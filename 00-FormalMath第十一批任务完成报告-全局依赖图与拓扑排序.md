- **数据模型**
  - `Concept` 概念数据类
  - `KnowledgeLevel` 知识层次枚举

  - D6 数论：10个概念
  - D7 离散数学：10个概念
  - D8 交叉领域：10个概念

- **核心算法**
  - `topological_sort_kahn()` - Kahn算法
  - `topological_sort_dfs()` - DFS算法（验证用）
  - `topological_sort_with_levels()` - 层次化排序

- **学习路径生成**
  - `LearningPathGenerator` 类
  - 个性化路径生成
  - 学习时间估计
  - 并行学习组识别

- **可视化输出**
  - `generate_mermaid_graph()` - 全局Mermaid图
  - `generate_domain_subgraph()` - 领域子图

- **统计分析**
  - `analyze_graph()` - 图结构统计
  - `detect_cycles()` - 环检测

## 🔧 使用方法

### 快速开始

```bash
# 1. 运行主程序
cd project
python global_dependency_graph.py

# 2. 运行示例
cd project
python example_usage.py
```

### Python API

```python
from global_dependency_graph import (
    get_core_concepts,
    ConceptDependencyGraph,
    LearningPathGenerator,
    topological_sort_kahn
)

# 构建图
concepts = get_core_concepts()
graph = ConceptDependencyGraph()
graph.build_from_concepts(concepts)

# 拓扑排序
sorted_ids = topological_sort_kahn(graph)

# 生成学习路径
generator = LearningPathGenerator(graph)
path = generator.generate_path(["C017"], DifficultyLevel.INTERMEDIATE)
```
