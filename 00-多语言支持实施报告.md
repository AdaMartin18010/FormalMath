## 一、项目背景与目标

### 1.1 项目背景

FormalMath项目目前主要是中文内容。为了提升项目的国际影响力，服务全球数学学习者，需要提供多语言支持，特别是英文翻译，以及日/法/德文术语对照。

### 1.2 现有资源

- **00-标准术语表-8语言.md**: 包含8语言术语对照（v1.0，约460术语）
- **multilang_concept_table.json**: 多语言概念表（约200+概念）

### 1.3 任务目标

1. 核心文档英文翻译
2. 完善多语言术语对照表
3. 建立多语言支持框架

### 4.2 语言切换机制（建议）

#### 方案A：文档内链接
在文档顶部添加语言选择链接：
```markdown
**Languages**: [中文](README.md) | [English](README.en.md) | [日本語](README.ja.md)
```

#### 方案B：前端语言切换（Web部署）
建议前端实现：
```javascript
// 语言配置
const i18n = {
  zh: { name: '中文', path: '/zh/' },
  en: { name: 'English', path: '/en/' },
  ja: { name: '日本語', path: '/ja/' },
  de: { name: 'Deutsch', path: '/de/' },
  fr: { name: 'Français', path: '/fr/' }
};
```

#### 方案C：URL参数方案
```
https://formalmath.org/docs/overview?lang=en
https://formalmath.org/docs/overview?lang=zh
```
