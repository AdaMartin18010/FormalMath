**集合 (Set)**
```
定义：集合是由确定的不同元素组成的无序整体。
ZFC公理：外延性、空集、配对、并集、幂集、无穷、替换、选择
符号：A, B, C（集合）；a ∈ A（元素属于）；∅（空集）
```

**函数 (Function)**
```
定义：设A和B是集合，函数f: A→B是笛卡尔积A×B的子集，满足：
  - 完全性：∀a∈A, ∃b∈B: (a,b)∈f
  - 唯一性：(a,b₁)∈f ∧ (a,b₂)∈f ⇒ b₁=b₂
符号：f: A→B；dom(f)；im(f)
```

**自然数 (Natural Number)**
```
定义：自然数是用于计数的数集，记作ℕ = {0,1,2,3,...}
Peano公理：
  1. 0是自然数
  2. 每个自然数n有唯一后继S(n)
  3. 0不是任何自然数的后继
  4. 不同自然数有不同后继
  5. 归纳公理
```

#### 形式化定义摘要

**群 (Group)**
```
定义：群是有序对(G,·)，其中G是非空集合，·: G×G→G是二元运算：
  1. 结合律：∀a,b,c∈G: (a·b)·c = a·(b·c)
  2. 单位元：∃e∈G, ∀a∈G: e·a = a·e = a
  3. 逆元：∀a∈G, ∃a⁻¹∈G: a·a⁻¹ = a⁻¹·a = e
符号：e或1_G（单位元）；a⁻¹（逆元）；|G|（群的阶）
```

**环 (Ring)**
```
定义：环是有序三元组(R,+,·)：
  1. (R,+)是交换群
  2. (R,·)是半群（满足结合律）
  3. 分配律：a·(b+c) = a·b + a·c 且 (a+b)·c = a·c + b·c
特殊类型：交换环、含幺环、整环、域
```

**域 (Field)**
```
定义：域是交换环(F,+,·)，其中0≠1，且每个非零元有乘法逆元：
  ∀a∈F, a≠0 ⇒ ∃a⁻¹∈F: a·a⁻¹ = 1
等价表述：
  - (F,+)是交换群
  - (F\{0},·)是交换群
  - 分配律成立
符号：ℚ, ℝ, ℂ（常见域）；𝔽_p（p元有限域）
```

**向量空间 (Vector Space)**
```
定义：设F是域，F-向量空间是有序对(V,+,·)：
  1. (V,+)是交换群
  2. 标量乘法·: F×V→V满足：
     - a·(v+w) = a·v + a·w
     - (a+b)·v = a·v + b·v
     - (ab)·v = a·(b·v)
     - 1·v = v
符号：V（向量空间）；u,v,w（向量）；a,b,c（标量）
```

#### 形式化定义摘要

**极限 (Limit)**
```
定义（数列极限）：数列{aₙ}收敛于L，记作lim_{n→∞} aₙ = L：
  ∀ε>0, ∃N∈ℕ, ∀n≥N: |aₙ - L| < ε

定义（函数极限）：lim_{x→a} f(x) = L：
  ∀ε>0, ∃δ>0, ∀x: 0<|x-a|<δ ⇒ |f(x)-L|<ε

符号：ε（任意小正数）；δ（依赖于ε的正数）；aₙ→L
```

**导数 (Derivative)**
```
定义：函数f在点a处的导数：
  f'(a) = lim_{h→0} [f(a+h) - f(a)]/h

等价表述：f'(a) = lim_{x→a} [f(x) - f(a)]/(x-a)

导函数：若f在每点可导，则f': x↦f'(x)称为导函数

符号：f'(x)或df/dx（Leibniz记法）；∂f/∂x（偏导数）
```

#### 形式化定义摘要

**图 (Graph)**
```
定义：图G是有序对(V,E)：
  - 无向图：E ⊆ (V choose 2)（V的2元子集族）
  - 有向图：E ⊆ V×V
  - 多重图：允许重边和自环

度：顶点v的度deg(v)是与v关联的边数

符号：G=(V,E)；V(G)（顶点集）；E(G)（边集）；n=|V|；m=|E|
```

**组合数 (Binomial Coefficient)**
```
定义：对非负整数n≥k，组合数：
  C(n,k) = n!/(k!(n-k)!)

表示从n个元素中选取k个的方式数

推广：对任意实数r和非负整数k：
  C(r,k) = r(r-1)...(r-k+1)/k!

符号：C(n,k)或(n choose k)（二项式系数）；n!（阶乘）
```
