# 08-数学历史

## 目录说明

本目录研究数学的历史发展，包括各个时期的数学成就、思想演变和文化传承。

## 研究内容

### 1. 古代数学（约公元前3000年-公元500年）

- **古希腊数学**: 欧几里得几何、毕达哥拉斯定理、阿基米德方法
- **古中国数学**: 《九章算术》、勾股定理、圆周率计算
- **古印度数学**: 十进制系统、零的概念、代数方法
- **古阿拉伯数学**: 代数学、三角学、算法

### 2. 中世纪数学（约500年-1500年）

- **欧洲数学**: 经院哲学、数学教育
- **伊斯兰数学**: 代数学发展、几何学、天文学
- **中国宋元数学**: 高次方程、天元术、四元术

### 3. 近代数学（17-19世纪）

- **17世纪**: 微积分创立（牛顿、莱布尼茨）、解析几何（笛卡尔）
- **18世纪**: 分析学发展、概率论、数论
- **19世纪**: 非欧几何、群论、集合论、分析严格化

### 4. 现代数学（20-21世纪）

- **20世纪早期**: 集合论基础、拓扑学、抽象代数
- **20世纪中期**: 范畴论、同调代数、代数几何
- **20世纪后期**: 数论突破、几何分析、动力系统
- **21世纪**: 计算数学、应用数学、跨学科研究

## 核心研究问题

1. 数学是如何发展的？
2. 各个时期的数学成就是什么？
3. 数学思想是如何演变的？
4. 不同文化对数学发展的贡献是什么？
