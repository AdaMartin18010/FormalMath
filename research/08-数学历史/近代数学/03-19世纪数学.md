## 📋 一、概述

### 1.1 历史地位

**19世纪**是数学史上最具变革性的时期：

- 现代数学的**诞生世纪**
- 从计算工具转向**结构科学**
- **严格化运动**彻底改变数学风貌
- 为20世纪数学奠定基础

### 1.2 主要革命

| 革命 | 核心变化 | 代表人物 |
|------|---------|---------|
| 分析严格化 | 无穷小→ε-δ语言 | 柯西、魏尔斯特拉斯 |
| 几何革命 | 欧几里得唯一性打破 | 高斯、罗巴切夫斯基、黎曼 |
| 代数革命 | 从方程到结构 | 伽罗瓦、阿贝尔、戴德金 |
| 基础革命 | 集合论基础 | 康托尔、弗雷格 |

### 1.3 社会背景

- 法国大革命后的科学体制化
- 普鲁士/德国大学改革
- 专业数学杂志创刊（Crelle's Journal, 1826）
- 国际数学共同体形成

## 📐 二、分析的严格化

### 2.1 危机与动机

**18世纪遗留问题**：

- 微积分基于模糊的"无穷小"概念
- 发散级数的矛盾使用
- 连续函数的病态例子

**伯克利主教的批判**（1734）：
> "消逝量的幽灵"——无穷小既为零又不为零

### 2.2 柯西的贡献

**Augustin-Louis Cauchy（1789-1857）**

**《分析教程》（1821）**核心创新：

| 概念 | 柯西的定义 |
|------|-----------|
| 极限 | "变量无限趋近于固定值" |
| 连续 | lim_{x→a} f(x) = f(a) |
| 收敛 | 部分和趋向有限极限 |
| 导数 | (f(x+h)-f(x))/h 的极限 |
| 积分 | 黎曼和的极限 |

**柯西序列**：
> 若对任意 ε>0，存在 N 使得 m,n>N 时 |a_m - a_n| < ε

### 2.3 魏尔斯特拉斯的完善

**Karl Weierstrass（1815-1897）**

**ε-δ语言的精确化**：

**极限的现代定义**：

```
lim_{x→a} f(x) = L ⟺
∀ε>0, ∃δ>0: 0<|x-a|<δ → |f(x)-L|<ε
```

**关键贡献**：

- 消除所有"运动"隐喻
- 连续性定义不依赖极限
- 一致连续与逐点连续的区分
- 反例构造：处处连续处处不可微函数

**处处连续处处不可微函数**（1872）：

```
f(x) = Σ_{n=0}^∞ (1/2)^n cos(3^n πx)
```

### 2.4 实数的严格构造

**1872年三种独立构造**：

| 方法 | 创造者 | 核心思想 |
|------|-------|---------|
| 戴德金分割 | 戴德金 | 有理数集的分划 |
| 柯西序列 | 康托尔 | 有理柯西序列的等价类 |
| 无限小数 | 梅雷 | 十进制展开 |

**戴德金分割**：
实数 r 定义为有理数集 Q 的一个分划 (A,B)，满足：

- A ∪ B = Q, A ∩ B = ∅
- a ∈ A, b ∈ B → a < b
- A 无最大元

## 🌐 三、非欧几何革命

### 3.1 平行公设问题

**欧几里得第五公设**：
> 过直线外一点，有且只有一条直线与原直线平行

**2000年的困扰**：

- 尝试从其他公设导出
- 所有尝试都失败
- 暗示可能独立

### 3.2 非欧几何的发现

**三位独立发现者**：

**高斯（Carl Friedrich Gauss, 1777-1855）**：

- 1820年代私下发展
- 因害怕争议未发表
- 信件中提及"反欧几里得几何"

**罗巴切夫斯基（Nikolai Lobachevsky, 1792-1856）**：

- 1826年首次公开发表
- 称为"虚几何"
- 系统发展双曲几何

**鲍耶（János Bolyai, 1802-1860）**：

- 1832年发表《空间的绝对科学》
- 与父亲（高斯友人）的通信

**双曲几何核心性质**：

| 欧氏 | 双曲 |
|------|------|
| 一条平行线 | 无穷多条"平行线" |
| 三角形内角和 = π | 三角形内角和 < π |
| 相似三角形无穷多 | 相似必全等 |

### 3.3 黎曼几何

**Bernhard Riemann（1826-1866）**

**1854年就职讲演**：《论几何学基础假设》

**核心创新**：

1. **多重广延的概念（流形）**
   - n维空间的一般化
   - 局部坐标系

2. **度量张量**
   - ds² = Σ g_{ij} dx^i dx^j
   - 曲率由度量决定

3. **内蕴几何**
   - 不依赖嵌入的几何性质
   - 高斯的"绝妙定理"的推广

4. **曲率**
   - 黎曼曲率张量
   - 正/零/负曲率空间

**影响**：

- 广义相对论的数学基础
- 现代微分几何
- 拓扑学的源头

### 3.4 几何基础反思

**克莱因的埃尔朗根纲领（1872）**：
> 几何学是研究在变换群下不变的性质

| 几何类型 | 变换群 | 不变量 |
|---------|--------|-------|
| 欧氏几何 | 等距变换 | 距离、角度 |
| 仿射几何 | 仿射变换 | 平行性 |
| 射影几何 | 射影变换 | 交比 |
| 拓扑学 | 同胚 | 连通性、紧致性 |

## 🔷 四、抽象代数的诞生

### 4.1 五次方程问题

**历史背景**：

- 二次方程：巴比伦时代
- 三次方程：卡尔达诺（1545）
- 四次方程：费拉里（1545）
- 五次方程：无通用求根公式？

### 4.2 阿贝尔与伽罗瓦

**尼尔斯·阿贝尔（1802-1829）**：

- 1824年证明五次方程没有根式解
- 阿贝尔群的命名来源
- 27岁去世

**埃瓦里斯特·伽罗瓦（1811-1832）**：

- 揭示可解性的深层结构
- 发明"群"的概念
- 决斗前夜写下革命性论文
- 21岁去世

### 4.3 伽罗瓦理论

**核心思想**：

**伽罗瓦群**：
多项式的对称群，由保持多项式根之间代数关系的置换组成

**可解性定理**：
> 多项式方程可用根式求解 ⟺ 其伽罗瓦群可解

**可解群定义**：
群 G 可解，若存在链

```
{e} = G₀ ⊲ G₁ ⊲ ... ⊲ Gₙ = G
```

其中每个 G_{i+1}/G_i 是阿贝尔群

**五次方程不可解的原因**：

- 一般五次方程的伽罗瓦群是 S₅
- S₅ 不是可解群（A₅ 是非阿贝尔单群）

### 4.4 抽象群论的发展

**从置换到抽象**：

| 阶段 | 时间 | 代表 |
|------|------|------|
| 置换群 | 1830s | 伽罗瓦 |
| 几何群 | 1870s | 克莱因 |
| 抽象群 | 1880s | 戴德金、弗罗贝尼乌斯 |
| 公理化 | 1900s | 韦伯 |

### 4.5 其他代数发展

**四元数（Hamilton, 1843）**：

- 非交换代数的首例
- i² = j² = k² = ijk = -1
- 开启代数结构研究

**布尔代数（Boole, 1854）**：

- 逻辑的代数化
- 计算机科学的先驱

**理想论（Dedekind, Kummer）**：

- 代数数论的基础
- 唯一分解的推广

## ∞ 五、集合论与基础危机

### 5.1 康托尔的集合论

**Georg Cantor（1845-1918）**

**无穷的革命**：

| 发现 | 时间 | 意义 |
|------|------|------|
| 有理数可数 | 1873 | 可数无穷 |
| 实数不可数 | 1873 | 对角线论证 |
| 超穷基数 | 1874-1884 | ℵ₀, ℵ₁, ... |
| 连续统假设 | 1878 | 至今独立 |

**对角线论证**：
假设实数可列举为 r₁, r₂, r₃, ...
构造 r：第n位小数不等于 rₙ 的第n位
则 r ≠ rₙ 对所有 n，矛盾

**基数比较**：

- |ℕ| = ℵ₀
- |ℝ| = 2^{ℵ₀} = c（连续统势）
- 连续统假设：c = ℵ₁?

### 5.2 集合论悖论

**康托尔悖论（1899）**：
"所有集合的集合"导致矛盾

**罗素悖论（1901）**：
令 R = {x : x ∉ x}
则 R ∈ R ⟺ R ∉ R

**影响**：

- 朴素集合论不一致
- 数学基础危机
- 导致公理化集合论

### 5.3 基础的重建

**三大纲领**：

| 纲领 | 代表 | 核心主张 |
|------|------|---------|
| 逻辑主义 | 弗雷格、罗素 | 数学可还原为逻辑 |
| 形式主义 | 希尔伯特 | 数学是符号操作 |
| 直觉主义 | 布劳威尔 | 数学是心智构造 |

**弗雷格的逻辑主义**：

- 《概念文字》（1879）：现代逻辑的诞生
- 《算术基础》（1884）：数的逻辑定义
- 被罗素悖论击溃
