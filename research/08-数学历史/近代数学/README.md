# 近代数学

## 目录说明

本目录研究近代数学的发展（17-19世纪），包括微积分、代数、几何、分析等领域的重大发展。

## 研究内容

### 1. 17世纪数学

- 微积分的发明
- 解析几何
- 概率论

### 2. 18世纪数学

- 微积分的发展
- 分析学
- 变分法

### 3. 19世纪数学

- 分析严格化
- 非欧几何
- 代数发展
- 集合论
