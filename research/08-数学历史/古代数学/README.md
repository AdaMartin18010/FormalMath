# 古代数学

## 目录说明

本目录研究古代数学的发展，包括古希腊、古中国、古印度、古阿拉伯等文明的数学成就。

## 研究内容

### 1. 古希腊数学

- 欧几里得几何
- 毕达哥拉斯学派
- 阿基米德方法
- 阿波罗尼奥斯圆锥曲线

### 2. 古中国数学

- 《九章算术》
- 勾股定理
- 圆周率计算
- 天元术

### 3. 古印度数学

- 十进制系统
- 零的概念
- 代数方法
- 三角学

### 4. 古阿拉伯数学

- 代数学
- 三角学
- 算法
- 数学翻译
