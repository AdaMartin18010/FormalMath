# 中世纪数学

## 目录说明

本目录研究中世纪数学的发展，包括欧洲、伊斯兰、中国宋元等时期的数学成就。

## 研究内容

### 1. 欧洲中世纪数学

- 经院哲学
- 数学教育
- 数学翻译

### 2. 伊斯兰数学

- 代数学发展
- 三角学发展
- 数学翻译和传播

### 3. 中国宋元数学

- 天元术和四元术
- 高次方程求解
- 数学应用
