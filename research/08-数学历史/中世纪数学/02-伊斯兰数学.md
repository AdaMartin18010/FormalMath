## 📋 一、概述

### 1.1 历史背景

**伊斯兰数学黄金时代**（约750-1450年）：

| 时期 | 特征 | 中心 |
|------|------|------|
| 早期（750-900）| 翻译运动、吸收 | 巴格达 |
| 鼎盛期（900-1200）| 原创发展、创新 | 巴格达、开罗、科尔多瓦 |
| 后期（1200-1450）| 传播、深化 | 撒马尔罕、伊斯坦布尔 |

**翻译运动（智慧之家）**：

- 阿拔斯王朝哈里发支持
- 翻译希腊、印度、波斯数学著作
- 欧几里得《几何原本》、托勒密《天文学大成》
- 印度数学著作（包括数字系统）

### 1.2 主要特点

| 传承性 | 保存并传播古典数学 |

## 👥 二、重要数学家

### 2.1 花拉子密（al-Khwārizmī, 约780-850）

**"代数之父"**

**主要著作**：

| 著作 | 内容 | 意义 |
|------|------|------|
| 《还原与对消的计算》| 系统的代数方法 | "Algebra"一词来源 |
| 《印度数字算术》| 印度-阿拉伯数字 | "Algorithm"一词来源 |
| 《天文表》| 天文计算 | 三角学应用 |

**代数贡献**：

六种标准方程形式（现代记法）：

```
ax² = bx        ax² = c        bx = c
ax² + bx = c    ax² + c = bx   bx + c = ax²
```

**几何证明**：
用几何方法证明代数公式，如完成平方法

### 2.2 奥马尔·海亚姆（Omar Khayyam, 1048-1131）

**波斯诗人与数学家**

**数学贡献**：

| 领域 | 贡献 |
|------|------|
| 代数 | 三次方程的几何解法 |
| 几何 | 平行线公理研究 |
| 数论 | 二项式系数研究 |

**三次方程解法**：

- 用圆锥曲线相交求解
- 分类讨论各种情况
- 首次系统处理三次方程

**平行公设研究**：

- 研究欧几里得第五公设
- 提出等价命题
- 为非欧几何开先河

### 2.3 阿布·瓦法（Abū al-Wafā', 940-998）

**三角学大师**

**贡献**：

| 贡献 | 内容 |
|------|------|
| 三角函数 | 引入正切、余切、正割、余割 |
| 三角恒等式 | 系统整理三角恒等式 |
| 计算表 | 编制精确的三角函数表 |
| 球面三角 | 发展球面三角学 |

### 2.4 纳西尔丁·图西（Nasīr al-Dīn al-Tūsī, 1201-1274）

**贡献**：

| 领域 | 贡献 |
|------|------|
| 三角学 | 《论四边形》——首部独立三角学著作 |
| 天文学 | 马拉盖天文台 |
| 几何学 | 平行公设研究 |

**图西圆对**：
用两个嵌套圆的运动解释行星运动

### 2.5 卡西（Jamshīd al-Kāshī, 约1380-1429）

**计算大师**

**贡献**：

| 贡献 | 内容 |
|------|------|
| π的计算 | 计算到16位小数 |
| 小数系统 | 系统使用十进制小数 |
| 三次方程 | 数值求解方法 |

## 📐 三、主要数学成就

### 3.1 代数学

**从修辞代数到符号萌芽**：

| 发展阶段 | 特征 |
|---------|------|
| 修辞代数 | 完全用文字表述 |
| 简略记法 | 开始使用缩写 |
| 符号萌芽 | 部分符号使用 |

**主要内容**：

- 一元二次方程的系统解法
- 三次方程的几何解法
- 多项式运算
- 根号运算

**示例：完成平方法解 x² + 10x = 39**

花拉子密的几何方法：

```
     5
  ┌─────┬─────┐
  │     │     │
  │  x² │ 5x  │ x
  │     │     │
  ├─────┼─────┤
  │ 5x  │ 25  │ 5
  └─────┴─────┘
     x     5

(x + 5)² = 39 + 25 = 64
x + 5 = 8
x = 3
```

### 3.2 三角学

**从天文学附属到独立学科**：

| 发展 | 内容 |
|------|------|
| 函数扩展 | 六个三角函数完整体系 |
| 恒等式 | 和差公式、倍角公式等 |
| 球面三角 | 完整的球面三角学 |
| 计算表 | 高精度三角函数表 |

**正弦定律**（al-Tūsī形式）：

```
a/sin A = b/sin B = c/sin C
```

### 3.3 数字系统

**印度-阿拉伯数字的传播**：

| 阶段 | 内容 |
|------|------|
| 引入 | 从印度引入位值制 |
| 发展 | 完善零的概念和用法 |
| 传播 | 传入欧洲（斐波那契）|

**小数记法**：

- 卡西系统使用十进制小数
- 比欧洲早约200年

### 3.4 数论

**主要贡献**：

| 内容 | 数学家 |
|------|--------|
| 完全数研究 | Ibn Qurra |
| 友爱数 | Ibn Qurra（发现第二对）|
| 二项式系数 | 海亚姆（"帕斯卡"三角形）|

### 3.5 几何学

**平行公设研究**：

| 数学家 | 贡献 |
|--------|------|
| 海亚姆 | 用四边形分析 |
| 图西 | 尝试证明、实际上发现等价命题 |
| 萨其里 | 继续研究（影响欧洲）|

**海亚姆四边形**：
研究底边两邻角相等的四边形，开启非欧几何先声

## 🌍 五、影响与遗产

### 5.1 对欧洲的影响

**传播路径**：

```
伊斯兰世界 → 西班牙（托莱多翻译学派）→ 欧洲
伊斯兰世界 → 意大利（商业接触）→ 欧洲
```

**关键人物**：

- 阿德拉德（Bath的Adelard）：翻译欧几里得
- 克雷莫纳的杰拉德：大量翻译
- 斐波那契：引入印度-阿拉伯数字

### 5.2 主要遗产

| 遗产 | 内容 |
|------|------|
| 代数学 | 系统化的代数方法 |
| 数字系统 | 印度-阿拉伯数字 |
| 三角学 | 独立的三角学学科 |
| 术语 | "algebra"、"algorithm"等 |

### 5.3 保存古典知识

**关键作用**：

- 翻译保存希腊数学著作
- 在欧洲中世纪黑暗期保护知识
