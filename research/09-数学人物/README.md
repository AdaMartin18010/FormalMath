# 09-数学人物

## 目录说明

本目录研究重要数学家的生平、贡献、思想，特别关注他们的语言表达、数学表达风格、论证风格、解释方式和写作特点。

## 研究内容

### 1. 古代数学家

- **古希腊**: 欧几里得、阿基米德、阿波罗尼奥斯、毕达哥拉斯
- **古中国**: 刘徽、祖冲之、秦九韶、李冶
- **古印度**: 婆罗摩笈多、婆什迦罗
- **古阿拉伯**: 花拉子米、奥马尔·海亚姆

### 2. 近代数学家（17-19世纪）

- **17世纪**: 牛顿、莱布尼茨、笛卡尔、费马
- **18世纪**: 欧拉、拉格朗日、拉普拉斯、高斯
- **19世纪**: 柯西、黎曼、康托尔、希尔伯特

### 3. 现代数学家（20-21世纪）

- **20世纪早期**: 庞加莱、希尔伯特、哥德尔、图灵
- **20世纪中期**: 格罗滕迪克、阿蒂亚、塞尔、朗兰兹
- **20世纪后期**: 怀尔斯、佩雷尔曼、陶哲轩
- **21世纪**: 当代重要数学家

## 研究重点

### 特别关注

1. **数学家的语言和表达方式**
   - 数学术语的使用
   - 符号系统的选择
   - 表达风格的特色

2. **数学表达风格**
   - 简洁 vs 详细
   - 直观 vs 抽象
   - 几何 vs 代数

3. **论证风格**
   - 严格性程度
   - 证明方法偏好
   - 逻辑结构特点

4. **解释方式**
   - 如何解释数学概念
   - 如何解释数学定理
   - 如何解释数学思想

5. **写作特点**
   - 写作风格
   - 组织结构
   - 读者定位

## 核心研究问题

1. 重要数学家的贡献是什么？
2. 他们的数学思想是什么？
3. 他们的表达和论证风格有什么特点？
4. 他们的语言和写作对数学发展有什么影响？
