## 📋 一、生平简介

### 1.1 基本信息

- **全名**：Leonhard Euler
- **生卒年**：1707年4月15日 - 1783年9月18日
- **出生地**：瑞士巴塞尔
- **主要领域**：几乎所有数学领域、物理学、天文学

### 1.2 生平轨迹

**早年（1707-1727）**：

- 父亲是牧师，受过数学训练
- 13岁进入巴塞尔大学
- 师从约翰·伯努利
- 1727年受邀前往圣彼得堡

**圣彼得堡第一期（1727-1741）**：

- 在俄国科学院工作
- 产出惊人的研究成果
- 1738年右眼失明
- 1741年应普鲁士王邀请前往柏林

**柏林时期（1741-1766）**：

- 在普鲁士科学院工作25年
- 与弗里德里希大帝关系复杂
- 完成大量著作
- 1766年返回俄国

**圣彼得堡第二期（1766-1783）**：

- 1771年完全失明
- 产出未减反增
- 1783年9月18日工作中去世
- 据说去世当天还在计算

### 1.3 惊人产出

**数量统计**：

| 类别 | 数量 |
|------|------|
| 发表论文 | 约500篇 |
| 著作 | 约25部 |
| 书信 | 数千封 |
| 全集 | 80卷（仍在出版）|

**著名轶事**：
> 圣彼得堡科学院在欧拉去世后仍花了47年才发表完他留下的论文

## 📐 二、分析学贡献

### 2.1 函数概念的确立

**函数定义的革命**：

| 阶段 | 函数观 |
|------|--------|
| 欧拉之前 | 几何曲线的表达式 |
| 欧拉 | 由变量组成的任意解析表达式 |
| 现代 | 集合间的映射 |

**《无穷小分析引论》（1748）**：

- 函数成为分析的核心对象
- 系统发展无穷级数
- 研究超越函数

### 2.2 级数理论

**无穷级数的大师**：

| 贡献 | 内容 |
|------|------|
| 巴塞尔问题 | ∑ 1/n² = π²/6 |
| 指数级数 | e^x = ∑ x^n/n! |
| 三角级数 | sin x, cos x 的级数 |
| 调和级数 | ∑ 1/n 发散 |

**欧拉-麦克劳林公式**：
连接求和与积分

```
∑f(k) ≈ ∫f(x)dx + 修正项（涉及伯努利数）
```

### 2.3 欧拉公式

**最美公式**：

```
e^(iπ) + 1 = 0
```

**一般形式**：

```
e^(ix) = cos x + i sin x
```

**意义**：

- 连接指数函数与三角函数
- 连接实数与复数
- 复分析的基础

### 2.4 微分方程

**贡献**：

| 类型 | 方法/结果 |
|------|----------|
| 常微分方程 | 常数变易法 |
| 线性方程 | 特征根方法 |
| 欧拉方程 | x^n y^(n) + ... 的求解 |
| 变分法 | 欧拉-拉格朗日方程 |

**欧拉-拉格朗日方程**：

```
∂L/∂y - d/dx(∂L/∂y') = 0
```

变分问题的基本方程

### 2.5 特殊函数

**伽马函数**：

```
Γ(n) = (n-1)!  （推广阶乘到实数/复数）
Γ(z) = ∫₀^∞ t^(z-1) e^(-t) dt
```

**贝塔函数**：

```
B(x,y) = ∫₀^1 t^(x-1)(1-t)^(y-1) dt
```

**欧拉积分**：开创特殊函数理论

## 🔢 三、数论贡献

### 3.1 费马定理的证明

**费马小定理**：
若 p 为素数，gcd(a,p)=1，则

```
a^(p-1) ≡ 1 (mod p)
```

欧拉首次给出完整证明（1736年）

### 3.2 欧拉定理

**推广费马小定理**：
若 gcd(a,n)=1，则

```
a^φ(n) ≡ 1 (mod n)
```

**欧拉函数 φ(n)**：
小于n且与n互素的正整数个数

**计算公式**：

```
φ(n) = n ∏(1 - 1/p)
```

其中积取遍n的所有素因子p

### 3.3 素数研究

**欧拉乘积公式**：

```
∑ 1/n^s = ∏ 1/(1 - p^(-s))
```

左边对所有正整数n求和，右边对所有素数p求积

**意义**：

- 连接分析与数论
- 黎曼ζ函数的起源
- 素数无穷的分析证明

**欧拉证明素数无穷**：
若素数有限，∏ 1/(1 - 1/p) 有限，但 ∑ 1/n = ∞

### 3.4 二次互反律

- 发现并猜想二次互反律
- 未能证明（后由高斯证明）
- 开启代数数论方向

### 3.5 数论函数

| 函数 | 定义 |
|------|------|
| φ(n) | 欧拉函数 |
| σ(n) | 因子和函数 |
| d(n) | 因子个数函数 |

## 📊 四、几何与组合贡献

### 4.1 多面体公式

**欧拉公式**：

```
V - E + F = 2
```

- V = 顶点数
- E = 边数
- F = 面数

**意义**：

- 拓扑学的开端
- 拓扑不变量的首例
- 欧拉示性数的起源

### 4.2 图论的创立

**柯尼斯堡七桥问题（1736）**：

```
    A
   /|\
  / | \
 B--C--D
  \ | /
   \|/
    E
```

能否一次走过所有桥各一次？

**欧拉的分析**：

- 引入"图"的概念
- 定义"度"（连接边数）
- **定理**：存在欧拉路径当且仅当奇度顶点数为0或2

**开创图论学科**

### 4.3 拉丁方与组合设计

**拉丁方**：
n×n矩阵，每行每列恰好包含每个元素一次

**36军官问题**：
能否用36个军官（6种军衔，6个团）排成6×6方阵，使每行每列各军衔各团各恰一人？

欧拉猜想不可能（1959年证明正确）

### 4.4 分拆理论

**整数分拆**：
将正整数写成正整数之和的方式

**生成函数方法**：

```
∑ p(n) x^n = ∏ 1/(1-x^k)
```

**五边形数定理**：

```
∏(1-x^n) = ∑ (-1)^k x^(k(3k-1)/2)
```
