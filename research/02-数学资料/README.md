# 02-数学资料

## 目录说明

本目录收集和整理数学的核心概念、理论、应用和工具，作为项目的基础资料库。

## 内容分类

### 1. 核心概念

- 基础数学概念（集合、函数、关系、数系等）
- 各分支核心概念（群、环、域、拓扑空间等）
- 高级数学概念（范畴、层、概形等）

### 2. 数学理论

- 基础理论（集合论、逻辑学、数论等）
- 代数理论（群论、环论、域论、表示论等）
- 分析理论（实分析、复分析、泛函分析等）
- 几何理论（欧氏几何、微分几何、代数几何等）
- 拓扑理论（点集拓扑、代数拓扑、微分拓扑等）

### 3. 数学应用

- 自然科学应用（物理、化学、生物等）
- 工程技术应用（计算机、通信、控制等）
- 社会科学应用（经济、社会、心理等）

### 4. 数学工具

- 计算工具（数值计算、符号计算等）
- 证明工具（形式化证明、定理证明器等）
- 可视化工具（图形、动画、交互式演示等）

## 文件组织

- `核心概念/` - 33个核心概念的三视角版本
- `数学理论/` - 各分支数学理论
- `数学应用/` - 数学在各领域的应用
- `数学工具/` - 数学工具和方法

- `05-解释/` - 数学解释和论证
