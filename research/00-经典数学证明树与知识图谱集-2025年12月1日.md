### 1.1 勾股定理证明树

**几何证明（面积法）**：

```text
                    [勾股定理: a² + b² = c²]
                              │
                 ┌────────────┴────────────┐
                 │                          │
        [大正方形面积计算]           [四个三角形面积计算]
                 │                          │
         (a + b)² = a² + 2ab + b²      4 × (½ab) = 2ab
                 │                          │
                 └────────────┬────────────┘
                              │
                   [大正方形 = 小正方形 + 4三角形]
                              │
                   (a + b)² = c² + 2ab
                              │
                   a² + 2ab + b² = c² + 2ab
                              │
                        a² + b² = c²  ∎
```

**代数证明树**：

```text
                    [勾股定理: a² + b² = c²]
                              │
           ┌──────────────────┴──────────────────┐
           │                                      │
    [设直角三角形顶点]                     [利用相似三角形]
           │                                      │
    A(0,0), B(a,0), C(0,b)                △ABC ~ △ACD ~ △BCD
           │                                      │
    [计算斜边长度]                        [面积/边长比例]
           │                                      │
    c = √[(a-0)² + (0-b)²]                 a²/c = c·(a²/c²)
           │                                      b²/c = c·(b²/c²)
    c² = a² + b²  ∎                               │
                                           a² + b² = c²  ∎
```

### 1.3 √2无理性证明树

```text
                        [√2 是无理数]
                               │
                        [反证法]
                               │
                    [假设 √2 = p/q 是有理数]
                    [其中 p,q 互素（最简分数）]
                               │
                        [两边平方]
                               │
                        2 = p²/q²
                               │
                        p² = 2q²
                               │
                ┌──────────────┴──────────────┐
                │                              │
         [p² 是偶数]                    [因此 p 是偶数]
                │                              │
         [设 p = 2k]                    [代入 p² = 2q²]
                │                              │
         (2k)² = 2q²                           │
                │                              │
         4k² = 2q²                             │
                │                              │
         2k² = q²                              │
                │                              │
                └──────────────┬──────────────┘
                               │
                        [q² 是偶数]
                               │
                        [因此 q 是偶数]
                               │
                   [p 和 q 都是偶数，与互素矛盾]
                               │
                        √2 是无理数  ∎
```

### 1.4 Cantor对角线论证

**实数不可数证明树**：

```text
                   [R（实数集）不可数]
                            │
                    [反证法]
                            │
           [假设 (0,1) 中的实数可列]
                            │
           设为 r₁, r₂, r₃, ...
           每个 rᵢ = 0.d_{i1}d_{i2}d_{i3}...
                            │
          ┌─────────────────┴─────────────────┐
          │                                    │
    [构造对角线数列]                      [构造新数]
          │                                    │
    d₁₁, d₂₂, d₃₃, ...                  x = 0.e₁e₂e₃...
          │                              其中 eᵢ ≠ dᵢᵢ
          │                                    │
          └─────────────────┬─────────────────┘
                            │
              [x ∈ (0,1) 但 x ≠ rᵢ 对所有i]
                            │
         [x 与 r₁ 的第1位不同 → x ≠ r₁]
         [x 与 r₂ 的第2位不同 → x ≠ r₂]
         [...]
                            │
              [x 不在列表中，与假设矛盾]
                            │
                    实数不可数  ∎
```

### 1.5 微积分基本定理证明树

**第一部分证明树**：

```text
          [微积分基本定理第一部分]
          [F(x) = ∫ₐˣ f(t)dt，则 F'(x) = f(x)]
                         │
              ┌──────────┴──────────┐
              │                      │
        [计算 F'(x)]           [使用极限定义]
              │                      │
        F'(x) = lim[h→0]            │
              (F(x+h) - F(x))/h      │
              │                      │
              └──────────┬──────────┘
                         │
            [F(x+h) - F(x) = ∫ₓˣ⁺ʰ f(t)dt]
                         │
                  [积分中值定理]
                         │
            [存在 c ∈ [x, x+h] 使得]
            [∫ₓˣ⁺ʰ f(t)dt = f(c)·h]
                         │
            [F'(x) = lim[h→0] f(c)·h/h]
                         │
            [F'(x) = lim[h→0] f(c)]
                         │
            [h→0 时 c→x，由f连续]
                         │
                 F'(x) = f(x)  ∎
```

### 1.6 中值定理证明树

```text
                    [中值定理]
       [若 f 在 [a,b] 连续，(a,b) 可微]
       [则存在 c ∈ (a,b) 使 f'(c) = (f(b)-f(a))/(b-a)]
                         │
              ┌──────────┴──────────┐
              │                      │
        [构造辅助函数]          [应用Rolle定理]
              │                      │
        g(x) = f(x) - f(a)          │
              - [(f(b)-f(a))/(b-a)]·(x-a)
              │                      │
              └──────────┬──────────┘
                         │
              [验证 g(a) = 0 = g(b)]
                         │
         ┌───────────────┴───────────────┐
         │                                │
    [g(a) = f(a) - f(a) - 0]       [g(b) = f(b) - f(a)]
         │                         [- (f(b)-f(a))]
    g(a) = 0  ✓                          │
                                    g(b) = 0  ✓
                                         │
              └───────────────┬───────────────┘
                              │
                   [由Rolle定理]
                              │
               [存在 c ∈ (a,b) 使 g'(c) = 0]
                              │
          [g'(x) = f'(x) - (f(b)-f(a))/(b-a)]
                              │
          [0 = f'(c) - (f(b)-f(a))/(b-a)]
                              │
          f'(c) = (f(b)-f(a))/(b-a)  ∎
```

                     │

                     │
    ┌────────────────┴────────────────┐
    │                                  │

    │                                  │

    │                           │             │
[G被陪集覆盖]              [陪集等大]   [陪集不交]
    │                           │             │

    │                      (双射 h↦ah)    aH ∩ bH = ∅
    │                           │             │
    └────────────────┬────────────────┘       │
                     │                         │

                     │

                     │

                     │

### 7.1 群论核心定理证明树

#### 7.1.1 Sylow第一定理证明树

```text
                [Sylow第一定理]
    [若|G|=p^n·m，gcd(p,m)=1，则G有p^n阶子群]
                     │
            [使用群作用证明]
                     │
        ┌────────────┴────────────┐
        │                          │
    [G作用于X]                [计算轨道]
        │                          │
    X = {S⊂G : |S|=p^n}           │
        │                      |X| ≡ 0 (mod p)
        │                     但也 ≡ m (mod p)
        │                          │
        └────────────┬────────────┘
                     │
            [存在轨道大小=1]
                     │
            [存在S使gS=S对所有g∈G]
                     │
            [稳定子群 G_S = {g : gS=S}]
                     │
            |G_S| ≥ p^n
                     │
        ┌────────────┴────────────┐
        │                          │
    [|G_S| ≤ p^n]            [因此|G_S| = p^n]
        │                          │
    (由S的大小限制)                │
        │                          │
        └────────────┬────────────┘
                     │
            G有p^n阶子群(Sylow p-子群)  ∎
```

#### 7.1.2 同态基本定理证明树

```text
                [群同态基本定理]
        [若 φ: G → H 是群同态，则 G/ker(φ) ≅ im(φ)]
                        │
            ┌───────────┴───────────┐
            │                        │
    [构造映射ψ]                [证明ψ是同构]
            │                        │
    ψ: G/ker(φ) → im(φ)              │
    ψ(g·ker(φ)) = φ(g)               │
            │                        │
            └───────────┬───────────┘
                        │
        ┌───────────────┼───────────────┐
        │               │               │
    [良定义性]      [同态性]        [双射性]
        │               │               │
    若g₁K = g₂K      ψ(aK·bK)        单射：ker(ψ)={K}
    则g₁⁻¹g₂∈K       = ψ(abK)        满射：定义域到im
    即φ(g₁)=φ(g₂)    = φ(ab)              │
        │            = φ(a)φ(b)           │
        │            = ψ(aK)ψ(bK)         │
        │               │               │
        └───────────────┴───────────────┘
                        │
                G/ker(φ) ≅ im(φ)  ∎
```

#### 7.1.3 Cayley定理证明树

```text
                [Cayley定理]
        [每个群G同构于某个置换群的子群]
                     │
            [构造嵌入映射]
                     │
            φ: G → Sym(G)
            φ(g) = λ_g  (左乘映射)
            λ_g(x) = gx
                     │
        ┌────────────┴────────────┐
        │                          │
    [φ是同态]                  [φ是单射]
        │                          │
    φ(gh) = λ_{gh}             若φ(g) = φ(h)
    λ_{gh}(x) = (gh)x          则λ_g = λ_h
             = g(hx)           λ_g(e) = λ_h(e)
             = λ_g(λ_h(x))     g = h
             = (λ_g ∘ λ_h)(x)      │
        │                          │
    φ(gh) = φ(g)∘φ(h)              │
        │                          │
        └────────────┬────────────┘
                     │
            G ≅ im(φ) ≤ Sym(G)  ∎
```
