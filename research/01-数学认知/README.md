# 01-数学认知

## 目录说明

本目录专注于数学认知的科学研究，从多个学科视角探索数学思维的内在机制。

## 研究领域

### 1. 神经科学视角

- 脑区定位：顶叶、额叶在数学认知中的作用
- 神经关联：数学符号处理的神经机制
- 空间认知关系：数学认知与空间认知的神经重叠

### 2. 认知心理学视角

- CPFS结构理论：概念域、概念系、命题域、命题系
- 问题解决过程：数学问题解决的认知策略
- 概念形成：从具体到抽象的认知转换

### 3. 认知语言学视角

- 数学语言处理：数学符号与自然语言的认知差异
- 术语认知：数学术语的生成、使用与演变

### 4. 认知模型视角

- LIDA认知架构：从知觉到推理的完整认知模型
- 神经符号学习：连接主义与符号系统的整合

### 5. 发展心理学视角

- 认知发展阶段：数学认知能力的发展轨迹
- 教育干预：如何促进数学认知发展

## 核心研究问题

1. 数学认知的神经基础是什么？
2. 数学认知的结构如何组织？
3. 数学思维的过程如何展开？
4. 数学认知如何发展？
