# 07-数学哲学

## 目录说明

本目录研究数学哲学的基础问题，包括本体论、认识论、方法论和各种数学主义。

## 研究领域

### 1. 数学本体论

**核心问题**: 数学对象是否存在？如果存在，它们是什么？

**主要主义**:

- **实在论（Realism）**: 数学对象独立于人类思维而存在
- **形式主义（Formalism）**: 数学是符号游戏，没有独立存在
- **直觉主义（Intuitionism）**: 数学是心智构造，存在依赖于构造
- **构造主义（Constructivism）**: 数学对象必须能够被构造
- **逻辑主义（Logicism）**: 数学可以还原为逻辑
- **结构主义（Structuralism）**: 数学对象是结构中的位置
- **虚构主义（Fictionalism）**: 数学对象是虚构的，但数学仍然有用

### 2. 数学认识论

**核心问题**: 我们如何获得数学知识？数学知识的性质是什么？

- **数学知识来源**: 直觉、经验、推理、实验
- **数学知识性质**: 必然性、客观性、相对性、确定性
- **数学知识验证**: 证明、实验、应用、共识

### 3. 数学方法论

**核心问题**: 数学研究的方法是什么？如何发现和证明数学真理？

- **研究方法**: 公理化方法、构造方法、抽象方法
- **证明方法**: 演绎证明、归纳证明、反证法、构造性证明
- **发现方法**: 直觉、类比、实验、计算

### 4. 数学主义对比

- 各主义的理论基础
- 各主义的优缺点
- 各主义的应用范围
- 各主义的发展历史

## 核心研究问题
