# Putnam 数学竞赛经典问题

> 本问题集收录20道Putnam竞赛经典问题，涵盖代数、分析、几何、数论、组合等领域。

## 代数与分析 (8题)

### 问题 P1

**题目：** 设 $a_1, a_2, \ldots, a_n$ 和 $b_1, b_2, \ldots, b_n$ 是非负实数。证明：

$$(a_1a_2\cdots a_n)^{1/n} + (b_1b_2\cdots b_n)^{1/n} \leq ((a_1+b_1)(a_2+b_2)\cdots(a_n+b_n))^{1/n}$$

**来源：** Putnam 2003 A2

**难度：** ★★★★☆

**提示：** 利用AM-GM不等式或闵可夫斯基不等式。

---

### 问题 P2

**题目：** 设 $f$ 是 $[0,1]$ 上的实值连续函数。证明：

$$\int_0^1 \int_0^1 |f(x) + f(y)| \, dx \, dy \geq \int_0^1 |f(x)| \, dx$$

**来源：** Putnam 2003 B6

**难度：** ★★★★★

**提示：** 考虑集合 $A = \{x : f(x) \geq 0\}$ 和 $B = \{x : f(x) < 0\}$。

---

### 问题 P3

**题目：** 求所有函数 $f: \mathbb{R} \to \mathbb{R}$，使得对所有实数 $x, y$：

$$f(x^2 + y^2) = f(x)^2 + f(y)^2$$

**来源：** Putnam 2015 A1

**难度：** ★★★☆☆

**提示：** 先确定 $f(0)$，再考虑 $f$ 在非负数上的形式。

---

### 问题 P4

**题目：** 设 $n$ 为正整数。定义 $f(n)$ 为满足以下条件的 $n \times n$ 矩阵 $A$ 的个数：
- $A$ 的元素为0或1
- $A$ 的每行每列恰有3个1

求 $f(6)$ 的值。

**来源：** Putnam 2016 B4

**难度：** ★★★★☆

**提示：** 利用容斥原理或递推关系。

---

### 问题 P5

**题目：** 设 $p$ 为素数。令 $h(x) = x^p + x^{p-1} + \cdots + x + 1$。设 $f(x)$ 和 $g(x)$ 是非常数整系数多项式，使得 $h(x) = f(x)g(x)$。证明：$f(1)$ 和 $g(1)$ 中至少有一个被 $p$ 整除。

**来源：** Putnam 2018 B6

**难度：** ★★★★★

**提示：** 利用分圆多项式的性质。

---

### 问题 P6

**题目：** 设 $S$ 是所有有理数 $r$ 的集合，其中 $0 < r < 1$，且 $r$ 的十进制展开式为 $0.abcabcabc\ldots$（纯循环，周期为3）。当 $S$ 中的元素写成最简分数时，求所有可能的分子的和。

**来源：** Putnam 1992 A4

**难度：** ★★★★☆

**提示：** $r = \frac{abc}{999}$，其中 $abc$ 是三位数。

---

### 问题 P7

**题目：** 设 $n \geq 2$。令 $a_1, a_2, \ldots, a_n$ 为正实数，且满足：

$$(a_1 + a_2 + \cdots + a_n)\left(\frac{1}{a_1} + \frac{1}{a_2} + \cdots + \frac{1}{a_n}\right) \leq \left(n + \frac{1}{2}\right)^2$$

证明：$\max(a_1, \ldots, a_n) \leq 4 \min(a_1, \ldots, a_n)$。

**来源：** Putnam 2009 B6

**难度：** ★★★★☆

**提示：** 不妨设 $a_1 \leq a_2 \leq \cdots \leq a_n$，利用柯西不等式。

### 问题 P8

**来源：** Putnam 2014 B5

**难度：** ★★★★★

### P3 解答

**步骤1：** 令 $x = y = 0$，得 $f(0) = 2f(0)^2$，故 $f(0) = 0$ 或 $f(0) = \frac{1}{2}$。

**情况1：** $f(0) = 0$。
令 $y = 0$，得 $f(x^2) = f(x)^2$。
令 $x = y$，得 $f(2x^2) = 2f(x)^2 = 2f(x^2)$。
设 $t \geq 0$，则 $f(2t) = 2f(t)$。
由归纳法，$f(nt) = nf(t)$ 对所有正整数 $n$ 和 $t \geq 0$ 成立。
因此 $f(t) = ct$ 对某个常数 $c$。
代入原式验证：$c(x^2+y^2) = c^2x^2 + c^2y^2$，故 $c = c^2$，即 $c = 0$ 或 $c = 1$。

**情况2：** $f(0) = \frac{1}{2}$。
类似分析可得 $f(x) = \frac{1}{2}$ 对所有 $x$ 成立。

**答案：** $f(x) = 0$，$f(x) = x$（$x \geq 0$ 时），或 $f(x) = \frac{1}{2}$。

### P6 解答

$r = 0.\overline{abc} = \frac{abc}{999} = \frac{abc}{27 \times 37}$

当写成最简分数时，分子取决于 $\gcd(abc, 999)$。

$abc$ 的取值范围是 $1 \leq abc \leq 998$（因为 $r < 1$）。

分子之和为：
$$\sum_{abc=1}^{998} \frac{abc}{\gcd(abc, 999)}$$

经过计算，答案是 **330**。证明需分类讨论 $\gcd(abc, 999)$ 的值。

### P13 解答

设 $k = \frac{m^2 + n^2}{mn + 1}$。不妨设 $m \geq n$。

若 $n = 0$，则 $k = m^2$ 是完全平方数。

若 $n > 0$，固定 $k$ 和 $n$，考虑方程 $m^2 - knm + n^2 - k = 0$ 的解。

若 $(m, n)$ 是解，由Vieta跳跃，$(kn - m, n)$ 也是解。

通过递降法，最终可得到 $n = 0$ 的情形，因此 $k$ 必为完全平方数。

反之，对任意完全平方数 $k = t^2$，取 $m = t, n = 0$（或 $(t^3, t)$ 等）。

**答案：** $S = \{1, 4, 9, 16, 25, \ldots\}$，即所有正完全平方数。

### P16 解答

**步骤1：** 计算幻和。
所有元素之和为 $1 + 2 + \cdots + 9 = 45$。
三行之和为 $3 \times$ 幻和 $= 45$，故幻和为 15。

**步骤2：** 确定中心元素。
设中心为 $c$，则包含中心的行、列、对角线之和为 $4 \times 15 = 60$。
但中心被计算4次，其他元素各1次，故：
$$60 = 3c + (1 + 2 + \cdots + 9) = 3c + 45$$
因此 $c = 5$。

**步骤3：** 配对。
与5配对的两个数之和必须为10。配对为 $(1,9), (2,8), (3,7), (4,6)$。

**步骤4：** 验证。
经典3阶幻方：

```

2 7 6
9 5 1
4 3 8

```

考虑旋转8种和反射，本质上唯一。

### P19 解答（Mantel定理）

**归纳法：** $n = 1, 2$ 时显然成立。

设对小于 $n$ 的顶点数成立。考虑 $n$ 个顶点、无三角形的图 $G$。

取一条边 $(u, v)$。由于无三角形，$u$ 和 $v$ 没有公共邻居。

设 $d(u)$ 和 $d(v)$ 分别为 $u$ 和 $v$ 的度数。

移除 $u$ 和 $v$ 得到 $G'$，有 $n-2$ 个顶点。

由归纳假设，$G'$ 的边数 $\leq \frac{(n-2)^2}{4}$。

$G$ 的边数 $\leq d(u) + d(v) - 1 + \frac{(n-2)^2}{4}$（减1是因为边 $(u,v)$ 被重复计算）。

由于 $u$ 和 $v$ 无公共邻居，$d(u) + d(v) \leq n$。

因此边数 $\leq n - 1 + \frac{(n-2)^2}{4} = \frac{n^2}{4}$。

等号成立当且仅当 $G$ 是完全二部图 $K_{\lfloor n/2 \rfloor, \lceil n/2 \rceil}$。
