## 问题1：中国剩余定理（含历史背景）⭐⭐☆☆☆

### 问题描述

求解同余方程组：
x ≡ 2 (mod 3)
x ≡ 3 (mod 5)
x ≡ 2 (mod 7)

### 历史背景

**中国剩余定理（CRT）**最早记载于中国古代数学著作《孙子算经》（约公元3-5世纪）。

**孙子问题**："今有物不知其数，三三数之剩二，五五数之剩三，七七数之剩二，问物几何？"

这是中国数学对世界数学的重要贡献之一。

### 解答

**步骤1**：确认条件
模数3, 5, 7两两互素，N = 3×5×7 = 105

**步骤2**：计算辅助量
- N₁ = 35, y₁ = 2（因为35×2 ≡ 1 mod 3）
- N₂ = 21, y₂ = 1（因为21 ≡ 1 mod 5）
- N₃ = 15, y₃ = 1（因为15 ≡ 1 mod 7）

**步骤3**：构造解
x = 2×35×2 + 3×21×1 + 2×15×1 = 140 + 63 + 30 = 233

**步骤4**：求最小正整数解
233 = 2×105 + 23

**答案**：x = 23 + 105k，最小正整数解为23

### 现代应用

**密码学**：
- RSA解密加速：利用CRT将大数模幂运算分解
- 秘密共享方案
- 错误校正码

## 问题2：高次同余方程求解 ⭐⭐⭐☆☆

### 问题描述

求解同余方程 x³ ≡ 8 (mod 19) 的所有解。

### 多种解法

#### 解法一：枚举法（适合小模数）

计算x³ mod 19，x = 0, 1, ..., 18：

| x | x³ mod 19 |
|:---:|:---:|
| 2 | 8 ✓ |
| 3 | 8 ✓ |
| 14 | 8 ✓ |

**解**：x ≡ 2, 3, 14 (mod 19)

#### 解法二：原根法（系统性方法）

**步骤1**：找原根
验证2是模19的原根（阶为18）

**步骤2**：取离散对数
设x = 2ᵏ，则2^{3k} ≡ 2³ (mod 19)

**步骤3**：求解指数方程
3k ≡ 3 (mod 18)

由于gcd(3,18) = 3，方程有3个解：
k ≡ 1, 7, 13 (mod 18)

**步骤4**：求出所有解
- k=1: x = 2¹ = 2
- k=7: x = 2⁷ = 14
- k=13: x = 2¹³ = 3

### 理论分析

**定理**：在F_p上，方程xⁿ ≡ a有解当且仅当a^{(p-1)/d} ≡ 1 (mod p)，其中d = gcd(n, p-1)。若有解，则恰有d个解。

本题：d = gcd(3, 18) = 3，检验8⁶ ≡ 1 (mod 19) ✓

## 问题3：费马素性测试与卡迈克尔数 ⭐⭐⭐☆☆

### 问题描述

1. 使用费马测试检验n = 341是否为素数
2. 验证561 = 3×11×17是卡迈克尔数
3. 证明卡迈克尔数必为无平方因子数

### 历史背景

**费马小定理**（1640年）：若p是素数，则对所有与p互素的a，有a^{p-1} ≡ 1 (mod p)。

**Pierre de Fermat**（1607-1665）是法国数学家，被誉为"现代数论之父"。

### 解答

#### 部分1：检验n = 341

计算2^{340} (mod 341)：

使用快速幂：
2^{340} = 2^{256}·2^{64}·2^{16}·2⁴ ≡ 1 (mod 341)

**结论**：341通过费马测试（以2为底），但341 = 11×31是合数！

341是**以2为底的伪素数**。

#### 部分2：验证561是卡迈克尔数

**卡迈克尔数定义**：合数n满足对所有与n互素的a，有a^{n-1} ≡ 1 (mod n)。

**Korselt判别法（1899）**：合数n是卡迈克尔数当且仅当：
1. n无平方因子
2. 若素数p|n，则p-1|n-1

验证561 = 3×11×17：
- 3-1 = 2 | 560 ✓
- 11-1 = 10 | 560 ✓
- 17-1 = 16 | 560 ✓

**结论**：561是卡迈克尔数。

（证明略，使用反证法）

### 现代意义

**密码学教训**：
- 费马测试无法检测卡迈克尔数
- 实际应用中应使用**米勒-拉宾测试**

卡迈克尔数有无穷多个（Alford-Granville-Pomerance, 1994）。

## 问题4：米勒-拉宾素性测试 ⭐⭐⭐⭐☆

### 问题描述

1. 使用米勒-拉宾测试检验n = 561
2. 证明素数必通过米勒-拉宾测试
3. 讨论错误概率上界

### 解答

#### 米勒-拉宾测试原理

写n-1 = 2ˢ·d（d为奇数）

测试序列：aᵈ, a^{2d}, ..., a^{2ˢd} = a^{n-1}

- 若第一项为1，通过
- 若某相邻对为(-1, 1)，通过
- 否则，失败（合数）

#### 检验n = 561

n-1 = 560 = 16×35，s = 4, d = 35

取a = 2：
- x = 2³⁵ ≡ 263 (mod 561)
- x² ≡ 166
- x⁴ ≡ 67
- 未出现-1，测试失败！

**结论**：561被检测为合数。

#### 现代应用

**实际应用**：
- 生成大素数（RSA密钥生成）
- 概率性素性测试的标准方法
- 重复k次，错误概率 < 4^{-k}

## 问题5：原根存在性 ⭐⭐⭐⭐☆

### 问题描述

1. 求模17的所有原根
2. 证明模m存在原根当且仅当m = 2, 4, pᵏ, 2pᵏ（p为奇素数）

### 历史背景

**原根**的概念由高斯在《算术研究》（Disquisitiones Arithmeticae, 1801）中系统研究。

**Carl Friedrich Gauss**（1777-1855）被誉为"数学王子"，他的《算术研究》奠定了现代数论的基础。

### 解答

#### 部分1：求模17的原根

φ(17) = 16 = 2⁴

**判定条件**：g是原根 ⇔ g⁸ ≢ 1 (mod 17)

检验g = 3：
3⁸ ≡ -1 ≢ 1 ✓

所以3是原根。

**所有原根**：3ᵏ，其中gcd(k,16) = 1

k ∈ {1,3,5,7,9,11,13,15}

计算得原根集合：{3, 5, 6, 7, 10, 11, 12, 14}

#### 部分2：原根存在性定理

**(Z/mZ)*是循环群**当且仅当m = 2, 4, pᵏ, 2pᵏ

**证明概要**：
- 利用中国剩余定理分解
- 循环群直积是循环群当且仅当阶互素
- 分析各情形的结构

### 现代应用

**离散对数问题**：
- Diffie-Hellman密钥交换
- ElGamal加密
- DSA数字签名

## 问题6：离散对数问题与密码学 ⭐⭐⭐⭐☆

### 问题描述

1. 以3为底，构造模17的指标表
2. 用指标法解同余方程 7x⁵ ≡ 11 (mod 17)
3. 讨论离散对数问题的计算复杂性

### 解答

#### 部分1：构造指标表

计算3ᵏ mod 17，k = 0, 1, ..., 15：

| k | 3ᵏ |
|:---:|:---:|
| 0 | 1 |
| 1 | 3 |
| 2 | 9 |
| ... | ... |

**指标表**（反向查找）：

| a | ind₃(a) |
|:---:|:---:|
| 1 | 0 |
| 2 | 14 |
| ... | ... |

#### 部分2：用指标法解方程

方程：7x⁵ ≡ 11 (mod 17)

取指标：
ind(7) + 5·ind(x) ≡ ind(11) (mod 16)

11 + 5k ≡ 7 (mod 16)
5k ≡ -4 ≡ 12 (mod 16)

5的逆元模16是13：
k ≡ 12×13 = 156 ≡ 12 (mod 16)

**解**：ind(x) = 12，查表得x = 4

#### 部分3：计算复杂性

| 算法 | 时间复杂度 | 适用场景 |
|:---:|:---:|:---|
| 穷举法 | O(p) | 极小群 |
| 小步大步法（BSGS） | O(√p) | 中等规模 |
| Pohlig-Hellman | O(∑eᵢ log p + max√pᵢ^{eᵢ}) | 光滑阶群 |
| 指数演算 | L_p[1/3, c] | 素域乘法群 |
| 数域筛 | L_p[1/3, 1.923] | 大素域 |

### 现代密码学应用

**安全参数**：
- 模素数p：p至少2048位
- 椭圆曲线：阶至少256位

**实际应用**：
1. **Diffie-Hellman密钥交换**
2. **ElGamal加密**
3. **DSA数字签名**
4. **椭圆曲线密码（ECC）**

## 问题7：模幂运算与快速幂算法 ⭐⭐⭐☆☆

### 问题描述

1. 计算7^{2024} (mod 1000)
2. 分析快速幂算法的时间复杂度
3. 实现蒙哥马利模乘优化

### 解答

#### 部分1：计算大模幂

**欧拉定理**：φ(1000) = φ(8)×φ(125) = 4×100 = 400

由于gcd(7,1000) = 1：
7^{2024} = 7^{5×400+24} ≡ 7^{24} (mod 1000)

**快速幂计算**：
- 7¹ = 7
- 7² = 49
- 7⁴ = 49² = 2401 ≡ 401
- 7⁸ = 401² ≡ 801
- 7^{16} = 801² ≡ 601

7^{24} = 7^{16}·7⁸ = 601×801 ≡ 401 (mod 1000)

**答案**：401

#### 部分2：算法分析

**快速幂**：O(log n)次乘法
**朴素算法**：O(n)次乘法

对于n = 2024，快速幂只需约12次乘法，而朴素算法需要2024次。
