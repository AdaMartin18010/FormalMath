# IMO 国际数学奥林匹克经典问题

> 本问题集收录20道IMO经典问题，涵盖代数、几何、数论、组合四大领域。

## 代数问题 (5题)

### 问题 A1

**题目：** 设 $a, b, c$ 为正实数，且 $abc = 1$。证明：

$$\frac{1}{a^3(b+c)} + \frac{1}{b^3(c+a)} + \frac{1}{c^3(a+b)} \geq \frac{3}{2}$$

**来源：** IMO 1995 第2题

**难度：** ★★★★☆

**提示：** 令 $a = \frac{1}{x}, b = \frac{1}{y}, c = \frac{1}{z}$，利用柯西不等式或排序不等式。

---

### 问题 A2

**题目：** 求所有实系数多项式 $P(x)$，使得对所有实数 $x$，有：

$$P(x^2) = (P(x))^2$$

**来源：** IMO 1962 第4题（变式）

**难度：** ★★★☆☆

**提示：** 考虑多项式的次数和根的性质。

---

### 问题 A3

**题目：** 设 $n \geq 2$ 为正整数，$x_1, x_2, \ldots, x_n$ 为实数，满足：

$$\sum_{i=1}^{n} |x_i| = 1, \quad \sum_{i=1}^{n} x_i = 0$$

证明：

$$\left|\sum_{i=1}^{n} \frac{x_i}{i}\right| \leq \frac{1}{2} - \frac{1}{2n}$$

**来源：** IMO 1989 第2题

**难度：** ★★★★☆

**提示：** 利用分部求和或归纳法。

---

### 问题 A4

**题目：** 设 $a, b, c$ 是三角形的边长。证明：

$$a^2b(a-b) + b^2c(b-c) + c^2a(c-a) \geq 0$$

并确定等号成立的条件。

**来源：** IMO 1983 第6题

**难度：** ★★★★★

**提示：** 设 $a = y+z, b = z+x, c = x+y$，其中 $x,y,z > 0$。

---

### 问题 A5

**题目：** 设 $n \geq 3$ 为正整数，$x_1, x_2, \ldots, x_n$ 为正实数，满足 $x_1 + x_2 + \cdots + x_n = 1$。证明：

$$\sum_{i=1}^{n} \frac{x_i}{\sqrt{1-x_i}} \geq \frac{1}{\sqrt{n-1}} \sum_{i=1}^{n} \sqrt{x_i}$$

**来源：** IMO 1991 第3题

**难度：** ★★★★☆

**提示：** 利用詹森不等式或柯西不等式。

## 几何问题 (5题)

### 问题 G1

**题目：** 设 $\triangle ABC$ 是锐角三角形，$H$ 是垂心。以 $BC$ 为直径的圆分别交 $AB$、$AC$ 于 $E$、$F$。证明：$EF$ 的垂直平分线经过 $AH$ 的中点。

**来源：** IMO 1990 第2题（变式）

**难度：** ★★★★☆

**提示：** 利用九点圆的性质或坐标几何。

---

### 问题 G2

**题目：** 凸四边形 $ABCD$ 的边 $AB$、$BC$、$CD$、$DA$ 上分别有点 $K$、$L$、$M$、$N$，使得：

$$\frac{AK}{KB} = \frac{BL}{LC} = \frac{CM}{MD} = \frac{DN}{NA} = k$$

证明：四边形 $KLMN$ 的面积不小于四边形 $ABCD$ 面积的 $\frac{1}{2}$。

**来源：** IMO 1978 第4题

**难度：** ★★★★☆

**提示：** 利用面积比和向量方法。

---

### 问题 G3

**题目：** 在 $\triangle ABC$ 中，$AB = AC$。在 $BC$ 上取点 $D$，在 $AC$ 上取点 $E$，使得 $AD = AE$。证明：$\angle BAD = 2\angle EDC$。

**来源：** IMO 1992 第4题

**难度：** ★★★☆☆

**提示：** 利用等腰三角形性质和角度计算。

---

### 问题 G4

**题目：** 圆 $\omega$ 内切于四边形 $ABCD$。设 $I$ 为圆心，且满足：

$$(AI + DI)^2 + (BI + CI)^2 = (AB + CD)^2$$

证明：$ABCD$ 是等腰梯形。

**来源：** IMO 2004 第4题

**难度：** ★★★★★

**提示：** 利用内切圆性质和托勒密不等式。

---

### 问题 G5

**题目：** 设 $ABCDEF$ 是凸六边形，满足 $AB = BC = CD$，$DE = EF = FA$，且 $\angle BCD = \angle EFA = 60°$。设 $G$ 和 $H$ 是六边形内部两点，使得 $\angle AGB = \angle DHE = 120°$。证明：

$$AG + GB + GH + DH + HE \geq CF$$

**来源：** IMO 1995 第5题

**难度：** ★★★★★

**提示：** 利用旋转和费马点性质。

### 问题 N3

**题目：** 确定所有大于3的整数 $n$，使得在平面上存在 $n$ 个点 $A_1, A_2, \ldots, A_n$ 和实数 $r_1, r_2, \ldots, r_n$，满足以下条件：

(i) $A_1, A_2, \ldots, A_n$ 中任意三点不共线；
(ii) 对每个三元组 $i, j, k$（$1 \leq i < j < k \leq n$），$\triangle A_iA_jA_k$ 的面积等于 $r_i + r_j + r_k$。

## 组合问题 (5题)

### 问题 C1

**题目：** 在一个 $n \times n$ 棋盘上放置 $n$ 个车，使得每行每列恰有一个车。证明：存在一种放置方式，使得任意两个车不在同一对角线上（即不互相攻击的对角线）。

**来源：** IMO 1979 第2题（变式）

**难度：** ★★★☆☆

**提示：** 利用拉丁方或构造性证明。

### 问题 C2

**题目：** 设 $n$ 为正整数。$S$ 是平面上 $n$ 个点的集合，其中任意三点不共线。设 $k$ 为正整数，$S$ 被染成红色或蓝色。证明：存在 $k$ 个同色点构成凸 $k$ 边形的顶点。

**来源：** IMO 2006 第2题（Erdős–Szekeres定理变式）

### 问题 C3

**来源：** IMO 1988 第2题（变式）

### A1 解答

**解法一（代换法）：** 令 $a = \frac{1}{x}, b = \frac{1}{y}, c = \frac{1}{z}$，则 $xyz = 1$。

原式化为：$$\sum_{cyc} \frac{x^2}{y+z} \geq \frac{3}{2}$$

由柯西不等式：$$\sum_{cyc} \frac{x^2}{y+z} \geq \frac{(x+y+z)^2}{2(x+y+z)} = \frac{x+y+z}{2} \geq \frac{3\sqrt[3]{xyz}}{2} = \frac{3}{2}$$

### A2 解答

设 $\deg P = n$，则 $\deg P(x^2) = 2n$，$\deg (P(x))^2 = 2n$。

若 $P(x) = a_nx^n + \cdots + a_0$，比较最高次项得 $a_n = a_n^2$，故 $a_n = 0$ 或 $a_n = 1$。

若 $a_n = 0$，则 $P(x) = 0$ 或 $P(x) = 1$。

若 $a_n = 1$，则 $P(x) = x^n$，验证得 $n = 0$ 或 $P(x) = x^k$ 对任意 $k$ 成立。

**答案：** $P(x) = 0, 1, x^k$（$k$ 为非负整数）

### A3 解答

设 $S^+ = \{i : x_i > 0\}$，$S^- = \{i : x_i < 0\}$。

令 $P = \sum_{i \in S^+} x_i$，$N = -\sum_{i \in S^-} x_i$，则 $P = N = \frac{1}{2}$。

$$\sum_{i=1}^{n} \frac{x_i}{i} = \sum_{i \in S^+} \frac{x_i}{i} - \sum_{i \in S^-} \frac{|x_i|}{i}$$

利用 $\frac{1}{i} - \frac{1}{i+1} = \frac{1}{i(i+1)}$ 进行分部求和。

### G4 解答

设内切圆半径为 $r$，切点将四边形分成8个直角三角形。

利用切线长相等和勾股定理，证明 $(AI + DI)^2 + (BI + CI)^2 = (AB + CD)^2$ 等价于 $AD \parallel BC$。

### N2 解答

考虑 $n! + 2, n! + 3, \ldots, n! + (n+1)$ 这 $n$ 个连续整数。

对 $2 \leq k \leq n+1$，$n! + k$ 被 $k$ 整除且 $n! + k > k$，故为合数。

进一步构造使得这些数都不是素数幂。
