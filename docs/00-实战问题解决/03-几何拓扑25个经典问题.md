## 类型1：流形构造问题

### 问题1：球面 S^n 的标准光滑结构验证

#### 问题陈述

验证 n 维球面 S^n = {x ∈ ℝ^(n+1) : |x| = 1} 配备标准坐标卡后构成光滑流形。

#### 问题类型识别

| 维度 | 类型 | 核心任务 |
|:---:|:---|:---|
| 对象类型 | 光滑流形 | 验证流形公理 |
| 方法类型 | 坐标卡构造 | 建立图册并验证相容性 |
| 难度等级 | ⭐⭐ | 基础验证类问题 |

#### 方法选择决策

```

┌─────────────────────────────────────────────────────────┐
│  问题：验证 S^n 的光滑结构                                 │
│  ↓                                                       │
│  选择：标准球极投影坐标卡                                  │
│  ↓                                                       │
│  北半球卡：φ_N: S^n\{N} → R^n                            │
│  南半球卡：φ_S: S^n\{S} → R^n                            │
│  ↓                                                       │
│  验证：φ_S ∘ φ_N^(-1) 是光滑微分同胚                      │
└─────────────────────────────────────────────────────────┘

```

#### 完整解决流程

**Step 1：坐标卡定义**

取北极 N = (0, ..., 0, 1) 和南极 S = (0, ..., 0, -1)。

**北半球坐标卡** (U_N, φ_N)：

- U_N = S^n \\ {N}
- 球极投影：φ_N(x_1, ..., x_{n+1}) = (1/(1-x_{n+1}))(x_1, ..., x_n)

**南半球坐标卡** (U_S, φ_S)：

- U_S = S^n \\ {S}
- 球极投影：φ_S(x_1, ..., x_{n+1}) = (1/(1+x_{n+1}))(x_1, ..., x_n)

**Step 2：覆盖性验证**

显然 U_N ∪ U_S = S^n，因为 S^n 上每点至少不等于北极或南极之一。

**Step 3：转移映射计算**

设 (y_1, ..., y_n) = φ_N(x)，则转移映射为：
φ_S ∘ φ_N^(-1)(y) = y/|y|^2

这是 ℝ^n \\ {0} 上的反演变换，显然是光滑的。

**Step 4：Hausdorff与第二可数**

S^n 作为 ℝ^(n+1) 的子空间，继承 Hausdorff 和第二可数性质。

#### 验证与反思

**正确性验证：**

- ✅ 转移映射的光滑性：反演变换 y ↦ y/|y|^2 在 ℝ^n \\ {0} 上无限可微

- ✅ 图册覆盖性：两个开集覆盖整个球面
- ✅ 拓扑性质：子空间拓扑保持

**方法反思：**
> 球极投影是球面几何中的标准工具。关键观察：球极投影将"无穷远"映射到对径点，转移映射的反演特性反映了这一几何直观。

#### 相关变体

| 变体 | 问题 | 方法调整 |
|:---:|:---|:---|
| 实射影空间 | ℝP^n 的光滑结构 | 使用齐次坐标卡 |
| 复射影空间 | ℂP^n 的复结构 | 全纯坐标卡 |
| 一般超曲面 | f^(-1)(0) 的正则值原像 | 隐函数定理 |

### 问题2：积流形 S^1 × S^1（环面）的切丛构造

#### 问题陈述

构造环面 T^2 = S^1 × S^1 的切丛 TT^2，证明其平凡性。

#### 问题类型识别

| 维度 | 类型 | 核心任务 |
|:---:|:---|:---|
| 对象类型 | 积流形 | 切丛构造与性质 |
| 方法类型 | 直积构造 | 利用积流形结构 |
| 难度等级 | ⭐⭐⭐ | 中等构造类问题 |

#### 方法选择决策

```

┌─────────────────────────────────────────────────────────┐
│  问题：T^2 的切丛平凡性                                    │
│  ↓                                                       │
│  策略：利用 T(T^2) ≅ T(S^1) × T(S^1)                     │
│  ↓                                                       │
│  关键：S^1 的切丛 TS^1 ≅ S^1 × R 是平凡的                  │
│  ↓                                                       │
│  结论：T^2 是平行化流形                                    │
└─────────────────────────────────────────────────────────┘

```

#### 完整解决流程

**Step 1：环面的参数化**

环面可参数化为：
Φ(θ, φ) = ((R + r·cos θ)·cos φ, (R + r·cos θ)·sin φ, r·sin θ)

其中 R > r > 0，θ, φ ∈ [0, 2π)。

**Step 2：切空间计算**

在点 p = Φ(θ_0, φ_0) 处，切空间由偏导数张成：
∂Φ/∂θ = (-r·sin θ_0·cos φ_0, -r·sin θ_0·sin φ_0, r·cos θ_0)
∂Φ/∂φ = (-(R+r·cos θ_0)·sin φ_0, (R+r·cos θ_0)·cos φ_0, 0)

**Step 3：切丛构造**

利用积流形性质：
T(T^2) = T(S^1 × S^1) ≅ TS^1 × TS^1 ≅ (S^1 × ℝ) × (S^1 × ℝ) ≅ T^2 × ℝ^2

**Step 4：平凡性证明**

构造全局标架场：

- X_1 = ∂/∂θ（经线方向）
- X_2 = ∂/∂φ（纬线方向）

这两个向量场处处线性无关，构成切丛的全局平凡化。

#### 验证与反思

**正确性验证：**

- ✅ 切空间维度：dim T_pT^2 = 2 = dim T^2 ✓
- ✅ 向量场光滑性：参数化 Φ 是光滑的，偏导数也是光滑的
- ✅ 线性无关性：∂Φ/∂θ × ∂Φ/∂φ ≠ 0

**方法反思：**
> 环面的切丛平凡性反映了其"平坦"的拓扑性质。这一结论可推广：n 维环面 T^n 都是平行化的。但注意：S^2 的切丛不平凡（hairy ball theorem 的深层原因）。

#### 相关变体

| 变体 | 问题 | 关键区别 |
|:---:|:---|:---|
| 高维环面 | T^n 的切丛 | 类似的平凡性 |
| 球面 S^2 | TS^2 是否平凡 | 不平凡，需要示性类证明 |
| 李群 | 任何李群的切丛 | 总是平凡的 |

### 问题3：莫比乌斯带的光滑向量场构造

#### 问题陈述

构造莫比乌斯带上的光滑向量场，使其恰有一个零点，并分析该向量场的拓扑指数。

#### 问题类型识别

| 维度 | 类型 | 核心任务 |
|:---:|:---|:---|
| 对象类型 | 非定向流形 | 向量场构造 |
| 方法类型 | 局部拼接 | 利用商结构 |
| 难度等级 | ⭐⭐⭐⭐ | 高级构造类问题 |

#### 方法选择决策

```

┌─────────────────────────────────────────────────────────┐
│  问题：莫比乌斯带上的向量场                                │
│  ↓                                                       │
│  方法：将莫比乌斯带表示为 [0,1]×R 的商空间                  │
│  ↓                                                       │
│  商映射：(t,x) ~ (t+1, -x)                               │
│  ↓                                                       │
│  构造：在商空间上定义相容的向量场                          │
└─────────────────────────────────────────────────────────┘

```

#### 完整解决流程

**Step 1：莫比乌斯带的商空间模型**

莫比乌斯带 M 可表示为：
M = ([0, 1] × ℝ) / ~
其中 (0, x) ~ (1, -x)。

**Step 2：局部坐标卡**

- 卡1：U_1 = (0, 1) × ℝ，坐标 (t, x)
- 卡2：U_2 = ([0, 1/2) ∪ (1/2, 1]) × ℝ / ~

**Step 3：向量场构造**

在商空间上定义向量场：
V(t, x) = sin(2πt)·∂/∂t + x·cos(2πt)·∂/∂x

验证相容性：在重叠区域，利用 t ↦ t+1 和 x ↦ -x 变换，向量场保持不变。

**Step 4：零点分析**

零点满足：sin(2πt) = 0 且 x·cos(2πt) = 0。

解得：t = 0, 1/2（模1）且 x = 0。

但 t=0 和 t=1 在商空间中是同一点（经过 x ↦ -x 等同），所以只有一个零点在 [(0, 0)] = [(1, 0)]。

**Step 5：拓扑指数计算**

在零点处，线性化矩阵为：
DV = [[2π, 0], [0, 1]]

特征值为正，指数 ind(V, 0) = +1。

#### 验证与反思

**正确性验证：**

- ✅ 商空间模型正确：莫比乌斯带的拓扑由对径翻转识别给出
- ✅ 向量场相容性：验证变换下的不变性
- ✅ 零点唯一性：通过等同关系减少零点数量

**方法反思：**
> 商空间上的向量场构造需要仔细验证相容性条件。莫比乌斯带的非定向性使其上的向量场构造比环面复杂得多。

#### 相关变体

| 变体 | 问题 | 挑战 |
|:---:|:---|:---|
| 克莱因瓶 | 无零点向量场存在性 | 非定向闭曲面 |
| 射影平面 | 向量场零点下界 | 欧拉示性数为1 |
| 任意曲面 | 向量场与示性类关系 | Poincaré-Hopf定理 |

### 问题4：Grassmann流形 G(k, n) 的切空间刻画

#### 问题陈述

刻画Grassmann流形 G(k, n)（ℝ^n 中 k 维子空间的集合）在某点 Π ∈ G(k, n) 处的切空间 T_Π G(k, n)。

#### 问题类型识别

| 维度 | 类型 | 核心任务 |
|:---:|:---|:---|
| 对象类型 | 齐性空间 | 切空间的代数刻画 |
| 方法类型 | 群作用 | 利用李群结构 |
| 难度等级 | ⭐⭐⭐⭐⭐ | 高级理论问题 |

#### 方法选择决策

```

┌─────────────────────────────────────────────────────────┐
│  问题：G(k,n) 的切空间                                    │
│  ↓                                                       │
│  方法1：Plücker嵌入                                      │
│  方法2：商空间 O(n)/(O(k)×O(n-k))                        │
│  ↓                                                       │
│  选择方法2，利用齐性空间结构                              │
│  ↓                                                       │
│  结论：T_Π G(k,n) ≅ Hom(Π, Π^⊥)                          │
└─────────────────────────────────────────────────────────┘

```

#### 完整解决流程

**Step 1：Grassmann流形作为齐性空间**

G(k, n) = O(n) / (O(k) × O(n-k))

正交群 O(n) 传递地作用在 k 维子空间集合上。

**Step 2：切空间的群论刻画**

对于齐性空间 G/H，有：
T_[e](G/H) ≅ 𝔤/𝔥

其中 𝔤 = 𝔰𝔬(n)，𝔥 = 𝔰𝔬(k) ⊕ 𝔰𝔬(n-k)。

**Step 3：具体计算**

𝔰𝔬(n) 由反对称矩阵组成。分解为分块形式：
X = [[A, -B^T], [B, D]]

其中 A ∈ 𝔰𝔬(k)，D ∈ 𝔰𝔬(n-k)，B ∈ M_{(n-k)×k}(ℝ)。

商空间 𝔰𝔬(n)/𝔰𝔬(k)⊕𝔰𝔬(n-k) 由 B 参数化。

**Step 4：几何解释**

矩阵 B 对应线性映射 Π → Π^⊥：对 v ∈ Π，选择曲线 γ(t) 使得 γ(0) = v 且方向由 B 决定。

因此：
T_Π G(k, n) ≅ Hom(Π, Π^⊥)

维数：dim G(k, n) = k(n-k) = dim Hom(Π, Π^⊥) ✓

#### 验证与反思

**正确性验证：**

- ✅ 维数匹配：k(n-k) = k(n-k) ✓
- ✅ 几何直观：切方向对应子空间的"旋转"
- ✅ 特例验证：G(1, n) = ℝP^(n-1)，T_[v]ℝP^(n-1) ≅ v^⊥ ≅ Hom(span(v), v^⊥) ✓

**方法反思：**
> Grassmann流形是代数几何和微分几何的交汇点。齐性空间方法虽然抽象，但给出了切空间最清晰的刻画。Plücker嵌入提供了另一种外代数视角。

#### 相关变体

| 变体 | 问题 | 方法 |
|:---:|:---|:---|
| 复Grassmann流形 | G_ℂ(k, n) | 类似，使用 U(n) |
| 辛Grassmann流形 | Lagrangian Grassmannian | 辛群作用 |
| 旗流形 | 嵌套子空间族 | 推广上述方法 |

### 问题5：流形间的光滑映射：隐函数定理应用

#### 问题陈述

设 f: ℝ^3 → ℝ，f(x, y, z) = x^2 + y^2 - z^2。

- (a) 证明 M = f^(-1)(1) 是光滑流形
- (b) 求 M 在某点的切空间
- (c) 参数化 M 并计算诱导度量

#### 问题类型识别

| 维度 | 类型 | 核心任务 |
|:---:|:---|:---|
| 对象类型 | 正则值原像 | 隐函数定理应用 |
| 方法类型 | 微分判定 | 计算雅可比矩阵 |
| 难度等级 | ⭐⭐⭐ | 标准应用类问题 |

#### 方法选择决策

```

┌─────────────────────────────────────────────────────────┐
│  问题：双叶双曲面的光滑结构                                │
│  ↓                                                       │
│  方法：正则值定理 (Sard定理的推论)                         │
│  ↓                                                       │
│  验证：df ≠ 0 在 f^(-1)(1) 上                            │
│  ↓                                                       │
│  参数化：双曲坐标                                         │
└─────────────────────────────────────────────────────────┘

```

#### 完整解决流程

**Step 1：正则值验证**

计算微分 df = (2x, 2y, -2z)。

若 df = 0，则 x = y = z = 0，此时 f(0, 0, 0) = 0 ≠ 1。

因此 1 是 f 的正则值，M = f^(-1)(1) 是光滑 2 维流形。

**Step 2：切空间计算**

在点 p = (x_0, y_0, z_0) ∈ M 处：
T_pM = ker df_p = {v ∈ ℝ^3 : 2x_0·v_1 + 2y_0·v_2 - 2z_0·v_3 = 0}

即 T_pM 是梯度 ∇f(p) = (2x_0, 2y_0, -2z_0) 的正交补空间。

**Step 3：参数化**

对于 z > 0 的叶，使用双曲坐标：
x = cosh u · cos v, y = cosh u · sin v, z = sinh u

其中 u ∈ ℝ，v ∈ [0, 2π)。

验证：x^2 + y^2 - z^2 = cosh^2 u - sinh^2 u = 1 ✓

**Step 4：诱导度量**

计算切向量：
∂/∂u = (sinh u · cos v, sinh u · sin v, cosh u)
∂/∂v = (-cosh u · sin v, cosh u · cos v, 0)

第一基本形式：
E = ||∂/∂u||^2 = sinh^2 u + cosh^2 u = cosh(2u)

F = ∂/∂u · ∂/∂v = 0
G = ||∂/∂v||^2 = cosh^2 u

度量张量：
g = cosh(2u) du^2 + cosh^2 u dv^2

#### 验证与反思

**正确性验证：**

- ✅ 参数化有效性：雅可比矩阵秩为2
- ✅ 度量正定性：E > 0 且 EG - F^2 = cosh(2u)·cosh^2 u > 0
- ✅ 几何直观：M 是连通的双叶曲面，每叶微分同胚于 ℝ^2

**方法反思：**
> 正则值定理是构造流形的强大工具。关键在于验证微分的秩条件。双曲坐标的选择源于问题的对称性——这是处理二次曲面的一般策略。

#### 相关变体

| 变体 | 方程 | 结果 |
|:---:|:---|:---|
| 单叶双曲面 | x^2 + y^2 - z^2 = -1 | 连通，拓扑 S^1 × ℝ |
| 椭球面 | x^2 + y^2 + z^2 = 1 | 紧致流形 S^2 |
| 一般二次曲面 | x^T A x = 1 | 依赖 A 的符号 |

### 问题6：带边流形的边界坐标卡

#### 问题陈述

设 M = {(x, y, z) ∈ ℝ^3 : x^2 + y^2 + z^2 ≤ 1, z ≥ 0}（上半球体）。

- (a) 证明 M 是带边 3 维流形
- (b) 构造边界和内点的坐标卡
- (c) 计算边界的诱导定向

#### 问题类型识别

| 维度 | 类型 | 核心任务 |
|:---:|:---|:---|
| 对象类型 | 带边流形 | 边界处理 |
| 方法类型 | 半空间模型 | 标准坐标卡 |
| 难度等级 | ⭐⭐⭐ | 基础构造类问题 |

#### 方法选择决策

```

┌─────────────────────────────────────────────────────────┐
│  问题：带边流形的坐标卡                                    │
│  ↓                                                       │
│  模型：上半空间 H^n = {(x_1,...,x_n) : x_n ≥ 0}          │
│  ↓                                                       │
│  边界 ∂M = M ∩ {z = 0}  (闭圆盘)                         │
│  内部 int(M) = {x^2+y^2+z^2 < 1, z > 0}                  │
│  ↓                                                       │
│  坐标卡：球坐标限制 + 边界平坦化                          │
└─────────────────────────────────────────────────────────┘

```

#### 完整解决流程

**Step 1：内部坐标卡**

对于内部点 p = (x, y, z) 满足 x^2 + y^2 + z^2 < 1 且 z > 0，使用球坐标：
φ(x, y, z) = (ρ, θ, φ_球)

其中 ρ ∈ [0, 1)，θ ∈ [0, 2π)，φ_球 ∈ (0, π/2]。

这是到 ℝ^3 开集的光滑同胚。

**Step 2：边界坐标卡**

对于边界点 p = (x, y, 0) 满足 x^2 + y^2 < 1，需要映射到上半空间 H^3 = {(u, v, w) : w ≥ 0}。

定义：ψ(x, y, z) = (x, y, z)

这是恒等映射，将边界 {z = 0} 映射到 {w = 0}，内部 {z > 0} 映射到 {w > 0}。

**Step 3：边界流形结构**

∂M = {(x, y, 0) : x^2 + y^2 ≤ 1} 是闭圆盘，是带边 2 维流形。

**Step 4：诱导定向**

M 从 ℝ^3 继承标准定向，体积形式为 dx ∧ dy ∧ dz。

在边界上，外法向为 ν = (0, 0, -1)（指向下方，离开 M）。

诱导定向由 ι_ν(dx ∧ dy ∧ dz) = -dx ∧ dy 给出。

若以 (x, y) 为边界坐标，定向由 -dx ∧ dy 或等价地 dy ∧ dx 给出。

#### 验证与反思

**正确性验证：**

- ✅ 内部坐标卡：标准球坐标，光滑性显然
- ✅ 边界坐标卡：恒等映射的限制，满足带边流形定义
- ✅ 定向相容性：斯托克斯定理要求边界定向与内部定向相容

**方法反思：**

### 问题8：曲面的高斯曲率：悬链面

#### 问题陈述

悬链面是悬链线绕轴旋转得到的曲面，参数化为：
X(u, v) = (cosh u · cos v, cosh u · sin v, u)

计算其第一、第二基本形式及高斯曲率 K。

#### 问题类型识别

| 维度 | 类型 | 核心任务 |
|:---:|:---|:---|
| 对象类型 | 旋转曲面 | 高斯曲率计算 |
| 方法类型 | 直接计算 | 第一、第二基本形式 |
| 难度等级 | ⭐⭐⭐ | 标准计算类问题 |

#### 方法选择决策

```

┌─────────────────────────────────────────────────────────┐
│  问题：悬链面的高斯曲率                                    │
│  ↓                                                       │
│  方法：第一、第二基本形式 → 形状算子 → 高斯曲率              │
│  ↓                                                       │
│  K = det(II) / det(I) = (LN - M^2) / (EG - F^2)          │
│  ↓                                                       │
│  期望：K ≤ 0（极小曲面的特征）                             │
└─────────────────────────────────────────────────────────┘

```

#### 完整解决流程

**Step 1：切向量计算**

X_u = (sinh u · cos v, sinh u · sin v, 1)
X_v = (-cosh u · sin v, cosh u · cos v, 0)

**Step 2：第一基本形式**

E = X_u · X_u = sinh^2 u + 1 = cosh^2 u
F = X_u · X_v = 0
G = X_v · X_v = cosh^2 u

所以：
I = cosh^2 u · du^2 + cosh^2 u · dv^2

**Step 3：单位法向量**

X_u × X_v = (-cosh u · cos v, -cosh u · sin v, sinh u · cosh u)

|X_u × X_v| = cosh u · √(cosh^2 u + sinh^2 u) = cosh u · √cosh(2u)

单位法向量：
N = (-cos v, -sin v, sinh u) / √cosh(2u)

**Step 4：第二基本形式**

计算二阶导数：
X_uu = (cosh u · cos v, cosh u · sin v, 0)
X_uv = (-sinh u · sin v, sinh u · cos v, 0)
X_vv = (-cosh u · cos v, -cosh u · sin v, 0)

第二基本形式系数：
L = X_uu · N = -cosh u / √cosh(2u)
M = X_uv · N = 0
N = X_vv · N = cosh u / √cosh(2u)

**Step 5：高斯曲率**

K = (LN - M^2) / (EG - F^2) = (-cosh^2 u / cosh(2u)) / cosh^4 u = -1 / (cosh^2 u · cosh(2u))

利用 cosh(2u) = 2·cosh^2 u - 1：
K = -1 / (cosh^2 u · (2·cosh^2 u - 1)) < 0

#### 验证与反思

**正确性验证：**

- ✅ 符号：K < 0 处处成立，符合双曲型曲面特征
- ✅ 极小曲面验证：平均曲率 H = (1/2)·(EN + GL - 2FM)/(EG-F^2) = (1/2)·(cosh^3 u - cosh^3 u)/(...) = 0 ✓
- ✅ 渐近行为：u → ∞ 时 K → 0

**方法反思：**
> 悬链面是唯一的旋转极小曲面。高斯曲率的负号反映了曲面的鞍形特征——沿经线方向弯曲向下，沿纬线方向弯曲向上。

| 球面 | S^2 | K = 1/R^2 > 0 |
| 圆柱 | S^1 × ℝ | K = 0 |

### 问题10：测地线方程：双曲平面的测地线

#### 问题陈述

上半平面模型 ℍ^2 = {(x, y) : y > 0} 配备庞加莱度量：
ds^2 = (dx^2 + dy^2) / y^2

- (a) 计算测地线方程
- (b) 证明测地线是垂直于 x 轴的半圆或直线

#### 问题类型识别

| 维度 | 类型 | 核心任务 |
|:---:|:---|:---|
| 对象类型 | 双曲空间 | 测地线计算 |
| 方法类型 | Euler-Lagrange | 变分法 |
| 难度等级 | ⭐⭐⭐⭐ | 高级计算类问题 |

#### 方法选择决策

```

┌─────────────────────────────────────────────────────────┐
│  问题：双曲平面的测地线                                    │
│  ↓                                                       │
│  方法1：Christoffel符号 + 测地线方程                       │
│  方法2：变分法（Euler-Lagrange）                           │
│  方法3：等距不变性（利用对称性）                            │
│  ↓                                                       │
│  选择方法2（计算更直接）                                   │
└─────────────────────────────────────────────────────────┘

```

#### 完整解决流程

**Step 1：拉格朗日量**

能量泛函的被积函数：
L = (ẋ^2 + ẏ^2) / y^2

其中 ẋ = dx/dt，ẏ = dy/dt。

**Step 2：Euler-Lagrange 方程**

对于 x：
d/dt(∂L/∂ẋ) = d/dt(2ẋ/y^2) = 2ẍ/y^2 - 4ẋẏ/y^3

∂L/∂x = 0

方程：ẍ - 2ẋẏ/y = 0

对于 y：
d/dt(∂L/∂ẏ) = 2ÿ/y^2 - 4ẏ^2/y^3

∂L/∂y = -2(ẋ^2 + ẏ^2)/y^3

方程：ÿ + (ẋ^2 - ẏ^2)/y = 0

**Step 3：首次积分**

拉格朗日量不显含 x，故守恒量：
p_x = ∂L/∂ẋ = 2ẋ/y^2 = c_1

若 c_1 = 0，则 ẋ = 0，即垂直线 x = const。

若 c_1 ≠ 0：
ẋ = c_1·y^2/2

**Step 4：求解轨迹**

利用能量守恒（以弧长参数化）：
(ẋ^2 + ẏ^2)/y^2 = 1

代入 ẋ = c_1·y^2/2：
(c_1^2·y^4/4 + ẏ^2)/y^2 = 1

ẏ^2 = y^2 - c_1^2·y^4/4 = y^2(1 - c_1^2·y^2/4)

分离变量并求解，消去参数后得：
(x - x_0)^2 + y^2 = 4/c_1^2

这是中心在 (x_0, 0)、半径为 2/|c_1| 的圆，在上半平面的部分。

#### 验证与反思

**正确性验证：**

- ✅ 垂直直线：令 c_1 → ∞ 的极限情形
- ✅ 垂直于边界：圆心都在 y=0 上，故与边界垂直
- ✅ 等距不变性：分式线性变换保持此类曲线

**方法反思：**
> 双曲几何的核心特征是"直线"（测地线）与欧氏几何不同。垂直于边界的圆弧是双曲平面的"直线"，这一性质使双曲几何具有丰富的三角形性质（内角和小于 π）。

### 问题11：高斯-博内定理应用：曲面的拓扑限制

- (a) 陈述高斯-博内定理
- (b) 若 K > 0 处处成立，证明 S ≅ S^2

```
│  ↓                                                       │

└─────────────────────────────────────────────────────────┘
```

#### 完整解决流程

**Step 1：高斯-博内定理**

对于无边紧致定向黎曼曲面 (S, g)：
∫_S K dA = 2π χ(S)

其中 χ(S) = 2 - 2g 是欧拉示性数，g 是亏格。

**Step 2：正曲率情形**

∫_S K dA > 0

因此 χ(S) > 0。

亏格为0的定向闭曲面微分同胚于 S^2。

**Step 3：平坦情形分析**

假设 K = 0 处处成立，则：
∫_S K dA = 0 = 2π χ(S)

因此 χ(S) = 0，即 g = 1。

**Step 4：平坦环面的存在性**

需要证明：任何亏格1的曲面都可配备平坦度量。

构造：S = ℝ^2 / Λ，其中 Λ 是格点群。商度量是平坦的。

反之，任何配备平坦度量的紧致曲面都是 T^2。

#### 验证与反思

**正确性验证：**

- ✅ 球面：χ(S^2) = 2，可配备 K = 1 的度量 ✓
- ✅ 环面：χ(T^2) = 0，标准平坦度量 K = 0 ✓
- ✅ 高亏格：χ < 0，必须 K 变号或恒负

**方法反思：**
> 高斯-博内定理是微分几何中最深刻的结果之一，它建立了局部几何量（曲率）与整体拓扑量（欧拉示性数）的联系。这是Chern-Weil理论的原型。

### 问题12：形状算子与主曲率：椭球面

```
│  问题：椭球面的形状算子                                    │
```

#### 完整解决流程

**Step 1：隐式表示**

F(x, y, z) = x^2/a^2 + y^2/b^2 + z^2/c^2 - 1 = 0

梯度（外法向）：
∇F = (2x/a^2, 2y/b^2, 2z/c^2)

**Step 2：参数化与第一基本形式**

在 (x, 0, z) 附近参数化（y = 0 平面）：
y = ±b·√(1 - x^2/a^2 - z^2/c^2)

取 y > 0 分支，切平面由基向量张成。

**Step 3：形状算子**

形状算子 S = -dN 满足 ⟨S(X), Y⟩ = II(X, Y)。

对于隐式曲面，形状算子在切向量 v 上的作用：
S(v) = (1/|∇F|)·(I - (∇F ⊗ ∇F)/|∇F|^2)·D^2F(v)

**Step 4：主曲率计算**

在点 (x, 0, z)，由对称性，x 和 z 方向是主方向。

计算得主曲率与 a, b, c 以及点的位置有关。

**Step 5：脐点分析**

脐点满足 κ_1 = κ_2。

这需要复杂的代数分析。结果表明：椭球面有4个脐点，位于 y = 0 平面上的曲率线交点处。

具体位置满足：
x^2/(a^2(a^2-b^2)) = z^2/(c^2(b^2-c^2))

#### 验证与反思

**正确性验证：**

- ✅ 球面特例：a=b=c=R 时，所有点都是脐点，κ_1 = κ_2 = 1/R ✓
- ✅ 对称性：脐点在 x-z 平面上成对出现
- ✅ 数量：4个脐点符合Jacobi关于3轴椭球面的经典结果

**方法反思：**
> 椭球面是研究曲面理论的试验场。脐点的分布反映了曲面的内在对称性。4个脐点是最小数量，这与Poincaré-Hopf定理关于向量场零点的结论相关。

### 问题13：指数映射与法坐标：局部几何

#### 问题陈述

- (a) 定义指数映射 exp_p: T_pM → M
- (b) 证明 exp_p 在 0 ∈ T_pM 处是局部微分同胚

#### 问题类型识别

| 维度 | 类型 | 核心任务 |

| 对象类型 | 黎曼流形 | 指数映射理论 |

```

┌─────────────────────────────────────────────────────────┐
│  问题：指数映射与法坐标                                    │
│  ↓                                                       │
│  定义：exp_p(v) = γ_v(1)，其中 γ_v 是从 p 出发初速为 v 的测地线│
│  ↓                                                       │
│  关键性质：d(exp_p)_0 = id（由测地线方程）                 │

└─────────────────────────────────────────────────────────┘

```

#### 完整解决流程

对于 v ∈ T_pM，设 γ_v 是满足：

- γ_v(0) = p
- γ_v'(0) = v

的唯一测地线。

定义：exp_p(v) = γ_v(1)（当 γ_v 定义到 t=1 时）。

**Step 2：局部微分同胚性**

由测地线方程的性质：
γ_{tv}(s) = γ_v(ts)

因此：exp_p(tv) = γ_v(t)

d/dt|_{t=0} exp_p(tv) = γ_v'(0) = v

这说明 d(exp_p)_0(v) = v，即 d(exp_p)_0 = id_{T_pM}。

**Step 3：法坐标**

(x^1, ..., x^n) ↦ exp_p(x^i·e_i)

- g_ij(0) = δ_ij
- ∂_k g_ij(0) = 0（法坐标的性质）

二阶项与曲率相关：
g_ij(x) = δ_ij + (1/3)·R_ikjl(p)·x^k·x^l + O(|x|^3)

**Step 5：体积元展开**

√det g = 1 - (1/6)·R_ij(p)·x^i·x^j + O(|x|^3)

其中 R_ij 是Ricci曲率。

**正确性验证：**

- ✅ 维数1：M = ℝ，exp_p(v) = p + v ✓

- ✅ 曲率联系：展开式中的曲率项反映了局部几何

> 法坐标是黎曼几何的"标准模型"。在法坐标下，流形在一点附近看起来最像欧氏空间。曲率出现在度量的二阶展开中，这是曲率作为"二阶几何量"的精确含义。

- U = S^n \\ {N}（北半球去掉极点）

0 → H_k(S^n) → H_{k-1}(S^{n-1}) → 0

因此 H_k(S^n) ≅ H_{k-1}(S^{n-1})。

**Step 4：初始情形**

S^0 = {-1, 1}（两点）：
H_k(S^0) = ℤ ⊕ ℤ (k = 0), 0 (k > 0)

**Step 5：一般公式**

由归纳：
H_k(S^n) = ℤ (k = 0, n), 0（其他）

验证：S^n 连通，故 H_0 = ℤ；只有 n 维"洞"，故 H_n = ℤ。

#### 验证与反思

**正确性验证：**

- ✅ 维数1：H_1(S^1) = ℤ，与基本群 H_1 = π_1^{ab} 一致 ✓
- ✅ 维数2：H_2(S^2) = ℤ，反映2维"球面包围的洞" ✓

**方法反思：**
> Mayer-Vietoris序列是计算同调群的有力工具。关键技巧是将复杂空间分解为简单部分的并。球面的分解是最经典的应用。

```
┌─────────────────────────────────────────────────────────┐
```

- H_0 = ℤ

```
└─────────────────────────────────────────────────────────┘

```

#### 完整解决流程

将 T^2 视为正方形 [0,1] × [0,1] 对边等同。

取：

- U = 正方形去掉中间水平带
- V = 正方形去掉中间垂直带

**Step 2：同伦类型分析**

- U 同伦等价于 S^1 ∨ S^1（两个圆的楔和）
- V 可缩
- U ∩ V 同伦等价于 S^1（圆盘的边界）

... → H_k(U) ⊕ H_k(V) → H_k(T^2) → H_{k-1}(U ∩ V) → ...

由于 U ≅ S^1 ∨ S^1，H_k(U) = 0 对 k ≥ 2。
V 可缩，H_k(V) = 0 对 k ≥ 1。

因此 H_k(T^2) = 0 对 k ≥ 3。

**Step 4：低维计算**

对于 k = 2：
H_2(U) ⊕ H_2(V) → H_2(T^2) → H_1(U ∩ V) → H_1(U) ⊕ H_1(V)

即：
0 → H_2(T^2) → ℤ →[α] ℤ^2 ⊕ 0

映射 α 是包含诱导的 H_1(S^1) → H_1(S^1 ∨ S^1)，它将生成元映为0。

因此 α = 0，H_2(T^2) ≅ ℤ。

**Step 5：一般公式**

H_k(T^2) = ℤ (k = 0), ℤ^2 (k = 1), ℤ (k = 2), 0 (k > 2)

#### 验证与反思

**正确性验证：**

- ✅ Euler示性数：χ = 1 - 2 + 1 = 0 = 2 - 2g（g=1）✓
- ✅ 与基本群一致：H_1 = π_1^{ab} = ℤ^2 ✓
- ✅ Poincaré对偶：H_1 ≅ H^1 ≅ H_1（自对偶）✓

**方法反思：**
> 环面是最简单的非球面闭曲面。其同调群反映了两个独立的"洞"——一个经向，一个纬向。Mayer-Vietoris序列的要点在于选择合适的分解，使各部分同调已知。

### 问题18：相对同调与切除定理

#### 问题陈述

设 A ⊂ X 是子空间，U ⊂ A 满足 Ū ⊂ int(A)。

- (a) 陈述切除定理
- (b) 计算 H_*(D^n, S^{n-1})
- (c) 证明 H_*(X, A) ≅ H̃_*(X/A)（当 A 是闭子空间且邻域形变收缩核时）

#### 问题类型识别

| 维度 | 类型 | 核心任务 |
|:---:|:---|:---|
| 对象类型 | 空间对 | 相对同调计算 |
| 方法类型 | 切除定理 | 商空间同调 |
| 难度等级 | ⭐⭐⭐⭐ | 高级理论问题 |

#### 方法选择决策

```

┌─────────────────────────────────────────────────────────┐
│  问题：相对同调与切除                                       │
│  ↓                                                       │
│  切除定理：H_n(X, A) ≅ H_n(X\U, A\U)                    │
│  ↓                                                       │
│  应用：H_n(D^n, S^(n-1)) ≅ H_n(S^n)                     │
│  通过将 D^n 收缩到一点，保持边界相对                       │
└─────────────────────────────────────────────────────────┘

```

#### 完整解决流程

**Step 1：切除定理**

**定理**：若 U ⊂ A ⊂ X 满足 Ū ⊂ int(A)，则包含映射诱导同构：
H_n(X \\ U, A \\ U) → H_n(X, A)

**Step 2：(D^n, S^{n-1}) 的同调**

长正合列：
... → H_k(S^{n-1}) → H_k(D^n) → H_k(D^n, S^{n-1}) → H_{k-1}(S^{n-1}) → ...

由于 D^n 可缩，H_k(D^n) = 0 对 k > 0。

因此：
H_k(D^n, S^{n-1}) ≅ H_{k-1}(S^{n-1})

计算得：
H_k(D^n, S^{n-1}) = ℤ (k = n), 0（其他）

**Step 3：与球面同调的联系**

商空间 D^n / S^{n-1} ≅ S^n。

断言：H_k(D^n, S^{n-1}) ≅ H̃_k(S^n)。

**证明**：考虑映射 (D^n, S^{n-1}) → (S^n, *)，将 S^{n-1} 捏合到一点。由切除和形变收缩，诱导同调同构。

**Step 4：一般商空间结果**

若 A ⊂ X 是闭子空间且是邻域形变收缩核，则：
H_*(X, A) ≅ H̃_*(X/A)

这通过将 A 的邻域"捏合"到一点来证明。

#### 验证与反思

**正确性验证：**

- ✅ n=1：(D^1, S^0)，H_1 = ℤ，D^1/S^0 ≅ S^1 ✓
- ✅ 维数匹配：dim H_n(D^n, S^{n-1}) = 1 = dim H̃_n(S^n) ✓
- ✅ 长正合列：与对 (X, A) 的相对同调序列相容

**方法反思：**
> 切除定理是建立相对同调与商空间同调联系的关键。它表明"切除"一个内部子空间不改变相对同调。这是计算商空间同调的有力工具。

| 变体 | 情形 | 应用 |
|:---:|:---|:---|

### 问题19：Künneth公式：积空间的同调

#### 问题陈述

叙述并应用Künneth公式计算 H_*(S^m × S^n)。

#### 问题类型识别

| 维度 | 类型 | 核心任务 |
|:---:|:---|:---|
| 对象类型 | 积空间 | 同调群计算 |
| 方法类型 | Künneth公式 | 代数分解 |
| 难度等级 | ⭐⭐⭐⭐ | 高级理论问题 |

#### 方法选择决策

```

┌─────────────────────────────────────────────────────────┐
│  问题：积空间的同调                                        │
│  ↓                                                       │
│  Künneth公式：H_*(X × Y) 由 H_*(X) 和 H_*(Y) 决定         │
│  ↓                                                       │
│  简化版（域系数）：H_n(X × Y) ≅ ⊕_{i+j=n} H_i(X) ⊗ H_j(Y)│
│  ↓                                                       │
│  对 S^m × S^n：十字交叉项产生中间维数同调                   │
└─────────────────────────────────────────────────────────┘

```

#### 完整解决流程

**Step 1：Künneth公式**

对于空间 X, Y，有分裂短正合列：
0 → ⊕_{i+j=n} H_i(X) ⊗ H_j(Y) → H_n(X × Y) → ⊕_{i+j=n-1} Tor(H_i(X), H_j(Y)) → 0

对于域系数，Tor项消失，公式简化为：
H_n(X × Y) ≅ ⊕_{i+j=n} H_i(X) ⊗ H_j(Y)

**Step 2：应用于 S^m × S^n**

H_*(S^m) 和 H_*(S^n) 只在0维和各自最高维有 ℤ。

计算 H_k(S^m × S^n)：

**情形1**：k = 0

- H_0(S^m) ⊗ H_0(S^n) = ℤ ⊗ ℤ = ℤ

**情形2**：0 < k < min(m, n)

- 无贡献，因为需要 i > 0 和 j > 0 同时成立
- H_k = 0

**情形3**：k = m（假设 m ≤ n）

- H_m(S^m) ⊗ H_0(S^n) = ℤ
- H_m = ℤ

**情形4**：k = n（假设 n > m）

- H_0(S^m) ⊗ H_n(S^n) = ℤ
- H_n = ℤ

**情形5**：k = m + n

- H_m(S^m) ⊗ H_n(S^n) = ℤ
- H_{m+n} = ℤ

**Step 3：结果汇总**

假设 m ≤ n：
H_k(S^m × S^n) = ℤ (k = 0, m, n, m+n)（若 m ≠ n）
H_m = ℤ^2 (若 m = n)
0（其他）

若 m = n，则 H_m = ℤ^2（两个因子贡献）。

#### 验证与反思

**正确性验证：**

- ✅ T^2 = S^1 × S^1：H_0 = H_2 = ℤ，H_1 = ℤ^2 ✓
- ✅ S^2 × S^3：H_0 = H_2 = H_3 = H_5 = ℤ ✓
- ✅ Euler示性数：χ(X × Y) = χ(X)χ(Y) = (1+(-1)^m)(1+(-1)^n) ✓

**方法反思：**
> Künneth公式是代数拓扑中最重要的结构性结果之一。它将积空间的几何"分解"为因子空间的几何。十字项（当 m ≠ n 时）反映了积空间的"新"拓扑特征。

| 上同调Künneth | H^*(X × Y) | 杯积结构 |

- 非满射的像可缩，故 f ≃ const

由Hurewicz：
π_n(S^n) ≅ H_n(S^n) = ℤ

**Step 4：生成元**

恒等映射 id: S^n → S^n：

- id_*[S^n] = [S^n] 生成 H_n(S^n)
- 故 [id] 生成 π_n(S^n)

映射度：f: S^n → S^n 诱导 f_*: H_n(S^n) → H_n(S^n)，即乘以整数 deg(f)。

**Step 5：π_k(S^n) 对 k > n**

虽然 H_k(S^n) = 0 对 k > n，但 π_k(S^n) 可以非平凡。

**Hopf纤维化**：S^3 → S^2 是非平凡的，实际上：
π_3(S^2) = ℤ

生成元是Hopf映射 η: S^3 → S^2。

这是同伦论发展的重要里程碑，证明了同伦群比同调群"更精细"。

#### 验证与反思

**正确性验证：**

- ✅ π_1(S^1) = ℤ（已知）✓
- ✅ π_2(S^2) = ℤ，生成元为恒等映射 ✓
- ✅ Hopf纤维化：S^1 → S^3 → S^2，长正合列给出 π_3(S^2) = ℤ ✓

**方法反思：**
> Hurewicz定理是同伦论与同调论之间的桥梁。对于球面，低维同伦群由Hurewicz定理完全确定，但高维同伦群极其复杂，是代数拓扑的核心难题之一。

#### 相关变体

| 变体 | 空间/问题 | 结果 |
|:---:|:---|:---|
| Freudenthal纬悬 | π_k(S^n) → π_{k+1}(S^{n+1}) | 稳定同伦群 |
| 球面稳定同伦群 | π_k^S = lim_{n→∞} π_{n+k}(S^n) | 有限生成阿贝尔群 |
| Serre定理 | π_k(S^n) 对 k > n | 有限群（除特定情形） |

### 问题25：纤维丛的同伦长正合列

#### 问题陈述

设 p: E → B 是Serre纤维丛，纤维 F = p^(-1)(b_0)。

- (a) 陈述同伦长正合列
- (b) 应用于Hopf纤维化 S^1 → S^3 → S^2 计算 π_k(S^2)
- (c) 讨论纤维化的同调谱序列

#### 问题类型识别

| 维度 | 类型 | 核心任务 |
|:---:|:---|:---|
| 对象类型 | 纤维丛 | 同伦计算工具 |
| 方法类型 | 长正合列 | 纤维化理论 |
| 难度等级 | ⭐⭐⭐⭐⭐ | 最高级理论问题 |

#### 方法选择决策

```

┌─────────────────────────────────────────────────────────┐
│  问题：纤维化的同伦正合列                                  │
│  ↓                                                       │
│  长正合列：... → π_n(F) → π_n(E) → π_n(B) → π_{n-1}(F) → ...│
│  ↓                                                       │
│  Hopf纤维化 S^1 → S^3 → S^2                              │
│  应用：利用 π_k(S^1) 和 π_k(S^3) 计算 π_k(S^2)            │
└─────────────────────────────────────────────────────────┘

```

#### 完整解决流程

**Step 1：同伦长正合列**

对于纤维丛 F → E → B，存在长正合列：
... → π_n(F) →[i_*] π_n(E) →[p_*] π_n(B) →[∂] π_{n-1}(F) → ...

其中：

- i: F ↪ E 是包含
- p: E → B 是投影
- ∂ 是连接同态（由提升性质定义）

**Step 2：Hopf纤维化**

S^1 →[i] S^3 →[p] S^2

其中 p(z_1, z_2) = [z_1 : z_2]（复射影映射）。

**Step 3：计算 π_k(S^2)**

利用长正合列和已知的 π_k(S^1)、π_k(S^3)：

k = 2：
π_2(S^1) → π_2(S^3) → π_2(S^2) → π_1(S^1) → π_1(S^3)

即：
0 → 0 → π_2(S^2) → ℤ → 0

因此 π_2(S^2) = ℤ。

k = 3：
π_3(S^1) → π_3(S^3) → π_3(S^2) → π_2(S^1)

即：
0 → ℤ → π_3(S^2) → 0

因此 π_3(S^2) = ℤ，生成元是Hopf映射 η。

k ≥ 4：
π_k(S^1) = 0 ⇒ π_k(S^3) ≅ π_k(S^2)

因此 π_4(S^2) = π_4(S^3) = ℤ/2ℤ（由Serre计算）。

**Step 4：Leray-Serre谱序列**

同调版本的工具是Leray-Serre谱序列：
E^2_{p,q} = H_p(B; H_q(F)) ⇒ H_{p+q}(E)

对于Hopf纤维化：

- E^2_{p,q} = H_p(S^2) ⊗ H_q(S^1)
- E^2 页只有 q = 0, 1 行非零
- 谱序列坍缩，验证 H_*(S^3) 的正确性

#### 验证与反思

**正确性验证：**

- ✅ π_2(S^2) = ℤ：与Hurewicz一致 ✓
- ✅ π_3(S^2) = ℤ：Hopf不变量为1的映射存在 ✓
- ✅ 同调：H_*(S^3) 从 S^2 和 S^1 的扭转积正确恢复 ✓

**方法反思：**
> 纤维化的同伦长正合列是计算同伦群的最有力工具之一。Hopf纤维化是典范例子，展示了即使对于简单空间，同伦群也包含丰富信息。谱序列则是系统处理复杂纤维化的代数工具。

#### 相关变体

| 变体 | 纤维化 | 应用 |
|:---:|:---|:---|
| 路径纤维化 | ΩX → PX → X | 计算环路空间同伦 |
| Eilenberg-MacLane | K(G, n) | 上同调运算 |
| 球丛 | S^k → E → B | 示性类理论 |
