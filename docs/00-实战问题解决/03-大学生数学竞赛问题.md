# 大学生数学竞赛经典问题

> 本问题集收录20道适合大学生的数学竞赛问题，涵盖数学分析、高等代数、微分方程等领域。

## 数学分析 (8题)

### 问题 U1

**题目：** 设 $f: [0,1] \to \mathbb{R}$ 是连续函数，满足 $\int_0^1 f(x) \, dx = 0$ 和 $\int_0^1 xf(x) \, dx = 0$。证明：存在 $c \in (0,1)$，使得 $f(c) = 0$。

**来源：** 全国大学生数学竞赛

**难度：** ★★★☆☆

**提示：** 利用积分的线性组合，考虑函数 $g(x) = ax + b$。

---

### 问题 U2

**题目：** 计算极限：

$$\lim_{n \to \infty} n \int_0^1 \frac{x^n}{1+x} \, dx$$

**来源：** CMC（中国数学竞赛）

**难度：** ★★★☆☆

**提示：** 分部积分或变量替换 $t = x^n$。

---

### 问题 U3

**题目：** 设 $f$ 在 $\mathbb{R}$ 上二阶可导，且 $f''(x) > 0$ 对所有 $x$ 成立。证明：若 $f$ 有上界，则 $f$ 必为常数。

**来源：** 全国大学生数学竞赛

**难度：** ★★★★☆

**提示：** 反证法，利用凸函数的性质。

---

### 问题 U4

**题目：** 设 $\{a_n\}$ 是正数列，满足 $\sum_{n=1}^{\infty} a_n = +\infty$。证明：

$$\sum_{n=1}^{\infty} \frac{a_n}{a_1 + a_2 + \cdots + a_n} = +\infty$$

**来源：** CMC

**难度：** ★★★★☆

**提示：** 令 $S_n = a_1 + \cdots + a_n$，利用比较判别法。

---

### 问题 U5

**题目：** 设 $f: [0,1] \to \mathbb{R}$ 是连续函数。证明：

$$\lim_{n \to \infty} \int_0^1 f(x^n) \, dx = f(0)$$

**来源：** 全国大学生数学竞赛

**难度：** ★★★☆☆

**提示：** 分段估计，利用一致连续性。

---

### 问题 U6

**题目：** 设 $f$ 在 $[0,1]$ 上连续，在 $(0,1)$ 内可导，且 $f(0) = 0$，$f(1) = 1$。证明：对任意正数 $a, b$，存在 $\xi, \eta \in (0,1)$，$\xi \neq \eta$，使得：

$$\frac{a}{f'(\xi)} + \frac{b}{f'(\eta)} = a + b$$

**来源：** CMC

**难度：** ★★★★☆

**提示：** 应用中值定理两次，适当选择中间点。

---

### 问题 U7

**题目：** 设 $f$ 在 $[a,b]$ 上连续，在 $(a,b)$ 内可导，且 $f(a) = f(b) = 0$。证明：存在 $c \in (a,b)$，使得：

$$f'(c) = f(c)$$

**来源：** 全国大学生数学竞赛

**难度：** ★★★☆☆

### 问题 U8

**题目：** 设 $f$ 在 $[0,+\infty)$ 上连续，且 $\lim_{x \to +\infty} f(x) = L$（$L$ 为有限数）。证明：

### U2 解答

**方法1（分部积分）：**

令 $I_n = n \int_0^1 \frac{x^n}{1+x} \, dx$。

分部积分：
$$I_n = n \left[ \frac{x^{n+1}}{(n+1)(1+x)} \right]_0^1 + n \int_0^1 \frac{x^{n+1}}{(n+1)(1+x)^2} \, dx$$

$$= \frac{n}{2(n+1)} + \frac{n}{n+1} \int_0^1 \frac{x^{n+1}}{(1+x)^2} \, dx$$

当 $n \to \infty$ 时，第一项趋于 $\frac{1}{2}$。

对于第二项：
$$0 \leq \int_0^1 \frac{x^{n+1}}{(1+x)^2} \, dx \leq \int_0^1 x^{n+1} \, dx = \frac{1}{n+2} \to 0$$

因此 $\lim_{n \to \infty} I_n = \frac{1}{2}$。

**答案：** $\frac{1}{2}$

### U4 解答

令 $S_n = a_1 + a_2 + \cdots + a_n$。

对于 $n \geq 2$：
$$\frac{a_n}{S_n} = \frac{S_n - S_{n-1}}{S_n} = 1 - \frac{S_{n-1}}{S_n}$$

因此：
$$\sum_{n=2}^{N} \frac{a_n}{S_n} = \sum_{n=2}^{N} \left(1 - \frac{S_{n-1}}{S_n}\right)$$

利用不等式 $1 - x \geq -\ln x$（$x > 0$）：
$$\sum_{n=2}^{N} \left(1 - \frac{S_{n-1}}{S_n}\right) \geq \sum_{n=2}^{N} \ln\frac{S_n}{S_{n-1}} = \ln\frac{S_N}{S_1} \to +\infty$$

因此原级数发散到 $+\infty$。

### U7 解答

设 $g(x) = f(x)e^{-x}$，则 $g(a) = g(b) = 0$。

由罗尔定理，存在 $c \in (a,b)$，使得 $g'(c) = 0$。

计算：
$$g'(x) = f'(x)e^{-x} - f(x)e^{-x} = (f'(x) - f(x))e^{-x}$$

因此 $g'(c) = 0$ 意味着 $f'(c) - f(c) = 0$，即 $f'(c) = f(c)$。

### U9 解答

设 $\lambda$ 是 $A$ 的特征值，则 $\lambda^3 = \lambda + 1$。

即 $\lambda$ 满足 $\lambda^3 - \lambda - 1 = 0$。

设三个根为 $\lambda_1, \lambda_2, \lambda_3$（可能为复数）。

由韦达定理：
- $\lambda_1 + \lambda_2 + \lambda_3 = 0$
- $\lambda_1\lambda_2 + \lambda_2\lambda_3 + \lambda_3\lambda_1 = -1$
- $\lambda_1\lambda_2\lambda_3 = 1$

由于 $\det A$ 是特征值的乘积，若 $A$ 有 $k$ 个特征值为 $\lambda_1$，$m$ 个为 $\lambda_2$，$n$ 个为 $\lambda_3$，则：
$$\det A = \lambda_1^k \lambda_2^m \lambda_3^n$$

实际上，复特征值成共轭对出现。设 $\lambda_1$ 为实根，$\lambda_2, \lambda_3$ 为共轭复根。

可以验证 $\lambda_1 > 0$（因为 $p(1) = -1 < 0$，$p(2) = 5 > 0$）。

对于复根 $\lambda_2, \lambda_3 = \bar{\lambda_2}$，有 $|\lambda_2|^2 = \lambda_2\lambda_3 = \frac{1}{\lambda_1} > 0$。

因此 $\det A = \lambda_1^k |\lambda_2|^{2m} > 0$。

### U11 解答

由 $AB - BA = A$，对两边取迹：
$$\text{tr}(AB - BA) = \text{tr}(A)$$

但 $\text{tr}(AB) = \text{tr}(BA)$，故 $\text{tr}(A) = 0$。

归纳证明 $\text{tr}(A^k) = 0$ 对所有 $k \geq 1$ 成立：

假设 $\text{tr}(A^k) = 0$，考虑 $A^{k+1}$：
$$A^{k+1} = A^k \cdot A = A^k(AB - BA) = A^kAB - A^kBA$$

因此：
$$\text{tr}(A^{k+1}) = \text{tr}(A^{k+1}B) - \text{tr}(A^kBA) = \text{tr}(A^{k+1}B) - \text{tr}(A^{k+1}B) = 0$$

由于 $A$ 的所有幂的迹为零，$A$ 是幂零矩阵。

### U15 解答

**步骤1：求齐次解**

特征方程：$r^2 + 1 = 0$，得 $r = \pm i$。

齐次解：$y_h = C_1 \cos x + C_2 \sin x$。

**步骤2：求特解**

由于 $\sin x$ 与齐次解中的 $\sin x$ 重复，设特解形式为 $y_p = x(A\cos x + B\sin x)$。

计算：
$$y_p' = A\cos x + B\sin x + x(-A\sin x + B\cos x)$$
$$y_p'' = -2A\sin x + 2B\cos x - x(A\cos x + B\sin x)$$

代入方程：
$$y_p'' + y_p = -2A\sin x + 2B\cos x = \sin x$$

比较系数：$A = -\frac{1}{2}$，$B = 0$。

因此 $y_p = -\frac{1}{2}x\cos x$。

**步骤3：通解与特解**

通解：$y = C_1 \cos x + C_2 \sin x - \frac{1}{2}x\cos x$。

由 $y(0) = 0$：$C_1 = 0$。

由 $y'(0) = 1$：$y' = C_2\cos x - \frac{1}{2}\cos x + \frac{1}{2}x\sin x$，故 $C_2 - \frac{1}{2} = 1$，即 $C_2 = \frac{3}{2}$。

**答案：** $y = \frac{3}{2}\sin x - \frac{1}{2}x\cos x$

### U20 解答

**步骤1：分离变量**

设 $u(x,t) = X(x)T(t)$，代入方程：
$$X(x)T'(t) = X''(x)T(t)$$
$$\frac{T'(t)}{T(t)} = \frac{X''(x)}{X(x)} = -\lambda$$

**步骤2：求解空间部分**

$X'' + \lambda X = 0$，$X(0) = X(1) = 0$。

特征值：$\lambda_n = (n\pi)^2$，$X_n(x) = \sin(n\pi x)$，$n = 1, 2, 3, \ldots$

**步骤3：求解时间部分**

$T_n'(t) = -(n\pi)^2 T_n(t)$，故 $T_n(t) = e^{-(n\pi)^2 t}$。

**步骤4：叠加**

$$u(x,t) = \sum_{n=1}^{\infty} c_n \sin(n\pi x) e^{-(n\pi)^2 t}$$

由初始条件 $u(x,0) = \sin(\pi x)$：
$$\sum_{n=1}^{\infty} c_n \sin(n\pi x) = \sin(\pi x)$$

因此 $c_1 = 1$，$c_n = 0$（$n \geq 2$）。

**答案：** $u(x,t) = \sin(\pi x) e^{-\pi^2 t}$
