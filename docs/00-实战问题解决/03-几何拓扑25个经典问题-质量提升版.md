## 问题1：球面S^n的光滑结构验证 ⭐⭐☆☆☆

### 问题描述

验证n维球面 S^n = {x ∈ R^{n+1} : |x| = 1} 配备标准坐标卡后构成光滑流形。

### 可视化描述

**球极投影的几何直观**：
- 从北极N向平面投影：将球面上除N外的点映射到切平面
- 转移映射是反演变换，具有优美的几何性质

### 解答

**坐标卡构造**：
- 北半球卡 φ_N: S^n\{N} → R^n，φ_N(x) = (1/(1-x_{n+1}))(x_1,...,x_n)
- 南半球卡 φ_S: S^n\{S} → R^n，φ_S(x) = (1/(1+x_{n+1}))(x_1,...,x_n)

**转移映射**：φ_S ∘ φ_N^{-1}(y) = y/|y|²（反演变换）

反演变换在 R^n\{0} 上光滑，故 S^n 是光滑流形。

### 应用背景

球面是物理学和工程学中最常见的流形：
- 天球：天体位置的表示
- 地球表面：导航和地图投影
- 量子力学：自旋态的Bloch球

## 问题2：环面T²的切丛构造 ⭐⭐⭐☆☆

### 问题描述

构造环面 T² = S¹ × S¹ 的切丛 TT²，证明其平凡性。

### 可视化描述

**环面的参数化**：
```
Φ(θ, φ) = ((R + r·cos θ)cos φ, (R + r·cos θ)sin φ, r·sin θ)
```

这描绘了一个在3维空间中旋转的圆环。

### 解答

**利用积流形性质**：
T(T²) = T(S¹ × S¹) ≅ TS¹ × TS¹ ≅ (S¹ × R) × (S¹ × R) ≅ T² × R²

**全局标架场**：
- X₁ = ∂/∂θ（经线方向）
- X₂ = ∂/∂φ（纬线方向）

这两个向量场处处线性无关，构成切丛的全局平凡化。

### 深入分析

**对比S²**：S²的切丛不平凡（hairy ball theorem），这反映了拓扑的根本差异。

**定理**：n维环面T^n都是平行化的。

## 问题3：莫比乌斯带上的向量场 ⭐⭐⭐⭐☆

### 问题描述

构造莫比乌斯带上的光滑向量场，使其恰有一个零点，并分析拓扑指数。

### 可视化描述

**商空间模型**：
M = ([0,1] × R) / ~，其中 (0,x) ~ (1,-x)

这种"翻转粘贴"创造了一边形的拓扑结构。

### 解答

**向量场构造**：
V(t,x) = sin(2πt)·∂/∂t + x·cos(2πt)·∂/∂x

**零点分析**：
- sin(2πt) = 0 且 x·cos(2πt) = 0
- t = 0, 1/2 且 x = 0
- 由于等同关系，只有一个零点 [(0,0)] = [(1,0)]

**拓扑指数**：ind(V,0) = +1（特征值均为正）

### 应用背景

莫比乌斯带在：
- 传送带设计（均匀磨损）
- 分子结构（Möbius aromaticity）
- 艺术和建筑中的美学应用

## 问题4：Grassmann流形的切空间 ⭐⭐⭐⭐⭐

### 问题描述

刻画Grassmann流形 G(k,n) 在点 Π 处的切空间 T_Π G(k,n)。

### 解答

**齐性空间表示**：
G(k,n) = O(n) / (O(k) × O(n-k))

**切空间**：
T_Π G(k,n) ≅ Hom(Π, Π^⊥)

维数：dim G(k,n) = k(n-k)

### 几何解释

切方向对应子空间的"旋转"：矩阵B描述如何将Π中的向量映射到正交补空间。

## 问题5：隐函数定理应用 ⭐⭐⭐☆☆

### 问题描述

设 f(x,y,z) = x² + y² - z²，M = f^{-1}(1)。

1. 证明M是光滑流形
2. 求某点的切空间
3. 参数化M并计算诱导度量

### 可视化描述

M是**双叶双曲面**，形状类似两个对顶的碗，沿z轴方向无限延伸。

### 解答

**步骤1**：df = (2x, 2y, -2z)，在f^{-1}(1)上df ≠ 0，1是正则值。

**步骤2**：切空间 T_pM = ker df_p = {v : 2x₀v₁ + 2y₀v₂ - 2z₀v₃ = 0}

**步骤3**：双曲坐标参数化（z>0的叶）：
x = cosh u · cos v, y = cosh u · sin v, z = sinh u

**诱导度量**：g = cosh(2u) du² + cosh²u dv²

## 问题6：带边流形的边界坐标卡 ⭐⭐⭐☆☆

### 问题描述

设 M = {(x,y,z) ∈ R³ : x²+y²+z² ≤ 1, z ≥ 0}（上半球体）。

1. 证明M是带边3维流形
2. 构造边界和内点的坐标卡
3. 计算边界的诱导定向

### 可视化描述

- 内部：开上半球 {x²+y²+z² < 1, z > 0}
- 边界：闭圆盘 {x²+y² ≤ 1, z = 0}

### 解答

**内部坐标卡**：标准球坐标限制

**边界坐标卡**：恒等映射 ψ(x,y,z) = (x,y,z)，将边界{z=0}映射到{w=0}

**诱导定向**：
- M继承R³的标准定向：dx∧dy∧dz
- 外法向 ν = (0,0,-1)
- 诱导定向：ι_ν(dx∧dy∧dz) = -dx∧dy

## 问题7：平面曲线的曲率计算 ⭐⭐☆☆☆

### 问题描述

设 γ(t) = (t, t²) 为抛物线。

1. 计算弧长参数化
2. 求曲率 κ(s)
3. 分析几何意义

### 可视化描述

抛物线在顶点处弯曲最剧烈，远离顶点时逐渐变平。

### 解答

**一般参数曲率公式**：
κ = |x'y'' - x''y'| / (x'² + y'²)^{3/2}

**计算**：
- x' = 1, x'' = 0
- y' = 2t, y'' = 2
- κ(t) = 2 / (1 + 4t²)^{3/2}

**分析**：
- t = 0（顶点）：κ_max = 2
- |t| → ∞：κ → 0

### 应用背景

抛物线的曲率性质使其成为：
- 卫星天线设计（焦点性质）
- 桥梁和建筑结构
- 逼近曲面的重要工具

## 问题8：悬链面的高斯曲率 ⭐⭐⭐☆☆

### 问题描述

悬链面 X(u,v) = (cosh u · cos v, cosh u · sin v, u)，计算第一、第二基本形式及高斯曲率K。

### 可视化描述

悬链面是唯一的**旋转极小曲面**，形状像拉紧的肥皂膜在两个圆环之间形成的曲面。

### 解答

**第一基本形式**：
E = cosh²u, F = 0, G = cosh²u

**第二基本形式**：
L = -cosh u/√cosh(2u), M = 0, N = cosh u/√cosh(2u)

**高斯曲率**：
K = (LN-M²)/(EG-F²) = -1/(cosh²u · cosh(2u)) < 0

**极小曲面验证**：平均曲率H = 0 ✓

## 问题9：球面的Riemann曲率张量 ⭐⭐⭐⭐☆

### 问题描述

设 S² ⊂ R³ 为半径R的球面，计算Christoffel符号、Riemann曲率张量、Ricci曲率和标量曲率。

### 解答

**度量**：g = R² dθ² + R² sin²θ dφ²

**Christoffel符号**：
Γ^θ_φφ = -sin θ cos θ
Γ^φ_θφ = Γ^φ_φθ = cot θ

**Riemann曲率**：
R^θ_φθφ = sin²θ

**Ricci曲率**：R_ij = (1/R²)g_ij

**标量曲率**：R = 2/R²

### 物理意义

Einstein场方程 R_ij - (1/2)Rg_ij = 8πT_ij 中，球面是真空解（T=0）的特例。

## 问题10：双曲平面的测地线 ⭐⭐⭐⭐☆

### 问题描述

上半平面模型 H² = {(x,y) : y > 0}，配备庞加莱度量 ds² = (dx²+dy²)/y²。

1. 计算测地线方程
2. 证明测地线是垂直于x轴的半圆或直线

### 可视化描述

双曲几何中的"直线"是垂直于边界的圆弧，这与欧氏几何截然不同。

### 解答

**Lagrange量**：L = (ẋ² + ẏ²)/y²

**Euler-Lagrange方程**：
ẍ - 2ẋẏ/y = 0
ÿ + (ẋ² - ẏ²)/y = 0

**首次积分**：p_x = 2ẋ/y² = c₁

**求解轨迹**：
(x - x₀)² + y² = 4/c₁²

这是中心在(x₀, 0)、半径为2/|c₁|的圆在上半平面的部分。

### 几何意义

双曲几何的核心特征：三角形内角和小于π，这使得双曲几何具有丰富的三角形性质。
