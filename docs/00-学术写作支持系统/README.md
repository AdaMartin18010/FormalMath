## 系统概述

FormalMath学术写作支持系统是FormalMath项目的T5.1任务成果，旨在为数学专业学生提供全方位的学术写作支持。本系统对标哈佛大学数学系学术写作标准，涵盖说明性论文、毕业论文等多种论文类型的写作指南。

### 系统目标

- ✅ 提供标准化的数学论文写作规范
- ✅ 建立可直接使用的写作模板库
- ✅ 整理数学写作常用资源和工具
- ✅ 提供优秀论文示例供参考学习

### 适用对象

- 数学专业本科生（写作课程论文、毕业论文）
- 数学专业研究生（写作研究论文、综述）
- 数学教师（指导学生写作、课程设计）
- 自学者（系统学习数学学术写作）

## 快速开始

### 第一步：了解基础

阅读[数学学术写作概述](写作指南/01-数学学术写作概述.md)，了解数学写作的特点和基本要求。

### 第二步：根据类型选择指南

| 论文类型 | 推荐指南 | 推荐模板 |
|---------|---------|---------|
| 说明性论文（5-10页） | [说明性论文专项指南](写作指南/08-说明性论文专项指南.md) | [课程论文模板](写作模板/01-课程论文LaTeX模板.tex) |
| 学期论文（15-20页） | [论文结构与组织](写作指南/02-论文结构与组织.md) | [课程论文模板](写作模板/01-课程论文LaTeX模板.tex) |
| 毕业论文（50+页） | [毕业论文专项指南](写作指南/09-毕业论文专项指南.md) | [毕业论文模板](写作模板/02-毕业论文LaTeX模板.tex) |
| 学术演讲 | [学术演讲与海报](写作指南/10-学术演讲与海报.md) | [Beamer模板](写作模板/03-Beamer幻灯片模板.tex) |
| 学术海报 | [学术演讲与海报](写作指南/10-学术演讲与海报.md) | [海报模板](写作模板/04-学术海报模板.tex) |

### 第三步：参考示例学习

查看对应方向的[示例论文](示例论文/)，学习优秀论文的写作方式。

### 第四步：使用资源工具

写作过程中随时查阅[写作资源](写作资源/)中的速查表和检查清单。

## 使用建议

### 初学者路径

```

1. 阅读《数学学术写作概述》
2. 根据论文类型阅读对应专项指南
3. 下载对应模板开始写作
4. 参考示例论文学习写法
5. 使用检查清单进行自查

```

### 进阶使用

```

1. 研究资源中的速查表，提高效率
2. 使用文献综述模板进行文献整理
3. 使用同行评议表单进行互评
4. 建立个人写作习惯和工作流

```

### 教师使用

```

1. 将本系统作为课程辅助材料
2. 根据具体课程选择相关指南
3. 使用模板作为作业提交格式
4. 使用示例作为评分标准参考

```
