## 示例1: 定义写作 - "连续性"

### 水平A: 初学者

```

连续就是函数图像没有断开。如果一个函数在某点连续，
那么它的图像在这点不会跳跃。比如x²在所有点都连续。

```

**问题**: 模糊、不严格、缺乏数学定义

### 水平B: 中级

```

定义: 函数f在点a连续，如果lim_{x→a} f(x) = f(a)。
这意味着当x接近a时，f(x)接近f(a)。

```

**问题**: 有定义但缺乏动机和例子

### 水平C: 优秀

```

**连续性的直观与严格定义**

直观上，连续函数是图像"一笔画成"的函数——没有
跳跃、断裂或孔洞。要将这一直观严格化，我们需要
量化的语言。

**定义 1.1** (连续性). 设 f: ℝ → ℝ，a ∈ ℝ。称 f 在 a
点连续，如果对于任意 ε > 0，存在 δ > 0，使得当

|x - a| < δ 时，有 |f(x) - f(a)| < ε。

这一定义（由Cauchy和Weierstrass发展）捕捉了直观
的核心: 自变量的小变化只导致函数值的小变化。
参数 ε 和 δ 量化了"小"的概念。

**例子 1.2**.
(1) 常数函数 f(x) = c 处处连续。对任意 ε > 0，
    任取 δ > 0 即可，因为 |f(x) - f(a)| = 0 < ε。

(2) 恒等函数 f(x) = x 处处连续。给定 ε > 0，
    取 δ = ε，则 |x - a| < δ ⟹ |f(x) - f(a)| =
    |x - a| < ε。

(3) 函数 f(x) = { 1, x ≥ 0; 0, x < 0 } 在 0 点不连续。
    取 ε = 1/2，对任意 δ > 0，存在 x = -δ/2 满足

    |x - 0| < δ 但 |f(x) - f(0)| = 1 > ε。

```

**优点**: 直观-定义-例子三层递进，严格且易懂

## 示例2: 证明写作 - "√2 的无理性"

### 修改前

```

证明√2是无理数。假设√2=p/q，则2q²=p²，所以p²是偶数，
p是偶数，设p=2k，则2q²=4k²，q²=2k²，q也是偶数，矛盾。

```

**问题**: 过于简洁，缺乏解释，难以理解

### 修改后

```

**命题**: √2 是无理数。

**证明**. 我们使用反证法。假设 √2 是有理数，则可以写成
既约分数形式 √2 = p/q，其中 p, q ∈ ℤ，q ≠ 0，且
gcd(p, q) = 1。

两边平方得:
    2 = p²/q²  ⟹  p² = 2q².                    (1)

由 (1) 知 p² 是偶数。我们断言 p 必为偶数: 若 p 为奇数，
设 p = 2k + 1，则 p² = 4k² + 4k + 1 也是奇数，矛盾。

因此可设 p = 2k，k ∈ ℤ。代入 (1):
    (2k)² = 2q²  ⟹  4k² = 2q²  ⟹  q² = 2k².

这表明 q² 是偶数，从而 q 也是偶数。

但此时 p 和 q 都是偶数，与 gcd(p, q) = 1 的假设矛盾。
因此 √2 不能是有理数，即 √2 是无理数。 □

```

**改进**:

- 明确证明策略（反证法）
- 每个步骤都有解释
- 关键断言有理由
- 清晰的结束标记

## 示例3: 论文结构 - 摘要对比

### 原稿

```

这篇论文讨论了群论。我们证明了一些定理。主要结果是
Sylow定理。我们也给出了一些应用。

```

**问题**: 过于笼统，没有具体信息

### 修改稿

```

**摘要**: 本文系统阐述有限群Sylow理论的基本结果。
首先回顾群作用和轨道-稳定子定理，作为研究Sylow
子群的基础工具。然后证明Sylow三定理: (1) Sylow
p-子群的存在性，(2) 所有Sylow p-子群的共轭性，
(3) Sylow p-子群数目的同余条件。证明采用群作用
在集合上的轨道分析方法。作为应用，我们分类了
阶为pq（p, q为不同素数）的群，并证明30阶群
不是单群。本文适合具有基础群论知识的读者。

**关键词**: Sylow定理，有限群，群作用，单群分类

```

**改进**:

- 具体说明内容
- 明确主要结果
- 说明方法
- 给出应用
- 指明读者对象

## 优秀写作的共同特点

1. **清晰的目标读者定位**
2. **从直观到严格的概念引入**
3. **充分的例子和反例**
4. **证明的逐步解释**
5. **适当的动机说明**
6. **良好的视觉组织**
7. **准确的引用和参考**
