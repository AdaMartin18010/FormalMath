# 图表与可视化

## 一、数学论文中的图表

### 1.1 图表的作用

数学论文中的图表承担以下功能：
- **说明概念**：复杂定义的直观解释
- **展示结构**：群、范畴、格的结构图
- **可视化数据**：数值结果、统计信息
- **说明关系**：交换图、蕴含图

### 1.2 图表选择指南

| 图表类型 | 适用场景 | LaTeX工具 |
|----------|----------|-----------|
| 函数图像 | 分析、微积分 | pgfplots, TikZ |
| 几何图形 | 几何、拓扑 | TikZ, PSTricks |
| 交换图 | 代数、范畴论 | tikz-cd |
| 示意图 | 一般说明 | TikZ |
| 数据图表 | 统计、计算 | pgfplots |
| 树形图 | 递归、博弈 | forest, TikZ |

## 二、TikZ基础

### 2.1 基本结构

```latex
\begin{tikzpicture}
  % 绘图命令
\end{tikzpicture}

```

### 2.2 基本元素

**点与线**：

```latex
\begin{tikzpicture}
  \draw (0,0) -- (2,0);        % 直线
  \draw (0,0) -- (2,2) -- (0,2) -- cycle;  % 三角形
  \fill (1,1) circle (2pt);    % 点
\end{tikzpicture}

```

**节点**：

```latex
\begin{tikzpicture}
  \node (A) at (0,0) {$A$};
  \node (B) at (2,0) {$B$};
  \draw[->] (A) -- (B);
\end{tikzpicture}

```

### 2.3 样式设置

```latex
\begin{tikzpicture}[
    node distance=2cm,
    auto,
    thick
  ]
  \node[draw, circle] (A) {$A$};
  \node[draw, rectangle, right of=A] (B) {$B$};
  \draw[->, blue, dashed] (A) -- (B);
\end{tikzpicture}

```

## 三、交换图

### 3.1 基本交换图

```latex
\begin{tikzcd}
  A \arrow{r}{f} \arrow{d}{g} & B \arrow{d}{h} \\
  C \arrow{r}{k} & D
\end{tikzcd}

```

效果：

```

  A → B
  ↓   ↓
  C → D

```

### 3.2 高级交换图

**弯曲箭头**：

```latex
\begin{tikzcd}
  A \arrow[bend left]{r}{f} 
    \arrow[bend right]{r}[swap]{g} & B
\end{tikzcd}

```

**长箭头**：

```latex
\begin{tikzcd}
  A \arrow{rr}{f} \arrow{dr}[swap]{g} & & B \\
  & C \arrow{ur}[swap]{h} &
\end{tikzcd}

```

**双箭头**：

```latex
\begin{tikzcd}
  A \arrow[r, shift left, "f"] 
    \arrow[r, shift right, swap, "g"] & B
\end{tikzcd}

```

**等号**：

```latex
\begin{tikzcd}
  A \arrow{r}{f} \arrow{d}{g} & B \arrow{d}{h} \\
  C \arrow{r}{k} & D
  \arrow[from=1-1, to=2-2, phantom, "\circlearrowleft"]
\end{tikzcd}

```

## 四、函数图像

### 4.1 基本图像

```latex
\begin{tikzpicture}
  \begin{axis}[
    xlabel=$x$,
    ylabel=$y$,
    xmin=-5, xmax=5,
    ymin=-5, ymax=5,
    grid=both,
    axis lines=middle
  ]
    \addplot[domain=-3:3, blue, thick] {x^2};
    \addplot[domain=-3:3, red, thick] {x^3};
    \legend{$x^2$, $x^3$}
  \end{axis}
\end{tikzpicture}

```

### 4.2 参数曲线

```latex
\begin{tikzpicture}
  \begin{axis}
    \addplot[domain=0:360, samples=100, thick]
      ({cos(x)}, {sin(x)});
  \end{axis}
\end{tikzpicture}

```

### 4.3 三维图形

```latex
\begin{tikzpicture}
  \begin{axis}[
    xlabel=$x$, ylabel=$y$, zlabel=$z$
  ]
    \addplot3[surf, domain=-2:2, samples=30]
      {x^2 - y^2};
  \end{axis}
\end{tikzpicture}

```

## 五、几何图形

### 5.1 几何构造

```latex
\begin{tikzpicture}
  % 三角形
  \coordinate (A) at (0,0);
  \coordinate (B) at (4,0);
  \coordinate (C) at (2,3);
  
  \draw (A) -- (B) -- (C) -- cycle;
  
  % 标注
  \node[below] at (A) {$A$};
  \node[below] at (B) {$B$};
  \node[above] at (C) {$C$};
  
  % 角度
  \draw (A) ++(0.5,0) arc (0:56:0.5);
\end{tikzpicture}

```

### 5.2 单位圆

```latex
\begin{tikzpicture}[scale=2]
  \draw[->] (-1.2,0) -- (1.2,0) node[right] {$x$};
  \draw[->] (0,-1.2) -- (0,1.2) node[above] {$y$};
  \draw (0,0) circle (1);
  
  \draw[thick] (0,0) -- (30:1) node[right] {$P$};
  \draw[dashed] (30:1) -- (0.866,0);
  \draw (0.2,0) arc (0:30:0.2) node[midway, right] {$\theta$};
\end{tikzpicture}

```

## 六、图表最佳实践

### 6.1 设计原则

**简洁性**：
- 移除不必要的装饰
- 保持最小数据-墨水比
- 每个元素都应有明确目的

**清晰性**：
- 标签清晰可读
- 比例适当
- 使用图例说明多个元素

**一致性**：
- 同类型图表风格统一
- 颜色方案一致
- 字体大小一致

### 6.2 颜色使用

**推荐配色**：

```latex
\definecolor{primary}{RGB}{31, 78, 121}
\definecolor{secondary}{RGB}{192, 80, 77}
\definecolor{tertiary}{RGB}{155, 187, 89}

```

**注意事项**：
- 考虑黑白打印效果
- 使用图案区分（不仅依赖颜色）
- 色盲友好配色

### 6.3 图形尺寸

**行内图形**：

```latex
\begin{tikzpicture}[scale=0.5]
  % 小图形内容
\end{tikzpicture}

```

**独立图形**：

```latex
\begin{figure}[htbp]
  \centering
  \begin{tikzpicture}
    % 图形内容
  \end{tikzpicture}
  \caption{图形标题}
  \label{fig:example}
\end{figure}

```

## 七、高级技巧

### 7.1 递归图形

```latex
\begin{tikzpicture}
  \def\tree#1#2{
    \ifnum#2>0
      \draw (0,0) -- (0,#1);
      \begin{scope}[shift={(0,#1)}, scale=0.7]
        \begin{scope}[rotate=30]
          \tree{#1}{\numexpr#2-1\relax}
        \end{scope}
        \begin{scope}[rotate=-30]
          \tree{#1}{\numexpr#2-1\relax}
        \end{scope}
      \end{scope}
    \fi
  }
  \tree{1}{4}
\end{tikzpicture}

```

### 7.2 计算与循环

```latex
\begin{tikzpicture}
  \foreach \i in {1,...,6} {
    \foreach \j in {1,...,6} {
      \node[draw, minimum size=0.5cm] at (\i, \j) {\pgfmathparse{int(\i*\j)}\pgfmathresult};
    }
  }
\end{tikzpicture}

```
