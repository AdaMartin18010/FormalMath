# 毕业论文专项指南

## 一、毕业论文概述

### 1.1 毕业论文的定位

毕业论文（Senior Thesis/Master's Thesis）是：

- 本科/硕士学业的**culmination**（顶点项目）
- 展示**独立研究能力**的主要证据
- 进入研究生院或学术界的重要**敲门砖**

### 1.2 哈佛标准要求

哈佛大学数学系毕业论文要求：

**内容要求**：

- 展示对某一数学领域的深度理解
- 包含原创性成分（可以是新的证明、新的视角、或原创研究）
- 长度通常在50-100页之间

**过程要求**：

- 需要导师全程指导
- 通常需要开题报告
- 需要最终答辩

**质量标准**：

- 达到可发表或接近发表水平
- 能够经受同行评议

## 二、论文类型选择

### 2.1 原创性研究论文

**特点**：

- 包含新的数学结果
- 证明新的定理
- 发现新的现象

**适合情况**：

- 学生已有研究基础
- 导师提供合适问题
- 有充分时间（1-2年）

**风险**：

- 可能无法获得结果
- 需要更多背景知识

### 2.2 综合阐述型论文

**特点**：

- 系统整理一个主题
- 提供新的组织视角
- 可能包含新的证明或解释

**适合情况**：

- 广泛的文献调研
- 喜欢教学/写作
- 时间相对有限

### 2.3 计算/实验型论文

**特点**：

- 大量计算验证
- 数据分析和模式发现
- 猜想提出

**适合情况**：

- 计算能力强
- 适合数论、代数几何等领域
- 导师有计算项目

### 2.4 类型比较

| 维度 | 原创研究 | 综合阐述 | 计算实验 |
|------|---------|---------|---------|
| 创新性 | 最高 | 中等 | 中等 |
| 风险 | 高 | 低 | 中等 |
| 时间需求 | 最长 | 中等 | 中等 |
| 适合PhD申请 | 最佳 | 良好 | 良好 |

## 三、时间管理

### 3.1 理想时间表（1年周期）

```

9月-10月    [选题与文献调研]
            - 与潜在导师沟通
            - 广泛阅读确定方向
            - 撰写选题报告

11月-12月   [深度阅读与问题确定]
            - 精读核心文献
            - 确定具体问题
            - 撰写开题报告

1月-3月     [主要工作阶段]
            - 进行研究/整理
            - 定期与导师会面
            - 完成核心章节

4月-5月     [写作与完善]
            - 完成初稿
            - 导师反馈与修改
            - 同行评议

6月         [定稿与答辩]
            - 最终修改
            - 准备答辩
            - 提交论文

```

### 3.2 里程碑设置

| 时间点 | 里程碑 | 检查点 |
|--------|--------|--------|
| 第1月末 | 选题确定 | 导师确认 |
| 第3月末 | 开题报告 | 委员会批准 |
| 第6月末 | 核心完成 | 主要定理/结果 |
| 第8月末 | 初稿完成 | 导师审阅 |
| 第10月末 | 终稿完成 | 格式审查 |
| 第12月 | 答辩通过 | 委员会认可 |

## 四、论文结构

### 4.1 标准结构

```

前页部分
  ├── 标题页
  ├── 摘要（中英文）
  ├── 致谢
  └── 目录

正文部分
  ├── 第1章 引言 (5-10页)
  ├── 第2章 预备知识 (10-15页)
  ├── 第3章 主要结果 (20-30页)
  ├── 第4章 应用/例子 (5-10页)
  └── 第5章 结论与展望 (3-5页)

后页部分
  ├── 参考文献
  ├── 附录（可选）
  └── 索引（可选）

```

### 4.2 各章节详解

**第1章 引言（5-10页）**

- 研究背景与动机
- 问题陈述
- 文献综述
- 论文贡献
- 论文结构

**第2章 预备知识（10-15页）**

- 必要的定义和记号
- 相关已知定理
- 基础理论框架
- 符号表

**第3章 主要结果（20-30页）**

- 核心定理陈述
- 详细证明
- 相关引理
- 证明技术分析

**第4章 应用/例子（5-10页）**

- 具体例子
- 数值验证（如适用）
- 与其他工作的联系

**第5章 结论与展望（3-5页）

- 工作总结
- 局限性讨论
- 未来研究方向

## 五、与导师的合作

### 5.1 导师角色

**学术指导**：

- 确定研究方向
- 提供文献建议
- 审阅论文草稿

**职业发展**：

- 推荐信
- 学术交流机会
- 职业建议

### 5.2 有效沟通

**定期会面**：

- 建议频率：每周或每两周一次
- 准备议程
- 记录讨论要点

**沟通内容**：

- 进展汇报
- 遇到的困难
- 下一步计划
- 需要的资源

### 5.3 常见问题处理

**导师太忙**：

- 提前邮件预约
- 准备充分提高会面效率
- 利用助教或博士后资源

**研究方向分歧**：

- 坦诚沟通
- 寻求折中方案
- 必要时更换导师（尽早）

## 六、写作过程

### 6.1 写作策略

**自上而下 vs 自下而上**：

自上而下：

```

先写大纲 → 扩展章节 → 完善细节

```

自下而上：

```

先写技术部分 → 整合章节 → 补充引言和结论

```

**建议**：结合两种方式，先写核心结果，再补充框架。

### 6.2 初稿写作

**克服写作障碍**：

- 先写简单的部分
- 不要追求完美
- 保持写作习惯

**初稿目标**：

- 完整的内容
- 不必完美
- 可以被理解

### 6.3 修改策略

**多轮修改**：

第一轮：内容完整性
第二轮：逻辑流畅性
第三轮：语言表达
第四轮：格式规范

**寻求反馈**：

- 导师意见
- 同行评议
- 写作中心

## 七、答辩准备

### 7.1 答辩形式

**标准流程**：

1. 论文陈述（20-30分钟）
2. 委员会提问（30-60分钟）
3. 委员会评议

### 7.2 答辩陈述

**结构建议**：

```

1. 引言与动机 (3分钟)
2. 核心结果陈述 (10分钟)
3. 关键证明思路 (10分钟)
4. 应用与例子 (5分钟)
5. 总结与展望 (2分钟)

```

**准备技巧**：

- 制作清晰的幻灯片
- 准备可能的问答
- 演练时间控制

### 7.3 常见问题

**技术问题**：

- 证明的某个步骤
- 结果的适用范围
- 与相关工作的比较

**方法问题**：

- 为什么选择这个方法
- 其他方法的尝试
- 局限性讨论

**延伸问题**：

- 未来研究方向
- 可能的应用
- 进一步推广

## 八、发表与后续

### 8.1 论文发表

**期刊选择**：

- 研究水平合适的期刊
- 导师建议
- 审稿周期考虑

**改写要点**：

- 精简引言
- 聚焦核心结果
- 符合期刊格式

### 8.2 研究生院申请

**论文作为材料**：

- 写作样本
- 研究陈述基础
- 推荐信素材

**面试准备**：

- 能够简洁介绍论文
- 讨论技术细节
- 展示研究潜力

## 九、检查清单

### 内容质量

- [ ] 数学内容准确
- [ ] 结果具有足够深度
- [ ] 证明完整严谨
- [ ] 有足够的原创性成分

### 写作质量

- [ ] 结构清晰合理
- [ ] 语言流畅专业
- [ ] 符号使用一致
- [ ] 图表清晰有效

### 学术规范

- [ ] 引用规范完整
- [ ] 无抄袭问题
- [ ] 致谢恰当
- [ ] 格式符合要求

### 答辩准备

- [ ] 陈述清晰有条理
- [ ] 能够回答技术问题
- [ ] 时间控制得当
- [ ] 幻灯片质量高
