# 参考文献管理

## 一、参考文献的重要性

### 1.1 学术诚信
- 承认他人的贡献
- 允许读者追溯来源
- 避免抄袭指控

### 1.2 学术功能
- 定位论文在学术传统中的位置
- 为读者提供进一步阅读资源
- 支持论文的论证

## 二、引用格式：AMS风格

数学领域主要使用AMS（美国数学学会）引用格式。

### 2.1 文内引用

**直接引用**：

```latex
Smith proved that $\pi$ is transcendental [12].

```

**作者-年份格式**（部分期刊使用）：

```latex
Smith (1882) proved that $\pi$ is transcendental.

```

**多文献引用**：

```latex
This result has been studied extensively [3, 7, 12].

```

**连续编号**：

```latex
For more details, see [3, 7-10, 12].

```

### 2.2 参考文献列表格式

**书籍**：

```

[1] J. P. Serre, A Course in Arithmetic, Graduate Texts in Mathematics, 
    vol. 7, Springer-Verlag, New York, 1973.

```

**期刊文章**：

```

[2] A. Wiles, Modular elliptic curves and Fermat's Last Theorem, 
    Ann. of Math. (2) 141 (1995), no. 3, 443-551.

```

**预印本**：

```

[3] T. Tao, The prime number theorem in arithmetic progressions, 
    arXiv:math/0309380 [math.NT], 2003.

```

**会议论文**：

```

[4] L. G. Valiant, A theory of the learnable, Proceedings of the 
    16th Annual ACM Symposium on Theory of Computing (1984), 436-445.

```

**在线资源**：

```

[5] The Stacks Project, https://stacks.math.columbia.edu, 
    accessed April 2026.

```

## 三、BibTeX使用指南

### 3.1 BibTeX文件结构

创建 `references.bib` 文件：

```bibtex
@book{serre1973,
  author    = {Jean-Pierre Serre},
  title     = {A Course in Arithmetic},
  series    = {Graduate Texts in Mathematics},
  volume    = {7},
  publisher = {Springer-Verlag},
  address   = {New York},
  year      = {1973}
}

@article{wiles1995,
  author  = {Andrew Wiles},
  title   = {Modular elliptic curves and {Fermat's Last Theorem}},
  journal = {Annals of Mathematics},
  volume  = {141},
  number  = {3},
  pages   = {443--551},
  year    = {1995}
}

@unpublished{tao2003,
  author      = {Terence Tao},
  title       = {The prime number theorem in arithmetic progressions},
  year        = {2003},
  note        = {arXiv:math/0309380 [math.NT]}
}

@inproceedings{valiant1984,
  author    = {Leslie G. Valiant},
  title     = {A theory of the learnable},
  booktitle = {Proceedings of the 16th Annual ACM Symposium on Theory of Computing},
  year      = {1984},
  pages     = {436--445}
}

@misc{stacks,
  author       = {{The Stacks Project Authors}},
  title        = {The Stacks Project},
  howpublished = {\url{https://stacks.math.columbia.edu}},
  year         = {2026},
  note         = {Accessed: 2026-04-03}
}

```

### 3.2 LaTeX中引用

```latex
\documentclass{article}
\usepackage[numbers]{natbib}  % 或 \usepackage[numbers]{natbib}

\begin{document}

As shown by \cite{serre1973}, ...

This result was proved in \cite{wiles1995}.

\bibliographystyle{amsplain}  % 或 amsalpha, plain, alpha
\bibliography{references}

\end{document}

```

### 3.3 BibTeX条目类型

| 类型 | 用途 |
|------|------|
| @article | 期刊文章 |
| @book | 书籍 |
| @inbook | 书籍中的章节 |
| @incollection | 文集收录 |
| @inproceedings | 会议论文 |
| @unpublished | 未发表论文 |
| @misc | 杂项 |
| @phdthesis | 博士论文 |
| @mastersthesis | 硕士论文 |
| @techreport | 技术报告 |

### 3.4 BibTeX字段说明

```bibtex
@article{key,
  author       = {},  % 作者，格式：Last, First and Last2, First2
  title        = {},  % 标题，专有名词加括号保护
  journal      = {},  % 期刊名
  year         = {},  % 年份
  volume       = {},  % 卷号
  number       = {},  % 期号
  pages        = {},  % 页码范围
  month        = {},  % 月份
  note         = {},  % 备注
  doi          = {},  % DOI
  url          = {},  % URL
  eprint       = {},  % 预印本编号
  eprinttype   = {},  % 预印本类型 (arXiv)
}

```

## 四、Zotero文献管理

### 4.1 Zotero简介

Zotero是免费开源的文献管理工具，适合数学学术写作。

### 4.2 安装与配置

1. 下载安装：[zotero.org](https://www.zotero.org/)
2. 安装浏览器插件（自动捕获文献）
3. 安装Zotero Connector

### 4.3 添加文献

**方式1：浏览器捕获**
- 访问MathSciNet、arXiv、期刊网站
- 点击Zotero图标自动导入

**方式2：手动添加**
- 点击"New Item"按钮
- 选择条目类型
- 填写字段

**方式3：导入BibTeX**
- File → Import
- 选择.bib文件

### 4.4 组织文献

**集合（Collections）**：
- 按主题创建集合
- 论文、参考、工具书分类

**标签（Tags）**：
- 添加颜色标签
- 按阅读状态标记

**搜索功能**：
- 全文搜索
- 高级搜索

### 4.5 与LaTeX集成

**Better BibTeX插件**：
1. 安装插件
2. 设置citekey格式
3. 导出BibTeX文件
4. 设置自动导出

**从Zotero生成BibTeX**：
1. 选中要导出的条目
2. 右键 → Export Selected Items
3. 选择BibTeX格式
4. 保存为.bib文件

### 4.6 在Word中使用

Zotero提供Word插件，可以：
- 插入引用
- 自动生成参考文献列表
- 切换引用格式

## 五、引用最佳实践

### 5.1 何时引用

**必须引用**：
- 直接引用他人原话
- 使用他人定理或结果
- 参考他人方法或证明
- 使用他人数据

**一般不需要引用**：
- 数学公理
- 标准定义
- 众所周知的事实（如勾股定理）
- 你自己的原创结果

### 5.2 引用位置

**首次提及**：

```latex
The Riemann hypothesis, one of the most important open problems 
in mathematics [12], states that...

```

**多次提及**：
- 第一次引用完整信息
- 后续简写："As Smith noted [12]"或"Smith [12]"

### 5.3 引用质量

**优先引用**：
- 原始出处（优先引用原作者）
- 权威来源（知名期刊、出版社）
- 最新研究（除非经典文献）

**谨慎引用**：
- 未审阅的预印本
- 非学术来源（博客、论坛）
- Wikipedia（可作为入门参考，但不应作为主要引用）

### 5.4 避免过度引用

**问题**：过多引用会分散注意力
**原则**：只引用对论证有直接帮助或需要提及的相关工作

## 六、常见问题

### 问题1：同一作者多篇论文

```bibtex
% 使用字母区分
\cite{smith2023a, smith2023b}

```

### 问题2：多位作者

```bibtex
% 标准格式
author = {Smith, John and Jones, Mary and Brown, Bob}

% et al.自动处理

```

### 问题3：arXiv引用

```bibtex
@unpublished{paper,
  author      = {Author Name},
  title       = {Paper Title},
  year        = {2024},
  note        = {arXiv:2401.12345 [math.AG]}
}

```

## 七、检查清单

- [ ] 所有引用都有参考文献条目
- [ ] BibTeX格式正确
- [ ] 作者姓名格式一致
- [ ] 标题中数学符号保护（加花括号）
- [ ] 页码范围使用双横线（--）
- [ ] 期刊名使用标准缩写
- [ ] DOI或URL可访问
- [ ] 引用编号按出现顺序排列
- [ ] 无孤立引用（文内未引用的参考文献）
