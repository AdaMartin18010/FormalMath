# 定理与证明书写

## 1. 定理陈述

### 1.1 定理环境使用

LaTeX标准定理环境：

```latex
\begin{theorem}[定理名称或引用]
  定理陈述内容
\end{theorem}
```

**常用环境类型**：

| 环境 | 用途 | 示例 |
|------|------|------|
| theorem | 主要结果 | 核心定理、主要结论 |
| lemma | 引理 | 为证明定理服务的辅助结果 |
| proposition | 命题 | 次要但独立的结果 |
| corollary | 推论 | 从定理直接得出的结论 |
| definition | 定义 | 概念的定义 |
| example | 例子 | 具体实例 |
| remark | 注记 | 补充说明 |
| conjecture | 猜想 | 未证明的假设 |

### 1.2 定理陈述的结构

标准定理包含三个部分：

```
┌─────────────────────────────────────────┐
│  假设条件 (Hypotheses)                  │
│     ↓                                   │
│  结论 (Conclusion)                      │
│     ↓                                   │
│  名称/引用 (Label) [可选]               │
└─────────────────────────────────────────┘
```

**标准格式**：

```latex
\begin{theorem}[Euclid]\label{thm:euclid}
  There are infinitely many prime numbers.
\end{theorem}
```

### 1.3 复杂定理的陈述

**多条件定理**：

```latex
\begin{theorem}
  Let $G$ be a finite group satisfying the following conditions:
  \begin{enumerate}[(i)]
    \item $G$ is solvable;
    \item Every Sylow subgroup of $G$ is abelian;
    \item $|G|$ is odd.
  \end{enumerate}
  Then $G$ is nilpotent.
\end{theorem}
```

**等价性定理**：

```latex
\begin{theorem}
  For a topological space $X$, the following are equivalent:
  \begin{enumerate}[(a)]
    \item $X$ is compact;
    \item Every open cover of $X$ has a finite subcover;
    \item Every collection of closed sets with the finite intersection property has nonempty intersection.
  \end{enumerate}
\end{theorem}
```

## 2. 证明书写

### 2.1 证明的基本结构

```
证明框架：
  1. 设置和假设回顾
  2. 策略概述（可选但推荐）
  3. 主要论证步骤
  4. 结论验证
```

**示例**：

```latex
\begin{proof}
  We prove the theorem by induction on $n$.

  \textbf{Base case:} When $n = 1$, ...

  \textbf{Inductive step:} Assume the result holds for $n = k$.
  We must show it holds for $n = k+1$.

  [详细论证...]

  This completes the induction, and the theorem is proved.
\end{proof}
```

### 2.2 证明技巧与写法

**反证法**：

```latex
\begin{proof}[Proof by contradiction]
  Suppose, for the sake of contradiction, that the conclusion does not hold.
  Then ... [推导出矛盾]
  This contradiction establishes the theorem.
\end{proof}
```

**构造性证明**：

```latex
\begin{proof}
  We construct the desired object explicitly.
  Define $f(x) = ...$ [具体构造]

  We verify that $f$ satisfies the required properties:
  \begin{enumerate}[(i)]
    \item [验证性质1]
    \item [验证性质2]
  \end{enumerate}

  This completes the construction.
\end{proof}
```

**分类讨论**：

```latex
\begin{proof}
  We consider two cases based on the parity of $n$.

  \textbf{Case 1:} $n$ is even. Then ...

  \textbf{Case 2:} $n$ is odd. Then ...

  In both cases, the conclusion holds.
\end{proof}
```

### 2.3 证明书写原则

**应该做的**：

- ✅ 明确陈述证明策略
- ✅ 每一步都说明理由
- ✅ 使用"因此"、"于是"等连接词
- ✅ 标注关键步骤
- ✅ 检查所有假设都被使用

**不应该做的**：

- ❌ 省略"显然"的步骤（对你显然≠对读者显然）
- ❌ 使用过长的句子
- ❌ 突然引入未定义的符号
- ❌ 证明中包含与结论无关的内容

## 3. 定义书写

### 3.1 定义的结构

```
定义 = 被定义项 + 条件/特征
```

**标准格式**：

```latex
\begin{definition}\label{def:group}
  A \emph{group} is a set $G$ equipped with a binary operation
  $\cdot: G \times G \to G$ satisfying the following axioms:
  \begin{enumerate}[(G1)]
    \item (Associativity) $(ab)c = a(bc)$ for all $a,b,c \in G$;
    \item (Identity) There exists $e \in G$ such that $ea = ae = a$ for all $a \in G$;
    \item (Inverse) For each $a \in G$, there exists $a^{-1} \in G$ such that $aa^{-1} = a^{-1}a = e$.
  \end{enumerate}
\end{definition}
```

### 3.2 定义的最佳实践

**先直观后正式**：

```latex
Roughly speaking, a manifold is a space that locally looks like Euclidean space.
More precisely:

\begin{definition}
  An $n$-dimensional \emph{manifold} is a second countable Hausdorff space $M$
  such that every point has a neighborhood homeomorphic to an open subset of $\mathbb{R}^n$.
\end{definition}
```

**提供例子**：
每个定义后应紧跟例子，帮助理解。

```latex
\begin{example}
  The set of integers $\mathbb{Z}$ with addition forms a group.
  The identity element is $0$, and the inverse of $n$ is $-n$.
\end{example}
```

## 4. 符号与标记规范

### 4.1 定理引用

**首次引用**：

```latex
By Theorem \ref{thm:main}, we have...
```

**重复引用**：

```latex
As mentioned in Theorem \ref{thm:main}...
```

**多定理引用**：

```latex
Theorems \ref{thm:first} and \ref{thm:second} imply...
```

### 4.2 证明结束标记

LaTeX自动在证明结束时添加证毕符号（∎）。

特殊情况：

```latex
\begin{proof}
  [证明在定理陈述中已完成]
\renewcommand{\qedsymbol}{}  % 取消证毕符号
\end{proof}
```

## 5. 常见错误与避免

### 错误1：定理条件不完整

❌ "如果 $f$ 是连续的，那么 $f$ 可导。"
✅ "如果 $f$ 是实值函数且在点 $a$ 处连续，且...，那么 $f$ 在 $a$ 处可导。"

### 错误2：循环论证

确保证明中引用的结果不依赖于待证定理。

### 错误3：混淆充分与必要

- 充分条件：$P \Rightarrow Q$
- 必要条件：$Q \Rightarrow P$
- 充要条件：$P \Leftrightarrow Q$

### 错误4：证明不完整

避免"显然"、"不难看出"等模糊表述，除非确实显然。

- [ ] 所有条件都明确

- [ ] 符号已定义
- [ ] 引用标签正确

### 证明检查

- [ ] 策略明确
- [ ] 每一步有依据
- [ ] 所有假设被使用
- [ ] 结论正确
- [ ] 无循环论证

### 定义检查

- [ ] 被定义项突出
- [ ] 条件完整
- [ ] 有例子说明
- [ ] 与相关概念区分清楚
