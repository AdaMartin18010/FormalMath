# 学术演讲与海报

## 1. 学术演讲

### 1.1 演讲类型

| 类型 | 时长 | 场合 | 目标 |
|------|------|------|------|
| 快速报告 | 5-10分钟 | 研讨会 | 介绍想法 |
| 标准报告 | 20-30分钟 | 学术会议 | 完整结果 |
| 邀请报告 | 45-60分钟 | 重要会议 | 深度阐述 |
| 答辩报告 | 20-30分钟 | 学位答辩 | 展示能力 |
| 大众科普 | 15-20分钟 | 公众场合 | 普及知识 |

### 1.2 演讲准备流程

```
1. 确定核心信息（1句话总结）
        ↓
2. 设计故事线（起承转合）
        ↓
3. 制作幻灯片（视觉辅助）
        ↓
4. 准备讲稿（不是逐字稿）
        ↓
5. 反复演练（计时、调整）
        ↓
6. 准备问答（预判问题）
```

### 1.3 演讲结构设计

**20分钟演讲示例**：

```
  开场 (2分钟)
    └── 问候、题目、自己
    └── 为什么听众应该关心
    
  背景 (4分钟)
    └── 问题背景
    └── 已知结果
    
  主要结果 (8分钟)
    └── 核心定理陈述
    └── 证明策略概述
    └── 关键步骤
    
  应用/例子 (4分钟)
    └── 具体例子
    └── 数值结果（如适用）
    
  结论 (2分钟)
    └── 总结
    └── 未来工作
```

### 1.4 幻灯片设计

**设计原则**：

1. **少即是多**
   - 每页幻灯片一个核心观点
   - 文字不超过6行
   - 留白充足

2. **视觉优先**
   - 使用图表代替文字
   - 颜色简洁（3种以内）
   - 字体清晰（24pt以上）

3. **渐进展示**
   - 逐步揭示复杂内容
   - 避免一次性展示过多信息

**幻灯片模板**：

```latex
\documentclass{beamer}
\usetheme{Madrid}  % 或其他主题
\usecolortheme{default}

\title{演讲标题}
\author{你的名字}
\institute{学校/机构}
\date{日期}

\begin{document}

\begin{frame}
  \titlepage
\end{frame}

\begin{frame}{大纲}
  \tableofcontents
\end{frame}

% 内容幻灯片
\begin{frame}{标题}
  内容...
\end{frame}

\end{document}
```

**数学公式处理**：

```latex
% 避免过长的行内公式
\begin{frame}{定理}
  \begin{theorem}
    陈述定理，使用显示公式
    \[
      E = mc^2
    \]
  \end{theorem}
\end{frame}

% 分步展示
\begin{frame}{证明}
  \begin{itemize}
    \item<1-> 第一步：...
    \item<2-> 第二步：...
    \item<3-> 结论
  \end{itemize}
\end{frame}
```

### 1.5 演讲技巧

**开场技巧**：
- 用问题开场："你们知道...吗？"
- 用例子开场："考虑这个例子..."
- 用故事开场："当高斯还是孩子时..."

**声音控制**：
- 语速适中（每分钟120-150词）
- 重要内容放慢
- 适当停顿

**肢体语言**：
- 面向观众
- 适当走动（不要一直站在电脑旁）
- 手势自然

**眼神交流**：
- 扫视全场
- 与不同区域观众有眼神接触
- 避免只盯着一个方向

### 1.6 问答环节

**准备策略**：
- 预判5-10个可能的问题
- 准备简洁的回答
- 知道哪些内容可以详细展开

**回答技巧**：
- 听清楚问题再回答
- 如果不懂，诚实承认
- 简洁回答，不要过度展开
- 必要时用"这是个好问题"争取思考时间

**困难情况**：
- 尖锐批评：保持冷静，感谢反馈
- 不懂的问题：诚实说"我需要再思考"
- 打断：礼貌地说"我后面会讲到"

## 2. 学术海报

### 2.1 海报尺寸

常见尺寸：
- A0 (841mm × 1189mm)
- 36" × 48" (91cm × 122cm)
- 48" × 36" (122cm × 91cm)

### 2.2 海报结构

```
┌─────────────────────────────────────┐
│           标题                      │
│     作者、机构、联系信息            │
├─────────────┬───────────────────────┤
│             │                       │
│  1. 引言    │   4. 主要结果         │
│             │                       │
├─────────────┤                       │
│             ├───────────────────────┤
│ 2. 预备知识 │                       │
│             │   5. 例子/应用        │
├─────────────┤                       │
│             ├───────────────────────┤
│ 3. 方法     │                       │
│             │   6. 结论与参考文献   │
│             │                       │
└─────────────┴───────────────────────┘
```

### 2.3 海报设计原则

**视觉层次**：
```
标题：72pt 或更大
章节标题：36-48pt
正文：24-32pt
注释：18-20pt
```

**颜色方案**：
- 背景：浅色（白、浅灰）
- 标题：深色（黑、深蓝）
- 强调：一种强调色（红、蓝）

**内容密度**：
- 总字数：500-800字
- 图表：4-8个
- 公式：不超过5个复杂的

### 2.4 海报内容写作

**标题**：
- 准确但吸引人
- 可从3米外看清

**摘要/引言**：
- 3-5句话
- 说明问题和动机

**方法/结果**：
- 使用流程图
- 核心定理框出
- 公式简洁

**结论**：
- 2-3个要点
- 明确主要发现

### 2.5 LaTeX海报模板

```latex
\documentclass[a0paper,portrait]{baposter}

\usepackage{amsmath,amssymb}
\usepackage{graphicx}

\begin{document}

\begin{poster}{
  grid=false,
  columns=2,
  headerheight=0.1\textheight,
  background=plain,
  bgColorOne=white,
}{
  % 标题
  \bf\Huge 海报标题\\[0.3em]
  \large 作者 \quad 机构 \quad 邮箱
}{
  % 左侧列
  \begin{posterbox}{1. 引言}
    内容...
  \end{posterbox}
  
  \begin{posterbox}{2. 预备知识}
    内容...
  \end{posterbox}
  
  \begin{posterbox}{3. 方法}
    内容...
  \end{posterbox}
}{
  % 右侧列
  \begin{posterbox}{4. 主要结果}
    内容...
  \end{posterbox}
  
  \begin{posterbox}{5. 例子}
    内容...
  \end{posterbox}
  
  \begin{posterbox}{6. 结论}
    内容...
  \end{posterbox}
}

\end{poster}
\end{document}
```

### 2.6 海报展示

**准备工作**：
- 准备1分钟介绍
- 准备5分钟详细介绍
- 准备额外材料（论文、名片）

**互动技巧**：
- 主动问候经过的人
- 询问他们的研究兴趣
- 根据听众调整介绍深度

## 3. 在线演讲特别指南

### 3.1 技术准备

**设备检查**：
- 稳定的网络连接
- 良好的麦克风和摄像头
- 备用方案（手机热点）

**软件准备**：
- 提前测试平台（Zoom等）
- 熟悉屏幕共享
- 准备PDF备用

### 3.2 演讲调整

**节奏调整**：
- 在线注意力更短，加快节奏
- 增加互动（投票、提问）
- 更多视觉变化

**幻灯片调整**：
- 更大的字体
- 更简洁的内容
- 避免过细的线条
