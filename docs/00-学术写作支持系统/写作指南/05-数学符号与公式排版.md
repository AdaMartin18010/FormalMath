# 数学符号与公式排版

## 1. 符号使用原则

### 1.1 选择符号的原则

**清晰性优先**：

- 避免与常用符号冲突
- 同一符号不要在不同含义间重用
- 使用标准记号

**系统性原则**：

```
标准对应关系：
  n, m, k  → 整数
  p, q, r  → 素数或有理数
  x, y, z  → 实数或变量
  f, g, h  → 函数
  G, H     → 群
  V, W     → 向量空间
```

**可读性原则**：

- 多字符标识符使用正体：$\mathrm{Hom}$, $\ker$, $\dim$
- 单字符变量使用斜体：$x$, $n$, $G$
- 下标选择有意义的符号：$a_i$ 而非 $a1$, $a2$

### 1.2 符号表建议

每篇论文应在预备知识部分提供符号表：

```latex
\begin{tabular}{ll}
  $\mathbb{N}$ & set of natural numbers \\
  $\mathbb{Z}$ & set of integers \\
  $\mathbb{Q}$ & set of rational numbers \\
  $\mathbb{R}$ & set of real numbers \\
  $\mathbb{C}$ & set of complex numbers \\
  $GL_n(F)$    & general linear group over field $F$
\end{tabular}
```

## 2. 公式排版

### 2.1 行内公式与显示公式

**行内公式**：用于短公式，不破坏文本流

```latex
The quadratic formula $x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$ gives the roots.
```

**显示公式**：用于重要或复杂公式

```latex
The quadratic formula gives the roots:
\begin{equation}\label{eq:quadratic}
  x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}.
\end{equation}
```

**选择指南**：

| 情况 | 建议 |
|------|------|
| 简单分数（如 $1/2$） | 行内 |
| 复杂分数 | 显示 |
| 积分、求和 | 显示 |
| 矩阵 | 显示 |
| 多行等式 | 显示 |

### 2.2 多行公式环境

**align环境**（推荐）：

```latex
\begin{align}
  (a+b)^2 &= a^2 + 2ab + b^2 \\
          &= a^2 + b^2 + 2ab.
\end{align}
```

**多行等号对齐**：

```latex
\begin{align}
  \sum_{i=1}^n i &= 1 + 2 + 3 + \cdots + n \\
                 &= \frac{n(n+1)}{2}.
\end{align}
```

**cases环境**（分段函数）：

```latex
\begin{equation}
  |x| = \begin{cases}
    x  & \text{if } x \geq 0, \\
    -x & \text{if } x < 0.
  \end{cases}
\end{equation}
```

### 2.3 公式编号

**自动编号**：

```latex
\begin{equation}
  E = mc^2
\end{equation}
```

**手动编号**：

```latex
\begin{equation}\tag{3.1}
  E = mc^2
\end{equation}
```

**不编号**：

```latex
\begin{equation*}
  E = mc^2
\end{equation*}
```

或

```latex
\[
  E = mc^2
\]
```

## 4. 字体与样式

### 4.1 数学字体

```latex
\mathbb{R}    % 黑板粗体：ℝ (数集)
\mathcal{C}   % 花体：𝒞 (范畴、曲线)
\mathfrak{g}  % 哥特体：𝔤 (李代数)
\mathrm{Hom}  % 罗马体：Hom (算子)
\mathbf{v}    % 粗体：v (向量)
\mathsf{A}    % 无衬线：𝗔
\mathtt{A}    % 打字机：𝙰
```

### 4.2 上下标位置

**自动调整**：

```latex
\sum_{i=1}^n      % 显示模式下标在下方
\sum\limits_{i=1}^n  % 强制下标在下方
```

**多重上下标**：

```latex
x^{i^j}, x_i^j, x^{ij}, x_{ij}
```

## 5. 公式间距与排版细节

### 5.1 手动间距

| 命令 | 效果 | 用途 |
|------|------|------|
| `\,` | 小间距 | 积分 $dx$ 前 |
| `\:` | 中间距 | 偶尔使用 |
| `\;` | 大间距 | 偶尔使用 |
| `\!` | 负间距 | 调整间距 |
| `\quad` | 1em | 分隔 |
| `\qquad` | 2em | 分隔 |

**积分示例**：

```latex
\int_0^1 f(x)\,dx  % 推荐：dx前有小间距
```

### 5.2 公式中的文本

```latex
\text{这里写文本}  % 需要amsmath宏包
\mathrm{Hom}      % 罗马体
\mbox{文本}        % 旧方法，不推荐
```

**示例**：

```latex
f(x) = \begin{cases}
  1 & \text{if } x \in \mathbb{Q}, \\
  0 & \text{if } x \notin \mathbb{Q}.
\end{cases}
```

## 7. 常见排版错误

### 错误1：积分微分符号未空格

❌ `\int f(x)dx`
✅ `\int f(x)\,dx`

### 错误2：省略号使用错误

❌ `...`
✅ `\ldots` (水平), `\cdots` (垂直居中), `\vdots` (垂直), `\ddots` (对角)

### 错误3：文本在公式中未正确设置

❌ `if x > 0`
✅ `\text{if } x > 0`

### 错误4：多字符函数名未用罗马体

❌ `$sin x$`
✅ `$\sin x$`

### 错误5：行内分数过于复杂

❌ `$\frac{\sum_{i=1}^n a_i}{\sum_{j=1}^m b_j}$`
✅ 使用显示公式或简化
