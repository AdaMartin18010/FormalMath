# 数学英语常用表达

## 1. 逻辑连接词

### 1.1 递进关系

| 英文表达 | 中文含义 | 使用场景 |
|---------|---------|---------|
| Furthermore | 此外 | 添加相关信息 |
| Moreover | 而且 | 强调额外要点 |
| In addition | 另外 | 补充内容 |
| Besides | 除此之外 | 列举更多 |
| What is more | 更有甚者 | 强调重要性 |
| Equally important | 同样重要的是 | 并列重要观点 |

**示例**：
```
The function is continuous. Furthermore, it is differentiable 
at all points except x = 0.
```

### 1.2 转折关系

| 英文表达 | 中文含义 | 使用场景 |
|---------|---------|---------|
| However | 然而 | 一般转折 |
| Nevertheless | 尽管如此 | 强调让步 |
| Nonetheless | 尽管如此 | 正式场合 |
| On the other hand | 另一方面 | 对比不同角度 |
| Conversely | 相反地 | 反向论证 |
| In contrast | 相比之下 | 对比差异 |
| Although | 虽然 | 引导让步从句 |
| Even though | 即使 | 强调让步 |

**示例**：
```
The proof is elementary. Nevertheless, it requires careful 
attention to detail.
```

### 1.3 因果关系

| 英文表达 | 中文含义 | 使用场景 |
|---------|---------|---------|
| Therefore | 因此 | 正式结论 |
| Thus | 于是 | 推导结果 |
| Consequently | 结果是 | 强调后果 |
| Hence | 因此 | 简洁结论 |
| As a result | 结果 | 说明结果 |
| It follows that | 由此可得 | 推导陈述 |
| This implies that | 这意味着 | 逻辑推导 |
| Because of this | 因此 | 说明原因 |

**示例**：
```
The sequence is bounded and monotonic. Therefore, it converges.
```

### 1.4 举例说明

| 英文表达 | 中文含义 | 使用场景 |
|---------|---------|---------|
| For example | 例如 | 一般举例 |
| For instance | 比如 | 正式举例 |
| Namely | 即 | 具体说明 |
| Specifically | 具体而言 | 详细说明 |
| In particular | 特别是 | 强调特例 |
| To illustrate | 为了说明 | 引入例子 |
| As an example | 作为例子 | 举例引入 |

**示例**：
```
Many sequences converge to zero; for example, a_n = 1/n.
```

## 2. 数学陈述常用句型

### 2.1 定义

**标准定义句型**：
```
Definition 1. An X is called a Y if ...
定义1. 如果一个X满足...，则称之为Y。

We say that X is Y when ...
我们说X是Y，当...时。

By a/an X, we mean ...
所谓X，我们指的是...

Recall that X is defined as ...
回顾：X定义为...
```

**示例**：
```
Definition 2.1. A group G is called abelian if ab = ba for all 
a, b in G.
```

### 2.2 定理陈述

**标准定理句型**：
```
Theorem 1. Let X satisfy ... Then Y holds.
定理1. 设X满足...，则Y成立。

Theorem 2. Suppose that ... Then ...
定理2. 假设...，则...

Theorem 3. If X, then Y.
定理3. 如果X，则Y。

Theorem 4. For any X satisfying ..., we have Y.
定理4. 对任何满足...的X，我们有Y。
```

**条件强化**：
```
Under the hypothesis that ..., we have ...
在...假设下，我们有...

Assuming ..., it follows that ...
假设...，可得...

Provided that ..., we obtain ...
倘若...，我们得到...
```

### 2.3 证明引导

**证明开始**：
```
Proof. We prove the theorem by induction on n.
证明. 我们对n用归纳法证明该定理。

The proof proceeds in three steps.
证明分三步进行。

We argue by contradiction.
我们用反证法论证。

The proof is divided into two cases.
证明分两种情况。
```

**证明步骤**：
```
First, we show that ...
首先，我们证明...

Next, we prove that ...
接下来，我们证明...

It remains to show that ...
还需证明...

To complete the proof, we verify that ...
为完成证明，我们验证...
```

**证明结论**：
```
This completes the proof.
证明完成。

The theorem is proved.
定理得证。

Thus the claim follows.
因此结论成立。

This establishes the desired result.
这建立了所需的结果。
```

### 2.4 例子引入

```
Example 1. Consider the function f(x) = x^2.
例子1. 考虑函数f(x) = x²。

As an illustration, take X = ...
作为说明，取X = ...

To see this, let us consider ...
为说明这一点，让我们考虑...
```

## 3. 数学关系表达

### 3.1 等式与不等式

**陈述等式**：
```
We have A = B.
我们有A = B。

It follows that A = B.
可得A = B。

This yields A = B.
这给出A = B。

A can be written as B.
A可以写成B。

Expanding A, we obtain B.
展开A，我们得到B。
```

**陈述不等式**：
```
We obtain the estimate ...
我们得到估计...

This gives the bound ...
这给出界...

By (2.3), we have the inequality ...
由(2.3)，我们有不等式...

A is bounded above by B.
A被B上界。

A is bounded below by B.
A被B下界。
```

### 3.2 收敛与极限

```
The sequence converges to L.
该序列收敛于L。

The limit exists and equals L.
极限存在且等于L。

As n tends to infinity, a_n approaches L.
当n趋于无穷时，a_n趋近L。

Taking the limit as x → 0, we get ...
令x → 0取极限，我们得到...
```

### 3.3 存在与唯一性

**存在性**：
```
There exists an element x such that ...
存在元素x使得...

For every ε > 0, there is a δ > 0 such that ...
对任意ε > 0，存在δ > 0使得...

Such an element exists.
这样的元素存在。
```

**唯一性**：
```
This element is unique.
该元素是唯一的。

There is exactly one such element.
恰好存在这样一个元素。

Uniqueness follows from ...
唯一性由...得出。
```

## 4. 论文结构用语

### 4.1 章节引导

**引言**：
```
This paper is organized as follows.
本文结构如下。

In Section 2, we review ...
在第2节，我们回顾...

Section 3 contains our main results.
第3节包含我们的主要结果。

The proof of Theorem 1 is given in Section 4.
定理1的证明在第4节给出。

We conclude with some open questions in Section 5.
我们在第5节以一些开放问题结束。
```

**章节过渡**：
```
Having established ..., we now turn to ...
在建立了...之后，我们现在转向...

Before proving the main theorem, we need some preliminary results.
在证明主要定理之前，我们需要一些预备结果。

With these preparations, we can now state our main theorem.
有了这些准备，我们现在可以陈述主要定理。
```

### 4.2 文献引用

```
This problem was first studied by Smith [12].
这个问题最早由Smith [12]研究。

The method we use goes back to [3].
我们使用的方法可追溯到[3]。

A related result was obtained by Jones [5, Theorem 2].
Jones [5, 定理2]得到了一个相关结果。

For a comprehensive treatment, see [7] and references therein.
综合处理见[7]及其参考文献。

We refer the reader to [4] for more details.
更多细节我们请读者参考[4]。
```
