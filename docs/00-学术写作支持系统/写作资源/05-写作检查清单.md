# 写作检查清单

## 一、内容检查

### 1.1 数学准确性

- [ ] 所有定理陈述准确完整
- [ ] 所有证明逻辑严密，无漏洞
- [ ] 所有计算经过验证
- [ ] 符号使用一致，无混淆
- [ ] 引用结果准确（定理编号、页码）
- [ ] 例子正确，能说明问题

### 1.2 内容完整性

- [ ] 引言明确研究问题
- [ ] 背景知识充分但不冗余
- [ ] 主要结果完整陈述
- [ ] 证明详细程度适当
- [ ] 结论总结到位
- [ ] 未来方向明确

### 1.3 原创性与贡献

- [ ] 论文贡献清晰说明
- [ ] 与已有工作明确区分
- [ ] 新结果充分强调
- [ ] 适当引用相关工作

## 二、结构检查

### 2.1 整体结构

- [ ] 结构符合数学论文惯例
- [ ] 各章节比例适当
- [ ] 章节标题清晰准确
- [ ] 章节过渡自然流畅

### 2.2 段落组织

- [ ] 每段有明确的主题句
- [ ] 段落长度适中（3-8行）
- [ ] 段落间有逻辑联系
- [ ] 无孤立段落

### 2.3 论文流程

- [ ] 从一般到具体的进展
- [ ] 论证环环相扣
- [ ] 前后呼应
- [ ] 结论与引言对应

## 三、写作风格检查

### 3.1 语言清晰性

- [ ] 句子简洁明了
- [ ] 无冗长或歧义表达
- [ ] 专业术语使用恰当
- [ ] 数学英语规范

### 3.2 一致性

- [ ] 术语使用一致
- [ ] 风格统一
- [ ] 时态一致（通常现在时）
- [ ] 人称一致（推荐第三人称）

### 3.3 可读性

- [ ] 目标读者能理解
- [ ] 难度渐进
- [ ] 复杂概念有解释
- [ ] 适当使用例子

## 四、数学表达检查

### 4.1 符号系统

- [ ] 符号表完整
- [ ] 符号选择符合惯例
- [ ] 新符号有明确定义
- [ ] 无符号冲突

### 4.2 公式排版

- [ ] 行内公式不过于复杂
- [ ] 重要公式使用显示模式
- [ ] 公式编号系统一致
- [ ] 引用公式正确

### 4.3 定理环境

- [ ] 定理编号连续
- [ ] 定理类型使用恰当
- [ ] 证明结束标记正确
- [ ] 标签命名规范

## 五、图表检查

### 5.1 图形

- [ ] 图形清晰可读
- [ ] 坐标轴标注完整
- [ ] 图例清晰
- [ ] 标题准确
- [ ] 图形引用正确

### 5.2 表格

- [ ] 表格格式规范
- [ ] 标题完整
- [ ] 单位标注清楚
- [ ] 表格引用正确

## 六、参考文献检查

### 6.1 引用规范

- [ ] 所有引用都有参考文献条目
- [ ] 引用格式一致
- [ ] 无孤立引用
- [ ] 无未引用的参考文献

### 6.2 引用质量

- [ ] 引用原始出处
- [ ] 引用权威来源
- [ ] 引用最新文献（如适用）
- [ ] 引用格式正确

### 6.3 BibTeX

- [ ] 作者姓名格式一致
- [ ] 标题中数学符号保护
- [ ] 期刊名缩写规范
- [ ] 页码范围使用双横线

## 七、LaTeX技术检查

### 7.1 编译

- [ ] 文档能无错误编译
- [ ] 无警告（或警告已处理）
- [ ] 交叉引用正确
- [ ] 目录生成正确

### 7.2 排版

- [ ] 页面边距正确
- [ ] 字体大小适当
- [ ] 行间距合适
- [ ] 页眉页脚正确

### 7.3 格式细节

- [ ] 标点符号使用正确
- [ ] 括号匹配
- [ ] 引号使用正确
- [ ] 破折号使用正确

## 八、最终审查

### 8.1 细节检查

- [ ] 拼写检查通过
- [ ] 无打字错误
- [ ] 页码正确
- [ ] 日期正确

### 8.2 格式符合要求

- [ ] 符合学校/期刊格式要求
- [ ] 页数符合要求
- [ ] 摘要长度符合要求
- [ ] 参考文献数量适当

### 8.3 提交准备

- [ ] 最终PDF生成
- [ ] 源文件备份
- [ ] 所有附属文件齐全
- [ ] 提交信息完整

## 九、分级检查表

### 基础级（必须）
- [ ] 无数学错误
- [ ] 无语法错误
- [ ] 结构完整
- [ ] 能编译通过

### 良好级（推荐）
- [ ] 写作清晰流畅
- [ ] 符号使用规范
- [ ] 引用完整规范
- [ ] 图表质量高

### 优秀级（追求）
- [ ] 有独特视角
- [ ] 组织巧妙
- [ ] 表达精炼
- [ ] 细节完美

## 十、提交前最终确认

### 自我审查后
- [ ] 放置24小时以上再读
- [ ] 大声朗读检查流畅性
- [ ] 反向阅读检查错误

### 同行评议
- [ ] 请同学/同事审阅
- [ ] 收集反馈意见
- [ ] 根据反馈修改

### 导师审核
- [ ] 提交导师审阅
- [ ] 处理导师意见
- [ ] 获得导师批准
