# 数学符号大全

## 1. 数集与代数结构

### 1.1 标准数集

| 符号 | LaTeX | 含义 | 说明 |
|------|-------|------|------|
| $\mathbb{N}$ | `\mathbb{N}` | 自然数集 | $\{0, 1, 2, ...\}$ 或 $\{1, 2, 3, ...\}$ |
| $\mathbb{Z}$ | `\mathbb{Z}` | 整数集 | $\mathbb{Z}^+$: 正整数，$\mathbb{Z}^-$: 负整数 |
| $\mathbb{Q}$ | `\mathbb{Q}` | 有理数集 | 需要`\mathbb` |
| $\mathbb{R}$ | `\mathbb{R}` | 实数集 | $\mathbb{R}^+$: 正实数 |
| $\mathbb{C}$ | `\mathbb{C}` | 复数集 | $a + bi$，$a,b \in \mathbb{R}$ |
| $\mathbb{H}$ | `\mathbb{H}` | 四元数集 | Hamilton四元数 |
| $\mathbb{F}_q$ | `\mathbb{F}_q` | 有限域 | $q = p^n$，$p$为素数 |
| $\mathbb{Z}_n$ | `\mathbb{Z}_n` | 模n整数环 | 或记为$\mathbb{Z}/n\mathbb{Z}$ |
| $\mathbb{K}$ | `\mathbb{K}` | 一般域 | 通常在域上讨论时使用 |
