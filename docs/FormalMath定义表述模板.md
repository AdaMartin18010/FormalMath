# FormalMath定义表述模板

## 标准化数学定义表述格式指南

---

## 📋 模板概述

本模板为FormalMath项目的数学定义表述提供统一、准确、标准化的格式指导。所有定义都遵循国际数学标准，确保在项目中的一致使用。

**模板原则**：

- **准确性**：定义表述准确无误
- **一致性**：定义格式保持一致
- **完整性**：定义包含必要的信息
- **国际化**：符合国际数学标准
