# 逻辑学推理判断树

## 概述

本文档构建逻辑学的完整推理链条，涵盖命题逻辑、谓词逻辑、集合论三大核心领域，共约45个核心定理。

## 一、命题逻辑推理链

### 1.1 语法理论

**公理体系（Hilbert系统）**：
```
公理模式
├── A1: φ → (ψ → φ)
├── A2: (φ → (ψ → χ)) → ((φ → ψ) → (φ → χ))
└── A3: (¬φ → ¬ψ) → (ψ → φ)

推理规则
└── MP (分离规则): 从 φ → ψ 和 φ 推出 ψ
```

**定理 PL1：演绎定理**
- **陈述**：$\Gamma \cup \{\varphi\} \vdash \psi$ $\iff$ $\Gamma \vdash \varphi \to \psi$
- **证明思路**：对推演长度归纳
- **父节点**：Hilbert公理系统
- **子节点**：可证性判定
- **判断逻辑**：**将推演转化为蕴含，简化证明构造**

**定理 PL2：合式公式性质**
- **唯一可读性**：每个合式公式有唯一语法树
- **归纳原理**：对公式结构归纳
- **父节点**：归纳定义
- **子节点**：语法分析、语义解释

### 1.2 语义理论

**定理 PL3：真值赋值**
- **定义**：$v: \text{Prop} \to \{0, 1\}$ 扩展到所有公式
- **真值表方法**：有限公式的真值可计算
- **父节点**：命题连接词语义
- **子节点**：重言式判定

**定理 PL4：重言式与可满足性**
- **定义**：
  - **重言式（永真式）**：所有赋值下为真
  - **可满足式**：存在赋值为真
  - **矛盾式**：所有赋值下为假
- **关系**：$\varphi$ 重言式 $\iff$ $\neg\varphi$ 不可满足
- **父节点**：真值赋值（PL3）

**定理 PL5：紧致性定理（命题逻辑）**
- **陈述**：公式集 $\Gamma$ 可满足 $\iff$ 每个有限子集可满足
- **证明方法**：
  1. 拓扑法（乘积拓扑的紧致性）
  2. 模型论构造
- **父节点**：语义理论、拓扑紧致性
- **应用**：无穷情形的可满足性

### 1.3 完备性定理

**定理 PL6：可靠性定理**
- **陈述**：$\vdash \varphi$ $\Rightarrow$ $\vDash \varphi$（可证则永真）
- **证明思路**：公理永真，MP保持永真性
- **父节点**：公理验证、语义定义
- **子节点**：完备性定理

**定理 PL7：完备性定理（命题逻辑）**
- **陈述**：$\vDash \varphi$ $\Rightarrow$ $\vdash \varphi$（永真则可证）
- **证明方法**：
  1. **Lindenbaum代数法**：构造极大一致集
  2. **语义表法**：反证法构造证明
- **父节点**：可靠性定理（PL6）、Lindenbaum引理
- **意义**：**语法与语义的等价，公理系统的充分性**

**定理 PL8：可判定性**
- **陈述**：命题逻辑是可判定的（真值表法）
- **复杂度**：co-NP完全
- **父节点**：真值表方法
- **判断逻辑**：**命题逻辑在计算上是可处理的**

### 1.4 自然演绎与相继演算

**定理 PL9：自然演绎系统**
- **特点**：引入/消去规则对应直觉
- **例子**：
  - $\to$引入：从假设 $\varphi$ 推出 $\psi$，得 $\varphi \to \psi$
  - $\to$消去（MP）：从 $\varphi \to \psi$ 和 $\varphi$ 得 $\psi$
- **与Hilbert系统等价**

**定理 PL10：相继演算（Gentzen系统）**
- **相继式**：$\Gamma \Rightarrow \Delta$（假设推出结论）
- **结构规则**：弱化、收缩、交换、切割
- **切割消去定理**：任何证明可转化为切割自由证明
- **子公式性质**：切割自由证明中只出现子公式
- **父节点**：自然演绎
- **应用**：自动化证明、一致性证明

## 二、谓词逻辑推理链

### 2.1 语法理论

**定理 FOL1：项与合式公式**
- **项的形成**：变元、常元、函数符号作用
- **公式的形成**：原子公式 + 连接词 + 量词
- **自由变元与约束变元**
- **父节点**：一阶语言定义
- **子节点**：替换理论、语义解释

**定理 FOL2：替换定理**
- **陈述**：项 $t$ 对公式 $\varphi(x)$ 中 $x$ 可自由替换，则
  $$\vDash \forall x \varphi(x) \to \varphi(t)$$
- **边界条件**：避免变元捕获
- **父节点**：语法定义
- **子节点**：量词规则

### 2.2 语义理论

**定理 FOL3：结构与满足**
- **结构**：$\mathcal{M} = (M, \{c^\mathcal{M}\}, \{f^\mathcal{M}\}, \{R^\mathcal{M}\})$
- **赋值**：$s: \text{Var} \to M$
- **满足关系**：$\mathcal{M} \vDash \varphi[s]$
- **父节点**：集合论、语义定义
- **子节点**：逻辑蕴涵、有效性

**定理 FOL4：量词语义**
- **全称量词**：$\mathcal{M} \vDash \forall x \varphi[s]$ $\iff$ 对所有 $a \in M$，$\mathcal{M} \vDash \varphi[s(x|a)]$
- **存在量词**：$\mathcal{M} \vDash \exists x \varphi[s]$ $\iff$ 存在 $a \in M$，$\mathcal{M} \vDash \varphi[s(x|a)]$
- **对偶性**：$\forall x \varphi \equiv \neg \exists x \neg \varphi$

**定理 FOL5：有效性与可满足性**
- **逻辑有效**：所有结构满足（$\vDash \varphi$）
- **可满足**：某结构某赋值满足
- **关系**：$\varphi$ 有效 $\iff$ $\neg\varphi$ 不可满足
- **父节点**：满足关系（FOL4）

### 2.3 完备性定理

**定理 FOL6：可靠性定理**
- **陈述**：$\vdash \varphi$ $\Rightarrow$ $\vDash \varphi$
- **证明思路**：公理有效，推理规则保有效性
- **父节点**：语义定义、公理验证
- **子节点**：Gödel完备性定理

**定理 FOL7：Gödel完备性定理（1930）**
- **陈述**：$\vDash \varphi$ $\Rightarrow$ $\vdash \varphi$（一阶逻辑中有效公式可证）
- **等价形式**：一致集可满足
- **证明要点**：
  1. 将一致集扩张为极大一致集
  2. 构造Henkin模型（加入常量作为见证）
  3. 商结构得到模型
- **父节点**：可靠性定理（FOL6）、Lindenbaum引理
- **意义**：**一阶逻辑的语法与语义完全对应**

**定理 FOL8：紧致性定理（一阶逻辑）**
- **陈述**：公式集 $\Gamma$ 有模型 $\iff$ 每个有限子集有模型
- **证明思路**：完备性定理 + 证明的有限性
- **父节点**：Gödel完备性定理（FOL7）
- **应用**：
  - 非标准模型的存在性
  - 无穷结构的刻画

**定理 FOL9：Löwenheim-Skolem定理**
- **向下形式**：若 $\Gamma$ 有无穷模型，则有任意无穷基数的模型
- **向上形式**：若 $\Gamma$ 有无穷模型，则有任意大基数的模型
- **父节点**：紧致性定理（FOL8）
- **意义**：**一阶逻辑不能刻画无穷基数**
- **应用**：非标准分析、Skolem悖论

### 2.4 不可判定性与不完备性

**定理 FOL10：Church定理**
- **陈述**：一阶逻辑是不可判定的（有效性不可判定）
- **证明思路**：通过停机问题或Post对应问题归约
- **父节点**：可计算性理论
- **意义**：**与命题逻辑的本质区别**

**定理 FOL11：Gödel第一不完备性定理（1931）**
- **陈述**：包含基本算术的一致递归公理系统不完全
- **证明要点**：
  1. Gödel编码：将语法对象编码为自然数
  2. 自指语句：构造"我不可证"语句 $G$
  3. 若 $T \vdash G$，则 $T$ 不一致
  4. 若 $T \vdash \neg G$，则 $T \vdash G$，矛盾
- **父节点**：递归函数、可表示性定理
- **意义**：**任何形式系统都无法完全捕捉算术真理**

**定理 FOL12：Gödel第二不完备性定理**
- **陈述**：一致系统 $T$ 不能证明自身的一致性
- **形式化**：$T \not\vdash \text{Con}(T)$
- **父节点**：第一不完备性定理（FOL11）
- **意义**：**希尔伯特计划的本质障碍**

## 三、集合论推理链

### 3.1 ZFC公理系统

**公理列表**：
```
ZFC公理系统
├── 外延公理：集合由元素唯一确定
├── 空集公理：存在空集
├── 对集公理：{a, b} 是集合
├── 并集公理：∪a 是集合
├── 幂集公理：P(a) 是集合
├── 无穷公理：存在无穷集
├── 分离公理模式：{x∈a : φ(x)} 是集合
├── 替换公理模式：函数像保持集合性
├── 正则公理：无无穷降链
└── 选择公理（AC）：非空集族的乘积非空
```

**定理 ST1：基本集合运算**
- **并、交、补、差**：良定义性
- **幂集性质**：$|P(A)| = 2^{|A|}$（$A$ 有限时）
- **父节点**：外延公理、其他构造公理

**定理 ST2：自然数构造**
- **von Neumann构造**：
  - $0 = \emptyset$
  - $n + 1 = n \cup \{n\} = \{0, 1, \ldots, n\}$
- **归纳原理**：分离公理 + 无穷公理
- **父节点**：无穷公理、外延公理
- **子节点**：算术运算定义

### 3.2 序数与基数理论

**定理 ST3：良序与序数**
- **良序集**：全序 + 每个非空子集有最小元
- **序数**：传递集 + $\in$ 良序
- **性质**：
  - 序数类是良序类
  - 每个良序集同构于唯一序数
- **父节点**：正则公理、替换公理
- **子节点**：超限归纳、基数定义

**定理 ST4：超限归纳与递归**
- **超限归纳**：若 $P$ 对所有小于 $\alpha$ 的序数成立则 $P(\alpha)$ 成立，则 $P$ 对所有序数成立
- **超限递归**：可在序数上递归定义函数
- **父节点**：序数良序性（ST3）
- **应用**：序数算术、层级构造

**定理 ST5：基数理论**
- **定义**：$|A| = $ 与 $A$ 等势的最小序数
- **Cantor定理**：$|A| < |P(A)|$
- **连续统假设**：$2^{\aleph_0} = \aleph_1$？
- **父节点**：序数理论、等势关系
- **子节点**：算术公理、大基数

### 3.3 选择公理及其等价形式

**定理 ST6：选择公理等价定理群**

以下命题等价于AC：
1. **Zorn引理**：偏序集每个链有上界则有极大元
2. **良序定理**：每个集合可良序化
3. **Tukey引理**：有限特征的集族有极大元
4. **Hausdorff极大原理**：偏序集有极大链
5. **向量空间基存在性**：每个向量空间有基
6. **Tychonoff定理**：紧致空间的乘积紧致
7. **Zermelo定理**：基数可比较

**定理 ST7：Zorn引理**
- **陈述**：$(P, \leq)$ 偏序，若每个链有上界，则 $P$ 有极大元
- **证明思路**：假设无极大元，用选择函数构造严格递增链，矛盾
- **父节点**：选择公理
- **应用**：极大理想存在、基扩张、Hahn-Banach定理

**定理 ST8：良序定理**
- **陈述**：每个集合存在良序
- **证明思路**：用选择函数递归构造良序
- **父节点**：选择公理、超限递归
- **应用**：基数定义、归纳论证

### 3.4 连续统假设与大基数

**定理 ST9：连续统假设的独立性**
- **Gödel（1938）**：CH与ZFC一致（可构造模型 $L$）
- **Cohen（1963）**：$\neg$CH与ZFC一致（力迫法）
- **结论**：CH在ZFC中不可判定
- **父节点**：模型论、力迫法
- **意义**：**集合论公理的不完备性实例**

**定理 ST10：大基数层次**
```
大基数层次（强度递增）
├── 不可达基数
├── Mahlo基数
├── 弱紧致基数
├── 可测基数
├── 强紧致基数
├── 超紧致基数
├── 巨大基数
└── I0（最高层）
```
- **性质**：
  - 大基数的存在性超过ZFC证明能力
  - 作为额外公理研究
  - 决定性公理（AD）与选择公理矛盾
- **父节点**：基数理论、可定义性
- **应用**：描述集合论、内模型理论

### 3.5 模型论基础

**定理 ST11：初等等价与同构**
- **初等等价**：$\mathcal{M} \equiv \mathcal{N}$（满足相同句子）
- **初等嵌入**：保持所有公式
- **关系**：同构 $\Rightarrow$ 初等等价（逆不成立）
- **父节点**：语义定义

**定理 ST12：超积构造**
- **超滤**：极大真滤子
- **超积**：$\prod_{i \in I} \mathcal{M}_i / \mathcal{U}$
- **Łoś定理**：$\prod \mathcal{M}_i / \mathcal{U} \vDash \varphi$ $\iff$ $\{i : \mathcal{M}_i \vDash \varphi\} \in \mathcal{U}$
- **父节点**：滤子理论、商构造
- **应用**：紧致性定理证明、非标准分析

### 4.3 关键判断逻辑

#### 证明策略选择树

```
待证命题分析
├── 命题逻辑？
│   ├── 永真式？ → 真值表/语义表
│   ├── 可满足？ → 找模型
│   └── 证明构造？ → 自然演绎/Hilbert系统
├── 谓词逻辑？
│   ├── 有效性？ → 完备性定理保证可证
│   ├── 可满足？ → 模型构造
│   └── 算术命题？ → Gödel不完备性注意
└── 集合论？
    ├── 存在性？ → 考虑AC/Zorn引理
    ├── 基数比较？ → 良序定理
    └── 一致性？ → 相对一致性证明
```

#### 逻辑系统选择策略

```
应用场景分析
├── 有限情形、计算效率重要？
│   └── 命题逻辑
├── 数学定理形式化？
│   └── 一阶谓词逻辑
│       ├── 代数结构 → 等式逻辑
│       ├── 序结构 → 带序的逻辑
│       └── 算术性质 → 皮亚诺算术
├── 高阶性质（如完备性、紧性）？
│   └── 高阶逻辑（注意不完备性）
├── 集合论基础？
│   └── ZFC（标准）
│       ├── 需要确定性？ → ZF + AD（描述集合论）
│       └── 大基数性质？ → ZFC + 大基数公理
└── 构造主义要求？
    └── 直觉主义逻辑（Heyting算术）
```
