# 几何学推理判断树

## 概述

本文档构建几何学的完整推理链条，涵盖欧氏几何、非欧几何、微分几何、代数几何四大核心领域，共约50个核心定理。

---

## 一、欧氏几何推理链

### 1.1 欧几里得公理体系

**公理体系（希尔伯特公理化）**：

```

希尔伯特公理组
├── 关联公理（8条）：点线面关联关系
├── 顺序公理（4条）：介于关系
├── 合同公理（5条）：线段、角的合同
├── 平行公理（1条）：过线外一点有唯一平行线
└── 连续公理（2条）：Archimedes公理、完备性公理

```

**定理 E1：三角形基本定理**

**定理 E1-1：三角形内角和**
- **陈述**：三角形三内角和等于 $\pi$
- **证明思路**：过顶点作平行线，利用平行线性质
- **依赖公理**：平行公理
- **父节点**：平行公理
- **子节点**：多边形内角和、相似三角形理论
- **等价形式**：平行公理 $\Leftrightarrow$ 三角形内角和 $= \pi$

**定理 E1-2：全等三角形判定**
- **SAS**：两边及其夹角对应相等
- **ASA**：两角及其夹边对应相等
- **SSS**：三边对应相等
- **证明思路**：利用合同公理、叠合论证
- **父节点**：合同公理
- **子节点**：等腰三角形性质、垂直平分线性质

**定理 E1-3：相似三角形判定**
- **AA**：两角对应相等
- **SAS**：两边成比例且夹角相等
- **SSS**：三边成比例
- **父节点**：平行线分线段成比例定理
- **子节点**：勾股定理的证明、三角函数定义

**定理 E2：勾股定理**
- **陈述**：直角三角形中，$a^2 + b^2 = c^2$
- **证明方法**：
  1. 相似三角形法（欧几里得《原本》）
  2. 面积割补法（赵爽弦图）
  3. 代数法（现代坐标法）
- **父节点**：相似三角形（E1-3）
- **子节点**：距离公式、三角恒等式
- **逆定理**：$a^2 + b^2 = c^2$ 则三角形为直角三角形

**定理 E3：圆的基本性质**
- **圆周角定理**：同弧所对圆周角相等，等于圆心角的一半
- **弦切角定理**：弦切角等于其所夹弧对的圆周角
- **幂定理**：点对圆的幂相等
- **父节点**：等腰三角形、角度关系
- **应用**：圆内接四边形、切线性质

### 1.2 平行公理等价形式

**定理 E4：平行公理等价定理群**

以下命题与平行公理等价：
1. **Playfair公理**：过线外一点有唯一平行线
2. **三角形内角和** $= \pi$
3. **相似三角形存在**：存在不全等但相似的三角形
4. **勾股定理成立**
5. **平行线等距**：平行线间距离相等
6. **存在矩形**

**判断逻辑**：**任一欧氏几何定理的证明若依赖上述任一形式，则实质依赖平行公理**

## 二、非欧几何推理链

### 2.1 双曲几何（负曲率）

**公理差异**：用双曲平行公设替代欧氏平行公理
- **双曲平行公设**：过线外一点至少有两条平行线（不相交直线）

**定理 H1：双曲几何基本定理群**

**定理 H1-1：三角形内角和**
- **陈述**：双曲三角形内角和 $< \pi$
- **角度缺陷**：$\delta = \pi - (\alpha + \beta + \gamma) > 0$
- **面积关系**：面积与角度缺陷成正比（Gauss-Bonnet）
- **父节点**：双曲平行公设
- **等价形式**：平行公设 $\Leftrightarrow$ 存在内角和 $\neq \pi$ 的三角形

**定理 H1-2：平行线性质**
- **陈述**：双曲平面上，过线外一点有无穷多条平行线
- **极限平行线**：两条"最极端"的平行线（渐近平行）
- **父节点**：双曲公设
- **与欧氏差异**：平行线不保持等距，有公共垂线

**定理 H1-3：相似与全等**
- **陈述**：双曲几何中，相似必全等
- **推论**：不存在任意大小的相似图形
- **证明思路**：由角度决定三角形，AAA即全等

### 2.2 椭圆几何（正曲率）

**公理差异**：无平行线（任意两直线相交）

**定理 H2：椭圆几何基本性质**

**定理 H2-1：直线性质**
- **陈述**：任意两条直线必相交
- **模型**：球面几何（将球面对径点等同）
- **与欧氏差异**：直线（大圆）有有限长度，封闭

**定理 H2-2：三角形内角和**
- **陈述**：椭圆三角形内角和 $> \pi$
- **角度盈余**：$\varepsilon = (\alpha + \beta + \gamma) - \pi > 0$
- **面积关系**：面积与角度盈余成正比

### 2.3 统一视角：Gauss-Bonnet定理

**定理 H3：Gauss-Bonnet定理（常曲率情形）**

**定理 H3-1：测地三角形**
- **陈述**：常曲率 $K$ 的曲面上，测地三角形
  $$\alpha + \beta + \gamma = \pi + K \cdot \text{Area}$$
- **三种几何对应**：
  - $K = 0$：欧氏几何，内角和 $= \pi$
  - $K < 0$：双曲几何，内角和 $< \pi$
  - $K > 0$：椭圆几何，内角和 $> \pi$
- **父节点**：测地线理论、曲率
- **判断逻辑**：**曲率是区分三种几何的本质量**

## 三、微分几何推理链

### 3.1 曲线论

**定理 DG1：Frenet-Serret公式**
- **设定**：$\gamma(s)$ 是弧长参数化的正则曲线
- **Frenet标架**：$(T, N, B)$，切向量、主法向量、副法向量
- **公式**：
  $$\frac{d}{ds}\begin{pmatrix} T \\ N \\ B \end{pmatrix} = \begin{pmatrix} 0 & \kappa & 0 \\ -\kappa & 0 & \tau \\ 0 & -\tau & 0 \end{pmatrix} \begin{pmatrix} T \\ N \\ B \end{pmatrix}$$
- **曲率** $\kappa$：切向量转动速率
- **挠率** $\tau$：副法向量转动速率
- **父节点**：弧长参数化、正交标架
- **子节点**：曲线分类定理

**定理 DG2：曲线基本定理**
- **存在性**：给定连续函数 $\kappa(s) > 0$ 和 $\tau(s)$，存在以之为曲率、挠率的曲线
- **唯一性**：相差一个刚体运动（平移+旋转）
- **父节点**：Frenet-Serret公式（DG1）
- **判断逻辑**：**曲率和挠率完全决定曲线形状**

### 3.2 曲面论

**定理 DG3：第一基本形式**
- **陈述**：曲面的诱导度量
  $$I = E du^2 + 2F dudv + G dv^2$$
- **几何意义**：弧长、角度、面积的度量
- **父节点**：曲面的参数表示
- **子节点**：测地线方程

**定理 DG4：第二基本形式**
- **陈述**：曲面在 $ \mathbb{R}^3 $ 中的弯曲程度
  $$II = L du^2 + 2M dudv + N dv^2$$
- **几何意义**：曲面偏离切平面的程度
- **父节点**：曲面的定向、单位法向量
- **子节点**：主曲率、Gauss曲率

**定理 DG5：主曲率与Gauss曲率**

**定理 DG5-1：主曲率**
- **陈述**：法曲率的极值，记为 $\kappa_1, \kappa_2$
- **计算**：Weingarten映射的特征值
- **父节点**：第二基本形式（DG4）、Weingarten映射

**定理 DG5-2：Gauss曲率**
- **定义**：$K = \kappa_1 \kappa_2 = \frac{LN - M^2}{EG - F^2}$
- **几何意义**：曲面在某点的内在弯曲程度
- **父节点**：主曲率（DG5-1）
- **子节点**：Gauss绝妙定理、Gauss-Bonnet定理

**定理 DG6：Gauss绝妙定理（Theorema Egregium）**
- **陈述**：Gauss曲率 $K$ 是内蕴量，仅依赖第一基本形式
- **证明思路**：通过Christoffel符号表达 $K$
- **父节点**：Gauss曲率（DG5-2）、Codazzi-Mainardi方程
- **意义**：**曲面弯曲的可测量与嵌入无关，是内蕴几何的基石**
- **应用**：等距变换保持 $K$，球面不能等距展开为平面

**定理 DG7：曲面的结构方程**

**Gauss方程**：$K$ 与第一基本形式的关系
**Codazzi-Mainardi方程**：第二基本形式的相容性条件

**曲面基本定理**：第一、二基本形式满足Gauss-Codazzi方程 $\Rightarrow$ 曲面局部存在唯一（相差刚体运动）

### 3.3 黎曼几何

**定理 DG8：Levi-Civita联络**
- **陈述**：黎曼流形 $(M, g)$ 上存在唯一的无挠、与度量相容的联络
- **Christoffel符号**：
  $$\Gamma^k_{ij} = \frac{1}{2}g^{kl}(\partial_i g_{jl} + \partial_j g_{il} - \partial_l g_{ij})$$
- **父节点**：仿射联络、度量相容性
- **子节点**：测地线、曲率张量

**定理 DG9：曲率张量**

**定义**：
$$R(X, Y)Z = \nabla_X\nabla_Y Z - \nabla_Y\nabla_X Z - \nabla_{[X,Y]}Z$$

**定理 DG9-1：Riemann曲率张量性质**
- **代数对称性**：
  - $R_{ijkl} = -R_{jikl} = -R_{ijlk} = R_{klij}$
  - 第一Bianchi恒等式：$R_{ijkl} + R_{iklj} + R_{iljk} = 0$
- **父节点**：Levi-Civita联络（DG8）

**定理 DG9-2：截面曲率**
- **定义**：二维子空间的曲率
  $$K(\sigma) = \frac{\langle R(X,Y)Y, X\rangle}{|X|^2|Y|^2 - \langle X,Y\rangle^2}$$

- **与Gauss曲率关系**：二维情形即Gauss曲率
- **父节点**：曲率张量（DG9-1）

**定理 DG9-3：Ricci曲率与标量曲率**
- **Ricci曲率**：$R_{ij} = R^k_{ikj}$
- **标量曲率**：$R = g^{ij}R_{ij}$
- **父节点**：曲率张量缩并
- **应用**：Einstein场方程

**定理 DG10：测地线理论**

**定理 DG10-1：测地线方程**
- **陈述**：$\frac{d^2x^k}{dt^2} + \Gamma^k_{ij}\frac{dx^i}{dt}\frac{dx^j}{dt} = 0$
- **变分特征**：弧长泛函的临界点
- **父节点**：Levi-Civita联络（DG8）

**定理 DG10-2：指数映射**
- **陈述**：$\exp_p: T_pM \to M$，$v \mapsto \gamma_v(1)$
- **局部微分同胚**：在0附近是微分同胚
- **父节点**：测地线存在唯一性
- **子节点**：正规坐标、Jacobi场

**定理 DG10-3：Hopf-Rinow定理**
- **陈述**：黎曼流形 $(M, g)$ 以下等价：
  1. 度量完备（Cauchy列收敛）
  2. 测地线完备（所有测地线可无限延拓）
  3. 有界闭集紧致
  4. 任意两点可由最短测地线连接
- **父节点**：测地线理论、度量空间完备性
- **判断逻辑**：**黎曼几何中完备性的核心定理**

**定理 DG11：Gauss-Bonnet定理（一般形式）**
- **陈述**：紧致定向二维黎曼流形
  $$\int_M K dA = 2\pi \chi(M) = 2\pi(2 - 2g)$$
- **拓扑不变量**：Euler示性数 $\chi(M)$
- **几何-拓扑联系**：曲率积分 = 拓扑不变量
- **父节点**：曲面论、拓扑学
- **高维推广**：Chern-Gauss-Bonnet定理

### 3.4 指标形式与Morse理论

**定理 DG12：Morse指标定理（几何形式）**
- **陈述**：测地线段的指标（共轭点个数，计重数）等于指标形式负定子空间的最大维数
- **共轭点**：Jacobi场消失的点
- **父节点**：Jacobi场、指标形式
- **应用**：最短测地线判定

## 四、代数几何推理链

### 4.1 仿射代数几何

**定理 AG1：Hilbert零点定理（Nullstellensatz）**

**定理 AG1-1：弱零点定理**
- **陈述**：$k$ 代数闭，$I \subseteq k[x_1, \ldots, x_n]$ 真理想，则 $V(I) \neq \emptyset$
- **等价形式**：$k[x_1, \ldots, x_n]$ 的极大理想形如 $(x_1 - a_1, \ldots, x_n - a_n)$
- **证明思路**：Noether正规化 + 域论
- **父节点**：Noether环、域扩张
- **子节点**：强零点定理

**定理 AG1-2：强零点定理（Hilbert Nullstellensatz）**
- **陈述**：$k$ 代数闭，$I \subseteq k[x_1, \ldots, x_n]$，则
  $$I(V(I)) = \sqrt{I}$$
- **代数-几何对应**：
  - 仿射簇 $\leftrightarrow$ 根式理想
  - 不可约簇 $\leftrightarrow$ 素理想
  - 点 $\leftrightarrow$ 极大理想
- **父节点**：弱零点定理（AG1-1）
- **判断逻辑**：**代数几何的基础，建立代数与几何的等价**

**定理 AG2：仿射簇的拓扑**
- **Zariski拓扑**：闭集为代数集
- **性质**：
  - 非Hausdorff
  - 紧致（quasi-compact）
  - 不可约性对应整环
- **父节点**：零点定理
- **子节点**：层上同调

### 4.2 射影代数几何

**定理 AG3：射影空间的性质**
- **紧致性**：射影簇是紧致的（在经典拓扑下）
- **完备性**：射影簇是完备的（在代数意义下）
- **父节点**：射影空间定义
- **应用**：相交理论、Bezout定理

**定理 AG4：Bezout定理**
- **陈述**：$\mathbb{P}^2$ 中，两条无公共分支的曲线 $C, D$ 次数分别为 $m, n$，则
  $$\sum_{p \in C \cap D} \text{mult}_p(C, D) = mn$$
- **重数**：交点的相交重数
- **父节点**：相交理论、射影空间性质
- **应用**：代数曲线计数

### 4.3 层上同调与对偶

**定理 AG5：层上同调**
- **陈述**：$H^i(X, \mathcal{F})$ 测量整体截面缺失
- **长正合序列**：短正合列诱导长正合上同调序列
- **父节点**：Abel范畴、导出函子
- **子节点**：Serre对偶、Riemann-Roch定理

**定理 AG6：Serre对偶**
- **陈述**：$X$ 光滑射影簇，维数 $n$，$\mathcal{F}$ 局部自由层，则
  $$H^i(X, \mathcal{F}) \cong H^{n-i}(X, \mathcal{F}^\vee \otimes \omega_X)^\vee$$
- **对偶层**：$\omega_X = \Omega^n_X$
- **父节点**：层上同调（AG5）、对偶层
- **应用**：计算上同调

**定理 AG7：Riemann-Roch定理**

**曲线情形**：
- **陈述**：$C$ 光滑射影曲线，亏格 $g$，$D$ 除子，则
  $$l(D) - l(K - D) = \deg(D) + 1 - g$$
- **项解释**：
  - $l(D)$：与 $D$ 关联的线丛的整体截面空间维数
  - $K$：典范除子
  - $g$：亏格（拓扑不变量）
- **父节点**：层上同调、Serre对偶（AG6）
- **应用**：曲线分类

**高维推广**：Hirzebruch-Riemann-Roch，Grothendieck-Riemann-Roch

### 5.3 关键判断逻辑

#### 几何类型判定策略

```

空间几何性质分析
├── 三角形内角和？
│   ├── = π → 欧氏几何（K=0）
│   ├── < π → 双曲几何（K<0）
│   └── > π → 椭圆几何（K>0）
├── 平行线存在性？
│   ├── 唯一存在 → 欧氏
│   ├── 无穷多条 → 双曲
│   └── 不存在 → 椭圆
└── 曲率符号？
    ├── K = 0 → 欧氏/平坦
    ├── K < 0 → 双曲/负曲率
    └── K > 0 → 球面/正曲率

```

#### 曲面分析策略

```

曲面几何分析
├── 等距变换？
│   └── Gauss绝妙定理：K 保持不变
├── 内蕴量 vs 外蕴量
│   ├── 内蕴（仅依赖第一基本形式）：
│   │   ├── Gauss曲率 K
│   │   ├── 测地曲率
│   │   └── 测地线
│   └── 外蕴（依赖第二基本形式）：
│       ├── 主曲率 κ₁, κ₂
│       └── 平均曲率 H
└── 形状判定
    ├── K > 0：椭圆点（局部像球面）
    ├── K < 0：双曲点（局部像马鞍）
    └── K = 0：抛物点（局部可展）

```

#### 代数几何问题策略

```

代数簇分析
├── 仿射簇？
│   └── Hilbert零点定理：簇 ↔ 根式理想
├── 射影簇？
│   ├── 齐次理想
│   └── Bezout定理：相交次数 = 次数乘积
├── 上同调计算？
│   ├── 短正合列 → 长正合序列
│   ├── Serre对偶
│   └── Riemann-Roch（曲线情形）
└── 奇点分析？
    ├── Jacobi矩阵秩
    └── 重数、切锥

```

### 5.4 公理依赖关系

```

几何公理依赖图

欧氏几何公理

│   ├── 关联公理
```
