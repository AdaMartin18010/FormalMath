## 一、实数系统推理链

### 1.1 核心公理系统

```
实数公理系统
├── 域公理（加法/乘法结构）
├── 序公理（全序关系）
└── 完备性公理（关键区分特征）
    ├── Dedekind完备性：有上界必有上确界
    └── Cauchy完备性：Cauchy列必收敛
```

**定理 R1：实数完备性等价定理群**

| 等价形式 | 关键内容 | 证明难度 |
|---------|---------|---------|
| 确界原理 | 非空有上界集必有上确界 | ★★☆☆☆ |
| 单调有界收敛 | 单调有界序列必收敛 | ★★☆☆☆ |
| 区间套定理 | 闭区间套有唯一公共点 | ★★☆☆☆ |
| Bolzano-Weierstrass | 有界序列必有收敛子列 | ★★★☆☆ |
| Cauchy收敛准则 | Cauchy列 ⟺ 收敛列 | ★★★☆☆ |
| 有限覆盖定理 | 闭区间开覆盖有有限子覆盖 | ★★★☆☆ |
| 聚点定理 | 无穷有界集必有聚点 | ★★☆☆☆ |

**定理 R1-1：确界原理**
- **陈述**：设 $E \subseteq \mathbb{R}$ 非空且有上界，则 $E$ 在 $\mathbb{R}$ 中存在上确界 $\sup E$
- **证明思路**：
  1. 利用Dedekind分割或十进制展开构造
  2. 或从完备性公理直接推出
- **依赖公理**：实数完备性公理
- **后继定理**：单调有界收敛定理（R1-2）
- **应用场景**：极值存在性证明、最优控制理论

**定理 R1-2：单调有界收敛定理**
- **陈述**：单调递增有上界（或递减有下界）的实数序列必收敛
- **证明思路**：
  1. 设 $\{a_n\}$ 递增有上界，令 $s = \sup\{a_n\}$
  2. 由确界定义，$\forall \epsilon > 0$，$\exists N$ 使 $s - \epsilon < a_N \leq s$
  3. 由单调性，$n \geq N$ 时 $s - \epsilon < a_n \leq s$
- **父节点**：确界原理（R1-1）
- **子节点**：Bolzano-Weierstrass定理（R1-4）、级数比较判别法
- **等价形式**：下降链条件（序对偶）
- **反例**：在有理数域中不成立（如 $a_n = (1+1/n)^n$ 的极限 $e$ 不在 $\mathbb{Q}$ 中）

**定理 R1-3：区间套定理**
- **陈述**：设 $[a_n, b_n]$ 为闭区间套（$[a_{n+1}, b_{n+1}] \subseteq [a_n, b_n]$ 且 $b_n - a_n \to 0$），则存在唯一的 $c$ 属于所有区间
- **证明思路**：
  1. $\{a_n\}$ 递增有上界，$\{b_n\}$ 递减有下界
  2. 由单调有界定理，两者都收敛
  3. $b_n - a_n \to 0$ 保证极限相同
- **父节点**：单调有界定理（R1-2）
- **子节点**：有限覆盖定理（R1-6）、Baire纲定理
- **边界条件**：必须为**闭**区间，开区间套可能无公共点
- **应用**：证明Bolzano-Weierstrass、建立实数不可数性

**定理 R1-4：Bolzano-Weierstrass定理**
- **陈述**：$\mathbb{R}$ 中的有界序列必有收敛子列
- **证明思路**（二分法）：
  1. 序列在 $[a, b]$ 中，二分区间
  2. 至少一半包含无穷多项，递归选择
  3. 由区间套定理，子列收敛于唯一公共点
- **父节点**：区间套定理（R1-3）
- **子节点**：Cauchy收敛准则（R1-5）、Arzelà-Ascoli定理
- **等价形式**：聚点定理、Weierstrass聚点原理
- **推广**：$\mathbb{R}^n$ 中有界序列必有收敛子列

**定理 R1-5：Cauchy收敛准则**
- **陈述**：序列 $\{a_n\}$ 收敛 $\iff$ $\forall \epsilon > 0$，$\exists N$，当 $m, n \geq N$ 时 $|a_m - a_n| < \epsilon$
- **证明思路**：
  - $(\Rightarrow)$ 由极限定义直接得
  - $(\Leftarrow)$ Cauchy列有界，用Bolzano-Weierstrass得收敛子列，再证全列收敛
- **父节点**：Bolzano-Weierstrass定理（R1-4）
- **子节点**：级数Cauchy判别、函数项级数一致收敛Cauchy准则、完备度量空间理论
- **判断逻辑**：**判定收敛的首选工具**（无需预先知道极限值）
- **反例**：在 $\mathbb{Q}$ 中存在Cauchy列不收敛

**定理 R1-6：有限覆盖定理（Heine-Borel）**
- **陈述**：闭区间 $[a, b]$ 的任意开覆盖必有有限子覆盖
- **证明思路**（反证法）：
  1. 假设 $[a, b]$ 不能被有限覆盖
  2. 二分区间，至少一半不能被有限覆盖
  3. 构造区间套，由区间套定理得公共点 $c$
  4. $c$ 在某开集中，该开集包含某小区间，矛盾
- **父节点**：区间套定理（R1-3）
- **等价定理群**：与 R1-1 至 R1-5 互相等价
- **推广**：$\mathbb{R}^n$ 中：紧致 $\iff$ 有界闭集（Heine-Borel定理）
- **边界条件**：闭区间不可省略，开区间 $(0,1)$ 的开覆盖 $\{(1/n, 1)\}$ 无有限子覆盖

### 1.2 判断逻辑：等价定理的选择策略

```
证明任务分析
├── 涉及序列收敛性？
│   ├── 有单调性？ → 用单调有界定理
│   ├── 有有界性？ → 用Bolzano-Weierstrass
│   └── 无具体信息？ → 用Cauchy准则
├── 涉及集合极值？
│   └── 用确界原理
├── 涉及紧致性论证？
│   └── 用有限覆盖定理
└── 涉及存在唯一性？
    └── 用区间套定理
```

## 二、极限与连续性推理链

### 2.1 极限理论基础

**定理 L1：序列极限与函数极限等价性**
- **陈述**：$\lim_{x \to a} f(x) = L \iff$ 对所有满足 $x_n \to a$（$x_n \neq a$）的序列，$\lim_{n \to \infty} f(x_n) = L$
- **证明思路**：
  - $(\Rightarrow)$ 由定义直接验证
  - $(\Leftarrow)$ 反证法：若极限不为 $L$，构造反例序列
- **父节点**：序列极限定义、函数极限定义
- **应用场景**：将函数极限问题转化为序列极限问题（Heine方法）
- **判断逻辑**：**证明函数极限不存在**的有力工具

**定理 L2：夹逼定理（三明治定理）**
- **陈述**：若 $g(x) \leq f(x) \leq h(x)$ 且 $\lim g(x) = \lim h(x) = L$，则 $\lim f(x) = L$
- **证明思路**：直接由 $\epsilon$-$\delta$ 定义验证
- **父节点**：极限定义、不等式保号性
- **子节点**：重要极限 $\lim_{x \to 0} \frac{\sin x}{x} = 1$ 的证明
- **判断逻辑**：**估计未知极限的首选方法**

**定理 L3：极限运算法则**
- **陈述**：设 $\lim f(x) = A$，$\lim g(x) = B$，则
  - $\lim(f \pm g) = A \pm B$
  - $\lim(f \cdot g) = A \cdot B$
  - $B \neq 0$ 时 $\lim(f/g) = A/B$
- **证明思路**：利用不等式技巧估计 $|f(x) - A|$ 和 $|g(x) - B|$
- **边界条件**：商法则要求分母极限非零
- **反例**：$\infty - \infty$、$0 \cdot \infty$、$\infty/\infty$ 为不定式

### 2.2 连续性推理链

**定理 C1：连续函数的运算**
- **陈述**：连续函数的四则运算（除数非零）仍连续，连续函数的复合仍连续
- **证明思路**：利用极限运算法则或 $\epsilon$-$\delta$ 直接验证
- **父节点**：连续性定义、极限运算法则
- **应用**：初等函数在其定义域内连续

**定理 C2：闭区间连续函数的性质**

```
连续性 + 闭区间 → 三大核心定理
├── 有界性定理：f 在 [a,b] 有界
├── 最值定理：f 在 [a,b] 取到最大/最小值
└── 介值定理（含零点定理）
    ├── 介值定理：f(a) < c < f(b) ⇒ ∃ξ, f(ξ) = c
    └── 零点定理：f(a)·f(b) < 0 ⇒ ∃ξ, f(ξ) = 0
```

**定理 C2-1：有界性定理**
- **陈述**：$f \in C[a, b]$ 则 $f$ 在 $[a, b]$ 上有界
- **证明思路**（有限覆盖）：
  1. 每点有邻域使 $f$ 有界（连续性）
  2. 开覆盖有有限子覆盖
  3. 有限个有界集的并仍有界
- **父节点**：有限覆盖定理（R1-6）、连续性定义
- **边界条件**：开区间不成立（如 $f(x) = 1/x$ 在 $(0, 1)$）

**定理 C2-2：最值定理（Weierstrass）**
- **陈述**：$f \in C[a, b]$ 则 $f$ 在 $[a, b]$ 上取到最大、最小值
- **证明思路**：
  1. 由有界性，$M = \sup f([a, b])$ 存在
  2. 构造序列 $f(x_n) \to M$
  3. 由Bolzano-Weierstrass，$x_n$ 有收敛子列 $x_{n_k} \to \xi$
  4. 由连续性，$f(\xi) = M$
- **父节点**：有界性定理（C2-1）、Bolzano-Weierstrass（R1-4）
- **应用场景**：极值问题、优化理论

**定理 C2-3：介值定理（零点定理）**
- **陈述**：$f \in C[a, b]$，$f(a)$ 与 $f(b)$ 异号，则 $\exists \xi \in (a, b)$，$f(\xi) = 0$
- **证明思路**（区间套）：
  1. 二分区间，选择使端点异号的子区间
  2. 构造区间套，由区间套定理得 $\xi$
  3. 由连续性，$f(\xi) = 0$
- **父节点**：区间套定理（R1-3）、连续性定义
- **子节点**：反函数连续性定理、不动点定理
- **判断逻辑**：**证明方程根存在性的核心工具**

**定理 C3：一致连续性（Heine-Cantor）**
- **陈述**：$f \in C[a, b]$ 则 $f$ 在 $[a, b]$ 上一致连续
- **证明思路**（有限覆盖）：
  1. 每点有邻域使 $|f(x) - f(y)| < \epsilon/2$
  2. 开覆盖有有限子覆盖
  3. 取 Lebesgue 数 $\delta$，$|x - y| < \delta$ 时在同一邻域
- **父节点**：有限覆盖定理（R1-6）
- **等价条件**：Cantor定理、闭区间上连续函数的一致连续性
- **边界条件**：开区间不成立（如 $\sin(1/x)$ 在 $(0, 1)$）
- **判断逻辑**：**积分可积性的关键前提**

## 三、微分学推理链

### 3.1 导数基础链

**定理 D1：求导法则**
- **陈述**：线性性、乘积法则、商法则、链式法则
- **证明思路**：由导数定义通过极限运算推导
- **父节点**：导数定义、极限运算法则
- **子节点**：初等函数导数公式、隐函数求导

**定理 D2：Fermat引理**
- **陈述**：$f$ 在 $x_0$ 可导且取极值，则 $f'(x_0) = 0$
- **证明思路**：由左右导数符号相反推出导数为零
- **父节点**：导数定义、极值定义
- **子节点**：Rolle定理（D3-1）
- **边界条件**：可导性不可省略（如 $f(x) = |x|$ 在 $x=0$）

### 3.2 微分中值定理链

```
中值定理层次结构
├── Rolle定理（基础）
│   └── Lagrange中值定理（核心）
│       ├── Cauchy中值定理（推广）
│       │   └── L'Hôpital法则（应用）
│       ├── 函数单调性判定
│       ├── Taylor定理（高阶推广）
│       └── 凸性判定
```

**定理 D3-1：Rolle定理**
- **陈述**：$f \in C[a, b]$，在 $(a, b)$ 可导，$f(a) = f(b)$，则 $\exists \xi \in (a, b)$，$f'(\xi) = 0$
- **证明思路**：
  1. 由最值定理，$f$ 取到最大/最小值
  2. 若极值点在内部，由Fermat引理，导数为零
  3. 若极值都在端点，$f$ 为常数，导数处处为零
- **父节点**：最值定理（C2-2）、Fermat引理（D2）
- **子节点**：Lagrange中值定理（D3-2）
- **判断逻辑**：**证明导数零点的核心工具**

**定理 D3-2：Lagrange中值定理**
- **陈述**：$f \in C[a, b]$，在 $(a, b)$ 可导，则 $\exists \xi \in (a, b)$，
  $$f'(\xi) = \frac{f(b) - f(a)}{b - a}$$
- **证明思路**：构造辅助函数 $g(x) = f(x) - f(a) - \frac{f(b) - f(a)}{b - a}(x - a)$，应用Rolle定理
- **父节点**：Rolle定理（D3-1）
- **子节点**：Cauchy中值定理、单调性判定、Taylor定理
- **等价形式**：$f(b) - f(a) = f'(\xi)(b - a)$（微分增量公式）
- **判断逻辑**：**建立函数增量与导数联系的核心桥梁**

**定理 D3-3：Cauchy中值定理**
- **陈述**：$f, g \in C[a, b]$，在 $(a, b)$ 可导，$g'(x) \neq 0$，则 $\exists \xi \in (a, b)$，
  $$\frac{f(b) - f(a)}{g(b) - g(a)} = \frac{f'(\xi)}{g'(\xi)}$$
- **证明思路**：构造辅助函数，应用Rolle定理
- **父节点**：Rolle定理（D3-1）或 Lagrange中值定理（D3-2）
- **子节点**：L'Hôpital法则（D4）
- **判断逻辑**：**处理比值的微分中值问题**

**定理 D4：L'Hôpital法则**
- **陈述**：$\lim_{x \to a} f(x) = \lim_{x \to a} g(x) = 0$（或 $\infty$），则
  $$\lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)}$$
  （右端极限存在或为无穷）
- **证明思路**：应用Cauchy中值定理，$\frac{f(x)}{g(x)} = \frac{f(x) - f(a)}{g(x) - g(a)} = \frac{f'(\xi)}{g'(\xi)}$
- **父节点**：Cauchy中值定理（D3-3）
- **判断逻辑**：**不定式极限计算的首选方法**
- **边界条件**：
  - 必须为 $0/0$ 或 $\infty/\infty$ 型
  - 导数比的极限必须存在（或为无穷）
  - 可多次应用

### 3.3 Taylor定理与极值理论

**定理 D5：Taylor定理**
- **陈述**：$f$ 在 $x_0$ 的某邻域有 $n+1$ 阶导数，则
  $$f(x) = \sum_{k=0}^{n} \frac{f^{(k)}(x_0)}{k!}(x-x_0)^k + R_n(x)$$
  其中 Lagrange 余项：$R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!}(x-x_0)^{n+1}$
- **证明思路**：反复应用Cauchy中值定理
- **父节点**：Lagrange中值定理（D3-2）
- **子节点**：极值判定（D6）、函数逼近、数值分析
- **等价形式**：Peano余项、Cauchy余项、积分余项
- **判断逻辑**：**函数局部逼近的核心工具**

**定理 D6：极值判定定理**
- **必要条件**：$f'(x_0) = 0$（Fermat引理）
- **第一充分条件**：$f'$ 在 $x_0$ 两侧变号
- **第二充分条件**：$f'(x_0) = 0$，$f''(x_0) \neq 0$
  - $f''(x_0) > 0$：极小值
  - $f''(x_0) < 0$：极大值
- **高阶条件**：$f'(x_0) = \cdots = f^{(n-1)}(x_0) = 0$，$f^{(n)}(x_0) \neq 0$
  - $n$ 偶：极值；$n$ 奇：非极值
- **父节点**：Taylor定理（D5）
- **判断逻辑**：**极值问题分析的层次结构**

**定理 D7：凸性判定**
- **陈述**：$f$ 在区间 $I$ 上凸 $\iff$ $f''(x) \geq 0$（二阶可导时）
- **证明思路**：利用Taylor展开或中值定理
- **父节点**：Lagrange中值定理、Taylor定理
- **应用**：Jensen不等式、优化理论

## 四、积分学推理链

### 4.1 Riemann积分理论

**定理 I1：可积性条件**

```
可积性判定层次
├── 必要条件：f 有界
├── 充分条件：
│   ├── f 连续 → 可积
│   ├── f 有有限个间断点 → 可积
│   └── f 单调 → 可积
└── Riemann可积准则：
    上积分 = 下积分
    ⇔ ∀ε>0，存在分划使上和-下和<ε
```

**定理 I1-1：连续函数可积性**
- **陈述**：$f \in C[a, b]$ 则 $f$ 在 $[a, b]$ 上Riemann可积
- **证明思路**：
  1. 由一致连续性，$\forall \epsilon > 0$，$\exists \delta > 0$
  2. 当分划细度 $< \delta$ 时，振幅 $< \epsilon/(b-a)$
  3. 上和 - 下和 $< \epsilon$
- **父节点**：一致连续性（C3）
- **子节点**：微积分基本定理

**定理 I2：微积分基本定理（Newton-Leibniz）**

**定理 I2-1：变上限积分求导**
- **陈述**：$f$ 可积，$F(x) = \int_a^x f(t) dt$，则 $F$ 连续；若 $f$ 在 $x_0$ 连续，则 $F'(x_0) = f(x_0)$
- **证明思路**：
  $$\frac{F(x+h) - F(x)}{h} = \frac{1}{h}\int_x^{x+h} f(t) dt$$
  由积分中值定理和连续性得极限为 $f(x)$
- **父节点**：积分定义、连续性
- **子节点**：Newton-Leibniz公式（I2-2）

**定理 I2-2：Newton-Leibniz公式**
- **陈述**：$f$ 可积且有原函数 $F$，则 $\int_a^b f(x) dx = F(b) - F(a)$
- **证明思路**：对分划应用中值定理
- **父节点**：变上限积分求导（I2-1）、微分中值定理
- **判断逻辑**：**定积分计算的核心公式**
- **边界条件**：需要原函数存在且被积函数可积

### 4.2 Lebesgue积分理论

**定理 I3：Lebesgue控制收敛定理**
- **陈述**：$f_n \to f$ a.e.，$|f_n| \leq g$ 可积，则 $f$ 可积且
  $$\lim_{n \to \infty} \int f_n = \int \lim_{n \to \infty} f_n = \int f$$
- **父节点**：Fatou引理、单调收敛定理
- **应用场景**：积分与极限交换
- **判断逻辑**：**Lebesgue积分的核心优势所在**

**定理 I4：Fubini定理**
- **陈述**：$f$ 在乘积空间可积，则
  $$\int_{X \times Y} f = \int_X \left(\int_Y f(x,y) dy\right) dx = \int_Y \left(\int_X f(x,y) dx\right) dy$$
- **父节点**：乘积测度理论、单调类定理
- **边界条件**：需要 $f$ 可积（或 $f \geq 0$）
- **反例**：非负不可测函数不能交换积分次序

**定理 I5：$L^p$空间完备性（Riesz-Fischer）**
- **陈述**：$L^p$空间是Banach空间（$1 \leq p \leq \infty$）
- **父节点**：控制收敛定理、Holder不等式
- **应用场景**：泛函分析、偏微分方程

## 五、级数推理链

### 5.1 数项级数理论

**定理 S1：级数收敛的必要条件**
- **陈述**：$\sum a_n$ 收敛 $\Rightarrow$ $a_n \to 0$
- **证明思路**：$a_n = S_n - S_{n-1} \to S - S = 0$
- **父节点**：序列极限运算法则
- **逆否命题**：$a_n \not\to 0$ 则级数发散（发散判别法）
- **边界条件**：必要条件非充分（如调和级数）

**定理 S2：正项级数判别法**

```
正项级数判别法层次
├── 比较判别法（基础）
│   ├── 直接比较
│   └── 极限比较
├── 比值判别法（D'Alembert）
├── 根值判别法（Cauchy）
└── 积分判别法
```

**定理 S2-1：比较判别法**
- **陈述**：$0 \leq a_n \leq b_n$，$\sum b_n$ 收敛 $\Rightarrow$ $\sum a_n$ 收敛
- **父节点**：单调有界定理、级数部分和有界
- **判断逻辑**：**最基础但最强大的判别法**
- **常用比较对象**：几何级数、$p$-级数

**定理 S2-2：比值判别法**
- **陈述**：$\limsup \left|\frac{a_{n+1}}{a_n}\right| = L$，则
  - $L < 1$：绝对收敛
  - $L > 1$：发散
  - $L = 1$：不定
- **父节点**：与几何级数比较
- **边界条件**：$L = 1$ 时不定（如 $\sum 1/n^2$ 收敛，$\sum 1/n$ 发散）

**定理 S2-3：根值判别法**
- **陈述**：$\limsup \sqrt[n]{|a_n|} = L$，判别结论同比值法
- **父节点**：与几何级数比较
- **与比值法关系**：根值法比比值法更强
- **证明**：利用 $\liminf \frac{a_{n+1}}{a_n} \leq \liminf \sqrt[n]{a_n} \leq \limsup \sqrt[n]{a_n} \leq \limsup \frac{a_{n+1}}{a_n}$

**定理 S3：绝对收敛与条件收敛**
- **绝对收敛**：$\sum |a_n|$ 收敛 $\Rightarrow$ $\sum a_n$ 收敛
- **条件收敛**：$\sum a_n$ 收敛但 $\sum |a_n|$ 发散
- **Riemann重排定理**：条件收敛级数可重排收敛到任意值（或发散）
- **父节点**：Cauchy收敛准则
- **判断逻辑**：**绝对收敛允许重排，条件收敛需谨慎**

### 5.2 函数项级数理论

**定理 S4：一致收敛判别法**

**定理 S4-1：Weierstrass M判别法**
- **陈述**：$|f_n(x)| \leq M_n$，$\sum M_n$ 收敛，则 $\sum f_n$ 一致收敛
- **父节点**：Cauchy一致收敛准则、比较判别法
- **判断逻辑**：**判断一致收敛的首选工具**

**定理 S5：一致收敛的性质**
- **连续性**：$f_n$ 连续且一致收敛于 $f$，则 $f$ 连续
- **逐项积分**：在有限区间上可逐项积分
- **逐项微分**：需要导函数列一致收敛
- **父节点**：一致收敛定义、相应定理
- **应用场景**：幂级数、Fourier级数运算

**定理 S6：幂级数理论**

**定理 S6-1：收敛半径（Cauchy-Hadamard）**
- **陈述**：$R = 1/\limsup \sqrt[n]{|a_n|}$
- **性质**：$|x| < R$ 绝对收敛，$|x| > R$ 发散
- **父节点**：根值判别法
- **子节点**：Abel定理、Taylor级数展开

**定理 S6-2：Abel定理**
- **陈述**：幂级数在收敛域端点收敛，则在闭区间上一致收敛
- **应用**：证明 $\sum (-1)^n/n = \ln 2$ 等
- **父节点**：Abel变换、级数一致收敛

**定理 S7：Fourier级数**

**定理 S7-1：Dirichlet收敛定理**
- **陈述**：$f$ 分段光滑（或分段单调），则Fourier级数在每点收敛于 $\frac{f(x^+) + f(x^-)}{2}$
- **父节点**：Riemann-Lebesgue引理、Dirichlet核估计
- **判断逻辑**：**Fourier级数点态收敛的基本定理**

**定理 S7-2：Parseval等式**
- **陈述**：$\frac{1}{\pi}\int_{-\pi}^{\pi} |f(x)|^2 dx = \frac{a_0^2}{2} + \sum_{n=1}^{\infty}(a_n^2 + b_n^2)$
- **父节点**：Fourier系数正交性、一致收敛
- **应用**：级数求和、能量守恒

