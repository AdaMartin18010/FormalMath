## 🎓 使用指南

### 如何查找术语

1. **按数学分支查找**
   - 确定术语所属的数学分支
   - 打开对应的术语词典
   - 使用词典内的索引或速查表

2. **按字母顺序查找**
   - 打开任意词典
   - 使用"术语速查表"中的"按字母顺序速查"
   - 跳转到术语定义

3. **按主题查找**
   - 打开任意词典
   - 使用"术语速查表"中的"按主题速查"
   - 找到相关主题的术语

4. **使用对比表**
   - 打开任意词典
   - 使用"术语对比表"
   - 理解相关概念的区别和联系

### 学习建议

1. **初学者**
   - 从基础数学术语词典开始
   - 按照术语优先级学习顺序学习
   - 完成基础练习，参考练习题答案

2. **中级学习者**
   - 选择感兴趣的数学分支
   - 使用术语关系网络图理解概念关系
   - 完成进阶练习，深入理解概念

3. **高级学习者**
   - 使用术语应用指南
   - 参考扩展阅读建议
   - 将术语应用到实际研究中
