# 31.04-拟阵理论

拟阵（Matroid）是线性无关性的抽象，统一了图论、线性代数和组合几何中的多种结构。

## 🎯 核心主题

### 拟阵基础
- **独立集公理**：遗传性与交换性
- **基与圈**：极大独立集、极小相关集
- **秩函数**：子集的秩
- ** flats 与闭包**：几何闭包算子

### 对偶性与运算
- **对偶拟阵**：对偶的构造与性质
- **删除与收缩**：拟阵的 minors
- **直和与张量积**：组合运算

### 拟阵与优化
- **贪婪算法**：最优基的选择
- **拟阵交**：最大公共独立集
- **拟阵划分**：Edmonds定理

### 可表示拟阵
- **线性拟阵**：域上的矩阵表示
- **图拟阵**：图的圈拟阵
- **代数拟阵**：代数相关性
