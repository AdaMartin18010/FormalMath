# 31.03-设计理论

设计理论研究有限集合的特定子集族配置，在实验设计、编码理论、密码学中有重要应用。

## 🎯 核心主题

### 区组设计
- **平衡不完全区组设计 (BIBD)**：参数、存在条件
- **Steiner系统**：$S(t,k,v)$，Steiner三元系
- **对称设计**：Fisher不等式取等情形
- **可分解设计**：Kirkman女学生问题

### 有限几何
- **射影平面**：阶为 $q$ 的射影平面
- **仿射平面**：性质与构作
- **高维几何**：射影空间 $PG(n,q)$
- **卵形与 blocking sets**

### 拉丁方
- **正交拉丁方**：Euler三十六军官问题
- **Mutually Orthogonal Latin Squares (MOLS)**
- **拉丁方与设计的联系**
