# 31.01-计数组合学

计数组合学研究离散结构的计数问题，包括基本计数原理、生成函数、递推关系等核心方法。

## 🎯 核心主题

### 基本计数方法
- **排列与组合**：有重复/无重复、圆排列
- **二项式定理**：牛顿二项式、多项式定理
- **容斥原理**：筛法公式、Möbius反演

### 生成函数理论
- **普通生成函数 (OGF)**：形式幂级数、收敛性
- **指数生成函数 (EGF)**：标记结构计数
- **多元生成函数**：多元组合枚举

### 递推关系
- **线性递推**：常系数、变系数
- **特征方程法**：齐次与非齐次方程
- **母函数法**：递推与生成函数的关系
