# 01-基础概念

## 一、对称函数基础

### 1.1 定义与环结构

**定义 1.1** (对称函数)：$\Lambda_n = \mathbb{C}[x_1, \ldots, x_n]^{S_n}$ 为 $n$ 变元对称多项式环。

**定义 1.2** (形式对称函数)：$\Lambda = \varprojlim \Lambda_n$，$n \to \infty$ 时的逆极限。

**分次**：$\Lambda = \bigoplus_{k \geq 0} \Lambda^k$，$k$ 次齐次对称函数。

### 1.2 基础对称函数

**定义 1.3** (单项式对称函数)：对分拆 $\lambda = (\lambda_1 \geq \lambda_2 \geq \cdots)$，
$$m_\lambda = \sum_{\alpha} x^\alpha$$

其中 $\alpha$ 取遍 $\lambda$ 的所有不同排列。

**定义 1.4** (初等对称函数)：
$$e_k = m_{(1^k)} = \sum_{i_1 < \cdots < i_k} x_{i_1} \cdots x_{i_k}$$
$$e_\lambda = e_{\lambda_1} e_{\lambda_2} \cdots$$

**定义 1.5** (完全齐次对称函数)：
$$h_k = \sum_{i_1 \leq \cdots \leq i_k} x_{i_1} \cdots x_{i_k}$$
$$h_\lambda = h_{\lambda_1} h_{\lambda_2} \cdots$$

**定义 1.6** (幂和对称函数)：
$$p_k = m_{(k)} = \sum_i x_i^k$$
$$p_\lambda = p_{\lambda_1} p_{\lambda_2} \cdots$$

### 1.3 生成函数

**定理 1.7**：
$$E(t) = \sum_{k \geq 0} e_k t^k = \prod_{i \geq 1}(1 + x_i t)$$

$$H(t) = \sum_{k \geq 0} h_k t^k = \prod_{i \geq 1}\frac{1}{1 - x_i t}$$

$$P(t) = \sum_{k \geq 1} p_k \frac{t^k}{k} = \log \prod_{i \geq 1}\frac{1}{1-x_i t}$$

**推论 1.8**：$E(t)H(-t) = 1$，即 $\sum_{r=0}^n (-1)^r e_r h_{n-r} = 0$（$n \geq 1$）。

## 二、Schur函数

### 2.1 定义

**定义 2.1** (Schur函数)：对分拆 $\lambda$（$l(\lambda) \leq n$），
$$s_\lambda(x_1, \ldots, x_n) = \frac{\det(x_i^{\lambda_j + n - j})_{1 \leq i,j \leq n}}{\det(x_i^{n-j})_{1 \leq i,j \leq n}}$$

分母为 Vandermonde 行列式 $\prod_{i<j}(x_i - x_j)$。

**定义 2.2** (组合定义)：
$$s_\lambda = \sum_T x^T$$

其中 $T$ 取遍形状为 $\lambda$ 的半标准杨表（SSYT），$x^T = \prod_i x_i^{\text{$i$在$T$中出现次数}}$。

### 2.2 基本性质

**定理 2.3**：$\{s_\lambda : |\lambda| = n\}$ 构成 $\Lambda^n$ 的标准正交基（关于Hall内积）。

**定理 2.4** (Cauchy恒等式)：
$$\sum_\lambda s_\lambda(x)s_\lambda(y) = \prod_{i,j}\frac{1}{1-x_i y_j}$$

**定理 2.5** (对偶性)：
$$\omega(s_\lambda) = s_{\lambda'}$$

其中 $\lambda'$ 为共轭分拆，$\omega$ 为对合：$\omega(e_k) = h_k$。

## 三、Young图与杨表

### 3.1 基本定义

**定义 3.1** (分拆)：$\lambda = (\lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_k > 0)$，$|\lambda| = \sum \lambda_i$。

**定义 3.2** (Young图)：分拆 $\lambda$ 的图由左对齐的行组成，第 $i$ 行 $\lambda_i$ 个方格。

**定义 3.3** (半标准杨表 SSYT)：在Young图中填入正整数，行弱增，列严格增。

**定义 3.4** (标准杨表 SYT)：$[n]$ 填入，行和列都严格增，$n = |\lambda|$。

**例**：形状 $(3,2)$ 的 SYT：
$$\begin{array}{ccc} 1 & 2 & 4 \\ 3 & 5 \end{array}$$

### 3.2 钩长公式

**定义 3.5** (钩)：方格 $v$ 的钩含 $v$ 及同行右侧、同列下方的方格。

**钩长** $h(v)$：钩中方格数。

**定理 3.6** (Frame-Robinson-Thrall钩长公式)：
$$f^\lambda = \#\{\text{形状 } \lambda \text{ 的 SYT}\} = \frac{|\lambda|!}{\prod_{v \in \lambda} h(v)}$$

**例**：$\lambda = (n)$，$f^{(n)} = 1$（唯一递增填法）。

$\lambda = (1^n)$，$f^{(1^n)} = 1$。

$\lambda = (2,1)$，$f^{(2,1)} = \frac{3!}{3 \cdot 1 \cdot 1} = 2$。

## 四、Littlewood-Richardson规则

### 4.1 问题陈述

**问题**：展开 $s_\mu s_\nu = \sum_\lambda c_{\mu\nu}^\lambda s_\lambda$，求结构常数 $c_{\mu\nu}^\lambda$。

### 4.2 LR规则

**定义 4.1** (格排列)：字 $w = w_1 w_2 \cdots w_n$，任意前缀中 $i$ 的个数 $\geq i+1$ 的个数（对所有 $i$）。

**定义 4.2** (逆序读取)：从右到左、从上到下读取杨表。

**定理 4.3** (Littlewood-Richardson规则)：

$c_{\mu\nu}^\lambda$ 为形状 $\lambda/\mu$、权 $\nu$ 的 SSYT 数，其逆序读取为格排列。

**例**：$c_{(1),(1)}^{(2)} = 1$，$c_{(1),(1)}^{(1,1)} = 1$。

即 $s_{(1)}^2 = s_{(2)} + s_{(1,1)}$，对应 $e_1^2 = h_2 + e_2$（实际是 $h_1^2 = h_2 + e_2$）。

### 4.3 特殊情形

**Pieri规则**：
$$s_\lambda s_{(k)} = \sum_\mu s_\mu$$

其中 $\mu/\lambda$ 为水平 $k$-条带（无两格同列）。

$$s_\lambda s_{(1^k)} = \sum_\mu s_\mu$$

其中 $\mu/\lambda$ 为垂直 $k$-条带（无两格同行）。

## 五、表示论联系

### 5.1 对称群的表示

**定理 5.1**：$\{S^\lambda\}$（Specht模）构成 $S_n$ 的所有不可约表示。

$$\dim S^\lambda = f^\lambda$$

**特征标**：$\chi^\lambda(\mu)$ 为形状 $\lambda$、权 $\mu$ 的 SSYT 数（Murnaghan-Nakayama规则）。

### 5.2 Frobenius特征标

**定义 5.2** (Frobenius映射)：
$$\text{ch}: R(S_n) \to \Lambda^n$$
$$\chi^\lambda \mapsto s_\lambda$$

**定理 5.3**：Frobenius映射为等距同构（适当归一化内积）。
