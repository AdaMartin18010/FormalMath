# 31.06-代数组合学

代数组合学运用代数方法研究组合结构，连接组合数学与表示论、代数几何、对称函数理论。

## 🎯 核心主题

### 对称函数
- **基础对称函数**：单项式、初等、完全齐次、幂和
- **Schur函数**：最重要的基，与表示论联系
- **Littlewood-Richardson规则**：结构常数
- **Hall-Littlewood与Macdonald多项式**：$q,t$-变形

### 组合表示论
- **对称群的表示**：Specht模
- **Young图与杨表**：组合框架
- **RSK对应**：排列与杨表的双射
- **Plancherel测度**：随机杨表

### 代数组合结构
- **组合Hopf代数**：对称函数代数
- **群作用与计数**：Burnside引理、Pólya理论
- **关联代数**：偏序集上的代数
