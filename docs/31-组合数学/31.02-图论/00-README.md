# 31.02-图论

图论研究图的性质、结构和算法，是组合数学的核心分支，在计算机科学、网络分析、运筹学中有广泛应用。

## 🎯 核心主题

### 基础图论
- **图的基本概念**：简单图、多重图、有向图
- **连通性**：连通分支、割点、桥、 $k$-连通
- **树与森林**：生成树、最小生成树、树遍历
- **匹配与覆盖**：最大匹配、完美匹配、Kőnig定理

### 极值图论
- **Turán定理**：无完全子图的极值图
- **Erdős-Stone定理**：极值图的渐近行为
- **Szemerédi正则性引理**：图的结构分解

### 随机图论
- **Erdős-Rényi模型**：$G(n,p)$ 和 $G(n,m)$
- **相变现象**：连通性阈值、巨分量涌现
- **小世界网络**：Watts-Strogatz模型

### 代数图论
- **图谱理论**：邻接矩阵、拉普拉斯矩阵、特征值
- **图同态**：图染色、图同构
- **凯莱图**：群论与图论的联系
