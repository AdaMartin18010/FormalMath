## 📋 说明

本目录存放**工作示例**（Worked Examples）：步骤完整、解释充分的例题与证明，用于降低认知负荷、支持概念理解与技能迁移。目标规模：150+ 示例（基础 100+，进阶 50+）。

**分类**:

| 分类 | 说明 | 目标数量 |
|------|------|----------|
| 概念理解 | 概念定义、等价表述、典型例与反例 | 50+ |
| 计算示例 | 计算步骤完整、中间结果与检验 | 40+ |
| 证明示例 | 证明步骤拆解、思路与关键步骤 | 40+ |
| 应用示例 | 定理/方法在实际情境中的应用 | 20+ |

**难度层级**（与 L0–L3 对应）:

- **L0**: 入门，步骤最细、解释最多
- **L1**: 中级，适当省略重复步骤
- **L2**: 进阶，侧重思路与关键步骤
- **L3**: 研究导向，可与前沿或开放问题衔接

**命名与格式**:

- 文件名：`序号-主题-工作示例.md`
- 每篇含：元信息（领域、难度、前置知识）、题目/命题、完整解答、关键步骤说明、相关概念链接
