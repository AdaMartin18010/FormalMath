## 题目

写出线性规划 \(\max c^T x\)，\(Ax \le b\)，\(x\ge 0\) 的对偶问题，并说明弱对偶：可行 \(x,y\) 满足 \(c^T x \le b^T y\)。

---

## 完整解答

对偶：\(\min b^T y\)，\(A^T y \ge c\)，\(y \ge 0\)。弱对偶：\(c^T x \le (A^T y)^T x = y^T A x \le y^T b = b^T y\)（因 \(Ax \le b\)，\(y \ge 0\)）。
