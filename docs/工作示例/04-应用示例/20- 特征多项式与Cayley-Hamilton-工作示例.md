## 题目

设 \(A\) 满足 \(A^2=A\)（幂等）。用 Cayley–Hamilton 说明 \(A\) 的特征值仅为 0 或 1。

---

## 完整解答

\(A^2-A=0\)，故 \(A\) 的极小多项式整除 \(t^2-t=t(t-1)\)，特征值仅为 0 或 1。
