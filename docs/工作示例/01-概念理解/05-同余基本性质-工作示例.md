## 题目

设 \(n \in \mathbb{Z}^+\)。说明「\(a \equiv b \pmod{n}\)」与「\(n \mid (a-b)\)」的等价性，并证明：若 \(a \equiv b\)、\(c \equiv d \pmod{n}\)，则 \(a+c \equiv b+d\)、\(ac \equiv bd \pmod{n}\)。

---

## 完整解答（工作示例）

**步骤 1**：\(a \equiv b \pmod{n}\) 定义为 \(n \mid (a-b)\)，故二者等价。

**步骤 2**：加法。由 \(n \mid (a-b)\) 与 \(n \mid (c-d)\) 得 \(n \mid ((a-b)+(c-d)) = (a+c)-(b+d)\)，故 \(a+c \equiv b+d \pmod{n}\)。

**步骤 3**：乘法。\(ac - bd = ac - bc + bc - bd = c(a-b) + b(c-d)\)。由 \(n \mid (a-b)\) 与 \(n \mid (c-d)\) 得 \(n \mid (ac-bd)\)，故 \(ac \equiv bd \pmod{n}\)。

---

## 关键步骤说明

- **等价性**：同余的定义即「差被模整除」。
- **运算保持**：同余对加、减、乘封闭；对除法需与模互素时才有逆元。
- **易错点**：\(ac \equiv bd\) 的证明中要凑出 \(a-b\) 与 \(c-d\) 的倍数。
