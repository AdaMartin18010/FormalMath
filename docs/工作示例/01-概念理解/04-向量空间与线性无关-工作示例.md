## 题目

设 \(V\) 为 \(\mathbb{R}\) 上的向量空间，\(v_1, v_2, v_3 \in V\)。说明「\(\{v_1, v_2, v_3\}\) 线性无关」与「任意 \(v \in V\) 至多有一种方式写成 \(v_1, v_2, v_3\) 的线性组合」的等价性。

---

## 完整解答（工作示例）

**步骤 1**：若 \(\{v_1, v_2, v_3\}\) 线性无关，设 \(v = a_1 v_1 + a_2 v_2 + a_3 v_3 = b_1 v_1 + b_2 v_2 + b_3 v_3\)。则 \((a_1-b_1)v_1 + (a_2-b_2)v_2 + (a_3-b_3)v_3 = 0\)，由线性无关得 \(a_i = b_i\)，故表示唯一。

**步骤 2**：若任意 \(v\) 至多有一种表示，特别地取 \(v = 0\)。则 \(0\) 只有一种方式写成 \(v_1,v_2,v_3\) 的线性组合，即 \(0 \cdot v_1 + 0 \cdot v_2 + 0 \cdot v_3\)。若 \(c_1 v_1 + c_2 v_2 + c_3 v_3 = 0\)，则 \(0\) 有两种表示（系数全 0 与 \((c_1,c_2,c_3)\)），由「至多一种」得 \(c_1 = c_2 = c_3 = 0\)，故线性无关。

---

## 关键步骤说明

- **线性无关**：仅当系数全为 0 时线性组合为 0。
- **唯一表示**：线性无关等价于每个向量在该组下的表示（若存在）唯一。
- **易错点**：「至多一种」包含「不存在」的情况；证明中用 \(v=0\) 推出系数全 0。
