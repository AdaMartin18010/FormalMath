## 题目

设 \(f_n : [a,b] \to \mathbb{R}\) 连续，\(f_n \rightrightarrows f\)（一致收敛）。证明 \(f\) 连续。

---

## 完整解答（工作示例）

**步骤 1**：取定 \(x_0 \in [a,b]\)。对 \(\varepsilon>0\)，由一致收敛存在 \(N\) 使 \(n \geq N \Rightarrow |f_n(x)-f(x)| < \varepsilon/3\) 对所有 \(x\)。取 \(n=N\)。

**步骤 2**：\(f_N\) 连续，存在 \(\delta>0\) 使 \(|x-x_0|<\delta \Rightarrow |f_N(x)-f_N(x_0)| < \varepsilon/3\)。

**步骤 3**：当 \(|x-x_0|<\delta\) 时，\(|f(x)-f(x_0)| \leq |f(x)-f_N(x)| + |f_N(x)-f_N(x_0)| + |f_N(x_0)-f(x_0)| < \varepsilon/3 + \varepsilon/3 + \varepsilon/3 = \varepsilon\)。故 \(f\) 在 \(x_0\) 连续。
