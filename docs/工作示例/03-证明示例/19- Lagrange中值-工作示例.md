## 定理陈述

**Lagrange中值定理**: 

设函数 \(f\) 满足：
1. 在闭区间 \([a,b]\) 上**连续**
2. 在开区间 \((a,b)\) 上**可导**

则存在 \(\xi \in (a,b)\)，使得：
$$f(b) - f(a) = f'(\xi)(b-a)$$

**几何意义**: 曲线上存在一点，其切线与连接端点的弦平行。

---

## 完整证明

### 证明思路
构造辅助函数，使其在端点值相等，然后应用Rolle定理。

### 详细证明

**步骤1：构造辅助函数**

令弦的方程为：
$$L(x) = f(a) + \frac{f(b)-f(a)}{b-a}(x-a)$$

定义辅助函数：
$$F(x) = f(x) - L(x) = f(x) - f(a) - \frac{f(b)-f(a)}{b-a}(x-a)$$

**步骤2：验证F(x)满足Rolle定理条件**

- **连续性**: \(F(x)\) 在 \([a,b]\) 连续（连续函数的和差）
- **可导性**: \(F(x)\) 在 \((a,b)\) 可导（可导函数的和差）
- **端点值**:
  - \(F(a) = f(a) - f(a) - 0 = 0\)
  - \(F(b) = f(b) - f(a) - (f(b)-f(a)) = 0\)

**步骤3：应用Rolle定理**

由Rolle定理，存在 \(\xi \in (a,b)\) 使得 \(F'(\xi) = 0\)。

**步骤4：计算导数并求解**

$$F'(x) = f'(x) - \frac{f(b)-f(a)}{b-a}$$

令 \(F'(\xi) = 0\)：
$$f'(\xi) - \frac{f(b)-f(a)}{b-a} = 0$$

因此：
$$f'(\xi) = \frac{f(b)-f(a)}{b-a}$$

即：
$$\boxed{f(b) - f(a) = f'(\xi)(b-a)}$$

证毕。

## 几何解释

```
    y
    │      ╭────╮
    │    ╱   ξ  ╲
    │   │   ●    │  ← 切线（斜率f'(ξ)）
    │  ╱  ╱│╲    ╲
    │ ╱  ╱ │ ╲    ╲
    │╱  ╱  │  ╲    ╲
    ●────────┼────────●
    a         │        b
              │
    弦（连接(a,f(a))和(b,f(b))）
```

**核心思想**: 连续光滑的曲线必有某点的切线与弦平行。

## 应用与推论

| 推论 | 内容 |
|-----|------|
| **零导数** | f'(x)≡0 ⇒ f(x)≡C |
| **相等导数** | f'=g' ⇒ f=g+C |
| **单调性** | f'>0 ⇒ f单调增 |
