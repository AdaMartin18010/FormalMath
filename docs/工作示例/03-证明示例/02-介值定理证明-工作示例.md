## 命题

**定理**：设 \(f : [a,b] \to \mathbb{R}\) 连续，\(f(a) \ne f(b)\)，\(\mu\) 介于 \(f(a)\) 与 \(f(b)\) 之间。则存在 \(c \in (a,b)\) 使得 \(f(c) = \mu\)。

---

## 完整证明（工作示例）

**步骤 1**：不妨设 \(f(a) < \mu < f(b)\)（否则考虑 \(-f\)）。

**步骤 2**：令 \(S = \{ x \in [a,b] : f(x) \le \mu \}\)。\(S \ne \varnothing\)（\(a \in S\)），\(S \subseteq [a,b]\) 故有上确界，记 \(c = \sup S\)，则 \(c \in [a,b]\)。

**步骤 3**：由 \(f\) 连续与 \(c = \sup S\)，可证 \(f(c) \le \mu\) 且 \(f(c) \ge \mu\)。

- 取 \(x_n \in S\) 且 \(x_n \to c\)，则 \(f(x_n) \le \mu\)，由连续性 \(f(c) = \lim f(x_n) \le \mu\)。
- 若 \(f(c) < \mu\)，则由连续性存在 \(\delta > 0\) 使 \(x \in (c, c+\delta) \cap [a,b] \Rightarrow f(x) < \mu\)，故 \((c, c+\delta) \cap [a,b] \subseteq S\)，与 \(c = \sup S\) 矛盾（\(c\) 应为 \(S\) 的上界），故 \(f(c) \ge \mu\)。

**步骤 4**：故 \(f(c) = \mu\)。若 \(c = a\) 或 \(c = b\) 则 \(f(c)\) 为端点值，与 \(\mu\) 介于两端点值之间矛盾，故 \(c \in (a,b)\)。

---

## 关键步骤说明

- **思路**：用确界构造 \(c\)，再由连续性推出 \(f(c) = \mu\)。
- **技巧**：\(S\) 的构造保证 \(f(c) \le \mu\)；反证 \(f(c) < \mu\) 得出 \(c\) 非上确界。
- **易错点**：需说明 \(c \in (a,b)\)，否则只能得到 \(f(c) = \mu\) 在 \([a,b]\) 上。
