## 题目

设 \(X,Y\) 独立同分布服从 \(\mathrm{Exp}(1)\)。求 \(\mathbb{E}[X \mid X+Y]\)。

---

## 完整解答

由对称性 \(\mathbb{E}[X \mid X+Y] = \mathbb{E}[Y \mid X+Y]\)，故 \(\mathbb{E}[X \mid X+Y] = (X+Y)/2\)（因 \(X+Y\) 可测，且 \(X+Y = \mathbb{E}[X+Y \mid X+Y]\)）。
