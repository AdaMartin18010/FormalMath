## 题目

求初值问题 \(y' + y = x\)，\(y(0)=1\) 的解。

---

## 完整解答（工作示例）

**步骤 1**：齐次解 \(y'+y=0\) 得 \(y_h = C e^{-x}\)。特解可设 \(y_p = ax+b\)，代入得 \(a + ax + b = x\)，故 \(a=1\)，\(a+b=0\) 即 \(b=-1\)，\(y_p = x-1\)。通解 \(y = C e^{-x} + x - 1\)。

**步骤 2**：由 \(y(0)=1\) 得 \(C - 1 = 1\)，\(C=2\)。故解为 \(y = 2 e^{-x} + x - 1\)。

---

## 关键步骤说明

- **一阶线性**：通解 = 齐次通解 + 特解；齐次用分离变量，特解可用待定系数。
- **初值**：代入初值确定常数 \(C\)。
