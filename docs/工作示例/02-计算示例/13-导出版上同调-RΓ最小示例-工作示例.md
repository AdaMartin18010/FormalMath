## 目的

用最简情形看清 **RΓ** 与经典 **H^i** 的对应，降低认知负荷；便于与 Stacks / Hartshorne 讲义例题对照。

---

## 设定

- **X** = Spec(k)（一点概形，k 为域）。
- **F** = k̃（常层：F(X)=k）。
- **Γ(X,-)** = 取全局截面，故 Γ(X,F)=k。

---

## 完整解答（最小可运行示例）

**步骤 1**：经典层上同调 H^i(X,F)。

- k̃ 在一点概形上为内射层（一点上 Abel 层均内射），故可取内射消解 0 → k̃ → 0 → …。
- 因此 **H^0(X,k̃)=k**，**H^{>0}(X,k̃)=0**。

**步骤 2**：导出版 RΓ(X,F)。

- RΓ(X,k̃) ∈ D^+(Ab) 拟同构于「复形仅在第 0 位为 k、其余为 0」的对象，即 **k[0]**。
- 故 **H^i(RΓ(X,k̃)) = H^i(k[0])**：i=0 时为 k，否则为 0，与 H^i(X,k̃) 一致。

**步骤 3**：对应关系小结。

- **RΓ** 把层 F 变成 D^+(Ab) 中一个对象。
- 「取第 i 个上同调」**H^i** 得到的就是经典 **H^i(X,F)**。
- 在仿射概形上对拟凝聚层，常有 RΓ 拟同构于「集中在 degree 0」的复形（消失定理）。

---

## 关键步骤说明

| 步骤 | 要点 |
|------|------|
| 经典 H^i | 内射消解 → Γ(X,-) 作用 → 取上同调 |
| RΓ | 层 → 导出范畴中对象；H^i(RΓ(F))=H^i(X,F) |
| 一点概形 | 计算 trivial，便于第一次建立 RΓ 与 H^i 的对应 |
