## 题目

设 \(f(x,y) = x^2 y + e^x \sin y\)，求 \(\frac{\partial f}{\partial x}\)、\(\frac{\partial f}{\partial y}\) 在 \((0,0)\) 处的值。

---

## 完整解答（工作示例）

**步骤 1**：对 \(x\) 求偏导时视 \(y\) 为常数：
\[
\frac{\partial f}{\partial x} = 2xy + e^x \sin y.
\]
故 \(\frac{\partial f}{\partial x}\big|_{(0,0)} = 0 + 1\cdot 0 = 0\)。

**步骤 2**：对 \(y\) 求偏导时视 \(x\) 为常数：
\[
\frac{\partial f}{\partial y} = x^2 + e^x \cos y.
\]
故 \(\frac{\partial f}{\partial y}\big|_{(0,0)} = 0 + 1\cdot 1 = 1\)。

---

## 关键步骤说明

- **偏导数**：\(\frac{\partial f}{\partial x}\) 为将 \(y\) 固定后对 \(x\) 的导数。
- **几何意义**：表示曲面在坐标方向上的切线斜率。
