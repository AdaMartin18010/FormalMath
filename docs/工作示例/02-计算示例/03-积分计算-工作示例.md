## 题目

求 \(\displaystyle \int_0^1 x e^{x^2} \, dx\)。

---

## 完整解答（工作示例）

**步骤 1**：令 \(u = x^2\)，则 \(du = 2x\,dx\)，即 \(x\,dx = \frac{1}{2} du\)。当 \(x = 0\) 时 \(u = 0\)；当 \(x = 1\) 时 \(u = 1\)。

**步骤 2**：代入。
\[
\int_0^1 x e^{x^2} \, dx = \int_0^1 e^u \cdot \frac{1}{2} du = \frac{1}{2} \int_0^1 e^u \, du = \frac{1}{2} \bigl[ e^u \bigr]_0^1 = \frac{1}{2}(e - 1).
\]

**步骤 3**：答案 \(\displaystyle \frac{e-1}{2}\)。

---

## 关键步骤说明

- **换元**：\(x e^{x^2}\) 中 \(x\,dx\) 与 \(d(x^2)\) 成比例，故设 \(u = x^2\)。
- **上下限**：换元后积分限随之变为 \(u\) 的 0 与 1。
- **易错点**：不要漏写 \(\frac{1}{2}\) 或忘记改积分限。
