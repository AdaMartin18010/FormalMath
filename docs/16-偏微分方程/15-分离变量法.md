# 分离变量法

**将PDE分解为ODE — Fourier、Bessel与Legendre的协奏曲**

---

## 1. 基本思想

### 1.1 方法概述

**分离变量法**假设解可写为单变量函数的乘积：
$$u(x,t) = X(x)T(t)$$
或高维情形
$$u(x_1, \dots, x_n) = X_1(x_1) \cdots X_n(x_n)$$

代入PDE，若方程与边界条件允许，可将PDE分解为若干常微分方程（ODE）。

### 1.2 适用条件

分离变量法适用于：
- 线性齐次方程；
- 边界条件为齐次Dirichlet/Neumann/Robin；
- 区域具有坐标对称性（矩形、圆、球、柱等）。

---

## 2. Sturm-Liouville理论

### 2.1 正则Sturm-Liouville问题

**Sturm-Liouville问题**：求 $\lambda$ 与非零 $X(x)$ 满足
$$-(p(x)X')' + q(x)X = \lambda w(x)X, \quad a < x < b$$
配以边界条件（如 $X(a) = X(b) = 0$ 或更一般的分离边界条件）。

假设 $p, w > 0$，$p, q, w$ 在 $[a,b]$ 上连续。

### 2.2 谱定理

**定理**：正则Sturm-Liouville问题有：
1. 离散谱 $\lambda_1 < \lambda_2 < \dots \to \infty$；
2. 特征函数 $\{X_n\}$ 构成 $L^2([a,b], wdx)$ 的完备正交基；
3. $X_n$ 在 $(a,b)$ 内恰有 $n-1$ 个零点。

---

## 3. 一维热方程

### 3.1 分离变量

对 $u_t = u_{xx}$，$x \in (0,L)$，$u(0,t) = u(L,t) = 0$，设 $u(x,t) = X(x)T(t)$。

代入得
$$\frac{T'}{T} = \frac{X''}{X} = -\lambda$$

$X$ 满足S-L问题
$$X'' + \lambda X = 0, \quad X(0) = X(L) = 0$$
解得 $\lambda_n = (n\pi/L)^2$，$X_n(x) = \sin(n\pi x/L)$。

$T_n(t) = e^{-(n\pi/L)^2 t}$。

### 3.2 Fourier级数解

通解为
$$u(x,t) = \sum_{n=1}^\infty b_n e^{-(n\pi/L)^2 t} \sin\frac{n\pi x}{L}$$

由初值 $u(x,0) = f(x)$，
$$b_n = \frac{2}{L}\int_0^L f(x) \sin\frac{n\pi x}{L} dx$$

**例1**：$f(x) = x(L-x)$。计算
$$b_n = \frac{2}{L}\int_0^L x(L-x)\sin\frac{n\pi x}{L} dx = \frac{4L^2}{n^3\pi^3}(1 - (-1)^n)$$
故
$$u(x,t) = \sum_{n=1,\text{odd}}^\infty \frac{8L^2}{n^3\pi^3} e^{-(n\pi/L)^2 t} \sin\frac{n\pi x}{L}$$

## 4. 二维Laplace方程

### 4.1 矩形区域

对 $\Delta u = 0$ 在 $(0,a) \times (0,b)$，$u=0$ 在三边，$u(x,b) = f(x)$。

设 $u(x,y) = X(x)Y(y)$，得
$$\frac{X''}{X} = -\frac{Y''}{Y} = -\lambda$$

$X_n(x) = \sin(n\pi x/a)$，$\lambda_n = (n\pi/a)^2$。
$Y_n(y) = \sinh(n\pi y/a)$（满足 $Y(0)=0$）。

解为
$$u(x,y) = \sum_{n=1}^\infty c_n \sin\frac{n\pi x}{a} \sinh\frac{n\pi y}{a}$$
由 $u(x,b) = f(x)$ 定 $c_n$。

### 4.2 圆盘

对圆盘 $r < R$，极坐标下 $\Delta u = u_{rr} + \frac{1}{r}u_r + \frac{1}{r^2}u_{\theta\theta} = 0$。

设 $u(r,\theta) = R(r)\Theta(\theta)$，得
$$\frac{r^2 R'' + rR'}{R} = -\frac{\Theta''}{\Theta} = \lambda$$

周期性要求 $\lambda = n^2$（$n = 0,1,2,\dots$），$\Theta_n = \cos(n\theta), \sin(n\theta)$。

$R$ 满足Euler方程 $r^2R'' + rR' - n^2R = 0$，解 $R_n = r^n$（有界）或 $r^{-n}$（舍去）。

解为Poisson积分或Fourier级数
$$u(r,\theta) = a_0 + \sum_{n=1}^\infty r^n(a_n \cos n\theta + b_n \sin n\theta)$$

**例2**：Dirichlet数据 $u(R,\theta) = \sin(2\theta)$。则 $u(r,\theta) = (r/R)^2 \sin(2\theta)$。

---

## 5. 波动方程的分离变量

### 5.1 一维有界弦

$u_{tt} = c^2 u_{xx}$，$u(0,t) = u(L,t) = 0$。

分离得 $X_n = \sin(n\pi x/L)$，$T_n(t) = A_n \cos(\omega_n t) + B_n \sin(\omega_n t)$，$\omega_n = cn\pi/L$。

解为模态叠加
$$u(x,t) = \sum_{n=1}^\infty \sin\frac{n\pi x}{L}\left(A_n \cos\omega_n t + B_n \sin\omega_n t\right)$$

这是**驻波**（standing wave），每个模态在固定位置振动。

### 5.2 膜振动

对矩形膜 $(0,a) \times (0,b)$，$u_{tt} = c^2 \Delta u$，$u=0$ 在边界。

分离得特征值 $\lambda_{mn} = \pi^2(m^2/a^2 + n^2/b^2)$，特征函数 $\sin(m\pi x/a)\sin(n\pi y/b)$。

基频为 $c\pi\sqrt{1/a^2 + 1/b^2}$。泛音不一定是基频的整数倍（非谐和性），这解释了鼓声与弦乐音色差异。

---

## 6. 高维与特殊函数

### 6.1 球坐标

三维Laplace方程球坐标分离得**球谐函数** $Y_l^m(\theta,\varphi)$ 与径向方程
$$(r^2 R')' - l(l+1)R = 0$$
解为 $R = r^l$ 或 $r^{-(l+1)}$。

### 6.2 柱坐标

柱坐标分离给出**Bessel函数** $J_n(kr)$ 与修正Bessel函数。圆盘上的热方程/波动方程涉及Bessel函数的零点作为特征值。

**例3（圆膜振动）**：圆形鼓膜的特征频率由 $J_n$ 的零点 $j_{n,m}$ 给出：$\omega_{n,m} = c j_{n,m}/R$。

---

## 7. 分离变量与谱理论

### 7.1 统一框架

分离变量法的本质是寻找Laplace算子（或更一般的椭圆算子）在特定区域上的**谱分解**。特征函数的正交完备性保证了Fourier展开的合法性。

### 7.2 局限性

- 只适用于线性方程；
- 只适用于具有对称性的区域；
- 对复杂边界或非齐次方程，解析解可能不可得。

现代替代方法：有限元、谱方法、边界元等数值方法。

---

## 8. 小结

分离变量法是求解线性PDE最经典也最优美的方法，它将偏微分方程的复杂性分解为常微分方程与代数本征值问题。Sturm-Liouville理论为这一方法提供了严格的数学基础，Fourier级数、Bessel函数与球谐函数则是其自然的产物。从弦的振动到鼓的共鸣，从静电势到热传导，分离变量法不仅给出显式解，更揭示了物理系统的模态结构与谱分解的普适语言。尽管在复杂几何与现代非线性问题中力有不逮，其思想——将问题约化到本征方向——仍是数学物理中最根本的策略之一。
