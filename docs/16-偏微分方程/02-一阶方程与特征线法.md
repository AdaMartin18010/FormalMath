# 一阶方程与特征线法

**沿特征线约化 — 从ODE到PDE的桥梁**

---

## 1. 线性一阶方程

### 1.1 特征线法

考虑**线性输运方程**
$$a(x,y) u_x + b(x,y) u_y = c(x,y)u + d(x,y)$$
其中 $a, b, c, d$ 为已知光滑函数。

**特征曲线**是向量场 $(a,b)$ 的积分曲线，满足ODE系统
$$\frac{dx}{dt} = a(x,y), \quad \frac{dy}{dt} = b(x,y)$$
沿特征曲线，$u$ 满足ODE
$$\frac{du}{dt} = c(x(t),y(t))u + d(x(t),y(t))$$

**定理**：若初值曲线 $\Gamma$ 处处非特征（即 $(a,b)$ 不切于 $\Gamma$），则在 $\Gamma$ 附近存在唯一解。

*证明*：由ODE存在唯一性定理，特征曲线覆盖 $\Gamma$ 邻域。沿每条特征，$u$ 满足线性ODE，初值由 $\Gamma$ 上数据给出。由于特征不重叠（局部），解唯一确定。$\square$

### 1.2 例子：常系数输运

**例1**：$u_t + c u_x = 0$（$c$ 为常数）。特征线为 $x - ct = \text{const}$。解为 $u(x,t) = f(x - ct)$，即初值 $u(x,0) = f(x)$ 以速度 $c$ 传播。

**验证**：$u_t = -cf'(x-ct)$，$u_x = f'(x-ct)$，故 $u_t + cu_x = 0$。

**例2**：$u_t + c u_x = g(x,t)$。沿特征线 $x = x_0 + ct$，
$$\frac{d}{dt}u(x_0 + ct, t) = g(x_0 + ct, t)$$
积分得
$$u(x,t) = f(x - ct) + \int_0^t g(x - c(t-s), s) ds$$

### 1.3 一般线性方程

对 $n$ 个空间变量，
$$\partial_t u + \sum_{i=1}^n b_i(x,t) \partial_{x_i} u + c(x,t)u = f(x,t)$$
特征曲线为 $(x(t), t)$ 满足 $\frac{dx_i}{dt} = b_i(x(t),t)$。沿特征，$u$ 满足
$$\frac{du}{dt} + c(x(t),t)u = f(x(t),t)$$

## 2. 拟线性一阶方程

### 2.1 Burgers型方程

考虑**拟线性方程**
$$u_t + a(u) u_x = 0$$
特征线方程为
$$\frac{dx}{dt} = a(u(x,t)), \quad \frac{du}{dt} = 0$$

故 $u$ 沿特征线为常数，特征线为直线 $x = x_0 + a(f(x_0))t$（$f(x) = u(x,0)$）。

**例3（Burgers方程 $u_t + uu_x = 0$）**：特征线斜率为 $f(x_0)$。若 $f$ 递减（如 $f(x) = -x$），特征线相交，导致**梯度爆破**（gradient blow-up）。

### 2.2 首次积分法

对一般一阶方程 $F(x,y,u,u_x,u_y) = 0$，**Lagrange-Charpit方法**通过寻找首次积分（沿特征为常数的函数）来构造全积分。

对两个自变量的完全非线性方程 $F(x,y,u,p,q) = 0$（$p=u_x, q=u_y$），特征方程为
$$\frac{dx}{F_p} = \frac{dy}{F_q} = \frac{du}{pF_p + qF_q} = -\frac{dp}{F_x + pF_u} = -\frac{dq}{F_y + qF_u}$$

## 3. 守恒律与弱解

### 3.1 积分形式守恒律

许多物理定律以**守恒律**表述：
$$\partial_t u + \partial_x F(u) = 0$$
其中 $F(u)$ 为**通量**（flux）。对光滑解，展开得拟线性方程
$$u_t + F'(u) u_x = 0$$

### 3.2 弱解的定义

当经典解不存在（特征线相交），需引入**弱解**。

**定义**：$u \in L^\infty_{\text{loc}}$ 称为守恒律的**弱解**，若对任意测试函数 $\varphi \in C_c^\infty(\mathbb{R} \times [0,\infty))$，
$$\int_0^\infty \int_{-\infty}^\infty \left(u \varphi_t + F(u) \varphi_x\right) dx dt + \int_{-\infty}^\infty u(x,0)\varphi(x,0) dx = 0$$

弱解允许间断，且满足**Rankine-Hugoniot跳跃条件**。

### 3.3 Rankine-Hugoniot条件

设解沿曲线 $x = s(t)$ 有跳跃间断，左侧极限 $u_-$，右侧极限 $u_+$。**激波速度** $\dot{s}$ 满足
$$\dot{s} = \frac{F(u_+) - F(u_-)}{u_+ - u_-}$$

*推导*：在 $[a,b] \times [t_1,t_2]$ 上积分守恒律，令区间缩至激波，由Gauss定理得跳跃条件。$\square$

**例4（Burgers方程的激波）**：$F(u) = u^2/2$。若 $u_- = 1, u_+ = 0$，则 $\dot{s} = (0 - 1/2)/(0 - 1) = 1/2$。激波以速度 $1/2$ 传播。

**例5（Buckley-Leverett方程）**：多孔介质中流体的驱替由 $F(u) = u^2/(u^2 + a(1-u)^2)$ 描述，产生复合激波结构。

## 4. 熵条件与唯一性

### 4.1 非唯一性问题

弱解不唯一。例如Burgers方程初值 $u(x,0) = \begin{cases} 0 & x < 0 \\ 1 & x > 0 \end{cases}$ 可有：
- **激波解**：$u(x,t) = \begin{cases} 0 & x < t/2 \\ 1 & x > t/2 \end{cases}$
- **稀疏波**（rarefaction wave）：$u(x,t) = \begin{cases} 0 & x < 0 \\ x/t & 0 < x < t \\ 1 & x > t \end{cases}$

两者均为弱解，但物理上稀疏波更符合耗散极限。

### 4.2 Lax熵条件

**Lax熵条件**（对凸通量 $F'' > 0$）：激波为**可接受的**，若
$$F'(u_-) > \dot{s} > F'(u_+)$$
即特征线"进入"激波。

对Burgers方程，$F'(u) = u$，故要求 $u_- > \dot{s} > u_+$。

**Oleinik熵条件**（对非凸通量）：要求
$$\frac{F(u) - F(u_-)}{u - u_-} \geq \dot{s} \geq \frac{F(u) - F(u_+)}{u - u_+}$$
对所有 $u$ 在 $u_+$ 与 $u_-$ 之间。

### 4.3 熵解的等价刻画

**Kružkov定理**：标量守恒律存在唯一的**熵解**，满足对任意凸熵对 $(\eta, q)$（$\eta'' \geq 0$，$q' = \eta' F'$），在分布意义下
$$\partial_t \eta(u) + \partial_x q(u) \leq 0$$

对Burgers方程，取 $\eta(u) = u^2/2$，则 $q(u) = u^3/3$，熵不等式给出
$$\partial_t(u^2/2) + \partial_x(u^3/3) \leq 0$$

**例6（物理熵）**：对气体动力学，熵在激波处增加，对应热力学第二定律。

## 5. Hamilton-Jacobi方程

### 5.1 特征线法

**Hamilton-Jacobi方程**
$$\partial_t u + H(x, \nabla u) = 0$$
的特征方程为Hamilton系统
$$\dot{x} = \nabla_p H, \quad \dot{p} = -\nabla_x H, \quad \dot{z} = p \cdot \nabla_p H - H$$
其中 $p = \nabla u$，$z = u$。

**例7（Eikonal方程）**：$H(x,p) = |p|$，描述几何光学中的波前传播。特征线为光线，满足 $\dot{x} = p/|p|$。

### 5.2 粘性解

Hamilton-Jacobi方程一般不存在经典全局解。**粘性解**（viscosity solution）由Crandall-Lions引入，通过比较原理定义：

$u$ 为粘性解，若对任意光滑测试函数 $\varphi$，当 $u - \varphi$ 在 $x_0$ 取局部极大时，$\partial_t \varphi + H(x_0, \nabla \varphi) \leq 0$；取局部极小时 $\geq 0$。

**定理**：有界连续初值的Hamilton-Jacobi方程存在唯一的粘性解。

### 5.3 Lax-Oleinik公式

对凸Hamiltonian $H(p)$，Cauchy问题的粘性解有显式表示
$$u(x,t) = \inf_y \left\{g(y) + t H^*\left(\frac{x-y}{t}\right)\right\}$$
其中 $H^*$ 为Legendre-Fenchel变换（凸共轭）。

---

## 6. 高维守恒律

### 6.1 多维特征

对 $u_t + \sum_i F_i(u)_{x_i} = 0$，特征方向由 $F'(u)$ 决定。激波为 $n$ 维空间中的超曲面，Rankine-Hugoniot条件为
$$\dot{s} [u] = [F(u)] \cdot \nu$$
其中 $\nu$ 为激波法向，$[\cdot]$ 为跳跃。

### 6.2 系统守恒律

对方程组 $u_t + F(u)_x = 0$（$u \in \mathbb{R}^m$），特征值为 $F'(u)$ 的特征值 $\lambda_1 < \dots < \lambda_m$。

- ** genuinely nonlinear**：$\nabla \lambda_k \cdot r_k \neq 0$（$r_k$ 为右特征向量）
- **线性退化**：$\nabla \lambda_k \cdot r_k = 0$

Lax熵条件要求激波两侧的特征进入激波。

---

## 7. 小结

一阶PDE的特征线法将偏微分方程约化为常微分方程组，是理解信息传播路径的基本工具。拟线性方程的特征线可能相交导致激波，守恒律的弱解框架与Rankine-Hugoniot条件描述了间断解的物理规律，而熵条件则从不唯一的弱解中筛选出物理可接受的解。Hamilton-Jacobi方程的特征方程揭示了几何光学与经典力学的深刻联系，粘性解理论则为完全非线性一阶方程提供了稳健的解概念。从交通流到气体动力学，从最优控制到微分博弈，一阶PDE的特征线方法与弱解理论构成了应用数学中不可或缺的基石。
