# 几何态射与逻辑态射

## 引言

Topos 之间的映射有两种主要类型：**几何态射**（geometric morphism）和**逻辑态射**（logical morphism）。几何态射保持几何结构，对应于连续映射或概形态射的推广；逻辑态射保持逻辑结构，对应于模型之间的初等嵌入。理解这两种态射的区别与联系，是掌握 Topos 理论的关键。

本教程将系统介绍几何态射、逻辑态射、Topos 的点以及分类 Topos 的概念。

---

## 一、几何态射

### 1.1 定义回顾

设 $\mathcal{E}, \mathcal{F}$ 是 Grothendieck Topos。一个**几何态射** $f: \mathcal{E} \to \mathcal{F}$ 是一对伴随函子：

$$f^*: \mathcal{F} \rightleftarrows \mathcal{E}: f_*$$

满足：
- $f^*$ 是 $f_*$ 的左伴随；
- $f^*$ 保持有限极限（即它是**左正合的**）。

### 1.2 例子

**拓扑空间**：若 $g: X \to Y$ 是拓扑空间的连续映射，则诱导几何态射 $g: \mathbf{Sh}(X) \to \mathbf{Sh}(Y)$，其中：
- $g_*$ 是层的直接像；
- $g^*$ 是层的逆像。

**常值层**：$\mathbf{Set} \to \mathbf{Sh}(X)$ 的几何态射对应于 $X$ 上的常值层：
- $f^*(S) = \underline{S}$（$S$ 上的常值层）；
- $f_*(\mathcal{F}) = \Gamma(X, \mathcal{F})$。

**代数几何**：étale 位型之间的态射诱导几何态射。例如，概形态射 $f: X \to Y$ 诱导 $f: \mathbf{Sh}(X_{\text{ét}}) \to \mathbf{Sh}(Y_{\text{ét}})$。

### 1.3 点的几何态射

Topos $\mathcal{E}$ 的一个**点**（point）是几何态射 $p: \mathbf{Set} \to \mathcal{E}$。它由一对函子 $p^* \dashv p_*$ 组成：
- $p^*: \mathcal{E} \to \mathbf{Set}$ 是**茎函子**（stalk functor），它将层映到其在点 $p$ 上的茎；
- $p_*: \mathbf{Set} \to \mathcal{E}$ 是** skyscraper 层**函子。

在拓扑空间 $X$ 上，点 $x \in X$ 对应于点 $p_x: \mathbf{Set} \to \mathbf{Sh}(X)$，其中 $p_x^*(\mathcal{F}) = \mathcal{F}_x$。但一般 Topos 可能有"非经典"的点，即不来自拓扑空间中的点。

## 二、逻辑态射

### 2.1 定义

设 $\mathcal{E}, \mathcal{F}$ 是 Topos。一个**逻辑态射** $f: \mathcal{E} \to \mathcal{F}$ 是一个保持所有有限极限和幂对象的函子 $f^*: \mathcal{E} \to \mathcal{F}$。

等价地，逻辑态射保持：
- 子对象分类器：$f^*(\Omega_\mathcal{E}) \cong \Omega_\mathcal{F}$；
- 幂对象：$f^*(P(A)) \cong P(f^*(A))$；
- 有限极限和所有余极限。

### 2.2 与几何态射的区别

| 性质 | 几何态射 | 逻辑态射 |
|------|---------|---------|
| 保持有限极限 | 是 | 是 |
| 保持所有余极限 | 不一定 | 是 |
| 保持幂对象 | 不一定 | 是 |
| 保持子对象分类器 | 不一定 | 是 |
| 保持几何逻辑 | 是 | 是 |
| 保持高阶逻辑 | 不一定 | 是 |

### 2.3 例子

- **等价**：Topos 等价是逻辑态射。
- **超幂构造**：在模型理论中，超积构造可以给出逻辑态射。
- **层化**：预层范畴到层范畴的层化函子 $a: \mathbf{PSh}(\mathcal{C}) \to \mathbf{Sh}(\mathcal{C}, J)$ 不是逻辑态射（一般不保持幂对象），但左伴随 $i: \mathbf{Sh}(\mathcal{C}, J) \to \mathbf{PSh}(\mathcal{C})$ 的复合构成几何态射。

## 三、分类 Topos

### 3.1 几何理论的模型

设 $\mathbb{T}$ 是一个几何理论。对任意 Topos $\mathcal{E}$，可以谈论 $\mathbb{T}$ 在 $\mathcal{E}$ 中的**模型**。这些模型构成的范畴记为 $\mathbf{Mod}(\mathbb{T}, \mathcal{E})$。

### 3.2 分类 Topos 的存在性

**定理**（Joyal, Reyes）：对任意几何理论 $\mathbb{T}$，存在唯一的（在同构意义下）Grothendieck Topos $\mathcal{B}(\mathbb{T})$，称为 $\mathbb{T}$ 的**分类 Topos**（classifying topos），使得对任意 Topos $\mathcal{E}$，有自然等价：

$$\text{Geom}(\mathcal{E}, \mathcal{B}(\mathbb{T})) \cong \mathbf{Mod}(\mathbb{T}, \mathcal{E})$$

换言之，到分类 Topos 的几何态射一一对应于目标 Topos 中的模型。

### 3.3 例子

- **对象分类子**：分类"单类型对象"理论的 Topos 是 $\mathbf{PSh}(\mathbf{Fin}^{\text{op}})$，即有限集的预层范畴。到它的几何态射对应于 Topos 中的对象。
- **环分类子**：分类交换环理论的 Topos 是 $\mathbf{Sh}(\mathbf{CRing}^{\text{op}}, \text{Zariski})$，即交换环 Zariski 位型上的层范畴。这就是为什么 Zariski 层天然带有环结构！
- **局部环分类子**：在环分类子上施加"局部环"几何理论，得到局部环的分类 Topos。

## 四、应用：代数几何中的平展 Topos

### 4.1 平展几何态射

设 $f: X \to Y$ 是概形的平展态射。则诱导的几何态射 $f: \mathbf{Sh}(X_{\text{ét}}) \to \mathbf{Sh}(Y_{\text{ét}})$ 具有特殊性质：$f^*$ 不仅是左正合的，而且是一个**等价**的局部化。这类几何态射称为**平展几何态射**（étale geometric morphism），是研究平展上同调和基本群的核心。

### 4.2 Galois 理论与基本群

设 $X$ 是连通概形，$\bar{x}$ 是几何点。$X$ 的**平展基本群** $\pi_1^{\text{ét}}(X, \bar{x})$ 可以定义为：有限平展覆叠范畴（对应于有限平展几何态射）在该点上的纤维函子的自同构群。

这与 Topos 理论中的**Galois Topos**概念完全一致：有限平展覆叠的范畴等价于一个 profinite 群上的有限连续集范畴。

## 五、习题

设 $f: X \to Y$ 是连续映射。证明诱导的几何态射 $f: \mathbf{Sh}(X) \to \mathbf{Sh}(Y)$ 中，$f_*$ 确实是 $f^*$ 的右伴随。

### 习题 2
证明：层化函子 $a: \mathbf{PSh}(\mathcal{C}) \to \mathbf{Sh}(\mathcal{C}, J)$ 一般不保持子对象分类器，因此不是逻辑态射。

### 习题 3
