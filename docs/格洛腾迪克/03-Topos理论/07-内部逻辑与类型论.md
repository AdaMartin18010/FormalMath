# 内部逻辑与类型论

## 引言

Grothendieck Topos 不仅是一个几何对象，也是一个构造性集合论的模型。自 1970 年代以来，数学家逐渐发现 Topos 理论与 **Martin-Löf 类型论**（Martin-Löf type theory）以及后来的**同伦类型论**（Homotopy Type Theory, HoTT）之间存在着深刻的联系。Topos 提供了一种"语义"，使得类型论中的命题可以在具体的几何或计算环境中得到解释。

本教程将介绍 Topos 内部逻辑与类型论的基本联系，以及这一联系对现代数学的影响。

---

## 一、Martin-Löf 类型论回顾

### 1.1 类型作为命题

Martin-Löf 类型论的核心理念是**命题即类型**（propositions as types）：
- **类型** $A$ = 命题
- **项** $a: A$ = 命题 $A$ 的证明
- **函数类型** $A \to B$ = 蕴涵 $A \Rightarrow B$
- **积类型** $A \times B$ = 合取 $A \wedge B$
- **和类型** $A + B$ = 析取 $A \vee B$
- **依赖积** $\prod_{x:A} B(x)$ = 全称量词 $\forall x:A. B(x)$
- **依赖和** $\sum_{x:A} B(x)$ = 存在量词 $\exists x:A. B(x)$

### 1.2 身份类型

对类型 $A$ 和项 $a, b: A$，**身份类型**（identity type）$\text{Id}_A(a, b)$ 表示 "$a$ 和 $b$ 相等"的命题。在经典类型论中，$\text{Id}_A(a, b)$ 要么是可证明的（当 $a = b$ 时），要么不可证明。

但在**同伦类型论**中，$\text{Id}_A(a, b)$ 可以有多个不同的证明，这些证明本身构成一个"空间"——这使得类型对应于 **$\infty$-群胚**（$\infty$-groupoid）。

## 二、Topos 中的类型论语义

### 2.1 类型 = 对象

在 Grothendieck Topos $\mathcal{E}$ 中，可以解释 Martin-Löf 类型论：
- **类型** = $\mathcal{E}$ 中的对象 $A$；
- **项** $a: A$ = 态射 $a: 1 \to A$（整体元素）或更一般地 $a: U \to A$（广义元素）；
- **函数类型** $A \to B$ = 指数对象 $B^A$；
- **积类型** = 范畴积 $A \times B$；
- **和类型** = 范畴余积 $A + B$；
- **依赖积** = 局部积（local product，即右伴随于 pullback 的函子）；
- **依赖和** = 局部余积（local coproduct，即 pullback 本身）。

### 2.2 身份类型的解释

在一般 Topos 中，身份类型 $\text{Id}_A(a, b)$ 解释为**等化子**：

$$\text{Eq}(a, b) = \{(x, y) \in A \times A \mid x = y\} \times_{A \times A} 1$$

即 $a$ 和 $b$ 相等的"真值"。在 $\mathbf{Set}$ 中，这退化为 $
\{\star \mid a = b\}$，即要么是真（单点集），要么是假（空集）。

### 2.3 构造性选择公理

在 Topos 中，**选择公理**一般不成立。具体来说，满射 $e: A \twoheadrightarrow B$ 不一定有截面 $s: B \to A$ 使得 $e \circ s = \text{id}_B$。这与构造性数学和计算理论的要求一致：不能从一个存在性命题中提取构造，除非有具体的算法。

## 三、同伦类型论与 $\infty$-Topos

### 3.1 从 Topos 到 $\infty$-Topos

同伦类型论中的类型对应于 $\infty$-群胚。这启发数学家发展 **$\infty$-Topos** 理论：它是 Grothendieck Topos 的高阶推广，其中的"对象"不再是集合或层，而是 **$\infty$-群胚**或**单纯集合**。

### 3.2 Lurie 的理论

Jacob Lurie 在《Higher Topos Theory》中系统发展了 $\infty$-Topos 的理论。一个 $(\infty, 1)$-Topos 是满足 Giraud 公理的 $\infty$-范畴，其中的对象具有高阶同伦结构。

**基本对应**：
| 概念 | 经典数学 | 同伦类型论 | $\infty$-Topos |
|------|---------|-----------|---------------|
| 类型 | 集合 | 类型 | 对象 |
| 相等 | 命题 | 身份类型 | 道路空间 |
| 高阶相等 | 平凡 | 高阶身份类型 | 高阶同伦群 |
| 逻辑 | 经典/直觉主义 | 构造性 | 构造性 |

### 3.3 未解问题：初始猜想

Vladimir Voevodsky 提出了**初始猜想**（Initiality Conjecture）：语法上定义的同伦类型论是某个 $\infty$-Topos 的初始对象。这一定理若成立，将给出 HoTT 语义学的完整基础。虽然该猜想的弱化版本已被证明，但完整版本仍是活跃的研究领域。

## 四、可实现性 Topos 与计算

### 4.1 可实现性 Topos

**可实现性 Topos**（realizability topos）是构造性数学和计算理论中的重要例子。在这个 Topos 中：
- 对象由"可实现集"组成；
- 态射是可计算函数；
- 内部逻辑对应于可实现性逻辑（realizability logic）。

### 4.2 有效 Topos

**有效 Topos**（effective topos）是最著名的可实现性 Topos。它由 Hyland 引入，其中每个命题都有"可实现数"（realizer）作为证明的编码。这使得有效 Topos 成为研究构造性分析和递归数学的理想模型。

## 五、习题

### 习题 1
在 $\mathbf{Set}$ 中，解释 Martin-Löf 类型论的"命题即类型"对应：函数类型 $A \to B$ 如何对应于逻辑蕴涵？

### 习题 2
设 $\mathcal{E} = \mathbf{Sh}(X)$。描述类型 $\mathbb{N}$（自然数类型）在这个 Topos 中的解释。

### 习题 3
解释为什么在一般 Topos 中，选择公理不成立，并给出一个层 Topos 中的反例。

### 习题 4
设 $A$ 是类型（对象），$a, b: A$。在同伦类型论中，$\text{Id}_A(a, b)$ 可能有多个不同的元素。这在 $\infty$-Topos 中对应于什么几何现象？
