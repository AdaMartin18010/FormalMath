## 一、定义-定理-证明规范

### 1.1 环境顺序与格式

```

标准环境顺序
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

定义 (Definition)
目的: 精确说明数学概念
格式: 定义号 + 被定义术语 + 定义内容
示例:
定义 2.1 (群). 群是一个集合 G 配备二元运算 ·
满足以下公理: ...

定理 (Theorem)
目的: 陈述重要数学结果
格式: 定理号 + 定理名称(可选) + 定理陈述
示例:
定理 3.2 (Lagrange). 若 H 是有限群 G 的子群,
则 |H| 整除 |G|。

引理 (Lemma)
目的: 辅助定理的预备结果
格式: 同定理，但重要性较低

推论 (Corollary)
目的: 从定理直接得到的结果
格式: 简短陈述，常引用定理

命题 (Proposition)
目的: 独立但不太重要的结果
格式: 同定理

证明 (Proof)
目的: 严格证明定理/引理/命题
格式: 以 □ (证毕符号) 结束
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

### 1.2 证明写作要点

```

证明写作的黄金法则
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 清晰的分段
   - 每个主要步骤独立成段
   - 关键断言用子引理

2. 逻辑连接词
   使用: 因此, 故, 从而, 于是
   避免: 口语化连接

3. 正向 vs 反向
   正向: 从假设推导结论
   反向: 明确标注"反设"

4. 证明技巧标注
   - "使用归纳法..."
   - "反设结论不成立..."
   - "考虑以下两种情况..."

5. 引用标注
   - 引用方程: 由 (3.2) 得...
   - 引用定理: 根据定理 2.1...
   - 引用定义: 由定义可知...

示例证明结构:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
证明. (可选: 证明策略概述)

(步骤1). 初始推导...
由假设...可得...

(步骤2). 关键步骤...
应用引理 3.4 得...

(步骤3). 结论...
因此...证毕。 □
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

## 二、符号与记号规范

### 2.1 符号命名原则

| 类别 | 推荐符号 | 避免使用 |
|-----|---------|---------|
| 集合 | A, B, C, X, Y | a, b, c (小写) |
| 元素 | a, b, x, y | A, B (大写) |
| 函数 | f, g, h, φ, ψ | 容易混淆的符号 |
| 数域 | ℝ, ℂ, ℚ, ℤ, ℕ | R, C, Q, Z, N (非黑板粗体) |
| 向量 | v, w, u, x | 单字母易混淆 |
| 矩阵 | A, B, M | 避免与集合混淆 |
| 算子 | T, L, A | 具体说明类型 |

### 2.2 常用记号规范

```

标准记号使用
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

集合论:
• 属于: x ∈ A
• 包含: A ⊆ B (子集), A ⊂ B (真子集)
• 并交: A ∪ B, A ∩ B
• 补集: A^c 或 X\A
• 笛卡尔积: A × B
• 幂集: P(A) 或 2^A

逻辑:
• 蕴含: A ⟹ B
• 等价: A ⟺ B
• 存在: ∃
• 任意: ∀
• 使得: s.t. 或 |

函数:
• 映射: f: X → Y
• 像: f(A), f⁻¹(B)
• 复合: g ∘ f
• 限制: f|_A

分析:
• 极限: lim_{x→a} f(x)
• 导数: f', df/dx, ∂f/∂x
• 积分: ∫_a^b f(x)dx
• 求和: Σ_{i=1}^n a_i
• 乘积: Π_{i=1}^n a_i

线性代数:
• 矩阵: (a_{ij}) 或 [a_{ij}]
• 转置: A^T
• 行列式: det(A) 或 |A|
• 范数: ||x||

• 内积: ⟨x, y⟩ 或 x·y
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

## 三、数学写作风格

### 3.1 语言风格指南

```

数学写作的DO和DON'T
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

DO (推荐):
• 使用主动语态 (当主语明确时)
• 精确表述: "存在一个唯一的..."
• 逐步展开: 从简单到复杂
• 提供直观解释
• 使用"我们"包含读者

DON'T (避免):
• 模糊表述: "显然", "不难看出"
  (如果显然，不需要写；如果不显然，解释清楚)
• 过度使用被动语态
• 符号滥用: 一个符号多个含义
• 过长的句子 (>3行)
• 口语化表达

示例对比:
❌ 不难看出这个结果是正确的。
✓ 由引理 3.2 和三角不等式，我们有...

❌ 我们做以下假设。
✓ 假设以下条件成立:
  (H1) ...
  (H2) ...
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

### 3.2 常见句式模板

**引入定义:**

- "我们称...为..."
- "...的定义如下:"
- "令...表示..."

**引入定理:**

- "本节的主要结果如下:"
- "我们有以下定理:"
- "下列结果是本文的核心:"

**证明开始:**

- "证明."
- "证明: 首先..."
- "我们使用归纳法证明..."

**推导过程:**

- "由...可得..."
- "因此,..."
- "从而,..."
- "注意到,..."

**证明结束:**

- "证毕。"
- "□"
- "这就完成了证明。"
