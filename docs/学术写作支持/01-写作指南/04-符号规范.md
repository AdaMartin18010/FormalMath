## 一、字体规范

### 1.1 LaTeX字体命令

```
数学字体规范 (LaTeX)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

类别          LaTeX命令      示例          使用场景
────────────────────────────────────────────────
变量          默认斜体       $x, y, n$     变量名
向量          \mathbf        $\mathbf{v}$  向量
矩阵          大写斜体       $A, B$        矩阵
集合          黑板粗体       $\mathbb{R}$  数域
算子          \mathcal       $\mathcal{L}$ 算子
特殊常数      正体           $\mathrm{e}$  指数常数
微分          正体           $\mathrm{d}x$ 微分符号
描述文字      \text          $\text{原}$   文字标注

示例:
变量: $x, y, z, t$ (斜体)
向量: $\mathbf{x}, \mathbf{v}, \mathbf{w}$ (粗体)
矩阵: $A, B, M$ (大写斜体)
数域: $\mathbb{R}, \mathbb{C}, \mathbb{Q}$ (黑板粗体)
函数空间: $\mathcal{C}, \mathcal{L}$ (花体)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 1.2 正体 vs 斜体

| 使用正体 | 使用斜体 |
|---------|---------|
| 微分 d: dx, dy | 变量: x, y |
| 常数: e, i, π | 函数: f, g |
| 算子名: sin, cos, log | 集合: A, B |
| 描述词: max, min | 几何点: P, Q |

## 三、符号使用原则

### 3.1 符号选择原则

```
符号选择原则
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 一致性
   • 同一符号在同一文档中含义唯一
   • 标准符号优先 (如 ℝ 表示实数)

2. 可读性
   • 避免易混淆符号 (l, 1, I, |)
   • 避免过多上下标

3. 习惯性
   • 遵循学科惯例
   • 群: G, H
   • 环: R, S
   • 域: F, K
   • 向量空间: V, W

4. 系统性
   • 相关概念使用相关符号
   • 对偶对象: V, V*
   • 子对象: H ≤ G
   • 商对象: G/N

5. 简洁性
   • 使用标准缩写
   • 复杂表达式定义中间符号
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
