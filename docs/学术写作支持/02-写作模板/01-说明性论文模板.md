# 说明性论文模板 (5页)

**制定日期**: 2026年4月2日
**页数**: 5页
**用途**: 解释数学概念或小结果

---

```markdown

**作者**: [姓名]

[1-2段，200词以内。说明: (1)本文讨论的主题，(2)主要结果或观点，(3)意义]

[引用或陈述必要的已知结果]
```

## 使用说明

### 页数分配

| 部分 | 页数 | 说明 |
|-----|------|------|
| 标题/摘要 | 0.5页 | 简洁明了 |
| 引言 | 1页 | 背景+问题 |
| 预备知识 | 1页 | 必要的定义 |
| 主要结果 | 2页 | 核心内容 |
| 讨论 | 0.5页 | 总结展望 |

### 写作要点

1. **聚焦**: 5页限制要求内容聚焦，避免枝节
2. **清晰**: 面向非专家读者，解释充分
3. **精确**: 数学陈述严格准确
4. **简洁**: 每词必争，避免冗余
