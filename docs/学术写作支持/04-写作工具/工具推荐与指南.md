## 一、LaTeX排版系统

### 1.1 LaTeX发行版

| 工具 | 平台 | 特点 | 推荐度 |
|-----|------|------|--------|
| TeX Live | Win/Mac/Linux | 最完整，持续更新 | ★★★★★ |
| MiKTeX | Windows | 按需安装包 | ★★★★ |
| MacTeX | Mac | Mac优化版TeX Live | ★★★★★ |

### 1.2 LaTeX编辑器

| 工具 | 平台 | 特点 | 适用 |
|-----|------|------|------|
| VS Code + LaTeX Workshop | 全平台 | 功能强大，可扩展 | 进阶用户 |
| TeXstudio | 全平台 | 专为LaTeX设计，易用 | 初学者 |
| Overleaf | 在线 | 实时协作，模板丰富 | 团队协作 |
| TeXShop | Mac | Mac原生，简洁 | Mac用户 |

### 1.3 常用宏包

```latex
% 数学写作必备宏包
\documentclass{amsart}  % AMS文章类

\usepackage{amsmath,amssymb,amsthm}  % AMS数学包
\usepackage{geometry}                % 页面设置
\usepackage{graphicx}                % 图片支持
\usepackage{hyperref}                % 超链接
\usepackage{tikz}                    % 绘图
\usepackage{cleveref}                % 智能引用

% 定理环境设置
\theoremstyle{plain}
\newtheorem{theorem}{Theorem}
\newtheorem{lemma}[theorem]{Lemma}
\newtheorem{proposition}[theorem]{Proposition}

\theoremstyle{definition}
\newtheorem{definition}[theorem]{Definition}

\theoremstyle{remark}
\newtheorem{remark}[theorem]{Remark}

```

## 二、文献管理工具

### 2.1 Zotero

**特点**:

- 免费开源
- 浏览器插件一键保存
- 支持多种引用格式
- 与Word/LaTeX集成

**基本使用**:

```

1. 安装Zotero和浏览器插件
2. 浏览文献时点击插件保存
3. 在Zotero中组织分类
4. 写作时插入引用
5. 自动生成参考文献列表

```

### 2.2 JabRef

**特点**:

- 专为BibTeX设计
- LaTeX用户首选
- 支持分组和搜索

### 2.3 Mendeley

**特点**:

- Elsevier出品
- PDF阅读和批注
- 社交功能
- 跨平台同步

## 三、在线协作工具

### 3.1 Overleaf

**功能**:

- 在线LaTeX编辑
- 实时协作
- 版本历史
- 丰富模板库

**适用场景**:

- 导师-学生协作
- 多作者论文
- 远程团队合作

### 3.2 Git + GitHub

**用途**:

- 版本控制
- 论文版本管理
- 备份与同步

**基本流程**:

```bash
git init
git add paper.tex
git commit -m "Initial version"
git push origin main

```

## 五、推荐工作流

### 初学者工作流

```

TeX Live + TeXstudio → Zotero → Grammarly

```

### 进阶用户工作流

```

VS Code + LaTeX Workshop → Git → Overleaf (协作)

```

### 团队协作工作流

```

Overleaf (主文档) + Zotero (文献) + Git (版本备份)

```
