# 镜像对称

## 📋 概述

镜像对称是弦理论中两个不同Calabi-Yau流形之间的对偶性，它预言了数学上看似无关的量之间的深刻联系。这一猜想由物理学家发现，后来成为代数几何和辛几何研究的核心问题之一。镜像对称连接了复几何和辛几何，体现了现代数学物理中物理直觉与严格数学证明的深刻互动。

**核心思想**: 两个Calabi-Yau流形 X 和 X̂ 是镜像的，如果IIB弦在 X 上的物理等价于IIA弦在 X̂ 上的物理。

---

## 📝 基础概念

### 1.1 镜像对称的陈述

**定义 1.1** (镜像对)
两个Calabi-Yau三folds X 和 X̂ 构成**镜像对**，如果：

$$h^{1,1}(X) = h^{2,1}(X̂), h^{2,1}(X) = h^{1,1}(X̂)$$

**Hodge数的交换**:

| 量 | X (IIB) | X̂ (IIA) |
|----|---------|----------|
| 复结构模 | h^{2,1} | h^{1,1} |
| Kähler模 | h^{1,1} | h^{2,1} |

---

### 1.2 SYZ猜想

**定义 1.2** (特殊拉格朗日纤维)
CY_3 的**SYZ纤维**是 f: X → B，纤维是特殊拉格朗日三维环面。

**猜想 1.1** (SYZ, 1996)
镜像对 (X, X̂) 有对偶的SYZ纤维化，通过T-对偶联系：

- 纤维方向T-对偶
- 基 B 保持不变

**T-对偶**: 在每个三维环面上对半径 R 和 1/R 对偶。

---

### 1.3 同源镜像对称

**定义 1.3** (同源)
两个流形的**同源**（homological mirror symmetry）是指：
$$D^b(X) ≅ Fuk(X̂)$$

其中：

- D^b(X): X 上的有界导出范畴（复几何）
- Fuk(X̂): X̂ 上的Fukaya范畴（辛几何）

**Kontsevich猜想** (1994): 镜像对称等价于导出范畴的等价。

## 🎯 核心定理

### 2.1 Gromov-Witten不变量

**定义 2.1** (Gromov-Witten不变量)
计数以特定拓扑类型映入 X 的全纯曲线数。

**生成函数**:
$$F_0(t) = Σ_{β∈H_2(X)} N_β q^β, q = e^{2πit}$$

其中 N_β 是GW不变量。

**镜像映射**: t 是复化Kähler参数。

---

### 2.2 周期的镜像对称

**定理 2.1** (周期的变换)
X 上的周期积分（复结构模空间）对应于 X̂ 上的GW生成函数。

**Picard-Fuchs方程**: 周期满足线性微分方程。

**例子** (quintic):
quintic的镜像是orbifold X^5/Z_5^3，其周期给出quintic的GW不变量。

---

### 2.3 枚举几何的预测

**定理 2.2** (Candelas-de la Ossa-Green-Parkes, 1991)
镜像对称预言了quintic CY_3 上度数为 d 的有理曲线的数目 n_d：

**预言验证**:

- n_1 = 2875 （已知）
- n_2 = 609250 （后来数学证明）
- n_3 = 317206375

## 💡 实战问题

### 问题1：quintic的镜像

**问题**: 描述quintic CY_3 的镜像构造。

**解答**:

**构造步骤**:

1. 取 P^4 中的五次超曲面族：
   X_ψ: z_0^5 + z_1^5 + z_2^5 + z_3^5 + z_4^5 - 5ψ z_0z_1z_2z_3z_4 = 0

2. 除以 (Z_5)^3 对称性（对角scale）

3. 解消奇点得到光滑流形 X̂

**镜像性质**:

- h^{1,1}(X̂) = 101 = h^{2,1}(X)
- h^{2,1}(X̂) = 1 = h^{1,1}(X)

**周期计算**: X̂ 的周期由超几何函数给出，给出 X 的GW不变量。

---

### 问题2：T-对偶

**问题**: 解释弦理论中的T-对偶及其与镜像对称的关系。

**解答**:

**T-对偶**: 在半径 R 的圆上，弦理论与在半径 α'/R 的圆上等价。

**动量和缠绕**:

- 动量模式: p = n/R （Kaluza-Klein）
- 缠绕模式: w = mR/α' （弦特有）

**对偶交换**: R ↔ α'/R 交换动量和缠绕量子数。

**SYZ对偶**:

- IIA弦在 X 上 ↔ IIB弦在 X̂ 上
- 纤维方向T-对偶
- 基 B 保持不变

这是镜像对称的几何解释。

---

### 问题3：导出范畴

**问题**: 描述同源镜像对称中导出范畴与Fukaya范畴的对应。

**解答**:

**D^b(X)** (有界导出范畴):

- 对象是复形的链
- 态射是链映射的等价类
- BPS D-膜由复形描述

**Fuk(X̂)** (Fukaya范畴):

- 对象是特殊拉格朗日子流形
- 态射是Floer同调
- A-无穷结构由全纯圆盘计数

**镜像对应**:

| D^b(X) | Fuk(X̂) |
|--------|--------|
| 点（skyscraper层） | 特殊拉格朗日环面纤维 |
| 矢量丛 | 特殊拉格朗日子流形 |
| Ext群 | Floer同调 |

Kontsevich猜想说这两个范畴是等价的（作为三角范畴）。
