# Calabi-Yau紧化

**创建日期**: 2026年4月4日
**最后更新**: 2026年4月4日
**版本**: v1.0
**状态**: 完成

---

## 📋 概述

超弦理论要求十维时空，而现实世界看起来是四维的。Calabi-Yau紧化提供了一种将额外六维卷曲成紧致流形的方法，同时保持部分超对称性。Calabi-Yau流形具有丰富的数学结构，是代数几何、复几何和数学物理交汇的核心研究对象。

**核心思想**: 额外维度紧致化在Calabi-Yau流形上，保留超对称性，得到四维有效理论。

---

## 📝 基础概念

### 1.1 Calabi-Yau流形

**定义 1.1** (Calabi-Yau流形)
**Calabi-Yau n-折**是满足以下条件的紧Kähler流形 X：

1. 第一陈类为零：c_1(X) = 0
2. 典范丛平凡：K_X = O_X
3. 和乐群为 SU(n)

**物理维数**: 弦理论中 n = 3（复三维，实六维）。

**拓扑不变量**:

- Hodge数: h^{p,q} = dim H^{p,q}(X)
- Euler示性数: χ(X) = Σ(-1)^{p+q}h^{p,q}

---

### 1.2 Kähler几何

**定义 1.2** (Kähler度量)
**Kähler度量**满足 dω = 0，其中Kähler形式：
$$ω = ig_{ij̄}dz^i ∧ dz̄^j$$

**Ricci形式**:
$$R = iR_{ij̄}dz^i ∧ dz̄^j$$

**Calabi-Yau条件**: c_1 = 0 ⇔ 存在Ricci平坦Kähler度量。

---

### 1.3 超对称性与紧化

**定义 1.3** (超对称性生成元)
超对称变换由旋量参数 ε 生成。在紧化后，保留的超对称性数目取决于 ε 在Calabi-Yau上的协变常数解。

**定理 1.1** (超对称性与和乐群)
在 CY_3 紧化后，保留 N=2 超对称性（II型弦）或 N=1（杂弦）。

---

## 🎯 核心定理

### 2.1 Calabi-Yau定理

**定理 2.1** (Yau, 1978)
设 (X, ω_0) 是紧Kähler流形，c_1(X) = 0。则存在唯一的Ricci平坦Kähler度量 g，满足 [ω] = [ω_0]。

**等价表述**: Monge-Ampère方程：
$$det(g_{ij̄} + ∂_i∂_{j̄}φ) = e^f det(g_{ij̄})$$

有唯一解（规范意义下）。

**意义**: 证明了Calabi猜想，为弦理论提供了紧化解的数学保证。

---

### 2.2 Hodge钻石

**定理 2.2** (Calabi-Yau的Hodge数)
CY_3 的Hodge数满足对称结构：

- h^{0,0} = h^{3,3} = 1
- h^{1,1} = h^{2,2}
- h^{3,0} = h^{0,3} = 1
- h^{2,1} = h^{1,2}

**物理意义**:

- h^{1,1}: Kähler模数目（决定大小和形状）
- h^{2,1}: 复结构模数目

**Euler示性数**: χ = 2(h^{1,1} - h^{2,1})

---

### 2.3 有效场论

**定理 2.3** (四维有效理论)
CY_3 紧化后，四维有效理论是 N=2 超引力（IIA/IIB）或 N=1 超引力（杂弦）。

**无质量谱**:

- 引力子多重态
- 矢量多重态: h^{1,1} 个（IIA）或 h^{2,1} 个（IIB）
- 超多重态: h^{2,1} 个（IIA）或 h^{1,1} 个（IIB）

## 💡 实战问题

### 问题1：quintic Calabi-Yau

**问题**: 描述五次超曲面 quintic CY_3 的基本性质。

**解答**:

**定义**: P^4 中的五次超曲面：
$$X = {[z_0:...:z_4] ∈ P^4 : z_0^5 + z_1^5 + z_2^5 + z_3^5 + z_4^5 = 0}$$

**Hodge数**:

- h^{1,1} = 1（来自超平面类）
- h^{2,1} = 101（由形变计算）
- χ = 2(1 - 101) = -200

**模空间**:

- Kähler模: 1维（体积）
- 复结构模: 101维

这是最简明的 CY_3 例子。

---

### 问题2：orbifold紧化

**问题**: 描述 T^6/Z_3 orbifold紧化。

**解答**:

**orbifold构造**:
取六维环面 T^6 = T^2 × T^2 × T^2，除以 Z_3 作用：
$$z_i → e^{2πi/3}z_i, i = 1, 2, 3$$

**奇点**: 固定点处有 A_2 型奇点（27个）。

**blowing up**: 用例外除子解消奇点，得到光滑 CY_3。

**Hodge数**: h^{1,1} = 36，h^{2,1} = 0（刚性）。

**物理**: 杂弦在此紧化后得 N=1 超对称，规范群 E_6 × E_8。

---

### 问题3：特殊拉格朗日子流形

**问题**: 解释Calabi-Yau中的特殊拉格朗日子流形及其物理意义。

**解答**:

**定义**: L ⊂ X 是**特殊拉格朗日子流形**，若：

1. Lagrangian: ω|_L = 0
2. 特殊: Im(e^{-iθ}Ω)|_L = 0，其中 Ω 是全纯体积形式

**物理意义**:

- D-膜可缠绕在SLag上
- 镜像对称中，SLag对应镜像上的相干层
- 稳定SLag对应BPS态

**例子**: T^6 中的三维环面 T^3 是SLag。

SYZ猜想（Strominger-Yau-Zaslow）: 镜像对称是T-对偶，纤维是SLag环面。
