# 弦动力学与共形场论

**创建日期**: 2026年4月4日
**最后更新**: 2026年4月4日
**版本**: v1.0
**状态**: 完成

## 📋 概述

弦理论将基本粒子从点状推广到一维的弦。这一简单的推广带来了深刻的数学和物理后果：弦的世界面是二维流形，自然地与共形场论联系；弦的振动模式对应于不同的粒子；弦理论还预言了引力的量子化。共形场论是弦理论的核心数学结构，也是数学物理中研究最深入的领域之一。

**核心思想**: 弦的世界面是二维共形场论，弦的激发给出粒子的谱。

---

## 📝 基础概念

### 1.1 弦的作用量

**定义 1.1** (Nambu-Goto作用量)
**Nambu-Goto作用量**是弦的世界面面积：
$$S_{NG} = -T\int d\tau d\sigma \sqrt{-\det(\partial_\alpha X^\mu \partial_\beta X^\nu \eta_{\mu\nu})}$$

其中：

- $T = 1/(2\pi\alpha')$ 是弦张力
- $\alpha'$ 是Regge斜率参数
- $X^\mu(\tau, \sigma)$ 是世界面到靶时空的嵌入

**定义 1.2** (Polyakov作用量)
**Polyakov作用量**引入独立的世界面度量 $g_{\alpha\beta}$：
$$S_P = -\frac{T}{2}\int d^2\sigma \sqrt{-g} g^{\alpha\beta}\partial_\alpha X^\mu \partial_\beta X^\nu \eta_{\mu\nu}$$

在经典水平上，与Nambu-Goto作用量等价。

---

### 1.2 共形对称性

**定义 1.3** (共形变换)
**共形变换**是保持度量共形等价的变换：$g'_{\alpha\beta} = e^{2\omega}g_{\alpha\beta}$。

在二维中，共形群是无限维的（Witt代数/Virasoro代数）。

**定义 1.4** (Virasoro代数)
**Virasoro代数**是共形对称性的中心扩张：
$$[L_m, L_n] = (m-n)L_{m+n} + \frac{c}{12}m(m^2-1)\delta_{m+n,0}$$

其中 $c$ 是中心荷。

---

### 1.3 弦的量子化

**定义 1.5** (弦模式展开)
开弦解满足Neumann边界条件，模式展开：
$$X^\mu(\tau, \sigma) = x^\mu + \alpha' p^\mu \tau + i\sqrt{2\alpha'}\sum_{n\neq 0}\frac{\alpha_n^\mu}{n}e^{-in\tau}\cos(n\sigma)$$

对易关系：
$$[\alpha_m^\mu, \alpha_n^\nu] = m\delta_{m+n,0}\eta^{\mu\nu}$$

**Virasoro生成元**:
$$L_m = \frac{1}{2}\sum_{n=-\infty}^{\infty}\alpha_{m-n}\cdot\alpha_n$$

## 🎯 核心定理

### 2.1 临界维数

**定理 2.1** (临界维数)
弦理论仅在特定时空维数下相容：

- 玻色弦: $D = 26$
- 超弦: $D = 10$

**证明概要**:

1. 量子化后，Virasoro代数有异常
2. 物理态条件（$L_n|\psi\rangle = 0$，$n > 0$；$(L_0 - a)|\psi\rangle = 0$）要求异常抵消

3. 计算得 $c = D - 26$（玻色弦），需要 $c = 0$，即 $D = 26$
4. 超弦类似，得 $D = 10$$\square$

---

### 2.2 无鬼定理

**定理 2.2** (无鬼定理)
在 $D = 26$（玻色弦）下，物理Hilbert空间无负范数态（鬼态）。

**证明** (Brower, 1972; Goddard-Thorn, 1972):
利用no-ghost定理，证明在物理态子空间中内积正定。

---

### 2.3 顶点算子

**定义 2.1** (顶点算子)
**顶点算子**对应于弦的激发态：
$$V(k) = :e^{ik\cdot X}:$$

**Tachyon顶点算子**: $V_0(k) = :e^{ik\cdot X}:$

**光子顶点算子**: $V_1(k, \epsilon) = \epsilon_\mu :\partial X^\mu e^{ik\cdot X}:$

**顶点算子代数**: 顶点算子的算子乘积展开（OPE）定义了顶点算子代数结构。

## 💡 实战问题

### 问题1：弦的质量谱

**问题**: 计算玻色弦的闭弦质量谱。

**解答**:

闭弦模式展开（左右 movers）：
$$X^\mu = x^\mu + \alpha' p^\mu \tau + \frac{i\sqrt{\alpha'}}{2}\sum_{n\neq 0}\left(\frac{\alpha_n^\mu}{n}e^{-2in(\tau-\sigma)} + \frac{\tilde{\alpha}_n^\mu}{n}e^{-2in(\tau+\sigma)}\right)$$

质量壳条件：
$$M^2 = \frac{4}{\alpha'}(N + \tilde{N} - 2)$$

其中 $N = \sum_{n=1}^\infty \alpha_{-n}\cdot\alpha_n$ 是数算子。

**质量谱**:

- 基态 ($N = \tilde{N} = 0$): $M^2 = -4/\alpha'$（快子，不稳定）
- 第一激发态 ($N = \tilde{N} = 1$): $M^2 = 0$
  - 无质量态: $g_{\mu\nu}$（引力子），$B_{\mu\nu}$（反对称张量），$\Phi$（伸缩子）

---

### 问题2：Virasoro代数表示

**问题**: 验证 $[L_m, L_n] = (m-n)L_{m+n}$（忽略中心项）。

**解答**:

$$L_m = \frac{1}{2}\sum_k \alpha_{m-k}\cdot\alpha_k$$

$$[L_m, \alpha_n^\mu] = -n\alpha_{m+n}^\mu$$

利用此计算 $[L_m, L_n]$：
$$[L_m, L_n] = \frac{1}{2}\sum_k [L_m, \alpha_{n-k}\cdot\alpha_k]$$
$$= \frac{1}{2}\sum_k \left(-(n-k)\alpha_{m+n-k}\cdot\alpha_k - k\alpha_{n-k}\cdot\alpha_{m+k}\right)$$

重新标记求和指标，得到：
$$= (m-n)L_{m+n} + \text{中心项}$$

中心项来自正规序的调整。$\square$

---

### 问题3：自由费米子CFT

**问题**: 描述自由Majorana费米子的CFT及其中心荷。

**解答**:

作用量：
$$S = \frac{1}{4\pi}\int d^2z \left(\psi\bar{\partial}\psi + \bar{\psi}\partial\bar{\psi}\right)$$

OPE：
$$\psi(z)\psi(w) = \frac{1}{z-w} + \cdots$$

能量-动量张量：
$$T(z) = -\frac{1}{2}:\psi\partial\psi:$$

中心荷计算：
$$\langle T(z)T(w)\rangle = \frac{c/2}{(z-w)^4}$$

对Majorana费米子，$c = 1/2$。

超弦要求 $c = 10$（5对左右费米子 + 共形鬼）。$\square$
