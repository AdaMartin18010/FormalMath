# 微分几何基础

**创建日期**: 2026年4月4日

## 📋 概述

广义相对论将引力描述为时空几何的弯曲。Einstein的洞见是：物质和能量决定时空的度量，而度量决定物质的轨迹。这一理论需要微分几何的语言——流形、张量、联络、曲率。本节建立广义相对论所需的数学基础。

**核心思想**: 引力即几何；时空是四维Lorentz流形，其曲率反映物质-能量分布。

---

## 📝 基础概念

### 1.1 光滑流形

**定义 1.1** (光滑流形)
$n$ 维**光滑流形** $M$ 是局部像 $\mathbb{R}^n$ 的拓扑空间，有光滑的坐标变换。

**切空间**: 点 $p \in M$ 的切空间 $T_pM$ 是过 $p$ 的曲线的速度向量空间。

**余切空间**: $T_p^*M$ 是 $T_pM$ 的对偶空间，包含余向量（1-形式）。

---

### 1.2 张量场

**定义 1.2** (张量)
$(k, l)$ 型**张量**是多线性映射：
$$T: \underbrace{T_p^*M \times \cdots \times T_p^*M}_{k} \times \underbrace{T_pM \times \cdots \times T_pM}_{l} \to \mathbb{R}$$

**例子**:
- $(0,1)$：向量 $V^\mu$
- $(1,0)$：余向量（1-形式）$\omega_\mu$
- $(1,1)$：线性变换 $T^\mu_\nu$
- $(0,2)$：度量 $g_{\mu\nu}$，曲率 $R_{\mu\nu}$
- $(1,3)$：Riemann张量 $R^\rho_{\sigma\mu\nu}$

---

### 1.3 度量与Lorentz几何

**定义 1.3** (度量张量)
**度量**是对称、非退化的 $(0,2)$ 张量场。

**Riemann度量**: 正定（Euclidean符号）
**Lorentz度量**: 符号 $(-, +, +, +)$ 或 $(+, -, -, -)$

**Lorentz流形**: 装备Lorentz度量的流形，描述时空。

---

## 🎯 核心定理

### 2.1 联络与协变导数

**定义 2.1** (Levi-Civita联络)
**Levi-Civita联络**是唯一的无挠、度量相容的联络：
1. **无挠**: $\Gamma^\lambda_{\mu\nu} = \Gamma^\lambda_{\nu\mu}$
2. **度量相容**: $\nabla_\lambda g_{\mu\nu} = 0$

**Christoffel符号**:
$$\Gamma^\lambda_{\mu\nu} = \frac{1}{2}g^{\lambda\sigma}(\partial_\mu g_{\nu\sigma} + \partial_\nu g_{\mu\sigma} - \partial_\sigma g_{\mu\nu})$$

**定理 2.1** (Levi-Civita联络的存在唯一性)
对任意伪Riemann流形，存在唯一的Levi-Civita联络。

---

### 2.2 曲率张量

**定义 2.2** (Riemann曲率张量)
**Riemann张量**度量协变导数的非交换性：
$$[\nabla_\mu, \nabla_\nu]V^\rho = R^\rho_{\sigma\mu\nu}V^\sigma$$

显式公式：
$$R^\rho_{\sigma\mu\nu} = \partial_\mu\Gamma^\rho_{\nu\sigma} - \partial_\nu\Gamma^\rho_{\mu\sigma} + \Gamma^\rho_{\mu\lambda}\Gamma^\lambda_{\nu\sigma} - \Gamma^\rho_{\nu\lambda}\Gamma^\lambda_{\mu\sigma}$$

**Ricci张量**: $R_{\mu\nu} = R^\lambda_{\mu\lambda\nu}$
**标量曲率**: $R = g^{\mu\nu}R_{\mu\nu}$
**Einstein张量**: $G_{\mu\nu} = R_{\mu\nu} - \frac{1}{2}g_{\mu\nu}R$

---

### 2.3 测地线

**定义 2.3** (测地线)
**测地线**是局部极值化固有时的曲线，满足：
$$\frac{d^2x^\mu}{d\tau^2} + \Gamma^\mu_{\nu\lambda}\frac{dx^\nu}{d\tau}\frac{dx^\lambda}{d\tau} = 0$$

**物理解释**: 自由下落粒子沿测地线运动（Einstein等效原理）。

## 💡 实战问题

### 问题1：球面上的曲率

**问题**: 计算二维球面 $S^2$（半径 $R$）的Riemann曲率。

**解答**:

度量（球坐标）：
$$ds^2 = R^2(d\theta^2 + \sin^2\theta \, d\phi^2)$$

非零Christoffel符号：
$$\Gamma^\theta_{\phi\phi} = -\sin\theta\cos\theta, \quad \Gamma^\phi_{\theta\phi} = \cot\theta$$

Riemann张量（2维只有一个独立分量）：
$$R^\theta_{\phi\theta\phi} = \sin^2\theta$$

$$R_{\theta\phi\theta\phi} = g_{\theta\theta}R^\theta_{\phi\theta\phi} = R^2\sin^2\theta$$

标量曲率：
$$R = \frac{2}{R^2}$$

球面有正常数曲率。

---

### 问题2：Schwarzschild度量的Christoffel符号

**问题**: 计算Schwarzschild度量：
$$ds^2 = -\left(1 - \frac{2M}{r}\right)dt^2 + \left(1 - \frac{2M}{r}\right)^{-1}dr^2 + r^2d\Omega^2$$

的非零Christoffel符号。

**解答**:

设 $A(r) = 1 - 2M/r$，则 $g_{tt} = -A$，$g_{rr} = A^{-1}$。

非零Christoffel符号：
$$\Gamma^t_{tr} = \Gamma^t_{rt} = \frac{A'}{2A}$$
$$\Gamma^r_{tt} = \frac{AA'}{2}, \quad \Gamma^r_{rr} = -\frac{A'}{2A}$$
$$\Gamma^r_{\theta\theta} = -rA, \quad \Gamma^r_{\phi\phi} = -rA\sin^2\theta$$
$$\Gamma^\theta_{r\theta} = \Gamma^\theta_{\theta r} = \frac{1}{r}, \quad \Gamma^\theta_{\phi\phi} = -\sin\theta\cos\theta$$
$$\Gamma^\phi_{r\phi} = \Gamma^\phi_{\phi r} = \frac{1}{r}, \quad \Gamma^\phi_{\theta\phi} = \Gamma^\phi_{\phi\theta} = \cot\theta$$

其中 $A' = dA/dr = 2M/r^2$。

---

### 问题3：平行输运与曲率

**问题**: 解释曲率与向量沿闭合曲线平行输运的关系。

**解答**:

向量 $V$ 沿曲线 $\gamma$ 平行输运满足：$\nabla_{\dot{\gamma}}V = 0$。

沿无穷小闭合曲线（由 $u^\mu$ 和 $v^\nu$ 张成）平行输运一周，向量的变化：
$$\delta V^\rho = R^\rho_{\sigma\mu\nu}V^\sigma u^\mu v^\nu$$

这是Riemann张量的几何解释：度量平行输运的路径依赖性。

在平直时空（$R^\rho_{\sigma\mu\nu} = 0$），平行输运与路径无关。$\square$
