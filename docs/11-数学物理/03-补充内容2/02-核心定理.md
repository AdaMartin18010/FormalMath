## 一、Noether定理

### 1.1 对称性与守恒律

**定理 1.1** (Noether定理)
若拉格朗日量在连续变换下不变，则存在相应的守恒量。

**形式表述**: 设单参数变换群 $q_i \mapsto q_i + \epsilon \psi_i(q, \dot{q}, t)$ 保持作用量不变，则：

$$Q = \sum_i p_i \psi_i$$

是运动常数，即 $\frac{dQ}{dt} = 0$。

### 1.2 经典例子

**时间平移不变** $\Rightarrow$ **能量守恒**

拉格朗日量不显含时间：$\partial L/\partial t = 0$

守恒量：$H = \sum_i p_i \dot{q}_i - L = E$

**空间平移不变** $\Rightarrow$ **动量守恒**

拉格朗日量在 $q_i \to q_i + \epsilon$ 下不变：

守恒量：$P = \sum_i p_i$

**空间旋转不变** $\Rightarrow$ **角动量守恒**

拉格朗日量在旋转变换下不变：

守恒量：$L = \sum_i q_i \times p_i$

### 1.3 证明概要

考虑变分 $\delta q_i = \epsilon \psi_i$。

作用量变分：
$$\delta S = \int \sum_i \left(\frac{\partial L}{\partial q_i}\psi_i + \frac{\partial L}{\partial \dot{q}_i}\dot{\psi}_i\right) dt$$

利用Euler-Lagrange方程：
$$= \int \sum_i \left(\frac{d}{dt}\frac{\partial L}{\partial \dot{q}_i}\psi_i + \frac{\partial L}{\partial \dot{q}_i}\dot{\psi}_i\right) dt$$

$$= \int \frac{d}{dt}\left(\sum_i p_i \psi_i\right) dt = 0$$

因此 $\sum_i p_i \psi_i = \text{const}$。∎

## 二、谱定理

### 2.1 自伴算子的谱定理

**定理 2.1** (谱定理 - 有界自伴算子)
设 $A$ 是希尔伯特空间 $\mathcal{H}$ 上的有界自伴算子，则存在唯一的谱测度 $E$ 使得：

$$A = \int_{\sigma(A)} \lambda dE(\lambda)$$

**性质**:

1. $E(\Delta)$ 是投影算子，对每个Borel集 $\Delta \subset \mathbb{R}$
2. $E(\emptyset) = 0$，$E(\mathbb{R}) = I$
3. $E(\Delta_1)E(\Delta_2) = E(\Delta_1 \cap \Delta_2)$
4. $E$ 是单位分解：$\int dE(\lambda) = I$

### 2.2 函数演算

**定理 2.2** (Borel函数演算)
对任意Borel函数 $f$，定义：

$$f(A) = \int_{\sigma(A)} f(\lambda) dE(\lambda)$$

**性质**:

- $(f + g)(A) = f(A) + g(A)$
- $(fg)(A) = f(A)g(A)$
- $\|f(A)\| = \|f\|_{\infty}$

### 2.3 量子力学应用

**时间演化**: $U(t) = e^{-iHt/\hbar} = \int e^{-i\lambda t/\hbar} dE(\lambda)$

**能量测量**: 测得能量在 $[a, b]$ 的概率：

$$P(E \in [a, b]) = \langle \psi | E([a, b]) | \psi \rangle$$

**离散谱情况**: 若 $A$ 只有离散谱 $\{\lambda_n\}$：

$$A = \sum_n \lambda_n P_n$$

其中 $P_n$ 是到特征空间的投影。

## 三、遍历定理

### 3.1 Birkhoff遍历定理

**定理 3.1** (Birkhoff遍历定理)
设 $(X, \mathcal{B}, \mu)$ 是概率空间，$T: X \to X$ 是保测变换。

对任意 $f \in L^1(X, \mu)$：

$$\lim_{n \to \infty} \frac{1}{n}\sum_{k=0}^{n-1} f(T^k x) = \bar{f}(x)$$

几乎处处存在，且 $\bar{f} \in L^1$，$\bar{f} \circ T = \bar{f}$。

若 $T$ 遍历，则 $\bar{f} = \int f d\mu$ a.e.

### 3.2 统计力学意义

**时间平均 = 系综平均** (对遍历系统)

$$\lim_{T \to \infty} \frac{1}{T}\int_0^T f(\phi_t(x)) dt = \int f d\mu$$

这为统计力学的基本假设提供了数学基础。

### 3.3 von Neumann遍历定理

**定理 3.2** (von Neumann遍历定理 - $L^2$版本)
设 $U$ 是希尔伯特空间上的酉算子，$P$ 到不变子空间的投影。

则：

$$\lim_{n \to \infty} \frac{1}{n}\sum_{k=0}^{n-1} U^k = P$$

在强算子拓扑下。

**物理意义**: 长时间平均收敛到平衡态。

## 四、Stone定理

### 4.1 酉群的生成

**定理 4.1** (Stone定理)
设 $U(t)$ 是希尔伯特空间上的强连续单参数酉群，则存在唯一的自伴算子 $A$ 使得：

$$U(t) = e^{iAt}$$

即 $A$ 是酉群的无穷小生成元：

$$iA\psi = \lim_{t \to 0} \frac{U(t)\psi - \psi}{t}$$

### 4.2 量子动力学

**定理 4.2** (薛定谔方程的解)
设 $H$ 是自伴哈密顿算子，则初值问题：

$$i\hbar \frac{d}{dt}\psi(t) = H\psi(t), \quad \psi(0) = \psi_0$$

有唯一解：

$$\psi(t) = e^{-iHt/\hbar}\psi_0$$

且 $\|\psi(t)\| = \|\psi_0\|$（概率守恒）。

### 4.3 海森堡绘景

**定理 4.3** (海森堡运动方程)
在海森堡绘景中，可观测量随时间演化：

$$A(t) = e^{iHt/\hbar} A e^{-iHt/\hbar}$$

满足：

$$\frac{dA(t)}{dt} = \frac{i}{\hbar}[H, A(t)]$$

这与经典力学的刘维尔方程 $\frac{df}{dt} = \{H, f\}$ 对应（量子化：$\{,\} \to \frac{1}{i\hbar}[,]$）。

## 总结

本章节介绍了物理数学的核心定理：

- **Noether定理**: 对称性与守恒律的一一对应
- **谱定理**: 自伴算子的对角化，量子测量的数学基础
- **遍历定理**: 时间平均与系综平均的等价性
- **Stone定理**: 酉群与自伴算子的对应，量子动力学基础

这些定理揭示了物理定律背后的深刻数学结构。
