## 问题1：拉格朗日力学与双摆系统 ⭐⭐⭐☆☆

### 📝 问题描述

**双摆系统**（经典混沌系统）：

- 摆长 $l_1 = l_2 = 1$ m
- 质量 $m_1 = m_2 = 1$ kg
- 重力加速度 $g = 9.8$ m/s²

**广义坐标**: $\theta_1, \theta_2$（两摆与竖直方向的夹角）

**任务**:
1. 推导系统的动能和势能表达式
2. 建立拉格朗日函数
3. 导出Euler-Lagrange运动方程
4. 数值模拟小角度和大角度初始条件的运动
5. 分析系统的混沌行为

---

### ✅ 解答

#### 步骤1: 位置和速度

**第一摆（上摆）**:
$$x_1 = l_1 \sin\theta_1, \quad y_1 = -l_1 \cos\theta_1$$
$$\dot{x}_1 = l_1\dot{\theta}_1\cos\theta_1, \quad \dot{y}_1 = l_1\dot{\theta}_1\sin\theta_1$$

**第二摆（下摆）**:
$$x_2 = l_1\sin\theta_1 + l_2\sin\theta_2$$
$$y_2 = -l_1\cos\theta_1 - l_2\cos\theta_2$$

速度分量：
$$\dot{x}_2 = l_1\dot{\theta}_1\cos\theta_1 + l_2\dot{\theta}_2\cos\theta_2$$
$$\dot{y}_2 = l_1\dot{\theta}_1\sin\theta_1 + l_2\dot{\theta}_2\sin\theta_2$$

#### 步骤2: 动能和势能

**动能**:
$$T_1 = \frac{1}{2}m_1 l_1^2 \dot{\theta}_1^2$$

$$T_2 = \frac{1}{2}m_2\left[l_1^2\dot{\theta}_1^2 + l_2^2\dot{\theta}_2^2 + 2l_1 l_2\dot{\theta}_1\dot{\theta}_2\cos(\theta_1 - \theta_2)\right]$$

**势能**:
$$V_1 = -m_1 g l_1 \cos\theta_1$$
$$V_2 = -m_2 g (l_1 \cos\theta_1 + l_2 \cos\theta_2)$$

#### 步骤3: 拉格朗日函数

$$L = T - V = T_1 + T_2 - V_1 - V_2$$

展开：
$$L = \frac{1}{2}(m_1+m_2)l_1^2\dot{\theta}_1^2 + \frac{1}{2}m_2l_2^2\dot{\theta}_2^2 + m_2 l_1 l_2 \dot{\theta}_1\dot{\theta}_2\cos(\theta_1-\theta_2)$$
$$+ (m_1+m_2)g l_1 \cos\theta_1 + m_2 g l_2 \cos\theta_2$$

#### 步骤4: Euler-Lagrange方程

对 $\theta_1$:
$$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{\theta}_1}\right) - \frac{\partial L}{\partial \theta_1} = 0$$

计算得：
$$(m_1+m_2)l_1\ddot{\theta}_1 + m_2 l_2\ddot{\theta}_2\cos(\theta_1-\theta_2) + m_2 l_2\dot{\theta}_2^2\sin(\theta_1-\theta_2) + (m_1+m_2)g\sin\theta_1 = 0$$

对 $\theta_2$:
$$m_2 l_2\ddot{\theta}_2 + m_2 l_1\ddot{\theta}_1\cos(\theta_1-\theta_2) - m_2 l_1\dot{\theta}_1^2\sin(\theta_1-\theta_2) + m_2 g\sin\theta_2 = 0$$

#### 步骤5: 小角度线性化

当 $\theta_1, \theta_2 \ll 1$：

$$(m_1+m_2)l_1\ddot{\theta}_1 + m_2 l_2\ddot{\theta}_2 + (m_1+m_2)g\theta_1 = 0$$
$$l_1\ddot{\theta}_1 + l_2\ddot{\theta}_2 + g\theta_2 = 0$$

矩阵形式（设 $m_1=m_2=m, l_1=l_2=l$）：
$$\begin{pmatrix} 2 & 1 \\ 1 & 1 \end{pmatrix}\begin{pmatrix} \ddot{\theta}_1 \\ \ddot{\theta}_2 \end{pmatrix} + \frac{g}{l}\begin{pmatrix} 2 & 0 \\ 0 & 1 \end{pmatrix}\begin{pmatrix} \theta_1 \\ \theta_2 \end{pmatrix} = 0$$

**简正模式频率**（设 $\omega_0^2 = g/l$）：
$$\omega_1 = \sqrt{2-\sqrt{2}}\omega_0 \approx 0.54\omega_0$$
$$\omega_2 = \sqrt{2+\sqrt{2}}\omega_0 \approx 1.85\omega_0$$

import numpy as np
from scipy.integrate import odeint
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation

def double_pendulum(y, t, m1, m2, l1, l2, g):
    """
    双摆运动方程
    y = [theta1, theta2, omega1, omega2]
    """
    theta1, theta2, omega1, omega2 = y
    
    # 计算系数矩阵
    delta = theta1 - theta2
    den1 = (m1 + m2) * l1
    den2 = m2 * l2
    
    # 简化的运动方程（解耦后的形式）
    num1 = (-m2 * l1 * omega1**2 * np.sin(delta) * np.cos(delta)
            - m2 * l2 * omega2**2 * np.sin(delta)
            - (m1 + m2) * g * np.sin(theta1))
    
    num2 = (m2 * l2 * omega2**2 * np.sin(delta) * np.cos(delta)
            + (m1 + m2) * l1 * omega1**2 * np.sin(delta)
            - (m1 + m2) * g * np.sin(theta2))
    
    # 求解角加速度
    cos_delta = np.cos(delta)
    sin_delta = np.sin(delta)
    
    # 更准确的实现
    f1 = -(m2 * l1 * omega1**2 * sin_delta * cos_delta
           + m2 * l2 * omega2**2 * sin_delta
           + (m1 + m2) * g * np.sin(theta1))
    
    f2 = (m2 * l2 * omega2**2 * sin_delta * cos_delta
          + (m1 + m2) * l1 * omega1**2 * sin_delta
          - (m1 + m2) * g * np.sin(theta2) * cos_delta)
    
    # 简化的实现（来自标准推导）
    alpha1 = -(m2 * l2 * omega2**2 * np.sin(delta) 
               + (m1 + m2) * g * np.sin(theta1))
    alpha2 = l1 * omega1**2 * np.sin(delta) - g * np.sin(theta2)
    
    M11 = (m1 + m2) * l1
    M12 = m2 * l2 * np.cos(delta)
    M21 = l1 * np.cos(delta)
    M22 = l2
    
    det = M11 * M22 - M12 * M21
    
    alpha_theta1 = (M22 * alpha1 - M12 * alpha2) / det
    alpha_theta2 = (-M21 * alpha1 + M11 * alpha2) / det
    
    return [omega1, omega2, alpha_theta1, alpha_theta2]

def simulate_double_pendulum(theta1_0, theta2_0, omega1_0=0, omega2_0=0, 
                              t_max=10, dt=0.01):
    """模拟双摆系统"""
    m1, m2 = 1.0, 1.0
    l1, l2 = 1.0, 1.0
    g = 9.8
    
    t = np.arange(0, t_max, dt)
    y0 = [theta1_0, theta2_0, omega1_0, omega2_0]
    
    solution = odeint(double_pendulum, y0, t, args=(m1, m2, l1, l2, g))
    
    return t, solution

# 模拟两种初始条件
# 1. 小角度（线性区域）
t1, sol1 = simulate_double_pendulum(np.pi/10, np.pi/10, t_max=20)

# 2. 大角度（混沌区域）
t2, sol2 = simulate_double_pendulum(np.pi/2, np.pi/2, t_max=20)

# 3. 敏感依赖（混沌特征）
t3, sol3a = simulate_double_pendulum(np.pi/2, np.pi/2, t_max=20)
t3, sol3b = simulate_double_pendulum(np.pi/2 + 0.001, np.pi/2, t_max=20)

# 可视化
fig, axes = plt.subplots(2, 2, figsize=(14, 12))

# 1. 小角度时间序列
axes[0, 0].plot(t1, sol1[:, 0], 'b-', label=r'$\theta_1$')
axes[0, 0].plot(t1, sol1[:, 1], 'r-', label=r'$\theta_2$')
axes[0, 0].set_xlabel('时间 (s)')
axes[0, 0].set_ylabel('角度 (rad)')
axes[0, 0].set_title('小角度初始条件 (规则运动)')
axes[0, 0].legend()
axes[0, 0].grid(True)

# 2. 大角度时间序列
axes[0, 1].plot(t2, sol2[:, 0], 'b-', label=r'$\theta_1$', alpha=0.7)
axes[0, 1].plot(t2, sol2[:, 1], 'r-', label=r'$\theta_2$', alpha=0.7)
axes[0, 1].set_xlabel('时间 (s)')
axes[0, 1].set_ylabel('角度 (rad)')
axes[0, 1].set_title('大角度初始条件 (混沌运动)')
axes[0, 1].legend()

# 3. 相空间轨迹
axes[1, 0].plot(sol2[:, 0], sol2[:, 2], 'b-', alpha=0.5, linewidth=0.5)

# 4. 敏感依赖（Lyapunov特征）

## 问题3：量子谐振子 ⭐⭐⭐☆☆

### 📝 问题描述

一维量子谐振子哈密顿量：
$$\hat{H} = \frac{\hat{p}^2}{2m} + \frac{1}{2}m\omega^2\hat{x}^2$$

**任务**:
1. 引入升降算子并对角化哈密顿量
2. 推导能级公式和本征态
3. 计算位置算符的矩阵元
4. 可视化前几个本征波函数
5. 计算不确定性关系

---

### ✅ 解答

#### 步骤1: 升降算子

定义：
$$\hat{a} = \sqrt{\frac{m\omega}{2\hbar}}\left(\hat{x} + \frac{i\hat{p}}{m\omega}\right)$$
$$\hat{a}^\dagger = \sqrt{\frac{m\omega}{2\hbar}}\left(\hat{x} - \frac{i\hat{p}}{m\omega}\right)$$

**逆关系**:
$$\hat{x} = \sqrt{\frac{\hbar}{2m\omega}}(\hat{a} + \hat{a}^\dagger)$$
$$\hat{p} = i\sqrt{\frac{\hbar m\omega}{2}}(\hat{a}^\dagger - \hat{a})$$

#### 步骤2: 对易关系与哈密顿量

$$[\hat{a}, \hat{a}^\dagger] = 1$$

哈密顿量改写：
$$\hat{H} = \hbar\omega\left(\hat{a}^\dagger\hat{a} + \frac{1}{2}\right) = \hbar\omega\left(\hat{n} + \frac{1}{2}\right)$$

#### 步骤3: 能级与本征态

设 $|n\rangle$ 满足 $\hat{n}|n\rangle = n|n\rangle$

**能级**:
$$E_n = \hbar\omega\left(n + \frac{1}{2}\right), \quad n = 0, 1, 2, ...$$

**升降算子作用**:
$$\hat{a}|n\rangle = \sqrt{n}|n-1\rangle$$
$$\hat{a}^\dagger|n\rangle = \sqrt{n+1}|n+1\rangle$$

#### 步骤4: 波函数（位置表象）

定义特征长度：$x_0 = \sqrt{\hbar/(m\omega)}$

**基态波函数**:
$$\psi_0(x) = \frac{1}{(\pi x_0^2)^{1/4}} \exp\left(-\frac{x^2}{2x_0^2}\right)$$

**激发态波函数**:
$$\psi_n(x) = \frac{1}{\sqrt{2^n n!}} \left(\frac{x}{x_0} - x_0\frac{d}{dx}\right)^n \psi_0(x)$$

用Hermite多项式表示：
$$\psi_n(x) = \frac{1}{\sqrt{2^n n!}} \left(\frac{1}{\pi x_0^2}\right)^{1/4} H_n\left(\frac{x}{x_0}\right) \exp\left(-\frac{x^2}{2x_0^2}\right)$$

#### 步骤5: 不确定性关系

基态：
$$\langle x \rangle = 0, \quad \langle p \rangle = 0$$
$$\langle x^2 \rangle = \frac{\hbar}{2m\omega}, \quad \langle p^2 \rangle = \frac{\hbar m\omega}{2}$$

$$\Delta x = \sqrt{\frac{\hbar}{2m\omega}}, \quad \Delta p = \sqrt{\frac{\hbar m\omega}{2}}$$

$$\Delta x \cdot \Delta p = \frac{\hbar}{2}$$

达到不确定性原理下限！

### 💻 Python实现与可视化

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.special import hermite, factorial

def quantum_harmonic_oscillator_wavefunction(x, n, x0=1):
    """
    量子谐振子波函数
    
    Args:
        x: 位置数组
        n: 量子数
        x0: 特征长度 sqrt(hbar/(m*omega))
    """
    xi = x / x0
    # Hermite多项式
    H_n = hermite(n)
    normalization = 1 / np.sqrt(2**n * factorial(n)) * (1 / (np.pi * x0**2))**0.25
    psi = normalization * H_n(xi) * np.exp(-xi**2 / 2)
    return psi

def quantum_harmonic_oscillator_energy(n, hbar=1, omega=1):
    """能级"""
    return hbar * omega * (n + 0.5)

# 可视化前5个本征态
x = np.linspace(-6, 6, 1000)
x0 = 1  # 特征长度
n_max = 5

fig, axes = plt.subplots(2, 1, figsize=(12, 10))

# 波函数
axes[0].axhline(y=0, color='k', linewidth=0.5)
colors = plt.cm.viridis(np.linspace(0, 1, n_max))
for n in range(n_max):
    psi = quantum_harmonic_oscillator_wavefunction(x, n, x0)
    E_n = quantum_harmonic_oscillator_energy(n)
    axes[0].plot(x, psi + E_n, color=colors[n], linewidth=2, 
                label=f'n={n}, E={E_n:.1f}ℏω')
    axes[0].axhline(y=E_n, color=colors[n], linestyle='--', alpha=0.3)

axes[0].set_xlabel('位置 x (x₀单位)')
axes[0].set_ylabel('波函数 ψₙ(x) + Eₙ')
axes[0].set_title('量子谐振子本征波函数与能级')
axes[0].legend(loc='upper right')
axes[0].set_xlim(-6, 6)
axes[0].grid(True, alpha=0.3)

# 概率密度
axes[1].axhline(y=0, color='k', linewidth=0.5)
for n in range(n_max):
    psi = quantum_harmonic_oscillator_wavefunction(x, n, x0)
    prob = np.abs(psi)**2
    axes[1].plot(x, prob + n*0.1, color=colors[n], linewidth=2, label=f'n={n}')

axes[1].set_xlabel('位置 x (x₀单位)')
axes[1].set_ylabel('|ψₙ(x)|² (偏移)')
axes[1].set_title('概率密度分布')
axes[1].legend()
axes[1].set_xlim(-6, 6)
axes[1].grid(True, alpha=0.3)

plt.tight_layout()
plt.savefig('quantum_harmonic_oscillator.png', dpi=150)
plt.show()

# 计算不确定性
n = 0  # 基态
psi = quantum_harmonic_oscillator_wavefunction(x, n, x0)
dx = x[1] - x[0]

# 归一化检查
norm = np.trapz(np.abs(psi)**2, x)
print(f"归一化: {norm:.6f}")

# 期望值
x_mean = np.trapz(x * np.abs(psi)**2, x)
x2_mean = np.trapz(x**2 * np.abs(psi)**2, x)

# 动量期望值（数值微分）
dpsi_dx = np.gradient(psi, x)
p_mean = -1j * np.trapz(np.conj(psi) * dpsi_dx, x)  # 应为0
p2_mean = -np.trapz(np.conj(psi) * np.gradient(dpsi_dx, x), x)

dx_val = np.sqrt(x2_mean - x_mean**2)
dp_val = np.sqrt(np.real(p2_mean))

print(f"\n基态不确定性 (ℏ=1, m=1, ω=1):")
print(f"  Δx = {dx_val:.4f} (理论值: {np.sqrt(0.5):.4f})")
print(f"  Δp = {dp_val:.4f} (理论值: {np.sqrt(0.5):.4f})")
print(f"  Δx·Δp = {dx_val * dp_val:.4f} (理论最小值: 0.5)")

# 位置算符矩阵元（选n=0,1,2）
print("\n位置算符矩阵元 <m|x|n>:")
for m in range(3):
    for n in range(3):
        psi_m = quantum_harmonic_oscillator_wavefunction(x, m, x0)
        psi_n = quantum_harmonic_oscillator_wavefunction(x, n, x0)
        x_mn = np.trapz(np.conj(psi_m) * x * psi_n, x)
        print(f"  <{m}|x|{n}> = {x_mn:8.4f}", end="")
    print()
```
