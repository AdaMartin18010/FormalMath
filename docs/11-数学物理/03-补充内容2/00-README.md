# 物理数学 / Mathematical Physics

**最后更新**: 2026年4月4日

---

## 📋 分支概述

物理数学是研究物理问题中的数学结构的学科，运用严格的数学方法分析经典力学、量子力学、统计力学等领域的理论基础。核心内容包括拉格朗日和哈密顿力学、希尔伯特空间中的量子理论、算子谱理论、统计系综理论等。物理数学架起了物理学与纯粹数学之间的桥梁。

## 📚 核心人物

- **Joseph-Louis Lagrange (1736-1813)**: 分析力学奠基人
- **William Rowan Hamilton (1805-1865)**: 哈密顿力学
- **Emmy Noether (1882-1935)**: 对称性与守恒律
- **John von Neumann (1903-1957)**: 量子力学数学基础
- **Paul Dirac (1902-1984)**: 狄拉克方程、量子力学符号体系
- **Ludwig Boltzmann (1844-1906)**: 统计力学奠基人
- **Vladimir Arnold (1937-2010)**: 经典力学数学理论

---

## 🔗 与其他分支的联系

- **分析学**: 泛函分析、微分方程、变分法
- **几何学**: 辛几何、黎曼几何、纤维丛
- **代数学**: 李群李代数、表示论、算子代数
- **概率论**: 随机过程、遍历理论、统计力学
- **拓扑学**: 拓扑不变量、指标定理
- **微分方程**: 可积系统、孤子理论

1. **先修知识**: 经典力学、量子力学基础、泛函分析
2. **第一阶段**: 拉格朗日和哈密顿力学
3. **第二阶段**: 量子力学的希尔伯特空间理论
4. **第三阶段**: 谱理论和散射理论
5. **第四阶段**: 统计力学、场论、可积系统
