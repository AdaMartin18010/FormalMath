# 体（数学）

## 定義

**体** $(K, +, \cdot)$ は、零でないすべての元が乗法の逆元を持つ可換環である：

$$\forall a \in K \setminus \{0\}: \exists a^{-1} \in K \text{ により } a \cdot a^{-1} = 1$$

## 性質

### 標数（特性）
体の**標数**とは、以下が成り立つ最小の正の整数 $p$ である：
$$\underbrace{1 + 1 + \cdots + 1}_{p \text{ 個}} = 0$$

そのような整数が存在しない場合、体の標数は 0 である。

| 標数 | 例 |
|------|-----|
| 0 | $\mathbb{Q}, \mathbb{R}, \mathbb{C}$ |
| $p$（素数）| $\mathbb{F}_p = \mathbb{Z}/p\mathbb{Z}$ |

## 体の拡大

**体の拡大** $L/K$ とは、体の包含 $K \subseteq L$ のことである。

### 拡大次数
**拡大次数** $[L : K]$ は、$L$ を $K$-ベクトル空間とみなしたときの次元である：
$$[L : K] = \dim_K(L)$$

### 代数的元
元 $\alpha \in L$ が $K$ 上**代数的**であるとは、$f(\alpha) = 0$ となるような $f \in K[X] \setminus \{0\}$ が存在することをいう。

### 超越的元
元 $\alpha \in L$ が $K$ 上**超越的**であるとは、代数的でないことをいう。

## 代数閉体

体 $K$ が**代数的閉体**であるとは、$K[X]$ の任意の定数でない多項式が $K$ に根を持つことをいう。

**代数閉包** $\overline{K}$ は、$K$ を含む最小の代数的閉体である。

## 有限体

素数べき $q = p^n$ に対して、$q$ 個の元を持つ有限体が同型を除いて一意に存在する。これを $\mathbb{F}_q$ または $\operatorname{GF}(q)$ と表す。

### 有限体の性質
- $\mathbb{F}_q^\times$ は巡回群
- $\mathbb{F}_{p^m} \subseteq \mathbb{F}_{p^n}$ であるための必要十分条件は $m \mid n$
- フロベニウス自己同型：$\varphi(x) = x^p$
