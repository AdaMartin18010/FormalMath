# Group (Mathematics)

## Definition

A **group** $(G, \cdot)$ is a set $G$ together with a binary operation $\cdot: G \times G \rightarrow G$ satisfying the following axioms:

### (G1) Associativity
For all $a, b, c \in G$:
$$(a \cdot b) \cdot c = a \cdot (b \cdot c)$$

### (G2) Identity Element
There exists an element $e \in G$ such that for all $a \in G$:
$$e \cdot a = a \cdot e = a$$

### (G3) Inverse Element
For each $a \in G$, there exists an element $a^{-1} \in G$ such that:
$$a \cdot a^{-1} = a^{-1} \cdot a = e$$

## Types of Groups

| Term | Definition | Example |
|------|------------|---------|
| Abelian Group | Commutative group: $a \cdot b = b \cdot a$ | $(\mathbb{Z}, +)$ |
| Finite Group | $|G| < \infty$ | Symmetric group $S_n$ |
| Cyclic Group | Generated by a single element | $\mathbb{Z}/n\mathbb{Z}$ |
| Simple Group | No nontrivial normal subgroups | $A_n$ for $n \geq 5$ |

## Fundamental Theorems

### Lagrange's Theorem
If $H \leq G$ is a subgroup of a finite group $G$, then:
$$|G| = |H| \cdot [G : H]$$

### First Isomorphism Theorem
For a group homomorphism $\varphi: G \rightarrow H$:
$$G / \ker(\varphi) \cong \operatorname{im}(\varphi)$$

### Cayley's Theorem
Every group $G$ is isomorphic to a subgroup of the symmetric group $\operatorname{Sym}(G)$.

## Applications

Groups have applications in:
- **Geometry**: Symmetry groups
- **Physics**: Representation theory
- **Cryptography**: Prime number groups
- **Chemistry**: Molecular symmetry
