## 📋 分支概览

基础数学分支是FormalMath项目的入门模块，涵盖数学大厦的根基内容：

- **集合论**：现代数学的通用语言

- **证明技巧**：数学思维的核心方法
- **范畴论**：数学结构的统一视角
- **图论与组合**：离散数学的核心工具
