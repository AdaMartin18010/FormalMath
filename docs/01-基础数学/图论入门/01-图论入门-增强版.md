## 📚 概述

图论是研究图（由顶点和边组成的结构）的数学分支。它起源于欧拉对哥尼斯堡七桥问题的研究，经过两个多世纪的发展，已成为离散数学的核心领域。

图论在计算机科学、运筹学、网络分析、生物信息学和社交网络分析等领域有着广泛的应用，是理解和解决复杂系统问题的重要工具。

## 🕰️ 历史发展

### 起源

**1736年**：莱昂哈德·欧拉解决哥尼斯堡七桥问题，创立图论。

**问题**：能否走遍7座桥，每座桥恰好经过一次？

**欧拉方法**：将陆地表示为顶点，桥表示为边，问题转化为是否存在欧拉回路。

### 19世纪发展

**四色问题**（1852年）：

- 弗朗西斯·格思里提出猜想
- 任何地图可用4种颜色着色，使相邻区域颜色不同
- 1976年由Appel和Haken用计算机证明

**基尔霍夫**（1847年）：

- 研究电路网络，提出树的概念

### 20世纪发展

**1930年代**：

- 柯尼希：第一本图论专著
- 埃尔德什：随机图论的开创者

**1940-50年代**：

- 网络流理论（Ford-Fulkerson）
- 匹配理论

**现代发展**：

- 算法图论
- 复杂网络分析
- 图神经网络

## 🏗️ 基本概念

### 图的定义

**定义 1.1（无向图）**：
图 $G = (V, E)$ 由：

- 顶点集 $V$（非空有限集）
- 边集 $E \subseteq \{\{u, v\} : u, v \in V, u \neq v\}$

**定义 1.2（有向图）**：
有向图 $D = (V, A)$ 中，$A \subseteq V \times V$ 是有序对（弧）。

**基本术语**：

- **阶**：顶点数 $|V|$
- **边数**：$|E|$
- **邻接**：顶点 $u$ 和 $v$ 有边相连
- **关联**：边 $e = uv$ 与顶点 $u, v$ 关联

### 特殊图类

**完全图 $K_n$**：

- 每对不同顶点间都有边
- 边数：$\binom{n}{2} = \frac{n(n-1)}{2}$

**空图 $\bar{K}_n$**：

- 没有边的图

**路径图 $P_n$**：

- $n$ 个顶点排成一条路径
- 边数：$n-1$

**圈图 $C_n$**：

- $n$ 个顶点形成一个环（$n \geq 3$）
- 边数：$n$

**轮图 $W_n$**：

- $C_{n-1}$ 加一个中心顶点与所有外围顶点相连

**完全二部图 $K_{m,n}$**：

- 顶点分为两部分，各 $m$ 和 $n$ 个
- 两部分间所有可能的边都存在
- 边数：$mn$

**正则图**：

- 每个顶点度数相同（$k$-正则）
- $K_n$ 是 $(n-1)$-正则
- $C_n$ 是2-正则

### 子图与运算

**子图**：$H = (V', E')$ 是 $G = (V, E)$ 的子图，如果 $V' \subseteq V$，$E' \subseteq E$。

**诱导子图**：$G[V']$ 包含 $V'$ 中所有原图中的边。

**图的运算**：

- **并**：$G_1 \cup G_2 = (V_1 \cup V_2, E_1 \cup E_2)$
- **交**：$G_1 \cap G_2 = (V_1 \cap V_2, E_1 \cap E_2)$
- **补图**：$\bar{G} = (V, \bar{E})$，其中 $\bar{E}$ 包含所有非边

## 📊 图的基本性质

### 度与握手定理

**定义 2.1（度）**：
顶点 $v$ 的度 $\deg(v)$ 是与 $v$ 关联的边数。

**握手定理**：
$$\sum_{v \in V} \deg(v) = 2|E|$$

**推论**：

- 图中奇度顶点的个数为偶数
- 平均度数为 $\frac{2|E|}{|V|}$

**有向图的度**：

- **出度** $\deg^+(v)$：从 $v$ 出发的弧数
- **入度** $\deg^-(v)$：指向 $v$ 的弧数
- $\sum \deg^+(v) = \sum \deg^-(v) = |A|$

### 路径与连通性

**定义 2.2**：

- **途径**：顶点序列 $v_0, v_1, \ldots, v_k$，相邻顶点有边
- **迹**：边不重的途径
- **路径**：顶点不重的途径
- **距离** $d(u, v)$：最短路径的长度
- **直径**：$\max_{u,v} d(u, v)$

**连通性**：

- **连通图**：任意两顶点间有路径
- **连通分支**：极大连通子图
- **割点**：移除后增加连通分支数的顶点
- **割边（桥）**：移除后增加连通分支数的边

### 树与森林

**定义 2.3（树）**：
无圈的连通图。

**等价刻画**：
对图 $G$（$|V| = n$），以下等价：

1. $G$ 是树（连通无圈）
2. $G$ 连通且有 $n-1$ 条边
3. $G$ 无圈且有 $n-1$ 条边
4. $G$ 中任意两顶点间有唯一路径
5. $G$ 连通，但移除任意边后不连通
6. $G$ 无圈，但添加任意边后产生圈

**森林**：无圈的图（树的并）。

**生成树**：包含所有顶点的树形子图。

**叶子**：度为1的顶点。任何树（$n \geq 2$）至少有两个叶子。

## 🔍 重要算法

### 遍历算法

**深度优先搜索（DFS）**：

```
DFS(v):
    标记 v 为已访问
    for v 的每个邻居 u:
        if u 未访问:
            DFS(u)
```

**广度优先搜索（BFS）**：

```
BFS(s):
    队列 Q = {s}
    标记 s 为已访问
    while Q 非空:
        v = Q.dequeue()
        for v 的每个邻居 u:
            if u 未访问:
                标记 u
                Q.enqueue(u)
```

**应用**：

- DFS：连通性检测、拓扑排序、寻找强连通分支
- BFS：最短路径（无权图）、层次遍历

### 最短路径

**Dijkstra算法**（单源最短路径，边权非负）：

```
Dijkstra(G, s):
    dist[s] = 0, dist[v] = ∞ (v ≠ s)
    优先队列 Q = V
    while Q 非空:
        u = Q.extract_min()
        for u 的邻居 v:
            if dist[u] + w(u,v) < dist[v]:
                dist[v] = dist[u] + w(u,v)
```

时间复杂度：$O((V + E) \log V)$（使用二叉堆）

**Floyd-Warshall算法**（全对最短路径）：

```
Floyd-Warshall(G):
    for k = 1 to n:
        for i = 1 to n:
            for j = 1 to n:
                dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])
```

时间复杂度：$O(V^3)$

### 最小生成树

**Kruskal算法**：

```
Kruskal(G):
    按边权排序所有边
    T = ∅
    for 每条边 e = uv:
        if u 和 v 不在同一连通分支:
            T = T ∪ {e}
    return T
```

使用并查集：时间复杂度 $O(E \log E)$

**Prim算法**：

```
Prim(G, s):
    T = {s}
    while |T| < |V|:
        选取连接 T 和 V\T 的最小权边 e
        T = T ∪ {e}
```

时间复杂度：$O(E \log V)$（使用优先队列）

## 💡 详细示例

### 示例1：七桥问题

**问题**：哥尼斯堡城有7座桥连接4块陆地，能否一次走遍每座桥恰好一次？

**分析**：
将4块陆地表示为顶点，7座桥表示为边。

问题转化为：图中是否存在欧拉迹（经过每条边恰好一次的迹）？

**欧拉定理**：
连通图有欧拉迹当且仅当：

- 没有奇度顶点（欧拉回路），或
- 恰好两个奇度顶点（欧拉通路）

**结论**：哥尼斯堡图有4个奇度顶点，故无解。$\square$

### 示例2：四色定理应用

**问题**：给平面地图着色，相邻国家颜色不同。

**图论模型**：

- 每个国家表示为顶点
- 相邻国家间有边

**问题**：图的色数 $\chi(G) \leq 4$？

**四色定理**：任何平面图满足 $\chi(G) \leq 4$。

### 示例3：握手问题

**问题**：聚会中每个人都与其他人握手，共有66次握手。有多少人？

**解**：
设 $n$ 人，握手次数为完全图的边数：
$$\binom{n}{2} = 66$$
$$\frac{n(n-1)}{2} = 66$$
$$n(n-1) = 132 = 12 \times 11$$

答案：$n = 12$ 人。

### 示例4：网络可靠性

**问题**：通信网络中，最少需要多少条边才能保证任意两点连通？

**解**：
需要一棵生成树，边数为 $n-1$。

如果要求 $k$-连通（移除任意 $k-1$ 个顶点后仍连通），则需要更多边。

## 📚 总结

### 核心定理速查

| 定理 | 内容 |
|-----|------|
| 握手定理 | $\sum \deg(v) = 2|E|$ |
| 欧拉迹存在 | 0或2个奇度顶点 |
| 树的等价 | 连通 + $|V|-1$ 条边 |
| 四色定理 | 平面图 $\chi \leq 4$ |

### 重要算法

| 算法 | 问题 | 复杂度 |
|-----|------|--------|
| DFS/BFS | 遍历 | $O(V+E)$ |
| Dijkstra | 单源最短路径 | $O((V+E)\log V)$ |
| Floyd-Warshall | 全对最短路径 | $O(V^3)$ |
| Kruskal/Prim | 最小生成树 | $O(E \log V)$ |

### 应用领域

- **计算机网络**：路由算法、拓扑设计
- **社交网络分析**：社区发现、影响力传播
- **生物信息学**：基因调控网络、蛋白质相互作用
- **运筹学**：网络流、匹配问题
- **计算机科学**：编译器、数据库、人工智能
