## 1. 引言

基础数学是整个数学大厦的基石，为所有数学分支提供严格的概念框架和逻辑基础。本模块系统涵盖**集合论**、**数系构造**、**序关系**和**基本代数结构**等核心内容，从空集的存在性出发，逐步构建自然数、整数、有理数、实数和复数的完整数系。

## 2. 核心概念与定义

### 2.1 集合论基础

**定义 2.1** (集合): 集合是确定的不同对象的整体，这些对象称为集合的元素。

**定义 2.2** (子集): 设 $A, B$ 为集合，若 $\forall x (x \in A \Rightarrow x \in B)$，则称 $A$ 是 $B$ 的子集，记作 $A \subseteq B$。

**定义 2.3** (幂集): 集合 $A$ 的幂集 $\mathcal{P}(A)$ 是 $A$ 的所有子集构成的集合：
$$\mathcal{P}(A) = \{X : X \subseteq A\}$$

**定理 2.1** (Cantor定理): 对任意集合 $A$，有 $|A| < |\mathcal{P}(A)|$。

**证明**: 假设存在满射 $f: A \to \mathcal{P}(A)$，构造 $B = \{x \in A : x \notin f(x)\}$。则 $B \in \mathcal{P}(A)$，但不存在 $a \in A$ 使得 $f(a) = B$（否则导致矛盾）。$\square$

---

### 2.2 关系与函数

**定义 2.4** (笛卡尔积): $A \times B = \{(a, b) : a \in A, b \in B\}$

**定义 2.5** (关系): $A$ 上的关系 $R$ 是 $A \times A$ 的子集。

**定义 2.6** (等价关系): 关系 $R$ 是等价关系，当且仅当满足：

1. **自反性**: $\forall a \in A, (a, a) \in R$
2. **对称性**: $(a, b) \in R \Rightarrow (b, a) \in R$
3. **传递性**: $(a, b) \in R \land (b, c) \in R \Rightarrow (a, c) \in R$

**定义 2.7** (函数): 从 $A$ 到 $B$ 的函数 $f: A \to B$ 是 $A \times B$ 的子集，满足：
$$\forall a \in A, \exists! b \in B : (a, b) \in f$$

**定理 2.2** (函数复合的结合律): 设 $f: A \to B$, $g: B \to C$, $h: C \to D$，则：
$$(h \circ g) \circ f = h \circ (g \circ f)$$

---

### 2.3 序关系

**定义 2.8** (偏序): 集合 $P$ 上的偏序 $\leq$ 满足：

1. **自反性**: $a \leq a$
2. **反对称性**: $a \leq b \land b \leq a \Rightarrow a = b$
3. **传递性**: $a \leq b \land b \leq c \Rightarrow a \leq c$

**定义 2.9** (全序): 偏序 $(P, \leq)$ 是全序，若任意两元素可比：
$$\forall a, b \in P, a \leq b \lor b \leq a$$

**定理 2.3** (Zorn引理): 若偏序集 $(P, \leq)$ 中每个链都有上界，则 $P$ 存在极大元。

**注**: Zorn引理与选择公理、良序定理等价（ZFC公理系统）。

## 3. 数系构造

### 3.1 自然数

**定义 3.1** (Peano公理): 自然数集 $\mathbb{N}$ 满足：

1. $0 \in \mathbb{N}$
2. 每个 $n \in \mathbb{N}$ 有唯一后继 $S(n) \in \mathbb{N}$
3. $0$ 不是任何元素的后继
4. $S(m) = S(n) \Rightarrow m = n$
5. **归纳原理**: $P(0) \land (P(n) \Rightarrow P(S(n))) \Rightarrow \forall n P(n)$

**定义 3.2** (加法递归定义):

- $n + 0 = n$
- $n + S(m) = S(n + m)$

**定理 3.1** (数学归纳法原理): 若性质 $P$ 满足 $P(0)$ 和 $P(n) \Rightarrow P(n+1)$，则 $P$ 对所有自然数成立。

---

### 3.2 整数与有理数

**定义 3.3** (整数): 整数集 $\mathbb{Z}$ 是自然数集在减法下的完备化：
$$\mathbb{Z} = (\mathbb{N} \times \mathbb{N}) / \sim$$
其中 $(a, b) \sim (c, d) \Leftrightarrow a + d = b + c$

**定义 3.4** (有理数): 有理数集 $\mathbb{Q}$ 是整数集在除法下的完备化：
$$\mathbb{Q} = (\mathbb{Z} \times \mathbb{Z}^*) / \sim$$
其中 $(a, b) \sim (c, d) \Leftrightarrow ad = bc$

**定理 3.2** (有理数的稠密性): 任意两个不同有理数之间存在无限多个有理数。

---

### 3.3 实数构造

**定义 3.5** (Dedekind分割): 实数是有理数集的Dedekind分割 $(A, B)$，其中：

1. $A \cup B = \mathbb{Q}$, $A \cap B = \emptyset$
2. $A \neq \emptyset$, $B \neq \emptyset$
3. $\forall a \in A, \forall b \in B, a < b$
4. $A$ 无最大元

**定义 3.6** (实数完备性): 实数集 $\mathbb{R}$ 满足确界原理：
$$\text{有上界的非空子集必有上确界}$$

**定理 3.3** (实数完备性等价形式): 以下条件等价：

1. 确界原理
2. 单调有界定理
3. 闭区间套定理
4. 有限覆盖定理 (Heine-Borel)
5. 聚点定理 (Bolzano-Weierstrass)
6. Cauchy收敛准则

## 4. 基础运算性质

### 4.1 基本运算律

| 运算律 | 加法形式 | 乘法形式 |
|--------|----------|----------|
| **交换律** | $a + b = b + a$ | $ab = ba$ |
| **结合律** | $(a+b)+c = a+(b+c)$ | $(ab)c = a(bc)$ |
| **分配律** | $a(b+c) = ab + ac$ | — |
| **单位元** | $a + 0 = a$ | $a \cdot 1 = a$ |
| **逆元** | $a + (-a) = 0$ | $a \cdot a^{-1} = 1$ (当 $a \neq 0$) |

### 4.2 数学归纳法变体

**定理 4.1** (强归纳法): 若 $P(n)$ 满足
$$(\forall m < n, P(m)) \Rightarrow P(n)$$
则 $\forall n, P(n)$。

**定理 4.2** (结构归纳法): 对递归定义的结构（如公式、树），若性质对所有基本结构成立，且在构造运算下保持，则对所有结构成立。

## 7. 示例

**示例 7.1** (等价类): 在整数集上定义模 $n$ 等价关系：
$$a \sim b \Leftrightarrow n \mid (a - b)$$
则等价类 $[a] = \{b \in \mathbb{Z} : a \sim b\}$ 构成商集 $\mathbb{Z}_n$。

**示例 7.2** (良序集): 自然数集 $(\mathbb{N}, \leq)$ 是良序集（每个非空子集有最小元），但整数集 $(\mathbb{Z}, \leq)$ 不是。

**示例 7.3** (归纳证明): 证明 $1 + 2 + \cdots + n = \frac{n(n+1)}{2}$

- **基础**: $n=1$ 时，$1 = \frac{1 \cdot 2}{2}$ ✓
- **归纳**: 假设对 $n$ 成立，则
  $$1 + 2 + \cdots + n + (n+1) = \frac{n(n+1)}{2} + (n+1) = \frac{(n+1)(n+2)}{2}$$ ✓
