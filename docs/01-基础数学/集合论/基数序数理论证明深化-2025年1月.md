## 📚 1. 基数理论证明深化

### 1.1 康托尔定理的完整证明

#### 定理 1.1.1 (康托尔定理 / Cantor's Theorem)

对于任意集合 $A$，有：
$$|A| < |\mathcal{P}(A)|$$

#### 证明思路

1. **第一步**: 证明 $|A| \leq |\mathcal{P}(A)|$
2. **第二步**: 证明 $|A| \neq |\mathcal{P}(A)|$
3. **第三步**: 结合前两步得到严格不等式

#### 详细证明

**步骤 1**: 证明 $|A| \leq |\mathcal{P}(A)|$

定义函数 $f : A \rightarrow \mathcal{P}(A)$ 为：
$$f(x) = \{x\}$$

**引理 1.1.1**: $f$ 是单射。

**证明**:

- 假设 $f(x) = f(y)$，则 $\{x\} = \{y\}$
- 由集合的外延性，$x = y$
- 因此 $f$ 是单射

由单射的存在性，我们得到 $|A| \leq |\mathcal{P}(A)|$。

**步骤 2**: 证明 $|A| \neq |\mathcal{P}(A)|$

假设存在双射 $g : A \rightarrow \mathcal{P}(A)$。

**引理 1.1.2** (对角线引理 / Diagonal Lemma):
定义集合：
$$D = \{x \in A : x \notin g(x)\}$$

则 $D \in \mathcal{P}(A)$ 且 $D \notin \text{range}(g)$。

**证明**:

- 由于 $g$ 是满射，存在 $d \in A$ 使得 $g(d) = D$
- **情况 1**: 如果 $d \in D$，则 $d \notin g(d) = D$，矛盾
- **情况 2**: 如果 $d \notin D$，则 $d \in g(d) = D$，矛盾
- 因此，不存在这样的 $d$，即 $D \notin \text{range}(g)$

这与 $g$ 是满射矛盾，因此 $|A| \neq |\mathcal{P}(A)|$。

**步骤 3**: 结合步骤 1 和步骤 2，得到 $|A| < |\mathcal{P}(A)|$。

#### 证明技巧分析

1. **对角线法**: 这是康托尔定理证明的核心技巧，通过构造"对角线集合"来证明不存在满射
2. **反证法**: 假设存在双射，然后导出矛盾
3. **分类讨论**: 对元素是否属于集合进行分类讨论

#### 应用案例

**案例 1.1.1**: 证明自然数集和实数集不等势

- 使用康托尔定理，$\mathbb{N}$ 和 $\mathcal{P}(\mathbb{N})$ 不等势
- 由于 $|\mathbb{R}| = |\mathcal{P}(\mathbb{N})|$，因此 $|\mathbb{N}| < |\mathbb{R}|$

**案例 1.1.2**: 证明存在不可数集

- 由康托尔定理，$\mathcal{P}(\mathbb{N})$ 是不可数集
- 这提供了构造不可数集的直接方法

### 1.2 康托尔-伯恩斯坦定理的完整证明

#### 定理 1.2.1 (康托尔-伯恩斯坦定理 / Cantor-Bernstein Theorem)

如果 $|A| \leq |B|$ 且 $|B| \leq |A|$，则 $|A| = |B|$。

#### 证明思路

1. **第一步**: 利用单射构造双射
2. **第二步**: 使用"链"的概念
3. **第三步**: 通过链的划分构造双射

#### 详细证明

**步骤 1**: 设 $f : A \rightarrow B$ 和 $g : B \rightarrow A$ 都是单射。

**定义 1.2.1** (链 / Chain):
对于 $a \in A$，定义链：
$$C(a) = \{a, g(f(a)), g(f(g(f(a)))), \ldots\}$$

**引理 1.2.1**: 链的性质

- 如果 $a \in C(b)$，则 $C(a) = C(b)$
- 链要么是有限循环，要么是无限链

**步骤 2**: 定义函数 $h : A \rightarrow B$：

对于 $a \in A$：

- 如果 $C(a)$ 是有限循环且 $a$ 在循环中的位置是奇数，则 $h(a) = f(a)$
- 否则，$h(a) = g^{-1}(a)$（如果 $a \in \text{range}(g)$）

**引理 1.2.2**: $h$ 是双射。

**证明**:

- **单射性**: 通过链的性质和 $f, g$ 的单射性证明
- **满射性**: 通过构造逆映射证明

**步骤 3**: 由双射的存在性，得到 $|A| = |B|$。

#### 证明技巧分析

1. **链方法**: 这是康托尔-伯恩斯坦定理证明的核心技巧
2. **构造性证明**: 通过构造双射来证明等势
3. **分类讨论**: 根据链的性质进行分类

#### 应用案例

**案例 1.2.1**: 证明 $|\mathbb{R}| = |\mathbb{R}^2|$

- 构造单射 $f : \mathbb{R} \rightarrow \mathbb{R}^2$ 和 $g : \mathbb{R}^2 \rightarrow \mathbb{R}$
- 应用康托尔-伯恩斯坦定理

**案例 1.2.2**: 证明 $|\mathbb{Q}| = |\mathbb{N}|$

- 构造单射 $f : \mathbb{Q} \rightarrow \mathbb{N}$ 和 $g : \mathbb{N} \rightarrow \mathbb{Q}$
- 应用康托尔-伯恩斯坦定理

## 📚 2. 序数理论证明深化

### 2.1 序数运算的结合律证明

#### 定理 2.1.1 (序数加法的结合律)

对于任意序数 $\alpha, \beta, \gamma$：
$$(\alpha + \beta) + \gamma = \alpha + (\beta + \gamma)$$

#### 证明思路

1. **第一步**: 对 $\gamma$ 进行超限归纳
2. **第二步**: 分别处理 $\gamma = 0$、后继序数和极限序数的情况
3. **第三步**: 使用序数的良序性质

#### 详细证明

**基础情况**: $\gamma = 0$

$$(\alpha + \beta) + 0 = \alpha + \beta = \alpha + (\beta + 0)$$

**后继情况**: 假设对 $\gamma$ 成立，证明对 $\gamma + 1$ 成立

$$(\alpha + \beta) + (\gamma + 1) = ((\alpha + \beta) + \gamma) + 1 = (\alpha + (\beta + \gamma)) + 1 = \alpha + ((\beta + \gamma) + 1) = \alpha + (\beta + (\gamma + 1))$$

**极限情况**: 假设 $\gamma$ 是极限序数，且对所有 $\delta < \gamma$ 成立

$$(\alpha + \beta) + \gamma = \sup\{(\alpha + \beta) + \delta : \delta < \gamma\} = \sup\{\alpha + (\beta + \delta) : \delta < \gamma\} = \alpha + (\beta + \gamma)$$

#### 证明技巧分析

1. **超限归纳法**: 这是序数理论证明的核心方法
2. **分类讨论**: 根据序数的类型（0、后继、极限）进行分类
3. **上确界性质**: 利用序数的上确界性质处理极限情况

### 2.2 序数乘法的分配律证明

#### 定理 2.2.1 (序数乘法的左分配律)

对于任意序数 $\alpha, \beta, \gamma$：
$$\alpha \cdot (\beta + \gamma) = \alpha \cdot \beta + \alpha \cdot \gamma$$

#### 证明思路

1. **第一步**: 对 $\gamma$ 进行超限归纳
2. **第二步**: 使用序数加法和乘法的定义
3. **第三步**: 利用序数的良序性质

#### 详细证明

**基础情况**: $\gamma = 0$

$$\alpha \cdot (\beta + 0) = \alpha \cdot \beta = \alpha \cdot \beta + 0 = \alpha \cdot \beta + \alpha \cdot 0$$

**后继情况**: 假设对 $\gamma$ 成立，证明对 $\gamma + 1$ 成立

$$\alpha \cdot (\beta + (\gamma + 1)) = \alpha \cdot ((\beta + \gamma) + 1) = \alpha \cdot (\beta + \gamma) + \alpha = (\alpha \cdot \beta + \alpha \cdot \gamma) + \alpha = \alpha \cdot \beta + (\alpha \cdot \gamma + \alpha) = \alpha \cdot \beta + \alpha \cdot (\gamma + 1)$$

**极限情况**: 假设 $\gamma$ 是极限序数，且对所有 $\delta < \gamma$ 成立

$$\alpha \cdot (\beta + \gamma) = \sup\{\alpha \cdot (\beta + \delta) : \delta < \gamma\} = \sup\{\alpha \cdot \beta + \alpha \cdot \delta : \delta < \gamma\} = \alpha \cdot \beta + \sup\{\alpha \cdot \delta : \delta < \gamma\} = \alpha \cdot \beta + \alpha \cdot \gamma$$

## 📚 3. 大基数理论证明深化

### 3.1 不可达基数的存在性

#### 定理 3.1.1 (不可达基数的性质)

如果 $\kappa$ 是不可达基数，则 $V_\kappa$ 是ZFC的模型。

#### 证明思路

1. **第一步**: 证明 $V_\kappa$ 满足外延公理
2. **第二步**: 证明 $V_\kappa$ 满足其他ZFC公理
3. **第三步**: 利用不可达基数的性质

#### 详细证明

**步骤 1**: 证明 $V_\kappa$ 满足外延公理

由于 $V_\kappa$ 是传递的，外延公理自然满足。

**步骤 2**: 证明 $V_\kappa$ 满足幂集公理

对于 $x \in V_\kappa$，有 $|x| < \kappa$。

由于 $\kappa$ 是强极限基数，$|\mathcal{P}(x)| = 2^{|x|} < \kappa$。

因此 $\mathcal{P}(x) \in V_\kappa$。

**步骤 3**: 证明 $V_\kappa$ 满足替换公理模式

对于函数 $f : x \rightarrow V_\kappa$，有 $|f(x)| \leq |x| < \kappa$。

由于 $\kappa$ 是正则基数，$f(x) \in V_\kappa$。

## 📚 4. 形式化实现

### 4.1 Lean 4 实现

```lean
-- 康托尔定理的形式化证明
theorem cantor_theorem_complete {α : Type u} (A : Set α) :
  Cardinal.mk A < Cardinal.mk (Set A) := by
  -- 步骤 1: 证明 |A| ≤ |P(A)|
  let f : A → Set A := fun x => {x}
  have h1 : Injective f := by
    intro x y h
    simp [f] at h
    exact h
  -- 步骤 2: 证明 |A| ≠ |P(A)|
  intro g
  assume h2 : Surjective g
  let D : Set A := {x : A | x ∉ g x}
  have h3 : D ∉ Set.range g := by
    intro h
    cases h with
    | intro a ha =>
      have h4 : a ∈ D ↔ a ∉ g a := by rw [ha]
      -- 分类讨论
      by_cases h5 : a ∈ D
      · have h6 : a ∉ g a := h4.mp h5
        rw [ha] at h6
        contradiction
      · have h7 : a ∈ g a := by
          rw [← ha]
          exact h4.mpr h5
        contradiction
  -- 步骤 3: 得到严格不等式
  exact ⟨h1, h3⟩
```
