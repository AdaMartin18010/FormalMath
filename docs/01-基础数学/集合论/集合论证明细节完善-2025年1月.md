## 一、ZFC公理证明补充

### 1.1 外延公理证明

**定理 (外延公理)**: 两个集合相等当且仅当它们具有相同的元素。

$$
\forall A \, \forall B \, (\forall x \, (x \in A \leftrightarrow x \in B) \to A = B)
$$

**形式化证明**：

```lean
theorem axiom_of_extensionality_proof {α : Type u} (A B : Set α) :
  (∀ x : α, x ∈ A ↔ x ∈ B) → A = B := by
  intro h
  ext x
  exact h x
```

**证明解析**：
1. `intro h`：引入假设 $\forall x, x \in A \leftrightarrow x \in B$
2. `ext x`：使用外延性策略，将目标转化为证明 $x \in A \leftrightarrow x \in B$
3. `exact h x`：直接应用假设

**注意**：在Lean中，`Set α` 的类型已经内置了外延公理（通过 `ext` 策略可用）。这个证明更多是展示外延公理如何在证明中被使用。

---

### 1.2 空集公理证明

**定理 (空集公理)**: 存在不包含任何元素的集合。

$$
\exists A \, \forall x \, (x \notin A)
$$

**形式化证明**：

```lean
theorem axiom_of_empty_set_proof {α : Type u} :
  ∃ A : Set α, ∀ x : α, x ∉ A := by
  existsi (∅ : Set α)
  intro x
  simp [not_mem_empty]
```

**证明解析**：
1. `existsi (∅ : Set α)`：显式构造空集 $\emptyset$ 作为见证
2. `intro x`：任取元素 $x$
3. `simp [not_mem_empty]`：使用空集的定义性质化简

**构造性说明**：在Lean的类型论框架中，空集 $\emptyset$ 定义为 `fun x => False`，即对每个 $x$，-membership 命题为假。这与 ZFC 中的原始空集公理略有不同（Lean的空集是构造性给出的，而 ZFC 只是断言其存在）。

## 二、基数理论证明完善

### 2.1 康托尔定理证明

**定理 (康托尔定理)**: 对任意集合 $A$，$|A| < |\mathcal{P}(A)|$。

形式化地，不存在从 $A$ 到其幂集 $\mathcal{P}(A)$ 的满射。

```lean
theorem cantor_theorem_complete {α : Type u} (A : Set α) :
  Cardinal.mk A < Cardinal.mk (Set A) := by
  let f : A → Set A := fun x => {x}
  have h1 : Injective f := by
    intro x y h
    simp [f] at h
    exact h
  intro g
  assume h2 : Surjective g
  let D : Set A := {x : A | x ∉ g x}

  have h3 : D ∉ Set.range g := by
    intro h
    cases h with a ha
    have h4 : a ∈ D ↔ a ∉ g a := by rw [ha]
    contradiction
  contradiction
```

**证明详细解析**：

**步骤1：建立单射 $A \hookrightarrow \mathcal{P}(A)$**

定义 $f(x) = \{x\}$。证明 $f$ 是单射：若 $\{x\} = \{y\}$，则 $x = y$。

这说明 $|A| \leq |\mathcal{P}(A)|$。

**步骤2：证明不存在满射 $A \twoheadrightarrow \mathcal{P}(A)$**

假设存在满射 $g: A \to \mathcal{P}(A)$。构造对角线集合：
$$D = \{x \in A \mid x \notin g(x)\}$$

**步骤3：导出矛盾**

由于 $g$ 是满射，存在 $a \in A$ 使 $g(a) = D$。

考虑 $a \in D$ 是否成立：
- 若 $a \in D$，则根据定义 $a \notin g(a) = D$。矛盾！
- 若 $a \notin D$，则根据定义 $a \in g(a) = D$。矛盾！

因此不存在这样的满射 $g$。

**步骤4：综合结论**

$|A| \leq |\mathcal{P}(A)|$（存在单射）且 $|A| \neq |\mathcal{P}(A)|$（不存在双射）。

故 $|A| < |\mathcal{P}(A)|$。

**对角线论证的意义**：康托尔的对角线方法是数学中最重要的证明技巧之一。它不仅证明了不可数集合的存在，还启发了哥德尔不完备性定理和图灵停机问题的证明。

## 三、集合运算证明完善

### 3.1 德摩根律证明

**定理 (德摩根律)**: $(A \cup B)^c = A^c \cap B^c$

```lean
theorem de_morgan_law_union {α : Type u} (A B : Set α) :
  (A ∪ B)ᶜ = Aᶜ ∩ Bᶜ := by
  ext x
  constructor
  · intro h
    constructor
    · intro h1; contradiction
    · intro h2; contradiction
  · intro h
    intro h1
    cases h with h2 h3
    cases h1 with h4 h5
    · contradiction
    · contradiction
```

**证明详细解析**：

**方法**：使用集合的外延性，证明两边包含相同的元素。

**方向1**：$x \in (A \cup B)^c \Rightarrow x \in A^c \cap B^c$

假设 $x \in (A \cup B)^c$，即 $x \notin A \cup B$。

这意味着 $x \notin A$ 且 $x \notin B$（因为若 $x \in A$ 或 $x \in B$，则 $x \in A \cup B$）。

因此 $x \in A^c$ 且 $x \in B^c$，即 $x \in A^c \cap B^c$。

**方向2**：$x \in A^c \cap B^c \Rightarrow x \in (A \cup B)^c$

假设 $x \in A^c \cap B^c$，即 $x \notin A$ 且 $x \notin B$。

假设 $x \in A \cup B$，则 $x \in A$ 或 $x \in B$。但两者都不成立，矛盾！

因此 $x \notin A \cup B$，即 $x \in (A \cup B)^c$。

**对偶形式**：对称地，有 $(A \cap B)^c = A^c \cup B^c$。

## 四、高级主题补充

### 4.1 选择公理与Zorn引理

**选择公理 (AC)**：对任意非空集合的族 $\{A_i\}_{i \in I}$，存在选择函数 $f: I \to \bigcup A_i$，使得 $f(i) \in A_i$。

**Zorn引理**：若偏序集 $(P, \leq)$ 中的每条链都有上界，则 $P$ 存在极大元。

**等价性**：在ZF系统中，AC $\Leftrightarrow$ Zorn引理 $\Leftrightarrow$ 良序定理。

**应用**：Zorn引理是证明许多存在性定理的关键工具：
- 向量空间的基的存在性
- 域的代数闭包的存在性
- 极大理想的 existence（Krull定理）
- Hahn-Banach定理

### 4.2 超限归纳法

**定理 (超限归纳)**：设 $(W, <)$ 是良序集，$P$ 是关于 $W$ 中元素的性质。若对任意 $x \in W$，$P(y)$ 对所有 $y < x$ 成立蕴含 $P(x)$ 成立，则 $P(x)$ 对所有 $x \in W$ 成立。

**与数学归纳法的区别**：
- 数学归纳法：$\mathbb{N}$ 上的归纳（离散、可数）
- 超限归纳法：任意良序集上的归纳（可连续、不可数）
