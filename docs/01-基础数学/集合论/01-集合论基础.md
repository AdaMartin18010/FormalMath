## 1.2 基本概念 / Basic Concepts (编号: B.01.01.02)

### 1.2.1 集合的定义 / Definition of Set

**定义 1.1** (集合 / Set)
设 $X$ 是一个类。如果存在集合 $A$ 使得 $X \in A$，则称 $X$ 是一个集合。

**形式化表述**: 集合是满足外延公理的对象，即两个集合相等当且仅当它们包含相同的元素。

**Definition 1.1** (Set)
Let $X$ be a class. If there exists a set $A$ such that $X \in A$, then $X$ is called a set.

**Formal statement**: A set is an object satisfying the axiom of extensionality, i.e., two sets are equal if and only if they contain the same elements.

**符号说明 / Symbol Explanation**:

- $X$: 任意类 (arbitrary class)
- $A$: 集合 (set)
- $\in$: 属于关系 (element relation)

**条件说明 / Condition Explanation**:

- 外延性: 集合由其元素唯一确定
- 基础性: 集合是数学的基础对象

## 📚 概念深度分析：集合 / Deep Concept Analysis: Set

### 1. 概念内涵 / Concept Intension

#### 1.1 本质属性 / Essential Properties

**核心特征**:

- 集合是一个**数学对象**，用于表示具有某种共同性质的事物的总体
- 集合由其**元素**唯一确定（外延性）
- 集合是**不可定义的原始概念**，通过ZFC公理系统刻画

**不可分离属性**:

- **外延性**: 两个集合相等当且仅当它们包含相同的元素
  - 形式化：$\forall x \forall y [\forall z(z \in x \leftrightarrow z \in y) \rightarrow x = y]$
- **元素关系**: 集合与其元素之间存在明确的属于关系 $\in$
- **确定性**: 对于任意对象和任意集合，要么该对象属于该集合，要么不属于（排中律）

#### 1.2 定义方式 / Definition Methods

**原始定义**:
在ZFC公理系统中，集合是满足外延公理的对象。这是集合论的基础，集合本身是不可定义的原始概念。

**等价定义**:

1. **外延性定义**: 集合是满足外延公理的对象
2. **类定义**: 集合是某个类的元素（即存在集合$A$使得$X \in A$）
3. **公理化定义**: 集合是ZFC公理系统中讨论的对象

**构造性定义**:
集合可以通过以下方式构造：

- **列举法**: $\{a, b, c\}$ 表示包含元素$a, b, c$的集合
- **描述法**: $\{x : P(x)\}$ 表示满足性质$P$的所有对象$x$的集合
- **公理构造**: 通过ZFC公理（空集公理、配对公理、并集公理等）构造

**公理化定义**:
集合是ZFC公理系统中讨论的基本对象，通过公理系统刻画其性质。

#### 1.3 条件分析 / Condition Analysis

**必要条件**:

- 如果$X$是集合，则必须满足外延公理
- 如果$X$是集合，则必须能够判断任意对象是否属于$X$

**充分条件**:

- 如果$X$满足外延公理且是某个集合的元素，则$X$是集合
- 如果$X$可以通过ZFC公理构造，则$X$是集合

**充要条件**:

- $X$是集合当且仅当$X$满足外延公理且存在集合$A$使得$X \in A$

### 2. 概念外延 / Concept Extension

#### 2.1 直接外延 / Direct Extension

**典型实例**:

- **有限集**: $\{1, 2, 3\}$（包含三个元素的集合）
- **空集**: $\emptyset = \{\}$（不包含任何元素的集合）
- **自然数集**: $\mathbb{N} = \{0, 1, 2, 3, \ldots\}$（所有自然数的集合）
- **实数集**: $\mathbb{R}$（所有实数的集合）

**标准实例**:

- **单元素集**: $\{a\}$（只包含一个元素的集合）
- **有序对**: $\{\{a\}, \{a, b\}\}$（有序对$(a, b)$的集合表示）
- **幂集**: $\mathcal{P}(A) = \{B : B \subseteq A\}$（集合$A$的所有子集的集合）

#### 2.2 间接外延 / Indirect Extension

**相关实例**:

- **类**: 比集合更广泛的概念，某些类不是集合（如所有集合的类）
- **族**: 集合的集合，即集合族
- **序列**: 可以看作特殊的集合（有序集合）

**推广实例**:

- **多集**: 允许元素重复的集合
- **模糊集**: 元素属于集合的程度在$[0,1]$之间的集合
- **范畴论中的集合**: 在范畴论中，集合是集合范畴的对象

#### 2.3 边界情况 / Boundary Cases

**极限情况**:

- **空集**: $\emptyset$（不包含任何元素的集合，是任何集合的子集）
- **单元素集**: $\{a\}$（只包含一个元素的集合）
- **全集**: 在给定论域中，包含所有对象的集合（在某些上下文中）

**退化情况**:

- **空集**: 集合的"零"情况
- **单元素集**: 集合的"最小非空"情况

#### 2.4 反例 / Counterexamples

**不满足概念的例子**:

- **真类**: 所有集合的类$V$不是集合（会导致罗素悖论）
- **罗素类**: $\{x : x \notin x\}$不是集合（罗素悖论）
- **不满足外延性的对象**: 如果两个对象包含相同元素但不相等，则不是集合

**常见误解**:

- 误解：所有"集合"都是集合（错误：真类不是集合）
- 误解：集合必须有元素（错误：空集是集合但没有元素）
- 误解：集合的元素必须是"东西"（错误：集合的元素可以是任何对象，包括其他集合）

#### 3.1 固有属性 / Intrinsic Properties

**定义蕴含的属性**:

## 1.3 ZFC公理系统 / ZFC Axiom System (编号: B.01.01.03)

### 1.3.1 外延公理 / Axiom of Extensionality

**公理 1.1** (外延公理 / Axiom of Extensionality)
对于任意集合 $x$ 和 $y$，如果它们包含相同的元素，则它们相等：

$$\forall x \forall y [\forall z(z \in x \leftrightarrow z \in y) \rightarrow x = y]$$

**Axiom 1.1** (Axiom of Extensionality)
For any sets $x$ and $y$, if they contain the same elements, then they are equal:

$$\forall x \forall y [\forall z(z \in x \leftrightarrow z \in y) \rightarrow x = y]$$

**直观解释 / Intuitive Explanation**:
外延公理表达了集合的基本性质：集合由其元素唯一确定。这意味着两个集合相等当且仅当它们包含完全相同的元素。

**形式化解释 / Formal Explanation**:
这个公理确保了集合的"外延性"，即集合的"内容"（元素）决定了集合的"身份"（相等性）。

### 1.3.2 空集公理 / Axiom of Empty Set

**公理 1.2** (空集公理)
存在一个不包含任何元素的集合：

$$\exists x \forall y (y \notin x)$$

**定义 1.3** (空集 / Empty Set)
空集记作 $\emptyset$，满足 $\forall x (x \notin \emptyset)$。

### 1.3.3 配对公理 / Axiom of Pairing

**公理 1.3** (配对公理)
对于任意两个集合，存在一个包含它们的集合：

$$\forall x \forall y \exists z \forall w(w \in z \leftrightarrow w = x \vee w = y)$$

**定义 1.4** (无序对 / Unordered Pair)
集合 $\{a, b\}$ 定义为包含 $a$ 和 $b$ 的集合。

### 1.3.4 并集公理 / Axiom of Union

**公理 1.4** (并集公理)
对于任意集合族，存在一个包含所有成员集合元素的集合：

$$\forall F \exists A \forall x(x \in A \leftrightarrow \exists B(B \in F \wedge x \in B))$$

**定义 1.5** (并集 / Union)
$$\bigcup F = \{x : \exists B(B \in F \wedge x \in B)\}$$

### 1.3.5 幂集公理 / Axiom of Power Set

**公理 1.5** (幂集公理)
对于任意集合，存在一个包含其所有子集的集合：

$$\forall x \exists y \forall z(z \in y \leftrightarrow z \subseteq x)$$

**定义 1.6** (幂集 / Power Set)
$$\mathcal{P}(A) = \{B : B \subseteq A\}$$

## 📚 概念深度分析：并集 / Deep Concept Analysis: Union

### 1. 概念内涵 / Concept Intension

#### 1.1 本质属性 / Essential Properties

**核心特征**:

- 并集是**两个集合的所有元素的集合**，描述了集合的合并操作
- 并集是**集合论的基础运算**，满足交换律、结合律和分配律
- 并集是**布尔代数**的核心运算，与交集构成对偶关系

**不可分离属性**:

- **合并性**: 并集包含两个集合的所有元素
  - 形式化：$x \in A \cup B \leftrightarrow x \in A \lor x \in B$
- **包含性**: 两个集合都是并集的子集
  - 形式化：$A \subseteq A \cup B$且$B \subseteq A \cup B$
- **最小性**: 并集是包含$A$和$B$的最小集合

#### 1.2 定义方式 / Definition Methods

**原始定义**:
集合$A$和$B$的并集是所有属于$A$或属于$B$的元素的集合。

**等价定义**:

1. **元素定义**: $A \cup B = \{x : x \in A \lor x \in B\}$
2. **子集定义**: $A \cup B$是包含$A$和$B$的最小集合
3. **补集定义**: $(A \cup B)^c = A^c \cap B^c$（De Morgan律）

**构造性定义**:

#### 1.3 条件分析 / Condition Analysis

**必要条件**:

- 如果$A \cup B$是并集，则$A$和$B$必须是集合
- 如果$x \in A \cup B$，则$x \in A$或$x \in B$

**充分条件**:

- 如果$A$和$B$是集合，则$A \cup B$是集合
- 如果集合$C$包含所有满足$x \in A \lor x \in B$的元素$x$，且$A \subseteq C$和$B \subseteq C$，则$C = A \cup B$

**充要条件**:

- $C = A \cup B$当且仅当$C = \{x : x \in A \lor x \in B\}$

### 1. 概念内涵 / Concept Intension

### 1.5.1 有序对 / Ordered Pair

**定义 1.13** (有序对 / Ordered Pair)
$$(a, b) = \{\{a\}, \{a, b\}\}$$

**定理 1.1** (有序对性质)
$(a, b) = (c, d)$ 当且仅当 $a = c$ 且 $b = d$

**证明**：
假设 $(a, b) = (c, d)$，则：
$$\{\{a\}, \{a, b\}\} = \{\{c\}, \{c, d\}\}$$

如果 $a = b$，则 $\{\{a\}\} = \{\{c\}, \{c, d\}\}$，这要求 $c = d = a$。
如果 $a \neq b$，则 $\{a\} \neq \{a, b\}$，因此 $\{a\} = \{c\}$ 且 $\{a, b\} = \{c, d\}$，从而 $a = c$ 且 $b = d$。

### 1.5.2 笛卡尔积定义 / Cartesian Product Definition

**定义 1.14** (笛卡尔积 / Cartesian Product)
$$A \times B = \{(a, b) : a \in A \wedge b \in B\}$$

## 📚 概念深度分析：笛卡尔积 / Deep Concept Analysis: Cartesian Product

### 1. 概念内涵 / Concept Intension

#### 1.1 本质属性 / Essential Properties

**核心特征**:

- 笛卡尔积是**两个集合的所有有序对的集合**，描述了集合之间的组合关系
- 笛卡尔积是**关系、函数和映射的基础**，为这些概念提供了构造方法
- 笛卡尔积是**集合论的核心运算**，用于构造多维结构和复合对象

**不可分离属性**:

- **有序性**: 笛卡尔积中的元素是有序对，$(a, b) \neq (b, a)$（除非$a = b$）
  - 形式化：$(a, b) \in A \times B \leftrightarrow a \in A \land b \in B$
- **组合性**: 笛卡尔积包含所有可能的组合
  - 形式化：$A \times B = \{(a, b) : a \in A \land b \in B\}$
- **集合性**: 笛卡尔积是集合（有序对的集合）

#### 1.2 定义方式 / Definition Methods

**原始定义**:
集合$A$和$B$的笛卡尔积是所有有序对$(a, b)$的集合，其中$a \in A$且$b \in B$。

**等价定义**:

1. **有序对定义**: $A \times B = \{(a, b) : a \in A \land b \in B\}$
2. **关系定义**: $A \times B$是所有满足$a \in A$且$b \in B$的有序对$(a, b)$的集合
3. **函数定义**: $A \times B$可以看作从$\{1, 2\}$到$A \cup B$的函数集合

**构造性定义**:
笛卡尔积可以通过以下方式构造：

- **直接构造**: 通过有序对的定义直接构造
- **递归构造**: $A_1 \times A_2 \times \cdots \times A_n = (A_1 \times A_2 \times \cdots \times A_{n-1}) \times A_n$
- **幂集构造**: 通过幂集和分离公理构造

**公理化定义**:
在ZFC公理系统中，笛卡尔积通过配对公理、并集公理和分离公理定义。

#### 1.3 条件分析 / Condition Analysis

**必要条件**:

- 如果$A \times B$是笛卡尔积，则$A$和$B$必须是集合
- 如果$(a, b) \in A \times B$，则$a \in A$且$b \in B$

**充分条件**:

- 如果$A$和$B$是集合，则$A \times B$是集合
- 如果集合$C$包含所有满足$a \in A$且$b \in B$的有序对$(a, b)$，则$C = A \times B$

**充要条件**:

- $C = A \times B$当且仅当$C = \{(a, b) : a \in A \land b \in B\}$

### 2. 概念外延 / Concept Extension

#### 2.1 直接外延 / Direct Extension

**典型实例**:

- **平面坐标**: $\mathbb{R} \times \mathbb{R} = \mathbb{R}^2$（实数平面的笛卡尔积）
- **三维空间**: $\mathbb{R} \times \mathbb{R} \times \mathbb{R} = \mathbb{R}^3$（三维实数空间）
- **有限集合**: $\{1, 2\} \times \{a, b\} = \{(1,a), (1,b), (2,a), (2,b)\}$

**标准实例**:

- **幂**: $A^n = A \times A \times \cdots \times A$（$n$个$A$的笛卡尔积）
- **无限笛卡尔积**: $\prod_{i \in I} A_i$（索引集$I$上的笛卡尔积）

#### 2.2 间接外延 / Indirect Extension

**相关实例**:

- **关系**: 关系是笛卡尔积的子集
- **函数**: 函数是笛卡尔积的子集（满足特定条件）

- **直积**: 在群论、环论中，直积是笛卡尔积的推广

#### 2.3 边界情况 / Boundary Cases

**极限情况**:

- **空集**: $\emptyset \times A = A \times \emptyset = \emptyset$（空集的笛卡尔积是空集）
- **单元素集**: $\{a\} \times \{b\} = \{(a, b)\}$（单元素集的笛卡尔积是单元素集）

**退化情况**:

#### 2.4 反例 / Counterexamples

**不满足概念的例子**:

- **不是有序对的集合**: 如果集合包含非有序对的元素，则不是笛卡尔积
- **不完整的组合**: 如果集合不包含所有可能的组合，则不是笛卡尔积

**常见误解**:

- 误解：笛卡尔积和并集是同一个概念（错误：笛卡尔积是有序对的集合，并集是元素的集合）
- 误解：笛卡尔积满足交换律（错误：$A \times B \neq B \times A$，除非$A = B$）
- 误解：笛卡尔积满足结合律（错误：$(A \times B) \times C \neq A \times (B \times C)$，但同构）

### 1.6.1 关系 / Relations

**定义 1.15** (关系 / Relation)
集合 $R$ 是关系当且仅当 $R \subseteq A \times B$，其中 $A$ 和 $B$ 是集合。

**定义 1.16** (定义域和值域 / Domain and Range)

- 定义域：$\text{dom}(R) = \{x : \exists y((x, y) \in R)\}$
- 值域：$\text{ran}(R) = \{y : \exists x((x, y) \in R)\}$

### 1.6.2 函数 / Functions

**定义 1.17** (函数 / Function)
关系 $f$ 是函数当且仅当：
$$\forall x \forall y \forall z(((x, y) \in f \wedge (x, z) \in f) \rightarrow y = z)$$

**定义 1.18** (函数记号 / Function Notation)
如果 $f$ 是函数且 $(x, y) \in f$，记作 $f(x) = y$。

## 1.7 等价关系 / Equivalence Relations (编号: B.01.01.07)

### 1.7.1 等价关系定义 / Equivalence Relation Definition

**定义 1.19** (等价关系 / Equivalence Relation)
关系 $R$ 是等价关系当且仅当：

1. 自反性：$\forall x(x \in A \rightarrow (x, x) \in R)$
2. 对称性：$\forall x \forall y((x, y) \in R \rightarrow (y, x) \in R)$
3. 传递性：$\forall x \forall y \forall z(((x, y) \in R \wedge (y, z) \in R) \rightarrow (x, z) \in R)$

### 1.7.2 等价类 / Equivalence Classes

**定义 1.20** (等价类 / Equivalence Class)
对于等价关系 $R$ 和元素 $a$，$a$ 的等价类为：
$$[a]_R = \{x : (a, x) \in R\}$$

**定理 1.2** (等价类性质)
如果 $R$ 是等价关系，则：

1. $a \in [a]_R$
2. $[a]_R = [b]_R$ 当且仅当 $(a, b) \in R$
3. $[a]_R \cap [b]_R = \emptyset$ 或 $[a]_R = [b]_R$

## 1.8 形式化代码示例 / Formal Code Examples (编号: B.01.01.08)

### 1.8.1 Lean 4 实现 / Lean 4 Implementation

```lean
-- 集合论基础实现
import Mathlib.Data.Set.Basic
import Mathlib.Data.Set.Lattice

-- 集合定义
def Set (α : Type u) := α → Prop

-- 元素关系
def mem {α : Type u} (a : α) (A : Set α) : Prop := A a

-- 子集关系
def subset {α : Type u} (A B : Set α) : Prop :=
  ∀ x, mem x A → mem x B

-- 交集
def intersection {α : Type u} (A B : Set α) : Set α :=
  fun x => mem x A ∧ mem x B

-- 并集
def union {α : Type u} (A B : Set α) : Set α :=
  fun x => mem x A ∨ mem x B

-- 差集
def difference {α : Type u} (A B : Set α) : Set α :=
  fun x => mem x A ∧ ¬mem x B

-- 等价关系
structure EquivalenceRelation (α : Type u) where
  relation : α → α → Prop
  reflexive : ∀ x, relation x x
  symmetric : ∀ x y, relation x y → relation y x
  transitive : ∀ x y z, relation x y → relation y z → relation x z

-- 等价类
def equivalenceClass {α : Type u} (R : EquivalenceRelation α) (a : α) : Set α :=
  fun x => R.relation a x
```

### 1.8.2 Haskell 实现 / Haskell Implementation

```haskell
-- 集合类型定义
type Set a = a -> Bool

-- 元素关系
member :: a -> Set a -> Bool
member x set = set x

-- 子集关系
subset :: Eq a => Set a -> Set a -> Bool
subset a b = all (\x -> member x a `implies` member x b) universe
  where
    implies p q = not p || q
    universe = [] -- 需要根据具体类型定义全域

-- 交集
intersection :: Set a -> Set a -> Set a
intersection a b = \x -> member x a && member x b

-- 并集
union :: Set a -> Set a -> Set a
union a b = \x -> member x a || member x b

-- 差集
difference :: Set a -> Set a -> Set a
difference a b = \x -> member x a && not (member x b)

-- 等价关系
data EquivalenceRelation a = EquivalenceRelation
  { relation :: a -> a -> Bool
  , reflexive :: a -> Bool
  , symmetric :: a -> a -> Bool
  , transitive :: a -> a -> a -> Bool
  }

-- 等价类
equivalenceClass :: EquivalenceRelation a -> a -> Set a
equivalenceClass eq a = \x -> relation eq a x
```
