## 1. 基础练习题

### 1.1 集合基本概念

**练习1.1**: 判断以下命题的真假

1. ∅ ∈ ∅
2. ∅ ⊆ ∅
3. {∅} ∈ {∅}
4. {∅} ⊆ {∅}

**解答**:

```lean
theorem basic_concepts_exercise :
  ∅ ∉ ∅ ∧ ∅ ⊆ ∅ ∧ {∅} ∈ {∅} ∧ {∅} ⊆ {∅} := by
  constructor
  · simp [not_mem_empty]
  constructor
  · simp [empty_subset]
  constructor
  · simp [mem_singleton]
  · simp [subset_singleton_iff]
```

**练习1.2**: 设A = {1,2,3}, B = {2,3,4}, 计算：

1. A ∪ B
2. A ∩ B
3. A \ B
4. A △ B

**解答**:

```lean
example (A B : Set ℕ) :
  A = {1,2,3} → B = {2,3,4} →
  A ∪ B = {1,2,3,4} ∧ A ∩ B = {2,3} ∧
  A \ B = {1} ∧ A △ B = {1,4} := by
  intro hA hB
  constructor
  · rw [hA, hB]; simp [union_def]
  constructor
  · rw [hA, hB]; simp [inter_def]
  constructor
  · rw [hA, hB]; simp [diff_def]
  · rw [hA, hB]; simp [symm_diff_def]
```

## 2. 进阶练习题

### 2.1 ZFC公理应用

**练习2.1**: 使用ZFC公理证明存在集合{∅, {∅}, {{∅}}}

**解答**:

```lean
theorem zfc_application :
  ∃ A : Set (Set (Set (Set ℕ))),
  A = {∅, {∅}, {{∅}}} := by
  -- 使用配对公理和并集公理
  have h1 : ∃ A, ∀ x, x ∈ A ↔ x = ∅ ∨ x = {∅} :=
    axiom_of_pairing ∅ {∅}
  have h2 : ∃ B, ∀ x, x ∈ B ↔ x = {∅} ∨ x = {{∅}} :=
    axiom_of_pairing {∅} {{∅}}
  have h3 : ∃ C, ∀ x, x ∈ C ↔ x ∈ A ∨ x ∈ B :=
    axiom_of_union {A, B}
  existsi C
  ext x
  simp [h1, h2, h3]
```

### 2.2 函数性质

**练习2.2**: 函数复合性质

设f: A → B, g: B → C, 证明：如果f和g都是单射，则g∘f是单射

**解答**:

```lean
theorem function_composition_injective {α β γ : Type u} 
  (f : α → β) (g : β → γ) :
  Injective f → Injective g → Injective (g ∘ f) := by
  intro hf hg
  intro x y h
  have h1 : g (f x) = g (f y) := h
  have h2 : f x = f y := hg h1
  exact hf h2
```

## 3. 高级练习题

### 3.1 基数理论

**练习3.1**: 康托尔定理

证明：对于任何集合A，|A| < |P(A)|

**解答**:

```lean
theorem cantor_theorem {α : Type u} (A : Set α) :
  Cardinal.mk A < Cardinal.mk (Set A) := by
  let f : A → Set A := fun x => {x}
  have h1 : Injective f := by
    intro x y h
    simp [f] at h
    exact h
  intro g
  assume h2 : Surjective g
  let D : Set A := {x : A | x ∉ g x}
  have h3 : D ∉ Set.range g := by
    intro h
    cases h with a ha
    have h4 : a ∈ D ↔ a ∉ g a := by rw [ha]
    contradiction
  contradiction
```
