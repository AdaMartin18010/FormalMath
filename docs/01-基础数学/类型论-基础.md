# 类型论基础

类型论（type theory）是数理逻辑与理论计算机科学的交汇领域，将"命题"与"类型"、"证明"与"程序"统一起来。从 Russell 的简单类型论（1908）到 Martin-Löf 的直觉主义类型论（1970s），再到 Homotopy Type Theory（2006s），类型论已成为形式化数学与程序语言设计的基石。它不仅是构造性数学的精确语言，也是现代 proof assistant（如 Lean、Coq、Agda）的理论基础。

## 1. 简单类型 λ-演算

### 1.1 类型与项

**定义 1.1（简单类型）**. 简单类型由以下文法归纳生成：

$$A, B ::= \iota \mid o \mid A \to B$$

其中 $\iota$ 为个体类型（基类型），$o$ 为命题类型（或布尔类型），$A \to B$ 为函数类型。

**定义 1.2（λ-项）**. 对每个类型 $A$，有可数无穷个变元 $x^A, y^A, \dots$。λ-项由以下规则生成：
1. **变元**：若 $x$ 为类型 $A$ 的变元，则 $x : A$；
2. **应用**：若 $M : A \to B$ 且 $N : A$，则 $MN : B$；
3. **抽象**：若 $M : B$ 且 $x : A$，则 $\lambda x^A . M : A \to B$。

### 1.2 β-归约与 Church-Rosser

**定义 1.3（β-归约）**. $(\lambda x.M)N \to_\beta M[N/x]$，其中 $M[N/x]$ 表示将 $M$ 中自由出现的 $x$ 替换为 $N$（必要时进行 $\alpha$-变换避免捕获）。

**定理 1.4（Church-Rosser）**. $\to_\beta$ 满足合流性（confluence）：若 $M \to_\beta^* N_1$ 且 $M \to_\beta^* N_2$，则存在 $P$ 使得 $N_1 \to_\beta^* P$ 且 $N_2 \to_\beta^* P$。

*证明*. 我们通过引入并行归约 $\Rrightarrow$ 来证明。定义 $M \Rrightarrow N$ 归纳如下：
1. $x \Rrightarrow x$（变元）；
2. 若 $M \Rrightarrow M'$ 且 $N \Rrightarrow N'$，则 $MN \Rrightarrow M'N'$；
3. 若 $M \Rrightarrow M'$，则 $\lambda x.M \Rrightarrow \lambda x.M'$；
4. 若 $M \Rrightarrow M'$ 且 $N \Rrightarrow N'$，则 $(\lambda x.M)N \Rrightarrow M'[N'/x]$。

关键引理：若 $M \Rrightarrow M'$ 且 $N \Rrightarrow N'$，则 $M[N/x] \Rrightarrow M'[N'/x]$（对 $M$ 的结构归纳）。

取 $M \Rrightarrow M_1$ 和 $M \Rrightarrow M_2$，需找 $M_3$ 使 $M_1 \Rrightarrow M_3$ 且 $M_2 \Rrightarrow M_3$。对 $M$ 的结构归纳，利用关键引理即得。由于 $\to_\beta^*$ 是 $\Rrightarrow^*$ 的子关系，合流性得证。$\square$

**推论 1.5**. 每个 λ-项至多有一个 β-范式。

### 1.3 强归一化

**定理 1.6（强归一化）**. 简单类型 λ-演算中，所有良类型的项都是强归一化的（不存在无限 β-归约序列）。

*证明概要*. 利用可约候选法（reducibility candidates）或 Tait 的可计算性方法。对类型 $A$ 定义可计算项集合 $\mathsf{SN}_A$：
- $\mathsf{SN}_\iota$ 为所有强归一化的基类型项；
- $M \in \mathsf{SN}_{A \to B}$ 当且仅当对所有 $N \in \mathsf{SN}_A$，有 $MN \in \mathsf{SN}_B$。

通过对项的结构归纳，证明所有良类型的项都是可计算的，从而强归一化。$\square$

## 2. 直觉主义类型论

### 2.1 依赖类型

**定义 2.1（依赖函数类型，Π-类型）**. 设 $A$ 为类型，$B(x)$ 为依赖类型（依赖于 $x : A$）。Π-类型表示依赖函数空间：

$$\Pi_{x:A} B(x) = \{ f : \forall x \in A,\ f(x) \in B(x) \}.$$

当 $B$ 不依赖 $x$ 时，$\Pi_{x:A} B = A \to B$。

**定义 2.2（依赖和类型，Σ-类型）**. Σ-类型表示依赖对空间：

$$\Sigma_{x:A} B(x) = \{ (a, b) : a \in A,\ b \in B(a) \}.$$

当 $B$ 不依赖 $x$ 时，$\Sigma_{x:A} B = A \times B$。

### 2.2 构造演算（Calculus of Constructions）

**定义 2.3（类型宇宙）**. 引入类型宇宙的层级 $\mathcal{U}_0 : \mathcal{U}_1 : \mathcal{U}_2 : \dots$（或累积层级 $\mathcal{U}_i : \mathcal{U}_{i+1}$），避免 Girard 悖论。

**定理 2.4（强归一化）**. 构造演算中良类型的项都是强归一化的（不存在无限归约序列）。

*证明*. Coquand 与 Gallier 通过可约性候选与超解释（super-interpretation）方法证明。核心思想是将每个类型赋予一个饱和集（saturated set），证明类型判断保持可计算性。$\square$

## 3. Curry-Howard 同构

### 3.1 命题即类型

**定理 3.1（Curry-Howard）**. 在直觉主义命题逻辑与简单类型 λ-演算之间存在以下对应：

| 逻辑 | 类型论 |
|------|--------|
| 命题 $A$ | 类型 $A$ |
| 证明 $p$ | 项 $p : A$ |
| $A \to B$ | 函数类型 $A \to B$ |
| $A \wedge B$ | 积类型 $A \times B$ |
| $A \vee B$ | 和类型 $A + B$ |
| $\forall x A(x)$ | $\Pi_{x:D} A(x)$ |
| $\exists x A(x)$ | $\Sigma_{x:D} A(x)$ |
| 矛盾 $\perp$ | 空类型 $\mathbf{0}$ |

**推论 3.2**. 命题 $A$ 在直觉主义逻辑中可证，当且仅当类型 $A$ 被某个闭 λ-项居住（inhabited）。

### 3.2 排中律的不可居住性

经典逻辑中的排中律 $A \vee \neg A$ 对应类型 $A + (A \to \mathbf{0})$。在直觉主义类型论中，该类型并非对所有 $A$ 可居住——除非接受双重否定消去或添加相应的公理。

**定理 3.3**. 不存在闭项 $t : \Pi_{A:\mathcal{U}} (A + (A \to \mathbf{0}))$。

*证明*. 假设存在这样的 $t$。取 $A = \mathbf{0}$，则 $t(\mathbf{0}) : \mathbf{0} + (\mathbf{0} \to \mathbf{0})$。若 $t(\mathbf{0}) = \mathsf{inl}(a)$，则 $a : \mathbf{0}$，矛盾。故 $t(\mathbf{0}) = \mathsf{inr}(f)$ 对某 $f : \mathbf{0} \to \mathbf{0}$。但这不能给出一般 $A$ 的排中律。更严格的证明利用 realizability 或拓扑语义：在某种模型中，$\Pi_{A:\mathcal{U}} (A + \neg A)$ 的解释为空。$\square$

## 4. 归纳类型

### 4.1 自然数类型

**定义 4.1（自然数类型）**. 自然数类型 $\mathbb{N}$ 由以下构造子定义：
- $0 : \mathbb{N}$；
- $\operatorname{succ} : \mathbb{N} \to \mathbb{N}$。

**原理 4.2（归纳原理）**. 为证明 $\forall n : \mathbb{N},\ P(n)$，只需证明：
1. $P(0)$（基例）；
2. $\forall n : \mathbb{N},\ P(n) \to P(\operatorname{succ}(n))$（归纳步）。

这恰好对应数学归纳法。在类型论中，归纳原理体现为依赖消去子（dependent eliminator）$\mathsf{ind}_\mathbb{N}$：

$$\mathsf{ind}_\mathbb{N} : \Pi_{P:\mathbb{N}\to\mathcal{U}} P(0) \to (\Pi_{n:\mathbb{N}} P(n) \to P(\operatorname{succ}(n))) \to \Pi_{n:\mathbb{N}} P(n).$$

### 4.2 身份类型（Identity Type）

**定义 4.3**. 对类型 $A$ 和 $a, b : A$，身份类型 $\operatorname{Id}_A(a, b)$（或 $a =_A b$）表示"$a$ 与 $b$ 相等"的命题。其唯一构造子为自反性 $\operatorname{refl}_a : a =_A a$。

**定理 4.4（J-消去）**. 对任意类型族 $C : \Pi_{x,y:A} (x =_A y) \to \mathcal{U}$ 和 $c : \Pi_{x:A} C(x, x, \operatorname{refl}_x)$，存在：

$$J(C, c) : \Pi_{x,y:A} \Pi_{p:x=y} C(x, y, p).$$

*证明*. 这是身份类型的归纳原理。在 Martin-Löf 类型论中，J-消去是身份类型的原始消去规则。对 $p = \operatorname{refl}_x$，定义 $J(C, c, x, x, \operatorname{refl}_x) = c(x)$。由归纳原理的唯一性，这唯一确定了 $J$。$\square$

在 Homotopy Type Theory 中，身份类型对应路径空间，允许证明的层级化（$p = q$，$p = q = r$ 等）。

## 5. 例子

### 5.1 例子：S-组合子的类型推导

考虑命题 $(A \to B \to C) \to (A \to B) \to A \to C$（蕴含的分配律，S-组合子）。对应的 λ-项为：

$$\lambda f^{A \to B \to C}. \lambda g^{A \to B}. \lambda x^A. f x (g x) : (A \to B \to C) \to (A \to B) \to A \to C.$$

类型验证：
- $f : A \to B \to C$，$x : A$ $ \Rightarrow $ $f x : B \to C$；
- $g : A \to B$，$x : A$ $ \Rightarrow $ $g x : B$；
- $f x (g x) : C$；
- 故 $\lambda x. f x (g x) : A \to C$；
- 故 $\lambda g. \lambda x. f x (g x) : (A \to B) \to A \to C$；
- 故整体项具有所求类型。

该项直接给出了命题 $(A \to B \to C) \to (A \to B) \to A \to C$ 在直觉主义逻辑中的证明。

### 5.2 例子：列表类型与 fold

列表类型 $\operatorname{List}(A)$ 的构造子为：
- $\operatorname{nil} : \operatorname{List}(A)$；
- $\operatorname{cons} : A \to \operatorname{List}(A) \to \operatorname{List}(A)$。

归纳定义函数如长度、映射、折叠（fold）等。例如 fold 的类型为：

$$\operatorname{fold} : (A \to B \to B) \to B \to \operatorname{List}(A) \to B.$$

其定义为：
- $\operatorname{fold}(f, b, \operatorname{nil}) = b$；
- $\operatorname{fold}(f, b, \operatorname{cons}(a, l)) = f(a, \operatorname{fold}(f, b, l))$。

fold 的重要性在于它编码了列表上的原始递归。例如，求和函数 $\mathsf{sum} : \operatorname{List}(\mathbb{N}) \to \mathbb{N}$ 可定义为 $\mathsf{sum} = \operatorname{fold}(+, 0, -)$。
