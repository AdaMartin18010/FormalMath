## 📚 概述

计数原理是组合数学的基础，研究如何计算有限集合中元素的个数。从简单的排列组合到复杂的容斥原理，计数技巧在数学、计算机科学、概率论和统计学中都有广泛应用。

本增强版文档将系统介绍计数的基本原理和方法，通过丰富的示例展示如何解决各种计数问题。

## 🏗️ 基本原理

### 加法原理

**原理**：若完成一件事有 $n$ 类不同方法，第 $i$ 类有 $m_i$ 种方法，则总共有：
$$m_1 + m_2 + \cdots + m_n$$
种方法。

**条件**：各类方法互斥（不重叠）。

**示例**：从甲地到乙地，可乘飞机（2班）、火车（3班）、汽车（4班）。共有 $2+3+4=9$ 种选择。

### 乘法原理

**原理**：若完成一件事需 $n$ 个步骤，第 $i$ 步有 $m_i$ 种方法，则总共有：
$$m_1 \times m_2 \times \cdots \times m_n$$
种方法。

**条件**：各步骤独立。

**示例**：

- 3男4女中选1男1女组队：$3 \times 4 = 12$ 种
- 3位数字密码（每位0-9）：$10^3 = 1000$ 种

### 补集原理

**原理**：若全集 $U$ 中满足性质 $P$ 的元素难数，可先数不满足 $P$ 的元素：
$$|P| = |U| - |\neg P|$$

**示例**：1到100中不被3整除的整数个数。

**解**：总数100减去被3整除的个数 $\lfloor 100/3 \rfloor = 33$，答案为67。

### 容斥原理

**两集合形式**：
$$|A \cup B| = |A| + |B| - |A \cap B|$$

**三集合形式**：
$$|A \cup B \cup C| = |A| + |B| + |C| - |A \cap B| - |B \cap C| - |A \cap C| + |A \cap B \cap C|$$

**一般形式**：
$$\left|\bigcup_{i=1}^n A_i\right| = \sum_{k=1}^n (-1)^{k+1} \sum_{1 \leq i_1 < \cdots < i_k \leq n} |A_{i_1} \cap \cdots \cap A_{i_k}|$$

**示例**：1到100中能被2或3整除的整数个数。

**解**：

- $|A_2| = 50$（被2整除）
- $|A_3| = 33$（被3整除）
- $|A_2 \cap A_3| = 16$（被6整除）

答案：$50 + 33 - 16 = 67$

## 📊 排列与组合

### 排列

**定义**：从 $n$ 个不同元素中取 $r$ 个排成一列的方式数。

**公式**：
$$P(n, r) = A_n^r = \frac{n!}{(n-r)!}$$

**特例**：

- $P(n, n) = n!$（全排列）
- $P(n, 1) = n$
- $P(n, 0) = 1$

**示例**：从5人中选3人排成一列：$P(5,3) = 5 \times 4 \times 3 = 60$。

### 组合

**定义**：从 $n$ 个不同元素中取 $r$ 个的方式数（不考虑顺序）。

**公式**：
$$C(n, r) = \binom{n}{r} = \frac{n!}{r!(n-r)!}$$

**性质**：

- $C(n, r) = C(n, n-r)$（对称性）
- $C(n, 0) = C(n, n) = 1$
- $C(n, r) = C(n-1, r) + C(n-1, r-1)$（帕斯卡恒等式）

**示例**：从5人中选3人：$C(5,3) = 10$。

### 二项式定理

**定理**：
$$(x + y)^n = \sum_{k=0}^n \binom{n}{k} x^{n-k} y^k$$

**特例**：

- $(1+1)^n = \sum_{k=0}^n \binom{n}{k} = 2^n$
- $(1-1)^n = \sum_{k=0}^n (-1)^k \binom{n}{k} = 0$（$n \geq 1$）

**组合意义**：$n$ 元素集合的子集数为 $2^n$。

## 🎯 高级计数技术

### 重复组合

**问题**：从 $n$ 类物品中可重复地取 $r$ 个的方式数。

**公式**：
$$C(n+r-1, r) = C(n+r-1, n-1)$$

**证明**（隔板法）：
$r$ 个物品排成一列，用 $n-1$ 个隔板分成 $n$ 组。
共 $n+r-1$ 个位置，选 $r$ 个放物品（或选 $n-1$ 个放隔板）。

**示例**：买3个相同的苹果、梨、桃，共买8个水果的方式数：
$$C(3+8-1, 8) = C(10, 8) = 45$$

### 分配问题

**类型1**：$n$ 个不同球放入 $r$ 个不同盒子，每盒至少一个。

**公式**（斯特林数）：
$$r! S(n, r)$$
其中 $S(n, r)$ 是第二类斯特林数。

**类型2**：$n$ 个相同球放入 $r$ 个不同盒子，每盒至少一个。

**公式**：
$$C(n-1, r-1)$$

**类型3**：$n$ 个不同球放入 $r$ 个相同盒子。

**公式**：
$$S(n, r)$$

### 卡特兰数

**定义**：
$$C_n = \frac{1}{n+1}\binom{2n}{n} = \frac{(2n)!}{(n+1)!n!}$$

**前几项**：$1, 1, 2, 5, 14, 42, 132, \ldots$

**应用场景**：

1. $n$ 对括号的合法匹配数
2. $n+1$ 个叶子的满二叉树个数
3. 从 $(0,0)$ 到 $(n,n)$ 不越过对角线的路径数
4. 凸 $n+2$ 边形的三角剖分数

**递推公式**：
$$C_{n+1} = \sum_{i=0}^n C_i C_{n-i}, \quad C_0 = 1$$

**示例**：3对括号的合法序列：

```
((()))  (()())  (())()  ()(())  ()()()
```

共 $C_3 = 5$ 个。

## 💡 详细示例

### 示例1：错位排列

**问题**：$n$ 个元素的排列中，每个元素都不在原来位置的排列数 $D_n$。

**解**：用容斥原理。

设 $A_i$ 为第 $i$ 个元素在原位置的排列集合。

$$D_n = n! - \left|\bigcup_{i=1}^n A_i\right|$$

由容斥：
$$\left|\bigcup_{i=1}^n A_i\right| = \sum_{k=1}^n (-1)^{k+1} \binom{n}{k} (n-k)! = \sum_{k=1}^n (-1)^{k+1} \frac{n!}{k!}$$

因此：
$$D_n = n! \sum_{k=0}^n \frac{(-1)^k}{k!} \approx \frac{n!}{e}$$

**前几项**：$D_1 = 0, D_2 = 1, D_3 = 2, D_4 = 9, D_5 = 44$

### 示例2：子集和问题

**问题**：集合 $\{1, 2, \ldots, n\}$ 有多少个子集的元素和为偶数？

**解**：

**方法一**：生成函数
$$(1+x)(1+x^2)\cdots(1+x^n)$$
中偶数次项系数和减去奇数次项系数和。

令 $x = -1$：$(1-1)(1+1)\cdots = 0$（$n \geq 1$），故相等。

总数 $2^n$，故各为 $2^{n-1}$。

**方法二**：配对
对每个子集 $S$，考虑 $S \triangle \{1\}$（对称差）。
若 $1 \in S$，去掉；若 $1 \notin S$，加入。
这会改变和的奇偶性，形成偶数和与奇数和子集间的一一对应。

故各为 $2^{n-1}$。

### 示例3：路径计数

**问题**：从 $(0,0)$ 到 $(m,n)$，每次只能向右或向上，有多少条路径？

**解**：
共需 $m$ 次右移和 $n$ 次上移，总计 $m+n$ 步。

选择其中 $m$ 步为右移（或 $n$ 步为上移）：
$$\binom{m+n}{m} = \binom{m+n}{n}$$

### 示例4：整数解计数

**问题**：方程 $x_1 + x_2 + x_3 = 10$ 的非负整数解个数。

**解**：隔板法。

10个1排成一列，用2个隔板分成3组。
共12个位置，选2个放隔板：
$$C(12, 2) = 66$$

若要求 $x_i \geq 1$，令 $y_i = x_i - 1$，则 $y_1 + y_2 + y_3 = 7$，答案为 $C(9, 2) = 36$。

## 📚 总结

### 核心公式速查

| 问题类型 | 公式 |
|---------|------|
| $n$ 个不同元素全排列 | $n!$ |
| 从 $n$ 个取 $r$ 个排列 | $P(n,r) = \frac{n!}{(n-r)!}$ |
| 从 $n$ 个取 $r$ 个组合 | $C(n,r) = \frac{n!}{r!(n-r)!}$ |
| 可重复取 $r$ 个 | $C(n+r-1, r)$ |
| 错位排列 | $D_n = n! \sum_{k=0}^n \frac{(-1)^k}{k!}$ |
| 卡特兰数 | $C_n = \frac{1}{n+1}\binom{2n}{n}$ |

### 解题策略

1. **明确问题**：区分有序/无序、可重复/不可重复
2. **选择原理**：加法（分类）或乘法（分步）
3. **容斥原理**：处理"至少"、"或"类问题
4. **等价转化**：将复杂问题转化为已知模型
5. **验证答案**：小例子检验公式的正确性

### 应用领域

- **概率论**：样本空间计数
- **计算机科学**：算法分析、数据结构
- **统计学**：抽样方法
- **生物信息学**：序列分析
