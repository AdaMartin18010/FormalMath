## 📚 概述

图论算法是计算机科学和离散数学的核心领域，研究在图上求解各种计算问题的有效方法。从最基本的遍历算法到复杂的网络优化，图论算法在社交网络分析、路径规划、调度优化等领域有广泛应用。

本增强版文档将详细阐述：

- 图遍历与最短路径算法
- 网络流与匹配算法
- 最小生成树与连通性算法
- 丰富的计算示例与实现

## 🕰️ 历史发展脉络

### 图论起源 (1736-1850)

**欧拉与七桥问题 (1736)**：
莱昂哈德·欧拉解决哥尼斯堡七桥问题，开创图论。

**基尔霍夫定律 (1847)**：
电路网络分析引入树的概念。

**凯莱公式 (1889)**：
$$\tau(K_n) = n^{n-2}$$
完全图的生成树计数。

### 算法时代 (1950-1980)

**Dijkstra算法 (1959)**：
Edsger Dijkstra发明单源最短路径算法。

**Ford-Fulkerson算法 (1956)**：
网络流问题的开创性方法。

**Prim与Kruskal算法**：
最小生成树的高效算法。

### 现代发展 (1980-至今)

**线性时间算法**：
Gabow-Tarjan等提出近线性时间算法。

**近似与随机算法**：
处理NP难问题的有效方法。

## 🏗️ 基础图算法

### 图的表示

**邻接矩阵**：
$$A_{ij} = \begin{cases} 1 & (i,j) \in E \\ 0 & \text{否则} \end{cases}$$

**邻接表**：
每个顶点维护其邻居列表，空间复杂度 $O(|V| + |E|)$。

### 图遍历算法

**广度优先搜索 (BFS)**：

```
BFS(G, s):
    for each vertex u in G.V - {s}:
        u.color = WHITE
        u.d = ∞
        u.π = NIL
    s.color = GRAY
    s.d = 0
    s.π = NIL
    Q = ∅
    ENQUEUE(Q, s)
    while Q ≠ ∅:
        u = DEQUEUE(Q)
        for each v in G.Adj[u]:
            if v.color == WHITE:
                v.color = GRAY
                v.d = u.d + 1
                v.π = u
                ENQUEUE(Q, v)
        u.color = BLACK
```

**时间复杂度**：$O(|V| + |E|)$

**性质**：

- $v.d$ 等于从 $s$ 到 $v$ 的最短边数（无权图）
- BFS树是最短路径树

---

**深度优先搜索 (DFS)**：

```
DFS(G):
    for each vertex u in G.V:
        u.color = WHITE
        u.π = NIL
    time = 0
    for each vertex u in G.V:
        if u.color == WHITE:
            DFS-VISIT(G, u)

DFS-VISIT(G, u):
    time = time + 1
    u.d = time
    u.color = GRAY
    for each v in G.Adj[u]:
        if v.color == WHITE:
            v.π = u
            DFS-VISIT(G, v)
    u.color = BLACK
    time = time + 1
    u.f = time
```

**时间戳**：

- $u.d$：发现时间
- $u.f$：完成时间
- 区间嵌套性质：$v$ 是 $u$ 的后代当且仅当 $[v.d, v.f] \subseteq [u.d, u.f]$

### 最短路径算法

**Dijkstra算法**：
适用于非负权图的单源最短路径。

```
DIJKSTRA(G, w, s):
    INITIALIZE-SINGLE-SOURCE(G, s)
    S = ∅
    Q = G.V
    while Q ≠ ∅:
        u = EXTRACT-MIN(Q)
        S = S ∪ {u}
        for each vertex v in G.Adj[u]:
            RELAX(u, v, w)
```

**松弛操作**：

```
RELAX(u, v, w):
    if v.d > u.d + w(u, v):
        v.d = u.d + w(u, v)
        v.π = u
```

**时间复杂度**：

- 数组实现：$O(V^2)$
- 二叉堆：$O(E \log V)$
- 斐波那契堆：$O(V \log V + E)$

---

**Bellman-Ford算法**：
处理负权边（无负权环）的单源最短路径。

```
BELLMAN-FORD(G, w, s):
    INITIALIZE-SINGLE-SOURCE(G, s)
    for i = 1 to |G.V| - 1:
        for each edge (u, v) in G.E:
            RELAX(u, v, w)
    for each edge (u, v) in G.E:
        if v.d > u.d + w(u, v):
            return FALSE  // 存在负权环
    return TRUE
```

**时间复杂度**：$O(VE)$

**应用**：检测套利机会（汇率转换）。

## 📐 网络流算法

### 最大流问题

**定义**：
流网络 $G = (V, E)$，容量函数 $c: E \to \mathbb{R}^+$，源点 $s$，汇点 $t$。

流 $f$ 满足：

1. **容量限制**：$0 \leq f(u,v) \leq c(u,v)$
2. **流量守恒**：对所有 $u \in V \setminus \{s,t\}$，$\sum_v f(v,u) = \sum_v f(u,v)$

**最大流**：最大化 $|f| = \sum_v f(s,v) - \sum_v f(v,s)$

---

**Ford-Fulkerson方法**：

**残差网络**：
边 $(u,v)$ 的残差容量：
$$c_f(u,v) = \begin{cases} c(u,v) - f(u,v) & (u,v) \in E \\ f(v,u) & (v,u) \in E \\ 0 & \text{否则} \end{cases}$$

**增广路径**：残差网络中的 $s$-$t$ 路径。

**算法**：

```
FORD-FULKERSON(G, s, t):
    for each edge (u, v) in G.E:
        (u, v).f = 0
    while there exists a path p from s to t in G_f:
        c_f(p) = min{c_f(u, v) : (u, v) in p}
        for each edge (u, v) in p:
            if (u, v) in G.E:
                (u, v).f = (u, v).f + c_f(p)
            else:
                (v, u).f = (v, u).f - c_f(p)
```

**最大流最小割定理**：
$$|f^*| = c(S, T)$$
最大流值等于最小割容量。

---

**Edmonds-Karp算法**：
使用BFS找最短增广路径，时间复杂度 $O(VE^2)$。

**Dinic算法**：
分层网络 + 阻塞流，时间复杂度 $O(V^2E)$。

**Push-Relabel算法**：
更高效的实现，时间复杂度 $O(V^3)$ 或 $O(V^2\sqrt{E})$。

### 二分图匹配

**定义**：
二分图 $G = (X \cup Y, E)$，匹配 $M \subseteq E$ 使得无两条边共享端点。

**最大匹配**：基数最大的匹配。

**匈牙利算法**：

```
HUNGARIAN(G):
    M = ∅
    for each unmatched u in X:
        if FIND-AUGMENTING-PATH(u):
            AUGMENT(M)
    return M
```

**Konig定理**：
二分图中，最大匹配大小 = 最小顶点覆盖大小。

## 🔄 高级图算法

### 最小生成树

**Kruskal算法**：
按权排序边，贪心选择不形成环的边。

```
KRUSKAL(G, w):
    A = ∅
    for each vertex v in G.V:
        MAKE-SET(v)
    sort edges by weight
    for each edge (u, v) in sorted order:
        if FIND-SET(u) ≠ FIND-SET(v):
            A = A ∪ {(u, v)}
            UNION(u, v)
    return A
```

**时间复杂度**：$O(E \log E)$（排序主导）

---

**Prim算法**：
从单点开始，逐步扩展树。

```
PRIM(G, w, r):
    for each u in G.V:
        u.key = ∞
        u.π = NIL
    r.key = 0
    Q = G.V
    while Q ≠ ∅:
        u = EXTRACT-MIN(Q)
        for each v in G.Adj[u]:
            if v in Q and w(u, v) < v.key:
                v.π = u
                v.key = w(u, v)
```

**时间复杂度**：$O(E \log V)$（二叉堆）

---

### 强连通分量

**Kosaraju算法**：

1. 在 $G$ 上运行DFS，按完成时间逆序排列顶点
2. 在转置图 $G^T$ 上按此顺序DFS，每棵DFS树是一个SCC

**Tarjan算法**：
单遍DFS使用栈追踪SCC，时间复杂度 $O(V + E)$。

## 💡 详细示例

### 示例 1：Dijkstra算法执行

**图结构**：

```
    4
s ----→ a
|       | 2
7|       ↓
|       b
↓       | 3
c ←---- d
    1
```

**执行过程**：

| 步骤 | 当前u | 距离d | 说明 |
|-----|-------|-------|------|
| 0 | - | s:0, a:∞, b:∞, c:∞, d:∞ | 初始化 |
| 1 | s | a:4, c:7 | 松弛s的邻居 |
| 2 | a | b:6 | 松弛a的邻居 |
| 3 | d | c:7, b:6 | d尚未访问 |
| 4 | b | - | b已确定 |
| 5 | c | - | c已确定 |

**结果**：

- s→a: 4
- s→b: 6 (s→a→b)
- s→c: 7
- s→d: 不存在路径

### 示例 2：最大流计算

**网络**：

```
s --10→ a --9→  t
|      ↓4     ↑10
|       b ----|
|      ↓6
└----9→ c
```

**迭代过程**：

**第1轮**：增广路径 s→a→t，流量9

- 残差：s→a剩1，a→t满，建立反向边

**第2轮**：增广路径 s→a→b→t，流量1

- s→a饱和，a→b:4→3，b→t:10→9

**第3轮**：增广路径 s→c→b→t，流量6

- s→c:9→3，c→b建立反向，b→t:9→3

**第4轮**：增广路径 s→c→b→a→t（反向），不可行

**最大流**：9 + 1 + 6 = 16

---

### 示例 3：二分图匹配

**二分图**：

- X = {a, b, c}
- Y = {1, 2, 3}
- E = {(a,1), (a,2), (b,2), (b,3), (c,1), (c,3)}

**匈牙利算法执行**：

**初始**：M = ∅

**匹配a**：

- 尝试(a,1)，成功，M = {(a,1)}

**匹配b**：

- 尝试(b,2)，成功，M = {(a,1), (b,2)}

**匹配c**：

- 尝试(c,1)，1已匹配给a
- 尝试让a匹配2，但2已匹配给b
- 尝试让b匹配3，成功
- 增广：c→1, a→2, b→3
- M = {(c,1), (a,2), (b,3)}

**最大匹配大小**：3（完美匹配）

## 🔧 技术实现表征

### Python 图算法库

```python
from collections import deque, defaultdict
from heapq import heappush, heappop
from typing import Dict, List, Set, Tuple, Optional

class Graph:
    """图类实现"""

    def __init__(self, directed: bool = False):
        self.directed = directed
        self.adj: Dict[int, List[Tuple[int, float]]] = defaultdict(list)
        self.vertices: Set[int] = set()

    def add_edge(self, u: int, v: int, weight: float = 1.0):
        """添加边"""
        self.adj[u].append((v, weight))
        self.vertices.add(u)
        self.vertices.add(v)
        if not self.directed:
            self.adj[v].append((u, weight))

    def bfs(self, start: int) -> Dict[int, int]:
        """广度优先搜索"""
        dist = {v: float('inf') for v in self.vertices}
        dist[start] = 0
        queue = deque([start])

        while queue:
            u = queue.popleft()
            for v, _ in self.adj[u]:
                if dist[v] == float('inf'):
                    dist[v] = dist[u] + 1
                    queue.append(v)
        return dist

    def dijkstra(self, start: int) -> Dict[int, float]:
        """Dijkstra最短路径"""
        dist = {v: float('inf') for v in self.vertices}
        dist[start] = 0
        pq = [(0, start)]

        while pq:
            d, u = heappop(pq)
            if d > dist[u]:
                continue
            for v, w in self.adj[u]:
                if dist[u] + w < dist[v]:
                    dist[v] = dist[u] + w
                    heappush(pq, (dist[v], v))
        return dist

    def ford_fulkerson(self, source: int, sink: int) -> float:
        """Ford-Fulkerson最大流"""
        # 残差容量
        residual = defaultdict(lambda: defaultdict(float))
        for u in self.adj:
            for v, w in self.adj[u]:
                residual[u][v] += w

        max_flow = 0.0

        def bfs_path() -> Optional[List[int]]:
            """找增广路径"""
            parent = {source: None}
            queue = deque([source])
            while queue and sink not in parent:
                u = queue.popleft()
                for v in residual[u]:
                    if v not in parent and residual[u][v] > 0:
                        parent[v] = u
                        queue.append(v)
            if sink not in parent:
                return None
            # 重构路径
            path = []
            v = sink
            while v is not None:
                path.append(v)
                v = parent[v]
            return path[::-1]

        while True:
            path = bfs_path()
            if path is None:
                break
            # 计算瓶颈容量
            flow = min(residual[path[i]][path[i+1]] for i in range(len(path)-1))
            # 更新残差网络
            for i in range(len(path)-1):
                u, v = path[i], path[i+1]
                residual[u][v] -= flow
                residual[v][u] += flow
            max_flow += flow

        return max_flow

# 使用示例
if __name__ == "__main__":
    # 创建有向图
    g = Graph(directed=True)
    g.add_edge(0, 1, 10)
    g.add_edge(0, 2, 5)
    g.add_edge(1, 2, 2)
    g.add_edge(1, 3, 1)
    g.add_edge(2, 1, 3)
    g.add_edge(2, 3, 9)

    # 最短路径
    dist = g.dijkstra(0)
    print("从0出发的最短距离:", dist)

    # 最大流
    flow = g.ford_fulkerson(0, 3)
    print(f"最大流: {flow}")
```

## 📚 总结

### 核心算法回顾

| 问题 | 算法 | 时间复杂度 |
|-----|------|----------|
| 图遍历 | BFS/DFS | $O(V+E)$ |
| 单源最短路径(非负权) | Dijkstra | $O(E \log V)$ |
| 单源最短路径(含负权) | Bellman-Ford | $O(VE)$ |
| 最大流 | Edmonds-Karp | $O(VE^2)$ |
| 二分图匹配 | 匈牙利算法 | $O(VE)$ |
| 最小生成树 | Kruskal/Prim | $O(E \log V)$ |

### 与数理逻辑体系的关联

- **可计算性**：图论问题的复杂度分类（P, NP, NPC）
- **组合数学**：图计数、Ramsey理论
- **拓扑学**：图作为1维复形

### 应用价值

1. **网络路由**：OSPF, BGP协议
2. **社交网络**：影响力传播、社区发现
3. **物流优化**：车辆路径、设施选址
4. **电路设计**：布线、时序分析
