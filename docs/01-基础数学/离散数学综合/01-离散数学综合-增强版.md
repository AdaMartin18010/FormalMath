## 📚 概述

离散数学是计算机科学的数学基础，研究离散而非连续的对象。它涵盖逻辑、集合论、组合数学、图论、代数结构等多个领域，为算法设计、数据结构、密码学和计算理论提供了严格的数学框架。

本增强版文档将综合阐述：

- 命题逻辑与谓词逻辑的推演系统
- 集合运算与关系理论
- 代数结构基础
- 丰富的计算示例与应用

## 🕰️ 历史发展脉络

### 逻辑的起源 (公元前4世纪-1850)

**亚里士多德的三段论**：
最早的形式逻辑系统，研究命题间的推理关系。

**布尔代数 (1847)**：
乔治·布尔将逻辑代数化，建立命题与代数运算的对应。

### 集合论的创立 (1874-1900)

**康托尔的革命**：
格奥尔格·康托尔建立朴素集合论，引入无限集合的层次。

**罗素悖论 (1901)**：
揭示朴素集合论的矛盾，推动公理化集合论的发展。

### 现代离散数学 (1930-至今)

**图灵与可计算性 (1936)**：
艾伦·图灵定义算法概念，奠定计算机科学基础。

**香农的信息论 (1948)**：
将布尔代数应用于电路设计，开启数字时代。

## 🏗️ 逻辑与证明

### 命题逻辑

**语法**：
命题公式递归定义：

1. 原子命题 $p, q, r, \ldots$
2. 若 $\varphi$ 是公式，则 $\neg\varphi$ 也是
3. 若 $\varphi, \psi$ 是公式，则 $\varphi \wedge \psi, \varphi \vee \psi, \varphi \rightarrow \psi$ 也是

**真值表**：

| $p$ | $q$ | $\neg p$ | $p \wedge q$ | $p \vee q$ | $p \rightarrow q$ |
|-----|-----|---------|-------------|-----------|------------------|
| T | T | F | T | T | T |
| T | F | F | F | T | F |
| F | T | T | F | T | T |
| F | F | T | F | F | T |

**永真式**：

- 排中律：$p \vee \neg p$
- 矛盾律：$\neg(p \wedge \neg p)$
- 蕴含等价：$p \rightarrow q \equiv \neg p \vee q$

---

### 推理规则

**自然演绎系统**：

**合取规则**：
$$\wedge I: \frac{\varphi \quad \psi}{\varphi \wedge \psi}$$
$$\wedge E_1: \frac{\varphi \wedge \psi}{\varphi} \quad \wedge E_2: \frac{\varphi \wedge \psi}{\psi}$$

**蕴含规则**：
$$\rightarrow I: \frac{[\varphi] \quad \vdots \quad \psi}{\varphi \rightarrow \psi}$$
$$\rightarrow E: \frac{\varphi \rightarrow \psi \quad \varphi}{\psi}$$

**否定规则**：
$$\neg I: \frac{[\varphi] \quad \vdots \quad \perp}{\neg\varphi}$$
$$\neg E: \frac{\varphi \quad \neg\varphi}{\perp}$$

---

### 谓词逻辑

**量词**：

- 全称量词：$\forall x \varphi(x)$ —— "对所有 $x$，$\varphi(x)$ 成立"
- 存在量词：$\exists x \varphi(x)$ —— "存在 $x$ 使 $\varphi(x)$ 成立"

**量词否定**：

- $\neg\forall x \varphi(x) \equiv \exists x \neg\varphi(x)$
- $\neg\exists x \varphi(x) \equiv \forall x \neg\varphi(x)$

**推理规则**：
$$\forall I: \frac{\varphi(x)}{\forall x \varphi(x)} \quad (x \text{ 不在假设中自由出现})$$
$$\forall E: \frac{\forall x \varphi(x)}{\varphi(t)}$$
$$\exists I: \frac{\varphi(t)}{\exists x \varphi(x)}$$
$$\exists E: \frac{\exists x \varphi(x) \quad [\varphi(y)] \quad \vdots \quad \psi}{\psi} \quad (y \text{ 新})$$

## 📐 集合与关系

### 集合运算

**基本运算**：

- 并集：$A \cup B = \{x : x \in A \vee x \in B\}$
- 交集：$A \cap B = \{x : x \in A \wedge x \in B\}$
- 差集：$A \setminus B = \{x : x \in A \wedge x \notin B\}$
- 补集：$A^c = \{x : x \notin A\}$（相对于全集）

**德摩根律**：

- $(A \cup B)^c = A^c \cap B^c$
- $(A \cap B)^c = A^c \cup B^c$

**幂集**：
$$\mathcal{P}(A) = \{B : B \subseteq A\}$$
若 $|A| = n$，则 $|\mathcal{P}(A)| = 2^n$。

---

### 关系理论

**定义**：
$A$ 到 $B$ 的关系 $R \subseteq A \times B$。

**性质**（$A$ 上的关系）：

- **自反**：$\forall a \in A, (a, a) \in R$
- **对称**：$(a, b) \in R \Rightarrow (b, a) \in R$
- **传递**：$(a, b) \in R \wedge (b, c) \in R \Rightarrow (a, c) \in R$
- **反对称**：$(a, b) \in R \wedge (b, a) \in R \Rightarrow a = b$

**特殊关系**：

- **等价关系**：自反、对称、传递
- **偏序**：自反、反对称、传递
- **全序**：偏序 + 可比性

**等价类**：
$$[a]_R = \{b \in A : (a, b) \in R\}$$
等价类构成 $A$ 的划分。

---

### 函数

**定义**：
关系 $f \subseteq A \times B$ 是函数，如果：
$$\forall a \in A, \exists! b \in B, (a, b) \in f$$

**性质**：

- **单射**：$f(a_1) = f(a_2) \Rightarrow a_1 = a_2$
- **满射**：$\forall b \in B, \exists a \in A, f(a) = b$
- **双射**：单射 + 满射

**逆函数**：
双射 $f$ 有逆函数 $f^{-1}$，满足 $f^{-1}(f(a)) = a$。

## 🔄 代数结构

### 群论基础

**定义**：
群 $(G, \cdot)$ 满足：

1. **封闭**：$a \cdot b \in G$
2. **结合**：$(a \cdot b) \cdot c = a \cdot (b \cdot c)$
3. **单位元**：$\exists e, a \cdot e = e \cdot a = a$
4. **逆元**：$\exists a^{-1}, a \cdot a^{-1} = e$

**例子**：

- $(\mathbb{Z}, +)$：整数加法群
- $(\mathbb{Z}_n, +_n)$：模 $n$ 加法群
- 对称群 $S_n$：$n$ 元置换群

**拉格朗日定理**：
若 $H \leq G$，则 $|H|$ 整除 $|G|$。

---

### 布尔代数

**定义**：
$(B, \wedge, \vee, ', 0, 1)$ 满足：

- 交换律、结合律、分配律
- 同一律：$x \wedge 1 = x, x \vee 0 = x$
- 补律：$x \wedge x' = 0, x \vee x' = 1$

**Stone表示定理**：
每个布尔代数同构于某集合的幂集代数。

**应用**：

- 数字电路设计
- 数据库查询优化
- 形式验证

### 示例 1：逻辑证明

**证明**：$p \rightarrow q, q \rightarrow r \vdash p \rightarrow r$（假言三段论）

```
1. p → q        [前提]
2. q → r        [前提]
3.   p          [假设]
4.   q          [→E, 1, 3]
5.   r          [→E, 2, 4]
6. p → r        [→I, 3-5]
```

### 示例 2：集合恒等式证明

**证明**：$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$

```
(⊆) 设 x ∈ A ∩ (B ∪ C)
    则 x ∈ A 且 x ∈ B ∪ C
    情况1：x ∈ B，则 x ∈ A ∩ B，故 x ∈ (A ∩ B) ∪ (A ∩ C)
    情况2：x ∈ C，则 x ∈ A ∩ C，故 x ∈ (A ∩ B) ∪ (A ∩ C)

(⊇) 设 x ∈ (A ∩ B) ∪ (A ∩ C)
    情况1：x ∈ A ∩ B，则 x ∈ A 且 x ∈ B ⊆ B ∪ C
           故 x ∈ A ∩ (B ∪ C)
    情况2：x ∈ A ∩ C，类似可证
```

### 示例 3：等价关系与划分

**问题**：定义 $\mathbb{Z}$ 上的关系 $R$：$(a, b) \in R$ 当且仅当 $3 | (a - b)$

**证明 $R$ 是等价关系**：

- 自反：$a - a = 0 = 3 \cdot 0$，故 $(a, a) \in R$
- 对称：若 $3 | (a - b)$，则 $3 | (b - a)$，故 $(b, a) \in R$
- 传递：若 $3 | (a - b)$ 且 $3 | (b - c)$，则 $3 | ((a - b) + (b - c)) = (a - c)$

**等价类**：

- $[0] = \{\ldots, -6, -3, 0, 3, 6, \ldots\}$
- $[1] = \{\ldots, -5, -2, 1, 4, 7, \ldots\}$
- $[2] = \{\ldots, -4, -1, 2, 5, 8, \ldots\}$

这就是模3同余类 $\mathbb{Z}_3$。

### 示例 4：群论计算

**问题**：在 $S_3$ 中计算 $(1\ 2\ 3) \circ (1\ 2)$

**计算**：
$(1\ 2)$ 的映射：$1 \mapsto 2, 2 \mapsto 1, 3 \mapsto 3$

$(1\ 2\ 3)$ 的映射：$1 \mapsto 2, 2 \mapsto 3, 3 \mapsto 1$

复合 $(1\ 2\ 3) \circ (1\ 2)$：

- $1 \mapsto 2 \mapsto 3$，故 $1 \mapsto 3$
- $2 \mapsto 1 \mapsto 2$，故 $2 \mapsto 2$
- $3 \mapsto 3 \mapsto 1$，故 $3 \mapsto 1$

结果为 $(1\ 3)$。

## 🔧 技术实现表征

### Python 离散数学工具

```python
from typing import Set, Tuple, List, Callable, TypeVar
from functools import reduce
import itertools

T = TypeVar('T')

class SetOperations:
    """集合运算工具"""

    @staticmethod
    def union(A: Set[T], B: Set[T]) -> Set[T]:
        return A | B

    @staticmethod
    def intersection(A: Set[T], B: Set[T]) -> Set[T]:
        return A & B

    @staticmethod
    def difference(A: Set[T], B: Set[T]) -> Set[T]:
        return A - B

    @staticmethod
    def symmetric_difference(A: Set[T], B: Set[T]) -> Set[T]:
        return A ^ B

    @staticmethod
    def power_set(A: Set[T]) -> Set[Tuple[T, ...]]:
        """幂集"""
        result = set()
        for r in range(len(A) + 1):
            for subset in itertools.combinations(A, r):
                result.add(subset)
        return result

    @staticmethod
    def cartesian_product(A: Set[T], B: Set[T]) -> Set[Tuple[T, T]]:
        return set(itertools.product(A, B))

class Relation:
    """关系类"""

    def __init__(self, pairs: Set[Tuple[T, T]]):
        self.pairs = pairs
        self.domain = {x for x, _ in pairs}
        self.codomain = {y for _, y in pairs}

    def is_reflexive(self, universe: Set[T]) -> bool:
        return all((a, a) in self.pairs for a in universe)

    def is_symmetric(self) -> bool:
        return all((b, a) in self.pairs for a, b in self.pairs)

    def is_transitive(self) -> bool:
        for a, b in self.pairs:
            for c, d in self.pairs:
                if b == c and (a, d) not in self.pairs:
                    return False
        return True

    def is_equivalence(self, universe: Set[T]) -> bool:
        return (self.is_reflexive(universe) and
                self.is_symmetric() and
                self.is_transitive())

    def equivalence_classes(self, universe: Set[T]) -> Set[Tuple[T, ...]]:
        """计算等价类"""
        if not self.is_equivalence(universe):
            raise ValueError("Not an equivalence relation")

        classes = []
        remaining = set(universe)
        while remaining:
            a = remaining.pop()
            cls = {a}
            cls.update({b for b in remaining if (a, b) in self.pairs})
            remaining -= cls
            classes.append(tuple(sorted(cls)))
        return set(classes)

class BooleanAlgebra:
    """布尔代数运算"""

    @staticmethod
    def AND(x: bool, y: bool) -> bool:
        return x and y

    @staticmethod
    def OR(x: bool, y: bool) -> bool:
        return x or y

    @staticmethod
    def NOT(x: bool) -> bool:
        return not x

    @staticmethod
    def IMPLIES(x: bool, y: bool) -> bool:
        return (not x) or y

    @staticmethod
    def truth_table(expr: Callable[..., bool], n_vars: int):
        """生成真值表"""
        variables = list(itertools.product([False, True], repeat=n_vars))
        table = []
        for vals in variables:
            result = expr(*vals)
            table.append(vals + (result,))
        return table

# 使用示例
if __name__ == "__main__":
    # 集合运算
    A = {1, 2, 3}
    B = {2, 3, 4}
    print(f"A ∪ B = {SetOperations.union(A, B)}")
    print(f"A ∩ B = {SetOperations.intersection(A, B)}")
    print(f"P(A) 的大小 = {len(SetOperations.power_set(A))}")

    # 关系
    R = Relation({(1, 1), (2, 2), (3, 3), (1, 2), (2, 1)})
    print(f"R 是自反的: {R.is_reflexive({1, 2, 3})}")
    print(f"R 是对称的: {R.is_symmetric()}")
    print(f"R 是传递的: {R.is_transitive()}")

    # 真值表
    def implies(p, q):
        return BooleanAlgebra.IMPLIES(p, q)

    print("\n蕴含的真值表:")
    table = BooleanAlgebra.truth_table(implies, 2)
    for row in table:
        print(f"p={row[0]}, q={row[1]} → {row[2]}")
```
