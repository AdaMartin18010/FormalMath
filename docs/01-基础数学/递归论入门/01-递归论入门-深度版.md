## 📚 概述

递归论（又称可计算性理论）是研究算法本质和计算极限的数理逻辑分支。由哥德尔、丘奇、图灵和克林等数学家在1930年代创立，为计算机科学奠定了理论基础。

本深度版文档将系统阐述：

- 图灵机与λ演算的计算模型
- 可判定性与半可判定性的精确刻画
- 停机问题与Rice定理等不可解性结果
- 递归可枚举集的结构与归约理论

## 🕰️ 历史发展脉络

### 可计算性的探索 (1931-1936)

**哥德尔的递归函数 (1931)**：
在证明不完备性定理时，哥德尔引入了原始递归函数类，用于将元数学命题算术化。

**丘奇的λ演算 (1936)**：
阿尔onzo·丘奇提出λ演算作为可计算性的形式定义，并提出**丘奇论题**：
> 直观可计算函数恰好是λ可定义函数。

**图灵的图灵机 (1936)**：
艾伦·图灵独立提出图灵机模型，证明**停机问题不可解**，并证明与λ可计算性的等价性。

### Post与递归可枚举集 (1940s)

埃米尔·波斯特发展了递归可枚举集理论，提出：

- **Post问题**：是否存在严格介于可判定集与完全不可判定集之间的递归可枚举集
- **归约概念**：多一归约、图灵归约

### 可计算性理论的成熟 (1950-1970)

**优先方法**：
弗里德伯格和穆奇尼克独立解决Post问题，发明优先方法。

**度理论**：
研究不可解性的精细结构，建立图灵度的代数理论。

## 🏗️ 可计算性理论

### 图灵机模型

**定义 1.1** (图灵机)
图灵机 $M = (Q, \Sigma, \Gamma, \delta, q_0, q_\text{accept}, q_\text{reject})$，其中：

- $Q$：有限状态集
- $\Sigma$：输入字母表
- $\Gamma$：带字母表（$\Sigma \subseteq \Gamma$，含空白符$\sqcup$）
- $\delta: Q \times \Gamma \to Q \times \Gamma \times \{L, R\}$：转移函数
- $q_0$：初始状态
- $q_\text{accept}, q_\text{reject}$：接受、拒绝状态

**计算过程**：
配置 $C = (q, w_1, a, w_2)$ 表示状态$q$，带头读$a$，左侧$w_1$，右侧$w_2$。

一步计算 $C \vdash_M C'$ 由 $\delta$ 确定。

**接受条件**：
$M$ 接受输入$w$如果存在计算序列：
$$(q_0, \varepsilon, \sqcup, w) \vdash_M^* (q_\text{accept}, w_1, a, w_2)$$

---

### 递归函数类

**基本函数**：

1. **零函数**：$Z(x) = 0$
2. **后继函数**：$S(x) = x + 1$
3. **投影函数**：$P^n_i(x_1, \ldots, x_n) = x_i$

**构造运算**：

1. **复合**：$h(\vec{x}) = f(g_1(\vec{x}), \ldots, g_m(\vec{x}))$
2. **原始递归**：
   $$f(0, \vec{x}) = g(\vec{x})$$
   $$f(n+1, \vec{x}) = h(n, f(n, \vec{x}), \vec{x})$$
3. **极小化**：$f(\vec{x}) = \mu y [g(y, \vec{x}) = 0]$

**定义 1.2** (部分递归函数)
由基本函数经复合、原始递归和极小化生成的函数类。

**丘奇-图灵论题**：
直观可计算的部分函数 = 图灵可计算函数 = 部分递归函数

### 可判定性

**定义 1.3** (特征函数)
语言 $L \subseteq \Sigma^*$ 的特征函数：
$$\chi_L(w) = \begin{cases} 1 & w \in L \\ 0 & w \notin L \end{cases}$$

**定义 1.4** (可判定/递归)
$L$ 是**可判定**的（或**递归**的），如果 $\chi_L$ 是全递归函数。

等价：存在图灵机 $M$ 对所有输入停机，且接受恰好 $L$。

**例子**：

- 空集、全集可判定
- 偶数集、素数集可判定
- 命题逻辑可满足性可判定

## 📐 不可解性与不完备性

### 停机问题

**定理 2.1** (停机问题不可解)
语言 $H = \{\langle M, w \rangle : M \text{ 接受 } w\}$ 是不可判定的。

**证明**（对角线论证）：

假设 $H$ 可判定，设图灵机 $D$ 判定 $H$。

构造图灵机 $N$：

- 输入 $\langle M \rangle$
- 模拟 $D$ 在 $\langle M, \langle M \rangle \rangle$ 上
- 若 $D$ 接受，则拒绝；若 $D$ 拒绝，则接受

考虑 $N(\langle N \rangle)$：

- 若 $N$ 接受 $\langle N \rangle$，则 $D$ 接受 $\langle N, \langle N \rangle \rangle$，即 $N$ 接受 $\langle N \rangle$，则 $N$ 拒绝 $\langle N \rangle$，矛盾
- 若 $N$ 拒绝 $\langle N \rangle$，类似导出矛盾

故 $H$ 不可判定。$\square$

**推论 2.2**：
补语言 $\overline{H}$ 不是递归可枚举的。

---

### Rice定理

**定理 2.3** (Rice定理)
任何关于程序语义的非平凡性质都是不可判定的。

**形式陈述**：
设 $P$ 是部分递归函数的性质，$P$ 非平凡（对某些函数成立，对某些不成立）。

则 $L_P = \{\langle M \rangle : M \text{ 计算的函数具有性质 } P\}$ 不可判定。

**证明概要**：
假设 $P$ 可判定，且空函数 $\emptyset$ 不具有 $P$。

归约 $H \leq_m L_P$：

- 输入 $\langle M, w \rangle$
- 构造 $M'$：输入$x$时，先模拟 $M(w)$
  - 若 $M$ 停机，则计算具有性质 $P$ 的函数
  - 否则不停机

$M$ 接受 $w$ 当且仅当 $M'$ 计算的函数具有 $P$。

若 $L_P$ 可判定，则 $H$ 可判定，矛盾。$\square$

---

### 哥德尔不完备性定理的证明论版本

**定理 2.4** (语义形式)
任何足够强的一致形式系统 $S$ 都存在不可判定命题。

**证明**（使用停机问题）：

假设 $S$ 可判定所有关于图灵机的命题。

则可判定 $H$：

- 输入 $\langle M, w \rangle$
- 构造公式 $\varphi_{M,w}$ 表示"$M$ 接受 $w$"
- 判定 $S \vdash \varphi_{M,w}$

这与停机问题不可解矛盾。$\square$

## 🔄 递归可枚举集

### 定义与等价刻画

**定义 3.1** (递归可枚举/r.e.)
语言 $L$ 是**递归可枚举**的，如果：

1. 存在图灵机 $M$ 接受 $L$（对 $w \in L$ 停机接受，对 $w \notin L$ 可能不停机）
2. 等价：$L$ 是某个部分递归函数的值域
3. 等价：$L$ 可有效地枚举（存在全递归函数枚举 $L$）

**定理 3.2** (Post定理)
$L$ 是可判定的当且仅当 $L$ 和 $\overline{L}$ 都是递归可枚举的。

**证明**：
($\Rightarrow$) 显然。

($\Leftarrow$) 设 $M_1$ 枚举 $L$，$M_2$ 枚举 $\overline{L}$。

对输入$w$，交错运行 $M_1$ 和 $M_2$：

- 若 $M_1$ 输出 $w$，接受
- 若 $M_2$ 输出 $w$，拒绝

必有其一发生，故可判定。$\square$

---

### 完备集与可归约性

**定义 3.3** (多一归约)
$A \leq_m B$ 如果存在全递归函数 $f$ 使得：
$$x \in A \Leftrightarrow f(x) \in B$$

**定义 3.4** (图灵归约)
$A \leq_T B$ 如果存在图灵机 $M$ 可判定 $A$，使用 $B$ 作为谕示(oracle)。

**定义 3.5** (完备r.e.集)
递归可枚举集 $C$ 是**完备的**，如果对任意r.e.集 $A$，$A \leq_m C$。

**定理 3.6** (停机集完备)
$H$ 是完备递归可枚举集。

**证明**：
设 $A$ 被 $M_A$ 接受。定义：
$$f(w) = \langle M_A, w \rangle$$

则 $w \in A$ 当且仅当 $M_A$ 接受 $w$ 当且仅当 $f(w) \in H$。

故 $A \leq_m H$。$\square$

### 度理论简介

**定义 3.7** (图灵度)
$\deg_T(A) = \{B : B \equiv_T A\}$

**度结构**：

- 偏序：$\mathbf{a} \leq \mathbf{b}$ 如果 $A \leq_T B$
- 最小元：$\mathbf{0}$（可判定集的度）
- 跳运算：$A' = \{e : \varphi_e^A(e) \downarrow\}$

**定理 3.8** (弗里德伯格-穆奇尼克)
存在不可比较的递归可枚举度 $\mathbf{a}, \mathbf{b}$（即 $\mathbf{a} \not\leq \mathbf{b}$ 且 $\mathbf{b} \not\leq \mathbf{a}$）。

## 🧠 思维过程表征

### 对角线方法

**核心模式**：

1. 假设存在完备列表（所有可计算函数、所有可判定集等）
2. 构造对角化对象，在第$n$步与第$n$个对象不同
3. 得出矛盾，证明列表不完备

**应用实例**：

- 康托尔：实数不可数
- 图灵：停机问题不可解
- 哥德尔：算术不完备

### 归约方法

**问题归约**：
将问题$A$转化为问题$B$，证明 $A \leq B$。

**归约链**：
$$A_{TM} \leq_m H_{TM} \leq_m \text{REGULAR}_{TM} \leq_m E_{TM}$$

## 💡 哲学分析与应用

### 丘奇-图灵论题的哲学地位

**论题陈述**：
所有"直观可计算"的函数都是图灵可计算的。

**地位争议**：

- 不是数学定理（涉及直观概念）
- 被普遍接受的科学假说
- 多种计算模型的收敛证据支持

### 应用价值

1. **复杂性理论**：P vs NP问题的理论基础
2. **密码学**：单向函数与不可计算性
3. **程序验证**：停机问题的可判定片段
4. **人工智能**：可学习性的理论极限

## 🔧 技术实现表征

### Python 图灵机模拟器

```python
from dataclasses import dataclass
from typing import Dict, Tuple, Optional

@dataclass
class TuringMachine:
    """图灵机实现"""
    states: set
    alphabet: set
    transitions: Dict[Tuple[str, str], Tuple[str, str, str]]
    start_state: str
    accept_state: str
    reject_state: str

    def step(self, state: str, tape: Dict[int, str], head: int) -> Optional[Tuple]:
        """执行一步计算"""
        symbol = tape.get(head, '_')
        if (state, symbol) not in self.transitions:
            return None
        new_state, new_symbol, direction = self.transitions[(state, symbol)]
        tape[head] = new_symbol
        head += 1 if direction == 'R' else -1
        return new_state, tape, head

    def run(self, input_str: str, max_steps: int = 1000) -> str:
        """运行图灵机"""
        tape = {i: c for i, c in enumerate(input_str)}
        state = self.start_state
        head = 0
        steps = 0

        while steps < max_steps:
            if state == self.accept_state:
                return "ACCEPT"
            if state == self.reject_state:
                return "REJECT"
            result = self.step(state, tape, head)
            if result is None:
                return "REJECT"
            state, tape, head = result
            steps += 1

        return "TIMEOUT"
```

### Lean 4 可计算性形式化

```lean
-- 部分递归函数的定义
inductive Partrec : (ℕ →. ℕ) → Prop
  | zero : Partrec (λ _ => 0)
  | succ : Partrec Nat.succ
  | left {n} (i : Fin n) : Partrec (λ v => v.get i)
  | comp {f g} : Partrec f → Partrec g → Partrec (λ x => f (g x))
  | prec {f g} : Partrec f → Partrec g → Partrec (Nat.rec f g)
  | rfind {f} : Partrec f → Partrec (λ x => Nat.find (f x))

-- 递归可枚举集的定义
def RE (A : Set ℕ) : Prop :=
  ∃ f : ℕ →. ℕ, Partrec f ∧ A = {n | f n ↓}
```
