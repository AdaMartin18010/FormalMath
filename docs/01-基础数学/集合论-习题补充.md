# 集合论习题补充

## 基础题

**习题1** 证明：$A \subseteq B$ 当且仅当 $A \cap B = A$。

**证明**：
($\Rightarrow$) 若 $A \subseteq B$，则对任意 $x \in A$，$x \in B$。
所以 $x \in A \cap B$，$A \subseteq A \cap B$。
显然 $A \cap B \subseteq A$，所以 $A \cap B = A$。

($\Leftarrow$) 若 $A \cap B = A$，则对任意 $x \in A$，$x \in A \cap B$，所以 $x \in B$。
因此 $A \subseteq B$。∎

---

**习题2** 证明德摩根律：$(A \cup B)^c = A^c \cap B^c$。

**证明**：
$x \in (A \cup B)^c \Leftrightarrow x \notin A \cup B \Leftrightarrow x \notin A \text{ 且 } x \notin B \Leftrightarrow x \in A^c \cap B^c$。∎

---

**习题3** 判断：若 $A \cup C = B \cup C$，是否必有 $A = B$？

**解答**：否。

反例：$A = \{1\}$，$B = \{2\}$，$C = \{1, 2\}$。
$A \cup C = B \cup C = \{1, 2\}$，但 $A \neq B$。

---

## 映射与函数

**习题4** 证明：$f: X \to Y$ 是单射当且仅当对任意 $A, B \subseteq X$，$f(A \cap B) = f(A) \cap f(B)$。

**证明**：
($\Rightarrow$) 设 $f$ 单射。
$y \in f(A \cap B) \Rightarrow \exists x \in A \cap B, f(x) = y \Rightarrow y \in f(A) \cap f(B)$。
反之，$y \in f(A) \cap f(B) \Rightarrow \exists a \in A, b \in B, f(a) = f(b) = y$。
由单射，$a = b \in A \cap B$，$y \in f(A \cap B)$。

($\Leftarrow$) 设 $f(a) = f(b)$。
$f(\{a\} \cap \{b\}) = f(\{a\}) \cap f(\{b\}) = \{f(a)\} \cap \{f(b)\} = \{f(a)\} \neq \emptyset$。
所以 $\{a\} \cap \{b\} \neq \emptyset$，$a = b$。∎

---

**习题5** 证明：若 $f: X \to Y$，$g: Y \to Z$ 都是双射，则 $g \circ f$ 也是双射。

**证明**：

- 单射：$(g \circ f)(x_1) = (g \circ f)(x_2) \Rightarrow g(f(x_1)) = g(f(x_2)) \Rightarrow f(x_1) = f(x_2) \Rightarrow x_1 = x_2$
- 满射：对任意 $z \in Z$，存在 $y \in Y$ 使 $g(y) = z$，存在 $x \in X$ 使 $f(x) = y$，所以 $(g \circ f)(x) = z$。∎

---

## 基数

**习题6** 证明：$[0, 1]$ 与 $(0, 1)$ 等势。

**证明**：构造双射 $f: [0, 1] \to (0, 1)$：

$f(0) = 1/2$，$f(1) = 1/3$，$f(1/2) = 1/4$，$f(1/3) = 1/5$，...
$f(x) = x$ 对其他 $x$。

这是双射，所以等势。∎

---

**习题7** 证明：$\mathbb{R}$ 不可数。

**证明**：（Cantor对角线法）

假设 $[0, 1]$ 可数，列举为 $x_1, x_2, ...$。

构造 $y \in [0, 1]$，$y$ 的第 $n$ 位小数与 $x_n$ 的第 $n$ 位不同。

则 $y$ 不在列表中，矛盾。所以 $\mathbb{R}$ 不可数。∎

## 反例

### 反例1：$f(A \cap B) \neq f(A) \cap f(B)$

**反例**：$f: \{1, 2\} \to \{0\}$，$f(1) = f(2) = 0$。
$A = \{1\}$，$B = \{2\}$。
$f(A \cap B) = f(\emptyset) = \emptyset$，$f(A) \cap f(B) = \{0\} \cap \{0\} = \{0\}$。

---

### 反例2：传递关系的并集不传递

**反例**：$R_1 = \{(1, 2)\}$，$R_2 = \{(2, 3)\}$。
$R_1, R_2$ 都传递，但 $R_1 \cup R_2 = \{(1, 2), (2, 3)\}$ 不传递（缺少 $(1, 3)$）。
