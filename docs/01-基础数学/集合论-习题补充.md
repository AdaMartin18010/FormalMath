# 集合论习题补充

集合论是现代数学的基础，由 Cantor 于 19 世纪末创立，经 Zermelo、Fraenkel、von Neumann 等人发展为 ZFC 公理化体系。本节通过精选习题展示基数理论、序数、选择公理与良序定理的核心技巧。

## 1. 集合运算与映射

### 1.1 包含与交并运算

**习题 1.1**. 证明：$A \subseteq B$ 当且仅当 $A \cap B = A$。

**证明**.

$(\Rightarrow)$ 设 $A \subseteq B$。对任意 $x \in A$，有 $x \in B$，故 $x \in A \cap B$，即 $A \subseteq A \cap B$。显然 $A \cap B \subseteq A$，故 $A \cap B = A$。

$(\Leftarrow)$ 设 $A \cap B = A$。对任意 $x \in A$，$x \in A \cap B \subseteq B$，故 $A \subseteq B$。$\square$

**习题 1.2（De Morgan 律）**. 证明 $(A \cup B)^c = A^c \cap B^c$。

**证明**.

$$x \in (A \cup B)^c \Leftrightarrow x \notin A \cup B \Leftrightarrow x \notin A \text{ 且 } x \notin B \Leftrightarrow x \in A^c \cap B^c. \quad \square$$

### 1.2 映射的性质

**习题 1.3**. 证明 $f: X \to Y$ 是单射当且仅当对任意 $A, B \subseteq X$，$f(A \cap B) = f(A) \cap f(B)$。

**证明**.

$(\Rightarrow)$ 设 $f$ 单射。$f(A \cap B) \subseteq f(A) \cap f(B)$ 恒成立。反之，设 $y \in f(A) \cap f(B)$，则存在 $a \in A, b \in B$ 使 $f(a) = f(b) = y$。由单射，$a = b \in A \cap B$，故 $y \in f(A \cap B)$。

$(\Leftarrow)$ 设 $f(a) = f(b)$。取 $A = \{a\}, B = \{b\}$，则 $f(A) \cap f(B) = \{f(a)\} \neq \emptyset$。由条件 $f(A \cap B) = f(A) \cap f(B) \neq \emptyset$，故 $A \cap B \neq \emptyset$，即 $a = b$。$\square$

**习题 1.4**. 若 $f: X \to Y$，$g: Y \to Z$ 都是双射，则 $g \circ f$ 也是双射。

**证明**.

- **单射**：$(g \circ f)(x_1) = (g \circ f)(x_2) \Rightarrow g(f(x_1)) = g(f(x_2)) \Rightarrow f(x_1) = f(x_2) \Rightarrow x_1 = x_2$。
- **满射**：对任意 $z \in Z$，存在 $y \in Y$ 使 $g(y) = z$，存在 $x \in X$ 使 $f(x) = y$，故 $(g \circ f)(x) = z$。$\square$

## 2. 基数理论

### 2.1 可数与不可数

**习题 2.1**. 证明 $[0, 1]$ 与 $(0, 1)$ 等势。

**证明**. 构造双射 $f: [0, 1] \to (0, 1)$。枚举 $(0, 1)$ 中的序列 $a_n = \frac{1}{n+1}$（$n \geq 1$）。定义：

$$f(0) = a_1 = \frac{1}{2}, \quad f(1) = a_2 = \frac{1}{3}, \quad f(a_n) = a_{n+2} \text{（$n \geq 1$）},$$

$f(x) = x$ 对其他 $x \in (0, 1) \setminus \{a_n\}$。验证 $f$ 为双射。$\square$

**习题 2.2（Cantor 定理）**. 对任意集合 $A$，$|A| < |\mathcal{P}(A)|$。

**证明**. 假设存在满射 $f: A \to \mathcal{P}(A)$。构造 $B = \{x \in A : x \notin f(x)\} \in \mathcal{P}(A)$。若 $B = f(b)$ 对某 $b \in A$：

- 若 $b \in B$，则由定义 $b \notin f(b) = B$，矛盾；
- 若 $b \notin B$，则由定义 $b \in f(b) = B$，矛盾。

故不存在满射，$|A| < |\mathcal{P}(A)|$。$\square$

**习题 2.3**. 证明 $\mathbb{R}$ 不可数。

**证明（Cantor 对角线法）**. 假设 $[0, 1]$ 可数，列举为 $x_1, x_2, \dots$，十进制展开 $x_n = 0.a_{n1}a_{n2}\dots$。构造 $y = 0.b_1b_2\dots$，其中 $b_n \neq a_{nn}$（且避免 9-循环）。则 $y \in [0, 1]$ 但 $y \neq x_n$ 对所有 $n$，矛盾。$\square$

### 2.2 Schröder-Bernstein 定理

**习题 2.4**. 若存在单射 $f: A \to B$ 和 $g: B \to A$，则存在双射 $h: A \to B$。

**证明概要**. 定义 $A$ 上的"祖先"映射。对 $x \in A$，追溯 $g^{-1}(x), f^{-1}(g^{-1}(x)), \dots$（若存在）。按祖先链终止于 $A$ 或 $B$ 或无限，将 $A$ 分为三部分。类似分 $B$。在有限链终止于 $A$ 的部分用 $f$，终止于 $B$ 的部分用 $g^{-1}$，无限链部分任意（如 $f$），构造双射。$\square$

## 3. 选择公理及其等价形式

### 3.1 Zorn 引理

**习题 3.1**. 证明选择公理蕴含 Zorn 引理：若偏序集 $(P, \leq)$ 中每个链都有上界，则 $P$ 存在极大元。

**证明概要**. 假设 $P$ 无极大元。对每个链 $C$，选取严格上界 $u(C) > C$。利用 Hartogs 数构造超限链，得到从序数到 $P$ 的严格递增映射。但 $P$ 的集合大小有限制，矛盾。$\square$

### 3.2 良序定理

**习题 3.2**. 证明良序定理与选择公理等价。

**证明概要**.

- AC $\Rightarrow$ 良序定理：对集合 $X$，考虑所有 $(A, <_A)$ 其中 $A \subseteq X$ 且 $<_A$ 为 $A$ 上的良序。按序型比较定义偏序，用 Zorn 引理得极大元，必为 $X$ 本身。
- 良序定理 $\Rightarrow$ AC：对集合族 $\{A_i\}_{i \in I}$，$\bigsqcup A_i$ 可良序，取每个 $A_i$ 的最小元构造选择函数。$\square$

## 4. 序数与超限归纳

### 4.1 序数的定义

**习题 4.1**. 证明每个良序集都与唯一序数同构。

**证明**. 对良序集 $(W, <)$ 用超限归纳：定义 $W_\alpha = \{x \in W : \{y : y < x\} \cong \alpha\}$。由良序性，$W = \bigcup_\alpha W_\alpha$，且恰有一个 $\beta$ 使 $W_\beta = W$，即 $W \cong \beta$。唯一性由序数间同构即相等保证。$\square$

### 4.2 超限递归

**习题 4.2**. 利用超限递归定义序数的加法：$\alpha + 0 = \alpha$，$\alpha + (\beta + 1) = (\alpha + \beta) + 1$，$\alpha + \lambda = \sup_{\beta < \lambda} (\alpha + \beta)$（$\lambda$ 为极限序数）。

验证：$1 + \omega = \omega \neq \omega + 1$，故序数加法不交换。

## 5. 例子

### 5.1 例子：$\aleph_0$ 的性质

**习题 5.1**. 证明 $\aleph_0 + \aleph_0 = \aleph_0 \cdot \aleph_0 = \aleph_0$。

**证明**. $\mathbb{N} \cup \mathbb{N} \cong \mathbb{N}$（交错排列），$\mathbb{N} \times \mathbb{N} \cong \mathbb{N}$（Cantor 配对函数 $(m,n) \mapsto \frac{(m+n)(m+n+1)}{2} + n$）。$\square$

### 5.2 例子：连续统假设

**习题 5.2**. 陈述连续统假设（CH）及其在 ZFC 中的独立性。

**解答**. CH：$2^{\aleph_0} = \aleph_1$。Gödel (1938) 证明 $L \models \text{ZFC} + \text{CH}$；Cohen (1963) 用力迫法证明 $\neg\text{CH}$ 也与 ZFC 相容。故 CH 独立于 ZFC。$\square$
