## 学习路径

### 基础路径
```
线性系统理论 → 状态空间分析 → 可控性/可观性 → 状态反馈设计
```

### 进阶路径
```
经典控制 → 现代控制 → 最优控制 → 非线性/鲁棒控制
```

### 应用路径
- **机器人**: 状态空间 + 非线性控制 + 运动规划
- **航空航天**: 最优控制 + 制导导航 + 姿态确定
- **过程控制**: PID + 模型预测控制 + 系统辨识
- **自动驾驶**: 状态估计 + MPC + 强化学习

## 核心定理速查

| 定理 | 描述 | 应用 |
|------|------|------|
| **可控性秩条件** | $\text{rank}[B, AB, \ldots, A^{n-1}B] = n$ | 状态反馈设计 |
| **可观性秩条件** | $\text{rank}[C^T, A^T C^T, \ldots] = n$ | 观测器设计 |
| **Lyapunov稳定性** | 存在 $V > 0$ 使 $\dot{V} < 0$ | 稳定性分析 |
| **Pontryagin极大值原理** | 最优控制的必要条件 | 最优控制 |
| **分离原理** | 控制器和观测器可分别设计 | LQG控制 |

## 系统分类

| 系统类型 | 数学描述 | 分析方法 |
|----------|----------|----------|
| LTI | $\dot{x} = Ax + Bu$ | 频域/时域 |
| 时变 | $\dot{x} = A(t)x + B(t)u$ | 状态转移矩阵 |
| 非线性 | $\dot{x} = f(x, u)$ | Lyapunov/线性化 |
| 离散 | $x_{k+1} = Ax_k + Bu_k$ | Z变换 |
| 随机 | $dx = Ax dt + B dw$ | 伊藤微积分 |
