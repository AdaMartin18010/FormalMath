# 控制论 / Control Theory

**最后更新**: 2026年4月4日

---

## 📋 分支概述

控制论是研究系统动态行为和控制方法的数学理论，广泛应用于工程、生物学、经济学等领域。它提供了分析和设计控制系统的数学工具，从经典的传递函数方法到现代的状态空间理论。

## 📚 核心人物

- **Norbert Wiener (1894-1964)**: 控制论创始人，《控制论》作者
- **Rudolf E. Kálmán (1930-2016)**: 卡尔曼滤波器、可控可观性理论
- **Lev Pontryagin (1908-1988)**: 极大值原理
- **Richard Bellman (1920-1984)**: 动态规划
- **Alexander Lyapunov (1857-1918)**: 稳定性理论奠基人

## 🔗 与其他分支的联系

- **微分方程**: 系统动力学建模
- **线性代数**: 状态空间分析
- **优化理论**: 最优控制、LQR设计
- **概率论**: 随机系统、滤波理论
- **泛函分析**: 无穷维系统控制

## 📖 学习路径建议

1. **先修知识**: 线性代数、微分方程、复变函数
2. **第一阶段**: 经典控制（传递函数、稳定性判据）
3. **第二阶段**: 现代控制（状态空间、可控可观性）
4. **第三阶段**: 最优控制（变分法、LQR）
5. **第四阶段**: 高级专题（非线性控制、自适应控制、鲁棒控制）
