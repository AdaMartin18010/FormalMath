## 1. 问题建模

### 1.1 从物理系统到状态空间

**示例：质量-弹簧-阻尼系统**

质量为 $m$ 的物体连接弹簧(刚度 $k$)和阻尼器(阻尼系数 $c$)，受外力 $u(t)$ 作用。

**牛顿第二定律**:
$$m\ddot{y} + c\dot{y} + ky = u$$

**状态变量选择**:
- $x_1 = y$ (位置)
- $x_2 = \dot{y}$ (速度)

**状态空间模型**:
$$
\begin{aligned}
\dot{x}_1 &= x_2 \\
\dot{x}_2 &= -\frac{k}{m}x_1 - \frac{c}{m}x_2 + \frac{1}{m}u
\end{aligned}
$$

矩阵形式:
$$
\dot{x} = \underbrace{\begin{bmatrix} 0 & 1 \\ -k/m & -c/m \end{bmatrix}}_{A} x + 
\underbrace{\begin{bmatrix} 0 \\ 1/m \end{bmatrix}}_{B} u
$$

输出方程（测量位置）:
$$y = \underbrace{\begin{bmatrix} 1 & 0 \end{bmatrix}}_{C} x$$

### 1.2 一般形式

**定义 1.1** (LTI状态空间): 
$$
\begin{aligned}
\dot{x}(t) &= Ax(t) + Bu(t), \quad x(0) = x_0 \\
y(t) &= Cx(t) + Du(t)
\end{aligned}
$$

其中 $x \in \mathbb{R}^n$ (状态), $u \in \mathbb{R}^m$ (输入), $y \in \mathbb{R}^p$ (输出)。

### 1.3 建模实例：倒立摆

**系统描述**: 小车质量 $M$，摆杆质量 $m$，长度 $l$，摆角 $\theta$。

**非线性方程** (推导略):
$$\begin{aligned}
(M+m)\ddot{x} + ml\ddot{\theta}\cos\theta - ml\dot{\theta}^2\sin\theta &= u \\
l\ddot{\theta} + \ddot{x}\cos\theta - g\sin\theta &= 0
\end{aligned}$$

**线性化** (在 $\theta \approx 0$ 处):
$$\begin{aligned}
(M+m)\ddot{x} + ml\ddot{\theta} &= u \\
\ddot{x} + l\ddot{\theta} - g\theta &= 0
\end{aligned}$$

**状态空间** (定义 $x_1 = x, x_2 = \dot{x}, x_3 = \theta, x_4 = \dot{\theta}$):
$$\dot{x} = Ax + Bu$$

其中:
$$A = \begin{bmatrix} 
0 & 1 & 0 & 0 \\
0 & 0 & -mg/M & 0 \\
0 & 0 & 0 & 1 \\
0 & 0 & (M+m)g/(Ml) & 0
\end{bmatrix}, \quad 
B = \begin{bmatrix} 0 \\ 1/M \\ 0 \\ -1/(Ml) \end{bmatrix}$$

## 2. 数学理论

### 2.1 解的存在唯一性

**定理 2.1** (状态方程解): LTI系统 $\dot{x} = Ax + Bu$ 的解为：
$$x(t) = e^{At}x_0 + \int_0^t e^{A(t-\tau)}Bu(\tau) d\tau$$

其中矩阵指数:
$$e^{At} = \sum_{k=0}^{\infty} \frac{(At)^k}{k!}$$

**定理 2.2** (矩阵指数性质):
1. $\frac{d}{dt}e^{At} = Ae^{At} = e^{At}A$
2. $e^{A(t_1+t_2)} = e^{At_1}e^{At_2}$
3. $e^{0} = I$
4. $(e^{At})^{-1} = e^{-At}$

### 2.2 可控性

**定义 2.1** (可控性): 系统 $(A, B)$ 是可控的，若对任意初始状态 $x_0$ 和目标状态 $x_f$，存在控制输入 $u(t)$ 在有限时间 $T$ 内将系统从 $x_0$ 转移到 $x_f$。

**定理 2.3** (可控性秩条件): 系统 $(A, B)$ 可控当且仅当：
$$\text{rank}\mathcal{C} = \text{rank}[B, AB, A^2B, \ldots, A^{n-1}B] = n$$

**证明概要**: 
- 充分性: 可控性格拉姆矩阵 $W_c = \int_0^T e^{At}BB^Te^{A^Tt}dt$ 正定
- 控制律: $u(t) = B^T e^{A^T(T-t)} W_c^{-1}(x_f - e^{AT}x_0)$

### 2.3 可观性

**定义 2.2** (可观性): 系统 $(A, C)$ 是可观的，若通过输出 $y(t)$ 在有限时间内的测量，可唯一确定初始状态 $x_0$。

**定理 2.4** (可观性秩条件): 系统 $(A, C)$ 可观当且仅当：
$$\text{rank}\mathcal{O} = \text{rank}\begin{bmatrix} C \\ CA \\ \vdots \\ CA^{n-1} \end{bmatrix} = n$$

### 2.4 稳定性

**定义 2.3** (Lyapunov稳定性): 系统 $\dot{x} = Ax$ 是渐近稳定的，若对所有 $x_0$，$x(t) \to 0$ 当 $t \to \infty$。

**定理 2.5** (稳定性判据): 以下等价：
1. $\dot{x} = Ax$ 渐近稳定
2. $A$ 的所有特征值实部为负
3. 存在 $P \succ 0$ 使得 $A^TP + PA \prec 0$ (Lyapunov不等式)

## 3. 算法与方法

### 3.1 状态反馈设计

**问题**: 设计 $u = -Kx$ 使闭环系统 $\dot{x} = (A - BK)x$ 稳定。

**方法1: 极点配置**

**定理 3.1** (极点配置): 若 $(A, B)$ 可控，则对任意期望特征值集合 $\{\lambda_1, \ldots, \lambda_n\}$，存在状态反馈增益 $K$ 使 $A - BK$ 的特征值为期望特征值。

**算法** (Ackermann公式):
$$K = [0, \ldots, 0, 1]\mathcal{C}^{-1}\phi(A)$$

其中 $\phi(s) = \prod_{i=1}^n(s - \lambda_i)$ 是期望特征多项式。

**方法2: LQR设计**

**优化问题**:
$$\min_u \int_0^{\infty} (x^TQx + u^TRu) dt$$

**解**: $u = -Kx$，其中 $K = R^{-1}B^TP$，$P$ 满足Riccati方程：
$$A^TP + PA - PBR^{-1}B^TP + Q = 0$$

### 3.2 观测器设计

**Luenberger观测器**:
$$\dot{\hat{x}} = A\hat{x} + Bu + L(y - C\hat{x})$$

误差动态: $\dot{e} = (A - LC)e$，其中 $e = x - \hat{x}$

**设计**: 选择 $L$ 使 $A - LC$ 的特征值具有负实部。

### 3.3 分离原理

**定理 3.2** (分离原理): 控制器和观测器可独立设计。组合系统的特征值是控制器极点和观测器极点的并集。

## 4. 具体案例

### 4.1 案例：二阶系统控制

**系统**: 质量-弹簧-阻尼系统，$m=1, k=4, c=0.5$

$$A = \begin{bmatrix} 0 & 1 \\ -4 & -0.5 \end{bmatrix}, \quad B = \begin{bmatrix} 0 \\ 1 \end{bmatrix}$$

**开环特征值**: $\lambda = -0.25 \pm 1.98i$ (稳定但响应慢)

**目标**: 将极点配置到 $\lambda_{1,2} = -2$ (更快响应)

**计算**:
1. 可控性矩阵: $\mathcal{C} = [B, AB] = \begin{bmatrix} 0 & 1 \\ 1 & -0.5 \end{bmatrix}$
2. 期望特征多项式: $\phi(s) = (s+2)^2 = s^2 + 4s + 4$
3. Ackermann公式: $K = [0, 1]\mathcal{C}^{-1}\phi(A)$

$$\phi(A) = A^2 + 4A + 4I = \begin{bmatrix} 0 & 3.5 \\ -14 & -2 \end{bmatrix}$$

$$K = [0, 1]\begin{bmatrix} 0.5 & 1 \\ 1 & 0 \end{bmatrix}\begin{bmatrix} 0 & 3.5 \\ -14 & -2 \end{bmatrix} = [3.5, 4]$$

**闭环系统**: $\dot{x} = (A - BK)x = \begin{bmatrix} 0 & 1 \\ -7.5 & -4.5 \end{bmatrix}x$

### 4.2 案例：倒立摆稳定化

**参数**: $M=1, m=0.1, l=0.5, g=9.8$

**开环特征值**: $\{0, 0, 4.43, -4.43\}$ (不稳定！)

**LQR设计**: 
- $Q = \text{diag}(10, 1, 10, 1)$ (惩罚位置和角度)
- $R = 1$ (控制代价)

**求解Riccati方程得**: $K \approx [-3.16, -5.83, 29.8, 8.5]$

**闭环特征值**: $\{-5.2, -4.1, -2.3 \pm 1.5i\}$ (全部稳定)

## 5. 代码实现

### 5.1 Python (Control库)

```python
import numpy as np
import control as ctrl
import matplotlib.pyplot as plt

# 质量-弹簧-阻尼系统参数
m, k, c = 1.0, 4.0, 0.5

# 状态空间矩阵
A = np.array([[0, 1], [-k/m, -c/m]])
B = np.array([[0], [1/m]])
C = np.array([[1, 0]])
D = np.array([[0]])

# 创建系统
sys = ctrl.ss(A, B, C, D)

# 开环极点
print(f"开环极点: {ctrl.poles(sys)}")

# 可控性检验
C_matrix = ctrl.ctrb(A, B)
print(f"可控性矩阵秩: {np.linalg.matrix_rank(C_matrix)}")

# 极点配置
poles = [-2, -2]
K = ctrl.acker(A, B, poles)
print(f"状态反馈增益 K: {K.flatten()}")

# 闭环系统
A_cl = A - B @ K
sys_cl = ctrl.ss(A_cl, B, C, D)
print(f"闭环极点: {ctrl.poles(sys_cl)}")

# 阶跃响应
t, y = ctrl.step_response(sys_cl)
plt.figure(figsize=(10, 4))
plt.plot(t, y, 'b-', linewidth=2)
plt.xlabel('Time (s)')
plt.ylabel('Position')
plt.title('Closed-Loop Step Response')
plt.grid(True)
plt.savefig('step_response.png', dpi=150)
plt.show()
```

### 5.2 LQR设计实现

```python
import numpy as np
from scipy import linalg

def solve_lqr(A, B, Q, R):
    """
    求解连续时间LQR问题
    
    返回: 增益矩阵 K, Riccati解 P
    """
    # 求解代数Riccati方程: A^T P + P A - P B R^{-1} B^T P + Q = 0
    P = linalg.solve_continuous_are(A, B, Q, R)
    
    # 计算增益
    K = np.linalg.inv(R) @ B.T @ P
    
    return K, P

# 倒立摆系统
M, m, l, g = 1.0, 0.1, 0.5, 9.8

A = np.array([
    [0, 1, 0, 0],
    [0, 0, -m*g/M, 0],
    [0, 0, 0, 1],
    [0, 0, (M+m)*g/(M*l), 0]
])

B = np.array([[0], [1/M], [0], [-1/(M*l)]])

# 权重矩阵
Q = np.diag([10, 1, 10, 1])
R = np.array([[1]])

# 求解LQR
K, P = solve_lqr(A, B, Q, R)

print(f"LQR增益矩阵 K: {K.flatten()}")
print(f"闭环特征值: {np.linalg.eigvals(A - B @ K)}")

# 验证Riccati方程
riccati_check = A.T @ P + P @ A - P @ B @ np.linalg.inv(R) @ B.T @ P + Q
print(f"Riccati方程残差范数: {np.linalg.norm(riccati_check):.2e}")
```
