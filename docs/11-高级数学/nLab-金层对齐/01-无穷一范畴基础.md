# (∞,1)-范畴基础：从单纯集到拟范畴

## 1. 引言

** (∞,1)-范畴理论**是当代数学中最具影响力的范式之一，它统一了范畴论与同伦论两种语言，为高阶结构的研究提供了严格而灵活的框架。根据 nLab 的阐述，(∞,1)-范畴是一种弱 ∞-范畴，其中所有 $n \geq 2$ 阶的态射都是可逆的（nLab, [(∞,1)-category](https://ncatlab.org/nlab/show/%28infinity%2C1%29-category)）。这一概念最早可以追溯到 Boardman 与 Vogt 在 1973 年关于同伦不变代数结构的研究，他们引入了**弱 Kan 复形**（weak Kan complex）作为拓扑空间代数模型的推广。随后，André Joyal 在 2002 年的开创性论文 *Quasi-categories and Kan complexes* 中系统发展了**拟范畴**（quasi-category）理论，证明了拟范畴可以承载完整的范畴论构造：极限、余极限、Yoneda 引理、Grothendieck 构造等。Jacob Lurie 在其 2009 年的巨著 *Higher Topos Theory* 中进一步将这一理论推向巅峰，建立了 (∞,1)-层理论与高阶 topos 理论的坚实基础。

本专题旨在建立从**单纯集**（simplicial set）到**拟范畴**再到 **(∞,1)-范畴中极限与余极限**的完整知识链条。我们将严格遵循 nLab 的核心页面与 Lurie 的原始文献，给出精确的数学定义，嵌入 Lean4 形式化代码，并在专节中批判性地审视 nLab 的写作风格及其教学适用性。

## 2. 单纯集：组合拓扑的代数语言

### 2.1 单纯范畴 Δ 与单纯集的定义

**单纯集**是代数拓扑与 (∞,1)-范畴论的基本构件。其抽象定义如下：

**定义 2.1**（nLab, [simplicial set](https://ncatlab.org/nlab/show/simplicial+set)；Eilenberg–Zilber, 1950）。设 **Δ** 为**单纯范畴**（simplex category），其对象为有限全序集
$$
[n] = \{0 < 1 < \cdots < n\}, \quad n \in \mathbb{N},
$$
态射为保序映射。一个**单纯集** $X$ 是指一个从 $\Delta^{\mathrm{op}}$ 到 $\mathbf{Set}$ 的函子：
$$
X : \Delta^{\mathrm{op}} \longrightarrow \mathbf{Set}.
$$
记 $X_n = X([n])$ 为 $X$ 的 **$n$-单形集**（set of $n$-simplices）。对态射 $\delta^i : [n-1] \to [n]$（在第 $i$ 个位置插入，像集遗漏 $i$），函子 $X$ 给出**面映射**（face map）
$$
d_i = X(\delta^i) : X_n \longrightarrow X_{n-1},
$$
而对态射 $\sigma^i : [n+1] \to [n]$（将 $i$ 与 $i+1$ 合并），函子 $X$ 给出**退化映射**（degeneracy map）
$$
s_i = X(\sigma^i) : X_n \longrightarrow X_{n+1}.
$$
这些映射满足如下**单纯恒等式**（simplicial identities）：

$$
\begin{aligned}
d_i d_j &= d_{j-1} d_i \quad &&(i < j), \\
s_i s_j &= s_{j+1} s_i \quad &&(i \leq j), \\
d_i s_j &= \begin{cases} s_{j-1} d_i & (i < j), \\ \mathrm{id} & (i = j \text{ 或 } i = j+1), \\ s_j d_{i-1} & (i > j+1). \end{cases}
\end{aligned}
$$

单纯集的范畴记为 $\mathbf{sSet} = [\Delta^{\mathrm{op}}, \mathbf{Set}]$。

**例 2.2**（标准 $n$-单形）。对每个 $n \geq 0$，Yoneda 函子 $\Delta(-, [n])$ 给出一个单纯集，记为 $\Delta[n]$，称为**标准 $n$-单形**。其 $m$-单形集为
$$
\Delta[n]_m = \mathrm{Hom}_{\Delta}([m], [n]),
$$
按组合数计算可知 $|\Delta[n]_m| = \binom{m+n+1}{n}$。

**例 2.3**（拓扑空间的奇异单纯集）。设 $X$ 为拓扑空间，其**奇异单纯集** $S(X)$ 定义为
$$
S(X)_n = \mathrm{Hom}_{\mathbf{Top}}(\Delta^n_{\mathrm{top}}, X),
$$
其中 $\Delta^n_{\mathrm{top}} = \{(t_0, \ldots, t_n) \in \mathbb{R}^{n+1} : \sum t_i = 1, \; t_i \geq 0\}$ 为标准拓扑 $n$-单形。奇异单纯集 $S(X)$ 总是一个 **Kan 复形**（见定义 2.5）。

### 2.2 角与 Kan 复形

在单纯集中，**角**（horn）是标准单形的某种“边界缺失一片”的子复形。

**定义 2.4**（nLab, [Kan complex](https://ncatlab.org/nlab/show/Kan+complex)）。对 $0 \leq k \leq n$，$n$-单形的 **$k$-角** $\Lambda^k[n]$ 是 $\Delta[n]$ 的子单纯集，由所有满足 $i \neq k$ 的面 $d_i$ 生成。换言之，$\Lambda^k[n]$ 包含 $\Delta[n]$ 的所有真面，唯独缺失与顶点 $k$ 相对的那一面。

**定义 2.5**（Kan 复形）。一个单纯集 $X$ 称为 **Kan 复形**，如果对于任意 $n \geq 1$、任意 $0 \leq k \leq n$，以及任意映射 $f : \Lambda^k[n] \to X$（称为一个**角**），都存在扩张 $g : \Delta[n] \to X$ 使得下图交换：
$$
\xymatrix{
\Lambda^k[n] \ar[r]^{f} \ar[d] & X \\
\Delta[n] \ar@{-->}[ur]_{g}
}
$$

Kan 复形是 **∞-群胚**（∞-groupoid）的单纯模型：每个 $n$-单形都可视为一个 $n$-维可逆胞腔，而 Kan 条件保证了所有高维组合都有“复合”与“逆元”。

## 3. 拟范畴：弱 Kan 复形与 (∞,1)-范畴的单纯模型

### 3.1 拟范畴的定义

如果我们放松 Kan 条件，仅要求**内角**（inner horns）——即 $0 < k < n$ 的角——有填充，就得到了 **(∞,1)-范畴**的单纯模型。

**定义 3.1**（Joyal, 2002; nLab, [quasi-category](https://ncatlab.org/nlab/show/quasi-category)）。一个单纯集 $C$ 称为**拟范畴**（quasi-category）或**弱 Kan 复形**（weak Kan complex），如果对所有 $n \geq 2$ 及所有满足 $0 < k < n$ 的内角 $\Lambda^k[n]$，任意映射 $f : \Lambda^k[n] \to C$ 都可扩张为 $g : \Delta[n] \to C$。

 equivalently（Lurie, HTT, Corollary 2.3.2.2），这等价于要求以下映射为平凡 Kan 纤维化：
$$
\mathrm{Hom}_{\mathbf{sSet}}(\Delta[2], C) \longrightarrow \mathrm{Hom}_{\mathbf{sSet}}(\Lambda^1[2], C).
$$

**直观解释**：
- $C$ 的 0-单形是**对象**；
- $C$ 的 1-单形是**态射**（不必可逆）；
- $C$ 的 2-单形是**2-态射**或“三角形的交换证据”，它表明两条 1-单形的复合与第三条 1-单形之间的同伦；
- 对 $n \geq 2$，$n$-单形编码了高阶同伦信息，而内 Kan 条件保证了高阶态射的可逆性（因为外角不要求填充，1-态射不必可逆）。

**例 3.2**。1. 每个 Kan 复形都是拟范畴。2. 每个普通小范畴 $\mathcal{C}$ 的** nerve** $N(\mathcal{C})$ 都是拟范畴。事实上，$N(\mathcal{C})$ 是 Kan 复形当且仅当 $\mathcal{C}$ 是群胚。

### 3.2 拟范畴中的等价与映射空间

在拟范畴中，1-单形 $f : x \to y$ 称为**等价**（equivalence），如果它在同伦范畴 $hC$ 中是一个同构。同伦范畴 $hC$ 定义为：对象是 $C$ 的 0-单形，态射由 1-单形的同伦类给出，复合由 2-单形的存在性诱导（Joyal, HTT §1.2）。

对于两个拟范畴 $C, D$，它们之间的**映射拟范畴** $\mathrm{Fun}(C, D)$ 简单地就是内部 Hom 单纯集 $\underline{\mathrm{Hom}}_{\mathbf{sSet}}(C, D)$。Joyal 证明了：若 $D$ 是拟范畴，则 $\mathrm{Fun}(C, D)$ 也是拟范畴（Joyal, 2002, Theorem 1.3）。这为 (∞,1)-范畴论中的函子、自然变换、伴随等概念提供了天然的栖身之所。

### 3.3 从拟范畴到 (∞,1)-范畴

**定义 3.3**（nLab, [(∞,1)-category](https://ncatlab.org/nlab/show/%28infinity%2C1%29-category)）。一个 **(∞,1)-范畴**是指任意一个与某个拟范畴范畴等价的模型。具体模型包括：
1. **拟范畴**（quasi-categories）；
2. **单纯 enriching 范畴**（simplicial categories），其中 hom-对象均为 Kan 复形；
3. **Segal 范畴**（Segal categories）；
4. **完备 Segal 空间**（complete Segal spaces）；
5. **A_∞-范畴**（A_∞-categories）。

Joyal、Lurie、Bergner、Dugger–Spivak、Rezk 等人的工作共同表明：上述所有模型在适当的 Quillen 等价意义下都描述了同一个高阶范畴——即 **(∞,1)-范畴的“同伦理论”**本身（Bergner, 2007; Lurie, HTT §2.2–2.3）。

## 4. (∞,1)-范畴中的极限与余极限

### 4.1 锥与同伦锥

在普通范畴论中，函子 $F : \mathcal{I} \to \mathcal{C}$ 的极限由一个**锥**（cone）$(L, \{\pi_i\}_{i \in \mathcal{I}})$ 给出，满足泛性质。在 (∞,1)-范畴中，由于高阶同伦的存在，锥的定义需要“同伦化”。

**定义 4.1**（Lurie, HTT, Definition 1.2.13.4）。设 $C$ 为拟范畴，$K$ 为单纯集（通常取 $K = N(\mathcal{I})$ 为指标范畴的 nerve）。$K$-**形变图**是指一个映射 $p : K \to C$。一个**锥**（或** $K$-左纤维化上的点**）是指 $K^{\triangleleft} \to C$ 的扩张，其中 $K^{\triangleleft}$ 是 $K$ 的**左锥**（left cone），即在 $K$ 上添加一个顶点并将该顶点与 $K$ 的所有顶点连接。

**定义 4.2**（Lurie, HTT, Definition 1.2.13.4）。设 $p : K \to C$ 为拟范畴 $C$ 中的图。$p$ 的**极限**是指左锥扩张 $\bar{p} : K^{\triangleleft} \to C$ 满足以下泛性质：对任意左锥的映射 $q : (K^{\triangleleft})^{\triangleleft} \to C$ 若其在第二个锥顶上的限制与 $\bar{p}$ 一致，则存在唯一的（同伦意义下）填充到第二个锥顶。等价地，若记 $C_{/p}$ 为** slice 拟范畴**（over-quasi-category），则 $\bar{p}$ 是极限当且仅当它在 $C_{/p}$ 中是一个终对象（terminal object）。

**定理 4.3**（Lurie, HTT, Proposition 1.2.13.8）。设 $C$ 为拟范畴，$p : K \to C$ 为任意图。若 $C$ 是**完备**的（complete），则 $p$ 的极限存在；若 $C$ 是**余完备**的（cocomplete），则 $p$ 的余极限存在。

**证明思路**。极限的存在性依赖于 slice 拟范畴 $C_{/p}$ 中的终对象存在。由于 $C$ 是完备拟范畴，可以构造一个映射 $K^{\triangleleft} \to C$ 使得其锥顶对象 $L$ 满足：对任意对象 $X \in C_0$，映射空间 $\mathrm{Map}_C(X, L)$ 等价于 $\mathrm{Map}_{\mathrm{Fun}(K, C)}(\mathrm{const}_X, p)$，其中 $\mathrm{const}_X$ 为常值函子。这一等价性正是极限的泛性质在 (∞,1)-范畴论中的同伦版本。Lurie 的证明使用了 Joyal 模型结构中的内纤维化（inner fibration）与 Cartesian 纤维化技术，详见 HTT §2.4。∎

### 4.2 与经典范畴论的比较

下表总结了 (∞,1)-范畴论与普通 1-范畴论在极限理论中的对应关系：

| 1-范畴论 | (∞,1)-范畴论 | 说明 |
|---------|--------------|------|
| 函子 $F : \mathcal{I} \to \mathcal{C}$ | 映射 $p : N(\mathcal{I}) \to C$ | 指标范畴的 nerve |
| 锥（cone） | 左锥扩张 $K^{\triangleleft} \to C$ | 包含高阶同伦信息 |
| 泛性质（唯一性） | 泛性质（同伦唯一性） | 填充在同伦意义下唯一 |
| 极限对象 $L$ | 极限对象 $L$ | $C_{/p}$ 中的终对象 |
| 等化子（equalizer） | 同伦等化子（homotopy equalizer） | 由 pullback 沿对角线给出 |
| 积（product） | 同伦积（homotopy product） | 由 $K$-锥的填充给出 |

一个关键差异在于：**(∞,1)-极限在同伦意义下唯一，但不必严格唯一**。这与模型范畴中“同伦极限”（homotopy limit）的概念相吻合：若一个模型范畴 $\mathcal{M}$  presents 一个完备拟范畴 $C$，则 $C$ 中的 (∞,1)-极限对应于 $\mathcal{M}$ 中的同伦极限（Lurie, HTT, Proposition 4.2.4.1）。

## 5. Lean4 形式化桥接

项目中的 `FormalMath-Enhanced/lean4/FormalMath/FormalMath/HigherCategoryTheory.lean` 提供了 (∞,1)-范畴核心概念的形式化框架。以下代码展示了单纯集、内 Kan 条件与 ∞-范畴定义在 Lean4 中的表达：

```lean
import FormalMath.CategoryTheory
import FormalMath.MathlibStub.Topology.Homotopy.Basic
import FormalMath.MathlibStub.AlgebraicTopology.SimplicialSet

namespace HigherCategoryTheory

open CategoryTheory

-- Mathlib 中已有单纯集的定义
#check SSet  -- Simplicial Set

/-
## 内 Kan 条件的形式化

内 Kan 条件要求：对任意内角 Λⁿᵢ（0 < i < n），
任意映射 f : Λⁿᵢ → X 都可扩张为 g : Δⁿ → X。
-/

structure InnerHorn (n : ℕ) (i : Fin (n + 1)) where
  inner : 0 < i.val ∧ i.val < n
  boundary : Set (Fin (n + 1) → ℕ)

def IsInnerKan {X : SSet} : Prop :=
  ∀ (n : ℕ) (i : Fin (n + 1)) (h : 0 < i.val ∧ i.val < n)
    (f : InnerHorn n i → X.obj (Opposite.op (SimplexCategory.mk n))),
  ∃ (g : Fin (n + 1) → X.obj (Opposite.op (SimplexCategory.mk n))),
    -- g 扩展 f
    sorry

/-
## ∞-范畴的定义（Lurie）

∞-范畴是满足内 Kan 条件的单纯集，即拟范畴。
-/

structure InfinityCategory where
  underlying : SSet
  is_inner_kan : IsInnerKan (X := underlying)

/-
## 与经典范畴的嵌入

每个普通范畴都可以通过 nerve 嵌入为 ∞-范畴。
-/

def nerve_as_infinity_category (C : Type*) [Category C] : InfinityCategory where
  underlying := nerve C  -- Mathlib 中的 nerve 构造
  is_inner_kan := by
    -- 证明：范畴的 nerve 自动满足内 Kan 条件
    sorry
```

上述代码中，`SSet` 来自 Mathlib4 的代数拓扑库，`nerve` 是范畴 nerve 的已有构造。`IsInnerKan` 用存在量词 `∃` 编码了内角的填充条件，而 `InfinityCategory` 结构则将一个单纯集与其满足的内 Kan 证明打包。需要指出的是，Mathlib4 目前尚未完整覆盖 Joyal 模型结构与 Cartesian 纤维化理论，因此诸如极限的泛性质、slice 拟范畴的终对象特征等高级内容在当前代码中仍以 `sorry` 占位。这恰恰反映了 (∞,1)-范畴形式化是当代 Lean/Mathlib 社区的前沿挑战之一。

## 6. nLab 对照与批判性评论

### 6.1 nLab 的内容深度与准确性

nLab 关于 (∞,1)-category、quasi-category 与 simplicial set 的三个页面在数学准确性上无可挑剔。它们系统地梳理了从 Boardman–Vogt 到 Joyal、Lurie 的完整学术谱系，并精确引用了 HTT 的定理编号（如 Corollary 2.3.2.2、Proposition 1.2.13.8）。页面中对“模型独立方法”（model-independent approach）的讨论尤其出色，直接引用了 Bergner、Barwick–Kan、Toën 等人的工作，展示了不同模型之间的 Quillen 等价网络。

### 6.2 nLab 的写作风格与教学适用性

然而，nLab 的写作风格对于非专家读者而言具有显著的**认知壁垒**：

1. **超链接迷宫**：nLab 页面充满了密集的内部超链接，几乎每个技术术语都链接到另一个页面。这种“网络化”结构虽然有利于研究者快速跳转，但对于系统学习者来说却造成了严重的**注意力碎片化**。例如，在阅读 quasi-category 的定义时，读者可能连续被引导至 Kan complex、inner fibration、model structure for quasi-categories 等页面，最终迷失了最初的学习目标。

2. **前置知识的黑洞**：nLab 假定读者已经熟悉模型范畴、Dwyer–Kan 局部化、homotopy coherent nerve 等高级概念。它很少为初学者提供“从 0 到 1”的直观桥梁。例如，quasi-category 页面虽然提到“弱 Kan 复形是 (∞,1)-范畴的几何模型”，但并未详细解释为什么内角的填充对应于高阶可逆性。

3. **缺乏形式化代码**：作为一个以范畴论与高阶结构为核心的资源，nLab 几乎完全不包含 Lean、Coq 或 Agda 的形式化代码。这与当代数学形式化运动（如 Voevodsky 的 Univalent Foundations）形成了鲜明对比。对于希望将理论转化为可验证证明的学习者而言，nLab 更像是一座“概念图书馆”而非“实践工坊”。

### 6.3 对 FormalMath 项目的启示

FormalMath 的 nLab 金层对齐任务应当吸收 nLab 在**概念精确性**与**文献完整性**上的优势，同时弥补其在**教学结构化**与**形式化桥接**上的不足。具体而言：
- 每篇金层文档必须设定明确的前置知识边界，避免无节制的概念跳转；
- 数学定义之后必须紧跟 Lean4 代码示例，即使当前只能给出 stub；
- 应当建立从“直观解释”到“严格定义”再到“形式化实现”的三段式叙述结构，降低读者的认知负荷。
