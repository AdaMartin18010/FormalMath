# (∞,1)-范畴基础：从单纯形集到拟范畴

> **对标 nLab 页面**：(infinity,1)-category, quasi-category, simplicial set
> **核心文献**：Lurie《Higher Topos Theory》Ch. 1–2; Joyal (2002); Cisinski (2019)

---

## 1. 为什么需要 (∞,1)-范畴？

经典范畴论中，两个对象之间的态射集合 $\operatorname{Hom}(X, Y)$ 是一个**集合**。但在代数拓扑和同伦论中，我们更关心的是**空间之间的连续映射空间** $\operatorname{Map}(X, Y)$，它不仅包含映射本身，还包含映射之间的同伦、高阶同伦等信息。

**核心问题**：
> 如何在严格的范畴论框架中， encode "态射之间的高阶同伦"？

Grothendieck 在《Pursuing Stacks》（1983）中首次系统提出了这一问题。Joyal（2002）和 Lurie（2009）给出的答案是：**拟范畴**（quasi-category）模型。

## 2. 单纯形集回顾

### 2.1 单纯范畴 $\Delta$

**定义 2.1**（Joyal; Lurie HTT 1.2.1）。
**单纯范畴**（simplicial category）$\Delta$ 定义为：

- 对象：有限序数 $[n] = \{0, 1, \dots, n\}$（$n \geq 0$）；
- 态射：保序映射 $[m] \to [n]$。

**定义 2.2**。
一个**单纯形集**（simplicial set）是从 $\Delta^{\mathrm{op}}$ 到集合范畴 $\mathbf{Set}$ 的一个函子：
$$X: \Delta^{\mathrm{op}} \longrightarrow \mathbf{Set}.$$

我们记 $X_n := X([n])$ 为 **$n$-单形**（$n$-simplices），并对每个保序映射 $\alpha: [m] \to [n]$ 有**面映射** $d_i: X_n \to X_{n-1}$ 和**退化映射** $s_i: X_n \to X_{n+1}$。

### 2.2 神经函子与 Yoneda 嵌入

**定义 2.3**（Lurie HTT 1.1.5）。
设 $\mathcal{C}$ 为一个 small 范畴。其**nerve**（神经）是一个单纯形集 $N(\mathcal{C})$，定义为
$$N(\mathcal{C})_n := \operatorname{Fun}([n], \mathcal{C}),$$
即长度为 $n$ 的态射串 $C_0 \to C_1 \to \cdots \to C_n$ 的集合。

**定理 2.4**（Segal 条件; nLab: nerve）。
一个单纯形集 $X$ 同构于某个范畴的神经 $N(\mathcal{C})$ 当且仅当对所有 $n \geq 2$，自然映射
$$X_n \longrightarrow X_1 \times_{X_0} X_1 \times_{X_0} \cdots \times_{X_0} X_1$$
是双射。

**证明概要**。
$(\Rightarrow)$ 若 $X = N(\mathcal{C})$，则 $n$-单形恰好由 $n$ 个可复合的 1-单形唯一决定，且复合唯一。
$(\Leftarrow)$ 反之，若 Segal 条件成立，则可由 $X_0$ 为对象、$X_1$ 为态射，通过 $X_2 \cong X_1 \times_{X_0} X_1$ 定义态射复合，associativity 由 $X_3$ 的唯一提升保证。 ∎

## 3. 拟范畴：弱 Kan 复形

### 3.1 内 Horn 填充条件

**定义 3.1**（Joyal; Lurie HTT 1.1.2.4）。
设 $0 < i < n$。**内 Horn**（inner horn）$\Lambda^n_i$ 是标准 $n$-单形 $\Delta^n$ 中去掉第 $i$ 个面和内部后剩余的子复形。

一个单纯形集 $X$ 称为 **拟范畴**（quasi-category），如果对所有 $0 < i < n$ 和所有映射 $\Lambda^n_i \to X$，都存在**填充**（filler）$\Delta^n \to X$ 使得下图交换：
$$\begin{array}{ccc}
\Lambda^n_i & \longrightarrow & X \\
\downarrow & \nearrow & \\
\Delta^n & &
\end{array}$$

**注记 3.2**。若 $X$ 满足对所有 $0 \leq i \leq n$ 的 Horn 填充条件（包括外 Horn $i=0, n$），则 $X$ 是一个 **Kan 复形**（Kan complex），即拓扑空间的同伦型模型。拟范畴只要求**内 Horn**可填充，因此它允许"有向的"高阶箭头。

### 3.2 从拟范畴到 (∞,1)-范畴

**定理 3.3**（Joyal; Lurie HTT 1.1.2.14）。
设 $\mathcal{C}$ 为一个拟范畴。则：
1. $0$-单形构成 **对象**；
2. $1$-单形 $f: X \to Y$ 构成 **1-态射**（1-morphisms）；
3. $2$-单形 $\sigma$ 的边界为
   $$\begin{array}{ccc}
   & Y & \\
   f \nearrow & \downarrow{h} & \nwarrow g \\
   X & \xrightarrow{\phantom{h}} & Z
   \end{array}$$
   给出 $h$ 与 $g \circ f$ 之间的**同伦**（homotopy），即复合只在同伦意义下良定义；
4. 更高维单形给出高阶同伦信息。

**推论 3.4**。拟范畴中的态射复合在同伦意义下是结合的，且存在同伦等价类层面的真正范畴结构：
$$\operatorname{h}\mathcal{C}(X, Y) := \pi_0\bigl(\operatorname{Map}_{\mathcal{C}}(X, Y)\bigr),$$
其中 $\operatorname{Map}_{\mathcal{C}}(X, Y)$ 是连接 $X$ 与 $Y$ 的 1-单形构成的空间。

## 4. 同伦范畴与等价性

### 4.1 同伦范畴 $\operatorname{h}\mathcal{C}$

**定义 4.1**（Lurie HTT 1.2.3.1）。
设 $\mathcal{C}$ 为拟范畴。其**同伦范畴**（homotopy category）$\operatorname{h}\mathcal{C}$ 是一个普通范畴，定义为：
- 对象：$\mathcal{C}$ 的 $0$-单形；
- 态射：$\operatorname{Hom}_{\operatorname{h}\mathcal{C}}(X, Y) = \pi_0\bigl(\mathcal{C}_{X/\!/Y}\bigr)$，即连接 $X, Y$ 的 1-单形在 2-单形同伦关系下的连通分支集；
- 复合：由 2-单形的填充诱导。

**定理 4.2**（Lurie HTT 1.2.3.7）。
函子 $N(\operatorname{h}\mathcal{C}) \to \mathcal{C}$（将普通范畴的神经嵌入拟范畴）在拟范畴的 Joyal 模型结构中是余纤维化（cofibration），且 $\operatorname{h}\mathcal{C}$ 是满足泛性质的唯一范畴。

### 4.2 等价（Equivalences）

**定义 4.3**（Lurie HTT 1.2.3.6）。
设 $\mathcal{C}$ 为拟范畴，$f: X \to Y$ 为一个 1-态射。若存在 $g: Y \to X$ 使得
$$g \circ f \simeq \operatorname{id}_X, \qquad f \circ g \simeq \operatorname{id}_Y$$
在同伦范畴 $\operatorname{h}\mathcal{C}$ 中成立，则称 $f$ 为 **等价**（equivalence）。

**定理 4.4**（Joyal; Lurie HTT 1.2.4.3）。
一个 1-态射 $f: X \to Y$ 在拟范畴 $\mathcal{C}$ 中是等价当且仅当对所有对象 $Z$，诱导映射
$$\operatorname{Map}_{\mathcal{C}}(Z, X) \longrightarrow \operatorname{Map}_{\mathcal{C}}(Z, Y)$$
是 Kan 复形之间的弱同伦等价。

## 5. 极限与余极限

### 5.1 锥体与泛性质

**定义 5.1**（Lurie HTT 1.2.13.4）。
设 $\mathcal{C}$ 为拟范畴，$K$ 为单纯形集（视为"指标图"）。$K$-**图**（diagram）是一个映射 $p: K \to \mathcal{C}$。$p$ 的**锥体**（cone）是一个映射 $\bar{p}: K^{\triangleleft} \to \mathcal{C}$，使得 $\bar{p}|_K = p$，其中 $K^{\triangleleft}$ 是在 $K$ 中加入一个新的初始顶点得到的单纯形集。

**定义 5.2**（Lurie HTT 1.2.13.5）。
一个锥体 $\bar{p}: K^{\triangleleft} \to \mathcal{C}$ 称为 **($\infty$-)极限锥**（limit cone），如果对所有 $n \geq 0$，映射
$$\operatorname{Map}_{\operatorname{Set}_\Delta}(\Delta^n, \mathcal{C}_{/p}) \longrightarrow \operatorname{Map}_{\operatorname{Set}_\Delta}(\Delta^n, \mathcal{C}_{/\bar{p}(v)})$$
是弱同伦等价，其中 $v$ 是锥体的顶点。

**注记 5.3**。这一定义将经典范畴论中的极限泛性质提升到了同伦层次：不是要求唯一性，而是要求"所有可能的填充"构成一个可缩空间（contractible space）。

### 5.2 与经典范畴论的比较

**定理 5.4**（Lurie HTT 1.2.12.9）。
设 $\mathcal{C}$ 为一个普通范畴，$p: K \to N(\mathcal{C})$ 为神经中的一个图。则 $p$ 在拟范畴 $N(\mathcal{C})$ 中有极限当且仅当对应的图在普通范畴 $\mathcal{C}$ 中有极限，且二者给出的极限对象一致。

**证明概要**。因为 $N(\mathcal{C})$ 中的 Horn 填充是唯一的（Segal 条件），所以"可缩的填充空间"退化为"唯一的填充"，这正是经典极限的定义。 ∎

## 6. nLab 对照与批判性评论

### 6.1 nLab 的写作特征

nLab 页面 `(infinity,1)-category` 的一个显著特征是：
- **高度压缩**：大量使用交叉链接，单页内容往往只给出核心定义和指向其他页面的链接；
- **类型论导向**：强调 (∞,1)-范畴与 **同伦类型论**（HoTT）的联系，特别是 Univalence Axiom；

## 7. Lean4 形式化片段

Mathlib4 目前尚未完整实现拟范畴理论，但已具备相关的组合类型基础。以下给出概念性的 Lean4 `structure` 代码框架：

```lean
import Mathlib.CategoryTheory.Category.Basic
import Mathlib.AlgebraicTopology.SimplicialSet

/-- 拟范畴：满足内 Horn 填充条件的单纯形集 -/
structure QuasiCategory where
  X : SSet  -- 单纯形集
  innerHornFilling : ∀ (n : ℕ) (i : Fin (n + 1)),
    0 < i.val → i.val < n →
    ∀ (f : (SSet.horn n i).obj ⟶ X.obj),
    ∃ (g : (SSet.standardSimplex n).obj ⟶ X.obj),
      f = g ∘ (SSet.hornInclusion n i).obj

/-- 拟范畴中的 1-态射（边） -/
def OneMorphism (Q : QuasiCategory) (x y : Q.X.obj 0) : Type _ :=
  { e : Q.X.obj 1 // Q.X.map (SimplexCategory.δ 0) e = y ∧
                        Q.X.map (SimplexCategory.δ 1) e = x }

/-- 同伦关系：由 2-单形连接的 1-态射 -/
def Homotopic (Q : QuasiCategory) {x y} (f g : OneMorphism Q x y) : Prop :=
  ∃ (σ : Q.X.obj 2),
    Q.X.map (SimplexCategory.δ 0) σ = g.1 ∧
    Q.X.map (SimplexCategory.δ 1) σ = f.1 ∧
    Q.X.map (SimplexCategory.δ 2) σ = Q.X.map (SimplexCategory.σ 0) x
  -- 注：此处为教学性框架，完整的等价关系需引入更精细的组合条件
```

**说明**：Mathlib4 的 `AlgebraicTopology.SimplicialSet` 模块正在发展中。完整的拟范畴理论实现（包括 Joyal 模型结构、Cartesian 闭性、极限理论）是当代形式化数学的重要开放问题之一。

## 8. 习题

**习题 8.1**（Joyal）。证明：一个单纯形集 $X$ 是 Kan 复形当且仅当它是某个群胚的 nerve 的"同伦化"版本。更精确地说，若 $X$ 满足所有 Horn 填充条件（包括外 Horn），则其同伦范畴 $\operatorname{h}X$ 是一个群胚（groupoid），且 $X$ 的每个连通分支都是某个拓扑空间的奇异复形。

**习题 8.2**（Lurie HTT 1.2.4.4）。设 $\mathcal{C}$ 为拟范畴，$f: X \to Y$ 为 1-态射。证明：$f$ 是等价当且仅当存在 2-单形 $\sigma, \tau$ 使得
$$d_0(\sigma) = f,\; d_1(\sigma) = g,\; d_2(\sigma) = \operatorname{id}_X$$
$$d_0(\tau) = g,\; d_1(\tau) = f,\; d_2(\tau) = \operatorname{id}_Y$$
对某个 1-态射 $g: Y \to X$ 成立。

**习题 8.3**（Segal 条件的弱化）。设 $X$ 为单纯形集，满足：对所有 $n \geq 2$ 和所有 $0 < i < n$，Horn $\Lambda^n_i \to X$ 的填充**唯一**。证明 $X$ 同构于某个普通范畴的神经 $N(\mathcal{C})$。

**习题 8.4**（综合）。解释为什么 (∞,1)-范畴中的极限不需要"唯一"，而只需要"可缩的填充空间"。用具体例子说明：在拓扑空间范畴 $\mathbf{Top}$ 中，乘积 $X \times Y$ 作为空间的同伦型，为什么自然地构成一个 (∞,1)-极限？

## 9. 总结

本节从单纯形集出发，经 Kan 复形与拟范畴，系统建立了 (∞,1)-范畴的 Joyal-Lurie 模型：

1. **单纯形集** $X: \Delta^{\mathrm{op}} \to \mathbf{Set}$ 提供了组合基础；
2. **拟范畴**通过**内 Horn 填充条件**弱化了普通范畴的 Segal 条件，允许"同伦意义下的复合"；
3. **同伦范畴** $\operatorname{h}\mathcal{C}$ 将拟范畴投影到普通范畴，而**等价**概念替代了传统同构；
4. **(∞-)极限**通过可缩的锥体填充空间定义，统一了经典极限与同伦极限。

这一理论是现代高阶代数几何（如 derived algebraic geometry）、数学物理（如 TQFT）和形式化数学（如同伦类型论）的共同语言基础。
