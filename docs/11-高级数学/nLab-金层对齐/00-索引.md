# nLab与高等范畴论：$(\infty,1)$-范畴与Topos理论基础

## 1. 单纯形集与拟范畴

**定义 1.1（单纯形集）**. 单纯形集是函子 $X: \Delta^{\text{op}} \to \mathbf{Set}$，其中 $\Delta$ 为有限序数范畴。

**定义 1.2（拟范畴）**. 拟范畴是满足内 Kan 条件的单纯形集：每个内角 $ \Lambda^n_k \to X$（$0 < k < n$）可填充为 $\Delta^n \to X$。

**定理 1.3（Joyal）**. 拟范畴模型 $(\infty, 1)$-范畴：对象是 0-单纯形，1-态射是 1-单纯形，高阶态射是高维单纯形。

## 2. Topos 理论

**定义 2.1（Grothendieck Topos）**. Grothendieck topos 是范畴等价于某小范畴上预层范畴的左正合局部化。

**定理 2.2（Giraud）**. 范畴 $\mathcal{E}$ 是 Grothendieck topos 当且仅当它满足完备性、指数对象存在、子对象分类子存在和有小的生成族。

## 3. 同伦类型论

**定义 3.1（Martin-Löf 类型论）**. 依值类型论，含 Π-类型、Σ-类型、恒等类型和归纳类型。

**定理 3.2（Univalence）**. 等价即相等：$(A =_\mathcal{U} B) \simeq (A \simeq B)$。

## 4. 例子

### 4.1 例子：空间作为 ∞-群胚

在 HoTT 中，类型 $A$ 的恒等类型给出其路径空间，形成 ∞-群胚结构。

### 4.2 例子：有效 Topos

有效 topos $\mathbf{Eff}$ 实现了构造性数学的 realizability 语义，其内部逻辑是高阶直觉主义算术。
