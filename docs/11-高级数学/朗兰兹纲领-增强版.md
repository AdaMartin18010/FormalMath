# 朗兰兹纲领 (Langlands Program)

## 1. 朗兰兹纲领概述

### 1.1 历史背景

朗兰兹纲领由加拿大数学家**罗伯特·朗兰兹**(Robert Langlands)于1967年在一封写给安德烈·韦伊(André Weil)的信中首次提出。这封信虽然简短，却开启了一个连接数学多个分支的宏伟研究计划，被誉为"数学的大统一理论"。

朗兰兹当时年仅31岁，在普林斯顿高等研究院工作。他深受**阿廷互反律**和**赫克理论**的启发，试图建立一个统一的框架来理解数论中的L函数与表示论中的自守形式之间的深刻联系。

### 1.2 核心猜想

朗兰兹纲领的核心是**朗兰兹互反猜想**(Langlands Reciprocity Conjecture)，它断言：

> 对于数域上的每一个n维伽罗瓦表示，都存在唯一的自守表示，使得两者的L函数相等。

这一猜想可以视为**非阿贝尔类域论**的推广。经典类域论描述了阿贝尔伽罗瓦扩张与理想类群之间的关系，而朗兰兹纲领试图将其推广到非阿贝尔情形。

核心数学表述：设$K$为数域，$G_K = \text{Gal}(\overline{K}/K)$为其绝对伽罗瓦群，则存在对应关系：

$$\{\rho: G_K \to GL_n(\mathbb{C})\} \longleftrightarrow \{\text{自守表示 } \pi \text{ of } GL_n(\mathbb{A}_K)\}$$

### 1.3 数论与表示论的桥梁

朗兰兹纲领建立了以下三个数学分支之间的深刻联系：

| 数论 | 表示论 | 几何 |
|------|--------|------|
| 伽罗瓦表示 | 自守表示 | 代数簇的上同调 |
| Artin L-函数 | 自守L-函数 | Hasse-Weil L-函数 |
| 互反律 | 函子性提升 | 镜像对称 |

这种对应关系不仅具有理论意义，还为解决具体数论问题提供了强大工具。

## 2. 局部朗兰兹对应

### 2.1 p进域上的表示

**局部朗兰兹对应**研究的是p进域$F$（如$\mathbb{Q}_p$）上的表示理论。设$F$为p进局部域，其剩余域为$\mathbb{F}_q$。

**定义**：局部朗兰兹对应建立了以下等价关系：

$$\{\text{连续表示 } \rho: W_F \to GL_n(\mathbb{C})\} \cong \{\text{光滑不可约表示 of } GL_n(F)\}$$

其中$W_F$是F的**韦伊群**(Weil group)，是伽罗瓦群的稠密子群。

### 2.2 Weil群

韦伊群$W_F$是局部域$F$的绝对伽罗瓦群$G_F$的一个修正版本，定义为：

$$1 \to I_F \to W_F \to \mathbb{Z} \to 1$$

其中$I_F$是惯性群。韦伊群捕捉了伽罗瓦表示的算术信息，特别是 Frobenius 元素的作用。

**关键性质**：韦伊群上的连续表示等价于在有限指数开子群上平凡的表示，这使得复表示理论得以建立。

### 2.3 L函数

局部L因子是朗兰兹纲领的核心不变量。对于$GL_n(F)$的表示$\pi$，其局部L函数定义为：

$$L(s, \pi) = \prod_{i=1}^{n} (1 - \alpha_i q^{-s})^{-1}$$

其中$\alpha_i$是Satake参数。这些L函数满足重要的函数方程，并且与伽罗瓦表示的Artin L函数相对应。

**定理**（Harris-Taylor, Henniart）：局部朗兰兹对应对于$GL_n$在p进域上是成立的。

## 3. 全局朗兰兹对应

### 3.1 自守表示

**自守表示**是定义在adele环$\mathbb{A}_K$上的表示。设$K$为整体域，$GL_n(\mathbb{A}_K)$的右正则表示作用于自守形式空间：

$$L^2(GL_n(K)\backslash GL_n(\mathbb{A}_K), \omega)$$

其中$\omega$是中心特征。

**关键概念**：
- **尖点表示**(Cuspidal representation)：满足特定衰减条件的自守表示
- **朗兰兹分解**：任何自守表示都可分解为尖点表示的诱导

### 3.2 伽罗瓦表示

全局伽罗瓦表示$\rho: G_K \to GL_n(\overline{\mathbb{Q}}_\ell)$来自：
- 代数簇的$\ell$-进上同调
- 椭圆曲线的Tate模
- 模形式的Galois表示

**兼容性条件**：全局表示在几乎所有位上必须是非分歧的，并且局部化后满足局部朗兰兹对应。

### 3.3 函子性原理

**朗兰兹函子性**(Functoriality)是纲领中最一般的猜想：

> 对于任意两个约化群$G$和$H$，以及它们L群之间的同态$^L G \to {}^L H$，存在相应的自守表示提升映射。

具体地，若$\rho: {}^L G \to {}^L H$为L-同态，则对每个$G$的自守表示$\pi$，应存在$H$的自守表示$\rho_*\pi$，使得：

$$L(s, \rho_*\pi, r) = L(s, \pi, r \circ \rho)$$

对所有$^L H$的有限维表示$r$成立。

## 4. 几何朗兰兹

### 4.1 几何对应

**几何朗兰兹纲领**由Drinfeld和Laumon发展，将算术朗兰兹对应翻译到几何语言中。

设$X$为有限域上的光滑射影曲线，几何朗兰兹对应断言：

$$\{\text{GL}_n\text{-主丛上的Hecke特征层}\} \leftrightarrow \{\text{Bun}_{GL_n}上的不可约}\text{ perverse层}\}$$

其中$\text{Bun}_{GL_n}$是$X$上的$GL_n$-丛的模空间。

**关键工具**：
- **Perverse层**：具有优良性质的几何对象
- **Fourier-Mukai变换**：几何版本的函子性
- **Hecke算子**：作用在模空间上的对应

### 4.2 与物理的关联

几何朗兰兹与理论物理有深刻联系：

**与量子场论的关联**：
- 4维超对称规范理论中的**S-对偶性**与几何朗兰兹对应密切相关
- Kapustin和Witten证明了在特定极限下，S-对偶性蕴含几何朗兰兹对应

**与弦论的关联**：
- D膜配置可以编码朗兰兹对应
- 镜像对称提供了理解函子性的新视角

## 5. 最新进展

### 5.1 费马大定理的证明

朗兰兹纲领最辉煌的成就之一是**费马大定理**的证明。Andrew Wiles在1994年证明了半稳定椭圆曲线的Taniyama-Shimura猜想：

**定理**（Wiles-Taylor）：每条半稳定椭圆曲线都是模的。

这一证明依赖于建立**2维伽罗瓦表示与模形式之间的对应**，这正是$GL_2$情形下的朗兰兹对应。

### 5.2 其他重大突破

**函数域情形**（L. Lafforgue, 2002）：证明了函数域上$GL_n$的全局朗兰兹对应，获菲尔兹奖。

**数域局部情形**（Harris-Taylor, Henniart）：建立了p进域上$GL_n$的局部朗兰兹对应。

**正特征情形**（Drinfeld）：对函数域证明了$GL_2$的对应。

**最新进展**（Peter Scholze）：使用perfectoid空间发展了新工具，为p进朗兰兹对应提供了新框架。

## 6. 未来展望

朗兰兹纲领仍然是数学中最活跃的研究领域之一，当前的研究方向包括：

1. **拓展到更一般的约化群**
2. **建立p进朗兰兹对应的完备理论**
3. **深化与物理的联系**
4. **发展计算工具验证具体猜想**

朗兰兹纲领的成功不仅在于其数学深度，更在于它展示了数学统一性的美妙图景。
