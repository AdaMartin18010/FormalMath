## 1. 概述

同伦论与代数拓扑是20世纪数学最具影响力的领域之一。2024-2025年，这一领域在等变同伦论、动机同伦论和Chromatic同伦论等方向取得了重大突破。

**2024-2025年核心主题**:

| 领域 | 关键进展 | 主要贡献者 |
|------|---------|-----------|
| 等变同伦论 | 等变$v_1$-自映射、Chromatic等变理论 | Balderrama, Hou, Zhang |
| Motivic同伦论 | 完美概形上的理论、对偶性结果 | Bachmann, Hekking, Wolters |
| Chromatic同伦论 | 望远镜猜想的反例、机器证明 | Burklund等; Lin, Wang, Xu |
| 同伦类型论 | 立方体类型论的高效实现 | Anders Mortberg等 |

**数学框架**:

同伦论研究空间在连续形变下的不变量。核心问题包括：

$$\pi_n(S^k) = ? \quad \text{(球面的同伦群)}$$

这是数学中最困难的问题之一，即使在今天，我们对其了解仍然有限。

## 2. 等变稳定同伦论的进展

### 2.1 等变稳定同伦范畴

**等变同伦论**研究带有群作用的空间的同伦性质。对于有限群$G$，**等变稳定同伦范畴**$Sp^G$是拓扑学的基本工具。

**关键构造**:

- **genuine $G$-谱**: 对所有$G$的闭子群$H$有Mackey函子结构
- **Borel谱**: 忽略非平凡子群的信息
- **几何不动点**: 提取子群$H$的几何不动点

**Mackey函子**: 对于$G$-谱$X$，其同伦群构成Mackey函子：

$$\underline{\pi}_n(X): G/H \mapsto \pi_n^H(X)$$

这是等变同伦论的核心代数不变量。

### 2.2 2025年突破：等变Chromatic同伦论

**Chromatic分层**将稳定同伦论按照复杂程度分层。在等变情形，这一理论在2025年取得重要进展。

**$v_1$-自映射** (Balderrama-Hou-Zhang, arXiv:2503.15852, 2025):

**定理**: 对于有限循环群$C_{p^n}$，存在等变$v_{1,\beta}$-自映射。

这在等变Chromatic同伦论中具有重要意义，因为它提供了等变$K(1)$-局部化的计算工具。

**等变Chromatic谱**:

$$Sp^G \supset Sp^G_{K(0)} \subset Sp^G_{K(1)} \subset \cdots \subset Sp^G_{K(n)} \subset \cdots$$

**等变Bousfield格**的结构仍然是活跃的研究方向。

### 2.3 Kervaire不变量问题的新视角

**Kervaire不变量问题**是球面同伦群理论中的核心问题。

**历史回顾**:

- Adams (1960): 解决Hopf不变量1问题
- Hill-Hopkins-Ravenel (2016): 证明$j \geq 7$时$h_j^2$支撑非平凡微分
- **2025年进展**: 对$h_j^3$的微分进行计算

**Adams谱序列** (Behrens等, Inventiones 2025):

在Adams滤过3中，对于$j \geq 6$，证明了$h_j^3$支撑非平凡的$d_4$-微分：

$$d_4(h_j^3) \neq 0$$

这确认了Mahowald的猜想。

**关键创新**: 使用两种不同的稳定同伦论形变：

1. $\mathbb{C}$-动机稳定同伦论
2. $\mathbb{F}_2$-合成同伦论

## 3. Motivic同伦论与代数几何

### 3.1 Motivic上同调理论

**Motivic同伦论**由Voevodsky开创，将代数拓扑的方法应用于代数几何。

**核心构造**: 对于概形$X$，定义**动机稳定同伦范畴**$SH(X)$。

**动机上同调**:

$$H^{p,q}_{\mathcal{M}}(X, \mathbb{Z}) = [X_+, S^{p,q} \wedge H\mathbb{Z}]$$

其中$S^{p,q} = S^{p-q} \wedge \mathbb{G}_m^{\wedge q}$是双分次球面。

**与Chow群的关系**:

$$H^{2n,n}_{\mathcal{M}}(X, \mathbb{Z}) \cong CH^n(X)$$

### 3.2 2025年新发展

**完美概形上的Motivic同伦论** (Dahlhausen-Hekking-Wolters, arXiv:2510.01390, 2025):

**定理**: 对于完美概形，动机同伦论具有良好的性质，包括紧生成性和对偶性。

**动机对偶性** (arXiv:2508.00064, 2025):

对于$KGL$-模（动机$K$-理论谱），建立了对偶性理论：

$$D_X: KGL\text{-Mod}_X^{op} \to KGL\text{-Mod}_X$$

**动机稳定性**的新结果:

| 结果 | 作者 | arXiv | 时间 |
|------|------|-------|------|
| 正则余连通环的$K$-理论 | Haoran | arXiv:2112.14723 | 2025年讨论 |
| 动机同伦论中的完美化 | Timmy | arXiv:1812.07506 | 2025年讨论 |
| 不稳定动机与实étale同伦论 | Levi | arXiv:2501.15651 | 2025-02 |

**层范畴的动机** (Jiantong, arXiv:2503.11338, 2025):

**定理**: 每个动机都是某个稳定$\infty$-范畴的动机。

这建立了动机理论与高阶范畴论之间的深刻联系。

### 3.3 与代数K理论的联系

**Bloch-Kato猜想**的证明（Voevodsky）展示了动机理论与代数$K$-理论的密切联系。

**2025年进展**:

- **Motivic作用猜想** (Horawa-Li, arXiv:2506.01699): 对于Doi-Naganuma提升的动机作用猜想
- **对数动机的同伦t-结构** (Merici, arXiv:2512.05051): 对数动机的计算

**Atiyah-Hirzebruch谱序列**:

$$E_2^{p,q} = H^{p,-q}_{\mathcal{M}}(X, \mathbb{Z}(n)) \Rightarrow K_{-p-q}(X)^{(n)}$$

这连接了动机上同调与代数$K$-理论。

## 4. Chromatic同伦论分层

### 4.1 Morava E-理论与稳定同伦群

**Chromatic分层**是理解球面稳定同伦群的强大框架。

**Chromatic层**:

对于素数$p$，稳定同伦论按照高度$n$分层：

- **高度0**: 有理同伦论 ($K(0) = H\mathbb{Q}$)
- **高度1**: $K$-理论和$K(1)$-局部化（$p$-完全$K$-理论）
- **高度$n$**: Morava $K$-理论$K(n)$

**Morava $E$-理论**$E_n$是高度$n$的Lubin-Tate理论，在Chromatic同伦论中扮演核心角色。

**$K(n)$-局部化**:

$$L_{K(n)}: Sp \to Sp_{K(n)}$$

这是理解稳定同伦群的主要工具。

### 4.2 望远镜猜想的新进展

**望远镜猜想** (Ravenel, 1984) 是Chromatic同伦论中最重要的未解决问题之一。

**陈述**: 望远镜局部化$L_n^f$与Smith-Toda复形$V_n$的$B(n)$-局部化一致。

**历史**:

- 1984: Ravenel提出
- 1990: 证明$n=1$情形
- **2023**: Burklund-Hahn-Levy-Schlank提出$K$-理论反例

**2025年影响**:

望远镜猜想的反例彻底改变了我们对Chromatic分层的理解，表明实际的稳定同伦论比预期更复杂。

**新的研究方向**:

1. 理解望远镜局部化的真实性质
2. 寻找替代的分层方法
3. 计算$L_n^f$的精确行为

### 4.3 机器证明在同伦论中的应用

**2025年突破**: Weinan Lin, Guozhen Wang, 和 Zhouli Xu (arXiv:2412.10876) 使用**机器证明**解决Adams微分问题。

**定理**: 对于Adams谱序列中的微分和扩张问题，机器证明提供了严格的验证。

**技术方法**:

1. 将同伦论问题转化为代数计算
2. 使用计算机代数系统验证
3. 人工验证关键步骤

**意义**:

- 这是同伦论中首次大规模使用机器证明
- 为其他复杂计算问题提供了模板
- 展示了人机协作在纯数学中的潜力

**Adams微分公式** (Behrens等, 2025):

对于$h_j^3$类，在Adams谱序列中：

$$d_4(h_j^3) \neq 0 \quad \text{for} \quad j \geq 6$$

这使用了$\mathbb{C}$-动机和合成同伦论的结合。

## 5. 同伦类型论（HoTT）的新发展

### 5.1 立方体类型论

**同伦类型论** (HoTT) 由Vladimir Voevodsky开创，将同伦论与类型论统一。

**立方体类型论**是HoTT的变体，使用立方体而非路径作为基本构造。

**2025年进展**:

- **高效类型检查器**: Anders Mortberg等人开发了新的立方体类型检查器，显著提高了计算效率
- **合成同伦论**: 在立方体类型论中综合地证明经典同伦论结果

**立方体类型论的基本构造**:

```
Path A a b : Type
  -- 类型A中从a到b的路径

Cube : Type
  -- 高维立方体，表示高阶同伦

Composition : ...
  -- 立方体的复合运算
```

### 5.2 综合同伦论

**综合同伦论**在类型论中直接处理同伦概念，无需显式的拓扑空间。

**关键原理**:

1. **单值性公理** (Univalence Axiom):
   $$(A =_\mathcal{U} B) \simeq (A \simeq B)$$
   相等与等价一致

2. **高阶归纳类型** (HITs):
   允许通过生成元和关系直接构造空间

**经典结果的综合证明**:

| 结果 | 证明者 | 时间 |
|------|--------|------|
| $\pi_1(S^1) \cong \mathbb{Z}$ | Licata-Shulman | 2013 |
| $\pi_n(S^n) \cong \mathbb{Z}$ | 多种形式化 | 2015-2020 |
| 五引理 | HoTT库 | 2020 |
| Blakers-Massey定理 | Favonia等 | 2016 |

### 5.3 2025年形式化进展

**Lean/Mathlib4中的同伦论** (2025):

- 基本群的形式化
- 覆盖空间理论
- 纤维丛的形式化开始

**Cubical Agda的发展**:

- 新的计算规则
- 性能优化
- 更多经典结果的计算机验证

**未来方向**:

1. **高维同伦群**: 形式化更复杂的计算
2. **谱序列**: 在类型论中处理谱序列
3. **等变HoTT**: 发展等变同伦论的类型论版本

## 6. 开放问题与未来方向

### 6.1 核心开放问题

**问题1**: 球面稳定同伦群的完全计算

即使对于简单的维数，球面同伦群仍然未知：

$$\pi_n^S = \pi_{n+k}(S^k) \quad (k \gg n)$$

已知的结果仅覆盖低维情形。

**问题2**: 等变Chromatic分层的完整理解

等变$K(n)$-局部化的结构仍然神秘，特别是：

- 等变望远镜猜想
- 等变Morava $E$-理论的几何
- $RO(G)$-分次的Chromatic谱序列

**问题3**: Motivic同伦论中的Bloch-Beilinson猜想

动机上同调与代数循环的精确关系仍然是中心问题。
