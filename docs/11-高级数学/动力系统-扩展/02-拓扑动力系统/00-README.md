# 拓扑动力系统

## 概述

拓扑动力系统研究连续映射或流在拓扑空间上的迭代行为，关注轨道的长期统计性质和拓扑结构。

- 移位空间 $\Sigma_n^+$

- 编码应用

### 遍历理论 (Ergodic Theory)
- 保测变换
- 遍历性、混合性、弱混合
- 熵理论 (Kolmogorov-Sinai熵)
- 谱理论

## 数学思想

拓扑动力系统揭示：**混沌可以通过拓扑工具严格定义**。即使系统不可积，统计方法仍能给出确定性结论。

## 历史发展

- **Poincaré**: 回归定理
- **Birkhoff**: 遍历定理
- **Kolmogorov & Sinai**: 熵理论 (1958)
- **Ornstein**:  Bernoulli移位分类 (1970)
- **Bowen & Ruelle**: 热力学形式化

## 与其他主题的联系
