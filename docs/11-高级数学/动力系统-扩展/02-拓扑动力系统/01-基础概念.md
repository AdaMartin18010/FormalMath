# 拓扑动力系统 - 基础概念

## 一、离散与连续动力系统

### 离散动力系统

$$(X, f), \quad f: X \to X$$

其中 $X$ 是拓扑空间，$f$ 是连续映射。

**轨道**: $O(x) = \{f^n(x): n \geq 0\}$

**例子**:

- 圆周旋转：$R_\alpha(x) = x + \alpha \pmod{1}$
- 倍角映射：$T(x) = 2x \pmod{1}$
- 帐篷映射

### 连续动力系统 (流)

$$(X, \phi_t), \quad \phi_t: X \to X, \quad t \in \mathbb{R}$$

满足群性质：$\phi_0 = \text{id}$，$\phi_{t+s} = \phi_t \circ \phi_s$

**关系**: 时间-1映射 $f = \phi_1$ 生成离散系统

## 二、拓扑传递性与混合性

### 拓扑传递性 (Topological Transitivity)

**定义**: 存在 $x \in X$ 使得轨道 $O(x)$ 在 $X$ 中稠密。

**等价条件**: 对任意非空开集 $U, V$，存在 $n > 0$ 使得 $f^n(U) \cap V \neq \emptyset$

### 拓扑混合性 (Topological Mixing)

**定义**: 对任意非空开集 $U, V$，存在 $N$ 使得对所有 $n \geq N$，$f^n(U) \cap V \neq \emptyset$

**层次**:

```

混合 > 弱混合 > 传递

```

### 精确性 (Exactness)

对任意正测度集 $A$，$\bigcup_{n=0}^\infty f^n(A)$ 满空间（保测情形）。

## 三、极小性

### 极小系统 (Minimal System)

**定义**: 每个轨道都在 $X$ 中稠密。

**等价**: 无真闭不变子集

**例子**:

- 无理旋转 $R_\alpha$（$\alpha \notin \mathbb{Q}$）
- Sturmian序列
- 符号空间的极小子移位

### 唯一遍历性 (Unique Ergodicity)

**定义**: 唯一的不变Borel概率测度。

**定理**: 唯一遍历 + 极小 $\Leftrightarrow$ 一致分布

**意义**: 时间平均 = 空间平均（对所有点）

## 四、拓扑熵

### 定义 (Bowen-Dinaburg)

**$(n, \epsilon)$-分离集**: 子集 $E \subset X$ 满足 $x \neq y \in E$ 时存在 $0 \leq k < n$ 使得 $d(f^k(x), f^k(y)) \geq \epsilon$

**拓扑熵**:
$$h_{\text{top}}(f) = \lim_{\epsilon \to 0} \limsup_{n \to \infty} \frac{\log s_n(\epsilon)}{n}$$

其中 $s_n(\epsilon)$ 是最大$(n, \epsilon)$-分离集基数。

### 基本性质

- 共轭不变性
- $h(f^n) = |n|h(f)$

- $h(f \times g) = h(f) + h(g)$

### 计算示例

| 系统 | 拓扑熵 |
|------|--------|
| 圆周旋转 | 0 |
| 倍角映射 | $\log 2$ |
| 全移位 $\sigma$ 在 $\Sigma_n$ | $\log n$ |

## 五、符号空间与移位

### 全移位空间

**符号空间**: $\Sigma_n = \{1, 2, ..., n\}^{\mathbb{Z}}$（双侧）或 $\Sigma_n^+ = \{1, ..., n\}^{\mathbb{N}}$（单侧）

**度量**:
$$d(x, y) = 2^{-k}, \quad k = \min\{|i|: x_i \neq y_i\}$$

**移位映射**: $(\sigma x)_i = x_{i+1}$

### 子移位 (Subshift)

闭不变子集 $Y \subset \Sigma_n$。

**有限型子移位 (SFT)**: 由有限个禁用词定义。

**Sofic移位**: SFT的因子系统。

### 邻接矩阵

对于由有向图定义的SFT，邻接矩阵 $A$ 给出：

- 周期点数：$\text{Fix}(\sigma^n) = \text{tr}(A^n)$
- 熵：$h = \log \lambda_{\max}(A)$

## 六、回归与回复

### Poincaré回归

**定理**: 保测变换下，几乎每个点都无穷多次回归任意正测度集。

**拓扑版本**: 回复点集在紧系统中稠密。

### 非游荡集 (Non-wandering Set)

**定义**: $\Omega(f) = \{x: \forall U \ni x \text{ 开}, \exists n > 0: f^n(U) \cap U \neq \emptyset\}$

**性质**: 闭不变集，包含所有长期动力学。

## 七、概念关联图

```

拓扑动力系统
├── 传递性
│   ├── 极小性（最强）
│   └── 混合性
├── 熵（复杂性度量）
│   ├── 零熵（简单）

```

## 八、重要例子

### 圆周旋转 $R_\alpha$

- $\alpha \in \mathbb{Q}$: 周期点稠密
- $\alpha \notin \mathbb{Q}$: 极小唯一遍历

### 倍角映射 $T(x) = 2x \pmod{1}$

- 混沌（敏感依赖、拓扑传递、周期点稠密）
- 熵 = $\log 2$
- 与移位 $\sigma$ 半共轭

### 符号移位 $\sigma$

- 混合性
- 周期点稠密
- 熵 = $\log n$

## 九、圆周旋转与符号动力学的深入分析

### 9.1 无理旋转的极小性证明

**定理**：圆周旋转 $R_\alpha(x) = x + \alpha \pmod 1$ 当 $\alpha \notin \mathbb{Q}$ 时是极小的。

**证明**：设 $U$ 为任意非空开弧。由Dirichlet逼近定理，存在无穷多对整数 $(p,q)$ 使 $|q\alpha - p| < 1/q$。故 $R_\alpha^q$ 近似平移 $p/q$，迭代可将任意点移入 $U$。形式化地，轨道 $\{R_\alpha^n(x)\}$ 是无限集且圆周紧，故必有聚点；由等距性，任意两点间距离可被任意小地逼近，从而轨道稠密。

### 9.2 全移位的拓扑混合性

**定理**：双侧全移位 $(\Sigma_n, \sigma)$ 是拓扑混合的。

**证明**：柱集 $C_{a_{-k}\dots a_m} = \{x : x_i = a_i \text{ for } -k \leq i \leq m\}$ 构成拓扑基。对任意两个柱集 $U = C_{u_{-k}\dots u_m}$ 和 $V = C_{v_{-k'}\dots v_{m'}}$，取 $N = m + k' + 1$。则对 $n \geq N$，$\sigma^n(U)$ 的第 $-k' \dots m'$ 位可由 $U$ 的尾部与 $V$ 的头部拼接得到，故 $\sigma^n(U) \cap V \neq \emptyset$。

### 9.3 拓扑熵的开覆盖定义

对紧度量空间 $X$ 和开覆盖 $\mathcal{U}$，定义：
$$\mathcal{U}^{(n)} = \bigvee_{i=0}^{n-1} f^{-i}\mathcal{U} = \{U_0 \cap f^{-1}(U_1) \cap \dots \cap f^{-1}(U_{n-1}) : U_i \in \mathcal{U}\}$$

令 $N(\mathcal{U})$ 为 $\mathcal{U}$ 的最小子覆盖基数。则：
$$h_{\text{top}}(f) = \sup_{\mathcal{U}} \lim_{n\to\infty} \frac{1}{n} \log N(\mathcal{U}^{(n)})$$

**与分离集定义的等价性**：开覆盖的直径趋于0时，$N(\mathcal{U}^{(n)})$ 与最大 $(n,\epsilon)$-分离集基数 $s_n(\epsilon)$ 的对数渐近等价。

### 9.4 Kronecker旋转与等分布

**Weyl等分布定理**：若 $\alpha \notin \mathbb{Q}$，则对任意Riemann可积函数 $f: S^1 \to \mathbb{R}$：
$$\lim_{N\to\infty} \frac{1}{N} \sum_{n=0}^{N-1} f(R_\alpha^n(x)) = \int_{S^1} f \, d\mu$$

这等价于 $R_\alpha$ 是**唯一遍历**的（Lebesgue测度是唯一不变测度），且时间平均等于空间平均。

## 十、共轭关系与拓扑熵的乘积公式

### 10.1 半共轭的典型例子

**倍角映射与移位映射**：定义 $T: S^1 \to S^1$, $T(x) = 2x \pmod 1$。用二进制展开 $x = 0.a_1 a_2 a_3 \dots$，则 $T(x) = 0.a_2 a_3 \dots$。

映射 $\pi: \Sigma_2^+ \to S^1$，$\pi(a_1 a_2 \dots) = \sum_{n=1}^\infty a_n 2^{-n}$ 是连续满射，且满足：
$$\pi \circ \sigma = T \circ \pi$$

因此 $T$ 与单侧移位 $\sigma$ **半共轭**。注意 $\pi$ 不是单射（如 $0.0111\dots = 0.1000\dots$），故不是共轭。

**意义**：半共轭将符号动力学的结构（周期点稠密、拓扑传递、敏感依赖）传递到倍角映射，证明后者是混沌的。

### 10.2 拓扑熵的乘积公式

对两个拓扑动力系统 $(X, f)$ 和 $(Y, g)$，其乘积系统为 $(X \times Y, f \times g)$。

**定理**：$h_{\text{top}}(f \times g) = h_{\text{top}}(f) + h_{\text{top}}(g)$。

**证明概要**：对分离集而言，$X \times Y$ 上的 $(n,\epsilon)$-分离集可取为 $X$ 上 $(n,\epsilon)$-分离集与 $Y$ 上 $(n,\epsilon)$-分离集的笛卡尔积，故基数相乘，取对数后熵相加。
