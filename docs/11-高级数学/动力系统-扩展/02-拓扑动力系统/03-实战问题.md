# 拓扑动力系统 - 实战问题

## 问题1: 符号移位分析

### 问题陈述

设 $\Sigma_2 = \{0, 1\}^{\mathbb{Z}}$ 是双侧移位空间，$\sigma$ 是移位映射。

(a) 证明 $\sigma$ 是拓扑混合的
(b) 计算周期为 $n$ 的周期点数
(c) 计算拓扑熵

### 解答

**(a) 拓扑混合性**

设 $U, V$ 是非空开集。由柱集构成基，存在柱集：

- $C_m(x) = \{y: y_{-m}...y_m = x_{-m}...x_m\} \subset U$
- $C_n(z) \subset V$

对 $k > m + n$，$\sigma^k(C_m(x))$ 的坐标 $[-m+k, m+k]$ 与 $C_n(z)$ 的坐标 $[-n, n]$ 可以重叠。

取 $k$ 足够大使 $m + k > n$，则：
$$\sigma^k(C_m(x)) \cap C_n(z) \neq \emptyset$$

因此 $\sigma^k(U) \cap V \neq \emptyset$ for all $k$ 足够大。 ✓

**(b) 周期点数**

$\sigma^n(x) = x$ 意味着 $x$ 是周期为 $n$ 的序列：
$$x = (...w|w|w...), \quad w \in \{0, 1\}^n$$

周期恰好为 $n$ 的点：$2^n - \sum_{d|n, d<n} \text{Fix}(\sigma^d)$

总周期点数（周期整除 $n$）：$|\text{Fix}(\sigma^n)| = 2^n$

**(c) 拓扑熵**

由分离集定义：$s_n(\epsilon) \sim 2^{n \cdot c(\epsilon)}$

或由周期点数：
$$h_{\text{top}}(\sigma) = \lim_{n \to \infty} \frac{\log |\text{Fix}(\sigma^n)|}{n} = \lim_{n \to \infty} \frac{n \log 2}{n} = \log 2$$

## 问题2: 圆周旋转的唯一遍历性

### 问题陈述

设 $R_\alpha(x) = x + \alpha \pmod{1}$，$\alpha \notin \mathbb{Q}$。

证明 $R_\alpha$ 是唯一遍历的，并确定唯一不变测度。

### 解答

**Step 1: 不变测度**

Lebesgue测度 $\lambda$ 显然不变：
$$\lambda(R_\alpha^{-1}(A)) = \lambda(A - \alpha) = \lambda(A)$$

**Step 2: 唯一性证明**

设 $\mu$ 是任意不变概率测度。证明 $\mu = \lambda$。

对连续函数 $f$，考虑Fourier系数：
$$\hat{\mu}(n) = \int e^{2\pi i n x} d\mu$$

不变性意味着：
$$\hat{\mu}(n) = \int e^{2\pi i n (x+\alpha)} d\mu = e^{2\pi i n \alpha} \hat{\mu}(n)$$

对 $n \neq 0$：$e^{2\pi i n \alpha} \neq 1$（无理旋转），故 $\hat{\mu}(n) = 0$。

$\hat{\mu}(0) = 1$（概率测度）。

因此 $\mu$ 有与Lebesgue测度相同的Fourier系数，$\mu = \lambda$。

**结论**: 唯一遍历，唯一不变测度是Lebesgue测度。

## 问题3: 倍角映射的熵

### 问题陈述

设 $T: [0, 1] \to [0, 1]$，$T(x) = 2x \pmod{1}$。

(a) 证明 $T$ 与单侧移位 $\sigma: \Sigma_2^+ \to \Sigma_2^+$ 半共轭
(b) 计算 $h_{\text{top}}(T)$
(c) 确定最大熵测度

### 解答

**(a) 半共轭**

定义二进制展开映射 $\pi: \Sigma_2^+ \to [0, 1]$：
$$\pi(x_0 x_1 x_2 ...) = \sum_{k=0}^\infty \frac{x_k}{2^{k+1}}$$

则：
$$T(\pi(x)) = 2\sum_{k=0}^\infty \frac{x_k}{2^{k+1}} \pmod{1} = \sum_{k=0}^\infty \frac{x_{k+1}}{2^{k+1}} = \pi(\sigma(x))$$

即 $\pi \circ \sigma = T \circ \pi$。

$\pi$ 是连续的满射，但在二进有理数处非单射（例如 $0.0111... = 0.1000...$）。

**(b) 拓扑熵**

由半共轭：$h_{\text{top}}(T) \leq h_{\text{top}}(\sigma) = \log 2$

反之，$T$ 有 $2^n$ 个 $n$-周期点，故：
$$h_{\text{top}}(T) \geq \lim_{n \to \infty} \frac{\log 2^n}{n} = \log 2$$

因此 $h_{\text{top}}(T) = \log 2$。

**(c) 最大熵测度**

Lebesgue测度 $\lambda$ 是最大熵测度：
$$h_\lambda(T) = \log 2 = h_{\text{top}}(T)$$

这可由 $\lambda = \pi_*(\mu_{1/2})$（$(1/2, 1/2)$ Bernoulli测度的像）验证。

## 问题4: SFT的熵计算

### 问题陈述

设 $A$ 是 $k \times k$ 的0-1矩阵，$\Sigma_A$ 是相应的有限型子移位。

(a) 证明周期点数 $|\text{Fix}(\sigma^n)| = \text{tr}(A^n)$
(b) 证明 $h_{\text{top}}(\sigma|_{\Sigma_A}) = \log \lambda_{\max}(A)$

### 解答

**(a) 周期点数**

周期为 $n$ 的点对应 $n$-环：
$$i_0 \xrightarrow{A_{i_0i_1}=1} i_1 \xrightarrow{} ... \xrightarrow{} i_{n-1} \xrightarrow{} i_0$$

这正是 $A^n$ 对角元：$(A^n)_{ii} = $ 从 $i$ 出发长为 $n$ 回到 $i$ 的路径数。

因此：$|\text{Fix}(\sigma^n)| = \sum_{i=1}^k (A^n)_{ii} = \text{tr}(A^n)$

**(b) 拓扑熵**

由Perron-Frobenius定理，$A$ 有最大特征值 $\lambda_{\max} > 0$ 对应正特征向量。

$\text{tr}(A^n) = \sum_{i=1}^k \lambda_i^n = \lambda_{\max}^n(1 + o(1))$

因此：
$$h_{\text{top}} = \lim_{n \to \infty} \frac{\log \text{tr}(A^n)}{n} = \log \lambda_{\max}$$

## 问题5: 极小性判定

### 问题陈述

设 $(X, T)$ 是紧度量空间上的动力系统。证明以下等价：
(a) $(X, T)$ 是极小的
(b) 每个轨道稠密
(c) 若 $A \subset X$ 闭且 $T(A) \subset A$，则 $A = \emptyset$ 或 $A = X$

### 解答

**(a) $\Rightarrow$ (b)**: 极小性定义

**(b) $\Rightarrow$ (c)**:

设 $A$ 是真闭不变子集，$x \in A$。则 $O(x) \subset A \neq X$，不稠密，矛盾。

**(c) $\Rightarrow$ (a)**:

设 $x \in X$，考虑 $\overline{O(x)}$。

- $\overline{O(x)}$ 是闭的
- $T(\overline{O(x)}) \subset \overline{O(x)}$（连续性）
- 非空（包含 $x$）

由 (c)，$\overline{O(x)} = X$。即轨道稠密。
