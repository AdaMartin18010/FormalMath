# 动力系统 (Dynamical Systems)

## 分支概述

动力系统是研究随时间演化的系统行为的数学分支，涵盖从经典力学到混沌理论的广泛领域。

## 核心主题

### [01-常微分方程定性理论](01-常微分方程定性理论/00-README.md)
- 相图与平衡点分析
- 极限环与Poincaré-Bendixson定理
- 稳定性理论（Lyapunov方法）
- 分支理论

### [02-拓扑动力系统](02-拓扑动力系统/00-README.md)
- 符号动力学与子移位
- 遍历理论与Birkhoff定理
- 熵与混合性
- 极小系统与唯一遍历性

### [03-光滑动力系统](03-光滑动力系统/00-README.md)
- 双曲理论与结构稳定性
- Smale马蹄与混沌
- Anosov系统与公理A系统
- 奇怪吸引子

### [04-哈密顿系统](04-哈密顿系统/00-README.md)
- 辛几何与Hamilton方程
- 可积系统
- KAM理论与近可积系统
- Arnold扩散

### [05-复动力系统](05-复动力系统/00-README.md)
- Julia集与Fatou集
- Mandelbrot集
- 双曲分量与重整化
- 多项式与有理映射动力学

### [06-无穷维动力系统](06-无穷维动力系统/00-README.md)
- 偏微分方程动力学
- 吸引子与惯性流形
- Navier-Stokes方程
- 反应扩散方程

## 与其他分支的联系

- **微分方程**: 演化方程的基础
- **拓扑学**: 定性分析的工具
- **概率论**: 随机动力系统、遍历理论
- **几何**: 辛几何、微分几何
- **数论**: 丢番图逼近与动力系统

## 应用实例

- 天体力学与轨道稳定性
- 流体力学中的湍流
- 生态系统的种群动力学
- 神经网络动力学
- 气候系统建模

## 推荐学习路径

1. **入门**: 从常微分方程定性理论开始
2. **基础**: 学习拓扑动力系统
3. **进阶**: 探索光滑动力系统和哈密顿系统
4. **专题**: 深入研究复动力系统或无穷维系统
