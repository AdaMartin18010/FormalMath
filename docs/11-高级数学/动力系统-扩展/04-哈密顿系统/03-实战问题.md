# 哈密顿系统 - 实战问题

## 问题1: 谐振子的作用-角变量

### 问题陈述

一维谐振子：$H = \frac{1}{2}(p^2 + q^2)$

(a) 构造作用-角变量
(b) 验证正则性
(c) 写出显式解

### 解答

**(a) 作用-角变量构造**

能量曲面：$H = E$，即 $p^2 + q^2 = 2E$

**作用变量**:
$$I = \frac{1}{2\pi}\oint p \, dq$$

参数化：$q = \sqrt{2E}\cos\theta$，$p = \sqrt{2E}\sin\theta$

$$I = \frac{1}{2\pi}\int_0^{2\pi} \sqrt{2E}\sin\theta \cdot (-\sqrt{2E}\sin\theta) d\theta = \frac{2E}{2\pi}\int_0^{2\pi} \sin^2\theta d\theta = E$$

所以 $I = H = E$。

**角变量**: $\theta = \arctan(p/q)$

**(b) 正则性验证**

计算泊松括号：
$$\{I, \theta\} = \frac{\partial I}{\partial q}\frac{\partial \theta}{\partial p} - \frac{\partial I}{\partial p}\frac{\partial \theta}{\partial q}$$

$I = \frac{1}{2}(p^2 + q^2)$，故：
- $\frac{\partial I}{\partial q} = q$，$\frac{\partial I}{\partial p} = p$

$\theta = \arctan(p/q)$，故：
- $\frac{\partial \theta}{\partial q} = -\frac{p}{p^2+q^2}$
- $\frac{\partial \theta}{\partial p} = \frac{q}{p^2+q^2}$

$$\{I, \theta\} = q \cdot \frac{q}{p^2+q^2} - p \cdot \left(-\frac{p}{p^2+q^2}\right) = \frac{q^2+p^2}{p^2+q^2} = 1$$

满足正则条件 $\{I, \theta\} = 1$。 ✓

**(c) 显式解**

$\dot{\theta} = \frac{\partial H}{\partial I} = 1$，$\dot{I} = 0$

$$\theta(t) = \theta_0 + t, \quad I(t) = I_0 = E$$

回到原变量：
$$q(t) = \sqrt{2E}\cos(t + \theta_0), \quad p(t) = \sqrt{2E}\sin(t + \theta_0)$$

## 问题2: 可积性判定

### 问题陈述

二维谐振子：$H = \frac{1}{2}(p_1^2 + p_2^2) + \frac{1}{2}(\omega_1^2 q_1^2 + \omega_2^2 q_2^2)$

(a) 证明系统可积
(b) 频率满足什么条件时运动周期？拟周期？

### 解答

**(a) 可积性**

取 $F_1 = H$，$F_2 = \frac{1}{2}(p_1^2 + \omega_1^2 q_1^2)$（第一个振子能量）。

验证对合：
$$\{F_1, F_2\} = \{H_1 + H_2, H_1\} = \{H_1, H_1\} + \{H_2, H_1\} = 0$$

（$H_1, H_2$ 独立变量，泊松括号为零）

$F_1, F_2$ 独立（一般点），故系统可积。

**(b) 周期性与拟周期性**

频率比 $\frac{\omega_1}{\omega_2}$：
- **有理数** $\frac{m}{n}$：周期运动（周期 $T = \frac{2\pi m}{\omega_1} = \frac{2\pi n}{\omega_2}$）
- **无理数**：拟周期运动，轨道在环面上稠密

## 问题3: KAM条件验证

### 问题陈述

考虑扰动系统：
$$H = \frac{1}{2}(I_1^2 + I_2^2) + \epsilon \sin(\theta_1 + \theta_2)$$

(a) 验证未扰动系统的非退化条件
(b) 哪些频率满足Diophantine条件？

### 解答

**(a) 非退化条件**

未扰动Hamiltonian：$H_0 = \frac{1}{2}(I_1^2 + I_2^2)$

Hessian矩阵：
$$\frac{\partial^2 H_0}{\partial I_i \partial I_j} = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}$$

$\det = 1 \neq 0$，非退化条件满足。 ✓

**(b) Diophantine条件**

频率：$\omega_1 = I_1$，$\omega_2 = I_2$

需要：$|k_1\omega_1 + k_2\omega_2| \geq \frac{\gamma}{(|k_1|+|k_2|)^\tau}$

等价于频率比的无理性：$\frac{\omega_1}{\omega_2} \notin \mathbb{Q}$ 且满足Diophantine逼近条件（非Liouville数）。

## 问题4: 开普勒问题的作用变量

### 问题陈述

开普勒问题：$H = \frac{|p|^2}{2} - \frac{1}{|q|}$

(a) 找出所有运动常数
(b) 计算作用变量
(c) 证明频率满足 $\omega_r = \omega_\theta = \omega_\phi$（简并性）

### 解答

**(a) 运动常数**

- 能量：$H = E$
- 角动量：$L = q \times p$（3个分量，但 $L \perp$ 轨道平面，2个独立）
- Laplace-Runge-Lenz向量：$A = p \times L - \frac{q}{|q|}$

共5个独立积分（$\dim M = 6$，$n = 3$），超可积。

**(b) 作用变量**

对椭圆轨道：
- $I_\phi = L_z$（$z$方向角动量）
- $I_\theta = |L|$（总角动量）
- $I_r = \frac{1}{2\pi}\oint p_r dr = -|L| + \sqrt{-\frac{1}{2E}}$

**(c) 简并性**

Hamiltonian：
$$H = -\frac{1}{2(I_r + I_\theta)^2}$$

频率：
$$\omega_r = \omega_\theta = \frac{1}{(I_r + I_\theta)^3}$$

这解释了闭合轨道的存在（Bertrand定理）。

## 问题5: 标准映射分析

### 问题陈述

Chirikov标准映射：
$$\begin{cases}
I_{n+1} = I_n + K\sin(\theta_n) \\
\theta_{n+1} = \theta_n + I_{n+1} \pmod{2\pi}
\end{cases}$$

(a) 证明它是保面积的
(b) 分析 $K \ll 1$ 时的KAM环面
(c) 估计 $K_c$（最后KAM环面被破坏的临界值）

### 解答

**(a) 保面积性**

计算Jacobi行列式：
$$\frac{\partial(I_{n+1}, \theta_{n+1})}{\partial(I_n, \theta_n)} = \det\begin{pmatrix} 1 & K\cos\theta_n \\ 1 & 1 + K\cos\theta_n \end{pmatrix} = 1$$

是辛映射。 ✓

**(b) KAM环面**

小 $K$ 时，映射可看作可积扭转映射的扰动：
$$I_{n+1} = I_n, \quad \theta_{n+1} = \theta_n + I_n$$

满足Diophantine条件的环面在 $K \ll 1$ 时保持。

**(c) 临界值估计**

数值实验给出：
$$K_c \approx 0.9716$$

Greene方法：通过分析周期轨道的稳定性来确定。
