# 复动力系统 - 实战问题

## 问题1: 二次多项式的Julia集

### 问题陈述

分析 $f_c(z) = z^2 + c$ 在以下参数的Julia集：

(a) $c = 0$（标准圆）
(b) $c = -1$（Basilica）
(c) $c = i$（兔子）

### 解答

**(a) $c = 0$**

$f_0(z) = z^2$

- Fatou集：$F = \{z: |z| < 1\} \cup \{z: |z| > 1\}$
- Julia集：$J = S^1 = \{z: |z| = 1\}$

- 动力学：$z = re^{i\theta} \mapsto r^2 e^{i2\theta}$

单位圆上的动力学是倍角映射，混沌。

**(b) $c = -1$（Basilica）**

$f_{-1}(z) = z^2 - 1$

- 临界点 $z = 0$ 的轨道：$0 \to -1 \to 0$（周期2）
- $z = 0$ 是超吸引周期点
- Julia集：连通的树状分形

周期轨道：$f^2(z) = z$ 的解

- $(z^2-1)^2 - 1 = z$
- $z^4 - 2z^2 - z = 0$
- $z(z+1)(z^2-z-1) = 0$

周期1：$z = 0$（超吸引），$z = -1$（超吸引）
周期2：$z^2 - z - 1 = 0$ 的解 $z = \frac{1 \pm \sqrt{5}}{2}$

**(c) $c = i$（兔子）**

$f_i(z) = z^2 + i$

- 临界点轨道：$0 \to i \to -1+i \to -i \to -1+i$（周期3）
- 超吸引周期3轨道
- Julia集：连通的"兔子"形状

## 问题2: Mandelbrot集的性质

### 问题陈述

(a) 证明若 $|c| > 2$，则 $c \notin \mathcal{M}$

(b) 求Main Cardioid的方程
(c) 找出周期2双曲分量的边界

### 解答

**(a) 逃逸判据**

若 $|z| > \max(2, |c|)$，则：
$$|z^2 + c| \geq |z|^2 - |c| > |z|^2 - |z| = |z|(|z|-1) > |z|$$

若 $|c| > 2$，则 $|f_c(0)| = |c| > 2$，且：
$$|f_c^n(0)| \to \infty$$

故 $c \notin \mathcal{M}$。 ✓

**(b) Main Cardioid**

$c$ 有吸引不动点当且仅当存在 $|\lambda| < 1$ 使得：

$$f_c(z) = z, \quad f_c'(z) = \lambda$$

即：

- $z^2 + c = z$
- $2z = \lambda$

解得：$c = z - z^2 = \frac{\lambda}{2} - \frac{\lambda^2}{4}$

令 $\lambda = e^{i\theta}$（边界 $|\lambda| = 1$）：

$$c = \frac{e^{i\theta}}{2} - \frac{e^{2i\theta}}{4}$$

这是心脏形曲线。

**(c) 周期2分量**

$f_c^2(z) = z$ 有4个解，其中2个是周期1点。

周期2点满足：
$$\frac{f_c^2(z) - z}{f_c(z) - z} = z^2 + z + c + 1 = 0$$

吸引条件：$|(f_c^2)'(z)| < 1$

$(f_c^2)'(z) = 4z^3 + 4cz = 4z(z^2+c) = 4z(f_c(z)) = 4z \cdot z = 4z^2$（在周期点上）

由 $z^2 = -z - c - 1$：
$$|(f_c^2)'| = |4(-z-c-1)| = 4|z+c+1|$$

周期2双曲分量是以 $c = -1$ 为中心的圆盘。

## 问题3: Siegel盘的存在性

### 问题陈述

考虑 $f(z) = \lambda z + z^2$，$\lambda = e^{2\pi i \theta}$。

(a) 什么条件下存在Siegel盘？
(b) 验证 $\theta = \frac{\sqrt{5}-1}{2}$（黄金比例）满足条件

### 解答

**(a) Siegel盘条件**

由Siegel-Yoccoz定理：

存在Siegel盘 当且仅当 $\theta$ 满足Diophantine条件

等价于Brjuno条件：
$$\sum_{n=0}^\infty \frac{\log q_{n+1}}{q_n} < \infty$$

其中 $p_n/q_n$ 是 $\theta$ 的连分数逼近。

**(b) 黄金比例**

$\phi = \frac{\sqrt{5}-1}{2} = [0; 1, 1, 1, ...]$（连分数全1）

逼近：$q_n = F_n$（Fibonacci数），$q_{n+1} \approx \phi^{-1} q_n$

Brjuno和：
$$\sum_{n} \frac{\log F_{n+1}}{F_n} \sim \sum_n \frac{n}{\phi^{-n}} < \infty$$

收敛！故 $\phi$ 满足Brjuno条件，存在Siegel盘。 ✓

## 问题4: 周期点计数

### 问题陈述

对 $f_c(z) = z^2 + c$，计算周期整除 $n$ 的周期点数。

### 解答

**迭代公式**: $f_c^n(z)$ 是 $2^n$ 次多项式

$f_c^n(z) = z$ 有 $2^n$ 个解（计入重数）。

**精确周期**：

记 $P_n$ 为周期整除 $n$ 的点数（不动点 of $f^n$）。

则周期恰好为 $n$ 的点数：
$$Q_n = P_n - \sum_{d|n, d<n} Q_d$$

或：
$$Q_n = \sum_{d|n} \mu(n/d) P_d$$

其中 $P_n = 2^n$（一般位置）。

**例子**:

- $n = 1$: $Q_1 = 2$（两个不动点）
- $n = 2$: $Q_2 = P_2 - P_1 = 4 - 2 = 2$
- $n = 3$: $Q_3 = P_3 - P_1 = 8 - 2 = 6$

## 问题5: 填充Julia集的容量

### 问题陈述

定义多项式 $f$ 的Green函数：
$$G(z) = \lim_{n \to \infty} \frac{1}{d^n} \log^+|f^n(z)|$$

(a) 证明 $G$ 在 $F(f)$ 上调和
(b) 计算 $f_c(z) = z^2 + c$ 的容量

### 解答

**(a) Green函数性质**

- $G(f(z)) = d \cdot G(z)$（函数方程）
- $G \geq 0$
- $G(z) = 0 \Leftrightarrow z \in K(f)$
- 在 $\mathbb{C} \setminus K(f)$ 上调和

调和性来自 $\log|z|$ 的调和性与一致收敛。

**(b) 容量计算**

填充Julia集 $K_c$ 的对数容量：
$$\text{cap}(K_c) = e^{-G_c(\infty)}$$

对 $f_c(z) = z^2 + c$：

在 $\infty$ 附近展开：
$$f_c^n(z) = z^{2^n} + O(z^{2^n-1})$$

Bottcher坐标：
$$\Phi_c(z) = \lim (f_c^n(z))^{1/2^n} = z + \frac{c}{2z} + ...$$

$$
$$\text{cap}(K_c) = 1$$

对所有 $c$ 成立！
