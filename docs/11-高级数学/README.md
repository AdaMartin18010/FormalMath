# 11-高级数学

本目录面向具备扎实基础的研究型学习者，系统整理代数几何、同伦论、数论几何、数学物理、导出代数几何、无穷范畴理论、高阶同伦论与代数 K 理论等前沿高级主题。文档总数近百份，采用专题综述与深度讲义相结合的形式，跟踪现代数学研究的核心议程与最新进展。每份文件不仅陈述高级定理，还力图阐明其历史动机、与其他分支的交叉关系，以及在形式化数学项目中的潜在应用，是通往研究前沿的重要知识枢纽。
