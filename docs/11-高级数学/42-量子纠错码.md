## 1. 引言：为什么需要量子纠错

### 1.1 量子退相干与噪声

**现实挑战**：
- **T1（能量弛豫时间）**：量子比特从 $|1\rangle$ 衰变到 $|0\rangle$ 的特征时间
- **T2（退相干时间）**：量子叠加态失去相位信息的特征时间
- **当前水平**：T1 ~ 100μs - 1ms（超导量子比特）

**错误率**：
- 单量子比特门：~0.1% - 0.01%
- 双量子比特门：~0.5% - 0.1%
- 测量：~1% - 0.1%

### 1.2 量子纠错的特殊性

**与经典纠错的区别**：

| 特性 | 经典纠错 | 量子纠错 |
|-----|---------|---------|
| 错误类型 | 比特翻转 | 比特翻转 + 相位翻转 |
| 测量问题 | 无限制 | 不可克隆定理，测量破坏叠加 |
| 连续错误 | 离散 | 连续（任意小旋转） |
| 资源需求 | 较小 | 较大（物理比特/逻辑比特比：10:1 到 1000:1） |

**不可克隆定理**：
$$\nexists \, U : U|\psi\rangle|0\rangle = |\psi\rangle|\psi\rangle$$

## 2. 量子错误模型

### 2.1 Pauli错误模型

**定义 2.1**（Pauli错误）：任何单量子比特错误可以表示为Pauli算符的线性组合：

$$\mathcal{E}(\rho) = \sum_{i=0}^{3} E_i \rho E_i^\dagger$$

其中 $E_i \in \{I, X, Y, Z\}$。

**错误类型**：
- **I**：无错误
- **X**：比特翻转（$|0\rangle \leftrightarrow |1\rangle$）
- **Z**：相位翻转（$|+\rangle \leftrightarrow |-\rangle$）
- **Y**：比特+相位翻转（$Y = iXZ$）

### 2.2 一般错误分解

**定理 2.1**（错误分解）：任意单量子比特幺正错误可以分解为：

$$U = e^{i\alpha} \left(\cos\frac{\theta}{2} I - i \sin\frac{\theta}{2} (\vec{n} \cdot \vec{\sigma})\right)$$

**离散化**：通过**量子纠错码**，连续错误被离散化为Pauli错误。

### 2.3 退相干模型

**定义 2.2**（退极化信道）：
$$\mathcal{E}_{\text{depol}}(\rho) = (1-p)\rho + \frac{p}{3}(X\rho X + Y\rho Y + Z\rho Z)$$

**定义 2.3**（幅值阻尼信道）：
$$\mathcal{E}_{\text{AD}}(\rho) = E_0 \rho E_0^\dagger + E_1 \rho E_1^\dagger$$

其中：
$$E_0 = \begin{pmatrix} 1 & 0 \\ 0 & \sqrt{1-\gamma} \end{pmatrix}, \quad
E_1 = \begin{pmatrix} 0 & \sqrt{\gamma} \\ 0 & 0 \end{pmatrix}$$

### 2.4 错误率与电路深度

**累积错误**：对于深度为 $D$、每门错误率为 $p$ 的电路：
$$P_{\text{success}} \approx (1-p)^D \approx e^{-pD}$$

**示例**：
- $p = 0.1\%$，$D = 100$：成功率 $\approx 90\%$
- $p = 0.1\%$，$D = 1000$：成功率 $\approx 37\%$
- $p = 0.1\%$，$D = 10000$：成功率 $\approx 0.0045\%$

## 3. 量子纠错的基本原理

### 3.1 三量子比特比特翻转码

**编码**：
$$|0\rangle_L = |000\rangle, \quad |1\rangle_L = |111\rangle$$

**叠加态**：
$$\alpha|0\rangle + \beta|1\rangle \to \alpha|000\rangle + \beta|111\rangle$$

**错误检测**：

| 综合征 | 测量 $Z_1 Z_2$ | 测量 $Z_2 Z_3$ | 错误位置 |
|-------|--------------|--------------|---------|
| 无 | +1 | +1 | 无 |
| $X_1$ | -1 | +1 | 第1位 |
| $X_2$ | -1 | -1 | 第2位 |
| $X_3$ | +1 | -1 | 第3位 |

**错误纠正**：在检测到错误的位置应用 $X$ 门。

### 3.2 三量子比特相位翻转码

**编码**：
$$|0\rangle_L = |+++\rangle, \quad |1\rangle_L = |---\rangle$$

其中 $|+\rangle = (|0\rangle + |1\rangle)/\sqrt{2}$，$|-\rangle = (|0\rangle - |1\rangle)/\sqrt{2}$。

** syndrome测量**：$X_1 X_2$ 和 $X_2 X_3$。

### 3.3 Shor码 [[9,1,3]]

**构造**：组合比特翻转码和相位翻转码

$$|0\rangle_L = \frac{1}{2\sqrt{2}}(|000\rangle + |111\rangle)^{\otimes 3}$$

$$|1\rangle_L = \frac{1}{2\sqrt{2}}(|000\rangle - |111\rangle)^{\otimes 3}$$

**参数**：
- $n = 9$：物理量子比特数
- $k = 1$：逻辑量子比特数
- $d = 3$：码距（可纠正1个任意错误）

## 4. Stabilizer形式化理论

### 4.1 Pauli群

**定义 4.1**（Pauli群）：$n$ 量子比特Pauli群 $G_n$：

$$G_n = \{i^k P_1 \otimes P_2 \otimes \cdots \otimes P_n : P_j \in \{I, X, Y, Z\}, k \in \{0,1,2,3\}\}$$

**性质**：
- $|G_n| = 4^{n+1}$
- 元素满足 $P^2 = \pm I$
- 对易关系：$[P, Q] = 0$ 或 $\{P, Q\} = 0$

### 4.2 Stabilizer群

**定义 4.2**（Stabilizer）：算子 $S$ **稳定**态 $|\psi\rangle$ 如果：
$$S|\psi\rangle = |\psi\rangle$$

**定义 4.3**（Stabilizer群）：一个Stabilizer群 $\mathcal{S}$ 是Pauli群的阿贝尔子群，且 $-I \notin \mathcal{S}$。

**定理 4.1**（Stabilizer空间的维度）：如果Stabilizer群有 $n-k$ 个独立生成元，则稳定空间的维度为 $2^k$。

### 4.3 Stabilizer码

**定义 4.4**（Stabilizer码）：由Stabilizer群 $\mathcal{S} = \langle g_1, g_2, \ldots, g_{n-k} \rangle$ 定义的量子码：

$$\mathcal{C} = \{ |\psi\rangle : g_i |\psi\rangle = |\psi\rangle, \forall i \}$$

**参数**：$[[n, k, d]]$ 码
- $n$：物理量子比特数
- $k$：逻辑量子比特数
- $d$：码距

### 4.4 syndrome测量

**定义 4.5**（syndrome）：对于错误 $E \in G_n$，syndrome是向量 $(s_1, s_2, \ldots, s_{n-k})$：

$$s_i = \begin{cases} 0 & \text{if } [E, g_i] = 0 \\ 1 & \text{if } \{E, g_i\} = 0 \end{cases}$$

**纠正条件**：如果 $E$ 和 $F$ 有相同的syndrome，则 $E^\dagger F \in N(\mathcal{S})$（正规化子）。

### 4.5 正规化子与逻辑算子

**定义 4.6**（正规化子）：
$$N(\mathcal{S}) = \{g \in G_n : ghg^{-1} \in \mathcal{S}, \forall h \in \mathcal{S}\}$$

**逻辑算子**：$N(\mathcal{S}) / \mathcal{S}$ 的元素对应于逻辑Pauli算子。

**定理 4.2**（码距）：
$$d = \min\{wt(E) : E \in N(\mathcal{S}) \setminus \mathcal{S}\}$$

## 5. 重要量子纠错码

### 5.1 Steane码 [[7,1,3]]

**构造**：基于经典Hamming码 $[7,4,3]$。

**Stabilizer生成元**：

$$g_1 = XXXXIII$$
$$g_2 = XXIIXXI$$
$$g_3 = XIXIXIX$$
$$g_4 = ZZZZIII$$
$$g_5 = ZZIIZZI$$
$$g_6 = ZIZIZIZ$$

**逻辑算子**：
$$\bar{X} = XXXXXXX, \quad \bar{Z} = ZZZZZZZ$$

**特点**：
- 最小的能纠正1个任意错误的CSS码
- 支持横向（transversal）门操作

### 5.2 CSS码

**定义 5.1**（CSS码）：Calderbank-Shor-Steane码由两个经典线性码 $C_1, C_2$ 构造：
- $C_2 \subset C_1$
- $C_1$：$[n, k_1, d_1]$ 码
- $C_2^\perp$：$[n, k_2, d_2]$ 码

**参数**：$[[n, k_1 - k_2, \min(d_1, d_2)]]$

**Stabilizer结构**：
- X型生成元：来自 $C_2^\perp$ 的校验矩阵
- Z型生成元：来自 $C_1$ 的校验矩阵

**优点**：X和Z错误可以独立纠正。

### 5.3 五量子比特码 [[5,1,3]]

**定理 5.1**（完美码）：$[[5,1,3]]$ 码是最小的能纠正1个任意错误的量子码。

**生成元**：
$$g_1 = XZZXI$$
$$g_2 = IXZZX$$
$$g_3 = XIXZZ$$
$$g_4 = ZXIXZ$$

**特点**：
- 非CSS码
- 完美码（达到量子Hamming界）

### 5.4 编码理论界限

**定理 5.2**（量子Hamming界）：对于 $[[n, k, 2t+1]]$ 码：
$$2^k \sum_{j=0}^{t} 3^j \binom{n}{j} \leq 2^n$$

**定理 5.3**（量子Singleton界）：
$$n - k \geq 2(d - 1)$$

满足等式的码称为**量子MDS码**。

**定理 5.4**（量子Gilbert-Varshamov界）：存在 $[[n, k, d]]$ 码如果：
$$2^{n-k} - 1 \geq \sum_{j=1}^{d-1} 3^j \binom{n}{j}$$

## 6. Surface Code拓扑纠错

### 6.1 表面码的格点结构

**定义 6.1**（表面码）：定义在二维方格上的Stabilizer码。

**布局**：
```
  ●───○───●───○───●
  │   │   │   │   │
  ○───●───○───●───○
  │   │   │   │   │
  ●───○───●───○───●
  │   │   │   │   │
  ○───●───○───●───○
  │   │   │   │   │
  ●───○───●───○───●
```

- **数据量子比特**（●）：存储逻辑信息
- **测量量子比特**（○）：测量Stabilizer

### 6.2 Stabilizer测量

**顶点算子（A-型）**：
$$A_v = \prod_{j \in \text{star}(v)} X_j$$

**面算子（B-型）**：
$$B_p = \prod_{j \in \text{boundary}(p)} Z_j$$

**测量**：
- 每个顶点测量 $A_v$
- 每个面测量 $B_p$

### 6.3 错误综合征

**X错误**：
- 在相邻顶点产生syndrome
- 形成字符串的两端

**Z错误**：
- 在相邻面产生syndrome
- 形成字符串的两端

**综合征图**：
```
  ●───○───●───○───●
  │   │   │   │   │
  ○───●───○───●───○
  │   │  ×│   │   │   ← Z错误产生的syndrome
  ●───○───●───○───●
  │   │   │   │   │
  ○───●───○───●───○
  │   │   │  ×│   │   ← Z错误产生的syndrome
  ●───○───●───○───●
```

### 6.4 解码算法

**最小权重完美匹配（MWPM）**：
1. 构建综合征节点之间的完全图
2. 边权重 = 曼哈顿距离
3. 找到最小权重的完美匹配
4. 沿匹配路径应用纠正操作

**复杂度**：$O(n^3)$ 使用Blossom算法。

### 6.5 逻辑算子与码距

**逻辑X算子**：
连接左右边界的任意路径上的 $X$ 算子乘积。

**逻辑Z算子**：
连接上下边界的任意路径上的 $Z$ 算子乘积。

**码距**：$d = L$（格点边长）

**参数**：$[[L^2, 1, L]]$

### 6.6 表面码的优势

| 特性 | 优势 |
|-----|------|
| **局部性** | 仅需要最近邻连接 |
| **高阈值** | ~1% 错误率阈值 |
| **容错门** | 支持Clifford门的横向实现 |
| **可扩展性** | 易于扩展到大规模系统 |

## 7. 容错量子计算与阈值定理

### 7.1 容错操作的要求

**定义 7.1**（容错）：一个操作是容错的，如果单点故障最多导致一个逻辑错误。

**要求**：
1. 错误传播受控
2. syndrome测量可靠
3. 纠正操作正确

### 7.2 横向门

**定义 7.2**（横向门）：门操作在每个物理量子比特上独立进行，不引入物理量子比特之间的交互。

**定理 7.1**（Eastin-Knill定理）：不存在能够横向实现所有逻辑门的量子纠错码。

**Clifford门的横向实现**：
- Hadamard：横向H门
- Phase：横向S门
- CNOT：横向CNOT

**非Clifford门（T门）**：
- 需要魔法态蒸馏（magic state distillation）
- 资源开销大

### 7.3 阈值定理

**定理 7.2**（阈值定理）：如果物理错误率 $p$ 低于某个阈值 $p_{\text{th}}$，则对于任意计算，存在容错方案使得逻辑错误率任意小。

$$p_L \sim \left(\frac{p}{p_{\text{th}}}\right)^{(d+1)/2}$$

**不同码的阈值**：

| 码 | 阈值估计 |
|---|---------|
| Steane码 | ~0.1% |
| Surface code | ~0.5% - 1% |
| 颜色码 | ~0.1% |
| LDPC码 | ~0.1% - 0.5% |

### 7.4 逻辑错误率分析

**表面码的缩放**：
$$p_L \approx C \left(\frac{p}{p_{\text{th}}}\right)^{(L+1)/2}$$

其中 $L$ 是码距。

**资源估计**：
- 要达到 $p_L = 10^{-15}$（Shor算法要求）
- 假设 $p = 0.1\%$，$p_{\text{th}} = 1\%$
- 需要 $L \approx 21$（441物理量子比特/逻辑量子比特）

### 7.5 魔法态蒸馏

**目标**：制备高保真度的 $|A\rangle = |0\rangle + e^{i\pi/4}|1\rangle$ 态。

**蒸馏方案**：
$$|A\rangle^{\otimes n} \to |A\rangle_{\text{pure}}$$

**资源开销**：
- 制备1个高保真T门需要 $O(\log^\gamma(1/\epsilon))$ 个原始魔法态
- 通常 $\gamma \approx 2$

"""
量子纠错码 - Python实现
Stabilizer码、Surface Code、错误纠正

import numpy as np
from itertools import product
import matplotlib.pyplot as plt
from matplotlib.patches import Circle, FancyBboxPatch
from typing import List, Tuple, Dict, Set


class StabilizerCode:
    """
    通用Stabilizer码模拟器
    支持任意[[n,k,d]] Stabilizer码
    """
    
    def __init__(self, n: int, k: int, stabilizers: List[str]):
        """
        初始化Stabilizer码
        
        参数:
            n: 物理量子比特数
            k: 逻辑量子比特数
            stabilizers: Stabilizer生成元列表，格式如 "XXIZ"
                X: Pauli-X, Z: Pauli-Z, Y: Pauli-Y, I: Identity
        """
        self.n = n
        self.k = k
        self.stabilizers = stabilizers
        self.r = len(stabilizers)  # syndrome比特数
        
        print(f"初始化Stabilizer码 [[{n},{k},{self.r}]]")
        print(f"物理比特数: {n}")
        print(f"逻辑比特数: {k}")
        print(f"Stabilizer生成元:")
        for i, stab in enumerate(stabilizers):
            print(f"   g_{i+1} = {stab}")
    
    def pauli_to_matrix(self, pauli_str: str) -> np.ndarray:
        """
        将Pauli字符串转换为矩阵表示
        用于小规模的验证计算
        """
        I = np.eye(2, dtype=complex)
        X = np.array([[0, 1], [1, 0]], dtype=complex)
        Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
        Z = np.array([[1, 0], [0, -1]], dtype=complex)
        
        matrices = {'I': I, 'X': X, 'Y': Y, 'Z': Z}
        
        result = np.array([[1]], dtype=complex)
        for p in pauli_str:
            result = np.kron(result, matrices[p])
        
        return result
    
    def compute_syndrome(self, error: str) -> Tuple[int, ...]:
        """
        计算给定错误的syndrome
        
        返回:
            syndrome元组，每个元素为0（对易）或1（反对易）
        """
        syndrome = []
        
        for stab in self.stabilizers:
            # 计算对易子
            commutes = True
            for i, (e, s) in enumerate(zip(error, stab)):
                if e != 'I' and s != 'I' and e != s:
                    # X和Z（或Y）反对易
                    if set([e, s]) == {'X', 'Z'} or 'Y' in [e, s]:
                        commutes = not commutes
            
            syndrome.append(0 if commutes else 1)
        
        return tuple(syndrome)
    
    def get_error_table(self) -> Dict[Tuple[int, ...], List[str]]:
        """
        生成完整的syndrome查找表
        映射 syndrome -> 可能的错误
        """
        error_table = {}
        
        # 生成所有单Pauli错误
        paulis = ['I', 'X', 'Y', 'Z']
        
        for error in product(paulis, repeat=self.n):
            error_str = ''.join(error)
            syndrome = self.compute_syndrome(error_str)
            
            if syndrome not in error_table:
                error_table[syndrome] = []
            error_table[syndrome].append(error_str)
        
        return error_table

class SurfaceCode:
    """
    表面码模拟器
    支持错误综合征生成和最小权重匹配解码
    """
    
    def __init__(self, L: int):
        """
        初始化L×L表面码
        
        参数:
            L: 码距（格点边长）
        """
        self.L = L
        self.d = L
        self.n_data = L * L
        self.n_ancilla_x = (L-1) * L  # X-syndrome测量
        self.n_ancilla_z = L * (L-1)  # Z-syndrome测量
        
        print(f"\n初始化表面码 (d={L})")
        print(f"   数据量子比特: {self.n_data}")
        print(f"   X-syndrome测量: {self.n_ancilla_x}")
        print(f"   Z-syndrome测量: {self.n_ancilla_z}")
        print(f"   总物理资源: {self.n_data + self.n_ancilla_x + self.n_ancilla_z}")
        print(f"   编码率: 1/{self.n_data}")
    
    def create_lattice(self) -> np.ndarray:
        """
        创建格点可视化矩阵
        """
        # 创建2L+1 × 2L+1的格点
        lattice = np.zeros((2*self.L+1, 2*self.L+1))
        
        # 数据量子比特位置
        for i in range(self.L):
            for j in range(self.L):
                lattice[2*i+1, 2*j+1] = 1  # 数据比特
        
        # X-syndrome测量位置
        for i in range(self.L-1):
            for j in range(self.L):
                lattice[2*i+2, 2*j+1] = 2
        
        # Z-syndrome测量位置
        for i in range(self.L):
            for j in range(self.L-1):
                lattice[2*i+1, 2*j+2] = 3
        
        return lattice
    
    def visualize_lattice(self, x_errors: Set[Tuple[int, int]] = None,
                          z_errors: Set[Tuple[int, int]] = None,
                          x_syndrome: Set[Tuple[int, int]] = None,
                          z_syndrome: Set[Tuple[int, int]] = None):
        """
        可视化表面码格点
        """
        fig, ax = plt.subplots(figsize=(10, 10))
        
        # 绘制格点
        lattice = self.create_lattice()
        
        for i in range(2*self.L+1):
            for j in range(2*self.L+1):
                val = lattice[i, j]
                
                if val == 1:  # 数据量子比特
                    color = 'blue'
                    if x_errors and (i//2, j//2) in x_errors:
                        color = 'red'  # X错误
                    if z_errors and (i//2, j//2) in z_errors:
                        color = 'green'  # Z错误

    def simulate_errors(self, p_x: float, p_z: float) -> Tuple[Set, Set]:
        """
        模拟随机错误
        
        参数:
            p_x: X错误概率
            p_z: Z错误概率
        """
        x_errors = set()
        z_errors = set()
        
        for i in range(self.L):
            for j in range(self.L):
                if np.random.random() < p_x:
                    x_errors.add((i, j))
                if np.random.random() < p_z:
                    z_errors.add((i, j))
        
        return x_errors, z_errors
    
    def measure_syndrome(self, x_errors: Set, z_errors: Set) -> Tuple[Set, Set]:
        """
        测量syndrome
        """
        x_syndrome = set()
        z_syndrome = set()
        
        # X-syndrome（检测Z错误）
        for i in range(self.L-1):
            for j in range(self.L):
                # 顶点(i+0.5, j)处的X-syndrome
                # 检测相邻的Z错误
                count = 0
                neighbors = [(i, j), (i+1, j)]
                for ni, nj in neighbors:
                    if (ni, nj) in z_errors:
                        count += 1
                if count % 2 == 1:
                    x_syndrome.add((2*i+2, 2*j+1))
        
        # Z-syndrome（检测X错误）
        for i in range(self.L):
            for j in range(self.L-1):
                # 面(i, j+0.5)处的Z-syndrome
                count = 0
                neighbors = [(i, j), (i, j+1)]
                for ni, nj in neighbors:
                    if (ni, nj) in x_errors:
                        count += 1
                if count % 2 == 1:
                    z_syndrome.add((2*i+1, 2*j+2))
        
        return x_syndrome, z_syndrome

class MWPMDecoder:
    """
    最小权重完美匹配解码器
    简化版实现用于教学演示
    """
    
    def __init__(self, L: int):
        self.L = L
    
    def decode(self, syndrome: Set[Tuple[int, int]], error_type: str = 'X') -> Set[Tuple[int, int]]:
        """
        使用MWPM解码syndrome
        
        简化实现：返回连接syndrome点的最短路径
        """
        if len(syndrome) <= 1:
            return set()
        
        # 简化的解码：配对最近的syndrome
        syndrome_list = list(syndrome)
        correction = set()
        
        # 贪心配对（实际应使用Blossom算法）
        paired = set()
        for i, s1 in enumerate(syndrome_list):
            if i in paired:
                continue
            
            min_dist = float('inf')
            min_j = -1
            
            for j, s2 in enumerate(syndrome_list[i+1:], start=i+1):
                if j in paired:
                    continue
                dist = abs(s1[0] - s2[0]) + abs(s1[1] - s2[1])
                if dist < min_dist:
                    min_dist = dist
                    min_j = j
            
            if min_j != -1:
                paired.add(i)
                paired.add(min_j)
                
                # 添加路径上的纠正
                path = self._get_path(syndrome_list[i], syndrome_list[min_j])
                correction.update(path)
        
        return correction
    
    def _get_path(self, p1: Tuple[int, int], p2: Tuple[int, int]) -> Set[Tuple[int, int]]:
        """获取两点之间的路径"""
        path = set()
        
        # 简化：曼哈顿路径
        x1, y1 = p1
        x2, y2 = p2
        
        # 水平移动
        step = 1 if x2 > x1 else -1
        for x in range(x1, x2, step):
            path.add((x, y1))
        
        # 垂直移动
        step = 1 if y2 > y1 else -1
        for y in range(y1, y2, step):
            path.add((x2, y))
        
        return path
