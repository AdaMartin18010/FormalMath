# 群论习题与反例补充

群论是代数学的基石，研究对称性的代数结构。本节通过精选习题与反例，深入探讨 Lagrange 定理的逆问题、正规子群、Sylow 定理与可解性的经典技巧。

## 1. 基础计算

### 1.1 对称群的运算

**习题 1.1**. 在 $S_3$ 中计算：

1. $(12) \circ (23)$；
2. $(123) \circ (12)$；
3. $(123)^{-1}$。

**解答**.

1. $(12) \circ (23)$：$1 \mapsto 1 \mapsto 2$，$2 \mapsto 3 \mapsto 3$，$3 \mapsto 2 \mapsto 1$，故为 $(123)$。
2. $(123) \circ (12)$：$1 \mapsto 2 \mapsto 3$，$2 \mapsto 1 \mapsto 2$，$3 \mapsto 3 \mapsto 1$，故为 $(13)$。
3. $(123)^{-1} = (132)$（逆序轮换）。$\square$

### 1.2 循环群的子群

**习题 1.2**. 找出 $\mathbb{Z}_{12}$ 的所有子群。

**解答**. 循环群的子群也是循环群。对 $n = 12$ 的每个正因子 $d$，存在唯一的 $d$ 阶子群 $\langle 12/d \rangle$：

- $d = 1$：$\langle 0 \rangle = \{0\}$；
- $d = 2$：$\langle 6 \rangle = \{0, 6\}$；
- $d = 3$：$\langle 4 \rangle = \{0, 4, 8\}$；
- $d = 4$：$\langle 3 \rangle = \{0, 3, 6, 9\}$；
- $d = 6$：$\langle 2 \rangle = \{0, 2, 4, 6, 8, 10\}$；
- $d = 12$：$\langle 1 \rangle = \mathbb{Z}_{12}$。

共 6 个子群。$\square$

## 2. 重要定理的证明

### 2.1 偶数阶群含二阶元

**习题 2.1**. 证明偶数阶群必含二阶元（$a \neq e$ 且 $a^2 = e$）。

**证明**. 元素按 $a \sim a^{-1}$ 配对。$e$ 与自身配对。若所有非单位元都与不同元素配对，则非单位元个数为偶数，总阶数为奇数，矛盾。故存在 $a \neq e$ 使 $a = a^{-1}$，即 $a^2 = e$。$\square$

### 2.2 $p^2$ 阶群为 Abel

**习题 2.2**. 若 $|G| = p^2$（$p$ 为素数），则 $G$ 为 Abel 群。

**证明**. 由类方程：

$$|G| = |Z(G)| + \sum_{i} [G : C_G(x_i)],$$

其中 $x_i$ 为非中心共轭类代表。共轭类大小整除 $|G| = p^2$，故为 $1, p$ 或 $p^2$。非中心共轭类大小 $\geq p$，而 $|Z(G)| \geq 1$。

若 $|Z(G)| = p$，则 $G/Z(G)$ 的阶为 $p$，故循环。取生成元 $gZ(G)$，则任意 $a, b \in G$ 可写为 $g^i z_1, g^j z_2$（$z_i \in Z(G)$），显然可交换。于是 $G$ 为 Abel，即 $Z(G) = G$，矛盾。

故 $|Z(G)| = p^2$，$G = Z(G)$ 为 Abel。$\square$

### 2.3 Sylow 定理的应用

**习题 2.3**. 证明 15 阶群必为循环群。

**证明**. $|G| = 15 = 3 \cdot 5$。由 Sylow 第三定理，$n_3 \equiv 1 \pmod{3}$ 且 $n_3 | 5$，故 $n_3 = 1$。同理 $n_5 = 1$。

设 $P_3$ 为唯一 Sylow 3-子群，$P_5$ 为唯一 Sylow 5-子群。二者均正规，$P_3 \cap P_5 = \{e\}$，$|P_3 P_5| = 15 = |G|$。故 $G = P_3 \times P_5 \cong \mathbb{Z}_3 \times \mathbb{Z}_5 \cong \mathbb{Z}_{15}$（因 $\gcd(3,5) = 1$）。$\square$

## 3. 经典反例

### 3.1 非正规子群

**反例 3.1**. 在 $S_3$ 中，$H = \{e, (12)\}$ 不是正规子群。

**验证**. $(123)(12)(123)^{-1} = (123)(12)(132) = (23) \notin H$。故 $H \not\trianglelefteq S_3$。$\square$

### 3.2 Lagrange 定理之逆不成立

**反例 3.2**. $A_4$（12 阶交错群）没有 6 阶子群。

**验证**. 假设 $H \leq A_4$，$|H| = 6$。则 $[A_4 : H] = 2$，故 $H \trianglelefteq A_4$。$A_4/H \cong \mathbb{Z}_2$。对任意 $g \in A_4$，$g^2 \in H$。

$A_4$ 中有 8 个 3-轮换，每个满足 $g^3 = e$，$g \neq e$，故 $g = g^{-2} \in H$。于是 $H$ 含所有 8 个 3-轮换，$|H| \geq 9$，矛盾。$\square$

### 3.3 同态不保持阶

**反例 3.3**. $\varphi: \mathbb{Z} \to \mathbb{Z}_2$，$\varphi(n) = n \pmod{2}$。$1 \in \mathbb{Z}$ 的阶为 $\infty$，但 $\varphi(1)$ 的阶为 $2$。

## 4. 例子

### 4.1 例子：二面体群 $D_4$

**习题 4.1**. 计算 $D_4$（正方形对称群）中每个元素的阶。

**解答**. $D_4 = \langle r, s \mid r^4 = s^2 = e, srs = r^{-1} \rangle$，8 个元素：$e, r, r^2, r^3, s, sr, sr^2, sr^3$。

- $o(e) = 1$；
- $o(r) = o(r^3) = 4$；
- $o(r^2) = 2$；
- $o(s) = o(sr) = o(sr^2) = o(sr^3) = 2$（所有反射）。

### 4.2 例子：$GL_2(\mathbb{R})$ 中的元素阶

**习题 4.2**. 计算 $A = \begin{pmatrix} 0 & -1 \\ 1 & 0 \end{pmatrix} \in GL_2(\mathbb{R})$ 的阶。

**解答**. $A^2 = \begin{pmatrix} -1 & 0 \\ 0 & -1 \end{pmatrix} = -I$，$A^4 = I$。$A^k \neq I$ 对 $k = 1, 2, 3$，故 $o(A) = 4$。几何上，$A$ 为逆时针旋转 $90^\circ$。$\square$
