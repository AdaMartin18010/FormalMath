## 13.8.1 引言

模空间理论是现代代数几何的核心领域之一，研究代数几何对象的参数化空间。曲线模空间 $M_g$ 参数化亏格 $g$ 的光滑代数曲线，是代数几何、数论和弦理论交汇的重要对象。Deligne和Mumford在1969年证明 $M_g$ 是不可约的拟射影簇，并构造了其紧化 $\overline{M}_g$，开创了现代模空间理论。

**Moduli space theory is one of the central areas of modern algebraic geometry, studying parameter spaces of algebraic geometric objects. The moduli space $M_g$ of curves parameterizes smooth algebraic curves of genus $g$, serving as an important object where algebraic geometry, number theory, and string theory intersect.**

## 13.8.2 模问题的函子描述

### 模函子的定义

**定义 13.8.1** (模函子 / Moduli Functor)

设 $\mathcal{M}$ 是概形范畴到集合范畴的函子。对概形 $S$，$\mathcal{M}(S)$ 是 $S$ 上某种几何对象的同构类集合，对态射 $f: S' \to S$，$\mathcal{M}(f)$ 是拉回映射。

**定义 13.8.2** (精细模空间 / Fine Moduli Space)

概形 $M$ 是函子 $\mathcal{M}$ 的精细模空间，如果存在泛族（universal family）$\mathcal{U} \to M$ 使得：
$$\text{Hom}(S, M) \cong \mathcal{M}(S)$$
对任意 $S$，通过拉回 $\mathcal{U}$ 实现。

**定义 13.8.3** (粗糙模空间 / Coarse Moduli Space)

概形 $M$ 是函子 $\mathcal{M}$ 的粗糙模空间，如果存在自然变换 $\Phi: \mathcal{M} \to h_M$ 使得：
- $\Phi(k): \mathcal{M}(k) \to M(k)$ 是双射（对代数闭域 $k$）
- $M$ 满足万有性质：对任意 $M'$ 和自然变换 $\Psi: \mathcal{M} \to h_{M'}$，存在唯一态射 $f: M \to M'$ 使得 $\Psi = h_f \circ \Phi$

## 13.8.3 光滑曲线的模空间 $M_g$

### $M_g$ 的定义

**定义 13.8.4** (曲线模空间 $M_g$)

亏格 $g$ 的光滑曲线的模空间 $M_g$ 是以下模函子的粗糙模空间：
$$\mathcal{M}_g(S) = \{\text{$S$ 上光滑亏格 $g$ 曲线的同构类}\}$$

### 维数与存在性

**定理 13.8.1** (Riemann计算曲线模空间的维数)

对 $g \geq 2$，$\dim M_g = 3g - 3$。

对 $g = 0$，$M_0 = \text{Spec}(k)$（只有一个点，代表 $\mathbb{P}^1$）。

对 $g = 1$，$\dim M_1 = 1$（$j$-线）。

**证明概要**: 曲线 $C$ 的形变由 $H^1(C, T_C)$ 控制。由Serre对偶：
$$h^1(T_C) = h^0(K_C^{\otimes 2}) = 3g - 3$$
（对 $g \geq 2$，由Riemann-Roch计算）。∎

## 13.8.4 稳定曲线与Deligne-Mumford紧化

### 稳定曲线的定义

**定义 13.8.5** (稳定曲线 / Stable Curve)

亏格 $g$ 的稳定曲线是满足以下条件的连通射影曲线 $C$：
- $C$ 只有节点（nodes）作为奇点
- 每条有理成分（同构于 $\mathbb{P}^1$）至少与曲线的其他部分在3点相交
- 数曲线的自同构群是有限的

等价地，若 $C$ 的正规化 $\tilde{C}$ 的亏格为 $g$，且对 $\tilde{C}$ 的每个连通分支 $C_i$，有：
$$2g(C_i) - 2 + \#\{\text{特殊点 on } C_i\} > 0$$

### Deligne-Mumford紧化

**定理 13.8.2** (Deligne-Mumford紧化)

存在射影模空间 $\overline{M}_g$ 是 $\mathcal{M}_g$ 的Deligne-Mumford紧化，即：
- $\overline{M}_g$ 是射影概形
- $M_g \subseteq \overline{M}_g$ 是稠密开子概形
- $\overline{M}_g \setminus M_g$ 参数化稳定曲线（带有节点的曲线）
- 边界 $\Delta = \overline{M}_g \setminus M_g$ 是正规交叉除子

边界分支分解为：
$$\Delta = \bigcup_{i=0}^{[g/2]} \Delta_i$$
其中 $\Delta_0$ 参数化不可约节点曲线，$\Delta_i$（$i > 0$）参数化具有两个分别亏格 $i$ 和 $g-i$ 的成分的曲线。

## 13.8.5 详细示例：椭圆曲线模空间 $M_1$ 的结构

### 复椭圆曲线的参数化

**示例 13.8.1** ($M_1$ 的结构)

复椭圆曲线同构于复环面 $\mathbb{C}/\Lambda$，其中 $\Lambda = \mathbb{Z} + \tau\mathbb{Z}$，$\tau \in \mathbb{H}$（上半平面）。

两个格 $\Lambda_\tau$ 和 $\Lambda_{\tau'}$ 给出同构的椭圆曲线当且仅当存在 $\gamma \in \text{SL}(2, \mathbb{Z})$ 使得：
$$\tau' = \frac{a\tau + b}{c\tau + d}, \quad \gamma = \begin{pmatrix} a & b \\ c & d \end{pmatrix}$$

**定理 13.8.3** ($M_1$ 的结构)

$$M_1 \cong \mathbb{H}/\text{SL}(2, \mathbb{Z}) \cong \mathbb{C}$$

### $j$-不变量

**定义 13.8.6** ($j$-不变量)

椭圆曲线 $E: y^2 = x^3 + ax + b$ 的 $j$-不变量定义为：
$$j(E) = 1728 \frac{4a^3}{4a^3 + 27b^2}$$

**性质**:
- $j(E) = j(E')$ 当且仅当 $E \cong E'$（代数闭域上）
- $j: M_1 \to \mathbb{A}^1$ 是同构

### 详细计算：Legendre形式

任何特征不为2的域上的椭圆曲线都可写成Legendre形式：
$$E_\lambda: y^2 = x(x-1)(x-\lambda), \quad \lambda \neq 0, 1$$

$j$-不变量计算：
$$j(E_\lambda) = 256 \frac{(1-\lambda+\lambda^2)^3}{\lambda^2(1-\lambda)^2}$$

**证明**: 将方程写成Weierstrass形式 $y^2 = x^3 + ax + b$：

展开：$y^2 = x^3 - (1+\lambda)x^2 + \lambda x$

令 $x' = x - \frac{1+\lambda}{3}$ 消去二次项，得：
$$a = -\frac{(1-\lambda+\lambda^2)}{3}, \quad b = -\frac{(1+\lambda)(2\lambda^2-5\lambda+2)}{27}$$

判别式：$\Delta = -16(4a^3 + 27b^2) = 16\lambda^2(1-\lambda)^2$

代入 $j$-不变量公式即得。∎

### $M_1$ 的紧化

$\overline{M}_1$ 是通过添加尖点（cusp）$j = \infty$ 得到的：
$$\overline{M}_1 \cong \mathbb{P}^1$$

边界点 $\Delta_0$ 对应于 nodal cubic（带一个节点的三次曲线）。

## 13.8.6 模空间的整体性质

### 一般性质

**定理 13.8.4** ($M_g$ 和 $\overline{M}_g$ 的性质)

对 $g \geq 3$：
- $M_g$ 是不可约拟射影簇
- $\overline{M}_g$ 是射影的、不可约的、正规簇
- $\overline{M}_g$ 是 Deligne-Mumford 栈的粗模空间
- $\text{Pic}(M_g) \otimes \mathbb{Q} \cong \mathbb{Q}\lambda$，其中 $\lambda$ 是Hodge类

### 大亏格行为

**定理 13.8.5** (Harris-Mumford, Eisenbud-Harris)

对 $g \geq 24$，$M_g$ 是一般型的（general type）。

对 $g \leq 15$，$M_g$ 是单有理的（unirational）。

## 13.8.7 形式化实现

```lean
-- 曲线模空间的形式化框架 (Lean 4风格)

-- 光滑曲线的定义
structure SmoothCurve (k : Type*) [Field k] (g : ℕ) where
  scheme : Scheme k
  smooth : IsSmooth scheme
  proj : IsProjective scheme
  genus : ArithmeticGenus scheme = g

-- 稳定曲线的定义  
structure StableCurve (k : Type*) [Field k] (g : ℕ) where
  curve : Scheme k
  connected : IsConnected curve
  proj : IsProjective curve
  onlyNodes : HasOnlyNodes curve  -- 只有节点奇点
  stability : StabilityCondition curve  -- 稳定性条件
  genus : ArithmeticGenus curve = g

-- 模函子的定义
def ModuliFunctorCurves (g : ℕ) : Functor Schemeᵒᵖ Type where
  obj S := {C : Scheme S // SmoothGenusGCurve C g} / Isomorphism
  map f := fun C => PullbackCurve f C

-- $M_g$ 存在性公理（或定理）
axiom M_g_exists (g : ℕ) : ∃ M : Scheme k, 
  IsCoarseModuliSpace M (ModuliFunctorCurves g)

-- Deligne-Mumford紧化
structure DeligneMumfordCompactification (g : ℕ) where
  M_bar : Scheme k
  M_open : Subscheme M_bar
  isCompact : IsProjective M_bar
  boundary : Divisor M_bar
  boundaryParam : StableCurves (boundary.support)
```
