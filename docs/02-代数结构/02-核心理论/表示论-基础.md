# 表示论基础

表示论（representation theory）通过将抽象代数结构（群、代数、Lie 代数）具体化为线性变换来研究其结构。对有限群而言，复表示论在二十世纪初由 Frobenius、Burnside、Schur 奠基，成为现代数学与物理的通用语言。

## 1. 群表示的基本定义

### 1.1 表示与同构

**定义 1.1（表示）**. 设 $G$ 为群，$V$ 为域 $\mathbb{F}$ 上的向量空间。$G$ 在 $V$ 上的表示（representation）是群同态：

$$\rho: G \to GL(V),$$

即对任意 $g, h \in G$，$\rho(gh) = \rho(g)\rho(h)$ 且 $\rho(e) = I_V$。称 $V$ 为表示空间，$\dim V$ 为表示的维数。

**定义 1.2（等价表示）**. 两个表示 $(\rho_1, V_1)$ 与 $(\rho_2, V_2)$ 称为等价，若存在线性同构 $T: V_1 \to V_2$ 使得对所有 $g \in G$：

$$T \circ \rho_1(g) = \rho_2(g) \circ T.$$

### 1.2 子表示与不可约性

**定义 1.3（不变子空间）**. 子空间 $W \subset V$ 称为不变的（invariant），若 $\rho(g)(W) \subset W$ 对所有 $g \in G$ 成立。

**定义 1.4（不可约表示）**. 非零表示 $(\rho, V)$ 称为不可约（irreducible），若其不变子空间只有 $\{0\}$ 和 $V$。

**定理 1.5（Maschke）**. 设 $G$ 为有限群，$\operatorname{char}(\mathbb{F}) \nmid |G|$。则 $G$ 的每个有限维表示都可完全分解为不可约表示的直和。

*证明*. 设 $W \subset V$ 为不变子空间。取任意投影 $P: V \to W$，定义平均化算子：

$$\tilde{P} = \frac{1}{|G|} \sum_{g \in G} \rho(g) P \rho(g)^{-1}.$$

对 $w \in W$，$\rho(g)^{-1}w \in W$，故 $P\rho(g)^{-1}w = \rho(g)^{-1}w$，从而 $\tilde{P}w = w$。又对任意 $v \in V$，$h \in G$：

$$\rho(h)\tilde{P}v = \frac{1}{|G|} \sum_{g} \rho(hg) P \rho(g)^{-1} v = \frac{1}{|G|} \sum_{g'} \rho(g') P \rho(g')^{-1} \rho(h) v = \tilde{P}\rho(h)v,$$

故 $\ker \tilde{P}$ 为不变补空间。$\square$

## 2. 特征标理论

### 2.1 特征标的定义

**定义 2.1（特征标）**. 表示 $(\rho, V)$ 的特征标（character）为类函数：

$$\chi_\rho(g) = \operatorname{tr}(\rho(g)).$$

**命题 2.2**. 特征标满足：
1. $\chi_\rho(e) = \dim V$；
2. $\chi_\rho(hgh^{-1}) = \chi_\rho(g)$（类函数）；
3. $\chi_{\rho_1 \oplus \rho_2} = \chi_{\rho_1} + \chi_{\rho_2}$；
4. $\chi_{\rho_1 \otimes \rho_2} = \chi_{\rho_1} \cdot \chi_{\rho_2}$。

### 2.2 正交关系

**定理 2.3（Schur 正交关系）**. 设 $(\rho_i, V_i)$ 为有限群 $G$ 的互不等价的不可约复表示，维数为 $n_i$。
1. 矩阵元正交：
   $$\frac{1}{|G|} \sum_{g \in G} \overline{\rho_i(g)_{kl}} \rho_j(g)_{mn} = \frac{\delta_{ij}\delta_{km}\delta_{ln}}{n_i}.$$
2. 特征标正交：
   $$\langle \chi_i, \chi_j \rangle := \frac{1}{|G|} \sum_{g \in G} \overline{\chi_i(g)} \chi_j(g) = \delta_{ij}.$$

*证明 (2)*. 设 $T: V_j \to V_i$ 为线性映射。定义平均 $\\tilde{T} = \frac{1}{|G|}\sum_g \rho_i(g) T \rho_j(g)^{-1}$。由 Schur 引理，$\\tilde{T} = 0$（$i \neq j$）或 $\\tilde{T} = \lambda I$（$i = j$）。取 $T$ 为仅在 $(m,n)$ 位置为 1 的矩阵，计算迹即得。$\square$

**推论 2.4**. 有限群 $G$ 的不可约复表示个数等于其共轭类个数。

### 2.3 特征标表

**定理 2.5（正则表示分解）**. 左正则表示 $\mathbb{C}[G]$ 分解为：

$$\mathbb{C}[G] \cong \bigoplus_{i} (\dim V_i) V_i,$$

其中求和遍历所有不可约表示。特别地，$\sum_i (\dim V_i)^2 = |G|$。

## 3. 诱导表示与 Frobenius 互反

### 3.1 诱导表示

**定义 3.1（诱导表示）**. 设 $H \leq G$，$(\sigma, W)$ 为 $H$ 的表示。诱导表示 $\operatorname{Ind}_H^G(\sigma)$ 为：

$$\operatorname{Ind}_H^G(W) = \{ f: G \to W : f(hg) = \sigma(h)f(g),\ \forall h \in H, g \in G \},$$

$G$ 作用为右平移：$(g \cdot f)(x) = f(xg)$。

**定理 3.2（Frobenius 互反）**. 设 $H \leq G$，$\sigma$ 为 $H$ 的表示，$\rho$ 为 $G$ 的表示。则：

$$\operatorname{Hom}_G(\rho, \operatorname{Ind}_H^G(\sigma)) \cong \operatorname{Hom}_H(\operatorname{Res}_H^G(\rho), \sigma).$$

等价地，$\langle \chi_\rho, \operatorname{Ind}_H^G(\chi_\sigma) \rangle_G = \langle \operatorname{Res}_H^G(\chi_\rho), \chi_\sigma \rangle_H$。

## 4. 例子

### 4.1 例子：$S_3$ 的表示

对称群 $S_3$ 有 3 个共轭类：$\{e\}$，$\{(12), (13), (23)\}$，$\{(123), (132)\}$。

不可约表示：
1. **平凡表示** $\mathbb{C}$：$\rho(g) = 1$，$\chi_1 = (1, 1, 1)$；
2. **符号表示** $\mathbb{C}_\epsilon$：$\rho(g) = \operatorname{sgn}(g)$，$\chi_\epsilon = (1, -1, 1)$；
3. **标准表示** $V$（二维）：$S_3$ 在 $\{x_1 + x_2 + x_3 = 0\} \subset \mathbb{C}^3$ 上的作用。

特征标值：$\chi_V(e) = 2$，$\chi_V((12)) = 0$（无不动点），$\chi_V((123)) = -1$（旋转 $2\pi/3$）。验证正交性：

$$\langle \chi_V, \chi_V \rangle = \frac{1}{6}(4 + 0 + 2) = 1,$$

$$\langle \chi_1, \chi_V \rangle = \frac{1}{6}(2 + 0 - 2) = 0.$$

验证维数公式：$1^2 + 1^2 + 2^2 = 6 = |S_3|$。

### 4.2 例子：四元数群 $Q_8$

$Q_8 = \{\pm 1, \pm i, \pm j, \pm k\}$，$i^2 = j^2 = k^2 = ijk = -1$。不可约复表示：
- 四个一维表示（通过 $Q_8/\{\pm 1\} \cong \mathbb{Z}_2 \times \mathbb{Z}_2$）；
- 一个二维表示：$\rho(i) = \begin{pmatrix} i & 0 \\ 0 & -i \end{pmatrix}$，$\rho(j) = \begin{pmatrix} 0 & 1 \\ -1 & 0 \end{pmatrix}$。

维数公式：$4 \cdot 1^2 + 2^2 = 8$。
