# 域论习题补充

域论（field theory）是研究域扩张及其 Galois 群的代数分支。从古希腊的三大几何作图难题到 Abel-Ruffini 定理（五次以上方程无根式解），域论始终是代数学的核心。

## 1. 域扩张基础

### 1.1 二次扩张

**习题 1.1**. 证明 $\mathbb{Q}(\sqrt{2}) = \{a + b\sqrt{2} : a, b \in \mathbb{Q}\}$ 是域。

**证明**.
- **加法封闭**：$(a + b\sqrt{2}) + (c + d\sqrt{2}) = (a+c) + (b+d)\sqrt{2} \in \mathbb{Q}(\sqrt{2})$。
- **乘法封闭**：$(a + b\sqrt{2})(c + d\sqrt{2}) = (ac + 2bd) + (ad + bc)\sqrt{2} \in \mathbb{Q}(\sqrt{2})$。
- **逆元**：对非零 $a + b\sqrt{2}$，

$$(a + b\sqrt{2})^{-1} = \frac{a - b\sqrt{2}}{a^2 - 2b^2} = \frac{a}{a^2 - 2b^2} - \frac{b}{a^2 - 2b^2}\sqrt{2} \in \mathbb{Q}(\sqrt{2}).$$

分母 $a^2 - 2b^2 \neq 0$（否则 $\sqrt{2} = a/b \in \mathbb{Q}$，矛盾）。$\square$

### 1.2 极小多项式

**习题 1.2**. 求 $\alpha = \sqrt{2} + \sqrt{3}$ 在 $\mathbb{Q}$ 上的极小多项式。

**解答**. 

$$\alpha^2 = 2 + 2\sqrt{6} + 3 = 5 + 2\sqrt{6},$$
$$(\alpha^2 - 5)^2 = 24,$$
$$\alpha^4 - 10\alpha^2 + 25 = 24,$$
$$\alpha^4 - 10\alpha^2 + 1 = 0.$$

多项式 $f(x) = x^4 - 10x^2 + 1$ 在 $\mathbb{Q}$ 上不可约（Eisenstein 判别法经 $x \mapsto x+1$ 不适用，改用：若可约，必分解为两个二次式，通过比较系数可证无此分解；或用 $[\mathbb{Q}(\sqrt{2},\sqrt{3}):\mathbb{Q}] = 4$）。故 $f$ 为极小多项式。$\square$

## 2. 有限域

### 2.1 存在性与唯一性

**习题 2.1**. 证明有限域的阶必为 $q = p^n$（$p$ 为素数），且对每这样的 $q$ 存在唯一（同构意义下）的有限域 $\mathbb{F}_q$。

**证明**. 设 $F$ 为有限域，特征为 $p$。则 $F$ 为 $\mathbb{F}_p$ 上的向量空间，设维数为 $n$，故 $|F| = p^n$。

$F^\times$ 为 $q-1$ 阶乘法群，故 $a^{q-1} = 1$ 对所有 $a \neq 0$ 成立，即 $F$ 中所有元为 $x^q - x = 0$ 的根。故 $F$ 是 $x^q - x$ 在 $\mathbb{F}_p$ 上的分裂域，由分裂域唯一性得唯一性。$\square$

### 2.2 Frobenius 自同构

**习题 2.2**. 证明 $\mathbb{F}_q$ 的 Galois 群 $\operatorname{Gal}(\mathbb{F}_q/\mathbb{F}_p)$ 是 $n$ 阶循环群，由 Frobenius 自同构 $\sigma(x) = x^p$ 生成。

**证明**. $\sigma$ 为自同构：$(ab)^p = a^p b^p$，$(a+b)^p = a^p + b^p$（特征 $p$ 下二项式系数 $\binom{p}{k} \equiv 0 \pmod{p}$ 对 $0 < k < p$）。

$\sigma^n(x) = x^{p^n} = x^q = x$ 对所有 $x \in \mathbb{F}_q$ 成立。对 $k < n$，$\sigma^k(x) = x^{p^k} = x$ 的解至多为 $p^k < q$ 个，故 $\sigma^k \neq \operatorname{id}$。因此 $\sigma$ 的阶恰为 $n$。

由 Galois 理论基本定理，$[\mathbb{F}_q : \mathbb{F}_p] = n$，故 $|\operatorname{Gal}| = n = |\langle \sigma \rangle|$。$\square$

## 3. Galois 理论

### 3.1 分裂域与 Galois 群

**习题 3.1**. 求 $f(x) = x^3 - 2$ 在 $\mathbb{Q}$ 上的分裂域 $K$ 及 $\operatorname{Gal}(K/\mathbb{Q})$。

**解答**. $f$ 的根为 $\sqrt[3]{2}$，$\sqrt[3]{2}\omega$，$\sqrt[3]{2}\omega^2$，其中 $\omega = e^{2\pi i/3} = \frac{-1 + i\sqrt{3}}{2}$。

分裂域 $K = \mathbb{Q}(\sqrt[3]{2}, \omega)$。

扩张次数：$[\mathbb{Q}(\sqrt[3]{2}):\mathbb{Q}] = 3$（$f$ 在 $\mathbb{Q}$ 上 Eisenstein 不可约），$[\mathbb{Q}(\omega):\mathbb{Q}] = 2$（极小多项式 $x^2 + x + 1$）。因 $2 \nmid 3$，$[K:\mathbb{Q}] = 6$。

Galois 群作用在三个根上，给出单射 $\operatorname{Gal}(K/\mathbb{Q}) \hookrightarrow S_3$。因 $|S_3| = 6 = [K:\mathbb{Q}]$，故 $\operatorname{Gal}(K/\mathbb{Q}) \cong S_3$。

生成元：$\sigma: \sqrt[3]{2} \mapsto \sqrt[3]{2}\omega$，$\omega \mapsto \omega$（3-轮换 $(123)$）；$\tau: \sqrt[3]{2} \mapsto \sqrt[3]{2}$，$\omega \mapsto \omega^2$（对换 $(23)$）。$\square$

### 3.2 不可解多项式

**习题 3.2**. 证明 $f(x) = x^5 - 4x + 2$ 在 $\mathbb{Q}$ 上不可约，且其 Galois 群为 $S_5$，从而 $f$ 不可根式解。

**证明**.
- **不可约性**：Eisenstein 判别法，取 $p = 2$。
- **实根分析**：$f(-2) = -32 + 8 + 2 = -22$，$f(-1) = 5$，$f(1) = -1$，$f(2) = 26$。由介值定理，至少 3 个实根。$f'(x) = 5x^4 - 4$，恰有两个实临界点，故恰有 3 个实根、2 个共轭复根。
- **Galois 群含对换**：复共轭给出对换（交换两个复根）。
- **Galois 群可迁**：$f$ 不可约，故群可迁地作用在 5 个根上。
- **$S_5$ 的判定**：可迁且含对换的 $S_5$ 子群必为 $S_5$（或直接由判别式非平方得）。

因 $S_5$ 不可解（$[S_5, S_5] = A_5$ 为单群），由 Galois 理论，$f$ 不可根式解。$\square$

## 4. 例子

### 4.1 例子：分圆扩张

**习题 4.1**. 证明 $\operatorname{Gal}(\mathbb{Q}(\zeta_n)/\mathbb{Q}) \cong (\mathbb{Z}/n\mathbb{Z})^\times$。

**解答**. $\zeta_n = e^{2\pi i/n}$ 的极小多项式为分圆多项式 $\Phi_n(x)$，次数为 $\varphi(n)$。自同构由 $\sigma_k: \zeta_n \mapsto \zeta_n^k$ 确定，其中 $\gcd(k,n) = 1$。$\sigma_k \circ \sigma_l = \sigma_{kl}$，故对应 $(\mathbb{Z}/n\mathbb{Z})^\times$。$\square$

### 4.2 例子：尺规作图的不可能性

**习题 4.2**. 证明三等分任意角与倍立方体不可能用尺规完成。

**解答**.
- **倍立方体**：等价于构造 $\sqrt[3]{2}$。$[\mathbb{Q}(\sqrt[3]{2}):\mathbb{Q}] = 3$ 不是 2 的幂，而尺规可作量对应扩张次数为 $2^k$，故不可能。
- **三等分角**：如 $60^\circ$ 角。$\cos 20^\circ$ 满足 $4\cos^3\theta - 3\cos\theta = \cos 60^\circ = 1/2$，即 $8x^3 - 6x - 1 = 0$。令 $y = 2x$，$y^3 - 3y - 1 = 0$，在 $\mathbb{Q}$ 上不可约（无有理根），故 $[\mathbb{Q}(\cos 20^\circ):\mathbb{Q}] = 3$，非 2 的幂。$\square$
