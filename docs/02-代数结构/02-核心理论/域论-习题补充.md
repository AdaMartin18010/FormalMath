# 域论习题补充

## 基础题

**习题1** 证明：$\mathbb{Q}(\sqrt{2}) = \{a + b\sqrt{2} : a, b \in \mathbb{Q}\}$是域。

**证明**：

- 加法封闭：$(a+b\sqrt{2}) + (c+d\sqrt{2}) = (a+c) + (b+d)\sqrt{2} \in \mathbb{Q}(\sqrt{2})$
- 乘法封闭：$(a+b\sqrt{2})(c+d\sqrt{2}) = (ac+2bd) + (ad+bc)\sqrt{2} \in \mathbb{Q}(\sqrt{2})$
- 逆元：$(a+b\sqrt{2})^{-1} = \frac{a-b\sqrt{2}}{a^2-2b^2} \in \mathbb{Q}(\sqrt{2})$

---

**习题2** 求$\sqrt{2} + \sqrt{3}$在$\mathbb{Q}$上的极小多项式。

**解答**：设$\alpha = \sqrt{2} + \sqrt{3}$
$$\alpha^2 = 2 + 2\sqrt{6} + 3 = 5 + 2\sqrt{6}$$
$$(\alpha^2 - 5)^2 = 24$$
$$\alpha^4 - 10\alpha^2 + 25 = 24$$
$$\alpha^4 - 10\alpha^2 + 1 = 0$$

极小多项式为$x^4 - 10x^2 + 1$。

---

**习题3** 证明：有限域的阶必为$p^n$（$p$为素数）。

**证明**：设$F$是有限域，特征为$p$。
则$F$是$\mathbb{F}_p$上的向量空间。
设维数为$n$，则$|F| = p^n$。

---

## 伽罗瓦理论

**习题4** 求$x^3 - 2$在$\mathbb{Q}$上的分裂域及其伽罗瓦群。

**解答**：

- 根：$\sqrt[3]{2}$，$\sqrt[3]{2}\omega$，$\sqrt[3]{2}\omega^2$（其中$\omega = e^{2\pi i/3}$）
- 分裂域：$\mathbb{Q}(\sqrt[3]{2}, \omega)$
- 扩张次数：$[\mathbb{Q}(\sqrt[3]{2}, \omega):\mathbb{Q}] = 6$
- 伽罗瓦群：$S_3$（3次对称群）

---

**习题5** 证明：$x^5 - 4x + 2$在$\mathbb{Q}$上不可约，且其伽罗瓦群是$S_5$。

**证明**：

- Eisenstein判别（$p=2$）：不可约
- 分析实根个数（3个实根，2个复根）
- 伽罗瓦群包含复共轭（一个对换）
- 由群论知识，包含对换的$\text{Gal}(f)$是$S_5$
