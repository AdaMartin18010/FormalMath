# 导出范畴 - 深度版

> **"导出范畴是同调代数最深刻的构造之一，它统一了上同调理论的种种变体。"** —— Alexander Grothendieck的变革性贡献

---

## 📚 概述

导出范畴（Derived Category）是格洛腾迪克在20世纪60年代引入的革命性概念，它将链复形的同伦范畴进行局部化，使得拟同构成为同构。这一构造统一了各种上同调理论（如层上同调、群上同调、Hochschild上同调），为代数几何和表示论提供了强大的工具。

---

## 🕰️ 历史发展与数学家理念

### 格洛腾迪克的革命

**格洛腾迪克数学理念**关于导出范畴的核心洞察：

1. **相对观点**：不应单独研究对象的上同调，而应研究导出范畴中的对象本身
2. **函子性**：导出函子应在导出范畴的层次定义
3. **对偶性**：Serre对偶、Verdier对偶在导出范畴中自然表达

**历史背景**：

- 1960年代，Grothendieck与Verdier在巴黎IHES发展导出范畴理论
- 动机：为对偶定理提供适当的框架
- 首次发表：Verdier的博士论文（1967），但直到1996年才出版

### Verdier的贡献

Jean-Louis Verdier在格洛腾迪克指导下完成了导出范畴的基础工作：

- **Verdier局部化**：在三角范畴中反转拟同构
- **t-结构**：从导出范畴中提取阿贝尔范畴的方法
- **稳定无穷范畴**：导出范畴的现代推广

## 🏗️ 核心概念与完整证明

### 1. 链复形与同伦范畴

**定义 19.1**（链复形）
阿贝尔范畴$\mathcal{A}$中的**链复形**$A^\bullet$是对象序列$\{A^n\}_{n \in \mathbb{Z}}$，带微分$d^n: A^n \to A^{n+1}$满足$d^{n+1} \circ d^n = 0$。

**定义 19.2**（同伦）
两个链映射$f, g: A^\bullet \to B^\bullet$称为**同伦**，如果存在**同伦算子**$h^n: A^n \to B^{n-1}$使得：
$$f^n - g^n = d_B^{n-1} \circ h^n + h^{n+1} \circ d_A^n$$

**定义 19.3**（同伦范畴）
**同伦范畴**$K(\mathcal{A})$：

- 对象：链复形
- 态射：链映射的同伦等价类

### 2. 导出范畴的构造

**定义 19.4**（拟同构）
链映射$f: A^\bullet \to B^\bullet$称为**拟同构**，如果对每个$n$，$H^n(f): H^n(A^\bullet) \to H^n(B^\bullet)$是同构。

**定义 19.5**（导出范畴）
**导出范畴**$D(\mathcal{A})$是$K(\mathcal{A})$关于拟同构的**局部化**。

**构造过程**（Verdier）：

**步骤1**：定义乘法系。

拟同构$Q$满足：

- 包含所有恒等态射
- 对复合封闭
- 满足Ore条件（左右可延拓）

**步骤2**：定义态射为 roofs。

$D(\mathcal{A})$中$A^\bullet \to B^\bullet$的态射由**roof**表示：

```
    C^•
   /   \
  s     f
 /       \
A^•     B^•
```

其中$s$是拟同构。

**步骤3**：定义等价关系。

两个roofs等价如果存在更大roofs使下图交换：

```
      D^•
     /   \
    /     \
   C₁^•   C₂^•
   |\     /|
   | \   / |
   |  \ /  |
   A^•  B^•
```

**步骤4**：定义复合。

利用Ore条件构造roofs的复合。

### 3. 导出函子

**定理 19.1**（导出函子的存在性）
设$F: \mathcal{A} \to \mathcal{B}$是左正合函子。若$\mathcal{A}$有足够内射对象，则存在**右导出函子**：
$$RF: D^+(\mathcal{A}) \to D^+(\mathcal{B})$$

**构造**：

**步骤1**：内射分解。

对$A^\bullet \in D^+(\mathcal{A})$，选取拟同构$A^\bullet \to I^\bullet$，其中$I^\bullet$是内射复形。

**步骤2**：应用函子。

$$RF(A^\bullet) = F(I^\bullet)$$

**步骤3**：验证良定性。

内射分解在同伦范畴中唯一，故$RF$良定。

**经典例子**：

- **$R\Gamma$**：整体截面函子的导出
- **$R\text{Hom}$**：Hom函子的导出，给出Ext
- **$\otimes^L$**：张量积的导出，给出Tor

### 4. t-结构与 hearts

**定义 19.6**（t-结构）
导出范畴$D$上的**t-结构**是一对满子范畴$(D^{\leq 0}, D^{\geq 0})$满足：

1. **包含**：$D^{\leq 0}[1] \subseteq D^{\leq 0}$，$D^{\geq 0}[-1] \subseteq D^{\geq 0}$
2. **Hom消失**：$\text{Hom}(X, Y) = 0$对$X \in D^{\leq 0}$，$Y \in D^{\geq 1}$
3. **截断存在**：任意$Z \in D$存在 distinguished triangle：
$$\tau^{\leq 0} Z \to Z \to \tau^{\geq 1} Z \to$$

**定理 19.2**（Verdier）：t-结构的**heart**$D^\heartsuit = D^{\leq 0} \cap D^{\geq 0}$是阿贝尔范畴。

**标准t-结构**：

- $D^{\leq 0} = \{A^\bullet : H^n(A^\bullet) = 0, \forall n > 0\}$
- $D^{\geq 0} = \{A^\bullet : H^n(A^\bullet) = 0, \forall n < 0\}$

heart给出原始阿贝尔范畴$\mathcal{A}$。

### 5. 应用：Serre对偶

**定理 19.3**（Serre对偶）
设$X$是$n$维光滑射影簇，$\omega_X$是典范丛。则存在自然同构：
$$\text{Ext}^i(\mathcal{F}, \omega_X) \cong H^{n-i}(X, \mathcal{F})^*$$

在导出范畴中，这等价于：
$$R\text{Hom}(\mathcal{F}, \omega_X[n]) \cong R\Gamma(X, \mathcal{F})^*$$
