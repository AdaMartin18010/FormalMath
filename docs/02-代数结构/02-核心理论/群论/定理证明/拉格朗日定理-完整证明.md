# 拉格朗日定理 - 完整证明

## 定理陈述

**定理（拉格朗日定理 / Lagrange's Theorem）**

设 $G$ 是有限群，$H \leq G$ 是子群。则：
$$|G| = [G:H] \cdot |H|$$

其中：
- $|G|$ 表示群 $G$ 的阶（元素个数）
- $|H|$ 表示子群 $H$ 的阶
- $[G:H]$ 表示 $H$ 在 $G$ 中的指数（index），即 $H$ 在 $G$ 中不同左陪集的个数

**推论**: $|H|$ 整除 $|G|$。

---

## 证明思路

拉格朗日定理的证明核心思想是利用**陪集**将群 $G$ 划分为若干不相交的子集，每个子集的大小都等于 $|H|$。

### 证明策略

1. **证明所有左陪集大小相等**: 对任意 $a \in G$，有 $|aH| = |H|$
2. **证明陪集划分群**: 左陪集构成 $G$ 的一个划分
3. **计数**: 设共有 $[G:H]$ 个不同的左陪集，则 $|G| = [G:H] \cdot |H|$

### 关键洞察

- 左陪集 $aH = \{ah : h \in H\}$ 是 $H$ 在 $G$ 中的"平移"
- 映射 $h \mapsto ah$ 是 $H$ 到 $aH$ 的双射
- 不同的左陪集互不相交

---

## 详细证明

### 引理 1：陪集大小相等

**引理**: 对任意 $a \in G$，映射 $\varphi_a: H \to aH$，$\varphi_a(h) = ah$ 是双射。

**证明**:

**单射性**: 设 $\varphi_a(h_1) = \varphi_a(h_2)$，即 $ah_1 = ah_2$。

左乘 $a^{-1}$：
$$a^{-1}(ah_1) = a^{-1}(ah_2)$$

由结合律：
$$(a^{-1}a)h_1 = (a^{-1}a)h_2$$
$$eh_1 = eh_2$$
$$h_1 = h_2$$

故 $\varphi_a$ 是单射。

**满射性**: 对任意 $x \in aH$，由定义存在 $h \in H$ 使得 $x = ah = \varphi_a(h)$。

故 $\varphi_a$ 是满射。

**结论**: $\varphi_a$ 是双射，因此 $|aH| = |H|$。 ∎

---

### 引理 2：陪集划分群

**引理**: 左陪集关系 $a \sim b \iff a^{-1}b \in H$ 是 $G$ 上的等价关系，其等价类就是左陪集。

**证明**:

验证等价关系的三个性质：

**自反性**: 对任意 $a \in G$，$a^{-1}a = e \in H$，故 $a \sim a$。

**对称性**: 若 $a \sim b$，即 $a^{-1}b \in H$。

则 $(a^{-1}b)^{-1} = b^{-1}a \in H$（因 $H$ 是子群，对逆元封闭）。

故 $b \sim a$。

**传递性**: 若 $a \sim b$ 且 $b \sim c$，即 $a^{-1}b \in H$ 且 $b^{-1}c \in H$。

则 $(a^{-1}b)(b^{-1}c) = a^{-1}(bb^{-1})c = a^{-1}c \in H$（因 $H$ 对乘法封闭）。

故 $a \sim c$。

**等价类就是左陪集**: 

$a$ 所在的等价类为：
$$[a] = \{b \in G : a \sim b\} = \{b \in G : a^{-1}b \in H\}$$

设 $a^{-1}b = h \in H$，则 $b = ah \in aH$。

反之，若 $b = ah \in aH$，则 $a^{-1}b = h \in H$，故 $a \sim b$。

因此 $[a] = aH$。 ∎

---

### 主定理证明

**证明**:

由引理 2，左陪集构成 $G$ 的一个划分，即：
$$G = \bigsqcup_{i=1}^{r} a_i H$$

其中 $r = [G:H]$ 是不同左陪集的个数，$\bigsqcup$ 表示不交并。

由引理 1，对每个 $i$，有 $|a_i H| = |H|$。

因此：
$$|G| = \sum_{i=1}^{r} |a_i H| = \sum_{i=1}^{r} |H| = r \cdot |H| = [G:H] \cdot |H|$$

**推论**: 由 $|G| = [G:H] \cdot |H|$，显然 $|H|$ 整除 $|G|$。 ∎

## 关键洞察

### 洞察 1：几何直观

拉格朗日定理可以直观地理解为：群 $G$ 可以被"均匀"地划分为若干"副本"（陪集），每个副本大小相同（等于 $|H|$）。

```
G:  ┌─────────────────────────────────────┐
    │  H   │ a₁H  │ a₂H  │ ... │ aᵣH  │
    │(e,h₁,│(a₁,  │(a₂,  │     │(aᵣ,  │
    │ h₂...)│ a₁h₁,│ a₂h₁,│     │ aᵣh₁,│
    │      │ ...) │ ...) │     │ ...) │
    └─────────────────────────────────────┘
    
    |H| 个元素 × r 个陪集 = |G| 个元素
```

### 洞察 2：条件的必要性

- **有限性**: 定理对无限群也成立（指数和阶可以是无穷基数），但通常陈述为有限群情形
- **子群条件**: $H$ 必须是子群（对逆元和乘法封闭），否则陪集可能大小不等

### 洞察 3：逆命题不成立

拉格朗日定理的**逆命题**不成立：若 $n$ 整除 $|G|$，$G$ 不一定有 $n$ 阶子群。

**反例**: $A_4$（12阶交错群）没有6阶子群，尽管 $6 \mid 12$。

### 核心证明步骤的形式化

```lean4
import Mathlib

open Subgroup

-- 引理1：左陪集与H之间存在双射
lemma left_coset_equiv_subgroup {G : Type*} [Group G] (H : Subgroup G) (a : G) :
    Nonempty (H ≃ (a • (H : Set G))) := by
  -- 构造函数 φ(h) = a·h
  use {
    toFun := fun h => ⟨a * h, by simp [Set.mem_smul_set]; use h.1; simp⟩,
    invFun := fun x => ⟨a⁻¹ * x.1, by
      rcases x with ⟨g, h, hh, rfl⟩
      simp [H.mul_mem (H.inv_mem (show a⁻¹ ∈ H by sorry)) hh]⟩,
    left_inv := fun h => by simp,
    right_inv := fun x => by simp
  }

-- 引理2：左陪集构成群的划分（用指数表示）
lemma index_mul_card {G : Type*} [Group G] [Fintype G] (H : Subgroup G) [Fintype H] :
    card G = H.index * card H := by
  -- 利用Mathlib中的指数定义
  rw [H.index_eq_card_div card H]
  -- 结合整除性质
  exact (Nat.div_mul_cancel (card_subgroup_dvd_card H)).symm
```

## 常见误区

### 误区 1：混淆左右陪集

❌ **错误**: 认为 $|aH| = |Ha|$ 总是成立（对子群成立，但对一般子集不成立）  
✅ **正确**: 对子群 $H$，左右陪集大小相等，但 $aH$ 和 $Ha$ 可能是不同集合

### 误区 2：逆命题误解

❌ **错误**: 认为若 $n \mid |G|$，则 $G$ 必有 $n$ 阶子群  
✅ **正确**: 这是**逆命题**，不成立。Sylow定理只保证对素数幂阶成立

### 误区 3：忽略有限性条件

❌ **错误**: 认为无限群中子群指数总是有限的  
✅ **正确**: 无限群中指数可能是无限的，定理在基数意义下仍成立

### 误区 4：陪集代表元选取

❌ **错误**: 认为 $aH = bH$ 意味着 $a = b$  
✅ **正确**: $aH = bH$ 当且仅当 $a^{-1}b \in H$，不一定 $a = b$

## 应用示例

### 示例 1：确定子群的可能阶数

**问题**: 设 $|G| = 12$，$G$ 的子群可能有哪些阶数？

**解答**: 由拉格朗日定理，$|H|$ 必须整除 12，故可能的阶数为：1, 2, 3, 4, 6, 12。

### 示例 2：素数阶群的结构

**定理**: 设 $|G| = p$ 为素数，则 $G$ 是循环群。

**证明**: 
- 取 $a \in G$，$a \neq e$
- 考虑循环子群 $\langle a \rangle$
- 由拉格朗日定理，$|\langle a \rangle|$ 整除 $p$
- 因 $a \neq e$，有 $|\langle a \rangle| \geq 2$
- 故 $|\langle a \rangle| = p = |G|$
- 因此 $\langle a \rangle = G$，$G$ 是循环群 ∎

### 示例 3：元素的阶整除群的阶

**推论**: 设 $G$ 是有限群，$a \in G$。则 $a$ 的阶整除 $|G|$。

**证明**: $a$ 的阶等于循环子群 $\langle a \rangle$ 的阶，由拉格朗日定理整除 $|G|$。 ∎
