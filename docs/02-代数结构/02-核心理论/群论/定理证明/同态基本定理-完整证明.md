# 同态基本定理 - 完整证明

## 定理陈述

**定理（同态基本定理 / 第一同构定理）**

设 $\varphi: G \to G'$ 是群的满同态，$N = \ker \varphi$ 是同态的核。则：
$$G/N \cong G'$$

同构映射为 $\bar{\varphi}: G/N \to G'$，定义为：
$$\bar{\varphi}(aN) = \varphi(a)$$

更一般的形式（不要求满射）：
$$G/\ker \varphi \cong \text{im } \varphi$$

---

## 证明思路

同态基本定理是群论中最重要的定理之一，它建立了**商群**与**同态像**之间的同构关系。

### 证明策略

1. **定义映射**: 构造 $\bar{\varphi}: G/N \to G'$，$\bar{\varphi}(aN) = \varphi(a)$
2. **验证良定义性**: 证明映射值与代表元选取无关
3. **验证同态性**: 证明 $\bar{\varphi}$ 保持群运算
4. **验证单射性**: 证明不同的陪集映射到不同的元素
5. **验证满射性**: 利用 $\varphi$ 的满射性

### 关键洞察

- **核的正规性**: 同态的核 $N = \ker \varphi$ 总是 $G$ 的正规子群，保证商群 $G/N$ 良定义
- **代表元无关性**: 若 $aN = bN$，则 $\varphi(a) = \varphi(b)$，保证 $\bar{\varphi}$ 良定义
- **结构保持**: 商群 $G/N$ "遗忘"了核中的信息，恰好保留了 $\varphi$ 的像

---

## 详细证明

### 准备：核的正规性

**引理**: 设 $\varphi: G \to G'$ 是群同态，则 $N = \ker \varphi \trianglelefteq G$（$N$ 是 $G$ 的正规子群）。

**证明**:

首先验证 $N$ 是子群：
- $e \in N$：因 $\varphi(e) = e'$
- 封闭性：若 $n_1, n_2 \in N$，则 $\varphi(n_1 n_2) = \varphi(n_1)\varphi(n_2) = e'e' = e'$，故 $n_1 n_2 \in N$
- 逆元：若 $n \in N$，则 $\varphi(n^{-1}) = \varphi(n)^{-1} = e'^{-1} = e'$，故 $n^{-1} \in N$

再验证正规性：对任意 $g \in G$，$n \in N$：
$$\varphi(gng^{-1}) = \varphi(g)\varphi(n)\varphi(g)^{-1} = \varphi(g)e'\varphi(g)^{-1} = e'$$

故 $gng^{-1} \in N$，即 $N \trianglelefteq G$。 ∎

---

### 步骤 1：定义映射

定义 $\bar{\varphi}: G/N \to G'$ 为：
$$\bar{\varphi}(aN) = \varphi(a)$$

---

### 步骤 2：验证良定义性

**证明**: 需证若 $aN = bN$，则 $\bar{\varphi}(aN) = \bar{\varphi}(bN)$。

设 $aN = bN$，即 $a^{-1}b \in N = \ker \varphi$。

则：
$$\varphi(a^{-1}b) = e'$$
$$\varphi(a)^{-1}\varphi(b) = e'$$
$$\varphi(a) = \varphi(b)$$

因此 $\bar{\varphi}(aN) = \varphi(a) = \varphi(b) = \bar{\varphi}(bN)$。 ∎

---

### 步骤 3：验证同态性

**证明**: 需证 $\bar{\varphi}((aN)(bN)) = \bar{\varphi}(aN)\bar{\varphi}(bN)$。

计算左边：
$$\bar{\varphi}((aN)(bN)) = \bar{\varphi}((ab)N) = \varphi(ab) = \varphi(a)\varphi(b)$$

计算右边：
$$\bar{\varphi}(aN)\bar{\varphi}(bN) = \varphi(a)\varphi(b)$$

两边相等，故 $\bar{\varphi}$ 是同态。 ∎

---

### 步骤 4：验证单射性

**证明**: 需证若 $\bar{\varphi}(aN) = \bar{\varphi}(bN)$，则 $aN = bN$。

设 $\bar{\varphi}(aN) = \bar{\varphi}(bN)$，即 $\varphi(a) = \varphi(b)$。

则：
$$\varphi(a)^{-1}\varphi(b) = e'$$
$$\varphi(a^{-1}b) = e'$$
$$a^{-1}b \in \ker \varphi = N$$

因此 $aN = bN$。 ∎

---

### 步骤 5：验证满射性

**证明**: 对任意 $g' \in G'$，需找 $aN \in G/N$ 使得 $\bar{\varphi}(aN) = g'$。

因 $\varphi$ 是满射，存在 $a \in G$ 使得 $\varphi(a) = g'$。

则：
$$\bar{\varphi}(aN) = \varphi(a) = g'$$

故 $\bar{\varphi}$ 是满射。 ∎

---

### 结论

**结论**: $\bar{\varphi}: G/N \to G'$ 是双射同态，即同构。

因此：
$$G/N \cong G'$$
或更一般地：
$$G/\ker \varphi \cong \text{im } \varphi$$ ∎

---

## 关键洞察

### 洞察 1：商群的本质

商群 $G/N$ 可以直观地理解为：
- 将 $N$ 中所有元素"坍缩"为单位元
- 保留 $G$ 中其他元素的结构信息
- 同态 $\varphi$ 恰好将 $N$ 映射到单位元

```
G:      e    n₁    n₂   ...   a    an₁   an₂   ...   b    bn₁   bn₂   ...
        ↓    ↓     ↓          ↓    ↓     ↓          ↓    ↓     ↓
       ═════════════════════════════════════════════════════════════════
        │    全部映射到     │         │  映射到        │         │
        │    e' ∈ G'       │         │  φ(a) ∈ G'     │         │
        └──────────────────┘         └────────────────┘         │
                 N                           aN                  │
                                                                  │
                                                          映射到 φ(b)
```

### 洞察 2：同态分解

任何同态 $\varphi: G \to G'$ 都可以分解为：
$$G \xrightarrow{\pi} G/\ker \varphi \xrightarrow{\bar{\varphi}} \text{im } \varphi \hookrightarrow G'$$

其中：
- $\pi$ 是标准投影（满射）
- $\bar{\varphi}$ 是同构（双射）
- $\hookrightarrow$ 是包含映射（单射）

### 洞察 3：核作为"遗忘"

同态 $\varphi$ "遗忘"了核 $N$ 中的信息，而商群 $G/N$ 恰好编码了这种遗忘后的结构。

## 常见误区

### 误区 1：忽略良定义性验证

❌ **错误**: 直接定义 $\bar{\varphi}(aN) = \varphi(a)$ 而不验证代表元无关性  
✅ **正确**: 必须证明若 $aN = bN$，则 $\varphi(a) = \varphi(b)$

### 误区 2：混淆核与像

❌ **错误**: 认为 $G/\text{im } \varphi \cong \ker \varphi$  
✅ **正确**: 正确的关系是 $G/\ker \varphi \cong \text{im } \varphi$

### 误区 3：忽略满射条件

❌ **错误**: 认为 $G/\ker \varphi \cong G'$ 对任意同态成立  
✅ **正确**: 必须 $\varphi$ 满射，否则只同构于像 $\text{im } \varphi$

### 误区 4：商群元素理解错误

❌ **错误**: 认为 $aN$ 是 $G$ 的子集所以不能参与运算  
✅ **正确**: $aN$ 是商群 $G/N$ 的元素，陪集乘法是良定义的

## 应用示例

### 示例 1：整数加法群的商群

**问题**: 证明 $\mathbb{Z}/n\mathbb{Z} \cong C_n$（$n$ 阶循环群）。

**证明**:
- 定义 $\varphi: \mathbb{Z} \to C_n = \langle g \rangle$，$\varphi(k) = g^k$
- 验证这是满同态
- $\ker \varphi = n\mathbb{Z}$
- 由同态基本定理：$\mathbb{Z}/n\mathbb{Z} \cong C_n$ ∎

### 示例 2：行列式映射

**问题**: 证明 $GL_n(\mathbb{R})/SL_n(\mathbb{R}) \cong \mathbb{R}^*$。

**证明**:
- 定义 $\varphi: GL_n(\mathbb{R}) \to \mathbb{R}^*$，$\varphi(A) = \det(A)$
- 验证这是满同态（行列式可以取任意非零实数）
- $\ker \varphi = \{A : \det(A) = 1\} = SL_n(\mathbb{R})$
- 由同态基本定理：$GL_n(\mathbb{R})/SL_n(\mathbb{R}) \cong \mathbb{R}^*$ ∎

### 示例 3：符号同态

**问题**: 证明 $S_n/A_n \cong C_2$。

**证明**:
- 定义符号同态 $\varepsilon: S_n \to \{1, -1\} \cong C_2$
- 这是满同态
- $\ker \varepsilon = A_n$（交错群）
- 由同态基本定理：$S_n/A_n \cong C_2$ ∎

## 第二、第三同构定理

### 第二同构定理（Diamond Isomorphism）

**定理**: 设 $H \leq G$，$N \trianglelefteq G$。则：
$$HN/N \cong H/(H \cap N)$$

### 第三同构定理

**定理**: 设 $N \trianglelefteq G$，$K \trianglelefteq G$，$N \subseteq K$。则：
$$(G/N)/(K/N) \cong G/K$$
