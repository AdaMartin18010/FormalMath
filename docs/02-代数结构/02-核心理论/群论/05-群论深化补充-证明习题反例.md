## 一、核心定理完整证明

### 1.1 拉格朗日定理完整证明

**定理 1.1** (拉格朗日定理 / Lagrange's Theorem)

设 $G$ 是有限群，$H \leq G$ 是子群，则：
$$|G| = |H| \cdot [G : H]$$

其中 $[G : H]$ 是 $H$ 在 $G$ 中的指数（左陪集的个数）。

**完整证明**：

**步骤 1**：定义左陪集

对于 $a \in G$，定义 $a$ 关于 $H$ 的左陪集为：
$$aH = \{ah : h \in H\}$$

**步骤 2**：证明不同左陪集不相交

**引理**：若 $aH \cap bH \neq \emptyset$，则 $aH = bH$。

*证明*：设 $x \in aH \cap bH$，则存在 $h_1, h_2 \in H$ 使得 $x = ah_1 = bh_2$。

因此 $a = bh_2h_1^{-1} \in bH$。

对任意 $ah \in aH$，有：
$$ah = (bh_2h_1^{-1})h = b(h_2h_1^{-1}h) \in bH$$

所以 $aH \subseteq bH$。同理可证 $bH \subseteq aH$，故 $aH = bH$。

**步骤 3**：证明每个左陪集大小等于 $|H|$

定义映射 $\phi: H \to aH$，$\phi(h) = ah$。

- **单射**：若 $\phi(h_1) = \phi(h_2)$，则 $ah_1 = ah_2$，由消去律得 $h_1 = h_2$。
- **满射**：对任意 $ah \in aH$，有 $h \in H$ 使得 $\phi(h) = ah$。

因此 $|aH| = |H|$。

**步骤 4**：完成证明

由于 $G$ 是有限群，存在有限个不同的左陪集 $a_1H, a_2H, \ldots, a_kH$，使得：
$$G = a_1H \cup a_2H \cup \cdots \cup a_kH$$

且这些陪集两两不相交。因此：
$$|G| = |a_1H| + |a_2H| + \cdots + |a_kH| = k \cdot |H| = [G : H] \cdot |H|$$

**证毕**。

---

**推论 1.1**：有限群中元素的阶整除群的阶。

*证明*：设 $g \in G$，则 $\langle g \rangle$ 是 $G$ 的子群，且 $|\langle g \rangle| = |g|$（$g$ 的阶）。

由拉格朗日定理，$|\langle g \rangle|$ 整除 $|G|$，即 $|g|$ 整除 $|G|$。

### 1.2 西罗定理完整证明

**定理 1.2** (西罗定理 / Sylow Theorems)

设 $G$ 是有限群，$|G| = p^a m$，其中 $p$ 是素数，$p \nmid m$。

**第一西罗定理** (存在性)：$G$ 存在阶为 $p^a$ 的子群（称为西罗 $p$-子群）。

**第二西罗定理** (共轭性)：$G$ 的所有西罗 $p$-子群彼此共轭。

**第三西罗定理** (计数)：设 $n_p$ 是西罗 $p$-子群的个数，则：
- $n_p \equiv 1 \pmod{p}$
- $n_p \mid m$

**前置引理**：

**引理 1.1** (Cauchy定理)：若素数 $p \mid |G|$，则 $G$ 中存在 $p$ 阶元素。

*证明*：考虑集合 $X = \{(g_1, \ldots, g_p) \in G^p : g_1 g_2 \cdots g_p = e\}$。

循环群 $C_p = \langle \sigma \rangle$ 作用在 $X$ 上通过循环移位：$\sigma \cdot (g_1, \ldots, g_p) = (g_p, g_1, \ldots, g_{p-1})$。

$|X| = |G|^{p-1}$（前 $p-1$ 个元素任意，最后一个被确定）。

由轨道-稳定子定理，轨道大小整除 $|C_p| = p$，故轨道大小为 1 或 $p$。

大小为 1 的轨道对应元素 $(g, g, \ldots, g)$ 满足 $g^p = e$。

$(e, \ldots, e)$ 是一个不动点。若这是唯一的不动点，则 $|X| \equiv 1 \pmod{p}$。

但 $p \mid |G|$ 意味着 $|X| = |G|^{p-1} \equiv 0 \pmod{p}$，矛盾。

故存在 $g \neq e$ 使得 $g^p = e$，即 $|g| = p$。

---

**第一西罗定理证明**（归纳法）：

**基例**：$|G| = 1$ 时显然成立。

**归纳步骤**：设对阶小于 $|G|$ 的群定理成立。

考虑类方程：
$$|G| = |Z(G)| + \sum_{i} [G : C_G(g_i)]$$

其中 $g_i$ 取遍非中心共轭类的代表元。

**情况 1**：$p \mid |Z(G)|$

由Cauchy定理，$Z(G)$ 存在 $p$ 阶子群 $N \trianglelefteq G$（因为 $N \subseteq Z(G)$）。

考虑 $G/N$，$|G/N| = p^{a-1} m$。由归纳假设，$G/N$ 有西罗 $p$-子群 $\bar{P}$，阶为 $p^{a-1}$。

设 $P$ 是 $\bar{P}$ 在自然同态 $G \to G/N$ 下的原像，则 $|P| = p \cdot p^{a-1} = p^a$。

**情况 2**：$p \nmid |Z(G)|$

则存在某个 $g_i$ 使得 $p \nmid [G : C_G(g_i)]$（否则 $|G| \equiv 0 \pmod{p}$ 与 $p \nmid |Z(G)|$ 矛盾）。

由于 $|G| = |C_G(g_i)| \cdot [G : C_G(g_i)]$，且 $p^a \mid |G|$，$p \nmid [G : C_G(g_i)]$，故 $p^a \mid |C_G(g_i)|$。

又 $[G : C_G(g_i)] > 1$，所以 $|C_G(g_i)| < |G|$。

由归纳假设，$C_G(g_i)$ 有西罗 $p$-子群，阶为 $p^a$，也是 $G$ 的西罗 $p$-子群。

---

**第二西罗定理证明**：

设 $P$ 和 $Q$ 是 $G$ 的两个西罗 $p$-子群，$|P| = |Q| = p^a$。

考虑 $Q$ 通过左乘作用在左陪集空间 $G/P$ 上：
$$g \cdot (xP) = (gx)P \quad \text{对 } g \in Q, xP \in G/P$$

由轨道-稳定子定理，$Q$-轨道的长度整除 $|Q| = p^a$。

注意 $|G/P| = [G:P] = m$，$p \nmid m$。

设不动点为 $(G/P)^Q = \{xP : qxP = xP \text{ 对所有 } q \in Q\}$。

若 $xP \in (G/P)^Q$，则 $Qx \subseteq xP$，即 $Q \subseteq xPx^{-1}$。

由于 $|Q| = |xPx^{-1}| = p^a$，得 $Q = xPx^{-1}$。

由轨道分解：$m = |G/P| = |(G/P)^Q| + \sum_{\text{非平凡轨道}} |\text{轨道}|$

非平凡轨道大小是 $p$ 的正幂次，故模 $p$ 为 0。因此 $|(G/P)^Q| \equiv m \not\equiv 0 \pmod{p}$。

所以 $(G/P)^Q \neq \emptyset$，存在不动点，即 $Q$ 与 $P$ 共轭。

---

**第三西罗定理证明**：

**步骤 1**：证明 $n_p = [G : N_G(P)]$

由第二西罗定理，所有西罗 $p$-子群共轭。$G$ 通过共轭作用在西罗 $p$-子群集合上，该作用可递。

$P$ 的稳定子是 $N_G(P) = \{g \in G : gPg^{-1} = P\}$。

由轨道-稳定子定理，$n_p = [G : N_G(P)]$，故 $n_p \mid |G| = p^a m$。

**步骤 2**：证明 $n_p \equiv 1 \pmod{p}$

固定一个西罗 $p$-子群 $P$。让 $P$ 通过共轭作用在所有西罗 $p$-子群的集合 $Syl_p(G)$ 上：
$$g \cdot Q = gQg^{-1}, \quad g \in P, Q \in Syl_p(G)$$

对 $Q \in Syl_p(G)$，若 $Q$ 是不动点，则 $P \subseteq N_G(Q)$。

$P$ 和 $Q$ 都是 $N_G(Q)$ 的西罗 $p$-子群，在 $N_G(Q)$ 中共轭。

但 $Q \trianglelefteq N_G(Q)$，故 $Q$ 是 $N_G(Q)$ 的唯一西罗 $p$-子群，因此 $P = Q$。

所以唯一的不动点是 $P$ 本身。

由轨道分解，$n_p = |Syl_p(G)| = 1 + \sum_{\text{非平凡轨道}} |\text{轨道}|$

非平凡轨道大小整除 $|P| = p^a$，故模 $p$ 为 0。因此 $n_p \equiv 1 \pmod{p}$。

**步骤 3**：证明 $n_p \mid m$

由步骤 1，$n_p = [G : N_G(P)] = \frac{|G|}{|N_G(P)|} = \frac{p^a m}{|N_G(P)|}$。

由于 $P \subseteq N_G(P)$，$p^a \mid |N_G(P)|$，设 $|N_G(P)| = p^a k$。

则 $n_p = \frac{p^a m}{p^a k} = \frac{m}{k}$，故 $n_p \mid m$。

**证毕**。

### 1.3 同态基本定理完整证明

**定理 1.3** (第一同构定理 / First Isomorphism Theorem)

设 $\phi: G \to H$ 是群同态，则：
$$G/\ker(\phi) \cong \text{im}(\phi)$$

**完整证明**：

**步骤 1**：证明 $\ker(\phi)$ 是正规子群

对任意 $g \in G$ 和 $k \in \ker(\phi)$：
$$\phi(gkg^{-1}) = \phi(g)\phi(k)\phi(g)^{-1} = \phi(g)e_H\phi(g)^{-1} = e_H$$

因此 $gkg^{-1} \in \ker(\phi)$，所以 $\ker(\phi) \trianglelefteq G$。

**步骤 2**：定义映射 $\bar{\phi}: G/\ker(\phi) \to \text{im}(\phi)$

定义 $\bar{\phi}(g\ker(\phi)) = \phi(g)$。

**步骤 3**：证明 $\bar{\phi}$ 是良定义的

若 $g_1\ker(\phi) = g_2\ker(\phi)$，则 $g_1^{-1}g_2 \in \ker(\phi)$。

因此：
$$\phi(g_1^{-1}g_2) = e_H \Rightarrow \phi(g_1)^{-1}\phi(g_2) = e_H \Rightarrow \phi(g_1) = \phi(g_2)$$

所以 $\bar{\phi}(g_1\ker(\phi)) = \bar{\phi}(g_2\ker(\phi))$，$\bar{\phi}$ 是良定义的。

**步骤 4**：证明 $\bar{\phi}$ 是群同态

$$\bar{\phi}((g_1\ker(\phi))(g_2\ker(\phi))) = \bar{\phi}((g_1g_2)\ker(\phi)) = \phi(g_1g_2) = \phi(g_1)\phi(g_2) = \bar{\phi}(g_1\ker(\phi))\bar{\phi}(g_2\ker(\phi))$$

**步骤 5**：证明 $\bar{\phi}$ 是单射

若 $\bar{\phi}(g_1\ker(\phi)) = \bar{\phi}(g_2\ker(\phi))$，则 $\phi(g_1) = \phi(g_2)$。

因此 $\phi(g_1^{-1}g_2) = e_H$，所以 $g_1^{-1}g_2 \in \ker(\phi)$，即 $g_1\ker(\phi) = g_2\ker(\phi)$。

**步骤 6**：证明 $\bar{\phi}$ 是满射

对任意 $h \in \text{im}(\phi)$，存在 $g \in G$ 使得 $\phi(g) = h$。

因此 $\bar{\phi}(g\ker(\phi)) = \phi(g) = h$。

**结论**：$\bar{\phi}$ 是群同构，因此 $G/\ker(\phi) \cong \text{im}(\phi)$。

**证毕**。

## 二、具体例子库

### 2.1 群的例子

**例子 2.1**：对称群 $S_3$

$S_3$ 是集合 $\{1, 2, 3\}$ 上所有置换构成的群。

**元素**：$S_3 = \{e, (12), (13), (23), (123), (132)\}$，共 6 个元素。

**群表**（部分）：

| $\circ$ | $e$ | $(12)$ | $(123)$ |
|---------|-----|--------|---------|
| $e$ | $e$ | $(12)$ | $(123)$ |
| $(12)$ | $(12)$ | $e$ | $(23)$ |
| $(123)$ | $(123)$ | $(13)$ | $(132)$ |

**子群**：
- $H_1 = \{e, (12)\}$ （阶为2）
- $H_2 = \{e, (13)\}$ （阶为2）
- $H_3 = \{e, (23)\}$ （阶为2）
- $A_3 = \{e, (123), (132)\}$ （阶为3，正规子群）

**验证拉格朗日定理**：$|S_3| = 6$，子群阶分别为 1, 2, 3, 6，都是 6 的因子。

---

**例子 2.2**：二面体群 $D_4$

$D_4$ 是正方形的对称群，有 8 个元素。

**生成元**：$r$（旋转90°），$s$（反射）

**关系**：$r^4 = e$，$s^2 = e$，$sr = r^{-1}s$

**元素**：$D_4 = \{e, r, r^2, r^3, s, sr, sr^2, sr^3\}$

**子群结构**：
- $\langle r \rangle = \{e, r, r^2, r^3\}$ （阶为4的循环子群）
- $\langle r^2 \rangle = \{e, r^2\}$ （阶为2，中心子群）
- $\langle s \rangle = \{e, s\}$ 
- $\langle sr \rangle = \{e, sr\}$

**共轭类**：
- $\{e\}$
- $\{r^2\}$
- $\{r, r^3\}$
- $\{s, sr^2\}$
- $\{sr, sr^3\}$

---

**例子 2.3**：四元数群 $Q_8$

$Q_8 = \{\pm 1, \pm i, \pm j, \pm k\}$，其中：
- $i^2 = j^2 = k^2 = -1$
- $ij = k$，$jk = i$，$ki = j$
- $ji = -k$，$kj = -i$，$ik = -j$

**性质**：
- 8 阶非交换群
- 每个子群都是正规子群
- 不是 $D_4$ 的同构（因为 $D_4$ 有5个二阶元素，$Q_8$ 只有1个）

---

### 2.2 子群的例子

**例子 2.4**：整数加群的子群

$(\mathbb{Z}, +)$ 的子群都是 $n\mathbb{Z} = \{nk : k \in \mathbb{Z}\}$ 的形式。

**验证**：
- $0 = n \cdot 0 \in n\mathbb{Z}$
- 若 $nk_1, nk_2 \in n\mathbb{Z}$，则 $nk_1 - nk_2 = n(k_1 - k_2) \in n\mathbb{Z}$

**商群**：$\mathbb{Z}/n\mathbb{Z} \cong \mathbb{Z}_n$

---

**例子 2.5**：特殊线性群

$SL_n(F) = \{A \in GL_n(F) : \det(A) = 1\}$ 是 $GL_n(F)$ 的正规子群。

**验证正规性**：对 $B \in GL_n(F)$，$A \in SL_n(F)$：
$$\det(BAB^{-1}) = \det(B)\det(A)\det(B)^{-1} = \det(A) = 1$$

所以 $BAB^{-1} \in SL_n(F)$。

**同构**：$GL_n(F)/SL_n(F) \cong F^*$（通过行列式同态）

---

### 2.3 西罗子群的例子

**例子 2.6**：$S_4$ 的西罗子群

$|S_4| = 24 = 2^3 \cdot 3$

**西罗 2-子群**（阶为8）：
- $n_2 \equiv 1 \pmod{2}$ 且 $n_2 \mid 3$，故 $n_2 = 1$ 或 $3$。
- 实际上 $n_2 = 3$，三个西罗 2-子群都同构于 $D_4$。

**西罗 3-子群**（阶为3）：
- $n_3 \equiv 1 \pmod{3}$ 且 $n_3 \mid 8$，故 $n_3 = 1$ 或 $4$。
- 实际上 $n_3 = 4$，由 3-循环生成，如 $\langle(123)\rangle$、$\langle(124)\rangle$ 等。

## 三、反例集锦

### 3.1 拉格朗日定理的逆命题不成立

**反例 3.1**：存在有限群 $G$ 和 $n \mid |G|$，但 $G$ 没有 $n$ 阶子群。

**具体例子**：$A_4$（12阶交错群）没有6阶子群。

**证明**：
- $|A_4| = 12$，6 整除 12。
- $A_4$ 的元素：单位元、8个3-循环、3个双对换。
- 假设存在6阶子群 $H$。
- 由于 $[A_4 : H] = 2$，$H \trianglelefteq A_4$。
- 3-循环在 $A_4$ 中两两共轭，若 $H$ 包含一个3-循环，则包含所有8个3-循环，但 $|H| = 6$，矛盾。
- 因此 $H$ 只能包含单位元和3个双对换，但这不是子群（不封闭）。

**结论**：$A_4$ 没有6阶子群。

---

### 3.2 非正规子群

**反例 3.2**：$H = \{e, (12)\} \leq S_3$ 不是正规子群。

**验证**：
$$(13)H(13)^{-1} = (13)\{e, (12)\}(13) = \{e, (23)\} \neq H$$

因此 $H$ 不是正规子群。

---

### 3.3 非交换群

**反例 3.3**：$S_3$ 是最小的非交换群。

**验证**：
$$(12)(13) = (132) \neq (123) = (13)(12)$$

---

### 3.4 非循环的交换群

**反例 3.4**：$\mathbb{Z}_2 \times \mathbb{Z}_2$（Klein四元群）是交换群但不是循环群。

**验证**：
- 所有元素的阶都是1或2，没有4阶元素，因此不是循环群。

## 四、计算实践

### 4.1 SageMath计算示例

```python
# SageMath群论计算示例

# 1. 定义对称群 S_4
S4 = SymmetricGroup(4)
print(f"S_4 的阶: {S4.order()}")  # 24

# 2. 定义元素并计算
s1 = S4('(1,2)')
s2 = S4('(2,3)')
print(f"(12)(23) = {s1 * s2}")  # (1,2,3)

# 3. 子群计算
H = S4.subgroup([S4('(1,2,3,4)'), S4('(1,3)')])
print(f"子群的阶: {H.order()}")  # 8 (D_4)

# 4. 共轭类
print("S_4 的共轭类:")
for conj_class in S4.conjugacy_classes():
    print(f"  代表元: {conj_class.representative()}, 大小: {len(conj_class)}")

# 5. 西罗子群
print(f"西罗 2-子群: {S4.sylow_subgroup(2)}")
print(f"西罗 3-子群: {S4.sylow_subgroup(3)}")

# 6. 正规子群
print("正规子群:")
for N in S4.normal_subgroups():
    print(f"  {N}, 阶: {N.order()}")

# 7. 商群
A4 = AlternatingGroup(4)
Q = S4.quotient(A4)
print(f"S_4/A_4 同构于: {Q}")  # C2
```

```gap
Print("西罗 2-子群: ", P2, "\n");
Print("西罗 3-子群: ", P3, "\n");

# 6. 正规子群
Print("正规子群:\n");
for N in NormalSubgroups(S4) do
    Print("  ", N, ", 阶: ", Size(N), "\n");
od;

# 7. 商群
A4 := AlternatingGroup(4);
Q := S4 / A4;
Print("S_4/A_4 同构于: ", Q, "\n");  # C2
```

### 4.3 Python计算示例

```python
import numpy as np
from itertools import permutations

# 置换群计算

def compose(p1, p2):
    """置换复合: p1 ∘ p2"""
    return tuple(p1[i-1] for i in p2)

def inverse(p):
    """置换的逆"""
    inv = [0] * len(p)
    for i, j in enumerate(p):
        inv[j-1] = i + 1
    return tuple(inv)

def order(p):
    """置换的阶"""
    n = len(p)
    visited = [False] * n
    result = 1
    for i in range(n):
        if not visited[i]:
            cycle_len = 0
            j = i
            while not visited[j]:
                visited[j] = True
                j = p[j] - 1
                cycle_len += 1
            if cycle_len > 0:
                result = lcm(result, cycle_len)
    return result

def lcm(a, b):
    from math import gcd
    return a * b // gcd(a, b)

# S_3 的元素
S3 = list(permutations([1, 2, 3]))
print(f"S_3 有 {len(S3)} 个元素")

# 计算置换阶
for p in S3:
    print(f"{p} 的阶: {order(p)}")
```

### 习题参考答案（部分）

**习题 5.1.3 解答**：

$\mathbb{Z}_{12}$ 的子群由 12 的正因子决定：
- $\langle 0 \rangle = \{0\}$（阶1）
- $\langle 6 \rangle = \{0, 6\}$（阶2）
- $\langle 4 \rangle = \{0, 4, 8\}$（阶3）
- $\langle 3 \rangle = \{0, 3, 6, 9\}$（阶4）
- $\langle 2 \rangle = \{0, 2, 4, 6, 8, 10\}$（阶6）
- $\langle 1 \rangle = \mathbb{Z}_{12}$（阶12）

**习题 5.2.2 解答**：

设 $[G:H] = 2$，则 $G$ 被 $H$ 分成两个左陪集：$H$ 和 $G \setminus H$。

对任意 $g \in G$：
- 若 $g \in H$，则 $gH = H = Hg$。
- 若 $g \notin H$，则 $gH = G \setminus H = Hg$（因为右陪集也是两个：$H$ 和 $G \setminus H$）。

因此对所有 $g \in G$，$gH = Hg$，即 $H \trianglelefteq G$。
