# 矩阵论习题

矩阵论是线性代数的核心组成部分，涵盖特征值问题、矩阵分解、范数理论与矩阵函数。本节通过精选习题展示 Jordan 标准形、SVD、正定矩阵与谱理论的经典技巧。

## 1. 特征值与 Jordan 标准形

### 1.1 特征值的代数与几何重数

**习题 1.1**. 设 $A \in M_n(\mathbb{C})$，$\lambda$ 为特征值。定义代数重数为特征多项式根 $\lambda$ 的重数，几何重数为 $\dim \ker(A - \lambda I)$。证明几何重数 $\leq$ 代数重数。

**证明**. 设 $k = \dim \ker(A - \lambda I)$。取特征空间的基 $v_1, \dots, v_k$，扩充为 $\mathbb{C}^n$ 的基。在此基下 $A$ 的矩阵为：

$$A' = \begin{pmatrix} \lambda I_k & B \\ 0 & C \end{pmatrix}.$$

故 $\det(A' - tI) = (\lambda - t)^k \det(C - tI)$，$\lambda$ 的代数重数至少为 $k$。$\square$

### 1.2 Jordan 标准形的存在性

**习题 1.2**. 证明任意复方阵 $A$ 都相似于唯一的 Jordan 标准形（不计 Jordan 块顺序）。

**证明概要**. 对 $V = \mathbb{C}^n$ 考虑 $A$ 的广义特征空间分解：

$$V = \bigoplus_{\lambda} G_\lambda, \quad G_\lambda = \ker((A - \lambda I)^n).$$

在每个 $G_\lambda$ 上，$N = A - \lambda I$ 为幂零算子。由循环子空间定理，$G_\lambda$ 可分解为 $N$-循环子空间的直和，每个对应一个 Jordan 块。唯一性由初等因子或不变因子理论保证。$\square$

## 2. 奇异值分解

### 2.1 SVD 的存在性与唯一性

**习题 2.1**. 证明任意 $A \in M_{m \times n}(\mathbb{R})$ 存在奇异值分解 $A = U\Sigma V^T$，其中 $U, V$ 为正交矩阵，$\Sigma$ 为对角元非负的对角矩阵。

**证明**. 考虑对称半正定矩阵 $A^T A \in M_n(\mathbb{R})$。由谱定理，存在正交 $V$ 使 $A^T A = V D V^T$，$D = \operatorname{diag}(\lambda_1, \dots, \lambda_n)$，$\lambda_i \geq 0$。令 $\sigma_i = \sqrt{\lambda_i}$，$\Sigma = \operatorname{diag}(\sigma_1, \dots, \sigma_r, 0, \dots)$。

对 $i = 1, \dots, r$，令 $u_i = \frac{1}{\sigma_i} A v_i$。验证 $\{u_i\}$ 标准正交：

$$u_i^T u_j = \frac{1}{\sigma_i \sigma_j} v_i^T A^T A v_j = \frac{\sigma_j^2}{\sigma_i \sigma_j} v_i^T v_j = \delta_{ij}.$$

扩充为 $\mathbb{R}^m$ 的标准正交基得 $U$。则 $AV = U\Sigma$，即 $A = U\Sigma V^T$。$\square$

### 2.2 低秩逼近

**习题 2.2（Eckart-Young-Mirsky）**. 设 $A = U\Sigma V^T$ 为 SVD，$\sigma_1 \geq \dots \geq \sigma_r > 0$。则对任意秩 $k < r$ 矩阵 $B$：

$$\|A - B\|_F \geq \|A - A_k\|_F = \sqrt{\sigma_{k+1}^2 + \dots + \sigma_r^2},$$

其中 $A_k = U \operatorname{diag}(\sigma_1, \dots, \sigma_k, 0, \dots) V^T$，$\|\cdot\|_F$ 为 Frobenius 范数。

**证明**. 设 $B$ 秩为 $k$，则 $\dim \ker(B) \geq n - k$。对 $x \in \ker(B)$，$\|(A-B)x\|_2 = \|Ax\|_2$。由极小极大原理：

$$\sigma_{k+1}(A) = \min_{\dim W = n-k} \max_{x \in W, \|x\|=1} \|Ax\|_2 \leq \max_{x \in \ker(B), \|x\|=1} \|Ax\|_2.$$

故 $\|A-B\|_2 \geq \sigma_{k+1}$。Frobenius 范数情形类似，利用 $\|A\|_F^2 = \sum \sigma_i^2$。$\square$

## 3. 正定矩阵

### 3.1 正定性判据

**习题 3.1**. 证明对 Hermitian 矩阵 $A \in M_n(\mathbb{C})$，以下等价：
1. $A$ 正定（$x^*Ax > 0$ 对所有非零 $x$）；
2. $A$ 的所有特征值 $> 0$；
3. 存在可逆矩阵 $B$ 使 $A = B^*B$；
4. $A$ 的所有顺序主子式 $> 0$（Sylvester 判据）。

**证明**. $(1) \Leftrightarrow (2)$：由谱定理 $A = UDU^*$，$x^*Ax = \sum \lambda_i |y_i|^2$（$y = U^*x$）。

$(1) \Rightarrow (3)$：取 $B = U\sqrt{D}U^*$（正定平方根）。

$(3) \Rightarrow (1)$：$x^*Ax = (Bx)^*(Bx) = \|Bx\|^2 > 0$（$B$ 可逆）。

$(1) \Leftrightarrow (4)$：由 Cholesky 分解或归纳法。$\square$

### 3.2 行列式不等式

**习题 3.2（Hadamard 不等式）**. 设 $A$ 为正定 Hermitian 矩阵，列向量为 $a_1, \dots, a_n$。则：

$$\det A \leq \prod_{i=1}^n a_{ii},$$

等号成立当且仅当 $A$ 对角。

**证明**. 对 $n$ 归纳。写 $A = \begin{pmatrix} a_{11} & b^* \\ b & C \end{pmatrix}$。由 Schur 补：

$$\det A = a_{11} \det(C - a_{11}^{-1}bb^*).$$

$C - a_{11}^{-1}bb^*$ 正定（合同于 $A$），由归纳假设：

$$\det(C - a_{11}^{-1}bb^*) \leq \prod_{i=2}^n (a_{ii} - a_{11}^{-1}|b_i|^2) \leq \prod_{i=2}^n a_{ii}.$$

$\square$

## 4. 例子

### 4.1 例子：具体 SVD 计算

**习题 4.1**. 求 $A = \begin{pmatrix} 2 & 1 \\ 1 & 2 \end{pmatrix}$ 的 SVD。

**解答**. $A^T A = \begin{pmatrix} 5 & 4 \\ 4 & 5 \end{pmatrix}$。特征值 $\lambda_1 = 9, \lambda_2 = 1$，故奇异值 $\sigma_1 = 3, \sigma_2 = 1$。

特征向量：$v_1 = \frac{1}{\sqrt{2}}(1, 1)^T$，$v_2 = \frac{1}{\sqrt{2}}(1, -1)^T$。

$u_1 = \frac{1}{\sigma_1}Av_1 = \frac{1}{\sqrt{2}}(1, 1)^T$，$u_2 = \frac{1}{\sigma_2}Av_2 = \frac{1}{\sqrt{2}}(1, -1)^T$。

故：

$$A = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix} \begin{pmatrix} 3 & 0 \\ 0 & 1 \end{pmatrix} \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}^T.$$

（因 $A$ 对称正定，SVD 与谱分解一致。）

### 4.2 例子：Cayley-Hamilton 的应用

**习题 4.2**. 利用 Cayley-Hamilton 定理计算 $A^{100}$，其中 $A = \begin{pmatrix} 1 & 2 \\ 3 & 4 \end{pmatrix}$。

**解答**. 特征多项式 $p(\lambda) = \lambda^2 - 5\lambda - 2$。由 Cayley-Hamilton，$A^2 = 5A + 2I$。递推：$A^n = 5A^{n-1} + 2A^{n-2}$。设 $A^n = a_n A + b_n I$，则 $a_n = 5a_{n-1} + 2a_{n-2}$，$a_0 = 0, a_1 = 1$。利用特征根 $\frac{5 \pm \sqrt{33}}{2}$ 得显式公式。$\square$
