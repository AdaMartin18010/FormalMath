## 一、核心定理完整证明

### 1.1 理想对应定理完整证明

**定理 1.1** (环的理想对应定理 / Ideal Correspondence Theorem)

设 $R$ 是环，$I$ 是 $R$ 的理想，$\pi: R \to R/I$ 是自然同态。则在 $R$ 的包含 $I$ 的理想与 $R/I$ 的理想之间存在一一对应：
$$J \mapsto J/I = \{j + I : j \in J\}$$

并且：

1. 这个对应保持包含关系：$J_1 \subseteq J_2$ 当且仅当 $J_1/I \subseteq J_2/I$
2. 正规子群对应正规子群
3. $(R/I)/(J/I) \cong R/J$

**完整证明**：

**步骤 1**：建立映射

定义 $\Phi: \{J : I \subseteq J \subseteq R, J \text{ 是理想}\} \to \{K : K \subseteq R/I, K \text{ 是理想}\}$

为 $\Phi(J) = J/I = \{j + I : j \in J\}$。

**步骤 2**：证明 $\Phi(J)$ 是 $R/I$ 的理想

设 $j_1 + I, j_2 + I \in J/I$，则：

- $(j_1 + I) - (j_2 + I) = (j_1 - j_2) + I \in J/I$（因为 $j_1 - j_2 \in J$）

设 $r + I \in R/I$，$j + I \in J/I$，则：

- $(r + I)(j + I) = rj + I \in J/I$（因为 $rj \in J$，$J$ 是理想）
- $(j + I)(r + I) = jr + I \in J/I$（因为 $jr \in J$，$J$ 是理想）

因此 $J/I$ 是 $R/I$ 的理想。

**步骤 3**：建立逆映射

定义 $\Psi: \{K : K \subseteq R/I, K \text{ 是理想}\} \to \{J : I \subseteq J \subseteq R, J \text{ 是理想}\}$

为 $\Psi(K) = \pi^{-1}(K) = \{r \in R : r + I \in K\}$。

**步骤 4**：证明 $\Psi(K)$ 是 $R$ 的理想且包含 $I$

设 $r_1, r_2 \in \Psi(K)$，则 $r_1 + I, r_2 + I \in K$，所以 $(r_1 + I) - (r_2 + I) = (r_1 - r_2) + I \in K$，故 $r_1 - r_2 \in \Psi(K)$。

设 $r \in R$，$j \in \Psi(K)$，则 $j + I \in K$，所以 $(r + I)(j + I) = rj + I \in K$，故 $rj \in \Psi(K)$。同理 $jr \in \Psi(K)$。

因此 $\Psi(K)$ 是 $R$ 的理想。

对任意 $i \in I$，$i + I = 0 + I \in K$（因为 $K$ 是理想，包含零元），所以 $I \subseteq \Psi(K)$。

**步骤 5**：证明 $\Phi$ 和 $\Psi$ 互为逆映射

- $\Psi(\Phi(J)) = \Psi(J/I) = \{r \in R : r + I \in J/I\} = J$（因为 $r + I \in J/I$ 当且仅当 $r \in J$）

- $\Phi(\Psi(K)) = \Phi(\pi^{-1}(K)) = \{r + I : r \in \pi^{-1}(K)\} = K$

**步骤 6**：证明保持包含关系

设 $J_1 \subseteq J_2$，则 $J_1/I = \{j + I : j \in J_1\} \subseteq \{j + I : j \in J_2\} = J_2/I$。

反之，设 $J_1/I \subseteq J_2/I$，对 $j_1 \in J_1$，$j_1 + I \in J_1/I \subseteq J_2/I$，所以存在 $j_2 \in J_2$ 使得 $j_1 + I = j_2 + I$，即 $j_1 - j_2 \in I \subseteq J_2$，因此 $j_1 = j_2 + (j_1 - j_2) \in J_2$。

**步骤 7**：证明 $(R/I)/(J/I) \cong R/J$

定义 $\phi: R/I \to R/J$ 为 $\phi(r + I) = r + J$。

- **良定义**：若 $r_1 + I = r_2 + I$，则 $r_1 - r_2 \in I \subseteq J$，所以 $r_1 + J = r_2 + J$。
- **同态**：$\phi((r_1 + I) + (r_2 + I)) = \phi((r_1 + r_2) + I) = (r_1 + r_2) + J = (r_1 + J) + (r_2 + J) = \phi(r_1 + I) + \phi(r_2 + I)$。乘法同理。
- **满射**：显然。
- **核**：$\ker(\phi) = \{r + I : r + J = J\} = \{r + I : r \in J\} = J/I$。

由第一同构定理，$(R/I)/(J/I) \cong R/J$。

**证毕**。

### 1.2 唯一分解定理完整证明

**定理 1.2** (主理想整环的唯一分解定理 / Unique Factorization Theorem)

设 $R$ 是主理想整环（PID），则 $R$ 中每个非零非单位元素都可以唯一地（在相伴意义下，不考虑因子顺序）分解为不可约元素的乘积。

**定义回顾**：

- **不可约元**：$p \in R$ 非零非单位，若 $p = ab$ 则 $a$ 或 $b$ 是单位。
- **相伴**：$a \sim b$ 若存在单位 $u$ 使得 $a = ub$。

**完整证明**：

**部分 1：分解的存在性**

设 $a \in R$ 非零非单位。若 $a$ 不可约，则证毕。

否则，$a = a_1 b_1$，其中 $a_1, b_1$ 都是非单位。若 $a_1, b_1$ 都不可约，则证毕。

否则，继续分解。假设这个过程无限进行，则得到一个严格升链：
$$(a) \subsetneq (a_1) \subsetneq (a_2) \subsetneq \cdots$$

设 $I = \bigcup_{i=1}^{\infty} (a_i)$。$I$ 是理想（因为若 $x, y \in I$，则存在 $n$ 使得 $x, y \in (a_n)$，所以 $x - y \in (a_n) \subseteq I$；对任意 $r \in R$，$rx \in (a_n) \subseteq I$）。

由于 $R$ 是 PID，$I = (d)$ 对某个 $d \in R$。

则 $d \in I$，所以存在 $n$ 使得 $d \in (a_n)$，因此 $(d) \subseteq (a_n) \subseteq (a_{n+1}) \subseteq \cdots \subseteq I = (d)$。

这意味着 $(a_n) = (a_{n+1}) = \cdots$，与严格包含矛盾。

因此分解过程必在有限步终止，得到 $a = p_1 p_2 \cdots p_n$，其中每个 $p_i$ 都不可约。

**部分 2：分解的唯一性**

设 $a = p_1 p_2 \cdots p_n = q_1 q_2 \cdots q_m$，其中 $p_i, q_j$ 都不可约。

**引理**：在主理想整环中，不可约元是素元。

*证明*：设 $p$ 不可约，$p \mid ab$。若 $p \nmid a$，则 $(p, a) = (1)$（因为 $R$ 是 PID，$(p, a) = (d)$，$d \mid p$，所以 $d \sim p$ 或 $d \sim 1$；若 $d \sim p$ 则 $p \mid a$，矛盾，故 $d \sim 1$）。

因此 $1 = px + ay$，所以 $b = pbx + aby$，故 $p \mid b$。所以 $p$ 是素元。

回到唯一性证明（对 $n$ 归纳）：

**基例**：$n = 1$，$a = p_1 = q_1 \cdots q_m$。若 $m > 1$，则 $p_1 = q_1(q_2 \cdots q_m)$，由于 $p_1$ 不可约，$q_1$ 或 $q_2 \cdots q_m$ 是单位，矛盾。所以 $m = 1$。

**归纳步骤**：设对少于 $n$ 个因子成立。由 $p_1 \mid q_1 \cdots q_m$，$p_1$ 是素元，所以存在 $j$ 使得 $p_1 \mid q_j$。

重排使得 $j = 1$。则 $q_1 = p_1 u$，$u$ 是单位（因为 $q_1$ 不可约）。

所以 $p_1 p_2 \cdots p_n = p_1 u q_2 \cdots q_m$，消去 $p_1$ 得：
$$p_2 \cdots p_n = (u q_2) q_3 \cdots q_m$$

由归纳假设，$n - 1 = m - 1$，所以 $n = m$，且重排后 $p_i \sim q_i$（$i \geq 2$）。

因此唯一性成立。

**证毕**。

### 1.3 同态基本定理完整证明

**定理 1.3** (环的第一同构定理 / First Isomorphism Theorem for Rings)

设 $\phi: R \to S$ 是环同态，则：
$$R/\ker(\phi) \cong \text{im}(\phi)$$

**完整证明**：

**步骤 1**：证明 $\ker(\phi)$ 是 $R$ 的理想

设 $a, b \in \ker(\phi)$，则 $\phi(a) = \phi(b) = 0$，所以 $\phi(a - b) = \phi(a) - \phi(b) = 0$，故 $a - b \in \ker(\phi)$。

设 $r \in R$，$a \in \ker(\phi)$，则：

- $\phi(ra) = \phi(r)\phi(a) = \phi(r) \cdot 0 = 0$，所以 $ra \in \ker(\phi)$
- $\phi(ar) = \phi(a)\phi(r) = 0 \cdot \phi(r) = 0$，所以 $ar \in \ker(\phi)$

因此 $\ker(\phi)$ 是 $R$ 的理想。

**步骤 2**：定义映射 $\bar{\phi}: R/\ker(\phi) \to \text{im}(\phi)$

定义 $\bar{\phi}(r + \ker(\phi)) = \phi(r)$。

**步骤 3**：证明 $\bar{\phi}$ 是良定义的

若 $r_1 + \ker(\phi) = r_2 + \ker(\phi)$，则 $r_1 - r_2 \in \ker(\phi)$。

因此 $\phi(r_1 - r_2) = 0$，即 $\phi(r_1) = \phi(r_2)$。

所以 $\bar{\phi}(r_1 + \ker(\phi)) = \bar{\phi}(r_2 + \ker(\phi))$，$\bar{\phi}$ 是良定义的。

**步骤 4**：证明 $\bar{\phi}$ 是环同态

- **加法**：$\bar{\phi}((r_1 + \ker(\phi)) + (r_2 + \ker(\phi))) = \bar{\phi}((r_1 + r_2) + \ker(\phi)) = \phi(r_1 + r_2) = \phi(r_1) + \phi(r_2) = \bar{\phi}(r_1 + \ker(\phi)) + \bar{\phi}(r_2 + \ker(\phi))$

- **乘法**：$\bar{\phi}((r_1 + \ker(\phi))(r_2 + \ker(\phi))) = \bar{\phi}(r_1 r_2 + \ker(\phi)) = \phi(r_1 r_2) = \phi(r_1)\phi(r_2) = \bar{\phi}(r_1 + \ker(\phi))\bar{\phi}(r_2 + \ker(\phi))$

**步骤 5**：证明 $\bar{\phi}$ 是单射

若 $\bar{\phi}(r_1 + \ker(\phi)) = \bar{\phi}(r_2 + \ker(\phi))$，则 $\phi(r_1) = \phi(r_2)$。

因此 $\phi(r_1 - r_2) = 0$，所以 $r_1 - r_2 \in \ker(\phi)$，即 $r_1 + \ker(\phi) = r_2 + \ker(\phi)$。

**步骤 6**：证明 $\bar{\phi}$ 是满射

对任意 $s \in \text{im}(\phi)$，存在 $r \in R$ 使得 $\phi(r) = s$。

因此 $\bar{\phi}(r + \ker(\phi)) = \phi(r) = s$。

**结论**：$\bar{\phi}$ 是环同构，因此 $R/\ker(\phi) \cong \text{im}(\phi)$。

**证毕**。

## 二、具体例子库

### 2.1 环的例子

**例子 2.1**：整数环 $\mathbb{Z}$

**性质**：

- 交换环，单位元为 1
- 整环（无零因子）
- 欧几里得整环（取 $d(n) = |n|$）
- 主理想整环（每个理想都是主理想 $(n) = n\mathbb{Z}$）

**理想**：$n\mathbb{Z} = \{nk : k \in \mathbb{Z}\}$

**商环**：$\mathbb{Z}/n\mathbb{Z} \cong \mathbb{Z}_n$

**不可约元**：素数 $p$（及其相反数 $-p$）

**单位**：$\pm 1$

---

**例子 2.2**：多项式环 $\mathbb{Q}[x]$

**性质**：

- 无限整环
- 欧几里得整环（取 $d(f) = \deg(f)$）
- 主理想整环

**理想**：$(f(x)) = \{f(x)g(x) : g(x) \in \mathbb{Q}[x]\}$

**商环**：$\mathbb{Q}[x]/(f(x))$ 是域当且仅当 $f(x)$ 不可约

**不可约多项式**：

- $x^2 + 1$（在 $\mathbb{Q}$ 上不可约）
- $x^2 - 2$（在 $\mathbb{Q}$ 上不可约）
- $x^3 - 2$（在 $\mathbb{Q}$ 上不可约）

**可约多项式**：

- $x^2 - 1 = (x-1)(x+1)$

---

**例子 2.3**：高斯整数环 $\mathbb{Z}[i]$

**定义**：$\mathbb{Z}[i] = \{a + bi : a, b \in \mathbb{Z}\}$

**性质**：

- 交换整环
- 欧几里得整环（取 $d(a + bi) = a^2 + b^2$）
- 主理想整环

**范数**：$N(a + bi) = a^2 + b^2 = |a + bi|^2$

**单位**：$\pm 1, \pm i$（因为 $N(u) = 1$ 当且仅当 $u \in \{\pm 1, \pm i\}$）

**不可约元**：

- $1 + i$（$N(1+i) = 2$，是素数）
- 有理素数 $p \equiv 3 \pmod{4}$ 在 $\mathbb{Z}[i]$ 中仍不可约
- 若 $p \equiv 1 \pmod{4}$，则 $p = (a + bi)(a - bi)$ 可分解

**分解示例**：

- $5 = (2 + i)(2 - i)$
- $2 = (1 + i)^2 \cdot (-i)$

---

**例子 2.4**：矩阵环 $M_2(\mathbb{R})$

**定义**：$2 \times 2$ 实矩阵构成的环

**性质**：

- 非交换环
- 有单位元（单位矩阵）
- 有零因子（如 $\begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix} \begin{pmatrix} 0 & 0 \\ 0 & 1 \end{pmatrix} = 0$）

**理想**：

- $M_2(\mathbb{R})$ 是单环（没有非平凡双边理想）
- 有左理想：如 $\left\{\begin{pmatrix} a & 0 \\ b & 0 \end{pmatrix} : a, b \in \mathbb{R}\right\}$
- 有右理想：如 $\left\{\begin{pmatrix} a & b \\ 0 & 0 \end{pmatrix} : a, b \in \mathbb{R}\right\}$

---

### 2.2 理想的例子

**例子 2.5**：$\mathbb{Z}[x]$ 中的理想

$\mathbb{Z}[x]$ 不是主理想整环。

**非主理想的例子**：$I = (2, x) = \{2f(x) + xg(x) : f, g \in \mathbb{Z}[x]\}$

*证明 $I$ 不是主理想*：

假设 $I = (h(x))$，则 $2 = h(x)f(x)$，$x = h(x)g(x)$。

由 $2 = h(x)f(x)$，$h(x)$ 必为常数 $c \in \mathbb{Z}$。

由 $x = cg(x)$，$c$ 必须整除 $x$ 的所有系数，所以 $c = \pm 1$。

但 $(1) = \mathbb{Z}[x]$，而 $I \neq \mathbb{Z}[x]$（因为常数项是偶数）。

矛盾！因此 $I$ 不是主理想。

---

### 2.3 商环的例子

**例子 2.6**：$\mathbb{R}[x]/(x^2 + 1) \cong \mathbb{C}$

**同构映射**：$a + bx + (x^2 + 1) \mapsto a + bi$

**验证**：

- $(x + (x^2+1))^2 = x^2 + (x^2+1) = -1 + (x^2+1)$
- 所以 $x + (x^2+1)$ 在商环中充当 $i$ 的角色

### 3.2 非唯一分解整环

**反例 3.2**：$\mathbb{Z}[\sqrt{-5}]$ 不是唯一分解整环。

**具体反例**：
$$6 = 2 \cdot 3 = (1 + \sqrt{-5})(1 - \sqrt{-5})$$

**验证**：

- $N(2) = 4$，$N(3) = 9$，$N(1 \pm \sqrt{-5}) = 6$
- $2$ 不可约：若 $2 = \alpha\beta$，则 $4 = N(\alpha)N(\beta)$。若 $N(\alpha) = 2$，设 $\alpha = a + b\sqrt{-5}$，则 $a^2 + 5b^2 = 2$ 无整数解。
- $3$ 不可约：若 $3 = \alpha\beta$，则 $9 = N(\alpha)N(\beta)$。若 $N(\alpha) = 3$，则 $a^2 + 5b^2 = 3$ 无整数解。
- $1 \pm \sqrt{-5}$ 不可约：若 $1 + \sqrt{-5} = \alpha\beta$，则 $6 = N(\alpha)N(\beta)$。若 $N(\alpha) \in \{2, 3\}$，无整数解。

因此6有两种不可约分解，不唯一。

## 四、计算实践

### 4.1 SageMath计算示例

```python
# SageMath环论计算示例

# 1. 定义多项式环
R.<x> = PolynomialRing(QQ)
print(f"多项式环: {R}")

# 2. 多项式运算
f = x^3 - 2*x^2 + x - 2
g = x^2 - 1
print(f"f = {f}")
print(f"g = {g}")
print(f"f + g = {f + g}")
print(f"f * g = {f * g}")
print(f"f 除以 g 的商和余式: {f.quo_rem(g)}")

# 3. 因式分解
print(f"f 的因式分解: {f.factor()}")
print(f"x^2 + 1 在 QQ 上是否可约: {not (x^2 + 1).is_irreducible()}")

# 4. 欧几里得算法
f1 = x^4 - 1
f2 = x^3 - 1
gcd_fg = f1.gcd(f2)
print(f"gcd(x^4-1, x^3-1) = {gcd_fg}")

# 5. 商环
S = R.quotient(x^2 + 1, 'i')
i = S.gen()
print(f"在 QQ[i] 中, i^2 = {i^2}")
print(f"(1 + i)^2 = {(1 + i)^2}")

# 6. 高斯整数
Zi = GaussianIntegers()
a = Zi(2 + 3*I)
b = Zi(1 + I)
print(f"高斯整数 2+3i 的范数: {a.norm()}")
print(f"(2+3i) / (1+i) = {a / b}")
print(f"2+3i 的因式分解: {a.factor()}")

# 7. 理想运算
I = R.ideal(x^2 - 2)
J = R.ideal(x - 2)
print(f"理想 I = {I}")
print(f"I + J = {I + J}")
print(f"I ∩ J = {I.intersection(J)}")
```

x = symbols('x')

g = x**2 - 1

# 3. 欧几里得算法
f1 = x**4 - 1
f2 = x**3 - 1
gcd_val = gcd(f1, f2)
lcm_val = lcm(f1, f2)
print(f"gcd(x^4-1, x^3-1) = {gcd_val}")
print(f"lcm(x^4-1, x^3-1) = {lcm_val}")

# 4. 扩展欧几里得算法
f = x**3 - x + 1
g = x**2 + x + 1
gcd_val, s, t = gcdex(f, g)
print(f"gcd = {gcd_val}")
print(f"s = {s}")
print(f"t = {t}")
print(f"验证: s*f + t*g = {expand(s*f + t*g)}")

# 5. 矩阵环计算
A = Matrix([[1, 2], [3, 4]])
B = Matrix([[0, 1], [1, 0]])
print(f"A = {A}")
print(f"B = {B}")
print(f"AB = {A * B}")

print(f"AB = BA? {A*B == B*A}")  # 非交换

# 幂零矩阵
N = Matrix([[0, 1], [0, 0]])

### 习题参考答案（部分）

**习题 5.1.1 解答**：

在 $\mathbb{Z}_{12}$ 中：

- $\overline{5} + \overline{9} = \overline{14} = \overline{2}$
- $\overline{7} \cdot \overline{8} = \overline{56} = \overline{8}$
- $\overline{5}$ 的逆元：找 $x$ 使得 $5x \equiv 1 \pmod{12}$。$5 \cdot 5 = 25 \equiv 1 \pmod{12}$，所以逆元是 $\overline{5}$。

**习题 5.2.2 解答**：

设 $R$ 是欧几里得整环，度函数为 $d$。设 $I$ 是 $R$ 的理想。

若 $I = \{0\}$，则 $I = (0)$ 是主理想。

若 $I \neq \{0\}$，取 $b \in I \setminus \{0\}$ 使得 $d(b)$ 最小。

对任意 $a \in I$，由欧几里得算法，存在 $q, r \in R$ 使得：
$$a = bq + r$$
其中 $r = 0$ 或 $d(r) < d(b)$。

由于 $r = a - bq \in I$，若 $r \neq 0$ 则 $d(r) < d(b)$ 与 $d(b)$ 最小矛盾。

因此 $r = 0$，即 $a = bq$，所以 $I = (b)$ 是主理想。
