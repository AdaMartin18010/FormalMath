# 理想类群 - 深度版

> **"理想类群是代数数域最深刻的算术不变量之一，它连接了代数、分析和几何。"**

---

## 1. 分式理想

### 1.1 定义

**定义 1.1**（分式理想）
设 $K$ 是数域，$\mathcal{O}_K$ 是其整数环。$K$ 的**分式理想**是 $\mathcal{O}_K$ 的子 $\mathcal{O}_K$-模 $\mathfrak{a}$，满足存在非零 $c \in \mathcal{O}_K$ 使得 $c\mathfrak{a} \subseteq \mathcal{O}_K$。

等价地，分式理想可写为 $\mathfrak{a} = \frac{1}{c}\mathfrak{b}$，其中 $\mathfrak{b}$ 是 $\mathcal{O}_K$ 的整理想。

**定义 1.2**（主分式理想）
分式理想 $\mathfrak{a}$ 称为**主的**，如果 $\mathfrak{a} = (\alpha) = \alpha\mathcal{O}_K$ 对某个 $\alpha \in K^\times$。

**定理 1.1**
数域 $K$ 的所有非零分式理想在乘法下构成群 $I_K$，单位元为 $\mathcal{O}_K$，逆元满足 $\mathfrak{a}\mathfrak{a}^{-1} = \mathcal{O}_K$。

## 2. 理想类群

### 2.1 定义

**定义 2.1**（理想类群）
数域 $K$ 的**理想类群**定义为：
$$Cl(K) = I_K / P_K$$

其中 $I_K$ 是分式理想群，$P_K$ 是主分式理想子群。

**类数**：$h_K = |Cl(K)|$。

**定理 2.1**
以下等价：
1. $h_K = 1$
2. $\mathcal{O}_K$ 是主理想整环（PID）
3. $\mathcal{O}_K$ 是唯一分解整环（UFD）

### 2.2 类数的有限性

**定理 2.2**（Minkowski界）
设 $K$ 是数域，$[K:\mathbb{Q}] = n$，$r_1$ 个实嵌入，$2r_2$ 个复嵌入。则每个理想类包含整理想 $\mathfrak{a}$ 满足：
$$N(\mathfrak{a}) \leq M_K = \frac{n!}{n^n}\left(\frac{4}{\pi}\right)^{r_2}\sqrt{|d_K|}$$

其中 $d_K$ 是判别式，$N(\mathfrak{a}) = |\mathcal{O}_K/\mathfrak{a}|$ 是理想的范数。

**推论 2.3**
$h_K < \infty$（类数有限）。

**证明**：
由Minkowski界，每个理想类包含范数 $\leq M_K$ 的整理想。而对任意正整数 $m$，范数为 $m$ 的整理想只有有限多个（整理想的范数对应于格的指数）。

**Minkowski界证明概要**：

设 $\mathfrak{a}$ 是整理想。$\mathcal{O}_K$ 可嵌入到 $\mathbb{R}^n$ 中作为格，其共体积为 $2^{-r_2}\sqrt{|d_K|}N(\mathfrak{a})$。

考虑凸对称集：
$$S = \{(y_1,\ldots,y_{r_1},z_1,\ldots,z_{r_2}) \in \mathbb{R}^{r_1} \times \mathbb{C}^{r_2} : \sum|y_i| + 2\sum|z_j| \leq t\}$$

其体积为 $2^{r_1}\left(\frac{\pi}{2}\right)^{r_2}\frac{t^n}{n!}$。

选取 $t$ 使 $\text{vol}(S) > 2^n \cdot \text{covol}(\mathfrak{a})$，由Minkowski凸体定理，$S$ 包含非零格点 $\alpha \in \mathfrak{a}$。

计算 $N(\alpha\mathcal{O}_K) = |N_{K/\mathbb{Q}}(\alpha)| \leq \frac{t^n}{n^n}$，取最优 $t$ 得到界。 $\square$

## 3. 例子

**例 1**：虚二次域 $\mathbb{Q}(\sqrt{-1})$

$\mathcal{O}_K = \mathbb{Z}[i]$ 是欧几里得整环，故 $h_K = 1$。

**例 2**：虚二次域 $\mathbb{Q}(\sqrt{-5})$

$\mathcal{O}_K = \mathbb{Z}[\sqrt{-5}]$。理想 $(2, 1+\sqrt{-5})$ 非主理想：

假设 $(2, 1+\sqrt{-5}) = (a+b\sqrt{-5})$，则 $2 = (a+b\sqrt{-5})(c+d\sqrt{-5})$。

取范数：$4 = (a^2+5b^2)(c^2+5d^2)$。

若 $a^2+5b^2 = 2$，无整数解。故 $(2, 1+\sqrt{-5})$ 非主理想，$h_K > 1$（实际上 $h_K = 2$）。 $\square$

**例 3**：实二次域 $\mathbb{Q}(\sqrt{10})$

$\mathcal{O}_K = \mathbb{Z}[\sqrt{10}]$，$d_K = 40$。Minkowski界 $M_K \approx 3.16$。

检查素理想 $(2)$ 和 $(3)$ 的分解，可证 $h_K = 2$。 $\square$

## 4. 解析类数公式

**定理 4.1**（解析类数公式）
设 $K$ 是数域，$\zeta_K(s)$ 是Dedekind zeta函数。则：
$$\lim_{s \to 1} (s-1)\zeta_K(s) = \frac{2^{r_1}(2\pi)^{r_2}h_K R_K}{w_K\sqrt{|d_K|}}$$

其中：
- $r_1, r_2$：实嵌入数和复嵌入对数
- $h_K$：类数
- $R_K$：调节子（单位群的基本域体积）
- $w_K$：单位根数
- $d_K$：判别式

**证明概要**：

**步骤1**：$\zeta_K(s) = \sum_{\mathfrak{a}} N(\mathfrak{a})^{-s} = \sum_{m=1}^\infty a_m m^{-s}$，其中 $a_m = \#\{\mathfrak{a} : N(\mathfrak{a}) = m\}$。

**步骤2**：利用理想类群的分解：
$$\zeta_K(s) = \sum_{[\mathfrak{a}] \in Cl(K)} \zeta_{[\mathfrak{a}]}(s)$$

其中 $\zeta_{[\mathfrak{a}]}(s) = \sum_{\mathfrak{b} \sim \mathfrak{a}} N(\mathfrak{b})^{-s}$。

**步骤3**：对主理想类，结合单位群的结构：
$$\zeta_{[\mathcal{O}_K]}(s) \sim \frac{2^{r_1+r_2}\pi^{r_2}R_K}{w_K\sqrt{|d_K|}} \cdot \frac{1}{s-1}$$

**步骤4**：所有理想类贡献相同（相差常数），共 $h_K$ 个类，得到公式。 $\square$

## 5. 类域论简介

**定理 5.1**（类域论基本定理）
设 $K$ 是数域，则存在包含关系的反序一一对应：
$$\{\text{有限Abel扩张 } L/K\} \longleftrightarrow \{\text{射线类群 } Cl_K(\mathfrak{m})\}$$

特别地，Hilbert类域 $H$（最大非分歧Abel扩张）满足：
$$\text{Gal}(H/K) \cong Cl(K)$$

**推论**：$[H:K] = h_K$。
