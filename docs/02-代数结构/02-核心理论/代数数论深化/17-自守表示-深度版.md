# 自守表示 - 深度版

> **"自守表示是Langlands纲领的核心，它统一了数论、表示论和调和分析。"** —— Robert Langlands的宏伟愿景

---

## 📚 概述

自守表示是约化群在Adele群上的不可约表示，是现代数论的核心研究对象。Langlands纲领预言了自守表示与Galois表示之间的深刻联系，这一联系在$GL(2)$情形已通过模性猜想和费马大定理的证明而证实。

---

## 🕰️ 历史发展与数学家理念

### Langlands的宏伟纲领

**Robert Langlands**在1967年给Weil的信中提出了革命性的纲领：

- **函子性原理**：不同群的自守表示之间的对应
- **局部Langlands对应**：局部域上的对应
- **整体Langlands对应**：L-函数和Galois表示的联系

### 格洛腾迪克的Motive理论

**格洛腾迪克数学理念**关于**Motive**的构想：

- Motive是"所有上同调理论的一般化"
- 自守表示对应于Motive的实现在有理点上的实现
- 标准猜想与Langlands纲领的联系

### 调和分析的视角

自守表示可通过调和分析研究：

- **Selberg迹公式**：紧黎曼曲面上的拉普拉斯算子
- **Arthur-Selberg迹公式**：一般约化群上的调和分析

## 🏗️ 核心概念

### 1. Adele环与Idele群

**定义 17.1**（Adele环）
数域$K$的**Adele环**：
$$\mathbb{A}_K = \{(x_v) \in \prod_v K_v : x_v \in \mathcal{O}_v \text{ 对几乎所有 } v\}$$

其中$v$遍历所有位（包括无穷位）。

**定义 17.2**（Idele群）
**Idele群**：$\mathbb{A}_K^\times$，带有限制积拓扑。

### 2. 自守表示的定义

**定义 17.3**（自守形式）
设$G$是约化群，$\omega$是Idele类特征标。函数$f: G(\mathbb{A}) \to \mathbb{C}$称为**自守形式**，如果：

1. **左不变性**：$f(\gamma g) = f(g)$对所有$\gamma \in G(K)$
2. **中心特征标**：$f(zg) = \omega(z)f(g)$对$z \in Z(\mathbb{A})$
3. **有限性条件**：在适当的函数空间中满足增长条件
4. **$K$-有限性**：在极大紧子群$K$作用下有限生成

**定义 17.4**（自守表示）
**自守表示**是$A(G, \omega)$（自守形式空间）的不可约子表示。

### 3. 局部-整体对应

**定理 17.1**（$GL(1)$的局部类域论）
$$\text{Hom}(K_v^\times, \mathbb{C}^\times) \longleftrightarrow \text{Hom}(Gal(\overline{K_v}/K_v), \mathbb{C}^\times)$$

**定理 17.2**（$GL(2)$的局部Langlands对应）
$$\{\text{不可约可容许表示 of } GL(2, K_v)\} \longleftrightarrow \{\text{2维Galois表示}\}$$

### 4. Langlands L-函数

**定义 17.5**（局部L-因子）
对自守表示$\pi$的每个局部分量$\pi_v$，定义局部L-因子$L(s, \pi_v)$。

**定义 17.6**（整体L-函数）
$$L(s, \pi) = \prod_v L(s, \pi_v)$$

**定理 17.3**（Jacquet-Langlands）：$L(s, \pi)$有解析延拓和函数方程。
