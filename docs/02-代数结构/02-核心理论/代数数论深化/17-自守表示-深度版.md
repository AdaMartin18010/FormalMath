# 自守表示理论 - 深度版

## 1. 引言

### 1.1 什么是自守表示？

**自守表示**（Automorphic Representation）是现代数论的核心概念，是经典模形式的深刻推广。它统一了数论、表示论和调和分析，是**朗兰兹纲领**（Langlands Program）的基石。

**核心思想**: 将数论问题转化为表示论问题，通过研究自守表示的L-函数来理解算术对象。

### 1.2 历史背景

| 时期 | 进展 |
|------|------|
| 1920s | 赫克（Hecke）引入赫克算子，研究L-函数的解析延拓和函数方程 |
| 1950s | 志村五郎研究模曲线的Jacobian，建立志村曲线理论 |
| 1967-1970 | 朗兰兹提出朗兰兹纲领，建立自守表示与伽罗瓦表示的联系 |
| 1990s | 怀尔斯证明费马大定理，利用自守表示的形变理论 |
| 2000s | 哈里斯-泰勒、Clozel等人证明局部朗兰兹对应 |

## 2. 经典模形式回顾

### 2.1 模形式的定义

**定义 2.1**:
设 $k$ 为正整数，函数 $f: \mathbb{H} \to \mathbb{C}$ 称为权 $k$ 的**模形式**，如果：

1. $f$ 在上半平面 $\mathbb{H} = \{z \in \mathbb{C} : \text{Im}(z) > 0\}$ 上全纯
2. 对任意 $\gamma = \begin{pmatrix} a & b \\ c & d \end{pmatrix} \in SL_2(\mathbb{Z})$：
   $$f\left(\frac{az+b}{cz+d}\right) = (cz+d)^k f(z)$$
3. $f$ 在无穷远处全纯

**傅里叶展开**:
$$f(z) = \sum_{n=0}^{\infty} a_n q^n, \quad q = e^{2\pi i z}$$

### 2.2 赫克算子

**定义 2.2** (赫克算子 $T_n$):
对素数 $p$，定义：
$$(T_p f)(z) = p^{k-1} f(pz) + \frac{1}{p} \sum_{j=0}^{p-1} f\left(\frac{z+j}{p}\right)$$

**性质**: 赫克算子彼此交换，可同时 diagonalization。

**定理 2.1** (赫克):
设 $f$ 是归一化赫克特征形式，则：
1. $T_n f = a_n f$
2. L-函数 $L(s, f) = \sum_{n=1}^{\infty} \frac{a_n}{n^s}$ 有欧拉积：
   $$L(s, f) = \prod_p \frac{1}{1 - a_p p^{-s} + p^{k-1-2s}}$$

## 3. 自守表示的定义

### 3.1 Adele环

**定义 3.1** (Adele环 $\mathbb{A}$):
$$\mathbb{A} = \mathbb{A}_\mathbb{Q} = \left\{(x_\infty, x_2, x_3, x_5, \ldots) \in \mathbb{R} \times \prod_p \mathbb{Q}_p : x_p \in \mathbb{Z}_p \text{ 对几乎所有 } p\right\}$$

**性质**: 
- $\mathbb{Q}$ 在 $\mathbb{A}$ 中离散嵌入
- $\mathbb{A}/\mathbb{Q}$ 是紧群

### 3.2 自守形式

**定义 3.2**:
函数 $\phi: GL_2(\mathbb{A}) \to \mathbb{C}$ 称为**自守形式**，如果：

1. 左 $\mathbb{Q}$-不变：$\phi(\gamma g) = \phi(g)$，对所有 $\gamma \in GL_2(\mathbb{Q})$
2. 右 $K$-有限：$\phi$ 在 $K = O(2) \times \prod_p GL_2(\mathbb{Z}_p)$ 的右作用下生成有限维空间
3. 满足适当的增长条件
4. 是某个Casimir算子的特征函数

### 3.3 自守表示

**定义 3.3**:
**自守表示** $\pi$ 是 $GL_2(\mathbb{A})$ 在自守形式空间上的不可约子表示。

**结构定理**: 
$$L^2(GL_2(\mathbb{Q}) \backslash GL_2(\mathbb{A}), \omega) = L^2_{cusp} \oplus L^2_{Eis}$$

其中 $L^2_{cusp}$ 是尖点形式空间，$L^2_{Eis}$ 是Eisenstein级数生成的空间。

## 4. 朗兰兹对应

### 4.1 局部朗兰兹对应

**定理 4.1** (哈里斯-泰勒, 2001; Henniart, 2000):
对 $p$-adic域 $F$，存在双射：

$$\{\text{GL}_n(F) \text{的不可约光滑表示}\} \longleftrightarrow \{\text{GL}_n \text{的 } n\text{-维Weil-Deligne表示}\}$$

保持L-因子和 $\varepsilon$-因子。

### 4.2 整体朗兰兹对应

**朗兰兹互反律猜想**:
存在对应：

$$\{\text{GL}_n(\mathbb{A}) \text{ 的尖点自守表示}\} \longleftrightarrow \{\text{Gal}(\overline{\mathbb{Q}}/\mathbb{Q}) \text{ 的 } n\text{-维不可约表示}\}$$

**定理 4.2** (怀尔斯, 泰勒; Breuil-Conrad-Diamond-Taylor):
当 $n = 2$ 时，对满足某些条件的伽罗瓦表示，朗兰兹对应成立。

## 5. L-函数与函子性

### 5.1 自守L-函数

**定义 5.1**:
设 $\pi$ 是 $GL_n(\mathbb{A})$ 的自守表示，其**标准L-函数**定义为：

$$L(s, \pi) = \prod_{v} L(s, \pi_v)$$

其中局部L-因子：
$$L(s, \pi_p) = \prod_{i=1}^{n} \frac{1}{1 - \alpha_{i,p} p^{-s}}$$

**定理 5.1** (Jacquet, Shalika, 1976):
$L(s, \pi)$ 可以解析延拓到整个复平面，满足函数方程：

$$L(s, \pi) = \varepsilon(s, \pi) L(1-s, \tilde{\pi})$$

### 5.2 朗兰兹函子性原理

**原理**: 对代数群之间的映射 $H \to G$，应该有对应的自守表示的转移：

$$\{\text{H的自守表示}\} \longrightarrow \{\text{G的自守表示}\}$$

保持L-函数：
$$L(s, \pi, r \circ \rho) = L(s, r(\pi), \rho)$$

## 6. 应用

### 6.1 费马大定理的证明

怀尔斯证明的关键步骤：
1. 将椭圆曲线 $E$ 的Galois表示 $\rho_{E,p}$ 与模形式联系起来
2. 利用自守表示的形变理论
3. 证明所有半稳定椭圆曲线都是模的

### 6.2 Sato-Tate猜想

**定理** (Taylor等人):
对不具复乘的椭圆曲线，其Frobenius迹的分布服从Sato-Tate分布。

**工具**: 证明相关的对称幂L-函数的全纯性。
