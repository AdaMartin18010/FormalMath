# 分圆域理论 - 深度版

> **"分圆域是代数数论最优美的研究对象之一，它连接了数论、代数与几何。"** —— 致敬Kummer的革命性工作

---

## 📚 概述

分圆域是通过添加单位根得到的数域，是代数数论中最基本也是研究最深入的数域类。Kummer在19世纪中叶研究分圆域的整数环和理想分解，创立了理想数理论，为现代代数数论奠定了基础。分圆域理论在费马大定理的证明、Iwasawa理论和朗兰兹纲领中都有核心应用。

---

## 🕰️ 历史发展与数学家理念

### Kummer的理想数理论

Ernst Kummer在1847年研究费马大定理时，发现分圆域$\mathbb{Q}(\zeta_p)$的整数环中唯一分解不成立：

- **理想数**：Kummer引入"理想数"恢复唯一分解
- **正则素数**：定义使$p \nmid h_{\mathbb{Q}(\zeta_p)}$的素数为正则素数
- **费马大定理**：Kummer证明了正则素数情形的费马大定理

### 伽罗瓦与分圆方程

**伽罗瓦数学理念**关于**分圆方程可解性**的研究：

- $x^n - 1 = 0$的Galois群是阿贝尔群$(\mathbb{Z}/n\mathbb{Z})^\times$
- 分圆域是Abel扩张的基本例子
- Kronecker-Weber定理：任何Abel扩张都包含于分圆域中

### 格洛腾迪克的几何化

**格洛腾迪克数学理念**通过**étale上同调**研究分圆域：

- 分圆单位与K-理论的联系
- Iwasawa理论的$\mathbb{Z}_p$-扩张

## 🏗️ 核心概念与完整证明

### 1. 分圆域的定义

**定义 14.1**（分圆域）
设$n \geq 1$，**$n$次分圆域**定义为：
$$\mathbb{Q}(\zeta_n) = \mathbb{Q}(e^{2\pi i/n})$$

其中$\zeta_n = e^{2\pi i/n}$是本原$n$次单位根。

**定理 14.1**：$[\mathbb{Q}(\zeta_n) : \mathbb{Q}] = \varphi(n)$（欧拉函数）。

**证明**：$\zeta_n$的极小多项式是**分圆多项式**：
$$\Phi_n(x) = \prod_{\substack{1 \leq k \leq n \\ \gcd(k,n)=1}} (x - \zeta_n^k)$$

次数为$\varphi(n)$。

### 2. 整数环结构

**定理 14.2**（分圆域的整数环）
$$\mathcal{O}_{\mathbb{Q}(\zeta_n)} = \mathbb{Z}[\zeta_n]$$

**证明概要**：

**步骤1**：计算判别式。
$$d_{\mathbb{Q}(\zeta_p)} = (-1)^{\frac{p-1}{2}}p^{p-2}$$对素数$p$。

**步骤2**：证明整基。
利用判别式的素因子分析证明$\{1, \zeta_n, \ldots, \zeta_n^{\varphi(n)-1}\}$是整基。

### 3. Galois群结构

**定理 14.3**：$Gal(\mathbb{Q}(\zeta_n)/\mathbb{Q}) \cong (\mathbb{Z}/n\mathbb{Z})^\times$

**证明**：
$\sigma \in Gal(\mathbb{Q}(\zeta_n)/\mathbb{Q})$由$\sigma(\zeta_n) = \zeta_n^a$确定，其中$\gcd(a, n) = 1$。

映射$\sigma \mapsto a \pmod{n}$给出同构。

### 4. Kummer引理与单位

**定理 14.4**（Kummer）：分圆单位
设$p$是奇素数，$\zeta = \zeta_p$。则：
$$\frac{1-\zeta^a}{1-\zeta^b}$$
是单位（当$p \nmid a, b$）。
