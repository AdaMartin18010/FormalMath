# 分圆域理论 - 深度版

## 1. 引言与历史背景

### 1.1 什么是分圆域？

**分圆域**（Cyclotomic Field）是指由单位根生成的数域。具体而言，对于正整数 $n \geq 3$，第 $n$ 个分圆域 $\mathbb{Q}(\zeta_n)$ 是通过在有理数域 $\mathbb{Q}$ 上添加本原 $n$ 次单位根 $\zeta_n = e^{2\pi i/n}$ 而得到的域扩张。

**历史意义**: 分圆域理论是代数数论的核心分支之一，起源于高斯对正多边形可作图性的研究，后经库默尔、克罗内克、希尔伯特等人的发展，成为现代代数数论的重要工具。

### 1.2 发展历程

| 时期 | 数学家 | 贡献 |
|------|--------|------|
| 1801 | 高斯 | 《算术研究》：分圆方程的可解性，正17边形的尺规作图 |
| 1847 | 库默尔 | 创立理想数理论，证明费马大定理对部分指数成立 |
| 1850s | 克罗内克 | 克罗内克-韦伯定理：任何阿贝尔扩张都包含于分圆域 |
| 1897 | 希尔伯特 | 希尔伯特第12问题，类域论的建立 |
| 1950s | 岩泽健吉 | 岩泽理论：分圆域的 $\mathbb{Z}_p$-扩张理论 |

## 2. 基本定义与性质

### 2.1 分圆多项式

**定义 2.1** (分圆多项式):
第 $n$ 个分圆多项式 $\Phi_n(x)$ 定义为所有本原 $n$ 次单位根的最小多项式：

$$\Phi_n(x) = \prod_{\substack{1 \leq k \leq n \\ (k,n) = 1}} (x - \zeta_n^k)$$

**性质**:
1. $\Phi_n(x)$ 是首一整系数不可约多项式
2. $\deg \Phi_n(x) = \varphi(n)$ （欧拉函数）
3. $x^n - 1 = \prod_{d|n} \Phi_d(x)$

**示例**:
- $\Phi_3(x) = x^2 + x + 1$
- $\Phi_4(x) = x^2 + 1$
- $\Phi_5(x) = x^4 + x^3 + x^2 + x + 1$
- $\Phi_8(x) = x^4 + 1$

**定理 2.1** (高斯):
分圆多项式 $\Phi_n(x)$ 在 $\mathbb{Q}$ 上不可约。

**证明概要**: 
利用Eisenstein判别法的变体。设 $p$ 是不整除 $n$ 的素数，证明若 $f(x)$ 是 $\zeta_n$ 的极小多项式，则 $f(\zeta_n^p) = 0$。通过迭代可知所有本原单位根都是 $f$ 的根，故 $f = \Phi_n$。 $\square$

### 2.2 分圆域的次数

**定理 2.2**:
分圆域 $\mathbb{Q}(\zeta_n)$ 是 $\mathbb{Q}$ 的 $\varphi(n)$ 次扩张：

$$[\mathbb{Q}(\zeta_n) : \mathbb{Q}] = \varphi(n)$$

**证明**: 
由定理2.1，$\zeta_n$ 的极小多项式为 $\Phi_n(x)$，其次数为 $\varphi(n)$。 $\square$

### 2.3 伽罗瓦群结构

**定理 2.3**:
$\text{Gal}(\mathbb{Q}(\zeta_n)/\mathbb{Q}) \cong (\mathbb{Z}/n\mathbb{Z})^\times$

具体对应为：对 $a \in (\mathbb{Z}/n\mathbb{Z})^\times$，定义自同构 $\sigma_a: \zeta_n \mapsto \zeta_n^a$。

**推论**: 分圆域是 $\mathbb{Q}$ 的阿贝尔扩张（即伽罗瓦群为阿贝尔群）。

## 3. 整数环与素数分解

### 3.1 整数环

**定理 3.1**:
分圆域 $\mathbb{Q}(\zeta_n)$ 的整数环为 $\mathbb{Z}[\zeta_n]$。

### 3.2 素数分解

**定理 3.2** (分圆域中的素数分解):
设 $p$ 是不整除 $n$ 的素数，$f$ 是 $p$ 模 $n$ 的阶（即使得 $p^f \equiv 1 \pmod{n}$ 的最小正整数），则 $p$ 在 $\mathbb{Q}(\zeta_n)$ 中分解为：

$$(p) = \mathfrak{p}_1 \mathfrak{p}_2 \cdots \mathfrak{p}_g$$

其中 $g = \varphi(n)/f$，每个 $\mathfrak{p}_i$ 的剩余次数为 $f$。

**示例** (在 $\mathbb{Q}(\zeta_5)$ 中):
- $p = 2$: 模5的阶为4，故 $(2)$ 保持素性（惯性）
- $p = 11$: 模5的阶为1，故 $(11) = \mathfrak{p}_1\mathfrak{p}_2\mathfrak{p}_3\mathfrak{p}_4$ 完全分裂

## 4. 类数与库默尔理论

### 4.1 分圆域的类数

**定义 4.1**:
分圆域 $\mathbb{Q}(\zeta_p)$ （$p$ 为奇素数）的类数记为 $h_p$。

**库默尔判据**:
若 $p \nmid h_p$（即 $p$ 是**正则素数**），则费马大定理对指数 $p$ 成立。

**定理 4.1** (库默尔):
素数 $p$ 是正则的当且仅当 $p$ 不整除伯努利数 $B_2, B_4, \ldots, B_{p-3}$ 的分子。

**已知的正则素数**: 3, 5, 7, 11, 13, 17, 19, ...
**不规则素数**: 37, 59, 67, 101, ...

### 4.2 岩泽理论简介

**定理 4.2** (岩泽):
对于固定的素数 $p$，考虑分圆 $\mathbb{Z}_p$-扩张 $K_\infty = \bigcup_{n \geq 0} \mathbb{Q}(\zeta_{p^{n+1}})$。则存在整数 $\lambda, \mu, \nu$ 使得对充分大的 $n$：

$$\text{ord}_p(h_{p^{n+1}}^-) = \lambda n + \mu p^n + \nu$$

其中 $h_{p^{n+1}}^-$ 是相对类数。

## 5. 克罗内克-韦伯定理

**定理 5.1** (克罗内克-韦伯):
设 $K/\mathbb{Q}$ 是有限阿贝尔扩张，则存在正整数 $n$ 使得 $K \subseteq \mathbb{Q}(\zeta_n)$。

**意义**: 这是类域论在 $\mathbb{Q}$ 上的特例，完全刻画了 $\mathbb{Q}$ 的所有阿贝尔扩张。

**推论**: $\mathbb{Q}$ 的极大阿贝尔扩张为 $\mathbb{Q}^{ab} = \bigcup_{n \geq 1} \mathbb{Q}(\zeta_n)$。

## 6. 现代应用

### 6.1 费马大定理

库默尔利用分圆域理论证明了：若 $p$ 是正则素数，则 $x^p + y^p = z^p$ 无非零整数解。

### 6.2 代数K理论

分圆域在代数K理论中起重要作用，特别是Milnor K群与Galois上同调的关系。

### 6.3 密码学

分圆域在格密码学和全同态加密中有应用，如NTRU密码系统的安全性分析与分圆域结构相关。
