# 赋值与局部域 - 深度版

赋值论（valuation theory）由 Kurt Hensel 于 1897 年创立，将数域的算术性质局部化，建立了局部-整体原则（Local-Global Principle）。局部域（$p$-adic 域与形式实域）为 Diophantine 方程、类域论与 Langlands 纲领提供了基本框架。

## 1. 赋值的基本理论

### 1.1 定义与例子

**定义 1.1（赋值）**. 域 $K$ 上的赋值是映射 $v: K^\times \to \Gamma$（$\Gamma$ 为全序 Abel 群），满足：
1. $v(xy) = v(x) + v(y)$；
2. $v(x+y) \geq \min(v(x), v(y))$（非 Archimedean 三角不等式）。

延拓 $v(0) = +\infty$。

**定义 1.2（赋值环）**. $\mathcal{O}_v = \{x \in K : v(x) \geq 0\}$，极大理想 $\mathfrak{m}_v = \{x : v(x) > 0\}$，剩余域 $k_v = \mathcal{O}_v / \mathfrak{m}_v$。

**例子 1.3**.
- **$p$-adic 赋值**：$v_p(p^n \frac{a}{b}) = n$（$p \nmid a,b$），$\Gamma = \mathbb{Z}$；
- **平凡赋值**：$v(x) = 0$（$x \neq 0$）；
- **次数赋值**：$K = k(t)$，$v(f/g) = \deg g - \deg f$。

### 1.2 赋值的等价与分类

**定义 1.4（绝对值）**. 与赋值 $v$ 对应的绝对值 $|x|_v = c^{-v(x)}$（$c > 1$）。

**定理 1.5（Ostrowski）**. $\mathbb{Q}$ 上的非平凡绝对值（在等价意义下）恰为：
1. $p$-adic 绝对值 $|\cdot|_p$（对每个素数 $p$）；
2. 通常绝对值 $|\cdot|_\infty$。

且满足乘积公式：$\prod_{p \leq \infty} |x|_p = 1$（$x \in \mathbb{Q}^\times$）。

## 2. $p$-adic 数

### 2.1 定义与拓扑

**定义 2.1**. $\mathbb{Q}_p$ 为 $\mathbb{Q}$ 关于 $v_p$ 的完备化：

$$\mathbb{Q}_p = \left\{\sum_{n=N}^{\infty} a_n p^n : N \in \mathbb{Z},\ a_n \in \{0, \dots, p-1\}\right\}.$$

**$p$-adic 整数环**：$\mathbb{Z}_p = \{x \in \mathbb{Q}_p : v_p(x) \geq 0\} = \{\sum_{n=0}^{\infty} a_n p^n\}$。

**定理 2.2**. $\mathbb{Z}_p$ 是局部主理想整环，非零理想恰为 $p^n \mathbb{Z}_p$（$n \geq 0$），剩余域 $\mathbb{Z}_p / p\mathbb{Z}_p \cong \mathbb{F}_p$。

### 2.2 Hensel 引理

**定理 2.3（Hensel 引理）**. 设 $f(x) \in \mathbb{Z}_p[x]$，$a_0 \in \mathbb{Z}_p$ 满足：
- $v_p(f(a_0)) > 0$；
- $v_p(f'(a_0)) = 0$。

则存在唯一 $a \in \mathbb{Z}_p$ 使 $f(a) = 0$ 且 $v_p(a - a_0) > 0$。

*证明（Newton 迭代）*. 定义 $a_{n+1} = a_n - \frac{f(a_n)}{f'(a_n)}$。

**步骤 1**：Taylor 展开：

$$f(a_{n+1}) = f(a_n) + f'(a_n)(a_{n+1} - a_n) + \sum_{k \geq 2} \frac{f^{(k)}(a_n)}{k!}(a_{n+1}-a_n)^k = \sum_{k \geq 2} \dots$$

故 $v_p(f(a_{n+1})) \geq 2v_p(a_{n+1} - a_n) = 2v_p(f(a_n))$（因 $v_p(f'(a_n)) = 0$）。

**步骤 2**：$v_p(a_{n+1} - a_n) = v_p(f(a_n)) \geq 2^n v_p(f(a_0)) \to \infty$，故 $\{a_n\}$ 为 Cauchy 列。

**步骤 3**：取极限 $a = \lim a_n$，$f(a) = 0$。唯一性由 $f'(a_0)$ 可逆保证。$\square$

## 3. 局部-整体原则

### 3.1 Hasse-Minkowski 定理

**定理 3.1（Hasse-Minkowski）**. 有理二次型 $Q$ 在 $\mathbb{Q}$ 上表示 0（有非平凡零点）当且仅当它在所有完备化 $\mathbb{Q}_v$（包括 $\mathbb{Q}_\infty = \mathbb{R}$）上表示 0。

*证明概要*. 对二元二次型由 Legendre 符号的二次互反律保证。对三元用 Dirichlet 算术级数定理。一般情形归纳。$\square$

### 3.2 反例

**定理 3.2（Selmer）**. $3x^3 + 4y^3 + 5z^3 = 0$ 在所有 $\mathbb{Q}_p$ 上有非平凡解，但在 $\mathbb{Q}$ 上无解。

这说明 Hasse-Minkowski 对三次型不成立。更一般地，Shafarevich-Tate 群 $Ш$ 度量了整体解与局部解的偏差。

## 4. 局部域的结构

### 4.1 局部域的分类

**定义 4.1（局部域）**. 局部域是 $\mathbb{R}$、$\mathbb{C}$、有限扩张的 $p$-adic 域，或 $\mathbb{F}_q((t))$（形式 Laurent 级数域）。

**定理 4.2**. 局部域 $K$ 具有以下性质：
1. 局部紧（作为拓扑域）；
2. 赋值离散；
3. 剩余域有限。

### 4.2 分歧理论

设 $L/K$ 为局部域的有限扩张。

**定义 4.3**. $e = [v(L^\times) : v(K^\times)]$ 为分歧指数，$f = [k_L : k_K]$ 为剩余次数。

**定理 4.4**. $[L : K] = ef$。

**定义 4.5**. 扩张称为非分歧的若 $e = 1$；完全分歧的若 $f = 1$。

## 5. 例子

### 5.1 例子：$\mathbb{Q}_p$ 中的平方元

**习题 5.1**. $x \in \mathbb{Q}_p^\times$ 为平方元当且仅当 $v_p(x)$ 为偶数且 $\bar{x} \in \mathbb{F}_p^\times$ 为平方（$p$ 奇）。对 $p = 2$ 需更精细的条件。

**解答**. 设 $x = p^n u$，$u \in \mathbb{Z}_p^\times$。若 $x = y^2$，则 $n = 2v_p(y)$ 为偶数，且 $\bar{u} = \bar{y}^2$。反之，若 $n = 2m$ 且 $\bar{u}$ 为平方，用 Hensel 引理提升 $\bar{u}$ 的平方根至 $\mathbb{Z}_p$。$\square$

### 5.2 例子：局部域中的指数与对数

**习题 5.2**. 对 $p$-adic 域 $K$，指数映射 $\exp(x) = \sum_{n=0}^{\infty} \frac{x^n}{n!}$ 在 $v_p(x) > \frac{1}{p-1}$ 时收敛，给出同构：

$$\exp: \{x : v_p(x) > \frac{1}{p-1}\} \xrightarrow{\sim} \{u \in 1 + \mathfrak{m}_K : v_p(u-1) > \frac{1}{p-1}\}.$$
