# 代数整数环 - 深度版

> **"代数整数是数域中的'整数'，它们的环结构是代数数论的基础。"**

---

## 1. 代数整数的定义

### 1.1 基本定义

**定义 1.1**（代数整数）
复数 $\alpha$ 称为**代数整数**，如果它是某个首一整系数多项式的根：
$$\alpha^n + a_{n-1}\alpha^{n-1} + \cdots + a_1\alpha + a_0 = 0, \quad a_i \in \mathbb{Z}$$

**等价条件**：$\alpha$ 的极小多项式（在 $\mathbb{Q}$ 上）是首一的整系数多项式。

**定理 1.1**
代数整数构成 $\mathbb{C}$ 的子环，记为 $\overline{\mathbb{Z}}$。

**证明**：
设 $\alpha, \beta$ 是代数整数，需证 $\alpha + \beta$ 和 $\alpha\beta$ 也是代数整数。

设 $\alpha$ 满足 $f(x) = x^m + a_{m-1}x^{m-1} + \cdots + a_0 = 0$，$\beta$ 满足 $g(x) = x^n + b_{n-1}x^{n-1} + \cdots + b_0 = 0$。

令 $V = \text{span}_{\mathbb{Z}}\{\alpha^i\beta^j : 0 \leq i < m, 0 \leq j < n\}$。

则 $(\alpha + \beta)V \subseteq V$ 且 $(\alpha\beta)V \subseteq V$。

考虑 $\alpha + \beta$ 在 $V$ 的基 $\{\alpha^i\beta^j\}$ 下的作用矩阵 $A$，其特征多项式是首一整系数的。由Cayley-Hamilton，$\alpha + \beta$ 是其特征多项式的根，故为代数整数。同理 $\alpha\beta$。 $\square$

### 1.2 数域的整数环

**定义 1.2**（数域的整数环）
数域 $K$ 的**整数环**：
$$\mathcal{O}_K = \{\alpha \in K : \alpha \text{ 是代数整数}\} = K \cap \overline{\mathbb{Z}}$$

**定理 1.2**
$\mathcal{O}_K$ 是 $K$ 的子环，且是秩为 $[K:\mathbb{Q}]$ 的自由 $\mathbb{Z}$-模。

**证明**：

**步骤1**：设 $[K:\mathbb{Q}] = n$，取 $K$ 的 $\mathbb{Q}$-基 $\alpha_1, \ldots, \alpha_n$。通过乘以公分母，可设 $\alpha_i \in \mathcal{O}_K$。

**步骤2**：考虑**迹形式**：
$$T(\alpha, \beta) = \text{Tr}_{K/\mathbb{Q}}(\alpha\beta)$$

这是 $K$ 上的非退化对称双线性型（因为 $K/\mathbb{Q}$ 可分）。

**步骤3**：令 $M = \mathbb{Z}\alpha_1 + \cdots + \mathbb{Z}\alpha_n \subseteq \mathcal{O}_K$。

考虑对偶模：
$$M^\vee = \{\beta \in K : \text{Tr}(\beta\alpha) \in \mathbb{Z}, \forall \alpha \in M\}$$

则 $M^\vee$ 也是秩为 $n$ 的 $\mathbb{Z}$-模，且 $M \subseteq \mathcal{O}_K \subseteq M^\vee$。

**步骤4**：由于 $M^\vee/M$ 有限，$\mathcal{O}_K$ 夹在两个秩 $n$ 的自由模之间，故也是秩 $n$ 的自由 $\mathbb{Z}$-模。 $\square$

## 2. Dedekind整环

### 2.1 定义

**定义 2.1**（Dedekind整环）
整环 $R$ 称为**Dedekind整环**，如果满足：

1. **Noetherian**：每个理想升链稳定
2. **整闭**：在分式域中整闭
3. **维数1**：每个非零素理想是极大的

**定理 2.1**
数域的整数环 $\mathcal{O}_K$ 是Dedekind整环。

**证明**：
1. **Noetherian**：$\mathcal{O}_K$ 是有限生成 $\mathbb{Z}$-模，故理想也是有限生成的。
2. **整闭**：由定义，$\mathcal{O}_K$ 包含 $K$ 中所有整元素。
3. **素理想极大**：设 $\mathfrak{p}$ 是非零素理想，则 $\mathfrak{p} \cap \mathbb{Z} = (p)$ 对某个素数 $p$。$\mathcal{O}_K/\mathfrak{p}$ 是 $\mathbb{F}_p$ 的有限整环，故为域，$\mathfrak{p}$ 极大。 $\square$

### 2.2 理想的唯一分解

**定理 2.2**（Dedekind唯一分解定理）
Dedekind整环 $R$ 中每个非零理想可唯一分解为素理想的乘积（不计顺序）。

**证明**：

**存在性**：

设 $S$ 是不能分解为素理想乘积的非零理想集合。若 $S \neq \emptyset$，由Noetherian条件，$S$ 有极大元 $\mathfrak{a}$。

$\mathfrak{a}$ 不是素的，故存在 $b, c \in R$ 使 $bc \in \mathfrak{a}$ 但 $b, c \notin \mathfrak{a}$。

则 $\mathfrak{a} \subsetneq (\mathfrak{a}, b)$，$\mathfrak{a} \subsetneq (\mathfrak{a}, c)$，且 $(\mathfrak{a}, b)(\mathfrak{a}, c) \subseteq \mathfrak{a}$。

由极大性，$(\mathfrak{a}, b)$ 和 $(\mathfrak{a}, c)$ 可分解，故 $\mathfrak{a}$ 也可分解，矛盾。

**唯一性**：

设 $\mathfrak{p}_1\cdots\mathfrak{p}_r = \mathfrak{q}_1\cdots\mathfrak{q}_s$。

则 $\mathfrak{p}_1 \supseteq \mathfrak{q}_1\cdots\mathfrak{q}_s$，故 $\mathfrak{p}_1 \supseteq \mathfrak{q}_i$ 对某个 $i$。由极大性，$\mathfrak{p}_1 = \mathfrak{q}_i$。

利用理想的可逆性（Dedekind整环中所有非零理想可逆），消去 $\mathfrak{p}_1 = \mathfrak{q}_i$，归纳得证。 $\square$

## 3. 素理想分解

### 3.1 Kummer-Dedekind定理

**定理 3.1**（Kummer-Dedekind）
设 $K = \mathbb{Q}(\theta)$，$\theta \in \mathcal{O}_K$，$f(x)$ 是 $\theta$ 的极小多项式。设 $p \nmid [\mathcal{O}_K : \mathbb{Z}[\theta]]$。

在 $\mathbb{F}_p[x]$ 中分解 $\bar{f}(x) = \bar{g}_1(x)^{e_1}\cdots\bar{g}_r(x)^{e_r}$，其中 $\bar{g}_i$ 互异首一不可约。

令 $g_i(x) \in \mathbb{Z}[x]$ 是 $\bar{g}_i$ 的提升。则：
$$(p) = \mathfrak{p}_1^{e_1}\cdots\mathfrak{p}_r^{e_r}, \quad \mathfrak{p}_i = (p, g_i(\theta))$$

且 $N(\mathfrak{p}_i) = p^{\deg g_i}$。

### 3.2 例子

**例 1**：$K = \mathbb{Q}(\sqrt{-5})$，$\mathcal{O}_K = \mathbb{Z}[\sqrt{-5}]$

$(2)$ 的分解：$x^2 + 5 \equiv (x+1)^2 \pmod{2}$

故 $(2) = (2, 1+\sqrt{-5})^2 = \mathfrak{p}^2$（分歧）。 $\square$

**例 2**：$K = \mathbb{Q}(\sqrt{5})$，$\mathcal{O}_K = \mathbb{Z}[\frac{1+\sqrt{5}}{2}]$

$(5)$ 的分解：$x^2 - x - 1 \equiv x(x-1) \pmod{5}$

故 $(5) = (5, \frac{1+\sqrt{5}}{2})(5, \frac{1+\sqrt{5}}{2}-1) = \mathfrak{p}_1\mathfrak{p}_2$（完全分裂）。 $\square$

## 4. 判别式与整基

### 4.1 判别式

**定义 4.1**（判别式）
设 $\alpha_1, \ldots, \alpha_n$ 是 $K$ 的 $\mathbb{Q}$-基，$\sigma_1, \ldots, \sigma_n$ 是 $K$ 到 $\mathbb{C}$ 的嵌入。定义：
$$d(\alpha_1, \ldots, \alpha_n) = \det(\sigma_i(\alpha_j))^2$$

**定理 4.1**
$d(\alpha_1, \ldots, \alpha_n) \in \mathbb{Q}$，且若 $\alpha_i \in \mathcal{O}_K$，则 $d(\alpha_1, \ldots, \alpha_n) \in \mathbb{Z}$。

**定义 4.2**（域判别式）
设 $\omega_1, \ldots, \omega_n$ 是 $\mathcal{O}_K$ 的 $\mathbb{Z}$-基（整基），定义域的判别式：
$$d_K = d(\omega_1, \ldots, \omega_n)$$

**定理 4.2**
$d_K \equiv 0$ 或 $1 \pmod{4}$（Stickelberger定理）。

### 4.2 整基的计算

**定理 4.3**
设 $\theta \in \mathcal{O}_K$，$K = \mathbb{Q}(\theta)$。若 $f(x)$ 是 $\theta$ 的极小多项式，则：
$$d(1, \theta, \ldots, \theta^{n-1}) = (-1)^{\frac{n(n-1)}{2}}N_{K/\mathbb{Q}}(f'(\theta))$$

**例**：$K = \mathbb{Q}(\sqrt{d})$，$d$ 无平方因子

- 若 $d \equiv 2, 3 \pmod{4}$：$\mathcal{O}_K = \mathbb{Z}[\sqrt{d}]$，$d_K = 4d$
- 若 $d \equiv 1 \pmod{4}$：$\mathcal{O}_K = \mathbb{Z}[\frac{1+\sqrt{d}}{2}]$，$d_K = d$
