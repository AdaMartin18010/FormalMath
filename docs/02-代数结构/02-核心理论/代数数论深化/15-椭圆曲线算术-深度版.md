# 椭圆曲线算术 - 深度版

> **"椭圆曲线是数论最丰富的研究对象之一，它连接了代数、几何、分析和密码学。"** —— Barry Mazur

---

## 📚 概述

椭圆曲线是亏格1的光滑射影曲线，具有自然的阿贝尔群结构。椭圆曲线的算术理论是现代数论的核心领域，在费马大定理的证明（Wiles-Taylor）、BSD猜想和密码学（ECC）中都有核心应用。

---

## 🕰️ 历史发展与数学家理念

### 阿贝尔的椭圆函数理论

**阿贝尔数学理念**关于**椭圆函数**的开创性研究：

- **椭圆积分反演**：Abel发现研究椭圆积分的反函数更有效
- **双周期性**：椭圆函数具有两个独立的周期
- **Abel定理**：描述椭圆曲线上点的加性结构

### 格洛腾迪克的概形理论

**格洛腾迪克数学理念**通过**Abel簇**的框架研究椭圆曲线：

- **通用椭圆曲线**：模概形上的椭圆曲线族
- **高度配对**：连接算术与几何
- **Motive理论**：椭圆曲线的深层结构

### Weil和Mordell的贡献

- **Mordell定理**（1922）：$E(\mathbb{Q})$是有限生成阿贝尔群
- **Weil的推广**：任意数域上的Mordell-Weil定理
- **Tate模**：$\ell$-进表示研究椭圆曲线

## 🏗️ 核心概念与完整证明

### 1. 椭圆曲线的定义

**定义 15.1**（Weierstrass方程）
域$K$上的**椭圆曲线**$E$是非奇异射影曲线，具有仿射Weierstrass方程：
$$y^2 + a_1xy + a_3y = x^3 + a_2x^2 + a_4x + a_6$$

特征不为2,3时可简化为：
$$y^2 = x^3 + Ax + B$$

判别式$\Delta = -16(4A^3 + 27B^2) \neq 0$保证非奇异。

### 2. 群结构

**定理 15.1**（Bezout定理的几何）
椭圆曲线$E$上的点构成交换群，运算为：

- **零点**：无穷远点$\mathcal{O}$
- **加法**：$P + Q + R = \mathcal{O}$当$P, Q, R$共线

**显式公式**（特征$\neq 2, 3$）：

设$P = (x_1, y_1)$，$Q = (x_2, y_2)$：

**情形1**（$P \neq Q$，$x_1 \neq x_2$）：
$$\lambda = \frac{y_2 - y_1}{x_2 - x_1}, \quad \nu = \frac{y_1x_2 - y_2x_1}{x_2 - x_1}$$

**情形2**（$P = Q$，$y_1 \neq 0$）：
$$\lambda = \frac{3x_1^2 + A}{2y_1}, \quad \nu = \frac{-x_1^3 + Ax_1 + 2B}{2y_1}$$

则$P + Q = (\lambda^2 - x_1 - x_2, -\lambda^3 + \lambda(x_1 + x_2) - \nu)$。

### 3. Mordell-Weil定理

**定理 15.2**（Mordell-Weil）
数域$K$上椭圆曲线$E$的有理点群$E(K)$是有限生成阿贝尔群：
$$E(K) \cong E(K)_{tors} \oplus \mathbb{Z}^r$$

其中$r$称为**秩**，$E(K)_{tors}$是挠子群。

**证明概要**：

**步骤1**：高度函数。

定义**对数高度**$h: E(K) \to \mathbb{R}_{\geq 0}$，满足：

- 仅有有限多点有界高度（Northcott性质）
- $h(2P) = 4h(P) + O(1)$（抛物性）

**步骤2**：下降论证。

证明$E(K)/2E(K)$有限（弱Mordell-Weil定理）。

**步骤3**：生成性。

利用高度论证有限生成。

### 4. BSD猜想

**猜想 15.1**（Birch-Swinnerton-Dyer）
$$\text{rank}(E(\mathbb{Q})) = \text{ord}_{s=1} L(E, s)$$

其中$L(E, s)$是Hasse-Weil $L$-函数。
