# 椭圆曲线算术 - 深度版

> **"椭圆曲线是数论最丰富的研究对象之一，它连接了代数、几何、分析和密码学。"** —— Barry Mazur

---

## 1. 椭圆曲线的定义

### 1.1 Weierstrass方程

**定义 1.1**（椭圆曲线）
域 $K$ 上的**椭圆曲线** $E$ 是亏格为1的光滑射影曲线，具有至少一个 $K$-有理点。

在特征 $\text{char}(K) \neq 2, 3$ 时，$E$ 可由**短Weierstrass方程**给出：
$$E : y^2 = x^3 + Ax + B, \quad A, B \in K$$

其**判别式**为：
$$\Delta = -16(4A^3 + 27B^2)$$

光滑性条件等价于 $\Delta \neq 0$。

**定义 1.2**（$j$-不变量）
椭圆曲线 $E: y^2 = x^3 + Ax + B$ 的**$j$-不变量**为：
$$j(E) = -1728\frac{(4A)^3}{\Delta}$$

**定理 1.1**
两个椭圆曲线在代数闭包上同构当且仅当它们有相同的 $j$-不变量。

## 2. 群结构

### 2.1 几何加法

**定理 2.1**（Mordell-Weil群结构）
椭圆曲线 $E$ 上的点集 $E(\bar{K})$ 具有自然的交换群结构：

- **零点**：无穷远点 $\mathcal{O} = [0:1:0]$（射影坐标）
- **加法**：$P + Q + R = \mathcal{O}$ 当且仅当 $P, Q, R$ 共线
- **逆元**：若 $P = (x,y)$，则 $-P = (x, -y)$

### 2.2 显式公式

设 $P = (x_1, y_1)$，$Q = (x_2, y_2)$，$P + Q = (x_3, y_3)$：

**情形1**（$P \neq Q$，$x_1 \neq x_2$）：
$$\lambda = \frac{y_2 - y_1}{x_2 - x_1}, \quad \nu = \frac{y_1 x_2 - y_2 x_1}{x_2 - x_1}$$

**情形2**（$P = Q$，$y_1 \neq 0$，倍点公式）：
$$\lambda = \frac{3x_1^2 + A}{2y_1}, \quad \nu = \frac{-x_1^3 + Ax_1 + 2B}{2y_1}$$

则：
$$x_3 = \lambda^2 - x_1 - x_2, \quad y_3 = -\lambda x_3 - \nu = \lambda(x_1 - x_3) - y_1$$

**证明**：
过 $P$ 和 $Q$ 的直线为 $y = \lambda x + \nu$。代入Weierstrass方程：
$$(\lambda x + \nu)^2 = x^3 + Ax + B$$

这是关于 $x$ 的三次方程，已知根为 $x_1, x_2$，设第三个根为 $x_3$。由Vieta公式：
$$x_1 + x_2 + x_3 = \lambda^2$$

故 $x_3 = \lambda^2 - x_1 - x_2$。第三个交点为 $R = (x_3, \lambda x_3 + \nu)$，而 $P + Q = -R = (x_3, -\lambda x_3 - \nu)$。 $\square$

## 3. Mordell-Weil定理

### 3.1 定理陈述

**定理 3.1**（Mordell-Weil）
设 $K$ 是数域，$E$ 是 $K$ 上的椭圆曲线，则 $E(K)$ 是有限生成阿贝尔群：
$$E(K) \cong E(K)_{\text{tors}} \oplus \mathbb{Z}^r$$

其中 $r$ 称为**秩**（Mordell-Weil秩），$E(K)_{\text{tors}}$ 是有限挠子群。

### 3.2 弱Mordell-Weil定理

**定理 3.2**（弱Mordell-Weil）
对任意正整数 $m$，$E(K)/mE(K)$ 是有限群。

**证明概要**（$m = 2$ 情形）：

设 $E: y^2 = (x - e_1)(x - e_2)(x - e_3)$，$e_i \in K$。

定义同态 $\phi: E(K) \to K^\times/(K^\times)^2 \times K^\times/(K^\times)^2$：
- $\phi(\mathcal{O}) = (1, 1)$
- $\phi((e_1, 0)) = ((e_1-e_2)(e_1-e_3), e_1-e_2)$
- 对 $P = (x,y)$，$x \neq e_i$：$\phi(P) = (x-e_1, x-e_2)$

**关键步骤**：证明 $\ker\phi = 2E(K)$，且 $\text{im}\phi$ 有限（由 $K$ 的 $S$-单位群有限性）。 $\square$

### 3.3 高度函数

**定义 3.1**（对数高度）
设 $P = [x_0:\cdots:x_n] \in \mathbb{P}^n(\mathbb{Q})$，取整坐标 $\gcd(x_0,\ldots,x_n) = 1$，定义：
$$H(P) = \max\{|x_0|, \ldots, |x_n|\}, \quad h(P) = \log H(P)$$

对 $P \in E(\mathbb{Q})$，$h(P) = h(x(P))$。

**定理 3.3**（高度性质）
1. **有限性**：对任意 $C > 0$，$\{P \in E(\mathbb{Q}) : h(P) \leq C\}$ 有限（Northcott）
2. **抛物性**：$h(2P) = 4h(P) + O(1)$
3. **下降**：$h(P - P_0) = 2h(P) + O(\sqrt{h(P)})$

**定理 3.4**（ canonical高度）
存在唯一的二次型 $\hat{h}: E(\mathbb{Q}) \to \mathbb{R}_{\geq 0}$ 使得 $\hat{h}(P) = h(P) + O(1)$，且：
$$\hat{h}(mP) = m^2\hat{h}(P), \quad \hat{h}(P+Q) + \hat{h}(P-Q) = 2\hat{h}(P) + 2\hat{h}(Q)$$

### 3.4 Mordell-Weil定理的完整证明

**证明**：
由弱Mordell-Weil定理，$E(\mathbb{Q})/2E(\mathbb{Q})$ 有限。设其代表元为 $Q_1, \ldots, Q_m$。

**关键引理**：存在常数 $C$ 使得对任意 $P \in E(\mathbb{Q})$，存在分解 $P = 2P_1 + Q_i$ 满足 $\hat{h}(P_1) \leq \frac{3}{4}\hat{h}(P) + C$。

**归纳下降**：
反复应用引理：
$$P = 2P_1 + Q_{i_1} = 2(2P_2 + Q_{i_2}) + Q_{i_1} = \cdots$$

经过 $n$ 步：
$$P = 2^n P_n + \sum_{j=1}^n 2^{j-1}Q_{i_j}$$

当 $n$ 足够大时，$\hat{h}(P_n)$ 有界。故 $P_n$ 属于有限集，而 $\sum 2^{j-1}Q_{i_j}$ 由有限个 $Q_i$ 生成。

因此 $E(\mathbb{Q})$ 由有限集生成。 $\square$

## 4. 例子

**例 1**：$E: y^2 = x^3 - x$

有理点：$E(\mathbb{Q}) = \{\mathcal{O}, (0,0), (1,0), (-1,0)\} \cong \mathbb{Z}/2\mathbb{Z} \times \mathbb{Z}/2\mathbb{Z}$，秩 $r = 0$。 $\square$

**例 2**：$E: y^2 = x^3 + 1$

挠点：$E(\mathbb{Q})_{\text{tors}} = \{\mathcal{O}, (0,\pm 1), (-1,0), (2,\pm 3)\} \cong \mathbb{Z}/6\mathbb{Z}$，秩 $r = 0$。 $\square$

**例 3**：$E: y^2 = x^3 - 2$

由Siegel定理或 descent 论证，$E(\mathbb{Q}) = \{\mathcal{O}\}$。 $\square$

## 5. BSD猜想

**猜想 5.1**（Birch-Swinnerton-Dyer）
设 $E/\mathbb{Q}$ 是椭圆曲线，$L(E,s)$ 是Hasse-Weil $L$-函数，则：
$$\text{rank}(E(\mathbb{Q})) = \text{ord}_{s=1} L(E,s)$$

且 leading coefficient 满足：
$$\lim_{s \to 1} \frac{L(E,s)}{(s-1)^r} = \frac{\Omega_E \cdot \text{Reg}_E \cdot \#Ш(E/\mathbb{Q}) \cdot \prod_p c_p}{\#E(\mathbb{Q})_{\text{tors}}^2}$$

其中 $Ш(E/\mathbb{Q})$ 是Tate-Shafarevich群，$\text{Reg}_E$ 是调节子，$c_p$ 是Tamagawa数。

**已知结果**：
- Coates-Wiles（1977）：对复乘椭圆曲线，若 $L(E,1) \neq 0$ 则 $r = 0$
- Gross-Zagier（1986）：$L'(E,1)$ 与Heegner点的关系
- Kolyvagin（1988）：$r \leq 1$ 时BSD秩公式成立
