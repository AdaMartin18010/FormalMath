# 模论基础

## 定义

### 模

设$R$是环，$R$-模$M$是配备标量乘法$R \times M \to M$的交换群，满足：

- $r(m_1 + m_2) = rm_1 + rm_2$
- $(r_1 + r_2)m = r_1m + r_2m$
- $(r_1r_2)m = r_1(r_2m)$
- $1m = m$（若$R$有单位元）

### 例子

- $R^n$：自由模
- 向量空间：域上的模
- 理想：环作为自身的模

### 同态

$f: M \to N$是$R$-模同态若$f(rm_1 + m_2) = rf(m_1) + f(m_2)$。
