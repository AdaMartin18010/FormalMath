## 📚 概述

同调代数是20世纪中期发展起来的一门数学分支，它通过代数方法研究拓扑空间、代数结构和几何对象的不变量。同调代数提供了一套统一的语言和工具，将拓扑学、代数学和几何学中的许多问题转化为代数问题。

同调代数的核心思想是构造"链复形"并研究其"同调群"，这些群捕捉了原对象的重要结构信息。同调代数在代数拓扑、代数几何、表示论和数论中都有着不可或缺的地位。

## 🏗️ 核心概念

### 链复形

**定义 1.1（链复形）**：
R-模的**链复形** $(C_\bullet, d_\bullet)$ 是模和同态的序列：
$$\cdots \xrightarrow{d_{n+2}} C_{n+1} \xrightarrow{d_{n+1}} C_n \xrightarrow{d_n} C_{n-1} \xrightarrow{d_{n-1}} \cdots$$

满足：**$d_n \circ d_{n+1} = 0$**（即 $\text{Im}(d_{n+1}) \subseteq \text{Ker}(d_n)$）

**术语**：

- $C_n$：$n$-链模
- $d_n$：边缘算子/微分
- $Z_n = \text{Ker}(d_n)$：$n$-闭链
- $B_n = \text{Im}(d_{n+1})$：$n$-边缘链

### 同调群

**定义 1.2（同调群）**：
链复形 $C_\bullet$ 的**$n$次同调**：
$$H_n(C_\bullet) = Z_n / B_n = \text{Ker}(d_n) / \text{Im}(d_{n+1})$$

**意义**：

- 衡量"闭链"中有多少不是"边缘"
- 捕捉空间的"洞"的信息

**上链复形**：
$$\cdots \xrightarrow{\delta^{n-2}} C^{n-1} \xrightarrow{\delta^{n-1}} C^n \xrightarrow{\delta^n} C^{n+1} \xrightarrow{\delta^{n+1}} \cdots$$

**上同调**：$H^n(C^\bullet) = \text{Ker}(\delta^n) / \text{Im}(\delta^{n-1})$

### 正合序列

**定义 1.3（正合）**：
序列 $A \xrightarrow{f} B \xrightarrow{g} C$ 在 $B$ **正合**，如果 $\text{Im}(f) = \text{Ker}(g)$。

**短正合序列**：
$$0 \to A \xrightarrow{i} B \xrightarrow{p} C \to 0$$

- $i$ 单射
- $p$ 满射
- $\text{Im}(i) = \text{Ker}(p)$，即 $C \cong B/A$

**分裂**：若存在 $s: C \to B$ 使 $p \circ s = \text{id}_C$，则 $B \cong A \oplus C$。

## 📊 导出函子

### 投射分解

**定义 2.1（投射模）**：
模 $P$ 是**投射**的，如果：

- 任意满射 $M \to N$ 和映射 $P \to N$ 可提升为 $P \to M$
- 等价：$P$ 是自由模的直和项

**定义 2.2（投射分解）**：
模 $M$ 的**投射分解**是正合序列：
$$\cdots \to P_2 \to P_1 \to P_0 \to M \to 0$$

其中每个 $P_i$ 投射。

**存在性**：R-模都有投射分解（自由分解）。

### Ext与Tor

**定义 2.3（左导出函子）**：
设 $F$ 是右正合加性函子，$L_nF$ 是第 $n$ 左导出函子。

**Tor函子**：
$$\text{Tor}_n^R(M, N) = L_n(- \otimes_R N)(M)$$

**Ext函子**：
$$\text{Ext}_R^n(M, N) = R^n\text{Hom}_R(M, -)(N)$$

性质：

- $\text{Tor}_0^R(M, N) = M \otimes_R N$
- $\text{Ext}_R^0(M, N) = \text{Hom}_R(M, N)$

### 长正合序列

**定理 2.1**：
短正合序列 $0 \to A \to B \to C \to 0$ 诱导长正合序列：
$$\cdots \to L_{n+1}F(C) \to L_nF(A) \to L_nF(B) \to L_nF(C) \to L_{n-1}F(A) \to \cdots$$

**对Ext**：
$$0 \to \text{Hom}(C, N) \to \text{Hom}(B, N) \to \text{Hom}(A, N) \to \text{Ext}^1(C, N) \to \cdots$$

## 🔍 重要定理

### 蛇引理

**定理 3.1（蛇引理）**：
给定交换图：

```
A ──→ B ──→ C ──→ 0
│      │      │
↓f     ↓g     ↓h
0 ──→ A' ──→ B' ──→ C'
```

两行正合，则有正合序列：
$$\text{Ker}(f) \to \text{Ker}(g) \to \text{Ker}(h) \xrightarrow{\delta} \text{Coker}(f) \to \text{Coker}(g) \to \text{Coker}(h)$$

**应用**：构造连接同态，证明长正合序列。

### 五引理

**定理 3.2（五引理）**：
给定交换图：

```
A₁ ──→ A₂ ──→ A₃ ──→ A₄ ──→ A₅
│       │       │       │       │
↓f₁     ↓f₂     ↓f₃     ↓f₄     ↓f₅
B₁ ──→ B₂ ──→ B₃ ──→ B₄ ──→ B₅
```

两行正合，$f_1, f_2, f_4, f_5$ 同构，则 $f_3$ 也是同构。

### 万有系数定理

**定理 3.3**：
对拓扑空间 $X$：
$$H_n(X; G) \cong (H_n(X) \otimes G) \oplus \text{Tor}(H_{n-1}(X), G)$$

**上同调版本**：
$$H^n(X; G) \cong \text{Hom}(H_n(X), G) \oplus \text{Ext}(H_{n-1}(X), G)$$

## 💡 详细示例

### 示例1：圆周的同调

$S^1$ 的链复形：
$$0 \to \mathbb{Z}[e] \xrightarrow{\partial} \mathbb{Z}[v] \to 0$$

其中 $\partial(e) = v - v = 0$。

同调：

- $H_0 = \mathbb{Z}[v] / 0 = \mathbb{Z}$
- $H_1 = \mathbb{Z}[e] / 0 = \mathbb{Z}$
- $H_n = 0$（$n \geq 2$）

### 示例2：Klein瓶

用胞腔分解计算：

- 1个0-胞腔
- 2个1-胞腔 $a, b$
- 1个2-胞腔，边界 $aba^{-1}b$

链复形：
$$0 \to \mathbb{Z} \xrightarrow{\partial_2} \mathbb{Z}^2 \xrightarrow{\partial_1} \mathbb{Z} \to 0$$

微分矩阵：

- $\partial_1 = 0$
- $\partial_2 = \begin{pmatrix} 2 & 0 \end{pmatrix}$

同调：

- $H_0 = \mathbb{Z}$
- $H_1 = \mathbb{Z} \oplus \mathbb{Z}_2$
- $H_2 = 0$

### 示例3：Ext的计算

计算 $\text{Ext}^1_\mathbb{Z}(\mathbb{Z}_m, \mathbb{Z}_n)$。

用投射分解：
$$0 \to \mathbb{Z} \xrightarrow{m} \mathbb{Z} \to \mathbb{Z}_m \to 0$$

应用 $\text{Hom}(-, \mathbb{Z}_n)$：
$$0 \to \text{Hom}(\mathbb{Z}_m, \mathbb{Z}_n) \to \mathbb{Z}_n \xrightarrow{m} \mathbb{Z}_n \to \text{Ext}^1(\mathbb{Z}_m, \mathbb{Z}_n) \to 0$$

因此：
$$\text{Ext}^1_\mathbb{Z}(\mathbb{Z}_m, \mathbb{Z}_n) \cong \mathbb{Z}_n / m\mathbb{Z}_n \cong \mathbb{Z}_{(m,n)}$$

## 📚 总结

### 核心概念图

```
链复形
  ↓
同调群 H_n = Ker d_n / Im d_{n+1}
  ↓
导出函子（Ext, Tor）
  ↓
长正合序列
  ↓
拓扑/代数应用
```

### 重要函子

| 函子 | 定义 | 性质 |
|-----|------|------|
| Tor | 张量积的左导出 | Tor₀ = ⊗ |
| Ext | Hom的右导出 | Ext⁰ = Hom |
