## 一、核心定理完整证明

### 1.1 伽罗瓦理论基本定理完整证明

**定理 1.1** (伽罗瓦理论基本定理 / Fundamental Theorem of Galois Theory)

设 $F/K$ 是有限伽罗瓦扩张，$G = \text{Gal}(F/K)$。则：

1. 存在一一对应（伽罗瓦对应）：
   $$\{\text{中间域 } L : K \subseteq L \subseteq F\} \leftrightarrow \{\text{子群 } H : H \leq G\}$$

   具体地：
   - $L \mapsto \text{Gal}(F/L) = \{\sigma \in G : \sigma|_L = \text{id}_L\}$
   - $H \mapsto F^H = \{x \in F : \sigma(x) = x \text{ 对所有 } \sigma \in H\}$

2. **包含关系反转**：$L_1 \subseteq L_2$ 当且仅当 $\text{Gal}(F/L_2) \leq \text{Gal}(F/L_1)$

3. **次数与指数关系**：$[L : K] = [G : \text{Gal}(F/L)]$，$|H| = [F : F^H]$

4. **正规性对应**：$L/K$ 是正规扩张当且仅当 $\text{Gal}(F/L) \trianglelefteq G$。此时：
   $$\text{Gal}(L/K) \cong G/\text{Gal}(F/L)$$

**前置定义与引理**：

**定义**：扩张 $F/K$ 是**伽罗瓦扩张**，如果它是正规且可分的。

**定义**：扩张是**正规**的，如果每个不可约多项式 $f(x) \in K[x]$ 只要在 $F$ 中有根，就在 $F$ 中分裂。

**定义**：扩张是**可分**的，如果每个元素在 $K$ 上的最小多项式没有重根。

**引理 1.1** (Artin引理)：设 $F$ 是域，$G$ 是 $\text{Aut}(F)$ 的有限子群，$K = F^G$。则 $[F : K] \leq |G|$。

**引理 1.2**：有限扩张 $F/K$ 是伽罗瓦扩张当且仅当 $|\text{Gal}(F/K)| = [F : K]$。

---

**定理 1.1 的完整证明**：

**步骤 1**：证明 $\text{Gal}(F/F^H) = H$ 且 $[F : F^H] = |H|$

设 $H \leq G$，$L = F^H$。由定义，$H \subseteq \text{Gal}(F/L)$。

由 Artin 引理，$[F : L] \leq |H|$。

由引理 1.2（对伽罗瓦扩张 $F/K$），$|G| = [F : K]$。

由塔定律，$[F : K] = [F : L][L : K]$。

由定义，$|G| = |H| \cdot [G : H]$。

注意到 $[G : H] = |G|/|H| \geq [F : K]/[F : L] = [L : K]$。

但 $H \subseteq \text{Gal}(F/L)$ 意味着 $|H| \leq |\text{Gal}(F/L)| \leq [F : L]$（引理 1.2 的不等式方向）。

结合 $[F : L] \leq |H|$，得 $[F : L] = |H|$ 且 $\text{Gal}(F/L) = H$。

**步骤 2**：证明 $F^{\text{Gal}(F/L)} = L$

设 $L'$ 是中间域，$H = \text{Gal}(F/L')$。由步骤 1，$F^H = L'$。

这建立了双向对应。

**步骤 3**：证明包含关系反转

设 $L_1 \subseteq L_2$。若 $\sigma \in \text{Gal}(F/L_2)$，则 $\sigma$ 固定 $L_2$，从而固定 $L_1$，所以 $\sigma \in \text{Gal}(F/L_1)$。

因此 $\text{Gal}(F/L_2) \leq \text{Gal}(F/L_1)$。

反之，设 $\text{Gal}(F/L_2) \leq \text{Gal}(F/L_1)$。则 $L_1 = F^{\text{Gal}(F/L_1)} \subseteq F^{\text{Gal}(F/L_2)} = L_2$。

**步骤 4**：证明次数与指数关系

由步骤 1，$|H| = [F : F^H]$。

对中间域 $L$，$[L : K] = [F : K]/[F : L] = |G|/|\text{Gal}(F/L)| = [G : \text{Gal}(F/L)]$。

**步骤 5**：证明正规性对应

($\Rightarrow$) 设 $L/K$ 是正规扩张。对任意 $\sigma \in G$，$\sigma(L)$ 是 $K$ 的共轭域。由于 $L/K$ 正规，$\sigma(L) = L$。

因此 $\sigma \text{Gal}(F/L) \sigma^{-1} = \text{Gal}(F/\sigma(L)) = \text{Gal}(F/L)$，所以 $\text{Gal}(F/L) \trianglelefteq G$。

($\Leftarrow$) 设 $\text{Gal}(F/L) \trianglelefteq G$。对任意 $\sigma \in G$，$\sigma \text{Gal}(F/L) \sigma^{-1} = \text{Gal}(F/L)$。

由包含关系反转，$\sigma(L) = L$，即 $L$ 在 $G$ 作用下不变。

设 $\alpha \in L$，$f(x)$ 是其在 $K$ 上的最小多项式。$f$ 的根是 $\{\sigma(\alpha) : \sigma \in G\}$（因为 $F/K$ 正规）。

由于每个 $\sigma(\alpha) \in L$，$f$ 在 $L$ 中分裂，所以 $L/K$ 正规。

**步骤 6**：证明同构 $\text{Gal}(L/K) \cong G/\text{Gal}(F/L)$

定义 $\phi: G \to \text{Gal}(L/K)$ 为 $\phi(\sigma) = \sigma|_L$（限制映射）。

由于 $L/K$ 正规，$\sigma(L) = L$，所以 $\sigma|_L \in \text{Gal}(L/K)$。

- $\phi$ 是同态：$\phi(\sigma_1 \sigma_2) = (\sigma_1 \sigma_2)|_L = \sigma_1|_L \circ \sigma_2|_L = \phi(\sigma_1)\phi(\sigma_2)$
- $\ker(\phi) = \{\sigma \in G : \sigma|_L = \text{id}_L\} = \text{Gal}(F/L)$
- $\phi$ 是满射：对任意 $\tau \in \text{Gal}(L/K)$，由于 $F/L$ 是伽罗瓦扩张，可以将 $\tau$ 扩张为 $F$ 的自同构（嵌入扩张定理）。

由第一同构定理，$G/\text{Gal}(F/L) \cong \text{Gal}(L/K)$。

**证毕**。

### 1.2 代数基本定理的域论证明

**定理 1.2** (代数基本定理 / Fundamental Theorem of Algebra)

复数域 $\mathbb{C}$ 是代数闭域，即每个非常数多项式 $f(x) \in \mathbb{C}[x]$ 在 $\mathbb{C}$ 中有根。

**证明**：

**步骤 1**：只需证明每个实系数多项式在 $\mathbb{C}$ 中有根

若 $f(x) \in \mathbb{C}[x]$，考虑 $\overline{f}(x)$（系数取共轭），则 $f(x)\overline{f}(x) \in \mathbb{R}[x]$。

若 $f(x)\overline{f}(x)$ 在 $\mathbb{C}$ 中有根 $\alpha$，则 $f(\alpha) = 0$ 或 $\overline{f}(\alpha) = 0$。

若 $\overline{f}(\alpha) = 0$，则 $f(\overline{\alpha}) = \overline{\overline{f}(\alpha)} = 0$。

因此只需证明实系数多项式在 $\mathbb{C}$ 中有根。

**步骤 2**：实系数奇次多项式有实根

由介值定理，奇次实系数多项式在 $\mathbb{R}$ 中有根。

**步骤 3**：$\mathbb{C} = \mathbb{R}(i)$ 且 $[\mathbb{C} : \mathbb{R}] = 2$

**步骤 4**：设 $f(x) \in \mathbb{R}[x]$，$E$ 是 $f$ 在 $\mathbb{C}$ 上的分裂域，$G = \text{Gal}(E/\mathbb{R})$。

设 $|G| = 2^n m$，$m$ 为奇数。

**步骤 5**：$G$ 有西罗 2-子群 $P$，$|P| = 2^n$

由伽罗瓦对应，存在中间域 $\mathbb{R} \subseteq K \subseteq E$ 使得 $[K : \mathbb{R}] = m$。

**步骤 6**：$K/\mathbb{R}$ 是可分扩张（特征0），由本原元素定理，$K = \mathbb{R}(\alpha)$。

$\alpha$ 的最小多项式次数为 $m$（奇数）。由步骤 2，该多项式在 $\mathbb{R}$ 中有根，所以 $m = 1$。

因此 $|G| = 2^n$。

**步骤 7**：若 $n = 0$，则 $E = \mathbb{R} \subseteq \mathbb{C}$，证毕。

若 $n \geq 1$，$G$ 有指数为2的子群（2-群性质）。由伽罗瓦对应，存在中间域 $\mathbb{R} \subseteq L \subseteq E$ 使得 $[L : \mathbb{R}] = 2$。

**步骤 8**：$L = \mathbb{R}(\sqrt{d})$，$d < 0$（因为 $\mathbb{R}$ 上仅有的2次扩张是 $\mathbb{R}(\sqrt{d})$，$d < 0$ 时 $\sqrt{d} = i\sqrt{|d|} \in \mathbb{C}$）

因此 $L \subseteq \mathbb{C}$。

**步骤 9**：继续此过程，最终得 $E \subseteq \mathbb{C}$。由于 $f$ 在 $E$ 中分裂，$f$ 在 $\mathbb{C}$ 中有根。

**证毕**。

### 1.3 有限域结构定理完整证明

**定理 1.3** (有限域结构定理 / Structure Theorem for Finite Fields)

1. **存在性**：对每个素数幂 $q = p^n$，存在唯一的（同构意义下）有限域 $\mathbb{F}_q$，其中 $|\mathbb{F}_q| = q$。

2. **乘法群结构**：$\mathbb{F}_q^*$ 是 $q-1$ 阶循环群。

3. **子域结构**：$\mathbb{F}_{p^m} \subseteq \mathbb{F}_{p^n}$ 当且仅当 $m \mid n$。

**完整证明**：

**部分 1：存在性**

考虑多项式 $f(x) = x^{p^n} - x \in \mathbb{F}_p[x]$。

设 $F$ 是 $f(x)$ 在 $\mathbb{F}_p$ 上的分裂域。

$f'(x) = p^n x^{p^n-1} - 1 = -1 \neq 0$，所以 $f(x)$ 没有重根。

设 $S = \{\alpha \in F : f(\alpha) = 0\}$，则 $|S| = p^n$。

**验证 $S$ 是域**：

- $0, 1 \in S$（显然）
- 若 $\alpha, \beta \in S$，则 $(\alpha - \beta)^{p^n} = \alpha^{p^n} - \beta^{p^n} = \alpha - \beta$，所以 $\alpha - \beta \in S$
- 若 $\alpha, \beta \in S$，$\beta \neq 0$，则 $(\alpha/\beta)^{p^n} = \alpha^{p^n}/\beta^{p^n} = \alpha/\beta$，所以 $\alpha/\beta \in S$

因此 $S$ 是 $F$ 的子域，且 $f(x)$ 在 $S$ 中分裂。由分裂域的极小性，$F = S$。

**唯一性**：

设 $F_1, F_2$ 都是 $p^n$ 元域。则 $F_1^*$ 和 $F_2^*$ 都是 $p^n - 1$ 阶循环群。

$F_1$ 是 $x^{p^n} - x$ 在 $\mathbb{F}_p$ 上的分裂域，同理 $F_2$ 也是。

分裂域在 $\mathbb{F}_p$ 上同构，所以 $F_1 \cong F_2$。

**部分 2：乘法群结构**

设 $\alpha \in \mathbb{F}_q^*$，则 $\alpha^{q-1} = 1$。

由有限交换群结构定理，$\mathbb{F}_q^* \cong \mathbb{Z}_{d_1} \times \cdots \times \mathbb{Z}_{d_k}$，$d_1 \mid \cdots \mid d_k$，$d_1 \cdots d_k = q-1$。

所有元素的阶整除 $d_k$，所以每个元素满足 $x^{d_k} - 1 = 0$。

多项式 $x^{d_k} - 1$ 在 $\mathbb{F}_q$ 中至多有 $d_k$ 个根，所以 $q-1 \leq d_k$。

但 $d_k \mid q-1$，所以 $d_k = q-1$，$k = 1$，即 $\mathbb{F}_q^*$ 是循环群。

**部分 3：子域结构**

($\Rightarrow$) 设 $\mathbb{F}_{p^m} \subseteq \mathbb{F}_{p^n}$。则 $[\mathbb{F}_{p^n} : \mathbb{F}_{p^m}] = n/m$，所以 $m \mid n$。

($\Leftarrow$) 设 $m \mid n$。则 $p^m - 1 \mid p^n - 1$，所以 $x^{p^m-1} - 1 \mid x^{p^n-1} - 1$。

因此 $x^{p^m} - x \mid x^{p^n} - x$。

$\mathbb{F}_{p^n}$ 是 $x^{p^n} - x$ 的分裂域，包含 $x^{p^m} - x$ 的所有根，即包含 $\mathbb{F}_{p^m}$。

**证毕**。

## 二、具体例子库

### 2.1 二次扩张

**例子 2.1**：$\mathbb{Q}(\sqrt{2})/\mathbb{Q}$

**性质**：

- 伽罗瓦扩张（正规、可分）
- $[\mathbb{Q}(\sqrt{2}) : \mathbb{Q}] = 2$
- 自同构群：$\text{Gal}(\mathbb{Q}(\sqrt{2})/\mathbb{Q}) = \{\text{id}, \sigma\} \cong \mathbb{Z}_2$

其中 $\sigma(\sqrt{2}) = -\sqrt{2}$，$\sigma(a + b\sqrt{2}) = a - b\sqrt{2}$。

**伽罗瓦对应**：

| 子群 | 固定域 |
|-----|-------|
| $\{id\}$ | $\mathbb{Q}(\sqrt{2})$ |
| $\{id, \sigma\}$ | $\mathbb{Q}$ |

---

**例子 2.2**：$\mathbb{Q}(\sqrt{2}, \sqrt{3})/\mathbb{Q}$

**性质**：

- 伽罗瓦扩张
- $[\mathbb{Q}(\sqrt{2}, \sqrt{3}) : \mathbb{Q}] = 4$
- 自同构群：$\text{Gal} \cong \mathbb{Z}_2 \times \mathbb{Z}_2$（Klein四元群）

**自同构**：

- $\sigma: \sqrt{2} \mapsto -\sqrt{2}$，$\sqrt{3} \mapsto \sqrt{3}$
- $\tau: \sqrt{2} \mapsto \sqrt{2}$，$\sqrt{3} \mapsto -\sqrt{3}$
- $\sigma\tau: \sqrt{2} \mapsto -\sqrt{2}$，$\sqrt{3} \mapsto -\sqrt{3}$

**伽罗瓦对应**：

| 子群 | 固定域 | 次数 $[L:\mathbb{Q}]$ |
|-----|-------|---------------------|
| $\{id\}$ | $\mathbb{Q}(\sqrt{2}, \sqrt{3})$ | 4 |
| $\langle \sigma \rangle$ | $\mathbb{Q}(\sqrt{3})$ | 2 |
| $\langle \tau \rangle$ | $\mathbb{Q}(\sqrt{2})$ | 2 |
| $\langle \sigma\tau \rangle$ | $\mathbb{Q}(\sqrt{6})$ | 2 |
| $\langle \sigma, \tau \rangle$ | $\mathbb{Q}$ | 1 |

---

### 2.2 分圆扩张

**例子 2.3**：$\mathbb{Q}(\zeta_5)/\mathbb{Q}$

其中 $\zeta_5 = e^{2\pi i/5}$ 是本原5次单位根。

**性质**：

- 伽罗瓦扩张
- $[\mathbb{Q}(\zeta_5) : \mathbb{Q}] = \varphi(5) = 4$
- 自同构群：$\text{Gal} \cong (\mathbb{Z}/5\mathbb{Z})^* \cong \mathbb{Z}_4$

**自同构**：$\sigma_k(\zeta_5) = \zeta_5^k$，$k = 1, 2, 3, 4$

**伽罗瓦对应**：

| 子群 | 固定域 | 次数 |
|-----|-------|-----|
| $\{id\}$ | $\mathbb{Q}(\zeta_5)$ | 4 |
| $\langle \sigma_4 \rangle \cong \mathbb{Z}_2$ | $\mathbb{Q}(\zeta_5 + \zeta_5^{-1}) = \mathbb{Q}(\cos(2\pi/5))$ | 2 |
| $\langle \sigma_2 \rangle \cong \mathbb{Z}_4$ | $\mathbb{Q}$ | 1 |

---

### 2.3 三次扩张

**例子 2.4**：$\mathbb{Q}(\sqrt[3]{2})/\mathbb{Q}$

**性质**：

- **不是**伽罗瓦扩张（不正规）
- $[\mathbb{Q}(\sqrt[3]{2}) : \mathbb{Q}] = 3$

**最小多项式**：$x^3 - 2$，根为 $\sqrt[3]{2}$，$\sqrt[3]{2}\omega$，$\sqrt[3]{2}\omega^2$，其中 $\omega = e^{2\pi i/3}$。

**伽罗瓦闭包**：$\mathbb{Q}(\sqrt[3]{2}, \omega)/\mathbb{Q}$，次数为6。

---

**例子 2.5**：$\mathbb{Q}(\sqrt[3]{2}, \omega)/\mathbb{Q}$

**性质**：

- 伽罗瓦扩张
- 次数：$[\mathbb{Q}(\sqrt[3]{2}, \omega) : \mathbb{Q}] = 6$
- 自同构群：$\text{Gal} \cong S_3$

**自同构**：

- 置换三个根 $\sqrt[3]{2}$，$\sqrt[3]{2}\omega$，$\sqrt[3]{2}\omega^2$

**子群与固定域**：

| 子群 | 固定域 | 是否正规 |
|-----|-------|---------|
| $\{id\}$ | $\mathbb{Q}(\sqrt[3]{2}, \omega)$ | 是 |
| $A_3 \cong \mathbb{Z}_3$ | $\mathbb{Q}(\omega)$ | 是 |
| 三个 $\mathbb{Z}_2$ | $\mathbb{Q}(\sqrt[3]{2})$，$\mathbb{Q}(\sqrt[3]{2}\omega)$，$\mathbb{Q}(\sqrt[3]{2}\omega^2)$ | 否 |
| $S_3$ | $\mathbb{Q}$ | 是 |

## 三、反例集锦

### 3.1 非正规扩张

**反例 3.1**：$\mathbb{Q}(\sqrt[3]{2})/\mathbb{Q}$ 不是正规扩张。

**证明**：$x^3 - 2$ 在 $\mathbb{Q}$ 上不可约，有一个根 $\sqrt[3]{2} \in \mathbb{Q}(\sqrt[3]{2})$，但其他两个根（复数）不在此域中。

---

### 3.2 不可分扩张

**反例 3.2**：在特征 $p$ 域中，扩张可能不可分。

设 $\mathbb{F}_p(t)$ 是有理函数域。考虑 $f(x) = x^p - t \in \mathbb{F}_p(t)[x]$。

设 $\alpha$ 是 $f$ 的根，则 $\alpha^p = t$，$f(x) = x^p - \alpha^p = (x - \alpha)^p$。

$f$ 有重根，所以 $f$ 不可分。

---

### 3.3 伽罗瓦群小于扩张次数

**反例 3.3**：$\mathbb{Q}(\sqrt[3]{2})/\mathbb{Q}$ 中，$|\text{Gal}| = 1 < 3 = [\mathbb{Q}(\sqrt[3]{2}) : \mathbb{Q}]$。

**原因**：扩张不正规。

```python
# SageMath域论计算示例

# 1. 定义数域

for H in G.subgroups():
    print(f"  子群: {H}, 固定域: {H.fixed_field()}")
```

```python
from sympy import *

alpha = sqrt(2) + sqrt(3)
```

### 习题参考答案（部分）

**习题 5.1.1 解答**：

$\mathbb{Q}(\sqrt{2}, \sqrt{3})$ 在 $\mathbb{Q}$ 上的基是 $\{1, \sqrt{2}, \sqrt{3}, \sqrt{6}\}$。

**验证线性无关性**：设 $a + b\sqrt{2} + c\sqrt{3} + d\sqrt{6} = 0$，$a, b, c, d \in \mathbb{Q}$。

整理得：$(a + b\sqrt{2}) + (c + d\sqrt{2})\sqrt{3} = 0$。

若 $c + d\sqrt{2} \neq 0$，则 $\sqrt{3} = -\frac{a + b\sqrt{2}}{c + d\sqrt{2}} \in \mathbb{Q}(\sqrt{2})$，矛盾（因为 $[\mathbb{Q}(\sqrt{2}, \sqrt{3}) : \mathbb{Q}] = 4 > 2 = [\mathbb{Q}(\sqrt{2}) : \mathbb{Q}]$）。

所以 $c + d\sqrt{2} = 0$，从而 $a + b\sqrt{2} = 0$。

由 $\{1, \sqrt{2}\}$ 在 $\mathbb{Q}$ 上线性无关，$a = b = 0$，同理 $c = d = 0$。

**习题 5.2.2 解答**：

$x^3 - 2$ 在 $\mathbb{Q}$ 上不可约（Eisenstein判别法，$p = 2$），有一个根 $\sqrt[3]{2} \in \mathbb{Q}(\sqrt[3]{2})$。

但其他两个根是 $\sqrt[3]{2}\omega$，$\sqrt[3]{2}\omega^2$，其中 $\omega = e^{2\pi i/3} = \frac{-1 + i\sqrt{3}}{2}$ 不在 $\mathbb{Q}(\sqrt[3]{2}) \subseteq \mathbb{R}$ 中。

因此 $\mathbb{Q}(\sqrt[3]{2})$ 不包含 $x^3 - 2$ 的所有根，扩张不正规。
