# 代数扩张理论 - 深度版

> **"域扩张是代数结构的'几何化'，它将抽象的代数问题转化为域包含的结构分析。"** —— 继承伽罗瓦的革命性思想

---

## 📚 概述

代数扩张理论是域论的核心分支，研究域的代数扩张的结构和性质。这一理论由Evariste Galois在19世纪初创立，彻底解决了代数方程的可解性问题，并为现代代数学、代数数论和代数几何奠定了基础。

---

## 🕰️ 历史发展与数学家理念

### 伽罗瓦的革命性突破

**伽罗瓦数学理念**的核心在于**通过对称性研究代数结构**：

1. **域与群的对应**：域扩张$K/F$ ↔ 伽罗瓦群$Gal(K/F)$
2. **可解性判据**：多项式根式可解当且仅当伽罗瓦群可解
3. **结构主义先驱**：伽罗瓦的抽象方法预示了现代代数学的发展方向

### 格洛腾迪克的相对观点

**格洛腾迪克数学理念**强调**相对性**和**函子性**：

- 域扩张应视为态射$F \to K$
- Galois理论是更一般的étale基本理论的特例
- 相对维数和光滑性概念

### Dedekind和Weber的贡献

Richard Dedekind在19世纪后期对伽罗瓦理论进行了现代化表述，引入了：

- 域的抽象概念
- 理想理论
- 现代伽罗瓦对应

## 🏗️ 核心概念与完整证明

### 1. 代数扩张的基本定义

**定义 9.1**（代数元与超越元）
设$K/F$是域扩张，$\alpha \in K$。

- $\alpha$称为**代数元**，如果存在非零多项式$f(x) \in F[x]$使得$f(\alpha) = 0$
- $\alpha$称为**超越元**，如果它不是代数元

**定义 9.2**（代数扩张）
扩张$K/F$称为**代数扩张**，如果$K$中每个元素都是$F$上的代数元。

**定义 9.3**（最小多项式）
设$\alpha$是$F$上的代数元，**最小多项式**$\min_F(\alpha)$是$F[x]$中以$\alpha$为根的不可约首一多项式。

**定理 9.1**：最小多项式存在且唯一。

### 2. 单扩张结构

**定理 9.2**（单扩张结构）
设$\alpha$是$F$上的代数元，$n = \deg(\min_F(\alpha))$。则：
$$F(\alpha) \cong F[x] / (\min_F(\alpha))$$

且$[F(\alpha) : F] = n$，基为$\{1, \alpha, \ldots, \alpha^{n-1}\}$。

**证明**：

**步骤1**：构造同态。
$$\phi: F[x] \to F(\alpha), \quad f(x) \mapsto f(\alpha)$$

**步骤2**：确定核。
$$\ker(\phi) = (\min_F(\alpha))$$

由定义，$\min_F(\alpha)$是$\ker(\phi)$中次数最小的非零首一多项式，且$F[x]$是PID，故$\ker(\phi) = (\min_F(\alpha))$。

**步骤3**：应用同态基本定理。
$$F[x]/(\min_F(\alpha)) \cong \text{Im}(\phi) = F(\alpha)$$

**步骤4**：确定基。
由除法算法，$F[x]/(p(x))$中每个元素唯一表示为：
$$a_0 + a_1\bar{x} + \cdots + a_{n-1}\bar{x}^{n-1}$$
其中$\bar{x}$是$x$的等价类。

对应到$F(\alpha)$：
$$F(\alpha) = \{a_0 + a_1\alpha + \cdots + a_{n-1}\alpha^{n-1} : a_i \in F\}$$

**证毕**。

### 3. 代数闭包的存在性

**定义 9.4**（代数闭域）
域$F$称为**代数闭域**，如果$F[x]$中每个非常数多项式在$F$中有根。

**等价条件**：$F[x]$中不可约多项式都是一次的。

**定理 9.3**（代数闭包存在性）
每个域$F$都存在代数闭包$\overline{F}$，即包含$F$的代数闭代数扩张，且在$F$-同构意义下唯一。

**证明概要**：

**步骤1**：构造单扩张塔。

- 枚举$F[x]$中所有非常数多项式：$f_1, f_2, \ldots$
- 构造$F_1$使得$f_1$在$F_1$中有根
- 归纳构造$F_n$使得$f_1, \ldots, f_n$在$F_n$中分裂
- $F^{(1)} = \bigcup_n F_n$

**步骤2**：迭代构造。
构造塔$F \subseteq F^{(1)} \subseteq F^{(2)} \subseteq \cdots$

**步骤3**：取并集。
$$\overline{F} = \bigcup_{i=1}^{\infty} F^{(i)}$$

**步骤4**：验证代数闭性。
$\overline{F}$中任何多项式的系数在某$F^{(i)}$中，故在$F^{(i+1)}$中分裂。

**步骤5**：唯一性。
使用Zorn引理证明任何两个代数闭包$F$-同构。

**证毕**。

### 4. 可分扩张

**定义 9.5**（可分元与可分扩张）
设$K/F$是代数扩张。

- 不可约多项式$f(x) \in F[x]$称为**可分**的，如果在分裂域中无重根
- $\alpha \in K$称为**可分元**，如果$\min_F(\alpha)$可分
- $K/F$称为**可分扩张**，如果每个元素可分

**定义 9.6**（完全域）
域$F$称为**完全域**，如果$F[x]$中每个不可约多项式都可分。

**定理 9.4**：特征0域都是完全域。

**证明**：特征0时，$f'(x) = 0$当且仅当$f$是常数。若$f$不可约，则$\gcd(f, f') = 1$，故$f$无重根。

**定理 9.5**：特征$p$域$F$完全当且仅当$F = F^p$（每个元素都是$p$次幂）。

### 5. 本原元定理

**定理 9.6**（本原元定理）
有限可分扩张$K/F$是单扩张，即存在$\alpha \in K$使得$K = F(\alpha)$。

**证明**（$F$无限情形）：

设$K = F(\alpha_1, \ldots, \alpha_n)$，对$n$归纳。

**步骤1**：只需证$F(\alpha, \beta) = F(\theta)$对某$\theta$。

**步骤2**：设$f, g$分别是$\alpha, \beta$的最小多项式。

在分裂域中，设$f$的根为$\alpha = \alpha_1, \ldots, \alpha_m$，$g$的根为$\beta = \beta_1, \ldots, \beta_n$。

**步骤3**：选择$c \in F$使得：
$$\alpha_i + c\beta_j \neq \alpha + c\beta \quad (j \neq 1)$$

这是可能的（$F$无限）。

**步骤4**：令$\theta = \alpha + c\beta$。

**步骤5**：证明$\beta \in F(\theta)$。

考虑$h(x) = f(\theta - cx) \in F(\theta)[x]$。

$h(\beta) = f(\alpha) = 0$，且对$j \neq 1$，$h(\beta_j) = f(\alpha + c\beta - c\beta_j) \neq 0$。

因此$\gcd(g, h) = x - \beta$在分裂域中，故$\beta \in F(\theta)$。

**步骤6**：$\alpha = \theta - c\beta \in F(\theta)$。

**证毕**。

## 🧮 应用示例

### 经典尺规作图问题

**定理 9.7**：$\sqrt[3]{2}$不能用尺规构造。

**证明**：尺规可构造数属于2-幂次扩张，而$[\mathbb{Q}(\sqrt[3]{2}) : \mathbb{Q}] = 3$不是2的幂。
