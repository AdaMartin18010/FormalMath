# 群的扩张与上同调 - 深度版

> **"上同调是研究代数结构变形与扩张的自然语言。"** —— Alexander Grothendieck

---

## 📚 概述

群的扩张理论研究如何通过较小的群构造更大的群，而群上同调则为这种构造提供了系统化的代数工具。这一理论由Schreier、Eilenberg和MacLane在20世纪30-40年代建立，是现代代数学、代数拓扑和代数数论的基石。

---

## 🕰️ 历史发展与数学家理念

### 格洛腾迪克的上同调革命

**格洛腾迪克数学理念**对上同调理论的发展产生了深远影响：

1. **相对观点**：上同调不应局限于具体构造，而应视为函子
2. **导出范畴**：将上同调纳入导出范畴的框架
3. **层的上同调**：将群上同调推广到拓扑空间的上同调

**格洛腾迪克洞察**：群$G$的上同调$H^n(G, M)$本质上是从群范畴到阿贝尔群范畴的导出函子。

### 伽罗瓦理论与群上同调

**伽罗瓦数学理念**关于域扩张与群对应的思想，与群上同调有密切联系：

- **伽罗瓦上同调**：研究伽罗瓦群的上同调，应用于域论和代数数论
- **Hilbert 90定理**：$H^1(Gal(L/K), L^\times) = 0$
- **Brauer群**：$H^2(Gal(L/K), L^\times)$分类中心单代数

### Eilenberg-MacLane的贡献

Samuel Eilenberg和Saunders MacLane在1945年建立了同调代数的一般框架：

- **导出函子**：系统地定义了$Tor$和$Ext$
- **群上同调**：作为$Ext$函子的特例
- **谱序列**：计算复杂上同调的工具

## 🏗️ 核心概念与完整证明

### 1. 群扩张的定义

**定义 4.1**（群扩张）
设$N$和$Q$是群。$G$称为$N$被$Q$的**扩张**，如果存在短正合序列：
$$1 \longrightarrow N \overset{\iota}{\longrightarrow} G \overset{\pi}{\longrightarrow} Q \longrightarrow 1$$

**等价扩张**：两个扩张$G_1$和$G_2$等价，如果存在同态$\phi: G_1 \to G_2$使得下图交换：

```
1 → N → G₁ → Q → 1
    ↓   ↓φ   ↓
1 → N → G₂ → Q → 1
```

**分裂扩张**：扩张**分裂**如果存在同态$s: Q \to G$使得$\pi \circ s = \text{id}_Q$。此时$G \cong N \rtimes Q$（半直积）。

### 2. 群上同调的定义

**定义 4.2**（群上同调）
设$G$是群，$M$是$G$-模（即$\mathbb{Z}[G]$-模）。**群上同调**$H^n(G, M)$定义为：
$$H^n(G, M) = \text{Ext}^n_{\mathbb{Z}[G]}(\mathbb{Z}, M)$$

其中$\mathbb{Z}$视为平凡$G$-模。

**Bar消解的显式构造**：

设$B_n$是以$G^n$为基的$\mathbb{Z}[G]$-自由模，元素记为$[g_1|\cdots|g_n]$。

边界算子$\partial_n: B_n \to B_{n-1}$：
\begin{align*}
\partial_n([g_1|\cdots|g_n]) &= g_1[g_2|\cdots|g_n] \\
&\quad + \sum_{i=1}^{n-1} (-1)^i [g_1|\cdots|g_ig_{i+1}|\cdots|g_n] \\
&\quad + (-1)^n [g_1|\cdots|g_{n-1}]
\end{align*}

上链复形$C^n(G, M) = \text{Hom}_{\mathbb{Z}[G]}(B_n, M) \cong \text{Map}(G^n, M)$。

上边缘算子$\delta^n: C^n \to C^{n+1}$：
\begin{align*}
(\delta^n f)(g_1, \ldots, g_{n+1}) &= g_1 \cdot f(g_2, \ldots, g_{n+1}) \\
&\quad + \sum_{i=1}^{n} (-1)^i f(g_1, \ldots, g_ig_{i+1}, \ldots, g_{n+1}) \\
&\quad + (-1)^{n+1} f(g_1, \ldots, g_n)
\end{align*}

**定理 4.1**：$H^n(G, M) = Z^n(G, M) / B^n(G, M)$，其中$Z^n = \ker(\delta^n)$是上闭链，$B^n = \text{im}(\delta^{n-1})$是上边缘。

### 3. 低维上同调的解释

**$H^0(G, M)$**：
$$H^0(G, M) = M^G = \{m \in M : g \cdot m = m, \forall g \in G\}$$
即$G$-不变元。

**$H^1(G, M)$**：
1-上闭链是交叉同态：$f: G \to M$满足$f(gh) = f(g) + g \cdot f(h)$
1-上边缘是主交叉同态：$f(g) = g \cdot m - m$

若$M$是平凡$G$-模：$H^1(G, M) = \text{Hom}(G, M)$

**$H^2(G, M)$**（核心应用）：
2-上闭链是因子组$f: G \times G \to M$满足：
$$g \cdot f(h, k) - f(gh, k) + f(g, hk) - f(g, h) = 0$$

2-上边缘是$f(g, h) = g \cdot \alpha(h) - \alpha(gh) + \alpha(g)$对某个$\alpha: G \to M$。

### 4. 群扩张的分类定理

**定理 4.2**（Schreier）：设$N$是阿贝尔群，$Q$是群。则$N$被$Q$的扩张等价类（保持$Q$在$N$上的作用）与$H^2(Q, N)$一一对应。

**完整证明**：

**步骤1**：从扩张构造因子组。

设$1 \to N \to G \to Q \to 1$是扩张。选择截面（集合论意义）$s: Q \to G$。

定义：

- **作用**：$q \cdot n = s(q) n s(q)^{-1}$（共轭作用）
- **因子组**：$f(q_1, q_2) = s(q_1)s(q_2)s(q_1q_2)^{-1} \in N$

**步骤2**：验证因子组条件。

利用结合律$(s(q_1)s(q_2))s(q_3) = s(q_1)(s(q_2)s(q_3))$：
$$f(q_1, q_2)f(q_1q_2, q_3) = (q_1 \cdot f(q_2, q_3))f(q_1, q_2q_3)$$

重排得2-上闭链条件：
$$q_1 \cdot f(q_2, q_3) - f(q_1q_2, q_3) + f(q_1, q_2q_3) - f(q_1, q_2) = 0$$

**步骤3**：不同截面的影响。

若选择另一个截面$s'$，设$s'(q) = \alpha(q)s(q)$，$\alpha: Q \to N$。

计算新因子组$f'$：
$$f'(q_1, q_2) = f(q_1, q_2) + q_1 \cdot \alpha(q_2) - \alpha(q_1q_2) + \alpha(q_1)$$

因此$f' - f = \delta\alpha$是2-上边缘。

**步骤4**：逆构造。

给定$f \in Z^2(Q, N)$，在集合$Q \times N$上定义乘法：
$$(q_1, n_1)(q_2, n_2) = (q_1q_2, n_1 + q_1 \cdot n_2 + f(q_1, q_2))$$

验证结合律依赖于$f$是2-上闭链。

**步骤5**：等价性验证。

两个上闭链相差上边缘当且仅当相应扩张等价。

**证毕**。

**定理 4.3**（分裂扩张判据）：扩张$1 \to N \to G \to Q \to 1$分裂当且仅当对应的$[f] = 0 \in H^2(Q, N)$。

### 5. 伽罗瓦上同调的应用

**定理 4.4**（Hilbert 90）：设$L/K$是有限伽罗瓦扩张，$G = Gal(L/K)$。则：
$$H^1(G, L^\times) = 0$$

**证明概要**：

设$f: G \to L^\times$是1-上闭链，即$f(\sigma\tau) = f(\sigma) \cdot \sigma(f(\tau))$。

由Artin线性无关性，存在$a \in L^\times$使得：
$$b = \sum_{\tau \in G} f(\tau) \cdot \tau(a) \neq 0$$

对任意$\sigma \in G$：
\begin{align*}
\sigma(b) &= \sum_{\tau} \sigma(f(\tau)) \cdot \sigma\tau(a) \\
&= \sum_{\tau} f(\sigma)^{-1}f(\sigma\tau) \cdot \sigma\tau(a) \\
&= f(\sigma)^{-1} \sum_{\tau'} f(\tau') \cdot \tau'(a) = f(\sigma)^{-1}b
\end{align*}

因此$f(\sigma) = b/\sigma(b) = \sigma(b^{-1})/b^{-1}$是1-上边缘。

**证毕**。

## 🔧 形式化实现

### Lean 4形式化

```lean
-- 群上同调定义
variable (G : Type) [Group G] (M : Type) [AddCommGroup M] [Module (MonoidAlgebra ℤ G) M]

-- 上链复形
def CochainComplex (n : ℕ) := (Fin n → G) → M

-- 上边缘算子
def coboundary {n : ℕ} (f : CochainComplex G M n) : CochainComplex G M (n+1) :=
  fun g => g 0 • f (fun i => g (i+1)) +
    ∑ i : Fin n, (-1)^(i.val+1) • f (fun j =>
      if j.val < i.val then g j.val
      else if j.val = i.val then g i.val * g (i.val+1)
      else g (j.val+1)) +
    (-1)^(n+1) • f (fun i => g i.val)

-- 上同调群
def GroupCohomology (n : ℕ) : Type :=
  {f : CochainComplex G M n // coboundary f = 0} ⧸
  (Set.range (coboundary (n := n-1)))
```
