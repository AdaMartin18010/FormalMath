# 自由群与群展示 - 深度版

> **"自由群是所有群中'最自由'的，它只受群公理的约束，没有其他关系。"**

---

## 📚 概述

自由群是群论中最基本的构造之一，它是由一组生成元生成的、除群公理外没有任何关系的群。群展示则是通过生成元和关系描述群的标准方法。这一理论由von Dyck、Dehn和Tietze在19世纪末20世纪初建立，是组合群论、几何群论和代数拓扑的核心工具。

---

## 🕰️ 历史发展与数学家理念

### 格洛腾迪克的结构主义视角

**格洛腾迪克数学理念**强调**泛性质**和**自由构造**：

- **自由对象**：自由群是从集合到群的自由函子的左伴随
- **泛性质**：自由群在同构意义下由泛性质唯一确定
- **相对观点**：群展示应理解为自由群对正规子群的商

### 阿贝尔理念与自由阿贝尔群

**阿贝尔数学理念**关于**交换性**的思想在自由阿贝尔群中有直接体现：

- **自由阿贝尔群**：自由群在阿贝尔范畴中的对应
- **Abel化函子**：从群范畴到阿贝尔群范畴的左伴随
- **关系**：$[x, y] = xyx^{-1}y^{-1} = 1$（交换关系）

### Dehn的几何群论起源

Max Dehn在20世纪初研究曲面基本群时，提出了：

- **字问题**：给定展示，判断一个字是否等于单位元
- **共轭问题**：判断两个字是否共轭
- **同构问题**：判断两个展示是否定义同构的群

这些问题后来被发现是算法不可解的（Novikov, Boone）。

## 🏗️ 核心概念与完整证明

### 1. 自由群的定义

**定义 5.1**（字与约化字）
设$S$是集合，$S^{-1} = \{s^{-1} : s \in S\}$是形式逆元集合。

**字**：有限序列$w = x_1x_2\cdots x_n$，其中$x_i \in S \cup S^{-1}$。

**约化字**：字$w$称为**约化的**，如果不含形如$ss^{-1}$或$s^{-1}s$的子串。

**约化运算**：从字中反复删除$ss^{-1}$和$s^{-1}s$直到不能删除，得到的唯一约化字记为$\text{red}(w)$。

**定义 5.2**（自由群）
集合$S$上的**自由群**$F(S)$是所有约化字构成的集合，运算为：
$$w_1 \cdot w_2 = \text{red}(w_1w_2)$$

**定理 5.1**：$F(S)$在上述运算下构成群。

**证明**：

**封闭性**：两个约化字的连接再约化仍是约化字。

**结合律**：需证$(w_1 \cdot w_2) \cdot w_3 = w_1 \cdot (w_2 \cdot w_3)$。

关键是证明约化过程的终结果与顺序无关。这可通过分析字的"消去模式"来证明。

**单位元**：空字$\varepsilon$。

**逆元**：$(x_1\cdots x_n)^{-1} = x_n^{-1}\cdots x_1^{-1}$。

**证毕**。

### 2. 泛性质

**定理 5.2**（自由群的泛性质）
设$F(S)$是集合$S$上的自由群，$i: S \to F(S)$是包含映射。

对任意群$G$和任意函数$f: S \to G$，存在唯一的群同态$\bar{f}: F(S) \to G$使得$\bar{f} \circ i = f$。

```
S --i--> F(S)
|      ↓ ∃!f̄
f      G
```

**证明**：

**存在性**：对约化字$w = x_1^{\epsilon_1}\cdots x_n^{\epsilon_n}$（$\epsilon_i = \pm 1$），定义：
$$\bar{f}(w) = f(x_1)^{\epsilon_1} \cdots f(x_n)^{\epsilon_n}$$

验证这是群同态：$\bar{f}(w_1 \cdot w_2) = \bar{f}(w_1)\bar{f}(w_2)$。

**唯一性**：由$F(S)$由$S$生成，同态由生成元上的值唯一确定。

**证毕**。

### 3. 群展示

**定义 5.3**（群展示）
群$G$的**展示**$\langle S \mid R \rangle$包含：

- **生成元集合**$S$
- **关系集合**$R \subseteq F(S)$

满足$G \cong F(S) / \langle\langle R \rangle\rangle$，其中$\langle\langle R \rangle\rangle$是$R$生成的正规闭包。

**标准记号**：

- $x^n$表示$x$的$n$次幂
- $[x, y] = xyx^{-1}y^{-1}$表示换位子
- $x^y = y^{-1}xy$表示共轭

**经典示例**：

| 群 | 展示 | 说明 |
|----|------|------|
| 循环群$\mathbb{Z}_n$ | $\langle a \mid a^n = 1 \rangle$ | 单一生成元 |
| 二面体群$D_n$ | $\langle r, s \mid r^n = s^2 = 1, srs = r^{-1} \rangle$ | 正$n$边形的对称 |
| 四元数群$Q_8$ | $\langle i, j \mid i^4 = 1, i^2 = j^2, jij^{-1} = i^{-1} \rangle$ | 非交换8阶群 |
| 自由阿贝尔群$\mathbb{Z}^n$ | $\langle x_1, \ldots, x_n \mid [x_i, x_j] = 1, \forall i,j \rangle$ | 交换关系 |

### 4. Nielsen-Schreier定理

**定理 5.3**（Nielsen-Schreier，1921）
自由群的任意子群都是自由群。

**证明概要**：

设$F$是自由群，$H \leq F$。

**步骤1**：构造Schreier陪集图。

- 顶点：$H$在$F$中的陪集$Hg$
- 边：$Hg \overset{s}{\longrightarrow} Hgs$对每个生成元$s$

**步骤2**：选择Schreier横截。
选择陪集代表元$\mathcal{T}$，满足：

- 每个陪集恰好一个代表元
- 代表元的任意前缀也是代表元

**步骤3**：构造$H$的生成元。
对每个$t \in \mathcal{T}$和生成元$s$，定义：
$$g(t, s) = ts(\overline{ts})^{-1}$$
其中$\overline{ts}$是$Hts$的代表元。

**步骤4**：验证这些生成元是自由的。
通过分析Schreier图上的回路结构证明没有非平凡关系。

**证毕**。

**定理 5.4**（Schreier指数公式）
若$F$是秩为$n$的自由群，$H \leq F$是指数为$k$的子群，则$H$的秩为：
$$\text{rank}(H) = k(n-1) + 1$$

### 5. Tietze变换

**定理 5.5**（Tietze，1908）
两个展示定义同构群当且仅当可以通过以下变换相互转化：

**T1（添加生成元）**：
$$\langle S \mid R \rangle \to \langle S, t \mid R, t = w \rangle$$
其中$w \in F(S)$。

**T2（删除生成元）**：
若生成元$t$可由其他生成元表示，可删除$t$及相应关系。

**T3（添加关系）**：
添加可由已有关系推出的关系。

**T4（删除关系）**：
删除可由其他关系推出的关系。

## 🧮 应用示例

### 基本群的展示

**定理 5.6**（van Kampen）
拓扑空间$X = U \cup V$（$U, V$开，$U \cap V$道路连通），则：
$$\pi_1(X) = \pi_1(U) *_{\pi_1(U \cap V)} \pi_1(V)$$

**示例**：亏格$g$曲面的基本群：
$$\pi_1(\Sigma_g) = \langle a_1, b_1, \ldots, a_g, b_g \mid \prod_{i=1}^g [a_i, b_i] = 1 \rangle$$
