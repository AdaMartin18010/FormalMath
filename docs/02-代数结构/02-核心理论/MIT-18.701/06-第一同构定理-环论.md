## 1. 引言

环的第一同构定理与群论版本平行，建立了环同态、理想与商环之间的同构关系，是研究环结构的基本工具。

## 2. 预备知识

**定义 2.1（环同态的核）** 设 $\varphi: R \to S$ 为环同态。其**核**定义为
$$\ker\varphi = \{r \in R \mid \varphi(r) = 0_S\}。$$

**引理 2.2**：$\ker\varphi$ 是 $R$ 的理想。

**证明**：

1. 对任意 $a, b \in \ker\varphi$，有 $\varphi(a-b) = \varphi(a) - \varphi(b) = 0 - 0 = 0$，故 $a-b \in \ker\varphi$。
2. 对任意 $r \in R, a \in \ker\varphi$，有 $\varphi(ra) = \varphi(r)\varphi(a) = \varphi(r) \cdot 0 = 0$，故 $ra \in \ker\varphi$。同理 $ar \in \ker\varphi$。
因此 $\ker\varphi$ 是 $R$ 的双边理想。 ∎

## 3. 第一同构定理

### 定理 3.1

设 $\varphi: R \to S$ 为环同态，则
$$R / \ker\varphi \cong \operatorname{im}\varphi。$$

**证明**：
构造映射 $\psi: R/\ker\varphi \to \operatorname{im}\varphi$，$\psi(r + \ker\varphi) = \varphi(r)$。

1. **良定义性**：若 $r_1 + \ker\varphi = r_2 + \ker\varphi$，则 $r_1 - r_2 \in \ker\varphi$，于是
   $$\varphi(r_1) - \varphi(r_2) = \varphi(r_1 - r_2) = 0,$$
   故 $\varphi(r_1) = \varphi(r_2)$。

2. **保持加法**：
   $$\psi((r_1 + \ker) + (r_2 + \ker)) = \psi(r_1 + r_2 + \ker) = \varphi(r_1 + r_2) = \varphi(r_1) + \varphi(r_2) = \psi(r_1 + \ker) + \psi(r_2 + \ker)。$$

3. **保持乘法**：
   $$\psi((r_1 + \ker)(r_2 + \ker)) = \psi(r_1 r_2 + \ker) = \varphi(r_1 r_2) = \varphi(r_1)\varphi(r_2) = \psi(r_1 + \ker)\psi(r_2 + \ker)。$$

4. **保持单位元**：$\psi(1 + \ker) = \varphi(1) = 1_S$。

5. **单射性**：若 $\psi(r_1 + \ker) = \psi(r_2 + \ker)$，则 $\varphi(r_1) = \varphi(r_2)$，于是 $\varphi(r_1 - r_2) = 0$，即 $r_1 - r_2 \in \ker\varphi$，故 $r_1 + \ker = r_2 + \ker$。

6. **满射性**：对任意 $y \in \operatorname{im}\varphi$，存在 $r \in R$ 使得 $y = \varphi(r) = \psi(r + \ker)$。

综上，$\psi$ 是环同构。 ∎

**Lean4 双语对照**：

```lean
import Mathlib.RingTheory.Ideal.Quotient
import Mathlib.RingTheory.Ideal.Basic
import Mathlib.Algebra.Ring.Hom.Defs

universe u v

namespace RingFirstIsomorphismTheorem

open Ideal RingHom

variable {R S : Type*} [CommRing R] [CommRing S]

-- 环同态的核是理想（自动实例）
theorem ker_is_ideal (f : R →+* S) : (ker f).IsIdeal := by
  infer_instance

-- 第一同构定理的显式构造
def firstIsomorphism (f : R →+* S) : R ⧸ ker f →+* range f where
  toFun := fun x =>
    Quotient.liftOn' x (fun r => ⟨f r, by simp [range]⟩) (by
      intro r₁ r₂ h
      simp at h ⊢
      have h_ker : r₁ - r₂ ∈ ker f := by
        simpa using h
      simp [mem_ker] at h_ker
      calc
        f r₁ = f r₁ + 0 := by rw [add_zero]
        _ = f r₁ + (f r₂ - f r₂) := by rw [sub_self]; ring
        _ = f r₂ + (f r₁ - f r₂) := by ring
        _ = f r₂ + f (r₁ - r₂) := by rw [map_sub]
        _ = f r₂ + 0 := by rw [h_ker]; ring
        _ = f r₂ := by rw [add_zero]
    )
  map_one' := by
    refine Quotient.inductionOn' (1 : R ⧸ ker f) (fun r => ?_)
    simp
  map_mul' := by
    intro x y
    refine Quotient.inductionOn₂' x y (fun r₁ r₂ => ?_)
    simp
    rw [map_mul]
  map_add' := by
    intro x y
    refine Quotient.inductionOn₂' x y (fun r₁ r₂ => ?_)
    simp
    rw [map_add]

-- 使用 Mathlib 的内置同构
theorem firstIsomorphismEquiv (f : R →+* S) :
    R ⧸ ker f ≃+* range f := by
  exact Ideal.quotientKerEquivRange f

end RingFirstIsomorphismTheorem
```

**代码解读**：

- `Quotient.liftOn'` 用于在商环上定义映射，并验证其与理想代表元无关。
- `map_add'`、`map_mul'`、`map_one'` 分别对应环同态的加法、乘法、单位元保持条件。
- `Ideal.quotientKerEquivRange` 是 Mathlib 中第一同构定理的现成实现。

## 4. 习题与详细解答

### 习题 1

设 $R$ 为交换环，$I, J$ 为 $R$ 的理想。证明
$$(I + J)/J \cong I/(I \cap J)。$$

**解答**：
考虑自然投影 $\pi: R \to R/J$ 在 $I$ 上的限制 $\pi|_I: I \to R/J$。其像为 $(I+J)/J$，核为 $I \cap J$。由环的第一同构定理即得结论。 ∎

### 习题 2

设 $\varphi: \mathbb{Z} \to \mathbb{Z}/n\mathbb{Z}$ 为自然同态。求 $\ker\varphi$ 并应用第一同构定理。

**解答**：
$\ker\varphi = n\mathbb{Z}$，$\operatorname{im}\varphi = \mathbb{Z}/n\mathbb{Z}$。于是
$$\mathbb{Z}/n\mathbb{Z} \cong \mathbb{Z}/n\mathbb{Z}。$$
此例虽然平凡，但说明商环 $\mathbb{Z}/n\mathbb{Z}$ 恰为 $\mathbb{Z}$ 模去核 $n\mathbb{Z}$。 ∎

### 习题 3

设 $R$ 为交换环，$M$ 为极大理想。证明 $R/M$ 是域。

**解答**：
设 $\bar{a} \neq 0$ 为 $R/M$ 中的元素，取代表元 $a \in R \setminus M$。考虑理想 $(a) + M$。因为 $M$ 是极大理想且 $a \notin M$，有 $(a) + M = R$。于是存在 $r \in R, m \in M$ 使得 $ra + m = 1$。在 $R/M$ 中，$\bar{r}\bar{a} = \bar{1}$，故 $\bar{a}$ 可逆。因此 $R/M$ 中每个非零元可逆，即 $R/M$ 是域。 ∎

## 5. 常见误区分析

1. **混淆群同态与环同态的核**：群同态的核是子群（且正规），而环同态的核是理想。理想对环中任意元素的左右乘法封闭，这是关键区别。
2. **遗漏乘法保持性**：证明环同构时，除了证明双射和加法保持，还必须验证乘法保持与单位元保持。
3. **良定义性证明不完整**：与群的情形类似，定义 $\psi(r + \ker) = \varphi(r)$ 时必须验证与代表元选取无关，初学者常遗漏此步。
