## 1. 引言

拉格朗日定理是有限群论中最基本的定理之一。它揭示了子群的阶与群的阶之间的整除关系，是后续 Sylow 定理、Cauchy 定理等的基石。

## 2. 基本定义

**定义 2.1（左陪集）** 设 $G$ 为群，$H \le G$ 为子群，$a \in G$。称集合
$$aH = \{ah \mid h \in H\}$$
为 $H$ 在 $G$ 中关于代表元 $a$ 的**左陪集**（left coset）。

**定义 2.2（指数）** 设 $G$ 为有限群，$H \le G$。$H$ 在 $G$ 中的不同左陪集的个数称为 $H$ 在 $G$ 中的**指数**（index），记为 $[G:H]$。

## 3. 关键引理

### 引理 3.1：左陪集的基数等于子群的基数

**陈述**：对任意 $a \in G$，映射 $f: H \to aH$，$f(h) = ah$ 是双射，从而 $|aH| = |H|$。

**证明**：

1. **单射**：设 $h_1, h_2 \in H$ 且 $ah_1 = ah_2$。左乘 $a^{-1}$ 得 $h_1 = h_2$。故 $f$ 为单射。
2. **满射**：对任意 $x \in aH$，由定义存在 $h \in H$ 使得 $x = ah = f(h)$。故 $f$ 为满射。
3. 因此 $f$ 是双射，$|aH| = |H|$。 ∎

**Lean4 双语对照**：

```lean
-- 左陪集的基数等于子群的基数
theorem leftCoset_card_eq_subgroup_card {G : Type u} [Group G] [Fintype G]
    (H : Subgroup G) [Fintype H] (a : G) :
    (a • (H : Set G)).toFinset.card = Fintype.card H := by
  /- 证明思路：
     1. 定义映射 f: H → aH, f(h) = ah
     2. 证明 f 是双射
     3. 因此 |aH| = |H|
  -/
  let f : H → (a • (H : Set G)).toFinset := fun h => ⟨a * h, by
    simp [leftCoset]
    exact ⟨h, h.2, rfl⟩
  ⟩

  have h_inj : Function.Injective f := by
    intro h₁ h₂ heq
    simp [f] at heq
    /- 由 ah₁ = ah₂，左乘 a⁻¹ 得到 h₁ = h₂ -/
    have : a * h₁ = a * h₂ := by simpa using heq
    exact (mul_left_inj a).mp this

  have h_surj : Function.Surjective f := by
    intro x
    rcases x with ⟨x, hx⟩
    simp [leftCoset] at hx
    rcases hx with ⟨h, hh, rfl⟩
    exact ⟨⟨h, hh⟩, by simp [f]⟩

  -- 双射意味着基数相等
  have h_bij : Function.Bijective f := ⟨h_inj, h_surj⟩
  exact (Fintype.card_of_bijective h_bij).symm
```

**代码解读**：

- `let f : H → ...` 对应“定义映射 $f$”。
- `have h_inj` 对应“证明 $f$ 是单射”。
- `have h_surj` 对应“证明 $f$ 是满射”。
- 最后使用 `Fintype.card_of_bijective` 得到基数相等。

### 引理 3.2：陪集要么相等要么不相交

**陈述**：对任意 $a, b \in G$，要么 $aH = bH$，要么 $aH \cap bH = \varnothing$。

**证明**：
设 $aH \cap bH \neq \varnothing$，则存在 $h_1, h_2 \in H$ 使得 $ah_1 = bh_2$。
于是 $a = bh_2h_1^{-1} \in bH$，故 $aH \subseteq bH$。
同理 $b = ah_1h_2^{-1} \in aH$，故 $bH \subseteq aH$。
因此 $aH = bH$。 ∎

**Lean4 双语对照**：

```lean
-- 陪集要么相等，要么不相交
theorem leftCoset_eq_or_disjoint {G : Type u} [Group G] (H : Subgroup G) (a b : G) :
    a • (H : Set G) = b • (H : Set G) ∨ (a • (H : Set G)) ∩ (b • (H : Set G)) = ∅ := by
  /- 证明思路：
     如果 aH ∩ bH ≠ ∅，则存在 h₁, h₂ ∈ H 使得 ah₁ = bh₂
     于是 a = bh₂h₁⁻¹ ∈ bH，因此 aH ⊆ bH
     同理 bH ⊆ aH，所以 aH = bH
  -/
  by_cases h : (a • (H : Set G)) ∩ (b • (H : Set G)) = ∅
  · right; exact h
  · left
    have : ∃ x, x ∈ (a • (H : Set G)) ∩ (b • (H : Set G)) := by
      by_contra h'
      push_neg at h'
      have : (a • (H : Set G)) ∩ (b • (H : Set G)) = ∅ := by
        ext x
        simp [h']
      contradiction

    rcases this with ⟨x, hxa, hxb⟩
    simp [leftCoset] at hxa hxb
    rcases hxa with ⟨h₁, hh₁, rfl⟩
    rcases hxb with ⟨h₂, hh₂, heq⟩

    /- ah₁ = bh₂ 推出 a = b * h₂ * h₁⁻¹ -/
    have ha : a = b * h₂ * h₁⁻¹ := by
      rw [←heq]
      group

    /- 证明 aH = bH -/
    ext y
    simp [leftCoset, ha]
    constructor
    · intro hy
      rcases hy with ⟨h, hh, rfl⟩
      use h₂ * h₁⁻¹ * h
      constructor
      · exact H.mul_mem (H.mul_mem hh₂ (H.inv_mem hh₁)) hh
      · group
    · intro hy
      rcases hy with ⟨h, hh, rfl⟩
      use h₁ * h₂⁻¹ * h
      constructor
      · exact H.mul_mem (H.mul_mem hh₁ (H.inv_mem hh₂)) hh
      · group
```

**代码解读**：

- `by_cases` 分两种情况讨论。
- 在相交非空的情况下，先取出公共元素 `x`，再分别用 `rcases` 分解为 $h_1, h_2$。
- `ext y` 证明集合相等采用“双向包含”。

## 4. 主定理

### 定理 4.1（拉格朗日定理）

设 $G$ 为有限群，$H \le G$ 为子群，则
$$|H| \mid |G| \quad\text{且}\quad |G| = [G:H] \cdot |H|。$$

**证明**：
由引理 3.2，$G$ 可分解为互不相交的左陪集的并：
$$G = a_1 H \cup a_2 H \cup \dots \cup a_k H,$$
其中 $k = [G:H]$。由引理 3.1，每个左陪集的大小均为 $|H|$，因此
$$|G| = \sum_{i=1}^k |a_i H| = k \cdot |H| = [G:H] \cdot |H|。$$
于是 $|H|$ 整除 $|G|$。 ∎

**Lean4 双语对照**：

```lean
-- 拉格朗日定理：子群的阶整除群的阶
theorem lagrange_theorem {G : Type u} [Group G] [Fintype G]
    (H : Subgroup G) [Fintype H] :
    Fintype.card H ∣ Fintype.card G := by
  /- 使用Mathlib4的index_mul_card定理 -/
  rw [← Subgroup.index_mul_card H]
  exact dvd_mul_left (Fintype.card H) (H.index)

-- 拉格朗日定理的完整形式：|G| = [G:H] · |H|
theorem lagrange_theorem_full {G : Type u} [Group G] [Fintype G]
    (H : Subgroup G) [Fintype H] :
    Fintype.card G = H.index * Fintype.card H := by
  exact (Subgroup.index_mul_card H).symm
```

**代码解读**：

- `Subgroup.index_mul_card` 已经将群的阶表示为指数乘以子群的阶。
- `dvd_mul_left` 证明左因子整除乘积。

### 推论 4.2

对有限群 $G$ 中的任意元素 $a$，有 $\operatorname{order}(a) \mid |G|$，且 $a^{|G|} = 1$。

**证明**：
考虑循环子群 $\langle a \rangle = \{a^n \mid n \in \mathbb{Z}\}$。其阶恰为 $a$ 的阶 $\operatorname{order}(a)$。由拉格朗日定理，$|\langle a \rangle| \mid |G|$。设 $|G| = \operatorname{order}(a) \cdot k$，则
$$a^{|G|} = a^{\operatorname{order}(a) \cdot k} = (a^{\operatorname{order}(a)})^k = 1^k = 1。$$ ∎

**Lean4 双语对照**：

```lean
-- 元素的阶整除群的阶
theorem order_of_dvd_card {G : Type u} [Group G] [Fintype G] (a : G) :
    orderOf a ∣ Fintype.card G := by
  /- 利用循环子群 <a> 的阶等于元素 a 的阶 -/
  have h : orderOf a = Fintype.card (zpowers a) := by
    rw [orderOf_eq_card_zpowers]
  rw [h]
  /- 应用拉格朗日定理于循环子群 -/
  exact lagrange_theorem (zpowers a)

-- 费马小定理（群论版本）
theorem fermat_little_theorem_group {G : Type u} [Group G] [Fintype G] (a : G) :
    a ^ Fintype.card G = 1 := by
  have h : orderOf a ∣ Fintype.card G := order_of_dvd_card a
  rcases h with ⟨k, hk⟩
  calc
    a ^ Fintype.card G = a ^ (orderOf a * k) := by rw [hk]
    _ = (a ^ orderOf a) ^ k := by rw [pow_mul]
    _ = 1 ^ k := by rw [pow_orderOf_eq_one a]
    _ = 1 := by simp
```

## 5. 习题与详细解答

### 习题 1

证明：若 $|G| = p$ 为素数，则 $G$ 为循环群。

**解答**：
取 $a \in G$ 且 $a \neq 1$。由拉格朗日定理，$|\langle a \rangle|$ 整除 $|G| = p$。因为 $p$ 为素数且 $|\langle a \rangle| > 1$（至少包含 $1, a$），故 $|\langle a \rangle| = p = |G|$。于是 $\langle a \rangle = G$，即 $G$ 为循环群。 ∎

### 习题 2

证明：指数为 2 的子群必为正规子群。

**解答**：
设 $H \le G$ 且 $[G:H] = 2$。则 $G$ 关于 $H$ 只有两个左陪集：$H$ 与 $aH$（$a \notin H$）。同理也只有两个右陪集：$H$ 与 $Ha$。因为 $aH \neq H$ 且 $Ha \neq H$，必有 $aH = Ha$。于是对所有 $g \in G$ 有 $gH = Hg$，即 $H \trianglelefteq G$。 ∎

### 习题 3

设 $G$ 为有限群，$a, b \in G$。证明 $\operatorname{order}(ab) = \operatorname{order}(ba)$。

**解答**：
注意 $ab$ 与 $ba$ 在 $G$ 中共轭：$ba = a^{-1}(ab)a$。因为共轭元素的阶相同，故 $\operatorname{order}(ab) = \operatorname{order}(ba)$。 ∎

## 6. 常见误区分析

1. **逆命题不成立**：拉格朗日定理仅说明“若 $H \le G$，则 $|H| \mid |G|$”。其逆命题“若 $d \mid |G|$，则 $G$ 必有 $d$ 阶子群”一般不成立。反例：$A_4$ 的阶为 12，但没有 6 阶子群。
2. **左右陪集混淆**：在一般情况下，左陪集 $aH$ 与右陪集 $Ha$ 不一定相等。只有当 $H$ 是正规子群时，$aH = Ha$ 对所有 $a$ 成立。
3. **忽略有限性条件**：拉格朗日定理要求 $G$ 为有限群。对无限群，子群的“指数”可以定义，但 $|G| = [G:H] \cdot |H|$ 的表述不再适用。
