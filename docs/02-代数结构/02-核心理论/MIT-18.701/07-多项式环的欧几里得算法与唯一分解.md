## 1. 引言

多项式环 $k[x]$（$k$ 为域）具有良好的整除结构：它是欧几里得整环，从而也是主理想整环与唯一分解整环。本节给出严格证明，并建立与 Lean4 的对照。

## 2. 多项式环的欧几里得算法

**定理 2.1（带余除法）** 设 $k$ 为域，$f, g \in k[x]$ 且 $g \neq 0$。则存在唯一的 $q, r \in k[x]$ 使得
$$f = qg + r, \quad \text{且 } r = 0 \text{ 或 } \deg r < \deg g。$$

**证明**：
对 $\deg f$ 使用归纳法。

- 若 $\deg f < \deg g$，取 $q = 0, r = f$ 即可。
- 设 $\deg f \ge \deg g$。令 $m = \deg f, n = \deg g$，设 $f, g$ 的首项分别为 $a_m x^m, b_n x^n$。构造
  $$f_1 = f - \frac{a_m}{b_n} x^{m-n} g。$$
  则 $\deg f_1 < \deg f$。由归纳假设，存在 $q_1, r$ 使得 $f_1 = q_1 g + r$ 且 $\deg r < \deg g$（或 $r=0$）。于是
  $$f = \left(\frac{a_m}{b_n} x^{m-n} + q_1\right) g + r。$$
  取 $q = \frac{a_m}{b_n} x^{m-n} + q_1$ 即得所求。

唯一性：设 $f = qg + r = q'g + r'$，则 $(q-q')g = r' - r$。若 $q \neq q'$，则左边次数 $\ge \deg g$，而右边次数 $< \deg g$，矛盾。故 $q = q', r = r'$。 ∎

## 3. 唯一分解定理

### 定理 3.1

设 $k$ 为域，则多项式环 $k[x]$ 是唯一分解整环（UFD）。即每个次数 $\ge 1$ 的多项式都可以唯一地（在相伴和顺序意义下）分解为不可约多项式的乘积。

**证明**：
由**定理 2.1**，$k[x]$ 是欧几里得整环。我们证明：每个欧几里得整环都是主理想整环（PID），每个 PID 都是 UFD。

**步骤 1：欧几里得整环 $\Rightarrow$ PID**。
设 $R$ 为欧几里得整环，$I \subseteq R$ 为非零理想。取 $I$ 中“次数” $\delta(d)$ 最小的非零元 $d$。对任意 $a \in I$，由带余除法存在 $q, r$ 使得 $a = qd + r$ 且 $r = 0$ 或 $\delta(r) < \delta(d)$。因为 $r = a - qd \in I$，由 $d$ 的最小性，必有 $r = 0$。于是 $a = qd \in (d)$。因此 $I = (d)$，即 $R$ 是 PID。

**步骤 2：PID $\Rightarrow$ UFD**。
设 $R$ 为 PID。

- **存在性**：考虑 $R$ 中不能分解为不可约元乘积的元素构成的集合 $S$。若 $S \neq \varnothing$，则 $S$ 中关于整除关系存在极大元 $a$（由主理想升链条件，PID 满足 ACC）。因为 $a$ 可约，存在 $a = bc$ 且 $b, c$ 均非单位。由极大性，$b, c \notin S$，均可分解为不可约元乘积，从而 $a$ 也可分解，矛盾。故 $S = \varnothing$。
- **唯一性**：在整环中，素元必是不可约元；反之，在 PID 中不可约元必是素元（因为 $(p)$ 是极大理想，从而是素理想）。利用素元性质，通过归纳法可证分解唯一。具体地，设
  $$a = p_1 p_2 \cdots p_n = q_1 q_2 \cdots q_m$$
  为两个不可约分解。因为 $p_1 \mid q_1 \cdots q_m$ 且 $p_1$ 是素元，存在某个 $j$ 使得 $p_1 \mid q_j$。因为 $q_j$ 不可约，$p_1$ 与 $q_j$ 相伴。两边约去 $p_1$（在整环中消去律成立），继续归纳即可得 $n=m$ 且分解在相伴意义下唯一。

综上，$k[x]$ 是 UFD。 ∎

## 4. 习题与详细解答

### 习题 1

在 $\mathbb{Q}[x]$ 中，对 $f(x) = x^3 + 2x^2 - x - 2$ 与 $g(x) = x^2 - 1$ 执行欧几里得算法，求 $\gcd(f, g)$。

**解答**：
注意到
$$(x+2)(x^2 - 1) = x^3 + 2x^2 - x - 2 = f(x)。$$
所以 $f = (x+2)g + 0$，即 $g \mid f$。因此 $\gcd(f, g) = g = x^2 - 1$。 ∎

### 习题 2

证明：在 $\mathbb{R}[x]$ 中，$x^2 + 1$ 是不可约多项式。

**解答**：
假设 $x^2 + 1 = p(x)q(x)$，则 $\deg p + \deg q = 2$。若 $\deg p = \deg q = 1$，设 $p(x) = ax + b$，则 $p$ 在 $\mathbb{R}$ 中有根 $-b/a$。但 $x^2 + 1 = 0$ 在 $\mathbb{R}$ 中无解，矛盾。因此 $x^2 + 1$ 不能分解为两个非常数多项式的乘积，即不可约。 ∎

### 习题 3

设 $k$ 为域，$f \in k[x]$ 为次数 $n \ge 1$ 的多项式。证明 $f$ 在 $k[x]$ 中最多有 $n$ 个不同的首一不可约因子（计重数）。

**解答**：
因为 $k[x]$ 是 UFD，$f$ 有唯一分解 $f = u \cdot p_1^{e_1} \cdots p_m^{e_m}$，其中 $u$ 为单位，$p_i$ 为互不相同的首一不可约多项式。比较次数得
$$n = \deg f = \sum_{i=1}^m e_i \deg p_i \ge \sum_{i=1}^m e_i。$$
因此 $\sum e_i \le n$，即首一不可约因子的总个数（计重数）不超过 $n$。 ∎

## 5. 常见误区分析

1. **忽略域的条件**：带余除法要求除式多项式的首项系数可逆，这在域上自然成立；在一般整环上不一定成立。
2. **混淆“不可约”与“无根”**：在次数 $\ge 3$ 时，多项式无根并不一定不可约（例如 $\mathbb{Q}[x]$ 中的 $(x^2+1)(x^2+2)$）。仅对二次或三次多项式，无根等价于不可约。
3. **唯一性的误解**：UFD 中的唯一性是在“相伴”与“顺序”意义下的。例如 $6 = 2 \cdot 3 = (-2) \cdot (-3)$ 在系数域中视为相同的分解。
