# 线性代数习题补充

## 习题1

证明：若$A$是实对称矩阵，则存在正交矩阵$Q$使$Q^T A Q = \Lambda$（对角阵）。

**证明**：

1. 实对称矩阵的特征值都是实数
2. 不同特征值对应的特征向量正交
3. 对重特征值用Gram-Schmidt正交化
4. 单位化后得到正交矩阵$Q$

## 习题2

求矩阵$A = \begin{pmatrix} 4 & 2 \\ 1 & 3 \end{pmatrix}$的特征值和特征向量。

**解答**：
特征方程：$\det(A - \lambda I) = (4-\lambda)(3-\lambda) - 2 = \lambda^2 - 7\lambda + 10 = 0$

特征值：$\lambda_1 = 5, \lambda_2 = 2$

特征向量：

- $\lambda_1 = 5$：$(A-5I)v = 0 \Rightarrow v_1 = (2, 1)^T$
- $\lambda_2 = 2$：$(A-2I)v = 0 \Rightarrow v_2 = (1, -1)^T$

## 习题3

证明Cayley-Hamilton定理：矩阵满足其特征方程。

**证明**：设$p(\lambda) = \det(A - \lambda I) = \lambda^n + a_{n-1}\lambda^{n-1} + \cdots + a_0$

则$p(A) = A^n + a_{n-1}A^{n-1} + \cdots + a_0 I = 0$。

**关键步骤**：利用伴随矩阵$(A - \lambda I)^{-1} = \frac{1}{p(\lambda)} \text{adj}(A - \lambda I)$。
