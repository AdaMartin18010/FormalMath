# 张量积与平坦模

## 1. 引言

张量积（tensor product）是多重线性代数的核心构造，它将两个模"粘合"为一个新的模，同时保持双线性的结构。在交换代数中，张量积的重要性远远超出了线性代数的范畴：它是基变换（标量扩张）的工具，是纤维积的代数对应，也是定义平坦性的基础。平坦模（flat module）是一类保持正合性的模——与张量积后序列仍正合。这一性质在代数几何中至关重要，因为概形的纤维积正是通过张量积构造的，而平坦态射保证了纤维的"连续变化"。本文从张量积的泛性质出发，建立其基本理论，讨论平坦性的各种判据，并引入Tor函子作为张量积左导出函子。

## 2. 模的张量积

### 2.1 泛性质定义

**定义 2.1**。设 $R$ 为交换环，$M, N$ 为 $R$-模。$M$ 与 $N$ 的**张量积** $M \otimes_R N$ 是配备双线性映射 $\otimes: M \times N \to M \otimes_R N$ 的 $R$-模，满足泛性质：对任意双线性映射 $f: M \times N \to P$，存在唯一的线性映射 $\bar{f}: M \otimes_R N \to P$ 使 $\bar{f}(m \otimes n) = f(m, n)$。

**构造**。$M \otimes_R N = F / K$，其中 $F$ 为自由 $R$-模（基为 $M \times N$），$K$ 为由以下关系生成的子模：
- $(m + m', n) - (m, n) - (m', n)$；
- $(m, n + n') - (m, n) - (m, n')$；
- $(rm, n) - r(m, n)$；
- $(m, rn) - r(m, n)$。

记 $m \otimes n$ 为 $(m, n)$ 的等价类。

### 2.2 基本性质

**命题 2.2**。张量积满足：
1. **交换律**：$M \otimes N \cong N \otimes M$；
2. **结合律**：$(M \otimes N) \otimes P \cong M \otimes (N \otimes P)$；
3. **单位元**：$R \otimes_R M \cong M$；
4. **分配律**：$M \otimes (\bigoplus_i N_i) \cong \bigoplus_i (M \otimes N_i)$。

**命题 2.3**。$\operatorname{Hom}_R(M \otimes N, P) \cong \operatorname{Bilin}_R(M \times N, P) \cong \operatorname{Hom}_R(M, \operatorname{Hom}_R(N, P))$。即 $- \otimes N \dashv \operatorname{Hom}(N, -)$（伴随对）。

### 2.3 例子

**例 2.4**。$\mathbb{Z}/m\mathbb{Z} \otimes_{\mathbb{Z}} \mathbb{Z}/n\mathbb{Z} \cong \mathbb{Z}/\gcd(m,n)\mathbb{Z}$。

*证明*。$a \otimes b = ab(1 \otimes 1)$。$m(1 \otimes 1) = m \otimes 1 = 0 \otimes 1 = 0$，同理 $n(1 \otimes 1) = 0$。故 $1 \otimes 1$ 的阶整除 $\gcd(m, n)$。反向构造满射。$\square$

**例 2.5**。$M \otimes_R R[x] \cong M[x]$（多项式模）。$M \otimes_R R_f \cong M_f$（局部化模）。

## 3. 平坦模

### 3.1 定义

**定义 3.1**。$R$-模 $M$ 称为**平坦的**（flat），若函子 $- \otimes_R M: \mathbf{Mod}_R \to \mathbf{Mod}_R$ 正合（即保持短正合序列）。

由右伴随性，$- \otimes M$ 自动保右正合，故平坦性等价于保单射：若 $N' \hookrightarrow N$，则 $N' \otimes M \hookrightarrow N \otimes M$。

### 3.2 基本例子

**例 3.2**。自由模平坦（因 $- \otimes R^n \cong (-)^n$）。投射模平坦（自由模的直和项）。

**例 3.3**。$\mathbb{Q}$ 作为 $\mathbb{Z}$-模平坦：$\mathbb{Q} \otimes_{\mathbb{Z}} -$ 为局部化函子，正合（见[01-局部化](01-局部化.md)）。

**非例 3.4**。$\mathbb{Z}/n\mathbb{Z}$（$n > 1$）作为 $\mathbb{Z}$-模非平坦：$\mathbb{Z} \hookrightarrow \mathbb{Q}$，但 $\mathbb{Z}/n \otimes \mathbb{Z} \cong \mathbb{Z}/n \not\hookrightarrow 0 = \mathbb{Z}/n \otimes \mathbb{Q}$。

### 3.3 平坦性判据

**定理 3.5**（局部判据）。$M$ 平坦当且仅当对所有素理想 $\mathfrak{p}$，$M_{\mathfrak{p}}$ 为平坦 $R_{\mathfrak{p}}$-模。当 $R$ Noetherian 且 $M$ 有限生成时，等价于 $M_{\mathfrak{m}}$ 对所有极大理想 $\mathfrak{m}$ 自由（此时 $M$ 局部自由，从而平坦）。

**定理 3.6**（Tor判据）。$M$ 平坦当且仅当 $\operatorname{Tor}_1^R(N, M) = 0$ 对所有 $N$。

**定理 3.7**（理想判据）。$M$ 平坦当且仅当对 $R$ 的每个（有限生成）理想 $I$，自然映射 $I \otimes M \to IM$ 为同构。

### 3.4 忠实平坦

**定义 3.8**。$M$ 称为**忠实平坦的**（faithfully flat），若 $M$ 平坦且 $N \otimes M = 0$ 蕴含 $N = 0$。

**定理 3.9**。$M$ 忠实平坦当且仅当 $M$ 平坦且对所有极大理想 $\mathfrak{m}$，$\mathfrak{m}M \neq M$（即 $M/\mathfrak{m}M \neq 0$）。

**例 3.10**。多项式扩张 $R \to R[x_1, \ldots, x_n]$ 忠实平坦。局部化 $R \to R_{\mathfrak{p}}$ 平坦但一般非忠实平坦。

## 4. Tor函子

### 4.1 定义

**定义 4.1**。$\operatorname{Tor}_n^R(M, N)$ 为 $- \otimes_R N$（或 $M \otimes_R -$）的左导出函子。

具体计算：取 $M$ 的投射分解
$$\cdots \to P_2 \to P_1 \to P_0 \to M \to 0,$$
张量积 $N$，得复形
$$\cdots \to P_2 \otimes N \to P_1 \otimes N \to P_0 \otimes N \to 0.$$
则 $\operatorname{Tor}_n(M, N) = H_n(P_\bullet \otimes N)$。

### 4.2 基本性质

**命题 4.2**。Tor函子满足：
- $\operatorname{Tor}_0(M, N) = M \otimes N$；
- 若 $M$ 或 $N$ 投射，则 $\operatorname{Tor}_n = 0$（$n > 0$）；
- 对称性：$\operatorname{Tor}_n(M, N) \cong \operatorname{Tor}_n(N, M)$；
- 对短正合序列 $0 \to M' \to M \to M'' \to 0$，有长正合序列
  $$\cdots \to \operatorname{Tor}_1(M'', N) \to M' \otimes N \to M \otimes N \to M'' \otimes N \to 0.$$

### 4.3 计算例子

**例 4.3**。$\operatorname{Tor}_1^{\mathbb{Z}}(\mathbb{Z}/m, \mathbb{Z}/n) \cong \mathbb{Z}/\gcd(m, n)$。

*证明*。取 $\mathbb{Z}/m$ 的投射分解 $0 \to \mathbb{Z} \xrightarrow{m} \mathbb{Z} \to \mathbb{Z}/m \to 0$。张量 $\mathbb{Z}/n$：
$$0 \to \mathbb{Z}/n \xrightarrow{m} \mathbb{Z}/n \to \mathbb{Z}/m \otimes \mathbb{Z}/n \to 0.$$
$\ker(m: \mathbb{Z}/n \to \mathbb{Z}/n) = \{x : mx \equiv 0 \pmod{n}\}$，阶为 $\gcd(m, n)$。$\square$

## 5. 与项目其他部分的关联

张量积与平坦模是交换代数的枢纽概念。在代数几何（见`docs/02-代数结构/范畴论/08-范畴论在代数几何中的应用.md`）中，概形的纤维积通过张量积定义：$\operatorname{Spec}(A) \times_{\operatorname{Spec}(C)} \operatorname{Spec}(B) = \operatorname{Spec}(A \otimes_C B)$。平坦态射 $f: X \to Y$ 保证了纤维的维数和Hilbert多项式的"连续性"，是形变理论的基础。在范畴论视角（见`docs/02-代数结构/范畴论/`）下，张量积函子是Hom函子的左伴随。Tor函子属于同调代数的标准工具（见`docs/02-代数结构/范畴论/05-Abel范畴.md`），与Ext函子对偶。在泛函分析（见`docs/03-分析学/03-泛函分析/`）中，张量积是算子理论的基本构造。
