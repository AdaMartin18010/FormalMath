# 交换代数基础

## 局部化

### 定义

$S^{-1}R = \{\frac{r}{s} : r \in R, s \in S\}$，$S$是乘法闭子集。

### 例子

- $R_\mathfrak{p}$：在素理想$\mathfrak{p}$处的局部化
- $K = S^{-1}R$，$S = R \setminus \{0\}$

## 诺特环

### 定义

理想升链稳定的环。

### Hilbert基定理

若$R$诺特，则$R[x]$诺特。
