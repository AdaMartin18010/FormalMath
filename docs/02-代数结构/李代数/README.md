# 李代数 / Lie Algebras

## 项目概述 / Project Overview

李代数是代数学的重要分支，研究满足雅可比恒等式的代数结构。李代数与李群密切相关，在数学物理、微分几何、表示论等领域有重要应用。

本分支致力于构建完整的李代数知识体系，包括核心理论、应用案例、形式化实现和算法实现。
