详细安装说明请参考 [安装部署指南](00-Python实现-代数结构安装部署指南.md)。

### 基础使用示例

```python
# 群论示例
from group_theory import cyclic_group, find_all_subgroups

G = cyclic_group(6)
subgroups = find_all_subgroups(G)
print(f"Z_6的子群数量: {len(subgroups)}")

# 环论示例
from ring_theory import IntegerModRing, find_all_ideals

Z5 = IntegerModRing(5)
ideals = find_all_ideals(Z5)
print(f"Z_5的理想数量: {len(ideals)}")

# 域论示例
from field_theory import FiniteField, find_primitive_element

GF16 = FiniteField(2, 4)
primitive = find_primitive_element(GF16)
print(f"GF(16)的本原元: {primitive}")
```

更多示例请参考 [快速参考](00-Python实现-代数结构快速参考.md) 和 [示例项目](00-Python实现-代数结构示例项目.md)。
