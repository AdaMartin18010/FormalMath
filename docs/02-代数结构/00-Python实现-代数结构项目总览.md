## 快速开始

### 1. 安装

```bash
# 克隆项目
git clone https://github.com/FormalMath/algebraic-structures.git
cd algebraic-structures

# 安装依赖
pip install -r requirements.txt
```

### 2. 基本使用

```python
from algebraic_structures.groups import FiniteGroup

# 创建对称群S3
S3 = FiniteGroup.symmetric_group(3)

# 计算群的阶
print(f"S3的阶: {S3.order()}")

# 查找子群
subgroups = S3.find_subgroups()
print(f"子群数量: {len(subgroups)}")
```
