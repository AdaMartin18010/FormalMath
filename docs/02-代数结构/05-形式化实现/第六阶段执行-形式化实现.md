## 形式化代码示例

### 群论定理

```lean
-- 拉格朗日定理：子群的阶整除群的阶
theorem lagrange_theorem {G : Type*} [Group G] [Fintype G] (H : Subgroup G) :
  Fintype.card H ∣ Fintype.card G := by
  apply H.index_mul_card.symm ▸ dvd_mul_left _ _
```

### 环论定理

```lean
-- 第一同构定理
theorem first_isomorphism_ring {R S : Type*} [Ring R] [Ring S]
  (f : R →+* S) :
  R ⧸ (RingHom.ker f) ≃+* (RingHom.range f : Subring S) := by
  apply Ideal.quotientKerEquivRange
```

### 域论定理

```lean
-- 有限域乘法群是循环群
theorem finite_field_units_cyclic {p : ℕ} [Fact p.Prime] (n : ℕ) :
  IsCyclic (Units (FiniteField p n)) := by
  unfold FiniteField
  infer_instance
```

### 模论定理

```lean
-- 同态基本定理
theorem first_isomorphism_module {R : Type*} [Ring R] {M N : Type*}
  [AddCommGroup M] [AddCommGroup N] [Module R M] [Module R N]
  {f : M →ₗ[R] N} :
  M ⧸ (LinearMap.ker f) ≃ₗ[R] (LinearMap.range f) := by
  apply LinearMap.quotKerEquivRange
```

### 李代数定理

-- 伴随表示
def adjointRepresentation {R L : Type*} [CommRing R] [LieRing L]
