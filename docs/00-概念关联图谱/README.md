# 00-概念关联图谱

本目录通过可视化网络与结构化图谱的形式，呈现数学各分支内部及跨分支之间的概念关联，共收录 30 余份文档。与线性教材不同，这里的每一份文件都力图揭示概念之间的双向或多重联系：例如群环域的转化链、线性代数与模论的关联网络、几何拓扑对象的分类映射等。图谱按代数、几何拓扑与跨分支三大主题组织，是学习者在构建整体数学认知框架时的重要导航工具。

## 子目录/文件导航

| 子目录 | 说明 |
|--------|------|
| `代数/` | 群环域、模论、表示理论与代数结构关联图 |
| `几何拓扑/` | 流形、纤维丛、同调与同伦概念网络 |
| `跨分支/` | 代数几何、数论、分析学与数学物理的交叉关联 |

**代表性文件**：
