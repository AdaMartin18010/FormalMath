# 范畴语义、代数语义与逻辑模型的统一理论

语义模型是连接形式逻辑与数学结构的桥梁。从 Tarski 的集合论语义到 Lawvere 的范畴论语义，从 Scott 的连续格到 Girard 的相干空间，语义学的发展深刻改变了我们对逻辑、计算与数学基础的理解。范畴语义学尤为深刻：它将逻辑系统解释为范畴中的结构，使得证明论、类型论与代数几何之间的深层联系得以显现。

## 1. 范畴语义学核心构造

### 1.1 Lawvere 理论

**定义 1.1（Lawvere 理论）**. 代数理论是范畴 $\mathcal{T}$，对象为自然数幂 $[0], [1], [2], \dots$，$[n]$ 为 $[1]$ 的 $n$ 次范畴积。

**定理 1.2**. 群、环、模等代数结构可由 Lawvere 理论描述，其模型为保持有限积的函子 $\mathcal{T} \to \mathbf{Set}$。

### 1.2 线性逻辑的相干语义

**定义 1.3（相干空间）**. 相干空间 $X$ 是带自反、对称关系 $\sim$ 的集合。线性 implication $X \multimap Y$ 由保持相干性的映射组成。

## 2. Topos 理论与几何逻辑

### 2.1 Grothendieck Topos

**定义 2.1（Site）**. Site 是范畴 $\mathcal{C}$ 配备 Grothendieck 拓扑 $J$，对每个对象 $U$ 指定覆盖筛的集族。

**定义 2.2（层）**. 预层 $F: \mathcal{C}^{\text{op}} \to \mathbf{Set}$ 是层，若对覆盖满足粘接条件。

**定理 2.3（Giraud）**. Grothendieck topos 等价于层范畴 $\mathbf{Sh}(\mathcal{C}, J)$。

## 3. 例子

### 3.1 例子：集合 Topos

$\mathbf{Set}$ 是 topos，子对象分类子为 $\Omega = \{0, 1\}$。这对应经典逻辑的布尔值语义。

### 3.2 例子：有效 Topos

有效 topos $\mathbf{Eff}$ 的对象为带等价关系 $\sim_A$ 的集合 $A$，态射为可计算的部分等价关系。这实现了构造性数学的 realizability 语义。
