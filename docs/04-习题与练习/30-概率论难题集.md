## 一、概率不等式

### 习题 1.1 ⭐⭐
**知识点**: Chebyshev不等式、Markov不等式

设 $X$ 是非负随机变量，$a > 0$。证明：
(a) Markov不等式：$P(X \geq a) \leq \frac{E[X]}{a}$
(b) Chebyshev不等式：若 $X$ 有方差，则 $P(|X - E[X]| \geq a) \leq \frac{\text{Var}(X)}{a^2}$

<details>
<summary>详细解答</summary>

**(a)** 
$$E[X] = \int_0^\infty P(X > t) dt \geq \int_0^a P(X > t) dt \geq a \cdot P(X \geq a)$$

或直接：$E[X] \geq E[X \cdot \mathbf{1}_{X \geq a}] \geq a \cdot P(X \geq a)$。

**(b)** 对 $(X - E[X])^2$ 用Markov不等式：
$$P(|X - E[X]| \geq a) = P((X - E[X])^2 \geq a^2) \leq \frac{E[(X-E[X])^2]}{a^2} = \frac{\text{Var}(X)}{a^2}$$
</details>

**相关概念**: [Markov不等式](../../concept/概率不等式.md), [Chebyshev不等式](../../concept/大数定律.md)

### 习题 1.2 ⭐⭐⭐
**知识点**: Chernoff界

设 $X \sim \text{Binomial}(n, p)$，$\mu = np$。证明对 $0 < \delta < 1$：
$$P(X \leq (1-\delta)\mu) \leq \left(\frac{e^{-\delta}}{(1-\delta)^{1-\delta}}\right)^\mu$$

<details>
<summary>关键思路点拨</summary>

利用矩母函数和Markov不等式，优化指数参数。
</details>

<details>
<summary>详细解答</summary>

对任意 $t < 0$，由Markov不等式：
$$P(X \leq (1-\delta)\mu) = P(e^{tX} \geq e^{t(1-\delta)\mu}) \leq \frac{E[e^{tX}]}{e^{t(1-\delta)\mu}}$$

$X = \sum_{i=1}^n X_i$，$X_i \sim \text{Bernoulli}(p)$ i.i.d.。

$$E[e^{tX}] = (E[e^{tX_1}])^n = (1 - p + pe^t)^n = (1 + p(e^t - 1))^n \leq e^{np(e^t-1)} = e^{\mu(e^t-1)}$$

因此：$P(X \leq (1-\delta)\mu) \leq e^{\mu(e^t - 1 - t(1-\delta))}$

优化 $t = \ln(1-\delta) < 0$：
$$e^t - 1 - t(1-\delta) = (1-\delta) - 1 - (1-\delta)\ln(1-\delta) = -\delta - (1-\delta)\ln(1-\delta) = \ln\left(\frac{e^{-\delta}}{(1-\delta)^{1-\delta}}\right)$$
</details>

**变式与推广**: 类似可得上尾界，Chernoff界在算法分析和统计学习中有广泛应用。

### 习题 1.3 ⭐⭐⭐⭐
**知识点**: Hoeffding不等式

设 $X_1, \ldots, X_n$ 是独立随机变量，$X_i \in [a_i, b_i]$，$S_n = \sum_{i=1}^n X_i$。证明对任意 $t > 0$：
$$P(S_n - E[S_n] \geq t) \leq \exp\left(-\frac{2t^2}{\sum_{i=1}^n (b_i - a_i)^2}\right)$$

<details>
<summary>详细解答</summary>

**引理**（Hoeffding引理）：若 $X \in [a, b]$，$E[X] = 0$，则 $E[e^{\lambda X}] \leq e^{\lambda^2(b-a)^2/8}$。

**证明**：由凸性，$e^{\lambda x} \leq \frac{b-x}{b-a}e^{\lambda a} + \frac{x-a}{b-a}e^{\lambda b}$ 对 $x \in [a, b]$。

取期望，令 $p = -a/(b-a)$，$L(\lambda) = \ln(pe^{\lambda(b-a)} + (1-p))$，验证 $L''(\lambda) \leq (b-a)^2/4$ 且 $L(0) = L'(0) = 0$，得 $L(\lambda) \leq \lambda^2(b-a)^2/8$。

**主证明**：设 $Y_i = X_i - E[X_i]$，则 $E[e^{\lambda S_n}] = \prod_i E[e^{\lambda Y_i}] \leq e^{\lambda^2\sum(b_i-a_i)^2/8}$。

由Chernoff方法：
$$P(S_n - E[S_n] \geq t) \leq e^{-\lambda t} E[e^{\lambda(S_n - E[S_n])}] \leq e^{-\lambda t + \lambda^2\sum(b_i-a_i)^2/8}$$

取 $\lambda = 4t/\sum(b_i-a_i)^2$ 使指数最小，得结论。
</details>

### 习题 1.4 ⭐⭐⭐⭐
**知识点**: Doob极大不等式

设 $(M_n)$ 是非负下鞅。证明对 $\lambda > 0$：
$$\lambda P(\max_{0 \leq k \leq n} M_k \geq \lambda) \leq E[M_n]$$

<details>
<summary>详细解答</summary>

设 $T = \min\{k: M_k \geq \lambda\} \wedge n$（停时）。

由下鞅的停时定理（可选停时定理），$E[M_T] \leq E[M_n]$。

而 $M_T \geq \lambda \cdot \mathbf{1}_{\{\max M_k \geq \lambda\}}$，因若停时触发，$M_T \geq \lambda$；否则 $T = n$。

故 $E[M_T] \geq \lambda P(\max M_k \geq \lambda)$，结合得不等式。
</details>

### 习题 1.5 ⭐⭐⭐⭐⭐
**知识点**: 鞅差序列、Azuma不等式

设 $(X_k)$ 是鞅差序列，$|X_k| \leq c_k$ a.s.。证明对任意 $t > 0$：
$$P\left(\sum_{k=1}^n X_k \geq t\right) \leq \exp\left(-\frac{t^2}{2\sum c_k^2}\right)$$

<details>
<summary>详细解答</summary>

这是Azuma不等式，是鞅的Hoeffding不等式。

**证明**：令 $S_n = \sum_{k=1}^n X_k$，则 $(S_n)$ 是鞅。

$E[e^{\lambda S_n}] = E[e^{\lambda S_{n-1}} E[e^{\lambda X_n} | \mathcal{F}_{n-1}]]$。

条件Hoeffding引理：给定 $\mathcal{F}_{n-1}$，$X_n$ 条件均值为0且 $|X_n| \leq c_n$，故 $E[e^{\lambda X_n} | \mathcal{F}_{n-1}] \leq e^{\lambda^2 c_n^2/2}$。

归纳得 $E[e^{\lambda S_n}] \leq e^{\lambda^2 \sum c_k^2/2}$。

Chernoff优化得结论。
</details>

## 二、极限定理

### 习题 2.1 ⭐⭐
**知识点**: 弱大数定律

设 $(X_n)$ 是i.i.d.随机变量序列，$E[X_1] = \mu$，$\text{Var}(X_1) = \sigma^2 < \infty$。证明 $\bar{X}_n = \frac{1}{n}\sum_{i=1}^n X_i \xrightarrow{P} \mu$。

<details>
<summary>详细解答</summary>

**Chebyshev方法**：
$$E[\bar{X}_n] = \mu, \quad \text{Var}(\bar{X}_n) = \frac{\sigma^2}{n}$$

对任意 $\varepsilon > 0$：
$$P(|\bar{X}_n - \mu| \geq \varepsilon) \leq \frac{\text{Var}(\bar{X}_n)}{\varepsilon^2} = \frac{\sigma^2}{n\varepsilon^2} \to 0$$

因此 $\bar{X}_n \xrightarrow{P} \mu$。

**注**：此证需有限方差。一般情况（仅需 $E|X_1| < \infty$）可用截断法或特征函数法证明。
</details>

---

### 习题 2.2 ⭐⭐⭐
**知识点**: 强大数定律

在习题2.1条件下，证明 $\bar{X}_n \xrightarrow{a.s.} \mu$。

<details>
<summary>关键思路点拨</summary>

用Borel-Cantelli引理，但对直接用Chebyshev不够。需用截断或子序列论证。
</details>

<details>
<summary>详细解答</summary>

**方法**：先证 $E[X_1^4] < \infty$ 的强情形，再逼近。

设 $E[X^4] < \infty$，$E[X] = 0$（不妨）。计算：
$$E[\bar{X}_n^4] = \frac{1}{n^4}\sum_{i,j,k,l} E[X_i X_j X_k X_l]$$

非零项：$i=j=k=l$（$n$ 项，各 $E[X^4]$）或两对相等（$3n(n-1)$ 项，各 $(E[X^2])^2$）。

$E[\bar{X}_n^4] = O(1/n^2)$，故 $\sum_n E[\bar{X}_n^4] < \infty$，$\sum_n \bar{X}_n^4 < \infty$ a.s.，即 $\bar{X}_n \to 0$ a.s.。

**一般情形**：用截断 $X_k' = X_k \mathbf{1}_{|X_k| \leq k}$，验证Borel-Cantelli条件等。
</details>

---

### 习题 2.3 ⭐⭐⭐
**知识点**: 中心极限定理

设 $(X_n)$ i.i.d.，$E[X_1] = \mu$，$\text{Var}(X_1) = \sigma^2 < \infty$。证明：
$$\frac{\sqrt{n}(\bar{X}_n - \mu)}{\sigma} \xrightarrow{d} N(0, 1)$$

<details>
<summary>详细解答</summary>

**特征函数法**：设 $\varphi(t) = E[e^{itX_1}]$。

标准化：设 $\mu = 0$，$\sigma = 1$。令 $S_n = \sum_{k=1}^n X_k$。

$$\varphi_{S_n/\sqrt{n}}(t) = E[e^{itS_n/\sqrt{n}}] = (\varphi(t/\sqrt{n}))^n$$

Taylor展开：$\varphi(s) = 1 + isE[X] - \frac{s^2}{2}E[X^2] + o(s^2) = 1 - \frac{s^2}{2} + o(s^2)$。

故 $\varphi(t/\sqrt{n}) = 1 - \frac{t^2}{2n} + o(1/n)$。

$$\varphi_{S_n/\sqrt{n}}(t) = \left(1 - \frac{t^2}{2n} + o(1/n)\right)^n \to e^{-t^2/2}$$

右边是标准正态的特征函数，由连续性定理得证。
</details>

**相关概念**: [中心极限定理](../../concept/概率极限理论.md), [特征函数](../../concept/概率论.md)

---

### 习题 2.4 ⭐⭐⭐⭐
**知识点**: 重对数律

设 $(X_n)$ i.i.d.，$E[X_1] = 0$，$\text{Var}(X_1) = 1$。证明：
$$\limsup_{n \to \infty} \frac{S_n}{\sqrt{2n\ln\ln n}} = 1 \quad \text{a.s.}$$

<details>
<summary>关键思路点拨</summary>

这是Hartman-Wintner重对数律。证明复杂，需精细的Borel-Cantelli论证和指数不等式。
</details>

<details>
<summary>详细解答（概要）</summary>

**上界**：对任意 $\varepsilon > 0$，证 $P(S_n > (1+\varepsilon)\sqrt{2n\ln\ln n} \text{ i.o.}) = 0$。

用指数不等式（如Bernstein），对子序列 $n_k = \lfloor (1+\eta)^k \rfloor$ 论证。

**下界**：证 $P(S_n > (1-\varepsilon)\sqrt{2n\ln\ln n} \text{ i.o.}) = 1$。

利用独立增量性质和第二个Borel-Cantelli引理。

**精确计算**：通过Skorokhod嵌入或鞅方法可得更精确结果。
</details>

### 习题 2.5 ⭐⭐⭐⭐⭐
**知识点**: 大偏差原理

设 $(X_n)$ i.i.d.，矩母函数 $M(t) = E[e^{tX}] < \infty$ 在某区间成立。证明对 $a > E[X]$：
$$\lim_{n \to \infty} \frac{1}{n}\ln P(S_n \geq na) = -I(a)$$
其中 $I(a) = \sup_{t > 0}(ta - \ln M(t))$ 是速率函数。

<details>
<summary>详细解答</summary>

**Cramér定理**。

**上界**：由Chernoff界，对任意 $t > 0$：
$$P(S_n \geq na) \leq e^{-nta}M(t)^n = e^{-n(ta - \ln M(t))}$$

优化 $t$ 得 $P(S_n \geq na) \leq e^{-nI(a)}$，即上界 $\leq -I(a)$。

**下界**：换测度（tilting）。定义新测度 $d\tilde{P} = \frac{e^{tX}}{M(t)}dP$，在此测度下 $E_{\tilde{P}}[X] = \frac{M'(t)}{M(t)}$。

选取 $t$ 使 $E_{\tilde{P}}[X] = a$（即 $\frac{d}{dt}\ln M(t) = a$）。

在此测度下用中心极限定理，可证 $P(S_n \approx na)$ 多项式衰减，而Radon-Nikodym导数贡献指数因子 $e^{-nI(a)}$。

综合得下界 $\geq -I(a)$。
</details>

## 三、随机游走

### 习题 3.1 ⭐⭐
**知识点**: 简单对称随机游走

设 $(S_n)$ 是 $\mathbb{Z}$ 上的简单对称随机游走，$S_0 = 0$。证明：
(a) $P(S_{2n} = 0) = \binom{2n}{n} 2^{-2n} \sim \frac{1}{\sqrt{\pi n}}$
(b) 游走是常返的（以概率1返回原点无穷多次）

<details>
<summary>详细解答</summary>

**(a)** $S_{2n} = 0$ 需 $n$ 步向右，$n$ 步向左，概率为 $\binom{2n}{n}/2^{2n}$。

由Stirling公式：$n! \sim \sqrt{2\pi n}(n/e)^n$

$$\binom{2n}{n} = \frac{(2n)!}{(n!)^2} \sim \frac{\sqrt{4\pi n}(2n/e)^{2n}}{2\pi n (n/e)^{2n}} = \frac{4^n}{\sqrt{\pi n}}$$

故 $\binom{2n}{n}2^{-2n} \sim \frac{1}{\sqrt{\pi n}}$。

**(b)** 由 (a)，$\sum_{n=1}^\infty P(S_{2n} = 0) = \infty$。

由Borel-Cantelli（适用于独立事件，这里需用对称性和Markov性质），返回原点无穷多次的概率为1。
</details>

---

### 习题 3.2 ⭐⭐⭐
**知识点**: 随机游走的常返性

证明 $\mathbb{Z}^d$ 上的简单对称随机游走在 $d = 1, 2$ 时常返，$d \geq 3$ 时暂态。

<details>
<summary>详细解答</summary>

**关键**：$P(S_{2n} = 0) \sim c_d n^{-d/2}$（局部中心极限定理）。

$d = 1$：$P(S_{2n} = 0) \sim \frac{1}{\sqrt{\pi n}}$，$\sum n^{-1/2} = \infty$，常返。

$d = 2$：$P(S_{2n} = 0) \sim \frac{1}{\pi n}$，$\sum n^{-1} = \infty$，常返。

$d \geq 3$：$P(S_{2n} = 0) \sim c n^{-d/2}$，$\sum n^{-d/2} < \infty$（$d/2 > 1$），暂态。

**详细计算**（$d = 2$）：
$$P(S_{2n} = 0) = \sum_{k=0}^n \binom{2n}{2k}\binom{2k}{k}\binom{2n-2k}{n-k} 4^{-2n} = 4^{-2n}\binom{2n}{n}\sum_k \binom{n}{k}^2 = 4^{-2n}\binom{2n}{n}^2$$

由 (a) 得 $P(S_{2n} = 0) \sim \frac{1}{\pi n}$。
</details>

**相关概念**: [随机游走](../../concept/随机过程.md), [常返性](../../concept/Markov链.md)

### 习题 3.3 ⭐⭐⭐⭐
**知识点**: 反正弦律

设 $(S_n)$ 是简单对称随机游走。令 $L_{2n}$ 为前 $2n$ 步中正值游走的次数。证明：
$$P(L_{2n} = 2k) = \binom{2k}{k}\binom{2n-2k}{n-k} 4^{-n}$$
并由此推导反正弦分布。

<details>
<summary>详细解答</summary>

**组合论证**（André反射原理的变体）：$L_{2n} = 2k$ 意味着游走 $2k$ 步在正侧，$2n-2k$ 步在负侧（或零）。

由 ballot 定理类型的计数，概率为：
$$P(L_{2n} = 2k) = u_{2k} u_{2n-2k}$$

其中 $u_{2m} = \binom{2m}{m}4^{-m} = P(S_{2m} = 0)$。

**反正弦分布**：对 $0 < a < b < 1$：
$$P(a \leq L_{2n}/(2n) \leq b) \to \int_a^b \frac{dx}{\pi\sqrt{x(1-x)}} = \frac{2}{\pi}(\arcsin\sqrt{b} - \arcsin\sqrt{a})$$

这说明 arcsine 分布是 Occupation time 的极限分布，违反直觉（大多数人猜测是均匀或正态）。
</details>

### 习题 3.4 ⭐⭐⭐⭐
**知识点**: 随机游走的尺度极限

证明适当尺度化的简单随机游走收敛于布朗运动（Donsker不变原理）。

<details>
<summary>详细解答</summary>

**构造**：令 $X^{(n)}(t) = \frac{S_{\lfloor nt \rfloor}}{\sqrt{n}}$，线性插值得连续路径。

**有限维分布**：由中心极限定理，对固定 $t_1 < \cdots < t_k$：
$$(X^{(n)}(t_1), X^{(n)}(t_2)-X^{(n)}(t_1), \ldots) \xrightarrow{d} (B(t_1), B(t_2)-B(t_1), \ldots)$$

即增量独立且服从适当方差的正态分布，与布朗运动一致。

**紧性**：验证 $\{X^{(n)}\}$ 在 $C[0,1]$ 中胎紧。用Kolmogorov-Chentsov准则或鞅不等式证等度连续性。

具体：对任意 $\varepsilon > 0$，
$$P(\sup_{|s-t| < \delta} |X^{(n)}(s) - X^{(n)}(t)| > \varepsilon) \to 0$$

当 $\delta \to 0$ 对 $n$ 一致。

由Prokhorov定理，胎紧加有限维收敛推出弱收敛于布朗运动。
</details>

### 习题 3.5 ⭐⭐⭐⭐⭐
**知识点**: 局部时、游程理论

设 $(B_t)$ 是标准布朗运动，$L_t^a$ 是 $a$ 点的局部时。证明 occupation density 公式：
$$\int_0^t f(B_s) ds = \int_{-\infty}^\infty f(a) L_t^a da$$

<details>
<summary>详细解答（概要）</summary>

**Tanaka公式**：
$$|B_t - a| = |B_0 - a| + \int_0^t \text{sgn}(B_s - a) dB_s + L_t^a$$

这是Itô公式的推广（$|x|$ 不可导于0）。

**occupation density**：对 $f \geq 0$ 可测，由Fubini和Tanaka：
$$\int_0^t f(B_s) ds = \int_{-\infty}^\infty f(a) \left(\int_0^t \delta(B_s - a) ds\right) da$$

形式上将 $L_t^a$ 解释为 $\int_0^t \delta(B_s - a) ds$。

**严格证明**：用光滑逼近 $\delta$ 函数，取极限，利用局部时的连续性等性质。

**应用**：局部时 $L_t^a$ 度量布朗运动在 $a$ 附近"花费"的时间密度。
</details>

## 四、鞅的应用

### 习题 4.1 ⭐⭐⭐
**知识点**: 鞅停时定理

设 $(M_n)$ 是鞅，$T$ 是有界停时。证明 $E[M_T] = E[M_0]$。

<details>
<summary>详细解答</summary>

设 $T \leq N$ a.s.。将 $M_T$ 写成望远镜和：
$$M_T = M_0 + \sum_{n=0}^{N-1} (M_{n+1} - M_n)\mathbf{1}_{T > n}$$

取期望：
$$E[M_T] = E[M_0] + \sum_{n=0}^{N-1} E[(M_{n+1} - M_n)\mathbf{1}_{T > n}]$$

关键：$\{T > n\} = \{T \leq n\}^c \in \mathcal{F}_n$，故：
$$E[(M_{n+1} - M_n)\mathbf{1}_{T > n}] = E[E[M_{n+1} - M_n | \mathcal{F}_n]\mathbf{1}_{T > n}] = 0$$

因此 $E[M_T] = E[M_0]$。

**注**：$T$ 有界条件可放松为 $E[T] < \infty$ 且 $(M_n)$ 有界增量等条件。
</details>

---

### 习题 4.2 ⭐⭐⭐⭐
**知识点**: Wald等式

设 $(X_n)$ i.i.d.，$E|X_1| < \infty$，$T$ 是关于 $(\mathcal{F}_n = \sigma(X_1, \ldots, X_n))$ 的停时，$E[T] < \infty$。证明：
$$E\left[\sum_{n=1}^T X_n\right] = E[T] \cdot E[X_1]$$

<details>
<summary>详细解答</summary>

设 $\mu = E[X_1]$。定义 $M_n = \sum_{k=1}^n (X_k - \mu)$，这是鞅。

$M_T = \sum_{k=1}^T X_k - \mu T$。

由鞅停时定理（验证条件）：需 $E|M_T| < \infty$ 或 $E[T] < \infty$ 且 $X_k$ 有界等。

严格验证：设 $S_n = \sum_{k=1}^n X_k$，则 $S_{T \wedge n}$ 是鞅（有界停时 $T \wedge n$）。

$E[S_{T \wedge n}] = \mu E[T \wedge n]$。

由单调收敛，$E[T \wedge n] \uparrow E[T]$。

若 $X_k \geq 0$ 或 $E[\sum_{k=1}^T |X_k|] < \infty$，可用控制收敛或单调收敛得 $E[S_T] = \mu E[T]$。
</details>

---

### 习题 4.3 ⭐⭐⭐⭐
**知识点**: 鞅收敛定理

设 $(M_n)$ 是上鞅，$\sup_n E|M_n| < \infty$。证明 $M_n$ 几乎必然收敛于可积随机变量 $M_\infty$。

<details>
<summary>详细解答</summary>

**上穿不等式**：设 $U_n(a, b)$ 是 $M_0, \ldots, M_n$ 上穿区间 $[a, b]$ 的次数。

Doob上穿不等式：$(b-a)E[U_n(a,b)] \leq E[(M_n - a)^-] \leq |a| + E|M_n|$。

故 $\sup_n E[U_n(a,b)] < \infty$，$U_\infty(a,b) = \lim_n U_n(a,b) < \infty$ a.s.。

**收敛性**：有理数对 $(a, b)$ 可数，故 a.s. 对所有有理 $a < b$，$U_\infty(a, b) < \infty$。

若 $M_n$ 不收敛，则 $\liminf M_n < \limsup M_n$，存在有理 $a < b$ 使上穿无限次，矛盾。

因此 $M_n \to M_\infty$ a.s.。

$E|M_\infty| \leq \liminf E|M_n| < \infty$（Fatou）。
</details>

### 习题 4.4 ⭐⭐⭐⭐⭐
**知识点**: 可选停时定理、博弈策略

考虑公平博弈，每次输赢1元概率各1/2。赌徒采用martingale策略（输后加倍下注直到赢）。证明：
(a) 以概率1最终赢1元；
(b) 但期望损失无限。

<details>
<summary>详细解答</summary>

**martingale策略**：第一次下注1元，若输则下注2元，再输则4元，以此类推。赢时重置为1元。

**(a)** 设 $T$ 是首次赢的局数。$P(T = n) = 2^{-n}$，$P(T < \infty) = \sum_{n=1}^\infty 2^{-n} = 1$。

赢时赢得 $2^n - (1 + 2 + \cdots + 2^{n-1}) = 2^n - (2^n - 1) = 1$ 元。

**(b)** 第 $n$ 局下注 $2^{n-1}$ 元，若 $T = n$，总损失 $1 + 2 + \cdots + 2^{n-1} = 2^n - 1$。

期望损失：$E[2^T - 1] = \sum_{n=1}^\infty (2^n - 1) \cdot 2^{-n} = \sum_{n=1}^\infty (1 - 2^{-n}) = \infty$。

这违反了可选停时定理的条件（期望停时 $E[T] = \sum n 2^{-n} = 2 < \infty$，但鞅 $(M_n)$（财富过程）不满足有界性条件）。
</details>

### 习题 4.5 ⭐⭐⭐⭐⭐
**知识点**: 鞅方法在金融数学中的应用

在Black-Scholes模型中，证明欧式看涨期权的定价公式。

<details>
<summary>详细解答（概要）</summary>

**模型**：股票价格 $dS_t = \mu S_t dt + \sigma S_t dW_t$。

**风险中性测度**：存在等价鞅测度 $Q$，使得 $e^{-rt}S_t$ 是 $Q$-鞅。

在 $Q$ 下：$dS_t = rS_t dt + \sigma S_t d\tilde{W}_t$。

**定价**：期权在 $t$ 时刻价值 $V(t, S_t) = e^{-r(T-t)}E_Q[(S_T - K)^+ | \mathcal{F}_t]$。

**求解**：$S_T = S_t \exp\left((r - \sigma^2/2)(T-t) + \sigma(\tilde{W}_T - \tilde{W}_t)\right)$。

$\ln(S_T/S_t) \sim N((r-\sigma^2/2)\tau, \sigma^2\tau)$，$\tau = T-t$。

直接计算期望：
$$V(t, s) = s\Phi(d_1) - Ke^{-r\tau}\Phi(d_2)$$

其中 $d_1 = \frac{\ln(s/K) + (r + \sigma^2/2)\tau}{\sigma\sqrt{\tau}}$，$d_2 = d_1 - \sigma\sqrt{\tau}$，$\Phi$ 是标准正态CDF。

这就是著名的Black-Scholes公式。
</details>
