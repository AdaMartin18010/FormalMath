# 数学习题与练习精选

本节汇集从基础到高等数学的经典习题，涵盖分析、代数、几何与拓扑等核心分支。每道习题配有详细解答，展示证明技巧与解题策略。

## 1. 分析学习题

### 1.1 极限与连续性

**习题 1.1**. 证明 $f(x) = x^2$ 在 $\mathbb{R}$ 上连续。

**解答**. 对任意 $a \in \mathbb{R}$，$|f(x) - f(a)| = |x^2 - a^2| = |x+a||x-a|$。当 $|x-a| < 1$ 时，$|x+a| \leq |x-a| + 2|a| < 1 + 2|a|$。故对 $\epsilon > 0$，取 $\delta = \min(1, \epsilon/(1+2|a|))$，则 $|x-a| < \delta$ 蕴含 $|f(x)-f(a)| < \epsilon$。$\square$

### 1.2 级数收敛

**习题 1.2**. 判断 $\sum_{n=1}^\infty \frac{1}{n^2}$ 的收敛性。

**解答**. 由积分判别法或 Cauchy 凝聚判别法，$\sum 2^n \cdot \frac{1}{(2^n)^2} = \sum \frac{1}{2^n}$ 收敛，故原级数收敛。事实上其和为 $\pi^2/6$（Euler）。$\square$

## 2. 代数学习题

### 2.1 群论

**习题 2.1**. 证明 $S_3$ 有 6 个元素，并列出所有子群。

**解答**. $S_3 = \{e, (12), (13), (23), (123), (132)\}$。子群：$\{e\}$，$\langle (12) \rangle$，$\langle (13) \rangle$，$\langle (23) \rangle$，$A_3 = \{e, (123), (132)\}$，$S_3$。正规子群：$\{e\}$，$A_3$，$S_3$。$\square$

### 2.2 环论

**习题 2.2**. 证明 $\mathbb{Z}[i]$ 是 Euclidean 整环。

**解答**. 范数 $N(a+bi) = a^2 + b^2$。对 $\alpha, \beta \neq 0$，$\alpha/\beta = q + r'$（$q \in \mathbb{Z}[i]$，$|r'| < 1$）。取 $\gamma = q$，则 $\alpha = \beta\gamma + \rho$，$N(\rho) = N(\beta)N(r') < N(\beta)$。$\square$

## 3. 几何学习题

### 3.1 微分几何

**习题 3.1**. 计算单位球面 $S^2$ 的 Gauss 曲率。

**解答**. 参数化 $r(u,v) = (\cos u \sin v, \sin u \sin v, \cos v)$。计算 $E = \sin^2 v$，$F = 0$，$G = 1$；$L = \sin^2 v$，$M = 0$，$N = 1$。$K = (LN - M^2)/(EG - F^2) = 1$。$\square$

## 4. 拓扑学习题

### 4.1 基本群

**习题 4.1**. 计算 $\pi_1(S^1 \times S^1)$。

**解答**. 由乘积公式，$\pi_1(S^1 \times S^1) = \pi_1(S^1) \times \pi_1(S^1) = \mathbb{Z} \times \mathbb{Z}$。$\square$
