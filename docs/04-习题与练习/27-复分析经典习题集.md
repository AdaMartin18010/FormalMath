## 一、全纯函数性质

### 习题 1.1 ⭐⭐
**知识点**: Cauchy-Riemann方程

设 $f(z) = u(x,y) + iv(x,y)$ 在区域 $D$ 内可微。证明：$f$ 在 $D$ 内全纯当且仅当 $u, v$ 满足Cauchy-Riemann方程：
$$\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}, \quad \frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}$$

<details>
<summary>详细解答</summary>

**必要性**：设 $f$ 全纯，则极限 $f'(z) = \lim_{h \to 0} \frac{f(z+h)-f(z)}{h}$ 存在且与 $h$ 趋于0的方式无关。

取 $h = \Delta x \to 0$（实数）：
$$f'(z) = \frac{\partial u}{\partial x} + i\frac{\partial v}{\partial x}$$

取 $h = i\Delta y \to 0$（纯虚数）：
$$f'(z) = -i\frac{\partial u}{\partial y} + \frac{\partial v}{\partial y}$$

比较实部虚部即得C-R方程。

**充分性**：由C-R方程和实可微性，可验证差商极限与路径无关。
</details>

**相关概念**: [全纯函数](../../concept/复变函数.md), [Cauchy-Riemann方程](../../concept/复微分.md)

---

### 习题 1.2 ⭐⭐⭐
**知识点**: 解析延拓、唯一性定理

设 $f$ 和 $g$ 是区域 $D$ 内的全纯函数，且 $\{z \in D: f(z) = g(z)\}$ 在 $D$ 内有聚点。证明 $f \equiv g$ 在 $D$ 内成立。

<details>
<summary>关键思路点拨</summary>

全纯函数的零点若有聚点必恒为零。考虑 $h = f - g$。
</details>

<details>
<summary>详细解答</summary>

令 $h = f - g$，则 $h$ 全纯且零点集 $Z(h) = \{z: f(z) = g(z)\}$ 有聚点 $z_0 \in D$。

由于 $h$ 全纯，在 $z_0$ 附近有Taylor展开：
$$h(z) = \sum_{n=0}^\infty a_n (z-z_0)^n$$

若 $h \not\equiv 0$，设 $a_m$ 是第一个非零系数，则：
$$h(z) = (z-z_0)^m \sum_{n=0}^\infty a_{m+n}(z-z_0)^n = (z-z_0)^m \varphi(z)$$

其中 $\varphi(z_0) = a_m \neq 0$。由连续性，$\varphi \neq 0$ 在 $z_0$ 某邻域内，故 $h \neq 0$ 在该去心邻域内，与 $z_0$ 是聚点矛盾。

因此 $h \equiv 0$，即 $f \equiv g$。
</details>

**变式与推广**: 若 $f, g$ 在曲线或具有极限点的集合上相等，结论同样成立。

### 习题 1.3 ⭐⭐⭐
**知识点**: 最大模原理

设 $f$ 在有界区域 $D$ 内全纯，在 $\bar{D}$ 上连续，且 $f$ 不恒为常数。证明 $|f|$ 在 $\bar{D}$ 上的最大值只能在边界 $\partial D$ 上达到。

<details>
<summary>关键思路点拨</summary>

用平均值性质。若内部某点达最大模，则在该点邻域内 $|f|$ 为常数，进而 $f$ 为常数。
</details>

<details>
<summary>详细解答</summary>

**步骤1**：假设 $|f|$ 在 $z_0 \in D$ 达到最大值 $M$。

**步骤2**：由平均值性质，对充分小 $r > 0$：
$$f(z_0) = \frac{1}{2\pi} \int_0^{2\pi} f(z_0 + re^{i\theta}) d\theta$$

取模：$M = |f(z_0)| \leq \frac{1}{2\pi} \int_0^{2\pi} |f(z_0 + re^{i\theta})| d\theta \leq M$

因此等号成立，即 $|f(z_0 + re^{i\theta})| = M$ 对所有 $\theta$ 成立。

**步骤3**：由 $r$ 的任意性，$|f| = M$ 在 $z_0$ 的某邻域内恒成立。

**步骤4**：由开映射定理的推论，若 $|f|$ 在区域内为常数，则 $f$ 为常数，矛盾。
</details>

**相关概念**: [最大模原理](../../concept/复分析.md), [平均值性质](../../concept/调和函数.md)

### 习题 1.4 ⭐⭐⭐⭐
**知识点**: Schwarz引理

设 $f: \mathbb{D} \to \mathbb{D}$ 全纯，$f(0) = 0$，其中 $\mathbb{D} = \{z: |z| < 1\}$。证明：
(a) $|f(z)| \leq |z|$ 对所有 $z \in \mathbb{D}$ 成立；
(b) $|f'(0)| \leq 1$；
(c) 若某点等号成立，则 $f(z) = e^{i\theta}z$。

<details>
<summary>详细解答</summary>

**(a)** 定义 $g(z) = f(z)/z$（$z \neq 0$），$g(0) = f'(0)$。由Riemann可去奇点定理，$g$ 在 $\mathbb{D}$ 内全纯。

对 $|z| = r < 1$，$|g(z)| = |f(z)|/|z| \leq 1/r$。令 $r \to 1$，由最大模原理，$|g(z)| \leq 1$ 在 $\mathbb{D}$ 内，即 $|f(z)| \leq |z|$。

**(b)** $|f'(0)| = |g(0)| \leq 1$。

**(c)** 若在某 $z_0 \neq 0$ 有 $|f(z_0)| = |z_0|$，则 $|g(z_0)| = 1$。由最大模原理，$|g| \equiv 1$，即 $g(z) = e^{i\theta}$ 为常数，$f(z) = e^{i\theta}z$。
</details>

### 习题 1.5 ⭐⭐⭐⭐
**知识点**: Liouville定理

设 $f$ 是整函数（在整个 $\mathbb{C}$ 上全纯），且存在常数 $M, R > 0$ 和正整数 $n$ 使得 $|f(z)| \leq M|z|^n$ 对所有 $|z| > R$ 成立。证明 $f$ 是次数不超过 $n$ 的多项式。

<details>
<summary>详细解答</summary>

对整函数 $f$，有Taylor展开 $f(z) = \sum_{k=0}^\infty a_k z^k$，其中 $a_k = \frac{f^{(k)}(0)}{k!} = \frac{1}{2\pi i} \oint_{|z|=r} \frac{f(z)}{z^{k+1}} dz$。

对 $r > R$：
$$|a_k| \leq \frac{1}{2\pi} \cdot 2\pi r \cdot \frac{Mr^n}{r^{k+1}} = \frac{M}{r^{k-n}}$$

当 $k > n$，令 $r \to \infty$ 得 $|a_k| = 0$。

因此 $f(z) = a_0 + a_1 z + \cdots + a_n z^n$ 是次数 $\leq n$ 的多项式。
</details>

## 二、积分计算

### 习题 2.1 ⭐⭐
**知识点**: Cauchy积分公式

设 $f$ 在 $|z - z_0| < R$ 内全纯，$0 < r < R$。证明：
$$f(z_0) = \frac{1}{2\pi} \int_0^{2\pi} f(z_0 + re^{i\theta}) d\theta$$

<details>
<summary>详细解答</summary>

由Cauchy积分公式，对圆周 $C: |z - z_0| = r$：
$$f(z_0) = \frac{1}{2\pi i} \oint_C \frac{f(z)}{z - z_0} dz$$

参数化 $z = z_0 + re^{i\theta}$，$dz = ire^{i\theta} d\theta$：
$$f(z_0) = \frac{1}{2\pi i} \int_0^{2\pi} \frac{f(z_0 + re^{i\theta})}{re^{i\theta}} \cdot ire^{i\theta} d\theta = \frac{1}{2\pi} \int_0^{2\pi} f(z_0 + re^{i\theta}) d\theta$$
</details>

---

### 习题 2.2 ⭐⭐⭐
**知识点**: 围道积分

计算积分 $\int_{-\infty}^\infty \frac{dx}{1+x^4}$。

<details>
<summary>关键思路点拨</summary>

上半平面有两个极点：$e^{i\pi/4}$ 和 $e^{3i\pi/4}$。用留数定理。
</details>

<details>
<summary>详细解答</summary>

考虑 $f(z) = \frac{1}{1+z^4}$，极点为 $z_k = e^{i(\pi + 2k\pi)/4}$，$k = 0, 1, 2, 3$。

上半平面：$z_0 = e^{i\pi/4}$，$z_1 = e^{3i\pi/4}$。

留数计算（单极点）：
$$\text{Res}(f, z_k) = \frac{1}{4z_k^3} = \frac{z_k}{4z_k^4} = -\frac{z_k}{4}$$

（因为 $z_k^4 = -1$）

$$\text{Res}(f, z_0) = -\frac{e^{i\pi/4}}{4}, \quad \text{Res}(f, z_1) = -\frac{e^{3i\pi/4}}{4}$$

留数和：
$$-\frac{1}{4}(e^{i\pi/4} + e^{3i\pi/4}) = -\frac{1}{4} \cdot 2i\sin(\pi/4) = -\frac{i\sqrt{2}}{4}$$

（利用 $e^{i\pi/4} + e^{3i\pi/4} = 2i\sin(\pi/4) = i\sqrt{2}$）

实际上直接计算：$e^{i\pi/4} = \frac{1+i}{\sqrt{2}}$，$e^{3i\pi/4} = \frac{-1+i}{\sqrt{2}}$，和为 $\frac{2i}{\sqrt{2}} = i\sqrt{2}$。

积分值：$2\pi i \cdot (-\frac{i\sqrt{2}}{4}) = \frac{\pi\sqrt{2}}{2}$。
</details>

---

### 习题 2.3 ⭐⭐⭐⭐
**知识点**: Jordan引理、Fourier型积分

对 $a > 0$，计算 $\int_0^\infty \frac{\cos(ax)}{1+x^2} dx$。

<details>
<summary>详细解答</summary>

考虑 $f(z) = \frac{e^{iaz}}{1+z^2}$，在实轴上的实部即为所求。

上半平面极点：$z = i$。

$$\text{Res}(f, i) = \frac{e^{iai}}{2i} = \frac{e^{-a}}{2i}$$

由Jordan引理，沿上半大圆弧的积分趋于0（因 $a > 0$）。

$$\int_{-\infty}^\infty \frac{e^{iax}}{1+x^2} dx = 2\pi i \cdot \frac{e^{-a}}{2i} = \pi e^{-a}$$

取实部：
$$\int_{-\infty}^\infty \frac{\cos(ax)}{1+x^2} dx = \pi e^{-a}$$

由被积函数偶性：$\int_0^\infty \frac{\cos(ax)}{1+x^2} dx = \frac{\pi e^{-a}}{2}$。
</details>

### 习题 2.4 ⭐⭐⭐⭐
**知识点**: 多值函数、分支切割

计算积分 $\int_0^\infty \frac{x^{a-1}}{1+x} dx$，其中 $0 < a < 1$。

<details>
<summary>关键思路点拨</summary>

考虑 $f(z) = \frac{z^{a-1}}{1+z}$，取分支切割沿正实轴，用钥匙孔围道。
</details>

<details>
<summary>详细解答</summary>

**围道**：大圆 $|z| = R$，小圆 $|z| = \varepsilon$，连接上下岸的钥匙孔围道。

极点：$z = -1 = e^{i\pi}$。

**上岸边**（$z = xe^{i0}$）：$z^{a-1} = x^{a-1}$

**下岸边**（$z = xe^{i2\pi}$）：$z^{a-1} = x^{a-1}e^{i2\pi(a-1)} = x^{a-1}e^{2\pi i(a-1)}$

留数：
$$\text{Res}(f, -1) = (-1)^{a-1} = e^{i\pi(a-1)} = -e^{i\pi a}$$

围道积分：$2\pi i \cdot (-e^{i\pi a}) = -2\pi i e^{i\pi a}$

上下岸贡献：
$$\int_\varepsilon^R \frac{x^{a-1}}{1+x} dx + \int_R^\varepsilon \frac{x^{a-1}e^{2\pi i(a-1)}}{1+x} dx = (1 - e^{2\pi ia})\int_0^\infty \frac{x^{a-1}}{1+x} dx$$

令 $R \to \infty$，$\varepsilon \to 0$，大圆小圆积分趋于0（需验证）。

因此：
$$(1 - e^{2\pi ia})I = -2\pi i e^{i\pi a}$$

$$I = \frac{-2\pi i e^{i\pi a}}{1 - e^{2\pi ia}} = \frac{-2\pi i}{e^{-i\pi a} - e^{i\pi a}} = \frac{\pi}{\sin(\pi a)}$$
</details>

**相关概念**: [Gamma函数](../../concept/特殊函数.md), [Beta函数](../../concept/特殊函数.md)

### 习题 2.5 ⭐⭐⭐⭐⭐
**知识点**: 围道积分技巧

计算 $\int_0^\infty \frac{\ln x}{1+x^2} dx$。

<details>
<summary>详细解答</summary>

**方法**：考虑 $f(z) = \frac{(\ln z)^2}{1+z^2}$，分支切割沿正实轴。

极点：$z = i = e^{i\pi/2}$，$z = -i = e^{3i\pi/2}$（在分支切割下）。

上岸边：$\ln x$；下岸边：$\ln x + 2\pi i$。

围道积分（钥匙孔）：
$$\int_0^\infty \frac{(\ln x)^2 - (\ln x + 2\pi i)^2}{1+x^2} dx = -4\pi i \int_0^\infty \frac{\ln x}{1+x^2} dx + 4\pi^2 \int_0^\infty \frac{dx}{1+x^2}$$

右边等于 $2\pi i[\text{Res}(f, i) + \text{Res}(f, -i)]$。

$$\text{Res}(f, i) = \frac{(i\pi/2)^2}{2i} = -\frac{\pi^2}{8i}$$
$$\text{Res}(f, -i) = \frac{(3i\pi/2)^2}{-2i} = \frac{9\pi^2}{8i}$$

和为 $\frac{\pi^2}{i} = -i\pi^2$。

围道积分 $= 2\pi i \cdot (-i\pi^2) = 2\pi^3$。

又 $\int_0^\infty \frac{dx}{1+x^2} = \frac{\pi}{2}$。

比较虚部：$-4\pi I = 0$，故 $I = 0$。
</details>

## 三、留数应用

### 习题 3.1 ⭐⭐
**知识点**: 留数定理基础

求下列函数在孤立奇点处的留数：
(a) $f(z) = \frac{1}{z(z-1)}$ 在 $z = 0, 1$
(b) $f(z) = \frac{e^z}{z^2}$ 在 $z = 0$
(c) $f(z) = \sin\frac{1}{z}$ 在 $z = 0$

<details>
<summary>详细解答</summary>

**(a)** 单极点：
$$\text{Res}(f, 0) = \lim_{z \to 0} z \cdot \frac{1}{z(z-1)} = -1$$
$$\text{Res}(f, 1) = \lim_{z \to 1} (z-1) \cdot \frac{1}{z(z-1)} = 1$$

**(b)** 二阶极点：
$$\text{Res}(f, 0) = \lim_{z \to 0} \frac{d}{dz}(z^2 \cdot \frac{e^z}{z^2}) = \lim_{z \to 0} e^z = 1$$

或展开：$\frac{e^z}{z^2} = \frac{1}{z^2} + \frac{1}{z} + \frac{1}{2} + \cdots$，$z^{-1}$ 系数为1。

**(c)** 本质奇点：
$$\sin\frac{1}{z} = \sum_{n=0}^\infty \frac{(-1)^n}{(2n+1)!} z^{-(2n+1)}$$

$z^{-1}$ 系数为1，故留数为1。
</details>

---

### 习题 3.2 ⭐⭐⭐
**知识点**: 辐角原理

设 $f$ 在简单闭曲线 $C$ 内及上全纯，在 $C$ 上无零点。证明：
$$\frac{1}{2\pi i} \oint_C \frac{f'(z)}{f(z)} dz = N - P$$
其中 $N$ 是 $f$ 在 $C$ 内的零点个数（计重数），$P$ 是极点个数。

<details>
<summary>详细解答</summary>

设 $f$ 在 $C$ 内有零点 $a_1, \ldots, a_k$，重数分别为 $m_1, \ldots, m_k$。

在 $a_j$ 附近：$f(z) = (z-a_j)^{m_j} g(z)$，$g(a_j) \neq 0$ 且全纯。

$$\frac{f'(z)}{f(z)} = \frac{m_j}{z-a_j} + \frac{g'(z)}{g(z)}$$

$\frac{g'}{g}$ 在 $a_j$ 全纯，故 $\text{Res}(\frac{f'}{f}, a_j) = m_j$。

由留数定理：$\frac{1}{2\pi i} \oint_C \frac{f'}{f} dz = \sum_{j=1}^k m_j = N$。

若有极点，类似分析得贡献为 $-P$。
</details>

### 习题 3.3 ⭐⭐⭐⭐
**知识点**: Rouché定理

求方程 $z^4 - 5z + 1 = 0$ 在圆盘 $|z| < 1$ 和 $1 < |z| < 2$ 内根的个数。

<details>
<summary>详细解答</summary>

**在 $|z| < 1$**：令 $f(z) = -5z$，$g(z) = z^4 + 1$。

在 $|z| = 1$ 上：$|f(z)| = 5$，$|g(z)| \leq |z|^4 + 1 = 2 < 5 = |f(z)|$。

由Rouché定理，$f + g = z^4 - 5z + 1$ 与 $f = -5z$ 在 $|z| < 1$ 内零点数相同，即1个。

**在 $|z| < 2$**：令 $f(z) = z^4$，$g(z) = -5z + 1$。

在 $|z| = 2$ 上：$|f(z)| = 16$，$|g(z)| \leq 10 + 1 = 11 < 16$。

$f + g$ 与 $f$ 在 $|z| < 2$ 内零点数相同，即4个（$z=0$ 是四重根）。

因此在 $1 < |z| < 2$ 内有 $4 - 1 = 3$ 个根。
</details>

### 习题 3.4 ⭐⭐⭐⭐
**知识点**: 留数与级数求和

利用留数定理求 $\sum_{n=1}^\infty \frac{1}{n^2}$。

<details>
<summary>关键思路点拨</summary>

考虑 $f(z) = \frac{\pi \cot(\pi z)}{z^2}$，利用其留数与级数的关系。
</details>

<details>
<summary>详细解答</summary>

函数 $\pi \cot(\pi z)$ 在所有整数 $n$ 处有单极点，留数为1。

考虑 $f(z) = \frac{\pi \cot(\pi z)}{z^2}$：
- 在 $z = n \neq 0$：留数为 $\frac{1}{n^2}$
- 在 $z = 0$：需计算

$\cot z = \frac{1}{z} - \frac{z}{3} - \frac{z^3}{45} - \cdots$

$\pi \cot(\pi z) = \frac{1}{z} - \frac{\pi^2 z}{3} - \cdots$

$f(z) = \frac{1}{z^3} - \frac{\pi^2}{3z} - \cdots$，在0处留数为 $-\frac{\pi^2}{3}$。

取大正方形围道 $C_N$ 包围 $\pm 1, \ldots, \pm N$，由留数定理：
$$\frac{1}{2\pi i} \oint_{C_N} f(z) dz = -\frac{\pi^2}{3} + 2\sum_{n=1}^N \frac{1}{n^2}$$

可证当 $N \to \infty$，左边积分 $\to 0$（因 $|\cot(\pi z)|$ 在围道上有界）。

因此 $2\sum_{n=1}^\infty \frac{1}{n^2} = \frac{\pi^2}{3}$，即 $\sum_{n=1}^\infty \frac{1}{n^2} = \frac{\pi^2}{6}$。
</details>

### 习题 3.5 ⭐⭐⭐⭐⭐
**知识点**: 无穷远点留数

设 $f$ 在扩充复平面上只有有限个孤立奇点。证明所有奇点（包括 $\infty$）的留数之和为零。

<details>
<summary>详细解答</summary>

**定义**：$\text{Res}(f, \infty) = -\frac{1}{2\pi i} \oint_{|z|=R} f(z) dz$（$R$ 充分大，顺时针）。

等价地，令 $w = 1/z$，则：
$$\text{Res}(f, \infty) = -\text{Res}\left(\frac{1}{w^2}f\left(\frac{1}{w}\right), 0\right)$$

设有限奇点为 $z_1, \ldots, z_n$。取大圆 $|z| = R$ 包围所有有限奇点。

由留数定理（逆时针）：
$$\frac{1}{2\pi i} \oint_{|z|=R} f(z) dz = \sum_{k=1}^n \text{Res}(f, z_k)$$

由无穷远点留数定义：
$$\text{Res}(f, \infty) = -\sum_{k=1}^n \text{Res}(f, z_k)$$

即 $\sum_{k=1}^n \text{Res}(f, z_k) + \text{Res}(f, \infty) = 0$。
</details>

## 四、共形映射

### 习题 4.1 ⭐⭐
**知识点**: 分式线性变换

求将单位圆盘 $\mathbb{D}$ 映射到自身的所有分式线性变换。

<details>
<summary>详细解答</summary>

设 $T(z) = \frac{az+b}{cz+d}$，$ad-bc \neq 0$。

条件：$T(\mathbb{D}) = \mathbb{D}$，则 $T$ 将单位圆周映射到自身。

由Schwarz引理的推广，这样的变换必有形式：
$$T(z) = e^{i\theta} \frac{z-\alpha}{1-\bar{\alpha}z}$$

其中 $|\alpha| < 1$，$\theta \in \mathbb{R}$。

**验证**：当 $|z| = 1$，$|z - \alpha| = |z||1 - \bar{\alpha}z| = |1 - \bar{\alpha}z|$（因 $|z|=1$ 时 $\bar{z} = 1/z$）。

故 $|T(z)| = 1$，即圆周映射到圆周。由连续性，内部映射到内部或外部，但 $T(\alpha) = 0 \in \mathbb{D}$，故内部映射到内部。
</details>

---

### 习题 4.2 ⭐⭐⭐
**知识点**: 共形映射构造

求将上半平面 $\mathbb{H} = \{z: \text{Im } z > 0\}$ 共形映射到单位圆盘 $\mathbb{D}$ 的具体映射。

<details>
<summary>详细解答</summary>

固定 $z_0 \in \mathbb{H}$（如 $z_0 = i$），要求 $f(z_0) = 0$。

分式线性变换将实轴映射到单位圆周。取：
$$f(z) = e^{i\theta} \frac{z - z_0}{z - \bar{z_0}}$$

验证：当 $z \in \mathbb{R}$，$|z - z_0| = |z - \bar{z_0}|$（因实轴是 $z_0$ 和 $\bar{z_0}$ 的中垂线），故 $|f(z)| = 1$。

取 $\theta = 0$，$z_0 = i$：
$$f(z) = \frac{z-i}{z+i}$$

验证 $f(i) = 0 \in \mathbb{D}$，故上半平面映射到单位圆盘。
</details>

### 习题 4.3 ⭐⭐⭐⭐
**知识点**: Riemann映射定理

**Riemann映射定理**：设 $D \subsetneq \mathbb{C}$ 是单连通区域，$z_0 \in D$。则存在唯一的共形映射 $f: D \to \mathbb{D}$ 满足 $f(z_0) = 0$，$f'(z_0) > 0$。

(a) 证明唯一性部分。
(b) 若 $D$ 是单位圆盘，$z_0 = 0$，求 $f$。

<details>
<summary>详细解答</summary>

**(a) 唯一性**：设 $f, g: D \to \mathbb{D}$ 都满足条件。令 $h = f \circ g^{-1}: \mathbb{D} \to \mathbb{D}$。

则 $h(0) = f(g^{-1}(0)) = f(z_0) = 0$。

由Schwarz引理，$|h(w)| \leq |w|$。

同理 $h^{-1}: \mathbb{D} \to \mathbb{D}$ 也满足 $h^{-1}(0) = 0$，故 $|h^{-1}(w)| \leq |w|$，即 $|w| \leq |h(w)|$。

因此 $|h(w)| = |w|$，由Schwarz引理等号条件，$h(w) = e^{i\theta}w$。

又 $h'(0) = f'(z_0) \cdot (g^{-1})'(0) = \frac{f'(z_0)}{g'(z_0)} > 0$，故 $e^{i\theta} = 1$，$h(w) = w$，即 $f = g$。

**(b)** 对 $\mathbb{D}$，$z_0 = 0$，唯一映射为 $f(z) = z$（因 $f'(0) > 0$ 且 $|f(z)| = |z|$ 要求 $f(z) = z$）。
</details>

### 习题 4.4 ⭐⭐⭐⭐
**知识点**: 共形映射与Dirichlet问题

利用共形映射求解上半平面的Dirichlet问题：求 $\mathbb{H}$ 内的调和函数 $u$ 使得 $u(x, 0) = \frac{1}{1+x^2}$。

<details>
<summary>详细解答</summary>

**方法**：将问题转化为单位圆盘，利用Poisson核求解，再映射回来。

映射 $w = \frac{z-i}{z+i}$ 将 $\mathbb{H} \to \mathbb{D}$，实轴 $\to$ 单位圆周。

实轴上的边界条件：设 $z = x \in \mathbb{R}$，则 $w = \frac{x-i}{x+i}$，$|w| = 1$。

需要求 $u(z) = U(w)$，其中 $U$ 是 $\mathbb{D}$ 内的调和函数。

由Poisson积分公式（对圆周 $|w| = 1$）：
$$U(w) = \frac{1}{2\pi} \int_0^{2\pi} \frac{1-|w|^2}{|w - e^{i\varphi}|^2} \phi(e^{i\varphi}) d\varphi$$

计算可得 $u(z) = \frac{1+y}{x^2 + (1+y)^2}$，其中 $z = x + iy$。

**验证**：直接验证 $\Delta u = 0$ 在 $\mathbb{H}$ 内，且当 $y \to 0^+$，$u(x, y) \to \frac{1}{1+x^2}$。
</details>

### 习题 4.5 ⭐⭐⭐⭐⭐
**知识点**: Schwarz-Christoffel变换

求将上半平面共形映射到矩形 $R = \{z: 0 < \text{Re } z < a, 0 < \text{Im } z < b\}$ 的Schwarz-Christoffel映射。

<details>
<summary>详细解答</summary>

Schwarz-Christoffel公式：
$$f(z) = C \int_0^z \prod_{k=1}^n (\zeta - x_k)^{\alpha_k - 1} d\zeta + D$$

其中 $\alpha_k \pi$ 是多边形在顶点 $w_k = f(x_k)$ 处的内角。

矩形有4个顶点，内角均为 $\frac{\pi}{2}$，故 $\alpha_k = \frac{1}{2}$。

选择 $x_1 = -1, x_2 = 1, x_3 = 1/k, x_4 = -1/k$（$0 < k < 1$）对应矩形的四个顶点。

$$f(z) = C \int_0^z \frac{d\zeta}{\sqrt{(1-\zeta^2)(1-k^2\zeta^2)}}$$

这是第一类椭圆积分，记作 $F(z, k)$。

矩形边长：
$$a = 2C K(k), \quad b = C K(k')$$

其中 $K(k)$ 是Complete elliptic integral of the first kind，$k' = \sqrt{1-k^2}$。

$$K(k) = \int_0^1 \frac{dt}{\sqrt{(1-t^2)(1-k^2t^2)}}$$
</details>
