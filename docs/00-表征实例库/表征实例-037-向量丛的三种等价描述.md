## 一、问题陈述

### 1.1 核心问题

> 向量丛有几种等价描述？局部平凡化、转移函数和主G-丛三种视角如何相互对应？

### 1.2 直观理解

向量丛是"参数化的向量空间"：
- **局部平凡化**：每点有邻域使丛看起来像乘积
- **转移函数**：描述局部平凡化之间的"粘合"
- **主G-丛**：通过对角线作用得到向量丛

## 三、符号表征（形式化）

### 3.1 定义1：局部平凡化定义

**定义**：底空间 X 上的 **秩r向量丛** 是映射 π: E → X，使得：

1. 每纤维 E_x := π⁻¹(x) 是 r 维 k-向量空间
2. 存在开覆盖 {U_α} 和局部平凡化：
   ```
   φ_α: π⁻¹(U_α) → U_α × k^r
   ```
   使得图表交换：
   ```
   π⁻¹(U_α) ──φ_α──→ U_α × k^r
       │                │
     π │                │ pr₁
       ▼                ▼
      U_α ═══════════ U_α
   ```
3. φ_α 限制到每纤维是线性同构

### 3.2 定义2：转移函数定义

**定义**：给定开覆盖 {U_α}，**转移函数**是一族映射：

```
g_{αβ}: U_{αβ} := U_α ∩ U_β → GL(r, k)
```

满足：

1. **上循环条件**：在 U_{αβγ} = U_α ∩ U_β ∩ U_γ 上
   ```
   g_{αβ} · g_{βγ} = g_{αγ}
   ```

2. **归一化**：g_{αα} = I（单位矩阵）

3. **逆性质**：g_{βα} = g_{αβ}⁻¹

**构造丛**：

```
E := (⊔_α U_α × k^r) / ~

其中 (x, v) ∈ U_α × k^r ~ (x, g_{αβ}(x)·v) ∈ U_β × k^r
```

### 3.3 定义3：主G-丛定义

**定义**：**主G-丛**是映射 π: P → X 配备右G-作用，使得：

1. G 在 P 上自由且正常作用
2. π 诱导同构 P/G ≅ X
3. 局部平凡：存在 {U_α} 和G-等变局部平凡化
   ```
   ψ_α: π⁻¹(U_α) → U_α × G
   ```

**从主丛构造向量丛**：

给定表示 ρ: G → GL(V)，定义**伴丛**：

```
E := P ×_ρ V := (P × V) / G

其中 g·(p, v) = (p·g⁻¹, ρ(g)·v)
```

### 3.4 三种描述的等价性

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    三种描述的等价转换                                       │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  局部平凡化 ─────────→ 转移函数                                             │
│  ═════════════════════════════════                                          │
│                                                                             │
│  给定 φ_α: E|_{U_α} ≅ U_α × k^r                                            │
│                                                                             │
│  在 U_{αβ} 上：φ_β ∘ φ_α⁻¹: U_{αβ} × k^r → U_{αβ} × k^r                   │
│                                                                             │
│  这是纤维上的线性映射，给出：                                               │
│  g_{αβ}: U_{αβ} → GL(r, k)                                                │
│                                                                             │
│  满足：g_{αβ} · g_{βγ} = g_{αγ}（由 (φ_γ ∘ φ_β⁻¹) ∘ (φ_β ∘ φ_α⁻¹)          │
│                                          = φ_γ ∘ φ_α⁻¹）                    │
│                                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  转移函数 ─────────→ 主GL(r)-丛                                             │
│  ═════════════════════════════════                                          │
│                                                                             │
│  给定转移函数 {g_{αβ}}，构造标架丛 Fr(E)：                                  │
│                                                                             │
│  Fr(E) = { (x, (e₁,...,e_r)) | x ∈ X, (e_i) 是 E_x 的基 }                  │
│                                                                             │
│  局部平凡化：ψ_α: Fr(E)|_{U_α} → U_α × GL(r, k)                           │
│                                                                             │
│  转移函数：ψ_β ∘ ψ_α⁻¹ 对应左乘 g_{αβ}(x)                                   │
│                                                                             │
│  这是一个主GL(r, k)-丛！                                                    │
│                                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  主G-丛 + 表示 ─────────→ 向量丛                                            │
│  ═══════════════════════════════════                                        │
│                                                                             │
│  给定主G-丛 P → X 和表示 ρ: G → GL(V)                                      │
│                                                                             │
│  构造 E = P ×_ρ V = (P × V)/G                                               │
│                                                                             │
│  局部平凡化：由 P 的局部平凡化诱导                                          │
│                                                                             │
│  E|_{U_α} ≅ (U_α × G × V)/G ≅ U_α × V                                      │
│                                                                             │
│  转移函数：ρ(g_{αβ})，其中 g_{αβ} 是 P 的转移函数                           │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 六、洞察与总结

### 6.1 核心洞察

| 洞察 | 内容 |
|------|------|
| **三种等价** | 局部平凡化、转移函数、主G-丛+表示是同一对象的不同视角 |
| **分类问题** | 转移函数的上同调类分类向量丛（见Cech上同调） |
| **结构约化** | 主丛视角允许约化结构群（如从GL(r)到O(r)得到度量） |
| **规范理论** | 主丛是物理中规范场（杨-Mills）的数学框架 |

### 6.3 关联概念

| 概念 | 关系 |
|------|------|
| **示性类** | 转移函数的上同调不变量（陈类、Stiefel-Whitney类）|
