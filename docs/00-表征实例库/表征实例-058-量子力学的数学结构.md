## 1. 引言：量子力学的数学基础

### 1.1 从物理到数学
量子力学的数学基础由von Neumann在1932年奠定，核心洞见：

```
量子系统 ⟺ Hilbert空间中的向量
可观测量 ⟺ 自伴算子
测量过程 ⟺ 谱分解
```

### 1.2 核心数学结构
量子力学的数学框架包含三个层次：
- **态空间**: Hilbert空间 $\mathcal{H}$
- **可观测量**: C*-代数/算子代数
- **对称性**: 群表示论

## 2. 多维表征

### 2.1 表征方式一：Hilbert空间框架

**基本设定**:
```
量子态: |ψ⟩ ∈ ℋ  (复Hilbert空间)
内积: ⟨φ|ψ⟩ ∈ ℂ
范数: ||ψ|| = √⟨ψ|ψ⟩
归一化: ||ψ|| = 1
```

**Dirac符号系统**:
```
|ψ⟩ : 列向量 (ket)
⟨ψ| : 行向量 (bra) = |ψ⟩†

⟨φ|ψ⟩ : 内积 (复数)
|ψ⟩⟨φ| : 外积 (算子)

正交归一基 {|n⟩}:
⟨m|n⟩ = δ_{mn}
完备性: Σ_n |n⟩⟨n| = I

展开: |ψ⟩ = Σ_n |n⟩⟨n|ψ⟩ = Σ_n c_n |n⟩
概率幅: c_n = ⟨n|ψ⟩, |c_n|² = 概率
```

**测量公设**:
```
可观测量: 自伴算子 A = A†

谱分解: A = Σ_n a_n |n⟩⟨n|

测量结果: 本征值 a_n
测量概率: |⟨n|ψ⟩|²
测量后状态: 投影到对应本征态 |n⟩

期望值: ⟨A⟩ = ⟨ψ|A|ψ⟩
不确定性: ΔA = √(⟨A²⟩ - ⟨A⟩²)
```

---

### 2.2 表征方式二：算子代数

**C*-代数框架**:
```
动机: 更一般的量子系统描述

C*-代数 𝒜:
• Banach代数
• 对合运算 *: 𝒜 → 𝒜
• C*条件: ||a*a|| = ||a||²

态 (State): 正线性泛函 ω: 𝒜 → ℂ
           满足 ω(I) = 1

Gelfand-Naimark-Segal (GNS) 构造:
每个C*-代数都可以表示为Hilbert空间上的算子代数
```

**von Neumann代数**:
```
定义: C*-代数 𝒜 ⊂ B(ℋ) 满足以下之一:
• 弱算子拓扑闭
• 二次交换子: 𝒜 = 𝒜''

类型分类:
• Type I: 离散谱（标准量子力学）
• Type II: 连续谱（量子统计）
• Type III: 因子（量子场论）

物理意义:
不同代数类型对应不同的物理情境
```

**量子统计力学**:
```
密度矩阵: ρ = Σ_i p_i |ψ_i⟩⟨ψ_i|
          p_i ≥ 0, Σp_i = 1

性质:
• ρ ≥ 0 (正定)
• Tr(ρ) = 1 (归一化)

期望值: ⟨A⟩ = Tr(ρA)

纯态: ρ² = ρ (秩1投影)
混合态: ρ² ≠ ρ

von Neumann熵: S(ρ) = -Tr(ρ log ρ)
```

### 2.3 表征方式三：表示论与对称性

**对称性与守恒量**:
```
Wigner定理:
量子系统的对称性是Hilbert空间上的
酉算子或反酉算子

Stone定理:
强连续单参酉群 U(t) = e^{-iHt/ℏ}
存在自伴生成元 H (Hamiltonian)

Noether定理 (量子版本):
对称性 ⟺ 守恒量
• 时间平移 ⟺ 能量守恒
• 空间平移 ⟺ 动量守恒
• 空间转动 ⟺ 角动量守恒
```

**角动量表示**:
```
角动量代数:
[J_i, J_j] = iℏ ε_{ijk} J_k

Casimir算子: J² = J_x² + J_y² + J_z²

不可约表示:
标记为 j = 0, 1/2, 1, 3/2, ...
维数: 2j + 1

例子:
• j = 0: 单态 (scalar)
• j = 1/2: 旋量 (qubit)
• j = 1: 矢量 (光子)
```

**Galilei群与Poincaré群**:
```
非相对论量子力学: Galilei群表示
相对论量子力学: Poincaré群表示

基本粒子分类:
由Poincaré群的Casimir算子标记
• 质量 m (p_μp^μ = m²c²)
• 自旋 s (小群表示)

→ 基本粒子的数学定义！
```

## 4. 深层联系

### 4.1 量子力学与经典力学的对应

```
对应原理 (Correspondence Principle):

经典力学          量子力学
─────────         ────────
相空间 (p,q)      Hilbert空间 |ψ⟩
Poisson括号 {,}   对易子 [ , ]/(iℏ)
可观测函数        自伴算子
Dirac δ函数       投影算子 |x⟩⟨x|
正则变换          酉变换

经典极限 ℏ → 0:
[x, p] = iℏ → 0  (可交换)
量子 → 经典
```

### 4.2 量子信息与量子计算

```
量子比特 (Qubit):
|ψ⟩ = α|0⟩ + β|1⟩, |α|² + |β|² = 1

布洛赫球 (Bloch Sphere):
|ψ⟩ = cos(θ/2)|0⟩ + e^{iφ}sin(θ/2)|1⟩

量子门:
• 单比特门: 布洛赫球上的旋转
• 双比特门: CNOT (纠缠生成)

量子算法:
利用叠加和纠缠实现并行计算
Shor算法、Grover算法等
```
