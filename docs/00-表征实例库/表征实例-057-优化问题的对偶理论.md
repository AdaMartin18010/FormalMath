## 1. 引言：对偶性的优化智慧

### 1.1 核心思想
每个优化问题（原问题，Primal）都可以关联一个对偶问题（Dual）：
- **弱对偶**: 对偶值 ≤ 原问题值
- **强对偶**: 在某些条件下，对偶值 = 原问题值

### 1.2 为什么对偶重要
```
对偶理论的价值:

• 计算: 对偶问题可能更容易求解
• 分析: 对偶提供最优性条件
• 灵敏度: 对偶变量 = 影子价格
• 结构: 揭示问题的深层结构
• 算法: 设计高效算法的基础
```

## 2. 多维表征

### 2.1 表征方式一：线性规划对偶

**标准形式**:
```
原问题 (P):              对偶问题 (D):
min cᵀx                  max bᵀy
s.t. Ax = b              s.t. Aᵀy ≤ c
     x ≥ 0
```

**对偶构造规则**:
```
原问题约束  ⟷  对偶变量
原问题变量  ⟷  对偶约束

min ⟷ max
等式约束 ⟷ 自由变量
不等式约束 ⟷ 非负变量
变量非负 ⟷ ≤ 约束
变量自由 ⟷ = 约束
```

**弱对偶与强对偶**:
```
弱对偶定理:
对于任何可行解 x, y:
bᵀy ≤ cᵀx

⟹ 对偶最优值 ≤ 原问题最优值

强对偶定理:
若 (P) 或 (D) 有有限最优解，
则两者都有，且最优值相等。

互补松弛条件:
x_i > 0 ⟹ (Aᵀy)_i = c_i
(Aᵀy)_i < c_i ⟹ x_i = 0
```

**经典例子:  max-flow min-cut**:
```
最大流问题 (原):
max 从s到t的流量
s.t. 容量约束、流量守恒

对偶 ⟺ 最小割问题

max flow = min cut  ⟹  强对偶的具体体现
```

### 2.2 表征方式二：凸优化对偶

**拉格朗日对偶**:
```
原问题:
min f₀(x)
s.t. f_i(x) ≤ 0, i=1,...,m
     h_j(x) = 0, j=1,...,p

拉格朗日函数:
L(x,λ,ν) = f₀(x) + Σ λ_i f_i(x) + Σ ν_j h_j(x)

对偶函数:
g(λ,ν) = inf_x L(x,λ,ν)

对偶问题:
max g(λ,ν)
s.t. λ ≥ 0
```

**对偶性质**:
```
弱对偶 (通用):
g(λ,ν) ≤ f₀(x) 对于所有可行对
⟹ d* ≤ p* (对偶最优 ≤ 原问题最优)

对偶间隙: p* - d* ≥ 0

强对偶 (需要条件):
对于凸问题 + Slater条件
p* = d*

Slater条件: 存在严格可行点
            f_i(x) < 0 (不等式严格)
            h_j(x) = 0
```

**KKT条件**:
```
对于可微凸问题，以下等价:
1. x* 是最优点
2. 存在 (λ*,ν*) 满足KKT:

   • 原始可行性: f_i(x*) ≤ 0, h_j(x*) = 0
   • 对偶可行性: λ* ≥ 0
   • 互补松弛: λ*_i f_i(x*) = 0
   • 平稳性: ∇f₀(x*) + Σλ*_i∇f_i(x*) + Σν*_j∇h_j(x*) = 0

当强对偶成立时，KKT条件是必要且充分的。
```

### 2.3 表征方式三：拉格朗日对偶的变分形式

**鞍点解释**:
```
原问题可以写为:
p* = inf_x sup_{λ≥0} L(x,λ)

对偶问题:
d* = sup_{λ≥0} inf_x L(x,λ)

弱对偶: 
sup_λ inf_x L ≤ inf_x sup_λ L
        (max min ≤ min max)

强对偶成立 ⟺ 可以交换 inf 和 sup
          ⟺ 存在鞍点
```

**鞍点定理**:
```
(x*,λ*) 是鞍点 ⟺
• x* 最小化 L(x,λ*) over x
• λ* 最大化 L(x*,λ) over λ ≥ 0
• 等价于KKT条件
```

**Fenchel对偶 (共轭函数视角)**:
```
共轭函数: f*(y) = sup_x (yᵀx - f(x))

对于问题: min f(x) + g(Ax)
对偶问题: max -f*(Aᵀy) - g*(-y)

例子:
f(x) = ½||x||²  ⟹  f*(y) = ½||y||²
g(x) = I_{b}(x) (示性函数) ⟹ g*是支撑函数
```

## 4. 深层联系

### 4.1 对偶性的几何解释

```
几何视角:

原问题: 寻找在约束集上最小化目标函数的点
对偶问题: 寻找支撑超平面

强对偶: 存在支撑超平面恰好接触目标水平集

对于凸集: 支撑超平面总是存在 (Hahn-Banach)
⟹ 强对偶成立

对于非凸: 支撑超平面可能不存在
⟹ 可能有对偶间隙
```

### 4.2 经济学解释

```
影子价格 (Shadow Prices):

对偶变量 λ*_i = 约束i的边际价值
= 约束右端项增加1单位时，
  目标函数的改善量

经济学意义:
• λ* ≥ 0: 约束越紧，价值越高
• λ* = 0: 约束不紧(松弛)，无价值

应用: 资源分配、生产计划、投资组合
```

## 5. 应用与意义

### 5.1 现代优化算法

| 算法 | 对偶性应用 |
|------|-----------|
| **对偶上升** | 直接最大化对偶函数 |
| **增广拉格朗日** | 加惩罚项保证收敛 |
| **ADMM** | 交替方向乘子法，分布式优化 |
| **内点法** | 通过对偶路径跟踪求解 |
| **SVM** | 对偶问题更易求解，核技巧 |

### 5.2 机器学习中的应用

```
支持向量机 (SVM) 的对偶:

原问题:
min ½||w||² + CΣξ_i
s.t. y_i(w·x_i + b) ≥ 1 - ξ_i

对偶问题:
max Σα_i - ½ΣΣα_iα_jy_iy_j(x_i·x_j)
s.t. 0 ≤ α_i ≤ C, Σα_iy_i = 0

优势:
• 只涉及内积 (x_i·x_j)
• 可核化: 替换为 K(x_i, x_j)
• 多数 α_i = 0 (稀疏性)
```
