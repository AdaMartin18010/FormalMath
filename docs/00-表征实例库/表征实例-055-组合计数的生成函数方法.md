### 1.1 核心思想
生成函数将离散序列编码为解析函数，将组合问题转化为分析问题：

$$A(x) = \sum_{n=0}^{\infty} a_n x^n$$

**哲学**: "代数操作 ⟺ 组合构造"

```

```

## 2. 多维表征

### 2.1 表征方式一：普通生成函数 (OGF)

**定义**: 对于序列 $\{a_n\}$，其OGF为
$$A(x) = \sum_{n \geq 0} a_n x^n$$

**基本操作**:

```
组合构造 ⟷ 生成函数运算

1. 不交并 (Disjoint Union):
   A = B ⊔ C  ⟹  A(x) = B(x) + C(x)

2. 笛卡尔积 (Cartesian Product):
   A = B × C  ⟹  A(x) = B(x) · C(x)

3. 序列构造 (Sequence):
   A = Seq(B)  ⟹  A(x) = 1/(1 - B(x))

4. 集合构造 (Set):
   A = Set(B)  ⟹  A(x) = exp(B(x))
   
5. 循环构造 (Cycle):
   A = Cyc(B)  ⟹  A(x) = log(1/(1 - B(x)))
```

**经典例子**:

```
1. 组合数 C(n,k):
   Σ_k C(n,k) x^k = (1+x)^n
   
2. 分拆数 p(n):
   Σ_n p(n) x^n = ∏_{k≥1} 1/(1-x^k)
   
3. Catalan数:
   C(x) = Σ_n C_n x^n = (1-√(1-4x))/(2x)
   C_n = (1/(n+1))C(2n,n)
```

---

### 2.2 表征方式二：指数生成函数 (EGF)

**定义**: 对于序列 $\{a_n\}$，其EGF为
$$\hat{A}(x) = \sum_{n \geq 0} a_n \frac{x^n}{n!}$$

**为何需要EGF**:
```
EGF适合标记结构(labeled structures)

原因: 标记对象的排列有n!种方式
      除以n!消除标记的依赖性
```

**基本操作**:

```
标记结构的组合构造:

1. 不交并:
   A = B ⊔ C  ⟹  Â(x) = B̂(x) + Ĉ(x)

2. 标记积 (Labeled Product):
   A = B ⋆ C  ⟹  Â(x) = B̂(x) · Ĉ(x)
   
   解释: 分割标记集，分别给B和C

3. 标记序列:
   A = Seq(B)  ⟹  Â(x) = 1/(1 - B̂(x))

4. 集合 (标记):
   A = Set(B)  ⟹  Â(x) = exp(B̂(x))
   
   例子: 集合划分 S(n,k)
   Σ_k S(n,k) x^k/k! → Bell数
```

**经典例子**:

```
1. 排列数 n!:
   Σ_n n! x^n/n! = 1/(1-x)
   
2. 错排数 D_n:
   D̂(x) = exp(-x)/(1-x)
   D_n = n! Σ_{k=0}^n (-1)^k/k!
   
3. Bell数 B_n (集合划分数):
   B̂(x) = exp(exp(x) - 1)
```

### 2.3 表征方式三：Dirichlet生成函数 (DGF)

**定义**: 对于序列 $\{a_n\}$，其DGF为
$$\tilde{A}(s) = \sum_{n \geq 1} \frac{a_n}{n^s}$$

**应用场景**:
```
DGF适合乘法结构:
• 数论函数
• 积性函数
• 算术级数
```

**基本性质**:

```
1. 乘法卷积:
   c_n = Σ_{d|n} a_d b_{n/d}
   ⟹  C̃(s) = Ã(s) · B̃(s)

2. 经典例子:
   ζ(s) = Σ_n 1/n^s  (Riemann zeta)
   
   若 a_n 是积性函数:
   Ã(s) = ∏_p (1 + a_p/p^s + a_{p²}/p^{2s} + ...)
```

**数论应用**:

```
• 除数函数: Σ_n d(n)/n^s = ζ(s)²
• Euler函数: Σ_n φ(n)/n^s = ζ(s-1)/ζ(s)
• von Mangoldt: Σ_n Λ(n)/n^s = -ζ'(s)/ζ(s)
• Möbius: Σ_n μ(n)/n^s = 1/ζ(s)
```

## 4. 深层联系

### 4.1 生成函数与组合类

```
组合类的结构:

一个组合类是一组对象，每个对象有大小(size)函数

OGF: 按大小枚举
EGF: 按大小枚举，考虑标记

符号方法 (Symbolic Method):
允许直接从组合描述写出生成函数
```

### 4.2 生成函数与复分析

```
复分析方法:

1. 奇点分析:
   系数 a_n 的渐近行为由 A(z) 的
   最近奇点决定

2. Cauchy积分公式:
   a_n = (1/2πi) ∮ A(z)/z^{n+1} dz

3. 鞍点法:
   选择通过鞍点的积分路径
   局部展开得到渐近估计
```
