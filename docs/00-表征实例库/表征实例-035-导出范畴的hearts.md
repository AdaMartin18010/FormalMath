## 一、问题陈述

### 1.1 核心问题

> 导出范畴 D(A) 本身不是Abel范畴，如何在其中恢复Abel结构？t-结构和heart提供了什么框架？

### 1.2 背景：导出范畴的问题

**Abel范畴的局限**：
- 复形范畴 C(A) 是Abel范畴，但拟同构不总是同构
- 需要"反转"拟同构得到导出范畴 D(A)

**导出范畴 D(A) 的性质**：
- 是**三角范畴**，不是Abel范畴
- 短正合序列的概念不再适用
- 需要新的方法恢复"Abel结构"

## 三、符号表征（形式化）

### 3.1 t-结构的严格定义

**定义**：设 D 是三角范畴，**t-结构**是一对全子范畴 (D^{≤0}, D^{≥0}) 满足：

1. **Hom消失**：对 A ∈ D^{≤0}, B ∈ D^{≥1}，有 Hom_D(A, B) = 0
   （其中 D^{≥1} := D^{≥0}[-1]）

2. **平移封闭**：D^{≤0}[1] ⊂ D^{≤0} 且 D^{≥0} ⊂ D^{≥0}[1]
   
   （等价：D^{≤-1} ⊂ D^{≤0} 且 D^{≥1} ⊂ D^{≥0}）

3. **分解存在**：对任意 X ∈ D，存在正合三角形
   
   ```
   A → X → B → A[1]
   ```
   
   其中 A ∈ D^{≤0}, B ∈ D^{≥1}

**定义**：t-结构的 **heart** 定义为：

```
C := D^{≤0} ∩ D^{≥0}
```

### 3.2 截断函子

**定理**：t-结构诱导一对伴随函子（截断函子）：

```
τ_{≤0}: D → D^{≤0}    （左截断）
τ_{≥0}: D → D^{≥0}    （右截断）

满足：
• τ_{≤0} 是包含 D^{≤0} ↪ D 的右伴随
• τ_{≥0} 是包含 D^{≥0} ↪ D 的左伴随
• 对任意 X ∈ D，有正合三角形：

  τ_{≤0} X → X → τ_{≥1} X → (τ_{≤0} X)[1]
```

### 3.3 上同调函子

**定义**：t-结构定义**上同调函子**：

```
H^0 := τ_{≤0} ∘ τ_{≥0} ≅ τ_{≥0} ∘ τ_{≤0}: D → C

H^n(X) := H^0(X[n])
```

**性质**：
- H^0 是上同调的（将三角变为长正合序列）
- H^n(X) = 0 对所有 n ⟺ X ≅ 0
- 若 X ∈ D^{≤0}，则 H^n(X) = 0 对 n > 0
- 若 X ∈ D^{≥0}，则 H^n(X) = 0 对 n < 0

### 3.4 Lean 4 形式化

```lean
import Mathlib.CategoryTheory.Triangulated.TStructure
import Mathlib.CategoryTheory.Triangulated.Triangle

variable {C : Type*} [Category C] [HasShift C ℤ] [Pretriangulated C]

-- t-结构
def TStructure (C : Type*) [Category C] [HasShift C ℤ] [Pretriangulated C] :=
  { truncLE : Subcategory C // 
    { truncGE : Subcategory C //
      -- Hom消失条件
      (∀ (X ∈ truncLE) (Y ∈ truncGE.shift (-1)), IsZero (X ⟶ Y)) ∧
      -- 平移封闭条件
      (truncLE.shift 1 ≤ truncLE) ∧
      (truncGE ≤ truncGE.shift 1) ∧
      -- 分解存在条件
      (∀ X, ∃ (T : Triangle C) (hT : T ∈ distTriang C),
        T.obj₁ ∈ truncLE ∧ T.obj₃ ∈ truncGE.shift (-1) ∧ T.mor₁ = 𝟙 X) } }

-- heart 的定义
def heart (t : TStructure C) : Subcategory C :=
  t.1 ⊓ t.2.1

-- heart 是 Abel 范畴
theorem heart_is_abelian (t : TStructure C) :

-- 上同调函子
def cohomologyFunctor (t : TStructure C) (n : ℤ) : C ⥤ (heart t).category :=
  (truncLEFunctor t ⋙ truncGEFunctor t).shift n
```

## 七、洞察与总结

|------|------|
| **三角vs Abel** | t-结构在三角范畴中构造Abel子范畴（heart） |
