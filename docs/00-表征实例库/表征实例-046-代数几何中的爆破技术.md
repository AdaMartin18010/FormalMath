## 1. 概念本质

### 1.1 核心思想
爆破（Blow-up）是将代数簇沿子簇"打开"的手术操作，用子簇的法丛的射影化代替原来的奇点或子簇，从而改善几何性质。

### 1.2 直观理解
```
想象将子簇 Z ⊂ X "膨胀"成一个Exceptional除子 E:

爆破前:     爆破后:
    X           Bl_Z(X)
    │            │  \
    │            │   \
    Z            E    X\Z
    │           /      │
    │          /       │
   pt         ℙ(N)    同构
   
Exceptional除子 E ≅ ℙ(N_{Z/X}) 记录了Z在X中的法向信息
```

## 2. 多维表征

### 2.1 表征方式一：奇点消解

**Hironaka定理**:
```
对于特征0的域上的任意代数簇 X，
存在光滑簇 X̃ 和真双有理映射:
    π: X̃ → X

使得 π 是爆破光滑中心的复合。
```

**消解步骤**:
```
步骤1: 找出最大Hilbert-Samuel strata
步骤2: 对该stratum进行爆破
步骤3: 验证不变量严格下降
步骤4: 重复直至光滑
```

**局部描述**:
```
设 X ⊂ 𝔸^n 由方程 f(x₁,...,x_n) = 0 定义
在 0 处有孤立奇点

爆破 𝔸^n 于原点:
    Bl_0(𝔸^n) = {((x₁,...,x_n), [y₁:...:y_n]) | x_i y_j = x_j y_i}

严格变换:
    X̃ = closure of π^{-1}(X \ {0})
    
例外除子 E = π^{-1}(0) ≅ ℙ^{n-1}
```

---

### 2.2 表征方式二：严格变换

**定义**:
```
设 π: X̃ → X 为 Z ⊂ X 的爆破

对于子簇 Y ⊂ X:
    全逆像: π^*(Y) = π^{-1}(Y)
    严格变换: Ỹ = closure(π^{-1}(Y \ Z))
    
例外除子的贡献: π^*(Y) = Ỹ + m·E （m = Y沿Z的重数）
```

**交截性质**:
```
严格变换保持横截相交:
若 Y₁, Y₂ ⊂ X 横截相交，且 Z 与它们都横截
则 Ỹ₁, Ỹ₂ ⊂ X̃ 仍然横截相交

相交数的保持:
Y₁ · Y₂ = Ỹ₁ · Ỹ₂ + （例外贡献）
```

**计算规则**:
```
设 Z = center of blow-up, E = exceptional divisor

对于除子 D:
    π^*D = D̃ + mult_Z(D)·E
    
相交公式:
    E^n = (-1)^{n-1}·deg(Z) （Z为点时）
    D̃ · E^{n-1} = mult_Z(D)
```

---

### 2.3 表征方式三：例外除子

**几何意义**:
```
例外除子 E = π^{-1}(Z) 是爆破的"痕迹":

结构: E ≅ ℙ(N_{Z/X}) = Proj(Sym(N_{Z/X}^∨))

法丛: N_{E/X̃} = 𝒪_E(-1) （tautological丛的负）
```

**典范丛公式**:
```
对于光滑簇 X 和光滑中心 Z，codim Z = r:

K_{X̃} = π^*K_X + (r-1)·E

证明: 局部计算 + 分歧公式
```

**除子类群**:
```
Cl(X̃) ≅ Cl(X) ⊕ ℤ·E

例外除子生成一个新的自由因子
```

### 3.2 爆破的局部结构

```
中心 Z = V(x,y) ⊂ 𝔸³ = Spec k[x,y,z]

爆破后的坐标卡:
                    Bl_Z(𝔸³)
                       │
          ┌────────────┼────────────┐
          ↓            ↓            ↓
       U_x            U_y         U_z
         │            │            │
         ▼            ▼            ▼
    k[x, y/x, z]  k[x/y, y, z]  k[x/z, y/z, z]
    x·(y/x)=y      (x/y)·y=x    z≠0 时同构
    
例外除子 E:
    在 U_x: x = 0, y/x 自由
    在 U_y: y = 0, x/y 自由
    整体: E ≅ ℙ¹ × 𝔸¹ ≅ ℙ(N_{Z/𝔸³})
```

## 4. 经典应用

### 4.1 曲线的奇点消解

```
平面曲线 C ⊂ 𝔸² 在原点有节点:
    xy = 0

爆破原点:
    在卡 U_x: x ≠ 0, 令 y = x·v
    方程变为: x²v = 0
    严格变换: v = 0 （即 y/x = 0）
    
    在卡 U_y: y ≠ 0, 令 x = y·u  
    方程变为: y²u = 0
    严格变换: u = 0 （即 x/y = 0）

结果:
    C̃ ∩ E 有两点，对应节点的两个分支
    C̃ 在例外除子处与E横截相交
```

### 4.2 曲面奇点的极小消解

```
Du Val奇点（有理二重点）:
    A_n: x² + y² + z^{n+1} = 0
    D_n: x² + y^{n-1} + yz² = 0
    E_6, E_7, E_8: 其他型

极小消解性质:
    例外除子 = 配置为相应Dynkin图的ℙ¹链
    K_{X̃} = π^*K_X （典范奇点）
```

### 4.3 Castelnuovo收缩定理

```
定理: 曲面 S 上的曲线 E ≅ ℙ¹，满足 E² = -1
则可收缩E为光滑点:
    ∃ S' 和双有理映射 π: S → S'
    s.t. π(E) = pt，π|_{S\E} 为同构

逆过程: 这就是在某点爆破
```

## 5. 现代发展

### 5.1 强分解vs弱分解

```
弱Hironaka: 存在光滑簇 X̃ → X （双有理）

强Hironaka: 给定 X̃ → X，可分解为爆破的光滑中心的复合

Abhyankar问题: 正特征情形仍有开放问题
```

### 5.2 叠上的爆破

```
Deligne-Mumford叠的爆破:
    可沿光滑闭子叠进行爆破
    保持DM性质
    
应用: 模空间的紧化（如M̄_g）
```

### 5.3 导出爆破

```
在导出代数几何中:
    可沿导出中心爆破
    严格变换有更精细的定义
    
应用: 导出形变理论
```
