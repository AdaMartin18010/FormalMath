## 1. 概念本质

### 1.1 核心思想
局部-整体原则研究"局部"信息（在所有完备化上的行为）是否能决定"整体"信息（在有理数域或整体域上的行为），是连接数论与几何的桥梁。

### 1.2 核心问题
```
给定整体域 K（如ℚ）上的对象 X（如代数簇、方程）:

局部可解 ⟹ 整体可解 ?

即: 若 X 在所有完备化 K_v 上有解，X 是否在 K 上有解？
```

## 2. 多维表征

### 2.1 表征方式一：Hasse原理

**经典形式**:
```
对于二次型（Hilbert, Hasse, Minkowski）:

定理: 二次型在 ℚ 上表示0（有非平凡零点）
      ⟺ 它在 ℝ 和所有 ℚ_p 上表示0

等价: 局部可解 ⟺ 整体可解（对二次型成立）
```

**强逼近**:
```
设 G 为 ℚ 上的单连通半单代数群:

G(ℚ) 在 G(𝔸) 中稠密

其中 𝔸 = ℝ × ∏'_p ℚ_p 是adele环

这推广了Hasse原理到齐性空间
```

**Hasse-Minkowski定理**:
```
对于二次型 Q(x₁,...,x_n) = 0:
- n = 2: 需要额外条件（Legendre符号条件）
- n ≥ 3: Hasse原理成立

例子: x² + y² + z² = 0 在 ℚ 上无非平凡解
      ⟺ 在 ℝ 上无非平凡解（显然）
      或等价于在 ℚ_2 上无解
```

---

### 2.2 表征方式二：Brauer-Manin阻碍

**动机**:
```
Hasse原理对三次曲面不成立（Selmer反例）！

3x³ + 4y³ + 5z³ = 0
在 ℝ, ℚ_p 上都有解，但在 ℚ 上无解

需要更精细的阻碍理论
```

**Brauer群**:
```
定义: Br(X) = H²_{ét}(X, 𝔾_m)

对于光滑射影簇 X/ℚ:
- 有自然配对: X(𝔸) × Br(X) → ℚ/ℤ
- 记 X(𝔸)^{Br} = {x ∈ X(𝔸) | ⟨x, b⟩ = 0 ∀b ∈ Br(X)}

包含关系:
    X(ℚ) ⊂ X(𝔸)^{Br} ⊂ X(𝔸)
```

**Brauer-Manin原理**:
```
猜想 (Colliot-Thélène & Sansuc):
对于有理簇（甚至更广的类），
X(ℚ) 在 X(𝔸)^{Br} 中稠密

已知: 对Châtelet曲面、某些del Pezzo曲面成立
```

**计算Brauer群**:
```
对于有理簇:
Br(X)/Br(ℚ) ≅ H¹(ℚ, Pic(X̄))

计算步骤:
1. 计算几何Picard群 Pic(X̄)
2. 计算Galois上同调
3. 确定Brauer群生成元
```

---

### 2.3 表征方式三：高阶阻碍

**Descent障碍**:
```
利用torsors（主齐性空间）:

对于代数群 G，G-torsor f: Y → X:
    X(ℚ) = ⊔_{[Y]∈H¹(ℚ,G)} f(Y(ℚ))

Descent集合:
    X(𝔸)^f = ∪_Y f(Y(𝔸)) ⊂ X(𝔸)
```

**Étale Brauer-Manin**:
```
结合étale上同调:

X(𝔸)^{ét,Br} = ∩_{有限平展 f: Y→X} X(𝔸)^f

Skorobogatov证明存在
X(𝔸)^{ét,Br} ≠ ∅ 但 X(ℚ) = ∅ 的例子
```

**高阶阻碍的层次**:
```
X(ℚ) ⊂ ... ⊂ X(𝔸)^{ét,Br} ⊂ X(𝔸)^{Br} ⊂ X(𝔸)^{Br[2]} ⊂ X(𝔸)

越来越精细的逼近，但永远可能不完备
```

## 4. 经典实例分析

### 4.1 Selmer三次曲面

```
方程: 3x³ + 4y³ + 5z³ = 0 在 ℙ²

分析:
1. 实点: 显然存在（如 x=1, y=1, z=-(7/5)^{1/3}）
2. p进点: 对所有 p，局部可解（Hensel引理）
3. 有理点: 不存在！

原因: Cassels证明这是Brauer-Manin阻碍
```

### 4.2 Châtelet曲面

```
形式: y² - az² = P(x)  （P为三次多项式）

定理 (Colliot-Thélène, Sansuc, Swinnerton-Dyer):
Châtelet曲面满足Hasse原理
（实际上是Brauer-Manin原理）

证明关键: Br(X)/Br(ℚ) ≅ ℤ/2ℤ
```

### 4.3 椭圆曲线与BSD猜想

```
Hasse-Weil L-函数:
    L(E/ℚ, s) = ∏_p L_p(p^{-s})^{-1}

BSD猜想:
    rank E(ℚ) = ord_{s=1} L(E/ℚ, s)

局部信息（L-函数）决定整体秩！
```
