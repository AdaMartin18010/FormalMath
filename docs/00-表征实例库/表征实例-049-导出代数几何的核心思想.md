## 1. 概念本质

### 1.1 核心动机
传统代数几何将空间看作点的集合，导出代数几何（Derived Algebraic Geometry, DAG）则将空间看作"函数的代数"的谱，并进一步考虑到高阶同伦信息，使得相交理论、模问题等获得更自然的处理。

### 1.2 核心洞察
```
经典: X ↦ Γ(X, 𝒪_X) 形成层
导出: X ↦ RΓ(X, 𝒪_X) 形成复形

关键观察:
1. 复形携带了比上同调更多的信息
2. 非横截相交有自然的导出解释
3. 商叠和模空间的自然构造
```

## 2. 多维表征

### 2.1 表征方式一：导出概形

**动机问题**:
```
纤维积问题:
    设 Y → Z 是非正则浸入，X → Z 任意映射
    经典纤维积 X ×_Z Y 可能有"错误"维数
    
例子: ℙ² 中两条切线的交:
    应该是"二重点"而非单点
```

**导出纤维积**:
```
在DAG中:
    X ×^h_Z Y 替代 X ×_Z Y

局部模型:
    Spec B ⊗^L_A C 替代 Spec(B ⊗_A C)
    
其中 ⊗^L 是导出张量积
```

**结构层**:
```
导出概形 (X, 𝒪_X^•):
- 底空间: 经典概形 X_{cl}
- 结构层: 交换微分分次代数（cdga）
    𝒪_X^• = 𝒪_{X_{cl}} ⊕ 𝒪^{(1)} ⊕ 𝒪^{(2)} ⊕ ...
    
π_i(𝒪_X^•) 给出高阶结构信息
```

---

### 2.2 表征方式二：商叠

**经典困难**:
```
GIT商的问题:
- 许多自然作用没有好商
- 需要稳定性条件的选择
- 信息丢失
```

**导出商叠**:
```
对于群 G 作用在 X 上:
    [X/G]^{der} 是导出Artin叠
    
与经典商的关系:
    π_0([X/G]^{der}) = [X/G]_{cl}
    
例子:
    [pt/G]^{der} = BG （分类叠）
    有自然的导出结构
```

**模叠的优势**:
```
在DAG中:
- 自然函子可表性
- 完美障碍理论自动出现
- 虚拟基本类自然构造

应用: Gromov-Witten理论、Donaldson-Thomas理论
```

---

### 2.3 表征方式三：形变理论

**经典形变理论**:
```
对于概形 X:
    T^i = Ext^i(Ω_X, 𝒪_X)  （Lichtenbaum-Schlessinger）
    
障碍: Ob = T^2
切空间: T^1
    
复杂性: 需要复杂的计算
```

**导出形变理论**:
```
导出切复形:
    𝕋_X = RHom(𝕃_X, 𝒪_X)
    
其中 𝕃_X 是 cotangent complex

形变函子自动由导出增强表示
```

**优势**:
```
1. 统一的框架（不需要分情况）
2. 自然的高阶结构
3. 与叠的形变兼容
```

## 4. 典型应用

### 4.1 完美障碍理论

```
问题: 模空间 M 可能奇异，需要定义"虚拟"基本类

DAG解决方案:
    M^{der} 有自然的切复形 𝕋_M
    若 𝕋_M 是 perfect complex（局部等价于有限秩自由复形）
    
则: [M]^{vir} ∈ A_{vdim}(M) 自然定义
    vdim = rank(𝕋_M)

应用: GW, DT, PT理论
```

### 4.2 导出相交理论

```
非横截相交的导出解释:

设 X, Y ⊂ Z 子概形
经典: X ∩ Y 可能有嵌入维数下降
导出: X ×^h_Z Y 保持正确的"交截维数"

K-理论解释:
    [𝒪_X ⊗^L 𝒪_Y] = Σ_i (-1)^i [Tor^Z_i(𝒪_X, 𝒪_Y)]
    这是正确的相交积
```

### 4.3 导出Hilbert概形

```
经典问题: Hilbert概形可能非光滑、维数不正确

导出Hilbert概形 Hilb^{der}:
- 总是"光滑"的（导出意义下）
- 正确的切空间由导出Ext给出
- 经典Hilb是其截断
```
