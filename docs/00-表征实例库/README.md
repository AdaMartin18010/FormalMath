# 00-表征实例库

本目录收录了 30 余份精选的“表征实例”文档，每一份文件都针对一个高度抽象或容易混淆的数学概念，通过多种等价定义、不同视角的对比、可视化描述与具体计算，帮助学习者建立深刻的概念表征。主题涵盖代数闭包、紧致性、同调代数的长正合序列、层上同调、Lie 群定义、导出范畴 hearts、曲面分类定理、向量丛等价描述等，横跨代数、几何、拓扑与分析多个领域，是概念深化与认知诊断的珍贵素材库。
