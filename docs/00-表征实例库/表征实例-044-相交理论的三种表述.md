## 1. 概念本质

### 1.1 核心问题
相交理论研究代数簇的子簇如何相交，核心问题是：给定两个子簇 X, Y ⊂ Z，如何定义和计算它们的"相交积" X · Y？

### 1.2 核心困难
- **维数问题**: 当 codim X + codim Y > dim Z 时，通常不相交
- **奇点问题**: 非横截相交时如何计数
- **紧性问题**: 相交数在形变下的不变性

---

## 2. 多维表征

### 2.1 表征方式一：代数循环

**基本对象**: 代数循环 = 子簇的形式线性组合

```
设 X 为光滑射影簇

Z_k(X) = {k维代数循环} = ⊕_{dim Y = k} ℤ·[Y]
```

**相交积的朴素定义**:
```
对于横截相交的子簇:
    [Y] · [Z] = [Y ∩ Z]  （作为循环）

条件: dim(Y ∩ Z) = dim Y + dim Z - dim X
```

**有理等价**:
```
两个循环 α ~_{rat} β ⇔ 
    存在 W ⊂ X × ℙ¹ 使得
    α = W ∩ (X × {0})
    β = W ∩ (X × {∞})

Chow群: A_k(X) = Z_k(X) / ~_{rat}
```

**相交积的构造**:
```
第一步: 形变到横截位置（移动引理）
    [Y] ↦ [Y'] 其中 Y' 横截于 Z

第二步: 定义 [Y]·[Z] = [Y' ∩ Z]

第三步: 证明与移动方式无关
```

---

### 2.2 表征方式二：Chow环

**环结构**:
```
A^*(X) = ⊕_{k=0}^{dim X} A_{dim X - k}(X)

次数: A^k(X) × A^l(X) → A^{k+l}(X)
      (α, β) ↦ α ∪ β
```

**基本性质**:
```
1. 结合律: (α ∪ β) ∪ γ = α ∪ (β ∪ γ)
2. 交换律: α ∪ β = (-1)^{kl} β ∪ α （分次交换）
3. 恒等元: [X] ∈ A^0(X)
```

**函子性**:
```
真映射 f: X → Y 诱导:
    f_*: A_k(X) → A_k(Y) （推出）

平坦映射或 l.c.i. 映射诱导:
    f^*: A^k(Y) → A^k(X) （拉回）

投影公式: f_*(α ∪ f^*β) = f_*α ∪ β
```

**陈类**:
```
对于秩 r 向量丛 E → X:
    c(E) = 1 + c_1(E) + ... + c_r(E) ∈ A^*(X)

分裂原理: 形式上写 E = L_1 ⊕ ... ⊕ L_r
    c(E) = ∏(1 + x_i) 其中 x_i = c_1(L_i)
```

---

### 2.3 表征方式三：上同调类

**周期映射**:
```
cl: A^k(X) → H^{2k}(X, ℤ)
    [Z] ↦ 积分闭链类

对于有向紧致流形: cl(α ∪ β) = cl(α) ∪ cl(β)
```

**Hodge理论视角**:
```
cl(A^k(X) ⊗ ℚ) ⊂ H^{k,k}(X) ∩ H^{2k}(X, ℚ)

Hodge猜想: 上述包含是等号（未解决！）
```

**不同上同调理论**:
```
1. Betti上同调: H^*(X, ℤ) — 拓扑方法
2. de Rham上同调: H^*_{dR}(X/ℂ) — 微分形式
3. ℓ进上同调: H^*_{ét}(X, ℚ_ℓ) — 算术几何
4. 晶体上同调: H^*_{cris}(X/W) — 正特征
```

**标准猜想**:
```
Grothendieck标准猜想预测:
1. Lefschetz型算子给出 SL_2 作用
2. 代数循环在同调中由数值等价刻画

蕴含: 不同上同调理论本质相同（纯粹动机理论）
```

## 4. 深层联系

### 4.1 从几何到数值

```
几何相交 → 代数循环 → Chow环 → 上同调 → 数值不变量

具体路径:
[Y] ∩ [Z] ↦ [Y]·[Z] ∈ A_*(X) ↦ cl([Y]·[Z]) ∈ H^*(X)
        ↦ ∫_X cl([Y]) ∪ cl([Z]) = 相交数
```

### 4.2 三种观点的优势对比

**代数循环**: 
- ✅ 保持几何信息
- ✅ 适用于任意域
- ❌ 等价关系难以计算

**Chow环**:
- ✅ 丰富的代数结构
- ✅ 陈类理论自然
- ❌ 仅对光滑簇定义良好

**上同调**:
- ✅ 强大的计算工具
- ✅ 与Hodge理论联系
- ❌ 丢失 torsion 信息

## 5. 经典应用

### 5.1 Bezout定理的三种证明

```
问题: ℙ²中两条一般曲线 C_d, C_e 的交点数?

Chow环方法:
    [C_d] = d·[H], [C_e] = e·[H]
    [C_d]·[C_e] = de·[H]² = de·[pt]
    ∴ 相交数 = de

上同调方法:
    H^*(ℙ²) = ℤ[h]/(h³)
    cl([C_d]) = d·h, cl([C_e]) = e·h
    ∫ cl([C_d])∪cl([C_e]) = ∫ de·h² = de
```

### 5.2 Grassmannian的Schubert演算

```
G(k, n) = {ℙ^{k-1} ⊂ ℙ^{n-1}}

Schubert循环: σ_λ 对应Young图
相交规则: Littlewood-Richardson规则

例子: σ_{(1)}·σ_{(1)} = σ_{(2)} + σ_{(1,1)}
```

### 5.3 Riemann-Roch定理

```
对于线丛 L 在光滑射影簇 X 上:

χ(X, L) = ∫_X ch(L) td(X)  （Hirzebruch-Riemann-Roch）

其中 ch(L) = e^{c_1(L)}, td(X) = Todd类
```

## 6. 现代发展

### 6.1 虚拟基本类

```
当模空间 M 不光滑时（如Gromov-Witten理论）:

[M]^{vir} ∈ A_{vdim}(M)

通过完美障碍理论构造:
    [E^{-1} → E^0]  切空间复形
```

### 6.2 导出相交

```
导出纤维积:
    X ×^h_Z Y 替代 X ×_Z Y
    
在导出代数几何中，即使传统相交维度"错误"，
仍有自然的相交积定义。
```
