## 一、问题陈述

### 1.1 核心问题

> Galois对应如何在域扩张的子域与Galois群的子群之间建立联系？这一对应的多维结构是什么？

### 1.2 直观理解

Galois对应是"对称性"与"结构"的桥梁：
- **域扩张** L/K 包含中间域的信息
- **Galois群** Gal(L/K) 包含对称性的信息
- **Galois对应**：中间域 ⟷ 子群，反转包含关系

## 三、符号表征（形式化）

### 3.1 Galois对应的严格陈述

**基本设定**：
- L/K 是有限Galois扩张
- G := Gal(L/K) = Aut_K(L)

**Galois对应**：

```
{中间域 F | K ⊆ F ⊆ L}  ⟷  {子群 H | H ⊆ G}

        F  ─────────→  Gal(L/F) := {σ ∈ G | σ|_F = id_F}
        
        L^H ←────────  H
        
其中 L^H := {x ∈ L | σ(x) = x, ∀σ ∈ H} 是 H 的固定域
```

**定理（Galois理论基本定理）**：

1. **双射**：上述对应是包含关系反转的双射
2. **次数公式**：|H| = [L : L^H]，[G : H] = [L^H : K]
3. **正规性**：H ⊲ G ⟺ L^H/K 是Galois扩张
4. **商群**：若 H ⊲ G，则 Gal(L^H/K) ≅ G/H

## 七、洞察与总结

### 7.1 核心洞察

| 洞察 | 内容 |
|------|------|
| **反转对应** | 域越大，Galois群越小（保持它的自同构越少） |
| **正规性对应** | 正规子群对应正规子扩张 |
| **可解性判据** | 方程可根式解 ⟺ Galois群可解 |
| **étale基本群** | Galois理论是代数几何中étale基本群的特例 |
