## 一、问题陈述

### 1.1 核心问题

> 复射影曲面如何分类？Enriques-Kodaira分类的决策树是什么？

### 1.2 直观理解

曲面分类是代数几何的里程碑：
- **曲线**（1维）：由亏格 g 完全分类
- **曲面**（2维）：需要更多不变量（Kodaira维数、陈类、Betti数等）

## 三、符号表征（形式化）

### 3.1 基本不变量

设 X 是复射影曲面，定义：

**数值不变量**：

```
q := h¹(X, O_X)           (非正则性)
p_g := h²(X, O_X)         (几何亏格)
χ := χ(O_X) = 1 - q + p_g  (Euler示性数)
b_i := dim H^i(X, ℚ)      (Betti数)
e := Σ(-1)^i b_i = c₂(X)  (拓扑Euler示性数)
```

**相交理论**：

```
K_X := 典范除子（余切丛的行列式）
K_X² := K_X · K_X         (自交数，典范体积)

Neron-Severi群：NS(X) := Pic(X)/Pic⁰(X)
ρ := rank NS(X)           (Picard数)
```

### 3.2 Kodaira维数的定义

**定义**：曲面 X 的 **Kodaira维数** 为：

```
κ(X) := tr.deg_ℂ ⊕_{n≥0} H⁰(X, O_X(nK_X)) - 1

或者等价地：

κ(X) = limsup_{n→∞} log h⁰(nK_X) / log n
```

**取值**：
- κ = -∞：对所有 n > 0，h⁰(nK_X) = 0
- κ = 0：h⁰(nK_X) 有界（但非零对某n）
- κ = 1：h⁰(nK_X) ~ cn
- κ = 2：h⁰(nK_X) ~ cn²/2（最大增长）

```
│                    分类判别准则                                              │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  κ = -∞  ⟺  12χ = K_X² + e 且 K_X² < 0                                    │

```

## 六、洞察与总结

### 6.1 核心洞察

| 洞察 | 内容 |
|------|------|
| **Kodaira维数是核心** | 将曲面分为4大类，类似于曲线的亏格 |
| **κ=2最一般** | 一般型曲面类似于g≥2的曲线，理论最丰富 |
| **κ=0最特殊** | 包含K3、Enriques、阿贝尔曲面等"珍宝" |
| **双有理分类** | 分类是双有理的，需要进一步研究极小模型 |

### 6.3 关联概念

| 概念 | 关系 |
|------|------|
| **极小模型** | 曲面的典范模型，分类的基础 |
| **Castelnuovo准则** | 收缩(-1)-曲线的判据 |
| **Bogomolov不等式** | 对曲面向量丛的约束 |
| **Miyaoka-Yau不等式** | K² ≤ 9χ，对一般型曲面的约束 |
