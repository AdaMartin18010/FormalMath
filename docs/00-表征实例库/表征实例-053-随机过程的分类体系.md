## 1. 引言：随机过程的数学景观

### 1.1 基本定义
随机过程是概率空间 $(\Omega, \mathcal{F}, P)$ 上的一族随机变量 $\{X_t\}_{t \in T}$，其中指标集 $T$ 通常表示时间。

### 1.2 分类维度
随机过程可以从多个维度进行分类：
- **时间**: 离散时间 vs 连续时间
- **状态空间**: 离散状态 vs 连续状态
- **结构性质**: 马尔可夫性、鞅性、平稳性等

---

## 2. 多维表征

### 2.1 表征方式一：马尔可夫过程

**核心性质**: 无记忆性
$$P(X_{t+s} = y | X_u, u \leq t) = P(X_{t+s} = y | X_t)$$

未来只依赖于现在，与过去无关。

```
马尔可夫过程的层次:

离散时间马尔可夫链 (DTMC):
  状态空间: 可数集 S = {0, 1, 2, ...}
  时间: 离散 T = {0, 1, 2, ...}
  刻画: 转移矩阵 P = (p_{ij})
  
连续时间马尔可夫链 (CTMC):
  状态空间: 可数集 S
  时间: 连续 T = [0, ∞)
  刻画: 生成元矩阵 Q = (q_{ij})
  
扩散过程:
  状态空间: ℝ^n
  刻画: SDE dX_t = μ(X_t)dt + σ(X_t)dW_t
  
Feller过程:
  半群: T_t f(x) = E^x[f(X_t)]
  生成元: 无穷小生成元
```

**关键工具**:
- **Chapman-Kolmogorov方程**: 转移概率的复合性质
- **Kolmogorov前向/后向方程**: 演化方程
- **强马尔可夫性**: 停时后的无记忆性

---

### 2.2 表征方式二：鞅

**核心性质**: 条件期望保持
$$E[X_{t+s} | \mathcal{F}_t] = X_t$$

公平游戏的数学抽象。

```
鞅的层次结构:

离散时间鞅:
  定义: M_n 适应于 ℱ_n, E[M_{n+1}|ℱ_n] = M_n
  例子: 随机游走 S_n = Σ X_i (若 E[X_i]=0)
  
连续时间鞅:
  定义: M_t 适应于 ℱ_t, E[M_{t+s}|ℱ_t] = M_t
  例子: 布朗运动 W_t, W_t^2 - t
  
局部鞅:
  定义: 存在停时序列 τ_n ↑ ∞，使 M_{t∧τ_n} 是鞅
  重要性: Itô积分 ∫ H_s dW_s 是局部鞅
  
一致可积鞅:
  性质: 收敛性更好，Doob收敛定理适用
```

**鞅的关键定理**:

| 定理 | 内容 | 应用 |
|------|------|------|
| Doob可选停时定理 | 停时保持鞅性 | 赌博策略分析 |
| Doob极大不等式 | 控制最大值 | 概率估计 |
| 鞅收敛定理 | 一致可积鞅a.s.收敛 | 极限理论 |
| 鞅表示定理 | 鞅可表示为随机积分 | 金融定价 |

### 2.3 表征方式三：高斯过程

**核心性质**: 所有有限维分布都是多元正态分布

```
高斯过程的刻画:

均值函数: μ(t) = E[X_t]
协方差函数: C(s,t) = Cov(X_s, X_t)

性质:
• 完全由 μ 和 C 决定
• 线性泛函仍高斯
• 条件分布仍高斯
```

**经典例子**:

```
1. 布朗运动 (Wiener过程):
   μ(t) = 0, C(s,t) = min(s,t)
   独立增量，路径连续但处处不可微

2. Ornstein-Uhlenbeck过程:
   dX_t = -θX_t dt + σ dW_t
   均值回归，平稳分布为N(0, σ²/2θ)
   
3. 分数布朗运动:
   C(s,t) = ½(s^{2H} + t^{2H} - |t-s|^{2H})
   Hurst参数 H ∈ (0,1)
   • H = 1/2: 标准布朗运动
   • H > 1/2: 长程相关
   • H < 1/2: 反持续性
   
4. 布朗桥:
   X_0 = X_1 = 0 的条件布朗运动
   协方差: C(s,t) = min(s,t) - st
```

### 4.2 半鞅的统摄地位

```
半鞅 (Semimartingale):

定义: 可以分解为局部鞅 + 有限变差过程

X_t = X_0 + M_t + A_t

其中 M 是局部鞅，A 是有界变差过程

重要性:
• Itô随机积分 ∫ H_s dX_s 只对半鞅定义良好
• 包含所有常见随机过程
• 金融数学的基础框架
```

## 5. 应用与意义

### 5.1 应用领域

| 领域 | 典型过程 | 核心应用 |
|------|---------|---------|
| **金融数学** | 布朗运动、几何布朗运动 | Black-Scholes期权定价 |
| **统计物理** | 扩散过程、Langevin方程 | 粒子运动建模 |
| **排队论** | 生灭过程、CTMC | 系统性能分析 |
| **信号处理** | 高斯过程、噪声过程 | 滤波、预测 |
| **机器学习** | 高斯过程回归 | 贝叶斯优化 |

### 5.2 金融数学中的应用

```
Black-Scholes模型:

dS_t = μS_t dt + σS_t dW_t

其中 W_t 是布朗运动（维纳过程）

性质:
• S_t 是几何布朗运动
• S_t 是马尔可夫过程
• log(S_t) 是高斯过程
• (discounted) S_t 是鞅（在风险中性测度下）

期权定价:
• 风险中性定价: E_Q[e^{-rT} payoff]
• 利用鞅表示定理
• 利用马尔可夫性得到PDE (Black-Scholes PDE)
```
