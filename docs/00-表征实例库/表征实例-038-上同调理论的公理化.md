## 一、问题陈述

### 1.1 核心问题

> 什么是上同调理论的Eilenberg-Steenrod公理？不同上同调理论（奇异、Čech、de Rham等）如何统一？

### 1.2 历史背景

1945年，Eilenberg和Steenrod提出了一套公理，刻画了"好的"上同调理论：
- **目的**：统一各种上同调理论
- **结果**：满足公理的上同调理论在同伦等价意义下唯一
- **例外**：广义上同调理论（如K-理论）不满足维数公理

## 三、符号表征（形式化）

### 3.1 上同调理论的严格定义

**定义（上同调理论）**：一个 **上同调理论** 是一对 (H^*, δ)，其中：

1. **H^q**：对每个整数 q ≥ 0，给出函子
   ```
   H^q: {拓扑空间对} → {Abel群}
   ```

2. **δ^q**：连接同态（自然变换）
   ```
   δ^q: H^q(A) → H^{q+1}(X, A)
   ```

3. 满足Eilenberg-Steenrod公理0-4

4. **普通上同调理论**还满足维数公理5

### 3.2 长正合序列的构造

**公理3的详细说明**：

对空间对 (X, A)，包含映射诱导：

```
i: A ↪ X,    j: (X, ∅) ↪ (X, A)

得到序列：
... → H^q(X, A) --j*--> H^q(X) --i*--> H^q(A) --δ--> H^{q+1}(X, A) → ...
```

**正合性**意味着：
- Im(j*) = Ker(i*)
- Im(i*) = Ker(δ)
- Im(δ) = Ker(j*^{q+1})

### 3.3 Lean 4 形式化

```lean
import Mathlib.AlgebraicTopology.EilenbergSteenrod

variable {C : Type*} [Category C] [HasForget C]

-- 上同调理论的结构
structure CohomologyTheory where
  H : ℕ → TopPair ⥤ Ab
  δ : ∀ n, H n ⋙ forget₂ _ (HomologicalComplex _ (ComplexShape.down ℕ))
      ⟶ H (n+1) ⋙ forget₂ _ (HomologicalComplex _ _)
  
  -- 公理1: 同伦不变性
  homotopy_invariance : ∀ {X Y : TopPair} {f g : X ⟶ Y},
    Homotopic f g → ∀ n, (H n).map f = (H n).map g
  
  -- 公理2: 切除
  excision : ∀ {X : Top} {A U : Set X},
    closure U ⊆ interior A →
    ∀ n, (H n).map (excisionMap X A U) 是同构
  
  -- 公理3: 正合性
  exactness : ∀ (X : TopPair) n,
    (longExactSequence H δ X).Exact
  
  -- 公理4: 可加性
  additivity : ∀ {ι : Type*} (X : ι → Top),
    (∀ n, (H n).obj (TopPair.mk (Σ X) _) ≅ Π i, (H n).obj (TopPair.mk (X i) _))

-- 维数公理
structure OrdinaryCohomologyTheory extends CohomologyTheory where
  dimension_axiom : ∀ n > 0, (H n).obj (TopPair.mk PUnit _) = 0
  coeff_group : (H 0).obj (TopPair.mk PUnit _)

-- 唯一性定理
theorem uniqueness (H H' : OrdinaryCohomologyTheory) 
    [H.coeff_group ≅ H'.coeff_group] :
    ∀ X, H.H ≅ H'.H := by
```

## 六、洞察与总结

### 6.1 核心洞察

| 洞察 | 内容 |
|------|------|
| **公理的力量** | 7条公理完全刻画了普通上同调理论 |
| **唯一性** | 满足公理的上同调理论在CW复形上唯一 |
| **广义化** | 去掉维数公理得到广义上同调理论（K-理论、配边等）|
| **可计算性** | 公理提供了计算上同调的工具（Mayer-Vietoris等）|
