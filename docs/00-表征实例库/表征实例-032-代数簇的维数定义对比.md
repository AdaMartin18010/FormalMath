## 一、问题陈述

### 1.1 核心问题

> 代数簇的"维数"有几种等价定义？它们之间如何相互联系？

### 1.2 直观理解

维数是代数簇"自由度"的度量：
- **几何直观**：曲线(1维) → 曲面(2维) → 立体(3维)
- **代数对应**：函数域的超越次数
- **拓扑对应**：不可约闭子集链的长度

## 三、符号表征（形式化）

### 3.1 五种定义的严格陈述

**定义 1（Krull维数）**：设 R 是交换环，

```
dim_{Krull}(R) := sup{n | ∃ p₀ ⊂ p₁ ⊂ ... ⊂ p_n, p_i ∈ Spec(R)}
```

对仿射簇 X = Spec A，定义 dim X := dim_{Krull}(A)。

---

**定义 2（超越次数）**：设 X 是不可约代数簇，函数域 K(X)，

```
dim(X) := tr.deg_k K(X)
```

即 K(X) 中在 k 上代数无关的元素的最大个数。

---

**定义 3（切空间维数）**：设 P ∈ X，m_P 是局部环 O_{X,P} 的极大理想，

```
dim T_P(X) := dim_k m_P/m_P²
```

在非奇异点，dim T_P(X) = dim X。

---

**定义 4（微分形式秩）**：设 X 是光滑簇，Ω_{X/k} 是 Kähler 微分层，

```
dim(X) := rank_{O_X} Ω_{X/k}
```

---

**定义 5（覆盖维数）**：

```
dim_{top}(X) := sup{n | ∃ 不可约闭子集链 Z₀ ⊂ Z₁ ⊂ ... ⊂ Z_n}
```

## 七、洞察与总结

### 7.1 核心洞察

| 洞察 | 内容 |
|------|------|
| **多种等价** | 维数的等价定义反映了代数几何的深度统一性 |
| **局部=整体** | 在非奇异点，局部切空间维数等于整体维数 |
| **Noether正规化** | 任何仿射簇都是仿射空间的有限覆盖，维数传递 |
| **奇点检测** | 切空间维数 > Krull维数 ⟺ 点是奇异的 |

### 7.2 重要定理回顾

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    维数理论的核心定理                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  1. Noether正规化引理                                                    │
│     任何有限生成k-代数A都有多项式子环                                     │
│     k[x₁,...,xₙ] ⊂ A 使得 A 在其上整                                      │
│     且 n = dim A                                                        │
│                                                                         │
│  2. Krull主理想定理                                                      │
│     设 R 是Noether环，f ∈ R 不是零因子                                     │
│     则 ht(f) ≤ 1，即由f定义的子簇维数降1                                   │
│                                                                         │
│  3. 维数公式（正则局部环）                                                │
│     对正则局部环 (R, m)：                                                │
│     dim R = dim_k m/m² = depth R                                         │
│                                                                         │
│  4. Serre定理                                                             │
│     对光滑射影簇，维数 = 结构层的上同调维数                                 │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 7.3 关联概念

| 概念 | 关系 |
|------|------|
| **深度** | depth ≤ dim，等号成立时是Cohen-Macaulay环 |
| **高度** | ht(p) + dim(R/p) = dim R |
| **Codimension** | codim_Z X = dim X - dim Z |
| **Hilbert多项式** | deg(Hilbert多项式) = dim 射影簇 |
