## 1. 引言：PDE理论的核心问题

### 1.1 Hadamard适定性概念
偏微分方程的适定性（well-posedness）由Hadamard在1902年提出，包含三个基本要求：

1. **存在性** (Existence): 解存在
2. **唯一性** (Uniqueness): 解唯一
3. **稳定性** (Stability): 解连续依赖于数据

### 1.2 为什么适定性重要
```
适定性 ⟺ 问题的数学表述正确

不适定的问题:
• 可能没有解（过度约束）
• 可能有无穷多解（约束不足）
• 可能对数据极度敏感（数值计算困难）
```

## 2. 多维表征

### 2.1 表征方式一：存在性理论

**主要方法**: 从显式构造到抽象泛函分析

```
存在性证明的演进:

古典方法 (18-19世纪):
├── 分离变量法 (Fourier)
├── 特征线法 (一阶方程)
├── Green函数法
└── 显式积分公式

现代方法 (20世纪以来):
├── 能量方法 (双曲/抛物方程)
│   └── 先验估计 + 弱收敛
├── 变分方法 (椭圆方程)
│   └── 极小化泛函
├── 半群理论 (演化方程)
│   └── Hille-Yosida定理
├── 不动点方法 (非线性方程)
│   ├── Banach不动点
│   ├── Schauder不动点
│   └── Leray-Schauder度
└── 粘性解理论 (Hamilton-Jacobi)
```

**能量方法示例** (波动方程):
```
□u = f 在 ℝ^n × (0,T)
u(·,0) = g, ∂_t u(·,0) = h

能量: E(t) = ½∫(|∂_t u|² + |∇u|²)dx

关键观察:
dE/dt = 0 (当 f=0 时)

⟹ 能量守恒 ⟹ 先验估计 ⟹ 存在性
```

### 2.2 表征方式二：唯一性理论

**核心思想**: 两个解的差满足齐次方程

```
唯一性证明的典型策略:

1. 能量不等式法:
   设 u₁, u₂ 都是解，令 w = u₁ - u₂
   w 满足齐次方程 + 零初边值
   ⟹ 能量 E[w] = 0
   ⟹ w = 0
   ⟹ u₁ = u₂

2. 极值原理法 (椭圆/抛物):
   若 Lu = 0 在 Ω, u|_∂Ω = 0
   且 L 满足极值原理
   ⟹ max|u| ≤ max|边界值| = 0
   ⟹ u = 0

3. 对偶方法:
   利用伴随方程的唯一性
   通过泛函分析论证

4. 解析延拓 (特殊情形):
   解的唯一性由边界值决定
   利用解析函数的刚性
```

**Holmgren唯一性定理**:
```
线性PDE的解析唯一性:

若 P(x,D) 是系数解析的线性微分算子，
且 Cauchy问题的数据在解析曲面上给出，
则解在解析函数类中唯一。

意义: 即使C^∞解不唯一，解析解也唯一
```

### 2.3 表征方式三：稳定性与连续依赖

**连续依赖的数学表述**:

```
解映射的连续性:

数据空间 X ∋ (f, g, h) ⟼ u ∈ Y (解空间)

要求: ||u₁ - u₂||_Y ≤ C||(f₁,g₁,h₁) - (f₂,g₂,h₂)||_X

即: 解算子 S: X → Y 是有界线性算子
```

**不稳定问题的例子**:

```
Hadamard经典例子 - 反向热方程:

∂_t u = -∂_x² u  (注意符号!)
u(x,0) = φ(x)

问题: 若 φ_n(x) = φ(x) + (1/n)sin(nx)
     则解 u_n(x,t) 在 t>0 时指数增长
     
     ||φ_n - φ||_∞ → 0
     但 ||u_n(·,t) - u(·,t)||_∞ → ∞

结论: 反向热方程不适定！
      (热扩散的不可逆性)
```

**条件稳定性**:
```
不适定问题的正则化:

若限制在紧集上，可能恢复稳定性:

Tikhonov正则化:
u_α = argmin ||Au - f||² + α||u||²

⟹ 不适定问题转化为适定的优化问题
```

## 4. 深层联系

### 4.1 适定性与物理定律

```
物理意义对应:

适定性 ⟺ 物理可预测性

• 存在性: 自然现象会发生
• 唯一性: 给定条件，结果确定
• 稳定性: 微小测量误差不会导致巨大偏差

例子:
• 热方程正向: 适定 (热量自然扩散)
• 热方程反向: 不适定 (无法追溯历史温度)
• 波动方程: 适定 (波动可预测传播)
```

### 4.2 适定性与数值计算

```
数值计算的适定性要求:

连续问题适定 ⟹ 离散格式稳定

Lax等价定理:
对于适定的线性初值问题，
一致性 + 稳定性 ⟺ 收敛性

⟹ 适定性是数值方法的数学基础
```

## 5. 应用与意义

### 5.1 各领域的适定性问题

| 领域 | 典型方程 | 适定性问题 |
|------|---------|-----------|
| **流体力学** | Navier-Stokes | 千年难题：三维解的存在唯一性 |
| **广义相对论** | Einstein方程 | Cauchy问题的适定性 |
| **量子力学** | Schrödinger方程 | 初值问题的适定性 |
| **最优控制** | Hamilton-Jacobi-Bellman | 粘性解框架 |
| **图像处理** | 非线性扩散方程 | 正则化方法 |

### 5.2 不适定问题的处理

```
不适定问题的正则化策略:

1. Tikhonov正则化:
   min ||Au - f||² + α||Lu||²
   (添加惩罚项)

2. 截断SVD:
   截断小奇异值
   稳定解但损失精度

3. 迭代正则化:
   Landweber迭代等
   早停防止过拟合

4. 贝叶斯方法:
   引入先验分布
   最大后验估计
```
