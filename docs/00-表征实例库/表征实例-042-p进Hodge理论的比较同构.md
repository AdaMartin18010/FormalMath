## 一、问题陈述

### 1.1 核心问题

> p进Hodge理论如何统一de Rham、étale和晶体上同调？比较定理的内容是什么？

### 1.2 历史背景

经典Hodge理论建立了复代数簇上不同上同调理论的联系：
- 比较同构：H^i_{dR}(X/ℂ) ≅ H^i_B(X, ℚ) ⊗_ℚ ℂ

p进Hodge理论的目标是在p进域上建立类似的理论：
- 连接 de Rham、étale、晶体上同调
- 由Tate、Faltings、Fontaine等数学家发展

```

    K : p进域（ℚ_p 的有限扩张）

═══════════════════════════════════════════════════════════════════════

    H^i_{dR}(X/K) ⊗_K B_{dR} ≅ H^i_ét(X_{K̄}, ℚ_p) ⊗_{ℚ_p} B_{dR}
    
    这是滤过等变同构
    
    推论：若 X 有良好约化，则上同调信息由étale完全决定

Step 3: de Rham ↔ 晶体 比较 (Berthelot, Ogus)
═══════════════════════════════════════════════════════════════════════

    H^i_{dR}(X/K) ≅ H^i_{cris}(𝒳_k/W(k)) ⊗_{W(k)} K
    
    这是Frobenius等变同构

Step 4: 晶体 ↔ étale 比较
═══════════════════════════════════════════════════════════════════════

    通过 de Rham 作为桥梁，或直接用Fontaine-Messing理论
    
    H^i_{cris}(𝒳_k/W(k)) ⊗_{W(k)} B_{cris} ≅ H^i_ét(X_{K̄}, ℚ_p) ⊗_{ℚ_p} B_{cris}

Step 5: 统一：p进Hodge分解
═══════════════════════════════════════════════════════════════════════

    H^i_ét(X_{K̄}, ℚ_p) ⊗_{ℚ_p} ℂ_p ≅ ⊕_{j=0}^i H^{i-j}(X, Ω^j_X) ⊗_K ℂ_p(-j)
    
    这是Hodge-Tate分解，最简单的比较形式
```

## 三、符号表征（形式化）

### 3.1 Fontaine环的定义

**设定**：K/ℚ_p 有限扩张，C_K = K̂（完备化）

**Hodge-Tate环**：

```
B_{HT} := ⊕_{n∈ℤ} C_K(n)

其中 C_K(n) = C_K ⊗ ℚ_p(n) 是Tate扭曲
```

**de Rham周期环** B_{dR}：

```
B_{dR}^+ := lim←_n (W(R)[1/p])/Ker(θ)^n

其中 R = lim←_{x↦x^p} O_{C_K}/p，θ: W(R) → O_{C_K}

B_{dR} := Frac(B_{dR}^+)，有自然的下降滤过
```

**晶体周期环** B_{cris}：

```
B_{cris}^+ := { Σ_{n≥0} a_n ω^n/n! | a_n ∈ W(R)[1/p] → 0 }

其中 ω = [ε] - 1，ε = (1, ζ_p, ζ_{p²}, ...)

B_{cris} := B_{cris}^+[1/t]，t = log([ε])

有Frobenius φ 和到 B_{dR} 的嵌入
```

**半稳定周期环** B_{st}：

```
B_{st} := B_{cris}[log([π])]

其中 π = p 的相容系统

有额外的单值化算子 N = -d/dlog([π])
```

### 3.2 p进表示的分类

**定义**：设 V 是p进Galois表示（ℚ_p-向量空间 + Gal(K̄/K)-作用）

```
V 是 Hodge-Tate  ⟺  dim_K (V ⊗_{ℚ_p} B_{HT})^{Gal} = dim_{ℚ_p} V

V 是 de Rham    ⟺  dim_K (V ⊗_{ℚ_p} B_{dR})^{Gal} = dim_{ℚ_p} V

V 是 半稳定     ⟺  dim_{K_0} (V ⊗_{ℚ_p} B_{st})^{Gal} = dim_{ℚ_p} V

V 是 晶体       ⟺  dim_{K_0} (V ⊗_{ℚ_p} B_{cris})^{Gal} = dim_{ℚ_p} V
```

**包含关系**：

```
{晶体} ⊂ {半稳定} ⊂ {de Rham} ⊂ {Hodge-Tate} ⊂ {所有表示}
```

### 3.3 比较定理的陈述

**de Rham比较定理**（Faltings, Tsuji）：

```
定理：设 X/K 光滑真，则存在自然的同构

H^i_{dR}(X/K) ⊗_K B_{dR} ≅ H^i_ét(X_{K̄}, ℚ_p) ⊗_{ℚ_p} B_{dR}

这是滤过等变的，且与Gal(K̄/K)作用相容
```

**晶体比较定理**（Fontaine-Messing, Faltings）：

```
定理：设 𝒳/W(k) 光滑真，则存在自然的同构

H^i_{cris}(𝒳_k/W) ⊗_W B_{cris} ≅ H^i_ét(𝒳_{K̄}, ℚ_p) ⊗_{ℚ_p} B_{cris}

这是φ-等变的
```
