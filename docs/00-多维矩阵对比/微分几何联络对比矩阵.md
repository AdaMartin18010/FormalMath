# 微分几何联络对比矩阵

> 系统对比微分几何中六种核心联络：Levi-Civita、仿射联络、度量联络、共形联络、射影联络、辛联络。

---

## 总览矩阵

| 维度 | Levi-Civita | 仿射联络 | 度量联络 | 共形联络 | 射影联络 | 辛联络 |
|------|-------------|---------|---------|---------|---------|--------|
| **存在性** | 唯一（给定度量）| 大量 | 大量（给定度量）| 大量 | 大量 | 大量（给定辛形式）|
| **无挠要求** | 是 ($T=0$) | 可选 | 可选 | 是 | 可选 | 是 ($T=0$) |
| **度量相容** | 是 ($\nabla g=0$) | 否 | 是 | 共形 ($\nabla g=\omega\otimes g$) | 否 | 辛相容 ($\nabla\omega=0$) |
| **曲率类型** | Riemann 曲率 | 一般曲率 | 度量曲率 | Weyl 曲率 | 射影曲率 | 辛曲率 |
| **标准例子** | 欧氏空间、球面 | 平坦空间任意标架 | 伪 Riemann 流形 | 共形平坦流形 | 射影平坦流形 | 辛流形 |
| **变换群** | 等距群 | 仿射变换群 | 相似变换群 | 共形变换群 | 射影变换群 | 辛群 |
| **最大对称维数** | $n(n+1)/2$ | $n(n+1)$ | $n(n+1)/2$ | $(n+1)(n+2)/2$ | $n(n+2)$ | 无限维（局部）|

---

## Levi-Civita 联络

| 性质 | 内容 |
|------|------|
| **定理** | 基本定理 of Riemann 几何 |
| **存在唯一性** | 给定伪 Riemann 度量 $g$，存在唯一的无挠且度量相容的联络 |
| **Christoffel 符号** | $\Gamma^k_{ij}=\frac{1}{2}g^{kl}(\partial_i g_{jl}+\partial_j g_{il}-\partial_l g_{ij})$ |
| **曲率** | Riemann 曲率张量 $R^k_{\ lij}$ |
| **Ricci 张量** | $R_{ij}=R^k_{\ ikj}$ |
| **标量曲率** | $R=g^{ij}R_{ij}$ |
| **测地线** | $\nabla_{\dot\gamma}\dot\gamma=0$，局部最短路径 |
| **例子** | 欧氏空间（$\Gamma=0$）、球面（常曲率）、双曲空间 |

---

## 仿射联络 vs 度量联络

| 性质 | 仿射联络 | 度量联络 |
|------|---------|---------|
| **定义** | 流形切丛上的联络，无额外结构 | 保持度量 $g$ 的联络 ($\nabla g=0$) |
| **挠率** | $T(X,Y)=\nabla_XY-\nabla_YX-[X,Y]$，可非零 | 可非零 |
| **曲率** | $R(X,Y)Z=[\nabla_X,\nabla_Y]Z-\nabla_{[X,Y]}Z$ | 同上，但有度量降指标 |
| **测地线** | 由联络定义 | 由联络定义 |
| **例子** | 平坦联络（$\Gamma=0$ 在某坐标系）| Levi-Civita、Weyl 联络 |
| **结构群** | $GL(n,\mathbb{R})$ | $O(p,q)$ |

---

## 共形联络

| 性质 | 内容 |
|------|------|
| **定义** | 满足 $\nabla g=\omega\otimes g$（$\omega$ 为 1-形式）|
| **Weyl 联络** | 无挠的共形联络 |
| **曲率分解** | Riemann 曲率 = Weyl 张量 + Ricci 部分 + 标量部分 |
| **Weyl 张量** | 共形不变量，$C_{ijkl}$ |
| **共形平坦** | Weyl 张量 = 0（$n\ge 4$）|
| **例子** | 球面、欧氏空间、双曲空间（互相共形）|

---

## 射影联络

| 性质 | 内容 |
|------|------|
| **定义** | 保持无参数化测地线集合的联络等价类 |
| **射影曲率** | 测地线偏离方程的曲率不变量 |
| **射影平坦** | 局部同胚于欧氏空间（测地线为直线）|
| **Weyl 射影张量** | $W_{ijkl}=R_{ijkl}-\frac{1}{n-1}(g_{ik}R_{jl}-g_{il}R_{jk})$ |
| **例子** | 常曲率空间射影平坦 |

---

## 辛联络

| 性质 | 内容 |
|------|------|
| **定义** | 满足 $\nabla\omega=0$ 且 $T=0$ 的联络 |
| **存在性** | 任意辛流形上存在（不唯一）|
| **曲率** | 满足 $\omega(R(X,Y)Z,W)=\omega(R(Z,W)X,Y)$ |
| **例子** | Kähler 流形上的 Levi-Civita 联络（自动辛相容）|
| **与复结构关系** | Kähler = 辛 + 复 + 相容 |

---

## 蕴含与兼容性

```
Levi-Civita = 仿射联络 + 无挠 + 度量相容

Kähler 联络 = Levi-Civita + 辛相容 + 复结构相容

共形联络 ⟹ 射影结构（共形测地线是测地线）
```

---

## 教学速记

| 你要保持的结构 | 选择联络 |
|--------------|---------|
| 度量 + 无挠 | Levi-Civita |
| 仅仿射结构 | 仿射联络 |
| 度量（允许有挠）| 度量联络 |
| 共角结构 | Weyl（共形）联络 |
| 测地线结构（无参数）| 射影联络 |
| 辛结构 + 无挠 | 辛联络 |
| 复 + 辛 + 度量 | Kähler / Levi-Civita |
