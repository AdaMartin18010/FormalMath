# 偏微分方程类型对比矩阵

> 系统对比椭圆型、抛物型、双曲型与混合型 PDE 的理论特征、解法与典型例子。

---

## 总览矩阵

| 维度 | 椭圆型 (Elliptic) | 抛物型 (Parabolic) | 双曲型 (Hyperbolic) | 混合型 |
|------|------------------|-------------------|--------------------|--------|
| **标准例子** | Laplace $\Delta u=0$ | 热方程 $u_t=\Delta u$ | 波动方程 $u_{tt}=\Delta u$ | Tricomi 方程 $yu_{xx}+u_{yy}=0$ |
| **特征** | 无实特征 | 一族实重特征 | 两族不同实特征 | 区域依赖 |
| **适定性** | 边值问题适定 | 初边值问题适定 | 初值问题适定 | 复杂 |
| **信息传播** | 全局（椭圆正则性）| 单向（时间正向）| 有限速度（特征锥）| 混合 |
| **光滑性** | 解自动光滑（内部）| 无穷光滑 ($t>0$) | 保持初值正则性 | 复杂 |
| **极值原理** | 有 | 有 | 无 | 局部有 |
| **能量** | 守恒/衰减 | 耗散 | 守恒 | 复杂 |
| **Green 函数** | 对称 | 非对称 | 分布意义 | — |

---

## 二阶线性 PDE 分类（两个自变量）

$$Au_{xx}+2Bu_{xy}+Cu_{yy}+\text{低阶项}=f$$

| 判别式 $\Delta=B^2-AC$ | 类型 | 标准形 | 特征方程 |
|------------------------|------|--------|---------|
| $\Delta<0$ | 椭圆型 | $u_{\xi\xi}+u_{\eta\eta}+\cdots=f$ | 无实特征 |
| $\Delta=0$ | 抛物型 | $u_{\xi\xi}+\cdots=f$ | 一族实重特征 |
| $\Delta>0$ | 双曲型 | $u_{\xi\eta}+\cdots=f$ 或 $u_{\xi\xi}-u_{\eta\eta}+\cdots=f$ | 两族不同实特征 |

---

## 椭圆型 PDE

| 性质 | 内容 |
|------|------|
| **代表方程** | Laplace $\Delta u=0$，Poisson $\Delta u=f$ |
| **边值问题** | Dirichlet、Neumann、Robin 问题 |
| **适定性** | Dirichlet 问题在光滑区域上适定 |
| **正则性** | 若 $f\in C^k$，则 $u\in C^{k+2}$（Schauder 估计）；若 $f$ 光滑，$u$ 光滑 |
| **极值原理** | 调和函数在区域内部不取最大/最小值（除非常数）|
| **均值性质** | $u(x)=\frac{1}{|B_r|}\int_{B_r(x)}u\,dy$ |
| **Green 函数** | $G(x,y)$ 对称，$u(x)=\int_\Omega G(x,y)f(y)\,dy+\text{边界项}$ |
| **变分结构** | Euler-Lagrange 方程 of $\int|\nabla u|^2$ |
| **例子** | 静电势、稳态热分布、不可压无旋流 |

---

## 抛物型 PDE

| 性质 | 内容 |
|------|------|
| **代表方程** | 热方程 $u_t=\Delta u$，反应扩散 $u_t=\Delta u+f(u)$ |
| **初边值问题** | 需给 $t=0$ 的初值 + 空间边界条件 |
| **适定性** | 初边值问题适定（时间正向）|
| **正则性** | $t>0$ 时解自动光滑（无穷可微），即使初值仅 $L^2$ |
| **极值原理** | 最大值原理：$\max u$ 在抛物边界上取到 |
| **基本解** | 热核 $K(t,x)=(4\pi t)^{-n/2}e^{-|x|^2/4t}$ |
| **传播速度** | 无穷（扰动瞬间传遍全空间）|
| **时间可逆性** | 不可逆（不适定反向求解）|
| **例子** | 热传导、布朗运动、Black-Scholes 方程 |

---

## 双曲型 PDE

| 性质 | 内容 |
|------|------|
| **代表方程** | 波动方程 $u_{tt}=c^2\Delta u$，传输方程 $u_t+cu_x=0$ |
| **初值问题** | Cauchy 问题：$u(0,x)$ 和 $u_t(0,x)$ |
| **适定性** | 初值问题适定 |
| **正则性** | 解的正则性与初值相同（不增光滑性）|
| **极值原理** | 无（解可振荡）|
| **特征方法** | 沿特征线传播信息 |
| **传播速度** | 有限（特征锥：$|x|=ct$）|
| **时间可逆性** | 可逆（$t\to -t$ 不变）|
| **能量守恒** | $E(t)=\int(u_t^2+c^2|\nabla u|^2)\,dx$ 守恒 |
| **例子** | 声波、电磁波、弹性波、弦振动 |

## 解法对比

| 解法 | 椭圆型 | 抛物型 | 双曲型 |
|------|--------|--------|--------|
| **分离变量** | ✅ 适用 | ✅ 适用 | ✅ 适用 |
| **Green 函数** | ✅ 核心工具 | ⚠️ 热核可用 | ❌ 一般不用 |
| **特征线法** | ❌ 无特征 | ❌ 无特征（除退化）| ✅ 核心工具 |
| **Fourier 变换** | ⚠️ 全空间可用 | ✅ 核心工具 | ✅ 核心工具 |
| **能量方法** | ⚠️ 变分形式 | ⚠️ 衰减估计 | ✅ 核心工具 |
| **最大值原理** | ✅ 核心工具 | ✅ 核心工具 | ❌ 不存在 |
| **半群理论** | ⚠️ 椭圆算子生成 | ✅ 热方程半群 | ⚠️ 波动方程群 |
| **Sobolev 空间** | $H^1$ 自然框架 | $L^2$ 自然框架 | $H^1\times L^2$ 能量空间 |

## 高维与变系数推广

| 类型 | 高维判据 | 变系数 |
|------|---------|--------|
| **椭圆** | 系数矩阵 $A(x)$ 处处正定 | 一致椭圆：$\lambda|\xi|^2\le a_{ij}\xi_i\xi_j\le\Lambda|\xi|^2$ |
| **抛物** | 时间一阶 + 空间椭圆 | $u_t-\sum a_{ij}(x,t)\partial_{ij}u+\cdots=f$ |
| **双曲** | 主部符号有正负特征值 | 对称双曲组 |

## 教学速记

```
椭圆 = 稳态、光滑、极值原理、边值问题
抛物 = 演化、 smoothing、耗散、初边值问题、不可逆
双曲 = 波动、守恒、特征线、初值问题、可逆
```

| 物理直觉 | 类型 |
|---------|------|
| 平衡态 | 椭圆 |
| 扩散/耗散 | 抛物 |
| 传播/波动 | 双曲 |
