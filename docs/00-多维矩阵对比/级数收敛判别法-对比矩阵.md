# 级数收敛判别法多维对比矩阵

**适用**: 正项级数 $\sum a_n$（$a_n > 0$）  
**难度**: 本科实分析（MIT 18.100A Week 10-11）

---

## 判别法总览矩阵

| 判别法 | 所需条件 | 收敛判据 | 发散判据 | 无法判定 | 适用场景 |
|--------|----------|----------|----------|----------|----------|
| **比较法** | 已知$\sum b_n$ | $a_n \leq b_n$且$\sum b_n$收敛 | $a_n \geq b_n$且$\sum b_n$发散 | 不等式不成立 | 易找比较级数 |
| **极限比较** | $\lim \frac{a_n}{b_n} = L$ | $0 < L < \infty$且$\sum b_n$收敛 | $L > 0$且$\sum b_n$发散 | $L = 0$或$\infty$ | 渐近等价 |
| **比值法** | $\lim \frac{a_{n+1}}{a_n} = L$ | $L < 1$ | $L > 1$ | $L = 1$ | 含阶乘/指数 |
| **根值法** | $\lim \sqrt[n]{a_n} = L$ | $L < 1$ | $L > 1$ | $L = 1$ | 含n次幂 |
| **积分法** | $a_n = f(n)$, $f$递减 | $\int_1^\infty f < \infty$ | $\int_1^\infty f = \infty$ | - | 连续函数 |
| **Raabe** | 比值=1时 | $\lim n(\frac{a_n}{a_{n+1}}-1) > 1$ | $< 1$ | $= 1$ | 比值失效 |

---

## 判别法选择决策树

```
级数 ∑aₙ (aₙ > 0)
│
├─ 是否容易找到比较级数？
│  ├─ 是 → 比较法/极限比较法
│  └─ 否 →
│
├─ 是否含阶乘/指数？
│  ├─ 是 → 比值法
│  └─ 否 →
│
├─ 是否含n次幂？
│  ├─ 是 → 根值法
│  └─ 否 →
│
├─ 是否为连续函数值？
│  ├─ 是 → 积分法
│  └─ 否 →
│
└─ 比值=1 → Raabe判别法
```

## 典型例子速查

| 级数 | 推荐判别法 | 结果 |
|------|------------|------|
| $\sum \frac{1}{n^p}$ | 积分法 | $p > 1$收敛 |
| $\sum \frac{n!}{n^n}$ | 比值法 | 收敛 |
| $\sum \frac{1}{2^n + 1}$ | 比较法 | 收敛 |
| $\sum \frac{n^2+1}{n^4+2}$ | 极限比较 | 收敛 |
