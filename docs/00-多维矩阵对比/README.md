# 00-多维矩阵对比

本目录通过矩阵对比的形式，将数学中易混淆或高度相关的概念并置呈现，共收录 16 份对比文档。每份文件以多维矩阵为框架，横向比较不同对象在定义、性质、定理、例子与反例等维度上的异同，帮助学习者快速识别概念边界、强化记忆并建立结构化认知。覆盖范围包括代数结构、几何对象、分析方法、拓扑性质、数学家贡献与证明方法等多个主题，是高效复习与备考的实用工具库。

## 子目录/文件导航

本目录为扁平结构，主要文件包括：

| 文件 | 说明 |
|------|------|
| `01-代数结构完整对比矩阵.md` | 群、环、域、模的完整多维对比 |
| `01-代数结构对比矩阵.md` | 代数结构核心性质的精简对比 |
| `02-几何对象完整对比矩阵.md` | 流形、代数簇、复形等几何对象对比 |
| `02-几何对象对比矩阵.md` | 几何对象基础性质的快速对照 |
| `03-分析方法对比矩阵.md` | 不同分析技术（实分析、复分析、泛函）对比 |
| `03-拓扑性质对比矩阵.md` | 紧致性、连通性、分离公理等拓扑性质矩阵 |
| `04-分析方法对比矩阵.md` | 分析方法进阶维度对照 |
| `04-拓扑性质对比矩阵.md` | 拓扑性质在不同空间类的表现对比 |
| `05-数学家贡献矩阵.md` | 各领域奠基数学家的核心贡献一览 |
| `05-证明方法对比矩阵.md` | 常见证明策略及其适用场景对比 |
