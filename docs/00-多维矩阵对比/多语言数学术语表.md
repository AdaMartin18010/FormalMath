# FormalMath 多语言数学术语表

## 说明

本术语表为 FormalMath 项目的中英双语数学术语对照，支持银层和金层文档的国际对齐。

## 基础数学术语

| 中文 | 英文 | 备注 |
|------|------|------|
| 向量 | vector | 既有大小又有方向的量 |
| 标量 | scalar | 只有大小的量 |
| 点积 | dot product / inner product | 内积 |
| 叉积 | cross product | 外积，仅定义于三维 |
| 梯度 | gradient | $\nabla f$ |
| 散度 | divergence | $\nabla \cdot \mathbf{F}$ |
| 旋度 | curl | $\nabla \times \mathbf{F}$ |
| 偏导数 | partial derivative | $\partial f / \partial x$ |
| 全微分 | total differential | $df = f_x dx + f_y dy$ |
| 方向导数 | directional derivative | $D_{\mathbf{u}} f$ |

## 线性代数术语

| 中文 | 英文 | 备注 |
|------|------|------|
| 矩阵 | matrix | 矩形数组 |
| 行列式 | determinant | $\det(A)$ |
| 特征值 | eigenvalue | 特征根 |
| 特征向量 | eigenvector | 特征向量 |
| 秩 | rank | 矩阵的秩 |
| 零空间 | null space / kernel | $\ker(A)$ |
| 列空间 | column space | 列向量的张成空间 |
| 正交 | orthogonal | 内积为零 |
| 对称矩阵 | symmetric matrix | $A = A^T$ |
| 奇异值分解 | singular value decomposition (SVD) | $A = U\Sigma V^T$ |
