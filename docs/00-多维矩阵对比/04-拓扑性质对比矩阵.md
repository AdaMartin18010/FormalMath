## 一、核心拓扑性质对比

### 1.1 六大基本拓扑性质

| 对比维度 | Hausdorff (T₂) | 紧致 (Compact) | 连通 (Connected) | 单连通 (Simply Connected) | 可度量化 (Metrizable) | 仿紧 (Paracompact) |
|:--------:|:--------------:|:--------------:|:----------------:|:-------------------------:|:---------------------:|:------------------:|
| **分离性等级** | T₂ (强分离) | - | - | - | T₂+额外条件 | T₂+额外条件 |
| **核心定义** | 任意两点有不相交邻域 | 任意开覆盖有有限子覆盖 | 不能分解为两个非空不交开集 | 连通+基本群平凡 | 拓扑可由度量诱导 | 任意开覆盖有局部有限加细 |
| **直观理解** | 点可分离 | 有限覆盖/有界闭类比 | 一体/无洞 | 无"一维洞" | 距离化 | 局部有限结构 |
| **典型例子** | ℝⁿ, 度量空间, 流形 | [0,1], Sⁿ, 闭球, 有限集 | ℝⁿ, 连通和, 区间 | ℝ², S²(n≥2), Dⁿ | 度量空间 | 流形, 度量空间 |
| **经典反例** | Zariski拓扑<br>有限补拓扑(无限集)<br>箭头空间 | (0,1), ℝ<br>无理数集<br>离散无限集 | [0,1)∪(1,2]<br>ℚ⊂ℝ<br>康托集 | S¹, 环面T²<br>ℝ²-{0}<br>射影平面 | 非Hausdorff空间<br> Zariski拓扑<br>不可数积(无度量) | 某些病态空间<br>不可数闭 ordinal |
| **核心定理** | 紧+Hausdorff→正规<br>紧Hausdorff空间丰富 | 连续像保持紧致<br>Tychonoff定理<br>极值定理 | 连通和保持<br>连续像保持连通 | 单值化定理<br>Riemann映射定理<br>Hurewicz同构 | Urysohn度量化<br>Nagata-Smirnov<br>度量化定理 | 单位分解存在<br>流形必仿紧<br> fine sheaf |
| **保持性** | 子空间(继承)<br>积空间(保持)<br>连续像(不保持) | 闭子空间保持<br>连续像保持<br>有限积保持 | 连续像保持<br>闭包保持<br>积空间(不一定) | 同伦等价保持<br>(非同胚保持) | 子空间保持<br>可数积保持 | 闭子空间保持<br>可数积保持 |
| **应用价值** | 分析学基础<br>极限唯一性<br>序列刻画拓扑 | 极值定理<br>一致连续性<br>有限性论证 | 介值定理<br>连续性论证<br>整体性结论 | 复分析基础<br>复叠空间理论<br>规范场分类 | 几何学基础<br>分析学标准框架<br>第一可数+T₁ | 流形理论核心<br>层上同调<br>微分形式积分 |
| **判定难度** | 中等 | 中等 | 较易 | 困难(需计算π₁) | 困难 | 中等 |
| **与其他性质关系** | 紧+Hausdorff=正规 | 度量空间⇒仿紧+Hausdorff | 道路连通⊂连通 | 单连通⊂连通 | 度量⇒仿紧+Hausdorff | 仿紧+T₂⇒正规 |

### 1.2 分离性公理层级

| 分离性 | 定义 | 性质强度 | 典型例子 | 与紧性结合 |
|--------|------|:--------:|----------|-----------|
| **T₀** | 任意两点拓扑可区分 | ★☆☆☆☆ | Sierpinski空间 | - |
| **T₁** | 任意两点各有一个不包含另一点的邻域 | ★★☆☆☆ | Zariski拓扑 | - |
| **T₂ (Hausdorff)** | 有不相交邻域 | ★★★☆☆ | 度量空间 | 紧+T₂=正规 |
| **T₂½ (完全Hausdorff)** | 有不相交闭包邻域 | ★★★☆☆ | 正则空间 | - |
| **T₃ (正则)** | T₁+点与闭集可分离 | ★★★☆☆ | 拓扑群 | - |
| **T₃½ (Tychonoff)** | 完全正则 | ★★★☆☆ | 拓扑群 | 紧Hausdorff空间 |
| **T₄ (正规)** | T₁+不相交闭集可分离 | ★★★★☆ | 度量空间 | 紧+T₂⇒T₄ |
| **T₅ (完全正规)** | 所有子空间正规 | ★★★★☆ | 线性序拓扑 | - |
| **T₆ (完美正规)** | 闭集是Gδ集 | ★★★★★ | 度量空间 | - |

**包含关系链：**

```

T₆ ⇒ T₅ ⇒ T₄ ⇒ T₃½ ⇒ T₃ ⇒ T₂½ ⇒ T₂ ⇒ T₁ ⇒ T₀

```

## 二、连通性相关的精细分类

### 2.1 各种连通性对比

| 性质 | 定义 | 强度 | 典型例子 | 反例 |
|------|------|:----:|----------|------|
| **连通** | 不能分成两个非空开集 | ★☆☆☆ | 区间, ℝⁿ | ℚ, [0,1)∪(1,2] |
| **道路连通** | 任意两点可用道路连接 | ★★☆☆ | 凸集, ℝⁿ | 拓扑学家的正弦曲线 |
| **弧连通** | 道路可用弧(嵌入的[0,1])连接 | ★★★☆ | 流形 | 某些病态空间 |
| **局部连通** | 每点有连通的邻域基 | ★★☆☆ | 流形, 局部有限CW复形 | 拓扑学家的正弦曲线 |
| **局部道路连通** | 每点有道路连通的邻域基 | ★★★☆ | 流形 | Warsaw圆 |
| **单连通** | 道路连通+π₁平凡 | ★★★★ | ℝⁿ(n≥2), S² | S¹, 环面 |
| **n-连通** | k≤n时πₖ平凡 | ★★★★★ | S^{n+1} | Sⁿ |
| **绝对收缩核(AR)** | 强拓扑性质 | ★★★★★ | 凸集, Hilbert立方体 | 圆周 |

### 2.2 连通分支与局部结构

| 概念 | 定义 | 性质 | 计算/判定 |
|------|------|------|----------|
| **连通分支** | 极大连通子集 | 闭集，两两不交 | 一般难以显式计算 |
| **道路分支** | 极大道路连通子集 | 可能不闭 | 通常等同于连通分支(局部道路连通) |
| **拟分支** | 所有既开又闭集之交 | 包含于连通分支 | 理论工具 |
| **分支数** | 分支的基数 | 拓扑不变量 | 可数或连续统 |

## 三、紧致性的推广与变体

### 3.1 各种紧致性概念

| 性质 | 定义 | 与标准紧致关系 | 典型空间 | 应用场景 |
|------|------|:------------:|----------|----------|
| **紧致** | 任意开覆盖有有限子覆盖 | = | [0,1], Sⁿ | 最核心性质 |
| **序列紧致** | 任意序列有收敛子列 | 度量空间等价 | 第一可数+紧致⇒序列紧致 | 分析学常用 |
| **可数紧致** | 任意可数开覆盖有有限子覆盖 | 紧致⇒可数紧致 | ω₁ | 一般拓扑 |
| **极限点紧致** | 无限集有极限点 | 紧致⇒极限点紧致 | 可数紧致⇔极限点紧致 | 古典分析 |
| **局部紧致** | 每点有紧邻域 | 弱于紧致 | ℝⁿ, 流形 | 调和分析 |
| **σ-紧致** | 可数紧子集之并 | 弱于紧致 | ℝⁿ | 测度论 |
| **预紧致/相对紧致** | 闭包紧致 | - | 有界集(度量空间) | 泛函分析 |
| **Lindelöf** | 任意开覆盖有可数子覆盖 | 紧致⇒Lindelöf | 第二可数空间 | 度量化问题 |
| **伪紧致** | 连续实函数有界 | 紧致⇒伪紧致 | 非紧例子存在 | C(X)研究 |
| **实紧致** | 可嵌入某ℝᴶ | 紧致⇒实紧致 | 离散空间 | 环论应用 |

### 3.2 紧致化方法对比

| 紧致化 | 构造方法 | 唯一性 | 性质保留 | 典型应用 |
|--------|----------|:------:|----------|----------|
| **单点紧致化** | 添加∞点 | 同胚意义唯一 | 局部紧致⇔Hausdorff | 复分析(黎曼球面) |
| **Stone-Čech紧致化** | βX = Max(C_b(X)) | 最大紧致化 | Tychonoff空间 | C*-代数 |
| **端点紧致化(Freudenthal)** | 添加端点 | 唯一(某些条件) | 局部连通 | 几何群论 |
| **群紧致化** | Bohr紧化 | 范畴性质 | 拓扑群 | 调和分析 |

## 四、可度量化与相关性质

### 4.1 度量化条件对比

| 定理/条件 | 条件内容 | 充分性 | 必要性 | 重要性 |
|-----------|----------|:------:|:------:|:------:|
| **Urysohn度量化** | T₃ + 第二可数 | ✓ | ✗(有反例) | ★★★★★ |
| **Nagata-Smirnov** | T₃ + σ-局部有限基 | ✓ | ✓ | ★★★★★ |
| **Smirnov** | T₂ + σ-局部紧致+Hausdorff+仿紧 | ✓ | ✓ | ★★★☆☆ |
| **Bing** | T₃ + σ-离散基 | ✓ | ✓ | ★★★☆☆ |
| **紧+T₂** | 紧Hausdorff | ✓(弱) | ✗ | ★★★★☆ |

### 4.2 度量不变量

| 不变量 | 定义 | 完备性依赖 | 紧致性等价条件 |
|--------|------|:----------:|----------------|
| **全有界** | 任意ε>0可被有限ε球覆盖 | 全有界+完备⇔紧致 | 紧致⇔全有界+完备 |
| **可分** | 有可数的稠密子集 | 否 | - |
| **完备** | Cauchy列收敛 | 是 | 完备+全有界⇔紧致 |
| **Polish** | 可分+完备可度量化 | 是 | - |
| **绝对收缩核** | 任意嵌入为收缩核 | - | - |

## 五、仿紧性与层论应用

### 5.1 仿紧性相关性质

| 性质 | 定义 | 与仿紧关系 | 典型例子 | 层论意义 |
|------|------|:----------:|----------|----------|
| **仿紧** | 局部有限加细 | = | 度量空间, 流形 | 好层(fine sheaf) |
| **可数仿紧** | 可数开覆盖有局部有限加细 | 弱于仿紧 | - | - |
| **强仿紧** | 星有限加细 | 强于仿紧 | 度量空间 | - |
| **可收缩覆盖** | 开覆盖可逐点收缩 | - | 流形 | 好覆盖(good cover) |
| **软层(Soft)** | 截面可延拓 | 仿紧⇒软⇔优 | 光滑函数层 | 零上同调 |
| **优层(Fine)** | 有单位分解 | 仿紧+软⇔优 | 光滑函数层 | 零上同调 |

### 5.2 单位分解对比

| 类型 | 存在条件 | 光滑性 | 应用 |
|------|:--------:|:------:|------|
| **连续单位分解** | 仿紧+T₂ | C⁰ | 向量丛存在性 |
| **光滑单位分解** | 光滑流形 | C^∞ | 流形理论核心 |
| **从属于覆盖** | 给定开覆盖 | 取决于覆盖 | 局部到整体 |

## 六、拓扑性质的应用决策树

### 6.1 问题类型与所需性质

| 数学领域 | 核心拓扑性质 | 理由 | 典型定理 |
|----------|-------------|------|----------|
| **数学分析** | 可度量化, 局部紧致 | 极限、连续性、积分 | Ascoli-Arzelà, Riesz表示 |
| **泛函分析** | 局部凸, 完备 | 对偶理论, 分布 | Hahn-Banach, 开映射 |
| **代数拓扑** | 仿紧, 局部道路连通 | 同伦, 复叠空间 | Hurewicz, 提升定理 |
| **微分几何** | 仿紧, 第二可数 | 流形基本假设 | Whitney嵌入, Sard定理 |
| **复分析** | 单连通, Hausdorff | 全纯函数理论 | Cauchy定理, Riemann映射 |
| **动力系统** | 紧致, 连通 | 遍历理论, 稳定性 | Poincaré回归, Birkhoff |
| **代数几何** | Zariski拓扑(T₁) | 代数簇结构 | 层上同调, 射影态射 |

### 6.2 性质组合的效果

| 组合 | 命名 | 典型例子 | 特殊定理 |
|------|------|----------|----------|
| **紧+T₂** | 紧Hausdorff | [0,1], Sⁿ, 紧Lie群 | Urysohn引理, Tietze延拓, 正规 |
| **局部紧+T₂** | 局部紧Hausdorff | ℝⁿ, 流形 | 单点紧致化, 测度论 |
| **连通+局部道路连通** | - | 流形, CW复形 | 复叠空间分类 |
| **单连通+局部紧** | - | ℝⁿ, S²(n≥2) | 万有复叠空间=自身 |
| **可度量化+完备** | Polish空间 | ℝⁿ, Baire空间 | Baire纲定理, 描述集合论 |
| **仿紧+光滑流形** | 好空间 | 所有光滑流形 | 单位分解, de Rham定理 |
