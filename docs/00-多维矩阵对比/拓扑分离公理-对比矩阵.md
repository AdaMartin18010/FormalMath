# 拓扑分离公理多维对比矩阵

**适用**: 点集拓扑基础  
**表征类型**: 多维矩阵对比

---

## 分离公理总表

| 公理 | 定义 | 等价条件 | 典型例子 | 反例 |
|------|------|----------|----------|------|
| **T₀** | 任意两点，存在开集包含其一而不含另一 | 不可区分点相等 | 实直线、Sierpinski空间 | 密着拓扑 |
| **T₁** | 任意两点，各自有开集不含另一点 | 单点集是闭集 | 实直线、余有限拓扑 | Sierpinski空间 |
| **T₂** (Hausdorff) | 任意两点，存在不相交开集分别包含它们 | 对角线是闭集 | 度量空间、流形 | 实直线模去有理数 |
| **T₂½** | 任意两点，存在闭包不相交的开集 | - | 正则+T₁ | - |
| **T₃** (正则) | T₁ + 点与闭集可用开集分离 | - | 度量空间 |  Zariski拓扑 |
| **T₃½** (完全正则) | T₁ + 点与闭集可用连续函数分离 | Urysohn引理前件 | 拓扑群 | - |
| **T₄** (正规) | T₁ + 不相交闭集可用开集分离 | Urysohn引理 | 紧Hausdorff | Zariski拓扑 |

---

## 蕴含关系

```
T₄ (正规)
│
├─ T₃½ (完全正则)
│
├─ T₃ (正则)
│
├─ T₂½
│
├─ T₂ (Hausdorff) ⭐最常用
│
├─ T₁
│
└─ T₀
```

## 度量空间的优势

**定理**: 度量空间满足所有分离公理（T₀到T₄）

| 性质 | 度量空间 | 一般拓扑空间 |
|------|----------|--------------|
| T₀ | ✅ | 不一定 |
| T₁ | ✅ | 不一定 |
| T₂ | ✅ | 不一定 |
| T₃ | ✅ | 不一定 |
| T₄ | ✅ | 不一定 |
