## 二、如何使用银层文档

### 推荐学习路径

1. **课前预习**: 阅读"核心定义"和"直观解释"部分，建立直觉
2. **课堂对照**: 用"严格陈述"对照教材/讲义，确认符号体系一致
3. **课后巩固**: 独立完成"典型例题"，然后对照解答
4. **自测**: 完成"习题"部分，使用"提示"但尽量不看"解答"
5. **纠偏**: 阅读"常见误区"，检查自己是否犯了同样的错误
6. **深化**: 如有 Lean4 代码，尝试理解形式化证明的结构

### 文档结构速查

每篇银层文档均包含以下板块：

```
YAML 元数据（课程对齐信息）
├── 核心定义（严格陈述 + 直观解释）
├── 核心定理与完整证明
│   └── 双语对照（Lean4 代码，如适用）
├── 典型例题（1–2 道）
├── 常见误区与纠正
├── 思想方法提炼
└── 习题（≥10 道，含完整解答）
```

## 三、学习建议

### 时间规划

| 课程 | 建议总学时 | 每章学时 | 习题比例 |
|------|-----------|---------|---------|
| MIT 18.06 | 60–80 小时 | 4–5 小时 | 40% |
| MIT 18.100A | 50–70 小时 | 6–8 小时 | 50% |
| MIT 18.701 | 60–80 小时 | 8–10 小时 | 45% |
| Harvard 232br | 按需 | 每题 1–2 小时 | 80% |
| Stanford FOAG | 按需 | 每章 8–12 小时 | 60% |

### 常见学习模式

- **考试冲刺模式**: 重点阅读"核心定理"+"典型例题"+"常见误区"，快速建立解题框架
- **系统学习模式**: 按章节顺序完整阅读，每章独立完成全部习题
- **查缺补漏模式**: 针对薄弱知识点，直接跳到对应章节，重点看"误区分析"
- **研究预备模式**: 重点关注金层专题（格洛腾迪克等）与银层的衔接部分
