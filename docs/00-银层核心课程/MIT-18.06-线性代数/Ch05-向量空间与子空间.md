# Ch.5 向量空间与子空间（Vector Spaces and Subspaces）

> **课程**: MIT 18.06 Linear Algebra | **章节**: Chapter 5
> **学习目标**: 严格掌握向量空间的公理化定义，理解子空间的判定准则，熟悉 $\mathbb{R}^n$ 中的典型子空间（特别是列空间），建立从"具体方程"到"抽象空间"的思维跃迁。

---

## 一、核心定义

### 定义 5.1（向量空间）

**严格陈述**: 设 $V$ 是一个非空集合，$\mathbb{F}$ 是一个域（通常 $\mathbb{F} = \mathbb{R}$ 或 $\mathbb{C}$）。在 $V$ 上定义两种运算：

- **加法**：对任意 $\mathbf{u}, \mathbf{v} \in V$，有 $\mathbf{u} + \mathbf{v} \in V$；
- **标量乘法**：对任意 $c \in \mathbb{F}$，$\mathbf{v} \in V$，有 $c\mathbf{v} \in V$。

若这两种运算满足以下八条公理，则称 $V$ 为 $\mathbb{F}$ 上的**向量空间**（vector space），$V$ 中的元素称为**向量**：

1. **加法交换律**：$\mathbf{u} + \mathbf{v} = \mathbf{v} + \mathbf{u}$；
2. **加法结合律**：$(\mathbf{u} + \mathbf{v}) + \mathbf{w} = \mathbf{u} + (\mathbf{v} + \mathbf{w})$；
3. **零向量存在**：存在 $\mathbf{0} \in V$，使得 $\mathbf{v} + \mathbf{0} = \mathbf{v}$ 对所有 $\mathbf{v} \in V$；
4. **负向量存在**：对每个 $\mathbf{v} \in V$，存在 $-\mathbf{v} \in V$ 使得 $\mathbf{v} + (-\mathbf{v}) = \mathbf{0}$；
5. **标量乘法与域乘法相容**：$c(d\mathbf{v}) = (cd)\mathbf{v}$；
6. **单位标量**：$1\mathbf{v} = \mathbf{v}$（其中 $1$ 是 $\mathbb{F}$ 的乘法单位元）；
7. **标量乘法对向量加法的分配律**：$c(\mathbf{u} + \mathbf{v}) = c\mathbf{u} + c\mathbf{v}$；
8. **标量乘法对域加法的分配律**：$(c + d)\mathbf{v} = c\mathbf{v} + d\mathbf{v}$。

**直观解释**: 向量空间是"可以相加和数乘，且运算规则与欧几里得向量一致"的抽象集合。八条公理看似繁琐，但每一条都排除了某种"病态"情形，确保代数运算的顺利进行。

```lean4
import Mathlib

-- 定义 5.1：向量空间的 Lean4 表示
-- Mathlib 使用 Module R V 来表示环 R 上的模（向量空间是域上的模）
-- 对于实向量空间，使用 Module ℝ V

-- ℝⁿ 作为标准向量空间：Fin n → ℝ 自动带有 Module ℝ 结构
example (n : ℕ) : Module ℝ (Fin n → ℝ) := by infer_instance

-- 验证八条公理中的关键几条（加法交换律、零向量存在性等）
open Module in
lemma vector_space_add_comm {V : Type} [AddCommGroup V] [Module ℝ V] (u v : V) :
    u + v = v + u := by
  -- 由 AddCommGroup 的定义直接得到
  exact add_comm u v

open Module in
lemma vector_space_zero_vec {V : Type} [AddCommGroup V] [Module ℝ V] (v : V) :
    v + 0 = v := by
  -- 由 AddCommGroup 的定义直接得到
  exact add_zero v
```

**严格陈述**: 设 $V$ 是 $\mathbb{F}$ 上的向量空间，$W \subseteq V$ 是 $V$ 的非空子集。若 $W$ 关于 $V$ 中定义的加法和标量乘法也构成向量空间，则称 $W$ 是 $V$ 的**子空间**（subspace）。

**等价判定准则**: $W \subseteq V$ 是子空间当且仅当满足以下三条：

1. **包含零向量**：$\mathbf{0} \in W$；
2. **对加法封闭**：若 $\mathbf{u}, \mathbf{v} \in W$，则 $\mathbf{u} + \mathbf{v} \in W$；
3. **对标量乘法封闭**：若 $c \in \mathbb{F}$，$\mathbf{v} \in W$，则 $c\mathbf{v} \in W$。

**直观解释**: 子空间是向量空间中的"自包含"子集——它足够大，包含零向量；又足够"平直"，对线性组合封闭。在 $\mathbb{R}^3$ 中，子空间包括：原点 $\{\mathbf{0}\}$、过原点的直线、过原点的平面、以及整个 $\mathbb{R}^3$。

---

### 定义 5.3（$\mathbb{R}^n$ 与标准向量空间）

**严格陈述**: $n$ 维实向量空间定义为

$$\mathbb{R}^n = \left\{ (x_1, x_2, \ldots, x_n)^T : x_i \in \mathbb{R} \right\}.$$

其上的加法和标量乘法按分量进行：

$$(x_1, \ldots, x_n)^T + (y_1, \ldots, y_n)^T = (x_1 + y_1, \ldots, x_n + y_n)^T,$$
$$c(x_1, \ldots, x_n)^T = (cx_1, \ldots, cx_n)^T.$$

**直观解释**: $\mathbb{R}^n$ 是我们最熟悉的向量空间，也是 MIT 18.06 的主要舞台。但向量空间的抽象定义允许更广泛的例子：多项式空间、函数空间、矩阵空间等。

---

### 定义 5.4（列空间）

**严格陈述**: 设 $A \in \mathbb{R}^{m \times n}$，其列向量为 $\mathbf{a}_1, \mathbf{a}_2, \ldots, \mathbf{a}_n \in \mathbb{R}^m$。$A$ 的**列空间**（column space）定义为这些列向量的所有线性组合构成的集合：

$$C(A) = \operatorname{span}\{\mathbf{a}_1, \ldots, \mathbf{a}_n\} = \{ x_1\mathbf{a}_1 + \cdots + x_n\mathbf{a}_n : x_i \in \mathbb{R} \} = \{ A\mathbf{x} : \mathbf{x} \in \mathbb{R}^n \}.$$

---

### 定义 5.5（零空间）

**严格陈述**: 设 $A \in \mathbb{R}^{m \times n}$。$A$ 的**零空间**（null space / kernel）定义为齐次方程组 $A\mathbf{x} = \mathbf{0}$ 的解集：

$$N(A) = \{ \mathbf{x} \in \mathbb{R}^n : A\mathbf{x} = \mathbf{0} \}.$$

---

## 二、核心定理与完整证明

### 定理 5.1（子空间判定准则）

**定理陈述**: 设 $V$ 是向量空间，$W \subseteq V$ 非空。则 $W$ 是 $V$ 的子空间当且仅当 $W$ 对线性组合封闭：对任意 $\mathbf{u}, \mathbf{v} \in W$ 和任意标量 $c, d$，有 $c\mathbf{u} + d\mathbf{v} \in W$。

**证明**:

**必要性**（$\Rightarrow$）：设 $W$ 是子空间。由定义，$W$ 对加法和标量乘法分别封闭。因此若 $\mathbf{u}, \mathbf{v} \in W$，则 $c\mathbf{u} \in W$，$d\mathbf{v} \in W$，进而 $c\mathbf{u} + d\mathbf{v} \in W$。

**充分性**（$\Leftarrow$）：设 $W$ 对任意线性组合封闭。

- **零向量**：取 $c = d = 0$，则 $0\mathbf{u} + 0\mathbf{v} = \mathbf{0} \in W$。
- **加法封闭**：取 $c = d = 1$，则 $\mathbf{u} + \mathbf{v} \in W$。
- **标量乘法封闭**：取 $d = 0$，则 $c\mathbf{u} \in W$。

因此 $W$ 满足子空间的三条判定准则，是 $V$ 的子空间。$\square$

**证明要点提示**: 子空间判定的核心是"一条准则替代三条"——对线性组合封闭等价于分别对加法和标量乘法封闭。这在实际验证中往往更便利。

```lean4
-- 定义 5.2：子空间的 Lean4 表示
-- Mathlib 使用 Submodule R V 表示 V 的 R-子模（即子空间）
-- Submodule 已经封装了包含零向量、对加法封闭、对标量乘法封闭的三条准则

open Submodule in
-- 子空间判定准则：W 是子空间 ⟺ W 对任意线性组合封闭
-- 在 Mathlib 中，Submodule 的 carrier 集合自动满足此性质
theorem subspace_criterion {V : Type} [AddCommGroup V] [Module ℝ V]
    (W : Submodule ℝ V) (u v : V) (c d : ℝ)
    (hu : u ∈ W) (hv : v ∈ W) :
    c • u + d • v ∈ W := by
  -- 证明：Submodule 对加法和标量乘法分别封闭
  apply W.add_mem
  · exact W.smul_mem c hu
  · exact W.smul_mem d hv

-- 示例：ℝ³ 中平面 W = {(x,y,z) : x + y + z = 0} 是子空间
-- 它可以看作矩阵 A = [1 1 1] 的零空间
def plane_subspace : Submodule ℝ (Fin 3 → ℝ) where
  carrier := {v | v 0 + v 1 + v 2 = 0}
  zero_mem' := by simp
  add_mem' := by
    intro a b ha hb
    simp at ha hb ⊢
    linarith
  smul_mem' := by
    intro c a ha
    simp at ha ⊢
    linarith
```

**定理陈述**: 设 $A \in \mathbb{R}^{m \times n}$，则 $C(A)$ 是 $\mathbb{R}^m$ 的子空间。

**证明**:

验证子空间的三条准则：

1. **零向量**：$\mathbf{0} = A\mathbf{0} \in C(A)$（取 $\mathbf{x} = \mathbf{0}$）。

2. **加法封闭**：设 $\mathbf{b}_1, \mathbf{b}_2 \in C(A)$。则存在 $\mathbf{x}_1, \mathbf{x}_2 \in \mathbb{R}^n$ 使得 $\mathbf{b}_1 = A\mathbf{x}_1$，$\mathbf{b}_2 = A\mathbf{x}_2$。

   则 $\mathbf{b}_1 + \mathbf{b}_2 = A\mathbf{x}_1 + A\mathbf{x}_2 = A(\mathbf{x}_1 + \mathbf{x}_2)$。由于 $\mathbf{x}_1 + \mathbf{x}_2 \in \mathbb{R}^n$，故 $\mathbf{b}_1 + \mathbf{b}_2 \in C(A)$。

3. **标量乘法封闭**：设 $\mathbf{b} \in C(A)$，$c \in \mathbb{R}$。则存在 $\mathbf{x}$ 使得 $\mathbf{b} = A\mathbf{x}$。

   则 $c\mathbf{b} = cA\mathbf{x} = A(c\mathbf{x})$。由于 $c\mathbf{x} \in \mathbb{R}^n$，故 $c\mathbf{b} \in C(A)$。

因此 $C(A)$ 是 $\mathbb{R}^m$ 的子空间。$\square$

```lean4
-- 定义 5.4：列空间的 Lean4 表示
-- 列空间 = A 的列向量的张成空间 = {A * x | x ∈ ℝⁿ}
-- 在 Mathlib 中对应 LinearMap.range (A.mulVec)

open Matrix in
-- A 的列空间是 ℝᵐ 的子空间
theorem colSpace_is_subspace {m n : ℕ} (A : Matrix (Fin m) (Fin n) ℝ) :
    ∃ W : Submodule ℝ (Fin m → ℝ), ∀ b, b ∈ W ↔ ∃ x, A.mulVec x = b := by
  -- 证明思路：列空间 = LinearMap.range 的 carrier
  -- LinearMap.range 自动是子空间
  let f := A.mulVecLinear
  use LinearMap.range f
  intro b
  simp [f, LinearMap.mem_range]

-- 定义 5.5：零空间的 Lean4 表示
-- 零空间 = {x | A * x = 0} = LinearMap.ker (A.mulVec)
open Matrix in
theorem nullSpace_is_subspace {m n : ℕ} (A : Matrix (Fin m) (Fin n) ℝ) :
    ∃ W : Submodule ℝ (Fin n → ℝ), ∀ x, x ∈ W ↔ A.mulVec x = 0 := by
  -- 证明思路：零空间 = LinearMap.ker 的 carrier
  -- LinearMap.ker 自动是子空间
  let f := A.mulVecLinear
  use LinearMap.ker f
  intro x
  simp [f, LinearMap.mem_ker]
```

**定理陈述**: 设 $A \in \mathbb{R}^{m \times n}$，则 $N(A)$ 是 $\mathbb{R}^n$ 的子空间。

**证明**:

1. **零向量**：$A\mathbf{0} = \mathbf{0}$，故 $\mathbf{0} \in N(A)$。

2. **加法封闭**：设 $\mathbf{x}_1, \mathbf{x}_2 \in N(A)$，即 $A\mathbf{x}_1 = \mathbf{0}$，$A\mathbf{x}_2 = \mathbf{0}$。

   则 $A(\mathbf{x}_1 + \mathbf{x}_2) = A\mathbf{x}_1 + A\mathbf{x}_2 = \mathbf{0} + \mathbf{0} = \mathbf{0}$，故 $\mathbf{x}_1 + \mathbf{x}_2 \in N(A)$。

3. **标量乘法封闭**：设 $\mathbf{x} \in N(A)$，$c \in \mathbb{R}$。

   则 $A(c\mathbf{x}) = c(A\mathbf{x}) = c\mathbf{0} = \mathbf{0}$，故 $c\mathbf{x} \in N(A)$。

因此 $N(A)$ 是 $\mathbb{R}^n$ 的子空间。$\square$

```lean4
-- 综合示例：验证列空间对线性组合封闭（对应定理 5.2）
open Matrix in
example {m n : ℕ} (A : Matrix (Fin m) (Fin n) ℝ)
    (b₁ b₂ : Fin m → ℝ) (c d : ℝ)
    (hb₁ : ∃ x₁, A.mulVec x₁ = b₁)
    (hb₂ : ∃ x₂, A.mulVec x₂ = b₂) :
    ∃ x, A.mulVec x = c • b₁ + d • b₂ := by
  -- 证明：若 b₁ = A*x₁, b₂ = A*x₂，则 c*b₁ + d*b₂ = A*(c*x₁ + d*x₂)
  rcases hb₁ with ⟨x₁, hx₁⟩
  rcases hb₂ with ⟨x₂, hx₂⟩
  use c • x₁ + d • x₂
  rw [map_add, map_smul, map_smul, hx₁, hx₂]
  -- 这里使用了 LinearMap 的性质：A.mulVecLinear 自动保持加法和数乘
  all_goals try simp [Matrix.mulVecLinear]

-- 综合示例：验证零空间对加法封闭（对应定理 5.3）
open Matrix in
example {m n : ℕ} (A : Matrix (Fin m) (Fin n) ℝ)
    (x₁ x₂ : Fin n → ℝ)
    (h₁ : A.mulVec x₁ = 0)
    (h₂ : A.mulVec x₂ = 0) :
    A.mulVec (x₁ + x₂) = 0 := by
  -- 证明：A*(x₁ + x₂) = A*x₁ + A*x₂ = 0 + 0 = 0
  rw [map_add]
  rw [h₁, h₂]
  simp
```

### 例题 5.1（判断给定集合是否为子空间）

**题目**: 判断下列 $W \subseteq \mathbb{R}^3$ 是否为子空间：

(a) $W = \{ (x, y, z)^T : x + y + z = 0 \}$

(b) $W = \{ (x, y, z)^T : x + y + z = 1 \}$

(c) $W = \{ (x, y, z)^T : x \geq 0 \}$

**解答**:

**(a)** $W$ 是过原点的平面。验证三条准则：

- **零向量**：$0 + 0 + 0 = 0$，故 $\mathbf{0} \in W$。✓
- **加法封闭**：设 $\mathbf{u} = (u_1, u_2, u_3)^T$，$\mathbf{v} = (v_1, v_2, v_3)^T \in W$。则 $u_1 + u_2 + u_3 = 0$，$v_1 + v_2 + v_3 = 0$。

  $(\mathbf{u} + \mathbf{v})_1 + (\mathbf{u} + \mathbf{v})_2 + (\mathbf{u} + \mathbf{v})_3 = (u_1 + v_1) + (u_2 + v_2) + (u_3 + v_3) = 0 + 0 = 0$。

  故 $\mathbf{u} + \mathbf{v} \in W$。✓

- **标量乘法封闭**：设 $\mathbf{u} \in W$，$c \in \mathbb{R}$。则 $cu_1 + cu_2 + cu_3 = c(u_1 + u_2 + u_3) = c \cdot 0 = 0$。

  故 $c\mathbf{u} \in W$。✓

**$W$ 是子空间**。

注意到 $W = N(A)$ 其中 $A = \begin{pmatrix} 1 & 1 & 1 \end{pmatrix}$，因此也可用定理 5.3 直接得出。

**(b)** $W$ 是不过原点的平面（$x + y + z = 1$）。

- **零向量**：$0 + 0 + 0 = 0 \neq 1$，故 $\mathbf{0} \notin W$。

不满足子空间条件。**$W$ 不是子空间**。

**(c)** $W$ 是右半空间（包含 $yz$-平面）。

- **零向量**：$0 \geq 0$，故 $\mathbf{0} \in W$。✓
- **加法封闭**：设 $\mathbf{u} = (1, 0, 0)^T \in W$（$1 \geq 0$），$\mathbf{v} = (1, 0, 0)^T \in W$。则 $\mathbf{u} + \mathbf{v} = (2, 0, 0)^T \in W$。这没问题，但...
- **标量乘法封闭**：取 $\mathbf{u} = (1, 0, 0)^T \in W$，$c = -1$。则 $c\mathbf{u} = (-1, 0, 0)^T$，而 $-1 < 0$，故 $c\mathbf{u} \notin W$。

不满足标量乘法封闭。**$W$ 不是子空间**。

**关键步骤解析**: 判断子空间时，首先检查零向量是最快捷的否决法（如 (b)）。其次检查标量乘法封闭往往能找到反例（如 (c) 中取负标量）。

### 例题 5.2（列空间的具体描述）

**题目**: 设 $A = \begin{pmatrix} 1 & 2 & 3 \\ 2 & 4 & 5 \\ 3 & 6 & 8 \end{pmatrix}$。描述 $C(A)$ 并判断 $\mathbf{b} = \begin{pmatrix} 6 \\ 11 \\ 17 \end{pmatrix}$ 是否属于 $C(A)$。

**解答**:

将 $A$ 化为行阶梯形以确定主元列：

$R_2 \to R_2 - 2R_1$，$R_3 \to R_3 - 3R_1$：

$$\begin{pmatrix} 1 & 2 & 3 \\ 0 & 0 & -1 \\ 0 & 0 & -1 \end{pmatrix}$$

$R_3 \to R_3 - R_2$：

$$\begin{pmatrix} 1 & 2 & 3 \\ 0 & 0 & -1 \\ 0 & 0 & 0 \end{pmatrix}$$

主元列为第 1 列和第 3 列。因此 $C(A)$ 由 $A$ 的第 1 列和第 3 列张成：

$$C(A) = \operatorname{span}\left\{ \begin{pmatrix} 1 \\ 2 \\ 3 \end{pmatrix}, \begin{pmatrix} 3 \\ 5 \\ 8 \end{pmatrix} \right\}.$$

注意第 2 列 $\begin{pmatrix} 2 \\ 4 \\ 6 \end{pmatrix} = 2 \begin{pmatrix} 1 \\ 2 \\ 3 \end{pmatrix}$ 是第 1 列的倍数，属于 $C(A)$ 但不增加新的维度。

判断 $\mathbf{b} = \begin{pmatrix} 6 \\ 11 \\ 17 \end{pmatrix} \in C(A)$：

即判断是否存在 $c_1, c_3$ 使得 $c_1 \begin{pmatrix} 1 \\ 2 \\ 3 \end{pmatrix} + c_3 \begin{pmatrix} 3 \\ 5 \\ 8 \end{pmatrix} = \begin{pmatrix} 6 \\ 11 \\ 17 \end{pmatrix}$。

从前两个分量：$c_1 + 3c_3 = 6$，$2c_1 + 5c_3 = 11$。

第一式乘 2：$2c_1 + 6c_3 = 12$。减去第二式：$c_3 = 1$。于是 $c_1 = 3$。

验证第三个分量：$3(3) + 1(8) = 9 + 8 = 17$ ✓。

因此 $\mathbf{b} = 3\mathbf{a}_1 + 1\mathbf{a}_3 \in C(A)$。$\square$

## 四、常见误区与纠正

### 误区 5.1: "$\mathbb{R}^2$ 是 $\mathbb{R}^3$ 的子空间"

**错误观点**: 初学者常认为平面 $\mathbb{R}^2$ "自然包含于"三维空间 $\mathbb{R}^3$ 中，因此是 $\mathbb{R}^3$ 的子空间。

**纠正**:

严格来说，$\mathbb{R}^2 = \{ (x, y)^T : x, y \in \mathbb{R} \}$ 与 $\mathbb{R}^3 = \{ (x, y, z)^T : x, y, z \in \mathbb{R} \}$ 的元素类型不同——一个是二元组，一个是三元组。因此 $\mathbb{R}^2$ **不是** $\mathbb{R}^3$ 的子集，更谈不上子空间。

**正确的说法**是：

- 集合 $W = \{ (x, y, 0)^T : x, y \in \mathbb{R} \}$（即 $xy$-平面）**是** $\mathbb{R}^3$ 的子空间；
- 集合 $W = \{ (x, y, z)^T : z = 0 \}$ **是** $\mathbb{R}^3$ 的子空间，且与 $\mathbb{R}^2$ "同构"（isomorphic）；
- 但 $\mathbb{R}^2$ 本身不是 $\mathbb{R}^3$ 的子空间。

**为什么错**: 混淆了"同构"与"包含"。$\mathbb{R}^2$ 与 $xy$-平面在结构上是"一样"的（同构），但它们作为集合并不相等。子空间要求是子集，因此必须是形如 $(x, y, 0)^T$ 的三元组集合。

**正确理解**:

- $\mathbb{R}^n$ 的子空间必须是由 $n$ 元组构成的集合；
- $\mathbb{R}^3$ 中与 $\mathbb{R}^2$ "对应"的子空间是过原点的平面（如 $z = 0$、$x + y + z = 0$ 等）；
- 不过原点的平面（如 $z = 1$）不是子空间，因为它不包含零向量。

## 五、思想方法提炼

**本章核心思想**：

1. **从方程到空间：视角的升维**: Ch.1–4 关注"如何求解 $A\mathbf{x} = \mathbf{b}$"，Ch.5 则将问题提升为"$\mathbf{b}$ 落在 $A$ 的列空间的什么位置"。这种从"点"到"空间"的视角跃迁，是线性代数从计算走向结构理论的关键一步。

2. **子空间 = "平直 + 过原点"**: 子空间判定的核心直觉是"过原点的直线/平面/超平面"。任何不过原点的集合都不是子空间（因为缺少零向量），任何"弯曲"的集合（如球面）也不是子空间（因为对线性组合不封闭）。

3. **四个基本子空间的萌芽**: 本章定义的 $C(A)$ 和 $N(A)$ 是后续"四大基本子空间"（列空间、零空间、行空间、左零空间）的核心成员。理解它们之间的关系，是掌握线性代数深层结构的关键。

**与后续章节的联系**：

- Ch.6 将系统研究 $C(A)$ 和 $N(A)$ 的维数（秩与零化度）；
- Ch.8 将引入"基"和"维数"的严格概念，从而可以谈论 $C(A)$ 的维数 = $A$ 的秩；
- Ch.9 将完成四大基本子空间的拼图，并揭示它们之间的正交关系。

### 习题 5.1

**题目**: 验证 $\mathbb{R}^n$ 满足向量空间的所有八条公理。

**解答**:

设 $\mathbf{u}, \mathbf{v}, \mathbf{w} \in \mathbb{R}^n$，$c, d \in \mathbb{R}$。

1. **加法交换律**：$(\mathbf{u} + \mathbf{v})_i = u_i + v_i = v_i + u_i = (\mathbf{v} + \mathbf{u})_i$。✓
2. **加法结合律**：$((\mathbf{u} + \mathbf{v}) + \mathbf{w})_i = (u_i + v_i) + w_i = u_i + (v_i + w_i) = (\mathbf{u} + (\mathbf{v} + \mathbf{w}))_i$。✓
3. **零向量**：$\mathbf{0} = (0, \ldots, 0)^T$，$\mathbf{u} + \mathbf{0} = \mathbf{u}$。✓
4. **负向量**：$-\mathbf{u} = (-u_1, \ldots, -u_n)^T$，$\mathbf{u} + (-\mathbf{u}) = \mathbf{0}$。✓
5. **标量乘法与域乘法相容**：$c(d\mathbf{u})_i = c(du_i) = (cd)u_i = ((cd)\mathbf{u})_i$。✓
6. **单位标量**：$(1\mathbf{u})_i = 1 \cdot u_i = u_i$。✓
7. **对向量加法分配**：$(c(\mathbf{u} + \mathbf{v}))_i = c(u_i + v_i) = cu_i + cv_i = (c\mathbf{u} + c\mathbf{v})_i$。✓
8. **对域加法分配**：$((c+d)\mathbf{u})_i = (c+d)u_i = cu_i + du_i = (c\mathbf{u} + d\mathbf{u})_i$。✓ $\square$

### 习题 5.2

**题目**: 判断下列集合是否为 $\mathbb{R}^3$ 的子空间：

(a) $W_1 = \{ (x, y, z)^T : x = y = z \}$

(b) $W_2 = \{ (x, y, z)^T : x^2 + y^2 + z^2 \leq 1 \}$

(c) $W_3 = \{ (x, y, z)^T : x + 2y = 3z \}$

**解答**:

**(a)** 是子空间。$W_1$ 是过原点的直线（方向向量 $(1,1,1)^T$）。验证：

- $\mathbf{0} = (0,0,0)^T \in W_1$；
- 若 $(t,t,t), (s,s,s) \in W_1$，则 $(t+s, t+s, t+s) \in W_1$；
- 若 $(t,t,t) \in W_1$，$c \in \mathbb{R}$，则 $(ct, ct, ct) \in W_1$。

**(b)** 不是子空间。单位球虽然包含原点，但对标量乘法不封闭：$(1,0,0)^T \in W_2$，但 $2(1,0,0)^T = (2,0,0)^T \notin W_2$（因为 $4 > 1$）。

**(c)** 是子空间。可写成 $x + 2y - 3z = 0$，即 $N(A)$ 其中 $A = \begin{pmatrix} 1 & 2 & -3 \end{pmatrix}$。或验证：

- $\mathbf{0}$ 满足 $0 + 0 = 0$；
- 若 $x_1 + 2y_1 = 3z_1$，$x_2 + 2y_2 = 3z_2$，则 $(x_1+x_2) + 2(y_1+y_2) = 3(z_1+z_2)$；
- 若 $x + 2y = 3z$，则 $cx + 2(cy) = c(x+2y) = 3(cz)$。$\square$

### 习题 5.3

**题目**: 设 $A = \begin{pmatrix} 1 & 2 & 3 \\ 2 & 4 & 6 \end{pmatrix}$。描述 $C(A)$ 并求其一组生成元。

**解答**:

$A$ 的列向量为 $\mathbf{a}_1 = \begin{pmatrix}1\\2\end{pmatrix}$，$\mathbf{a}_2 = \begin{pmatrix}2\\4\end{pmatrix}$，$\mathbf{a}_3 = \begin{pmatrix}3\\6\end{pmatrix}$。

注意到 $\mathbf{a}_2 = 2\mathbf{a}_1$，$\mathbf{a}_3 = 3\mathbf{a}_1$。

因此 $C(A) = \operatorname{span}\{\mathbf{a}_1, \mathbf{a}_2, \mathbf{a}_3\} = \operatorname{span}\{\mathbf{a}_1\} = \left\{ t\begin{pmatrix}1\\2\end{pmatrix} : t \in \mathbb{R} \right\}$。

这是 $\mathbb{R}^2$ 中过原点、方向为 $(1,2)^T$ 的**直线**。$\square$

### 习题 5.4

**题目**: 设 $A \in \mathbb{R}^{m \times n}$，$\mathbf{b} \in \mathbb{R}^m$。证明：$A\mathbf{x} = \mathbf{b}$ 有解当且仅当 $\mathbf{b} \in C(A)$。

**解答**:

这正是 Ch.1 定理 1.1 的重述。$A\mathbf{x} = \mathbf{b}$ 有解 $\Leftrightarrow$ 存在 $\mathbf{x}$ 使得 $A\mathbf{x} = \mathbf{b}$ $\Leftrightarrow$ $\mathbf{b}$ 可表示为 $A$ 的列的线性组合 $\Leftrightarrow$ $\mathbf{b} \in C(A)$。$\square$

### 习题 5.5

**题目**: 设 $W_1, W_2$ 是向量空间 $V$ 的两个子空间。证明 $W_1 \cap W_2$ 也是 $V$ 的子空间。

**解答**:

- **零向量**：$\mathbf{0} \in W_1$ 且 $\mathbf{0} \in W_2$，故 $\mathbf{0} \in W_1 \cap W_2$。
- **加法封闭**：设 $\mathbf{u}, \mathbf{v} \in W_1 \cap W_2$。则 $\mathbf{u}, \mathbf{v} \in W_1$，故 $\mathbf{u} + \mathbf{v} \in W_1$；同理 $\mathbf{u} + \mathbf{v} \in W_2$。因此 $\mathbf{u} + \mathbf{v} \in W_1 \cap W_2$。
- **标量乘法封闭**：设 $\mathbf{v} \in W_1 \cap W_2$，$c \in \mathbb{F}$。则 $\mathbf{v} \in W_1$，故 $c\mathbf{v} \in W_1$；同理 $c\mathbf{v} \in W_2$。因此 $c\mathbf{v} \in W_1 \cap W_2$。

$W_1 \cap W_2$ 是子空间。$\square$

### 习题 5.6

**题目**: 设 $A = \begin{pmatrix} 1 & 2 \\ 3 & 6 \end{pmatrix}$。求 $N(A)$ 并描述其几何意义。

**解答**:

$A\mathbf{x} = \mathbf{0}$ 即

$$\begin{cases} x + 2y = 0 \\ 3x + 6y = 0 \end{cases}$$

第二式是第一式的 3 倍。由第一式：$x = -2y$。令 $y = t$，则 $\mathbf{x} = t\begin{pmatrix}-2\\1\end{pmatrix}$。

$$N(A) = \left\{ t\begin{pmatrix}-2\\1\end{pmatrix} : t \in \mathbb{R} \right\}.$$

几何上，$N(A)$ 是 $\mathbb{R}^2$ 中过原点、方向为 $(-2, 1)^T$ 的**直线**。$\square$

### 习题 5.7

**题目**: 设 $V$ 为所有 $2 \times 2$ 实矩阵的集合。验证 $V$ 是向量空间（指出零向量和负向量）。

**解答**:

$V = \mathbb{R}^{2 \times 2}$。加法和标量乘法按分量进行。

- 零向量：$\mathbf{0} = \begin{pmatrix}0&0\\0&0\end{pmatrix}$。
- 负向量：$-\begin{pmatrix}a&b\\c&d\end{pmatrix} = \begin{pmatrix}-a&-b\\-c&-d\end{pmatrix}$。

八条公理均按分量满足（因每个分量都满足 $\mathbb{R}$ 上的对应性质）。$\square$

### 习题 5.8

**题目**: 设 $W$ 为所有 $2 \times 2$ 对称矩阵的集合。证明 $W$ 是 $\mathbb{R}^{2 \times 2}$ 的子空间。

**解答**:

$W = \left\{ \begin{pmatrix}a&b\\b&d\end{pmatrix} : a, b, d \in \mathbb{R} \right\}$。

- 零矩阵 $\begin{pmatrix}0&0\\0&0\end{pmatrix}$ 对称，$\in W$。
- 若 $A, B \in W$，则 $A^T = A$，$B^T = B$。$(A+B)^T = A^T + B^T = A + B$，故 $A+B \in W$。
- 若 $A \in W$，$c \in \mathbb{R}$，则 $(cA)^T = cA^T = cA$，故 $cA \in W$。

$W$ 是子空间。$\square$

### 习题 5.9

**题目**: 设 $A \in \mathbb{R}^{m \times n}$，$B \in \mathbb{R}^{n \times p}$。证明：$C(AB) \subseteq C(A)$。

**解答**:

设 $\mathbf{y} \in C(AB)$。则存在 $\mathbf{x} \in \mathbb{R}^p$ 使得 $\mathbf{y} = (AB)\mathbf{x} = A(B\mathbf{x})$。

令 $\mathbf{z} = B\mathbf{x} \in \mathbb{R}^n$。则 $\mathbf{y} = A\mathbf{z}$，故 $\mathbf{y} \in C(A)$。

因此 $C(AB) \subseteq C(A)$。$\square$

### 习题 5.10

**题目**: 设 $A = \begin{pmatrix} 1 & 2 & 1 \\ 2 & 4 & 3 \\ 3 & 6 & 4 \end{pmatrix}$。

(a) 求 $C(A)$ 的一组生成元；

(b) 求 $N(A)$ 并描述其几何意义。

**解答**:

(a) 消元：$R_2 \to R_2 - 2R_1$，$R_3 \to R_3 - 3R_1$：

$$\begin{pmatrix} 1 & 2 & 1 \\ 0 & 0 & 1 \\ 0 & 0 & 1 \end{pmatrix} \xrightarrow{R_3 - R_2} \begin{pmatrix} 1 & 2 & 1 \\ 0 & 0 & 1 \\ 0 & 0 & 0 \end{pmatrix}$$

主元列为第 1 列和第 3 列。故 $C(A) = \operatorname{span}\left\{\begin{pmatrix}1\\2\\3\end{pmatrix}, \begin{pmatrix}1\\3\\4\end{pmatrix}\right\}$，是 $\mathbb{R}^3$ 中过原点的一个平面。

(b) $A\mathbf{x} = \mathbf{0}$：

由行阶梯形：$x_3 = 0$，$x_1 + 2x_2 = 0 \Rightarrow x_1 = -2x_2$。令 $x_2 = t$，则 $\mathbf{x} = t\begin{pmatrix}-2\\1\\0\end{pmatrix}$。

$$N(A) = \left\{ t\begin{pmatrix}-2\\1\\0\end{pmatrix} : t \in \mathbb{R} \right\}.$$

几何上，$N(A)$ 是 $\mathbb{R}^3$ 中过原点、方向为 $(-2, 1, 0)^T$ 的**直线**。$\square$

### 习题 5.11

**题目**: 设 $W_1, W_2$ 是 $V$ 的子空间。$W_1 \cup W_2$ 是否一定是子空间？证明或举反例。

**解答**:

**不一定**。反例：在 $\mathbb{R}^2$ 中，设 $W_1 = \operatorname{span}\{(1,0)^T\}$（$x$-轴），$W_2 = \operatorname{span}\{(0,1)^T\}$（$y$-轴）。

$W_1 \cup W_2$ 包含 $(1,0)^T$ 和 $(0,1)^T$，但不包含 $(1,0)^T + (0,1)^T = (1,1)^T$（因为 $(1,1)^T$ 不在任何一轴上）。

因此 $W_1 \cup W_2$ 对加法不封闭，不是子空间。

**注**: $W_1 \cup W_2$ 是子空间当且仅当 $W_1 \subseteq W_2$ 或 $W_2 \subseteq W_1$。$\square$

### 习题 5.12

**题目**: 设 $\mathcal{P}_2$ 为所有次数不超过 2 的实系数多项式构成的集合：$\mathcal{P}_2 = \{ at^2 + bt + c : a, b, c \in \mathbb{R} \}$。验证 $\mathcal{P}_2$ 是向量空间，并判断 $W = \{ p \in \mathcal{P}_2 : p(1) = 0 \}$ 是否为其子空间。

**解答**:

**$\mathcal{P}_2$ 是向量空间**：

多项式的加法和标量乘法按通常方式定义。八条公理均满足（因实数的加法和乘法满足对应性质）。零向量是零多项式 $0t^2 + 0t + 0 = 0$。

**$W$ 是子空间**：

- 零多项式在 $t = 1$ 处取值为 0，故 $\mathbf{0} \in W$。
- 若 $p, q \in W$，则 $p(1) = 0$，$q(1) = 0$。$(p+q)(1) = p(1) + q(1) = 0 + 0 = 0$，故 $p+q \in W$。
- 若 $p \in W$，$c \in \mathbb{R}$，则 $(cp)(1) = c \cdot p(1) = c \cdot 0 = 0$，故 $cp \in W$。

因此 $W$ 是 $\mathcal{P}_2$ 的子空间。

**注**: $W = \{ a(t-1)(t-b) : a, b \in \mathbb{R} \}$ 的更直接描述是 $W = \operatorname{span}\{t-1, t^2-1\}$，维数为 2。$\square$
