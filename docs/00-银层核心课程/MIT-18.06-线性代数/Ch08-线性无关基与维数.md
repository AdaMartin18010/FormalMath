# Ch.8 线性无关、基与维数（Independence, Basis, Dimension）

> **课程**: MIT 18.06 Linear Algebra | **章节**: Chapter 8
> **学习目标**: 严格掌握线性无关、基、维数三大核心概念，理解维数定理的深刻含义，熟练进行基的判定与坐标计算。

---

## 一、核心定义

### 定义 8.1（线性无关）

**严格陈述**: 设 $V$ 是 $\mathbb{F}$ 上的向量空间，$\mathbf{v}_1, \mathbf{v}_2, \ldots, \mathbf{v}_k \in V$。称这组向量**线性无关**（linearly independent），如果方程

$$c_1 \mathbf{v}_1 + c_2 \mathbf{v}_2 + \cdots + c_k \mathbf{v}_k = \mathbf{0}$$

的唯一解是 $c_1 = c_2 = \cdots = c_k = 0$（平凡解）。

若存在不全为零的标量 $c_1, \ldots, c_k$ 使上式成立，则称这组向量**线性相关**（linearly dependent）。

**直观解释**: 线性无关意味着"没有一个向量是多余的"——每个向量都指向一个其他向量无法补偿的方向。线性相关则意味着至少有一个向量可以"被其他人表示"，从而"冗余"。

```lean4
import Mathlib

-- **定义 8.1**：线性无关的 Lean4 表示
-- Mathlib 使用 LinearIndependent R v 来表示族 v 是 R-线性无关的
-- 其中 v : ι → V 是从指标集 ι 到向量空间 V 的族

open Module

-- 线性无关的定义：∑ cᵢ • vᵢ = 0 ⟹ 所有 cᵢ = 0
-- Mathlib 中 LinearIndependent 的定义与此等价
theorem linear_independent_def {V : Type} [AddCommGroup V] [Module ℝ V]
    {ι : Type} {v : ι → V} :
    LinearIndependent ℝ v ↔
    ∀ (s : Finset ι) (c : ι → ℝ), ∑ i ∈ s, c i • v i = 0 → ∀ i ∈ s, c i = 0 := by
  -- 证明：这是 Mathlib 中 LinearIndependent 的等价刻画
  simp [LinearIndependent, Finsupp.linearCombination_eq_zero_iff]
  -- 完整证明需展开 Finsupp 的定义，此处展示核心逻辑
  sorry

-- 示例：ℝ³ 中标准基向量 e₁, e₂, e₃ 线性无关
example : LinearIndependent ℝ (fun i : Fin 3 => EuclideanSpace.basisFun (Fin 3) ℝ i) := by
  -- 标准基自动线性无关
  exact EuclideanSpace.instBasisFun (Fin 3) ℝ |>.linearIndependent
```

**严格陈述**: 设 $V$ 是向量空间，$\mathcal{B} = \{\mathbf{v}_1, \ldots, \mathbf{v}_k\} \subseteq V$。称 $\mathcal{B}$ 是 $V$ 的一组**基**（basis），如果：

1. $\mathcal{B}$ 线性无关；
2. $\mathcal{B}$ 张成 $V$（即 $\operatorname{span}(\mathcal{B}) = V$）。

**直观解释**: 基是向量空间的"最小生成集"——它刚好足够张成全空间，没有任何冗余元素。同时，基也是"最大线性无关集"——再加入任何向量都会破坏线性无关性。

```lean4
-- **定义 8.2**：基的 Lean4 表示
-- Mathlib 使用 Basis ι R V 表示 V 作为 R-模的一组基
-- 其中 ι 是指标集， Basis ι R V 蕴含了线性无关和张成性

open Module

-- Basis 的两个核心性质
-- 1. 线性无关性
theorem basis_linear_independent {V : Type} [AddCommGroup V] [Module ℝ V]
    {ι : Type} (b : Basis ι ℝ V) :
    LinearIndependent ℝ b := by
  exact b.linearIndependent

-- 2. 张成性（Span = ⊤，即全空间）
theorem basis_span {V : Type} [AddCommGroup V] [Module ℝ V]
    {ι : Type} (b : Basis ι ℝ V) :
    Submodule.span ℝ (Set.range b) = ⊤ := by
  exact b.span_eq

-- 示例：ℝ³ 的标准基
example : Basis (Fin 3) ℝ (Fin 3 → ℝ) := by
  -- Pi.basisFun 构造了函数空间的“标准基”
  exact Pi.basisFun (Fin 3) ℝ
```

**严格陈述**: 设 $V$ 是向量空间。若 $V$ 存在由有限个向量构成的基，则称 $V$ 是**有限维的**（finite-dimensional）。此时，$V$ 的**维数**（dimension）定义为其任意一组基所含向量的个数，记为 $\dim V$。

**注**: 我们将在**定理 8.2** 中证明，同一向量空间的所有基大小相同，因此维数是良定义的。

**直观解释**: 维数是向量空间"自由度"的度量。$\mathbb{R}^3$ 的维数为 3，因为需要 3 个独立方向才能描述空间中的任意向量；一条直线的维数为 1；一个平面的维数为 2。

---

### 定义 8.4（坐标）

**严格陈述**: 设 $\mathcal{B} = \{\mathbf{v}_1, \ldots, \mathbf{v}_n\}$ 是向量空间 $V$ 的一组基。对任意 $\mathbf{v} \in V$，存在唯一的标量 $c_1, \ldots, c_n \in \mathbb{F}$ 使得

$$\mathbf{v} = c_1 \mathbf{v}_1 + \cdots + c_n \mathbf{v}_n.$$

称列向量 $[\mathbf{v}]_\mathcal{B} = \begin{pmatrix} c_1 \\ \vdots \\ c_n \end{pmatrix}$ 为 $\mathbf{v}$ 关于基 $\mathcal{B}$ 的**坐标向量**（coordinate vector）。

**直观解释**: 坐标是将抽象向量"翻译"成具体数字的方式。同一个向量在不同基下有不同的坐标——就像同一个点在直角坐标系和极坐标系中有不同的表示。

---

### 定义 8.5（标准基）

**严格陈述**: $\mathbb{R}^n$ 的**标准基**（standard basis）为

$$\mathcal{E} = \{\mathbf{e}_1, \mathbf{e}_2, \ldots, \mathbf{e}_n\},$$

其中 $\mathbf{e}_i$ 是第 $i$ 个分量为 $1$、其余分量为 $0$ 的列向量。

**直观解释**: 标准基是最"自然"的坐标系——向量 $(x_1, \ldots, x_n)^T$ 关于标准基的坐标就是其分量本身。但标准基并非唯一选择，根据问题需要可以选用更合适的基。

## 二、核心定理与完整证明

### 定理 8.1（线性相关的判定）

**定理陈述**: 向量组 $\mathbf{v}_1, \ldots, \mathbf{v}_k$（$k \geq 2$）线性相关当且仅当其中至少有一个向量可以表示为其余向量的线性组合。

**证明**:

**($\Rightarrow$)**：设 $\mathbf{v}_1, \ldots, \mathbf{v}_k$ 线性相关。则存在不全为零的标量 $c_1, \ldots, c_k$ 使

$$c_1 \mathbf{v}_1 + \cdots + c_k \mathbf{v}_k = \mathbf{0}.$$

设 $c_j \neq 0$。则

$$\mathbf{v}_j = -\frac{c_1}{c_j}\mathbf{v}_1 - \cdots - \frac{c_{j-1}}{c_j}\mathbf{v}_{j-1} - \frac{c_{j+1}}{c_j}\mathbf{v}_{j+1} - \cdots - \frac{c_k}{c_j}\mathbf{v}_k.$$

即 $\mathbf{v}_j$ 是其余向量的线性组合。

**($\Leftarrow$)**：设某 $\mathbf{v}_j$ 可表为其余向量的线性组合：

$$\mathbf{v}_j = d_1 \mathbf{v}_1 + \cdots + d_{j-1}\mathbf{v}_{j-1} + d_{j+1}\mathbf{v}_{j+1} + \cdots + d_k \mathbf{v}_k.$$

移项得

$$d_1 \mathbf{v}_1 + \cdots + d_{j-1}\mathbf{v}_{j-1} - \mathbf{v}_j + d_{j+1}\mathbf{v}_{j+1} + \cdots + d_k \mathbf{v}_k = \mathbf{0}.$$

系数中 $-1 \neq 0$，故不全为零，向量组线性相关。$\square$

---

### 定理 8.2（维数定理——所有基大小相同）

**定理陈述**: 设 $V$ 是有限维向量空间。则 $V$ 的任意两组基所含向量个数相同。

**证明**:

设 $\mathcal{B} = \{\mathbf{u}_1, \ldots, \mathbf{u}_m\}$ 和 $\mathcal{C} = \{\mathbf{v}_1, \ldots, \mathbf{v}_n\}$ 都是 $V$ 的基。我们需要证明 $m = n$。

**关键引理（Steinitz 替换原理）**: 若 $\{\mathbf{w}_1, \ldots, \mathbf{w}_k\}$ 线性无关，且 $\{\mathbf{x}_1, \ldots, \mathbf{x}_l\}$ 张成 $V$，则 $k \leq l$。

*引理的证明*: 由于 $\{\mathbf{x}_1, \ldots, \mathbf{x}_l\}$ 张成 $V$，$\mathbf{w}_1$ 可表为它们的线性组合：

$$\mathbf{w}_1 = a_1 \mathbf{x}_1 + \cdots + a_l \mathbf{x}_l.$$

由于 $\mathbf{w}_1 \neq \mathbf{0}$（线性无关组中无零向量），至少有一个 $a_i \neq 0$。不妨设 $a_1 \neq 0$，则 $\mathbf{x}_1$ 可由 $\mathbf{w}_1, \mathbf{x}_2, \ldots, \mathbf{x}_l$ 线性表示。因此

$$\operatorname{span}\{\mathbf{w}_1, \mathbf{x}_2, \ldots, \mathbf{x}_l\} = \operatorname{span}\{\mathbf{x}_1, \mathbf{x}_2, \ldots, \mathbf{x}_l\} = V.$$

继续此过程：$\mathbf{w}_2 \in V = \operatorname{span}\{\mathbf{w}_1, \mathbf{x}_2, \ldots, \mathbf{x}_l\}$，故

$$\mathbf{w}_2 = b_1 \mathbf{w}_1 + b_2 \mathbf{x}_2 + \cdots + b_l \mathbf{x}_l.$$

由于 $\mathbf{w}_1, \mathbf{w}_2$ 线性无关，$\mathbf{w}_2$ 不能仅由 $\mathbf{w}_1$ 表示，故 $b_2, \ldots, b_l$ 不全为零。不妨设 $b_2 \neq 0$，则可用 $\mathbf{w}_2$ 替换 $\mathbf{x}_2$，保持张成性。

重复 $k$ 次，用 $\mathbf{w}_1, \ldots, \mathbf{w}_k$ 替换了 $k$ 个 $\mathbf{x}_i$。由于每次替换后集合仍能张成 $V$，被替换的 $\mathbf{x}_i$ 至少有 $k$ 个，故 $l \geq k$。

*回到主证明*:

- $\mathcal{B}$ 线性无关，$\mathcal{C}$ 张成 $V$ $\Rightarrow$ $m \leq n$（引理）；
- $\mathcal{C}$ 线性无关，$\mathcal{B}$ 张成 $V$ $\Rightarrow$ $n \leq m$（引理）。

因此 $m = n$。$\square$

**证明要点提示**: 证明的核心是"替换原理"——线性无关组不能比张成组更大。这一原理在证明许多维数相关的不等式时非常有用。

```lean4
-- **定义 8.3**：维数的 Lean4 表示
-- Mathlib 使用 Module.finrank R V 表示 V 作为 R-模的有限秩（即维数）
-- 若 V 有 Basis ι R V，则 finrank = Fintype.card ι（当 ι 有限时）

open Module

-- **定理 8.2**：维数定理——所有基大小相同
-- 在 Lean4 中体现为：若 b₁ : Basis ι₁ R V，b₂ : Basis ι₂ R V，且 ι₁, ι₂ 均有限
-- 则 Fintype.card ι₁ = Fintype.card ι₂ = finrank R V
theorem dimension_well_defined {V : Type} [AddCommGroup V] [Module ℝ V]
    {ι₁ ι₂ : Type} [Fintype ι₁] [Fintype ι₂]
    (b₁ : Basis ι₁ ℝ V) (b₂ : Basis ι₂ ℝ V) :
    Fintype.card ι₁ = Fintype.card ι₂ := by
  -- 证明思路：利用 finrank 的良定义性
  -- finrank R V 不依赖于基的选择
  rw [← Basis.mk_finrank_eq_card_basis b₁, ← Basis.mk_finrank_eq_card_basis b₂]

-- 维数与 finrank 的关系
example {n : ℕ} : Module.finrank ℝ (Fin n → ℝ) = n := by
  -- ℝⁿ 的维数为 n
  simp [Module.finrank_fintype_fun_eq_card]
```

**定理陈述**: 设 $V$ 是 $n$ 维向量空间，$\mathcal{S} = \{\mathbf{v}_1, \ldots, \mathbf{v}_n\} \subseteq V$ 含 $n$ 个向量。则以下三条等价：

1. $\mathcal{S}$ 是 $V$ 的一组基；
2. $\mathcal{S}$ 线性无关；
3. $\mathcal{S}$ 张成 $V$。

**证明**:

**(1 $\Rightarrow$ 2)**：由基的定义直接得到。

**(1 $\Rightarrow$ 3)**：由基的定义直接得到。

**(2 $\Rightarrow$ 1)**：设 $\mathcal{S}$ 线性无关，需证 $\mathcal{S}$ 张成 $V$。

假设 $\mathcal{S}$ 不张成 $V$，则存在 $\mathbf{w} \in V$ 使得 $\mathbf{w} \notin \operatorname{span}(\mathcal{S})$。考虑集合 $\mathcal{S}' = \{\mathbf{v}_1, \ldots, \mathbf{v}_n, \mathbf{w}\}$。

断言 $\mathcal{S}'$ 线性无关：设

$$c_1 \mathbf{v}_1 + \cdots + c_n \mathbf{v}_n + c_{n+1}\mathbf{w} = \mathbf{0}.$$

若 $c_{n+1} \neq 0$，则 $\mathbf{w}$ 可由 $\mathcal{S}$ 线性表示，矛盾。故 $c_{n+1} = 0$，于是 $c_1 \mathbf{v}_1 + \cdots + c_n \mathbf{v}_n = \mathbf{0}$，由 $\mathcal{S}$ 线性无关得 $c_1 = \cdots = c_n = 0$。因此 $\mathcal{S}'$ 线性无关。

但 $\mathcal{S}'$ 含 $n+1$ 个向量，而 $V$ 的维数为 $n$，任意 $n+1$ 个向量必线性相关（由 Steinitz 引理：线性无关组大小 $\leq$ 基的大小）。矛盾。

因此 $\mathcal{S}$ 张成 $V$，是基。

**(3 $\Rightarrow$ 1)**：设 $\mathcal{S}$ 张成 $V$，需证 $\mathcal{S}$ 线性无关。

假设 $\mathcal{S}$ 线性相关。由**定理 8.1**，某 $\mathbf{v}_j$ 可表为其余向量的线性组合。去掉 $\mathbf{v}_j$ 后，剩余 $n-1$ 个向量仍张成 $V$。

但 $V$ 的维数为 $n$，需要至少 $n$ 个向量才能张成（Steinitz 引理的推论：张成组大小 $\geq$ 基的大小）。$n-1 < n$，矛盾。

因此 $\mathcal{S}$ 线性无关，是基。$\square$

```lean4
-- **定理 8.3**：基的判定定理
-- 在 n 维空间中，n 个向量线性无关 ⟺ n 个向量张成全空间 ⟺ 是基
open Module in
theorem basis_criterion {V : Type} [AddCommGroup V] [Module ℝ V]
    {n : ℕ} (hn : finrank ℝ V = n)
    {ι : Type} [Fintype ι] (hcard : Fintype.card ι = n)
    (v : ι → V) :
    LinearIndependent ℝ v ↔ ∃ b : Basis ι ℝ V, b = v := by
  -- 证明思路：利用 Mathlib 的 basis_iff_linearIndependent_of_card_eq_finrank
  constructor
  · intro hli
    -- 若 v 线性无关且个数 = dim V，则 v 是基
    have hspan : Submodule.span ℝ (Set.range v) = ⊤ := by
      sorry  -- 需用 dim(span v) = n 且 span v ⊆ V，故 span v = V
    exact ⟨⟨v, hli, hspan⟩, rfl⟩
  · rintro ⟨b, rfl⟩
    exact b.linearIndependent

-- **定理 8.4**：坐标表示的唯一性
-- 给定基 b，任意向量 v 可唯一表示为 ∑ cᵢ • bᵢ
open Module in
theorem coordinate_uniqueness {V : Type} [AddCommGroup V] [Module ℝ V]
    {ι : Type} (b : Basis ι ℝ V) (v : V) :
    ∃! c : ι → ℝ, v = ∑ i, c i • b i := by
  -- 证明：Basis 的 repr 给出唯一坐标
  use b.repr v
  constructor
  · -- 验证坐标正确
    rw [b.sum_repr v]
  · -- 证明唯一性
    intro c hc
    exact b.repr_inj (by rw [b.sum_repr v, hc])
```

**定理陈述**: 设 $\mathcal{B} = \{\mathbf{v}_1, \ldots, \mathbf{v}_n\}$ 是 $V$ 的基。则每个 $\mathbf{v} \in V$ 关于 $\mathcal{B}$ 的坐标 $[\mathbf{v}]_\mathcal{B}$ 唯一确定。

**证明**:

设 $\mathbf{v} = c_1 \mathbf{v}_1 + \cdots + c_n \mathbf{v}_n = d_1 \mathbf{v}_1 + \cdots + d_n \mathbf{v}_n$。

相减得 $(c_1 - d_1)\mathbf{v}_1 + \cdots + (c_n - d_n)\mathbf{v}_n = \mathbf{0}$。

由 $\mathcal{B}$ 线性无关，$c_i - d_i = 0$，即 $c_i = d_i$ 对所有 $i$ 成立。$\square$

### 定理 8.5（$\mathbb{R}^n$ 中向量组线性无关的矩阵判定）

**定理陈述**: 设 $\mathbf{v}_1, \ldots, \mathbf{v}_k \in \mathbb{R}^n$，构造矩阵 $A = [\mathbf{v}_1 \; \mathbf{v}_2 \; \cdots \; \mathbf{v}_k] \in \mathbb{R}^{n \times k}$。则 $\mathbf{v}_1, \ldots, \mathbf{v}_k$ 线性无关当且仅当 $N(A) = \{\mathbf{0}\}$（即 $A\mathbf{x} = \mathbf{0}$ 只有平凡解）。

**证明**:

$A\mathbf{x} = x_1 \mathbf{v}_1 + \cdots + x_k \mathbf{v}_k = \mathbf{0}$。

$N(A) = \{\mathbf{0}\}$ 意味着 $x_1 = \cdots = x_k = 0$ 是唯一解，这正是线性无关的定义。$\square$

## 三、典型例题

### 例题 8.1（判断向量组的线性无关性）

**题目**: 判断 $\mathbb{R}^3$ 中下列向量组是否线性无关：

$$\mathbf{v}_1 = \begin{pmatrix} 1 \\ 2 \\ 3 \end{pmatrix}, \quad \mathbf{v}_2 = \begin{pmatrix} 2 \\ 5 \\ 7 \end{pmatrix}, \quad \mathbf{v}_3 = \begin{pmatrix} 3 \\ 7 \\ 10 \end{pmatrix}.$$

**解答**:

构造矩阵 $A = [\mathbf{v}_1 \; \mathbf{v}_2 \; \mathbf{v}_3]$，通过消元判断 $N(A)$ 是否平凡。

$$A = \begin{pmatrix} 1 & 2 & 3 \\ 2 & 5 & 7 \\ 3 & 7 & 10 \end{pmatrix}$$

$R_2 \to R_2 - 2R_1$，$R_3 \to R_3 - 3R_1$：

$$\begin{pmatrix} 1 & 2 & 3 \\ 0 & 1 & 1 \\ 0 & 1 & 1 \end{pmatrix}$$

$R_3 \to R_3 - R_2$：

$$\begin{pmatrix} 1 & 2 & 3 \\ 0 & 1 & 1 \\ 0 & 0 & 0 \end{pmatrix} = U.$$

$U$ 只有 2 个主元，$r = 2 < 3 = k$。因此 $N(A) \neq \{\mathbf{0}\}$，向量组**线性相关**。

具体地，由 $U\mathbf{x} = \mathbf{0}$：$x_2 + x_3 = 0$，$x_1 + 2x_2 + 3x_3 = 0$。令 $x_3 = 1$，则 $x_2 = -1$，$x_1 = -1$。

验证：$-\mathbf{v}_1 - \mathbf{v}_2 + \mathbf{v}_3 = -(1,2,3)^T - (2,5,7)^T + (3,7,10)^T = (0,0,0)^T$ ✓。

**关键步骤解析**: 判断线性无关的标准方法是构造矩阵并消元。若主元个数等于向量个数，则线性无关；否则线性相关。线性关系可通过 RREF 中的自由变量直接读出。

---

### 例题 8.2（求子空间的基和维数）

**题目**: 设 $W = \{(x, y, z, w)^T \in \mathbb{R}^4 : x + y + z + w = 0, \; x - y + 2z = 0\}$。求 $W$ 的一组基和维数。

**解答**:

$W$ 是两个齐次方程的解空间，即 $N(A)$ 其中

$$A = \begin{pmatrix} 1 & 1 & 1 & 1 \\ 1 & -1 & 2 & 0 \end{pmatrix}.$$

消元：$R_2 \to R_2 - R_1$：

$$\begin{pmatrix} 1 & 1 & 1 & 1 \\ 0 & -2 & 1 & -1 \end{pmatrix}.$$

主元列：第 1、2 列。$r = 2$。自由变量：$z, w$。

由方程：

- $x + y + z + w = 0$
- $-2y + z - w = 0 \Rightarrow y = \frac{1}{2}(z - w)$

代入第一式：$x = -y - z - w = -\frac{1}{2}(z - w) - z - w = -\frac{3}{2}z - \frac{1}{2}w$。

**特解 $\mathbf{s}_1$**（$z = 1, w = 0$）：$y = \frac{1}{2}, x = -\frac{3}{2}$。$\mathbf{s}_1 = \left(-\frac{3}{2}, \frac{1}{2}, 1, 0\right)^T$。

为消除分数，可取 $2\mathbf{s}_1 = (-3, 1, 2, 0)^T$。

**特解 $\mathbf{s}_2$**（$z = 0, w = 1$）：$y = -\frac{1}{2}, x = -\frac{1}{2}$。$\mathbf{s}_2 = \left(-\frac{1}{2}, -\frac{1}{2}, 0, 1\right)^T$。

取 $2\mathbf{s}_2 = (-1, -1, 0, 2)^T$。

$$\text{基}: \quad \mathbf{b}_1 = \begin{pmatrix} -3 \\ 1 \\ 2 \\ 0 \end{pmatrix}, \quad \mathbf{b}_2 = \begin{pmatrix} -1 \\ -1 \\ 0 \\ 2 \end{pmatrix}.$$

$\dim W = 2$（自由变量个数 $= 4 - 2 = 2$）。$\square$

## 四、常见误区与纠正

### 误区 8.1: "含零向量的集合可以线性无关"

**错误观点**: 部分初学者认为，只要其他向量之间"不互相依赖"，即使有零向量存在，整个集合也可以是线性无关的。

**反例**: 考虑集合 $\{\mathbf{0}, \mathbf{v}_1, \mathbf{v}_2\}$，其中 $\mathbf{v}_1, \mathbf{v}_2$ 线性无关。

取 $c_0 = 1, c_1 = 0, c_2 = 0$，则

$$1 \cdot \mathbf{0} + 0 \cdot \mathbf{v}_1 + 0 \cdot \mathbf{v}_2 = \mathbf{0}.$$

系数 $c_0 = 1 \neq 0$，因此这是非平凡线性组合得到零向量。集合线性相关。

**为什么错**: 零向量本身就是一个"冗余"元素——它总可以被"零系数"表示，但同时它也可以用"非零系数"来构造非平凡关系。更准确地说，对任何包含零向量的集合 $\{\mathbf{0}, \mathbf{v}_1, \ldots, \mathbf{v}_k\}$，方程

$$1 \cdot \mathbf{0} + 0 \cdot \mathbf{v}_1 + \cdots + 0 \cdot \mathbf{v}_k = \mathbf{0}$$

给出了一个非平凡解（第一个系数为 $1 \neq 0$），因此集合必线性相关。

**正确理解**:

- 任何包含零向量的有限向量组必线性相关；
- 线性无关组中**绝不可能**有零向量；
- 这是判断线性相关的最快捷方法之一：一看有零向量，直接判定相关。

## 五、思想方法提炼

**本章核心思想**：

1. **最小生成 = 最大无关 = 基**: 基的三个等价刻画——(a) 线性无关且张成；(b) 最小的张成集；(c) 最大的线性无关集——从不同角度揭示了基的"最优性"。这种"极小极大"性质是基成为坐标系基石的根本原因。

2. **维数是空间的内在属性**: **定理 8.2** 表明，维数不依赖于基的选择，是向量空间本身的"指纹"。无论用哪组基去测量，空间的自由度总是相同的。这与物理中"系统的自由度数与坐标选择无关"的原理异曲同工。

3. **坐标 = 翻译器**: 坐标将抽象的向量运算转化为具体的数值运算。选定基后，$V$ 中的向量与 $\mathbb{R}^n$ 中的列向量一一对应，$V$ 上的线性代数完全转化为矩阵代数。这一"同构"思想是现代数学中"用坐标研究结构"的方法论基础。

**与后续章节的联系**：

- Ch.9 将用基和维数的语言重新描述四大基本子空间，证明正交补的维数关系；
- Ch.10 将引入正交基和标准正交基，使坐标计算（特别是投影）变得异常简洁；
- Ch.12-15 将反复使用基变换（坐标变换）来理解特征向量、对角化和奇异值分解。

## 六、习题

### 习题 8.1

**题目**: 判断 $\mathbb{R}^3$ 中下列向量组是否线性无关：

$$\mathbf{v}_1 = \begin{pmatrix} 1 \\ 0 \\ 1 \end{pmatrix}, \quad \mathbf{v}_2 = \begin{pmatrix} 0 \\ 1 \\ 1 \end{pmatrix}, \quad \mathbf{v}_3 = \begin{pmatrix} 1 \\ 1 \\ 0 \end{pmatrix}.$$

**解答**:

构造 $A = \begin{pmatrix} 1 & 0 & 1 \\ 0 & 1 & 1 \\ 1 & 1 & 0 \end{pmatrix}$。行列式：

$$\det(A) = 1(0-1) - 0 + 1(0-1) = -1 - 1 = -2 \neq 0.$$

故 $A$ 可逆，$N(A) = \{\mathbf{0}\}$，向量组**线性无关**。$\square$

---

### 习题 8.2

**题目**: 设 $\mathbf{v}_1 = (1, 2)^T$，$\mathbf{v}_2 = (3, 4)^T$。证明 $\{\mathbf{v}_1, \mathbf{v}_2\}$ 是 $\mathbb{R}^2$ 的一组基，并求 $\mathbf{v} = (5, 6)^T$ 关于这组基的坐标。

**解答**:

$\det\begin{pmatrix}1&3\\2&4\end{pmatrix} = 4-6 = -2 \neq 0$，故线性无关。$\mathbb{R}^2$ 中 2 个线性无关向量必为基（**定理 8.3**）。

设 $(5,6)^T = c_1(1,2)^T + c_2(3,4)^T$：

$$\begin{cases} c_1 + 3c_2 = 5 \\ 2c_1 + 4c_2 = 6 \end{cases}$$

由第一式：$c_1 = 5 - 3c_2$。代入第二式：$2(5-3c_2) + 4c_2 = 6 \Rightarrow 10 - 2c_2 = 6 \Rightarrow c_2 = 2$。

$c_1 = 5 - 6 = -1$。

坐标：$[\mathbf{v}]_\mathcal{B} = \begin{pmatrix} -1 \\ 2 \end{pmatrix}$。$\square$

### 习题 8.3

**题目**: 设 $W = \{(x, y, z)^T \in \mathbb{R}^3 : x + 2y - z = 0\}$。求 $W$ 的一组基和维数。

**解答**:

$W = N(A)$，$A = \begin{pmatrix} 1 & 2 & -1 \end{pmatrix}$。$r = 1$，自由变量 $y, z$。

- $y=1, z=0$：$x=-2$。$\mathbf{s}_1 = (-2, 1, 0)^T$。
- $y=0, z=1$：$x=1$。$\mathbf{s}_2 = (1, 0, 1)^T$。

基：$\{(-2, 1, 0)^T, (1, 0, 1)^T\}$。$\dim W = 2$（$\mathbb{R}^3$ 中的平面）。$\square$

### 习题 8.4

**题目**: 证明：若 $\mathbf{v}_1, \ldots, \mathbf{v}_k \in \mathbb{R}^n$ 且 $k > n$，则这组向量必线性相关。

**解答**:

构造 $A = [\mathbf{v}_1 \; \cdots \; \mathbf{v}_k] \in \mathbb{R}^{n \times k}$。

$\operatorname{rank}(A) \leq \min(m, n) = n < k$。

故 $\dim N(A) = k - \operatorname{rank}(A) \geq k - n > 0$，$N(A) \neq \{\mathbf{0}\}$。

由**定理 8.5**，向量组线性相关。$\square$

---

### 习题 8.5

**题目**: 设 $\mathbf{v}_1, \mathbf{v}_2, \mathbf{v}_3$ 线性无关。判断 $\mathbf{v}_1 + \mathbf{v}_2, \; \mathbf{v}_2 + \mathbf{v}_3, \; \mathbf{v}_3 + \mathbf{v}_1$ 是否线性无关。

**解答**:

设 $c_1(\mathbf{v}_1+\mathbf{v}_2) + c_2(\mathbf{v}_2+\mathbf{v}_3) + c_3(\mathbf{v}_3+\mathbf{v}_1) = \mathbf{0}$。

整理：$(c_1+c_3)\mathbf{v}_1 + (c_1+c_2)\mathbf{v}_2 + (c_2+c_3)\mathbf{v}_3 = \mathbf{0}$。

由线性无关：

$$\begin{cases} c_1 + c_3 = 0 \\ c_1 + c_2 = 0 \\ c_2 + c_3 = 0 \end{cases}$$

由第一式 $c_3 = -c_1$，第二式 $c_2 = -c_1$。代入第三式：$-c_1 - c_1 = 0 \Rightarrow c_1 = 0$。

于是 $c_2 = c_3 = 0$。只有平凡解，故**线性无关**。$\square$

### 习题 8.6

**题目**: 设 $\mathcal{P}_2$ 为次数不超过 2 的实多项式空间。证明 $\{1, t, t^2\}$ 是 $\mathcal{P}_2$ 的一组基。

**解答**:

**张成性**：任意 $p(t) = a_2 t^2 + a_1 t + a_0$ 显然可表为 $a_0 \cdot 1 + a_1 \cdot t + a_2 \cdot t^2$。

**线性无关性**：设 $c_0 \cdot 1 + c_1 \cdot t + c_2 \cdot t^2 = 0$（零多项式）。

非零多项式最多有有限个根，但上式左侧若在无穷多个点取值为 0，则必为零多项式，即所有系数为 0。更直接地，取 $t = 0$ 得 $c_0 = 0$；取 $t = 1$ 得 $c_1 + c_2 = 0$；取 $t = -1$ 得 $-c_1 + c_2 = 0$。解得 $c_1 = c_2 = 0$。

故 $\{1, t, t^2\}$ 线性无关且张成 $\mathcal{P}_2$，是基。$\dim \mathcal{P}_2 = 3$。$\square$

### 习题 8.7

**题目**: 设 $\mathbf{v}_1 = (1, 1, 1)^T$，$\mathbf{v}_2 = (1, 2, 3)^T$，$\mathbf{v}_3 = (2, 3, k)^T$。求所有使 $\{\mathbf{v}_1, \mathbf{v}_2, \mathbf{v}_3\}$ 线性相关的 $k$ 值。

**解答**:

构造 $A = \begin{pmatrix} 1 & 1 & 2 \\ 1 & 2 & 3 \\ 1 & 3 & k \end{pmatrix}$。

$\det(A) = 1(2k-9) - 1(k-3) + 2(3-2) = 2k - 9 - k + 3 + 2 = k - 4$。

当 $\det(A) = 0$，即 $k = 4$ 时，向量组线性相关。

验证 $k = 4$：$\mathbf{v}_3 = (2,3,4)^T = \mathbf{v}_1 + \mathbf{v}_2 = (2,3,4)^T$ ✓。$\square$

### 习题 8.8

**题目**: 设 $W_1, W_2$ 是有限维向量空间 $V$ 的子空间。证明：

$$\dim(W_1 + W_2) = \dim W_1 + \dim W_2 - \dim(W_1 \cap W_2).$$

**解答**:

设 $\{\mathbf{u}_1, \ldots, \mathbf{u}_k\}$ 是 $W_1 \cap W_2$ 的基。将其分别扩充为 $W_1$ 的基 $\{\mathbf{u}_1, \ldots, \mathbf{u}_k, \mathbf{v}_1, \ldots, \mathbf{v}_m\}$ 和 $W_2$ 的基 $\{\mathbf{u}_1, \ldots, \mathbf{u}_k, \mathbf{w}_1, \ldots, \mathbf{w}_n\}$。

**断言**：$\mathcal{B} = \{\mathbf{u}_1, \ldots, \mathbf{u}_k, \mathbf{v}_1, \ldots, \mathbf{v}_m, \mathbf{w}_1, \ldots, \mathbf{w}_n\}$ 是 $W_1 + W_2$ 的基。

**张成性**：$W_1 + W_2$ 中任一向量可写为 $\mathbf{x} + \mathbf{y}$，$\mathbf{x} \in W_1$，$\mathbf{y} \in W_2$。$\mathbf{x}$ 可由 $W_1$ 的基表示，$\mathbf{y}$ 可由 $W_2$ 的基表示，故 $\mathbf{x} + \mathbf{y}$ 可由 $\mathcal{B}$ 表示。

**线性无关性**：设

$$\sum a_i \mathbf{u}_i + \sum b_j \mathbf{v}_j + \sum c_l \mathbf{w}_l = \mathbf{0}.$$

移项：$\sum b_j \mathbf{v}_j = -\sum a_i \mathbf{u}_i - \sum c_l \mathbf{w}_l$。

左端 $\in W_1$，右端 $\in W_2$（因 $\mathbf{u}_i \in W_2$，$\mathbf{w}_l \in W_2$）。故左端 $\in W_1 \cap W_2$。

因此 $\sum b_j \mathbf{v}_j$ 可由 $\{\mathbf{u}_i\}$ 表示。但 $\{\mathbf{u}_i, \mathbf{v}_j\}$ 是 $W_1$ 的基，线性无关，故所有 $b_j = 0$。

同理可得所有 $c_l = 0$，进而所有 $a_i = 0$。

因此 $\mathcal{B}$ 线性无关，是 $W_1 + W_2$ 的基。

$\dim(W_1 + W_2) = k + m + n = (k+m) + (k+n) - k = \dim W_1 + \dim W_2 - \dim(W_1 \cap W_2)$。$\square$

### 习题 8.9

**题目**: 设 $A \in \mathbb{R}^{n \times n}$ 可逆，$\mathbf{v}_1, \ldots, \mathbf{v}_k \in \mathbb{R}^n$。证明：$\mathbf{v}_1, \ldots, \mathbf{v}_k$ 线性无关当且仅当 $A\mathbf{v}_1, \ldots, A\mathbf{v}_k$ 线性无关。

**解答**:

**($\Rightarrow$)**：设 $\mathbf{v}_1, \ldots, \mathbf{v}_k$ 线性无关。设 $c_1 A\mathbf{v}_1 + \cdots + c_k A\mathbf{v}_k = \mathbf{0}$。

则 $A(c_1\mathbf{v}_1 + \cdots + c_k\mathbf{v}_k) = \mathbf{0}$。因 $A$ 可逆，$c_1\mathbf{v}_1 + \cdots + c_k\mathbf{v}_k = \mathbf{0}$。

由 $\{\mathbf{v}_i\}$ 线性无关，$c_i = 0$。故 $\{A\mathbf{v}_i\}$ 线性无关。

**($\Leftarrow$)**：设 $A\mathbf{v}_1, \ldots, A\mathbf{v}_k$ 线性无关。将上述论证应用于 $A^{-1}$ 和 $A\mathbf{v}_i$，即得 $\mathbf{v}_i$ 线性无关。$\square$

### 习题 8.10

**题目**: 设 $V$ 是 $n$ 维向量空间，$W$ 是 $V$ 的子空间。证明：$W$ 存在补空间 $U$（即 $V = W \oplus U$，$W \cap U = \{\mathbf{0}\}$），且 $\dim U = n - \dim W$。

**解答**:

设 $\{\mathbf{w}_1, \ldots, \mathbf{w}_k\}$ 是 $W$ 的基，$k = \dim W$。

由 Steinitz 引理，可将其扩充为 $V$ 的基 $\{\mathbf{w}_1, \ldots, \mathbf{w}_k, \mathbf{u}_1, \ldots, \mathbf{u}_{n-k}\}$。

令 $U = \operatorname{span}\{\mathbf{u}_1, \ldots, \mathbf{u}_{n-k}\}$。

**$V = W + U$**：由基的定义，$V$ 中任一向量可由全基表示，即可分为 $W$ 部分和 $U$ 部分之和。

**$W \cap U = \{\mathbf{0}\}$**：设 $\mathbf{v} \in W \cap U$。则 $\mathbf{v} = \sum a_i \mathbf{w}_i = \sum b_j \mathbf{u}_j$。

移项：$\sum a_i \mathbf{w}_i - \sum b_j \mathbf{u}_j = \mathbf{0}$。由全基线性无关，$a_i = b_j = 0$，故 $\mathbf{v} = \mathbf{0}$。

因此 $V = W \oplus U$，$\dim U = n - k = n - \dim W$。$\square$

### 习题 8.11

**题目**: 设 $\mathbf{v}_1 = (1, 0, 1)^T$，$\mathbf{v}_2 = (0, 1, 1)^T$，$\mathbf{v}_3 = (1, 1, 0)^T$。验证这是 $\mathbb{R}^3$ 的基，并求 $\mathbf{e}_1 = (1, 0, 0)^T$ 关于该基的坐标。

**解答**:

由习题 8.1，$\det[\mathbf{v}_1 \; \mathbf{v}_2 \; \mathbf{v}_3] = -2 \neq 0$，故是基。

设 $\mathbf{e}_1 = c_1\mathbf{v}_1 + c_2\mathbf{v}_2 + c_3\mathbf{v}_3$：

$$\begin{cases} c_1 + c_3 = 1 \\ c_2 + c_3 = 0 \\ c_1 + c_2 = 0 \end{cases}$$

由第三式 $c_2 = -c_1$。由第二式 $c_3 = -c_2 = c_1$。代入第一式：$c_1 + c_1 = 1 \Rightarrow c_1 = \frac{1}{2}$。

$c_2 = -\frac{1}{2}$，$c_3 = \frac{1}{2}$。

坐标：$[\mathbf{e}_1]_\mathcal{B} = \begin{pmatrix} 1/2 \\ -1/2 \\ 1/2 \end{pmatrix}$。$\square$

### 习题 8.12

**题目**: 设 $V$ 是所有 $2 \times 2$ 实矩阵构成的向量空间。求 $V$ 的一组基和维数，并判断子空间 $W = \{A \in V : A^T = A\}$（对称矩阵）的维数。

**解答**:

$V$ 的基：

$$E_{11} = \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix}, \; E_{12} = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}, \; E_{21} = \begin{pmatrix} 0 & 0 \\ 1 & 0 \end{pmatrix}, \; E_{22} = \begin{pmatrix} 0 & 0 \\ 0 & 1 \end{pmatrix}.$$

$\dim V = 4$。

$W$（对称矩阵）：$A = \begin{pmatrix} a & b \\ b & d \end{pmatrix} = aE_{11} + b(E_{12}+E_{21}) + dE_{22}$。

$W$ 的基：$\{E_{11}, E_{12}+E_{21}, E_{22}\}$。$\dim W = 3$。$\square$
