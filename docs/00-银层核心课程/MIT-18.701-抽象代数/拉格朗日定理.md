# 拉格朗日定理（Lagrange's Theorem）

> **课程**: MIT 18.701 抽象代数 I | **主题**: 有限群的子群结构
> **学习目标**: 理解陪集划分的基本思想，掌握拉格朗日定理的完整证明及其核心推论，能够运用该定理分析有限群的基本性质

---

## 一、核心定义

### 定义 1.1（子群）

**严格陈述**: 设 $(G, \cdot)$ 为群，$H \subseteq G$ 为非空子集。若 $H$ 在 $G$ 的运算下也构成群，则称 $H$ 为 $G$ 的**子群**（subgroup），记作 $H \leq G$。

等价地，$H \leq G$ 当且仅当满足以下三条：

1. **封闭性**: $\forall a, b \in H,\; a \cdot b \in H$；
2. **单位元**: $1_G \in H$；
3. **逆元**: $\forall a \in H,\; a^{-1} \in H$。

**直观解释**: 子群是原群的一个"自包含"的子结构，它继承了原群的运算规则，同时自身也满足群的四条公理（结合律自动继承自 $G$）。

> **双语对照**:
>
> ```lean4
> -- Lean4 中子群由 Subgroup 结构表示
> variable {G : Type u} [Group G]
> variable (H : Subgroup G)
> -- H 封装了 carrier（载体集合）以及封闭性、单位元、逆元的证明
> ```

---

### 定义 1.2（左陪集与右陪集）

**严格陈述**: 设 $G$ 为群，$H \leq G$，$a \in G$。集合
$$aH \;:=\; \{ah \mid h \in H\}$$
称为 $H$ 在 $G$ 中关于代表元 $a$ 的**左陪集**（left coset）。类似地，集合
$$Ha \;:=\; \{ha \mid h \in H\}$$
称为 $H$ 关于 $a$ 的**右陪集**（right coset）。

**直观解释**: 左陪集 $aH$ 是将子群 $H$ 中的每个元素都"左乘" $a$ 后得到的集合。可以把 $H$ 想象成过原点的直线，$aH$ 则是将该直线平移到经过点 $a$ 的位置。

> **双语对照**:
>
> ```lean4
> -- Mathlib4 中左陪集由左作用 (•) 定义
> -- a • (H : Set G) 表示左陪集 aH
> theorem leftCoset_card_eq_subgroup_card
>     {G : Type u} [Group G] [Fintype G]
>     (H : Subgroup G) [Fintype H] (a : G) :
>     (a • (H : Set G)).toFinset.card = Fintype.card H := by
> ```

---

### 定义 1.3（陪集代表元）

**严格陈述**: 若 $aH$ 是 $H$ 的一个左陪集，则称 $a$ 为该陪集的一个**代表元**（coset representative）。注意：陪集的代表元不唯一；事实上，$aH = bH$ 当且仅当 $a^{-1}b \in H$。

**直观解释**: 同一陪集中的任意元素都可以作为该陪集的代表元。选择代表元的自由性在构造商群时至关重要。

---

### 定义 1.4（指数）

**严格陈述**: 设 $G$ 为群，$H \leq G$。$H$ 在 $G$ 中不同左陪集的个数称为 $H$ 在 $G$ 中的**指数**（index），记作 $[G : H]$。当 $G$ 为有限群时，$[G : H]$ 是一个有限正整数。

> **双语对照**:
>
> ```lean4
> -- Mathlib4 中指数由 Subgroup.index 定义
> -- H.index 表示 [G : H]
> theorem lagrange_theorem_full {G : Type u} [Group G] [Fintype G]
>     (H : Subgroup G) [Fintype H] :
>     Fintype.card G = H.index * Fintype.card H := by
>   exact (Subgroup.index_mul_card H).symm
> ```

---

## 二、核心定理与完整证明

### 定理 2.1（拉格朗日定理）

**定理陈述**: 设 $G$ 为有限群，$H \leq G$。则 $|H|$ 整除 $|G|$，且
$$|G| \;=\; [G : H] \cdot |H|.$$

**证明**: 我们分三步完成证明。

**第一步：证明任意两个左陪集要么相等，要么不相交。**

设 $a, b \in G$。假设 $aH \cap bH \neq \varnothing$，则存在 $x \in aH \cap bH$。于是存在 $h_1, h_2 \in H$ 使得
$$x \;=\; ah_1 \;=\; bh_2.$$
由此得 $a = bh_2h_1^{-1}$。因为 $H$ 是子群，$h_2h_1^{-1} \in H$，所以 $a \in bH$。于是对任意 $ah \in aH$，有
$$ah \;=\; (bh_2h_1^{-1})h \;=\; b(h_2h_1^{-1}h) \;\in\; bH,$$
故 $aH \subseteq bH$。对称地可证 $bH \subseteq aH$。因此 $aH = bH$。

这说明：若两个左陪集有公共元素，则它们完全重合。换言之，不同的左陪集两两不相交。

> **双语对照**:
>
> ```lean4
> theorem leftCoset_eq_or_disjoint {G : Type u} [Group G] (H : Subgroup G) (a b : G) :
>     a • (H : Set G) = b • (H : Set G) ∨
>     (a • (H : Set G)) ∩ (b • (H : Set G)) = ∅ := by
>   by_cases h : (a • (H : Set G)) ∩ (b • (H : Set G)) = ∅
>   · right; exact h
>   · left
>     have : ∃ x, x ∈ (a • (H : Set G)) ∩ (b • (H : Set G)) := by
>       by_contra h'
>       push_neg at h'
>       have : (a • (H : Set G)) ∩ (b • (H : Set G)) = ∅ := by
>         ext x; simp [h']
>       contradiction
>     rcases this with ⟨x, hxa, hxb⟩
>     simp [leftCoset] at hxa hxb
>     rcases hxa with ⟨h₁, hh₁, rfl⟩
>     rcases hxb with ⟨h₂, hh₂, heq⟩
>     have ha : a = b * h₂ * h₁⁻¹ := by
>       rw [←heq]; group
>     ext y
>     simp [leftCoset, ha]
>     constructor
>     · intro hy
>       rcases hy with ⟨h, hh, rfl⟩
>       use h₂ * h₁⁻¹ * h
>       constructor
>       · exact H.mul_mem (H.mul_mem hh₂ (H.inv_mem hh₁)) hh
>       · group
>     · intro hy
>       rcases hy with ⟨h, hh, rfl⟩
>       use h₁ * h₂⁻¹ * h
>       constructor
>       · exact H.mul_mem (H.mul_mem hh₁ (H.inv_mem hh₂)) hh
>       · group
> ```

**第二步：证明每个左陪集与 $H$ 等势（基数相同）。**

定义映射 $f : H \to aH$ 为 $f(h) = ah$。

- **单射性**: 若 $f(h_1) = f(h_2)$，即 $ah_1 = ah_2$，左乘 $a^{-1}$ 得 $h_1 = h_2$。
- **满射性**: 对任意 $x \in aH$，由定义存在 $h \in H$ 使得 $x = ah = f(h)$。

因此 $f$ 是双射，$|aH| = |H|$。

> **双语对照**:
>
> ```lean4
> theorem leftCoset_card_eq_subgroup_card {G : Type u} [Group G] [Fintype G]
>     (H : Subgroup G) [Fintype H] (a : G) :
>     (a • (H : Set G)).toFinset.card = Fintype.card H := by
>   let f : H → (a • (H : Set G)).toFinset := fun h =>
>     ⟨a * h, by simp [leftCoset]; exact ⟨h, h.2, rfl⟩⟩
>   have h_inj : Function.Injective f := by
>     intro h₁ h₂ heq
>     simp [f] at heq
>     have : a * h₁ = a * h₂ := by simpa using heq
>     exact (mul_left_inj a).mp this
>   have h_surj : Function.Surjective f := by
>     intro x
>     rcases x with ⟨x, hx⟩
>     simp [leftCoset] at hx
>     rcases hx with ⟨h, hh, rfl⟩
>     exact ⟨⟨h, hh⟩, by simp [f]⟩
>   have h_bij : Function.Bijective f := ⟨h_inj, h_surj⟩
>   exact (Fintype.card_of_bijective h_bij).symm
> ```

**第三步：计数论证。**

由于不同左陪集两两不相交，且所有左陪集的并恰好是整个群 $G$（因为对任意 $g \in G$，有 $g \in gH$），所以 $G$ 被分解为 $[G : H]$ 个互不相交的左陪集的并。每个左陪集的大小均为 $|H|$，因此
$$|G| \;=\; [G : H] \cdot |H|.$$

特别地，$|H|$ 整除 $|G|$。 $\square$

> **双语对照**:
>
> ```lean4
> theorem lagrange_theorem {G : Type u} [Group G] [Fintype G]
>     (H : Subgroup G) [Fintype H] :
>     Fintype.card H ∣ Fintype.card G := by
>   rw [← Subgroup.index_mul_card H]
>   exact dvd_mul_left (Fintype.card H) (H.index)
>
> theorem lagrange_theorem_full {G : Type u} [Group G] [Fintype G]
>     (H : Subgroup G) [Fintype H] :
>     Fintype.card G = H.index * Fintype.card H := by
>   exact (Subgroup.index_mul_card H).symm
> ```

> **证明要点提示**: 本证明的核心思想是"划分—等势—计数"三部曲。左陪集将群划分为大小相等的块，这是群论中最基本的几何直觉之一。

### 推论 2.2（元素阶整除群的阶）

**定理陈述**: 设 $G$ 为有限群，$a \in G$。则元素 $a$ 的阶 $\operatorname{ord}(a)$ 整除 $|G|$，且 $a^{|G|} = 1_G$。

**证明**: 考虑由 $a$ 生成的循环子群 $\langle a \rangle = \{a^n \mid n \in \mathbb{Z}\}$。已知 $|\langle a \rangle| = \operatorname{ord}(a)$。由拉格朗日定理，$|\langle a \rangle|$ 整除 $|G|$，即 $\operatorname{ord}(a) \mid |G|$。设 $|G| = \operatorname{ord}(a) \cdot k$，则
$$a^{|G|} \;=\; a^{\operatorname{ord}(a) \cdot k} \;=\; \bigl(a^{\operatorname{ord}(a)}\bigr)^k \;=\; 1_G^k \;=\; 1_G.$$
$\square$

> **双语对照**:
>
> ```lean4
> theorem order_of_dvd_card {G : Type u} [Group G] [Fintype G] (a : G) :
>     orderOf a ∣ Fintype.card G := by
>   have h : orderOf a = Fintype.card (zpowers a) := by
>     rw [orderOf_eq_card_zpowers]
>   rw [h]
>   exact lagrange_theorem (zpowers a)
>
> theorem fermat_little_theorem_group {G : Type u} [Group G] [Fintype G] (a : G) :
>     a ^ Fintype.card G = 1 := by
>   have h : orderOf a ∣ Fintype.card G := order_of_dvd_card a
>   rcases h with ⟨k, hk⟩
>   calc
>     a ^ Fintype.card G = a ^ (orderOf a * k) := by rw [hk]
>     _ = (a ^ orderOf a) ^ k := by rw [pow_mul]
>     _ = 1 ^ k := by rw [pow_orderOf_eq_one a]
>     _ = 1 := by simp
> ```

## 三、典型例题

### 例题 3.1（素数阶群必为循环群）

**题目**: 设 $G$ 为有限群，$|G| = p$ 为素数。证明 $G$ 是循环群，且 $G \cong \mathbb{Z}/p\mathbb{Z}$。

**解答**:

取 $a \in G$，$a \neq 1_G$。考虑循环子群 $\langle a \rangle \leq G$。由拉格朗日定理，$|\langle a \rangle|$ 整除 $|G| = p$。因为 $p$ 是素数，$|\langle a \rangle|$ 只能是 $1$ 或 $p$。但 $a \neq 1_G$，所以 $\langle a \rangle$ 至少含有两个元素 $\{1_G, a\}$，故 $|\langle a \rangle| \geq 2$。因此 $|\langle a \rangle| = p = |G|$。

由 $\langle a \rangle \subseteq G$ 且两者元素个数相等，得 $\langle a \rangle = G$。所以 $G$ 是由 $a$ 生成的循环群。有限循环群的结构定理告诉我们，$p$ 阶循环群同构于 $\mathbb{Z}/p\mathbb{Z}$，即 $G \cong \mathbb{Z}/p\mathbb{Z}$。 $\square$

**关键步骤解析**: 本题的关键在于利用素数的"不可分割性"。拉格朗日定理将子群的可能阶数限制为 $1$ 或 $p$，而选取非单位元立即排除了平凡子群的情形。

---

### 例题 3.2（利用指数求子群阶）

**题目**: 设 $G$ 为 24 阶群，$H \leq G$ 且 $[G : H] = 3$。求 $|H|$。

**解答**: 由拉格朗日定理的完整形式 $|G| = [G : H] \cdot |H|$，代入得
$$24 \;=\; 3 \cdot |H|,$$
所以 $|H| = 8$。 $\square$

## 四、常见误区与纠正

### 误区 4.1: "拉格朗日定理的逆命题成立"

**错误观点**: 许多初学者认为：若正整数 $n$ 整除 $|G|$，则 $G$ 必存在 $n$ 阶子群。

**反例或纠正**: 这一命题**不成立**。经典的反例是交错群 $A_4$（4 元交错群）。$|A_4| = 12$，$6 \mid 12$，但 $A_4$ 不存在 6 阶子群。

**详细验证**: $A_4$ 由 4 元集上的所有偶置换组成，其元素为：

- 单位元 $e$（1 个）；
- 3-循环，如 $(123), (132), (124), (142), (134), (143), (234), (243)$（共 8 个）；
- 双对换，如 $(12)(34), (13)(24), (14)(23)$（共 3 个）。

假设 $H \leq A_4$ 且 $|H| = 6$。由拉格朗日定理，$[A_4 : H] = 2$，故 $H \trianglelefteq A_4$（指数为 2 的子群必正规）。考虑 $A_4$ 中的 3-循环 $g$。因为 $g^3 = e$，在商群 $A_4/H$ 中有 $(gH)^3 = H$。但 $|A_4/H| = 2$，所以 $(gH)^2 = H$，于是 $gH = H$，即 $g \in H$。这说明所有 8 个 3-循环都在 $H$ 中，与 $|H| = 6$ 矛盾。因此 $A_4$ 不存在 6 阶子群。

> **为什么错**: 拉格朗日定理只给出了子群阶数的**必要条件**（必须整除群的阶），而非**充分条件**。子群的存在性还受到群结构（如正规性、共轭类分布等）的深层约束。Sylow 定理部分地弥补了这一点：当 $n$ 是素数幂时，逆命题成立。

## 五、思想方法提炼

**本章核心思想**:

1. **划分与计数**: 拉格朗日定理的本质是将群按子群的陪集划分为大小相等的"格子"，这是将几何直觉（平移不变性）代数化的典范。

2. **必要 vs. 充分**: 拉格朗日定理提供的是子群阶数的必要条件，而非充分条件。在群论中，区分必要条件和充分条件是避免错误猜想的关键。

3. **从结构到元素**: 由拉格朗日定理可导出元素阶整除群的阶，这是从"子群结构"下降到"元素性质"的典型推理路径。

**与后续章节的联系**:

- 拉格朗日定理是证明 **Sylow 第一定理** 的基础工具之一（归纳步骤中需要利用子群的指数和阶的关系）。
- 在 **第一同构定理** 的证明中，利用拉格朗日定理的推论可得同态的阶公式 $|G| = |\ker\varphi| \cdot |\operatorname{im}\varphi|$。
- 商群的阶 $|G/N| = [G : N]$ 直接由拉格朗日定理得出。

## 六、习题

### 习题 6.1

**题目**: 设 $G$ 为有限群，$K \leq H \leq G$。证明 $[G : K] = [G : H] \cdot [H : K]$（指数的传递性）。

**提示**: 利用拉格朗日定理分别计算 $|G|$、$|H|$、$|K|$ 之间的关系。

**解答**: 由拉格朗日定理：
$$|G| = [G : H] \cdot |H| = [G : H] \cdot [H : K] \cdot |K|.$$
另一方面，$|G| = [G : K] \cdot |K|$。比较两式并消去 $|K|$（$|K| \geq 1$），得 $[G : K] = [G : H] \cdot [H : K]$。 $\square$

---

### 习题 6.2

**题目**: 设 $G$ 为群，$|G| = p^2$（$p$ 为素数）。证明 $G$ 是交换群。

**提示**: 考虑群的中心 $Z(G)$，利用类方程证明 $|Z(G)| > 1$，再分情况讨论。

**解答**: 由类方程，$|G| = |Z(G)| + \sum_i [G : C_G(g_i)]$，其中 $g_i$ 取遍非中心共轭类的代表元。每个 $[G : C_G(g_i)] > 1$ 且整除 $p^2$，故 $[G : C_G(g_i)] = p$ 或 $p^2$。因此每个求和项都被 $p$ 整除。由于 $p^2$ 被 $p$ 整除，得 $p \mid |Z(G)|$。于是 $|Z(G)| \geq p > 1$。

若 $|Z(G)| = p^2$，则 $Z(G) = G$，$G$ 交换。

若 $|Z(G)| = p$，则 $G/Z(G)$ 的阶为 $p$，由例题 3.1 知 $G/Z(G)$ 是循环群。设 $G/Z(G) = \langle aZ(G) \rangle$。则任意 $g_1, g_2 \in G$ 可写为 $g_1 = a^{m_1}z_1$，$g_2 = a^{m_2}z_2$（$z_1, z_2 \in Z(G)$）。于是
$$g_1g_2 = a^{m_1}z_1a^{m_2}z_2 = a^{m_1}a^{m_2}z_1z_2 = a^{m_2}a^{m_1}z_2z_1 = g_2g_1,$$
故 $G$ 交换。 $\square$

### 习题 6.3

**题目**: 设 $G$ 为有限群，$H, K \leq G$。证明 $|HK| = \dfrac{|H| \cdot |K|}{|H \cap K|}$，其中 $HK = \{hk \mid h \in H, k \in K\}$。

**提示**: 考虑 $H \times K$ 在 $HK$ 上的作用，或构造从 $H \times K$ 到 $HK$ 的满射并计算每个像的原像个数。

**解答**: 定义映射 $\varphi : H \times K \to HK$ 为 $\varphi(h, k) = hk$。显然 $\varphi$ 是满射。对任意 $x \in HK$，设 $x = h_0k_0$，则
$$\varphi^{-1}(x) = \{(h, k) \in H \times K \mid hk = h_0k_0\}.$$
由 $hk = h_0k_0$ 得 $h_0^{-1}h = k_0k^{-1} \in H \cap K$。令 $t = h_0^{-1}h$，则 $h = h_0t$，$k = t^{-1}k_0$。反之，对任意 $t \in H \cap K$，$(h_0t, t^{-1}k_0) \in \varphi^{-1}(x)$。因此 $|\varphi^{-1}(x)| = |H \cap K|$。

所以 $|H \times K| = |HK| \cdot |H \cap K|$，即 $|H| \cdot |K| = |HK| \cdot |H \cap K|$。整理即得所求。 $\square$

### 习题 6.4

**题目**: 证明 6 阶群中必存在 3 阶子群。

**提示**: 分 $G$ 有 6 阶元素和没有 6 阶元素两种情况讨论。

**解答**: 若 $G$ 有 6 阶元素 $a$，则 $\langle a^2 \rangle$ 是 3 阶子群（因为 $(a^2)^3 = a^6 = 1$，且 $a^2 \neq 1$）。

若 $G$ 没有 6 阶元素，则所有非单位元的阶为 2 或 3。若所有非单位元的阶均为 2，则 $G$ 是交换群（因为 $x^2 = 1$ 对所有 $x$ 成立意味着 $x = x^{-1}$，于是 $xy = (xy)^{-1} = y^{-1}x^{-1} = yx$）。取 $a, b \in G$ 为不同非单位元，则 $\{1, a, b, ab\}$ 构成 4 阶子群，与拉格朗日定理（$4 \nmid 6$）矛盾。因此 $G$ 必有 3 阶元素，其生成的循环子群即为 3 阶子群。 $\square$

### 习题 6.5

**题目**: 设 $G$ 为有限群，$|G| = 15$。证明 $G$ 是循环群。

**提示**: 利用 Cauchy 定理（或拉格朗日定理的推论）找出 3 阶和 5 阶元素，证明它们生成的子群相交平凡，再利用阶的公式。

**解答**: 由 Cauchy 定理（见本文档后续主题），$G$ 中存在 3 阶元素 $a$ 和 5 阶元素 $b$。则 $|\langle a \rangle| = 3$，$|\langle b \rangle| = 5$。由拉格朗日定理，$|\langle a \rangle \cap \langle b \rangle|$ 整除 3 和 5，故 $|\langle a \rangle \cap \langle b \rangle| = 1$，即两子群交为平凡子群。

考虑 $|\langle a \rangle \langle b \rangle| = \dfrac{3 \cdot 5}{1} = 15 = |G|$（习题 6.3），所以 $G = \langle a \rangle \langle b \rangle$。进一步可证 $ab = ba$（因为 $\langle a \rangle \trianglelefteq G$，$\langle b \rangle \trianglelefteq G$，这由 Sylow 定理的唯一性可得），于是 $\operatorname{ord}(ab) = 15$，$G = \langle ab \rangle$ 为循环群。 $\square$

### 习题 6.6

**题目**: 设 $G$ 为群，$H \leq G$ 且 $[G : H] = 2$。证明 $H \trianglelefteq G$（$H$ 是正规子群）。

**提示**: 验证对任意 $g \in G$，$gH = Hg$。分 $g \in H$ 和 $g \notin H$ 两种情况。

**解答**: 当 $g \in H$ 时，$gH = H = Hg$。当 $g \notin H$ 时，左陪集只有两个：$H$ 和 $gH$，且它们不相交，故 $gH = G \setminus H$。同理右陪集也只有 $H$ 和 $Hg$，且 $Hg = G \setminus H$。因此 $gH = Hg$。由正规子群的定义，$H \trianglelefteq G$。 $\square$

### 习题 6.7

**题目**: 设 $G$ 为有限群，$a, b \in G$。证明 $\operatorname{ord}(ab) = \operatorname{ord}(ba)$。

**提示**: 证明 $\langle ab \rangle$ 与 $\langle ba \rangle$ 共轭，或直接验证 $(ba)^n = a^{-1}(ab)^na$。

**解答**: 对任意正整数 $n$，
$$(ba)^n \;=\; \underbrace{(ba)(ba)\cdots(ba)}_{n\text{ 个}} \;=\; b\underbrace{(ab)(ab)\cdots(ab)}_{n-1\text{ 个}}a \;=\; b(ab)^{n-1}a.$$
更简洁地，注意到 $ba = a^{-1}(ab)a$，即 $ba$ 与 $ab$ 共轭。共轭元素的阶相等，故 $\operatorname{ord}(ab) = \operatorname{ord}(ba)$。 $\square$

### 习题 6.8

**题目**: 设 $G$ 为有限群，$|G| = p^n$（$p$ 为素数，$n \geq 1$）。证明 $G$ 的中心 $Z(G)$ 非平凡（即 $|Z(G)| > 1$）。

**提示**: 利用类方程，注意共轭类的大小整除 $|G|$。

**解答**: 由类方程
$$|G| = |Z(G)| + \sum_{i} [G : C_G(g_i)],$$
其中 $g_i$ 遍历非中心共轭类的代表元。对每个 $i$，$g_i \notin Z(G)$，故 $C_G(g_i) \neq G$，于是 $[G : C_G(g_i)] > 1$。又因为 $[G : C_G(g_i)]$ 整除 $|G| = p^n$，所以 $p \mid [G : C_G(g_i)]$。因此求和项被 $p$ 整除。左边 $|G| = p^n$ 也被 $p$ 整除，故 $p \mid |Z(G)|$。由于 $1 \in Z(G)$，$|Z(G)| \geq p > 1$。 $\square$

### 习题 6.9

**题目**: 设 $G$ 为群，$H, K \leq G$ 且 $H \subseteq K$。证明 $H \leq K$（即 $H$ 也是 $K$ 的子群）。

**提示**: 直接验证子群定义的三条。

**解答**: 因为 $H \leq G$，所以 $H$ 非空。对任意 $a, b \in H \subseteq K$，由于 $H \leq G$，有 $ab \in H$ 且 $a^{-1} \in H$。同时 $1_G \in H$。而 $K \leq G$ 保证了 $K$ 中的运算与 $G$ 一致，所以 $H$ 在 $K$ 的运算下也封闭，包含单位元，包含逆元。故 $H \leq K$。 $\square$

### 习题 6.10

**题目**: 设 $G$ 为有限交换群，$|G| = n$。证明对任意 $m \mid n$，$G$ 存在 $m$ 阶子群。（注：这是有限交换群情形下拉格朗日定理逆命题的一个正面结果。）

**提示**: 利用有限交换群的结构定理：$G \cong \mathbb{Z}/n_1\mathbb{Z} \times \cdots \times \mathbb{Z}/n_k\mathbb{Z}$，其中 $n_1 \mid n_2 \mid \cdots \mid n_k$ 且 $n_1n_2\cdots n_k = n$。

**解答**: 由有限交换群基本定理，$G \cong \mathbb{Z}/n_1\mathbb{Z} \times \cdots \times \mathbb{Z}/n_k\mathbb{Z}$，其中 $n_1 \mid n_2 \mid \cdots \mid n_k$ 且乘积为 $n$。设 $m \mid n$。对每个 $i$，令 $d_i = \gcd(m, n_i)$。由于 $n_1 \mid n_2 \mid \cdots \mid n_k$，可取适当的 $d_i$ 使得 $d_1d_2\cdots d_k = m$（通过对 $m$ 的素因子分解逐层分配）。对每个 $i$，$\mathbb{Z}/n_i\mathbb{Z}$ 有唯一的 $d_i$ 阶子群（由 $\overline{n_i/d_i}$ 生成）。这些子群的直积给出 $G$ 的一个 $m$ 阶子群。 $\square$

**习题 1.2**。设 $H,K$ 是有限群 $G$ 的子群。证明 $|HK| = \\frac{|H||K|}{|H\\cap K|}$。
