# 多项式环的欧几里得算法与唯一分解定理

> **课程**: MIT 18.701 抽象代数 I | **主题**: 多项式环的算术性质
> **学习目标**: 掌握多项式环的带余除法、欧几里得算法与唯一分解定理的完整证明链条，能够计算具体多项式的最大公因式并进行因式分解

---

## 一、核心定义

### 定义 1.1（多项式环）

**严格陈述**: 设 $F$ 为域。$F$ 上的**一元多项式环**（polynomial ring in one variable）$F[x]$ 定义为所有形式表达式
$$f(x) \;=\; a_0 + a_1x + a_2x^2 + \cdots + a_nx^n \;=\; \sum_{i=0}^{n} a_i x^i,$$
其中 $a_i \in F$，$n \geq 0$，且 $x$ 是**不定元**（indeterminate）。多项式的加法和乘法按通常方式定义（合并同类项、分配律展开）。

$F[x]$ 在加法和乘法下构成交换环，且若 $F$ 含幺则 $F[x]$ 也含幺（常数多项式 $1$ 是单位元）。

**直观解释**: 多项式环是域上"最自由"的交换环扩张——添加一个不定元 $x$ 而不施加任何代数关系。$F[x]$ 中的元素可以看作系数在 $F$ 中的有限序列 $(a_0, a_1, \ldots, a_n)$，其乘法由卷积给出。

> **双语对照**:
>
> ```lean4
> -- Lean4 中多项式环由 Polynomial F（或 F[X]）表示
> variable {F : Type*} [Field F]
> -- F[X] 是 F 上的多项式环
> example : CommRing F[X] := by infer_instance
> example : IsDomain F[X] := by infer_instance -- 多项式环是整环
> ```

---

### 定义 1.2（次数与首一多项式）

**严格陈述**: 设 $f(x) = \sum_{i=0}^{n} a_i x^i \in F[x]$，且 $f \neq 0$。

- **次数**（degree）: $\deg(f) := \max\{i \mid a_i \neq 0\}$，即最高非零项的指数。约定 $\deg(0) = -\infty$。
- **首项系数**（leading coefficient）: $a_n$，其中 $n = \deg(f)$。
- **首一多项式**（monic polynomial）: 首项系数为 $1$ 的多项式，即 $a_n = 1$。

**次数的基本性质**: 对非零多项式 $f, g \in F[x]$，
$$\deg(fg) = \deg(f) + \deg(g).$$
特别地，$F[x]$ 是整环（无零因子）。

**证明零因子性质**: 设 $f, g \neq 0$，$\deg(f) = m$，$\deg(g) = n$。则 $fg$ 的首项为 $a_mb_n x^{m+n}$，其中 $a_m, b_n \neq 0$。因为 $F$ 是域，$a_mb_n \neq 0$，故 $fg \neq 0$。 $\square$

> **双语对照**:
>
> ```lean4
> -- Mathlib4 中次数由 Polynomial.degree 定义
> open Polynomial
> variable {F : Type*} [Field F] {f g : F[X]}
>
> theorem degree_mul : (f * g).degree = f.degree + g.degree := by
>   by_cases hf : f = 0
>   · simp [hf]
>   by_cases hg : g = 0
>   · simp [hg]
>   exact degree_mul' (mul_ne_zero hf hg)
>
> -- 首一多项式
> def Monic (f : F[X]) : Prop := leadingCoeff f = 1
> ```

### 定义 1.3（不可约多项式）

**严格陈述**: 设 $F$ 为域，$f(x) \in F[x]$ 为非常数多项式。若 $f$ 不能分解为两个非常数多项式的乘积，即
$$f = gh \quad \Longrightarrow \quad \deg(g) = 0 \text{ 或 } \deg(h) = 0,$$
则称 $f$ 在 $F$ 上是**不可约的**（irreducible）。否则称 $f$ 是**可约的**（reducible）。

**等价刻画**: $f$ 不可约当且仅当 $(f)$ 是 $F[x]$ 的极大理想（因为 $F[x]$ 是 PID，见**定理 2.3**）。

**直观解释**: 不可约多项式是多项式环中的"素数"——它们不能再被非平凡地分解。但要注意：可约性与基域密切相关。例如 $x^2 + 1$ 在 $\mathbb{R}$ 上不可约，但在 $\mathbb{C}$ 上可约为 $(x - i)(x + i)$。

> **双语对照**:
>
> ```lean4
> -- 不可约多项式在 Mathlib4 中
> def Irreducible (f : F[X]) : Prop :=
>   f ≠ 0 ∧ ¬IsUnit f ∧ ∀ g h, f = g * h → IsUnit g ∨ IsUnit h
> -- 等价于：非零、非单位，且任何分解必有一个因子是单位
> ```

---

### 定义 1.4（欧几里得整环、主理想整环与唯一分解整环）

**严格陈述**: 设 $R$ 为整环。

1. **欧几里得整环**（Euclidean Domain, ED）: 若存在**欧几里得度**（Euclidean norm）$N : R \setminus \{0\} \to \mathbb{N}$，使得对任意 $a, b \in R$，$b \neq 0$，存在 $q, r \in R$ 满足
   $$a = qb + r, \quad \text{其中 } r = 0 \text{ 或 } N(r) < N(b).$$

2. **主理想整环**（Principal Ideal Domain, PID）: 若 $R$ 的每个理想都是主理想，即对任意理想 $I \subseteq R$，存在 $a \in R$ 使得 $I = (a)$。

3. **唯一分解整环**（Unique Factorization Domain, UFD）: 若 $R$ 中每个非零非单位元 $r$ 都可以（在相伴意义下）唯一地分解为不可约元的乘积：
   $$r = u \cdot p_1^{e_1} p_2^{e_2} \cdots p_k^{e_k},$$
   其中 $u$ 是单位，$p_i$ 是互不相伴的不可约元，$e_i \geq 1$。

**关系链**: ED $\Rightarrow$ PID $\Rightarrow$ UFD。后文将证明 $F[x]$ 是 ED，从而 $F[x]$ 也是 PID 和 UFD。

> **双语对照**:
>
> ```lean4
> -- 欧几里得整环
> class EuclideanDomain (R : Type*) extends CommRing R, IsDomain R where
>   degree : R → ℕ
>   degree_lt_mul_right : ∀ a b, b ≠ 0 → a ≠ 0 → degree a ≤ degree (a * b)
>   div : R → R → R
>   mod : R → R → R
>   mod_add_div : ∀ a b, b ≠ 0 → mod a b + b * div a b = a
>   mod_lt : ∀ a b, b ≠ 0 → degree (mod a b) < degree b
>
> -- 主理想整环
> class IsPrincipalIdealRing (R : Type*) [Ring R] : Prop where
>   principal : ∀ (I : Ideal R), I.IsPrincipal
>
> -- 唯一分解整环
> class UniqueFactorizationMonoid (α : Type*) [CancelCommMonoidWithZero α] : Prop where
>   irreducible_iff_prime : ∀ {a : α}, Irreducible a ↔ Prime a
> ```

## 二、核心定理与完整证明

### 定理 2.1（多项式环的带余除法）

**定理陈述**: 设 $F$ 为域，$f(x), g(x) \in F[x]$，$g \neq 0$。则存在唯一的多项式 $q(x), r(x) \in F[x]$ 使得
$$f(x) = q(x)g(x) + r(x),$$
其中 $r = 0$ 或 $\deg(r) < \deg(g)$。

**证明**:

**存在性**: 对 $\deg(f)$ 进行归纳。

- **基例**: 若 $f = 0$，取 $q = 0, r = 0$。若 $\deg(f) < \deg(g)$，取 $q = 0, r = f$。

- **归纳步骤**: 设 $\deg(f) = n \geq m = \deg(g)$。令
  $$f(x) = a_n x^n + \cdots, \qquad g(x) = b_m x^m + \cdots.$$
  构造
  $$f_1(x) \;:=\; f(x) - \frac{a_n}{b_m} x^{n-m} g(x).$$
  这是良定义的，因为 $b_m \neq 0$ 且 $F$ 是域，$a_n/b_m \in F$。

  $f_1$ 的首项 $a_n x^n$ 被消去，故 $\deg(f_1) < n = \deg(f)$。由归纳假设，存在 $q_1, r$ 使得
  $$f_1 = q_1 g + r, \quad r = 0 \text{ 或 } \deg(r) < \deg(g).$$

  于是
  $$f = f_1 + \frac{a_n}{b_m} x^{n-m} g = \left(q_1 + \frac{a_n}{b_m} x^{n-m}\right) g + r.$$
  令 $q = q_1 + \dfrac{a_n}{b_m} x^{n-m}$，即得所求。

**唯一性**: 设 $f = qg + r = q'g + r'$，其中 $r, r'$ 满足次数条件。则
$$(q - q')g = r' - r.$$
若 $q \neq q'$，则左边 $\deg((q - q')g) = \deg(q - q') + \deg(g) \geq \deg(g)$。但右边 $\deg(r' - r) \leq \max(\deg(r'), \deg(r)) < \deg(g)$，矛盾。因此 $q = q'$，从而 $r = r'$。 $\square$

> **双语对照**:
>
> ```lean4
> -- Mathlib4 中带余除法由 Polynomial.divModByMonic 提供
> open Polynomial
> variable {F : Type*} [Field F] {f g : F[X]} (hg : g ≠ 0)
>
> theorem division_algorithm : ∃ q r, f = q * g + r ∧ (r = 0 ∨ degree r < degree g) := by
>   by_cases hf : f = 0
>   · use 0, 0; simp [hf]
>   · -- 对 degree f 进行归纳
>     generalize hd : degree f = d
>     induction' d using Nat.strongRecOn with d ih generalizing f
>     by_cases hfg : degree f < degree g
>     · use 0, f; simp [hfg]
>     · -- 消去首项
>       have h_le : degree g ≤ degree f := by linarith
>       let c := leadingCoeff f / leadingCoeff g
>       let n := (degree f - degree g).natAbs
>       let f₁ := f - C c * X ^ n * g
>       have hf₁_lt : degree f₁ < degree f := by
>         -- 验证首项被消去
>         sorry
>       rcases ih (degree f₁) (by omega) f₁ (by rfl) with ⟨q₁, r, h_eq, hr⟩
>       use q₁ + C c * X ^ n, r
>       constructor
>       · rw [h_eq]
>         ring
>       · exact hr
> ```

### 定理 2.2（$F[x]$ 是欧几里得整环）

**定理陈述**: 设 $F$ 为域。则 $F[x]$ 是欧几里得整环，其欧几里得度为 $N(f) = \deg(f)$。

**证明**: 由**定理 2.1**，对任意 $f, g \in F[x]$，$g \neq 0$，存在 $q, r$ 使得 $f = qg + r$，其中 $r = 0$ 或 $\deg(r) < \deg(g)$。这正是欧几里得整环的定义，取 $N(f) = \deg(f)$ 即可。 $\square$

> **双语对照**:
>
> ```lean4
> instance {F : Type*} [Field F] : EuclideanDomain F[X] where
>   degree := fun f => if f = 0 then 0 else f.degree.natSucc
>   -- 度函数：0 映射到 0，非零映射到 degree + 1（保证正值）
>   degree_lt_mul_right := by
>     intro a b hb ha
>     simp [ha, hb]
>     rw [degree_mul]
>     exact Nat.le_add_right _ _
>   div := fun a b => if b = 0 then 0 else (a / b : F[X])
>   mod := fun a b => if b = 0 then a else (a % b : F[X])
>   -- 带余除法的存在性由 Polynomial.modByMonic_add_div 保证
>   mod_add_div := sorry -- 需要基于 monic 的完整除法算法
>   mod_lt := sorry
> ```

### 定理 2.3（ED $\Rightarrow$ PID）

**定理陈述**: 每个欧几里得整环都是主理想整环。

**证明**: 设 $R$ 为欧几里得整环，度函数为 $N$，$I \subseteq R$ 为非零理想。取 $I$ 中使 $N(b)$ 最小的非零元素 $b \in I$（良序原理保证存在性）。

断言 $I = (b)$。显然 $(b) \subseteq I$（因为 $b \in I$）。反之，设 $a \in I$。由带余除法，存在 $q, r \in R$ 使得
$$a = qb + r, \quad r = 0 \text{ 或 } N(r) < N(b).$$

由于 $r = a - qb \in I$（因为 $a, b \in I$），若 $r \neq 0$，则 $N(r) < N(b)$ 与 $b$ 的选取矛盾。因此 $r = 0$，$a = qb \in (b)$。故 $I \subseteq (b)$。

综上，$I = (b)$ 是主理想。 $\square$

---

### 定理 2.4（PID $\Rightarrow$ UFD）

**定理陈述**: 每个主理想整环都是唯一分解整环。

**证明**: 需要证明分解的**存在性**和**唯一性**。

**存在性**: 设 $R$ 为 PID，$a \in R$ 非零非单位。若 $a$ 不可约，则分解完成。若 $a$ 可约，则 $a = a_1 b_1$，其中 $a_1, b_1$ 均非单位。于是 $(a) \subsetneq (a_1)$（因为 $a_1 \notin (a)$，否则 $a_1 = ac$，代入得 $a = acb_1$，由整环消去律 $1 = cb_1$，$b_1$ 为单位，矛盾）。

若 $a_1$ 仍不可约，继续分解。这样得到理想的升链：
$$(a) \subsetneq (a_1) \subsetneq (a_2) \subsetneq \cdots$$

**关键引理**: PID 中理想的升链 stabilize。设 $I = \bigcup_{i} (a_i)$，则 $I$ 是理想（PID 中任意理想的并仍是理想）。因为 $R$ 是 PID，$I = (d)$。由 $d \in I$，存在 $n$ 使得 $d \in (a_n)$，故 $(d) \subseteq (a_n) \subseteq (a_{n+1}) \subseteq \cdots \subseteq (d)$。因此 $(a_n) = (a_{n+1}) = \cdots$，链 stabilize。

因此分解过程必在有限步终止，存在性得证。

**唯一性**: 先证 PID 中不可约元是素元。设 $p$ 不可约，$p \mid ab$。考虑理想 $(p, a)$。因为 $R$ 是 PID，$(p, a) = (d)$。由 $p \in (d)$，$d \mid p$。因为 $p$ 不可约，$d$ 是单位或 $d$ 与 $p$ 相伴。

- 若 $d$ 是单位，则 $(p, a) = R$，存在 $x, y \in R$ 使得 $xp + ya = 1$。两边乘 $b$ 得 $xpb + yab = b$。因为 $p \mid ab$，$p \mid yab$，又 $p \mid xpb$，故 $p \mid b$。
- 若 $d$ 与 $p$ 相伴，则 $(p, a) = (p)$，故 $a \in (p)$，$p \mid a$。

因此 $p$ 是素元。在整环中，素因子分解的唯一性是标准结论（通过归纳和素元的定义直接得到）。 $\square$

> **双语对照**:
>
> ```lean4
> -- PID ⇒ UFD 在 Mathlib4 中的实例链
> instance {R : Type*} [CommRing R] [IsDomain R] [IsPrincipalIdealRing R] :
>     UniqueFactorizationMonoid R := by
>   infer_instance -- Mathlib 已内置此推导
>
> -- 多项式环是 UFD
> instance {F : Type*} [Field F] : UniqueFactorizationMonoid F[X] := by
>   have : IsPrincipalIdealRing F[X] := by
>     infer_instance -- F[X] 是 ED ⇒ PID
>   infer_instance -- PID ⇒ UFD
> ```

### 推论 2.5（$F[x]$ 中的最大公因式）

**定理陈述**: 设 $F$ 为域，$f, g \in F[x]$ 不全为零。则存在唯一的首一多项式 $d(x) \in F[x]$ 使得：

1. $d \mid f$ 且 $d \mid g$；
2. 若 $h \mid f$ 且 $h \mid g$，则 $h \mid d$。

此 $d$ 称为 $f$ 和 $g$ 的**最大公因式**（greatest common divisor），记作 $\gcd(f, g)$。进一步，存在 $u, v \in F[x]$ 使得
$$d = uf + vg$$
（Bezout 等式）。

**证明**: 考虑理想 $I = (f, g) = \{af + bg \mid a, b \in F[x]\}$。因为 $F[x]$ 是 PID，$I = (d)$ 对某个 $d \in F[x]$。取 $d$ 的首一版本（乘以适当的单位，即 $F$ 中非零常数）。

- 因为 $f, g \in (d)$，$d \mid f$ 且 $d \mid g$。
- 若 $h \mid f$ 且 $h \mid g$，则 $h \mid (af + bg)$ 对所有 $a, b \in F[x]$。特别地，$h \mid d$（因为 $d \in (f, g)$）。

Bezout 等式直接由 $d \in (f, g)$ 得到。 $\square$

## 三、典型例题

### 例题 3.1（$\mathbb{Q}[x]$ 和 $\mathbb{F}_5[x]$ 中分解 $x^4 - 1$）

**题目**: 分别在 $\mathbb{Q}[x]$ 和 $\mathbb{F}_5[x]$ 中将 $x^4 - 1$ 分解为不可约多项式的乘积。

**解答**:

**在 $\mathbb{Q}[x]$ 中**:

首先用平方差公式：
$$x^4 - 1 = (x^2 - 1)(x^2 + 1) = (x - 1)(x + 1)(x^2 + 1).$$

$x - 1$ 和 $x + 1$ 是一次式，显然不可约。$x^2 + 1$ 在 $\mathbb{Q}[x]$ 中不可约：若可约，则必有有理根 $r/s$（既约形式），满足 $(r/s)^2 + 1 = 0$，即 $r^2 + s^2 = 0$。在 $\mathbb{Q}$ 中这要求 $r = s = 0$，矛盾。或者由 Eisenstein 判别法的变体：$x^2 + 1$ 没有有理根（$\pm 1$ 都不是根），故不可约。

因此 $\mathbb{Q}[x]$ 中的分解为：
$$x^4 - 1 = (x - 1)(x + 1)(x^2 + 1).$$

**在 $\mathbb{F}_5[x]$ 中**:

同样在 $\mathbb{F}_5$ 中：
$$x^4 - 1 = (x^2 - 1)(x^2 + 1) = (x - 1)(x + 1)(x^2 + 1).$$

但 $x^2 + 1$ 在 $\mathbb{F}_5$ 中是否可约？检查：在 $\mathbb{F}_5$ 中，$x^2 + 1 = 0 \iff x^2 = -1 = 4 \iff x = \pm 2$（因为 $2^2 = 4$，$3^2 = 9 = 4$）。

因此 $x^2 + 1 = (x - 2)(x + 2) = (x - 2)(x - 3)$ 在 $\mathbb{F}_5[x]$ 中。

完整分解为：
$$x^4 - 1 = (x - 1)(x + 1)(x - 2)(x - 3) \quad \text{在 } \mathbb{F}_5[x] \text{ 中}.$$

注意 $x^4 - 1$ 在 $\mathbb{F}_5$ 中还有另一种看法：由 Fermat 小定理，$\mathbb{F}_5^\times$ 是 4 阶循环群，故所有非零元都是 $x^4 - 1$ 的根。这也验证了上述分解。 $\square$

**关键步骤解析**: 本题展示了可约性与基域的密切关系。同一多项式在不同域上可能有完全不同的分解行为。在 $\mathbb{F}_5$ 中，$x^2 + 1$ "分裂"为一次因子，因为 $-1$ 是模 5 的二次剩余。

### 例题 3.2（求 $\gcd(x^3 + 2x^2 + x, x^2 - 1)$ 的欧几里得算法）

**题目**: 在 $\mathbb{Q}[x]$ 中，利用欧几里得算法求 $f(x) = x^3 + 2x^2 + x$ 和 $g(x) = x^2 - 1$ 的最大公因式，并表示为 Bezout 等式。

**解答**:

**第一步：分解观察。**

$f(x) = x(x^2 + 2x + 1) = x(x + 1)^2$，$g(x) = (x - 1)(x + 1)$。

由观察，$\gcd(f, g) = x + 1$。下面用欧几里得算法严格计算。

**第二步：执行欧几里得算法。**

- $f = x^3 + 2x^2 + x$，$g = x^2 - 1$。
- 用 $f$ 除以 $g$：
  $$x^3 + 2x^2 + x = (x + 2)(x^2 - 1) + (2x + 2).$$
  验证：$(x + 2)(x^2 - 1) = x^3 + 2x^2 - x - 2$，$f - (x + 2)g = x^3 + 2x^2 + x - (x^3 + 2x^2 - x - 2) = 2x + 2$。
  余式 $r_1 = 2x + 2$，$\deg(r_1) = 1 < 2 = \deg(g)$。

- 用 $g$ 除以 $r_1$：
  $$x^2 - 1 = \frac{1}{2}(x - 1)(2x + 2) + 0.$$
  验证：$\dfrac{1}{2}(x - 1)(2x + 2) = (x - 1)(x + 1) = x^2 - 1$。

余式为零，算法终止。最后一个非零余式是 $2x + 2$。取首一版本得
$$\gcd(f, g) = x + 1.$$

**第三步：回代求 Bezout 等式。**

由第一步：
$$2x + 2 = f - (x + 2)g.$$

两边除以 2：
$$x + 1 = \frac{1}{2}f - \frac{1}{2}(x + 2)g = \frac{1}{2}f + \left(-\frac{x}{2} - 1\right)g.$$

因此
$$\gcd(f, g) = \frac{1}{2} \cdot f + \left(-\frac{x}{2} - 1\right) \cdot g. \quad \square$$

## 四、常见误区与纠正

### 误区 4.1: "$\mathbb{Z}[x]$ 是欧几里得整环"

**错误观点**: 有学生认为：既然 $\mathbb{Z}$ 是欧几里得整环（度为绝对值），$F[x]$ 也是欧几里得整环（度为次数），那么 $\mathbb{Z}[x]$ 也应该是欧几里得整环。

**纠正**: $\mathbb{Z}[x]$ **不是**欧几里得整环，甚至**不是**主理想整环（PID）。但 $\mathbb{Z}[x]$ **是**唯一分解整环（UFD）。

**证明 $\mathbb{Z}[x]$ 不是 PID**: 考虑理想 $I = (2, x) = \{2a(x) + xb(x) \mid a, b \in \mathbb{Z}[x]\}$。假设 $I = (f(x))$ 是主理想。则 $2 \in (f)$，故 $f \mid 2$。在 $\mathbb{Z}[x]$ 中，$2$ 的因子只有 $\pm 1, \pm 2$（因为 $2$ 是常数）。

- 若 $f = \pm 1$，则 $I = \mathbb{Z}[x]$，但 $1 \notin I$（因为 $I$ 中元素的常数项都是偶数）。
- 若 $f = \pm 2$，则 $I = (2)$，但 $x \in I$ 而 $x \notin (2)$。

因此 $I$ 不是主理想，$\mathbb{Z}[x]$ 不是 PID，从而不是 ED。

**$\mathbb{Z}[x]$ 是 UFD**: 这是一个经典定理（Gauss 引理）：若 $R$ 是 UFD，则 $R[x]$ 也是 UFD。因为 $\mathbb{Z}$ 是 UFD，$\mathbb{Z}[x]$ 是 UFD。类似地，$F[x_1, \ldots, x_n]$（$F$ 为域）也是 UFD，但 $n \geq 2$ 时不是 PID。

> **为什么错**: 这一误区的根源在于忽视了带余除法对系数环的要求。$F[x]$ 中的带余除法依赖于首项系数的可逆性（需要除以 $b_m$）。在 $\mathbb{Z}[x]$ 中，除法后系数可能不再是整数（例如 $x^2$ 除以 $2x$ 得 $\dfrac{1}{2}x$）。因此次数函数不能作为 $\mathbb{Z}[x]$ 的欧几里得度。

## 五、思想方法提炼

**本章核心思想**:

1. **带余除法是算法之源**: 多项式环 $F[x]$ 中的带余除法是欧几里得算法、Bezout 等式、唯一分解的根基。它使得 $F[x]$ 具备了与整数 $\mathbb{Z}$ 完全平行的算术结构。

2. **理想升链与有限性**: PID $\Rightarrow$ UFD 的证明中，理想的升链条件（ACC）保证了分解过程的终止。这是代数中"有限性论证"的典型范式——通过反证法和升链条件排除无限递降/递升的可能性。

3. **基域决定可约性**: 多项式的可约性不是内禀性质，而是相对于系数域而言的。这揭示了"基域扩张"的重要性：通过扩域，可以使不可约多项式分裂（如 $\mathbb{R}[x]$ 中 $x^2 + 1$ 在 $\mathbb{C}$ 中分裂）。

**与后续章节的联系**:

- **域扩张**: 单代数扩张 $F(\alpha)$ 的结构由 $\alpha$ 的极小多项式决定，该多项式在 $F[x]$ 中不可约。第一同构定理给出 $F[x]/(m_\alpha(x)) \cong F(\alpha)$。
- **Galois 理论**: 多项式的分裂域、Galois 群的结构都依赖于其不可约因子的性质。
- **代数几何**: $\mathbb{C}[x_1, \ldots, x_n]$ 中的理想与仿射簇的对应（Hilbert 零点定理）以 UFD 性质为基础。

## 六、习题

### 习题 6.1

**题目**: 在 $\mathbb{Q}[x]$ 中，用带余除法求 $x^4 + 2x^3 - x + 1$ 除以 $x^2 + x + 1$ 的商和余式。

**提示**: 按次数逐项消去，注意系数的符号。

**解答**:

$x^4 + 2x^3 - x + 1$ 除以 $x^2 + x + 1$：

- $x^4 / x^2 = x^2$，$x^2(x^2 + x + 1) = x^4 + x^3 + x^2$。
  余：$(x^4 + 2x^3 - x + 1) - (x^4 + x^3 + x^2) = x^3 - x^2 - x + 1$。

- $x^3 / x^2 = x$，$x(x^2 + x + 1) = x^3 + x^2 + x$。
  余：$(x^3 - x^2 - x + 1) - (x^3 + x^2 + x) = -2x^2 - 2x + 1$。

- $-2x^2 / x^2 = -2$，$-2(x^2 + x + 1) = -2x^2 - 2x - 2$。
  余：$(-2x^2 - 2x + 1) - (-2x^2 - 2x - 2) = 3$。

因此
$$x^4 + 2x^3 - x + 1 = (x^2 + x - 2)(x^2 + x + 1) + 3. \quad \square$$

### 习题 6.2

**题目**: 在 $\mathbb{Q}[x]$ 中，求 $\gcd(x^3 - 1, x^4 + x^2 + 1)$，并表示为 Bezout 等式。

**提示**: 先分解 $x^3 - 1 = (x - 1)(x^2 + x + 1)$，再用欧几里得算法。

**解答**: $f = x^3 - 1$，$g = x^4 + x^2 + 1$。

- $g = x \cdot f + (x^2 + x + 1)$。验证：$xf = x^4 - x$，$g - xf = x^4 + x^2 + 1 - (x^4 - x) = x^2 + x + 1$。
- $f = (x - 1)(x^2 + x + 1) + 0$。验证：$(x - 1)(x^2 + x + 1) = x^3 - 1 = f$。

余式为零，$\gcd(f, g) = x^2 + x + 1$（已是首一）。

回代：$x^2 + x + 1 = g - xf = (-x)f + 1 \cdot g$。 $\square$

### 习题 6.3

**题目**: 证明 $x^3 - 2$ 在 $\mathbb{Q}[x]$ 中不可约。

**提示**: 使用有理根定理：若 $f(x) = a_nx^n + \cdots + a_0 \in \mathbb{Z}[x]$ 有有理根 $p/q$（既约），则 $p \mid a_0$，$q \mid a_n$。

**解答**: $x^3 - 2$ 是三次多项式。若可约，则必有一次因子，即存在有理根 $p/q$（既约）。由有理根定理，$p \mid 2$，$q \mid 1$。可能的有理根为 $\pm 1, \pm 2$。

验证：$(\pm 1)^3 - 2 = -1 \neq 0$，$(\pm 2)^3 - 2 = 6 \neq 0$。因此 $x^3 - 2$ 无有理根，在 $\mathbb{Q}[x]$ 中不可约。 $\square$

### 习题 6.4

**题目**: 在 $\mathbb{F}_2[x]$ 中，将 $x^4 + x^2 + 1$ 分解为不可约多项式的乘积。

**提示**: $\mathbb{F}_2[x]$ 中一次不可约多项式只有 $x$ 和 $x + 1$；二次不可约多项式需要逐一检验。

**解答**: $\mathbb{F}_2[x]$ 中次数 $\leq 2$ 的多项式：

- 一次：$x, x + 1$（均不可约）；
- 二次：$x^2, x^2 + 1 = (x + 1)^2, x^2 + x = x(x + 1), x^2 + x + 1$（最后一个无根：$0^2 + 0 + 1 = 1 \neq 0$，$1^2 + 1 + 1 = 1 \neq 0$），故 $x^2 + x + 1$ 不可约。

检验 $x^4 + x^2 + 1$：

- $x = 0$：值为 $1 \neq 0$，故 $x \nmid (x^4 + x^2 + 1)$；
- $x = 1$：值为 $1 + 1 + 1 = 1 \neq 0$（在 $\mathbb{F}_2$ 中），故 $(x + 1) \nmid (x^4 + x^2 + 1)$。

尝试分解为两个二次式的乘积。在 $\mathbb{F}_2[x]$ 中，二次不可约式只有 $x^2 + x + 1$。

$$(x^2 + x + 1)^2 = x^4 + x^2 + 1 + 2(x^3 + x^2 + x) = x^4 + x^2 + 1$$
（因为在 $\mathbb{F}_2$ 中 $2 = 0$）。

因此 $x^4 + x^2 + 1 = (x^2 + x + 1)^2$ 在 $\mathbb{F}_2[x]$ 中。 $\square$

### 习题 6.5

**题目**: 证明 $\mathbb{R}[x]/(x^2 - 2) \cong \mathbb{R}[\sqrt{2}] := \{a + b\sqrt{2} \mid a, b \in \mathbb{R}\}$。

**提示**: 构造代入同态 $x \mapsto \sqrt{2}$，利用第一同构定理。

**解答**: 定义 $\varphi : \mathbb{R}[x] \to \mathbb{R}$ 为 $\varphi(f) = f(\sqrt{2})$。这是环同态（赋值同态）。

- **像**: 对任意 $a + b\sqrt{2} \in \mathbb{R}[\sqrt{2}]$，取 $f(x) = a + bx$，则 $\varphi(f) = a + b\sqrt{2}$。故 $\operatorname{im}\varphi = \mathbb{R}[\sqrt{2}]$。
- **核**: $f \in \ker\varphi \iff f(\sqrt{2}) = 0$。因为 $x^2 - 2$ 是 $\sqrt{2}$ 在 $\mathbb{R}$ 上的极小多项式（在 $\mathbb{R}[x]$ 中不可约，因为根为 $\pm\sqrt{2} \notin \mathbb{R}$ 的代数结构使得 $x^2 - 2$ 无一次因子），$\ker\varphi = (x^2 - 2)$。

由环第一同构定理，$\mathbb{R}[x]/(x^2 - 2) \cong \mathbb{R}[\sqrt{2}]$。 $\square$

### 习题 6.6

**题目**: 设 $F$ 为域，$f(x) \in F[x]$ 为 $n$ 次多项式。证明 $f$ 在 $F[x]$ 中最多有 $n$ 个根（计重数）。

**提示**: 若 $a$ 是根，用带余除法证明 $(x - a) \mid f(x)$，然后对商归纳。

**解答**: 对 $n$ 归纳。

- $n = 0$：常数多项式非零时无根。
- 归纳步骤：设 $f$ 有根 $a \in F$。由带余除法，$f(x) = q(x)(x - a) + r$，其中 $r$ 为常数（因为 $\deg(x - a) = 1$）。代入 $x = a$ 得 $0 = f(a) = r$，故 $r = 0$，$f(x) = q(x)(x - a)$。

$\deg(q) = n - 1$。由归纳假设，$q$ 最多有 $n - 1$ 个根。$f$ 的根是 $a$ 加上 $q$ 的根，故最多有 $n$ 个根。 $\square$

### 习题 6.7

**题目**: 设 $R$ 为整环。证明 $R[x]$ 中单位恰好是 $R$ 中的单位（即常数单位）。

**提示**: 利用次数公式 $\deg(fg) = \deg(f) + \deg(g)$。

**解答**: 若 $f \in R[x]$ 是单位，则存在 $g \in R[x]$ 使得 $fg = 1$。取次数：$\deg(f) + \deg(g) = \deg(1) = 0$。因为次数非负，$\deg(f) = \deg(g) = 0$。故 $f, g$ 都是常数多项式，且 $fg = 1$ 在 $R$ 中，即 $f$ 是 $R$ 中的单位。

反之，$R$ 中的单位显然是 $R[x]$ 中的单位。 $\square$

### 习题 6.8

**题目**: 证明 Wilson 定理：若 $p$ 为素数，则 $(p - 1)! \equiv -1 \pmod{p}$。

**提示**: 在 $\mathbb{F}_p[x]$ 中考虑多项式 $x^{p-1} - 1$，利用 Fermat 小定理和根与因子的关系。

**解答**: 在 $\mathbb{F}_p[x]$ 中，由 Fermat 小定理，对任意 $a \in \mathbb{F}_p^\times$ 有 $a^{p-1} = 1$。因此 $\mathbb{F}_p^\times$ 的 $p - 1$ 个元素都是 $x^{p-1} - 1$ 的根。

由习题 6.6，$x^{p-1} - 1$ 最多有 $p - 1$ 个根。而 $\mathbb{F}_p^\times$ 恰好提供了 $p - 1$ 个不同的根 $1, 2, \ldots, p - 1$。因此
$$x^{p-1} - 1 = \prod_{a=1}^{p-1} (x - a) \quad \text{在 } \mathbb{F}_p[x] \text{ 中}.$$

代入 $x = 0$：
$$0^{p-1} - 1 = -1 = \prod_{a=1}^{p-1} (0 - a) = (-1)^{p-1} \prod_{a=1}^{p-1} a = (-1)^{p-1} (p - 1)!.$$

在 $\mathbb{F}_p$ 中，若 $p = 2$，$(-1)^1 = -1 = 1$，$(2 - 1)! = 1 = -1 \pmod{2}$，成立。

若 $p > 2$，$p - 1$ 为偶数，$(-1)^{p-1} = 1$，故 $(p - 1)! = -1$ 在 $\mathbb{F}_p$ 中，即 $(p - 1)! \equiv -1 \pmod{p}$。 $\square$

### 习题 6.9

**题目**: 设 $f(x) = x^n + a_{n-1}x^{n-1} + \cdots + a_0 \in \mathbb{Z}[x]$ 为首一多项式。证明：若 $f$ 有有理根 $r/s$（既约），则 $s = \pm 1$，即有理根必为整数，且该整数整除 $a_0$。

**提示**: 代入 $x = r/s$，两边乘 $s^n$，分析整除性。

**解答**: 设 $f(r/s) = 0$，$\gcd(r, s) = 1$，$s > 0$。则
$$\left(\frac{r}{s}\right)^n + a_{n-1}\left(\frac{r}{s}\right)^{n-1} + \cdots + a_0 = 0.$$

两边乘 $s^n$：
$$r^n + a_{n-1}r^{n-1}s + \cdots + a_0s^n = 0.$$

整理：$r^n = -s(a_{n-1}r^{n-1} + \cdots + a_0s^{n-1})$。因此 $s \mid r^n$。因为 $\gcd(r, s) = 1$，$\gcd(r^n, s) = 1$，故 $s = 1$。

因此有理根必为整数 $r$。由 $r^n + a_{n-1}r^{n-1} + \cdots + a_0 = 0$，
$$r(r^{n-1} + a_{n-1}r^{n-2} + \cdots + a_1) = -a_0,$$
故 $r \mid a_0$。 $\square$

### 习题 6.10

**题目**: 设 $F$ 为域，$f, g \in F[x]$ 不全为零。证明 $F[x]$ 中 $f$ 和 $g$ 的 gcd 在相伴意义下唯一，且存在唯一的首一 gcd。

**提示**: 设 $d_1, d_2$ 都是 gcd，利用 gcd 定义中的互相整除性。

**解答**: 设 $d_1, d_2$ 都是 $f$ 和 $g$ 的 gcd。由定义：

- $d_1 \mid f$ 且 $d_1 \mid g$，且 $d_2$ 是 gcd，故 $d_1 \mid d_2$；
- $d_2 \mid f$ 且 $d_2 \mid g$，且 $d_1$ 是 gcd，故 $d_2 \mid d_1$。

因此 $d_1 = u d_2$ 对某个单位 $u \in F[x]$。由习题 6.7，$F[x]$ 中的单位是非零常数。故 $d_1$ 与 $d_2$ 相伴。

设 $d$ 是一个 gcd，首项系数为 $c \neq 0$。令 $d' = c^{-1}d$，则 $d'$ 首一，且 $d'$ 与 $d$ 相伴，故 $d'$ 也是 gcd。若 $d''$ 是另一个首一 gcd，则 $d'' = u d'$ 对某个单位 $u$。比较首项系数：$1 = u \cdot 1$，故 $u = 1$，$d'' = d'$。因此首一 gcd 唯一。 $\square$
