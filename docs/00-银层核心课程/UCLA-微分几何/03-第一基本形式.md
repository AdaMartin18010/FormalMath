# 第一基本形式（First Fundamental Form）

> **课程**: UCLA 微分几何 | **章节**: Ch. 2 — 曲面的度量结构
> **学习目标**: 掌握第一基本形式的定义与系数计算；理解内蕴几何的含义；能够计算曲面上曲线的弧长和区域的面积

---

## 一、核心定义

### 定义 3.1（第一基本形式 / First Fundamental Form）

**严格陈述**: 设 \(S\) 为正则曲面，\(p \in S\)。定义 \(p\) 处的**第一基本形式** \(I_p: T_pS \times T_pS \to \mathbb{R}\) 为切平面上的内积：

$$I_p(v, w) = \langle v, w \rangle_{\mathbb{R}^3}, \qquad v, w \in T_pS.$$

在参数化 \(\mathbf{x}(u, v)\) 下，记

$$E = \langle \mathbf{x}_u, \mathbf{x}_u \rangle, \quad F = \langle \mathbf{x}_u, \mathbf{x}_v \rangle, \quad G = \langle \mathbf{x}_v, \mathbf{x}_v \rangle.$$

则对 \(v = a\mathbf{x}_u + b\mathbf{x}_v\)，\(w = c\mathbf{x}_u + d\mathbf{x}_v\)，

$$I_p(v, w) = Eac + F(ad + bc) + Gbd.$$

矩阵形式为

$$I = \begin{pmatrix} E & F \\ F & G \end{pmatrix}.$$

**直观解释**: 第一基本形式是曲面从周围三维欧氏空间"继承"来的度量结构。它告诉我们如何在曲面上测量长度、角度和面积——而这一切"只依赖于曲面本身"，不依赖于曲面如何嵌入三维空间。系数 \(E, F, G\) 完全刻画了曲面的局部度量性质。

---

### 定义 3.2（弧长元与面积元）

**严格陈述**: 在参数化 \(\mathbf{x}(u, v)\) 下：

- **弧长元**（line element）：
  $$ds^2 = E\, du^2 + 2F\, du\, dv + G\, dv^2.$$

- **面积元**（area element）：
  $$dA = \sqrt{EG - F^2}\, du\, dv = |\mathbf{x}_u \times \mathbf{x}_v|\, du\, dv.$$

**直观解释**: 弧长元 \(ds^2\) 是勾股定理在曲面上的推广——曲面上无穷小位移 \((du, dv)\) 的实际长度平方由第一基本形式给出。面积元则是"无穷小平行四边形"的面积，由叉积的模长给出。注意 \(EG - F^2 = |\mathbf{x}_u \times \mathbf{x}_v|^2 > 0\)（正则性条件），保证了面积元的良定性。

---

### 定义 3.3（内蕴几何 / Intrinsic Geometry）

**严格陈述**: 曲面的一个几何性质或量称为**内蕴的**（intrinsic），若它可以仅由第一基本形式及其导数表达，而不依赖于曲面在 \(\mathbb{R}^3\) 中的具体嵌入方式。

**直观解释**: 内蕴几何是"曲面上的居民"能够感知到的几何——不需要离开曲面去三维空间中测量。例如，曲面上曲线的弧长、两条曲线的夹角、区域的面积都是内蕴的。Gauss 的绝妙定理（Theorema Egregium）断言：Gauss 曲率也是内蕴的——这是微分几何中最惊人的发现之一。

> **双语对照**:
>
> ```lean4
> import Mathlib
>
> -- 第一基本形式的系数
> structure FirstFundamentalForm where
>   E : ℝ → ℝ → ℝ
>   F : ℝ → ℝ → ℝ
>   G : ℝ → ℝ → ℝ
>   positive_def : ∀ u v, E u v > 0 ∧ E u v * G u v - F u v ^ 2 > 0
>
> -- 面积元
> def areaElement (I : FirstFundamentalForm) (u v : ℝ) : ℝ :=
>   Real.sqrt (I.E u v * I.G u v - I.F u v ^ 2)
>

> def curveLength (I : FirstFundamentalForm) (u v : ℝ → ℝ) (t₀ t₁ : ℝ) : ℝ :=
>   ∫ t in t₀..t₁, Real.sqrt (I.E (u t) (v t) * (deriv u t)^2 +
>     2 * I.F (u t) (v t) * deriv u t * deriv v t +
>     I.G (u t) (v t) * (deriv v t)^2)
> ```

## 二、核心定理与完整证明

### 定理 3.1（弧长与面积公式）

**定理陈述**: 设 \(\mathbf{x}(u, v)\) 为正则曲面的参数化，\(I = (E, F, G)\) 为其第一基本形式。

1. 若 \(\gamma(t) = \mathbf{x}(u(t), v(t))\)（\(t \in [a, b]\)）为曲面上的曲线，则其弧长为
   $$L(\gamma) = \int_a^b \sqrt{E(u')^2 + 2Fu'v' + G(v')^2}\, dt.$$

2. 若 \(R \subseteq U\) 为参数域中的区域，则 \(\mathbf{x}(R)\) 的面积为
   $$A(\mathbf{x}(R)) = \iint_R \sqrt{EG - F^2}\, du\, dv.$$

**证明**:

**(1) 弧长公式**: \(\gamma'(t) = u'(t)\mathbf{x}_u + v'(t)\mathbf{x}_v\)。由第一基本形式的定义，

$$|\gamma'(t)|^2 = \langle \gamma'(t), \gamma'(t) \rangle = E(u')^2 + 2Fu'v' + G(v')^2.$$

弧长 \(L = \int_a^b |\gamma'(t)|\, dt\)，代入即得。

**(2) 面积公式**: 在参数域中考虑无穷小矩形 \([u, u+du] \times [v, v+dv]\)。它在曲面上映射为以 \(\mathbf{x}_u du\) 和 \(\mathbf{x}_v dv\) 为边的平行四边形，其面积为

$$|\mathbf{x}_u du \times \mathbf{x}_v dv| = |\mathbf{x}_u \times \mathbf{x}_v|\, du\, dv = \sqrt{EG - F^2}\, du\, dv.$$

对区域 \(R\) 积分即得总面积。\(\square\)

---

### 定理 3.2（第一基本形式在坐标变换下的规律）

**定理陈述**: 设 \(\mathbf{x}(u, v)\) 和 \(\tilde{\mathbf{x}}(\tilde{u}, \tilde{v})\) 为曲面的两个参数化，坐标变换为 \((u, v) = \Phi(\tilde{u}, \tilde{v})\)，Jacobi 矩阵为 \(J = d\Phi\)。设两参数化下的第一基本形式矩阵分别为 \(I\) 和 \(\tilde{I}\)。则

$$\tilde{I} = J^T I J.$$

**证明**: 由链式法则，

$$\tilde{\mathbf{x}}_{\tilde{u}} = \frac{\partial u}{\partial \tilde{u}} \mathbf{x}_u + \frac{\partial v}{\partial \tilde{u}} \mathbf{x}_v, \quad \tilde{\mathbf{x}}_{\tilde{v}} = \frac{\partial u}{\partial \tilde{v}} \mathbf{x}_u + \frac{\partial v}{\partial \tilde{v}} \mathbf{x}_v.$$

即 \((\tilde{\mathbf{x}}_{\tilde{u}}, \tilde{\mathbf{x}}_{\tilde{v}}) = (\mathbf{x}_u, \mathbf{x}_v) J\)。因此

$$\tilde{I} = \begin{pmatrix} \tilde{\mathbf{x}}_{\tilde{u}} \\ \tilde{\mathbf{x}}_{\tilde{v}} \end{pmatrix} \begin{pmatrix} \tilde{\mathbf{x}}_{\tilde{u}} & \tilde{\mathbf{x}}_{\tilde{v}} \end{pmatrix} = J^T \begin{pmatrix} \mathbf{x}_u \\ \mathbf{x}_v \end{pmatrix} \begin{pmatrix} \mathbf{x}_u & \mathbf{x}_v \end{pmatrix} J = J^T I J.$$

\(\square\)

## 三、习题

### 习题 3.1

**题目**: 计算平面 \(z = 0\)（参数化 \(\mathbf{x}(u, v) = (u, v, 0)\)）的第一基本形式，并验证曲面上曲线的弧长公式退化为平面上弧长公式。

**提示**: 计算 \(\mathbf{x}_u, \mathbf{x}_v\) 及其内积。

**解答**: \(\mathbf{x}_u = (1, 0, 0)\)，\(\mathbf{x}_v = (0, 1, 0)\)。\(E = 1, F = 0, G = 1\)。故 \(ds^2 = du^2 + dv^2\)。曲线 \(\gamma(t) = (u(t), v(t), 0)\) 的弧长为

$$L = \int_a^b \sqrt{(u')^2 + (v')^2}\, dt,$$

这正是平面上曲线的弧长公式。\(\square\)

---

### 习题 3.2

**题目**: 计算半径为 \(R\) 的球面的第一基本形式（用球坐标参数化 \(\mathbf{x}(\theta, \phi) = (R\sin\theta\cos\phi, R\sin\theta\sin\phi, R\cos\theta)\)），并求其总面积。

**提示**: 注意 \(\theta\) 为极角（与 \(z\) 轴夹角），\(\phi\) 为方位角。

**解答**: \(\mathbf{x}_\theta = (R\cos\theta\cos\phi, R\cos\theta\sin\phi, -R\sin\theta)\)，
\(\mathbf{x}_\phi = (-R\sin\theta\sin\phi, R\sin\theta\cos\phi, 0)\)。

$$E = R^2, \quad F = 0, \quad G = R^2\sin^2\theta.$$

$$ds^2 = R^2 d\theta^2 + R^2\sin^2\theta\, d\phi^2.$$

$$\sqrt{EG - F^2} = R^2\sin\theta.$$

面积：

$$A = \int_0^{2\pi} \int_0^{\pi} R^2\sin\theta\, d\theta\, d\phi = 2\pi R^2 \cdot [-\cos\theta]_0^{\pi} = 4\pi R^2.$$

\(\square\)

---

### 习题 3.3

**题目**: 设圆柱面参数化为 \(\mathbf{x}(u, v) = (\cos u, \sin u, v)\)。求其第一基本形式，并证明圆柱面与平面是**局部等距的**（locally isometric）——即存在保持第一基本形式的局部映射。

**提示**: 考虑将圆柱面沿母线剪开展平。

**解答**: \(\mathbf{x}_u = (-\sin u, \cos u, 0)\)，\(\mathbf{x}_v = (0, 0, 1)\)。\(E = 1, F = 0, G = 1\)。故 \(ds^2 = du^2 + dv^2\)。

平面的第一基本形式也是 \(d\tilde{u}^2 + d\tilde{v}^2\)。映射 \((u, v) \mapsto (u, v)\)（将圆柱坐标直接视为平面直角坐标）保持第一基本形式不变。这说明圆柱面可以"不拉伸、不撕裂"地展平为平面。\(\square\)

---

### 习题 3.4

**题目**: 证明曲面上两条曲线 \(\gamma_1(t) = \mathbf{x}(u_1(t), v_1(t))\) 和 \(\gamma_2(t) = \mathbf{x}(u_2(t), v_2(t))\) 在交点处的夹角 \(\theta\) 满足

$$\cos\theta = \frac{E u_1' u_2' + F(u_1' v_2' + v_1' u_2') + G v_1' v_2'}{\sqrt{E(u_1')^2 + 2Fu_1'v_1' + G(v_1')^2} \sqrt{E(u_2')^2 + 2Fu_2'v_2' + G(v_2')^2}}.$$

**提示**: 用第一基本形式表达 \(\langle \gamma_1', \gamma_2' \rangle\) 和 \(|\gamma_1'|, |\gamma_2'|\)。

**解答**: \(\gamma_1' = u_1'\mathbf{x}_u + v_1'\mathbf{x}_v\)，\(\gamma_2' = u_2'\mathbf{x}_u + v_2'\mathbf{x}_v\)。由内积的双线性，

$$\langle \gamma_1', \gamma_2' \rangle = E u_1' u_2' + F(u_1' v_2' + v_1' u_2') + G v_1' v_2'.$$

又 \(|
\gamma_1'|^2 = E(u_1')^2 + 2Fu_1'v_1' + G(v_1')^2\)，同理 \(|
\gamma_2'|^2\)。由 \(\cos\theta = \dfrac{\langle \gamma_1', \gamma_2' \rangle}{|\gamma_1'||\gamma_2'|}\) 即得。\(\square\)

---

### 习题 3.5

**题目**: 设曲面参数化为 \(\mathbf{x}(u, v) = (u, v, u^2 + v^2)\)（旋转抛物面）。求其第一基本形式，并计算曲线 \(u = t, v = t\)（\(t \in [0, 1]\)）的弧长。

**提示**: 先计算 \(E, F, G\)，再代入弧长公式。

**解答**: \(\mathbf{x}_u = (1, 0, 2u)\)，\(\mathbf{x}_v = (0, 1, 2v)\)。

$$E = 1 + 4u^2, \quad F = 4uv, \quad G = 1 + 4v^2.$$

曲线 \(u = t, v = t\)，\(u' = v' = 1\)。

$$|\gamma'|^2 = E + 2F + G = (1+4t^2) + 8t^2 + (1+4t^2) = 2 + 16t^2.$$

$$L = \int_0^1 \sqrt{2 + 16t^2}\, dt = \int_0^1 \sqrt{2}\sqrt{1 + 8t^2}\, dt.$$

令 \(t = \dfrac{\sinh\theta}{\sqrt{8}}\)，可得

$$L = \frac{\sqrt{2}}{\sqrt{8}} \int_0^{\operatorname{arcsinh}\sqrt{8}} \cosh^2\theta\, d\theta = \frac{1}{2} \left[\frac{\theta + \sinh\theta\cosh\theta}{2}\right]_0^{\operatorname{arcsinh}\sqrt{8}}.$$

数值上 \(L \approx 2.056\)。\(\square\)

## 四、Lean4 形式化框架

```lean4
import Mathlib

-- 第一基本形式的系数结构
structure FirstFundamentalForm where
  E : ℝ → ℝ → ℝ
  F : ℝ → ℝ → ℝ
  G : ℝ → ℝ → ℝ

-- 第一基本形式作为对称双线性型
def firstFundamentalForm (I : FirstFundamentalForm) (u v : ℝ)
  (a b c d : ℝ) : ℝ :=
  I.E u v * a * c + I.F u v * (a * d + b * c) + I.G u v * b * d

-- 弧长计算
def arcLength (I : FirstFundamentalForm) (u v : ℝ → ℝ) (t₀ t₁ : ℝ) : ℝ :=
  ∫ t in t₀..t₁, Real.sqrt (I.E (u t) (v t) * (deriv u t)^2 +
    2 * I.F (u t) (v t) * deriv u t * deriv v t +
    I.G (u t) (v t) * (deriv v t)^2)

-- 面积计算
def surfaceArea (I : FirstFundamentalForm) (R : Set (ℝ × ℝ)) : ℝ :=
  ∫⁻ p : ℝ × ℝ in R, Real.sqrt (I.E p.1 p.2 * I.G p.1 p.2 - I.F p.1 p.2 ^ 2)
```
