# Ch01 向量与向量运算

## 1. 向量的定义与表示

**定义 1.1（向量）**。一个**向量（vector）**是既有大小（magnitude）又有方向（direction）的量。在 $\mathbb{R}^n$ 中，向量可以表示为有序 $n$ 元组：

$$\mathbf{v} = (v_1, v_2, \ldots, v_n)$$

向量的**长度（模）**定义为：

$$|\mathbf{v}| = \sqrt{v_1^2 + v_2^2 + \cdots + v_n^2}$$

**定义 1.2（单位向量）**。长度等于 1 的向量称为**单位向量（unit vector）**。任意非零向量 $\mathbf{v}$ 的方向单位向量为：

$$\hat{\mathbf{v}} = \frac{\mathbf{v}}{|\mathbf{v}|}$$

**定义 1.3（零向量）**。所有分量均为 0 的向量称为**零向量**，记作 $\mathbf{0} = (0, 0, \ldots, 0)$。零向量的长度为 0，方向不确定。

## 2. 向量的线性运算

**定义 2.1（向量加法）**。设 $\mathbf{u} = (u_1, u_2, u_3)$，$\mathbf{v} = (v_1, v_2, v_3)$，则：

$$\mathbf{u} + \mathbf{v} = (u_1 + v_1, u_2 + v_2, u_3 + v_3)$$

几何解释：平行四边形法则或首尾相接法则。将两个向量的起点放在同一点，以它们为邻边作平行四边形，对角线即为和向量。

**定义 2.2（数乘）**。设 $c \in \mathbb{R}$，则：

$$c\mathbf{v} = (cv_1, cv_2, cv_3)$$

- 当 $c > 0$ 时，$c\mathbf{v}$ 与 $\mathbf{v}$ 同向，长度为 $c|\mathbf{v}|$
- 当 $c < 0$ 时，$c\mathbf{v}$ 与 $\mathbf{v}$ 反向，长度为 $|c||\mathbf{v}|$
- 当 $c = 0$ 时，$c\mathbf{v} = \mathbf{0}$

**定理 2.3（向量空间的性质）**。$\mathbb{R}^3$ 中的向量配备加法和数乘运算，构成实数域上的向量空间。具体地，对任意向量 $\mathbf{u}, \mathbf{v}, \mathbf{w}$ 和标量 $a, b$，满足：

1. 交换律：$\mathbf{u} + \mathbf{v} = \mathbf{v} + \mathbf{u}$
2. 结合律：$(\mathbf{u} + \mathbf{v}) + \mathbf{w} = \mathbf{u} + (\mathbf{v} + \mathbf{w})$
3. 零元：$\mathbf{u} + \mathbf{0} = \mathbf{u}$
4. 负元：$\mathbf{u} + (-\mathbf{u}) = \mathbf{0}$
5. 数乘结合律：$a(b\mathbf{u}) = (ab)\mathbf{u}$
6. 数乘分配律（对向量）：$a(\mathbf{u} + \mathbf{v}) = a\mathbf{u} + a\mathbf{v}$
7. 数乘分配律（对标量）：$(a + b)\mathbf{u} = a\mathbf{u} + b\mathbf{u}$
8. 单位元：$1 \cdot \mathbf{u} = \mathbf{u}$

*证明*：直接按分量验证八条公理。以交换律为例：

$$\mathbf{u} + \mathbf{v} = (u_1 + v_1, u_2 + v_2, u_3 + v_3) = (v_1 + u_1, v_2 + u_2, v_3 + u_3) = \mathbf{v} + \mathbf{u}$$

其余类似。$\square$

## 3. 标准基向量

在 $\mathbb{R}^3$ 中，定义**标准基向量**：

$$\mathbf{i} = (1, 0, 0), \quad \mathbf{j} = (0, 1, 0), \quad \mathbf{k} = (0, 0, 1)$$

任意向量可唯一表示为：

$$\mathbf{v} = v_1\mathbf{i} + v_2\mathbf{j} + v_3\mathbf{k}$$

这一表示的唯一性来自于 $\{\mathbf{i}, \mathbf{j}, \mathbf{k}\}$ 是 $\mathbb{R}^3$ 的一组基。

## 4. 向量的几何应用

**定义 4.1（位置向量）**。空间中一点 $P(x, y, z)$ 的**位置向量**是从原点 $O$ 指向 $P$ 的向量：

$$\vec{OP} = (x, y, z)$$

**定理 4.2（两点间向量）**。从点 $P(x_1, y_1, z_1)$ 到点 $Q(x_2, y_2, z_2)$ 的向量为：

$$\vec{PQ} = (x_2 - x_1, y_2 - y_1, z_2 - z_1)$$

两点间距离为 $|\vec{PQ}| = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2}$。

## 5. 典型例题

**例题 5.1**。求从点 $P(1, 2, 3)$ 到点 $Q(4, 0, 6)$ 的向量及其长度。

*解答*：$\vec{PQ} = Q - P = (4-1, 0-2, 6-3) = (3, -2, 3)$。

$$|\vec{PQ}| = \sqrt{3^2 + (-2)^2 + 3^2} = \sqrt{9 + 4 + 9} = \sqrt{22}$$

**例题 5.2**。求向量 $\mathbf{v} = (2, -1, 2)$ 方向的单位向量。

*解答*：

$$|\mathbf{v}| = \sqrt{2^2 + (-1)^2 + 2^2} = \sqrt{4 + 1 + 4} = 3$$

$$\hat{\mathbf{v}} = \frac{\mathbf{v}}{|\mathbf{v}|} = \left(\frac{2}{3}, -\frac{1}{3}, \frac{2}{3}\right)$$

**例题 5.3**。证明平行四边形的对角线互相平分。

*解答*：设平行四边形的顶点依次为 $A, B, C, D$，则 $\vec{AB} = \vec{DC}$。对角线 $AC$ 的中点位置向量为 $\frac{1}{2}(\vec{OA} + \vec{OC}) = \frac{1}{2}(\vec{OA} + \vec{OD} + \vec{DC}) = \frac{1}{2}(\vec{OA} + \vec{OD} + \vec{AB}) = \frac{1}{2}(\vec{OB} + \vec{OD})$，这正是对角线 $BD$ 的中点。$\square$

## 6. 常见错误分析

1. **混淆点与向量**：点表示位置，向量表示位移。虽然都用坐标表示，但概念不同。例如，$P = (1, 2, 3)$ 是一个点，而 $\vec{OP} = (1, 2, 3)$ 是从原点到 $P$ 的向量。

2. **向量加法与标量加法混淆**：向量加法是对应分量相加，不是简单的数值加。$(1, 2) + (3, 4) = (4, 6)$，而不是 $1+2+3+4 = 10$。

3. **忽略方向**：只看大小不看方向是常见错误。例如，$\mathbf{v}$ 和 $-\mathbf{v}$ 大小相同但方向相反。

4. **单位向量计算错误**：忘记除以向量的长度，或者长度计算错误（如漏掉平方根）。

## 7. 练习题

### 基础练习

1. 已知 $\mathbf{a} = (1, 2, 3)$，$\mathbf{b} = (4, 5, 6)$，求 $2\mathbf{a} - 3\mathbf{b}$ 和 $|2\mathbf{a} - 3\mathbf{b}|$。
2. 求与 $\mathbf{a} = (1, 1, 1)$ 和 $\mathbf{b} = (1, -1, 0)$ 都垂直的单位向量。
3. 设 $A(1, 0, 0)$，$B(0, 1, 0)$，$C(0, 0, 1)$，求三角形 $ABC$ 的重心位置向量。

### 进阶练习

1. 证明三角不等式：$|\mathbf{u} + \mathbf{v}| \leq |\mathbf{u}| + |\mathbf{v}|$。等号何时成立？
2. 设 $\mathbf{a}, \mathbf{b}, \mathbf{c}$ 为不共面向量，证明任意向量 $\mathbf{r}$ 可唯一表示为 $\mathbf{r} = x\mathbf{a} + y\mathbf{b} + z\mathbf{c}$。
3. 在四面体 $OABC$ 中，$D, E, F$ 分别为 $OA, OB, OC$ 的中点。证明 $\vec{DE} = \frac{1}{2}\vec{AB}$，$\vec{DF} = \frac{1}{2}\vec{AC}$。

## 8. Lean4 形式化对照

本节展示向量运算核心概念在 Lean4 / Mathlib4 中的形式化表达。

### 8.1 向量空间基础

```lean4
import Mathlib

-- R^n 中的向量
variable (n : ℕ)

-- EuclideanSpace 是 Mathlib 中对 R^n 的标准实现
example (v : EuclideanSpace ℝ (Fin n)) : ℝ :=
  ‖v‖  -- 范数（长度）

-- 向量加法
example (u v : EuclideanSpace ℝ (Fin 3)) : EuclideanSpace ℝ (Fin 3) :=
  u + v

-- 数乘
example (c : ℝ) (v : EuclideanSpace ℝ (Fin 3)) : EuclideanSpace ℝ (Fin 3) :=
  c • v

-- 零向量
example : EuclideanSpace ℝ (Fin 3) :=
  0

-- 标准基向量 e₁
example : EuclideanSpace ℝ (Fin 3) :=
  fun i => if i = 0 then 1 else 0
```

### 8.2 向量长度的性质

```lean4
import Mathlib

-- 三角不等式在 Mathlib 中已有证明
example (u v : EuclideanSpace ℝ (Fin 3)) : ‖u + v‖ ≤ ‖u‖ + ‖v‖ := by
  apply norm_add_le

-- 数乘对范数的影响
example (c : ℝ) (v : EuclideanSpace ℝ (Fin 3)) : ‖c • v‖ = |c| * ‖v‖ := by
  simp [norm_smul]
```

### 8.3 单位向量

```lean4
import Mathlib

-- 非零向量的单位化
example (v : EuclideanSpace ℝ (Fin 3)) (hv : v ≠ 0) : EuclideanSpace ℝ (Fin 3) :=
  (1 / ‖v‖) • v

-- 单位向量的范数为1
example (v : EuclideanSpace ℝ (Fin 3)) (hv : v ≠ 0) : ‖(1 / ‖v‖) • v‖ = 1 := by
  simp [norm_smul, hv]
```

## 9. 应用与拓展

### 9.1 物理学中的应用

向量是物理学的基础语言：

- **力**：力是向量，有大小和方向
- **速度**：物体的速度是位置向量的导数
- **位移**：物体位置的变化是向量差

### 9.2 计算机图形学

在计算机图形学中，向量用于：

- 表示三维模型的顶点位置
- 计算光照（需要法向量）
- 相机变换和投影

### 9.3 机器学习

在机器学习中，数据点通常表示为高维向量：

- 特征向量：每个维度对应一个特征
- 词嵌入：单词表示为高维语义向量
- 距离度量：欧氏距离、余弦相似度等

## 习题

**习题 1.1**。已知 $\mathbf{a} = (1, 2, -1)$，$\mathbf{b} = (3, 0, 2)$，计算：
(1) $\mathbf{a} + 2\mathbf{b}$；(2) $|\mathbf{a}|$；(3) $\mathbf{a}$ 方向的单位向量。

*解答*：
(1) $\mathbf{a} + 2\mathbf{b} = (1, 2, -1) + (6, 0, 4) = (7, 2, 3)$
(2) $|\mathbf{a}| = \sqrt{1^2 + 2^2 + (-1)^2} = \sqrt{6}$
(3) $\hat{\mathbf{a}} = \frac{1}{\sqrt{6}}(1, 2, -1)$

---

**习题 1.2**。证明：对任意向量 $\mathbf{u}, \mathbf{v}, \mathbf{w} \in \mathbb{R}^3$，有 $(\mathbf{u} + \mathbf{v}) + \mathbf{w} = \mathbf{u} + (\mathbf{v} + \mathbf{w})$。

*解答*：按分量计算：
$$((u_1+v_1)+w_1, (u_2+v_2)+w_2, (u_3+v_3)+w_3) = (u_1+(v_1+w_1), u_2+(v_2+w_2), u_3+(v_3+w_3))$$
由实数加法结合律即得。$\square$

---

**习题 1.3**。求顶点为 $A(1,0,0)$, $B(0,2,0)$, $C(0,0,3)$ 的三角形的重心位置向量。

*解答*：重心 $G = \frac{A+B+C}{3} = \frac{1}{3}(1, 2, 3) = (\frac{1}{3}, \frac{2}{3}, 1)$。$\square$
