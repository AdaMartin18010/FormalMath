# Ch04 多元函数与偏导数

## 1. 多元函数的定义

**定义 1.1**。一个**多元函数（function of several variables）**是映射 $f: D \subseteq \mathbb{R}^n \to \mathbb{R}$，即对每个输入 $(x_1, x_2, \ldots, x_n) \in D$，输出唯一的实数 $f(x_1, x_2, \ldots, x_n)$。

**定义 1.2（定义域与值域）**。函数 $f$ 的**定义域**是所有使 $f$ 有定义的输入集合，**值域**是所有可能的输出值集合。

常见的二元函数例子：

- $f(x, y) = x^2 + y^2$（旋转抛物面）
- $f(x, y) = \sqrt{x^2 + y^2}$（锥面）
- $f(x, y) = \sin(x)\cos(y)$（波动面）
- $f(x, y) = e^{-(x^2+y^2)}$（高斯钟形面）

**定义 1.3（等值线/等值面）**。函数 $f$ 的**等值线（level curve）**是集合 $\{(x, y) \mid f(x, y) = c\}$，等值面类似定义于三维。

## 2. 偏导数

**定义 2.1（偏导数）**。函数 $f(x, y)$ 对 $x$ 的**偏导数**定义为：

$$\frac{\partial f}{\partial x} = \lim_{h \to 0} \frac{f(x+h, y) - f(x, y)}{h}$$

类似地，对 $y$ 的偏导数为：

$$\frac{\partial f}{\partial y} = \lim_{h \to 0} \frac{f(x, y+h) - f(x, y)}{h}$$

记号：$f_x = \frac{\partial f}{\partial x}$，$f_y = \frac{\partial f}{\partial y}$，$f_{xx} = \frac{\partial^2 f}{\partial x^2}$，$f_{xy} = \frac{\partial^2 f}{\partial x \partial y}$ 等。

几何意义：固定 $y$，沿 $x$ 方向的变化率，即曲面在 $x$ 方向的切线斜率。

**定理 2.2（Clairaut定理/混合偏导数相等）**。若 $f_{xy}$ 和 $f_{yx}$ 在某开区域连续，则：

$$\frac{\partial^2 f}{\partial x \partial y} = \frac{\partial^2 f}{\partial y \partial x}$$

*证明*：由中值定理，考虑差分：

$$D = f(x+h, y+k) - f(x+h, y) - f(x, y+k) + f(x, y)$$

先对 $x$ 后对 $y$ 应用中值定理，以及先对 $y$ 后对 $x$ 应用中值定理，得到两个表达式，当 $h, k \to 0$ 时，由 $f_{xy}$ 和 $f_{yx}$ 的连续性，两者必须相等。$\square$

## 3. 全微分与可微性

**定义 3.1（可微性）**。$f(x, y)$ 在 $(a, b)$ **可微**，若存在线性函数 $L(x, y)$ 使得：

$$f(a+h, b+k) - f(a, b) = L(h, k) + o(\sqrt{h^2 + k^2})$$

其中 $L(h, k) = f_x(a, b)h + f_y(a, b)k$。

**定义 3.2（全微分）**。$f$ 在 $(a, b)$ 的**全微分**为：

$$df = f_x(a, b) \, dx + f_y(a, b) \, dy$$

**定理 3.3**。若 $f_x$ 和 $f_y$ 存在且在 $(a, b)$ 连续，则 $f$ 在 $(a, b)$ 可微。

*证明*：利用中值定理：

$$f(a+h, b+k) - f(a, b) = [f(a+h, b+k) - f(a, b+k)] + [f(a, b+k) - f(a, b)]$$

$$= f_x(a+\theta_1 h, b+k)h + f_y(a, b+\theta_2 k)k$$

由连续性，$f_x(a+\theta_1 h, b+k) = f_x(a, b) + \epsilon_1$，$f_y(a, b+\theta_2 k) = f_y(a, b) + \epsilon_2$，其中 $\epsilon_1, \epsilon_2 \to 0$。代入即得可微性定义。$\square$

## 4. 链式法则

**定理 4.1（多元链式法则）**。设 $z = f(x, y)$，$x = x(t)$，$y = y(t)$，则：

$$\frac{dz}{dt} = \frac{\partial z}{\partial x}\frac{dx}{dt} + \frac{\partial z}{\partial y}\frac{dy}{dt}$$

对于 $z = f(x, y)$，$x = x(u, v)$，$y = y(u, v)$：

$$\frac{\partial z}{\partial u} = \frac{\partial z}{\partial x}\frac{\partial x}{\partial u} + \frac{\partial z}{\partial y}\frac{\partial y}{\partial u}$$

$$\frac{\partial z}{\partial v} = \frac{\partial z}{\partial x}\frac{\partial x}{\partial v} + \frac{\partial z}{\partial y}\frac{\partial y}{\partial v}$$

*证明*：利用全微分的链式结构和可微性定义。$\square$

**定理 4.2（隐函数求导）**。若 $F(x, y) = 0$ 定义 $y$ 为 $x$ 的隐函数，则：

$$\frac{dy}{dx} = -\frac{F_x}{F_y}$$

## 5. 典型例题

**例题 5.1**。设 $f(x, y) = x^3 y^2 + \sin(xy)$，求 $f_x$、$f_y$、$f_{xx}$、$f_{xy}$。

*解答*：

$$f_x = 3x^2 y^2 + y\cos(xy)$$

$$f_y = 2x^3 y + x\cos(xy)$$

$$f_{xx} = 6xy^2 - y^2\sin(xy)$$

$$f_{xy} = 6x^2 y + \cos(xy) - xy\sin(xy)$$

**例题 5.2**。设 $w = f(x, y, z) = xyz$，$x = t^2$，$y = e^t$，$z = \sin t$，求 $\frac{dw}{dt}$。

*解答*：

$$\frac{dw}{dt} = yz \cdot 2t + xz \cdot e^t + xy \cdot \cos t = 2te^t\sin t + t^2 e^t \sin t + t^2 e^t \cos t$$

**例题 5.3**。验证 $f(x, y) = \arctan(y/x)$ 满足 Laplace 方程 $f_{xx} + f_{yy} = 0$（在 $x \neq 0$ 处）。

*解答*：

$$f_x = \frac{-y}{x^2 + y^2}, \quad f_{xx} = \frac{2xy}{(x^2+y^2)^2}$$

$$f_y = \frac{x}{x^2 + y^2}, \quad f_{yy} = \frac{-2xy}{(x^2+y^2)^2}$$

所以 $f_{xx} + f_{yy} = 0$。$\square$

## 6. 常见错误

1. **混淆偏导数与全导数**：$\frac{\partial f}{\partial x}$ 是在固定其他变量下的导数，$\frac{df}{dx}$ 是所有变量都依赖于 $x$ 时的总变化率。

2. **链式法则漏项**：当多个中间变量依赖于同一自变量时，需要对每个路径求导并相加。

3. **忽略可微性条件**：偏导数存在不能保证函数可微，需要偏导数连续。

## 7. 练习题

### 基础练习

1. 设 $f(x, y) = e^{xy} \ln(x^2 + y^2)$，求 $f_x$ 和 $f_y$。
2. 设 $u = f(x - y, y - z, z - x)$，证明 $\frac{\partial u}{\partial x} + \frac{\partial u}{\partial y} + \frac{\partial u}{\partial z} = 0$。
3. 设 $x^3 + y^3 + z^3 - 3xyz = 1$，求 $\frac{\partial z}{\partial x}$ 和 $\frac{\partial z}{\partial y}$。

### 进阶练习

1. 证明：若 $f(x, y)$ 满足 $x f_x + y f_y = n f$，则 $f$ 是 $n$ 次齐次函数（Euler齐次函数定理）。
2. 设 $u = f(\frac{y}{x})$，证明 $x^2 u_{xx} + 2xy u_{xy} + y^2 u_{yy} = 0$。
3. 设 $z = f(x + y) + g(x - y)$，证明 $z_{xx} - z_{yy} = 0$（波动方程）。

## 8. Lean4 形式化对照

```lean4
import Mathlib

-- 偏导数通过 restrict 和 deriv 实现
-- 函数 f(x,y) = x^2 + y^2 对 x 的偏导数
example : deriv (fun x : ℝ => x^2 + (3 : ℝ)^2) 2 = 4 := by
  simp [deriv_pow]

-- 梯度作为 Fréchet 导数
example (f : EuclideanSpace ℝ (Fin 2) → ℝ) (hf : Differentiable ℝ f) :
    ∃ (L : EuclideanSpace ℝ (Fin 2) →L[ℝ] ℝ), HasFDerivAt f L (fun i => 0) := by
  exact ⟨fderiv ℝ f (fun i => 0), hf.differentiableAt.hasFDerivAt⟩

-- 链式法则
example (f : ℝ → ℝ) (g : ℝ → ℝ) (x : ℝ) (hf : DifferentiableAt ℝ f (g x))
    (hg : DifferentiableAt ℝ g x) :
    deriv (fun x => f (g x)) x = deriv f (g x) * deriv g x := by
  rw [deriv.comp]
  all_goals assumption
```

## 9. 应用与拓展

### 9.1 热传导方程

一维热传导方程：$u_t = k u_{xx}$，其中 $u(x, t)$ 为温度分布。这是偏微分方程的最经典例子。

### 9.2 经济学中的边际分析

在经济学中，多元函数用于描述生产函数 $Q = f(K, L)$（资本和劳动的产出）。偏导数 $\frac{\partial Q}{\partial K}$ 和 $\frac{\partial Q}{\partial L}$ 分别表示资本的边际产出和劳动的边际产出。

### 9.3 机器学习中的梯度下降

梯度下降算法利用偏导数来最小化损失函数：$\theta_{new} = \theta_{old} - \alpha \nabla L(\theta_{old})$。
