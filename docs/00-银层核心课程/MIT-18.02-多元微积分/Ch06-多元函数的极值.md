# Ch06 多元函数的极值

## 1. 局部极值的必要条件

**定理 1.1**。若 $f(x, y)$ 在 $(a, b)$ 处有局部极值且偏导数存在，则：

$$f_x(a, b) = 0, \quad f_y(a, b) = 0$$

满足上述条件的点称为**临界点（critical point）**或**驻点（stationary point）**。

## 2. 二阶导数判别法

**定理 2.1**。设 $(a, b)$ 为临界点，记 $D = f_{xx}(a,b)f_{yy}(a,b) - [f_{xy}(a,b)]^2$。

- $D > 0$ 且 $f_{xx} > 0$：局部极小值
- $D > 0$ 且 $f_{xx} < 0$：局部极大值
- $D < 0$：**鞍点（saddle point）**
- $D = 0$：判别法失效

*证明概要*：Taylor 展开到二阶：

$$f(a+h, b+k) - f(a,b) = \frac{1}{2}(f_{xx}h^2 + 2f_{xy}hk + f_{yy}k^2) + o(h^2+k^2)$$

二次型 $Q(h,k) = f_{xx}h^2 + 2f_{xy}hk + f_{yy}k^2$ 的定号性由判别式 $D$ 决定。$\square$

## 3. 条件极值与Lagrange乘数法

**定理 3.1（Lagrange乘数法）**。求 $f(x, y)$ 在约束 $g(x, y) = 0$ 下的极值，解方程组：

$$\nabla f = \lambda \nabla g, \quad g(x, y) = 0$$

## 4. 典型例题

**例题 4.1**。求 $f(x, y) = x^3 + y^3 - 3xy$ 的临界点并分类。

*解答*：$f_x = 3x^2 - 3y = 0$，$f_y = 3y^2 - 3x = 0$，得 $(0, 0)$ 和 $(1, 1)$。

在 $(0, 0)$：$f_{xx} = 0$，$f_{yy} = 0$，$f_{xy} = -3$，$D = -9 < 0$（鞍点）。

在 $(1, 1)$：$f_{xx} = 6$，$f_{yy} = 6$，$f_{xy} = -3$，$D = 27 > 0$，$f_{xx} > 0$（极小值，$f(1,1) = -1$）。

**例题 4.2**。在 $x^2 + y^2 = 1$ 上求 $f(x, y) = x + 2y$ 的最大值和最小值。

*解答*：$g(x, y) = x^2 + y^2 - 1 = 0$。$\nabla f = (1, 2)$，$\nabla g = (2x, 2y)$。

由 $1 = 2\lambda x$，$2 = 2\lambda y$ 得 $y = 2x$。结合 $x^2 + y^2 = 1$ 得 $x = \pm \frac{1}{\sqrt{5}}$。

最大值 $f(\frac{1}{\sqrt{5}}, \frac{2}{\sqrt{5}}) = \sqrt{5}$，最小值 $f(-\frac{1}{\sqrt{5}}, -\frac{2}{\sqrt{5}}) = -\sqrt{5}$。

## 5. 练习题

1. 求 $f(x, y) = x^4 + y^4 - 4xy$ 的所有临界点并分类。
2. 用 Lagrange 乘数法求原点到平面 $x + 2y + 3z = 4$ 的最短距离。
3. 证明：在约束 $x + y + z = 1$，$x, y, z \geq 0$ 下，$f(x, y, z) = xyz$ 的最大值为 $\frac{1}{27}$。
4. 求椭球 $\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1$ 内接长方体的最大体积。

## 7. 应用

### 7.1 最小二乘法

最小二乘拟合可以视为多元函数的极值问题：最小化误差平方和 $E(a, b) = \sum (y_i - ax_i - b)^2$。

### 7.2 经济学中的效用最大化

消费者理论中，在预算约束下最大化效用函数 $U(x, y)$，这正是Lagrange乘数法的经典应用。
