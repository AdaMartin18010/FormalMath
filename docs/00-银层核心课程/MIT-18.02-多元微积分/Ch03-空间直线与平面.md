# Ch03 空间直线与平面

## 1. 空间直线的参数方程

**定义 1.1（直线的参数方程）**。过点 $P_0(x_0, y_0, z_0)$ 且方向向量为 $\mathbf{v} = (a, b, c)$ 的直线参数方程为：

$$\mathbf{r}(t) = \mathbf{r}_0 + t\mathbf{v} = (x_0 + at, y_0 + bt, z_0 + ct), \quad t \in \mathbb{R}$$

**定义 1.2（直线的对称式）**。若 $a, b, c \neq 0$，可消去参数 $t$ 得：

$$\frac{x - x_0}{a} = \frac{y - y_0}{b} = \frac{z - z_0}{c}$$

**定义 1.3（直线的两点式）**。过两点 $P_1(x_1, y_1, z_1)$ 和 $P_2(x_2, y_2, z_2)$ 的直线，方向向量为 $\vec{P_1P_2}$，参数方程为：

$$\mathbf{r}(t) = P_1 + t(P_2 - P_1), \quad t \in \mathbb{R}$$

## 2. 平面的方程

**定义 2.1（平面的点法式）**。过点 $P_0(x_0, y_0, z_0)$ 且法向量为 $\mathbf{n} = (A, B, C)$ 的平面方程为：

$$A(x - x_0) + B(y - y_0) + C(z - z_0) = 0$$

或简写为 $Ax + By + Cz = D$，其中 $D = Ax_0 + By_0 + Cz_0$。

**定理 2.2（平面的一般式）**。任意平面都可表示为 $Ax + By + Cz + D = 0$，其中 $(A, B, C) \neq (0, 0, 0)$。

*证明*：给定平面上一点 $P_0$ 和法向量 $\mathbf{n} = (A, B, C)$，平面上任一点 $P$ 满足 $\vec{P_0P} \perp \mathbf{n}$，即 $\mathbf{n} \cdot \vec{P_0P} = 0$，展开即得。$\square$

**定义 2.3（平面的三点式）**。过不共线三点 $P_1, P_2, P_3$ 的平面，法向量为 $\vec{P_1P_2} \times \vec{P_1P_3}$。

## 3. 直线与平面的位置关系

**定理 3.1（直线与平面的夹角）**。设直线方向向量为 $\mathbf{v}$，平面法向量为 $\mathbf{n}$，则夹角 $\phi$（直线与平面法向量的夹角的余角）满足：

$$\sin\phi = \frac{|\mathbf{v} \cdot \mathbf{n}|}{|\mathbf{v}| |\mathbf{n}|}$$

**定理 3.2（点到平面的距离）**。点 $P(x_1, y_1, z_1)$ 到平面 $Ax + By + Cz + D = 0$ 的距离：

$$d = \frac{|Ax_1 + By_1 + Cz_1 + D|}{\sqrt{A^2 + B^2 + C^2}}$$

*证明*：取平面上任一点 $P_0$ 满足 $Ax_0 + By_0 + Cz_0 + D = 0$。则距离等于向量 $\vec{P_0P}$ 在法向量方向投影的绝对值：

$$d = |\vec{P_0P} \cdot \hat{\mathbf{n}}| = \frac{|(x_1-x_0, y_1-y_0, z_1-z_0) \cdot (A, B, C)|}{\sqrt{A^2+B^2+C^2}} = \frac{|Ax_1+By_1+Cz_1+D|}{\sqrt{A^2+B^2+C^2}}$$

$\square$

**定理 3.3（两平面的夹角）**。设两平面法向量分别为 $\mathbf{n}_1, \mathbf{n}_2$，则夹角 $\theta$ 满足：

$$\cos\theta = \frac{|\mathbf{n}_1 \cdot \mathbf{n}_2|}{|\mathbf{n}_1| |\mathbf{n}_2|}$$

## 4. 典型例题

**例题 4.1**。求过点 $(1, 2, 3)$ 且垂直于平面 $2x - y + 3z = 5$ 的直线方程。

*解答*：直线方向向量即为平面法向量 $(2, -1, 3)$。

参数方程：$x = 1 + 2t$，$y = 2 - t$，$z = 3 + 3t$。

对称式：$\frac{x-1}{2} = \frac{y-2}{-1} = \frac{z-3}{3}$。

**例题 4.2**。求两平面 $x + y + z = 1$ 和 $x - y + 2z = 3$ 的交线。

*解答*：交线方向向量：

$$\mathbf{v} = \mathbf{n}_1 \times \mathbf{n}_2 = (1, 1, 1) \times (1, -1, 2) = (3, -1, -2)$$

求交线上一点：令 $z = 0$，解 $x + y = 1$，$x - y = 3$，得 $x = 2, y = -1$。所以点 $(2, -1, 0)$ 在交线上。

交线参数方程：$x = 2 + 3t$，$y = -1 - t$，$z = -2t$。

**例题 4.3**。求点 $(1, 2, 3)$ 到平面 $x + 2y + 2z = 10$ 的距离。

*解答*：

$$d = \frac{|1 + 4 + 6 - 10|}{\sqrt{1 + 4 + 4}} = \frac{1}{3}$$

## 5. 练习题

### 基础练习

1. 求过三点 $(1,0,0)$、$(0,1,0)$、$(0,0,1)$ 的平面方程。
2. 求点 $(1, 2, 3)$ 到平面 $x + 2y + 2z = 10$ 的距离。
3. 求直线 $x = 1 + t, y = 2 - t, z = 3 + 2t$ 与平面 $x + y + z = 6$ 的交点。

### 进阶练习

1. 证明：若平面 $Ax + By + Cz = D$ 与坐标轴的交点分别为 $(a, 0, 0)$、$(0, b, 0)$、$(0, 0, c)$，则 $\frac{1}{a} + \frac{1}{b} + \frac{1}{c} = \frac{D}{abc}$（截距式）。
2. 求过点 $(1, 0, 0)$ 且与两条直线 $L_1: x = y = z$ 和 $L_2: x = -y = z$ 都平行的平面方程。

## 7. 应用与拓展

### 7.1 计算机图形学中的射线追踪

在光线追踪渲染中，需要频繁计算光线（直线）与物体表面（平面或曲面）的交点。平面交点计算是其中最基础的操作。

### 7.2 机器人运动规划

机器人末端执行器的运动轨迹通常是空间直线或曲线。平面约束用于定义机器人的工作空间边界。

## 习题

**习题 3.1**。求过点 $P(1, 2, 3)$ 且方向向量为 $\mathbf{v} = (2, -1, 4)$ 的直线的参数方程和对称式方程。

*解答*：
参数方程：$x = 1 + 2t$, $y = 2 - t$, $z = 3 + 4t$
对称式：$\frac{x-1}{2} = \frac{y-2}{-1} = \frac{z-3}{4}$。$\square$

---

**习题 3.2**。求过三点 $A(1,0,0)$, $B(0,1,0)$, $C(0,0,1)$ 的平面方程。

*解答*：法向量 $\mathbf{n} = \overrightarrow{AB} \times \overrightarrow{AC} = (1, 1, 1)$。
平面方程：$x + y + z = 1$。$\square$

---

**习题 3.3**。求点 $Q(2, 1, -1)$ 到平面 $2x - y + 2z + 3 = 0$ 的距离。

*解答*：$d = \frac{|2(2) - 1(1) + 2(-1) + 3|}{\sqrt{4+1+4}} = \frac{|4-1-2+3|}{3} = \frac{4}{3}$。$\square$
