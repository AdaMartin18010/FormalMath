# 共形映射（Conformal Mappings）

> **课程**: Princeton 复分析 | **章节**: Ch. 4 — 共形映射与 Riemann 映射定理
> **学习目标**: 理解共形映射的几何意义；掌握 Möbius 变换的基本性质；了解 Riemann 映射定理的陈述与应用

---

## 一、核心定义

### 定义 5.1（共形映射 / Conformal Mapping）

**严格陈述**: 设 \(U, V \subseteq \mathbb{C}\) 为开集。映射 \(f: U \to V\) 称为**共形的**（conformal），若 \(f\) 在 \(U\) 上全纯且 \(f'(z) \neq 0\) 对所有 \(z \in U\) 成立。若 \(f\) 还是双射，则称 \(f\) 为**共形等价**（conformal equivalence）或**双全纯映射**（biholomorphic map），称 \(U\) 与 \(V\) **共形等价**。

**直观解释**: 共形映射是"保持角度"的映射。设两条光滑曲线在 \(z_0\) 处相交，夹角为 \(\theta\)。经过共形映射 \(f\) 后，像曲线在 \(f(z_0)\) 处的夹角仍为 \(\theta\)（包括定向）。这是因为：对无穷小向量 \(h\)，\(f(z_0 + h) - f(z_0) \approx f'(z_0) h\)，即局部上 \(f\) 近似为乘以复数 \(f'(z_0)\)——乘以一个非零复数等价于旋转加伸缩，两者均保持角度。

---

### 定义 5.2（Möbius 变换 / Fractional Linear Transformation）

**严格陈述**: 形如

$$f(z) = \frac{az + b}{cz + d}, \qquad a, b, c, d \in \mathbb{C}, \quad ad - bc \neq 0$$

的映射称为 **Möbius 变换**（Möbius transformation）或**分式线性变换**。矩阵条件 \(ad - bc \neq 0\) 保证映射非退化。Möbius 变换可延拓为扩充复平面 \(\hat{\mathbb{C}} = \mathbb{C} \cup \{\infty\}\) 上的双射，其中 \(f(-d/c) = \infty\)（若 \(c \neq 0\)），\(f(\infty) = a/c\)。

**直观解释**: Möbius 变换是复平面上最"灵活"的共形映射。它们构成一个群（在复合运算下），同构于 \(PSL(2, \mathbb{C}) = SL(2, \mathbb{C})/\{\pm I\}\)。Möbius 变换将圆和直线映射为圆和直线（在 \(\hat{\mathbb{C}}\) 中，直线可视为过无穷远点的圆），并保持交比不变。它们在双曲几何（Poincaré 圆盘/上半平面模型）中扮演着等距变换的角色。

---

### 定义 5.3（单连通区域 / Simply Connected Domain）

**严格陈述**: 区域（连通开集）\(U \subseteq \mathbb{C}\) 称为**单连通的**，若 \(\mathbb{C} \setminus U\) 的补集无界连通分支；等价地，\(U\) 内任意闭合曲线可在 \(U\) 内连续缩为一点。

**直观解释**: 单连通区域"没有洞"。单位圆盘 \(\{z : |z| < 1\}\)、上半平面 \(\{z : \operatorname{Im}(z) > 0\}\)、整个复平面 \(\mathbb{C}\) 都是单连通的。去心圆盘、圆环不是单连通的。

> **双语对照**:
>
> ```lean4

>
> open Complex
>
> -- 共形映射：双全纯等价
> example {f : ℂ → ℂ} (hf : Differentiable ℂ f) (hf' : ∀ z, deriv f z ≠ 0)
>   (hbi : Function.Bijective f) : True := trivial

>

> def crossRatio (z₁ z₂ z₃ z₄ : ℂ) : ℂ :=
>   (z₁ - z₃) * (z₂ - z₄) / ((z₁ - z₄) * (z₂ - z₃))
> ```

## 二、核心定理与完整证明

### 定理 5.1（Riemann 映射定理）

**定理陈述**: 设 \(U \subsetneq \mathbb{C}\) 为单连通区域，且 \(U \neq \mathbb{C}\)。则对任意 \(z_0 \in U\)，存在唯一的共形映射 \(f: U \to \mathbb{D}\)（其中 \(\mathbb{D} = \{z : |z| < 1\}\) 为单位圆盘），满足

$$f(z_0) = 0, \qquad f'(z_0) > 0.$$

**证明思路**: Riemann 映射定理是复分析中最深刻的存在性定理之一。其标准证明分为以下几步：

1. **构造非空函数族**: 考虑所有单射全纯映射 \(g: U \to \mathbb{D}\) 的集合 \(\mathcal{F}\)。先证明 \(\mathcal{F} \neq \varnothing\)（这需要 \(U \neq \mathbb{C}\) 的条件——利用平方根分支的存在性）。

2. **极值原理**: 考虑在 \(z_0\) 处使 \(|g'(z_0)|\) 最大的映射。由 Montel 定理（正规族理论），这个极值可以达到。

3. **证明极值映射是满射**: 若极值映射 \(f\) 不是满射，设 \(w_0 \in \mathbb{D} \setminus f(U)\)。通过适当的 Möbius 变换和平方根运算，可构造另一个单射全纯映射 \(\tilde{g}: U \to \mathbb{D}\) 使得 \(|
\tilde{g}'(z_0)| > |f'(z_0)|\)，与极值性矛盾。

4. **唯一性**: 由 Schwarz 引理保证。\(\square\)

> **证明要点提示**:
>
> 1. **正规族与紧性**: Riemann 映射定理的证明依赖于全纯函数族的紧性（Montel 定理）。这使得"取最大导数"的极值论证成为可能。
> 2. **平方根的妙用**: 单连通性保证了全纯函数平方根的存在性——这是连接单连通性与共形等价的关键桥梁。

---

### 定理 5.2（Schwarz 引理）

**定理陈述**: 设 \(f: \mathbb{D} \to \mathbb{D}\) 全纯，且 \(f(0) = 0\)。则

1. \(|f(z)| \leq |z|\) 对所有 \(z \in \mathbb{D}\)；
2. \(|f'(0)| \leq 1\)。

若任一不等式在某非零点取等号，则 \(f(z) = e^{i\theta} z\) 为旋转。

**证明**: 定义 \(g(z) = f(z)/z\)（\(z \neq 0\)），\(g(0) = f'(0)\)。由最大模原理，对 \(|z| \leq r < 1\)，

$$|g(z)| \leq \max_{|\zeta| = r} |g(\zeta)| = \max_{|\zeta| = r} \frac{|f(\zeta)|}{|\zeta|} \leq \frac{1}{r}.$$

令 \(r \to 1\)，得 \(|g(z)| \leq 1\)。故 \(|f(z)| \leq |z|\) 且 \(|f'(0)| = |g(0)| \leq 1\)。

若在某 \(z_0 \neq 0\) 处 \(|f(z_0)| = |z_0|\)，则 \(|g(z_0)| = 1\)。由最大模原理，\(g\) 为常数 \(e^{i\theta}\)，故 \(f(z) = e^{i\theta} z\)。\(\square\)

## 三、习题

### 习题 5.1

**题目**: 求将上半平面 \(\{z : \operatorname{Im}(z) > 0\}\) 共形映射到单位圆盘 \(\mathbb{D}\) 的映射，且满足 \(f(i) = 0\)，\(f'(i) > 0\)。

**提示**: 考虑 Cayley 变换 \(C(z) = \dfrac{z - i}{z + i}\)。

**解答**: Cayley 变换 \(C(z) = \dfrac{z - i}{z + i}\) 将上半平面映到 \(\mathbb{D}\)（因为 \(z\) 在上半平面时 \(|z - i| < |z + i|\)），且 \(C(i) = 0\)。计算 \(C'(z) = \dfrac{2i}{(z+i)^2}\)，\(C'(i) = \dfrac{2i}{(2i)^2} = \dfrac{1}{2i} = -\dfrac{i}{2}\)，不满足正实数条件。取 \(f(z) = e^{i\pi/2} C(z) = i \cdot \dfrac{z-i}{z+i}\)。则 \(f'(i) = i \cdot (-i/2) = 1/2 > 0\)。\(\square\)

---

### 习题 5.2

**题目**: 求所有将单位圆盘 \(\mathbb{D}\) 共形映射到自身的 Möbius 变换。

**提示**: 利用 Schwarz 引理和自同构的性质。

**解答**: 这类变换恰为

$$f(z) = e^{i\theta} \frac{z - a}{1 - \overline{a}z}, \qquad |a| < 1, \quad \theta \in \mathbb{R}.$$

验证：当 \(|z| = 1\) 时，\(|z - a| = |\overline{z} - \overline{a}| = |1/z - \overline{a}| = |1 - \overline{a}z|/|z|\)，故 \(|f(z)| = 1\)。由最大模原理，\(|f(z)| < 1\) 对 \(|z| < 1\)。逆映射同理。\(\square\)

---

### 习题 5.3

**题目**: 证明 Möbius 变换将圆和直线映射为圆和直线。

**提示**: 先验证平移、伸缩、旋转和反演 \(z \mapsto 1/z\) 保持"广义圆"（圆或直线）不变。

**解答**: 任何 Möbius 变换可分解为平移、伸缩、旋转和反演的复合。平移、伸缩、旋转显然保持圆和直线。对反演 \(w = 1/z\)，圆的方程 \(|z - z_0| = R\) 变为

$$\left|\frac{1}{w} - z_0\right| = R \Rightarrow |1 - z_0 w| = R|w|.$$

这是关于 \(w\) 的二次方程，表示圆或直线（取决于原圆是否过原点）。\(\square\)

---

### 习题 5.4

**题目**: 设 \(f\) 在包含闭单位圆盘 \(\overline{\mathbb{D}}\) 的区域上全纯，且 \(f\) 将单位圆周映射到单位圆周。证明 \(f\) 将单位圆盘映射到单位圆盘。

**提示**: 用最大模原理。

**解答**: 对 \(|z| < 1\)，由最大模原理，\(|f(z)| \leq \max_{|\zeta| = 1} |f(\zeta)| = 1\)。若在某内点 \(z_0\) 处 \(|f(z_0)| = 1\)，则最大模原理推出 \(f\) 为常数，与 \(f\) 将圆周映到圆周矛盾。故 \(|f(z)| < 1\) 对 \(|z| < 1\)。\(\square\)

---

### 习题 5.5

**题目**: 求将区域 \(\{z : |z| < 1, \operatorname{Im}(z) > 0\}\)（上半单位圆盘）共形映射到单位圆盘 \(\mathbb{D}\) 的映射。

**提示**: 先用 \(z \mapsto \dfrac{1+z}{1-z}\) 将上半圆盘映射到第一象限，再用平方映射到上半平面，最后用 Cayley 变换。

**解答**: 

**第一步**: \(w_1 = \dfrac{1+z}{1-z}\) 将上半圆盘映到第一象限 \(\{w_1 : \operatorname{Re}(w_1) > 0, \operatorname{Im}(w_1) > 0\}\)。

**第二步**: \(w_2 = w_1^2\) 将第一象限映到上半平面。

**第三步**: \(w = \dfrac{w_2 - i}{w_2 + i}\)（Cayley 变换）将上半平面映到 \(\mathbb{D}\)。

复合映射：

$$f(z) = \frac{\left(\frac{1+z}{1-z}\right)^2 - i}{\left(\frac{1+z}{1-z}\right)^2 + i}.$$

\(\square\)
