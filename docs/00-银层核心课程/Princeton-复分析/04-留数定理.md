# 留数定理（Residue Theorem）

> **课程**: Princeton 复分析 | **章节**: Ch. 3 — 留数计算与应用
> **学习目标**: 熟练掌握留数的计算方法；能够运用留数定理计算复围道积分和实积分；理解辐角原理与 Rouché 定理

---

## 一、核心定义

### 定义 4.1（留数 / Residue）

**严格陈述**: 设 \(f\) 在穿孔圆盘 \(D_r^*(z_0) = \{z : 0 < |z - z_0| < r\}\) 上全纯，其 Laurent 展开为

$$f(z) = \sum_{n=-\infty}^{\infty} a_n (z - z_0)^n.$$

称系数 \(a_{-1}\) 为 \(f\) 在 \(z_0\) 处的**留数**（residue），记作

$$\operatorname{Res}(f, z_0) = a_{-1}.$$

等价地，对任意 \(0 < \rho < r\)，

$$\operatorname{Res}(f, z_0) = \frac{1}{2\pi i} \oint_{|z - z_0| = \rho} f(z)\, dz.$$

**直观解释**: 留数是 Laurent 展开中唯一的"积分敏感"系数——只有 \((z - z_0)^{-1}\) 项的积分沿闭合围道非零（因为 \(\oint (z-z_0)^n dz = 0\) 对 \(n \neq -1\)）。因此，计算围道积分的问题被转化为计算各奇点处留数的代数问题。

---

### 定义 4.2（简单极点处的留数计算法则）

**严格陈述**: 设 \(z_0\) 为 \(f\) 的**一阶极点**（simple pole），则

$$\operatorname{Res}(f, z_0) = \lim_{z \to z_0} (z - z_0) f(z).$$

若 \(f(z) = \dfrac{g(z)}{h(z)}\)，其中 \(g, h\) 在 \(z_0\) 处全纯，\(g(z_0) \neq 0\)，\(h(z_0) = 0\)，\(h'(z_0) \neq 0\)（即 \(z_0\) 为 \(h\) 的一阶零点），则

$$\operatorname{Res}(f, z_0) = \frac{g(z_0)}{h'(z_0)}.$$

**直观解释**: 一阶极点是留数计算中最常见的情形。第一个公式直接从 Laurent 展开读出：若 \(f(z) = a_{-1}(z-z_0)^{-1} + a_0 + \cdots\)，则 \((z-z_0)f(z) = a_{-1} + O(z-z_0)\)。第二个公式是计算有理函数型奇点留数的高效工具，避免了显式求 Laurent 展开。

---

### 定义 4.3（\(m\) 阶极点处的留数计算法则）

**严格陈述**: 设 \(z_0\) 为 \(f\) 的 \(m\) 阶极点，则

$$\operatorname{Res}(f, z_0) = \frac{1}{(m-1)!} \lim_{z \to z_0} \frac{d^{m-1}}{dz^{m-1}} \left[(z - z_0)^m f(z)\right].$$

**直观解释**: 对 \(m\) 阶极点，乘以 \((z-z_0)^m\) "消去"了奇性，得到一个全纯函数，其 \((m-1)\) 阶导数在 \(z_0\) 处的取值恰好与 \(a_{-1}\) 相关。当 \(m = 1\) 时，此公式退化为简单极点的公式。

> **双语对照**:
>
> ```lean4

>   simp [h, hg]
> ```

## 二、核心定理与完整证明

### 定理 4.1（留数定理 / Residue Theorem）

**定理陈述**: 设 \(U \subseteq \mathbb{C}\) 为单连通区域，\(z_1, z_2, \ldots, z_n\) 为 \(U\) 内互不相同的点。设 \(f\) 在 \(U \setminus \{z_1, \ldots, z_n\}\) 上全纯，\(\gamma\) 为 \(U\) 内一条分段光滑的简单闭合围道，且 \(z_1, \ldots, z_n\) 均在 \(\gamma\) 的内部。则

$$\oint_{\gamma} f(z)\, dz = 2\pi i \sum_{k=1}^{n} \operatorname{Res}(f, z_k).$$

**证明**:

**第一步：构造小圆周。**

对每个奇点 \(z_k\)，取充分小的 \(r_k > 0\) 使得闭圆盘 \(\overline{D}_{r_k}(z_k)\) 互不相交且均含于 \(\gamma\) 的内部。记 \(C_k\) 为圆周 \(|z - z_k| = r_k\)，取逆时针方向。

**第二步：应用多连通区域的 Cauchy 定理。**

考虑区域

$$V = \operatorname{Int}(\gamma) \setminus \bigcup_{k=1}^{n} D_{r_k}(z_k),$$

其中 \(\operatorname{Int}(\gamma)\) 为 \(\gamma\) 的内部。\(f\) 在 \(V\) 的闭包上全纯。由多连通区域的 Cauchy 定理，

$$\oint_{\partial V} f(z)\, dz = 0.$$

边界 \(\partial V\) 由外围道 \(\gamma\)（逆时针）和各小圆周 \(C_k\)（顺时针，即 \(C_k\) 的反向）组成。故

$$\oint_{\gamma} f(z)\, dz + \sum_{k=1}^{n} \oint_{-C_k} f(z)\, dz = 0,$$

即

$$\oint_{\gamma} f(z)\, dz = \sum_{k=1}^{n} \oint_{C_k} f(z)\, dz.$$

**第三步：利用 Laurent 展开计算各小圆周积分。**

在每个 \(z_k\) 附近，\(f\) 有 Laurent 展开

$$f(z) = \sum_{n=-\infty}^{\infty} a_n^{(k)} (z - z_k)^n.$$

沿 \(C_k\) 逐项积分。由 \(\oint_{C_k} (z - z_k)^n dz = 0\) 对 \(n \neq -1\)，以及 \(\oint_{C_k} (z - z_k)^{-1} dz = 2\pi i\)，得

$$\oint_{C_k} f(z)\, dz = 2\pi i \cdot a_{-1}^{(k)} = 2\pi i \cdot \operatorname{Res}(f, z_k).$$

代入第二步的等式即得所求。\(\square\)

> **证明要点提示**:
>
> 1. **局部到全局**: 留数定理的精髓在于将全局围道积分分解为各奇点邻域的局部贡献。每个奇点只贡献其留数乘以 \(2\pi i\)。
> 2. **Cauchy 定理的推广**: 留数定理是多连通区域 Cauchy 定理的直接推论。小圆周的引入是为了"隔离"各奇点，使 Laurent 展开可用。

---

### 定理 4.2（辐角原理 / Argument Principle）

**定理陈述**: 设 \(f\) 在单连通区域 \(U\) 上亚纯（meromorphic），\(\gamma\) 为 \(U\) 内简单闭合围道，\(f\) 在 \(\gamma\) 上无零点和极点。则

$$\frac{1}{2\pi i} \oint_{\gamma} \frac{f'(z)}{f(z)}\, dz = N - P,$$

其中 \(N\) 为 \(f\) 在 \(\gamma\) 内部零点的个数（计重数），\(P\) 为极点个数（计重数）。

**证明思路**: \(\dfrac{f'}{f}\) 在 \(f\) 的零点 \(z_0\) 附近，若 \(f(z) = (z-z_0)^m g(z)\)（\(g(z_0) \neq 0\)），则

$$\frac{f'(z)}{f(z)} = \frac{m}{z - z_0} + \frac{g'(z)}{g(z)}.$$

故 \(\operatorname{Res}(f'/f, z_0) = m\)（零点贡献 +m）。类似地，在 \(m\) 阶极点处贡献 \(-m\)。由留数定理即得结论。\(\square\)

## 三、习题

### 习题 4.1

**题目**: 计算 \(f(z) = \dfrac{e^z}{z^2 + 1}\) 在其所有孤立奇点处的留数。

**提示**: 分母 \(z^2 + 1 = (z-i)(z+i)\)，故有两个一阶极点。

**解答**: 奇点为 \(z = i\) 和 \(z = -i\)。

$$\operatorname{Res}(f, i) = \lim_{z \to i} (z - i) \frac{e^z}{(z-i)(z+i)} = \frac{e^i}{2i}.$$

$$\operatorname{Res}(f, -i) = \lim_{z \to -i} (z + i) \frac{e^z}{(z-i)(z+i)} = \frac{e^{-i}}{-2i} = -\frac{e^{-i}}{2i}.$$

\(\square\)

---

### 习题 4.2

**题目**: 用留数定理计算 \(\displaystyle\int_{-\infty}^{\infty} \frac{dx}{1 + x^4}\)。

**提示**: 考虑上半平面的围道积分，利用 \(1 + z^4 = 0\) 的根。

**解答**: \(z^4 = -1 = e^{i\pi}\)，根为 \(z_k = e^{i(\pi + 2k\pi)/4}\)，\(k = 0, 1, 2, 3\)。上半平面的根为 \(z_0 = e^{i\pi/4}\) 和 \(z_1 = e^{i3\pi/4}\)。

对 \(f(z) = 1/(1 + z^4)\)，各极点为一阶。由公式 \(\operatorname{Res}(f, z_k) = 1/(4z_k^3) = z_k/(4z_k^4) = -z_k/4\)（因为 \(z_k^4 = -1\)）。

$$\operatorname{Res}(f, z_0) = -\frac{e^{i\pi/4}}{4}, \quad \operatorname{Res}(f, z_1) = -\frac{e^{i3\pi/4}}{4}.$$

留数之和：

$$-\frac{1}{4}\left(e^{i\pi/4} + e^{i3\pi/4}\right) = -\frac{1}{4}\left(\frac{1+i}{\sqrt{2}} + \frac{-1+i}{\sqrt{2}}\right) = -\frac{i}{2\sqrt{2}}.$$

由留数定理，沿上半大圆弧围道（半径 \(R \to \infty\)），弧上积分趋于零（\(|f(z)| \sim 1/R^4\)，弧长 \(\pi R\)）。故

$$\int_{-\infty}^{\infty} \frac{dx}{1+x^4} = 2\pi i \cdot \left(-\frac{i}{2\sqrt{2}}\right) = \frac{\pi}{\sqrt{2}}.$$

\(\square\)

---

### 习题 4.3

**题目**: 计算 \(\displaystyle\oint_{|z|=2} \frac{z^2 + 1}{(z-1)^2(z+1)}\, dz\)。

**提示**: 确定围道内部的奇点及其类型，分别计算留数。

**解答**: 围道 \(|z| = 2\) 内部包含 \(z = 1\)（二阶极点）和 \(z = -1\)（一阶极点）。

在 \(z = -1\)（简单极点）：

$$\operatorname{Res}(f, -1) = \lim_{z \to -1} (z+1) \frac{z^2+1}{(z-1)^2(z+1)} = \frac{2}{4} = \frac{1}{2}.$$

在 \(z = 1\)（二阶极点）：令 \(g(z) = (z-1)^2 f(z) = \dfrac{z^2+1}{z+1}\)。则

$$\operatorname{Res}(f, 1) = \lim_{z \to 1} g'(z) = \lim_{z \to 1} \frac{2z(z+1) - (z^2+1)}{(z+1)^2} = \frac{4-2}{4} = \frac{1}{2}.$$

积分 \(= 2\pi i \cdot (1/2 + 1/2) = 2\pi i\)。\(\square\)

---

### 习题 4.4

**题目**: 用留数定理计算 \(\displaystyle\int_0^{2\pi} \frac{d\theta}{2 + \cos\theta}\)。

**提示**: 令 \(z = e^{i\theta}\)，则 \(\cos\theta = (z + z^{-1})/2\)，\(d\theta = dz/(iz)\)。将实积分化为单位圆周上的围道积分。

**解答**: 令 \(z = e^{i\theta}\)，\(d\theta = dz/(iz)\)，\(\cos\theta = (z^2 + 1)/(2z)\)。

$$I = \oint_{|z|=1} \frac{1}{2 + \frac{z^2+1}{2z}} \cdot \frac{dz}{iz} = \oint_{|z|=1} \frac{2z}{4z + z^2 + 1} \cdot \frac{dz}{iz} = \frac{2}{i} \oint_{|z|=1} \frac{dz}{z^2 + 4z + 1}.$$

分母零点：\(z = -2 \pm \sqrt{3}\)。在单位圆内的是 \(z_0 = -2 + \sqrt{3}\)（\(|-2+\sqrt{3}| \approx 0.268 < 1\)），另一根 \(|-2-\sqrt{3}| > 1\) 在外。

$$\operatorname{Res}\left(\frac{1}{z^2+4z+1}, z_0\right) = \frac{1}{2z_0 + 4} = \frac{1}{2\sqrt{3}}.$$

故

$$I = \frac{2}{i} \cdot 2\pi i \cdot \frac{1}{2\sqrt{3}} = \frac{2\pi}{\sqrt{3}}.$$

\(\square\)

---

### 习题 4.5

**题目**: 设 \(f(z) = z^7 - 5z^3 + z - 2\)。用 Rouché 定理证明 \(f\) 在单位圆盘 \(|z| < 1\) 内恰有 3 个零点（计重数）。

**提示**: 在单位圆周上比较 \(-5z^3\) 与 \(z^7 + z - 2\) 的大小。

**解答**: 在单位圆周 \(|z| = 1\) 上，令 \(g(z) = -5z^3\)，\(h(z) = z^7 + z - 2\)。

\(|g(z)| = 5\)，\(|h(z)| \leq |z^7| + |z| + 2 = 4 < 5 = |g(z)|\)。

由 Rouché 定理，\(f = g + h\) 与 \(g\) 在 \(|z| < 1\) 内有相同数目的零点。\(g(z) = -5z^3\) 在 \(z = 0\) 处有三重零点，故 \(f\) 在 \(|z| < 1\) 内恰有 3 个零点。\(\square\)
