# 多元函数与偏导数：多变量微分学基础

## 1. 引言

多元函数的微积分是单变量微积分的自然推广，但其理论远比单变量情形丰富和复杂。本章将建立多元函数的连续性、偏导数和可微性的严格理论。

## 2. 多元函数的极限与连续

### 2.1 极限的定义

**定义 2.1（极限）**。设 $f: D \subseteq \mathbb{R}^n \to \mathbb{R}^m$。称 $\lim_{x \to a} f(x) = L$，如果：

$$\forall \varepsilon > 0, \exists \delta > 0: 0 < |x - a| < \delta \Rightarrow |f(x) - L| < \varepsilon$$

### 2.2 连续性的定义

**定义 2.2（连续性）**。$f$ 在 $a$ 处**连续**，如果 $\lim_{x \to a} f(x) = f(a)$。

**定理 2.3**。连续函数的和、差、积、商（分母非零）和复合仍连续。

### 2.3 路径极限

**重要现象**：在多元函数中，极限可能沿不同路径趋近时取不同值。

**例 2.4**。$f(x, y) = \frac{xy}{x^2 + y^2}$（补充 $f(0, 0) = 0$）。

沿 $y = kx$：$f(x, kx) = \frac{kx^2}{x^2 + k^2x^2} = \frac{k}{1 + k^2}$。

不同 $k$ 给出不同极限，所以 $\lim_{(x,y) \to (0,0)} f(x, y)$ 不存在。

## 3. 偏导数

### 3.1 定义

**定义 3.1（偏导数）**。$f(x, y)$ 对 $x$ 的**偏导数**：

$$\frac{\partial f}{\partial x} = \lim_{h \to 0} \frac{f(x + h, y) - f(x, y)}{h}$$

类似定义 $\frac{\partial f}{\partial y}$。

### 3.2 几何意义

$\frac{\partial f}{\partial x}(a, b)$ 是曲面 $z = f(x, y)$ 与平面 $y = b$ 的交线在 $x = a$ 处的切线斜率。

### 3.3 高阶偏导数

**定义 3.2**。二阶偏导数：

$$\frac{\partial^2 f}{\partial x^2}, \quad \frac{\partial^2 f}{\partial y^2}, \quad \frac{\partial^2 f}{\partial x \partial y}, \quad \frac{\partial^2 f}{\partial y \partial x}$$

**定理 3.3（Clairaut定理）**。若 $\frac{\partial^2 f}{\partial x \partial y}$ 和 $\frac{\partial^2 f}{\partial y \partial x}$ 在区域内连续，则：

$$\frac{\partial^2 f}{\partial x \partial y} = \frac{\partial^2 f}{\partial y \partial x}$$

## 4. 全微分与可微性

### 4.1 全微分

**定义 4.1（全微分）**。$f(x, y)$ 在 $(a, b)$ 处的**全微分**为：

$$df = \frac{\partial f}{\partial x}dx + \frac{\partial f}{\partial y}dy$$

### 4.2 可微性的严格定义

**定义 4.2（可微性）**。$f$ 在 $a$ 处**可微**，如果存在线性映射 $L$ 使得：

$$\lim_{h \to 0} \frac{|f(a + h) - f(a) - L(h)|}{|h|} = 0$$

**定理 4.3**。若所有偏导数在 $a$ 的邻域内连续，则 $f$ 在 $a$ 处可微。

### 4.3 链式法则

**定理 4.4（多元链式法则）**。设 $z = f(x, y)$，$x = x(t)$，$y = y(t)$。则：

$$\frac{dz}{dt} = \frac{\partial z}{\partial x}\frac{dx}{dt} + \frac{\partial z}{\partial y}\frac{dy}{dt}$$

## 5. 梯度与方向导数

### 5.1 梯度

**定义 5.1（梯度）**：

$$\nabla f = \left(\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z}\right)$$

### 5.2 方向导数

**定义 5.2（方向导数）**。$f$ 沿单位向量 $u$ 的**方向导数**：

$$D_u f = \nabla f \cdot u$$

**定理 5.3**。梯度方向是函数增长最快的方向，且最大增长率为 $|\nabla f|$。

## 6. 例题

**例 6.1**。设 $f(x, y) = x^2y + y^3$。求 $\nabla f$ 和 $f$ 在 $(1, 2)$ 沿方向 $(3, 4)$ 的方向导数。

*解*：$\frac{\partial f}{\partial x} = 2xy$，$\frac{\partial f}{\partial y} = x^2 + 3y^2$。

$\nabla f(1, 2) = (4, 13)$。

方向 $(3, 4)$ 的单位向量为 $(\frac{3}{5}, \frac{4}{5})$。

$D_u f = (4, 13) \cdot (\frac{3}{5}, \frac{4}{5}) = \frac{12 + 52}{5} = \frac{64}{5}$。$\square$

## 8. 总结

多元函数的偏导数和全微分是理解多变量微积分的核心概念。方向导数和梯度提供了函数在空间中变化的几何直观，而链式法则则是计算复合函数导数的基本工具。这些概念将在后续的多元积分和向量分析中得到深入应用。
