# 第一部分：快速开始指南

## 欢迎使用 FormalMath

FormalMath 是一个面向数学教育的综合性知识库，致力于：

- 🎯 **建立完整知识体系**：从基础数学到现代数学前沿的结构化内容
- 🧠 **数学家理念体系**：62位数学家的思想、贡献与现代影响深度分析
- 🌍 **国际权威对齐**：与MIT、Harvard、Stanford、Princeton等九校121+门课程详细对应
- 💻 **形式化证明**：20个Lean4定理形式化，与Mathlib4 v4.29.0对齐
- 🤖 **智能学习系统**：认知诊断、学习评估、自适应学习路径三大系统

### 核心统计数据

| 资源类型 | 数量 | 说明 |
|---------|------|------|
| 数学文档 | 6,500+ | 15个数学分支结构化内容 |
| 数学家理念 | 62位 | 从阿基米德到肖尔策 |
| 国际课程映射 | 121+门 | 九校课程详细对应 |
| Lean4定理 | 20个 | 可运行的形式化证明 |
| 智能学习系统 | 3大系统 | 认知诊断、评估、自适应路径 |

## 第三步：开始使用智能学习系统

**功能**: 评估您在 L0-L3 四个知识层次的能力

**知识层次说明**：

| 层次 | 名称 | 能力要求 |
|------|------|----------|

| L2 | 定理应用层 | 复杂证明、综合应用 |
| L3 | 综合证明层 | 创新性研究、开放问题 |

3. 查看诊断报告，了解薄弱环节

## 二、智能学习系统

### 认知诊断系统

基于 HCD（层次约束感知认知诊断）框架，评估 L0-L3 四个知识层次的能力。

**诊断报告包含**：
- 综合评估：整体水平和置信度
- L0-L3能力分析：各层次得分和等级

- 薄弱环节：需要重点提升的知识点
- 个性化建议：学习路径和练习推荐

### 自适应学习路径系统

基于知识图谱和 A* 算法，生成个性化学习路径。

**核心功能**：
- 学习者特征分析
- 路径生成（A*算法）
- 实时调整
- 间隔重复复习

### 评估系统

多维度、过程化、发展性的学习评估平台。

**四维评价模型**：

| 维度 | 权重 | 子指标 |

| 问题解决能力 | 25% | 问题分析、策略选择、执行验证 |

# 第五部分：故障排除指南

## 文档访问问题

### 无法打开 Markdown 文档

**解决方案**:
- 使用 VS Code + Markdown 扩展
- 使用 Typora 等专业编辑器
- 转换为 HTML: `pandoc file.md -o file.html`

### 公式显示不正确

**解决方案**:
- 使用支持 MathJax 的阅读器
- VS Code + Markdown Preview Enhanced
- 在线渲染: `pandoc file.md -o file.html --mathjax`

### 中文显示乱码

**解决方案**:
- 确保文件为 UTF-8 编码
- 转换编码: `iconv -f GBK -t UTF-8 input.md > output.md`

## 智能学习系统问题

### 认知诊断系统无法启动

**排查步骤**:
1. 检查 Python 版本（需要 3.9+）
2. 安装依赖: `pip install -r requirements.txt`
3. 检查端口占用
4. 查看日志

### 学习路径生成失败

**排查步骤**:
1. 检查学习者档案
2. 检查知识图谱连接
3. 简化目标概念数量
