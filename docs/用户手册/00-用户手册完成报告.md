## 📱 输出格式说明

### Markdown 格式

- **用途**: 源码格式，便于版本控制和编辑
- **阅读方式**: 任何 Markdown 阅读器或编辑器
- **推荐工具**: VS Code, Typora, Obsidian

### HTML 格式（在线文档）

- **文件**: `index.html`
- **用途**: 在线浏览和阅读
- **特点**:
  - 响应式设计，支持移动端
  - 现代化样式，易于阅读
  - 可直接打印为 PDF
  - 支持代码高亮
  - 表格样式优化

**打开方式**:
```bash
# 直接用浏览器打开
double-click index.html

# 或使用本地服务器
python -m http.server 8000
# 然后访问 
```

### PDF 格式

- **生成方式**: 参考 `生成PDF指南.md`
- **推荐方法**: 使用 pandoc
- **命令**:
```bash
pandoc 用户手册-合并版.md -o 用户手册.pdf --pdf-engine=xelatex -V CJKmainfont="SimSun" --toc
```

## 🚀 使用建议

### 新用户推荐流程

```
1. 打开 index.html（在线文档）
   或阅读 README.md
   ↓
2. 阅读 01-快速开始指南
   ↓
3. 根据水平选择学习起点
   ↓
4. 深入使用时参考 02-功能使用说明
   ↓
5. 规划长期学习使用 03-学习路径指南
   ↓
6. 遇到问题查阅 04-FAQ 或 05-故障排除
```

### 教师/机构使用建议

1. **打印版本**: 使用 `用户手册-完整版.md` 或生成 PDF
2. **在线嵌入**: 使用 `index.html` 嵌入到学习平台
3. **内容定制**: 基于 Markdown 源文件进行定制

## 🔧 维护说明

### 更新流程

1. 编辑相应的 Markdown 文件
2. 运行 `python build_html.py` 更新 HTML
3. 如有需要，重新生成 PDF

- 所有 Markdown 源文件应纳入版本控制

- PDF 文件较大，建议不纳入版本控制
