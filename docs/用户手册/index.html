第一步：选择您的起点

根据您的数学背景，选择合适的学习起点：

🌱 初学者（高中/大一水平）

如果您符合以下情况：
- 刚接触高等数学
- 需要巩固数学基础
- 准备大学数学课程

推荐学习路径：

  1. 集合论基础 → docs/01-基础数学/集合论/01-集合论基础-国际标准版.md

  3. 线性代数 → docs/02-代数结构/线性代数/线性代数-深度扩展版.md

预计学习时间: 3-4个月

🌿 中级学习者（大二/大三水平）

如果您符合以下情况：
- 已掌握线性代数和微积分
- 希望深入学习抽象代数或分析学
- 准备研究生入学考试

推荐学习路径：

  1. 群论 → docs/02-代数结构/群论/群论-深度扩展版.md

预计学习时间: 4-6个月

如果您符合以下情况：
- 正在攻读数学研究生学位
- 从事数学研究
- 需要深入理解现代数学前沿

推荐学习路径：

  1. 格洛腾迪克数学理念 → 数学家理念体系/格洛腾迪克数学理念/02-数学内容深度分析/

第二步：了解文档命名规则

FormalMath 的文档采用统一的命名规则，帮助您快速识别内容深度：

后缀 含义 适合人群 特点
-基础版.md 概念介绍 初学者 直观解释，较少证明
-增强版.md 定理+例子 中级 包含主要定理和应用
-深度扩展版.md 完整证明 高级 严格证明，深入分析

docs/02-代数结构/群论/
├── 群论-基础版.md          # 概念介绍

诊断算法原理

HCD 算法的核心特点：

  1. 层次感知：显式建模 L0→L1→L2→L3 的依赖关系
  2. 约束优化：使用层次约束指导知识状态估计
  3. 多维度输出：同时输出知识点掌握度和层次能力
  4. 可解释性：诊断结果有明确的层次结构解释

一、入门问题

Q1: 我完全没有任何高等数学基础，可以学习FormalMath吗？

A: 可以！FormalMath 提供从基础到前沿的完整内容。

推荐起点：

  1. docs/01-基础数学/集合论/01-集合论基础-国际标准版.md

Q2: FormalMath 适合什么水平的学习者？

A: FormalMath 涵盖 L0-L3 四个知识层次：

层次 适合人群 内容深度
L0 高中/大一 概念介绍、直观理解
L1 大二/大三 形式化定义、基本定理
L2 研究生 复杂证明、综合应用
L3 研究者 前沿内容、开放问题

A: 不需要。

  • 形式化证明: 需要学习 Lean4（可选）
  • 智能学习系统: 通过界面使用（无需编程）

时间: 约6个月

情况B：大二/大三数学系学生

路径: 实分析 → 抽象代数 → 拓扑学
时间: 约4个月

路径: 根据研究方向选择专题
例如: 概形理论 → 层上同调 → 导出范畴

三、智能学习系统问题

Q11: 认知诊断系统是如何工作的？

A: 基于 HCD（层次约束感知认知诊断）框架：

  1. 测试: 完成30-45分钟的诊断测试
  2. 分析: 系统分析 L0-L3 各层次表现
  3. 报告: 生成能力分析和薄弱环节报告
  4. 建议: 提供个性化学习路径建议

诊断准确度: >85%

Q12: 认知诊断测试需要多长时间？

A:

  • 默认设置: 30题，约30-45分钟
  • 快速诊断: 15题，约15-20分钟
  • 深度诊断: 50题，约60-75分钟

Q13: 自适应学习路径系统如何生成路径？

A: 使用 A* 算法结合多目标优化：

  1. 分析学习者: 认知风格、能力水平、可用时间
  2. 知识缺口分析: 识别达到目标所需的概念
  3. 拓扑排序: 根据先修关系确定顺序
  4. 路径优化: 平衡时间、深度、广度、平滑度

Q14: 学习路径会动态调整吗？

A: 是的，系统会实时监控：

检测指标 触发条件 调整策略
时间比例 > 2.0 降低难度
错误率 > 50% 插入复习节点
帮助请求 ≥ 3次 添加支持内容
掌握度 ≥ 0.9 提高难度

Q15: 评估系统的四个维度是什么意思？

A:

维度 权重 说明
知识掌握度 30% 概念理解、定理掌握、方法熟练
技能熟练度 25% 计算、证明、建模能力
问题解决能力 25% 分析、策略、验证能力
创新思维能力 20% 发散性、创造性、批判性思维

git clone failed 配置 git 代理

cd docs/09-形式化证明/Lean4/

lake update
lake build
