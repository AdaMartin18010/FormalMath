## 🚀 快速开始

### 5分钟快速上手

1. **确定您的水平**
   - 初学者 → 阅读 [01-快速开始指南#初学者](./01-快速开始指南.md#初学者高中大一水平)
   - 中级 → 阅读 [01-快速开始指南#中级](./01-快速开始指南.md#中级学习者大二大三水平)
   - 高级 → 阅读 [01-快速开始指南#高级](./01-快速开始指南.md#高级学习者研究生水平)

2. **选择第一个文档**
   - 根据您的水平，选择对应的推荐文档
   - 从 `-基础版` 或 `-国际标准版` 开始

3. **了解文档命名规则**
   - `-基础版`: 概念介绍
   - `-增强版`: 定理+例子
   - `-深度扩展版`: 完整证明
   - `-国际标准版`: 国际课程对齐

### 推荐阅读顺序

```
首次使用 FormalMath
    ↓
阅读 01-快速开始指南 (15分钟)
    ↓
选择适合水平的学习起点
    ↓
开始阅读第一个文档
    ↓
深入使用时参考 02-功能使用说明
    ↓
规划长期学习使用 03-学习路径指南
    ↓
遇到问题查阅 04-FAQ 或 05-故障排除指南
```

## 📖 项目简介

FormalMath 是一个**面向数学教育的综合性知识库**，致力于：

- 🎯 **建立完整知识体系**：从基础数学到现代数学前沿的结构化内容
- 🧠 **数学家理念体系**：62位数学家的思想、贡献与现代影响深度分析
- 🌍 **国际权威对齐**：与MIT、Harvard、Stanford、Princeton等九校121+门课程详细对应
- 💻 **形式化证明**：20个Lean4定理形式化，与Mathlib4 v4.29.0对齐
- 🤖 **智能学习系统**：认知诊断、学习评估、自适应学习路径三大系统

### 核心统计数据

| 资源类型 | 数量 | 说明 |
|---------|------|------|
| 数学文档 | 6,500+ | 15个数学分支结构化内容 |
| 数学家理念 | 62位 | 从阿基米德到肖尔策 |
| 国际课程映射 | 121+门 | 九校课程详细对应 |
| Lean4定理 | 20个 | 可运行的形式化证明 |
| 智能学习系统 | 3大系统 | 认知诊断、评估、自适应路径 |
