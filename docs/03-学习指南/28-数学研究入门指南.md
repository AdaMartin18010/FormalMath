# 数学研究入门指南

> 本指南为有志于从事数学研究的读者提供系统性的指导，涵盖研究方向选择、文献检索、笔记管理、学术交流及学术道德等方面。

---

## 一、如何选择研究方向

### 1.1 认识自己的兴趣与能力

#### 自我评估维度
| 维度 | 评估问题 | 重要性 |
|------|----------|--------|
| **兴趣** | 什么数学问题让你感到兴奋？ | ★★★★★ |
| **能力** | 你在哪些领域表现突出？ | ★★★★☆ |
| **背景** | 哪些基础知识已经掌握？ | ★★★★☆ |
| **资源** | 周围有哪些导师和资源？ | ★★★☆☆ |
| **前景** | 该领域的就业/发展前景如何？ | ★★★☆☆ |

#### 探索方法
1. **广泛阅读**：浏览不同领域的教材和综述
2. **参加讲座**：听不同方向的学术报告
3. **交流讨论**：与学长学姐、导师交流
4. **尝试小课题**：做小型研究项目体验
5. **跟随直觉**：选择让你"心动"的问题

### 1.2 主流数学研究方向概览

#### 纯数学分支

| 分支 | 核心问题 | 所需基础 | 活跃领域 |
|------|----------|----------|----------|
| **代数几何** | 代数方程的几何 | 交换代数、复分析 | 模空间、枚举几何 |
| **数论** | 整数性质 | 代数、分析 | 朗兰兹纲领、椭圆曲线 |
| **微分几何** | 曲线曲面的几何 | 拓扑、PDE | 几何流、度量几何 |
| **代数拓扑** | 拓扑不变量 | 代数、范畴 | 同伦论、K理论 |
| **偏微分方程** | 物理方程的数学 | 分析、泛函 | 流体方程、几何PDE |
| **动力系统** | 迭代与演化 | 分析、拓扑 | 混沌理论、遍历论 |
| **表示论** | 对称性的代数 | 代数、李群 | 几何表示论 |
| **数学物理** | 物理的数学基础 | 物理、几何 | 弦论、量子场论 |
| **数理逻辑** | 数学的基础 | 集合论 | 模型论、证明论 |

#### 应用数学方向

| 分支 | 应用领域 | 核心技能 |
|------|----------|----------|
| **计算数学** | 科学计算 | 数值分析、编程 |
| **优化理论** | 运筹学、ML | 凸分析、算法 |
| **概率统计** | 数据科学 | 随机过程、推断 |
| **金融数学** | 量化金融 | 随机分析、PDE |
| **生物数学** | 生命科学 | 建模、动力系统 |
| **信息论** | 通信、编码 | 组合、概率 |

### 1.3 选择策略

#### 短期策略（本科/硕士）
- 打好广泛的数学基础
- 尝试2-3个不同方向
- 跟随课程项目深入
- 参加暑期学校

#### 长期策略（博士/职业）
- 选择有持续兴趣的领域
- 考虑导师的研究方向
- 评估学术资源和机会
- 思考职业发展规划

#### 经典建议
> "选择研究问题时，找一个你认为重要的问题，一个你认为你能解决的问题，以及一个别人还没解决的问题。这三者的交集就是你的机会。" —— John Tukey

## 二、文献检索方法

### 2.1 主要数学数据库

#### zbMATH（最权威）
- **网址**: zbmath.org
- **特点**: 最完整的数学文献数据库
- **覆盖**: 从19世纪至今的数学文献
- **功能**: 作者索引、分类检索、引用分析
- **使用**: 机构订阅或个人付费

#### MathSciNet（带评论）
- **网址**: mathscinet.ams.org
- **特点**: 美国数学学会出品
- **特色**: 专家撰写的文献评论
- **覆盖**: 数学及相关领域
- **使用**: AMS会员或机构订阅

#### arXiv（最新进展）
- **网址**: arxiv.org
- **特点**: 免费预印本服务器
- **优势**: 第一时间获取最新研究
- **分类**: math.*（纯数学）和其他
- **订阅**: 邮件订阅感兴趣的分类

### 2.2 文献检索技巧

#### 检索策略
```
1. 从综述文章开始
   - 查找领域综述 (survey/review)
   - 阅读权威教科书
   
2. 追溯经典文献
   - 找到领域的奠基性论文
   - 追踪引用该论文的后续工作
   
3. 跟踪最新进展
   - 订阅arXiv每日更新
   - 关注顶级期刊新刊
   - 参加学术会议
   
4. 建立文献网络
   - 识别领域核心作者
   - 了解作者间的合作关系
   - 绘制文献引用图谱
```

#### 检索示例
```
关键词检索:
- "algebraic geometry" + "moduli space"
- "Riemann hypothesis" + "prime number"
- author:"Tao, Terence"

MSC分类检索:
- @: 代数几何的上同调理论
- @: 解析数论
- @: 整体微分几何
```

### 2.3 文献阅读方法

#### 三遍阅读法

**第一遍（5-10分钟）**：
- 读标题、摘要、结论
- 浏览章节标题和图表
- 判断相关性

**第二遍（1-2小时）**：
- 读全文但不深究细节
- 关注主要结果和方法
- 标记关键定理和证明思路
- 记录问题和想法

**第三遍（数小时）**：
- 深入理解证明细节
- 验证关键步骤
- 复现核心计算
- 与自己的工作联系

#### 笔记系统
```
文献笔记模板:

【基本信息】
- 标题:
- 作者:
- 年份/期刊:
- 关键词:

【主要内容】
- 研究问题:
- 主要结果:
- 核心方法:

【个人思考】
- 创新点:
- 局限性:
- 与我的关系:
- 待解决问题:

【引用关系】
- 重要参考文献:
```

## 五、学术道德规范

### 5.1 基本原则

#### 核心准则
1. **诚实**：如实报告结果
2. **原创**：承认他人贡献
3. **公正**：公平评价同行
4. **责任**：对研究负责

### 5.2 引用规范

#### 何时引用
✅ **必须引用**：
- 直接引用他人文字
- 使用他人定理/结果
- 借鉴他人方法
- 受他人思想启发

❌ **不需要引用**：
- 数学常识
- 标准教科书中的基础结果
- 自己已发表的工作（可提及）

#### 引用格式
```
文中引用:
- 根据Smith [1] 的结果...
- 这一问题已被广泛研究 [2,3,4]。
- 见文献 [5, 第3章] 中的详细讨论。

参考文献列表:
[1] J. Smith, Title of Paper, Journal, vol, year.
[2] A. Author, Book Title, Publisher, year.
```

### 5.3 作者署名

#### 署名标准（ICMJE建议）
满足以下所有条件才能署名：
1. 实质性参与研究设计/执行
2. 参与论文起草或修改
3. 同意最终版本
4. 对研究内容负责

#### 作者顺序
- **纯数学**：通常按字母顺序
- **应用数学**：按贡献大小
- **通讯作者**：负责对外联系

### 5.4 常见不当行为

| 行为 | 说明 | 后果 |
|------|------|------|
| **抄袭** | 不注明来源使用他人成果 | 学术处分 |
| **伪造数据** | 编造或篡改数据 | 撤稿、禁发 |
| **一稿多投** | 同时投多个期刊 | 拒稿 |
| **重复发表** | 已发表内容再次发表 | 撤稿 |
| **不当署名** | 挂名或遗漏贡献者 | 学术纠纷 |
| **审稿泄露** | 利用审稿信息 | 职业禁入 |

## 六、职业发展建议

### 6.1 学术职业路径

```
本科 → 硕士 → 博士 → 博士后 → 教职
       ↓        ↓        ↓
    工业界   工业界   研究员
       ↓        ↓
    数据科学  量化金融
```

### 6.2 建立学术声誉

#### 长期策略
1. **质量优先**：发表有影响力的工作
2. **持续产出**：保持稳定的研究节奏
3. **学术交流**：积极参与学术社区
4. **教学服务**：承担教学和社会服务
5. **学术诚信**：维护良好的职业记录

#### 短期建议
- 完成博士学位要求
- 发表2-3篇高质量论文
- 建立合作网络
- 参加重要会议
- 申请研究基金
