### 1.4 公式排版规范

#### 行内公式与显示公式
```latex
% 行内公式
函数 $f(x) = x^2$ 在 $x=0$ 处可导。

% 独立显示公式
\begin{equation}
    \int_a^b f(x) \, dx = F(b) - F(a)
    \label{eq:fundamental}
\end{equation}

% 多行对齐
\begin{align}
    (a+b)^2 &= a^2 + 2ab + b^2 \\
    &= b^2 + 2ab + a^2
\end{align}

% 多行无编号
\begin{align*}
    e^x &= \sum_{n=0}^{\infty} \frac{x^n}{n!} \\
    &= 1 + x + \frac{x^2}{2!} + \cdots
\end{align*}
```

#### 矩阵排版
```latex
\begin{pmatrix}  % 圆括号
    a & b \\
    c & d
\end{pmatrix}

\begin{bmatrix}  % 方括号
    1 & 0 \\
    0 & 1
\end{bmatrix}

\begin{vmatrix}  % 行列式
    a & b \\
    c & d
\end{vmatrix}
```

#### 分段函数
```latex
f(x) = \begin{cases}
    x^2, & x \geq 0, \\
    -x, & x < 0.
\end{cases}
```

## 二、数学论文结构

### 2.1 标准论文结构

```
1. 标题 (Title)
2. 摘要 (Abstract)
3. 关键词 (Keywords)
4. 引言 (Introduction)
5. 预备知识 (Preliminaries)
6. 主要结果 (Main Results)
7. 证明 (Proofs)
8. 应用/例子 (Applications/Examples)
9. 结论 (Conclusion)
10. 致谢 (Acknowledgments)
11. 参考文献 (References)
12. 附录 (Appendix) - 可选
```

### 2.2 各部分写作要点

#### 标题 (Title)
**原则**：
- 简洁准确，通常10-15个词
- 突出核心贡献
- 避免缩写（除非广为人知）
- 使用关键词便于检索

**示例**：
- 好："A New Proof of the Prime Number Theorem"
- 差："Some Results on Numbers"

#### 摘要 (Abstract)
**结构（通常150-250词）**：
1. **背景** (1-2句)：研究背景和问题
2. **方法** (1-2句)：使用的方法
3. **结果** (2-3句)：主要发现
4. **意义** (1句)：贡献和价值

**写作要点**：
- 独立完成（不引用文献）
- 包含关键词
- 避免公式（必要时可包含1-2个）
- 用第三人称

#### 引言 (Introduction)
**内容组织**：
1. **研究背景**：为什么这个问题重要？
2. **文献综述**：前人做了什么？
3. **研究空白**：还有什么没解决？
4. **本文贡献**：我们做了什么？
5. **论文结构**：后续章节安排

**写作技巧**：
- 由宏观到具体
- 引用关键文献
- 突出创新点
- 设定合理期望

#### 主要结果 (Main Results)
**呈现方式**：
- 先陈述定理，后给证明
- 按逻辑顺序排列
- 标注依赖关系
- 图示辅助理解

**定理写作模板**：
```
定理 X.Y (定理名称). 
设 ... (条件). 则 ... (结论).
```

### 2.3 论文长度建议

| 论文类型 | 页数 | 摘要长度 |
|----------|------|----------|
| 注 (Note) | 3-5页 | 50-100词 |
| 研究论文 | 10-25页 | 150-250词 |
| 综述论文 | 20-50页 | 200-300词 |
| 学位论文 | 50-200页 | 300-500词 |

## 三、证明书写技巧

### 3.1 证明结构原则

#### 清晰分层
```
证明: (分情况讨论)

情况1: [情况描述]
[详细论证]

情况2: [情况描述]
[详细论证]

综上，结论成立。 □
```

#### 常见证明策略标识
- **直接证明**："我们有..."
- **反证法**："假设结论不成立..."
- **归纳法**："对n作数学归纳法..."
- **构造法**："构造如下..."
- **分类讨论**："分以下情况..."

### 3.2 证明书写规范

#### 语言规范
✅ **推荐写法**：
- "设 $x$ 为满足...的任意元素"
- "由假设可知..."
- "根据定理X.Y..."
- "由此可得..."
- "证毕。"

❌ **避免写法**：
- "显然..."（除非真的显然）
- "不难看出..."（同样）
- "众所周知..."（引用来源）
- "易得..."（给出步骤）
- "同理可证..."（如非显然，给出要点）

#### 符号使用规范
```latex
% 证明结束符号
\qed  % 证毕方块
\qedhere  % 在公式后

% 证明环境
\begin{proof}
证明内容...
\end{proof}

% 带名的证明
\begin{proof}[定理1.1的证明]
...
\end{proof}
```

### 3.3 长篇证明的组织

#### 分步骤证明
```latex
\begin{proof}
我们将证明分为三个步骤。

\textbf{步骤1:} [步骤描述]
[详细证明]

\textbf{步骤2:} [步骤描述]
[详细证明]

\textbf{步骤3:} [步骤描述]
[详细证明]

综合以上步骤，结论得证。
\end{proof}
```

#### 引用引理
```latex
\begin{proof}
由引理\ref{lem:1}，我们有...

应用引理\ref{lem:2}，得到...

结合上述结果，即得所需结论。
\end{proof}
```

## 四、学术演讲准备

### 4.1 演讲结构

#### 标准学术报告结构 (20-30分钟)
```
1. 开场 (2分钟)
   - 自我介绍
   - 演讲主题
   - 合作者致谢

2. 背景与动机 (5分钟)
   - 研究问题是什么？
   - 为什么重要？
   - 已有工作

3. 主要结果 (10分钟)
   - 核心定理/发现
   - 证明思路（略去细节）
   - 关键技巧

4. 应用/例子 (5分钟)
   - 具体实例
   - 数值结果

5. 总结与展望 (3分钟)
   - 主要贡献
   - 未解决问题
   - 未来方向

6. 问答 (5-10分钟)
```

### 4.2 幻灯片制作原则

#### 基本规范
| 要素 | 建议 |
|------|------|
| **页数** | 每分钟1-1.5页 |
| **字体大小** | 标题28-36pt，正文18-24pt |
| **行数** | 每页不超过6-8行 |
| **颜色** | 深色背景浅色字，或相反 |
| **公式** | 一页不超过2-3个复杂公式 |

#### Beamer模板推荐
```latex
\documentclass[aspectratio=169]{beamer}
\usetheme{Madrid}  % 经典主题
\usecolortheme{beaver}  % 颜色主题

\title{演讲标题}
\author{演讲者}
\institute{单位}
\date{\today}

\begin{document}
\begin{frame}
\titlepage
\end{frame}

\begin{frame}{大纲}
\tableofcontents
\end{frame}

\section{引言}
\begin{frame}{研究背景}
...
\end{frame}

\end{document}
```

### 4.3 演讲技巧

#### 开场技巧
- **吸引注意**：提出有趣问题
- **建立联系**：与听众已知知识联系
- **明确目标**：说明演讲目的

#### 内容呈现
- **由简到难**：循序渐进
- **可视化**：多用图表
- **强调重点**：重复关键信息
- **语速控制**：重要内容放慢

#### 问答环节
- **倾听完整**：不打断提问
- **确认理解**：复述问题
- **诚实回答**：不知道就承认
- **简明扼要**：避免过度展开

## 五、海报制作指南

| 类型 | 尺寸 | 适用场景 |

| A0纵向 | 841×1189mm | 国际会议标准 |
| A0横向 | 1189×841mm | 较宽内容 |

#### 推荐布局
```
+------------------+------------------+
|      标题区      |      标题区      |
+------------------+------------------+
|  作者/单位信息   |  作者/单位信息   |
+--------+---------+---------+--------+
| 引言   | 方法   | 结果   | 结论   |
|        |        |        |        |
|        |        |        |        |
+--------+--------+--------+--------+
|              参考文献/致谢         |
+-----------------------------------+
```

### 5.2 海报设计原则

#### 视觉层次
1. **标题**：最大字体，3-5米外可读
2. **节标题**：次大，清晰分隔
3. **正文**：最小24pt
4. **图表**：占总面积的40-50%

#### LaTeX海报模板
```latex
\documentclass[a0paper,portrait]{baposter}

\usepackage{amsmath,amssymb}
\usepackage{graphicx}

\begin{document}
\begin{poster}
{
columns=4,
headerheight=0.12\textheight,
}
{\bf\LARGE 海报标题}
{作者1, 作者2\\单位名称}

\headerbox{引言}{name=intro,column=0,row=0}{
引言内容...
}

\headerbox{方法}{name=method,column=1}{
方法内容...
}

\headerbox{结果}{name=results,column=2}{
结果内容...
}

\headerbox{结论}{name=conclusion,column=3}{
结论内容...
}

\end{poster}
\end{document}
```

### 5.3 海报展示技巧

#### 准备阶段
- **打印测试**：提前打印检查
- **准备讲稿**：30秒、2分钟、5分钟版本
- **备用材料**：小论文摘要

#### 展示期间
- **主动招呼**：邀请观众
- **灵活讲解**：根据背景调整深度
- **记录反馈**：准备笔记本

## 七、常见错误与建议

### 7.1 写作常见错误

#### 语言错误
| 错误 | 正确 | 说明 |
|------|------|------|
| We prove that X is Y. | We prove that X is Y. | 陈述定理用现在时 |
| We proved... | We prove... | 论文中用现在时 |
| This theorem is very important. | This theorem plays a crucial role in... | 具体说明重要性 |
| It is easy to see... | 直接给出论证 | 避免"显然" |

#### 逻辑错误
- **循环论证**：用结论证明结论
- **偷换概念**：中途改变术语含义
- **过度泛化**：特例推广到一般
- **忽略条件**：未验证定理条件

### 7.2 演讲常见错误

| 错误 | 改进建议 |
|------|----------|
| 幻灯片文字过多 | 每页只留关键词 |
| 读幻灯片 | 讲解时面向听众 |
| 时间失控 | 提前演练计时 |
| 忽视听众 | 观察反馈调整 |
| 无眼神交流 | 与不同区域观众交流 |
