## 1. 分析学核心公式

### 1.1 极限与连续

| 公式名称 | 表达式 | 适用条件 |
|---------|-------|---------|
| **重要极限1** | lim(x→0) sin(x)/x = 1 | 0/0型 |
| **重要极限2** | lim(x→∞) (1+1/x)^x = e | 1^∞型 |
| **指数极限** | lim(x→0) (a^x-1)/x = ln(a) | a > 0 |
| **对数极限** | lim(x→0) ln(1+x)/x = 1 | 等价无穷小 |
| **幂指极限** | lim(x→0) (1+x)^(1/x) = e | 等价形式 |

### 1.2 导数公式

#### 基本导数表

| 函数 f(x) | 导数 f'(x) | 备注 |
|-----------|------------|-----|
| x^n | n*x^(n-1) | n ∈ R |
| e^x | e^x | 指数函数 |
| a^x | a^x * ln(a) | a > 0, a ≠ 1 |
| ln(x) | 1/x | x > 0 |
| log_a(x) | 1/(x*ln(a)) | 换底公式推导 |
| sin(x) | cos(x) | 三角函数 |
| cos(x) | -sin(x) | 注意符号 |
| tan(x) | sec^2(x) | cos(x) ≠ 0 |
| arcsin(x) | 1/√(1-x²) | \|x\| < 1 |
| arccos(x) | -1/√(1-x²) | \|x\| < 1 |
| arctan(x) | 1/(1+x²) | 全定义域 |

#### 求导法则

**四则运算：**
- 线性: (u ± v)' = u' ± v'
- 乘法: (uv)' = u'v + uv'
- 除法: (u/v)' = (u'v - uv')/v²

**链式法则：** (f(g(x)))' = f'(g(x)) · g'(x)

**高阶导数莱布尼茨公式：**
$$ (uv)^{(n)} = \sum_{k=0}^{n} C_n^k u^{(n-k)} v^{(k)} $$

### 1.3 中值定理

| 定理 | 条件 | 结论 | 几何意义 |
|-----|-----|-----|---------|
| **Rolle** | f在[a,b]连续，(a,b)可导，f(a)=f(b) | 存在ξ∈(a,b), f'(ξ)=0 | 水平切线 |
| **Lagrange** | f在[a,b]连续，(a,b)可导 | f(b)-f(a)=f'(ξ)(b-a) | 割线=切线 |
| **Cauchy** | f,g满足Lagrange条件，g'≠0 | [f(b)-f(a)]/[g(b)-g(a)]=f'(ξ)/g'(ξ) | 参数曲线 |

### 1.4 Taylor展开

#### 常见函数的Maclaurin展开

| 函数 | 展开式 | 收敛域 |
|-----|-------|-------|
| e^x | Σ(n=0→∞) x^n/n! | (-∞, +∞) |
| sin(x) | Σ(n=0→∞) (-1)^n x^(2n+1)/(2n+1)! | (-∞, +∞) |
| cos(x) | Σ(n=0→∞) (-1)^n x^(2n)/(2n)! | (-∞, +∞) |
| ln(1+x) | Σ(n=1→∞) (-1)^(n-1) x^n/n | (-1, 1] |
| (1+x)^α | Σ(n=0→∞) C(α,n) x^n | \|x\| < 1 |
| arctan(x) | Σ(n=0→∞) (-1)^n x^(2n+1)/(2n+1) | [-1, 1] |

### 1.5 积分公式

#### 基本积分表

| 被积函数 | 原函数 | 备注 |
|---------|-------|-----|
| x^n (n≠-1) | x^(n+1)/(n+1) + C | 幂函数积分 |
| 1/x | ln\|x\| + C | 特殊情况 |
| e^x | e^x + C | 指数函数 |
| a^x | a^x/ln(a) + C | a>0, a≠1 |
| sin(x) | -cos(x) + C | 三角函数 |
| cos(x) | sin(x) + C | - |
| tan(x) | -ln\|cos(x)\| + C | - |
| sec²(x) | tan(x) + C | - |
| 1/√(1-x²) | arcsin(x) + C | 反三角 |
| 1/(1+x²) | arctan(x) + C | - |
| 1/√(a²-x²) | arcsin(x/a) + C | a>0 |
| 1/(a²+x²) | (1/a)arctan(x/a) + C | a≠0 |
| 1/(x²-a²) | (1/2a)ln\|(x-a)/(x+a)\| + C | 分式分解 |
| 1/√(x²±a²) | ln\|x+√(x²±a²)\| + C | 双曲换元 |
| sec(x) | ln\|sec(x)+tan(x)\| + C | 常用公式 |
| csc(x) | -ln\|csc(x)+cot(x)\| + C | 配对记忆 |

#### 分部积分公式
$$ \int u \, dv = uv - \int v \, du $$

**LIATE选择优先级：** 对数(Log) < 反三角(Inv) < 代数(Alg) < 三角(Tri) < 指数(Exp)

### 1.6 多元微积分

#### 偏导数与全微分

| 概念 | 公式 | 说明 |
|-----|-----|-----|
| **全微分** | df = (∂f/∂x)dx + (∂f/∂y)dy | 二元函数 |
| **链式法则** | ∂z/∂t = (∂z/∂x)(∂x/∂t) + (∂z/∂y)(∂y/∂t) | 复合函数 |
| **隐函数** | dy/dx = -F_x/F_y | F(x,y)=0 |
| **梯度** | ∇f = (∂f/∂x, ∂f/∂y, ∂f/∂z) | 最速上升方向 |
| **方向导数** | ∂f/∂l = ∇f · l̂ | 单位方向向量 |

#### 积分变换公式

| 变换 | 公式 | Jacobian |
|-----|-----|---------|
| **极坐标** | x = r·cos(θ), y = r·sin(θ) | r |
| **柱坐标** | x = r·cos(θ), y = r·sin(θ), z = z | r |
| **球坐标** | x = r·sin(φ)cos(θ), y = r·sin(φ)sin(θ), z = r·cos(φ) | r²·sin(φ) |

#### 三大公式

**Green公式：**
$$ \oint_L P\,dx + Q\,dy = \iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) dx\,dy $$

**Gauss公式（散度定理）：**
$$ \oiint_\Sigma \vec{F} \cdot d\vec{S} = \iiint_\Omega \nabla \cdot \vec{F} \, dV $$

**Stokes公式：**
$$ \oint_L \vec{F} \cdot d\vec{r} = \iint_\Sigma (\nabla \times \vec{F}) \cdot d\vec{S} $$

## 2. 代数学核心公式

### 2.1 多项式与方程

| 公式/定理 | 表达式 | 说明 |
|----------|-------|-----|
| **Vieta定理** | Σx_i = -a_{n-1}/a_n, Πx_i = (-1)^n a_0/a_n | 根与系数关系 |
| **余数定理** | f(a) = r, f(x) = (x-a)q(x) + r | 除法算法 |
| **二项式定理** | (a+b)^n = Σ(k=0→n) C(n,k) a^(n-k) b^k | 展开公式 |
| **判别式** | Δ = b² - 4ac | 二次方程根的性质 |

### 2.2 矩阵与行列式

#### 基本运算

| 运算 | 公式 | 备注 |
|-----|-----|-----|
| **转置** | (A^T)_ij = A_ji | (AB)^T = B^T A^T |
| **逆矩阵** | A^(-1) = (1/det A) A* | det A ≠ 0 |
| **伴随矩阵** | A* = (A_ji), A_ij是代数余子式 | AA* = A*A = det(A)·I |
| **行列式** | det A = Σ_σ sgn(σ) Π_i a_{i,σ(i)} | 排列展开 |

#### 行列式性质
- 转置不变: det(A^T) = det(A)
- 行交换变号
- 行倍加不变
- 行倍乘提取: det(kA) = k^n det(A)
- 乘法公式: det(AB) = det(A)·det(B)
- Laplace展开

#### 特征值与特征向量

| 概念 | 公式 | 性质 |
|-----|-----|-----|
| **特征方程** | det(A - λI) = 0 | n次多项式 |
| **特征值和** | Σλ_i = tr(A) | 迹 |
| **特征值积** | Πλ_i = det(A) | 行列式 |
| **Cayley-Hamilton** | p(A) = 0 | 矩阵满足自身特征方程 |

### 2.3 线性空间

| 公式/定理 | 表达式 | 说明 |
|----------|-------|-----|
| **维数公式** | dim(U+V) = dim U + dim V - dim(U∩V) | 子空间和 |
| **秩-零化度** | dim ker A + dim im A = n | 对 m×n 矩阵 |
| **基变换** | [v]_B = P_{B←C} [v]_C | 过渡矩阵 |
| **坐标变换** | A' = P^(-1)AP | 相似变换 |

### 2.4 内积空间

| 公式 | 表达式 | 几何意义 |
|-----|-------|---------|
| **内积** | <u,v> = u^T v̄ | 标准内积 |
| **范数** | \|v\| = √<v,v> | 长度 |
| **Cauchy-Schwarz** | \|<u,v>\| ≤ \|u\|·\|v\| | 内积有界 |
| **正交投影** | proj_u(v) = (<v,u>/<u,u>) u | 投影到u方向 |
| **Gram-Schmidt** | e_k = v_k - Σ(j<k)<v_k,e_j>e_j, 再归一化 | 正交化 |

### 2.5 二次型

| 公式 | 表达式 | 说明 |
|-----|-------|-----|
| **矩阵表示** | Q(x) = x^T A x | A对称 |
| **标准形** | Q = y_1² + ... + y_p² - y_{p+1}² - ... - y_r² | 惯性定理 |
| **正惯性指数** | p | 正特征值个数 |
| **符号差** | p - (r-p) = 2p - r | 分类不变量 |
| **正定性判定** | 顺序主子式 > 0 | 正定的充要条件 |

## 3. 几何学核心公式

### 3.1 解析几何

#### 平面几何

| 概念 | 公式 | 说明 |
|-----|-----|-----|
| **两点距离** | d = √[(x₂-x₁)² + (y₂-y₁)²] | 欧氏距离 |
| **直线方程** | Ax + By + C = 0 | 一般式 |
| **点线距离** | d = \|Ax₀ + By₀ + C\|/√(A²+B²) | 点到直线 |
| **夹角公式** | tan(θ) = \|(k₂-k₁)/(1+k₁k₂)\| | 两直线夹角 |

#### 空间几何

| 概念 | 公式 | 说明 |
|-----|-----|-----|
| **两点距离** | d = √[(x₂-x₁)² + (y₂-y₁)² + (z₂-z₁)²] | 三维距离 |
| **平面方程** | Ax + By + Cz + D = 0 | 法向量(A,B,C) |
| **点面距离** | d = \|Ax₀ + By₀ + Cz₀ + D\|/√(A²+B²+C²) | 点到平面 |
| **直线对称式** | (x-x₀)/m = (y-y₀)/n = (z-z₀)/p | 方向向量(m,n,p) |

### 3.2 曲线与曲面

#### 常见曲线

| 曲线 | 方程 | 参数 |
|-----|-----|-----|
| **圆** | (x-a)² + (y-b)² = R² | 圆心(a,b), 半径R |
| **椭圆** | x²/a² + y²/b² = 1 | 半轴a,b |
| **双曲线** | x²/a² - y²/b² = 1 | 渐近线y=±(b/a)x |
| **抛物线** | y² = 2px 或 x² = 2py | 焦点参数p |

#### 常见曲面

| 曲面 | 方程 | 特征 |
|-----|-----|-----|
| **球面** | (x-a)² + (y-b)² + (z-c)² = R² | 球心(a,b,c) |
| **椭球面** | x²/a² + y²/b² + z²/c² = 1 | 半轴a,b,c |
| **双曲面(单叶)** | x²/a² + y²/b² - z²/c² = 1 | 直母线 |
| **双曲面(双叶)** | x²/a² - y²/b² - z²/c² = 1 | 两部分 |
| **锥面** | x²/a² + y²/b² - z²/c² = 0 | 顶点在原点 |
| **抛物面(椭圆)** | x²/a² + y²/b² = 2z | 开口向上 |
| **抛物面(双曲)** | x²/a² - y²/b² = 2z | 马鞍形 |
| **柱面** | f(x,y) = 0 | 母线平行z轴 |

### 3.3 曲线论（微分几何）

| 概念 | 公式 | 说明 |
|-----|-----|-----|
| **弧长参数** | s(t) = ∫(t₀→t) \|r'(τ)\| dτ | 自然参数 |
| **切向量** | T = dr/ds | 单位切向量 |
| **曲率** | κ = \|dT/ds\| = \|r'×r''\|/\|r'\|³ | 弯曲程度 |
| **曲率半径** | ρ = 1/κ | - |
| **挠率** | τ = (r',r'',r''')/\|r'×r''\|² | 偏离平面程度 |
| **Frenet公式** | dT/ds = κN, dN/ds = -κT + τB, dB/ds = -τN | 标架运动 |

### 3.4 曲面论

| 概念 | 公式 | 说明 |
|-----|-----|-----|
| **第一基本形式** | I = E du² + 2F dudv + G dv² | 内蕴几何 |
| **第二基本形式** | II = L du² + 2M dudv + N dv² | 外蕴几何 |
| **高斯曲率** | K = (LN-M²)/(EG-F²) | 内蕴量 |
| **平均曲率** | H = (EN-2FM+GL)/[2(EG-F²)] | - |
| **主曲率** | κ₁, κ₂ = H ± √(H²-K) | 极值曲率 |
| **测地线方程** | 满足变分原理 δ∫ds = 0 | 曲面上"直线" |

## 4. 概率统计公式

### 4.1 概率基础

| 概念 | 公式 | 说明 |
|-----|-----|-----|
| **加法公式** | P(A∪B) = P(A) + P(B) - P(A∩B) | 一般情况 |
| **条件概率** | P(A\|B) = P(A∩B)/P(B) | P(B) > 0 |
| **乘法公式** | P(A∩B) = P(A)P(B\|A) = P(B)P(A\|B) | - |
| **全概率公式** | P(A) = Σ P(B_i)P(A\|B_i) | 完备事件组 |
| **Bayes公式** | P(B_i\|A) = P(B_i)P(A\|B_i)/Σ P(B_j)P(A\|B_j) | 逆概率 |
| **独立性** | P(A∩B) = P(A)P(B) | A,B独立 |

### 4.2 随机变量

#### 数字特征

| 特征 | 离散型 | 连续型 | 性质 |
|-----|-------|-------|-----|
| **期望** | E[X] = Σ x_i p_i | E[X] = ∫ x f(x) dx | 线性性 |
| **方差** | Var(X) = E[(X-μ)²] | 同上 | Var(aX+b) = a²Var(X) |
| **标准差** | σ = √Var(X) | 同上 | - |
| **协方差** | Cov(X,Y) = E[(X-μ_x)(Y-μ_y)] | 同上 | Cov(X,X) = Var(X) |
| **相关系数** | ρ = Cov(X,Y)/(σ_x σ_y) | 同上 | \|ρ\| ≤ 1 |
| **切比雪夫不等式** | P(\|X-μ\| ≥ kσ) ≤ 1/k² | - | 通用界限 |

#### 常用分布

| 分布 | 概率质量/密度函数 | 期望 | 方差 |
|-----|-----------------|-----|-----|
| **Bernoulli(p)** | P(X=1)=p, P(X=0)=1-p | p | p(1-p) |
| **Binomial(n,p)** | C(n,k)p^k(1-p)^(n-k) | np | np(1-p) |
| **Poisson(λ)** | e^(-λ) λ^k/k! | λ | λ |
| **Uniform(a,b)** | 1/(b-a) | (a+b)/2 | (b-a)²/12 |
| **Exponential(λ)** | λe^(-λx) | 1/λ | 1/λ² |
| **Normal(μ,σ²)** | (1/√(2π)σ)e^[-(x-μ)²/(2σ²)] | μ | σ² |
| **Gamma(α,β)** | β^α x^(α-1)e^(-βx)/Γ(α) | α/β | α/β² |

### 4.3 大数定律与中心极限定理

| 定理 | 条件 | 结论 |
|-----|-----|-----|
| **弱大数定律** | X_i iid, E[X_i] = μ | (1/n)ΣX_i →P μ |
| **强大数定律** | X_i iid, E[\|X_i\|] < ∞ | (1/n)ΣX_i →a.s. μ |
| **中心极限定理** | X_i iid, E[X_i]=μ, Var(X_i)=σ² | (√n)(X̄_n-μ)/σ →d N(0,1) |

### 4.4 统计推断

#### 参数估计

| 方法 | 公式 | 说明 |
|-----|-----|-----|
| **样本均值** | X̄ = (1/n)ΣX_i | μ的无偏估计 |
| **样本方差** | S² = [1/(n-1)]Σ(X_i-X̄)² | σ²的无偏估计 |
| **极大似然** | θ̂ = argmax L(θ\|x) | 似然函数最大化 |
| **矩估计** | 样本矩 = 理论矩 | 解方程得估计 |

#### 假设检验

| 检验类型 | 检验统计量 | 分布 |
|---------|-----------|-----|
| **单样本Z检验** | Z = (X̄-μ₀)/(σ/√n) | N(0,1) |
| **单样本t检验** | t = (X̄-μ₀)/(S/√n) | t(n-1) |
| **双样本t检验** | t = (X̄₁-X̄₂)/S_p√(1/n₁+1/n₂) | t(n₁+n₂-2) |
| **卡方检验** | χ² = Σ(O_i-E_i)²/E_i | χ²(df) |
| **F检验** | F = S₁²/S₂² | F(n₁-1,n₂-1) |

#### 置信区间

| 参数 | 置信区间 | 条件 |
|-----|---------|-----|
| **μ (σ已知)** | X̄ ± z_(α/2)·σ/√n | 大样本或正态 |
| **μ (σ未知)** | X̄ ± t_(α/2,n-1)·S/√n | 正态总体 |
| **σ²** | [(n-1)S²/χ²_(α/2), (n-1)S²/χ²_(1-α/2)] | 正态总体 |
| **p (比例)** | p̂ ± z_(α/2)√[p̂(1-p̂)/n] | 大样本 |

## 5. 数值方法公式

### 5.1 方程求根

| 方法 | 迭代公式 | 收敛阶 | 适用条件 |
|-----|---------|-------|---------|
| **二分法** | [a,b]中点 | 线性 | 连续, f(a)f(b)<0 |
| **Newton法** | x_{n+1} = x_n - f(x_n)/f'(x_n) | 二次 | f'(x)≠0 |
| **割线法** | x_{n+1} = x_n - f(x_n)(x_n-x_{n-1})/[f(x_n)-f(x_{n-1})] | 1.618 | 避免求导 |
| **不动点迭代** | x_{n+1} = g(x_n) | 线性 | \|g'(x)\|<1 |

### 5.2 插值方法

| 方法 | 公式 | 误差估计 |
|-----|-----|---------|
| **Lagrange插值** | L_n(x) = Σ y_i l_i(x), l_i(x) = Π_{j≠i}(x-x_j)/(x_i-x_j) | f^(n+1)(ξ)/(n+1)! · ω(x) |
| **Newton插值** | N_n(x) = f[x_0] + Σ f[x_0,...,x_k]ω_k(x) | 同Lagrange |
| **Hermite插值** | 含导数值的插值 | - |
| **三次样条** | 分段三次多项式, 二阶导数连续 | O(h⁴) |

### 5.3 数值积分

| 方法 | 公式 | 代数精度 |
|-----|-----|---------|
| **梯形公式** | (b-a)[f(a)+f(b)]/2 | 1 |
| **Simpson公式** | (b-a)[f(a)+4f((a+b)/2)+f(b)]/6 | 3 |
| **复合梯形** | h[(f_0+f_n)/2 + Σf_i] | O(h²) |
| **复合Simpson** | (h/3)[f_0 + 4Σf_{2i-1} + 2Σf_{2i} + f_n] | O(h⁴) |
| **Gauss-Legendre** | Σ w_i f(x_i) | 2n+1 |

### 5.4 常微分方程数值解

| 方法 | 迭代公式 | 阶数 |
|-----|---------|-----|
| **Euler法** | y_{n+1} = y_n + hf(x_n, y_n) | 1 |
| **改进Euler** | 预测-校正 | 2 |
| **Runge-Kutta4** | y_{n+1} = y_n + (h/6)(K₁+2K₂+2K₃+K₄) | 4 |
| **Adams-Bashforth** | 多步法 | 多阶 |

### 5.5 线性方程组数值解

| 方法 | 公式/特点 | 适用性 |
|-----|----------|-------|
| **Gauss消元** | 化为上三角 | 一般稠密矩阵 |
| **LU分解** | A = LU | 多次右端项 |
| **Cholesky** | A = LL^T (A对称正定) | 对称正定 |
| **Jacobi迭代** | x_i^(k+1) = (b_i - Σ_{j≠i}a_{ij}x_j^(k))/a_{ii} | 对角占优 |
| **Gauss-Seidel** | 使用最新分量 | 通常比Jacobi快 |
| **SOR** | 带松弛因子 | 加速收敛 |

### 5.6 矩阵特征值

| 方法 | 特点 | 适用 |
|-----|-----|-----|
| **幂法** | 求模最大特征值 | 占优特征值 |
| **逆幂法** | 移位求特定特征值 | 接近给定值 |
| **QR迭代** | 收敛到上三角 | 全部特征值 |
| **Jacobi方法** | 平面旋转对角化 | 对称矩阵 |
