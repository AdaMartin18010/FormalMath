# FormalMath 快速开始指南

**最后更新**: 2026年4月5日  
**版本**: 2.0  
**目标读者**: 数学学习者、研究者、教育工作者

---

## 1. 项目简介

FormalMath是一个开源的数学知识库项目，旨在以结构化、可验证的方式呈现现代数学的完整体系。项目涵盖从基础数学到前沿研究的各个领域，采用标准化的文档格式，支持多语言，并与国际数学标准（MSC2020）对齐。

### 1.1 核心特色

- **三视角内容**: 每个概念提供形式化定义、直观理解和实战应用
- **国际标准对齐**: 与MIT、Harvard、Princeton等顶级课程对标
- **MSC编码标注**: 完整的数学主题分类编码
- **多语言支持**: 中英双语，支持扩展其他语言
- **知识图谱**: 概念间的依赖关系和前置要求

## 3. 如何开始学习

### 3.1 根据背景选择路径

#### 初学者（大学预科/大一）

**推荐起点**: `docs/01-基础数学/`

1. **阅读**: 00-README.md 了解基础数学全貌
2. **学习**: 集合论基础 → 数系构造 → 基本代数
3. **练习**: 完成实战问题文档中的习题
4. **进阶**: 进入 `docs/02-代数结构/` 或 `docs/03-分析学/`

#### 本科生（数学专业）

**推荐起点**: 根据兴趣选择核心分支

- **代数方向**: `docs/02-代数结构/` → 群论 → 环论 → 域论
- **分析方向**: `docs/03-分析学/` → 实分析 → 复分析 → 泛函分析
- **几何方向**: `docs/04-几何与拓扑/` → 微分几何 → 拓扑学
- **离散方向**: `docs/09-组合数学与离散数学/` → 组合学 → 图论

#### 研究生/研究者

**推荐起点**: `docs/11-高级数学/`

- **动力系统**: 遍历理论 → 混沌理论
- **代数几何**: 概形理论 → 层上同调
- **指标理论**: Atiyah-Singer定理及应用

### 3.2 文档阅读指南

每篇文档遵循统一格式：

1. **Frontmatter**: 标题、MSC分类、处理时间
2. **引言**: 概念概述和历史背景
3. **核心定义**: 严格的数学定义
4. **定理与证明**: 重要结果及其证明
5. **示例**: 具体例子帮助理解
6. **目录结构**: 相关文档索引
7. **学习路径**: 前置知识和后续发展

**提示**: 文档标题旁标注了质量等级（⭐数量），五星文档代表研究级内容。

## 4. 核心概念三问

每个数学概念都可以通过三个视角深入理解：

### 4.1 它是什么？（形式化）

- 严格定义是什么？
- 符号表示是什么？
- 属于哪个数学结构？

**示例**（群）:
> 群是集合 $G$ 配备运算 $\cdot: G \times G \to G$，满足结合律、单位元、逆元三公理。

### 4.2 它意味着什么？（直观化）

- 几何直观是什么？
- 直观例子有哪些？
- 为什么这个概念重要？

**示例**（群）:
> 群描述对称性。置换群描述物体排列的对称，矩阵群描述线性变换。

### 4.3 它有什么用？（实战化）

- 有什么应用场景？
- 相关的问题有哪些？
- 如何用于解决问题？

**示例**（群）:
> 群论用于晶体学分类、密码学算法、粒子物理标准模型。

## 5. 推荐学习顺序

### 5.1 标准路径（3-4年）

```
Year 1: 基础数学 → 线性代数 → 微积分
Year 2: 代数结构 → 分析学 → 概率统计
Year 3: 微分几何/拓扑学 → 数论/组合数学
Year 4: 高级数学专题 → 研究项目
```

### 5.2 快速路径（有基础者）

```
阶段1: 查漏补缺（评估现有知识）
阶段2: 核心分支深入学习（选择1-2个方向）
阶段3: 跨分支连接（发现数学统一性）
阶段4: 前沿专题研究
```

## 8. 常见问题

### Q: 如何找到特定主题？

A: 使用文档索引或全局搜索功能，也可按MSC编码查找。

### Q: 文档的难度如何判断？

A: 查看Frontmatter中的MSC编码和文档质量等级标注。

### Q: 是否有配套练习？

A: 多数分支包含"实战问题"子目录，提供习题和解答。

### Q: 如何报告错误？

A: 通过GitHub Issues提交，或使用文档中的反馈链接。
