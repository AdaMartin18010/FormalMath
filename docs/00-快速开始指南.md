# FormalMath 快速开始指南

**版本**: v1.0  
**适用对象**: 所有学习者  
**阅读时间**: 约10分钟

---

## 🎯 本指南目标

帮助您快速了解 FormalMath 项目，找到适合自己水平的学习路径，并高效使用项目资源。

---

## 📚 项目简介

FormalMath 是一个面向数学教育的综合性知识库，包含：

- **6,573+篇文档** - 涵盖从基础到前沿的数学内容
- **62位数学家** - 深度分析数学思想与贡献
- **20个Lean4定理** - 形式化证明验证
- **121+门国际课程** - 与MIT/Harvard等九校对齐

## 📖 学习路径推荐

### 路径A：基础数学 → 代数 → 分析 → 拓扑

适合希望建立完整数学基础的学习者。

```
第一阶段：基础构建 (1-2个月)
├── 01-集合论基础
├── 02-数系与运算
├── 03-函数与映射
└── 04-关系与等价

第二阶段：代数结构 (2-3个月)
├── 线性代数
├── 群论基础
├── 环论基础
└── 域论基础

第三阶段：分析学 (2-3个月)
├── 实分析
├── 复分析
└── 泛函分析

第四阶段：几何拓扑 (2-3个月)
├── 点集拓扑
├── 微分几何
└── 代数拓扑
```

### 路径B：按兴趣专题学习

适合有明确学习目标的学习者。

#### 代数几何方向
```
预备：交换代数 → 代数几何基础 → 概形理论
进阶：层上同调 → 导出范畴 → 格洛腾迪克理念
```

#### 数论方向
```
预备：初等数论 → 代数数论 → 解析数论
进阶：类域论 → 椭圆曲线 → 模形式
```

#### 逻辑与基础方向
```
预备：命题逻辑 → 谓词逻辑 → 模型论
进阶：集合论 → 证明论 → 可计算性理论
```

## ❓ 常见问题

### Q1: 我应该从哪里开始？

**A**: 根据您的水平选择：
- 高中/大一 → [集合论基础](docs/01-基础数学/集合论/01-集合论基础-国际标准版.md)
- 大二/大三 → [群论](docs/02-代数结构/群论/群论-深度扩展版.md) 或 [实分析](docs/03-分析学/01-实分析/实分析-深度扩展版.md)
- 研究生 → [格洛腾迪克理念](数学家理念体系/格洛腾迪克数学理念/README.md)

### Q2: 文档的版本有什么区别？

**A**: 
- 基础版：概念介绍，适合入门
- 增强版：包含定理和例子，适合深入学习
- 深度扩展版：完整证明和深度分析，适合研究

### Q3: 如何找到与我课程对应的内容？

**A**: 查看 [国际课程对照表](project/00-国际课程与机构对齐对照表-2026年4月.md)，找到您学校的课程对应项目内容。

### Q4: Lean4形式化证明怎么使用？

**A**: 
1. 安装Lean 4.29.0
2. 克隆项目代码
3. 进入 `docs/09-形式化证明/Lean4/`
4. 使用 `lake build` 编译验证

详细指南: [Lean4入门指南](docs/09-形式化证明/Lean4/00-README.md)

### Q5: 项目多久更新一次？

**A**: 
- 核心内容：按需更新，季度复核
- 前沿追踪：arXiv月度摘要
- 错误修正：实时处理
