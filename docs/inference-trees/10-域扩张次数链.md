# 域扩张次数链

## 核心概念

**域扩张**：若 $F \subseteq E$ 都是域，称 $E/F$ 为域扩张。

**扩张次数**：$[E:F] = \dim_F E$（作为 $F$-向量空间的维数）

---

## 推理树

```mermaid
graph TD
    A[域扩张 E/F] --> B[E是F-向量空间]
    B --> C[基的存在性<br/>{αᵢ} 基]
    C --> D[扩张次数<br/>[E:F] = dimF E]
    
    D --> E[有限扩张<br/>[E:F] < ∞]
    D --> F[无限扩张<br/>[E:F] = ∞]
    
    E --> G[塔法则<br/>F⊆K⊆E]
    G --> H[[E:F] = [E:K][K:F]]
    
    H --> I[有限扩张的传递<br/>E/F, K/E有限 ⇒ K/F有限]
    H --> J[素数次扩张<br/>无中间域]
    
    K[代数元α∈E] --> L[单扩张F(α)<br/>= F[α]]
    L --> M[极小多项式<br/>irr(α,F)]
    M --> N[[F(α):F] = deg irr(α,F)]
    
    N --> O[代数扩张<br/>所有元代数]
    O --> P[有限扩张<br/>⇒ 代数扩张]
    P --> Q[代数闭包<br/>存在性与唯一性]

```

## 塔法则（次数乘法公式）

**定理**：设 $F \subseteq K \subseteq E$ 是域扩张，则
$$[E:F] = [E:K] \cdot [K:F]$$

```mermaid
graph TD
    F[F] --> K[K] --> E[E]
    
    subgraph 塔结构
    E1[E] -.->|基{γⱼ}| K1[[E:K]]
    K1 -.->|基{βᵢ}| F1[[K:F]]

    end
    
    E1 -.->|积基{βᵢγⱼ}| F2[[E:F]=[E:K][K:F]]

```

**证明**：

1. 设 $\{\beta_i\}_{i=1}^m$ 是 $K/F$ 的基，$\{\gamma_j\}_{j=1}^n$ 是 $E/K$ 的基
2. **证明 $\{\beta_i \gamma_j\}$ 是 $E/F$ 的基**：
   - **生成**：$E \ni \alpha = \sum_j k_j \gamma_j = \sum_j (\sum_i f_{ij} \beta_i) \gamma_j = \sum_{i,j} f_{ij} (\beta_i \gamma_j)$
   - **线性无关**：设 $\sum_{i,j} f_{ij} \beta_i \gamma_j = 0$
     - 写成 $\sum_j (\sum_i f_{ij} \beta_i) \gamma_j = 0$
     - $\{\gamma_j\}$ 在 $K$ 上线性无关，故 $\sum_i f_{ij} \beta_i = 0$ 对所有 $j$
     - $\{\beta_i\}$ 在 $F$ 上线性无关，故 $f_{ij} = 0$ ∎

## 代数元与单扩张

### 代数元判定

**定义**：$\alpha \in E$ 称为 $F$-代数的，若存在非零 $f(x) \in F[x]$ 使 $f(\alpha) = 0$。

**等价条件**：

```mermaid
graph TD
    A[α代数/F] --> B[存在极小多项式<br/>irr(α,F)]
    B --> C[F(α) = F[α]]
    C --> D[[F(α):F] < ∞]
    D --> E[F(α) ≅ F[x]/(irr(α,F))]
    
    F[α超越/F] --> G[无极小多项式]
    G --> H[F(α) ≅ F(x)]
    H --> I[[F(α):F] = ∞]

```

### 单扩张结构定理

**定理**：
- 若 $\alpha$ 代数，$[F(\alpha):F] = \deg(\text{irr}(\alpha, F))$
- 若 $\alpha$ 超越，$[F(\alpha):F] = \infty$

## 代数扩张

```mermaid
graph TD
    A[E/F代数扩张] --> B[任意α∈E<br/>[F(α):F] < ∞]
    B --> C[E是F的<br/>代数闭包子域]
    
    D[有限扩张] --> E[[E:F] < ∞]
    E --> F[生成有限<br/>E = F(α₁,...,αₙ)]
    F --> A
    
    G[代数扩张的传递] --> H[E/F, K/E代数<br/>⇒ K/F代数]
    H --> I[代数闭包<br/>F̄/F代数]
    
    J[代数闭包性质] --> K[存在性<br/>Zorn引理]
    J --> L[唯一性<br/>同构延拓]
    
    style E fill:#f9f,stroke:#333,stroke-width:2px

```

### 代数扩张的塔法则

**命题**：$E/F$ 代数 $\Leftrightarrow$ 存在中间域塔 $F = F_0 \subseteq F_1 \subseteq \cdots \subseteq E$，其中每步都是有限代数扩张。

## 例子表

| 扩张 | 次数 | 极小多项式/基 | 说明 |
|-----|------|-------------|------|
| $\mathbb{C}/\mathbb{R}$ | 2 | $x^2 + 1$, $\{1, i\}$ | 经典例子 |
| $\mathbb{Q}(\sqrt{2})/\mathbb{Q}$ | 2 | $x^2 - 2$, $\{1, \sqrt{2}\}$ | 二次扩张 |
| $\mathbb{Q}(\sqrt[3]{2})/\mathbb{Q}$ | 3 | $x^3 - 2$, $\{1, \sqrt[3]{2}, \sqrt[3]{4}\}$ | 纯不可分 |
| $\mathbb{Q}(\zeta_p)/\mathbb{Q}$ | $p-1$ | 分圆多项式 $\Phi_p$ | Galois扩张 |
| $\mathbb{Q}(\sqrt{2}, \sqrt{3})/\mathbb{Q}$ | 4 | $\{1, \sqrt{2}, \sqrt{3}, \sqrt{6}\}$ | 双二次 |
| $\mathbb{R}/\mathbb{Q}$ | $\infty$ | 超越基 | 不可数维 |
