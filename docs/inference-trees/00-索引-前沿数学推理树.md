## 阅读指南

### 推荐学习路径

1. **同伦类型论入门**: 类型论基础 → 同伦层次 → 泛等公理
2. **导出代数几何**: 无穷范畴 → 导出函子 → 拟凝聚层
3. **Langlands纲领**: 类域论 → 自守形式 → 局部/整体对应
4. **镜像对称**: SYZ猜想 → 同源镜像对称
5. **椭圆上同调**: 形式群律 → 拓扑模形式

### 前置知识要求

| 主题 | 前置知识 |
|------|----------|
| Langlands纲领 | 代数数论、表示论、代数几何基础 |
| 同伦类型论 | 数理逻辑、类型论、基础拓扑学 |
| 导出代数几何 | 同调代数、代数几何、范畴论 |
| 镜像对称 | 复几何、辛几何、代数几何 |
| 椭圆上同调 | 代数拓扑、同伦论、代数几何 |
