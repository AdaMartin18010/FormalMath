## 形式群律详解

### 1. 定义

交换环R上的形式群律是幂级数：
```
F(x,y) ∈ R[[x,y]]
```

满足：
- **结合性**: F(F(x,y),z) = F(x,F(y,z))
- **交换性**: F(x,y) = F(y,x)
- **单位元**: F(x,0) = x, F(0,y) = y
- **逆元**: 存在ι(x)使得F(x,ι(x)) = 0

### 2. 例子

**加法形式群律**:
```
G_a(x,y) = x + y
```

**乘法形式群律**:
```
G_m(x,y) = x + y + xy = (1+x)(1+y) - 1
```

**欧拉形式群律**:
```
F_E(x,y) = (x√P(y) + y√P(x)) / (1 - εx²y²)
```

### 3. Lazard环

L是**通用**形式群律的系数环：
```
F_L(x,y) = x + y + Σ a_{ij} x^i y^j
```

**Lazard定理**: 
```
L ≅ ℤ[x₁, x₂, x₃, ...], |x_i| = 2i
```

## Quillen定理

**定理**: 
```
MU_* ≅ L  (Lazard环)
```

**证明思路**:
1. MU_*是形式群律的通用环
2. CP^∞上的形式群结构给出映射 L → MU_*
3. 计算验证这是同构

## 复配边与形式群

### 形式群结构

MU^*(CP^∞) ≅ MU^*[[x]]，其中x是第一Chern类。

投影公式CP^∞ × CP^∞ → CP^∞诱导形式群结构：
```
MU^*(CP^∞ × CP^∞) ≅ MU^*[[x,y]]

x ↦ F(x,y)
```

### 通用性质

任何复定向上同调理论E^*有形式群律，由分类映射：
```
MU^* → E^*
```

## 椭圆形式群律

### 椭圆曲线的形式群

设E是椭圆曲线，形式完备Ê在恒等元处有形式群结构。

**Weierstrass方程**:
```
y² + a₁xy + a₃y = x³ + a₂x² + a₄x + a₆
```

形式坐标t = -x/y给出形式群坐标。

### 高度

椭圆形式群律的典型高度为2：
- 在特征p，[p](t) = u·t^{p²} + ...

## 与椭圆上同调的关系

### Landweber正合函子定理

给定平坦形式群(E_*, F_E)，可构造上同调理论：
```
X ↦ MU_*(X) ⊗_{MU_*} E_*
```

### 椭圆上同调

椭圆上同调是复定向上同调，其形式群律来自椭圆曲线。

**Landweber-Ravenel-Stong构造**: 使用椭圆曲线的通用形变。
