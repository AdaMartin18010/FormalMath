## 测地线方程详解

### 1. 变分法导出
能量泛函：

```

E[γ] = ½ ∫_a^b g_ij(γ(t)) ẋ^i ẋ^j dt

```

Euler-Lagrange方程给出测地线方程：

```

ẍ^k + Γ^k_ij ẋ^i ẋ^j = 0

```

### 2. 自平行条件
测地线是切向量沿自身平行移动的曲线：

```

∇_γ' γ' = 0

```

展开后得到相同方程。

### 3. 局部最短性
测地线是弧长泛函的临界点（第一变分为零）。

## 几何解释

### 测地曲率
测地线的测地曲率 κ_g = 0，即曲线在曲面内"不弯曲"。

### 指数映射

```

Exp_p: T_p M → M,  v ↦ γ_v(1)

```

将切向量映射到沿测地线行走单位时间的点。

## 特殊空间的测地线

| 空间 | 测地线 |
|------|--------|
| ℝⁿ | 直线 |
| Sⁿ | 大圆 |
| Hⁿ (双曲) | 垂直于边界的圆/直线 |
| 平坦环面 | 直线（模周期） |

## Jacobi场

描述测地线族的无穷小展开：

```

J''(t) + R(γ', J)γ' = 0

```

- **共轭点**: Jacobi场的零点
- **几何意义**: 无穷小测地线族开始聚焦的点

## 重要定理

### Hopf-Rinow定理
对于黎曼流形，以下等价：
1. 作为度量空间完备
2. 测地线可无限延伸（测地完备）
3. 任意两点可由最短测地线连接
4. 有界闭集紧致

## 应用

1. **广义相对论**: 自由粒子沿时空测地线运动
2. **测地线流**: 切丛上的Hamilton系统
3. **比较几何**: Rauch比较、Toponogov定理
