本推理树展示导出函子的理论框架，包括Abel范畴的导出范畴、全导出函子、以及在有界性条件下的实现，这是同调代数在导出代数几何中的核心工具。

## 核心概念详解

### 1. 链复形

**链复形**: C• = {C_n, d_n}
```
... → C_{n+1} → C_n → C_{n-1} → ...
```
满足 d² = 0。

**同调**: H_n(C) = Ker(d_n) / Im(d_{n+1})

### 2. 同伦范畴 K(A)

**链同伦**: f ≃ g，如果存在h: f - g = dh + hd

**同伦等价**: 存在g使得 fg ≃ id, gf ≃ id

### 3. 导出范畴 D(A)

**拟同构**: f: C• → D•诱导同调同构H_n(f)

**导出范畴**: 
```
D(A) := K(A)[{拟同构}⁻¹]
```

**有界性**: 
- D⁺(A): 下有界
- D⁻(A): 上有界  
- D^b(A): 有界

## 导出函子构造

### 左导出函子

对于右正合函子F: A → B：

1. 取投射分解P• → X → 0
2. 应用F: FP•
3. 取同调: L_nF(X) = H_n(FP•)

### 右导出函子

对于左正合函子F: A → B：

1. 取内射分解0 → X → I•
2. 应用F: FI•
3. 取上同调: R^nF(X) = H^n(FI•)

## 经典导出函子

| 函子 | 类型 | 导出 | 经典记号 |
|------|------|------|----------|
| Hom(X,-) | 左正合 | 右导出 | Ext^n(X,-) |
| Hom(-,Y) | 左正合 | 右导出 | Ext^n(-,Y) |
| X ⊗ - | 右正合 | 左导出 | Tor_n(X,-) |
| Γ(X,-) | 左正合 | 右导出 | H^n(X,-) |

## Grothendieck谱序列

对于F: A → B, G: B → C，若F将内射映到G-零调：
```
E²_{p,q} = R^pG(R^qF(X)) ⇒ R^{p+q}(GF)(X)
```

## 无穷版本

在稳定无穷范畴中：
- 导出范畴是自然的无穷范畴结构
- 全导出函子是左/右Kan扩张
- 与经典导出函子在有界情形下一致
