# 行列式性质推导

## 核心定义

**行列式**：设 $A = (a_{ij})$ 是 $n \times n$ 矩阵，定义
$$\det(A) = \sum_{\sigma \in S_n} \text{sgn}(\sigma) \prod_{i=1}^n a_{i,\sigma(i)}$$

---

```mermaid
graph TD
    A[n×n矩阵A] --> B[置换展开<br/>Σ sgn(σ) a_{1σ1}...a_{nσn}]

    C --> D[多重线性<br/>det关于每行线性]
```

## 基本性质推导

### 1. 多重线性

**命题**：$\det$ 关于每行是线性的。

**证明**：对第一行（其他行类似）
- $\det(r_1 + r_1', r_2, \ldots) = \sum_\sigma \text{sgn}(\sigma)(a_{1\sigma(1)} + a'_{1\sigma(1)})a_{2\sigma(2)}\cdots$
- $= \sum_\sigma \text{sgn}(\sigma)a_{1\sigma(1)}a_{2\sigma(2)}\cdots + \sum_\sigma \text{sgn}(\sigma)a'_{1\sigma(1)}a_{2\sigma(2)}\cdots$
- $= \det(r_1, r_2, \ldots) + \det(r_1', r_2, \ldots)$ ∎

### 2. 反对称性

**命题**：交换两行，行列式变号。

**证明**：设交换行 $i$ 和 $j$，对应置换 $\tau = (i \ j)$
- $\det(A') = \sum_\sigma \text{sgn}(\sigma) a_{\tau(1)\sigma(1)} \cdots$
- 令 $\sigma' = \sigma \circ \tau$，则 $\text{sgn}(\sigma') = -\text{sgn}(\sigma)$
- $\det(A') = \sum_{\sigma'} (-\text{sgn}(\sigma')) a_{1\sigma'(1)} \cdots = -\det(A)$ ∎

### 3. 乘积公式

**定理**：$\det(AB) = \det(A)\det(B)$

**证明**：

```mermaid
graph TD
    A[固定B] --> B[φ_B: A ↦ det(AB)]
    B --> C[φ_B 多重线性<br/>反对称]
    C --> D[唯一性<br/>n重交错形式]
    D --> E[φ_B(A) = c·det(A)]
    E --> F[取A=I<br/>c = det(B)]
    F --> G[det(AB) = det(A)det(B)]
    
    style G fill:#f9f,stroke:#333,stroke-width:2px

```

```mermaid
graph TD
    A[计算det(A)] --> B[LU分解<br/>A = LU]
    B --> C[det(A) = det(U)<br/>对角元乘积]
    
    A --> D[行变换化为<br/>上三角]
    D --> E[记录行交换<br/>符号变化]
    E --> F[上三角行列式<br/>易算]
    
    A --> G[Laplace展开<br/>降阶]
    G --> H[2×2/3×3公式]
    
    I[特殊矩阵] --> J[对角矩阵<br/>∏ aᵢᵢ]
    I --> K[三角矩阵<br/>∏ aᵢᵢ]
    I --> L[正交矩阵<br/>det = ±1]
    I --> M[酉矩阵<br/>|det| = 1]

```

### Laplace展开

**按第 $i$ 行展开**：
$$\det(A) = \sum_{j=1}^n (-1)^{i+j} a_{ij} M_{ij}$$

其中 $M_{ij}$ 是删除第 $i$ 行第 $j$ 列的子式。

## 重要公式

| 公式 | 说明 |
|-----|------|
| $\det(A^T) = \det(A)$ | 转置不变 |
| $\det(A^{-1}) = \det(A)^{-1}$ | 逆矩阵 |
| $\det(cA) = c^n \det(A)$ | 数乘 |
| $\det(A^k) = \det(A)^k$ | 幂 |
| $\det(e^A) = e^{\text{tr}(A)}$ | 指数（Jacobi公式） |
| $\det(I + tA) = 1 + t \cdot \text{tr}(A) + O(t^2)$ | 微分 |
