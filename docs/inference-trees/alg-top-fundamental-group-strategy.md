## 概述

本推理树展示拓扑空间基本群 π₁(X, x₀) 的系统计算策略与常用方法。

## 计算策略详解

### 1. 可缩空间
- **判定**: 恒等映射同伦于常值映射
- **结论**: π₁(X) = 0（平凡群）
- **典型**: ℝⁿ, Dⁿ, 星形区域, 树

### 2. 圆周 S¹
- **基本群**: π₁(S¹) ≅ ℤ
- **生成元**: 绕圆周一圈的回路 [ω]
- **同构**: [ωⁿ] ↔ n ∈ ℤ（绕数）
- **证明**: 覆叠空间理论或复分析绕数

### 3. 球面 Sⁿ (n≥2)
- **基本群**: π₁(Sⁿ) = 0（单连通）
- **直观**: 回路可"收缩"到一点
- **推论**: 与 S¹ 有本质拓扑差异

### 4. 乘积空间
- **公式**: π₁(X × Y) ≅ π₁(X) × π₁(Y)
- **环面 T² = S¹ × S¹**: π₁(T²) ≅ ℤ × ℤ
- **高维环面**: π₁(Tⁿ) ≅ ℤⁿ

### 5. 贴空间（Seifert-van Kampen）
**定理**: X = U ∪ V，U,V,U∩V 道路连通
```
π₁(X) ≅ π₁(U) * π₁(V) / N
```
其中 N 是正规闭包，由 i*(γ)·j*(γ)⁻¹ 生成

**应用**:
- 贴 n-胞腔 (n≥3): 不改变基本群
- 贴 2-胞腔: 引入关系（贴入映射在基本群中为1）

### 6. 覆叠空间
- **单射**: p*: π₁(E) → π₁(X) 是单射
- **指数**: [π₁(X) : p*π₁(E)] = 覆叠叶数
- **万有覆叠**: E 单连通时，π₁(X) ≅ Aut(E)

## 常用公式速查

| 空间 | 基本群 | 计算方法 |
|------|--------|----------|
| ℝⁿ, Dⁿ | 0 | 可缩 |
| S¹ | ℤ | 覆叠空间 |
| Sⁿ (n≥2) | 0 | 单连通 |
| Tⁿ = (S¹)ⁿ | ℤⁿ | 乘积公式 |
| S¹ ∨ S¹ | F₂（自由群） | van Kampen |
| 亏格g曲面 | ⟨a₁,b₁,...,a₉,b₉ \| ∏[aᵢ,bᵢ]=1⟩ | 标准表示 |
| ℝPⁿ | ℤ/2ℤ (n≥2) | 覆叠 Sⁿ → ℝPⁿ |
| 透镜空间 L(p,q) | ℤ/pℤ | 覆叠 |

## 计算流程图

```
空间X
  ↓
是否可缩？ → 是 → π₁ = 0
  ↓ 否
是否S¹？ → 是 → π₁ ≅ ℤ
  ↓ 否
是否球面Sⁿ(n≥2)？ → 是 → π₁ = 0
  ↓ 否
是否乘积？ → 是 → 用乘积公式
  ↓ 否
能否分解为U∪V？ → 是 → van Kampen定理
  ↓ 否
有覆叠空间？ → 是 → 覆叠空间理论
  ↓ 否
用CW复形/胞腔逼近
```
