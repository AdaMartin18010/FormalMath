## 拟凝聚层详解

### 1. 定义

设(X, O_X)是环层空间。O_X-模F是拟凝聚的，如果对每点x∈X，存在邻域U使得：

```

F|_U ≅ M̃

```

其中M是O_X(U)-模，M̃是对应的层。

### 2. 仿射情形

对于仿射概形X = Spec A：

```

Qcoh(X) ≅ Mod(A)

```

通过F ↦ Γ(X,F) 和 M ↦ M̃。

### 3. 等价刻画

F是拟凝聚的 ⇔ 以下条件之一：
- 每个点有邻域U使得F|_U是模化层
- 对仿射开集Spec A ⊆ X，F|_Spec A ≅ Γ(F)^

## 导出拟凝聚层

### 导出范畴

```

D(Qcoh(X)): 拟凝聚层链复形的导出范畴

```

关键性质：
- 若X是拟紧分离概形，则D(Qcoh(X))是紧生成
- 完美复形是紧对象

### 完美复形

E ∈ D(Qcoh(X))是**完美**的，如果局部等价于有界复形，其项是有限投射的。

等价条件：
- E是D(Qcoh(X))中的紧对象
- E局部有有限Tor振幅

## 导出概形

### 定义

导出概形(X, O_X)其中：
- X是拓扑空间
- O_X是X上的层dg-代数
- (X, H^0(O_X))是经典概形

### 截断

```

t₀(X, O_X) = (X, H^0(O_X))

```

### 拟凝聚导出层

F是导出拟凝聚层，如果：
- F是O_X的dg-模层
- H^i(F)是H^0(O_X)的拟凝聚层

## 关键定理

| 定理 | 陈述 |
|------|------|
| Serre消失 | X仿射，F拟凝聚，则H^i(X,F)=0, i>0 |
| 紧生成 | X拟紧分离，D(Qcoh(X))由完美复形紧生成 |
| Grothendieck对偶 | 适当态射有右伴随f^! |
| 平坦下降 | 拟凝聚层满足平坦下降 |

导出代数几何使用E_∞-环谱代替交换环。

### 谱概形

(X, O_X)其中O_X取值于E_∞-环谱。

### 拟凝聚谱层

模谱层，满足拟凝聚条件。
