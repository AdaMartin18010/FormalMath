## 1. 引言

零知识证明(Zero-Knowledge Proof, ZKP)是密码学中最迷人的概念之一，允许证明者向验证者证明某个陈述为真，而无需透露任何额外信息。

### 1.1 历史里程碑

| 年份 | 突破 | 贡献者 |
|-----|------|--------|
| 1985 | 零知识证明概念提出 | Goldwasser, Micali, Rackoff |
| 1986 | 第一个ZKP构造 | Goldreich, Micali, Wigderson |

### 1.2 应用场景

- **区块链隐私**: Zcash (zk-SNARKs), Monero (环签名)
- **身份认证**: 证明身份属性而不泄露具体信息
- **计算外包验证**: 验证云计算结果的正确性
- **合规证明**: 证明满足监管要求而不泄露敏感数据
- **可验证投票**: 确保选举完整性同时保护隐私

## 2. 基础理论

### 2.1 零知识证明的定义

**定义 2.1** (交互式证明系统)：对于语言 $L$，一个**交互式证明系统**是证明者 $\mathcal{P}$ 和验证者 $\mathcal{V}$ 之间的协议，满足：

1. **完备性 (Completeness)**：若 $x \in L$，则 $\Pr[\langle \mathcal{P}, \mathcal{V} \rangle(x) = 1] \geq 1 - \epsilon$
2. **可靠性 (Soundness)**：若 $x \notin L$，则对任意 $\mathcal{P}^*$，$\Pr[\langle \mathcal{P}^*, \mathcal{V} \rangle(x) = 1] \leq \delta$

**定义 2.2** (零知识)：证明系统是**零知识**的，如果对于任意概率多项式时间验证者 $\mathcal{V}^*$，存在模拟器 $\mathcal{S}$ 使得：

$$\{\text{View}_{\mathcal{V}^*}[\langle \mathcal{P}, \mathcal{V}^* \rangle(x)]\}_{x \in L} \approx \{\mathcal{S}(x)\}_{x \in L}$$

即验证者从交互中学到的信息，可以由仅知道 $x$ 的模拟器生成。

**定义 2.3** (零知识强度层级)：

- **完美零知识**：分布完全相同
- **统计零知识**：统计距离可忽略
- **计算零知识**：计算不可区分

### 2.2 zk-SNARKs定义

**定义 2.4** (zk-SNARK)：一个**简洁非交互式零知识论证** (Succinct Non-interactive ARgument of Knowledge) 包含三个算法：

1. **Setup**($1^\lambda$) → $(pk, vk)$：生成证明密钥和验证密钥
2. **Prove**($pk, x, w$) → $\pi$：生成证明 $\pi$，其中 $x$ 是公开输入，$w$ 是见证
3. **Verify**($vk, x, \pi$) → $\{0, 1\}$：验证证明

**性质要求**：

- **简洁性**: $|\pi| = O(\text{polylog}(|C|))$，验证时间 $O(|x| + \text{polylog}(|C|))$
- **完备性**: 诚实验证者接受正确证明
- **可靠性**: 虚假陈述的证明无法通过验证
- **零知识**: 证明不泄露见证信息
- **知识论证**: 证明者"知道"见证（非仅存在性）

### 2.3 核心数学工具

#### 2.3.1 二次算术程序 (QAP)

**定义 2.5** (算术电路)：由加法门和乘法门组成的布尔电路推广，计算有限域 $\mathbb{F}$ 上的多项式函数。

**定义 2.6** (R1CS - Rank-1 Constraint System)：约束系统形式为：

$$(A \cdot w) \circ (B \cdot w) = C \cdot w$$

其中 $w$ 是见证向量，$A, B, C$ 是约束矩阵，$\circ$ 表示Hadamard积。

**定义 2.7** (QAP - Quadratic Arithmetic Program)：给定R1CS，QAP构造多项式 $A_i(x), B_i(x), C_i(x)$ 和目标点集 $\{r_1, \ldots, r_m\}$ 使得：

$$w \text{ 满足R1CS} \iff \exists H(x): \left(\sum w_i A_i(x)\right)\left(\sum w_i B_i(x)\right) - \sum w_i C_i(x) = H(x) \cdot Z(x)$$

其中 $Z(x) = \prod (x - r_i)$ 是目标多项式。

**核心思想**：将电路可满足性转化为多项式整除关系验证。

#### 2.3.2 多项式承诺方案

**定义 2.8** (多项式承诺)：允许证明者承诺一个多项式 $P(x)$，之后证明 $P(z) = y$。

**KZG承诺** (Kate-Zaverucha-Goldberg)：

- 设置: 可信设置生成 $\{g^{\tau^i}\}_{i=0}^d$，其中 $\tau$ 是陷门
- 承诺: $C = g^{P(\tau)} = g^{\sum p_i \tau^i} = \prod (g^{\tau^i})^{p_i}$
- 打开证明: 利用多项式除法，$P(x) - y = (x - z) \cdot Q(x)$

## 3. 构造方法详解

### 3.1 从程序到QAP

**转换流程**：

```
程序/电路 → R1CS → QAP → 多项式 → 承诺证明
```

**示例**：证明知道 $x$ 使得 $x^3 + x + 5 = 35$ (即 $x = 3$)

**步骤1**: 展开为约束

```
v1 = x * x       (x^2)
v2 = v1 * x      (x^3)
v3 = v2 + x      (x^3 + x)
out = v3 + 5     (x^3 + x + 5)
out = 35
```

**步骤2**: 转换为R1CS
每个约束 $\vec{a} \cdot \vec{w} \times \vec{b} \cdot \vec{w} = \vec{c} \cdot \vec{w}$

见证向量: $w = [1, out, x, v1, v2, v3]$

**步骤3**: 构造QAP
为每个约束选择根 $r_1, r_2, r_3, r_4$，构造Lagrange插值多项式。

### 3.2 Groth16协议

**协议流程**：

**Setup**:

1. 生成随机 $\alpha, \beta, \gamma, \delta, \tau \in \mathbb{F}_r$
2. 计算CRS (Common Reference String):
   - $\sigma_1 = (g^\alpha, g^\beta, g^\delta, \{g^{\tau^i}\}_{i=0}^{n-1}, \{g^{\beta u_i(\tau) + \alpha v_i(\tau) + w_i(\tau)}\}_{i=0}^l)$
   - $\sigma_2 = (h^\beta, h^\gamma, h^\delta, \{h^{\tau^i}\}_{i=0}^{n-1})$

**Prove**$(\sigma, w = (a_0, \ldots, a_m))$:

1. 计算 $A = g^{\alpha + \sum_{i=0}^m a_i u_i(\tau) + r\delta}$
2. 计算 $B = h^{\beta + \sum_{i=0}^m a_i v_i(\tau) + s\delta}$
3. 计算 $C = g^{(\sum_{i=l+1}^m a_i (\beta u_i(\tau) + \alpha v_i(\tau) + w_i(\tau)) + h(\tau)t(\tau))/\delta} \cdot A^s \cdot B^{-r} \cdot g^{-rs\delta}$

输出证明 $\pi = (A, B, C)$ (3个群元素!)

**Verify**$(vk, x = (a_0, \ldots, a_l), \pi = (A, B, C))$:
验证配对等式:
$$e(A, B) = e(g^\alpha, h^\beta) \cdot e(\prod_{i=0}^l g^{a_i(\beta u_i(\tau) + \alpha v_i(\tau) + w_i(\tau))/\gamma}, h^\gamma) \cdot e(C, h^\delta)$$

**正确性**：
展开左边：$e(A, B) = e(g, h)^{(\alpha + U + r\delta)(\beta + V + s\delta)}$

右边：$e(g, h)^{\alpha\beta + UV_1 + W_2}$

其中 $U = \sum a_i u_i(\tau)$，$V = \sum a_i v_i(\tau)$，$W = \sum a_i w_i(\tau)$

等式成立当且仅当 $UV - W$ 被 $t(\tau)$ 整除，即原始约束满足。

### 3.3 PLONK协议

PLONK相比Groth16的优势：

- **通用可信设置**: 不依赖特定电路
- **可更新设置**: 多方逐步增强安全性
- **更灵活的约束**: 支持自定义门

**核心思想**：

1. 使用置换论证(permutation argument)处理电路连线
2. 多项式承诺替代QAP的复杂结构
3. 门约束和连线约束分离验证

```python
"""
零知识证明基础 - Python实现
包含QAP构造、R1CS约束、简化的证明系统演示

参考: GGPR13, Groth16, Vitalik Buterin的教程
"""

import hashlib
import random
from typing import List, Tuple, Optional, Dict
from dataclasses import dataclass
from functools import reduce

# ============================================================================
# 第1部分：有限域和多项式运算
# ============================================================================

class FieldElement:
    """有限域 F_p 上的元素"""

    def __init__(self, value: int, prime: int):
        self.prime = prime
        self.value = value % prime

    def __add__(self, other: 'FieldElement') -> 'FieldElement':
        return FieldElement((self.value + other.value) % self.prime, self.prime)

    def __sub__(self, other: 'FieldElement') -> 'FieldElement':
        return FieldElement((self.value - other.value) % self.prime, self.prime)

    def __mul__(self, other) -> 'FieldElement':
        if isinstance(other, FieldElement):
            return FieldElement((self.value * other.value) % self.prime, self.prime)
        return FieldElement((self.value * other) % self.prime, self.prime)

    def __pow__(self, exp: int) -> 'FieldElement':
        return FieldElement(pow(self.value, exp, self.prime), self.prime)

    def __truediv__(self, other: 'FieldElement') -> 'FieldElement':
        return self * other.inverse()

    def inverse(self) -> 'FieldElement':
        """费马小定理求逆元"""
        return self ** (self.prime - 2)

    def __neg__(self) -> 'FieldElement':
        return FieldElement((-self.value) % self.prime, self.prime)

    def __eq__(self, other) -> bool:
        return isinstance(other, FieldElement) and self.value == other.value

    def __repr__(self):
        return f"F{self.value}"


class Polynomial:
    """有限域上的多项式"""

    def __init__(self, coefficients: List[FieldElement]):
        """
        coefficients: [a_0, a_1, ..., a_n] 表示 a_0 + a_1*x + ... + a_n*x^n
        """
        self.coeffs = coefficients
        self._trim()

    def _trim(self):
        """移除高次零系数"""
        while len(self.coeffs) > 1 and self.coeffs[-1].value == 0:
            self.coeffs.pop()

    def degree(self) -> int:
        return len(self.coeffs) - 1

    def __add__(self, other: 'Polynomial') -> 'Polynomial':
        max_len = max(len(self.coeffs), len(other.coeffs))
        result = []
        for i in range(max_len):
            a = self.coeffs[i] if i < len(self.coeffs) else FieldElement(0, self.coeffs[0].prime)
            b = other.coeffs[i] if i < len(other.coeffs) else FieldElement(0, other.coeffs[0].prime)
            result.append(a + b)
        return Polynomial(result)

    def __sub__(self, other: 'Polynomial') -> 'Polynomial':
        max_len = max(len(self.coeffs), len(other.coeffs))
        result = []
        for i in range(max_len):
            a = self.coeffs[i] if i < len(self.coeffs) else FieldElement(0, self.coeffs[0].prime)
            b = other.coeffs[i] if i < len(other.coeffs) else FieldElement(0, other.coeffs[0].prime)
            result.append(a - b)
        return Polynomial(result)

    def __mul__(self, other) -> 'Polynomial':
        if isinstance(other, FieldElement):
            return Polynomial([c * other for c in self.coeffs])

        # 多项式乘法
        result_len = len(self.coeffs) + len(other.coeffs) - 1
        result = [FieldElement(0, self.coeffs[0].prime) for _ in range(result_len)]
        for i, a in enumerate(self.coeffs):
            for j, b in enumerate(other.coeffs):
                result[i + j] = result[i + j] + a * b
        return Polynomial(result)

    def __rmul__(self, scalar: FieldElement) -> 'Polynomial':
        return self.__mul__(scalar)

    def __call__(self, x: FieldElement) -> FieldElement:
        """多项式求值"""
        result = FieldElement(0, self.coeffs[0].prime)
        power = FieldElement(1, self.coeffs[0].prime)
        for coeff in self.coeffs:
            result = result + coeff * power
            power = power * x
        return result

    def __mod__(self, other: 'Polynomial') -> 'Polynomial':
        """多项式取模 (简化版)"""
        if other.degree() == 0:
            return Polynomial([FieldElement(0, self.coeffs[0].prime)])

        remainder = self
        divisor_lc = other.coeffs[-1]

        while remainder.degree() >= other.degree():
            factor_degree = remainder.degree() - other.degree()
            factor = remainder.coeffs[-1] / divisor_lc

            # 构造 factor * x^factor_degree * other
            shift_coeffs = [FieldElement(0, self.coeffs[0].prime)] * factor_degree + list(other.coeffs)
            subtrahend = Polynomial([c * factor for c in shift_coeffs])

            # 扩展remainder到相同长度
            new_coeffs = list(remainder.coeffs)
            while len(new_coeffs) < len(subtrahend.coeffs):
                new_coeffs.append(FieldElement(0, self.coeffs[0].prime))

            result_coeffs = []
            for i in range(max(len(new_coeffs), len(subtrahend.coeffs))):
                a = new_coeffs[i] if i < len(new_coeffs) else FieldElement(0, self.coeffs[0].prime)
                b = subtrahend.coeffs[i] if i < len(subtrahend.coeffs) else FieldElement(0, self.coeffs[0].prime)
                result_coeffs.append(a - b)

            remainder = Polynomial(result_coeffs)

        return remainder

    def __eq__(self, other) -> bool:
        return isinstance(other, Polynomial) and self.coeffs == other.coeffs

    def __repr__(self):
        terms = []
        for i, c in enumerate(self.coeffs):
            if c.value != 0:
                if i == 0:
                    terms.append(str(c.value))
                elif i == 1:
                    terms.append(f"{c.value}x")
                else:
                    terms.append(f"{c.value}x^{i}")
        return " + ".join(terms) if terms else "0"


def interpolate(points: List[Tuple[FieldElement, FieldElement]]) -> Polynomial:
    """
    Lagrange插值
    给定点 (x_i, y_i)，构造通过所有点的多项式
    """
    if not points:
        return Polynomial([FieldElement(0, 2)])

    p = points[0][0].prime
    result = Polynomial([FieldElement(0, p)])

    for i, (x_i, y_i) in enumerate(points):
        # 计算基多项式 L_i(x)
        numerator = Polynomial([FieldElement(1, p)])
        denominator = FieldElement(1, p)

        for j, (x_j, _) in enumerate(points):
            if i != j:
                # (x - x_j)
                term = Polynomial([FieldElement(-x_j.value, p), FieldElement(1, p)])
                numerator = numerator * term
                denominator = denominator * (x_i - x_j)

        # L_i(x) = numerator / denominator
        basis = numerator * (y_i / denominator)
        result = result + basis

    return result


# ============================================================================
# 第2部分：R1CS约束系统
# ============================================================================

@dataclass
class R1CSConstraint:
    """单个R1CS约束: (A · w) * (B · w) = (C · w)"""
    A: Dict[int, int]  # 变量索引 -> 系数
    B: Dict[int, int]
    C: Dict[int, int]

    def check(self, witness: List[int], prime: int) -> bool:
        """验证约束"""
        def dot(vec: Dict[int, int], w: List[int]) -> int:
            return sum(vec.get(i, 0) * w[i] for i in vec.keys()) % prime

        left = dot(self.A, witness) * dot(self.B, witness) % prime
        right = dot(self.C, witness) % prime
        return left == right


class R1CS:
    """R1CS约束系统"""

    def __init__(self, num_vars: int, prime: int = 21888242871839275222246405745257275088548364400416034343698204186575808495617):
        self.num_vars = num_vars  # 变量数量 (包含常数1)
        self.prime = prime
        self.constraints: List[R1CSConstraint] = []

    def add_constraint(self, A: Dict[int, int], B: Dict[int, int], C: Dict[int, int]):
        """添加约束"""
        self.constraints.append(R1CSConstraint(A, B, C))

    def is_satisfied(self, witness: List[int]) -> bool:
        """检查见证是否满足所有约束"""
        for i, constraint in enumerate(self.constraints):
            if not constraint.check(witness, self.prime):
                print(f"约束 {i} 不满足")
                return False
        return True

    def to_qap(self) -> 'QAP':
        """转换为QAP"""
        # 为每个约束选择根: 1, 2, 3, ..., m
        roots = [FieldElement(i + 1, self.prime) for i in range(len(self.constraints))]

        # 构造目标多项式 Z(x) = (x-1)(x-2)...(x-m)
        p = self.prime
        Z = Polynomial([FieldElement(1, p)])
        for r in roots:
            Z = Z * Polynomial([FieldElement(-r.value, p), FieldElement(1, p)])

        # 为每个变量构造A_i(x), B_i(x), C_i(x)
        A_polys = []
        B_polys = []
        C_polys = []

        for var_idx in range(self.num_vars):
            # 收集该变量在所有约束中的系数
            a_points = []
            b_points = []
            c_points = []

            for i, constraint in enumerate(self.constraints):
                root = roots[i]
                a_points.append((root, FieldElement(constraint.A.get(var_idx, 0), p)))
                b_points.append((root, FieldElement(constraint.B.get(var_idx, 0), p)))
                c_points.append((root, FieldElement(constraint.C.get(var_idx, 0), p)))

            A_polys.append(interpolate(a_points))
            B_polys.append(interpolate(b_points))
            C_polys.append(interpolate(c_points))

        return QAP(A_polys, B_polys, C_polys, Z, roots, self.prime)

# 第3部分：QAP (二次算术程序)
# ============================================================================

class QAP:
    """二次算术程序"""

    def __init__(self, A: List[Polynomial], B: List[Polynomial],
                 C: List[Polynomial], Z: Polynomial,
                 roots: List[FieldElement], prime: int):
        self.A = A  # A_i(x) 多项式
        self.B = B  # B_i(x) 多项式
        self.C = C  # C_i(x) 多项式
        self.Z = Z  # 目标多项式
        self.roots = roots
        self.prime = prime

    def compute_polynomials(self, witness: List[int]) -> Tuple[Polynomial, Polynomial, Polynomial]:
        """计算A(x), B(x), C(x)"""
        p = self.prime

        # A(x) = sum(w_i * A_i(x))
        A_poly = Polynomial([FieldElement(0, p)])
        B_poly = Polynomial([FieldElement(0, p)])
        C_poly = Polynomial([FieldElement(0, p)])

        for i, w in enumerate(witness):
            w_fe = FieldElement(w, p)
            A_poly = A_poly + self.A[i] * w_fe
            B_poly = B_poly + self.B[i] * w_fe
            C_poly = C_poly + self.C[i] * w_fe

        return A_poly, B_poly, C_poly

    def check_divisibility(self, witness: List[int]) -> Tuple[bool, Polynomial]:
        """
        检查 A(x)*B(x) - C(x) 是否能被 Z(x) 整除
        返回: (是否可整除, 商多项式H(x))
        """
        A_poly, B_poly, C_poly = self.compute_polynomials(witness)

        # P(x) = A(x) * B(x) - C(x)
        P = A_poly * B_poly - C_poly

        # 检查P在所有根处为0
        for root in self.roots:
            if P(root).value != 0:
                return False, Polynomial([FieldElement(0, self.prime)])

        # 多项式除法得到 H(x) = P(x) / Z(x)
        # 简化: 这里我们验证可整除性
        return True, P  # 简化返回

def build_example_circuit():
    """
    构建示例电路: 证明知道 x 使得 x^3 + x + 5 = 35

    电路约束:
    v1 = x * x
    v2 = v1 * x
    v3 = v2 + x
    out = v3 + 5
    out = 35

    见证向量: [1, out, x, v1, v2, v3]
    索引:      0,  1,   2,  3,  4,  5
    """
    prime = 21888242871839275222246405745257275088548364400416034343698204186575808495617

    r1cs = R1CS(num_vars=6, prime=prime)

    # 约束1: v1 = x * x  ->  x * x = v1
    r1cs.add_constraint(
        A={2: 1},      # x
        B={2: 1},      # x
        C={3: 1}       # v1
    )

    # 约束2: v2 = v1 * x  ->  v1 * x = v2
    r1cs.add_constraint(
        A={3: 1},      # v1
        B={2: 1},      # x
        C={4: 1}       # v2
    )

    # 约束3: v3 = v2 + x  ->  (v2 + x) * 1 = v3
    r1cs.add_constraint(
        A={4: 1, 2: 1},  # v2 + x
        B={0: 1},        # 1 (常数)
        C={5: 1}         # v3
    )

    # 约束4: out = v3 + 5  ->  (v3 + 5) * 1 = out
    r1cs.add_constraint(
        A={5: 1, 0: 5},  # v3 + 5
        B={0: 1},        # 1
        C={1: 1}         # out
    )

    # 约束5: out = 35  ->  out * 1 = 35
    r1cs.add_constraint(
        A={1: 1},        # out
        B={0: 1},        # 1
        C={0: 35}        # 35
    )

    return r1cs


def build_range_proof_circuit(bit_length: int = 8):
    """
    构建范围证明电路: 证明 0 <= x < 2^n

    方法: 将x分解为n个比特
    x = b_0 + 2*b_1 + 4*b_2 + ... + 2^(n-1)*b_{n-1}
    且每个 b_i ∈ {0, 1}
    """
    prime = 21888242871839275222246405745257275088548364400416034343698204186575808495617
    num_bits = bit_length

    # 变量: [1, x, b_0, b_1, ..., b_{n-1}]
    r1cs = R1CS(num_vars=2 + num_bits, prime=prime)

    # 每个比特约束: b_i * (1 - b_i) = 0 => b_i - b_i^2 = 0
    for i in range(num_bits):
        bit_idx = 2 + i
        r1cs.add_constraint(
            A={bit_idx: 1},        # b_i
            B={0: 1, bit_idx: -1}, # 1 - b_i
            C={}                   # 0
        )

    # x = sum(2^i * b_i)
    # 约束: x * 1 = sum(2^i * b_i)
    # 这需要特殊的约束处理，简化表示

    return r1cs

class SimpleZKP:
    """
    简化的零知识证明演示
    展示核心思想，非完整实现
    """

    def __init__(self, prime: int):
        self.prime = prime

    def commit(self, polynomial: Polynomial, hiding: int = None) -> int:
        """
        简化的多项式承诺 (基于求值)
        实际应使用KZG或Bulletproofs
        """
        # 简化为在随机点的求值
        if hiding is None:
            hiding = random.randint(1, self.prime - 1)
        point = FieldElement(hiding, self.prime)
        return polynomial(point).value

    def prove_knowledge(self, qap: QAP, witness: List[int]) -> dict:
        """
        生成证明 (简化版)
        """
        # 1. 检查见证满足约束
        valid, H = qap.check_divisibility(witness)
        if not valid:
            raise ValueError("见证不满足约束")

        # 2. 计算多项式
        A_poly, B_poly, C_poly = qap.compute_polynomials(witness)

        # 3. 承诺 (简化)
        proof = {
            'A_commit': self.commit(A_poly),
            'B_commit': self.commit(B_poly),
            'C_commit': self.commit(C_poly),
            'witness_hash': hashlib.sha256(str(witness).encode()).hexdigest()[:16]
        }

        return proof

    def verify(self, proof: dict, public_input: List[int]) -> bool:
        """
        验证证明 (简化版)
        """
        # 简化验证: 仅检查承诺存在
        # 实际应验证配对等式
        required = ['A_commit', 'B_commit', 'C_commit']
        return all(k in proof for k in required)
```

## 5. 安全性分析

### 5.1 安全假设

**定义 5.1** (d-Power Knowledge of Exponent, d-PKE)：
给定 $(g, g^s, \ldots, g^{s^d}, h, h^s, \ldots, h^{s^d})$，难以生成 $(C, D)$ 使得 $e(C, h) = e(g, D)$ 但不知道多项式 $P$ 使得 $C = g^{P(s)}, D = h^{P(s)}$。

**定义 5.2** (q-Power Diffie-Hellman, q-PDH)：
给定 $(g, g^s, \ldots, g^{s^q}, h, h^s, \ldots, h^{s^q})$，计算 $g^{s^{q+1}}$ 是困难的。

### 5.2 可信设置的安全性

**问题**: Setup阶段需要生成随机数 $\tau$，若泄露则系统不安全。

**解决方案**:

1. **多方计算 (MPC)**: 多个参与方共同生成CRS，只要一人诚实即可保证安全
2. **可更新设置**: PLONK允许逐步增强设置安全性
3. **透明设置**: STARKs无需可信设置

### 5.3 量子安全性

zk-SNARKs基于椭圆曲线配对，**不抗量子**。

**后量子替代方案**:

- **STARKs**: 基于哈希函数，抗量子
- **Ligero**: 基于交互式Oracle证明
- **Lattice-based ZK**: 基于格困难问题
