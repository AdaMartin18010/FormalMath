## 1. 引言

同态加密(Homomorphic Encryption, HE)是密码学中的"圣杯"，允许在加密数据上直接进行计算，而无需解密。这一特性使其成为隐私保护计算的基石技术。

### 1.1 历史发展

| 年份 | 里程碑 | 贡献者 |
|-----|--------|--------|
| 1978 | 同态加密概念提出 | Rivest, Adleman, Dertouzos |
| 2009 | **首个全同态加密方案** | **Craig Gentry** |
| 2011 | BGV方案 (分层级) | Brakerski, Gentry, Vaikuntanathan |
| 2012 | BFV/FV方案 | Fan, Vercauteren; Brakerski |
| 2016 | CKKS方案 (近似计算) | Cheon, Kim, Kim, Song |
| 2018 | TFHE (快速启动) | Chillotti et al. |

### 1.2 核心问题

> **是否可以在不知道明文的情况下计算加密数据？**

传统加密：$E(m_1), E(m_2) \rightarrow$ 必须解密才能计算 $m_1 + m_2$
同态加密：$E(m_1), E(m_2) \rightarrow E(m_1 + m_2)$ （无需解密）

## 2. 数学基础

### 2.1 同态加密的定义

**定义 2.1** (同态加密)：加密方案 $(KeyGen, Enc, Dec)$ 是**同态**的，如果存在高效算法 $Eval$ 使得：

$$Dec(sk, Eval(pk, f, c_1, \ldots, c_n)) = f(m_1, \ldots, m_n)$$

其中 $c_i = Enc(pk, m_i)$，$f$ 是某个函数类中的函数。

**定义 2.2** (同态性分级)：

| 类型 | 定义 | 示例方案 |
|-----|------|---------|
| **部分同态 (PHE)** | 支持单一操作无限次 | RSA, ElGamal, Paillier |
| **些许同态 (SHE)** | 支持有限深度的任意电路 | BGN |
| **分级全同态 (LHE)** | 支持任意电路，深度受限 | BGV, BFV |
| **全同态 (FHE)** | 支持任意电路，深度无限 | Gentry09, BGV+Bootstrapping |

### 2.2 格密码学基础

**定义 2.3** (格)：设 $\mathbf{b}_1, \ldots, \mathbf{b}_n \in \mathbb{R}^m$ 是线性无关向量，**格** $L$ 定义为：

$$L = \mathcal{L}(\mathbf{B}) = \left\{\sum_{i=1}^n z_i \mathbf{b}_i : z_i \in \mathbb{Z}\right\}$$

其中 $\mathbf{B} = [\mathbf{b}_1, \ldots, \mathbf{b}_n]$ 是基矩阵。

**定义 2.4** (最短向量问题, SVP)：给定格 $L$，找到非零向量 $\mathbf{v} \in L$ 使得 $\|\mathbf{v}\|$ 最小。

**定义 2.5** (最近向量问题, CVP)：给定格 $L$ 和目标向量 $\mathbf{t} \in \mathbb{R}^m$，找到 $\mathbf{v} \in L$ 使得 $\|\mathbf{t} - \mathbf{v}\|$ 最小。

**定理 2.6** (格问题困难性)：SVP和CVP在任意范数下都是NP难问题。在多项式时间因子内近似也是困难的。

### 2.3 理想格与多项式环

**定义 2.7** (多项式环)：设 $R = \mathbb{Z}[x]/(f(x))$ 是多项式商环，其中 $f(x)$ 是 $n$ 次首一多项式。

常用选择：

- **幂次分圆**: $f(x) = x^n + 1$，其中 $n = 2^k$
- 优点：FFT友好，支持快速多项式乘法

**定义 2.8** (理想格)：设 $I$ 是 $R$ 的理想，对应 $I$ 的格称为**理想格**。

**系数嵌入**：多项式 $a(x) = a_0 + a_1x + \ldots + a_{n-1}x^{n-1}$ 对应向量 $(a_0, a_1, \ldots, a_{n-1}) \in \mathbb{Z}^n$。

**定义 2.9** (环LWE问题)：设 $R_q = \mathbb{Z}_q[x]/(f(x))$，秘密 $s \in R_q$，样本 $(a, b = a \cdot s + e \mod q)$，其中 $a \leftarrow R_q$ 均匀随机，$e$ 是小误差。

**环LWE假设**：给定多项式数量的样本，区分 $(a, b)$ 与均匀随机对是困难的。

### 2.4 RLWE问题的数学结构

**定理 2.10** (Regev, Lyubashevsky-Peikert-Regev)：

- 在某些参数选择下，决策环LWE问题的困难性可归约到理想格上的SVP
- 量子归约：量子算法可以解决SVP $\Rightarrow$ 量子算法可以解决RLWE
- 经典归约在某些参数下也存在

**误差分布**：

- 离散高斯分布 $\chi$：$\Pr[x] \propto e^{-\pi x^2/\sigma^2}$
- 参数 $\sigma$ 控制误差大小
- 标准差 $\sigma$ 通常是安全参数的多项式

## 3. BFV/BGV方案详解

### 3.1 方案概览

**BFV (Brakerski-Fan-Vercauteren)** 是目前最常用的同态加密方案之一。

**参数**：

- $n$：多项式次数（通常是2的幂）
- $q$：密文模数（大整数）
- $t$：明文模数（$t < q$，通常是2或小的素数）
- $\chi$：误差分布

### 3.2 密钥生成

$$KeyGen(1^\lambda):$$

1. 选择 $s \leftarrow \chi$ （秘密密钥）
2. 选择 $a \leftarrow R_q$ 均匀随机，$e \leftarrow \chi$
3. 计算 $b = -(a \cdot s + e) \in R_q$
4. 公钥 $pk = (b, a)$，私钥 $sk = s$

### 3.3 加密

$$Enc(pk, m \in R_t):$$

1. 编码：$\Delta = \lfloor q/t \rfloor$，$m' = \Delta \cdot m \in R_q$
2. 选择 $u \leftarrow \chi$，$e_1, e_2 \leftarrow \chi$
3. 计算：
   $$c_0 = b \cdot u + e_1 + m'$$
   $$c_1 = a \cdot u + e_2$$
4. 输出密文 $ct = (c_0, c_1) \in R_q^2$

**解密方程**：
$$c_0 + c_1 \cdot s = \Delta \cdot m + e_{enc}$$

其中 $e_{enc} = e \cdot u + e_1 + e_2 \cdot s$ 是加密误差。

### 3.4 解密

$$Dec(sk, ct = (c_0, c_1)):$$

1. 计算 $m_{q} = c_0 + c_1 \cdot s \in R_q$
2. 解码：$m = \lfloor t/q \cdot m_q \rceil \mod t$

**正确性条件**：$|e_{enc}| < q/(2t)$

### 3.5 同态运算

**加法**：给定 $ct_1 = (c_{1,0}, c_{1,1})$，$ct_2 = (c_{2,0}, c_{2,1})$
$$ct_{add} = (c_{1,0} + c_{2,0}, c_{1,1} + c_{2,1})$$

解密：$Dec(ct_{add}) = m_1 + m_2 \mod t$（误差相加）

**乘法**（更复杂）：

1. 计算张量积：$(c_{1,0} + c_{1,1} \cdot s)(c_{2,0} + c_{2,1} \cdot s)$
2. 展开得到关于 $s$ 的二次多项式
3. **重线性化 (Relinearization)**：将3个分量的密文转换回2个分量
4. 结果密文 $ct_{mult}$ 满足 $Dec(ct_{mult}) = m_1 \cdot m_2 \mod t$

**乘法误差增长**：误差大约平方增长，限制计算深度。

### 3.6 模切换 (Modulus Switching)

BGV方案的关键技术，用于管理噪声增长：

$$ct \in R_q^2 \rightarrow ct' \in R_{q'}^2$$

通过缩放密文分量：$ct' = \lfloor q'/q \cdot ct \rceil$

效果：降低噪声绝对值，但增加相对噪声

### 3.7 启动 (Bootstrapping)

**问题**：乘法后噪声增长，最终会导致解密失败。

**Gentry的突破**：同态地评估解密电路！

$$Bootstrapping(ct_{noisy}) = Enc(Dec(sk, ct_{noisy})) = ct_{fresh}$$

通过将解密电路表示为同态可计算的布尔电路，可以用加密的私钥对密文进行"刷新"。

**TFHE方案**优化了启动过程，使其仅需毫秒级时间。

"""

import numpy as np
from numpy.polynomial import polynomial as P
from typing import Tuple, List, Optional
import hashlib
import secrets

# ============================================================================
# 第1部分：多项式环运算
# ============================================================================

class PolynomialRing:
    """
    多项式环 R = Z[x] / (x^n + 1)
    使用数论变换(NTT)加速乘法
    """

    def __init__(self, coeffs: np.ndarray, modulus: int):
        """
        coeffs: 系数数组 [a0, a1, ..., a_{n-1}]
        modulus: 模数
        """
        self.n = len(coeffs)
        self.q = modulus
        self.coeffs = np.mod(coeffs, modulus).astype(np.int64)

    @classmethod
    def random(cls, n: int, modulus: int, min_val: int = 0, max_val: int = None) -> 'PolynomialRing':
        """生成随机多项式"""
        if max_val is None:
            max_val = modulus
        coeffs = np.random.randint(min_val, max_val, size=n)
        return cls(coeffs, modulus)

    @classmethod
    def gaussian(cls, n: int, modulus: int, sigma: float = 3.2) -> 'PolynomialRing':
        """高斯分布多项式 (误差分布)"""
        coeffs = np.random.normal(0, sigma, size=n).astype(np.int64)
        return cls(coeffs, modulus)

    @classmethod
    def from_constant(cls, n: int, modulus: int, value: int) -> 'PolynomialRing':
        """常数多项式"""
        coeffs = np.zeros(n, dtype=np.int64)
        coeffs[0] = value % modulus
        return cls(coeffs, modulus)

    def __add__(self, other: 'PolynomialRing') -> 'PolynomialRing':
        """多项式加法"""
        assert self.n == other.n and self.q == other.q
        return PolynomialRing(np.mod(self.coeffs + other.coeffs, self.q), self.q)

    def __sub__(self, other: 'PolynomialRing') -> 'PolynomialRing':
        """多项式减法"""
        assert self.n == other.n and self.q == other.q
        return PolynomialRing(np.mod(self.coeffs - other.coeffs, self.q), self.q)

    def __mul__(self, other) -> 'PolynomialRing':
        """
        多项式乘法 (模 x^n + 1)
        使用朴素O(n^2)算法，实际应使用NTT
        """
        if isinstance(other, int):
            return PolynomialRing(np.mod(self.coeffs * other, self.q), self.q)

        assert self.n == other.n and self.q == other.q

        # 卷积乘法
        result = np.convolve(self.coeffs, other.coeffs)

        # 模 x^n + 1: x^n = -1
        # 高次项折回
        if len(result) > self.n:
            # x^n = -1, x^{n+k} = -x^k
            for i in range(self.n, len(result)):
                result[i - self.n] = (result[i - self.n] - result[i]) % self.q
            result = result[:self.n]

        return PolynomialRing(np.mod(result, self.q), self.q)

    def __rmul__(self, scalar: int) -> 'PolynomialRing':
        return self.__mul__(scalar)

    def __neg__(self) -> 'PolynomialRing':
        return PolynomialRing(np.mod(-self.coeffs, self.q), self.q)

    def __eq__(self, other) -> bool:
        return (self.n == other.n and self.q == other.q and
                np.array_equal(self.coeffs, other.coeffs))

    def __repr__(self):
        terms = []
        for i, c in enumerate(self.coeffs):
            if c != 0:
                if i == 0:
                    terms.append(str(c))
                elif i == 1:
                    terms.append(f"{c}x")
                else:
                    terms.append(f"{c}x^{i}")
        return " + ".join(terms) if terms else "0"

    def copy(self) -> 'PolynomialRing':
        return PolynomialRing(self.coeffs.copy(), self.q)

    def infinity_norm(self) -> int:
        """无穷范数: 最大绝对值系数"""
        # 将系数映射到 [-q/2, q/2] 范围
        centered = np.where(self.coeffs > self.q // 2,
                           self.coeffs - self.q,
                           self.coeffs)
        return int(np.max(np.abs(centered)))

class BFVParameters:
    """BFV方案参数"""

    def __init__(self, n: int, q: int, t: int, sigma: float = 3.2):
        """
        n: 多项式次数 (必须是2的幂)
        q: 密文模数
        t: 明文模数
        sigma: 误差分布标准差
        """
        assert (n & (n - 1)) == 0, "n必须是2的幂"
        self.n = n
        self.q = q
        self.t = t
        self.sigma = sigma

        # 缩放因子
        self.delta = q // t


class BFVKeyPair:
    """BFV密钥对"""

    def __init__(self, secret_key: PolynomialRing,
                 public_key: Optional[Tuple[PolynomialRing, PolynomialRing]] = None):
        self.sk = secret_key
        self.pk = public_key


class BFVCiphertext:
    """BFV密文"""

    def __init__(self, c0: PolynomialRing, c1: PolynomialRing):
        self.c0 = c0
        self.c1 = c1

    def __add__(self, other: 'BFVCiphertext') -> 'BFVCiphertext':
        return BFVCiphertext(self.c0 + other.c0, self.c1 + other.c1)

    def __sub__(self, other: 'BFVCiphertext') -> 'BFVCiphertext':
        return BFVCiphertext(self.c0 - other.c0, self.c1 - other.c1)

    def __repr__(self):
        return f"CT(c0={self.c0.coeffs[:3]}..., c1={self.c1.coeffs[:3]}...)"


class BFVScheme:
    """
    BFV同态加密方案简化实现

    支持:
    - 加密/解密
    - 同态加法
    - 密文-明文乘法
    - 密文-密文乘法 (简化版)
    """

    def __init__(self, params: BFVParameters):
        self.params = params

    def keygen(self) -> BFVKeyPair:
        """
        密钥生成
        sk = s ← χ (误差分布)
        pk = (b, a) where b = -(a·s + e), a ← R_q uniform, e ← χ
        """
        n, q = self.params.n, self.params.q

        # 秘密密钥
        s = PolynomialRing.gaussian(n, q, self.params.sigma)

        # 公钥
        a = PolynomialRing.random(n, q)
        e = PolynomialRing.gaussian(n, q, self.params.sigma)

        # b = -(a*s + e)
        b = -(a * s + e)

        return BFVKeyPair(s, (b, a))

    def encode(self, plaintext: int) -> PolynomialRing:
        """
        编码: 整数 → 多项式 (Delta * m)
        """
        m_poly = PolynomialRing.from_constant(
            self.params.n, self.params.q,
            (plaintext % self.params.t) * self.params.delta
        )
        return m_poly

    def decode(self, poly: PolynomialRing) -> int:
        """
        解码: 多项式 → 整数
        m = round(t/q * m_q) mod t
        """
        # 取常数项
        val = poly.coeffs[0]

        # 缩放并四舍五入
        scaled = round(val * self.params.t / self.params.q)

        return int(scaled) % self.params.t

    def encrypt(self, pk: Tuple[PolynomialRing, PolynomialRing],
                plaintext: int) -> BFVCiphertext:
        """
        加密
        c0 = b*u + e1 + Delta*m
        c1 = a*u + e2
        """
        n, q = self.params.n, self.params.q
        b, a = pk

        # 编码明文
        m_encoded = self.encode(plaintext)

        # 随机元素
        u = PolynomialRing.gaussian(n, q, self.params.sigma)
        e1 = PolynomialRing.gaussian(n, q, self.params.sigma)
        e2 = PolynomialRing.gaussian(n, q, self.params.sigma)

        # 计算密文
        c0 = b * u + e1 + m_encoded
        c1 = a * u + e2

        return BFVCiphertext(c0, c1)

    def decrypt(self, sk: PolynomialRing, ct: BFVCiphertext) -> int:
        """
        解密
        m_q = c0 + c1 * s
        m = decode(m_q)
        """
        # 计算 c0 + c1 * s
        m_poly = ct.c0 + ct.c1 * sk

        return self.decode(m_poly)

    def add(self, ct1: BFVCiphertext, ct2: BFVCiphertext) -> BFVCiphertext:
        """同态加法"""
        return ct1 + ct2

    def add_plain(self, ct: BFVCiphertext, pt: int) -> BFVCiphertext:
        """密文 + 明文"""
        pt_encoded = self.encode(pt)
        return BFVCiphertext(ct.c0 + pt_encoded, ct.c1)

    def multiply_plain(self, ct: BFVCiphertext, pt: int) -> BFVCiphertext:
        """密文 * 明文"""
        # 明文编码为多项式
        pt_poly = PolynomialRing.from_constant(
            self.params.n, self.params.q,
            (pt % self.params.t) * self.params.delta
        )
        return BFVCiphertext(ct.c0 * pt_poly, ct.c1 * pt_poly)

    def noise_budget(self, sk: PolynomialRing, ct: BFVCiphertext) -> float:
        """
        估计剩余噪声预算
        返回可用乘法深度
        """
        # 计算当前噪声
        m_poly = ct.c0 + ct.c1 * sk
        noise = m_poly.infinity_norm()

        # 最大允许噪声
        max_noise = self.params.q // (2 * self.params.t)

        # 预算 (对数尺度)
        if noise >= max_noise:
            return 0.0

        budget = np.log2(max_noise / noise)
        return max(0.0, budget)

def demo_basic_encryption():
    """基本加密演示"""
    print("=" * 60)

    print("=" * 60)

    # 密钥生成
    keypair = bfv.keygen()

    # 加密
    plaintext = 42
    print(f"\n明文: {plaintext}")

    ct = bfv.encrypt(keypair.pk, plaintext)
    print(f"密文生成完成")

    # 解密
    decrypted = bfv.decrypt(keypair.sk, ct)
    print(f"解密结果: {decrypted}")
    print(f"正确性: {'✓' if decrypted == plaintext else '✗'}")

    # 噪声预算
    budget = bfv.noise_budget(keypair.sk, ct)
    print(f"噪声预算: {budget:.2f} bits")

    print()


def demo_homomorphic_addition():
    """同态加法演示"""
    print("=" * 60)
    print("同态加法演示")
    print("=" * 60)

    params = BFVParameters(n=2048, q=2**54, t=65537, sigma=3.2)
    bfv = BFVScheme(params)
    keypair = bfv.keygen()

    # 加密两个数
    m1, m2 = 10, 25
    ct1 = bfv.encrypt(keypair.pk, m1)
    ct2 = bfv.encrypt(keypair.pk, m2)

    print(f"加密: {m1} 和 {m2}")

    # 同态加法
    ct_sum = bfv.add(ct1, ct2)
    print("执行同态加法...")

    # 解密结果
    result = bfv.decrypt(keypair.sk, ct_sum)
    expected = (m1 + m2) % params.t

    print(f"解密结果: {result}")
    print(f"期望结果: {expected}")
    print(f"正确性: {'✓' if result == expected else '✗'}")

    # 噪声预算变化
    budget_before = bfv.noise_budget(keypair.sk, ct1)
    budget_after = bfv.noise_budget(keypair.sk, ct_sum)
    print(f"噪声预算: 加密后={budget_before:.2f}, 加法后={budget_after:.2f}")

    print()

    """加密计算演示"""
    print("=" * 60)
    print("加密计算演示: f(x) = 2x^2 + 3x + 5")
    print("=" * 60)

    # 使用更大的参数支持更多运算
    params = BFVParameters(n=4096, q=2**109, t=65537, sigma=3.2)
    bfv = BFVScheme(params)
    keypair = bfv.keygen()

    x = 4
    print(f"输入 x = {x}")

    # 加密输入
    ct_x = bfv.encrypt(keypair.pk, x)
    print("加密完成")

    # 计算 2x^2 + 3x + 5
    # 步骤1: x^2 = x * x
    ct_x2 = bfv.multiply_plain(ct_x, x)  # x * x
    print("计算 x^2...")

    # 步骤2: 2x^2
    ct_2x2 = bfv.multiply_plain(ct_x2, 2)
    print("计算 2x^2...")

    # 步骤3: 3x
    ct_3x = bfv.multiply_plain(ct_x, 3)
    print("计算 3x...")

    # 步骤4: 2x^2 + 3x
    ct_sum = bfv.add(ct_2x2, ct_3x)
    print("计算 2x^2 + 3x...")

    # 步骤5: + 5
    ct_result = bfv.add_plain(ct_sum, 5)
    print("计算 + 5...")

    # 解密
    result = bfv.decrypt(keypair.sk, ct_result)
    expected = 2 * x**2 + 3 * x + 5

    print(f"\n解密结果: {result}")
    print(f"期望结果: {expected}")
    print(f"正确性: {'✓' if result == expected else '✗'}")

    budget = bfv.noise_budget(keypair.sk, ct_result)
    print(f"剩余噪声预算: {budget:.2f} bits")

    print()


def demo_batch_encryption():
    """批处理加密演示 (SIMD)"""
    print("=" * 60)
    print("批处理加密演示 (单密文多明文)")
    print("=" * 60)

    print("""
BFV/BGV支持批处理 (batching):
- 使用中国剩余定理 (CRT)
- 将多个明文编码到单个密文的不同"槽"中
- 单次运算同时作用于所有槽
- 类似于SIMD指令

槽数 = n (多项式次数)
每个槽可存储 mod t 的整数

应用:
- 加密向量运算
- 机器学习推理
- 数据库查询
    """)

    params = BFVParameters(n=2048, q=2**54, t=65537, sigma=3.2)
    bfv = BFVScheme(params)
    keypair = bfv.keygen()

    # 模拟批处理: 每个系数存储一个值
    batch_size = min(8, params.n)  # 演示用小的批大小
    plaintexts = list(range(1, batch_size + 1))

    print(f"批处理大小: {batch_size}")
    print(f"明文值: {plaintexts}")

    # 创建批处理明文多项式
    coeffs = np.zeros(params.n, dtype=np.int64)
    for i, pt in enumerate(plaintexts):
        coeffs[i] = (pt * params.delta) % params.q

    m_batch = PolynomialRing(coeffs, params.q)

    # 手动构造加密 (简化)
    u = PolynomialRing.gaussian(params.n, params.q, params.sigma)
    e1 = PolynomialRing.gaussian(params.n, params.q, params.sigma)
    e2 = PolynomialRing.gaussian(params.n, params.q, params.sigma)

    b, a = keypair.pk
    c0 = b * u + e1 + m_batch
    c1 = a * u + e2

    ct_batch = BFVCiphertext(c0, c1)
    print("批处理加密完成")

    # 解密并提取
    m_decoded = c0 + c1 * keypair.sk

    print("解密结果:")
    for i in range(batch_size):
        val = round(m_decoded.coeffs[i] * params.t / params.q) % params.t
        print(f"  槽 {i}: {val} (期望: {plaintexts[i]})")

    print()

## 5. 安全性分析

### 5.1 安全参数选择

**定理 5.1** (RLWE安全性)：对于适当选择的参数 $(n, q, \sigma)$，RLWE问题的困难性与最坏情况下理想格上近似SVP的困难性相关。

**推荐参数** (128-bit安全级别)：

| 方案 | $n$ | $\log_2 q$ | $\sigma$ | 乘法深度 |
|-----|-----|-----------|----------|---------|
| BFV | 4096 | 109 | 3.2 | ~10 |
| BFV | 16384 | 438 | 3.2 | ~20 |
| CKKS | 8192 | 218 | 3.2 | ~15 |

### 5.2 已知攻击

**1. 对偶攻击 (Dual Attack)**：

- 利用格的约化基找到短向量
- BKZ算法及其变体
- 参数选择需考虑量子BKZ

**2. 代数攻击**：

- 利用分圆域的特殊结构
- 针对特定NTRU参数的亚指数攻击
- 当前参数设置下不可行

**3. 侧信道攻击**：

- 密钥恢复攻击
- 噪声分析攻击
- 需要恒定时间实现

### 5.3 量子安全性

格密码学是**后量子密码**的主要候选：

- 目前没有已知的量子算法能高效解决SVP/CVP
- Shor算法不适用于格问题
- NIST后量子标准化进程包含多个格方案
