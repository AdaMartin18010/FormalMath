# 基本群思维导图

## 概述
基本群是代数拓扑中最基本的不变量，通过研究空间中环路（基于点的闭道路）的同伦类来刻画空间的"洞"结构。

## 常见空间的基本群

| 空间 | 基本群 | 说明 |
|------|--------|------|
| **ℝⁿ, Dⁿ** | 0 | 可缩空间 |
| **S¹** | ℤ | 非平凡最基本 |
| **Sⁿ (n≥2)** | 0 | 单连通 |
| **Tⁿ = (S¹)ⁿ** | ℤⁿ | n维环面 |
| **RPⁿ (n≥2)** | ℤ/2ℤ | 实射影空间 |
| **楔形∨S¹ (k个)** | Fₖ | k生成元自由群 |

## van Kampen定理

若 $X = U \cup V$，$U, V, U \cap V$ 都道路连通，则：
$$\pi_1(X) \cong \pi_1(U) *_{\pi_1(U \cap V)} \pi_1(V)$$

（自由积的融合积）
