# 代数簇思维导图

## 概述
代数簇是多项式方程组的零点集，是代数几何的基本研究对象，连接了代数和几何，为研究方程的解提供了几何框架。

## 代数簇的基本对应

```

仿射簇 V ⊂ Aⁿ  ⟷  根理想 I(V) ⊂ k[x₁,...,xₙ]
     ↓                           ↓
   不可约簇     ⟷     素理想
     ↓                           ↓
   点         ⟷     极大理想

```

## 常见代数簇

| 簇 | 定义 | 维数 | 特点 |
|----|------|------|------|
| **Aⁿ** | 仿射空间 | n | 非紧 |
| **Pⁿ** | 射影空间 | n | 紧、完备 |
| **V(xy-1)** | 双曲线 | 1 | 同构于A¹-{0} |
| **V(y-x²)** | 抛物线 | 1 | 同构于A¹ |
| **V(y²-x³)** | 尖点曲线 | 1 | 奇异点在原点 |

## 核心定理

| 定理 | 内容 |
|------|------|
| **Hilbert零点** | I(V(J)) = √J（代数闭域） |
| **Bezout** | 次数d,e的射影平面曲线交于de点（计重数） |
| **Hironaka** | 特征0域上奇点可消解 |
