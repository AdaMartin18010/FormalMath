# 层思维导图

## 概述
层是将局部数据系统地粘合为整体结构的数学工具，是现代代数几何、复几何和代数拓扑的基础，为研究局部-整体关系提供了框架。

## 层的定义层次

```
预层 (Presheaf): 只有限制映射
    ↓ + 层公理
层 (Sheaf): 局部性 + 可粘合性
    ↓ + Abel群结构
Abel层: 截面是Abel群
    ↓ + O_X-模结构
O_X-模: 结构层上的模
```

## 层的几何对应

| 几何对象 | 层论对应 | 说明 |
|---------|---------|------|
| 向量丛 | 局部自由层 | 截面层 |
| 除子 | 可逆层 | O_X(D) |
| 闭子簇 | 结构层理想 | I_Y |
| 微分形式 | Ω^p | Kähler微分 |
| 上同调 | 导出函子H^i | 整体截面 |

## 核心定理

| 定理 | 内容 |
|------|------|
| **Serre对偶** | Hⁱ(X,F) ≅ H^{n-i}(X, F*⊗ω_X)* |
| **Cartan定理B** | Stein流形上凝聚层的高阶上同调为零 |
| **消失定理** | 正线丛的高阶上同调消失(Kodaira) |
