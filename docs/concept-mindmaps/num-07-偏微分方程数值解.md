# 偏微分方程数值解 思维导图

## 1. 偏微分方程分类

### 1.1 方程类型
- **椭圆型**
  - Laplace方程: ∇²u = 0
  - Poisson方程: ∇²u = f
  - Helmholtz方程
- **抛物型**
  - 热方程: uₜ = α∇²u
  - 扩散方程
- **双曲型**
  - 波动方程: uₜₜ = c²∇²u
  - 对流方程: uₜ + cuₓ = 0

### 1.2 定解条件
- **初值条件**
- **边界条件**
  - Dirichlet
  - Neumann
  - Robin
- **适定性**

## 2. 有限差分法

### 2.1 差分格式
- **空间离散**
  - 中心差分: O(h²)
  - 向前/向后差分: O(h)
- **时间离散**
  - 显式格式
  - 隐式格式
  - Crank-Nicolson

### 2.2 抛物型方程
- **显式格式**
  - 条件稳定
  - CFL条件: r = αΔt/Δx² ≤ 0.5
- **隐式格式**
  - 无条件稳定
  - 需要解线性系统
- **Crank-Nicolson**
  - 二阶精度
  - 无条件稳定

### 2.3 双曲型方程
- **迎风格式**
  - 根据特征方向
- **Lax-Friedrichs**
- **Lax-Wendroff**
  - 二阶精度
- **蛙跳格式**

### 2.4 椭圆型方程
- **五点差分**
  - 2D Laplace
- **九点差分**
  - 高精度
- **稀疏线性系统**

### 2.5 稳定性与收敛
- **von Neumann分析**
- **CFL条件**
- **Lax等价定理**
