# 切空间思维导图

## 概述
切空间是微分流形上某点处所有切向量的集合，它是流形在该点的线性近似，构成了流形上微分计算的基础。

## 典型切空间维度

| 流形 | 切空间维数 | 切丛维数 |
|------|-----------|---------|
| ℝⁿ | n | 2n |
| Sⁿ | n | 2n |
| T² | 2 | 4 |
| GL(n,ℝ) | n² | 2n² |
