# 非线性方程求解 思维导图

## 1. 单变量方程求根

### 1.1 二分法
- **算法原理**
  - 区间对半分割
  - 中间值定理
- **收敛性**
  - 线性收敛
  - 误差界: (b-a)/2ⁿ⁺¹
- **优缺点**
  - 稳健但慢
  - 保证收敛

### 1.2 不动点迭代
- **迭代格式**
  - xₖ₊₁ = φ(xₖ)
- **收敛条件**
  - |φ'(x*)| < 1
  - 局部收敛
- **加速技术**
  - Aitken Δ²加速
  - Steffensen方法

### 1.3 Newton法
- **算法原理**
  - xₖ₊₁ = xₖ - f(xₖ)/f'(xₖ)
- **收敛性**
  - 二阶收敛
  - 局部收敛
- **改进版本**
  - 简化Newton法
  - 修正Newton法

### 1.4 割线法
- **算法原理**
  - 差商代替导数
  - xₖ₊₁ = xₖ - f(xₖ)(xₖ-xₖ₋₁)/(f(xₖ)-f(xₖ₋₁))
- **收敛性**
  - 超线性收敛 (阶≈1.618)
- **多点割线法**

### 1.5 抛物线法 (Muller法)
- **二次插值**
- **收敛阶**
  - ≈1.839

## 2. 非线性方程组

### 2.1 Newton-Raphson法
- **算法原理**
  - xₖ₊₁ = xₖ - J⁻¹(xₖ)F(xₖ)
- **Jacobian矩阵**
  - Jᵢⱼ = ∂fᵢ/∂xⱼ
- **收敛性**
  - 二阶收敛
  - 局部收敛

### 2.2 拟Newton法
- **Broyden方法**
  - Jacobian更新
  - 秩1更新
- **BFGS方法**
  - 秩2更新
  - 对称正定
- **DFP方法**

### 2.3 不动点迭代
- **Jacobi迭代**
- **Gauss-Seidel迭代**
- **SOR方法**
