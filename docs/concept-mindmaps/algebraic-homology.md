# 同调群思维导图

## 概述
同调群是代数拓扑中的核心不变量，通过代数化的方式刻画空间的"洞"结构（包括高维洞），比基本群更易于计算且适用范围更广。

## 常见空间的同调群

| 空间 | H₀ | H₁ | H₂ | Hₙ (n≥3) |
|------|----|----|----|---------|
| **点** | ℤ | 0 | 0 | 0 |
| **S¹** | ℤ | ℤ | 0 | 0 |
| **Sⁿ** | ℤ | 0 | ... | ℤ (n维) |
| **T²** | ℤ | ℤ² | ℤ | 0 |
| **RP²** | ℤ | ℤ/2ℤ | 0 | 0 |
| **Klein瓶** | ℤ | ℤ⊕ℤ/2ℤ | 0 | 0 |

## 核心定理

| 定理 | 内容 |
|------|------|
| **同伦不变性** | 同伦等价的空间有同构的同调群 |
| **Mayer-Vietoris** | 分解空间计算同调的正合序列 |
| **万有系数** | Hₙ(X;G)由Hₙ(X;ℤ)和Hₙ₋₁(X;ℤ)决定 |
