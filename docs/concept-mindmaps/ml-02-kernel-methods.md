# 核方法 - 思维导图

## 核心思想

### 核技巧本质
- **基本思想**: 隐式映射到高维特征空间
- **关键洞察**: 只需计算内积，无需显式映射
- **核函数**: K(x, x') = φ(x)ᵀφ(x')
- **计算优势**: 避免高维计算灾难

### 特征空间映射
- **原始空间**: x ∈ ℝᵈ
- **特征空间**: φ(x) ∈ ℝᴰ (D ≫ d 或无限维)
- **线性不可分 → 线性可分**: 通过适当映射
- **示例**: XOR问题通过二次映射可解

## 核函数理论

### 正定核定义
- **对称性**: K(x, x') = K(x', x)
- **正定性**: ∀cᵢ, xᵢ, Σᵢⱼ cᵢcⱼK(xᵢ, xⱼ) ≥ 0
- **Mercer定理**: 连续对称核正定的充要条件
- **核矩阵**: Gram矩阵半正定

### 常见核函数

#### 线性核
- **形式**: K(x, x') = xᵀx'
- **特点**: 原始空间线性模型
- **适用**: 高维稀疏数据
- **计算**: O(d)

#### 多项式核
- **形式**: K(x, x') = (γxᵀx' + r)ᵈ
- **参数**:
  - γ: 缩放系数
  - r: 偏置项
  - d: 多项式次数
- **特征空间**: 所有次数≤d的单项式
- **计算**: O(d)

#### 高斯RBF核
- **形式**: K(x, x') = exp(-γ‖x - x'‖²)
- **参数**: γ = 1/(2σ²)
- **特征空间**: 无限维
- **局部性**: 近邻点影响大
- **通用近似**: 可逼近任意连续函数

#### Sigmoid核
- **形式**: K(x, x') = tanh(γxᵀx' + r)
- **类比**: 神经网络激活函数
- **限制**: 不一定正定
- **用途**: 模拟神经网络行为

### 核函数构造
- **缩放**: c·K(x, x')
- **平移**: K(x, x') + c
- **加法**: K₁ + K₂
- **乘法**: K₁ · K₂
- **张量积**: K₁ ⊗ K₂
- **复合**: f(K(x, x')) 其中f为正定函数

## 核方法算法

### 核化线性回归
- **原始形式**: ŷ = wᵀφ(x)
- **核形式**: ŷ = Σᵢ αᵢ K(xᵢ, x)
- **解**: α = (K + λI)⁻¹y
- **计算复杂度**: O(n³) vs O(D³)

### 支持向量机 (SVM)
- **原始优化**: min ½‖w‖² + CΣξᵢ
- **对偶形式**: max Σαᵢ - ½ΣαᵢαⱼyᵢyⱼK(xᵢ, xⱼ)
- **约束**: 0 ≤ αᵢ ≤ C, Σαᵢyᵢ = 0
- **决策函数**: ŷ = sign(ΣαᵢyᵢK(xᵢ, x) + b)

### 核PCA
- **目标**: 特征空间中的主成分分析
- **中心化核矩阵**: K̃ = K - 1ₙK - K1ₙ + 1ₙK1ₙ
- **特征分解**: K̃v = λv
- **投影**: 新点投影到第k个主成分

### 高斯过程
- **先验**: f ~ GP(0, K)
- **后验**: f*|X*,X,y ~ N(μ*, Σ*)

- **预测均值**: μ* = K*ᵀ(K + σ²I)⁻¹y
- **预测协方差**: Σ* = K** - K*ᵀ(K + σ²I)⁻¹K*
- **核函数**: 定义函数平滑性

### 核K-means
- **目标**: 特征空间中的聚类
- **距离计算**: ‖φ(x) - μⱼ‖² = K(x,x) - (2/nⱼ)ΣK(x,xᵢ) + (1/nⱼ²)ΣΣK(xᵢ,xₖ)
- **算法**: 标准K-means的核版本

## 核的数学理论

### 再生核希尔伯特空间 (RKHS)
- **定义**: 具有再生性质的Hilbert函数空间
- **再生性质**: f(x) = ⟨f, K(x,·)⟩_H
- **表示定理**: 最优解可表示为核的线性组合
- **范数**: ‖f‖_H 控制函数复杂度

### 表示定理
- **一般形式**: min L(f(x₁),...,f(xₙ)) + Ω(‖f‖_H)
- **解的形式**: f*(x) = Σᵢ αᵢ K(xᵢ, x)
- **意义**: 无限维优化→有限维优化
- **应用**: 所有基于RKHS正则化的方法

### 核矩阵分析
- **条件数**: 数值稳定性指标
- **谱分析**: 特征值衰减与泛化
- **低秩近似**: Nyström方法
- **稀疏近似**: 诱导点方法

### 泛化理论
- **Rademacher复杂度**: 核方法的复杂度度量
- **覆盖数**: 函数空间的覆盖程度
- **泛化界**: 与核范数、样本量相关
- **大间隔理论**: SVM的几何解释

## 计算优化

### 大规模核方法
- **问题**: n大时O(n²)存储，O(n³)计算
- **近似策略**:
  - 低秩近似
  - 随机特征
  - 分块计算
  - 在线学习

### 随机傅里叶特征
- **Bochner定理**: 平移不变核的谱表示
- **近似**: K(x,x') ≈ z(x)ᵀz(x')
- **特征映射**: z(x) = √(2/D)[cos(w₁ᵀx+b₁), ..., cos(wᴅᵀx+bᴅ)]
- **复杂度**: 从O(n²)降到O(nD)

### Nyström方法
- **思想**: 用m个诱导点近似核矩阵
- **近似**: K ≈ Kₙₘ Kₘₘ⁻¹ Kₘₙ
- **选择策略**:
  - 随机采样
  - 列范数采样
  - K-means聚类中心
- **复杂度**: O(m²n + m³)

## 核选择与设计

### 核选择准则
- **交叉验证**: 数据驱动的选择
- **边际最大化**: SVM中的核选择
- **贝叶斯方法**: 核作为超参数
- **多核学习**: 自动学习核组合

### 多核学习
- **形式**: K = Σₘ dₘ Kₘ, dₘ ≥ 0
- **优化**: 联合学习权重和模型
- **应用**: 异构数据融合
- **方法**: 单级/两级优化

### 深度学习中的核
- **神经正切核 (NTK)**: 宽神经网络的核极限
- **核回归等价**: 梯度下降等价于核方法
- **泛化分析**: 通过NTK理解神经网络
- **动态核**: 训练过程中的核演化

## 高级主题

### 非正定核
- **不定核**: 不满足正定性
- **Kreĭn空间**: 不定内积空间
- **应用**: 相似性学习、距离替代

### 条件正定核
- **定义**: 在约束条件下正定
- **例子**: 薄板样条核
- **应用**: 插值、散点数据拟合

### 算子值核
- **向量值函数**: 多输出学习
- **矩阵值核**: K(x,x') ∈ ℝᵐˣᵐ
- **应用**: 多任务学习
