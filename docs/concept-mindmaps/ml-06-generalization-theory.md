# 泛化理论 - 思维导图

## 基础概念

### 泛化问题
- **核心问题**: 为什么训练好的模型能在测试集上表现好？
- **过拟合**: 训练误差低但测试误差高
- **欠拟合**: 训练和测试误差都高
- **泛化误差**: 期望风险与经验风险的差距

### 风险分解
- **期望风险 (R(f))**: 总体分布上的期望损失
- **经验风险 (R̂(f))**: 训练集上的平均损失
- **泛化误差**: R(f) - R̂(f)
- **目标**: 最小化期望风险

### 偏差-方差分解
- **分解**: E[(Y - f̂(X))²] = Bias² + Variance + Noise
- **偏差**: 模型假设导致的系统误差
  - 高偏差 → 欠拟合
- **方差**: 数据扰动导致的误差
  - 高方差 → 过拟合
- **噪声**: 数据本身的不可约误差

### 学习框架
- **经验风险最小化 (ERM)**: 最小化训练误差
- **结构风险最小化 (SRM)**: ERM + 正则化
- **一致收敛**: 经验风险→期望风险
- **PAC学习**: 概率近似正确

## 经典泛化理论

### VC维理论

#### 定义
- **打散 (Shattering)**: 假设类能拟合所有可能标记
- **VC维**: 能打散的最大样本数
- **生长函数**: 假设类在不同样本上的最大分类数
- **Sauer引理**: 生长函数的多项式上界

#### 泛化界
- **一致收敛**: 以高概率
  - R(f) ≤ R̂(f) + O(√(VC(H)log(n)/n))
- **VC维与参数**: 通常与参数数量相关
- **样本复杂度**: n = O(VC(H)/ε) 达到精度ε

#### 计算复杂度
- **ERM的困难**: 即使VC维小，ERM也可能NP难
- **不可知学习**: 最小化经验风险可能困难

### Rademacher复杂度

#### 定义
- **Rademacher变量**: εᵢ ∈ {-1, +1} 均匀随机
- **经验Rademacher复杂度**:
  - R̂ₙ(F) = E_ε[sup_{f∈F}(1/n)Σᵢ εᵢf(xᵢ)]
- **Rademacher复杂度**: Rₙ(F) = E[R̂ₙ(F)]
- **解释**: 假设类拟合随机噪声的能力

#### 性质
- **范围**: 0 ≤ Rₙ(F) ≤ sup|f|

- **单调性**: F ⊆ G → Rₙ(F) ≤ Rₙ(G)
- **Lipschitz复合**: Lipschitz函数不改变复杂度阶
- **有限类**: Rₙ(F) ≤ C√(log|F|/n)

#### 泛化界
- **基于Rademacher**: 更紧的界
  - R(f) ≤ R̂(f) + 2Rₙ(F) + O(√(log(1/δ)/n))
- **优点**: 数据依赖、通常比VC界紧

### 覆盖数与填充数

#### 覆盖数
- **定义**: N(F, ε, d) — ε-覆盖的最小集合大小
- **度量**: L₂距离、L∞距离
- **关系**: 与Rademacher复杂度相关
- **Dudley熵积分**: 覆盖数的积分控制复杂度

#### 填充数
- **定义**: M(F, ε, d) — ε-分离的最大集合大小
- **关系**: M(F, 2ε) ≤ N(F, ε) ≤ M(F, ε)

### 一致稳定性

#### 定义
- **稳定性**: 改变一个样本对输出的影响
- **均匀稳定性**: sup |ℓ(f_S, z) - ℓ(f_{Sⁱ}, z)| ≤ β

- **假设稳定性**: 期望意义下的稳定性

#### 泛化界
- **稳定性→泛化**: 稳定算法泛化好
- **正则化ERM**: Tikhonov正则化保证稳定性
- **与复杂度**: 稳定性与假设类复杂度相关

## 现代泛化理论

### 过参数化现象

#### 双重下降
- **经典U型**: 欠拟合→最优→过拟合
- **现代观察**: 过参数化后测试误差再次下降
- **插值阈值**: 能完美拟合训练集的参数数量
- **良性过拟合**: 过参数化反而帮助泛化

#### 过参数化神经网络
- **参数数量**: 远大于样本数
- **零训练误差**: 仍能泛化良好
- **挑战经典理论**: VC维失效
- **隐式正则化**: 优化算法的隐式偏好

### 神经正切核 (NTK) 理论

#### NTK定义
- **无限宽极限**: 神经网络→核方法
- **NTK**: Θ(x,x') = E[∂f/∂θ · ∂f/∂θ]
- **常数核**: 训练过程中近似不变
- **线性化**: 网络在初始化附近线性

#### NTK下的学习
- **等价于核回归**: 用NTK的核回归
- **泛化分析**: 通过核方法分析
- **收敛性**: 过参数化下梯度下降收敛
- **局限性**: 实际网络可能不在NTK区域

### 隐式正则化

#### 优化算法偏好
- **梯度下降**: 倾向于最小L₂范数解
- **SGD**: 引入噪声，偏好平坦极小值
- **早停**: 防止过拟合的正则化
- **初始化**: 影响最终解

#### 平坦极小值
- **Hessian特征值**: 曲率度量
- **锐度**: 损失曲面尖锐程度
- **泛化关联**: 平坦极小值泛化更好
- **SAM优化器**: 显式寻找平坦极小值

#### 边际理论
- **间隔**: 分类决策边界的距离
- **大间隔**: 更好的泛化
- **SVM**: 显式最大化间隔
- **神经网络**: 隐式最大化间隔

### PAC-Bayes理论

#### 框架
- **先验**: 假设的先验分布
- **后验**: 数据依赖的分布
- **KL散度**: D_{KL}(Q‖P)

#### PAC-Bayes界
- **形式**: 以高概率对所有后验Q
  - E_{f~Q}[R(f)] ≤ E_{f~Q}[R̂(f)] + √(KL(Q‖P)+log(1/δ)/2n)
- **紧致性**: 比VC界通常更紧
- **应用**: 随机神经网络、集成方法

### 信息论泛化界

#### 互信息界
- **输入-输出互信息**: I(S; A(S))
- **泛化界**: |E[R - R̂]| ≤ √(2I(S;A(S))/n)

- **解释**: 算法输出与数据的依赖程度

#### 条件互信息
- **个体样本**: I(Zᵢ; A(S)|Z⁻ⁱ)

- **更紧的界**: 利用样本间的结构
- **CMI框架**: 条件互信息方法

### 基于压缩的理论

#### 压缩方法
- **核心思想**: 可压缩的模型泛化好
- **压缩界**: 泛化误差与压缩率相关
- **信息瓶颈**: 保留相关信息，压缩表示

#### 彩票假说
- **发现**: 随机初始化网络中存在小子网络
- **特性**: 单独训练可达原网络性能
- **意义**: 参数冗余性、结构重要性
- **泛化含义**: 有效复杂度可能远小于参数数

## 深度学习泛化

### 深度网络的挑战
- **参数爆炸**: 百万至十亿级参数
- **非凸优化**: 多个局部最优
- **非稳定性**: 梯度消失/爆炸
- **理论缺口**: 经典理论失效

### 实验观察

#### 泛化差距
- **定义**: 训练误差 - 测试误差
- **影响因素**:
  - 批量大小
  - 学习率
  - 正则化强度
  - 网络深度/宽度

#### 深度与泛化
- **深度优势**: 表示能力提升
- **优化困难**: 深层更难训练
- **残差连接**: 缓解梯度问题
- **宽度vs深度**: 权衡关系

### 正则化技术理论

#### Dropout理论
- **集成解释**: 训练指数级子网络
- **正则化效果**: 防止共适应
- **贝叶斯解释**: 近似贝叶斯推断
- **与L2**: 类似权重衰减效果

#### 批归一化理论
- **协变量偏移**: 减少内部协变量偏移
- **平滑效果**: 损失曲面更平滑
- **正则化**: 引入噪声
- **学习率**: 允许更大学习率

#### 数据增强理论
- **隐式正则化**: 扩大有效训练集
- **不变性学习**: 学习不变特征
- **Vicinal Risk Minimization**: 邻域风险最小化
- **Mixup**: 线性插值的理论分析

### 优化与泛化

#### 批量大小
- **大批量**: 尖锐极小值，泛化差
- **小批量**: 平坦极小值，泛化好
- **比例规则**: 学习率与批量大小成比例
- **临界批量**: 计算效率与泛化的平衡

#### 学习率
- **学习率调度**: warmup, decay, cyclical
- **学习率与泛化**: 影响收敛点
- **边缘锐度**: 与学习率相关
- **学习率范围测试**: 寻找最优范围

#### 早停
- **验证监控**: 在验证集最优时停止
- **正则化效果**: 限制模型复杂度
- **等价性**: 与L2正则化的关系
- **最优迭代**: 存在最优停止点

## 高级主题

### 对抗鲁棒性与泛化
- **对抗样本**: 微小扰动导致错误
- **鲁棒泛化**: 对抗攻击下的泛化
- **鲁棒性-准确性权衡**: 两者间的矛盾
- **理论**: 对抗Rademacher复杂度

### 领域自适应
- **问题设定**: 训练与测试分布不同
- **距离度量**: H-divergence, discrepancy
- **泛化界**: 涉及领域间距离
- **自适应方法**: 领域对抗训练

### 迁移学习理论
- **源域-目标域**: 知识迁移
- **泛化界**: 涉及源域性能+领域差异
- **特征重用**: 预训练特征的有效性
- **微调理论**: 何时微调更好

### 元学习泛化
- **任务分布**: 从任务分布学习
- **双层优化**: 内外循环
- **泛化**: 对新任务的适应能力
- **理论**: 任务多样性与泛化

### 神经正切特征 (NTF)
- **有限网络**: 与NTK的关系
- **特征学习**: 训练过程中的变化
- **泛化改进**: 特征学习的好处
- **理论**: 均值场极限
