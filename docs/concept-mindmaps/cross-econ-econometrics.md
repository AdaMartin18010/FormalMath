# 数学×经济学：计量经济学的统计推断

## 概述

计量经济学运用统计学和数学方法对经济理论进行实证检验和经济关系进行定量分析。从经典线性回归到现代时间序列分析，从面板数据到因果推断，计量方法为经济学研究提供了严谨的经验基础。

## 因果推断策略

| 方法 | 识别假设 | 适用场景 | 估计方法 |
|------|----------|----------|----------|
| 工具变量 | 相关性、外生性、排他性 | 遗漏变量、测量误差 | 2SLS, GMM |
| 双重差分 | 平行趋势、无预期效应 | 政策评估 | DID估计量 |
| 断点回归 | 局部连续性 | 门槛政策 | 局部线性回归 |
| 匹配 | 条件独立、共同支撑 | 观测研究 | 倾向得分匹配 |
| 合成控制 | 可预测性、权重凸性 | 单一处理单元 | 合成控制法 |

## 现代计量挑战

- **高维数据**: 变量选择、正则化方法
- **网络数据**: 社会交互、同伴效应
- **大数据**: 算法可扩展性、近似推断
- **因果ML**: 异质处理效应、因果森林
- **结构估计**: 动态离散选择、博弈估计
