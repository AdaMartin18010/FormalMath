# 深度学习数学 - 思维导图

## 神经网络基础

### 神经元模型
- **生物启发**: 简化神经元模型
- **数学表达**: y = σ(wᵀx + b)
- **组成部分**:
  - 输入: x ∈ ℝᵈ
  - 权重: w ∈ ℝᵈ
  - 偏置: b ∈ ℝ
  - 激活: σ(·)

### 前馈神经网络
- **层结构**: 输入层 → 隐藏层 → 输出层
- **全连接**: 每层神经元与下层全连接
- **深度**: 隐藏层数量
- **宽度**: 每层神经元数量
- **万能近似定理**: 单隐藏层可逼近任意连续函数

### 前向传播
- **逐层计算**: h⁽ˡ⁺¹⁾ = σ(W⁽ˡ⁾h⁽ˡ⁾ + b⁽ˡ⁾)
- **矩阵形式**: 批量并行计算
- **输出层**:
  - 回归: 线性激活
  - 二分类: Sigmoid
  - 多分类: Softmax

## 激活函数

### Sigmoid
- **公式**: σ(x) = 1/(1 + e⁻ˣ)
- **输出范围**: (0, 1)
- **优点**: 平滑可导，概率解释
- **缺点**: 梯度消失，非零均值

### Tanh
- **公式**: tanh(x) = (eˣ - e⁻ˣ)/(eˣ + e⁻ˣ)
- **输出范围**: (-1, 1)
- **优点**: 零均值，比Sigmoid梯度强
- **缺点**: 仍有梯度消失问题

### ReLU
- **公式**: ReLU(x) = max(0, x)
- **优点**: 计算简单，缓解梯度消失
- **缺点**: 神经元死亡问题
- **变体**:
  - Leaky ReLU: max(αx, x)
  - PReLU: 可学习α
  - ELU: 平滑负半轴

### Softmax
- **公式**: σ(z)ⱼ = eᶻʲ / Σₖ eᶻᵏ
- **性质**: 输出概率分布
- **数值稳定性**: 减去最大值
- **交叉熵**: 标准配合损失

### 其他激活函数
- **Swish**: x · σ(x)
- **GELU**: 高斯误差线性单元
- **Mish**: 自门控激活
- **选择原则**: 任务、网络深度、实验验证

## 反向传播算法

### 链式法则
- **单变量**: dy/dx = dy/du · du/dx
- **多变量**: 偏导数累加
- **向量化**: Jacobian矩阵
- **计算图**: 自动微分基础

### 反向传播过程
- **前向**: 计算并保存中间值
- **反向**: 从输出向输入传播梯度
- **参数更新**: θ ← θ - η·∂L/∂θ
- **复杂度**: 与前向传播同阶

### 梯度计算
- **损失梯度**: ∂L/∂y
- **权重梯度**: ∂L/∂W = ∂L/∂y · ∂y/∂W
- **激活梯度**: ∂L/∂x = ∂L/∂y · σ'(z)
- **层间传播**: δ⁽ˡ⁾ = (W⁽ˡ⁾)ᵀδ⁽ˡ⁺¹⁾ ⊙ σ'(z⁽ˡ⁾)

## 损失函数

### 回归损失
- **MSE**: 均方误差，L2损失
- **MAE**: 平均绝对误差，L1损失
- **Huber损失**: MSE与MAE的平滑组合
- **对数cosh**: 类似MSE但更平滑

### 分类损失
- **交叉熵**: -Σyᵢ log(ŷᵢ)
- **二元交叉熵**: 二分类专用
- **多分类交叉熵**: Softmax配合
- **Focal Loss**: 处理类别不平衡

### 高级损失函数
- **对比损失**: 度量学习
- **Triplet Loss**: 三元组学习
- **Center Loss**: 增大类间距离
- **ArcFace**: 加性角度边距

## 优化算法

### 梯度下降变体
- **批量GD**: 全数据集，稳定但慢
- **SGD**: 单样本，快但噪声大
- **小批量GD**: 折中方案，最常用
- **动量**: 累积历史梯度方向

### 自适应学习率
- **AdaGrad**: 累积平方梯度
- **RMSprop**: 指数移动平均
- **Adam**: 动量 + RMSprop
- **AdamW**: 解耦权重衰减

### 高级优化器
- **Nadam**: Nesterov + Adam
- **AMSGrad**: 修正Adam收敛性
- **LARS/LAMB**: 分层学习率
- **二阶方法**: L-BFGS，自然梯度

## 正则化技术

### 权重正则化
- **L1正则**: 稀疏性，特征选择
- **L2正则**: 权重衰减，平滑
- **弹性网络**: L1 + L2组合
- **谱正则**: 控制谱范数

### Dropout
- **训练**: 随机丢弃神经元
- **测试**: 缩放或MC采样
- **解释**: 模型集成，稀疏性
- **变体**:
  - DropConnect: 丢弃连接
  - Spatial Dropout: 空间维度
  - DropBlock: 结构化丢弃

### 批归一化 (BatchNorm)
- **操作**: 标准化每批数据
- **公式**: y = γ(x - μ)/√(σ² + ε) + β
- **效果**: 加速训练，稳定分布
- **位置**: 激活前或激活后

### 其他归一化
- **层归一化 (LayerNorm)**: 每样本标准化
- **实例归一化**: 图像风格迁移
- **组归一化**: 通道分组
- **权重归一化**: 权重标准化

### 数据增强
- **几何变换**: 旋转、翻转、裁剪
- **颜色变换**: 亮度、对比度、饱和度
- **Mixup/CutMix**: 样本插值
- **AutoAugment**: 学习增强策略

## 架构设计

### 卷积神经网络 (CNN)
- **卷积操作**: 局部连接，权值共享
- **关键层**:
  - 卷积层: 特征提取
  - 池化层: 降维，平移不变
  - 全连接层: 分类决策
- **经典架构**:
  - LeNet → AlexNet → VGG
  - ResNet → DenseNet → EfficientNet

### 循环神经网络 (RNN)
- **序列建模**: hₜ = σ(Wₓxₜ + Wₕhₜ₋₁ + b)
- **变体**:
  - LSTM: 门控机制，解决长程依赖
  - GRU: 简化LSTM
  - 双向RNN: 利用未来信息
- **应用**: NLP、时间序列、语音

### 注意力机制
- **核心思想**: 动态加权输入
- **Self-Attention**: 序列内元素间关系
- **多头注意力**: 多子空间并行
- **Transformer**: 全注意力架构
  - 编码器-解码器结构
  - 位置编码
  - 层归一化+残差

### 生成模型
- **VAE**: 变分自编码器
  - 编码器: q(z|x)
  - 解码器: p(x|z)

  - 损失: 重构 + KL散度
- **GAN**: 生成对抗网络
  - 生成器: 伪造数据
  - 判别器: 真假鉴别
  -  minimax博弈
- **Diffusion模型**: 渐进去噪
  - 前向: 加噪过程
  - 反向: 去噪过程

## 深度学习的数学理论

### 表达能力
- **万能近似定理**: 有限宽网络的能力
- **深度分离**: 深度vs宽度的优势
- **函数空间**: 神经网络表示的函数类
- **Barron空间**: 特定激活函数的理论

### 优化理论
- **非凸性**: 损失景观复杂
- **局部极小值**: 高维中并非问题
- **鞍点**: 主要障碍
- **收敛性**: 过参数化下的收敛

### 泛化理论
- **经典理论**: 容量控制失效
- **隐式正则化**: SGD的隐式偏好
- **PAC-Bayes**: 贝叶斯视角
- **双重下降**: 过参数化现象

### 神经正切核 (NTK)
- **宽网络极限**: 神经网络→核方法
- **训练动态**: 梯度下降等价于核回归
- **线性化**: 网络在初始化附近线性
- **泛化分析**: 通过NTK理解泛化
