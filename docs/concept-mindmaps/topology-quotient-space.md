# 商空间思维导图

## 概述
商空间是通过等价关系将拓扑空间中的点"粘合"在一起得到的新空间，是构造复杂拓扑空间的重要方法。

## 常见商空间构造

| 构造 | 原始空间 | 等价关系 | 结果 |
|------|---------|---------|------|
| **圆柱面** | [0,1]×[0,1] | (0,t)∼(1,t) | S¹×[0,1] |
| **莫比乌斯带** | [0,1]×[0,1] | (0,t)∼(1,1-t) | 不可定向带 |
| **环面** | [0,1]×[0,1] | 对边分别叠合 | S¹×S¹ |
| **克莱因瓶** | [0,1]×[0,1] | 一对扭转叠合 | 不可定向闭曲面 |
| **实射影平面** | S² | 对径点叠合 | RP² |

## 商映射的性质

```

商映射 f: X → Y
- 满射
- V ⊂ Y 开 ⇔ f⁻¹(V) ⊂ X 开
- 连续的满开（或满闭）映射是商映射

```
