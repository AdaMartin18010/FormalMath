# 线性方程组求解 思维导图

## 一、直接法 (Direct Methods)

### 1.1 Gauss消去法
- **基本消去**
  - 前向消去
  - 回代求解
  - 计算复杂度: O(n³)
- **主元选取**
  - 列主元
  - 全主元
  - 数值稳定性

### 1.2 LU分解
- **Doolittle分解**
  - L单位下三角
  - U上三角
- **Crout分解**
  - U单位上三角
  - L下三角
- **Cholesky分解**
  - 对称正定矩阵
  - A = LLᵀ
  - 计算复杂度: O(n³/3)

### 1.3 特殊矩阵
- **三对角矩阵**
  - 追赶法
  - O(n)复杂度
- **带状矩阵**
  - 稀疏存储
- **对称矩阵**
  - 对称消去

## 二、迭代法 (Iterative Methods)

### 2.1 经典迭代法
- **Jacobi迭代**
  - 分量形式
  - 矩阵形式: x⁽ᵏ⁺¹⁾ = D⁻¹(b - (L+U)x⁽ᵏ⁾)
  - 收敛条件: 严格对角占优
- **Gauss-Seidel迭代**
  - 分量形式
  - 矩阵形式: x⁽ᵏ⁺¹⁾ = (D-L)⁻¹(b + Ux⁽ᵏ⁾)
  - SOR加速
- **SOR方法**
  - 松弛因子 ω
  - 最优松弛因子

### 2.2 收敛性分析
- **迭代矩阵谱半径**
  - ρ(B) < 1 收敛
- **对角占优矩阵**
  - 严格对角占优
  - 不可约对角占优
- **对称正定矩阵**
  - Gauss-Seidel收敛

### 2.3 共轭梯度法 (CG)
- **算法原理**
  - Krylov子空间
  - 共轭方向
- **收敛性**
  - 最多n步收敛
  - 误差估计
- **预处理CG**
  - Jacobi预处理
  - 不完全Cholesky

## 三、Krylov子空间方法

### 3.1 Arnoldi迭代
- **Arnoldi过程**
  - Hessenberg矩阵
  - 正交基构造
- **GMRES**
  - 最小残差法
  - 重启策略

### 3.2 Lanczos迭代
- **对称矩阵**
  - 三对角化
- **MINRES**
  - 对称不定系统

### 3.3 双正交化方法
- **BiCG**
  - 双共轭梯度
- **BiCGSTAB**
  - 稳定化版本
- **QMR**
  - 拟最小残差

## 四、预处理技术

### 4.1 预处理目标
- **改善谱分布**
- **加速收敛**

### 4.2 预处理方法
- **Jacobi预处理**
  - 对角预处理
- **不完全LU分解 (ILU)**
  - ILU(0)
  - ILU(k)
- **不完全Cholesky (IC)**
  - 对称正定
- **多重网格**
  - 粗网格校正

## 五、最小二乘问题

### 5.1 正规方程
- **法方程**
  - AᵀAx = Aᵀb
  - 条件数平方

### 5.2 QR分解
- **Householder变换**
  - 反射矩阵
- **Givens旋转**
  - 平面旋转
- **Gram-Schmidt**
  - 经典/修正

### 5.3 SVD方法
- **奇异值分解**
  - A = UΣVᵀ
- **伪逆**
  - A⁺ = VΣ⁺Uᵀ
- **秩亏问题**
