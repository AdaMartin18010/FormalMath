# 马尔可夫链 - 思维导图

## 概述

马尔可夫链(Markov Chain)是以Andrey Markov命名的随机过程，具有"无记忆性"的核心特征：给定现在，未来与过去独立。这一特性使其成为建模随机系统演化的强大工具，在物理、化学、生物、计算机科学、经济学等领域有广泛应用。

## 重要定理汇总

| 定理名称 | 条件 | 结论 |
|----------|------|------|
| **Chapman-Kolmogorov** | 任意时齐MC | $p_{ij}^{(m+n)} = \sum_k p_{ik}^{(m)} p_{kj}^{(n)}$ |
| **常返性判别** | 不可约链 | $\sum_n p_{ii}^{(n)} = \infty$ ⇔ 常返 |
| **遍历定理** | 不可约遍历 | $\lim_{n\to\infty} p_{ij}^{(n)} = \pi_j$ |
| **强大数定律** | 不可约正常返 | $\frac{1}{n}\sum_{k=1}^n f(X_k) \to \sum_i \pi_i f(i)$ |
| **中心极限定理** | 遍历链 | 标准化部分和收敛于正态分布 |

## 关键公式速查

| 公式 | 说明 |
|------|------|
| $p_{ij} = P(X_{n+1}=j \mid X_n=i)$ | 一步转移概率 |
| $p_{ij}^{(n)} = (P^n)_{ij}$ | n步转移概率 |
| $p_{ij}^{(m+n)} = \sum_k p_{ik}^{(m)} p_{kj}^{(n)}$ | Chapman-Kolmogorov方程 |
| $\pi = \pi P$ | 平稳分布方程 |
| $\pi_j = \lim_{n\to\infty} p_{ij}^{(n)}$ | 极限分布(遍历链) |
| $\pi_j = 1/\mu_j$ | 平稳概率与平均返回时间 |
| $f_{ij} = P(\text{从}i\text{最终到达}j)$ | 首达概率 |
| $\mu_i = \sum_n n f_{ii}^{(n)}$ | 平均返回时间 |
