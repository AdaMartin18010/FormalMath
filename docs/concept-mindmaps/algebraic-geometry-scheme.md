# 概形思维导图

## 概述
概形是代数簇概念的推广，由Grothendieck引入，允许定义在任意交换环上，包含"几何点"和"算术点"，是现代代数几何的基础语言。

## 概形的层次结构

```

概形
 ├── 局部诺特概形
 │    ├── 整概形
 │    │    └── 代数簇（几何整+分离+有限型）
 │    └── 约化概形
 │         └── 代数簇（古典定义）
 └── 仿射概形 (=Spec A)

```

## 重要的概形例子

| 概形 | 描述 | 特点 |
|------|------|------|
| **Spec ℤ** | 整数环的谱 | 算术曲线的"点" |
| **Spec k** | 域的谱 | 单点 |
| **Spec k[ε]/(ε²)** | 对偶数 | 幂零、非约化 |
| **Aⁿ_k** | 仿射空间 | 几何直观 |
| **Pⁿ_k** | 射影空间 | 完备紧化 |

## 概形 vs 代数簇

| 特性 | 代数簇 | 概形 |
|------|--------|------|
| 基环 | 代数闭域 | 任意交换环 |
| 函数 | 多项式环 | 任意环 |
| 幂零 | 无 | 允许 |
| 泛性质 | 弱 | 强（纤维积等） |
