# 优化算法 - 思维导图

## 优化基础

### 优化问题形式
- **一般形式**: min_{x∈X} f(x)
- **无约束优化**: X = ℝⁿ
- **约束优化**: X = {x | gᵢ(x) ≤ 0, hⱼ(x) = 0}
- **机器学习**: 损失函数最小化

### 问题分类
- **凸优化**: 凸函数 + 凸集
  - 局部最优 = 全局最优
  - 多项式时间可解
- **非凸优化**: 存在多个局部最优
  - 一般NP难
  - 深度学习的挑战
- **光滑性**: Lipschitz梯度、Hessian

### 收敛性概念
- **收敛速率**: 迭代次数与精度的关系
- **线性收敛**: ‖xₖ - x*‖ ≤ ρᵏ‖x₀ - x*‖
- **次线性收敛**: O(1/k), O(1/√k)
- **超线性/二次收敛**: 更快收敛

## 梯度方法

### 梯度下降 (GD)
- **更新规则**: xₖ₊₁ = xₖ - η∇f(xₖ)
- **步长选择**:
  - 固定步长
  - 精确线搜索
  - 回溯线搜索 (Armijo条件)
- **收敛性**:
  - 凸函数: O(1/k)
  - 强凸函数: O(ρᵏ)

### 随机梯度下降 (SGD)
- **更新**: xₖ₊₁ = xₖ - ηₖ∇fᵢ(xₖ)
- **批量vs随机**:
  - GD: 全梯度，稳定但慢
  - SGD: 单样本，快但噪声大
- **学习率衰减**: ηₖ = O(1/k) 或 O(1/√k)
- **收敛**: 凸函数 O(1/√k)，强凸 O(1/k)

### 小批量梯度下降
- **折中方案**: 批量大小 b
- **方差缩减**: σ²/b
- **并行化**: 利用硬件加速
- **批量大小选择**: 内存、计算、泛化权衡

### 加速梯度方法

#### Nesterov加速梯度 (NAG)
- **更新**:
  - yₖ = xₖ + β(xₖ - xₖ₋₁)
  - xₖ₊₁ = yₖ - η∇f(yₖ)
- **直觉**: 预测下一步位置
- **收敛**: 凸函数 O(1/k²) — 最优
- **动量解释**: 加速收敛，抑制震荡

#### 动量方法
- **更新**:
  - vₖ₊₁ = βvₖ + ∇f(xₖ)
  - xₖ₊₁ = xₖ - ηvₖ₊₁
- **物理类比**: 质量、速度、摩擦力
- **效果**: 加速相关方向，抑制震荡
- **参数**: β ≈ 0.9

## 自适应方法

### AdaGrad
- **思想**: 自适应调整每个参数的学习率
- **累积平方梯度**: rₖ = rₖ₋₁ + ∇f(xₖ) ⊙ ∇f(xₖ)
- **更新**: xₖ₊₁ = xₖ - (η/√(rₖ+ε)) ⊙ ∇f(xₖ)
- **优点**: 适合稀疏梯度
- **缺点**: 学习率单调递减，可能过早停止

### RMSprop
- **改进**: 指数移动平均
- **更新**: rₖ = ρrₖ₋₁ + (1-ρ)∇f(xₖ) ⊙ ∇f(xₖ)
- **优点**: 解决AdaGrad学习率问题
- **参数**: ρ = 0.9, η = 0.001

### Adam (Adaptive Moment Estimation)
- **一阶矩估计**: mₖ = β₁mₖ₋₁ + (1-β₁)∇f(xₖ)
- **二阶矩估计**: vₖ = β₂vₖ₋₁ + (1-β₂)∇f(xₖ)²
- **偏差修正**:
  - m̂ₖ = mₖ / (1-β₁ᵏ)
  - v̂ₖ = vₖ / (1-β₂ᵏ)
- **更新**: xₖ₊₁ = xₖ - η·m̂ₖ/(√v̂ₖ+ε)
- **默认**: β₁=0.9, β₂=0.999, η=0.001

### Adam变种
- **AdaMax**: L∞范数替代L2
- **Nadam**: Nesterov + Adam
- **AMSGrad**: 修正Adam收敛性
  - v̂ₖ = max(v̂ₖ₋₁, vₖ)
- **AdamW**: 解耦权重衰减
  - 权重衰减不作用于梯度

## 二阶方法

### 牛顿法
- **更新**: xₖ₊₁ = xₖ - [∇²f(xₖ)]⁻¹∇f(xₖ)
- **二次收敛**: 接近最优时极快
- **问题**:
  - 计算Hessian: O(n²)存储
  - 求逆: O(n³)计算
  - 可能非正定
- **修正**: 信赖域、线搜索

### 拟牛顿法
- **思想**: 近似Hessian或其逆
- **BFGS**: 最流行的拟牛顿法
  - 秩二更新
  - 保持正定性
- **L-BFGS**: 有限内存BFGS
  - 只保存最近m步信息
  - O(mn)存储和计算
- **适用**: 中小规模问题

### 自然梯度
- **信息几何**: 参数空间的黎曼度量
- **Fisher信息矩阵**: G(θ) = E[∇log p ∇log pᵀ]
- **自然梯度**: ∇̃f = G⁻¹∇f
- **不变性**: 参数化不变
- **KFAC**: 神经网络中的近似

## 约束优化

### 拉格朗日方法
- **拉格朗日函数**: L(x,λ,ν) = f(x) + Σλᵢgᵢ(x) + Σνⱼhⱼ(x)
- **KKT条件**:
  - 平稳性: ∇f + Σλᵢ∇gᵢ + Σνⱼ∇hⱼ = 0
  - 原始可行: gᵢ(x) ≤ 0, hⱼ(x) = 0
  - 对偶可行: λᵢ ≥ 0
  - 互补松弛: λᵢgᵢ(x) = 0

### 投影梯度法
- **更新**: xₖ₊₁ = Proj_X(xₖ - η∇f(xₖ))
- **投影**: 到可行集的最短距离
- **例子**: 非负约束、盒约束
- **收敛**: 类似梯度下降

### 罚函数法
- **外点法**: 违反约束时惩罚
  - P(x,μ) = f(x) + μΣ[max(0,gᵢ(x))]²
- **内点法**: 在可行域内，接近边界时惩罚
  - 障碍函数: -μΣlog(-gᵢ(x))
- **增广拉格朗日**: 结合乘子与惩罚

### 对偶方法
- **拉格朗日对偶**: max_λ min_x L(x,λ)
- **对偶问题**: 凸优化，即使原问题非凸
- **弱对偶**: d* ≤ p*
- **强对偶**: d* = p* (凸问题+约束规范)
- **KKT条件**: 强对偶下的最优条件

### 原始-对偶方法
- **同时更新**: 原始变量x和对偶变量λ
- **ADMM**: 交替方向乘子法
  - 分解问题
  - 适用于分布式优化
- **Mirror Descent**: 基于Bregman散度

## 非凸优化

### 局部最优
- **一阶条件**: ∇f(x) = 0
- **二阶条件**: ∇²f(x) ⪰ 0
- **鞍点**: ∇f = 0但Hessian不定
- **逃离鞍点**: 扰动、二阶信息

### 深度学习的优化景观
- **局部极小值**: 高维中并非主要问题
- **鞍点**: 主要障碍
- **平坦极小值**: 与泛化相关
- **对称性**: 导致等价的多个解

### 扰动梯度下降
- **PGD**: 添加随机扰动
- **逃离鞍点**: 以高概率逃离严格鞍点
- **收敛**: 收敛到二阶稳定点

### 二阶方法在非凸中
- **信赖域**: 限制步长
- **立方正则化**: 三阶模型
- **Hessian-free**: 矩阵-向量乘积
- **曲率利用**: 加速收敛

## 方差缩减方法

### SAG (Stochastic Average Gradient)
- **存储**: 每个样本的梯度
- **更新**: 使用所有历史梯度的平均
- **收敛**: 线性收敛 for 强凸
- **存储**: O(n)梯度存储

### SVRG (Stochastic Variance Reduced Gradient)
- **双循环**:
  - 外循环: 计算全梯度
  - 内循环: 方差缩减随机更新
- **修正**: ∇fᵢ(x) - ∇fᵢ(x̃) + ∇f(x̃)
- **无偏**: E[修正项] = ∇f(x)
- **方差**: 当x→x*时→0

### SAGA
- **SAG + SVRG**: 结合两者优点
- **增量更新**: 不需要双循环
- **无偏**: 天然无偏
- **适用**: 更广泛的场景

### Katyusha
- **加速方差缩减**: Nesterov + 方差缩减
- **收敛**: 最优的oracle复杂度
- **特点**: 负动量，防止震荡

## 特殊问题结构

### 复合优化
- **形式**: min f(x) + g(x)
- **f**: 光滑，易求梯度
- **g**: 可能非光滑，有 prox
- **例子**: Lasso, 稀疏优化

### 近端梯度法
- **proximal operator**: prox_{ηg}(v) = argmin_x {g(x) + (1/2η)‖x-v‖²}
- **近端梯度更新**:
  - xₖ₊₁ = prox_{ηg}(xₖ - η∇f(xₖ))
- **ISTA**: 梯度下降 + 软阈值
- **FISTA**: Nesterov加速ISTA

### 坐标下降
- **思想**: 每次只更新一个坐标
- **更新**: xᵢ⁽ᵏ⁺¹⁾ = argmin_{xᵢ} f(x₁⁽ᵏ⁺¹⁾, ..., xᵢ, ..., xₙ⁽ᵏ⁾)
- **Gauss-Seidel**: 使用最新值
- **适用**: 坐标可分离或近似可分离
- **加速**: 随机坐标下降、加速坐标下降

### 随机优化
- **有限和**: f(x) = (1/n)Σfᵢ(x)
- **在线优化**: 数据流式到达
- **后悔界**: Regret = Σfₜ(xₜ) - min Σfₜ(x)
- **算法**: Online GD, FTRL, Mirror Descent

## 优化理论

### 凸分析基础
- **凸函数**: f(λx+(1-λ)y) ≤ λf(x)+(1-λ)f(y)
- **次梯度**: 凸函数在不可导点的推广
- **共轭函数**: f*(y) = sup_x {yᵀx - f(x)}
- **对偶性**: 原问题与对偶问题

### 光滑性与强凸性
- **L-光滑**: ‖∇f(x) - ∇f(y)‖ ≤ L‖x - y‖
  - 等价: f(y) ≤ f(x) + ∇f(x)ᵀ(y-x) + (L/2)‖y-x‖²
- **μ-强凸**: f(y) ≥ f(x) + ∇f(x)ᵀ(y-x) + (μ/2)‖y-x‖²
- **条件数**: κ = L/μ
- **收敛依赖**: 条件数决定收敛速度

### 收敛分析
- **Lyapunov函数**: 证明收敛的工具
- **势能方法**: 分析复杂度
- **连续极限**: ODE视角理解优化
- **离散化**: 分析数值方案

### 下界理论
- **Oracle复杂度**: 查询下界
- **Nemirovski-Yudin**: 一阶方法下界
- **最优算法**: 达到下界的算法
- **加速必要性**: 证明加速的优越性
