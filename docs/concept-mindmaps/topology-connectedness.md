# 连通性思维导图

## 概述
连通性描述拓扑空间"不可分离"的性质，是拓扑空间的基本分类标准之一。

## 思维导图

```mermaid
mindmap
  root((连通性<br/>Connectedness))
    基本连通
      定义
        不能分割为两个非空开集
        只有平凡既开又闭集
      等价刻画
        不存在连续满射到离散两点集
        任意连续映射到离散空间为常值
      保持性
        连续像保持连通
        连通子集的并保持连通
    道路连通
      定义
        任意两点有道路连接
        连续映射γ:[0,1]→X
      与连通的关系
        道路连通⇒连通
        连通⇏道路连通
        拓扑学家的正弦曲线
      道路分支
        极大道路连通子集
        道路分支的交为空
    局部连通
      定义
        每点有连通邻域基
      局部道路连通
        每点有道路连通邻域基
      性质
        连通+局部道路⇒道路连通
    分支结构
      连通分支
        极大连通子集
        空间的分划
      分支的性质
        闭集
        两两不交
        并等于全空间
    高维连通
      单连通
        基本群平凡
        任意环路可缩
      n-连通
        低维同伦群平凡
        π₁,...,πₙ都平凡
      可缩空间
        同伦等价于点
        强于单连通
    应用与例子
      介值定理
        连通性的分析表述
      维数不变性
        Rⁿ与Rᵐ不同胚(n≠m)
      常见空间
        Sⁿ连通(n≥1)
        球面连通
        环面连通
    乘积空间
      连通性的积
        连通×连通⇒连通
      道路连通的积
        道路连通×道路⇒道路

```

## 连通性层次

```

可缩空间 ⇒ n-连通 ⇒ 单连通 ⇒ 道路连通 ⇒ 连通

```

## 核心例子

| 空间 | 连通性 | 道路连通性 | 单连通性 |
|------|--------|-----------|---------|
| ℝⁿ | ✓ | ✓ | ✓ |
| S¹ | ✓ | ✓ | ✗ |
| Sⁿ (n≥2) | ✓ | ✓ | ✓ |
| 拓扑学家的正弦曲线 | ✓ | ✗ | - |
