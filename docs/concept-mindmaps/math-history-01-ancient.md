## 详细内容

### 埃及数学

| 方面 | 内容 |
|------|------|
| **时间** | 公元前3000年 - 公元前300年 |
| **文献** | 莱茵德纸草书(约1650BC)、莫斯科纸草书(约1850BC) |
| **记数系统** | 十进制，象形数字符号 |
| **主要成就** | 单位分数运算、几何体积公式、线性方程求解 |

**关键数学家**：
- **阿默斯**：莱茵德纸草书抄写者
- **未知抄写员**：莫斯科纸草书作者

### 巴比伦数学

| 方面 | 内容 |
|------|------|
| **时间** | 公元前2000年 - 公元前500年 |
| **文献** | 普林顿322号泥板、YBC 7289泥板 |
| **记数系统** | 六十进制位置记数法 |
| **主要成就** | 二次方程、平方根近似、三角函数表 |

**关键成就**：
- √2的精确近似：1;24,51,10（≈1.414213）
- 毕达哥拉斯三元组的系统研究

## 历史意义

1. **证明的诞生**：从经验计算到逻辑证明
2. **公理化方法**：欧几里得建立数学范式
3. **抽象思维**：从具体问题到一般理论
4. **几何传统**：影响西方数学两千余年
