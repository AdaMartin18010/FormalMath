# 微分思维导图

## 概述

微分学研究函数的变化率，是分析学的核心工具。微分概念刻画了函数的局部线性特征，为研究函数性质、优化问题和物理定律提供了数学基础。

## 求导公式速查表

| 函数类型 | 公式 | 备注 |
|---------|------|------|
| 常数 | $(c)' = 0$ | $c \in \mathbb{R}$ |
| 幂函数 | $(x^a)' = ax^{a-1}$ | $a \in \mathbb{R}$ |
| 指数 | $(e^x)' = e^x$ | 不变性 |
| 指数(一般) | $(a^x)' = a^x \ln a$ | $a > 0$ |
| 对数 | $(\ln x)' = \frac{1}{x}$ | $x > 0$ |
| 对数(一般) | $(\log_a x)' = \frac{1}{x \ln a}$ | $a > 0, a \neq 1$ |
| 正弦 | $(\sin x)' = \cos x$ | - |
| 余弦 | $(\cos x)' = -\sin x$ | - |
| 正切 | $(\tan x)' = \sec^2 x$ | $x \neq \frac{\pi}{2} + k\pi$ |

---

## 求导法则

| 法则 | 公式 | 条件 |
|------|------|------|
| 线性性 | $(af + bg)' = af' + bg'$ | $a, b$ 为常数 |
| 乘法 | $(fg)' = f'g + fg'$ | 乘积法则 |
| 除法 | $(f/g)' = (f'g - fg')/g^2$ | $g \neq 0$ |
| 链式 | $(f(g(x)))' = f'(g(x)) \cdot g'(x)$ | 复合函数 |
| 反函数 | $(f^{-1})'(y) = 1/f'(x)$ | $y = f(x)$ |
| 隐函数 | $\frac{dy}{dx} = -\frac{F_x}{F_y}$ | $F(x,y) = 0$ |

## 极值判定方法

| 方法 | 第一充分条件 | 第二充分条件 |
|------|-------------|-------------|
| 条件 | $f'$ 在 $x_0$ 两侧变号 | $f'(x_0)=0$ 且 $f''(x_0) \neq 0$ |
| 极大值 | $f'$ 由正变负 | $f''(x_0) < 0$ |
| 极小值 | $f'$ 由负变正 | $f''(x_0) > 0$ |

## 与其他概念的联系

- **积分学**: 微积分基本定理揭示微分与积分的互逆关系
- **微分方程**: 微分是构建ODE和PDE的基础
- **优化理论**: 梯度是优化算法的核心
- **几何学**: 切空间、法向量等几何概念的基础
- **物理学**: 描述变化率和守恒定律
