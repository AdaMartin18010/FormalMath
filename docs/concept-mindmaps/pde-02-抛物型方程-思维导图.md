# 抛物型方程 - 思维导图

## 概述

抛物型偏微分方程描述扩散、输运和演化过程，其典型代表是热传导方程。这类方程具有时间不可逆性、解的光滑化效应和极值原理，在物理、金融、生物等领域有广泛应用。

## 关键公式速查

| 公式 | 名称 | 说明 |
|------|------|------|
| $K(x,t) = \frac{1}{(4\pi t)^{n/2}}e^{-|x|^2/4t}$ | 热核/基本解 | 高斯分布形式 |
| $u(x,t) = \int_{\mathbb{R}^n} K(x-y,t)u_0(y)dy$ | Poisson公式 | Cauchy问题解 |
| $u(t) = e^{t\Delta}u_0$ | 半群表示 | 算子形式 |
| $\|u(t)\|_{L^2} \leq \|u_0\|_{L^2}$ | 能量衰减 | L²收缩性 |
| $\sup_{\Omega_T} u \leq \sup_{\partial_p \Omega_T} u^+$ | 抛物极值原理 | 最大值估计 |
