## 核心要点

### 范数公理
**定义**: 映射 ‖·‖: X → [0,∞) 满足:
1. ‖x‖ = 0 ⇔ x = 0
2. ‖αx‖ = |α|‖x‖

3. ‖x+y‖ ≤ ‖x‖ + ‖y‖

### 范数等价
**定义**: 范数 ‖·‖₁ 与 ‖·‖₂ 等价，若 ∃c,C>0:
$$c\|x\|_1 \leq \|x\|_2 \leq C\|x\|_1$$

**定理**: 有限维空间上所有范数等价。

### Riesz引理
**定理**: 设 M 是赋范空间 X 的真闭子空间，则 ∀θ∈(0,1), ∃x∈X, ‖x‖=1:
$$\text{dist}(x, M) \geq \theta$$

### 重要空间

| 空间 | 范数 | 性质 |
|------|------|------|
| ℝⁿ | ‖x‖ₚ = (Σ\|xᵢ\|ᵖ)^(1/p) | 有限维 |
| lᵖ | ‖x‖ₚ = (Σ\|xₙ\|ᵖ)^(1/p) | 完备 |
| C[a,b] | ‖f‖∞ = sup\|f(x)\| | 完备 |
