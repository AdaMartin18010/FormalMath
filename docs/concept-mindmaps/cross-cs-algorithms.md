# 数学×计算机科学：算法分析的离散数学

## 概述

算法是计算机科学的核心，而算法的设计与分析深深植根于离散数学。从复杂度理论到图论，从组合优化到概率分析，数学工具为理解和改进算法提供了严谨的框架。

## 复杂度类层次

| 复杂度类 | 定义 | 典型问题 |
|----------|------|----------|
| P | 确定性图灵机多项式时间可解 | 排序、最短路径、矩阵乘法 |
| NP | 解可在多项式时间内验证 | 哈密顿回路、子集和 |
| NP完全 | NP中最难的问题 | SAT、3-SAT、旅行商 |
| co-NP | 补问题在NP中 | 素数判定(PRIMES∈P) |
| PSPACE | 多项式空间可解 | 量化布尔公式、游戏必胜策略 |
| EXPTIME | 指数时间可解 | 推广的围棋、国际象棋 |

## 主定理速查

```mermaid
mindmap
  root((主定理<br/>Master Theorem))
    递归形式
      T(n) = aT(n/b) + f(n)
      a ≥ 1, b > 1
      f(n)渐近正
    情况1
      f(n) = O(n^(log_b a - ε))
      T(n) = Θ(n^(log_b a))
      递归项主导
    情况2
      f(n) = Θ(n^(log_b a) log^k n)
      T(n) = Θ(n^(log_b a) log^(k+1) n))
      平衡情况
    情况3
      f(n) = Ω(n^(log_b a + ε))
      af(n/b) ≤ cf(n)
      T(n) = Θ(f(n))
      非递归项主导
    应用实例
      归并排序
        a=2,b=2,f(n)=O(n)
        T(n) = Θ(n log n)
      二分搜索
        a=1,b=2,f(n)=O(1)
        T(n) = Θ(log n)
      Strassen矩阵乘法
        a=7,b=2,f(n)=O(n²)
        T(n) = Θ(n^log₂7) ≈ O(n^2.81)

```

## 近似算法与不可近似性

- **顶点覆盖**: 2-近似存在，(2-ε)-近似假设UGC困难
- **集合覆盖**: ln(n)-近似最优，除非NP⊆DTIME(n^O(log log n))
- **旅行商**: 一般情况无有限近似，度量空间1.5-近似(Christofides)
- **最大割**: 0.878-近似(Goeman-Williamson)，假设UGC最优
