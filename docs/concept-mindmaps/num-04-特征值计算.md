# 特征值计算 思维导图

## 1. 基本理论与性质

### 1.1 特征值问题
- **标准特征值问题**
  - Ax = λx
  - 特征多项式
- **广义特征值问题**
  - Ax = λBx
  - 对称正定B

### 1.2 特征值性质
- **Gershgorin圆盘定理**
  - 特征值定位
- **谱分解**
  - 正规矩阵
  - A = QΛQᴴ
- **条件数**
  - 特征值敏感性

## 2. 幂法与反幂法

### 2.1 幂法 (Power Method)
- **算法原理**
  - 迭代: xₖ₊₁ = Axₖ/‖Axₖ‖
  - 收敛到主特征向量
- **收敛分析**
  - 收敛速度: |λ₂/λ₁|
  - 归一化处理
- **Rayleigh商**
  - λ ≈ xᴴAx/xᴴx

### 2.2 反幂法
- **算法原理**
  - 迭代: xₖ₊₁ = A⁻¹xₖ/‖A⁻¹xₖ‖
  - 收敛到最小特征值
- **位移反幂法**
  - (A - μI)⁻¹
  - 收敛到最近特征值

### 2.3 子空间迭代
- **块幂法**
- **正交化过程**

## 3. QR算法

### 3.1 基本QR算法
- **QR迭代**
  - Aₖ = QₖRₖ
  - Aₖ₊₁ = RₖQₖ
- **收敛性**
  - 对角收敛到特征值

### 3.2 实用QR算法
- **上Hessenberg化**
  - Householder变换
  - 保持结构
- **位移策略**
  - Rayleigh商位移
  - Wilkinson位移
- **隐式QR**
  - 双重位移
  - 实Schur型

### 3.3 对称QR算法
- **三对角化**
- **快速收敛**
- **特征向量计算**

## 4. 分而治之法

### 4.1 对称三对角矩阵
- **矩阵分割**
- **特征值合并**
- **计算复杂度**
  - O(n²)或O(n logᵖ n)

### 4.2 快速算法
- **Cuppen算法**
- **并行实现**

## 5. 二分法与对分法

### 5.1 Sturm序列
- **主子式序列**
- **符号变化计数**
- **特征值计数**

### 5.2 对分法
- **区间分割**
- **指定特征值**
- **高精度计算**

## 6. Lanczos与Arnoldi方法

### 6.1 Lanczos算法
- **对称矩阵**
  - 三对角化
  - 正交基构造
- **理论性质**
  - Ritz值收敛
- **实用考虑**
  - 重正交化
  - 部分重正交化

### 6.2 Arnoldi算法
- **非对称矩阵**
  - Hessenberg化
- **隐式重启Arnoldi (IRA)**
  - 滤除不需要的特征值
