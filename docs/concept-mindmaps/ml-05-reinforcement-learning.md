# 强化学习数学 - 思维导图

## 基础概念

### 强化学习框架
- **智能体 (Agent)**: 决策者、学习者
- **环境 (Environment)**: 智能体交互的外部世界
- **状态 (State)**: 环境在时刻t的描述 Sₜ
- **动作 (Action)**: 智能体在状态s下可执行的操作 Aₜ
- **奖励 (Reward)**: 环境对动作的反馈 Rₜ
- **目标**: 最大化累积奖励

### 马尔可夫决策过程 (MDP)
- **定义**: ⟨S, A, P, R, γ⟩
- **状态空间 S**: 所有可能的状态
- **动作空间 A**: 所有可能的动作
- **转移概率 P**: P(s'|s,a) — 动态模型

- **奖励函数 R**: R(s,a,s') 或 R(s,a)
- **折扣因子 γ**: [0,1]，平衡即时与远期奖励

### 马尔可夫性质
- **定义**: P(Sₜ₊₁|Sₜ, Aₜ) = P(Sₜ₊₁|S₁,...,Sₜ, Aₜ)

- **意义**: 未来只依赖于现在，与过去无关
- **完全可观测**: 状态包含所有相关信息
- **部分可观测 (POMDP)**: 状态不完全可见

## 策略与价值函数

### 策略 (Policy)
- **定义**: 状态到动作的映射
- **确定性策略**: a = π(s)
- **随机策略**: π(a|s) — 概率分布

- **目标**: 找到最优策略 π*

### 回报 (Return)
- **折扣回报**: Gₜ = Σₖ₌₀^∞ γᵏ Rₜ₊ₖ₊₁
- **有限时域**: Gₜ = Σₖ₌₀^T Rₜ₊ₖ₊₁
- **平均回报**: 长期平均奖励

### 价值函数

#### 状态价值函数 V(s)
- **定义**: Vᵅ(s) = Eᵅ[Gₜ | Sₜ = s]

- **含义**: 从状态s出发，遵循策略π的期望回报
- **贝尔曼方程**:
  - Vᵅ(s) = Σₐ π(a|s) Σₛ' P(s'|s,a)[R(s,a,s') + γVᵅ(s')]

#### 动作价值函数 Q(s,a)
- **定义**: Qᵅ(s,a) = Eᵅ[Gₜ | Sₜ = s, Aₜ = a]

- **含义**: 在状态s执行动作a后遵循策略π的期望回报
- **贝尔曼方程**:
  - Qᵅ(s,a) = Σₛ' P(s'|s,a)[R(s,a,s') + γΣₐ' π(a'|s')Qᵅ(s',a')]

#### 关系
- **V与Q**: Vᵅ(s) = Σₐ π(a|s)Qᵅ(s,a)
- **Q与V**: Qᵅ(s,a) = Σₛ' P(s'|s,a)[R(s,a,s') + γVᵅ(s')]

### 最优价值函数
- **最优状态价值**: V*(s) = maxᵅ Vᵅ(s)
- **最优动作价值**: Q*(s,a) = maxᵅ Qᵅ(s,a)
- **最优策略**: π* = argmaxᵅ Vᵅ(s)

## 贝尔曼方程

### 贝尔曼期望方程
- **状态价值**: Vᵅ = Rᵅ + γPᵅVᵅ
- **矩阵形式**: V = (I - γPᵅ)⁻¹Rᵅ
- **迭代求解**: 动态规划基础

### 贝尔曼最优方程
- **最优状态价值**:
  - V*(s) = maxₐ Σₛ' P(s'|s,a)[R(s,a,s') + γV*(s')]

- **最优动作价值**:
  - Q*(s,a) = Σₛ' P(s'|s,a)[R(s,a,s') + γ maxₐ' Q*(s',a')]

- **非线性**: max运算导致非线性

### 压缩映射
- **贝尔曼算子 T**: (TV)(s) = maxₐ ...
- **压缩性**: ‖TV₁ - TV₂‖∞ ≤ γ‖V₁ - V₂‖∞
- **Banach不动点定理**: 迭代收敛到唯一解
- **收敛速度**: 指数收敛

## 动态规划方法

### 策略迭代
- **策略评估**: 固定策略，计算Vᵅ
  - 迭代: Vₖ₊₁(s) = Σₐ π(a|s)Σₛ'P(s'|s,a)[R+γVₖ(s')]

  - 解析解: V = (I - γPᵅ)⁻¹Rᵅ
- **策略改进**: 贪心更新策略
  - π'(s) = argmaxₐ Qᵅ(s,a)
- **收敛**: 有限步收敛到最优

### 价值迭代
- **更新规则**:
  - Vₖ₊₁(s) = maxₐ Σₛ' P(s'|s,a)[R(s,a,s') + γVₖ(s')]

- **与策略迭代**: 每步都隐含策略改进
- **收敛**: 当‖Vₖ₊₁ - Vₖ‖ < ε时停止
- **提取策略**: π(s) = argmaxₐ ...

### 异步动态规划
- **in-place更新**: 立即使用新值
- **优先扫描**: 优先更新变化大的状态
- **实时动态规划**: 只访问相关状态

## 蒙特卡洛方法

### 基本思想
- **经验平均**: 用采样回报估计期望
- **无模型**: 不需要P和R
- **回合制**: 等待回合结束才更新

### 蒙特卡洛预测
- **首次访问**: 每回合每个状态只算第一次
- **每次访问**: 每次出现都计入
- **收敛性**: 大数定律保证

### 蒙特卡洛控制
- **ε-贪心探索**: 以ε概率随机，1-ε概率贪心
- **GLIE**: 极限无穷探索
- **蒙特卡洛ES**: 探索起点
- **离策略**: 重要性采样

### 重要性采样
- **目标**: 评估策略π，但用策略b采样
- **比率**: ρ = Π π(aₜ|sₜ)/b(aₜ|sₜ)

- **普通重要性采样**: 无偏但方差大
- **加权重要性采样**: 有偏但方差小

## 时序差分学习

### TD(0) - 单步TD
- **更新**: V(sₜ) ← V(sₜ) + α[Rₜ₊₁ + γV(sₜ₊₁) - V(sₜ)]
- **TD误差**: δ = R + γV(s') - V(s)
- **自举**: 用当前估计更新
- **对比**: MC(无偏，高方差) vs TD(有偏，低方差)

### SARSA (On-Policy)
- **更新**: Q(sₜ,aₜ) ← Q(sₜ,aₜ) + α[Rₜ₊₁ + γQ(sₜ₊₁,aₜ₊₁) - Q(sₜ,aₜ)]
- **五元组**: S,A,R,S',A' → SARSA
- **同策略**: 学习和采样用同一策略
- **收敛**: 适当学习率下收敛

### Q-Learning (Off-Policy)
- **更新**: Q(sₜ,aₜ) ← Q(sₜ,aₜ) + α[Rₜ₊₁ + γ maxₐ' Q(sₜ₊₁,a') - Q(sₜ,aₜ)]
- **离策略**: 学习最优策略，采样用探索策略
- **收敛性**: 保证收敛到Q*
- **探索-利用**: ε-贪心、UCB、Boltzmann

### 期望SARSA
- **更新**: 用期望替代采样
- Q(s,a) ← Q(s,a) + α[R + γΣₐ' π(a'|s')Q(s',a') - Q(s,a)]

- **方差**: 比SARSA方差小

### n步方法
- **n步回报**: Gₜ:ₜ₊ₙ = Rₜ₊₁ + ... + γⁿ⁻¹Rₜ₊ₙ + γⁿV(Sₜ₊ₙ)
- **TD(λ)**: 指数加权平均
- **前向视图**: 理论形式
- **后向视图**: 资格迹实现

### 资格迹 (Eligibility Traces)
- **ET(λ)算法**: SARSA(λ), Q(λ)
- **迹**: Eₜ(s,a) = γλEₜ₋₁(s,a) + 1(Sₜ=s,Aₜ=a)
- **更新**: 所有状态按迹加权更新
- **λ=0**: 纯TD; **λ=1**: 纯MC

## 函数近似

### 线性函数近似
- **形式**: V̂(s,w) = wᵀφ(s)
- **特征**: φ(s) — 状态特征向量
- **SGD更新**: w ← w + α[G - V̂(s,w)]∇V̂(s,w)
- **收敛性**: 线性TD保证收敛

### 非线性函数近似
- **神经网络**: DQN及其变体
- **收敛挑战**: 非线性+自举可能发散
- **经验回放**: 打破样本相关性
- **目标网络**: 稳定学习过程

### DQN (Deep Q-Network)
- **损失函数**: L = (R + γ max Q(s',a';θ⁻) - Q(s,a;θ))²
- **关键技术**:
  - 经验回放 (Experience Replay)
  - 目标网络 (Target Network)
  - 奖励裁剪
- **改进**: Double DQN, Dueling DQN, Prioritized Replay

### 策略梯度方法
- **思想**: 直接参数化策略
- **目标**: J(θ) = E[R] 或 V^{π_θ}(s₀)
- **梯度**: ∇J = E[∇log π(a|s) · G]

- **REINFORCE**: 蒙特卡洛策略梯度

### Actor-Critic
- **Actor**: 策略网络 π(a|s;θ)

- **Critic**: 价值网络 V(s;w) 或 Q(s,a;w)
- **TD误差**: δ = R + γV(s') - V(s)
- **更新**:
  - Critic: w ← w + αδ∇V(s)
  - Actor: θ ← θ + βδ∇log π(a|s)

### 优势Actor-Critic (A2C/A3C)
- **优势函数**: A(s,a) = Q(s,a) - V(s)
- **减少方差**: 基线减去了状态价值
- **异步**: 多线程并行 (A3C)
- **同步**: 批量更新 (A2C)

### PPO (Proximal Policy Optimization)
- **问题**: 策略梯度更新步长敏感
- **TRPO**: 信任域约束
- **PPO**: 裁剪目标函数
  - L^{CLIP}(θ) = E[min(rₜ(θ)Aₜ, clip(rₜ(θ),1-ε,1+ε)Aₜ)]
  - rₜ(θ) = π_θ(aₜ|sₜ) / π_{θ_old}(aₜ|sₜ)

- **优势**: 简单、稳定、高效

## 模型基础方法

### 学习模型
- **模型**: 学习 P̂(s'|s,a) 和 R̂(s,a)

- **监督学习**: 回归问题
- **规划**: 用学习模型模拟
- **Dyna架构**: 结合学习与规划

### 蒙特卡洛树搜索 (MCTS)
- **四步骤**:
  1. 选择: UCB选择子节点
  2. 扩展: 添加新节点
  3. 模拟: 随机 rollout
  4. 反向传播: 更新统计量
- **UCB1**: 平衡探索与利用
- **应用**: AlphaGo、AlphaZero

### 模型预测控制 (MPC)
- **思想**: 滚动优化
- **交叉熵方法 (CEM)**: 采样优化
- **随机射击**: 随机采样动作序列
- **模型集成**: 减少模型误差

## 高级主题

### 多智能体强化学习
- **合作**: 共同最大化团队奖励
- **竞争**: 零和博弈
- **混合**: 一般和博弈
- **方法**: 独立学习、联合学习

### 分层强化学习
- **选项 (Options)**: 时间抽象
  - 选项: ⟨I, π, β⟩ — 初始集、策略、终止
- **半马尔可夫决策过程**: 考虑时间抽象
- **MAXQ分解**: 分层价值函数分解
- ** feudal网络**: 管理者-工人架构

### 模仿学习
- **行为克隆**: 监督学习
  - 问题: 分布漂移
- **DAgger**: 迭代收集专家数据
- **逆强化学习**: 从专家行为恢复奖励

### 离线强化学习
- **问题**: 从固定数据集学习
- **挑战**: 分布外动作
- **方法**: 
  - 保守Q学习 (CQL)
  - 加权行为克隆
  - 策略约束

### 探索方法
- **ε-贪心**: 简单基线
- **UCB**: 置信区间上界
- **Thompson采样**: 贝叶斯方法
- **内在奖励**: 好奇心、计数
- **信息增益**: 贝叶斯最优探索

## 理论分析

### 收敛性
- **表格情况**: 各种算法的收敛保证
- **函数近似**: 线性TD收敛，非线性挑战
- **策略梯度**: 收敛到局部最优

### 样本复杂度
- **PAC-MDP**: 概率近似正确
- **遗憾分析**: 与最优策略的累积差距
- **下界**: 问题固有的复杂度

### 近似误差
- **模型误差**: 模型与学习模型的差距
- **值函数误差**: 函数近似误差
- **策略梯度方差**: 估计方差
