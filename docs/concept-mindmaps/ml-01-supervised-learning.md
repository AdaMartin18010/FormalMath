# 监督学习基础 - 思维导图

## 核心概念

### 什么是监督学习
- **定义**: 从标记数据中学习输入到输出的映射函数
- **目标**: 最小化预测误差
- **关键要素**:
  - 输入空间 X
  - 输出空间 Y
  - 训练数据集 {(xᵢ, yᵢ)}ᵢ₌₁ⁿ
  - 假设空间 H

### 学习范式
- **回归**: 连续输出预测
  - 线性回归
  - 多项式回归
  - 非参数回归
- **分类**: 离散标签预测
  - 二分类
  - 多分类
  - 多标签分类

## 线性模型

### 线性回归
- **模型**: f(x) = wᵀx + b
- **损失函数**: MSE = (1/n)Σ(yᵢ - ŷᵢ)²
- **解析解**: w = (XᵀX)⁻¹Xᵀy
- **几何解释**: 最小二乘投影

### 逻辑回归
- **模型**: P(y=1|x) = σ(wᵀx + b)

- **Sigmoid函数**: σ(z) = 1/(1+e⁻ᶻ)
- **损失函数**: 交叉熵
- **决策边界**: 线性超平面

### 感知机
- **模型**: ŷ = sign(wᵀx + b)
- **学习规则**: 误差驱动更新
- **收敛性**: 线性可分保证收敛
- **局限**: 无法处理非线性

## 损失函数与优化

### 常见损失函数
- **平方损失**: L = (y - ŷ)²
- **绝对值损失**: L = |y - ŷ|

- **Huber损失**: 结合两者优点
- **Hinge损失**: max(0, 1 - y·ŷ)
- **交叉熵损失**: -Σy log(ŷ)

### 正则化技术
- **L1正则 (Lasso)**: λ‖w‖₁ → 稀疏解
- **L2正则 (Ridge)**: λ‖w‖₂² → 平滑解
- **弹性网络**: αL1 + (1-α)L2
- **贝叶斯视角**: 先验分布

### 优化算法
- **梯度下降 (GD)**: 全批量更新
- **随机梯度下降 (SGD)**: 单样本更新
- **小批量梯度下降**: 折中方案
- **学习率调度**: 动态调整η

## 概率视角

### 最大似然估计 (MLE)
- **目标**: 最大化 P(D|θ)

- **等价形式**: 最小化负对数似然
- **与高斯噪声**: 等价于最小二乘

### 最大后验估计 (MAP)
- **目标**: 最大化 P(θ|D) ∝ P(D|θ)P(θ)

- **先验作用**: 正则化效果
- **与高斯先验**: 等价于L2正则

### 贝叶斯方法
- **完整后验**: P(θ|D)
- **预测分布**: ∫P(y|x,θ)P(θ|D)dθ

- **近似方法**: 变分推断、MCMC

## 模型评估

### 评估指标
- **回归指标**:
  - MSE, RMSE, MAE
  - R² 决定系数
  - 解释方差
- **分类指标**:
  - 准确率、精确率、召回率
  - F1分数、AUC-ROC
  - 混淆矩阵

### 交叉验证
- **K折交叉验证**: 数据分割策略
- **留一法**: n折交叉验证
- **分层采样**: 保持类别比例
- **时间序列**: 前向链式验证

### 偏差-方差分解
- **偏差²**: 模型假设导致的误差
- **方差**: 数据扰动导致的误差
- **不可约误差**: 噪声项
- **权衡**: 模型复杂度平衡

## 扩展方法

### 基函数扩展
- **多项式特征**: x, x², x³, ...
- **样条函数**: 分段多项式
- **径向基函数 (RBF)**: 局部响应
- **傅里叶特征**: 周期函数

### 局部加权方法
- **K近邻 (KNN)**: 局部投票/平均
- **核回归**: 加权最小二乘
- **LOESS**: 局部多项式拟合

### 支持向量机 (SVM)
- **最大间隔**: 几何间隔最大化
- **软间隔**: 允许分类错误
- **核技巧**: 非线性扩展
- **对偶问题**: 稀疏解

## 数学工具

### 矩阵运算
- **矩阵求逆**: (XᵀX)⁻¹
- **特征值分解**: 谱分析
- **SVD**: 奇异值分解
- **Moore-Penrose伪逆**: 广义逆

### 凸优化
- **凸函数**: f(λx+(1-λ)y) ≤ λf(x)+(1-λ)f(y)
- **凸集**: 任意两点连线在集合内
- **KKT条件**: 最优性条件
- **强对偶性**: 原始与对偶等价

### 统计学习理论
- **Hoeffding不等式**: 浓度不等式
- **一致收敛**: 经验风险→期望风险
- **VC维**: 模型复杂度度量
- **泛化界**: 理论保证

## 实践考虑

### 特征工程
- **标准化**: 零均值单位方差
- **归一化**: 缩放到[0,1]
- **特征选择**: 过滤法、包裹法、嵌入法
- **降维**: PCA、特征哈希

### 超参数调优
- **网格搜索**: 穷举搜索
- **随机搜索**: 随机采样
- **贝叶斯优化**: 先验引导
- **早停**: 防止过拟合
