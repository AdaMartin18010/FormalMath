# 整环与域 - 思维导图

## 概述

整环和域是环论中最重要、最基础的两类结构。整环推广了整数的算术性质，是研究唯一分解理论的理想舞台；而域则是代数运算最完善的结构，在其中所有非零元都可逆，使得域上的线性代数（向量空间理论）成为可能。从整环到域的转化通过局部化实现，这是代数几何和代数数论的核心构造。

## 重要定理总结

| 定理 | 陈述 | 应用 |
|------|------|------|
| **算术基本定理** | ℤ是UFD | 数论基础 |
| **Gauss引理** | R是UFD ⇒ R[x]是UFD | 多项式环 |
| **Bézout等式** | PID中存在as+bt=gcd | 算法基础 |
| **域上多项式UFD** | k[x]是欧几里得整环 | 多项式理论 |
| **Freshman法则** | char=p时 $(a+b)^p = a^p + b^p$ | 有限域计算 |
| **塔定律** | $[K:F] = [K:L][L:F]$ | 域扩张度计算 |
