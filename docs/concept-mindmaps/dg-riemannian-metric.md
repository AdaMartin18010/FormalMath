# 黎曼度量思维导图

## 概述
黎曼度量是流形上的内积结构，使得每一点的切空间成为欧几里得空间，为测量长度、角度和体积提供了可能，是黎曼几何的核心概念。

## 常曲率空间

| 曲率 | 空间 | 度量 | 拓扑 |
|------|------|------|------|
| K=1 | 球面Sⁿ | dr²+sin²r ds²_{n-1} | 紧致 |
| K=0 | 欧氏ℝⁿ | 标准度量 | 非紧 |
| K=-1 | 双曲Hⁿ | dr²+sinh²r ds²_{n-1} | 非紧 |

## 核心定理

| 定理 | 内容 |
|------|------|
| **Nash嵌入** | 任何黎曼流形可等距嵌入欧氏空间 |
| **Hopf-Rinow** | 度量完备 ⟺ 测地完备 |
| **Gauss-Bonnet** | 2维总曲率=2π欧拉示性数 |
