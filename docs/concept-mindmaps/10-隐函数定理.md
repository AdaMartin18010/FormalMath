## 核心要点

### 一元隐函数定理
**条件**:
1. F(x₀,y₀) = 0
2. F, F_y 在邻域连续
3. F_y(x₀,y₀) ≠ 0

**结论**: ∃! y = f(x), F(x,f(x)) ≡ 0

### 导数公式
$$\frac{dy}{dx} = -\frac{F_x}{F_y}$$

### 多元情形
对于 F(x,y) = 0，其中 x∈ℝⁿ, y∈ℝᵐ:
**条件**: det(∂F/∂y) ≠ 0

**结论**: 存在隐函数 y = f(x)

### 反函数定理
**条件**: f: ℝⁿ→ℝⁿ, det(Df(x₀)) ≠ 0

**结论**: f 在 x₀ 邻域是局部微分同胚
