# 同伦论思维导图

## 概述
同伦论研究空间在连续形变下的不变性质，是代数拓扑的核心分支，高阶同伦群提供了比同调更精细但更难计算的不变量。

## 核心定理

| 定理 | 内容 |
|------|------|
| **Hurewicz** | 第一个非零同伦群与同调群同构 |
| **Whitehead** | CW复形间的弱同伦等价是同伦等价 |
| **Freudenthal** | 悬垂在稳定范围内是同构 |
