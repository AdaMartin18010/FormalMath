# 张量分析思维导图

## 概述
张量分析是研究流形上多线性代数结构的数学分支，为物理学的场论和微分几何提供了统一的数学语言和计算框架。

## 张量类型汇总

| 类型 | 表示 | 阶数 | 例子 |
|------|------|------|------|
| **标量** | (0,0) | 0 | 函数f |
| **向量** | (1,0) | 1 | 切向量X |
| **余向量** | (0,1) | 1 | 微分df |
| **线性变换** | (1,1) | 2 | 矩阵Aⁱⱼ |
| **双线性型** | (0,2) | 2 | 度量gᵢⱼ |
| **曲率** | (1,3) | 4 | Rⁱⱼₖₗ |

## 重要恒等式

```
Cartan公式: L_X = d∘i_X + i_X∘d
外微分: d(ω∧η) = dω∧η + (-1)^deg(ω) ω∧dη
Bianchi: ∇_X R(Y,Z)W + 循环 = 0
```
