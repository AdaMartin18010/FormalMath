# 微分流形思维导图

## 概述
微分流形是局部类似于欧几里得空间的拓扑空间，拥有光滑的结构，是现代微分几何、广义相对论和规范场论的数学基础。

## 流形的分类（低维）

| 维数 | 紧致连通 | 非紧致连通 |
|------|---------|-----------|
| **1** | S¹ | ℝ |
| **2** | 可定向：Σ_g (g≥0) | 同上 |
| **2** | 不可定向：N_k (k≥1) | 同上 |

## 球面的光滑结构

| 维数 | 光滑结构数 |
|------|-----------|
| n≤3 | 1（唯一） |
| 4 | 不可数无穷 |
| 5-6 | 1 |
| 7 | 28 |
| n≥5 | 有限，与群同伦有关 |
