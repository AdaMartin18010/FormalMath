# 测地线思维导图

## 概述
测地线是黎曼流形上局部最短的曲线，是直线的推广，在广义相对论中描述自由粒子的运动轨迹，是黎曼几何的核心研究对象。

## 测地线的变分特征

```

长度泛函 L(γ) = ∫|γ'|dt

    ↓ 一阶变分
测地线方程 ∇_{γ'}γ' = 0
    ↓ 二阶变分
Jacobi场方程 J'' + R(J,γ')γ' = 0

```

## 不同空间的测地线

| 空间 | 测地线 | 特点 |
|------|--------|------|
| **ℝⁿ** | 直线 | 全局最短 |
| **Sⁿ** | 大圆 | 周期闭合 |
| **Hⁿ** | 垂直于边界的圆弧 | 无限延伸 |
| **柱面** | 螺线/经线/纬线 | 分类明确 |
| **环面** | 缠绕曲线 | 有理/无理斜率 |

## 核心定理

| 定理 | 内容 |
|------|------|
| **Hopf-Rinow** | 度量完备 ⟺ 测地完备 ⟺ 任意两点间有极小测地线 |
| **Gauss引理** | 指数映射保持径向距离 |
| **Morse指标** | Morse指标=共轭点（计重数）个数 |
