# 数学×物理学：量子力学的数学基础

## 概述

量子力学是20世纪物理学的革命性突破，其数学基础建立在泛函分析、算子理论和概率论之上。希尔伯特空间成为描述量子态的自然舞台，而算子理论则刻画了物理量的测量。

## 数学严格化

- **Hilbert空间公理化**: von Neumann (1932)
- **C*-代数方法**: Segal, Haag-Kastler
- **Rigged Hilbert空间**: 处理连续谱 (Gelfand)
- **量子测量理论**: 正定算子值测度 (POVM)
- **退相干理论**: 环境诱导超选择
