# 随机微分方程 - 思维导图

## 概述

随机微分方程(Stochastic Differential Equation, SDE)是描述随机系统动态演化的微分方程，其中包含随机噪声项（通常由布朗运动驱动）。由Kiyoshi Itô开创的随机分析理论为SDE提供了严格的数学基础，在金融工程、物理、生物、控制论等领域有广泛应用。

## 经典SDE模型

```mermaid
graph TD
    subgraph 几何布朗运动
        A[dS_t = μS_t dt + σS_t dW_t] --> B[解: S_t = S_0 exp((μ-σ²/2)t + σW_t)]
        B --> C[Black-Scholes模型]
    end
    
    subgraph Ornstein-Uhlenbeck
        D[dX_t = -θ(X_t-μ)dt + σ dW_t] --> E[解: X_t ~ N(μ + (X_0-μ)e^(-θt), σ²(1-e^(-2θt))/2θ)]
        E --> F[均值回归过程]
    end
    
    subgraph CIR模型
        G[dr_t = a(b-r_t)dt + σ√r_t dW_t] --> H[保证r_t ≥ 0]
        H --> I[利率建模]
    end
    
    subgraph Heston模型
        J[dS_t = μS_t dt + √v_t S_t dW_t¹] --> K[dv_t = κ(θ-v_t)dt + ξ√v_t dW_t²]
        K --> L[随机波动率]
    end
    
    style A fill:#e3f2fd
    style D fill:#fff3e0
    style G fill:#e8f5e9

```

## 重要定理与公式

| 名称 | 内容 | 应用 |
|------|------|------|
| **Itô引理** | $df = f' dX + \frac{1}{2}f'' d\langle X \rangle$ | SDE求解 |
| **乘积法则** | $d(XY) = X dY + Y dX + d\langle X,Y \rangle$ | 多变量SDE |
| **Girsanov定理** | 测度变换: $d\tilde{W}_t = dW_t + \theta_t dt$ | 风险中性测度 |
| **Feynman-Kac公式** | PDE与SDE的联系 | 定价公式 |
| **鞅表示定理** | $M_t = M_0 + \int H_s dW_s$ | 完全市场 |

## 关键公式速查

| 公式 | 说明 |
|------|------|
| $dX_t = \mu(X_t,t)dt + \sigma(X_t,t)dW_t$ | SDE标准形式 |
| $df(X_t) = f'(X_t)dX_t + \frac{1}{2}f''(X_t)d\langle X \rangle_t$ | Itô引理 |
| $d\langle X \rangle_t = \sigma^2(X_t,t)dt$ | 二次变差 |
| $S_t = S_0 \exp((\mu-\frac{\sigma^2}{2})t + \sigma W_t)$ | 几何布朗运动解 |
| $\mathbb{E}[\int_0^t H_s dW_s] = 0$ | Itô积分零均值 |
| $\mathbb{E}[(\int_0^t H_s dW_s)^2] = \mathbb{E}[\int_0^t H_s^2 ds]$ | Itô等距 |
| $V_t = e^{-r(T-t)}\mathbb{E}^Q[V_T|\mathcal{F}_t]$ | 风险中性定价 |
| $d\tilde{W}_t = dW_t + \theta_t dt$ | Girsanov变换 |
