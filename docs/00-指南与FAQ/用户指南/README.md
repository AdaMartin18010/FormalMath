## 📖 指南详细介绍

### 00-快速开始指南

**目标**: 帮助您快速了解 FormalMath 项目，找到适合自己水平的学习路径

**主要内容**:
- 项目简介（6,573+篇文档、62位数学家、20个Lean4定理、121+门国际课程）
- 5分钟快速上手（确认学习水平、选择第一篇文档）
- 文档命名规则解析（基础版、增强版、深度扩展版、国际标准版）
- 学习路径推荐（基础→代数→分析→拓扑，或按兴趣专题学习）
- 资源查找方法
- 智能学习系统使用（认知诊断、自适应学习）
- 常见问题解答

**推荐首先阅读本文档**

### 01-项目使用指南

**目标**: 详细介绍如何使用FormalMath的各个功能模块

**主要内容**:
1. **数学家理念体系浏览**
   - 标准目录结构解析
   - 推荐浏览路径
   - 深度等级识别
   - 重点推荐数学家

2. **形式化证明模块使用**
   - 环境要求（Lean 4.29.0）
   - 20个验证定理列表
   - 代码结构说明
   - 与Mathlib4对齐

3. **智能学习系统使用**
   - 认知诊断系统（HCD模型）
   - 评估系统（多维度评价）
   - 自适应学习路径系统（A*算法）
   - 三系统协同使用流程

4. **国际课程对应查询**
   - 七所学校96+门课程映射
   - 三种查找方法
   - 权威资源引用

### 02-新手入门指南

**目标**: 帮助首次参与项目的新贡献者完成第一次贡献

**主要内容**:
1. **项目概览**
   - 项目结构说明
   - 内容五级分类（L0-L4）

2. **贡献方式选择**
   - 内容贡献（无需编程）
   - 代码贡献（Lean4、工具开发）

3. **第一次贡献步骤**
   - 准备工作（GitHub账号、Fork项目）
   - 寻找任务（good first issue标签）
   - 创建分支、修改、提交
   - 创建PR并等待审核

4. **开发环境配置**
   - 必需工具（Git、Python、编辑器）
   - 可选工具（Lean4、Node.js、Docker）

5. **常见问题解答**
   - 获取帮助的渠道
   - 导师计划介绍
