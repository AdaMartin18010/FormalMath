## 一、项目简介

### 1.1 什么是FormalMath？

FormalMath是一个**面向数学教育的综合性知识库**，旨在：

- 建立从基础数学到现代数学前沿的完整知识体系
- 通过"数学家理念体系"提供独特的历史与哲学视角
- 提供形式化证明（Lean4）与权威数学库（Mathlib4）对齐的学习资源
- 为数学学习者提供结构化、个性化的学习路径

### 1.2 核心特色

| 特色 | 说明 |
|------|------|
| **数学家理念体系** | 62位数学家的思想与贡献，从阿基米德到肖尔策 |
| **国际权威对齐** | 与MIT、Harvard、Stanford、ETH、Princeton课程详细对应 |
| **形式化证明** | 20个可运行的Lean4定理，与Mathlib4对齐 |
| **智能学习系统** | 认知诊断、学习评估、自适应学习路径 |

### 1.3 内容结构

```
FormalMath/
├── docs/                    # 数学分支文档（01-15）
├── 数学家理念体系/           # 62位数学家的深度分析
├── project/                 # 项目文档、课程映射
├── ref/                     # 参考文献
└── concept/                 # 概念分析
```

## 二、快速开始

### 2.1 选择您的起点

根据您的背景选择合适的学习起点：

#### 初学者（高中/大一水平）

**推荐路径**:

1. `docs/01-基础数学/集合论/01-集合论基础-国际标准版.md`
2. `docs/01-基础数学/集合论/02-数系与运算-深度扩展版.md`
3. `docs/02-代数结构/线性代数/线性代数-深度扩展版.md`

#### 中级学习者（大二/大三水平）

**推荐路径**:

1. `docs/02-代数结构/群论/群论-深度扩展版.md`
2. `docs/03-分析学/01-实分析/实分析-深度扩展版.md`
3. `docs/05-拓扑学/点集拓扑-深度扩展版.md`

#### 高级学习者（研究生水平）

**推荐路径**:

1. `数学家理念体系/格洛腾迪克数学理念/02-数学内容深度分析/`
2. `docs/13-代数几何/概形理论-深度扩展版.md`
3. `docs/15-同调代数/导出范畴-深度扩展版.md`

### 2.2 阅读文档的技巧

#### 文档命名规则

| 后缀 | 含义 | 适合人群 |
|------|------|----------|
| `-基础版.md` | 概念介绍 | 初学者 |
| `-增强版.md` | 定理+例子 | 中级 |
| `-深度扩展版.md` | 完整证明 | 高级 |
| `-国际标准版.md` | 国际对齐 | 所有 |

#### 文档结构

每个深度文档通常包含：

1. **定义** - 严格的形式化定义
2. **定理** - 核心定理陈述
3. **证明思路** - 理解证明的关键步骤
4. **例子** - 具体例子帮助理解
5. **历史背景** - 数学史视角
6. **参考文献** - 权威教材引用

## 五、使用智能学习系统

### 5.1 认知诊断系统

**使用指南**: `docs/00-认知诊断系统使用指南.md`

**功能**:

- 评估L0-L3四个知识层次的能力
- 识别学习优势与薄弱环节
- 生成个性化学习建议

**使用步骤**:

1. 进入系统，选择"开始诊断测试"
2. 选择测试参数（层次、领域、题目数）
3. 完成测试（约30-45分钟）
4. 查看诊断报告
5. 根据建议开始学习

### 5.2 评估系统

**使用指南**: `docs/00-评估系统使用指南.md`

**评估维度**:

- 知识掌握度 (30%)
- 技能熟练度 (25%)
- 问题解决能力 (25%)
- 创新思维能力 (20%)

**评估类型**:

- 形成性评价（学习过程中）
- 总结性评价（阶段结束）
- 过程性评价（持续跟踪）
- 增值评价（进步幅度）

### 5.3 自适应学习路径系统

**使用指南**: `docs/00-自适应学习路径系统使用指南.md`

**功能**:

- 分析学习者特征
- 生成个性化学习路径
- 实时调整学习进度
- 提供学习支持

**使用步骤**:

1. 导入认知诊断结果
2. 设定学习目标
3. 系统生成最优路径
4. 跟随路径学习
5. 系统根据进度动态调整
