## 概述

FormalMath 采用**文件系统为主、数据库存储为辅**的混合存储架构：

| 数据类型 | 存储方式 | 格式 | 说明 |
|----------|----------|------|------|
| 文档内容 | 文件系统 | Markdown | 数学文档 |
| 元数据 | 文件系统 + JSON | YAML/JSON | 文档元数据 |
| 知识图谱 | 文件系统 | YAML/JSON | 概念关系 |
| 用户数据 | SQLite/JSON | SQLite/JSON | 学习数据 |
| 配置数据 | 文件系统 | YAML/JSON | 系统配置 |

### 设计原则

1. **Git 友好**: 所有数据支持版本控制
2. **人类可读**: 使用文本格式，便于审查
3. **工具无关**: 不依赖特定数据库软件
4. **易于迁移**: 标准格式，方便数据迁移

## 元数据存储

### 元数据提取与缓存

```python
# tools/metadata-system/metadata_extractor.py

class MetadataExtractor:
    """扫描文档并提取元数据"""

    def scan_directory(self, pattern: str = '**/*.md') -> List[MetadataRecord]:
        """扫描目录提取所有文档元数据"""
        records = []
        for file_path in glob(pattern, recursive=True):
            record = self.extract_from_file(file_path)
            records.append(record)
        return records

    def extract_from_file(self, file_path: str) -> MetadataRecord:
        """从单个文件提取元数据"""
        with open(file_path, 'r', encoding='utf-8') as f:
            content = f.read()

        # 解析 YAML Frontmatter
        if content.startswith('---'):
            parts = content.split('---', 2)
            if len(parts) >= 3:
                metadata = yaml.safe_load(parts[1])
                body = parts[2]

        # 计算统计信息
        word_count = len(body.split())

        return MetadataRecord(
            file_path=file_path,
            metadata=metadata,
            word_count=word_count,
            # ...
        )

    def export_to_json(self, output_path: str):
        """导出元数据到 JSON"""
        records = self.scan_directory()
        with open(output_path, 'w', encoding='utf-8') as f:
            json.dump(records, f, ensure_ascii=False, indent=2)
```

### 元数据查询

```python
# tools/metadata-system/metadata_query.py

class MetadataQuery:
    """元数据查询系统"""

    def __init__(self, metadata_file: str):
        with open(metadata_file, 'r') as f:
            self.records = json.load(f)

    def query(
        self,
        category: Optional[str] = None,
        difficulty: Optional[str] = None,
        has_proofs: Optional[bool] = None,
        # ...
    ) -> List[MetadataRecord]:
        """多条件查询"""
        results = self.records

        if category:
            results = [r for r in results if r.get('category') == category]

        if difficulty:
            results = [r for r in results if r.get('difficulty') == difficulty]

        if has_proofs is not None:
            results = [r for r in results if r.get('has_proofs') == has_proofs]

        return results

    def search(self, keyword: str) -> List[MetadataRecord]:
        """全文搜索"""
        return [
            r for r in self.records
            if keyword in r.get('title', '')
            or keyword in r.get('tags', [])
            or keyword in str(r.get('metadata', {}))
        ]
```

## 用户数据存储

### SQLite 存储方案

```sql
-- 用户表
CREATE TABLE users (
    id TEXT PRIMARY KEY,
    name TEXT NOT NULL,
    email TEXT UNIQUE NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    preferences JSON
);

-- 概念掌握度表
CREATE TABLE concept_mastery (
    user_id TEXT REFERENCES users(id),
    concept_id TEXT NOT NULL,
    mastery_level REAL CHECK (mastery_level BETWEEN 0 AND 100),
    last_studied TIMESTAMP,
    study_count INTEGER DEFAULT 0,
    confidence REAL,
    PRIMARY KEY (user_id, concept_id)
);

-- 学习历史表
CREATE TABLE learning_history (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id TEXT REFERENCES users(id),
    timestamp TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    activity_type TEXT CHECK (activity_type IN ('read', 'practice', 'assessment', 'review')),
    concept_id TEXT,
    duration_minutes INTEGER,
    performance_score REAL
);

-- 能力评估表
CREATE TABLE ability_assessments (
    id TEXT PRIMARY KEY,
    user_id TEXT REFERENCES users(id),
    timestamp TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    scores JSON,  -- 五维能力评分
    overall_score REAL,
    level TEXT,
    recommendations JSON
);

-- 创建索引
CREATE INDEX idx_history_user_time ON learning_history(user_id, timestamp);
CREATE INDEX idx_mastery_user ON concept_mastery(user_id);
CREATE INDEX idx_assessments_user ON ability_assessments(user_id);
```

## 备份与恢复

### 备份策略

| 数据类型 | 备份方式 | 频率 | 保留期 |
|----------|----------|------|--------|
| 文档内容 | Git 仓库 | 每次提交 | 永久 |
| 元数据 | Git 仓库 + 导出 | 每日 | 30天 |
| 用户数据 | 定期导出 | 每日 | 90天 |
| 配置数据 | Git 仓库 | 每次变更 | 永久 |

### 备份脚本

```bash
#!/bin/bash
# backup.sh

BACKUP_DIR="/backup/formalmath/$(date +%Y%m%d)"
mkdir -p $BACKUP_DIR

# 备份 Git 仓库
git bundle create $BACKUP_DIR/repo.bundle --all

# 备份元数据
python tools/metadata-system/metadata_cli.py export \
  --format json \
  --output $BACKUP_DIR/metadata.json

# 备份用户数据 (如使用 SQLite)
cp data/users.db $BACKUP_DIR/

# 压缩
 tar -czf $BACKUP_DIR.tar.gz $BACKUP_DIR
rm -rf $BACKUP_DIR

# 清理旧备份 (保留 30 天)
find /backup/formalmath -name "*.tar.gz" -mtime +30 -delete
```

### 恢复流程

```bash
#!/bin/bash
# restore.sh

BACKUP_FILE=$1
RESTORE_DIR="/tmp/restore"

# 解压
tar -xzf $BACKUP_FILE -C $RESTORE_DIR

# 恢复 Git 仓库
git clone $RESTORE_DIR/repo.bundle formalmath-restored

# 恢复元数据
cp $RESTORE_DIR/metadata.json tools/metadata-system/

# 恢复用户数据
cp $RESTORE_DIR/users.db data/

echo "恢复完成"
```
