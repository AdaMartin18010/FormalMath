## 项目概述

FormalMath-Interactive 是 FormalMath 项目的交互式学习前端，基于 React + TypeScript + Vite 构建。

### 技术栈

| 类别 | 技术 | 版本 | 说明 |
|------|------|------|------|
| 框架 | React | 18.x | UI 框架 |
| 语言 | TypeScript | 5.x | 类型安全 |
| 构建 | Vite | 5.x | 构建工具 |
| 路由 | React Router | 6.x | 页面路由 |
| 状态 | Zustand | 4.x | 状态管理 |
| 查询 | React Query | 5.x | 数据获取 |
| 样式 | Tailwind CSS | 3.x | CSS 框架 |
| 组件 | shadcn/ui | - | UI 组件库 |
| 图标 | Lucide React | - | 图标库 |
| 数学 | KaTeX | 0.16.x | 公式渲染 |
| 图表 | D3.js | 7.x | 数据可视化 |
| 协作 | Yjs | - | 实时协作 |

## 开发环境

### 环境要求

- Node.js 18+
- npm 9+ 或 pnpm 8+
- Git

### 快速开始

```bash
# 克隆项目
git clone https://github.com/formalmath/FormalMath.git
cd FormalMath/FormalMath-Interactive

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 运行测试
npm run test
```

```bash

# .env.production
VITE_API_BASE_URL=https://api.formalmath.org/api/v1
```

## 状态管理

### Zustand Store 结构

```typescript
// stores/userStore.ts
import { create } from 'zustand';
import { persist } from 'zustand/middleware';

interface UserState {
  // 状态
  user: User | null;
  preferences: UserPreferences;
  learningProgress: LearningProgress;

  // 操作
  setUser: (user: User) => void;
  updateProgress: (conceptId: string, mastery: number) => void;
  setPreference: (key: keyof UserPreferences, value: any) => void;
}

export const useUserStore = create<UserState>()(
  persist(
    (set, get) => ({
      user: null,
      preferences: defaultPreferences,
      learningProgress: {},

      setUser: (user) => set({ user }),

      updateProgress: (conceptId, mastery) => {
        set((state) => ({
          learningProgress: {
            ...state.learningProgress,
            [conceptId]: mastery,
          },
        }));
      },

      setPreference: (key, value) => {
        set((state) => ({
          preferences: { ...state.preferences, [key]: value },
        }));
      },
    }),
    {
      name: 'user-storage',
    }
  )
);
```

### React Query 数据获取

```typescript
// hooks/useConcept.ts
import { useQuery, useMutation } from '@tanstack/react-query';
import { conceptApi } from '@/services/api';

export const useConcept = (conceptId: string) => {
  return useQuery({
    queryKey: ['concept', conceptId],
    queryFn: () => conceptApi.getById(conceptId),
    staleTime: 5 * 60 * 1000, // 5 分钟
  });
};

export const useUpdateProgress = () => {
  return useMutation({
    mutationFn: conceptApi.updateProgress,
    onSuccess: () => {
      // 成功后刷新相关查询
      queryClient.invalidateQueries({ queryKey: ['learning-path'] });
    },
  });
};
```

## 数学公式渲染

### KaTeX 集成

```typescript
// components/math/KaTeXRenderer.tsx
import katex from 'katex';
import 'katex/dist/katex.min.css';

interface KaTeXRendererProps {
  math: string;
  displayMode?: boolean;
}

export const KaTeXRenderer: React.FC<KaTeXRendererProps> = ({
  math,
  displayMode = false,
}) => {
  const html = useMemo(() => {
    return katex.renderToString(math, {
      displayMode,
      throwOnError: false,
      strict: false,
    });
  }, [math, displayMode]);

  return (
    <span
      className={displayMode ? 'katex-display' : 'katex-inline'}
      dangerouslySetInnerHTML={{ __html: html }}
    />
  );
};
```

## 可视化组件

### 概念图谱 (D3.js)

```typescript
// components/visualization/ConceptGraph.tsx
import * as d3 from 'd3';

interface ConceptGraphProps {
  data: GraphData;
  width: number;
  height: number;
  onNodeClick?: (node: Node) => void;
}

export const ConceptGraph: React.FC<ConceptGraphProps> = ({
  data,
  width,
  height,
  onNodeClick,
}) => {
  const svgRef = useRef<SVGSVGElement>(null);

  useEffect(() => {
    if (!svgRef.current) return;

    const svg = d3.select(svgRef.current);

    // 创建力导向图模拟
    const simulation = d3.forceSimulation(data.nodes)
      .force('link', d3.forceLink(data.links).id((d: any) => d.id))
      .force('charge', d3.forceManyBody().strength(-300))
      .force('center', d3.forceCenter(width / 2, height / 2));

    // 绘制连接线
    const link = svg.append('g')
      .selectAll('line')
      .data(data.links)
      .join('line')
      .attr('stroke', '#999')
      .attr('stroke-width', 2);

    // 绘制节点
    const node = svg.append('g')
      .selectAll('circle')
      .data(data.nodes)
      .join('circle')
      .attr('r', (d) => d.importance * 10)
      .attr('fill', (d) => getNodeColor(d.category))
      .on('click', (event, d) => onNodeClick?.(d));

    // 更新位置
    simulation.on('tick', () => {
      link
        .attr('x1', (d: any) => d.source.x)
        .attr('y1', (d: any) => d.source.y)
        .attr('x2', (d: any) => d.target.x)
        .attr('y2', (d: any) => d.target.y);

      node
        .attr('cx', (d: any) => d.x)
        .attr('cy', (d: any) => d.y);
    });

    return () => simulation.stop();
  }, [data, width, height]);

  return <svg ref={svgRef} width={width} height={height} />;
};
```

## 最佳实践

### 性能最佳实践

1. **避免不必要的重渲染**
   - 使用 `React.memo` 缓存组件
   - 使用 `useMemo` 缓存计算结果
   - 使用 `useCallback` 缓存回调函数

2. **优化图片加载**
   - 使用适当的图片格式 (WebP)
   - 使用懒加载
   - 提供响应式图片

3. **代码分割**
   - 路由级懒加载
   - 组件级懒加载
   - 动态导入非关键代码

### 可访问性最佳实践

1. **语义化 HTML**
   - 使用正确的标题层级
   - 使用语义化标签
   - 提供适当的 ARIA 标签

2. **键盘导航**
   - 确保所有功能可键盘访问
   - 提供焦点指示器
   - 管理焦点顺序

3. **对比度**
   - 文字与背景对比度至少 4.5:1
   - 大文字对比度至少 3:1
