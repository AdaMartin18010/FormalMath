## 一、符号表征（形式化）

### 1.1 严格陈述

**单调收敛定理**：设 (X, Σ, μ) 是测度空间，{fₙ} 是非负可测函数列，满足：

1. fₙ ≤ fₙ₊₁ a.e.（单调递增）
2. fₙ → f a.e.（逐点收敛）

则：
$$\lim_{n\to\infty} \int_X f_n \, d\mu = \int_X f \, d\mu = \int_X \lim_{n\to\infty} f_n \, d\mu$$

**简记**：积分与极限可交换（对单调递增非负函数列）。

### 1.2 Lean 4 形式化

```lean
import Mathlib.MeasureTheory.Integral.Lebesgue

-- 单调收敛定理
theorem monotone_convergence {α : Type*} [MeasurableSpace α]
    {μ : Measure α} {f : ℕ → α → ℝ≥0∞}
    (hf_meas : ∀ n, Measurable (f n))
    (hf_mono : ∀ x, Monotone (fun n => f n x)) :
    ∫⁻ x, ⨆ n, f n x ∂μ = ⨆ n, ∫⁻ x, f n x ∂μ := by
  exact lintegral_iSup hf_meas fun x => hf_mono x
```

### 1.3 证明框架

```text
        [MCT证明结构]
              │
    ┌─────────┴─────────┐
    │                   │
[≤方向]             [≥方向]
    │                   │
fₙ ≤ f ⟹           用简单函数
∫fₙ ≤ ∫f           逼近法
    │                   │
取sup即得≤         ∫f ≤ sup∫fₙ
```

### 2.1 核心图示

```text
        [单调逼近]

    │    ╱────────
    │   ╱ f₃
    │  ╱─────

    │╱────

    └─────────────→ x

    面积逐步增加：∫f₁ ≤ ∫f₂ ≤ ∫f₃ → ∫f
```

### 2.2 阶梯逼近

```text
    │    ┌───────────
    │  ┌─┘ f₃
    │┌─┘   f₂
    ├┘     f₁
    └─────────────────

    积分 = 阶梯下的面积
    面积递增收敛到极限函数下的面积
```

## 三、直觉表征（类比）

### 3.1 一句话解释

> **MCT**：单调爬楼梯，楼层面积单调增加，最终面积等于顶楼面积

### 3.2 蓄水池类比

```text
水位逐渐上升：
├─ 水面高度 = fₙ(x)
├─ 水量 = ∫fₙ
├─ 单调递增：水只进不出
└─ 极限水量 = 最终水量
```

### 3.3 为什么成立？

```text
关键：非负 + 单调递增

├─ 非负：积分非负，有下界
├─ 单调：积分序列递增
├─ 有界：如果极限积分有限
└─ 单调有界必收敛，且收敛到sup
```

## 四、计算表征（算法）

### 4.1 Python验证

```python
import numpy as np
from scipy import integrate

def verify_mct():
    """
    验证单调收敛定理

    fₙ(x) = x^n · χ_{[0,1]}  不是好例子（非单调递增）

    改用：fₙ(x) = min(n, x²) 在 [0, 2]
    """
    def f_n(x, n):
        return np.minimum(n, x**2)

    def f_limit(x):
        return x**2

    print("验证MCT: fₙ(x) = min(n, x²) 在 [0, 2]")
    print()

    integrals = []
    for n in [1, 2, 4, 8, 16, 100]:
        result, _ = integrate.quad(lambda x: f_n(x, n), 0, 2)
        integrals.append(result)
        print(f"∫f_{n} = {result:.6f}")

    limit_integral, _ = integrate.quad(f_limit, 0, 2)
    print(f"\n∫f = ∫x² = {limit_integral:.6f}")
    print(f"理论值: 8/3 ≈ {8/3:.6f}")

verify_mct()
```

### 4.2 简单函数逼近

```python
def simple_function_approximation(f, a, b, n_levels):
    """
    用简单函数从下方逼近
    这是MCT证明的构造性版本
    """
    # 划分值域
    y_max = max(f(x) for x in np.linspace(a, b, 1000))
    levels = np.linspace(0, y_max, n_levels + 1)

    integral = 0
    for i in range(n_levels):
        level = levels[i]
        # 测度{x: f(x) ≥ level}
        xs = np.linspace(a, b, 10000)
        measure = (b - a) * np.mean(f(xs) >= level)
        integral += (levels[i+1] - levels[i]) * measure

    return integral
```

## 五、范畴表征（抽象）

### 5.1 序结构视角

```text
L¹函数空间有偏序：f ≤ g ⟺ f(x) ≤ g(x) a.e.

MCT说：
├─ 积分是保序映射
├─ 积分与定向极限交换
└─ 积分是"连续"的（对单调极限）
```

### 5.2 格论视角

```text
非负可测函数构成完备格
├─ 上确界存在
├─ MCT：积分保持可数上确界（对链）
└─ 这是 Scott 连续性的测度论版本
```

### 5.3 与其他收敛定理的关系

```text
        [收敛定理家族]
              │
    ┌─────────┼─────────┐
    │         │         │
  [MCT]    [Fatou]    [DCT]
单调递增   下极限    被控制
    │         │         │
    └────→────┴────←────┘
           │
    MCT ⟹ Fatou ⟹ DCT
```
