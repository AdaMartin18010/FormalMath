## 一、符号表征（形式化）

### 1.1 严格陈述

**代数基本定理**：每个次数≥1的复多项式在ℂ中有零点。

等价地：ℂ是代数闭域。

---

## 二、几何表征（可视化）

```text
n次多项式有n个复根（计重数）：

    p(z) = (z-z₁)(z-z₂)...(z-zₙ)
```

---

## 三、直觉表征（类比）

> **代数基本定理**：在复数域中，多项式方程总有解

---

## 四、计算表征（算法）

```python
import numpy as np

def find_roots(coeffs):
    return np.roots(coeffs)  # 所有n个根
```
