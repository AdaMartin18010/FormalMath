## 一、符号表征（形式化）

### 1.1 严格陈述

**代数基本定理**（Fundamental Theorem of Algebra）：设 $p(z) = a_n z^n + a_{n-1} z^{n-1} + \cdots + a_1 z + a_0$ 是次数 $n \geq 1$ 的复系数多项式（$a_n \neq 0$），则 $p(z)$ 在 $\mathbb{C}$ 中至少有一个零点。

**等价表述**：

1. **完全分解**：每个 $n$ 次复多项式可以完全分解为 $n$ 个一次因子的乘积：
   $$p(z) = a_n (z - z_1)(z - z_2) \cdots (z - z_n)$$
   其中 $z_1, z_2, \ldots, z_n \in \mathbb{C}$（允许重复）。

2. **代数闭性**：$\mathbb{C}$ 是**代数闭域**，即每个非常数复系数多项式在 $\mathbb{C}$ 中都有根。

3. **根的存在性**：方程 $p(z) = 0$ 在 $\mathbb{C}$ 中总有解（可能有重根）。

### 1.2 重根计数

如果 $p(z) = a_n (z - z_1)^{m_1} (z - z_2)^{m_2} \cdots (z - z_k)^{m_k}$，其中 $z_i$ 互不相同，$m_i \geq 1$，则：
- $z_i$ 是**$m_i$重根**
- $m_1 + m_2 + \cdots + m_k = n$（计重数共 $n$ 个根）

### 1.3 实系数情况

对于**实系数多项式** $p(x)$：
- 如果 $z$ 是根，则 $\bar{z}$（共轭）也是根
- 实根对应一次因子 $(x - r)$
- 非实根成对出现，对应二次因子 $(x^2 + bx + c)$

## 二、几何表征（可视化）

### 2.1 复平面上的根

    p(z) = a_n (z-z₁)(z-z₂)...(z-zₙ)

### 2.2 多项式的图像

```text
实系数多项式的根分布：

情况1：所有根都是实数
    ●───●───●───●
    z₁  z₂  z₃  z₄
    
情况2：有复根（成对出现）
    ●───●───●
    z₁  z₂  z₃
         │
        z̄₂ (共轭)
```

### 2.3 根与系数的关系

**Vieta公式**：如果 $p(z) = z^n + a_{n-1} z^{n-1} + \cdots + a_0 = (z - z_1) \cdots (z - z_n)$，则：

$$z_1 + z_2 + \cdots + z_n = -a_{n-1}$$
$$z_1 z_2 \cdots z_n = (-1)^n a_0$$

## 三、直觉表征（类比）

### 3.1 物理类比

> **代数基本定理**：在复数域中，多项式方程总有解

**类比1：寻找平衡点**

- 多项式方程 $p(z) = 0$ 类似于寻找系统的平衡点
- 代数基本定理保证：在复数域中，平衡点总是存在
- 这类似于物理系统的稳定性分析

**类比2：分解与合成**

- 多项式可以"分解"为一次因子
- 这类似于化学中的"分解反应"
- 每个因子对应一个"基本成分"（根）

### 3.2 几何类比

**类比**：多项式方程的解类似于：
- **交点**：多项式曲线与 $x$ 轴的交点
- **零点**：函数的零点
- **固定点**：映射的不动点

def factor_polynomial(coeffs):
    """
    将多项式分解为一次因子
    
    参数:
        coeffs: 系数列表
    
    返回:
        factors: 因子列表 [(z - z₁), (z - z₂), ...]

## 五、范畴表征（抽象）

### 5.1 代数闭域

**代数基本定理**说明 $\mathbb{C}$ 是**代数闭域**：

**定义**：域 $F$ 是代数闭的，如果每个非常数多项式 $p(x) \in F[x]$ 在 $F$ 中都有根。

**性质**：
- $\mathbb{C}$ 是代数闭域（代数基本定理）
- $\mathbb{R}$ 不是代数闭域（如 $x^2 + 1 = 0$ 无实根）
- 代数闭包：每个域都有唯一的代数闭包

### 5.2 Galois理论

**代数基本定理**是**Galois理论**的基础：

1. **根的存在性**：保证多项式方程有解
2. **Galois群**：研究根的可解性
3. **可解性**：五次及以上方程一般无根式解（Abel-Ruffini定理）

### 5.3 复分析视角

在**复分析**中，代数基本定理是以下结果的基础：

1. **Liouville定理**：有界整函数是常数
2. **最大模原理**：全纯函数的模在边界上达到最大值
3. **Rouché定理**：用于定位根的个数

## 六、应用实例

### 6.1 因式分解

**例子1**：分解 $p(z) = z^4 - 1$

使用代数基本定理，$p(z)$ 有4个根：$\pm 1, \pm i$

$$p(z) = (z-1)(z+1)(z-i)(z+i) = (z^2-1)(z^2+1)$$

### 6.2 求解方程

**例子2**：求解 $z^3 - 8 = 0$

根为：$z = 2, 2e^{2\pi i/3}, 2e^{4\pi i/3}$

即：$z = 2, -1 + i\sqrt{3}, -1 - i\sqrt{3}$

### 6.3 实系数多项式的分解

**例子3**：分解 $p(x) = x^4 + 1$（实系数）

在 $\mathbb{C}$ 中：$p(x) = (x - e^{i\pi/4})(x - e^{3i\pi/4})(x - e^{5i\pi/4})(x - e^{7i\pi/4})$

在 $\mathbb{R}$ 中：$p(x) = (x^2 + \sqrt{2}x + 1)(x^2 - \sqrt{2}x + 1)$

## 八、证明思路

### 8.1 分析证明（Liouville定理）

**思路**：
1. 假设 $p(z)$ 无零点
2. 则 $1/p(z)$ 是整函数
3. 当 $|z| \to \infty$ 时，$|1/p(z)| \to 0$
4. 由Liouville定理，$1/p(z)$ 是常数
5. 矛盾，故 $p(z)$ 有零点

### 8.2 拓扑证明（度理论）

**思路**：
1. 考虑映射 $f: S^1 \to S^1$，$f(z) = p(z)/|p(z)|$
2. 计算映射度（degree）
3. 当 $|z|$ 大时，$f(z) \approx z^n$
4. 度不为零，故 $f$ 是满射
5. 因此 $p(z) = 0$ 有解

### 8.3 代数证明（Galois理论）

**思路**：
1. 考虑 $p(z)$ 的分裂域
2. 使用Galois理论
3. 证明分裂域是 $\mathbb{C}$ 的子域
4. 因此根在 $\mathbb{C}$ 中

## 九、推广与变体

### 9.1 代数闭包

**定理**：每个域 $F$ 都有唯一的代数闭包 $\bar{F}$。

**例子**：
- $\bar{\mathbb{Q}} = \mathbb{A}$（代数数）
- $\bar{\mathbb{R}} = \mathbb{C}$

### 9.2 有限域

**有限域** $\mathbb{F}_q$（$q = p^n$）不是代数闭的，但其代数闭包存在且唯一。

### 9.3 超越数

代数基本定理只保证**代数数**的存在，不涉及**超越数**（如 $\pi, e$）。
