## 一、符号表征（形式化）

### 1.1 严格陈述

**Krull主理想定理**：设R是Noether环，x∈R非零非单位，P是包含(x)的极小素理想，则ht(P) ≤ 1。

推广：若I = (x₁,...,xₙ)，P是包含I的极小素理想，则ht(P) ≤ n。

## 四、计算表征（算法）

```python
def check_krull(I, P, R):
    # 验证：P包含I的极小素，则ht(P) ≤ #generators(I)
    return height(P) <= len(generators(I))
```
