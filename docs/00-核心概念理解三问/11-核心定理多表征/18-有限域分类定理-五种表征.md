## 一、符号表征（形式化）

### 1.1 严格陈述

**有限域分类定理**：每个有限域F的阶为pⁿ，其中p是素数，n≥1。对每个pⁿ，在同构意义下存在唯一的有限域，记为𝔽_{pⁿ}。

$$|F| = p^n \text{ 且 } F \cong \mathbb{F}_{p^n}$$

### 1.2 构造

𝔽_{pⁿ} = 𝔽_p[x] / ⟨f(x)⟩，其中f是𝔽_p上的n次不可约多项式。

### 1.3 Lean 4 形式化

```lean
import Mathlib.FieldTheory.Finite.Basic

theorem finite_field_card {F : Type*} [Field F] [Fintype F] :
    ∃ (p : ℕ) [Fact p.Prime] (n : ℕ), Fintype.card F = p ^ n := by
  exact FiniteField.card F

theorem finite_field_unique {F₁ F₂ : Type*} [Field F₁] [Field F₂]
    [Fintype F₁] [Fintype F₂] (h : Fintype.card F₁ = Fintype.card F₂) :
    Nonempty (F₁ ≃+* F₂) := by
  exact FiniteField.equiv_of_card_eq h
```

## 二、几何表征（可视化）

### 2.1 有限域结构

```text
𝔽_{pⁿ} 的结构：
    ┌─────────────┐
    │   𝔽_{pⁿ}     │
    │  (pⁿ个元素)   │
    └──────┬───────┘
           │
    ┌──────┴──────┐
    │             │
  𝔽_p         扩域
  (p个元素)    (n次)
```

### 2.2 例子：𝔽₄

```text
𝔽₄ = {0, 1, α, α+1}
其中 α² = α + 1

加法表：
    + │ 0  1  α  α+1
    ──┼─────────────
    0 │ 0  1  α  α+1
    1 │ 1  0  α+1 α
    α │ α  α+1 0  1
  α+1│ α+1 α  1  0
```

## 三、直觉表征（类比）

### 3.1 一句话解释

> **有限域分类**：有限域就像"时钟算术"的推广，只有pⁿ种可能的值

### 3.2 时钟类比

- 𝔽_p：p小时制的时钟
- 𝔽_{pⁿ}：pⁿ小时制的时钟
- 每个"小时数"唯一确定一个有限域

### 3.3 为什么唯一？

有限域由阶唯一确定，就像"只有一种pⁿ小时制的时钟"

    """构造𝔽_{pⁿ}"""
    from sympy import GF, Poly, symbols

## 五、范畴表征（抽象）

### 5.1 范畴视角

```text
有限域范畴：
├─ 对象：有限域
├─ 态射：域同态
├─ 同构类 = {pⁿ : p素数, n≥1}
└─ 每个同构类只有一个对象（在同构下）
```

### 5.2 万有性质

𝔽_{pⁿ}是𝔽_p的n次代数闭包在有限域中的"代表"

### 5.3 推广

- 无限域：不唯一（如ℚ, ℝ, ℂ）
- 局部域：p-adic域的分类更复杂
