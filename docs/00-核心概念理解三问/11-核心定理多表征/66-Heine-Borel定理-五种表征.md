## 一、符号表征（形式化）

### 1.1 严格陈述

**Heine-Borel定理**：ℝⁿ中子集K紧致 ⟺ K有界且闭。

---

## 二、几何表征（可视化）

```text
ℝⁿ中紧致集：

紧致 = 有界 + 闭
    = 任意开覆盖有有限子覆盖
```

---

## 三、直觉表征（类比）

> **Heine-Borel**：在ℝⁿ中，"有边界且包含边界"等价于"紧致"

---

## 四、计算表征（算法）

```python
def is_compact_Rn(K, is_bounded, is_closed):
    return is_bounded(K) and is_closed(K)
```

---

## 五、范畴表征（抽象）

Heine-Borel刻画ℝⁿ的局部紧致性，是度量空间紧致性理论的基础。
