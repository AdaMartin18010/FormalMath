## 一、符号表征（形式化）

### 1.1 有限维版本

**谱定理（对称矩阵）**：设A是n×n实对称矩阵，则存在正交矩阵Q使得Q^T A Q = Λ是对角矩阵，对角元是A的特征值。

$$A = Q \Lambda Q^T = \sum_{i=1}^n \lambda_i P_i$$

其中Pᵢ是到特征空间的正交投影。

### 1.2 无穷维版本

**谱定理（自伴算子）**：设H是Hilbert空间，T: H→H是有界自伴算子，则存在谱测度E使得：

$$T = \int_{\sigma(T)} \lambda \, dE_\lambda$$

## 二、几何表征（可视化）

### 2.1 特征分解

```text
对称矩阵A的作用：

    沿特征向量方向缩放

    A = λ₁|e₁⟩⟨e₁| + λ₂|e₂⟩⟨e₂| + ...
         ↑           ↑
    投影到e₁方向   投影到e₂方向
    再缩放λ₁倍     再缩放λ₂倍
```

### 2.2 正交分解

```text
H = E₁ ⊕ E₂ ⊕ ... ⊕ Eₖ (特征空间直和)
     │     │         │
    λ₁    λ₂        λₖ

T在每个特征空间上是数乘
```

### 3.1 一句话解释

## 四、计算表征（算法）

### 4.1 Python实现

```python
import numpy as np
from scipy.linalg import eigh

def spectral_decomposition(A):
    """
    对称矩阵的谱分解
    A = Q Λ Q^T = Σ λᵢ Pᵢ
    """
    # eigh用于对称/Hermite矩阵
    eigenvalues, eigenvectors = eigh(A)

    # 构造投影算子
    n = A.shape[0]
    projections = []
    for i in range(n):
        v = eigenvectors[:, i:i+1]
        P = v @ v.T
        projections.append(P)

    # 验证: A = Σ λᵢ Pᵢ
    A_reconstructed = sum(eigenvalues[i] * projections[i] for i in range(n))

    print(f"特征值: {eigenvalues}")
    print(f"重构误差: {np.linalg.norm(A - A_reconstructed):.2e}")

    return eigenvalues, projections

# 示例
A = np.array([[2, 1], [1, 2]])
eigenvalues, projections = spectral_decomposition(A)
```

### 4.2 无穷维：数值逼近

```python
def spectral_integral_approx(f, spectrum, projections):
    """
    计算 f(T) = ∫ f(λ) dE_λ
    """
    result = sum(f(lam) * P for lam, P in zip(spectrum, projections))
    return result

# 示例：计算 e^A
A = np.array([[0, 1], [-1, 0]])
eigenvalues, projections = spectral_decomposition(A)
exp_A = spectral_integral_approx(np.exp, eigenvalues, projections)
```

## 五、范畴表征（抽象）

### 5.1 范畴视角

```text
谱定理 = C*-代数表示
├─ 自伴算子生成交换C*-代数
├─ Gelfand同构：C*(T) ≅ C(σ(T))
└─ 函数演算：f(T) 对应 f ∈ C(σ(T))
```

### 5.2 推广

- **正规算子**：复谱定理
- **无界算子**：von Neumann代数
- **非交换**：C*-代数的谱理论

### 5.3 量子力学

```text
量子力学中的谱定理：
├─ 可观测量 = 自伴算子
├─ 测量结果 = 特征值
└─ 测量后状态 = 特征向量
```
