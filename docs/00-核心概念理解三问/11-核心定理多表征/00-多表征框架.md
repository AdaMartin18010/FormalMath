## 📋 多表征框架

### 五种表征类型

```text
        [定理的五种表征]
                │
    ┌───────────┼───────────┐
    │     │     │     │     │
[符号]  [几何]  [直觉]  [计算]  [范畴]
    │     │     │     │     │
形式化  可视化  类比   代码   抽象
证明    图示    解释   实现   视角
```

| 表征类型 | 内容 | 受众 |
|----------|------|------|
| **符号表征** | 严格数学表述+Lean形式化 | 研究者 |
| **几何表征** | 图形/图示/可视化 | 视觉学习者 |
| **直觉表征** | 类比/故事/一句话解释 | 初学者 |
| **计算表征** | 算法/代码实现 | 程序员 |
| **范畴表征** | 范畴论统一视角 | 高级学习者 |
