## 一、符号表征（形式化）

### 1.1 严格陈述

**中值定理**：若f在[a,b]连续，在(a,b)可导，则存在c∈(a,b)使得：

$$f'(c) = \frac{f(b) - f(a)}{b - a}$$

---

## 二、几何表征（可视化）

    │●╱
    └──────────

## 三、直觉表征（类比）

> **中值定理**：平均速度必在某刻等于瞬时速度

## 四、计算表征（算法）

```python
def mean_value_c(f, f_prime, a, b):
    avg_rate = (f(b) - f(a)) / (b - a)
    # 找c使得f'(c) = avg_rate
    from scipy.optimize import brentq
    return brentq(lambda x: f_prime(x) - avg_rate, a, b)
```
