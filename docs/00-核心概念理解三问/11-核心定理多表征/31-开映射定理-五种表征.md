## 一、符号表征（形式化）

### 1.1 严格陈述

**开映射定理**：设X, Y是Banach空间，T: X→Y是有界线性满射，则T是开映射（把开集映为开集）。

$$T \text{ 满射} \Rightarrow T(U) \text{ 是开集}, \forall U \text{ 开集}$$

### 1.2 等价形式

存在常数c > 0，使得B_Y(0, c) ⊂ T(B_X(0, 1))。

### 1.3 Lean 4 形式化

```lean
import Mathlib.Analysis.NormedSpace.OperatorNorm

-- 开映射定理
theorem open_mapping_theorem {E F : Type*}
    [NormedAddCommGroup E] [NormedSpace ℝ E] [CompleteSpace E]
    [NormedAddCommGroup F] [NormedSpace ℝ F] [CompleteSpace F]
    {T : E →L[ℝ] F} (hT : Function.Surjective T) :
    IsOpenMap T := by
  exact T.isOpenMap hT
```

## 二、几何表征（可视化）

### 2.1 开集映为开集

```text
X空间                Y空间
┌──────┐            ┌──────┐
│  U   │     T      │ T(U) │
│ (开) │ ────────>  │ (开) │
└──────┘            └──────┘

T把X中的开球映为Y中包含开球的集合
```

### 2.2 核心估计

```text
存在c > 0：

    X中的单位球 B_X(0,1)
          │
          T
          ↓
    包含Y中半径c的球 B_Y(0,c)
```

## 三、直觉表征（类比）

### 3.1 一句话解释

> **开映射定理**：满射的有界线性算子不会"压缩"开集，开集仍映为开集

### 3.2 投影类比

- T：像从高维投影到低维
- 满射：投影覆盖整个目标空间
- 开映射：投影不会把"大区域"压成"点"

### 3.3 为什么需要完备？

完备性（Baire纲定理）保证像不是"稀薄"的

## 四、计算表征（算法）

### 4.1 Python实现

```python
import numpy as np

def verify_open_mapping(T, domain_radius=1.0, n_samples=1000):
    """
    数值验证开映射性质
    检查T(B(0,1))是否包含某个开球
    """
    # 随机采样单位球中的点
    n = T.shape[1]
    samples = np.random.randn(n_samples, n)
    samples = samples / np.linalg.norm(samples, axis=1, keepdims=True)
    samples *= np.random.uniform(0, domain_radius, (n_samples, 1))

    # 计算像
    images = (T @ samples.T).T

    # 估计像中的最大内接球半径
    image_norms = np.linalg.norm(images, axis=1)
    min_norm_on_sphere = np.min(image_norms[np.linalg.norm(samples, axis=1) > 0.9])

    print(f"估计的内接球半径: {min_norm_on_sphere:.4f}")
    return min_norm_on_sphere

# 示例：满秩矩阵
T = np.array([[1, 0, 0], [0, 2, 0]])  # 2×3矩阵，满射
verify_open_mapping(T)
```

### 4.2 应用：求逆的有界性

```python
def bounded_inverse_estimate(T):
    """
    如果T是双射，则T⁻¹有界
    估计 ‖T⁻¹‖
    """
    # 对于有限维，使用SVD
    U, s, Vh = np.linalg.svd(T)

    if np.min(s) > 1e-10:  # 满秩
        inv_norm = 1 / np.min(s)
        print(f"‖T⁻¹‖ ≤ {inv_norm:.4f}")
        return inv_norm
    else:
        print("T不可逆")
        return np.inf
```

## 五、范畴表征（抽象）

### 5.1 范畴视角

```text
开映射定理 = Banach空间范畴的性质
├─ 满射态射（在Ban中）自动是开映射
├─ 等价于：满射 ⟹ 商映射
└─ 核+像的分解：X/ker(T) ≅ Im(T)
```

### 5.2 推广

- **Fréchet空间**：更一般的设置
- **局部凸空间**：需要额外条件
- **同调代数**：短正合列的性质

### 5.3 与闭图像定理

```text
闭图像定理：T闭图像 ⟹ T有界
开映射定理：T满射有界 ⟹ T开映射
等价（通过图的闭性）
```
