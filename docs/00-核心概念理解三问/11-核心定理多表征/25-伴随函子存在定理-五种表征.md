## 一、符号表征（形式化）

### 1.1 严格陈述

**伴随函子存在定理**：设F: C → D是函子，则F有右伴随G: D → C当且仅当对每个d ∈ D，逗号范畴(d ↓ F)有初始对象。此时G(d)就是该初始对象的C-分量。

$$F \dashv G \iff \forall d \in D, (d \downarrow F) \text{ 有初始对象}$$

等价地，F有左伴随当且仅当对每个c ∈ C，逗号范畴(F ↓ c)有余终对象。

### 1.2 单位-余单位表述

F ⊣ G ⟺ 存在自然变换：

- η: 1_C → G∘F（单位）
- ε: F∘G → 1_D（余单位）

满足三角恒等式。

## 二、几何表征（可视化）

### 2.1 伴随关系

```text
    C              D
    │              │
    │   F          │
    │  ────>       │
    │              │
    │   G          │
    │  <────       │
    │              │

F ⊣ G: F是G的左伴随
意味着：Hom_D(Fc, d) ≅ Hom_C(c, Gd)
```

### 2.2 万有性质

```text
对每个d ∈ D，找G(d)：
    d ──f──> F(c)
    │        ↑
    │        │
    │        ∃! g
    │        │
    └───> G(d) ──> c

G(d)是"最接近"d的C-对象
```

## 三、直觉表征（类比）

### 3.1 一句话解释

> **伴随函子存在**：一个函子有"最佳逆"（伴随）当且仅当它"足够好"（满足万有性质）

### 3.2 翻译类比

- F：从语言C翻译到语言D
- G：从语言D翻译回语言C
- 伴随：翻译"不丢失信息"（在同构意义下）

### 3.3 为什么需要初始对象？

初始对象 = "最佳选择"，保证伴随的唯一性

## 四、计算表征（算法）

### 4.1 Python实现（简化：有限范畴）

```python
class Functor:
    def __init__(self, on_objects, on_morphisms):
        self.obj_map = on_objects
        self.mor_map = on_morphisms

def has_right_adjoint(F, C, D):
    """
    检查F是否有右伴随
    通过检查每个d的逗号范畴是否有初始对象
    """
    for d in D.objects:
        comma_cat = comma_category(d, F)
        if not has_initial_object(comma_cat):
            return False
    return True

def construct_right_adjoint(F, C, D):
    """
    构造F的右伴随G
    """
    def G_obj(d):
        # G(d) = 逗号范畴(d↓F)的初始对象的C-分量
        comma = comma_category(d, F)
        initial = initial_object(comma)
        return initial.c_component

    def G_mor(f):
        # 使用万有性质构造态射
        return universal_morphism(f, F)

    return Functor(G_obj, G_mor)

# 示例：自由-遗忘伴随
# F: Set → Mon (自由幺半群函子)
# G: Mon → Set (遗忘函子)
# F ⊣ G
```

## 五、范畴表征（抽象）

### 5.1 范畴视角

```text
伴随 = 范畴间的"最佳对应"
├─ F ⊣ G: Hom_D(Fc, d) ≅ Hom_C(c, Gd)
├─ 单位-余单位: η, ε
└─ 三角恒等式: 一致性条件
```

### 5.2 万有性质

伴随是"万有的"：G(d)是使得Hom_D(F(-), d)可表的对象

### 5.3 推广

- 2-伴随：2-范畴中的伴随
- 无穷伴随：∞-范畴中的伴随
