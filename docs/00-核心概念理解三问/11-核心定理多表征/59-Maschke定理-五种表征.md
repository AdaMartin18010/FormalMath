## 一、符号表征（形式化）

### 1.1 严格陈述

**Maschke定理**：设G是有限群，k是域，char(k)不整除|G|，则群代数k[G]是半单的。

等价地：每个有限维k[G]-模是完全可约的。

## 四、计算表征（算法）

```python
def maschke_projection(V, W, G):
    # 构造补空间的投影
    # P = (1/|G|) Σ_g g·π·g⁻¹
    P = sum(g @ pi @ g.inv() for g in G) / len(G)
    return P
```
