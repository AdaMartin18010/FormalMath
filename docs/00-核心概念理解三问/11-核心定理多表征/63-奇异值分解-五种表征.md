## 一、符号表征（形式化）

### 1.1 严格陈述

**SVD定理**：任意m×n矩阵A可分解为：

$$A = U \Sigma V^T$$

其中U是m×m正交矩阵，V是n×n正交矩阵，Σ是m×n对角矩阵（奇异值）。

---

## 二、几何表征（可视化）

```text
A的作用分解：

    V^T: 旋转输入空间
    Σ: 沿坐标轴缩放
    U: 旋转输出空间

    A = 旋转 → 缩放 → 旋转
```

---

## 三、直觉表征（类比）

> **SVD**：任何线性变换 = 旋转 + 拉伸 + 旋转

---

## 四、计算表征（算法）

```python
import numpy as np

def svd_decomposition(A):
    U, s, Vt = np.linalg.svd(A)
    Sigma = np.zeros_like(A, dtype=float)
    np.fill_diagonal(Sigma, s)
    return U, Sigma, Vt

# 低秩近似
def low_rank_approx(A, k):
    U, s, Vt = np.linalg.svd(A)
    return U[:, :k] @ np.diag(s[:k]) @ Vt[:k, :]
```

## 五、范畴表征（抽象）

SVD是矩阵在正交群作用下的标准形，给出最优低秩近似。
