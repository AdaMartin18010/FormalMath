## 一、符号表征（形式化）

### 1.1 严格陈述

**Rolle定理**：若f在[a,b]连续，在(a,b)可导，且f(a)=f(b)，则存在c∈(a,b)使得f'(c)=0。

---

## 二、几何表征（可视化）

```text
f(a) = f(b)时，曲线必有水平切线：

    │     ╱╲
    │    ╱  ╲  ← 水平切点
    │   ╱    ╲
    └───●─────●───
        a     b
```

## 三、直觉表征（类比）

> **Rolle**：从A点出发又回到同高度，必有"转折点"

---

## 四、计算表征（算法）

```python
def rolle_find_c(f, f_prime, a, b, n=1000):
    for x in np.linspace(a, b, n):
        if abs(f_prime(x)) < 1e-10:
            return x
```
