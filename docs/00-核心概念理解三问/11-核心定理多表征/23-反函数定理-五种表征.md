## 一、符号表征（形式化）

### 1.1 严格陈述

**反函数定理**：设f: ℝⁿ → ℝⁿ在a点C¹，且Df(a)可逆，则存在a的邻域U和f(a)的邻域V，使得f|_U: U → V是双射，且反函数f⁻¹: V → U也是C¹，且：

$$D(f^{-1})(f(a)) = [Df(a)]^{-1}$$

### 1.2 一维情形

若f'(a) ≠ 0，则f在a附近局部可逆。

### 1.3 Lean 4 形式化

```lean
import Mathlib.Analysis.Calculus.InverseFunctionTheorem

theorem exists_deriv_at_section {f : E → F} {f' : E →L[𝕜] F}
    (hf : HasStrictFderivAt f f' a) (hf' : Function.Bijective f') :
    ∃ (g : F → E) (hg : ContinuousAt g (f a)),
      f (g y) = y ∧ g (f a) = a ∧
      HasStrictFderivAt g (f'.symm : F →L[𝕜] E) (f a) := by
  exact HasStrictFderivAt.to_localInverse hf hf'

```

## 二、几何表征（可视化）

### 2.1 局部双射

```text
    U (a的邻域)          V (f(a)的邻域)
    ┌──────┐            ┌──────┐
    │  a   │  f         │ f(a) │
    │  •   │ ────────>  │  •   │
    │      │            │      │
    └──────┘            └──────┘
         f⁻¹ <────────

f在U上可逆，且f⁻¹连续可微

```

### 2.2 一维例子

```text
f(x) = x² 在 x > 0：
f'(x) = 2x > 0 ⟹ 局部可逆
f⁻¹(y) = √y

```

## 三、直觉表征（类比）

### 3.1 一句话解释

> **反函数定理**：如果映射在一点"不折叠"（导数可逆），则在该点附近可以"反转"

### 3.2 地图类比

- f：地图投影
- Df可逆：投影不"压扁"（保持维度）
- 反函数：可以"反投影"回去

### 3.3 为什么需要可逆？

可逆 ⟹ 局部"一对一"，才能定义反函数

## 四、计算表征（算法）

### 4.1 Python实现

```python
import numpy as np
from scipy.optimize import fsolve

def inverse_function_theorem(f, df, a, y_target, tol=1e-6):
    """
    使用反函数定理找f⁻¹(y_target)
    f: 函数
    df: 导数（Jacobi矩阵）
    a: 初始点
    """
    # 检查Df(a)可逆
    J = df(a)
    if np.linalg.det(J) == 0:
        raise ValueError("导数不可逆")

    # 初始猜测：线性近似
    x0 = a + np.linalg.solve(J, y_target - f(a))

    # 牛顿法精化
    def F(x):
        return f(x) - y_target

    x_sol = fsolve(F, x0, fprime=df)
    return x_sol

# 示例：f(x,y) = (x²+y, x+y²)
def f(x):
    return np.array([x[0]**2 + x[1], x[0] + x[1]**2])

def df(x):
    return np.array([[2*x[0], 1], [1, 2*x[1]]])

a = np.array([1.0, 1.0])
y_target = np.array([2.0, 2.0])
x_inv = inverse_function_theorem(f, df, a, y_target)
print(f"f⁻¹({y_target}) ≈ {x_inv}")

```

## 五、范畴表征（抽象）

### 5.1 范畴视角

```text
反函数定理 = 局部同构
├─ f: M → N (流形间的映射)
├─ Df可逆 ⟹ f是局部微分同胚
└─ 在微分流形范畴中：局部可逆

```
