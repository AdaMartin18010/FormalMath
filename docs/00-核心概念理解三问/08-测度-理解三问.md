## 一、为什么？(WHY)

### 1.1 历史动机

```text
        [测度概念的起源]
                │
    ┌───────────┼───────────┐
    │           │           │
[长度推广]   [积分基础]   [概率公理化]
    │           │           │
Cantor      Lebesgue    Kolmogorov
1880s       1902        1933
    │           │           │
点集大小    积分的域    概率测度
```

**核心危机**：Riemann积分的局限

```text
Dirichlet函数: D(x) = 1 if x∈ℚ, 0 if x∉ℚ

问题：∫₀¹ D(x)dx = ?

├─ Riemann积分：不存在！（上下和不等）
├─ 但直觉上：ℚ"太小"，积分应该是0
└─ 需要：更好的"大小"概念 → 测度
```

### 1.2 为什么这样定义？

**测度的公理**：

| 公理 | 内容 | 为什么需要？ |
|------|------|--------------|
| **非负性** | μ(A) ≥ 0 | "大小"非负 |
| **空集为零** | μ(∅) = 0 | 无→大小为0 |
| **可数可加** | μ(∪Aₙ) = Σμ(Aₙ) | 可分割成部分 |

**为什么是σ-代数而不是所有子集？**

```text
Vitali集：存在ℝ的子集不可测！

├─ 假设所有子集可测
├─ 构造Vitali集V（用AC）
├─ 推出矛盾：μ([0,1])应该是0或∞
└─ 结论：必须限制可测集的范围
```

### 1.3 核心问题

> **测度要回答的核心问题**：如何给集合赋予合理的"大小"？

- 推广"长度/面积/体积"
- 为积分提供基础
- 公理化概率

## 二、是什么？(WHAT)

### 2.1 正式定义

**σ-代数**：设X是集合，Σ ⊆ P(X)是σ-代数当且仅当：

1. X ∈ Σ
2. A ∈ Σ ⟹ Aᶜ ∈ Σ
3. {Aₙ}ₙ∈ℕ ⊆ Σ ⟹ ∪ₙAₙ ∈ Σ

**测度**：设(X, Σ)是可测空间，μ: Σ → [0,∞]是测度当且仅当：

1. μ(∅) = 0
2. 若{Aₙ}两两不交，则μ(∪ₙAₙ) = Σₙμ(Aₙ)

**Lean 4形式化**：

```lean
class MeasureSpace (α : Type*) extends MeasurableSpace α where
  volume : Measure α

class Measure (α : Type*) [MeasurableSpace α] where
  measureOf : Set α → ℝ≥0∞
  empty : measureOf ∅ = 0
  mono : s ⊆ t → measureOf s ≤ measureOf t
  iUnion_nat : (∀ i j, i ≠ j → Disjoint (s i) (s j)) →
    measureOf (⋃ i, s i) = ∑' i, measureOf (s i)
```

### 2.2 直觉理解

**一句话本质**：
> 测度 = 集合的"大小"函数，满足可数可加性

**直觉类比**：

```text
测度 ≈ 天平称重
├─ 物体的质量（非负）
├─ 空盘子质量为0
├─ 多个物体的总质量=各部分之和
└─ 可以处理无穷多个微小部分
```

**Lebesgue测度**：

```text
ℝ上的Lebesgue测度：
├─ 区间(a,b)的测度 = b-a（长度）
├─ 可数集测度 = 0（ℚ的测度是0）
├─ Cantor集测度 = 0（但不可数！）
└─ 推广到ℝⁿ：体积
```

### 2.3 多重表征

| 表征类型 | 内容 | 示例 |
|----------|------|------|
| **长度表征** | 区间长度的推广 | Lebesgue测度 |
| **概率表征** | 事件的可能性 | P: Σ → [0,1] |
| **质量表征** | 分布的密度 | ∫f dμ |
| **计数表征** | 离散点的个数 | 计数测度 |

### 2.4 典型例子

| 测度 | 空间 | μ(A) |
|------|------|------|
| **Lebesgue** | ℝⁿ | 体积 |
| **计数测度** | 任意集合 | \|A\| |
| **Dirac** | X, x₀∈X | 1 if x₀∈A, 0 otherwise |
| **概率测度** | (Ω,Σ,P) | P(A) ∈ [0,1] |
| **Hausdorff** | ℝⁿ | 分形维数 |

**零测集**：

| 集合 | Lebesgue测度 | 特点 |
|------|--------------|------|
| ℚ ∩ [0,1] | 0 | 可数但稠密 |
| Cantor集 | 0 | 不可数！ |
| 光滑曲线 | 0 (在ℝ²中) | 维数更低 |

### 4.2 常见误解

| 误解 | 正确理解 |
|------|----------|
| "测度=长度" | 测度是更抽象的概念 |
| "所有集合可测" | Vitali集不可测 |
| "零测集=空集" | 零测集可以不可数（Cantor集）|
| "可数可加=有限可加" | 可数可加更强 |
| "测度有限" | 测度可以取∞ |
