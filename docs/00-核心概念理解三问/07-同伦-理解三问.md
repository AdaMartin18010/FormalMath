## 一、为什么？(WHY)

### 1.1 历史动机

```text
        [同伦概念的起源]
                │
    ┌───────────┼───────────┐
    │           │           │
[路径等价]   [映射分类]   [形变]
    │           │           │
Poincaré    Brouwer     连续变形
1895        1910s       的直觉
    │           │           │
基本群      不动点定理   橡皮泥几何
```

**核心问题**：什么时候两个映射"本质相同"？

- **同胚**太强：要求双射
- **需要更弱的等价**：允许"连续变形"
- **直觉**：可以通过连续变形从一个映射变到另一个

### 1.2 为什么这样定义？

**同伦的定义**：

```text
f ≃ g 当且仅当存在连续映射 H: X × [0,1] → Y
使得 H(x,0) = f(x), H(x,1) = g(x)

            H
    X×[0,1] ──→ Y
       │
       │  "连续变形"
       │
    t=0: f
    t=1: g
```

**为什么用 [0,1]？**

| 选择 | 原因 |
|------|------|
| 紧致 | 保证良好的拓扑性质 |
| 连通 | 中间状态都存在 |
| 有序 | 提供"时间"参数 |
| 标准 | 便于组合和逆 |

### 1.3 核心问题

> **同伦要回答的核心问题**：拓扑空间之间的映射如何分类？

- 不是按点对点对应分类
- 而是按"能否连续变形"分类
- 捕捉映射的"整体特征"

## 二、是什么？(WHAT)

### 2.1 正式定义

**映射的同伦**：设 f, g: X → Y 是连续映射。f 和 g 同伦（记作 f ≃ g）当且仅当存在连续映射 H: X × [0,1] → Y 使得：

- H(x, 0) = f(x)  ∀x ∈ X
- H(x, 1) = g(x)  ∀x ∈ X

H称为从f到g的**同伦**。

**同伦等价**：空间 X 和 Y 同伦等价（X ≃ Y）当且仅当存在连续映射 f: X → Y 和 g: Y → X 使得：

- g ∘ f ≃ id_X
- f ∘ g ≃ id_Y

**Lean 4形式化**：

```lean
structure ContinuousMap.Homotopy (f g : C(X, Y)) where
  toFun : X × I → Y
  continuous_toFun : Continuous toFun
  map_zero_left : ∀ x, toFun (x, 0) = f x
  map_one_left : ∀ x, toFun (x, 1) = g x
```

### 2.2 直觉理解

**一句话本质**：
> 同伦 = 映射之间的"连续变形" = 映射空间中的道路

**橡皮泥直觉**：

```text
想象X是橡皮泥，Y是目标空间

f: 把橡皮泥放入Y的一种方式
g: 另一种方式

f ≃ g ⟺ 可以连续地从f"揉"成g
        （不撕裂、不粘合）
```

**道路直觉**：

```text
同伦H是映射空间C(X,Y)中的一条道路

    C(X,Y)
      │
      │  H
      ↓
   f ─────→ g
     "道路"
```

### 2.3 多重表征

| 表征类型 | 内容 |
|----------|------|
| **参数表征** | H: X × [0,1] → Y |
| **道路表征** | H: [0,1] → C(X,Y)（映射空间中的道路）|
| **柱面表征** | 通过映射柱面 M_f 理解 |
| **范畴表征** | 2-态射（高阶范畴） |

### 2.4 典型例子

**同伦的映射**：

| f | g | 同伦 H |
|---|---|--------|
| id: ℝⁿ→ℝⁿ | 0: x↦0 | H(x,t)=(1-t)x |
| id: S¹→S¹ | id | H(x,t)=x（平凡同伦）|
| f: S¹→S¹, z↦z | g: z↦z² | 不同伦！（度数不同）|

**同伦等价**：

| X | Y | 同伦等价？ |
|---|---|-----------|
| ℝⁿ | {点} | ✅（ℝⁿ可缩）|
| S¹ | {点} | ❌（基本群不同）|
| 实心环面 | S¹ | ✅ |
| 莫比乌斯带 | S¹ | ✅ |

### 2.5 同伦vs同胚

```text
        [等价关系层次]
              │
    同胚 ⟹ 同伦等价（反之不成立）
              │
    ┌─────────┼─────────┐
    │         │         │
[同胚]    [同伦等价]   [同调等价]
 最强       中等        最弱
    │         │         │
 保持全部   保持同伦   保持同调
 拓扑性质   不变量     不变量
```

| 误解 | 正确理解 |
|------|----------|
| "同伦=同胚" | 同伦等价弱于同胚 |

## 五、同伦不变量

### 5.1 保持的性质

| 不变量 | 说明 |
|--------|------|
| 基本群 π₁ | 闭路的同伦类 |
