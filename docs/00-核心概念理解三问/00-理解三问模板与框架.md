## 📊 直觉来源分类体系

### 一、几何直觉

```text
[几何直觉]
    │
    ├─ 空间直觉：位置、形状、距离
    ├─ 变换直觉：旋转、平移、伸缩
    ├─ 连续直觉：不断裂、可变形
    └─ 维度直觉：点→线→面→体
```

### 二、代数直觉

```text
[代数直觉]
    │
    ├─ 运算直觉：组合、逆、结合
    ├─ 结构直觉：对称、守恒、不变
    ├─ 抽象直觉：共性提取、模式识别
    └─ 等价直觉：本质相同、表现不同
```

### 三、分析直觉

```text
[分析直觉]
    │
    ├─ 逼近直觉：越来越接近
    ├─ 无穷小直觉：微观局部
    ├─ 积累直觉：无穷多小量累加
    └─ 极限直觉：过程的终点
```

### 四、组合直觉

```text
[组合直觉]
    │
    ├─ 计数直觉：数有多少种方式
    ├─ 递归直觉：大问题分解为小问题
    ├─ 对称直觉：不同角度看相同
    └─ 构造直觉：从简单搭建复杂
```

## 🗺️ 认知地图格式规范

### 概念形成路径图

```text
        [概念认知地图]
              │
    ┌─────────┼─────────┐
    │         │         │
[动机层]   [核心层]   [扩展层]
    │         │         │
├─问题背景 ├─正式定义 ├─推广
├─直觉来源 ├─关键性质 ├─变体
├─历史     ├─核心定理 ├─应用
```

### 理解层次模型

```text
L0 [操作层] ─ 能使用公式计算
      │
L1 [结构层] ─ 理解内部结构
      │
L2 [关系层] ─ 看到与其他概念的联系
      │
L3 [创造层] ─ 能提出新问题、发现新联系
```
