## 🎯 学习路径建议

### 路径1：代数方向

```text
集合 → 函数 → 二元运算 → 群 → 环 → 域 → 模
                          ↓
                       Galois理论
```

### 路径2：分析方向

```text
实数 → 数列 → 极限 → 连续 → 导数 → 积分 → 测度论 → 泛函分析
```

### 路径3：拓扑方向

```text
集合 → 度量空间 → 拓扑空间 → 连续/紧致/连通 → 基本群 → 同调论
```

### 路径4：几何方向

```text
拓扑空间 → 流形 → 切丛 → 联络 → 曲率 → Gauss-Bonnet
```
