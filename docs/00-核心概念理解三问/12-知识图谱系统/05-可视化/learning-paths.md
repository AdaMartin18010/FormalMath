## 🔧 个性化路径生成算法

### 输入
- 当前知识状态（已掌握概念集合）
- 目标概念/定理

### 算法

```python
def generate_learning_path(known_concepts, target):
    """生成从当前知识到目标的最短学习路径"""

    # 1. 获取目标的所有前置依赖
    all_prereqs = get_all_prerequisites(target)

    # 2. 移除已知概念
    to_learn = all_prereqs - known_concepts

    # 3. 拓扑排序（按依赖关系）
    sorted_concepts = topological_sort(to_learn)

    # 4. 按难度分层
    layers = group_by_difficulty(sorted_concepts)

    # 5. 生成推荐顺序
    path = []
    for layer in layers:
        path.extend(prioritize_by_importance(layer))

    return path
```
