## 一、为什么？(WHY)

### 1.1 历史动机

```text
        [群概念的历史起源]
                │
    ┌───────────┼───────────┐
    │           │           │
[方程求解]  [几何对称]   [数论问题]
Galois      Klein        Gauss
    │           │           │
五次方程    变换群      模n剩余类
不可解性    Erlangen纲领 二次互反律
```

**核心问题**：为什么五次及以上方程没有根式解？

- **Galois (1832)**: 发现方程的可解性与其"对称性"有关
- 方程的根之间的"置换"构成一个代数结构
- 这个结构的性质决定了方程是否可解

**几何角度 (Klein, 1872)**：

- 什么是"几何"？研究在某种变换下不变的性质
- 欧氏几何 = 在旋转、平移、反射下不变
- 变换的集合构成群！

### 1.2 为什么这样定义？

**群的四条公理：**

| 公理 | 内容 | 为什么需要？ |
|------|------|--------------|
| **封闭性** | a·b ∈ G | 保证运算在集合内完成 |
| **结合律** | (a·b)·c = a·(b·c) | 保证运算顺序不重要 |
| **单位元** | ∃e: e·a = a·e = a | 提供"什么都不做"的操作 |
| **逆元** | ∀a ∃a⁻¹: a·a⁻¹ = e | 保证每个操作可以"撤销" |

**如果去掉某条公理？**

```text
去掉封闭性 → 不是代数结构
去掉结合律 → 拟群、Loop（更弱结构）
去掉单位元 → 半群
去掉逆元   → 幺半群
```

### 1.3 核心问题

> **群要回答的核心问题**：如何刻画"对称性"的代数结构？

- 对称性 = 保持某种结构不变的变换
- 变换可以复合（乘法）
- 变换可以撤销（逆）
- 存在"什么都不做"（单位元）

## 二、是什么？(WHAT)

### 2.1 正式定义

**定义**：群是一个二元组 (G, ·)，其中G是非空集合，·: G × G → G 是二元运算，满足：

1. **结合律**：∀a,b,c ∈ G, (a·b)·c = a·(b·c)
2. **单位元**：∃e ∈ G, ∀a ∈ G, e·a = a·e = a
3. **逆元**：∀a ∈ G, ∃a⁻¹ ∈ G, a·a⁻¹ = a⁻¹·a = e

**Lean 4形式化**：

```lean
class Group (G : Type*) extends Monoid G, Inv G where
  inv_mul_cancel : ∀ a : G, a⁻¹ * a = 1
```

### 2.2 直觉理解

**一句话本质**：
> 群是"可逆操作"的集合，操作可以组合，遵循结合律。

**几何直觉**：

```text
想象一个正方形：
        A ─── B
        │     │
        D ─── C

保持正方形"看起来一样"的操作：
├─ 旋转90°、180°、270°
├─ 关于对角线翻转
├─ 关于中线翻转
├─ 什么都不做
共8个操作 → 二面体群D₄
```

**代数直觉**：

```text
整数加法 (ℤ, +)：
├─ 任意两个整数相加还是整数（封闭）
├─ (a+b)+c = a+(b+c)（结合）
├─ 0是单位元：a+0 = a
├─ -a是a的逆元：a+(-a) = 0
```

### 2.3 多重表征

| 表征类型 | 内容 | 示例 |
|----------|------|------|
| **符号表征** | (G, ·) 满足群公理 | (ℤ, +), (ℝ*, ×) |
| **几何表征** | 对称变换群 | 正多边形的对称群 |
| **矩阵表征** | 可逆矩阵群 | GL(n,ℝ), SO(n) |
| **置换表征** | 集合的置换 | Sₙ对称群 |
| **生成元表征** | 由生成元和关系给出 | ⟨a,b \| a²=b²=(ab)³=e⟩ |

### 2.4 典型例子

**最简单的例子**：

| 群 | 元素 | 运算 | 阶 |
|----|------|------|-----|
| 平凡群 | {e} | e·e=e | 1 |
| ℤ/2ℤ | {0,1} | 模2加法 | 2 |

**典型例子**：

| 群 | 描述 | 特点 |
|----|------|------|
| (ℤ, +) | 整数加法群 | 无限循环群 |
| (ℚ*, ×) | 非零有理数乘法群 | 无限阿贝尔群 |
| Sₙ | n元素置换群 | \|Sₙ\| = n!, 非交换(n≥3) |
| GL(n,ℝ) | n阶可逆实矩阵 | 无限非交换群 |
| SO(3) | 三维旋转群 | 李群，描述物理旋转 |

**反例/边界情况**：

| 结构 | 为什么不是群？ |
|------|--------------|
| (ℕ, +) | 缺少逆元（没有负数） |
| (ℤ, ×) | 缺少逆元（1/2不是整数） |
| (M₂(ℝ), ×) | 缺少逆元（奇异矩阵无逆） |
| 八元数 (𝕆, ×) | 不满足结合律 |

### 4.2 常见误解与障碍

| 误解 | 正确理解 |
|------|----------|
| "群必须是数的集合" | 群可以是任何满足公理的集合 |
| "群运算必须是乘法" | 运算可以是任何二元运算（+、∘、*等） |
| "所有群都是交换的" | 非交换群很常见（Sₙ, GL(n)等） |
| "子群一定是正规子群" | 正规子群需要额外条件 gHg⁻¹=H |
| "群同态保持所有性质" | 只保持群结构，不保持其他性质 |
