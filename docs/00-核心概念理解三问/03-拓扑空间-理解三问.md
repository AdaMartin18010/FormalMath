## 一、为什么？(WHY)

### 1.1 历史动机

```text

                │
    ┌───────────┼───────────┐
    │           │           │

Weierstrass  Poincaré    Hausdorff

```

**核心问题**：什么是"连续"和"接近"的本质？

- **度量空间**：用距离定义接近 → 但很多情况不需要距离
- **例子**：函数空间上不同"收敛"概念不能统一用距离描述
- **需求**：抽象出"接近性"的本质结构

### 1.2 为什么这样定义？

**拓扑公理（开集系统）**：

设 X 是集合，τ ⊆ P(X) 称为 X 上的拓扑，如果：

| 公理 | 内容 | 为什么需要？ |
|------|------|--------------|
| **T1** | ∅, X ∈ τ | 极端情况：空集开、全空间开 |
| **T2** | 有限交封闭 | 有限个开集的公共部分仍"开" |
| **T3** | 任意并封闭 | 任意多开集的合并仍"开" |

**为什么用开集定义？**

```text
[问题] 如何抽象"邻域"概念？

度量空间：B(x,ε) = {y : d(x,y) < ε}
    │
    ▼
开集 = 开球的并
    │
    ▼
[洞察] 开集满足的性质：
├─ 空集和全空间是开的
├─ 有限交是开的
├─ 任意并是开的
    │
    ▼
直接用这些性质作为公理！
```

### 1.3 核心问题

> **拓扑空间要回答的核心问题**：不用距离，如何定义"接近"和"连续"？

- 什么是"邻近"？→ 邻域/开集
- 什么是"连续"？→ 开集原像是开集
- 什么是"不变性"？→ 同胚下的不变量

## 二、是什么？(WHAT)

### 2.1 正式定义

**定义**：拓扑空间是一个有序对 (X, τ)，其中：

- X 是一个集合
- τ ⊆ P(X) 满足：
  1. ∅ ∈ τ, X ∈ τ
  2. U₁, U₂ ∈ τ ⟹ U₁ ∩ U₂ ∈ τ
  3. {Uᵢ}ᵢ∈I ⊆ τ ⟹ ∪ᵢUᵢ ∈ τ

τ 中的元素称为**开集**。

**Lean 4形式化**：

```lean
class TopologicalSpace (X : Type*) where
  IsOpen : Set X → Prop
  isOpen_univ : IsOpen Set.univ
  isOpen_inter : ∀ s t, IsOpen s → IsOpen t → IsOpen (s ∩ t)
  isOpen_sUnion : ∀ S, (∀ s ∈ S, IsOpen s) → IsOpen (⋃₀ S)
```

### 2.2 直觉理解

**一句话本质**：
> 拓扑空间 = 定义了"开集"概念的集合 = 有"邻近"概念的空间

**几何直觉**：

```text
开集 ≈ "没有边界"的区域
    │
    ├─ 内部每个点都有"活动空间"
    │
    ├─ 不包含自己的边界
    │
    └─ 可以向各方向"稍微移动"而不离开

闭集 = 开集的补 ≈ 包含边界的区域
```

**邻域直觉**：

```text
x的邻域 = 包含x的某个开集的超集
    │
    ├─ "x周围的区域"
    │
    └─ 不需要知道"多大"，只需要"存在这样的区域"
```

### 2.3 多重表征

| 表征类型 | 内容 |
|----------|------|
| **开集表征** | τ = {开集} 满足三公理 |
| **闭集表征** | 闭集系：任意交、有限并封闭 |
| **邻域表征** | 每点的邻域系满足特定公理 |
| **闭包表征** | Kuratowski闭包公理 |
| **内部表征** | 内部算子公理 |
| **基表征** | 开集基生成的拓扑 |

### 2.4 典型例子

**最简单的例子**：

| 拓扑 | 集合X | 开集τ |
|------|-------|-------|
| 平凡拓扑 | 任意X | {∅, X} |
| 离散拓扑 | 任意X | P(X) (所有子集) |

**典型例子**：

| 拓扑空间 | 描述 | 特点 |
|----------|------|------|
| (ℝ, 欧氏) | 开区间生成 | 最熟悉的拓扑 |
| (ℝ, 余有限) | 补有限的集合开 | 非Hausdorff |
| (ℝ, 下限) | [a,b)型集合为基 | 更细的拓扑 |
| Sⁿ (n维球面) | 诱导拓扑 | 紧致流形 |

**反例/边界情况**：

```text
余有限拓扑（ℝ上）：
├─ 开集 = ∅ 或 补集有限
├─ 任意两个非空开集相交非空
├─ 不是Hausdorff的
└─ 序列收敛性"太弱"
```

### 4.2 常见误解与障碍

| 误解 | 正确理解 |
|------|----------|
| "开集必须是开区间" | 开集是满足公理的任意集合 |
| "闭集是闭的开集" | 闭集=开集的补，与"闭合"无关 |
| "点都是闭的" | 只在T₁空间中成立 |
| "序列收敛唯一确定拓扑" | 仅在第一可数空间 |
| "拓扑必须来自度量" | 很多拓扑不能度量化 |
