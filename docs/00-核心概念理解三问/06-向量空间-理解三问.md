## 一、为什么？(WHY)

### 1.1 历史动机

```text
        [向量空间的起源]
                │
    ┌───────────┼───────────┐
    │           │           │
[几何向量]   [方程组]    [函数空间]
    │           │           │
Hamilton    Cayley     Fourier
四元数      矩阵理论    级数展开
    │           │           │
    └───────────┼───────────┘
                │
        Grassmann/Peano
        公理化(1888)
```

**核心问题**：如何统一描述"可以相加和数乘"的对象？

- **几何**：平面/空间中的箭头（位移）
- **代数**：方程组的解集
- **分析**：函数的线性组合
- **发现**：这些不同对象有相同的代数结构！

### 1.2 为什么这样定义？

**向量空间公理**：

| 公理类型 | 公理 | 为什么需要？ |
|----------|------|--------------|
| **加法群** | (V,+)是阿贝尔群 | 向量可以相加 |
| **数乘相容** | 1·v = v | 单位元作用 |
| **数乘结合** | a(bv) = (ab)v | 与域乘法相容 |
| **分配律1** | a(u+v) = au+av | 数乘对加法分配 |
| **分配律2** | (a+b)v = av+bv | 加法对数乘分配 |

**为什么需要"域"？**

```text
数乘需要"除法"：
├─ 如果 av = 0 且 a ≠ 0，希望 v = 0
├─ 需要 a⁻¹ 存在
└─ 所以系数必须来自域（或至少整环）
```

### 1.3 核心问题

> **向量空间要回答的核心问题**：什么是"线性结构"的本质？

- 可加性：对象可以相加
- 可伸缩性：对象可以乘以系数
- 这两个操作相容

## 二、是什么？(WHAT)

### 2.1 正式定义

**定义**：设F是域，F上的向量空间是一个四元组(V, +, ·, F)满足：

1. (V, +)是阿贝尔群
2. ·: F × V → V（数乘）
3. 对所有 a,b ∈ F, u,v ∈ V:
   - a(u+v) = au + av
   - (a+b)v = av + bv
   - a(bv) = (ab)v
   - 1·v = v

**Lean 4形式化**：

```lean
class Module (R : Type*) (M : Type*) [Semiring R] [AddCommMonoid M] extends
    SMul R M where
  one_smul : ∀ x : M, (1 : R) • x = x
  mul_smul : ∀ (r s : R) (x : M), (r * s) • x = r • s • x
  smul_add : ∀ (r : R) (x y : M), r • (x + y) = r • x + r • y
  add_smul : ∀ (r s : R) (x : M), (r + s) • x = r • x + s • x
```

### 2.2 直觉理解

**一句话本质**：
> 向量空间 = 可以做"线性组合"的对象集合

**几何直觉**：

```text
ℝ² 中的向量：
        y
        ↑  v=(2,3)
        │ ↗
        │/
        └──────→ x

v = 2·e₁ + 3·e₂ （线性组合）

向量可以：
├─ 相加（平行四边形法则）
├─ 数乘（伸缩）
└─ 任意线性组合
```

**抽象直觉**：

```text
向量空间 ≈ "平直"的空间
├─ 没有"弯曲"
├─ 原点是特殊的（零向量）
├─ 通过原点的直线是子空间
└─ 所有方向"等价"
```

### 2.3 多重表征

| 表征类型 | 内容 | 示例 |
|----------|------|------|
| **坐标表征** | Fⁿ列向量 | ℝ³ |
| **函数表征** | F→F的函数 | 多项式空间 |
| **矩阵表征** | m×n矩阵 | M_{m×n}(F) |
| **几何表征** | 箭头/位移 | 物理向量 |
| **抽象表征** | 满足公理的集合 | 任意向量空间 |

### 2.4 典型例子

**有限维**：

| 空间 | 维数 | 说明 |
|------|------|------|
| {0} | 0 | 平凡空间 |
| F | 1 | 域本身 |
| Fⁿ | n | 坐标空间 |
| M_{m×n}(F) | mn | 矩阵空间 |
| F[x]_{≤n} | n+1 | 次数≤n的多项式 |

**无限维**：

| 空间 | 说明 |
|------|------|
| F[x] | 所有多项式 |
| F^∞ | 有限支撑序列 |
| C[0,1] | 连续函数 |
| L²([0,1]) | 平方可积函数 |

**反例**：

| 结构 | 为什么不是向量空间？ |
|------|---------------------|
| ℤⁿ (整数系数) | ℤ不是域（无乘法逆） |
| 正实数 | 加法不封闭 |
| 单位球 | 数乘不封闭 |

### 3.3 统一视角

**范畴论视角**：

```text
Vect_F = F-向量空间范畴
├─ 对象：F-向量空间
├─ 态射：F-线性映射
├─ 积：直积V×W
├─ 余积：直和V⊕W（=直积！）

特点：Vect_F是阿贝尔范畴
```

**线性代数的核心定理**：

```text
        [向量空间结构定理]
                │
    ┌───────────┼───────────┐
    │           │           │
[基底存在]  [维数不变]   [同构分类]
    │           │           │
每个向量空间  同一空间    V ≅ W ⟺
有基底       不同基底    dim V = dim W
             大小相同
```

### 3.4 维数的重要性

```text
        [维数决定一切]
              │
    有限维向量空间完全由维数决定
              │
    dim V = n ⟹ V ≅ Fⁿ
              │
    ┌─────────┼─────────┐
    │         │         │
[子空间]   [商空间]   [线性映射]
    │         │         │
dim W ≤ n  dim V/W    rank + nullity
           = n-dim W   = n
```

### 4.2 常见误解

| 误解 | 正确理解 |
|------|----------|
| "向量=箭头" | 向量可以是函数、矩阵等 |
| "维数=坐标个数" | 维数是基底大小 |
| "所有空间有限维" | 函数空间常无限维 |
| "基底唯一" | 基底有无穷多种选择 |
| "内积是标配" | 内积是额外结构 |
