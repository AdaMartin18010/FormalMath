## 一、模形式的定义与基本性质

### 1.1 模群与全模群

**定义 1.1**（全模群）：**全模群**（Full Modular Group）定义为：
$$\text{SL}(2, \mathbb{Z}) = \left\{ \gamma = \begin{pmatrix} a & b \\ c & d \end{pmatrix} : a, b, c, d \in \mathbb{Z}, \ ad - bc = 1 \right\}$$

**定理 1.2**（生成元）：$\text{SL}(2, \mathbb{Z})$ 由两个元素生成：
$$S = \begin{pmatrix} 0 & -1 \\ 1 & 0 \end{pmatrix}, \quad T = \begin{pmatrix} 1 & 1 \\ 0 & 1 \end{pmatrix}$$

满足关系：$S^2 = (ST)^3 = -I$，且这是 $\text{PSL}(2, \mathbb{Z}) = \text{SL}(2, \mathbb{Z})/\{\pm I\}$ 的完整表示。

**定义 1.3**（上半平面）：**Poincaré上半平面**：
$$\mathbb{H} = \{ z \in \mathbb{C} : \text{Im}(z) > 0 \}$$

**模群作用**：$\gamma(z) = \frac{az + b}{cz + d}$，满足 $\text{Im}(\gamma(z)) = \frac{\text{Im}(z)}{|cz + d|^2}$。

### 1.2 权k模形式

**定义 1.4**（权k模形式）：函数 $f: \mathbb{H} \to \mathbb{C}$ 称为**权为 $k$ 的（全纯）模形式**，如果：

1. **全纯性**：$f$ 在 $\mathbb{H}$ 上全纯
2. **模变换**：对所有 $\gamma = \begin{pmatrix} a & b \\ c & d \end{pmatrix} \in \text{SL}(2, \mathbb{Z})$，
   $$f(\gamma(z)) = (cz + d)^k f(z)$$
3. **尖点条件**：$f$ 在 $\infty$ 处全纯（Fourier展开 $f(z) = \sum_{n=0}^{\infty} a_n q^n$，$q = e^{2\pi i z}$）

**定义 1.5**（尖点形式）：若 $a_0 = 0$，称 $f$ 为**尖点形式**（cusp form）。

**例 1.6**（Eisenstein级数）：对偶数 $k \geq 4$，
$$G_k(z) = \sum_{(m,n) \neq (0,0)} \frac{1}{(mz + n)^k}$$

是权为 $k$ 的模形式，满足 $G_k(\infty) = 2\zeta(k)$。

**归一化Eisenstein级数**：
$$E_k(z) = \frac{1}{2\zeta(k)} G_k(z) = 1 - \frac{2k}{B_k} \sum_{n=1}^{\infty} \sigma_{k-1}(n) q^n$$

其中 $B_k$ 是Bernoulli数，$\sigma_{k-1}(n) = \sum_{d|n} d^{k-1}$。

**定理 1.7**（模形式空间的维数）：
$$\dim M_k(\text{SL}(2, \mathbb{Z})) = \begin{cases} \lfloor k/12 \rfloor & k \equiv 2 \pmod{12} \\ \lfloor k/12 \rfloor + 1 & k \not\equiv 2 \pmod{12} \end{cases}$$

## 二、Hecke算子与特征形式

### 2.1 Hecke算子的代数结构

**定义 2.1**（Hecke算子 $T_n$）：对正整数 $n$，**Hecke算子** $T_n$ 作用于权 $k$ 模形式空间：
$$(T_n f)(z) = n^{k-1} \sum_{d|n} d^{-k} \sum_{b=0}^{d-1} f\left(\frac{nz + bd}{d^2}\right)$$

等价地，对素数 $p$：
$$(T_p f)(z) = p^{k-1} f(pz) + \frac{1}{p} \sum_{j=0}^{p-1} f\left(\frac{z+j}{p}\right)$$

**定理 2.2**（Hecke算子的性质）：

1. $T_n$ 保持 $M_k$ 和 $S_k$（尖点形式空间）
2. Hecke算子交换：$T_m T_n = T_n T_m$
3. 乘性关系：$T_{mn} = T_m T_n$ 当 $(m,n) = 1$
4. 对素数 $p$：$T_{p^r} T_p = T_{p^{r+1}} + p^{k-1} T_{p^{r-1}}$

### 2.2 本征形式与乘性

**定义 2.3**（Hecke本征形式）：模形式 $f = \sum a_n q^n$ 称为**归一化Hecke本征形式**，如果：

1. $a_1 = 1$
2. 对所有 $n$，$T_n f = a_n f$

**定理 2.4**（本征形式的乘性）：设 $f$ 是归一化Hecke本征形式，则Fourier系数满足：

1. $a_{mn} = a_m a_n$，当 $(m,n) = 1$
2. $a_{p^{r+1}} = a_p a_{p^r} - p^{k-1} a_{p^{r-1}}$

**例 2.5**（判别式模形式）：
$$\Delta(z) = q \prod_{n=1}^{\infty} (1 - q^n)^{24} = \sum_{n=1}^{\infty} \tau(n) q^n$$

是权12的尖点形式，Ramanujan的$\tau$函数满足乘性：
$$\tau(mn) = \tau(m)\tau(n), \quad (m,n) = 1$$

**Ramanujan猜想**（Deligne定理）：$|\tau(p)| \leq 2p^{11/2}$，即 $|\tau(n)| \leq d(n) \cdot n^{11/2}$。

## 三、L函数的解析理论

### 3.1 模形式L函数的函数方程

**定义 3.1**（模形式L函数）：对模形式 $f = \sum_{n=1}^{\infty} a_n q^n$，定义**L函数**：
$$L(s, f) = \sum_{n=1}^{\infty} \frac{a_n}{n^s}$$

**定理 3.2**（Euler乘积）：若 $f$ 是归一化Hecke本征形式，则：
$$L(s, f) = \prod_p \frac{1}{1 - a_p p^{-s} + p^{k-1-2s}}$$

**定理 3.3**（函数方程）：设 $f$ 是权 $k$ 尖点形式，定义：
$$\Lambda(s, f) = (2\pi)^{-s} \Gamma(s) L(s, f)$$

则 $\Lambda(s, f)$ 可全纯延拓到 $\mathbb{C}$，且：
$$\Lambda(s, f) = (-1)^{k/2} \Lambda(k-s, f)$$

**证明思路**：

1. Mellin变换：$\Lambda(s, f) = \int_0^{\infty} f(iy) y^{s-1} dy$
2. 模变换 $z \mapsto -1/z$ 给出 $f(i/y) = (iy)^k f(iy)$
3. 分裂积分 $\int_0^{\infty} = \int_0^1 + \int_1^{\infty}$，用变换公式证明解析延拓

### 3.2 Converse定理

**定理 3.4**（Hecke converse定理）：设 $\{a_n\}$ 满足适当增长条件，$L(s) = \sum a_n n^{-s}$ 满足：

1. 函数方程：$\Lambda(s) = (2\pi)^{-s}\Gamma(s)L(s)$ 可解析延拓且 $\Lambda(s) = (-1)^{k/2}\Lambda(k-s)$
2. $L(s)$ 在有限平面全纯且在 $\text{Re}(s) > \sigma$ 绝对收敛

则 $f(z) = \sum a_n q^n$ 是权 $k$ 模形式。

**意义**：L函数的解析性质完全刻画模形式。

## 四、模形式与椭圆曲线的联系

### 4.1 谷山-志村定理

**历史背景**：

- **谷山丰（1955）**：提出椭圆曲线的L函数与模形式相关
- **志村五郎（1950s-60s）**：系统发展复乘理论
- **Weil（1967）**：提出完善猜想形式

**谷山-志村-Weil猜想**：每条定义在 $\mathbb{Q}$ 上的椭圆曲线都是模的（modular），即存在权2模形式 $f$ 使得 $L(E, s) = L(s, f)$。

**定理 4.1**（Wiles-Taylor, 1994）：每条定义在 $\mathbb{Q}$ 上的半稳定椭圆曲线都是模的。

**定理 4.2**（Breuil-Conrad-Diamond-Taylor, 2001）：谷山-志村-Weil猜想完全成立。

### 4.2 权2模形式与椭圆曲线

**构造 4.3**（从模形式到椭圆曲线）：设 $f = \sum a_n q^n$ 是权2归一化Hecke本征形式，$a_n \in \mathbb{Z}$。通过Eichler-Shimura构造，存在椭圆曲线 $E_f$ 满足：
$$a_p(E_f) = a_p$$

即 $L(E_f, s) = L(s, f)$。

**Eichler-Shimura同构**：
$$S_2(\Gamma_0(N)) \cong H^0(X_0(N), \Omega^1) \cong \text{Hom}(J_0(N), \mathbb{C})$$

其中 $J_0(N)$ 是模曲线 $X_0(N)$ 的Jacobian簇。

**例 4.4**：权2本征形式 $f(z) = q \prod_{n=1}^{\infty}(1-q^n)^2(1-q^{11n})^2$ 对应椭圆曲线：
$$E: y^2 + y = x^3 - x^2 - 10x - 20$$
（导子 $N = 11$ 的最小模型）

## 五、Galois表示与Langlands纲领

### 5.1 与模形式关联的Galois表示

**定理 5.1**（Deligne, 1969）：设 $f$ 是权 $k$ 归一化Hecke本征形式，$a_n \in \mathcal{O}_L$。则对每个素数 $\ell$，存在连续Galois表示：
$$\rho_{f,\ell}: \text{Gal}(\overline{\mathbb{Q}}/\mathbb{Q}) \to \text{GL}(2, L_\lambda)$$

其中 $\lambda | \ell$，满足：对 $p \nmid N\ell$，$\rho_{f,\ell}$ 在 $p$ 处非分歧，且：

$$\text{char}(\rho_{f,\ell}(\text{Frob}_p)) = X^2 - a_p X + p^{k-1}$$

**推论 5.2**（Ramanujan猜想的证明）：Deligne证明 $|\rho_{f,\ell}(\text{Frob}_p)| \leq 2p^{(k-1)/2}$，导出 $|a_p| \leq 2p^{(k-1)/2}$。

### 5.2 Langlands纲领的联系

**Langlands纲领的核心思想**：

| 自守表示 | Galois表示 |
|----------|------------|
| $\text{GL}(2, \mathbb{A}_\mathbb{Q})$ 的尖点表示 | 2维Galois表示 |
| L-函数 $L(s, \pi)$ | Artin L-函数 $L(s, \rho)$ |
| Hecke本征形式 | 模Galois表示 |

**定理 5.3**（局部Langlands对应，已证）：对局部域 $F$，存在典范对应：
$$\{\text{GL}(n, F) \text{ 的不可约光滑表示}\} \longleftrightarrow \{n \text{ 维 } W_F \text{-表示}\}$$

其中 $W_F$ 是Weil群。

**应用 5.4**（Serre猜想）：

**Serre猜想**（已证，Khare-Wintenberger, 2008）：任意奇、不可约、模 $\ell$ 的2维Galois表示 $\rho: G_\mathbb{Q} \to \text{GL}(2, \overline{\mathbb{F}}_\ell)$ 来自权 $k$、级 $N$ 的模形式。

**意义**：Serre猜想的证明是Langlands纲领的巨大进展，也证明了谷山-志村定理的更广泛形式。
