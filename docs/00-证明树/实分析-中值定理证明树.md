# 实分析 - 中值定理证明树

**定理**: 中值定理 (Mean Value Theorem)  
**表征类型**: 证明树 (Proof Tree)

---

## 证明树结构

```
中值定理证明
│
├─ 目标: 证明 ∃c∈(a,b), f'(c) = (f(b)-f(a))/(b-a)
│
├─ 核心策略: 构造辅助函数，转化为Rolle定理
│
├─ 步骤1: 构造辅助函数 g(x) = f(x) - [(f(b)-f(a))/(b-a)](x-a)
│  ├─ 性质1: g(a) = f(a) ✓
│  ├─ 性质2: g(b) = f(a) ✓（关键计算）
│  └─ 性质3: g 满足Rolle定理条件
│
├─ 步骤2: 应用Rolle定理
│  └─ ∃c∈(a,b), g'(c) = 0
│
├─ 步骤3: 计算 g'(c) = f'(c) - (f(b)-f(a))/(b-a) = 0
│
└─ 步骤4: 结论 f'(c) = (f(b)-f(a))/(b-a) ✓
```

## 关键洞察

**辅助函数的几何意义**: $g(x)$ 表示 $f(x)$ 与割线之间的垂直距离
