# 代数 - 同态基本定理证明树

**定理**: 第一同构定理 (First Isomorphism Theorem)  
**适用**: MIT 18.701 Algebra I, Week 3-4

---

## 定理陈述

设 $\varphi: G \to H$ 是群同态，则：
$$G / \ker(\varphi) \cong \text{im}(\varphi)$$

---

## 证明树结构

```
第一同构定理证明
│
├─ 目标: 构造同构 ψ: G/ker(φ) → im(φ)
│
├─ 步骤1: 定义映射
│  │
│  └─ ψ(g·ker(φ)) := φ(g)
│
├─ 步骤2: 验证良定性
│  │
│  ├─ 问题: 若 g·ker(φ) = g'·ker(φ)，是否有 φ(g) = φ(g')?
│  │
│  ├─ 条件分析: g⁻¹g' ∈ ker(φ)
│  │
│  ├─ 推导: φ(g⁻¹g') = e_H
│  │
│  ├─ 推导: φ(g)⁻¹φ(g') = e_H
│  │
│  └─ 结论: φ(g) = φ(g') ✓
│
├─ 步骤3: 验证同态性
│  │
│  ├─ ψ((g₁·ker)(g₂·ker)) = ψ(g₁g₂·ker)
│  ├─ = φ(g₁g₂)
│  ├─ = φ(g₁)φ(g₂)
│  └─ = ψ(g₁·ker)ψ(g₂·ker) ✓
│
├─ 步骤4: 验证单射
│  │
│  ├─ 设 ψ(g·ker) = e_H
│  ├─ 即 φ(g) = e_H
│  ├─ 即 g ∈ ker(φ)
│  └─ 即 g·ker = ker (单位元) ✓
│
├─ 步骤5: 验证满射
│  │
│  ├─ 对任意 h ∈ im(φ)，存在 g ∈ G 使得 φ(g) = h
│  └─ 则 ψ(g·ker) = h ✓
│
└─ 结论: ψ 是同构，定理得证
```

---

## 关键洞察

**核心思想**: 商群 $G/\ker(\varphi)$ "消除"了同态的核，使得映射变为单射。

**几何直观**: 将 $G$ 中映射到同一像的元素"粘合"为一个陪集。
