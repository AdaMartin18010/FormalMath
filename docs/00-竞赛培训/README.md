# 数学竞赛培训模块

**最后更新**: 2026年4月4日

---

## 📋 模块概述

本模块为FormalMath项目的数学竞赛培训专题，涵盖国际数学奥林匹克（IMO）、Putnam竞赛、大学生数学竞赛等重要赛事的培训内容。

|------|----------|----------|
| [IMO培训专题](IMO培训专题.md) | IMO代数/几何/数论/组合四大专题 | ★★★★★ |

## 🎯 培训路径

### 初级路径（高中竞赛）

```
竞赛解题策略 → IMO培训专题 → 真题解析集
```

### 高级路径（大学竞赛）

```
竞赛解题策略 → Putnam竞赛准备 → 大学生数学竞赛 → 真题解析集
```
