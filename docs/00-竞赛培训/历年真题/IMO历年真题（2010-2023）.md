# IMO历年真题汇编（2010-2023）

**最后更新**: 2026年4月4日

---

## 📋 概述

本文档汇编2010年至2023年IMO的全部试题，按年份和领域分类。

---

## 2023 IMO (日本千叶)

### 第一天

**第1题** (代数): 确定所有满足以下条件的合数正整数 $n$：若 $d_1 < d_2 < \cdots < d_k$ 是 $n$ 的所有正因数，则存在整数 $0 \leq a_1 < a_2 < \cdots < a_k$ 使得
$$n = d_1 \cdot 2^{a_1} + d_2 \cdot 2^{a_2} + \cdots + d_k \cdot 2^{a_k}$$

**第2题** (几何): 在锐角三角形 $ABC$ 中，$AB < AC$，$\Omega$ 是其外接圆。$M$ 是 $BC$ 的中点。$D$ 和 $E$ 分别在 $\Omega$ 的弧 $AB$ 和弧 $AC$ 上（不含 $C$ 和 $B$），使得 $MD = ME$。直线 $BD$ 和 $CE$ 交于 $P$。证明 $AP$ 过定点。

**第3题** (组合): 一个正整数 $n$ 称为**好的**，如果存在排列 $a_1, a_2, \ldots, a_n$ 使得 $k + a_k$ 对所有 $k = 1, 2, \ldots, n$ 都是完全平方数。求所有好数。

### 第二天

**第4题** (几何): 设 $ABC$ 是一个锐角三角形，$AB < AC$。设 $P$ 和 $Q$ 是 $\Omega$ 上的两点，使得 $\angle ABP = \angle CBQ < \angle ABC$。设 $R$ 是 $\Omega$ 上一点，使得 $PR \parallel AC$。证明 $QR$ 过 $\triangle ABC$ 的垂心。

**第5题** (数论): 求所有函数 $f: \mathbb{Z}^+ \to \mathbb{Z}^+$ 使得对任意正整数 $a, b$：
- $f(ab) = f(a)f(b)$
- $f(a+b) \mid f(a^2+b^2)$

**第6题** (组合/代数): 设 $n$ 是正整数。一个 $n$-元组 $(a_1, \ldots, a_n)$ 称为**好的**如果 $a_1 + 2a_2 + \cdots + na_n$ 是 $n$ 的倍数。求好 $n$-元组的个数。

## 2022 IMO (挪威奥斯陆)

### 第一天

**第1题** (代数): 设 $n$ 是正整数。求所有函数 $f: \mathbb{R} \to \mathbb{R}$ 使得对任意实数 $x, y$：
$$f(x + f(y)) = f(x) + f(y)^n$$

**第2题** (组合): 设 $n$ 是正整数。一个北欧方阵是由 $+$ 和 $-$ 组成的 $n \times n$ 方阵，满足以下条件：不存在任何由 $+$ 组成的 $2 \times 2$ 子方阵，也不存在任何由 $-$ 组成的 $2 \times 2$ 子方阵。求所有可能的 $n$ 值。

**第3题** (几何): 设 $ABC$ 是锐角三角形，$A'$ 是 $A$ 关于 $BC$ 的对称点，$B'$ 是 $B$ 关于 $AC$ 的对称点，$C'$ 是 $C$ 关于 $AB$ 的对称点。设 $P$ 是 $\triangle ABC$ 内一点，使得 $PA', PB', PC'$ 分别与 $BC, CA, AB$ 交于 $D, E, F$。证明若 $PD = PE = PF$，则 $P$ 是 $\triangle ABC$ 的外心。

### 第二天

**第4题** (几何): 设 $ABCDE$ 是凸五边形，满足 $BC = DE$。假设在 $ABCDE$ 内部有一点 $T$ 使得 $TB = TD$，$TC = TE$，且 $\angle ABT = \angle TEA$。设直线 $AB$ 与 $CD$ 交于 $P$，直线 $AE$ 与 $CD$ 交于 $Q$。证明 $PT = QT$。

**第5题** (数论): 求所有三元正整数组 $(a, b, p)$ 使得 $p$ 是素数，$\gcd(a, b) = 1$，且：
$$a^p + b^p = p^p$$

**第6题** (组合): 设 $n$ 是正整数。一个 $n$-tuple $(a_1, a_2, \ldots, a_n)$ 称为**递增的**如果 $a_1 \leq a_2 \leq \cdots \leq a_n$。一个 $n$-tuple $(b_1, b_2, \ldots, b_n)$ 称为**好的**如果存在递增的 $n$-tuple $(c_1, c_2, \ldots, c_n)$ 使得对所有 $i$ 有 $b_i = c_i + i$。求所有好的 $n$-tuple 的个数。

## 2021 IMO (俄罗斯圣彼得堡，线上)

### 第一天

**第1题** (几何): 设 $n \geq 100$ 是整数。Ivan 将 $n, n+1, \ldots, 2n$ 的每个数写在不同的卡片上。然后他将这 $n+1$ 张卡片洗牌并分成两堆。证明至少有一堆中包含两张卡片，其上的数之和是完全平方数。

**第2题** (几何): 证明对任意实数 $x_1, x_2, \ldots, x_n$，有：
$$\sum_{i=1}^{n} \sum_{j=1}^{n} \sqrt{|x_i - x_j|} \leq \sum_{i=1}^{n} \sum_{j=1}^{n} \sqrt{|x_i + x_j|}$$

**第3题** (代数): 设 $D$ 是锐角三角形 $ABC$ 内一点，满足 $\angle ADB = \angle ACB + 90°$ 且 $AC \cdot BD = AD \cdot BC$。
- (a) 求 $\frac{AB \cdot CD}{AC \cdot BD}$
- (b) 证明 $\triangle ACD$ 和 $\triangle BCD$ 的外接圆在 $C$ 处的切线互相垂直

### 第二天

**第4题** (组合): 设 $\Gamma$ 是一个圆，圆心为 $I$，$ABCD$ 是 $\Gamma$ 的内接四边形。设 $E$ 是 $AC$ 与 $BD$ 的交点。设 $F$ 是 $AD$ 与 $BC$ 的交点。证明 $\frac{EF^2}{EI^2} = \frac{FA \cdot FB}{CA \cdot CB}$。

**第5题** (数论): 两个跳蚤最初位于数轴上的整数点 $A$ 和 $B$（$A < B$）。它们按以下规则跳跃：
- 在点 $x$ 的跳蚤可以跳到点 $2x - y$，其中 $y$ 是另一只跳蚤的位置
证明通过有限次跳跃，两只跳蚤可以到达任意给定的整数点 $C$ 和 $D$（$C < D$），且 $D - C = B - A$。

**第6题** (组合): 设 $m \geq 2$ 是整数，$A$ 是有限整数集，$B_1, B_2, \ldots, B_m$ 是 $A$ 的子集。假设对任意 $k = 1, 2, \ldots, m$，$B_k$ 中所有元素之和为 $m^k$。证明 $A$ 至少包含 $m/2$ 个元素。

## 2020 IMO (俄罗斯，线上)

### 第一天

**第1题** (几何): 考虑凸四边形 $ABCD$。点 $P$ 在 $ABCD$ 内部。假设以下等式成立：
$$\angle PAD : \angle PBA : \angle DPA = 1 : 2 : 3 = \angle CBP : \angle BAP : \angle BPC$$
证明 $\angle ADP$ 的内角平分线、$\angle PCB$ 的内角平分线和线段 $AB$ 的垂直平分线三线共点。

**第2题** (组合): 给定实数 $a, b, c, d$ 满足 $a \geq b \geq c \geq d > 0$ 且 $a + b + c + d = 1$。证明：
$$(a + 2b + 3c + 4d) a^a b^b c^c d^d < 1$$

**第3题** (组合): 有 $4n$ 枚硬币，重量分别为 $1, 2, \ldots, 4n$。每枚硬币被染成 $n$ 种颜色之一，每种颜色恰有4枚硬币。证明可以将硬币分成两堆，使得：
- 每堆总重量相等
- 每堆包含每种颜色的硬币恰2枚

### 第二天

**第4题** (几何): 点 $P$ 在三角形 $ABC$ 的外接圆上。设 $D, E, F$ 分别是 $P$ 到 $BC, CA, AB$ 的垂足。证明 $D, E, F$ 共线（Simson线）。

**第5题** (数论): 一个由正整数组成的集合 $S$ 称为**幸运的**，如果存在正整数 $a > 1$ 使得 $|S| = a^2$ 且 $S$ 中所有元素之和也是 $a^2$。证明对任意正整数 $n$，集合 $\{1, 2, \ldots, n\}$ 可以划分为若干幸运的子集。

**第6题** (组合): 证明存在正整数 $M$ 使得以下游戏对任意 $n \geq M$ 都成立：Alice 和 Bob 在 $n \times n$ 棋盘上玩游戏。Alice 先手，轮流在空格上写 $A$ 或 $B$。当所有格子都被填满时，统计行数和列数，如果某行或某列中 $A$ 比 $B$ 多，则 Alice 得1分，否则 Bob 得1分。得分多者获胜。

## 2019 IMO (英国巴斯)

### 第一天

**第1题** (代数): 求所有函数 $f: \mathbb{Z} \to \mathbb{Z}$ 使得对任意整数 $a, b$：
$$f(2a) + 2f(b) = f(f(a+b))$$

**第2题** (几何): 在三角形 $ABC$ 中，$AB = AC$，$\angle A < 60°$。点 $D, E$ 分别在边 $AB, AC$ 上，使得 $BD = DE = EC$。证明 $AD = AE$。

**第3题** (数论/组合): 一个社交圈有 $2019$ 个人。一些人互为朋友（友谊是相互的）。一个**陌生人操作**是选择三个人 $A, B, C$ 使得 $A$ 与 $B, C$ 都是朋友，但 $B$ 与 $C$ 不是朋友，然后删除 $A$ 与 $B$ 的友谊。证明经过有限次陌生人操作后，社交圈可以被划分为若干团（团内两两互为朋友，团间无友谊）。

### 第二天

**第4题** (几何): 求所有函数 $f: \mathbb{R}^+ \to \mathbb{R}^+$ 使得对任意 $x, y > 0$：
$$f(x + f(x) + y) = 2x + f(y)$$

**第5题** (组合): 设 $n$ 是正整数。Harry 有 $n$ 枚硬币，排成一行。每枚硬币正面朝上或反面朝上。Harry 反复执行以下操作：若存在相邻三枚硬币 $C_1, C_2, C_3$ 满足 $C_1$ 和 $C_3$ 相同但都与 $C_2$ 不同，则翻转 $C_2$。证明Harry最多进行 $n-1$ 次操作。

**第6题** (代数): 设 $I$ 是三角形 $ABC$ 的内心。直线 $AI$ 与 $ABC$ 的外接圆交于 $D \neq A$。设 $F$ 和 $E$ 分别是 $B$ 和 $C$ 在 $AI$ 上的投影。证明 $ID^2 = IF \cdot IE$。
