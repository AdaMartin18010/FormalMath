# 竞赛解题策略

**最后更新**: 2026年4月4日

---

## 📋 概述

数学竞赛解题不仅需要扎实的数学知识，更需要系统的解题策略和方法论。本文档总结各类数学竞赛中常用的解题策略和思维方法。

---

## 🧠 通用解题思维

### Polya解题四步法

1. **理解问题**
   - 未知是什么？已知是什么？条件是什么？
   - 能否画图？引入符号？

2. **制定计划**
   - 以前见过类似问题吗？
   - 能否转化为已知问题？
   - 能否分解为子问题？

3. **执行计划**
   - 逐步验证每一步
   - 记录关键思路

4. **回顾反思**
   - 检验结果
   - 寻找更优解法
   - 总结方法

### 核心思维原则

| 原则 | 说明 | 应用场景 |
|------|------|----------|
| 从简单入手 | 先考虑特殊情况 | 复杂问题 |
| 逆向思维 | 从结论倒推 | 证明题 |
| 类比联想 | 联系已知问题 | 新颖问题 |
| 分解组合 | 分解为子问题 | 综合性问题 |

## 🔧 常用解题技巧

### 1. 特殊化与一般化

#### 特殊化
从一般到特殊，寻找规律。

**示例**: 证明 $n^3 - n$ 对所有正整数 $n$ 能被 6 整除。
- 先验证 $n = 1, 2, 3$ 的情况
- 发现规律：$n^3 - n = n(n-1)(n+1)$ 是三个连续整数之积

#### 一般化
从特殊到一般，提升抽象层次。

**示例**: 计算 $1 + 2 + \cdots + n$
- 考虑一般情形：求等差数列和
- 使用配对法或数学归纳法

### 2. 构造法

#### 直接构造
直接构造满足条件的对象。

**示例**: 证明存在无理数 $a, b$ 使得 $a^b$ 是有理数。
- 考虑 $a = \sqrt{2}$，$b = \log_{\sqrt{2}} 3$
- 或分类讨论：若 $\sqrt{2}^{\sqrt{2}}$ 是有理数则得证；若是无理数，则 $(\sqrt{2}^{\sqrt{2}})^{\sqrt{2}} = 2$ 是有理数

#### 反证构造
假设反面成立，推出矛盾。

**示例**: 证明素数有无穷多个（Euclid证法）。
- 假设只有有限个素数 $p_1, p_2, \ldots, p_n$
- 构造 $N = p_1 p_2 \cdots p_n + 1$
- 推出矛盾

### 3. 不变量与单调量

#### 不变量
在变化过程中保持不变的量。

**应用场景**: 组合游戏、几何变换

**示例**: 棋盘覆盖问题中，染色不变量。

#### 单调量
在变化过程中单调变化（递增或递减）的量。

**应用场景**: 极值问题、过程终止性

**示例**: 证明辗转相除法在有限步内终止。
- 余数严格递减且非负

### 4. 极端原理

寻找极端情况（最大、最小、最远等）来推导结论。

**示例**: 平面上有 $n$ 个点，任意两点距离不超过1。证明存在一个半径为 $\frac{\sqrt{3}}{3}$ 的圆覆盖所有点。
- 取距离最大的两点，以其中点为圆心

### 5. 抽屉原理

**基本形式**: $n+1$ 个物体放入 $n$ 个抽屉，至少有一个抽屉有至少2个物体。

**推广形式**: $mn+1$ 个物体放入 $n$ 个抽屉，至少有一个抽屉有至少 $m+1$ 个物体。

**应用场景**: 存在性证明、极值问题

**示例**: 从 $\{1, 2, \ldots, 2n\}$ 中选出 $n+1$ 个数，必有两数互素。
- 将数分为 $n$ 组：$\{1,2\}, \{3,4\}, \ldots, \{2n-1, 2n\}$
- 由抽屉原理，必有两个数在同一组

### 6. 染色与标记

#### 棋盘染色
用于覆盖问题、路径问题。

#### 图染色
用于组合问题、图论问题。

**示例**: 证明 $8 \times 8$ 棋盘去掉对角两格后不能被 $1 \times 2$ 骨牌覆盖。
- 黑白染色，发现去掉的两格同色
- 每个骨牌覆盖一黑一白，无法平衡

### 7. 数学归纳法

#### 第一归纳法
- 基础步：验证 $n = 1$
- 归纳步：假设 $n = k$ 成立，证明 $n = k+1$ 成立

#### 第二归纳法（强归纳法）
- 归纳步：假设对所有 $m < k$ 成立，证明 $n = k$ 成立

#### 其他变体
- 倒推归纳法
- 螺旋归纳法
- 双重归纳法

**示例**: 证明 $n$ 元集合有 $2^n$ 个子集。
- 基础：$n = 0$ 时，空集有 $2^0 = 1$ 个子集（空集自身）
- 归纳：假设 $n = k$ 时成立，考虑 $n = k+1$ 时，新增元素可选择加入或不加入任一子集，故子集数翻倍

### 8. 反证法

假设结论不成立，推出矛盾。

**适用情形**: 
- 结论含"不存在"、"不可能"等否定词
- 直接证明困难

**示例**: 证明 $\sqrt{2}$ 是无理数。
- 假设 $\sqrt{2} = \frac{p}{q}$（最简分数）
- 推出 $p, q$ 都是偶数，矛盾

### 9. 配对法

将元素两两配对，简化计数或求和。

**示例**: 计算 $\sum_{k=0}^{n} (-1)^k \binom{n}{k}$。
- 利用 $(1-1)^n = 0$ 的组合意义
- 或配对 $(1+x)^n$ 展开式中 $x^k$ 与 $x^{n-k}$ 系数

### 10. 递推方法

建立递推关系求解。

**示例**: Hanoi塔问题。
- 设 $H_n$ 为移动 $n$ 个盘的最少步数
- 递推关系：$H_n = 2H_{n-1} + 1$，$H_1 = 1$
- 解得 $H_n = 2^n - 1$

## ⏱️ 时间管理与应试策略

### 题目选择

```
优先级:
1. 熟悉类型 → 先拿稳分
2. 中等难度 → 争取突破
3. 高难度题 → 选择性攻克
```

### 书写规范

1. **清晰表达**: 逻辑链条完整
2. **符号规范**: 使用标准数学符号
3. **步骤完整**: 关键步骤不可省略
4. **答案标注**: 明确写出最终答案

### 检查策略

- 时间分配：做题与检查比例约 4:1
- 重点检查：计算过程、边界条件
- 验证方法：特例验证、量纲分析
