## 📋 概述

不等式是数学竞赛的核心内容之一，涵盖经典不等式、技巧方法和高级技巧。本专题系统梳理竞赛中常见的不等式问题。

## 🔷 经典不等式

### 1. 均值不等式 (AM-GM)

**基本形式**: 对正实数 $a_1, a_2, \ldots, a_n$：
$$\frac{a_1 + a_2 + \cdots + a_n}{n} \geq \sqrt[n]{a_1 a_2 \cdots a_n}$$

等号成立当且仅当 $a_1 = a_2 = \cdots = a_n$。

**加权形式**: 对正实数 $a_i$ 和正权重 $w_i$（$\sum w_i = 1$）：
$$\sum w_i a_i \geq \prod a_i^{w_i}$$

### 2. Cauchy-Schwarz不等式

**基本形式**: 对实数 $a_i, b_i$：
$$\left(\sum a_i^2\right)\left(\sum b_i^2\right) \geq \left(\sum a_i b_i\right)^2$$

等号成立当且仅当 $(a_1, \ldots, a_n)$ 与 $(b_1, \ldots, b_n)$ 成比例。

**Engel形式** (Titu引理):
$$\sum \frac{a_i^2}{b_i} \geq \frac{(\sum a_i)^2}{\sum b_i}$$

其中 $b_i > 0$。

### 3. Jensen不等式

**凸函数定义**: $f$ 在区间 $I$ 上是凸函数，如果对任意 $x, y \in I$ 和 $t \in [0,1]$：
$$f(tx + (1-t)y) \leq tf(x) + (1-t)f(y)$$

**Jensen不等式**: 若 $f$ 是凸函数，则对任意 $x_i \in I$ 和正权重 $w_i$（$\sum w_i = 1$）：
$$f\left(\sum w_i x_i\right) \leq \sum w_i f(x_i)$$

**常用凸函数**: $e^x$, $-\ln x$, $x^p$ ($p \geq 1$ 或 $p \leq 0$)

### 4. Hölder不等式

**基本形式**: 对正实数 $a_{ij}$：
$$\sum_{i=1}^{n} \prod_{j=1}^{m} a_{ij} \leq \prod_{j=1}^{m} \left(\sum_{i=1}^{n} a_{ij}^{p_j}\right)^{1/p_j}$$

其中 $\sum \frac{1}{p_j} = 1$。

**特例** (Cauchy-Schwarz): $m=2, p_1 = p_2 = 2$

### 5. 排序不等式

**定理**: 设 $a_1 \leq a_2 \leq \cdots \leq a_n$ 和 $b_1 \leq b_2 \leq \cdots \leq b_n$ 是两组实数，则对任意排列 $\sigma$：
$$\sum a_i b_{n+1-i} \leq \sum a_i b_{\sigma(i)} \leq \sum a_i b_i$$

即：顺序和 $\geq$ 乱序和 $\geq$ 逆序和

### 6. Chebyshev不等式

**定理**: 若 $a_1 \leq a_2 \leq \cdots \leq a_n$ 和 $b_1 \leq b_2 \leq \cdots \leq b_n$，则：
$$\frac{1}{n}\sum a_i b_i \geq \left(\frac{1}{n}\sum a_i\right)\left(\frac{1}{n}\sum b_i\right)$$

若一组递增一组递减，则不等号反向。

### 7. Schur不等式

**基本形式**: 对非负实数 $a, b, c$ 和 $r > 0$：
$$a^r(a-b)(a-c) + b^r(b-c)(b-a) + c^r(c-a)(c-b) \geq 0$$

**特例** ($r=1$):
$$a^3 + b^3 + c^3 + 3abc \geq ab(a+b) + bc(b+c) + ca(c+a)$$

等价于：
$$a^3 + b^3 + c^3 + 3abc \geq \sum_{sym} a^2b$$

## 🔧 常用技巧

### 1. 齐次化

若不等式是齐次的，可以添加条件简化问题。

**示例**: 证明对正实数 $a, b, c$，有
$$\frac{a}{b+c} + \frac{b}{c+a} + \frac{c}{a+b} \geq \frac{3}{2}$$

可设 $a + b + c = 1$，转化为证明 $\sum \frac{a}{1-a} \geq \frac{3}{2}$。

### 2. 归一化

通过设定特定值（如和为1、积为1等）简化表达式。

### 3. 对称化

利用对称性，假设变量有序或相等。

**常用假设**: $a \geq b \geq c$ 或 $a = \max(a,b,c)$

### 4. 换元法

**常用换元**:
- 三角换元：$a = \tan A$ 等
- 代数换元：$x = a + b$, $y = b + c$, $z = c + a$
- 齐次换元：设 $a + b + c = 1$

### 5. 放缩法

通过适当放缩简化表达式。

**原则**: 放缩方向要正确，不要过度放缩

## 📐 高级技巧

### 1. SOS方法 (Sum of Squares)

将不等式表示为平方和的形式。

**标准形式**:
$$S_a(b-c)^2 + S_b(c-a)^2 + S_c(a-b)^2 \geq 0$$

### 2. uvw方法

对于三元对称不等式，设：
- $3u = a + b + c$
- $3v^2 = ab + bc + ca$
- $w^3 = abc$

**核心定理**: 对称不等式的极值通常在两个变量相等或一个变量为0时取到。

### 3. pqr方法

类似uvw，用于不同场景。

### 4. Lagrange乘数法

用于求条件极值。

## 📝 训练题目

### 基础题

**题目1**: 设 $a, b, c > 0$，证明：
$$(a+b)(b+c)(c+a) \geq 8abc$$

**提示**: 使用AM-GM不等式。

**题目2**: 设 $a, b, c > 0$ 且 $a + b + c = 1$，证明：
$$\frac{1}{a} + \frac{1}{b} + \frac{1}{c} \geq 9$$

**提示**: Cauchy-Schwarz或AM-HM。

### 中等题

**题目3** (Nesbitt不等式): 设 $a, b, c > 0$，证明：
$$\frac{a}{b+c} + \frac{b}{c+a} + \frac{c}{a+b} \geq \frac{3}{2}$$

**题目4**: 设 $a, b, c > 0$，证明：
$$\frac{a^2}{b} + \frac{b^2}{c} + \frac{c^2}{a} \geq a + b + c$$

**提示**: Cauchy-Schwarz（Engel形式）。

### 难题

**题目5**: 设 $a, b, c \geq 0$ 且 $a + b + c = 3$，证明：
$$a^2 + b^2 + c^2 + abc \geq 4$$

**题目6**: 设 $a, b, c > 0$，证明：
$$\sum_{cyc} \frac{a^3}{a^2 + ab + b^2} \geq \frac{a + b + c}{3}$$
