# 组合计数专题训练

**最后更新**: 2026年4月4日

---

## 📋 概述

组合数学是数学竞赛的重要分支，涵盖计数原理、组合恒等式、图论基础等内容。本专题系统梳理组合计数的核心方法。

---

## 🔷 基本计数原理

### 1. 加法原理

若完成一件事有 $n$ 类方法，第 $i$ 类有 $m_i$ 种方法，则共有 $m_1 + m_2 + \cdots + m_n$ 种方法。

**关键**: 分类不重复、不遗漏。

### 2. 乘法原理

若完成一件事需 $n$ 个步骤，第 $i$ 步有 $m_i$ 种方法，则共有 $m_1 \cdot m_2 \cdots m_n$ 种方法。

**关键**: 步骤相互独立。

### 3. 排列与组合

**排列**: 从 $n$ 个不同元素中取 $k$ 个的有序选取数：
$$P(n, k) = A_n^k = \frac{n!}{(n-k)!}$$

**组合**: 从 $n$ 个不同元素中取 $k$ 个的无序选取数：
$$C(n, k) = \binom{n}{k} = \frac{n!}{k!(n-k)!}$$

**性质**:
- $\binom{n}{k} = \binom{n}{n-k}$
- $\binom{n}{k} = \binom{n-1}{k-1} + \binom{n-1}{k}$ (Pascal恒等式)

## 🔶 二项式定理与恒等式

### 二项式定理

$$(x + y)^n = \sum_{k=0}^{n} \binom{n}{k} x^{n-k} y^k$$

### 重要恒等式

**基本恒等式**:
$$\sum_{k=0}^{n} \binom{n}{k} = 2^n$$

**交错和**:
$$\sum_{k=0}^{n} (-1)^k \binom{n}{k} = 0$$

**加权和**:
$$\sum_{k=0}^{n} k \binom{n}{k} = n \cdot 2^{n-1}$$

**Vandermonde恒等式**:
$$\sum_{k=0}^{r} \binom{m}{k} \binom{n}{r-k} = \binom{m+n}{r}$$

**组合证明**: 从 $m$ 个男性和 $n$ 个女性中选 $r$ 人，按男性人数分类计数。

**Hockey-stick恒等式**:
$$\sum_{i=r}^{n} \binom{i}{r} = \binom{n+1}{r+1}$$

## 🔸 容斥原理

### 基本形式

对有限集合 $A_1, A_2, \ldots, A_n$：
$$\left|\bigcup_{i=1}^{n} A_i\right| = \sum_{i} |A_i| - \sum_{i<j} |A_i \cap A_j| + \sum_{i<j<k} |A_i \cap A_j \cap A_k| - \cdots + (-1)^{n-1} |A_1 \cap \cdots \cap A_n|$$

### 补集形式

$$\left|\bigcap_{i=1}^{n} \overline{A_i}\right| = |U| - \sum_{i} |A_i| + \sum_{i<j} |A_i \cap A_j| - \cdots$$

### 经典应用

**错排问题**: $n$ 个元素的排列，每个元素都不在原来位置的排列数 $D_n$：
$$D_n = n! \sum_{k=0}^{n} \frac{(-1)^k}{k!} \approx \frac{n!}{e}$$

## 🔹 递推关系

### 线性递推

**一阶线性**: $a_n = p a_{n-1} + q$
- 若 $p \neq 1$，设 $a_n = b_n + c$，求特解 $c = \frac{q}{1-p}$
- 转化为等比数列

**二阶线性齐次**: $a_n = p a_{n-1} + q a_{n-2}$
- 特征方程：$r^2 = pr + q$
- 若特征根为 $r_1, r_2$，则 $a_n = A r_1^n + B r_2^n$

### Catalan数

**定义**: $C_n = \frac{1}{n+1}\binom{2n}{n}$

**递推**: $C_0 = 1$，$C_{n+1} = \sum_{i=0}^{n} C_i C_{n-i}$

**应用场景**:
- 括号匹配方案数
- 二叉树形态数
- 不交叉对角线划分
- 栈排序方案数

## 📐 生成函数

### 定义

序列 $\{a_n\}$ 的生成函数：
$$G(x) = \sum_{n=0}^{\infty} a_n x^n$$

### 常用生成函数

| 序列 | 生成函数 |
|------|----------|
| $1, 1, 1, \ldots$ | $\frac{1}{1-x}$ |
| $1, 2, 3, \ldots$ | $\frac{1}{(1-x)^2}$ |
| $\binom{n}{0}, \binom{n}{1}, \ldots$ | $(1+x)^n$ |
| $C_0, C_1, C_2, \ldots$ | $\frac{1-\sqrt{1-4x}}{2x}$ |

### 应用

**组合计数**: 从多类物品中选取的生成函数为各类型生成函数的乘积。

## 🔷 图论基础

### 基本概念

**图**: 由顶点集 $V$ 和边集 $E$ 组成的结构 $G = (V, E)$。

**度**: 顶点 $v$ 的度 $d(v)$ 是与 $v$ 关联的边数。

**握手定理**: $\sum_{v \in V} d(v) = 2|E|$

### 特殊图

**完全图** $K_n$: $n$ 个顶点，每对顶点间都有边。
- 边数：$\binom{n}{2}$

**二分图**: 顶点可分为两部分，边只在两部分之间。

**树**: 连通无圈图。
- 边数：$|V| - 1$

### 经典定理

**Euler定理**: 连通图有Euler回路当且仅当每个顶点度为偶数。

**Dirac定理**: 若 $n \geq 3$ 且每个顶点度至少为 $n/2$，则图有Hamilton圈。

**Turán定理**: $n$ 个顶点不含 $K_{r+1}$ 的图的最大边数。

## 📝 训练题目

### 基础题

**题目1**: 从5男4女中选3人，要求至少1名女性，有多少种选法？

**题目2**: 证明：$\sum_{k=0}^{n} \binom{n}{k}^2 = \binom{2n}{n}$

**提示**: Vandermonde恒等式或组合意义。

### 中等题

**题目3**: $n$ 对夫妻围圆桌就坐，要求每对夫妻不相邻，求就坐方案数。

**题目4**: 求由1, 2, 3组成的 $n$ 位数中，1、2、3都出现的数的个数。

**提示**: 容斥原理。

### 难题

**题目5**: 证明 $n$ 个顶点的不同标记树的数量为 $n^{n-2}$（Cayley公式）。

**题目6**: (IMO 1995) 设 $p$ 是奇素数。求集合 $\{1, 2, \ldots, 2p\}$ 的满足以下条件的子集 $A$ 的个数：
- $|A| = p$
- $A$ 中所有元素之和被 $p$ 整除
