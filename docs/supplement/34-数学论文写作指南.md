# 数学论文写作指南

---

## 论文结构

### 标准结构

**1. 标题 (Title)**
- 准确、简洁
- 反映核心贡献
- 避免过度夸张

**2. 摘要 (Abstract)**
- 200-300字
- 背景、方法、结果、结论
- 独立成篇

**3. 引言 (Introduction)**
- 问题背景
- 相关工作
- 主要贡献
- 论文结构

**4. 预备知识 (Preliminaries)**
- 符号说明
- 基本概念
- 已有结果

**5. 主体内容**
- 定理、引理、证明
- 算法、分析
- 例子、应用

**6. 结论 (Conclusion)**
- 总结贡献
- 未来工作

**7. 参考文献**
- 规范格式
- 完整引用

## 写作原则

### 清晰性

**定义先行**
- 新符号立即定义
- 避免歧义
- 符号保持一致

**逻辑清晰**
- 段落有主题
- 过渡自然
- 论证完整

### 准确性

**数学严谨**
- 定理条件完整
- 证明无漏洞
- 引用准确

**语言精确**
- 避免模糊词汇
- 量化表述
- 区分"充分"与"必要"

### 简洁性

**删除冗余**
- 去除重复
- 简化表达
- 合并相似内容

**有效使用空间**
- 重要结果详细
- 已知结果简述
- 附录放细节

## 证明写作

### 证明结构

```
定理 X.X [定理名称]
[陈述]

证明:
[步骤1] ...
[步骤2] ...
...
因此，[结论]。 ∎
```

### 证明技巧

**分情况讨论**
- 清晰标注每种情况
- 确保覆盖所有可能

**引用先前结果**
- 明确标注引理/定理编号
- 验证条件满足

**使用图表**
- 复杂结构可视化
- 辅助理解

## 符号与记号

### 符号选择原则

| 类型 | 常用符号 | 示例 |
|-----|---------|------|
| 集合 | 大写字母 | $A, B, X$ |
| 元素 | 小写字母 | $x, y, z$ |
| 函数 | $f, g, h$ | $f(x)$ |
| 常数 | $c, k, C$ | $c = 2$ |
| 索引 | $i, j, k, n, m$ | $x_i$ |

### 常用记号

- $\mathbb{N}$: 自然数
- $\mathbb{Z}$: 整数
- $\mathbb{Q}$: 有理数
- $\mathbb{R}$: 实数
- $\mathbb{C}$: 复数
- $\forall$: 对所有
- $\exists$: 存在
- $\Rightarrow$: 蕴含
- $\Leftrightarrow$: 当且仅当

## LaTeX数学排版

### 常用环境

```latex
% 定理环境
\begin{theorem}
定理内容
\end{theorem}

% 证明
\begin{proof}
证明内容
\end{proof}

% 公式
\begin{equation}
E = mc^2
\end{equation}

% 对齐公式
\begin{align}
a &= b + c \\
  &= d + e
\end{align}
```

### 排版建议

- 公式编号重要时编号
- 长公式适当换行
- 使用合适的括号大小
- 注意间距

## 投稿与审稿

### 选择期刊

**考虑因素**
- 研究领域匹配
- 影响因子
- 审稿周期
- 开放获取政策

### 审稿意见处理

**建设性回应**
- 感谢审稿人
- 逐条回复
- 修改明确标注
- 合理解释不修改处

### 常见问题

| 问题 | 解决 |
|-----|------|
| 证明不完整 | 补充细节 |
| 结果不显著 | 强调意义 |
| 相关文献遗漏 | 补充引用 |
| 表述不清 | 重写简化 |

## 学术伦理

### 引用规范

- 适当引用前人工作
- 区分原创与引用
- 避免过度自引

### 合作作者

- 贡献者都列名
- 通讯作者明确
- 致谢贡献者

### 数据与代码

- 可重复性
- 数据共享
- 代码开源
