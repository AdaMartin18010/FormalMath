## 1. 分析学决策树

### 1.1 收敛性判定决策树

```mermaid
flowchart TD
    A[判定序列/函数收敛] --> B{类型}
    B -->|数列| C{已知信息}
    B -->|函数列| D{收敛模式}
    B -->|级数| E{级数类型}
    
    C -->|单调| F[单调收敛定理]
    C -->|有界| G[Bolzano-Weierstrass]
    C -->|递推| H[压缩映射/不动点]
    
    D -->|逐点收敛| I[固定x，判数列收敛]
    D -->|一致收敛| J[sup范数趋于0]
    D -->|L收敛| K[积分趋于0]
    
    E -->|正项级数| L{判别法选择}
    E -->|交错级数| M[Leibniz判别]
    E -->|一般级数| N[绝对收敛?]
    
    L -->|比值/根值| O[计算极限]
    L -->|比较| P[找参照级数]
    L -->|积分| Q[积分判别法]
```

### 1.2 积分计算决策树

```mermaid
flowchart TD
    A[计算积分] --> B{积分类型}
    
    B -->|不定积分| C{被积函数}
    B -->|定积分| D{区间/奇点}
    B -->|重积分| E{区域类型}
    
    C -->|有理函数| F[部分分式分解]
    C -->|三角函数| G[万能代换/降幂]
    C -->|根式| H[换元消根]
    C -->|指数对数| I[分部积分]
    C -->|乘积| J[分部积分]
    
    D -->|对称区间| K[奇偶性]
    D -->|无穷区间| L[反常积分收敛性]
    D -->|奇点| M[瑕积分判定]
    
    E -->|矩形| N[迭代积分]
    E -->|极坐标| O[r, θ换元]
    E -->|一般区域| P[坐标变换]
```

### 1.3 连续性判定决策树

| 判定条件 | 方法 | 关键检查点 |
|---------|------|-----------|
| **点连续** | ε-δ定义 | $\lim_{x \to a} f(x) = f(a)$ |
| **区间连续** | 每点连续 | 端点单侧连续 |
| **一致连续** | Cantor定理/直接估计 | $|\delta$ 不依赖点 |
| **绝对连续** | 定义/Lipschitz | 变差有限 |
| **Lipschitz** | 差商有界 | $|f(x)-f(y)| \leq L|x-y|$ |

## 2. 代数学决策树

### 2.1 群结构分析决策树

```mermaid
flowchart TD
    A[分析群G的结构] --> B{有限/无限}
    
    B -->|有限| C{阶}
    B -->|无限| D{生成方式}
    
    C -->|素数阶| E[循环群 Z_p]
    C -->|小阶| F[查分类表]
    C -->|一般| G[Sylow分析]
    
    D -->|有限生成| H[Abel群结构定理]
    D -->|自由| I[自由群/自由Abel群]
    
    G --> J{p-子群结构}
    G --> K{正规子群?}
    
    K -->|单群| L[不可分解]
    K -->|可解| M[合成列分析]
```

### 2.2 子群判定决策树

```mermaid
flowchart TD
    A[判定H是否为G的子群] --> B{方法选择}
    
    B -->|两步法| C[封闭性检验]
    C --> D[逆元检验]
    
    B -->|一步法| E[∀a,b∈H, ab⁻¹∈H]
    
    B -->|有限子群| F[仅需封闭性]
    
    B -->|核/像| G[同态的ker/im]
    
    D --> H[是子群]
    E --> H
    F --> H
    G --> H
```

### 2.3 环论问题决策树

| 问题类型 | 判定步骤 | 关键定理 |
|---------|---------|---------|
| **整环** | 无零因子 | $ab=0 \Rightarrow a=0$ or $b=0$ |
| **PID** | 理想皆主 | 欧几里得算法 |
| **UFD** | 唯一分解 | 素元=不可约元 |
| **域** | 非零元可逆 | 无真理想 |
| **素理想** | $R/P$ 整环 | 吸收乘法 |
| **极大理想** | $R/M$ 域 | 无真包含理想 |

## 3. 线性代数决策树

### 3.1 矩阵问题决策树

```mermaid
flowchart TD
    A[矩阵问题] --> B{问题类型}
    
    B -->|可逆性| C{方法}
    B -->|对角化| D{判定}
    B -->|解方程| E{Ax=b}
    
    C -->|行列式| F[det≠0]
    C -->|秩| G[rank=n]
    C -->|方程| H[Ax=0只有零解]
    
    D -->|特征值| I[代数重数=几何重数]
    D -->|极小多项式| J[无重根]
    
    E -->|相容性| K[rank(A)=rank([A|b])]
    E -->|解结构| L[特解+零空间]
```

### 3.2 线性系统求解决策

| 情况 | 条件 | 解的结构 |
|-----|------|---------|
| **唯一解** | $\det A \neq 0$ | $x = A^{-1}b$ |
| **无穷解** | $\text{rank}(A) = \text{rank}([A|b]) < n$ | 特解+零空间基 |
| **无解** | $\text{rank}(A) < \text{rank}([A|b])$ | 最小二乘解 |

## 4. 拓扑学决策树

### 4.1 紧性判定决策树

```mermaid
flowchart TD
    A[判定空间X的紧性] --> B{空间类型}
    
    B -->|度量空间| C{Heine-Borel?}
    B -->|一般拓扑| D[开覆盖定义]
    B -->|序列紧| E[任意列有收敛子列]
    
    C -->|ℝⁿ子集| F[闭且有界]
    C -->|一般度量| G[完全+全有界]
    
    D --> H[任意开覆盖有有限子覆盖]
    
    E --> I[度量空间⇔紧]
```

### 4.2 连通性判定决策树

| 性质 | 判定方法 | 反例 |
|-----|---------|-----|
| **连通** | 不能分离为不交开集 | 单点集 |
| **道路连通** | 任意两点有道路 | 拓扑学家正弦曲线 |
| **单连通** | 道路连通+基本群平凡 | $S^1$ |
| **局部连通** | 每点有连通邻域基 | Topologist's sine closure |

## 5. 微分方程决策树

### 5.1 ODE类型判定与求解

```mermaid
flowchart TD
    A[ODE问题] --> B{阶数}
    
    B -->|一阶| C{类型}
    B -->|高阶| D{线性?}
    
    C -->|可分离| E[分离变量]
    C -->|线性| F[积分因子]
    C -->|恰当| G[求势函数]
    C -->|齐次| H[换元u=y/x]
    
    D -->|是| I{常系数?}
    D -->|否| J[数值/级数解]
    
    I -->|是| K[特征方程]
    I -->|否| L[降阶/级数]
    
    K --> M[齐次解+特解]
```

### 5.2 稳定性分析决策树

```mermaid
flowchart TD
    A[系统X'=AX稳定性] --> B{特征值}
    
    B -->|实部皆负| C[渐近稳定]
    B -->|有正实部| D[不稳定]
    B -->|纯虚| E[中心/Lyapunov稳定]
```

## 6. 概率统计决策树

### 6.1 分布选择决策树

```mermaid
flowchart TD
    A[选择概率模型] --> B{数据类型}
    
    B -->|离散| C{取值范围}
    B -->|连续| D{支撑集}
    
    C -->|0-1| E[Bernoulli]
    C -->|计数| F{稀有?}
    C -->|等待次数| G[几何/负二项]
    
    F -->|是| H[Poisson]
    F -->|否| I[二项]
    
    D -->|正实数| J{无记忆?}
    D -->|实数| K{对称?}
    D -->|有界| L[Beta/均匀]
    
    J -->|是| M[指数]
    J -->|否| N[Gamma/Weibull]
    
    K -->|是| O[正态/Student-t]
```

### 6.2 假设检验决策树

| 步骤 | 决策 | 方法 |
|-----|------|-----|
| **1. 建立假设** | $H_0$ vs $H_1$ | 明确原/备择 |
| **2. 选择统计量** | 参数/非参数 | Z, t, $\chi^2$, F |
| **3. 确定拒绝域** | 显著性水平 $\alpha$ | 单侧/双侧 |
| **4. 计算p值** | 与$\alpha$比较 | 统计软件/表 |
| **5. 决策** | 拒绝/不拒绝$H_0$ | 结论表述 |

## 7. 综合决策框架

### 7.1 问题识别流程

```mermaid
flowchart TD
    A[遇到数学问题] --> B{识别领域}
    
    B -->|涉及极限/连续| C[分析学]
    B -->|涉及结构/运算| D[代数学]
    B -->|涉及空间/形状| E[几何拓扑]
    B -->|涉及变化/演化| F[微分方程]
    B -->|涉及随机/数据| G[概率统计]
    
    C --> H[选择对应决策树]
    D --> H
    E --> H
    F --> H
    G --> H
```

### 7.2 证明策略选择

| 证明目标 | 推荐策略 | 典型案例 |
|---------|---------|---------|
| **存在性** | 构造法/介值定理 | 零点存在 |
| **唯一性** | 反证法/单调性 | 不动点唯一 |
| **对所有成立** | 直接证明/归纳法 | 恒等式 |
| **不存在** | 反证法 | 无理数证明 |
| **等价性** | 双向证明 | 充要条件 |
