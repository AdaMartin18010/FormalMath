# FormalMath 项目索引

## 项目概述

FormalMath项目是一个全面的数学模型知识库，包含截至2025年所有最成熟的数学理论模型、形式模型、语义模型和科学模型。项目采用中英双语，提供严格的形式化证明和代码实现。
