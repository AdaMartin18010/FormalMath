## 1. 概述

### 1.1 目的与范围

本文档建立FormalMath项目的**统一数学符号规范**，确保：

- ✅ 所有文档使用**一致的数学符号**
- ✅ 符号系统符合**国际标准**（ISO 80000-2, AMS）
- ✅ 便于**跨文档引用**和**理解**
- ✅ 支持**国际化**交流

**适用范围**：
- 所有数学概念文档
- 形式化证明文档
- 教学资源与示例
- 跨学科应用文档

### 1.2 参考标准

| 标准 | 说明 | 优先级 |
|------|------|--------|
| **ISO 80000-2** | 量和单位—第2部分：数学 | 最高 |
| **AMS数学符号标准** | 美国数学学会标准 | 高 |
| **LaTeX数学排版** | 学术排版标准 | 高 |
| **Unicode数学符号** | 符号编码标准 | 中 |
| **MathML** | 数学标记语言 | 中 |

### 1.3 符号表示原则

1. **一致性原则**：同一概念在全项目使用相同符号
2. **国际标准原则**：优先采用ISO和AMS标准
3. **清晰性原则**：符号应直观、易于理解
4. **兼容性原则**：与主流数学教材兼容
5. **可扩展性原则**：预留符号扩展空间

| 粗体 | `\mathbf{v}` | $\mathbf{v}$ | 向量、矩阵 |

### 8.3 下标与上标

**下标使用规则**：
- 索引：$a_1, a_2, \ldots, a_n$
- 元素：$x_i \in X$
- 限制：$f|_S$
- 偏导：$f_x, f_{xy}$

**上标使用规则**：
- 幂次：$x^2, x^n$
- 逆元：$g^{-1}$
- 转置：$A^T$
- 共轭：$\bar{z}$ 或 $z^*$
- 对偶：$V^*$
- 张量：$T^{ij}$

## 9. 符号一致性检查清单

### 9.1 检查项目

- [ ] 所有数集使用黑板粗体：$\mathbb{N}, \mathbb{Z}, \mathbb{Q}, \mathbb{R}, \mathbb{C}$
- [ ] 映射箭头使用 `\to` 而非 `\rightarrow`
- [ ] 极限使用 `\lim_{x \to a}` 格式
- [ ] 微分符号$d$使用罗马体：`\mathrm{d}`
- [ ] 函数名使用罗马体：`\sin`, `\cos`, `\log`
- [ ] 向量使用粗体：`\mathbf{v}`
- [ ] 逻辑与使用 `\land` 而非 `\wedge`
- [ ] 逻辑或使用 `\lor` 而非 `\vee`
- [ ] 子集使用 `\subseteq`（一般子集）或 `\subset`（真子集）
- [ ] 群、环、域使用大写字母表示
- [ ] 自然数是否包含0在文中明确说明
- [ ] 复合函数使用 $f \circ g$ 表示 $(f \circ g)(x) = f(g(x))$

### 9.2 禁止用法

| ❌ 错误用法 | ✅ 正确用法 | 说明 |
|------------|------------|------|
| $N, Z, Q, R, C$ | $\mathbb{N}, \mathbb{Z}, \mathbb{Q}, \mathbb{R}, \mathbb{C}$ | 数集必须用黑板粗体 |
| $\rightarrow$ | $\to$ | 映射用短箭头 |
| $\Longrightarrow$ | $\Rightarrow$ | 蕴含用单箭头 |
| $dx$ | $\mathrm{d}x$ | 微分用罗马体d |
| $\vec{v}$ | $\mathbf{v}$ | 向量用粗体 |
| $\text{sin}$ | `\sin` | 函数名用预定义命令 |
| $\subset$（表示一般子集） | $\subseteq$ | 区分真子集和一般子集 |
| $\setminus$ 以外表示差集 | $\setminus$ | 统一用 `\setminus` |

### 10.2 常见错误与纠正

**错误1：符号混用**
```markdown
❌ $f: A \rightarrow B$（使用长箭头）
✅ $f: A \to B$（使用短箭头）
```

**错误2：数集字体错误**
```markdown
❌ $R$ 或 **R**（表示实数集）
✅ $\mathbb{R}$（使用黑板粗体）
```

**错误3：微分符号格式错误**
```markdown
❌ $dx$（斜体d）
✅ $\mathrm{d}x$（罗马体d）
```

**错误4：向量表示不一致**
```markdown
❌ $\vec{v}$ 或 $\overrightarrow{v}$
✅ $\mathbf{v}$（统一使用粗体）
```

**错误5：子集符号混淆**
```markdown
❌ $A \subset B$（含义不明确）
✅ $A \subseteq B$（A是B的子集）
✅ $A \subsetneq B$ 或 $A \subsetneqq B$（A是B的真子集）
```

**错误6：极限符号不规范**
```markdown
❌ $\lim_{x \rightarrow a}$（使用长箭头）
✅ $\lim_{x \to a}$（使用短箭头）
```
