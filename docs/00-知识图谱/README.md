# 00-知识图谱

本目录以网络图谱的形式呈现数学知识的宏观结构，共收录 20 余份高阶知识图谱文档。这些图谱不仅展示单个分支内的定理依赖关系（如群论定理网络、导出函子关系图），还跨越分支边界绘制宏大图景（如 Langlands 纲领关系图、p-进 Hodge 理论全景图、数学基础体系层次图）。部分文件专门对齐形式化数学工具，例如 Lean4 定理映射图，为从传统数学学习到形式化实践的过渡提供全景式导航。

## 子目录/文件导航

本目录为扁平结构，主要文件包括：

| 文件 | 说明 |
|------|------|
| `代数-群论定理依赖网络.md` | 群论核心定理的依赖与推导网络 |
| `同调代数-导出函子关系图.md` | 导出函子、Ext 与 Tor 的关系图谱 |
| `实分析-核心概念关系图谱.md` | 极限、连续、可微、可积概念网络 |
| `形式化-Lean4定理映射图.md` | Lean4 Mathlib 中定理的分类映射 |
| `知识图谱-016-代数几何五大定理关系图.md` | 代数几何奠基性定理的相互关联 |
| `知识图谱-017-同调代数核心概念网络.md` | 同调代数范畴与技术的知识网络 |
| `知识图谱-018-p进Hodge理论全景图.md` | p-进 Hodge 理论的宏观结构 |
| `知识图谱-019-代数曲面分类决策树.md` | 代数曲面分类的知识节点分支 |
| `知识图谱-020-数学基础体系层次图.md` | 集合论、类型论与范畴论基础层次 |
| `知识图谱-021-Langlands纲领关系图.md` | Langlands 纲领的跨分支关系概览 |
