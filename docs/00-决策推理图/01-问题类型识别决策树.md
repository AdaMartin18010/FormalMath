# 问题类型识别决策树

> 本决策树用于快速识别数学问题的类型，并推荐相应的解决方法和关键定理。

---

```

├── 代数问题？

├── 分析问题？
│   ├── 收敛性？
│   ├── 存在性？
```

## 一、代数问题分支

### 1.1 方程求解

```

方程求解
├── 线性方程组
│   ├── 判断条件: 未知量次数均为1，无乘积项
│   ├── 推荐方法: 矩阵方法/Gauss消元法
│   ├── 关键定理: 
│   │   - Cramer法则（理论解）
│   │   - LU分解（数值计算）
│   │   - 秩-零化度定理
│   └── 典型例子: 解 Ax = b，求特征值问题
│
├── 多项式方程
│   ├── 判断条件: 方程可表示为多项式等于零
│   ├── 推荐方法: 因式分解/Galois理论
│   ├── 关键定理:
│   │   - 代数基本定理（根的存在性）
│   │   - Abel-Ruffini定理（五次以上无根式解）
│   │   - Sturm定理（实根计数）
│   └── 典型例子: x⁵ - x + 1 = 0，求根问题
│
└── 超越方程
    ├── 判断条件: 包含超越函数（exp, sin, log等）
    ├── 推荐方法: 数值方法/迭代法
    ├── 关键定理:
    │   - 介值定理（根的存在性）
    │   - Newton迭代收敛定理
    │   - 压缩映射原理
    └── 典型例子: eˣ = x + 2，cos x = x

```

### 1.2 结构分类

```

代数结构分类
├── 有限群
│   ├── 判断条件: 有限集合 + 群公理
│   ├── 推荐方法: Sylow定理/表示论
│   ├── 关键定理:
│   │   - Sylow定理（p-子群存在性）
│   │   - Lagrange定理（子群阶整除）
│   │   - Jordan-Hölder定理（合成列）
│   │   - Burnside定理（p^aq^b可解）
│   └── 典型例子: 对称群Sₙ，交错群Aₙ，二面体群Dₙ
│
├── Lie群
│   ├── 判断条件: 光滑流形 + 群结构 + 相容性
│   ├── 推荐方法: Lie代数/指数映射
│   ├── 关键定理:
│   │   - Lie群-Lie代数对应
│   │   - Baker-Campbell-Hausdorff公式
│   │   - Peter-Weyl定理
│   └── 典型例子: GL(n,ℝ), SO(n), SU(n), 辛群Sp(2n)
│
└── 代数簇
    ├── 判断条件: 多项式零点集
    ├── 推荐方法: 概形理论/层上同调
    ├── 关键定理:
    │   - Hilbert零点定理
    │   - Bézout定理（交点计数）
    │   - Riemann-Roch定理
    │   - Serre对偶性
    └── 典型例子: 椭圆曲线，射影簇，环面簇

```

### 1.3 不变量计算

```

不变量计算
├── 特征值问题
│   ├── 判断条件: 线性算子的谱分析
│   ├── 推荐方法: 特征多项式/Jordan标准形
│   ├── 关键定理:
│   │   - Cayley-Hamilton定理
│   │   - 谱分解定理
│   │   - Jordan标准形定理
│   │   - Gershgorin圆盘定理（特征值定位）
│   └── 典型例子: 矩阵对角化，稳定性分析
│
├── 同调群计算
│   ├── 判断条件: 拓扑空间的代数不变量
│   ├── 推荐方法: 链复形/正合序列
│   ├── 关键定理:
│   │   - Mayer-Vietoris序列
│   │   - 万有系数定理
│   │   - Künneth公式
│   │   - Poincaré对偶性
│   └── 典型例子: 球面同调，射影空间，环面
│
└── K-理论
    ├── 判断条件: 向量丛或投射模的分类
    ├── 推荐方法: 向量丛/投射模的Grothendieck群
    ├── 关键定理:
    │   - Bott周期性定理
    │   - Atiyah-Singer指标定理
    │   - 周期性定理（K₀, K₁）
    └── 典型例子: 拓扑K-理论，代数K-理论

```

```
│
├── 级数收敛
│   ├── 判断条件: 无穷求和的收敛性
│   ├── 推荐方法: 判别法/比较测试
│   ├── 关键定理:
│   │   - 比较判别法
│   │   - 比值判别法（d'Alembert）
│   │   - 根值判别法（Cauchy）
│   │   - Leibniz判别法（交错级数）
```
