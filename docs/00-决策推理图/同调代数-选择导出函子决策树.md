# 同调代数 - 选择导出函子决策树

**适用**: 同调代数、代数几何  
**表征类型**: 决策树

---

## 决策树

```
需要计算导出函子
│
├─ 左导出还是右导出？
│  │
│  ├─ 左正合函子 → 右导出 R^iF
│  │   ├─ Hom(A, -) → Ext^i(A, -)
│  │   └─ Γ(X, -) → H^i(X, -) （层上同调）
│  │
│  └─ 右正合函子 → 左导出 L_iF
│      ├─ A ⊗ - → Tor_i(A, -)
│      └─ - ⊗ B → Tor_i(-, B)
│
├─ 需要计算什么？
│  │
│  ├─ 模的扩张 → Ext^n(A, B)
│  ├─ 挠子 → Tor_n(A, B)
│  ├─ 层上同调 → H^n(X, F)
│  └─ 高阶直接像 → R^nf_*F
│
└─ 用什么分解？
   │
   ├─ 右导出 → 内射分解
   │   └─ 0 → A → I^0 → I^1 → I^2 → ...
   │
   └─ 左导出 → 投射分解
       └─ ... → P_2 → P_1 → P_0 → A → 0
```

## 快速参考表

| 函子 | 正合性 | 导出 | 用途 | 分解 |
|------|--------|------|------|------|
| Hom(A, -) | 左正合 | Ext^i(A, -) | 扩张分类 | 内射 |
| - ⊗ B | 右正合 | Tor_i(-, B) | 挠子 | 投射 |
| Γ(X, -) | 左正合 | H^i(X, -) | 层上同调 | 内射 |
| f_* | 左正合 | R^if_* | 高阶直接像 | 内射 |

---

## 关键问题

**Q: 什么时候用Ext？**
A: 研究模的扩张、分类问题

**Q: 什么时候用Tor？**
A: 研究张量积的精确性、挠子

**Q: 层上同调与导出函子关系？**
A: 层上同调 = 整体截面函子的右导出
