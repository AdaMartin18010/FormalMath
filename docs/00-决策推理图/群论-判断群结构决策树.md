# 群论 - 判断群结构决策树

**适用**: MIT 18.701 Algebra I, Week 1-4  
**表征类型**: 决策树

---

## 判断群结构决策树

```
给定集合 G 与运算 *
│
├─ 步骤1: 验证封闭性
│  │
│  └─ ∀a,b∈G, a*b∈G ?
│     ├─ 否 → [不是群]
│     └─ 是 → 继续
│
├─ 步骤2: 验证结合律
│  │
│  └─ ∀a,b,c∈G, (a*b)*c = a*(b*c) ?
│     ├─ 否 → [不是群]
│     └─ 是 → 继续
│
├─ 步骤3: 验证单位元
│  │
│  └─ ∃e∈G, ∀a∈G: e*a = a*e = a ?
│     ├─ 否 → [不是群]
│     └─ 是 → 继续
│
├─ 步骤4: 验证逆元
│  │
│  └─ ∀a∈G, ∃a⁻¹∈G: a*a⁻¹ = a⁻¹*a = e ?
│     ├─ 否 → [不是群]
│     └─ 是 → [是群!] 继续判断性质
│
└─ [判断群的性质]
   │
   ├─ 是否交换？
   │  ├─ 是 → [Abel群]
   │  └─ 否 → [非交换群]
   │
   ├─ 是否循环？
   │  ├─ 是 → [循环群]
   │  └─ 否 → [非循环群]
   │
   └─ 是否单群？（复杂判断）
```

## 快速判断指南

### 常见群的判断

| 结构 | 封闭 | 结合 | 单位元 | 逆元 | 结果 |
|------|------|------|--------|------|------|
| $(\mathbb{Z}, +)$ | ✅ | ✅ | 0 | $-a$ | 是群，Abel |
| $(\mathbb{N}, +)$ | ✅ | ✅ | 无 | - | 不是群 |
| $(\mathbb{Q}^*, \times)$ | ✅ | ✅ | 1 | $1/a$ | 是群，Abel |
| $(\mathbb{Z}_n, +)$ | ✅ | ✅ | 0 | $n-a$ | 是群，Abel，循环 |
| $(S_n, \circ)$ | ✅ | ✅ | $id$ | 逆置换 | 是群，$n\geq3$非Abel |
| $(GL_n(\mathbb{R}), \times)$ | ✅ | ✅ | $I$ | $A^{-1}$ | 是群，非Abel ($n\geq2$) |
| $(\mathbb{Z}, \times)$ | ✅ | ✅ | 1 | 大多无 | 不是群 |

## 教学要点

### 关键检查清单

- [ ] 封闭性（最容易被忽视）
- [ ] 结合律（通常由运算继承）
- [ ] 单位元（必须验证存在性）
- [ ] 逆元（必须对所有元素存在）

### 常见错误

❌ **错误**: 只检查三个公理，忘记封闭性

✅ **正确**: 封闭性是群的定义前提
