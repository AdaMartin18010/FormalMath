# 层上同调计算方法决策树

**适用**: 代数几何、复几何  
**表征类型**: 决策树

---

## 决策树

```
需要计算H^i(X, F)
│
├─ X是什么？
│  │
│  ├─ 仿射概形 → [H^i = 0 for i > 0] ✅
│  │
│  ├─ 射影空间 → [用欧拉序列计算]
│  │
│  ├─ 曲线 → [用Riemann-Roch]
│  │
│  └─ 一般概形 → 继续
│
├─ F是什么？
│  │
│  ├─ 线丛 O(k) → [用标准分解]
│  ├─ 丰沛层 → [Serre消失定理]
│  ├─ 凝聚层 → [用分解或归纳]
│  └─ 一般层 → [用Čech或导出]
│
├─ 用什么方法？
│  │
│  ├─ 有仿射覆盖？ → [Čech上同调]
│  │   └─ 计算Čech复形
│  │
│  ├─ 有内射分解？ → [导出函子定义]
│  │   └─ 找内射分解I^•
│  │
│  └─ 有松层分解？ → [Godement分解]
│      └─ 计算Γ(X, C^•(F))
│
└─ 需要更高工具？
   │
   ├─ 谱序列 → [Leray谱序列]
   ├─ 基变换 → [上同调与基变换定理]
   └─ 对偶 → [Serre/Grothendieck对偶]
```
