# 实分析 - 判断级数收敛性决策树

**适用场景**: 判断数项级数 $\sum_{n=1}^{\infty} a_n$ 的收敛性
**难度范围**: 本科实分析（MIT 18.100A Week 10-11）
**更新时间**: 2026年4月9日

---

## 决策树总览

```
级数 ∑aₙ
│
├─ 第一步：检查必要条件
│  │
│  └─ lim aₙ = 0 ?
│     ├─ 否 → [级数发散] （由发散判别法）
│     └─ 是 → 继续下一步
│
├─ 第二步：识别级数类型
│  │
│  ├─ 是否为交错级数？
│  │  ├─ 是 → [莱布尼茨判别法流程]
│  │  └─ 否 → 继续
│  │
│  ├─ 是否为几何级数？
│  │  ├─ 是 → [几何级数判别法]
│  │  └─ 否 → 继续
│  │
│  ├─ 是否为p-级数？
│  │  ├─ 是 → [p-级数判别法]
│  │  └─ 否 → 继续
│  │
│  └─ 是否为正项级数？
│     ├─ 是 → [正项级数判别法流程]
│     └─ 否 → [一般级数判别法流程]
│
└─ [绝对收敛性检验]
```

## 详细决策流程

### 节点1: 必要条件检验

**问题**: $\lim_{n \to \infty} a_n = 0$ 是否成立？

**判断方法**:

```
计算 lim aₙ
│
├─ 极限不存在（或不存在且不为∞）
│  └─ [发散] （必要条件不满足）
│
├─ 极限存在但 ≠ 0
│  └─ [发散] （由发散判别法）
│
└─ 极限 = 0
   └─ [继续下一步] （必要条件满足，但非充分）
```

**教学提示**:
> ⚠️ **常见错误**: 认为 $a_n \to 0$ 蕴含级数收敛
>
> **反例**: 调和级数 $\sum \frac{1}{n}$ 发散，尽管 $\frac{1}{n} \to 0$

---

### 节点2: 莱布尼茨判别法流程（交错级数）

**适用条件**: 级数形式为 $\sum (-1)^n b_n$ 或 $\sum (-1)^{n+1} b_n$，其中 $b_n > 0$

**决策流程**:

```
交错级数 ∑(-1)ⁿbₙ
│
├─ 条件1: bₙ 单调递减？
│  │
│  ├─ 否 → [无法判定，尝试其他方法]
│  │
│  └─ 是 → 继续
│
├─ 条件2: lim bₙ = 0？
│  │
│  ├─ 否 → [无法判定，尝试其他方法]
│  │
│  └─ 是 →
│     │
│     └─ [条件收敛] （莱布尼茨判别法）
│         备注：需进一步检验绝对收敛性
│
└─ [检验绝对收敛性] → 节点5
```

**示例分析**:

| 级数 | bₙ | 单调递减 | lim bₙ=0 | 结论 |
|------|-----|----------|----------|------|
| $\sum \frac{(-1)^n}{n}$ | $1/n$ | ✅ | ✅ | 条件收敛 |
| $\sum \frac{(-1)^n}{n^2}$ | $1/n^2$ | ✅ | ✅ | 绝对收敛 |
| $\sum (-1)^n$ | $1$ | ✅ | ❌ | 发散 |

### 节点3: 正项级数判别法流程

**适用条件**: $a_n > 0$ 对所有 $n$

```
正项级数 ∑aₙ
│
├─ 尝试：比较判别法
│  │
│  ├─ 是否容易找到比较级数？
│  │  ├─ 是 → [比较判别法 / 极限比较法]
│  │  └─ 否 → 继续
│  │
│  ├─ 大O分析：aₙ = O(1/nᵖ) ?
│  │  ├─ p > 1 → [收敛] （与p-级数比较）
│  │  ├─ p = 1 → 临界情况
│  │  └─ p < 1 → [发散]
│  │
│  └─ 继续下一步
│
├─ 尝试：比值判别法
│  │
│  ├─ 计算 L = lim |aₙ₊₁/aₙ|
│  │
│  ├─ L < 1 → [绝对收敛]
│  ├─ L > 1 → [发散]
│  └─ L = 1 → [无法判定] → 继续
│
├─ 尝试：根值判别法
│  │
│  ├─ 计算 L = lim sup |aₙ|^(1/n)
│  │
│  ├─ L < 1 → [绝对收敛]
│  ├─ L > 1 → [发散]
│  └─ L = 1 → [无法判定] → 继续
│
├─ 尝试：积分判别法
│  │
│  ├─ 是否存在递减正函数 f 使得 f(n) = aₙ ?
│  │
│  ├─ 是 → [积分判别法]
│  │   ├─ ∫₁^∞ f(x)dx 收敛 → [级数收敛]
│  │   └─ ∫₁^∞ f(x)dx 发散 → [级数发散]
│  │
│  └─ 否 → 继续
│
└─ 尝试：Raabe判别法（比值=1时）
   │
   └─ 计算 lim n(|aₙ/aₙ₊₁| - 1)
      ├─ > 1 → [收敛]
      ├─ < 1 → [发散]
      └─ = 1 → [无法判定]
```

### 节点4: 一般级数判别法流程

**适用条件**: 非正项、非交错的任意级数

```
一般级数 ∑aₙ
│
├─ [检验绝对收敛性]
│  │
│  └─ 对 ∑|aₙ| 应用正项级数判别法
│     │
│     ├─ 收敛 → [绝对收敛] （蕴含原级数收敛）
│     └─ 发散 → 继续
│
├─ [检验条件收敛性]
│  │
│  ├─ 是否为交错级数？
│  │  ├─ 是 → [莱布尼茨判别法]
│  │  └─ 否 → 继续
│  │
│  ├─ Dirichlet判别法
│  │  ├─ 部分和有界？
│  │  ├─ bₙ 单调趋于0？
│  │  └─ 两者皆满足 → [条件收敛]
│  │
│  ├─ Abel判别法
│  │  ├─ ∑aₙ 收敛？
│  │  ├─ bₙ 单调有界？
│  │  └─ 两者皆满足 → [条件收敛]
│  │
│  └─ 其他特殊方法
│
└─ [无法判定 / 发散]
```

### 节点5: 绝对收敛性检验

```
检验 ∑|aₙ| 的收敛性
│
├─ [应用正项级数判别法]
│
├─ 收敛
│  │
│  └─ [级数绝对收敛]
│     ├─ 推论1: 原级数收敛
│     ├─ 推论2: 重排不变性
│     └─ 推论3: 乘积级数性质
│
└─ 发散
   │
   └─ 原级数可能：
      ├─ 发散
      └─ 条件收敛（非绝对收敛）
```

## 常见级数类型速查

| 级数类型 | 形式 | 收敛条件 | 判别法 |
|----------|------|----------|--------|
| 几何级数 | $\sum r^n$ | $|r| < 1$ | 公式求和 |
| p-级数 | $\sum 1/n^p$ | $p > 1$ | 积分判别法 |
| 调和级数 | $\sum 1/n$ | 发散 | 积分判别法 |
| 交错调和 | $\sum (-1)^n/n$ | 条件收敛 | 莱布尼茨 |
| 对数p-级数 | $\sum 1/(n(\ln n)^p)$ | $p > 1$ | 积分判别法 |

## 多维对比矩阵

### 正项级数判别法对比

| 判别法 | 适用条件 | 收敛条件 | 发散条件 | 无法判定 |
|--------|----------|----------|----------|----------|
| **比较法** | 易找比较级数 | $a_n \leq b_n$, $\sum b_n$ 收敛 | $a_n \geq b_n$, $\sum b_n$ 发散 | 不等式不成立 |
| **比值法** | 含阶乘/指数 | $\lim \frac{a_{n+1}}{a_n} < 1$ | $> 1$ | $= 1$ |
| **根值法** | 含n次幂 | $\lim \sqrt[n]{a_n} < 1$ | $> 1$ | $= 1$ |
| **积分法** | $a_n = f(n)$, $f$ 递减 | $\int_1^\infty f < \infty$ | 积分发散 | - |
| **Raabe** | 比值=1时 | $\lim n(\frac{a_n}{a_{n+1}}-1) > 1$ | $< 1$ | $= 1$ |

### 收敛类型层次

```
绝对收敛
│
├─ 条件：∑|aₙ| 收敛
├─ 性质：
│   ├─ 原级数必收敛
│   ├─ 重排不变
│   └─ 柯西乘积收敛
│
条件收敛
│
├─ 条件：∑aₙ 收敛但 ∑|aₙ| 发散
├─ 性质：
│   ├─ 黎曼重排定理（可重排至任意和）
│   └─ 柯西乘积可能发散
│
发散
│
└─ 不满足任何收敛判别法
```

## 教学提示

### 常见误区

| 误区 | 正确理解 |
|------|----------|
| $a_n \to 0$ 则级数收敛 | 仅必要条件，非充分 |
| 正项级数收敛则必绝对收敛 | 正项级数绝对收敛=自身收敛 |
| 比值=1时级数发散 | 无法判定，需用其他方法 |
| 条件收敛级数可任意重排 | 会导致和改变（黎曼定理） |

### 解题策略

1. **先检验必要条件** $a_n \to 0$
2. **识别特殊类型**（几何、p-级数、交错）
3. **正项级数**优先尝试比较法和比值法
4. **比值=1**时尝试Raabe或积分法
5. **一般级数**先检验绝对收敛性
