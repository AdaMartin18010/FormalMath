## 决策树总览

### 什么是数学证明策略决策树？

数学证明策略决策树是一种系统化的方法论工具，帮助数学学习者和研究者：

- **快速定位**适用的证明方法
- **理解**不同方法之间的逻辑关系
- **避免**盲目尝试低效的证明路径
- **建立**结构化的数学思维框架

### 决策树的使用原则

1. **自上而下**：从待证命题的特征出发，逐步细化
2. **多路径并存**：同一问题可能有多种有效解法
3. **动态调整**：实际证明中可能需要回退和重选
4. **经验积累**：决策树是指导而非束缚

### 四大核心决策树

```

数学证明策略体系
│
├─ 决策树1：证明方法选择（逻辑层面）
│  └─ 基于命题结构的通用方法
│
├─ 决策树2：分析学证明策略
│  └─ 极限、连续、微分、积分
│
├─ 决策树3：代数学证明策略
│  └─ 群、环、域、模
│
└─ 决策树4：拓扑学证明策略
   └─ 开集、连续映射、连通性、紧致性

```

### 1.2 节点详解：何时用反证法

#### 适用条件

反证法（Proof by Contradiction）是数学中最强大的证明技巧之一，特别适用于以下情形：

**直接证明路径不明确时**

- 待证命题"P ⇒ Q"的直接证明无从下手
- 假设¬Q后能产生更多可利用的条件
- 问题的否定形式更易刻画

**需要证明"不存在"或"不能"时**

- 不存在性证明
- 不可解性证明
- 不可判定性证明

**结论涉及否定词时**

- 结论本身包含"非"、"不"、"无理"等否定性表述
- 例如：证明某数是无理数、证明某方程无解

#### 经典例子

**例1：√2是无理数**

```

假设√2是有理数，则存在互质的整数p,q使得√2 = p/q
⇒ 2 = p²/q² ⇒ 2q² = p²
⇒ p²是偶数 ⇒ p是偶数（设p=2k）
⇒ 2q² = 4k² ⇒ q² = 2k²
⇒ q²是偶数 ⇒ q是偶数
⇒ p,q都是偶数，与互质矛盾！

```

**例2：素数有无穷多个（欧几里得）**

```

假设素数有限，为p₁, p₂, ..., pₙ
考虑N = p₁p₂...pₙ + 1
N不被任何pᵢ整除，但N>1必有素因子
⇒ 存在不在列表中的素数，矛盾！

```

**例3：实数不可数（康托尔对角线法）**

```

假设[0,1]区间实数可数，排列为r₁, r₂, r₃, ...
构造新数x：第n位小数≠rₙ的第n位
则x∈[0,1]但x≠rₙ对所有n成立
⇒ 矛盾！

```

#### 注意事项

1. **构造性vs非构造性**
   - 反证法往往给出非构造性证明
   - 若需要显式构造，可能需要转换思路

2. **矛盾的清晰度**
   - 必须明确指出矛盾的对象
   - 避免"得出矛盾"而结束证明

3. **与逆否命题的区别**
   - 反证法：假设P且¬Q，推出矛盾
   - 逆否命题：证明¬Q ⇒ ¬P
   - 两者逻辑等价但证明结构不同

4. **常见陷阱**
   - 循环论证：无意中使用了待证结论
   - 矛盾虚假：推导过程中出错导致"矛盾"

### 1.3 节点详解：何时用归纳法

#### 适用条件

数学归纳法适用于以下情况：

**命题与自然数相关**

- 涉及"对所有n∈ℕ"的陈述
- 递推定义的序列或结构
- 有限集上的组合性质

**存在递归结构**

- 问题的规模n可由n-1或更小规模解决
- 具有"搭积木"式的构造特性

**良基关系上**

- 不仅限于ℕ，任何良序集上的归纳
- 超限归纳用于序数
- 结构归纳用于树、公式等

#### 强归纳 vs 弱归纳

| 特征 | 弱归纳（简单归纳） | 强归纳（完全归纳） |
|------|-------------------|-------------------|
| 归纳假设 | 仅假设P(k)成立 | 假设P(i)对所有i≤k成立 |
| 归纳步 | P(k)⇒P(k+1) | (∀i≤k, P(i))⇒P(k+1) |
| 适用场景 | 仅依赖前一项 | 依赖前面多项或任意前项 |
| 等价性 | 两者逻辑等价 | 表达更灵活 |

**弱归纳例子：算术基本定理（分解存在性）**

```

基础步：n=2，显然成立
归纳步：假设n=k时成立
  若k+1是素数，则分解为自身
  若k+1是合数，则k+1=ab（2≤a,b≤k）
  由归纳假设，a,b可分解
  故k+1可分解

```

**强归纳例子：任何n≥2可写成素数乘积**

```

基础步：n=2，成立
强归纳步：假设对所有2≤m≤k成立
  考虑k+1：
  - 若k+1是素数，显然
  - 若k+1=ab（2≤a,b<k+1）
  - 由强归纳，a,b都是素数乘积
  - 故k+1也是素数乘积

```

#### 结构归纳

结构归纳是数学归纳法在递归定义结构上的推广：

**适用结构**：

- 语法树（逻辑公式、程序表达式）
- 树形数据结构
- 递归类型定义的对象

**归纳原理**：

1. 证明基础结构满足性质
2. 证明若子结构满足性质，则复合结构也满足

**例子：证明任何合式公式的括号匹配**

```

基础：原子命题（无括号），显然成立
归纳：
  - 若A,B括号匹配，则(A∧B)也匹配
  - 若A括号匹配，则(¬A)也匹配
  - 若A括号匹配，则(∃xA)也匹配
由结构归纳，所有合式公式括号匹配

```

### 1.4 节点详解：构造性证明 vs 非构造性证明

#### 构造性证明

**特点**：

- 给出满足条件的对象的具体描述
- 提供算法可计算出该对象
- 直觉主义逻辑所接受

**类型**：

1. **显式构造**：直接给出公式或算法
2. **递归构造**：通过递归过程定义
3. **迭代构造**：通过极限过程逼近

**例子**：

```

命题：存在无理数a,b使得aᵇ是有理数

构造性证明：
令a = √2，b = log₂9 = 2log₂3

情况1：若√2^(log₂9)是有理数，得证

情况2：若√2^(log₂9)是无理数
  令a = √2^(log₂9)，b = √2
  则aᵇ = (√2^(log₂9))^√2 = √2^(√2·log₂9)
  这个证明略有技巧，更好的构造：

经典构造：a = √2^√2
  - 若a是有理数，取a=√2^√2，b=√2
  - 若a是无理数，取a=(√2^√2)，b=√2
    则aᵇ = (√2^√2)^√2 = √2² = 2（有理数）

```

#### 非构造性证明

**特点**：

- 证明存在性但不给出具体对象
- 常使用反证法或选择公理
- 经典逻辑中广泛应用

**常用技巧**：

1. **反证法**：假设不存在，推出矛盾
2. **紧致性**：无穷集上的存在性传递到有限子集
3. **选择公理/Zorn引理**：从无穷选择中构造

**例子**：

```

命题：存在不可测集的实数子集

证明：
利用选择公理，在[0,1]上定义等价关系：
x ~ y 当且仅当 x-y∈ℚ
每个等价类选择一代表元，构成Vitali集V
证明V不可测

这是一个纯存在性证明，无法显式构造V

```

#### 选择策略

| 情形 | 建议策略 | 理由 |
|------|----------|------|
| 需要实际计算 | 构造性 | 给出算法或公式 |
| 仅关心存在性 | 两者皆可 | 非构造性可能更简单 |
| 有计算复杂度要求 | 构造性 | 分析效率 |
| 涉及无穷选择 | 非构造性 | 可能无法构造 |

### 1.5 节点详解：不等式证明方法

#### 离散不等式

**归纳法**：

```

伯努利不等式：(1+x)ⁿ ≥ 1+nx（x≥-1, n∈ℕ）
基础：n=1时等号成立
归纳：假设对n=k成立
  (1+x)^(k+1) = (1+x)^k(1+x)
  ≥ (1+kx)(1+x) = 1+(k+1)x+kx²
  ≥ 1+(k+1)x

```

**组合论证**：

```

证明：C(n,k) ≤ nᵏ/k!
组合解释：从n元素选k有序组有n(n-1)...(n-k+1)种
无序组为此除以k!
故C(n,k) = n(n-1)...(n-k+1)/k! ≤ nᵏ/k!

```

#### 连续不等式

**微分法**：

```

证明：eˣ ≥ 1+x（对所有实数x）
令f(x) = eˣ - 1 - x
f'(x) = eˣ - 1
f'(x) = 0 当且仅当 x = 0
f''(x) = eˣ > 0，故x=0是最小值点
f(0) = 0，故f(x) ≥ 0对所有x成立

```

**积分比较**：

```

证明：∑(k=1 to n) 1/k² < 2
利用∫(1 to n+1) 1/x² dx = 1 - 1/(n+1) < 1
且1/k² < ∫(k-1 to k) 1/x² dx（k≥2）
故∑(k=2 to n) 1/k² < ∫(1 to n) 1/x² dx < 1
加上1/1² = 1，总和< 2

```

### 2.2 节点详解：收敛性证明

#### 数列收敛的核心定理

**单调有界定理**：

```

定理：ℝ中单调有界数列必收敛

应用步骤：
1. 证明数列单调（递增或递减）
2. 证明数列有界（上界或下界）
3. 得出结论：收敛
4. （可选）求极限值

例子：证明a₁=1, aₙ₊₁=√(2+aₙ)收敛
单调性：归纳证明aₙ₊₁>aₙ
有界性：归纳证明aₙ<2
由定理知收敛，设极限为L
L=√(2+L) ⇒ L²=2+L ⇒ L=2（舍去负根）

```

**柯西准则**：

```

定理：完备度量空间中，数列收敛⇔它是柯西列

柯西列定义：∀ε>0, ∃N, 使得m,n>N时|aₙ-aₘ|<ε

应用：当单调性难以证明时，验证柯西条件

例子：证明压缩映射序列收敛
设f是压缩映射，|f(x)-f(y)|≤c|x-y|（c<1）

aₙ₊₁=f(aₙ)
则|aₙ₊₁-aₙ|≤cⁿ|a₁-a₀|
对m>n，|aₘ-aₙ|≤∑(k=n to m-1)|aₖ₊₁-aₖ|
≤|a₁-a₀|·cⁿ/(1-c)→0（n→∞）

故是柯西列，收敛

```

#### 级数收敛判别法选择

| 判别法 | 适用条件 | 成功标准 |
|--------|----------|----------|
| 比较判别法 | 已知收敛/发散的参照级数 | 0≤aₙ≤bₙ |
| 极限比较 | aₙ/bₙ有有限正极限 | lim aₙ/bₙ = L∈(0,∞) |
| 比值判别法 | 阶乘、指数、幂混合 | lim aₙ₊₁/aₙ = L<1 |
| 根值判别法 | 含n次幂的形式 | lim aₙ^(1/n) = L<1 |
| 积分判别法 | f(n)形式，f单调减 | ∫₁^∞ f(x)dx收敛 |
| Raabe判别法 | 比值法失效（L=1） | lim n(aₙ/aₙ₊₁-1)>1 |

### 2.3 节点详解：连续性证明

#### ε-δ方法详解

**标准模板**：

```

要证f在x₀连续：
∀ε>0, 需找δ>0使得|x-x₀|<δ ⇒ |f(x)-f(x₀)|<ε

步骤：
1. 分析|f(x)-f(x₀)|，用代数变形
2. 建立与|x-x₀|的关系

3. 根据ε确定δ
4. 验证δ满足要求

例子：证明f(x)=x²在x=1连续

|f(x)-f(1)| = |x²-1| = |x+1||x-1|
若|x-1|<1，则0<x<2，|x+1|<3
故|f(x)-f(1)|<3|x-1|

取δ=min(1, ε/3)
则|x-1|<δ ⇒ |f(x)-1|<ε

```

#### 一致连续性

**定义对比**：

```

点态连续：∀x₀∀ε>0∃δ>0, |x-x₀|<δ⇒|f(x)-f(x₀)|<ε
一致连续：∀ε>0∃δ>0∀x₀, |x-x₀|<δ⇒|f(x)-f(x₀)|<ε

关键区别：点态连续中δ依赖于x₀，一致连续中δ是全局的

例子：f(x)=1/x在(0,1)上
- 点态连续：是（每点都连续）
- 一致连续：否（靠近0时变化率无限增大）

```

**判定定理**：

1. **Lipschitz条件**：|f(x)-f(y)|≤L|x-y| ⇒ 一致连续

2. **康托尔定理**：紧致集上的连续函数必一致连续
3. **导数有界**：|f'|≤M ⇒ Lipschitz（由中值定理）

### 2.4 节点详解：微分证明

#### 中值定理体系

```

Rolle定理 ──→ Lagrange中值定理 ──→ Cauchy中值定理
│               │                    │
└─ f(a)=f(b)   └─ 参数化曲线        └─ L'Hôpital法则
   则f'(c)=0      的切线斜率          的证明基础

应用：
1. 证明不等式
2. 判定单调性
3. 估计函数值
4. 证明唯一性

```

**证明不等式例子**：

```

证明：ln(1+x) ≤ x（x>-1）
令f(t)=ln(1+t)，在[0,x]上应用中值定理
存在c∈(0,x)使得
ln(1+x)-ln(1) = f'(c)·x = x/(1+c)
若x>0，c>0，则1/(1+c)<1，故ln(1+x)<x
若-1<x<0，类似论证
x=0时等号成立

```

#### 隐函数定理

**定理陈述**：

```

设F: ℝⁿ⁺ᵐ→ℝᵐ是C¹函数，F(x₀,y₀)=0
若∂F/∂y在(x₀,y₀)处可逆
则存在x₀的邻域U和唯一函数y=f(x)使得
F(x,f(x))=0对所有x∈U成立

且dy/dx = -(∂F/∂y)⁻¹(∂F/∂x)

```

**应用步骤**：

1. 验证F连续可微
2. 找到满足F(x₀,y₀)=0的点
3. 计算∂F/∂y，验证其可逆（行列式≠0）
4. 得出结论：局部存在隐函数
5. （可选）计算导数

### 2.5 节点详解：积分理论

#### Riemann可积的Lebesgue判别

```

定理：有界函数f在[a,b]上Riemann可积
当且仅当f的不连续点集是零测集

应用：
- 单调函数：至多可数不连续点 ⇒ 可积
- 连续函数：无处不连续 ⇒ 可积
```

#### 反常积分收敛判别

| 类型 | 判别法 | 条件 |
|------|--------|------|
| ∫ₐ^∞ f(x)dx | Cauchy判别 | ∀ε>0, ∃A, A₂>A₁>A⇒|∫(A₁ to A₂)|<ε |

### 3.2 节点详解：群论证明

#### Lagrange定理及其应用

```

Lagrange定理：若H是有限群G的子群，则|H|整除|G|

推论：
1. 元素的阶整除群的阶
2. |G|素数 ⇒ G是循环群

3. Fermat小定理（应用）

证明思路：
- 陪集划分G
- 每个陪集与H等势
- G是陪集的不交并

应用例子：
证明4阶群只有两种结构：ℤ₄或ℤ₂×ℤ₂
由Lagrange，非单位元阶为2或4
若有4阶元，则为ℤ₄
若无非4阶元，则所有元阶为2，可证是ℤ₂×ℤ₂

```

#### 同态基本定理

```

第一同构定理：
设φ: G→H是同态，则
G/Ker φ ≅ Im φ

应用步骤：
1. 构造/识别同态φ
2. 确定Ker φ
3. 确定Im φ
4. 得出同构结论

例子：证明GL(n,ℝ)/SL(n,ℝ) ≅ ℝ*
定义φ(A) = det(A)
Ker φ = {A: det(A)=1} = SL(n,ℝ)
Im φ = ℝ*
故GL(n,ℝ)/SL(n,ℝ) ≅ ℝ*

```

#### Sylow定理

```

第一Sylow定理：
若pᵏ整除|G|，则G有pᵏ阶子群
特别地，p-Sylow子群（pᵏ||G|但pᵏ⁺¹∤|G|）存在

第二Sylow定理：
所有p-Sylow子群彼此共轭

第三Sylow定理：
nₚ ≡ 1 (mod p) 且 nₚ | [G:P]

应用：判定群的结构

例子：证明15阶群是循环群

|G|=15=3×5
n₅ ≡ 1 (mod 5) 且 n₅|3 ⇒ n₅=1
n₃ ≡ 1 (mod 3) 且 n₃|5 ⇒ n₃=1或5

若有唯一5-Sylow P₅和唯一3-Sylow P₃
则都正规，G ≅ P₅×P₃ ≅ ℤ₅×ℤ₃ ≅ ℤ₁₅

```

```

唯一分解整环的两个条件：

1. 分解存在：
   每个非零非单位元可写成有限个不可约元的乘积

   证明方法：
   - 诺特性：升链条件保证过程终止
   - 范数：整数环中|N(a)|递减

   若a=p₁...pₙ=q₁...qₘ，则n=m且重排后相伴

   不可约元都是素元

例子：ℤ是UFD
- 算术基本定理保证分解存在唯一
- 或验证不可约整数就是素数

```

### 4.2 节点详解：连续性证明

#### 开集原像法

```

定理：f: X→Y连续当且仅当Y中任意开集的原像在X中开

应用步骤：
1. 取Y中任意开集V
2. 计算f⁻¹(V) = {x∈X: f(x)∈V}
3. 证明f⁻¹(V)是X中开集

例子：证明投影映射π₁: X×Y→X连续
设U⊆X开，则π₁⁻¹(U) = U×Y
由乘积拓扑定义，U×Y开
故π₁连续

```

#### 粘接引理

```

定理：设X=A∪B，A,B闭
若f|ₐ和f|ᵦ连续，则f连续

证明要点：
F⊆Y闭 ⇒ f⁻¹(F) = (f|ₐ)⁻¹(F) ∪ (f|ᵦ)⁻¹(F)

右边是闭集的有限并，故闭

例子：定义f: [0,2]→ℝ
f(x) = x, x∈[0,1]
f(x) = 2-x, x∈[1,2]
在[0,1]和[1,2]上分别连续
[0,1]和[1,2]闭，[0,2]=[0,1]∪[1,2]
由粘接引理，f在[0,2]连续

```

### 4.3 节点详解：连通性证明

#### 连通性判定

```

定义：X连通当且仅当不能写成两个非空无交开集的并

等价判定：
1. 既开又闭的子集只有∅和X
2. 任意连续函数f: X→{0,1}（离散）是常值
3. 任意分解X=A∪B（A,B非空无交开）不可能

证明策略：
- 直接法：假设分解，推出矛盾
- 连续像法：找到连续满射到连通空间
- 覆盖法：用连通子集覆盖，有公共点

例子：证明[0,1]连通
设[0,1]=A∪B，A,B非空无交开
设0∈A，考虑sup{x: [0,x]⊆A}
利用A,B开，推出矛盾

```

#### 道路连通

```

定义：X道路连通当且仅当任意两点可由道路连接
道路：连续映射γ: [0,1]→X，γ(0)=x，γ(1)=y

证明步骤：
1. 任取两点x,y∈X
2. 显式构造或论证存在γ: [0,1]→X
3. 验证γ连续且端点正确

例子：ℝⁿ道路连通
对任意a,b∈ℝⁿ，定义γ(t) = (1-t)a + tb
这是直线段，显然连续
故ℝⁿ道路连通

重要关系：
道路连通 ⇒ 连通
连通 + 局部道路连通 ⇒ 道路连通

```

### 4.4 节点详解：紧致性证明

#### 开覆盖法

```

定义：X紧致当且仅当任意开覆盖有有限子覆盖

证明步骤：
1. 取任意开覆盖{Uₐ}
2. 论证存在有限子覆盖
   - 直接构造
   - 反证法（假设不存在有限子覆盖，推出矛盾）
   - 利用已知紧致性

例子：证明[0,1]紧致（Heine-Borel定理的一部分）
设{Uₐ}是[0,1]的开覆盖
令S = {x∈[0,1]: [0,x]可被有限覆盖}
证明sup S = 1且1∈S

```

#### 紧致性的保持

```

定理：
1. 紧致集的连续像紧致
2. 紧致空间的闭子集紧致
3. Hausdorff空间中紧致子集闭
4. 紧致集的有限积紧致
5. （Tychonoff）紧致集的任意积紧致

应用：
- 证明空间紧致：作为紧致空间的像或子集
- 证明子集紧致：证明它是某紧致集的闭子集

例子：单位球面Sⁿ⊆ℝⁿ⁺¹紧致
证明：
1. Sⁿ = {x: ||x||=1}是f(x)=||x||的逆像

2. {1}在ℝ中闭，故Sⁿ在ℝⁿ⁺¹中闭
3. Sⁿ有界（||x||=1≤1）

4. 由Heine-Borel，Sⁿ紧致

```

### 4.5 节点详解：分离性公理

#### Hausdorff空间

```

定义：任意不同两点有不相交邻域

重要性质：
1. 紧致子集可分离：K紧致，x∉K，则有不相交开集分离
2. 收敛极限唯一：序列/网收敛至多一个点
3. 紧致+Hausdorff ⇒ 正规

证明Hausdorff的步骤：
1. 任取x≠y
2. 找开集U∋x，V∋y
3. 证明U∩V=∅

例子：度量空间是Hausdorff
对x≠y，令d=d(x,y)>0
取U=B(x,d/2)，V=B(y,d/2)
由三角不等式，U∩V=∅

```

#### Urysohn引理

```

定理（正规空间）：
设X正规，A,B是不相交闭集
则存在连续函数f: X→[0,1]使得
f|ₐ = 0，f|ᵦ = 1

应用：
- 构造连续函数分离闭集
- 证明可度量化
- 建立单位分解

证明思路：
利用正规性，在[0,1]中稠密集上定义
通过连续性延拓到全体

```

### 模式详解

#### 模式1-4：基础逻辑模式

**模式1：直接证明**

```

结构：
1. 假设P成立
2. 通过逻辑推演
3. 得出Q成立

示例：证明两个偶数之和为偶数
设a,b是偶数，则a=2k，b=2ℓ
a+b = 2k+2ℓ = 2(k+ℓ)
故a+b是偶数

```

**模式2：逆否证明**

```

结构：
要证P⇒Q，改为证¬Q⇒¬P

示例：若n²是偶数，则n是偶数
逆否：若n是奇数，则n²是奇数
证明：设n=2k+1，则n²=4k²+4k+1=2(2k²+2k)+1是奇数

```

**模式3-4：存在与全称**

```

全称∀：设x任意，证明P(x)
存在∃：找到具体的x₀使得P(x₀)

结合使用：
∀ε>0 ∃δ>0: ... （ε-δ定义）
证明时：先设任意ε，再构造δ(ε)

```

#### 模式5-8：复合命题模式

**模式5：唯一性证明**

```

两步：存在性 + 唯一性

唯一性证明：
假设x₁,x₂都满足，证明x₁=x₂

示例：证明方程2x+3=7有唯一解
存在：x=2满足
唯一：设2x₁+3=7且2x₂+3=7
则2x₁=2x₂=4，故x₁=x₂=2

```

**模式6：等价证明**

```

双向证明A⇔B：
A⇒B 和 B⇒A

或等价链：
A⇔A₁⇔A₂⇔...⇔B

示例：矩阵可逆的三个等价条件
(1) A可逆
(2) det(A)≠0
(3) Ax=0只有零解

```

#### 模式9-10：分析学核心模式

**模式9：数列收敛**

```

单调有界法：
1. 证明单调（归纳/代数变形）
2. 证明有界（估计/归纳）
3. 得出收敛
4. （可选）取极限求值

柯西法：
∀ε>0, ∃N, m,n>N⇒|aₙ-aₘ|<ε

```

**模式10：级数收敛**

```

比较判别模板：
0 ≤ aₙ ≤ bₙ，∑bₙ收敛 ⇒ ∑aₙ收敛

比值判别模板：
lim |aₙ₊₁/aₙ| = L < 1 ⇒ 绝对收敛

```

#### 模式11-14：分析学进阶模式

**模式11-12：连续性证明**

```

点态连续：
∀ε>0 ∃δ>0: |x-x₀|<δ ⇒ |f(x)-f(x₀)|<ε

一致连续：
∀ε>0 ∃δ>0 ∀x₀: |x-x₀|<δ ⇒ |f(x)-f(x₀)|<ε

```

**模式13：可微性**

```

定义法：
lim(h→0) [f(x+h)-f(x)]/h 存在

隐函数法：
F(x,y)=0，∂F/∂y≠0 ⇒ y是x的可微函数

```

#### 模式15-16：代数学核心模式

**模式15：同构证明**

```

步骤：
1. 定义映射φ: G→H
2. 验证同态：φ(ab)=φ(a)φ(b)
3. 验证单射：Ker φ={e}
4. 验证满射：Im φ=H

或使用同态定理：
证明G/N ≅ H

```

**模式16：正规子群**

```

验证gNg⁻¹=N

或验证是某同态的核

特殊情况：[G:H]=2则H◁G

```

#### 模式17-20：拓扑学核心模式

**模式17：连续性**

```

开集原像：V开 ⇒ f⁻¹(V)开

序列刻画：xₙ→x ⇒ f(xₙ)→f(x)

```

**模式18：连通性**

```

道路连通：构造γ: [0,1]→X连接两点

连通：证明既开又闭集只有∅和X

```

**模式19：紧致性**

```

有限覆盖：任意开覆盖有有限子覆盖

序列紧致：任意序列有收敛子列
（第一可数+T₁空间）

```

**模式20：分离性**

```

Hausdorff：不相交点有不相交邻域

正规：不相交闭集有不交开邻域

```
