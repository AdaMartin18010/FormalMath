# 证明策略选择决策树

> 本决策树用于根据证明目标选择合适的证明策略，提高证明效率与成功率。

---

## 总览图

```

证明目标
├── 存在性证明？
│   ├── 可以显式构造？
│   ├── 唯一性也需要？
│   └── 是无穷对象？
├── 全称命题？
│   ├── 关于自然数？
│   ├── 反证法可行？
│   └── 可分情况？
├── 不等式证明？
│   ├── 可以配方？
│   ├── 凸性相关？
│   └── 积分相关？
├── 同构/等价证明？
│   ├── 可以显式构造映射？
│   ├── 需要验证所有公理？
│   └── 需要函子性？
├── 唯一性证明？
│   ├── 假设两个对象？
│   └── 证明相等？
└── 递归/归纳定义？
    ├── 良基关系？
    └── 归纳步骤？

```

## 一、存在性证明

### 1.1 显式构造 vs 非构造性

```

存在性证明路径
│
├─[Q1] 可以显式构造目标对象？
│   │
│   ├─ 是 → 构造法（Constructive Proof）
│   │   ├── 适用条件:
│   │   │   - 对象可算法生成
│   │   │   - 解有闭式表达式
│   │   │   - 递归可定义
│   │   ├── 执行步骤:
│   │   │   1. 分析对象需满足的条件
│   │   │   2. 设计构造算法/公式
│   │   │   3. 验证构造结果满足要求
│   │   │   4. （可选）证明唯一性
│   │   ├── 经典例子:
│   │   │   - √2的存在性：构造小数逼近序列
│   │   │   - 方程解：给出求根公式
│   │   │   - 稠密性：构造有理数逼近
│   │   └── 注意事项:
│   │       - 构造必须良定义
│   │       - 需验证所有条件
│   │       - 算法复杂度可接受
│   │
│   └─ 否 → 非构造性存在证明
│       ├── 路径A: 紧致性论证
│       │   ├── 适用条件:
│       │   │   - 对象空间紧致（或局部紧致）
│       │   │   - 性质在极限下保持
│       │   ├── 关键定理:
│       │   │   - Bolzano-Weierstrass定理
│       │   │   - Tychonoff定理
│       │   │   - Alaoglu定理（弱*紧致）
│       │   └── 例子: 有界序列有收敛子列
│       │
│       ├── 路径B: 选择公理/Zorn引理
│       │   ├── 适用条件:
│       │   │   - 极大元存在性问题
│       │   │   - 偏序集的全序子集有上界
│       │   ├── 执行步骤:
│       │   │   1. 定义适当的偏序关系
│       │   │   2. 验证全序子集有上界
│       │   │   3. 应用Zorn引理得极大元
│       │   │   4. 证明极大元即为所求
│       │   └── 例子: 向量空间存在基，极大理想存在
│       │
│       └── 路径C: 概率方法（存在性）
│           ├── 适用条件:
│           │   - 对象空间可赋予概率测度
│           │   - 期望或概率可计算
│           ├── 关键原理:
│           │   - 正概率⇒存在性
│           │   - 期望论证
│           │   - Lovász局部引理
│           └── 例子: Ramsey数下界，编码存在性
│
├─[Q2] 唯一性也需要证明？
│   │
│   ├─ 是 → 先证存在 + 再证唯一
│   │   ├── 唯一性证明策略:
│   │   │   - 假设存在两个，证明相等
│   │   │   - 利用对象的特殊性质
│   │   │   - 反证：假设不同导出矛盾
│   │   └── 例子: 极限唯一性，代数闭包唯一性
│   │
│   └─ 否 → 仅需存在性论证
│       └── 可采用上述任意存在性证明方法
│
└─[Q3] 是无穷对象？
    │
    ├─ 是 → Zorn引理/超限归纳
    │   ├── 适用场景:
    │   │   - 无穷维向量空间的基
    │   │   - 代数闭包
    │   │   - 极大理想
    │   │   - Hahn-Banach延拓
    │   ├── 超限归纳步骤:
    │   │   1. 基始：验证对最小元成立
    │   │   2. 后继：假设对α成立，证对α+1成立
    │   │   3. 极限：假设对所有β<α成立，证对α成立
    │   └── 例子: 良序定理证明，超限递归定义
    │
    └─ 否 → 有限搜索/枚举
        ├── 适用条件:
        │   - 搜索空间有限且可遍历
        │   - 有有效剪枝条件
        ├── 算法:
        │   - 回溯算法
        │   - 分支限界
        │   - SAT求解器
        └── 例子: 组合配置的存在性，博弈必胜策略

```

## 二、全称命题证明

### 2.1 自然数相关全称命题

```

全称命题：自然数情形
│
├─[Q1] 命题是否关于所有自然数成立？
│   │
│   ├─ 是 → 数学归纳法
│   │   ├── 标准归纳法
│   │   │   ├── 适用条件: P(n)⇒P(n+1) 自然成立
│   │   │   ├── 执行步骤:
│   │   │   │   1. 基始：证明 P(0) 或 P(1)
│   │   │   │   2. 归纳假设：假设 P(k) 成立
│   │   │   │   3. 归纳步骤：证明 P(k+1)
│   │   │   └── 例子: 求和公式，整除性
│   │   │
│   │   ├── 强归纳法（完全归纳）
│   │   │   ├── 适用条件: 需要用到前面所有情形
│   │   │   ├── 执行步骤:
│   │   │   │   1. 基始：证明基础情形
│   │   │   │   2. 归纳假设：假设对所有 m≤k，P(m) 成立
│   │   │   │   3. 归纳步骤：证明 P(k+1)
│   │   │   └── 例子: 算术基本定理，Fibonacci性质
│   │   │
│   │   ├── 结构归纳法
│   │   │   ├── 适用条件: 递归定义的结构（树、公式等）
│   │   │   ├── 执行步骤:
│   │   │   │   1. 基始：证明对原子元素成立
│   │   │   │   2. 归纳：假设对子结构成立，证对复合结构成立
│   │   │   └── 例子: 逻辑公式的性质，程序正确性
│   │   │
│   │   └── 超限归纳法
│   │       ├── 适用条件: 良序集（不仅是自然数）
│   │       └── 例子: Borel集的性质，递归定义函数
│   │
│   └─ 否 → 一般全称命题方法
│       ├── 直接证明
│       ├── 反证法
│       └── 分情况讨论
│
└─[Q2] 反证法是否可行且有效？
    │
    ├─ 是 → 反证法
    │   ├── 适用条件:
    │   │   - 结论的否定形式更具体
    │   │   - 易于导出矛盾
    │   │   - 存在性命题的否定是全称
    │   ├── 执行步骤:
    │   │   1. 假设结论不成立（否定命题）
    │   │   2. 逻辑推演
    │   │   3. 导出矛盾（与已知/公理/假设矛盾）
    │   │   4. 否定假设，得证
    │   ├── 经典例子:
    │   │   - √2的无理性
    │   │   - 素数无穷多
    │   │   - 实数不可数
    │   └── 注意事项:
    │       - 矛盾必须明确
    │       - 逻辑链条完整
    │       - 双重否定要处理清楚
    │
    └─ 否 → 直接证明/构造性证明
        ├── 直接证明
        │   ├── 适用条件: 条件⇒结论的逻辑路径清晰
        │   ├── 方法:
        │   │   - 逻辑推演
        │   │   - 已知定理直接应用
        │   │   - 等价变形
        │   └── 例子: 代数恒等式，直接计算
        │
        └── 分情况讨论
            ├── 适用条件:
            │   - 问题可分解为互斥情形
            │   - 每种情形有不同的处理
            ├── 执行步骤:
            │   1. 完备分类（穷举所有可能）
            │   2. 对每种情形分别证明
            │   3. 统一结论
            ├── 分类原则:
            │   - 互斥性：各类不交
            │   - 完备性：覆盖所有情况
            │   - 简洁性：分类标准自然
            └── 例子: 整数奇偶分类，符号讨论

```

## 三、不等式证明

```

不等式证明决策树
│
├─[Q1] 是否可以通过配方解决？
│   │
│   ├─ 是 → 配方法
│   │   ├── 适用条件:
│   │   │   - 多项式形式
│   │   │   - 可写成平方和
│   │   ├── 技术要点:
│   │   │   - 完全平方公式
│   │   │   - Lagrange恒等式
│   │   │   - 二次型正定性
│   │   └── 例子:
│   │       - AM-GM不等式的特殊情形
│   │       - a² + b² ≥ 2ab
│   │       - Cauchy-Schwarz的代数证明
│   │
│   └─ 否 → 继续其他方法
│
├─[Q2] 是否与凸性相关？
│   │
│   ├─ 是 → 凸函数方法
│   │   ├── Jensen不等式
│   │   │   ├── 条件: 函数f是凸/凹函数
│   │   │   ├── 形式: f(∑λᵢxᵢ) ≤ ∑λᵢf(xᵢ)
│   │   │   └── 例子: AM-GM不等式，熵不等式
│   │   │
│   │   ├── Karamata不等式（控制不等式）
│   │   │   ├── 条件: 向量控制关系 + 凸函数
│   │   │   └── 例子: 优化不等式
│   │   │
│   │   └── Hermite-Hadamard不等式
│   │       ├── 条件: 凸函数在区间上的积分
│   │       └── 形式: 函数平均值与积分的关系
│   │
│   └─ 否 → 继续其他方法
│
├─[Q3] 是否涉及积分或求和？
│   │
│   ├─ 是 → 积分不等式方法
│   │   ├── Cauchy-Schwarz不等式（积分形式）
│   │   │   └── (∫fg)² ≤ (∫f²)(∫g²)
│   │   │
│   │   ├── Hölder不等式
│   │   │   └── ∫|fg| ≤ (∫|f|^p)^(1/p) · (∫|g|^q)^(1/q)

│   │   │
│   │   ├── Minkowski不等式
│   │   │   └── 三角不等式的L^p版本
│   │   │
│   │   ├── Chebyshev和不等式
│   │   │   └── 同序序列乘积和的不等式
│   │   │
│   │   ├── 重排不等式
│   │   │   └── 序列重排后的极值性质
│   │   │
│   │   └── 积分比较法
│   │       └── 通过被积函数比较
│   │
│   └─ 否 → 代数不等式方法
│
├─[Q4] 是否涉及导数或单调性？
│   │
│   ├─ 是 → 微分方法
│   │   ├── 单调性分析
│   │   │   ├── f'(x) ≥ 0 ⇒ f 单调增
│   │   │   └── 用于比较函数值
│   │   │
│   │   ├── 极值方法
│   │   │   ├── 求导找临界点
│   │   │   └── 二阶导判断极值性质
│   │   │
│   │   ├── Lagrange中值定理
│   │   │   └── 连接函数值与导数
│   │   │
│   │   └── Taylor展开
│   │       └── 局部逼近分析
│   │
│   └─ 否 → 其他代数技巧
│
└─[Q5] 是否对称不等式？
    │
    ├─ 是 → 对称化方法
    │   ├── 对称多项式基本定理
    │   ├── Muirhead不等式
    │   ├── Schur不等式
    │   └── pqr方法（三元对称）
    │
    └─ 否 → 一般方法
        ├── 归一化（齐次不等式）
        ├── 增量法（uvw方法）
        └── 混合变量法

```

## 四、同构/等价证明

```

同构/等价证明决策树
│
├─[Q1] 是否可以显式构造映射？
│   │
│   ├─ 是 → 构造同构映射
│   │   ├── 执行步骤:
│   │   │   1. 定义映射 φ: A → B
│   │   │   2. 证明良定义
│   │   │   3. 证明同态性（保持结构）
│   │   │   4. 证明双射（单射+满射）
│   │   │   5. （可选）构造逆映射
│   │   ├── 验证清单:
│   │   │   - 线性映射：验证线性性 + 核为零 + 维数相同
│   │   │   - 群同构：验证同态 + 双射
│   │   │   - 拓扑同胚：连续性 + 逆连续
│   │   │   - 微分同胚：光滑 + 逆光滑
│   │   └── 例子:
│   │       - 向量空间同构
│   │       - 群同构定理
│   │       - 覆盖空间提升
│   │
│   └─ 否 → 泛性质/范畴论证
│       ├── 泛性质刻画
│       │   ├── 原理: 泛性质唯一确定对象（在同构意义下）
│       │   ├── 适用: 自由对象，张量积，完备化
│       │   └── 方法: 验证两者满足同一泛性质
│       │
│       └── 范畴等价
│           ├── 原理: 寻找等价函子
│           ├── 验证:
│           │   - 函子忠实且全
│           │   - 本质满射
│           └── 例子: Gelfand对偶，Serre-Swan定理
│
├─[Q2] 需要验证所有公理/结构？
│   │
│   ├─ 是 → 逐项验证
│   │   ├── 代数结构:
│   │   │   - 封闭性
│   │   │   - 结合律
│   │   │   - 单位元
│   │   │   - 逆元
│   │   │   - 交换律（如适用）
│   │   │   - 分配律（环情形）
│   │   │
│   │   ├── 拓扑结构:
│   │   │   - 开集原像是开集
│   │   │   - 连续性
│   │   │   - 同胚验证
│   │   │
│   │   └── 序结构:
│   │       - 保序性
│   │       - 完备性保持
│   │
│   └─ 否 → 生成元验证
│       ├── 原理: 验证在生成元上一致即可
│       ├── 适用:
│       │   - 群由生成元表出
│       │   - 线性映射由基向量决定
│       │   - 环由生成元决定
│       └── 方法:
│           1. 确定生成集
│           2. 验证映射在生成元上定义良好
│           3. 验证关系保持
│
└─[Q3] 需要函子性或自然性？
    │
    ├─ 是 → 范畴论方法
    │   ├── 自然变换
    │   │   └── 验证图交换
    │   │
    │   ├── 伴随函子
    │   │   └── 验证Hom集同构
    │   │
    │   └── 可表函子
    │       └── 验证表示对象
    │
    └─ 否 → 具体集合论方法
        ├── 直接构造双射
        ├── 利用选择公理
        └── 基数论证

```

    └── 例子:

        - 完备化（度量空间）的存在唯一性

```

## 六、递归/归纳定义

```

递归定义验证
│
├─[Q1] 是否良基递归？
│   │
