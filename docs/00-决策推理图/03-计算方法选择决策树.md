# 计算方法选择决策树

> 本决策树用于指导数值计算和符号计算方法的选择，平衡精度、效率和稳定性。

---

## 总览图

```

计算问题
├── 线性代数计算？
│   ├── 矩阵规模？
│   ├── 稀疏矩阵？
│   └── 精度要求？
├── 微分方程数值解？
│   ├── 常微分方程？
│   └── 偏微分方程？
├── 优化问题？
│   ├── 凸优化？
│   ├── 约束条件？
│   └── 问题规模？
├── 积分计算？
│   ├── 一维积分？
│   └── 高维积分？
├── 插值与逼近？
│   ├── 数据类型？
│   └── 光滑性要求？
└── 特征值计算？
    ├── 矩阵性质？
    └── 所需特征值？

```

## 一、线性代数计算

### 1.1 线性方程组求解

```

线性方程组 Ax = b
│
├─[Q1] 矩阵规模？
│   │
│   ├─ 小规模（n < 1000）→ 直接法
│   │   ├── LU分解
│   │   │   ├── 适用: 一般稠密矩阵
│   │   │   ├── 复杂度: O(n³)
│   │   │   ├── 存储: O(n²)
│   │   │   ├── 稳定性: 选主元提高稳定性
│   │   │   └── 算法: Doolittle, Crout, 选主元LU
│   │   │
│   │   ├── Cholesky分解
│   │   │   ├── 适用: 对称正定矩阵
│   │   │   ├── 复杂度: O(n³/3)
│   │   │   ├── 存储: O(n²/2)
│   │   │   ├── 稳定性: 无需选主元
│   │   │   └── 算法: 平方根法
│   │   │
│   │   └── QR分解
│   │       ├── 适用: 最小二乘问题，稳定性要求高
│   │       ├── 复杂度: O(2n³/3)
│   │       ├── 稳定性: 数值稳定
│   │       ├── 算法: Householder, Givens, Gram-Schmidt
│   │       └── 应用: 病态问题，QR迭代求特征值
│   │
│   └─ 大规模（n ≥ 1000）→ 迭代法
│       ├── 共轭梯度法（CG）
│       │   ├── 适用: 对称正定矩阵
│       │   ├── 收敛: 有限步收敛（精确算术）
│       │   ├── 复杂度: 每步O(nnz)
│       │   ├── 存储: O(nnz) + O(n)
│       │   └── 预处理: 推荐（不完全Cholesky等）
│       │
│       ├── GMRES
│       │   ├── 适用: 一般非对称矩阵
│       │   ├── 收敛: Krylov子空间方法
│       │   ├── 重启: 限制内存使用（GMRES(k)）
│       │   ├── 复杂度: 每步递增，重启后重置
│       │   └── 预处理: 通常必需
│       │
│       ├── BiCGSTAB
│       │   ├── 适用: 非对称矩阵
│       │   ├── 特点: 短递推，低存储
│       │   ├── 复杂度: 每步O(nnz)
│       │   └── 稳定性: 可能不稳定
│       │
│       └── Jacobi/Gauss-Seidel/SOR
│           ├── 适用: 特定结构（如对角占优）
│           ├── 收敛: 理论条件严格
│           ├── 复杂度: 每步O(nnz)
│           └── 实际: 常作预处理子
│
├─[Q2] 矩阵是否稀疏？
│   │
│   ├─ 是 → 稀疏算法
│   │   ├── 存储格式:
│   │   │   - CSR（压缩稀疏行）
│   │   │   - CSC（压缩稀疏列）
│   │   │   - COO（坐标格式）
│   │   │   - Block稀疏格式
│   │   ├── 稀疏直接法
│   │   │   ├── 排序策略: AMD, Nested Dissection
│   │   │   ├── 填充控制: 符号分解
│   │   │   └── 软件: UMFPACK, SuperLU, MUMPS
│   │   └── 稀疏迭代法
│   │       ├── 矩阵-向量乘法优化
│   │       └── 预处理器设计
│   │
│   └─ 否 → 稠密算法
│       ├── BLAS优化
│       ├── 分块算法
│       └── 并行化（OpenMP/MPI）
│
└─[Q3] 精度要求？
    │
    ├─ 高精度（>机器精度）→ 符号计算/任意精度
    │   ├── 符号计算
    │   │   ├── 方法: 有理数运算，代数数运算
    │   │   ├── 软件: Mathematica, Maple, SymPy
    │   │   └── 适用: 精确解，小矩阵
    │   └── 任意精度浮点
    │       ├── 方法: 高精度算术库
    │       ├── 软件: MPFR, ARPREC
    │       └── 适用: 病态问题，验证
    │
    └─ 普通精度 → 浮点计算
        ├── 单精度: float32
        ├── 双精度: float64（推荐）
        └── 扩展精度: float128（特殊需要）

```

### 1.2 特征值计算

```

特征值问题 Ax = λx
│
├─[Q1] 矩阵性质？
│   │
│   ├─ 对称/Hermite矩阵
│   │   ├── QR算法（对称版本）
│   │   │   ├── 先化为三对角形式
│   │   │   ├── 隐式QR迭代
│   │   │   └── 收敛: 立方收敛
│   │   │
│   │   ├── 分治算法
│   │   │   ├── 适用于大矩阵
│   │   │   └── 可并行化
│   │   │
│   │   └── 对分法 + 反迭代
│   │       ├── 仅需部分特征值
│   │       └── 区间定位
│   │
│   ├─ 一般矩阵
│   │   ├── QR算法（一般版本）
│   │   │   ├── 先化为Hessenberg形式
│   │   │   ├── 位移策略（Wilkinson, Francis）
│   │   │   └── 双位移处理复特征值
│   │   │
│   │   └── Arnoldi/Lanczos迭代
│   │       ├── 大规模矩阵
│   │       ├── 仅需部分特征值（最大/最小模）
│   │       └── 隐式重启技术
│   │
│   └─ 特殊结构
│       ├── 三对角: 特殊QR算法
│       ├── Toeplitz: 快速算法
│       └── 稀疏: 迭代特征值算法
│
└─[Q2] 需要多少特征值？
    │
    ├─ 全部 → QR算法及相关变体
    │
    └─ 部分 → 迭代方法
        ├── 最大/最小模: 幂法/反幂法
        ├── 最大/小实部: 隐式重启Arnoldi
        ├── 区间内: 谱变换 + Lanczos
        └── 聚类附近: 位移-求逆技术

```

## 二、微分方程数值解

### 2.1 常微分方程初值问题

```

ODE初值问题: dy/dt = f(t,y), y(t₀) = y₀
│
├─[Q1] 方程是否刚性？
│   │
│   ├─ 刚性（ stiffness ratio > 10）→ 隐式方法
│   │   ├── 判断标准:
│   │   │   - 特征值差异大
│   │   │   - 稳定性限制步长过小
│   │   │   - 刚性比 |Re(λ_max)|/|Re(λ_min)| >> 1

│   │   │
│   │   ├── 向后Euler法
│   │   │   ├── 阶: 一阶
│   │   │   ├── 稳定性: A-稳定
│   │   │   └── 需要: 每步解非线性方程
│   │   │
│   │   ├── BDF方法（向后差分公式）
│   │   │   ├── 阶: 1-6阶
│   │   │   ├── 稳定性: A-稳定（1-2阶），A(α)-稳定（3-6阶）
│   │   │   └── 应用: DASSL, LSODE
│   │   │
│   │   └── 隐式Runge-Kutta
│   │       ├── Gauss-Legendre: 2s阶，A-稳定
│   │       ├── Radau IIA: 2s-1阶，L-稳定
│   │       └── Lobatto IIIA: 2s-2阶，A-稳定
│   │
│   └─ 非刚性 → 显式方法
│       ├── Euler法
│       │   ├── 阶: 一阶
│       │   ├── 稳定性: 条件稳定
│       │   └── 用途: 教学，简单测试
│       │
│       ├── 改进Euler（Heun法）
│       │   ├── 阶: 二阶
│       │   └── 稳定性: 条件稳定
│       │
│       ├── 经典Runge-Kutta（RK4）
│       │   ├── 阶: 四阶
│       │   ├── 稳定性: 条件稳定
│       │   └── 应用: 通用问题
│       │
│       ├── Runge-Kutta-Fehlberg（RKF45）
│       │   ├── 阶: 4(5)阶嵌入对
│       │   ├── 特点: 自适应步长
│       │   └── 误差控制: 自动调整
│       │
│       └── Dormand-Prince（DOPRI5）
│           ├── 阶: 5(4)阶嵌入对
│           ├── 特点: MATLAB ode45默认
│           └── 效率: 优化局部截断误差
│
├─[Q2] 精度要求？
│   │
│   ├─ 高精度 → 高阶方法
│   │   ├── 高阶Runge-Kutta
│   │   ├── 外推法（Gragg-Bulirsch-Stoer）
│   │   └── 谱延迟校正
│   │
│   └─ 低精度 → 低阶方法
│       ├── Euler法
│       └── 改进Euler
│
└─[Q3] 特殊考虑
    │
    ├─ 守恒量保持 → 辛几何方法
    │   ├── 辛Runge-Kutta
    │   ├── 分裂方法
    │   └── 变分积分器
    │
    ├─ 长时间模拟 → 稳定性优先
    │   └── 选择A-稳定或辛方法
    │
    └─ 事件检测 → 间断处理
        └── 结合根查找算法

```

## 三、优化问题

```

优化问题 min f(x) s.t. constraints
│

│   │   │   │
│   │   │   ├── 拟牛顿法
│   │   │   │   ├── BFGS: 最流行的拟牛顿法

│   │   │   │
│   │   │   └── 随机梯度下降（SGD）

│   │   │       ├── 变体: Momentum, Adam, RMSprop
│   │   │       └── 学习率调度
│   │   │
│   │   └── 约束凸优化
│   │       ├── 等式约束: 拉格朗日乘子法
│   │       ├── 不等式约束
│   │       │   ├── 内点法
│   │       │   │   ├── 原始-对偶路径跟踪
│   │       │   │   └── 障碍函数法
│   │       │   │
│   │       │   ├── 投影梯度法
│   │       │   │   └── 适用于简单约束集
│   │       │   │
│   │       │   ├── 对偶方法
│   │       │   │   └── Uzawa算法，增广拉格朗日

│   │       │   └── 近端方法（Proximal Methods）

│   │
│   └─ 否 → 全局/非凸优化
│       ├── 多起点局部搜索
│       ├── 模拟退火（SA）
│       ├── 遗传算法（GA）
│       ├── 粒子群优化（PSO）

│   │
│   └─ 混合约束
│       └── 序列二次规划（SQP）
│           └── 内点法
│
└─[Q3] 问题规模？
    │
    ├─ 小规模（n < 1000）→ 精确算法
    │   ├── 牛顿法
    │   ├── 内点法

        └── 随机近似方法

## 四、积分计算

### 4.1 一维数值积分

```

一维数值积分 ∫[a,b] f(x)dx
│
├─[Q1] 被积函数是否光滑？
│   │
│   ├─ 光滑（解析或高阶可导）→ 高阶求积
│   │   ├── Gauss-Legendre求积
