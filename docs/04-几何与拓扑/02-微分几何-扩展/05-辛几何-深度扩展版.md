## 14.5.1 引言 / Introduction

辛几何是研究辛流形的数学分支，起源于经典力学的Hamilton形式。与黎曼几何不同，辛几何描述的是相空间的几何结构，具有反对称而非对称的基本形式。辛几何在经典力学、量子力学、表示论和代数几何中有广泛应用。Gromov在1985年引入的伪全纯曲线理论引发了辛拓扑的革命性发展。

**Symplectic geometry is the mathematical study of symplectic manifolds, originating from the Hamiltonian formulation of classical mechanics. Unlike Riemannian geometry, symplectic geometry describes the geometry of phase space with a skew-symmetric rather than symmetric fundamental form.**

## 14.5.2 辛形式与辛流形 / Symplectic Forms and Symplectic Manifolds

**定义 14.5.1** (辛形式 / Symplectic Form)
设 $M$ 是一个 $2n$ 维光滑流形。$M$ 上的一个**辛形式** $\omega$ 是一个闭的、非退化的 $2$-形式，即：

1. **闭性**: $d\omega = 0$
2. **非退化性**: 对所有 $p \in M$，若 $\omega_p(X, Y) = 0$ 对所有 $Y \in T_pM$ 成立，则 $X = 0$

等价地，非退化性意味着 $\omega^n = \omega \wedge \cdots \wedge \omega$（$n$ 次）是一个处处非零的体积形式。

**Definition 14.5.1** (Symplectic Form)
Let $M$ be a $2n$-dimensional smooth manifold. A **symplectic form** $\omega$ on $M$ is a closed, non-degenerate 2-form.

**定义 14.5.2** (辛流形 / Symplectic Manifold)
一个**辛流形** $(M, \omega)$ 是一个光滑流形 $M$ 配备一个辛形式 $\omega$。

**线性辛几何**: 在向量空间 $V$ 上，辛形式 $\Omega$ 满足：
- $\Omega: V \times V \to \mathbb{R}$ 是双线性、反对称、非退化的
- 存在基 $\{e_1, \ldots, e_n, f_1, \ldots, f_n\}$ 使得：
$$\Omega(e_i, e_j) = 0, \quad \Omega(f_i, f_j) = 0, \quad \Omega(e_i, f_j) = \delta_{ij}$$
这称为**标准辛基**。

**定理 14.5.1** (辛流形的维数 / Dimension of Symplectic Manifolds)
任何辛流形都是偶数维的。

**证明思路**: 在线性代数中，任何反对称双线性形式在奇数维空间中必退化。

**辛体积**: 辛形式 $\omega$ 诱导一个自然的体积形式：
$$\text{vol}_\omega = \frac{1}{n!} \omega^n$$
这是辛几何中Liouville测度的基础。

## 14.5.3 Darboux定理 / Darboux's Theorem

**定理 14.5.2** (Darboux定理 / Darboux's Theorem)
设 $(M, \omega)$ 是一个 $2n$ 维辛流形。对于任意点 $p \in M$，存在局部坐标 $(q^1, \ldots, q^n, p_1, \ldots, p_n)$（称为**Darboux坐标**或**标准辛坐标**），使得：
$$\omega = \sum_{i=1}^n dq^i \wedge dp_i$$

**Theorem 14.5.2** (Darboux's Theorem)
Let $(M, \omega)$ be a $2n$-dimensional symplectic manifold. For any point $p \in M$, there exist local coordinates in which $\omega$ takes the standard form.

**证明思路**: 
1. 由线性代数，在 $T_pM$ 上可选择标准辛基
2. 利用Moser同痕技巧，通过一族辛形式 $\omega_t = (1-t)\omega_0 + t\omega$ 的稳定性
3. 应用Poincaré引理保证局部闭形式的恰当性

**重要性**: Darboux定理表明辛几何没有局部不变量（不像黎曼几何的曲率），所有辛流形在局部都是标准空间 $(\mathbb{R}^{2n}, \omega_0)$。辛几何的全局不变量才是研究的重点。

**线性Darboux定理**: 任何有限维辛向量空间 $(V, \Omega)$ 都存在辛基，使得 $\Omega$ 的矩阵为：
$$J = \begin{pmatrix} 0 & I_n \\ -I_n & 0 \end{pmatrix}$$

## 14.5.4 Hamilton力学 / Hamiltonian Mechanics

**定义 14.5.3** (Hamilton向量场 / Hamiltonian Vector Field)
设 $(M, \omega)$ 是辛流形，$H: M \to \mathbb{R}$ 是光滑函数（称为**Hamilton函数**）。**Hamilton向量场** $X_H$ 由以下条件唯一确定：
$$\iota_{X_H}\omega = dH$$
即对任意向量场 $Y$：
$$\omega(X_H, Y) = dH(Y) = Y(H)$$

**Definition 14.5.3** (Hamiltonian Vector Field)
The Hamiltonian vector field $X_H$ is uniquely defined by $\iota_{X_H}\omega = dH$.

**非退化性的作用**: 由于 $\omega$ 非退化，映射 $X \mapsto \iota_X\omega$ 是向量场与 $1$-形式之间的同构，因此 $X_H$ 唯一存在。

**Hamilton方程**: 在Darboux坐标 $(q^i, p_i)$ 下：
$$X_H = \sum_{i=1}^n \left(\frac{\partial H}{\partial p_i} \frac{\partial}{\partial q^i} - \frac{\partial H}{\partial q^i} \frac{\partial}{\partial p_i}\right)$$
因此Hamilton方程为：
$$\dot{q}^i = \frac{\partial H}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial H}{\partial q^i}$$

**定理 14.5.3** (能量守恒 / Conservation of Energy)
$H$ 沿 $X_H$ 的流线是常数，即 $X_H(H) = 0$。

**证明**: $X_H(H) = dH(X_H) = \omega(X_H, X_H) = 0$（由反对称性）。

**例子 14.5.1** (谐振子 / Harmonic Oscillator)
$M = T^*\mathbb{R} \cong \mathbb{R}^2$，$\omega = dq \wedge dp$，$H = \frac{1}{2}(p^2 + q^2)$。
Hamilton方程：$\dot{q} = p$, $\dot{p} = -q$，解为圆 $q^2 + p^2 = \text{const}$。

**例子 14.5.2** (Kepler问题 / Kepler Problem)
$H = \frac{1}{2}|p|^2 - \frac{\mu}{|q|}$，描述行星运动。轨道为椭圆、抛物线或双曲线。

## 14.5.5 辛同胚与守恒量 / Symplectomorphisms and Conserved Quantities

**定义 14.5.4** (辛同胚 / Symplectomorphism)
设 $(M_1, \omega_1)$ 和 $(M_2, \omega_2)$ 是辛流形。微分同胚 $\varphi: M_1 \to M_2$ 称为**辛同胚**（或**正则变换**），若：
$$\varphi^*\omega_2 = \omega_1$$

**定义 14.5.5** (辛群 / Symplectic Group)
辛向量空间 $(V, \Omega)$ 的线性辛自同构构成**辛群** $Sp(V, \Omega)$。在标准基下：
$$Sp(2n, \mathbb{R}) = \{A \in GL(2n, \mathbb{R}) : A^TJA = J\}$$

**定理 14.5.4** (Liouville定理 / Liouville's Theorem)
Hamilton流保持辛体积，即若 $\varphi_t$ 是 $X_H$ 的流，则 $\varphi_t^*\omega = \omega$。

**泊松括号**: 对 $f, g \in C^\infty(M)$，定义：
$$\{f, g\} = \omega(X_f, X_g) = X_f(g) = -X_g(f)$$
在Darboux坐标下：
$$\{f, g\} = \sum_{i=1}^n \left(\frac{\partial f}{\partial q^i}\frac{\partial g}{\partial p_i} - \frac{\partial f}{\partial p_i}\frac{\partial g}{\partial q^i}\right)$$

**定理 14.5.5** (泊松括号的性质)
$(C^\infty(M), \{\cdot, \cdot\})$ 构成Lie代数：
1. **反对称**: $\{f, g\} = -\{g, f\}$
2. **Jacobi恒等式**: $\{f, \{g, h\}\} + \{g, \{h, f\}\} + \{h, \{f, g\}\} = 0$
3. **Leibniz法则**: $\{fg, h\} = f\{g, h\} + g\{f, h\}$

**Noether定理**: 若Hamilton函数 $H$ 在辛同胚 $\varphi$ 下不变，则对应的生成函数是守恒量。

## 14.5.6 重要例子 / Important Examples

**例子 14.5.3** (余切丛 / Cotangent Bundle)
任何流形 $Q$ 的余切丛 $M = T^*Q$ 有标准辛结构。设 $\pi: T^*Q \to Q$ 是投影，**Liouville $1$-形式** $\theta$ 定义为：
$$\theta_\alpha(X) = \alpha(d\pi(X)), \quad \alpha \in T^*Q, X \in T_\alpha(T^*Q)$$
辛形式为 $\omega = -d\theta$。在局部坐标 $(q^i, p_i)$ 下：$\theta = \sum p_i dq^i$，$\omega = \sum dq^i \wedge dp_i$。

**例子 14.5.4** (Kähler流形 / Kähler Manifolds)
Kähler流形 $(M, g, J, \omega)$ 同时是黎曼流形、复流形和辛流形，其中 $\omega(X, Y) = g(JX, Y)$。例如：
- $\mathbb{C}^n$ 配备标准结构
- 复射影空间 $\mathbb{C}P^n$ 配备Fubini-Study形式
- 复环面
- 紧Riemann曲面

**例子 14.5.5** (环面 $T^{2n}$)
$\mathbb{R}^{2n}/\mathbb{Z}^{2n}$ 配备标准辛形式 $\omega = \sum dx^i \wedge dy^i$ 是一个紧辛流形。

## 14.5.7 历史发展 / Historical Development

### 经典力学起源 (1687-1900)
- **1687**: Newton《自然哲学的数学原理》
- **1788**: Lagrange《分析力学》，Lagrange形式
- **1833**: Hamilton引入Hamilton形式
- **1834-1835**: Hamilton发表关于光线和动力学的论文

### 数学形式化 (1900-1980)
- **1890**: Poincaré在《天体力学新方法》中研究相空间几何
- **1905**: Cartan发展外微分形式
- **1940s**: Weyl引入"symplectic"一词（来自希腊语"复杂"）
- **1960s**: Arnold、Smale、Moser发展现代辛几何

### 辛拓扑革命 (1980-至今)
- **1985**: Gromov引入伪全纯曲线
- **1988**: Floer创建Floer同调
- **1990s**: 格罗莫夫-威滕不变量、镜像对称

## 14.5.8 现代发展 / Modern Developments

### 辛拓扑
- **Gromov-Witten不变量**: 计数伪全纯曲线
- **Floer同调**: 无穷维Morse理论
- **辛场论**: Eliashberg-Givental-Hofer

### 镜像对称
Kontsevich的同调镜像对称猜想：两个Calabi-Yau流形的导出范畴等价与它们的辛几何和复几何对偶相关。

### 应用领域
- **天体力学**: 三体问题、KAM理论
- **分子动力学**: 辛积分器保持能量
- **控制理论**: 几何控制
- **量子力学**: 几何量子化
