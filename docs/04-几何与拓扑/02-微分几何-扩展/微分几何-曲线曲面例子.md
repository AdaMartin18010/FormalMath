# 微分几何曲线与曲面例子

## 曲线例子

### 1. 螺旋线

**参数方程**：$\gamma(t) = (a\cos t, a\sin t, bt)$

**曲率和挠率**：
$$\kappa = \frac{a}{a^2 + b^2}, \quad \tau = \frac{b}{a^2 + b^2}$$

### 2. 悬链线

**方程**：$y = a \cosh(x/a)$

**性质**：曲线上每点的张力水平分量相同。

---

## 曲面例子

### 1. 球面

**参数方程**：
$$X(u,v) = (R\sin u \cos v, R\sin u \sin v, R\cos u)$$

**第一基本形式**：
$$I = R^2 du^2 + R^2\sin^2 u \, dv^2$$

**高斯曲率**：$K = \frac{1}{R^2}$（常数正曲率）

### 2. 环面

**参数方程**：
$$X(u,v) = ((R + r\cos u)\cos v, (R + r\cos u)\sin v, r\sin u)$$

**高斯曲率**：
$$K = \frac{\cos u}{r(R + r\cos u)}$$

- $K > 0$：外侧
- $K < 0$：内侧
- $K = 0$：顶部和底部圆周

### 3. 悬链面（极小曲面）

**参数方程**：
$$X(u,v) = (a\cosh(v/a)\cos u, a\cosh(v/a)\sin u, v)$$

**平均曲率**：$H = 0$（极小曲面）

## 习题

### 习题1

计算圆$\gamma(t) = (R\cos t, R\sin t)$的曲率。

**解答**：
$\gamma'(t) = (-R\sin t, R\cos t)$，$|\gamma'(t)| = R$
$\gamma''(t) = (-R\cos t, -R\sin t)$

$\kappa = \frac{|\gamma' \times \gamma''|}{|\gamma'|^3} = \frac{R^2}{R^3} = \frac{1}{R}$

### 习题2

证明：曲面上曲线的测地曲率满足$\kappa_g = 0$当且仅当曲线是测地线。

**证明**：测地曲率是曲率向量在切平面上的投影。测地线定义为加速度向量处处垂直于曲面的曲线，即测地曲率为零。

### 习题3

计算平面$z = 0$的高斯曲率。

**解答**：$K = 0$（平面是可展曲面）。
