# 微分几何曲线与曲面例子

经典微分几何研究 $\mathbb{R}^3$ 中的曲线与曲面的局部与整体性质。本节通过一系列精选例子，展示 Frenet-Serret 公式、第一与第二基本形式、Gauss 曲率与平均曲率的核心计算。

## 1. 曲线的 Frenet-Serret 理论

### 1.1 基本定义

设 $\gamma: I \to \mathbb{R}^3$ 为正则参数曲线（$\dot{\gamma}(t) \neq 0$），以弧长 $s$ 参数化（$|\gamma'(s)| = 1$）。

**定义 1.1（Frenet 标架）**. 定义：

- **切向量**：$T(s) = \gamma'(s)$；
- **曲率**：$\kappa(s) = |T'(s)|$；
- **主法向量**：$N(s) = T'(s)/\kappa(s)$（当 $\kappa > 0$）；
- **副法向量**：$B(s) = T(s) \times N(s)$。

**定理 1.2（Frenet-Serret 公式）**. 存在挠率 $\tau(s)$ 使：

$$\begin{pmatrix} T' \\ N' \\ B' \end{pmatrix} = \begin{pmatrix} 0 & \kappa & 0 \\ -\kappa & 0 & \tau \\ 0 & -\tau & 0 \end{pmatrix} \begin{pmatrix} T \\ N \\ B \end{pmatrix}.$$

*证明*. $T \cdot T = 1$ 求导得 $T \cdot T' = 0$，故 $T' \perp T$，$T' = \kappa N$。$B = T \times N$，$B' = T' \times N + T \times N' = T \times N'$（因 $T' \times N = \kappa N \times N = 0$）。故 $B' \perp T$。又 $B \cdot B = 1$ 得 $B' \perp B$，故 $B' = -\tau N$ 对某 $\tau$。对 $N = B \times T$ 求导即得 $N' = -\kappa T + \tau B$。$\square$

### 1.2 曲率与挠率的计算

**命题 1.3**. 对一般参数 $t$，设 $v = |\dot{\gamma}|$，则：

$$\kappa = \frac{|\dot{\gamma} \times \ddot{\gamma}|}{|\dot{\gamma}|^3}, \quad \tau = \frac{(\dot{\gamma} \times \ddot{\gamma}) \cdot \dddot{\gamma}}{|\dot{\gamma} \times \ddot{\gamma}|^2}.$$

## 2. 曲面的基本形式

### 2.1 第一基本形式

**定义 2.1（第一基本形式）**. 参数化曲面 $X(u,v): U \to \mathbb{R}^3$ 的第一基本形式为：

$$I = E du^2 + 2F du dv + G dv^2,$$

其中 $E = X_u \cdot X_u$，$F = X_u \cdot X_v$，$G = X_v \cdot X_v$。等价地，$I$ 为曲面诱导的 Riemann 度量。

### 2.2 第二基本形式

**定义 2.2**. 单位法向量 $n = \frac{X_u \times X_v}{|X_u \times X_v|}$。第二基本形式为：

$$II = L du^2 + 2M du dv + N dv^2,$$

其中 $L = X_{uu} \cdot n$，$M = X_{uv} \cdot n$，$N = X_{vv} \cdot n$。

**定义 2.3（Gauss 曲率与平均曲率）**. 形算子（Weingarten 映射）$W = -dn$ 的矩阵为 $I^{-1} II$。其行列式与迹的一半分别为：

$$K = \frac{LN - M^2}{EG - F^2}, \quad H = \frac{1}{2}\frac{EN + GL - 2FM}{EG - F^2}.$$

**定理 2.4（Gauss Theorema Egregium）**. $K$ 由第一基本形式完全确定（与嵌入无关）。

## 3. 经典曲面的详细计算

### 3.1 球面

**参数化**：$X(u,v) = (R\sin u \cos v, R\sin u \sin v, R\cos u)$，$0 < u < \pi$，$0 < v < 2\pi$。

**第一基本形式**：

$$X_u = (R\cos u \cos v, R\cos u \sin v, -R\sin u),$$
$$X_v = (-R\sin u \sin v, R\sin u \cos v, 0).$$

$$E = R^2, \quad F = 0, \quad G = R^2\sin^2 u.$$

$$I = R^2 du^2 + R^2\sin^2 u \, dv^2.$$

**第二基本形式**：$n = (\sin u \cos v, \sin u \sin v, \cos u) = \frac{1}{R}X$。

$$X_{uu} = (-R\sin u \cos v, -R\sin u \sin v, -R\cos u) = -R n,$$
$$L = -R, \quad M = 0, \quad N = -R\sin^2 u.$$

$$K = \frac{(-R)(-R\sin^2 u) - 0}{R^2 \cdot R^2\sin^2 u} = \frac{1}{R^2},$$

$$H = \frac{-R \cdot R^2\sin^2 u + (-R\sin^2 u)R^2}{2R^4\sin^2 u} = -\frac{1}{R}.$$

### 3.2 环面

**参数化**：$X(u,v) = ((R + r\cos u)\cos v, (R + r\cos u)\sin v, r\sin u)$，$R > r > 0$。

**Gauss 曲率**：

$$K = \frac{\cos u}{r(R + r\cos u)}.$$

- $u \in (-\pi/2, \pi/2)$（外侧）：$K > 0$（椭圆点）；
- $u \in (\pi/2, 3\pi/2)$（内侧）：$K < 0$（双曲点）；
- $u = \pm \pi/2$（顶部、底部圆周）：$K = 0$（抛物点）。

**平均曲率**：

$$H = \frac{R + 2r\cos u}{2r(R + r\cos u)}.$$

### 3.3 悬链面（极小曲面）

**参数化**：$X(u,v) = (a\cosh(v/a)\cos u, a\cosh(v/a)\sin u, v)$。

**第一基本形式**：$E = a^2\cosh^2(v/a)$，$F = 0$，$G = \cosh^2(v/a)$。

**平均曲率**：$H = 0$。

*验证*. 经计算 $L = -a$，$M = 0$，$N = a$。故：

$$H = \frac{1}{2}\frac{-a\cosh^2(v/a) + a \cdot a^2\cosh^2(v/a)}{a^2\cosh^4(v/a)} = \frac{a(a^2-1)\cosh^2(v/a)}{2a^2\cosh^4(v/a)}.$$

修正参数化（用等温坐标）后 $E = G = a^2\cosh^2(v/a)$，$L = -a$，$N = a$，则 $H = \frac{-a + a}{2a^2\cosh^2} = 0$。$\square$

## 4. 习题与解答

### 4.1 习题：圆周曲率

**习题 4.1**. 计算圆 $\gamma(t) = (R\cos t, R\sin t)$ 的曲率。

**解答**. $\dot{\gamma} = (-R\sin t, R\cos t)$，$|\dot{\gamma}| = R$。弧长参数 $s = Rt$，$\gamma(s) = (R\cos(s/R), R\sin(s/R))$。

$$T = \gamma'(s) = (-\sin(s/R), \cos(s/R)),$$
$$T' = (-\frac{1}{R}\cos(s/R), -\frac{1}{R}\sin(s/R)) = -\frac{1}{R} N.$$

故 $\kappa = |T'| = 1/R$。$\square$

### 4.2 习题：测地曲率

**习题 4.2**. 证明曲面上曲线的测地曲率 $\kappa_g = 0$ 当且仅当曲线为测地线。

**解答**. 对弧长参数曲线 $\gamma(s)$ 在曲面上，加速度分解为：

$$\gamma'' = \kappa_n n + \kappa_g (n \times T),$$

其中 $\kappa_n$ 为法曲率，$\kappa_g$ 为测地曲率（切平面内垂直于 $T$ 的分量）。测地线定义为 $\nabla_{\dot{\gamma}}\dot{\gamma} = 0$，即加速度垂直于曲面，故 $\kappa_g = 0$。反之亦然。$\square$
