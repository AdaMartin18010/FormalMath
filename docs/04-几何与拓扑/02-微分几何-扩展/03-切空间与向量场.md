## 14.3.1 引言 / Introduction

切空间（tangent space）是微分几何中最核心也最具抽象性的概念之一。与欧氏空间中我们可以"看到"的切平面不同，抽象流形上的切空间没有外在的包容空间作为参照，必须通过内禀的方式加以定义。Princeton MAT 355课程在第四至六周详细探讨了切空间的三种等价定义——速度向量（geometric curves）、导子（derivations）与余切空间的对偶——并在此基础上建立了向量场、李括号与弗罗贝尼乌斯可积性理论的完整框架。

这一理论的历史脉络可追溯至19世纪末嘉当（Élie Cartan）对活动标架法（method of moving frames）的系统化发展，以及20世纪初外尔（Hermann Weyl）和李（Sophus Lie）将群论与微分几何相结合的尝试。切向量作为"无穷小位移"的严格化，使得我们可以在没有坐标的情况下表述平行输运、曲率与测地线等核心几何概念。Lee（2012）的第三章至第四章以及第十四章至第十九章，为切空间与向量场理论提供了现代最清晰、最完整的教科书级阐述。

向量场不仅是几何对象，也是动力系统（dynamical system）的生成元。一个光滑向量场$X$在流形$M$上定义了一个常微分方程组，其解曲线称为$X$的积分曲线。当$k$个向量场满足特定的对合（involutive）条件时，弗罗贝尼乌斯定理保证存在积分子流形，这一定理在偏微分方程、控制论以及广义相对论（如Frobenius定理保证超曲面正交性条件）中具有根本重要性。

## 14.3.2 切空间的多重等价定义 / Equivalent Definitions of Tangent Spaces

### 几何定义：速度向量 / Geometric Definition: Velocity Vectors

设$M$为$n$维光滑流形，$p \\in M$。考虑所有经过$p$点的光滑曲线的集合：
\\[
\\mathcal{C}_p = \\{\\gamma: (-\\epsilon, \\epsilon) \\to M : \\gamma \\text{ 光滑}, \\gamma(0) = p\\}.
\\]

**定义 2.1**（切向量的几何定义）. 两条曲线$\\gamma_1, \\gamma_2 \\in \\mathcal{C}_p$称为在$p$点**等价**，若存在$p$附近的某个坐标卡$(U, \\varphi)$，使得
\\[
(\\varphi \\circ \\gamma_1)'(0) = (\\varphi \\circ \\gamma_2)'(0) \\in \\mathbb{R}^n.
\\]
可以验证这一定义不依赖于坐标卡的选取。$\\gamma$的等价类记为$\\gamma'(0)$，称为$p$点的一个**切向量**。$p$点所有切向量构成的集合称为$M$在$p$的**切空间**，记为$T_p M$。

**命题 2.2**. $T_p M$具有自然的$n$维实向量空间结构。

*证明思路*：取定$p$附近的一个坐标卡$(U, \\varphi)$，坐标为$(x^1, \\ldots, x^n)$。映射$\\Phi: T_p M \\to \\mathbb{R}^n$，$\\Phi([\\gamma]) = (\\varphi \\circ \\gamma)'(0)$是良定义的双射。通过$\\Phi$将$\\mathbb{R}^n$的向量空间结构转移到$T_p M$上。若换用另一坐标卡$(V, \\psi)$，转移映射为 Jacobi 矩阵$\\frac{\\partial \\psi^i}{\\partial x^j}(\\varphi(p))$，这是$\\mathbb{R}^n$上的线性同构，因此所诱导的向量空间结构与坐标卡的选取无关。$\\square$

在坐标$(x^1, \\ldots, x^n)$下，切空间$T_p M$的一组标准基由**坐标切向量**
\\[
\\left.\\frac{\\partial}{\\partial x^1}\\right|_p, \\ldots, \\left.\\frac{\\partial}{\\partial x^n}\\right|_p

\\]
给出，其中$\\left.\\frac{\\partial}{\\partial x^i}\\right|_p$表示第$i$条坐标曲线的速度向量：$\\gamma_i(t) = \\varphi^{-1}(\\varphi(p) + t e_i)$。任意切向量$v \\in T_p M$可唯一表示为

\\[
v = \\sum_{i=1}^n v^i \\left.\\frac{\\partial}{\\partial x^i}\\right|_p.

\\]

### 代数定义：导子 / Algebraic Definition: Derivations

切向量的第二种等价定义强调其作为"方向导数"的作用。

**定义 2.3**（导子）. 映射$v: C^\\infty(M) \\to \\mathbb{R}$称为$p$点的一个**导子**（derivation），若满足：
1. **线性性**：$v(af + bg) = a v(f) + b v(g)$，对$a, b \\in \\mathbb{R}$，$f, g \\in C^\\infty(M)$；
2. **莱布尼茨法则**：$v(fg) = f(p) v(g) + g(p) v(f)$。

$p$点所有导子构成的集合记为$\\mathcal{D}_p M$，同样构成$n$维实向量空间。

**定理 2.4**（切空间与导子的同构）. 存在典范同构$T_p M \\cong \\mathcal{D}_p M$。

*证明思路*：给定速度向量$v = [\\gamma] \\in T_p M$，定义对应的导子为
\\[
v(f) = \\left.\\frac{d}{dt}\\right|_{t=0} f(\\gamma(t)).

\\]
可以直接验证这满足导子的两条公理。反之，给定导子$D \\in \\mathcal{D}_p M$，在坐标卡下定义曲线$\\gamma(t) = \\varphi^{-1}(\\varphi(p) + t (D(x^1), \\ldots, D(x^n)))$，则$D$恰好对应$\\gamma'(0)$。这一对应是线性同构。$\\square$

导子定义的优越性在于完全不依赖于曲线，仅通过光滑函数代数$C^\\infty(M)$即可刻画切向量。这种观点为后期代数几何中Zariski切空间的定义提供了原型，也凸显了微分几何与交换代数的深刻联系。

### 余切空间与微分 / Cotangent Spaces and Differentials

与切空间对偶的概念是**余切空间**（cotangent space）。

**定义 2.5**（余切空间）. $p$点的余切空间定义为切空间的对偶空间：
\\[
T_p^* M = (T_p M)^* = \\{\omega: T_p M \\to \\mathbb{R} : \\omega \\text{ 线性}\\}.
\\]
余切空间的元素称为**余切向量**或**余向量**（covector）。

在坐标卡$(U, \\varphi)$下，$T_p^* M$的标准基为$\\{dx^1|_p, \\ldots, dx^n|_p\\}$，满足

\\[
dx^i|_p \\left(\\left.\\frac{\\partial}{\\partial x^j}\\right|_p\\right) = \\delta^i_j.

\\]

**定义 2.6**（微分）. 设$F: M \\to N$为光滑映射，$p \\in M$。$F$在$p$点的**微分**（differential）是指线性映射$dF_p: T_p M \\to T_{F(p)} N$，定义为
\\[
dF_p(v)(f) = v(f \\circ F), \\quad \\forall v \\in T_p M, \\ f \\in C^\\infty(N).
\\]

在局部坐标下，$dF_p$的矩阵表示正是$F$的 Jacobi 矩阵$\\frac{\\partial \\widehat{F}^i}{\\partial x^j}(\\varphi(p))$。这一定义统一了多元微积分中的全微分概念，并直接导出了浸入、淹没与嵌入的秩条件描述。

**命题 2.7**（链式法则）. 若$F: M \\to N$与$G: N \\to P$为光滑映射，则
\\[
d(G \\circ F)_p = dG_{F(p)} \\circ dF_p.
\\]

*证明*：对任意$v \\in T_p M$和$h \\in C^\\infty(P)$，
\\[
d(G \\circ F)_p(v)(h) = v(h \\circ G \\circ F) = dF_p(v)(h \\circ G) = dG_{F(p)}(dF_p(v))(h). \\square
\\]

## 14.3.3 向量场与李括号 / Vector Fields and Lie Brackets

### 光滑向量场 / Smooth Vector Fields

**定义 3.1**（向量场）. $M$上的一个**向量场**（vector field）$X$是指在每个点$p \\in M$指定一个切向量$X_p \\in T_p M$的映射。若对任意$f \\in C^\\infty(M)$，函数$X(f): M \\to \\mathbb{R}$，$p \\mapsto X_p(f)$都是光滑的，则称$X$为**光滑向量场**。

所有光滑向量场构成的集合记为$\\mathfrak{X}(M)$。在坐标卡$(U, \\varphi)$下，向量场可局部表示为
\\[
X|_U = \\sum_{i=1}^n X^i \\frac{\\partial}{\\partial x^i},

\\]
其中$X^i: U \\to \\mathbb{R}$为光滑函数，称为$X$的**分量**。

$\\mathfrak{X}(M)$不仅是实向量空间，还是$C^\\infty(M)$-模：对$f \\in C^\\infty(M)$和$X \\in \\mathfrak{X}(M)$，$(fX)_p = f(p) X_p$定义了新的光滑向量场$fX$。

### 李括号 / Lie Bracket

两个光滑向量场$X, Y$的复合$X \\circ Y$和$Y \\circ X$作为$C^\\infty(M)$上的算子一般不再是导子（不满足莱布尼茨法则），但它们的交换子却是一个新的导子。

**定义 3.2**（李括号）. 设$X, Y \\in \\mathfrak{X}(M)$。$X$与$Y$的**李括号**（Lie bracket）定义为
\\[
[X, Y] = X \\circ Y - Y \\circ X,
\\]
即对$f \\in C^\\infty(M)$，
\\[
[X, Y](f) = X(Y(f)) - Y(X(f)).
\\]

**命题 3.3**. $[X, Y] \\in \\mathfrak{X}(M)$，即李括号仍是光滑向量场。

*证明思路*：直接验证$[X, Y]$满足导子的莱布尼茨法则。对$f, g \\in C^\\infty(M)$，
\\[
\begin{aligned}
[X, Y](fg) &= X(Y(fg)) - Y(X(fg)) \\
&= X(f Y(g) + g Y(f)) - Y(f X(g) + g X(f)) \\
&= X(f) Y(g) + f X(Y(g)) + X(g) Y(f) + g X(Y(f)) \\
&\\quad - Y(f) X(g) - f Y(X(g)) - Y(g) X(f) - g Y(X(f)) \\
&= f [X, Y](g) + g [X, Y](f). \\square
\end{aligned}
\\]

**定理 3.4**（李括号的基本性质）. 李括号满足以下性质：
1. **双线性性**：$[aX + bY, Z] = a[X, Z] + b[Y, Z]$，$[Z, aX + bY] = a[Z, X] + b[Z, Y]$；
2. **反对称性**：$[X, Y] = -[Y, X]$；
3. **Jacobi 恒等式**：$[X, [Y, Z]] + [Y, [Z, X]] + [Z, [X, Y]] = 0$；
4. **与光滑函数的交互**：$[fX, gY] = fg[X, Y] + f X(g) Y - g Y(f) X$。

满足上述性质的代数结构称为**李代数**（Lie algebra）。因此$(\\mathfrak{X}(M), [\\cdot, \\cdot])$是实数域上的一个无穷维李代数。

在局部坐标下，李括号有显式的分量表达式。设$X = \\sum_i X^i \\partial_i$，$Y = \\sum_j Y^j \\partial_j$（这里$\\partial_i = \\frac{\\partial}{\\partial x^i}$），则
\\[
[X, Y] = \\sum_{i,j} \\left(X^i \\frac{\\partial Y^j}{\\partial x^i} - Y^i \\frac{\\partial X^j}{\\partial x^i}\\right) \\frac{\\partial}{\\partial x^j}.
\\]
特别地，$[\\partial_i, \\partial_j] = 0$，即坐标向量场总是互相对易的。

### 向量场的流 / Flows of Vector Fields

向量场$X$可视为常微分方程
\\[
\\frac{d}{dt} \\gamma(t) = X_{\\gamma(t)}
\\]
的右端项。该ODE的解给出了$X$的**积分曲线**（integral curve）与**流**（flow）。

**定义 3.5**（积分曲线）. 光滑曲线$\\gamma: I \\to M$（$I \\subset \\mathbb{R}$为开区间）称为$X$的积分曲线，若对所有$t \\in I$有$\\gamma'(t) = X_{\\gamma(t)}$。

**定理 3.6**（积分曲线的存在唯一性）. 设$X \\in \\mathfrak{X}(M)$，$p \\in M$。则存在包含$0$的开区间$I_p$和唯一的积分曲线$\\gamma_p: I_p \\to M$满足$\\gamma_p(0) = p$。

若$X$是**完备的**（complete），即所有极大积分曲线都定义在整个$\\mathbb{R}$上，则对任意$t \\in \\mathbb{R}$可定义全局**流映射**
\\[
\\theta_t: M \\to M, \\quad \\theta_t(p) = \\gamma_p(t).
\\]

**定理 3.7**（流的性质）. 若$X$完备，则流映射$\\{\\theta_t\\}_{t \\in \\mathbb{R}}$满足：
1. $\\theta_0 = \\mathrm{id}_M$；
2. $\\theta_t \\circ \\theta_s = \\theta_{t+s}$，对所有$t, s \\in \\mathbb{R}$；
3. 对每个$t$，$\\theta_t: M \\to M$是微分同胚。

满足上述性质的$\\mathbb{R}$参数微分同胚族称为$M$上的一个**单参数微分同胚群**（one-parameter group of diffeomorphisms）。

**命题 3.8**（李括号与流的交换性）. 设$X, Y \\in \\mathfrak{X}(M)$，其流分别为$\\theta_t, \\phi_s$。则$[X, Y] = 0$当且仅当$\\theta_t \\circ \\phi_s = \\phi_s \\circ \\theta_t$对所有充分小的$t, s$成立。

这一命题深刻揭示了李括号的几何意义：$[X, Y]$度量了两个向量场所生成流之间的"非交换性"。

## 14.3.4 分布、积分曲线与弗罗贝尼乌斯定理 / Distributions, Integral Curves, and Frobenius Theorem

### 分布与积分子流形 / Distributions and Integral Submanifolds

**定义 4.1**（分布）. $M$上一个**$k$维分布**（$k$-dimensional distribution）$D$是指在每个点$p \\in M$指定一个$k$维子空间$D_p \\subset T_p M$的赋值。若对每点$p \\in M$存在邻域$U$和$k$个光滑向量场$X_1, \\ldots, X_k \\in \\mathfrak{X}(U)$，使得对所有$q \\in U$有$D_q = \\mathrm{span}\\{X_1|_q, \\ldots, X_k|_q\\}$，则称$D$为**光滑分布**。

**定义 4.2**（积分子流形）. 一个浸入子流形$N \\subset M$称为分布$D$的**积分子流形**（integral manifold），若对所有$q \\in N$有$T_q N = D_q$。

**定义 4.3**（对合分布）. 分布$D$称为**对合的**（involutive），若对任意在$D$中的光滑向量场$X, Y$（即$X_p, Y_p \\in D_p$对所有$p$），有$[X, Y]$仍在$D$中。

显然，若$D$存在过每点的积分子流形，则$D$必须是对合的：因为积分子流形本身是一个流形，其上任意两个切向量场的李括号仍是切向量场。弗罗贝尼乌斯定理断言：这一必要条件也是充分的。

**定理 4.4**（弗罗贝尼乌斯定理，局部形式）. 设$D$是$M$上的$k$维光滑对合分布。则对任意$p \\in M$，存在$p$的坐标邻域$(U, \\varphi)$，坐标为$(x^1, \\ldots, x^n)$，使得
\\[
D|_U = \\mathrm{span}\\left\\{\\frac{\\partial}{\\partial x^1}, \\ldots, \\frac{\\partial}{\\partial x^k}\\right\\}.

\\]
等价地，坐标切片$\\{x^{k+1} = c^{k+1}, \\ldots, x^n = c^n\\}$是$D$的积分子流形。

*证明思路*（归纳法概要）：这是Princeton MAT 355第六周的顶点定理之一。

**$k=1$情形**：一维分布由单个非零向量场$X$张成。由常微分方程的存在唯一性定理，过每点存在唯一的积分曲线，这正是一维积分子流形。此时弗罗贝尼乌斯定理退化为ODE基本定理。

**归纳步骤**：假设定理对$k-1$维分布成立。设$D$为$k$维对合分布，$p \\in M$。选取在$p$附近张成$D$的向量场$X_1, \\ldots, X_k$。通过对流$\\theta_t$的适当坐标变换（这里$\\theta_t$是$X_1$的流），可以假设$X_1 = \\frac{\\partial}{\\partial x^1}$。然后考虑由$X_2, \\ldots, X_k$在坐标切片$x^1 = 0$上诱导的$(k-1)$维分布$\\widetilde{D}$。利用对合性条件$[X_1, X_i] \\in D$以及归纳假设，可构造出使$\\widetilde{D}$坐标化的局部坐标，再延拓到整个邻域。详细证明见Lee（2012）定理19.21。$\\square$

弗罗贝尼乌斯定理有重要的全局形式：若$D$是$M$上的$k$维对合分布，则$M$可被$D$的极大连通积分子流形所**叶化**（foliation）。这些积分子流形称为**叶片**（leaves），它们两两不相交或完全重合。

**应用 4.5**（超曲面正交性）. 在广义相对论中，一个类时向量场$u$定义了时空的一个$3$维空间分布（空间超曲面分布）。弗罗贝尼乌斯定理断言：这一分布可积（即存在整体的时间切片）当且仅当$u$的旋度（vorticity）为零。这是证明宇宙学中FLRW度规存在同步坐标系的基础。

## 14.3.5 计算实例 / Computational Examples

### 实例 1：$S^2$上的向量场

考虑$S^2 \\subset \\mathbb{R}^3$，参数化为$(x, y, z)$满足$x^2 + y^2 + z^2 = 1$。定义
\\[
X = -y \\frac{\\partial}{\\partial x} + x \\frac{\\partial}{\\partial y}, \\quad Y = -z \\frac{\\partial}{\\partial y} + y \\frac{\\partial}{\\partial z}.
\\]
这里$X, Y$应理解为$\\mathbb{R}^3$中切于$S^2$的向量场。首先验证$X$切于$S^2$：考虑函数$f(x, y, z) = x^2 + y^2 + z^2 - 1$，则对任意$p \\in S^2$，$X_p(f) = -y \\cdot 2x + x \\cdot 2y = 0$。同理$Y_p(f) = 0$。因此$X, Y$都是$S^2$上的光滑向量场。

计算李括号$[X, Y]$：
\\[
\begin{aligned}
[X, Y] &= X(Y(x))\\frac{\\partial}{\\partial x} + X(Y(y))\\frac{\\partial}{\\partial y} + X(Y(z))\\frac{\\partial}{\\partial z} \\
&\\quad - Y(X(x))\\frac{\\partial}{\\partial x} - Y(X(y))\\frac{\\partial}{\\partial y} - Y(X(z))\\frac{\\partial}{\\partial z}.
\end{aligned}
\\]
由于$Y(x) = 0$，$Y(y) = -z$，$Y(z) = y$，以及$X(x) = -y$，$X(y) = x$，$X(z) = 0$，我们有
\\[
\begin{aligned}
X(Y(x)) &= 0, & Y(X(x)) &= Y(-y) = z, \\
X(Y(y)) &= X(-z) = 0, & Y(X(y)) &= Y(x) = 0, \\
X(Y(z)) &= X(y) = x, & Y(X(z)) &= Y(0) = 0.
\end{aligned}
\\]
因此
\\[
[X, Y] = -z \\frac{\\partial}{\\partial x} + x \\frac{\\partial}{\\partial z}.
\\]
这是$S^2$上的另一个光滑向量场。注意$[X, Y] \\neq 0$，说明由$X$和$Y$生成的单参数流不交换。

著名的**毛球定理**（Hairy Ball Theorem）断言：$S^2$上不存在处处非零的连续向量场。换言之，任何光滑向量场$X \\in \\mathfrak{X}(S^2)$必有零点。上述$X$在北极$(0, 0, 1)$和南极$(0, 0, -1)$处为零。这一拓扑结果是Poincaré-Hopf定理的特例，深刻联系于$\\chi(S^2) = 2$。

### 实例 2：李群上的左不变向量场

设$G$为李群，$e \\in G$为单位元。对任意$a \\in G$，左平移$L_a: G \\to G$，$L_a(g) = ag$是微分同胚。

**定义**. 向量场$X \\in \\mathfrak{X}(G)$称为**左不变的**（left-invariant），若对所有$a \\in G$有$(L_a)_* X = X$，即$(dL_a)_g(X_g) = X_{ag}$对所有$g \\in G$成立。

左不变向量场完全由其在单位元处的值$X_e \\in T_e G$决定：$X_a = (dL_a)_e(X_e)$。因此左不变向量场的集合与$T_e G$典范同构。$T_e G$配上李括号结构称为$G$的**李代数**，记为$\\mathfrak{g}$。

**例子**：考虑$G = GL(n, \\mathbb{R})$，单位元$e = I_n$。$T_{I_n} GL(n, \\mathbb{R}) \\cong M_n(\\mathbb{R})$。对$A \\in M_n(\\mathbb{R})$，对应的左不变向量场在$g \\in GL(n, \\mathbb{R})$处为$X_g = gA$（矩阵乘法）。李括号$[X_A, X_B]$对应的单位元处切向量为$AB - BA$。因此$\\mathfrak{gl}(n, \\mathbb{R})$的李代数结构就是矩阵交换子$[A, B] = AB - BA$。

### 实例 3：分布的弗罗贝尼乌斯检验

在$\\mathbb{R}^3$上考虑由向量场
\\[
X = \\frac{\\partial}{\\partial x} + y \\frac{\\partial}{\\partial z}, \\quad Y = \\frac{\\partial}{\\partial y}
\\]
张成的2维分布$D$。计算李括号：
\\[
[X, Y] = X(Y(x))\\partial_x + X(Y(y))\\partial_y + X(Y(z))\\partial_z - Y(X(x))\\partial_x - Y(X(y))\\partial_y - Y(X(z))\\partial_z.
\\]
由于$Y(x) = Y(y) = Y(z) = 0$为常数，前三个项为零。而$X(x) = 1$，$X(y) = 0$，$X(z) = y$，故$Y(X(x)) = 0$，$Y(X(y)) = 0$，$Y(X(z)) = 1$。因此
\\[
[X, Y] = -\\frac{\\partial}{\\partial z}.
\\]
检验$[X, Y]$是否在$D$中：$D$由$X = \\partial_x + y \\partial_z$和$Y = \\partial_y$张成。假设$[X, Y] = a X + b Y$，则
\\[
-\\partial_z = a(\\partial_x + y \\partial_z) + b \\partial_y = a \\partial_x + b \\partial_y + ay \\partial_z.
\\]
比较系数得$a = 0$，$b = 0$，但$a y = 0 \\neq -1$，矛盾。因此$[X, Y] \\not\\in D$，$D$不是对合的，不存在过每点的2维积分子流形。

再考虑分布$D' = \\mathrm{span}\\{\\partial_x, \\partial_y\\}$。显然$[\\partial_x, \\partial_y] = 0 \\in D'$，故$D'$是对合的。其积分子流形是平行于$xy$平面的平面$\\{z = c\\}$。
