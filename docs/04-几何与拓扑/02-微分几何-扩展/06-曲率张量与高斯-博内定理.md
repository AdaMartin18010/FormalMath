## 14.6.1 引言 / Introduction

曲率是微分几何的灵魂概念，它精确量化了空间在某点处偏离平直的程度。从欧几里得几何的平行公设，到高斯（Carl Friedrich Gauss）1827年关于曲面内禀曲率的革命性发现，再到黎曼（Bernhard Riemann）将曲率推广到任意$n$维流形，曲率理论经历了近两百年不间断的深化与发展。高斯-博内定理（Gauss-Bonnet Theorem）是这一理论中最璀璨的明珠之一：它将曲面上高斯曲率的积分——一个纯粹的分析量——与曲面的欧拉示性数——一个纯粹的拓扑不变量——紧密联系起来，堪称局部几何与整体拓扑完美融合的典范。

Princeton MAT 355课程在第十三至十五周深入探讨了曲率张量的代数结构、各种曲率不变量（截面曲率、Ricci曲率、数量曲率）的几何意义，以及高斯-博内定理的证明与应用。do Carmo（1992）《Riemannian Geometry》第四章和第八章，以及Lee（2012）第七章和第十三章，是国际上最权威的教学资源。本文档在系统呈现这些理论的同时，配以丰富的计算实例，帮助读者从抽象定义过渡到具体计算。

曲率在当代数学与物理学中具有核心地位。爱因斯坦场方程将时空的Ricci曲率与物质-能量分布直接关联；在几何分析中，曲率流（如Ricci flow、均值曲率流）通过演化度量来改变曲率，从而解决深刻的分类问题；在计算机图形学与机器学习中，离散曲率为曲面重建和流形学习提供了基本工具。

## 14.6.2 黎曼曲率张量 / Riemann Curvature Tensor

### 曲率算子与曲率张量 / Curvature Operator and Curvature Tensor

Levi-Civita联络的曲率度量了向量沿无穷小环路平行输运后的变化。

**定义 2.1**（曲率算子）. 设$(M, g)$为黎曼流形，$\\nabla$为Levi-Civita联络。**曲率算子**（curvature operator）$R(X, Y): \\mathfrak{X}(M) \\to \\mathfrak{X}(M)$定义为
\\[
R(X, Y)Z = \\nabla_X \\nabla_Y Z - \\nabla_Y \\nabla_X Z - \\nabla_{[X, Y]} Z.
\\]

**定义 2.2**（黎曼曲率张量）. **黎曼曲率张量**（Riemann curvature tensor）$Rm$是$(1, 3)$型张量场，定义为
\\[
Rm(X, Y, Z, W) = g(R(X, Y)Z, W).
\\]
在局部坐标下，其分量为
\\[
R^l_{ijk} = \\partial_i \\Gamma^l_{jk} - \\partial_j \\Gamma^l_{ik} + \\sum_m \\Gamma^l_{im} \\Gamma^m_{jk} - \\sum_m \\Gamma^l_{jm} \\Gamma^m_{ik},
\\]
\\[
R_{ijkl} = \\sum_m g_{im} R^m_{jkl}.
\\]

**定理 2.3**（黎曼曲率张量的代数恒等式）. 黎曼曲率张量满足以下对称性：
1. **反对称性**：$R_{ijkl} = -R_{jikl} = -R_{ijlk}$；
2. **第一Bianchi恒等式**：$R_{ijkl} + R_{iklj} + R_{iljk} = 0$；
3. **配对对称性**：$R_{ijkl} = R_{klij}$；
4. **第二Bianchi恒等式**：$\\nabla_i R_{jklm} + \\nabla_j R_{kilm} + \\nabla_k R_{ijlm} = 0$。

*证明思路*：恒等式1-3是纯粹的代数性质，可通过曲率算子的定义直接验证。第一Bianchi恒等式反映了无挠性的几何约束；配对对称性则来源于度量相容性。恒等式4（微分Bianchi恒等式）需要利用$R(X, Y)Z$的定义以及Jacobi恒等式，经过仔细的协变导数运算得到。详见Lee（2012）命题7.4与定理7.5。$\\square$

上述对称性将黎曼曲率张量的独立分量数从$n^4$大幅减少。在$n$维黎曼流形上，独立分量的数目为$\\frac{n^2(n^2 - 1)}{12}$。例如：
- $n = 2$：仅1个独立分量（高斯曲率）；
- $n = 3$：6个独立分量；
- $n = 4$：20个独立分量。

### 曲率与平行输运 / Curvature and Parallel Transport

曲率张量的几何意义可通过平行输运来阐释。设$p \\in M$，$v, w \\in T_p M$为两个线性无关的切向量。考虑由$v$和$w$张成的无穷小平行四边形：沿$v$移动$\\epsilon$，再沿$w$移动$\\epsilon$，然后沿$-v$移动$\\epsilon$，最后沿$-w$移动$\\epsilon$回到$p$。将向量$Z \\in T_p M$沿此环路平行输运一周后，其变化量的一阶近似为$\\epsilon^2 R(v, w)Z$。

这一解释说明：曲率为零当且仅当平行输运与路径无关（至少局部上）。因此，平坦黎曼流形（如欧氏空间$\\mathbb{R}^n$和环面$T^n$配备平坦度量）的黎曼曲率张量恒为零。反之，若$Rm \\equiv 0$，则流形局部等距于$\\mathbb{R}^n$。

## 14.6.3 截面曲率、Ricci曲率与数量曲率 / Sectional, Ricci, and Scalar Curvature

### 截面曲率 / Sectional Curvature

**定义 3.1**（截面曲率）. 设$\\Pi \\subset T_p M$为一个2维平面（截面），$\\{v, w\\}$为$\\Pi$的一组基。$\\Pi$的**截面曲率**（sectional curvature）定义为
\\[
K(\\Pi) = \\frac{Rm(v, w, w, v)}{|v \\wedge w|^2} = \\frac{g(R(v, w)w, v)}{g(v, v)g(w, w) - g(v, w)^2}.

\\]

截面曲率度量了以$\\Pi$为切平面的无穷小测地三角形的内角和与$\\pi$的偏离。$K(\\Pi) > 0$意味着该方向上的"三角形"内角和大于$\\pi$（如球面）；$K(\\Pi) < 0$意味着内角和小于$\\pi$（如双曲空间）。

**定理 3.2**（Schur引理）. 设$M$为维数$n \\geq 3$的连通黎曼流形。若截面曲率$K(\\Pi)$在每点$p$与截面$\\Pi$的选取无关（即$K(\\Pi) = f(p)$仅依赖于点），则$f$必为常数。此时$M$称为**常曲率空间**（space form）。

*证明思路*：由第二Bianchi恒等式和$n \\geq 3$的条件，可以推出$\\nabla f = 0$，因此$f$为常数。$\\square$

完备、单连通的常曲率空间在等距意义下只有三种：
- $K = 0$：欧氏空间$\\mathbb{R}^n$；
- $K = 1$：单位球面$S^n$；
- $K = -1$：双曲空间$\\mathbb{H}^n$。

### Ricci曲率与数量曲率 / Ricci and Scalar Curvature

对黎曼曲率张量的两个指标进行缩并（contraction），得到Ricci曲率张量。

**定义 3.3**（Ricci曲率）. **Ricci曲率张量**$Rc$是$(0, 2)$型对称张量，定义为
\\[
Rc(X, Y) = \\mathrm{tr}(Z \\mapsto R(Z, X)Y) = \\sum_{i=1}^n Rm(e_i, X, Y, e_i),
\\]
其中$\\{e_i\\}$为$T_p M$的标准正交基。在坐标下，$R_{ij} = \\sum_k R^k_{ikj}$。

Ricci曲率度量了度量体积元与欧氏体积元的二阶偏离。具体来说，在黎曼法坐标下，体积元有如下展开：
\\[
\\sqrt{\\det(g_{ij})} = 1 - \\frac{1}{6} \\sum_{i,j} R_{ij} x^i x^j + O(|x|^3).

\\]

**定义 3.4**（数量曲率）. **数量曲率**（scalar curvature）$S$是Ricci曲率的迹：
\\[
S = \\mathrm{tr}_g Rc = \\sum_{i,j} g^{ij} R_{ij} = \\sum_i Rc(e_i, e_i).
\\]

数量曲率是一个点函数，反映了该点处无穷小球面的平均面积与欧氏空间中相应球面面积的偏离。$S > 0$意味着局部上面积收缩得比欧氏空间快；$S < 0$意味着扩张得更快。

### Einstein流形 / Einstein Manifolds

**定义 3.5**（Einstein流形）. 若黎曼流形$(M, g)$的Ricci曲率与度量成比例，即存在常数$\\lambda$使得
\\[
Rc = \\lambda g,
\\]
则称$(M, g)$为**Einstein流形**。

当$n \\geq 3$时，由第二Bianchi恒等式可证$\\lambda$必为常数。Einstein流形是爱因斯坦场方程$Rc - \\frac{1}{2} S g + \\Lambda g = T$在真空情形（$T = 0$）下的解，因此在广义相对论中具有特殊重要性。$S^n$、$\\mathbb{R}^n$、$\\mathbb{H}^n$以及某些紧致齐性空间（如复射影空间$\\mathbb{C}P^n$配备Fubini-Study度量）都是Einstein流形的例子。

## 14.6.4 高斯-博内定理 / Gauss-Bonnet Theorem

### 二维情形 / The 2-Dimensional Case

**定理 4.1**（高斯-博内定理，全局形式）. 设$(M, g)$为紧致可定向的2维黎曼流形（无边）。则
\\[
\\int_M K \\ dA = 2\\pi \\chi(M) = 2\\pi (2 - 2g),
\\]
其中$K$为高斯曲率（即数量曲率，在2维时$K = S/2$），$dA$为面积元，$\\chi(M)$为欧拉示性数，$g$为亏格（genus）。

*证明思路*（概述）：高斯-博内定理的证明有多种途径，这里给出基于向量场指标理论的拓扑证明梗概（Hopf的"umkehrmappe"方法）。

1. 在$M$上选取一个只有孤立零点的光滑向量场$X$。由Poincaré-Hopf定理，$X$的零点指标和等于$\\chi(M)$。
2. 在每个零点$p$附近，选取适当的局部坐标使$X$形如$X = u \\partial_u + v \\partial_v$（指标为+1）或类似形式。
3. 在$M$挖去零点小圆盘后的区域$M'$上，$X$非零。考虑$X$方向上的角度1-形式$\\omega = d\\theta$，它度量了沿曲线移动时$X$的旋转角度。
4. 在$M'$的边界（即围绕各零点的小圆周）上积分$\\omega$，利用联络形式与曲率形式的关系$d\\omega = -K \\ dA$，可以证明每个零点对积分的贡献恰好是其指标乘以$2\\pi$。
5. 综合所有零点的贡献，得到$\\int_M K \\ dA = 2\\pi \\sum \\mathrm{ind}(p) = 2\\pi \\chi(M)$。

完整的分析证明（利用测地三角形剖分）可见do Carmo（1992）定理5.3。$\\square$

**例子 4.2**.
- $S^2$（$g = 0$，$\\chi = 2$）：$\\int_{S^2} K \\ dA = 4\\pi$。由于$K \\equiv 1$（单位球面），面积为$4\\pi$，吻合。
- $T^2$（$g = 1$，$\\chi = 0$）：$\\int_{T^2} K \\ dA = 0$。对平坦环面，$K \\equiv 0$，显然成立。
- 亏格$g$的闭曲面：$\\int_M K \\ dA = 4\\pi(1 - g)$。

高斯-博内定理深刻揭示了拓扑对几何的约束：无论度量如何变化，高斯曲率在曲面上的积分——一个全局几何量——必须由拓扑完全锁定。这是局部-整体原理的最纯粹体现之一。

### 带边界情形与推广 / Boundary Case and Generalizations

**定理 4.3**（带边界的高斯-博内定理）. 设$M$为紧致可定向带边2维黎曼流形，$\\partial M$为分段光滑边界，$\\kappa_g$为边界的测地曲率，$\\theta_i$为角点处的转角。则
\\[
\\int_M K \\ dA + \\int_{\\partial M} \\kappa_g \\ ds + \\sum_i \\theta_i = 2\\pi \\chi(M).
\\]

**高维推广**（Chern-Gauss-Bonnet定理）：陈省身（Shiing-Shen Chern）在1944年将高斯-博内定理推广到偶数维黎曼流形。对$2n$维紧致定向黎曼流形，
\\[
\\int_M \\mathrm{Pf}(\\Omega) = (2\\pi)^n \\chi(M),
\\]
其中$\\mathrm{Pf}(\\Omega)$是由曲率2-形式矩阵$\\Omega$定义的Pfaffian。这是陈省身最著名的数学成就之一，也是现代示性类理论的起点。

## 14.6.5 计算实例 / Computational Examples

### 实例 1：$S^2$的曲率计算

对$S^2$的度量$g = d\\phi^2 + \\sin^2\\phi \\ d\\theta^2$，已求得克里斯托费尔符号：
\\[
\\Gamma^\\phi_{\\theta\\theta} = -\\sin\\phi \\cos\\phi, \\quad \\Gamma^\\theta_{\\phi\\theta} = \\Gamma^\\theta_{\\theta\\phi} = \\cot\\phi.
\\]

计算黎曼曲率分量$R^\\phi_{\\theta\\phi\\theta}$：
\\[
\begin{aligned}
R^\\phi_{\\theta\\phi\\theta} &= \\partial_\\phi \\Gamma^\\phi_{\\theta\\theta} - \\partial_\\theta \\Gamma^\\phi_{\\phi\\theta} + \\Gamma^\\phi_{\\phi\\phi}\\Gamma^\\phi_{\\theta\\theta} + \\Gamma^\\phi_{\\phi\\theta}\\Gamma^\\theta_{\\theta\\theta} - \\Gamma^\\phi_{\\theta\\phi}\\Gamma^\\phi_{\\theta\\theta} - \\Gamma^\\phi_{\\theta\\theta}\\Gamma^\\theta_{\\theta\\theta} \\
&= \\partial_\\phi(-\\sin\\phi \\cos\\phi) - 0 + 0 + 0 - 0 - 0 \\
&= -\\cos^2\\phi + \\sin^2\\phi = \\sin^2\\phi - \\cos^2\\phi.
\end{aligned}
\\]

 lowering指标：$R_{\\phi\\theta\\phi\\theta} = g_{\\phi\\phi} R^\\phi_{\\theta\\phi\\theta} = 1 \\cdot (\\sin^2\\phi - \\cos^2\\phi) \\ldots$

等等，让我重新仔细计算。实际上对2维流形，黎曼张量只有一个独立分量，且$R_{1212} = \\det(g) \\cdot K$。对$g = \\begin{pmatrix} 1 & 0 \\ 0 & \\sin^2\\phi \\end{pmatrix}$，$\\det(g) = \\sin^2\\phi$。已知$K \\equiv 1$（单位球面），所以$R_{\\phi\\theta\\phi\\theta} = \\sin^2\\phi$。

让我用更标准的公式验证。用$R_{ijkl}$的定义：
\\[
R_{ijkl} = g_{im} R^m_{jkl} = g_{im}(\\partial_k \\Gamma^m_{lj} - \\partial_l \\Gamma^m_{kj} + \\Gamma^m_{kp}\\Gamma^p_{lj} - \\Gamma^m_{lp}\\Gamma^p_{kj}).
\\]

取$i = \\phi, j = \\theta, k = \\phi, l = \\theta$：
\\[
R^\\phi_{\\theta\\phi\\theta} = \\partial_\\phi \\Gamma^\\phi_{\\theta\\theta} - \\partial_\\theta \\Gamma^\\phi_{\\phi\\theta} + \\Gamma^\\phi_{\\phi p}\\Gamma^p_{\\theta\\theta} - \\Gamma^\\phi_{\\theta p}\\Gamma^p_{\\phi\\theta}.
\\]
只有$\\Gamma^\\phi_{\\theta\\theta} = -\\sin\\phi \\cos\\phi$和$\\Gamma^\\theta_{\\phi\\theta} = \\cot\\phi$非零。因此：
\\[
R^\\phi_{\\theta\\phi\\theta} = \\partial_\\phi(-\\sin\\phi \\cos\\phi) - 0 + 0 - 0 = -\\cos^2\\phi + \\sin^2\\phi.
\\]

Hmm, 这似乎不对。让我检查另一分量$R^\\theta_{\\phi\\theta\\phi}$：
\\[
R^\\theta_{\\phi\\theta\\phi} = \\partial_\\theta \\Gamma^\\theta_{\\phi\\phi} - \\partial_\\phi \\Gamma^\\theta_{\\theta\\phi} + \\Gamma^\\theta_{\\theta p}\\Gamma^p_{\\phi\\phi} - \\Gamma^\\theta_{\\phi p}\\Gamma^p_{\\theta\\phi}.
\\]
$\\Gamma^\\theta_{\\phi\\theta} = \\cot\\phi$，所以$\\partial_\\phi \\Gamma^\\theta_{\\theta\\phi} = -\\csc^2\\phi$。最后一项：$\\Gamma^\\theta_{\\phi\\theta}\\Gamma^\\theta_{\\theta\\phi} = \\cot\\phi \\cdot \\cot\\phi = \\cot^2\\phi$。
\\[
R^\\theta_{\\phi\\theta\\phi} = 0 - (-\\csc^2\\phi) + 0 - \\cot^2\\phi = \\csc^2\\phi - \\cot^2\\phi = 1.
\\]

现在$R_{\\theta\\phi\\theta\\phi} = g_{\\theta\\theta} R^\\theta_{\\phi\\theta\\phi} = \\sin^2\\phi \\cdot 1 = \\sin^2\\phi$。
利用对称性$R_{\\phi\\theta\\phi\\theta} = R_{\\theta\\phi\\theta\\phi} = \\sin^2\\phi$。

因此
\\[
K = \\frac{R_{\\phi\\theta\\phi\\theta}}{\\det(g)} = \\frac{\\sin^2\\phi}{\\sin^2\\phi} = 1.
\\]
这与单位球面高斯曲率恒为1的预期一致。

### 实例 2：Poincaré圆盘的曲率

Poincaré圆盘模型$D^2 = \\{(x, y) : x^2 + y^2 < 1\\}$上的度量为
\\[
g = \\frac{4}{(1 - r^2)^2} (dx^2 + dy^2), \\quad r^2 = x^2 + y^2.
\\]

由于度量是共形平坦的（conformally flat），$g_{ij} = \\lambda^2 \\delta_{ij}$，其中$\\lambda = \\frac{2}{1 - r^2}$。对共形度量$g = \\lambda^2 g_0$（$g_0$为欧氏度量），高斯曲率有显式公式：
\\[
K = -\\frac{1}{\\lambda^2} \\Delta_0 \\ln \\lambda,
\\]
其中$\\Delta_0 = \\partial_x^2 + \\partial_y^2$为欧氏拉普拉斯算子。

计算$\\ln \\lambda = \\ln 2 - \\ln(1 - r^2)$，
\\[
\\partial_x \\ln(1 - r^2) = \\frac{-2x}{1 - r^2}, \\quad \\partial_x^2 \\ln(1 - r^2) = \\frac{-2(1 - r^2) - 4x^2}{(1 - r^2)^2} = \\frac{-2 - 2x^2 + 2y^2}{(1 - r^2)^2}.
\\]
同理$\\partial_y^2 \\ln(1 - r^2) = \\frac{-2 + 2x^2 - 2y^2}{(1 - r^2)^2}$。因此
\\[
\\Delta_0 \\ln \\lambda = -\\Delta_0 \\ln(1 - r^2) = \\frac{4}{(1 - r^2)^2} = \\lambda^2.
\\]
于是$K = -\\frac{1}{\\lambda^2} \\cdot \\lambda^2 = -1$。这验证了Poincaré圆盘是常曲率$-1$的双曲空间模型。

### 实例 3：乘积度量$S^2 \\times \\mathbb{R}$的Ricci曲率

考虑$M = S^2 \\times \\mathbb{R}$，配备乘积度量$g = g_{S^2} + dt^2$。对乘积度量，联络、曲率张量均按因子分解。

设$v \\in TS^2$，$w \\in T\\mathbb{R} \\cong \\mathbb{R}$。则：
- 在$S^2$方向上，$Rc(v, v) = K \\|v\\|^2 = \\|v\\|^2$（因为$K = 1$）；

- 在$\\mathbb{R}$方向上，$Rc(\\partial_t, \\partial_t) = 0$（因为$\\mathbb{R}$平坦）；
- 混合项$Rc(v, \\partial_t) = 0$。

因此Ricci曲率矩阵为
\\[
Rc = \\begin{pmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 0 \\end{pmatrix}
\\]
（在标准正交基下）。数量曲率$S = 1 + 1 + 0 = 2$。

由于$Rc \\neq \\lambda g$（$g$的对角元全为1，而$Rc$的$\\mathbb{R}$方向分量为0），$S^2 \\times \\mathbb{R}$不是Einstein流形。这一例子说明：即使每个因子都是Einstein的，其乘积也未必是Einstein流形。
