## 📋 目录概述

## 📚 核心专题

### 黎曼几何 (Riemannian Geometry)
- **参考**: do Carmo "Riemannian Geometry", Lee "Riemannian Manifolds"
- **核心内容**: 黎曼度量、Levi-Civita联络、曲率张量、测地线
- **MSC**: 53B20, 53C20

### 辛几何 (Symplectic Geometry)
- **参考**: Cannas da Silva "Lectures on Symplectic Geometry", Arnold "Mathematical Methods of Classical Mechanics"
- **核心内容**: 辛形式、Darboux定理、Hamilton向量场
- **MSC**: 53D05, 53Dxx

### 复几何 (Complex Geometry)
- **参考**: Huybrechts "Complex Geometry", Voisin "Hodge Theory and Complex Algebraic Geometry"
- **核心内容**: 复流形、全纯向量丛、Kähler流形、Hodge分解
- **MSC**: 32Q15, 32L05, 14C30

### 指标理论 (Index Theory)
- **参考**: Berline-Getzler-Vergne "Heat Kernels and Dirac Operators"
- **核心内容**: Atiyah-Singer指标定理、Dirac算子、热核方法
- **MSC**: 58J20, 19K56, 58Jxx
