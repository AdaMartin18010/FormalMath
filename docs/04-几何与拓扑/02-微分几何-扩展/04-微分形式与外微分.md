## 14.4.1 引言 / Introduction

微分形式（differential forms）是嘉当（Élie Cartan）在20世纪初为统一和推广多元微积分中的积分理论而发明的数学工具。与传统的向量分析相比，微分形式具有坐标无关性、自然的反对称结构以及完美的外微分运算，使得格林公式、高斯公式与斯托克斯公式可以统一表述为一个极为简洁的定理：$\\int_M d\\omega = \\int_{\\partial M} \\omega$。

Princeton MAT 355课程在第七至九周深入讲授了微分形式的代数基础、外微分的运算规则、德拉姆上同调（de Rham cohomology）的初步理论，以及嘉当结构方程在现代几何中的应用。Lee（2012）《Introduction to Smooth Manifolds》第十一到第十六章提供了这一主题的系统性现代阐述。本文档在介绍基础理论的同时，特别强调了嘉当活动标架法（method of moving frames）的精神：通过选择适当的局部标架（frame）和余标架（coframe），将曲率、挠率等几何量表述为结构方程，从而避免直接处理复杂的克里斯托费尔符号（Christoffel symbols）。

微分形式的重要性不仅限于几何学。在物理学中，电磁场强$F$是2-形式，规范势$A$是1-形式；在拓扑学中，德拉姆上同调群建立了分析与代数拓扑之间的桥梁；在信息几何与机器学习领域，微分形式为研究统计流形上的信息度量提供了自然语言。

## 14.4.2 反对称张量与外代数 / Antisymmetric Tensors and Exterior Algebra

### 交错张量 / Alternating Tensors

设$V$为$n$维实向量空间，$V^*$为其对偶空间。一个$(0, k)$型张量$\\alpha \\in T^k(V^*) = (V^*)^{\\otimes k}$称为**交错的**（alternating）或**反对称的**（antisymmetric），若对任意向量$v_1, \\ldots, v_k \\in V$和任意置换$\\sigma \\in S_k$，有
\\[
\\alpha(v_{\\sigma(1)}, \\ldots, v_{\\sigma(k)}) = \\mathrm{sgn}(\\sigma) \\alpha(v_1, \\ldots, v_k).
\\]
所有$k$阶交错张量构成的子空间记为$\\Lambda^k(V^*)$。约定$\\Lambda^0(V^*) = \\mathbb{R}$，$\\Lambda^1(V^*) = V^*$。

**定义 2.1**（交错化算子）. 对任意$\\alpha \\in T^k(V^*)$，其**交错化**$\\mathrm{Alt}(\\alpha)$定义为
\\[
\\mathrm{Alt}(\\alpha)(v_1, \\ldots, v_k) = \\frac{1}{k!} \\sum_{\\sigma \\in S_k} \\mathrm{sgn}(\\sigma) \\alpha(v_{\\sigma(1)}, \\ldots, v_{\\sigma(k)}).
\\]
算子$\\mathrm{Alt}: T^k(V^*) \\to \\Lambda^k(V^*)$是到交错子空间上的投影。

**命题 2.2**（维数）. $\\dim \\Lambda^k(V^*) = \\binom{n}{k}$。特别地，当$k > n$时$\\Lambda^k(V^*) = \\{0\\}$；当$k = n$时$\\dim \\Lambda^n(V^*) = 1$。

### 楔积 / Wedge Product

**定义 2.3**（楔积）. 设$\\omega \\in \\Lambda^k(V^*)$，$\\eta \\in \\Lambda^l(V^*)$。它们的**楔积**（wedge product）$\\omega \\wedge \\eta \\in \\Lambda^{k+l}(V^*)$定义为
\\[
\\omega \\wedge \\eta = \\frac{(k+l)!}{k! \\cdot l!} \\mathrm{Alt}(\\omega \\otimes \\eta).
\\]

**定理 2.4**（楔积的基本性质）. 楔积满足：
1. **双线性性**：$(a\\omega_1 + b\\omega_2) \\wedge \\eta = a(\\omega_1 \\wedge \\eta) + b(\\omega_2 \\wedge \\eta)$；
2. **结合律**：$(\\omega \\wedge \\eta) \\wedge \\theta = \\omega \\wedge (\\eta \\wedge \\theta)$；
3. **分次交换性**：$\\omega \\wedge \\eta = (-1)^{kl} \\eta \\wedge \\omega$；
4. 若$\\omega^1, \\ldots, \\omega^n$为$V^*$的基，则$\\{\\omega^{i_1} \\wedge \\cdots \\wedge \\omega^{i_k} : 1 \\leq i_1 < \\cdots < i_k \\leq n\\}$构成$\\Lambda^k(V^*)$的基。

由性质4，任意$\\omega \\in \\Lambda^k(V^*)$可唯一写成
\\[
\\omega = \\sum_{1 \\leq i_1 < \\cdots < i_k \\leq n} \\omega_{i_1 \\cdots i_k} \\omega^{i_1} \\wedge \\cdots \\wedge \\omega^{i_k}.
\\]

**例子 2.5**. 在$\\mathbb{R}^3$中，设$dx, dy, dz$为标准余切基。则
\\[
dx \\wedge dy = -dy \\wedge dx, \\quad dx \\wedge dx = 0,
\\]
\\[
dx \\wedge dy \\wedge dz = dy \\wedge dz \\wedge dx = dz \\wedge dx \\wedge dy.
\\]
一般的2-形式可写为$\\omega = P \\ dy \\wedge dz + Q \\ dz \\wedge dx + R \\ dx \\wedge dy$，这正是经典向量分析中向量场$(P, Q, R)$的对偶表示。

## 14.4.3 微分形式与外微分 / Differential Forms and Exterior Derivative

### 微分形式丛 / Differential Form Bundles

设$M$为$n$维光滑流形。对每个$p \\in M$，考虑余切空间$\\Lambda^k(T_p^* M)$。这些空间随$p$光滑变化，构成了$M$上的**$k$次外形式丛**$\\Lambda^k(T^* M)$。一个$k$次**微分形式**（differential $k$-form）是指该丛的一个光滑截面$\\omega: M \\to \\Lambda^k(T^* M)$，即对每个$p$指定$\\omega_p \\in \\Lambda^k(T_p^* M)$，且$\\omega$光滑地依赖于$p$。所有$k$-形式的集合记为$\\Omega^k(M)$。

在局部坐标$(x^1, \\ldots, x^n)$下，$\\omega \\in \\Omega^k(M)$可写成
\\[
\\omega = \\sum_{1 \\leq i_1 < \\cdots < i_k \\leq n} \\omega_{i_1 \\cdots i_k}(x) \\ dx^{i_1} \\wedge \\cdots \\wedge dx^{i_k},
\\]
其中$\\omega_{i_1 \\cdots i_k}$为光滑函数。

### 外微分 / Exterior Derivative

**定理/定义 3.1**（外微分）. 存在唯一一族线性映射$d: \\Omega^k(M) \\to \\Omega^{k+1}(M)$，满足：
1. **线性性**：$d(a\\omega + b\\eta) = a \\ d\\omega + b \\ d\\eta$；
2. **与微分的兼容性**：对$f \\in \\Omega^0(M) = C^\\infty(M)$，$df$即通常的微分；
3. **楔积法则**（分次莱布尼茨法则）：$d(\\omega \\wedge \\eta) = d\\omega \\wedge \\eta + (-1)^k \\omega \\wedge d\\eta$，其中$\\omega \\in \\Omega^k(M)$；
4. **幂零性**：$d(d\\omega) = 0$对所有$\\omega$成立。

*存在性与唯一性证明思路*：
- **唯一性**：由条件1-3，只需确定$d$在局部坐标下对基形式$dx^{i_1} \\wedge \\cdots \\wedge dx^{i_k}$的作用。由于$d(dx^i) = d(df) = 0$（这里$f = x^i$），利用条件3归纳可得$d(dx^{i_1} \\wedge \\cdots \\wedge dx^{i_k}) = 0$。因此对$\\omega = \\sum_I \\omega_I dx^I$有
\\[
d\\omega = \\sum_I d\\omega_I \\wedge dx^I = \\sum_I \\sum_j \\frac{\\partial \\omega_I}{\\partial x^j} dx^j \\wedge dx^I.
\\]
这唯一确定了$d$。
- **存在性**：在每个坐标卡上按上述公式定义$d$，然后验证在坐标变换下不变。具体地，设$(y^1, \\ldots, y^n)$为另一组坐标，利用链式法则$\\frac{\\partial}{\\partial x^j} = \\sum_k \\frac{\\partial y^k}{\\partial x^j} \\frac{\\partial}{\\partial y^k}$和$dx^j = \\sum_k \\frac{\\partial x^j}{\\partial y^k} dy^k$，可以验证两种坐标下的计算结果一致。$\\square$

**命题 3.2**（外微分的坐标表达式）. 对$\\omega = \\sum_{|I|=k} \\omega_I dx^I$，

\\[
d\\omega = \\sum_{|I|=k} \\sum_{j=1}^n \\frac{\\partial \\omega_I}{\\partial x^j} dx^j \\wedge dx^I = \\sum_{|I|=k} d\\omega_I \\wedge dx^I.

\\]

外微分的幂零性$d^2 = 0$是微分形式理论的核心。它意味着$\\mathrm{Im}(d: \\Omega^{k-1} \\to \\Omega^k) \\subset \\mathrm{Ker}(d: \\Omega^k \\to \\Omega^{k+1})$，这为定义德拉姆上同调奠定了基础。

**定义 3.3**（闭形式与恰当形式）. 
- 若$d\\omega = 0$，称$\\omega$为**闭形式**（closed form）；
- 若存在$\\eta$使得$\\omega = d\\eta$，称$\\omega$为**恰当形式**（exact form）。

由$d^2 = 0$，任何恰当形式都是闭形式。反之是否成立依赖于流形的拓扑，这正是德拉姆上同调所量化的内容。

### 德拉姆上同调简介 / Introduction to de Rham Cohomology

**定义 3.4**（德拉姆上同调群）. $M$的**$k$阶德拉姆上同调群**定义为
\\[
H^k_{\\mathrm{dR}}(M) = \\frac{\\mathrm{Ker}(d: \\Omega^k(M) \\to \\Omega^{k+1}(M))}{\\mathrm{Im}(d: \\Omega^{k-1}(M) \\to \\Omega^k(M))}.
\\]

**定理 3.5**（德拉姆定理，de Rham's Theorem）. 对光滑流形$M$，存在自然的同构
\\[
H^k_{\\mathrm{dR}}(M) \\cong H^k(M; \\mathbb{R}),
\\]
其中右侧是$M$的实系数奇异上同调群。

这一定理表明：闭形式模恰当形式完全刻画了流形的拓扑性质。例如，$H^1_{\\mathrm{dR}}(S^1) \\cong \\mathbb{R}$，反映了$S^1$上存在非恰当的闭1-形式（如$d\\theta$）；而$H^1_{\\mathrm{dR}}(S^2) = 0$，对应于$S^2$上任何闭1-形式都是恰当的（Poincaré引理的推论）。

## 14.4.4 积分理论与斯托克斯定理 / Integration Theory and Stokes' Theorem

### 流形上的积分 / Integration on Manifolds

微分形式的自然几何意义在于它们可以被积分。$n$维流形上的$n$-形式在局部坐标下形如$f(x) dx^1 \\wedge \\cdots \\wedge dx^n$，这恰好对应于多元积分中的被积表达式。

**定义 4.1**（定向流形）. $M$上的一个**定向**（orientation）是指$\\Lambda^n(T^* M)$的一个处处非零的光滑截面，即一个处处非零的$n$-形式。配备了定向的流形称为**可定向流形**（orientable manifold）。等价地，定向也可定义为在每点$p$指定切空间$T_p M$的一个等价类基（右手系或左手系），且这些选择随$p$连续变化。

**定义 4.2**（紧支集形式的积分）. 设$M$为定向$n$维流形，$\\omega \\in \\Omega^n(M)$具有紧致支撑（compact support）。取一个与定向相容的图册$\\{(U_\\alpha, \\varphi_\\alpha)\\}$和从属于该覆盖的单位分解$\\{\\psi_\\alpha\\}$。定义
\\[
\\int_M \\omega = \\sum_\\alpha \\int_{\\varphi_\\alpha(U_\\alpha)} (\\varphi_\\alpha^{-1})^* (\\psi_\\alpha \\omega).
\\]
右侧是$\\mathbb{R}^n$中开集上的通常Lebesgue积分。

**命题 4.3**. 上述积分定义不依赖于坐标卡的选取和单位分解的选取。

*证明思路*：关键在于转移映射的Jacobi行列式。若两组坐标卡都与定向相容，则转移映射的Jacobi行列式恒正，因此变量替换公式给出相同的积分值。$\\square$

**定理 4.4**（斯托克斯定理）. 设$M$为带边界的定向$n$维光滑流形（$\\partial M$可空），$\\omega \\in \\Omega^{n-1}(M)$具有紧致支撑。则
\\[
\\int_M d\\omega = \\int_{\\partial M} \\omega,
\\]
其中$\\partial M$具有由$M$的定向诱导的**斯托克斯定向**（Stokes orientation）。

*证明思路*：利用单位分解，只需在单个坐标卡上验证。将$\\omega$拉回$\\mathbb{R}^n$中，问题约化为带边区域上的经典微积分基本定理。具体地，设$\\omega = \\sum_{i=1}^n (-1)^{i-1} f_i dx^1 \\wedge \\cdots \\wedge \\widehat{dx^i} \\wedge \\cdots \\wedge dx^n$，则
\\[
d\\omega = \\left(\\sum_{i=1}^n \\frac{\\partial f_i}{\\partial x^i}\\right) dx^1 \\wedge \\cdots \\wedge dx^n.
\\]
由Fubini定理和一元微积分基本定理，$\\int_M \\frac{\\partial f_i}{\\partial x^i} dx^1 \\cdots dx^n$恰好等于$\\omega$在边界$\\partial M$上对应项的积分。对所有$i$求和即得结论。$\\square$

斯托克斯定理的统一性是惊人的：
- 当$M = [a, b] \\subset \\mathbb{R}$时，它退化为$\\int_a^b f'(x) dx = f(b) - f(a)$；
- 当$M$是$\\mathbb{R}^2$中的区域时，它对应格林公式；
- 当$M$是$\\mathbb{R}^3$中的曲面时，它对应经典斯托克斯公式；
- 当$M$是$\\mathbb{R}^3$中的立体区域时，它对应高斯散度定理。

## 14.4.5 嘉当结构方程与活动标架法 / Cartan's Structure Equations and Moving Frames

### 活动标架 / Moving Frames

嘉当活动标架法的核心思想是：在流形的每一点（或每一点的一个邻域）选取一组“好”的标架，使得几何量的计算大为简化。

设$(M, g)$为$n$维黎曼流形。$M$上的一个**局部正交标架**（local orthonormal frame）是指定义在开集$U \\subset M$上的一组光滑向量场$\\{e_1, \\ldots, e_n\\}$，满足$g(e_i, e_j) = \\delta_{ij}$。对应的对偶余标架（coframe）记为$\\{\\theta^1, \\ldots, \\theta^n\\}$，满足$\\theta^i(e_j) = \\delta^i_j$。

在局部坐标下，度量可表示为$g = \\sum_{i,j} g_{ij} dx^i \\otimes dx^j$。寻找正交标架等价于对度量矩阵$G = (g_{ij})$进行逐点“开方”：$G = A^T A$，然后令$e_i = \\sum_j (A^{-1})_{ji} \\frac{\\partial}{\\partial x^j}$，$\\theta^i = \\sum_j A_{ij} dx^j$。

### 嘉当结构方程 / Cartan's Structure Equations

给定正交标架$\\{e_i\\}$和对偶余标架$\\{\\theta^i\\}$，Levi-Civita联络的联络形式（connection 1-forms）$\\omega^i_j$由以下第一结构方程定义：
\\[
d\\theta^i = -\\sum_j \\omega^i_j \\wedge \\theta^j.
\\]
由Levi-Civita联络的无挠性（torsion-free）和度量相容性，联络形式满足反对称条件$\\omega^i_j + \\omega^j_i = 0$（对正交标架而言）。

**定义 5.1**（曲率形式）. **曲率2-形式**$\\Omega^i_j$由第二结构方程定义：
\\[
d\\omega^i_j = -\\sum_k \\omega^i_k \\wedge \\omega^k_j + \\Omega^i_j.
\\]

**定理 5.2**（嘉当结构方程组）. 对黎曼流形$(M, g)$上的局部正交标架，有
\\[
\begin{cases}
d\\theta^i + \\sum_j \\omega^i_j \\wedge \\theta^j = 0 & \\text{(第一结构方程)} \\
d\\omega^i_j + \\sum_k \\omega^i_k \\wedge \\omega^k_j = \\Omega^i_j & \\text{(第二结构方程)}
\end{cases}
\\]
其中$\\omega^i_j + \\omega^j_i = 0$。

曲率2-形式与黎曼曲率张量的关系为
\\[
\\Omega^i_j = \\frac{1}{2} \\sum_{k,l} R^i_{jkl} \\theta^k \\wedge \\theta^l.
\\]

嘉当结构方程的优点在于：一旦选定了正交标架，所有几何量（联络、曲率）都可以通过求解外微分方程得到，而无需显式计算克里斯托费尔符号$\\Gamma^k_{ij}$。这在处理对称性较高的空间（如球面、双曲空间、李群）时尤为高效。

## 14.4.6 计算实例 / Computational Examples

### 实例 1：$\\mathbb{R}^3$中2-形式的外微分

在$\\mathbb{R}^3$中考虑2-形式$\\omega = x \\ dy \\wedge dz + y \\ dz \\wedge dx + z \\ dx \\wedge dy$。计算其外微分：
\\[
\begin{aligned}
d\\omega &= dx \\wedge dy \\wedge dz + dy \\wedge dz \\wedge dx + dz \\wedge dx \\wedge dy \\
&= 3 \\ dx \\wedge dy \\wedge dz.
\\]
\end{aligned}
\\]
（因为$dy \\wedge dz \\wedge dx = dx \\wedge dy \\wedge dz$，$dz \\wedge dx \\wedge dy = dx \\wedge dy \\wedge dz$）。

设$B_R \\subset \\mathbb{R}^3$为半径$R$的球体。由斯托克斯定理，
\\[
\\int_{B_R} d\\omega = 3 \\int_{B_R} dx \\wedge dy \\wedge dz = 3 \\cdot \\frac{4}{3} \\pi R^3 = 4\\pi R^3.
\\]
另一方面，$\\partial B_R = S^2_R$。在$S^2_R$上，$\\omega$的拉回为面积元的$R$倍（因为$x^2 + y^2 + z^2 = R^2$时，$\\omega|_{S^2}$与标准面积形式成比例）。直接计算可得$\\int_{S^2_R} \\omega = 4\\pi R^3$，验证了斯托克斯定理。

### 实例 2：$S^2$上的标准体积形式

考虑$S^2 \\subset \\mathbb{R}^3$的标准体积形式（area form）
\\[
\\omega = x \\ dy \\wedge dz + y \\ dz \\wedge dx + z \\ dx \\wedge dy,
\\]
限制在$S^2$上。为计算$\\int_{S^2} \\omega$，使用球坐标参数化：
\\[
x = \\sin\\phi \\cos\\theta, \\quad y = \\sin\\phi \\sin\\theta, \\quad z = \\cos\\phi,
\\]
其中$\\phi \\in [0, \\pi]$，$\\theta \\in [0, 2\\pi]$。计算得
\\[
dx \\wedge dy = \\sin\\phi \\cos\\phi \\cos\\theta \\ d\\phi \\wedge d\\theta,
\\]
等，最终
\\[
\\omega|_{S^2} = \\sin\\phi \\ d\\phi \\wedge d\\theta.

\\]
因此
\\[
\\int_{S^2} \\omega = \\int_0^{2\\pi} \\int_0^\\pi \\sin\\phi \\ d\\phi \\ d\\theta = 2\\pi \\cdot 2 = 4\\pi.
\\]
这与$\\chi(S^2) = 2$和高斯-博内定理的预期一致。

### 实例 3：环面$T^2$的德拉姆上同调

$T^2 = S^1 \\times S^2$可用角坐标$(\\theta, \\phi)$参数化，其中$\\theta, \\phi \\in [0, 2\\pi]$且边界等同。考虑1-形式$\\alpha = d\\theta$，$\\beta = d\\phi$。

显然$d\\alpha = d(d\\theta) = 0$，$d\\beta = 0$，故$\\alpha, \\beta$都是闭形式。它们不是恰当形式：若$\\alpha = df$，则$f(\\theta, \\phi) = \\theta + g(\\phi)$，但$\\theta$在$S^1$上不是单值函数，矛盾。因此$[\\alpha], [\\beta]$是$H^1_{\\mathrm{dR}}(T^2)$中的非零元。

进一步可以证明$H^1_{\\mathrm{dR}}(T^2) \\cong \\mathbb{R}^2$，由$[\\alpha]$和$[\\beta]$张成。更高阶的上同调为$H^0_{\\mathrm{dR}}(T^2) \\cong \\mathbb{R}$（常数函数），$H^2_{\\mathrm{dR}}(T^2) \\cong \\mathbb{R}$（由体积形式$\\alpha \\wedge \\beta = d\\theta \\wedge d\\phi$生成）。这完全匹配$T^2$的贝蒂数：$b_0 = 1, b_1 = 2, b_2 = 1$。
