## 14.5.1 引言 / Introduction

1854年，黎曼（Bernhard Riemann）在哥廷根大学发表了其著名的就职演讲《关于几何基础的假设》。在这篇演讲中，黎曼首次系统提出了$n$维流形的内禀几何概念，并引入了今天我们称之为黎曼度量（Riemannian metric）的基本结构：在每一点的切空间上指定一个光滑变化的正定双线性型。这一思想彻底颠覆了欧几里得几何的范式，为后来的广义相对论、规范场论以及现代几何分析奠定了基石。

黎曼流形$(M, g)$是微分几何研究的中心对象。度量$g$使我们能够定义长度、角度、体积、曲率以及两点之间的最短路径——测地线（geodesic）。Princeton MAT 355课程在第十至十二周集中讲授了黎曼度量的存在性、Levi-Civita联络的唯一性、测地线方程的推导、指数映射的性质以及第一变分公式。do Carmo（1992）的《Riemannian Geometry》第二章至第五章，以及Lee（2012）第十一至第十三章，是国际上最广泛采用的标准教材。本文档严格对齐这些课程与教材的核心内容。

测地线不仅是几何学中的"直线"类比，也是物理学中自由粒子运动轨迹的数学描述。在广义相对论中，测地线方程描述了物质在引力场中的运动；在机器人学中，测地线为构型空间中的最优路径规划提供了理论基础。理解黎曼度量与测地线的关系，是从局部微分几何走向整体几何分析的关键一步。

## 14.5.2 黎曼度量与黎曼流形 / Riemannian Metrics and Riemannian Manifolds

### 黎曼度量的定义 / Definition of Riemannian Metric

**定义 2.1**（黎曼度量）. 设$M$为光滑流形。$M$上的一个**黎曼度量**（Riemannian metric）$g$是指在每个点$p \\in M$的切空间$T_p M$上指定一个正定对称双线性型$g_p: T_p M \\times T_p M \\to \\mathbb{R}$，且$g_p$光滑地依赖于$p$。配备了黎曼度量的流形称为**黎曼流形**（Riemannian manifold），记为$(M, g)$。

"光滑地依赖于$p$"的精确含义是：对任意光滑向量场$X, Y \\in \\mathfrak{X}(M)$，函数$g(X, Y): M \\to \\mathbb{R}$，$p \\mapsto g_p(X_p, Y_p)$是光滑函数。

在局部坐标$(x^1, \\ldots, x^n)$下，度量$g$可表示为
\\[
g = \\sum_{i,j=1}^n g_{ij}(x) \\ dx^i \\otimes dx^j,
\\]
其中$g_{ij}(x) = g_x(\\partial_i, \\partial_j)$为光滑函数，且矩阵$(g_{ij}(x))$在每点处对称正定。度量张量的分量$g_{ij}$完全刻画了局部几何。

**定理 2.2**（黎曼度量的存在性）. 任何光滑流形$M$上都存在黎曼度量。

*证明思路*：利用单位分解定理。取$M$的一个由坐标卡组成的开覆盖$\\{U_\\alpha\\}$，在每个$U_\\alpha$上定义标准的欧氏度量$g^{(\\alpha)} = \\sum_i (dx^i)^2$。选取从属于该覆盖的单位分解$\\{\\psi_\\alpha\\}$，定义
\\[
g = \\sum_\\alpha \\psi_\\alpha g^{(\\alpha)}.
\\]
由于每个$g^{(\\alpha)}$都是正定的，且$\\psi_\\alpha \\geq 0$、$\\sum_\\alpha \\psi_\\alpha = 1$，上述加权和在每点仍是正定双线性型。光滑性由单位分解和局部度量的光滑性保证。$\\square$

这一证明展示了单位分解在整体几何构造中的威力。值得注意的是，**伪黎曼度量**（如洛伦兹度量）的存在性则没有如此简单的保证，这是广义相对论中经常需要额外条件的根源。

### 基本几何量 / Basic Geometric Quantities

给定黎曼度量$g$，可以定义以下基本几何量：

**长度（Length）**：对光滑曲线$\\gamma: [a, b] \\to M$，其长度为
\\[
L(\\gamma) = \\int_a^b \\sqrt{g_{\\gamma(t)}(\\gamma'(t), \\gamma'(t))} \\ dt = \\int_a^b \\|\\gamma'(t)\\|_g \\ dt.

\\]

**角度（Angle）**：对非零切向量$v, w \\in T_p M$，其夹角$\\theta$满足
\\[
\\cos\\theta = \\frac{g_p(v, w)}{\\|v\\|_g \\|w\\|_g}.

\\]

**体积元（Volume Element）**：在定向黎曼流形上，体积形式为
\\[
dV_g = \\sqrt{\\det(g_{ij})} \\ dx^1 \\wedge \\cdots \\wedge dx^n.
\\]
对紧致区域$U \\subset M$，其$n$维体积为$\\mathrm{Vol}(U) = \\int_U dV_g$。

### 经典例子 / Classical Examples

**例子 2.3**（欧氏空间）. $(\\mathbb{R}^n, g_{\\mathrm{Euc}})$，其中$g_{\\mathrm{Euc}} = \\sum_{i=1}^n (dx^i)^2$。这是最基本的平坦黎曼流形，其克里斯托费尔符号全为零，测地线为通常的直线。

**例子 2.4**（球面）. $S^n \\subset \\mathbb{R}^{n+1}$的**诱导度量**（induced metric）由$\\mathbb{R}^{n+1}$的欧氏度量限制到$S^n$的切空间上得到。在球坐标下，$S^2$的度量为
\\[
g_{S^2} = d\\phi^2 + \\sin^2\\phi \\ d\\theta^2.
\\]

**例子 2.5**（双曲空间）. $n$维双曲空间$\\mathbb{H}^n$有多种等价的模型，其中Poincaré圆盘模型上的度量为
\\[
g_{\\mathbb{H}^n} = \\frac{4}{(1 - |x|^2)^2} \\sum_{i=1}^n (dx^i)^2, \\quad |x| < 1.

\\]
上半空间模型上的度量为
\\[
g_{\\mathbb{H}^n} = \\frac{1}{x_n^2} \\sum_{i=1}^n (dx^i)^2, \\quad x_n > 0.
\\]
双曲空间是截面曲率恒为$-1$的完备单连通空间形式。

**例子 2.6**（乘积度量）. 若$(M, g_M)$和$(N, g_N)$为黎曼流形，则$M \\times N$上的**乘积度量**为$g = g_M \\oplus g_N$。在局部坐标下，若$g_M = \\sum g_{ij} dx^i dx^j$，$g_N = \\sum h_{\\alpha\\beta} dy^\\alpha dy^\\beta$，则
\\[
g = \\sum g_{ij} dx^i dx^j + \\sum h_{\\alpha\\beta} dy^\\alpha dy^\\beta.
\\]
例如，平坦环面$T^2 = S^1 \\times S^1$配备乘积度量后为平坦黎曼流形。

## 14.5.3 Levi-Civita联络与协变导数 / Levi-Civita Connection and Covariant Derivative

### 仿射联络 / Affine Connections

为了能够在流形上"微分"向量场，需要引入**联络**（connection）的概念。

**定义 3.1**（仿射联络）. $M$上的一个**仿射联络**（affine connection）$\\nabla$是指映射$\\nabla: \\mathfrak{X}(M) \\times \\mathfrak{X}(M) \\to \\mathfrak{X}(M)$，记为$(X, Y) \\mapsto \\nabla_X Y$，满足：
1. $\\nabla_{fX + gZ} Y = f \\nabla_X Y + g \\nabla_Z Y$（对第一个变量$C^\\infty(M)$-线性）；
2. $\\nabla_X (aY + bZ) = a \\nabla_X Y + b \\nabla_X Z$（对第二个变量$\\mathbb{R}$-线性）；
3. $\\nabla_X (fY) = X(f) Y + f \\nabla_X Y$（Leibniz法则）。

$\\nabla_X Y$称为向量场$Y$沿$X$方向的**协变导数**（covariant derivative）。

在局部坐标下，联络由**克里斯托费尔符号**（Christoffel symbols）$\\Gamma^k_{ij}$完全刻画：
\\[
\\nabla_{\\partial_i} \\partial_j = \\sum_{k=1}^n \\Gamma^k_{ij} \\partial_k.
\\]
对一般的向量场$X = \\sum X^i \\partial_i$，$Y = \\sum Y^j \\partial_j$，由定义中的三条公理可得
\\[
\\nabla_X Y = \\sum_{k=1}^n \\left( X(Y^k) + \\sum_{i,j} X^i Y^j \\Gamma^k_{ij} \\right) \\partial_k.
\\]

### Levi-Civita定理 / The Fundamental Theorem of Riemannian Geometry

**定理 3.2**（Levi-Civita, 1917）. 对任意黎曼流形$(M, g)$，存在唯一的仿射联络$\\nabla$满足：
1. **无挠性**（torsion-free）：$\\nabla_X Y - \\nabla_Y X = [X, Y]$；
2. **度量相容性**（metric compatibility）：$X(g(Y, Z)) = g(\\nabla_X Y, Z) + g(Y, \\nabla_X Z)$。

这一联络称为$(M, g)$的**Levi-Civita联络**。

*证明思路*：唯一性通过对称化论证得到。由度量相容性，
\\[
\begin{aligned}
X(g(Y, Z)) &= g(\\nabla_X Y, Z) + g(Y, \\nabla_X Z), \\
Y(g(Z, X)) &= g(\\nabla_Y Z, X) + g(Z, \\nabla_Y X), \\
Z(g(X, Y)) &= g(\\nabla_Z X, Y) + g(X, \\nabla_Z Y).
\end{aligned}
\\]
将第一式加第二式减第三式，并利用无挠性$\\nabla_X Y - \\nabla_Y X = [X, Y]$等，可得
\\[
2g(\\nabla_X Y, Z) = X(g(Y,Z)) + Y(g(Z,X)) - Z(g(X,Y)) + g([X,Y], Z) - g([Y,Z], X) + g([Z,X], Y).
\\]
由于$g$正定，右侧唯一确定了$\\nabla_X Y$。存在性通过验证按此公式定义的$\\nabla$确实满足联络公理、无挠性和度量相容性得到。$\\square$

**推论 3.3**（克里斯托费尔符号的显式公式）. 在局部坐标下，Levi-Civita联络的克里斯托费尔符号为
\\[
\\Gamma^k_{ij} = \\frac{1}{2} \\sum_l g^{kl} \\left( \\frac{\\partial g_{jl}}{\\partial x^i} + \\frac{\\partial g_{il}}{\\partial x^j} - \\frac{\\partial g_{ij}}{\\partial x^l} \\right),
\\]
其中$(g^{kl})$是度量矩阵$(g_{ij})$的逆矩阵。

这一公式是黎曼几何计算的核心。虽然符号复杂，但其结构高度对称，只依赖于度量的一阶导数。

## 14.5.4 测地线与指数映射 / Geodesics and Exponential Map

### 测地线方程 / Geodesic Equation

**定义 4.1**（测地线）. 光滑曲线$\\gamma: I \\to M$称为**测地线**（geodesic），若其切向量场$\\gamma'(t)$沿$\\gamma$的协变导数为零：
\\[
\\nabla_{\\gamma'(t)} \\gamma'(t) = 0.
\\]

在局部坐标下，设$\\gamma(t) = (x^1(t), \\ldots, x^n(t))$，则$\\gamma'(t) = \\sum_i \\dot{x}^i(t) \\partial_i$，测地线方程化为二阶常微分方程组：
\\[
\\ddot{x}^k + \\sum_{i,j} \\Gamma^k_{ij} \\dot{x}^i \\dot{x}^j = 0, \\quad k = 1, \\ldots, n.
\\]

**定理 4.2**（测地线的存在唯一性）. 对任意$p \\in M$和$v \\in T_p M$，存在$\\epsilon > 0$和唯一的测地线$\\gamma_v: (-\\epsilon, \\epsilon) \\to M$满足$\\gamma_v(0) = p$和$\\gamma_v'(0) = v$。此外，测地线关于初始条件光滑依赖。

*证明思路*：测地线方程是二阶ODE的初值问题。由ODE标准理论（Picard-Lindelöf定理或更一般的光滑依赖性定理），对给定初值$(p, v)$，存在唯一的局部解，且解关于初值光滑依赖。$\\square$

**命题 4.3**（常速度性）. 若$\\gamma$是测地线，则$\\|\\gamma'(t)\\|_g$为常数。

*证明*：$\\frac{d}{dt} g(\\gamma', \\gamma') = 2g(\\nabla_{\\gamma'} \\gamma', \\gamma') = 0$。$\\square$

### 指数映射 / Exponential Map

**定义 4.4**（指数映射）. 设$p \\in M$。对充分小的$v \\in T_p M$，定义**指数映射**
\\[
\\exp_p(v) = \\gamma_v(1),
\\]
其中$\\gamma_v$是以$p$为起点、初速度为$v$的测地线。$\\exp_p$在$0 \\in T_p M$的某个邻域上是良好定义的光滑映射。

**定理 4.5**（高斯引理）. 指数映射$\\exp_p: T_p M \\to M$在$0$处的微分是恒等映射：$d(\\exp_p)_0 = \\mathrm{id}_{T_p M}$。因此，$\\exp_p$在$0$附近是局部微分同胚。

*证明思路*：对$v \\in T_p M$，考虑径向曲线$\\sigma(t) = tv$，其在$\\exp_p$下的像为$\\exp_p(tv) = \\gamma_{tv}(1) = \\gamma_v(t)$。因此
\\[
d(\\exp_p)_0(v) = \\left.\\frac{d}{dt}\\right|_{t=0} \\gamma_v(t) = v. \\square

\\]

高斯引理保证了我们可以利用指数映射将切空间$T_p M$上的线性结构"移植"到流形$M$上$p$的附近。这是**法坐标**（normal coordinates）和**测地极坐标**（geodesic polar coordinates）的基础。

**定义 4.6**（法坐标）. 取$T_p M$的一组标准正交基$\\{e_1, \\ldots, e_n\\}$，通过$\\exp_p$将$T_p M$的坐标转移到$M$上$p$的邻域中，这样得到的局部坐标称为**黎曼法坐标**（Riemannian normal coordinates）。在法坐标下，$g_{ij}(0) = \\delta_{ij}$且$\\Gamma^k_{ij}(0) = 0$。

## 14.5.5 变分公式与距离函数 / Variational Formulas and Distance Functions

### 第一变分公式 / First Variation Formula

测地线的基本几何性质是它们局部最小化长度。这一性质通过第一变分公式精确刻画。

设$\\gamma: [a, b] \\to M$为光滑曲线，$\\Gamma: (-\\epsilon, \\epsilon) \\times [a, b] \\to M$为$\\gamma$的一个**变分**（variation），即$\\Gamma(0, t) = \\gamma(t)$。记变分向量场为$V(t) = \\partial_s \\Gamma(0, t)$。

**定理 5.1**（第一变分公式）. 设$L(s)$为曲线$\\gamma_s(t) = \\Gamma(s, t)$的长度。若$\\gamma$以常速度参数化（即$\\|\\gamma'(t)\\| = c$为常数），则

\\[
\\left.\\frac{dL}{ds}\\right|_{s=0} = -\\frac{1}{c} \\int_a^b g(\\nabla_{\\gamma'(t)} \\gamma'(t), V(t)) \\ dt + \\frac{1}{c} \\left[ g(\\gamma'(t), V(t)) \\right]_a^b.

\\]

*证明思路*：直接计算$\\frac{d}{ds} \\int_a^b \\|\\partial_t \\Gamma\\| \\ dt$，利用度量相容性、无挠性和分部积分。详细推导见do Carmo（1992）命题9.2。$\\square$

**推论 5.2**. 对固定端点的变分（$V(a) = V(b) = 0$），$\\gamma$是长度泛函的临界点当且仅当$\\nabla_{\\gamma'} \\gamma' = 0$，即$\\gamma$是测地线。

这一定理确立了测地线作为"局部最短路径"的几何地位。需要注意的是，"测地线"是一个局部概念：即使$\\gamma$是测地线，如果它延伸到足够远，也可能不再是最短路径。例如，球面上的大圆是测地线，但超过半圆后就不是连接两端点的最短路径了。

### 黎曼距离函数 / Riemannian Distance Function

**定义 5.3**（黎曼距离）. 对连通黎曼流形$(M, g)$，两点$p, q \\in M$之间的**黎曼距离**定义为
\\[
d_g(p, q) = \\inf \\{L(\\gamma) : \\gamma \\text{ 为连接 } p, q \\text{ 的分段光滑曲线}\\}.
\\]

**定理 5.4**（Hopf-Rinow定理，核心形式）. 设$(M, g)$为连通黎曼流形。则以下等价：
1. $(M, d_g)$作为度量空间是完备的（即所有柯西列收敛）；
2. 对某点$p \\in M$，$\\exp_p$在整个$T_p M$上有定义；
3. 对所有$p \\in M$，$\\exp_p$在整个$T_p M$上有定义。

若上述条件满足，则任意两点$p, q \\in M$可由一条极小测地线连接。

Hopf-Rinow定理是整体黎曼几何的基石。所有紧致黎曼流形都是完备的，因为紧致度量空间必然完备。欧氏空间$\\mathbb{R}^n$、球面$S^n$和双曲空间$\\mathbb{H}^n$也都是完备的。

## 14.5.6 计算实例 / Computational Examples

### 实例 1：$S^2$的克里斯托费尔符号与测地线

考虑$S^2$的诱导度量$g = d\\phi^2 + \\sin^2\\phi \\ d\\theta^2$。度量矩阵为
\\[
(g_{ij}) = \\begin{pmatrix} 1 & 0 \\ 0 & \\sin^2\\phi \\end{pmatrix}, \\quad (g^{ij}) = \\begin{pmatrix} 1 & 0 \\ 0 & \\sin^{-2}\\phi \\end{pmatrix}.
\\]

计算克里斯托费尔符号：
\\[
\\Gamma^\\phi_{\\theta\\theta} = -\\sin\\phi \\cos\\phi, \\quad \\Gamma^\\theta_{\\phi\\theta} = \\Gamma^\\theta_{\\theta\\phi} = \\cot\\phi,
\\]
其余$\\Gamma^k_{ij} = 0$。

测地线方程为：
\\[
\begin{cases}
\\ddot{\\phi} - \\sin\\phi \\cos\\phi \\ \\dot{\\theta}^2 = 0, \\
\\ddot{\\theta} + 2\\cot\\phi \\ \\dot{\\phi} \\dot{\\theta} = 0.
\end{cases}
\\]

若$\\dot{\\theta} = 0$，则$\\ddot{\\phi} = 0$，解为$\\phi(t) = at + b$，即经线（meridian）。可以验证所有大圆（great circles）都是测地线，这符合球面几何的直观。

### 实例 2：Poincaré上半平面的测地线

在$\\mathbb{H}^2 = \\{(x, y) : y > 0\\}$上，度量为$g = \\frac{dx^2 + dy^2}{y^2}$。度量矩阵$g_{ij} = \\frac{1}{y^2} \\delta_{ij}$。

计算克里斯托费尔符号：
\\[
\\Gamma^x_{xy} = \\Gamma^x_{yx} = -\\frac{1}{y}, \\quad \\Gamma^y_{xx} = \\frac{1}{y}, \\quad \\Gamma^y_{yy} = -\\frac{1}{y},
\\]
其余为零。

测地线方程：
\\[
\begin{cases}
\\ddot{x} - \\frac{2}{y} \\dot{x} \\dot{y} = 0, \\
\\ddot{y} + \\frac{1}{y}(\\dot{x}^2 - \\dot{y}^2) = 0.
\end{cases}
\\]

可以验证，垂直于$x$轴的直线$x = \\text{const}$和以$x$轴上点为圆心的半圆都是测地线。这是双曲几何的核心结果：在Poincaré模型中，"直线"是垂直于边界的欧氏直线或半圆。

### 实例 3：$T^2$上的平坦度量

平坦环面$T^2 = \\mathbb{R}^2 / \\mathbb{Z}^2$上的度量由$\\mathbb{R}^2$的欧氏度量诱导。在坐标$(\\theta, \\phi)$下，$g = d\\theta^2 + d\\phi^2$。所有克里斯托费尔符号为零，测地线方程为$\\ddot{\\theta} = 0$，$\\ddot{\\phi} = 0$，解为
\\[
\\theta(t) = at + \\theta_0, \\quad \\phi(t) = bt + \\phi_0.
\\]
在$T^2$上，这类曲线是斜率为$b/a$的直线（模去整数格）。当$b/a$为有理数时，测地线是闭合曲线；当$b/a$为无理数时，测地线在$T^2$上稠密。这与$\\mathbb{R}^2$上所有测地线都是直线形成鲜明对比，体现了商空间拓扑对测地线整体行为的影响。
