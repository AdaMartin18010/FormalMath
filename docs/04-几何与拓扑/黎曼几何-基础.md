# 黎曼几何基础

## Riemann度量

### 定义

Riemann度量$g$是光滑流形$M$上每点$p$处的内积$g_p: T_p M \times T_p M \to \mathbb{R}$，光滑依赖于$p$。

局部坐标：$g = g_{ij} dx^i \otimes dx^j$

### Levi-Civita联络

唯一满足：

1. 无挠：$\nabla_X Y - \nabla_Y X = [X, Y]$
2. 度量相容：$X(g(Y, Z)) = g(\nabla_X Y, Z) + g(Y, \nabla_X Z)$

Christoffel符号：$\Gamma^k_{ij} = \frac{1}{2}g^{kl}(\partial_i g_{jl} + \partial_j g_{il} - \partial_l g_{ij})$

### 曲率

Riemann曲率张量：$R^l_{ijk} = \partial_j \Gamma^l_{ik} - \partial_k \Gamma^l_{ij} + \Gamma^l_{jm}\Gamma^m_{ik} - \Gamma^l_{km}\Gamma^m_{ij}$

Ricci曲率：$R_{ij} = R^k_{ikj}$

标量曲率：$R = g^{ij}R_{ij}$
