# 黎曼几何基础

Riemann 几何是研究配备 Riemann 度量的光滑流形的数学分支，由 Bernhard Riemann 于 1854 年就任 Göttingen 大学讲师时的著名就职演说《论几何基础中的假设》中创立。其核心是内蕴地测量长度、角度、体积与曲率。

## 1. Riemann 度量

### 1.1 定义与例子

**定义 1.1（Riemann 度量）**. 光滑流形 $M$ 上的 Riemann 度量 $g$ 是对每点 $p \in M$ 指定切空间上的正定对称双线性型：

$$g_p: T_p M \times T_p M \to \mathbb{R},$$

且光滑依赖于 $p$（即对任意光滑向量场 $X, Y$，$g(X, Y) \in C^\infty(M)$）。

在局部坐标 $(x^1, \dots, x^n)$ 下：

$$g = g_{ij} dx^i \otimes dx^j, \quad g_{ij} = g\left(\frac{\partial}{\partial x^i}, \frac{\partial}{\partial x^j}\right).$$

**例子 1.2**.
- **Euclidean 空间**：$\mathbb{R}^n$ 上的标准度量 $g = \sum_{i=1}^n (dx^i)^2$；
- **球面**：$S^n \subset \mathbb{R}^{n+1}$ 上的诱导度量；
- **双曲空间**：Poincaré 圆盘上的 $g = \frac{4}{(1-|x|^2)^2} \sum (dx^i)^2$；
- **Lorentz 度量**：伪 Riemann 度量（如广义相对论中的时空度量）。

### 1.2 度量诱导的结构

**定义 1.3（长度与角度）**. 曲线 $\gamma: [a,b] \to M$ 的长度：

$$L(\gamma) = \int_a^b \sqrt{g(\dot{\gamma}(t), \dot{\gamma}(t))} dt.$$

两切向量 $v, w \in T_p M$ 的夹角：

$$\cos \theta = \frac{g(v,w)}{|v||w|}.$$

**定义 1.4（体积形式）**. 定向 Riemann 流形上的体积形式：

$$dV_g = \sqrt{\det(g_{ij})} dx^1 \wedge \cdots \wedge dx^n.$$

## 2. Levi-Civita 联络

### 2.1 仿射联络

**定义 2.1（仿射联络）**. 仿射联络 $\nabla$ 是映射 $\mathfrak{X}(M) \times \mathfrak{X}(M) \to \mathfrak{X}(M)$，记 $\nabla_X Y$，满足：
1. $\nabla_{fX + gZ} Y = f\nabla_X Y + g\nabla_Z Y$；
2. $\nabla_X (Y + Z) = \nabla_X Y + \nabla_X Z$；
3. $\nabla_X (fY) = X(f)Y + f\nabla_X Y$（Leibniz 法则）。

**定理 2.2（Levi-Civita）**. 对任意 Riemann 流形 $(M, g)$，存在唯一的无挠（$\nabla_X Y - \nabla_Y X = [X, Y]$）且与度量相容（$X g(Y, Z) = g(\nabla_X Y, Z) + g(Y, \nabla_X Z)$）的联络，称为 Levi-Civita 联络。

*证明*. 由度量相容性和无挠性，轮换指标得：

$$X g(Y, Z) = g(\nabla_X Y, Z) + g(Y, \nabla_X Z),$$

$$Y g(Z, X) = g(\nabla_Y Z, X) + g(Z, \nabla_Y X),$$

$$Z g(X, Y) = g(\nabla_Z X, Y) + g(X, \nabla_Z Y).$$

前两式相加减第三式，利用无挠性 $\nabla_X Y - \nabla_Y X = [X,Y]$：

$$2g(\nabla_X Y, Z) = X g(Y,Z) + Y g(Z,X) - Z g(X,Y)$$

$$- g([X,Y], Z) - g([Y,Z], X) + g([Z,X], Y).$$

右边唯一确定 $\nabla_X Y$（由 $g$ 的正定性），且可验证满足联络公理。$\square$

### 2.2 Christoffel 符号

在坐标下，$\nabla_{\partial_i} \partial_j = \Gamma^k_{ij} \partial_k$，其中：

$$\Gamma^k_{ij} = \frac{1}{2} g^{kl} \left( \partial_i g_{jl} + \partial_j g_{il} - \partial_l g_{ij} \right).$$

## 3. 曲率张量

### 3.1 Riemann 曲率张量

**定义 3.1（Riemann 曲率）**. Levi-Civita 联络的曲率张量 $R$ 定义为：

$$R(X, Y)Z = \nabla_X \nabla_Y Z - \nabla_Y \nabla_X Z - \nabla_{[X,Y]} Z.$$

在坐标下：

$$R^l_{ijk} = \partial_j \Gamma^l_{ik} - \partial_k \Gamma^l_{ij} + \Gamma^l_{jm}\Gamma^m_{ik} - \Gamma^l_{km}\Gamma^m_{ij}.$$

**命题 3.2（对称性）**.
1. $R(X, Y)Z = -R(Y, X)Z$；
2. $g(R(X,Y)Z, W) = -g(R(X,Y)W, Z)$；
3. 第一 Bianchi 恒等式：$R(X,Y)Z + R(Y,Z)X + R(Z,X)Y = 0$；
4. $g(R(X,Y)Z, W) = g(R(Z,W)X, Y)$。

### 3.2 Ricci 与标量曲率

**定义 3.3（Ricci 曲率）**. $R_{ij} = R^k_{ikj}$。

**定义 3.4（标量曲率）**. $R = g^{ij}R_{ij}$。

**定义 3.5（截面曲率）**. 对 $2$-平面 $\Pi = \operatorname{span}\{v, w\} \subset T_p M$：

$$K(\Pi) = \frac{g(R(v,w)w, v)}{g(v,v)g(w,w) - g(v,w)^2}.$$

## 4. 测地线

### 4.1 定义与方程

**定义 4.1（测地线）**. 曲线 $\gamma$ 称为测地线，若 $\nabla_{\dot{\gamma}} \dot{\gamma} = 0$。

在坐标下，测地线方程为：

$$\ddot{x}^k + \Gamma^k_{ij} \dot{x}^i \dot{x}^j = 0.$$

**定理 4.2**. 测地线是局部长度极小的曲线。

*证明概要*. 考虑固定端点的变分 $\gamma_s$，第一变分公式给出：

$$\left.\frac{d}{ds}\right|_{s=0} L(\gamma_s) = -\int_a^b g\left(\frac{D}{dt}\dot{\gamma}, V\right) dt,$$

其中 $V = \partial_s \gamma|_{s=0}$。对任意变分向量场 $V$，极值要求 $\frac{D}{dt}\dot{\gamma} = 0$。$\square$

## 5. 例子

### 5.1 例子：常曲率空间

**定理 5.1（空间形式）**. 单连通、完备、常截面曲率 $K$ 的 $n$ 维 Riemann 流形等距于：
- $K > 0$：半径 $1/\sqrt{K}$ 的球面 $S^n$；
- $K = 0$：Euclidean 空间 $\mathbb{R}^n$；
- $K < 0$：半径 $1/\sqrt{-K}$ 的双曲空间 $H^n$。

### 5.2 例子：二维曲面的 Gauss 绝妙定理

**定理 5.2（Gauss Theorema Egregium）**. 曲面的 Gauss 曲率 $K$ 由第一基本形式（度量）完全确定，与第二基本形式（嵌入方式）无关。

*证明概要*. 对参数化曲面 $X(u,v)$，$K = \frac{LN - M^2}{EG - F^2}$。直接计算可验证 $K$ 只依赖于 $E, F, G$ 及其导数。$\square$
