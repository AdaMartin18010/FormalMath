## 5.11.1 引言 / Introduction

计算拓扑学（computational topology）是21世纪初兴起的一门交叉学科，它将代数拓扑的抽象理论与计算机科学的算法设计相结合，为高维数据分析、形状识别、材料科学、神经科学和机器学习等领域提供了全新的数学工具。与传统的纯数学拓扑学不同，计算拓扑学关注的是：如何在有限数据、有限计算资源的情况下，有效地计算拓扑不变量，并将这些不变量用于实际问题的建模与求解。

持续同调（persistent homology）是计算拓扑学中最核心的概念。它通过在多尺度下追踪拓扑特征（如连通分支、环、空洞）的"寿命"，将离散的采样数据转化为稳定的拓扑签名（topological signature）——即**条形码**（barcode）或**持续图**（persistence diagram）。这种签名对数据的微小扰动具有鲁棒性，因此在机器学习中作为特征提取工具具有独特优势。

本文档对齐Edelsbrunner与Harer（2010）《Computational Topology: An Introduction》、Carlsson（2009）关于拓扑数据分析（TDA）的综述、Ghrist（2008）的讲义，以及UW Math AI Lab 2026在计算拓扑与人工智能交叉领域的最新研究进展。核心内容包括：持续同调的代数与算法基础、Mapper算法、离散Morse理论，以及计算拓扑在材料科学、神经科学和AI中的具体应用。

## 5.11.2 持续同调 / Persistent Homology

### 过滤与持续模 / Filtrations and Persistence Modules

**定义 2.1**（单纯复形过滤）. 一个**单纯复形过滤**（simplicial complex filtration）是指一族单纯复形$\\{K_\\epsilon\\}_{\\epsilon \\in \\mathbb{R}}$，满足$\\epsilon_1 \\leq \\epsilon_2$时$K_{\\epsilon_1} \\subset K_{\\epsilon_2}$。

最常见的构造方式是基于点云的**Vietoris-Rips复形**：给定度量空间$(X, d)$中的有限点集$P$和尺度参数$\\epsilon \\geq 0$，$VR_\\epsilon(P)$是以$P$为顶点集的单纯复形，其中一组顶点$\\{p_0, \\ldots, p_k\\}$张成一个$k$-单纯形当且仅当它们两两之间的距离$d(p_i, p_j) \\leq \\epsilon$对所有$i, j$成立。

另一种常用构造是**Cech复形**：$\\{p_0, \\ldots, p_k\\}$构成一个单纯形当且仅当这些点的$\\epsilon/2$-半径球有非空交集。

**定义 2.2**（持续同调）. 对过滤$\\{K_\\epsilon\\}$，包含映射$i_{\\epsilon_1, \\epsilon_2}: K_{\\epsilon_1} \\hookrightarrow K_{\\epsilon_2}$诱导同调群的同态
\\[
(i_{\\epsilon_1, \\epsilon_2})_*: H_n(K_{\\epsilon_1}) \\to H_n(K_{\\epsilon_2}).
\\]
**$n$维持续同调群**定义为这些同态的像：
\\[
H_n^{\\epsilon_1, \\epsilon_2} = \\mathrm{im}((i_{\\epsilon_1, \\epsilon_2})_*: H_n(K_{\\epsilon_1}) \\to H_n(K_{\\epsilon_2})).
\\]

**定义 2.3**（持续模）. 一族向量空间$\\{V_\\epsilon\\}_{\\epsilon \\in \\mathbb{R}}$和线性映射$\\phi_{\\epsilon_1, \\epsilon_2}: V_{\\epsilon_1} \\to V_{\\epsilon_2}$（$\\epsilon_1 \\leq \\epsilon_2$）满足$\\phi_{\\epsilon, \\epsilon} = id$和$\\phi_{\\epsilon_2, \\epsilon_3} \\circ \\phi_{\\epsilon_1, \\epsilon_2} = \\phi_{\\epsilon_1, \\epsilon_3}$，称为**持续模**（persistence module）。

### 结构定理与条形码 / Structure Theorem and Barcodes

**定理 2.4**（持续模的结构定理）. 设$\\{V_\\epsilon\\}$为有限型持续模（即每个$V_\\epsilon$有限维，且只有有限多个临界参数使结构发生变化）。则$\\{V_\\epsilon\\}$可唯一分解为区间不可约模的直和：
\\[
V_\\epsilon \\cong \\bigoplus_{i} I_{[b_i, d_i)}(\\epsilon),
\\]
其中$I_{[b, d)}$是在$[b, d)$上为$\\mathbb{F}$（某个域）、其余处为零的区间模，转移映射为恒等映射。

这些区间$[b_i, d_i)$构成了持续同调的**条形码**（barcode）表示。区间的左端点$b_i$称为**诞生时间**（birth），右端点$d_i$称为**死亡时间**（death），$d_i - b_i$称为该拓扑特征的**持续长度**（persistence）。

长持续长度的特征（长条）通常对应于数据的真实拓扑信号，而短持续长度的特征（短条）通常被视为噪声。这一"信噪分离"能力是持续同调在数据分析中极具价值的原因。

### 算法复杂度 / Algorithmic Complexity

**定理 2.5**（矩阵简化算法）. 对具有$N$个单纯形的过滤复形，计算所有维数的持续同调可在$O(N^3)$时间内完成。对于 Vietoris-Rips 复形，通过优化可降至接近$O(N^2)$的期望时间。

*算法思路*：将边界矩阵$\\partial$通过列操作化为简化形式（reduced form）。简化后的矩阵中非零低对角元恰好对应于条形码中的区间端点。详见Edelsbrunner-Harer（2010）第七章。$\\square$

## 5.11.3 Mapper算法与拓扑机器学习 / Mapper Algorithm and Topological Machine Learning

### Mapper算法 / The Mapper Algorithm

Mapper算法由Singh, Mémoli和Carlsson在2007年提出，是拓扑数据分析中用于高维数据可视化和聚类的核心工具。

**Mapper算法步骤**：
1. **过滤函数**：选择一个（或多个）过滤函数$f: X \\to \\mathbb{R}^d$（或$\\mathbb{R}$），将高维数据投影到低维空间；
2. **覆盖**：在像空间$\\mathbb{R}^d$上定义一个开覆盖$\\{U_\\alpha\\}$；
3. **聚类**：对每个原像$f^{-1}(U_\\alpha)$中的数据点进行聚类；
4. **构建图/复形**：若两个聚类有非空交集，则在图中连接对应的节点。

Mapper的输出是一个**Reeb图的离散近似**，可以用图的结构来理解数据的整体形状。

**定理 3.1**（Mapper收敛性）. 在适当条件下，当覆盖足够精细且数据点足够密集时，Mapper输出收敛于真实数据的Reeb图。

### 拓扑特征在机器学习中的应用 / Topological Features in ML

持续同调和Mapper的输出可以作为机器学习模型的输入特征：
- **持续图核**（persistence diagram kernels）：定义持续图之间的距离和核函数，用于支持向量机（SVM）和核方法；
- **向量化方法**：将条形码映射到有限维向量空间（如持续景观 persistence landscapes、持续图像 persistence images），直接输入神经网络；
- **拓扑正则化**（topological regularization）：在损失函数中加入拓扑惩罚项，强制神经网络学习具有特定拓扑结构的表示。

UW Math AI Lab 2026的最新研究表明，将拓扑特征与深度学习结合可以显著提高模型在分子性质预测、蛋白质结构分类和材料相变检测等任务上的性能。

## 5.11.4 离散Morse理论 / Discrete Morse Theory

### 离散梯度向量场 / Discrete Gradient Vector Field

离散Morse理论由Forman在1998年提出，它将经典Morse理论中的光滑梯度流推广到单纯复形和CW复形上。

**定义 4.1**（离散Morse函数）. 设$K$为有限单纯复形。函数$f: K \\to \\mathbb{R}$称为**离散Morse函数**，若对任意$p$-单纯形$\\sigma \\in K$，以下两个集合中至多有一个非空：
- $\\{\\tau^{(p+1)} > \\sigma : f(\\tau) \\leq f(\\sigma)\\}$；
- $\\{\\nu^{(p-1)} < \\sigma : f(\\nu) \\geq f(\\sigma)\\}$。

若两个集合都为空，称$\\sigma$为**临界单纯形**（critical simplex）。

**定义 4.2**（离散梯度向量场）. $K$上的**离散梯度向量场**（discrete gradient vector field）$V$是$K$中单纯形配对$\\{\\alpha^{(p)} < \\beta^{(p+1)}\\}$的集合，满足每个单纯形至多出现在一个配对中。

离散Morse函数自然诱导离散梯度向量场：将满足$f(\\beta) \\leq f(\\alpha)$的相邻单纯形配对。反之，任何离散梯度向量场都可以由一个离散Morse函数实现。

### 离散Morse不等式 / Discrete Morse Inequalities

**定理 4.3**（离散Morse不等式）. 设$K$为有限单纯复形，$m_p$为指标为$p$的临界单纯形的个数。则
\\[
c_p - c_{p-1} + \\cdots \\pm c_0 \\geq b_p - b_{p-1} + \\cdots \\pm b_0,
\\]
其中$c_p$为$K$的$p$-胞腔数，$b_p$为第$p$贝蒂数。当取等号时（即$K$具有完美Morse函数），$m_p = b_p$对所有$p$成立。

离散Morse理论在计算中的核心价值在于：**Morse复形**（critical cells的链复形）与原始复形$K$具有相同的同调，但规模可能小得多。这为高维数据的同调计算提供了强大的预处理工具。

## 5.11.5 应用实例 / Application Examples

### 实例 1：材料科学中的相变检测

在晶体学和材料科学中，材料的微观结构可以通过点云或图像数据表示。持续同调被用来追踪材料在加热、加压或形变过程中拓扑特征（如孔洞、环、通道）的演化。

例如，在研究多孔材料（如沸石）时，$H_1$中的长条对应于材料中的通道结构，这些通道决定了分子的传输性质。通过比较不同材料的持续条形码，研究人员可以预测其催化活性和吸附性能。

### 实例 2：神经科学中的脑网络拓扑

大脑可以被建模为一个复杂的网络（connectome），其中节点是神经元或脑区，边是神经连接。Mapper算法和持续同调被用来分析脑网络的多尺度拓扑结构。

研究发现，健康大脑的功能网络具有丰富的环结构（反映在$H_1$中），而某些神经退行性疾病（如阿尔茨海默病）会导致这些环结构的显著减少。持续同调为量化这种拓扑变化提供了客观指标。

### 实例 3：分子结构与蛋白质折叠

蛋白质的3D结构决定了其生物功能。Mapper算法可以将蛋白质构象空间的高维点云转化为低维图结构，揭示折叠路径中的关键中间态和能垒。

持续同调也被用于分子描述符的构建：分子的$H_0$条形码反映原子团的聚类结构，$H_1$反映环状结构（如芳香环），$H_2$反映笼状结构。这些拓扑描述符在定量构效关系（QSAR）建模中表现出了优于传统化学描述符的预测能力。

### 实例 4：UW Math AI Lab 2026 研究进展

UW Math AI Lab 2026在计算拓扑与深度学习的交叉领域取得了多项进展：
- **拓扑注意力网络**（Topological Attention Networks）：将单纯复形的邻域结构融入图注意力机制，用于分子图表示学习；
- **持续同调生成模型**（Persistent Homology Generative Models）：训练生成对抗网络（GAN）生成具有特定拓扑特征的合成数据；
- **拓扑可解释AI**（Topological XAI）：利用Mapper和持续同调解释深度神经网络的决策边界和隐藏层表示的拓扑结构。

这些研究表明，计算拓扑不仅是一种数学工具，更是连接纯数学与人工智能应用的桥梁。
