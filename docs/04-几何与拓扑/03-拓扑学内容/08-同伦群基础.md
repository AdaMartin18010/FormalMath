## 5.8.1 引言 / Introduction

基本群$\\pi_1(X)$捕捉了空间中一维洞的结构，但拓扑空间可能含有更高维的洞——例如，$S^2$是单连通的（$\\pi_1(S^2) = 0$），但显然它"包围"着一个2维的空腔。高阶同伦群$\\pi_n(X)$（$n \\geq 2$）正是为了量化这些高维洞而发明的。1935年，Hurewicz引入了高阶同伦群的概念，并证明了它们都是阿贝尔群，这为同伦论作为代数拓扑的核心分支奠定了基础。

与基本群不同，高阶同伦群的计算极为困难。即使是对最简单的空间——球面$S^n$——其同伦群$\\pi_k(S^n)$至今也没有完全确定。这种复杂性恰恰反映了同伦论深刻的内涵：球面同伦群的计算不仅是拓扑学的核心问题，还与代数K-理论、配边理论、微分拓扑中的球面微分同胚分类以及弦论中的brane分类有着千丝万缕的联系。

本文档对齐Hatcher（2002）《Algebraic Topology》第四章的内容，并参考Whitehead（1978）《Elements of Homotopy Theory》与May（1999）《A Concise Course in Algebraic Topology》。核心内容包括：高阶同伦群的定义、基本性质、纤维序列的长正合序列、Whitehead定理、胞腔逼近定理，以及球面同伦群的部分计算结果。

## 5.8.2 高阶同伦群的定义 / Definition of Higher Homotopy Groups

### 定义 / Definition

设$X$为拓扑空间，$x_0 \\in X$为基点。$S^n$的标准基点取为$(1, 0, \\ldots, 0) \\in S^n \\subset \\mathbb{R}^{n+1}$。

**定义 2.1**（高阶同伦群）. $X$的**第$n$阶同伦群**$\\pi_n(X, x_0)$定义为所有连续映射$f: (S^n, s_0) \\to (X, x_0)$在保持基点的同伦等价关系下的集合，配备运算：
\\[
(f + g)(t_1, \\ldots, t_n) = \\begin{cases} f(2t_1, t_2, \\ldots, t_n) & 0 \\leq t_1 \\leq 1/2, \\\ g(2t_1 - 1, t_2, \\ldots, t_n) & 1/2 \\leq t_1 \\leq 1. \\end{cases}
\\]

等价地，利用$I^n = [0, 1]^n$和$\\partial I^n$为其边界，$\\pi_n(X, x_0)$也可定义为映射$(I^n, \\partial I^n) \\to (X, x_0)$的同伦类集合。

**定理 2.2**（阿贝尔性）. 对$n \\geq 2$，$\\pi_n(X, x_0)$是阿贝尔群。

*证明思路*：当$n \\geq 2$时，可以在两个坐标方向上交换$f$和$g$的"拼接位置"，从而证明$f + g \\simeq g + f$。具体地，将$I^n$中$f$和$g$的定义区域绕某轴旋转180度，即可构造出所需的同伦。$\\square$

**命题 2.3**（基点变换）. 若$X$道路连通，则$\\pi_n(X, x_0) \\cong \\pi_n(X, x_1)$对任意$x_0, x_1 \\in X$成立（但同构一般不是典范的）。

### 与基本群的作用 / Action of $\\pi_1$

**定理 2.4**. $\\pi_1(X, x_0)$通过共轭作用在$\\pi_n(X, x_0)$上：对$[\\gamma] \\in \\pi_1(X, x_0)$和$[f] \\in \\pi_n(X, x_0)$，
\\[
\\gamma \\cdot [f] = [\\gamma \\circ f \\circ \\gamma^{-1}].
\\]
当$n \\geq 2$时，这是$\\pi_1$在阿贝尔群$\\pi_n$上的一个群作用。

若$\\pi_1$平凡（即$X$单连通），则$\\pi_n(X)$不依赖于基点选择，且是良定义的函子。

## 5.8.3 基本性质与纤维序列 / Basic Properties and Fiber Sequences

### 诱导同态 / Induced Homomorphisms

连续映射$f: (X, x_0) \\to (Y, y_0)$诱导群同态$f_*: \\pi_n(X, x_0) \\to \\pi_n(Y, y_0)$，$f_*([g]) = [f \\circ g]$。

**定理 3.1**.
1. $(f \\circ g)_* = f_* \\circ g_*$；
2. $id_* = id$；
3. 若$f \\simeq g$（保持基点的同伦），则$f_* = g_*$。

### 纤维序列的长正合序列 / Long Exact Sequence of a Fibration

**定义 3.2**（纤维丛/纤维化）. 映射$p: E \\to B$称为**纤维化**（fibration），若对任意空间$X$和交换图
\\[
\begin{array}{ccc}
X \\times \\{0\\} & \\xrightarrow{\\widetilde{H}_0} & E \\
\\downarrow & & \\downarrow p \\
X \\times [0, 1] & \\xrightarrow{H} & B
\\end{array}
\\]
存在提升$\\widetilde{H}: X \\times [0, 1] \\to E$使图表交换。纤维为$F = p^{-1}(b_0)$。

**定理 3.3**（纤维序列的长正合同伦序列）. 对纤维化$p: E \\to B$及纤维$F = p^{-1}(b_0)$，存在长正合序列
\\[
\\cdots \\to \\pi_n(F) \\xrightarrow{i_*} \\pi_n(E) \\xrightarrow{p_*} \\pi_n(B) \\xrightarrow{\\partial} \\pi_{n-1}(F) \\to \\cdots \\to \\pi_0(E) \\to \\pi_0(B) \\to 0.
\\]

*证明思路*：关键在于构造连接同态$\\partial: \\pi_n(B) \\to \\pi_{n-1}(F)$。给定$\\alpha: (S^n, s_0) \\to (B, b_0)$，利用$f: S^n \\cong D^n / S^{n-1}$，先提升$D^n \\times \\{0\\} \\subset D^n \\times [0, 1]$到$E$中（取常值映射到$e_0$），然后通过同伦提升性质将$\\alpha$的零伦提升为$E$中的同伦，其限制在"边界"$S^{n-1}$上给出$F$中的映射。详细构造见Hatcher第四章。$\\square$

**推论 3.4**. 若$E$可缩，则$\\pi_n(B) \\cong \\pi_{n-1}(F)$对所有$n \\geq 1$成立。

这一定理是计算同伦群的最重要工具之一。例如，对覆叠空间$\\mathbb{R} \\to S^1$，纤维为离散的$\\mathbb{Z}$，由长正合序列可再次得到$\\pi_1(S^1) \\cong \\mathbb{Z}$，$\\pi_n(S^1) = 0$对$n \\geq 2$。

### Hopf纤维化 / Hopf Fibration

**定理 3.5**（Hopf纤维化）. 存在纤维化$S^1 \\hookrightarrow S^3 \\xrightarrow{\\eta} S^2$。

由长正合序列及$\\pi_n(S^1) = 0$对$n \\geq 2$，$\\pi_n(S^3) = 0$对$n \\leq 2$（因$S^3$单连通且$\\pi_2(S^3) = 0$），可得：
\\[
\\pi_n(S^2) \\cong \\pi_n(S^3) \\text{ 对 } n \\geq 3.
\\]
特别地，$\\pi_3(S^2) \\cong \\pi_3(S^3) \\cong \\mathbb{Z}$。Hopf映射$\\eta: S^3 \\to S^2$生成了这个无限循环群。

Hopf纤维化是同伦论历史上第一个非平凡的、从低维球面到高维球面的非零同伦群映射，它彻底改变了人们对高维球面拓扑结构的认识。

## 5.8.4 Whitehead定理与胞腔逼近 / Whitehead Theorem and Cellular Approximation

### 相对同伦群 / Relative Homotopy Groups

对空间对$(X, A)$，基点$x_0 \\in A$，定义**相对同伦群**$\\pi_n(X, A, x_0)$为映射$(I^n, \\partial I^n, J^{n-1}) \\to (X, A, x_0)$的同伦类集合，其中$J^{n-1} = \\partial I^n \\setminus (I^{n-1} \\times \\{0\\})$。

**定理 4.1**（空间对的长正合序列）. 对空间对$(X, A)$，存在长正合序列
\\[
\\cdots \\to \\pi_n(A) \\xrightarrow{i_*} \\pi_n(X) \\xrightarrow{j_*} \\pi_n(X, A) \\xrightarrow{\\partial} \\pi_{n-1}(A) \\to \\cdots.
\\]

### Whitehead定理 / Whitehead Theorem

**定理 4.2**（Whitehead定理）. 设$f: X \\to Y$为CW复形之间的映射。若$f$诱导所有同伦群的同构$f_*: \\pi_n(X) \\to \\pi_n(Y)$，则$f$是同伦等价。

*证明思路*：通过映射柱（mapping cylinder）将$f$转化为包含映射$X \\hookrightarrow M_f$，然后利用同伦群的长正合序列和Hurewicz定理逐步构造形变收缩。详见Hatcher第四章§4.1。$\\square$

Whitehead定理是同伦论中最深刻的结果之一。它说明：对CW复形而言，所有同伦群完全决定了空间的同伦型。但需要注意的是，这一定理要求空间是CW复形；对一般拓扑空间，结论不一定成立。

### 胞腔逼近定理 / Cellular Approximation Theorem

**定理 4.3**（胞腔逼近定理）. 设$f: X \\to Y$为CW复形之间的映射，$A \\subset X$为子复形。若$f|_A$是胞腔映射（即将$A$的每个胞腔映射到$Y$的等维或低维胞腔中），则$f$相对于$A$同伦于一个胞腔映射。

这一定理是同调计算中许多简化步骤的理论基础。例如，它可以用来证明：若$X$的维数小于$n$，则$\\pi_n(X) = 0$（任何$S^n \\to X$的映射都可胞腔逼近为将$S^n$映射到$X$的$n$-骨架，但$\\dim X < n$意味着像必须落在更低维的骨架中，从而零伦）。

## 5.8.5 球面同伦群的计算 / Computations of Homotopy Groups of Spheres

### 稳定同伦群 / Stable Homotopy Groups

对$k < n - 1$，同伦群$\\pi_{n+k}(S^n)$与$n$无关，称为**稳定同伦群**$\\pi_k^S$。这一现象由Freudenthal悬挂定理解释。

**定理 5.1**（Freudenthal悬挂定理）. 悬挂同态$E: \\pi_k(S^n) \\to \\pi_{k+1}(S^{n+1})$在$k \\leq 2n - 2$时是满射，在$k < 2n - 1$时是单射。因此$\\pi_{n+k}(S^n)$在$k < n - 1$时稳定。

**例子 5.2**.
- $\\pi_0^S = \\mathbb{Z}$；
- $\\pi_1^S = \\mathbb{Z}/2\\mathbb{Z}$（由Hopf映射$\\eta$生成）；
- $\\pi_2^S = \\mathbb{Z}/2\\mathbb{Z}$；
- $\\pi_3^S = \\mathbb{Z}/24\\mathbb{Z}$。

### 部分已知结果 / Partial Known Results

尽管完整计算球面同伦群是极为困难的开放问题，但以下经典结果是已知的（Hatcher第四章末尾表格）：

| $k$ | $\\pi_k(S^2)$ | $\\pi_k(S^3)$ | $\\pi_k(S^4)$ | $\\pi_k(S^5)$ |
|-----|--------------|--------------|--------------|--------------|
| 1   | 0            | 0            | 0            | 0            |
| 2   | $\\mathbb{Z}$ | 0            | 0            | 0            |
| 3   | $\\mathbb{Z}$ | $\\mathbb{Z}$ | $\\mathbb{Z}$ | 0            |
| 4   | $\\mathbb{Z}/2$ | $\\mathbb{Z}/2$ | $\\mathbb{Z}/2$ | $\\mathbb{Z}$ |
| 5   | $\\mathbb{Z}/2$ | $\\mathbb{Z}/2$ | $\\mathbb{Z}/2$ | $\\mathbb{Z}/2$ |

特别值得注意的是：
- $\\pi_3(S^2) = \\mathbb{Z}$由Hopf映射生成；
- $\\pi_4(S^3) = \\mathbb{Z}/2\\mathbb{Z}$由Hopf映射的悬挂生成；
- $\\pi_n(S^n) = \\mathbb{Z}$对所有$n \\geq 1$成立；
- $\\pi_k(S^n) = 0$对$k < n$成立。

### 同伦群的复杂性 / Complexity of Homotopy Groups

**定理 5.3**（Serre有限性定理）. 对$k > n \\geq 1$，$\\pi_k(S^n)$是有限阿贝尔群，除非$n$为偶数且$k = 2n - 1$（此时$\\pi_{2n-1}(S^n)$含有一个$\\mathbb{Z}$直和项）。

球面同伦群的计算催生了代数拓扑中许多强大工具：谱序列（spectral sequences）、Steenrod运算、Postnikov塔、Adams谱序列等。Adams谱序列及其现代推广至今仍是系统计算稳定同伦群的主要方法。
