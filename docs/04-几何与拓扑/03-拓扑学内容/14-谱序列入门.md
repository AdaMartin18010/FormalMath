## 5.14.1 引言 / Introduction

谱序列（Spectral Sequence）被誉为"二十世纪最复杂但最有用的代数发明"之一。这个由Leray在1940年代为研究层上同调而引入的工具，如今已成为同调代数、代数拓扑、代数几何乃至表示论中不可或缺的计算框架。

谱序列的核心思想是通过一系列"页面"（pages）逼近一个难以直接计算的目标。每一页$E^r$都是一个双分次模（bigraded module），配备微分$d^r$，其同调给出下一页$E^{r+1} = H(E^r, d^r)$。在理想情况下，当$r$足够大时，页面趋于稳定，极限页$E^\infty$与目标对象通过某种滤过（filtration）相联系。

1950年代，Serre将Leray的谱序列应用于纤维化计算，开创了**Leray-Serre谱序列**，这是代数拓扑中最常用的谱序列。此后，Grothendieck引入了**Grothendieck谱序列**用于导出函子的复合，Eilenberg-Moore构造了**Eilenberg-Moore谱序列**研究同伦拉回，Adams发明了**Adams谱序列**计算稳定同伦群。

本文档对齐McCleary（2001）《A User's Guide to Spectral Sequences》、Hatcher（2002）第一章、Weibel（1994）《An Introduction to Homological Algebra》第五章以及Stacks Project的相关内容。

## 5.14.2 谱序列的定义与收敛 / Definition and Convergence of Spectral Sequences

### 谱序列的形式定义 / Formal Definition

**定义 2.1**（同调谱序列）. 一个**同调谱序列**（homological spectral sequence）$(E^r, d^r)_{r \geq a}$由以下数据组成：
1. 对每个整数$r \geq a$（通常$a = 1$或$2$），一个双分次$R$-模$E^r = \bigoplus_{p,q \in \mathbb{Z}} E^r_{p,q}$；
2. **微分**$d^r: E^r \to E^r$是双次数为$(-r, r-1)$的同态，即$d^r: E^r_{p,q} \to E^r_{p-r, q+r-1}$；
3. 满足$(d^r)^2 = 0$，且$E^{r+1}_{p,q} = H(E^r_{p,q}) = \ker(d^r: E^r_{p,q} \to E^r_{p-r, q+r-1}) / \mathrm{im}(d^r: E^r_{p+r, q-r+1} \to E^r_{p,q})$。

**定义 2.2**（上同调谱序列）. **上同调谱序列**（cohomological spectral sequence）的微分次数为$(r, 1-r)$，即$d_r: E_r^{p,q} \to E_r^{p+r, q-r+1}$。

### 收敛性 / Convergence

**定义 2.3**（谱序列的收敛）. 设$H_\bullet$为分次$R$-模，配备有限递减滤过$F^\bullet H_n$：
$$\cdots \supset F^p H_n \supset F^{p+1} H_n \supset \cdots.$$
谱序列$(E^r)$**收敛**到$H_\bullet$（记为$E^a_{p,q} \Rightarrow H_{p+q}$），若对每个$p, q$，存在$R$使得对所有$r \geq R$有$E^r_{p,q} = E^R_{p,q} =: E^\infty_{p,q}$，且
$$E^\infty_{p,q} \cong F^p H_{p+q} / F^{p+1} H_{p+q}.$$

**直观理解**：$E^\infty$的$(p,q)$-分量给出$H_{p+q}$的第$p$个滤过商。通过对$E^\infty$沿对角线求和，可以"拼回"$H_\bullet$（在扩张问题可解的情况下）。

### 边缘同态 / Edge Homomorphisms

**定义 2.4**（边缘同态）. 设$E^2_{p,q} \Rightarrow H_{p+q}$为收敛谱序列。若$E^2_{p,q} = 0$对$p < 0$或$q < 0$（第一象限谱序列），则存在自然映射：
- **左边缘**：$H_n \twoheadrightarrow E^\infty_{n,0} \hookrightarrow E^2_{n,0}$（若$E^2_{n,0}$存活到$E^\infty$）；
- **上边缘**：$E^2_{0,n} \twoheadrightarrow E^\infty_{0,n} \hookrightarrow H_n$。

这些映射在应用中提供了重要的几何信息。

## 5.14.3 正合对与导出正合对 / Exact Couples and Derived Exact Couples

### Massey正合对 / Massey's Exact Couple

**定义 3.1**（正合对）. 一个**正合对**（exact couple）$(D, E, i, j, k)$是如下形式的三角形：
$$D \xrightarrow{i} D \xrightarrow{j} E \xrightarrow{k} D$$
满足$\mathrm{im}(i) = \ker(k)$，$\mathrm{im}(k) = \ker(j)$，$\mathrm{im}(j) = \ker(i)$。

**构造**（从滤过链复形）：设$C_\bullet$为链复形，配备滤过$F^p C_n$。定义
- $D^1_{p,q} = H_{p+q}(F^p C_\bullet)$，由滤过包含诱导$i: D^1_{p,q} \to D^1_{p-1, q+1}$；
- $E^1_{p,q} = H_{p+q}(F^p C_\bullet / F^{p+1} C_\bullet)$；
- 长正合序列给出$j$和$k$。

### 导出正合对 / Derived Exact Couple

**定理 3.2**（导出）. 给定正合对$(D, E, i, j, k)$，定义$d = j \circ k: E \to E$。由$d^2 = 0$，可构造**导出正合对**$(D', E', i', j', k')$：
- $D' = \mathrm{im}(i)$，$i' = i|_{D'}$；
- $E' = H(E, d) = \ker(d) / \mathrm{im}(d)$；
- $j'(i(x)) = [j(x)]$，$k'([y]) = k(y)$。

重复此过程得到谱序列$E^r$，其中$E^1 = E$，$E^{r+1} = H(E^r, d^r)$。

**定理 3.3**（收敛定理）. 设$C_\bullet$为有界滤过链复形（即对每个$n$，存在$p_0, p_1$使得$F^p C_n = 0$对$p > p_0$，$F^p C_n = C_n$对$p < p_1$）。则上述构造的谱序列收敛到$H_*(C_\bullet)$：
$$E^1_{p,q} = H_{p+q}(F^p / F^{p+1}) \Rightarrow H_{p+q}(C_\bullet).$$

## 5.14.4 Leray-Serre谱序列 / Leray-Serre Spectral Sequence

### 纤维化的谱序列 / Spectral Sequence of a Fibration

**定理 4.1**（Leray-Serre谱序列）. 设$F \hookrightarrow E \xrightarrow{\pi} B$为Serre纤维化，$B$道路连通。则存在第一象限同调谱序列：
$$E^2_{p,q} = H_p(B; \mathcal{H}_q(F)) \Rightarrow H_{p+q}(E),$$
其中$\mathcal{H}_q(F)$是$q$阶同调系数的局部系统（由$\pi_1(B)$在$H_q(F)$上的单值作用给出）。

若$\pi_1(B)$在$H_*(F)$上作用平凡（如$B$单连通），则$E^2_{p,q} = H_p(B; H_q(F))$。

**上同调版本**：
$$E_2^{p,q} = H^p(B; \mathcal{H}^q(F)) \Rightarrow H^{p+q}(E).$$

### 乘法结构 / Multiplicative Structure

**定理 4.2**. 若$H^*(F)$和$H^*(B)$为分次交换环，且局部系统是平凡的，则Leray-Serre谱序列有乘法结构：
$$E_r^{p,q} \times E_r^{p',q'} \to E_r^{p+p', q+q'},$$
微分满足Leibniz法则$d_r(xy) = d_r(x)y + (-1)^{p+q}xd_r(y)$对$x \in E_r^{p,q}$。

**应用**：这一乘法结构是计算环面等空间上同调环的关键。

### 经典计算实例 / Classical Computation Examples

**例子 4.3**（$S^n$的环路空间）. 考虑path space fibration $\Omega S^n \hookrightarrow PS^n \to S^n$。由于$PS^n$可缩，谱序列给出：
$$E^2_{p,q} = H_p(S^n; H_q(\Omega S^n)) \Rightarrow H_{p+q}(\text{pt}).$$
分析微分可得$H_q(\Omega S^n) = \mathbb{Z}$对$q = k(n-1)$，$k \geq 0$；否则为0。这给出了著名的**Bott周期性**的雏形。

**例子 4.4**（Hopf纤维化）. 对$S^1 \hookrightarrow S^3 \to S^2$，谱序列有$E^2_{p,q} = H_p(S^2; H_q(S^1))$。非平凡微分$d^2: E^2_{2,0} \to E^2_{0,1}$给出了Hopf映射的同调解释。

## 5.14.5 其他重要谱序列 / Other Important Spectral Sequences

### Grothendieck谱序列 / Grothendieck Spectral Sequence

**定理 5.1**（Grothendieck谱序列）. 设$F: \mathcal{A} \to \mathcal{B}$和$G: \mathcal{B} \to \mathcal{C}$为Abel范畴间的左正合函子，$\mathcal{A}$有足够内射对象，且$F$将内射对象映为$G$-零调对象。则对任意对象$A \in \mathcal{A}$，存在谱序列：
$$E_2^{p,q} = (R^p G)(R^q F)(A) \Rightarrow R^{p+q}(G \circ F)(A).$$

**应用**：这是导出函子复合的基本工具。例如，层上同调的 Leray谱序列：对连续映射$f: X \to Y$，有$E_2^{p,q} = H^p(Y; R^q f_* \mathcal{F}) \Rightarrow H^{p+q}(X; \mathcal{F})$。

### Lyndon-Hochschild-Serre谱序列 / LHS Spectral Sequence

**定理 5.2**（LHS谱序列）. 设$G$为群，$N \trianglelefteq G$为正规子群。则对$G$-模$M$，存在谱序列：
$$E_2^{p,q} = H^p(G/N; H^q(N; M)) \Rightarrow H^{p+q}(G; M).$$

这是群上同调中研究群扩张的基本工具。

### Adams谱序列 / Adams Spectral Sequence

**定理 5.3**（Adams谱序列）. 存在谱序列：
$$E_2^{s,t} = \mathrm{Ext}_{\mathcal{A}_p}^{s,t}(H^*(X; \mathbb{Z}/p), \mathbb{Z}/p) \Rightarrow \pi_{t-s}^S(X)_p^\wedge,$$
其中$\mathcal{A}_p$为模$p$ Steenrod代数，$\pi_*^S$为稳定同伦群，$_p^\wedge$表示$p$-进完备化。

这是计算球面同伦群及其稳定化的最强大工具。

## 5.14.6 实战问题 / Practical Problems

### 问题1：用Leray-Serre谱序列计算$\mathbb{C}P^n$的同调

**问题**：利用纤维化的Leray-Serre谱序列计算$\mathbb{C}P^n$的整系数同调群。

**解答思路**：

**选择适当的纤维化**：
1. $S^1 \hookrightarrow S^{2n+1} \to \mathbb{C}P^n$（Hopf纤维化推广）；
2. 这是主$U(1)$-丛，$S^{2n+1}$是$\mathbb{C}P^n$上的万有丛的全空间。

**谱序列设置**：
1. $E^2_{p,q} = H_p(\mathbb{C}P^n; H_q(S^1))$；
2. 由于$\mathbb{C}P^n$单连通，无局部系统扭曲；
3. $H_q(S^1) = \mathbb{Z}$对$q = 0, 1$，否则为0。

**利用目标信息**：
1. 谱序列收敛到$H_*(S^{2n+1})$；
2. $H_k(S^{2n+1}) = \mathbb{Z}$对$k = 0, 2n+1$，否则为0。

**分析微分**：
1. 唯一可能的非零微分是$d^2: E^2_{p,0} \to E^2_{p-2, 1}$；
2. 由于$H_{2n+1}(S^{2n+1}) = \mathbb{Z}$，且$E^\infty_{2n+1, 0} = \mathbb{Z}$，$E^\infty_{0, 2n+1} = 0$；
3. 归纳分析：从$E^2_{0,0} = \mathbb{Z}$开始；
4. $d^2: E^2_{2,0} \to E^2_{0,1}$必须是同构，否则$E^\infty$会有过多非零项；
5. 同理，$d^2: E^2_{4,0} \to E^2_{2,1}$是同构，等等；
6. 最终$E^2_{p,0} = \mathbb{Z}$对$p = 0, 2, 4, \ldots, 2n$，微分将它们两两配对消去，只留$E^\infty_{0,0} = E^\infty_{2n+1, 0} = \mathbb{Z}$。

**结论**：$H_k(\mathbb{C}P^n; \mathbb{Z}) = \begin{cases} \mathbb{Z} & k = 0, 2, 4, \ldots, 2n, \\ 0 & \text{otherwise}. \end{cases}$

这给出了复射影空间的贝蒂数，且所有同调无挠。

### 问题2：Atiyah-Hirzebruch谱序列与K-理论

**问题**：设$X$为有限CW复形。描述Atiyah-Hirzebruch谱序列
$$E_2^{p,q} = H^p(X; K^q(\text{pt})) \Rightarrow K^{p+q}(X),$$
并计算$K^0(\mathbb{C}P^1)$验证结果。

**解答思路**：

**谱序列背景**：
1. Atiyah-Hirzebruch谱序列将普通上同调与广义上同调（如$K$-理论）联系；
2. 这是Brown可表性定理和Postnikov塔构造的推论。

**$K$-理论的系数**：
1. $K^q(\text{pt}) = \mathbb{Z}$对$q$偶数，$0$对$q$奇数（Bott周期性）；
2. $K^0$对应复向量丛的同构类群（形式差），$K^1$与酉群的同伦有关。

**对$\mathbb{C}P^1 = S^2$的计算**：
1. $E_2^{p,q} = H^p(S^2; K^q(\text{pt}))$；
2. 非零项：$E_2^{0,0} = E_2^{2,0} = E_2^{0,-2} = E_2^{2,-2} = \mathbb{Z}$，及$q = 2k$的其他项；
3. 由Bott周期性，谱序列2-周期：$E_2^{p,q} = E_2^{p, q+2}$；
4. 因此只需考虑$q = 0, -2$的"带"。

**分析微分和扩张**：
1. 对$S^2$，唯一可能的微分是$d_2: E_2^{0,-2} \to E_2^{2,-3}$，但$E_2^{2,-3} = 0$；
2. 同理，$d_2: E_2^{0,0} \to E_2^{2,-1} = 0$；
3. 因此谱序列在$E_2$页塌缩：$E_\infty = E_2$；
4. $K^0(S^2)$的滤过：$F^2 \subset F^1 \subset F^0 = K^0(S^2)$；
5. $E_\infty^{0,0} = F^0/F^1 = \mathbb{Z}$，$E_\infty^{2,-2} = F^2 = \mathbb{Z}$；
6. 扩张问题：$0 \to \mathbb{Z} \to K^0(S^2) \to \mathbb{Z} \to 0$。

**结论**：$K^0(S^2) = \mathbb{Z} \oplus \mathbb{Z}$，由平凡丛和超平面丛（Hopf丛）生成。这与Bott周期性$\widetilde{K}^0(S^2) \cong K^0(\text{pt}) = \mathbb{Z}$一致。

### 问题3：群上同调的LHS谱序列计算

**问题**：设$G = D_8$为8阶二面体群，$N = \mathbb{Z}/4\mathbb{Z}$为其正规子群（旋转子群）。利用Lyndon-Hochschild-Serre谱序列计算$H^2(D_8; \mathbb{Z})$。

**解答思路**：

**群结构**：
1. $D_8 = \langle r, s \mid r^4 = s^2 = 1, srs = r^{-1} \rangle$；
2. $N = \langle r \rangle \cong \mathbb{Z}/4\mathbb{Z}$；
3. $G/N = \langle s \rangle \cong \mathbb{Z}/2\mathbb{Z}$。

**LHS谱序列设置**：
$$E_2^{p,q} = H^p(G/N; H^q(N; \mathbb{Z})) \Rightarrow H^{p+q}(G; \mathbb{Z}).$$

**计算$H^q(N; \mathbb{Z})$**：
1. $N = \mathbb{Z}/4\mathbb{Z}$的整系数群上同调：
   - $H^0(N; \mathbb{Z}) = \mathbb{Z}^N = \mathbb{Z}$（不变元）；
   - $H^{2k+1}(N; \mathbb{Z}) = 0$；
   - $H^{2k}(N; \mathbb{Z}) = \mathbb{Z}/4\mathbb{Z}$对$k \geq 1$。
2. $G/N \cong \mathbb{Z}/2\mathbb{Z}$在$H^q(N; \mathbb{Z})$上的作用：
   - 在$H^0$上平凡（总是如此）；
   - 在$H^{2k}$上，$s$作用为$\pm 1$（取决于具体扩张）。

**对$q = 0$**：
1. $E_2^{p,0} = H^p(\mathbb{Z}/2\mathbb{Z}; \mathbb{Z}) = \begin{cases} \mathbb{Z} & p = 0, \\ 0 & p \text{ 奇数}, \\ \mathbb{Z}/2\mathbb{Z} & p > 0 \text{ 偶数}. \end{cases}$

**对$q = 2$**：
1. $H^2(N; \mathbb{Z}) = \mathbb{Z}/4\mathbb{Z}$；
2. 共轭作用$srs^{-1} = r^{-1}$在$N = \langle r \rangle$上是取逆；
3. 在$H_{2k}(N) = \mathbb{Z}/4\mathbb{Z}$上，取逆诱导乘以$(-1)^{2k} = 1$；
4. 由万有系数定理，在$H^{2k}(N; \mathbb{Z})$上也是平凡作用。

**计算$E_2^{p,2}$**：
1. $E_2^{0,2} = (\mathbb{Z}/4\mathbb{Z})^{\mathbb{Z}/2\mathbb{Z}} = \mathbb{Z}/4\mathbb{Z}$；
2. $E_2^{1,2} = H^1(\mathbb{Z}/2\mathbb{Z}; \mathbb{Z}/4\mathbb{Z}) = \mathbb{Z}/2\mathbb{Z}$；
3. $E_2^{2,2} = H^2(\mathbb{Z}/2\mathbb{Z}; \mathbb{Z}/4\mathbb{Z}) = \mathbb{Z}/2\mathbb{Z}$。

**收敛到$H^2(G; \mathbb{Z})$**：
1. $H^2(G; \mathbb{Z})$由$E_\infty^{2,0}$，$E_\infty^{1,1}$，$E_\infty^{0,2}$贡献；
2. $E_\infty^{1,1} = 0$（因为$H^1(N) = 0$）；
3. 分析微分后得到$H^2(D_8; \mathbb{Z}) = (\mathbb{Z}/2\mathbb{Z})^2$。

**验证**：用Schur乘子：$H_2(G; \mathbb{Z}) = \mathbb{Z}/2\mathbb{Z}$，由万有系数$H^2(G; \mathbb{Z}) = \mathrm{Ext}(H_1, \mathbb{Z}) = \mathbb{Z}/2\mathbb{Z} \oplus \mathbb{Z}/2\mathbb{Z}$。

**结论**：$H^2(D_8; \mathbb{Z}) = (\mathbb{Z}/2\mathbb{Z})^2$，由两个2阶元素生成。
