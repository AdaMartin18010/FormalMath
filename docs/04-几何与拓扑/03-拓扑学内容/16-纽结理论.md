## 5.16.1 引言 / Introduction

纽结理论（Knot Theory）是拓扑学中最直观、最古老却又最活跃的分支之一。从19世纪Kelvin勋爵关于以太涡旋的原子模型猜想，到20世纪Alexander、Reidemeister和Seifert等人的奠基性工作，再到1984年Vaughan Jones发现Jones多项式引发的革命，纽结理论始终是低维拓扑的核心驱动力。

纽结理论研究的是$S^1$在$\mathbb{R}^3$（或$S^3$）中的嵌入，以及这些嵌入在同痕（isotopy）意义下的分类。表面上看，这是一个简单的问题：给定两个纽结图，如何判断它们表示相同的纽结？然而，这个问题的困难程度远超直觉——两个看似截然不同的纽结图可能代表同一个纽结（如活结可以解开），而两个极其相似的图可能代表不同的纽结。

纽结不变量（knot invariants）是区分纽结的基本工具。从早期的三色性、桥数、亏格，到经典的Alexander多项式，再到现代的三维流形不变量、量子不变量和纽结同调，每一种不变量都揭示了纽结的不同侧面，也反映了数学其他分支（如表示论、代数几何、统计力学）的深刻联系。

本文档对齐Reidemeister（1933）《Knotentheorie》、Rolfsen（1976）《Knots and Links》、Lickorish（1997）《An Introduction to Knot Theory》以及Jones（1985）原始论文的相关内容。

## 5.16.2 纽结与链环的基本概念 / Basic Concepts of Knots and Links

### 纽结与链环的定义 / Definition of Knots and Links

**定义 2.1**（纽结）. $S^1$在$\mathbb{R}^3$（或$S^3$）中的**嵌入**（embedding）$K: S^1 \hookrightarrow \mathbb{R}^3$的像称为一个**纽结**（knot）。两个纽结$K_1, K_2$**等价**（记为$K_1 \cong K_2$），若存在ambient同痕$H: \mathbb{R}^3 \times [0,1] \to \mathbb{R}^3$使得$H(-, 0) = \mathrm{id}$且$H(K_1, 1) = K_2$。

**定义 2.2**（链环）. **链环**（link）$L = K_1 \cup \cdots \cup K_n$是$S^1$的不交并（$n$个分支）在$\mathbb{R}^3$中的嵌入。$n = 1$时即为纽结。

**定义 2.3**（平凡纽结）. 与$S^1 \subset \mathbb{R}^2 \subset \mathbb{R}^3$（平面圆）等价的纽结称为**平凡纽结**（unknot）或**平凡结**（trivial knot）。

### 纽结图与交叉点 / Knot Diagrams and Crossings

**定义 2.4**（纽结图）. 通过投影映射$\pi: \mathbb{R}^3 \to \mathbb{R}^2$，$(x, y, z) \mapsto (x, y)$，纽结$K$的像称为**纽结图**（knot diagram）。图在**一般位置**（general position）时满足：
1. 仅有有限个二重点（双交叉）；
2. 无三重或更高重点；
3. 交叉处上下关系明确（通过断开下弧线表示）。

**定义 2.5**（交叉数）. 纽结$K$的**交叉数**（crossing number）$c(K)$是所有$K$的图中交叉点个数的最小值。

**基本纽结分类（按交叉数）**：
- $c = 0$：只有平凡结；
- $c = 3$：只有三叶结（trefoil knot，左右手性不同）；
- $c = 4$：只有八字结（figure-eight knot）；
- $c = 5$：两个纽结（$5_1$和$5_2$）；
- $c = 6$：三个纽结；
- $c = 7$：七个纽结。

### 基本纽结不变量 / Basic Knot Invariants

**定义 2.6**（三色性）. 纽结图是**三色的**（tricolorable），若可对每个弧线赋予红、绿、蓝三色之一，使得：
1. 至少两种颜色被使用；
2. 在每个交叉处，三弧线颜色全同或全不同。

三色性是纽结不变量：三叶结是三色的，平凡结不是，因此三叶结不平凡。

**定义 2.7**（桥数）. 纽结$K$的**桥数**（bridge number）$b(K)$是使$K$的某图中所有上跨弧线（over-arcs）连通分支数为$n$的最小$n$。

**性质**：$b(K) = 1$当且仅当$K$是平凡结。三叶结有$b = 2$，八字结也有$b = 2$。

**定义 2.8**（亏格）. 纽结$K$的**亏格**（genus）$g(K)$是$K$的所有Seifert面的最小亏格。

**Seifert面**：以$K$为边界的紧致可定向曲面$F \subset S^3$。

**性质**：$g(K) = 0$当且仅当$K$是平凡结。三叶结有$g = 1$，八字结也有$g = 1$。

## 5.16.3 Reidemeister定理与纽结不变量 / Reidemeister Theorem and Knot Invariants

### Reidemeister定理 / Reidemeister's Theorem

**定理 3.1**（Reidemeister, 1927）. 两个纽结图表示等价纽结当且仅当它们可以通过一系列**Reidemeister移动**及其逆相互转化：

- **R-I移动**：添加或移除一个纽结（twist）；
- **R-II移动**：添加或移除两个交叉（kink）；
- **R-III移动**：滑动一个弧线过交叉点（slide）。

**推论 3.2**. 任何从纽结图计算的、在Reidemeister移动下不变的量都是纽结不变量。

**证明思路**：Reidemeister证明了任何两个等价纽结图的ambient同痕可以通过投影后的一连串标准移动（即R-I, R-II, R-III）实现。

### 纽结群 / The Knot Group

**定义 3.3**（纽结群）. 纽结$K$的**纽结群**（knot group）定义为
$$\pi(K) = \pi_1(S^3 \setminus K).$$

**定理 3.4**. 纽结群是纽结不变量：若$K_1 \cong K_2$，则$\pi(K_1) \cong \pi(K_2)$。

**Wirtinger表现**：给定纽结图，纽结群有如下表现：
- **生成元**：每个弧线对应一个生成元$x_i$；
- **关系**：每个交叉给出关系$x_j = x_k x_i x_k^{-1}$（取决于交叉类型）。

**定理 3.5**（Wirtinger）. 上述表现给出$\pi(K)$。

**例子**：三叶结的Wirtinger表现：$\langle x, y \mid x y x = y x y \rangle$。

### 亚历山大多项式的组合定义 / Combinatorial Definition of Alexander Polynomial

**定义 3.6**（Alexander多项式，初等定义）. 对纽结$K$，考虑其Alexander-Conway多项式$\nabla_K(z)$，满足：
1. $\nabla_{\text{unknot}}(z) = 1$；
2. **skein关系**：对三个仅在某一交叉处不同的纽结$L_+, L_-, L_0$，
   $$\nabla_{L_+}(z) - \nabla_{L_-}(z) = z \nabla_{L_0}(z).$$

**与经典Alexander多项式的关系**：$\Delta_K(t) = \nabla_K(t^{1/2} - t^{-1/2})$。

**例子**：
- 三叶结：$\nabla(z) = 1 + z^2$，$\Delta(t) = t - 1 + t^{-1}$；
- 八字结：$\nabla(z) = 1 - z^2$，$\Delta(t) = -t + 3 - t^{-1}$。

## 5.16.4 Alexander多项式与Jones多项式 / Alexander and Jones Polynomials

### Alexander多项式的拓扑定义 / Topological Definition of Alexander Polynomial

**定义 4.1**（Alexander模）. 设$\widetilde{X}$为$X = S^3 \setminus K$的万有阿贝尔覆叠（对应核$[\pi, \pi]$）。$H_1(\widetilde{X}; \mathbb{Z})$是$\mathbb{Z}[t, t^{-1}]$-模，称为**Alexander模**。

**定义 4.2**（Alexander多项式）. Alexander多项式$\Delta_K(t)$是Alexander模的初等理想$E_1$的生成元（在$\mathbb{Z}[t, t^{-1}]$的单位倍数意义下）。

**性质**：
1. $\Delta_K(t^{-1}) = \Delta_K(t)$（对$\pm t^n$倍数）；
2. $\Delta_K(1) = \pm 1$；
3. 若$K$是slice纽结，则$\Delta_K(t) = f(t)f(t^{-1})$对某$f$。

### Jones多项式 / The Jones Polynomial

**定理 4.3**（Jones, 1984）. 存在纽结不变量$V_K(t) \in \mathbb{Z}[t^{1/2}, t^{-1/2}]$满足：
1. $V_{\text{unknot}} = 1$；
2. **Kauffman-Jones skein关系**：
   $$t^{-1} V_{L_+}(t) - t V_{L_-}(t) = (t^{1/2} - t^{-1/2}) V_{L_0}(t).$$

**计算示例**：三叶结的Jones多项式。
- 通过skein关系约化到平凡结；
- $V_{\text{trefoil}} = t^{-1} + t^{-3} - t^{-4}$。

**Jones多项式的性质**：
1. 可区分三叶结的左右手性（Alexander不能）；
2. 对镜像$\overline{K}$，$V_{\overline{K}}(t) = V_K(t^{-1})$；
3. 对连通和，$V_{K_1 \# K_2} = V_{K_1} \cdot V_{K_2}$。

### HOMFLY多项式 / The HOMFLY Polynomial

**定理 4.4**（HOMFLY, 1985）. 存在双变量多项式不变量$P_K(a, z) \in \mathbb{Z}[a^{\pm 1}, z^{\pm 1}]$满足：
1. $P_{\text{unknot}} = 1$；
2. **HOMFLY skein关系**：
   $$a P_{L_+} - a^{-1} P_{L_-} = z P_{L_0}.$$

**特殊性**：
- $a = t^{-1}, z = t^{1/2} - t^{-1/2}$给出Jones多项式；
- $a = 1, z = z$给出Alexander-Conway多项式。

## 5.16.5 纽结同调与范畴化 / Knot Homologies and Categorification

### Khovanov同调 / Khovanov Homology

**定理 5.1**（Khovanov, 2000）. 对每个纽结$K$，存在分次Abel群$\mathit{Kh}^{*,*}(K)$满足：
1. Euler示性数给出Jones多项式：$\sum (-1)^i q^j \dim \mathit{Kh}^{i,j}(K) = V_K(q^2)$；
2. $Kh$是纽结不变量；
3. 包含关于$K$的拓扑信息（如纽结的四亏格）。

**构造思路**（概要）：
1. 对纽结图$K$，构造**立方体分解**（cube of resolutions）：每个交叉有0-解析和1-解析两种选择；
2. $n$个交叉给出$n$维立方体，顶点对应完全解析（无交叉的圆集），边对应改变一个解析；
3. 给每个圆分配向量空间$V = \mathbb{Z}[x]/(x^2)$（分次为$\deg(1) = 0, \deg(x) = -2$）；
4. 边映射由乘法$m: V \otimes V \to V$和余元$\Delta: V \to V \otimes V$给出；
5. 立方体的反交换图经**扁平化**（flattening）后成为链复形，其同调为$\mathit{Kh}(K)$。

**定理 5.2**（不变性）. $Kh(K)$在Reidemeister移动下同构。

### Knot Floer同调 / Knot Floer Homology

**定义 5.3**（Knot Floer同调）. 由Ozsváth-Szabó和Rasmussen独立引入，$\widehat{\mathit{HFK}}(K)$是纽结$K$的**Heegaard Floer同调**，满足：
1. 分次$\widehat{\mathit{HFK}}(K) = \bigoplus_{i,j} \widehat{\mathit{HFK}}_i(K, j)$；
2. Euler示性数给出Alexander多项式：$\sum (-1)^i t^j \dim \widehat{\mathit{HFK}}_i(K, j) = \Delta_K(t)$；
3. **亏格检测**：$g(K) = \max\{j : \widehat{\mathit{HFK}}_*(K, j) \neq 0\}$；
4. **slice判据**：若$K$是slice纽结，则$\widehat{\mathit{HFK}}_0(K, g(K)) = 0$。

Knot Floer同调是目前已知最强的经典纽结不变量之一，能够检测亏格、slice性和纤维性。

### 范畴化的意义 / Significance of Categorification

**范畴化哲学**：将多项式不变量"提升"为分次同调理论，使得：
1. 多项式是同调群的Euler示性数；
2. 同调群包含比多项式更丰富的信息；
3. 映射诱导同调映射，而非仅仅是数。

**应用**：
- **Rasmussen$s$-不变量**：从Khovanov同调定义的slice同态不变量，给出了Milnor猜想的拓扑证明；
- **纽结检测**：$Kh(K)$是否能完全检测纽结？目前未知，但对许多小纽结成立。

## 5.16.6 实战问题 / Practical Problems

### 问题1：Wirtinger表现与Alexander多项式计算

**问题**：
1. 写出三叶结的Wirtinger表现并证明$\pi(K) \cong \langle x, y \mid x y x = y x y \rangle$；
2. 利用Fox微分计算三叶结的Alexander多项式。

**解答思路**：

**第一部分：Wirtinger表现**

三叶结的标准图有3个弧线（生成元$x, y, z$）和3个交叉（关系）：
1. 观察三叶结图：三弧线环绕三个交叉；
2. 选择生成元：设三弧线标记为$x, y, z$；
3. 交叉关系（假设都是正交叉）：
   - 交叉1（$x$上跨$y$）：$z = x y x^{-1}$；
   - 交叉2（$y$上跨$z$）：$x = y z y^{-1}$；
   - 交叉3（$z$上跨$x$）：$y = z x z^{-1}$。

**简化**：
1. 由第一个关系，$z = x y x^{-1}$；
2. 代入第二个：$x = y (x y x^{-1}) y^{-1} = y x y x^{-1} y^{-1}$；
3. 化简：$x y x = y x y$；
4. 验证第三个关系自动满足。

**第二部分：Fox微分计算Alexander多项式**

**Fox微分规则**：
1. $\frac{\partial}{\partial x}(x) = 1$，$\frac{\partial}{\partial x}(y) = 0$（$y \neq x$）；
2. $\frac{\partial}{\partial x}(uv) = \frac{\partial}{\partial x}(u) + u \frac{\partial}{\partial x}(v)$；
3. $\frac{\partial}{\partial x}(x^{-1}) = -x^{-1}$。

**计算**：
1. 关系$r = x y x y^{-1} x^{-1} y^{-1} = 1$（由$x y x = y x y$变形）；
2. $\frac{\partial r}{\partial x} = 1 + x y - x y x y^{-1} x^{-1}$；
3. Abel化（将群环映到$\mathbb{Z}[t, t^{-1}]$，$x, y \mapsto t$）：
   - $\frac{\partial r}{\partial x} \mapsto 1 + t^2 - t = 1 - t + t^2$。

**结论**：三叶结的Alexander多项式为$\Delta(t) = t - 1 + t^{-1}$（或标准化为$1 - t + t^2$）。

### 问题2：Jones多项式的skein计算

**问题**：利用Kauffman-Jones skein关系计算八字结（figure-eight knot）的Jones多项式。

**解答思路**：

**skein关系回顾**：
$$t^{-1} V_{L_+} - t V_{L_-} = (t^{1/2} - t^{-1/2}) V_{L_0}.$$

**八字结的交叉分析**：
1. 八字结有4个交叉，可选择其中一个应用skein关系；
2. 设八字结为$K$，在某交叉处分解为$K_+$（正交叉）、$K_-$（负交叉）、$K_0$（解开）。

**skein树**：
1. 从八字结开始，递归应用skein关系；
2. 每次应用将纽结约化为更简单的纽结；
3. 直到所有叶节点都是平凡结。

**具体计算**（概要）：
1. 八字结$K$在某交叉处：$K_+ = K$，$K_-$为更简单纽结，$K_0$为链环；
2. 继续对$K_-$和$K_0$应用skein关系；
3. 经过有限步，所有叶节点都是平凡结（Jones多项式为1）。

**结果**：
$$V_{\text{figure-eight}}(t) = t^{-2} - t^{-1} + 1 - t + t^2.$$

**验证**：
- 对$t = 1$，$V(1) = 1$（对所有纽结成立，因为$V(1) = 1$是归一化条件）；
- 对镜像$\overline{K}$，$V_{\overline{K}}(t) = V_K(t^{-1}) = t^2 - t + 1 - t^{-1} + t^{-2}$；
- 八字结是** amphichiral**（与镜像等价），因此$V_K(t) = V_K(t^{-1})$，验证通过。

### 问题3：Seifert矩阵与亏格计算

**问题**：给定纽结$K$的Seifert面$F$，定义Seifert矩阵$V$并证明Alexander多项式满足$\Delta_K(t) = \det(V - tV^T)$。用此方法计算三叶结的亏格。

**解答思路**：

**Seifert面的构造**：
1. 对纽结图应用Seifert算法：在每个交叉处按方向连接弧线，得到Seifert圆；
2. 用扭转带（twisted bands）连接Seifert圆，得到以$K$为边界的可定向曲面$F$。

**Seifert矩阵的定义**：
1. 设$F$的亏格为$g$，则$H_1(F) = \mathbb{Z}^{2g}$，基为$\{a_1, \ldots, a_{2g}\}$；
2. **Seifert矩阵**$V$的$(i, j)$元为$lk(a_i^+, a_j)$，其中$a_i^+$是$a_i$沿正法向推离$F$的副本，$lk$为环绕数。

**Alexander多项式的公式**：
$$\Delta_K(t) = \det(V - tV^T).$$

**三叶结的计算**：
1. 三叶结的Seifert面是穿孔环面（once-punctured torus），亏格$g = 1$；
2. $H_1(F) = \mathbb{Z}^2$，基$a, b$；
3. 计算Seifert矩阵：$V = \begin{pmatrix} 1 & 0 \\ 1 & 1 \end{pmatrix}$（具体值依赖于基的选择）；
4. $\Delta(t) = \det\begin{pmatrix} 1 - t & -t \\ 1 & 1 - t \end{pmatrix} = (1-t)^2 + t = 1 - t + t^2$。

**亏格的下界**：
1. $\deg(\Delta_K(t)) \leq 2g(K)$；
2. 对三叶结，$\deg(\Delta) = 2$，因此$g(K) \geq 1$；
3. 由于Seifert面给出$g \leq 1$，故$g(K) = 1$。

**结论**：三叶结的亏格为1，这是纽结理论中最重要的基本结果之一。亏格的计算展示了从Seifert矩阵提取拓扑信息的威力。
