# 胞腔同调理论 - 深度版

## 1. 引言

### 1.1 什么是胞腔同调？

**胞腔同调**（Cellular Homology）是计算拓扑空间同调群的有效方法，特别适用于**CW复形**。与奇异同调相比，胞腔同调通过利用空间的胞腔分解，大大降低了计算的复杂度。

**核心思想**: 将空间分解为不同维数的胞腔（Cell），通过研究边界映射的代数性质来计算拓扑不变量。

### 1.2 历史发展

| 时期 | 数学家 | 贡献 |
|------|--------|------|
| 1895 | 庞加莱 | 创立代数拓扑，引入贝蒂数 |
| 1920s | 诺特 | 将同调公理化，引入群论方法 |
| 1930s | 亚历山德罗夫、霍普夫 | 发展胞腔复形理论 |
| 1949 | 怀特海 | 引入CW复形的概念 |
| 1952 | Eilenberg-Steenrod | 建立同调论的公理化体系 |

## 2. CW复形

### 2.1 胞腔的粘贴

**定义 2.1** (n-胞腔):
**n-胞腔** $e^n$ 是同胚于开球 $\mathring{D}^n = \{x \in \mathbb{R}^n : |x| < 1\}$ 的拓扑空间。

**定义 2.2** (特征映射):
对n-胞腔 $e^n_\alpha$，其**特征映射**是连续映射：

$$\Phi_\alpha: D^n \to X$$

满足：
1. $\Phi_\alpha$ 限制在 $\mathring{D}^n$ 上是到 $e^n_\alpha$ 的同胚
2. $\Phi_\alpha(\partial D^n)$ 包含于 $X^{(n-1)}$ （$(n-1)$-骨架）

### 2.2 CW复形的定义

**定义 2.3** (CW复形):
**CW复形** $X$ 是通过归纳方式构造的空间：

1. **0-骨架**: $X^{(0)}$ 是离散点集
2. **归纳步骤**: $X^{(n)}$ 通过将n-胞腔沿 attaching maps $\varphi_\alpha: S^{n-1} \to X^{(n-1)}$ 粘贴到 $X^{(n-1)}$ 得到
3. $X = \bigcup_{n \geq 0} X^{(n)}$，带有弱拓扑

**示例**:
1. **球面 $S^n$**: 一个0-胞腔 + 一个n-胞腔
2. **环面 $T^2$**: 一个0-胞腔 + 两个1-胞腔 + 一个2-胞腔
3. **射影空间 $\mathbb{R}P^n$**: $k$-骨架为 $\mathbb{R}P^k$，每个维度添加一个胞腔

## 3. 胞腔链复形

### 3.1 胞腔链群

**定义 3.1** (胞腔链群):
设 $X$ 是CW复形，其**n维胞腔链群**定义为：

$$C_n^{cell}(X) := H_n(X^{(n)}, X^{(n-1)})$$

**定理 3.1**:
$C_n^{cell}(X)$ 是以n-胞腔为基的自由阿贝尔群：

$$C_n^{cell}(X) \cong \bigoplus_{\alpha} \mathbb{Z} \cdot e^n_\alpha$$

### 3.2 边界算子

**定义 3.2** (胞腔边界算子):
胞腔边界算子 $\partial_n^{cell}: C_n^{cell}(X) \to C_{n-1}^{cell}(X)$ 定义为复合：

$$\partial_n^{cell}: H_n(X^{(n)}, X^{(n-1)}) \xrightarrow{\partial} H_{n-1}(X^{(n-1)}) \xrightarrow{j_*} H_{n-1}(X^{(n-1)}, X^{(n-2)})$$

其中 $\partial$ 是长正合序列的边界映射，$j_*$ 是包含诱导的映射。

### 3.3 胞腔链复形

**定义 3.3**:
$(C_*^{cell}(X), \partial_*^{cell})$ 构成链复形，即 $\partial_{n-1}^{cell} \circ \partial_n^{cell} = 0$。

## 4. 胞腔同调定理

### 4.1 主要定理

**定理 4.1** (胞腔同调定理):
对CW复形 $X$，胞腔同调与奇异同调同构：

$$H_n^{cell}(X) := \frac{\ker \partial_n^{cell}}{\text{im } \partial_{n+1}^{cell}} \cong H_n(X)$$

**证明概要**:
1. 利用 $(X^{(n)}, X^{(n-1)})$ 的长正合序列
2. 证明 $H_k(X^{(n)}) = 0$ 对 $k > n$
3. 证明 $H_k(X^{(n)}) \cong H_k(X)$ 对 $k < n$
4. 建立胞腔链复形与奇异链复形之间的链映射，证明诱导同构

### 4.2 计算示例

**示例 1: 球面 $S^n$**

胞腔结构：$e^0 \cup e^n$

链复形：
$$0 \to \mathbb{Z} \xrightarrow{0} 0 \to \cdots \to 0 \xrightarrow{0} \mathbb{Z} \to 0$$

同调群：
$$H_k(S^n) = \begin{cases} \mathbb{Z} & k = 0, n \\ 0 & \text{其他} \end{cases}$$

**示例 2: 环面 $T^2 = S^1 \times S^1$**

胞腔结构：$e^0 \cup e^1_a \cup e^1_b \cup e^2$

边界算子：
- $\partial_1 = 0$ （两个1-胞腔都是loop）
- $\partial_2(e^2) = 0$ （2-胞腔的边界是 $aba^{-1}b^{-1}$，在交换化后为0）

同调群：
$$H_k(T^2) = \begin{cases} \mathbb{Z} & k = 0, 2 \\ \mathbb{Z}^2 & k = 1 \\ 0 & \text{其他} \end{cases}$$

**示例 3: 实射影空间 $\mathbb{R}P^n$**

胞腔结构：每个维度一个胞腔 $e^0 \cup e^1 \cup \cdots \cup e^n$

边界算子：$\partial_k(e^k) = 2e^{k-1}$ 或 $0$（取决于 $k$ 的奇偶性）

同调群：
$$H_k(\mathbb{R}P^n) = \begin{cases} \mathbb{Z} & k = 0 \text{ 或 } k = n \text{ (n奇数)} \\ \mathbb{Z}/2\mathbb{Z} & k \text{ 奇}, 0 < k < n \\ 0 & \text{其他} \end{cases}$$

## 5. 胞腔上同调

### 5.1 上链复形

**定义 5.1**:
胞腔上链群 $C^n_{cell}(X; G) = \text{Hom}(C_n^{cell}(X), G)$

上边缘算子 $\delta^n: C^n_{cell} \to C^{n+1}_{cell}$ 是 $\partial_{n+1}$ 的对偶。

### 5.2 杯积结构

**定理 5.1**:
胞腔上同调中的杯积可以通过对角映射和胞腔逼近显式计算。

**示例**: $H^*(\mathbb{C}P^n; \mathbb{Z}) = \mathbb{Z}[x]/(x^{n+1})$，其中 $|x| = 2$。

## 6. 应用

### 6.1 欧拉示性数

**定理 6.1**:
对有限CW复形 $X$：

$$\chi(X) = \sum_{n} (-1)^n \text{rank } H_n(X) = \sum_{n} (-1)^n c_n$$

其中 $c_n$ 是n-胞腔的个数。

### 6.2 Morse理论

**定理 6.2** (Morse):
若 $f: M \to \mathbb{R}$ 是光滑流形上的Morse函数，则 $M$ 有CW结构，其中 $k$-临界点对应 $k$-胞腔。
