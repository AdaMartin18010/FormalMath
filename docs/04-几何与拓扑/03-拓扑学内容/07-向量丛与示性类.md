## 5.7.1 引言 / Introduction

向量丛（vector bundle）是现代拓扑学、微分几何和数学物理的核心对象。从拓扑学角度看，向量丛是纤维丛的特例，其纤维为向量空间；从几何学角度看，切丛、余切丛以及各种张量丛都是向量丛的自然例子；从物理学角度看，规范场论中的物质场就是向量丛的截面。示性类（characteristic classes）则是研究向量丛整体拓扑性质的最有力工具：它们将向量丛映射到底空间的上同调类中，从而用可计算的代数不变量来刻画丛的非平凡性。

Stiefel-Whitney类、Chern类和Pontryagin类是三种最重要的示性类。Stiefel-Whitney类定义在实向量丛上，取值于$\\mathbb{Z}/2\\mathbb{Z}$系数的上同调；Chern类定义在复向量丛上，取值于整数系数上同调；Pontryagin类则是实向量丛的复化丛的Chern类的"实部"。这些示性类不仅具有丰富的代数结构（如Whitney和公式），还与流形的可定向性、自旋结构、指标定理以及配边分类有着深刻联系。

本文档综合Hatcher（2003）《Vector Bundles and K-Theory》、Milnor-Stasheff（1974）《Characteristic Classes》以及Bott-Tu（1982）《Differential Forms in Algebraic Topology》等权威教材，系统介绍向量丛的定义、示性类的公理化构造与计算方法，以及它们在具体空间上的应用。

## 5.7.2 向量丛的定义与例子 / Definition and Examples of Vector Bundles

### 定义 / Definition

**定义 2.1**（实向量丛）. 一个**秩$k$实向量丛**（rank $k$ real vector bundle）是指三元组$(E, M, \\pi)$，其中$E$（全空间）、$M$（底空间）为拓扑空间，$\\pi: E \\to M$为连续满射，满足：
1. 对每点$x \\in M$，纤维$E_x = \\pi^{-1}(x)$具有$k$维实向量空间结构；
2. 局部平凡性：对每点$x \\in M$，存在邻域$U \\subset M$和同胚$\\phi: \\pi^{-1}(U) \\to U \\times \\mathbb{R}^k$，使得$\\pi = pr_1 \\circ \\phi$，且$\\phi|_{E_y}: E_y \\to \\mathbb{R}^k$为线性同构对所有$y \\in U$。

若$E$和$M$为光滑流形，$\\pi$为光滑淹没，且局部平凡化$\\phi$为微分同胚，则称$E$为**光滑向量丛**。

**定义 2.2**（复向量丛）. 将上述定义中的$\\mathbb{R}^k$替换为$\\mathbb{C}^k$，即得**秩$k$复向量丛**的定义。

### 基本例子 / Basic Examples

**例子 2.3**（平凡丛）. $M \\times \\mathbb{R}^k \\xrightarrow{pr_1} M$是最简单的向量丛，称为**平凡丛**（trivial bundle）。若向量丛$E$同构于平凡丛，称$E$是**平凡的**（trivial）。

**例子 2.4**（切丛）. $n$维光滑流形$M$的**切丛**$TM = \\bigsqcup_{p \\in M} T_p M$是秩$n$光滑实向量丛。

**例子 2.5**（Möbius丛）. 在$S^1$上定义秩1实向量丛：将$[0, 1] \\times \\mathbb{R}$的两端$(0, t)$与$(1, -t)$粘合。这称为**Möbius线丛**（Möbius line bundle），它是非平凡丛的最简单例子。

**例子 2.6**（ tautological 丛）. 在$\\mathbb{R}P^n$上，**tautological线丛**$\\gamma^1$的纤维在点$\\ell \\in \\mathbb{R}P^n$（$\\ell$为$\\mathbb{R}^{n+1}$中过原点的直线）处就是直线$\\ell$本身。类似地可定义$\\mathbb{C}P^n$上的复tautological线丛。

### 丛的运算 / Bundle Operations

对给定向量丛可进行多种标准运算：
- **Whitney和**（直和）：$(E \\oplus F)_x = E_x \\oplus F_x$；
- **张量积**：$(E \\otimes F)_x = E_x \\otimes F_x$；
- **外幂**：$\\Lambda^k(E)$；
- **对偶丛**：$E^*$；
- **复化**：对实丛$E$，$E \\otimes_\\mathbb{R} \\mathbb{C}$为复丛。

**定理 2.7**（Whitney和公式，预示）. 对示性类$c$（Stiefel-Whitney、Chern或Pontryagin），有
\\[
c(E \\oplus F) = c(E) \\smile c(F).
\\]
这将在下节作为示性类的公理之一出现。

## 5.7.3 Stiefel-Whitney类 / Stiefel-Whitney Classes

### 公理化定义 / Axiomatic Definition

**定理/定义 3.1**（Stiefel-Whitney类）. 对任意实向量丛$E \\to M$，存在唯一的上同调类$w_i(E) \\in H^i(M; \\mathbb{Z}/2\\mathbb{Z})$（$i = 0, 1, 2, \\ldots$），满足：
1. $w_0(E) = 1$，$w_i(E) = 0$对$i > \\mathrm{rank}(E)$；
2. **自然性**：对丛映射$f^*E \\to E$，有$w_i(f^*E) = f^*w_i(E)$；
3. **Whitney和公式**：$w(E \\oplus F) = w(E) \\smile w(F)$，其中$w(E) = 1 + w_1(E) + w_2(E) + \\cdots$为**全Stiefel-Whitney类**；
4. **规范性**：对$\\mathbb{R}P^1$上的tautological线丛$\\gamma^1$，$w_1(\\gamma^1) \\neq 0 \\in H^1(\\mathbb{R}P^1; \\mathbb{Z}/2\\mathbb{Z}) \\cong \\mathbb{Z}/2\\mathbb{Z}$。

### 可定向性与自旋结构 / Orientability and Spin Structures

**定理 3.2**. $n$维流形$M$可定向当且仅当其切丛的$w_1(TM) = 0$。

*证明思路*：$w_1(TM)$是切丛从结构群$O(n)$到$\\mathbb{Z}/2\\mathbb{Z} \\cong O(n)/SO(n)$的阻碍类。$w_1 = 0$意味着结构群可约化为$SO(n)$，即可定向。$\\square$

**定理 3.3**. 可定向流形$M$存在自旋结构（spin structure）当且仅当$w_2(TM) = 0$。

自旋结构在指标定理和量子场论中具有根本重要性：Dirac算子只能在具有自旋结构的流形上定义。

### 分裂原理 / Splitting Principle

**定理 3.4**（分裂原理）. 对任意复向量丛$E \\to M$，存在底空间的映射$f: F(E) \\to M$使得$f^*E$分裂为线丛的直和$f^*E = L_1 \\oplus \\cdots \\oplus L_k$，且$f^*: H^*(M) \\to H^*(F(E))$是单射。

分裂原理极大地简化了对示性类的计算：只需验证公式在线丛直和上成立，则对一般丛也成立。

## 5.7.4 Chern类与Pontryagin类 / Chern and Pontryagin Classes

### Chern类 / Chern Classes

**定理/定义 4.1**（Chern类）. 对任意复向量丛$E \\to M$，存在唯一的上同调类$c_i(E) \\in H^{2i}(M; \\mathbb{Z})$（$i = 0, 1, 2, \\ldots$），满足与Stiefel-Whitney类类似的公理，但取值于整数上同调，且规范性条件为：对$\\mathbb{C}P^1$上的tautological线丛$\\gamma^1$，$c_1(\\gamma^1)$为$H^2(\\mathbb{C}P^1; \\mathbb{Z}) \\cong \\mathbb{Z}$的生成元。

**全Chern类**定义为$c(E) = 1 + c_1(E) + c_2(E) + \\cdots$。

**例子 4.2**（$\\mathbb{C}P^n$的切丛）. $T\\mathbb{C}P^n$的Chern类为
\\[
c(T\\mathbb{C}P^n) = (1 + \\alpha)^{n+1},
\\]
其中$\\alpha \\in H^2(\\mathbb{C}P^n; \\mathbb{Z})$为标准生成元。具体展开得
\\[
c_k(T\\mathbb{C}P^n) = \\binom{n+1}{k} \\alpha^k.
\\]

### Pontryagin类 / Pontryagin Classes

**定义 4.3**（Pontryagin类）. 设$E$为实向量丛。其**复化丛**$E \\otimes_\\mathbb{R} \\mathbb{C}$是复向量丛。$E$的第$k$个**Pontryagin类**定义为
\\[
p_k(E) = (-1)^k c_{2k}(E \\otimes_\\mathbb{R} \\mathbb{C}) \\in H^{4k}(M; \\mathbb{Z}).
\\]

全Pontryagin类为$p(E) = 1 + p_1(E) + p_2(E) + \\cdots$。

**定理 4.4**. Pontryagin类满足：
1. $p_k(E) = 0$对$2k > \\mathrm{rank}(E)$；
2. 自然性和Whitney和公式（在$\\mathbb{Z}[1/2]$系数下成立，因为$c_{odd}(E \\otimes \\mathbb{C}) = 0$的消除需要除以2）。

**例子 4.5**（$S^{2n}$的切丛）. $TS^{2n}$是平凡丛与法丛的直和（由$\\mathbb{R}^{2n+1}$的平凡丛限制得到）。因此$p(TS^{2n}) = 1$。类似地，所有球面的切丛的Pontryagin类均为平凡。

## 5.7.5 示性数与配边 / Characteristic Numbers and Cobordism

### 示性数 / Characteristic Numbers

设$M$为$4k$维闭定向光滑流形。对$Pontryagin$类的任意多项式$P(p_1, \\ldots, p_k)$，若其次数为$4k$（即$\\deg P = 4k$），则可定义**Pontryagin数**
\\[
P[M] = \\langle P(p_1(TM), \\ldots, p_k(TM)), [M] \\rangle \\in \\mathbb{Z}.
\\]
类似可定义Stiefel-Whitney数（模2）和Chern数。

**定理 5.1**（Thom-Pontryagin配边定理）. 两个闭定向光滑流形$M, N$是**定向配边的**（oriented cobordant）当且仅当它们的所有Stiefel-Whitney数和Pontryagin数都相等。

配边理论将流形的分类问题转化为示性数的代数问题，是代数拓扑在微分拓扑中应用的典范。Thom因此于1958年获得菲尔兹奖。

### Hirzebruch符号差定理 / Hirzebruch Signature Theorem

对$4k$维闭定向流形$M$， cup product 在$H^{2k}(M; \\mathbb{R})$上给出一个对称双线性型，其符号差（正特征值数减负特征值数）称为$M$的**符号差**（signature），记为$\\sigma(M)$。

**定理 5.2**（Hirzebruch符号差定理）. $\\sigma(M)$可表示为Pontryagin类的$L$-多项式：
\\[
\\sigma(M) = \\langle L_k(p_1, \\ldots, p_k), [M] \\rangle.
\\]

例如：
- 对$k = 1$（$\\dim M = 4$）：$\\sigma(M) = \\frac{1}{3} \\langle p_1(TM), [M] \\rangle$；
- 对$k = 2$（$\\dim M = 8$）：$\\sigma(M) = \\frac{1}{45}(7p_2 - p_1^2)[M]$。

Hirzebruch符号差定理是Atiyah-Singer指标定理的一个特例，深刻联系着拓扑、分析与代数几何。

## 5.7.6 计算实例 / Computational Examples

### 实例 1：$\\mathbb{R}P^n$的Stiefel-Whitney类

**定理**. $w(T\\mathbb{R}P^n) = (1 + \\alpha)^{n+1}$，其中$\\alpha \\in H^1(\\mathbb{R}P^n; \\mathbb{Z}/2\\mathbb{Z})$为生成元。

*推导*：$\\mathbb{R}P^n$的切丛满足$\\epsilon^1 \\oplus T\\mathbb{R}P^n \\cong (\\gamma^1)^{\\oplus (n+1)}$，其中$\\epsilon^1$为平凡线丛，$\\gamma^1$为tautological线丛。因此
\\[
w(T\\mathbb{R}P^n) = w((\\gamma^1)^{\\oplus (n+1)}) = w(\\gamma^1)^{n+1} = (1 + \\alpha)^{n+1}.
\\]

例如，$w_1(T\\mathbb{R}P^n) = (n+1)\\alpha$。当$n$为偶数时，$w_1 \\neq 0$，故$\\mathbb{R}P^n$不可定向；当$n$为奇数时，$w_1 = 0$，故$\\mathbb{R}P^n$可定向。

### 实例 2：$S^2$上复线丛的陈类

任何$S^2$上的复线丛$L$由其一阶陈类$c_1(L) \\in H^2(S^2; \\mathbb{Z}) \\cong \\mathbb{Z}$完全分类。整数$k = \\langle c_1(L), [S^2] \\rangle$称为线丛的**度数**（degree）。

霍普夫丛$\\eta: S^3 \\to S^2$对应的复线丛（通过将$S^2$视为$\\mathbb{C}P^1$）满足$c_1(\\eta) = -1$。更一般地，$\\mathcal{O}(k)$丛在$\\mathbb{C}P^1$上的陈类为$k$倍的生成元。

### 实例 3：$T^2$的切丛是平凡的

$TT^2$同构于两个平凡线丛的直和：$TT^2 \\cong \\epsilon^1 \\oplus \\epsilon^1$。因此所有Stiefel-Whitney类、Chern类和Pontryagin类均为平凡。这是平坦环面作为平坦流形的直接推论。
