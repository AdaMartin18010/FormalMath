# Atiyah-Singer 指标定理及其应用

Atiyah-Singer 指标定理是二十世纪数学最伟大的成就之一，它将微分算子的解析指标（核与余核的维数差）与底流形的拓扑不变量（示性类）联系起来。这一定理统一了微分几何、代数拓扑与泛函分析，在数学物理、指标理论与非交换几何中都有深远影响。

## 1. 椭圆算子与指标

### 1.1 微分算子与象征

**定义 1.1（微分算子）**. 设 $E, F$ 为紧流形 $M$ 上的向量丛。$k$ 阶微分算子 $D: \Gamma(E) \to \Gamma(F)$ 局部可写为：

$$D = \sum_{|\alpha| \leq k} A_\alpha(x) \frac{\partial^{|\alpha|}}{\partial x^\alpha},$$

其中 $A_\alpha$ 为矩阵值函数。

**定义 1.2（象征与椭圆性）**. $D$ 的主象征（principal symbol）为：

$$\sigma_D(x, \xi) = \sum_{|\alpha| = k} A_\alpha(x) (i\xi)^\alpha \in \operatorname{Hom}(E_x, F_x), \quad \xi \in T^*_x M.$$

$D$ 称为椭圆的，若 $\sigma_D(x, \xi)$ 对所有 $x \in M$，$\xi \neq 0$ 可逆。

### 1.2 解析指标

**定义 1.3（解析指标）**. 椭圆算子 $D$ 的解析指标定义为：

$$\operatorname{ind}(D) = \dim \ker D - \dim \operatorname{coker} D = \dim \ker D - \dim \ker D^*.$$

由椭圆正则性，$\ker D$ 和 $\ker D^*$ 都是有限维的，故指标良定义。

## 2. Atiyah-Singer 指标定理

### 2.1 定理陈述

**定理 2.1（Atiyah-Singer）**. 设 $D: \Gamma(E) \to \Gamma(F)$ 为紧定向流形 $M$ 上的椭圆微分算子。则：

$$\operatorname{ind}(D) = \int_M \operatorname{ch}(E) \smile \operatorname{td}(TM) \smile \frac{1}{e(TM)} \smile \text{(拓扑修正项)},$$

或更具体地，对 Dirac 型算子：

$$\operatorname{ind}(D) = \int_M \widehat{A}(TM) \smile \operatorname{ch}(E).$$

### 2.2 经典复形的特例

**de Rham 复形**：$d + d^*: \Omega^{\text{even}} \to \Omega^{\text{odd}}$。指标 = Euler 示性数 $\chi(M)$。Gauss-Bonnet 定理：

$$\chi(M) = \int_M \operatorname{Pf}(\frac{-R}{2\pi}).$$

**Dolbeault 复形**：$\bar{\partial} + \bar{\partial}^*$。指标 = Euler 示性数（全纯）$= \sum (-1)^q h^{0,q}$。Riemann-Roch 定理：

$$\chi(\mathcal{O}_X) = \int_M \operatorname{td}(T^{1,0}M).$$

**Dirac 算子**：在旋量流形上，$\slashed{D}: \Gamma(S^+) \to \Gamma(S^-)$。指标 = $\widehat{A}$-亏格：

$$\widehat{A}(M) = \int_M \widehat{A}(TM).$$

## 3. 证明思路

### 3.1 K-理论方法

Atiyah 与 Singer 的原始证明利用 K-理论：

1. 将椭圆算子 $D$ 提升为 K-理论中的类 $[\sigma_D] \in K(T^*M)$；
2. 构造解析指标同态 $\operatorname{ind}: K(T^*M) \to \mathbb{Z}$；
3. 证明拓扑指标 $\operatorname{ind}_t: K(T^*M) \to \mathbb{Z}$ 与之相等（Thom 同构与 Bott 周期性）；
4. 对生成元验证等式（平凡情形），由同态的唯一性推广到一般情形。

### 3.2 热核方法

**定理 3.1（McKean-Singer）**. 

$$\operatorname{ind}(D) = \operatorname{Tr}(e^{-tD^*D}) - \operatorname{Tr}(e^{-tDD^*})$$

对任意 $t > 0$。

*证明*. $e^{-tD^*D}$ 和 $e^{-tDD^*}$ 有相同的正特征值（因 $D$ 给出 $V_\lambda$ 与 $W_\lambda$ 之间的同构，$\lambda > 0$），故正特征值贡献相消。$t \to 0$ 时利用热核渐近展开即得指标公式。$\square$

## 4. 例子

### 4.1 例子：球面 $S^2$ 的指标

$S^2$ 的 Euler 示性数：$\chi(S^2) = 2$。

Gauss-Bonnet：$\int_{S^2} K dA = 4\pi = 2\pi \chi(S^2)$（因 $K = 1$ 对单位球面）。

Dolbeault 指标：$\chi(\mathcal{O}_{\mathbb{P}^1}) = h^{0,0} - h^{0,1} = 1 - 0 = 1$。Riemann-Roch：$\chi(\mathcal{O}) = \int \operatorname{td}(T\mathbb{P}^1) = \int (1 + c_1/2) = 1$（因 $\int c_1 = 2$）。

### 4.2 例子： torus $T^2$ 的 Dirac 指标

平坦 torus $T^2 = \mathbb{R}^2/\mathbb{Z}^2$，旋量丛平凡。Dirac 算子 $\slashed{D} = \begin{pmatrix} 0 & \partial_z \\ \partial_{\bar{z}} & 0 \end{pmatrix}$。

正旋量的核：常数函数（1 维）。负旋量的核：常数函数（1 维）。故 $\widehat{A}(T^2) = 1 - 1 = 0$，与平坦 torus 的 Pontryagin 类为零一致。
