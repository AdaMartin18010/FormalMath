# 高等黎曼几何曲率专题

## 曲率与拓扑

### Bonnet-Myers定理

Ricci曲率$\geq (n-1)k > 0$ $\Rightarrow$ 紧致，直径$\leq \pi/\sqrt{k}$。

### 分裂定理

Ricci曲率$\geq 0$且有直线 $\Rightarrow$ 分裂出$\mathbb{R}$因子。
