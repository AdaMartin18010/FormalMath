# 辛几何基础

## 辛流形

### 定义

辛流形$(M, \omega)$是光滑流形$M$配备闭的非退化2-形式$\omega$。

### 例子

$\mathbb{R}^{2n}$，$\omega = \sum_{i=1}^n dp_i \wedge dq_i$

### 哈密顿力学

哈密顿向量场$X_H$：$\iota_{X_H}\omega = dH$

哈密顿方程：$\dot{q} = \frac{\partial H}{\partial p}$，$\dot{p} = -\frac{\partial H}{\partial q}$
