# 辛几何基础

辛几何（symplectic geometry）是研究辛流形的数学分支，起源于经典力学的 Hamilton 形式体系。与 Riemann 几何不同，辛结构是反对称的、退化的（但在辛流形上非退化），这种结构刚性极强：Darboux 定理断言所有辛流形局部等价于标准辛空间，不存在局部不变量。

## 1. 辛向量空间

### 1.1 基本定义

**定义 1.1（辛形式）**. 设 $V$ 为有限维实向量空间。双线性型 $\omega: V \times V \to \mathbb{R}$ 称为辛形式（symplectic form），若：
1. **反对称**：$\omega(u, v) = -\omega(v, u)$ 对所有 $u, v \in V$；
2. **非退化**：若 $\omega(u, v) = 0$ 对所有 $v \in V$ 成立，则 $u = 0$。

称 $(V, \omega)$ 为辛向量空间（symplectic vector space）。

**命题 1.2**. 辛向量空间必为偶数维。若 $\dim V = 2n$，则存在基 $\{e_1, \dots, e_n, f_1, \dots, f_n\}$ 使得：

$$\omega(e_i, e_j) = 0, \quad \omega(f_i, f_j) = 0, \quad \omega(e_i, f_j) = \delta_{ij}.$$

这样的基称为辛基（symplectic basis）。

*证明*. 取非零 $e_1 \in V$，由非退化性存在 $f_1$ 使 $\omega(e_1, f_1) = 1$。令 $W = \operatorname{span}\{e_1, f_1\}$，考虑其辛正交补：

$$W^\omega = \{ v \in V : \omega(v, w) = 0,\ \forall w \in W \}.$$

由非退化性，$\dim W^\omega = \dim V - 2$。归纳构造即得。$\square$

**定义 1.3（标准辛空间）**. $\mathbb{R}^{2n}$ 上的标准辛形式为：

$$\omega_0 = \sum_{i=1}^n dp_i \wedge dq_i = dp \wedge dq.$$

在矩阵表示下，$\omega_0(u, v) = u^T J v$，其中 $J = \begin{pmatrix} 0 & I_n \\ -I_n & 0 \end{pmatrix}$。

### 1.2 辛群

**定义 1.4（辛群）**. 线性变换 $A \in GL(V)$ 称为辛变换，若 $A^*\omega = \omega$。全体辛变换构成辛群 $Sp(V, \omega)$。

在标准辛空间上：

$$Sp(2n, \mathbb{R}) = \{ A \in GL(2n, \mathbb{R}) : A^T J A = J \}.$$

**命题 1.5**. $\det A = 1$ 对所有 $A \in Sp(2n, \mathbb{R})$ 成立。故 $Sp(2n, \mathbb{R}) \subset SL(2n, \mathbb{R})$。

*证明*. 由 $A^T J A = J$ 取行列式得 $(\det A)^2 \det J = \det J$，而 $\det J = 1$，故 $(\det A)^2 = 1$。进一步利用 $Sp(2n, \mathbb{R})$ 的连通性（它同伦等价于 $U(n)$）及 $\det I = 1$ 得 $\det A = 1$。$\square$

## 2. 辛流形

### 2.1 定义与基本例子

**定义 2.1（辛流形）**. 光滑流形 $M$ 配备闭的非退化 2-形式 $\omega \in \Omega^2(M)$（即 $d\omega = 0$ 且对所有 $p \in M$，$\omega_p$ 为 $T_p M$ 上的辛形式），称为辛流形，记 $(M, \omega)$。

**定理 2.2（Darboux）**. 设 $(M, \omega)$ 为 $2n$ 维辛流形，$p \in M$。则存在 $p$ 的邻域坐标卡 $(U; q_1, \dots, q_n, p_1, \dots, p_n)$ 使得：

$$\omega|_U = \sum_{i=1}^n dp_i \wedge dq_i.$$

*证明概要*. 对 $n$ 归纳。$n=1$ 时由闭形式在二维局部恰当直接得到。归纳步利用 Moser 技巧：构造一族辛形式 $\omega_t = (1-t)\omega_0 + t\omega$，通过求解同伦算子方程得到微分同胚将 $\omega$ 拉回到 $\omega_0$。$\square$

**推论 2.3**. 辛流形没有局部不变量（如曲率），所有不变量均为整体性的。

### 2.2 余切丛上的典范辛结构

**定理 2.4**. 设 $Q$ 为光滑流形，$M = T^*Q$ 其余切丛。则 $M$ 上存在典范辛结构。

*构造*. 定义 Liouville 1-形式 $\theta \in \Omega^1(T^*Q)$：对 $(q, p) \in T^*Q$（其中 $p \in T_q^*Q$）及 $X \in T_{(q,p)}(T^*Q)$，令：

$$\theta_{(q,p)}(X) = p(d\pi_{(q,p)}(X)),$$

其中 $\pi: T^*Q \to Q$ 为投影。则 $\omega = -d\theta$ 为辛形式。

在局部坐标 $(q_i, p_i)$ 下，$\theta = \sum p_i dq_i$，故：

$$\omega = \sum_{i=1}^n dq_i \wedge dp_i = \sum_{i=1}^n dp_i \wedge dq_i.$$

这是 Hamilton 力学的自然几何框架。

## 3. Hamilton 力学

### 3.1 Hamilton 向量场

**定义 3.1（Hamilton 向量场）**. 设 $(M, \omega)$ 为辛流形，$H \in C^\infty(M)$ 为 Hamilton 函数。由 $\omega$ 的非退化性，存在唯一的向量场 $X_H$ 使得：

$$\iota_{X_H}\omega = dH,$$

即 $\omega(X_H, Y) = dH(Y)$ 对所有 $Y \in \mathfrak{X}(M)$ 成立。$X_H$ 称为 $H$ 的 Hamilton 向量场。

**定理 3.2**. Hamilton 向量场的流保持辛形式：$(\phi_{X_H}^t)^*\omega = \omega$，即 $\phi_{X_H}^t$ 为辛同胚。

*证明*. 计算 Lie 导数：

$$\mathcal{L}_{X_H}\omega = d\iota_{X_H}\omega + \iota_{X_H}d\omega = d(dH) + 0 = 0.$$

由 Cartan 公式及 $d\omega = 0$。$\square$

### 3.2 Hamilton 方程

在 Darboux 坐标 $(q_i, p_i)$ 下，$\omega = \sum dp_i \wedge dq_i$，设 $X_H = \sum (\dot{q}_i \partial_{q_i} + \dot{p}_i \partial_{p_i})$，则：

$$\iota_{X_H}\omega = \sum_i (\dot{q}_i dp_i - \dot{p}_i dq_i) = dH = \sum_i \left( \frac{\partial H}{\partial q_i} dq_i + \frac{\partial H}{\partial p_i} dp_i \right).$$

比较系数得 **Hamilton 方程**：

$$\dot{q}_i = \frac{\partial H}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial H}{\partial q_i}.$$

**定理 3.3（能量守恒）**. $H$ 沿 Hamilton 流的积分曲线为常数：$\frac{d}{dt}H(q(t), p(t)) = 0$。

*证明*. $$\frac{dH}{dt} = dH(X_H) = \omega(X_H, X_H) = 0,$$

由 $\omega$ 的反对称性。$\square$

## 4. Lagrangian 子流形

### 4.1 定义与性质

**定义 4.1（Lagrangian 子空间）**. 设 $(V, \omega)$ 为 $2n$ 维辛向量空间。子空间 $L \subset V$ 称为 Lagrangian 子空间，若 $L = L^\omega$（即 $L$ 等于其辛正交补）。等价地，$\dim L = n$ 且 $\omega|_{L \times L} = 0$。

**定义 4.2（Lagrangian 子流形）**. 辛流形 $(M, \omega)$ 的 $n$ 维子流形 $\Lambda \subset M$（$\dim M = 2n$）称为 Lagrangian 子流形，若 $T_x\Lambda$ 为 $T_x M$ 的 Lagrangian 子空间对所有 $x \in \Lambda$ 成立，即 $\omega|_\Lambda = 0$。

**定理 4.3**. 设 $S: Q \to \mathbb{R}$ 为光滑函数，则其图像的导映射：

$$\Lambda_S = \{ (q, dS_q) : q \in Q \} \subset T^*Q$$

为 Lagrangian 子流形。

*证明*. 设 $i: \Lambda_S \hookrightarrow T^*Q$ 为嵌入。由 $\Lambda_S$ 的定义，$i^*\theta = dS$，故：

$$i^*\omega = -i^*d\theta = -d(i^*\theta) = -d(dS) = 0.$$

$\square$

## 5. 例子

### 5.1 例子：标准辛空间中的线性 Lagrangian 子空间

在 $(\mathbb{R}^{2n}, \omega_0)$ 中，$L_0 = \mathbb{R}^n \times \{0\} = \{ (q, 0) \}$ 是 Lagrangian 子空间。更一般地，对任意对称矩阵 $A \in \operatorname{Sym}_n(\mathbb{R})$：

$$L_A = \{ (q, Aq) : q \in \mathbb{R}^n \}$$

是 Lagrangian 子空间，因为 $\omega_0((q, Aq), (q', Aq')) = q^T A q' - (Aq)^T q' = 0$。

### 5.2 例子：一维谐振子

考虑 $Q = \mathbb{R}$，$T^*Q = \mathbb{R}^2$，$\omega = dp \wedge dq$。Hamilton 函数：

$$H(q, p) = \frac{p^2}{2m} + \frac{1}{2}m\omega_0^2 q^2.$$

Hamilton 方程为：

$$\dot{q} = \frac{p}{m}, \quad \dot{p} = -m\omega_0^2 q.$$

解为 $q(t) = A\cos(\omega_0 t + \phi)$，轨道为相空间中的椭圆 $H = E$。每个能量面都是一维 Lagrangian 子流形（因 $\dim = 1 = n$ 且 1-形式在曲线上自然为零）。
