# 04-几何与拓扑

**空间形式的数学 — 从欧几里得几何到现代代数拓扑**

---

## 📋 分支概览

几何与拓扑分支是FormalMath的核心分支之一，涵盖空间形式的数学理论：

- **欧几里得几何**：经典几何的严格化
- **解析几何**：代数方法研究几何
- **微分几何**：曲线与曲面的微分理论
- **非欧几何**：双曲几何与椭圆几何
- **点集拓扑**：拓扑空间的基本理论
- **代数拓扑**：用代数工具研究拓扑
- **微分拓扑**：光滑流形的拓扑

**难度等级**: ⭐⭐⭐⭐（高级）
