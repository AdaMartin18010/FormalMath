# 量子上同调简介

量子上同调（quantum cohomology）是辛几何与代数几何交汇的前沿领域，由 Gromov-Witten 理论驱动。它将经典上同调的交换乘积形变为结合但非交换的"量子积"，编码了流形中伪全纯曲线的计数信息，是镜像对称（mirror symmetry）的数学核心。

## 1. Gromov-Witten 不变量

### 1.1 伪全纯曲线

设 $(M, \omega, J)$ 为紧近复流形（或辛流形配备相容的近复结构 $J$）。

**定义 1.1（伪全纯曲线）**. 光滑映射 $u: (\Sigma, j) \to (M, J)$ 称为伪全纯（pseudo-holomorphic）的（或 $J$-全纯的），若：

$$\bar{\partial}_J u = \frac{1}{2}(du + J \circ du \circ j) = 0.$$

当 $M$ 为复流形且 $J$ 为可积复结构时，此即通常的全纯映射。

### 1.2 模空间

**定义 1.2（Gromov-Witten 模空间）**. 固定同调类 $\beta \in H_2(M; \mathbb{Z})$ 和亏格 $g$。Gromov-Witten 模空间 $\overline{\mathcal{M}}_{g,n}(M, \beta)$ 参数化满足以下条件的稳定映射 $(C, p_1, \dots, p_n, u)$：
- $C$ 为具有 $n$ 个标记点的亏格 $g$ 稳定曲线；
- $u: C \to M$ 为 $J$-全纯映射；
- $u_*[C] = \beta \in H_2(M)$；
- 稳定条件：$u$ 的每个亏格零不可约分量至少有三个特殊点（节点或标记点），亏格一分量至少有一个。

**定理 1.3（Gromov 紧性）**. 对固定的 $\beta$，$\overline{\mathcal{M}}_{g,n}(M, \beta)$ 是紧的拓扑空间（在 Gromov-Hausdorff 意义下）。

### 1.3 Gromov-Witten 不变量

**定义 1.4（GW 不变量）**. 利用 evaluation 映射 $\operatorname{ev}_i: \overline{\mathcal{M}}_{g,n}(M, \beta) \to M$，$\operatorname{ev}_i(C, u) = u(p_i)$，定义：

$$\langle \alpha_1, \dots, \alpha_n \rangle_{g,n,\beta} = \int_{[\overline{\mathcal{M}}_{g,n}(M, \beta)]^{\text{vir}}} \operatorname{ev}_1^*\alpha_1 \smile \cdots \smile \operatorname{ev}_n^*\alpha_n,$$

其中 $[\cdot]^{\text{vir}}$ 为虚拟基本类（virtual fundamental class），用于处理模空间非光滑或维数跳跃的情形。

## 2. 量子上同调环

### 2.1 小量子积

**定义 2.1（量子上同调）**. $M$ 的小量子上同调（small quantum cohomology）$QH^*(M)$ 作为向量空间等于 $H^*(M; \mathbb{C}) \otimes \Lambda$，其中 $\Lambda$ 为 Novikov 环。乘法（量子积）定义为：

$$(a * b, c) = \sum_\beta \langle a, b, c \rangle_{0,3,\beta} q^\beta,$$

其中 $q^\beta$ 为形式参数，$(\cdot, \cdot)$ 为 Poincaré 配对。

**定理 2.2**. 量子积是结合的、分次交换的（带 grading shift），且以经典上同调为经典极限（$q \to 0$ 时量子积退化为杯积）。

*证明概要*. 结合性对应于模空间 $\overline{\mathcal{M}}_{0,4}(M, \beta)$ 的分解：将四点曲线分裂为两个三点曲线的不同方式给出相同的 GW 不变量。这是 divisor axiom 与 splitting axiom 的结果。$\square$

### 2.2 例子：复射影空间

对 $M = \mathbb{CP}^n$，$H^*(\mathbb{CP}^n) = \mathbb{C}[h]/(h^{n+1})$，其中 $h = c_1(\mathcal{O}(1))$。

**定理 2.3**. $\mathbb{CP}^n$ 的量子上同调满足：

$$h * h^n = q \cdot 1,$$

即 $QH^*(\mathbb{CP}^n) \cong \mathbb{C}[h, q]/(h^{n+1} - q)$。

*推导*. 在 $\mathbb{CP}^n$ 中，过两点的直线唯一（类 $\beta = [\mathbb{CP}^1]$）。通过计算三点 GW 不变量，$\langle h^i, h^j, h^k \rangle_{0,3,\beta} = 1$ 当 $i+j+k = n+1$，否则为 0（适当范围内）。$\square$

## 3. 镜像对称

### 3.1 物理起源与数学表述

镜像对称断言：Calabi-Yau 三维流形 $X$ 的辛几何（A-模型）与其"镜像" $X^\vee$ 的复几何（B-模型）等价。

**定义 3.1（Calabi-Yau 流形）**. 紧 Kähler 流形 $X$ 称为 Calabi-Yau，若 $c_1(X) = 0$（等价地，存在 Ricci 平坦 Kähler 度量）。

**定理 3.2（Candelas-de la Ossa-Green-Parkes）**. 对五次超曲面 $X_5 \subset \mathbb{CP}^4$（CY 三维流形），其亏格零 GW 生成函数可通过对偶的 Picard-Fuchs 方程计算。

### 3.2 Hodge 结构的镜像

**定理 3.3**. 镜像对称意味着 Hodge 数的对称：

$$h^{p,q}(X) = h^{n-p,q}(X^\vee).$$

特别地，$h^{1,1}(X) = h^{2,1}(X^\vee)$，即 Kähler 模空间维数等于 $X^\vee$ 的复结构模空间维数。

## 4. 例子

### 4.1 例子：$S^2$ 的量子上同调

$H^*(S^2) = \mathbb{C}[h]/(h^2)$，$\deg h = 2$。量子关系：

$$h * h = \langle h, h, \text{pt} \rangle q^{[S^2]} \cdot 1 = q \cdot 1,$$

因为过 $S^2$ 上两点有唯一的球面（自身）。故 $QH^*(S^2) \cong \mathbb{C}[h, q]/(h^2 - q)$。

### 4.2 例子：镜像流形的构造

Batryev-Borisov 构造：对 Fano 完全交 $X \subset \mathbb{CP}^n$，通过其对偶多面体（reflexive polytope）构造镜像 $X^\vee$。例如，

$$X = \{x_0^5 + x_1^5 + x_2^5 + x_3^5 + x_4^5 = 0\} \subset \mathbb{CP}^4$$

的镜像为 Greene-Plesser 轨形商：$X^\vee = X / (\mathbb{Z}_5)^3$。
