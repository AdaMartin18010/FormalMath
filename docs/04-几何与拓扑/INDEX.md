## 📚 学习路径推荐

### 路径A：点集拓扑 → 代数拓扑（标准路线）

```
学期1: 点集拓扑基础
├── 拓扑空间与连续映射
├── 分离公理与可数性
├── 紧致性与连通性
└── Urysohn引理与Tychonoff定理

学期2: 代数拓扑基础  
├── 基本群与覆叠空间
├── 奇异同调论
├── 胞腔同调与CW复形
└── 上同调环

学期3: 高级代数拓扑
├── 同伦论
├── 纤维丛与示性类
└── 谱序列
```

### 路径B：微分几何 → 黎曼几何（物理导向）

```
学期1: 曲线曲面论
├── 曲线论（Frenet标架）
├── 曲面论（第一、第二基本形式）
├── 高斯曲率与平均曲率
└── Gauss-Bonnet定理

学期2: 光滑流形
├── 光滑流形基础
├── 切空间与向量场
├── 张量分析与微分形式
└── 积分与Stokes定理

学期3: 黎曼几何
├── 黎曼度量与联络
├── 曲率张量
├── 测地线与指数映射
└── 比较几何
```

### 路径C：几何拓扑 → 低维拓扑（研究导向）

```
学期1: 微分拓扑
├── 光滑流形
├── Sard定理与横截性
├── Whitney嵌入定理
└── 向量丛

学期2: 低维拓扑
├── 纽结理论
├── 三维流形
├── 几何化猜想
└── Floer同调
```
