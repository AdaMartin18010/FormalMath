## 📚 概述

计算几何研究几何问题的算法设计与分析，是计算机科学和离散几何的交叉领域。本文档介绍计算几何的基本数据结构、核心算法及其在实际问题中的应用，包含详细的算法示例和复杂度分析。

## 📐 基本数据结构

### 点与向量

**点表示**：
```
struct Point {
    double x, y;
    // 二维点
};

struct Point3D {
    double x, y, z;
    // 三维点
};
```

**向量运算**：
- 加法：$\vec{u} + \vec{v} = (u_x + v_x, u_y + v_y)$
- 点积：$\vec{u} \cdot \vec{v} = u_x v_x + u_y v_y = |u||v|\cos\theta$
- 叉积：$\vec{u} \times \vec{v} = u_x v_y - u_y v_x = |u||v|\sin\theta$（二维）

**几何意义**：
- 点积 $> 0$：夹角为锐角
- 点积 $= 0$：垂直
- 叉积 $> 0$：$\vec{v}$ 在 $\vec{u}$ 的逆时针方向

---

### 线段与射线

**线段表示**：
```
struct Segment {
    Point p1, p2;
};
```

**射线表示**：
```
struct Ray {
    Point origin;
    Vector direction;
};
```

**直线方程形式**：
1. **点斜式**：$y - y_0 = k(x - x_0)$
2. **两点式**：$\frac{x - x_1}{x_2 - x_1} = \frac{y - y_1}{y_2 - y_1}$
3. **一般式**：$Ax + By + C = 0$
4. **参数式**：$P = P_0 + t\vec{v}$

---

### 多边形表示

**多边形分类**：
- **简单多边形**：边不自交
- **凸多边形**：内角均 $< 180°$
- **星形多边形**：存在一点可见整个多边形

**表示方法**：
```
struct Polygon {
    vector<Point> vertices;  // 按顺序排列的顶点
    bool isConvex() const;
    double area() const;
    bool contains(Point p) const;
};
```

## 🔍 基本算法

### 凸包算法

**问题**：给定 $n$ 个点，求包含所有点的最小凸多边形。

**Graham扫描算法**（$O(n \log n)$）：

```python
def graham_scan(points):
    # 1. 找到最左下角的点作为起点
    start = min(points, key=lambda p: (p.y, p.x))
    
    # 2. 按极角排序其他点
    sorted_points = sort_by_polar_angle(points, start)
    
    # 3. 构建凸包
    hull = [start, sorted_points[0]]
    for p in sorted_points[1:]:
        while len(hull) >= 2 and not is_left_turn(hull[-2], hull[-1], p):
            hull.pop()
        hull.append(p)
    
    return hull
```

**复杂度分析**：
- 排序：$O(n \log n)$
- 扫描：$O(n)$
- 总复杂度：$O(n \log n)$

**Jarvis行进算法**（$O(nh)$，$h$为凸包顶点数）：

逐个寻找凸包顶点，适合凸包顶点数少的情况。

---

### 线段相交检测

**问题**：判断两条线段是否相交。

**方向判定**：
- $\text{orientation}(p, q, r) = (q-p) \times (r-p)$
- $> 0$：逆时针
- $< 0$：顺时针
- $= 0$：共线

**相交判定算法**：

```python
def segments_intersect(p1, q1, p2, q2):
    o1 = orientation(p1, q1, p2)
    o2 = orientation(p1, q1, q2)
    o3 = orientation(p2, q2, p1)
    o4 = orientation(p2, q2, q1)
    
    # 一般情况
    if o1 != o2 and o3 != o4:
        return True
    
    # 特殊共线情况
    if o1 == 0 and on_segment(p1, p2, q1): return True
    if o2 == 0 and on_segment(p1, q2, q1): return True
    if o3 == 0 and on_segment(p2, p1, q2): return True
    if o4 == 0 and on_segment(p2, q1, q2): return True
    
    return False
```

**复杂度**：$O(1)$

### 点包含测试

**问题**：判断点 $p$ 是否在多边形 $P$ 内。

**射线法**（$O(n)$）：

从点 $p$ 向右发射水平射线，计算与多边形边界的交点数。
- 交点数为奇数 $\Rightarrow$ 点在内部
- 交点数为偶数 $\Rightarrow$ 点在外部

```python
def point_in_polygon(point, polygon):
    x, y = point
    n = len(polygon)
    inside = False
    
    for i in range(n):
        x1, y1 = polygon[i]
        x2, y2 = polygon[(i + 1) % n]
        
        # 检查边是否与水平射线相交
        if ((y1 > y) != (y2 > y)):
            xinters = (x2 - x1) * (y - y1) / (y2 - y1) + x1
            if x < xinters:
                inside = not inside
    
    return inside
```

**绕数法**（更精确）：

计算多边形绕点的总角度，$2\pi$ 的整数倍即为绕数。

## 📊 三角剖分

### Delaunay三角剖分

**定义**：点集 $P$ 的Delaunay三角剖分满足空圆性质：
> 任意三角形的外接圆内不含其他点。

**性质**：
- 最大化最小角（避免狭长三角形）
- 唯一性（一般位置假设下）

**算法**（Bowyer-Watson，$O(n \log n)$）：

```python
def delaunay_triangulation(points):
    # 1. 创建包含所有点的超级三角形
    triangulation = [super_triangle]
    
    for point in points:
        # 2. 找出外接圆包含该点的所有三角形
        bad_triangles = [t for t in triangulation if in_circumcircle(point, t)]
        
        # 3. 找到这些三角形的边界
        boundary = find_boundary(bad_triangles)
        
        # 4. 删除坏三角形，添加新三角形
        for t in bad_triangles:
            triangulation.remove(t)
        for edge in boundary:
            triangulation.append(Triangle(edge.p1, edge.p2, point))
    
    # 5. 移除与超级三角形相关的三角形
    return [t for t in triangulation if not involves_super(t)]
```

### Voronoi图

**定义**：给定点集 $P = \{p_1, ..., p_n\}$，$p_i$ 的Voronoi区域为：
$$V(p_i) = \{x : d(x, p_i) \leq d(x, p_j), \forall j \neq i\}$$

**与Delaunay三角剖分的关系**：
- Delaunay三角剖分的对偶图即为Voronoi图
- Voronoi顶点是Delaunay三角形外接圆心

**应用**：
- 最近邻搜索
- 设施选址问题
- 自然邻插值

---

### 约束三角剖分

**问题**：在存在约束边（如边界、障碍物）的情况下进行三角剖分。

**Constrained Delaunay Triangulation (CDT)**：

放松空圆条件，允许约束边穿过外接圆。

**算法**：
1. 插入所有约束边
2. 恢复Delaunay性质（不删除约束边）
3. 局部优化（边翻转）

## 📈 几何搜索

### 范围搜索

**问题**：报告矩形（或其他范围）内的所有点。

**平面扫描算法**（$O(n \log n + k)$，$k$ 为报告点数）：

```python
def range_search(points, query_rect):
    # 1. 按x坐标排序事件点
    events = sorted([(p.x, 'in', p) for p in points] + 
                   [(query_rect.x2, 'out', p) for p in points])
    
    # 2. 使用平衡BST维护活动区间
    active_set = BalancedBST(key=lambda p: p.y)
    result = []
    
    for x, typ, p in events:
        if typ == 'in' and query_rect.contains_x(p):
            active_set.insert(p)
        elif typ == 'out':
            active_set.remove(p)
        
        # 查询y范围
        if x >= query_rect.x1:
            result.extend(active_set.range(query_rect.y1, query_rect.y2))
    
    return result
```

### 最近邻搜索

**问题**：找到距离查询点最近的点。

**蛮力**：$O(n)$

**kd树优化**（$O(\log n)$ 平均）：

递归搜索，优先访问包含查询点的子树，必要时检查另一子树。

---

### kd树

**定义**：k维空间的二叉空间分割树。

**构建**（$O(n \log n)$）：

```python
def build_kdtree(points, depth=0):
    if not points:
        return None
    
    # 选择分割维度
    axis = depth % k
    
    # 按该维度排序并选取中位数
    points.sort(key=lambda p: p[axis])
    median = len(points) // 2
    
    return Node(
        point=points[median],
        left=build_kdtree(points[:median], depth + 1),
        right=build_kdtree(points[median+1:], depth + 1),
        axis=axis
    )
```

**应用**：
- 最近邻搜索
- 范围搜索
- 点云处理

## 💡 应用示例

### 计算机图形学

**光线追踪**：
- 求光线与三角形的交点
- 使用BVH（包围盒层次）加速

**碰撞检测**：
- 分离轴定理（SAT）
- GJK算法（Gilbert-Johnson-Keerthi）

**示例代码**（光线-三角形相交）：

```python
def ray_triangle_intersect(ray_origin, ray_dir, triangle):
    # Möller-Trumbore算法
    edge1 = triangle.v1 - triangle.v0
    edge2 = triangle.v2 - triangle.v0
    
    h = ray_dir.cross(edge2)
    a = edge1.dot(h)
    
    if abs(a) < EPSILON:
        return None  # 平行
    
    f = 1.0 / a
    s = ray_origin - triangle.v0
    u = f * s.dot(h)
    
    if u < 0 or u > 1:
        return None
    
    q = s.cross(edge1)
    v = f * ray_dir.dot(q)
    
    if v < 0 or u + v > 1:
        return None
    
    t = f * edge2.dot(q)
    return ray_origin + ray_dir * t if t > EPSILON else None
```
