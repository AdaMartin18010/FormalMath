## 📚 概述

双曲几何（Hyperbolic Geometry）是负曲率非欧几何，与欧几里得几何和椭圆几何（球面几何）并称三大经典几何。在双曲几何中，过直线外一点存在无穷多条平行线，三角形内角和小于 $180°$，圆周长大于 $2\pi r$。庞加莱圆盘模型、上半平面模型、克莱因模型等提供了研究双曲几何的不同视角。

本深度版将系统阐述双曲几何的模型理论、核心定理及其与数学家理念的深度关联。

## 🕰️ 历史发展

**欧几里得第五公设问题**：

自欧几里得以来，数学家试图从其他公设证明第五公设（平行公设）。这一努力持续了两千年。

**非欧几何的诞生**：

- **罗巴切夫斯基**（Lobachevsky, 1829）和 **鲍耶**（Bolyai, 1832）独立发现：否定第五公设可得到相容的新几何
- **高斯**私下研究但未发表

**模型的建立**：

- **贝尔特拉米**（Beltrami, 1868）：伪球面模型，证明双曲几何的相容性
- **庞加莱**（Poincaré）：圆盘模型和上半平面模型
- **克莱因**（Klein）：射影模型

**黎曼的统一**：

黎曼几何框架将三种几何统一：曲率 $K > 0$（椭圆）、$K = 0$（欧氏）、$K < 0$（双曲）。

## 🏗️ 双曲几何的公理系统

双曲几何采用与欧几里得几何相同的前四条公设，但用以下公设代替第五公设：

**双曲平行公设**：

给定直线 $l$ 和线外一点 $P$，过 $P$ 至少存在两条直线与 $l$ 不相交（平行）。

**等价表述**：

存在内角和小于两直角（$180°$）的三角形。

## 📐 主要模型详解

### 庞加莱圆盘模型

**定义**：

在单位圆盘 $D = \{z \in \mathbb{C} : |z| < 1\}$ 上定义度量：

$$ds^2 = \frac{4(dx^2 + dy^2)}{(1 - x^2 - y^2)^2}$$

**基本性质**：

- **测地线**：与单位圆正交的圆弧（包括直径）
- **距离公式**：$d(z_1, z_2) = \text{arcosh}\left(1 + \frac{2|z_1 - z_2|^2}{(1-|z_1|^2)(1-|z_2|^2)}\right)$
- **等距群**：Möbius变换 $z \mapsto e^{i\theta}\frac{z-a}{1-\bar{a}z}$，其中 $|a| < 1$

### 庞加莱上半平面模型

**定义**：

在上半平面 $H = \{z \in \mathbb{C} : \text{Im}(z) > 0\}$ 上定义度量：

$$ds^2 = \frac{dx^2 + dy^2}{y^2}$$

**基本性质**：

- **测地线**：垂直于实轴的半圆和垂直直线
- **距离公式**：$d(z_1, z_2) = \text{arcosh}\left(1 + \frac{|z_1 - z_2|^2}{2\text{Im}(z_1)\text{Im}(z_2)}\right)$
- **等距群**：$PSL(2, \mathbb{R})$：$z \mapsto \frac{az+b}{cz+d}$，$ad-bc=1$

### 克莱因模型

**定义**：

在单位圆盘 $K = \{x \in \mathbb{R}^2 : |x| < 1\}$ 上定义度量：

$$ds^2 = \frac{dx^2 + dy^2}{1 - x^2 - y^2} + \frac{(x\,dx + y\,dy)^2}{(1 - x^2 - y^2)^2}$$

**基本性质**：

- **测地线**：欧几里得直线段
- **角度**：与欧几里得角度不同（不保角）
- **优点**：直观性，测地线为直线

### 双曲面模型

**定义**：

在双叶双曲面的上半叶 $H^2 = \{(x,y,z) \in \mathbb{R}^3 : x^2 + y^2 - z^2 = -1, z > 0\}$ 上定义度量（由Minkowski空间诱导）。

**Minkowski内积**：$\langle u, v \rangle = u_1v_1 + u_2v_2 - u_3v_3$

**度量**：$ds^2 = dx^2 + dy^2 - dz^2$（限制在双曲面上）

**基本性质**：

- **测地线**：双曲面与过原点的平面的交线
- **高斯曲率**：$K = -1$（常数）
- **等距群**：$SO^+(2,1)$

## 📐 基本定理的完整证明

### 定理1：双曲三角形内角和定理

**定理**：双曲三角形内角和小于 $\pi$（$180°$）。

**证明**（庞加莱圆盘模型）：

**步骤1**：简化情形

通过等距变换，可将三角形的一个顶点移到原点，一边沿实轴。

**步骤2**：分析角度

在原点处的角度与欧几里得角度相同（因为度量在原点附近接近欧氏度量）。

另外两个顶点在圆盘内部，测地线是与单位圆正交的圆弧。

**步骤3**：比较欧氏三角形

将双曲三角形的三个顶点用欧几里得直线段连接，形成欧氏三角形。

欧氏三角形的内角和为 $\pi$。

**步骤4**：角度比较

在双曲三角形与单位圆的交点处，双曲测地线垂直于边界圆，而欧氏直线不垂直。

通过几何分析可以证明：双曲三角形的内角小于对应欧氏三角形的内角。

因此，双曲三角形的内角和小于欧氏三角形的内角和，即小于 $\pi$。

**证毕**。∎

### 定理2：双曲余弦定理

**定理**：对于边长为 $a, b, c$ 的双曲三角形：

$$\cosh c = \cosh a \cosh b - \sinh a \sinh b \cos \gamma$$

**证明**（双曲面模型）：

**步骤1**：参数化

设三角形顶点为 $A, B, C$ 在双曲面 $x^2 + y^2 - z^2 = -1$ 上。

边长定义为：$a = d(B, C)$ 等，其中距离由Minkowski度量给出。

**步骤2**：Minkowski距离

对于双曲面上的两点 $P, Q$，双曲距离为：

$$d(P, Q) = \text{arcosh}(-\langle P, Q \rangle)$$

因此：$\cosh a = -\langle B, C \rangle$，等等。

**步骤3**：向量关系

利用向量恒等式和Minkowski内积的性质，可以导出：

$$\langle A, B \rangle = \langle A, C \rangle \langle C, B \rangle - \sinh d(A,C) \sinh d(C,B) \cos \gamma$$

转换为双曲函数：

$$-\cosh c = (-\cosh b)(-\cosh a) - \sinh b \sinh a \cos \gamma$$

$$\cosh c = \cosh a \cosh b - \sinh a \sinh b \cos \gamma$$

**证毕**。∎

### 定理3：双曲正弦定理

**定理**：对于双曲三角形：

$$\frac{\sinh a}{\sin \alpha} = \frac{\sinh b}{\sin \beta} = \frac{\sinh c}{\sin \gamma}$$

**证明**：

从双曲余弦定理出发，通过代数变换（类似于球面情形的证明），可以导出正弦定理。

考虑恒等式：

$$\frac{\sin^2 \gamma}{\sinh^2 c} = \frac{1 - \cos^2 \gamma}{\sinh^2 c}$$

利用余弦定理消去 $\cos \gamma$，经过代数运算可得：

$$\frac{\sin^2 \gamma}{\sinh^2 c} = \frac{1 - \cosh^2 a - \cosh^2 b - \cosh^2 c + 2\cosh a \cosh b \cosh c}{\sinh^2 a \sinh^2 b \sinh^2 c}$$

右边关于三边对称，因此：

$$\frac{\sin^2 \alpha}{\sinh^2 a} = \frac{\sin^2 \beta}{\sinh^2 b} = \frac{\sin^2 \gamma}{\sinh^2 c}$$

取正平方根即得正弦定理。

**证毕**。∎

## 💡 数学家理念关联

### 黎曼的内蕴几何

黎曼将双曲几何纳入其几何框架：

> "空间可以具有负的常数曲率。"

双曲几何是 $K = -1$ 的常曲率空间，是黎曼几何的重要特例。

### 庞加莱的模型与自守函数

庞加莱发现双曲几何与复分析、自守函数的深刻联系：

- **Fuchs群**：双曲平面的离散等距群
- **自守函数**：在Fuchs群作用下不变的函数
- **单值化定理**：黎曼曲面的统一化

庞加莱回忆发现Fuchs群的经历：

> "当我踏上脚踏车，突然意识到Fuchs群的变换与双曲几何的变换是相同的。"

### 希尔伯特的相容性证明

希尔伯特使用解析模型证明了双曲几何的相容性：

> "如果欧几里得几何是相容的，那么双曲几何也是相容的。"

贝尔特拉米的伪球面模型给出了双曲几何的欧几里得实现，建立了两种几何的联系。

## 🔗 现代应用

### 复分析和Teichmüller理论

双曲几何是研究黎曼曲面的基本工具。Teichmüller空间参数化了紧致曲面的复结构。

### 三维流形

瑟斯顿的几何化猜想（后由佩雷尔曼证明）指出：三维流形可分解为具有八种几何结构的部分，其中双曲几何是最丰富的一种。

### 狭义相对论

Minkowski空间的几何与双曲几何密切相关。速度空间具有双曲结构（rapidities相加）。

### 网络科学

双曲几何模型用于描述互联网、社交网络等复杂网络的分层结构。

### 机器学习

双曲嵌入用于表示层次化数据，如词嵌入、知识图谱等。
