## 📚 概述

非欧几何是不同于欧几里得几何的公理体系，通过否定或修改欧几里得第五公设（平行公设）而建立。主要分为双曲几何（否定平行公设，过直线外一点有无穷多条平行线）和椭圆几何（否定平行公设，过直线外一点没有平行线）。非欧几何的发现是19世纪数学最重要的突破之一，深刻影响了现代数学和物理学的发展。

---

## 🏛️ 历史背景

### 欧几里得第五公设

**第五公设（平行公设）**：

> 若一直线与两直线相交，且同侧内角之和小于两直角，则两直线延长后必在该侧相交。

**等价形式（Playfair公理）**：

> 过直线外一点，有且仅有一条直线与已知直线平行。

**历史疑难**：

- 公元前300年-1800年：数学家试图从其他公设证明第五公设
- 所有"证明"都隐含地使用了等价假设
- 18世纪：意识到第五公设的独立性

---

### 非欧几何的诞生

**双曲几何**（1820s-1830s）：

| 数学家 | 贡献 | 时间 |
|--------|------|------|
| Gauss | 未发表研究 | 1810s |
| Lobachevsky | 系统发表双曲几何 | 1829 |
| Bolyai | 独立发现 | 1832 |

**椭圆几何**（1854）：

- Riemann在他的著名就职演讲中提出
- 基于球面几何，但将对径点等同

**Klein模型**（1871）：

- 在欧氏空间内构造非欧几何模型
- 证明非欧几何的相对相容性

**Poincaré圆盘模型**（1882）：

- 更直观的非欧几何实现
- 在复分析中有重要应用

## 📐 双曲几何

### 双曲公理系统

**修改后的公理**：将欧几里得第五公设替换为：

> **双曲平行公设**：过直线外一点，存在至少两条直线与已知直线不相交。

**等价表述**：存在一对平行线，它们在一端渐近。

---

### Poincaré圆盘模型

**定义**：双曲平面等同于单位开圆盘 $D = \{z \in \mathbb{C}: |z| < 1\}$

**测地线（直线）**：
- 与边界正交的圆弧（包括直径）

**双曲度量**：
$$ds^2 = \frac{4(dx^2 + dy^2)}{(1 - x^2 - y^2)^2}$$

或复形式：
$$ds = \frac{2|dz|}{1 - |z|^2}$$

**距离公式**：
$$\cosh d(z, w) = 1 + \frac{2|z - w|^2}{(1 - |z|^2)(1 - |w|^2)}$$

**性质**：
- 边界圆 $|z| = 1$ 在无穷远处
- 测地线向边界渐近
- 三角形内角和小于 $\pi$

---

### 上半平面模型

**定义**：双曲平面等同于上半平面 $\mathbb{H} = \{z \in \mathbb{C}: \text{Im}(z) > 0\}$

**测地线**：
- 垂直于实轴的半直线
- 圆心在实轴上的半圆

**双曲度量**：
$$ds^2 = \frac{dx^2 + dy^2}{y^2}$$

**距离公式**：
$$\cosh d(z, w) = 1 + \frac{|z - w|^2}{2\text{Im}(z)\text{Im}(w)}$$

**与圆盘模型的关系**：

通过Cayley变换互变：
$$w = \frac{z - i}{z + i}, \quad z = i\frac{1 + w}{1 - w}$$

---

### Klein模型

**定义**：双曲平面等同于单位开圆盘

**测地线**：欧氏直线段（弦）

**度量**（非共形）：
$$\cosh d(O, P) = \frac{1}{\sqrt{1 - r^2}}$$

其中 $r = |OP|$。

**特点**：
- 直线是欧氏直线（直观）
- 但角度失真（非共形模型）

---

## 🔺 椭圆几何

### 椭圆公理系统

**修改后的公理**：将欧几里得第五公设替换为：

> **椭圆平行公设**：过直线外一点，不存在与已知直线不相交的直线。

**推论**：任意两条直线必相交。

---

### 球面模型

**定义**：单位球面 $S^2 = \{(x, y, z) \in \mathbb{R}^3: x^2 + y^2 + z^2 = 1\}$

**测地线**：大圆（球与过原点的平面的交线）

**度量**：球面度量
$$ds^2 = d\theta^2 + \sin^2\theta \, d\phi^2$$

**距离公式**：
$$d(P, Q) = \arccos(\langle P, Q \rangle)$$

**性质**：
- 测地线闭合（大圆周长为 $2\pi$）
- 不存在平行线
- 三角形内角和大于 $\pi$

---

### 射影平面模型

**定义**：将 $S^2$ 上对径点等同，得到实射影平面 $\mathbb{R}P^2$

**测地线**：射影直线（$\mathbb{R}P^2$ 中的直线）

**与球面模型的区别**：
- 球面：每条测地线有无穷多对径点
- 射影平面：测地线不自交，每条长 $\pi$

---

## 📏 非欧度量

### 双曲度量

**统一形式**：

在二维，常曲率 $K = -1$ 的度量：
$$ds^2 = dr^2 + \sinh^2 r \, d\theta^2$$

其中 $r$ 为到原点的双曲距离。

**体积元**：
$$dV = \sinh r \, dr \, d\theta$$

**圆周长与圆面积**：

- 半径为 $r$ 的圆：周长 $C = 2\pi \sinh r$，面积 $A = 2\pi(\cosh r - 1)$
- 当 $r \to \infty$，指数增长（与欧氏几何的线性增长不同）

---

### 椭圆度量

**统一形式**：

常曲率 $K = 1$ 的度量：
$$ds^2 = dr^2 + \sin^2 r \, d\theta^2$$

**体积元**：
$$dV = \sin r \, dr \, d\theta$$

**圆周长与圆面积**：

- 半径为 $r$ 的圆：周长 $C = 2\pi \sin r$，面积 $A = 2\pi(1 - \cos r)$
- 当 $r \to \pi$，周长和面积趋于0（拓扑闭合）

---

### 曲率与几何

| 曲率 $K$ | 几何类型 | 平行线 | 三角形内角和 | 圆周长 |
|----------|----------|--------|--------------|--------|
| $K > 0$ | 椭圆几何 | 无 | $> \pi$ | $< 2\pi r$ |
| $K = 0$ | 欧氏几何 | 唯一 | $= \pi$ | $= 2\pi r$ |
| $K < 0$ | 双曲几何 | 无穷多 | $< \pi$ | $> 2\pi r$ |

---

## 📊 三角形性质

### 双曲三角形

**性质**：
- 内角和 $\alpha + \beta + \gamma < \pi$
- 角亏 $\delta = \pi - (\alpha + \beta + \gamma) > 0$

**边长关系**（双曲余弦定理）：
$$\cosh c = \cosh a \cosh b - \sinh a \sinh b \cos \gamma$$

**面积公式**：
$$\text{Area} = \pi - (\alpha + \beta + \gamma) = \delta$$

即面积等于角亏！

---

### 椭圆三角形

**性质**：
- 内角和 $\alpha + \beta + \gamma > \pi$
- 角盈 $\varepsilon = (\alpha + \beta + \gamma) - \pi > 0$

**边长关系**（球面余弦定理）：
$$\cos c = \cos a \cos b + \sin a \sin b \cos \gamma$$

**面积公式**：
$$\text{Area} = (\alpha + \beta + \gamma) - \pi = \varepsilon$$

即面积等于角盈！

---

### 面积公式

**统一表述**：

$$\text{Area} = |K|^{-1} \cdot |\text{角亏/角盈}|$$

其中 $K$ 为高斯曲率。

---

## 🔄 等距变换

### 双曲等距

**Poincaré圆盘模型的等距**：

**Möbius变换**：
$$f(z) = e^{i\theta} \frac{z - a}{1 - \bar{a}z}, \quad |a| < 1$$

**性质**：
- 保向等距群同构于 $PSU(1, 1) \cong PSL(2, \mathbb{R})$
- 所有等距由反射生成

**上半平面模型的等距**：
$$f(z) = \frac{az + b}{cz + d}, \quad a, b, c, d \in \mathbb{R}, \quad ad - bc = 1$$

即 $PSL(2, \mathbb{R})$。

---

### 椭圆等距

**球面模型的等距**：

旋转群 $SO(3)$。

**射影平面的等距**：

$SO(3)$ 作用在对径点等同的球面上。

---

### 离散群

**Fuchs群**：双曲平面上的离散等距群。

**Klein群**：扩充复平面上的离散Möbius变换群。

**应用**：
- 双曲曲面的分类
- 自守函数理论
- 三维流形几何化

---

## 🧠 统一理论

### 黎曼几何观点

**黎曼度量**：

在二维，度量 $ds^2 = E du^2 + 2F du dv + G dv^2$ 的高斯曲率：

$$K = \frac{LN - M^2}{EG - F^2}$$

其中 $L, M, N$ 为第二基本形式系数。

**非欧几何的统一定义**：
- 常曲率 $K < 0$：双曲几何
- 常曲率 $K = 0$：欧氏几何
- 常曲率 $K > 0$：椭圆几何

---

### 常曲率空间

**完备单连通常曲率空间**（二维）：

| 曲率 | 空间 | 万有覆盖 |
|------|------|----------|
| $K < 0$ | 双曲平面 $\mathbb{H}^2$ | 自身 |
| $K = 0$ | 欧氏平面 $\mathbb{E}^2$ | 自身 |
| $K > 0$ | 球面 $S^2$ | 自身 |

**高维推广**：
- 常曲率 $-1$：双曲空间 $\mathbb{H}^n$
- 常曲率 $0$：欧氏空间 $\mathbb{E}^n$
- 常曲率 $1$：球面 $S^n$ / 射影空间 $\mathbb{R}P^n$

---

### Gauss-Bonnet定理

**定理**（二维情形）：

设 $M$ 为紧定向二维黎曼流形，则：
$$\int_M K \, dA = 2\pi \chi(M)$$

其中 $\chi(M)$ 为Euler示性数。

**推论**：

- 球面 $S^2$：$\chi = 2$，$\int K dA = 4\pi$
- 环面 $T^2$：$\chi = 0$，$K = 0$（平坦）
- 高亏格曲面：$\chi < 0$，双曲度量

**非欧几何的意义**：

双曲几何为亏格 $\geq 2$ 的曲面提供了自然度量。

## 💻 形式化实现

```lean
import geometry.euclidean.basic
import geometry.hyperbolic.basic

namespace non_euclidean_geometry

-- 双曲平面（上半平面模型）
def hyperbolic_plane : Type := {z : ℂ // z.im > 0}

-- 双曲度量
def hyperbolic_metric (z w : hyperbolic_plane) : ℝ :=
  real.arccosh (1 + (z.val - w.val).norm_sq / (2 * z.val.im * w.val.im))

-- 双曲等距（PSL(2,ℝ)）
structure psl2r :=
  (a b c d : ℝ)
  (det : a * d - b * c = 1)

def psl2r.action (g : psl2r) (z : hyperbolic_plane) : hyperbolic_plane :=
⟨(g.a * z.val + g.b) / (g.c * z.val + g.d), by field_simp [g.det]; linarith [z.property]⟩

-- 双曲三角形的面积定理
theorem hyperbolic_triangle_area (α β γ : ℝ) (h : α + β + γ < real.pi) :
  ∃ T : set hyperbolic_plane, 
    hyperbolic_triangle T ∧ 
    hyperbolic_area T = real.pi - (α + β + γ) :=
sorry

-- 椭圆几何（球面模型）
def spherical_distance (p q : metric.sphere (0 : euclidean_space ℝ (fin 3)) 1) : ℝ :=
  real.arccos (inner p q)

-- 球面三角形的面积
theorem spherical_triangle_area (α β γ : ℝ) (h : α + β + γ > real.pi) (h' : α + β + γ < 3 * real.pi) :
  ∃ T : set (metric.sphere (0 : euclidean_space ℝ (fin 3)) 1),
    spherical_triangle T ∧
    spherical_area T = (α + β + γ) - real.pi :=
sorry

end non_euclidean_geometry
```
