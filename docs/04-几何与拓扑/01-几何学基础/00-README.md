## 1. 引言与定义

### 1.1 什么是几何学？

**几何学**（Geometry，源自希腊语 "geo" 大地 + "metron" 测量）是数学的一个基础分支，研究空间、形状、大小、位置以及图形性质的科学。几何学提供了描述物理世界空间结构的数学语言。

### 1.2 几何学的核心概念

| 概念 | 定义 | 符号表示 |
|------|------|----------|
| **点** | 空间中无大小的基本元素 | $P, Q, A, B$ |
| **线** | 由点组成的一维集合 | $l, m, \overleftrightarrow{AB}$ |
| **面** | 由线组成的二维集合 | $\alpha, \beta, \pi$ |
| **空间** | 所有点的三维集合 | $\mathbb{R}^3, \mathbb{E}^3$ |
| **距离** | 两点间的度量 | $d(P, Q)$ 或 $\|PQ\|$ |
| **角度** | 两条射线之间的度量 | $\angle ABC$ |

## 2. 几何学的主要分支

### 2.1 欧几里得几何 (Euclidean Geometry)

**定义**: 基于欧几里得《几何原本》的公理系统，研究平直空间中的几何性质。

**核心公理**:
1. **关联公理**: 两点确定唯一直线
2. **顺序公理**: 线上的点有顺序关系
3. **合同公理**: 图形的全等关系
4. **平行公理**（第五公设）: 过直线外一点有且只有一条平行线

**定理示例**: 

**定理 2.1** (勾股定理): 在直角三角形中，斜边的平方等于两直角边的平方和。

$$c^2 = a^2 + b^2$$

**证明**: 
构造边长为 $a+b$ 的正方形，内部放置四个全等的直角三角形，剩余区域形成边长为 $c$ 的正方形。
$$(a+b)^2 = 4 \cdot \frac{1}{2}ab + c^2$$
$$a^2 + 2ab + b^2 = 2ab + c^2$$
$$\therefore a^2 + b^2 = c^2 \quad \square$$

### 2.2 非欧几何 (Non-Euclidean Geometry)

**定义**: 否定欧几里得平行公理而建立的几何体系。

**类型**:
- **双曲几何** (Hyperbolic Geometry): 过直线外一点存在多条平行线，曲率 $< 0$
- **椭圆几何** (Elliptic Geometry): 不存在平行线，曲率 $> 0$

**曲率关系**:

$$K = \begin{cases} 
0 & \text{欧几里得几何} \\
< 0 & \text{双曲几何} \\
> 0 & \text{椭圆几何}
\end{cases}$$

### 2.3 射影几何 (Projective Geometry)

**定义**: 研究图形在射影变换下不变性质的几何学。

**核心定理**:

**定理 2.2** (德萨格定理): 如果两个三角形对应顶点的连线共点，则它们对应边的交点共线。

### 2.4 微分几何 (Differential Geometry)

**定义**: 运用微积分和微分方程研究曲线、曲面和流形的几何学。

**基本概念**:
- **曲线**: 光滑映射 $\gamma: I \to \mathbb{R}^n$
- **曲率**: $\kappa = \|\gamma''(t)\|$ （曲线的弯曲程度）
- **曲面**: 参数化映射 $\mathbf{r}(u,v): D \to \mathbb{R}^3$
- **高斯曲率**: $K = \kappa_1 \cdot \kappa_2$ （主曲率的乘积）

**高斯-博内定理**:

$$\int_M K \, dA + \int_{\partial M} k_g \, ds = 2\pi \chi(M)$$

其中 $\chi(M)$ 是流形 $M$ 的欧拉示性数。

## 3. 几何学的代数化

### 3.1 解析几何

**定义**: 通过坐标系将几何问题转化为代数问题。

**核心对应**:

| 几何对象 | 代数表示 |
|----------|----------|
| 点 | $(x, y, z) \in \mathbb{R}^3$ |
| 直线 | $ax + by + c = 0$ |
| 平面 | $ax + by + cz + d = 0$ |
| 圆 | $(x-a)^2 + (y-b)^2 = r^2$ |
| 球面 | $(x-a)^2 + (y-b)^2 + (z-c)^2 = r^2$ |

### 3.2 向量几何

**向量运算**:
- **加法**: $\mathbf{a} + \mathbf{b} = (a_1+b_1, a_2+b_2, a_3+b_3)$
- **点积**: $\mathbf{a} \cdot \mathbf{b} = |\mathbf{a}||\mathbf{b}|\cos\theta = a_1b_1 + a_2b_2 + a_3b_3$
- **叉积**: $|\mathbf{a} \times \mathbf{b}| = |\mathbf{a}||\mathbf{b}|\sin\theta$

**应用示例**:
计算三角形的面积：
$$S = \frac{1}{2}|\overrightarrow{AB} \times \overrightarrow{AC}|$$
