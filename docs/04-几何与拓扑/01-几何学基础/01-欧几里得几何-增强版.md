## 📚 概述

欧几里得几何是历史上第一个公理化数学体系，由古希腊数学家欧几里得在《几何原本》中系统阐述。它基于五条公设和五条公理，通过逻辑演绎建立完整的平面和立体几何理论。本增强版文档包含详细的几何示例和经典问题解析。

---

## 📐 基本公理

### 五条公设

1. **连接公设**：任意两点可以确定一条直线
2. **延长公设**：直线可以无限延长
3. **圆公设**：以任意点为中心、任意长为半径可以作圆
4. **直角公设**：所有直角都相等
5. **平行公设**：过直线外一点，有且仅有一条直线与已知直线平行

### 五条公理

1. **等于同量的量相等**
2. **等量加等量，和相等**
3. **等量减等量，差相等**
4. **互相重合的量相等**
5. **整体大于部分**

## 🔗 平面几何

### 三角形

**基本元素**：
- 三边：$a, b, c$
- 三角：$A, B, C$
- 三高、三中线的交点：垂心、重心

**重要定理**：

| 定理 | 内容 |
|------|------|
| 内角和 | $A + B + C = 180°$ |
| 外角定理 | 外角等于不相邻两内角之和 |
| 中位线 | 平行于第三边且等于其一半 |
| 重心 | 中线交点，分中线为2:1 |
| 垂心 | 三高交点 |
| 外心 | 三边垂直平分线交点 |
| 内心 | 三角平分线交点，内切圆心 |

**示例**：证明三角形内角和为180°

过顶点作平行于对边的直线，利用内错角相等可证。

---

### 四边形

| 类型 | 性质 |
|------|------|
| 平行四边形 | 对边平行且相等，对角相等，对角线互相平分 |
| 矩形 | 平行四边形 + 一个直角 |
| 菱形 | 平行四边形 + 邻边相等 |
| 正方形 | 矩形 + 菱形 |
| 梯形 | 只有一对对边平行 |

**面积公式**：
- 平行四边形：$S = bh$（底×高）
- 梯形：$S = \frac{1}{2}(a+b)h$（两底和的一半×高）

---

### 圆的性质

**基本定理**：

1. **圆周角定理**：同弧所对圆周角等于圆心角的一半
2. **弦切角定理**：弦切角等于所夹弧对的圆周角
3. **相交弦定理**：$PA \cdot PB = PC \cdot PD$
4. **切割线定理**：$PT^2 = PA \cdot PB$

**示例**：证明半圆上的圆周角为直角

设 $AB$ 为直径，$C$ 在圆上。圆心角 $AOB = 180°$，故圆周角 $ACB = 90°$。

## 📊 相似与全等

### 全等判定

| 判定 | 条件 |
|------|------|
| SSS | 三边对应相等 |
| SAS | 两边及其夹角对应相等 |
| ASA | 两角及其夹边对应相等 |
| AAS | 两角及其中一角的对边对应相等 |
| HL | 直角三角形斜边和一条直角边对应相等 |

---

### 相似判定

| 判定 | 条件 |
|------|------|
| SSS | 三边对应成比例 |
| SAS | 两边对应成比例且夹角相等 |
| AA | 两角对应相等 |

**相似比性质**：
- 对应线段比等于相似比
- 面积比等于相似比的平方

## 📈 度量关系

### 勾股定理

**定理**：直角三角形中，$a^2 + b^2 = c^2$

**证明**（面积法）：

四个全等直角三角形拼成边长为 $(a+b)$ 的大正方形，中间小正方形边长为 $c$。

$$(a+b)^2 = 4 \cdot \frac{1}{2}ab + c^2$$
$$a^2 + 2ab + b^2 = 2ab + c^2$$
$$a^2 + b^2 = c^2$$

**逆定理**：若 $a^2 + b^2 = c^2$，则以 $a, b, c$ 为边的三角形为直角三角形。

---

### 三角函数

**定义**（直角三角形）：

$$\sin A = \frac{a}{c}, \quad \cos A = \frac{b}{c}, \quad \tan A = \frac{a}{b}$$

**基本恒等式**：
- $\sin^2 A + \cos^2 A = 1$
- $\tan A = \frac{\sin A}{\cos A}$

**正弦定理**：
$$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$$

**余弦定理**：
$$c^2 = a^2 + b^2 - 2ab\cos C$$

---

### 面积公式

| 图形 | 面积公式 |
|------|----------|
| 三角形 | $S = \frac{1}{2}bh = \frac{1}{2}ab\sin C = \sqrt{s(s-a)(s-b)(s-c)}$（Heron） |
| 矩形 | $S = lw$ |
| 圆 | $S = \pi r^2$ |
| 扇形 | $S = \frac{1}{2}r^2\theta$（$\theta$为弧度） |

## 💡 经典例题

### 几何证明题

**例题1**：证明等腰三角形两底角相等

**证明**：
设 $\triangle ABC$ 中 $AB = AC$。

连接顶点 $A$ 与底边中点 $D$。

在 $\triangle ABD$ 和 $\triangle ACD$ 中：
- $AB = AC$（已知）
- $BD = CD$（$D$为中点）
- $AD = AD$（公共边）

由 SSS，$\triangle ABD \cong \triangle ACD$，故 $\angle B = \angle C$。

---

### 作图问题

**例题2**：过圆外一点作圆的切线

**作法**：
1. 连接点 $P$ 与圆心 $O$
2. 以 $OP$ 为直径作圆，与原圆交于 $A, B$
3. $PA, PB$ 即为所求切线

**证明**：$\angle OAP = 90°$（半圆上的圆周角），故 $PA \perp OA$，$PA$ 为切线。

---

### 计算问题

**例题3**：已知 $\triangle ABC$ 中 $a=5$, $b=7$, $c=8$，求面积

**解**：

用Heron公式，$s = \frac{5+7+8}{2} = 10$

$$S = \sqrt{10(10-5)(10-7)(10-8)} = \sqrt{10 \cdot 5 \cdot 3 \cdot 2} = \sqrt{300} = 10\sqrt{3}$$

---

**例题4**：证明正五边形的对角线与边长之比为黄金比例

**解**：

设边长为1，对角线长为 $\phi$。

利用相似三角形可证：$\phi = 1 + \frac{1}{\phi}$

解得：$\phi = \frac{1+\sqrt{5}}{2} \approx 1.618$
