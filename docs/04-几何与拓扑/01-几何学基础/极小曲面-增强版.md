## 1. 引言

极小曲面是平均曲率处处为零的曲面。这一概念起源于18世纪关于肥皂膜物理性质的观察：肥皂膜在表面张力作用下自动调整形状以最小化表面积。

**历史发展**:

| 年份 | 发展 | 贡献者 |
|------|------|--------|
| 1760 | 极小曲面的变分法基础 | Lagrange |
| 1776 | 悬链面的发现 | Meusnier |
| 1835 | 螺旋面的发现 | Scherk |
| 1866 | Weierstrass表示 | Weierstrass, Enneper |
| 1930 | Plateau问题的解 | Douglas, Radó |
| 1976 | 周期极小曲面的系统研究 | Schoen |
| 1982 | 嵌入极小曲面的构造 | Costa, Hoffman, Meeks |

**物理背景**:

肥皂膜的表面积泛函：

$$A(\Sigma) = \int_\Sigma dA$$

极小曲面是这一泛函的临界点。

## 2. 平均曲率

### 2.1 定义与几何意义

**定义 2.1** (平均曲率)

设 $\Sigma \subset \mathbb{R}^3$ 是正则曲面，$p \in \Sigma$。主曲率 $\kappa_1, \kappa_2$ 是曲面的最大和最小法曲率。平均曲率定义为：

$$H = \frac{\kappa_1 + \kappa_2}{2}$$

**等价的计算公式**:

设曲面的第一基本形式为 $E du^2 + 2F du dv + G dv^2$，第二基本形式为 $L du^2 + 2M du dv + N dv^2$，则：

$$H = \frac{EN - 2FM + GL}{2(EG - F^2)}$$

**定义 2.2** (极小曲面)

曲面 $\Sigma$ 称为极小曲面，如果其平均曲率处处为零：

$$H \equiv 0$$

等价地，$\kappa_1 + \kappa_2 = 0$，即主曲率互为相反数。

**几何意义**: 在每一点，曲面在两个正交方向上的弯曲程度相等但方向相反。这种"平衡"使得曲面具有最小化面积的变分性质。

### 2.2 极小曲面的变分特征

**定理 2.1** (极小曲面的变分特征)

曲面 $\Sigma$ 是极小曲面当且仅当它是面积泛函的一阶变分临界点。

**证明**:

设 $\Sigma$ 是参数曲面 $X: D \to \mathbb{R}^3$，考虑法向变分：

$$X_\epsilon(u, v) = X(u, v) + \epsilon f(u, v) N(u, v)$$

其中 $f$ 是紧支光滑函数，$N$ 是单位法向量。

**步骤 1**: 计算第一变分

面积泛函：

$$A(\Sigma_\epsilon) = \int_D |X_{\epsilon,u} \times X_{\epsilon,v}| du dv$$

计算变分后的切向量：

$$X_{\epsilon,u} = X_u + \epsilon(f_u N + f N_u)$$
$$X_{\epsilon,v} = X_v + \epsilon(f_v N + f N_v)$$

**步骤 2**: 计算法向量的变化

由Weingarten方程：

$$N_u = -S(X_u), \quad N_v = -S(X_v)$$

其中 $S$ 是形状算子。

**步骤 3**: 计算面积的一阶变分

展开到一阶：

$$\left.\frac{d}{d\epsilon}\right|_{\epsilon=0} A(\Sigma_\epsilon) = -2\int_D f H |X_u \times X_v| du dv$$

**步骤 4**: 结论

由于 $f$ 是任意的，一阶变分为零当且仅当 $H \equiv 0$。

**证毕**。

**定理 2.2** (面积的二阶变分)

对于极小曲面，面积的二阶变分为：

$$\left.\frac{d^2}{d\epsilon^2}\right|_{\epsilon=0} A(\Sigma_\epsilon) = \int_\Sigma \left(|\nabla f|^2 - (\kappa_1^2 + \kappa_2^2)f^2\right) dA$$

由于 $\kappa_1 = -\kappa_2$（在极小曲面上），有：

$$\left.\frac{d^2}{d\epsilon^2}\right|_{\epsilon=0} A(\Sigma_\epsilon) = \int_\Sigma \left(|\nabla f|^2 - 2K f^2\right) dA$$

其中 $K = \kappa_1 \kappa_2$ 是高斯曲率。

## 3. Plateau问题

### 3.1 问题陈述

**Plateau问题**:

给定 $\mathbb{R}^3$ 中的简单闭曲线 $\Gamma$，是否存在以 $\Gamma$ 为边界的极小曲面？

这个问题以比利时物理学家Joseph Plateau命名，他在19世纪系统研究了肥皂膜的物理性质。

### 3.2 解的存在性

**定理 3.1** (Douglas, 1931; Radó, 1930)

对于任意可求长的简单闭曲线 $\Gamma \subset \mathbb{R}^3$，存在以 $\Gamma$ 为边界的极小圆盘。

**证明概要** (Douglas方法):

**步骤 1**: 参数化问题

寻找调和映射 $X: D \to \mathbb{R}^3$，其中 $D$ 是单位圆盘，使得：
- $\Delta X = 0$（调和性）
- $|X_u|^2 - |X_v|^2 = 0$，$X_u \cdot X_v = 0$（共形性）
- $X|_{\partial D}$ 是 $\Gamma$ 的单调参数化

**步骤 2**: Dirichlet泛函

Douglas考虑Dirichlet泛函而非面积泛函：

$$D(X) = \frac{1}{2} \int_D (|X_u|^2 + |X_v|^2) du dv$$

对于共形参数化，$D(X) = A(X)$。

**步骤 3**: 极小化序列

取极小化序列 $\{X_n\}$，使得 $D(X_n) \to \inf D$。

**步骤 4**: 紧性论证

利用Courant-Lebesgue引理证明序列有紧性。

**步骤 5**: 极限是解

证明极限映射满足调和性和共形性条件。

## 4. Weierstrass表示

### 4.1 Weierstrass-Enneper表示

**定理 4.1** (Weierstrass-Enneper表示)

设 $f$ 是单位圆盘 $D$ 上的全纯函数，$g$ 是 $D$ 上的亚纯函数，满足 $fg^2$ 全纯。则：

$$X(\zeta) = \text{Re} \int_{\zeta_0}^\zeta \left(\frac{1}{2}f(1-g^2), \frac{i}{2}f(1+g^2), fg\right) d\zeta$$

定义一个极小曲面。

反之，任意非平面的极小曲面都可以这样表示。

**证明**:

**步骤 1**: 验证极小曲面条件

设 $X = (X_1, X_2, X_3)$，计算：

$$\phi_1 = \frac{1}{2}f(1-g^2), \quad \phi_2 = \frac{i}{2}f(1+g^2), \quad \phi_3 = fg$$

这些是满足 $\phi_1^2 + \phi_2^2 + \phi_3^2 = 0$ 的全纯函数（称为Enneper-Weierstrass数据）。

**步骤 2**: 验证共形性

由于 $\phi_1^2 + \phi_2^2 + \phi_3^2 = 0$，有：

$$|X_u|^2 - |X_v|^2 = 0, \quad X_u \cdot X_v = 0$$

因此参数化是共形的。

**步骤 3**: 验证极小性

对于共形参数化，平均曲率为零等价于各分量是调和函数。

由于 $X_j = \text{Re} \int \phi_j$ 是全纯函数的实部，因此是调和的。

**证毕**。

### 4.2 具体例子

**例子 4.1** (悬链面 / Catenoid)

取 $f(\zeta) = -e^{-\zeta}$，$g(\zeta) = -e^{\zeta}$，则：

$$X(u, v) = (\cosh u \cos v, \cosh u \sin v, u)$$

这是旋转极小曲面，唯一完备嵌入的旋转极小曲面。

**例子 4.2** (螺旋面 / Helicoid)

取 $f(\zeta) = -ie^{-i\zeta}$，$g(\zeta) = e^{i\zeta}$，则：

$$X(u, v) = (\sinh u \cos v, \sinh u \sin v, v)$$

这是螺旋形的极小曲面，是唯一的直纹极小曲面（除平面外）。

**例子 4.3** (Enneper曲面)

取 $f(\zeta) = 1$，$g(\zeta) = \zeta$，则：

$$X(u, v) = \left(u - \frac{u^3}{3} + uv^2, -v + \frac{v^3}{3} - u^2v, u^2 - v^2\right)$$

这是代数极小曲面，自相交。

## 5. 极小曲面的性质与例子

**定理 5.1** (Bernstein定理)

设 $\Sigma \subset \mathbb{R}^3$ 是定义在整个平面上的极小图 $z = f(x, y)$，则 $\Sigma$ 是平面。

**注**: 这是极小曲面唯一的Liouville型定理。

**定理 5.2** (Osserman完备性定理)

完备极小曲面要么是平面，要么其Gauss映射取几乎所有球面上的点（至多遗漏4个点）。

**经典极小曲面表**:

| 曲面 | 发现者 | 年份 | 特点 |
|------|--------|------|------|
| 悬链面 | Meusnier | 1776 | 旋转极小曲面 |
| 螺旋面 | Meusnier | 1776 | 直纹极小曲面 |
| Scherk曲面 | Scherk | 1835 | 周期极小曲面 |
| Enneper曲面 | Enneper | 1864 | 代数极小曲面 |
| Costa曲面 | Costa | 1982 | 嵌入极小曲面 |
| Gyroid | Schoen | 1970 | 三重周期极小曲面 |

## 6. 现代发展

**嵌入极小曲面理论**:

在1980年代以前，唯一已知的完备嵌入极小曲面是平面、悬链面和螺旋面。

Costa (1982) 发现了新的完备嵌入极小曲面，Hoffman和Meeks (1985) 证明它是嵌入的。

**三重周期极小曲面 (TPMS)**:

在材料科学和晶体学中有重要应用，如Gyroid、P-surface、D-surface等。

**极小曲面的应用**:

1. **建筑学**: 帐篷结构、膜结构的最优设计
2. **材料科学**: 纳米多孔材料的设计
3. **生物学**: 细胞膜、昆虫翅膀结构
4. **数学物理**: 弦理论中的世界面
