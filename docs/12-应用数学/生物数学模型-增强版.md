## 1. 种群动力学

### 1.1 单种群模型

**指数增长模型**：

$$\frac{dN}{dt} = rN$$

解：$N(t) = N_0 e^{rt}$

**Logistic增长模型**：

$$\frac{dN}{dt} = rN\left(1 - \frac{N}{K}\right)$$

其中 $r$ 是内禀增长率，$K$ 是环境容纳量。

**解析解**：

$$N(t) = \frac{K}{1 + \left(\frac{K-N_0}{N_0}\right)e^{-rt}}$$

**Allee效应模型**：

$$\frac{dN}{dt} = rN\left(\frac{N}{A} - 1\right)\left(1 - \frac{N}{K}\right)$$

其中 $A$ 是Allee阈值，当 $N < A$ 时种群趋于灭绝。

### 1.2 捕食者-猎物模型

**Lotka-Volterra模型**：

$$\frac{dX}{dt} = \alpha X - \beta XY$$
$$\frac{dY}{dt} = \delta XY - \gamma Y$$

其中：
- $X$：猎物数量
- $Y$：捕食者数量
- $\alpha$：猎物增长率
- $\beta$：捕食率
- $\delta$：捕食者转化效率
- $\gamma$：捕食者死亡率

**平衡点分析**：

1. 平凡平衡点：$(0, 0)$ - 不稳定
2. 非平凡平衡点：$(\frac{\gamma}{\delta}, \frac{\alpha}{\beta})$ - 中心点

**带承载能力的修正模型**：

$$\frac{dX}{dt} = \alpha X\left(1 - \frac{X}{K}\right) - \beta XY$$
$$\frac{dY}{dt} = \delta XY - \gamma Y$$

### 1.3 竞争与共生模型

**竞争模型**（Lotka-Volterra竞争）：

$$\frac{dN_1}{dt} = r_1N_1\left(1 - \frac{N_1 + \alpha_{12}N_2}{K_1}\right)$$
$$\frac{dN_2}{dt} = r_2N_2\left(1 - \frac{N_2 + \alpha_{21}N_1}{K_2}\right)$$

**共生模型**：

$$\frac{dN_1}{dt} = r_1N_1\left(1 - \frac{N_1}{K_1} + \frac{\alpha_{12}N_2}{K_1}\right)$$
$$\frac{dN_2}{dt} = r_2N_2\left(1 - \frac{N_2}{K_2} + \frac{\alpha_{21}N_1}{K_2}\right)$$

## 2. 流行病模型

### 2.1 SIR模型

**基本模型**：

$$\frac{dS}{dt} = -\beta SI$$
$$\frac{dI}{dt} = \beta SI - \gamma I$$
$$\frac{dR}{dt} = \gamma I$$

其中：
- $S$：易感者比例
- $I$：感染者比例
- $R$：康复者比例
- $\beta$：感染率
- $\gamma$：恢复率

**基本再生数**：

$$R_0 = \frac{\beta S_0}{\gamma}$$

**阈值定理**：
- $R_0 > 1$：疫情爆发
- $R_0 < 1$：疫情消退

**最终规模方程**：

$$\ln\frac{S_0}{S_\infty} = R_0\left(1 - \frac{S_\infty}{N}\right)$$

### 2.2 SEIR模型

增加潜伏者 compartment：

$$\frac{dS}{dt} = -\beta SI$$
$$\frac{dE}{dt} = \beta SI - \sigma E$$
$$\frac{dI}{dt} = \sigma E - \gamma I$$
$$\frac{dR}{dt} = \gamma I$$

其中 $\sigma$ 是潜伏期到感染期的转化率。

**潜伏期影响**：
- 增加系统时滞
- 改变疫情动力学
- 影响控制策略效果

### 2.3 网络传播模型

**SIS网络模型**：

$$\frac{dI_i}{dt} = -\delta I_i + \beta (1-I_i)\sum_j A_{ij}I_j$$

其中 $A_{ij}$ 是网络的邻接矩阵。

**异质性网络**：
- 无标度网络：存在超级传播者
- 小世界网络：快速全局传播

## 3. 生物化学系统

### 3.1 酶动力学

**Michaelis-Menten动力学**：

$$E + S \underset{k_{-1}}{\stackrel{k_1}{\rightleftharpoons}} ES \stackrel{k_2}{\rightarrow} E + P$$

**反应速率**：

$$v = \frac{d[P]}{dt} = \frac{V_{max}[S]}{K_m + [S]}$$

其中：
- $V_{max} = k_2[E]_0$
- $K_m = \frac{k_{-1} + k_2}{k_1}$

**Lineweaver-Burk图**：

$$\frac{1}{v} = \frac{K_m}{V_{max}} \cdot \frac{1}{[S]} + \frac{1}{V_{max}}$$

**抑制类型**：
1. 竞争性抑制：$V_{max}$ 不变，$K_m$ 增大
2. 非竞争性抑制：$V_{max}$ 减小，$K_m$ 不变
3. 反竞争性抑制：$V_{max}$ 和 $K_m$ 都减小

### 3.2 基因调控网络

**转录调控模型**：

$$\frac{dm}{dt} = \alpha_m \cdot f(p) - \delta_m m$$
$$\frac{dp}{dt} = \alpha_p m - \delta_p p$$

其中 $f(p)$ 是调控函数：

- 激活：$f(p) = \frac{p^n}{K^n + p^n}$
- 抑制：$f(p) = \frac{K^n}{K^n + p^n}$

**Toggle Switch**：

两个相互抑制的基因：

$$\frac{dp_1}{dt} = \frac{\alpha_1}{1 + p_2^n} - p_1$$
$$\frac{dp_2}{dt} = \frac{\alpha_2}{1 + p_1^n} - p_2$$

双稳态系统，可作为生物记忆单元。

## 4. 神经网络模型

**Hodgkin-Huxley模型**：

$$C_m\frac{dV}{dt} = I_{ext} - \bar{g}_{Na}m^3h(V-E_{Na}) - \bar{g}_Kn^4(V-E_K) - g_L(V-E_L)$$

门控变量动力学：
$$\frac{dm}{dt} = \alpha_m(V)(1-m) - \beta_m(V)m$$

**FitzHugh-Nagumo模型**（简化）：

$$\frac{dv}{dt} = v - \frac{v^3}{3} - w + I$$
$$\tau\frac{dw}{dt} = v + a - bw$$

## 5. 计算示例

### 5.1 种群动态模拟

**Logistic增长模拟**：

```python
def logistic_growth(r, K, N0, t_max, dt=0.01):
    """Logistic增长数值模拟"""
    t = np.arange(0, t_max, dt)
    N = np.zeros_like(t)
    N[0] = N0
    
    for i in range(1, len(t)):
        dN = r * N[i-1] * (1 - N[i-1]/K) * dt
        N[i] = N[i-1] + dN
    
    return t, N

# 参数设置
r = 0.5    # 内禀增长率
K = 1000   # 环境容纳量
N0 = 50    # 初始种群

# 解析解
def logistic_analytical(r, K, N0, t):
    return K / (1 + ((K - N0)/N0) * np.exp(-r*t))

# 数值验证
t, N_num = logistic_growth(r, K, N0, 20)
N_ana = logistic_analytical(r, K, N0, t)

error = np.max(np.abs(N_num - N_ana))
print(f"最大误差: {error:.2e}")
```

**Lotka-Volterra模拟**：

```python
def lotka_volterra(alpha, beta, gamma, delta, X0, Y0, t_max, dt=0.01):
    """Lotka-Volterra系统模拟"""
    t = np.arange(0, t_max, dt)
    X = np.zeros_like(t)
    Y = np.zeros_like(t)
    X[0], Y[0] = X0, Y0
    
    for i in range(1, len(t)):
        dX = (alpha * X[i-1] - beta * X[i-1] * Y[i-1]) * dt
        dY = (delta * X[i-1] * Y[i-1] - gamma * Y[i-1]) * dt
        X[i] = X[i-1] + dX
        Y[i] = Y[i-1] + dY
    
    return t, X, Y

# 模拟
t, X, Y = lotka_volterra(1.0, 0.1, 1.5, 0.075, 10, 5, 20)

# 相图分析
plt.figure(figsize=(10, 5))
plt.subplot(1, 2, 1)
plt.plot(t, X, label='Prey')
plt.plot(t, Y, label='Predator')
plt.xlabel('Time')
plt.ylabel('Population')
plt.legend()

plt.subplot(1, 2, 2)
plt.plot(X, Y)
plt.xlabel('Prey')
plt.ylabel('Predator')
plt.title('Phase Portrait')
```

### 5.2 COVID-19传播模型

**扩展SEIR模型**（含干预措施）：

```python
def covid_seir_model(beta0, beta1, sigma, gamma, N, E0, I0, R0, 
                     intervention_day, t_max, dt=0.1):
    """
    COVID-19 SEIR模型
    beta0: 干预前感染率
    beta1: 干预后感染率
    """
    steps = int(t_max / dt)
    S, E, I, R = np.zeros(steps), np.zeros(steps), np.zeros(steps), np.zeros(steps)
    
    S[0], E[0], I[0], R[0] = N - E0 - I0 - R0, E0, I0, R0
    
    for t in range(1, steps):
        day = t * dt
        beta = beta1 if day >= intervention_day else beta0
        
        dS = -beta * S[t-1] * I[t-1] / N * dt
        dE = (beta * S[t-1] * I[t-1] / N - sigma * E[t-1]) * dt
        dI = (sigma * E[t-1] - gamma * I[t-1]) * dt
        dR = gamma * I[t-1] * dt
        
        S[t] = S[t-1] + dS
        E[t] = E[t-1] + dE
        I[t] = I[t-1] + dI
        R[t] = R[t-1] + dR
    
    return S, E, I, R

# 实际参数（基于2020年初数据）
params = {
    'beta0': 0.5,      # 干预前基本再生数约2.5
    'beta1': 0.15,     # 干预后基本再生数约0.75
    'sigma': 1/5.2,    # 潜伏期5.2天
    'gamma': 1/2.9,    # 感染期2.9天
    'N': 1000000,      # 人口
    'intervention_day': 30
}

S, E, I, R = covid_seir_model(**params, E0=100, I0=10, R0=0, t_max=200)

# 计算基本再生数
R0_before = params['beta0'] / params['gamma']
R0_after = params['beta1'] / params['gamma']
print(f"干预前 R0 = {R0_before:.2f}")
print(f"干预后 R0 = {R0_after:.2f}")
print(f"峰值感染人数: {int(np.max(I))}")
print(f"总感染比例: {R[-1]/params['N']*100:.1f}%")
```

### 5.3 基因表达模拟

**Toggle Switch模拟**：

```python
def toggle_switch(alpha1, alpha2, n, K, p1_0, p2_0, t_max, dt=0.01):
    """基因Toggle Switch模拟"""
    steps = int(t_max / dt)
    p1, p2 = np.zeros(steps), np.zeros(steps)
    p1[0], p2[0] = p1_0, p2_0
    
    for t in range(1, steps):
        # Hill函数抑制
        f1 = alpha1 / (1 + (p2[t-1]/K)**n)
        f2 = alpha2 / (1 + (p1[t-1]/K)**n)
        
        # 蛋白质动力学（含降解）
        dp1 = (f1 - p1[t-1]) * dt
        dp2 = (f2 - p2[t-1]) * dt
        
        p1[t] = p1[t-1] + dp1
        p2[t] = p2[t-1] + dp2
    
    return p1, p2

# 双稳态演示
# 初始条件1：p1高，p2低
p1_a, p2_a = toggle_switch(5, 5, 2, 1, 4, 0.1, 50)
# 初始条件2：p1低，p2高
p1_b, p2_b = toggle_switch(5, 5, 2, 1, 0.1, 4, 50)

print("初始条件 (4, 0.1) -> 稳态:", f"({p1_a[-1]:.2f}, {p2_a[-1]:.2f})")
print("初始条件 (0.1, 4) -> 稳态:", f"({p1_b[-1]:.2f}, {p2_b[-1]:.2f})")
```

    ∀ t, deriv N t = r * N t * (1 - N t / K) := by

-- Lotka-Volterra系统
structure LotkaVolterra where
  alpha beta gamma delta : ℝ
  X0 Y0 : ℝ

def lotkaVolterraSystem (lv : LotkaVolterra) (t : ℝ) : ℝ × ℝ :=
  let dX := lv.alpha * X - lv.beta * X * Y
  let dY := lv.delta * X * Y - lv.gamma * Y
  (dX, dY)

-- SIR模型基本再生数
def basicReproductionNumber (beta gamma S0 : ℝ) : ℝ :=
  beta * S0 / gamma

-- 阈值定理
theorem thresholdTheorem (beta gamma S0 : ℝ) (h : gamma > 0) :
    let R0 := basicReproductionNumber beta gamma S0
    R0 > 1 → ∃ t, I t > I 0 := by
  sorry

-- Michaelis-Menten动力学
def michaelisMentenRate (Vmax Km S : ℝ) : ℝ :=
  Vmax * S / (Km + S)

-- 酶动力学解的存在唯一性
theorem enzymeKineticsWellPosed (Vmax Km S0 : ℝ) (h : Km > 0) :
    ∃! S : ℝ → ℝ, 
      S 0 = S0 ∧ 
      ∀ t, deriv S t = -michaelisMentenRate Vmax Km (S t) := by
  sorry

-- 双稳态系统（Toggle Switch）
def toggleSwitch (alpha1 alpha2 n K : ℝ) (p1 p2 : ℝ) : ℝ × ℝ :=
  let dp1 := alpha1 / (1 + (p2 / K)^n) - p1
  let dp2 := alpha2 / (1 + (p1 / K)^n) - p2
  (dp1, dp2)

-- 平衡点分析
theorem toggleSwitchBistability (alpha1 alpha2 n K : ℝ)
    (h_params : alpha1 > 0 ∧ alpha2 > 0 ∧ n > 1 ∧ K > 0) :
    ∃ p1_star p2_star, 
      toggleSwitch alpha1 alpha2 n K p1_star p2_star = (0, 0) ∧
      p1_star > p2_star := by
