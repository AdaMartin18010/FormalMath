## 1. 凸优化理论

### 1.1 凸集与凸函数

**定义 1.1** (凸集)

集合 $C \subseteq \mathbb{R}^n$ 是凸集，如果对于任意 $x, y \in C$ 和 $\lambda \in [0,1]$：

$$\lambda x + (1-\lambda)y \in C$$

**定义 1.2** (凸函数)

函数 $f: \mathbb{R}^n \to \mathbb{R}$ 是凸函数，如果对于任意 $x, y \in \text{dom}(f)$ 和 $\lambda \in [0,1]$：

$$f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y)$$

**判定条件**：
- 一阶条件：$f(y) \geq f(x) + \nabla f(x)^T(y-x)$
- 二阶条件：$\nabla^2 f(x) \succeq 0$

**强凸性**：

$$f(y) \geq f(x) + \nabla f(x)^T(y-x) + \frac{\mu}{2}\|y-x\|^2$$

等价于：$\nabla^2 f(x) \succeq \mu I$

### 1.2 对偶理论

**Lagrange对偶**：

原问题：
$$\min_x f(x) \quad \text{s.t.} \quad g_i(x) \leq 0, h_j(x) = 0$$

Lagrange函数：
$$L(x, \lambda, \nu) = f(x) + \sum_i \lambda_i g_i(x) + \sum_j \nu_j h_j(x)$$

对偶函数：
$$g(\lambda, \nu) = \inf_x L(x, \lambda, \nu)$$

对偶问题：
$$\max_{\lambda \geq 0, \nu} g(\lambda, \nu)$$

**弱对偶性**：$d^* \leq p^*$

**强对偶性**：在约束规范下（如Slater条件），$d^* = p^*$

**KKT条件**：

1. 原始可行性：$g_i(x) \leq 0$, $h_j(x) = 0$
2. 对偶可行性：$\lambda_i \geq 0$
3. 互补松弛：$\lambda_i g_i(x) = 0$
4. 平稳性：$\nabla f(x) + \sum_i \lambda_i \nabla g_i(x) + \sum_j \nu_j \nabla h_j(x) = 0$

### 1.3 锥优化

**锥规划**：

$$\min_x c^Tx \quad \text{s.t.} \quad Ax = b, x \succeq_K 0$$

其中 $K$ 是锥（如非负象限、二阶锥、半正定锥）。

**SDP (半正定规划)**：

$$\min_X \langle C, X \rangle \quad \text{s.t.} \quad \langle A_i, X \rangle = b_i, X \succeq 0$$

## 2. 一阶优化方法

### 2.1 梯度下降及其加速

**标准梯度下降**：

$$x_{k+1} = x_k - \eta_k \nabla f(x_k)$$

**收敛性分析**：

对于 $L$-光滑凸函数，$\eta = 1/L$：

$$f(x_k) - f^* \leq \frac{2L\|x_0 - x^*\|^2}{k+4}$$

**Nesterov加速**：

$$y_k = x_k + \frac{k-1}{k+2}(x_k - x_{k-1})$$
$$x_{k+1} = y_k - \frac{1}{L}\nabla f(y_k)$$

收敛速度：$O(1/k^2)$

### 2.2 近端方法

**近端梯度下降**（用于复合函数 $f + g$）：

$$x_{k+1} = \text{prox}_{\eta g}(x_k - \eta \nabla f(x_k))$$

其中近端算子：

$$\text{prox}_g(x) = \arg\min_y \left\{g(y) + \frac{1}{2}\|y-x\|^2\right\}$$

**加速近端梯度(FISTA)**：

```python
def fista(grad_f, prox_g, x0, L, max_iter):
    """加速近端梯度法"""
    x = x0.copy()
    y = x0.copy()
    t = 1
    
    for k in range(max_iter):
        x_new = prox_g(y - grad_f(y) / L, 1/L)
        t_new = (1 + np.sqrt(1 + 4*t**2)) / 2
        y = x_new + (t - 1) / t_new * (x_new - x)
        
        x, t = x_new, t_new
    
    return x
```

### 2.3 随机优化

**SGD收敛性**：

对于凸函数，$\eta_k = O(1/\sqrt{k})$：

$$\mathbb{E}[f(\bar{x}_T)] - f^* \leq O(1/\sqrt{T})$$

对于强凸函数，$\eta_k = O(1/k)$：

$$\mathbb{E}[\|x_T - x^*\|^2] \leq O(1/T)$$

**方差缩减方法**：

- SVRG: 定期计算全梯度
- SAGA: 存储历史梯度
- Katyusha: 加速方差缩减

## 3. 二阶优化方法

### 3.1 Newton方法

**经典Newton法**：

$$x_{k+1} = x_k - [\nabla^2 f(x_k)]^{-1}\nabla f(x_k)$$

**局部收敛性**：

在极小点 $x^*$ 附近，二次收敛：

$$\|x_{k+1} - x^*\| \leq C\|x_k - x^*\|^2$$

**信赖域Newton法**：

```python
def trust_region_newton(f, grad_f, hess_f, x0, max_iter=100):
    """信赖域Newton法"""
    x = x0.copy()
    delta = 1.0  # 信赖域半径
    
    for k in range(max_iter):
        g = grad_f(x)
        H = hess_f(x)
        
        # 求解信赖域子问题
        p = solve_trust_region_subproblem(g, H, delta)
        
        # 计算实际下降与预测下降比
        rho = (f(x) - f(x + p)) / (-g.dot(p) - 0.5 * p.dot(H.dot(p)))
        
        if rho > 0.75:
            delta *= 2
            x = x + p
        elif rho > 0.25:
            x = x + p
        else:
            delta *= 0.5
    
    return x
```

### 3.2 拟Newton方法

**BFGS更新**：

$$B_{k+1} = B_k - \frac{B_k s_k s_k^T B_k}{s_k^T B_k s_k} + \frac{y_k y_k^T}{y_k^T s_k}$$

其中 $s_k = x_{k+1} - x_k$，$y_k = \nabla f_{k+1} - \nabla f_k$

**L-BFGS**：

有限内存BFGS，只存储最近 $m$ 步的信息。

### 3.3 内点法

**障碍函数法**：

$$\min_x f(x) - \mu \sum_i \log(-g_i(x))$$

**原始-对偶内点法**：

求解KKT系统的扰动版本。

## 4. 约束优化

### 4.1 罚函数法

**二次罚函数**：

$$\min_x f(x) + \frac{\rho}{2}\sum_i \max(0, g_i(x))^2 + \frac{\rho}{2}\sum_j h_j(x)^2$$

随着 $\rho \to \infty$，逼近原问题。

### 4.2 增广Lagrange法

**增广Lagrange函数**：

$$L_\rho(x, \lambda) = f(x) + \lambda^T h(x) + \frac{\rho}{2}\|h(x)\|^2$$

**更新规则**：

$$x_{k+1} = \arg\min_x L_\rho(x, \lambda_k)$$
$$\lambda_{k+1} = \lambda_k + \rho h(x_{k+1})$$

### 4.3 SQP方法

**序列二次规划**：

在每一步求解QP子问题：

$$\min_p \frac{1}{2}p^T H_k p + \nabla f_k^T p$$
$$\text{s.t. } \nabla g_i^T p + g_i \leq 0, \nabla h_j^T p + h_j = 0$$

## 5. 实际应用案例

### 5.1 供应链优化

**问题描述**：最小化总成本（生产+库存+运输）

**数学模型**：

$$\min \sum_t \sum_i (c_{it}^{prod} P_{it} + c_{it}^{inv} I_{it}) + \sum_t \sum_{ij} c_{ijt}^{trans} T_{ijt}$$

约束：
- 产能约束：$P_{it} \leq Cap_{it}$
- 库存平衡：$I_{it} = I_{i,t-1} + P_{it} - D_{it} + \sum_j T_{jit} - \sum_j T_{ijt}$
- 非负约束

**实际案例**：某制造业公司

```python
def supply_chain_optimization(demand_forecast, capacities, costs):
    """供应链优化"""
    T, n_facilities = demand_forecast.shape
    
    # 决策变量
    production = cp.Variable((T, n_facilities), nonneg=True)
    inventory = cp.Variable((T, n_facilities), nonneg=True)
    transport = cp.Variable((T, n_facilities, n_facilities), nonneg=True)
    
    # 目标函数
    obj = (cp.sum(cp.multiply(costs['production'], production)) +
           cp.sum(cp.multiply(costs['inventory'], inventory)) +
           cp.sum(cp.multiply(costs['transport'], transport)))
    
    constraints = []
    
    # 产能约束
    for t in range(T):
        for i in range(n_facilities):
            constraints.append(production[t, i] <= capacities[i])
    
    # 库存平衡
    for t in range(T):
        for i in range(n_facilities):
            flow_in = cp.sum(transport[:, :, i], axis=1)
            flow_out = cp.sum(transport[:, i, :], axis=1)
            if t == 0:
                constraints.append(
                    inventory[t, i] == production[t, i] - demand_forecast[t, i] 
                    + flow_in[t] - flow_out[t]
                )
            else:
                constraints.append(
                    inventory[t, i] == inventory[t-1, i] + production[t, i] 
                    - demand_forecast[t, i] + flow_in[t] - flow_out[t]
                )
    
    prob = cp.Problem(cp.Minimize(obj), constraints)
    prob.solve(solver=cp.GUROBI)
    
    return production.value, inventory.value, transport.value
```

**优化效果**：
- 总成本降低15%
- 库存周转率提高20%
- 服务水平保持在98%

### 5.2 电力系统调度

**经济调度问题**：

$$\min_{P_g} \sum_g C_g(P_g)$$
$$\text{s.t. } \sum_g P_g = D, \quad P_g^{min} \leq P_g \leq P_g^{max}$$

**DC OPF（最优潮流）**：

```python
def dc_opf(generators, loads, lines, B_matrix):
    """直流最优潮流"""
    n_buses = len(loads)
    n_gen = len(generators)
    
    # 决策变量
    P_gen = cp.Variable(n_gen)
    theta = cp.Variable(n_buses)
    
    # 目标：发电成本最小
    cost = sum(g['a']*P_gen[i]**2 + g['b']*P_gen[i] + g['c'] 
               for i, g in enumerate(generators))
    
    constraints = [
        # 功率平衡
        B_matrix @ theta == cp.hstack([
            sum(P_gen[g['bus']] for g in generators if g['bus'] == i) - loads[i]
            for i in range(n_buses)
        ]),
        # 发电机出力限制
        *[P_gen[i] >= g['Pmin'] for i, g in enumerate(generators)],
        *[P_gen[i] <= g['Pmax'] for i, g in enumerate(generators)],
        # 线路潮流限制
        *[cp.abs(B_matrix[i,j]*(theta[i]-theta[j])) <= lines[i,j]['limit']
          for i in range(n_buses) for j in range(i+1, n_buses) 
          if (i,j) in lines]
    ]
    
    prob = cp.Problem(cp.Minimize(cost), constraints)
    prob.solve()
    
    return P_gen.value, theta.value
```

### 5.3 投资组合优化

**马科维茨模型**：

$$\min_w w^T\Sigma w$$
$$\text{s.t. } \mu^T w \geq R_{target}, \mathbf{1}^Tw = 1, w \geq 0$$

**风险平价组合**：

$$\min_w \sum_{i=1}^n \left(\frac{w_i(\Sigma w)_i}{w^T\Sigma w} - \frac{1}{n}\right)^2$$

```python
def risk_parity_portfolio(Sigma, max_iter=1000):
    """风险平价投资组合"""
    n = Sigma.shape[0]
    w = np.ones(n) / n
    
    for _ in range(max_iter):
        # 计算边际风险贡献
        MRC = Sigma @ w
        RC = w * MRC
        
        # 更新权重
        target_RC = w.T @ Sigma @ w / n
        w_new = target_RC / MRC
        w_new = w_new / np.sum(w_new)
        
        if np.linalg.norm(w_new - w) < 1e-8:
            break
        w = w_new
    
    return w
```
