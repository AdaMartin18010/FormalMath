## 📋 概述

后量子密码学（Post-Quantum Cryptography, PQC）研究在量子计算威胁下仍然安全的密码系统。2024年NIST发布了首批PQC标准（ML-KEM、ML-DSA、SLH-DSA），标志着密码学进入后量子时代。本深度版系统阐述格密码、NTRU、哈希签名等核心方案的数学基础。

## 1. 格的基本概念

### 1.1 格的定义与参数

**定义 1.1（格）**:
$\mathbb{R}^n$中的**格**是$m$个线性无关向量$\mathbf{b}_1, \ldots, \mathbf{b}_m \in \mathbb{R}^n$的整数线性组合：
$$
\mathcal{L}(\mathbf{B}) = \left\{\sum_{i=1}^{m} z_i \mathbf{b}_i : z_i \in \mathbb{Z}\right\} = \mathbf{B}\mathbb{Z}^m
$$

其中$\mathbf{B} = [\mathbf{b}_1 | \cdots | \mathbf{b}_m] \in \mathbb{R}^{n \times m}$称为**基矩阵**。若$m=n$，称格为**满秩**。

**基本平行多面体**:
$$
\mathcal{P}(\mathbf{B}) = \{\mathbf{B}\mathbf{x} : \mathbf{x} \in \mathbb{R}^m, 0 \leq x_i < 1\}
$$

**行列式**（协体积）:
$$
\det(\mathcal{L}) = \sqrt{\det(\mathbf{B}^T\mathbf{B})}
$$
对满秩格：$\det(\mathcal{L}) = |\det(\mathbf{B})|$。

### 1.2 格的不变量

**逐次最小值**:
对$i = 1, \ldots, m$，第$i$逐次最小值：
$$
\lambda_i(\mathcal{L}) = \inf\{r : \dim(\mathrm{span}(\mathcal{L} \cap \bar{\mathcal{B}}(0, r))) \geq i\}
$$
其中$\bar{\mathcal{B}}(0, r)$为闭球。

**覆盖半径**:
$$
\mu(\mathcal{L}) = \inf\{r : \mathcal{L} + \bar{\mathcal{B}}(0, r) = \mathbb{R}^n\}
$$

**最短向量长度**: $\lambda_1(\mathcal{L}) = \min_{\mathbf{v} \in \mathcal{L} \setminus \{0\}} \|\mathbf{v}\|$。

**Minkowski定理**:
**定理 1.2（第一Minkowski定理）**:
$$
\lambda_1(\mathcal{L}) \leq \sqrt{n} \cdot \det(\mathcal{L})^{1/n}
$$

**定理 1.3（第二Minkowski定理）**:
$$
\prod_{i=1}^{n} \lambda_i(\mathcal{L}) \leq n^{n/2} \cdot \det(\mathcal{L})
$$

### 1.3 对偶格

**定义 1.4（对偶格）**:
格$\mathcal{L}$的**对偶格**：
$$
\mathcal{L}^* = \{\mathbf{w} \in \mathrm{span}(\mathcal{L}) : \langle \mathbf{w}, \mathbf{v} \rangle \in \mathbb{Z}, \forall \mathbf{v} \in \mathcal{L}\}
$$

**性质**:

- $(\mathcal{L}^*)^* = \mathcal{L}$
- $\det(\mathcal{L}^*) = \det(\mathcal{L})^{-1}$
- 若$\mathbf{B}$是$\mathcal{L}$的基，则$(\mathbf{B}^T)^{-1}$是$\mathcal{L}^*$的基

## 2. 格上的困难问题

### 2.1 最短向量问题(SVP)

**定义 2.1（SVP）**:
**最短向量问题**: 给定格基$\mathbf{B}$，找到最短非零向量：
$$
\text{SVP}: \text{find } \mathbf{v} \in \mathcal{L}(\mathbf{B}) \setminus \{\mathbf{0}\} \text{ with } \|\mathbf{v}\| = \lambda_1(\mathcal{L})
$$

**近似版本（$\gamma$-SVP）**:
找到$\mathbf{v} \in \mathcal{L}(\mathbf{B}) \setminus \{\mathbf{0}\}$满足：
$$
\|\mathbf{v}\| \leq \gamma \cdot \lambda_1(\mathcal{L})
$$

**计算复杂性**:

- **精确SVP**: NP-hard（Van Emde Boas, 1981; Ajtai, 1998）
- **近似SVP**: NP-hard for $\gamma = n^{O(1/\log\log n)}$（Khot, 2005）

**算法**:

1. **枚举算法（Kannan）**: $2^{O(n\log n)}$时间
2. **sieving算法**: $2^{0.292n + o(n)}$时间（量子: $2^{0.265n}$）
3. **BKZ算法**: 块Korkine-Zolotarev约化，实用中$\gamma = (\delta/2)^{n/k}$

### 2.2 最近向量问题(CVP)

**定义 2.2（CVP）**:
**最近向量问题**: 给定格基$\mathbf{B}$和目标向量$\mathbf{t} \in \mathbb{R}^n$，找到格中最接近$\mathbf{t}$的向量：
$$
\text{CVP}: \text{find } \mathbf{v} \in \mathcal{L}(\mathbf{B}) \text{ minimizing } \|\mathbf{t} - \mathbf{v}\|
$$

**近似版本（$\gamma$-CVP）**:
$$
\|\mathbf{t} - \mathbf{v}\| \leq \gamma \cdot \min_{\mathbf{w} \in \mathcal{L}} \|\mathbf{t} - \mathbf{w}\|
$$

**定理 2.3（CVP计算复杂性）**:
精确CVP是NP-hard，近似CVP在$\gamma = n^{O(1/\log\log n)}$时仍是NP-hard。

**Babai算法**（最近平面算法）:
给定近似正交基，CVP可在$O(n^3)$时间内解决，近似因子为$n \cdot (\max_i \|\mathbf{b}_i^*\| / \min_i \|\mathbf{b}_i^*\|)$。

### 2.3 其他困难问题

**最短独立向量问题(SIVP)**:
找到$n$个线性无关向量$\mathbf{v}_1, \ldots, \mathbf{v}_n \in \mathcal{L}$，使得$\max_i \|\mathbf{v}_i\|$最小化。

**有界距离解码(BDD)**:
CVP的变体，目标$\mathbf{t}$保证距离某个格向量小于$\lambda_1/(2\gamma)$。

**关系**:
$$
\text{SVP} \leq_P \text{SIVP} \leq_P \text{CVP}
$$

在近似因子多项式相关时，这些问题是等价的。

## 3. LWE问题与Regev加密

### 3.1 LWE问题定义

**定义 3.1（LWE分布）**:
设$n$为安全参数，$q = q(n)$为模数，$\chi$为$\mathbb{Z}_q$上的错误分布。对秘密向量$\mathbf{s} \in \mathbb{Z}_q^n$，**LWE分布**$A_{\mathbf{s}, \chi}$采样：
$$
(\mathbf{a}, b = \langle \mathbf{a}, \mathbf{s} \rangle + e \mod q) \in \mathbb{Z}_q^n \times \mathbb{Z}_q
$$
其中$\mathbf{a} \leftarrow U(\mathbb{Z}_q^n)$，$e \leftarrow \chi$。

**定义 3.2（LWE问题）**:
**决策LWE（DLWE）**: 区分$A_{\mathbf{s}, \chi}$与均匀分布$U(\mathbb{Z}_q^n \times \mathbb{Z}_q)$
**搜索LWE**: 给定LWE样本，恢复秘密$\mathbf{s}$

**定理 3.3（Regev归约）**:
若存在量子算法解决$\mathrm{DLWE}_{n,q,\chi}$，则存在量子算法解决$\tilde{O}(n \cdot q/\sigma)$-近似SVP，其中$\sigma$为错误标准差。

### 3.2 Regev加密方案

**参数**: $n, m, q, \chi$

**密钥生成**:

- 私钥: $\mathbf{s} \leftarrow \chi^n$
- 公钥: $(\mathbf{A}, \mathbf{b} = \mathbf{A}\mathbf{s} + \mathbf{e})$，其中$\mathbf{A} \leftarrow U(\mathbb{Z}_q^{m \times n})$，$\mathbf{e} \leftarrow \chi^m$

**加密**（单比特$\mu \in \{0, 1\}$）:

- 选随机向量$\mathbf{r} \in \{0, 1\}^m$
- 密文: $(\mathbf{c}_1 = \mathbf{r}^T\mathbf{A}, c_2 = \mathbf{r}^T\mathbf{b} + \mu \cdot \lfloor q/2 \rceil)$

**解密**:
$$
\mu' = \begin{cases}
0 & \text{if } |c_2 - \langle \mathbf{c}_1, \mathbf{s} \rangle| < q/4 \\
1 & \text{otherwise}
\end{cases}
$$

**正确性**:
解密错误概率$\leq 2^{-\Omega(n)}$（当错误界足够小时）。

**安全性**:
基于DLWE假设，具有IND-CPA安全性。

## 4. NTRU密码系统

### 4.1 NTRU数学基础

**环结构**:
$$
R = \mathbb{Z}[x]/(x^n - 1) \quad \text{或} \quad R_q = \mathbb{Z}_q[x]/(x^n - 1)
$$
现代变体使用$R = \mathbb{Z}[x]/(x^n + 1)$（NTRU Prime）。

**定义 4.1（NTRU问题）**:
给定$h = f/g \mod q \in R_q$，其中$f, g$为短多项式，恢复$f, g$。

**参数**: $(n, p, q, d_f, d_g, d_r)$

- $n$: 多项式次数
- $p, q$: 小模数和大模数，$\gcd(p, q) = 1$
- $d_f, d_g, d_r$: 稀疏度参数

### 4.2 NTRU加密

**密钥生成**:

1. 随机选$f \in R$，$d_f$个系数为1，$d_f-1$个为-1，其余为0，且$f$在$R_q$中可逆
2. 随机选$g \in R$，$d_g$个系数为1，$d_g$个为-1
3. 私钥: $(f, f_p)$，其中$f_p = f^{-1} \mod p$
4. 公钥: $h = p \cdot g \cdot f^{-1} \mod q \in R_q$

**加密**（消息$m \in R$，系数在$\{-1, 0, 1\}$）:

1. 随机选$r \in R$，$d_r$个系数为1，$d_r$个为-1
2. 密文: $e = r \cdot h + m \mod q$

**解密**:

1. 计算$a = f \cdot e \mod q = p \cdot r \cdot g + f \cdot m \mod q$
2. 中心提升：将$a$系数调整到$[-q/2, q/2)$
3. 计算$m' = f_p \cdot a \mod p$

**正确性条件**:
$$
\|p \cdot r \cdot g + f \cdot m\|_\infty < q/2
$$

### 4.3 NTRU安全性分析

**攻击方法**:

1. **格攻击**: NTRU格$L_{h} = \{(u, v) \in R^2 : u \cdot h = v \mod q\}$
2. **解密失败攻击**: 利用解密失败信息
3. **中间相遇攻击**: 针对稀疏密钥

**安全参数建议**（NTRU-HRSS）:

| 安全级别 | $n$ | $q$ |
|---------|-----|-----|
| 128-bit | 701 | 8192 |
| 192-bit | 821 | 4096 |
| 256-bit | 1229 | 4096 |

## 5. NIST PQC标准

### 5.1 ML-KEM（Kyber）

**基础**: 模块-LWE（MLWE）

**核心参数**:

| 参数 | ML-KEM-512 | ML-KEM-768 | ML-KEM-1024 |
|-----|-----------|-----------|------------|
| $n$ | 256 | 256 | 256 |
| $q$ | 3329 | 3329 | 3329 |
| $k$ | 2 | 3 | 4 |
| $\eta$ | 3 | 2 | 2 |

**密钥封装机制**:

1. **KeyGen**: 生成公钥$\mathbf{A} \in R_q^{k \times k}$，私钥$(\mathbf{s}, \mathbf{e})$
2. **Encaps**: 生成共享密钥$K$和密文$\mathbf{c}$
3. **Decaps**: 从密文恢复$K$

**安全性**: IND-CCA2安全，基于MLWE。

### 5.2 ML-DSA（Dilithium）

**基础**: 模块-LWE + 模块-SIS

**签名方案**:

1. **KeyGen**:
   - 私钥: 短向量$(\mathbf{s}_1, \mathbf{s}_2)$
   - 公钥: $\mathbf{A}, \mathbf{t} = \mathbf{A}\mathbf{s}_1 + \mathbf{s}_2$

2. **Sign($\mu$)**:
   - 选掩码$\mathbf{y}$
   - 计算$\mathbf{w} = \mathbf{A}\mathbf{y}$
   - 挑战$c = H(\mu \| \mathbf{w})$
   - 响应$\mathbf{z} = \mathbf{y} + c\mathbf{s}_1$
   - 输出$(c, \mathbf{z})$（可能重试）

3. **Verify**:
   - 检查$\|\mathbf{z}\|$小
   - 验证$c = H(\mu \| \mathbf{A}\mathbf{z} - c\mathbf{t})$

**参数**:

| 参数 | ML-DSA-44 | ML-DSA-65 | ML-DSA-87 |
|-----|-----------|-----------|-----------|
| $d$ | 13 | 13 | 13 |
| $\tau$ | 39 | 49 | 60 |
| $\gamma_1$ | $2^{17}$ | $2^{19}$ | $2^{19}$ |

### 5.3 SLH-DSA（SPHINCS+）

**基础**: 哈希函数（无结构化困难假设）

**结构**:

- 超树（Hypertree）签名XMSS实例
- 无状态设计，使用树叶的伪随机选择
- FORS（Forest of Random Subsets）Few-Time签名

**安全级别**:

| 变体 | 参数 | 公钥大小 | 签名大小 |
|-----|------|---------|---------|
| SLH-DSA-SHA2-128s | 128-bit | 32 B | 7.8 KB |
| SLH-DSA-SHA2-128f | 128-bit | 32 B | 17.0 KB |
| SLH-DSA-SHA2-256s | 256-bit | 64 B | 29.8 KB |

## 6. Python实现

### 6.1 LWE加密实现

```python
import numpy as np
from typing import Tuple

class LWECrypto:
    """LWE-based加密（简化Regev方案）"""

    def __init__(self, n: int = 50, q: int = 257, sigma: float = 3.0):
        self.n = n  # 维度
        self.q = q  # 模数
        self.sigma = sigma  # 错误标准差
        self.m = 2 * n  # 样本数

    def _sample_error(self, size: int) -> np.ndarray:
        """从离散高斯采样错误"""
        # 简化为中心二项分布
        return np.random.binomial(16, 0.5, size) - 8

    def keygen(self) -> Tuple[np.ndarray, np.ndarray, np.ndarray]:
        """密钥生成: 返回(私钥s, 公钥A, 公钥b)"""
        # 私钥: 短向量
        s = self._sample_error(self.n)

        # 公钥: A和b = As + e
        A = np.random.randint(0, self.q, size=(self.m, self.n))
        e = self._sample_error(self.m)
        b = (A @ s + e) % self.q

        return s, A, b

    def encrypt(self, A: np.ndarray, b: np.ndarray,
                message: int) -> Tuple[np.ndarray, int]:
        """加密单比特消息"""
        assert message in {0, 1}

        # 随机向量r ∈ {0,1}^m
        r = np.random.randint(0, 2, size=self.m)

        # 密文: (c1 = r^T A, c2 = r^T b + m*q/2)
        c1 = (r @ A) % self.q
        c2 = int((r @ b) % self.q + message * (self.q // 2)) % self.q

        return c1, c2

    def decrypt(self, s: np.ndarray, c1: np.ndarray, c2: int) -> int:
        """解密密文"""
        # 计算距离
        dist = (c2 - (c1 @ s) % self.q) % self.q

        # 调整到中心
        if dist > self.q // 2:
            dist -= self.q

        # 判断
        return 1 if abs(dist) > self.q // 4 else 0

# 测试
if __name__ == "__main__":
    lwe = LWECrypto(n=50, q=257)
    s, A, b = lwe.keygen()

    # 加密解密测试
    for msg in [0, 1]:
        c1, c2 = lwe.encrypt(A, b, msg)
        decrypted = lwe.decrypt(s, c1, c2)
        print(f"消息: {msg}, 解密: {decrypted}, 正确: {msg == decrypted}")
```

### 6.2 NTRU实现

```python
class NTRU:
    """NTRU加密实现"""

    def __init__(self, n: int = 7, p: int = 3, q: int = 41,
                 df: int = 2, dg: int = 2):
        self.n = n
        self.p = p
        self.q = q
        self.df = df
        self.dg = dg

    def _random_poly(self, num_ones: int, num_neg_ones: int) -> np.ndarray:
        """生成稀疏多项式"""
        coeffs = [1] * num_ones + [-1] * num_neg_ones + [0] * (self.n - num_ones - num_neg_ones)
        np.random.shuffle(coeffs)
        return np.array(coeffs)

    def _convolve(self, a: np.ndarray, b: np.ndarray) -> np.ndarray:
        """环R_q中的乘法（循环卷积）"""
        result = np.zeros(self.n, dtype=int)
        for i in range(self.n):
            for j in range(self.n):
                idx = (i + j) % self.n
                result[idx] += a[i] * b[j]
        return result

    def _mod_coeffs(self, poly: np.ndarray, mod: int) -> np.ndarray:
        """系数取模"""
        return ((poly % mod) + mod // 2) % mod - mod // 2

    def _extended_gcd_poly(self, a: np.ndarray, b: int) -> Tuple[np.ndarray, np.ndarray]:
        """扩展欧几里得算法（简化版）"""
        # 这里简化为模逆计算
        return np.array([pow(int(a[0]), -1, b)]), np.array([0])

    def keygen(self) -> Tuple[np.ndarray, np.ndarray]:
        """密钥生成: 返回(私钥f, 公钥h)"""
        # 生成f（可逆）
        while True:
            f = self._random_poly(self.df, self.df - 1)
            try:
                # 检查在模p和模q下是否可逆（简化）
                if f[0] % self.p != 0 and f[0] % self.q != 0:
                    break
            except:
                continue

        # 生成g
        g = self._random_poly(self.dg, self.dg)

        # 计算f^{-1} mod q（简化: 仅考虑常数项）
        f_inv_q = np.zeros(self.n)
        f_inv_q[0] = pow(int(f[0]), -1, self.q)

        # 公钥 h = p * g * f^{-1} mod q
        h = self._mod_coeffs(self.p * self._convolve(g, f_inv_q), self.q)

        return f, h

    def encrypt(self, h: np.ndarray, message: np.ndarray) -> np.ndarray:
        """加密消息"""
        assert len(message) == self.n

        # 随机r
        r = self._random_poly(self.df, self.df)

        # e = r * h + m mod q
        e = self._mod_coeffs(self._convolve(r, h) + message, self.q)
        return e

    def decrypt(self, f: np.ndarray, e: np.ndarray) -> np.ndarray:
        """解密"""
        # a = f * e mod q
        a = self._mod_coeffs(self._convolve(f, e), self.q)

        # 中心提升已在上一步完成
        # m' = a mod p
        m_prime = self._mod_coeffs(a, self.p)
        return m_prime

# 测试
if __name__ == "__main__":
    ntru = NTRU(n=7, p=3, q=41, df=2, dg=2)
    f, h = ntru.keygen()

    message = np.array([1, -1, 0, 1, 0, -1, 0])
    print(f"原始消息: {message}")

    ciphertext = ntru.encrypt(h, message)
    print(f"密文: {ciphertext}")

    decrypted = ntru.decrypt(f, ciphertext)
    print(f"解密: {decrypted}")
    print(f"正确: {np.array_equal(decrypted, message)}")
```

### 7.2 LWE问题的形式化

```lean4
-- LWE distribution
def LWEDistribution {n q : ℕ} (s : Fin n → ZMod q)
    (χ : PMF (ZMod q)) : PMF ((Fin n → ZMod q) × ZMod q) :=
  do
    let a ← PMF.uniformOfFinset (Finset.univ) (by simp)
    let e ← χ
    pure (a, (Finset.sum Finset.univ (fun i => a i * s i) + e))

-- Decision LWE assumption
def DecisionLWE (n q : ℕ) (χ : PMF (ZMod q)) : Prop :=
  ∀ s : Fin n → ZMod q,
  ∀ D : ((Fin n → ZMod q) × ZMod q) → Bool,
  |D '' (LWEDistribution s χ) - D '' (PMF.uniformOfFinset Finset.univ (by simp))| ≤ negl n

-- Search LWE assumption
def SearchLWE (n q : ℕ) (χ : PMF (ZMod q)) : Prop :=
  ∀ s : Fin n → ZMod q,
  ∀ A : PMF ((Fin n → ZMod q) × ZMod q) → PMF (Fin n → ZMod q),
  Pr[(do let samples ← PMF.iid (LWEDistribution s χ) (n+1); A samples) = s] ≥ 1 - negl n
```

### 7.3 NTRU问题的形式化

```lean4
-- Polynomial ring R = Z[x]/(x^n - 1)
def R (n : ℕ) := Polynomial ℤ ⧸ Ideal.span {Polynomial.X^n - 1}

-- R_q = Z_q[x]/(x^n - 1)
def Rq (n q : ℕ) := Polynomial (ZMod q) ⧸ Ideal.span {(Polynomial.X^n - 1 : Polynomial (ZMod q))}

-- NTRU instance
def NTRUInstance {n q : ℕ} (h : Rq n q) : Prop :=
  ∃ f g : R n,
    (∀ i, |(f.coeff i)| ≤ 1) ∧  -- f is short
    (∀ i, |(g.coeff i)| ≤ 1) ∧  -- g is short
    (f : Rq n q) * h = (g : Rq n q)  -- h = g/f in R_q

-- NTRU problem: find short f', g' such that f' * h = g'
def NTRUProblem (n q : ℕ) (h : Rq n q) : Set (R n × R n) :=
  { (f, g) | (∀ i, |(f.coeff i)| ≤ 1) ∧ (∀ i, |(g.coeff i)| ≤ 1) ∧
             (f : Rq n q) * h = (g : Rq n q) }
```
