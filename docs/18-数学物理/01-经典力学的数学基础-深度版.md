## 一、引言

经典力学是数学物理中最古老且最成熟的领域之一，其现代数学形式由Lagrange、Hamilton、Jacobi和Poincaré等人在18至19世纪建立。从数学视角看，经典力学的核心在于揭示物理系统演化背后的几何结构——特别是辛几何结构。

与Newton的向量力学不同，分析力学通过变分原理和相空间几何提供了一般性的数学框架。这一框架不仅适用于质点力学，也可推广至场论和连续介质力学，成为现代理论物理的基石。

**核心思想**: 经典力学系统的运动由作用量的驻定（极值）决定，相空间具有自然的辛流形结构，Hamilton流保持这一辛结构不变。

## 二、Lagrange力学与变分原理

### 2.1 构型空间与切丛

**定义 2.1** (构型空间)
一个力学系统的**构型空间**是光滑流形 $Q$，其点表示系统的所有可能位形。若系统有 $n$ 个自由度，则 $\dim Q = n$。

**定义 2.2** (切丛)
构型空间 $Q$ 的**切丛** $TQ$ 是所有切空间的并：
$$TQ = \bigcup_{q \in Q} T_q Q$$
其元素为 $(q, v)$，其中 $q \in Q$，$v \in T_q Q$。$TQ$ 是 $2n$ 维光滑流形。

### 2.2 Lagrange函数与作用量

**定义 2.3** (Lagrange函数)
**Lagrange函数**（或Lagrange量）是光滑映射 $L: TQ \times \mathbb{R} \to \mathbb{R}$，通常形式为：
$$L(q, \dot{q}, t) = T(q, \dot{q}) - V(q, t)$$
其中 $T$ 是动能，$V$ 是势能。

**定义 2.4** (作用量泛函)
给定曲线 $q: [t_1, t_2] \to Q$，**作用量**定义为：
$$S[q] = \int_{t_1}^{t_2} L(q(t), \dot{q}(t), t) \, dt$$
这是路径空间上的泛函。

### 2.3 Hamilton原理与Euler-Lagrange方程

**定理 2.1** (Hamilton原理 / Euler-Lagrange方程)
设 $L$ 是正则Lagrange量（Hessian矩阵 $\partial^2 L / \partial \dot{q}^i \partial \dot{q}^j$ 非退化）。曲线 $q(t)$ 使作用量 $S$ 取驻定值当且仅当满足**Euler-Lagrange方程**：
$$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}^i}\right) - \frac{\partial L}{\partial q^i} = 0, \quad i = 1, \ldots, n$$

**证明概要**: 考虑变分 $q_\epsilon(t) = q(t) + \epsilon \eta(t)$，其中 $\eta(t_1) = \eta(t_2) = 0$。由 $\frac{d}{d\epsilon}S[q_\epsilon]|_{\epsilon=0} = 0$ 分部积分即得。

**物理意义**: Hamilton原理表明，自然界选择使作用量取极值的路径——这一变分原理比Newton方程更具一般性，且易于推广到场论。

### 2.4 广义动量与Lagrange方程的几何形式

**定义 2.5** (广义动量)
**广义动量**（共轭动量）定义为：
$$p_i = \frac{\partial L}{\partial \dot{q}^i}$$
这定义了Legendre变换的局部形式。

**几何解释**: Euler-Lagrange方程可写为切丛 $TQ$ 上的向量场方程。存在唯一的**Lagrange向量场** $X_L$ 满足：
$$\iota_{X_L} \omega_L = dE_L$$
其中 $\omega_L$ 是Lagrange $2$-形式，$E_L = p_i \dot{q}^i - L$ 是能量函数。

## 三、Legendre变换与Hamilton力学

### 3.1 Legendre变换

**定义 3.1** (Legendre变换)
**Legendre变换**是映射 $\mathbb{F}L: TQ \to T^*Q$，定义为：
$$\mathbb{F}L(q, \dot{q}) = \left(q, \frac{\partial L}{\partial \dot{q}}\right) = (q, p)$$
对于正则Lagrange量，$\mathbb{F}L$ 是局部微分同胚。

**定理 3.1** (Legendre变换的等价性)
若Lagrange量 $L$ 是**超正则**的（即 $\mathbb{F}L$ 是全局微分同胚），则Lagrange力学与Hamilton力学等价。

### 3.2 Hamilton函数与Hamilton方程

**定义 3.2** (Hamilton函数)
**Hamilton函数** $H: T^*Q \times \mathbb{R} \to \mathbb{R}$ 通过Legendre变换定义：
$$H(q, p, t) = p_i \dot{q}^i - L(q, \dot{q}, t)$$
其中 $\dot{q}$ 通过 $p = \partial L / \partial \dot{q}$ 表示为 $(q, p)$ 的函数。

**定理 3.2** (Hamilton方程)
在Legendre变换下，Euler-Lagrange方程等价于**Hamilton方程**：
$$\dot{q}^i = \frac{\partial H}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial H}{\partial q^i}$$

**矩阵形式**: 设 $z = (q^1, \ldots, q^n, p_1, \ldots, p_n)^T$，$J = \begin{pmatrix} 0 & I_n \\ -I_n & 0 \end{pmatrix}$，则：
$$\dot{z} = J \nabla H$$

### 3.3 相空间与正则变换

**定义 3.3** (相空间)
**相空间**是构型空间的余切丛 $T^*Q$。它是描述力学系统状态的 $2n$ 维流形，装备有自然的辛结构。

**定义 3.4** (正则变换)
微分同胚 $\varphi: T^*Q \to T^*Q$ 称为**正则变换**（或辛变换），若保持辛形式：
$$\varphi^*\omega = \omega$$
其中 $\omega = dq^i \wedge dp_i$ 是标准辛形式。

## 四、辛流形上的Hamilton力学

### 4.1 辛流形

**定义 4.1** (辛形式与辛流形)
设 $M$ 是 $2n$ 维光滑流形。闭的非退化 $2$-形式 $\omega \in \Omega^2(M)$ 称为**辛形式**。二元组 $(M, \omega)$ 称为**辛流形**。

**标准辛形式**: 在 $T^*Q$ 上，标准辛形式为：
$$\omega = -d\theta = dq^i \wedge dp_i$$
其中 $\theta = p_i dq^i$ 是**Liouville $1$-形式**（或典范 $1$-形式）。

**定理 4.1** (Darboux定理)
对任意辛流形 $(M, \omega)$ 和点 $p \in M$，存在局部坐标（Darboux坐标）$(q^1, \ldots, q^n, p_1, \ldots, p_n)$ 使得：
$$\omega = \sum_{i=1}^n dq^i \wedge dp_i$$

**意义**: Darboux定理表明辛几何没有局部不变量——所有辛流形局部都是标准辛空间。

### 4.2 Hamilton向量场

**定义 4.2** (Hamilton向量场)
设 $(M, \omega)$ 是辛流形，$H \in C^\infty(M)$ 是**Hamilton函数**。**Hamilton向量场** $X_H$ 由以下唯一确定：
$$\iota_{X_H}\omega = dH$$
即对任意向量场 $Y$：$\omega(X_H, Y) = dH(Y) = Y(H)$。

**存在唯一性**: 由 $\omega$ 的非退化性，映射 $X \mapsto \iota_X\omega$ 是向量场与 $1$-形式之间的同构。

**Hamilton方程**: 在Darboux坐标下：
$$X_H = \sum_{i=1}^n \left(\frac{\partial H}{\partial p_i}\frac{\partial}{\partial q^i} - \frac{\partial H}{\partial q^i}\frac{\partial}{\partial p_i}\right)$$
因此 $X_H$ 的流线满足Hamilton方程。

### 4.3 Poisson括号与守恒量

**定义 4.3** (Poisson括号)
对 $f, g \in C^\infty(M)$，**Poisson括号**定义为：
$$\{f, g\} = \omega(X_f, X_g) = X_f(g) = -X_g(f)$$
在Darboux坐标下：
$$\{f, g\} = \sum_{i=1}^n \left(\frac{\partial f}{\partial q^i}\frac{\partial g}{\partial p_i} - \frac{\partial f}{\partial p_i}\frac{\partial g}{\partial q^i}\right)$$

**定理 4.2** (Poisson代数的Lie代数结构)
$(C^\infty(M), \{\cdot, \cdot\})$ 构成Lie代数：

1. **反对称性**: $\{f, g\} = -\{g, f\}$
2. **Jacobi恒等式**: $\{f, \{g, h\}\} + \{g, \{h, f\}\} + \{h, \{f, g\}\} = 0$
3. **Leibniz法则**: $\{fg, h\} = f\{g, h\} + g\{f, h\}$

**定理 4.3** (能量守恒)
$H$ 沿 $X_H$ 的流线是常数，即 $\frac{dH}{dt} = X_H(H) = \{H, H\} = 0$。

**Noether定理** (辛几何形式): 若 $f$ 是运动常数（即 $\{f, H\} = 0$），则 $X_f$ 生成的流保持Hamilton函数不变。

### 4.4 Liouville定理

**定理 4.4** (Liouville定理)
Hamilton流保持辛体积形式不变：若 $\varphi_t$ 是 $X_H$ 的流，则 $\varphi_t^*\omega = \omega$。因此相空间体积 $\omega^n/n!$ 守恒。

**推论**: 对有界能量曲面，Hamilton流保持Liouville测度不变。

## 五、可积系统与Liouville定理

### 5.1 可积系统的定义

**定义 5.1** (可积系统 / Liouville可积)
设 $(M, \omega)$ 是 $2n$ 维辛流形，$H$ 是Hamilton函数。若存在 $n$ 个函数 $F_1 = H, F_2, \ldots, F_n$ 满足：

1. **函数独立**: $dF_1, \ldots, dF_n$ 在 $M$ 上几乎处处线性无关
2. **互相对合**: $\{F_i, F_j\} = 0$ 对所有 $i, j$
3. **完备性**: Hamilton流是完备的（解对所有时间存在）

则称系统为**Liouville可积**（或完全可积）。

### 5.2 Arnold-Liouville定理

**定理 5.1** (Arnold-Liouville定理)
设 $(M, \omega, H)$ 是Liouville可积系统，$F_1, \ldots, F_n$ 是对合的独立守恒量。考虑水平集：
$$M_c = \{(q, p) \in M : F_i(q, p) = c_i, i = 1, \ldots, n\}$$
若 $M_c$ 是紧致且连通的，则：

1. $M_c$ 微分同胚于 $n$ 维环面 $\mathbb{T}^n$
2. 存在**作用-角变量** $(I_1, \ldots, I_n, \theta^1, \ldots, \theta^n)$ 使得 $\omega = dI_i \wedge d\theta^i$
3. Hamilton函数仅依赖于作用量：$H = H(I_1, \ldots, I_n)$
4. 运动方程为：$\dot{I}_i = 0$，$\dot{\theta}^i = \omega^i(I) = \frac{\partial H}{\partial I_i}$

**物理意义**: 可积系统的运动是准周期的，在环面上沿直线匀速运动。作用量是绝热不变量，角变量随时间线性增长。

### 5.3 重要例子

**谐振子**: $H = \frac{1}{2}(p^2 + \omega^2 q^2)$。作用-角变量为：
$$I = \frac{1}{2\omega}(p^2 + \omega^2 q^2), \quad \theta = \arctan\left(\frac{\omega q}{p}\right)$$

**Kepler问题**: $H = \frac{|p|^2}{2} - \frac{\mu}{|q|}$。除能量外还有角动量和Runge-Lenz向量守恒，系统可积。

## 六、历史发展与现代应用

### 6.1 历史脉络

**1687年**: Newton发表《自然哲学的数学原理》，建立Newton方程 $F = ma$，奠定经典力学基础。

**1788年**: Lagrange发表《分析力学》，引入广义坐标与变分原理，避免约束力，统一处理力学系统。

**1833-1835年**: Hamilton提出Hamilton形式，引入相空间 $(q, p)$ 与Hamilton方程。Jacobi发展Hamilton-Jacobi理论。

**1890年**: Poincaré发表《天体力学新方法》，开创动力系统理论，认识到三体问题的复杂性，为混沌理论奠基。

**1960-1970年代**: Arnold、Smale、Abraham-Marsden等将经典力学严格建立在辛几何基础上，《Mathematical Methods of Classical Mechanics》(1974) 和《Foundations of Mechanics》(1978) 是里程碑著作。

### 6.2 现代发展

**KAM理论** (Kolmogorov-Arnold-Moser): 研究近可积系统的稳定性。证明小扰动下大多数不变环面得以保持，解释了太阳系的稳定性。

**混沌理论**: 非可积系统的研究揭示了确定性系统中的随机行为。Poincaré截面、Lyapunov指数等工具发展起来。

**几何量子化**: 试图从经典力学的辛几何结构出发，系统地构造量子理论。预量子化、极化等概念是核心。

**辛拓扑**: Gromov (1985) 引入伪全纯曲线，开创了辛拓扑这一新领域。辛容量、Floer同调等不变量得到研究。

### 6.3 应用领域

| 领域 | 应用 |
|------|------|
| 天体力学 | 行星轨道演化、卫星姿态控制 |
| 分子动力学 | Hamilton分子动力学模拟、辛积分器 |
| 等离子体物理 | 带电粒子在电磁场中的运动 |
| 控制理论 | 几何控制、最优控制 |
| 量子化学 | 半经典近似、轨迹计算 |
