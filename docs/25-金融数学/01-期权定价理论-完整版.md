## 1. 问题建模

### 1.1 期权的基本概念

**定义 1.1** (欧式看涨期权): 赋予持有者在到期日 $T$ 以执行价 $K$ 购买标的资产的权利。

**收益函数**: $C_T = \max(S_T - K, 0) = (S_T - K)^+$

**期权类型对比**:

| 类型 | 看涨收益 | 看跌收益 | 行权时间 |
|------|----------|----------|----------|
| 欧式 | $(S_T - K)^+$ | $(K - S_T)^+$ | 到期日 |
| 美式 | $(S_T - K)^+$ | $(K - S_T)^+$ | 到期前任意时刻 |
| 亚式 | $(\bar{S} - K)^+$ | $(K - \bar{S})^+$ | 到期日 |
| 障碍 | 条件收益 | 条件收益 | 到期日 |

### 1.2 Black-Scholes模型假设

1. **标的资产价格**: 几何布朗运动
   $$dS_t = \mu S_t dt + \sigma S_t dW_t$$

2. **无风险利率**: $r$ 为常数

3. **无摩擦市场**: 无交易成本、税收，可连续交易

4. **无套利机会**: 市场完备

5. **标的资产无红利** (可扩展)

## 2. 数学理论

### 2.1 几何布朗运动

**定义 2.1** (几何布朗运动): 随机过程 $S_t$ 满足：
$$S_t = S_0 \exp\left(\left(\mu - \frac{\sigma^2}{2}\right)t + \sigma W_t\right)$$

**性质**:

- $\log S_t \sim \mathcal{N}(\log S_0 + (\mu - \sigma^2/2)t, \sigma^2 t)$
- $\mathbb{E}[S_t] = S_0 e^{\mu t}$
- $\text{Var}(S_t) = S_0^2 e^{2\mu t}(e^{\sigma^2 t} - 1)$

### 2.2 伊藤引理

**定理 2.1** (伊藤引理): 设 $X_t$ 满足 $dX_t = \mu_t dt + \sigma_t dW_t$，$f(t, x)$ 二次连续可微，则：
$$df(t, X_t) = \frac{\partial f}{\partial t}dt + \frac{\partial f}{\partial x}dX_t + \frac{1}{2}\frac{\partial^2 f}{\partial x^2}(dX_t)^2$$

其中 $(dX_t)^2 = \sigma_t^2 dt$。

### 2.3 Black-Scholes方程推导

**构造对冲组合**: $\Pi = V - \Delta S$

**选择** $\Delta = \frac{\partial V}{\partial S}$，使组合无风险：
$$d\Pi = r\Pi dt$$

**推导过程**:

1. 由伊藤引理: $dV = \frac{\partial V}{\partial t}dt + \frac{\partial V}{\partial S}dS + \frac{1}{2}\sigma^2S^2\frac{\partial^2 V}{\partial S^2}dt$

2. 组合变化: $d\Pi = dV - \Delta dS = \left(\frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2S^2\frac{\partial^2 V}{\partial S^2}\right)dt$

3. 无风险条件: $\frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2S^2\frac{\partial^2 V}{\partial S^2} = r(V - \Delta S) = rV - rS\frac{\partial V}{\partial S}$

**定理 2.2** (Black-Scholes PDE):
$$\frac{\partial V}{\partial t} + rS\frac{\partial V}{\partial S} + \frac{1}{2}\sigma^2S^2\frac{\partial^2 V}{\partial S^2} - rV = 0$$

边界条件: $V(T, S) = (S - K)^+$

### 2.4 Black-Scholes公式

**定理 2.3** (欧式看涨期权定价公式):
$$C(S, t) = S_0 N(d_1) - Ke^{-rT}N(d_2)$$

其中:
$$d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2)T}{\sigma\sqrt{T}}$$
$$d_2 = d_1 - \sigma\sqrt{T}$$

$N(\cdot)$ 是标准正态CDF。

**欧式看跌期权** (由看跌-看涨平价):
$$P = C - S_0 + Ke^{-rT} = Ke^{-rT}N(-d_2) - S_0N(-d_1)$$

## 3. 算法与方法

### 3.1 蒙特卡洛模拟

**算法**: 风险中性定价

```
对于每个路径 i = 1, ..., N:
    S_T^i = S_0 * exp((r - σ²/2)T + σ√T * Z_i)
    payoff_i = max(S_T^i - K, 0)

期权价格 = e^(-rT) * mean(payoff)
```

**误差分析**: 蒙特卡洛误差 $O(1/\sqrt{N})$

### 3.2 二叉树方法

**CRR模型参数**:
$$u = e^{\sigma\sqrt{\Delta t}}, \quad d = e^{-\sigma\sqrt{\Delta t}}, \quad p = \frac{e^{r\Delta t} - d}{u - d}$$

**向后归纳**:
$$V_{i,j} = e^{-r\Delta t}(pV_{i+1,j+1} + (1-p)V_{i+1,j})$$

### 3.3 有限差分法

**Crank-Nicolson格式**:
$$\frac{V_j^{n+1} - V_j^n}{\Delta t} = \frac{1}{2}(\mathcal{L}V_j^{n+1} + \mathcal{L}V_j^n)$$

无条件稳定，精度 $O(\Delta t^2, \Delta S^2)$

## 4. 具体案例

### 4.1 案例：欧式看涨期权定价

**参数**:

- 标的现价 $S_0 = 100$
- 执行价 $K = 100$
- 到期时间 $T = 1$ 年
- 无风险利率 $r = 0.05$
- 波动率 $\sigma = 0.2$

**计算**:
$$d_1 = \frac{\ln(1) + (0.05 + 0.02)}{0.2} = 0.35$$
$$d_2 = 0.35 - 0.2 = 0.15$$
$$N(d_1) = 0.6368, \quad N(d_2) = 0.5596$$
$$C = 100 \times 0.6368 - 100e^{-0.05} \times 0.5596 = 10.45$$

### 4.2 希腊字母计算

| 希腊字母 | 公式 | 数值 | 含义 |
|----------|------|------|------|
| Delta ($\Delta$) | $\frac{\partial C}{\partial S} = N(d_1)$ | 0.637 | 股价变化1元，期权变化0.637元 |
| Gamma ($\Gamma$) | $\frac{\partial^2 C}{\partial S^2} = \frac{N'(d_1)}{S\sigma\sqrt{T}}$ | 0.019 | Delta对股价的敏感度 |
| Vega | $\frac{\partial C}{\partial \sigma} = S\sqrt{T}N'(d_1)$ | 39.9 | 波动率变化1%，期权变化0.399元 |
| Theta ($\Theta$) | $-\frac{\partial C}{\partial t}$ | -6.4 | 时间衰减(每日) |
| Rho ($\rho$) | $\frac{\partial C}{\partial r} = KTe^{-rT}N(d_2)$ | 53.2 | 利率变化1%，期权变化0.532元 |

## 5. 代码实现

### 5.1 Black-Scholes解析解

```python
import numpy as np
from scipy.stats import norm

def black_scholes_call(S, K, T, r, sigma):
    """
    Black-Scholes欧式看涨期权定价

    参数:
        S: 标的资产现价
        K: 执行价格
        T: 到期时间（年）
        r: 无风险利率
        sigma: 波动率
    """
    if T <= 0:
        return max(S - K, 0)

    d1 = (np.log(S / K) + (r + 0.5 * sigma**2) * T) / (sigma * np.sqrt(T))
    d2 = d1 - sigma * np.sqrt(T)

    call_price = S * norm.cdf(d1) - K * np.exp(-r * T) * norm.cdf(d2)

    return call_price

def black_scholes_put(S, K, T, r, sigma):
    """Black-Scholes欧式看跌期权定价（看跌-看涨平价）"""
    call = black_scholes_call(S, K, T, r, sigma)
    return call - S + K * np.exp(-r * T)

def greeks(S, K, T, r, sigma):
    """计算希腊字母"""
    d1 = (np.log(S / K) + (r + 0.5 * sigma**2) * T) / (sigma * np.sqrt(T))
    d2 = d1 - sigma * np.sqrt(T)

    delta = norm.cdf(d1)
    gamma = norm.pdf(d1) / (S * sigma * np.sqrt(T))
    vega = S * norm.pdf(d1) * np.sqrt(T)
    theta = -(S * norm.pdf(d1) * sigma) / (2 * np.sqrt(T)) - r * K * np.exp(-r * T) * norm.cdf(d2)
    rho = K * T * np.exp(-r * T) * norm.cdf(d2)

    return {
        'delta': delta,
        'gamma': gamma,
        'vega': vega,
        'theta': theta,
        'rho': rho
    }

# 测试
S, K, T, r, sigma = 100, 100, 1.0, 0.05, 0.2

print("=" * 50)
print("Black-Scholes期权定价")
print("=" * 50)
print(f"参数: S={S}, K={K}, T={T}, r={r}, σ={sigma}")
print(f"\n看涨期权价格: {black_scholes_call(S, K, T, r, sigma):.4f}")
print(f"看跌期权价格: {black_scholes_put(S, K, T, r, sigma):.4f}")

print("\n希腊字母:")
gr = greeks(S, K, T, r, sigma)
for name, value in gr.items():
    print(f"  {name}: {value:.4f}")
```

### 5.2 蒙特卡洛模拟

```python
def monte_carlo_option(S, K, T, r, sigma, n_simulations=100000, option_type='call'):
    """
    蒙特卡洛期权定价
    """
    np.random.seed(42)

    # 生成随机路径
    Z = np.random.standard_normal(n_simulations)
    S_T = S * np.exp((r - 0.5 * sigma**2) * T + sigma * np.sqrt(T) * Z)

    # 计算收益
    if option_type == 'call':
        payoffs = np.maximum(S_T - K, 0)
    else:
        payoffs = np.maximum(K - S_T, 0)

    # 贴现
    option_price = np.exp(-r * T) * np.mean(payoffs)

    # 标准误差
    std_error = np.exp(-r * T) * np.std(payoffs) / np.sqrt(n_simulations)

    return option_price, std_error

# 比较解析解与蒙特卡洛
mc_price, mc_error = monte_carlo_option(S, K, T, r, sigma)
bs_price = black_scholes_call(S, K, T, r, sigma)

print("\n" + "=" * 50)
print("解析解 vs 蒙特卡洛")
print("=" * 50)
print(f"Black-Scholes: {bs_price:.4f}")
print(f"Monte Carlo:   {mc_price:.4f} ± {mc_error:.4f}")
print(f"相对误差: {abs(mc_price - bs_price) / bs_price * 100:.2f}%")
```
