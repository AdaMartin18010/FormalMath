## 一、随机微积分

### 1.1 布朗运动

**定义 1.1** (标准布朗运动)
随机过程 $\{W_t\}_{t \geq 0}$ 称为标准布朗运动，如果满足：

1. $W_0 = 0$（几乎必然）
2. 独立增量：对 $0 \leq s < t$，$W_t - W_s$ 独立于历史 $\{W_u: 0 \leq u \leq s\}$
3. 平稳增量：$W_t - W_s \sim N(0, t-s)$
4. 路径连续性：$t \mapsto W_t$ 几乎必然连续

**性质**:

- $E[W_t] = 0$, $\text{Var}(W_t) = t$
- 自相似性：对 $c > 0$，$\{W_{ct}\} \stackrel{d}{=} \{\sqrt{c}W_t\}$
- 二次变差：$\langle W \rangle_t = t$

### 1.2 伊藤积分

**定义 1.2** (伊藤积分)
设 $\{H_t\}$ 为适应过程且满足可积条件 $\int_0^T H_s^2 ds < \infty$，则伊藤积分定义为：

$$I_t = \int_0^t H_s dW_s = \lim_{n \to \infty} \sum_{i=0}^{n-1} H_{t_i}(W_{t_{i+1}} - W_{t_i})$$

**性质**:

1. **鞅性**: $I_t$ 是连续鞅
2. **等距性**: $E\left[\left(\int_0^t H_s dW_s\right)^2\right] = E\left[\int_0^t H_s^2 ds\right]$
3. **零均值**: $E\left[\int_0^t H_s dW_s\right] = 0$

### 1.3 伊藤引理

**定理 1.1** (伊藤引理)
设 $X_t$ 满足随机微分方程：

$$dX_t = \mu(X_t, t)dt + \sigma(X_t, t)dW_t$$

若 $f \in C^{2,1}(\mathbb{R} \times [0, T])$，则：

$$df(X_t, t) = \frac{\partial f}{\partial t}dt + \frac{\partial f}{\partial x}dX_t + \frac{1}{2}\frac{\partial^2 f}{\partial x^2}(dX_t)^2$$

其中 $(dX_t)^2 = \sigma^2 dt$（伊藤等距）。

**多维形式**：
对 $dX_t^i = \mu_i dt + \sum_{j=1}^d \sigma_{ij} dW_t^j$：

$$df = \frac{\partial f}{\partial t}dt + \sum_{i=1}^n \frac{\partial f}{\partial x_i}dX_t^i + \frac{1}{2}\sum_{i,j=1}^n \frac{\partial^2 f}{\partial x_i \partial x_j}dX_t^i dX_t^j$$

### 1.4 几何布朗运动

**定义 1.3** (几何布朗运动)
随机过程 $\{S_t\}$ 满足：

$$dS_t = \mu S_t dt + \sigma S_t dW_t$$

或等价地：

$$S_t = S_0 \exp\left[\left(\mu - \frac{\sigma^2}{2}\right)t + \sigma W_t\right]$$

**性质**:

- 解存在且唯一（Lipschitz条件满足）
- 价格非负（$S_t > 0$ 若 $S_0 > 0$）
- 对数正态分布：$\ln S_t \sim N(\ln S_0 + (\mu - \sigma^2/2)t, \sigma^2 t)$

## 二、期权定价基础

### 2.1 期权定义

**定义 2.1** (欧式期权)
欧式看涨期权赋予持有者在到期日 $T$ 以执行价 $K$ 购买标的资产的权利：

$$C_T = \max(S_T - K, 0) = (S_T - K)^+$$

欧式看跌期权：

$$P_T = \max(K - S_T, 0) = (K - S_T)^+$$

**美式期权**: 可在到期前任何时间行权。

### 2.2 无套利原理

**定义 2.2** (套利机会)
套利策略满足：

1. 初始成本为零或负：$V_0 \leq 0$
2. 未来价值非负：$P(V_T \geq 0) = 1$
3. 正收益概率为正：$P(V_T > 0) > 0$

**一价定律**: 若两个投资组合在所有状态下的未来价值相同，则当前价格相同。

### 2.3 风险中性定价

**基本思想**: 在风险中性测度 $\mathbb{Q}$ 下，所有资产的期望收益率等于无风险利率 $r$：

$$\frac{dS_t}{S_t} = r dt + \sigma d\tilde{W}_t$$

其中 $\tilde{W}_t$ 是 $\mathbb{Q}$-布朗运动。

**期权定价公式**: 在 $\mathbb{Q}$ 下，衍生品价格为其折现期望收益：

$$V_t = e^{-r(T-t)} \mathbb{E}^{\mathbb{Q}}[V_T | \mathcal{F}_t]$$

### 2.4 希腊字母

**定义 2.3** (风险参数/Greeks)

| 希腊字母 | 定义 | 含义 |
|---------|------|------|
| Delta ($\Delta$) | $\frac{\partial V}{\partial S}$ | 价格对标的资产价格的敏感度 |
| Gamma ($\Gamma$) | $\frac{\partial^2 V}{\partial S^2}$ | Delta的变化率 |
| Theta ($\Theta$) | $\frac{\partial V}{\partial t}$ | 时间衰减 |
| Vega | $\frac{\partial V}{\partial \sigma}$ | 波动率敏感度 |
| Rho ($\rho$) | $\frac{\partial V}{\partial r}$ | 利率敏感度 |

## 三、风险中性测度

### 3.1 等价鞅测度

**定义 3.1** (等价鞅测度)
测度 $\mathbb{Q}$ 称为等价鞅测度（风险中性测度），如果：

1. 等价性：$\mathbb{Q} \sim \mathbb{P}$（具有相同的零测集）
2. 鞅性：折现价格过程 $\tilde{S}_t = e^{-rt}S_t$ 是 $\mathbb{Q}$-鞅

### 3.2 Girsanov定理

**定理 3.1** (Girsanov定理)
设 $W_t$ 是 $\mathbb{P}$-布朗运动，$\theta_t$ 满足Novikov条件：

$$\mathbb{E}\left[\exp\left(\frac{1}{2}\int_0^T \theta_s^2 ds\right)\right] < \infty$$

定义：

$$Z_t = \exp\left(-\int_0^t \theta_s dW_s - \frac{1}{2}\int_0^t \theta_s^2 ds\right)$$

$$d\mathbb{Q} = Z_T d\mathbb{P}$$

则在 $\mathbb{Q}$ 下，$\tilde{W}_t = W_t + \int_0^t \theta_s ds$ 是布朗运动。

**金融应用**: 取 $\theta = \frac{\mu - r}{\sigma}$（风险的市场价格），则将真实测度转换为风险中性测度。

### 3.3 完全市场

**定义 3.2** (完全市场)
市场称为完全的，如果每个或有索取权（contingent claim）都可被复制。

**定理 3.2** (完全市场与唯一性)
若市场无套利且完全，则等价鞅测度存在且唯一。

**复制策略**: 对期权支付 $V_T$，存在自融资策略 $(\phi_t, \psi_t)$ 使得：

$$V_T = \phi_T S_T + \psi_T B_T$$

其中 $B_t = e^{rt}$ 是货币市场账户。

## 四、利率模型

### 4.1 短期利率模型

**定义 4.1** (短期利率)
瞬时无风险利率 $r_t$ 满足随机微分方程。

**Vasicek模型**:

$$dr_t = a(b - r_t)dt + \sigma dW_t$$

其中 $a > 0$（均值回归速度），$b$（长期均值），$\sigma > 0$（波动率）。

**性质**:

- 均值回归：利率趋向长期均值 $b$
- 正态分布：$r_t \sim N(r_0 e^{-at} + b(1 - e^{-at}), \frac{\sigma^2}{2a}(1 - e^{-2at}))$
- 可能为负（缺点）

**Cox-Ingersoll-Ross (CIR)模型**:

$$dr_t = a(b - r_t)dt + \sigma\sqrt{r_t} dW_t$$

- 保持利率非负（若 $2ab \geq \sigma^2$）
- 非中心卡方分布

### 4.2 零息债券定价

**定义 4.2** (零息债券)
在 $T$ 时刻支付1美元的债券，其在 $t$ 时刻的价格为：

$$P(t, T) = \mathbb{E}^{\mathbb{Q}}\left[\exp\left(-\int_t^T r_s ds\right) | \mathcal{F}_t\right]$$

**仿射期限结构**:
若 $P(t, T) = \exp(A(t, T) - B(t, T)r_t)$，则称为仿射模型。

Vasicek模型的解：

$$B(t, T) = \frac{1 - e^{-a(T-t)}}{a}$$

$$A(t, T) = \left(b - \frac{\sigma^2}{2a^2}\right)(B(t, T) - (T-t)) - \frac{\sigma^2 B(t, T)^2}{4a}$$

### 4.3 HJM框架

**Heath-Jarrow-Morton框架**: 直接对瞬时远期利率 $f(t, T)$ 建模：

$$df(t, T) = \alpha(t, T)dt + \sigma(t, T)dW_t$$

**无套利条件** (HJM漂移条件)：

$$\alpha(t, T) = \sigma(t, T) \int_t^T \sigma(t, u)du$$

漂移由波动率结构完全确定，保证无套利。

## 总结

本章节介绍了金融数学的基础概念：

- **随机微积分**: 布朗运动、伊藤积分、伊藤引理、几何布朗运动

- **风险中性测度**: 等价鞅测度、Girsanov定理、完全市场理论
- **利率模型**: Vasicek模型、CIR模型、零息债券定价、HJM框架

这些概念构成了金融衍生品定价的理论基础。
