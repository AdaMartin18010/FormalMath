## 一、Black-Scholes公式

### 1.1 Black-Scholes-Merton偏微分方程

**定理 1.1** (Black-Scholes PDE)
假设标的资产价格服从几何布朗运动：

$$dS_t = \mu S_t dt + \sigma S_t dW_t$$

则无套利条件下，衍生品价格 $V(S, t)$ 满足：

$$\frac{\partial V}{\partial t} + rS\frac{\partial V}{\partial S} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} - rV = 0$$

**边界条件**: 取决于衍生品类型（如 $V(S, T) = (S - K)^+$ 对看涨期权）。

**证明概要**:

1. **构建组合**: 持有期权空头和 $\Delta = \frac{\partial V}{\partial S}$ 份标的资产多头
2. **组合价值**: $\Pi = -V + \Delta S$
3. **组合变化**: 应用伊藤引理，随机项抵消：
   $$d\Pi = -\left(\frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2}\right)dt$$
4. **无套利**: $d\Pi = r\Pi dt = r(-V + \Delta S)dt$
5. **整理得PDE** ∎

### 1.2 欧式看涨期权定价公式

**定理 1.2** (Black-Scholes公式)
欧式看涨期权的价格为：

$$C(S, t) = S\Phi(d_1) - Ke^{-r(T-t)}\Phi(d_2)$$

其中：

$$d_1 = \frac{\ln(S/K) + (r + \sigma^2/2)(T-t)}{\sigma\sqrt{T-t}}$$

$$d_2 = d_1 - \sigma\sqrt{T-t} = \frac{\ln(S/K) + (r - \sigma^2/2)(T-t)}{\sigma\sqrt{T-t}}$$

$\Phi(\cdot)$ 是标准正态分布的累积分布函数。

**证明**:

**方法1：解PDE**

对看涨期权，边界条件 $V(S, T) = (S - K)^+$。通过变量替换将Black-Scholes PDE转化为热方程求解。

变量替换：
- $\tau = T - t$（时间反向）
- $x = \ln(S/K)$
- $V = Ku(x, \tau)$

变换后方程：
$$\frac{\partial u}{\partial \tau} = \frac{\sigma^2}{2}\frac{\partial^2 u}{\partial x^2} + \left(r - \frac{\sigma^2}{2}\right)\frac{\partial u}{\partial x} - ru$$

进一步令 $u = e^{\alpha x + \beta \tau}v$，选择适当 $\alpha, \beta$ 消去一阶项和零阶项，得到标准热方程。

**方法2：风险中性定价**

在风险中性测度 $\mathbb{Q}$ 下，$S_t$ 服从：

$$dS_t = rS_t dt + \sigma S_t d\tilde{W}_t$$

因此：

$$C(S, t) = e^{-r(T-t)} \mathbb{E}^{\mathbb{Q}}[(S_T - K)^+ | S_t = S]$$

计算期望：

$$S_T = S \exp\left[(r - \sigma^2/2)(T-t) + \sigma(\tilde{W}_T - \tilde{W}_t)\right]$$

令 $Y = \sigma(\tilde{W}_T - \tilde{W}_t) \sim N(0, \sigma^2(T-t))$，则：

$$\mathbb{E}[(S_T - K)^+] = \int_{\ln(K/S)}^{\infty} (Se^{(r-\sigma^2/2)(T-t)+y} - K) \frac{e^{-y^2/(2\sigma^2(T-t))}}{\sqrt{2\pi\sigma^2(T-t)}} dy$$

经过计算可得：

$$C(S, t) = S\Phi(d_1) - Ke^{-r(T-t)}\Phi(d_2)$$ ∎

### 1.3 欧式看跌期权定价公式

**定理 1.3** (Black-Scholes看跌期权公式)
欧式看跌期权的价格为：

$$P(S, t) = Ke^{-r(T-t)}\Phi(-d_2) - S\Phi(-d_1)$$

或等价地：

$$P(S, t) = S(\Phi(d_1) - 1) - Ke^{-r(T-t)}(\Phi(d_2) - 1)$$

**证明**: 由看跌-看涨平价关系或直接计算期望。

### 1.4 Black-Scholes公式中的希腊字母

**定理 1.4** (看涨期权的Greeks)

**Delta**:
$$\Delta = \frac{\partial C}{\partial S} = \Phi(d_1)$$

**Gamma**:
$$\Gamma = \frac{\partial^2 C}{\partial S^2} = \frac{\phi(d_1)}{S\sigma\sqrt{T-t}}$$

**Theta**:
$$\Theta = \frac{\partial C}{\partial t} = -\frac{S\phi(d_1)\sigma}{2\sqrt{T-t}} - rKe^{-r(T-t)}\Phi(d_2)$$

**Vega**:
$$\text{Vega} = \frac{\partial C}{\partial \sigma} = S\phi(d_1)\sqrt{T-t}$$

**Rho**:
$$\rho = \frac{\partial C}{\partial r} = K(T-t)e^{-r(T-t)}\Phi(d_2)$$

其中 $\phi(x) = \frac{1}{\sqrt{2\pi}}e^{-x^2/2}$ 是标准正态密度函数。

## 二、鞅表示定理

### 2.1 鞅表示定理

**定理 2.1** (鞅表示定理)
设 $W_t$ 是概率空间 $(\Omega, \mathcal{F}, \mathbb{P})$ 上的布朗运动，$\mathcal{F}_t = \sigma(W_s: 0 \leq s \leq t)$。若 $M_t$ 是平方可积鞅（相对于 $\mathcal{F}_t$），则存在唯一的适应过程 $\phi_t$ 满足：

$$M_t = M_0 + \int_0^t \phi_s dW_s$$

或微分形式：

$$dM_t = \phi_t dW_t$$

**金融意义**: 任何鞅都可以表示为布朗运动的伊藤积分，即可以通过适当的交易策略复制。

### 2.2 在衍生品定价中的应用

**定理 2.2** (完全市场的复制)
在Black-Scholes模型中，任何欧式衍生品都可被复制。

**证明**: 设衍生品在 $T$ 时刻支付 $V_T$，定义：

$$M_t = \mathbb{E}^{\mathbb{Q}}[e^{-rT}V_T | \mathcal{F}_t]$$

则 $M_t$ 是 $\mathbb{Q}$-鞅。由鞅表示定理：

$$M_t = M_0 + \int_0^t \phi_s d\tilde{W}_s$$

令 $\Delta_t = \frac{\phi_t}{\sigma \tilde{S}_t}$，其中 $\tilde{S}_t = e^{-rt}S_t$ 是折现价格。

构造组合：持有 $\Delta_t$ 份标的资产和 $\psi_t$ 份债券，使得：

$$\Delta_t S_t + \psi_t B_t = e^{rt}M_t = V_t$$

验证自融资条件：

$$dV_t = \Delta_t dS_t + \psi_t dB_t$$

因此 $(\Delta_t, \psi_t)$ 是复制策略。∎

## 三、Feynman-Kac公式

### 3.1 Feynman-Kac表示

**定理 3.1** (Feynman-Kac公式)
设 $u(x, t)$ 满足偏微分方程：

$$\frac{\partial u}{\partial t} + \mu(x, t)\frac{\partial u}{\partial x} + \frac{1}{2}\sigma^2(x, t)\frac{\partial^2 u}{\partial x^2} - r(x, t)u = 0$$

边界条件 $u(x, T) = g(x)$。

则解可表示为：

$$u(x, t) = \mathbb{E}\left[\exp\left(-\int_t^T r(X_s, s)ds\right)g(X_T) | X_t = x\right]$$

其中 $X_s$ 满足：

$$dX_s = \mu(X_s, s)ds + \sigma(X_s, s)dW_s$$

**金融解释**: 衍生品价格 = 折现期望收益，其中折现因子包含瞬时利率。

### 3.2 与Black-Scholes的关系

**推论 3.1**: Black-Scholes PDE是Feynman-Kac公式的特例。

取：
- $\mu(x, t) = rx$
- $\sigma(x, t) = \sigma x$
- $r(x, t) = r$
- $g(x) = (x - K)^+$

则解：

$$C(S, t) = e^{-r(T-t)} \mathbb{E}^{\mathbb{Q}}[(S_T - K)^+ | S_t = S]$$

这正是风险中性定价公式。

## 四、期权平价关系

### 4.1 看跌-看涨平价

**定理 4.1** (Put-Call Parity)
在无套利条件下，欧式看涨期权和看跌期权满足：

$$C - P = S - Ke^{-r(T-t)}$$

其中 $C$ 是看涨期权价格，$P$ 是看跌期权价格，$S$ 是标的资产现价，$K$ 是执行价，$r$ 是无风险利率，$T-t$ 是到期时间。

**证明**:

考虑两个投资组合：

**组合A**: 一份看涨期权 + $Ke^{-r(T-t)}$ 现金
**组合B**: 一份看跌期权 + 一份标的资产

在到期日 $T$：

- 若 $S_T \geq K$：
  - 组合A：$(S_T - K) + K = S_T$
  - 组合B：$0 + S_T = S_T$

- 若 $S_T < K$：
  - 组合A：$0 + K = K$
  - 组合B：$(K - S_T) + S_T = K$

两组合在 $T$ 时刻价值相同，由一价定律，当前价值相同：

$$C + Ke^{-r(T-t)} = P + S$$

整理得：$C - P = S - Ke^{-r(T-t)}$ ∎

### 4.2 应用

**验证定价一致性**: 若Black-Scholes公式正确，则：

$$C - P = S\Phi(d_1) - Ke^{-r(T-t)}\Phi(d_2) - Ke^{-r(T-t)}\Phi(-d_2) + S\Phi(-d_1)$$

$$= S(\Phi(d_1) + \Phi(-d_1)) - Ke^{-r(T-t)}(\Phi(d_2) + \Phi(-d_2))$$

$$= S - Ke^{-r(T-t)}$$

满足平价关系。

**套利策略**: 若市场违反平价关系，可以构建无风险套利组合。

## 总结

本章节介绍了金融数学的核心定理：

- **Black-Scholes公式**: 欧式期权定价的闭式解，基于PDE和风险中性定价两种推导
- **鞅表示定理**: 完全市场中任意衍生品可被复制，保证定价的唯一性
- **Feynman-Kac公式**: 连接PDE和随机表示，统一了偏微分方程和概率方法
- **期权平价关系**: 看涨看跌期权之间的无套利关系

这些定理构成了现代金融工程的理论基石。
