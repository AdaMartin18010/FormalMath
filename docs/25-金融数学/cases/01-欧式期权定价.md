# 欧式期权定价与对冲案例

## 问题描述

某交易员需要为以下欧式看涨期权定价和对冲：

| 参数 | 数值 |
|------|------|
| 标的资产现价 $S_0$ | 100 USD |
| 执行价格 $K$ | 105 USD |
| 到期时间 $T$ | 6个月 (0.5年) |
| 无风险利率 $r$ | 5% (年化) |
| 波动率 $\sigma$ | 25% |

## 定价分析

### Black-Scholes解析解

```python
import numpy as np
from scipy.stats import norm

def black_scholes_call(S, K, T, r, sigma):
    d1 = (np.log(S/K) + (r + 0.5*sigma**2)*T) / (sigma*np.sqrt(T))
    d2 = d1 - sigma*np.sqrt(T)
    return S*norm.cdf(d1) - K*np.exp(-r*T)*norm.cdf(d2)

# 参数
S, K, T, r, sigma = 100, 105, 0.5, 0.05, 0.25

# 定价
call_price = black_scholes_call(S, K, T, r, sigma)
print(f"欧式看涨期权价格: ${call_price:.4f}")

# 希腊字母
d1 = (np.log(S/K) + (r + 0.5*sigma**2)*T) / (sigma*np.sqrt(T))
d2 = d1 - sigma*np.sqrt(T)

delta = norm.cdf(d1)
gamma = norm.pdf(d1) / (S * sigma * np.sqrt(T))
vega = S * norm.pdf(d1) * np.sqrt(T)
theta = -(S*norm.pdf(d1)*sigma)/(2*np.sqrt(T)) - r*K*np.exp(-r*T)*norm.cdf(d2)
rho = K*T*np.exp(-r*T)*norm.cdf(d2)

print(f"\n希腊字母:")
print(f"  Delta: {delta:.4f}")
print(f"  Gamma: {gamma:.4f}")
print(f"  Vega:  {vega:.4f}")
print(f"  Theta: {theta:.4f} (每日)")
print(f"  Rho:   {rho:.4f}")
```

### 蒙特卡洛模拟验证

```python
def monte_carlo_option(S, K, T, r, sigma, n_paths=100000):
    np.random.seed(42)
    Z = np.random.standard_normal(n_paths)
    S_T = S * np.exp((r - 0.5*sigma**2)*T + sigma*np.sqrt(T)*Z)
    payoffs = np.maximum(S_T - K, 0)
    return np.exp(-r*T) * np.mean(payoffs)

mc_price = monte_carlo_option(S, K, T, r, sigma)
print(f"\n蒙特卡洛价格: ${mc_price:.4f}")
print(f"与解析解差异: {abs(mc_price - call_price)*100:.2f}美分")
```

## 动态对冲策略

### Delta对冲

```python
def delta_hedge_simulation(S0, K, T, r, sigma, n_steps=100, n_sims=1000):
    dt = T / n_steps
    hedge_pnl = []
    
    for _ in range(n_sims):
        S = S0
        portfolio_value = 0
        shares_held = 0
        
        for t in np.linspace(0, T, n_steps):
            # 计算当前Delta
            tau = T - t
            if tau > 0:
                d1 = (np.log(S/K) + (r + 0.5*sigma**2)*tau) / (sigma*np.sqrt(tau))
                delta = norm.cdf(d1)
            else:
                delta = 1 if S > K else 0
            
            # 调整持仓
            shares_to_buy = delta - shares_held
            portfolio_value -= shares_to_buy * S
            shares_held = delta
            
            # 标的资产价格演化
            S *= np.exp((r - 0.5*sigma**2)*dt + sigma*np.sqrt(dt)*np.random.normal())
        
        # 到期结算
        payoff = max(S - K, 0)
        portfolio_value += shares_held * S - payoff
        hedge_pnl.append(portfolio_value)
    
    return np.array(hedge_pnl)

pnl = delta_hedge_simulation(S, K, T, r, sigma)
print(f"\nDelta对冲P&L统计:")
print(f"  均值: ${np.mean(pnl):.4f}")
print(f"  标准差: ${np.std(pnl):.4f}")
print(f"  95% VaR: ${np.percentile(pnl, 5):.4f}")
```

## 结果分析

| 指标 | 数值 | 解释 |
|------|------|------|
| **期权价格** | $7.46 | 理论公平价格 |
| **Delta** | 0.5425 | 每1美元股价上涨，期权涨0.54美元 |
| **Gamma** | 0.0195 | Delta对股价的敏感度 |
| **Vega** | 27.23 | 波动率每增1%，期权价值增0.27美元 |
| **Theta** | -0.013 | 每日时间衰减0.013美元 |
| **对冲成本** | ~0 | Delta对冲可消除方向性风险 |

## 交易建议

1. **定价**: 若市场价格偏离$7.46，存在套利机会
2. **对冲**: 每卖出1份期权，需买入0.54股标的股票对冲
3. **风险管理**: 关注Gamma和Vega风险，定期进行再平衡
4. **波动率交易**: 若预期实际波动率高于25%，可买入期权
