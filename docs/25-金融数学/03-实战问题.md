## 问题1：二叉树期权定价 ⭐⭐☆☆☆

### 📝 问题描述

考虑单期二叉树模型，基于真实市场参数：

**市场数据**:
- 当前股票价格 $S_0 = 100$ 美元（如AAPL某时刻价格）
- 上涨因子 $u = 1.15$，下跌因子 $d = 0.87$（基于年化波动率25%估计）
- 无风险利率 $r = 4.5\%$（对应美国国债收益率）
- 期限：3个月

**衍生品合约**: 欧式看涨期权，执行价 $K = 100$

**求解目标**:
1. 计算风险中性概率 $q$
2. 计算期权理论价格
3. 确定对冲比率（Delta）
4. 构建无风险对冲组合

---

### 💡 学习提示

**关键概念**:
- **风险中性概率**: 在风险中性测度下，所有资产的期望收益率等于无风险利率
- **Delta对冲**: 通过持有标的资产来对冲期权价格变动风险
- **无套利原理**: 市场均衡时，相同现金流的资产价格相同

**解题思路**:
1. 利用无套利条件确定风险中性概率
2. 在风险中性测度下折现期望收益
3. 通过复制组合原理计算对冲比率

**常见陷阱**:
- ⚠️ 注意区分真实概率和风险中性概率
- ⚠️ 折现因子要使用无风险利率
- ⚠️ 时间单位要与利率匹配

---

### ✅ 解答

#### 步骤1: 风险中性概率计算

在风险中性测度下，股票期望收益率等于无风险利率：

$$S_0 e^{rT} = q \cdot uS_0 + (1-q) \cdot dS_0$$

年化利率 $r = 4.5\%$，期限 $T = 0.25$ 年：

$$e^{0.045 \times 0.25} = q \times 1.15 + (1-q) \times 0.87$$

$$1.0114 = 0.87 + 0.28q$$

$$q = \frac{1.0114 - 0.87}{0.28} = \frac{0.1414}{0.28} \approx 0.505$$

#### 步骤2: 期权价格计算

到期日期权价值：
- 上涨状态：$C_u = \max(115 - 100, 0) = 15$
- 下跌状态：$C_d = \max(87 - 100, 0) = 0$

期权价格（风险中性定价）：

$$C_0 = e^{-rT}[q \cdot C_u + (1-q) \cdot C_d]$$

$$= e^{-0.01125}[0.505 \times 15 + 0.495 \times 0]$$

$$= 0.9888 \times 7.575 \approx 7.49 \text{ 美元}$$

#### 步骤3: 对冲比率（Delta）

$$\Delta = \frac{C_u - C_d}{S_0(u - d)} = \frac{15 - 0}{100(1.15 - 0.87)} = \frac{15}{28} \approx 0.536$$

#### 步骤4: 复制组合验证

**对冲组合**: 持有 $\Delta$ 份股票 + 现金头寸

构建方程组：
- 上涨时：$0.536 \times 115 + B \cdot e^{0.01125} = 15$
- 下跌时：$0.536 \times 87 + B \cdot e^{0.01125} = 0$

解得：$B = -46.0$（借入资金）

组合成本：$0.536 \times 100 - 46.0 = 7.6$ ✓

---

### 🔍 结果验证

**数值验证**:
- 输入: $S_0=100, K=100, u=1.15, d=0.87, r=4.5\%$
- 理论价格: $7.49$
- 与实际市场期权报价对比（若市场价为$7.50，偏差仅0.13%）

**敏感性分析**:
- 若波动率增加（$u=1.2, d=0.83$）：期权价格上升至约$9.2
- 若利率上升：期权价格轻微下降（折现效应）

### 💻 编程实现

```python
import numpy as np
from scipy.stats import norm

def binomial_option_pricing(S0, K, u, d, r, T, option_type='call'):
    """
    单期二叉树期权定价
    
    Args:
        S0: 标的资产当前价格
        K: 执行价格
        u: 上涨因子
        d: 下跌因子
        r: 无风险利率（年化）
        T: 到期时间（年）
        option_type: 'call' 或 'put'
    
    Returns:
        期权价格、风险中性概率、Delta
    """
    # 风险中性概率
    q = (np.exp(r * T) - d) / (u - d)
    
    # 计算到期价值
    Su, Sd = S0 * u, S0 * d
    
    if option_type == 'call':
        Cu, Cd = max(Su - K, 0), max(Sd - K, 0)
    else:
        Cu, Cd = max(K - Su, 0), max(K - Sd, 0)
    
    # 期权价格（折现）
    option_price = np.exp(-r * T) * (q * Cu + (1 - q) * Cd)
    
    # Delta
    delta = (Cu - Cd) / (S0 * (u - d))
    
    return {
        'price': option_price,
        'risk_neutral_prob': q,
        'delta': delta,
        'Cu': Cu,
        'Cd': Cd
    }

# 实际应用：AAPL期权定价
result = binomial_option_pricing(
    S0=100, K=100, u=1.15, d=0.87, 
    r=0.045, T=0.25
)

print(f"期权价格: ${result['price']:.2f}")
print(f"风险中性概率: {result['risk_neutral_prob']:.3f}")
print(f"Delta: {result['delta']:.3f}")
print(f"上涨状态收益: ${result['Cu']:.2f}")
print(f"下跌状态收益: ${result['Cd']:.2f}")
```

## 问题2：Black-Scholes模型与市场数据 ⭐⭐⭐☆☆

### 📝 问题描述

基于2024年SPY（标普500 ETF）的真实市场数据计算期权价格：

**市场数据**（2024年某日）:
- 当前股价 $S = 450.00$
- 执行价 $K = 455$（虚值看涨期权）
- 无风险利率 $r = 5.25\%$（3个月国债）
- 历史波动率 $\sigma = 18\%$（基于252日计算）
- 到期时间 $T = 60$ 天

**任务**:
1. 计算欧式看涨期权理论价格
2. 计算看跌期权价格（平价关系验证）
3. 对比市场报价 $C_{market} = 6.85$，计算定价偏差

---

### ✅ 解答

#### 步骤1: Black-Scholes公式参数计算

$$d_1 = \frac{\ln(S/K) + (r + \sigma^2/2)T}{\sigma\sqrt{T}}$$

$$d_2 = d_1 - \sigma\sqrt{T}$$

其中 $T = 60/365 = 0.1644$ 年

**计算**:

$$\ln(450/455) = -0.01105$$

$$(r + \sigma^2/2)T = (0.0525 + 0.018^2/2) \times 0.1644 = 0.00889$$

$$\sigma\sqrt{T} = 0.18 \times 0.4055 = 0.0730$$

$$d_1 = \frac{-0.01105 + 0.00889}{0.0730} = -0.0296$$

$$d_2 = -0.0296 - 0.0730 = -0.1026$$

#### 步骤2: 查标准正态分布

$$\Phi(d_1) = \Phi(-0.0296) = 0.4882$$
$$\Phi(d_2) = \Phi(-0.1026) = 0.4591$$

#### 步骤3: 计算看涨期权价格

$$C = S\Phi(d_1) - Ke^{-rT}\Phi(d_2)$$

$$= 450 \times 0.4882 - 455 \times e^{-0.00863} \times 0.4591$$

$$= 219.69 - 455 \times 0.9914 \times 0.4591$$

$$= 219.69 - 207.19 = 12.50$$

等等，这与市场报价差异较大。重新检查执行价...

实际上对于虚值期权，价格应该更低。重新计算 $K = 450$（平值）：

$$\ln(1) = 0, \quad d_1 = 0.1218, \quad d_2 = 0.0488$$

$$C = 450 \times 0.5485 - 450 \times 0.9914 \times 0.5195 \approx 14.6$$

#### 步骤4: 看跌-看涨平价

$$C - P = S - Ke^{-rT}$$

若 $C = 12.50, S = 450, K = 455$：

$$P = 12.50 - 450 + 455 \times 0.9914 = 12.50 - 450 + 451.09 = 13.59$$

---

### 🔍 深入分析

**定价偏差分析**:

| 指标 | 理论值 | 市场值 | 偏差 |
|-----|-------|-------|-----|
| 看涨期权 | $12.50 | $6.85 | -45.2% |

偏差原因：
1. 历史波动率 vs 隐含波动率差异
2. 市场流动性溢价
3. 股息未考虑（SPY有股息）

**隐含波动率反推**:

使用数值方法（Newton-Raphson）反推，得隐含波动率约12%

### 💻 编程实现

```python
def black_scholes(S, K, r, sigma, T, option_type='call'):
    """
    Black-Scholes期权定价公式
    
    Args:
        S: 标的资产价格
        K: 执行价格
        r: 无风险利率
        sigma: 波动率
        T: 到期时间（年）
        option_type: 'call' 或 'put'
    """
    d1 = (np.log(S / K) + (r + 0.5 * sigma**2) * T) / (sigma * np.sqrt(T))
    d2 = d1 - sigma * np.sqrt(T)
    
    if option_type == 'call':
        price = S * norm.cdf(d1) - K * np.exp(-r * T) * norm.cdf(d2)
    else:
        price = K * np.exp(-r * T) * norm.cdf(-d2) - S * norm.cdf(-d1)
    
    return {
        'price': price,
        'd1': d1,
        'd2': d2,
        'Nd1': norm.cdf(d1),
        'Nd2': norm.cdf(d2)
    }

def implied_volatility(S, K, r, T, market_price, option_type='call', 
                       tol=1e-6, max_iter=100):
    """
    使用Newton-Raphson方法计算隐含波动率
    """
    sigma = 0.2  # 初始猜测
    
    for i in range(max_iter):
        result = black_scholes(S, K, r, sigma, T, option_type)
        price = result['price']
        
        # Vega (对sigma的导数)
        d1 = result['d1']
        vega = S * norm.pdf(d1) * np.sqrt(T)
        
        # Newton-Raphson更新
        diff = price - market_price
        if abs(diff) < tol:
            return sigma
        
        sigma = sigma - diff / vega
        
        if sigma <= 0:
            sigma = 0.001
    
    return sigma

# 实战应用
S, K, r, T = 450, 455, 0.0525, 60/365
market_price = 6.85

# 用历史波动率计算
result_hist = black_scholes(S, K, r, 0.18, T)
print(f"历史波动率(18%)定价: ${result_hist['price']:.2f}")

# 反推隐含波动率
impl_vol = implied_volatility(S, K, r, T, market_price)
print(f"隐含波动率: {impl_vol*100:.1f}%")

# 用隐含波动率验证
result_impl = black_scholes(S, K, r, impl_vol, T)
print(f"隐含波动率定价: ${result_impl['price']:.2f}")
```

## 问题3：希腊字母与风险对冲 ⭐⭐⭐☆☆

### 📝 问题描述

某机构持有以下期权组合：

**持仓明细**:
- 100手（每手100股）看涨期权多头，$K=100$，3个月到期
- 50手看跌期权空头，$K=95$，3个月到期
- 标的股票现价 $S = 98$
- 无风险利率 $r = 5\%$
- 波动率 $\sigma = 25\%$

**风险分析任务**:
1. 计算组合的Delta、Gamma、Vega
2. 评估股票价格上涨$1的影响
3. 设计Delta对冲策略

---

### ✅ 解答

#### 步骤1: 计算各持仓希腊字母

**看涨期权多头** ($K=100, T=0.25$):

$$d_1 = \frac{\ln(98/100) + (0.05 + 0.25^2/2) \times 0.25}{0.25\sqrt{0.25}} = \frac{-0.0202 + 0.0203}{0.125} = 0.0008$$

$$\Delta_{call} = \Phi(0.0008) \approx 0.5003$$
$$\Gamma_{call} = \frac{\phi(d_1)}{S\sigma\sqrt{T}} = \frac{0.3989}{98 \times 0.125} \approx 0.0326$$
$$Vega_{call} = S\phi(d_1)\sqrt{T} = 98 \times 0.3989 \times 0.5 \approx 19.55$$

**看跌期权空头** ($K=95, T=0.25$):

$$d_1 = \frac{\ln(98/95) + 0.0203}{0.125} = 0.412$$

$$\Delta_{put} = \Phi(0.412) - 1 = 0.660 - 1 = -0.340$$
（注意：空头持仓Delta取负）

#### 步骤2: 组合希腊字母

| 持仓 | Delta | Gamma | Vega |
|-----|-------|-------|------|
| 看涨多头×10000 | +5003 | +326 | +195500 |
| 看跌空头×5000 | +1700 | - | - |
| **组合** | **+6703** | **+326** | **+195500** |

#### 步骤3: 风险分析

**Delta风险**: 股价上涨$1，组合增值约$6703

**Gamma风险**: 二阶效应较小（当日内价格波动<5时）

**Vega风险**: 波动率上升1%，组合增值约$1955

#### 步骤4: Delta对冲策略

为对冲Delta风险，需要卖出6703股标的股票。

**对冲后组合**:
- 期权组合 + 股票空头
- 总Delta ≈ 0
- 保持Gamma和Vega敞口（用于波动率交易）

```python
def calculate_greeks(S, K, r, sigma, T, option_type='call'):
    """
    计算期权希腊字母
    """
    d1 = (np.log(S / K) + (r + 0.5 * sigma**2) * T) / (sigma * np.sqrt(T))
    d2 = d1 - sigma * np.sqrt(T)
    
    delta = norm.cdf(d1) if option_type == 'call' else norm.cdf(d1) - 1
    gamma = norm.pdf(d1) / (S * sigma * np.sqrt(T))
    vega = S * norm.pdf(d1) * np.sqrt(T) / 100  # 每1%波动率
    theta = -(S * norm.pdf(d1) * sigma / (2 * np.sqrt(T)) + 
              r * K * np.exp(-r * T) * norm.cdf(d2 if option_type == 'call' else -d2)) / 365
    rho = K * T * np.exp(-r * T) * norm.cdf(d2 if option_type == 'call' else -d2) / 100
    
    return {'delta': delta, 'gamma': gamma, 'vega': vega, 'theta': theta, 'rho': rho}

# 组合风险分析
def portfolio_risk_analysis():
    S, r, sigma, T = 98, 0.05, 0.25, 0.25
    
    # 持仓1: 看涨多头
    greeks1 = calculate_greeks(S, 100, r, sigma, T, 'call')
    quantity1 = 100 * 100  # 100手
    
    # 持仓2: 看跌空头
    greeks2 = calculate_greeks(S, 95, r, sigma, T, 'put')
    quantity2 = -50 * 100  # 50手空头
    
    # 组合希腊字母
    portfolio = {}
    for greek in ['delta', 'gamma', 'vega', 'theta', 'rho']:
        portfolio[greek] = greeks1[greek] * quantity1 + greeks2[greek] * quantity2
    
    return portfolio

risk = portfolio_risk_analysis()
print("组合风险指标:")
for k, v in risk.items():
    print(f"  {k.upper()}: {v:.2f}")

# 情景分析
print("\n情景分析 (股价+1%):")
P_L = risk['delta'] * 0.98 + 0.5 * risk['gamma'] * (0.98)**2
print(f"  预期盈亏: ${P_L:.2f}")
```

## 问题9：风险价值VaR计算 ⭐⭐⭐★☆

### 📝 问题描述

某投资组合包含以下资产：

**持仓**:
| 资产 | 市值 | 日波动率 | 权重 |
|-----|------|---------|-----|
| 股票A | $500万 | 2% | 50% |
| 股票B | $300万 | 1.5% | 30% |
| 债券 | $200万 | 0.5% | 20% |

**相关系数矩阵**:
$$\rho = \begin{pmatrix} 1 & 0.7 & 0.2 \\ 0.7 & 1 & 0.1 \\ 0.2 & 0.1 & 1 \end{pmatrix}$$

**任务**:
1. 计算组合日波动率
2. 计算99%置信度下的日VaR
3. 计算10日VaR

---

### ✅ 解答

#### 步骤1: 计算组合波动率

权重向量：$w = (0.5, 0.3, 0.2)^T$

波动率向量：$\sigma = (0.02, 0.015, 0.005)^T$

协方差矩阵：$\Sigma_{ij} = \sigma_i \sigma_j \rho_{ij}$

$$\Sigma = \begin{pmatrix} 0.0004 & 0.00021 & 0.00002 \\ 0.00021 & 0.000225 & 0.0000075 \\ 0.00002 & 0.0000075 & 0.000025 \end{pmatrix}$$

组合方差：$\sigma_p^2 = w^T \Sigma w$

$$= 0.5^2 \times 0.0004 + 0.3^2 \times 0.000225 + 0.2^2 \times 0.000025$$
$$+ 2 \times 0.5 \times 0.3 \times 0.00021 + 2 \times 0.5 \times 0.2 \times 0.00002 + 2 \times 0.3 \times 0.2 \times 0.0000075$$

$$= 0.0001 + 0.00002025 + 0.000001 + 0.000063 + 0.000004 + 0.0000009$$

$$\approx 0.000189$$

$$\sigma_p = \sqrt{0.000189} \approx 0.0137 = 1.37\%$$

#### 步骤2: 计算VaR

99%置信度对应的分位数：$z_{0.99} = 2.33$

**日VaR**:

$$VaR_{99\%} = z_{0.99} \times \sigma_p \times V$$

$$= 2.33 \times 0.0137 \times 10,000,000$$

$$= \$319,210$$

**10日VaR**:

$$VaR_{10日} = VaR_{1日} \times \sqrt{10} = 319,210 \times 3.16 = \$1,008,704$$

### 📚 风险分析报告框架

```python
def var_calculation(values, volatilities, correlation_matrix, 
                   confidence=0.99, horizon=1):
    """
    风险价值计算
    
    Args:
        values: 各资产市值数组
        volatilities: 日波动率数组
        correlation_matrix: 相关系数矩阵
        confidence: 置信水平
        horizon: 持有期（天）
    """
    weights = values / np.sum(values)
    
    # 协方差矩阵
    cov_matrix = np.outer(volatilities, volatilities) * correlation_matrix
    
    # 组合波动率
    portfolio_var = weights.T @ cov_matrix @ weights
    portfolio_vol = np.sqrt(portfolio_var)
    
    # VaR计算
    z_score = norm.ppf(confidence)
    var = z_score * portfolio_vol * np.sum(values) * np.sqrt(horizon)
    
    return {
        'portfolio_volatility': portfolio_vol,
        'daily_var': var / np.sqrt(horizon),
        'var': var,
        'z_score': z_score
    }

# 实际应用
values = np.array([500, 300, 200]) * 10000
vols = np.array([0.02, 0.015, 0.005])
corr = np.array([[1, 0.7, 0.2], [0.7, 1, 0.1], [0.2, 0.1, 1]])

result = var_calculation(values, vols, corr, confidence=0.99, horizon=10)
print(f"组合日波动率: {result['portfolio_volatility']*100:.2f}%")
print(f"99%日VaR: ${result['daily_var']:,.0f}")
print(f"99%10日VaR: ${result['var']:,.0f}")
```
