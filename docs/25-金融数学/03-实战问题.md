## 问题1：二叉树期权定价

### 问题描述

考虑单期二叉树模型：

- 当前股票价格 $S_0 = 100$
- 上涨因子 $u = 1.2$，下跌因子 $d = 0.8$
- 无风险利率 $r = 5\%$（单期）
- 欧式看涨期权，执行价 $K = 100$

计算：
1. 风险中性概率
2. 期权价格
3. 对冲比率（Delta）

### 解答

**1. 风险中性概率**

风险中性概率 $q$ 满足：

$$S_0 = \frac{1}{1+r}[q \cdot uS_0 + (1-q) \cdot dS_0]$$

$$1 + r = qu + (1-q)d$$

$$q = \frac{(1+r) - d}{u - d} = \frac{1.05 - 0.8}{1.2 - 0.8} = \frac{0.25}{0.4} = 0.625$$

**2. 期权价格**

到期日期权价值：
- 上涨状态：$C_u = (120 - 100)^+ = 20$
- 下跌状态：$C_d = (80 - 100)^+ = 0$

期权价格：

$$C_0 = \frac{1}{1+r}[q \cdot C_u + (1-q) \cdot C_d] = \frac{1}{1.05}[0.625 \times 20 + 0.375 \times 0] = \frac{12.5}{1.05} \approx 11.90$$

**3. 对冲比率**

$$\Delta = \frac{C_u - C_d}{S_0(u - d)} = \frac{20 - 0}{100(1.2 - 0.8)} = \frac{20}{40} = 0.5$$

验证复制组合：
- 持有 $0.5$ 份股票 + 债券空头 $B$
- $0.5 \times 120 - B(1.05) = 20$
- $0.5 \times 80 - B(1.05) = 0$

从第二式：$B = \frac{40}{1.05} \approx 38.10$

组合成本：$0.5 \times 100 - 38.10 = 11.90$ ✓

## 问题2：Black-Scholes计算

### 问题描述

计算欧式看涨期权价格：

- 当前股价 $S = 50$
- 执行价 $K = 50$
- 无风险利率 $r = 6\%$
- 波动率 $\sigma = 20\%$
- 到期时间 $T = 1$ 年

### 解答

**计算 $d_1$ 和 $d_2$**:

$$d_1 = \frac{\ln(50/50) + (0.06 + 0.2^2/2) \times 1}{0.2\sqrt{1}} = \frac{0 + 0.08}{0.2} = 0.4$$

$$d_2 = d_1 - 0.2\sqrt{1} = 0.4 - 0.2 = 0.2$$

**查标准正态分布表**:

$$\Phi(0.4) \approx 0.6554$$
$$\Phi(0.2) \approx 0.5793$$

**计算期权价格**:

$$C = 50 \times 0.6554 - 50 \times e^{-0.06} \times 0.5793$$
$$= 32.77 - 50 \times 0.9418 \times 0.5793$$
$$= 32.77 - 27.28$$
$$\approx 5.49$$

## 问题3：希腊字母计算

### 问题描述

基于问题2的参数，计算：
1. Delta ($\Delta$)
2. Gamma ($\Gamma$)
3. Vega (每1%波动率变化)
4. Theta ($\Theta$，按天计算)

### 解答

**1. Delta**:

$$\Delta = \Phi(d_1) = \Phi(0.4) \approx 0.6554$$

**2. Gamma**:

$$\phi(d_1) = \frac{1}{\sqrt{2\pi}}e^{-0.4^2/2} = \frac{1}{\sqrt{2\pi}}e^{-0.08} \approx 0.3683$$

$$\Gamma = \frac{\phi(d_1)}{S\sigma\sqrt{T}} = \frac{0.3683}{50 \times 0.2 \times 1} = \frac{0.3683}{10} \approx 0.0368$$

**3. Vega**（对 $\sigma = 0.2$，计算每1%变化）：

$$\text{Vega} = S\phi(d_1)\sqrt{T} = 50 \times 0.3683 \times 1 = 18.42$$

每1%波动率变化：$\frac{18.42}{100} = 0.1842$

即波动率增加1%，期权价格增加约0.18美元。

**4. Theta**（按年计算）：

$$\Theta = -\frac{S\phi(d_1)\sigma}{2\sqrt{T}} - rKe^{-rT}\Phi(d_2)$$

$$= -\frac{50 \times 0.3683 \times 0.2}{2} - 0.06 \times 50 \times e^{-0.06} \times 0.5793$$

$$= -1.84 - 1.64 = -3.48$$

按天计算：$\frac{-3.48}{365} \approx -0.0095$ 美元/天

期权每天因时间衰减损失约0.01美元。

## 问题4：风险中性测度转换

### 问题描述

假设在真实测度 $\mathbb{P}$ 下：

$$dS_t = 0.1 S_t dt + 0.25 S_t dW_t$$

无风险利率 $r = 0.05$。

1. 求风险的市场价格 $\theta$
2. 写出风险中性测度 $\mathbb{Q}$ 下的动态
3. 求 $\frac{d\mathbb{Q}}{d\mathbb{P}}$

### 解答

**1. 风险的市场价格**:

$$\theta = \frac{\mu - r}{\sigma} = \frac{0.1 - 0.05}{0.25} = \frac{0.05}{0.25} = 0.2$$

**2. 风险中性动态**:

在 $\mathbb{Q}$ 下，$\tilde{W}_t = W_t + \theta t = W_t + 0.2t$ 是布朗运动。

$$dS_t = rS_t dt + \sigma S_t d\tilde{W}_t = 0.05 S_t dt + 0.25 S_t d\tilde{W}_t$$

**3. Radon-Nikodym导数**:

$$\frac{d\mathbb{Q}}{d\mathbb{P}} = Z_T = \exp\left(-\theta W_T - \frac{1}{2}\theta^2 T\right) = \exp\left(-0.2 W_T - 0.02 T\right)$$

## 问题5：隐含波动率计算

### 问题描述

已知：
- $S = 100$, $K = 100$, $r = 5\%$, $T = 0.5$
- 市场观测到的看涨期权价格 $C_{market} = 8.50$

求隐含波动率 $\sigma_{imp}$（使用Newton-Raphson方法，初始猜测 $\sigma_0 = 0.2$）。

### 解答

**Newton-Raphson迭代**:

$$\sigma_{n+1} = \sigma_n - \frac{C(\sigma_n) - C_{market}}{\text{Vega}(\sigma_n)}$$

**迭代1** ($\sigma_0 = 0.2$):

$$d_1 = \frac{(0.05 + 0.5 \times 0.04) \times 0.5}{0.2\sqrt{0.5}} = \frac{0.035}{0.1414} \approx 0.2475$$
$$d_2 = 0.2475 - 0.1414 = 0.1061$$

$$\Phi(0.2475) \approx 0.5978, \quad \Phi(0.1061) \approx 0.5423$$

$$C(0.2) = 100 \times 0.5978 - 100 \times e^{-0.025} \times 0.5423$$
$$= 59.78 - 52.95 = 6.83$$

$$\text{Vega} = S\phi(d_1)\sqrt{T} = 100 \times 0.3875 \times 0.7071 \approx 27.40$$

$$\sigma_1 = 0.2 - \frac{6.83 - 8.50}{27.40} = 0.2 + 0.061 = 0.261$$

**迭代2** ($\sigma_1 = 0.261$):

类似计算可得 $C(0.261) \approx 8.45$，接近市场值。

**隐含波动率**: $\sigma_{imp} \approx 26.2\%$

## 问题6：看跌-看涨平价套利

### 问题描述

市场数据：
- $S = 50$, $r = 6\%$, $T = 0.5$
- 看涨期权：$K = 50$, $C = 4.50$
- 看跌期权：$K = 50$, $P = 3.00$

检验是否存在套利机会，若有，设计套利策略。

### 解答

**检验平价关系**:

$$C - P = 4.50 - 3.00 = 1.50$$

$$S - Ke^{-rT} = 50 - 50 \times e^{-0.06 \times 0.5} = 50 - 50 \times 0.9704 = 50 - 48.52 = 1.48$$

偏差：$1.50 - 1.48 = 0.02 > 0$

**套利策略**:

相对理论值，看涨期权定价偏高（或看跌偏低）。

构建组合（平价关系左边 < 右边，买入左边）：
- **买入**：看涨期权 (-$4.50)
- **卖出**：看跌期权 (+$3.00)
- **卖出**：股票 (+$50.00)
- **投资**：$Ke^{-rT} = 48.52$ 在债券中 (-$48.52)

初始现金流：$-4.50 + 3.00 + 50.00 - 48.52 = -0.02$（修正策略）

**正确策略**（平价关系左边 > 右边，卖出左边）：
- **卖出**：看涨期权 (+$4.50)
- **买入**：看跌期权 (-$3.00)
- **买入**：股票 (-$50.00)
- **借款**：$Ke^{-rT} = 48.52$ (+$48.52)

初始现金流：$4.50 - 3.00 - 50.00 + 48.52 = 0.02 > 0$

到期日 $T$ 时：
- 若 $S_T \geq K$：看涨被行权，交付股票获得 $K$；看跌无价值；偿还借款 $K$。净收益 = 0
- 若 $S_T < K$：看涨无价值；行权看跌，卖出股票获得 $K$；偿还借款 $K$。净收益 = 0

无风险收益：$0.02$（初始正现金流）

## 问题7：Vasicek债券定价

### 问题描述

Vasicek模型参数：
- $a = 0.3$, $b = 0.05$, $\sigma = 0.02$
- 当前利率 $r_0 = 0.04$

计算5年期零息债券价格 $P(0, 5)$。

### 解答

**仿射期限结构**:

$$P(0, T) = \exp(A(0, T) - B(0, T)r_0)$$

其中：

$$B(0, T) = \frac{1 - e^{-aT}}{a} = \frac{1 - e^{-1.5}}{0.3} = \frac{1 - 0.2231}{0.3} = 2.5896$$

$$A(0, T) = \left(b - \frac{\sigma^2}{2a^2}\right)(B(0, T) - T) - \frac{\sigma^2 B(0, T)^2}{4a}$$

$$b - \frac{\sigma^2}{2a^2} = 0.05 - \frac{0.0004}{0.18} = 0.05 - 0.0022 = 0.0478$$

$$B(0, T) - T = 2.5896 - 5 = -2.4104$$

$$\frac{\sigma^2 B(0, T)^2}{4a} = \frac{0.0004 \times 6.7060}{1.2} = \frac{0.0027}{1.2} = 0.0022$$

$$A(0, T) = 0.0478 \times (-2.4104) - 0.0022 = -0.1152 - 0.0022 = -0.1174$$

**债券价格**:

$$P(0, 5) = \exp(-0.1174 - 2.5896 \times 0.04) = \exp(-0.1174 - 0.1036) = e^{-0.221} \approx 0.8025$$

## 问题8：亚式期权定价

### 问题描述

考虑几何平均亚式看涨期权：

- $S_0 = 100$, $K = 100$, $r = 5\%$, $\sigma = 20\%$, $T = 1$
-  payoff: $(G_T - K)^+$，其中 $G_T = \exp\left(\frac{1}{T}\int_0^T \ln S_t dt\right)$

利用几何平均的对数正态性推导定价公式。

### 解答

**几何平均的性质**:

$$\ln G_T = \frac{1}{T}\int_0^T \ln S_t dt$$

在风险中性测度下：

$$\ln S_t = \ln S_0 + (r - \sigma^2/2)t + \sigma W_t$$

$$\int_0^T \ln S_t dt = T\ln S_0 + (r - \sigma^2/2)\frac{T^2}{2} + \sigma \int_0^T W_t dt$$

**计算均值和方差**:

$$\mathbb{E}[\ln G_T] = \ln S_0 + \frac{1}{2}(r - \sigma^2/2)T = \ln 100 + 0.5 \times 0.04 = 4.6052 + 0.02 = 4.6252$$

对于 $\int_0^T W_t dt$：
- 均值：0
- 方差：$\text{Var}\left(\int_0^T W_t dt\right) = \int_0^T\int_0^T \min(s, t) ds dt = \frac{T^3}{3}$

因此：

$$\text{Var}(\ln G_T) = \frac{\sigma^2}{T^2} \cdot \frac{T^3}{3} = \frac{\sigma^2 T}{3} = \frac{0.04}{3} \approx 0.0133$$

**等效参数**:

将几何平均亚式期权视为普通期权，参数调整为：
- $S_0^{eff} = S_0 \exp\left(\frac{(r - \sigma^2/2)T}{2} - \frac{\sigma^2 T}{12}\right)$
- $\sigma_{eff} = \frac{\sigma}{\sqrt{3}} \approx 0.1155$

使用调整后的Black-Scholes公式计算。

## 问题9：障碍期权定价

### 问题描述

向下敲出看涨期权（Down-and-Out Call）：
- $S = 100$, $K = 100$, $H = 90$（障碍）
- $r = 5\%$, $\sigma = 20\%$, $T = 1$
- 若股价触及 $H = 90$，期权作废

推导定价公式。

### 解答

**反射原理方法**:

向下敲出看涨期权价格 = 普通看涨期权 - 调整项

$$C_{DO} = C_{BS}(S, K) - \left(\frac{S}{H}\right)^{1 - 2r/\sigma^2} C_{BS}\left(\frac{H^2}{S}, K\right)$$

计算参数：
- $2r/\sigma^2 = 0.1/0.04 = 2.5$
- $1 - 2r/\sigma^2 = -1.5$

$$\left(\frac{100}{90}\right)^{-1.5} = (1.111)^{-1.5} \approx 0.853$$

计算普通看涨期权（问题2已算）：$C_{BS}(100, 100) \approx 10.45$

计算 $C_{BS}(81, 100)$（假设）：
- $S' = 81$, $K = 100$
- $d_1 = \frac{\ln(0.81) + 0.08}{0.2} = \frac{-0.211 + 0.08}{0.2} = -0.655$

该期权虚值较深，价值较低（约1-2）。

$$C_{DO} \approx 10.45 - 0.853 \times 2 \approx 8.74$$

## 问题10：蒙特卡洛模拟

### 问题描述

用蒙特卡洛模拟计算亚式期权价格（算术平均）：

- $S_0 = 100$, $K = 100$, $r = 5\%$, $\sigma = 20\%$, $T = 1$
-  payoff: $\left(\frac{1}{N}\sum_{i=1}^N S_{t_i} - K\right)^+$
- 时间步数 $N = 252$（交易日）
- 模拟路径数 $M = 100000$

描述算法并给出估计结果。

### 解答

**算法步骤**:

1. **离散化**: $\Delta t = T/N = 1/252 \approx 0.004$

2. **生成路径**（对 $j = 1, \ldots, M$）：
   - $S_0^{(j)} = 100$
   - 对 $i = 1, \ldots, N$：
     $$Z_i^{(j)} \sim N(0, 1)$$
     $$S_i^{(j)} = S_{i-1}^{(j)} \exp\left[(r - \sigma^2/2)\Delta t + \sigma\sqrt{\Delta t}Z_i^{(j)}\right]$$

3. **计算收益**:
   $$A^{(j)} = \frac{1}{N}\sum_{i=1}^N S_i^{(j)}$$
   $$V_T^{(j)} = (A^{(j)} - K)^+$$

4. **折现平均**:
   $$\hat{C} = e^{-rT} \cdot \frac{1}{M}\sum_{j=1}^M V_T^{(j)}$$

5. **计算标准误**:
   $$SE = \frac{e^{-rT} \cdot \sigma_V}{\sqrt{M}}$$

其中 $\sigma_V$ 是样本标准差。

**数值结果**（参考值）:

对于算术平均亚式期权：
- 精确值（PDE方法）：约 $5.7 - 6.0$
- 蒙特卡洛估计（$M=100000$）：$\hat{C} \approx 5.85 \pm 0.05$（95%置信区间）

**方差缩减技巧**:
- 对偶变量：每路径生成 $Z$ 和 $-Z$ 一对
- 控制变量：使用几何平均亚式期权的解析解作为控制
