# 并行计算基础

## MPI基础

### 点对点通信

```c
MPI_Send(buf, count, datatype, dest, tag, comm);
MPI_Recv(buf, count, datatype, source, tag, comm, status);
```

### 集合通信

- MPI_Bcast
- MPI_Reduce
- MPI_Allreduce

## OpenMP

### 并行区域

```c
#pragma omp parallel for
for (i = 0; i < n; i++) { ... }
```
