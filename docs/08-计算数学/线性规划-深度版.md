## 📚 概述

线性规划是运筹学和最优化理论的基石，研究在一组线性约束下最大化或最小化线性目标函数的问题。从Dantzig的单纯形法到Karmarkar的内点法，线性规划算法经历了两次革命性的发展。

本深度版将系统阐述线性规划的理论基础，提供对偶理论和收敛性的严格证明，并探讨其与格洛腾迪克数学体系——特别是层论、概形和热带几何——的深层联系。

## 🕰️ 历史发展与Dantzig革命

### 线性不等式系统的古典研究

**Fourier (1827)**：Fourier-Motzkin消元法。

**Minkowski (1896)**：凸多面体的表示定理。

**Weyl (1935)**：凸多面体的对偶描述。

### 单纯形法的发明

**Dantzig (1947)**：为美国空军规划问题时发明。

**核心理念**：在可行域的顶点间移动，每一步改进目标函数值。

**影响**：推动了整个运筹学学科的发展。

### 内点法的突破

**Karmarkar (1984)**：首个实际可行的多项式时间算法。

**重要性**：理论上的突破（多项式时间）和实践上的竞争力。

**后续发展**：原-对偶内点方法成为大规模问题的标准工具。

## 🏗️ 理论基础的完整阐述

### 凸多面体的几何理论

**表示定理**（Minkowski-Weyl）：
$$P = \{x : Ax \leq b\} = \text{conv}(V) + \text{cone}(R)$$

有界多面体（多面体）是其顶点的凸包。

**顶点的代数刻画**：$x$是顶点当且仅当活跃约束的矩阵满秩。

### Farkas引理与择一定理

**Farkas引理**：以下恰有一个成立：
1. $\exists x \geq 0: Ax = b$
2. $\exists y: A^T y \geq 0, b^T y < 0$

**证明框架**：
- 若(1)不成立，$b \notin K = \{Ax : x \geq 0\}$
- 分离超平面定理给出(2)

**重要性**：对偶理论的基石。

### 对偶理论的严格证明

**原始问题** (P)：$\min c^T x$ s.t. $Ax = b, x \geq 0$

**对偶问题** (D)：$\max b^T y$ s.t. $A^T y \leq c$

**弱对偶**：$c^T x \geq b^T y$对所有可行解成立。

**强对偶**：若(P)和(D)都可行，则最优值相等。

**互补松弛性**：$x_i(c_i - (A^T y)_i) = 0$对所有$i$。

## 🔬 单纯形法的完整分析

### 顶点与基本可行解

**基本解**：选择基$B$（$m$个线性无关列），设非基变量为0，解$Ax = b$。

**基本可行解（BFS）**：$x_B = A_B^{-1}b \geq 0$。

**定理**：BFS与可行域顶点一一对应。

### 单纯形迭代的代数结构

**目标函数改进**：
$$z = c_B^T x_B + c_N^T x_N = c_B^T A_B^{-1}b + (c_N^T - c_B^T A_B^{-1} A_N)x_N$$

**简约成本**：$\bar{c}_j = c_j - c_B^T A_B^{-1} a_j$

**进基规则**：选择$\bar{c}_j < 0$的非基变量进基。

**出基规则（最小比值检验）**：
$$i^* = \arg\min_i \left\{\frac{(A_B^{-1}b)_i}{(A_B^{-1}a_j)_i} : (A_B^{-1}a_j)_i > 0\right\}$$

### 收敛性与循环避免

**有限收敛**：若非退化，单纯形法在有限步内收敛（顶点有限）。

**退化与循环**：目标函数不改进的迭代可能导致循环。

**Bland规则**：选择下标最小的进基和出基变量，保证不循环。

## 📐 内点法的数学理论

### 障碍函数与中心路径

**对数障碍函数**：
$$\min c^T x - \mu \sum_{i=1}^n \ln x_i \quad \text{s.t. } Ax = b$$

**中心路径**：当$\mu \to 0$，障碍问题解$x(\mu)$形成趋于最优的路径。

**性质**：
- 对任意$\mu > 0$，$x(\mu) > 0$（严格可行）
- $\lim_{\mu \to 0} x(\mu) = x^*$

### 牛顿方向与路径跟踪

**KKT条件**：
$$\begin{cases} Ax = b \\ A^T y + s = c \\ x_i s_i = \mu, \quad i=1,\ldots,n \end{cases}$$

**牛顿步**：求解线性系统得到搜索方向。

**路径跟踪算法**：沿中心路径以几何速率减小$\mu$。

### 多项式时间收敛性证明

**定理**：内点法在$O(\sqrt{n}\ln(1/\epsilon))$次迭代内达到$\epsilon$-最优。

**证明框架**：
1. 势函数或邻域概念
2. 每次迭代减小对偶间隙的固定比例
3. 累积得到多项式复杂度

## ⚡ 扩展与变体

### 整数线性规划与割平面

**分支定界**：系统枚举整数解。

**割平面法**：添加有效不等式割去分数解。

**Gomory割平面**：从单纯形表生成。

### 鲁棒优化

**不确定性**：$b \in \mathcal{U}$（不确定集）

**鲁棒对应**：$\min c^T x$ s.t. $Ax \geq b, \forall b \in \mathcal{U}$

**可计算性**：对于特定不确定集，可转化为LP或SOCP。

### 大规模分解方法

**Dantzig-Wolfe分解**：列生成。

**Benders分解**：行生成。

**延迟列生成**：只生成必要的变量。

## 📊 计算复杂度与算法选择

### 单纯形法的平均复杂度

**Worst-case**：Klee-Minty例子，指数复杂度。

**平均情况**：Smale等人的分析表明，期望多项式复杂度。

### 内点法的迭代复杂度

**理论**：$O(\sqrt{n})$或$O(n)$迭代。

**实践**：通常30-50次迭代，与问题规模基本无关。

### 实际问题的算法选择

| 特征 | 推荐算法 |
|------|----------|
| 中小规模 | 单纯形法 |
| 大规模稀疏 | 内点法 |
| 需要基解 | 单纯形法 |
| 网络结构 | 网络单纯形 |
