# 08-计算数学

本目录系统整理计算数学领域的核心内容与实践资源，收录约 40 份文档，涵盖数值分析、优化理论、计算几何及基础概念。文档风格兼顾理论推导与算法实现，常配有伪代码、收敛性分析与误差估计，帮助学习者理解“可计算的数学”与“数学的计算”之间的双向互动。目录中还包含国际课程对齐对照表与内容建设完成报告，便于教育者快速定位教学资源并进行课程设计。
