## 1. 概述

### 1.1 算法复杂度对比

| 问题 | 朴素算法 | 快速算法 | 加速比 |
|-----|---------|---------|--------|
| DFT | $O(N^2)$ | $O(N\log N)$ | $N/\log N$ |
| 多项式乘法 | $O(N^2)$ | $O(N\log N)$ | $N/\log N$ |
| 矩阵乘法 | $O(N^3)$ | $O(N^{2.81})$ | $N^{0.19}$ |
| N体问题 | $O(N^2)$ | $O(N)$ | $N$ |
| 稀疏求解 | $O(N^3)$ | $O(N)$ | $N^2$ |

### 1.2 算法设计范式

- **分治**：分解、解决、合并
- **变换**：将问题转换到更易求解的域
- **近似**：牺牲精度换取速度
- **稀疏性**：利用问题结构减少计算

## 2. 快速傅里叶变换

### 2.1 Cooley-Tukey算法

#### 算法思想

DFT定义：$X_k = \sum_{n=0}^{N-1} x_n \omega_N^{nk}$，$\omega_N = e^{-2\pi i/N}$

**分治策略**：将序列分为偶数和奇数下标：

$$\begin{aligned}
X_k &= \sum_{m=0}^{N/2-1} x_{2m} \omega_N^{2mk} + \sum_{m=0}^{N/2-1} x_{2m+1} \omega_N^{(2m+1)k} \\
&= \underbrace{\sum_{m=0}^{N/2-1} x_{2m} \omega_{N/2}^{mk}}_{E_k} + \omega_N^k \underbrace{\sum_{m=0}^{N/2-1} x_{2m+1} \omega_{N/2}^{mk}}_{O_k}
\end{aligned}$$

**蝶形运算**：
$$X_k = E_k + \omega_N^k O_k$$
$$X_{k+N/2} = E_k - \omega_N^k O_k$$

```python
import numpy as np
import matplotlib.pyplot as plt

def cooley_tukey_fft(x):
    """
    Cooley-Tukey FFT（基2，递归实现）
    
    参数:
        x: 输入序列（长度必须是2的幂）
    """
    N = len(x)
    
    if N <= 1:
        return x
    
    # 分为偶数和奇数
    even = cooley_tukey_fft(x[0::2])
    odd = cooley_tukey_fft(x[1::2])
    
    # 蝶形运算
    T = np.exp(-2j * np.pi * np.arange(N) / N)
    
    result = np.zeros(N, dtype=complex)
    for k in range(N // 2):
        result[k] = even[k] + T[k] * odd[k]
        result[k + N // 2] = even[k] - T[k] * odd[k]
    
    return result

def iterative_fft(x):
    """迭代实现的FFT（位反转置换）"""
    N = len(x)
    
    # 位反转置换
    def bit_reverse(n, bits):
        result = 0
        for i in range(bits):
            result = (result << 1) | ((n >> i) & 1)
        return result
    
    bits = int(np.log2(N))
    X = np.array([x[bit_reverse(i, bits)] for i in range(N)], dtype=complex)
    
    # 迭代FFT
    step = 2
    while step <= N:
        half = step // 2
        for start in range(0, N, step):
            for k in range(half):
                idx = start + k
                twiddle = np.exp(-2j * np.pi * k / step)
                temp = twiddle * X[idx + half]
                X[idx + half] = X[idx] - temp
                X[idx] = X[idx] + temp
        step *= 2
    
    return X

# 测试
N = 1024
x = np.random.randn(N)

# 自定义FFT

# NumPy FFT
X_numpy = np.fft.fft(x)

# 验证
print(f"最大误差: {np.max(np.abs(X_custom - X_numpy)):.2e}")

# 性能比较

for N in sizes:
```

#### 复杂度分析

**递归关系**：$T(N) = 2T(N/2) + O(N)$

**Master定理**：$T(N) = O(N \log N)$

详细计算：
- 层数：$\log_2 N$
- 每层运算：$N$次复数乘法和加法
- 总计：$O(N \log N)$

### 2.2 其他FFT算法

#### Bluestein算法（任意长度）

$$X_k = \sum_{n=0}^{N-1} x_n e^{-2\pi ink/N} = e^{-\pi i k^2/N} \sum_{n=0}^{N-1} (x_n e^{-\pi i n^2/N}) e^{\pi i (k-n)^2/N}$$

转化为卷积，可用FFT计算。

#### Rader算法（素数长度）

利用原根将DFT转化为循环卷积。

```python
def bluestein_fft(x):
    """Bluestein算法（Chirp-Z变换）"""
    N = len(x)
    
    # 找到大于等于2N-1的最小2的幂
    M = 1
    while M < 2 * N - 1:
        M *= 2
    
    # Chirp序列
    n = np.arange(N)
    chirp = np.exp(1j * np.pi * n**2 / N)
    
    # 构造卷积序列
    a = x * chirp
    b = np.conj(chirp)
    
    # 补零
    a_padded = np.zeros(M, dtype=complex)
    b_padded = np.zeros(M, dtype=complex)
    a_padded[:N] = a
    b_padded[:N] = b
    b_padded[M-N+1:] = b[1:][::-1]  # 循环部分
    
    # FFT卷积
    A = np.fft.fft(a_padded)
    B = np.fft.fft(b_padded)
    C = A * B
    c = np.fft.ifft(C)
    
    # 提取结果
    result = chirp * c[:N]
    
    return result

# 测试非2的幂长度
N = 1000
x = np.random.randn(N)
X_bluestein = bluestein_fft(x)
X_dft = np.fft.fft(x)
print(f"Bluestein误差: {np.max(np.abs(X_bluestein - X_dft)):.2e}")
```

### 2.3 多维FFT

#### 可分离性

二维DFT可分解为行和列的一维DFT：

$$X_{k,l} = \sum_{m=0}^{M-1} \sum_{n=0}^{N-1} x_{m,n} e^{-2\pi i(mk/M + nl/N)}$$

**算法**：先对每行做FFT，再对每列做FFT。

```python
def fft2_manual(x):
    """二维FFT（手动实现）"""
    M, N = x.shape
    
    # 对每行做FFT
    X = np.array([iterative_fft(row) for row in x])
    
    # 对每列做FFT
    X = np.array([iterative_fft(col) for col in X.T]).T
    
    return X
```

## 3. 快速多项式乘法

### 3.1 问题描述

给定两个多项式：
$$p(x) = \sum_{i=0}^{n-1} a_i x^i, \quad q(x) = \sum_{j=0}^{m-1} b_j x^j$$

求乘积$r(x) = p(x)q(x)$。

**朴素算法**：$O(nm)$  
**FFT算法**：$O(N\log N)$，其中$N \geq n+m-1$

### 3.2 FFT-based乘法

```python
def polynomial_multiply_fft(a, b):
    """
    使用FFT进行多项式乘法
    
    参数:
        a, b: 多项式系数数组（从低次到高次）
    """
    n, m = len(a), len(b)
    result_size = n + m - 1
    
    # 找到大于等于result_size的最小2的幂
    N = 1
    while N < result_size:
        N *= 2
    
    # 补零并FFT
    A = np.fft.fft(np.pad(a, (0, N - n)))
    B = np.fft.fft(np.pad(b, (0, N - m)))
    
    # 逐点相乘
    C = A * B
    
    # 逆FFT并取整
    c = np.fft.ifft(C).real
    
    # 四舍五入到最近整数（如果是整数系数）
    c = np.round(c[:result_size]).astype(int)
    
    return c

def polynomial_multiply_naive(a, b):
    """朴素多项式乘法"""
    n, m = len(a), len(b)
    c = np.zeros(n + m - 1)
    for i in range(n):
        for j in range(m):
            c[i + j] += a[i] * b[j]
    return c

# 测试
np.random.seed(42)

b = np.random.randint(0, 10, n)

import time

t_fft = time.time() - t1

print(f"朴素方法: {t_naive:.4f}秒")

print(f"结果正确: {np.allclose(c_fft, c_naive)}")
```

### 3.3 数论变换(NTT)

在有限域上进行DFT，避免浮点误差。

```python
def mod_pow(base, exp, mod):
    """快速幂取模"""
    result = 1
    base %= mod
    while exp > 0:
        if exp & 1:
            result = (result * base) % mod
        base = (base * base) % mod
        exp >>= 1
    return result

def ntt(a, invert=False, mod=998244353, root=3):
    """
    数论变换
    
    参数:
        mod: 素数，形式为 c*2^k + 1
        root: 原根
    """
    n = len(a)
    j = 0
    for i in range(1, n):
        bit = n >> 1
        while j & bit:
            j ^= bit
            bit >>= 1
        j ^= bit
        if i < j:
            a[i], a[j] = a[j], a[i]
    
    length = 2
    while length <= n:
        wlen = mod_pow(root, (mod - 1) // length, mod)
        if invert:
            wlen = mod_pow(wlen, mod - 2, mod)
        for i in range(0, n, length):
            w = 1
            half = length // 2
            for j in range(i, i + half):
                u = a[j]
                v = a[j + half] * w % mod
                a[j] = (u + v) % mod
                a[j + half] = (u - v + mod) % mod
                w = w * wlen % mod
        length <<= 1
    
    if invert:
        n_inv = mod_pow(n, mod - 2, mod)
        for i in range(n):
            a[i] = a[i] * n_inv % mod
    
    return a

def polynomial_multiply_ntt(a, b, mod=998244353):
    """使用NTT进行精确多项式乘法"""
    n, m = len(a), len(b)
    result_size = n + m - 1
    
    N = 1
    while N < result_size:
        N <<= 1
    
    fa = a + [0] * (N - n)
    fb = b + [0] * (N - m)
    
    fa = ntt(fa)
    fb = ntt(fb)
    
    for i in range(N):
        fa[i] = fa[i] * fb[i] % mod
    
    result = ntt(fa, invert=True)
    return result[:result_size]

# 测试NTT
a = [1, 2, 3, 4, 5]
b = [5, 4, 3, 2, 1]
c_ntt = polynomial_multiply_ntt(a, b)
print(f"NTT结果: {c_ntt}")
```

## 4. 快速矩阵乘法

### 4.1 Strassen算法

将矩阵分块：
$$\begin{bmatrix} C_{11} & C_{12} \\ C_{21} & C_{22} \end{bmatrix} = 
\begin{bmatrix} A_{11} & A_{12} \\ A_{21} & A_{22} \end{bmatrix} 
\begin{bmatrix} B_{11} & B_{12} \\ B_{21} & B_{22} \end{bmatrix}$$

只需7次乘法而非8次：

```python
def strassen_multiply(A, B, threshold=64):
    """
    Strassen矩阵乘法
    
    参数:
        threshold: 低于此大小使用朴素算法
    """
    n = len(A)
    
    # 基本情况
    if n <= threshold:
        return np.dot(A, B)
    
    # 确保n是2的幂
    if n % 2 == 1:
        # 补零
        A_padded = np.zeros((n+1, n+1))
        B_padded = np.zeros((n+1, n+1))
        A_padded[:n, :n] = A
        B_padded[:n, :n] = B
        C = strassen_multiply(A_padded, B_padded, threshold)
        return C[:n, :n]
    
    # 分块
    mid = n // 2
    A11, A12 = A[:mid, :mid], A[:mid, mid:]
    A21, A22 = A[mid:, :mid], A[mid:, mid:]
    B11, B12 = B[:mid, :mid], B[:mid, mid:]
    B21, B22 = B[mid:, :mid], B[mid:, mid:]
    
    # 7个乘积
    M1 = strassen_multiply(A11 + A22, B11 + B22, threshold)
    M2 = strassen_multiply(A21 + A22, B11, threshold)
    M3 = strassen_multiply(A11, B12 - B22, threshold)
    M4 = strassen_multiply(A22, B21 - B11, threshold)
    M5 = strassen_multiply(A11 + A12, B22, threshold)
    M6 = strassen_multiply(A21 - A11, B11 + B12, threshold)
    M7 = strassen_multiply(A12 - A22, B21 + B22, threshold)
    
    # 组合结果
    C11 = M1 + M4 - M5 + M7
    C12 = M3 + M5
    C21 = M2 + M4
    C22 = M1 - M2 + M3 + M6
    
    # 合并
    C = np.vstack((np.hstack((C11, C12)),
                   np.hstack((C21, C22))))
    
    return C

# 测试
n = 512
A = np.random.randn(n, n)
B = np.random.randn(n, n)

t1 = time.time()
C_strassen = strassen_multiply(A, B, threshold=64)
t_strassen = time.time() - t1

t1 = time.time()
C_numpy = np.dot(A, B)
t_numpy = time.time() - t1

print(f"Strassen: {t_strassen:.4f}秒")
print(f"NumPy BLAS: {t_numpy:.4f}秒")
print(f"误差: {np.max(np.abs(C_strassen - C_numpy)):.2e}")
```

### 4.2 复杂度分析

**Strassen**：$O(N^{\log_2 7}) \approx O(N^{2.81})$

**Coppersmith-Winograd**：$O(N^{2.376})$（理论最优）

**AlphaTensor**：最近发现$O(N^{2.78})$的算法

## 5. 快速多极子法

### 5.1 N体问题

计算$N$个粒子之间的相互作用：
$$\Phi(x_i) = \sum_{j \neq i} \frac{m_j}{|x_i - x_j|}$$

朴素算法$O(N^2)$，快速多极子法$O(N)$。

### 5.2 核心思想

1. **多极展开**：远距离粒子群用多极矩近似
2. **局部展开**：将多极展开转换为局部展开
3. **分层网格**：自适应细分空间

```python
class FastMultipoleMethod:
    """
    二维快速多极子法（简化实现）
    """
    
    def __init__(self, p=5):
        """
        参数:
            p: 多极展开的截断阶数
        """
        self.p = p
    
    def multipole_expansion(self, sources, center):
        """
        计算多极展开系数
        
        M_k = Σ m_j (z_j - z_c)^k / k!
        """
        M = np.zeros(self.p, dtype=complex)
        for z, m in sources:
            dz = z - center
            for k in range(self.p):
                M[k] += m * (dz ** k)
        # 阶乘归一化
        for k in range(1, self.p):
            M[k] /= np.math.factorial(k)
        return M
    
    def evaluate_multipole(self, M, center, target):
        """评估多极展开在目标点的值"""
        dz = target - center
        result = 0
        for k in range(self.p):
            result += M[k] / (dz ** (k + 1))
        return result.real
    
    def direct_sum(self, sources, targets):
        """直接计算（用于验证和近场）"""
        result = np.zeros(len(targets))
        for i, (zt, _) in enumerate(targets):
            for zs, m in sources:
                if abs(zt - zs) > 1e-10:
                    result[i] += m / abs(zt - zs)
        return result
    
    def fmm(self, sources, targets, box_size=1.0, n_levels=3):
        """
        简化FMM实现
        """
        # 这里应实现完整的FMM算法
        # 为简化，仅演示概念
        
        # 将空间划分为层次网格
        # 1. 向上传递：计算每个盒子的多极展开
        # 2. 向下传递：M2L转换和局部展开
        # 3. 直接计算：近场相互作用
        
        # 简化版本：对远距离使用多极展开
        result = np.zeros(len(targets))
        
        for i, (zt, _) in enumerate(targets):
            for zs, m in sources:
                r = abs(zt - zs)
                if r < box_size / (2 ** n_levels):
                    # 近场：直接计算
                    if r > 1e-10:
                        result[i] += m / r
                else:
                    # 远场：多极近似（简化）
                    result[i] += m / r
        
        return result

# 测试

           zip(np.random.rand(N), np.random.rand(N))]
```

```python
    print(f"CSR存储需要: {(A_csr.data.nbytes + A_csr.indices.nbytes + A_csr.indptr.nbytes) / 1e6:.2f} MB")
    
    # 矩阵-向量乘法性能
    x = np.random.randn(n)
    
    import time
    
    # CSR乘法
    t1 = time.time()
    for _ in range(100):
        y = A_csr @ x
    t_csr = (time.time() - t1) / 100
    
    # 密集乘法（仅小矩阵）
    if n <= 2000:
        A_dense = A_csr.toarray()
        t1 = time.time()
        for _ in range(10):
            y = A_dense @ x
        t_dense = (time.time() - t1) / 10
        print(f"密集乘法: {t_dense:.6f}秒")
    
    print(f"CSR乘法: {t_csr:.6f}秒")
    print(f"加速比: {t_dense / t_csr if n <= 2000 else 'N/A'}")
```

### 7.1 图像压缩

```python
def image_compression_dct(image, keep_ratio=0.1):
    """
    使用DCT进行图像压缩
    """
    from scipy.fftpack import dct, idct
    
    def dct2(block):
        return dct(dct(block.T, norm='ortho').T, norm='ortho')
    
    def idct2(block):
        return idct(idct(block.T, norm='ortho').T, norm='ortho')
    
    h, w = image.shape
    block_size = 8
    
    compressed = np.zeros_like(image)
    
    for i in range(0, h, block_size):
        for j in range(0, w, block_size):
            block = image[i:i+block_size, j:j+block_size]
            if block.shape != (block_size, block_size):
                continue
            
            # DCT变换
            dct_block = dct2(block)
            
            # 保留低频分量
            n_keep = int(block_size * block_size * keep_ratio)
            mask = np.zeros((block_size, block_size))
            
            # Zig-zag扫描保留低频
            idx = 0
            for s in range(2 * block_size - 1):
                for i_idx in range(block_size):
                    j_idx = s - i_idx
                    if 0 <= j_idx < block_size:
                        if idx < n_keep:
                            mask[i_idx, j_idx] = 1
                        idx += 1
            
            dct_block *= mask
            
            # 逆变换
            compressed[i:i+block_size, j:j+block_size] = idct2(dct_block)
    
    return compressed
```

### 7.2 大规模图分析

```python
def pagerank_sparse(adj_matrix, alpha=0.85, tol=1e-8, max_iter=100):
    """稀疏矩阵PageRank"""
    n = adj_matrix.shape[0]
    
    # 归一化
    out_degrees = np.array(adj_matrix.sum(axis=1)).flatten()
    out_degrees[out_degrees == 0] = 1  # 处理悬停节点
    
    # 构造转移矩阵（稀疏）
    P = adj_matrix.T / out_degrees
    P = P.tocsr()
    
    # 迭代
    x = np.ones(n) / n
    
    for i in range(max_iter):
        x_new = alpha * P @ x + (1 - alpha) / n
        
        if np.linalg.norm(x_new - x) < tol:
            print(f"PageRank收敛于{i+1}次迭代")
            break
        
        x = x_new
    
    return x
```
