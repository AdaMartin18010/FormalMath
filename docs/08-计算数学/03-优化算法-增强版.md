## 📚 概述

优化算法是计算数学的核心分支，研究如何在给定约束条件下寻找目标函数的最优解。从数学规划到现代机器学习，优化算法支撑着科学计算和人工智能的理论基础。本教学文档系统介绍凸优化理论、梯度方法、牛顿法以及约束优化，着重阐述算法的收敛性分析和实际应用。

**核心问题**：对于目标函数 $f: \mathbb{R}^n \to \mathbb{R}$，求解
$$\min_{x \in \Omega} f(x)$$
其中 $\Omega \subseteq \mathbb{R}^n$ 是可行域，由等式和不等式约束定义。

## 🕰️ 历史发展脉络

### 凸优化的诞生 (1940-1970)

1947年，George Dantzig提出**单纯形法**，标志着线性规划的正式诞生。该方法在第二次世界大战期间用于军事资源调配，战后广泛应用于经济学和运筹学。

1950-60年代，非线性优化理论快速发展：

- **1951年**：Kuhn-Tucker提出约束优化的必要条件（KKT条件）
- **1960年**：Rosen提出梯度投影法处理约束优化
- **1963年**：Powell提出共轭梯度法，用于大规模优化

### 内点法的革命 (1984-至今)

1984年，Karmarkar提出**多项式时间内点法**，证明了线性规划可以在多项式时间内求解，开启了现代凸优化的篇章。

关键里程碑：

- **1988年**：Nesterov-Nemirovski建立自协调函数理论
- **1994年**：内点法扩展到半定规划(SDP)和二阶锥规划(SOCP)
- **2004年**：Boyd & Vandenberghe出版《Convex Optimization》，奠定现代教学基础

### 机器学习的推动 (2000-至今)

2000年后，大规模机器学习推动随机优化算法发展：

- **随机梯度下降(SGD)**：Robbins-Monro (1951) → Bottou (1998) → 现代深度学习
- **自适应方法**：AdaGrad (2011), RMSprop, Adam (2014)
- **分布式优化**：联邦学习、大规模并行训练

## 📐 核心理论

### 1. 凸函数与凸集判定

#### 凸集判定定理

**定义（凸集）**：集合 $C \subseteq \mathbb{R}^n$ 是凸集，如果对于任意 $x, y \in C$ 和 $\theta \in [0,1]$，有
$$\theta x + (1-\theta)y \in C$$

**判定方法**：

1. **交集保持**：任意凸集的交集仍是凸集
2. **仿射映射**：若 $C$ 凸，$f(x) = Ax + b$ 仿射，则 $f(C)$ 凸
3. **透视函数**：若 $C$ 凸且在 $\{(x,t) : t > 0\}$ 中，则透视像凸

**半定规划中的凸集**：
$$\mathcal{S}_+^n = \{X \in \mathbb{R}^{n \times n} : X = X^T, X \succeq 0\}$$
对称正定矩阵构成凸锥。

#### 凸函数判定条件

**定义（凸函数）**：$f: \mathbb{R}^n \to \mathbb{R}$ 是凸函数，如果定义域 $\text{dom}(f)$ 凸，且对于任意 $x, y \in \text{dom}(f)$ 和 $\theta \in [0,1]$：
$$f(\theta x + (1-\theta)y) \leq \theta f(x) + (1-\theta)f(y)$$

**一阶判定条件**（可微函数）：
$f$ 凸当且仅当对所有 $x, y \in \text{dom}(f)$：
$$f(y) \geq f(x) + \nabla f(x)^T(y-x)$$
函数图像位于切线之上。

**二阶判定条件**（二阶可微）：
$f$ 凸当且仅当Hessian矩阵半正定：
$$\nabla^2 f(x) \succeq 0, \quad \forall x \in \text{dom}(f)$$

**重要凸函数类**：

- 仿射函数：$f(x) = a^T x + b$
- 二次函数：$f(x) = \frac{1}{2}x^T P x + q^T x + r$，当 $P \succeq 0$ 时凸
- 指数函数：$e^{ax}$ 在 $\mathbb{R}$ 上凸
- 负对数：$-\log x$ 在 $\mathbb{R}_{++}$ 上凸
- 范数：任意范数 $||\cdot||$ 是凸函数

### 2. 梯度下降法收敛性分析

梯度下降法是最基本的优化算法，迭代格式为：
$$x_{k+1} = x_k - \alpha_k \nabla f(x_k)$$

#### 收敛性定理

**定理（Lipschitz梯度下的收敛）**：
设 $f$ 凸且可微，$\nabla f$ 是 $L$-Lipschitz连续的：
$$||\nabla f(x) - \nabla f(y)|| \leq L||x - y||, \quad \forall x, y$$

取固定步长 $\alpha_k = \alpha \in (0, 2/L)$，则：
$$f(x_k) - f(x^*) \leq \frac{||x_0 - x^*||^2}{2\alpha k}$$

**证明**：
由Lipschitz条件，有下降引理：
$$f(y) \leq f(x) + \nabla f(x)^T(y-x) + \frac{L}{2}||y-x||^2$$

令 $y = x_{k+1}$，$x = x_k$，$y - x = -\alpha \nabla f(x_k)$：
$$f(x_{k+1}) \leq f(x_k) - \alpha(1 - \frac{\alpha L}{2})||\nabla f(x_k)||^2$$

当 $\alpha < 2/L$ 时，$f(x_{k+1}) < f(x_k)$，函数值严格下降。

通过凸性 $f(x_k) - f(x^*) \leq \nabla f(x_k)^T(x_k - x^*)$ 和迭代关系，可得：
$$||x_{k+1} - x^*||^2 \leq ||x_k - x^*||^2 - \alpha(2 - \alpha L)(f(x_k) - f(x^*))$$

累加并整理即得收敛速率。

#### 收敛速率估计

| 函数类型 | 算法 | 收敛速率 | 复杂度 |
|---------|------|---------|--------|
| 凸函数 | 梯度下降 | $O(1/k)$ | $O(1/\epsilon)$ |
| 强凸函数 | 梯度下降 | $O(\rho^k)$, $\rho < 1$ | $O(\log(1/\epsilon))$ |
| 凸函数 | Nesterov加速 | $O(1/k^2)$ | $O(1/\sqrt{\epsilon})$ |

**强凸性**：若存在 $\mu > 0$ 使得 $f(x) - \frac{\mu}{2}||x||^2$ 凸，则 $f$ 是 $\mu$-强凸的。强凸函数有唯一全局最小值，梯度下降呈线性收敛。

### 3. 牛顿法的二阶收敛性

牛顿法利用二阶信息构造迭代：
$$x_{k+1} = x_k - [\nabla^2 f(x_k)]^{-1}\nabla f(x_k)$$

#### 局部二次收敛定理

**定理（牛顿法局部收敛）**：
设 $f$ 二阶连续可微，$x^*$ 是局部极小点且 $\nabla^2 f(x^*) \succ 0$。若初始点 $x_0$ 充分接近 $x^*$，则牛顿法二次收敛：
$$||x_{k+1} - x^*|| \leq C||x_k - x^*||^2$$

**证明**：
设 $e_k = x_k - x^*$，由泰勒展开：
$$\nabla f(x_k) = \nabla^2 f(x^*)e_k + O(||e_k||^2)$$

牛顿迭代：
$$x_{k+1} = x_k - [\nabla^2 f(x_k)]^{-1}\nabla f(x_k)$$

在 $x^*$ 附近，$\nabla^2 f(x_k) \approx \nabla^2 f(x^*)$，因此：
$$e_{k+1} = e_k - [\nabla^2 f(x^*)]^{-1}(\nabla^2 f(x^*)e_k + O(||e_k||^2)) = O(||e_k||^2)$$

**数值意义**：

- 若 $||e_k|| = 10^{-2}$，则 $||e_{k+1}|| \approx 10^{-4}$
- 若 $||e_{k+1}|| = 10^{-4}$，则 $||e_{k+2}|| \approx 10^{-8}$

仅需6-8次迭代即可达到机器精度！

#### 阻尼牛顿法

全局化策略：结合线搜索
$$x_{k+1} = x_k - \alpha_k [\nabla^2 f(x_k)]^{-1}\nabla f(x_k)$$

其中 $\alpha_k$ 由Wolfe条件或Armijo规则确定，保证全局收敛性。

**计算复杂度**：

- 梯度下降：$O(n)$ 每次迭代
- 牛顿法：$O(n^3)$（求解线性系统）或 $O(n^2)$（若使用预处理）

**拟牛顿法**（BFGS等）：通过低秩更新近似Hessian逆，达到超线性收敛，复杂度 $O(n^2)$。

### 4. 约束优化与KKT条件

#### 拉格朗日函数

标准约束优化问题：
$$
\begin{aligned}
\min_{x} \quad & f(x) \\
\text{s.t.} \quad & g_i(x) \leq 0, \quad i = 1, \ldots, m \\
& h_j(x) = 0, \quad j = 1, \ldots, p
\end{aligned}
$$

**拉格朗日函数**：
$$\mathcal{L}(x, \lambda, \nu) = f(x) + \sum_{i=1}^m \lambda_i g_i(x) + \sum_{j=1}^p \nu_j h_j(x)$$

其中 $\lambda_i \geq 0$ 是不等式约束的乘子，$\nu_j$ 是等式约束的乘子。

#### KKT最优性条件

**定理（KKT条件）**：
设 $f, g_i, h_j$ 可微，$x^*$ 是局部最优解且约束规范（如Slater条件或LICQ）成立，则存在 $\lambda^*, \nu^*$ 满足：

1. **平稳性**：$\nabla_x \mathcal{L}(x^*, \lambda^*, \nu^*) = 0$
2. **原始可行性**：$g_i(x^*) \leq 0$，$h_j(x^*) = 0$
3. **对偶可行性**：$\lambda_i^* \geq 0$
4. **互补松弛性**：$\lambda_i^* g_i(x^*) = 0$

**Slater条件**：若存在严格可行点 $x$ 使得 $g_i(x) < 0$ 且 $h_j(x) = 0$，则强对偶成立，KKT条件是充分的。

**数值实现**：

- 内点法：通过障碍函数 $-\mu \sum \log(-g_i(x))$ 处理不等式约束
- SQP（序列二次规划）：迭代求解二次规划子问题
- 增广拉格朗日法：结合惩罚项和乘子更新

## 🔍 实例表征

### 例1：梯度下降法求解二次函数

**问题**：求解 $\min_x f(x) = \frac{1}{2}x^T A x - b^T x$，其中
$$A = \begin{bmatrix} 4 & 1 \\ 1 & 3 \end{bmatrix}, \quad b = \begin{bmatrix} 1 \\ 2 \end{bmatrix}$$

**理论解**：$x^* = A^{-1}b = \begin{bmatrix} 0.0909 \\ 0.6364 \end{bmatrix}$

**梯度下降迭代**（步长 $\alpha = 0.25$，最大特征值 $\lambda_{\max}(A) = 4.618$，取 $\alpha < 2/\lambda_{\max}$）：

| 迭代 $k$ | $x_k[0]$ | $x_k[1]$ | $f(x_k)$ | $||\nabla f(x_k)||$ |
|---------|---------|---------|---------|-------------------|
| 0 | 0.0000 | 0.0000 | 0.0000 | 2.2361 |
| 1 | 0.2500 | 0.5000 | -0.6875 | 0.9014 |
| 2 | 0.0625 | 0.5938 | -0.7461 | 0.4330 |
| 3 | 0.1094 | 0.6289 | -0.7484 | 0.0824 |
| 4 | 0.0908 | 0.6353 | -0.7486 | 0.0217 |
| 5 | 0.0910 | 0.6363 | -0.7486 | 0.0043 |

**收敛分析**：

- 条件数 $\kappa(A) = \lambda_{\max}/\lambda_{\min} = 4.618/2.382 \approx 1.94$，收敛较快
- 收敛速率与 $(\frac{\kappa-1}{\kappa+1})^2$ 相关

### 例2：牛顿法求解非线性方程组

**问题**：求解 $\min_x f(x) = e^{x_1 + 3x_2 - 0.1} + e^{x_1 - 3x_2 - 0.1} + e^{-x_1 - 0.1}$

这是逻辑回归的负对数似然函数。

**牛顿迭代**：

| 迭代 $k$ | $x_k[0]$ | $x_k[1]$ | $f(x_k)$ | $||\nabla f(x_k)||$ |
|---------|---------|---------|---------|-------------------|
| 0 | 0.0000 | 0.0000 | 3.0446 | 5.4366 |
| 1 | -0.4197 | 0.0000 | 2.5008 | 1.1246 |
| 2 | -0.4550 | -0.0001 | 2.4536 | 0.0389 |
| 3 | -0.4550 | -0.0001 | 2.4535 | 0.0001 |

**观察**：3次迭代即收敛到机器精度，体现了二次收敛性。

### 例3：拉格朗日乘数法求解约束优化

**问题**：求解
$$\begin{aligned}
\min_{x,y} \quad & x^2 + y^2 \\
\text{s.t.} \quad & x + y = 1
\end{aligned}$$

**拉格朗日函数**：$\mathcal{L} = x^2 + y^2 + \lambda(1 - x - y)$

**KKT条件**：
$$\begin{cases}
2x - \lambda = 0 \\
2y - \lambda = 0 \\
x + y = 1
\end{cases}$$

**解得**：$x = y = 0.5$，$\lambda = 1$，最优值 $f^* = 0.5$

**数值验证**：代入 $x = 0.5 + \delta$，$y = 0.5 - \delta$，有 $f = 0.5 + 2\delta^2 \geq 0.5$。✓

## 🧠 机器学习中的应用

### 1. 线性回归的最小二乘优化

**问题**：给定数据 $(x_i, y_i)_{i=1}^n$，求解
$$\min_w \frac{1}{2n}\sum_{i=1}^n (w^T x_i - y_i)^2 + \frac{\lambda}{2}||w||^2$$

**解析解**（正则化正规方程）：
$$w^* = (X^T X + n\lambda I)^{-1}X^T y$$

**梯度下降实现**：
$$\nabla f(w) = \frac{1}{n}X^T(Xw - y) + \lambda w$$

**复杂度比较**：
- 解析解：$O(nd^2 + d^3)$，适合中小规模
- 梯度下降：$O(nd)$ 每次迭代，适合大规模数据

### 2. 逻辑回归的梯度下降

**问题**：二分类问题，最小化负对数似然
$$\min_w \frac{1}{n}\sum_{i=1}^n \log(1 + e^{-y_i w^T x_i}) + \frac{\lambda}{2}||w||^2$$

**梯度**：
$$\nabla f(w) = -\frac{1}{n}\sum_{i=1}^n \frac{y_i x_i}{1 + e^{y_i w^T x_i}} + \lambda w$$

**随机梯度下降(SGD)**：每次迭代随机选一个样本计算梯度，复杂度 $O(d)$。

### 3. 神经网络的反向传播与优化

**多层感知机**：$f(x) = W_L \sigma(W_{L-1} \sigma(\cdots \sigma(W_1 x)))$

**反向传播**：通过链式法则高效计算梯度
$$\frac{\partial \mathcal{L}}{\partial W_l} = \delta_{l+1} \cdot a_l^T$$

**优化算法选择**：
- SGD + Momentum：加速收敛，逃离鞍点
- Adam：自适应学习率，适合非平稳目标
- L-BFGS：适合小批量、高精度训练

## 🔧 技术实现表征

### Python实现

```python
import numpy as np
from typing import Callable, Tuple

class OptimizationAlgorithms:
    """优化算法实现"""

    @staticmethod
    def gradient_descent(f: Callable, grad_f: Callable, x0: np.ndarray,
                        alpha: float = 0.01, tol: float = 1e-6,
                        max_iter: int = 1000) -> Tuple[np.ndarray, int]:
        """梯度下降法"""
        x = x0.copy()
        for k in range(max_iter):
            g = grad_f(x)
            if np.linalg.norm(g) < tol:
                return x, k + 1
            x = x - alpha * g
        return x, max_iter

    @staticmethod
    def newton_method(f: Callable, grad_f: Callable, hess_f: Callable,
                     x0: np.ndarray, tol: float = 1e-6,
                     max_iter: int = 100) -> Tuple[np.ndarray, int]:
        """牛顿法"""
        x = x0.copy()
        for k in range(max_iter):
            g = grad_f(x)
            if np.linalg.norm(g) < tol:
                return x, k + 1
            H = hess_f(x)
            d = np.linalg.solve(H, -g)  # 求解牛顿方向
            x = x + d
        return x, max_iter

    @staticmethod
    def backtracking_line_search(f: Callable, grad_f: Callable, x: np.ndarray,
                                 d: np.ndarray, alpha: float = 1.0,
                                 rho: float = 0.5, c: float = 1e-4) -> float:
        """回溯线搜索（Armijo条件）"""
        while f(x + alpha * d) > f(x) + c * alpha * np.dot(grad_f(x), d):
            alpha *= rho
        return alpha

# 使用示例
if __name__ == "__main__":
    opt = OptimizationAlgorithms()

    # 二次函数优化
    A = np.array([[4, 1], [1, 3]])
    b = np.array([1, 2])
    f = lambda x: 0.5 * x.T @ A @ x - b.T @ x
    grad_f = lambda x: A @ x - b

    # 梯度下降
    x_gd, iter_gd = opt.gradient_descent(f, grad_f, np.zeros(2), alpha=0.25)
    print(f"梯度下降: x = {x_gd}, 迭代次数 = {iter_gd}")

    # 牛顿法
    hess_f = lambda x: A
    x_newton, iter_newton = opt.newton_method(f, grad_f, hess_f, np.zeros(2))
    print(f"牛顿法: x = {x_newton}, 迭代次数 = {iter_newton}")

```

## Julia实现

```julia
using LinearAlgebra

function gradient_descent(f, grad_f, x0; alpha=0.01, tol=1e-6, max_iter=1000)
    x = copy(x0)
    for k in 1:max_iter
        g = grad_f(x)
        norm(g) < tol && return x, k
        x -= alpha * g
    end
    return x, max_iter
end

function newton_method(f, grad_f, hess_f, x0; tol=1e-6, max_iter=100)
    x = copy(x0)
    for k in 1:max_iter
        g = grad_f(x)
        norm(g) < tol && return x, k
        d = -hess_f(x) \ g
        x += d
    end
    return x, max_iter
end

# 示例
A = [4.0 1.0; 1.0 3.0]
b = [1.0, 2.0]
f(x) = 0.5 * x' * A * x - b' * x
grad_f(x) = A * x - b
hess_f(x) = A

x_gd, iter_gd = gradient_descent(f, grad_f, zeros(2), alpha=0.25)
println("梯度下降: x = $x_gd, 迭代次数 = $iter_gd")

x_newton, iter_newton = newton_method(f, grad_f, hess_f, zeros(2))
println("牛顿法: x = $x_newton, 迭代次数 = $iter_newton")
```

## 📊 总结与展望

优化算法是连接理论与应用的桥梁。本教学文档系统阐述了：

1. **凸优化理论**：凸集与凸函数的判定，保证全局最优
2. **梯度方法**：$O(1/k)$ 收敛速率，适合大规模问题
3. **牛顿法**：二次收敛，适合高精度需求
4. **约束优化**：KKT条件提供最优性判定

**未来发展方向**：
- **非凸优化**：深度学习中的鞍点逃逸、局部最优分析
- **随机优化**：方差缩减、自适应方法的理论保证
- **分布式优化**：联邦学习、去中心化优化算法
- **量子优化**：量子退火、变分量子本征求解器(VQE)
