## 📋 目录概述

计算数学是连接数学理论与实际应用的桥梁，专注于开发算法、分析数值方法并在计算机上实现数学问题的求解。本目录系统涵盖计算数学的六大核心领域：**数值分析**研究插值、数值积分、微分方程数值解与误差分析，是科学计算的基础；**优化理论**涵盖凸优化、线性规划、非线性规划与对偶理论，为机器学习和运筹学提供方法论支撑；**计算几何**探索计算几何算法、Voronoi图与Delaunay三角剖分，支撑计算机图形学与CAD系统；**优化算法**详细阐述梯度下降、内点法、启发式算法与全局优化策略；**并行计算**聚焦并行算法设计、分布式计算与高性能计算架构；**符号计算**研究计算机代数系统、符号积分与Gröbner基理论。从牛顿迭代法到深度学习优化器，从有限元方法到量子计算模拟，计算数学是现代科学与工程不可或缺的技术支柱。

## 🗺️ 学习路径

### 入门阶段（建议时长：6-8周）

**推荐学习顺序：**

1. **起点**：[01-数值分析](01-数值分析.md) — 掌握浮点数运算、插值方法、数值积分
2. **深化**：[01-数值分析-增强版](01-数值分析-增强版.md) — 深入收敛性分析、稳定性理论、微分方程数值解
3. **基础**：[02-优化理论](02-优化理论.md) — 学习凸集、凸函数、线性规划基础

**配套练习：**

- 实现拉格朗日插值与样条插值
- 编写龙贝格积分程序
- 实现牛顿迭代法与割线法
- 用单纯形法求解线性规划问题

### 进阶方向

#### 方向A：科学计算与数值方法（科学与工程方向）

**学习序列：**

1. [01-数值分析-增强版](01-数值分析-增强版.md) — 偏微分方程数值解、有限元方法
2. [02-计算几何](02-计算几何.md) — 网格生成、几何处理算法

**先修要求：**

- 微积分（多变量微积分）
- 线性代数（矩阵计算）
- 常微分方程/偏微分方程
- Python/MATLAB/Julia编程

#### 方向B：优化与机器学习（AI/Data Science方向）

**学习序列：**

1. [02-优化理论-增强版](02-优化理论-增强版.md) — 凸优化、对偶理论、KKT条件
2. [03-优化算法](03-优化算法.md) — 梯度下降变体、随机优化、二阶方法

**先修要求：**

- 线性代数（矩阵分解、特征值）
- 概率论与统计
- Python编程（NumPy/SciPy/PyTorch）
- [02-优化理论](02-优化理论.md)基础

#### 方向C：高性能计算与符号计算（系统方向）

**学习序列：**

1. [04-并行计算](04-并行计算.md) — MPI/OpenMP、GPU计算、分布式算法
2. [05-符号计算](05-符号计算.md) — 符号积分、多项式运算、Gröbner基

**先修要求：**

- C/C++编程
- 计算机体系结构基础
- 抽象代数（用于符号计算）
- [01-数值分析](01-数值分析.md)基础
