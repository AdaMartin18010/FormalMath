## 📋 目录概述

本目录涵盖计算数学核心领域：数值分析（插值、求积、微分方程数值解）、优化理论（凸优化、非线性规划）、计算几何、优化算法、并行计算、符号计算。内容兼顾理论分析与算法实现。

## 🗺️ 学习路径建议

1. **入门**: 01-数值分析 → 02-优化理论  
2. **数值方向**: 01-数值分析-增强版（需线性代数、分析学）  
3. **优化方向**: 02-优化理论-增强版 → 03-优化算法  
4. **计算方向**: 02-计算几何、04-并行计算、05-符号计算（可按需选读）
