## 一、数值线性代数

### 1.1 LU分解

**定义 1.1** (LU分解)
矩阵 $A \in \mathbb{R}^{n \times n}$ 的LU分解：

$$A = LU$$

其中 $L$ 是单位下三角矩阵，$U$ 是上三角矩阵。

**存在条件**: 所有顺序主子式非零。

**算法** (Doolittle算法)：
对 $k = 1, \ldots, n$:
- $u_{kj} = a_{kj} - \sum_{m=1}^{k-1} l_{km}u_{mj}$，$j = k, \ldots, n$
- $l_{ik} = \frac{1}{u_{kk}}\left(a_{ik} - \sum_{m=1}^{k-1} l_{im}u_{mk}\right)$，$i = k+1, \ldots, n$

**计算复杂度**: $O(n^3/3)$

### 1.2 Cholesky分解

**定义 1.2** (Cholesky分解)
对称正定矩阵 $A$ 的Cholesky分解：

$$A = LL^T$$

其中 $L$ 是下三角矩阵。

**算法**：
$$l_{jj} = \sqrt{a_{jj} - \sum_{k=1}^{j-1} l_{jk}^2}$$
$$l_{ij} = \frac{1}{l_{jj}}\left(a_{ij} - \sum_{k=1}^{j-1} l_{ik}l_{jk}\right), \quad i > j$$

**计算复杂度**: $O(n^3/6)$（约为LU的一半）

### 1.3 QR分解

**定义 1.3** (QR分解)
矩阵 $A \in \mathbb{R}^{m \times n}$ ($m \geq n$) 的QR分解：

$$A = QR$$

其中 $Q \in \mathbb{R}^{m \times m}$ 是正交矩阵，$R \in \mathbb{R}^{m \times n}$ 是上三角矩阵。

**计算方法**:
- **Gram-Schmidt正交化**: 理论简单，数值不稳定
- **Householder变换**: 数值稳定，$O(mn^2 - n^3/3)$
- **Givens旋转**: 适合稀疏矩阵和并行计算

**Householder反射**: 
$$H = I - 2\frac{vv^T}{v^Tv}$$

### 1.4 迭代法求解线性系统

**定义 1.4** (Jacobi迭代)
将 $A = D - L - U$，其中 $D$ 对角，$L$ 严格下三角，$U$ 严格上三角。

$$Dx^{(k+1)} = (L + U)x^{(k)} + b$$

分量形式：
$$x_i^{(k+1)} = \frac{1}{a_{ii}}\left(b_i - \sum_{j \neq i} a_{ij}x_j^{(k)}\right)$$

**收敛条件**: $A$ 严格对角占优或对称正定。

**Gauss-Seidel迭代**:
$$(D - L)x^{(k+1)} = Ux^{(k)} + b$$

使用最新计算值，通常收敛更快。

**SOR (逐次超松弛)**:
$$x_i^{(k+1)} = (1-\omega)x_i^{(k)} + \frac{\omega}{a_{ii}}\left(b_i - \sum_{j < i} a_{ij}x_j^{(k+1)} - \sum_{j > i} a_{ij}x_j^{(k)}\right)$$

最优松弛参数 $\omega_{opt}$ 对特殊矩阵可解析确定。

### 1.5 共轭梯度法

**定义 1.5** (共轭梯度法 CG)
求解 $Ax = b$，其中 $A$ 对称正定。

**算法**：
1. 初始化：$r_0 = b - Ax_0$，$p_0 = r_0$
2. 对 $k = 0, 1, 2, \ldots$:
   - $\alpha_k = \frac{r_k^T r_k}{p_k^T A p_k}$
   - $x_{k+1} = x_k + \alpha_k p_k$
   - $r_{k+1} = r_k - \alpha_k A p_k$
   - $\beta_k = \frac{r_{k+1}^T r_{k+1}}{r_k^T r_k}$
   - $p_{k+1} = r_{k+1} + \beta_k p_k$

**性质**:
- 有限步收敛（理论上最多 $n$ 步）
- 实际中作为迭代法使用（预条件CG）
- 收敛速度取决于 $A$ 的条件数

## 二、有限元方法

### 2.1 弱形式与变分问题

**定义 2.1** (弱形式)
考虑边值问题：

$$-u'' = f \text{ in } (0, 1), \quad u(0) = u(1) = 0$$

弱形式：求 $u \in H_0^1(0,1)$ 使得

$$\int_0^1 u'v' dx = \int_0^1 fv dx, \quad \forall v \in H_0^1(0,1)$$

**变分原理**: 等价于最小化能量泛函
$$J(u) = \frac{1}{2}\int_0^1 (u')^2 dx - \int_0^1 fu dx$$

### 2.2 Galerkin逼近

**定义 2.2** (Galerkin方法)
选择有限维子空间 $V_h \subset H_0^1$，求 $u_h \in V_h$ 使得

$$a(u_h, v_h) = (f, v_h), \quad \forall v_h \in V_h$$

其中 $a(u, v) = \int_0^1 u'v' dx$。

**基函数展开**: $u_h = \sum_{j=1}^N c_j \phi_j$

线性系统：$Ac = F$，其中 $A_{ij} = a(\phi_j, \phi_i)$，$F_i = (f, \phi_i)$

### 2.3 有限元空间

**定义 2.3** (Lagrange有限元)
在网格 $\mathcal{T}_h = \{K_i\}$ 上定义分段多项式空间。

**P1元**（线性元）:
- 节点：网格顶点
- 基函数："帽函数" $\phi_i(x_j) = \delta_{ij}$
- 局部自由度：顶点值

**P2元**（二次元）:
- 节点：顶点和边中点
- 局部自由度：顶点值和边中点值

### 2.4 误差分析

**定理 2.1** (Céa引理)
设 $a(\cdot, \cdot)$ 连续且强制，$u$ 是真解，$u_h$ 是Galerkin解，则

$$\|u - u_h\|_V \leq \frac{M}{\alpha} \inf_{v_h \in V_h} \|u - v_h\|_V$$

其中 $M$ 是连续性常数，$\alpha$ 是强制性常数。

**收敛阶**:
- P1元：$\|u - u_h\|_{H^1} = O(h)$，$\|u - u_h\|_{L^2} = O(h^2)$
- P2元：$\|u - u_h\|_{H^1} = O(h^2)$，$\|u - u_h\|_{L^2} = O(h^3)$

## 三、谱方法

### 3.1 多项式逼近

**定义 3.1** (谱逼近)
用全局光滑函数（通常是多项式或三角函数）逼近解。

**Fourier谱方法**:
周期问题上用三角多项式：
$$u_N(x) = \sum_{k=-N/2}^{N/2-1} \hat{u}_k e^{ikx}$$

**Chebyshev谱方法**:
非周期问题上用Chebyshev多项式：
$$T_n(x) = \cos(n \arccos x), \quad x \in [-1, 1]$$

### 3.2 谱精度

**定理 3.1** (谱收敛)
若 $u$ 无穷可微，则谱逼近误差指数衰减：

$$\|u - u_N\| \leq Ce^{-cN}$$

若 $u$ 有 $m$ 阶连续导数，则
$$\|u - u_N\| = O(N^{-m})$$

对比有限元：需要 $h$ 指数减小才能达到相同精度。

### 3.3 快速Fourier变换(FFT)

**定义 3.2** (FFT)
计算离散Fourier变换的快速算法：

$$\hat{u}_k = \sum_{j=0}^{N-1} u_j e^{-2\pi i j k / N}$$

**计算复杂度**:
- 直接计算：$O(N^2)$
- FFT算法：$O(N \log N)$

**应用**: 谱微分、卷积计算、Poisson方程快速求解

## 总结

本章节介绍了计算数学的基础概念：

- **数值线性代数**: LU、Cholesky、QR分解，Jacobi、Gauss-Seidel、CG迭代法
- **有限元方法**: 弱形式、Galerkin逼近、有限元空间、误差分析
- **谱方法**: 全局逼近、谱收敛、FFT算法

这些方法是科学计算的核心工具。
