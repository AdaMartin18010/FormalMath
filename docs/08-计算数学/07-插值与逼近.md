## 1. 概述

### 1.1 问题定义

**插值问题**：给定数据点$(x_i, y_i)_{i=0}^n$，求函数$P(x)$使得
$$P(x_i) = y_i, \quad i = 0, 1, \ldots, n$$

**逼近问题**：给定函数$f(x)$，在函数空间$\mathcal{S}$中寻找$P(x)$使得某种误差度量最小。

### 1.2 问题分类

| 类型 | 特点 | 应用 |
|-----|------|------|
| 多项式插值 | 简单、解析性好 | 理论分析、小数据 |
| 样条插值 | 光滑、避免振荡 | CAD、图形学 |
| 三角插值 | 周期性数据 | 信号处理 |
| 有理插值 | 处理极点 | 特殊函数 |
| 最小二乘 | 噪声数据处理 | 数据拟合 |

## 2. 多项式插值

### 2.1 Lagrange插值

#### 直观理解

Lagrange插值构造基函数$\ell_i(x)$，每个基函数在$x_i$处为1，在其他节点处为0。

#### 形式定义

**Lagrange基函数**：
$$\ell_i(x) = \prod_{j=0, j \neq i}^n \frac{x - x_j}{x_i - x_j}$$

**插值多项式**：
$$P_n(x) = \sum_{i=0}^n y_i \ell_i(x)$$

#### 误差分析

**定理 2.1**（插值误差）：若$f \in C^{n+1}[a,b]$，则对任意$x \in [a,b]$，存在$\xi \in [a,b]$使得
$$f(x) - P_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!} \prod_{i=0}^n (x - x_i)$$

#### Python实现

```python
import numpy as np
import matplotlib.pyplot as plt

def lagrange_basis(x_nodes, i, x):
    """计算第i个Lagrange基函数在x处的值"""
    n = len(x_nodes)
    result = 1.0
    for j in range(n):
        if j != i:
            result *= (x - x_nodes[j]) / (x_nodes[i] - x_nodes[j])
    return result

def lagrange_interpolation(x_nodes, y_nodes, x):
    """Lagrange插值"""
    n = len(x_nodes)
    result = 0.0
    for i in range(n):
        result += y_nodes[i] * lagrange_basis(x_nodes, i, x)
    return result

# 向量化版本
def lagrange_interpolation_vec(x_nodes, y_nodes, x_array):
    """向量化Lagrange插值"""
    n = len(x_nodes)
    result = np.zeros_like(x_array, dtype=float)
    
    for i in range(n):
        # 计算第i个基函数在所有x处的值
        basis = np.ones_like(x_array)
        for j in range(n):
            if j != i:
                basis *= (x_array - x_nodes[j]) / (x_nodes[i] - x_nodes[j])
        result += y_nodes[i] * basis
    
    return result

# 测试：插值sin函数
x_nodes = np.linspace(0, np.pi, 5)
y_nodes = np.sin(x_nodes)
x_fine = np.linspace(0, np.pi, 200)
y_interp = lagrange_interpolation_vec(x_nodes, y_nodes, x_fine)

plt.figure(figsize=(10, 5))
plt.plot(x_fine, np.sin(x_fine), 'b-', label='sin(x)')
plt.plot(x_fine, y_interp, 'r--', label='Lagrange插值')
plt.scatter(x_nodes, y_nodes, c='red', zorder=5)
plt.legend()
plt.title('Lagrange插值 sin(x)')
plt.show()
```

**复杂度分析**：
- 计算一个点：$O(n^2)$
- 计算$m$个点：$O(mn^2)$

### 2.2 Newton插值

#### 直观理解

Newton插值使用**差商**递归构造，便于添加新节点。

#### 差商定义

**零阶差商**：$f[x_i] = f(x_i)$

**k阶差商**：
$$f[x_0, x_1, \ldots, x_k] = \frac{f[x_1, \ldots, x_k] - f[x_0, \ldots, x_{k-1}]}{x_k - x_0}$$

#### Newton插值公式

$$P_n(x) = f[x_0] + f[x_0, x_1](x-x_0) + f[x_0, x_1, x_2](x-x_0)(x-x_1) + \cdots$$

```python
def divided_differences(x_nodes, y_nodes):
    """计算差商表"""
    n = len(x_nodes)
    F = np.zeros((n, n))
    F[:, 0] = y_nodes
    
    for j in range(1, n):
        for i in range(n - j):
            F[i, j] = (F[i + 1, j - 1] - F[i, j - 1]) / (x_nodes[i + j] - x_nodes[i])
    
    return F[0, :]  # 返回第一行（各阶差商）

def newton_interpolation(x_nodes, y_nodes, x):
    """Newton插值"""
    coeffs = divided_differences(x_nodes, y_nodes)
    n = len(x_nodes)
    
    result = coeffs[-1]
    for i in range(n - 2, -1, -1):
        result = result * (x - x_nodes[i]) + coeffs[i]
    
    return result

# 添加新节点的高效实现
def newton_add_node(x_nodes, y_nodes, new_x, new_y):
    """向Newton插值添加新节点"""
    n = len(x_nodes)
    new_coeff = new_y
    
    # 计算新差商
    for i in range(n):
        new_coeff = (new_coeff - y_nodes[i]) / (new_x - x_nodes[i])
    
    return np.append(y_nodes, new_coeff)
```

### 2.3 Hermite插值

#### 形式定义

给定函数值和导数值：$f(x_i) = y_i$，$f'(x_i) = y'_i$

**Hermite插值多项式**满足：
$$H(x_i) = y_i, \quad H'(x_i) = y'_i$$

```python
def hermite_interpolation(x_nodes, y_nodes, dy_nodes, x):
    """Hermite插值（三次）"""
    n = len(x_nodes)
    result = 0.0
    
    for i in range(n):
        # 计算基函数
        L_i = 1.0
        dL_i = 0.0
        for j in range(n):
            if j != i:
                L_i *= (x - x_nodes[j]) / (x_nodes[i] - x_nodes[j])
                dL_i += 1.0 / (x_nodes[i] - x_nodes[j])
        
        # Hermite基函数
        h_i = (1 - 2 * (x - x_nodes[i]) * dL_i) * L_i**2
        hbar_i = (x - x_nodes[i]) * L_i**2
        
        result += y_nodes[i] * h_i + dy_nodes[i] * hbar_i
    
    return result
```

## 3. 样条插值

### 3.1 三次样条

#### 形式定义

**三次样条**$S(x)$满足：
1. $S(x_i) = y_i$（插值条件）
2. $S(x)$在每个子区间$[x_i, x_{i+1}]$上是三次多项式
3. $S(x) \in C^2[a,b]$（二阶连续可微）

#### 三弯矩方程

设$M_i = S''(x_i)$，则在$[x_i, x_{i+1}]$上：
$$S(x) = M_i \frac{(x_{i+1}-x)^3}{6h_i} + M_{i+1} \frac{(x-x_i)^3}{6h_i} + A_i(x-x_i) + B_i$$

其中$h_i = x_{i+1} - x_i$。

连续性条件导出**三弯矩方程**：
$$\mu_i M_{i-1} + 2M_i + \lambda_i M_{i+1} = d_i$$

```python
from scipy.sparse import diags
def cubic_spline_coeffs(x_nodes, y_nodes, bc_type='natural'):
    """计算三次样条系数
    bc_type: 'natural'(自然), 'clamped'(固定), 'periodic'(周期)
    """
    n = len(x_nodes) - 1
    h = np.diff(x_nodes)
    
    # 计算右端项
    d = np.zeros(n + 1)
    for i in range(1, n):
        d[i] = 6 * ((y_nodes[i+1] - y_nodes[i]) / h[i] - 
                    (y_nodes[i] - y_nodes[i-1]) / h[i-1])
    
    # 构造三对角矩阵
    main_diag = 2 * np.ones(n + 1)
    off_diag = np.zeros(n)
    
    for i in range(1, n):
        off_diag[i-1] = h[i-1] / (h[i-1] + h[i])
        main_diag[i] = 2
    
    # 边界条件
    if bc_type == 'natural':
        main_diag[0] = 1
        main_diag[-1] = 1
        d[0] = d[-1] = 0
    elif bc_type == 'clamped':
        # 需要额外参数指定端点导数
        pass
    
    # 求解三对角系统
    from scipy.linalg import solve_banded
    ab = np.zeros((3, n + 1))
    ab[0, 1:] = off_diag  # 上对角
    ab[1, :] = main_diag  # 主对角
    ab[2, :-1] = off_diag  # 下对角
    
    M = solve_banded((1, 1), ab, d)
    
    # 计算多项式系数
    a = y_nodes[:-1]
    b = (y_nodes[1:] - y_nodes[:-1]) / h - h * (2 * M[:-1] + M[1:]) / 6
    c = M[:-1] / 2
    d_coef = (M[1:] - M[:-1]) / (6 * h)
    
    return a, b, c, d_coef

def cubic_spline_eval(x_nodes, a, b, c, d, x):
    """评估三次样条"""
    # 找到x所在的区间
    idx = np.searchsorted(x_nodes, x) - 1
    idx = np.clip(idx, 0, len(x_nodes) - 2)
    
    dx = x - x_nodes[idx]
    return a[idx] + b[idx] * dx + c[idx] * dx**2 + d[idx] * dx**3

# 使用SciPy的简化版本
```

### 3.2 B-样条

#### 定义

**B-样条基函数**递归定义：
$$N_{i,0}(x) = \begin{cases} 1 & t_i \leq x < t_{i+1} \\ 0 & \text{otherwise} \end{cases}$$

$$N_{i,k}(x) = \frac{x - t_i}{t_{i+k} - t_i} N_{i,k-1}(x) + \frac{t_{i+k+1} - x}{t_{i+k+1} - t_{i+1}} N_{i+1,k-1}(x)$$

```python
from scipy.interpolate import BSpline, make_interp_spline
```

### 3.3 样条逼近

#### 最小二乘样条

当数据有噪声时，不要求精确通过每个点，而是最小化：
$$\min_S \sum_{i=0}^n w_i (S(x_i) - y_i)^2 + \lambda \int_a^b (S''(x))^2 dx$$

```python
from scipy.interpolate import make_lsq_spline

def smoothing_spline(x_data, y_data, lam=0.1):
    """平滑样条"""
    n = len(x_data)
    
    # 使用scipy的LSQUnivariateSpline
    from scipy.interpolate import LSQUnivariateSpline
    
    # 选择内部节点
    t = np.linspace(x_data[0], x_data[-1], 8)[1:-1]
    
    spl = LSQUnivariateSpline(x_data, y_data, t)
    
    return spl

# 演示
def demo_smoothing():
    x = np.linspace(0, 10, 100)
    y_clean = np.sin(x) + 0.1 * x

    plt.show()

demo_smoothing()
```

## 4. 最佳逼近

### 4.1 最小二乘逼近

#### 离散最小二乘

给定数据$(x_i, y_i)$，求$P(x) = \sum_{j=0}^m c_j \phi_j(x)$最小化：
$$E = \sum_{i=0}^n w_i (P(x_i) - y_i)^2$$

**法方程**：
$$\sum_{j=0}^m \langle \phi_j, \phi_k \rangle c_j = \langle y, \phi_k \rangle$$

其中$\langle f, g \rangle = \sum_{i=0}^n w_i f(x_i) g(x_i)$

```python
def least_squares_fit(x_data, y_data, degree, w=None):
    """多项式最小二乘拟合"""
    if w is None:
        w = np.ones(len(x_data))
    
    # 构造Vandermonde矩阵
    A = np.vander(x_data, degree + 1, increasing=True)
    
    # 加权
    W = np.diag(np.sqrt(w))
    Aw = W @ A
    yw = W @ y_data
    
    # 求解最小二乘问题
    coeffs, _, _, _ = np.linalg.lstsq(Aw, yw, rcond=None)
    
    return coeffs

# 使用numpy.polyfit
def polyfit_demo():
    x = np.linspace(0, 5, 50)
    y = 2 * x**2 - 3 * x + 1 + np.random.normal(0, 1, len(x))
    
    coeffs = np.polyfit(x, y, 2)
    p = np.poly1d(coeffs)
    
    x_fine = np.linspace(0, 5, 200)
    
    plt.figure(figsize=(10, 5))
    plt.scatter(x, y, alpha=0.5, label='数据')
    plt.plot(x_fine, 2*x_fine**2 - 3*x_fine + 1, 'g--', label='真实')
    plt.plot(x_fine, p(x_fine), 'r-', label='拟合')
    plt.legend()
    plt.show()
    
    print(f"拟合系数: {coeffs}")

polyfit_demo()
```

$$\|f - P\|_\infty < \epsilon$$

### 4.3 Chebyshev逼近

#### Chebyshev多项式

$$T_n(x) = \cos(n \arccos x), \quad x \in [-1, 1]$$

**正交性**：
$$\int_{-1}^1 \frac{T_m(x) T_n(x)}{\sqrt{1-x^2}} dx = \begin{cases} 0 & m \neq n \\ \pi & m = n = 0 \\ \pi/2 & m = n \neq 0 \end{cases}$$

```python
def chebyshev_poly(n, x):
    """计算Chebyshev多项式"""
    if n == 0:
        return np.ones_like(x)
    elif n == 1:
        return x
    else:
        T_prev2 = np.ones_like(x)
        T_prev1 = x
        for k in range(2, n + 1):
            T_curr = 2 * x * T_prev1 - T_prev2
            T_prev2 = T_prev1
            T_prev1 = T_curr
        return T_curr

def chebyshev_nodes(n, a=-1, b=1):
    """Chebyshev节点"""
    k = np.arange(n + 1)
    x = np.cos((2 * k + 1) * np.pi / (2 * (n + 1)))
    return (b - a) / 2 * x + (a + b) / 2

def chebyshev_interp(f, n, a=-1, b=1):
    """Chebyshev插值"""
    x_nodes = chebyshev_nodes(n, a, b)
    y_nodes = f(x_nodes)
    
    # 转换为[-1,1]区间
    x_std = 2 * (x_nodes - a) / (b - a) - 1
    
    # 计算Chebyshev系数
    c = np.zeros(n + 1)
    for k in range(n + 1):
        for j in range(n + 1):
            c[k] += y_nodes[j] * np.cos(k * (2 * j + 1) * np.pi / (2 * (n + 1)))
        c[k] *= 2 / (n + 1)
    c[0] /= 2
    
    return x_nodes, y_nodes, c

# 演示Chebyshev插值
x_fine = np.linspace(-1, 1, 500)
y_true = runge_function(x_fine)

plt.figure(figsize=(12, 4))
for i, n in enumerate([5, 10, 20]):
    plt.subplot(1, 3, i + 1)
    
    x_nodes, y_nodes, c = chebyshev_interp(runge_function, n)
    
    # 评估Chebyshev展开
    y_cheb = np.zeros_like(x_fine)
    for k in range(n + 1):
        y_cheb += c[k] * chebyshev_poly(k, x_fine)
    
    plt.plot(x_fine, y_true, 'b-', label='Runge函数')
    plt.plot(x_fine, y_cheb, 'r--', label=f'{n}阶Chebyshev')
    plt.scatter(x_nodes, y_nodes, c='red', zorder=5)
    plt.ylim(-0.5, 1.5)
    plt.legend()
    plt.title(f'Chebyshev插值 (n={n})')

plt.tight_layout()
plt.show()
```

```python
def bezier_curve(control_points, n_points=100):
    """计算Bézier曲线"""
    n = len(control_points) - 1
    t = np.linspace(0, 1, n_points)
    
    curve = np.zeros((n_points, 2))
    for i, ti in enumerate(t):
        for j, p in enumerate(control_points):
            # Bernstein多项式
            binom = np.math.comb(n, j)
            curve[i] += binom * (ti**j) * ((1-ti)**(n-j)) * p
    
    return curve

# 演示Bézier曲线
```

### 5.3 数据压缩：曲线简化

```python
def ramer_douglas_peucker(points, epsilon):
    """Ramer-Douglas-Peucker算法"""
    if len(points) < 3:
        return points
    
    # 找到距离线段最远的点
    start, end = points[0], points[-1]
    line_vec = end - start
    line_len_sq = np.dot(line_vec, line_vec)
    
    max_dist = 0
    max_idx = 0
    
    for i in range(1, len(points) - 1):
        point_vec = points[i] - start
        if line_len_sq == 0:
            dist = np.linalg.norm(point_vec)
        else:
            t = max(0, min(1, np.dot(point_vec, line_vec) / line_len_sq))
            projection = start + t * line_vec
            dist = np.linalg.norm(points[i] - projection)
        
        if dist > max_dist:
            max_dist = dist
            max_idx = i
    
    if max_dist > epsilon:
        # 递归简化
        left = ramer_douglas_peucker(points[:max_idx+1], epsilon)
        right = ramer_douglas_peucker(points[max_idx:], epsilon)
        return np.vstack([left[:-1], right])
    else:
        return np.array([start, end])

# 演示曲线简化
t = np.linspace(0, 4*np.pi, 200)
points = np.column_stack([t, np.sin(t) + 0.1 * np.random.randn(len(t))])

simplified_1 = ramer_douglas_peucker(points, 0.1)
simplified_2 = ramer_douglas_peucker(points, 0.5)

plt.figure(figsize=(12, 4))
plt.plot(points[:, 0], points[:, 1], 'gray', alpha=0.5, label='原始数据')
plt.plot(simplified_1[:, 0], simplified_1[:, 1], 'r-o', label=f'ε=0.1 ({len(simplified_1)}点)')
plt.plot(simplified_2[:, 0], simplified_2[:, 1], 'b-s', label=f'ε=0.5 ({len(simplified_2)}点)')
plt.legend()
plt.title('Ramer-Douglas-Peucker曲线简化')
plt.show()
```
