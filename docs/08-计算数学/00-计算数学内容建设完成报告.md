## 📚 新建核心文档详情

### 1. 06-数值线性代数.md

**文件大小**: 20,602字节
**MSC分类**: @

**内容结构**:

- 概述与基本概念（条件数、稳定性）
- 直接法：LU分解、Cholesky分解、QR分解、SVD
- 迭代法：Jacobi、Gauss-Seidel、CG、GMRES
- 特征值问题：幂迭代、QR算法、Lanczos
- 应用案例：图像去模糊、PageRank、PCA
- 习题：5理论 + 5编程

**特色**:

- 完整Python实现所有算法
- 手算示例配合代码验证
- 复杂度分析表格

### 2. 07-插值与逼近.md

**文件大小**: 22,100字节
**MSC分类**: 65D05

**内容结构**:

- 多项式插值：Lagrange、Newton、Hermite
- Runge现象与分段插值
- 样条插值：三次样条、B-样条、平滑样条
- 最佳逼近：最小二乘、Chebyshev多项式
- 应用案例：Bézier曲线、信号滤波、曲线简化
- 习题：5理论 + 5编程

**特色**:

- 可视化Runge现象
- 完整的样条实现（含边界条件处理）
- 收敛性分析图表

### 3. 08-数值微积分.md

**文件大小**: 26,468字节
**MSC分类**: 65D30, @

**内容结构**:

- 数值微分：有限差分、Richardson外推、复步长、自动微分
- 数值积分：Newton-Cotes、Gauss求积、Romberg、自适应
- ODE求解：Euler、RK4、Adams方法、刚性方程
- 应用案例：单摆、Lotka-Volterra、Black-Scholes
- 习题：5理论 + 5编程

**特色**:

- 数值稳定性实验（不同步长比较）
- 完整的ODE求解器实现
- 金融领域应用（期权定价）

### 4. 09-偏微分方程数值解.md

**文件大小**: 24,860字节
**MSC分类**: @

**内容结构**:

- 有限差分法：椭圆/抛物/双曲方程
- 稳定性分析：von Neumann方法
- 有限元法：变分形式、Galerkin、误差估计
- 多重网格：V-cycle、W-cycle
- 应用案例：顶盖驱动流、热传导、结构力学
- 习题：5理论 + 5编程

**特色**:

- 完整的2D Poisson求解器
- CFL条件实验
- 收敛性分析代码

### 5. 10-优化算法.md

**文件大小**: 29,827字节
**MSC分类**: 90C26

**内容结构**:

- 无约束优化：梯度下降（动量、Nesterov、Adam）、牛顿法、拟牛顿法
- 约束优化：罚函数法、内点法
- 全局优化：遗传算法、模拟退火、粒子群优化
- 随机优化：SGD
- 应用案例：神经网络训练、TSP
- 习题：5理论 + 5编程

**特色**:

- 完整的遗传算法实现
- 各种梯度下降变体比较
- 优化路径可视化

- 快速多极子法

**特色**:

## 📈 学习路径完善

### 推荐学习序列

#### 入门路径（8周）

```
Week 1-2: 01-数值分析 + 06-数值线性代数（基础部分）
Week 3-4: 08-数值微积分（差分、基础ODE）
Week 5-6: 07-插值与逼近
Week 7-8: 10-优化算法（梯度下降基础）
```

#### 进阶路径（12周）

```
Week 1-3: 06-数值线性代数（完整）+ 09-偏微分方程数值解
Week 4-6: 08-数值微积分（完整，含刚性方程）
Week 7-9: 10-优化算法（完整，含全局优化）
Week 10-12: 11-快速算法 + 04-并行计算
```

#### 专业方向

- **科学计算**：06 → 08 → 09 → 11
- **机器学习**：10 → 06（CG）→ 11（稀疏）
- **高性能计算**：11 → 04 → 09（多重网格）
