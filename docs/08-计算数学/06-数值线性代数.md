## 1. 概述与基本概念

### 1.1 什么是数值线性代数

数值线性代数是研究**用数值方法求解线性代数问题**的学科，主要包括：
- 线性方程组求解：$Ax = b$
- 特征值问题：$Ax = \lambda x$
- 最小二乘问题：$\min_x \|Ax - b\|_2$
- 矩阵分解：LU、QR、SVD等

### 1.2 核心问题分类

| 问题类型 | 数学描述 | 主要算法 | 复杂度 |
|---------|---------|---------|--------|
| 稠密线性方程组 | $Ax = b$, $A \in \mathbb{R}^{n \times n}$ | LU分解 | $O(n^3)$ |
| 稀疏线性方程组 | 同上，$A$稀疏 | 迭代法 | $O(n)$ 或 $O(n^{1.5})$ |
| 对称正定系统 | $Ax = b$, $A = A^T \succ 0$ | Cholesky, CG | $O(n^3)$ 或 $O(n)$ |
| 最小二乘 | $\min \|Ax - b\|_2$ | QR, SVD | $O(mn^2)$ |
| 特征值问题 | $Ax = \lambda x$ | QR算法, 幂迭代 | $O(n^3)$ 每步 |

### 1.3 数值稳定性与条件数

**条件数**衡量问题对扰动的敏感程度：
$$\kappa(A) = \|A\| \cdot \|A^{-1}\|$$

对于2-范数：$\kappa_2(A) = \frac{\sigma_{\max}(A)}{\sigma_{\min}(A)}$

**定理 1.1**（扰动分析）：设 $Ax = b$，$(A + \Delta A)(x + \Delta x) = b + \Delta b$，则
$$\frac{\|\Delta x\|}{\|x\|} \leq \frac{\kappa(A)}{1 - \kappa(A)\frac{\|\Delta A\|}{\|A\|}} \left( \frac{\|\Delta A\|}{\|A\|} + \frac{\|\Delta b\|}{\|b\|} \right)$$

## 2. 直接法：矩阵分解

### 2.1 LU分解

#### 直观理解

LU分解将矩阵$A$分解为下三角矩阵$L$和上三角矩阵$U$的乘积：
$$A = LU$$

这样$Ax = b$变为$LUx = b$，可通过前代和回代求解：
1. 解 $Ly = b$（前代）
2. 解 $Ux = y$（回代）

#### 形式定义

**定理 2.1**（LU分解存在性）：若$A$的所有顺序主子式非零，则存在唯一的单位下三角矩阵$L$和上三角矩阵$U$使得$A = LU$。

#### 算法描述

**Doolittle算法**（LU分解伪代码）：

```
算法: LU分解 (Doolittle)
输入: 矩阵 A ∈ ℝ^(n×n)
输出: 下三角矩阵 L，上三角矩阵 U

对于 k = 1 到 n:
    对于 j = k 到 n:
        U[k,j] = A[k,j] - Σ(L[k,m]·U[m,j], m=1到k-1)
    
    对于 i = k+1 到 n:
        L[i,k] = (A[i,k] - Σ(L[i,m]·U[m,k], m=1到k-1)) / U[k,k]

返回 L, U
```

**复杂度分析**：
- 乘法次数：$\sum_{k=1}^n (n-k)^2 = \frac{n(n-1)(2n-1)}{6} \approx \frac{n^3}{3}$
- 总复杂度：$O(n^3)$

#### 数值示例

**手算示例**：对矩阵 $A = \begin{bmatrix} 2 & 1 & 1 \\ 4 & 3 & 3 \\ 8 & 7 & 9 \end{bmatrix}$ 进行LU分解

步骤1：第一列
- $u_{11} = 2$, $u_{12} = 1$, $u_{13} = 1$
- $l_{21} = 4/2 = 2$, $l_{31} = 8/2 = 4$

步骤2：第二列
- $u_{22} = 3 - 2 \times 1 = 1$, $u_{23} = 3 - 2 \times 1 = 1$
- $l_{32} = (7 - 4 \times 1) / 1 = 3$

步骤3：第三列
- $u_{33} = 9 - 4 \times 1 - 3 \times 1 = 2$

结果：
$$L = \begin{bmatrix} 1 & 0 & 0 \\ 2 & 1 & 0 \\ 4 & 3 & 1 \end{bmatrix}, \quad U = \begin{bmatrix} 2 & 1 & 1 \\ 0 & 1 & 1 \\ 0 & 0 & 2 \end{bmatrix}$$

**Python实现**：

```python
import numpy as np

def lu_decomposition(A):
    """Doolittle LU分解"""
    n = len(A)
    L = np.eye(n)
    U = np.zeros((n, n))
    
    for k in range(n):
        # 计算U的第k行
        for j in range(k, n):
            U[k, j] = A[k, j] - np.dot(L[k, :k], U[:k, j])
        
        # 计算L的第k列
        for i in range(k + 1, n):
            L[i, k] = (A[i, k] - np.dot(L[i, :k], U[:k, k])) / U[k, k]
    
    return L, U

# 验证
A = np.array([[2, 1, 1], [4, 3, 3], [8, 7, 9]], dtype=float)
L, U = lu_decomposition(A)
print("L =\n", L)
print("U =\n", U)
print("LU =\n", L @ U)
print("残差:", np.max(np.abs(L @ U - A)))
```

#### 部分主元LU分解

为避免小主元导致的数值不稳定，采用**部分主元**策略：
$$PA = LU$$
其中$P$是置换矩阵。

```python
def lu_decomposition_pivot(A):
    """带部分主元的LU分解"""
    n = len(A)
    L = np.eye(n)
    U = A.copy()
    P = np.arange(n)
    
    for k in range(n - 1):
        # 选主元
        pivot_idx = np.argmax(np.abs(U[k:, k])) + k
        if pivot_idx != k:
            U[[k, pivot_idx]] = U[[pivot_idx, k]]
            P[[k, pivot_idx]] = P[[pivot_idx, k]]
            if k > 0:
                L[[k, pivot_idx], :k] = L[[pivot_idx, k], :k]
        
        # 消元
        for i in range(k + 1, n):
            L[i, k] = U[i, k] / U[k, k]
            U[i, k:] -= L[i, k] * U[k, k:]
    
    return L, U, P
```

### 2.2 Cholesky分解

#### 直观理解

对于**对称正定矩阵**$A = A^T \succ 0$，存在唯一的下三角矩阵$G$使得：
$$A = GG^T$$

比LU分解快约一倍，且数值稳定。

#### 算法描述

```
算法: Cholesky分解
输入: 对称正定矩阵 A ∈ ℝ^(n×n)
输出: 下三角矩阵 G

对于 j = 1 到 n:
    G[j,j] = √(A[j,j] - Σ(G[j,k]², k=1到j-1))
    
    对于 i = j+1 到 n:
        G[i,j] = (A[i,j] - Σ(G[i,k]·G[j,k], k=1到j-1)) / G[j,j]

返回 G
```

**复杂度**：$O(\frac{n^3}{6})$，约为LU分解的一半。

#### Python实现

```python
def cholesky(A):
    """Cholesky分解"""
    n = len(A)
    G = np.zeros((n, n))
    
    for j in range(n):
        # 对角元素
        temp = A[j, j] - np.dot(G[j, :j], G[j, :j])
        if temp <= 0:
            raise ValueError("矩阵不是正定的")
        G[j, j] = np.sqrt(temp)
        
        # 下三角部分
        for i in range(j + 1, n):
            G[i, j] = (A[i, j] - np.dot(G[i, :j], G[j, :j])) / G[j, j]
    
    return G

# 验证
A = np.array([[4, 2, -2], [2, 10, 2], [-2, 2, 5]], dtype=float)
G = cholesky(A)
print("G =\n", G)
print("GG^T =\n", G @ G.T)
```

### 2.3 QR分解

#### 直观理解

QR分解将矩阵$A$分解为正交矩阵$Q$和上三角矩阵$R$：
$$A = QR$$

应用：求解最小二乘问题、特征值计算、正交化。

#### 形式定义

**定理 2.2**（QR分解）：任何实矩阵$A \in \mathbb{R}^{m \times n}$（$m \geq n$）可分解为$A = QR$，其中$Q \in \mathbb{R}^{m \times m}$正交，$R \in \mathbb{R}^{m \times n}$上三角。

#### Gram-Schmidt正交化

```
算法: 改进Gram-Schmidt
输入: 矩阵 A ∈ ℝ^(m×n)
输出: 正交矩阵 Q，上三角矩阵 R

对于 k = 1 到 n:
    v_k = a_k
    对于 j = 1 到 k-1:
        R[j,k] = q_j^T · a_k
        v_k = v_k - R[j,k] · q_j
    R[k,k] = ||v_k||
    q_k = v_k / R[k,k]

返回 Q = [q_1, ..., q_n], R
```

#### Householder变换（数值稳定）

```python
def householder_qr(A):
    """Householder QR分解"""
    m, n = A.shape
    Q = np.eye(m)
    R = A.copy()
    
    for k in range(n):
        # 提取列向量
        x = R[k:, k]
        
        # 构造Householder向量
        e1 = np.zeros_like(x)
        e1[0] = np.sign(x[0]) * np.linalg.norm(x)
        v = x + e1
        v = v / np.linalg.norm(v)
        
        # 应用变换
        R[k:, k:] -= 2.0 * np.outer(v, v @ R[k:, k:])
        Q[:, k:] -= 2.0 * np.outer(Q[:, k:] @ v, v)
    
    return Q, R
```

### 2.4 奇异值分解(SVD)

#### 形式定义

**定理 2.3**（SVD）：任何矩阵$A \in \mathbb{R}^{m \times n}$可分解为：
$$A = U\Sigma V^T$$
其中$U \in \mathbb{R}^{m \times m}$，$V \in \mathbb{R}^{n \times n}$正交，$\Sigma$对角，对角元$\sigma_1 \geq \sigma_2 \geq \cdots \geq 0$为奇异值。

#### 应用

1. **伪逆**：$A^+ = V\Sigma^+ U^T$
2. **低秩逼近**：$A_k = \sum_{i=1}^k \sigma_i u_i v_i^T$
3. **主成分分析(PCA)**
4. **图像压缩**

```python
def svd_compression(A, k):
    """SVD低秩逼近压缩"""
    U, s, Vt = np.linalg.svd(A, full_matrices=False)
    # 取前k个奇异值
    Ak = U[:, :k] @ np.diag(s[:k]) @ Vt[:k, :]
    compression_ratio = (A.size) / (k * (U.shape[0] + 1 + Vt.shape[1]))
    return Ak, compression_ratio, s[:k]
```

## 3. 迭代法

### 3.1 Jacobi迭代

#### 算法思想

将$A = D - L - U$（对角、严格下三角、严格上三角），则：
$$Dx^{(k+1)} = (L + U)x^{(k)} + b$$

**分量形式**：
$$x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j \neq i} a_{ij} x_j^{(k)} \right)$$

#### 收敛性分析

**定理 3.1**：Jacobi迭代收敛当且仅当$\rho(D^{-1}(L+U)) < 1$，其中$\rho$为谱半径。

**充分条件**：若$A$严格对角占优或不可约对角占优，则Jacobi迭代收敛。

```python
def jacobi(A, b, x0, tol=1e-6, max_iter=1000):
    """Jacobi迭代法"""
    n = len(b)
    x = x0.copy()
    x_new = np.zeros_like(x)
    
    for k in range(max_iter):
        for i in range(n):
            x_new[i] = (b[i] - np.dot(A[i, :i], x[:i]) 
                       - np.dot(A[i, i+1:], x[i+1:])) / A[i, i]
        
        if np.linalg.norm(x_new - x) < tol:
            return x_new, k + 1
        
        x = x_new.copy()
    
    return x, max_iter
```

### 3.2 Gauss-Seidel迭代

#### 算法思想

使用最新计算的值：
$$(D - L)x^{(k+1)} = Ux^{(k)} + b$$

**分量形式**：
$$x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j < i} a_{ij} x_j^{(k+1)} - \sum_{j > i} a_{ij} x_j^{(k)} \right)$$

```python
def gauss_seidel(A, b, x0, tol=1e-6, max_iter=1000):
    """Gauss-Seidel迭代法"""
    n = len(b)
    x = x0.copy()
    
    for k in range(max_iter):
        x_old = x.copy()
        for i in range(n):
            x[i] = (b[i] - np.dot(A[i, :i], x[:i]) 
                   - np.dot(A[i, i+1:], x_old[i+1:])) / A[i, i]
        
        if np.linalg.norm(x - x_old) < tol:
            return x, k + 1
    
    return x, max_iter
```

### 3.3 共轭梯度法(CG)

#### 算法思想

用于求解**对称正定**线性系统，是最优的Krylov子空间方法。

**核心思想**：在Krylov子空间$\mathcal{K}_k(A, r_0)$中寻找使$A$-范数误差最小的解。

#### 算法描述

```
算法: 共轭梯度法 (CG)
输入: 对称正定矩阵 A, 向量 b, 初始猜测 x₀
输出: 近似解 x

r₀ = b - Ax₀
p₀ = r₀

对于 k = 0, 1, 2, ... 直到收敛:
    αₖ = (rₖᵀrₖ) / (pₖᵀApₖ)
    xₖ₊₁ = xₖ + αₖpₖ
    rₖ₊₁ = rₖ - αₖApₖ
    βₖ = (rₖ₊₁ᵀrₖ₊₁) / (rₖᵀrₖ)
    pₖ₊₁ = rₖ₊₁ + βₖpₖ
```

#### 收敛性分析

**定理 3.2**（CG收敛）：对于$A$的条件数$\kappa$，CG的误差满足：
$$\|x_k - x^*\|_A \leq 2 \left( \frac{\sqrt{\kappa} - 1}{\sqrt{\kappa} + 1} \right)^k \|x_0 - x^*\|_A$$

```python
def conjugate_gradient(A, b, x0=None, tol=1e-6, max_iter=None):
    """共轭梯度法求解 Ax = b"""
    n = len(b)
    if x0 is None:
        x0 = np.zeros(n)
    if max_iter is None:
        max_iter = n
    
    x = x0.copy()
    r = b - A @ x
    p = r.copy()
    rs_old = np.dot(r, r)
    
    for k in range(max_iter):
        Ap = A @ p
        alpha = rs_old / np.dot(p, Ap)
        x += alpha * p
        r -= alpha * Ap
        rs_new = np.dot(r, r)
        
        if np.sqrt(rs_new) < tol:
            return x, k + 1
        
        p = r + (rs_new / rs_old) * p
        rs_old = rs_new
    
    return x, max_iter

# 验证：使用2D泊松矩阵
from scipy.sparse import diags
def poisson_matrix(n):
    """生成2D泊松问题的稀疏矩阵"""
    main_diag = 4 * np.ones(n * n)
    off_diag = -np.ones(n * n - 1)
    off_diag[n-1::n] = 0  # 边界条件
    far_diag = -np.ones(n * n - n)
    
    A = (diags([far_diag, off_diag, main_diag, off_diag, far_diag], 
               [-n, -1, 0, 1, n]).toarray())
    return A

n = 20
A = poisson_matrix(n)
b = np.ones(n * n)
x, iters = conjugate_gradient(A, b, tol=1e-8)
print(f"迭代次数: {iters}")
print(f"残差: {np.linalg.norm(A @ x - b):.2e}")
```

### 3.4 GMRES

#### 算法思想

GMRES（广义最小残差法）用于**非对称**线性系统，在Krylov子空间中最小化残差范数。

```python
def gmres(A, b, x0=None, m=20, tol=1e-6, max_restart=10):
    """GMRES(m)算法"""
    n = len(b)
    if x0 is None:
        x0 = np.zeros(n)
    
    x = x0.copy()
    
    for restart in range(max_restart):
        # Arnoldi过程
        r = b - A @ x
        beta = np.linalg.norm(r)
        if beta < tol:
            return x, restart * m
        
        V = np.zeros((n, m + 1))
        H = np.zeros((m + 1, m))
        V[:, 0] = r / beta
        
        for j in range(m):
            w = A @ V[:, j]
            for i in range(j + 1):
                H[i, j] = np.dot(w, V[:, i])
                w -= H[i, j] * V[:, i]
            H[j + 1, j] = np.linalg.norm(w)
            if H[j + 1, j] < 1e-12:
                m = j + 1
                break
            V[:, j + 1] = w / H[j + 1, j]
        
        # 最小二乘问题
        e1 = np.zeros(m + 1)
        e1[0] = beta
        y, _, _, _ = np.linalg.lstsq(H[:m + 1, :m], e1, rcond=None)
        x += V[:, :m] @ y
        
        if np.linalg.norm(b - A @ x) < tol:
            return x, restart * m + m
    
    return x, max_restart * m
```

## 4. 特征值问题

### 4.1 幂迭代法

#### 算法思想

通过重复乘以矩阵来逼近**最大模特征值**对应的特征向量：
$$v^{(k+1)} = \frac{Av^{(k)}}{\|Av^{(k)}\|}$$

特征值估计：$\lambda^{(k)} = (v^{(k)})^T A v^{(k)}$

#### 收敛性分析

**定理 4.1**：设$|\lambda_1| > |\lambda_2| \geq \cdots \geq |\lambda_n|$，则
$$\|v^{(k)} - (\pm v_1)\| = O\left(\left|\frac{\lambda_2}{\lambda_1}\right|^k\right)$$

```python
def power_iteration(A, v0=None, tol=1e-6, max_iter=1000):
    """幂迭代法求主特征值"""
    n = A.shape[0]
    if v0 is None:
        v = np.random.randn(n)
    else:
        v = v0.copy()
    
    v = v / np.linalg.norm(v)
    
    for k in range(max_iter):
        Av = A @ v
        v_new = Av / np.linalg.norm(Av)
        
        # Rayleigh商
        lambda_est = v_new @ A @ v_new
        
        if np.linalg.norm(v_new - v) < tol:
            return lambda_est, v_new, k + 1
        
        v = v_new
    
    return lambda_est, v, max_iter
```

### 4.2 QR算法

#### 算法思想

通过迭代QR分解逼近Schur形式：
$$A_k = Q_k R_k, \quad A_{k+1} = R_k Q_k$$

$A_k$收敛到上三角矩阵（或拟上三角），对角元为特征值。

```python
def qr_algorithm(A, tol=1e-6, max_iter=1000):
    """基本QR算法（带位移）"""
    n = A.shape[0]
    T = A.copy()
    
    for k in range(max_iter):
        # Wilkinson位移
        if n > 1:
            d = (T[-2, -2] - T[-1, -1]) / 2
            mu = T[-1, -1] - np.sign(d) * T[-1, -2]**2 / (
                abs(d) + np.sqrt(d**2 + T[-1, -2]**2))
        else:
            mu = 0
        
        Q, R = np.linalg.qr(T - mu * np.eye(n))
        T = R @ Q + mu * np.eye(n)
        
        # 检查次对角线收敛
        off_diag_norm = np.sqrt(np.sum(T**2) - np.sum(np.diag(T)**2))
        if off_diag_norm < tol:
            break
    
    eigenvalues = np.diag(T)
    return eigenvalues, T, k + 1
```

### 4.3 Lanczos算法

#### 算法思想

将对称矩阵三对角化，用于大规模稀疏特征值问题。

```python
def lanczos(A, q1, m):
    """Lanczos算法生成三对角矩阵"""
    n = len(q1)
    Q = np.zeros((n, m))
    alpha = np.zeros(m)
    beta = np.zeros(m)
    
    Q[:, 0] = q1 / np.linalg.norm(q1)
    
    for j in range(m):
        v = A @ Q[:, j]
        alpha[j] = np.dot(v, Q[:, j])
        
        v = v - alpha[j] * Q[:, j]
        if j > 0:
            v = v - beta[j-1] * Q[:, j-1]
        
        if j < m - 1:
            beta[j] = np.linalg.norm(v)
            if beta[j] < 1e-12:
                break
            Q[:, j + 1] = v / beta[j]
    
    # 构造三对角矩阵
    T = np.diag(alpha) + np.diag(beta[:-1], 1) + np.diag(beta[:-1], -1)
    return T, Q
```

## 5. 应用案例

### 5.1 图像去模糊

图像模糊可建模为线性系统：$b = Ax + \eta$，其中$A$为模糊核矩阵。

```python
def deblur_tikhonov(A, b, lambda_reg=0.01):
    """Tikhonov正则化去模糊"""
    # 求解 (A^T A + λI)x = A^T b
    n = A.shape[1]
    x, _ = conjugate_gradient(A.T @ A + lambda_reg * np.eye(n), A.T @ b)
    return x
```

### 5.2 PageRank算法

Web图的特征值问题：求解$Gp = p$，其中$G$为Google矩阵。

```python
def pagerank(adj_matrix, alpha=0.85, tol=1e-8):
    """PageRank算法"""
    n = adj_matrix.shape[0]
    
    # 构造转移矩阵
    out_degree = np.sum(adj_matrix, axis=1)
    P = adj_matrix / out_degree[:, np.newaxis]
    P[out_degree == 0] = 1 / n  # 处理悬停节点
    
    # Google矩阵
    G = alpha * P + (1 - alpha) / n * np.ones((n, n))
    
    # 幂迭代
    p = np.ones(n) / n
    for _ in range(1000):
        p_new = G @ p
        if np.linalg.norm(p_new - p) < tol:
            break
        p = p_new
    
    return p
```

### 5.3 主成分分析(PCA)

```python
def pca_svd(X, k):
    """基于SVD的PCA"""
    # 中心化
    X_centered = X - np.mean(X, axis=0)
    
    # SVD
    U, s, Vt = np.linalg.svd(X_centered, full_matrices=False)
    
    # 投影
    components = Vt[:k].T
    projected = X_centered @ components
    
    return projected, components, s[:k]**2 / (X.shape[0] - 1)
```
