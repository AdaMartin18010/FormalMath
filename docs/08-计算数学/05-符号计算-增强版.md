## 📚 概述

符号计算（又称计算机代数）是使用计算机进行精确数学运算的技术，与数值计算不同，符号计算保持数学表达式的精确形式，避免舍入误差。符号计算系统能够进行多项式运算、微积分、方程求解、以及自动定理证明，是数学研究、科学计算和教育的核心工具。

本教学文档系统介绍符号计算的理论基础，包括多项式算法、符号积分、Groebner基理论以及自动定理证明，着重阐述算法的数学原理和实际应用。

**核心问题**：如何用算法精确表示和操作数学表达式，实现符号微分、积分、化简和求解？

## 📐 核心理论

### 1. 多项式GCD与因子分解

#### 欧几里得算法

**定理（多项式欧几里得算法）**：
设 $F$ 是域，$a(x), b(x) \in F[x]$ 不全为零。存在唯一的首一多项式 $d(x) = \gcd(a, b)$，满足：

1. $d \mid a$ 且 $d \mid b$
2. 若 $c \mid a$ 且 $c \mid b$，则 $c \mid d$

**欧几里得算法**：

```

输入: a, b ∈ F[x]
输出: gcd(a, b)

while b ≠ 0:
    r ← a mod b    // 多项式除法
    a ← b
    b ← r
return a (首一化)

```

**复杂度分析**：

- 设 $\deg(a) = m$，$\deg(b) = n$，$m \geq n$
- 朴素实现：$O(n^3)$ 域运算
- 快速算法（FFT加速）：$O(n \log^2 n)$

**扩展欧几里得算法**：
同时计算 $s, t$ 使得 $sa + tb = \gcd(a, b)$（Bezout等式）。

#### 模方法因子分解

**Berlekamp算法**（有限域 $\mathbb{F}_p$）：

1. **构造Berlekamp矩阵** $Q$，其中 $Q_{ij}$ 表示 $x^{ip} \mod f(x)$ 中 $x^j$ 的系数
2. **计算 $Q - I$ 的零空间**，基向量对应因子
3. **GCD计算**：$\gcd(f, h - s)$ 给出非平凡因子

**复杂度**：$O(n^3 + pn^2)$，其中 $n = \deg(f)$。

**Cantor-Zassenhaus算法**（大特征域）：

- 利用随机多项式分解
- 复杂度：$O(n^2 \log n \log p)$

**整数多项式因子分解**（Zassenhaus算法）：

1. **模 $p$ 因子分解**：在 $\mathbb{F}_p[x]$ 中分解
2. **Hensel提升**：从 $\mod p$ 提升到 $\mod p^k$
3. **组合试验**：找到正确的因子组合

### 2. 符号积分：Risch算法

#### Liouville定理

**定理（Liouville, 1835）**：
设 $f$ 是初等函数，$K$ 是 $f$ 的基域。若 $f$ 有初等积分，则：
$$\int f = v_0 + \sum_{i=1}^n c_i \log(v_i)$$
其中 $v_0, v_1, \ldots, v_n$ 是 $K$ 上的代数函数，$c_i$ 是常数。

**推论**：若 $f = g'/g$（对数导数形式），则积分是对数函数。

#### Risch算法框架

1968年，Risch提出了判定初等函数积分的完整算法。

**Risch结构定理**：
初等函数域 $K = \mathbb{Q}(x, \theta_1, \ldots, \theta_n)$，其中每个 $\theta_i$ 是超越或对数函数。

**算法步骤**：

1. **Hermite约化**（有理函数部分）：
   对于有理函数 $f = p/q$，分解为：
   $$\int f = \frac{s}{q_2} + \int \frac{t}{q_1}$$
   其中 $q = q_1 q_2^2$，$\gcd(q_1, q_2) = 1$，被积函数分母无平方因子。

2. **Rothstein-Trager算法**（对数部分）：
   计算 $R(z) = \text{res}_x(p - zq', q)$ 的根，得到对数项。

3. **超越情形**：
   对于指数函数 $e^{g(x)}$，构造微分方程求解。

**判定条件**：
Risch算法能够：

- 计算初等函数的积分（如果存在）
- 证明某些函数无初等积分（如 $e^{x^2}$、$\frac{1}{\ln x}$）

### 3. Groebner基与Buchberger算法

#### 单项式序

**定义**：单项式序 $<$ 是 $\mathbb{N}^n$ 上的全序，满足：

1. 良序性：任意非空子集有最小元
2. 相容性：$\alpha < \beta \Rightarrow \alpha + \gamma < \beta + \gamma$

**常用单项式序**：

1. **字典序(Lex)**：
   $x^\alpha <_{lex} x^\beta$ 当且仅当 $\alpha - \beta$ 的第一个非零分量 $< 0$。

   例：$x^2 <_{lex} xy <_{lex} y^2$

2. **全次数字典序(DegLex)**：
   先比较总次数，次数相同则用字典序。

   例：$x <_{deglex} y <_{deglex} x^2 <_{deglex} xy <_{deglex} y^2$

3. **全次数逆字典序(DegRevLex)**：
   先比较总次数，次数相同则从右边开始比较（取反）。

#### Buchberger算法

**定义（Groebner基）**：
设 $I$ 是多项式理想，$G = \{g_1, \ldots, g_t\} \subseteq I$。$G$ 是 $I$ 的Groebner基，如果：
$$\langle \text{LT}(g_1), \ldots, \text{LT}(g_t) \rangle = \langle \text{LT}(I) \rangle$$

**S-多项式**：
对于多项式 $f, g$，设 $x^\gamma = \text{lcm}(\text{LM}(f), \text{LM}(g))$，则：
$$S(f, g) = \frac{x^\gamma}{\text{LT}(f)} \cdot f - \frac{x^\gamma}{\text{LT}(g)} \cdot g$$

**Buchberger判定准则**：
$G$ 是Groebner基当且仅当对所有 $i \neq j$，$S(g_i, g_j) \xrightarrow{G} 0$。

**Buchberger算法**：

```

输入: F = {f₁, ..., fₛ}
输出: G = Groebner基

G ← F
B ← {(fᵢ, fⱼ) : 1 ≤ i < j ≤ s}

while B ≠ ∅:
    (f, g) ← B中的某对
    B ← B \ {(f, g)}

    S ← S(f, g) 除以 G 的余式

    if S ≠ 0:
        B ← B ∪ {(h, S) : h ∈ G}
        G ← G ∪ {S}

return G

```

#### 复杂度分析

**最坏情况**：双指数复杂度
对于 $n$ 个变量，$d$ 次多项式，Groebner基的复杂度为 $d^{2^{O(n)}}$。

**实际应用**：

- 对于零维理想（有限个解），FGLM算法可转换单项式序
- F4/F5算法：矩阵方法加速，可达多项式复杂度

**应用**：

1. **理想成员判定**：$f \in I$ 当且仅当 $f \xrightarrow{G} 0$
2. **方程求解**：计算消元理想，依次求解
3. **代数几何**：计算维数、次数、Hilbert函数

### 4. 自动定理证明

#### 几何定理证明

**吴方法（1978年）**：

1. 将几何条件代数化：点坐标、几何约束 $\rightarrow$ 多项式方程
2. **三角化**：将多项式组化为伪三角形式
3. **伪余式计算**：验证结论多项式在条件下为零

**例**：证明三角形的垂心存在

- 顶点：$A(0, 0), B(1, 0), C(u, v)$
- 高线方程：$h_1, h_2, h_3$
- 结论：三垂线共点

#### 代数方法

**Groebner基方法**：

1. 将几何假设编码为多项式理想 $I = \langle h_1, \ldots, h_m \rangle$
2. 结论为 $c = 0$
3. 验证 $c \in \sqrt{I}$（根理想成员）

**量词消去**：
对于实闭域上的初等几何，使用**柱形代数分解(CAD)**：

- 复杂度：双指数
- 应用：自动不等式证明

## 🔍 实例表征

### 例1：多项式GCD计算

**问题**：计算 $a(x) = x^4 - 1$ 和 $b(x) = x^3 - 1$ 的GCD。

**欧几里得算法**：

| 步骤 | $a$ | $b$ | $r = a \mod b$ |
|-----|-----|-----|---------------|
| 1 | $x^4 - 1$ | $x^3 - 1$ | $x - 1$ |
| 2 | $x^3 - 1$ | $x - 1$ | $0$ |

结果：$\gcd(x^4 - 1, x^3 - 1) = x - 1$

**验证**：

- $x^4 - 1 = (x - 1)(x^3 + x^2 + x + 1)$ ✓
- $x^3 - 1 = (x - 1)(x^2 + x + 1)$ ✓

### 例2：符号积分示例

**问题**：计算 $\int \frac{x^2 + 1}{x^3 - x} dx$

**步骤1：部分分式分解**
$$\frac{x^2 + 1}{x^3 - x} = \frac{x^2 + 1}{x(x-1)(x+1)} = \frac{A}{x} + \frac{B}{x-1} + \frac{C}{x+1}$$

解得：$A = -1, B = 1, C = 1$

**步骤2：逐项积分**
$$\int \left(-\frac{1}{x} + \frac{1}{x-1} + \frac{1}{x+1}\right) dx = -\ln|x| + \ln|x-1| + \ln|x+1| + C$$

**结果**：
$$\ln\left|\frac{x^2 - 1}{x}\right| + C$$

**不可积示例**：$\int e^{x^2} dx$

- 由Liouville定理可证：$e^{x^2}$ 无初等积分
- 只能用数值方法或级数展开

### 例3：Groebner基计算

**问题**：求理想 $I = \langle f_1, f_2 \rangle$ 的Groebner基，其中
$$f_1 = x^2 + y^2 - 4, \quad f_2 = xy - 1$$

**Buchberger算法**（字典序 $x > y$）：

**步骤1**：计算S-多项式
$$S(f_1, f_2) = \frac{x^2y}{x^2} f_1 - \frac{x^2y}{xy} f_2 = y(x^2 + y^2 - 4) - x(xy - 1) = y^3 - 4y + x$$

约化：$S(f_1, f_2) \xrightarrow{f_2} y^3 - 4y + x - (xy - 1)/y$ ...

**简化计算**：
$$f_3 = y^3 - 4y + x - xy^2 + y = y^3 - 3y + x(1 - y^2)$$

消去 $x$：由 $f_2$，$x = 1/y$，代入 $f_1$：
$$\frac{1}{y^2} + y^2 - 4 = 0 \Rightarrow 1 + y^4 - 4y^2 = 0$$

**Groebner基**：$\{y^4 - 4y^2 + 1, x - y^3 + 3y\}$

**求解**：

- $y^2 = 2 \pm \sqrt{3}$
- $y = \pm\sqrt{2 \pm \sqrt{3}}$
- $x = y^3 - 3y = \frac{1}{y}$

### 例4：几何定理自动证明

**定理**：等腰三角形的两底角相等。

**坐标设定**：

- $A(0, 0), B(c, 0), C(c/2, h)$
- 等腰条件：$|AC| = |BC|$（自动满足）

**多项式编码**：

- 斜率 $k_{AC} = \frac{h}{c/2} = \frac{2h}{c}$
- 斜率 $k_{BC} = \frac{h}{-c/2} = -\frac{2h}{c}$

**角度相等条件**：
$$\tan(\angle CAB) = \tan(\angle CBA)$$
即：$\frac{k_{AC} - k_{AB}}{1 + k_{AC}k_{AB}} = \frac{k_{BA} - k_{BC}}{1 + k_{BA}k_{BC}}$

**验证**：代入 $k_{AB} = 0$，验证恒等式成立。

## 🔧 技术实现表征

### Python/SymPy实现

```python
from sympy import *

# 定义符号
x, y, z = symbols('x y z')

# 多项式GCD计算
def polynomial_gcd_example():
    a = x**4 - 1
    b = x**3 - 1
    g = gcd(a, b)
    print(f"gcd({a}, {b}) = {g}")

    # 验证
    assert (a / g).expand().is_polynomial()
    assert (b / g).expand().is_polynomial()

# 符号积分
def symbolic_integration_example():
    # 有理函数积分
    f = (x**2 + 1) / (x**3 - x)
    result = integrate(f, x)
    print(f"∫{f}dx = {result} + C")

    # 验证（微分应得原函数）
    assert simplify(diff(result, x) - f) == 0

# Groebner基
def groebner_basis_example():
    f1 = x**2 + y**2 - 4
    f2 = x*y - 1
    G = groebner([f1, f2], x, y, order='lex')
    print(f"Groebner基: {G}")

    # 求解
    solutions = solve([f1, f2], [x, y])
    print(f"解: {solutions}")

if __name__ == "__main__":
    polynomial_gcd_example()
    symbolic_integration_example()
    groebner_basis_example()

```
