## 问题1：LU分解与线性系统求解 ⭐⭐☆☆☆

### 📝 问题描述

求解结构力学中的刚度方程组：

$$\mathbf{K}\mathbf{u} = \mathbf{f}$$

其中刚度矩阵 $\mathbf{K}$ 和载荷向量 $\mathbf{f}$：

$$\mathbf{K} = \begin{pmatrix} 
8 & -4 & 0 & 0 \\
-4 & 8 & -4 & 0 \\
0 & -4 & 8 & -4 \\
0 & 0 & -4 & 4
\end{pmatrix}, \quad 
\mathbf{f} = \begin{pmatrix} 0 \\ 100 \\ 0 \\ 0 \end{pmatrix}$$

**任务**:
1. 进行Doolittle LU分解（不带行交换）
2. 前代求解 $\mathbf{L}\mathbf{y} = \mathbf{f}$
3. 回代求解 $\mathbf{U}\mathbf{u} = \mathbf{y}$
4. 验证残差并分析误差

---

### ✅ 解答

#### 步骤1: LU分解

**分解原理**: $\mathbf{A} = \mathbf{L}\mathbf{U}$

对三对角矩阵，使用Crout变体：

$k=1$:
- $u_{11} = a_{11} = 8$
- $l_{21} = a_{21}/u_{11} = -4/8 = -0.5$

$k=2$:
- $u_{22} = a_{22} - l_{21}u_{12} = 8 - (-0.5)(-4) = 6$
- $l_{32} = a_{32}/u_{22} = -4/6 = -2/3$

$k=3$:
- $u_{33} = a_{33} - l_{32}u_{23} = 8 - (-2/3)(-4) = 16/3$
- $l_{43} = a_{43}/u_{33} = -4/(16/3) = -3/4$

$k=4$:
- $u_{44} = a_{44} - l_{43}u_{34} = 4 - (-3/4)(-4) = 1$

结果：

$$\mathbf{L} = \begin{pmatrix} 
1 & 0 & 0 & 0 \\
-0.5 & 1 & 0 & 0 \\
0 & -2/3 & 1 & 0 \\
0 & 0 & -3/4 & 1
\end{pmatrix}, \quad
\mathbf{U} = \begin{pmatrix} 
8 & -4 & 0 & 0 \\
0 & 6 & -4 & 0 \\
0 & 0 & 16/3 & -4 \\
0 & 0 & 0 & 1
\end{pmatrix}$$

#### 步骤2: 前代求解 $\mathbf{L}\mathbf{y} = \mathbf{f}$

$$\begin{cases}
y_1 = 0 \\
-0.5y_1 + y_2 = 100 \Rightarrow y_2 = 100 \\
-\frac{2}{3}y_2 + y_3 = 0 \Rightarrow y_3 = 200/3 \\
-\frac{3}{4}y_3 + y_4 = 0 \Rightarrow y_4 = 50
\end{cases}$$

$$\mathbf{y} = (0, 100, 200/3, 50)^T$$

#### 步骤3: 回代求解 $\mathbf{U}\mathbf{u} = \mathbf{y}$

$$\begin{cases}
u_4 = 50 \\
\frac{16}{3}u_3 - 4(50) = \frac{200}{3} \Rightarrow u_3 = 50 \\
6u_2 - 4(50) = 100 \Rightarrow u_2 = 50 \\
8u_1 - 4(50) = 0 \Rightarrow u_1 = 25
\end{cases}$$

$$\mathbf{u} = (25, 50, 50, 50)^T$$

#### 步骤4: 误差验证

**残差计算**:
$$\mathbf{r} = \mathbf{f} - \mathbf{K}\mathbf{u}$$

$$\mathbf{K}\mathbf{u} = (0, 100, 0, 0)^T$$
$$\mathbf{r} = (0, 0, 0, 0)^T$$

残差为零（精确解）！

---

### 💻 Python实现

```python
import numpy as np
from scipy.linalg import lu, solve_triangular

# 定义刚度矩阵和载荷向量
K = np.array([
    [8, -4, 0, 0],
    [-4, 8, -4, 0],
    [0, -4, 8, -4],
    [0, 0, -4, 4]
], dtype=float)

f = np.array([0, 100, 0, 0], dtype=float)

print("刚度矩阵 K:")
print(K)
print(f"\n条件数: {np.linalg.cond(K):.4f}")

# 方法1: 手动LU分解（Doolittle）
def lu_decomposition_manual(A):
    """Doolittle LU分解"""
    n = len(A)
    L = np.eye(n)
    U = np.zeros((n, n))
    
    for k in range(n):
        # U的第k行
        for j in range(k, n):
            U[k, j] = A[k, j] - sum(L[k, m] * U[m, j] for m in range(k))
        
        # L的第k列
        for i in range(k+1, n):
            L[i, k] = (A[i, k] - sum(L[i, m] * U[m, k] for m in range(k))) / U[k, k]
    
    return L, U

L_manual, U_manual = lu_decomposition_manual(K)
print("\n手动LU分解:")
print("L =", L_manual)
print("U =", U_manual)

# 验证 A = LU
print(f"\n验证 ||K - LU|| = {np.linalg.norm(K - L_manual @ U_manual):.2e}")

# 前代和回代求解
def solve_lu(L, U, b):
    """用LU分解求解线性系统"""
    n = len(b)
    
    # 前代: Ly = b
    y = np.zeros(n)
    for i in range(n):
        y[i] = b[i] - sum(L[i, j] * y[j] for j in range(i))
    
    # 回代: Ux = y
    x = np.zeros(n)
    for i in range(n-1, -1, -1):
        x[i] = (y[i] - sum(U[i, j] * x[j] for j in range(i+1, n))) / U[i, i]
    
    return x

u_manual = solve_lu(L_manual, U_manual, f)
print(f"\n手动求解结果: {u_manual}")

# 方法2: 使用SciPy（带部分主元）
P, L_scipy, U_scipy = lu(K)
u_scipy = solve_triangular(U_scipy, solve_triangular(L_scipy, P.T @ f, lower=True))
print(f"SciPy求解结果: {u_scipy}")

# 方法3: NumPy直接求解
u_numpy = np.linalg.solve(K, f)
print(f"NumPy求解结果: {u_numpy}")

# 误差分析
print("\n--- 误差分析 ---")
for method, u in [("手动", u_manual), ("SciPy", u_scipy), ("NumPy", u_numpy)]:
    residual = np.linalg.norm(f - K @ u)
    relative_error = residual / np.linalg.norm(f)
    print(f"{method}: 残差={residual:.2e}, 相对误差={relative_error:.2e}")
```

## 问题2：迭代法收敛性分析 ⭐⭐⭐☆☆

### 📝 问题描述

求解二维Poisson方程离散化后的大型稀疏系统：

$$\mathbf{A}\mathbf{x} = \mathbf{b}$$

其中 $\mathbf{A}$ 是 $n^2 \times n^2$ 矩阵（对应 $n \times n$ 网格）：

$$\mathbf{A} = \begin{pmatrix} 
\mathbf{T} & -\mathbf{I} & & \\
-\mathbf{I} & \mathbf{T} & -\mathbf{I} & \\
& \ddots & \ddots & \ddots \\
& & -\mathbf{I} & \mathbf{T}
\end{pmatrix}, \quad
\mathbf{T} = \begin{pmatrix} 
4 & -1 & & \\
-1 & 4 & -1 & \\
& \ddots & \ddots & \ddots \\
& & -1 & 4
\end{pmatrix}$$

**任务**:
1. 对 $n=20$（400维系统），比较Jacobi和Gauss-Seidel方法
2. 分析收敛速率与谱半径的关系
3. 实现SOR方法并寻找最优松弛因子
4. 与直接法比较计算效率

---

### ✅ 解答

#### 步骤1: 迭代格式

**Jacobi迭代**:
$$x_i^{(k+1)} = \frac{1}{a_{ii}}\left(b_i - \sum_{j \neq i} a_{ij}x_j^{(k)}\right)$$

**Gauss-Seidel迭代**:
$$x_i^{(k+1)} = \frac{1}{a_{ii}}\left(b_i - \sum_{j < i} a_{ij}x_j^{(k+1)} - \sum_{j > i} a_{ij}x_j^{(k)}\right)$$

**SOR迭代**:
$$x_i^{(k+1)} = (1-\omega)x_i^{(k)} + \frac{\omega}{a_{ii}}\left(b_i - \sum_{j < i} a_{ij}x_j^{(k+1)} - \sum_{j > i} a_{ij}x_j^{(k)}\right)$$

#### 步骤2: 收敛性分析

对于模型Poisson问题：
- Jacobi谱半径: $\rho_J = \cos(\pi/n) \approx 1 - \frac{\pi^2}{2n^2}$
- Gauss-Seidel谱半径: $\rho_{GS} = \rho_J^2 \approx 1 - \frac{\pi^2}{n^2}$

当 $n=20$：
- $\rho_J \approx 0.9877$
- $\rho_{GS} \approx 0.9755$

**渐进收敛速率**:
- Jacobi: $R_J = -\ln(\rho_J) \approx 0.0124$
- GS: $R_{GS} = -\ln(\rho_{GS}) \approx 0.0248$（快2倍）

#### 步骤3: 最优SOR参数

$$\omega_{opt} = \frac{2}{1 + \sqrt{1 - \rho_J^2}}$$

对于 $n=20$：
$$\omega_{opt} = \frac{2}{1 + \sqrt{1 - 0.9755}} \approx 1.56$$

### 💻 Python实现与比较

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.sparse import diags, kron, eye
from scipy.sparse.linalg import spsolve
import time

def build_poisson_matrix(n):
    """构建二维Poisson矩阵（稀疏格式）"""
    # 一维三对角矩阵
    T = diags([-1, 4, -1], [-1, 0, 1], shape=(n, n))
    I = eye(n)
    
    # 二维：A = I ⊗ T + T ⊗ I - 对角修正
    A = kron(I, T) + kron(T, I)
    return A

def jacobi(A, b, x0, max_iter=1000, tol=1e-10):
    """Jacobi迭代"""
    n = len(b)
    x = x0.copy()
    errors = []
    
    D = A.diagonal()
    R = A - np.diag(D)
    
    for k in range(max_iter):
        x_new = (b - R @ x) / D
        err = np.linalg.norm(x_new - x)
        errors.append(err)
        
        if err < tol:
            break
        x = x_new
    
    return x, k+1, errors

def gauss_seidel(A, b, x0, max_iter=1000, tol=1e-10):
    """Gauss-Seidel迭代"""
    n = len(b)
    x = x0.copy()
    errors = []
    
    for k in range(max_iter):
        x_old = x.copy()
        for i in range(n):
            sigma = sum(A[i, j] * x[j] for j in range(n) if j != i)
            x[i] = (b[i] - sigma) / A[i, i]
        
        err = np.linalg.norm(x - x_old)
        errors.append(err)
        
        if err < tol:
            break
    
    return x, k+1, errors

def sor(A, b, x0, omega, max_iter=1000, tol=1e-10):
    """SOR迭代"""
    n = len(b)
    x = x0.copy()
    errors = []
    
    for k in range(max_iter):
        x_old = x.copy()
        for i in range(n):
            sigma = sum(A[i, j] * x[j] for j in range(n) if j != i)
            x[i] = (1 - omega) * x[i] + omega * (b[i] - sigma) / A[i, i]
        
        err = np.linalg.norm(x - x_old)
        errors.append(err)
        
        if err < tol:
            break
    
    return x, k+1, errors

# 测试参数
n = 20
N = n * n

print(f"网格大小: {n}×{n}, 矩阵维度: {N}×{N}")

# 构建矩阵和右端项
A = build_poisson_matrix(n).toarray()
x_exact = np.ones(N)
b = A @ x_exact
x0 = np.zeros(N)

# 求解并计时
methods = []

# Jacobi
t_start = time.time()
x_jacobi, iter_j, err_j = jacobi(A, b, x0, max_iter=2000)
t_jacobi = time.time() - t_start
methods.append(('Jacobi', x_jacobi, iter_j, t_jacobi, err_j))

# Gauss-Seidel
t_start = time.time()
x_gs, iter_gs, err_gs = gauss_seidel(A, b, x0, max_iter=2000)
t_gs = time.time() - t_start
methods.append(('Gauss-Seidel', x_gs, iter_gs, t_gs, err_gs))

# SOR (不同omega)
omega_values = [1.0, 1.2, 1.4, 1.56, 1.7, 1.9]
sor_results = []
for omega in omega_values:
    x_sor, iter_s, err_s = sor(A, b, x0, omega, max_iter=1000)
    sor_results.append((omega, iter_s, err_s))
    if omega == 1.56:
        methods.append((f'SOR (ω={omega})', x_sor, iter_s, 0, err_s))

# 直接法
t_start = time.time()
x_direct = np.linalg.solve(A, b)
t_direct = time.time() - t_start

# 结果比较
print("\n--- 结果比较 ---")
print(f"{'方法':<20} {'迭代次数':<10} {'时间(s)':<12} {'相对误差':<15}")
print("-" * 60)

for name, x_sol, iters, t, _ in methods:
    rel_err = np.linalg.norm(x_sol - x_exact) / np.linalg.norm(x_exact)
    print(f"{name:<20} {iters:<10} {t:<12.4f} {rel_err:<15.2e}")

print(f"{'直接法 (LU)':<20} {'N/A':<10} {t_direct:<12.4f} "
      f"{np.linalg.norm(x_direct - x_exact)/np.linalg.norm(x_exact):<15.2e}")

# 可视化收敛历史
plt.figure(figsize=(12, 5))

plt.subplot(1, 2, 1)
for name, _, _, _, errors in methods:
    plt.semilogy(errors, label=name, linewidth=2)
plt.xlabel('迭代次数')
plt.ylabel('误差（对数尺度）')
plt.title('迭代法收敛比较')
plt.legend()
plt.grid(True)

plt.subplot(1, 2, 2)
for omega, _, errors in sor_results:
    plt.semilogy(errors, label=f'ω={omega}', linewidth=1.5)
plt.xlabel('迭代次数')
plt.ylabel('误差（对数尺度）')
plt.title('SOR不同松弛因子比较')
plt.legend()
plt.grid(True)

plt.tight_layout()
plt.savefig('iterative_methods.png', dpi=150)
plt.show()

# 最优omega分析
print("\n--- SOR松弛因子分析 ---")
print(f"{'ω':<8} {'迭代次数':<10}")
print("-" * 20)
for omega, iters, _ in sor_results:
    print(f"{omega:<8.2f} {iters:<10}")
```

## 问题5：有限差分稳定性分析 ⭐⭐⭐★☆

### 📝 问题描述

一维热传导方程：
$$u_t = \alpha u_{xx}, \quad x \in [0, 1], \quad t > 0$$

**初始条件**: $u(x, 0) = \sin(\pi x)$

**边界条件**: $u(0, t) = u(1, t) = 0$

**精确解**: $u(x, t) = e^{-\alpha \pi^2 t}\sin(\pi x)$

**数值方法**:
1. 显式格式（前向欧拉）
2. 隐式格式（后向欧拉）
3. Crank-Nicolson格式

**任务**:
1. 用von Neumann分析推导各格式的稳定性条件
2. 数值验证稳定性理论
3. 比较各格式的精度和计算成本

---

### ✅ 解答

#### 步骤1: 显式格式稳定性分析

离散格式：
$$\frac{u_j^{n+1} - u_j^n}{\Delta t} = \alpha \frac{u_{j+1}^n - 2u_j^n + u_{j-1}^n}{\Delta x^2}$$

令 $r = \alpha \Delta t / \Delta x^2$，得：
$$u_j^{n+1} = u_j^n + r(u_{j+1}^n - 2u_j^n + u_{j-1}^n)$$

**von Neumann分析**: 设 $u_j^n = G^n e^{ikj\Delta x}$

$$G = 1 + r(e^{ik\Delta x} - 2 + e^{-ik\Delta x}) = 1 - 4r\sin^2(k\Delta x/2)$$

稳定性要求 $|G| \leq 1$：
$$-1 \leq 1 - 4r \leq 1$$

得：**稳定性条件** $r \leq 0.5$，即 $\Delta t \leq \Delta x^2/(2\alpha)$

#### 步骤2: 隐式格式（无条件稳定）

$$-ru_{j-1}^{n+1} + (1+2r)u_j^{n+1} - ru_{j+1}^{n+1} = u_j^n$$

放大因子：
$$G = \frac{1}{1 + 4r\sin^2(k\Delta x/2)}$$

对所有 $r > 0$，$|G| < 1$，无条件稳定。

#### 步骤3: Crank-Nicolson格式

$$\frac{u_j^{n+1} - u_j^n}{\Delta t} = \frac{\alpha}{2}\left[\frac{u_{j+1}^{n+1} - 2u_j^{n+1} + u_{j-1}^{n+1}}{\Delta x^2} + \frac{u_{j+1}^n - 2u_j^n + u_{j-1}^n}{\Delta x^2}\right]$$

放大因子：
$$G = \frac{1 - 2r\sin^2(k\Delta x/2)}{1 + 2r\sin^2(k\Delta x/2)}$$

$|G| \leq 1$ 对所有 $r$，无条件稳定，且二阶精度。

---

### 💻 Python实现与数值验证

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.sparse import diags
from scipy.sparse.linalg import spsolve

def heat_exact(x, t, alpha=1):
    """精确解"""
    return np.exp(-alpha * np.pi**2 * t) * np.sin(np.pi * x)

def explicit_heat(u0, dx, dt, alpha, nt):
    """显式格式"""
    nx = len(u0)
    r = alpha * dt / dx**2
    u = u0.copy()
    u_all = [u.copy()]
    
    for n in range(nt):
        u_new = u.copy()
        for j in range(1, nx-1):
            u_new[j] = u[j] + r * (u[j+1] - 2*u[j] + u[j-1])
        u = u_new
        u_all.append(u.copy())
    
    return np.array(u_all)

def implicit_heat(u0, dx, dt, alpha, nt):
    """隐式格式（三对角系统）"""
    nx = len(u0)
    r = alpha * dt / dx**2
    u = u0.copy()
    u_all = [u.copy()]
    
    # 构建三对角矩阵
    main = (1 + 2*r) * np.ones(nx-2)
    off = -r * np.ones(nx-3)
    A = diags([off, main, off], [-1, 0, 1])
    
    for n in range(nt):
        u[1:-1] = spsolve(A, u[1:-1])
        u_all.append(u.copy())
    
    return np.array(u_all)

def crank_nicolson_heat(u0, dx, dt, alpha, nt):
    """Crank-Nicolson格式"""
    nx = len(u0)
    r = alpha * dt / dx**2
    u = u0.copy()
    u_all = [u.copy()]
    
    # 左端矩阵
    main_L = (1 + r) * np.ones(nx-2)
    off_L = -r/2 * np.ones(nx-3)
    A = diags([off_L, main_L, off_L], [-1, 0, 1])
    
    # 右端矩阵
    main_R = (1 - r) * np.ones(nx-2)
    off_R = r/2 * np.ones(nx-3)
    B = diags([off_R, main_R, off_R], [-1, 0, 1])
    
    for n in range(nt):
        rhs = B @ u[1:-1]
        u[1:-1] = spsolve(A, rhs)
        u_all.append(u.copy())
    
    return np.array(u_all)

# 参数设置
alpha = 1
nx = 50
dx = 1 / (nx - 1)
x = np.linspace(0, 1, nx)
T_final = 0.1
nt = 500
dt = T_final / nt

print(f"参数: nx={nx}, dx={dx:.4f}, dt={dt:.6f}")
print(f"CFL数 r = {alpha*dt/dx**2:.4f}")

# 初始条件
u0 = np.sin(np.pi * x)

# 精确解
u_exact = heat_exact(x, T_final, alpha)

# 数值求解
r = alpha * dt / dx**2

if r <= 0.5:
    u_explicit = explicit_heat(u0, dx, dt, alpha, nt)[-1]
    err_explicit = np.linalg.norm(u_explicit - u_exact)
else:
    print("显式格式不稳定！")
    err_explicit = None

u_implicit = implicit_heat(u0, dx, dt, alpha, nt)[-1]
err_implicit = np.linalg.norm(u_implicit - u_exact)

u_cn = crank_nicolson_heat(u0, dx, dt, alpha, nt)[-1]
err_cn = np.linalg.norm(u_cn - u_exact)

# 结果输出
print("\n--- 数值误差比较 (L2范数) ---")
if err_explicit:
    print(f"显式格式: {err_explicit:.6f}")
print(f"隐式格式: {err_implicit:.6f}")
print(f"Crank-Nicolson: {err_cn:.6f}")

# 可视化
plt.figure(figsize=(12, 5))

plt.subplot(1, 2, 1)
plt.plot(x, u_exact, 'k-', linewidth=2, label='精确解')
if err_explicit:
    plt.plot(x, u_explicit, 'r--', alpha=0.7, label='显式')
plt.plot(x, u_implicit, 'g:', linewidth=2, label='隐式')
plt.plot(x, u_cn, 'b-.', alpha=0.7, label='Crank-Nicolson')
plt.xlabel('x')
plt.ylabel('u')
plt.title(f'热方程数值解 (t={T_final})')
plt.legend()
plt.grid(True)

# 稳定性验证：不同r值
plt.subplot(1, 2, 2)
r_values = [0.3, 0.5, 0.6, 1.0]
for r_test in r_values:
    dt_test = r_test * dx**2 / alpha
    nt_test = int(T_final / dt_test)
    if r_test <= 0.5:
        u_test = explicit_heat(u0, dx, dt_test, alpha, nt_test)[-1]
        plt.plot(x, u_test, label=f'r={r_test} (稳定)' if r_test <= 0.5 else f'r={r_test} (不稳定)', 
                linewidth=2 if r_test <= 0.5 else 1, alpha=0.7)
    else:
        # 不稳定情况，只显示少量步数
        nt_short = min(nt_test, 50)
        try:
            u_test = explicit_heat(u0, dx, dt_test, alpha, nt_short)[-1]
            plt.plot(x, u_test, '--', label=f'r={r_test} (不稳定)', alpha=0.5)
        except:
            pass

plt.plot(x, u_exact, 'k-', linewidth=2, label='精确解')
plt.xlabel('x')
plt.ylabel('u')
plt.title('显式格式稳定性验证')
plt.legend()
plt.grid(True)

plt.tight_layout()
plt.savefig('heat_equation.png', dpi=150)
plt.show()
```
