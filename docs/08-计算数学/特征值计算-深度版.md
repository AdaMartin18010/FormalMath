## 📚 概述

特征值计算是数值线性代数的核心问题之一，在量子力学、结构分析、图论、数据科学等领域有广泛应用。从幂法的简单迭代到现代隐式重启Arnoldi方法，特征值算法经历了深刻的发展。

本深度版将系统阐述特征值计算的理论基础，提供严格的收敛性证明，并探讨其与格洛腾迪克数学体系——特别是概形上的谱理论、l-adic特征值和导出范畴——的深层联系。

## 🕰️ 历史发展与范式转变

### 古典代数时代

**特征多项式方法**：通过$\det(A - \lambda I) = 0$求根。

**问题**：对于$n \geq 5$，没有一般代数解；数值上不稳定。

### QR算法的革命

**Francis (1961)** 与 **Kublanovskaya (1961)**：独立发现QR算法。

**突破**：通过迭代$A_{k+1} = R_k Q_k$（其中$A_k = Q_k R_k$），矩阵收敛到Schur形。

**隐式位移**：Wilkinson引入位移策略加速收敛。

### 现代大规模方法

**Lanczos (1950)**：对称矩阵的三对角化。

**Arnoldi (1951)**：非对称推广。

**Sorensen (1992)**：隐式重启Arnoldi方法（ARPACK）。

## 🏗️ 经典算法的完整理论

### 幂法与反幂法的收敛性

**幂法**：$v_{k+1} = Av_k / \|Av_k\|$

**收敛定理**：设$A$有唯一最大模特征值$\lambda_1$，对应特征向量$x_1$，则
$$v_k \to \pm x_1, \quad \mu_k = v_k^T A v_k \to \lambda_1$$

收敛速度$O(|\lambda_2/\lambda_1|^k)$。

**反幂法**：对$(A - \sigma I)^{-1}$应用幂法，收敛到最接近$\sigma$的特征值。

### QR算法的全局收敛性

**基本QR迭代**：
$$A_k = Q_k R_k, \quad A_{k+1} = R_k Q_k = Q_k^T A_k Q_k$$

**定理**：在一定条件下，$A_k$收敛到上三角形式（Schur形）。

**收敛机制**：
- 最后一对角元$\alpha_{nn}^{(k)}$收敛到最小特征值
- 收敛后收缩（deflation）处理

### 隐式位移与双位移策略

**单位移QR**：
$$A_k - \sigma_k I = Q_k R_k, \quad A_{k+1} = R_k Q_k + \sigma_k I$$

**Wilkinson位移**：选择$\sigma_k$为右下角$2 \times 2$子矩阵的特征值。

**双位移**（对于实矩阵）：同时处理共轭复特征值对，保持实运算。

## 🔬 对称特征值问题的特殊结构

### 三对角化与Lanczos算法

**Householder三对角化**：$O(n^3)$运算，将对称矩阵化为三对角形式。

**Lanczos算法**：
$$\beta_{j+1}q_{j+1} = Aq_j - \alpha_j q_j - \beta_j q_{j-1}$$

**Krylov子空间**：$q_1, \ldots, q_k$构成$\mathcal{K}_k(A, q_1)$的正交基。

### 分治法

**思想**：将三对角矩阵分块为
$$T = \begin{bmatrix} T_1 & 0 \\ 0 & T_2 \end{bmatrix} + \rho vv^T$$

**秩一修改**：利用特征值交错性质，通过特征方程求解。

**复杂度**：$O(n^2)$，适合并行化。

### 相对扰动理论

**Demyko-Kahan理论**：对于对称三对角矩阵，微小相对扰动导致特征值的微小相对变化。

**重要性**：保证高精度计算的可能性。

## 📐 收敛性的严格证明

### 幂法的收敛速度分析

设$A = X \Lambda X^{-1}$可对角化，$v_0 = \sum_{i=1}^n \alpha_i x_i$：

$$A^k v_0 = \lambda_1^k\left(\alpha_1 x_1 + \sum_{i=2}^n \alpha_i \left(\frac{\lambda_i}{\lambda_1}\right)^k x_i\right)$$

当$k \to \infty$，第二项指数衰减。

### QR迭代的二次收敛

对于单特征值，Wilkinson位移给出渐近二次收敛：

$$|a_{n,n-1}^{(k+1)}| = O(|a_{n,n-1}^{(k)}|^2)$$

### Lanczos方法的近似理论

**Ritz值**：Lanczos过程给出的近似特征值。

**Kaniel-Paige-Saad理论**：给出Ritz值与真实特征值的误差界，与Krylov子空间的逼近性质相关。

## 🔄 格洛腾迪克视角下的谱理论

### 概形上的特征值问题

**向量丛的特征值**：在概形上研究向量丛自同态的特征值。

**与数值线性代数的联系**：有限维近似对应于概形上的离散化。

### l-adic特征值与Frobenius

**Weil猜想**：光滑射影簇的zeta函数满足函数方程，其零极点与Frobenius作用的特征值相关。

**Deligne定理**：证明Weil猜想，特征值的绝对值满足特定约束。

**深层联系**：有限域上矩阵的谱理论与l-adic上同调的深刻联系。

### 导出范畴中的谱序列

**谱序列**：计算同调/上同调的有力工具。

**与特征值的联系**：在某些情况下，谱序列的收敛与线性算子的谱分解相关。

## ⚡ 现代大规模方法

### Arnoldi与隐式重启

**Arnoldi算法**：非对称矩阵的Krylov子空间方法。

**隐式重启**：选择$k$个"最优"Ritz对，隐式进行QR步，压缩到维度$k$。

**ARPACK**：实现隐式重启Arnoldi，MATLAB的eigs函数基础。

### Jacobi-Davidson方法

**思想**：结合Jacobi正交化和Davidson方法。

**校正方程**：求解$(I - uu^T)(A - \theta I)(I - uu^T)t = -r$近似。

**优势**：适合内部特征值和预处理器。

### 滤波与谱变换

**多项式滤波**：$p(A)$，其中$p$在不需要的特征值附近为零。

**Cayley变换**：$(A - \sigma I)^{-1}(A - \tau I)$，将特征值映射。

## 📊 数值稳定性与后处理

### 向后稳定性分析

**QR算法**：向后稳定，计算的特征值是轻微扰动矩阵的精确特征值。

**条件数**：单个特征值的条件数为$1/|y^* x|$（左、右特征向量夹角）。

### 特征向量计算

**逆迭代**：对近似特征值$\hat{\lambda}$，求解$(A - \hat{\lambda}I)x = b$。

**精度**：即使$\hat{\lambda}$不精确，通常得到准确的特征向量。

### 广义特征值问题

**问题**：$Ax = \lambda Bx$

**QZ算法**：Moler与Stewart（1973），同时对$(A, B)$进行广义Schur分解。
