# 证明论核心定理

## 一、Hauptsatz (切割消去定理)

### 1.1 定理陈述

**Hauptsatz** (Gentzen, 1934): 在相继式演算中，任何可证明的相继式都存在无切割的证明。

**形式**: 若 $\vdash \Gamma \Rightarrow \Delta$，则存在不使用(Cut)规则的证明。

### 1.2 证明概要

**关键引理**: 切割消去引理

```
若  d₁ ⊢ Γ ⇒ Δ, A    (秩 < ρ)
   d₂ ⊢ A, Σ ⇒ Π     (秩 < ρ)
---------------------------------
则存在 d ⊢ Γ, Σ ⇒ Δ, Π  (秩 < ρ)
```

**主要归纳**: 对切割公式的复杂度和证明高度进行双重归纳。

### 1.3 切割消去的复杂性

**上界定理**: 若证明高度为 $n$，则消去切割后的证明高度为 $2^n$。

**超指数爆炸**: 切割消去通常导致证明大小的超指数增长。

**可判定性推论**: 切割消去给出了谓词逻辑半可判定性的构造性证明。

## 二、Hilbert第二问题

### 2.1 问题陈述

**Hilbert第二问题**: 证明算术公理的一致性。

### 2.2 Gödel第二不完备性定理

**定理**: 若PA一致，则PA不能证明自身的一致性。

**推论**: 在PA内部不能解决Hilbert第二问题。

### 2.3 Gentzen的解答

**定理** (Gentzen): 使用直到 $\varepsilon_0$ 的超限归纳可证明PA的一致性。

**一致性证明结构**:

1. 归约: 将PA证明转换为相继式演算
2. 消除: 切割消去
3. 赋值: 为每个证明分配序数 $< \varepsilon_0$
4. 归纳: 用超限归纳证明不可导出矛盾

### 2.4 意义与限制

**意义**: 
- 证明了超限方法在元数学中的必要性
- 开创了序数分析领域

**限制**:
- 使用了比PA更强的假设（超限归纳）
- Gödel定理保证不能在PA内完成

## 三、序数分析主要结果

### 3.1 Peano算术

**定理**: $|PA| = \varepsilon_0$

**解释**: PA的证明论序数是 $\varepsilon_0$。

**推论**: PA可证明的函数恰好是$< \varepsilon_0$-递归函数。

### 3.2 子系统二阶算术

| 系统 | 证明论序数 | 关键特征 |
|------|-----------|----------|
| PRA | $\omega^\omega$ | 原始递归算术 |
| RCA₀ | $\omega^\omega$ | 可计算分析基础 |
| WKL₀ | $\omega^\omega$ | 弱König引理 |
| ACA₀ | $\varepsilon_0$ | 算术概括 |
| ATR₀ | $\Gamma_0$ | 算术传递递归 |
| Π¹₁-CA₀ | $\psi(\Omega_\omega)$ | Π¹₁-概括 |

### 3.3 Feferman-Schütte序数

**定义**: $\Gamma_0$ 是第 $\Gamma_0$ 个不动点 $\varphi_\alpha(0) = \alpha$。

**意义**: $\Gamma_0$ 是ATO理论的证明论序数，也是"可预测性"的界限。

### 3.4 递归大序数

**Bachmann-Howard序数**: 出现在Kripke-Platek集合论的分析中。

**证明论序数层级**:

```
ε₀ < Γ₀ < ψ(Ω_ω) < ψ(Ω_Ω) < ... < 递归不可达 < ...
```

## 四、Curry-Howard对应

### 4.1 命题即类型

**对应**:

| 逻辑 | 类型论 | λ演算 |
|------|--------|-------|
| 命题 | 类型 | 项 |
| 证明 | 项/程序 | λ项 |
| 蕴涵消去 | 函数应用 | $f(a)$ |
| 蕴涵引入 | λ抽象 | $\lambda x.t$ |

### 4.2 归一化对应

**定理**: 证明规范化对应于λ项的正规化。

**切割消去**: 对应于β归约。

### 4.3 扩展对应

| 逻辑扩展 | 类型论扩展 |
|----------|-----------|
| 合取 | 积类型 |
| 析取 | 和类型 |
| 假 | 空类型 |
| 量词 | 依赖类型 |

## 五、Cook-Levin定理

### 5.1 定理陈述

**Cook-Levin定理** (1971): SAT是NP完全的。

**证明论意义**: 连接了证明复杂度与计算复杂度。

### 5.2 证明长度与计算

**定理**: 在扩展Frege系统中，存在证明长度与计算时间的对应。

**证明复杂度**: 研究证明系统的最小证明长度。

### 5.3 证明系统层级

| 系统 | 强度 | 对应计算类 |
|------|------|-----------|
| 命题归结 | 弱 | AC⁰ |
| 切消除 | 中 | P |
| 扩展Frege | 强 | P/poly |

## 六、构造性数学与排中律

### 6.1 Brouwer-Heyting-Kolmogorov解释

**构造性证明**: 证明 $\exists x P(x)$ 要求构造满足 $P$ 的 $x$。

**排中律**: $A \lor \neg A$ 在构造性数学中不成立。

### 6.2 可实现性 (Realizability)

**定义**: $e$ 实现 $\varphi$，记作 $e \Vdash \varphi$。

**例子**:
- $e \Vdash A \to B$: $e$ 是将 $A$ 的实现映射到 $B$ 的实现的程序
- $e \Vdash \exists x A(x)$: $e = (n, d)$ 其中 $d \Vdash A(n)$

### 6.3 构造性分析

**定理** (Bishop): 大部分经典分析可以在构造性框架中发展。

**选择原理**: 构造性数学中经常需要添加选择公理。
