# 集合论 (Set Theory)

## 模块概述

集合论是现代数学的基础，为几乎所有数学分支提供了严格的语言框架。本模块从ZFC公理系统出发，深入探讨大基数理论、强制法(forcing)等核心主题。

| [01-基础概念.md](./01-基础概念.md) | 大基数 | 不可达基数、可测基数、超紧基数 |
