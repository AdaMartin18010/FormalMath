# 集合论实战问题

## 一、连续统假设(CH)的独立性证明

### 问题描述

证明 $2^{\aleph_0} = \aleph_1$（CH）独立于ZFC公理系统。

### Cohen力迫法证明 $\neg$CH

**构造**: 设 $M \models \text{ZFC}$ 为可数传递模型。

**力迫偏序**: $\mathbb{P} = \text{Add}(\omega_2, 1)$，即添加$\omega_2$个Cohen实数的有限条件偏序。

**关键步骤**:

1. **c.c.c性质**: $\mathbb{P}$ 满足可数链条件
2. **基数保持**: c.c.c力迫保持基数
3. **添加实数**: $M[G]$ 中添加了$\omega_2$个不同的实数
4. **结论**: $M[G] \models 2^{\aleph_0} \geq \aleph_2$

**详细推导**:

设 $G$ 为 $M$-脱殊滤子，则:
- 对每个 $\alpha < \omega_2^M$，$G$ 提供Cohen实数 $c_\alpha: \omega \to 2$
- 这些实数在 $M[G]$ 中互不相同
- 因此 $M[G] \models |\mathcal{P}(\omega)| \geq \aleph_2$

### Gödel可构造性证明CH

**构造**: 内模型 $L$（可构造宇宙）

**关键定理**: $L \models \text{ZFC} + \text{CH}$

**证明概要**:

1. **可构造层次**: $L = \bigcup_{\alpha \in \text{Ord}} L_\alpha$
2. **良序性**: 在 $L$ 中，$\mathcal{P}(\omega)$ 可良序
3. **决定性**: 若 $x \in \mathcal{P}(\omega) \cap L$，则 $x \in L_{\omega_1}$
4. **计数**: $|\mathcal{P}(\omega) \cap L| \leq \aleph_1$

## 二、决定性公理(AD)的分析

### 问题：AD与选择公理的兼容性

**定理**: AD与AC在全域上不相容，但在$L(\mathbb{R})$中可相容。

### 实战分析

**情形1**: $L(\mathbb{R})$ 中的决定性

**假设**: 存在无限多个Woodin基数和可测基数

**结论**: $L(\mathbb{R}) \models \text{AD}$

**推论**:
- 所有实数集都是Lebesgue可测的
- 所有实数集都具有完美集性质
- 所有实数集都具有Baire性质

**情形2**: 选择公理的放弃

若全域满足AD:
- 无奇异基数（所有不可数基数都是正则的）
- 所有实数集都可测
- 无Well-ordering of $\mathbb{R}$

## 三、大基数层级构造

### 问题：从大基数假设推导结构性质

### 可测基数 $\Rightarrow V \neq L$

**定理** (Scott): 若存在可测基数，则 $V \neq L$。

**证明要点**:

1. 设 $\kappa$ 是可测基数，$U$ 是$\kappa$-完全非主超滤子
2. 构造超幂 $Ult(V, U)$ 和初等嵌入 $j: V \to Ult(V, U)$
3. 关键: $j(\kappa) > \kappa$（临界点性质）
4. 若 $V = L$，则 $j$ 的不动点导致矛盾

### Woodin基数与决定性

**构造**: Woodin基数序列 $\langle \delta_n : n < \omega \rangle$

**定理**: Woodin基数 $\Rightarrow$ $L(\mathbb{R})$ 中的决定性

**应用**: 描述集合论中的正则性性质

## 四、力迫法的实际应用

### 问题：MA + $\neg$CH 的相容性

**Martin's Axiom (MA)**: 对于满足c.c.c的偏序集，任意$< 2^{\aleph_0}$个稠密子集有共同相容滤子。

**力迫构造**:

**迭代力迫**: 有限支撑迭代c.c.c力迫

**步骤**:

1. 初始模型: $M \models \text{ZFC} + \text{CH}$
2. 迭代: 长度 $\omega_2$ 的有限支撑迭代
3. 每个阶段处理一个c.c.c偏序和稠密子集族
4. 最终模型: $M[G] \models \text{MA} + 2^{\aleph_0} = \aleph_2$

### 实际计算问题

**问题**: 在MA + $\neg$CH下，证明$\aleph_1 < 2^{\aleph_0}$

**解**: 由MA，$2^{\aleph_0}$至少是强不可达基数。

## 五、可构造性分析

### 问题：$0^\#$ 的存在性

**定义**: $0^\#$ 是编码 $L$ 中不可定义性的特定实数。

**等价条件**:

1. $0^\#$ 存在
2. 存在非平凡初等嵌入 $j: L \to L$
3. 每个不可数基数在 $L$ 中都是不可达基数
4. 存在 $L$-不可定义的序数真类

**意义**: $0^\#$ 的存在是超越 $L$ 的最小大基数假设。

**提示**: 验证每条ZFC公理在 $V_\kappa$ 中成立。

### 练习2

构造一个力迫扩张使得 $2^{\aleph_0} = \aleph_3$。

**提示**: 使用 $\text{Add}(\omega_3, 1)$。

### 练习3

分析：在AD下，证明每个实数集都是Lebesgue可测的。

**提示**: 利用Banach-Mazur博弈和决定性。
