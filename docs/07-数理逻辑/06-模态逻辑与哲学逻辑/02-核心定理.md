# 模态逻辑与哲学逻辑核心定理

## 一、对应定理 (Correspondence Theory)

### 1.1 Sahlqvist定理

**定理** (Sahlqvist): 每个Sahlqvist公式对应于一阶可定义框架条件。

**Sahlqvist公式类**: 特定语法形式的模态公式。

**例子**:

| 模态公式 | 框架条件 |
|----------|----------|
| □p → p | ∀x(xRx) 自反 |
| □p → □□p | ∀xyz(xRy ∧ yRz → xRz) 传递 |
| p → □◇p | ∀xy(xRy → yRx) 对称 |
| ◇□p → □◇p | McKinsey条件 |

### 1.2 对应算法

**方法**: 将模态公式转换为一阶公式。

**步骤**:

1. 将模态公式的否定转换为二阶形式
2. 应用Sahlqvist算法消除二阶量词
3. 得到一阶对应公式

### 1.3 无法对应的公式

**Löb公式**: □(□p → p) → □p

**对应**: 不存在一阶对应，对应于良基性和传递性。

**这是二阶的**: 良基性不是一阶可定义的。

## 二、Goldblatt-Thomason定理

### 2.1 定理陈述

**Goldblatt-Thomason定理**: 框架类在下列条件下是模态可定义的：

1. 对生成子框架封闭
2. 对不相交并封闭
3. 对有界态射像封闭
4. 对超滤扩展反射封闭

### 2.2 意义

**模态可定义性**: 刻画了哪些框架类可被模态公式定义。

**模型论联系**: 连接模态逻辑与一阶模型论。

### 2.3 应用

**不可定义性证明**: 若框架类不满足上述任一条件，则不是模态可定义的。

## 三、模态完全性定理

### 3.1 K系统完全性

**定理**: K关于所有Kripke框架类是强完全的。

**证明方法**: 典范模型构造

**构造**:

1. 取所有极大一致集作为世界
2. 定义可达关系: wRv 当且仅当 ∀φ(□φ∈w → φ∈v)
3. 验证真值引理

### 3.2 S4和S5完全性

**S4**: 完全关于预序框架（自反且传递）

**S5**: 完全关于等价关系框架

### 3.3 典范模型方法

**一般框架**: (W, R, P) 其中P是可允许集代数

**完全性策略**:

1. 证明系统关于一般框架可靠
2. 使用典范一般模型
3. 必要时细化到Kripke框架

## 四、有限模型性质

### 4.1 定理陈述

**定理**: 若模态公式φ可满足，则它在有限模型中可满足。

**方法**: 过滤(Filtration)技术

### 4.2 过滤技术

**给定**: 模型M = (W, R, V)，公式集Σ在子公式下封闭

**构造**: M^f = (W^f, R^f, V^f)

- W^f = {[w] : w ∈ W}，其中[w] = {v : ∀ψ∈Σ, M,w⊨ψ ↔ M,v⊨ψ}
- 定义适当的R^f
- V^f(p) = {[w] : M,w⊨p} 对p∈Σ

### 4.3 可判定性推论

**定理**: K、T、S4、S5等系统的可满足性问题是可判定的。

**复杂度**: PSPACE完全

## 五、时态逻辑模型检验

### 5.1 CTL模型检验

**算法**: 标记算法

**复杂度**: O(|M| × |φ|)，线性时间

**步骤**:

1. 从子公式开始标记状态
2. 递归计算复合公式的满足集
3. 检查初始状态是否在最终集合中

### 5.2 LTL模型检验

**算法**: 自动机方法

**复杂度**: PSPACE完全

**步骤**:

1. 将¬φ转换为Büchi自动机
2. 构造系统与自动机的积
3. 检查非空性

### 5.3 符号模型检验

**方法**: 使用BDD表示状态集合

**应用**: 硬件和协议验证

## 六、知识逻辑主要定理

### 6.1 公共知识特征

**定理**: 固定点刻画

$$C_G φ ↔ E_G(φ ∧ C_G φ)$$

即公共知识是"人人知道"算子的最大固定点。

### 6.2 知识更新

**公开宣告**: [!φ]ψ 表示"宣告φ后ψ为真"

**归约公理**:

$$[!φ]K_i ψ ↔ (φ → K_i(φ → [!φ]ψ))$$

### 6.3 泥孩子谜题分析

**定理**: n个孩子的泥孩子谜题在n轮迭代后解决。

**证明**: 使用知识算子的迭代消除。

**关键**: 父亲的公告创建了公共知识。

## 七、哲学逻辑问题

### 7.1 指称与意义

**Frege的区分**:

- **Sinn** (意义/涵义): 认知内容
- **Bedeutung** (指称): 真值

**例子**: "晨星"和"暮星"指称相同但意义不同。

### 7.2 真理论

**Tarski的T模式**:

$$T(\text{"φ"}) ↔ φ$$

**不可定义性**: 在足够强的语言中，真概念不能一致地定义。

### 7.3 模糊性

**连锁悖论**: 从明显为真的前提通过有限步骤导出明显为假的结论。

**多值逻辑**: 真值在[0,1]区间内。

**超赋值**: 使用不完全赋值。
