# 模型论基础

## 结构

### 定义

$L$-结构$\mathcal{M} = (M, \{c^\mathcal{M}\}, \{f^\mathcal{M}\}, \{R^\mathcal{M}\})$

## 完备性

### 理论完备

理论$T$完备若对任意句子$\phi$，$T \vdash \phi$或$T \vdash \neg\phi$。

### 量词消去

理论有量词消去若每个公式等价于无量词公式。
