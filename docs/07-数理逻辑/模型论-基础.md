# 模型论基础

模型论（model theory）是数理逻辑的一个分支，研究形式语言与其解释（模型）之间的关系。从 Tarski 的真之定义到 Morley 的 categoricity 定理，再到 Shelah 的分类理论，模型论为代数、分析与组合数学提供了强大的工具。本节从一阶逻辑出发，经紧致性定理到量词消去，建立模型论的核心框架。

## 1. 一阶语言与结构

### 1.1 一阶语言

**定义 1.1（一阶语言）**. 一阶语言 $\mathcal{L}$ 由以下符号组成：
1. **逻辑符号**：$\neg, \wedge, \vee, \to, \forall, \exists, =$；
2. **变元**：$x_1, x_2, \dots$；
3. **非逻辑符号**：函数符号、关系符号、常数符号。

**定义 1.2（$\mathcal{L}$-结构）**. $\mathcal{L}$-结构 $\mathcal{M}$ 由非空论域 $M$ 和对每个符号的解释组成：函数符号解释为 $M$ 上的函数，关系符号解释为 $M$ 上的关系，常数符号解释为 $M$ 中的元素。

### 1.2 满足关系

**定义 1.3（满足）**. 对 $\mathcal{L}$-结构 $\mathcal{M}$、赋值 $s: \text{Var} \to M$ 和公式 $\phi$，定义 $\mathcal{M} \models \phi[s]$（$\mathcal{M}$ 在赋值 $s$ 下满足 $\phi$）归纳于公式结构。

**例子**. $\mathcal{M} \models \forall x \phi(x)[s]$ 当且仅当对所有 $a \in M$，$\mathcal{M} \models \phi(x)[s(x/a)]$，其中 $s(x/a)$ 将 $x$ 映到 $a$ 其余不变。

## 2. 紧致性定理

### 2.1 完备性与紧致性

**定理 2.1（Gödel 完备性定理）**. 一阶逻辑中，$\Gamma \vdash \phi$（可证）当且仅当 $\Gamma \models \phi$（语义后承）。

**定理 2.2（紧致性定理）**. 公式集 $\Gamma$ 有模型当且仅当每个有限子集有模型。

*证明*. 由完备性定理，$\Gamma$ 有模型 $\Leftrightarrow$ $\Gamma$ 一致 $\Leftrightarrow$ 每个有限子集一致 $\Leftrightarrow$ 每个有限子集有模型。$\square$

### 2.2 应用：非标准模型

**定理 2.3**. 存在 $\mathbb{R}$ 的非标准初等扩张 $^*\mathbb{R}$，包含无穷小和无穷大元素。

*证明*. 考虑语言 $\mathcal{L} = \{+, \cdot, <, 0, 1, r\}_{r \in \mathbb{R}}$ 和理论

$$\Gamma = \text{Th}(\mathbb{R}) \cup \{0 < c < 1/n : n \in \mathbb{N}\}.$$

每个有限子集只涉及有限多个 $1/n$，故取 $c$ 为充分小的正实数即得模型。由紧致性，$\Gamma$ 有模型 $^*\mathbb{R}$。$\square$

## 3. Löwenheim-Skolem 定理

**定理 3.1（向上 Löwenheim-Skolem）**. 若 $\mathcal{L}$-结构 $\mathcal{M}$ 为无限，$\kappa \geq |\mathcal{L}|$ 为基数，则存在初等扩张 $\mathcal{N} \succ \mathcal{M}$ 使 $|N| = \kappa$。

**定理 3.2（向下 Löwenheim-Skolem）**. 若 $\mathcal{L}$-结构 $\mathcal{M}$ 无限，$A \subseteq M$，则存在初等子结构 $\mathcal{N} \prec \mathcal{M}$ 使 $A \subseteq N$ 且 $|N| \leq |A| + |\mathcal{L}|$。

*证明*. 构造闭包：从 $A$ 出发，对每个公式 $\exists x \phi(x, \bar{a})$（$\bar{a} \in N$），若 $\mathcal{M} \models \exists x \phi$，加入见证元。迭代 $|\mathcal{L}|$ 次，所得集合 $N$ 关于 Tarski 判别条件封闭，故为初等子结构。$\square$

## 4. 量词消去

### 4.1 定义与判定

**定义 4.1（量词消去）**. 理论 $T$ 允许量词消去，若对每个公式 $\phi(\bar{x})$，存在无量词公式 $\psi(\bar{x})$ 使得 $T \models \forall \bar{x}(\phi(\bar{x}) \leftrightarrow \psi(\bar{x}))$。

**定理 4.2**. 无端点稠密线性序 $(\mathbb{Q}, <)$ 的理论 DLO 允许量词消去。

*证明*. 对 $\exists x \phi(x, \bar{y})$，$\phi$ 无量词，分析 $x$ 相对于 $y_i$ 的位置关系（$x < y_i$，$x = y_i$，$x > y_i$）。由稠密性和无端点，存在性等价于有限个不等式的合取。$\square$

### 4.2 代数闭域

**定理 4.3（Tarski）**. 代数闭域的理论 ACF 允许量词消去。

**推论 4.4**. 代数闭域的可定义集是有限个代数簇与代数集补的布尔组合（Chevalley 定理）。

## 5. 例子

### 5.1 例子：实闭域

**定义 5.1（实闭域）**. 域 $F$ 称为实闭的，若 $-1$ 不是平方和，且 $F(i)$ 代数闭。

**定理 5.2（Tarski-Seidenberg）**. 实闭域的理论 RCF 允许量词消去。

*推论*. 半代数集（多项式等式与不等式定义的集合）的投影仍是半代数集。

### 5.2 例子：随机图理论

Erdős-Rényi 随机图 $G(n, 1/2)$ 的几乎必然理论是完备且 $\aleph_0$-categorical 的，允许量词消去。
