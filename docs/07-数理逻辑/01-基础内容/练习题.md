# 数理逻辑综合练习题

> **适用文档**: 01-基础内容全系列  
> **难度分级**: ⭐ 基础 | ⭐⭐ 进阶 | ⭐⭐⭐ 挑战  
> **目标**: 巩固命题逻辑、谓词逻辑、模态逻辑、直觉逻辑、多值逻辑的核心概念

---

## 一、命题逻辑 (Propositional Logic)

### ⭐ 基础练习

#### 练习 1.1: 真值表验证

**题目**: 构造以下公式的真值表，判断其类型(重言式、可满足式、矛盾式)：

1. $(p \rightarrow q) \leftrightarrow (\neg q \rightarrow \neg p)$
2. $(p \land q) \rightarrow (p \lor q)$
3. $(p \rightarrow q) \land (q \rightarrow r) \land \neg(p \rightarrow r)$

**解答**:

1. **逆否命题** (重言式)

| $p$ | $q$ | $\neg p$ | $\neg q$ | $p \rightarrow q$ | $\neg q \rightarrow \neg p$ | 等价 |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| T | T | F | F | T | T | T |
| T | F | F | T | F | F | T |
| F | T | T | F | T | T | T |
| F | F | T | T | T | T | T |

2. **蕴含关系** (重言式)

$p \land q$为真仅当$p,q$都为真，此时$p \lor q$必为真。

3. **假言三段论的反例** (矛盾式)

若前件$(p \rightarrow q) \land (q \rightarrow r)$为真，则$p \rightarrow r$必为真，故与$\neg(p \rightarrow r)$矛盾。

---

#### 练习 1.2: 逻辑等价变换

**题目**: 使用德摩根律和蕴含等价，将以下公式化为合取范式(CNF)：

$$\neg((p \rightarrow q) \land (r \lor \neg s))$$

**解答**:

第一步：消去蕴含
$$\neg((\neg p \lor q) \land (r \lor \neg s))$$

第二步：德摩根律
$$\neg(\neg p \lor q) \lor \neg(r \lor \neg s)$$

第三步：再次德摩根律
$$(p \land \neg q) \lor (\neg r \land s)$$

第四步：分配律化为CNF
$$(p \lor \neg r) \land (p \lor s) \land (\neg q \lor \neg r) \land (\neg q \lor s)$$

---

#### 练习 1.3: 自然演绎证明

**题目**: 用自然演绎证明：$p \rightarrow q, q \rightarrow r \vdash p \rightarrow r$

**解答**:

```
1. p → q      [前提]
2. q → r      [前提]
3. | p        [假设]
4. | q        [→E: 1, 3]
5. | r        [→E: 2, 4]
6. p → r      [→I: 3-5]
```

---

### ⭐⭐ 进阶练习

#### 练习 1.4: SAT问题分析

**题目**: 使用DPLL算法判断以下CNF公式的可满足性：

$$(p \lor q \lor r) \land (\neg p \lor \neg q) \land (\neg p \lor \neg r) \land (\neg q \lor \neg r)$$

**解答**:

**初始**: $\{\{p, q, r\}, \{\neg p, \neg q\}, \{\neg p, \neg r\}, \{\neg q, \neg r\}\}$

**步骤 1**: 无单元子句，选择分支变量$p = \text{true}$

**简化后**: $\{\{\neg q\}, \{\neg r\}, \{\neg q, \neg r\}\}$

**步骤 2**: 单元传播 $\neg q$，$\neg r$

**结果**: 所有子句满足 ✓

**满足赋值**: $p = \top, q = \bot, r = \bot$

验证：
- $(p \lor q \lor r) = \top \lor \bot \lor \bot = \top$ ✓
- $(\neg p \lor \neg q) = \bot \lor \top = \top$ ✓
- $(\neg p \lor \neg r) = \bot \lor \top = \top$ ✓
- $(\neg q \lor \neg r) = \top \lor \top = \top$ ✓

---

#### 练习 1.5: 可靠性定理证明

**题目**: 证明自然演绎的可靠性：若$\Gamma \vdash \varphi$，则$\Gamma \vDash \varphi$

**证明概要**:

对证明长度进行归纳。

**基例**: 
- 若$\varphi$是公理，验证其为重言式
- 若$\varphi \in \Gamma$，显然$\Gamma \vDash \varphi$

**归纳步**: 假设对长度$< n$的证明成立，考虑长度为$n$的证明最后一步：

- **→E**: 由$\psi$和$\psi \rightarrow \varphi$得到$\varphi$
  - 归纳假设：$\Gamma \vDash \psi$和$\Gamma \vDash \psi \rightarrow \varphi$
  - 故对任意满足$\Gamma$的赋值$v$，$v(\psi) = \top$且$v(\psi \rightarrow \varphi) = \top$
  - 由→语义，$v(\varphi) = \top$

- **∧I, ∨I, ¬I等**: 类似分析 ∎

---

### ⭐⭐⭐ 挑战练习

#### 练习 1.6: 完备性定理构造

**题目**: 设$\Gamma = \{p \rightarrow q, \neg q\}$，证明$\Gamma \vDash \neg p$并给出语法证明。

**语义证明**:

任取赋值$v$满足$\Gamma$：
- $v(\neg q) = \top \Rightarrow v(q) = \bot$
- $v(p \rightarrow q) = \top$且$v(q) = \bot \Rightarrow v(p) = \bot$
- 故$v(\neg p) = \top$ ∎

**语法证明**:

```
1. p → q      [前提]
2. ¬q         [前提]
3. | p        [假设]
4. | q        [→E: 1, 3]
5. | ⊥        [¬E: 2, 4]
6. ¬p         [¬I: 3-5]
```

## 二、一阶逻辑 (First-Order Logic)

### ⭐ 基础练习

#### 练习 2.1: 形式化翻译

**题目**: 将以下自然语言语句翻译为一阶逻辑公式(使用适当谓词)：

1. "所有人类都是会死的"
2. "存在一个大于10的素数"
3. "每个学生都有一位导师"
4. "没有大于2的偶素数"

**解答**:

1. $\forall x (Human(x) \rightarrow Mortal(x))$

2. $\exists x (Prime(x) \land x > 10)$

3. $\forall x (Student(x) \rightarrow \exists y (Advisor(y, x)))$
   
   或更精确：$\forall x (Student(x) \rightarrow \exists y (Professor(y) \land Mentors(y, x)))$

4. $\neg\exists x (Even(x) \land Prime(x) \land x > 2)$
   
   或等价地：$\forall x ((Even(x) \land Prime(x)) \rightarrow x \leq 2)$

---

#### 练习 2.2: 语义验证

**题目**: 设$\mathcal{M} = (\mathbb{N}, 0, S, +, =)$，验证$\mathcal{M} \vDash \forall x \exists y (y = S(x))$

**解答**:

对任意$n \in \mathbb{N}$：
- 取赋值$s(x) = n$
- 需证$\mathcal{M} \vDash \exists y (y = S(x))[s]$
- 即存在$m \in \mathbb{N}$使$\mathcal{M} \vDash y = S(x)[s[y/m]]$
- 取$m = n + 1$
- $S(x)[s] = n + 1 = m$
- 故$y[s[y/m]] = m = S(x)[s]$ ∎

---

### ⭐⭐ 进阶练习

#### 练习 2.3: 紧致性应用

**题目**: 证明：若每个有限子图都是平面图，则无限图也是平面图。

**解答**:

构造一阶语言：常数$c_v$对每个顶点$v$，谓词$E(x,y)$表示边。

公式集$\Gamma$包含：
1. $E(c_u, c_v)$ 对每条边$(u,v) \in E$
2. $\neg E(c_u, c_v)$ 对每条非边
3. Kuratowski定理的一阶近似：不包含$K_5$或$K_{3,3}$的子图

任取有限$\Gamma_0 \subseteq \Gamma$，涉及有限子图$G_0$，由假设$G_0$是平面图，故$\Gamma_0$可满足。

由紧致性定理，$\Gamma$可满足，即原图是平面图。 ∎

---

#### 练习 2.4: 量词否定

**题目**: 证明$\neg\forall x P(x)$与$\exists x \neg P(x)$逻辑等价。

**证明**:

**左到右**:
- 设$\neg\forall x P(x)$
- 即并非对所有$x$，$P(x)$成立
- 故存在某个$x$使$P(x)$不成立
- 即$\exists x \neg P(x)$

**形式证明** (自然演绎)：

```
1. ¬∀x P(x)           [假设]
2. | ¬∃x ¬P(x)        [假设(反证)]
3. | | ¬P(a)          [假设，a为新常数]
4. | | ∃x ¬P(x)       [∃I: 3]
5. | | ⊥              [¬E: 2, 4]
6. | ¬¬P(a)           [¬I: 3-5]
7. | P(a)             [¬¬E: 6]
8. | ∀x P(x)          [∀I: 7]
9. | ⊥                [¬E: 1, 8]
10. ¬¬∃x ¬P(x)        [¬I: 2-9]
11. ∃x ¬P(x)          [¬¬E: 10]
```

---

### ⭐⭐⭐ 挑战练习

#### 练习 2.5: Löwenheim-Skolem定理应用

**题目**: 证明：存在非标准模型$^*\mathbb{R}$使得$^*\mathbb{R} \equiv \mathbb{R}$但包含无限小元素。

**解答**:

**构造**:
- 语言：实数域的语言$(+, \times, <, 0, 1, c_r(r \in \mathbb{R}))$
- 理论：$Th(\mathbb{R})$（所有在$\mathbb{R}$中真的句子）
- 添加新常数$\epsilon$和公理：$0 < \epsilon < c_{1/n}$对所有$n \in \mathbb{N}^+$

**紧致性应用**:
- 任取有限子集，涉及$0 < \epsilon < 1/n_1, ..., 1/n_k$
- 取$\epsilon = 1/(\max(n_i) + 1)$可满足
- 由紧致性，整个理论可满足

**非标准模型**$^*\mathbb{R}$:
- $^*\mathbb{R} \vDash Th(\mathbb{R})$
- $^*\mathbb{R}$包含无限小$\epsilon$
- 由Löwenheim-Skolem上升定理，可假设$|^*\mathbb{R}| = |\mathbb{R}|$ ∎

## 三、模态逻辑 (Modal Logic)

### ⭐ 基础练习

#### 练习 3.1: Kripke模型验证

**题目**: 在以下模型中验证$w_1 \vDash \Box p \rightarrow \Box\Box p$：

```
    w1 → w2 → w3
    p    ¬p   p
```

**解答**:

**$w_1 \vDash \Box p$?**
- $w_1$可达：$w_1, w_2$
- $w_1 \vDash p$ ✓，$w_2 \vDash \neg p$ ✗
- 故$w_1 \not\vDash \Box p$

**$w_1 \vDash \Box p \rightarrow \Box\Box p$?**
- 前件假，整个蕴含为真 ✓

---

#### 练习 3.2: 公理系统证明

**题目**: 在系统K中证明：$\vdash \Box(p \land q) \rightarrow (\Box p \land \Box q)$

**证明**:

```
1. (p ∧ q) → p      [重言式]
2. □((p ∧ q) → p)   [Nec: 1]
3. □(p ∧ q) → □p    [K: 2]
4. (p ∧ q) → q      [重言式]
5. □((p ∧ q) → q)   [Nec: 4]
6. □(p ∧ q) → □q    [K: 5]
7. (□(p ∧ q) → □p) ∧ (□(p ∧ q) → □q)  [∧I: 3, 6]
8. □(p ∧ q) → (□p ∧ □q)              [重言式+MP]
```

---

### ⭐⭐ 进阶练习

#### 练习 3.3: 对应理论

**题目**: 证明：公理$\Box p \rightarrow p$对应关系的自反性。

**证明**:

**($\Rightarrow$) 公理有效→自反性**:

设$\mathcal{F} \vDash \Box p \rightarrow p$。

假设$\mathcal{F}$不自反，即存在$w$使得$w \not\rightarrow w$。

构造模型：$V(p) = W \setminus \{w\}$。

则$\mathcal{M}, w \vDash \Box p$ (因为$w$无自环，$w$看不到任何不满足$p$的世界)。

但$\mathcal{M}, w \not\vDash p$ (由构造)。

矛盾！故$\mathcal{F}$自反。

**($\Leftarrow$) 自反性→公理有效**:

设$\mathcal{F}$自反，任取模型$\mathcal{M}$和世界$w$。

设$\mathcal{M}, w \vDash \Box p$。

由自反性，$w \rightarrow w$。

故$\mathcal{M}, w \vDash p$。

因此$\mathcal{M}, w \vDash \Box p \rightarrow p$。 ∎

---

#### 练习 3.4: 泥孩子谜题 (简化版)

**题目**: 分析两个孩子的泥孩子谜题：父亲公告"至少一个孩子有泥"后，每个孩子能看到对方。经过几轮"不知道"后，有泥的孩子会知道？

**解答**:

**可能世界**: $\{MM, MC, CM, CC\}$ (M=有泥, C=干净)

**观察关系**:
- 孩子1能看到孩子2：无法区分$MM \leftrightarrow CM$，$MC \leftrightarrow CC$
- 孩子2能看到孩子1：无法区分$MM \leftrightarrow MC$，$CM \leftrightarrow CC$

**初始** (父亲公告后)：排除$CC$

**第一轮**:
- 在$MC$：孩子1看到$C$，知道必有泥 → 回答"知道"
- 在$CM$：孩子2看到$C$，知道必有泥 → 回答"知道"
- 在$MM$：都看到泥，不确定 → 都回答"不知道"

**第二轮** (听到"不知道"):
- 在$MM$：孩子1推理"若我干净，孩子2应看到并知道，但孩子2说不知道，故我有泥"
- 同理孩子2也推断有泥
- 都回答"知道"

**结论**: 两个孩子都有泥时，第二轮回答"知道"。

## 四、直觉逻辑 (Intuitionistic Logic)

### ⭐ 基础练习

#### 练习 4.1: BHK解释

**题目**: 给出以下公式的Brouwer-Heyting-Kolmogorov (BHK) 解释：

1. $p \rightarrow (q \rightarrow p)$
2. $(p \rightarrow q) \rightarrow ((q \rightarrow r) \rightarrow (p \rightarrow r))$

**解答**:

**BHK解释原则**:
- $p \land q$的证明：对$(p$的证明$, q$的证明$)$
- $p \lor q$的证明：要么$p$的证明，要么$q$的证明
- $p \rightarrow q$的证明：将$p$的证明转换为$q$的证明的函数

1. **$p \rightarrow (q \rightarrow p)$**
   - 输入：$p$的证明
   - 输出：函数(输入$q$的证明，输出$p$的证明)
   - 实现：$\lambda x. \lambda y. x$ (忽略$y$，返回$x$)

2. **假言三段论**
   - 输入：函数$f: p \rightarrow q$
   - 输出：函数(输入$g: q \rightarrow r$，输出函数$h: p \rightarrow r$)
   - 实现：$\lambda f. \lambda g. \lambda x. g(f(x))$ (函数复合)

---

#### 练习 4.2: 排中律不成立

**题目**: 解释为什么$p \lor \neg p$在直觉逻辑中不成立。

**解答**:

在直觉逻辑中，$p \lor \neg p$要求：
- 要么提供$p$的证明
- 要么提供$\neg p$的证明(即$p \rightarrow \bot$的函数)

但对于任意命题$p$，我们可能既无$p$的证明，也无$p$的反驳。

**例子**: 设$p$ = "存在奇完全数"
- 目前数学既未证明存在，也未证明不存在
- 故不能有$p \lor \neg p$的构造性证明

---

### ⭐⭐ 进阶练习

#### 练习 4.3: Curry-Howard对应

**题目**: 写出对应于以下证明的λ项：

```
p → q, q → r ⊢ p → r
```

**解答**:

**证明树**:
```
[p]¹    p → q
----------- →E
     q        q → r
     --------------- →E
            r
      --------- →I¹
       p → r
```

**对应λ项**: $\lambda x^{p}. g(f(x))$

其中：
- $f : p \rightarrow q$
- $g : q \rightarrow r$
- 类型: $p \rightarrow r$

## 五、多值逻辑 (Many-Valued Logic)

### ⭐ 基础练习

#### 练习 5.1: Łukasiewicz三值逻辑

**题目**: 在Łukasiewicz三值逻辑中，计算以下公式的真值($v(p) = 1/2, v(q) = 0$)：

1. $p \rightarrow q$
2. $\neg p \lor q$
3. $p \land \neg p$

**真值表**:

| $p$ | $\neg p$ | $p \land q$ | $p \lor q$ | $p \rightarrow q$ |
|:---:|:---:|:---:|:---:|:---:|
| 0 | 1 | $\min(p,q)$ | $\max(p,q)$ | $\min(1, 1-p+q)$ |
| 1/2 | 1/2 | | | |
| 1 | 0 | | | |

**解答**:

1. $p \rightarrow q = \min(1, 1 - 1/2 + 0) = \min(1, 1/2) = 1/2$

2. $\neg p = 1/2$，故$\neg p \lor q = \max(1/2, 0) = 1/2$

3. $p \land \neg p = \min(1/2, 1/2) = 1/2$

**注意**: 排中律不成立，$p \land \neg p$不是矛盾式(值为1/2而非0)。

## 六、综合挑战

### ⭐⭐⭐ 挑战 1: Gödel句子

**题目**: 设$T$是包含PA的一致理论，$G$是Gödel句子满足$T \vdash G \leftrightarrow \neg Prov_T(\ulcorner G \urcorner)$。证明若$T$一致，则$T \not\vdash G$且$T \not\vdash \neg G$。

**证明**:

**不可证性**:

假设$T \vdash G$。

1. 则存在$G$的证明，故$Prov_T(\ulcorner G \urcorner)$为真
2. 由可表示性，$T \vdash Prov_T(\ulcorner G \urcorner)$
3. 由$G$的定义，$T \vdash \neg Prov_T(\ulcorner G \urcorner)$
4. 故$T \vdash Prov_T(\ulcorner G \urcorner) \land \neg Prov_T(\ulcorner G \urcorner)$
5. $T$不一致，矛盾！

因此$T \not\vdash G$。

**不可否认性**: (需要更强的一致性条件) 类似论证。 ∎

---

### ⭐⭐⭐ 挑战 2: 停机问题

**题目**: 用对角线法证明停机问题不可判定。

**证明**:

假设存在判定器$H(\langle M \rangle, w)$判定$M$是否在$w$上停机。

构造$D$:
```
D on input ⟨M⟩:
  if H(⟨M⟩, ⟨M⟩) = "halt" then
    loop forever
  else
    halt
```

考虑$D(\langle D \rangle)$:
- 若$H(\langle D \rangle, \langle D \rangle)$ = "halt"，则$D$永不停机，矛盾
- 若$H(\langle D \rangle, \langle D \rangle)$ = "loop"，则$D$停机，矛盾

故$H$不存在。 ∎
