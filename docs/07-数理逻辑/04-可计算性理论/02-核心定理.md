# 可计算性理论核心定理

## 一、Church-Turing论题

### 1.1 论题陈述

**Church-Turing论题**: 直观上可计算的函数恰好是图灵可计算函数。

**注记**: 这不是数学定理，而是对"可计算"概念的经验性概括。

### 1.2 支持证据

**等价性定理**: 以下计算模型等价：

- 图灵机
- λ演算
- 递归函数
- 寄存器机
- 随机存取机 (RAM)
- 元胞自动机
- ...

**稳健性**: 所有合理的计算模型都定义了相同的可计算函数类。

### 1.3 物理Church-Turing论题

**强论题**: 任何物理过程都可被图灵机模拟。

**量子挑战**: 量子计算似乎可高效模拟，但不超越图灵可计算性。

## 二、停机问题不可判定性

### 2.1 Turing原始证明

**定理**: A_TM = {⟨M, w⟩ : M 接受 w} 是不可判定的。

**对角化证明**:

假设H判定A_TM，构造D：

```
D(⟨M⟩):
  运行 H(⟨M, ⟨M⟩⟩)
  若 H 接受，则拒绝
  若 H 拒绝，则接受
```

矛盾：D(⟨D⟩)既接受又拒绝。

### 2.2 归约方法

**HALT_TM ≤_m A_TM**: 停机问题可归约到接受问题

**推论**: HALT_TM是不可判定的。

### 2.3 列表更多不可判定问题

| 问题 | 描述 |
|------|------|
| E_TM | 空语言问题 |
| EQ_TM | 等价问题 |
| REGULAR_TM | 语言是否正则 |
| DECIDABLE_TM | 语言是否可判定 |
| 自我复制 | 程序打印自身 |

## 三、Rice定理

### 3.1 定理陈述

**Rice定理**: 任何关于图灵机语言的非平凡性质都是不可判定的。

**形式**: 设P是RE语言的非平凡性质，则：

$$L_P = \{⟨M⟩ : L(M) \in P\}$$

是不可判定的。

### 3.2 证明概要

1. 不失一般性，假设空语言 ∉ P
2. 取某个满足P的图灵机M₀
3. 从A_TM归约: 给定⟨M, w⟩，构造M'：
   - M'首先模拟M在w上
   - 若M接受w，则模拟M₀在输入上
   - 若M拒绝w，则拒绝
4. L(M') = L(M₀) ∈ P 当且仅当 M接受w

### 3.3 应用

以下都是不可判定的：

- 程序是否对任何输入停机
- 程序是否计算常值函数
- 程序是否接受有限语言
- 两个程序是否等价

## 四、Cook-Levin定理

### 4.1 定理陈述

**Cook-Levin定理** (1971): SAT是NP完全的。

**形式**:

1. SAT ∈ NP
2. ∀L ∈ NP, L ≤_p SAT

### 4.2 证明概要

**第一部分**: SAT ∈ NP

- 证书: 可满足赋值
- 验证器: 检查赋值是否满足所有子句
- 多项式时间验证

**第二部分**: NP ⊆ P^SAT

- 给定L ∈ NP，设N是在时间n^k内判定L的非确定图灵机
- 对每个输入w，构造布尔公式φ_w：
  - 变量编码计算历史
  - 约束确保计算有效
  - 大小多项式有界
- φ_w可满足 ⟺ N接受w

### 4.3 推论

- 3-SAT是NP完全的
- 若SAT ∈ P，则P = NP
- SAT的困难性是NP的"核心"

## 五、Savitch定理

### 5.1 定理陈述

**Savitch定理**: PSPACE = NPSPACE

**更强形式**: NSPACE(s(n)) ⊆ SPACE(s(n)²)，其中s(n) ≥ log n。

### 5.2 证明概要

**关键**: 可达性问题可在平方空间内解决。

**算法**: 对非确定图灵机M，判断M从配置c₁能否到达c₂：

```
REACHABLE(c₁, c₂, t):
  若 t = 0: 返回 (c₁ = c₂)
  若 t = 1: 返回 (c₁ → c₂ 直接转移)
  对中间配置c_m:
    若 REACHABLE(c₁, c_m, t/2) 且 REACHABLE(c_m, c₂, t/2):
      返回真
  返回假
```

**空间分析**: 递归深度O(log t)，每层O(s(n))空间。

### 5.3 推论

- NL ⊆ L²（实际上NL ⊆ L²不完全精确，但方向正确）
- PSPACE = NPSPACE = coNPSPACE

## 六、时间层次定理

### 6.1 确定性时间层次

**定理**: 若 f(n) 是时间可构造的，则：

$$\text{TIME}(f(n)) \subsetneq \text{TIME}(f(n) \cdot \log f(n))$$

**更强的**: TIME(f(n)) ⊊ TIME(f(n)²)

### 6.2 推论

- P ⊊ EXPTIME
- 存在严格的时间层次

### 6.3 空间层次

**定理**: 若 f(n) ≥ log n 是空间可构造的，则：

$$\text{SPACE}(f(n)) \subsetneq \text{SPACE}(f(n) \cdot \log f(n))$$

**推论**: L ⊊ PSPACE

## 七、Immerman-Szelepcsényi定理

### 7.1 定理陈述

**Immerman-Szelepcsényi定理**: NL = coNL

**更强形式**: 对于s(n) ≥ log n，NSPACE(s(n)) = coNSPACE(s(n))。

### 7.2 证明思想

**可达性计数**:

1. 先计算从起点可达的配置数N
2. 然后非确定地验证不可达性
3. 使用"归纳计数"技术

**关键**: 通过巧妙计数，在NL中验证补问题。

### 7.3 意义

- 解决了长期开放的NL = coNL问题
- 展示了非确定性空间类的对称性
- 不同于时间类（NP ?= coNP开放）

## 八、Cook-Levin定理的详细编码构造

### 8.1 从非确定图灵机到布尔公式

设 $N$ 是在时间 $n^k$ 内运行的非确定图灵机。对输入 $w$（长度 $n$），构造布尔公式 $\varphi_w$ 使得 $\varphi_w$ 可满足当且仅当 $N$ 接受 $w$。

**变量设计**：
- $T_{i,j,s}$：在时间 $i$，带方格 $j$ 包含符号 $s$。
- $H_{i,j}$：在时间 $i$，读写头位于方格 $j$。
- $Q_{i,q}$：在时间 $i$，状态为 $q$。

其中 $0 \leq i, j \leq n^k$。

**约束公式**：
1. **初始配置**：$Q_{0,q_0} \land H_{0,0} \land$ 输入 $w$ 编码在 $T_{0,j}$ 中。
2. **转移约束**（窗口条件）：对每个时间 $i$ 和位置 $j$，$(H_{i,j} \land Q_{i,q} \land T_{i,j,s}) \to \bigvee_{(q',s',D) \in \delta(q,s)} (Q_{i+1,q'} \land T_{i+1,j,s'} \land H_{i+1,j+D})$。
3. **接受状态**：$Q_{n^k, q_{\text{accept}}}$。

**大小分析**：变量数和子句数均为 $O(n^{2k})$，多项式有界。

## 九、Immerman-Szelepcsényi定理的归纳计数

### 9.1 可达性计数算法

**目标**：在NL中验证不可达性（即验证 $s$ 到 $t$ 无路径）。

**步骤1：计算可达配置数 $N$**

对 $i = 0, 1, \dots, n^k$（$n^k$ 是配置总数的多项式上界）：
- 维护计数器 $c_i$ = 从 $s$ 出发长度 $\leq i$ 可达的配置数。
- $c_0 = 1$（仅 $s$ 自身）。
- 对 $c_{i+1}$：对每个配置 $v$，非确定地猜测一条从 $s$ 到 $v$ 的长度 $\leq i+1$ 路径；同时用 $c_i$ 验证已统计所有中间配置。

**步骤2：验证不可达性**

利用已知 $N$（从 $s$ 可达的总配置数），非确定地猜测 $N$ 个不同的可达配置，验证它们均不是 $t$。若成功，则 $t$ 不可达。

**关键**：在NL中可维护计数器（对数空间），并通过非确定性枚举和交叉验证实现补集判定。

## 十、时间层次定理的对角线构造

### 10.1 确定性时间层次

**定理**：若 $f(n)$ 时间可构造且 $f(n) \geq n$，则：
$$\text{TIME}(f(n)) \subsetneq \text{TIME}(f(n) \cdot \log f(n))$$

**证明概要**：构造图灵机 $D$，在输入 $\langle M \rangle$ 上：
1. 模拟 $M(\langle M \rangle)$ 至多 $f(|\langle M \rangle|)$ 步。
2. 若 $M$ 停机并拒绝，$D$ 接受；若接受，$D$ 拒绝；若超时，$D$ 拒绝。

$D$ 的运行时间为 $O(f(n) \cdot \log f(n))$（模拟开销）。但对任何在 $f(n)$ 时间内运行的 $M$，$D(\langle M \rangle) \neq M(\langle M \rangle)$。因此 $L(D) \notin \text{TIME}(f(n))$。
