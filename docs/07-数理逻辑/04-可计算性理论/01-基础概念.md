# 可计算性理论基础概念

## 一、图灵机 (Turing Machine)

### 1.1 形式定义

**图灵机**: M = (Q, Σ, Γ, δ, q₀, q_accept, q_reject)

- Q: 有限状态集
- Σ: 输入字母表（不含空白符 ⊔）
- Γ: 带字母表，Σ ⊂ Γ，⊔ ∈ Γ
- δ: Q × Γ → Q × Γ × {L, R} 转移函数（部分函数）
- q₀: 初始状态
- q_accept, q_reject: 接受/拒绝状态

### 1.2 可计算性

**定义**: 函数 f: Σ*→ Σ* 是**图灵可计算**的，若存在图灵机M，对任意输入w，M停机并输出f(w)。

**判定器与识别器**:

- **判定器**: 对所有输入都停机的图灵机
- **识别器**: 可能永不停机的图灵机

**语言分类**:

- **递归 (R)**: 存在判定器的语言
- **递归可枚举 (RE)**: 存在识别器的语言

### 1.3 等价模型

**Church-Turing论题**: 直观可计算 = 图灵可计算

| 模型 | 等价性 |
|------|--------|
| λ演算 | 等价于图灵机 |
| 递归函数 | 等价于图灵机 |
| 寄存器机 | 等价于图灵机 |
| 元胞自动机 | 等价于图灵机 |
| 量子图灵机 | 计算能力相同（多项式时间？） |

## 二、不可判定性

### 2.1 停机问题

**定理** (Turing): 停机问题是不可判定的。

**问题**: A_TM = {⟨M, w⟩ : M 接受 w}

**证明** (对角化):

1. 假设存在判定器H判定A_TM
2. 构造图灵机D: "在输入⟨M⟩上，运行H(⟨M, ⟨M⟩⟩)，若H接受则拒绝，若H拒绝则接受"
3. 运行D(⟨D⟩)导致矛盾

### 2.2 Rice定理

**定理** (Rice): 关于程序语义的非平凡性质都是不可判定的。

**形式**: 设P是图灵机语言的非平凡性质，则语言 {⟨M⟩ : L(M) ∈ P} 是不可判定的。

**推论**: 以下问题都不可判定：

- 程序是否计算常值函数
- 程序是否接受空语言
- 两个程序是否等价

### 2.3 归约方法

**定义**: 语言A可归约到B (A ≤_m B)，若存在可计算函数f使得 w ∈ A ⟺ f(w) ∈ B。

**性质**:

- 若 A ≤_m B 且 B 可判定，则A可判定
- 若 A ≤_m B 且 B ∈ RE，则A ∈ RE

**应用**: 证明新问题不可判定

| 问题 | 归约来源 | 含义 |
|------|----------|------|
| 停机问题 | - | 基础不可判定问题 |
| Post对应问题 | 停机问题 | 组合问题 |
| 群字问题 | 停机问题 | 代数问题 |
| Hilbert第十问题 | 停机问题 | 数论问题 |

## 三、复杂度类

### 3.1 时间复杂度

**定义**: TIME(t(n)) = {L : 存在图灵机M判定L，M在O(t(n))时间内运行}

**重要类别**:

| 类别 | 定义 | 例子 |
|------|------|------|
| P | ∪_k TIME(n^k) | 图连通性、线性规划 |
| EXPTIME | ∪_k TIME(2^{n^k}) | 棋类游戏 |
| 2-EXPTIME | ∪_k TIME(2^{2^{n^k}}) | Presburger算术 |

### 3.2 空间复杂度

**定义**: SPACE(s(n)) = {L : 存在图灵机M判定L，M使用O(s(n))空间}

**重要类别**:

| 类别 | 定义 | 关系 |
|------|------|------|
| L | SPACE(log n) | L ⊆ P |
| NL | 非确定对数空间 | NL ⊆ P |
| PSPACE | ∪_k SPACE(n^k) | P ⊆ PSPACE |
| EXPSPACE | ∪_k SPACE(2^{n^k}) | PSPACE ⊆ EXPSPACE |

### 3.3 非确定性

**定义**: NTIME(t(n)) = {L : 存在非确定图灵机在O(t(n))时间内接受L}

**NP** (Nondeterministic Polynomial time):

- 验证版本: L ∈ NP若存在多项式时间验证器V，使得 w ∈ L ⟺ ∃c: V(w, c) = 1
- 证书长度|c| = poly(|w|)

**NP完全问题**:

- 可满足性问题 (SAT)
- 3-SAT
- 团问题 (Clique)
- 顶点覆盖
- 哈密尔顿路径
- 子集和

## 四、复杂度层次

### 4.1 基本包含关系

```

L ⊆ NL ⊆ P ⊆ NP ⊆ PSPACE ⊆ EXPTIME ⊆ EXPSPACE

```

**已知真包含**:

- L ⊂ PSPACE (空间层次定理)
- P ⊂ EXPTIME (时间层次定理)

**未解决问题**:

- L =? NL
- P =? NP (千禧年问题)
- NP =? PSPACE

### 4.2 多项式层次 (Polynomial Hierarchy)

**定义**:

- Σ₀^P = Π₀^P = P
- Σ_{k+1}^P = NP^{Σ_k^P}
- Π_{k+1}^P = coNP^{Σ_k^P}
- PH = ∪_k Σ_k^P

**层次结构**:

```

P = Σ₀^P = Π₀^P
  ⊆ Σ₁^P = NP    ⊆ Σ₂^P    ⊆ ... ⊆ PH
  ⊆ Π₁^P = coNP  ⊆ Π₂^P    ⊆ ...

```

**性质**: 若 PH 坍缩（某层等于下一层），则整个层次坍缩到该层。

### 4.3 概率复杂度类

| 类别 | 定义 | 特点 |
|------|------|------|
| RP | 单边错误，多项式时间 | 接受时必定正确 |
| coRP | RP的补 | 拒绝时必定正确 |
| BPP | 双边错误，多项式时间 | 错误概率 ≤ 1/3 |
| ZPP | RP ∩ coRP | 零错误，期望多项式 |
| PP | 多数正确 | 弱于PSPACE |

**关系**:

- ZPP = RP ∩ coRP
- P ⊆ ZPP ⊆ RP ⊆ NP
- BPP ⊆ Σ₂^P ∩ Π₂^P
