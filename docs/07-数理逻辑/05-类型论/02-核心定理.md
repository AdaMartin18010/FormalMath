# 类型论核心定理

## 一、Church-Rosser定理

### 1.1 定理陈述

**Church-Rosser定理** (合流性): 若 t →*u₁ 且 t →* u₂，则存在v使得 u₁ →*v 且 u₂ →* v。

**图示**:

```

    t
   / \
  ↓   ↓
 u₁   u₂
  \   /
   ↓ ↓
    v

```

### 1.2 证明概要

**菱形性质**: 若 t → u₁ 且 t → u₂，则存在v使得 u₁ →*v 且 u₂ →* v。

**步骤**:

1. 证明单步归约满足菱形性质
2. 通过归纳扩展到多步归约
3. 使用平行归约(parallel reduction)技术

### 1.3 推论

**定理**: 每个项至多有一个范式（若存在）。

**证明**: 假设u₁和u₂都是范式且t →*u₁, t →* u₂。
由CR，存在v使得u₁ →*v, u₂ →* v。
但u₁, u₂是范式，所以u₁ = v = u₂。

## 二、强正规化定理

### 2.1 定理陈述

**强正规化 (SN)**: 每个良类型的λ项都终止于范式（无无限归约序列）。

**形式**: ∀t:Γ⊢t:T, 不存在无限归约链 t → t₁ → t₂ → ...

### 2.2 证明方法

**Reducibility Candidates** (Girard):

1. 定义可约性谓词 Red_T
2. 对所有类型T，证明Red_T的性质
3. 证明每个良类型项属于对应可约性集合

**定义**:

- Red_σ = {t | t强正规化}
- Red_{σ→τ} = {t | ∀s∈Red_σ, t s ∈ Red_τ}

### 2.3 类型推断与终止

**定理**: 简单类型λ演算中，所有良类型项都是强正规化的。

**注意**: System F也是强正规化的，但System Fω不是。

## 三、Curry-Howard对应

### 3.1 命题即类型，证明即程序

**对应表**:

| 逻辑 | 类型论 | λ演算 |
|------|--------|-------|
| 命题 | 类型 | - |
| 证明 | 项/程序 | λ项 |
| 蕴涵 A→B | 函数类型 A→B | λ抽象与应用 |
| 合取 A∧B | 积类型 A×B | 对构造器 |
| 析取 A∨B | 和类型 A+B | 和构造器 |
| 真 | 单元类型 ⊤ | () |
| 假 | 空类型 ⊥ | - |
| 归一化 | 计算 | β归约 |

### 3.2 证明规范化

**定理**: 证明规范化对应于λ项的β归约。

**切割消去**: 对应于函数应用。

**例子**:

**逻辑证明**:

```
  A    A→B
  --------
      B
```

**类型论**:

```
  f : A→B    x : A
  -----------------
      f x : B
```

### 3.3 应用

**证明提取**: 从存在性证明提取计算内容。

**程序验证**: 类型即规范，证明即证据。

## 四、Univalence公理

### 4.1 定理陈述

**Univalence公理** (Voevodsky):

$$(A =_U B) \simeq (A \simeq B)$$

其中U是宇宙，A ≃ B表示等价（具有准逆的函数）。

### 4.2 意义

**结构不变性**: 同构的结构在类型论中可互换。

**函数外延性**: 点wise相等的函数相等。

**证明相关相等**: 相等性证明自身携带结构信息。

### 4.3 同伦层次

**定义**:

- **命题** (-1-截断): 所有元素相等的类型
- **集合** (0-截断): 相等性是命题的类型
- **群胚** (1-截断): 相等性相等性可互换

**IsTrunc n A**: 类型A是n-截断的。

### 4.4 立方类型论中的计算性

**问题**: HoTT中Univalence是公理，无计算规则。

**立方类型论解决方案**:

- Path类型替代Id类型
- 内部化路径概念
- 使Univalence可计算

## 五、构造演算 (Calculus of Constructions)

### 5.1 纯构造演算

**类型层级**:

```

* : □

```

**规则**:

- (*,*, *): 项级依赖类型 (Π)
- (□, *,*): 多态 (System F)
- (□, □, □): 类型构造器
- (*, □, □): 类型算子

### 5.2 构造归纳演算 (CIC)

**扩展**:

- 归纳类型
- 互归纳类型
- 归纳-递归定义

**Coq基础**: CIC是Coq证明助手的理论基础。

### 5.3 元理论性质

**一致性**: CIC是一致的（相对强假设）。

**强正规化**: 所有良类型项强正规化。

**表达能力**: 可表达大多数数学结构。

## 六、归纳类型的范畴语义

### 6.1 初始代数语义

**自然数**:

在范畴C中，自然数是初始代数的载体：

$$1 + N \xrightarrow{[zero, succ]} N$$

**归纳原理**: 从初始代数到任意代数的唯一同态。

### 6.2 最终余代数

**共归纳类型**: 最终余代数给出无限结构。

**流 (Stream)**:

$$S \xrightarrow{(head, tail)} A \times S$$

### 6.3 融合规则

**foldr-build融合**:

```haskell
foldr f z (build g) = g f z
```

**类型论意义**: 计算优化保持语义等价。
