# 类型论实战问题

## 一、类型推导算法

### 问题描述

实现Hindley-Milner类型推导算法，为λ项自动推断最一般类型。

### 算法步骤

**输入**: 无类型λ项

**输出**: 最一般类型（或类型错误）

**核心步骤**:

1. **分配类型变量**: 为每个子项分配新鲜类型变量

2. **生成约束**: 根据语法结构生成类型等式约束

   ```

   若 t = λx.e, 则 t: α→β, 其中x:α, e:β

   若 t = f a,  则 f: α→β, a:α, t:β

   ```

3. **统一求解**: 使用Robinson统一算法求解约束

**Robinson统一算法**:

```python

def unify(t1, t2):

    if t1 == t2: return {}

    if is_var(t1): return {t1: t2} if not occurs(t1, t2) else fail

    if is_var(t2): return {t2: t1} if not occurs(t2, t1) else fail

    if is_app(t1) and is_app(t2):

        s1 = unify(fun(t1), fun(t2))

        s2 = unify(s1(arg(t1)), s1(arg(t2)))

        return s2 ∘ s1

    fail

```

### 实战例子

**项**: λf.λx.f (f x)

**推导**:

1. f: α, x: β
2. f x: γ, 约束 α = β→γ
3. f (f x): δ, 约束 α = γ→δ
4. 统一: α = β→γ = γ→δ
5. 解: γ = β, δ = β, α = β→β
6. 结果: (β→β)→β→β

### let多态

**关键**: let绑定允许类型变量的一般化。

```

let id = λx.x in (id 1, id true)

```

id被赋予多态类型 ∀α.α→α。

## 二、向量安全操作实现

### 问题描述

利用依赖类型实现安全的向量操作，排除越界访问。

### 向量类型定义

```

inductive Vec (A:Type) : ℕ → Type where

  | nil : Vec A 0

  | cons : ∀n, A → Vec A n → Vec A (n+1)

```

### 安全操作实现

**head** (要求非空):

```

head : ∀A n, Vec A (n+1) → A

head A n (cons _ x _) = x

```

**tail**:

```

 tail : ∀A n, Vec A (n+1) → Vec A n

 tail A n (cons _ _ xs) = xs

```

**连接**:

```

append : ∀A m n, Vec A m → Vec A n → Vec A (m+n)

append A 0 n nil ys = ys

append A (m+1) n (cons _ x xs) ys =

  cons A (m+n) x (append A m n xs ys)

```

**查找** (有限索引):

```

lookup : ∀A n, Vec A n → Fin n → A

lookup A (n+1) (cons _ x _) (zero _) = x

lookup A (n+1) (cons _ _ xs) (succ _ i) = lookup A n xs i

```

**Fin n** 确保索引在范围[0, n-1]内。

### 实战练习

实现向量反转：

```

reverse : ∀A n, Vec A n → Vec A n

```

**提示**: 使用辅助函数和累积参数。

### 红黑树不变式

**数据类型**:

```

inductive Color : Type where

  | Red | Black

inductive RBTree (A:Type) : ℕ → Type where

  | Leaf : RBTree A 1

  | Node : ∀n, Color → RBTree A n → A → RBTree A n →

           RBTree A (if c == Red then n else n+1)

```

**不变式**:

1. 红黑性质: 红节点的子节点是黑的
2. 黑高度: 所有路径黑节点数相同
3. 有序性: 二叉搜索树性质

## 四、同伦类型论证明

### 问题描述

利用HoTT证明关于等价的定理。

### 基本证明技巧

**路径归纳** (J):

```

path_induction : ∀(A:Type)(C:∀(y:A), x=y → Type),

  C(x, refl_x) → ∀(y:A)(p:x=y), C(y, p)

```

**应用**: 证明所有基于路径的性质。

### Univalence应用

**结构传输**:

若我们知道关于群G的性质，且G ≅ H，则通过Univalence，
相同性质对H成立。

**形式化**:

```

transport : ∀(A B:Type)(P:Type → Type),

  A = B → P A → P B

```

结合Univalence:

```

equiv_transport : ∀(A B:Type)(P:Type → Type),

  A ≃ B → P A → P B

```

### 实战：基本群计算

**圆的基本群**: π₁(S¹) ≅ ℤ

**证明思路**:

1. 定义覆盖空间 R → S¹
2. 证明提升性质
3. 建立ℤ与路径的对应

## 五、归纳-递归定义

### 问题描述

理解并应用归纳-递归定义（如宇宙层次）。

### 类型宇宙

**定义**:

```

mutual

  inductive U : Type where

    | nat : U

    | pi : ∀(a:U), (T a → U) → U

  recursive T : U → Type where

    | T nat = ℕ

    | T (pi a b) = ∀(x:T a), T (b x)

```

**性质**: U同时是归纳定义，T在U上递归定义。

### 应用：正确编译器

**目标**: 验证编译器保持程序语义。

**方法**:

1. 源语言类型 Src
2. 目标语言类型 Tgt
3. 编译函数 compile : Src → Tgt
4. 证明语义保持: ⟦s⟧ = ⟦compile s⟧
