# 类型论基础概念

## 一、λ演算

### 1.1 无类型λ演算

**语法**:

```

t ::= x        (变量)

    | λx.t     (抽象)

    | t t      (应用)

```

**β归约**: (λx.t) s → t[x := s]

**η归约**: λx.t x → t（若x不在t中自由）

### 1.2 简单类型λ演算 (λ→)

**类型语法**:

```

τ ::= A        (基类型)

    | τ → τ    (函数类型)

```

**类型判断**: Γ ⊢ t : τ

**规则**:

```

Γ, x:τ ⊢ x:τ          (Var)

Γ, x:σ ⊢ t:τ
-----------------     (→I)
Γ ⊢ λx.t: σ→τ

Γ ⊢ t:σ→τ    Γ ⊢ s:σ
---------------------  (→E)
Γ ⊢ t s:τ

```

### 1.3 多态λ演算 (System F)

**类型量化**: ∀X.τ

**语法扩展**:

```

t ::= ... 

    | ΛX.t     (类型抽象)

    | t [τ]    (类型应用)

```

**规则**:

```

Γ ⊢ t:τ
----------------  (∀I)
Γ ⊢ ΛX.t:∀X.τ

Γ ⊢ t:∀X.τ
--------------    (∀E)
Γ ⊢ t[σ]:τ[X:=σ]

```

**性质**: System F可表达任何递归可计算函数，但类型推断不可判定。

## 二、依赖类型

### 2.1 Π类型（依赖函数类型）

**定义**: Π(x:A).B(x) 表示从x:A到B(x)的函数。

**语法**: (x:A) → B x 或 ∀(x:A), B x

**例子**:
- 向量长度: vec : Π(n:ℕ).Type
- 恒等函数: id : Π(A:Type).A → A

### 2.2 Σ类型（依赖对类型）

**定义**: Σ(x:A).B(x) 表示对(a, b)其中a:A, b:B(a)。

**语法**: (x:A) × B x 或 Σ(x:A), B x

**投影**:
- π₁ : Σ(x:A).B(x) → A
- π₂ : (p:Σ(x:A).B(x)) → B(π₁(p))

**例子**: 偶数类型 Σ(n:ℕ).(n mod 2 = 0)

### 2.3 归纳类型

**自然数**:

```

inductive ℕ : Type where

  | zero : ℕ

  | succ : ℕ → ℕ

```

**归纳原理**:

```

nat_ind : ∀(P:ℕ → Type),

  P zero →

  (∀n, P n → P (succ n)) →

  ∀n, P n

```

**列表**:

```

inductive List (A:Type) : Type where

  | nil : List A

  | cons : A → List A → List A

```

## 三、Martin-Löf类型论

### 3.1 判断形式

1. **类型判断**: A type（A是良型类型）
2. **成员判断**: a : A（a是A的元素）
3. **等式判断**: a ≡ b : A（a和b在A中等式）

### 3.2 类型构造器

| 类型 | 构造 | 消除 | 计算 |
|------|------|------|------|
| Π | λ | 应用 | (λx.t) a ≡ t[x:=a] |
| Σ | 对 | 投影 | π₁(a,b) ≡ a, π₂(a,b) ≡ b |
| + | inl, inr | case | case(inl a, f, g) ≡ f a |
| = | refl | J/transport | J(refl, d) ≡ d |

### 3.3 命题即类型

**对应**:

| 逻辑 | 类型 | 解释 |
|------|------|------|
| 真 | ⊤ (单元类型) | 可证 |
| 假 | ⊥ (空类型) | 不可证 |
| A ∧ B | A × B | 对 |
| A ∨ B | A + B | 和 |
| A → B | A → B | 函数 |
| ∀x:A.P(x) | Π(x:A).P(x) | 依赖函数 |
| ∃x:A.P(x) | Σ(x:A).P(x) | 依赖对 |
| x = y | Id_A(x,y) | 同一性类型 |

## 四、归纳族与索引类型

### 4.1 向量类型

```

inductive Vec (A:Type) : ℕ → Type where

  | nil : Vec A 0

  | cons : ∀n, A → Vec A n → Vec A (n+1)

```

**性质**: 类型中携带长度信息，保证安全操作。

**安全head**:

```

head : ∀A n, Vec A (n+1) → A

head A n (cons _ x _) = x

-- nil情况被类型系统排除！

```

### 4.2 有限类型

```

inductive Fin : ℕ → Type where

  | zero : ∀n, Fin (n+1)

  | succ : ∀n, Fin n → Fin (n+1)

```

**应用**: 安全数组索引

```

lookup : ∀A n, Vec A n → Fin n → A

```

## 五、同伦类型论 (HoTT)

### 5.1 同一性类型

**定义**: Id_A(a, b) 或 a =_A b 表示a和b的同一性证明。

**refl**: refl_a : a = a

**路径归纳** (J):

```

J : ∀(C:∀(y:A), x=y → Type),

  C(x, refl_x) →

  ∀(y:A)(p:x=y), C(y, p)

```

### 5.2 路径与等价

**路径运算**:
- 复合: p · q
- 逆: p⁻¹
- 恒等: refl

**同伦**: f ~ g 表示 ∀x, f(x) = g(x)

**等价**: A ≃ B 表示存在f:A→B具有准逆

### 5.3 Univalence公理

**公理** (Voevodsky):

$$(A = B) \simeq (A \simeq B)$$

**意义**: 同构类型可互换。

**推论**: 
- 结构不变性
- 函数外延性
- 同伦层次的定义

### 5.4 高阶归纳类型 (HIT)

**圆 S¹**:

```

inductive S¹ : Type where

  | base : S¹

  | loop : base = base

```

**区间 I**:

```

inductive I : Type where

  | zero : I

  | one : I

  | seg : zero = one

```

**商类型**:

```

inductive Quotient (A:Type) (R:A→A→Type) : Type where

  | proj : A → Quotient A R

  | eq : ∀(a b:A), R a b → proj a = proj b

```

## 六、立方类型论

### 6.1 路径抽象

**语法**: Path A a b 表示从a到b的路径类型。

**特征**:
- 内部化路径
- 直接处理边界条件
- 计算性更好的Univalence

### 6.2 合成运算

**transp**: 在类型族上的运输。

**hcomp**: 高维合成。

**意义**: 使路径填充可计算。

## 七、Martin-Löf类型论中的判断形式详解

### 7.1 四种判断

Martin-Löf类型论区分以下判断（judgement）：
1. **$\Gamma \vdash A\ \text{type}$**：在上下文 $\Gamma$ 中，$A$ 是良形的类型。
2. **$\Gamma \vdash a : A$**：项 $a$ 属于类型 $A$。
3. **$\Gamma \vdash a \equiv b : A$**：$a$ 与 $b$ 在类型 $A$ 上**定义相等**（计算相等）。
4. **$\Gamma \vdash A \equiv B\ \text{type}$**：类型 $A$ 与 $B$ 定义相等。

**与命题逻辑的区别**：在传统逻辑中，$A$ 本身是一个命题（可证或否）；在类型论中，$A$ 是一个类型，其"真"对应于存在元素 $a:A$（ inhabitation ）。

### 7.2 依赖类型的编程实例：安全向量

```
inductive Vec (A : Type) : ℕ → Type where
  | nil  : Vec A 0
  | cons : ∀ n, A → Vec A n → Vec A (n+1)
```

**安全 head 操作**：
```
head : ∀ A n, Vec A (n+1) → A
head A n (cons _ x _) = x
```
由于输入类型要求长度至少为1（$n+1$），`head` 在 `nil` 上无定义分支——类型系统静态排除了空向量取首元素的错误。

## 八、同一性类型的群胚结构

### 8.1 路径运算满足群胚律

对类型 $A$ 和路径 $p : a =_A b$, $q : b =_A c$, $r : c =_A d$：
- **结合律**：$(p \cdot q) \cdot r = p \cdot (q \cdot r)$
- **单位律**：$p \cdot \text{refl}_b = p$ 且 $\text{refl}_a \cdot p = p$
- **逆律**：$p \cdot p^{-1} = \text{refl}_a$ 且 $p^{-1} \cdot p = \text{refl}_b$

**证明**：使用路径归纳（J-eliminator）。例如对 $p$ 进行路径归纳，将 $p$ 化为 $\text{refl}_a$，此时 $p^{-1} = \text{refl}_a$，等式化为 $\text{refl}_a \cdot \text{refl}_a = \text{refl}_a$，由路径复合的定义直接成立。

### 8.2 高阶路径与∞-群胚

对 $p, q : a =_A b$，存在 $p =_{a=b} q$（路径之间的路径）。继续迭代得到高阶同伦结构。在HoTT中，这正是**弱∞-群胚**的代数结构。

## 九、高阶归纳类型 S¹ 的详细分析

### 9.1 圆的归纳原理

```
inductive S¹ : Type where
  | base : S¹
  | loop : base = base
```

**归纳原理**：给定 $P : S¹ \to \text{Type}$，要构造 $f : \prod_{x:S¹} P(x)$，需：
- 基点：$b : P(\text{base})$
- 路径：$\ell : \text{transport}^P(\text{loop}, b) =_{P(\text{base})} b$

### 9.2 计算基本群 $\pi_1(S¹) \cong \mathbb{Z}$

通过Univalence公理和覆盖空间理论，在HoTT中可证明：
$$\Omega(S¹, \text{base}) := (\text{base} =_{S¹} \text{base}) \simeq \mathbb{Z}$$

其中整数 $n$ 对应 loop 的 $n$ 次复合（$n > 0$ 为正方向迭代，$n < 0$ 为逆方向）。这给出了纯类型论/同伦论方法计算基本群的典范例子。

## 十、归纳类型的递归原理与证明辅助器实践

### 10.1 递归原理的推导

在Martin-Löf类型论中，每个归纳类型都配备其**归纳原理**（或称**递归原理**、**消去规则**）。以自然数为例：

```
 nat_rec : ∀(P:ℕ → Type), P 0 → (∀n, P n → P (succ n)) → ∀n, P n
```

这恰好是数学归纳法的类型论形式：要证明所有自然数满足性质 $P$，只需证明 $P(0)$ 成立，并证明若 $P(n)$ 成立则 $P(n+1)$ 成立。

在Coq/Lean等证明辅助器中，`match` 表达式和 `induction` 策略均基于此原理。区别仅在于：`match` 对应于非依赖的原始递归（`nat_rec`），而 `induction` 对应于依赖的归纳原理（`nat_ind`）。

### 10.2 Π类型与逻辑∀的完全对应

在依赖类型中，命题 $\forall x:A, B(x)$ 不仅断言"对所有 $x$ 有 $B(x)$"，而且要求提供一个**证明函数** $f: \prod_{x:A} B(x)$，使得对任意给定的 $x$，$f(x)$ 是 $B(x)$ 的证明。这与经典逻辑中"∀作为无限合取"的语义不同：在类型论中，∀证明必须是**构造性的、可计算的**。
