## 14.2.1 引言 / Introduction

光滑流形（smooth manifold）是现代微分几何的基石概念。1854年，黎曼（Bernhard Riemann）在其著名的就职演讲《关于几何基础的假设》（*Über die Hypothesen, welche der Geometrie zu Grunde liegen*）中首次提出了$n$维流形的直观思想：一个局部类似于欧几里得空间$\mathbb{R}^n$的连续对象。经过一个多世纪的发展，特别是惠特尼（Hassler Whitney）在1936年对流形抽象定义的严格化，以及斯泰恩罗德（Norman Steenrod）与嘉当（Élie Cartan）在纤维丛与活动标架理论上的贡献，光滑流形理论已成为连接拓扑学、分析学与代数学的枢纽。

Princeton MAT 355（Differential Geometry）课程在前三周系统构建了光滑流形的理论基础：从拓扑流形的定义出发，经过坐标卡图册（atlas）的构造，建立光滑结构（smooth structure），进而研究光滑映射、浸入（immersion）、嵌入（embedding）与淹没（submersion），最终抵达惠特尼嵌入定理（Whitney Embedding Theorem）与萨德定理（Sard's Theorem）的深度结果。本文档严格对齐该课程进度，并综合Lee（2012）《Introduction to Smooth Manifolds》第一章至第六章的核心内容，为后续切空间、向量场、微分形式与黎曼几何的学习奠定坚实基础。

从应用视角看，光滑流形理论不仅是广义相对论中时空模型的数学语言，也是规范场论、机器学习中的流形学习（manifold learning）以及机器人运动规划的基础框架。理解流形的局部-整体性质，是掌握现代几何分析的第一步。

## 14.2.2 流形的定义与例子 / Definition and Examples of Manifolds

### 拓扑流形 / Topological Manifolds

**定义 2.1**（拓扑流形）. 一个**拓扑流形**（topological manifold）$M$是指满足以下条件的拓扑空间：

1. **Hausdorff性**：$M$是Hausdorff空间，即任意两个不同点存在不相交的邻域；
2. **第二可数性**：$M$具有可数的拓扑基；
3. **局部欧几里得性**：对任意$p \\in M$，存在$p$的邻域$U \\subset M$同胚于$\mathbb{R}^n$的一个开子集。整数$n$称为流形的**维数**（dimension）。

上述三个条件中，Hausdorff性与第二可数性排除了某些病态空间（如长线long line），保证了流形上可以进行标准的分析操作；局部欧几里得性则是流形名称的来源——“ manifold ”一词本身即意味着“多样的统一体”，强调其局部平凡但整体可能非常复杂的拓扑结构。

**例子 2.2**（基本例子）.
- **欧几里得空间**：$\mathbb{R}^n$自身是$n$维拓扑流形，平凡图册仅含一个坐标卡$(\mathbb{R}^n, \mathrm{id})$。
- **单位球面**：$S^n = \\{x \\in \\mathbb{R}^{n+1} : \\|x\\| = 1\\}$是$n$维紧致拓扑流形。以$S^2$为例，可用球极投影（stereographic projection）构造两个坐标卡覆盖整个球面。

- **环面**：$n$维环面$T^n = S^1 \\times \\cdots \\times S^1$（$n$个因子）是$n$维拓扑流形，直观上可视为$\mathbb{R}^n$模去整数格$\mathbb{Z}^n$的商空间。

**定理 2.3**（维数不变性）. 连通的拓扑流形具有唯一的维数。即若$M$连通且局部同胚于$\mathbb{R}^n$又局部同胚于$\mathbb{R}^m$，则$n = m$。

*证明思路*：这是布劳威尔（Brouwer）不变域定理（Invariance of Domain）的推论。若存在开集$U \\subset \\mathbb{R}^n$与$V \\subset \\mathbb{R}^m$同胚，则不变域定理要求$n = m$。拓扑流形的维数不变性保证了后续所有局部计算中维数的一致性。

### 光滑图册与光滑结构 / Smooth Atlases and Smooth Structures

拓扑流形仅具有连续结构。为了进行微分运算，必须在流形上引入**光滑结构**。

**定义 2.4**（坐标卡与图册）. 设$M$为$n$维拓扑流形。$M$上的一个**坐标卡**（chart）是指二元组$(U, \\varphi)$，其中$U \\subset M$为开集，$\\varphi: U \\to \\widetilde{U} \\subset \\mathbb{R}^n$为同胚映射。$\\varphi$称为**局部坐标映射**，其分量$(x^1, \\ldots, x^n)$称为$U$上的**局部坐标**。$M$上的一族坐标卡$\\mathcal{A} = \\{(U_\\alpha, \\varphi_\\alpha)\\}_{\\alpha \\in A}$若满足$\\bigcup_\\alpha U_\\alpha = M$，则称为一个**图册**（atlas）。

**定义 2.5**（光滑相容性）. 设$(U, \\varphi)$与$(V, \\psi)$为$M$上的两个坐标卡。称它们是**光滑相容的**（smoothly compatible），若转移映射（transition map）
\\[
\\psi \\circ \\varphi^{-1}: \\varphi(U \\cap V) \\to \\psi(U \\cap V)
\\]
是$\\mathbb{R}^n$中开子集之间的光滑（$C^\\infty$）微分同胚。注意，当$U \\cap V = \\emptyset$时，两坐标卡自动光滑相容。

**定义 2.6**（光滑图册）. $M$上的图册$\\mathcal{A}$称为**光滑图册**（smooth atlas），若其中任意两个坐标卡都光滑相容。

**定义 2.7**（光滑结构）. $M$上的一个**光滑结构**是指一个极大的光滑图册$\\mathcal{A}$，即对任意与$\\mathcal{A}$中所有坐标卡都光滑相容的坐标卡$(U, \\varphi)$，必有$(U, \\varphi) \\in \\mathcal{A}$。配备了光滑结构的拓扑流形称为**光滑流形**（smooth manifold），记为$(M, \\mathcal{A})$或简记为$M$。

*注记*：由Zorn引理，任意光滑图册都唯一确定一个极大光滑图册。因此，实际构造中只需验证图册的光滑性，再取其极大化即可。

**例子 2.8**（$S^n$的标准光滑结构）. 考虑$S^n \\subset \\mathbb{R}^{n+1}$。定义北半球与南半球的球极投影坐标卡：
\\[
U_N = S^n \\setminus \\{(0, \\ldots, 0, -1)\\}, \\quad \\varphi_N(x^1, \\ldots, x^{n+1}) = \\left( \\frac{x^1}{1+x^{n+1}}, \\ldots, \\frac{x^n}{1+x^{n+1}} \\right),
\\]
\\[
U_S = S^n \\setminus \\{(0, \\ldots, 0, 1)\\}, \\quad \\varphi_S(x^1, \\ldots, x^{n+1}) = \\left( \\frac{x^1}{1-x^{n+1}}, \\ldots, \\frac{x^n}{1-x^{n+1}} \\right).
\\]
在交集$U_N \\cap U_S$上，转移映射为
\\[
\\varphi_S \\circ \\varphi_N^{-1}(y) = \\frac{y}{\\|y\\|^2}, \\quad y \\in \\mathbb{R}^n \\setminus \\{0\\},

\\]
这是$\\mathbb{R}^n \\setminus \\{0\\}$到自身的反演映射，显然是光滑的。因此$\\{(U_N, \\varphi_N), (U_S, \\varphi_S)\\}$构成$S^n$的一个光滑图册，从而确定了$S^n$上的标准光滑结构。

**例子 2.9**（实射影空间$\\mathbb{R}P^n$）. $\\mathbb{R}P^n$是$\\mathbb{R}^{n+1} \\setminus \\{0\\}$中过原点的直线的集合，可等价地定义为$S^n$在对径映射$x \\sim -x$下的商空间。对$i = 0, \\ldots, n$，定义
\\[
U_i = \\{[x^0 : \\cdots : x^n] \\in \\mathbb{R}P^n : x^i \\neq 0\\},
\\]
\\[
\\varphi_i([x^0 : \\cdots : x^n]) = \\left( \\frac{x^0}{x^i}, \\ldots, \\widehat{\\frac{x^i}{x^i}}, \\ldots, \\frac{x^n}{x^i} \\right) \\in \\mathbb{R}^n.
\\]
可以验证转移映射$\\varphi_j \\circ \\varphi_i^{-1}$是$\\mathbb{R}^n$中开子集之间的有理函数，因此是光滑的。这给出了$\\mathbb{R}P^n$上的标准光滑结构，使其成为$n$维紧致光滑连通流形。

**注记 2.10**（Milnor怪球面）. 米尔诺（John Milnor）在1956年发现了$S^7$上存在与标准光滑结构不同胚的光滑结构，这类流形称为**怪球面**（exotic spheres）。米尔诺因此获得1962年菲尔兹奖。怪球面的存在表明：拓扑流形上的光滑结构未必唯一。然而，对维数$n \\leq 3$，光滑结构是唯一的；对$S^n$，$n \\neq 4$时是否唯一结构的问题已解决，而$\\mathbb{R}^4$上甚至存在不可数多个互不同胚的光滑结构（Donaldson, Freedman）。

## 14.2.3 光滑结构与光滑映射 / Smooth Structures and Smooth Maps

### 光滑映射 / Smooth Maps

有了光滑流形的定义，便可定义流形之间的光滑映射，这是将欧氏空间中的微积分推广到流形上的关键步骤。

**定义 3.1**（光滑映射）. 设$M, N$为光滑流形，$F: M \\to N$为连续映射。称$F$是**光滑映射**（smooth map），若对任意$p \\in M$，存在$M$上包含$p$的坐标卡$(U, \\varphi)$和$N$上包含$F(p)$的坐标卡$(V, \\psi)$，使得$F(U) \\subset V$且坐标表示
\\[
\\widehat{F} = \\psi \\circ F \\circ \\varphi^{-1}: \\varphi(U) \\to \\psi(V) \\subset \\mathbb{R}^m
\\]
是欧氏空间之间的光滑映射。这里$\\dim M = n$，$\\dim N = m$。

上述定义的本质是：映射$F$的光滑性完全由其在局部坐标下的表示$\\widehat{F}$的光滑性决定，而转移映射的光滑相容性保证了这一定义不依赖于坐标卡的选择。

**命题 3.2**（光滑映射的复合）. 若$F: M \\to N$与$G: N \\to P$均为光滑映射，则$G \\circ F: M \\to P$也是光滑映射。

*证明思路*：取适当的坐标卡，复合映射的坐标表示为$(\\widehat{G \\circ F}) = \\widehat{G} \\circ \\widehat{F}$，而欧氏空间之间光滑映射的复合仍是光滑的。由坐标卡选取的独立性，结论得证。$\\square$

**定义 3.3**（微分同胚）. 若$F: M \\to N$是光滑双射，且其逆映射$F^{-1}: N \\to M$也是光滑的，则称$F$为**微分同胚**（diffeomorphism）。此时称$M$与$N$**微分同胚**。

微分同胚是光滑流形范畴中的同构概念。两个微分同胚的流形在光滑结构的意义下被视为“相同”。

**例子 3.4**.
- 任何坐标卡映射$\\varphi: U \\to \\widetilde{U} \\subset \\mathbb{R}^n$都是$U$（作为$M$的开子流形）到$\\widetilde{U}$的微分同胚。
- 反演映射$F: \\mathbb{R}^n \\setminus \\{0\\} \\to \\mathbb{R}^n \\setminus \\{0\\}$，$F(x) = x / \\|x\\|^2$是微分同胚，其逆映射即自身。

### 浸入、淹没与嵌入 / Immersions, Submersions, and Embeddings

光滑映射的局部行为由其微分（在后续文档中严格定义）的秩决定。这里先给出基于秩的三种基本映射类型的定义。

**定义 3.5**（浸入与淹没）. 设$F: M \\to N$为光滑映射，$\\dim M = m$，$\\dim N = n$。
- 若$F$在每点$p \\in M$处的微分$dF_p$的秩恒为$m$（即满秩），则称$F$为**浸入**（immersion）；
- 若$dF_p$的秩恒为$n$，则称$F$为**淹没**（submersion）。

由定义，浸入要求$m \\leq n$，淹没要求$m \\geq n$。

**定理 3.6**（秩定理 / Rank Theorem）. 设$F: M \\to N$为光滑映射，且在某点$p \\in M$的邻域内微分$dF$的秩恒为常数$r$。则存在$p$附近的坐标卡$(U, \\varphi)$和$F(p)$附近的坐标卡$(V, \\psi)$，使得$F$的局部坐标表示为
\\[
\\widehat{F}(x^1, \\ldots, x^m) = (x^1, \\ldots, x^r, 0, \\ldots, 0).
\\]

*证明思路*：这是反函数定理（Inverse Function Theorem）在流形上的推广。通过适当选取坐标卡，将问题约化到欧氏空间中的常秩映射，再利用反函数定理进行坐标变换。详细证明可见Lee（2012）定理4.12。$\\square$

**推论 3.7**.
- 若$F$是浸入，则局部上$F$相当于标准包含映射$\\mathbb{R}^m \\hookrightarrow \\mathbb{R}^n$，$(x^1, \\ldots, x^m) \\mapsto (x^1, \\ldots, x^m, 0, \\ldots, 0)$；
- 若$F$是淹没，则局部上$F$相当于标准投影映射$\\mathbb{R}^m \\to \\mathbb{R}^n$，$(x^1, \\ldots, x^m) \\mapsto (x^1, \\ldots, x^n)$。

**定义 3.8**（嵌入）. 若光滑映射$F: M \\to N$满足：
1. $F$是浸入；
2. $F$是$M$到$F(M) \\subset N$上的同胚（这里$F(M)$取$N$诱导的子空间拓扑），
则称$F$为**嵌入**（embedding），$F(M)$称为$N$的**嵌入子流形**（embedded submanifold）。

浸入与嵌入的关键区别在于拓扑条件：浸入仅要求局部是单射（微分满秩），而嵌入还要求整体是到像集上的同胚。经典的反例是“八字形浸入”$F: S^1 \\to \\mathbb{R}^2$，它在某点自交，因此是浸入但不是嵌入。

**定理 3.9**（正则水平集定理）. 设$F: M \\to N$为光滑淹没，$c \\in N$。则水平集$F^{-1}(c)$是$M$的维数为$\\dim M - \\dim N$的嵌入子流形。

*证明思路*：由秩定理，在$p \\in F^{-1}(c)$附近，$F$的局部坐标表示为标准投影。因此$F^{-1}(c)$局部上等于前$\\dim N$个坐标取固定值、后$\\dim M - \\dim N$个坐标自由变动的子集，这正是嵌入子流形的局部定义。$\\square$

**例子 3.10**.
- $S^n$可视为$F: \\mathbb{R}^{n+1} \\to \\mathbb{R}$，$F(x) = \\|x\\|^2$的正则水平集$F^{-1}(1)$。由于$dF_x = 2x^T$在$x \\neq 0$处秩为1，$F$在$\\mathbb{R}^{n+1} \\setminus \\{0\\}$上是淹没，故$S^n$是$\\mathbb{R}^{n+1}$的$n$维嵌入子流形。

- 正交群$O(n) = \\{A \\in M_n(\\mathbb{R}) : A^T A = I_n\\}$可视为$\\frac{n(n+1)}{2}$个约束方程$F(A) = A^T A - I_n = 0$的水平集。可以验证$0$是$F$的正则值，因此$O(n)$是$M_n(\\mathbb{R}) \\cong \\mathbb{R}^{n^2}$中的$\\frac{n(n-1)}{2}$维嵌入子流形，且是紧致的李群。

### 惠特尼嵌入定理 / Whitney Embedding Theorem

惠特尼嵌入定理是光滑流形理论中最深刻的结果之一，它断言任何光滑流形都可以作为欧氏空间的嵌入子流形来实现。

**定理 3.11**（惠特尼嵌入定理，1936）. 设$M$为$n$维光滑流形。则存在光滑嵌入$F: M \\hookrightarrow \\mathbb{R}^{2n}$。若$M$紧致，则$F$可取为到$\\mathbb{R}^{2n+1}$的嵌入。

*证明思路*（概要）：证明分为三个主要步骤，这是Princeton MAT 355课程第三周的核心内容，详见Lee（2012）第六章。

**第一步：嵌入到高维欧氏空间**。利用单位分解定理（见下节），可以将$M$分解为可数个坐标卡的并。在每个坐标卡上构造到$\\mathbb{R}^n$的光滑映射，再通过适当的截断函数（bump function）将其光滑地延拓到整个$M$。将这些映射拼接起来，可构造一个到某个高维欧氏空间$\\mathbb{R}^N$（$N$可能非常大）的光滑嵌入$F_0: M \\hookrightarrow \\mathbb{R}^N$。

**第二步：降维**。我们需要将$F_0$的像逐步投影到低维子空间，同时保持嵌入性质。考虑沿某个方向$v \\in S^{N-1}$将$\\mathbb{R}^N$投影到$v$的正交补空间$\\mathbb{R}^{N-1}$上。若投影$\\pi_v \\circ F_0$不是单射，则存在$p \\neq q$使得$F_0(p) - F_0(q)$平行于$v$。通过分析所有这类"坏"方向构成的集合，可以证明：当$N > 2n + 1$时，几乎所有的方向$v$都能给出$M$到$\\mathbb{R}^{N-1}$的嵌入。重复此过程，最终可将$M$嵌入到$\\mathbb{R}^{2n+1}$。

**第三步：降至$\\mathbb{R}^{2n}$**。要将紧致流形嵌入到$\\mathbb{R}^{2n}$，需要更精细的分析。惠特尼证明了通过"扭结"（whitney trick）可以消除自交点。对于非紧致流形，结论同样成立，但证明更为复杂。

惠特尼嵌入定理的重要意义在于：它允许我们将抽象的光滑流形视为欧氏空间中具有明确几何形象的子集，从而将抽象的微分几何问题转化为更具直观性的子流形问题。需要注意的是，$2n$这一维数通常不是最优的。例如，$S^n$可以自然嵌入到$\\mathbb{R}^{n+1}$中。但对于一般流形，$2n$是最优的普适上界：存在不能嵌入到$\\mathbb{R}^{2n-1}$中的$n$维流形（如实射影平面$\\mathbb{R}P^2$不能嵌入$\\mathbb{R}^3$，但可以嵌入$\\mathbb{R}^4$）。

## 14.2.4 单位分解与嵌入子流形 / Partitions of Unity and Embedded Submanifolds

### 单位分解定理 / Partitions of Unity

单位分解是将局部构造拼接为整体构造的核心工具，在微分几何中具有不可替代的作用。

**定义 4.1**（支撑的闭包）. 设$f: M \\to \\mathbb{R}$为连续函数。$f$的**支撑**（support）定义为集合$\\mathrm{supp}(f) = \\overline{\\{p \\in M : f(p) \\neq 0\\}}$。

**定义 4.2**（鼓包函数）. 设$A \\subset M$为闭子集，$U \\supset A$为开集。一个关于$A$支撑在$U$中的**鼓包函数**（bump function）是指光滑函数$f: M \\to [0, 1]$，满足$f|_A \\equiv 1$且$\\mathrm{supp}(f) \\subset U$。

**引理 4.3**（鼓包函数的存在性）. 对光滑流形$M$的任意闭子集$A$和包含$A$的开集$U$，存在鼓包函数$f$关于$A$支撑在$U$中。

*证明思路*：先在$\\mathbb{R}^n$上构造一个光滑函数$\\rho: \\mathbb{R}^n \\to [0, 1]$，使得$\\rho$在以原点为中心的单位闭球上取值为1，在半径为2的闭球外取值为0。然后通过坐标卡将$\\rho$转移到$M$上，并利用单位分解进行拼接。$\\square$

**定理 4.4**（单位分解定理）. 设$M$为光滑流形，$\\mathcal{U} = \\{U_\\alpha\\}_{\\alpha \\in A}$为$M$的任意开覆盖。则存在一列光滑函数$\\{\\psi_i: M \\to [0, 1]\\}_{i=1}^\\infty$，满足：
1. 对每个$i$，$\\mathrm{supp}(\\psi_i)$紧致且包含于某个$U_\\alpha$中；
2. 每点$p \\in M$有一个邻域仅与有限多个$\\mathrm{supp}(\\psi_i)$相交（局部有限性）；
3. 对所有$p \\in M$，$\\sum_{i=1}^\\infty \\psi_i(p) = 1$。

这样的$\\{\\psi_i\\}$称为从属于$\\mathcal{U}$的**单位分解**（partition of unity）。

*证明思路*：利用$M$的第二可数性和局部紧致性，可以构造一个由预紧坐标卡组成的局部有限可数开覆盖$\\{V_j\\}$，以及一个加细覆盖$\\{W_j\\}$使得$\\overline{W_j} \\subset V_j$。在每个$V_j$上利用鼓包函数构造光滑函数$\\varphi_j$支撑在$V_j$中且在$\\overline{W_j}$上为正。由于覆盖是局部有限的，$\\varphi = \\sum_j \\varphi_j$在每点邻域上都是有限和，因此是良好定义的光滑正函数。最后令$\\psi_j = \\varphi_j / \\varphi$即得所需单位分解。$\\square$

单位分解的应用极为广泛：
- **黎曼度量的存在性**：在每个坐标卡上取标准欧氏度量，再用单位分解加权求和，可整体构造出$M$上的黎曼度量。
- **向量丛截面的延拓**：局部定义的截面可以通过单位分解拼接成整体截面。
- **流形上积分的定义**：积分是局部定义的，单位分解保证了整体积分的良好定义性。

### 萨德定理 / Sard's Theorem

萨德定理是微分拓扑中的基本定理，它断言光滑映射的临界值集合在像空间中测度为零。

**定义 4.5**（临界值与正则值）. 设$F: M \\to N$为光滑映射。
- 若$p \\in M$处$dF_p$不是满射，则称$p$为$F$的**临界点**（critical point）。
- $c \\in N$若满足$F^{-1}(c)$中存在临界点，则称$c$为**临界值**（critical value）；否则称$c$为**正则值**（regular value）。

注意：即使$F^{-1}(c) = \\emptyset$，$c$仍然被视为正则值。

**定理 4.6**（萨德定理）. 设$F: M \\to N$为光滑映射。则$F$的临界值集合在$N$中具有零测度（Lebesgue测度意义下）。

*证明思路*（归纳法）：这是Princeton MAT 355第三周的经典内容。对$\\dim M = n$，$\\dim N = m$进行归纳。关键步骤是：
1. 记$C$为所有临界点的集合。将$C$分解为$C \\supset C_1 \\supset C_2 \\supset \\cdots$，其中$C_k$为$F$的$k$阶退化临界点集合。
2. 证明$F(C \\setminus C_1)$测度为零：利用秩定理局部上$F$形如$(x^1, \\ldots, x^n) \\mapsto (x^1, \\ldots, x^{r}, 0, \\ldots, 0)$加上高阶项，应用Fubini定理。
3. 证明$F(C_k \\setminus C_{k+1})$测度为零：在某点处至少有一个$k+1$阶偏导数非零，再次利用坐标变换和Fubini定理。
4. 证明$F(C_k)$测度为零对充分大的$k$：利用Taylor展开和覆盖论证。

萨德定理有诸多重要推论，包括Brouwer不动点定理、横截性定理以及Whitney浸入/嵌入定理的简化证明。

## 14.2.5 计算实例与具体构造 / Computational Examples and Concrete Constructions

### 实例 1：$S^2$的显式图册构造

为了加深对光滑结构的理解，我们显式构造$S^2$的一个光滑图册并计算转移映射。

设$S^2 = \\{(x, y, z) \\in \\mathbb{R}^3 : x^2 + y^2 + z^2 = 1\\}$。考虑六个半球坐标卡：
\\[
U_x^+ = \\{(x, y, z) \\in S^2 : x > 0\\}, \\quad \\varphi_x^+(x, y, z) = (y, z),
\\]
\\[
U_x^- = \\{(x, y, z) \\in S^2 : x < 0\\}, \\quad \\varphi_x^-(x, y, z) = (y, z),
\\]
以及类似定义的$U_y^\\pm, U_z^\\pm$与对应坐标映射。

在交集$U_z^+ \\cap U_x^+$上，点满足$x > 0, z > 0$且$x^2 + y^2 + z^2 = 1$。设$(u, v) = \\varphi_z^+(x, y, z) = (x, y)$，则$\\varphi_x^+(x, y, z) = (y, z) = (v, \\sqrt{1 - u^2 - v^2})$。因此转移映射为
\\[
\\varphi_x^+ \\circ (\\varphi_z^+)^{-1}(u, v) = \\left(v, \\sqrt{1 - u^2 - v^2}\\right),
\\]
定义域为$\\{(u, v) : u > 0, v^2 < 1 - u^2\\}$，这是$\\mathbb{R}^2$中的开集。该映射显然是光滑的。类似地，可以验证所有其他转移映射也是光滑的。因此这六个坐标卡构成了$S^2$的一个光滑图册。

此图册比球极投影图册更"细碎"，但计算转移映射更为直接，常用于具体计算中。

### 实例 2：$\\mathbb{R}P^2$的同伦群计算准备

$\\mathbb{R}P^2$是2维非定向紧致流形。其标准光滑结构由齐次坐标$[x^0 : x^1 : x^2]$给出。具体地，令
\\[
U_i = \\{[x^0 : x^1 : x^2] : x^i \\neq 0\\}, \\quad i = 0, 1, 2.
\\]
坐标映射为
\\[
\\varphi_0([1 : u : v]) = (u, v), \\quad \\varphi_1([u : 1 : v]) = (u, v), \\quad \\varphi_2([u : v : 1]) = (u, v).
\\]

计算$\\varphi_1 \\circ \\varphi_0^{-1}$：对$(u, v) \\in \\varphi_0(U_0 \\cap U_1)$，有$u \\neq 0$，且
\\[
\\varphi_0^{-1}(u, v) = [1 : u : v] = [1/u : 1 : v/u],
\\]
因此
\\[
\\varphi_1 \\circ \\varphi_0^{-1}(u, v) = \\left(\\frac{1}{u}, \\frac{v}{u}\\right).
\\]
这是$\\mathbb{R}^2 \\setminus \\{u = 0\\}$到自身的有理映射，显然是光滑的。类似可验证其余转移映射。

$\\mathbb{R}P^2$还可以等价地定义为$S^2$模去对径映射$x \\sim -x$的商空间。商映射$\\pi: S^2 \\to \\mathbb{R}P^2$是光滑的2对1覆叠映射。这一构造将在拓扑学文档"基本群与覆叠空间"中深入讨论。

### 实例 3：环面$T^2$作为商流形

$T^2 = S^1 \\times S^1$可以通过$\\mathbb{R}^2$模去格点群$\\mathbb{Z}^2$的等价关系得到：
\\[
(x, y) \\sim (x', y') \\iff x - x' \\in \\mathbb{Z}, \\ y - y' \\in \\mathbb{Z}.
\\]
商映射$\\pi: \\mathbb{R}^2 \\to T^2$是局部微分同胚。具体地，对任意$p = (x_0, y_0) \\in \\mathbb{R}^2$，取$U = (x_0 - 1/2, x_0 + 1/2) \\times (y_0 - 1/2, y_0 + 1/2)$，则$\\pi|_U: U \\to \\pi(U) \\subset T^2$是微分同胚。因此$\\{(\\pi(U_p), \\pi|_{U_p}^{-1})\\}$构成了$T^2$的一个光滑图册。

这一构造自然地推广到高维环面$T^n = \\mathbb{R}^n / \\mathbb{Z}^n$，且$T^n$上继承的平坦黎曼度量（flat metric）使其成为黎曼几何中最基本的例子之一。
