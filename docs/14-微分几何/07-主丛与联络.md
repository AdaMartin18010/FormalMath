## 14.7.1 引言 / Introduction

纤维丛（fiber bundle）理论是现代微分几何与理论物理的交汇点。从拓扑学角度看，纤维丛为研究流形的整体结构提供了强有力的代数工具；从物理学角度看，规范场论（gauge theory）中的规范势、场强以及粒子相互作用都可以用主丛（principal bundle）及其联络来精确表述。电磁学中的麦克斯韦方程、弱电统一理论、量子色动力学（QCD）以及更一般的杨-米尔斯理论（Yang-Mills theory），无不以主丛为数学骨架。

Princeton MAT 355课程在第十六至十八周引入了纤维丛与主丛的基本概念，并探讨了主丛联络与流形上一般线性联络的深刻联系。Kobayashi与Nomizu（1963）的《Foundations of Differential Geometry》第一卷是该领域无可争议的经典权威，Lee（2012）第十章则提供了更为现代和易于理解的入门路径。本文档在系统介绍主丛、伴随丛、联络形式、曲率形式与和乐群的同时，通过霍普夫丛（Hopf bundle）和杨-米尔斯方程的计算实例，揭示这一抽象理论的物理与几何内涵。

主丛理论的历史发展紧密关联于嘉当（Élie Cartan）的活动标架法、斯泰恩罗德（Norman Steenrod）对纤维丛拓扑的系统化，以及杨振宁与米尔斯（Robert Mills）在1954年开创的非阿贝尔规范场论。今天，主丛联络不仅是微分几何研究生课程的标准内容，也是弦论、膜世界模型以及拓扑量子场论的基本语言。

## 14.7.2 纤维丛与主丛 / Fiber Bundles and Principal Bundles

### 纤维丛的定义 / Definition of Fiber Bundles

**定义 2.1**（纤维丛）. 一个**纤维丛**（fiber bundle）是指四元组$(E, M, \\pi, F)$，其中$E$（全空间）、$M$（底空间）、$F$（纤维）为光滑流形，$\\pi: E \\to M$为光滑满射，满足：**局部平凡性**，即对每点$p \\in M$，存在邻域$U \\subset M$和微分同胚$\\phi: \\pi^{-1}(U) \\to U \\times F$，使得$\\pi = pr_1 \\circ \\phi$。

**定义 2.2**（转移函数）. 设$\\{(U_\\alpha, \\phi_\\alpha)\\}$为纤维丛的局部平凡化覆盖。在交集$U_\\alpha \\cap U_\\beta$上，转移映射形如$(p, f) \\mapsto (p, \\tau_{\\beta\\alpha}(p)(f))$，其中$\\tau_{\\beta\\alpha}: U_\\alpha \\cap U_\\beta \\to \\mathrm{Diff}(F)$为光滑映射，称为**转移函数**。转移函数满足上闭链条件：$\\tau_{\\gamma\\beta} \\circ \\tau_{\\beta\\alpha} = \\tau_{\\gamma\\alpha}$。

### 主丛的定义 / Definition of Principal Bundles

**定义 2.3**（主丛）. 一个**主G-丛**（principal G-bundle）是指纤维丛$(P, M, \\pi, G)$，其中$G$为李群，纤维$F = G$，且局部平凡化中的转移函数取值于$G$的左乘作用。

主丛的关键特征是右$G$作用$P \\times G \\to P$，$(p, g) \\mapsto p \\cdot g = R_g(p)$，满足：
1. $\\pi(p \\cdot g) = \\pi(p)$（作用保持纤维）；
2. 该作用在每个纤维上是自由且可迁的；
3. 局部平凡化与$G$作用相容。

由这些条件，底空间$M$可自然等同于轨道空间$P / G$。

**例子 2.4**（标架丛）. 设$M$为$n$维光滑流形。$M$上所有有序基的集合
\\[
FM = \\{(p, e_1, \\ldots, e_n) : p \\in M, \\ (e_1, \\ldots, e_n) \\text{ 为 } T_p M \\text{ 的基}\\}
\\]
构成一个主$GL(n, \\mathbb{R})$-丛，称为**标架丛**。右作用为基的右乘：$(e_1, \\ldots, e_n) \\cdot A = (\\sum_j e_j A^j_1, \\ldots, \\sum_j e_j A^j_n)$。

若$M$为黎曼流形，则正交标架的集合构成主$O(n)$-丛$OM \\subset FM$，称为**正交标架丛**。

**例子 2.5**（霍普夫丛）. 著名的**霍普夫丛**$S^1 \\hookrightarrow S^3 \\xrightarrow{\\pi} S^2$是一个主$U(1)$-丛。将$S^3$视为$\\mathbb{C}^2$中的单位球面$|z_1|^2 + |z_2|^2 = 1$，$U(1)$作用为$(z_1, z_2) \\cdot e^{i\\theta} = (z_1 e^{i\\theta}, z_2 e^{i\\theta})$。商映射$\\pi: S^3 \\to S^2$由$\\pi(z_1, z_2) = [z_1 : z_2] \\in \\mathbb{C}P^1 \\cong S^2$给出。

霍普夫丛是第一个非平凡的主丛例子，其陈数（Chern number）为$-1$，在量子力学中描述自旋$1/2$粒子的Berry相位。

### 伴随丛与伴丛 / Adjoint Bundles and Associated Bundles

**定义 2.6**（伴丛）. 设$P$为主$G$-丛，$\\rho: G \\to \\mathrm{Diff}(F)$为$G$在流形$F$上的光滑左作用。定义**伴丛**为商空间
\\[
P \\times_\\rho F = (P \\times F) / G,
\\]
其中$G$对角作用为$(p, f) \\cdot g = (pg, \\rho(g^{-1})f)$。

**例子 2.7**. 对标架丛$FM$和标准表示，伴丛$FM \\times_\\rho \\mathbb{R}^n$典范同构于切丛$TM$。

**定义 2.8**（伴随丛）. 李群$G$通过共轭作用$\\mathrm{Ad}_g(X) = gXg^{-1}$在其李代数$\\mathfrak{g}$上作用。对应的伴丛$P \\times_{\\mathrm{Ad}} \\mathfrak{g}$称为**伴随丛**，记为$\\mathrm{Ad} P$。

伴随丛在主丛联络理论中至关重要：主丛联络的曲率形式取值于$\\mathfrak{g}$，因此可视为$\\mathrm{Ad} P$-值的2-形式。

## 14.7.3 主丛上的联络 / Connections on Principal Bundles

### 主丛联络的定义 / Definition of Principal Connection

**定义 3.1**（主丛联络）. 主$G$-丛$P$上的一个**联络**是指对每点$p \\in P$指定一个子空间$H_p \\subset T_p P$，满足：
1. **直和分解**：$T_p P = H_p \\oplus V_p$，其中$V_p = \\ker(d\\pi_p)$为**垂直子空间**；
2. **G-等变性**：$(R_g)_* H_p = H_{pg}$对所有$g \\in G$成立；
3. $H_p$光滑地依赖于$p$。

子空间$H_p$称为**水平子空间**。给定联络，任何切向量$v \\in T_p P$可唯一分解为$v = v^H + v^V$。

### 联络形式 / Connection Form

垂直子空间$V_p$可典范等同于李代数$\\mathfrak{g}$：对$X \\in \\mathfrak{g}$，定义$P$上的**基本向量场**
\\[
\\underline{X}_p = \\left.\\frac{d}{dt}\\right|_{t=0} p \\cdot \\exp(tX).

\\]
映射$X \\mapsto \\underline{X}_p$给出了$\\mathfrak{g} \\cong V_p$的典范同构。

**定义 3.2**（联络1-形式）. 主丛联络等价地由一个$\\mathfrak{g}$-值的1-形式$\\omega \\in \\Omega^1(P; \\mathfrak{g})$刻画，满足：
1. **垂直条件**：对$X \\in \\mathfrak{g}$，$\\omega(\\underline{X}) = X$；
2. **G-等变性**：$(R_g)^* \\omega = \\mathrm{Ad}_{g^{-1}} \\circ \\omega$。

给定联络形式$\\omega$，水平子空间可恢复为$H_p = \\ker \\omega_p$。

**定理 3.3**（主丛联络与流形联络的对应）. 设$P = FM$为$M$的标架丛。则$FM$上的主$GL(n, \\mathbb{R})$-丛联络一一对应于$M$上的仿射联络$\\nabla$。

*证明思路*（概要）：给定$FM$上的联络形式$\\omega$，对$M$上的向量场$X, Y$，将$Y$提升为$FM$上的水平向量场$\\widetilde{Y}$，定义$\\nabla_X Y$为$\\widetilde{X}(\\widetilde{Y})$在底空间上的投影。反之，给定$M$上的仿射联络，可在$FM$上构造满足联络形式公理的$\\omega$。详见Kobayashi-Nomizu（1963）第三章。$\\square$

当$P = OM$为正交标架丛时，主$O(n)$-丛联络一一对应于$M$上的**度量联络**，而满足无挠条件的唯一度量联络正是Levi-Civita联络。这一对应揭示了主丛联络与经典黎曼几何之间的深层统一。

## 14.7.4 曲率形式与和乐群 / Curvature Forms and Holonomy Groups

### 曲率形式 / Curvature Form

**定义 4.1**（曲率形式）. 设$\\omega$为主$G$-丛$P$上的联络形式。其**曲率形式**$\\Omega \\in \\Omega^2(P; \\mathfrak{g})$定义为
\\[
\\Omega = d\\omega + \\frac{1}{2}[\\omega \\wedge \\omega],
\\]
其中$[\\omega \\wedge \\omega]$表示李代数值形式的外积再取李括号：对切向量$X, Y$，
\\[
[\\omega \\wedge \\omega](X, Y) = [\\omega(X), \\omega(Y)] - [\\omega(Y), \\omega(X)] = 2[\\omega(X), \\omega(Y)].
\\]

**定理 4.2**（比安基恒等式）. 曲率形式满足**结构方程**
\\[
d\\Omega = [\\Omega \\wedge \\omega] = \\mathrm{ad}(\\omega) \\wedge \\Omega.
\\]
等价地，$D\\Omega = 0$，其中$D$为外协变导数。

这一定理对应于黎曼几何中的第二Bianchi恒等式。实际上，当$P = FM$时，主丛的曲率形式 pullback 到底空间恰好给出黎曼曲率张量。

### 和乐群 / Holonomy Groups

**定义 4.3**（和乐群）. 固定$p \\in M$和$u \\in \\pi^{-1}(p) \\subset P$。设$\\gamma$为以$p$为基点的分段光滑闭曲线。将$u$沿$\\gamma$作水平提升得到$\\widetilde{\\gamma}$，终点为$u \\cdot g_\\gamma$（对某个$g_\\gamma \\in G$）。所有这样的$g_\\gamma$构成的子群
\\[
\\mathrm{Hol}_u(\\omega) = \\{g_\\gamma \\in G : \\gamma \\text{ 为基于 } p \\text{ 的闭曲线}\\}
\\]
称为联络$\\omega$在$u$处的**和乐群**（holonomy group）。仅考虑零伦闭曲线得到的子群称为**限制和乐群**$\\mathrm{Hol}^0_u(\\omega)$。

**定理 4.4**（和乐群约化定理）. 若$M$单连通且联络$\\omega$的曲率恒为零（$\\Omega \\equiv 0$），则$\\mathrm{Hol}_u(\\omega) = \\{e\\}$，此时主丛是平凡丛。更一般地，若$\\mathrm{Hol}_u(\\omega)$是$G$的真子群$H$，则联络$\\omega$可约化到主$H$-丛上。

**定理 4.5**（Ambrose-Singer定理）. 限制和乐群$\\mathrm{Hol}^0_u(\\omega)$的李代数由所有曲率形式$\\Omega(X, Y)$（$X, Y$为任意水平向量）在$\\mathfrak{g}$中生成的李子代数给出。

Ambrose-Singer定理深刻揭示了曲率与整体拓扑（通过闭曲线上的平行输运）之间的精确关系：曲率不仅局部地度量了平行输运的"失败"，其所有可能取值还生成了和乐群的李代数，从而决定了主丛联络的整体对称性。

## 14.7.5 计算实例：霍普夫丛与规范场 / Computational Examples

### 实例 1：霍普夫丛的联络与曲率

考虑主$U(1)$-丛$\\pi: S^3 \\to S^2$。$U(1)$的李代数$\\mathfrak{u}(1) \\cong i\\mathbb{R}$是交换的。联络1-形式可取为
\\[
\\omega = i \\mathrm{Im}(\\bar{z}_1 dz_1 + \\bar{z}_2 dz_2) = \\frac{i}{2}(\\bar{z}_1 dz_1 - z_1 d\\bar{z}_1 + \\bar{z}_2 dz_2 - z_2 d\\bar{z}_2).
\\]

在$S^3$上$|z_1|^2 + |z_2|^2 = 1$，因此$\\bar{z}_1 dz_1 + \\bar{z}_2 dz_2 + z_1 d\\bar{z}_1 + z_2 d\\bar{z}_2 = 0$。可以验证$\\omega$满足联络形式的两个条件：

1. 对基本向量场$\\underline{i} = i(z_1 \\partial_{z_1} + z_2 \\partial_{z_2}) + \\text{c.c.}$，$\\omega(\\underline{i}) = i$；
2. $U(1)$作用$z_j \\mapsto z_j e^{i\\theta}$使$\\omega$不变（因为$\\bar{z}_j dz_j$不变）。

曲率形式为
\\[
\\Omega = d\\omega + \\frac{1}{2}[\\omega \\wedge \\omega] = d\\omega,
\\]
因为$\\mathfrak{u}(1)$是交换的，李括号项为零。计算得
\\[
d\\omega = i(d\\bar{z}_1 \\wedge dz_1 + d\\bar{z}_2 \\wedge dz_2).
\\]
将$\\Omega$推前到底空间$S^2$上，得到$S^2$上的2-形式$F$。利用$S^2$上的齐次坐标$[z_1 : z_2]$，可验证$F$与$S^2$的标准面积形式成比例：
\\[
\\int_{S^2} F = -2\\pi.
\\]
这恰好对应于霍普夫丛的陈数（第一陈类）$c_1 = -1$。这一结果是磁单极子（Dirac monopole）的数学原型：$F$可视为电磁场强2-形式，其通过$S^2$的通量量子化为$2\\pi$的整数倍。

### 实例 2：杨-米尔斯方程

在物理中，规范场由主丛$P$上的联络$\\omega$描述，场强为曲率$\\Omega$。**杨-米尔斯泛函**定义为曲率的$L^2$范数：
\\[
S_{YM}(\\omega) = \\int_M \\|\\Omega\\|^2 \\ dV_g.

\\]
其中$M$为底空间（通常是4维洛伦兹或欧氏时空），$\\|\\Omega\\|^2$由基空间度量$g$和李代数上的伴随不变内积给出。

杨-米尔斯方程即上述泛函的Euler-Lagrange方程：
\\[
D^* \\Omega = 0,
\\]
其中$D^*$为外协变导数$D$的形式伴随。结合Bianchi恒等式$D\\Omega = 0$，这构成了规范场论的基本方程组。

当$G = U(1)$时，杨-米尔斯方程退化为真空中的麦克斯韦方程$dF = 0$，$d^*F = 0$。当$G$为非阿贝尔群（如$SU(2)$或$SU(3)$）时，方程变为非线性的，描述了弱相互作用和强相互作用。

**瞬子（Instanton）**是欧氏4维空间中自对偶（self-dual）或反自对偶（anti-self-dual）的杨-米尔斯联络，满足$\\Omega = \\pm *\\Omega$。在$SU(2)$主丛$S^3 \\times \\mathbb{R} \\to S^3$上，BPST瞬子给出了第一陈数为$\\pm 1$的显式解，是量子场论与微分几何交互影响的光辉典范。
