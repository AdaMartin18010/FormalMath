# 曲率张量

## 引言

曲率是Riemann几何的灵魂。它量化了流形在某点处的"弯曲"程度——即向量沿无穷小环路平行移动后产生的偏差。从高斯对曲面内蕴曲率的发现，到Riemann将曲率推广到任意维数，再到Einstein将曲率与物质的能量-动量联系起来，曲率张量始终是几何与物理交汇的核心。

Riemann曲率张量 $R(X,Y)Z$ 是一个 $(1,3)$-型张量，它通过Levi-Civita联络的二次协变微分定义。虽然其定义初看起来复杂，但它拥有深刻的对称性和恒等式（Bianchi恒等式），这些不仅简化了计算，也是广义相对论中场方程的数学基础。

本教程系统介绍Riemann曲率张量的定义、对称性、Bianchi恒等式，以及截面曲率的概念。

---

## 1. Riemann曲率张量

### 1.1 定义

**定义 1.1**。设 $(M, g)$ 为Riemann流形，$\nabla$ 为Levi-Civita联络。**Riemann曲率张量**定义为

$$R(X, Y)Z := \nabla_X \nabla_Y Z - \nabla_Y \nabla_X Z - \nabla_{[X,Y]} Z.$$

等价地，作为 $(0,4)$-张量：

$$R(X, Y, Z, W) := g(R(X, Y)Z, W).$$

### 1.2 坐标表示

在坐标下：

$$R^l_{\,kij} = \partial_i \Gamma^l_{jk} - \partial_j \Gamma^l_{ik} + \sum_m (\Gamma^m_{jk}\Gamma^l_{im} - \Gamma^m_{ik}\Gamma^l_{jm}).$$

---

## 2. 对称性

### 2.1 基本对称性

**命题 2.1**。$R(X,Y,Z,W)$ 满足：
1. **反对称性**：$R(X,Y,Z,W) = -R(Y,X,Z,W) = -R(X,Y,W,Z)$；
2. **第一Bianchi恒等式**：$R(X,Y,Z,W) + R(Y,Z,X,W) + R(Z,X,Y,W) = 0$；
3. **对称性**：$R(X,Y,Z,W) = R(Z,W,X,Y)$。

### 2.2 独立分量

**命题 2.2**。$n$ 维流形上Riemann张量的独立分量数为

$$\frac{n^2(n^2 - 1)}{12}.$$

---

## 3. Bianchi恒等式

### 3.1 第二Bianchi恒等式

**定理 3.1（第二Bianchi恒等式）**。设 $\nabla$ 为Levi-Civita联络。则

$$(\nabla_X R)(Y,Z,W,V) + (\nabla_Y R)(Z,X,W,V) + (\nabla_Z R)(X,Y,W,V) = 0.$$

*证明*。在正规坐标（normal coordinates）下，$\nabla$ 的Christoffel符号在一点为零，直接计算即可。$\square$

### 3.2 缩并Bianchi恒等式

**推论 3.2**。对Ricci曲率 $\operatorname{Ric}$ 和标量曲率 $S$：

$$\operatorname{div}(\operatorname{Ric}) = \frac{1}{2} dS,$$

即 $\nabla^j R_{ij} = \frac{1}{2} \partial_i S$。

这是Einstein场方程 $G = 8\pi T$ 中Bianchi恒等式 $\operatorname{div} G = 0$ 保证能量守恒的数学来源。

## 4. 重要例子

### 例子 1：常曲率空间

**例 4.1**。常截面曲率 $K$ 的流形满足

$$R(X,Y,Z,W) = K(g(X,Z)g(Y,W) - g(X,W)g(Y,Z)).$$

- $K = 0$：欧氏空间；
- $K > 0$：球面 $S^n(R)$，$K = 1/R^2$；
- $K < 0$：双曲空间 $\mathbb{H}^n(R)$，$K = -1/R^2$。

### 例子 2：乘积流形

**例 4.2**。$M \times N$ 的曲率由 $M$ 和 $N$ 的曲率直和给出。混合曲率为零：$R(X_M, Y_N) = 0$。
