# 超曲面曲率

## 引言

超曲面（hypersurface）是余维数为1的子流形。由于法丛是线丛，超曲面的第二基本形式退化为一个二次型（或等价地，形状算子是一个自伴线性映射），这使得超曲面的外蕴几何特别丰富且可计算。主曲率是形状算子的特征值，平均曲率是它们的平均，Gauss曲率是它们的乘积（在二维情形）。

极小曲面（minimal surface）——平均曲率为零的超曲面——是变分法中面积泛函的临界点。从肥皂膜的物理形态到广义相对论中的marginally trapped surface，极小曲面具有丰富的数学结构和深刻的物理应用。常平均曲率曲面（CMC）则是极小曲面的自然推广。

本教程从形状算子出发，系统建立超曲面的曲率理论，分析极小曲面的变分特征，并给出经典例子。

---

## 1. 形状算子与主曲率

### 1.1 超曲面的基本设定

设 $M^n \subseteq \widetilde{M}^{n+1}$ 为定向超曲面，$N$ 为光滑单位法向量场。因法丛1维，$II(X,Y) = h(X,Y)N$，$h$ 为标量值对称 $(0,2)$-张量。

**定义 1.1**。形状算子 $S: TM \to TM$ 定义为
$$\langle S(X), Y \rangle = h(X,Y) = \langle II(X,Y), N \rangle.$$
等价地，$S(X) = -\widetilde{\nabla}_X N$（切分量）。

**命题 1.1**。$S$ 关于诱导度量自伴。

**证明**：$h$ 对称。∎

### 1.2 主曲率与主方向

**定义 1.2**。自伴算子 $S$ 的实特征值 $\kappa_1, \dots, \kappa_n$ 称为 **主曲率**（principal curvatures）。对应的单位特征向量方向称为 **主方向**。

**命题 1.2**。不同主曲率对应的主方向正交。

### 1.3 曲率不变量

**定义 1.3**。由主曲率定义：
- **平均曲率**（mean curvature）：$H = \frac{1}{n}\sum_{i=1}^n \kappa_i = \frac{1}{n}\operatorname{tr} S$；
- **Gauss-Kronecker曲率**：$K = \prod_{i=1}^n \kappa_i = \det S$；
- **第 $k$ 阶平均曲率**：$\sigma_k = e_k(\kappa_1, \dots, \kappa_n)$（初等对称多项式）。

**注记**：$2H = \sigma_1$，$K = \sigma_n$。对 $n=2$，$K = \kappa_1 \kappa_2$，$H = (\kappa_1 + \kappa_2)/2$。

---

## 2. Gauss方程与曲率关系

### 2.1 超曲面的Gauss方程

**定理 2.1**。设 $\widetilde{M}$ 的截面曲率为 $\widetilde{K}$，$M$ 的截面曲率为 $K$。对由正交单位向量 $X, Y$ 张成的平面：
$$K(X,Y) = \widetilde{K}(X,Y) + \langle S(X), X \rangle \langle S(Y), Y \rangle - \langle S(X), Y \rangle^2.$$

当 $\widetilde{M} = \mathbb{R}^{n+1}$（平坦），$K(X,Y) = \kappa_X \kappa_Y$（若 $X, Y$ 为主方向）。

### 2.2 二维情形的Gauss绝妙定理

对 $M^2 \subseteq \mathbb{R}^3$，$K = \kappa_1 \kappa_2$ 仅依赖于诱导度量（Theorema Egregium）。

---

## 3. 极小曲面

### 3.1 第一变分公式

**定理 3.1**。设 $M$ 为紧致带边超曲面。对法向变分 $M_t$（速度场 $fN$，$f|_{\partial M} = 0$），面积泛函 $A(t) = \operatorname{Area}(M_t)$ 满足
$$\left.\frac{dA}{dt}\right|_{t=0} = -n \int_M f H \, dV.$$

**证明概要**：利用Gauss公式和散度定理。第一变分与平均曲率成正比。∎

**推论 3.1**。$H = 0$ 当且仅当 $M$ 是面积泛函的临界点（对紧支变分）。

### 3.2 第二变分与稳定性

**定理 3.2**。第二变分公式：
$$\left.\frac{d^2A}{dt^2}\right|_{t=0} = \int_M \left(|\nabla f|^2 - (|S|^2 + \widetilde{\operatorname{Ric}}(N,N))f^2\right) dV.$$

若对所有 $f$ 第二变分非负，称 $M$ **稳定**。

## 4. 具体例子

### 例子 4.1：$S^n(r) \subseteq \mathbb{R}^{n+1}$

单位外法向 $N = x/r$。$S(X) = -\widetilde{\nabla}_X N = -X/r$。

主曲率：$\kappa_1 = \dots = \kappa_n = -1/r$（依外法向约定）。

$H = -1/r$，$K = (-1/r)^n$。

### 例子 4.2：悬链面（Catenoid）

曲线 $r(z) = a \cosh(z/a)$ 绕 $z$ 轴旋转。参数化：
$$X(z,\theta) = (a \cosh(z/a) \cos\theta, a \cosh(z/a) \sin\theta, z).$$

主曲率：$\kappa_1 = -\kappa_2 = -\frac{1}{a \cosh^2(z/a)}$。

$H = 0$，故为极小曲面。这是 $\mathbb{R}^3$ 中唯一的旋转极小曲面（除平面外）。

### 例子 4.3：螺旋面（Helicoid）

$$X(u,v) = (u \cos v, u \sin v, av).$$

主曲率：$\kappa_{1,2} = \pm a/(u^2 + a^2)$，$H = 0$。

螺旋面与悬链面在局部上是等距的（同为Gauss曲率 $K = -a^2/(u^2+a^2)^2$ 的曲面），这一事实由Bonnet发现。

### 例子 4.4：$T^2 = S^1(a) \times S^1(b) \subseteq \mathbb{R}^4$

参数化：$(a\cos\theta, a\sin\theta, b\cos\varphi, b\sin\varphi)$。

法丛2维（在 $\mathbb{R}^4$ 中余维2，非超曲面）。若视为 $\mathbb{R}^3$ 中的环面 $T^2_{\text{std}}$（旋转环面）：

参数化 $((R+r\cos\phi)\cos\theta, (R+r\cos\phi)\sin\theta, r\sin\phi)$，$0 < r < R$。

主曲率：$\kappa_1 = \cos\phi/(R+r\cos\phi)$（子午线曲率），$\kappa_2 = 1/r$（平行圆曲率）。

$H = \frac{1}{2}\left(\frac{\cos\phi}{R+r\cos\phi} + \frac{1}{r}\right)$。仅当 $R = 0$（退化球面）时 $H=0$。故标准环面非极小。
