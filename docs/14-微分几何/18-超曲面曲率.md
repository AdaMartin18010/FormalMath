# 超曲面曲率

## 引言

超曲面（hypersurface）是余维数为1的子流形。由于法丛是线丛（秩1），超曲面的第二基本形式退化为一个二次型（或等价地，形状算子是一个自伴线性映射）。这使得超曲面的外蕴几何特别丰富且可计算：主曲率是形状算子的特征值，平均曲率是它们的平均，Gauss曲率是它们的乘积（在二维情形）。

极小曲面（minimal surface）——平均曲率为零的超曲面——是变分法中面积泛函的临界点，具有丰富的数学结构和物理应用（肥皂膜、广义相对论中的marginally trapped surface等）。

本教程介绍超曲面的主曲率、平均曲率、Gauss曲率以及极小曲面理论的基础。

---

## 1. 形状算子与主曲率

### 1.1 形状算子

设 $M^n \subseteq \widetilde{M}^{n+1}$ 为定向超曲面，$N$ 为单位法向量场。

**定义 1.1**。形状算子 $S: TM \to TM$ 定义为

$$S(X) := -\widetilde{\nabla}_X N$$

（切分量）。

**命题 1.2**。$S$ 关于诱导度量自伴：$\langle S(X), Y \rangle = \langle X, S(Y) \rangle$。

### 1.2 主曲率

**定义 1.3**。$S$ 的特征值 $\kappa_1, \dots, \kappa_n$ 称为**主曲率**。

**定义 1.4**。**平均曲率** $H := \frac{1}{n}\sum \kappa_i = \frac{1}{n}\operatorname{tr}(S)$。**Gauss曲率** $K := \prod \kappa_i = \det(S)$。

---

## 2. 极小曲面

**定义 2.1**。超曲面 $M$ 称为**极小的**（minimal），如果 $H = 0$。

**命题 2.2**。极小曲面是面积泛函的变分临界点（对紧支变分）。

---

## 3. 重要例子

### 例子 1：球面

**例 3.1**。$S^n(r) \subseteq \mathbb{R}^{n+1}$。$S = \frac{1}{r}\operatorname{id}$，故所有主曲率 $\kappa_i = 1/r$，$H = 1/r$，$K = 1/r^n$。

### 例子 2：极小曲面

**例 3.2**。悬链面（catenoid）和螺旋面（helicoid）是 $\mathbb{R}^3$ 中经典的极小曲面。
