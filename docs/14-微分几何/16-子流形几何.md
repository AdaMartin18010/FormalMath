# 子流形几何

## 引言

子流形是嵌入在更大流形中的低维流形，其几何同时受到内蕴结构和外蕴环境的影响。第一基本形式描述子流形的诱导度量——纯粹的内蕴信息；第二基本形式编码了子流形在环境空间中的弯曲方式——外蕴信息。从高斯的绝妙定理到现代子流形理论，第一和第二基本形式的 interplay 一直是微分几何的核心课题。在广义相对论中，时空的类空超曲面的外蕴几何与Einstein约束方程直接相关。

本教程从子流形的基本定义出发，严格建立第一和第二基本形式，引入形状算子，推导Gauss-Weingarten方程，并分析超曲面的特殊情形。

---

## 1. 第一基本形式

### 1.1 诱导度量

设 $F: M^m \to \widetilde{M}^n$ 为浸入（immersion），$\widetilde{g}$ 为 $\widetilde{M}$ 的Riemann度量。

**定义 1.1**。$M$ 上的 **诱导度量**（第一基本形式，first fundamental form）为
$$g(X, Y) := \widetilde{g}(dF(X), dF(Y)), \quad X, Y \in T_p M.$$

**命题 1.1**。$g$ 是 $M$ 上的Riemann度量。

**证明**：$dF_p$ 为单射（因 $F$ 为浸入），故 $g_p$ 正定。光滑性由 $F$ 和 $\widetilde{g}$ 的光滑性保证。∎

**例 1.1**。对 $S^n \subseteq \mathbb{R}^{n+1}$，诱导度量即球面度量。在球坐标下：
$$g_{S^n} = d\theta^2 + \sin^2\theta \, d\Omega_{n-1}^2.$$

## 2. 第二基本形式

### 2.1 法丛与正交分解

因 $F$ 为浸入，$dF(TM)$ 是 $T\widetilde{M}|_{F(M)}$ 的秩 $m$ 子丛。其正交补
$$NM := (dF(TM))^\perp \subseteq T\widetilde{M}|_{F(M)}$$
称为 **法丛**（normal bundle），秩为 $n-m$。

在每点 $p \in M$，有正交分解：
$$T_{F(p)}\widetilde{M} = dF_p(T_p M) \oplus N_p M.$$
记切投影 $\top$ 和法投影 $\perp$。

### 2.2 第二基本形式的定义

**定义 2.1**。$M$ 的 **第二基本形式**（second fundamental form）为
$$II(X, Y) := \left(\widetilde{\nabla}_{dF(X)} dF(Y)\right)^\perp - dF(\nabla_X Y) = \left(\widetilde{\nabla}_{dF(X)} dF(Y)\right)^\perp,$$
其中 $X, Y \in \mathfrak{X}(M)$，$\widetilde{\nabla}$ 和 $\nabla$ 分别为环境和诱导Levi-Civita联络。

**命题 2.1**。$II$ 是 $C^\infty(M)$-双线性的、对称的、取值于 $NM$ 的张量。

**证明**：双线性性由联络的Leibniz性质。对称性：因Levi-Civita联络无挠，$\widetilde{\nabla}_{dF(X)} dF(Y) - \widetilde{\nabla}_{dF(Y)} dF(X) = [dF(X), dF(Y)] = dF([X,Y])$（$F$ 为浸入），其法分量为零。故 $II(X,Y) = II(Y,X)$。∎

### 2.3 Gauss公式与Weingarten公式

**定理 2.1（Gauss公式）**。对 $X, Y \in \mathfrak{X}(M)$，
$$\widetilde{\nabla}_{dF(X)} dF(Y) = dF(\nabla_X Y) + II(X,Y). \tag{G}$$

**证明**：由定义，$\widetilde{\nabla}_{dF(X)} dF(Y) = (\widetilde{\nabla}_{dF(X)} dF(Y))^\top + (\widetilde{\nabla}_{dF(X)} dF(Y))^\perp$。切分量恰为 $dF(\nabla_X Y)$（由诱导联络的定义），法分量为 $II(X,Y)$。∎

**定理 2.2（Weingarten公式）**。对 $X \in \mathfrak{X}(M)$，$\xi \in \Gamma(NM)$，
$$\widetilde{\nabla}_{dF(X)} \xi = -dF(A_\xi X) + \nabla_X^\perp \xi, \tag{W}$$
其中 $A_\xi: TM \to TM$ 为形状算子，$\nabla^\perp$ 为法联络。

**证明**：分解 $\widetilde{\nabla}_X \xi$ 为切分量和法分量。切分量定义 $-A_\xi X$（负号为传统约定），法分量为 $\nabla_X^\perp \xi$。∎

### 2.4 形状算子

**定义 2.2**。对法向量 $\xi \in N_p M$，**形状算子**（shape operator / Weingarten map）$A_\xi: T_p M \to T_p M$ 定义为
$$\langle A_\xi(X), Y \rangle = \langle II(X, Y), \xi \rangle, \quad X, Y \in T_p M.$$

**命题 2.2**。$A_\xi$ 关于 $g$ 自伴：$\langle A_\xi(X), Y \rangle = \langle X, A_\xi(Y) \rangle$。

**证明**：由 $II$ 的对称性。∎

## 3. 超曲面的特殊情形

当 $M^{n-1} \subseteq \widetilde{M}^n$ 为超曲面时，法丛1维。取单位法向量场 $N$（局部存在，整体存在性要求 $M$ 可定向）。

**定义 3.1**。标量第二基本形式 $h$ 定义为
$$II(X, Y) = h(X, Y) N.$$

**定义 3.2**。形状算子 $A = A_N$ 满足 $\langle AX, Y \rangle = h(X, Y)$。

**定义 3.3（主曲率）**。$A$ 为自伴算子，其特征值 $\kappa_1, \dots, \kappa_{n-1}$ 称为 **主曲率**。对应的特征方向称为主方向。

**定义 3.4（平均曲率与Gauss曲率）**。

- **平均曲率** $H = \frac{1}{n-1}\operatorname{tr} A = \frac{1}{n-1}\sum \kappa_i$；
- **Gauss-Kronecker曲率** $K = \det A = \prod \kappa_i$。

**例 3.1（$S^{n-1}(r) \subseteq \mathbb{R}^n$）**。单位外法向 $N = x/r$。对切向量 $X$，$\widetilde{\nabla}_X N = X/r$。故 $A = -I/r$（符号依约定），主曲率均为 $-1/r$，$H = -1/r$，$K = (-1/r)^{n-1}$。

## 4. 具体例子

### 例子 4.1：旋转曲面

设 $\gamma(t) = (r(t), z(t))$ 为 $xz$-平面上的曲线（$r > 0$），绕 $z$ 轴旋转得曲面 $M \subseteq \mathbb{R}^3$：
$$F(t, \theta) = (r(t)\cos\theta, r(t)\sin\theta, z(t)).$$

经线切向量 $X = (r'\cos\theta, r'\sin\theta, z')$，纬线切向量 $Y = (-r\sin\theta, r\cos\theta, 0)$。

诱导度量：$g_{tt} = r'^2 + z'^2 = 1$（弧长参数），$g_{\theta\theta} = r^2$，$g_{t\theta} = 0$。

法向量 $N = (z'\cos\theta, z'\sin\theta, -r')$。

$II(\partial_t, \partial_t) = \langle \widetilde{\nabla}_{\partial_t} \partial_t, N \rangle = r'z'' - r''z'$（曲率）。

$II(\partial_\theta, \partial_\theta) = \langle \widetilde{\nabla}_{\partial_\theta} \partial_\theta, N \rangle = r z'$。

主曲率：$\kappa_1 = r'z'' - r''z'$（经线曲率），$\kappa_2 = z'/r$。Gauss曲率 $K = \kappa_1 \kappa_2 = -r''/r$（因 $r'^2 + z'^2 = 1$，$r'r'' + z'z'' = 0$）。

### 例子 4.2：$\mathbb{R}^3$ 中的极小曲面

**定义 4.1**。平均曲率 $H = 0$ 的曲面称为 **极小曲面**。

**例**：悬链面（catenoid）：$r(t) = \cosh t$，$z(t) = t$。$H = 0$。验证：$r' = \sinh t$，$z' = 1/\cosh t$？不，$r'^2 + z'^2 = \sinh^2 t + 1 = \cosh^2 t$ 不对。正确参数化：$(\cosh t \cos\theta, \cosh t \sin\theta, t)$。则 $r = \cosh t$，$z = t$，$r' = \sinh t$，$z' = 1$，$r'^2 + z'^2 = \sinh^2 t + 1 = \cosh^2 t$。取弧长参数 $s$：$ds/dt = \cosh t$。

$\kappa_1 = -1/\cosh^2 t$（经线曲率），$\kappa_2 = 1/\cosh^2 t$（纬线曲率）。$H = (\kappa_1 + \kappa_2)/2 = 0$。
