# 辛几何入门

## 引言

辛几何（symplectic geometry）是研究辛流形——配备闭非退化2-形式的偶维流形——的数学分支。它起源于经典力学的Hamilton形式：19世纪末，Poincaré在研究天体力学三体问题时发现了相空间中的积分不变量，Cartan则系统发展了外微分形式的语言。Weyl于1939年引入"symplectic"一词，而现代辛几何的繁荣则始于Arnold在1960年代的工作以及Gromov在1985年证明的辛非挤压定理（nonsqueezing theorem）——这一结果揭示了辛几何与Riemann几何的本质差异。

辛几何与Riemann几何形成有趣的对比：Riemann度量是正定的对称双线性形式，而辛形式是反对称非退化的；Riemann几何关注长度和角度，具有丰富的局部不变量（曲率），而辛几何关注面积和体积，Darboux定理表明所有辛流形局部上都是标准的——辛几何没有局部不变量！全局不变量的丰富性正是辛拓扑的核心课题。

本教程从辛流形的严格定义出发，证明Darboux定理，建立Hamilton力学的几何框架，并介绍辛容量的初步概念。

## 1. 辛形式与辛流形

### 1.1 线性辛几何

**定义 1.1**。有限维实向量空间 $V$ 上的 **辛形式** 是反对称双线性形式 $\omega: V \times V \to \mathbb{R}$，满足 **非退化性**：若 $\omega(v, w) = 0$ 对所有 $w \in V$，则 $v = 0$。

**命题 1.1**。若 $V$ 上存在辛形式，则 $\dim V = 2n$ 为偶数。

**证明**：反对称形式在奇数维上行列式为零，故必退化。∎

**定理 1.1（线性Darboux定理）**。对任意辛向量空间 $(V, \omega)$，存在基 $(e_1, \dots, e_n, f_1, \dots, f_n)$ 使
$$\omega = \sum_{i=1}^n e^i \wedge f^i,$$
即 $\omega(e_i, e_j) = \omega(f_i, f_j) = 0$，$\omega(e_i, f_j) = \delta_{ij}$。

### 1.2 辛流形

**定义 1.2**。光滑流形 $M$ 上的 **辛形式** 是闭的（$d\omega = 0$）、非退化的2-形式。**辛流形** 是二元组 $(M, \omega)$。

**命题 1.2**。辛流形是偶维的，且可定向（由体积形式 $\omega^n/n!$）。

**证明**：非退化性要求每点的切空间为辛向量空间，故偶维。$\omega^n$ 处处非零，给出处处非零的 $2n$-形式。∎

## 2. Darboux定理

### 2.1 陈述

**定理 2.1（Darboux定理）**。设 $(M, \omega)$ 为辛流形。对任意 $p \in M$，存在邻域坐标 $(q^1, \dots, q^n, p_1, \dots, p_n)$（称为 **Darboux坐标** 或 **典范坐标**），使得
$$\omega = \sum_{i=1}^n dq^i \wedge dp_i.$$

### 2.2 证明概要（Moser技巧）

**证明**：核心思想是通过一族辛形式的同伦，将 $\omega$ 变形为标准形式 $\omega_0$，同时用流来拉回坐标。

**步骤一**：在 $p$ 点，由线性代数可取切空间基使 $\omega_p = \sum dq^i \wedge dp_i$。在局部坐标下，$\omega = \omega_0 + O(|x|)$。

**步骤二**：考虑一族2-形式 $\omega_t = (1-t)\omega_0 + t\omega$（$t \in [0,1]$）。在 $p$ 附近，$\omega_t$ 对所有 $t$ 非退化（因非退化性在开条件下保持）。

**步骤三**：由Poincaré引理（$d\omega = d\omega_0 = 0$），存在1-形式 $\beta$ 使 $\omega - \omega_0 = d\beta$ 且 $\beta_p = 0$。

**步骤四**：设 $X_t$ 为由 $\iota_{X_t}\omega_t = -\beta$ 定义的向量场（良定因 $\omega_t$ 非退化）。令 $\psi_t$ 为 $X_t$ 的流，$\psi_0 = \mathrm{id}$。计算
$$\frac{d}{dt}(\psi_t^*\omega_t) = \psi_t^*(\mathcal{L}_{X_t}\omega_t + \frac{d\omega_t}{dt}) = \psi_t^*(d\iota_{X_t}\omega_t + \iota_{X_t}d\omega_t + \omega - \omega_0)$$
$$= \psi_t^*(-d\beta + 0 + d\beta) = 0.$$
故 $\psi_t^*\omega_t = \omega_0$。特别地，$\psi_1^*\omega = \omega_0$。

**步骤五**：$\psi_1$ 为局部微分同胚（因 $X_t(p) = 0$，流固定 $p$）。取 $\psi_1$ 的局部逆作为坐标变换，即得Darboux坐标。∎

**意义**：Darboux定理表明辛流形没有局部不变量——与Riemann几何形成根本对比。辛几何的全局不变量必须通过拓扑或渐近方法研究。

## 3. Hamilton力学

### 3.1 Hamilton向量场

**定义 3.1**。设 $H: M \to \mathbb{R}$ 为光滑函数（**Hamilton函数**）。由 $\omega$ 的非退化性，存在唯一的向量场 $X_H$ 使
$$\iota_{X_H}\omega = dH,$$
称为 $H$ 的 **Hamilton向量场**。

**命题 3.1**。$H$ 沿 $X_H$ 的流守恒：$X_H(H) = \omega(X_H, X_H) = 0$。

### 3.2 Hamilton方程

在Darboux坐标下，$\omega = \sum dq^i \wedge dp_i$，$dH = \frac{\partial H}{\partial q^i}dq^i + \frac{\partial H}{\partial p_i}dp_i$。

由 $\iota_{X_H}\omega = dH$：设 $X_H = \dot{q}^i \partial_{q^i} + \dot{p}_i \partial_{p_i}$，则
$$\iota_{X_H}(dq^i \wedge dp_i) = \dot{q}^i dp_i - \dot{p}_i dq^i = dH.$$
故
$$\dot{q}^i = \frac{\partial H}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial H}{\partial q^i}. \tag{Hamilton}$$

### 3.3 Poisson括号

**定义 3.2**。函数 $f, g$ 的 **Poisson括号** 为
$$\{f, g\} := \omega(X_f, X_g) = X_f(g).$$

**命题 3.2**。$(C^\infty(M), \{-,-\})$ 构成Lie代数，且 $\{f, gh\} = \{f,g\}h + g\{f,h\}$（Leibniz法则）。

## 4. 辛容量的非挤压定理

**定义 4.1（辛容量）**。映射 $c$ 从辛流形类到 $[0, +\infty]$ 称为 **辛容量**，若满足：

1. **单调性**：若 $\phi: (M, \omega) \hookrightarrow (N, \tau)$ 为辛嵌入，则 $c(M) \leq c(N)$；
2. **共形性**：$c(M, \lambda\omega) = |\lambda| c(M, \omega)$；
3. **规范性**：$c(B^{2n}(1), \omega_0) = c(Z^{2n}(1), \omega_0) = \pi$，其中 $B^{2n}(r)$ 为球，$Z^{2n}(r) = B^2(r) \times \mathbb{R}^{2n-2}$ 为柱。

**定理 4.1（Gromov非挤压定理, 1985）**。若存在辛嵌入 $B^{2n}(R) \hookrightarrow Z^{2n}(r)$，则 $R \leq r$。

**证明概要**：利用拟全纯曲线（pseudoholomorphic curves）技术。证明 $B^{2n}(R)$ 的辛容量为 $\pi R^2$，而 $Z^{2n}(r)$ 的辛容量为 $\pi r^2$。由单调性即得。∎

这一定理深刻表明：辛嵌入比体积保持嵌入严格得多——即使 $R > r$ 时 $B^{2n}(R)$ 的体积可任意小于 $Z^{2n}(r)$ 的体积（当 $n \geq 2$），辛嵌入仍不可能。

## 5. 具体例子

### 例子 5.1：余切丛

**定理 5.1**。设 $Q$ 为光滑流形，$M = T^*Q$ 有余切丛的 **典范辛形式** $\omega = d\theta$，其中 **Liouville 1-形式** $\theta$ 定义为
$$\theta_\alpha(V) = \alpha(d\pi_{\alpha}(V)), \quad \alpha \in T^*Q, \, V \in T_\alpha(T^*Q),$$
$\pi: T^*Q \to Q$ 为投影。

在局部坐标 $(q^i, p_i)$（$p_i$ 为余切坐标）下，$\theta = \sum p_i dq^i$，$\omega = \sum dq^i \wedge dp_i$。

**物理意义**：$Q$ 为位形空间，$T^*Q$ 为相空间，$\omega$ 为相空间的自然辛结构。Hamilton力学即在此框架下展开。

### 例子 5.2：Kähler流形

**定义 5.1**。Kähler流形是三元组 $(M, g, J)$，其中 $g$ 为Riemann度量，$J$ 为近复结构（$J^2 = -\mathrm{id}$），满足：

1. $g(JX, JY) = g(X, Y)$（$J$ 为等距）；
2. $\nabla J = 0$（$J$ 平行）。

**命题 5.1**。Kähler流形上有自然辛形式 $\omega(X, Y) = g(JX, Y)$，且 $d\omega = 0$。

**例**：$\mathbb{C}P^n$ 配Fubini-Study度量为紧Kähler流形，其辛形式生成上同调 $H^2(\mathbb{C}P^n, \mathbb{R}) \cong \mathbb{R}$。
