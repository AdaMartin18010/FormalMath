# 数学经典实例与反例集

数学中的反例与病态构造不仅是直觉的矫正器，更是理论深度的探针。本节系统整理分析、代数、几何与拓扑中的经典反例，每个反例配有严格的构造与证明。

## 1. 分析学反例

### 1.1 连续但无处可微函数

**定理 1.1（Weierstrass）**. 存在 $\mathbb{R}$ 上的连续函数处处不可微。

*构造*. 

$$W(x) = \sum_{n=0}^\infty a^n \cos(b^n \pi x),$$

其中 $0 < a < 1$，$b$ 为奇整数，$ab > 1 + \frac{3\pi}{2}$。

*证明概要*. 对任意 $x$，取 $h_m = \pm \frac{1}{2}b^{-m}$（符号选择使 $b^m(x+h_m)$ 与 $b^m x$ 的最近整数距离 $\geq 1/4$）。差商中第 $n$ 项贡献：$n > m$ 时振荡剧烈，$n \leq m$ 时同号叠加。验证 $|\frac{W(x+h_m)-W(x)}{h_m}| \to \infty$。$\square$

### 1.2 可积但无原函数的函数

**例子 1.2**. 

$$f(x) = \begin{cases} 0 & x = 0 \\ 2x\sin(1/x^2) - \frac{2}{x}\cos(1/x^2) & x \neq 0 \end{cases}$$

在 $[-1, 1]$ 上无界，故非 Riemann 可积。但其"形式原函数"$F(x) = x^2\sin(1/x^2)$（$x \neq 0$），$F(0) = 0$ 在 $x=0$ 处不可导（导数振荡无极限）。

修正例子：Volterra 函数——在 Cantor 集型零测集外可微且导数有界，但导数不 Riemann 可积。

## 2. 代数学反例

### 2.1 非交换有限群的最小例子

**例子 2.1**. $S_3$（阶 6）是最小的非交换群。$D_4$（阶 8，二面体群）是阶最小的非交换 $p$-群。

### 2.2 素元非不可约元的整环

**例子 2.2**. $\mathbb{Z}[\sqrt{-5}]$ 中，$2$ 是素元（生成素理想）但非不可约元：在扩环中可分解。

更准确例子：考虑 $\mathbb{Z}[x]$，$(2, x)$ 是素理想但非主理想。

## 3. 拓扑学反例

### 3.1 连通但非道路连通

**例子 3.1**. $\{(x, \sin(1/x)) : x > 0\} \cup \{(0, y) : -1 \leq y \leq 1\} \subset \mathbb{R}^2$（拓扑学家的正弦曲线）。

*证明*. 图像连通（连续像的闭包）。但 $(0,0)$ 与 $(1, \sin 1)$ 之间不存在道路：任何从 $(0,0)$ 出发的道路在 $x \to 0$ 时 $y$ 必须在 $[-1,1]$ 中振荡，无法定义连续极限。$\square$

### 3.2 紧致但非列紧（非度量空间）

**例子 3.2**. $[0,1]$ 配备余有限拓扑。紧致但非 Hausdorff，故在度量空间中不可能。

在度量空间中，紧致 $\Leftrightarrow$ 列紧 $\Leftrightarrow$ 完全有界且完备。

## 4. 例子

### 4.1 例子：Cantor 函数

Cantor 函数（魔鬼阶梯）$c: [0,1] \to [0,1]$：
- 连续、单调递增；
- 在 Cantor 集的补集（开集，测度 1）上为常数；
- $c(0) = 0$，$c(1) = 1$；
- 几乎处处可微且导数为零，但 $\int_0^1 c'(x)dx = 0 \neq 1 = c(1) - c(0)$。

这说明微积分基本定理对非绝对连续函数不成立。

### 4.2 例子：Peano 曲线

**定理 4.1（Peano）**. 存在连续满射 $f: [0,1] \to [0,1]^2$。

*构造*. Hilbert 曲线或空间填充曲线的极限构造：递归地将单位区间映射到越来越细分的网格中，一致收敛到满射。$\square$
