# 拍卖理论中的机制设计

## 应用领域

**学科**: 博弈论 / 机制设计 / 微观经济学  
**具体应用**: 频谱拍卖、在线广告拍卖、艺术品拍卖、政府采购  
**MSC分类**: 91A40 (Game-theoretic models), 91B26 (Market models), 91A10 (Noncooperative games)

---

## 数学概念

### 核心数学工具

1. **贝叶斯博弈**: 不完全信息博弈
2. **占优策略**: 无论对手如何选择都是最优的策略
3. **激励相容**: 说真话是最优策略
4. **个体理性**: 参与拍卖的收益非负
5. **收益等价定理**: 不同拍卖形式的预期收益关系

### 关键定义

- **估值**: 竞拍者对物品的私人价值
- **分配规则**: 谁获得物品
- **支付规则**: 获胜者支付多少

---

## 问题描述

### 拍卖设计问题

**卖方目标**: 最大化预期收益（或效率、社会福利）

**挑战**:
- 竞拍者估值私有
- 竞拍者策略性行为
- 需要激励说真话

### 四种基本拍卖形式

1. **英国式（升价拍卖）**: 价格递增，最后出价者获胜
2. **荷兰式（降价拍卖）**: 价格递减，首个接受者获胜
3. **一级密封价格**: 最高价获胜，支付自己出价
4. **二级密封价格（维克里拍卖）**: 最高价获胜，支付第二高价

## 数学模型

### 私人价值模型

**设置**:
- $n$ 个竞拍者
- 竞拍者 $i$ 的估值 $v_i \sim F_i$（独立同分布）
- $v_i$ 是私人信息

**效用**: 若获胜并支付 $p$，效用为 $u_i = v_i - p$；若失败，效用为0。

### 维克里拍卖分析

**机制**: 最高价获胜，支付第二高价

**定理**: 在维克里拍卖中，说真话（出价等于估值）是占优策略。

**证明**:

竞拍者 $i$ 出价 $b_i$，其他竞拍者出价 $b_{-i}$。令 $b^{(1)}_{-i} = \max_{j \neq i} b_j$。

**情况1**: $b_i > b^{(1)}_{-i}$
- 获胜，支付 $b^{(1)}_{-i}$
- 效用: $v_i - b^{(1)}_{-i}$

**情况2**: $b_i < b^{(1)}_{-i}$
- 失败，效用: 0

**分析**:
- 若 $v_i > b^{(1)}_{-i}$，出价 $b_i = v_i$ 获胜，获得正效用
- 若 $v_i < b^{(1)}_{-i}$，出价 $b_i = v_i$ 失败，避免负效用

因此 $b_i = v_i$ 是最优的，独立于他人出价。 ∎

### 一级密封价格拍卖

**均衡策略**: 假设估值在 $[0, 1]$ 上均匀分布

**对称贝叶斯纳什均衡**:

$$b(v) = \frac{n-1}{n} v$$

**推导**:

竞拍者出价 $b$，获胜概率（当其他人都出价低于 $b$）:

$$P(\text{win}) = P(b(v_j) < b \text{ for all } j \neq i) = \left(\frac{n}{n-1}b\right)^{n-1}$$

预期效用:

$$U(b) = (v - b) \cdot \left(\frac{n}{n-1}b\right)^{n-1}$$

最大化得:

$$b^*(v) = \frac{n-1}{n} v$$

### 收益等价定理

**定理**: 在满足以下条件的任何标准拍卖中：
1. 竞拍者风险中性
2. 估值独立同分布
3. 竞拍者对称
4. 拍卖给最高估值者

**预期收益相同**。

**证明思路**: 

任何竞拍者的预期支付仅取决于其分配概率和估值分布，与具体拍卖形式无关。

## 求解过程

### 步骤1：Myerson最优拍卖设计

**目标**: 最大化卖方预期收益

**机制设计要素**:
- 分配规则 $x_i(v)$: $i$ 获得物品的概率
- 支付规则 $p_i(v)$: $i$ 的预期支付

**约束**:
1. **激励相容(IC)**: 说真话最优
2. **个体理性(IR)**: 参与非负效用
3. **可行性**: $\sum_i x_i(v) \leq 1$

### 步骤2：虚拟估值

**定义**: 竞拍者 $i$ 的虚拟估值

$$\psi_i(v_i) = v_i - \frac{1 - F_i(v_i)}{f_i(v_i)}$$

其中 $f_i$ 是概率密度函数。

**Myerson定理**: 最优拍卖将物品分配给虚拟估值最高的竞拍者（若为负则保留）。

### 步骤3：带保留价的最优拍卖

**标准拍卖+最优保留价**:

对于均匀分布 $v_i \sim U[0, 1]$，最优保留价:

$$r^* = \frac{1 - F(r^*)}{f(r^*)} = \frac{1 - r^*}{1} \Rightarrow r^* = \frac{1}{2}$$

**收益提升**: 相比无保留价，收益增加。

## 结果分析

### 不同拍卖比较

| 拍卖形式 | 策略复杂度 | 效率 | 卖方收益 | 实际应用 |
|----------|-----------|------|----------|----------|
| **英国式** | 简单（占优策略） | 有效 | 等价 | 艺术品、古董 |
| **荷兰式** | 需预测他人 | 有效 | 等价 | 鲜花、鱼 |
| **一级密封** | 需均衡策略 | 有效 | 等价 | 政府采购 |
| **二级密封** | 简单（占优策略） | 有效 | 等价 | 在线广告 |

### 在线广告应用

**广义第二价格(GSP)拍卖**:

- 多个广告位
- 第 $i$ 位获胜者支付第 $(i+1)$ 位的出价
- 实际使用（Google、Bing）

**与VCG比较**:
- GSP: 简单易行，但不一定激励相容
- VCG: 激励相容，但复杂、收入可能低

### 频谱拍卖

**组合拍卖挑战**:
- 竞拍者对频率组合有估值
- 维克里拍卖的支付计算困难（NP-hard）

**实用机制**:
- 同时多轮拍卖(SMRA)
- 组合时钟拍卖(CCA)
