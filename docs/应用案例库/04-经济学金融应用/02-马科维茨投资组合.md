# 投资组合优化的马科维茨模型

## 应用领域

**学科**: 金融经济学 / 运筹学  
**具体应用**: 资产配置、养老基金管理、风险平价策略、智能投顾  
**MSC分类**: 91G10 (Portfolio theory), 90C20 (Quadratic programming), 62P05 (Statistical applications in actuarial sciences)

---

## 数学概念

### 核心数学工具

1. **均值-方差分析**: 收益与风险的权衡
2. **二次规划**: 凸优化问题
3. **有效前沿**: 最优投资组合集合
4. **资本资产定价模型(CAPM)**: 均衡定价
5. **协方差矩阵**: 资产间相关性

### 关键定义

- **预期收益**: $\mu = \mathbb{E}[R]$
- **风险(标准差)**: $\sigma = \sqrt{\text{Var}(R)}$
- **夏普比率**: $(R_p - R_f) / \sigma_p$

---

## 问题描述

### 投资组合选择问题

给定 $n$ 个风险资产，如何配置权重 $w = (w_1, ..., w_n)^T$ 以：
- 最大化预期收益
- 最小化风险（收益方差）

### 双目标优化

**目标1**: 最大化预期收益 $\mu_p = w^T \mu$

**目标2**: 最小化风险 $\sigma_p^2 = w^T \Sigma w$

其中:
- $\mu \in \mathbb{R}^n$: 资产预期收益率向量
- $\Sigma \in \mathbb{R}^{n \times n}$: 收益率协方差矩阵（对称正定）

---

## 数学模型

### 基本优化问题

**问题1: 给定目标收益，最小化风险**

$$\boxed{\min_w \frac{1}{2} w^T \Sigma w}$$

**约束**:
$$w^T \mu = \mu_{target}$$
$$w^T \mathbf{1} = 1$$

**问题2: 给定风险上限，最大化收益**

$$\max_w w^T \mu$$

**约束**:
$$w^T \Sigma w \leq \sigma_{max}^2$$
$$w^T \mathbf{1} = 1$$

**问题3: 风险厌恶参数化**

$$\max_w \left(w^T \mu - \frac{\lambda}{2} w^T \Sigma w\right)$$

其中 $\lambda > 0$ 是风险厌恶系数。

### 有效前沿

**定义**: 在所有可行投资组合中，给定风险水平下收益最高，或给定收益水平下风险最小的投资组合集合。

**解析解**（两基金定理）:

设 $A = \mathbf{1}^T \Sigma^{-1} \mathbf{1}$, $B = \mu^T \Sigma^{-1} \mathbf{1}$, $C = \mu^T \Sigma^{-1} \mu$

全局最小方差组合:

$$w_g = \frac{\Sigma^{-1} \mathbf{1}}{A}, \quad \sigma_g^2 = \frac{1}{A}$$

有效前沿上的组合:

$$w^* = \Sigma^{-1}(\alpha \mathbf{1} + \beta \mu)$$

其中 $\alpha, \beta$ 由约束条件确定。

**前沿方程**（双曲线）:

$$\sigma_p^2 = \frac{1}{AC - B^2}(A\mu_p^2 - 2B\mu_p + C)$$

---

## 求解过程

### 步骤1：拉格朗日方法

对于问题1，构造拉格朗日函数:

$$\mathcal{L} = \frac{1}{2} w^T \Sigma w - \lambda(w^T \mu - \mu_{target}) - \gamma(w^T \mathbf{1} - 1)$$

**KKT条件**:

$$\frac{\partial \mathcal{L}}{\partial w} = \Sigma w - \lambda \mu - \gamma \mathbf{1} = 0$$

解得:

$$w = \Sigma^{-1}(\lambda \mu + \gamma \mathbf{1})$$

由约束确定 $\lambda, \gamma$:

$$\begin{pmatrix} C & B \\ B & A \end{pmatrix} \begin{pmatrix} \lambda \\ \gamma \end{pmatrix} = \begin{pmatrix} \mu_{target} \\ 1 \end{pmatrix}$$

### 步骤2：引入无风险资产

**切线组合**: 与资本配置线(CAL)相切的风险资产组合。

**最优风险组合**:

$$w_t = \frac{\Sigma^{-1}(\mu - r_f \mathbf{1})}{\mathbf{1}^T \Sigma^{-1}(\mu - r_f \mathbf{1})}$$

**资本配置线**:

$$\mu_p = r_f + \frac{\mu_t - r_f}{\sigma_t} \sigma_p$$

斜率是夏普比率的最大值。

### 步骤3：数值优化（含约束）

**实际约束**:
- 不允许卖空: $w_i \geq 0$
- 权重上限: $w_i \leq u_i$
- 行业约束: 某些行业权重和约束
- 换手率限制

**二次规划求解**:

$$\min_w \frac{1}{2} w^T \Sigma w - q^T w$$

**约束**:
$$Gw \leq h$$
$$Aw = b$$

使用内点法或活动集法求解。

## 结果分析

### 分散化效应

**组合风险**:

$$\sigma_p^2 = \sum_{i=1}^n w_i^2 \sigma_i^2 + \sum_{i \neq j} w_i w_j \sigma_{ij}$$

当资产不相关且等权重:

$$\sigma_p = \frac{\bar{\sigma}}{\sqrt{n}} \to 0 \text{ as } n \to \infty$$

**分散化降低非系统性风险**。

### 均值-方差悖论

**估计误差**: 
- 样本均值估计误差大
- 协方差矩阵估计需要大量数据

**改进方法**:
- **收缩估计**: Ledoit-Wolf收缩
- **因子模型**: 降低参数数量
- **Bayes方法**: 引入先验
- **稳健优化**: 考虑不确定性集合

### 实际应用

| 策略 | 方法 | 特点 |
|------|------|------|
| **最小方差** | 忽略预期收益 | 对估计误差稳健 |
| **等权重** | $w_i = 1/n$ | 简单、分散化好 |
| **风险平价** | 各资产风险贡献相等 | 真正的分散化 |
| **最大分散化** | 最大化分散化比率 | 等风险权重 |

### 风险平价组合

**风险贡献**:

$$RC_i = w_i \frac{\partial \sigma_p}{\partial w_i} = \frac{w_i (\Sigma w)_i}{\sigma_p}$$

**风险平价条件**: $RC_i = RC_j$ 对所有 $i, j$

这转化为非线性方程组，需数值求解。
