# 强化学习基础

## 概述
- **应用领域**: 人工智能/自动控制/游戏
- **数学分支**: 随机过程/动态规划/优化
- **难度级别**: 中级（研究生）
- **关键词**: MDP、贝尔曼方程、Q学习、策略梯度、探索-利用

---

## 问题背景

### 核心问题

如何在未知环境中学习最优行为策略，通过试错获得最大累积奖励？

**应用场景**:
- 游戏AI (AlphaGo, Atari)
- 机器人控制
- 自动驾驶
- 资源调度
- 推荐系统

### 与监督学习的区别

| 特征 | 监督学习 | 强化学习 |
|------|----------|----------|
| 数据 | 静态数据集 | 交互生成 |
| 反馈 | 即时标签 | 延迟奖励 |
| 探索 | 无 | 需要 |
| 序列性 | 独立样本 | 状态依赖 |

## 数学建模

### 马尔可夫决策过程 (MDP)

**五元组**: $\mathcal{M} = (\mathcal{S}, \mathcal{A}, P, R, \gamma)$

- $\mathcal{S}$: 状态空间
- $\mathcal{A}$: 动作空间  
- $P(s'|s,a)$: 转移概率

- $R(s,a)$: 奖励函数
- $\gamma \in [0,1]$: 折扣因子

**马尔可夫性质**:
$$P(s_{t+1}|s_t, a_t, s_{t-1}, a_{t-1}, \ldots) = P(s_{t+1}|s_t, a_t)$$

### 策略与值函数

**策略**: $\pi(a|s)$ 状态下选择动作的概率分布

**状态值函数**:
$$V^\pi(s) = \mathbb{E}\left[\sum_{t=0}^\infty \gamma^t R(s_t, a_t) \bigg| s_0 = s, \pi\right]$$

**动作值函数 (Q函数)**:
$$Q^\pi(s, a) = \mathbb{E}\left[\sum_{t=0}^\infty \gamma^t R(s_t, a_t) \bigg| s_0 = s, a_0 = a, \pi\right]$$

**优势函数**:
$$A^\pi(s, a) = Q^\pi(s, a) - V^\pi(s)$$

## 数学方法

### 贝尔曼方程

**最优值函数**:
$$V^*(s) = \max_a \left[ R(s, a) + \gamma \sum_{s'} P(s'|s, a) V^*(s') \right]$$

**最优Q函数**:
$$Q^*(s, a) = R(s, a) + \gamma \sum_{s'} P(s'|s, a) \max_{a'} Q^*(s', a')$$

**贝尔曼最优算子** (压缩映射):
$$(\mathcal{T}V)(s) = \max_a \left[ R(s, a) + \gamma \sum_{s'} P(s'|s, a) V(s') \right]$$

### 动态规划方法

**值迭代**:
$$V_{k+1}(s) = \max_a \left[ R(s, a) + \gamma \sum_{s'} P(s'|s, a) V_k(s') \right]$$

收敛到 $V^*$，收敛速率 $O(\gamma^k)$。

**策略迭代**:
1. 策略评估: 解线性方程组求 $V^\pi$
2. 策略改进: $\pi'(s) = \arg\max_a Q^\pi(s, a)$
3. 保证收敛到最优策略

### 无模型方法

**Q学习** (Watkins, 1989):
$$Q(s, a) \leftarrow Q(s, a) + \alpha \left[ r + \gamma \max_{a'} Q(s', a') - Q(s, a) \right]$$

收敛保证: 在适当步长下收敛到 $Q^*$。

**SARSA** (on-policy):
$$Q(s, a) \leftarrow Q(s, a) + \alpha \left[ r + \gamma Q(s', a') - Q(s, a) \right]$$

### 策略梯度方法

**策略梯度定理**:
$$\nabla_\theta J(\theta) = \mathbb{E}_{\pi_\theta}\left[ \nabla_\theta \log \pi_\theta(a|s) \cdot Q^{\pi_\theta}(s, a) \right]$$

**REINFORCE算法**:
$$\theta \leftarrow \theta + \alpha \sum_t \nabla_\theta \log \pi_\theta(a_t|s_t) \cdot G_t$$

其中 $G_t = \sum_{k=t}^T \gamma^{k-t} r_k$ 是回报。

**Actor-Critic**:
- Actor: 参数化策略 $\pi_\theta$
- Critic: 估计值函数 $V_w$ 或 $Q_w$
- 用优势函数减少方差

## 求解过程

### 探索-利用困境

**$\epsilon$-贪心**:
- 以概率 $\epsilon$ 随机探索
- 以概率 $1-\epsilon$ 选择最优动作

**UCB (上置信界)**:
$$a_t = \arg\max_a \left[ \hat{Q}(s, a) + c\sqrt{\frac{\ln t}{N(s, a)}} \right]$$

理论保证: 次线性后悔。

**Thompson采样**:
- 维护奖励分布的后验
- 从后验采样选择动作

### 函数近似

**线性近似**:
$$\hat{Q}(s, a; w) = w^T \phi(s, a)$$

**神经网络 (DQN)**:
- 参数化 $Q(s, a; \theta)$
- 经验回放打破相关性
- 目标网络稳定学习

**损失函数**:
$$L(\theta) = \mathbb{E}_{(s,a,r,s') \sim D}\left[ \left( r + \gamma \max_{a'} Q(s', a'; \theta^-) - Q(s, a; \theta) \right)^2 \right]$$

## 结果分析

### 收敛性理论

**值迭代**: 指数收敛
$$\|V_k - V^*\|_\infty \leq \gamma^k \|V_0 - V^*\|_\infty$$

**Q学习**: 概率收敛
在递减步长 $\alpha_t$ 下收敛到最优。

**策略梯度**: 收敛到局部最优（非凸优化）

### 样本复杂度

| 方法 | 样本复杂度 | 适用场景 |
|------|------------|----------|
| 模型-based | $O(SA/\epsilon^2)$ | 小状态空间 |
| Q学习 | $O(SA/\epsilon^2(1-\gamma)^3)$ | 表格型 |
| 函数近似 | 多项式/指数 | 连续空间 |

## 拓展思考

### 深度强化学习

**关键突破**:
- DQN (2013): Atari游戏超人类
- A3C (2016): 异步并行
- PPO (2017): 稳定策略优化
- SAC (2018): 最大熵框架

**挑战**:
- 样本效率低
- 训练不稳定
- 泛化能力弱

### 多智能体强化学习

**博弈论框架**:
- 随机博弈 (Markov game)
- Nash均衡 vs 社会最优
- 涌现合作与竞争

### 与现实世界应用

**Sim-to-real迁移**:
- 领域随机化
- 域适应
- 安全约束
