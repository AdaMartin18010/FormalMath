# 支持向量机理论

## 概述
- **应用领域**: 机器学习/模式识别
- **数学分支**: 凸优化/泛函分析
- **难度级别**: 中级（研究生）
- **关键词**: SVM、核方法、凸优化、间隔最大化、VC维

---

## 问题背景

### 分类问题

给定训练数据 $\{(x_i, y_i)\}_{i=1}^n$，其中 $x_i \in \mathbb{R}^d$, $y_i \in \{-1, +1\}$，找到决策函数 $f(x)$ 对新样本正确分类。

### SVM的核心思想

**最大间隔分类器**: 不仅要求正确分类，还要最大化分类边界与最近样本的距离。

**优势**:
- 泛化能力强
- 非线性通过核方法
- 解是稀疏的（仅依赖支持向量）

---

## 数学建模

### 线性可分情况

**硬间隔SVM**:

寻找超平面 $w^T x + b = 0$，约束:
$$y_i(w^T x_i + b) \geq 1, \quad \forall i$$

**优化问题**:
$$\min_{w, b} \frac{1}{2}\|w\|^2$$

$$\text{s.t. } y_i(w^T x_i + b) \geq 1, \quad i = 1, \ldots, n$$

**几何解释**: 间隔为 $\frac{2}{\|w\|}$

### 不可分情况

**软间隔SVM**:

引入松弛变量 $\xi_i \geq 0$:
$$y_i(w^T x_i + b) \geq 1 - \xi_i$$

**优化问题**:
$$\min_{w, b, \xi} \frac{1}{2}\|w\|^2 + C \sum_{i=1}^n \xi_i$$

$$\text{s.t. } y_i(w^T x_i + b) \geq 1 - \xi_i, \quad \xi_i \geq 0$$

参数 $C$ 控制间隔与误分类的权衡。

---

## 数学方法

### 拉格朗日对偶

**原始拉格朗日**:
$$L(w, b, \alpha) = \frac{1}{2}\|w\|^2 - \sum_{i=1}^n \alpha_i[y_i(w^T x_i + b) - 1]$$

**KKT条件**:
$$\nabla_w L = w - \sum_i \alpha_i y_i x_i = 0 \Rightarrow w = \sum_i \alpha_i y_i x_i$$
$$\frac{\partial L}{\partial b} = \sum_i \alpha_i y_i = 0$$

**对偶问题**:
$$\max_\alpha \sum_{i=1}^n \alpha_i - \frac{1}{2}\sum_{i,j} \alpha_i \alpha_j y_i y_j x_i^T x_j$$
$$\text{s.t. } \sum_i \alpha_i y_i = 0, \quad 0 \leq \alpha_i \leq C$$

### 核方法

**特征映射**: $\phi: \mathcal{X} \rightarrow \mathcal{H}$ (高维特征空间)

**核技巧**: 不计算 $\phi(x)$，直接计算内积
$$K(x, x') = \langle \phi(x), \phi(x') \rangle$$

**常用核函数**:

| 核函数 | 表达式 | 参数 |
|--------|--------|------|
| 多项式 | $K(x, x') = (x^T x' + c)^d$ | $d$: 次数 |
| RBF (高斯) | $K(x, x') = \exp(-\gamma\|x - x'\|^2)$ | $\gamma$: 宽度 |
| Sigmoid | $K(x, x') = \tanh(\kappa x^T x' + \theta)$ | - |

**Mercer定理**: 核函数 $K$ 有效当且仅当对任意样本 Gram矩阵半正定。

### 求解算法

**SMO (Sequential Minimal Optimization)**:
- 每次优化两个拉格朗日乘子
- 解析解，无需数值优化
- 收敛快，内存效率高

**坐标下降**:
- 迭代优化单变量
- 适合大规模数据

---

## 求解过程

### 决策函数

$$f(x) = \text{sign}\left(\sum_{i=1}^n \alpha_i y_i K(x_i, x) + b\right)$$

**支持向量**: 满足 $\alpha_i > 0$ 的样本
- 位于间隔边界上或误分类
- 决定决策边界
- 通常稀疏（远少于训练样本）

### 间隔计算

**几何间隔**:
$$\gamma = \frac{1}{\|w\|} = \frac{1}{\sqrt{\sum_{i,j} \alpha_i \alpha_j y_i y_j K(x_i, x_j)}}$$

**泛化误差界** (VC理论):
以高概率:
$$R(f) \leq R_{emp}(f) + O\left(\sqrt{\frac{d_{VC}}{n}}\right)$$

SVM最小化 VC维上界。

---

## 结果分析

### 统计学习理论

**结构风险最小化**:
$$R(f) \leq R_{emp}(f) + \text{复杂度惩罚}$$

SVM通过最大化间隔控制模型复杂度。

**Rademacher复杂度**:
$$\mathcal{R}_n(\mathcal{F}) = \mathbb{E}_\sigma \left[\sup_{f \in \mathcal{F}} \frac{1}{n}\sum_{i=1}^n \sigma_i f(x_i)\right]$$

间隔大的函数类复杂度低。

### 实际考虑

| 问题 | 解决方案 |
|------|----------|
| 类别不平衡 | 类别权重、SMOTE |
| 多分类 | 一对一、一对其余 |
| 大规模数据 | SGD、线性SVM近似 |
| 核选择 | 交叉验证、MKL |

## 拓展思考

### 扩展变体

**支持向量回归 (SVR)**:
$$\min \frac{1}{2}\|w\|^2 + C \sum_i L_\epsilon(y_i - f(x_i))$$

其中 $L_\epsilon$ 是 $\epsilon$-不敏感损失。

**单类SVM**: 异常检测

**结构化SVM**: 结构化输出预测

### 与现代深度学习的联系

**神经正切核** (NTK):
- 宽神经网络等价于核方法
- SVM是特定核的极限情况

**核与深度网络比较**:
- 核: 固定特征，凸优化
- 深度网络: 学习特征，非凸
