# 量子纠缠与贝尔不等式

## 应用领域

**学科**: 量子信息 / 量子基础
**具体应用**: 量子密码学、量子计算、量子非定域性验证
**MSC分类**: 81P40 (Quantum entanglement), 81P15 (Quantum measurement theory)

---

## 数学概念

### 核心数学工具

1. **张量积空间**: 复合量子系统的Hilbert空间
2. **密度矩阵**: 混合态与纯态的统描述
3. **纠缠态**: 不可分解的量子关联
4. **贝尔不等式**: 经典相关性的边界

### 关键定义

- **纠缠态**: $|\psi\rangle$ 不能写成 $|\phi_1\rangle \otimes |\phi_2\rangle$ 形式

- **贝尔态**: 四个最大纠缠态（单态和三重态）

---

## 问题描述

### EPR佯谬背景

Einstein, Podolsky, Rosen (1935) 质疑量子力学的完备性：

- **定域性**: 对系统的测量不应瞬间影响遥远系统
- **实在性**: 物理实在的元素应在测量前就存在

**贝尔的贡献** (1964): 将哲学争论转化为可实验检验的数学不等式。

---

## 数学模型

### 贝尔实验设置

**系统**: 两个纠缠的光子，分别发送给Alice和Bob

**测量设置**:

- Alice选择测量方向 $\mathbf{a}$ 或 $\mathbf{a}'$
- Bob选择测量方向 $\mathbf{b}$ 或 $\mathbf{b}'$

**测量结果**: $A, B \in \{+1, -1\}$

### CHSH不等式（贝尔不等式的推广）

**定义关联函数**:

$$E(\mathbf{a}, \mathbf{b}) = \langle AB \rangle = \sum_{A,B} A \cdot B \cdot P(A,B|\mathbf{a},\mathbf{b})$$

**CHSH量**:

$$S = E(\mathbf{a}, \mathbf{b}) + E(\mathbf{a}, \mathbf{b}') + E(\mathbf{a}', \mathbf{b}) - E(\mathbf{a}', \mathbf{b}')$$

**贝尔不等式**（经典/定域实在论界限）:

$$|S| \leq 2$$

### 量子力学预测

对于单态 $|\psi^-\rangle = \frac{1}{\sqrt{2}}(|01\rangle - |10\rangle)$:

$$E_Q(\mathbf{a}, \mathbf{b}) = -\mathbf{a} \cdot \mathbf{b} = -\cos\theta$$

**最优角度选择**: $\theta_{ab} = \theta_{ab'} = \theta_{a'b} = 22.5°$, $\theta_{a'b'} = 67.5°$

**量子预测**:

$$S_Q = 2\sqrt{2} \approx 2.828$$

**违反贝尔不等式**！

---

## 求解过程

### 步骤1：隐变量理论的形式化

**隐变量假设**: 存在变量 $\lambda$ 决定测量结果

$$A(\mathbf{a}, \lambda) = \pm 1, \quad B(\mathbf{b}, \lambda) = \pm 1$$

**定域性条件**:

$$A(\mathbf{a}, \lambda) \text{ 不依赖于 } \mathbf{b}$$
$$B(\mathbf{b}, \lambda) \text{ 不依赖于 } \mathbf{a}$$

### 步骤2：推导CHSH界限

$$S = \int d\lambda \, \rho(\lambda) [A(\mathbf{a},\lambda)B(\mathbf{b},\lambda) + A(\mathbf{a},\lambda)B(\mathbf{b}',\lambda) + A(\mathbf{a}',\lambda)B(\mathbf{b},\lambda) - A(\mathbf{a}',\lambda)B(\mathbf{b}',\lambda)]$$

**关键观察**: 对任意 $\lambda$:

$$|A(\mathbf{a},\lambda)(B(\mathbf{b},\lambda) + B(\mathbf{b}',\lambda)) + A(\mathbf{a}',\lambda)(B(\mathbf{b},\lambda) - B(\mathbf{b}',\lambda))| \leq 2$$

因为 $B(\mathbf{b},\lambda), B(\mathbf{b}',\lambda) \in \{+1,-1\}$，所以要么 $B(\mathbf{b}) = B(\mathbf{b}')$，要么 $B(\mathbf{b}) = -B(\mathbf{b}')$。

**结论**: $|S| \leq 2$

### 步骤3：量子计算

对于单态，自旋关联：

$$\langle \sigma_\mathbf{a} \otimes \sigma_\mathbf{b} \rangle = -\cos\theta_{ab}$$

取:

- $\mathbf{a}$: x轴方向 (0°)
- $\mathbf{a}'$: 45°方向
- $\mathbf{b}$: 22.5°方向
- $\mathbf{b}'$: -22.5°方向

计算:

- $E(\mathbf{a}, \mathbf{b}) = -\cos(22.5°) = -\frac{\sqrt{2+\sqrt{2}}}{2}$
- $E(\mathbf{a}, \mathbf{b}') = -\cos(-22.5°) = -\frac{\sqrt{2+\sqrt{2}}}{2}$
- $E(\mathbf{a}', \mathbf{b}) = -\cos(-22.5°) = -\frac{\sqrt{2+\sqrt{2}}}{2}$
- $E(\mathbf{a}', \mathbf{b}') = -\cos(-67.5°) = \frac{\sqrt{2-\sqrt{2}}}{2}$

$$S_Q = -3 \cdot \frac{\sqrt{2+\sqrt{2}}}{2} + \frac{\sqrt{2-\sqrt{2}}}{2} = 2\sqrt{2}$$

## 结果分析

### 实验验证

| 实验 | 年份 | 结果 | 意义 |
|------|------|------|------|
| Aspect等 | 1982 | $S = 2.697 \pm 0.015$ | 首次严格验证 |
| Weihs等 | 1998 | 关闭局域性漏洞 | 随机测量选择 |
| Hensen等 | 2015 | 关闭探测漏洞 | Loophole-free |
| BIG Bell Test | 2016 | 人类随机选择 | 自由意志检验 |

### 物理意义

1. **量子非定域性**: 量子关联不能由任何定域隐变量理论解释
2. **无信号原理**: 虽然不能超光速通信，但关联是瞬时的
3. **设备无关量子密码**: 贝尔违反保证安全性，无需信任设备

### Tsirelson界限

**定理**: 对于量子力学，$|S| \leq 2\sqrt{2}$

这是C*代数结构的约束，非定域性存在理论上限。
