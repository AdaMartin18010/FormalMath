# 神经网络动力学

## 概述
- **应用领域**: 神经科学/计算生物学
- **数学分支**: 动力系统/微分方程
- **难度级别**: 高级（研究生）
- **关键词**: 神经网络、动力系统、平衡点、分岔、振荡

---

## 问题背景

### 神经元数学模型

从Hodgkin-Huxley模型到简化的现象学模型，数学帮助理解大脑信息处理的机制。

**核心问题**:
- 神经元如何产生动作电位？
- 神经网络如何产生集体节律？
- 记忆如何编码在网络连接中？

### 模型层次

| 层次 | 模型 | 变量 | 应用 |
|------|------|------|------|
| 离子通道 | Hodgkin-Huxley | 膜电位+门控变量 | 精确模拟 |
| 神经元 | Integrate-and-fire | 膜电位 | 大规模网络 |
| 群体 | 发放率模型 | 平均活动 | 认知建模 |

## 数学建模

### Hodgkin-Huxley模型 (1952)

**膜电位动力学**:
$$C_m \frac{dV}{dt} = -g_{Na}m^3h(V - E_{Na}) - g_K n^4(V - E_K) - g_L(V - E_L) + I_{ext}$$

**门控变量**:
$$\frac{dx}{dt} = \alpha_x(V)(1-x) - \beta_x(V)x, \quad x \in \{m, h, n\}$$

**生理学意义**:
- $m$: 钠通道激活
- $h$: 钠通道失活  
- $n$: 钾通道激活

### 发放率模型

**Wilson-Cowan方程**:
$$\tau_E \frac{dE}{dt} = -E + f_E(w_{EE}E - w_{EI}I + I_E)$$
$$\tau_I \frac{dI}{dt} = -I + f_I(w_{IE}E - w_{II}I + I_I)$$

其中 $f$ 是S型激活函数:
$$f(x) = \frac{1}{1 + e^{-\beta(x - \theta)}}$$

## 数学方法

### 分岔分析

**分岔**: 系统定性行为随参数变化而突变

**神经元中的典型分岔**:
- **鞍结分岔**: 平衡点出现/消失 → 激发阈值
- **Hopf分岔**: 平衡点失稳，产生极限环 → 振荡
- **同宿分岔**: 极限环与鞍点碰撞

**兴奋性类型**:
- **Type I**: 鞍结分岔在不变圆上 (SNIC)
  - 任意频率振荡
  - 例子: 皮层锥体神经元
  
- **Type II**: Hopf分岔
  - 非零最小频率
  - 例子: 小脑浦肯野细胞

### 同步理论

**Kuramoto模型**:
$$\frac{d\theta_i}{dt} = \omega_i + \frac{K}{N} \sum_{j=1}^N \sin(\theta_j - \theta_i)$$

**序参量** (order parameter):
$$r(t)e^{i\psi(t)} = \frac{1}{N} \sum_{j=1}^N e^{i\theta_j(t)}$$

- $r = 0$: 异步状态
- $r > 0$: 同步状态

**临界耦合强度**:
$$K_c = \frac{2}{\pi g(0)}$$

其中 $g(\omega)$ 是自然频率分布。

### 平均场理论

**随机神经网络**:
$$\tau \frac{dh_i}{dt} = -h_i + \sum_j J_{ij}\phi(h_j) + I_i$$

**混沌边缘**: 当连接强度达到临界值，网络进入混沌态，与计算能力相关。

## 求解过程

### 数值方法

**积分方法**:
- Euler: 简单但稳定性差
- Runge-Kutta: 精度高
- 事件驱动 (Integrate-and-fire): 精确检测阈值穿越

**分岔分析工具**:
- AUTO: 延续分岔图
- XPPAUT: 神经动力学专用

### 稳定性分析

**线性化**:
$$\frac{d\mathbf{x}}{dt} = J \mathbf{x}, \quad J_{ij} = \frac{\partial f_i}{\partial x_j}\bigg|_{x^*}$$

**特征值分析**:
- 全负实部: 稳定节点
- 正实部: 不稳定
- 虚部非零: 振荡

## 结果分析

### 计算特性

| 特性 | 数学机制 | 功能意义 |
|------|----------|----------|
| 发放 | 极限环 | 信息编码 |
| 双稳态 | 两个稳定平衡点 | 短时记忆 |
| 混沌 | 奇怪吸引子 | 复杂计算 |
| 同步 | 耦合振荡器 | 信息整合 |

### 与认知的关联

**决策模型** (Drift-diffusion model):
$$dx = \mu dt + \sigma dW$$

决策边界交叉时间对应反应时间。

**工作记忆**: 持续性活动源于再生反馈

## 拓展思考

### 深度学习与神经科学

**相似性**:
- 层级表征
- 误差反向传播 vs 生物可塑性

**差异**: 生物神经网络使用局部学习规则

### 神经形态计算

**脉冲神经网络** (SNN):
- 事件驱动，低功耗
- 时间编码信息
- 芯片实现: TrueNorth, Loihi

### 开放问题

- 意识的全局工作空间理论数学化
- 自由能原理的统一框架
- 神经代码: 发放率 vs 时间编码
