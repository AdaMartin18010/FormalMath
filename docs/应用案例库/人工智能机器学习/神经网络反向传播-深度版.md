## 1. 历史背景与动机

### 1.1 神经网络的发展简史

**1943**: McCulloch-Pitts神经元模型 — 神经网络的数学基础
**1958**: Rosenblatt的感知机 (Perceptron) — 首个可学习的神经网络
**1969**: Minsky & Papert《Perceptrons》— 指出单层网络的局限性，引发第一次AI寒冬
**1986**: **Rumelhart, Hinton, Williams**发表反向传播算法 — 突破多层网络训练瓶颈
**2006**: Hinton的深度信念网络预训练 — 深度学习复兴
**2012**: AlexNet在ImageNet获胜 — 深度学习爆发

### 1.2 反向传播的历史意义

在反向传播之前，多层神经网络训练面临**信用分配问题** (Credit Assignment Problem)：如何确定网络中每个权重对最终误差的贡献？

> **Rumelhart等人 (1986) 的关键洞察**: 利用**链式法则** (Chain Rule) 高效计算梯度，将输出层误差逐层反向传播到输入层。

这一突破使得训练具有隐藏层的神经网络成为可能，为现代深度学习奠定了算法基础。

## 2. 多层感知机 (MLP) 的数学描述

### 2.1 网络结构

**定义 2.1 (多层感知机)**: 具有 $L$ 层的MLP定义为：

- **输入层**: $a^{[0]} = x \in \mathbb{R}^{n_0}$
- **隐藏层** $l = 1, ..., L-1$:
  $$z^{[l]} = W^{[l]} a^{[l-1]} + b^{[l]}$$
  $$a^{[l]} = \sigma(z^{[l]})$$
- **输出层** $L$:
  $$z^{[L]} = W^{[L]} a^{[L-1]} + b^{[L]}$$
  $$\hat{y} = a^{[L]} = g(z^{[L]})$$

其中：

- $W^{[l]} \in \mathbb{R}^{n_l \times n_{l-1}}$: 第 $l$ 层权重矩阵
- $b^{[l]} \in \mathbb{R}^{n_l}$: 第 $l$ 层偏置向量
- $\sigma$: 隐藏层激活函数 (ReLU, sigmoid, tanh等)
- $g$: 输出层激活函数 (softmax, sigmoid, 线性等)
- $n_l$: 第 $l$ 层神经元数量

**参数总量**: $P = \sum_{l=1}^L (n_l \cdot n_{l-1} + n_l)$

### 2.2 激活函数

**定义 2.2 (常用激活函数)**:

| 函数 | 公式 | 导数 | 特点 |
|------|------|------|------|
| **Sigmoid** | $\sigma(z) = \frac{1}{1+e^{-z}}$ | $\sigma'(z) = \sigma(z)(1-\sigma(z))$ | 输出(0,1)，梯度消失问题 |
| **Tanh** | $\tanh(z) = \frac{e^z - e^{-z}}{e^z + e^{-z}}$ | $1 - \tanh^2(z)$ | 输出(-1,1)，零中心化 |
| **ReLU** | $\text{ReLU}(z) = \max(0, z)$ | $1_{z>0}$ | 计算高效，缓解梯度消失 |
| **Leaky ReLU** | $\max(\alpha z, z)$ | $\alpha$ (if $z<0$), 1 (if $z>0$) | 解决ReLU死亡问题 |
| **Softmax** | $\frac{e^{z_i}}{\sum_j e^{z_j}}$ | 复杂 (见下文) | 多分类输出层 |

**Softmax的导数**: 对于向量输出，Jacobian矩阵为：

$$\frac{\partial \text{softmax}(z)_i}{\partial z_j} = \text{softmax}(z)_i (\delta_{ij} - \text{softmax}(z)_j)$$

## 3. 损失函数构造

### 3.1 回归问题：均方误差 (MSE)

**定义 3.1 (MSE损失)**: 对于回归任务：

$$\mathcal{L}_{MSE} = \frac{1}{N} \sum_{i=1}^N \|\hat{y}_i - y_i\|^2 = \frac{1}{N} \sum_{i=1}^N \sum_{k=1}^{n_L} (\hat{y}_{i,k} - y_{i,k})^2$$

**梯度**:
$$\frac{\partial \mathcal{L}_{MSE}}{\partial \hat{y}} = \frac{2}{N}(\hat{y} - y)$$

### 3.2 分类问题：交叉熵损失

**定义 3.2 (二元交叉熵)**:

对于二分类问题：

$$\mathcal{L}_{BCE} = -\frac{1}{N}\sum_{i=1}^N \left[y_i \log(\hat{y}_i) + (1-y_i)\log(1-\hat{y}_i)\right]$$

其中 $\hat{y}_i = \sigma(z_i^{[L]}) \in (0, 1)$。

**定义 3.3 (多类交叉熵 / Softmax损失)**:

对于 $K$ 类分类问题：

$$\mathcal{L}_{CE} = -\frac{1}{N}\sum_{i=1}^N \sum_{k=1}^K y_{i,k} \log(\hat{y}_{i,k})$$

其中 $\hat{y}_{i,k} = \text{softmax}(z_i^{[L]})_k$。

**关键性质**: 当Softmax与交叉熵结合时，输出层梯度**简化**为：

$$\frac{\partial \mathcal{L}_{CE}}{\partial z^{[L]}} = \hat{y} - y$$

这一简化是深度学习实现高效训练的重要技巧。

### 3.3 损失函数的性质

**定理 3.4 (损失函数的凸性)**:

- MSE损失在 $\hat{y}$ 上是**凸**的
- 交叉熵损失在 $z^{[L]}$ 上是**凸**的（但整个网络损失通常非凸）

**引理 3.5**: 对于单层线性网络，MSE损失是凸函数。但对于多层非线性网络，损失函数通常具有**多个局部极小值**。

## 4. 链式法则与反向传播的完整推导

### 4.1 链式法则回顾

**单变量链式法则**: 若 $y = f(g(x))$，则

$$\frac{dy}{dx} = \frac{df}{dg} \cdot \frac{dg}{dx}$$

**多变量链式法则**: 若 $y = f(u_1, ..., u_m)$，$u_i = g_i(x_1, ..., x_n)$，则

$$\frac{\partial y}{\partial x_j} = \sum_{i=1}^m \frac{\partial y}{\partial u_i} \cdot \frac{\partial u_i}{\partial x_j}$$

**Jacobian矩阵**: 对于向量值函数 $\mathbf{f}: \mathbb{R}^n \to \mathbb{R}^m$：

$$J_{ij} = \frac{\partial f_i}{\partial x_j}$$

链式法则的矩阵形式：$J_{\mathbf{f} \circ \mathbf{g}} = J_\mathbf{f} \cdot J_\mathbf{g}$

### 4.2 反向传播算法推导

**目标**: 计算损失函数对所有参数的梯度 $\frac{\partial \mathcal{L}}{\partial W^{[l]}}$ 和 $\frac{\partial \mathcal{L}}{\partial b^{[l]}}$。

#### 步骤1: 前向传播 (Forward Pass)

对于每个样本 $x$：

```
a^[0] = x
for l = 1 to L:
    z^[l] = W^[l] a^[l-1] + b^[l]
    a^[l] = activation(z^[l])
```

保存所有中间结果 $z^{[l]}$ 和 $a^{[l]}$。

#### 步骤2: 计算输出层误差

定义**误差信号** $\delta^{[l]} = \frac{\partial \mathcal{L}}{\partial z^{[l]}}$。

对于输出层 $L$：

$$\delta^{[L]} = \frac{\partial \mathcal{L}}{\partial z^{[L]}} = \frac{\partial \mathcal{L}}{\partial a^{[L]}} \odot \sigma'(z^{[L]})$$

其中 $\odot$ 是逐元素乘法 (Hadamard积)。

**例子** (Softmax + CrossEntropy): $\delta^{[L]} = \hat{y} - y$

#### 步骤3: 反向传播误差

**关键推导**: 对于隐藏层 $l$：

$$\delta^{[l]} = \frac{\partial \mathcal{L}}{\partial z^{[l]}}$$

由链式法则：

$$\frac{\partial \mathcal{L}}{\partial z^{[l]}} = \frac{\partial \mathcal{L}}{\partial z^{[l+1]}} \cdot \frac{\partial z^{[l+1]}}{\partial a^{[l]}} \cdot \frac{\partial a^{[l]}}{\partial z^{[l]}}$$

计算各项：

- $\frac{\partial z^{[l+1]}}{\partial a^{[l]}} = (W^{[l+1]})^T$
- $\frac{\partial a^{[l]}}{\partial z^{[l]}} = \text{diag}(\sigma'(z^{[l]}))$

因此：

$$\boxed{\delta^{[l]} = ((W^{[l+1]})^T \delta^{[l+1]}) \odot \sigma'(z^{[l]})}$$

#### 步骤4: 计算参数梯度

**权重梯度**:

$$\frac{\partial \mathcal{L}}{\partial W^{[l]}} = \delta^{[l]} (a^{[l-1]})^T$$

**偏置梯度**:

$$\frac{\partial \mathcal{L}}{\partial b^{[l]}} = \delta^{[l]}$$

### 4.3 完整的反向传播算法

**算法 4.1 (反向传播)**:

**输入**: 网络输入 $x$，标签 $y$，网络参数 $\{W^{[l]}, b^{[l]}\}_{l=1}^L$

**前向传播**:

```
a^[0] = x
for l = 1 to L:
    z^[l] = W^[l] a^[l-1] + b^[l]
    a^[l] = activation(z^[l])
```

**反向传播**:

```
# 输出层误差
delta^[L] = dL/da^[L] * activation'(z^[L])

# 反向传播到各层
for l = L-1 down to 1:
    delta^[l] = (W^[l+1])^T delta^[l+1] * activation'(z^[l])
```

**计算梯度**:

```
for l = 1 to L:
    dW^[l] = delta^[l] (a^[l-1])^T
    db^[l] = delta^[l]
```

## 5. 具体数值例子

### 5.1 2-1-1网络（两层）

**网络结构**:

- 输入层: 2个神经元 ($x_1, x_2$)
- 隐藏层: 1个神经元，Sigmoid激活
- 输出层: 1个神经元，线性激活

**参数**:

- $W^{[1]} = [0.1, 0.2]$ (1×2)
- $b^{[1]} = [0.3]$
- $W^{[2]} = [0.4]$ (1×1)
- $b^{[2]} = [0.5]$

**输入**: $x = [1, 2]^T$
**标签**: $y = [1]$

### 5.2 前向传播计算

**隐藏层**:
$$z^{[1]} = W^{[1]}x + b^{[1]} = 0.1 \times 1 + 0.2 \times 2 + 0.3 = 0.8$$
$$a^{[1]} = \sigma(0.8) = \frac{1}{1 + e^{-0.8}} \approx 0.690$$

**输出层**:
$$z^{[2]} = W^{[2]}a^{[1]} + b^{[2]} = 0.4 \times 0.690 + 0.5 \approx 0.776$$
$$\hat{y} = a^{[2]} = z^{[2]} \approx 0.776$$ (线性激活)

**损失** (MSE):
$$\mathcal{L} = \frac{1}{2}(0.776 - 1)^2 \approx 0.0251$$

### 5.3 反向传播计算

**输出层误差**:
$$\delta^{[2]} = \frac{\partial \mathcal{L}}{\partial z^{[2]}} = \hat{y} - y = 0.776 - 1 = -0.224$$

**隐藏层误差**:
$$\sigma'(z^{[1]}) = \sigma(z^{[1]})(1 - \sigma(z^{[1]})) = 0.690 \times 0.310 \approx 0.214$$

$$\delta^{[1]} = (W^{[2]})^T \delta^{[2]} \odot \sigma'(z^{[1]})$$
$$= 0.4 \times (-0.224) \times 0.214 \approx -0.0192$$

**参数梯度**:
$$\frac{\partial \mathcal{L}}{\partial W^{[2]}} = \delta^{[2]} (a^{[1]})^T = -0.224 \times 0.690 \approx -0.155$$
$$\frac{\partial \mathcal{L}}{\partial b^{[2]}} = \delta^{[2]} = -0.224$$

$$\frac{\partial \mathcal{L}}{\partial W^{[1]}} = \delta^{[1]} (a^{[0]})^T = -0.0192 \times [1, 2] = [-0.0192, -0.0384]$$
$$\frac{\partial \mathcal{L}}{\partial b^{[1]}} = \delta^{[1]} = -0.0192$$

### 5.4 参数更新 (学习率 η = 0.1)

$$W^{[2]}_{new} = 0.4 - 0.1 \times (-0.155) = 0.4155$$
$$b^{[2]}_{new} = 0.5 - 0.1 \times (-0.224) = 0.5224$$
$$W^{[1]}_{new} = [0.1, 0.2] - 0.1 \times [-0.0192, -0.0384] = [0.1019, 0.2038]$$
$$b^{[1]}_{new} = 0.3 - 0.1 \times (-0.0192) = 0.3019$$

## 6. Python完整实现（PyTorch风格）

```python
"""
神经网络反向传播算法完整实现
包含前向传播、反向传播、自动微分验证
"""
import numpy as np
from typing import List, Tuple, Callable, Optional
import matplotlib.pyplot as plt


class Layer:
    """神经网络层基类"""

    def __init__(self, input_size: int, output_size: int,
                 activation: str = 'relu'):
        """
        初始化层

        参数:
            input_size: 输入维度
            output_size: 输出维度
            activation: 激活函数 ('relu', 'sigmoid', 'tanh', 'linear', 'softmax')
        """
        # He初始化
        self.W = np.random.randn(output_size, input_size) * np.sqrt(2.0 / input_size)
        self.b = np.zeros((output_size, 1))

        self.activation_name = activation
        self.activation = self._get_activation(activation)
        self.activation_derivative = self._get_activation_derivative(activation)

        # 缓存前向传播结果用于反向传播
        self.cache = {}

    def _get_activation(self, name: str) -> Callable:
        """获取激活函数"""
        activations = {
            'relu': lambda z: np.maximum(0, z),
            'sigmoid': lambda z: 1 / (1 + np.exp(-np.clip(z, -500, 500))),
            'tanh': np.tanh,
            'linear': lambda z: z,
            'softmax': lambda z: np.exp(z - np.max(z, axis=0, keepdims=True)) /
                                  np.sum(np.exp(z - np.max(z, axis=0, keepdims=True)), axis=0, keepdims=True)
        }
        return activations[name]

    def _get_activation_derivative(self, name: str) -> Callable:
        """获取激活函数导数"""
        def relu_deriv(z, a):
            return (z > 0).astype(float)

        def sigmoid_deriv(z, a):
            return a * (1 - a)

        def tanh_deriv(z, a):
            return 1 - a**2

        def linear_deriv(z, a):
            return np.ones_like(z)

        def softmax_deriv(z, a):
            # Softmax+CrossEntropy的简化梯度
            return np.ones_like(z)

        derivatives = {
            'relu': relu_deriv,
            'sigmoid': sigmoid_deriv,
            'tanh': tanh_deriv,
            'linear': linear_deriv,
            'softmax': softmax_deriv
        }
        return derivatives[name]

    def forward(self, A_prev: np.ndarray) -> np.ndarray:
        """
        前向传播

        参数:
            A_prev: 上一层的输出 (input_size × m)

        返回:
            A: 当前层输出 (output_size × m)
        """
        m = A_prev.shape[1]

        # 线性变换
        Z = self.W @ A_prev + self.b

        # 激活
        A = self.activation(Z)

        # 缓存
        self.cache = {
            'A_prev': A_prev,
            'Z': Z,
            'A': A
        }

        return A

    def backward(self, dA: np.ndarray) -> Tuple[np.ndarray, np.ndarray, np.ndarray]:
        """
        反向传播

        参数:
            dA: 来自下一层的梯度 (output_size × m)

        返回:
            dA_prev: 传递给前一层的梯度
            dW: 权重梯度
            db: 偏置梯度
        """
        A_prev = self.cache['A_prev']
        Z = self.cache['Z']
        A = self.cache['A']
        m = A_prev.shape[1]

        # 激活函数导数
        dZ = dA * self.activation_derivative(Z, A)

        # 计算梯度
        dW = (dZ @ A_prev.T) / m
        db = np.mean(dZ, axis=1, keepdims=True)
        dA_prev = self.W.T @ dZ

        return dA_prev, dW, db

    def update(self, dW: np.ndarray, db: np.ndarray, lr: float):
        """参数更新"""
        self.W -= lr * dW
        self.b -= lr * db


class NeuralNetwork:
    """神经网络类"""

    def __init__(self, layer_dims: List[int],
                 activations: List[str],
                 loss_type: str = 'mse'):
        """
        初始化神经网络

        参数:
            layer_dims: 各层维度列表 [input, hidden1, ..., output]
            activations: 各层激活函数列表
            loss_type: 损失函数类型 ('mse', 'bce', 'ce')
        """
        self.layers = []
        self.loss_type = loss_type

        for i in range(len(layer_dims) - 1):
            layer = Layer(layer_dims[i], layer_dims[i+1], activations[i])
            self.layers.append(layer)

    def forward(self, X: np.ndarray) -> np.ndarray:
        """
        前向传播

        参数:
            X: 输入 (input_size × m)

        返回:
            输出 (output_size × m)
        """
        A = X
        for layer in self.layers:
            A = layer.forward(A)
        return A

    def compute_loss(self, Y_pred: np.ndarray, Y: np.ndarray) -> float:
        """计算损失"""
        m = Y.shape[1]

        if self.loss_type == 'mse':
            return np.mean((Y_pred - Y)**2) / 2
        elif self.loss_type == 'bce':
            epsilon = 1e-15
            Y_pred = np.clip(Y_pred, epsilon, 1 - epsilon)
            return -np.mean(Y * np.log(Y_pred) + (1 - Y) * np.log(1 - Y_pred))
        elif self.loss_type == 'ce':
            epsilon = 1e-15
            Y_pred = np.clip(Y_pred, epsilon, 1)
            return -np.mean(np.sum(Y * np.log(Y_pred), axis=0))
        else:
            raise ValueError(f"未知损失类型: {self.loss_type}")

    def backward(self, Y_pred: np.ndarray, Y: np.ndarray) -> float:
        """
        反向传播

        返回:
            损失值
        """
        loss = self.compute_loss(Y_pred, Y)

        # 输出层梯度
        if self.loss_type == 'mse':
            dA = Y_pred - Y
        elif self.loss_type in ['bce', 'ce']:
            # Softmax + CrossEntropy 的简化梯度
            dA = Y_pred - Y

        # 反向传播各层
        for layer in reversed(self.layers):
            dA, dW, db = layer.backward(dA)

        return loss

    def update(self, lr: float):
        """更新所有层参数"""
        for layer in self.layers:
            # 从缓存中获取梯度（实际实现需要存储梯度）
            pass

    def train_step(self, X: np.ndarray, Y: np.ndarray, lr: float) -> float:
        """单步训练"""
        # 前向传播
        Y_pred = self.forward(X)

        # 计算损失和反向传播
        loss = self.compute_loss(Y_pred, Y)

        # 反向传播
        if self.loss_type == 'mse':
            dA = Y_pred - Y
        else:
            dA = Y_pred - Y

        # 存储梯度用于更新
        grads_W = []
        grads_b = []

        for layer in reversed(self.layers):
            dA, dW, db = layer.backward(dA)
            grads_W.insert(0, dW)
            grads_b.insert(0, db)

        # 更新参数
        for i, layer in enumerate(self.layers):
            layer.update(grads_W[i], grads_b[i], lr)

        return loss

    def fit(self, X: np.ndarray, Y: np.ndarray,
            epochs: int = 1000, lr: float = 0.01,
            verbose: bool = True) -> List[float]:
        """
        训练模型

        参数:
            X: 训练数据 (features × samples)
            Y: 标签 (outputs × samples)
            epochs: 训练轮数
            lr: 学习率
            verbose: 是否打印进度

        返回:
            损失历史
        """
        history = []

        for epoch in range(epochs):
            loss = self.train_step(X, Y, lr)
            history.append(loss)

            if verbose and (epoch + 1) % 100 == 0:
                print(f"Epoch {epoch + 1}/{epochs}, Loss: {loss:.6f}")

        return history

    def predict(self, X: np.ndarray) -> np.ndarray:
        """预测"""
        return self.forward(X)

    Y = np.random.randn(1, 5)

    # 数值梯度
    epsilon = 1e-7

    # 对第一层第一个权重进行梯度检查
    original_W = nn.layers[0].W[0, 0].copy()

    # f(theta + epsilon)
    nn.layers[0].W[0, 0] = original_W + epsilon
    loss_plus = nn.compute_loss(nn.forward(X), Y)

    # f(theta - epsilon)
    nn.layers[0].W[0, 0] = original_W - epsilon
    loss_minus = nn.compute_loss(nn.forward(X), Y)

    # 数值梯度
    numerical_grad = (loss_plus - loss_minus) / (2 * epsilon)

    # 反向传播梯度
    nn.layers[0].W[0, 0] = original_W
    Y_pred = nn.forward(X)
    dA = Y_pred - Y
    dA, dW, _ = nn.layers[0].backward(dA)
    backprop_grad = dW[0, 0]

    # 比较
    diff = abs(numerical_grad - backprop_grad)
    print(f"\n数值梯度: {numerical_grad:.8f}")
    print(f"反向传播梯度: {backprop_grad:.8f}")
    print(f"差异: {diff:.2e}")
    print(f"检查: {'通过' if diff < 1e-5 else '失败'}")


if __name__ == "__main__":
    # XOR演示
    history = demonstrate_xor()

    # 梯度检查
    demonstrate_gradient_check()
```

## 7. 梯度消失与梯度爆炸的数学分析

### 7.1 梯度消失问题

**定义 7.1 (梯度消失)**: 在反向传播中，靠近输入层的梯度变得极小，导致这些层的参数几乎不更新。

**数学分析**: 考虑一个深度网络，每层使用sigmoid激活函数。

对于sigmoid：$\sigma'(z) = \sigma(z)(1-\sigma(z)) \leq 0.25$（最大值在 $z=0$ 处）

反向传播时：
$$\delta^{[l]} = (W^{[l+1]})^T \delta^{[l+1]} \odot \sigma'(z^{[l]})$$

假设权重初始化为小值，$\|W^{[l]}\| \approx 1$：

$$\|\delta^{[l]}\| \leq 0.25 \|\delta^{[l+1]}\|$$

经过 $L$ 层：
$$\|\delta^{[1]}\| \leq (0.25)^{L-1} \|\delta^{[L]}\|$$

对于 $L = 10$：$(0.25)^9 \approx 3.8 \times 10^{-6}$，梯度几乎消失。

### 7.2 梯度爆炸问题

**定义 7.2 (梯度爆炸)**: 在反向传播中，梯度变得极大，导致参数更新过大，训练不稳定。

**数学分析**: 如果权重初始化为较大值，$\|W^{[l]}\| > 1$：

$$\|\delta^{[l]}\| \approx \|W^{[l+1]}\| \|\delta^{[l+1]}\|$$

经过 $L$ 层：
$$\|\delta^{[1]}\| \approx \prod_{l=2}^L \|W^{[l]}\| \cdot \|\delta^{[L]}\|$$

若 $\|W^{[l]}\| = 2$，经过10层：梯度放大 $2^9 = 512$ 倍。

### 7.3 解决方案

| 问题 | 解决方案 | 数学原理 |
|------|----------|----------|
| **梯度消失** | ReLU激活 | $\sigma'(z) = 1$ (for $z > 0$) |
| **梯度消失** | 残差连接 (ResNet) | 引入跳跃连接：$a^{[l+1]} = a^{[l]} + \mathcal{F}(a^{[l]})$ |
| **梯度消失** | 批归一化 (BatchNorm) | 标准化激活，稳定梯度分布 |
| **梯度爆炸** | 梯度裁剪 | $\text{if } \|g\| > \tau: g = \tau \frac{g}{\|g\|}$ |
| **梯度爆炸** | 权重正则化 | 限制权重大小 |
