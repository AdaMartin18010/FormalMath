# 数字滤波器设计的Z变换方法

## 应用领域

**学科**: 数字信号处理 / 控制工程  
**具体应用**: 音频处理、通信系统、生物医学信号、雷达系统  
**MSC分类**: 94A12 (Signal theory), 93E11 (Filtering), 39A06 (Linear difference equations)

---

## 数学概念

### 核心数学工具

1. **Z变换**: 离散时间信号的频域表示
2. **传递函数**: 系统输入输出关系
3. **极零点分析**: 系统稳定性与频率响应
4. **双线性变换**: 模拟到数字滤波器转换
5. **窗函数法**: FIR滤波器设计

### 关键定义

- **Z变换**: $X(z) = \sum_{n=-\infty}^{\infty} x[n]z^{-n}$
- **单位圆**: $|z| = 1$，对应离散时间傅里叶变换(DTFT)

---

## 问题描述

### 滤波器设计需求

**目标**: 设计离散时间系统，选择性通过或抑制特定频率成分。

**滤波器类型**:
- **低通**: 通过低频，抑制高频
- **高通**: 通过高频，抑制低频
- **带通**: 通过特定频带
- **带阻**: 抑制特定频带

### 设计指标

- **通带波纹** ($\delta_p$): 通带内增益变化
- **阻带衰减** ($\delta_s$): 阻带抑制程度
- **过渡带宽**: 通带与阻带之间的过渡
- **群延迟**: 相位线性度

---

## 数学模型

### 线性时不变(LTI)系统

**差分方程**:

$$\sum_{k=0}^{N} a_k y[n-k] = \sum_{k=0}^{M} b_k x[n-k]$$

**Z变换域**:

$$H(z) = \frac{Y(z)}{X(z)} = \frac{\sum_{k=0}^{M} b_k z^{-k}}{\sum_{k=0}^{N} a_k z^{-k}} = \frac{B(z)}{A(z)}$$

### 极零点表示

**因式分解**:

$$H(z) = G \frac{\prod_{k=1}^{M}(1 - z_k z^{-1})}{\prod_{k=1}^{N}(1 - p_k z^{-1})}$$

其中 $z_k$ 是零点，$p_k$ 是极点，$G$ 是增益。

### 稳定性条件

**定理**: 因果LTI系统BIBO稳定当且仅当所有极点满足 $|p_k| < 1$（位于单位圆内）。

### 频率响应

在单位圆上求值：$z = e^{j\omega}$，$\omega \in [0, 2\pi]$

$$H(e^{j\omega}) = \frac{B(e^{j\omega})}{A(e^{j\omega})}$$

**幅度响应**: $|H(e^{j\omega})|$  
**相位响应**: $\angle H(e^{j\omega})$

## 求解过程

### 步骤1：FIR滤波器设计（窗函数法）

**理想低通滤波器**（截止频率 $\omega_c$）:

$$h_d[n] = \frac{\omega_c}{\pi} \text{sinc}\left(\frac{\omega_c n}{\pi}\right) = \frac{\sin(\omega_c n)}{\pi n}$$

**问题**: 无限长、非因果

**窗函数截断**:

$$h[n] = h_d[n - \frac{N-1}{2}] \cdot w[n], \quad n = 0, 1, ..., N-1$$

**常用窗函数**:

| 窗函数 | 主瓣宽度 | 旁瓣衰减 | 特点 |
|--------|----------|----------|------|
| **矩形** | $4\pi/N$ | -13 dB | 最窄主瓣，旁瓣高 |
| **Hanning** | $8\pi/N$ | -31 dB | 良好折中 |
| **Hamming** | $8\pi/N$ | -41 dB | 低旁瓣 |
| **Blackman** | $12\pi/N$ | -57 dB | 宽主瓣，低旁瓣 |
| **Kaiser** | 可调 | 可调 | 参数化设计 |

### 步骤2：IIR滤波器设计（双线性变换）

**模拟原型**: Butterworth、Chebyshev、Elliptic

**双线性变换**:

$$s = \frac{2}{T} \cdot \frac{1 - z^{-1}}{1 + z^{-1}}$$

将模拟频率 $\Omega$ 映射到数字频率 $\omega$:

$$\Omega = \frac{2}{T} \tan\left(\frac{\omega}{2}\right)$$

**频率预畸变**: 补偿非线性映射

**Butterworth低通设计**:

1. 确定模拟原型阶数 $N$ 和截止频率 $\Omega_c$
2. 应用双线性变换得到 $H(z)$
3. 验证数字频率响应

### 步骤3：稳定性验证

**求解分母多项式根**:

```
poles = roots(A)  // A是a_k系数向量
for each pole p:
    if |p| >= 1:
        系统不稳定
```

**稳定化**（如有必要）:
- 将单位圆外极点映射到圆内: $p' = 1/p^*$
- 调整增益保持幅度响应

### 步骤4：实现结构

**直接型I**:
- 先实现零点，再实现极点
- 需要 $N + M + 1$ 个延迟单元

**直接型II（规范型）**:
- 最小延迟单元数: $\max(N, M)$
- 更数值稳定

**级联型**:
- 将 $H(z)$ 分解为二阶节(SOS)的乘积
- $$H(z) = \prod_{k=1}^{L} \frac{b_{0k} + b_{1k}z^{-1} + b_{2k}z^{-2}}{1 + a_{1k}z^{-1} + a_{2k}z^{-2}}$$
- 数值稳定性最好

## 结果分析

### FIR vs IIR比较

| 特性 | FIR | IIR |
|------|-----|-----|
| **稳定性** | 始终稳定 | 需要验证 |
| **线性相位** | 可以精确实现 | 只能近似 |
| **计算复杂度** | 高（阶数高） | 低（阶数低） |
| **设计难度** | 简单（窗函数） | 较复杂 |
| **延迟** | 大（群延迟 $N/2$） | 小 |

### 典型设计参数

**FIR低通滤波器**:
- 采样率: 44.1 kHz（CD音频）
- 通带: 0 - 18 kHz
- 阻带: 20 - 22.05 kHz
- 阶数: 约100-200（取决于窗函数和指标）

**IIR低通滤波器**:
- 相同指标: 8-12阶即可满足
- 使用椭圆滤波器实现最陡峭过渡

### 实际应用

**音频均衡器**:
- 多带FIR滤波器组
- 线性相位保持波形

**语音识别前端**:
- 预加重滤波器（高通）
- 抗混叠滤波器

**通信系统**:
- 匹配滤波器（最大化SNR）
- 根升余弦滤波器（限制带宽）
