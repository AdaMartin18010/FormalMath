# PID控制器的稳定性分析

## 应用领域

**学科**: 控制理论 / 自动化工程  
**具体应用**: 温度控制、电机调速、化工过程、飞行器控制  
**MSC分类**: 93C80 (Frequency-response methods), 93D05 (Lyapunov stability), 93B50 (Synthesis problems)

---

## 数学概念

### 核心数学工具

1. **传递函数**: 系统输入输出关系的频域表示
2. **拉普拉斯变换**: 连续时间系统的分析工具
3. **根轨迹**: 闭环极点随参数变化的轨迹
4. **奈奎斯特判据**: 基于开环频率响应的稳定性判据
5. **相位裕度/增益裕度**: 相对稳定性指标

### 关键定义

- **PID控制器**: $C(s) = K_p + \frac{K_i}{s} + K_d s$
- **闭环传递函数**: $T(s) = \frac{C(s)G(s)}{1 + C(s)G(s)}$

---

## 问题描述

### 控制目标

**问题**: 使系统输出 $y(t)$ 跟踪参考输入 $r(t)$，同时抑制干扰 $d(t)$。

**性能要求**:
- **稳定性**: 闭环系统BIBO稳定
- **稳态精度**: 跟踪误差趋于零或足够小
- **瞬态响应**: 上升时间、超调量、调节时间
- **鲁棒性**: 对模型不确定性不敏感

### PID控制原理

- **比例(P)**: 当前误差，即时响应
- **积分(I)**: 累积误差，消除稳态误差
- **微分(D)**: 误差变化率，预测趋势、抑制超调

---

## 数学模型

### PID控制器

**时域**:

$$u(t) = K_p e(t) + K_i \int_0^t e(\tau)d\tau + K_d \frac{de(t)}{dt}$$

**拉普拉斯域**:

$$C(s) = K_p + \frac{K_i}{s} + K_d s = K_p\left(1 + \frac{1}{T_i s} + T_d s\right)$$

其中 $T_i = K_p/K_i$ 是积分时间，$T_d = K_d/K_p$ 是微分时间。

### 闭环系统

**开环传递函数**: $L(s) = C(s)G(s)$

**闭环传递函数**:

$$T(s) = \frac{L(s)}{1 + L(s)} = \frac{C(s)G(s)}{1 + C(s)G(s)}$$

**特征方程**:

$$1 + L(s) = 0 \Rightarrow 1 + C(s)G(s) = 0$$

### 稳定性判据

**奈奎斯特判据**:

闭环系统稳定当且仅当开环频率响应 $L(j\omega)$ 的奈奎斯特图不包围 $(-1, 0)$ 点，假设开环稳定。

**相位裕度**: 
$$PM = 180° + \angle L(j\omega_{gc})$$

其中 $\omega_{gc}$ 是增益穿越频率（$|L(j\omega_{gc})| = 1$）。

**增益裕度**:
$$GM = \frac{1}{|L(j\omega_{pc})|}$$

其中 $\omega_{pc}$ 是相位穿越频率（$\angle L(j\omega_{pc}) = -180°$）。

## 求解过程

### 步骤1：Ziegler-Nichols整定法

**方法1: 阶跃响应法**（适用于自平衡过程）

从开环阶跃响应确定参数：
- $L$: 延迟时间
- $T$: 时间常数
- $K$: 过程增益

| 控制器 | $K_p$ | $T_i$ | $T_d$ |
|--------|-------|-------|-------|
| **P** | $T/(KL)$ | - | - |
| **PI** | $0.9T/(KL)$ | $3L$ | - |
| **PID** | $1.2T/(KL)$ | $2L$ | $0.5L$ |

**方法2: 临界增益法**（闭环法）

1. 仅使用比例控制，增大 $K_p$ 直至系统临界振荡
2. 记录临界增益 $K_u$ 和振荡周期 $P_u$

| 控制器 | $K_p$ | $T_i$ | $T_d$ |
|--------|-------|-------|-------|
| **P** | $0.5K_u$ | - | - |
| **PI** | $0.45K_u$ | $P_u/1.2$ | - |
| **PID** | $0.6K_u$ | $P_u/2$ | $P_u/8$ |

### 步骤2：根轨迹分析

**绘制根轨迹**: 随 $K$ 变化，闭环极点 $s$ 满足的方程：

$$1 + K \frac{(s-z_1)(s-z_2)...}{(s-p_1)(s-p_2)...} = 0$$

**PID对根轨迹的影响**:
- **积分**: 增加原点极点，改善稳态性能但降低稳定性
- **微分**: 增加零点，改善瞬态响应和稳定性

### 步骤3：频域设计

**目标**: 相位裕度 30°-60°，增益裕度 > 6 dB

**设计步骤**:

1. **P控制**: 调整 $K_p$ 使带宽和稳态误差满足要求
2. **加入I**: 降低 $K_p$ 约20%，加入积分消除稳态误差
3. **加入D**: 增加微分改善相位裕度和瞬态响应

**超前-滞后补偿**:

PID可视为超前-滞后补偿的特殊形式：
- PI ≈ 滞后补偿（改善低频增益）
- PD ≈ 超前补偿（改善相位裕度）

### 步骤4：数字实现

**后向欧拉离散化**:

$$s \approx \frac{1 - z^{-1}}{T_s}$$

**PID差分方程**:

$$u[n] = u[n-1] + q_0 e[n] + q_1 e[n-1] + q_2 e[n-2]$$

其中:
- $q_0 = K_p + K_i T_s + K_d/T_s$
- $q_1 = -K_p - 2K_d/T_s$
- $q_2 = K_d/T_s$

## 结果分析

### 性能指标

| 指标 | 定义 | 典型要求 |
|------|------|----------|
| **上升时间** ($t_r$) | 10%到90%终值时间 | 尽可能快 |
| **超调量** ($M_p$) | $(y_{max} - y_{ss})/y_{ss}$ | < 10-20% |
| **调节时间** ($t_s$) | 进入±2%或±5%终值带 | < 规定值 |
| **稳态误差** ($e_{ss}$) | $\lim_{t\to\infty}(r(t) - y(t))$ | = 0 或足够小 |

### 参数影响

| 参数 | 上升时间 | 超调量 | 调节时间 | 稳态误差 |
|------|----------|--------|----------|----------|
| **$K_p$增加** | 减小 | 增加 | 小变化 | 减小 |
| **$K_i$增加** | 减小 | 增加 | 增加 | 消除 |
| **$K_d$增加** | 小变化 | 减小 | 减小 | 无影响 |

### 实际应用考量

**积分饱和(Anti-windup)**:
- 问题: 执行器饱和时积分项累积
- 解决: 限制积分项或条件积分

**微分冲击**:
- 问题: 参考输入突变导致微分项冲击
- 解决: 对反馈而非误差微分，或添加低通滤波

**噪声敏感性**:
- 问题: 微分放大高频噪声
- 解决: 添加微分滤波器 $1/(1 + T_f s)$
