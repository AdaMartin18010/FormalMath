# 桥梁结构的有限元分析

## 应用领域

**学科**: 结构工程 / 计算力学  
**具体应用**: 桥梁设计、建筑结构、航空航天结构、机械部件分析  
**MSC分类**: 65N30 (Finite element methods), 74S05 (Finite element methods in solid mechanics), 74K10 (Rods, beams, columns)

---

## 数学概念

### 核心数学工具

1. **变分原理**: 虚功原理、最小势能原理
2. **有限元离散**: 将连续域离散为有限单元
3. **刚度矩阵**: 单元与整体刚度组装
4. **数值积分**: Gauss求积法
5. **线性系统求解**: 大规模稀疏矩阵

### 关键定义

- **形函数**: 单元内场变量的插值函数
- **等参元**: 几何与场变量使用相同形函数
- **弱形式**: 降低微分阶数的积分形式

---

## 问题描述

### 结构分析问题

**目标**: 求解桥梁在载荷作用下的：
- 位移（挠度）
- 应力分布
- 应变状态

**载荷类型**:
- 静载荷: 自重、桥面恒载
- 活载荷: 车辆、人群
- 风载荷: 横向压力
- 地震载荷: 惯性力

### 力学模型

**控制方程**（线弹性）:

**平衡方程**:
$$\nabla \cdot \boldsymbol{\sigma} + \mathbf{f} = \mathbf{0}$$

**本构关系**（广义胡克定律）:
$$\boldsymbol{\sigma} = \mathbf{C} : \boldsymbol{\varepsilon}$$

**几何方程**（小变形）:
$$\boldsymbol{\varepsilon} = \frac{1}{2}(\nabla \mathbf{u} + (\nabla \mathbf{u})^T)$$

---

## 数学模型

### 弱形式（虚功原理）

求位移场 $\mathbf{u} \in V$ 使得对所有虚位移 $\delta \mathbf{u} \in V_0$:

$$\int_\Omega \boldsymbol{\sigma} : \delta \boldsymbol{\varepsilon} \, d\Omega = \int_\Omega \mathbf{f} \cdot \delta \mathbf{u} \, d\Omega + \int_{\Gamma_t} \mathbf{t} \cdot \delta \mathbf{u} \, d\Gamma$$

其中:
- $\delta \boldsymbol{\varepsilon}$: 虚应变
- $\mathbf{f}$: 体积力
- $\mathbf{t}$: 表面牵引力

### 有限元离散

**单元插值**:

$$\mathbf{u}^h = \sum_{i=1}^{n_{node}} N_i(\boldsymbol{\xi}) \mathbf{d}_i = \mathbf{N} \mathbf{d}^e$$

其中:
- $N_i$: 形函数
- $\mathbf{d}_i$: 节点位移
- $\boldsymbol{\xi}$: 自然坐标

**应变-位移矩阵**:

$$\boldsymbol{\varepsilon}^h = \mathbf{B} \mathbf{d}^e$$

**单元刚度矩阵**:

$$\mathbf{K}^e = \int_{\Omega_e} \mathbf{B}^T \mathbf{D} \mathbf{B} \, d\Omega$$

其中 $\mathbf{D}$ 是材料本构矩阵。

### 整体系统

**组装**: 

$$\mathbf{K} = \bigwedge_{e=1}^{n_{elem}} \mathbf{K}^e$$

（$\bigwedge$ 表示按自由度组装）

**线性系统**:

$$\mathbf{K} \mathbf{d} = \mathbf{F}$$

其中:
- $\mathbf{K}$: 整体刚度矩阵（对称正定稀疏）
- $\mathbf{d}$: 整体位移向量
- $\mathbf{F}$: 整体载荷向量

---

## 求解过程

### 步骤1：前处理

**几何建模**:
- 桥梁结构的三维建模
- 定义材料属性（弹性模量 $E$、泊松比 $\nu$、密度 $\rho$）

**网格划分**:
- 将几何离散为有限单元
- 单元类型选择:
  - **桁架单元**: 只受轴力（斜拉索）
  - **梁单元**: 受弯矩、剪力（主梁、桥墩）
  - **板壳单元**: 桥面
  - **实体单元**: 支座、锚固区

**网格密度**:
- 应力集中区细化
- 长宽比控制（避免畸形单元）

### 步骤2：刚度矩阵计算与组装

**等参元变换**:

$$\mathbf{K}^e = \int_{-1}^{1}\int_{-1}^{1}\int_{-1}^{1} \mathbf{B}^T \mathbf{D} \mathbf{B} \, |\mathbf{J}| \, d\xi d\eta d\zeta$$

**Gauss数值积分**:

$$\mathbf{K}^e \approx \sum_{i=1}^{n_g} \sum_{j=1}^{n_g} \sum_{k=1}^{n_g} w_i w_j w_k \mathbf{B}^T \mathbf{D} \mathbf{B} \, |\mathbf{J}|$$

**边界条件处理**:
- **位移边界**: 直接消去或罚函数法
- **对称边界**: 减少计算量

### 步骤3：线性系统求解

**直接法**（小规模）:
- LDL^T分解（对称正定）
- 带状矩阵优化

**迭代法**（大规模）:
- 共轭梯度法(CG)
- 预处理技术（不完全Cholesky、多重网格）

**稀疏矩阵技术**:
- CSR/CSC存储格式
- 重排序减少填充（AMD、嵌套剖分）

### 步骤4：后处理

**单元应力恢复**:

$$\boldsymbol{\sigma} = \mathbf{D} \mathbf{B} \mathbf{d}^e$$

**节点平均**: 改善应力连续性

**结果可视化**:
- 变形图（放大显示）
- 应力云图
- 动画（动态载荷）

## 结果分析

### 验证与确认

**验证**(Verification): 数值解是否正确求解数学模型
- 网格收敛性分析
- 与解析解对比（简单情况）

**确认**(Validation): 数学模型是否反映物理现实
- 与实验数据对比
- 与规范要求对比

### 安全性评估

**应力准则**:

$$\sigma_{vm} = \sqrt{\frac{3}{2}s_{ij}s_{ij}} \leq \frac{\sigma_y}{FS}$$

其中:
- $\sigma_{vm}$: von Mises等效应力
- $\sigma_y$: 屈服强度
- $FS$: 安全系数（通常1.5-3.0）

**挠度准则**:

$$\delta_{max} \leq \frac{L}{250} \text{ 至 } \frac{L}{400}$$

（$L$ 为跨度）

### 典型结果

| 分析类型 | 输出 | 应用 |
|----------|------|------|
| **静力分析** | 位移、应力 | 常规设计校核 |
| **模态分析** | 固有频率、振型 | 风振、地震响应 |
| **屈曲分析** | 临界载荷 | 稳定性校核 |
| **疲劳分析** | 寿命预测 | 维护计划 |
