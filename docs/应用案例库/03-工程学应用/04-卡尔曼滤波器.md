# 卡尔曼滤波器的状态估计

## 应用领域

**学科**: 估计理论 / 随机系统 / 控制工程  
**具体应用**: 导航制导(GPS/INS)、目标跟踪、传感器融合、经济学  
**MSC分类**: 93E10 (Estimation and detection), 93E11 (Filtering), 60G35 (Signal detection and filtering)

---

## 数学概念

### 核心数学工具

1. **状态空间模型**: 动态系统的数学描述
2. **最小均方误差估计**: 最优估计准则
3. **递推贝叶斯估计**: 预测-更新框架
4. **Riccati方程**: 误差协方差传播
5. **高斯分布**: 噪声与状态的概率描述

### 关键定义

- **状态方程**: $x_k = F x_{k-1} + B u_{k-1} + w_{k-1}$
- **观测方程**: $z_k = H x_k + v_k$
- **卡尔曼增益**: $K_k$（最优权重）

---

## 问题描述

### 状态估计问题

**系统模型**:

**状态方程**（过程模型）:
$$x_k = F x_{k-1} + B u_{k-1} + w_{k-1}$$

**观测方程**（测量模型）:
$$z_k = H x_k + v_k$$

其中:
- $x_k$: 系统状态（待估计）
- $u_{k-1}$: 已知控制输入
- $z_k$: 观测值
- $w_{k-1} \sim \mathcal{N}(0, Q)$: 过程噪声
- $v_k \sim \mathcal{N}(0, R)$: 观测噪声

### 核心问题

如何从含噪声的观测 $\{z_1, ..., z_k\}$ 最优估计状态 $x_k$？

---

## 数学模型

### 卡尔曼滤波的两步递推

**预测步**（时间更新）:

$$\hat{x}_{k|k-1} = F \hat{x}_{k-1|k-1} + B u_{k-1}$$
$$P_{k|k-1} = F P_{k-1|k-1} F^T + Q$$

**更新步**（测量更新）:

$$K_k = P_{k|k-1} H^T (H P_{k|k-1} H^T + R)^{-1}$$
$$\hat{x}_{k|k} = \hat{x}_{k|k-1} + K_k (z_k - H \hat{x}_{k|k-1})$$
$$P_{k|k} = (I - K_k H) P_{k|k-1}$$

### 最优性证明

**定理**: 在高斯噪声假设下，卡尔曼滤波器是最小均方误差(MMSE)估计。

**证明概要**:

1. **预测**: 基于上一时刻最优估计的状态演化
2. **新息**: $y_k = z_k - H \hat{x}_{k|k-1}$（观测残差）
3. **卡尔曼增益**: 最优权重使后验协方差最小

通过求解:

$$\frac{\partial \text{tr}(P_{k|k})}{\partial K_k} = 0$$

得到卡尔曼增益公式。

### 稳态分析

对于时不变系统（$F, H, Q, R$ 恒定）：

**离散代数Riccati方程(DARE)**:

$$P = F P F^T - F P H^T (H P H^T + R)^{-1} H P F^T + Q$$

**稳态卡尔曼增益**:

$$K_{\infty} = P_{\infty} H^T (H P_{\infty} H^T + R)^{-1}$$

---

## 求解过程

### 步骤1：初始化

**初始状态估计**: $\hat{x}_0 = E[x_0]$（或先验知识）

**初始误差协方差**: $P_0 = E[(x_0 - \hat{x}_0)(x_0 - \hat{x}_0)^T]$

对于无先验知识，使用大协方差（如 $P_0 = \sigma^2 I$，$\sigma$ 很大）。

### 步骤2：预测-更新循环

```
for each time step k:
    // 预测
    x_pred = F * x_est + B * u
    P_pred = F * P_est * F' + Q
    
    // 更新
    y = z - H * x_pred        // 新息
    S = H * P_pred * H' + R   // 新息协方差
    K = P_pred * H' * inv(S)  // 卡尔曼增益
    
    x_est = x_pred + K * y
    P_est = (I - K * H) * P_pred
```

### 步骤3：数值稳定性改进

**Joseph形式**（保持协方差对称正定）:

$$P_{k|k} = (I - K_k H) P_{k|k-1} (I - K_k H)^T + K_k R K_k^T$$

**平方根滤波**:

传播 $P = S S^T$ 的平方根而非 $P$ 本身，保持数值稳定。

### 步骤4：扩展卡尔曼滤波(EKF)（非线性系统）

对于非线性系统:

$$x_k = f(x_{k-1}, u_{k-1}) + w_{k-1}$$
$$z_k = h(x_k) + v_k$$

**线性化**（一阶泰勒展开）:

$$F_k = \left.\frac{\partial f}{\partial x}\right|_{\hat{x}_{k-1|k-1}}, \quad H_k = \left.\frac{\partial h}{\partial x}\right|_{\hat{x}_{k|k-1}}$$

然后应用标准卡尔曼滤波。

## 结果分析

### 滤波器性能

**误差协方差** $P_{k|k}$ 表征估计精度：
- 对角元素是各状态变量的估计方差
- 越小表示估计越精确

**新息** $y_k$ 的性质（最优滤波）:
- 零均值: $E[y_k] = 0$
- 白噪声: $E[y_k y_j^T] = 0$ 对 $k \neq j$
- 协方差: $S_k = H P_{k|k-1} H^T + R$

可用于故障检测（新息过大表示模型不匹配）。

### 设计参数选择

| 参数 | 作用 | 调谐策略 |
|------|------|----------|
| **$Q$** | 过程噪声协方差 | 增大 → 更信任观测 |
| **$R$** | 观测噪声协方差 | 增大 → 更信任模型 |
| **$P_0$** | 初始不确定性 | 无先验时用大值 |

**自适应卡尔曼滤波**: 在线估计 $Q$ 和 $R$。

### 典型应用

**GPS/INS组合导航**:
- 状态: 位置、速度、姿态、传感器偏差
- 观测: GPS位置、IMU加速度/角速度
- 结果: 高精度、高频率的位置估计

**目标跟踪**:
- 状态: 目标位置、速度、加速度
- 观测: 雷达/视觉测量
- 结果: 平滑的轨迹估计

**传感器融合**:
- 多传感器数据融合（温度、湿度、压力）
- 不同采样率、不同精度的传感器
