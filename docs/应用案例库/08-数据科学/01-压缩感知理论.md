# 压缩感知理论

## 概述
- **应用领域**: 信号处理/成像/机器学习
- **数学分支**: 凸优化/随机矩阵/泛函分析
- **难度级别**: 高级（研究级）
- **关键词**: 稀疏恢复、L1最小化、RIP、受限等距性、随机投影

---

## 问题背景

### 传统采样定理

**奈奎斯特-香农采样定理**: 采样率必须是信号最高频率的2倍以上才能完美重建。

**问题**: 高带宽信号需要大量采样和存储。

### 压缩感知的核心思想

如果信号是稀疏的（或在某个基下稀疏），可以用远少于奈奎斯特定理的测量重建。

**关键洞察**: 直接获取压缩表示，跳过传统"采样-压缩"流程。

**应用**:
- MRI成像（加速扫描）
- 单像素相机
- 雷达成像
- 异常检测

---

## 数学建模

### 问题设置

**信号模型**:
- $x \in \mathbb{R}^n$: 原始信号
- 稀疏性: $\|x\|_0 = k \ll n$ (非零元素个数)

或 $x = \Psi \alpha$，其中 $\alpha$ 稀疏。

**测量过程**:
$$y = Ax$$

其中 $A \in \mathbb{R}^{m \times n}$ 是测量矩阵，$m \ll n$。

**重建问题**:
$$\min_x \|x\|_0 \quad \text{s.t. } y = Ax$$

### 稀疏基扩展

**可压缩信号**:
$$x = \sum_{i=1}^n \alpha_i \psi_i = \Psi \alpha$$

常用稀疏基:
- DCT/Fourier: 平滑信号
- Wavelet: 分段光滑信号
- Curvelet: 图像边缘

---

## 数学方法

### L1松弛

**NP难问题**: L0最小化是组合问题

**凸松弛**:
$$\min_x \|x\|_1 \quad \text{s.t. } y = Ax$$

**定理** (Candès, Tao, Romberg; Donoho):
在适当条件下，L1最小化等价于L0最小化。

### 受限等距性 (RIP)

**定义**: 矩阵 $A$ 满足RIP of order $k$，若存在 $\delta_k \in (0, 1)$ 使得:
$$(1 - \delta_k)\|x\|_2^2 \leq \|Ax\|_2^2 \leq (1 + \delta_k)\|x\|_2^2$$

对所有 $k$-稀疏 $x$ 成立。

**充分条件**:
若 $\delta_{2k} < \sqrt{2} - 1$，则L1最小化精确恢复任何 $k$-稀疏信号。

### 随机矩阵

**高斯测量矩阵**:
$A_{ij} \sim \mathcal{N}(0, 1/m)$

**性质**: 高概率满足RIP，只要
$$m \geq C \cdot k \log(n/k)$$

**其他构造**:
- Bernoulli随机矩阵
- 随机傅里叶采样
- 结构化随机矩阵

## 求解过程

### 基追踪 (Basis Pursuit)

**线性规划形式**:
$$\min_{x, u} \sum_i u_i$$
$$\text{s.t. } -u_i \leq x_i \leq u_i, \quad y = Ax$$

可用内点法求解。

### 迭代算法

**匹配追踪 (MP)**:
1. 初始化残差 $r = y$
2. 找到与残差最相关的列: $j = \arg\max_i |\langle a_i, r \rangle|$

3. 更新解和残差
4. 重复直到收敛

**正交匹配追踪 (OMP)**:
- 维护已选列的张成空间
- 每次迭代正交投影
- 最多 $k$ 次迭代

**迭代阈值**:
- ISTA/FISTA: 近端梯度方法
- AMP: 近似消息传递

### 噪声鲁棒性

**LASSO形式**:
$$\min_x \frac{1}{2}\|y - Ax\|_2^2 + \lambda \|x\|_1$$

或约束形式:
$$\min_x \|x\|_1 \quad \text{s.t. } \|y - Ax\|_2 \leq \epsilon$$

**误差界**:
$$\|\hat{x} - x\|_2 \leq C_0 \frac{\|x - x_k\|_1}{\sqrt{k}} + C_1 \epsilon$$

其中 $x_k$ 是最佳 $k$-项近似。

## 结果分析

### 相变现象

**$(\delta = m/n, \rho = k/m)$ 平面**:
- 精确恢复区域
- 失败区域
- 相变边界

### 应用实例

**MRI加速**:
- 传统: 256×256 = 65536采样
- CS: 20%采样即可重建
- 扫描时间从30分钟缩短到6分钟

**单像素相机**:
- DMD微镜阵列调制
- 单光电探测器
- 每次测量一个随机投影

## 拓展思考

### 矩阵感知与补全

**低秩矩阵恢复**:
$$\min_X \|X\|_* \quad \text{s.t. } \mathcal{A}(X) = b$$

其中 $\|\cdot\|_*$ 是核范数（奇异值之和）。

**矩阵补全** (Netflix问题):
- 仅观察部分条目
- 低秩假设
- 核范数最小化

### 结构化稀疏

- 组稀疏: 整组同时为零
- 树稀疏: 小波树结构
- 模型CS: 先验结构约束

### 深度学习联系

**深度压缩感知**:
- 学习测量矩阵
- 学习重建网络
- 端到端训练

**隐式先验**: 深度网络作为隐式正则化
