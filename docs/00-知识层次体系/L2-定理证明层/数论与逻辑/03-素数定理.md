# 素数定理

**定理编号**: L2-NL003  
**MSC分类**: 11N05 (素数分布)  
**难度等级**: ⭐⭐⭐⭐⭐  
**证明策略**: ANA (解析方法) + CST (复分析)

---

## 定理陈述

**定理（素数定理，PNT）**

设 $\pi(x)$ 为不超过 $x$ 的素数个数，则

$$\pi(x) \sim \frac{x}{\ln x}$$

即 $\displaystyle\lim_{x \to \infty} \frac{\pi(x)}{x/\ln x} = 1$。

**等价形式**：第 $n$ 个素数 $p_n \sim n \ln n$。

#### 步骤1-2：ζ函数与欧拉乘积

$$\zeta(s) = \sum_{n=1}^\infty \frac{1}{n^s} = \prod_p \frac{1}{1 - p^{-s}}$$

取对数微分得：
$$-\frac{\zeta'(s)}{\zeta(s)} = \sum_p \frac{\ln p}{p^s - 1}$$

#### 步骤3：零点分布详细证明

**关键定理**: $\zeta(s) \neq 0$ 对 $\text{Re}(s) \geq 1$。

*证明（Hadamard-de la Vallée Poussin）*:

**步骤3a: 在 $\text{Re}(s) > 1$ 无零点**

由欧拉乘积：
$$\zeta(s) = \prod_p \frac{1}{1 - p^{-s}}$$

每个因子在 $\text{Re}(s) > 0$ 非零，故 $\zeta(s) \neq 0$ 对 $\text{Re}(s) > 1$。

**步骤3b: 在 $\text{Re}(s) = 1$ 无零点**

利用恒等式（对 $\sigma > 1$）：
$$\ln|\zeta(s)| = \sum_p \sum_{m=1}^\infty \frac{1}{m} p^{-m\sigma} \cos(mt\ln p)$$

考虑：
$$\zeta(\sigma)^3 |\zeta(\sigma + it)|^4 |\zeta(\sigma + 2it)|$$

取对数：
$$3\ln\zeta(\sigma) + 4\ln|\zeta(\sigma+it)| + \ln|\zeta(\sigma+2it)|$$
$$= \sum_p \sum_{m=1}^\infty \frac{1}{m} p^{-m\sigma} (3 + 4\cos\theta + \cos 2\theta)$$

其中 $\theta = mt\ln p$。

利用 $3 + 4\cos\theta + \cos 2\theta = 2(1 + \cos\theta)^2 \geq 0$，得：
$$\zeta(\sigma)^3 |\zeta(\sigma + it)|^4 |\zeta(\sigma + 2it)| \geq 1$$

**步骤3c: 导出矛盾**

假设 $\zeta(1 + it) = 0$（$t \neq 0$）。

当 $\sigma \to 1^+$：
- $\zeta(\sigma) \sim \frac{1}{\sigma - 1}$
- $|\zeta(\sigma + it)| \sim C(\sigma - 1)$（由零点一阶性）
- $|\zeta(\sigma + 2it)|$ 有界

左边 $\sim (\sigma-1)^{-3} \cdot (\sigma-1)^4 \cdot O(1) = O(\sigma-1) \to 0$，与 $\geq 1$ 矛盾。

因此 $\zeta(1 + it) \neq 0$。 $\square$

#### 步骤4：显式公式详细推导

**von Mangoldt函数**: 
$$\Lambda(n) = \begin{cases} \ln p & n = p^k \\ 0 & \text{否则} \end{cases}$$

**Chebyshev函数**: $\psi(x) = \sum_{n \leq x} \Lambda(n)$

**显式公式推导**:

由Perron公式，对 $c > 1$：
$$\psi(x) = \frac{1}{2\pi i} \int_{c-i\infty}^{c+i\infty} \left(-\frac{\zeta'(s)}{\zeta(s)}\right) \frac{x^s}{s} ds$$

将积分线向左移动到 $\text{Re}(s) = -\infty$，经过：
- $s = 1$：一阶极点，留数为 $x$
- $s = \rho$（非平凡零点）：留数为 $-\frac{x^\rho}{\rho}$
- $s = 0$：留数为 $-\frac{\zeta'(0)}{\zeta(0)}$
- $s = -2, -4, \ldots$（平凡零点）：贡献为 $O(\ln^2 x)$

因此：
$$\psi(x) = x - \sum_\rho \frac{x^\rho}{\rho} - \frac{\zeta'(0)}{\zeta(0)} + O(\ln^2 x)$$ $\square$

#### 步骤5：渐近推导

**步骤5a: 从 $\psi(x)$ 到 $\pi(x)$**

由Abel求和：
$$\pi(x) = \sum_{p \leq x} 1 = \frac{\psi(x)}{\ln x} + \int_2^x \frac{\psi(t)}{t \ln^2 t} dt$$

**步骤5b: 估计余项**

由零free区域，存在 $c > 0$ 使得 $\text{Re}(\rho) \leq 1 - \frac{c}{\ln|\text{Im}(\rho)|}$。

由此可得：
$$\sum_\rho \frac{x^\rho}{\rho} = O(x \exp(-c'\sqrt{\ln x})) = o(x)$$

因此：
$$\psi(x) = x + o(x) \sim x$$

**步骤5c: 完成证明**

$$\pi(x) = \frac{x}{\ln x} + o\left(\frac{x}{\ln x}\right) + \int_2^x \frac{t}{t \ln^2 t} dt$$

计算积分：
$$\int_2^x \frac{1}{\ln^2 t} dt = \frac{x}{\ln^2 x} + O\left(\frac{x}{\ln^3 x}\right)$$

因此：
$$\pi(x) = \frac{x}{\ln x} + O\left(\frac{x}{\ln^2 x}\right) \sim \frac{x}{\ln x}$$ $\square$

## 推论与应用

### 重要推论

1. **第n素数**：$p_n \sim n \ln n$

2. **素数间隙**：$p_{n+1} - p_n$ 的平均阶为 $\ln p_n$

3. **孪生素数密度**：若孪生素数无限，则其密度为 $O(x/\ln^2 x)$

### 应用示例

| 应用 | 说明 |
|-----|------|
| 密码学 | RSA素数生成的规模估计 |
| 算法分析 | 涉及素数的算法复杂度 |
| 随机模型 | Cramér模型，素数的随机行为 |
