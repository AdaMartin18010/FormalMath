## 一、理论概述

### 1.1 理论定位

偏微分方程理论研究**多元函数的微分方程**，是描述物理、工程、金融等领域现象的核心数学工具。从经典的椭圆、抛物、双曲分类到现代的**Sobolev空间**和**正则性理论**，PDE理论已成为连接分析、几何和概率的庞大体系。

```mermaid

```

### 1.2 核心思想

| 核心思想 | 描述 | 重要性 |
|---------|------|-------|
| **弱解** | 分布意义下的解 | 存在性 |
| **Sobolev空间** | 弱可微函数空间 | 函数框架 |
| **先验估计** | 解的正则性控制 | 存在性证明 |
| **正则性** | 弱解的光滑性 | 经典解 |

## 三、支撑定理(L2)清单

| 定理名称 | 陈述 | 重要性 |
