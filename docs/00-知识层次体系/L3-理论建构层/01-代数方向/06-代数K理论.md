## 一、理论概述

### 1.1 理论定位

代数K理论研究**环**上的**线性代数**的不变量，从Grothendieck群构造出发，推广到高阶K群。

### 1.2 核心思想

- **K₀**: 投射模的Grothendieck群，维数的推广
- **K₁**: 可逆矩阵的Abel化，行列式的推广
- **K₂**: Steinberg群与符号，同调代数构造
- **高阶K**: Quillen的+Q构造，谱序列方法

---

## 二、核心定义(L1)清单

### 2.1 K₀群

设R是环，**K₀(R)** 定义为投射有限生成R-模的Grothendieck群：

$$K_0(R) = \frac{\text{自由阿贝尔群}\{[P] : P \text{投射}\}}{[P \oplus Q] = [P] + [Q]}$$

**约化K₀**: $$\tilde{K}_0(R) = K_0(R) / \mathbb{Z}$$

### 2.2 K₁群

$$K_1(R) = GL(R)^{ab} = GL(R) / [GL(R), GL(R)]$$

对交换环：$$K_1(R) \cong R^* \oplus SK_1(R)$$

### 2.3 K₂群

Milnor定义：$$K_2(R) = H_2(E(R), \mathbb{Z})$$

或用Steinberg群：$$K_2(R) = \ker(\text{St}(R) \to E(R))$$
