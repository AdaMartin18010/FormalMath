## 一、理论概述

### 1.1 理论定位

Galois理论建立了**域扩张**与**群论**之间的深刻联系，通过Galois群刻画多项式方程的可解性。从经典的有限Galois理论到现代的**无穷Galois理论**和**绝对Galois群**，这一理论成为数论、代数几何和算术几何的基石。


### 1.2 核心思想

| 核心思想 | 描述 | 重要性 |
|---------|------|-------|
| **对称性对应** | 域扩张 ↔ 子群 | 基本对偶 |
| **可解性** | 方程可解⇔群可解 | 代数方程 |
| **绝对Galois群** | G_Q = Gal(Q̄/Q) | 数论核心 |
| **逆Galois问题** | 哪些群是Galois群 | 开放问题 |
