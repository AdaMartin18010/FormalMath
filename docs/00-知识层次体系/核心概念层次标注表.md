- **L3**: 群论、群范畴 - 作为代数结构和范畴论的研究对象
- **L4**: 有限单群分类 - 20世纪数学的伟大成就，所有有限单群的完整分类

#### 示例2: 连续性 (分析学)
- **L0**: 不断裂的曲线 - 直观的连续概念
- **L1**: ε-δ定义 - Weierstrass的严格形式化定义
- **L2**: 介值定理、最值定理 - 连续函数的基本性质
- **L3**: 连续函数空间 - C(X)作为Banach空间的研究

#### 示例3: 基本群 (拓扑学)

- **L3**: 覆叠空间理论 - 基本群与覆叠空间的对应

## 层次标注方法论

### 标注原则

1. **层次递进性**: 每个概念的L0到L4应呈现明显的认知递进
2. **准确性**: L1定义应严格符合数学标准定义
3. **完整性**: 尽量覆盖概念的主要方面
4. **前沿性**: L4应反映当前数学研究的活跃领域

### 层次特征

| 层次 | 认知水平 | 典型表述 |
|------|----------|----------|
| L0 | 直觉认知 | 比喻、图示、实例 |
| L1 | 形式定义 | 符号、公理、严格表述 |
| L2 | 核心性质 | 定理、引理、判别法 |
| L3 | 理论整合 | 结构、系统、范畴 |
