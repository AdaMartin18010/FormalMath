## 一、命题逻辑概念

### 1. 命题 (Proposition)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "能判断真假的陈述句"，如"今天是星期一" |
| **L1** | 形式化定义 | 有确定真值（真或假）的语句 |
| **L2** | 定理证明 | 真值计算、真值表、逻辑等价 |
| **L3** | 理论建构 | 布尔代数、Heyting代数、范畴论语义 |
| **L4** | 前沿研究 | 量子命题、模糊逻辑 |

---

### 2. 逻辑联结词 (Logical Connectives)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "并且"、"或者"、"非"、"如果...那么..." |
| **L1** | 形式化定义 | $\land$ (合取)、$\lor$ (析取)、$\neg$ (否定)、$\to$ (蕴含)、$\leftrightarrow$ (等价) |
| **L2** | 定理证明 | 真值表完备性、De Morgan律、分配律 |
| **L3** | 理论建构 | 自由代数、范畴逻辑、proof net |
| **L4** | 前沿研究 | 线性逻辑联结词、量子逻辑 |

---

### 3. 命题逻辑 (Propositional Logic)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "研究命题及其联结词的推理系统" |
| **L1** | 形式化定义 | 语法：公式递归定义；语义：真值赋值 |
| **L2** | 定理证明 | 可靠性、完备性、紧致性 |
| **L3** | 理论建构 | 范畴语义、代数逻辑、拓扑斯语义 |
| **L4** | 前沿研究 | 量子命题逻辑、概率逻辑 |

## 二、谓词逻辑概念

### 4. 谓词 (Predicate)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "带变量的性质或关系"，如"x是素数"、"x大于y" |
| **L1** | 形式化定义 | $P(x_1, ..., x_n)$，从个体域到真值的函数 |
| **L2** | 定理证明 | 替换定理、变元的自由与约束 |
| **L3** | 理论建构 | 高阶谓词、类型论、依赖类型 |
| **L4** | 前沿研究 | 高阶抽象语法、名义集合 |

---

### 5. 量词 (Quantifier)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "所有"（$\forall$）和"存在"（$\exists$） |
| **L1** | 形式化定义 | $\forall x P(x)$、$\exists x P(x)$ 的真值定义 |
| **L2** | 定理证明 | 量词否定规则、量词分配律、前束范式 |
| **L3** | 理论建构 | Skolem化、Herbrand定理、高阶量词 |
| **L4** | 前沿研究 | 一般量词、依赖量词、线性量词 |

---

### 6. 谓词逻辑 (Predicate Logic / First-order Logic)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "包含量词的推理系统"，表达更丰富 |
| **L1** | 形式化定义 | 一阶语言：函数符号、谓词符号、量词 |
| **L2** | 定理证明 | Gödel完备性定理、Löwenheim-Skolem定理、紧致性 |
| **L3** | 理论建构 | 模型论基础、超积、型论 |
| **L4** | 前沿研究 | 连续逻辑、不稳定理论 |

**跃迁路径**: 

```

自然语言推理 → 一阶语法语义 → Gödel完备性 → 模型论 → 连续逻辑

```

## 三、证明论概念

### 7. 形式系统 (Formal System)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "严格公理化系统"，如欧氏几何公理 |
| **L1** | 形式化定义 | 语言、公理、推理规则三元组 $(\mathcal{L}, A, R)$ |
| **L2** | 定理证明 | 一致性、完备性、独立性 |
| **L3** | 理论建构 | 可计算性理论、证明复杂性 |
| **L4** | 前沿研究 | 证明挖掘、逆向数学 |

---

### 8. 证明 (Proof)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "从公理到定理的推导过程" |
| **L1** | 形式化定义 | 公式序列，每步由公理或规则得到 |
| **L2** | 定理证明 | 演绎定理、切割消除、范式 |
| **L3** | 理论建构 | 证明网、线性逻辑、Curry-Howard对应 |
| **L4** | 前沿研究 | 证明复杂度、交互式证明 |

---

### 9. Curry-Howard对应

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "证明与程序是一回事" |
| **L1** | 形式化定义 | 命题 = 类型，证明 = 程序，证明归约 = 程序求值 |
| **L2** | 定理证明 | 同构的证明、强规范化、Church-Rosser性质 |
| **L3** | 理论建构 | 依赖类型、Martin-Löf类型论、proof assistant |
| **L4** | 前沿研究 | 同伦类型论、立方类型论 |

---

## 四、模型论概念

### 10. 模型 (Model)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "使公式为真的数学结构" |
| **L1** | 形式化定义 | 结构 $\mathcal{M} = (D, I)$，$\mathcal{M} \models \phi$ |
| **L2** | 定理证明 | 可满足性、逻辑后承、初等等价 |
| **L3** | 理论建构 | 超积、紧致性应用、Morley定理 |
| **L4** | 前沿研究 | 连续逻辑模型论、稳定性理论 |

---

### 11. 紧致性定理 (Compactness Theorem)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "局部可满足则整体可满足" |
| **L1** | 形式化定义 | 若每个有限子集可满足，则整体可满足 |
| **L2** | 定理证明 | 拓扑证明（Tychonoff定理）、语法证明 |
| **L3** | 理论建构 | 型空间、饱和模型、超积构造 |
| **L4** | 前沿研究 | 连续逻辑紧致性、概率紧致性 |

---

### 12. Löwenheim-Skolem定理

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "无穷模型有不同基数的模型" |
| **L1** | 形式化定义 | 无穷模型有任意大的模型，也有可数模型 |
| **L2** | 定理证明 | 向下和向上LS定理、初等子模型 |
| **L3** | 理论建构 | Skolem函数、不可辨元、模型论力迫 |
| **L4** | 前沿研究 | 连续逻辑的LS定理、大基数效应 |

---

## 五、集合论概念

### 13. 集合论 (Set Theory)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "对象的汇集"、"集合的集合" |
| **L1** | 形式化定义 | ZFC公理系统：外延、空集、配对、并集、幂集、分离、替换、无穷、选择、正则 |
| **L2** | 定理证明 | 序数理论、基数理论、超限归纳 |
| **L3** | 理论建构 | 内模型、可构造宇宙L、力迫法 |
| **L4** | 前沿研究 | 大基数、决定性公理、内模型纲领 |

**跃迁路径**: 

```

朴素集合论 → ZFC公理 → 超限算术 → 力迫法/内模型 → 大基数理论

```

---

### 14. ZFC公理系统

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "现代数学的基础公理" |
| **L1** | 形式化定义 | 十条公理：外延、空集、配对、并集、幂集、分离、无穷、替换、选择、正则 |
| **L2** | 定理证明 | 公理的一致性、独立性、相对一致性 |
| **L3** | 理论建构 | 力迫法、内模型、传递模型 |
| **L4** | 前沿研究 | 新公理、决定性公理、大基数公理 |

---

### 15. 大基数 (Large Cardinals)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "比所有可构造集合都大的基数" |
| **L1** | 形式化定义 | 不可达基数、Mahlo基数、可测基数、超紧基数等 |
| **L2** | 定理证明 | 大基数的层次、一致性强度 |
| **L3** | 理论建构 | 内模型、决定性、描述集合论 |
| **L4** | 前沿研究 | 内模型纲领、终极L、力迫公理 |

## 六、递归论概念

### 16. 可计算性 (Computability)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "能用算法计算"的函数 |
| **L1** | 形式化定义 | Turing可计算、$\lambda$可计算、递归函数 |
| **L2** | 定理证明 | Church-Turing论题、通用图灵机、停机问题 |
| **L3** | 理论建构 | 可计算枚举集、算术层次、图灵度 |
| **L4** | 前沿研究 | 随机性、可计算分析、量子计算 |

---

### 17. Gödel不完备定理

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "足够强的形式系统不能证明所有真命题" |
| **L1** | 形式化定义 | 第一不完备：一致则不完备；第二不完备：不能证自身一致性 |
| **L2** | 定理证明 | Gödel编码、对角线引理、自指命题 |
| **L3** | 理论建构 | 可证性逻辑、算术化元数学 |
| **L4** | 前沿研究 | 不完全性的速度、物理中的不完备性 |

**跃迁路径**: 

```

元数学直觉 → Gödel编码 → 对角线引理 → 可证性逻辑 → 现代元数学

```

## 七、同伦类型论

### 18. 同伦类型论 (Homotopy Type Theory)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "类型就是空间，证明就是路径" |
| **L1** | 形式化定义 | Martin-Löf类型论 + 同伦论解释 |
| **L2** | 定理证明 | 同论等价即相等、高阶归纳类型 |
| **L3** | 理论建构 | 无穷群胚模型、立方类型论 |
| **L4** | 前沿研究 | 谱类型论、综合同伦论、计算机形式化 |
