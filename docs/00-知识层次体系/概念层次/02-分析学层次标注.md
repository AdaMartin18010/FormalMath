## 一、实分析核心概念

### 1. 实数 (Real Numbers)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | 数轴上的点，可以任意精确测量，"连续"的数 |
| **L1** | 形式化定义 | Dedekind分割：$(A,B)$ 分割有理数；或Cauchy序列的等价类 |
| **L2** | 定理证明 | 完备性定理、确界原理、区间套定理、Bolzano-Weierstrass定理 |
| **L3** | 理论建构 | 完备化理论、实数作为拓扑空间的泛性质 |
| **L4** | 前沿研究 | 非标准分析、超实数、可计算分析 |

**跃迁路径**:

```

数轴直觉 → Dedekind分割/Cauchy序列 → 完备性定理 → 完备化理论 → 非标准分析

```

---

### 2. 极限 (Limit)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "越来越接近某个值"，数列1, 1/2, 1/3, ... 接近0 |
| **L1** | 形式化定义 | $\forall \varepsilon > 0, \exists N, n > N \Rightarrow |a_n - L| < \varepsilon$ |
| **L2** | 定理证明 | 极限唯一性、夹逼定理、极限的运算性质 |
| **L3** | 理论建构 | 滤子极限、网的收敛、拓扑收敛 |
| **L4** | 前沿研究 | 非标准极限、超收敛 |

---

### 3. 连续性 (Continuity)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "一笔画成的曲线，没有断开"，无跳跃 |
| **L1** | 形式化定义 | $\varepsilon$-$\delta$定义：$\forall \varepsilon > 0, \exists \delta > 0, |x-a| < \delta \Rightarrow |f(x)-f(a)| < \varepsilon$ |
| **L2** | 定理证明 | 连续函数的运算性质、中间值定理、最值定理 |
| **L3** | 理论建构 | 拓扑连续性、连续函子、层的连续性 |
| **L4** | 前沿研究 | 可计算连续性、构造性分析 |

**跃迁路径**:

```

一笔画直觉 → ε-δ定义 → 连续函数性质 → 拓扑连续 → 可计算分析

```

---

### 4. 导数 (Derivative)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "切线的斜率"、"瞬时变化率"，汽车速度表的概念 |
| **L1** | 形式化定义 | $f'(x) = \lim_{h \to 0} \frac{f(x+h)-f(x)}{h}$ |
| **L2** | 定理证明 | 求导法则、链式法则、中值定理、Taylor展开 |
| **L3** | 理论建构 | 切丛理论、微分形式、分布理论 |
| **L4** | 前沿研究 | 非交换微分、分数阶导数、变形量化 |

**典型例子在各层次的表现**:

- L0: 曲线在某点的倾斜程度
- L1: $f(x) = x^2$ 的导数是 $2x$
- L2: 中值定理证明、Taylor展开
- L3: 切空间、微分形式
- L4: 非交换几何中的导数

---

### 5. 微分 (Differential)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "微小变化量"，$df$ 表示函数 $f$ 的微小变化 |
| **L1** | 形式化定义 | $df = f'(x)dx$，线性部分 |
| **L2** | 定理证明 | 微分中值定理、一阶近似、误差估计 |
| **L3** | 理论建构 | 切映射、微分形式的外微分、de Rham上同调 |
| **L4** | 前沿研究 | 非交换微分、微分几何中的微分 |

---

### 6. 积分 (Integral)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "面积的累加"，曲线下方的区域面积 |
| **L1** | 形式化定义 | Riemann积分：分割、求和、取极限 |
| **L2** | 定理证明 | Newton-Leibniz公式、积分中值定理、换元积分 |
| **L3** | 理论建构 | Lebesgue积分、测度论、分布的积分 |
| **L4** | 前沿研究 | 非交换积分、量子积分、随机积分 |

---

### 7. 微积分基本定理

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "微分和积分互为逆运算" |
| **L1** | 形式化定义 | $\int_a^b f'(x)dx = f(b) - f(a)$ |
| **L2** | 定理证明 | 使用中值定理证明、推广的Stokes定理 |
| **L3** | 理论建构 | Stokes定理、外微分的微积分基本定理 |
| **L4** | 前沿研究 | 非交换几何中的推广 |

---

### 8. 多元函数

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "多个变量的函数"，如 $z = f(x,y)$ 表示曲面 |
| **L1** | 形式化定义 | $f: \mathbb{R}^n \to \mathbb{R}^m$，多变量映射 |
| **L2** | 定理证明 | 连续性、极限的运算、向量值函数 |
| **L3** | 理论建构 | 多元映射的分析、Banach空间中的映射 |
| **L4** | 前沿研究 | 无穷维空间中的函数 |

---

### 9. 偏导数 (Partial Derivative)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "沿某一坐标轴方向的变化率" |
| **L1** | 形式化定义 | $\frac{\partial f}{\partial x_i} = \lim_{h \to 0} \frac{f(x+he_i)-f(x)}{h}$ |
| **L2** | 定理证明 | 混合偏导数相等、链式法则、隐函数求导 |
| **L3** | 理论建构 | 梯度、Hessian矩阵、Jacobi矩阵 |
| **L4** | 前沿研究 | 弱导数、Sobolev空间 |

---

### 10. 重积分 (Multiple Integral)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "多维区域的体积/面积" |
| **L1** | 形式化定义 | Riemann重积分、累次积分 |
| **L2** | 定理证明 | Fubini定理、变量替换公式、极坐标变换 |
| **L3** | 理论建构 | Lebesgue重积分、流形上的积分 |
| **L4** | 前沿研究 | 随机多重积分、无穷维积分 |

## 二、级数理论概念

### 11. 级数 (Series)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "无穷多个数相加"，$1 + \frac{1}{2} + \frac{1}{4} + ...$ |
| **L1** | 形式化定义 | 部分和序列 $S_n = \sum_{k=1}^n a_k$，级数收敛即 $S_n$ 收敛 |
| **L2** | 定理证明 | 收敛的必要条件、Cauchy收敛准则、比较判别法 |
| **L3** | 理论建构 | Banach空间中的级数、无条件收敛、弱收敛 |
| **L4** | 前沿研究 | 可求和性、超求和、渐近级数 |

---

### 12. 一致收敛 (Uniform Convergence)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "整体一起收敛"，收敛速度不依赖于点 |
| **L1** | 形式化定义 | $\forall \varepsilon > 0, \exists N, \forall n \geq N, \forall x: |f_n(x)-f(x)| < \varepsilon$ |
| **L2** | 定理证明 | Weierstrass M判别法、一致收敛保持连续性/可积性 |
| **L3** | 理论建构 | 函数空间中的收敛、Banach空间中的收敛 |
| **L4** | 前沿研究 | 拟一致收敛、随机一致收敛 |

---

## 三、复分析核心概念

### 13. 复数 (Complex Numbers)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "实数加上虚数 $i$"，复平面的点 |
| **L1** | 形式化定义 | $z = x + iy$，其中 $i^2 = -1$；或有序对 $(x,y)$ |
| **L2** | 定理证明 | 代数基本定理、共轭性质、模的性质 |
| **L3** | 理论建构 | 复数作为域、复数的几何、复结构 |
| **L4** | 前沿研究 | 超复数、四元数、八元数 |

---

### 14. 全纯函数 (Holomorphic Function)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "复可微的函数"，比实可微强得多的条件 |
| **L1** | 形式化定义 | 复导数存在：$\lim_{h \to 0} \frac{f(z+h)-f(z)}{h}$ |
| **L2** | 定理证明 | Cauchy-Riemann方程、解析性、幂级数展开 |
| **L3** | 理论建构 | 全纯层、复流形、Riemann面 |
| **L4** | 前沿研究 | 多复变函数、拟全纯映射、Gromov-Witten理论 |

---

### 15. Cauchy积分定理

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "全纯函数沿闭路的积分为零" |
| **L1** | 形式化定义 | $\oint_C f(z)dz = 0$，$f$ 全纯，$C$ 单连通区域内的闭路 |
| **L2** | 定理证明 | Goursat证明、Green定理方法、同调论证明 |
| **L3** | 理论建构 | 同调论视角、层的上同调、留数理论 |
| **L4** | 前沿研究 | 高维推广、Atiyah-Singer指标定理 |

**跃迁路径**:

```

复积分直觉 → 全纯条件 → Cauchy定理 → 层上同调 → 指标定理

```

---

### 16. 留数定理 (Residue Theorem)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "极点处的系数决定积分值" |
| **L1** | 形式化定义 | $\oint_C f(z)dz = 2\pi i \sum \text{Res}(f, a_k)$ |
| **L2** | 定理证明 | Laurent展开、留数计算、围道积分技巧 |
| **L3** | 理论建构 | Riemann-Roch定理、指标定理 |
| **L4** | 前沿研究 | 高维留数、Grothendieck留数 |

---

## 四、泛函分析核心概念

### 17. 度量空间 (Metric Space)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "有距离概念的空间"，可以测量点之间的距离 |
| **L1** | 形式化定义 | $(X, d)$，$d: X \times X \to \mathbb{R}_{\geq 0}$ 满足：正定性、对称性、三角不等式 |
| **L2** | 定理证明 | 开球、开集、收敛序列、Cauchy序列 |
| **L3** | 理论建构 | 完备化、拓扑空间、一致空间 |
| **L4** | 前沿研究 | 非交换度量空间、粗几何 |

---

### 18. Banach空间

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "完备的赋范空间"，向量空间加上长度概念和完备性 |
| **L1** | 形式化定义 | 赋范向量空间 $(V, \|\cdot\|)$，且关于范数完备 |
| **L2** | 定理证明 | 完备性判别、Banach空间中的级数、商空间 |
| **L3** | 理论建构 | 对偶空间、Hahn-Banach定理、一致有界原理 |
| **L4** | 前沿研究 | 算子代数、Banach空间几何 |

---

### 19. Hilbert空间

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "有内积的完备空间"，可以度量角度和长度 |
| **L1** | 形式化定义 | 完备的内积空间，内积诱导范数 |
| **L2** | 定理证明 | Cauchy-Schwarz不等式、正交性、正交分解 |
| **L3** | 理论建构 | 正交基、Riesz表示定理、谱定理 |
| **L4** | 前沿研究 | 量子力学基础、von Neumann代数 |

## 五、微分方程概念

### 20. 常微分方程 (ODE)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "含有未知函数及其导数的方程"，描述变化规律 |
| **L1** | 形式化定义 | $F(x, y, y', ..., y^{(n)}) = 0$ |
| **L2** | 定理证明 | 存在唯一性定理、解的结构、初值问题 |
| **L3** | 理论建构 | 动力系统、相空间、稳定性理论 |
| **L4** | 前沿研究 | 混沌理论、分岔理论、可积系统 |

---

### 21. 偏微分方程 (PDE)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "含有多元函数及其偏导数的方程"，描述物理场 |
| **L1** | 形式化定义 | $F(x_1,...,x_n, u, \frac{\partial u}{\partial x_1}, ...) = 0$ |
| **L2** | 定理证明 | 特征线法、分离变量法、格林函数 |
| **L3** | 理论建构 | Sobolev空间、弱解、变分方法 |
| **L4** | 前沿研究 | Navier-Stokes方程、几何PDE、随机PDE |

**跃迁路径**:

```

物理直觉 → PDE方程 → 解的存在唯一性 → Sobolev空间理论 → 几何PDE

```
