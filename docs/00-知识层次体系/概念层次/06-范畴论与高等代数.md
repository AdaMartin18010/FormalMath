## 一、范畴论基础概念

### 1. 范畴 (Category)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "由对象和态射构成的数学结构" |
| **L1** | 形式化定义 | 对象类、态射集、复合运算、单位态射，满足结合律和单位律 |
| **L2** | 定理证明 | 态射的性质、具体范畴的例子 |
| **L3** | 理论建构 | 范畴的范畴、2-范畴、高阶范畴 |
| **L4** | 前沿研究 | 无穷范畴、导出范畴、$(\infty,1)$-范畴 |

**跃迁路径**: 

```

结构直觉 → 范畴公理 → 范畴定理 → 高阶范畴论 → 无穷范畴

```

### 2. 函子 (Functor)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "保持结构的映射"，范畴之间的对应 |
| **L1** | 形式化定义 | $F: \mathcal{C} \to \mathcal{D}$，保持复合和单位 |
| **L2** | 定理证明 | 函子的复合、恒等函子、忠实/满函子 |
| **L3** | 理论建构 | 自然变换、函子范畴、Yoneda引理 |
| **L4** | 前沿研究 | 导出函子、DG函子、无穷函子 |

---

### 3. 自然变换 (Natural Transformation)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "函子之间的自然对应"，不依赖具体选择的映射 |
| **L1** | 形式化定义 | $\eta: F \Rightarrow G$，对每个对象给出态射，满足自然性条件 |
| **L2** | 定理证明 | 垂直与水平复合、自然同构 |
| **L3** | 理论建构 | 函子范畴、2-范畴结构、自然性即交换图 |
| **L4** | 前沿研究 | 高阶自然变换、无穷自然变换 |

---

### 4. Yoneda引理

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "一个对象由它与其他对象的关系完全确定" |
| **L1** | 形式化定义 | $\text{Nat}(h_c, F) \cong F(c)$，自然同构 |
| **L2** | 定理证明 | 引理的证明、Yoneda嵌入 |
| **L3** | 理论建构 | 可表函子、普遍元素、泛性质 |
| **L4** | 前沿研究 | 高阶Yoneda引理、$(\infty,1)$-Yoneda |

---

### 5. 泛性质 (Universal Property)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "在同构意义下唯一的最佳构造" |
| **L1** | 形式化定义 | 始对象、终对象、积、余积、等化子、余等化子 |

## 二、同调代数概念

### 6. 链复形 (Chain Complex)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "一串带有边缘映射的对象序列" |
| **L1** | 形式化定义 | $... \to C_{n+1} \xrightarrow{\partial_{n+1}} C_n \xrightarrow{\partial_n} C_{n-1} \to ...$，满足 $\partial^2 = 0$ |
| **L2** | 定理证明 | 边缘映射的性质、链映射、链同伦 |
| **L3** | 理论建构 | 同调群、长正合序列、同伦范畴 |
| **L4** | 前沿研究 | DG范畴、$A_\infty$代数、导出范畴 |

---

### 7. 同调群 (Homology Group)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "度量'洞'的代数不变量" |
| **L1** | 形式化定义 | $H_n = \ker\partial_n / \text{im}\partial_{n+1}$ |
| **L2** | 定理证明 | 同调的函子性、同伦不变性、切除定理 |
| **L3** | 理论建构 | 导出函子、Tor和Ext、谱序列 |
| **L4** | 前沿研究 | 周期同调、循环同调、高阶同调 |

---

### 8. 导出函子 (Derived Functor)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "测量函子'非正合性'的工具" |
| **L1** | 形式化定义 | $L_nF$ 和 $R^nF$，使用投射/内射分解 |
| **L2** | 定理证明 | 长正合序列、平衡性、Grothendieck谱序列 |
| **L3** | 理论建构 | 导出范畴、$t$-结构、导出张量积 |
| **L4** | 前沿研究 | 高阶导出、非交换导出、导出代数几何 |

---

### 9. Ext与Tor

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | Ext测量扩张，Tor测量挠率 |
| **L1** | 形式化定义 | $\text{Ext}^n_R(M,N) = R^n\text{Hom}_R(M,N)$，$\text{Tor}_n^R(M,N) = L_n(M \otimes_R N)$ |
| **L2** | 定理证明 | 长正合序列、Ext与扩张的对应、Tor与平坦性 |
| **L3** | 理论建构 | 同调维数、全局维数、局部对偶 |
| **L4** | 前沿研究 | 高阶Ext、微局部同调、矩阵因子化 |

### 11. 层上同调 (Sheaf Cohomology)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
