## 一、群论概念

### 1. 二元运算

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "两个元素合成一个"的操作直觉，如加法、乘法的直观感受 |
| **L1** | 形式化定义 | 映射 $*: G \times G \to G$，满足封闭性 |
| **L2** | 性质定理 | 运算表、运算的结合性检验、运算的交换性检验 |
| **L3** | 理论框架 | 泛代数中的运算概念、 operad 理论 |
| **L4** | 前沿研究 | 量子群中的变形运算、非交换几何 |

**跃迁路径**: 直观运算 → 映射定义 → 运算性质 → 泛代数框架 → 量子变形

---

### 2. 群 (Group)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "有对称性的对象"，如等边三角形的旋转对称、正方形的对称变换 |
| **L1** | 形式化定义 | 集合 $G$ 配备二元运算 $\cdot$，满足：结合律、单位元、逆元 |
| **L2** | 定理证明 | 单位元唯一性、逆元唯一性、消去律、方程 $ax=b$ 可解性 |
| **L3** | 理论建构 | Grp范畴、群作为群对象、群的表示论 |
| **L4** | 前沿研究 | 量子群、有限单群分类、几何群论 |

**跃迁路径**:

```

对称性直觉 → 群公理系统 → 基本定理网络 → 群范畴论 → 现代群论前沿

```

**典型例子在各层次的表现**:
- L0: 等边三角形的6个对称变换
- L1: $(\mathbb{Z}, +)$，$(\mathbb{Q}^*, \cdot)$，对称群 $S_n$
- L2: 群的同态性质、子群判定
- L3: 群作为范畴、群的分类问题
- L4: 群的上同调、 Langlands 对应

---

### 3. 子群 (Subgroup)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | 群的"一部分"，继承原群的运算 |
| **L1** | 形式化定义 | $H \subseteq G$，满足：非空、对运算封闭、对逆元封闭 |
| **L2** | 定理证明 | 子群判定定理、子群的交仍是子群、由子集生成的子群 |
| **L3** | 理论建构 | 子群格、子群作为子对象、格论视角 |
| **L4** | 前沿研究 | 子群复杂度、子群增长问题 |

**层次依赖**: 群 (L1) → 子群 (L1) → 陪集 (L2)

---

### 4. 正规子群 (Normal Subgroup)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "左右都一样的子群"，共轭作用下的不变子群 |
| **L1** | 形式化定义 | $N \leq G$，满足 $\forall g \in G: gNg^{-1} = N$ |
| **L2** | 定理证明 | 正规子群的等价刻画、核必为正规子群、商群良定义 |
| **L3** | 理论建构 | 正规子群作为核、正合列中的概念 |
| **L4** | 前沿研究 | 有限单群分类中的角色、几乎正规子群 |

**层次依赖**: 子群 (L1) → 共轭作用 (L2) → 正规子群 (L2)

---

### 5. 商群 (Quotient Group)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "把正规子群压成一个点"，类似整数模n剩余类 |
| **L1** | 形式化定义 | $G/N = \{gN : g \in G\}$，运算 $(gN)(hN) = (gh)N$ |
| **L2** | 定理证明 | 运算良定义性、自然同态、第一同构定理 |
| **L3** | 理论建构 | 商作为余等化子、商范畴、正合序列 |
| **L4** | 前沿研究 | 导出范畴中的商、商群的上同调 |

**层次依赖**: 正规子群 (L2) → 陪集 (L2) → 商群 (L2)

---

### 6. 群同态 (Group Homomorphism)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "保持群结构的映射"，如指数映射 $\exp: (\mathbb{R},+) \to (\mathbb{R}^+, \cdot)$ |
| **L1** | 形式化定义 | $\phi: G \to H$，满足 $\phi(ab) = \phi(a)\phi(b)$ |
| **L2** | 定理证明 | 同态保持单位元和逆元、同态的核与像、同态基本定理 |
| **L3** | 理论建构 | Grp范畴中的态射、同态诱导的映射、函子性 |
| **L4** | 前沿研究 | 高阶同态、导出范畴中的态射 |

---

### 7. 同构定理 (Isomorphism Theorems)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "结构相同的群看起来一样" |
| **L1** | 形式化定义 | 双射同态，$G \cong H$ |
| **L2** | 定理证明 | 第一同构定理：$G/\ker\phi \cong \text{im}\phi$；第二、第三同构定理 |
| **L3** | 理论建构 | Abel范畴中的同构定理、同构作为等价关系 |
| **L4** | 前沿研究 | 导出等价、Morita等价 |

---

### 8. Sylow定理

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "极大p子群的存在与控制" |
| **L1** | 形式化定义 | Sylow p子群：阶为 $p^n$ 其中 $p^n \mid |G|$ 但 $p^{n+1} \nmid |G|$ |
| **L2** | 定理证明 | 存在性、共轭性、计数定理 |
| **L3** | 理论建构 | Sylow理论、p群的结构理论 |
| **L4** | 前沿研究 | fusion系统、有限单群分类中的局部理论 |

## 二、环与域概念

### 9. 环 (Ring)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "有加法和乘法两种运算的系统"，如整数、多项式 |
| **L1** | 形式化定义 | $(R, +, \cdot)$：$(R,+)$ 是Abel群，$(R,\cdot)$ 是半群，满足分配律 |
| **L2** | 定理证明 | 零元性质、负元的性质、二项式定理 |
| **L3** | 理论建构 | Ring范畴、交换环几何、环的谱 |
| **L4** | 前沿研究 | 非交换几何、导出代数几何 |

**跃迁路径**:

```

算术直觉 → 环公理系统 → 环的结构定理 → 环的范畴论/代数几何 → 现代环论

```

---

### 10. 理想 (Ideal)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "类似正规子群的子环"，用于构造商环 |
| **L1** | 形式化定义 | $I \subseteq R$，满足：$(I,+)$ 子群，$\forall r \in R: rI \subseteq I, Ir \subseteq I$ |
| **L2** | 定理证明 | 理想的交与和、主理想、理想的生成 |
| **L3** | 理论建构 | 理想格、素谱 $Spec(R)$、理想层 |
| **L4** | 前沿研究 | 非交换理想理论、导出范畴中的理想 |

---

### 11. 商环 (Quotient Ring)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "模去一个理想的环"，如 $\mathbb{Z}/n\mathbb{Z}$ |
| **L1** | 形式化定义 | $R/I = \{r+I : r \in R\}$，自然定义的运算 |
| **L2** | 定理证明 | 运算良定义性、自然同态、同态基本定理 |
| **L3** | 理论建构 | 商环与环的表示、商的几何意义 |
| **L4** | 前沿研究 | 形式邻域、完备化 |

---

### 12. 整环 (Integral Domain)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "没有零因子的环"，如整数环、域 |
| **L1** | 形式化定义 | 交换环 $R$，满足：$1 \neq 0$，无零因子（$ab=0 \Rightarrow a=0$ 或 $b=0$） |
| **L2** | 定理证明 | 消去律成立、整环可嵌入域（分式域） |
| **L3** | 理论建构 | 整环的分类、UFD、PID、ED的关系 |
| **L4** | 前沿研究 | 高维代数几何中的整环 |

---

### 13. 域 (Field)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "能进行四则运算的数系"，如有理数、实数、复数 |
| **L1** | 形式化定义 | 交换环 $F$，满足：$1 \neq 0$，非零元都可逆 |
| **L2** | 定理证明 | 域的特征、素域的结构、有限域的分类 |
| **L3** | 理论建构 | 域扩张理论、Galois理论、域的代数闭包 |
| **L4** | 前沿研究 | p进域、函数域、类域论 |

**跃迁路径**:

```

数的直觉 → 域公理 → 域的结构定理 → Galois理论 → 现代数论

```

---

### 14. 域扩张 (Field Extension)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "在一个域中添加元素得到更大的域" |
| **L1** | 形式化定义 | 域 $F$ 和包含 $F$ 的域 $E$，记作 $E/F$ |
| **L2** | 定理证明 | 扩张次数、塔公式、代数元与超越元 |
| **L3** | 理论建构 | Galois对应、可分扩张、正规扩张 |
| **L4** | 前沿研究 | 逆Galois问题、绝对Galois群 |

---

### 15. Galois群

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "保持基域不动的对称变换群" |
| **L1** | 形式化定义 | $\text{Gal}(E/F) = \{\sigma \in \text{Aut}(E) : \sigma|_F = \text{id}\}$ |
| **L2** | 定理证明 | Galois群的阶与扩张次数的关系、Galois基本定理 |
| **L3** | 理论建构 | Galois理论、Galois上同调、绝对Galois群 |
| **L4** | 前沿研究 | 逆Galois问题、Langlands纲领中的Galois表示 |

## 三、模与线性代数概念

### 16. 向量空间 (Vector Space)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "有方向的量构成的空间"，如平面、空间中的向量 |
| **L1** | 形式化定义 | 域 $F$ 上的向量空间：Abel群 $(V,+)$，数乘 $F \times V \to V$，满足8条公理 |
| **L2** | 定理证明 | 基的存在性、维数不变性、维数公式 |
| **L3** | 理论建构 | Vect范畴、泛性质、对偶空间理论 |
| **L4** | 前沿研究 | 无穷维空间、拓扑向量空间 |

---

### 17. 线性映射 (Linear Map)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "保持线性结构的映射"，如旋转、缩放、投影 |
| **L1** | 形式化定义 | $T: V \to W$，满足 $T(av+bw) = aT(v) + bT(w)$ |
| **L2** | 定理证明 | 核与像、秩-零化度定理、矩阵表示 |
| **L3** | 理论建构 | 线性范畴、正合序列、函子性 |
| **L4** | 前沿研究 | 导出函子、谱序列 |

---

### 18. 模 (Module)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "环上的向量空间"，推广了向量空间的概念 |
| **L1** | 形式化定义 | 环 $R$ 上的模：Abel群 $(M,+)$，作用 $R \times M \to M$，满足公理 |
| **L2** | 定理证明 | 子模、商模、模同态、直和与直积 |
| **L3** | 理论建构 | 模范畴、正合序列、投射/内射/平坦模 |
| **L4** | 前沿研究 | 导出范畴、同调代数前沿 |

**跃迁路径**:

```

向量空间 → 模的定义 → 模的结构定理 → 同调代数 → 导出代数几何

```

## 四、李理论概念

### 19. 李代数 (Lie Algebra)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "无穷小变换的代数"，李群的切空间 |
| **L1** | 形式化定义 | 向量空间 $\mathfrak{g}$，双线性括号 $[\cdot,\cdot]$，满足反对称性和Jacobi恒等式 |
| **L2** | 定理证明 | 李子代数、理想、商代数、表示论基础 |
| **L3** | 理论建构 | 李代数分类、半单李代数、根系理论 |
| **L4** | 前沿研究 | Kac-Moody代数、顶点算子代数 |

---

### 20. 李群 (Lie Group)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "连续的群"，如旋转群、矩阵群 |
| **L1** | 形式化定义 | 既是群又是光滑流形，群运算是光滑映射 |
| **L2** | 定理证明 | 李群与李代数的对应、指数映射、伴随表示 |
| **L3** | 理论建构 | 李群表示论、齐性空间、对称空间 |
| **L4** | 前沿研究 | 无限维李群、量子群、几何Langlands纲领 |
