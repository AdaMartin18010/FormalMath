## 一、初等数论概念

### 1. 整除 (Divisibility)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "一个数能被另一个数除尽"，如6被2整除 |
| **L1** | 形式化定义 | $a \mid b \Leftrightarrow \exists q: b = aq$ |
| **L2** | 定理证明 | 整除的性质、传递性、最大公约数 |
| **L3** | 理论建构 | 赋值论、p进赋值、整除格 |
| **L4** | 前沿研究 | 高维整除理论、算术几何中的整除 |

---

### 2. 素数 (Prime Numbers)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "只能被1和自身整除的数"，如2, 3, 5, 7, ... |
| **L1** | 形式化定义 | $p > 1$，且只有平凡因子 |
| **L2** | 定理证明 | 欧几里得素数无穷定理、算术基本定理 |
| **L3** | 理论建构 | 素数分布、素数定理、筛法理论 |
| **L4** | 前沿研究 | Riemann假设、孪生素数猜想、Goldbach猜想 |

**跃迁路径**: 

```

数的乘法结构 → 素数定义 → 素数定理 → 解析数论 → Riemann假设

```

---

### 3. 同余 (Congruence)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "除以某数后余数相同"，如时钟算术 |
| **L1** | 形式化定义 | $a \equiv b \pmod{m} \Leftrightarrow m \mid (a-b)$ |
| **L2** | 定理证明 | 同余的运算性质、中国剩余定理、Fermat小定理 |
| **L3** | 理论建构 | 剩余类环、模算术、Galois表示 |
| **L4** | 前沿研究 | 高次同余、模形式的同余 |

---

### 4. 算术基本定理

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "每个数都能唯一地分解成素数乘积" |
| **L1** | 形式化定义 | 每个 $n > 1$ 可唯一写成 $n = p_1^{a_1}...p_k^{a_k}$ |
| **L2** | 定理证明 | 存在性证明（归纳法）、唯一性证明（Euclid引理） |
| **L3** | 理论建构 | UFD理论、Dedekind整环的因子分解 |
| **L4** | 前沿研究 | 非交换代数中的因子分解 |

---

### 5. 原根 (Primitive Root)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "生成整个乘法群的元素"，如模5的原根是2或3 |
| **L1** | 形式化定义 | $g$ 是模 $p$ 的原根 $\Leftrightarrow \text{ord}_p(g) = p-1$ |
| **L2** | 定理证明 | 原根存在定理、原根的个数、原根的判定 |
| **L3** | 理论建构 | 分圆域、Galois群、离散对数 |
| **L4** | 前沿研究 | Artin原根猜想、密码学中的应用 |

---

### 6. 二次剩余 (Quadratic Residue)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "模意义下的完全平方数"，如1, 4是模5的二次剩余 |
| **L1** | 形式化定义 | $a$ 是模 $p$ 的二次剩余 $\Leftrightarrow \exists x: x^2 \equiv a \pmod{p}$ |
| **L2** | 定理证明 | Legendre符号、Euler判别法、二次互反律 |
| **L3** | 理论建构 | Hilbert符号、类域论、局部-整体原则 |
| **L4** | 前沿研究 | 高次剩余、Langlands纲领中的对应 |

---

### 7. 互反律 (Reciprocity Law)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "二次剩余的对称规律"，高斯的黄金定理 |
| **L1** | 形式化定义 | $\left(\frac{p}{q}\right)\left(\frac{q}{p}\right) = (-1)^{\frac{p-1}{2}\frac{q-1}{2}}$ |
| **L2** | 定理证明 | Gauss的多个证明、高斯和、代数证明 |
| **L3** | 理论建构 | Artin互反律、类域论的核心 |
| **L4** | 前沿研究 | Langlands互反律、非交换互反律 |

## 二、代数数论概念

### 8. 代数数 (Algebraic Number)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "满足多项式方程的数"，如 $\sqrt{2}$ 满足 $x^2 - 2 = 0$ |
| **L1** | 形式化定义 | 存在 $0 \neq f \in \mathbb{Q}[x]$ 使得 $f(\alpha) = 0$ |
| **L2** | 定理证明 | 代数数的运算封闭性、代数数的次数 |
| **L3** | 理论建构 | 代数闭包、代数数域、Galois理论 |
| **L4** | 前沿研究 | 超越数论、Schanuel猜想 |

---

### 9. 代数整数 (Algebraic Integer)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "首一多项式的根"，整数的推广 |
| **L1** | 形式化定义 | 存在首一 $f \in \mathbb{Z}[x]$ 使得 $f(\alpha) = 0$ |
| **L2** | 定理证明 | 代数整数构成环、代数整数的判别 |
| **L3** | 理论建构 | 数域的整数环、Dedekind整环 |
| **L4** | 前沿研究 | 高度理论、算术几何中的点 |

---

### 10. 数域 (Number Field)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "在有理数上添加代数数得到的域"，如 $\mathbb{Q}(\sqrt{2})$ |
| **L1** | 形式化定义 | $\mathbb{Q}$ 的有限扩张 $K$，$[K:\mathbb{Q}] < \infty$ |
| **L2** | 定理证明 | 本原元定理、嵌入的数量、迹与范数 |
| **L3** | 理论建构 | Galois群、分歧理论、类域论 |
| **L4** | 前沿研究 | Langlands纲领、Iwasawa理论 |

**跃迁路径**: 

```

代数数的直觉 → 数域定义 → Galois理论 → 类域论 → Langlands纲领

```

### 11. 整数环 (Ring of Integers)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "数域中的'整数'"，如 $\mathbb{Z}[\sqrt{2}]$ |
| **L1** | 形式化定义 | $\mathcal{O}_K = \{\alpha \in K : \alpha$ 是代数整数$\}$ |
| **L2** | 定理证明 | $\mathcal{O}_K$ 是Dedekind整环、基的判别式 |
| **L3** | 理论建构 | 理想的唯一分解、分式理想、类群 |
| **L4** | 前沿研究 | Euler系统、Iwasawa理论 |

### 12. 理想类群 (Ideal Class Group)

| 层次 | 描述 | 关键内容 |

| **L4** | 前沿研究 | BSD猜想、类数问题、Euler系统 |

## 三、解析数论概念

### 14. Zeta函数

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "将素数信息编码的函数"，如 $\sum \frac{1}{n^s}$ |
| **L1** | 形式化定义 | $\zeta(s) = \sum_{n=1}^{\infty} \frac{1}{n^s} = \prod_p (1-p^{-s})^{-1}$ |
| **L2** | 定理证明 | 解析延拓、函数方程、Euler乘积 |
| **L3** | 理论建构 | Dedekind zeta函数、L函数、motive的L函数 |
| **L4** | 前沿研究 | Riemann假设、广义Riemann假设、BSD猜想 |

---

### 15. Riemann假设

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "zeta函数零点都在一条线上"，最重要的数学猜想之一 |
| **L1** | 形式化定义 | $\zeta(s) = 0$ 且 $0 < \text{Re}(s) < 1$ 则 $\text{Re}(s) = \frac{1}{2}$ |
| **L2** | 定理证明 | 计算验证、部分结果（Hardy定理、Conrey定理） |
| **L3** | 理论建构 | 谱解释、随机矩阵理论、Hilbert-Pólya猜想 |
| **L4** | 前沿研究 | 各种推广、 motivic L函数的零点 |

**跃迁路径**: 

```

素数分布直觉 → zeta函数定义 → 解析延拓理论 → 谱理论 → Riemann假设

```

### 16. Dirichlet定理

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "等差数列中有无穷多素数"，如4k+1型素数无穷多 |
| **L1** | 形式化定义 | $(a,m) = 1$ 则 $a, a+m, a+2m, ...$ 中有无穷多素数 |
| **L2** | 定理证明 | Dirichlet L函数、非零性证明 |
| **L3** | 理论建构 | Dirichlet特征标、Chebotarev密度定理 |
| **L4** | 前沿研究 | 算术级数中素数间的有界间隔 |

## 四、算术几何概念

### 17. 椭圆曲线 (Elliptic Curve)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "亏格为1的代数曲线"，如 $y^2 = x^3 + ax + b$ |
| **L1** | 形式化定义 | 光滑的射影曲线，亏格为1，带一个有理点 |
| **L2** | 定理证明 | 群结构、Mordell-Weil定理、挠点 |
| **L3** | 理论建构 | 复乘理论、Galois表示、模形式对应 |
| **L4** | 前沿研究 | BSD猜想、费马大定理证明、Gross-Zagier公式 |

---

### 18. 模形式 (Modular Form)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "满足特定变换性质的复函数"，高度对称 |
| **L1** | 形式化定义 | 上半平面上的全纯函数，满足 $f(\frac{az+b}{cz+d}) = (cz+d)^k f(z)$ |
| **L2** | 定理证明 | Fourier展开、Hecke算子、特征形式 |
| **L3** | 理论建构 | 模曲线、Galois表示、Langlands对应 |
| **L4** | 前沿研究 | Serre猜想、Buzzard-Gee猜想、p进模形式 |

---

### 19. BSD猜想 (Birch-Swinnerton-Dyer Conjecture)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "椭圆曲线的有理点与它的zeta函数有关" |
| **L1** | 形式化定义 | $\text{ord}_{s=1} L(E,s) = \text{rank}(E(\mathbb{Q}))$ |
| **L2** | 定理证明 | Coates-Wiles定理、Gross-Zagier定理、Kolyvagin定理 |
| **L3** | 理论建构 | Euler系统、Iwasawa理论、p进BSD |
| **L4** | 前沿研究 | 千禧年大奖难题、BSD的精确形式 |
