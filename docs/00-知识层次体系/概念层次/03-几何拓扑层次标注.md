## 一、点集拓扑概念

### 1. 拓扑空间 (Topological Space)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "邻近关系"、"点的邻近性"、连续性的抽象 |
| **L1** | 形式化定义 | $(X, \tau)$，$\tau \subseteq \mathcal{P}(X)$ 满足：$\emptyset, X \in \tau$；任意并封闭；有限交封闭 |
| **L2** | 定理证明 | 开集性质、闭集性质、邻域基的性质 |
| **L3** | 理论建构 | 位象(Locale)、Frame理论、Grothendieck拓扑 |
| **L4** | 前沿研究 | $\infty$-Topos、高阶拓扑斯 |

**跃迁路径**: 

```

邻近直觉 → 拓扑公理 → 拓扑性质定理 → 位象理论 → ∞-Topos

```

---

### 2. 开集与闭集

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "不包含边界的区域"（开集），"包含边界的区域"（闭集） |
| **L1** | 形式化定义 | 开集：$\tau$ 的元素；闭集：开集的补集 |
| **L2** | 定理证明 | 闭集的性质、闭包运算、内部运算 |
| **L3** | 理论建构 | Kuratowski闭包公理、开集格 |
| **L4** | 前沿研究 | 构造性拓扑、直觉主义拓扑 |

---

### 3. 连续映射 (Continuous Map)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "不撕裂、不跳跃的映射"，直观连续 |
| **L1** | 形式化定义 | $f: X \to Y$，开集的原像是开集 |
| **L2** | 定理证明 | 连续性的等价刻画、连续映射的复合 |
| **L3** | 理论建构 | Top范畴、连续函子、同伦连续 |
| **L4** | 前沿研究 | 可计算连续性、构造性连续性 |

---

### 4. 同胚 (Homeomorphism)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "可连续变形到另一个"，如咖啡杯和甜甜圈 |
| **L1** | 形式化定义 | 双射连续映射，且逆映射也连续 |
| **L2** | 定理证明 | 同胚是等价关系、同胚不变量 |
| **L3** | 理论建构 | 拓扑分类问题、模空间 |
| **L4** | 前沿研究 | 双曲几何中的拟共形同胚 |

---

### 5. 连通性 (Connectedness)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "连成一片"，不能分成不相交的两部分 |
| **L1** | 形式化定义 | 不能表示为两个非空不交开集的并 |
| **L2** | 定理证明 | 连通性的保持、连通分支分解 |
| **L3** | 理论建构 | 道路连通、局部连通、连通范畴 |
| **L4** | 前沿研究 | 高阶连通性、形状理论 |

---

### 6. 紧致性 (Compactness)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "有限范围"、"没有无限延伸的部分" |
| **L1** | 形式化定义 | 任意开覆盖有有限子覆盖 |
| **L2** | 定理证明 | Heine-Borel定理、紧集的闭子集紧、紧集的像紧 |
| **L3** | 理论建构 | 局部紧、紧Hausdorff空间、一点紧化 |
| **L4** | 前沿研究 | 非标准分析中的紧性、粗几何中的紧性 |

**跃迁路径**: 

```

有限直觉 → 开覆盖定义 → Heine-Borel定理 → 局部紧理论 → 现代几何

```

---

### 7. 分离公理 (Separation Axioms)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "能用开集把点分开"的程度 |
| **L1** | 形式化定义 | $T_0, T_1, T_2$ (Hausdorff), $T_3$, $T_4$ 等 |
| **L2** | 定理证明 | 分离公理的蕴含关系、Urysohn引理 |
| **L3** | 理论建构 | 可度量化定理、拓扑完备化 |
| **L4** | 前沿研究 | 非Hausdorff空间、代数几何中的Zariski拓扑 |

## 二、代数拓扑概念

### 8. 同伦 (Homotopy)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "连续变形"，一个映射可以连续地变成另一个 |
| **L1** | 形式化定义 | $H: X \times [0,1] \to Y$，$H(x,0) = f(x)$，$H(x,1) = g(x)$ |
| **L2** | 定理证明 | 同伦是等价关系、同伦的复合、同伦逆 |
| **L3** | 理论建构 | 同伦范畴、同伦型、形变收缩 |
| **L4** | 前沿研究 | 同伦类型论、高阶同伦、谱序列 |

---

### 9. 基本群 (Fundamental Group)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "空间中的洞的数量"，回路不可收缩的程度 |
| **L1** | 形式化定义 | $\pi_1(X, x_0)$：基于$x_0$的回路同伦类群 |
| **L2** | 定理证明 | 道路提升性质、覆盖空间的分类、van Kampen定理 |
| **L3** | 理论建构 | 高阶同伦群、纤维丛的同伦正合列 |
| **L4** | 前沿研究 | 基本群的表示、pro-有限完备化 |

**典型例子在各层次的表现**:
- L0: 甜甜圈有一个洞，球没有洞
- L1: $\pi_1(S^1) = \mathbb{Z}$，$\pi_1(S^n) = 0$ ($n \geq 2$)
- L2: 万有覆盖、提升定理
- L3: 高阶同伦群、谱序列
- L4: 代数几何中的平展上同调

---

### 10. 同调 (Homology)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "洞的高维推广"，用于计数高维洞 |
| **L1** | 形式化定义 | 链复形、边缘映射、同调群 $H_n = \ker\partial_n / \text{im}\partial_{n+1}$ |
| **L2** | 定理证明 | 同调的长正合序列、切除定理、Mayer-Vietoris序列 |
| **L3** | 理论建构 | 公理化同调、广义同调论、谱 |
| **L4** 前沿研究 | 导出范畴、 motivic 同调 |

---

### 11. 上同调 (Cohomology)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | 同调的对偶概念，用于分类空间上的"函数" |
| **L1** | 形式化定义 | 上链复形、上边缘、上同调群 |
| **L2** | 定理证明 | 万有系数定理、Künneth公式、杯积 |
| **L3** | 理论建构 | de Rham上同调、层的上同调、导出函子 |
| **L4** | 前沿研究 | 等变上同调、量子上同调 |

---

### 12. 纤维丛 (Fiber Bundle)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "在每点粘上相同的空间"，如切丛 |
| **L1** | 形式化定义 | $(E, B, F, p)$，$p: E \to B$，局部平凡 |
| **L2** | 定理证明 | 覆盖同伦性质、纤维序列的正合性 |
| **L3** | 理论建构 | 示性类、K理论、主丛 |
| **L4** | 前沿研究 | 弦论中的丛、导出代数拓扑 |

## 三、微分几何概念

### 13. 流形 (Manifold)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "局部像欧氏空间的曲面"，如地球表面局部像平面 |
| **L1** | 形式化定义 | 局部同胚于 $\mathbb{R}^n$ 的Hausdorff空间，有可数基 |
| **L2** | 定理证明 | 坐标卡、图册、流形的维数 |
| **L3** | 理论建构 | 切丛、余切丛、张量丛 |
| **L4** | 前沿研究 | 无穷维流形、分次流形、导出流形 |

**跃迁路径**: 

```

曲面直觉 → 局部坐标卡 → 流形结构定理 → 向量丛理论 → 现代几何

```

---

### 14. 切空间 (Tangent Space)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "在某点的所有可能方向"，如曲面上的切平面 |
| **L1** | 形式化定义 | $T_pM$：过$p$的曲线的等价类，或导子的空间 |
| **L2** | 定理证明 | 切空间的维数、切映射、切丛的可微结构 |
| **L3** | 理论建构 | 向量场、张量场、李导数 |
| **L4** | 前沿研究 | 广义切空间、super流形上的切空间 |

---

### 15. 微分形式 (Differential Form)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "可以积分的对象"，$dx$、$dy$等的反对称组合 |
| **L1** | 形式化定义 | 反对称协变张量场，$\omega \in \Omega^k(M)$ |
| **L2** | 定理证明 | 外微分、Cartan魔术公式、de Rham定理 |
| **L3** | 理论建构 | de Rham上同调、积分理论、Stokes定理 |
| **L4** | 前沿研究 | 广义复几何中的形式、弦论中的形式 |

---

### 16. Riemann度量 (Riemannian Metric)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "度量长度和角度的结构"，如曲面上的第一基本形式 |
| **L1** | 形式化定义 | 光滑的对称正定2阶协变张量场 $g$ |
| **L2** | 定理证明 | 度量的存在性、诱导距离、体积形式 |
| **L3** | 理论建构 | Levi-Civita联络、曲率张量、测地线 |
| **L4** | 前沿研究 | Ricci流、几何化猜想、Perelman证明 |

---

### 17. 曲率 (Curvature)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "空间的弯曲程度"，如球面正曲率、鞍面负曲率 |
| **L1** | 形式化定义 | 曲率张量 $R(X,Y)Z$，或截面曲率、Ricci曲率、数量曲率 |
| **L2** | 定理证明 | Gauss方程、Codazzi方程、Gauss-Bonnet定理 |
| **L3** | 理论建构 | 示性类、指标定理、比较几何 |
| **L4** | 前沿研究 | Ricci曲率下界、坍塌理论、正曲率流形分类 |

---

### 18. Gauss-Bonnet定理

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "曲面的总曲率与拓扑有关" |
| **L1** | 形式化定义 | $\int_M K dA = 2\pi \chi(M)$ |
| **L2** | 定理证明 | 局部Gauss-Bonnet、整体证明、多边形分解 |
| **L3** | 理论建构 | 指标定理、示性数、Atiyah-Singer指标定理 |
| **L4** | 前沿研究 | 高维推广、算术Gauss-Bonnet |

## 四、代数几何概念

### 19. 代数簇 (Algebraic Variety)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "多项式方程的零点集"，如圆锥曲线 |
| **L1** | 形式化定义 | 仿射簇：$\mathbb{A}^n$ 中多项式理想的零点集 |
| **L2** | 定理证明 | Hilbert零点定理、不可约分解、维数理论 |
| **L3** | 理论建构 | 概形、层、上同调 |
| **L4** | 前沿研究 | 导出代数几何、镜像对称 |

---

### 20. 概形 (Scheme)

| 层次 | 描述 | 关键内容 |
|------|------|----------|
| **L0** | 直观理解 | "代数簇的推广，带有'函数环'信息的几何对象" |
| **L1** | 形式化定义 | 局部环化空间 $(X, \mathcal{O}_X)$，局部同构于仿射概形 |
| **L2** | 定理证明 | 概形的纤维积、分离性、properness |
| **L3** | 理论建构 | 层上同调、平坦性、光滑性 |
| **L4** | 前沿研究 | 导出概形、稳定配对的不变式、完美oid空间 |

**跃迁路径**: 

```

方程的解 → 代数簇 → 概形定义 → 层上同调 → 现代代数几何前沿

```
