# L0-05: 十进制系统

**概念编号**: L0-05  
**所属家族**: 数与计数  
**核心定位**: 位置记数法的核心实例

---

## 📌 直观描述

### 1.1 核心直观

十进制系统是**"满十进一"**的记数规则——每数到十，就向前一位进一。

> **日常表述**: "十个一等于一个十"、"十个十等于一百"

### 1.2 位值原理

```

十进制的位值结构:

   3   2   7   5
   │   │   │   │
   │   │   │   └─ 个位: 5 × 1 = 5
   │   │   └───── 十位: 7 × 10 = 70
   │   └───────── 百位: 2 × 100 = 200
   └───────────── 千位: 3 × 1000 = 3000
   
总和: 5 + 70 + 200 + 3000 = 3275

```

### 1.3 位值表

| 位名 | 位权 | 示例 |
|-----|------|------|
| 个位 | 10⁰ = 1 | 5 = 5×1 |
| 十位 | 10¹ = 10 | 50 = 5×10 |
| 百位 | 10² = 100 | 500 = 5×100 |
| 千位 | 10³ = 1000 | 5000 = 5×1000 |

## 🌍 物理世界对应

### 2.1 为什么用十进制？

| 原因 | 解释 |
|-----|------|
| **手指计数** | 人类有10根手指 |
| **适中大小** | 进制太小数字太长，太大记忆困难 |
| **历史传承** | 印度-阿拉伯数字系统的传播 |

### 2.2 其他进制系统

| 进制 | 应用 | 特点 |
|-----|------|------|
| 二进制 | 计算机 | 只有0和1 |
| 十二进制 | 时间、打 | 因数多（2,3,4,6） |
| 六十进制 | 角度、时间 | 古巴比伦遗产 |
| 二十进制 | 玛雅文明 | 手指加脚趾 |
