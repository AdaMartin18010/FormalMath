# L0-11: 点的概念

**概念编号**: L0-11  
**所属家族**: 空间与几何  
**核心定位**: 几何的基本元素

---

## 📌 直观描述

### 1.1 核心直观

点是**位置的标记**——没有大小，只有位置。

> **日常表述**: "铅笔尖在纸上一戳"、"地图上的标记"、"定位一个位置"

### 1.2 多重表征

| 表征类型 | 具体表现 | 示例 |
|---------|---------|------|
| **笔尖表征** | 最小的标记 | 铅笔尖触纸 |
| **坐标表征** | 数对位置 | (3, 4)在坐标系中 |
| **位置表征** | 空间中的点 | "从这里出发" |
| **抽象表征** | 理想化的点 | 没有维度 |

### 1.3 点的悖论

```

点的直观矛盾:

现实: 任何标记都有大小
       ● ← 实际画出的点有面积
       
几何: 点是零维的
       · ← 理想化的点没有大小
       
解决: 点是位置的抽象

```

### 2.2 历史发展

```

点的概念发展:

欧几里得《几何原本》(约300BC):
"点是没有部分的东西"

现代观点:
- 仿射空间中的元素
- 拓扑空间中的点
- 流形上的点

```
