# L0-35: 是/否判断

**概念编号**: L0-35  
**所属家族**: 逻辑与关系  
**核心定位**: 二值判断基础

---

## 📌 直观描述

是/否判断是**对问题的肯定或否定回答**——对或错，真或假。

> **日常表述**: "是"、"否"、"对"、"错"、"真"、"假"

### 二值逻辑

| 判断 | 示例 |
|-----|------|
| 是 | "这是苹果" |
| 否 | "这不是香蕉" |
