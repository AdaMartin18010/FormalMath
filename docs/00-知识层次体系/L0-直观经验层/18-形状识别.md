# L0-18: 形状识别

**概念编号**: L0-18  
**所属家族**: 空间与几何  
**核心定位**: 图形分类的基础

---

## 📌 直观描述

### 1.1 核心直观

形状是**物体的外形轮廓**——通过边界特征来识别和分类。

> **日常表述**: "这是三角形"、"那个是正方形"、"圆形的盘子"

### 1.2 基本形状

| 形状 | 直观特征 | 示例 |
|-----|---------|------|
| **三角形** | 三条边、三个角 | 三明治、警告标志 |
| **正方形** | 四边相等、四角为直角 | 瓷砖、窗户 |
| **长方形** | 对边相等、四角为直角 | 门、书本 |
| **圆形** | 没有角、等距边界 | 硬币、盘子 |
| **椭圆形** | 压扁的圆 | 鸡蛋、轨道 |

### 1.3 形状识别的发展

```

儿童形状认知发展:

3岁: 识别圆形（最简单）
  ↓
4岁: 识别正方形、三角形
  ↓
5岁: 识别长方形、椭圆形
  ↓
6岁: 区分菱形、平行四边形
  ↓
7岁+: 理解形状性质（边数、角数）

```
