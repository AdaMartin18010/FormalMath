# L0-46: 可能性大小

**概念编号**: L0-46  
**所属家族**: 随机与可能  
**核心定位**: 概率直观

---

## 📌 直观描述

可能性大小是**事情发生的机会**——可能、很可能、不可能。

> **日常表述**: "有可能"、"大概会"、"肯定能"、"不可能"

### 可能性等级

| 等级 | 描述 | 示例 |
|-----|------|------|
| 必然 | 一定会发生 | 太阳东升 |
| 很可能 | 发生机会大 | 明天下雨（雨季） |
| 可能 | 不确定 | 掷骰子得6 |
| 不太可能 | 发生机会小 | 中彩票 |
| 不可能 | 不会发生 | 掷骰子得7 |
