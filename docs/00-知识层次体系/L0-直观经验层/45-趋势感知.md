# L0-45: 趋势感知

**概念编号**: L0-45  
**所属家族**: 变化与模式  
**核心定位**: 极限直观

---

## 📌 直观描述

趋势感知是**事物发展走向**的判断——向哪里发展、趋向何处。

> **日常表述**: "趋向于..."、"走向..."、"接近..."、"收敛到..."

### 趋势类型

| 趋势 | 描述 | 示例 |
|-----|------|------|
| 收敛 | 越来越接近某值 | 0.9,0.99,0.999→1 |
| 发散 | 越来越大 | 1,2,3,4,...→∞ |
| 振荡 | 来回波动 | 摆锤 |
