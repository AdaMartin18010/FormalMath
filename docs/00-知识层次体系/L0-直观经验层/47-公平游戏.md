# L0-47: 公平游戏

**概念编号**: L0-47  
**所属家族**: 随机与可能  
**核心定位**: 等可能直观  
**学习难度**: ⭐⭐ (入门级)

---

## 📌 直观描述

公平游戏是**机会均等**——每个人赢的机会一样。

> **日常表述**: "公平"、"机会一样"、"平等"、"不偏不倚"

### 公平性判断

| 游戏 | 公平性 | 分析 |
|-----|--------|------|
| **掷公平硬币** | 公平 | P(正面)=P(反面)=0.5 |
| **掷公平骰子** | 公平 | 6个面各1/6概率 |
| **猜拳** | 公平 | 石头、剪刀、布各1/3概率 |
| **彩票** | 不公平 | 庄家优势，期望收益为负 |
| **轮盘赌** | 不公平 | 有0和00，赌场优势 |

---

## 🌰 具体示例

### 示例1：掷硬币赌局
A和B玩掷硬币游戏：
- 正面：A赢1元
- 反面：B赢1元

**分析**:
- P(A赢) = P(B赢) = 0.5
- A的期望收益 = 1×0.5 + (-1)×0.5 = 0
- **结论**：公平游戏

### 示例2：不公平的骰子
某骰子被做了手脚：
- P(6) = 0.5
- P(1)=P(2)=P(3)=P(4)=P(5) = 0.1

若押6点赢5元，否则输1元：
- 期望收益 = 5×0.5 + (-1)×0.5 = 2元（对庄家有利）
- **结论**：不公平游戏

### 示例3：俄罗斯轮盘赌（危险游戏！）
6个弹仓，1发子弹：
- 每人轮流扣扳机
- 第一个人死亡概率 = 1/6
- 第二个人死亡概率 = 5/6 × 1/5 = 1/6

**分析**: 两人死亡概率相等，从概率角度是"公平"的，但极其危险！

### 示例4：彩票期望
购买1张2元彩票，头奖100万元，中奖概率1/500万：
- 期望收益 = 1000000×(1/5000000) + 0×(4999999/5000000) - 2
- = 0.2 - 2 = -1.8元

**结论**：期望损失1.8元，对玩家不公平。

## 📊 公平游戏的数学定义

### 期望收益准则

一个游戏对玩家**公平**当且仅当：
$$E[X] = 0$$

其中 \(X\) 为玩家的收益随机变量。

| 情况 | 数学表达 | 含义 |
|-----|---------|------|
| E[X] = 0 | 公平 | 长期看不输不赢 |
| E[X] > 0 | 对玩家有利 | 长期看玩家赢 |
| E[X] < 0 | 对玩家不利 | 长期看玩家输 |

## 🎲 公平性设计

### 如何设计公平游戏

1. **对称性**: 所有玩家规则相同
2. **等概率**: 所有结果概率相等
3. **零期望**: 收益与损失平衡

### 实例：切蛋糕问题
两人分蛋糕，如何公平？
- **方案1**: 一人切，另一人先选（激励切公平）
- **方案2**: 拍卖或博弈论方法

## ❓ 常见问题

**Q: 公平游戏意味着每次结果相等吗？**  
A: 不是。公平是指长期看期望收益为零，单次结果仍随机。

**Q: 为什么赌场能赚钱？**  
A: 赌场设计游戏规则使其期望收益为正（玩家期望为负），长期必然盈利。

**Q: 如何判断一个游戏是否公平？**  
A: 计算所有结果的期望收益，若为0则公平。
