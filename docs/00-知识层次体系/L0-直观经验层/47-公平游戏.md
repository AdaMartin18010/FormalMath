# L0-47: 公平游戏

**概念编号**: L0-47  
**所属家族**: 随机与可能  
**核心定位**: 等可能直观

---

## 📌 直观描述

公平游戏是**机会均等**——每个人赢的机会一样。

> **日常表述**: "公平"、"机会一样"、"平等"、"不偏不倚"

### 公平性判断

| 游戏 | 公平性 |
|-----|--------|
| 掷公平硬币 | 公平 |
| 掷公平骰子 | 公平 |
| 猜拳 | 公平 |
| 彩票 | 不公平（庄家优势） |
