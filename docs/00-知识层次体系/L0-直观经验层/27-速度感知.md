# L0-27: 速度感知

**概念编号**: L0-27  
**所属家族**: 量与测量  
**核心定位**: 运动的快慢

---

## 📌 直观描述

### 1.1 核心直观

速度是**运动的快慢**——单位时间内走的距离。

> **日常表述**: "开得太快了"、"时速100公里"、"比兔子跑得慢"

### 1.2 速度感知

| 类型 | 描述 | 示例 |
|-----|------|------|
| **平均速度** | 总路程÷总时间 | 全程平均 |
| **瞬时速度** | 某一时刻的速度 | 车速表读数 |
| **相对速度** | 相对于观察者的速度 | 超车时的相对速度 |
