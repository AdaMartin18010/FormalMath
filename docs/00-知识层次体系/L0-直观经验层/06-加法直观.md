# L0-06: 加法直观

**概念编号**: L0-06  
**所属家族**: 数与计数  
**核心定位**: 最基本的算术运算

---

## 📌 直观描述

### 1.1 核心直观

加法是**把东西合在一起**的操作——两堆东西合并成一堆。

> **日常表述**: "3个苹果加2个苹果等于5个苹果"、"合起来有多少"

### 1.2 多重表征

| 表征类型 | 具体表现 | 示例 |
|---------|---------|------|
| **合并表征** | 两堆东西合为一堆 | ●●● + ●● = ●●●●● |
| **数轴表征** | 向右移动 | 从3开始，向右移动2格到5 |
| **计数表征** | 接着数 | 3, 4, 5（再数2个） |
| **集合表征** | 并集 | |A∪B| = |A| + |B|（不相交时） |

### 1.3 加法直观特征

```

加法的直观性质:
├─ 交换性: 3+2 = 2+3（合并顺序不影响结果）
├─ 结合性: (1+2)+3 = 1+(2+3)（分组方式不影响）
├─ 单位元: a+0 = a（加零不变）
└─ 单调性: 若a<b，则a+c<b+c

```
