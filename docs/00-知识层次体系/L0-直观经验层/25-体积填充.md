# L0-25: 体积填充

**概念编号**: L0-25  
**所属家族**: 量与测量  
**核心定位**: 三维量的度量

---

## 📌 直观描述

### 1.1 核心直观

体积是**空间占据的大小**——用单位立方体填充，看需要多少。

> **日常表述**: "这个盒子能装多少"、"水箱容量多大"

### 1.2 测量方法

| 方法 | 描述 | 示例 |
|-----|------|------|
| **填充法** | 用小立方体填充 | 数立方体个数 |
| **排水法** | 排水体积 | 阿基米德原理 |
| **公式法** | 用公式计算 | 长×宽×高 |
