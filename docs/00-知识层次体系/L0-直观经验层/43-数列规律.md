# L0-43: 数列规律

**概念编号**: L0-43  
**所属家族**: 变化与模式  
**核心定位**: 序列推理  
**学习难度**: ⭐⭐ (入门级)

---

## 📌 直观描述

数列规律是**数字按某种规则排列**——找规律、填空。

> **日常表述**: "下一个数是什么"、"规律是什么"、"按顺序"

### 常见数列类型

| 数列 | 规律描述 | 通项公式 | 下一项 |
|-----|---------|---------|--------|
| 2,4,6,8,... | 每次+2（偶数）| aₙ=2n | 10 |
| 1,3,5,7,... | 每次+2（奇数）| aₙ=2n-1 | 9 |
| 1,1,2,3,5,... | 前两数之和 | Fₙ=Fₙ₋₁+Fₙ₋₂ | 8 |
| 2,4,8,16,... | 每次×2（等比）| aₙ=2ⁿ | 32 |
| 1,4,9,16,... | 完全平方数 | aₙ=n² | 25 |

---

## 🌰 具体示例

### 示例1：找规律填数
**序列**: 3, 6, 12, 24, ?

**分析**: 6÷3=2, 12÷6=2, 24÷12=2 → 每次×2  
**答案**: 24×2 = **48**

### 示例2：斐波那契数列
**序列**: 1, 1, 2, 3, 5, 8, 13, 21, ?

**规律**: 从第三项开始，每项等于前两项之和  
Fₙ = Fₙ₋₁ + Fₙ₋₂

**应用**: 
- 兔子繁殖问题
- 植物叶序排列
- 黄金分割近似

### 示例3：交错数列
**序列**: 1, -2, 3, -4, 5, ?

**规律**: 绝对值递增1，符号正负交替  
**答案**: **-6**

## 📊 数列分类

### 按规律类型分类

| 类型 | 特征 | 示例 |
|------|------|------|
| **等差数列** | 相邻项差相等 | 3, 7, 11, 15... (公差d=4) |
| **等比数列** | 相邻项比相等 | 2, 6, 18, 54... (公比r=3) |
| **周期数列** | 重复出现 | 1, 2, 3, 1, 2, 3... |
| **递推数列** | 由前项定义 | 斐波那契数列 |

### 通项与递推
- **通项公式**: aₙ = f(n)，直接计算第n项
- **递推公式**: aₙ = f(aₙ₋₁, aₙ₋₂, ...)，由前面项计算

## 🧮 求和公式

| 数列类型 | 前n项和公式 |
|---------|-----------|
| 等差数列 | Sₙ = n(a₁+aₙ)/2 |
| 等比数列 | Sₙ = a₁(1-rⁿ)/(1-r) |
| 自然数和 | 1+2+...+n = n(n+1)/2 |

**高斯故事**: 1+2+3+...+100 = 100×101/2 = 5050

## ❓ 常见问题

**Q: 如何快速判断是等差还是等比？**  
A: 计算相邻数的差，若相等是等差；计算相邻数的比，若相等是等比。

**Q: 斐波那契数列有通项公式吗？**  
A: 有，称为比内公式：Fₙ = (φⁿ - ψⁿ)/√5，其中φ=(1+√5)/2，ψ=(1-√5)/2
