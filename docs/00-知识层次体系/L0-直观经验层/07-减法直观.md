# L0-07: 减法直观

**概念编号**: L0-07  
**所属家族**: 数与计数  
**核心定位**: 加法的逆运算

---

## 📌 直观描述

### 1.1 核心直观

减法是**拿走**或**比较**的操作——从一堆中拿走一些，或者比较两堆的差异。

> **日常表述**: "从5个里拿走2个"、"3比7少多少"

### 1.2 多重表征

| 表征类型 | 具体表现 | 示例 |
|---------|---------|------|
| **拿走表征** | 去掉一部分 | ●●●●● - ●● = ●●● |
| **比较表征** | 求差值 | 7比5多2 |
| **数轴表征** | 向左移动 | 从5开始，向左移2格到3 |
| **逆运算表征** | 加法的逆 | 5-2=3 因为 3+2=5 |

### 1.3 两种减法类型

```

减法的情境意义:

类型1: 拿走型 (Take-away)
   "我有5颗糖，吃了2颗，还剩几颗？"
   5 - 2 = 3
   
类型2: 比较型 (Comparison)
   "小明有5本书，小红有2本，小明比小红多几本？"
   5 - 2 = 3
   
类型3: 缺失加数型 (Missing addend)
   "我有2元，想买5元的玩具，还差几元？"
   2 + ? = 5  →  5 - 2 = 3

```

### 2.2 减法的认知难点

```

儿童学习减法的困难:

困难1: 结果变小
   "为什么越减越少？"
   
困难2: 零的理解
   "5-5等于多少？"
   
困难3: 负数引入
   "3-5等于多少？"
   → 需要扩展到整数范围

```

| 直观问题 | 形式化挑战 | L1回答 |
