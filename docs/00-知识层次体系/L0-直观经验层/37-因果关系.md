# L0-37: 因果关系

**概念编号**: L0-37  
**所属家族**: 逻辑与关系  
**核心定位**: 推理的基础

---

## 📌 直观描述

因果关系是**原因导致结果**的关系——因为...所以...。

> **日常表述**: "因为下雨，所以地湿"、"因此..."、"结果是..."

### 因果类型

| 类型 | 示例 |
|-----|------|
| 直接因果 | 火→热 |
| 间接因果 | 云→雨→湿 |
| 多因一果 | 阳光+水→植物生长 |
