# L0-30: 分类思维

**概念编号**: L0-30  
**所属家族**: 集合与分类  
**核心定位**: 按特征分组

---

## 📌 直观描述

分类是按**共同特征**把事物分组——相似的放在一起。

> **日常表述**: "把红色的放一起"、"按大小分类"、"这些都是水果"

### 分类标准

| 标准 | 示例 |
|-----|------|
| 颜色 | 红、黄、蓝 |
| 形状 | 圆、方、三角 |
| 大小 | 大、中、小 |
| 功能 | 餐具、文具、玩具 |

---

## 🌍 物理世界对应

- 图书馆图书分类
- 商品货架分类
- 生物物种分类
- 垃圾分类
