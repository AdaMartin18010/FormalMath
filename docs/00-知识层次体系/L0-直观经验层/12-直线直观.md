# L0-12: 直线直观

**概念编号**: L0-12  
**所属家族**: 空间与几何  
**核心定位**: 最短的路径

---

## 📌 直观描述

### 1.1 核心直观

直线是**拉紧的绳子**——没有弯曲，向两边无限延伸。

> **日常表述**: "笔直的马路"、"拉紧的线"、"两点之间最短"

### 1.2 多重表征

| 表征类型 | 具体表现 | 示例 |
|---------|---------|------|
| **拉紧表征** | 绷紧的线 | 拉紧的绳子 |
| **光线表征** | 光的路径 | 激光笔的光线 |
| **边缘表征** | 平直的边缘 | 尺子的边 |
| **抽象表征** | 一维无限延伸 | 数轴 |

### 1.3 直线的性质

```

直线的直观特征:
├─ 无限延伸: 向两端无限延长
├─ 平直性: 没有弯曲
├─ 一维性: 只有长度，没有宽度
├─ 连续性: 没有断裂
└─ 最短性: 两点之间直线最短

```

### 2.1 日常实例

| 场景 | 直线体现 |
|-----|---------|
| **建筑** | 笔直的道路、墙角线 |
| **天文** | 光线传播路径 |
| **测量** | 水平线、垂直线 |
| **艺术** | 构图中的直线元素 |
| **运动** | 直线运动轨迹 |

### 2.2 直线公理

```

欧几里得第一公设:
"从任一点到任一点可以作一条直线"

等价表述:
- 过两点有且仅有一条直线
- 两点确定一条直线

```
