# L0-08: 乘法直观

**概念编号**: L0-08  
**所属家族**: 数与计数  
**核心定位**: 重复的加法

---

## 📌 直观描述

### 1.1 核心直观

乘法是**重复的加法**——相同的数多次相加的简便写法。

> **日常表述**: "3个5相加"、"每排6个，有4排"、"翻4倍"

### 1.2 多重表征

| 表征类型 | 具体表现 | 示例 |
|---------|---------|------|
| **重复表征** | 相同数相加 | 3×4 = 4+4+4 |
| **阵列表征** | 矩形排列 | 3行4列的阵列有12个 |
| **面积表征** | 矩形面积 | 3×4的矩形面积是12 |
| **缩放表征** | 比例变化 | 扩大到原来的3倍 |
| **组合表征** | 搭配组合 | 3件上衣×4条裤子=12种搭配 |

### 1.3 乘法直观特征

```

乘法的直观性质:
├─ 交换性: 3×4 = 4×3（阵列旋转）
├─ 结合性: (2×3)×4 = 2×(3×4)
├─ 分配性: 3×(4+5) = 3×4 + 3×5
├─ 单位元: a×1 = a
└─ 零元: a×0 = 0

```
