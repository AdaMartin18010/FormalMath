# L0-28: 比例直观

**概念编号**: L0-28  
**所属家族**: 量与测量  
**核心定位**: 部分与整体的关系

---

## 📌 直观描述

### 1.1 核心直观

比例是**部分与整体的关系**——一个量是另一个量的几分之几。

> **日常表述**: "一半的水"、"按1:3的比例混合"、"放大两倍"

### 1.2 比例表示

| 形式 | 描述 | 示例 |
|-----|------|------|
| **分数** | 部分/整体 | 1/2 |
| **比** | a:b | 1:3 |
| **百分数** | 百分之几 | 50% |
| **倍数** | 几倍 | 2倍 |
