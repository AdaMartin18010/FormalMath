# L0-41: 增长减少

**概念编号**: L0-41  
**所属家族**: 变化与模式  
**核心定位**: 单调性直观

---

## 📌 直观描述

增长减少是**数量变大或变小**的趋势——越来越多或越来越少。

> **日常表述**: "越来越..."、"增加了"、"减少了"、"趋势"

### 变化类型

| 类型 | 描述 | 示例 |
|-----|------|------|
| 增长 | 越来越大 | 身高增长 |
| 减少 | 越来越小 | 气温下降 |
| 不变 | 保持稳定 | 恒温 |
