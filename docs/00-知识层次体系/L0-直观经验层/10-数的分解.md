# L0-10: 数的分解

**概念编号**: L0-10  
**所属家族**: 数与计数  
**核心定位**: 数的结构分析

---

## 📌 直观描述

### 1.1 核心直观

数的分解是把一个数**拆开成几部分**，可以按不同方式拆分。

> **日常表述**: "10可以分成3和7"、"12是3个4"

### 1.2 多重表征

| 表征类型 | 具体表现 | 示例 |
|---------|---------|------|
| **加法分解** | 拆成和 | 10 = 3 + 7 = 4 + 6 |
| **乘法分解** | 因数分解 | 12 = 3 × 4 = 2 × 6 |
| **位值分解** | 按位拆分 | 327 = 300 + 20 + 7 |
| **质因数分解** | 质数乘积 | 12 = 2 × 2 × 3 |

### 1.3 分解的类型

```

数的分解方式:

加法分解:
   5 = 1+4 = 2+3 = 0+5 = ...
   
乘法分解（因数分解）:
   12 = 1×12 = 2×6 = 3×4
   
质因数分解:
   12 = 2² × 3
   
位值分解:
   234 = 2×100 + 3×10 + 4×1

```

## 🌍 物理世界对应

### 2.1 日常实例

| 场景 | 分解应用 |
|-----|---------|
| **换零钱** | 100元=50+50=20×5 |
| **时间安排** | 1小时=30+30分钟 |
| **物品分类** | 总数按类别分组 |
| **因数应用** | 矩形排列的可能性 |

### 2.2 质因数分解的重要性

```

质因数分解的应用:

- 最大公约数 (GCD)
- 最小公倍数 (LCM)  
- 分数化简
- 密码学基础（RSA算法）

```
