# L0-22: 重量感知

**概念编号**: L0-22  
**所属家族**: 量与测量  
**核心定位**: 质量的日常感知

---

## 📌 直观描述

### 1.1 核心直观

重量是物体**被地球吸引的程度**——用手掂量时的轻重感觉。

> **日常表述**: "这个很重"、"用天平称一下"、"减重5公斤"

### 1.2 测量方法

| 方法 | 描述 | 示例 |
|-----|------|------|
| **天平** | 比较平衡 | 天平两边相等 |
| **弹簧秤** | 形变测量 | 弹簧伸长 |
| **电子秤** | 传感器测量 | 数字显示 |
| **手感** | 估测 | 用手掂量 |
