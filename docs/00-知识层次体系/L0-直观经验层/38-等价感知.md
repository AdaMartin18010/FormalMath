# L0-38: 等价感知

**概念编号**: L0-38  
**所属家族**: 逻辑与关系  
**核心定位**: 相等的推广

---

## 📌 直观描述

等价感知是**认为两个事物"一样"**——在某种意义上相等。

> **日常表述**: "这两个一样"、"等价"、"相等价"

### 等价情形

| 情形 | 等价含义 |
|-----|---------|
| 数值 | 数值相等 |
| 形状 | 全等 |
| 分类 | 同类 |
