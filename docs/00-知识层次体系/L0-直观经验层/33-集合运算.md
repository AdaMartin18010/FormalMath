# L0-33: 集合运算

**概念编号**: L0-33  
**所属家族**: 集合与分类  
**核心定位**: 集合的组合操作  
**学习难度**: ⭐⭐ (入门级)

---

## 📌 直观描述

集合运算是**对集合进行操作**——合并、找共同部分、去掉。

> **日常表述**: "放在一起"、"共同的部分"、"去掉重复的"

### 基本运算

| 运算 | 直观描述 | 符号 | 数学定义 |
|-----|---------|------|----------|
| **并** | 合并在一起 | ∪ | A∪B = {x : x∈A 或 x∈B} |
| **交** | 共同部分 | ∩ | A∩B = {x : x∈A 且 x∈B} |
| **差** | 去掉一部分 | \\ | A\\B = {x : x∈A 且 x∉B} |
| **补** | 不在集合中 | ᶜ | Aᶜ = {x : x∉A} |

### Venn图示意

```

└─────┘         └─────┘
```

## 🌰 具体示例

### 示例1：水果集合
设 **A** = {苹果, 香蕉, 橙子}，**B** = {香蕉, 葡萄, 梨}

- **A∪B** = {苹果, 香蕉, 橙子, 葡萄, 梨} （所有水果，不重复）
- **A∩B** = {香蕉} （两种集合都有的水果）
- **A\\B** = {苹果, 橙子} （在A中但不在B中的水果）

### 示例2：数字集合
设 **E** = {2, 4, 6, 8}（偶数），**P** = {2, 3, 5, 7}（素数）

- **E∩P** = {2} （唯一的偶素数）
- **E∪P** = {2, 3, 4, 5, 6, 7, 8}

### 示例3：学生选课
- **数学班** = {张三, 李四, 王五}
- **物理班** = {李四, 赵六, 钱七}

则：
- 同时选两科的：**数学班 ∩ 物理班** = {李四}
- 选数学或物理的：**数学班 ∪ 物理班** = {张三, 李四, 王五, 赵六, 钱七}

## 📝 运算律

集合运算满足以下重要性质：

| 定律 | 公式 |
|------|------|
| **交换律** | A∪B = B∪A, A∩B = B∩A |
| **结合律** | (A∪B)∪C = A∪(B∪C) |
| **分配律** | A∪(B∩C) = (A∪B)∩(A∪C) |
| **德摩根律** | (A∪B)ᶜ = Aᶜ∩Bᶜ |

## ❓ 常见问题

**Q: 空集与任何集合的交集是什么？**  
A: 空集。∅∩A = ∅

**Q: 并集和交集的优先级？**  
A: 通常交集优先，但建议使用括号明确顺序。
