# L0-16: 距离直观

**概念编号**: L0-16  
**所属家族**: 空间与几何  
**核心定位**: 空间中的远近

---

## 📌 直观描述

### 1.1 核心直观

距离是**两点之间的远近程度**——可以用长度来度量。

> **日常表述**: "从这里到那里有多远"、"两地的距离"

### 1.2 多重表征

| 表征类型 | 具体表现 | 示例 |
|---------|---------|------|
| **直线路径** | 直线距离 | 两点之间直线 |
| **实际路径** | 行走距离 | 实际路线长度 |
| **时间距离** | 所需时间 | 走10分钟 |
| **代价距离** | 花费代价 | 车票价格 |

### 1.3 距离的性质

```

距离的直观特征:
├─ 非负性: 距离总是≥0
├─ 同一性: 点与自己的距离为0
├─ 对称性: A到B等于B到A
└─ 三角不等式: 两边之和≥第三边

```

### 2.2 不同空间中的距离

```

距离概念的推广:

欧氏距离: d = √(x²+y²)  （平面直线）
          
曼哈顿距离: d = |x|+|y|  （城市街道）
          
球面距离: 沿大圆测量     （地球表面）

```
