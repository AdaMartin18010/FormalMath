# L0-17: 方向直观

**概念编号**: L0-17  
**所属家族**: 空间与几何  
**核心定位**: 指向的概念

---

## 📌 直观描述

### 1.1 核心直观

方向是**指向某个方位的趋势**——东西南北，上下左右。

> **日常表述**: "向前走"、"向左转"、"往上"

### 1.2 多重表征

| 表征类型 | 具体表现 | 示例 |
|---------|---------|------|
| **罗盘表征** | 方位指向 | 东南西北 |
| **身体表征** | 相对身体 | 前后左右 |
| **坐标表征** | 坐标轴方向 | x,y,z轴方向 |
| **向量表征** | 有方向的量 | 箭头指向 |

### 1.3 基本方向

```

方向的直观体系:

平面方向:
          北(N)
           │
  西(W) ───┼─── 东(E)
           │
          南(S)

空间方向:
  上下、前后、左右（六方向）

```

## 🌍 物理世界对应

### 2.1 日常实例

| 场景 | 方向应用 |
|-----|---------|
| **导航** | 指示前进方向 |
| **建筑** | 房屋朝向 |
| **运动** | 运动方向 |
| **艺术** | 视觉引导方向 |
| **物理** | 力的方向 |

### 2.2 相对方向与绝对方向

```

方向的两类:

相对方向: 相对于观察者
  - 我的左边
  - 你的前面
  
绝对方向: 相对于地球/空间
  - 正北方向
  - 向上的方向

```
