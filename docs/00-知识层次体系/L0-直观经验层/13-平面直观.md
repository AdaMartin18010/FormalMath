# L0-13: 平面直观

**概念编号**: L0-13  
**所属家族**: 空间与几何  
**核心定位**: 二维无限延伸

---

## 📌 直观描述

### 1.1 核心直观

平面是**无限延伸的平整表面**——像桌面一样平，向四周无限展开。

> **日常表述**: "水平的桌面"、"平静的湖面"、"无限延伸的地面"

### 1.2 多重表征

| 表征类型 | 具体表现 | 示例 |
|---------|---------|------|
| **表面表征** | 平整表面 | 桌面、墙面 |
| **水面表征** | 平静水面 | 湖面、镜面 |
| **抽象表征** | 二维空间 | 坐标平面 |
| **展开表征** | 铺开的平面 | 展开的纸张 |

### 1.3 平面的性质

```

平面的直观特征:
├─ 平整性: 没有起伏
├─ 二维性: 有长度和宽度
├─ 无限延伸: 向四周无限扩展
├─ 连续性: 没有洞或断裂
└─ 包含性: 包含无限多条直线

```

## 🌍 物理世界对应

### 2.1 日常实例

| 场景 | 平面体现 |
|-----|---------|
| **建筑** | 地板、天花板 |
| **自然** | 平静湖面 |
| **艺术** | 画布、纸面 |
| **地图** | 地球表面的局部近似 |
| **投影** | 屏幕、墙面 |

### 2.2 三维空间中的平面

```

平面与空间的关系:

空间: 三维 ────────→ 有长度、宽度、高度
              │
平面: 二维 ────────→ 有长度、宽度，无厚度
              │
直线: 一维 ────────→ 只有长度
              │
点:   零维 ────────→ 只有位置

```
