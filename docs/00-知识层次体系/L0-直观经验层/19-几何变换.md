# L0-19: 几何变换

**概念编号**: L0-19  
**所属家族**: 空间与几何  
**核心定位**: 图形运动的变化

---

## 📌 直观描述

### 1.1 核心直观

几何变换是**图形在空间中移动或改变**——平移、旋转、翻转。

> **日常表述**: "把图移过去"、"转个方向"、"翻过来"

### 1.2 基本变换

| 变换 | 直观描述 | 示例 |
|-----|---------|------|
| **平移** | 沿直线移动 | 滑动窗户 |
| **旋转** | 绕点转动 | 钟表指针 |
| **反射** | 镜像翻转 | 照镜子 |
| **缩放** | 大小改变 | 放大缩小 |

### 1.3 变换的性质

```

基本变换的特征:

平移: 形状大小不变，位置改变
      
旋转: 形状大小不变，方向改变
      
反射: 形状大小不变，左右互换（镜像）
      
缩放: 位置不变，大小改变

```

## 🌍 物理世界对应

### 2.1 日常实例

| 变换 | 实际例子 |
|-----|---------|
| 平移 | 电梯上下、抽屉推拉 |
| 旋转 | 门开关、地球自转 |
| 反射 | 镜像、倒影 |
| 缩放 | 地图比例、复印 |
