# L0-15: 角度直观

**概念编号**: L0-15  
**所属家族**: 空间与几何  
**核心定位**: 转动的量度

---

## 📌 直观描述

### 1.1 核心直观

角度是**两条线张开的程度**——张开越大，角度越大。

> **日常表述**: "转弯的大小"、"张开的程度"、"转过多少"

### 1.2 多重表征

| 表征类型 | 具体表现 | 示例 |
|---------|---------|------|
| **开合表征** | 张合程度 | 剪刀的开合 |
| **旋转表征** | 转过的量 | 钟表指针转动 |
| **坡度表征** | 倾斜程度 | 斜坡的陡缓 |
| **方向表征** | 方向变化 | 转向的角度 |

### 1.3 常见角度

```

角度的直观认识:

0°:   ────→  没有张开
      
90°:  │      直角（垂直）
      └──→
      
180°: ←────→ 平角（直线）
      
360°: 完整旋转一周

```

| 单位 | 定义 | 应用 |
