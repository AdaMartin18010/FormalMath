# L0-50: 平均概念

**概念编号**: L0-50  
**所属家族**: 随机与可能  
**核心定位**: 集中趋势

---

## 📌 直观描述

平均是**数据的中间水平**——代表一般情况。

> **日常表述**: "平均..."、"一般水平"、"大多数"、"中等"

### 平均类型

| 类型 | 描述 | 计算 |
|-----|------|------|
| 算术平均 | 总和÷个数 | (a+b)/2 |
| 中位数 | 中间值 | 排序后取中间 |
| 众数 | 出现最多 | 频率最高 |
