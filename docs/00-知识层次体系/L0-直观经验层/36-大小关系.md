# L0-36: 大小关系

**概念编号**: L0-36  
**所属家族**: 逻辑与关系  
**核心定位**: 序关系直观  
**学习难度**: ⭐ (入门级)

---

## 📌 直观描述

大小关系是**比较两个事物的大小**——谁大谁小。

> **日常表述**: "A比B大"、"B比A小"、"一样大"

### 比较结果

| 关系 | 符号 | 示例 | 读法 |
|-----|------|------|------|
| **大于** | > | 5 > 3 | 5大于3 |
| **小于** | < | 3 < 5 | 3小于5 |
| **等于** | = | 3 = 3 | 3等于3 |
| **大于等于** | ≥ | x ≥ 0 | x大于等于0 |
| **小于等于** | ≤ | y ≤ 10 | y小于等于10 |
| **不等于** | ≠ | 2 ≠ 3 | 2不等于3 |

---

## 🌰 具体示例

### 示例1：数字比较
- 7 > 3（7比3大）
- -2 < 0（负数小于零）
- 1/2 > 1/3（分数比较）
- √2 ≈ 1.414 > 1（无理数比较）

### 示例2：生活中的比较

| 比较对象 | 关系 | 说明 |
|---------|------|------|
| 大象 vs 老鼠 | 大象 > 老鼠 | 体重/体型 |
| 珠穆朗玛峰 vs 泰山 | 珠峰 > 泰山 | 海拔高度 |
| 光速 vs 声速 | 光速 > 声速 | 速度 |
| 地球 vs 乒乓球 | 地球 > 乒乓球 | 体积 |

### 示例3：数轴表示

```
←─────────────────────────────────→
   -3  -2  -1   0   1   2   3   4   5
              ↑
           越往右越大
```

- -2 < 1（负数小于正数）
- 0 > -5（零大于负数）
- 3 = 3（自己等于自己）

## 📐 序的基本性质

### 三歧性
对任意两个实数 a, b，下列恰有一个成立：
- a < b
- a = b
- a > b

### 传递性
- 若 a > b 且 b > c，则 a > c
- 若 a < b 且 b < c，则 a < c

### 加法保序
- 若 a > b，则 a + c > b + c
- 若 a < b，则 a + c < b + c

### 乘法保序（正数）
- 若 a > b 且 c > 0，则 ac > bc
- 若 c < 0，则不等号方向改变：ac < bc

## 💡 学习建议

1. **数轴辅助**: 在数轴上标出数字，直观比较
2. **实际例子**: 用身高、年龄、成绩等熟悉的事物练习
3. **注意负数**: 负数的大小关系容易混淆，需要特别练习

## ❓ 常见问题

**Q: 为什么 -5 < -3？**  
A: 在数轴上，-5在-3的左边，所以-5更小。可以想：欠5元比欠3元更穷。

**Q: 0是正数还是负数？**  
A: 0既不是正数也不是负数，是中性数。

**Q: 如何比较两个分数？**  
A: 通分后比较分子，或转换为小数比较。
