## 📌 直观描述

重复模式是**同样的东西反复出现**——循环、周期、重复。

> **日常表述**: "反复出现"、"周期性"、"循环"、"每隔..."

### 周期现象

| 现象 | 周期 | 说明 |
|-----|------|------|
| **昼夜交替** | 24小时 | 地球自转 |
| **四季变化** | 1年 | 地球公转 |
| **心跳** | 约1秒 | 生理节律 |
| **月经周期** | 约28天 | 生理周期 |
| **经济周期** | 数年 | 繁荣-衰退循环 |

## 🌰 具体示例

### 示例1：日常生活中的周期
- **星期**: 每7天重复
- **月份**: 每12个月重复
- **交通信号灯**: 红→黄→绿→红循环
- **课程表**: 每周重复

### 示例2：自然界中的周期

| 周期 | 时长 | 原因 |
|-----|------|------|
| 潮汐 | 约12小时 | 月球引力 |
| 月相 | 约29.5天 | 月球绕地球 |
| 太阳黑子 | 约11年 | 太阳磁场活动 |
| 冰期 | 数万年 | 地球轨道变化 |

### 示例3：音乐中的重复
- **节拍**: 强弱弱强弱弱...（3/4拍）
- **旋律重复**: A段B段A段（三段体）
- **卡农**: 同一旋律在不同时间重复

### 示例4：数学中的周期
- **小数**: 1/3 = 0.333...（3循环）
- **数列**: 1, 2, 3, 1, 2, 3, ...（周期3）
- **函数**: sin(x) 周期为2π

## 📊 周期的数学描述

### 周期函数
若函数 f(x) 满足：
$$f(x + T) = f(x)$$
对所有 x 成立，则称 T 为 f 的**周期**。

### 最小正周期
所有周期中最小的正数称为**基本周期**。

| 函数 | 基本周期 |
|-----|---------|
| sin(x), cos(x) | 2π |
| tan(x) | π |
| 常函数 | 任意正数（无最小周期）|

## ❓ 常见问题

**Q: 所有重复都是周期性的吗？**  
A: 不是。有些重复是无规律的，周期性要求严格的时间/空间间隔。

**Q: 周期可以是无理数吗？**  
A: 可以。例如 f(x) = sin(x) + sin(√2x) 的周期与√2有关。

**Q: 如何确定一个函数的周期？**  
A: 寻找最小的T使得f(x+T)=f(x)对所有x成立。
