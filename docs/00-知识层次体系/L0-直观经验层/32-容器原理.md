# L0-32: 容器原理

**概念编号**: L0-32  
**所属家族**: 集合与分类  
**核心定位**: 集合的具象化

---

## 📌 直观描述

容器原理是**把集合想象成装东西的容器**——盒子、袋子、房间。

> **日常表述**: "这个盒子里有..."、"装不进去"、"倒出来"

### 容器类比

| 集合概念 | 容器类比 |
|---------|---------|
| 元素 | 容器里的东西 |
| 子集 | 小盒子套大盒子 |
| 空集 | 空盒子 |
| 并集 | 两个盒子倒在一起 |
| 交集 | 共同的物品 |
