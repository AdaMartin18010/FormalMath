# L0-23: 时间流逝

**概念编号**: L0-23  
**所属家族**: 量与测量  
**核心定位**: 变化的感知

---

## 📌 直观描述

### 1.1 核心直观

时间是**事件的先后顺序和持续**——过去、现在、未来的流动。

> **日常表述**: "过了5分钟"、"时间过得真快"、"还要多久"

### 1.2 时间感知

| 维度 | 描述 | 示例 |
|-----|------|------|
| **顺序** | 先后关系 | 先吃饭后睡觉 |
| **持续** | 过程长短 | 电影2小时 |
| **时点** | 特定时刻 | 3点钟 |
| **周期** | 重复间隔 | 一天24小时 |
