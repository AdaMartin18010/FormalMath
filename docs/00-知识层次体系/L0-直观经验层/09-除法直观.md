# L0-09: 除法直观

**概念编号**: L0-09  
**所属家族**: 数与计数  
**核心定位**: 乘法的逆运算

---

## 📌 直观描述

### 1.1 核心直观

除法是**均分**或**包含**的操作——把东西平均分配，或者看一个数里包含几个另一个数。

> **日常表述**: "把12个苹果平均分给4个人"、"20里面有几个5"

### 1.2 多重表征

| 表征类型 | 具体表现 | 示例 |
|---------|---------|------|
| **均分表征** | 平均分配 | 12÷4 = 每人3个 |
| **包含表征** | 包含几个 | 12÷4 = 包含3个4 |
| **逆运算表征** | 乘法的逆 | 12÷4=3 因为 3×4=12 |
| **比率表征** | 比较关系 | 12是4的3倍 |

### 1.3 两种除法类型

```

除法的情境意义:

类型1: 等分除 (Partitive)
   "把12块糖分给3个小朋友，每人几块？"
   12 ÷ 3 = 4
   
类型2: 包含除 (Quotitive)
   "12块糖，每人3块，可以分给几人？"
   12 ÷ 3 = 4

```

## 🌍 物理世界对应

### 2.1 日常实例

| 场景 | 除法应用 |
|-----|---------|
| **分配** | 平均分配资源 |
| **速率** | 路程÷时间=速度 |
| **单价** | 总价÷数量=单价 |
| **密度** | 质量÷体积=密度 |
| **比率** | 比较两个量的大小 |

### 2.2 除法的认知难点

```

除法学习的困难:

困难1: 结果变小（除数>1时）
   "为什么越除越少？"
   
困难2: 余数的处理
   "13÷4=3...1"，余数怎么办？
   
困难3: 除不尽
   "10÷3=?"，需要分数
   
困难4: 除以0
   "为什么不能除以0？"

```
