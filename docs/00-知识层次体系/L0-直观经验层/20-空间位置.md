# L0-20: 空间位置

**概念编号**: L0-20  
**所属家族**: 空间与几何  
**核心定位**: 拓扑直觉的基础

---

## 📌 直观描述

### 1.1 核心直观

空间位置描述物体**在哪里**以及**与其他物体的关系**——内外、相邻、隔离。

> **日常表述**: "在盒子里"、"在旁边"、"在对面"

### 1.2 位置关系

| 关系 | 直观描述 | 示例 |
|-----|---------|------|
| **内部/外部** | 在边界内还是外 | 球在盒子里 |
| **相邻** | 紧挨着 | 两个房间相邻 |
| **分离** | 不接触 | 两座山之间 |
| **连通** | 可以到达 | 从A能走到B |
| **包围** | 一个包着另一个 | 蛋壳包着蛋白 |

### 1.3 位置认知的发展

```

空间位置认知:

拓扑关系（最早掌握）:
  - 内外
  - 连通/分离
  
投影关系（较晚）:
  - 前后遮挡
  
欧氏关系（最晚）:
  - 精确距离
  - 角度测量

```
