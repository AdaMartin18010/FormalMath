# L0-44: 对称直观

**概念编号**: L0-44  
**所属家族**: 变化与模式  
**核心定位**: 对称性感知

---

## 📌 直观描述

对称直观是**两边一样**的感觉——折叠重合、镜像相同。

> **日常表述**: "对称"、"两边一样"、"镜像"、"平衡"

### 对称类型

| 类型 | 描述 | 示例 |
|-----|------|------|
| **轴对称** | 沿直线折叠重合 | 蝴蝶 |
| **中心对称** | 旋转180°重合 | 字母S |
| **旋转对称** | 旋转一定角度重合 | 雪花 |
