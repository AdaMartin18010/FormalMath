# 最优化 / Optimization

**最后更新**: 2026年4月4日

---

## 📋 分支概述

最优化是研究如何从众多可行方案中选择最优方案的数学分支。它在运筹学、经济学、工程、机器学习等领域有广泛应用。本分支系统构建最优化的完整理论体系，从凸优化到非线性规划，从理论分析到算法实现。

## 📚 核心人物

- **Joseph-Louis Lagrange (1736-1813)**: 拉格朗日乘数法
- **Leonid Kantorovich (1912-1986)**: 线性规划创始人，诺贝尔经济学奖
- **George Dantzig (1914-2005)**: 单纯形法发明者
- **Harold Kuhn (1925-2014) & Albert Tucker (1905-1995)**: KKT条件
- **Stephen Boyd**: 现代凸优化，著作《Convex Optimization》

## 🔗 与其他分支的联系

- **线性代数**: 矩阵运算、特征值分解在最优化中广泛应用
- **分析学**: 梯度、Hessian矩阵、收敛性分析
- **概率论**: 随机优化、随机梯度下降
- **图论**: 网络流问题、组合优化
- **机器学习**: 参数优化、损失函数最小化

---

## 📖 学习路径建议

1. **先修知识**: 线性代数、多元微积分
2. **第一阶段**: 凸优化基础（凸集、凸函数、凸问题）
3. **第二阶段**: 线性规划（单纯形法、对偶理论）
4. **第三阶段**: 非线性规划（无约束/约束优化、KKT条件）
5. **第四阶段**: 高级算法（内点法、一阶方法、随机优化）
