## 1. 问题建模

### 1.1 从实际问题到数学模型

**示例：生产计划问题**

某工厂生产两种产品A和B，需要消耗三种资源：劳动力、原材料和机器时间。

| 资源 | 产品A消耗 | 产品B消耗 | 可用总量 |
|------|-----------|-----------|----------|
| 劳动力 | 2小时 | 1小时 | 100小时 |
| 原材料 | 1单位 | 1单位 | 80单位 |
| 机器时间 | 1小时 | 3小时 | 120小时 |
| **利润** | 40元 | 60元 | - |

**决策变量**:
- $x_1$: 产品A的产量
- $x_2$: 产品B的产量

**数学模型**:
$$
\begin{aligned}
\max \quad & z = 40x_1 + 60x_2 \\
\text{s.t.} \quad & 2x_1 + x_2 \leq 100 \quad \text{(劳动力)} \\
& x_1 + x_2 \leq 80 \quad \text{(原材料)} \\
& x_1 + 3x_2 \leq 120 \quad \text{(机器时间)} \\
& x_1, x_2 \geq 0
\end{aligned}
$$

### 1.2 标准形式

**定义 1.1** (线性规划标准型):
$$
\begin{aligned}
\min \quad & c^T x \\
\text{s.t.} \quad & Ax = b \\
& x \geq 0
\end{aligned}
$$

其中 $A \in \mathbb{R}^{m \times n}$, $b \in \mathbb{R}^m$, $c \in \mathbb{R}^n$, 且 $\text{rank}(A) = m$。

**转化为标准型的方法**:
1. 最大化转最小化: $\max c^T x \Rightarrow \min (-c)^T x$
2. 不等式转等式: 引入松弛变量 $s \geq 0$
   $$a^T x \leq b \Rightarrow a^T x + s = b$$
3. 自由变量: $x = x^+ - x^-$, 其中 $x^+, x^- \geq 0$

## 2. 数学理论

### 2.1 基本几何概念

**定义 2.1** (凸集): 集合 $C \subseteq \mathbb{R}^n$ 是凸集，若对任意 $x, y \in C$ 和 $\lambda \in [0, 1]$:
$$\lambda x + (1-\lambda)y \in C$$

**定义 2.2** (极点/顶点): 凸集 $C$ 的点 $x$ 称为极点，若不存在 $y, z \in C$, $y \neq z$, 使得 $x = \lambda y + (1-\lambda)z$ 对某个 $\lambda \in (0, 1)$ 成立。

**定理 2.1** (可行域的凸性): 线性规划的可行域 $S = \{x : Ax = b, x \geq 0\}$ 是凸集。

*证明*: 设 $x_1, x_2 \in S$，则 $Ax_1 = b$, $Ax_2 = b$，且 $x_1, x_2 \geq 0$。
对任意 $\lambda \in [0, 1]$，令 $x = \lambda x_1 + (1-\lambda)x_2$，则：
$$Ax = A(\lambda x_1 + (1-\lambda)x_2) = \lambda Ax_1 + (1-\lambda)Ax_2 = \lambda b + (1-\lambda)b = b$$
且 $x = \lambda x_1 + (1-\lambda)x_2 \geq 0$。
因此 $x \in S$，$S$ 是凸集。$\square$

### 2.2 基本可行解

**定义 2.3** (基): 设 $A$ 是 $m \times n$ 矩阵，$\text{rank}(A) = m$。$A$ 的 $m \times m$ 非奇异子矩阵 $B$ 称为基矩阵。

**定义 2.4** (基本可行解): 对应于基 $B$ 的基本解为 $x = [B^{-1}b; 0]$。若 $B^{-1}b \geq 0$，则称为**基本可行解(BFS)**。

**定理 2.2** (基本可行解与顶点的等价性): $x$ 是可行域的顶点当且仅当 $x$ 是基本可行解。

### 2.3 最优性理论

**定理 2.3** (线性规划基本定理): 对于标准型线性规划：
1. 若存在最优解，则存在基本可行解为最优解
2. 若可行域非空且有界，则必存在最优解
3. 若可行域非空但无界，则要么存在最优解，要么问题无界

**定理 2.4** (最优性条件): 设 $x^*$ 是可行解，若存在 $y \in \mathbb{R}^m$ 使得：
$$c - A^T y \geq 0, \quad (c - A^T y)^T x^* = 0$$
则 $x^*$ 是最优解。

## 3. 算法与方法

### 3.1 单纯形法

**算法 3.1** (单纯形法)

```
输入: 初始基本可行解 x，基 B
输出: 最优解 x* 或无界判定

1. 计算简约成本: c̄_N = c_N - A_N^T (B^{-T} c_B)
2. 若 c̄_N ≥ 0，当前解最优，停止
3. 选择进基变量: j = argmin{c̄_j : c̄_j < 0}
4. 计算方向: d_B = -B^{-1} A_j
5. 若 d_B ≥ 0，问题无界，停止
6. 选择离基变量: 
   i = argmin{-(B^{-1}b)_k / (d_B)_k : (d_B)_k < 0}
7. 更新基: B ← B ∪ {j} \ {i}，转步骤1
```

**复杂度分析**:
- 每轮迭代: $O(mn)$
- 最坏情况: 指数级 (Klee-Minty 例子)
- 平均情况: 多项式级

### 3.2 对偶理论

**定义 3.1** (对偶问题): 原问题(P)的对偶问题(D)为：
$$
\begin{aligned}
\text{(P)} \quad \max \quad & c^T x \\
\text{s.t.} \quad & Ax \leq b \\
& x \geq 0
\end{aligned}
\quad \Leftrightarrow \quad
\begin{aligned}
\text{(D)} \quad \min \quad & b^T y \\
\text{s.t.} \quad & A^T y \geq c \\
& y \geq 0
\end{aligned}
$$

**定理 3.1** (弱对偶): 若 $x$ 是(P)的可行解，$y$ 是(D)的可行解，则 $c^T x \leq b^T y$。

**定理 3.2** (强对偶): 若(P)或(D)有有限最优值，则两者最优值相等。

**定理 3.3** (互补松弛性): 设 $x^*, y^*$ 分别是(P)和(D)的最优解，则：
1. $y_i^* (b_i - a_i^T x^*) = 0$ 对所有 $i$
2. $x_j^* (A_j^T y^* - c_j) = 0$ 对所有 $j$

### 3.3 内点法

**算法 3.2** (原始-对偶内点法)

```
输入: 初始点 (x, y, s) > 0，容差 ε > 0
输出: ε-最优解

1. 计算对偶间隙: μ = x^T s / n
2. 若 μ < ε，停止
3. 求解牛顿系统:
   [ 0   A^T  I  ] [ Δx ]   [ 0    ]
   [ A   0    0  ] [ Δy ] = [ r_b  ]
   [ S   0    X  ] [ Δs ]   [ r_c  ]
   
   其中 r_b = b - Ax, r_c = c - A^T y - s
4. 选择步长 α，保持 (x, s) > 0
5. 更新: (x, y, s) ← (x, y, s) + α(Δx, Δy, Δs)
6. 转步骤1
```

**复杂度**: $O(\sqrt{n} \cdot L)$ 次迭代，$L$ 为输入长度

## 4. 具体案例

### 4.1 案例：生产计划优化

**问题数据** (同1.1节)

**求解过程**:

标准型：引入松弛变量 $s_1, s_2, s_3$
$$
\begin{aligned}
\max \quad & 40x_1 + 60x_2 \\
\text{s.t.} \quad & 2x_1 + x_2 + s_1 = 100 \\
& x_1 + x_2 + s_2 = 80 \\
& x_1 + 3x_2 + s_3 = 120 \\
& x_1, x_2, s_1, s_2, s_3 \geq 0
\end{aligned}
$$

初始基: $B = \{s_1, s_2, s_3\}$, $x_B = [100, 80, 120]^T$

**迭代1**: 进基 $x_2$ (系数60最大)，离基 $s_3$ (最小比值120/3=40)

**迭代2**: 进基 $x_1$，离基 $s_2$

**最优解**: $x_1 = 30$, $x_2 = 50$, 最大利润 $z = 4200$ 元

### 4.2 敏感性分析

**对偶价格** (影子价格):
- 劳动力: 20元/小时
- 原材料: 0元/单位 (有剩余)
- 机器时间: 20元/小时

**解释**: 劳动力和机器时间是瓶颈资源，增加1小时可多赚20元。

## 5. 代码实现

### 5.1 Python实现 (SciPy)

```python
import numpy as np
from scipy.optimize import linprog

# 目标函数系数 (注意：linprog默认最小化，所以取负)
c = [-40, -60]

# 不等式约束 A_ub @ x <= b_ub
A_ub = [
    [2, 1],    # 劳动力
    [1, 1],    # 原材料
    [1, 3]     # 机器时间
]
b_ub = [100, 80, 120]

# 变量非负约束 (默认)
bounds = [(0, None), (0, None)]

# 求解
result = linprog(c, A_ub=A_ub, b_ub=b_ub, bounds=bounds, method='highs')

print(f"最优解: x1 = {result.x[0]:.2f}, x2 = {result.x[1]:.2f}")
print(f"最大利润: {-result.fun:.2f}")
print(f"迭代次数: {result.nit}")
print(f"是否成功: {result.success}")

# 对偶价格
print(f"\n对偶变量 (影子价格): {result.ineqlin.marginals}")
```

**输出**:
```
最优解: x1 = 30.00, x2 = 50.00
最大利润: 4200.00
迭代次数: 2
是否成功: True
对偶变量: [20.  0. 20.]
```

### 5.2 单纯形法实现

```python
import numpy as np

class SimplexSolver:
    def __init__(self, A, b, c):
        self.A = np.array(A, dtype=float)
        self.b = np.array(b, dtype=float)
        self.c = np.array(c, dtype=float)
        self.m, self.n = self.A.shape
        
    def solve(self):
        # 添加松弛变量
        A_ext = np.hstack([self.A, np.eye(self.m)])
        c_ext = np.hstack([self.c, np.zeros(self.m)])
        
        # 基变量索引
        basis = list(range(self.n, self.n + self.m))
        non_basis = list(range(self.n))
        
        iteration = 0
        max_iter = 1000
        
        while iteration < max_iter:
            iteration += 1
            
            # 计算基矩阵和逆
            B = A_ext[:, basis]
            B_inv = np.linalg.inv(B)
            x_B = B_inv @ self.b
            
            # 计算简约成本
            c_B = c_ext[basis]
            y = B_inv.T @ c_B
            reduced_cost = c_ext[non_basis] - A_ext[:, non_basis].T @ y
            
            # 最优性检验
            if np.all(reduced_cost >= -1e-10):
                x_opt = np.zeros(self.n + self.m)
                x_opt[basis] = x_B
                return {
                    'success': True,
                    'x': x_opt[:self.n],
                    'objective': c_ext @ x_opt,
                    'iterations': iteration
                }
            
            # 选择进基变量
            entering = non_basis[np.argmin(reduced_cost)]
            
            # 计算方向
            d = B_inv @ A_ext[:, entering]
            
            # 检查无界
            if np.all(d <= 1e-10):
                return {'success': False, 'message': 'Problem unbounded'}
            
            # 选择离基变量 (最小比值检验)
            ratios = np.where(d > 1e-10, x_B / d, np.inf)
            leaving_idx = np.argmin(ratios)
            leaving = basis[leaving_idx]
            
            # 更新基
            basis[leaving_idx] = entering
            non_basis = [i for i in range(self.n + self.m) if i not in basis]
        
        return {'success': False, 'message': 'Max iterations exceeded'}

# 求解
A = [[2, 1], [1, 1], [1, 3]]
b = [100, 80, 120]
c = [-40, -60]  # 最大化

solver = SimplexSolver(A, b, c)
result = solver.solve()
print(result)
```

### 习题 7.1 (建模)

某制药公司生产两种药物A和B，需要经过三个工序：原料处理、合成、包装。每单位产品的资源消耗和利润如下：

| 工序 | 药物A | 药物B | 日可用量 |
|------|-------|-------|----------|
| 原料处理 | 1小时 | 2小时 | 8小时 |
| 合成 | 2小时 | 1小时 | 10小时 |
| 包装 | 1小时 | 1小时 | 6小时 |
| 利润 | 3万元 | 2万元 | - |

**要求**: 
1. 建立线性规划模型
2. 求最优生产计划
3. 计算各工序的影子价格
4. 若原料处理时间可额外租用(500元/小时)，是否值得？

### 习题 7.2 (理论证明)

证明：若线性规划存在两个不同的最优基本可行解 $x^{(1)}$ 和 $x^{(2)}$，则存在无穷多个最优解。

### 习题 7.3 (单纯形法)

用单纯形法求解：
$$
\begin{aligned}
\max \quad & 3x_1 + 2x_2 + 5x_3 \\
\text{s.t.} \quad & x_1 + 2x_2 + x_3 \leq 430 \\
& 3x_1 + 2x_3 \leq 460 \\
& x_1 + 4x_2 \leq 420 \\
& x_1, x_2, x_3 \geq 0
\end{aligned}
$$

要求：
1. 列出每轮迭代的单纯形表
2. 指出进基和离基变量
3. 验证最优性条件

### 习题 7.4 (对偶问题)

写出下列问题的对偶问题，并证明强对偶性：
$$
\begin{aligned}
\min \quad & 2x_1 + 3x_2 + 5x_3 \\
\text{s.t.} \quad & x_1 + x_2 - x_3 \geq 4 \\
& 2x_1 - x_2 + 4x_3 = 5 \\
& x_1, x_2 \geq 0, \quad x_3 \text{自由}
\end{aligned}
$$
