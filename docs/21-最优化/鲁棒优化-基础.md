# 鲁棒优化基础

## 不确定性集合

### 盒式不确定性
$$\mathcal{U} = \{u: \|u\|_\infty \leq \hat{u}\}$$

### 椭球不确定性
$$\mathcal{U} = \{u: u^T \Sigma^{-1} u \leq \Gamma^2\}$$

## 鲁棒对应

### 线性规划
$$\min_x \{c^T x: Ax \leq b, \forall A \in \mathcal{U}_A\}$$
