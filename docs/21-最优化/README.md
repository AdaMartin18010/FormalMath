# 最优化理论与方法

最优化理论与方法在工程、经济、机器学习与运筹学中具有广泛应用。本节系统介绍线性规划、凸优化、组合优化与鲁棒优化的核心内容。

## 1. 线性规划

### 1.1 标准形式

**定义 1.1（线性规划）**. 标准形式为：

$$\min c^T x \quad \text{s.t.} \quad Ax = b, \quad x \geq 0.$$

### 1.2 对偶理论

**定理 1.2（强对偶）**. 若原问题有最优解且约束规格成立，则对偶问题有最优解，且最优值相等。

*证明*. 利用单纯形法或 Farkas 引理。$\square$

## 2. 凸优化

### 2.1 凸函数

**定义 2.1（凸函数）**. $f: \mathbb{R}^n \to \mathbb{R}$ 凸，若 $f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y)$（$\lambda \in [0,1]$）。

### 2.2 最优性条件

**定理 2.2（KKT）**. 对凸优化问题，在约束规格下，$x^*$ 最优当且仅当存在 $\lambda$ 使：

$$\nabla f(x^*) + \sum_i \lambda_i \nabla g_i(x^*) = 0, \quad \lambda_i \geq 0, \quad \lambda_i g_i(x^*) = 0.$$

## 3. 例子

### 3.1 例子：最小二乘

$$\min_x \|Ax - b\|_2^2$$

解为 $x = (A^TA)^{-1}A^Tb$（正规方程）。

### 3.2 例子：支持向量机

$$\min_{w,b} \frac{1}{2}\|w\|^2 \quad \text{s.t.} \quad y_i(w^Tx_i + b) \geq 1$$

对偶问题用核技巧扩展到非线性情形。
