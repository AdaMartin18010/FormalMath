# 生产计划优化案例

## 问题描述

某电子制造厂生产两种产品：智能手机和平板电脑。生产受以下资源约束：

| 资源 | 手机(单位消耗) | 平板(单位消耗) | 日可用量 |
|------|----------------|----------------|----------|
| 组装线(小时) | 2 | 3 | 100 |
| 质检人员(人时) | 1 | 2 | 60 |
| 原材料(单位) | 4 | 2 | 120 |
| 仓储空间(立方米) | 2 | 1 | 50 |
| **单位利润(元)** | 300 | 500 | - |

## 数学建模

### 决策变量
- $x_1$: 每日生产手机数量
- $x_2$: 每日生产平板数量

### 目标函数
$$\max Z = 300x_1 + 500x_2$$

### 约束条件
$$\begin{aligned}
2x_1 + 3x_2 &\leq 100 \quad \text{(组装线)} \\
x_1 + 2x_2 &\leq 60 \quad \text{(质检)} \\
4x_1 + 2x_2 &\leq 120 \quad \text{(原材料)} \\
2x_1 + x_2 &\leq 50 \quad \text{(仓储)} \\
x_1, x_2 &\geq 0
\end{aligned}$$

## Python求解

```python
from scipy.optimize import linprog
import numpy as np

# 目标函数系数 (最大化转最小化)
c = [-300, -500]

# 不等式约束矩阵
A_ub = [
    [2, 3],    # 组装线
    [1, 2],    # 质检
    [4, 2],    # 原材料
    [2, 1]     # 仓储
]

b_ub = [100, 60, 120, 50]

# 求解
result = linprog(c, A_ub=A_ub, b_ub=b_ub, bounds=[(0, None), (0, None)], 
                 method='highs')

print(f"最优解:")
print(f"  手机产量: {result.x[0]:.2f} 台/日")
print(f"  平板产量: {result.x[1]:.2f} 台/日")
print(f"最大日利润: {-result.fun:.2f} 元")
print(f"\n影子价格: {result.ineqlin.marginals}")
```

## 求解结果

**最优生产计划**:
- 手机: 10 台/日
- 平板: 25 台/日
- **最大日利润: 15,500 元**

**敏感性分析**:
- 组装线: 影子价格 150 元/小时 (瓶颈资源)
- 质检: 影子价格 0 (有剩余)
- 原材料: 影子价格 0 (有剩余)
- 仓储: 影子价格 0 (有剩余)

**管理建议**:
1. 组装线是瓶颈，增加组装线产能可显著提升利润
2. 质检和原材料有剩余，可缩减或重新配置
3. 每增加1小时组装线产能，日利润增加150元

## 结果验证
