## 学习路径

### 基础路径
```
线性代数 → 多元微积分 → 线性规划 → 凸优化 → 非线性规划
```

### 进阶路径
```
凸优化 → 组合优化/整数规划 → 动态规划 → 随机优化
```

### 应用路径
- **运筹学**: 线性规划 + 网络优化 + 库存管理
- **机器学习**: 凸优化 + 随机优化 + 正则化理论
- **金融工程**: 二次规划 + 随机规划 + 风险管理
- **工程设计**: 非线性规划 + 多目标优化 + 拓扑优化

## 核心定理速查

| 定理 | 描述 | 应用领域 |
|------|------|----------|
| **单纯形法收敛** | 有限步内找到最优解或判定无解 | 线性规划 |
| **强对偶定理** | 原问题与对偶问题最优值相等 | 优化理论 |
| **KKT条件** | 约束优化最优解的必要条件 | 非线性规划 |
| **凸优化全局最优** | 凸问题的局部最优即全局最优 | 凸优化 |
| **贝尔曼最优性** | 最优策略的子策略也是最优的 | 动态规划 |

## 算法复杂度汇总

| 算法 | 问题类型 | 时间复杂度 | 空间复杂度 |
|------|----------|------------|------------|
| 单纯形法 | 线性规划 | 多项式平均/指数最坏 | O(mn) |
| 内点法 | 凸优化 | O(√n log(1/ε)) | O(n²) |
| 分支定界 | 整数规划 | 指数级 | O(n) |
| 梯度下降 | 无约束优化 | O(1/ε) | O(n) |
| 牛顿法 | 无约束优化 | O(log log(1/ε)) | O(n²) |
| L-BFGS | 大规模优化 | O(1/√k) | O(mn) |
