## 1. 问题建模

### 1.1 凸优化问题定义

**定义 1.1** (凸优化问题): 形如下的优化问题称为凸优化问题：
$$
\begin{aligned}
\min \quad & f_0(x) \\
\text{s.t.} \quad & f_i(x) \leq 0, \quad i = 1, \ldots, m \\
& h_j(x) = 0, \quad j = 1, \ldots, p
\end{aligned}
$$
其中 $f_0, f_1, \ldots, f_m$ 是凸函数，$h_j$ 是仿射函数：$h_j(x) = a_j^T x - b_j$。

### 1.2 建模实例：投资组合优化

**问题**: 投资者有资金 $W$，可在 $n$ 种资产中分配，目标是最大化收益同时控制风险。

**决策变量**: $x \in \mathbb{R}^n$，$x_i$ 为投资于资产 $i$ 的比例

**数据**:
- $\mu \in \mathbb{R}^n$: 预期收益率向量
- $\Sigma \in \mathbb{S}_{++}^n$: 收益率协方差矩阵（正定）
- $R_{\min}$: 最低预期收益率要求

**Markowitz均值-方差模型**:
$$
\begin{aligned}
\min \quad & x^T \Sigma x \quad \text{(组合方差/风险)} \\
\text{s.t.} \quad & \mu^T x \geq R_{\min} \quad \text{(收益约束)} \\
& \mathbf{1}^T x = 1 \quad \text{(预算约束)} \\
& x \geq 0 \quad \text{(不允许卖空)}
\end{aligned}
$$

**凸性验证**:
- 目标函数: $x^T \Sigma x$ 是凸函数（因 $\Sigma \succ 0$）
- 不等式约束: $\mu^T x \geq R_{\min}$ 等价于 $R_{\min} - \mu^T x \leq 0$，是凸约束
- 等式约束: 仿射约束
- 不等式约束: $x \geq 0$ 是凸约束

### 1.3 常见凸优化问题类型

| 问题类型 | 标准形式 | 特点 |
|----------|----------|------|
| **LP** | $\min c^T x$ s.t. $Ax \leq b$ | 线性目标和约束 |
| **QP** | $\min \frac{1}{2}x^T P x + q^T x$ | $P \succeq 0$ |
| **QCQP** | QP + 二次约束 $x^T P_i x + q_i^T x \leq r_i$ | $P_i \succeq 0$ |
| **SOCP** | $\min f^T x$ s.t. $\|A_i x + b_i\|_2 \leq c_i^T x + d_i$ | 二阶锥约束 |
| **SDP** | $\min c^T x$ s.t. $x_1 F_1 + \cdots + x_n F_n + G \preceq 0$ | 半定约束 |

## 2. 数学理论

### 2.1 凸集与凸函数

**定义 2.1** (凸集): $C \subseteq \mathbb{R}^n$ 是凸集，若对任意 $x, y \in C$ 和 $\theta \in [0, 1]$:
$$\theta x + (1-\theta)y \in C$$

**定义 2.2** (凸函数): $f: \mathbb{R}^n \to \mathbb{R}$ 是凸函数，若 $\text{dom } f$ 是凸集，且对任意 $x, y \in \text{dom } f$，$\theta \in [0, 1]$:
$$f(\theta x + (1-\theta)y) \leq \theta f(x) + (1-\theta)f(y)$$

**定理 2.1** (一阶条件): 设 $f$ 可微，则 $f$ 是凸函数当且仅当：
$$f(y) \geq f(x) + \nabla f(x)^T (y - x), \quad \forall x, y \in \text{dom } f$$

**定理 2.2** (二阶条件): 设 $f$ 二阶可微，则 $f$ 是凸函数当且仅当Hessian矩阵半正定：
$$\nabla^2 f(x) \succeq 0, \quad \forall x \in \text{dom } f$$

### 2.2 最优性条件

**定理 2.3** (凸优化全局最优性): 对于凸优化问题，任何局部最优解都是全局最优解。

*证明*: 设 $x^*$ 是局部最优解，存在 $r > 0$ 使得 $f_0(x^*) \leq f_0(x)$ 对所有满足 $\|x - x^*\| < r$ 的可行 $x$ 成立。

假设 $x^*$ 不是全局最优，则存在可行解 $y$ 使得 $f_0(y) < f_0(x^*)$。

考虑 $z = \theta x^* + (1-\theta)y$，$\theta \in (0, 1)$。由凸性：
- $z$ 可行（可行域凸）
- $f_0(z) \leq \theta f_0(x^*) + (1-\theta)f_0(y) < f_0(x^*)$

取 $\theta$ 接近1使 $\|z - x^*\| < r$，则 $z$ 在 $x^*$ 的邻域内但 $f_0(z) < f_0(x^*)$，矛盾。$\square$

**定理 2.4** (无约束最优性): 对于无约束凸优化 $\min f(x)$，$x^*$ 是最优解当且仅当：
$$\nabla f(x^*) = 0$$

### 2.3 约束最优性：KKT条件

**定义 2.3** (拉格朗日函数): 
$$\mathcal{L}(x, \lambda, \nu) = f_0(x) + \sum_{i=1}^m \lambda_i f_i(x) + \sum_{j=1}^p \nu_j h_j(x)$$

**定理 2.5** (KKT条件): 对于凸优化问题，若Slater条件成立（存在严格可行点），则 $x^*$ 是最优解当且仅当存在 $(\lambda^*, \nu^*)$ 满足：

1. **平稳性**: $\nabla_x \mathcal{L}(x^*, \lambda^*, \nu^*) = 0$
2. **原始可行性**: $f_i(x^*) \leq 0$，$h_j(x^*) = 0$
3. **对偶可行性**: $\lambda_i^* \geq 0$
4. **互补松弛性**: $\lambda_i^* f_i(x^*) = 0$

## 3. 算法与方法

### 3.1 梯度下降法

**算法 3.1** (梯度下降)

```
输入: 初始点 x_0，步长序列 {α_k}
输出: 近似最优解

for k = 0, 1, 2, ... until convergence:
    x_{k+1} = x_k - α_k ∇f(x_k)
```

**收敛性分析** (L-光滑且 μ-强凸):
- 固定步长 $α \leq 1/L$: 
  $$f(x_k) - f^* \leq (1 - \mu/L)^k (f(x_0) - f^*)$$
- 迭代复杂度达到 ε-最优: $O(\kappa \log(1/\varepsilon))$，其中 $\kappa = L/\mu$ 是条件数

### 3.2 牛顿法

**算法 3.2** (牛顿法)

```
输入: 初始点 x_0，容差 ε
输出: 近似最优解

for k = 0, 1, 2, ...:
    求解: ∇²f(x_k) d_k = -∇f(x_k)   (牛顿方向)
    选择步长 α_k (回溯线搜索)
    x_{k+1} = x_k + α_k d_k
    若 ‖∇f(x_{k+1})‖ ≤ ε，停止
```

**收敛性**:
- 局部二次收敛: $\|x_{k+1} - x^*\| = O(\|x_k - x^*\|^2)$
- 每次迭代需要求解线性系统，复杂度 $O(n^3)$

### 3.3 内点法

**算法 3.3** (障碍函数内点法)

```
输入: 严格可行初始点 x_0，容差 ε > 0
输出: ε-最优解

for t = t_0, μt_0, μ²t_0, ... while m/t ≥ ε:
    中心点: x*(t) = argmin t·f_0(x) + φ(x)
           其中 φ(x) = -Σ log(-f_i(x)) 是对数障碍函数
    用牛顿法求解中心点问题
```

**复杂度**: $O(\sqrt{m} \log(1/\varepsilon))$ 次外迭代

## 4. 具体案例

### 4.1 案例：最小二乘拟合

**问题**: 给定数据点 $(t_i, y_i)$，$i = 1, \ldots, m$，拟合模型 $y = \sum_{j=1}^n x_j f_j(t)$

**优化问题**:
$$\min_x \|Ax - y\|_2^2 = \sum_{i=1}^m (\sum_{j=1}^n a_{ij} x_j - y_i)^2$$

其中 $a_{ij} = f_j(t_i)$。

**解析解**: $x^* = (A^T A)^{-1} A^T y$（正规方程）

### 4.2 案例：鲁棒回归

**问题**: 最小化Huber损失以处理异常值

$$\min_x \sum_{i=1}^m \phi_{\text{huber}}(a_i^T x - y_i)$$

Huber函数:
$$\phi_{\text{huber}}(u) = \begin{cases} u^2 & |u| \leq M \\ 2M|u| - M^2 & |u| > M \end{cases}$$

转化为QP:
$$
\begin{aligned}
\min \quad & \sum_{i=1}^m v_i \\
\text{s.t.} \quad & (a_i^T x - y_i)^2 \leq v_i, \quad |a_i^T x - y_i| \leq M \\
& 2M|a_i^T x - y_i| - M^2 \leq v_i, \quad |a_i^T x - y_i| > M
\end{aligned}
$$

## 5. 代码实现

### 5.1 CVXPY实现

```python
import cvxpy as cp
import numpy as np
import matplotlib.pyplot as plt

# 生成投资组合数据
np.random.seed(42)
n = 10  # 资产数量
mu = np.random.uniform(0.05, 0.15, n)  # 预期收益率 5%-15%
Sigma = np.random.rand(n, n)
Sigma = Sigma @ Sigma.T + 0.1 * np.eye(n)  # 协方差矩阵

# 决策变量
x = cp.Variable(n)

# 参数
R_min = 0.10  # 最低收益率10%

# 目标函数：最小化风险（方差）
objective = cp.Minimize(cp.quad_form(x, Sigma))

# 约束
constraints = [
    mu @ x >= R_min,  # 收益约束
    cp.sum(x) == 1,   # 预算约束
    x >= 0            # 不允许卖空
]

# 求解
problem = cp.Problem(objective, constraints)
result = problem.solve()

print(f"最优投资组合: {x.value.round(4)}")
print(f"组合风险 (标准差): {np.sqrt(result):.4f}")
print(f"预期收益率: {mu @ x.value:.4f}")

# 计算有效前沿
returns = np.linspace(0.06, 0.14, 50)
risks = []

for R in returns:
    x = cp.Variable(n)
    prob = cp.Problem(
        cp.Minimize(cp.quad_form(x, Sigma)),
        [mu @ x >= R, cp.sum(x) == 1, x >= 0]
    )
    prob.solve()
    risks.append(np.sqrt(prob.value))

# 绘制有效前沿
plt.figure(figsize=(10, 6))
plt.plot(risks, returns, 'b-', linewidth=2, label='Efficient Frontier')
plt.xlabel('Risk (Standard Deviation)')
plt.ylabel('Expected Return')
plt.title('Portfolio Optimization Efficient Frontier')
plt.grid(True)
plt.legend()
plt.savefig('efficient_frontier.png', dpi=150)
plt.show()
```

### 5.2 梯度下降实现

```python
import numpy as np

def gradient_descent(f, grad_f, x0, alpha=0.01, max_iter=1000, tol=1e-6):
    """
    梯度下降算法
    
    参数:
        f: 目标函数
        grad_f: 梯度函数
        x0: 初始点
        alpha: 步长
        max_iter: 最大迭代次数
        tol: 收敛容差
    """
    x = x0.copy()
    history = []
    
    for k in range(max_iter):
        gradient = grad_f(x)
        history.append(f(x))
        
        if np.linalg.norm(gradient) < tol:
            print(f"收敛于迭代 {k}")
            break
            
        x = x - alpha * gradient
    
    return x, history

# 测试：Rosenbrock函数 (非凸，用于演示)
def rosenbrock(x):
    return (1 - x[0])**2 + 100 * (x[1] - x[0]**2)**2

def grad_rosenbrock(x):
    dx0 = -2*(1 - x[0]) - 400*x[0]*(x[1] - x[0]**2)
    dx1 = 200*(x[1] - x[0]**2)
    return np.array([dx0, dx1])

# 求解
x0 = np.array([-1.0, 2.0])
x_opt, history = gradient_descent(rosenbrock, grad_rosenbrock, x0, 
                                   alpha=0.001, max_iter=10000)

print(f"最优解: {x_opt}")
print(f"最优值: {rosenbrock(x_opt):.10f}")
```

### 习题 7.4 (梯度下降分析)

设 $f(x) = \frac{1}{2}x^T Q x - b^T x$，其中 $Q \succ 0$，条件数 $\kappa = \lambda_{\max}/\lambda_{\min}$。

证明：
1. 固定步长梯度下降收敛当且仅当 $0 < \alpha < 2/\lambda_{\max}$
2. 最优步长 $\alpha^* = 2/(\lambda_{\max} + \lambda_{\min})$
3. 收敛率满足 $\|x_k - x^*\| \leq \left(\frac{\kappa - 1}{\kappa + 1}\right)^k \|x_0 - x^*\|$
