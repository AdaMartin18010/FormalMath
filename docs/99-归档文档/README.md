# 归档文档目录

## 目录说明

本目录用于存放与FormalMath项目主题和子主题无关的文档。

## 项目主题

FormalMath项目是一个**数学知识梳理和全面分析**项目，核心主题包括：

1. **理论基础** - ZFC公理体系、代数结构论证体系等
2. **核心理论** - 各数学分支的理论分析文档
3. **应用分析** - 各领域的应用场景分析
4. **知识梳理** - 知识梳理和结构关系分析文档
5. **形式化实现** - Lean4形式化实现文档
6. **技术实现参考** - Python等技术实现参考（作为参考，非核心）

## 归档分类

### 项目推进报告

存放项目推进过程中的周报、月报、阶段报告等。

### 批判性分析报告

存放项目的批判性分析报告和对标分析报告。

### 项目改进计划

存放项目的改进计划、执行计划等。

### 项目完成报告

存放项目的完成报告、确认报告等。

### 质量检查报告

存放历史的质量检查报告和对齐修正报告。
