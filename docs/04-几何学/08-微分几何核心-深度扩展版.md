## 🌐 二、流形基础

### 2.1 光滑流形定义

**定义**：n维光滑流形M是第二可数Hausdorff空间，配备极大光滑图册{(Uα, φα)}，使得：

1. M = ∪α Uα
2. φα: Uα → ℝⁿ 是同胚
3. 转移函数φβ ∘ φα⁻¹光滑

### 2.2 流形构造图

```text
        [流形定义层次]
              │
    ┌─────────┴─────────┐
    │                   │
[拓扑流形]          [光滑流形]
    │                   │
局部同胚于ℝⁿ       转移函数光滑
    │                   │
    └─────────┬─────────┘
              │
        [坐标卡覆盖]
              │
    ┌─────────┼─────────┐
    │         │         │
  (U₁,φ₁)  (U₂,φ₂)   ...
    │         │
    └─────────┘
        转移函数
    φ₂∘φ₁⁻¹: φ₁(U₁∩U₂) → φ₂(U₁∩U₂)
```

### 2.3 重要例子

| 流形 | 维数 | 描述 |
|------|------|------|
| Sⁿ | n | n维球面 |
| ℝPⁿ | n | 实射影空间 |
| Tⁿ | n | n维环面 |
| SO(n) | n(n-1)/2 | 特殊正交群 |
| Gr(k,n) | k(n-k) | Grassmann流形 |

## 📐 三、切丛与向量场

### 3.1 切空间定义

**定义（导数定义）**：p点处切空间T_pM是在p点处的导算子空间：

```
v: C^∞(M) → ℝ, 满足Leibniz法则
v(fg) = f(p)v(g) + g(p)v(f)
```

### 3.2 切丛结构图

```text
        [切丛 TM]
           │
    π: TM → M (投影)
           │
    ┌──────┴──────┐
    │             │
[切空间T_pM]   [整体结构]
    │             │
n维向量空间   2n维流形
    │             │
基: ∂/∂x^i     局部平凡化
    │          TU ≃ U×ℝⁿ
    │             │
    └──────┬──────┘
           │
    [向量场 X: M → TM]
    满足π∘X = id
           │
    局部: X = X^i ∂/∂x^i
```

### 3.3 向量场与Lie括号

**定义**：向量场X, Y的Lie括号[X,Y]定义为：

```
[X,Y]f = X(Yf) - Y(Xf)
```

**性质**：

- 双线性
- 反对称：[X,Y] = -[Y,X]
- Jacobi恒等式：[[X,Y],Z] + [[Y,Z],X] + [[Z,X],Y] = 0

## 🔗 四、联络与曲率

### 4.1 联络定义

**定义**：向量丛E上的联络∇是映射：

```
∇: Γ(TM) × Γ(E) → Γ(E)
(X, s) ↦ ∇_X s
```

满足：

1. C^∞(M)-线性于X
2. ℝ-线性于s
3. Leibniz法则：∇_X(fs) = (Xf)s + f∇_X s

### 4.2 Levi-Civita联络

**定理**：黎曼流形(M,g)上存在唯一的联络∇（Levi-Civita联络）满足：

1. **无挠**：∇_X Y - ∇_Y X = [X,Y]
2. **度量相容**：X⟨Y,Z⟩ = ⟨∇_X Y, Z⟩ + ⟨Y, ∇_X Z⟩

### 4.3 联络构造图

```text
        [Levi-Civita联络构造]
                │
    ┌───────────┴───────────┐
    │                       │
[度量g]                 [无挠条件]
    │                       │
⟨∂_i, ∂_j⟩ = g_ij      ∇_{∂_i}∂_j = ∇_{∂_j}∂_i
    │                       │
    └───────────┬───────────┘
                │
        [Christoffel符号]
                │
    Γ^k_ij = (1/2)g^{kl}(∂_i g_{jl} + ∂_j g_{il} - ∂_l g_{ij})
                │
        [联络公式]
                │
    ∇_{∂_i} ∂_j = Γ^k_{ij} ∂_k
```

### 4.4 曲率张量

**定义**：Riemann曲率张量：

```
R(X,Y)Z = ∇_X ∇_Y Z - ∇_Y ∇_X Z - ∇_{[X,Y]} Z
```

**分量形式**：

```
R^l_{ijk} = ∂_i Γ^l_{jk} - ∂_j Γ^l_{ik} + Γ^l_{im}Γ^m_{jk} - Γ^l_{jm}Γ^m_{ik}
```

### 4.5 曲率收缩

```text
        [曲率张量层次]
              │
    ┌─────────┼─────────┐
    │         │         │
[Riemann]  [Ricci]   [标量]
R^l_{ijk}  R_{ij}     R
(4,0)张量  (2,0)张量  函数
    │         │         │
完整信息   收缩        全收缩
           R_{ij}=R^k_{ikj}  R=g^{ij}R_{ij}
    │         │         │
    └─────────┴─────────┘
              │
        [几何意义]
              │
├─Riemann: 平行移动的路径相关性
├─Ricci: "平均"曲率
└─R: 总曲率
```

## 🌟 五、黎曼几何核心定理

### 5.1 Gauss-Bonnet定理

**定理（曲面）**：设M是紧致定向曲面，则：

```
∫_M K dA = 2πχ(M)
```

其中K是Gauss曲率，χ(M)是Euler示性数。

**证明框架**：

```text
        [Gauss-Bonnet证明]
                │
    ┌───────────┴───────────┐
    │                       │
[局部公式]              [整体公式]
    │                       │
三角形内角和           三角剖分
= π + ∫K dA            Σ(内角和-π)
    │                       │
角盈余 = 曲率积分       组合公式
    │                       │
    └───────────┬───────────┘
                │
        ∫K dA = 2π(V-E+F) = 2πχ(M) ∎
```

### 5.2 比较定理

**Bonnet-Myers定理**：若Ricci ≥ (n-1)K > 0，则diam(M) ≤ π/√K，特别地M紧致。

**Cartan-Hadamard定理**：若M完备单连通且截面曲率K ≤ 0，则exp_p: T_pM → M是微分同胚。

### 5.3 比较定理图

```text
        [曲率比较定理]
              │
    ┌─────────┴─────────┐
    │                   │
[正曲率]            [负曲率]
    │                   │
├─Bonnet-Myers      ├─Cartan-Hadamard
│ diam有界          │ 无共轭点
├─Synge             ├─负曲率流形
│ 偶维正曲率        │ 万有覆盖=ℝⁿ
│ ⟹ 单连通        │
├─测地线会聚        ├─测地线发散
```

### 5.4 Hodge理论

**定理（Hodge分解）**：紧致黎曼流形上：

```
Ω^k(M) = H^k(M) ⊕ dΩ^{k-1}(M) ⊕ δΩ^{k+1}(M)
```

其中H^k是调和形式空间。

**推论**：H^k(M) ≅ H^k_{dR}(M)（调和形式表示de Rham上同调）

## 💻 七、技术实现

### 7.1 Lean 4 形式化

```lean
-- 微分几何基础形式化（Mathlib风格）
import Mathlib.Geometry.Manifold.SmoothManifold

-- 光滑流形
structure SmoothManifold (n : ℕ) where
  carrier : Type*
  [topology : TopologicalSpace carrier]
  [manifold : ChartedSpace (EuclideanSpace ℝ (Fin n)) carrier]
  [smooth : SmoothManifoldWithCorners (𝓡 n) carrier]

-- 切向量
def TangentVector (M : SmoothManifold n) (p : M.carrier) :=
  TangentSpace (𝓡 n) p

-- Levi-Civita联络(简化)
structure LeviCivitaConnection (M : SmoothManifold n) where
  metric : RiemannianMetric M
  connection : Connection (TangentBundle M)
  torsion_free : ∀ X Y, connection.covariantDerivative X Y -
                        connection.covariantDerivative Y X =
                        LieBracket X Y
  metric_compatible : ∀ X Y Z, X (metric Y Z) =
                              metric (connection.covariantDerivative X Y) Z +
                              metric Y (connection.covariantDerivative X Z)
```

### 7.2 Python 数值计算

```python
import numpy as np
from scipy.linalg import eigh

def christoffel_symbols(g, dg, n):
    """
    计算Christoffel符号
    g: 度量张量 g_ij
    dg: 度量导数 ∂_k g_ij
    """
    g_inv = np.linalg.inv(g)
    Gamma = np.zeros((n, n, n))

    for k in range(n):
        for i in range(n):
            for j in range(n):
                for l in range(n):
                    Gamma[k,i,j] += 0.5 * g_inv[k,l] * (
                        dg[i,j,l] + dg[j,i,l] - dg[l,i,j]
                    )
    return Gamma

def riemann_curvature(Gamma, dGamma, n):
    """
    计算Riemann曲率张量
    R^l_ijk = ∂_i Γ^l_jk - ∂_j Γ^l_ik + Γ^l_im Γ^m_jk - Γ^l_jm Γ^m_ik
    """
    R = np.zeros((n, n, n, n))

    for l in range(n):
        for i in range(n):
            for j in range(n):
                for k in range(n):
                    R[l,i,j,k] = dGamma[l,j,k,i] - dGamma[l,i,k,j]
                    for m in range(n):
                        R[l,i,j,k] += Gamma[l,i,m]*Gamma[m,j,k] - Gamma[l,j,m]*Gamma[m,i,k]
    return R

# 球面S²的Gauss曲率
def sphere_curvature(R):
    """球面半径R的Gauss曲率为1/R²"""
    return 1.0 / (R**2)

print(f"单位球面S²的Gauss曲率: K = {sphere_curvature(1.0)}")
```
