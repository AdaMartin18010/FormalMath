## 引用格式标准

本项目采用 **AMS (美国数学学会)** 引用格式，数学领域的国际标准。

### 文内引用格式

```latex
% 单篇引用
This result was proved by Smith [12].

% 多篇引用
This has been studied extensively [3, 7, 12].

% 连续编号
For more details, see [3, 7-10, 12].

% 引用具体定理
The theorem [12, Thm. 3.1] states that...
```

### 参考文献列表格式

**书籍**:
```
[1] S. Lang, Algebra, 3rd ed., Springer, 2002. ISBN: 978-0-387-95385-4
```

**期刊文章**:
```
[2] A. Wiles, Modular elliptic curves and Fermat's Last Theorem,
    Ann. of Math. (2) 141 (1995), no. 3, 443-551. DOI: 10.2307/2118559

```

    arXiv:math/0309380 [math.NT], 2003.
