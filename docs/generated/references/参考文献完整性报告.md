## 3. 引用格式标准化

### 3.1 采用标准

本项目采用 **AMS (美国数学学会) 引用格式**，数学领域的国际标准。

### 3.2 书籍引用格式

```
[编号] 作者, 书名, 版次, 出版社, 出版地, 年份.
        ↑                                    ↑
        └ 作者名: 名 姓 格式                 └ ISBN: 可选

示例:
[1] M. Artin, Algebra, 2nd ed., Pearson, Boston, 2010.
    ISBN: 978-0-13-241377-0
```

### 3.3 期刊文章引用格式

```
[编号] 作者, 文章标题, 期刊名 卷号 (年份), 期号, 页码范围.
        ↑                                           ↑
        └ 标题中数学符号用LaTeX                     └ DOI: 必须

示例:
[2] A. Wiles, Modular elliptic curves and Fermat's Last Theorem,
    Ann. of Math. (2) 141 (1995), no. 3, 443-551.
    DOI: 10.2307/2118559
```

### 3.4 预印本引用格式

```
[编号] 作者, 文章标题, arXiv:编号 [分类], 年份.

示例:
[3] T. Tao, The prime number theorem in arithmetic progressions,
    arXiv:math/0309380 [math.NT], 2003.
```

### 3.5 在线资源引用格式

```
[编号] 作者/机构, 资源名称, URL, 访问日期.

示例:
[4] The Stacks Project Authors, The Stacks Project,
    https://stacks.math.columbia.edu[需更新[需更新]], accessed April 2026.
```

## 8. 使用指南

### 8.1 如何引用数据库中的文献

**LaTeX用法**:
```latex
\documentclass{article}
\bibliographystyle{amsplain}

\begin{document}
As shown by \cite{rudin1976}, ...

\bibliography{FormalMath-References}
\end{document}
```

**Markdown用法**:
```markdown
根据Rudin的经典教材[1]，我们可以...

## 参考文献

[1] W. Rudin, Principles of Mathematical Analysis, 3rd ed., 
    McGraw-Hill, 1976. ISBN: 978-0-07-054235-8
```

### 8.2 如何添加新文献

1. 在 `FormalMath-References.bib` 中添加BibTeX条目
2. 遵循现有格式规范
3. 确保包含DOI或ISBN
4. 更新本报告中的相关表格

### 8.3 文献选择建议

**初学者**: 优先选择带⭐⭐⭐⭐⭐的入门教材  
**进阶学习**: 参考"必读经典"和"进阶专著"  
**研究前沿**: 关注里程碑论文和arXiv预印本  
**历史背景**: 查阅"历史文献"分类

### 9.1 引用检查清单

- [ ] 所有引用都有对应的参考文献条目
- [ ] 作者姓名格式一致 (名 姓)
- [ ] 书名/文章名使用斜体
- [ ] 数学符号用LaTeX格式保护
- [ ] 包含DOI或ISBN (如适用)
- [ ] 在线资源注明访问日期
- [ ] 引用编号按出现顺序排列
- [ ] 无孤立引用 (正文中未引用的条目)

### 9.2 常见问题解答

**Q1: 应该引用哪个版本？**  
A: 优先引用最新版，但如果经典版本有特殊意义也可引用。

**Q2: 如何处理多作者？**  
A: 列出所有作者，除非超过6位，此时可用"et al."。

**Q3: 中文文献如何处理？**  
A: 使用英文翻译标题，注明"(in Chinese)"。

**Q4: 个人笔记能否引用？**  
A: 不能。只引用已发表或公开可用的文献。
