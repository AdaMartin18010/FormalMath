## BibTeX 条目模板

### 书籍

```bibtex
@book{key2020,
  author    = {First Last and First2 Last2},
  title     = {Book Title},
  edition   = {nth},
  publisher = {Publisher},
  address   = {City},
  year      = {2020},
  isbn      = {978-X-XXX-XXXXX-X}
}
```

### 期刊文章

```bibtex
@article{key2020,
  author  = {First Last},
  title   = {Article Title},
  journal = {Journal Name},
  volume  = {42},
  number  = {3},
  pages   = {123--145},
  year    = {2020},
  doi     = {10.XXXX/XXXXX}
}
```

### 预印本

```bibtex
@unpublished{key2020,
  author = {First Last},
  title  = {Paper Title},
  year   = {2020},
  note   = {arXiv:2001.12345 [math.XX]}
}
```

### 在线资源

```bibtex
@misc{key2020,
  author       = {{Author or Institution}},
  title        = {Resource Title},
  howpublished = {\url{https://example.com}},
  year         = {2020},
  note         = {Accessed: YYYY-MM-DD}
}
```

### 会议论文

```bibtex
@inproceedings{key2020,
  author    = {First Last},
  title     = {Paper Title},
  booktitle = {Proceedings of Conference Name},
  year      = {2020},
  pages     = {100--110},
  doi       = {10.XXXX/XXXXX}
}
```

### 博士论文

```bibtex
@phdthesis{key2020,
  author = {First Last},
  title  = {Thesis Title},
  school = {University Name},
  year   = {2020}
}
```

## 引用检查清单

- [ ] 所有引用都有对应的参考文献条目
- [ ] 作者姓名格式一致 (名 姓)
- [ ] 书名使用斜体 (`*text*`)
- [ ] 期刊名使用标准缩写
- [ ] 包含DOI或ISBN (如适用)
- [ ] 在线资源注明访问日期
- [ ] 页码范围使用双横线 (`--`)
- [ ] 数学符号用LaTeX格式保护 (`$...$`)
- [ ] 引用编号按出现顺序排列
- [ ] 无孤立引用 (正文中未引用的条目)
