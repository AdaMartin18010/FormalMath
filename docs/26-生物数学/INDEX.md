## 学习路径

### 基础路径
```
微分方程 → 种群模型 → 稳定性分析 → 分岔理论
```

### 进阶路径
```
动力系统 → 反应扩散方程 → 模式形成 → 网络动力学
```

### 应用路径
- **流行病学**: 传染病模型 + 网络分析 + 数据拟合
- **生态学**: 种群模型 + 竞争理论 + 保护生物学
- **神经科学**: 单神经元模型 + 神经网络 + 脑动力学

## 核心模型速查

| 模型 | 方程 | 应用领域 |
|------|------|----------|
| **Logistic** | $\frac{dN}{dt} = rN(1-\frac{N}{K})$ | 种群增长 |
| **Lotka-Volterra** | $\frac{dx}{dt} = ax - bxy$ | 捕食者-猎物 |
| **SIR** | $\frac{dS}{dt} = -\beta SI$ | 传染病 |
| **Hodgkin-Huxley** | $C\frac{dV}{dt} = -I_{ion} + I_{ext}$ | 神经传导 |
| **FitzHugh-Nagumo** | $\frac{dv}{dt} = v - v^3/3 - w + I$ | 神经简化 |

## 分析方法

| 方法 | 描述 | 适用场景 |
|------|------|----------|
| **线性稳定性** | 特征值分析 | 平衡点稳定性 |
| **分岔分析** | 参数变化引起定性改变 | 系统动态变化 |
| **相平面分析** | 2D可视化 | 二维系统 |
| **数值积分** | ODE求解器 | 复杂系统 |
