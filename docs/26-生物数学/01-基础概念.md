## 一、种群动力学

### 1.1 单种群模型

**定义 1.1** (指数增长模型)
无资源限制的种群增长：

$$\frac{dN}{dt} = rN$$

其中 $N(t)$ 是种群数量，$r$ 是内禀增长率。

解：$N(t) = N_0 e^{rt}$

**定义 1.2** (Logistic增长模型)
考虑环境承载力 $K$ 的种群增长：

$$\frac{dN}{dt} = rN\left(1 - \frac{N}{K}\right)$$

其中 $K$ 是环境承载力。

**性质**:

- 当 $N < K$ 时，种群增长
- 当 $N > K$ 时，种群减少
- 平衡点：$N^* = 0$（不稳定），$N^* = K$（稳定）
- 解：$N(t) = \frac{K}{1 + \left(\frac{K}{N_0} - 1\right)e^{-rt}}$

**Allee效应**: 种群密度过低时增长率下降：

$$\frac{dN}{dt} = rN\left(\frac{N}{A} - 1\right)\left(1 - \frac{N}{K}\right)$$

其中 $A$ 是Allee阈值（$A < K$）。

### 1.2 Lotka-Volterra竞争模型

**定义 1.3** (竞争模型)
两个物种竞争相同资源：

$$\frac{dN_1}{dt} = r_1 N_1\left(1 - \frac{N_1 + \alpha_{12}N_2}{K_1}\right)$$

$$\frac{dN_2}{dt} = r_2 N_2\left(1 - \frac{N_2 + \alpha_{21}N_1}{K_2}\right)$$

其中：
- $\alpha_{12}$：物种2对物种1的竞争系数
- $\alpha_{21}$：物种1对物种2的竞争系数

**平衡点分析**:

1. **灭绝平衡点**: $(0, 0)$（不稳定）
2. **物种1独占**: $(K_1, 0)$
3. **物种2独占**: $(0, K_2)$
4. **共存平衡点**: 解方程组

**共存条件**: 当且仅当
$$\alpha_{12} < \frac{K_1}{K_2} \quad \text{且} \quad \alpha_{21} < \frac{K_2}{K_1}$$

### 1.3 Lotka-Volterra捕食-被捕食模型

**定义 1.4** (捕食-被捕食模型)

$$\frac{dN}{dt} = rN - aNP = N(r - aP)$$

$$\frac{dP}{dt} = caNP - dP = P(caN - d)$$

其中：
- $N$：被捕食者（猎物）数量
- $P$：捕食者数量
- $r$：猎物内禀增长率
- $a$：捕食率
- $c$：转化效率
- $d$：捕食者死亡率

**平衡点**:

1. **平凡平衡点**: $(0, 0)$（不稳定）
2. **猎物无捕食者**: $(K, 0)$（若加入承载力）
3. **共存平衡点**: $(N^*, P^*) = \left(\frac{d}{ca}, \frac{r}{a}\right)$

**性质**:

- 系统具有守恒量（第一积分）
- 解形成闭合轨道（周期振荡）
- 振幅取决于初始条件

## 二、流行病模型

### 2.1 SIR模型

**定义 2.1** (SIR模型)
将人群分为三类：

- **S** (Susceptible)：易感者
- **I** (Infected)：感染者
- **R** (Recovered/Removed)：康复/移除者

动力学方程：

$$\frac{dS}{dt} = -\beta SI$$

$$\frac{dI}{dt} = \beta SI - \gamma I$$

$$\frac{dR}{dt} = \gamma I$$

其中：
- $\beta$：传染率
- $\gamma$：恢复率
- 总人口 $N = S + I + R$（常数）

### 2.2 基本再生数

**定义 2.2** (基本再生数 $R_0$)
一个感染者在完全易感人群中平均产生的二代感染数：

$$R_0 = \frac{\beta N}{\gamma}$$

**流行病学意义**:

- $R_0 < 1$：疾病消亡
- $R_0 > 1$：疾病流行
- $R_0 = 1$：临界阈值

**最终规模方程**:

设最终感染比例为 $R_\infty$，则：

$$R_\infty = 1 - e^{-R_0 R_\infty}$$

### 2.3 SEIR模型

**定义 2.3** (SEIR模型)
加入潜伏者类别E（Exposed）：

$$\frac{dS}{dt} = -\beta SI$$

$$\frac{dE}{dt} = \beta SI - \sigma E$$

$$\frac{dI}{dt} = \sigma E - \gamma I$$

$$\frac{dR}{dt} = \gamma I$$

其中 $\sigma$ 是潜伏期到发病期的转化率。

**潜伏期效应**: 增加了系统的时间延迟，影响疫情发展的时序。

### 2.4 SIS模型

**定义 2.4** (SIS模型)
无免疫的传染病（如普通感冒）：

$$\frac{dS}{dt} = -\beta SI + \gamma I$$

$$\frac{dI}{dt} = \beta SI - \gamma I$$

注意 $S + I = N$（常数）。

**平衡分析**:

- 无病平衡点：$I^* = 0$
- 地方病平衡点：$I^* = N(1 - 1/R_0)$（当 $R_0 > 1$ 时）

## 三、进化博弈论

### 3.1 复制者方程

**定义 3.1** (复制者方程)
描述策略频率演化的动力学：

$$\frac{dx_i}{dt} = x_i(f_i - \bar{f})$$

其中：
- $x_i$：策略 $i$ 的频率（$\sum_i x_i = 1$）
- $f_i$：策略 $i$ 的适应度
- $\bar{f} = \sum_j x_j f_j$：平均适应度

**性质**:

- 频率演化：适应度高于平均的策略增长，低于平均的减少
- 单态平衡点：纯策略（$x_i = 1$）
- 多态平衡点：混合策略

### 3.2 博弈矩阵

**定义 3.2** (支付矩阵)
两人对称博弈的支付矩阵 $A = (a_{ij})$：

|  | 策略1 | 策略2 |
|--|-------|-------|
| **策略1** | $a_{11}$ | $a_{12}$ |
| **策略2** | $a_{21}$ | $a_{22}$ |

适应度计算：

$$f_i = \sum_j a_{ij} x_j = (Ax)_i$$

### 3.3 进化稳定策略(ESS)

**定义 3.3** (ESS)
策略 $x^*$ 是进化稳定的，如果对任意 $y \neq x^*$：

1. $x^* \cdot Ax^* \geq y \cdot Ax^*$（纳什均衡条件）
2. 若等号成立，则 $x^* \cdot Ay > y \cdot Ay$

**等价定义**: $x^*$ 是ESS如果存在 $\epsilon_0 > 0$ 使得对所有 $0 < \epsilon < \epsilon_0$：

$$x^* \cdot A((1-\epsilon)x^* + \epsilon y) > y \cdot A((1-\epsilon)x^* + \epsilon y)$$

即小群体入侵者不能获得更高收益。

### 3.4 鹰鸽博弈

**经典例子**: 资源价值 $V$，冲突成本 $C$

|  | 鹰(H) | 鸽(D) |
|--|-------|-------|
| **鹰** | $(V-C)/2$ | $V$ |
| **鸽** | $0$ | $V/2$ |

**分析**:

- 若 $V > C$：鹰是ESS（纯策略）
- 若 $V < C$：混合ESS，鹰的频率 $p^* = V/C$

**复制者动态**:

$$\frac{dp}{dt} = p(1-p)[f_H - f_D]$$

其中 $p$ 是鹰策略频率。

## 总结

本章节介绍了生物数学的基础概念：

- **种群动力学**: 单种群增长模型、种间竞争、捕食-被捕食关系
- **流行病模型**: SIR、SEIR、SIS模型、基本再生数 $R_0$
- **进化博弈论**: 复制者方程、ESS、鹰鸽博弈

这些模型为理解生物系统的动态行为提供了数学框架。
