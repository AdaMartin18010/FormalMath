## 一、Lotka-Volterra稳定性

### 1.1 竞争模型的稳定性

**定理 1.1** (Lotka-Volterra竞争稳定性)
考虑竞争模型：

$$\frac{dN_1}{dt} = r_1 N_1\left(1 - \frac{N_1 + \alpha_{12}N_2}{K_1}\right)$$

$$\frac{dN_2}{dt} = r_2 N_2\left(1 - \frac{N_2 + \alpha_{21}N_1}{K_2}\right)$$

**四种情况**:

**情况I**：$\alpha_{12} < K_1/K_2$ 且 $\alpha_{21} < K_2/K_1$
- 共存平衡点 $(N_1^*, N_2^*)$ 局部渐近稳定
- 其中 $N_1^* = \frac{K_1 - \alpha_{12}K_2}{1 - \alpha_{12}\alpha_{21}}$，$N_2^* = \frac{K_2 - \alpha_{21}K_1}{1 - \alpha_{12}\alpha_{21}}$

**情况II**：$\alpha_{12} > K_1/K_2$ 且 $\alpha_{21} > K_2/K_1$
- 共存平衡点不稳定（鞍点）
- 两个边界平衡点 $(K_1, 0)$ 和 $(0, K_2)$ 都是局部稳定
- 系统收敛取决于初始条件（双稳态）

**情况III**：$\alpha_{12} < K_1/K_2$ 且 $\alpha_{21} > K_2/K_1$
- $(K_1, 0)$ 稳定，$(0, K_2)$ 不稳定
- 物种1竞争排除物种2

**情况IV**：$\alpha_{12} > K_1/K_2$ 且 $\alpha_{21} < K_2/K_1$
- $(0, K_2)$ 稳定，$(K_1, 0)$ 不稳定
- 物种2竞争排除物种1

**证明**（情况I）：

在平衡点 $(N_1^*, N_2^*)$ 线性化，Jacobian矩阵：

$$J = \begin{pmatrix} -\frac{r_1 N_1^*}{K_1} & -\frac{r_1 \alpha_{12} N_1^*}{K_1} \\ -\frac{r_2 \alpha_{21} N_2^*}{K_2} & -\frac{r_2 N_2^*}{K_2} \end{pmatrix}$$

迹：$\text{tr}(J) = -\frac{r_1 N_1^*}{K_1} - \frac{r_2 N_2^*}{K_2} < 0$

行列式：$\det(J) = \frac{r_1 r_2 N_1^* N_2^*}{K_1 K_2}(1 - \alpha_{12}\alpha_{21}) > 0$（当情况I条件满足）

因此特征值实部均为负，平衡点渐近稳定。∎

### 1.2 捕食-被捕食模型的周期解

**定理 1.2** (Lotka-Volterra振荡定理)
经典的Lotka-Volterra捕食-被捕食模型：

$$\frac{dN}{dt} = N(r - aP)$$

$$\frac{dP}{dt} = P(caN - d)$$

具有守恒量：

$$H(N, P) = cN - d\ln N + aP - r\ln P$$

所有非平衡点的轨线都是围绕平衡点 $(N^*, P^*) = (d/ca, r/a)$ 的闭合曲线。

**证明**:

验证 $dH/dt = 0$：

$$\frac{dH}{dt} = \left(c - \frac{d}{N}\right)\frac{dN}{dt} + \left(a - \frac{r}{P}\right)\frac{dP}{dt}$$

$$= \left(c - \frac{d}{N}\right)N(r - aP) + \left(a - \frac{r}{P}\right)P(caN - d)$$

$$= (cN - d)(r - aP) + (aP - r)(caN - d)$$

$$= (cN - d)(r - aP) - (r - aP)(caN - d) = 0$$

因此 $H(N, P) = \text{const}$，轨线沿等高线运动。由于等高线是闭合曲线（可验证），解是周期性的。∎

## 二、繁殖者方程收敛性

### 2.1 复制者动态的稳定性

**定理 2.1** (Nash均衡与平衡点)
在复制者方程中：

$$\frac{dx_i}{dt} = x_i(f_i - \bar{f})$$

**性质**:

1. 混合策略纳什均衡对应内部平衡点
2. 纯策略纳什均衡对应边界平衡点
3. ESS对应渐近稳定平衡点

**定理 2.2** (Lyapunov函数)
对于具有ESS $x^*$ 的对称博弈，函数：

$$V(x) = \sum_i x_i^* \ln\frac{x_i^*}{x_i}$$

是复制者动态的严格Lyapunov函数，即 $dV/dt < 0$ 对所有 $x \neq x^*$。

**证明概要**:

计算：

$$\frac{dV}{dt} = -\sum_i \frac{x_i^*}{x_i}\frac{dx_i}{dt} = -\sum_i \frac{x_i^*}{x_i} x_i(f_i - \bar{f})$$

$$= -\sum_i x_i^* f_i + \bar{f}\sum_i x_i^* = \bar{f} - x^* \cdot f$$

由于 $x^*$ 是ESS，可以证明 $x^* \cdot f > \bar{f}$ 对所有 $x \neq x^*$，因此 $dV/dt < 0$。∎

### 2.2 进化稳定策略的稳定性

**定理 2.3** (ESS与渐近稳定性)
在对称矩阵博弈中，ESS是渐近稳定的。

反之不成立：存在渐近稳定但不是ESS的策略。

**证明**: 由定理2.2的Lyapunov函数直接可得。

## 三、基本再生数阈值定理

### 3.1 SIR模型的阈值行为

**定理 3.1** (SIR阈值定理)
在SIR模型中：

$$\frac{dS}{dt} = -\beta SI, \quad \frac{dI}{dt} = \beta SI - \gamma I, \quad \frac{dR}{dt} = \gamma I$$

设 $R_0 = \beta S_0/\gamma$ 是基本再生数。

**结论**:

1. 若 $R_0 \leq 1$，$I(t)$ 单调递减至0，无流行病发生
2. 若 $R_0 > 1$，$I(t)$ 先增后减，发生流行病

峰值感染量 $I_{max}$ 满足：

$$S_{critical} = \frac{\gamma}{\beta} = \frac{S_0}{R_0}$$

当 $S$ 降至 $S_{critical}$ 时，$I$ 达到最大值。

**证明**:

由 $dI/dt = I(\beta S - \gamma)$，$I$ 的增长取决于 $\beta S - \gamma$ 的符号。

当 $S > \gamma/\beta = S_0/R_0$ 时，$dI/dt > 0$，感染增加。

当 $S < S_0/R_0$ 时，$dI/dt < 0$，感染减少。

由于 $S$ 单调递减，若 $S_0 \leq S_0/R_0$（即 $R_0 \leq 1$），则 $dI/dt < 0$ 始终成立。

若 $R_0 > 1$，则 $S_0 > S_{critical}$，$I$ 先增后减。∎

### 3.2 最终规模方程

**定理 3.2** (最终规模定理)
在SIR模型中，设 $S_0 \approx N$（初始几乎全易感），最终感染比例 $R_\infty = R(\infty)/N$ 满足：

$$R_\infty = 1 - e^{-R_0 R_\infty}$$

**证明**:

由 $dS/dR = -S/(\gamma I) \cdot \gamma I = -S/(R_0 N)$（利用 $\beta = R_0 \gamma/N$）

积分：$\ln S - \ln S_0 = -(R - R_0)/(R_0 N)$

当 $t \to \infty$，$S_\infty = S_0 e^{-R_\infty/R_0}$（假设 $R_0 = 0$）

由于 $N = S_\infty + R_\infty$（$I_\infty = 0$）：

$$R_\infty = N - S_0 e^{-R_\infty/N \cdot R_0} \approx N(1 - e^{-R_0 R_\infty/N})$$

归一化得最终规模方程。∎

## 四、Kolmogorov捕食系统

### 4.1 一般捕食系统

**定义 4.1** (Kolmogorov系统)
一般二维捕食系统：

$$\frac{dx}{dt} = xf(x, y)$$

$$\frac{dY}{dt} = yg(x, y)$$

其中：
- $x$：猎物种群
- $y$：捕食者种群
- $\partial f/\partial y < 0$（捕食者对猎物不利）
- $\partial g/\partial x > 0$（猎物对捕食者有利）

### 4.2 极限环存在性

**定理 4.1** (Kolmogorov极限环定理)
在一定条件下（猎物有承载力、功能反应函数为Holling Type II型），Kolmogorov系统存在稳定的极限环。

**条件**:

1. $f(x, y)$ 对 $y$ 递减
2. $g(x, y)$ 对 $x$ 递增
3. 猎物等倾线 $f(x, y) = 0$ 为驼峰形
4. 捕食者等倾线 $g(x, y) = 0$ 单调递增

**Holling Type II功能反应**:

$$p(x) = \frac{ax}{1 + ahx}$$

其中 $a$ 是攻击率，$h$ 是处理时间。

**生物学意义**: 捕食者饱和效应导致系统产生稳定振荡，而非Lotka-Volterra型的中性稳定。

## 总结

本章节介绍了生物数学的核心定理：

- **Lotka-Volterra稳定性**: 竞争模型的四种情况、捕食模型的周期解
- **繁殖者方程**: ESS与渐近稳定性、Lyapunov函数
- **阈值定理**: 基本再生数决定疾病动力学、最终规模方程
- **Kolmogorov系统**: 一般捕食系统的极限环存在性

这些定理揭示了生物系统动力学的深层数学结构。
