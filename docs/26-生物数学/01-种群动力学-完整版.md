## 1. 问题建模

### 1.1 单种种群模型

**问题**: 描述种群数量随时间的演变规律，考虑资源限制。

**Malthus模型** (指数增长):
$$\frac{dN}{dt} = rN$$
解: $N(t) = N_0 e^{rt}$

**局限性**: 无限增长不符合现实

**Logistic模型** (资源限制):
$$\frac{dN}{dt} = rN\left(1 - \frac{N}{K}\right)$$

其中:
- $r$: 内禀增长率
- $K$: 环境承载力
- $N/K$: 拥挤效应

**解**:
$$N(t) = \frac{K}{1 + \left(\frac{K}{N_0} - 1\right)e^{-rt}}$$

### 1.2 捕食者-猎物模型

**Lotka-Volterra模型**:

设 $x(t)$: 猎物数量, $y(t)$: 捕食者数量

$$\begin{aligned}
\frac{dx}{dt} &= ax - bxy = x(a - by) \\
\frac{dy}{dt} &= -cy + dxy = y(-c + dx)
\end{aligned}$$

**生物学意义**:
- $ax$: 猎物自然增长
- $-bxy$: 捕食造成的猎物死亡
- $-cy$: 捕食者自然死亡
- $dxy$: 捕食转化为捕食者增长

## 2. 数学理论

### 2.1 Logistic模型的分析

**定理 2.1** (Logistic模型解的存在唯一性): 对任意 $N_0 > 0$，存在唯一解 $N(t) > 0$ 对所有 $t \geq 0$。

**平衡点分析**:
- $N^* = 0$ (不稳定)
- $N^* = K$ (渐近稳定)

**证明**: 线性化，Jacobian $J = r(1 - 2N/K)$
- 在 $N=0$: $J = r > 0$，不稳定
- 在 $N=K$: $J = -r < 0$，稳定

### 2.2 Lotka-Volterra系统分析

**平衡点**:
- $E_0 = (0, 0)$: 灭绝平衡点
- $E_1 = (c/d, a/b)$: 共存平衡点

**定理 2.2** (Lotka-Volterra周期轨道): 系统存在一族周期轨道环绕平衡点 $E_1$。

**首次积分**:
$$H(x, y) = dx - c\ln x + by - a\ln y = \text{const}$$

**平均种群数量**:
$$\bar{x} = \frac{1}{T}\int_0^T x(t)dt = \frac{c}{d}, \quad \bar{y} = \frac{a}{b}$$

### 2.3 竞争模型

**两种群竞争模型**:
$$\begin{aligned}
\frac{dx}{dt} &= r_1 x\left(1 - \frac{x + \alpha_{12}y}{K_1}\right) \\
\frac{dy}{dt} &= r_2 y\left(1 - \frac{y + \alpha_{21}x}{K_2}\right)
\end{aligned}$$

**竞争结果分类**:

| 条件 | 结果 |
|------|------|
| $\alpha_{12} < K_1/K_2$ 且 $\alpha_{21} < K_2/K_1$ | 稳定共存 |
| $\alpha_{12} > K_1/K_2$ 且 $\alpha_{21} < K_2/K_1$ | x竞争排除y |
| $\alpha_{12} < K_1/K_2$ 且 $\alpha_{21} > K_2/K_1$ | y竞争排除x |
| $\alpha_{12} > K_1/K_2$ 且 $\alpha_{21} > K_2/K_1$ | 双稳态 |

## 3. 算法与方法

### 3.1 数值求解

**RK4方法** (四阶Runge-Kutta):
```python
def rk4(f, y0, t_span, h=0.01):
    """
    四阶Runge-Kutta方法
    """
    t = np.arange(t_span[0], t_span[1], h)
    y = np.zeros((len(t), len(y0)))
    y[0] = y0
    
    for i in range(len(t)-1):
        k1 = f(t[i], y[i])
        k2 = f(t[i] + h/2, y[i] + h*k1/2)
        k3 = f(t[i] + h/2, y[i] + h*k2/2)
        k4 = f(t[i] + h, y[i] + h*k3)
        y[i+1] = y[i] + h*(k1 + 2*k2 + 2*k3 + k4)/6
    
    return t, y
```

### 3.2 参数估计

**最小二乘拟合**:
$$\min_{\theta} \sum_{i=1}^n (N(t_i; \theta) - N_i^{obs})^2$$

**方法**: Levenberg-Marquardt算法

## 5. 代码实现

### 5.1 Logistic模型

```python
import numpy as np
from scipy.integrate import odeint
import matplotlib.pyplot as plt

def logistic(N, t, r, K):
    """
    Logistic增长模型
    dN/dt = r*N*(1 - N/K)
    """
    return r * N * (1 - N / K)

# 参数
r = 0.5      # 内禀增长率
K = 100      # 承载力
N0 = 10      # 初始种群

# 时间
T = np.linspace(0, 20, 1000)

# 求解
N = odeint(logistic, N0, T, args=(r, K))

# 解析解
N_analytic = K / (1 + (K/N0 - 1) * np.exp(-r * T))

# 绘图
plt.figure(figsize=(10, 6))
plt.plot(T, N, 'b-', linewidth=2, label='Numerical')
plt.plot(T, N_analytic, 'r--', linewidth=2, label='Analytic')
plt.axhline(y=K, color='g', linestyle=':', label=f'K={K}')
plt.xlabel('Time')
plt.ylabel('Population N(t)')
plt.title(f'Logistic Growth (r={r}, K={K})')
plt.legend()
plt.grid(True)
plt.savefig('logistic_growth.png', dpi=150)
plt.show()

print(f"稳态种群: {N[-1][0]:.2f}")
print(f"理论承载力: {K}")
```

### 5.2 Lotka-Volterra系统

```python
def lotka_volterra(Z, t, a, b, c, d):
    """
    Lotka-Volterra捕食者-猎物模型
    """
    x, y = Z
    dxdt = a*x - b*x*y
    dydt = -c*y + d*x*y
    return [dxdt, dydt]

# 参数 (兔-狐模型)
a, b = 1.0, 0.1    # 猎物参数
c, d = 0.5, 0.02   # 捕食者参数

# 初始条件
x0, y0 = 20, 5    # 初始猎物和捕食者数量

# 时间
T = np.linspace(0, 50, 5000)

# 求解
solution = odeint(lotka_volterra, [x0, y0], T, args=(a, b, c, d))
x, y = solution.T

# 绘制时间序列
fig, axes = plt.subplots(1, 2, figsize=(14, 5))

# 时间序列
axes[0].plot(T, x, 'b-', label='Prey (x)', linewidth=2)
axes[0].plot(T, y, 'r-', label='Predator (y)', linewidth=2)
axes[0].set_xlabel('Time')
axes[0].set_ylabel('Population')
axes[0].set_title('Lotka-Volterra Time Series')
axes[0].legend()
axes[0].grid(True)

# 相平面图
axes[1].plot(x, y, 'g-', linewidth=1.5)
axes[1].plot(c/d, a/b, 'ro', markersize=8, label='Equilibrium')
axes[1].set_xlabel('Prey (x)')
axes[1].set_ylabel('Predator (y)')
axes[1].set_title('Phase Portrait')
axes[1].legend()
axes[1].grid(True)

plt.tight_layout()
plt.savefig('lotka_volterra.png', dpi=150)
plt.show()

# 计算平均种群
print(f"猎物平均数量: {np.mean(x):.2f}")
print(f"捕食者平均数量: {np.mean(y):.2f}")
print(f"理论平均值: 猎物={c/d:.2f}, 捕食者={a/b:.2f}")
```

## 6. 结果分析

### 6.1 Logistic模型参数敏感性

| 参数 | 增加效应 | 生物学意义 |
|------|----------|------------|
| $r$ | 更快达到K | 高繁殖率 |
| $K$ | 更高稳态 | 资源丰富 |
| $N_0$ | 不影响稳态 | 初始条件无关 |

### 6.2 捕食者-猎物振荡特征

| 特征 | 数值 | 解释 |
|------|------|------|
| 周期 | $2\pi/\sqrt{ac}$ | 种群波动周期 |
| 振幅 | 依赖初始条件 | 保守系统 |
| 相位差 | 捕食者滞后 | 食物链效应 |
