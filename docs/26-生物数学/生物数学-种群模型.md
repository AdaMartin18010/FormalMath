# 生物数学种群模型与代码实现

## 1. Logistic增长模型

### 模型
$$\frac{dN}{dt} = rN\left(1 - \frac{N}{K}\right)$$

### Python实现

```python
import numpy as np
from scipy.integrate import odeint
import matplotlib.pyplot as plt

def logistic_growth(N, t, r, K):
    """Logistic增长模型"""
    return r * N * (1 - N/K)

# 参数
r = 0.5  # 内禀增长率
K = 1000  # 环境容纳量
N0 = 100  # 初始种群

# 时间
 t = np.linspace(0, 20, 200)

# 求解
N = odeint(logistic_growth, N0, t, args=(r, K))

plt.figure(figsize=(10, 6))
plt.plot(t, N, 'b-', linewidth=2, label='Population')
plt.axhline(y=K, color='r', linestyle='--', label='Carrying Capacity (K)')
plt.xlabel('Time')
plt.ylabel('Population Size')
plt.title('Logistic Growth Model')
plt.legend()
plt.grid(True)
plt.savefig('logistic_growth.png')
plt.show()
```

## 2. Lotka-Volterra捕食者-被捕食者模型

### 模型
$$\frac{dx}{dt} = ax - bxy$$
$$\frac{dy}{dt} = -cy + dxy$$

### Python实现

```python
def lotka_volterra(Z, t, a, b, c, d):
    """Lotka-Volterra模型"""
    x, y = Z
    dxdt = a*x - b*x*y
    dydt = -c*y + d*x*y
    return [dxdt, dydt]

# 参数
a, b, c, d = 1.0, 0.1, 1.5, 0.075

# 初始条件
x0, y0 = 10, 5

# 时间
t = np.linspace(0, 15, 1000)

# 求解
solution = odeint(lotka_volterra, [x0, y0], t, args=(a, b, c, d))
x, y = solution.T

# 绘制
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(14, 5))

# 时间序列
ax1.plot(t, x, 'b-', label='Prey')
ax1.plot(t, y, 'r-', label='Predator')
ax1.set_xlabel('Time')
ax1.set_ylabel('Population')
ax1.legend()
ax1.grid(True)

# 相图
ax2.plot(x, y, 'g-')
ax2.set_xlabel('Prey')
ax2.set_ylabel('Predator')
ax2.grid(True)

plt.tight_layout()
plt.savefig('lotka_volterra.png')
plt.show()
```

## 3. SIR传染病模型

### 模型
$$\frac{dS}{dt} = -\beta SI$$
$$\frac{dI}{dt} = \beta SI - \gamma I$$
$$\frac{dR}{dt} = \gamma I$$

### Python实现

```python
def sir_model(Y, t, beta, gamma):
    """SIR模型"""
    S, I, R = Y
    N = S + I + R
    dSdt = -beta * S * I / N
    dIdt = beta * S * I / N - gamma * I
    dRdt = gamma * I
    return [dSdt, dIdt, dRdt]

# 参数（COVID-19估计值）
beta = 0.5  # 传染率
gamma = 0.1  # 恢复率（1/平均感染期）

# 初始条件
N = 1000000  # 总人口
I0 = 100     # 初始感染者
S0 = N - I0
R0 = 0

# 时间（天）
t = np.linspace(0, 160, 1000)

# 求解
solution = odeint(sir_model, [S0, I0, R0], t, args=(beta, gamma))
S, I, R = solution.T

# 绘制
plt.figure(figsize=(12, 6))
plt.plot(t, S, 'b-', label='Susceptible')
plt.plot(t, I, 'r-', label='Infected')
plt.plot(t, R, 'g-', label='Recovered')
plt.xlabel('Days')
plt.ylabel('Number of People')
plt.title('SIR Model')
plt.legend()
plt.grid(True)
plt.savefig('sir_model.png')
plt.show()

# 基本再生数
R0 = beta / gamma
print(f"基本再生数 R0 = {R0:.2f}")
```

## 习题

### 习题1
求Logistic方程的解析解。

**解答**：分离变量：
$$\frac{dN}{N(1-N/K)} = r dt$$

部分分式分解后积分：
$$N(t) = \frac{K}{1 + \left(\frac{K-N_0}{N_0}\right)e^{-rt}}$$

### 习题2
求Lotka-Volterra模型的平衡点。

**解答**：令导数为0：
- $(0, 0)$：平凡平衡点（不稳定）
- $(c/d, a/b)$：非平凡平衡点（中心点，周期轨道）
