# 证明论基础概念

## 一、证明系统

### 1.1 自然演绎 (Natural Deduction)

**起源**: Gentzen (1934)

**特点**:
- 直观、接近数学实践
- 引入/消去规则配对

**核心规则**:

| 连接词 | 引入规则 | 消去规则 |
|--------|----------|----------|
| ∧ (与) | A, B ⊢ A∧B | A∧B ⊢ A, A∧B ⊢ B |
| ∨ (或) | A ⊢ A∨B, B ⊢ A∨B | A∨B, A→C, B→C ⊢ C |
| → (蕴含) | A ⊢ B ⊢ A→B | A→B, A ⊢ B |
| ∀ (全称) | ⊢φ(x) ⊢ ∀x φ(x) | ∀x φ(x) ⊢ φ(t) |
| ∃ (存在) | φ(t) ⊢ ∃x φ(x) | ∃x φ(x), φ(x)→C ⊢ C |

**归约规则**: 
- 引入后接消去可简化（proof reduction）
- 对应于λ演算中的β归约

### 1.2 相继式演算 (Sequent Calculus)

**相继式**: Γ ⊢ Δ，其中 Γ, Δ 是公式多重集。

**解释**: Γ 中合取式蕴含 Δ 中析取式。

**结构规则**:
- 弱化: Γ ⊢ Δ ⇒ Γ, A ⊢ Δ
- 收缩: Γ, A, A ⊢ Δ ⇒ Γ, A ⊢ Δ
- 置换: 公式顺序无关

**切割规则 (Cut)**:

$$
\frac{\Gamma \vdash \Delta, A \quad A, \Sigma \vdash \Pi}{\Gamma, \Sigma \vdash \Delta, \Pi}
$$

### 1.3 切割消去定理 (Hauptsatz)

**定理** (Gentzen): 任何相继式证明都可转化为无切割的证明。

**意义**:
1. 一致性: 空相继式 ⊢ 不可证
2. 子公式性质: 证明中只出现子公式
3. 可判定性: 命题逻辑可判定
4. 插值定理

## 二、序数分析

### 2.1 证明论序数

**定义**: 理论T的**证明论序数** |T| 是使得T可证明递归函数的最小序数。

**意义**: 测量理论的"证明强度"。

### 2.2 Gentzen对PA的分析

**Peano算术 (PA)**:
- 归纳公理: φ(0) ∧ ∀n(φ(n)→φ(n+1)) → ∀n φ(n)

**序数标记系统**:
- 使用Cantor正规形
- ε₀ = 最小的满足 ω^α = α 的序数

**定理** (Gentzen): 
1. PA的一致性可在 PRA + ε₀-归纳 中证明
2. PA ⊬ Con(PA)（Gödel第二定理）
3. PA的证明论序数是 ε₀

**证明方法**:
1. 将PA证明转化为无穷证明树
2. 分配序数 < ε₀ 给证明
3. 切割消去降低序数
4. 超限归纳（到ε₀）证明一致性

### 2.3 序数层级

| 理论 | 证明论序数 | 关键特征 |
|------|-----------|----------|
| PRA | ω^ω | 原始递归算术 |
| PA | ε₀ | 一阶归纳 |
| ATR₀ | Γ₀ | 算术超限递归 |
| Π¹₁-CA₀ | ψ(Ω_ω) | 分析归纳 |
| ZFC | ? | 未确定 |

### 2.4 序数记号系统

**Veblen层级**:
- φ₀(α) = ω^α
- φ_{β+1}(α) = φ_β 的第α个不动点
- 极限步骤取交

**Buchholz记号**: ψ(Ω_ω) 等，处理更强的理论。

## 三、反向数学

### 3.1 框架

**基础系统**: RCA₀ (Recursive Comprehension Axiom)
- 可计算集的存在性
- 受限的归纳

**目的**: 确定证明特定数学定理所需的最小公理系统。

### 3.2 五大子系统

| 系统 | 全称 | 核心公理 | 强度 |
|------|------|----------|------|
| RCA₀ | 递归理解公理 | Δ¹₁-概括, Σ⁰₁-归纳 | 基础 |
| WKL₀ | 弱柯尼希引理 | RCA₀ + 弱柯尼希引理 | 中等 |
| ACA₀ | 算术概括公理 | RCA₀ + 算术概括 | 强 |
| ATR₀ | 算术超限递归 | ACA₀ + 超限递归 | 很强 |
| Π¹₁-CA₀ | Π¹₁-概括公理 | 对Π¹₁公式概括 | 极强 |

### 3.3 弱柯尼希引理 (WKL)

**陈述**: 无限二叉树有无限分支。

**等价定理**:
- Heine-Borel定理（紧致性）
- 一致连续性的扩展
- 初等微积分的许多结果

**分离**: WKL₀ 严格强于 RCA₀，但弱于 ACA₀。

### 3.4 分析定理的等价类

| 定理 | 所需系统 | 备注 |
|------|----------|------|
| 中值定理 | RCA₀ | 可计算证明 |
| 极值定理 | WKL₀ | 需要紧致性 |
| Bolzano-Weierstrass | ACA₀ | 需要序列选择 |
| 可数序数的确定性 | ATR₀ | 超限归纳 |
| 良基性的等价形式 | Π¹₁-CA₀ | 强归纳原理 |

## 四、可证明递归函数

### 4.1 定义

**定义**: 函数f是T-可证明递归的，若T可证明f是全函数。

**例子**:
- PRA: 原始递归函数
- PA: < ε₀-递归函数
- 更强理论: 更快增长的函数

### 4.2 Hardy层级

**定义**:
- H₀(n) = n
- H_{α+1}(n) = H_α(n+1)
- H_λ(n) = H_{λ[n]}(n)（对角化）

**意义**: H_{ε₀} 不是PA-可证明递归的（Goodstein定理）。

### 4.3 Goodstein定理

**定理** (Goodstein): 每个Goodstein序列最终达到0。

**Goodstein序列构造**:
1. 将n写成纯基b表示
2. 增加基数到b+1，减1
3. 重复

**证明论意义**:
- 是Π₂语句
- PA不能证明（Kirby-Paris, 1982）
- 需要ε₀-归纳
