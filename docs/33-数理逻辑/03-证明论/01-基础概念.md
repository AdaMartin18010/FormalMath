# 证明论基础概念

## 一、自然演绎 (Natural Deduction)

### 1.1 引言规则与消去规则

**合取 (∧)**:

```
  A    B          A ∧ B          A ∧ B
  ----- (∧I)      ----- (∧E₁)    ----- (∧E₂)
  A ∧ B           A              B
```

**蕴涵 (→)**:

```
  [A]              A → B    A
  ...              ----------- (→E)
  B    (→I)        B
  -----
  A → B
```

**析取 (∨)**:

```
  A              B              A ∨ B    [A]    [B]
  ----- (∨I₁)    ----- (∨I₂)    ...      ...    ...
  A ∨ B          A ∨ B          C        C      C
                 ------------------------------- (∨E)
                 C
```

### 1.2 否定与矛盾

**否定** (~):

```
  [A]
  ...
  ⊥
  ----- (~I)
  ~A
```

**矛盾 (⊥)**:

```
  ⊥
  ----- (⊥E)
  A
```

**双重否定消去** (经典逻辑):

```
  ~~A
  ----- (DNE)
  A
```

### 1.3 规范化定理

**定理**: 每个证明都可转换为范式（无"交通堵塞"）。

**归约**: 消去规则紧跟在引入规则后。

**例子**:

```
  π₁    π₂              π₁
  ----  ----            ----
  A     B       →       A
  ------ (∧I)           
  A ∧ B
  ---- (∧E₁)
  A
```

## 二、相继式演算 (Sequent Calculus)

### 2.1 相继式表示

**相继式**: $\Gamma \vdash \Delta$，其中 $\Gamma, \Delta$ 是公式多重集。

**解释**: 从假设 $\Gamma$ 可推出结论 $\Delta$ 中的至少一个。

### 2.2 结构规则

**弱化 (Weakening)**:

```
  Γ ⊢ Δ
  ------- (WL)
  A, Γ ⊢ Δ

  Γ ⊢ Δ
  ------- (WR)
  Γ ⊢ Δ, A
```

**收缩 (Contraction)**:

```
  A, A, Γ ⊢ Δ
  ----------- (CL)
  A, Γ ⊢ Δ
```

**交换 (Exchange)**: 允许重排公式顺序。

**切割 (Cut)**:

```
  Γ ⊢ Δ, A    A, Σ ⊢ Π
  -------------------- (Cut)
  Γ, Σ ⊢ Δ, Π
```

### 2.3 逻辑规则

**合取**:

```
  Γ ⊢ Δ, A    Γ ⊢ Δ, B
  -------------------- (∧R)
  Γ ⊢ Δ, A ∧ B

  A, Γ ⊢ Δ
  ----------- (∧L)
  A ∧ B, Γ ⊢ Δ
```

**蕴涵**:

```
  A, Γ ⊢ Δ, B
  ------------- (→R)
  Γ ⊢ Δ, A → B

  Γ ⊢ Δ, A    B, Σ ⊢ Π
  -------------------- (→L)
  A → B, Γ, Σ ⊢ Δ, Π
```

## 三、序数标记系统

### 3.1 Cantor正规形式

**定理**: 每个序数 $\alpha > 0$ 可唯一表示为：

$$\alpha = \omega^{\beta_1} \cdot c_1 + \omega^{\beta_2} \cdot c_2 + \ldots + \omega^{\beta_n} \cdot c_n$$

其中 $\beta_1 > \beta_2 > \ldots > \beta_n$，$c_i \in \mathbb{N}^+$。

### 3.2 ε-数

**定义**: $\varepsilon_0$ 是最小满足 $\omega^\alpha = \alpha$ 的序数。

**表示**: $\varepsilon_0 = \omega^{\omega^{\omega^{\cdot^{\cdot^{\cdot}}}}}$

**重要性**: Gentzen使用 $\varepsilon_0$ 证明了PA的一致性。

### 3.3 序数递归

**定义**: 函数 $f: \mathbb{N} \to \mathbb{N}$ 是 **$\alpha$-递归** 的，若可通过序数小于 $\alpha$ 的递归来定义。

**定理**: PA可证明的函数恰好是$< \varepsilon_0$-递归函数。

## 四、序数分析

### 4.1 Gentzen对PA的分析

**定理** (Gentzen, 1936): 若使用直到 $\varepsilon_0$ 的归纳，则PA是一致的。

**证明概要**:

1. 将PA证明转换为相继式演算形式
2. 消除切割 (Cut-elimination)
3. 为每个证明分配序数 $< \varepsilon_0$
4. 证明消除步骤降低序数
5. 空证明不能从非空公理得到

### 4.2 序数分析框架

**一般模式**:

| 理论 | 证明论序数 |
|------|-----------|
| PA | $\varepsilon_0$ |
| ATR₀ | $\Gamma_0$ |
| Π¹₁-CA₀ | $\psi(\Omega_\omega)$ |
| ZFC + 大基数 | 更大序数 |

### 4.3 逆向结果

**定理** (Kreisel): 若使用直到 $\alpha$ 的原始递归序数归纳可证明 $
abla$ 的一致性，则 $\alpha \geq \varepsilon_0$。

## 五、反向数学 (Reverse Mathematics)

### 5.1 二阶算术子系统

**语言**: 数变量 $x, y, z, \ldots$ 和集合变量 $X, Y, Z, \ldots$

**基本系统 RCA₀** (Recursive Comprehension Axiom):

- 基本算术公理
- $\Delta^0_1$-概括: $\Delta^0_1$ 公式定义的集合存在
- $I\Sigma_1$: $\Sigma_1$ 归纳

### 5.2 五大子系统

| 系统 | 概括公理 | 等价定理 |
|------|----------|----------|
| RCA₀ | $\Delta^0_1$ | 可计算数学的基础 |
| WKL₀ | RCA₀ + 弱König引理 | Heine-Borel, Bolzano-Weierstrass |
| ACA₀ | $\Sigma^0_1$-概括 | Bolzano-Weierstrass, 序列收敛 |
| ATR₀ | 算术传递递归 | 可数良序集, 开决定性 |
| Π¹₁-CA₀ | Π¹₁-概括 | 支配收敛定理, 紧性 |

### 5.3 等价类例子

**弱König引理**: 每个无穷二叉树有无穷分支。

**定理**: 在RCA₀中，WKL等价于：
- [0,1]的Heine-Borel覆盖性质
- 有界序列的Bolzano-Weierstrass性质
- 形式语言紧致性

### 5.4 证明论强度

| 系统 | 证明论序数 | 可证明递归函数 |
|------|-----------|---------------|
| RCA₀ | $\omega^\omega$ | 原始递归函数 |
| WKL₀ | $\omega^\omega$ | 原始递归函数 |
| ACA₀ | $\varepsilon_0$ | $<_\varepsilon_0$-递归 |
| ATR₀ | $\Gamma_0$ | $<_\Gamma_0$-递归 |
| Π¹₁-CA₀ | $\psi(\Omega_\omega)$ | 更大递归类 |
