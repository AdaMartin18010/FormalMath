# 证明论核心定理

## 一、Gentzen Hauptsatz (切割消去定理)

### 1.1 定理陈述

**定理**: 对于相继式演算LJ（直觉主义）或LK（经典），若 Γ ⊢ Δ 可证，则存在无切割的证明。

### 1.2 证明概要

**核心问题**: 切割规则破坏了子公式性质。

**消去策略**:
1. 对证明树高度进行归纳
2. 关键情况：切割公式由左右两边同时引入
3. 将切割"推高"到公理层

**秩降低引理**: 若两边的前切割证明秩为 n，则可转化为秩 < n 的证明。

**量词情况**: 需处理 ∃ 和 ∀ 的切割。

### 1.3 重要推论

**一致性**: 若 ⊢ ⊥ 可证，则通过切割消去，必存在公理化证明，矛盾。

**子公式性质**: 无切割证明中只出现子公式。

**插值定理**: 若 ⊢ A → B，则存在I使得：
- ⊢ A → I
- ⊢ I → B
- I的语言是A、B语言的交

## 二、Gentzen对PA的分析

### 2.1 无穷证明系统

**ω-规则**:

$$
\frac{\Gamma \vdash \Delta, A(0) \quad \Gamma \vdash \Delta, A(1) \quad \Gamma \vdash \Delta, A(2) \quad \cdots}{\Gamma \vdash \Delta, \forall x A(x)}
$$

**深度**: 证明树的序数秩。

### 2.2 序数分配

**方法**: 将PA证明转化为无穷树，分配序数。

**关键**: 归纳公理通过ω-规则处理，深度为ω。

**切割消去**: 无穷证明的切割消去降低序数：
- 简单情况: α + 1 → α
- 复杂情况: ω^α → α

### 2.3 边界分析

**定理**: 
1. PA ⊢ φ ⟺ 存在深度 < ε₀ 的无穷证明
2. PA ⊢ Con(PA) 需要深度 ≥ ε₀

**意义**: ε₀ 是PA的证明论序数。

## 三、Gödel不完备性定理

### 3.1 第一不完备性定理

**定理**: 任何包含PA的一致递归可公理化理论T都是不完备的。

**证明方法** (通过对角化):
1. Gödel编码: 将语法对象编码为自然数
2. 构造自指语句 G: "G在T中不可证"
3. 若 T ⊢ G，则 T ⊢ "G可证"，即 T ⊢ ¬G，矛盾
4. 若 T ⊢ ¬G，则 T ⊢ "G可证"，即T证明自身不可证，矛盾

### 3.2 第二不完备性定理

**定理**: 若T一致，则 T ⊬ Con(T)。

**形式化**: Con(T) = ¬Prov_T(⌜0=1⌝)

**证明概要**:
1. 第一定理的形式化: T ⊢ Con(T) → G
2. 由第一定理证明的分析: T ⊢ G → ¬Con(T)（实际上 G ⟺ ¬Con(T)）
3. 因此 T ⊢ Con(T) → ¬Con(T)
4. 若 T ⊢ Con(T)，则 T ⊢ ¬Con(T)，即T不一致

### 3.3 Löb定理

**定理**: T ⊢ □φ → φ ⟺ T ⊢ φ

其中 □φ 表示"φ在T中可证"。

**意义**: 可证性谓词的自我指称性质。

## 四、Hilbert第二问题

### 4.1 问题陈述

**Hilbert第二问题**: 证明算术公理的一致性。

### 4.2 Gentzen的解答

**正面结果**: PA的一致性可证。

**代价**: 需要超限归纳（到ε₀），超出PA自身。

**形式**: PRA + TI(ε₀) ⊢ Con(PA)

### 4.3 Gödel的限制

**定理**: 在PA内不能证明PA的一致性（若PA一致）。

**元数学意义**: 
- 有限主义证明的限制
- Hilbert计划的修正

## 五、Friedman的有限形式

### 5.1 Kruskal定理

**定理** (Kruskal): 有限树的无限序列必有树嵌入一对。

**有限形式**: 对于每个n，存在N使得任何n个标签的树序列 t₁, ..., t_N 中存在嵌入 t_i ↪ t_j (i < j)。

**证明论强度**:
- 无限形式: 证明论序数 Γ₀（ATR₀强度）
- 有限形式: 不能在任何少于 Γ₀ 的系统中证明

### 5.2 Robertson-Seymour图小定理

**定理**: 图在子式关系下是良拟序。

**证明论强度**: 超过 Π¹₁-CA₀。

**意义**: 组合数学中自然的强不可证命题。

## 六、可计算性与可证明性

### 6.1 Church-Turing论题

**论题**: 直观可计算 = Turing可计算

**形式**: 递归函数类恰是直觉可计算的函数类。

### 6.2 Curry-Howard对应

**对应**:

| 逻辑 | 计算 |
|------|------|
| 命题 | 类型 |
| 证明 | 程序 |
| 蕴含消除 | 函数应用 |
| 蕴含引入 | λ抽象 |
| 切割消去 | β归约 |
| 范式 | 正规形式 |

**意义**: 证明论与类型论、程序语言的统一。

## 七、序数分析的现代发展

### 7.1 自举过程

**问题**: 如何为强理论（如ZFC）找到证明论序数？

**方法**: 迭代过程，从内向外分析。

### 7.2 Rathjen的工作

**结果**: Kripke-Platek集合论 + 可容许集 的序数分析。

**技术**: 局部谓词逻辑、 collapsible hierarchies。

### 7.3 反推数学的大间隙

**问题**: ATR₀ 与 Π¹₁-CA₀ 之间的系统。

**方法**: 使用更强的序数记号系统。
