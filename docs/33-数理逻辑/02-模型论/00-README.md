# 模型论 (Model Theory)

## 模块概述

模型论研究形式语言及其解释（模型）之间的关系。本模块从经典紧致性定理出发，深入探讨稳定性理论、o-极小性等现代核心主题。

| [01-基础概念.md](./01-基础概念.md) | 稳定性 | 稳定性分层、 Shelah 分类定理 |

| [03-实战问题.md](./03-实战问题.md) | 应用问题 | 代数闭域、实闭域、微分闭域分析 |

## 学习路径

```

语言与结构 → 紧致性 → 完备性与量词消去 → 稳定性 → 几何模型论

```
