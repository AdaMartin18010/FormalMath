# 模型论基础概念

## 1. 形式语言与结构

### 1.1 签名 (Signature)

**定义**: 签名 $\sigma = (\mathcal{C}, \mathcal{F}, \mathcal{R}, \text{ar})$ 包含：
- 常数符号集 $\mathcal{C}$
- 函数符号集 $\mathcal{F}$
- 关系符号集 $\mathcal{R}$
- 元数函数 $\text{ar}: \mathcal{F} \cup \mathcal{R} \to \mathbb{N}$

**例子**: 群论签名 $\sigma_{\text{grp}} = (\{e\}, \{\cdot, {}^{-1}\}, \emptyset)$

### 1.2 结构 (Structure)

**定义**: $\sigma$-结构 $\mathcal{M} = (M, (c^\mathcal{M})_{c \in \mathcal{C}}, (f^\mathcal{M})_{f \in \mathcal{F}}, (R^\mathcal{M})_{R \in \mathcal{R}})$
- 论域 $M$（非空集合）
- 常数、函数、关系的解释

### 1.3 公式与满足关系

**定义**: 
- 项 $t$ 的归纳定义
- 原子公式：$t_1 = t_2$ 或 $R(t_1, \ldots, t_n)$
- 公式：由原子公式通过 $\neg, \land, \lor, \rightarrow, \forall, \exists$ 构造

**满足关系**: $\mathcal{M} \models \varphi[a_1, \ldots, a_n]$（在赋值下的满足）

## 2. 紧致性定理 (Compactness Theorem)

### 2.1 定理陈述

**定理**: 一阶理论 $T$ 有模型当且仅当 $T$ 的每个有限子集有模型。

$$T \models \varphi \Leftrightarrow \exists T_0 \subseteq^{\text{fin}} T: T_0 \models \varphi$$

### 2.2 证明方法

**方法一: 超积 (Ultraproducts)**

**定义**: 设 $\{\mathcal{M}_i\}_{i \in I}$ 是结构族，$\mathcal{U}$ 是 $I$ 上的超滤子。

$$\prod_{i \in I} \mathcal{M}_i / \mathcal{U} = \{(a_i)_{i \in I} : a_i \in M_i\} / \sim$$

其中 $(a_i) \sim (b_i)$ 当且仅当 $\{i : a_i = b_i\} \in \mathcal{U}$

**Łoś定理**: $\prod \mathcal{M}_i / \mathcal{U} \models \varphi$ 当且仅当 $\{i : \mathcal{M}_i \models \varphi\} \in \mathcal{U}$

**紧致性证明**: 若 $T$ 有限可满足，取超积构造模型。

**方法二: 完备性定理**

语法可证 $\Leftrightarrow$ 语义有效

### 2.3 应用

**应用1: 非标准分析**
- 构造包含无穷小和无限大数的非标准实数域 $^*\mathbb{R}$
- 转移原理：一阶命题在 $\mathbb{R}$ 和 $^*\mathbb{R}$ 中等价

**应用2: 无穷图染色**
- 若有限子图可 $k$-染色，则全图可 $k$-染色

**应用3: Robinson原理**
- 代数命题在所有特征0域成立当且仅当在几乎所有特征 $p$ 域成立

## 3. 完备理论与量词消去

### 3.1 完备理论

**定义**: 理论 $T$ 是完备的，若对任意句子 $\varphi$，$T \models \varphi$ 或 $T \models \neg\varphi$。

**等价条件**: 
- 任意两个模型初等等价
- $T$ 的模型类在同构意义下只有一个可数模型（可数语言）

### 3.2 量词消去

**定义**: 理论 $T$ 允许量词消去，若对每个公式 $\varphi$，存在无量词公式 $\psi$ 使得 $T \models \varphi \leftrightarrow \psi$。

**判别准则** (QE判定): $T$ 允许QE当且仅当对任意原初公式 $\exists x \, \varphi(x, \bar{y})$，存在无量词公式 $\psi(\bar{y})$ 等价。

**经典例子**:

| 理论 | 量词消去 | 方法 |
|------|----------|------|
| ACF (代数闭域) | 是 | 结式、代数几何 |
| RCF (实闭域) | 是 | Sturm定理、柱面代数分解 |
| DCF$_0$ (微分闭域) | 是 | 微分代数 |
| Th$(\mathbb{Q}, <)$ | 是 | 直接构造 |
| PA (皮亚诺算术) | 否 | Gödel不完备性 |

### 3.3 模型完全性

**定义**: $T$ 是模型完全的，若 $T$ 的任意两个模型间的包含是初等嵌入。

**等价条件**:
- 每个公式等价于一个全称存在公式
- 若 $\mathcal{M} \subseteq \mathcal{N}$，则 $\mathcal{M} \preceq \mathcal{N}$

## 4. 稳定性理论

### 4.1 稳定性分层

**定义**: 理论 $T$ 是 **$\kappa$-稳定** 的，若对于 $|A| \leq \kappa$，$|S_n(A)| \leq \kappa$。

**稳定性分层**:

| 类别 | 定义 | 例子 |
|------|------|------|
| $\omega$-稳定 | $|S_n(A)| = \aleph_0$（$A$ 可数） | ACF, DCF$_0$ |
| 超稳定 | 稳定且有主导函数 | 严格稳定理论 |
| 稳定 | 对所有足够大的 $\kappa$ 是 $\kappa$-稳定 | 可分Hilbert空间 |
| 单纯 (simple) | 有独立关系的推广 | 伪有限域 |
| NIP | 无独立性质 | 稠密线性序 |
| 不稳定 | 以上皆非 | 随机图 |

### 4.2 分裂与分离

**定义**: 型 $p \in S(B)$ **在 $A \subseteq B$ 上分裂**，若存在 $\bar{b}_1, \bar{b}_2 \in B$ 在 $A$ 上同一型，但 $\varphi(\bar{x}, \bar{b}_1), \neg\varphi(\bar{x}, \bar{b}_2) \in p$。

**重要性**: 稳定性等价于型不分裂的有界性。

### 4.3 Shelah分类定理

**定理**: 每个完备可数理论 $T$ 属于以下之一：
1. **$\omega$-稳定**: 有Morley秩，结构可完全分类
2. **超稳定但非$\omega$-稳定**: 有某种分层结构
3. **严格稳定**: 更复杂但仍可控
4. **单纯不稳定**: 有随机成分
5. **非单纯**: 最复杂，如随机图

**意义**: 稳定性类别决定了理论的"可分类性"。

## 5. o-极小性 (O-minimality)

### 5.1 定义

**定义**: 全序结构 $(M, <, \ldots)$ 是 **o-极小** 的，若每个可定义子集是有限个区间和点的并。

**等价条件**: 每个一元可定义集形如：
$$\bigcup_{i=1}^n (a_i, b_i) \cup \bigcup_{j=1}^m \{c_j\}$$

### 5.2 经典例子

| 结构 | o-极小性 | 证明 |
|------|----------|------|
| $(\mathbb{R}, +, \cdot, <, 0, 1)$ | 是 (Tarski) | 量词消去 |
| $(\mathbb{R}, +, \cdot, \exp, <)$ | 是 (Wilkie) | 模型论分析 |
| $(\mathbb{R}, +, \cdot, \sin)$ | 否 | 定义 $\mathbb{Z}$ |
| $(\mathbb{R}_{\text{an}}, \exp)$ | 是 | 硬分析 |

### 5.3 o-极小结构的性质

**单调性定理**: 可定义函数 $f: (a, b) \to M$ 是分段单调的（分段连续且单调）。

**细胞分解**: 可定义集可分解为有限个"细胞"的不交并。

**维数理论**: 可定义集的维数良定义且具有良好的性质。

### 5.4 几何意义

** tame topology**: o-极小结构避免了Gödel现象，是"驯服"的。

**应用**:
- 实代数几何
- 解析几何
- Lie群的可定义子集
