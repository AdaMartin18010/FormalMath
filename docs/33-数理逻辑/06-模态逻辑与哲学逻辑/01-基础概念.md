# 模态逻辑与哲学逻辑基础概念

## 1. 经典模态逻辑

### 1.1 语法

**模态语言**: 在命题逻辑基础上添加模态算子

- **□** (Box): 必然性
- **◇** (Diamond): 可能性

**形成规则**:
```
φ ::= p | ⊥ | ¬φ | φ ∧ φ | φ ∨ φ | φ → φ | □φ | ◇φ
```

**对偶关系**: ◇φ ⟺ ¬□¬φ

### 1.2 公理系统

**K系统 (正规模态逻辑)**:
- 命题重言式
- **K**: □(p → q) → (□p → □q)
- **MP**: 从φ和φ→q推出q
- **Nec**: 从φ推出□φ

**常见扩展**:

| 系统 | 添加公理 | 语义特征 |
|------|----------|----------|
| T | T: □p → p | 自反关系 |
| B | B: p → □◇p | 对称关系 |
| S4 | 4: □p → □□p | 传递关系 |
| S5 | 5: ◇p → □◇p | 等价关系 |
| D | D: □p → ◇p | 持续性关系 |

### 1.3 关系语义 (Kripke语义)

**框架**: F = (W, R)
- W: 可能世界集
- R ⊆ W × W: 可达关系

**模型**: M = (W, R, V)
- V: 命题变元 → P(W)

**满足关系** (M, w ⊨ φ):

| 公式 | 条件 |
|------|------|
| M, w ⊨ p | w ∈ V(p) |
| M, w ⊨ ¬φ | M, w ⊭ φ |
| M, w ⊨ φ ∧ ψ | M, w ⊨ φ 且 M, w ⊨ ψ |
| M, w ⊨ □φ | ∀v: wRv ⟹ M, v ⊨ φ |
| M, w ⊨ ◇φ | ∃v: wRv 且 M, v ⊨ φ |

## 2. 对应理论

### 2.1 框架可定义性

**对应**: 模态公式与一阶公式的对应关系

**经典对应**:

| 模态公理 | 对应一阶条件 |
|----------|-------------|
| □p → p | ∀x: R(x, x) (自反) |
| □p → □□p | ∀xyz: R(x,y) ∧ R(y,z) → R(x,z) (传递) |
| p → □◇p | ∀xy: R(x,y) → R(y,x) (对称) |
| ◇p → □◇p | ∀xyz: R(x,y) ∧ R(x,z) → R(y,z) (欧几里得) |
| □p → ◇p | ∀x: ∃y: R(x,y) (持续性) |

### 2.2 Sahlqvist定理

**定理**: Sahlqvist公式有对应一阶条件，且完备性成立。

**Sahlqvist公式形式**: 正公式与负公式的特定组合。

**例子**: □p → p 是Sahlqvist的。

### 2.3 Goldblatt-Thomason定理

**定理**: 框架类在模态逻辑中可定义当且仅当它：
1. 在生成子框架下封闭
2. 在不相交并下封闭
3. 在有界态射像下封闭
4. 在超滤扩张下反射

## 3. 认知逻辑

### 3.1 知识算子

**Kᵢφ**: 主体i知道φ

**性质** (S5知识):
- **K**: Kᵢ(φ → ψ) → (Kᵢφ → Kᵢψ)
- **T**: Kᵢφ → φ (真实性)
- **4**: Kᵢφ → KᵢKᵢφ (正内省)
- **5**: ¬Kᵢφ → Kᵢ¬Kᵢφ (负内省)

### 3.2 公共知识

**Eφ**: 每个人都知φ

**Cφ**: φ是公共知识

**不动点定义**: Cφ ⟺ E(φ ∧ Cφ)

**意义**: 无限嵌套: 我知道，你知道，我知道你知道，...

### 3.3 信念

**Bᵢφ**: 主体i相信φ

**通常性质** (KD45):
- D: Bᵢφ → ¬Bᵢ¬φ (一致性)
- 4: Bᵢφ → BᵢBᵢφ (内省)
- 5: ¬Bᵢφ → Bᵢ¬Bᵢφ

**注意**: 信念不必真实（非T公理）。

### 3.4 知识更新

**公开宣告**: [!φ]ψ 表示"宣告φ后ψ成立"

**语义**: M, w ⊨ [!φ]ψ ⟺ 若 M, w ⊨ φ 则 M|φ, w ⊨ ψ

**模型更新**: M|φ 是从M中删除¬φ为真的世界

## 4. 时态逻辑

### 4.1 线性时态逻辑 (LTL)

**算子**:
- **X** (Next): Xφ - 下一时刻φ成立
- **F** (Future): Fφ - 将来某时刻φ成立
- **G** (Globally): Gφ - 永远φ成立
- **U** (Until): φ U ψ - φ成立直到ψ成立

**语义**: 在无限序列 σ = s₀, s₁, s₂, ... 上解释

### 4.2 分支时间逻辑 (CTL)

**路径量词**:
- **A**: 对所有路径
- **E**: 存在某路径

**组合**:
- AFφ: 对所有路径，将来φ成立
- EGφ: 存在路径，永远φ成立
- E[φ U ψ]: 存在路径，φ直到ψ

**CTL与LTL**: CTL* 包含两者作为片段。

### 4.3 区间时态逻辑

**基本单位**: 时间区间而非时间点

**算子**:
- ⟨A⟩φ: 对所有子区间φ
- ⟨B⟩φ: 对某开始子区间φ
- ⟨E⟩φ: 对某结束子区间φ

**应用**: 时间段的自然语言描述。

## 5. 道义逻辑与条件句

### 5.1 道义逻辑

**算子**:
- **Oφ**: φ是义务的
- **Pφ**: φ是许可的
- **Fφ**: φ是被禁止的

**关系**: Oφ ⟺ ¬P¬φ ⟺ F¬φ

**悖论**: 
- Ross悖论: Oφ 推出 O(φ ∨ ψ)
- 善意原则

### 5.2 条件句逻辑

**实质条件句**: φ → ψ

**严格条件句**: □(φ → ψ)

**变严格条件句**: φ □→ ψ (Lewis)

**语义**: 在最接近的φ-世界中ψ成立

## 6. 非经典逻辑

### 6.1 直觉主义逻辑

**特征**: 拒绝排中律 (p ∨ ¬p)

**语义**: 
- Kripke语义（信息增长）
- Brouwer-Heyting-Kolmogorov解释

**对应**: 与类型论通过Curry-Howard对应

### 6.2 多值逻辑

**三值逻辑** (Kleene):
- 真、假、未定义

**模糊逻辑**:
- 真值在[0,1]区间
- t-范数、余t-范数

### 6.3 相干逻辑与次协调逻辑

**相干逻辑**: 要求前提与结论有实质联系

**次协调逻辑 (paraconsistent)**: 
- 允许矛盾存在
- 非平凡推理

**应用**: 处理不一致知识库。

## 7. 动态逻辑

### 7.1 命题动态逻辑 (PDL)

**程序表达式**:
- a (原子动作)
- π₁; π₂ (顺序)
- π₁ ∪ π₂ (选择)
- π* (迭代)
- φ? (测试)

**公式**: [π]φ (执行π后φ必然成立)

### 7.2 游戏逻辑

**博弈论语义**: 将逻辑视为游戏
- 证明者 vs 反驳者
- 有效公式 = 证明者有必胜策略

**博弈代数**: 博弈的形式操作
