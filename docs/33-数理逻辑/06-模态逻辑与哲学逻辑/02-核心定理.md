# 模态逻辑与哲学逻辑核心定理

## 一、Kripke完备性定理

### 1.1 定理陈述

**定理**: 正规模态逻辑K关于所有Kripke框架类是强完备的。

**形式**: K ⊢ φ ⟺ φ在所有框架上有效

### 1.2 典范模型方法

**构造**:
1. 取所有极大一致集作为世界
2. 定义可达关系: ΓRΔ ⟺ □⁻¹Γ ⊆ Δ
3. 赋值: V(p) = {Γ : p ∈ Γ}

**关键引理**: 真值引理 - M^K, Γ ⊨ φ ⟺ φ ∈ Γ

### 1.3 特殊系统的完备性

| 系统 | 完备框架类 |
|------|-----------|
| K | 所有框架 |
| T | 自反框架 |
| S4 | 预序框架 |
| S5 | 等价关系框架 |
| GL | 有穷传递逆向良基框架 |

## 二、模态逻辑的判定性

### 2.1 基本模态逻辑

**定理**: 基本模态逻辑K是可判定的。

**证明方法**: 
- 树模型性质
- 有穷模型性质
-  filtration

**复杂度**: PSPACE完全

### 2.2 有穷模型性质

**定理**: 若φ可满足，则在某有穷模型可满足。

**filtration构造**:
1. 取真子公式闭包
2. 按等价类定义世界
3. 过滤关系

### 2.3 不可判定结果

**定理**: 以下系统不可判定：
- 带全称量词的模态谓词逻辑
- 某些时态逻辑扩展
- 某些混合逻辑

## 三、Goldblatt-Thomason定理

### 3.1 定理陈述

**定理**: 框架类K在模态逻辑中可定义当且仅当K在以下运算下封闭：
1. 生成子框架
2. 不相交并
3. 有界态射像
4. 超滤扩张反射

### 3.2 应用

**判断框架类可定义性**:
- 传递框架类: 可定义 (对应4公理)
- 良基框架类: 不可定义

### 3.3 一阶对应

**定理** (van Benthem): 模态公式与一阶公式等价当且仅当它保持超滤扩张。

## 四、对应理论深化

### 4.1 Sahlqvist对应定理

**定理**: 每个Sahlqvist公式φ有对应一阶条件α_φ，且φ关于α_φ完备。

**算法**: Sahlqvist-van Benthem算法从公式提取对应条件。

### 4.2 非Sahlqvist公式

**例子**: McKinsey公理 □◇p → ◇□p

**对应**: 无简单一阶对应（但有一阶无穷对应）

**性质**: 对应收敛条件（难以一阶表达）

## 五、模态代数与对偶性

### 5.1 模态代数

**定义**: 布尔代数 + 模态算子□满足：
- □⊤ = ⊤
- □(a ∧ b) = □a ∧ □b

### 5.2 Jónsson-Tarski对偶性

**定理**: 
- 模态代数 ⟺ 描述性一般框架
- 完全模态代数 ⟺ Kripke框架

**意义**: 代数和语义方法的统一。

## 六、认知逻辑的核心定理

### 6.1 公共知识特征

**定理**: C_Gφ 是满足以下条件的最小公式：
1. ⊨ C_Gφ → E_G(φ ∧ C_Gφ)
2. 若 ⊨ ψ → E_G(φ ∧ ψ) 则 ⊨ ψ → C_Gφ

**不动点**: C_Gφ ⟺ νX. E_G(φ ∧ X)

### 6.2 知识更新与AGM

**AGM公设** (信念修正):
- (K*1) K * φ 是信念集
- (K*2) φ ∈ K * φ
- (K*3) K * φ ⊆ Cn(K ∪ {φ})
- (K*4) 若¬φ ∉ K，则 Cn(K ∪ {φ}) ⊆ K * φ
- ...

**定理**: 满足AGM公设的修正算子存在。

## 七、时态逻辑的核心定理

### 7.1 Kamp定理

**定理** (Kamp): 在Dedekind完全序上，LTL与一阶逻辑在表达能力上等价。

**意义**: LTL可以表达所有时间性质（在该类序上）。

### 7.2 模型检验复杂度

| 逻辑 | 模型检验复杂度 |
|------|---------------|
| LTL | PSPACE完全 |
| CTL | P完全 |
| CTL* | PSPACE完全 |
| μ-演算 | UP ∩ coUP (拟多项式) |

## 八、哲学逻辑中的元定理

### 8.1 超赋值与真理

**超赋值语义**: 处理真值间隙

**定理**: 超赋值保持经典重言式。

### 8.2 直觉主义完备性

**定理**: 直觉主义命题逻辑对Kripke语义完备。

**方法**: 典范模型（使用饱和集而非极大一致集）

### 8.3 模糊逻辑表示定理

**定理**: t-范数模糊逻辑与连续[0,1]值解释对应。

**例子**: 
- Łukasiewicz t-范数 ⟺ Łukasiewicz逻辑
- Gödel t-范数 ⟺ Gödel逻辑
- 乘积t-范数 ⟺ 乘积逻辑
