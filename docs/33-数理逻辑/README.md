# 33-数理逻辑 (Mathematical Logic)

## 分支概述

数理逻辑是数学的基础分支，研究数学推理的形式结构和性质。本分支涵盖集合论、模型论、证明论、可计算性理论、类型论以及模态逻辑与哲学逻辑六大核心领域，为现代数学和计算机科学提供严格的理论基础。

## 学习路径建议

### 基础路径
```
集合论(ZFC) → 一阶逻辑 → 模型论基础 → 可计算性理论
```

### 理论深化路径
```
证明论 → 序数分析 → 类型论 → HoTT
```

### 应用导向路径
```
可计算性理论 → 复杂度理论 → 类型论 → 形式化验证
```

### 哲学交叉路径
```
模态逻辑 → 认知逻辑 → 非经典逻辑 → 哲学应用
```
