# 可计算性理论核心定理

## 1. Church-Turing论题

### 1.1 论题陈述

**Church-Turing论题**: 直观可计算的函数类恰是图灵可计算的函数类。

**形式**: 任何"算法"都可被图灵机模拟。

### 1.2 支持证据

**等价模型**:
- λ演算 (Church)
- 递归函数 (Gödel, Kleene)
- 图灵机 (Turing)
- 寄存器机
- 元胞自动机
- 随机存取机器 (RAM)

**结论**: 所有这些模型定义了相同的可计算函数类。

### 1.3 物理版本

**物理Church-Turing论题**: 任何物理过程都可被图灵机有效模拟。

**挑战**:
- 量子计算（可能多项式加速）
- 连续计算模型
- 超计算模型（如Malament-Hogarth时空）

## 2. 停机问题不可判定性

### 2.1 定理陈述

**定理** (Turing, 1936): 语言 A_TM = {⟨M, w⟩ : M 接受 w} 是不可判定的。

### 2.2 对角化证明

**假设**: 存在判定器H判定A_TM。

**构造**: 图灵机D定义如下：
```
D(⟨M⟩):
    运行 H(⟨M, ⟨M⟩⟩)
    若 H 接受，则拒绝
    若 H 拒绝，则接受
```

**矛盾**: 运行 D(⟨D⟩)：
- 若D接受⟨D⟩，则H(⟨D, ⟨D⟩⟩)接受，则D拒绝⟨D⟩
- 若D拒绝⟨D⟩，则H(⟨D, ⟨D⟩⟩)拒绝，则D接受⟨D⟩

### 2.3 推论

**定理**: 停机问题的补不是RE。

**定理**: RE在补运算下不封闭。

## 3. Rice定理

### 3.1 定理陈述

**定理** (Rice, 1953): 设P是图灵机语言的非平凡性质（即∃M₁: L(M₁)∈P, ∃M₂: L(M₂)∉P），则语言 L_P = {⟨M⟩ : L(M) ∈ P} 是不可判定的。

### 3.2 证明概要

**方法**: 从A_TM归约。

**步骤**:
1. 设M*是某固定图灵机，L(M*) ∈ P（或∉P）
2. 给定⟨M, w⟩，构造M'："忽略输入x，模拟M在w上"
3. 则 L(M') = Σ* 若M接受w，否则 = ∅
4. 适当选择P使得这构成归约

### 3.3 应用

**不可判定问题**:
- 程序是否接受空语言
- 程序是否接受正则语言
- 程序是否计算全函数
- 两个程序是否等价

## 4. Cook-Levin定理

### 4.1 定理陈述

**定理** (Cook, 1971; Levin, 1973): SAT是NP完全的。

### 4.2 证明概要

**第一部分**: SAT ∈ NP
- 证书: 真值赋值
- 验证: 多项式时间检查公式

**第二部分**: ∀L ∈ NP, L ≤_p SAT

**构造**: 
1. 设N是接受L的非确定图灵机，时间限制n^k
2. 对输入w，构造公式φ_w描述N的计算表
3. φ_w可满足 ⟺ N接受w

**计算表变量**: 
- T[i, j, a]: 第i步第j格为a
- H[i, j]: 第i步头在第j格
- S[i, q]: 第i步状态为q

**约束**: 初始条件、转移约束、接受条件

### 4.3 推论

**NP完全性**: 若SAT ∈ P，则P = NP。

**归约链**: SAT ≤_p 3-SAT ≤_p CLIQUE ≤_p VERTEX-COVER ≤_p ...

## 5. 空间层次定理

### 5.1 定理陈述

**空间层次定理**: 若f(n)是空间可构造函数且f(n) ≥ log n，则：

$$\text{SPACE}(o(f(n))) \subsetneq \text{SPACE}(f(n))$$

### 5.2 证明概要

**方法**: 对角化

**构造**: 图灵机D: "在输入⟨M⟩上，模拟M在⟨M⟩上，空间限制f(n)"
- 若M拒绝，则接受
- 若M接受，则拒绝
- 若M超出空间或循环，则接受

**关键**: 使用f(n)空间，D可以在o(f(n))空间中模拟任何使用o(f(n))的M。

### 5.3 推论

**严格包含**:
- L ⊂ PSPACE
- PSPACE ⊂ EXPSPACE

**分离**: L ≠ PSPACE，PSPACE ≠ EXPSPACE

## 6. Savitch定理

### 6.1 定理陈述

**定理** (Savitch, 1970): 对于f(n) ≥ log n：

$$\text{NSPACE}(f(n)) \subseteq \text{SPACE}(f(n)^2)$$

### 6.2 证明概要

**核心思想**: 用空间换非确定性。

**可达性算法**:
```
REACH(c₁, c₂, t):  // 从配置c₁能否在t步到达c₂
    若 t = 1: 检查直接转移
    对每个中间配置cₘ:
        若 REACH(c₁, cₘ, t/2) 且 REACH(cₘ, c₂, t/2):
            返回 真
    返回 假
```

**空间分析**: 递归深度log t，每层存储一个配置，总空间O(f(n)·log t) = O(f(n)²)。

### 6.3 推论

**PSPACE = NPSPACE**

**多项式空间闭包**: PSPACE在非确定性下封闭。

## 7. 时间层次定理

### 7.1 定理陈述

**时间层次定理**: 若f(n)是时间可构造函数，则：

$$\text{TIME}(o(f(n)/\log f(n))) \subsetneq \text{TIME}(f(n))$$

### 7.2 推论

**严格包含**:
- P ⊂ EXPTIME

**注意**: 证明比空间层次更复杂，因需要考虑模拟开销。

## 8. 概率算法的去随机化

### 8.1 Impagliazzo-Wigderson定理

**定理**: 若存在困难的平均情况问题，则BPP = P。

**条件**: E（指数时间）不包含大小为2^o(n)的电路的P/poly。

### 8.2 Nisan-Wigderson构造

**伪随机生成器**: 从短随机种子生成长伪随机串。

**应用**: 若存在困难函数，则BPP可去随机化。

## 9. 交互证明与IP

### 9.1 交互证明系统

**模型**: 证明者P（计算无界）与验证者V（概率多项式时间）交互。

**类别IP[k]**: k轮交互后V接受或拒绝。

### 9.2 Shamir定理

**定理** (Shamir, 1992): IP = PSPACE。

**意义**: 
- 交互证明极其强大
- 证明PSPACE问题可高效验证

### 9.3 PCP定理

**定理**: NP = PCP[O(log n), O(1)]。

**解释**: NP问题的证书可被概率验证，通过读取O(log n)随机位和查询证书O(1)位。

**应用**: 近似算法的困难性。
