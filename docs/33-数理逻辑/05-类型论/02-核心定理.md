# 类型论核心定理

## 一、Church-Rosser定理

### 1.1 定理陈述

**定理** (Church-Rosser): λ演算满足合流性（confluence）。

**形式**: 若 t →* u 且 t →* v，则存在w使得 u →* w 且 v →* w。

```

    t
   / \
  ↓   ↓
  u   v
   \ /
    ↓
    w

```

### 1.2 证明概要

**方法**: 平行归约 + 菱形性质

**平行归约 (⇉)**: 同时归约所有可约式

**引理**: 若 t ⇉ u 且 t ⇉ v，则存在w使得 u ⇉ w 且 v ⇉ w。

**传递**: →* 是 ⇉ 的传递闭包。

### 1.3 推论

**正规形式的唯一性**: 若t有正规形式，则唯一。

## 二、强正规化定理

### 2.1 定理陈述

**定理**: 简单类型λ演算 (λ→) 满足强正规化。

**形式**: 每个良类型的项都有有限的归约序列终止于正规形式。

### 2.2 证明方法

**方法一: Tait可计算性**

**定义**:
- 基类型A: Comp_A = {t | t强正规化}
- 函数类型σ→τ: Comp_{σ→τ} = {t | ∀s∈Comp_σ, t s ∈ Comp_τ}

**引理**: 若Γ ⊢ t:τ且对每个(x:σ)∈Γ有γ(x)∈Comp_σ，则t[γ]∈Comp_τ。

**方法二: 可逼近语义 (Girard)**

**可逼近映射**: 用有限信息逼近无限对象。

### 2.3 System F的正规化

**定理** (Girard): System F满足强正规化。

**方法**: 可计算性候选（candidats de réductibilité）

**复杂性**: 需要二阶可计算性谓词。

## 三、Curry-Howard同构

### 3.1 定理陈述

**同构**: 

```

逻辑命题  ⟷  类型
证明      ⟷  程序
证明归约  ⟷  程序计算

```

### 3.2 详细对应

| 逻辑系统 | 类型系统 | 计算模型 |
|----------|----------|----------|
| 命题直觉主义逻辑 | 简单类型λ演算 | 程序 |
| 二阶逻辑 | System F | 多态程序 |
| 构造演算 | 依赖类型 | 证明助手 |

### 3.3 规范化即提取

**定理**: 证明的规范化对应程序的执行。

**应用**: 从证明提取程序（程序综合）。

## 四、参数性 (Parametricity)

### 4.1 定理陈述

**定理** (Reynolds): 多态类型限制其实现。

**形式**: 若 t : ∀X.X → X，则 t 必然等于 ΛX.λx:X.x。

### 4.2 自由定理 (Free Theorems)

**例子**: 
- f : ∀X.List X → List X ⟹ f保持长度或改变但满足特定条件
- reverse : ∀X.List X → List X ⟹ reverse (map g xs) = map g (reverse xs)

### 4.3 关系语义

**定义**: 类型解释为关系而非集合。

**意义**: 捕捉类型约束的行为等价。

## 五、Univalence公理的一致性

### 5.1 定理陈述

**定理** (Voevodsky): Univalence公理在单纯集合模型中成立。

**定理** (Cohen-Coquand-Huber-Mörtberg): 立方类型论中Univalence可计算。

### 5.2 单纯集合模型

**构造**: 类型 ⟷  Kan复形

**路径**: 同伦路径

**意义**: HoTT有集合论语义。

### 5.3 立方类型论

**创新**: 直接构造路径而非公理化。

**结果**: 
- 可计算Univalence
- 更好的归约行为
- 实现于Cubical Agda

## 六、归纳原理的推导

### 6.1 定理陈述

**定理**: 归纳类型的归纳原理可从其构造子推导。

### 6.2 Church编码

**自然数**:

```

ℕ = ∀X. (X → X) → X → X
zero = λs z. z
succ = λn s z. s (n s z)

```

**缺点**: 无归纳原理（需公理化）。

### 6.3 W类型

**定义**: W(a:A).B(a) 表示良基树。

**通用性**: 所有归纳类型可归约到W类型（在扩展类型论中）。

## 七、类型论的一致性

### 7.1 定理陈述

**定理**: MLTT（Martin-Löf类型论）是一致的（若ZFC+存在无限多不可达基数一致）。

### 7.2 证明方法

**集合论模型**: 
- 类型 ⟷ 集合
- Π ⟷ 依赖积
- Σ ⟷ 依赖和
- Id ⟷ 对角线

**复杂性**: 宇宙层级需要大基数。

### 7.3 构造主义语义

**可实现性**: 类型解释为可计算对象。

**意义**: 构造主义类型论有计算语义。

## 八、归一化即同构

### 8.1 定理陈述

**定理** (Coquand): 证明的归一化给出同构：

$$\text{Normal}(A) \simeq \text{Proof}(A)$$

### 8.2 外延 vs 内涵

**外延类型论** (ETT):
- 同一性反映等式
- 可判定性差

**内涵类型论** (ITT):
- 同一性是归纳族
- 可判定性好

**联系**: ETT = ITT + 函数外延性 + UIP

### 8.3 Set-level类型论

**定义**: 类型满足UIP（同一性证明的唯一性）。

**等价**: Set-level类型论 ≈ 外延类型论。

**应用**: 数学形式化的实用选择。
