## 一、理论概述

### 1.1 理论定位

同调代数是研究数学对象"缺陷"的学科，通过链复形和导出函子来测量代数构造的非正合性。从经典的Tor和Ext函子到现代的导出范畴理论，同调代数已成为连接代数、拓扑、几何和数论的统一语言。

---

## 二、核心定义(L1)清单

| 定义名称 | 数学表述 | 层次 |
|---------|---------|-----|
| **链复形** | (C_•, d), d² = 0 | L1 |
| **同调群** | H_n(C) = ker d_n / im d_{n+1} | L1 |
| **导出函子** | L_nF, R^nF | L1 |
| **Tor** | Tor_n^R(M, N) | L1 |
| **Ext** | Ext_R^n(M, N) | L1 |
| **导出范畴** | D(A) = K(A)[Q^{-1}] | L1 |
| **谱序列** | (E_r^{p,q}, d_r) | L1 |
| **t-结构** | (D^{≤0}, D^{≥0}) | L1 |
