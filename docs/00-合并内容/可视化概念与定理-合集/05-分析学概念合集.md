# 哈代空间可视化

## 描述

哈代空间H^p是单位圆盘上满足特定增长条件的全纯函数空间。

## 可视化代码

```mermaid
graph TD
    A[单位圆盘 𝔻] --> B[H^p空间]
    B --> C[边界值 f* ∈ L^p]
    C --> D[泊松积分]

    E[H^∞] --> F[有界全纯函数]
end

```

## 参考

1. Duren, P. L. (1970). Theory of H^p Spaces.

# 巴拿赫空间理论

## 描述

巴拿赫空间是完备的赋范线性空间，是泛函分析的核心研究对象。

## 可视化

```mermaid
graph TD
    A[向量空间] --> B[范数 ||·||]
    B --> C[度量 d(x,y) = ||x-y||]

    C --> D[完备化]
    D --> E[巴拿赫空间]
    
    F[例子: L^p空间] --> G[ℓ^p空间]
end

```

## 参考

Rudin, W. (1991). Functional Analysis.

# 希尔伯特空间理论

## 描述

希尔伯特空间是完备的内积空间，是量子力学的数学基础。

## 可视化

```mermaid
graph TD
    A[内积空间] --> B[完备化]
    B --> C[希尔伯特空间 H]
    C --> D[正交基]
    D --> E[傅里叶展开]
end

```

## 参考

Reed, M., & Simon, B. (1980). Methods of Modern Mathematical Physics.
