# 同构定理 - 群同态分解

## 描述

群同态基本定理（第一同构定理）建立了群同态与商群之间的联系。

## 定理陈述

若 φ: G → H 是群同态，则 G/ker(φ) ≅ im(φ)。

## 可视化代码

```mermaid
graph TD
    A[G] -->|φ| B[H]
    C[G/kerφ] -->|同构| D[imφ ⊂ H]

    E[kerφ ⊲ G] --> F[正规子群]
end

```

## 参考

1. Dummit, D. S., & Foote, R. M. (2004). Abstract Algebra.

# 西罗定理完整版

## 描述

西罗三定理是有限群论中关于p子群的核心结果。

## 定理

1. 存在性: |G| = pⁿm，则存在pⁿ阶子群

2. 共轭性: 所有Sylow p子群共轭
3. n_p ≡ 1 mod p, n_p | m

## 可视化

```mermaid
graph TD
    A[西罗第一定理] --> B[p子群存在]
    C[西罗第二定理] --> D[p子群共轭]
    E[西罗第三定理] --> F[p子群计数]
end

```

## 参考

Dummit, D. S., & Foote, R. M. (2004). Abstract Algebra.

# 若尔当标准型定理

## 描述

任何复方阵都相似于唯一的若尔当标准型。

## 定理

A = PJP⁻¹, J为若尔当块对角阵。

## 可视化

```mermaid
graph TD
    A[矩阵 A] --> B[特征值 λ₁, λ₂, ...]
    B --> C[若尔当块 Jᵢ]
    C --> D[对角元 λ, 上次对角元 1]
end

```

## 参考

Hoffman, K., & Kunze, R. (1971). Linear Algebra.

# 希尔伯特基定理

## 描述

诺特环上的多项式环仍是诺特环。

## 定理

R 诺特 ⟹ R[x] 诺特。

## 可视化

```mermaid
graph TD
    A[诺特环 R] --> B[多项式环 R[x]]
    B --> C[升链条件]
    C --> D[理想有限生成]
end

```

## 参考

Atiyah, M. F., & Macdonald, I. G. (1969). Introduction to Commutative Algebra.
