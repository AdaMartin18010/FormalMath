# 同伦群序列可视化

## 描述

高阶同伦群 πₙ(X,x₀) 描述了n维球面到空间的映射同伦类。

## 可视化代码

```mermaid
graph LR
    A[π₁] --> B[基本群]
    C[π₂] --> D[二维洞]
    E[πₙ] --> F[n维洞]

    B --> G[非交换]
    D --> H[交换]
    F --> I[交换 n≥2]
end

```

## 参考

1. Hatcher, A. (2002). Algebraic Topology.

# CW复形结构

## 描述

CW复形是通过逐层粘贴胞腔构造的拓扑空间，是代数拓扑的主要研究对象。

## 可视化

```mermaid
graph TD
    A[0骨架 X⁰] --> B[粘贴1胞腔 X¹]
    B --> C[粘贴2胞腔 X²]
    C --> D[... Xⁿ]
    
    E[n维胞腔 eⁿ_α] --> F[粘贴映射 Sⁿ⁻¹ → Xⁿ⁻¹]
end

```

## 参考

Hatcher, A. (2002). Algebraic Topology.

# 层论基础

## 描述

层是拓扑空间上的一种数据结构，将代数对象系统地附着在空间的每一点上。

## 可视化

```mermaid
graph TD
    A[拓扑空间 X] --> B[层 F]
    B --> C[茎 F_x]
    B --> D[截面 F(U)]
    
    E[预层] --> F[满足粘合条件]
    F --> G[成为层]
end

```

## 参考

Hartshorne, R. (1977). Algebraic Geometry.

# 示性类理论

## 描述

示性类是将向量丛的拓扑信息映射到上同调类的函子性构造。

## 可视化

```mermaid
graph TD
    A[向量丛 E → M] --> B[示性类]
    B --> C[陈类 cᵢ(E) ∈ H²ⁱ(M;ℤ)]
    B --> D[庞特里亚金类 pᵢ(E)]
    B --> E[欧拉类 e(E)]
end

```

## 参考

Milnor, J. W., & Stasheff, J. D. (1974). Characteristic Classes.

# 莫尔斯理论

## 描述

莫尔斯理论研究流形上光滑函数的临界点与流形拓扑的关系。

## 可视化

```mermaid
graph TD
    A[光滑函数 f: M → ℝ] --> B[临界点 ∇f = 0]
    B --> C[黑塞矩阵非退化]
    C --> D[莫尔斯指标]
    D --> E[胞腔粘贴]
end

```

## 参考

Milnor, J. (1963). Morse Theory.
