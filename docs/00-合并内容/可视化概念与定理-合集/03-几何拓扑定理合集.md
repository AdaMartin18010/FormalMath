# 高斯-博内定理

## 描述

曲面的全曲率等于欧拉示性数的2π倍，连接了微分几何与拓扑。

## 定理

∫_M K dA = 2π χ(M)

## 可视化

```mermaid
graph TD
    A[高斯曲率 K] --> B[全曲率 ∫KdA]
    B --> C[2πχ(M)]
    C --> D[欧拉示性数]
    
    E[球面 K=1] --> F[χ=2, ∫KdA=4π]
    G[环面 K=0] --> H[χ=0, ∫KdA=0]
end

```

## 参考

do Carmo, M. P. (1992). Riemannian Geometry.

# 黎曼-罗赫定理

## 描述

紧致黎曼曲面上除子的维数公式。

## 定理

l(D) - l(K-D) = deg(D) + 1 - g

## 可视化

```mermaid
graph TD
    A[除子 D] --> B[l(D) = dim H⁰(D)]
    B --> C[deg(D)]
    C --> D[亏格 g]
    D --> E[黎曼-罗赫公式]
end

```

## 参考

Hartshorne, R. (1977). Algebraic Geometry.
