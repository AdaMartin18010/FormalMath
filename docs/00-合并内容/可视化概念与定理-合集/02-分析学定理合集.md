# 单调收敛定理

## 描述

单调收敛定理是勒贝格积分的基石，说明单调递增可测函数列的积分与极限可交换。

## 定理陈述

若 0 ≤ f₁ ≤ f₂ ≤ ... → f，则 ∫fₙ → ∫f。

## 可视化代码

```mermaid
graph TD
    A[f₁ ≤ f₂ ≤ f₃ ≤ ...] --> B[单调递增收敛]
    B --> C[∫ lim fₙ = lim ∫ fₙ]

    D[与黎曼积分的区别] --> E[允许逐点收敛]
end

```

## 参考

1. Rudin, W. (1987). Real and Complex Analysis.

# 控制收敛定理

## 描述

控制收敛定理给出了积分与极限交换的充分条件。

## 定理

若 |fₙ| ≤ g ∈ L¹ 且 fₙ → f a.e., 则 ∫fₙ → ∫f。

## 可视化

```mermaid
graph TD
    A[fₙ → f a.e.] --> B[控制函数 |fₙ| ≤ g]

    B --> C[lim ∫fₙ = ∫lim fₙ]
    
    D[反例: 无控制函数] --> E[结论不成立]
end

```

## 参考

Rudin, W. (1987). Real and Complex Analysis.

# 柯西积分公式 - 复平面积分可视化

## 描述

本可视化展示复分析中的柯西积分公式，说明全纯函数在圆盘内的值如何由其边界值完全确定。

## 数学概念

**柯西积分公式**: 若f在包含圆周C及其内部的区域全纯，则对C内部任意点z₀：
$$f(z_0) = \frac{1}{2\pi i} \oint_C \frac{f(z)}{z-z_0} dz$$

## 可视化代码

```mermaid
graph TB
    subgraph 柯西积分公式
    A[全纯函数 f] --> B[闭合路径 C]
    B --> C[内部点 z₀]

    D[公式] --> E[f(z₀) = 1/(2πi) ∮ f(z)/(z-z₀) dz]

    F[推论] --> G[高阶导数: f⁽ⁿ⁾(z₀) = n!/(2πi) ∮ f(z)/(z-z₀)^(n+1) dz]
    F --> H[均值性质: f(z₀) = 1/(2π) ∫₀^(2π) f(z₀+re^(iθ)) dθ]
    F --> I[最大模原理]
    end

    style A fill:#e3f2fd
    style E fill:#fff3e0
    style G fill:#e8f5e9

```

```

复平面上的柯西积分
═══════════════════════════════════════════════════════════════

积分路径示意:
───────────────────────────────────────────────────────────────

              Im
               │
               │    ╭──────╮
               │   ╱   C    ╲
               │  │  ●z₀    │
               │  │    ↑    │
               │   ╲   f(z) ╱
               │    ╰──────╯
               │
    ───────────┼───────────────→ Re
               │
               │

关键洞见:
┌─────────────────────────────────────────────────────────────┐
│  全纯函数在区域内部的值完全由边界值决定！                    │
│                                                             │
│  这意味着: 全纯函数不能随意变化，                            │
│  每一点的值都与周围所有点的值紧密关联                        │
└─────────────────────────────────────────────────────────────┘

证明思路:
1. 变形积分路径为以z₀为中心的小圆
2. 在小圆上 z = z₀ + re^(iθ)
3. 利用 f(z) ≈ f(z₀) + f'(z₀)(z-z₀) + ...
4. 计算留数得到结果

重要推论:
───────────────────────────────────────────────────────────────
• 全纯函数无穷可微
• 局部可展开为幂级数
• 零点的孤立性
• 刘维尔定理: 有界整函数必为常数

```

## 参考

1. Ahlfors, L. V. (1979). Complex Analysis.
2. Conway, J. B. (1978). Functions of One Complex Variable.

# 里斯表示定理

## 描述

希尔伯特空间上的连续线性泛函都可由内积表示。

## 定理

∀φ ∈ H*, ∃! y ∈ H, φ(x) = ⟨x,y⟩。

## 可视化

```mermaid
graph TD
    A[对偶空间 H*] --> B[同构 H]
    B --> C[φ ↔ y_φ]
    C --> D[φ(x) = ⟨x, y_φ⟩]
end

```

## 参考

Conway, J. B. (1990). A Course in Functional Analysis.
