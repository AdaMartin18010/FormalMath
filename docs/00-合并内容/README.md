## 合并说明

### 合并原则

1. **主题相关性**: 将同一主题或相关主题的小文件合并
2. **学习连贯性**: 合并后的文档应支持连续阅读
3. **保留原始文件**: 原始小文件保留在原位置，合并文档用于系统学习

### 文件大小对比

| 合并组 | 合并前平均大小 | 合并后大小 | 效果 |
|--------|--------------|-----------|------|
| L0-直观经验层 | ~1 KB/文件 | ~17 KB/文件 | 便于系统学习 |
| L3-理论建构层 | ~1 KB/文件 | ~19 KB/文件 | 便于系统学习 |
| 可视化思维导图 | ~10 KB/文件 | ~583 KB/文件 | 便于查阅 |
| 工作示例 | ~1.5 KB/文件 | ~40 KB/文件 | 便于参考 |

### 使用建议

1. **系统学习**: 使用合并后的文档进行系统学习
2. **快速查阅**: 合并文档提供了完整的目录索引
3. **原始参考**: 如需查看原始独立文件，仍可在原位置找到
