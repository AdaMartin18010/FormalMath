# L0-21: 长度测量

**概念编号**: L0-21  
**所属家族**: 量与测量  
**核心定位**: 一维量的度量

---

## 📌 直观描述

### 1.1 核心直观

长度测量是用**单位长度去量**物体的长短——看看包含多少单位。

> **日常表述**: "这根绳子有多长"、"用尺子量一下"

### 1.2 测量方法

| 方法 | 描述 | 示例 |
|-----|------|------|
| **直接测量** | 用尺子量 | 用卷尺量身高 |
| **间接测量** | 计算得出 | 用勾股定理求斜边 |
| **步测** | 用脚步估算 | 大约走了100步 |
| **比较测量** | 与已知比较 | 比这根绳子长 |

### 1.3 单位系统

```

常用长度单位:

公制:
  千米(km) → 米(m) → 厘米(cm) → 毫米(mm)
  
市制:
  里 → 丈 → 尺 → 寸
  
英制:
  英里(mile) → 码(yard) → 英尺(foot) → 英寸(inch)

```

---

|-----|---------|

| **建筑** | 房屋尺寸 |
| **运动** | 跑道长度 |
| **旅行** | 路程距离 |
| **制造** | 零件尺寸 |
