## 历史背景与核心思想

### 从高斯到现代代数数论

代数数论的诞生源于对Diophantine方程（特别是费马大定理）的研究。**卡尔·弗里德里希·高斯**在研究双二次互反律时引入了复整数（即高斯整数 $\mathbb{Z}[i]$），开创了用代数方法研究数论问题的先河。

> **核心洞察**: 代数数论的本质是用代数工具（域扩张、Galois理论、理想理论）研究数论问题。"理想数"的概念使得唯一分解定理在更广泛的代数整数环中得以恢复。

**历史发展时间线**:

| 年份 | 数学家 | 里程碑贡献 |
|------|--------|-----------|
| 1801 | 高斯 | 《算术研究》，二次型理论 |
| 1847 | Kummer | 理想数，分圆域 |
| 1871 | Dedekind | 理想理论，Dedekind域 |
| 1897 | Hilbert | 《数论报告》，类域论纲领 |
| 1920s | Artin, Takagi | 一般类域论 |
| 1950s | Tate, Iwasawa | 现代类域论 |

## 代数整数的环论结构

### 代数整数环的定义

**定义**: 复数 $\alpha$ 称为**代数整数**，如果它是某个首一整数系数多项式的根：
$$\alpha^n + a_{n-1}\alpha^{n-1} + \cdots + a_0 = 0, \quad a_i \in \mathbb{Z}$$

**定义**: 数域 $K$（$\mathbb{Q}$ 的有限扩张）的**整数环**定义为：
$$\mathcal{O}_K = \{\alpha \in K : \alpha \text{ 是代数整数}\}$$

**示例**:
- $K = \mathbb{Q}$: $\mathcal{O}_K = \mathbb{Z}$
- $K = \mathbb{Q}(i)$: $\mathcal{O}_K = \mathbb{Z}[i]$（高斯整数）
- $K = \mathbb{Q}(\sqrt{-5})$: $\mathcal{O}_K = \mathbb{Z}[\sqrt{-5}]$（无唯一分解）

### 定理1: 整数环是Dedekind域

**定理**: $\mathcal{O}_K$ 是Dedekind域，即满足：
1. Noether环
2. 整闭
3. 每个非零素理想都是极大理想

**证明**:

**步骤1: Noether性**

$\mathcal{O}_K$ 是秩为 $n = [K:\mathbb{Q}]$ 的自由 $\mathbb{Z}$-模。由于 $\mathbb{Z}$ 是Noether环，$\mathcal{O}_K$ 是Noether模，从而是Noether环。

**步骤2: 整闭性**

设 $\alpha \in K$ 在 $\mathcal{O}_K$ 上整。则 $\alpha$ 在 $\mathbb{Z}$ 上整（整性的传递性），故 $\alpha \in \mathcal{O}_K$。

**步骤3: 素理想极大性**

设 $\mathfrak{p} \subseteq \mathcal{O}_K$ 是非零素理想。则 $\mathfrak{p} \cap \mathbb{Z} = (p)$ 对某素数 $p$。

$\mathcal{O}_K/\mathfrak{p}$ 是 $\mathbb{F}_p = \mathbb{Z}/(p)$ 的有限扩张，从而是有限域，因此是域。故 $\mathfrak{p}$ 是极大理想。$\square$

## 理想分解与Dedekind域

### 唯一分解定理

**定理** (Dedekind): 设 $R$ 是Dedekind域，$\mathfrak{a} \subseteq R$ 是非零理想。则 $\mathfrak{a}$ 可唯一地分解为素理想的乘积：
$$\mathfrak{a} = \mathfrak{p}_1^{e_1} \cdots \mathfrak{p}_g^{e_g}$$

**证明概要**:

**存在性**:
1. 对任意非零理想 $\mathfrak{a}$，存在素理想 $\mathfrak{p}_1, \ldots, \mathfrak{p}_r$ 使得 $\mathfrak{p}_1 \cdots \mathfrak{p}_r \subseteq \mathfrak{a}$
2. 对包含 $\mathfrak{a}$ 的素理想个数进行归纳
3. 若 $\mathfrak{a}$ 是素理想，则已完成；否则分解 $\mathfrak{a} = \mathfrak{b}\mathfrak{c}$ 并应用归纳假设

**唯一性**:
1. 利用素理想的性质：若 $\mathfrak{p} \supseteq \mathfrak{ab}$，则 $\mathfrak{p} \supseteq \mathfrak{a}$ 或 $\mathfrak{p} \supseteq \mathfrak{b}$
2. 比较两边的分解，利用 $\mathfrak{p}^e \setminus \mathfrak{p}^{e+1} \neq \emptyset$ 确定指数

### 素理想分解

设 $K/\mathbb{Q}$ 是 $n$ 次数域，$\mathcal{O}_K$ 是其整数环。对素数 $p$，$(p)$ 在 $\mathcal{O}_K$ 中分解为：
$$(p) = \mathfrak{p}_1^{e_1} \cdots \mathfrak{p}_g^{e_g}$$

其中 $f_i = [\mathcal{O}_K/\mathfrak{p}_i : \mathbb{F}_p]$ 是剩余域次数。满足：
$$\sum_{i=1}^g e_i f_i = n$$

**定义**:
- $p$ **分歧**(ramifies)：若某 $e_i > 1$
- $p$ **完全分裂**(splits completely)：若 $g = n$，所有 $e_i = f_i = 1$
- $p$ **惰性**(inert)：若 $g = 1$，$e_1 = 1$，$f_1 = n$

### 定理2: 分歧判定

**定理**: 素数 $p$ 在 $K/\mathbb{Q}$ 中分歧当且仅当 $p \mid d_K$，其中 $d_K$ 是 $K$ 的判别式。

**证明概要**:

**步骤1**: $p$ 分歧 $\Leftrightarrow$ $\mathcal{O}_K/(p)$ 有幂零元

**步骤2**: 判别式 $d_K = \det(\text{Tr}(\omega_i\omega_j))$，其中 $\{\omega_i\}$ 是整基

**步骤3**: $p \mid d_K$ $\Leftrightarrow$ 模 $p$ 的迹形式退化 $\Leftrightarrow$ $\mathcal{O}_K/(p)$ 有幂零元$\square$

## 类域论基础

### 理想类群

**定义**: 数域 $K$ 的**理想类群**定义为：
$$\text{Cl}(K) = I_K / P_K$$

其中 $I_K$ 是分式理想群，$P_K$ 是主分式理想群。

**定理** (Minkowski): 理想类群是有限的。

**类数**: $h_K = |\text{Cl}(K)|$

**意义**: $h_K = 1$ $\Leftrightarrow$ $\mathcal{O}_K$ 是主理想整环 $\Leftrightarrow$ $\mathcal{O}_K$ 是唯一分解整环

### Hilbert类域

**定理** (Hilbert类域的存在性): 对任意数域 $K$，存在唯一的有限扩张 $H/K$ 使得：
1. $H/K$ 是Galois扩张且 $\text{Gal}(H/K) \cong \text{Cl}(K)$
2. $H/K$ 是非分歧的（在所有素理想处，包括无穷位）
3. $H$ 是满足上述性质的极大扩张

**意义**: Hilbert类域"消解"了 $K$ 的理想类，在 $H$ 中所有理想都变成主理想。

### Artin互反映射

设 $L/K$ 是数域的Galois扩张，$\mathfrak{p}$ 是 $K$ 的未分歧素理想。

**定义**: **Frobenius元素** $\left(\frac{L/K}{\mathfrak{p}}\right) \in \text{Gal}(L/K)$ 定义为：
$$\sigma(\alpha) \equiv \alpha^{N(\mathfrak{p})} \pmod{\mathfrak{P}}$$

对 $L$ 中任意素理想 $\mathfrak{P} \mid \mathfrak{p}$ 和 $\alpha \in \mathcal{O}_L$。

**Artin映射**: 对模 $L/K$ 的导子 $\mathfrak{f}$ 互素的理想 $\mathfrak{a} = \prod \mathfrak{p}_i^{e_i}$：
$$\left(\frac{L/K}{\mathfrak{a}}\right) = \prod \left(\frac{L/K}{\mathfrak{p}_i}\right)^{e_i}$$

**Artin互反律**: Artin映射诱导同构
$$I_K^{\mathfrak{f}} / P_K^{\mathfrak{f}} \cdot N_{L/K}(I_L^{\mathfrak{f}}) \cong \text{Gal}(L/K)^{ab}$$

## 重要定理的完整证明

### 定理3: Dirichlet单位定理

**定理**: 设 $K$ 是数域，$r_1$ 是实嵌入数，$r_2$ 是复嵌入对数。则单位群 $\mathcal{O}_K^\times$ 是有限生成Abel群：
$$\mathcal{O}_K^\times \cong \mu_K \times \mathbb{Z}^{r_1 + r_2 - 1}$$

其中 $\mu_K$ 是 $K$ 中单位根的群（有限循环群）。

**证明** (概要):

**步骤1: 对数嵌入**

定义对数嵌入 $\lambda: \mathcal{O}_K^\times \to \mathbb{R}^{r_1 + r_2}$：
$$\lambda(\alpha) = (\log|\sigma_1(\alpha)|, \ldots, \log|\sigma_{r_1+r_2}(\alpha)|)$$

**步骤2: 核的确定**

$\ker(\lambda) = \mu_K$（单位根群）。这是因为若 $|\sigma(\alpha)| = 1$ 对所有嵌入 $\sigma$，则 $\alpha$ 是单位根。

**步骤3: 像的离散性**

$\lambda(\mathcal{O}_K^\times)$ 是超平面 $\sum x_i = 0$ 中的离散子群。

**步骤4: 紧性论证**

利用Minkowski定理证明对数像生成一个格，秩为 $r_1 + r_2 - 1$。$\square$

### 定理4: Minkowski定理

**定理**: 设 $L \subset \mathbb{R}^n$ 是格，$\det(L) = \text{covolume}$。设 $S \subset \mathbb{R}^n$ 是凸对称集，体积 $\text{vol}(S) > 2^n \det(L)$。则 $S$ 包含 $L$ 的非零点。

**证明**:

**步骤1: 体积比较**

考虑基本域 $\mathcal{F} = \{t_1 v_1 + \cdots + t_n v_n : 0 \leq t_i < 1\}$，其中 $\{v_i\}$ 是 $L$ 的基。

**步骤2: 鸽巢原理**

若 $\frac{1}{2}S$ 的平移 $\{\frac{1}{2}S + v : v \in L\}$ 不交叠，则
$$\text{vol}\left(\bigcup_{v \in L} \left(\frac{1}{2}S + v\right) \cap \mathcal{F}\right) \leq \text{vol}(\mathcal{F}) = \det(L)$$

但左边 $= \text{vol}(\frac{1}{2}S) = 2^{-n}\text{vol}(S) > \det(L)$，矛盾。

**步骤3: 存在非零点**

存在 $v_1 \neq v_2 \in L$ 使得 $(\frac{1}{2}S + v_1) \cap (\frac{1}{2}S + v_2) \neq \emptyset$。

因此 $v_1 - v_2 \in S \cap L$。$\square$

## 与数学家理念体系的关联

### 高斯的代数数论起源

高斯对双二次互反律的研究引出了复整数 $\mathbb{Z}[i]$ 的概念。他认识到，在传统整数环中无法解决的问题（如某些Diophantine方程），在扩大的数环中可能变得可解。

**交叉引用**: [高斯数学理念/02-数学内容深度分析/02-数论贡献](../../数学家理念体系/高斯数学理念/)

### Kummer的理想数革命

Ernst Kummer为研究费马大定理引入了"理想数"(ideal number)的概念。这一思想后来由Dedekind发展为现代理想理论，彻底改变了代数的面貌。

### Hilbert的类域论纲领

David Hilbert在《数论报告》(Zahlbericht)和一系列论文中提出了类域论的完整纲领。他的工作展示了"用域扩张消解理想类"的深刻思想，是数论史上的里程碑。
