## 历史背景与发展脉络

### 从手工计算到量子时代

计算数论是数论与计算机科学的交叉学科，关注数论问题的算法设计与复杂度分析。随着计算机的发展，这一领域从理论研究走向实际应用，特别是在密码学中扮演核心角色。

> **核心洞察**: 数论中"困难问题"（如大整数分解、离散对数）的计算复杂性是现代密码学的安全基础。

## 素性测试算法

### 试除法与埃拉托斯特尼筛

**试除法**: 检验 $n$ 是否素数，只需试除至 $\sqrt{n}$。

**复杂度**: $O(\sqrt{n})$，仅适用于小整数。

**埃拉托斯特尼筛**: 生成不超过 $N$ 的所有素数。

**算法**:
```python
def sieve(n):
    is_prime = [True] * (n + 1)
    is_prime[0] = is_prime[1] = False
    for i in range(2, int(n**0.5) + 1):
        if is_prime[i]:
            for j in range(i*i, n+1, i):
                is_prime[j] = False
    return [i for i, prime in enumerate(is_prime) if prime]
```

**复杂度**: 时间 $O(n \log \log n)$，空间 $O(n)$。

### Fermat测试与伪素数

**Fermat测试**: 若 $n$ 是素数，$a^{n-1} \equiv 1 \pmod{n}$。

**伪素数**: 合数 $n$ 满足上述等式对某 $a$。

**Carmichael数**: 对所有与 $n$ 互素的 $a$ 都满足Fermat等式的合数（如561, 1105, 1729）。

### Miller-Rabin测试

**算法**:  probabilistic测试，错误概率可任意降低。

**步骤**:
1. 写 $n-1 = 2^r \cdot d$，$d$ 奇数
2. 随机选 $a \in [2, n-2]$
3. 计算 $x = a^d \pmod{n}$
4. 若 $x = 1$ 或 $x = n-1$，通过
5. 重复平方 $r-1$ 次，检查是否出现 $n-1$
6. 若从未出现，$n$ 是合数

**定理**: 对奇合数 $n$，随机 $a$ 通过测试的概率 $\leq 1/4$。

**应用**: 重复 $k$ 次，错误概率 $< 4^{-k}$。

### AKS素性测试

**定理** (Agrawal-Kayal-Saxena, 2002): 素性测试可以在确定性多项式时间内完成。

**算法概要**:
1. 检查 $n$ 是否是完全幂
2. 找最小的 $r$ 使得 $o_r(n) > \log^2 n$
3. 若 $1 < \gcd(a, n) < n$ 对某 $a \leq r$，$n$ 是合数
4. 若 $n \leq r$，$n$ 是素数
5. 检查 $(X + a)^n \equiv X^n + a \pmod{X^r - 1, n}$ 对 $a = 1, \ldots, \lfloor\sqrt{\phi(r)} \log n\rfloor$

**复杂度**: $O(\log^{12} n)$，后改进到 $O(\log^6 n)$。

## 整数分解方法

### Pollard's Rho算法

**思想**: 利用生日悖论，在随机序列中寻找碰撞。

**算法**:
```python
def pollard_rho(n):
    if n % 2 == 0:
        return 2
    x, y, d = 2, 2, 1
    f = lambda x: (x*x + 1) % n
    while d == 1:
        x = f(x)
        y = f(f(y))
        d = math.gcd(abs(x - y), n)
    return d if d != n else None
```

**复杂度**: 期望 $O(n^{1/4})$，适用于小因子。

### 椭圆曲线分解法 (ECM)

**Lenstra算法**: 利用椭圆曲线群的结构。

**步骤**:
1. 随机选择椭圆曲线 $E: y^2 = x^3 + ax + b$ 和点 $P$
2. 计算 $kP$ 对某光滑界 $B$
3. 若计算中需要求逆元模 $n$ 失败，得到因子

**复杂度**: 期望 $O(e^{(1+o(1))\sqrt{2 \ln p \ln \ln p}})$，其中 $p$ 是最小素因子。

**适用**: 找出中等大小（20-50位）的因子。

### 数域筛法 (NFS)

**目前最快的通用分解算法**。

**复杂度**: $L_n[1/3, c] = \exp((c + o(1))(\ln n)^{1/3}(\ln \ln n)^{2/3})$

**实际记录**:
- RSA-768 (232位): 2009年分解
- RSA-240 (795位): 2019年分解

## 离散对数问题

### 定义与重要性

**定义**: 在群 $G$ 中，给定 $g$ 和 $h = g^x$，求 $x$。

**重要性**: 
- Diffie-Hellman密钥交换
- ElGamal加密
- DSA数字签名
- 椭圆曲线密码学

### Baby-step Giant-step

**算法**: 
1. 设 $m = \lceil\sqrt{n}\rceil$
2. Baby steps: 计算 $g^j$，$j = 0, 1, \ldots, m-1$
3. Giant steps: 计算 $h \cdot g^{-mi}$，$i = 0, 1, \ldots, m-1$
4. 找到匹配 $g^j = h \cdot g^{-mi}$，则 $x = j + mi$

**复杂度**: 时间 $O(\sqrt{n})$，空间 $O(\sqrt{n})$。

### Pollard's Rho for DLP

**思想**: 类似分解的Rho方法，利用伪随机游走。

**复杂度**: 期望 $O(\sqrt{n})$，空间 $O(1)$。

### Index Calculus

**适用于乘法群 $\mathbb{F}_p^\times$**。

**步骤**:
1. 选择因子基 $S = \{p_1, \ldots, p_k\}$（小素数）
2. 收集关系：找 $a$ 使得 $g^a \mod p$ 是 $S$-光滑
3. 解线性方程组得到 $\log_g p_i$
4. 计算目标离散对数

**复杂度**: $L_p[1/3, c]$，亚指数时间。

## 椭圆曲线密码学

### 椭圆曲线的群结构

**Weierstrass方程**: $y^2 = x^3 + ax + b$，$4a^3 + 27b^2 \neq 0$

**群运算**: 几何定义的弦切法

**Hasse定理**: $\#E(\mathbb{F}_p) = p + 1 - t$，$|t| \leq 2\sqrt{p}$

### ECDLP

**椭圆曲线离散对数问题**: 给定 $P, Q = nP$，求 $n$。

**已知最佳算法**: 
- Pollard's Rho: $O(\sqrt{n})$
- Index calculus不适用（无好的因子基概念）

**安全参数**: 256位椭圆曲线 ≈ 3072位RSA

### 协议实例

**ECDH密钥交换**:
1. Alice选择私钥 $a$，发送公钥 $aG$
2. Bob选择私钥 $b$，发送公钥 $bG$
3. 共享密钥：$abG$

**ECDSA签名**:
1. 选择随机 $k$，计算 $R = kG$
2. $r = x_R \mod n$
3. $s = k^{-1}(h + rd) \mod n$
4. 签名: $(r, s)$

## 应用场景详解

### 应用1: TLS/SSL协议

**TLS握手**:
1. 客户端发送支持的密码套件
2. 服务器选择套件，发送证书（含RSA/ECDSA公钥）
3. 密钥交换（RSA加密或ECDH）
4. 生成会话密钥

**前向安全性**: 使用Ephemeral Diffie-Hellman，即使私钥泄露，过去的会话仍安全。

### 应用2: 区块链与加密货币

**比特币**:
- 地址: RIPEMD-160(SHA-256(公钥))
- 签名: ECDSA (secp256k1曲线)
- 私钥: 256位随机数

**以太坊**: 类似，使用Keccak-256哈希。

### 应用3: 零知识证明

**zk-SNARKs**: 利用椭圆曲线配对构造简洁非交互零知识证明。

**应用**: Zcash隐私交易、以太坊Layer 2扩容。

## 思维过程表征

### 计算数论的典型解决策略

**策略1: 随机化**
- 使用Las Vegas或Monte Carlo算法
- 重复测试降低错误概率

**策略2: 代数结构利用**
- 利用群、环、域的结构
- 将问题转化到更易处理的形式

**策略3: 光滑性**
- 寻找光滑数（只有小素因子的数）
- 利用因子基分解问题

## 形式化实现

```lean
-- 计算数论的Lean 4形式化
import Mathlib

-- 扩展欧几里得算法
def extendedGcd (a b : ℤ) : ℤ × ℤ × ℤ :=
  if b = 0 then (a, 1, 0)
  else
    let (g, x, y) := extendedGcd b (a % b)
    (g, y, x - (a / b) * y)

-- 模幂运算（快速幂）
def modPow (base exp mod : ℕ) : ℕ :=
  if exp = 0 then 1
  else if exp % 2 = 0 then
    let half := modPow base (exp / 2) mod
    (half * half) % mod
  else
    (base * modPow base (exp - 1) mod) % mod

-- Miller-Rabin测试
def millerRabinTest (n : ℕ) (k : ℕ) : Bool :=
  if n < 2 || n % 2 = 0 then n = 2
  else
    -- 写 n-1 = 2^r * d
    let r := (n - 1).factorization 2
    let d := (n - 1) / 2^r
    -- 重复 k 次测试
    List.range k |>.all (fun _ =>
      let a := Random.randNat 2 (n - 2)
      let x := modPow a d n
      if x = 1 || x = n - 1 then true
      else
        -- 重复平方检查
        let rec loop (i : ℕ) (x : ℕ) : Bool :=
          if i >= r - 1 then false
          else
            let x' := (x * x) % n
            if x' = n - 1 then true
            else loop (i + 1) x'
        loop 0 x)

-- Pollard's Rho
def pollardRho (n : ℕ) : Option ℕ :=
  if n % 2 = 0 then some 2
  else
    let rec loop (x y d : ℕ) : Option ℕ :=
      if d = 1 then
        let x' := (x * x + 1) % n
        let y' := (y * y + 1) % n
        let y'' := (y' * y' + 1) % n
        let d' := Nat.gcd (if x' > y'' then x' - y'' else y'' - x') n
        loop x' y'' d'
      else if d = n then none
      else some d
    loop 2 2 1
```

## 与数学家理念体系的关联

### Fermat的数论遗产

费马小定理是现代素性测试的基础。尽管费马从未给出证明，他提出的定理成为计算数论的基石。

**交叉引用**: [费马数学理念](../../数学家理念体系/费马数学理念/)

### RSA开创的公钥密码学

Rivest、Shamir和Adleman的RSA算法首次实现了实用的公钥密码系统。这一突破不仅革新了信息安全，也展示了数论抽象理论的实际价值。

### 椭圆曲线与代数几何

椭圆曲线密码学的发展受益于代数几何的深厚理论。Weil猜想、模形式等高等数学为椭圆曲线密码的安全性提供了理论基础。

**交叉引用**: [韦伊数学理念](../../数学家理念体系/韦伊数学理念/)
