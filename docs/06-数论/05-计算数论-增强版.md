## 1. 素性测试：从试除到概率算法

### 1.1 Fermat测试及其缺陷

**定理 1.1**（Fermat小定理）：若 $p$ 是素数，$a$ 满足 $\gcd(a, p) = 1$，则
$$a^{p-1} \equiv 1 \pmod{p}$$

**Fermat素性测试**：

1. 随机选取 $a \in \{2, \ldots, n-2\}$
2. 若 $a^{n-1} \not\equiv 1 \pmod{n}$，则 $n$ 是合数
3. 否则，$n$ **可能是**素数

**定义 1.2**（Fermat伪素数）：若 $n$ 合数但 $a^{n-1} \equiv 1 \pmod{n}$，称 $n$ 是以 $a$ 为底的**Fermat伪素数**。

**定义 1.3**（Carmichael数）：若 $n$ 合数且对所有满足 $\gcd(a, n) = 1$ 的 $a$，有 $a^{n-1} \equiv 1 \pmod{n}$，称 $n$ 为**Carmichael数**。

**例 1.4**：$561 = 3 \times 11 \times 17$ 是最小的Carmichael数。

验证：$\lambda(561) = \text{lcm}(2, 10, 16) = 80$，而 $560 = 7 \times 80$，故 $a^{560} \equiv 1 \pmod{561}$。

**问题**：Carmichael数无穷多（Alford-Granville-Pomerance，1994），Fermat测试不可靠。

### 1.2 Miller-Rabin测试详解

**关键观察**：若 $p$ 是奇素数，写 $p-1 = 2^s \cdot d$（$d$ 奇），则对任意 $a$：
$$a^d \equiv 1 \pmod{p}$$
或存在 $0 \leq r < s$ 使得
$$a^{2^r d} \equiv -1 \pmod{p}$$

**证明**：由Fermat小定理，$a^{2^s d} \equiv 1 \pmod{p}$。从 $a^d$ 开始连续平方，必在某步得到 $-1$ 或始终为1。

**定义 1.5**（强伪素数）：设 $n$ 奇合数，$n-1 = 2^s \cdot d$。若 $a$ 满足：

- $a^d \equiv 1 \pmod{n}$，或
- 存在 $0 \leq r < s$ 使得 $a^{2^r d} \equiv -1 \pmod{n}$

则称 $n$ 是以 $a$ 为底的**强伪素数**。

**Miller-Rabin算法**：

```
输入：奇数 n ≥ 3，安全参数 k
输出："合数" 或 "可能素数"

1. 写 n-1 = 2^s · d（d奇）
2. 重复k次：
   a. 随机选取 a ∈ {2, ..., n-2}
   b. x ← a^d mod n
   c. 若 x = 1 或 x = n-1，继续下一轮
   d. 重复 s-1 次：
      x ← x^2 mod n
      若 x = n-1，跳出继续下一轮
   e. 返回 "合数"
3. 返回 "可能素数"
```

**定理 1.6**（Rabin）：若 $n$ 是奇合数，则以随机 $a$ 为底的强伪素数概率 $< \frac{1}{4}$。

**推论**：进行 $k$ 轮测试，错误概率 $< 4^{-k}$。

**例 1.7**：测试 $n = 561$（Carmichael数）

- $560 = 2^4 \times 35$，$s = 4$，$d = 35$
- 取 $a = 2$：$2^{35} \equiv 263 \pmod{561}$
  - $263^2 \equiv 166 \pmod{561}$
  - $166^2 \equiv 67 \pmod{561}$
  - $67^2 \equiv 1 \pmod{561}$
- 从未达到 $-1 \equiv 560$，故561被识别为合数！

### 1.3 确定性测试：AKS算法

**定理 1.8**（Agrawal-Kayal-Saxena，2002）：存在确定性多项式时间素性测试算法。

**核心引理**：$n$ 是素数当且仅当在 $\mathbb{Z}[x]/(n, x^r - 1)$ 中：
$$(x + a)^n \equiv x^n + a \pmod{n, x^r - 1}$$

对所有 $1 \leq a \leq \sqrt{r} \log n$ 成立，其中 $r$ 适当选择。

**复杂度**：$O(\log^{12} n)$（后经改进到 $O(\log^{6} n)$）。

**注**：AKS理论重要，但实际中Miller-Rabin更常用（因更快）。

## 2. 因数分解算法

### 2.1 Pollard's p-1方法

**核心思想**：若 $n$ 有素因子 $p$ 使得 $p-1$ 是**光滑的**（所有素因子 $\leq B$），则可利用。

**Fermat小定理推论**：若 $p-1 | M$，则对所有满足 $\gcd(a, p) = 1$ 的 $a$，有 $a^M \equiv 1 \pmod{p}$。

**算法**：

1. 选择光滑界 $B$
2. 计算 $M = \text{lcm}(1, 2, \ldots, B)$ 或 $M = \prod_{p \leq B} p^{\lfloor \log_p n \rfloor}$
3. 随机选取 $a$，计算 $d = \gcd(a^M - 1, n)$
4. 若 $1 < d < n$，则 $d$ 是 $n$ 的因子

**复杂度**：$O(B \log^2 n)$，当 $p-1$ 是 $B$-光滑时成功。

**例 2.2**：分解 $n = 299 = 13 \times 23$

- $p = 13$：$p-1 = 12 = 2^2 \times 3$，对 $B = 3$ 光滑
- $M = \text{lcm}(1, 2, 3) = 6$
- 取 $a = 2$：$2^6 = 64$，$\gcd(63, 299) = \gcd(63, 299) = 1$
- 实际上需要 $M = 12$：$2^{12} = 4096 \equiv 92 \pmod{299}$，$\gcd(91, 299) = 13$ ✓

### 2.2 Pollard's Rho算法详解

**核心思想**：利用**生日悖论**在伪随机序列中寻找碰撞。

**算法描述**：

定义多项式映射 $f(x) = x^2 + c \pmod{n}$（通常 $c = 1$）。

从 $x_0 = 2$ 开始，生成两个序列：

- **龟**（慢）：$x_{i+1} = f(x_i)$
- **兔**（快）：$y_{i+1} = f(f(y_i))$

计算 $d = \gcd(|x_i - y_i|, n)$。

**理论分析**：

设 $p$ 是 $n$ 的最小素因子。序列 $\{x_i\}$ 模 $p$ 最终会进入周期（$\rho$ 形）。

**定理 2.3**：期望 $O(\sqrt{p})$ 步内找到碰撞，即 $x_i \equiv y_i \pmod{p}$。

**证明思路**：

- 序列模 $p$ 随机，由生日悖论，$O(\sqrt{p})$ 步内碰撞
- 碰撞时 $p | (x_i - y_i)$，但未必 $n | (x_i - y_i)$
- $\gcd(|x_i - y_i|, n)$ 给出非平凡因子

**Floyd循环检测**：

```python
def pollard_rho(n, c=1):
    if n % 2 == 0:
        return 2
    x, y, d = 2, 2, 1
    f = lambda x: (pow(x, 2, n) + c) % n
    while d == 1:
        x = f(x)
        y = f(f(y))
        d = math.gcd(abs(x - y), n)
    if d == n:
        return None  # 失败，尝试其他c
    return d
```

**复杂度**：$O(n^{1/4})$，适用于 $n < 10^{20}$。

**例 2.4**：分解 $n = 8051$

- $f(x) = x^2 + 1 \pmod{8051}$
- $(x_0, y_0) = (2, 2)$
- $x_1 = 5$，$y_1 = f(f(2)) = f(5) = 26$
- $\gcd(21, 8051) = 1$
- $x_2 = 26$，$y_2 = f(f(26)) = f(677) = 7474$
- $\gcd(7448, 8051) = 1$
- ...
- $x_3 = 677$，$y_3 = f(f(7474)) = 871$
- $\gcd(194, 8051) = 97$ ✓

故 $8051 = 97 \times 83$。

### 2.3 二次筛法与数域筛法

**二次筛法（QS）**：

寻找 $x, y$ 使得 $x^2 \equiv y^2 \pmod{n}$ 但 $x \not\equiv \pm y \pmod{n}$，则 $\gcd(x-y, n)$ 给出因子。

**步骤**：

1. 选择因子基 $F = \{p_1, \ldots, p_k\}$（小素数）
2. 寻找光滑数：$x_i^2 \equiv \prod_{j} p_j^{e_{ij}} \pmod{n}$
3. 寻找线性关系使指数全为偶数
4. 构造 $x^2 \equiv y^2 \pmod{n}$

**复杂度**：$L_n[1/2, 1] = \exp((1+o(1))\sqrt{\log n \log\log n})$

**数域筛法（NFS）**：

目前最快的通用算法，复杂度 $L_n[1/3, c]$。

**记录**：2020年，RSA-250（250位十进制数，829比特）被分解。

## 3. 椭圆曲线方法

### 3.1 Lenstra椭圆曲线分解

**核心思想**：Pollard's p-1 利用群 $\mathbb{Z}_p^*$（阶 $p-1$）。椭圆曲线方法利用**椭圆曲线群**，其阶在 $p + 1 \pm 2\sqrt{p}$ 范围内变化。

**椭圆曲线定义**：设 $p > 3$ 素数，$a, b \in \mathbb{F}_p$，$4a^3 + 27b^2 \not\equiv 0 \pmod{p}$：
$$E: y^2 \equiv x^3 + ax + b \pmod{p}$$

**群运算**：点加法（弦切线法）

**Hasse定理**：$|E(\mathbb{F}_p)| = p + 1 - t$，其中 $|t| \leq 2\sqrt{p}$。

**Lenstra算法**：

1. 随机选择椭圆曲线 $E$ 和点 $P \in E(\mathbb{Z}_n)$
2. 计算 $k! \cdot P$（利用椭圆曲线群运算）
3. 若在模某因子 $p|n$ 下 $E(\mathbb{F}_p)$ 的阶是光滑的，则计算中需要求逆时会遇到零因子
4. $\gcd$ 给出因子

**复杂度**：$O(\exp((\sqrt{2} + o(1))\sqrt{\log p \log\log p}))$，对小因子特别有效。

**优势**：通过更换曲线，"尝试不同的群阶"。

### 3.2 椭圆曲线素性证明

**Goldwasser-Kilian定理**：利用带复乘的椭圆曲线可构造素性证书。

**Atkin-Morain算法**：实际使用的椭圆曲线素性证明方法，对数千位数的素数有效。

## 4. 应用：现代密码学

### 4.1 RSA安全性分析

**RSA依赖**：大整数分解的困难性。

**安全参数**：

| 密钥长度 | 推荐年限 | 算法 |
|---------|---------|------|
| 1024位 | 已不安全 | QS/NFS |
| 2048位 | 2030年前 | NFS |
| 3072位 | 长期安全 | NFS |

**攻击类型**：

- **因子分解**：直接攻击
- **共模攻击**：同一模数不同公钥
- **低指数攻击**：小公钥指数 $e$
- **侧信道攻击**：功耗分析、时序分析

### 4.2 椭圆曲线密码学

**优势**：

- 相同安全级别下密钥更短（256位ECC ≈ 3072位RSA）
- 计算效率更高

**ECDLP**：给定 $P, Q \in E(\mathbb{F}_q)$，求 $k$ 使得 $Q = kP$。

**最佳已知算法**：Pollard's rho，复杂度 $O(\sqrt{n})$。

**应用**：

- **ECDSA**：数字签名算法（比特币、TLS）
- **ECDH**：密钥交换协议
- **配对密码学**：基于Weil配对的先进协议
