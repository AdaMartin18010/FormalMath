## 一、加法组合基础

组合数论研究整数集的**组合性质**，关注加法、乘法结构在集合上的表现。

### 1.1 和集与差集

**定义 1.1**（和集与差集）：对 $A, B \subseteq \mathbb{Z}$，

- **和集**：$A + B = \{a + b : a \in A, b \in B\}$
- **差集**：$A - B = \{a - b : a \in A, b \in B\}$
- **k倍和**：$kA = A + A + \cdots + A$（$k$ 个副本）

**例 1.2**：

- $A = \{1, 2, 3\}$，则 $A + A = \{2, 3, 4, 5, 6\}$，$|A + A| = 5$
- 若 $A = \{1, 2, 4, 8, \ldots, 2^{n-1}\}$，则 $|A + A| = \binom{n}{2} + n = \frac{n(n+1)}{2}$（所有和唯一）

**定理 1.3**（和集大小界限）：对有限集 $A \subseteq \mathbb{Z}$，
$$2|A| - 1 \leq |A + A| \leq \binom{|A|+1}{2}$$

左等号成立当且仅当 $A$ 是**等差数列**。

**证明**：设 $A = \{a_1 < a_2 < \cdots < a_n\}$。

- 下界：$a_1 + a_1 < a_1 + a_2 < \cdots < a_1 + a_n < a_2 + a_n < \cdots < a_n + a_n$，至少有 $2n-1$ 个不同元素
- 上界：显然不超过 $\binom{n}{2} + n$

**定义 1.4**（加法能量）：
$$E(A) = |\{(a_1, a_2, a_3, a_4) \in A^4 : a_1 + a_2 = a_3 + a_4\}|$$

**定理 1.5**（Cauchy-Schwarz）：
$$|A + A| \geq \frac{|A|^4}{E(A)}$$

$E(A)$ 大意味着 $A$ 有"加法结构"（如等差数列）。

### 1.2 Plünnecke-Ruzsa不等式

**定理 1.6**（Plünnecke-Ruzsa）：设 $A$ 是有限阿贝尔群中的子集，$|A + A| \leq K|A|$。则对任意非负整数 $m, n$：
$$|mA - nA| \leq K^{m+n}|A|$$

**意义**：如果一个集合的和集较小（有结构），则其多重和/差集也较小。

**例 1.7**：若 $A$ 是等差数列，$|A| = n$，则 $|A + A| = 2n-1 \approx 2|A|$，$K \approx 2$。

$$|3A| = 3|A| - 2 \leq K^3 |A| = 8|A|$$

实际更紧：$|3A| = 3n-2 \leq 8n$ 显然成立。

## 二、算术级数与Szemerédi定理

### 2.1 van der Waerden定理

**定理 2.1**（van der Waerden，1927）：对任意正整数 $k, r$，存在 $W(k, r)$ 使得 $\{1, 2, \ldots, W(k, r)\}$ 的任意 $r$-染色中，必存在长度为 $k$ 的单色等差数列。

**例 2.2**：$W(3, 2) = 9$。对 $\{1, \ldots, 9\}$ 的任意2-染色，必有同色3项等差数列。

证明：由鸽巢原理，考虑1, 2, 4, 5, 8的染色...（组合论证）。

**注**：$W(k, r)$ 增长极快，$W(3, 3) = 27$，$W(4, 2) = 35$，$W(5, 2) = 178$。

### 2.2 Szemerédi定理

**定理 2.3**（Szemerédi，1975）：对任意整数 $k \geq 1$ 和实数 $\delta > 0$，存在 $N_0(k, \delta)$ 使得对任意 $N \geq N_0$，任意子集 $A \subseteq \{1, \ldots, N\}$ 满足 $|A| \geq \delta N$，则 $A$ 必包含长度为 $k$ 的等差数列。

**历史背景**：

- $k = 1, 2$：平凡
- $k = 3$：Roth（1953），使用**Fourier分析**
- $k = 4$：Szemerédi（1969）
- 一般 $k$：Szemerédi（1975），组合方法

**密度解释**：任何具有正上密度（$\limsup |A \cap [1,N]|/N > 0$）的整数集必包含任意长等差数列。

**推论 2.4**（Erdős-Turán猜想的部分解决）：若 $\sum_{a \in A} 1/a$ 发散，则 $A$ 包含任意长等差数列。

（注：Green-Tao定理（2004）证明了素数包含任意长等差数列，但Erdős-Turán猜想至今未完全解决。）

### 2.3 证明思路：Szemerédi正则性引理

**定理 2.5**（Szemerédi正则性引理）：对任意 $\varepsilon > 0$，存在 $M(\varepsilon)$ 使得任意图 $G$ 的顶点集可划分为 $k \leq M$ 个部分 $V_1, \ldots, V_k$，满足：

- 除最多 $\varepsilon k^2$ 对外，$(V_i, V_j)$ 是**$\varepsilon$-正则的**

**定义**（$\varepsilon$-正则对）：$(A, B)$ 称为**$\varepsilon$-正则**，若对所有 $X \subseteq A$，$Y \subseteq B$，$|X| \geq \varepsilon|A|$，$|Y| \geq \varepsilon|B|$，有
$$|d(X, Y) - d(A, B)| < \varepsilon$$

其中 $d(A, B) = \frac{e(A, B)}{|A||B|}$ 是边密度。

**证明Szemerédi定理的策略**：

1. **密度增量**：若 $A$ 无 $k$-AP，则在某子结构上密度增加
2. **能量递增**：通过正则性引理，构造能量函数，证明有限步后矛盾
3. **Fourier分析**（Roth方法）：对 $k = 3$，利用Fourier系数检测等差数列

**Roth定理证明框架**：

设 $A \subseteq \mathbb{Z}/N\mathbb{Z}$，$|A| = \delta N$。定义

$$\hat{A}(r) = \sum_{a \in A} e^{-2\pi i ar/N}$$

$k$-AP计数：
$$T_k(A) = \frac{1}{N^2} \sum_{r} \hat{A}(r)^k \hat{A}(-2r)^{k-1} \cdots$$

对 $k = 3$：
$$T_3(A) = \frac{|A|^3}{N^2} + \frac{1}{N^2} \sum_{r \neq 0} \hat{A}(r)^2 \hat{A}(-2r)$$

若 $A$ 无3-AP，则 $T_3(A) = 0$，导出 $\hat{A}(r)$ 大，意味着 $A$ 在某等差数列上密度增加。

## 三、素数模式：格林-陶定理

### 3.1 定理陈述与意义

**定理 3.1**（Green-Tao，2004）：素数集包含任意长的等差数列。

即：对任意 $k \geq 3$，存在无穷多个 $k$ 项等差数列，其各项均为素数。

**例 3.2**：

- $k = 3$：$(3, 5, 7)$，$(7, 13, 19)$，等
- $k = 4$：$(5, 11, 17, 23)$，$(61, 67, 73, 79)$，等
- $k = 5$：$(5, 11, 17, 23, 29)$
- 目前已知最长：26项，由素数 $43,142,746,595,714,191 + 23,681,770,333,645,690 \cdot n$（$n = 0, \ldots, 25$）构成

**意义**：

- 素数在整体上是"伪随机"的（Dirichlet定理、素数定理）
- 但具有足够的"结构"以包含长等差数列
- 开启了**高阶Fourier分析**的发展

### 3.2 证明框架

**核心思想**：将素数嵌入一个"伪随机"的集合中，然后应用Szemerédi定理的相对版本。

**步骤 1**：构造权重函数

设 $\Lambda(n)$ 是von Mangoldt函数。构造修改版本（W-trick）：
$$\tilde{\Lambda}(n) = \frac{\phi(W)}{W} \Lambda(Wn + b)$$

其中 $W = \prod_{p \leq w} p$ 是素数阶乘，$b$ 与 $W$ 互素。

**目的**：消除小素数的影响，使剩余序列在算术级数上均匀分布。

**步骤 2**：伪随机性验证

需要验证 $\tilde{\Lambda}$ 满足**线性形式条件**：对任意有界线性形式，
$$\mathbb{E}[\tilde{\Lambda}(L_1(n)) \cdots \tilde{\Lambda}(L_k(n))] \approx 1$$

这基于**GPY筛法**和**Goldston-Pintz-Yıldırım**的工作。

**步骤 3**：相对Szemerédi定理

**定理 3.3**（Green-Tao）：设 $S \subseteq \mathbb{Z}_N$ 满足伪随机性条件，$A \subseteq S$，$|A| \geq \delta |S|$。则 $A$ 包含长等差数列。

**步骤 4**：应用到素数

- 素数在修改后的权重下有正密度
- 满足伪随机性（由Siegel-Walfisz定理和解析数论结果）
- 因此包含任意长等差数列

## 四、筛法：从Eratosthenes到现代

筛法是通过**逐步排除**来研究素数分布的工具。

### 4.1 容斥原理与Eratosthenes筛

**基本设置**：要计数区间 $[1, x]$ 中的素数，排除被小素数整除的数。

**容斥公式**：
$$\pi(x) - \pi(\sqrt{x}) + 1 = \sum_{d | P} \mu(d) \left\lfloor \frac{x}{d} \right\rfloor$$

其中 $P = \prod_{p \leq \sqrt{x}} p$，$\mu$ 是Möbius函数。

**问题**：误差项累积！$|\lfloor x/d \rfloor - x/d| \leq 1$，总误差 $\approx 2^{\pi(\sqrt{x})}$，远大于主项。

### 4.2 Brun组合筛

**核心思想**：截断容斥，控制误差。

**定理 4.1**（Brun，1919）：设 $A$ 是有限整数集，$P$ 是素数集。对 $z > 0$，令 $P(z) = \prod_{p \leq z, p \in P} p$。

$$S(A, P, z) = |\{a \in A : \gcd(a, P(z)) = 1\}|$$

Brun筛给出上下界：
$$S(A, P, z) \leq X \prod_{p \leq z}(1 - g(p)) + \text{error}$$

**应用**：Brun证明
$$\pi_2(x) = O\left(\frac{x}{\log^2 x}\right)$$

其中 $\pi_2(x)$ 是不超过 $x$ 的孪生素数对数（$p, p+2$ 皆素数）。

### 4.3 Selberg筛

**核心思想**：用二次型优化替代线性容斥。

**定理 4.2**（Selberg上界筛）：设 $\lambda_1 = 1$，$\lambda_d$（$d \leq z$）任意。则
$$S(A, P, z) \leq \sum_{d_1, d_2 \leq z} \lambda_{d_1}\lambda_{d_2} \frac{|A|}{[d_1, d_2]} + \text{error}$$

选择 $\lambda_d$ 最小化主项（解线性方程组）。

**例 4.3**（孪生素数上界）：

$$\pi_2(x) \leq (16 + o(1)) \prod_{p > 2} \left(1 - \frac{1}{(p-1)^2}\right) \frac{x}{\log^2 x}$$

其中常数 $16$（后经改进）来自优化选择。

## 五、开放问题与前沿

| 问题 | 状态 |
|-----|------|
| **Erdős等差数列猜想** | 未解决：$\sum_{a \in A} 1/a = \infty \Rightarrow$ 任意长AP |
| **孪生素数猜想** | 未解决：无穷多对 $(p, p+2)$ |
| **Goldbach猜想** | 未解决：每个偶数 $> 2$ 是两素数和 |
| **素数k元组猜想** | 未解决：线性方程组在素数中的解 |
| **Furstenberg-Sárközy定理** | 已解决：无平方子集中两元素差为平方数 |

**近期进展**：

- **张益唐（2013）**：存在无穷多对素数差 $< 70,000,000$（现改进到246）
- **Helfgott（2013）**：弱Goldbach猜想（三素数定理）完全解决
- **Maynard-Tao**：独立发展高维筛法，简化张益唐证明
