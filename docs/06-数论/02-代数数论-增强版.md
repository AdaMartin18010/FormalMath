## 1. 代数数域的基本构造

### 1.1 从二次域入手

**定义 1.1**（二次域）：设 $d \neq 0, 1$ 是无平方因子的整数，则
$$K = \mathbb{Q}(\sqrt{d}) = \{a + b\sqrt{d} : a, b \in \mathbb{Q}\}$$
称为**二次数域**。当 $d > 0$ 时为**实二次域**，当 $d < 0$ 时为**虚二次域**。

**例 1.2**（两个典型二次域）：

**(a)** $K = \mathbb{Q}(\sqrt{2})$ —— 实二次域的典型代表。

- **嵌入（Embeddings）**：由于 $\sqrt{2}$ 的共轭为 $\pm\sqrt{2}$，存在两个嵌入：
  $$\sigma_1: a + b\sqrt{2} \mapsto a + b\sqrt{2} \quad \text{(恒等)}$$
  $$\sigma_2: a + b\sqrt{2} \mapsto a - b\sqrt{2} \quad \text{(共轭)}$$

- **范数与迹**：对 $\alpha = a + b\sqrt{2}$，
  $$N_{K/\mathbb{Q}}(\alpha) = \sigma_1(\alpha)\sigma_2(\alpha) = a^2 - 2b^2$$
  $$\text{Tr}_{K/\mathbb{Q}}(\alpha) = \sigma_1(\alpha) + \sigma_2(\alpha) = 2a$$

**(b)** $K = \mathbb{Q}(\sqrt{-5})$ —— 虚二次域，类数大于1的经典例子。

- 嵌入：$\sigma_1(\sqrt{-5}) = \sqrt{-5}$，$\sigma_2(\sqrt{-5}) = -\sqrt{-5}$
- 范数：$N(a + b\sqrt{-5}) = a^2 + 5b^2$

### 1.2 代数整数环

**定义 1.3**（代数整数）：元素 $\alpha \in K$ 称为**代数整数**，如果它是首一整系数多项式的根。

**定理 1.4**（二次域的整数环）：设 $K = \mathbb{Q}(\sqrt{d})$，其中 $d$ 无平方因子。

$$
\mathcal{O}_K = \begin{cases}
\mathbb{Z}[\sqrt{d}] & \text{若 } d \equiv 2, 3 \pmod{4} \\[6pt]
\mathbb{Z}\left[\frac{1+\sqrt{d}}{2}\right] & \text{若 } d \equiv 1 \pmod{4}
\end{cases}
$$

**证明思路**：设 $\alpha = a + b\sqrt{d}$ 是代数整数，$a, b \in \mathbb{Q}$。其极小多项式为
$$X^2 - 2aX + (a^2 - db^2) = 0$$

要求系数为整数，故 $2a \in \mathbb{Z}$，$a^2 - db^2 \in \mathbb{Z}$。

- 若 $a \in \mathbb{Z}$，则 $db^2 \in \mathbb{Z}$，结合 $b$ 分母分析得 $b \in \mathbb{Z}$
- 若 $a = \frac{m}{2}$，$m$ 奇数，则需要 $d \equiv 1 \pmod{4}$，此时 $b = \frac{n}{2}$，$n$ 奇数

**例 1.5**：

- $\mathcal{O}_{\mathbb{Q}(\sqrt{2})} = \mathbb{Z}[\sqrt{2}]$（因 $2 \equiv 2 \pmod{4}$）
- $\mathcal{O}_{\mathbb{Q}(\sqrt{-3})} = \mathbb{Z}[\omega]$，其中 $\omega = \frac{-1+\sqrt{-3}}{2}$（因 $-3 \equiv 1 \pmod{4}$）
- $\mathcal{O}_{\mathbb{Q}(\sqrt{-5})} = \mathbb{Z}[\sqrt{-5}]$（因 $-5 \equiv 3 \pmod{4}$）

## 2. 理想理论：拯救唯一分解

代数数论诞生的核心动机之一是解决**唯一分解定理的失效**。

**反例 2.1**（在 $\mathbb{Z}[\sqrt{-5}]$ 中）：
$$6 = 2 \cdot 3 = (1+\sqrt{-5})(1-\sqrt{-5})$$

验证这四个因子都是不可约元但不相伴：

- $N(2) = 4$，$N(3) = 9$，$N(1\pm\sqrt{-5}) = 6$
- 若 $2 = \alpha\beta$，则 $N(\alpha)N(\beta) = 4$，但 $a^2 + 5b^2 = 2$ 无整数解

### 2.1 理想分解定理

**定义 2.2**（理想）：子集 $I \subseteq \mathcal{O}_K$ 称为**理想**，如果：

- $(I, +)$ 是 $(\mathcal{O}_K, +)$ 的子群
- 对任意 $r \in \mathcal{O}_K$，$x \in I$，有 $rx \in I$

**定理 2.3**（戴德金，理想唯一分解）：设 $K$ 是数域，$\mathcal{O}_K$ 是其整数环。则：

1. 每个非零真理想可唯一分解为素理想的乘积
2. 对任意非零理想 $\mathfrak{a}, \mathfrak{b}$，存在唯一 $\mathfrak{c}$ 使得 $\mathfrak{a}\mathfrak{c} = \mathfrak{b}$

**证明框架**：

- 关键：证明 $\mathcal{O}_K$ 是**戴德金整环**（Noetherian、整闭、维数1）
- 利用迹配对证明理想逆存在：对理想 $\mathfrak{a}$，定义
  $$\mathfrak{a}^{-1} = \{x \in K : x\mathfrak{a} \subseteq \mathcal{O}_K\}$$
- 证明 $\mathfrak{a} \cdot \mathfrak{a}^{-1} = \mathcal{O}_K$

### 2.2 详细示例：$\mathbb{Q}(\sqrt{-5})$

在 $\mathcal{O}_K = \mathbb{Z}[\sqrt{-5}]$ 中，考察主理想 $(6)$ 的分解。

**步骤 1**：分析素数在 $\mathcal{O}_K$ 中的分解行为

对素数 $p$，研究 $(p)$ 在 $\mathcal{O}_K$ 中的分解。利用**克罗内克符号** $(d/p)$：

- $(p) = \mathfrak{p}^2$（分歧）：若 $p \mid d$ 或 $p = 2, d \equiv 3 \pmod{4}$
- $(p) = \mathfrak{p}_1\mathfrak{p}_2$（分裂）：若 $(d/p) = 1$
- $(p)$ 保持素理想（惰性）：若 $(d/p) = -1$

**步骤 2**：具体分解

对 $K = \mathbb{Q}(\sqrt{-5})$，$d = -5$：

- $p = 2$：$d \equiv 3 \pmod{4}$，故 $(2)$ 分歧
  $$(2) = \mathfrak{p}_2^2, \quad \mathfrak{p}_2 = (2, 1+\sqrt{-5})$$

- $p = 3$：$(-5/3) = (-2/3) = (-1/3)(2/3) = (-1)(-1) = 1$，故 $(3)$ 分裂
  $$(3) = \mathfrak{p}_3 \cdot \mathfrak{p}_3', \quad \mathfrak{p}_3 = (3, 1+\sqrt{-5}), \quad \mathfrak{p}_3' = (3, 1-\sqrt{-5})$$

- $p = 5$：$5 \mid d$，故 $(5)$ 分歧
  $$(5) = \mathfrak{p}_5^2, \quad \mathfrak{p}_5 = (\sqrt{-5})$$

**步骤 3**：验证理想分解拯救了唯一性

$$(6) = (2)(3) = \mathfrak{p}_2^2 \cdot \mathfrak{p}_3 \cdot \mathfrak{p}_3'$$

同时：

- $(1+\sqrt{-5}) = \mathfrak{p}_2 \cdot \mathfrak{p}_3$（验证：范数为6）
- $(1-\sqrt{-5}) = \mathfrak{p}_2 \cdot \mathfrak{p}_3'$（范数也为6）

因此：
$$(1+\sqrt{-5})(1-\sqrt{-5}) = \mathfrak{p}_2^2 \cdot \mathfrak{p}_3 \cdot \mathfrak{p}_3' = (6)$$

**关键观察**：元素层面的分解不唯一，但理想层面的分解唯一！

## 3. 类群：测量分解失败的指标

**定义 3.1**（分式理想与类群）：

- **分式理想**：$K$ 的 $\mathcal{O}_K$-子模 $I$，存在 $r \in \mathcal{O}_K$ 使得 $rI \subseteq \mathcal{O}_K$
- **主分式理想**：形如 $(\alpha) = \alpha\mathcal{O}_K$ 的理想，$\alpha \in K^*$
- **理想类群**：$\text{Cl}(K) = \{\text{分式理想}\} / \{\text{主分式理想}\}$
- **类数**：$h_K = |\text{Cl}(K)|$

**定理 3.2**（类群有限性，戴德金）：对任意数域 $K$，类群 $\text{Cl}(K)$ 是有限群。

**意义**：

- $h_K = 1$ 当且仅当 $\mathcal{O}_K$ 是主理想整环（即唯一分解整环）
- 类群"测量"了唯一分解定理失效的程度

### 3.1 类数的计算思路

**步骤 1**：确定Minkowski界

**定理 3.3**（Minkowski界）：每个理想类包含一个整理想 $\mathfrak{a}$ 满足
$$N(\mathfrak{a}) \leq M_K = \frac{n!}{n^n}\left(\frac{4}{\pi}\right)^{r_2}\sqrt{|d_K|}$$

其中 $n = [K:\mathbb{Q}]$，$r_2$ 是复嵌入对数，$d_K$ 是判别式。

**步骤 2**：对二次域的具体计算

对 $K = \mathbb{Q}(\sqrt{d})$，$d < -5$：
$$M_K = \frac{2}{\pi}\sqrt{|d_K|}$$

**例 3.4**（计算 $\mathbb{Q}(\sqrt{-5})$ 的类数）：

- $d_K = -20$（因 $-5 \equiv 3 \pmod{4}$，$d_K = 4d = -20$）
- $M_K = \frac{2}{\pi}\sqrt{20} \approx 2.85$

因此只需考察范数 $\leq 2$ 的素理想：

- $(2) = \mathfrak{p}_2^2$，$N(\mathfrak{p}_2) = 2$

检查 $\mathfrak{p}_2 = (2, 1+\sqrt{-5})$ 是否主理想：

- 若 $\mathfrak{p}_2 = (\alpha)$，则 $N(\alpha) = 2$
- 但 $a^2 + 5b^2 = 2$ 无整数解

因此 $[\mathfrak{p}_2] \neq [\mathcal{O}_K]$，且 $[\mathfrak{p}_2]^2 = [(2)] = [\mathcal{O}_K]$（因 $(2)$ 主理想）

**结论**：$\text{Cl}(K) \cong \mathbb{Z}/2\mathbb{Z}$，$h_K = 2$

## 4. 单位群与Dirichlet单位定理

**定理 4.1**（Dirichlet单位定理）：设 $K$ 是数域，$[K:\mathbb{Q}] = n$，有 $r_1$ 个实嵌入，$r_2$ 对复嵌入。则
$$\mathcal{O}_K^* \cong \mu(K) \times \mathbb{Z}^{r_1 + r_2 - 1}$$

其中 $\mu(K)$ 是 $K$ 中单位根群。

**解读**：

- 单位群是有限生成阿贝尔群
- **自由秩**：$r = r_1 + r_2 - 1$
- **挠部分**：单位根（如 $K$ 包含的本原单位根）

### 4.1 实二次域的单位

对实二次域 $K = \mathbb{Q}(\sqrt{d})$，$d > 0$：

- $r_1 = 2$，$r_2 = 0$，故 $r = 1$
- $\mathcal{O}_K^* \cong \{\pm 1\} \times \mathbb{Z}$

**定义 4.2**（基本单位）：单位群生成元 $\varepsilon > 1$ 称为**基本单位**。

**例 4.3**（$\mathbb{Q}(\sqrt{2})$ 的单位）：

- 最小解 $a^2 - 2b^2 = \pm 1$ 为 $(a, b) = (1, 1)$，对应 $\varepsilon = 1 + \sqrt{2}$
- 验证：$N(1+\sqrt{2}) = 1 - 2 = -1$，故 $\varepsilon$ 是单位
- 所有单位：$\pm(1+\sqrt{2})^n$，$n \in \mathbb{Z}$

**注**：$\varepsilon = 1 + \sqrt{2} \approx 2.414$ 是最小的大于1的单位。

## 5. 与其他分支的联系

| 数学分支 | 联系 |
|---------|------|
| **代数几何** | 数域对应曲线的函数域；Spec($\mathcal{O}_K$)是一维概型 |
| **表示论** | 阿廷L-函数来自伽罗瓦表示；类域论 = 交换伽罗瓦表示的朗兰兹对应 |
| **解析数论** | 戴德金ζ函数 $\zeta_K(s) = \sum N(\mathfrak{a})^{-s}$；类数公式连接解析不变量与代数不变量 |
| **密码学** | 虚二次域类群用于密码（如Buchmann-Williams系统）；椭圆曲线有复乘理论 |

**现代应用**：

- **类群计算**：用于量子抗性密码设计
- **单位群**：用于计算 regulator，与 $K$-理论联系
- **高斯猜想**：虚二次域类数1问题（Heegner, Stark解决）
