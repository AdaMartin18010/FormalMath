## 📋 目录概述

数论是研究整数性质及其结构的数学分支，被誉为"数学的皇后"。本目录系统涵盖数论的五大核心领域：**初等数论**探索整数的整除性、同余理论与素数分布，奠定数论基础；**代数数论**研究代数整数、理想理论与类域论，架起代数与数论的桥梁；**解析数论**运用分析工具研究素数分布、黎曼ζ函数与L函数，解决经典难题；**组合数论**关注加法结构、 Ramsey理论与极值组合；**计算数论**聚焦算法设计与密码学应用，包括素性测试、因子分解与椭圆曲线密码。从欧几里得算法到费马大定理，从黎曼猜想到现代密码学，数论不仅是纯粹数学的核心，更是信息安全、编码理论与计算科学的理论基石。

## 🗺️ 学习路径

### 入门阶段（建议时长：4-6周）

**推荐学习顺序：**

1. **起点**：[01-初等数论](01-初等数论.md) — 掌握整除理论、欧几里得算法、同余方程
2. **深化**：[01-初等数论-增强版](01-初等数论-增强版.md) — 深入二次互反律、原根理论、高斯和

**配套练习：**

- 完成费马小定理的多种证明
- 实现欧几里得算法与扩展欧几里得算法
- 探索二次剩余的计算与应用

### 进阶方向

#### 方向A：代数数论（需先修抽象代数）

**学习序列：**

1. [02-代数数论](02-代数数论.md) — 代数整数环、理想分解、类群理论
2. 深入：类域论、椭圆曲线、模形式

**先修要求：**

- 抽象代数（群、环、域理论）
- Galois理论
- 交换代数基础

#### 方向B：解析数论（需先修复分析）

**学习序列：**

1. [03-解析数论](03-解析数论.md) — 素数定理、Dirichlet特征、ζ函数解析延拓
2. 深入：筛法、大筛法、自守形式

**先修要求：**

- 复分析（解析函数、围道积分）
- 实分析（测度论、傅里叶分析）
- 渐近分析方法

#### 方向C：计算数论与应用（需编程基础）

**学习序列：**

1. [05-计算数论](05-计算数论.md) — 算法复杂度、素性测试、因子分解
2. 实践：RSA实现、椭圆曲线密码学

**先修要求：**

- Python/C++编程
- 算法与数据结构
- [01-初等数论](01-初等数论.md)基础
