# 数论 / Number Theory

**最后更新**: 2026年2月10日  
**MSC 主类**: 11-XX (Number theory)  
**国际对标**: MIT 18.781, AMS MSC2020

---

## 📋 目录概述

本目录涵盖数论主要分支：初等数论（整除、同余、二次剩余）、代数数论（代数整数、理想、Galois 扩张）、解析数论（素数分布、L 函数、黎曼 ζ 函数）、组合数论、计算数论。

## 🗺️ 学习路径建议

1. **入门**: 01-初等数论 → 01-初等数论-增强版  
2. **代数方向**: 02-代数数论（先修：抽象代数、域论）  
3. **解析方向**: 03-解析数论（先修：复分析、实分析）  
4. **应用方向**: 05-计算数论（先修：初等数论、算法）
