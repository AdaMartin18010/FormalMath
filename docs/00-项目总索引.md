## 一、项目概览

### 1.1 项目简介

FormalMath 是一个面向数学教育的综合性知识库，致力于建立从基础数学到现代数学前沿的完整结构化内容，并结合数学家理念体系、国际权威课程对齐、形式化证明和智能学习系统。

### 1.2 核心统计数据

| 指标 | 数值 | 说明 |
|------|------|------|
| **总文档数** | 6,573+篇 | Markdown文档 |
| **数学家数量** | 62位 | 26位研究级深度 |
| **数学分支** | 15个 | 01-基础数学 到 15-同调代数 |
| **Lean4定理** | 20个 | 全部通过编译验证 |
| **国际课程** | 121+门 | MIT/Harvard/Stanford等九校 |
| **工作示例** | 101个 | 概念/计算/证明/应用 |
| **应用案例** | 129个 | 多学科深度案例 |
