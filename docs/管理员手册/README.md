# FormalMath 管理员手册

**版本**: v1.0  
**更新日期**: 2026年4月4日  
**适用对象**: 系统管理员、运维工程师、技术负责人  

---

## 手册概述

本手册为 FormalMath 项目的系统管理员提供全面的运维指导，涵盖系统架构、部署、监控、故障处理、安全管理、备份恢复等方面。

### FormalMath 项目简介

FormalMath 是一个面向数学教育的综合性知识库，包含：

- **6,600+篇文档** (约500万字)
- **62位数学家**深度理念体系 (3,800+篇)
- **15+数学分支**结构化内容
- **50个Lean4定理**形式化证明
- **三大智能学习系统** (认知诊断、评估、自适应路径)

### 关键路径

```
/opt/formalmath/           # 项目根目录
├── docs/                  # 文档内容
├── 数学家理念体系/         # 数学家体系
├── tools/                 # 工具脚本
│   ├── cognitive-diagnosis/
│   ├── assessment-system/
│   └── personalized-recommendation/
├── data/                  # 数据文件
│   ├── cognitive.db       # 认知诊断数据库
│   ├── assessment.db      # 评估数据库
│   └── adaptive.db        # 自适应学习数据库
└── logs/                  # 日志文件
```
