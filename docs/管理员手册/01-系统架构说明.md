## 一、系统概述

### 1.1 项目规模

| 维度 | 规模 | 备注 |
|------|------|------|
| 总文档数 | 6,600+篇 | 约500万字 |
| 数学家体系 | 62位 | 3,800+篇文档 |
| 数学分支 | 15+个 | 1,500+篇文档 |
| Lean4定理 | 50个 | ~3,600行代码 |
| 代码行数 | ~20,000行 | 形式化+系统代码 |
| 存储容量 | ~500MB | 文本+代码 |

### 1.2 系统定位

FormalMath 是面向数学教育的综合性知识库，包含：
- **内容系统**: 结构化数学文档体系
- **形式化系统**: Lean4定理证明
- **智能系统**: 认知诊断、评估、自适应学习
- **管理系统**: 元数据、索引、质量控制

### 1.3 架构原则

1. **静态优先**: 文档以静态Markdown为主，便于版本控制和分发
2. **服务解耦**: 智能系统作为独立服务部署
3. **分层存储**: 内容、索引、元数据分离存储
4. **可扩展性**: 模块化设计，支持水平扩展
