## 一、监控概述

### 1.1 监控范围

| 层级 | 监控对象 | 频率 | 优先级 |
|------|----------|------|--------|
| **基础设施** | CPU、内存、磁盘、网络 | 15秒 | P0 |
| **应用服务** | API响应时间、错误率 | 10秒 | P0 |
| **数据库** | 连接数、查询性能 | 30秒 | P1 |
| **业务指标** | 文档访问量、定理编译 | 1分钟 | P2 |
| **安全** | 登录异常、访问模式 | 实时 | P0 |

## 二、指标监控

### 2.1 系统指标

#### CPU 监控

```yaml
# Prometheus 规则
- alert: HighCPUUsage
  expr: 100 - (avg by (instance) (irate(node_cpu_seconds_total{mode="idle"}[5m])) * 100) > 80
  for: 5m
  labels:
    severity: warning
  annotations:
    summary: "CPU使用率过高"
    description: "实例 {{ $labels.instance }} CPU使用率超过80%"
```

**关键指标**:

- CPU使用率: < 70% 正常, 70-85% 警告, > 85% 严重
- 负载平均: load1 < 核心数 * 0.7
- 进程数量: < 1000

#### 内存监控

```yaml
- alert: HighMemoryUsage
  expr: (node_memory_MemTotal_bytes - node_memory_MemAvailable_bytes) / node_memory_MemTotal_bytes * 100 > 85
  for: 5m
  labels:
    severity: critical
  annotations:
    summary: "内存使用率过高"
    description: "实例 {{ $labels.instance }} 内存使用率超过85%"
```

**关键指标**:

- 内存使用率: < 70% 正常, 70-85% 警告, > 85% 严重
- 交换分区使用: < 10% 正常
- OOM事件: 0

#### 磁盘监控

```yaml
- alert: DiskSpaceLow
  expr: (node_filesystem_avail_bytes{mountpoint="/"} / node_filesystem_size_bytes{mountpoint="/"} * 100) < 15
  for: 5m
  labels:
    severity: warning
  annotations:
    summary: "磁盘空间不足"
    description: "磁盘 {{ $labels.mountpoint }} 剩余空间低于15%"

- alert: DiskWillFillIn4Hours
  expr: predict_linear(node_filesystem_avail_bytes{mountpoint="/"}[1h], 4 * 3600) < 0
  for: 5m
  labels:
    severity: critical
  annotations:
    summary: "磁盘将在4小时内满"
```

**关键指标**:

- 磁盘使用率: < 70% 正常, 70-85% 警告, > 85% 严重
- inode使用率: < 80%
- IO延迟: < 10ms

## 四、健康检查

### 4.1 端点检查

```python
# FastAPI 健康检查端点
from fastapi import FastAPI, status
from pydantic import BaseModel

app = FastAPI()

class HealthStatus(BaseModel):
    status: str
    version: str
    checks: dict

@app.get("/health", response_model=HealthStatus)
async def health_check():
    checks = {
        "database": await check_database(),
        "disk": check_disk_space(),
        "memory": check_memory(),
        "lean": check_lean_environment()
    }

    all_healthy = all(c["status"] == "healthy" for c in checks.values())

    return HealthStatus(
        status="healthy" if all_healthy else "unhealthy",
        version="1.0.0",
        checks=checks
    )

async def check_database():
    try:
        # 尝试执行简单查询
        await db.execute("SELECT 1")
        return {"status": "healthy", "response_time_ms": 5}
    except Exception as e:
        return {"status": "unhealthy", "error": str(e)}

def check_disk_space():
    import shutil
    stat = shutil.disk_usage("/")
    free_percent = stat.free / stat.total * 100
    return {
        "status": "healthy" if free_percent > 15 else "warning",
        "free_percent": free_percent
    }

def check_memory():
    import psutil
    mem = psutil.virtual_memory()
    return {
        "status": "healthy" if mem.percent < 85 else "warning",
        "used_percent": mem.percent
    }

def check_lean_environment():
    import subprocess
    try:
        result = subprocess.run(
            ["lean", "--version"],
            capture_output=True,
            timeout=5
        )
        return {
            "status": "healthy" if result.returncode == 0 else "unhealthy",
            "version": result.stdout.decode().strip()
        }
    except Exception as e:
        return {"status": "unhealthy", "error": str(e)}
```

### 4.2 深度检查

```python
@app.get("/health/deep")
async def deep_health_check():
    """深度健康检查 - 包含业务功能验证"""
    checks = {

        "cache": await check_cache(),
        "external_apis": await check_external_apis(),
        "lean_build": await check_lean_build(),
        "document_access": await check_document_access()
    }
    return checks

async def check_lean_build():
    """验证Lean项目可以正常构建"""
    import subprocess
    try:
        result = subprocess.run(
            ["lake", "build"],
            cwd="/app/docs/09-形式化证明/Lean4",
            capture_output=True,

        )
        return {
            "status": "healthy" if result.returncode == 0 else "unhealthy",
            "build_time_ms": 45000
        }
    except subprocess.TimeoutExpired:
        return {"status": "unhealthy", "error": "Build timeout"}
```

## 五、告警配置

### 5.1 告警规则

```yaml
# alertmanager.yml
groups:
  - name: formalmath-alerts
    rules:
      # 系统告警
      - alert: InstanceDown
        expr: up == 0
        for: 1m
        labels:
          severity: critical
        annotations:
          summary: "实例 {{ $labels.instance }} 宕机"

      # 服务告警
      - alert: ServiceHighErrorRate
        expr: rate(http_requests_total{status=~"5.."}[5m]) > 0.05
        for: 2m
        labels:
          severity: critical
        annotations:
          summary: "服务错误率过高"

      # 业务告警
      - alert: LeanBuildFailed
        expr: increase(lean_build_failures_total[1h]) > 0
        for: 0m
        labels:
          severity: high
        annotations:
          summary: "Lean构建失败"

      # 安全告警
      - alert: UnauthorizedAccess
        expr: increase(unauthorized_access_attempts_total[5m]) > 10
        for: 1m
        labels:
          severity: critical
        annotations:
          summary: "检测到大量未授权访问尝试"
```

### 5.2 告警分级

| 级别 | 响应时间 | 通知方式 | 示例 |
|------|----------|----------|------|
| **P0 - 严重** | 5分钟 | 电话+短信+邮件 | 服务宕机、数据丢失 |
| **P1 - 高** | 15分钟 | 短信+邮件 | 性能严重下降、安全告警 |
| **P2 - 中** | 1小时 | 邮件+IM | 资源警告、非核心功能异常 |
| **P3 - 低** | 1工作日 | 邮件 | 容量预警、优化建议 |

### 6.2 自定义监控脚本

```python
#!/usr/bin/env python3
# monitor.py - 系统监控脚本

import psutil
import requests
import json
import time
from datetime import datetime

class SystemMonitor:
    def __init__(self):
        self.thresholds = {
            'cpu': 80,
            'memory': 85,
            'disk': 85,
            'response_time': 500  # ms
        }

    def check_system(self):
        """检查系统资源"""
        return {
            'timestamp': datetime.now().isoformat(),
            'cpu_percent': psutil.cpu_percent(interval=1),
            'memory_percent': psutil.virtual_memory().percent,
            'disk_percent': psutil.disk_usage('/').percent,
            'connections': len(psutil.net_connections())
        }

    def check_services(self):
        """检查服务状态"""
        services = {
            'cognitive': 'http://localhost:8001/health',
            'assessment': 'http://localhost:8002/health',
            'adaptive': 'http://localhost:8003/health'
        }

        results = {}
        for name, url in services.items():
            try:
                start = time.time()
                resp = requests.get(url, timeout=5)
                elapsed = (time.time() - start) * 1000

                results[name] = {
                    'status': 'up' if resp.status_code == 200 else 'down',
                    'response_time_ms': round(elapsed, 2),
                    'status_code': resp.status_code
                }
            except Exception as e:
                results[name] = {
                    'status': 'down',
                    'error': str(e)
                }

        return results

    def check_lean(self):
        """检查Lean环境"""
        import subprocess
        try:
            result = subprocess.run(
                ['lake', 'build'],
                cwd='docs/09-形式化证明/Lean4',
                capture_output=True,
                timeout=120
            )
            return {
                'status': 'ok' if result.returncode == 0 else 'failed',
                'returncode': result.returncode
            }
        except Exception as e:
            return {'status': 'error', 'error': str(e)}

    def generate_report(self):
        """生成监控报告"""
        report = {
            'system': self.check_system(),
            'services': self.check_services(),
            'lean': self.check_lean()
        }

        # 检查阈值
        alerts = []
        sys = report['system']
        if sys['cpu_percent'] > self.thresholds['cpu']:
            alerts.append(f"CPU使用率过高: {sys['cpu_percent']}%")
        if sys['memory_percent'] > self.thresholds['memory']:
            alerts.append(f"内存使用率过高: {sys['memory_percent']}%")

        report['alerts'] = alerts
        return report

if __name__ == '__main__':
    monitor = SystemMonitor()
    report = monitor.generate_report()
    print(json.dumps(report, indent=2))
```
