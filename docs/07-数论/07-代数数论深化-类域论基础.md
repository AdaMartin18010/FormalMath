## 一、局部域的理论

### 1.1 p-adic数的构造

**历史背景**：Kurt Hensel (1861-1941) 于1897年引入p-adic数，类比于复分析中的幂级数展开。

**定义 1.1**（p-adic赋值）：设 $p$ 为素数。对非零有理数 $x = p^n \frac{a}{b}$（其中 $p \nmid ab$），定义 **p-adic赋值**
$$v_p(x) = n, \quad |x|_p = p^{-n}$$

约定 $|0|_p = 0$。

**性质**：

1. $|xy|_p = |x|_p |y|_p$（乘法性）
2. $|x+y|_p \leq \max(|x|_p, |y|_p)$（超度量不等式）
3. $|x+y|_p = \max(|x|_p, |y|_p)$ 当 $|x|_p \neq |y|_p$

**定义 1.2**（p-adic数域）：$\mathbb{Q}_p$ 是 $\mathbb{Q}$ 关于 $|\cdot|_p$ 的完备化。元素可表示为

$$x = \sum_{n=N}^{\infty} a_n p^n, \quad a_n \in \{0, 1, \ldots, p-1\}$$

**例 1.3**：在 $\mathbb{Q}_3$ 中，$-1 = 2 + 2 \cdot 3 + 2 \cdot 3^2 + \cdots = \sum_{n=0}^{\infty} 2 \cdot 3^n$

验证：$3 \sum_{n=0}^{\infty} 2 \cdot 3^n = \sum_{n=1}^{\infty} 2 \cdot 3^n$，故
$$-1 = 2 + \sum_{n=1}^{\infty} 2 \cdot 3^n - 3\sum_{n=0}^{\infty} 2 \cdot 3^n = 2 + 0 = 2 \pmod{3}$$

**定义 1.4**（p-adic整数环）：
$$\mathbb{Z}_p = \{ x \in \mathbb{Q}_p : |x|_p \leq 1 \} = \{ x \in \mathbb{Q}_p : v_p(x) \geq 0 \}$$

**定理 1.5**：$\mathbb{Z}_p$ 是局部环，极大理想 $p\mathbb{Z}_p$，剩余域 $\mathbb{Z}_p/p\mathbb{Z}_p \cong \mathbb{F}_p$。

### 1.2 局部域的分类

**定义 1.6**（局部域）：**局部域**是以下两类域之一：

1. **非阿基米德局部域**：关于离散赋值的完备域，剩余域有限
2. **阿基米德局部域**：$\mathbb{R}$ 或 $\mathbb{C}$

**定理 1.7**（局部域分类）：

**非阿基米德情形**：

- 特征0：$p$-adic数域的有限扩张（即 $p$-adic局部域）
- 特征 $p$：$\mathbb{F}_q((t))$（形式Laurent级数域），$q = p^n$

**阿基米德情形**：

- $K_v \cong \mathbb{R}$（实位）
- $K_v \cong \mathbb{C}$（复位）

**数域的完备化**：设 $K$ 是数域，$[K:\mathbb{Q}] = n$，则：

- 每个素理想 $\mathfrak{p}$ 给出非阿基米德完备化 $K_{\mathfrak{p}}$
- 每个实嵌入 $\sigma: K \hookrightarrow \mathbb{R}$ 给出实位 $K_v \cong \mathbb{R}$
- 每对复嵌入给出复位 $K_v \cong \mathbb{C}$

**例 1.8**（二次域的完备化）：设 $K = \mathbb{Q}(\sqrt{2})$，则：

- $p=2$：$(2) = \mathfrak{p}^2$ 分歧，$K_{\mathfrak{p}} = \mathbb{Q}_2(\sqrt{2})$（$\sqrt{2} \in \mathbb{Q}_2$）
- $p=7$：$(7) = \mathfrak{p}_1 \mathfrak{p}_2$ 分裂，$K_{\mathfrak{p}_i} \cong \mathbb{Q}_7$
- $p=5$：$(5)$ 惰性，$K_{\mathfrak{p}} = \mathbb{Q}_5(\sqrt{2})$（2次非分歧扩张）
- 两个实位：$v_1: \sqrt{2} \mapsto \sqrt{2}$，$v_2: \sqrt{2} \mapsto -\sqrt{2}$

## 二、Adele与Idele

### 2.1 Adele环的构造

**动机**：将数域的所有完备化统一处理。

**定义 2.1**（Adele环）：设 $K$ 是数域，其 **Adele环**
$$\mathbb{A}_K = \left\{ (x_v)_v \in \prod_v K_v : x_v \in \mathcal{O}_v \text{ 对几乎所有 } v \right\}$$

其中 $v$ 遍历 $K$ 的所有位（place），$\mathcal{O}_v$ 是 $K_v$ 的整数环（阿基米德位时 $\mathcal{O}_v = K_v$）。

**拓扑结构**：限制积拓扑。基本开集形如 $\prod_v U_v$，其中 $U_v = \mathcal{O}_v$ 对几乎所有 $v$，且 $U_v \subseteq K_v$ 开。

**定理 2.2**（强逼近）：$K$ 在 $\mathbb{A}_K$ 中离散，且商空间 $\mathbb{A}_K/K$ 紧致。

**定理 2.3**：$\mathbb{A}_K \cong \mathbb{A}_{\mathbb{Q}} \otimes_{\mathbb{Q}} K$，且 $\mathbb{A}_{\mathbb{Q}} = \mathbb{R} \times \prod'_p \mathbb{Q}_p$。

### 2.2 Idele群与理想类群

**定义 2.4**（Idele群）：
$$\mathbb{I}_K = \mathbb{A}_K^* = \left\{ (x_v)_v \in \prod_v K_v^* : x_v \in \mathcal{O}_v^* \text{ 对几乎所有 } v \right\}$$

**定义 2.5**（Idele类群）：$C_K = \mathbb{I}_K / K^*$

**范数映射**：对 $x = (x_v) \in \mathbb{I}_K$，定义
$$|x|_K = \prod_v |x_v|_v$$

其中 $|\cdot|_v$ 是归一化绝对值（使乘积公式成立）。

**定理 2.6**（乘积公式）：对 $\alpha \in K^*$，$|\alpha|_K = 1$。

**与理想类群的联系**：
$$\mathbb{I}_K / K^* \cdot \prod_{v \nmid \infty} \mathcal{O}_v^* \cong \text{Cl}(K)$$

**定理 2.7**（类数的Adele解释）：
$$C_K / \overline{K_\infty^* \prod_{v \nmid \infty} \mathcal{O}_v^*} \cong \text{Cl}(K)$$

其中 $K_\infty^* = \prod_{v | \infty} K_v^*$。

## 三、类域论基本定理

### 3.1 Artin互反律

**历史背景**：类域论由 **Hilbert (1898)**、**Takagi (1920)**、**Artin (1927)** 发展，是代数数论的高峰。

**Artin映射**：设 $L/K$ 是Abel扩张，$\mathfrak{p}$ 是 $K$ 的非分歧素理想。定义 **Frobenius元**
$$\sigma_{\mathfrak{p}} = \left[\frac{L/K}{\mathfrak{p}}\right] \in \text{Gal}(L/K)$$

满足 $\sigma_{\mathfrak{p}}(\alpha) \equiv \alpha^{N(\mathfrak{p})} \pmod{\mathfrak{P}}$ 对所有 $\alpha \in \mathcal{O}_L$。

**定理 3.1**（Artin互反律）：设 $L/K$ 是Abel扩张。存在满同态
$$r_{L/K}: C_K \to \text{Gal}(L/K)$$

核为 $N_{L/K}(C_L)$，诱导同构
$$C_K / N_{L/K}(C_L) \cong \text{Gal}(L/K)$$

**解释**：局部-整体原理的极致体现。Abel扩张完全由基域的算术决定。

### 3.2 存在性定理与范数指数

**定理 3.2**（存在性定理）：设 $K$ 是数域，$H \subseteq C_K$ 是开子群且有限指数。则存在唯一的Abel扩张 $L/K$ 使得 $N_{L/K}(C_L) = H$。

**推论**：开子群 $H \subseteq C_K$ 与有限Abel扩张 $L/K$ 一一对应（类域论对应）。

**定理 3.3**（范数指数不等式）：对任意Galois扩张 $L/K$（不必Abel），
$$[C_K : N_{L/K}(C_L)] \leq [L : K]$$

等号成立当且仅当 $L/K$ 是Abel扩张。

**证明思路**：

1. 第一不等式：$[C_K : N_{L/K}(C_L)] \leq [L : K]$（由Takagi建立）
2. 第二不等式：$[C_K : N_{L/K}(C_L)] \geq [L : K]$（由Artin证明，利用互反律）

## 四、类域论的应用

### 4.1 Kronecker-Weber定理

**定理 4.1**（Kronecker-Weber）：$\mathbb{Q}$ 的每个有限Abel扩张都包含在某个分圆域 $\mathbb{Q}(\zeta_n)$ 中。

等价表述：$\mathbb{Q}^{ab} = \bigcup_n \mathbb{Q}(\zeta_n)$

**证明概要（类域论视角）**：

**步骤 1**：Idele类群的结构
$$C_{\mathbb{Q}} = \mathbb{R}_{>0} \times \prod_p \mathbb{Z}_p^*$$

**步骤 2**：$\mathbb{Q}(\zeta_n)$ 对应的开子群。分圆域 $\mathbb{Q}(\zeta_n)$ 的导子是 $n$ 或 $2n$（$n$ 奇数时）。

**步骤 3**：局部对应。对每个 $p$，$\mathbb{Q}_p$ 的极大Abel扩张由局部类域论给出：
$$\mathbb{Q}_p^{ab} = \mathbb{Q}_p(\zeta_{p^\infty}, \zeta_{n, (p,n)=1})$$

**步骤 4**：全局-局部相容性。由Artin互反律的局部-整体相容，$\mathbb{Q}$ 的任何Abel扩张的局部化必为分圆扩张。

**历史注记**：Kronecker (1853) 提出，Weber (1886) 证明，Hilbert (1896) 完善。

### 4.2 Hilbert类域

**定义 4.2**（Hilbert类域）：数域 $K$ 的 **Hilbert类域** $H$ 是 $K$ 的极大非分歧Abel扩张。

**定理 4.3**（Hilbert类域的性质）：设 $H$ 是 $K$ 的Hilbert类域，则：

1. $\text{Gal}(H/K) \cong \text{Cl}(K)$（类群同构）
2. $[H : K] = h_K$（类数）
3. $H/K$ 中每个素理想完全分裂当且仅当它是主理想
4. $H$ 是 $K$ 上所有非分歧Abel扩张的合成

**证明**：由Artin互反律，取 $H = K^* \cdot \prod_{v} \mathcal{O}_v^*$，则
$$C_K / H \cong \text{Cl}(K)$$

**例 4.4**（虚二次域的类域）：设 $K = \mathbb{Q}(\sqrt{-5})$，$h_K = 2$。

- 类群：$\text{Cl}(K) \cong \mathbb{Z}/2\mathbb{Z}$
- Hilbert类域：$H = K(\sqrt{-1}) = \mathbb{Q}(\sqrt{-5}, i)$
- 验证：$[H : K] = 2 = h_K$

**主理想定理**（Hilbert猜想，1929年Furtwängler证明）：$H$ 的每个理想都是主理想。

等价表述：类域塔的终止性。虽然一般情况下类域塔可能无限（Golod-Shafarevich，1964），但Hilbert类域本身满足主理想定理。

**现代发展（2024-2025）**：

- **高维类域论**：对于高维概型，类域论被Kato-Saito理论推广
- **非交换类域论**：Langlands纲领的目标
- **Iwasawa理论**：研究 $\mathbb{Z}_p$-扩张和 $p$-adic L函数
