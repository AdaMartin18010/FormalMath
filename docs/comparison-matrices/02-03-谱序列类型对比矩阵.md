## 谱序列基础框架

### 谱序列的一般结构

**定义**：一个**谱序列**是一族微分对象 $(E_r, d_r)_{r \geq r_0}$，其中：
- $E_r = \bigoplus_{p,q} E_r^{p,q}$（双分次模）
- $d_r: E_r^{p,q} \to E_r^{p+r, q-r+1}$（微分，次数 $(r, -r+1)$）
- $E_{r+1} = H(E_r, d_r)$（同调/上同调递推）

**收敛**：若对固定的 $(p,q)$，当 $r$ 足够大时 $E_r^{p,q} = E_\infty^{p,q}$，则称谱序列**收敛**到某滤过对象。

### 谱序列构造来源

| 构造来源 | 典型谱序列 | 微分次数 | $E_2$ 页结构 | 收敛到 |
|---------|-----------|---------|-------------|-------|
| **滤过复形** | 滤过诱导 | $(r, -r+1)$ | $E_1^{p,q} = H^{p+q}(F^p/F^{p+1})$ | $H^*(Tot)$ |
| **双复形** | 两个谱序列 | $(r, -r+1)$ | $E_2^{p,q} = H^I H^{II}$ 或 $H^{II} H^I$ | $H^*(Tot)$ |
| **复合函子** | Grothendieck谱序列 | $(r, -r+1)$ | $E_2^{p,q} = R^p G(R^q F)$ | $R^{p+q}(G∘F)$ |
| **纤维丛** | Leray-Serre | $(r, -r+1)$ | $E_2^{p,q} = H^p(B, H^q(F))$ | $H^{p+q}(E)$ |
| **群扩张** | Lyndon-Hochschild-Serre | $(r, -r+1)$ | $E_2^{p,q} = H^p(Q, H^q(N))$ | $H^{p+q}(G)$ |

## 拓扑学谱序列对比

### 纤维丛谱序列

#### Serre谱序列

**设置**：纤维丛 $F \to E \to B$，B道路连通。

**谱序列**：
$$E_2^{p,q} = H^p(B, \mathcal{H}^q(F)) \Rightarrow H^{p+q}(E)$$

其中 $\mathcal{H}^q(F)$ 是局部系（若B单连通则为常值）。

**特征**：
- 适用于上同调（Serre）或同调
- 系数可以是局部系
- 乘积结构（上同调）

**典型应用**：
- 计算环路空间同调
- 球丛上同调
- 分类空间上同调

#### Leray谱序列

**设置**：连续映射 $f: X \to Y$，F是X上的层。

**谱序列**：
$$E_2^{p,q} = H^p(Y, R^q f_* F) \Rightarrow H^{p+q}(X, F)$$

**与Serre的关系**：当f是纤维丛时，Leray = Serre。

**应用**：
- 层上同调计算
- 几何映射的分析
- 代数几何中的基本工具

### 上同调运算谱序列

| 谱序列 | 来源 | $E_2$ 页 | 主要应用 |
|-------|-----|---------|---------|
| **Steenrod代数** | 上同调运算 | $Ext_A(H^*X, \mathbb{Z}/p)$ | 同伦群计算 |
| **Adams谱序列** | 稳定同伦范畴 | $Ext_{A}(H^*Y, H^*X)$ | $\{X, Y\}_*$ |
| **Atiyah-Hirzebruch** | 广义上同调 | $H^p(X, h^q(*))$ | 广义上同调计算 |

## 代数学谱序列对比

### 群上同调谱序列

#### Lyndon-Hochschild-Serre谱序列 (LHS)

**设置**：群扩张 $1 \to N \to G \to Q \to 1$，M是G-模。

**谱序列**：
$$E_2^{p,q} = H^p(Q, H^q(N, M)) \Rightarrow H^{p+q}(G, M)$$

**关键性质**：
- 连接群上同调与正规子群/商群
- 低维项给出扩张的阻碍
- 边缘同态与转移映射相关

**应用**：
- 计算大群的上同调
- 研究群扩张的分类
- 类域论中的基本工具

#### 实例：计算 $H^*(\mathbb{Z}/p^n \mathbb{Z})$

```

1 → ℤ/p^{n-1} → ℤ/p^n → ℤ/p → 1

LHS: E_2^{p,q} = H^p(ℤ/p, H^q(ℤ/p^{n-1}))
        ⇒ H^{p+q}(ℤ/p^n)
        
归纳计算可得完整上同调结构。

```

### 代数谱序列

| 谱序列 | 构造 | $E_2$ 页 | 收敛 |
|-------|-----|---------|-----|
| **Barr-Beck** | 单子的分解 | 各种形式 | 导出函子 |
| **André-Quillen** | 同调代数 | D_q Der | 同调/上同调 |
| **改变环** | -⊗_R S | Tor | 导出张量积 |

## 代数几何谱序列对比

### 层上同调谱序列

| 谱序列 | 设置 | $E_2$ 页 | 应用 |
|-------|-----|---------|-----|
| **Leray** | f: X → Y | $H^p(Y, R^qf_*F)$ | 高阶像的计算 |
| **局部-整体Ext** | X上凝聚层 | $H^p(X, Ext^q(F,G))$ | Ext群计算 |
| **Frölicher** | 复流形 | $H^q(X, \Omega^p)$ | Hodge分解 |
| **Hodge-de Rham** | 光滑簇 | $H^q(X, \Omega^p_{alg})$ | de Rham上同调 |

### Grothendieck谱序列详解

**定理**：设 A --F--> B --G--> C，其中：
- F: 将内射对象映到 G-零调对象
- G: 左正合

则存在谱序列：
$$E_2^{p,q} = R^pG(R^qF(X)) \Rightarrow R^{p+q}(G∘F)(X)$$

**关键应用**：

#### 复合映射的Leray谱序列

```

X --f--> Y --g--> Z

Leray谱序列:
E_2^{p,q} = R^pg_*(R^qf_*F) ⇒ R^{p+q}(g∘f)_*F

```

#### 局部上同调谱序列

```

H^i_Z(X, F) (支集在Z内的上同调)

若 Z ⊆ Y ⊆ X，则:
H^p_Z(Y, H^q_Y(X, F)) ⇒ H^{p+q}_Z(X, F)

```

## 收敛性质对比

### 收敛类型

| 收敛类型 | 定义 | 充分条件 | 典型例子 |
|---------|-----|---------|---------|
| **强收敛** | $E_\infty^{p,q} = G^p H^{p+q}$ | 滤过有限 | 有限CW复形 |
| **弱收敛** | 有边缘同态连接 | - | 无限复形 |
| **条件收敛** | lim/lim^1 消失 | Mittag-Leffler | 余极限 |
| **完全退化** | $E_2 = E_\infty$ | 微分全为零 | 乘积丛 |
| **在第一象限** | $E_r^{p,q} = 0$ (p<0 或 q<0) | 第一象限复形 | 大部分几何谱序列 |

### 退化条件

**退化（Collapse）**：若对某r，所有 $d_r = 0$，则 $E_r = E_\infty$。

**充分条件**：
1. **次数理由**：$E_r^{p,q} \neq 0$ 仅在一条对角线
2. **环结构**：乘法结构强制消失
3. **维数理由**：空间维数限制非零项

**退化实例** (Hodge分解)：

```

Frölicher谱序列:
E_1^{p,q} = H^q(X, \Omega^p) ⇒ H^{p+q}_{dR}(X)

紧Kähler流形上：E_1 = E_∞
得到 Hodge分解: H^k(X, ℂ) = ⊕_{p+q=k} H^{p,q}

```

## 计算策略与应用

### 谱序列计算的一般步骤

```

1. 确定谱序列来源 (滤过/双复形/复合函子/...)
        ↓
2. 计算 E_2 页
   - 通常涉及已知/简单的同调计算
        ↓
3. 分析微分 d_r
   - 次数分析
   - 已知边缘同态
   - 乘积结构
        ↓
4. 计算 E_∞ 页
        ↓
5. 解决延拓问题 (extension problem)
   - 从 E_∞ 恢复目标对象

```

### 第一象限谱序列的边界效应

**低维项分析**：

```

E_2^{0,0} = E_∞^{0,0} = H^0 (总有)

正合列 (5-项): 0 → E_2^{1,0} → H^1 → E_2^{0,1} → E_2^{2,0} → H^2

低维长正合列提供了有用的信息，即使高维无法完全计算。

```

### 不同领域的选择指南

| 问题类型 | 推荐谱序列 | 关键页面 | 计算重点 |
|---------|-----------|---------|---------|
| 纤维丛上同调 | Serre/Leray | $E_2 = H^*(B, H^*(F))$ | 局部系系数 |
| 群上同调计算 | LHS | $E_2 = H^*(Q, H^*(N))$ | 群作用 |
| 层上同调 | Leray | $E_2 = H^*(Y, R^qf_*F)$ | 高阶直像 |
| 导出范畴 | 各种Grothendieck | $E_2 = R^pG(R^qF)$ | 复合顺序 |
| 广义上同调 | AHSS | $E_2 = H^*(X, h^*)$ | 系数系统 |
