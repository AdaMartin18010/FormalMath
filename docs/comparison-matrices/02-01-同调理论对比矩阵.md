## 代数拓扑同调理论对比

### 奇异同调 vs 单纯同调 vs 胞腔同调

| 对比维度 | 奇异同调 H_*^sing(X) | 单纯同调 H_*^simp(K) | 胞腔同调 H_*^cell(X) |
|---------|-------------------|-------------------|-------------------|
| **适用对象** | 任意拓扑空间 | Δ-复形/单纯复形 | CW复形 |
| **链复形** | C_n = 自由 Abel 群 (连续映射 Δ^n → X) | C_n = 由 n-单形生成 | C_n = H_n(X^n, X^{n-1}) |
| **边缘映射** | 面上限制和 | 定向面求和 | 附着映射导出 |
| **同伦不变性** | 是 | 是 | 是 |
| **切除定理** | 是 | 是 | 是 |
| **Mayer-Vietoris** | 是 | 是 | 是 |
| **计算难度** | 难（连续映射太多） | 中等（组合） | 易（胞腔数有限） |
| **链同伦等价** | 非平凡 | 与奇异同调同构 | 与奇异同调同构 |
| **优势** | 普适性 | 几何直观 | 计算效率 |

### 奇异同调的构造

```
奇异 n-单形: σ: Δ^n → X (连续映射)

边缘算子: ∂_n(σ) = Σ(-1)^i σ|[v_0,...,v̂_i,...,v_n]

链复形: ... → C_n(X) --∂_n--> C_{n-1}(X) → ... → C_0(X) → 0

奇异同调: H_n(X) = ker(∂_n) / im(∂_{n+1})
```

### 单纯同调的构造

```
单纯复形 K = ∪ K_n (n-单形的集合)

定向 n-单形: [v_0, v_1, ..., v_n]

边缘算子: ∂[v_0,...,v_n] = Σ(-1)^i [v_0,...,v̂_i,...,v_n]

单纯同调: H_n^simp(K) = ker(∂_n) / im(∂_{n+1})
```

### 同调公理对比 (Eilenberg-Steenrod)

| 公理 | 奇异同调 | 满足说明 |
|-----|---------|---------|
| **同伦公理** | ✓ | 同伦等价诱导同构 |
| **正合公理** | ✓ | 空间对的正合列 |
| **切除公理** | ✓ | 子空间切除不改变同调 |
| **维数公理** | ✓ | H_n(*) = 0 (n > 0), H_0(*) = ℤ |
| **可加公理** | ✓ | 不交并的直和 |

### 特殊同调理论

#### Čech 同调

**定义**：基于开覆盖的神经构造

$$\check{H}_n(X) = \varprojlim_{\mathcal{U}} H_n(N(\mathcal{U}))$$

**特征**：
- 对紧Hausdorff空间定义良好
- 与奇异同调在局部可缩空间一致
- 满足连续性公理（逆向极限保持）

#### Vietoris 同调

**定义**：基于 Vietoris 复形

$$V_\epsilon(X) = \{\text{直径} < \epsilon \text{ 的有限子集}\}$$

**特征**：
- 适用于紧度量空间
- 与Čech同调等价
- 在计算拓扑（persistent homology）中有应用

## 同调代数中的导出函子对比

### Tor 与 Ext 对比

| 对比维度 | Tor_n^R(-,-) | Ext^n_R(-,-) |
|---------|-------------|-------------|
| **变元** | 两个右/左模 | 两个模（左/右） |
| **导出自** | -⊗_R- (张量积) | Hom_R(-,-) |
| **结果类型** | 模 | Abel群/模 |
| **消失** | 平坦模 ⇒ Tor_n = 0 (n > 0) | 投射/内射模 ⇒ Ext^n = 0 (n > 0) |
| **连接同态** | 长正合列（两个变元） | 长正合列（两个变元） |
| **低维解释** | Tor_1 = 挠子 | Ext^1 = 扩张分类 |
| **高维解释** | 高阶挠 | 高阶扩张 |

### Tor 函子的详细分析

**定义**：Tor_n^R(M, N) = H_n(P_• ⊗_R N) = H_n(M ⊗_R Q_•)

其中 P_• → M 和 Q_• → N 是投射分解。

**关键性质**：
1. **对称性**：Tor_n^R(M, N) ≅ Tor_n^R(N, M) （R交换时）
2. **平坦性判别**：M 平坦 ⟺ Tor_n^R(M, -) = 0 ∀ n > 0
3. **通用系数定理**：
   $$0 → H_n(X)⊗G → H_n(X;G) → Tor(H_{n-1}(X), G) → 0$$

### Ext 函子的详细分析

**定义**：Ext^n_R(M, N) = H^n(Hom_R(P_•, N)) = H^n(Hom_R(M, I^•))

**关键性质**：
1. **Yoneda 积**：Ext^m ⊗ Ext^n → Ext^{m+n}
2. **扩张解释**：Ext^1_R(M, N) = {N 被 M 的扩张} / 等价
3. **Baer 和**：Ext^1 上的 Abel 群结构

**Yoneda Ext**：
```
0 → N → E_1 → E_2 → ... → E_n → M → 0
```
两个 n-扩张等价如果存在链映射连接它们。

## 上同调理论对比

### 奇异上同调 vs de Rham 上同调

| 对比维度 | 奇异上同调 H^*_sing | de Rham 上同调 H^*_dR |
|---------|------------------|---------------------|
| **适用对象** | 任意拓扑空间 | 光滑流形 |
| **链复形** | 上链 C^n = Hom(C_n, R) | 微分形式 Ω^*(M) |
| **上边缘** | δ: C^n → C^{n+1} | d: Ω^k → Ω^{k+1} |
| **乘积结构** | 上积 (cup product) | 楔积 (wedge product) |
| **环结构** | 分次交换环 | 分次交换环 |
| **系数** | 任意 Abel 群/环 | ℝ (或 ℂ) |
| **计算工具** | 胞腔上链、谱序列 | 分部积分、Stokes定理 |
| **物理意义** | 拓扑不变量 | 保守场、势理论 |

### de Rham 定理

**定理**：对光滑流形 M，存在典范同构：

$$H^*_{dR}(M) \cong H^*_{sing}(M; \mathbb{R})$$

**同构映射**：
```
I: H^k_{dR}(M) → H^k_{sing}(M; ℝ)
[ω] ↦ ([σ] ↦ ∫_σ ω)
```

**意义**：连接分析与拓扑的桥梁。

### 层上同调对比

| 理论 | 层类型 | 上同调群 | 主要应用 |
|-----|-------|---------|---------|
| **拓扑层** | Ab_X (Abel群层) | H^i(X, F) | 拓扑不变量 |
| **代数几何层** | O_X-模层 | H^i(X, F) | 几何不变量 |
| **平展上同调** | ℓ进层 | H^i_ét(X, ℚ_ℓ) | Weil猜想 |
| **晶体上同调** | 晶体层 | H^i_{cris} | 特征p几何 |
| **德利涅上同调** | 混合Hodge结构 | H^i_D(X, ℚ) | Hodge理论 |

## 计算方法与工具对比

### 不同理论的计算工具

| 同调理论 | 主要计算工具 | 计算复杂度 | 软件支持 |
|---------|------------|----------|---------|
| **胞腔同调** | 胞腔链复形、附着映射 | O(n^3) | SageMath |
| **单纯同调** | Smith 正规形、持久同调 | O(n^3) | GUDHI, Ripser |
| **群上同调** | Bar 分解、Lyndon/Hochschild-Serre | 指数级 | GAP |
| **层上同调** | Čech 分解、Čech-de Rham | 多项式级 | Macaulay2 |
| **de Rham** | 分部积分、调和形式 | 分析级 | 手工/符号计算 |

### 持久同调 (Persistent Homology)

**核心思想**：追踪拓扑特征随尺度参数的变化。

```
过滤: X_0 ⊆ X_1 ⊆ X_2 ⊆ ... ⊆ X_n = X

条形码: 每个拓扑特征表示为 [birth, death)
持久图: (birth, death) 点集
```

**应用**：数据分析、形状识别、材料科学。
