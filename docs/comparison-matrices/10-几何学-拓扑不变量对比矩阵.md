# 几何学概念对比矩阵：拓扑不变量

## 概述

本文档对比代数拓扑中的主要不变量：同伦群(Homotopy Groups)、同调群(Homology Groups)、上同调(Cohomology)和欧拉示性数(Euler Characteristic)，阐明其定义、计算方法、相互关系和拓扑意义。

---

## 核心对比矩阵

| 不变量 | 维度 | 计算方法 | 可计算性 | 核心信息 |
|:---:|:---:|:---:|:---:|:---|
| **同伦群 πₙ** | 可数无穷 | 几何直观 | 极难 | 洞的高维形状 |
| **同调群 Hₙ** | 有限维(紧流形) | 代数组合 | 可计算 | 洞的"个数" |
| **上同调 Hⁿ** | 有限维 | 对偶 | 可计算 | 同调对偶，乘积结构 |
| **欧拉示性数 χ** | 标量 | 交错和 | 易计算 | 整体拓扑复杂度 |
| **Betti数 bₙ** | 标量(每个n) | dim Hₙ | 易计算 | n维洞的数量 |

## 详细对比分析

### 1. 同伦群 (Homotopy Groups)

**定义**: πₙ(X, x₀) = [Sⁿ, X]（保持基点的映射同伦类）

| 特征 | 说明 |
|:---:|:---|
| **π₁** | 基本群（可能非交换） |
| **πₙ (n≥2)** | 高阶同伦群（总是交换） |
| **几何意义** | n维球面到X的映射同伦类 |
| **计算难度** | 极其困难，即使对简单空间 |

**例子**:

| 空间 | π₁ | π₂ | π₃ | ... |
|:---:|:---:|:---:|:---:|:---:|
| S¹ | ℤ | 0 | 0 | ... |
| S² | 0 | ℤ | ℤ | ℤ/2, ℤ/2, ℤ/12... |
| Sⁿ (n≥2) | 0 | ... | ℤ | ... |
| T² = S¹×S¹ | ℤ² | 0 | 0 | ... |
| ℂP^∞ | 0 | ℤ | 0 | ... |

**困难之处**: 即使是球面的同伦群，至今也未完全计算出来！

### 2. 同调群 (Homology Groups)

**定义**: Hₙ(X) = ker ∂ₙ / im ∂ₙ₊₁ （闭链/边缘链）

| 特征 | 说明 |
|:---:|:---|
| **计算方法** | 胞腔分解、单纯形分解 |
| **可计算性** | 相对容易（线性代数） |
| **几何意义** | n维洞的"个数"（Betti数） |
| **长正合列** | 对(M, A)有连接同态 |

**Betti数**: bₙ = dim Hₙ(X; ℚ)
- b₀: 连通分支数
- b₁: 一维洞（环）数
- b₂: 二维洞（空腔）数

**例子**:

| 空间 | H₀ | H₁ | H₂ | Hₙ (n≥3) |
|:---:|:---:|:---:|:---:|:---:|
| Sⁿ | ℤ | 0 | ... | 0(n≠n), ℤ(k=n) |
| T² | ℤ | ℤ² | ℤ | 0 |
| ℂPⁿ | ℤ | 0 | ℤ | ... |
| ℝⁿ \\{0\\} | ℤ | 0/ℤ(n=1) | 0 | ... |

### 3. 上同调 (Cohomology)

**定义**: Hⁿ(X) = Hom(Hₙ(X), ℤ)（对偶）
或 Hⁿ(X) = ker dⁿ / im dⁿ⁻¹

| 特征 | 说明 |
|:---:|:---|
| **环结构** | 上积 ∪: Hᵖ × Hᵠ → Hᵖ⁺ᵠ |
| **自然性** | 与连续映射的函子性 |
| **计算** | 通常通过对偶计算 |
| **示性类** | 陈类、Pontryagin类等 |

**de Rham上同调**: 光滑流形上

```

Hⁿ_{dR}(M) = {闭n形式} / {恰当n形式}

定理: Hⁿ_{dR}(M) ≅ Hⁿ(M; ℝ) (光滑情形)

```

### 4. 欧拉示性数 (Euler Characteristic)

**定义**: χ(X) = Σ (-1)ⁿ rank Hₙ(X) = Σ (-1)ⁿ bₙ

| 特征 | 说明 |
|:---:|:---|
| **组合定义** | χ = V - E + F（多面体） |
| **同伦不变** | 同伦等价的 spaces 相同 χ |
| **可局部计算** | 可用向量场指标和计算 |
| **乘性** | χ(X×Y) = χ(X)·χ(Y) |

**例子**:

| 空间 | χ |
|:---:|:---:|
| Sⁿ | 1 + (-1)ⁿ |
| T² | 0 |
| Σ_g (亏格g曲面) | 2-2g |
| ℂPⁿ | n+1 |
| 可缩空间 | 1 |

**Poincaré-Hopf定理**:

```

对紧致流形上的向量场V:
Σ_{V(p)=0} ind(p) = χ(M)

零点指标和 = Euler示性数

```

## 关系与定理

### 万有系数定理

```

同调与上同调的关系:

Hⁿ(X; G) ≅ Hom(Hₙ(X), G) ⊕ Ext(Hₙ₋₁(X), G)

特别: Hⁿ(X; ℚ) ≅ Hom(Hₙ(X; ℚ), ℚ)
     （无挠时，就是单纯对偶）

```

### Hurewicz定理

```

联系同伦与同调:

若 X (n-1)-连通 (πᵢ = 0, i < n)，则:
    πₙ(X) ≅ Hₙ(X)

特例: 单连通空间，π₂ ≅ H₂

```

### Poincaré对偶

```

对n维可定向闭流形:
Hᵏ(M) ≅ Hₙ₋ᵏ(M)
或 Hᵏ(M) ≅ H^{n-k}(M)（上同调形式）

推论: bₖ = bₙ₋ₖ

```

### 切除定理

```

U ⊂ A ⊂ X, Ū ⊂ int(A)
─────────────────────────
Hₙ(X, A) ≅ Hₙ(X\\U, A\\U)

切除不改变相对同调

```

## 计算方法对比

| 方法 | 适用 | 难度 | 典型应用 |
|:---:|:---:|:---:|:---|
| **胞腔分解** | CW复形 | 中 | 射影空间、Grassmannian |
| **单纯形分解** | 三角剖分 | 高 | 抽象计算 |
| **Mayer-Vietoris** | 并集 | 中 | X = U ∪ V |
| **纤维丛谱序列** | 纤维丛 | 高 | 球丛、主丛 |
| **上同调环** | 乘积结构 | 中 | 乘积空间、射影空间 |
