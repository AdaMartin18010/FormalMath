# 可视化方法对比：静态可视化 vs 交互可视化

## 基本信息

| 维度 | 内容 |
|------|------|
| **对比主题** | 信息呈现的两种模式 |
| **核心问题** | 预计算 vs 探索性 |
| **技术演进** | 印刷时代 → 数字时代 |

---

## 核心特征矩阵

| 特征维度 | 静态可视化 (Static) | 交互可视化 (Interactive) |
|----------|---------------------|--------------------------|
| **信息状态** | 固定、预计算 | 动态、响应式 |
| **用户角色** | 被动接收 | 主动探索 |
| **认知模式** | 线性阅读 | 循环探索 |
| **技术需求** | 低（图像/印刷） | 高（计算+渲染） |
| **可复现性** | ⭐⭐⭐⭐⭐ 完全可复现 | 依赖交互路径 |
| **存储需求** | 小（图像文件） | 大（应用+数据） |

---

## 技术特征对比

| 技术指标 | 静态可视化 | 交互可视化 |
|----------|------------|------------|
| **制作成本** | 中等（设计） | 高（开发） |
| **分发便利性** | ⭐⭐⭐⭐⭐ 高（PDF/PNG） | ⭐⭐⭐ 中等（需要平台） |
| **信息容量** | 有限（一页） | 几乎无限（分层） |
| **个性化** | 无 | 高度可定制 |
| **实时性** | 离线 | 可实时数据 |
| **学习曲线** | 低 | 中等 |

---

## 交互技术分类

| 交互类型 | 描述 | 示例 |
|----------|------|------|
| **导航** | 平移、缩放、旋转 | 地图、3D模型 |
| **选择** | 点选、框选、刷选 | 数据子集分析 |
| **过滤** | 条件筛选 | 动态查询 |
| **重新配置** | 改变布局 | 排序、重排 |
| **编码** | 改变视觉映射 | 颜色、尺寸映射 |
| **抽象/具体** | 聚合/下钻 | 层次数据探索 |
| **连接** | 多视图联动 | 刷选+链接 |

## 数学可视化场景

### 静态优势场景

| 场景 | 描述 | 最佳实践 |
|------|------|----------|
| **论文插图** | 结论展示 | 高分辨率、清晰标注 |
| **教科书** | 概念说明 | 分步骤、标注关键 |
| **海报展示** | 远距离观看 | 简洁、对比强烈 |
| **打印材料** | 无屏幕环境 | 自包含、完整信息 |
| **存档** | 长期保存 | 标准格式、元数据 |

### 交互优势场景

| 场景 | 描述 | 最佳实践 |
|------|------|----------|
| **探索性分析** | 假设生成 | 多联动视图 |
| **参数空间** | 多参数影响 | 滑块、动画 |
| **复杂函数** | 高维可视化 | 切片、投影 |
| **教学演示** | 概念动态展示 | 逐步揭示 |
| **大数据** | 海量数据浏览 | 聚合+下钻 |

## 具体案例对比

### 案例：函数 $f(x,y) = e^{-(x^2+y^2)}\sin(10\sqrt{x^2+y^2})$

**静态可视化**

```

3D曲面图（固定视角）：
- 选择代表性视角
- 等高线叠加
- 颜色编码高度
局限：单一视角、隐藏细节

```

**交互可视化**

```

可旋转3D视图：
- 鼠标旋转视角
- 缩放查看细节
- 切片查看截面
- 参数滑块调整频率
优势：全方位理解

```

## 交互级别模型

```

Level 0: 静态（无交互）
    ↓
Level 1: 动态（自动播放）
    ↓
Level 2: 操作（缩放、平移）
    ↓
Level 3: 探索（过滤、查询）
    ↓
Level 4: 分析（多视图联动）
    ↓
Level 5: 创造（用户添加内容）

```

## 技术实现对比

|------|------|------|

---

## 认知负荷理论

| 类型 | 静态 | 交互 |
|------|------|------|
| **内在负荷** | 内容本身 | 内容+界面 |
| **外在负荷** | 低 | 可能高（复杂界面）|
| **关联负荷** | 集中于内容 | 分散于操作 |

**设计原则**：交互不应增加不必要的认知负担。
