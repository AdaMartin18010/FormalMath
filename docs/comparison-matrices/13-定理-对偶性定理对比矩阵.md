# 定理对比矩阵：对偶性定理家族

## 概述

本文档对比数学中重要的对偶性定理：线性规划对偶、Pontryagin对偶、Poincaré对偶、Serre对偶等，阐明其数学结构、应用领域和深层统一性。

## 详细对比分析

### 1. 线性规划对偶 (Linear Programming Duality)

**原问题 (Primal)**:
```
minimize    cᵀx
subject to  Ax ≥ b
            x ≥ 0
```

**对偶问题 (Dual)**:
```
maximize    bᵀy
subject to  Aᵀy ≤ c
            y ≥ 0
```

**核心定理**:
| 定理 | 内容 |
|:---:|:---|
| **弱对偶** | 任意可行解: bᵀy ≤ cᵀx |
| **强对偶** | 若原问题有最优解，对偶也有，且值相等 |
| **互补松弛** | 最优解满足特定互补条件 |

**应用**: 单纯形法、灵敏度分析、网络流

### 2. Lagrange对偶 (优化)

**原问题**:
```
minimize    f(x)
subject to  gᵢ(x) ≤ 0, hⱼ(x) = 0
```

**Lagrange函数**:
```
L(x,λ,ν) = f(x) + Σλᵢgᵢ(x) + Σνⱼhⱼ(x)
```

**对偶函数**: g(λ,ν) = infₓ L(x,λ,ν)

**对偶问题**: maximize g(λ,ν) subject to λ ≥ 0

**强对偶条件**: Slater条件（凸问题+严格可行）

### 3. Pontryagin对偶 (调和分析)

**定义**: 局部紧阿贝尔群G的特征群
```
Ĝ = Hom(G, 𝕋)  (连续同态到圆周群)
```

**Pontryagin对偶定理**: 
```
自然映射 G → Ĝ̂ 是同构

例子:
    ℤ̂ = 𝕋 (圆周)
    𝕋̂ = ℤ
    ℝ̂ = ℝ
    ℤ/nℤ̂ = ℤ/nℤ (自对偶)
```

**Fourier变换**:
```
ℱ: L²(G) → L²(Ĝ)
ℱ(f)(χ) = ∫_G f(x)χ̄(x)dx

Pontryagin对偶 ⟹ Fourier反演公式
```

### 4. Poincaré对偶 (代数拓扑)

**定理**: 对n维可定向闭流形M
```
Hᵏ(M; ℝ) ≅ Hₙ₋ᵏ(M; ℝ)

或: Hᵏ(M) × Hⁿ⁻ᵏ(M) → Hⁿ(M) ≅ ℝ 是非退化配对
```

**几何意义**: k维洞的数目 = (n-k)维洞的数目

**Cup积配对**: 
```
∪: Hᵏ(M) × Hⁿ⁻ᵏ(M) → Hⁿ(M) ≅ ℝ
(α, β) ↦ ∫_M α∪β
```

**推广**:
- Lefschetz对偶（带边流形）
- Alexander对偶（子空间补）
- Poincaré-Verdier对偶（层论）

### 5. Serre对偶 (代数几何)

**定理**: 对n维光滑射影簇X，凝聚层ℱ
```
Hⁱ(X, ℱ) ≅ Hⁿ⁻ⁱ(X, ℱ* ⊗ K_X)*

其中:
- ℱ* = Hom(ℱ, 𝒪_X) (对偶层)
- K_X = Ωⁿ_X (典范层)
```

**特例**: 曲线C上
```
h⁰(D) - h¹(D) = deg(D) + 1 - g    (Riemann-Roch)

Serre对偶 ⟹ h¹(D) = h⁰(K-D)
```

### 6. Tannaka对偶 (表示论)

**定理**: 紧致群G由其表示范畴Rep(G)重建
```
G ≅ Aut^⊗(ω)

其中:
- ω: Rep(G) → Vec (遗忘函子)
- Aut^⊗ 表示张量自同构
```

**推广**: 代数群、量子群、Hopf代数

**哲学**: 群由其对称性表示决定

## 对偶性的统一模式

### 范畴论观点

```
对偶 = 反变Hom函子

D = Hom(-, K)  对某个对偶对象K

例子:
- 向量空间对偶: V* = Hom(V, k)
- Pontryagin对偶: Ĝ = Hom(G, 𝕋)
- Cartier对偶: G^D = Hom(G, 𝔾ₘ)

双对偶: D² = Hom(Hom(-,K), K) ≈ id (在好条件下)
```

### 对偶性类型

| 类型 | 特征 | 例子 |
|:---:|:---:|:---|
| **完美对偶** | D² ≅ id | 有限维向量空间、Pontryagin |
| **非完美对偶** | D² ≠ id | 无限维空间、Banach空间 |
| **自对偶** | 对象与其对偶同构 | ℝⁿ, L², 辛向量空间 |
| **正交对偶** | 配对方非退化 | Poincaré, Serre |

## Langlands对偶（最高层次）

**纲领**: Robert Langlands (1960s) 提出的深远对偶性

```
数论侧                    表示论侧
────────                  ────────
Galois群表示      ⟷      自守表示
Artin L-函数      ⟷      自守L-函数
素数分解          ⟷      Hecke算子特征值

几何Langlands:
G-丛的模空间      ⟷      ^LG-局部系统的模空间
D-模              ⟷      局部系统层
n维表示          ⟷      n阶联络
```

**意义**: 统一了数论、表示论、代数几何中的多种对偶性
