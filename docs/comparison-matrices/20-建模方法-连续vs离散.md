|:---:|:---:|:---:|
| **基本对象** | 实数 $\mathbb{R}$、连续函数、流形 | 整数 $\mathbb{Z}$、图、有限集 |

## 理论基础对比

### 数学结构对比

| 结构 | 连续版本 | 离散版本 | 对应关系 |
|:---:|:---:|:---:|:---|
| **导数** | $\frac{df}{dx}$ | 差分 $\Delta f = f(x+1)-f(x)$ | 离散近似连续 |
| **积分** | $\int_a^b f(x)dx$ | 求和 $\sum_{i=a}^b f(i)$ | Riemann和 |
| **函数空间** | $C^k(\mathbb{R})$、Sobolev空间 | 序列空间 $\ell^p$、函数空间 | 采样对应 |
| **变换** | Fourier变换 | 离散Fourier变换（DFT） | 采样定理联系 |
| **概率** | 概率密度函数 | 概率质量函数 | 密度vs计数 |
| **优化** | 变分法、Euler-Lagrange | 组合优化、动态规划 | 连续松弛 |

| 方面 | 连续建模 | 离散建模 |
|:---:|:---:|:---:|
| **核心思想** | 局部线性化、无穷小分析 | 状态枚举、组合分析 |
| **分析重点** | 极限行为、渐近分析 | 有限情况、精确计数 |

## 转换与近似方法

### 连续 → 离散

| 方法 | 描述 | 误差阶 | 应用领域 |
|:---:|:---:|:---:|:---|
| **前向差分** | $f'(x) \approx \frac{f(x+h)-f(x)}{h}$ | $O(h)$ | 数值微分 |
| **中心差分** | $f'(x) \approx \frac{f(x+h)-f(x-h)}{2h}$ | $O(h^2)$ | 高精度微分 |
| **有限体积** | 守恒律的离散形式 | $O(h^p)$ | CFD、物理 |
| **有限元** | 变分问题的分段逼近 | 依单元阶数 | 结构力学 |
| **谱方法** | 全局基函数展开 | 指数（光滑解） | 高精度需求 |
| **采样定理** | Shannon采样 | 精确（带限） | 信号处理 |

### 离散 → 连续

| 方法 | 描述 | 应用 |
|:---:|:---:|:---|
| **连续极限** | 步长 $h \to 0$ | 差分方程→微分方程 |
| **均值场近似** | 大量粒子平均行为 | 统计物理 |
| **流体极限** | 大系统尺度行为 | 排队论、网络 |
| **连续松弛** | 整数→实数变量 | 组合优化 |
| **插值方法** | 离散点构造连续函数 | 数据拟合 |
| **粗粒化** | 细观→宏观平均 | 多尺度建模 |

### 误差分析

```
连续-离散转换误差来源：

离散化误差 = 连续解 - 离散解
          = 截断误差 + 舍入误差

截断误差：方法本身的近似误差（如O(h²)）
舍入误差：浮点数计算的有限精度

收敛性：当 h → 0，离散解 → 连续解
稳定性：误差不随计算传播放大
一致性：离散方程逼近连续方程

Lax等价定理：一致性 + 稳定性 ⟺ 收敛性
```

## 案例分析

### 案例1：种群动力学

#### 连续模型（Logistic方程）

$$\frac{dN}{dt} = rN\left(1 - \frac{N}{K}\right)$$

**解析解：**
$$N(t) = \frac{K}{1 + \left(\frac{K-N_0}{N_0}\right)e^{-rt}}$$

| 特征 | 说明 |
|:---:|:---|
| **优点** | 解析解、平滑行为、渐近分析 |
| **缺点** | 假设无限可分、忽略个体异质性、随机性 |
| **适用** | 大规模种群、长期趋势 |

#### 离散模型（Logistic映射）

$$N_{n+1} = rN_n\left(1 - \frac{N_n}{K}\right)$$

**动力学行为：**
| 参数r | 行为 |
|:---:|:---|
| 0 < r < 1 | 灭绝 |
| 1 < r < 3 | 稳定不动点 |
| 3 < r < 3.57 | 周期倍分岔 |
| r > 3.57 | 混沌 |

| 特征 | 说明 |
|:---:|:---|
| **优点** | 展现丰富动力学（周期、混沌）、个体世代 |
| **缺点** | 可能过度简化、混沌对初值敏感 |
| **适用** | 世代不重叠、小种群、复杂动力学 |

### 案例2：期权定价

| 方法 | 类型 | 特点 | 计算成本 |
|:---:|:---:|:---:|:---:|
| **Black-Scholes** | 连续（PDE） | 解析公式、快速 | 低 |
| **二叉树** | 离散 | 直观、美式期权 | 中等 |
| **蒙特卡洛** | 随机离散 | 路径依赖、多维 | 高 |
| **有限差分** | 离散化连续 | 复杂边界、早行权 | 中等 |

## 复杂度理论视角

### 计算模型对比

| 理论 | 连续 | 离散 |
|:---:|:---:|:---:|
| **计算模型** | BSS机、实数RAM | 图灵机 |
| **复杂度类** | $P_\mathbb{R}$、$NP_\mathbb{R}$ | P、NP |
| **可计算性** | 实数可计算性 | 经典可计算性 |
| **停机问题** | 不可判定 | 不可判定 |
| **完备问题** | 与离散不同 | SAT、3-SAT等 |

### BSS模型简介

```
实数RAM模型：
- 存储实数（无限精度）
- 操作：+,-,×,÷,比较
- 分支基于实数比较
- 与图灵机不同：实数计算能力更强

意义：分析数值算法的理论模型
局限：不考虑舍入误差
```

## 数值方法对比

### 微分方程数值解

| 问题 | 连续方法 | 离散方法 | 混合方法 |
|:---:|:---:|:---:|:---:|
| **ODE初值** | Runge-Kutta、Adams | 迭代方法 | 自适应步长 |
| **ODE边值** | 打靶法、有限差分 | 迭代求解 | 多重网格 |
| **PDE抛物型** | 有限差分、有限元 | 粒子方法 | 自适应网格 |
| **PDE双曲型** | 有限体积、谱方法 | 粒子追踪 | WENO等 |
| **PDE椭圆型** | 有限元、谱方法 | 多网格 | 区域分解 |

### 优化方法

| 问题类型 | 连续方法 | 离散方法 |
|:---:|:---:|:---:|
| **无约束优化** | 梯度下降、牛顿法 | 离散搜索、模式搜索 |
| **约束优化** | 拉格朗日乘子、内点法 | 分支定界、割平面 |
| **全局优化** | 模拟退火、遗传算法（连续） | 遗传算法、蚁群 |
| **组合优化** | 松弛+舍入 | 动态规划、分支定界 |

### 快速选择指南

| 如果你面对... | 推荐方法 | 理由 |
|:---:|:---:|:---|
| **光滑物理场** | 连续（PDE） | 自然规律通常是连续的 |
| **个体异质性重要** | 离散（Agent） | 个体行为差异 |
| **数字信号** | 离散 | 本质离散 |

| **小系统精确行为** | 离散 | 可穷举验证 |
| **复杂几何** | 离散或混合 | 边界处理灵活 |
| **多尺度现象** | 混合 | 各尺度不同方法 |

## 现代发展趋势

### 新兴方向

| 趋势 | 描述 | 代表技术 |
|:---:|:---:|:---:|
| **混合系统** | 连续+离散动态结合 | 混合动力系统、切换系统 |
| **多尺度建模** | 不同尺度不同模型耦合 | QM-MM、异质多尺度 |
| **不确定性量化** | 模型参数随机化 | 随机PDE、多项式混沌 |
| **数据驱动建模** | 从数据学习模型 | 物理信息神经网络(PINN) |
| **数字孪生** | 实时仿真与物理同步 | 混合建模+实时计算 |
| **可解释AI** | 结合物理约束的ML | 符号回归、约束神经网络 |

### 物理信息神经网络 (PINN)

```
融合连续建模与机器学习：

损失函数 = 数据拟合误差 + 物理方程残差

优势：
- 数据驱动（离散数据点）
- 满足物理约束（连续方程）
- 可处理逆问题

应用：流体、固体力学、热传导等
```

## 总结与建议

### 核心对比总结

| 维度 | 连续建模 | 离散建模 |
|:---:|:---:|:---:|
| **核心优势** | 解析洞察、渐近分析、平滑性 | 精确计数、算法可实现、个体追踪 |
| **主要局限** | 解析可解类有限、数值误差 | 组合爆炸、近似分析 |
| **最佳适用** | 光滑现象、大尺度、物理场 | 原子性现象、小系统、算法 |
| **现代趋势** | 与机器学习结合 | 大规模模拟、GPU加速 |

### 学习建议

| 阶段 | 连续建模 | 离散建模 |
|:---:|:---:|:---:|
| **基础** | 微积分、ODE | 离散数学、算法 |
| **进阶** | PDE、数值分析 | 图论、组合优化 |
| **高级** | 泛函分析、渐近方法 | 复杂性理论、概率方法 |
| **前沿** | 不确定性量化、PINN | 大规模模拟、量子计算 |
