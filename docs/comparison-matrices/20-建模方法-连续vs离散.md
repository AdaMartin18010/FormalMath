# 建模方法对比：连续建模 vs 离散建模

## 基本信息

| 维度 | 内容 |
|------|------|
| **对比主题** | 数学建模的两种基本世界观 |
| **核心问题** | 何时连续？何时离散？ |
| **应用领域** | 物理、工程、生物、经济等 |

---

## 核心特征矩阵

| 特征维度 | 连续建模 (Continuous) | 离散建模 (Discrete) |
|----------|----------------------|---------------------|
| **基本对象** | 实数、连续函数、流形 | 整数、图、有限集 |
| **变化观** | 平滑、无限可分 | 跳跃、原子性 |
| **主要工具** | 微积分、微分方程 | 差分方程、组合、图论 |
| **状态空间** | 连续（不可数） | 离散（可数/有限） |
| **典型方程** | $\frac{dy}{dt} = f(y,t)$ | $y_{n+1} = f(y_n, n)$ |
| **计算实现** | 数值积分、有限元 | 迭代、搜索、优化 |

---

## 技术特征对比

| 技术指标 | 连续建模 | 离散建模 |
|----------|----------|----------|
| **解析可解性** | 有限类（线性、特殊函数） | 部分可解（递推关系） |
| **数值稳定性** | 需要稳定性分析 | 通常更稳定 |
| **计算精度** | 截断+舍入误差 | 可能精确（整数运算） |
| **可视化** | 平滑曲线、曲面 | 网络、树、状态图 |
| **验证难度** | 分析困难 | 可穷举（小规模） |
| **复杂度分析** | 连续复杂度理论 | 经典离散复杂度 |

---

## 应用领域对比

| 应用领域 | 连续模型 | 离散模型 | 混合模型 |
|----------|----------|----------|----------|
| **物理学** | 经典力学、场论 | 晶格模型、自旋系统 | 量子场论格点化 |
| **生物学** | 种群动力学（ODE） | 基因网络、细胞自动机 | 混合系统 |
| **经济学** | 微分博弈、DSGE | 代理人模型 | 混合模型 |
| **计算机科学** | 神经网络（连续） | 算法、数据结构 | 混合建模 |
| **交通** | 流体动力学模型 | 元胞自动机 | 混合交通模型 |
| **金融** | Black-Scholes PDE | 二叉树模型 | 蒙特卡洛模拟 |

## 转换与近似

### 连续 → 离散

| 方法 | 描述 | 应用 |
|------|------|------|
| **差分法** | 导数→差分 | 数值PDE求解 |
| **采样** | 连续信号离散化 | 数字信号处理 |
| **格点近似** | 连续空间离散化 | 格点场论 |
| **有限元** | 变分问题离散化 | 结构力学 |

### 离散 → 连续

| 方法 | 描述 | 应用 |
|------|------|------|
| **连续极限** | 步长→0 | 差分方程→微分方程 |
| **均值场近似** | 大量粒子平均 | 统计物理 |
| **流体极限** | 大系统行为 | 排队论、网络 |
| **连续松弛** | 整数→实数 | 组合优化 |

## 案例分析：种群动力学

### 连续模型（Logistic方程）

```
dN/dt = rN(1 - N/K)

解：N(t) = K / (1 + ((K-N₀)/N₀)e^(-rt))

优点：解析解、平滑行为
缺点：假设无限可分、忽略随机性
```

### 离散模型（Logistic映射）

```
N_{n+1} = rN_n(1 - N_n/K)

行为：
- r < 3：稳定不动点
- 3 < r < 3.57：周期倍分岔
- r > 3.57：混沌

优点：展现丰富动力学
缺点：可能过度简化
```

## 复杂度理论视角

| 理论 | 连续 | 离散 |
|------|------|------|
| **计算模型** | BSS机、实数RAM | 图灵机 |
| **复杂度类** | P_R, NP_R | P, NP |
| **可计算性** | 实数可计算性 | 经典可计算性 |
| **停机问题** | 同样不可判定 | 不可判定 |

---

## 数值方法对比

| 问题 | 连续方法 | 离散方法 |
|------|----------|----------|
| **优化** | 梯度下降、牛顿法 | 离散搜索、遗传算法 |
| **积分** | 数值积分（Simpson等） | 蒙特卡洛求和 |
| **微分方程** | Runge-Kutta、有限差分 | 迭代方法 |
| **偏微分方程** | 有限元、谱方法 | 粒子方法、元胞自动机 |
