# 分析学概念对比矩阵：函数空间类型

## 概述

本文档对比分析学中的主要函数空间：Banach空间、Hilbert空间、Sobolev空间等，阐明其结构特性、包含关系和在不同数学领域的应用。

---

## 核心对比矩阵

| 维度 | Banach空间 | Hilbert空间 | Sobolev空间 |
|:---:|:---:|:---:|:---:|
| **定义** | 完备的赋范空间 | 完备的内积空间 | 带弱导数的Lᵖ空间 |
| **结构层次** | 范数结构 | 内积→范数 | 导数+积分 |
| **几何特性** | 凸性、一致凸 | 角度、正交性 | 光滑性层次 |
| **自反性** | 未必自反 | 总是自反 | Lᵖ型自反 |
| **对偶空间** | 范数对偶 | 等距同构自身 | 负指数空间 |
| **正交基** | 未必存在 | 标准正交基存在 | 广义Fourier基 |
| **最佳逼近** | 存在未必唯一 | 正交投影 | 变分形式 |
| **紧性** | 单位球紧⇔有限维 | 同上 | 嵌入定理 |

## Banach空间详解

### 定义与性质

**定义**: 完备的赋范向量空间 (X, ‖·‖)

| 性质 | 说明 |
|:---:|:---|
| **完备性** | 所有Cauchy列收敛 |
| **线性结构** | 向量加法、数乘连续 |
| **范数诱导度量** | d(x,y) = ‖x-y‖ |

### 经典例子

| 空间 | 范数 | 完备性 | 自反性 |
|:---:|:---:|:---:|:---:|
| Lᵖ(μ) (1<p<∞) | ‖f‖ₚ = (∫|f|ᵖ)^{1/p} | 完备 | 自反 |
| L¹(μ) | ‖f‖₁ = ∫|f| | 完备 | 不自反 |
| L∞(μ) | ‖f‖∞ = ess sup|f| | 完备 | 不自反 |
| C(K) (紧集K) | ‖f‖∞ = sup|f| | 完备 | 不自反 |
| c₀ | ‖x‖∞ = sup|xₙ| | 完备 | 不自反 |
| ℓᵖ (1≤p≤∞) | 离散形式 | 完备 | p∈(1,∞)自反 |

### 重要定理

| 定理 | 内容 | 应用 |
|:---:|:---:|:---|
| **Hahn-Banach** | 泛函延拓 | 对偶理论 |
| **一致有界原理** | 点态有界⇒一致有界 | 算子理论 |
| **开映射定理** | 满射开映射 | 逆算子 |
| **闭图像定理** | 图像闭⇒连续 | 算子判定 |

---

## Hilbert空间详解

### 定义与性质

**定义**: 完备的内积空间 (H, ⟨·,·⟩)

| 性质 | 说明 |
|:---:|:---|
| **内积诱导范数** | ‖x‖ = √⟨x,x⟩ |
| **平行四边形法则** | ‖x+y‖²+‖x-y‖²=2(‖x‖²+‖y‖²)（刻画） |
| **柯西-施瓦茨** | |⟨x,y⟩| ≤ ‖x‖·‖y‖ |

### 正交结构

```

正交补: M⊥ = {x: ⟨x,m⟩=0, ∀m∈M}
正交投影: 对闭子空间M, H = M ⊕ M⊥
标准正交基: {eₙ}, ⟨eᵢ,eⱼ⟩ = δᵢⱼ
Fourier展开: x = Σ ⟨x,eₙ⟩eₙ

```

### 经典例子

| 空间 | 内积 | 正交基 |
|:---:|:---:|:---|
| ℝⁿ, ℂⁿ | ⟨x,y⟩ = Σxᵢȳᵢ | 标准基 |
| L²([0,2π]) | ⟨f,g⟩ = ∫fḡ | {e^{inx}} |
| L²(ℝ) | ⟨f,g⟩ = ∫fḡ | Hermite函数 |
| ℓ² | ⟨x,y⟩ = Σxₙȳₙ | 标准基 |
| H²( Hardy空间) | 边界值L² | — |
| Sobolev空间 H¹ | ⟨f,g⟩₁ = ⟨f,g⟩ + ⟨f',g'⟩ | — |

### 核心定理

| 定理 | 内容 |
|:---:|:---|
| **Riesz表示** | ∀φ∈H*, ∃!y∈H, φ(x)=⟨x,y⟩ |
| **投影定理** | 对闭凸集C, ∃!最近点 |
| **Lax-Milgram** | 双线性形式与PDE弱解 |
| **谱定理** | 正规算子对角化 |

## Sobolev空间详解

### 定义与动机

**动机**: 经典解要求太高，需要"弱解"框架

**定义**: W^{k,p}(Ω) = {u ∈ Lᵖ(Ω): Dᵅu ∈ Lᵖ, |α|≤k}

范数: ‖u‖_{W^{k,p}} = (Σ_{|α|≤k} ‖Dᵅu‖ₚᵖ)^{1/p}

### 空间层次

```

        W^{k,p}(Ω)  (k阶导数在Lᵖ)
             │
    ┌────────┼────────┐
    ▼        ▼        ▼
 Hᵏ(Ω) = W^{k,2}   W^{k,p} (p≠2)   Cᵏ(Ω̄)
 (Hilbert)         (Banach)     (光滑函数)
    │                │             │
    ▼                ▼             ▼
 L²(Ω)            Lᵖ(Ω)          C(Ω̄)
 (零阶)           (零阶)         (连续)

```

### 特例: H¹空间

**定义**: H¹(Ω) = {u ∈ L²: ∇u ∈ (L²)ⁿ}

**内积**: ⟨u,v⟩_{H¹} = ⟨u,v⟩_{L²} + ⟨∇u,∇v⟩_{L²}

**应用**: 二阶椭圆型PDE的变分形式

### 嵌入定理 (Sobolev嵌入)

| 条件 | 嵌入关系 | 意义 |
|:---:|:---:|:---|
| kp < n | W^{k,p} ↪ L^q, q ≤ np/(n-kp) | 可积性提升 |
| kp = n | W^{k,p} ↪ L^q, ∀q < ∞ | 临界情况 |
| kp > n | W^{k,p} ↪ C^{m,α} | 连续性获得 |
| k₁≥k₂, p₁≤p₂ | W^{k₁,p₁} ↪ W^{k₂,p₂} | 层次关系 |

### 迹定理

**问题**: 边界值 ∂u/∂n|_{∂Ω} 有意义吗？

```

迹算子: γ: H¹(Ω) → H^{1/2}(∂Ω)
         γ₀: u ↦ u|_{∂Ω}  (边界值)

         γ₁: u ↦ ∂u/∂n    (法向导数)

定理: γ是满射且有界线性算子

```

## 三大空间对比

| 特性 | Banach | Hilbert | Sobolev |
|:---:|:---:|:---:|:---:|
| **范数来源** | 直接定义 | 内积诱导 | 导数+积分 |
| **几何工具** | 凸性、对偶 | 正交投影 | 嵌入、紧性 |
| **求解方法** | 不动点定理 | 正交投影 | 变分方法 |
| **紧性结果** | 有限维刻画 | 同上 | Rellich紧嵌入 |
| **对偶计算** | 复杂 | 简单(Riesz) | 负指数空间 |
| **谱理论** | 复杂 | 完整(正规算子) | 椭圆算子 |
| **最佳逼近** | 存在性 | 正交投影 | Galerkin方法 |

## PDE中的应用

### 椭圆型方程

**Poisson方程**: -Δu = f in Ω, u=0 on ∂Ω

| 框架 | 空间 | 方法 |
|:---:|:---:|:---|
| **古典解** | C²(Ω) | 极值原理 |
| **弱解** | H₀¹(Ω) | Lax-Milgram |
| **强解** | W^{2,p} | Calderón-Zygmund |
| **分布解** | 𝒟'(Ω) | 广义函数 |

### 发展方程

**热方程**: u_t = Δu

| 空间 | 应用 |
|:---:|:---|
| L²(Ω) | 能量估计 |
| H¹(Ω) | Galerkin逼近 |
| H⁻¹(Ω) | 对偶论证 |

---

## 对偶关系

```

Hilbert空间自对偶:
H* ≅ H (Riesz表示)

Banach空间对偶链:
Lᵖ* ≅ Lᵠ (1/p + 1/q = 1, p∈[1,∞))
(L¹)* = L∞
(L∞)* ≠ L¹ (更大)

Sobolev对偶:
(W^{k,p})* = W^{-k,q}
H^{-1} = (H₀¹)*

包含关系对偶:
H₀¹ ↪ L² ↪ H⁻¹
（Gelfand三重）

```

## 紧性工具

| 空间 | 紧性结果 | 应用 |
|:---:|:---:|:---|
| Banach | 单位球紧⇔有限维 | Riesz引理 |
| Hilbert | 闭单位球弱紧 | 弱收敛抽取子列 |
| Sobolev | **Rellich**: H¹ ↪ L²紧嵌入 | PDE存在性 |

## 学习路径

```

Level 1: 有限维空间
   ├── ℝⁿ, ℂⁿ
   ├── 内积与范数
   └── 矩阵表示

Level 2: 无穷维Banach空间
   ├── 完备性概念
   ├── 经典例子 (C(K), Lᵖ, ℓᵖ)
   └── 四大定理

Level 3: Hilbert空间
   ├── 内积结构
   ├── 正交性
   ├── 标准正交基
   └── Riesz表示

Level 4: Sobolev空间
   ├── 弱导数
   ├── W^{k,p}定义
   ├── 嵌入定理
   └── 迹定理

Level 5: 高级专题
   ├── 插值理论
   ├── 微分算子谱理论
   └── 非线性泛函分析

```
