# 伴随函子与单子对比矩阵

> **摘要**：本文档系统对比范畴论中的伴随函子和单子这两个密切相关但视角不同的概念。伴随函子描述两个范畴之间的结构性对应，而单子则在单一范畴内描述代数结构。通过多维度的对比分析，揭示二者的深刻联系。

## 基本概念对比

### 核心对比表

| 对比维度 | 伴随函子 (Adjunction) | 单子 (Monad) |
|---------|---------------------|-------------|
| **定义层次** | 两个范畴之间的关系 | 单一范畴上的代数结构 |
| **组成** | 函子对 L: C ⇄ D: R + 同构族 | 函子 T: C → C + η + μ |
| **直观理解** | "最佳近似"的双向对应 | "代数表达式"的形式化 |
| **方程视角** | 泛性质（存在唯一性） | 代数恒等式 |
| **范畴论地位** | 基本结构（如群论中的同态） | 代数工具（如群论中的群作用） |
| **典型记号** | L ⊣ R | (T, η, μ) |

### 概念的直观类比

```

伴随函子类比:
        L              R
    C <-----------> D
    (小学)          (中学)
    
    L: 升学 (自由/左伴随)
    R: 回顾基础 (遗忘/右伴随)
    
    伴随关系: 升学后的适应 = 回顾时的巩固

单子类比:
    C = 集合范畴
    T = "代数表达式构造"
    
    X --η--> T(X) = 变量嵌入
    T(T(X)) --μ--> T(X) = 表达式展开

```

## 定义与结构对比

### 伴随函子的定义

**定义**：函子 L: C → D 和 R: D → C 构成**伴随**，记作 L ⊣ R，如果存在自然同构：

$$\text{Hom}_D(LX, Y) \cong \text{Hom}_C(X, RY)$$

**结构要素**：
1. **单位 (Unit)**：η: Id_C ⇒ RL
2. **余单位 (Counit)**：ε: LR ⇒ Id_D

**三角恒等式**：

```

L --Lη--> LRL --εL--> L    =    id_L
R --ηR--> RLR --Rε--> R    =    id_R

```

### 单子的定义

**定义**：范畴 C 上的**单子**是一个三元组 (T, η, μ)，其中：
- T: C → C 是函子
- η: Id_C ⇒ T 是单位（return/pure）
- μ: T² ⇒ T 是乘法（join/flatten）

**公理**（交换图）：

```

结合律:              单位律:
T³ --Tμ--> T²         T --ηT--> T² --μ--> T

 |           |         |              |

 μT          μ         =              =
 v           v         v              v
T² --μ--->  T         T ======== T <--η-- T

```

### 结构要素对比

| 结构要素 | 伴随 L ⊣ R | 单子 (T, η, μ) |
|---------|-----------|---------------|
| **函子** | L (左), R (右) | T = RL |
| **单位** | η: Id ⇒ RL | η: Id ⇒ T |
| **乘法** | ε: LR ⇒ Id | μ = RεL: T² ⇒ T |
| **方向** | C ⇄ D | 在 C 上循环 |
| **方程** | 三角恒等式 | 结合律 + 单位律 |

## 典型实例对比

### 经典伴随实例

| 伴随 | L (左伴随) | R (右伴随) | 伴随同构含义 |
|-----|-----------|-----------|------------|
| **自由-遗忘** | F: Set → Grp | U: Grp → Set | 群同态 ⟺ 集合映射 |
| **张量-Hom** | (-)⊗A: Mod → Mod | Hom(A,-): Mod → Mod | 双线性 ⟺ 线性 |
| **离散-底** | Δ: Set → Top | U: Top → Set | 连续 ⟺ 任意 |
| **像-原像** | f*: P(Y) → P(X) | f_*: P(X) → P(Y) | 原像包含 ⟺ 像包含 |
| **存在-弱化** | ∃: Pred(X) → Pred(Y) | π*: Pred(Y) → Pred(X) | 存在量词伴随 |

### 经典单子实例

| 单子 | 函子 T | 单位 η | 乘法 μ | 应用 |
|-----|-------|-------|-------|-----|
| **幂集单子** | T(A) = P(A) | {−} | 并 | 非确定性计算 |
| **列表单子** | T(A) = A* | [−] | concat | 序列计算 |
| **状态单子** | T(A) = S → (A × S) | λs.(a,s) | 状态组合 | 状态传递 |
| **可能单子** | T(A) = A + 1 | Just | 错误传播 | 异常处理 |
| **连续单子** | T(A) = (A → R) → R | 嵌入 | 扩展 | 连续计算 |

### 自由-遗忘伴随 vs 自由单子

**自由-遗忘伴随**：

```

F: Set ⇄ Grp: U
    F ⊣ U
    
η_A: A → U(F(A))     将元素映射为生成元
ε_G: F(U(G)) → G     将字映射为群元素

```

**诱导的单子**：

```

T = U ∘ F: Set → Set
T(A) = A* (A上的字)
η_A: A → A*, a ↦ [a]
μ_A: A** → A*, 列表的列表 ↦ 拼接

```

这是**自由单子**（列表单子）。

## 构造与分解对比

### Eilenberg-Moore 代数 (单子的代数)

**定义**：设 (T, η, μ) 是 C 上的单子。一个 **T-代数** 是一个对 (A, α)，其中：
- A ∈ Ob(C)
- α: T(A) → A 是结构映射

满足：

```

α ∘ η_A = id_A        (单位律)
α ∘ μ_A = α ∘ T(α)    (结合律)

```

**代数同态**：f: (A, α) → (B, β) 使得 f ∘ α = β ∘ T(f)

### 伴随 vs 单子范畴

| 构造 | 伴随 L ⊣ R | 单子 T |
|-----|-----------|-------|
| **代数范畴** | D（原始范畴） | C^T (Eilenberg-Moore) |
| **比较函子** | K: D → C^T | 标准构造 |
| **满映性** | 不一定 | 由伴随决定 |
| **单子性** | 产生单子 T = RL | 产生伴随 F^T ⊣ U^T |

### 单子的分解

**任何伴随都诱导单子**：

```

L: C ⇄ D: R    ⟹    T = RL 是 C 上的单子

```

**任何单子都来自伴随**：

```

T 是 C 上的单子  ⟹  存在伴随 F^T ⊣ U^T 产生 T
    
F^T: C → C^T,    A ↦ (T(A), μ_A)
U^T: C^T → C,    (A, α) ↦ A

```

### Kleisli 范畴

**Kleisli 范畴** C_T：
- **对象**：与 C 相同
- **态射**：A ⇢ B 对应 C 中的态射 A → T(B)
- **复合** (Kleisli 复合)：g ∘_K f = μ ∘ T(g) ∘ f

**Kleisli 伴随**：

```

F_T: C → C_T,    f ↦ η ∘ f
U_T: C_T → C,    (f: A ⇢ B) ↦ (μ ∘ T(f): T(A) → T(B))
    
F_T ⊣ U_T 且产生原单子 T

```

## 应用与计算对比

### 在函数式编程中的应用

| 应用 | 伴随视角 | 单子视角 | 实现 |
|-----|---------|---------|-----|
| **I/O** | 效果与纯计算的分离 | IO 单子 | Haskell |
| **状态** | 状态转换伴随 | State 单子 | State s a |
| **异常** | 完全偏序伴随 | Maybe/Either 单子 | Maybe a |
| **非确定性** | 幂集伴随 | List 单子 | [a] |
| **延续** | 双重否定伴随 | Cont 单子 | (a -> r) -> r |

### 在代数中的应用

| 代数结构 | 伴随描述 | 单子描述 |
|---------|---------|---------|
| **群** | F ⊣ U: Grp → Set | 群单子 |
| **环** | F ⊣ U: Ring → Set | 环单子 |
| **模** | R⊗− ⊣ Hom(R,−) | - |
| **代数闭包** | 代数闭包伴随 | - |

### 计算复杂性对比

| 操作 | 伴随计算 | 单子计算 | 复杂度 |
|-----|---------|---------|-------|
| **映射** | L(f) 或 R(f) | T(f) = R∘L(f) | 取决于具体函子 |
| **单位** | η: Id ⇒ RL | η: Id ⇒ T | 通常是嵌入 |
| **乘法** | ε: LR ⇒ Id | μ = RεL | 通常是求值/归约 |
| **复合** | 函子复合 | Kleisli 复合 | 需要 μ |

## 伴随-单子对应定理

### 基本对应

```

{C 上的单子}  ≅  {产生 C 上给定单子的伴随} / 等价

```

更准确地说：

**定理**：设 T 是 C 上的单子。

1. **Eilenberg-Moore 伴随** F^T ⊣ U^T: C ⇄ C^T 是**终极**的（最右的）
2. **Kleisli 伴随** F_T ⊣ U_T: C ⇄ C_T 是**初始**的（最左的）
3. 任何产生 T 的伴随 L ⊣ R: C ⇄ D 都分解为：
   ```

   C ⇄ C_T → D → C^T ⇄ C
   ```

### 对应关系图

```

        任意伴随 L ⊣ R: C ⇄ D

                |
                | 产生单子 T = RL

                v
        +-------------------+

        |     C 上的单子 T    |

        +-------------------+
           /             \
          /               \
   Kleisli                Eilenberg-Moore
   范畴 C_T               范畴 C^T
   (最自由)                (最严格)
      \                   /
       \                 /
        v               v
        +---------------+

        |    D (原始)    |

        +---------------+

```

### 三种范畴的关系

| 范畴 | 描述 | T-代数 | 特点 |
|-----|------|-------|-----|
| **C_T (Kleisli)** | 自由代数范畴 | 仅自由代数 (T(A), μ_A) | 最自由 |
| **D (原始)** | 给定范畴 | 某些 T-代数 | 中间 |
| **C^T (E-M)** | 所有 T-代数 | 所有 T-代数 | 最完备 |

**比较函子**：
- J: C_T → D （Kleisli 到原始）
- K: D → C^T （原始到 E-M）

当 D ≅ C^T 时，称伴随是**单子性的**（monadic）。

## 总结

| 特征 | 伴随函子 L ⊣ R | 单子 (T, η, μ) |
|-----|---------------|---------------|
| **本质** | 双向结构对应 | 单范畴代数结构 |
| **组件** | 函子对 + 单位/余单位 | 自函子 + 单位 + 乘法 |
| **方程** | 三角恒等式 | 结合律 + 单位律 |
| **来源** | 原始概念 | 可由伴随诱导 |
| **完备化** | Kleisli/E-M 范畴 | 产生伴随 |
| **应用** | 结构对应、泛性质 | 代数化、计算效果 |

**核心洞见**：伴随和单子是同一枚硬币的两面——伴随提供"外在"的结构对应视角，单子提供"内在"的代数操作视角。它们在数学和计算机科学中都有广泛而深刻的应用。
