## 层的基本类型对比

### 层分类层次

```

预层 ──→ 层 (满足粘合公理)
  │
  ├──→ Abel群层 (Ab_X)
  │       │
  │       ├──→ 𝒪_X-模层 (Mod_{𝒪_X})
  │               │
  │               ├──→ 拟凝聚层 (QCoh_X)
  │               │       │
  │               │       └──→ 凝聚层 (Coh_X) （Noetherian上）
  │               │
  │               └──→ 局部自由层 (= 向量丛)
  │                       │
  │                       ├──→ 线丛 (秩1)
  │                       └──→ 平凡丛
  │
  └──→ 非Abel层 (层叠等)

```

### 𝒪_X-模层类型对比

| 类型 | 定义 | 局部结构 | 全局性质 | 典型例子 |
|-----|-----|---------|---------|---------|
| **预层** | 截面赋值 | - | - | - |
| **层** | 满足粘合 | 局部确定 | - | 连续函数层 |
| **𝒪_X-模** | 截面是环模 | 模结构 | - | 微分形式层 |
| **拟凝聚** | 局部 = M̃ | 模的相伴层 | 平坦极限 | 有理函数层 |
| **凝聚** | 局部有限生成+有限关系 | 有限展示 | Noetherian | 结构层、理想层 |
| **局部自由** | 局部 ≅ 𝒪ⁿ | 自由模 | 向量丛 | 切丛、余切丛 |
| **反身** | ℱ ≅ ℱ∨∨ | 无挠 | 深度≥2 | 法丛 |
| **平坦** | 张量正合 | 平坦模 | 保持正合 | 局部自由 |

## 拟凝聚层 vs 凝聚层

### 定义对比

| 性质 | 拟凝聚层 | 凝聚层 |
|-----|---------|-------|
| **局部结构** | 形如 M̃（模的相伴层） | 有限展示：coker(𝒪ᵐ → 𝒪ⁿ) |
| **有限生成** | 不要求 | 要求 |
| **有限关系** | 不要求 | 要求 |
| **核性质** | 核拟凝聚 | 核凝聚 |
| **有限拉回** | 保持 | 保持 |
| **重要性** | 导出范畴的基础 | 有限计算的基础 |

### 关键定理

**Serre定理**：设X是Noetherian概形。
- 拟凝聚层范畴 ≅ 准凝聚模范畴
- 凝聚层范畴 ≅ 有限生成模范畴（仿射情形）

**Serre有限性**：若 f: X → Y 是概形的真态射，Y是Noetherian的，则 $R^i f_* \mathcal{F}$ 是凝聚的（ℱ凝聚）。

### 构造方法对比

| 构造 | 拟凝聚 | 凝聚 | 条件 |
|-----|-------|-----|-----|
| **相伴层 M̃** | 是 | M有限生成时 | Spec(R) |
| **核** | 是 | 是（凝聚层间） | Noetherian |
| **余核** | 是 | 是 | Noetherian |
| **张量积** | 是 | 是 | - |
| **Hom** | 是 | 是（凝聚层） | Noetherian |
| **推出** | 是 | 不一定 | - |

## 上同调理论对比

### 上同调构造方法

| 方法 | 构造 | 适用场景 | 优点 | 缺点 |
|-----|-----|---------|-----|-----|
| **导出函子** | Hⁱ(X, ℱ) = RⁱΓ(X, ℱ) | 一般 | 理论完备 | 抽象 |
| **Čech** | Čech复形 | 仿射覆盖 | 可计算 | 需精细覆盖 |
| **双复形** | 导出范畴 | 导出函子复合 | 谱序列自然 | 技术复杂 |
| **解析** |  Dolbeault | 复流形 | 分析工具 | 需光滑性 |

### Čech上同调 vs 层上同调

**Čech上同调**：设 $\mathcal{U} = \{U_i\}$ 是开覆盖。

$$\check{C}^p(\mathcal{U}, \mathcal{F}) = \prod_{i_0 < ... < i_p} \mathcal{F}(U_{i_0} \cap ... \cap U_{i_p})$$

$$
(\delta \sigma)_{i_0...i_{p+1}} = \sum_{j=0}^{p+1} (-1)^j \sigma_{i_0...\hat{i_j}...i_{p+1}}|_{U_{i_0} \cap ... \cap U_{i_{p+1}}}

$$

**比较定理**：
- 若X仿射，ℱ拟凝聚，则 $\check{H}^i(\mathcal{U}, \mathcal{F}) = 0$ (i > 0)
- 一般情形：Čech上同调 → 层上同调的谱序列

**Leray定理**：若 $H^q(U_{i_0} \cap ... \cap U_{i_p}, \mathcal{F}) = 0$ ∀ q > 0，则 Čech = 层上同调。

### 上同调计算策略

```

计算 H^i(X, ℱ):

1. 选择仿射开覆盖 𝒰 = {U_i}
        ↓
2. 计算 Čech 复形 Č(𝒰, ℱ)
        ↓
3. 若 ℱ 拟凝聚且 X 分离：Čech = 层上同调
        ↓
4. 计算 Čech 复形的上同调
        ↓
5. 或使用谱序列：E_2^{p,q} = H^p(𝒰, ℋ^q(ℱ)) ⇒ H^{p+q}(X, ℱ)

```

## 特殊层的上同调

### 结构层上同调

**定理 (Serre)**：设 X 是仿射概形，则 $H^i(X, \mathcal{O}_X) = 0$ ∀ i > 0。

**射影空间上同调**：

| 层 | H⁰ | Hⁱ (0<i<n) | Hⁿ |
|---|-----|-----------|-----|
| **𝒪(m), m≥0** | $k[x_0,...,x_n]_m$ | 0 | 0 |
| **𝒪(m), -n-1<m<0** | 0 | 0 | 0 |
| **𝒪(-n-1)** | 0 | 0 | k |
| **𝒪(m), m≤-n-1** | 0 | 0 | $k[x_0,...,x_n]_{-m-n-1}^*$ |

**欧拉示性数**：$\chi(\mathcal{O}(m)) = \binom{m+n}{n}$（m ≥ 0）

### 切层与法层上同调

| 层 | 几何意义 | 上同调意义 | 典型计算 |
|---|---------|-----------|---------|
| **Θ_X = 𝒯_X** | 切丛 | 无穷小自同构 | H⁰ = 向量场 |
| **Ω¹_X** | 余切丛 | 1-形式 | H⁰ = 全纯1-形式 |
| **Ω^p_X** | p-形式 | de Rham上同调 | Hodge分解 |
| **N_{Y/X}** | 法丛 | 形变理论 | H⁰ = 无穷小形变 |

### 上同调长正合列

**短正合列**：$0 → \mathcal{F}' → \mathcal{F} → \mathcal{F}'' → 0$

诱导：

```

0 → H⁰(X, ℱ') → H⁰(X, ℱ) → H⁰(X, ℱ'')
  → H¹(X, ℱ') → H¹(X, ℱ) → ...

```

**应用**：通过已知层的上同调计算未知层。

## 消失定理对比

### 经典消失定理

| 定理 | 条件 | 结论 | 应用 |
|-----|-----|-----|-----|
| **Serre消失** | X射影，ℱ凝聚，L丰富 | Hⁱ(X, ℱ⊗Lⁿ) = 0 (i>0, n>>0) | 整体生成 |
| **Kodaira消失** | X光滑射影，L丰富 | Hⁱ(X, K_X ⊗ L) = 0 (i>0) | 嵌入定理 |
| **Nakano消失** | 同上 | H^q(X, Ω^p ⊗ L) = 0 (p+q>n) | Lefschetz |
| **Grauert-Riemenschneider** | f: X→Y双有理，X光滑 | Rⁱf_*K_X = 0 (i>0) | 奇点消解 |
| **Kawamata-Viehweg** | 对数典范除子 | 推广的Kodaira消失 | 极小模型 |

### 仿射概形上的消失

**定理**：设 X 是仿射概形，ℱ是拟凝聚层，则：
$$H^i(X, \mathcal{F}) = 0 \quad \forall i > 0$$

**逆定理 (Serre)**：若 X 是拟紧分离概形，且对所有拟凝聚层 ℱ 有 $H^1(X, \mathcal{F}) = 0$，则 X 仿射。

### 正性条件与消失

```

正性层次:

丰富 (ample) ──→ 有效 (effective)
     │
     v
消失定理: Hⁱ(X, K_X + ample) = 0 (i > 0)
     │
     v
应用: 射影嵌入、Riemann-Roch

```

## 应用实例

### Riemann-Roch定理

**曲线情形**：X 是光滑射影曲线，亏格 g。

$$\chi(\mathcal{L}) = \deg(\mathcal{L}) + 1 - g$$

** Hirzebruch-Riemann-Roch**：

$$\chi(X, \mathcal{E}) = \int_X \text{ch}(\mathcal{E}) \cdot \text{td}(X)$$

**Grothendieck-Riemann-Roch**：相对版本，涉及 Todd 类和 Chern 特征。

### 对偶性定理

**Serre对偶**（光滑射影簇 X，维数 n）：
$$H^i(X, \mathcal{F})^* \cong \text{Ext}^{n-i}_X(\mathcal{F}, \omega_X)$$

**特殊情况**：ℱ局部自由时，
$$H^i(X, \mathcal{F})^* \cong H^{n-i}(X, \mathcal{F}^\vee \otimes \omega_X)$$

**Grothendieck对偶**：概形态射 f: X → Y 的相对对偶。

### 形变理论中的应用

| 层 | 上同调 | 形变意义 |
|---|-------|---------|
| **H⁰(X, Θ_X)** | 整体向量场 | 无穷小自同构 |
| **H¹(X, Θ_X)** | - | 无穷小形变空间 |
| **H²(X, Θ_X)** | - | 阻碍空间 |
| **H¹(X, N_{Y/X})** | - | 子概形Y的形变 |

## 总结表：层与上同调选择指南

| 问题 | 推荐层类型 | 上同调工具 |
|-----|----------|-----------|
| 全局截面 | 拟凝聚层 | H⁰ |

| 复结构 | 全纯层 | Dolbeault |
