# 数学高级结构对比矩阵

本目录包含FormalMath项目中关于数学高级结构的15个对比矩阵文档，系统对比了现代数学核心领域的关键概念。

## 内容概览

### 范畴论结构对比（3个）

1. **范畴、函子、自然变换对比** - 范畴论的三层基础结构
2. **极限与余极限对比** - 泛性质、构造方法与对偶关系
3. **伴随函子与单子对比** - 伴随-单子对应、Kleisli范畴

### 同调代数对比（3个）

1. **各种同调理论对比** - 奇异、单纯、de Rham、层上同调
2. **导出函子对比** - Tor、Ext、高阶导出函子
3. **谱序列类型对比** - Leray、Serre、LHS等谱序列

### 表示论对比（3个）

1. **群表示类型对比** - 线性、置换、诱导表示
2. **李代数表示对比** - 最高权理论、Verma模
3. **模表示对比** - Brauer特征标、块理论

### 代数几何对比（3个）

1. **各种概形对比** - 仿射、射影、分离概形
2. **层与上同调对比** - 拟凝聚层、消失定理
3. **Motive理论对比** - 纯motive、混合motive

### 微分几何对比（3个）

1. **各种联络对比** - Levi-Civita、主丛联络
2. **各种曲率对比** - 截面、Ricci、Weyl曲率
3. **各种流形对比** - Riemann、辛、复、Kähler流形
