## 导出函子框架概览

### 导出函子的一般框架

**设置**：设 F: A → B 是Abel范畴之间的加性函子。

| 函子类型 | 变元类型 | 分解类型 | 导出方式 | 结果类型 |
|---------|---------|---------|---------|---------|
| **左导出 L_nF** | 右正合 | 投射分解 | 取同调 | 左导出函子 |
| **右导出 R^nF** | 左正合 | 内射分解 | 取上同调 | 右导出函子 |

### 导出函子构造图

```
左导出函子 L_nF:

M ← P_• (投射分解)
     ↓ F
F(P_•) 链复形
     ↓
H_n(F(P_•)) = L_nF(M)

右导出函子 R^nF:

M → I^• (内射分解)
     ↓ F
F(I^•) 上链复形
     ↓
H^n(F(I^•)) = R^nF(M)
```

### 经典导出函子分类

| 函子 | 变元 | 导出类型 | 经典名称 | 消失条件 |
|-----|-----|---------|---------|---------|
| M⊗_R - | 右 | 左导出 | Tor_n^R(M, -) | 平坦模 |
| -⊗_R N | 左 | 左导出 | Tor_n^R(-, N) | 平坦模 |
| Hom_R(M, -) | 左 | 右导出 | Ext^n_R(M, -) | 投射模 |
| Hom_R(-, N) | 右 | 右导出 | Ext^n_R(-, N) | 内射模 |
| f_* (直像) | - | 右导出 | R^if_* |  flasque层 |
| f^! (例外逆像) | - | 左导出 | L_if^! | 局部自由层 |
| Γ(X, -) | - | 右导出 | H^i(X, -) | 松软层 |
| lim | - | 右导出 | lim^i | Mittag-Leffler |

## Tor函子深度对比

### Tor函子的多重定义

| 定义方式 | 公式 | 适用条件 | 优势 |
|---------|-----|---------|-----|
| **第一变元导出** | Tor_n^R(M, N) = H_n(P_• ⊗ N) | M有投射分解 | 标准定义 |
| **第二变元导出** | Tor_n^R(M, N) = H_n(M ⊗ Q_•) | N有投射分解 | 对称性证明 |
| **平衡Tor** | Tor_n^R(M, N) = H_n(P_• ⊗ Q_•) | 双复形 | 不依赖选择 |
| **导出范畴** | M ⊗^L N (导出张量积) | 导出范畴 | 现代观点 |

### Tor函子的关键性质

#### 对称性

**定理**：若 R 是交换环，则：
$$\text{Tor}_n^R(M, N) \cong \text{Tor}_n^R(N, M)$$

**证明概要**：两者都计算 M ⊗^L N 的同调。

#### 长正合列

**短正合列诱导**：若 0 → M' → M → M'' → 0，则：

```
... → Tor_{n+1}(M'', N) → Tor_n(M', N) → Tor_n(M, N) 
    → Tor_n(M'', N) → Tor_{n-1}(M', N) → ...
```

### Tor函子的计算实例

#### 实例1：ℤ-模的Tor

```
Tor_n^ℤ(ℤ/mℤ, ℤ/nℤ) = ?

投射分解: 0 → ℤ --m--> ℤ → ℤ/mℤ → 0

Tor_0 = ℤ/mℤ ⊗ ℤ/nℤ = ℤ/gcd(m,n)ℤ
Tor_1 = ker(ℤ/n --m--> ℤ/n) = ℤ/gcd(m,n)ℤ
Tor_n = 0 (n ≥ 2)
```

**结论**：Tor_1^ℤ(ℤ/m, ℤ/n) = ℤ/gcd(m,n)ℤ，捕捉"公共挠子"。

#### 实例2：多项式环

```
R = k[x,y], M = R/(x), N = R/(y)

Tor_1^R(M, N) = ?

利用: 0 → R --x--> R → M → 0

Tor_1 = ker(N --x--> N) = {n ∈ N : xn = 0}
       = (y)/(xy) ≅ R/(x,y) = k
```

### Tor_1的直观解释

```
挠子的代数表现:

Tor_1^R(M, N) = 0  ⟺  "M和N在张量时不产生新的挠"

具体地，若 M = R/I, N = R/J:
Tor_1^R(R/I, R/J) = (I ∩ J) / (IJ)
```

## Ext函子深度对比

### Ext函子的多重定义

| 定义方式 | 公式 | 变元位置 | 特点 |
|---------|-----|---------|-----|
| **第一变元投射** | Ext^n(M, N) = H^n(Hom(P_•, N)) | 第一变元 | 标准 |
| **第二变元内射** | Ext^n(M, N) = H^n(Hom(M, I^•)) | 第二变元 | 等价 |
| **双复形** | H^n(Tot(Hom(P_•, I^•))) | 两者 | 平衡 |
| **导出Hom** | R^nHom(M, N) | 范畴论 | 统一视角 |
| **Yoneda Ext** | n-扩张的等价类 | 范畴论 | 构造性 |

### Ext^1的扩张解释

**定义**：一个 **n-扩张** 是正合列：

$$\xi: 0 → N → E_n → E_{n-1} → ... → E_1 → M → 0$$

**Baer 和**：对两个扩张 ξ, ξ'，Baer 和 ξ + ξ' 通过推出-拉回构造。

**定理**：Ext^1_R(M, N) ≅ {N被M的扩张} / 等价

#### 扩张构造示例

```
Ext^1_ℤ(ℤ/2ℤ, ℤ/2ℤ) = ℤ/2ℤ

两种扩张:
1. 分裂扩张: 0 → ℤ/2 → ℤ/2 ⊕ ℤ/2 → ℤ/2 → 0
2. 非分裂扩张: 0 → ℤ/2 → ℤ/4 → ℤ/2 → 0

对应 Ext^1 = {0, [非分裂]} ≅ ℤ/2ℤ
```

### Ext函子的长正合列

#### 第一变元正合列

若 0 → M' → M → M'' → 0，则：

```
0 → Hom(M'', N) → Hom(M, N) → Hom(M', N)
  → Ext^1(M'', N) → Ext^1(M, N) → Ext^1(M', N)
  → Ext^2(M'', N) → ...
```

#### 第二变元正合列

若 0 → N' → N → N'' → 0，则类似的长正合列。

### Ext与扩张的层次

| Ext^n | 解释 | 构造方法 |
|------|-----|---------|
| **Ext^0** | Hom(M, N) | 同态 |
| **Ext^1** | 扩张 | Baer 和 |
| **Ext^2** | 2-扩张 | Yoneda 复合 |
| **Ext^n** | n-扩张 | 高阶Yoneda积 |

### Yoneda积

**定义**：∪: Ext^m(A, B) × Ext^n(B, C) → Ext^{m+n}(A, C)

**构造**：扩张的拼接
```
ξ: 0 → B → E_m → ... → E_1 → A → 0
ζ: 0 → C → F_n → ... → F_1 → B → 0

ξ ∪ ζ: 0 → C → F_n → ... → F_1 → E_m → ... → E_1 → A → 0
```

## 高阶导出函子对比

### 层上同调的导出函子

| 函子 | 定义 | 计算工具 | 典型应用 |
|-----|-----|---------|---------|
| **H^i(X, F)** | R^iΓ(X, F) | Čech覆盖 | 几何不变量 |
| **R^if_*F** | 高阶直像 | Leray谱序列 | 纤维丛 |
| **Ext^i_X(F, G)** | 层Ext | 局部-整体谱序列 | 对偶性 |
| **H^i_c(X, F)** | 紧支上同调 | 紧化方法 | Poincaré对偶 |

### 局部上同调

**定义**：设 Z ⊆ X 是闭子集，F 是层。

$$H^i_Z(X, F) = R^i\Gamma_Z(X, F)$$

其中 Γ_Z 是支集在Z内的截面函子。

**长正合列**：
```
... → H^i_Z(X, F) → H^i(X, F) → H^i(X\Z, F) → H^{i+1}_Z(X, F) → ...
```

### 导出范畴视角

**导出范畴 D(A)**：
- 对象：A 中的链复形
- 态射：链映射的局部化（拟同构求逆）

**导出函子**：
```
LF: D(A) → D(B)  （左导出）
RF: D(A) → D(B)  （右导出）

经典导出函子 = 同调/上同调:
L_iF(X) = H_i(LF(X))
R^iF(X) = H^i(RF(X))
```

**优势**：
- 自然处理复合函子
- 谱序列的自然来源
- 三角范畴结构

## 导出函子计算技术对比

### 分解选择策略

| 函子 | 推荐分解 | 替代分解 | 效率考虑 |
|-----|---------|---------|---------|
| **Tor** | 自由分解 (模) | 平坦分解 | 平坦分解更短 |
| **Ext** | 投射分解 | 自由分解 | 自由分解可计算 |
| **层上同调** | 内射分解 | Čech分解 | Čech可具体计算 |
| **群上同调** | Bar分解 | 标准分解 | Bar分解通用 |

### 谱序列作为计算工具

#### Grothendieck谱序列

**设置**：A --F--> B --G--> C，G 左正合，F 将内射对象映到 G-零调对象。

**结论**：
$$E_2^{p,q} = R^pG(R^qF(X)) \Rightarrow R^{p+q}(G∘F)(X)$$

**应用实例**：
- Leray谱序列（f: X → Y）：H^p(Y, R^qf_*F) ⇒ H^{p+q}(X, F)
- 局部-整体Ext谱序列：H^p(X, Ext^q(F,G)) ⇒ Ext^{p+q}_X(F,G)

### 维度位移 (Dimension Shifting)

**技巧**：若 0 → M' → P → M → 0 中 P 投射，则：

$$\text{Ext}^n(M, N) \cong \text{Ext}^{n-1}(M', N), \quad n \geq 2$$

**用途**：通过归纳法证明关于Ext的一般命题。

## 应用实例对比

### 同调代数中的应用

| 应用 | 涉及的导出函子 | 核心结果 |
|-----|--------------|---------|
| **通用系数定理** | Tor | H_n(X;G) ≅ H_n(X)⊗G ⊕ Tor(H_{n-1}(X), G) |
| **万有系数定理(上同调)** | Ext | H^n(X;G) ≅ Hom(H_n(X), G) ⊕ Ext(H_{n-1}(X), G) |
| ** Künneth 公式** | Tor | H_*(X×Y) ≅ H_*(X)⊗H_*(Y) ⊕ Tor(H_*(X), H_*(Y)) |
| **对偶性定理** | Ext | 局部对偶、Grothendieck对偶 |
| ** finiteness定理** | R^if_* | 凝聚层的 pushforward 凝聚 |

### 代数几何中的应用

| 应用 | 导出函子 | 结果 |
|-----|---------|-----|
| **Serre对偶** | Ext^n_X | H^i(X, F)^* ≅ Ext^{n-i}_X(F, ω_X) |
| **Riemann-Roch** | χ(F) = Σ(-1)^i dim H^i(X, F) | 示性类公式 |
| **消失定理** | Kodaira, Nakano | H^q(X, K⊗L) = 0 (q > 0, L丰富) |
| **基变换** | L^if^* | 平坦基变换定理 |

### 表示论中的应用

| 应用 | 函子 | 结果 |
|-----|-----|-----|
| **诱导表示** | Ind_H^G | Frobenius互反: Hom_G(Ind V, W) ≅ Hom_H(V, Res W) |
| **Shapiro引理** | H^*(G, Ind V) ≅ H^*(H, V) | 诱导与上同调交换 |
| **Tate上同调** | 修正的群上同调 | 周期性与类域论 |
