# 建模方法对比：确定性建模 vs 随机建模

## 基本信息

| 维度 | 内容 |
|------|------|
| **对比主题** | 处理不确定性的两种范式 |
| **核心问题** | 内在随机性 vs 认知不确定性 |
| **哲学分歧** | 决定论 vs 随机论 |

---

## 核心特征矩阵

| 特征维度 | 确定性建模 (Deterministic) | 随机建模 (Stochastic) |
|----------|----------------------------|-----------------------|
| **世界观** | 拉普拉斯决定论 | 概率性、统计规律 |
| **预测特性** | 点预测、唯一轨迹 | 分布预测、多轨迹 |
| **方程类型** | 常微分/偏微分方程 | 随机微分方程、主方程 |
| **初始条件** | 精确给定 | 概率分布 |
| **可重复性** | 完全可重复 | 统计意义上可重复 |
| **典型输出** | 函数 y(t) | 概率分布 P(y,t) 或样本路径 |

---

## 技术特征对比

| 技术指标 | 确定性建模 | 随机建模 |
|----------|------------|----------|
| **信息量** | 单轨信息 | 分布信息（更完整） |
| **计算成本** | 单次模拟 | 多次模拟（蒙特卡洛） |
| **分析复杂度** | 经典分析工具 | 需要随机分析 |
| **参数敏感性** | 可能混沌 | 平滑化（平均效应） |
| **预测置信度** | 虚假精确 | 量化不确定性 |
| **模型识别** | 参数估计 | 分布估计 |

---

## 数学工具对比

| 工具类型 | 确定性 | 随机性 |
|----------|--------|--------|
| **基本方程** | ODE: ẋ = f(x) | SDE: dX = f(X)dt + g(X)dW |
| **解的性质** | 轨迹 | 扩散过程、鞅 |
| **数值方法** | Runge-Kutta | Euler-Maruyama、Milstein |
| **长期行为** | 不动点、极限环 | 稳态分布、遍历性 |
| **分析工具** | Lyapunov方法 | Itô公式、Fokker-Planck |

## 应用领域对比

| 领域 | 确定性模型 | 随机模型 | 选择依据 |
|------|------------|----------|----------|
| **经典力学** | 牛顿方程 | 朗之万方程 | 系统尺度 |
| **量子力学** | 薛定谔方程 | 量子随机过程 | 诠释 |
| **金融** | 贴现现金流 | Black-Scholes、随机波动率 | 风险管理 |
| **流行病学** | SIR ODE | 随机SIR、.agent模型 | 种群规模 |
| **神经网络** | 确定性动力学 | 随机梯度下降、Dropout | 训练/泛化 |
| **排队论** | 流体近似 | 随机过程 | 负载水平 |

## 随机微分方程（SDE）基础

### Itô过程

```
dX_t = μ(X_t,t)dt + σ(X_t,t)dW_t

其中：
- μ: 漂移系数（确定性趋势）
- σ: 扩散系数（波动强度）
- dW_t: 维纳过程增量
```

### 经典SDE示例

| 名称 | 方程 | 应用 |
|------|------|------|
| **几何布朗运动** | dS = μSdt + σSdW | 股票价格 |
| **Ornstein-Uhlenbeck** | dX = -θXdt + σdW | 利率、均值回归 |
| **CIR模型** | dr = a(b-r)dt + σ√rdW | 利率期限结构 |

## 不确定性来源分类

| 来源 | 描述 | 处理方式 |
|------|------|----------|
| **初始条件** | 测量不确定性 | 概率初始分布 |
| **参数** | 模型参数不确定 | 随机参数、贝叶斯 |
| **外部噪声** | 环境涨落 | 加性/乘性噪声 |
| **内在随机性** | 量子效应等 | 本质随机模型 |
| **模型误差** | 简化误差 | 模型平均、集成 |

## 蒙特卡洛方法

| 方面 | 描述 |
|------|------|
| **基本思想** | 用大量随机样本估计期望值 |
| **收敛速度** | O(1/√N)（与维度无关） |
| **方差缩减** | 重要性采样、控制变量、分层抽样 |
| **准蒙特卡洛** | 低差异序列替代随机数 |
| **马尔可夫链MCMC** | 复杂分布采样 |

---

## 从确定性到随机性

| 过渡路径 | 机制 | 示例 |
|----------|------|------|
| **微小噪声** | 小扰动分析 | 随机稳定性 |
| **大数极限** | 多粒子平均 | 从微观到宏观 |
| **混沌系统** | 确定性混沌 | 统计描述有效 |
| **粗粒化** | 忽略细节 | 有效方程 |

---

## 混合建模策略

| 策略 | 描述 | 应用 |
|------|------|------|
| **分段确定** | 随机切换确定性模式 | 马尔可夫切换模型 |
| **随机参数** | 参数为随机变量 | 随机有限元 |
| **平均化** | 快变量随机平均 | 多尺度系统 |
| **集合预报** | 多初值集合 | 天气预报 |

---

## 不确定性量化（UQ）

| 方法 | 类型 | 特点 |
|------|------|------|
| **多项式混沌** | 谱方法 | 快速收敛（光滑问题） |
| **随机配点** | 非侵入式 | 重用确定性求解器 |
| **Galerkin投影** | 侵入式 | 一次求解得分布 |
| **贝叶斯推断** | 统计 | 数据更新信念 |

---

## 选择决策矩阵

| 情况 | 推荐模型 | 理由 |
|------|----------|------|
| 系统简单、数据精确 | 确定性 | 简洁、可解释 |
| 大规模粒子系统 | 随机 | 统计描述有效 |
| 需要风险量化 | 随机 | 自然提供置信区间 |
| 混沌系统长期预测 | 随机 | 确定性预测失效 |
| 数据稀缺 | 确定性+贝叶斯 | 参数不确定性 |
