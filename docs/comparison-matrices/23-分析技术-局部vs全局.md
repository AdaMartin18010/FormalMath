# 分析技术对比：局部分析 vs 全局分析

## 基本信息

| 维度 | 内容 |
|------|------|
| **对比主题** | 观察数学对象的两种尺度 |
| **核心问题** | 微观行为 vs 宏观性质 |
| **几何对应** | 微分几何 vs 代数拓扑 |

---

## 核心特征矩阵

| 特征维度 | 局部分析 (Local) | 全局分析 (Global) |
|----------|------------------|-------------------|
| **观察尺度** | 点邻域、无穷小 | 整体空间、大范围 |
| **主要工具** | 导数、线性化、泰勒展开 | 积分、同调、指标定理 |
| **信息类型** | 变化率、切空间 | 拓扑不变量、整体性质 |
| **典型问题** | 可微性、曲率 | 连通性、紧致性、分类 |
| **分析方法** | 微分 | 积分、拓扑 |
| **稳定性** | 对扰动敏感 | 拓扑鲁棒 |

---

## 技术特征对比

| 技术指标 | 局部分析 | 全局分析 |
|----------|----------|----------|
| **计算复杂度** | 通常较低 | 可能很高 |
| **信息密度** | 高（某点详细信息） | 低（整体概括） |
| **反例可能性** | 局部性质不保证整体 | 整体性质约束局部 |
| **推广性** | 标准技术适用 | 需要全局结构 |
| **可视化** | 放大、局部分析 | 整体图像、全景 |
| **证明风格** | 构造性、计算性 | 存在性、拓扑性 |

---

## 微分几何视角

| 概念 | 局部版本 | 全局版本 |
|------|----------|----------|
| **曲率** | Gauss曲率（点定义） | 全曲率、Chern类 |
| **测地线** | 短程（局部） | 最短路径（全局） |
| **向量场** | 局部坐标表示 | 整体存在性（障碍类）|
| **流形结构** | 局部坐标卡 | 整体图册、分类 |

| 工具 | 描述 | 应用 |
|------|------|------|
| **导数** | 变化率、线性逼近 | 极值、单调性 |

## 经典桥梁：局部-全局原理

| 原理 | 陈述 | 领域 |
|------|------|------|
| **Stokes定理** | ∫_M dω = ∫_{∂M} ω | 微分形式 |
| **Gauss-Bonnet** | ∫_M K dA = 2πχ(M) | 微分几何 |
| **Poincaré-Hopf** | 向量场指数和 = χ(M) | 动力系统 |
| **指标定理** | Analytic index = Topological index | 全局分析 |
| **Hasse原理** | 局部可解 ⇏ 全局可解 | 数论 |

| 应用场景 | 局部分析 | 全局分析 |
|----------|----------|----------|
| **优化** | 梯度下降、牛顿法 | 全局最优性证明 |

## 具体案例对比

### 函数极值

**局部分析**
```
寻找临界点：f'(x) = 0
判断类型：f''(x) > 0 → 极小值
局限：可能找到鞍点或局部极值
```

**全局分析**
```
方法：
1. 比较所有临界点
2. 分析无穷远行为
3. 紧致集上极值定理
结果：全局最大/最小值
```

### 向量场指数

| 方面 | 局部 | 全局 |
|------|------|------|
| **定义** | 孤立奇点的指数 | 所有奇点指数之和 |
| **计算** | 局部环绕数 | χ(M)（拓扑不变）|
| **约束** | 自由 | 受欧拉示性数约束 |

## 现代发展

| 方向 | 描述 | 工具 |
|------|------|------|
| **非交换几何** | 空间的代数化 | C*-代数、谱三元组 |
| **量子上同调** | 代数曲线的计数 | Gromov-Witten不变量 |
| **持续同调** | 数据分析中的拓扑 | 滤过、barcode |
| **微局部分析** | 相空间局部化 | 拟微分算子 |

## 选择决策矩阵

| 问题特征 | 推荐分析 | 理由 |
|----------|----------|------|
| 需要具体数值 | 局部 | 可计算 |
| 分类问题 | 全局 | 不变量区分 |
| 稳定性分析 | 局部 | 线性化有效 |
| 存在性证明 | 全局 | 拓扑方法 |
| 渐近行为 | 局部+全局 | 两者结合 |
