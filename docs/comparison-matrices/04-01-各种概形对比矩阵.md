## 概形分类概览

### 概形层次结构

```

环 ──→ 仿射概形 Spec(R)
  │
  ├──→ 准仿射概形 (仿射开子概形)
  │
  ├──→ 射影概形 Proj(S)
  │
  ├──→ 一般概形 (局部仿射)
  │
  └──→ 代数空间 ( étale等价关系)
           │
           └──→ Deligne-Mumford叠 (有限稳定化子)
                    │
                    └──→ Artin叠 (一般叠)

```

### 概形类型总览

| 类型 | 定义特征 | 局部性质 | 整体性质 | 典型例子 |
|-----|---------|---------|---------|---------|
| **仿射概形** | Spec(R) | 整体仿射 | - | 𝔸ⁿ, Spec(ℤ) |
| **射影概形** | Proj(S) | 局部仿射 | 射影 | ℙⁿ, 射影簇 |
| **拟射影概形** | 开子概形 | 局部仿射 | 拟射影 | 仿射+射影 |
| **分离概形** | Δ闭浸入 | Hausdorff-like | - | 大多数概形 |
| **真概形** | 泛闭+分离+有限型 | 射影态射 | 紧性 | 射影概形 |
| **整概形** | 既约+不可约 | 整环 | 连通 | Spec(整环) |
| **正规概形** | 局部正规环 | 正规 | - | 正规簇 |
| **光滑概形** | 局部光滑 | 正则局部环 | - | 光滑簇 |

## 仿射概形 vs 射影概形

### 基本定义对比

| 对比维度 | 仿射概形 Spec(R) | 射影概形 Proj(S) |
|---------|-----------------|-----------------|
| **构造来源** | 交换环 R | 分次环 S = ⊕Sₙ |
| **点** | 素理想 𝔭 ∈ Spec(R) | 齐次素理想 𝔭 ⊉ S₊ |
| **拓扑** | Zariski拓扑：V(I)闭 | Zariski拓扑：V₊(I)闭 |
| **结构层** | 𝒪(U) = R在U上的正则函数 | 齐次局部化 |
| **整体截面** | Γ(Spec(R), 𝒪) = R | Γ(Proj(S), 𝒪(n)) = Sₙ |
| **函子性** | 环同态 → 概形态射 | 分次同态 → 概形态射 |

### 仿射概形的性质

**基本性质**：
- 仿射概形构成概形范畴的"基"（每个概形被仿射开覆盖）
- 仿射开子集是概形的"坐标卡"
- 仿射概形的范畴 ≅ 交换环的范畴^{op}

**关键定理**：
- **Serre定理**：X仿射 ⟺ X分离+拟紧+H^i(X, ℱ) = 0 ∀ i > 0, ∀拟凝聚层ℱ

**例子**：

| 环 R | Spec(R) | 几何意义 |
|-----|---------|---------|
| k[x₁,...,xₙ] | 𝔸ⁿ | 仿射n空间 |
| k[x,y]/(xy) | 两直线在原点相交 | 非既约 |
| ℤ | Spec(ℤ) | 算术点 |
| ℤ/pℤ | 点 (特征p) | 有限域 |

### 射影概形的性质

**构造**：对分次环 S = ⊕_{n≥0} Sₙ，Proj(S) 由齐次素理想（不含 S₊ = ⊕_{n>0} Sₙ）组成。

**射影空间**：
$$\mathbb{P}^n_k = \text{Proj}(k[x_0, x_1, ..., x_n])$$

**开覆盖**：$\mathbb{P}^n = U_0 \cup U_1 \cup ... \cup U_n$，其中 $U_i = D_+(x_i) \cong \mathbb{A}^n$。

**线丛**：射影概形上有自然的丰富线丛 𝒪(1)。

### 仿射 vs 射影的核心差异

| 性质 | 仿射 | 射影 |
|-----|-----|-----|
| **整体函数** | 多 | 少（常值） |
| **闭子概形** | 任意 | 由齐次理想给出 |
| **紧性** | 否 | 是（真概形） |
| **凝聚层** | 由模给出 | 复杂（需上同调） |
| **几何直观** | 开集 | 紧流形类比 |

## 概形性质层次

### Noetherian性质

| 层次 | 定义 | 例子 | 非例子 |
|-----|-----|-----|-------|
| **局部Noetherian** | 有仿射开覆盖Spec(Rᵢ)，Rᵢ Noetherian | 大多数概形 | 无限次爆破 |
| **Noetherian** | 局部Noetherian + 拟紧 | 有限型概形 | - |

**重要性**：Noetherian概形上凝聚层 = 有限生成层。

### 既约与整性

| 性质 | 定义 | 局部环 | 例子 |
|-----|-----|-------|-----|
| **既约** | 无幂零函数 | 既约（无幂零） | Spec(k[x,y]/(xy)) |
| **整** | 既约+不可约 | 整环 | Spec(k[x,y]) |
| **正规** | 局部正规环 | 整闭 | Spec(k[x,y]/(y²-x³)) 非正规 |
| **正则** | 局部正则环 | 正则局部环 | 光滑点 |

**层次关系**：

```

光滑 ──→ 正则 ──→ 正规 ──→ 整 ──→ 既约 ──→ 一般概形

```

### 维数与光滑性

**维数**：
- **Krull维数**：素理想链的最大长度
- **切维数**：切空间的维数

**光滑点**：切维数 = 维数，且局部环正则。

**奇异点**：非光滑点。

## 特殊概形类型

### 群概形

**定义**：概形 G/S 带有乘法、单位、逆元的态射，满足群公理。

| 类型 | 定义 | 例子 |
|-----|-----|-----|
| **加法群** 𝔾ₐ | Spec(ℤ[t])，群律 t ↦ t⊗1 + 1⊗t | 加法群 |
| **乘法群** 𝔾ₘ | Spec(ℤ[t,t⁻¹])，群律 t ↦ t⊗t | GL₁ |
| **一般线性群** GLₙ | 可逆矩阵概形 | 代数群 |
| **椭圆曲线** | 亏格1真群概形 | 阿贝尔簇 |

### 曲线与曲面

| 类型 | 维数 | 分类工具 | 主要不变量 |
|-----|-----|---------|----------|
| **曲线** | 1 | 亏格 g | g = h¹(X, 𝒪_X) |
| **曲面** | 2 | Enriques分类 | κ, $p_g, q, c_2$ |
| **阿贝尔簇** | g | 复环面/代数 | 极化、自同态 |

### 模空间概形

| 模空间 | 参数化对象 | 存在性 | 紧化 |
|-------|-----------|-------|-----|
| **ℳ_g** | 光滑亏格g曲线 | Deligne-Mumford叠 | Deligne-Mumford紧化 |
| **$\mathcal{A}_g$** | 主极化阿贝尔簇 | 概形（主极化） | Satake紧化 |
| **Bun_G(X)** | 曲线X上的G-丛 | Artin叠 | 各种紧化 |

## 概形态射对比

### 态射性质层次

| 性质 | 定义要点 | 几何意义 |
|-----|---------|---------|
| **局部有限型** | 局部：A → B，B有限生成A-代数 | 有限维纤维 |
| **有限型** | 局部有限型 + 拟紧 | 相对有限维 |
| **有限** | 局部：B有限生成A-模 | 真态射+有限纤维 |
| **仿射** | 原像仿射 ⟹ 源仿射 | 仿射丛 |
| **射影** | 因子分解：浸入 + ℙⁿ投影 | 紧纤维 |
| **真** | 泛闭 + 分离 + 有限型 | 紧性类比 |
| **平坦** | 局部：B平坦A-模 | 纤维维数常数 |
| **光滑** | 平坦 + 纤维光滑 | 纤维丛类比 |
| **平展** | 平坦 + 非分歧 + 局部有限 presentation | 局部同构 |

### 重要态射例子

| 态射 | 性质 | 应用 |
|-----|-----|-----|
| **闭浸入** | 有限、单 | 闭子概形 |
| **开浸入** | 平展 | 开子概形 |
| **范态射** | 有限 | Weil限制 |
| **Frobenius** | 纯不可分 | 特征p几何 |
