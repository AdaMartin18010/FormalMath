# 分析学概念对比矩阵：积分理论

## 概述

本文档对比分析学中的主要积分理论：黎曼积分(Riemann Integral)、勒贝格积分(Lebesgue Integral)和斯蒂尔杰斯积分(Riemann-Stieltjes Integral)，阐明其定义、性质、关系和适用范围。

---

## 核心对比矩阵

| 维度 | 黎曼积分 | 勒贝格积分 | 斯蒂尔杰斯积分 |
|:---:|:---:|:---:|:---:|
| **提出者** | B. Riemann (1854) | H. Lebesgue (1902) | T.J. Stieltjes (1894) |
| **分割基础** | 定义域分割 | 值域分割 | 广义区间长度 |
| **可积函数类** | 几乎处处连续函数 | 所有可测函数 | 取决于积分器 |
| **极限与积分交换** | 条件严格 | 控制收敛定理 | 条件类似黎曼 |
| **完备性** | 不完备 | 完备 (Banach空间) | 不完备 |
| **绝对可积性** | 独立概念 | 可积⇔绝对可积 | 视情况而定 |
| **推广性** | 有限区间 | 任意测度空间 | 有界变差函数 |
| **计算便利性** | Newton-Leibniz公式 | 需测度论基础 | 分部积分公式 |

## 详细对比分析

### 1. 黎曼积分 (Riemann Integral)

**定义**: 对[a,b]的分割 P: a=x₀<x₁<...<xₙ=b

```

黎曼和: S(P,f) = Σ f(ξᵢ)(xᵢ - xᵢ₋₁)
积分:   ∫ₐᵇ f(x)dx = lim_{‖P‖→0} S(P,f)

```

| 特征 | 说明 | 局限 |
|:---:|:---:|:---:|
| **可积条件** | 几乎处处连续（有界） | 不连续点太多则不可积 |
| **Darboux和** | 上和=下和时Riemann可积 | 判别法较复杂 |
| **性质保持** | 线性、区间可加 | 极限交换困难 |
| **微积分基本定理** | 成立 | 要求f连续 |

**Riemann可积的充要条件**: f有界且间断点集为零测集

**经典反例**: 
- Dirichlet函数（有理点取1，无理点取0）: 黎曼不可积
- 特征函数 ℚ∩[0,1]: 处处不连续，黎曼不可积

### 2. 勒贝格积分 (Lebesgue Integral)

**定义**: 通过简单函数逼近

```

简单函数: φ = Σ aᵢ·1_{Eᵢ}
积分:     ∫φ = Σ aᵢ·μ(Eᵢ)
一般积分: ∫f = sup{∫φ: 0≤φ≤f, φ简单}

```

| 特征 | 说明 | 优势 |
|:---:|:---:|:---:|
| **可积函数类** | 可测函数（包括病态函数） |  vastly larger |
| **收敛定理** | 单调收敛、控制收敛、Fatou | 极限交换容易 |
| **完备性** | L¹是Banach空间 | 分析学基础 |
| **Fubini定理** | 条件自然（可积即可） | 重积分理论简洁 |

**三大收敛定理**:
1. **单调收敛定理**: 0 ≤ fₙ ↗ f ⇒ ∫fₙ → ∫f
2. **Fatou引理**: ∫liminf fₙ ≤ liminf ∫fₙ
3. **控制收敛定理**: |fₙ| ≤ g, g可积, fₙ → f a.e. ⇒ ∫fₙ → ∫f

**Lebesgue可积但Riemann不可积**: Dirichlet函数（Lebesgue积分=0）

### 3. 斯蒂尔杰斯积分 (Riemann-Stieltjes Integral)

**定义**: 相对于有界变差函数α的积分

```

R-S和: S(P,f,α) = Σ f(ξᵢ)[α(xᵢ) - α(xᵢ₋₁)]
积分:  ∫ₐᵇ f(x)dα(x) = lim_{‖P‖→0} S(P,f,α)

```

| 特征 | 说明 | 应用 |
|:---:|:---:|:---:|
| **积分器α** | 有界变差函数 | 概率分布函数 |
| **离散质量** | α跳跃⇒点质量 | 离散+连续混合 |
| **分部积分** | ∫fdg = fg - ∫gdf | 计算工具 |
| **与黎曼关系** | α(x)=x时=黎曼积分 | 包含黎曼作为特例 |

**存在条件**: f连续且α有界变差 ⇒ R-S可积

**重要特例**:
- α(x) = θ(x-a) (Heaviside函数): ∫fdα = f(a)
- α跳跃: 积分=离散求和
- α绝对连续: 化为Lebesgue积分 ∫fα'dx

## 极限与积分交换

### 黎曼积分的困难

```

问题: 何时 ∫ lim fₙ = lim ∫ fₙ ?

黎曼需要: 一致收敛
   fₙ ⇉ f ⇒ ∫fₙ → ∫f

反例: fₙ(x) = n·1_{(0,1/n]}(x)
   - 逐点收敛到0
   - ∫₀¹ fₙ = 1 ↛ 0 = ∫₀¹ 0

```

### 勒贝格积分的优势

```

控制收敛定理 (DCT):

|fₙ| ≤ g ∈ L¹, fₙ → f a.e.

────────────────────────────
    ∫fₙ → ∫f

应用: 积分号下求导、级数逐项积分

```

### R-S积分

类似黎曼，通常需要一致收敛。

## 微积分基本定理对比

| 积分类型 | 基本定理形式 | 条件 |
|:---:|:---:|:---|
| 黎曼 | F'(x)=f(x) ⇒ ∫ₐᵇf = F(b)-F(a) | f连续 |
| 黎曼 | f可积，F(x)=∫ₐˣf(t)dt ⇒ F连续 | f可积 |
| 勒贝格 | f可积，F(x)=∫ₐˣf(t)dm ⇒ F'(x)=f(x) a.e. | — |
| 勒贝格 | F绝对连续 ⇒ F(x)-F(a)=∫ₐˣF'(t)dm | F绝对连续 |
| R-S | 分部积分 | f,g中一个连续，一个有界变差 |

---

## 多维积分与Fubini定理

| 积分类型 | Fubini定理条件 | 结论 |
|:---:|:---:|:---|
| 黎曼 | f连续（或几乎处处连续） | 重积分=累次积分 |
| 勒贝格 | f可积（或f≥0可测） | 重积分=累次积分 |
| R-S | 复杂，需要一致收敛 | 视情况而定 |

**关键差异**:
- 黎曼需要f在矩形区域上几乎处处连续
- Lebesgue只需要f在乘积测度空间上可积

## 概率论视角

```

概率空间 (Ω, F, P) 上的积分 = 期望

黎曼-Stieltjes积分 (分布函数F):
E[X] = ∫_{-∞}^{∞} x dF(x)

勒贝格积分:
E[X] = ∫_Ω X(ω) dP(ω)

关系:
当 X 有密度 f 时: dF(x) = f(x)dx
∫ x dF(x) = ∫ xf(x)dx

离散分布: dF 是纯点测度
∫ x dF(x) = Σ xᵢ P(X=xᵢ)

```

## 计算实例对比

### 计算 ∫₀¹ x² dx

| 方法 | 过程 | 结果 |
|:---:|:---:|:---:|
| 黎曼 | lim Σ (i/n)²·(1/n) = lim (n(n+1)(2n+1))/(6n³) | 1/3 |
| Lebesgue | 连续函数可积，用原函数 | 1/3 |
| Newton-Leibniz | [x³/3]₀¹ | 1/3 |

| 黎曼 | 不可积 | 处处不连续 |
| Lebesgue | 1 | D(x)=1 a.e.（有理数零测） |
| Lebesgue | = m([0,1]\ℚ) = 1 | |

**核心洞见**: 勒贝格积分通过改变分割方式（值域而非定义域）和引入测度概念，解决了黎曼积分的根本局限，为现代分析学奠定了坚实基础。
