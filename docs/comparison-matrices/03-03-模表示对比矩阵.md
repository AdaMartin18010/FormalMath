## 模表示论概述

### 什么是模表示

**定义**：设 G 是有限群，p 是素数。当域 K 的特征 char(K) = p 且 p 整除 |G| 时，KG-模称为**模表示**。

**关键区别**：
- **常表示**（char(K) = 0 或 p ∤ |G|）：Maschke定理成立，完全可约
- **模表示**（char(K) = p | |G|）：Maschke定理失效，存在不可分解但可约的模

### 模表示论的核心问题

```

给定有限群 G 和特征 p | |G|:

Q1: 不可分解表示的分类？
   ↓
Q2: 不可约表示的分类？（相对于Jacobson根）
   ↓
Q3: 投射覆盖的结构？
   ↓
Q4: 块代数的分解？
   ↓
Q5: 与常表示的联系？（分解数、上升）

```

## 常表示 vs 模表示

### 基本性质对比

| 性质 | 常表示 (char ∤ |G|) | 模表示 (char | |G|) |
|-----|-------------------|---------------------|
| **Maschke定理** | ✓ 完全可约 | ✗ 不完全可约 |
| **群代数结构** | 半单 | 非半单，有Jacobson根 |
| **不可约分类** | 有限，共轭类数 | 有限，p-正则共轭类数 |
| **不可分解** | 全部不可约 | 大多数非不可约 |
| **投射模** | 全部投射 (= 内射) | 非全部，分类重要 |
| **维数限制** | 整除|G| | 无限制（幂零群可任意大） |

### 群代数结构对比

**常表示**：$\mathbb{C}[G] \cong \prod_{i} M_{n_i}(\mathbb{C})$

**模表示**：$k[G] / J(k[G]) \cong \prod_{i} M_{m_i}(D_i)$（半单商）

其中 $J(k[G])$ 是Jacobson根，包含幂零元。

### 不可约表示数量

**定理 (Brauer)**：设 G 有 r 个共轭类，$r_p$ 个p-正则共轭类（元素阶数不被p整除）。

- 常表示：r 个不可约表示
- 模表示：$r_p$ 个不可约表示

**实例**：$G = S_3$, p = 3
- 共轭类：{1}, {(12),(13),(23)}, {(123),(132)} → r = 3
- 3-正则类：{1}, {(12),(13),(23)} → $r_3$ = 2
- 不可约表示：常情形3个，模情形2个

## Brauer特征标理论

### Brauer特征标的定义

**设置**：设 G 是有限群，p 是素数，K 是特征0的域（如 ℂ），k 是特征p的域。

**提升同态**：设 $\mu_{p'} \subseteq K$ 是单位根（阶数与p互素）。存在典范同构：
$$\mu_{p'}(K) \xrightarrow{\sim} \mu_{p'}(k) = k^\times$$

**Brauer特征标定义**：设 V 是kG-模，g ∈ G 是p-正则元（p ∤ o(g)）。

$$\phi_V(g) = \sum_{\text{本征值 } \lambda_i} [\lambda_i] \in K$$

其中 $[\lambda_i]$ 是 $\lambda_i \in k^\times$ 在K中的唯一提升。

### 特征标表对比

| 特征标 | 定义域 | 取值 | 类函数 | 正交性 |
|-------|-------|-----|-------|-------|
| **普通特征标** | 所有g ∈ G | ℂ | 所有共轭类 | ⟨χ, ψ⟩ = δ_{ij} |
| **Brauer特征标** | p-正则元 | K (特征0) | p-正则类 | ⟨φ, ψ⟩_p' = δ_{ij} |
| **模特征标** | - | k | - | - |

### 正交关系

**Brauer内积**：对 Brauer 特征标 φ, ψ：

$$\langle \phi, \psi \rangle_{p'} = \frac{1}{|G|}\sum_{g \in G_{p'}} \phi(g)\psi(g^{-1})$$

其中 $G_{p'}$ 是p-正则元集合。

**正交性**：不可约 Brauer 特征标关于此内积正交归一。

## 分解矩阵与Cartan矩阵

### 分解矩阵 D

**设置**：设 $\chi_1, ..., \chi_r$ 是普通不可约特征标，$\phi_1, ..., \phi_{r_p}$ 是不可约 Brauer 特征标。

**分解数**：
$$\chi_i|_{G_{p'}} = \sum_{j=1}^{r_p} d_{ij} \phi_j$$

**分解矩阵**：$D = (d_{ij})$，r × $r_p$ 矩阵，$d_{ij} \in \mathbb{Z}_{\geq 0}$

**性质**：
- D 的列线性无关
- 每个 $d_{ij}$ 表示 $L_j$ 在特征0表示 $V_i$ 的p-约化中出现的重数

### Cartan矩阵 C

**定义**：$C = D^T D$，$r_p \times r_p$ 对称矩阵。

**代数意义**：设 $P_i$ 是 $L_i$ 的投射覆盖，则：
$$c_{ij} = \dim_k \text{Hom}_{kG}(P_i, P_j) / \dim_k \text{End}_{kG}(L_i)$$

**性质**：
- C 是对称正定整数矩阵
- 对角元 $c_{ii} \geq 1$（Cartan不变量）
- 行列式是p的幂

### 实例：S₃ 在 p=3

**普通特征标**：χ₁(平凡), χ₂(符号), χ₃(2维)

**Brauer特征标**：φ₁(平凡), φ₂(2维约化)

**分解**：
- χ₁| = φ₁
- χ₂| = φ₁（符号在特征3下平凡化）
- χ₃| = φ₂

**分解矩阵**：$D = \begin{pmatrix} 1 & 0 \\ 1 & 0 \\ 0 & 1 \end{pmatrix}$

**Cartan矩阵**：$C = D^T D = \begin{pmatrix} 2 & 0 \\ 0 & 1 \end{pmatrix}$

## 块理论

### 块分解

**定理**：群代数分解为双向理想的直和：
$$k[G] = B_0 \oplus B_1 \oplus ... \oplus B_n$$

每个 $B_i$ 称为一个**p-块**（或简称为块）。

**块代数的性质**：
- 每个不可约模属于唯一的块
- 每个投射不可分解模属于唯一的块
- 块是连通分支（从Ext图看）

### 主块与缺陷

**主块** $B_0$：包含平凡模的块。

**缺陷群**：块B的**缺陷群**是某个p-子群D ⊆ G，满足：
- |D| = $p^d$，d 称为**缺陷**

- D 是G的Sylow p-子群的子群
- D 在同构意义下唯一

**重要定理**：
- **Brauer对应**：在G和 $N_G(D)$ 之间建立块对应
- **第一主定理**：块的个数 = 具有给定缺陷群的p-正则类数
- **零缺陷**：缺陷0的块对应半单代数（仅一个不可约）

### 块不变量

| 不变量 | 符号 | 含义 |
|-------|-----|-----|
| **缺陷** | d(B) | 缺陷群的阶的对数 |
| **不可约数** | k(B) | 块中不可约模的个数 |
| **投射不可分数** | l(B) | 块中投射不可分解模个数 |
| **Cartan行列式** | det(C) | 非零，p的幂 |

**Brauer k(B)-猜想**：$k(B) \leq p^{d(B)}$（仍开放）

**Brauer l(B)-猜想**：$l(B) \leq p^{d(B)}$（已解决）

## 投射模与内射模

### 投射模的分类

**定理**：在k[G]-模范畴中，投射模 = 内射模（自内射代数）。

**投射覆盖**：每个模M有唯一的投射覆盖 P(M) → M，核是多余的。

**投射不可分解模 (PIM)**：
- 与不可约模一一对应：$P_i \leftrightarrow L_i$
- $P_i / \text{rad}(P_i) \cong L_i$
- $P_i$ 是 $L_i$ 的投射覆盖

### Loewy结构

**Jacobson根列**：
$$P_i \supseteq \text{rad}(P_i) = JP_i \supseteq J^2P_i \supseteq ... \supseteq J^n P_i = 0$$

**Loewy层**：$L^j = J^{j-1}P_i / J^j P_i$（半单商）

**Loewy长度**：最小的n使得 $J^n P_i = 0$。

### Green对应

**设置**：设 P 是p-子群，$H = N_G(P)$。

**Green对应**：在G的P-顶模和H的P-顶模之间建立双射。

**顶 (Vertex)**：模M的顶是最小的p-子群Q，使得M是诱导模 $\text{Ind}_Q^G V$ 的直和项。

**源 (Source)**：相应的Q-模V称为源。

## 总结表：模表示论核心概念

| 概念 | 常表示 | 模表示 | 联系 |
|-----|-------|-------|-----|
| **完全可约** | 是 | 否 | 根的存在 |
| **不可约数** | r | $r_{p'}$ | Brauer |
| **特征标** | 普通 | Brauer | 分解数 |
| **群代数** | 半单 | 非半单 | $k[G]/J$ |
| **投射模** | 全部 | PIM分类 | 覆盖 |
| **分类工具** | 特征标表 | 分解矩阵+D+Cartan | 上升理论 |
