# 几何学概念对比矩阵：曲率类型

## 概述

本文档系统对比微分几何中的各类曲率：高斯曲率(Gaussian Curvature)、平均曲率(Mean Curvature)、里奇曲率(Ricci Curvature)和数量曲率(Scalar Curvature)，阐明其定义、几何意义和在Riemann几何中的作用。

---

## 核心对比矩阵

| 曲率类型 | 记号 | 定义域 | 维度依赖 | 几何意义 |
|:---:|:---:|:---:|:---:|:---|
| **高斯曲率** | K | 曲面（2维） | 内蕴 | 曲面局部形状 |
| **平均曲率** | H | 超曲面 | 外蕴 | 曲面嵌入方式 |
| **截面曲率** | K(Π) | 2维平面 | n维 | 子流形弯曲 |
| **里奇曲率** | Ric | 向量场 | n维 | 体积变化率 |
| **数量曲率** | R | 点函数 | n维 | 总曲率度量 |

## 详细对比分析

### 1. 高斯曲率 (Gaussian Curvature)

**定义**: 对曲面M ⊂ ℝ³，高斯映射 g: M → S² 的Jacobian行列式
```
K(p) = κ₁·κ₂ = det(dN_p) = (LN-M²)/(EG-F²)
```

**Gauss绝妙定理**: K是内蕴量，仅依赖于第一基本形式

| 几何意义 | K > 0 | K = 0 | K < 0 |
|:---:|:---:|:---:|:---:|
| **局部形状** | 椭圆点（球状） | 抛物点（柱/平面） | 双曲点（马鞍） |
| **测地线行为** | 汇聚 | 平行 | 发散 |
| **三角形内角和** | > π | = π | < π |
| **例子** | 球面、椭球 | 柱面、锥面、平面 | 伪球面、双曲平面 |

**高斯-博内定理**:
```
∫_M K dA = 2πχ(M)  （紧致可定向曲面）

χ(M) = 2-2g  （欧拉示性数，g为亏格）
```

### 2. 平均曲率 (Mean Curvature)

**定义**: 两个主曲率的平均
```
H = (κ₁ + κ₂)/2 = (1/2)tr(dN_p)
```

| 特征 | 说明 |
|:---:|:---|
| **外蕴性** | 依赖于嵌入方式，非内蕴 |
| **极小曲面** | H ≡ 0（肥皂膜形状） |
| **常平均曲率** | H ≡ const（肥皂泡形状） |
| **物理意义** | 表面张力平衡条件 |

**极小曲面例子**:
- 平面（平凡）
- 悬链面 (Catenoid)
- 螺旋面 (Helicoid)
- Scherk曲面

### 3. 截面曲率 (Sectional Curvature)

**定义**: 对2维平面 Π ⊂ T_pM
```
K(Π) = ⟨R(X,Y)Y, X⟩ / (|X|²|Y|² - ⟨X,Y⟩²)
```
其中 {X,Y} 张成Π

| 分类 | 条件 | 例子 |
|:---:|:---:|:---|
| **常正曲率** | K(Π) ≡ c > 0 | 球面 Sⁿ(r) |
| **常零曲率** | K(Π) ≡ 0 | 欧氏空间 ℝⁿ |
| **常负曲率** | K(Π) ≡ c < 0 | 双曲空间 Hⁿ |
| **变曲率** | K随方向变化 | 一般流形 |

**空间形式 (Space Forms)**:
```
K ≡ 常数 ⟹ M 局部等距于:
    K > 0: 球面 Sⁿ
    K = 0: 欧氏空间 ℝⁿ  
    K < 0: 双曲空间 Hⁿ
```

### 4. 里奇曲率 (Ricci Curvature)

**定义**: Riemann曲率张量的缩并
```
Ric(X,Y) = tr(Z ↦ R(Z,X)Y) = Σᵢ ⟨R(eᵢ,X)Y, eᵢ⟩
```

**里奇张量**: 对称2阶张量

| 几何意义 | 说明 |
|:---:|:---|
| **体积比较** | Ric > 0 ⟹ 测地球体积小于欧氏 |
| **Bochner公式** | 与Laplace算子联系 |
| **Einstein方程** | Ric - (R/2)g = T（引力场） |
| **Myers定理** | Ric ≥ (n-1)k > 0 ⟹ 紧流形 |

**里奇平坦 (Ricci-flat)**: Ric ≡ 0
- Calabi-Yau流形（弦理论）
- 超Kähler流形

### 5. 数量曲率 (Scalar Curvature)

**定义**: Ricci曲率的迹
```
R = tr(Ric) = Σᵢ Ric(eᵢ, eᵢ)
```

| 性质 | 说明 |
|:---:|:---|
| **最简单曲率** | 标量函数 |
| **平均曲率** | 所有方向曲率的平均 |
| **共形几何** | 共形变换下的变分 |
| **正质量定理** | 物理应用 |

**Yamabe问题**: 给定共形类，找常数量曲率的度量

## 维度特性对比

### 2维曲面

```
曲面 M² 上的曲率:
├── Riemann张量: 1个独立分量
├── 高斯曲率 K: 完全决定曲率
├── Ricci = K·g (与度量成比例)
└── 数量曲率 R = 2K

唯一非平凡曲率: 高斯曲率
```

### 3维流形

```
3维 Riemann流形:
├── Riemann张量: 6个独立分量
├── 里奇张量 Ric: 6个分量（决定曲率）
├── Weyl张量: 恒为零！
└── 曲率完全由Ricci决定

Thurston几何化纲领的核心
```

### n维流形 (n≥4)

```
n维流形:
├── Riemann张量: n²(n²-1)/12 个分量
├── Ricci张量: n(n+1)/2 个分量
├── Weyl张量: 共形曲率
└── 曲率分解: Riemann = Weyl + Ricci部分

Weyl张量捕捉共形不变的曲率信息
```

## 曲率与拓扑的关系

| 曲率条件 | 拓扑结论 | 定理 |
|:---:|:---:|:---|
| K > 0 | 有限基本群 | Synge定理 |
| Ric > 0 | 紧流形、有限π₁ | Myers定理 |
| R > 0 | 拓扑障碍 | Hitchin-Lichnerowicz |
| K ≤ 0 | K(π,1)流形 | Cartan-Hadamard |
| Ric ≥ 0 | 分裂定理 | Cheeger-Gromoll |
| K ≡ 0 | 平坦流形 | Bieberbach |

## 计算公式汇总

### 正交标架下的曲率

| 曲率 | 公式 | 备注 |
|:---:|:---:|:---|
| **截面曲率** | K(eᵢ,eⱼ) = R_{ijji} | 规范正交基 |
| **里奇曲率** | R_{ii} = Σⱼ R_{ijji} | 对角分量 |
| **数量曲率** | R = Σᵢ R_{ii} = Σ_{i,j} R_{ijji} | 总和 |

### 2维曲面的显式公式

```
第一基本形式: I = E du² + 2F dudv + G dv²
第二基本形式: II = L du² + 2M dudv + N dv²

高斯曲率:
K = (LN - M²) / (EG - F²)

平均曲率:
H = (EN - 2FM + GL) / 2(EG - F²)

主曲率: κ₁,₂ = H ± √(H² - K)
```

## 里奇流 (Ricci Flow)

```
∂g/∂t = -2Ric(g)

动机: 像热方程一样"平均化"曲率
应用: Hamilton-Perelman证明Poincaré猜想

2维类比: 高斯曲率流
∂g/∂t = -2K·g
趋向常曲率度量（单值化定理）
```
