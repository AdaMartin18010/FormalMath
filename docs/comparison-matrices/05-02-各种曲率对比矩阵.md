## 曲率张量基础

### Riemann曲率张量

**定义**：设 (M, g) 是 Riemann流形，∇ 是Levi-Civita联络。

$$R(X, Y)Z = \nabla_X \nabla_Y Z - \nabla_Y \nabla_X Z - \nabla_{[X,Y]} Z$$

**分量形式**：
$$R^l_{kij} = \partial_i \Gamma^l_{jk} - \partial_j \Gamma^l_{ik} + \Gamma^l_{im}\Gamma^m_{jk} - \Gamma^l_{jm}\Gamma^m_{ik}$$

**协变形式**：$R_{lkij} = g_{lm}R^m_{kij}$

### 曲率张量的对称性

| 恒等式 | 公式 | 独立分量 |
|-------|-----|---------|
| **反对称** | $R_{lkij} = -R_{klij} = -R_{lkji}$ | - |
| **交换** | $R_{lkij} = R_{ijkl}$ | - |
| **第一Bianchi** | $R_{lkij} + R_{lijk} + R_{ljki} = 0$ | 无挠时 |
| **第二Bianchi** | $\nabla_m R_{lkij} + \nabla_l R_{kmij} + \nabla_k R_{mlij} = 0$ | - |

**独立分量数**：$\frac{n^2(n^2-1)}{12}$

## 各类曲率定义对比

### 截面曲率

**定义**：对2维子空间 σ = span(X, Y) ⊆ T_pM，

$$K(\sigma) = \frac{g(R(X,Y)Y, X)}{g(X,X)g(Y,Y) - g(X,Y)^2}$$

**几何意义**：通过p点沿σ的测地曲面的Gauss曲率。

**特例**：

| 曲面 | 截面曲率 | 几何 |
|-----|---------|-----|
| **球面 Sⁿ(r)** | 1/r² | 正曲率 |
| **双曲空间 Hⁿ** | -1 | 负曲率 |
| **欧氏空间** | 0 | 平坦 |
| **柱面 S¹ × ℝ** | 0 | 平坦 |

### Ricci曲率

**定义**：曲率张量的缩并：
$$Ric(X, Y) = \text{tr}(Z \mapsto R(Z, X)Y)$$

**分量形式**：$R_{ij} = R^k_{ikj}$

**性质**：
- 对称：$R_{ij} = R_{ji}$
- n个独立分量（在n维）

**几何意义**：体积在测地线法向的变化率。

**应用**：
- Einstein方程：$G_{ij} = R_{ij} - \frac{1}{2}Rg_{ij}$
- 比较定理：Ricci曲率下界 → 体积比较

### 标量曲率

**定义**：Ricci的迹：
$$R = g^{ij}R_{ij} = \text{tr}_g(Ric)$$

**几何意义**：点p处小球的体积与欧氏体积的偏差：
$$\text{Vol}(B_r(p)) = \omega_n r^n \left(1 - \frac{R}{6(n+2)}r^2 + O(r^4)\right)$$

**应用**：
- Yamabe问题：共形形变使R常数
- 正质量定理

### Weyl共形曲率张量

**定义**（n ≥ 3）：
$$W = Riem - \frac{1}{n-2}(Ric - \frac{R}{n}g) \circ g - \frac{R}{2n(n-1)}g \circ g$$

**性质**：
- 无迹：$g^{ij}W_{ikjl} = 0$
- 共形不变：若 $\tilde{g} = e^{2f}g$，则 $\tilde{W} = e^{2f}W$
- n = 3 时，W = 0（由Bianchi恒等式）

**几何意义**：描述偏离共形平坦的程度。

### 曲率张量对比表

| 曲率 | 张量阶 | 维数 | 约束 | 主要几何信息 |
|-----|-------|-----|-----|------------|
| **Riemann** | (1,3)或(0,4) | $\frac{n^2(n^2-1)}{12}$ | Bianchi恒等式 | 全部曲率信息 |
| **截面** | 函数 | 无穷 | - | 2维子空间曲率 |
| **Ricci** | (0,2) | $\frac{n(n+1)}{2}$ | 对称 | 体积变化 |
| **标量** | 函数 | 1 | - | 总曲率 |
| **Weyl** | (0,4) | $\frac{(n+2)(n+1)n(n-3)}{12}$ | 无迹 | 共形不变部分 |

### 曲率分解（n ≥ 4）

$$Riem = Weyl + \frac{1}{n-2}Ric_0 \circ g + \frac{R}{2n(n-1)}g \circ g$$

其中：
- $Ric_0 = Ric - \frac{R}{n}g$（无迹Ricci）
- $g \circ g$ 是Kulkarni-Nomizu积

**分量数检查**：
$$\frac{n^2(n^2-1)}{12} = \frac{(n+2)(n+1)n(n-3)}{12} + \frac{(n+2)(n-1)}{2} + 1$$

## 常曲率空间

### 空间形式分类

**定义**：截面曲率为常数的空间称为**空间形式**。

| 曲率 K | 单连通完备空间 | 通用覆盖 |
|-------|---------------|---------|
| **K > 0** | 球面 Sⁿ(1/√K) | Sⁿ |
| **K = 0** | 欧氏空间 ℝⁿ | ℝⁿ |
| **K < 0** | 双曲空间 Hⁿ(1/√|K|) | Hⁿ |

### 曲率张量形式

**常曲率 K**：
$$R_{ijkl} = K(g_{ik}g_{jl} - g_{il}g_{jk})$$

**等价条件**：
- Riemann = $\frac{R}{n(n-1)} g \circ g$
- Weyl = 0, Ric₀ = 0

### 曲率齐性空间

| 类型 | 定义 | 例子 |
|-----|-----|-----|
| **曲率齐性** | 曲率张量在等距轨道上是常数 | 齐性空间 |
| **曲率齐性阶k** | 直到k阶导数常数 | - |
| **局部齐性** | 局部与齐性空间等距 | - |

## 曲率与拓扑

### Gauss-Bonnet定理

**2维**：M是紧无边Riemann曲面，
$$\int_M K \, dA = 2\pi \chi(M) = 2\pi(2-2g)$$

**高维推广**（Chern-Gauss-Bonnet）：
$$\int_M Pf(\Omega) = (2\pi)^n \chi(M)$$

其中 Pf 是Pfaffian，Ω 是曲率形式。

### 曲率与基本群

| 定理 | 曲率条件 | 拓扑结论 |
|-----|---------|---------|
| **Bonnet-Myers** | Ric ≥ (n-1)k > 0 | 紧，π₁有限 |
| **Synge** | 偶维，K > 0 | π₁ = 0 或 ℤ₂ |
| **Preissman** | K < 0 | 可解子群循环 |
| **Gromoll-Meyer** | Ric > 0 | π₂有限 |

### 球定理

**1/4-球定理** (Berger-Klingenberg)：
若 M 是紧单连通，且 1/4 < K ≤ 1，则 M 同胚于球面。

**1/4-微分球定理** (Brendle-Schoen)：
在上述条件下，M 微分同胚于标准球面。

## 特殊曲率条件

### Einstein流形

**定义**：$Ric = \lambda g$（对某常数 λ）

**等价条件**：Ricci张量与度量成比例。

**分类**（低维）：

| 维数 | Einstein流形 |
|-----|-------------|
| **n = 2** | 常曲率（Gauss-Bonnet） |
| **n = 3** | 常曲率 |
| **n = 4** | 丰富（Kähler、Toric等） |
| **n ≥ 5** | 各种构造 |

**例子**：
- 球面 Sⁿ：λ = n-1
- 双曲空间 Hⁿ：λ = -(n-1)
- 环面 Tⁿ：λ = 0
- Calabi-Yau：Ric = 0，Kähler

### 常标量曲率

**Yamabe问题**：给定(M,g)，找共形度量 $\tilde{g} = u^{\frac{4}{n-2}}g$ 使R常数。

**Yamabe不变量**：
$$Y(M, [g]) = \inf_{\tilde{g} \in [g]} \frac{\int R_{\tilde{g}} \, dvol_{\tilde{g}}}{(\int dvol_{\tilde{g}})^{\frac{n-2}{n}}}$$

** Aubin-Schoen定理**：Yamabe问题总有解。

### 其他特殊曲率条件

| 条件 | 定义 | 几何意义 |
|-----|-----|---------|
| **共形平坦** | Weyl = 0 | 局部共形于欧氏 |
| **Ricci平坦** | Ric = 0 | Kähler: Calabi-Yau |
| **标量平坦** | R = 0 | 体积二阶不变 |
| **非负曲率** | sec ≥ 0 或 Ric ≥ 0 | 比较几何基础 |
| **非正曲率** | sec ≤ 0 | 拓扑刚性 |

## 应用与比较几何

### 比较几何

**Rauch比较定理**：截面曲率控制Jacobi场增长。

**体积比较**（Bishop-Gromov）：
若 Ric ≥ (n-1)k，则：
$$\frac{\text{Vol}(B_r(p))}{\text{Vol}_k(B_r)} \text{ 关于 } r \text{ 非增}$$

其中 Vol_k 是空间形式中的体积。

**应用**：
- 直径估计
- 特征值估计
- 热核估计

### 广义相对论中的曲率

| 曲率 | 物理意义 | 约束条件 |
|-----|---------|---------|
| **Riemann** | 潮汐力 | - |
| **Ricci** | 物质分布 | Einstein方程 |
| **Weyl** | 引力辐射 | 真空：Weyl描述全部曲率 |
| **标量** | 物质密度 | - |

**Einstein场方程**：$G_{\mu\nu} = R_{\mu\nu} - \frac{1}{2}Rg_{\mu\nu} = 8\pi T_{\mu\nu}$

**真空Einstein**：$Ric = 0$（Ricci平坦）

### 几何流

| 流 | 方程 | 曲率作用 |
|---|-----|---------|
| **Ricci流** | $\partial_t g = -2Ric$ | 使曲率均匀化 |
| **平均曲率流** | $\partial_t X = H$ | 体积减小 |
| **Yamabe流** | $\partial_t g = -(R-r)g$ | 标量曲率常数化 |

**Poincaré猜想**（Perelman）：Ricci流 + 手术技术证明了3维Poincaré猜想。
