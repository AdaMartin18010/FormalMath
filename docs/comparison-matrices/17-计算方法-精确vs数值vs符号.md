### 核心特征矩阵

|:---:|:---:|:---:|:---:|
| **输出类型** | 精确有理数/代数数 | 浮点近似值 | 符号表达式 |

| **中间结果** | 精确有理数 | 浮点数 | 符号表达式 |

## 技术特征深度分析

### 精确计算技术细节

**有理数运算：**
```
表示：p/q，其中 p,q ∈ ℤ，q ≠ 0，gcd(p,q) = 1

运算规则：
- 加法：a/b + c/d = (ad + bc) / bd
- 乘法：(a/b) × (c/d) = (ac) / (bd)
- 约分：每次运算后约简

复杂度：
- 加法：O(n) 位运算（n为位数）
- 乘法：O(n log n)（Karatsuba，FFT）
- 除法：O(n log n)

问题：中间结果位数可能指数增长
例：计算 1/2 + 1/3 + ... + 1/n，分母为 lcm(2,3,...,n) ≈ e^n
```

**代数数运算：**
```
表示：代数数的极小多项式 + 隔离区间

例：√2 表示为 (x² - 2 = 0, [1,2])

运算复杂度：
- 加法/乘法：多项式结式计算
- 复杂度：O(d³)（d为次数）

应用：几何定理证明、代数数论
```

### 数值计算技术细节

**浮点数表示（IEEE 754）：**
```
单精度（32位）：1位符号 + 8位指数 + 23位尾数
双精度（64位）：1位符号 + 11位指数 + 52位尾数

双精度特性：
- 约16位十进制精度
- 范围：≈ ±10^±308
- 机器精度 ε ≈ 2.22 × 10^-16

特殊值：
- Inf（无穷）
- NaN（非数）
- 正零/负零
```

**误差来源：**

| 误差类型 | 描述 | 示例 | 控制方法 |
|:---:|:---:|:---:|:---|
| **截断误差** | 无穷过程的有限截断 | Taylor级数截断 | 自适应步长、高阶方法 |
| **舍入误差** | 浮点表示的有限精度 | 0.1 + 0.2 ≠ 0.3 | 高精度算术、Kahan求和 |
| **传播误差** | 误差在计算中累积 | 递推关系不稳定 | 数值稳定性分析、重正交化 |
| **病态问题** | 输入小扰动导致输出大变化 | 矩阵求逆 | 条件数分析、预处理 |

### 符号计算技术细节

**表达式表示：**
```
树形结构：
    +
   / \
  ×   3
 / \
x   x

表示：x² + 3

化简规则：
- 代数化简：x + x = 2x，x × x = x²
- 三角化简：sin²x + cos²x = 1
- 指数化简：e^x × e^y = e^{x+y}
- 对数化简：log a + log b = log(ab)
```

**核心算法：**

| 问题 | 算法 | 复杂度 | 代表系统 |
|:---:|:---:|:---:|:---:|
| **多项式GCD** | 欧几里得算法、模GCD | O(n²) - O(n log²n) | 所有CAS |
| **因式分解** | Berlekamp-Zassenhaus | 指数级（最坏） | Mathematica, Maple |
| **符号积分** | Risch算法 | 指数级 | Mathematica, Maple |
| **Gröbner基** | Buchberger算法 | 双指数级（最坏） | Singular, Macaulay2 |
| **微分方程** | 李群方法、Kovacic算法 | 依问题而定 | Maple, SymPy |

## 精度与误差分析

### 精度层次对比

```
精度层次（从高到低）：

无限精度 ────────────────────────────────► 有限精度

精确计算        符号计算         区间算术         数值计算
(有理/代数)     (形式精确)       (范围保证)       (浮点近似)
    │              │                │               │
    ▼              ▼                ▼               ▼
1/3 = 1/3      1/3 符号         [0.333,0.334]    0.333333...
√2 = 代数数    √2 符号          [1.414,1.415]    1.41421356...
π = 超越数     π 符号           [3.141,3.142]    3.14159265...
```

### 误差分析工具

| 方法 | 描述 | 应用场景 |
|:---:|:---:|:---|
| **前向误差分析** | 分析输入误差如何传播到输出 | 算法设计 |
| **后向误差分析** | 求等价扰动输入 | Wilkinson风格 |
| **区间算术** | 计算结果的范围而非点值 | 可靠性验证 |
| **任意精度** | 动态调整精度直到收敛 | 高精度需求 |

## 计算复杂度对比

### 基本操作复杂度

| 操作 | 精确计算 | 数值计算 | 符号计算 |
|:---:|:---:|:---:|:---:|
| **大整数加法** | O(n) | O(1) | O(n) |
| **大整数乘法** | O(n log n)（FFT） | O(1) | O(n log n)（FFT） |
| **多项式乘法** | O(n²)（经典） | - | O(n log n)（FFT） |
| **矩阵乘法(n×n)** | O(n³)（位运算大） | O(n³) | O(n³)（形式系数） |
| **矩阵求逆** | 可能指数位增长 | O(n³) | O(n³)（形式） |
| **整数分解** | 指数级 | - | 指数级 |
| **多项式GCD** | O(n²) | - | O(n²) |
| **符号积分** | - | - | 指数级（一般） |

### 位复杂度 vs 算术复杂度

```
数值计算：主要关注算术复杂度（浮点操作数）
精确/符号计算：需考虑位复杂度（位数增长）

例：计算 Fibonacci(n)
- 数值：O(n) 时间（但精度损失）
- 精确：O(n²)（位数与n成正比）
- 符号：O(n)（保持为表达式）
```

## 混合计算策略

### 经典混合策略

| 策略 | 描述 | 示例 | 优势 |
|:---:|:---:|:---:|:---|
| **符号-数值混合** | 先用符号简化，再数值求值 | 复杂公式化简后求值 | 减少数值误差 |
| **精确-数值验证** | 数值结果用精确计算验证 | 猜想检验 | 保证正确性 |
| **自适应精度** | 从低精度开始，逐步提高 | 数值积分 | 效率与精度平衡 |
| **区间算术** | 计算结果的范围而非点值 | 可靠性验证 | 严格误差界 |
| **混合线性代数** | 有理数 + 浮点数混合 | 稀疏矩阵 | 精确+高效 |

### 精确-数值混合示例

```python
# 例：计算多项式根

# 纯符号方法（可能太慢或不可行）
solve(x^5 - x + 1 = 0)  # 一般五次方程无根式解

# 混合策略：
# 1. 符号分析：确定实根个数（Sturm序列）
# 2. 数值隔离：用牛顿法近似根
# 3. 精确验证：验证近似根（精确算术）

结果：
- 1个实根（数值近似）
- 4个复根（共轭对）
- 代数度数：5
```

### 具体问题策略

| 问题类型 | 推荐方法 | 理由 |
|:---:|:---:|:---|
| **整数规划** | 精确（分支定界） | 需要最优性保证 |
| **微分方程数值解** | 数值（RK方法） | 解析解通常不存在 |
| **符号微分** | 符号 | 精确保持结构 |
| **大规模矩阵运算** | 数值 | 计算效率 |
| **代数几何计算** | 精确+符号 | Gröbner基需要精确 |
| **蒙特卡洛模拟** | 数值 | 统计采样 |
| **编译器优化** | 精确（整数） | 程序正确性 |
| **金融建模** | 数值（有时精确） | 精度vs速度权衡 |

### 最佳实践清单

**精确计算：**
- [ ] 评估位数增长风险
- [ ] 使用高效算法（Karatsuba, FFT）
- [ ] 考虑模运算替代（如可能）

**数值计算：**
- [ ] 进行误差分析
- [ ] 检查条件数
- [ ] 使用稳定算法
- [ ] 验证结果合理性

**符号计算：**
- [ ] 及时化简表达式
- [ ] 避免表达式膨胀
- [ ] 设置计算限制

## 总结与展望

### 核心对比总结

| 维度 | 精确计算 | 数值计算 | 符号计算 |
|:---:|:---:|:---:|:---:|
| **核心优势** | 无误差的数学保证 | 计算效率 | 结构保持 |
| **主要局限** | 计算成本高 | 精度有限 | 复杂度高 |
| **不可替代** | 数论、密码学 | 大规模仿真 | 公式推导 |
| **现代趋势** | 任意精度、可验证 | 混合精度、自动微分 | AI辅助、神经符号 |
