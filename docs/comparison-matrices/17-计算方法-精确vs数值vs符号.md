# 计算方法对比：精确计算 vs 数值计算 vs 符号计算

## 基本信息

| 维度 | 内容 |
|------|------|
| **对比主题** | 三种基本计算范式 |
| **核心问题** | 精度、效率、通用性的权衡 |
| **代表系统** | Mathematica, MATLAB, Python科学计算 |

---

## 核心特征矩阵

| 特征维度 | 精确计算 (Exact) | 数值计算 (Numerical) | 符号计算 (Symbolic) |
|----------|------------------|----------------------|---------------------|
| **输出类型** | 精确有理数/代数数 | 浮点近似值 | 符号表达式 |
| **精度特性** | 无限精度 | 有限精度（可配置） | 形式精确 |
| **计算速度** | 慢（数增长快） | 快（硬件优化） | 中等（表达式膨胀） |
| **内存需求** | 可能指数增长 | 固定 | 可能多项式膨胀 |
| **舍入误差** | 无 | 有（需误差分析） | 无（形式处理） |
| **可解释性** | 高（精确答案） | 低（需解释） | 高（保持结构） |

## 技术特征详细对比

| 技术指标 | 精确计算 | 数值计算 | 符号计算 |
|----------|----------|----------|----------|
| **1/3 + 1/3** | $\frac{2}{3}$ | $0.666666...$（截断） | $\frac{1}{3} + \frac{1}{3}$ |
| **√2 × √2** | $2$ | $1.999999...$ | $2$（化简后） |
| **sin(π/3)** | $\frac{\sqrt{3}}{2}$ | $0.866025...$ | $\frac{\sqrt{3}}{2}$ |
| **解方程** | 精确根式 | 近似根 | 符号公式 |
| **极限计算** | 可能不可判定 | 数值逼近 | 符号极限 |

## 误差与可靠性

### 数值计算的误差来源

| 误差类型 | 描述 | 控制方法 |
|----------|------|----------|
| **截断误差** | 无穷过程的有限截断 | 自适应步长、高阶方法 |
| **舍入误差** | 浮点表示的有限精度 | 高精度算术、区间算术 |
| **传播误差** | 误差在计算中累积 | 数值稳定性分析 |

### 精度比较

```
精确计算：  1/3 = 0.333333...（无限精度）
数值计算：  1/3 ≈ 0.333333（双精度约16位）
符号计算：  1/3（保持为分数）
```

## 混合计算策略

| 策略 | 描述 | 示例 |
|------|------|------|
| **符号-数值混合** | 先用符号简化，再数值求值 | 复杂公式求值 |
| **精确-数值验证** | 数值结果用精确计算验证 | 猜想检验 |
| **自适应精度** | 从低精度开始，逐步提高 | 数值积分 |
| **区间算术** | 计算结果的范围而非点值 | 可靠性验证 |

## 计算复杂度对比

| 操作 | 精确计算 | 数值计算 | 符号计算 |
|------|----------|----------|----------|
| **大整数加法** | O(n) | O(1) | O(n) |
| **大整数乘法** | O(n log n) | O(1) | O(n log n) |
| **多项式乘法** | O(n²) | - | O(n log n)（FFT）|
| **矩阵求逆** | 可能指数位增长 | O(n³) | O(n³)（形式）|
| **积分** | 可能不可计算 | 数值近似 | 启发式算法 |
