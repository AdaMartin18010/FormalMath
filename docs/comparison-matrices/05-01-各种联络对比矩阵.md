## 联络基础概念

### 什么是联络

**直观理解**：联络是"平行移动"的无穷小版本，提供了向量沿曲线"不变"比较的方式。

**数学定义**：在向量丛 E → M 上，**联络**是一个映射：
$$\nabla: \Gamma(E) \to \Gamma(T^*M \otimes E)$$

满足：
1. $\nabla(fs) = df \otimes s + f \nabla s$（Leibniz法则）
2. $\nabla(s_1 + s_2) = \nabla s_1 + \nabla s_2$（线性）

### 联络分类总览

| 联络类型 | 定义域 | 结构群 | 主要约束 | 典型例子 |
|---------|-------|-------|---------|---------|
| **仿射联络** | 切丛 TM | GL(n,ℝ) | 无 | 一般联络 |
| **Levi-Civita** | TM | O(n) | 无挠、度量相容 | Riemann几何 |
| **主丛联络** | 主丛 P → M | G | G-等变 | Yang-Mills |
| **向量丛联络** | 一般向量丛 | GL(r) | - | Chern-Weil理论 |
| **射影联络** | TM | PGL(n) | - | 射影几何 |
| **共形联络** | TM | CO(n) | 共形结构 | 共形几何 |
| **辛联络** | TM | Sp(2n) | 辛形式平行 | 辛几何 |

## 仿射联络对比

### 仿射联络的一般形式

**Christoffel符号**：在坐标 {xⁱ} 下，
$$\nabla_{\partial_i} \partial_j = \sum_k \Gamma^k_{ij} \partial_k$$

**协变导数**：对向量场 $X = X^i \partial_i$，
$$\nabla_X Y = X(Y^k) \partial_k + X^i Y^j \Gamma^k_{ij} \partial_k$$

### Levi-Civita联络

**定理 (Levi-Civita)**：给定 Riemann度量 g，存在唯一的无挠、度量相容联络。

**度量相容**：$\nabla g = 0$，即 $X(g(Y,Z)) = g(\nabla_X Y, Z) + g(Y, \nabla_X Z)$

**无挠**：$\nabla_X Y - \nabla_Y X = [X, Y]$

**Christoffel公式**：
$$\Gamma^k_{ij} = \frac{1}{2}g^{kl}(\partial_i g_{jl} + \partial_j g_{il} - \partial_l g_{ij})$$

### 仿射联络性质对比

| 性质 | 一般仿射联络 | Levi-Civita联络 |
|-----|-------------|----------------|
| **存在性** | 多（仿射空间） | 唯一（给定g） |
| **挠率** | 可能有 | 无 |
| **度量相容** | 不一定 | 是 |
| **Christoffel对称** | $\Gamma^k_{ij} \neq \Gamma^k_{ji}$ | $\Gamma^k_{ij} = \Gamma^k_{ji}$ |
| **测地线方程** | 存在 | 能量极值曲线 |

## 主丛联络与向量丛联络

### 主丛联络

**设置**：P → M 是主 G-丛。

**定义**：主丛联络是 G-等变分解：
$$TP = HP \oplus VP$$

其中 VP = ker(dπ) 是垂直子丛。

**联络形式**：ω ∈ Ω¹(P, 𝔤)，满足：
1. ω(A*) = A（基本向量场）
2. R_g^* ω = Ad_{g^{-1}} ω（等变性）

### 向量丛联络

**设置**：E → M 是秩 r 向量丛。

**定义**：联络是映射：
$$\nabla: \Gamma(E) \to \Gamma(T^*M \otimes E)$$

**局部形式**：在标架 {e_α} 下，
$$\nabla e_\alpha = \sum_\beta \omega^\beta_\alpha \otimes e_\beta$$

其中 ω = (ω^β_α) 是**联络矩阵**（值在 𝔤𝔩(r)）。

### 两种联络的对应

| 方面 | 主丛联络 | 向量丛联络 |
|-----|---------|-----------|
| **定义对象** | 主丛 P | 向量丛 E |
| **联络形式** | ω ∈ Ω¹(P, 𝔤) | ∇: Γ(E) → Ω¹(M, E) |
| **水平提升** | 曲线提升 | 平行截面 |
| **曲率** | Ω = dω + ½[ω,ω] | R(X,Y)s = [∇_X, ∇_Y]s - ∇_[X,Y]s |
| **结构群** | G | GL(r)（或子群） |

**对应关系**：给定主丛联络和表示 ρ: G → GL(V)，诱导相伴向量丛的联络。

## 特殊联络类型

### 辛联络

**设置**：(M, ω) 是辛流形，ω 是非退化闭2-形式。

**定义**：**辛联络**是满足 $\nabla \omega = 0$ 的联络。

**性质**：
- 总是存在（但不是唯一的）
- 无挠辛联络存在当且仅当某种阻碍消失

**与Riemann联络对比**：
| 结构 | 对称双线性型 g | 反对称双线性型 ω |
|-----|----------------|-----------------|
| **结构群** | O(n) | Sp(2n) |
| **联络约束** | 度量相容 | 辛形式平行 |
| **典范联络** | Levi-Civita | 不唯一 |

### 共形联络

**设置**：[g] 是共形等价类度量。

**定义**：**共形联络**是保持共形结构的联络。

**Weyl联络**：无挠共形联络，满足：
$$\nabla g = \theta \otimes g$$

对某1-形式 θ。

**性质**：
- 典范选择：Weyl结构
- 共形平坦：Weyl张量 = 0

### Kähler联络

**设置**：(M, g, J, ω) 是Kähler流形。

**定理**：Levi-Civita联络满足：
1. $\nabla J = 0$（复结构平行）
2. $\nabla \omega = 0$（辛形式平行）

**等价条件**：Kähler ⟺ Levi-Civita联络与复结构相容

### 特殊联络对比表

| 联络 | 结构 | 约束 | 存在性 | 唯一性 |
|-----|-----|-----|-------|-------|
| **Levi-Civita** | Riemann | 无挠+度量相容 | 是 | 是 |
| **辛** | 辛 | ∇ω = 0 | 是 | 否 |
| **Weyl** | 共形 | 无挠+共形 | 是 | 否 |
| **Hermite** | 复Hermite | ∇g = ∇J = 0 | 否 | - |
| **Kähler** | Kähler | Levi-Civita+∇J=0 | 是 | 是 |

## 联络的曲率与挠率

### 挠率张量

**定义**：仿射联络的**挠率**：
$$T(X, Y) = \nabla_X Y - \nabla_Y X - [X, Y]$$

**坐标形式**：$T^k_{ij} = \Gamma^k_{ij} - \Gamma^k_{ji}$

**性质**：
- T = 0 ⟺ Christoffel对称
- T 是张量（与坐标无关）

### 曲率张量

**定义**：**Riemann曲率**：
$$R(X, Y)Z = \nabla_X \nabla_Y Z - \nabla_Y \nabla_X Z - \nabla_{[X,Y]} Z$$

**坐标形式**：
$$R^l_{kij} = \partial_i \Gamma^l_{jk} - \partial_j \Gamma^l_{ik} + \Gamma^l_{im}\Gamma^m_{jk} - \Gamma^l_{jm}\Gamma^m_{ik}$$

**对称性**：
1. $R_{klij} = -R_{lkij} = -R_{klji}$
2. 第一Bianchi恒等式：$R_{klij} + R_{kilj} + R_{ijkl} = 0$（无挠时）
3. 第二Bianchi恒等式：$\nabla_m R_{klij} + \nabla_l R_{kmij} + \nabla_k R_{mlij} = 0$

### 曲率分解

| 张量 | 定义 | 维数 | 几何意义 |
|-----|-----|-----|---------|
| **Ricci曲率** | $R_{ij} = R^k_{ikj}$ | n(n+1)/2 | 体积变化 |
| **标量曲率** | $R = g^{ij}R_{ij}$ | 1 | 总曲率 |
| **Weyl张量** | 无迹部分 | n(n+1)(n+2)(n-3)/12 | 共形不变 |
| **Schouten张量** | $P_{ij} = \frac{1}{n-2}(R_{ij} - \frac{R}{2(n-1)}g_{ij})$ | n(n+1)/2 | 共形几何 |

**n ≥ 4 分解**：
$$Riem = Weyl + \frac{1}{n-2}Ric \circ g + \frac{R}{2n(n-1)}g \circ g$$

### 结构方程

**Cartan结构方程**：

设 {θⁱ} 是余标架，{ωⁱⱼ} 是联络形式。

1. **第一结构方程**（挠率）：
$$d\theta^i + \sum_j \omega^i_j \wedge \theta^j = \Theta^i = \frac{1}{2}T^i_{jk} \theta^j \wedge \theta^k$$

2. **第二结构方程**（曲率）：
$$d\omega^i_j + \sum_k \omega^i_k \wedge \omega^k_j = \Omega^i_j = \frac{1}{2}R^i_{jkl} \theta^k \wedge \theta^l$$

**Bianchi恒等式**：
- $d\Theta + \omega \wedge \Theta = \Omega \wedge \theta$
- $d\Omega + \omega \wedge \Omega = \Omega \wedge \omega$

## 应用领域对比

### 广义相对论

| 概念 | 物理意义 | 数学对应 |
|-----|---------|---------|
| **时空流形** | 宇宙的几何 | 4维Lorentz流形 |
| **联络** | 引力场 | Levi-Civita |
| **曲率** | 引力 | Einstein张量 |
| **测地线** | 自由落体 | 极值曲线 |

**Einstein方程**：$G_{\mu\nu} + \Lambda g_{\mu\nu} = \frac{8\pi G}{c^4} T_{\mu\nu}$

### 规范理论

| 物理概念 | 数学对应 | 说明 |
|---------|---------|-----|
| **规范场** | 主丛联络 | Yang-Mills场 |
| **规范群** | 结构群G | SU(3)×SU(2)×U(1) |
| **场强** | 曲率Ω | F_{μν} |
| **Bianchi恒等式** | d_∇Ω = 0 | 运动方程 |

| 概念 | 联络 | 曲率条件 |
|-----|-----|---------|
| **Hermite-Einstein** | 向量丛上的特殊联络 | ΛF = c·Id |
