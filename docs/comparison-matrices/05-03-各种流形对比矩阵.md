## 流形分类概览

### 几何结构层次

```

拓扑流形
    ↓ (光滑结构)
光滑流形
    ├──→ Riemann流形 (+度量)
    │       ├──→ Kähler流形 (+复+辛)
    │       ├──→ Einstein流形 (Ric=λg)
    │       └──→ 常曲率空间
    │
    ├──→ 辛流形 (+辛形式)
    │       └──→ Kähler流形
    │
    ├──→ 复流形 (+复结构)
    │       ├──→ Kähler流形
    │       └──→ 代数簇 (射影)
    │
    └──→ 伪Riemann流形 (+不定度量)
            └──→ Lorentz流形 (广义相对论)

```

### 主要流形类型总览

| 类型 | 结构 | 结构群 | 维度 | 典型不变量 |
|-----|-----|-------|-----|----------|
| **光滑流形** | 光滑图册 | GL(n,ℝ) | n | 同伦型、示性类 |
| **复流形** | 全纯图册 | GL(n,ℂ) | 2n (实) | Hodge数 |
| **Riemann流形** | 度量 g > 0 | O(n) | n | 曲率、特征值 |
| **辛流形** | 辛形式 ω | Sp(2n) | 2n | 上同调类、Gromov-Witten |
| **Kähler流形** | g + J + ω | U(n) | 2n | Hodge结构、Chern类 |
| **Lorentz流形** | 度量 (n-1,1) | O(n-1,1) | n | 测地线、奇点 |
| **旋量流形** | 旋结构 | Spin(n) | n | 指标、η-不变量 |

## 光滑流形 vs 复流形

### 基本定义对比

| 对比维度 | 光滑流形 | 复流形 |
|---------|---------|-------|
| **图册** | {(U_α, φ_α: U_α → ℝⁿ)} | {(U_α, φ_α: U_α → ℂⁿ)} |
| **转移函数** | 光滑 (C^∞) | 全纯 |
| **结构层** | C^∞函数层 | 全纯函数层 𝒪_X |
| **切空间** | 实向量空间 T_xM ≅ ℝⁿ | 复向量空间 T_xM ≅ ℂⁿ |
| **维数** | n（实） | n（复）= 2n（实） |

### 光滑流形结构

**定义**：拓扑流形配备**光滑图册**，即转移函数无限可微。

**关键问题**：
- **光滑化问题**：哪些拓扑流形有光滑结构？
- **唯一性**：光滑结构是否唯一？（高维不唯一）

**Donaldson理论**：4维流形有微妙的光滑结构现象。

**例子**：
- ℝⁿ：标准光滑结构
- Sⁿ：n ≤ 3 唯一；n ≥ 5 唯一；n = 4 未知（有怪球）
- 复射影空间 ℂPⁿ：标准光滑结构

### 复流形结构

**定义**：光滑流形配备**可积近复结构**J，即 $J^2 = -I$，且Nijenhuis张量 N_J = 0。

**近复 vs 复**：
- **近复**：J 存在（可定义全纯切空间）
- **复**：J 可积（有全纯坐标）

**Newlander-Nirenberg定理**：J 可积 ⟺ N_J = 0。

**例子**：

| 复流形 | 维数 | 性质 |
|-------|-----|-----|
| **ℂⁿ** | n | Stein |
| **ℂPⁿ** | n | 紧、Fano |
| **复环面** | n | 平行化、Kähler |
| **代数簇** | n | 射影/拟射影 |

### 实 vs 复的几何对比

| 几何 | 光滑流形 | 复流形 |
|-----|---------|-------|
| **函数** | 多（光滑函数） | 少（全纯函数紧流形上常数） |
| **结构刚性** | 低 | 高（复结构约束强） |
| **存在性** | 大部分拓扑流形可光滑化 | 很多光滑流形不可复化 |
| **分类** | 难（高维） | 难但有工具（Kodaira维数） |

## Riemann流形 vs 辛流形

### Riemann几何

**定义**：光滑流形配备**Riemann度量** g，即：
- 对称：g(X,Y) = g(Y,X)
- 正定：g(X,X) > 0（X ≠ 0）

**结构群**：O(n)（保持内积）

**典例联络**：Levi-Civita联络（唯一无挠度量联络）

**曲率**：Riemann → Ricci → 标量

### 辛几何

**定义**：光滑流形配备**辛形式** ω，即：
- 反对称：ω(X,Y) = -ω(Y,X)
- 非退化：ωⁿ ≠ 0
- 闭：dω = 0

**结构群**：Sp(2n)（保持辛形式）

**维数**：必是偶数 2n

**典例定理**：
- **Darboux定理**：辛流形局部同构于 (ℝ²ⁿ, ω₀ = Σ dpᵢ ∧ dqᵢ)
- **Moser定理**：辛结构的同伦类决定辛结构

### Riemann vs 辛对比

| 特征 | Riemann流形 | 辛流形 |
|-----|------------|-------|
| **结构** | 对称双线性型 g | 反对称双线性型 ω |
| **非退化** | 正定（强） | 非退化（弱） |
| **局部形式** | g_{ij}(x) 一般形式 | Darboux：标准形式 |
| **局部刚性** | 高（曲率区分） | 低（Darboux） |
| **整体刚性** | 有（比较几何） | 高（Gromov） |
| **典范联络** | Levi-Civita（唯一） | 辛联络（不唯一） |
| **测地线** | 能量极值 | 作用量极值（Hamilton） |

### 兼容结构：Kähler流形

**定义**：配备 (g, J, ω) 满足：
- g：Riemann度量
- J：近复结构
- ω(X,Y) = g(JX, Y)（辛形式）
- J 可积（复流形）
- dω = 0（辛条件）

**等价条件**：
- Kähler ⟺ g Hermite + ∇J = 0
- Kähler ⟺ 复流形上有闭的、正定的 (1,1)-形式

**Hodge分解**：紧Kähler流形上，
$$H^k(X, ℂ) = \bigoplus_{p+q=k} H^{p,q}(X)$$

## 特殊几何结构

### Lorentz几何（广义相对论）

**定义**：配备**Lorentz度量** g，符号 (n-1, 1)。

**结构群**：O(n-1, 1)

**分类向量**：
- **类时**：g(X,X) < 0（物理可观测）
- **类空**：g(X,X) > 0
- **类光/零**：g(X,X) = 0

**Einstein方程**：$G_{\mu\nu} = 8\pi T_{\mu\nu}$

### 旋量几何

**旋结构**：二重覆盖 Spin(n) → SO(n) 提升到标架丛。

**障碍**：第二Stiefel-Whitney类 $w_2(M) = 0$。

**旋量丛**：S → M，截面是旋量场。

**Dirac算子**：$D: \Gamma(S) \to \Gamma(S)$

**应用**：指标定理、正质量定理、Seiberg-Witten理论。

### Calabi-Yau流形

**定义**：紧Kähler流形满足：
- Ric = 0（Ricci平坦）
- c₁(X) = 0（第一陈类为零）
- 典范丛 K_X = ΛⁿT*X 平凡

**维度**：

| 维数 | 例子 | 物理意义 |
|-----|-----|---------|
| **1** | 椭圆曲线 | 弦紧致化 |
| **2** | K3曲面 | 超对称 |
| **3** | Calabi-Yau 3- folds | 弦理论紧化 |

**Yau定理**：紧Kähler流形，c₁(X) = 0 ⟹ 存在唯一Ricci平坦Kähler度量（每Kähler类中）。

## 流形上的结构兼容性

### 结构兼容性表

| 结构组合 | 兼容性条件 | 结果结构 | 例子 |
|---------|----------|---------|-----|
| **复 + Riemann** | g(JX, JY) = g(X,Y) | Hermite流形 | (ℂⁿ, 欧氏) |
| **复 + 辛** | ω(JX, JY) = ω(X,Y) | 近Kähler | - |
| **Riemann + 辛** | g(X,Y) = ω(X, JY) | 近Kähler | - |
| **复 + Riemann + 辛 + dω=0** | 全部相容 | **Kähler** | ℂPⁿ, 环面 |

### 不可兼容性

| 结构1 | 结构2 | 障碍 | 备注 |
|-------|-------|-----|-----|
| **辛** | **紧 + 边界** | Stokes定理 | 辛流形无边或体积无穷 |
| **复** | **S⁴** | 未知（开放） | 4维球无复结构（猜想） |
| **Kähler** | **Hopf曲面** | b₁奇数 | Hopf曲面复非Kähler |

### 退化与极限结构

```

结构退化链:

Kähler ──→ 辛 (forget J)
    ↓
复 (forget g)
    ↓
光滑 (forget J)

```

## 应用与例子

### 数学中的应用

| 领域 | 主要流形类型 | 核心问题 |
|-----|------------|---------|
| **代数几何** | 复射影簇 | 分类、模空间 |
| **辛几何** | 辛流形 | 同调、量子上同调 |
| **表示论** | 旗流形、齐性空间 | 几何表示 |
| **拓扑** | 光滑流形 | 手术、配边 |
| **分析** | Riemann流形 | 谱几何 |

### 物理学应用

| 理论 | 流形类型 | 应用 |
|-----|---------|-----|
| **经典力学** | 辛流形（余切丛） | Hamilton力学 |
| **广义相对论** | Lorentz流形 | 引力理论 |
| **弦理论** | Calabi-Yau 3- folds | 紧化 |
| **规范理论** | 主丛上的联络 | Yang-Mills |
| **凝聚态** | 环面、K3 | 拓扑绝缘体 |

### 经典例子对比

| 流形 | 类型 | 维数 | 关键性质 |
|-----|-----|-----|---------|
| **Sⁿ** | Riemann | n | 常正曲率 |
| **ℙⁿ(ℂ)** | Kähler | 2n | Fano，曲率 > 0 |
| **Tⁿ** | Kähler | n | 平坦，betti数最大 |
| **K3曲面** | Kähler | 4 | Ricci平坦，超Kähler |
| **ℂHⁿ** | Kähler | 2n | 负曲率，双曲空间 |
| **Flag流形** | Kähler | - | 齐性，有理 |

## 总结表：流形选择指南

| 研究目标 | 推荐流形 | 关键结构 |

| 曲率与距离 | Riemann流形 | Levi-Civita联络 |
| 复分析/代数几何 | 复流形/Kähler | 全纯结构 |
| 动力系统/力学 | 辛流形 | Hamilton结构 |
| 弦理论 | Calabi-Yau | SU(n)和乐 |
| 引力 | Lorentz流形 | Einstein方程 |
| 指标理论 | 旋量流形 | Dirac算子 |
