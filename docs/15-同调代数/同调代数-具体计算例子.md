# 同调代数具体计算例子

## 引言

同调代数不仅是一套抽象的理论框架，更是一种强大的计算工具。从链复形的同调群到Ext与Tor函子的具体值，许多看似抽象的结构在实例中呈现出清晰可计算的形态。本教程通过链复形、拓扑空间的同调群、以及Ext和Tor的典型计算，展示同调代数中核心构造的显式计算方法与技巧。

---

## 1. 链复形与同调群的基本计算

### 1.1 链复形的定义

**定义 1.1（链复形）**。一个 **链复形** $(C_\bullet, \partial)$ 是一列Abel群（或 $R$-模）$\{C_n\}_{n \in \mathbb{Z}}$ 连同 **边缘算子** $\partial_n: C_n \to C_{n-1}$，满足 $\partial_{n-1} \circ \partial_n = 0$ 对所有 $n$ 成立。等价地，$\operatorname{im}\partial_{n+1} \subseteq \ker\partial_n$。

**定义 1.2（同调群）**。链复形 $C_\bullet$ 的第 $n$ 个 **同调群** 定义为
$$H_n(C_\bullet) := \frac{\ker\partial_n}{\operatorname{im}\partial_{n+1}}.$$
元素 $[z] \in H_n(C_\bullet)$ 称为 **同调类**，其中 $z \in \ker\partial_n$ 为 **闭链**（cycle），$z \in \operatorname{im}\partial_{n+1}$ 为 **边缘链**（boundary）。

### 1.2 短链复形的计算

**例子 1.1**。考虑链复形
$$\cdots \to 0 \to \mathbb{Z} \xrightarrow{\times 2} \mathbb{Z} \to 0 \to \cdots$$
其中 $C_1 = C_0 = \mathbb{Z}$，其余 $C_n = 0$；$\partial_1 = \times 2$（乘以2），其余 $\partial_n = 0$。

**计算**：
- $n=0$：$\ker\partial_0 = \mathbb{Z}$（因 $C_{-1}=0$），$\operatorname{im}\partial_1 = 2\mathbb{Z}$。故
  $$H_0 = \mathbb{Z} / 2\mathbb{Z} = \mathbb{Z}_2.$$
- $n=1$：$\ker\partial_1 = \{n \in \mathbb{Z} : 2n=0\} = 0$，$\operatorname{im}\partial_2 = 0$。故
  $$H_1 = 0 / 0 = 0.$$
- $n \neq 0,1$：$C_n=0$，故 $H_n = 0$。

该复形的同调仅在0维非零，反映了 $\mathbb{Z}_2$ 的同调特征。

**例子 1.2（两个边缘算子的复合）**。考虑
$$\cdots \to 0 \to \mathbb{Z} \xrightarrow{\times 6} \mathbb{Z} \xrightarrow{\times 2} \mathbb{Z} \to 0 \to \cdots$$
其中 $C_2 = C_1 = C_0 = \mathbb{Z}$，$\partial_2 = \times 6$，$\partial_1 = \times 2$。注意 $\partial_1 \circ \partial_2 = \times 12 \neq 0$？不——这不构成链复形！

修正为：
$$\cdots \to 0 \to \mathbb{Z} \xrightarrow{\times 3} \mathbb{Z} \xrightarrow{\times 2} \mathbb{Z} \to 0 \to \cdots$$
此时 $\partial_1 \circ \partial_2 = \times 6$，仍不满足 $\partial^2 = 0$。

正确的构造：取 $C_2 = \mathbb{Z}$，$C_1 = \mathbb{Z}$，$C_0 = \mathbb{Z}/2\mathbb{Z}$，$\partial_2 = \pi \circ (\times 1)$（其中 $\pi: \mathbb{Z} \to \mathbb{Z}/2\mathbb{Z}$ 为投射），$\partial_1 = 0$。则 $\partial_1 \circ \partial_2 = 0$。

**计算**：
- $H_2 = \ker\partial_2 / 0 = 2\mathbb{Z} \cong \mathbb{Z}$（因 $\partial_2$ 的核为偶数）。
- $H_1 = \ker\partial_1 / \operatorname{im}\partial_2 = \mathbb{Z} / (\mathbb{Z}/2\mathbb{Z} \text{ 的原像}) = \mathbb{Z}/2\mathbb{Z}$？更仔细地：$\operatorname{im}\partial_2 = \mathbb{Z}/2\mathbb{Z}$ 视为 $C_1$ 的子群？实际上 $\partial_2: \mathbb{Z} \to \mathbb{Z}/2\mathbb{Z}$ 的像为整个 $\mathbb{Z}/2\mathbb{Z}$，但 $C_1 = \mathbb{Z}$，这里模结构不匹配。正确的做法应取 $C_1 = \mathbb{Z}/2\mathbb{Z}$。让我们重新构造：

$$C_2 = \mathbb{Z} \xrightarrow{\partial_2 = \pi} C_1 = \mathbb{Z}/2\mathbb{Z} \xrightarrow{\partial_1 = 0} C_0 = 0.$$

则 $H_2 = \ker\pi = 2\mathbb{Z} \cong \mathbb{Z}$，$H_1 = \mathbb{Z}/2\mathbb{Z}$。此复形的同调群为 $(\mathbb{Z}, \mathbb{Z}/2\mathbb{Z}, 0, \dots)$。

## 2. 拓扑空间的同调群

### 2.1 球面 $S^n$ 的同调

**定理 2.1**。$n$ 维球面 $S^n$ 的奇异同调群为
$$H_k(S^n; \mathbb{Z}) = \begin{cases} \mathbb{Z} & k = 0, n, \\ 0 & \text{否则}. \end{cases}$$

**证明概要**：利用Mayer-Vietoris序列。将 $S^n$ 分解为两个半球 $D_+^n, D_-^n$，其交 $D_+^n \cap D_-^n \simeq S^{n-1}$。每个半球的同调平凡（可缩空间）。对 $n \geq 2$，Mayer-Vietoris长正合列给出
$$\cdots \to H_k(D_+) \oplus H_k(D_-) \to H_k(S^n) \to H_{k-1}(S^{n-1}) \to H_{k-1}(D_+) \oplus H_{k-1}(D_-) \to \cdots$$
对 $k \geq 2$，首尾项为0，故 $H_k(S^n) \cong H_{k-1}(S^{n-1})$。由归纳法，$H_n(S^n) \cong H_1(S^1) = \mathbb{Z}$，$H_k(S^n)=0$ 对 $1 \leq k < n$。

对 $k=0$，$H_0(S^n) = \mathbb{Z}$ 反映 $S^n$ 的连通性。∎

**例 2.1（$S^2$ 的直观）**。$H_0(S^2) = \mathbb{Z}$：一个连通分支。$H_1(S^2) = 0$：任何环路在 $S^2$ 上可连续缩为一点（$S^2$ 单连通）。$H_2(S^2) = \mathbb{Z}$：$S^2$ 包围一个三维"洞"，二维闭链即球面本身，非边缘。

### 2.2 环面 $T^2 = S^1 \times S^1$ 的同调

**定理 2.2**。环面的同调群为
$$H_k(T^2; \mathbb{Z}) = \begin{cases} \mathbb{Z} & k = 0, 2, \\ \mathbb{Z}^2 & k = 1, \\ 0 & k > 2. \end{cases}$$

**证明**：利用Künneth公式：对拓扑空间 $X, Y$，
$$H_n(X \times Y) \cong \bigoplus_{p+q=n} H_p(X) \otimes H_q(Y) \oplus \bigoplus_{p+q=n-1} \operatorname{Tor}_1^\mathbb{Z}(H_p(X), H_q(Y)).$$

对 $X = Y = S^1$，$H_0(S^1) = H_1(S^1) = \mathbb{Z}$，其余为0。

- $n=0$：仅 $p=q=0$。$H_0 = \mathbb{Z} \otimes \mathbb{Z} = \mathbb{Z}$。
- $n=1$：$(p,q)=(1,0)$ 或 $(0,1)$。$H_1 = (\mathbb{Z} \otimes \mathbb{Z}) \oplus (\mathbb{Z} \otimes \mathbb{Z}) = \mathbb{Z}^2$。
- $n=2$：仅 $(p,q)=(1,1)$。$H_2 = \mathbb{Z} \otimes \mathbb{Z} = \mathbb{Z}$。

Tor项均消失因 $H_p(S^1)$ 皆为自由Abel群。∎

**几何解释**：$H_1(T^2) = \mathbb{Z}^2$ 的两个生成元对应环面的两个非平凡的独立环路——经圈（meridian）和纬圈（longitude）。$H_2(T^2) = \mathbb{Z}$ 的生成元是整个环面本身，它是一个非边缘的二维闭链。

### 2.3 实射影平面 $\mathbb{RP}^2$

**定理 2.3**。$\mathbb{RP}^2$ 的同调群为
$$H_k(\mathbb{RP}^2; \mathbb{Z}) = \begin{cases} \mathbb{Z} & k = 0, \\ \mathbb{Z}/2\mathbb{Z} & k = 1, \\ 0 & k \geq 2. \end{cases}$$

**证明**：利用胞腔分解。$\mathbb{RP}^2$ 有1个0-胞腔 $e^0$、1个1-胞腔 $e^1$、1个2-胞腔 $e^2$，附着映射为 $S^1 \to S^1$ 的二重覆盖 $z \mapsto z^2$。胞腔链复形为
$$0 \to \mathbb{Z} \xrightarrow{\times 2} \mathbb{Z} \xrightarrow{0} \mathbb{Z} \to 0.$$
故 $H_2 = 0$，$H_1 = \mathbb{Z}/2\mathbb{Z}$，$H_0 = \mathbb{Z}$。∎

**注记**：$H_1 = \mathbb{Z}/2\mathbb{Z}$ 反映了 $\mathbb{RP}^2$ 的**非定向性**。其上的任何环路要么可缩（边缘），要么绕两圈可缩，但绕一圈不可缩。

## 3. Ext与Tor的计算

### 3.1 在PID上的Ext

**定理 3.1**。设 $R$ 为主理想整环（PID），$M, N$ 为有限生成 $R$-模。则
$$\operatorname{Ext}^n_R(M, N) = 0 \quad \text{对 } n \geq 2.$$

这是因为PID上任何模的投射维数 $\leq 1$。

**例 3.1（$\operatorname{Ext}^1_\mathbb{Z}(\mathbb{Z}/m, \mathbb{Z}/n)$）**。取 $\mathbb{Z}/m\mathbb{Z}$ 的自由分解：
$$0 \to \mathbb{Z} \xrightarrow{\times m} \mathbb{Z} \to \mathbb{Z}/m\mathbb{Z} \to 0.$$
对 $\operatorname{Hom}_\mathbb{Z}(-, \mathbb{Z}/n\mathbb{Z})$ 应用，得
$$0 \to \operatorname{Hom}(\mathbb{Z}/m, \mathbb{Z}/n) \to \operatorname{Hom}(\mathbb{Z}, \mathbb{Z}/n) \xrightarrow{\times m} \operatorname{Hom}(\mathbb{Z}, \mathbb{Z}/n) \to \operatorname{Ext}^1(\mathbb{Z}/m, \mathbb{Z}/n) \to 0.$$
因 $\operatorname{Hom}(\mathbb{Z}, \mathbb{Z}/n) \cong \mathbb{Z}/n$，$\times m$ 的核为 $(\mathbb{Z}/n)[m] = \{x : mx=0\} \cong \mathbb{Z}/\gcd(m,n)$。由正合性，
$$\operatorname{Ext}^1_\mathbb{Z}(\mathbb{Z}/m, \mathbb{Z}/n) \cong \frac{\mathbb{Z}/n}{\operatorname{im}(\times m)} \cong \mathbb{Z}/\gcd(m,n)\mathbb{Z}.$$

**具体计算**：$\operatorname{Ext}^1_\mathbb{Z}(\mathbb{Z}/4, \mathbb{Z}/6) \cong \mathbb{Z}/\gcd(4,6)\mathbb{Z} = \mathbb{Z}/2\mathbb{Z}$。这对应于 $\mathbb{Z}/4\mathbb{Z}$ 由 $\mathbb{Z}/6\mathbb{Z}$ 的两个非等价扩张：分裂扩张 $\mathbb{Z}/6 \oplus \mathbb{Z}/4$ 和唯一非分裂扩张（$\mathbb{Z}/12 \oplus \mathbb{Z}/2$？不，需更仔细分类）。

### 3.2 在PID上的Tor

**定理 3.2**。设 $R$ 为PID，$M, N$ 为有限生成 $R$-模。则
$$\operatorname{Tor}_n^R(M, N) = 0 \quad \text{对 } n \geq 2.$$

**例 3.2（$\operatorname{Tor}^\mathbb{Z}_1(\mathbb{Z}/m, \mathbb{Z}/n)$）**。利用同一自由分解 $0 \to \mathbb{Z} \xrightarrow{\times m} \mathbb{Z} \to \mathbb{Z}/m \to 0$，取 $-\otimes_\mathbb{Z} \mathbb{Z}/n$：
$$0 \to \operatorname{Tor}_1(\mathbb{Z}/m, \mathbb{Z}/n) \to \mathbb{Z}/n \xrightarrow{\times m} \mathbb{Z}/n \to \mathbb{Z}/m \otimes \mathbb{Z}/n \to 0.$$
故
$$\operatorname{Tor}_1^\mathbb{Z}(\mathbb{Z}/m, \mathbb{Z}/n) \cong \ker(\times m: \mathbb{Z}/n \to \mathbb{Z}/n) \cong \mathbb{Z}/\gcd(m,n)\mathbb{Z}.$$

同时
$$\mathbb{Z}/m \otimes_\mathbb{Z} \mathbb{Z}/n \cong \mathbb{Z}/\gcd(m,n)\mathbb{Z}.$$

**具体计算**：$\operatorname{Tor}_1^\mathbb{Z}(\mathbb{Z}/4, \mathbb{Z}/6) \cong \mathbb{Z}/2\mathbb{Z}$。这反映了张量积 $-\otimes \mathbb{Z}/6$ 不保持单射 $0 \to \mathbb{Z} \xrightarrow{\times 4} \mathbb{Z}$：因 $2 \in \mathbb{Z}/6$ 满足 $4 \cdot 2 = 8 \equiv 2 \pmod 6 \neq 0$，实际上 $\ker(\times 4)$ 在 $\mathbb{Z}/6$ 中为 $\{0, 3\}$（因 $4 \cdot 3 = 12 \equiv 0$），故核为 $\mathbb{Z}/2\mathbb{Z}$。

## 4. 长正合序列的计算应用

### 4.1 由短正合序列导出同调

**定理 4.1**。给定短正合序列 of 链复形 $0 \to A_\bullet \to B_\bullet \to C_\bullet \to 0$，存在自然的长正合序列
$$\cdots \to H_n(A) \to H_n(B) \to H_n(C) \xrightarrow{\partial} H_{n-1}(A) \to H_{n-1}(B) \to \cdots$$

**例 4.1（计算 $\mathbb{RP}^2$ 的同调）**。令 $X = \mathbb{RP}^2$，$U$ 为去掉一点的邻域，$V$ 为包含该点的圆盘。$U \simeq S^1$，$V$ 可缩，$U \cap V \simeq S^1$。Mayer-Vietoris序列的一部分为
$$H_2(U) \oplus H_2(V) \to H_2(X) \to H_1(U \cap V) \xrightarrow{\phi} H_1(U) \oplus H_1(V).$$
因 $H_2(U)=H_2(V)=0$，$H_1(U \cap V)=\mathbb{Z}$，$H_1(U)=\mathbb{Z}$，$H_1(V)=0$，且 $\phi$ 为 $\times 2$（由 $U \cap V \to U$ 的二重覆盖），故
$$H_2(X) \cong \ker\phi = 0, \quad H_1(X) \cong \operatorname{coker}\phi = \mathbb{Z}/2\mathbb{Z}.$$
