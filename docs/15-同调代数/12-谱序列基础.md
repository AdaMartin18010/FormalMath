# 谱序列基础

## 引言

谱序列（spectral sequence）是同调代数中最强大但也最令人望而生畏的工具之一。它提供了一种系统性的方法，将一个复杂的双复形或滤过复形的同调信息"分层"计算。从Leray在1946年为层上同调发明谱序列，到Grothendieck将其推广到导出函子，再到今天谱序列成为代数拓扑、代数几何和表示论中的标准工具，谱序列理论经历了近80年的发展。

谱序列的核心思想是：不是直接计算目标同调群，而是构造一系列逐步逼近的"页"（pages）$E_r$，每一页都有微分 $d_r$，其同调给出下一页。在有利条件下，这个过程收敛到所求的同调群。

本教程介绍谱序列的基本定义、收敛概念以及第一象限谱序列的性质。

---

## 1. 谱序列的定义

### 1.1 基本构造

**定义 1.1**。一个**谱序列** $(E_r, d_r)_{r \geq r_0}$ 由以下数据组成：
1. 对每个 $r \geq r_0$，一族 $R$-模 $E_r = \{E_r^{p,q}\}_{p,q \in \mathbb{Z}}$；
2. 微分 $d_r: E_r^{p,q} \to E_r^{p+r, q-r+1}$（或适当双次数），满足 $d_r^2 = 0$；
3. $E_{r+1}^{p,q} \cong H(E_r^{p,q}, d_r)$。

### 1.2 收敛

**定义 1.2**。谱序列**收敛**到 $H^*$，记 $E_r \Rightarrow H^*$，如果存在 $H^n$ 的滤过

$$0 = F^{s+1} H^n \subseteq F^s H^n \subseteq \dots \subseteq F^0 H^n = H^n$$

使得 $E_\infty^{p,q} \cong F^p H^{p+q} / F^{p+1} H^{p+q}$。

---

## 2. 第一象限谱序列

### 2.1 定义

**定义 2.1**。谱序列称为**第一象限的**，如果 $E_r^{p,q} = 0$ 当 $p < 0$ 或 $q < 0$。

### 2.2 收敛性

**命题 2.2**。第一象限谱序列对每个 $(p,q)$ 稳定：当 $r > \max(p, q+1)$ 时，$E_r^{p,q} = E_{r+1}^{p,q} = \cdots = E_\infty^{p,q}$。

---

## 3. 重要例子

### 例子 1：滤过复形的谱序列

**例 3.1**。设 $C^\bullet$ 有下降滤过 $F^p C$。则存在谱序列

$$E_1^{p,q} = H^{p+q}(F^p C / F^{p+1} C) \Rightarrow H^{p+q}(C).$$

### 例子 2：双复形的谱序列

**例 3.2**。双复形 $C^{\bullet, \bullet}$ 有两种标准谱序列：
- **第一滤过**：$'E_2^{p,q} = H^p_H H^q_V(C) \Rightarrow H^{p+q}(\operatorname{Tot}(C))$。
- **第二滤过**：$''E_2^{p,q} = H^p_V H^q_H(C) \Rightarrow H^{p+q}(\operatorname{Tot}(C))$。
