# 15-同调代数

**代数拓扑的代数工具 — 从链复形到导出范畴**

---

## 📋 分支概览

同调代数是20世纪数学的重要创造，提供研究代数结构的统一框架：

- **链复形与上链复形**：同调代数的基本对象
- **导出函子**：Hom与⊗的派生
- **Tor与Ext**：同调代数的核心不变量
- **谱序列**：计算复杂同调的工具
- **导出范畴**：Verdier的三角范畴理论
- **三角范畴**：稳定同伦论与代数几何

**难度等级**: ⭐⭐⭐⭐⭐（研究级）
