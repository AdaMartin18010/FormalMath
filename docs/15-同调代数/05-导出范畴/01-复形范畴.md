# 复形范畴

**导出范畴的基石 — 从复形到同伦的提升**

---

## 1. 概念深度解析

### 1.1 代数直观

**复形范畴** $  ext{Ch}(  hetacal A)$ 是同调代数的出发点。在数学的各个分支中，我们频繁遇到"序列+微分"的结构：

- **代数拓扑**：奇异链复形 $C_  hetacalbullet(X)$
- **微分几何**：de Rham复形 $  hetacalOmega^*(M)$
- **交换代数**：Koszul复形、自由分解
- **代数几何**：层复形、Čech复形

复形范畴为这些结构提供了统一的抽象框架。

但直接使用复形范畴有问题：

- **拟同构（同调同构）应该被视为同构**：如果两个复形具有相同的同调，它们在几何/代数意义上应该是"等价"的
- **需要"反转"拟同构**：类似于同伦论中将弱等价视为同伦等价

**同伦范畴** $K(  hetacal A)$ 是第一步简化：将链同伦的映射视为相同。但同伦范畴中拟同构仍然不一定是同构，因此需要进一步局部化得到导出范畴。

### 1.2 范畴论语境

构造导出范畴的步骤：

```
复形范畴 Ch(A)
    ↓ 商去链同伦
同伦范畴 K(A)
    ↓ 局部化拟同构
导出范畴 D(A)
```

这个构造与拓扑空间范畴到同伦范畴的构造完全平行：

- $  ext{Ch}(  hetacal A)   hetacorresponds$ 拓扑空间范畴
- $K(  hetacal A)   hetacorresponds$ 同伦范畴（带基点空间）
- $D(  hetacal A)   hetacorresponds$ 稳定同伦范畴

### 1.3 形式定义

#### 定义 1.1 (复形范畴)

Abel范畴 $  hetacal A$ 的**复形范畴** $  ext{Ch}(  hetacal A)$：

- **对象**：链复形 $C_  hetacalbullet = (C_n,   hetapartial_n)$，满足 $  hetapartial_{n-1}   hetacalcirc   hetapartial_n = 0$
- **态射**：链映射 $f_  hetacalbullet : C_  hetacalbullet   hetato D_  hetacalbullet$，即交换图：

```
... → C_{n+1} → C_n → C_{n-1} → ...
      ↓ f_{n+1}   ↓ f_n    ↓ f_{n-1}
... → D_{n+1} → D_n → D_{n-1} → ...
```

**链复形的几何直观**：可以将链复形想象为一个"代数网格"，其中每个 $C_n$ 是 $n$ 维的"代数单元"，微分 $  hetapartial_n$ 描述了 $n$ 维单元的边界。条件 $  hetapartial^2 = 0$ 正是"边界的边界为空"的代数表达。

#### 定义 1.2 (同伦范畴)

**同伦范畴** $K(  hetacal A)$：

- **对象**：链复形
- **态射**：链映射的链同伦等价类

**链同伦**：两个链映射 $f, g: C   hetato D$ 是**链同伦**的，如果存在**同伦算子** $h_n: C_n   hetato D_{n+1}$ 使得：
$$f_n - g_n =   hetapartial_{n+1}^D   hetacalcirc h_n + h_{n-1}   hetacalcirc   hetapartial_n^C$$

**直观理解**：链同伦是链映射之间的"代数形变"。在拓扑中，两个连续映射之间的同伦是一个连续的形变过程；在代数中，链同伦用代数公式描述了两个映射如何"相差一个边缘"。

## 2. 属性与关系

### 2.1 三角结构

**定理 2.1 (映射锥)**
链映射 $f: C   hetato D$ 的**映射锥** $  ext{Cone}(f)$ 给出正合三角：
$$C   hetatoxrightarrow{f} D   hetato   ext{Cone}(f)   hetato C[1]$$

**映射锥的构造**：
$$  ext{Cone}(f)_n = D_n   hetacaloplus C_{n-1}$$
微分：
$$  hetadelta_n =   hetabegin{pmatrix}   hetapartial_n^D & f_{n-1} \\ 0 & -  hetapartial_{n-1}^C   hetaend{pmatrix}$$

**几何直观**：映射锥是代数版本的"映射柱"。在拓扑中，映射柱 $M_f$ 是将 $f: X   hetato Y$ 的像与 $Y$ 粘合得到的空间；代数中，映射锥 $  ext{Cone}(f)$ 编码了 $f$ 的"余核+1"信息。

**长正合上同调序列**：从映射锥三角得到的长正合序列正是：
$$  hetacdots   hetato H_n(C)   hetato H_n(D)   hetato H_n(  ext{Cone}(f))   hetato H_{n-1}(C)   hetato   hetacdots$$

### 2.2 同调函子

**定理 2.2**
同调 $H_n : K(  hetacal A)   hetato   hetacal A$ 是 well-defined 的函子。

**证明**：若 $f   hetasim g$（链同伦），则 $H_n(f) = H_n(g)$。这是因为对闭链 $z   hetacalin Z_n(C)$：
$$f_n(z) - g_n(z) =   hetapartial_{n+1}^D(h_n(z)) + h_{n-1}(  hetapartial_n^C(z)) =   hetapartial_{n+1}^D(h_n(z))   hetacalin B_n(D)$$

因此 $f_n(z)$ 与 $g_n(z)$ 相差一个边缘，在同调中相等。

## 3. 深入示例

### 3.1 例子：de Rham复形

设 $M$ 是光滑流形，$  hetacalOmega^*(M)$ 是其de Rham复形：
$$0   hetato   hetacalOmega^0(M)   hetatoxrightarrow{d}   hetacalOmega^1(M)   hetatoxrightarrow{d}   hetacalOmega^2(M)   hetato   hetacdots$$

**链映射例子**：设 $f: M   hetato N$ 是光滑映射，拉回 $f^*:   hetacalOmega^*(N)   hetato   hetacalOmega^*(M)$ 是链映射：
$$d(f^*  hetacalomega) = f^*(d  hetacalomega)$$

**同伦不变性**：若 $f   hetasim g$（光滑同伦），则 $f^*$ 与 $g^*$ 链同伦。这直接导致了de Rham上同调的同伦不变性：
$$H^*_{dR}(M)   hetacalcong H^*_{dR}(N)   hetatquad   ext{若 } M   hetasim N$$

### 3.2 例子：Koszul复形

设 $R$ 是交换环，$x   hetacalin R$。Koszul复形 $K(x)$：
$$0   hetato R   hetatoxrightarrow{x} R   hetato 0$$

对序列 $x_1, ..., x_n$，Koszul复形是张量积：
$$K(x_1, ..., x_n) = K(x_1)   hetacalotimes_R   hetacalcdots   hetacalotimes_R K(x_n)$$

**同调意义**：$H_0(K(x_1, ..., x_n)) = R/(x_1, ..., x_n)$。当 $x_1, ..., x_n$ 是正则序列时，Koszul复形是 $R/(x_1, ..., x_n)$ 的自由分解，即 $H_i = 0$（$i > 0$）。

**在导出范畴中**：Koszul复形给出了 $R/(x_1, ..., x_n)$ 的"投影分解"（在局部化意义下）。这在相交理论中至关重要：两个子簇的相交可以用张量积复形 $K(  hetacal I_X)   hetacalotimes K(  hetacal I_Y)$ 来描述。

## 4. 示例与习题

### 4.1 习题

#### 习题 1

证明链同伦是等价关系。

**详细证明**：

**自反性**：$f   hetasim f$，取 $h = 0$。

**对称性**：若 $f - g =   hetapartial h + h   hetapartial$，则 $g - f =   hetapartial(-h) + (-h)  hetapartial$，故 $g   hetasim f$。

**传递性**：若 $f   hetasim g$（通过 $h$）且 $g   hetasim k$（通过 $h'$），则：
$$f - k = (f - g) + (g - k) = (  hetapartial h + h   hetapartial) + (  hetapartial h' + h'   hetapartial) =   hetapartial(h + h') + (h + h')  hetapartial$$

故 $f   hetasim k$。$  hetacal square$

#### 习题 2

证明 $H_n$ 将链同伦的映射送到相等的映射。

**详细证明**：见定理2.2的证明。核心在于对任意闭链 $z   hetacalin Z_n(C)$（即 $  hetapartial z = 0$）：
$$f(z) - g(z) =   hetapartial(h(z))$$

因此在同调类 $[z]   hetacalin H_n(C)$ 上：
$$H_n(f)([z]) = [f(z)] = [g(z) +   hetapartial(h(z))] = [g(z)] = H_n(g)([z])$$

$  hetacal square$

#### 习题 3

证明：$f: C   hetato D$ 是链同伦等价当且仅当 $  ext{Cone}(f)$ 可缩。

**详细证明**：

**$(  hetacalRightarrow)$**：设 $g: D   hetato C$ 是 $f$ 的同伦逆，即 $g   hetacalcirc f   hetasim   ext{id}_C$，$f   hetacalcirc g   hetasim   ext{id}_D$。

需要证明 $  ext{Cone}(f)$ 的恒等映射链同伦于零。利用 $g$ 和链同伦 $h: C   hetato C$（$g   hetacalcirc f -   ext{id} =   hetapartial h + h   hetapartial$），可以显式构造收缩映射。

具体地，定义 $s:   ext{Cone}(f)_n   hetato   ext{Cone}(f)_{n+1}$ 为：
$$s(d, c) = (h'(d), g(d) + h(c))$$

其中 $h'$ 是 $f   hetacalcirc g   hetasim   ext{id}_D$ 的链同伦。可以验证 $  ext{id} =   hetapartial s + s   hetapartial$。

**$(  hetacalLeftarrow)$**：设 $  ext{Cone}(f)$ 可缩，即存在收缩 $s$ 使 $  ext{id} =   hetapartial s + s   hetapartial$。

由 $  ext{Cone}(f)_n = D_n   hetacaloplus C_{n-1}$，将 $s$ 写为矩阵：
$$s =   hetabegin{pmatrix}   hetacalalpha &   hetacalbeta \\   hetacalgamma &   hetacaldelta   hetaend{pmatrix}$$

从 $  hetapartial s + s   hetapartial =   ext{id}$ 的矩阵方程可以解出 $g = -  hetacalgamma$ 是 $f$ 的同伦逆。$  hetacal square$
