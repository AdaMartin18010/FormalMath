# 复形范畴

**导出范畴的基石 — 从复形到同伦的提升**

---

## 1. 概念深度解析

### 1.1 代数直观

**复形范畴** $\text{Ch}(\mathcal{A})$ 是同调代数的出发点：

- 对象：链复形
- 态射：链映射

但直接使用复形范畴有问题：

- 拟同构（同调同构）应该被视为同构
- 需要"反转"拟同构

### 1.2 范畴论语境

构造导出范畴的步骤：

```
复形范畴 Ch(A)
    ↓ 商去链同伦
同伦范畴 K(A)
    ↓ 局部化拟同构
导出范畴 D(A)
```

### 1.3 形式定义

#### 定义 1.1 (复形范畴)

Abel范畴 $\mathcal{A}$ 的**复形范畴** $\text{Ch}(\mathcal{A})$：

- **对象**：链复形 $C_\bullet = (C_n, \partial_n)$
- **态射**：链映射 $f_\bullet : C_\bullet \to D_\bullet$

#### 定义 1.2 (同伦范畴)

**同伦范畴** $K(\mathcal{A})$：

- **对象**：链复形
- **态射**：链映射的链同伦等价类

---

## 2. 属性与关系

### 2.1 三角结构

**定理 2.1 (映射锥)**
链映射 $f: C \to D$ 的**映射锥** $\text{Cone}(f)$ 给出正合三角：
$$C \xrightarrow{f} D \to \text{Cone}(f) \to C[1]$$

### 2.2 同调函子

**定理 2.2**
同调 $H_n : K(\mathcal{A}) \to \mathcal{A}$ 是 well-defined 的函子。

---

## 3. 示例与习题

### 3.1 习题

#### 习题 1

证明链同伦是等价关系。

#### 习题 2

证明 $H_n$ 将链同伦的映射送到相等的映射。

#### 习题 3

证明：$f: C \to D$ 是链同伦等价当且仅当 $\text{Cone}(f)$ 可缩。

## 4. 形式化实现 (Lean 4)

```lean4
import Mathlib.Algebra.Homology.ChainComplex

variable {C : Type*} [Category C] [Abelian C]

-- ============================================
-- 复形范畴
-- ============================================

def ChainComplexCat : Category (ChainComplex C (ComplexShape.down ℤ)) :=
  inferInstance

-- ============================================
-- 同伦范畴
-- ============================================

def HomotopyCategory := HomotopyCategory C (ComplexShape.down ℤ)

def quotientFunctor : ChainComplex C (ComplexShape.down ℤ) ⥤ HomotopyCategory C :=
  HomotopyCategory.quotient _ _
```
