# 拟同构与局部化

**导出范畴的构造 — 从拟同构到同构的转化**

---

## 1. 概念深度解析

### 1.1 代数直观

**拟同构 (Quasi-isomorphism)** 是同调同构的链映射：

- $f: C \to D$ 是拟同构如果 $H_n(f)$ 对所有n是同构
- 在代数拓扑中，这是"弱等价"
- 在导出范畴中，拟同构变为真正的同构

### 1.2 范畴论语境

**局部化**是范畴论中将某些态射"反转"的通用构造：

- 给定范畴 $\mathcal{C}$ 和态射类 $S$
- 构造 $S^{-1}\mathcal{C}$ 使得 $S$ 中态射成为同构

### 1.3 形式定义

#### 定义 1.1 (拟同构)

链映射 $f: C \to D$ 是**拟同构**，如果对所有n：
$$H_n(f) : H_n(C) \xrightarrow{\cong} H_n(D)$$

#### 定义 1.2 (导出范畴)

**导出范畴** $D(\mathcal{A})$ 是同伦范畴关于拟同构的局部化：
$$D(\mathcal{A}) = K(\mathcal{A})[\text{qis}^{-1}]$$

**态射描述**（ roofs / coroofs）：

```
    C'
   /  \
  f    g
 /      \
C        D
```

其中f是拟同构。

## 2. 属性与关系

### 2.1 导出范畴的泛性质

**定理 2.1**
$Q: K(\mathcal{A}) \to D(\mathcal{A})$ 满足：

- $Q(s)$ 是同构对所有拟同构s
- 任何将拟同构送到同构的函子唯一通过Q分解

### 2.2 有界导出范畴

**定义 2.1**

- $D^+(\mathcal{A})$：下有界复形
- $D^-(\mathcal{A})$：上有界复形
- $D^b(\mathcal{A})$：有界复形

**定理 2.2**
若 $\mathcal{A}$ 有足够多的内射对象，则 $D^+(\mathcal{A}) \cong K^+(\mathcal{I})$。

## 3. 示例与习题

### 3.1 习题

#### 习题 1

证明：两个拟同构的复合是拟同构。

#### 习题 2

构造两个拟同构 $C \xrightarrow{f} D \xleftarrow{g} E$，证明 roof 等价类构成态射。

#### 习题 3

证明 $D^b(\text{Vec}_k)$ 等价于分次向量空间范畴。
