# t-结构

**导出范畴的心脏 — 从三角到Abel的桥梁**

---

## 1. 概念深度解析

### 1.1 代数直观

**t-结构 (t-structure)** 是导出范畴中的"切片"：

- 将导出范畴分解为"≤0"和"≥0"两部分
- **心脏** (heart) 是三角范畴中的Abel子范畴
- 提供导出范畴的上同调理论

### 1.2 范畴论语境

**动机**：

- 导出范畴 $D(\mathcal{A})$ 是三角的但非Abel的
- t-结构允许在三角范畴中做"同调代数"
- 不同t-结构给出不同的心

### 1.3 形式定义

#### 定义 1.1 (t-结构)

**t-结构** $(D^{\leq 0}, D^{\geq 0})$ 满足：

1. $D^{\leq 0} \subseteq D^{\leq 1}$，$D^{\geq 0} \supseteq D^{\geq 1}$
2. $\text{Hom}(D^{\leq 0}, D^{\geq 1}) = 0$
3. 任意 $X$ 有三角 $A \to X \to B \to$ 其中 $A \in D^{\leq 0}$，$B \in D^{\geq 1}$

#### 定义 1.2 (心脏)

$$\mathcal{C} = D^{\leq 0} \cap D^{\geq 0}$$
是Abel范畴。

---

## 2. 属性与关系

### 2.1 上同调函子

**定理 2.1**
设 $(D^{\leq 0}, D^{\geq 0})$ 是t-结构，心脏 $\mathcal{C}$。

存在上同调函子 $H^0: \mathcal{D} \to \mathcal{C}$ 满足：

- 将三角送到长正合列
- $H^0(X) = X$ 对 $X \in \mathcal{C}$

### 2.2 标准t-结构

**例子**：$D(\mathcal{A})$ 的标准t-结构

- $D^{\leq 0} = \{C : H^i(C) = 0 \text{ 对 } i > 0\}$
- $D^{\geq 0} = \{C : H^i(C) = 0 \text{ 对 } i < 0\}$
- 心脏 $\cong \mathcal{A}$

### 2.3 反常t-结构

**Perverse t-结构**：

- 在导出层范畴上
- 用于相交上同调
- 心脏 = 反常层
