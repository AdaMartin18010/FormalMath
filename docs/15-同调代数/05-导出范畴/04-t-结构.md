# t-结构：导出范畴的心脏与Abel范畴的桥梁

## 1. 引言

导出范畴（derived category）$D(\mathcal{A})$ 是代数几何、代数拓扑与表示论中的核心工具，它将链复形间的拟同构逆转为真正的同构，使得层上同调、Ext与Tor等经典构造获得函子性的统一描述。然而，导出范畴本身是三角范畴而非Abel范畴——它没有良好的正合列结构，无法直接进行同调代数运算。

t-结构（t-structure）由Beilinson、Bernstein与Deligne在解决Kazhdan-Lusztig猜想时系统引入，它通过在导出范畴中定义"截断"操作，构造出一个Abel子范畴（称为心脏，heart），使得导出范畴重获同调代数的结构。反常层理论（perverse sheaves）正是t-结构在层范畴上的应用，在代数群表示论与朗兰兹纲领中发挥了革命性作用。

本文严格定义t-结构，证明心脏的Abel性，阐述标准t-结构与反常t-结构，并给出完整的计算例子。

---

## 2. 严格数学定义

### 2.1 t-结构

**定义 2.1（t-结构）**
设 $\mathcal{D}$ 为三角范畴。一个**t-结构** $(\mathcal{D}^{\leq 0}, \mathcal{D}^{\geq 0})$ 是一对全子范畴，满足：

1. **单调性**：$\mathcal{D}^{\leq 0} \subseteq \mathcal{D}^{\leq 1}$，$\mathcal{D}^{\geq 0} \supseteq \mathcal{D}^{\geq 1}$（其中 $\mathcal{D}^{\leq n} = \mathcal{D}^{\leq 0}[-n]$，$\mathcal{D}^{\geq n} = \mathcal{D}^{\geq 0}[-n]$）
2. **正交性**：$\text{Hom}(\mathcal{D}^{\leq 0}, \mathcal{D}^{\geq 1}) = 0$
3. **截断存在性**：对任意 $X \in \mathcal{D}$，存在 distinguished triangle
   $$A \to X \to B \to A[1]$$
   其中 $A \in \mathcal{D}^{\leq 0}$，$B \in \mathcal{D}^{\geq 1}$

**定义 2.2（心脏）**
t-结构的**心脏**（heart）定义为
$$\mathcal{C} = \mathcal{D}^{\leq 0} \cap \mathcal{D}^{\geq 0}$$

**定义 2.3（截断函子）**
由截断三角的唯一性（在同构意义下），可定义函子
$$\tau^{\leq 0}: \mathcal{D} \to \mathcal{D}^{\leq 0}, \quad \tau^{\geq 1}: \mathcal{D} \to \mathcal{D}^{\geq 1}$$
满足 $A \cong \tau^{\leq 0} X$，$B \cong \tau^{\geq 1} X$。

### 2.2 上同调函子

**定义 2.4（上同调函子）**
t-结构的**上同调函子** $H^0: \mathcal{D} \to \mathcal{C}$ 定义为
$$H^0(X) = \tau^{\geq 0}\tau^{\leq 0} X \cong \tau^{\leq 0}\tau^{\geq 0} X$$
高阶上同调：$H^n(X) = H^0(X[n]) \in \mathcal{C}$。

## 3. 核心定理与证明

### 定理 3.1（心脏是Abel范畴）

设 $(\mathcal{D}^{\leq 0}, \mathcal{D}^{\geq 0})$ 为三角范畴 $\mathcal{D}$ 上的t-结构，心脏 $\mathcal{C} = \mathcal{D}^{\leq 0} \cap \mathcal{D}^{\geq 0}$。则 $\mathcal{C}$ 是Abel范畴，且短正合列 $0 \to A \to B \to C \to 0$ 在 $\mathcal{C}$ 中正合当且仅当存在 distinguished triangle $A \to B \to C \to A[1]$ 在 $\mathcal{D}$ 中且 $A, B, C \in \mathcal{C}$。

**证明**：

**(1) 核与余核的存在性**：设 $f: A \to B$ 为 $\mathcal{C}$ 中的态射。在 $\mathcal{D}$ 中完成 distinguished triangle
$$A \xrightarrow{f} B \to C \to A[1]$$

定义 $K = H^0(C[-1]) \in \mathcal{C}$，$Q = H^0(C) \in \mathcal{C}$。由截断的性质，存在自然映射 $K \to A$（核）和 $B \to Q$（余核）。

**(2) 验证核的泛性质**：设 $g: D \to A$（$D \in \mathcal{C}$）满足 $f \circ g = 0$。由正交性，$g$ 通过 $K$ 分解。具体地，$D \in \mathcal{D}^{\geq 0}$，$C[-1] \in \mathcal{D}^{\leq 0}$（因 $C \in \mathcal{D}^{\geq -1}$ 由构造），故 $\text{Hom}(D, C[-1]) \cong \text{Hom}(D, H^0(C[-1])) = \text{Hom}(D, K)$。

**(3) 单射与满射的刻画**：$f$ 在 $\mathcal{C}$ 中单射 $\Leftrightarrow$ $K = 0$ $\Leftrightarrow$ $C[-1] \in \mathcal{D}^{\leq -1}$ $\Leftrightarrow$ $C \in \mathcal{D}^{\geq 0}$。此时 $C \in \mathcal{C}$ 且 $B \to C$ 为余核。对偶地，$f$ 满射 $\Leftrightarrow$ $Q = 0$ $\Leftrightarrow$ $C \in \mathcal{D}^{\leq 0}$，此时 $C \in \mathcal{C}$。

**(4) 同构定理**：$\text{Coim}(f) = \text{Coker}(K \to A) \cong \text{Ker}(B \to Q) = \text{Im}(f)$ 由三角范畴的八面体公理验证。$\square$

### 定理 3.2（标准t-结构的上同调）

设 $\mathcal{D} = D(\mathcal{A})$ 为Abel范畴的导出范畴，标准t-结构：
- $\mathcal{D}^{\leq 0} = \{C : H^i(C) = 0 \text{ 对所有 } i > 0\}$
- $\mathcal{D}^{\geq 0} = \{C : H^i(C) = 0 \text{ 对所有 } i < 0\}$

则心脏 $\mathcal{C} \cong \mathcal{A}$，且上同调函子 $H^0$ 就是通常的复形上同调。

**证明**：

**(1) 心脏识别**：$C \in \mathcal{C}$ 当且仅当 $H^i(C) = 0$（$i \neq 0$）。由导出范畴的性质，这样的复形同构于集中在0次的一个对象 $A[0]$（$A \in \mathcal{A}$）。故 $\mathcal{C} \simeq \mathcal{A}$。

**(2) 截断验证**：对复形 $C$，定义
$$(\tau^{\leq 0} C)^i = \begin{cases} C^i & i < 0 \\ \ker(d^0) & i = 0 \\ 0 & i > 0 \end{cases}$$
$$(\tau^{\geq 1} C)^i = \begin{cases} 0 & i < 1 \\ \text{coker}(d^0) & i = 1 \\ C^i & i > 1 \end{cases}$$

验证 $\tau^{\leq 0} C \to C \to \tau^{\geq 1} C \to$ 为 distinguished triangle：其 Cone 为短正合列 $0 \to \tau^{\leq 0} C \to C \to \tau^{\geq 1} C \to 0$ 的三角化。

**(3) 上同调函子**：$H^0(C) = H^0(\tau^{\leq 0} C) = \ker(d^0)/\text{im}(d^{-1})$，即通常的上同调。$\square$

## 4. 详细例子

### 例 4.1：$D^b(\text{Vec}_\mathbb{K})$ 的标准t-结构

设 $\mathcal{A} = \text{Vec}_\mathbb{K}$（有限维 $\mathbb{K}$-向量空间），$\mathcal{D} = D^b(\mathcal{A})$。

对复形 $C = [C^{-1} \xrightarrow{d} C^0 \xrightarrow{d} C^1]$（其余项为零），$H^0(C) = \ker(d: C^0 \to C^1) / \text{im}(d: C^{-1} \to C^0)$。

截断：
- $\tau^{\leq 0} C = [C^{-1} \to \ker(d: C^0 \to C^1)]$（在0次截断）
- $\tau^{\geq 1} C = [\text{coker}(d: C^{-1} \to C^0) \to C^1]$（从1次开始）

验证 distinguished triangle：短正合列
$$0 \to \tau^{\leq 0} C \to C \to \tau^{\geq 1} C \to 0$$
在复形范畴中正合，导出范畴中三角化。

### 例 4.2：反常t-结构与反常层

设 $X$ 为复代数簇，维度 $n$。导出层范畴 $D^b_c(X)$ 上的**中间反常t-结构**：
- ${}^p\mathcal{D}^{\leq 0} = \{K : \dim \text{supp}\, H^i(K) \leq -i, \forall i\}$
- ${}^p\mathcal{D}^{\geq 0} = \{K : \dim \text{supp}\, H^i(\mathbb{D}K) \leq -i, \forall i\}$

其中 $\mathbb{D}$ 为Verdier对偶。心脏为**反常层**（perverse sheaves）范畴 $\text{Perv}(X)$。

对光滑曲线 $X$ 上的局部系统 $\mathcal{L}$（秩 $r$），$\mathcal{L}[1]$ 是反常层（因支撑维度为1，移位使 $i = -1$ 时 $H^{-1}(\mathcal{L}[1]) = \mathcal{L}$ 满足条件）。

反常层的简单对象对应 intersection cohomology complexes $IC_Y(\mathcal{L})$，在Kazhdan-Lusztig猜想中给出李代数的Kazhdan-Lusztig多项式。

## 5. 进阶主题

### 5.1 t-结构的粘合

对开浸入 $j: U \hookrightarrow X$ 与闭浸入 $i: Z \hookrightarrow X$（$Z = X \setminus U$），$D^b_c(X)$ 上的t-结构可由 $U$ 与 $Z$ 上的t-结构粘合得到。这对应于"胶合数据"（recollement）：
$$j_! \dashv j^* \dashv j_* \quad \text{和} \quad i_* \dashv i^! \dashv i^!$$
反常t-结构正是由点上的标准t-结构通过这种粘合归纳构造的。

### 5.2 与导出代数几何的联系

在导出代数几何中，t-结构对应于"经典截断"（classical truncation）。对导出叠 $X$，其结构层 $\mathcal{O}_X$ 是微分分次环，t-结构给出叠的底拓扑空间 $|X|$ 与结构层 $\pi_0(\mathcal{O}_X)$。
