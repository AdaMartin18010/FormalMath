# 三角范畴

**同调代数的范畴框架 — Verdier的形式化公理**

---

## 1. 概念深度解析

### 1.1 代数直观

**三角范畴**是带有"正合三角"结构的加性范畴：

- 类比Abel范畴的正合序列
- 用三角 $A \to B \to C \to A[1]$ 代替短正合列
- 移位函子 $[1]$ 是核心结构

### 1.2 范畴论语境

**Verdier的动机**：

- 同伦范畴和导出范畴都有三角结构
- 三角结构是"同调"的恰当范畴框架
- 比Abel范畴更灵活

### 1.3 形式定义

#### 定义 1.1 (三角范畴)

**三角范畴** $(\mathcal{D}, [1], \mathcal{T})$ 包含：

- 加性范畴 $\mathcal{D}$
- 自等价函子 $[1]: \mathcal{D} \to \mathcal{D}$（移位）
- 三角类 $\mathcal{T}$（形如 $A \to B \to C \to A[1]$）

满足Verdier公理TR1-TR4。

#### 公理 TR1

- 每个 $f: A \to B$ 可扩充为三角
- 三角在同构下封闭
- $A \xrightarrow{\text{id}} A \to 0 \to A[1]$ 是三角

#### 公理 TR2 (旋转)

$(u, v, w)$ 是三角当且仅当 $(v, w, -u[1])$ 是三角。

#### 公理 TR3 (态射)

三角间的态射可扩充。

#### 公理 TR4 (八面体)

三角的复合性质。

---

## 2. 属性与关系

### 2.1 上同调函子

**定义 2.1**
$H: \mathcal{D} \to \mathcal{A}$ 是**上同调函子**，如果：

- 将三角送到长正合列
- 与移位相容

**例子**：

- $H^0 : D(\mathcal{A}) \to \mathcal{A}$

### 2.2 厚子范畴与Verdier商

**定义 2.2 (厚子范畴)**
$\mathcal{N} \subseteq \mathcal{D}$ 是**厚子范畴**，如果对任意三角中两个对象在 $\mathcal{N}$，第三个也在。

**定理 2.1**
可构造商范畴 $\mathcal{D}/\mathcal{N}$。

```lean4
import Mathlib.CategoryTheory.Triangulated.Basic

-- ============================================
-- 三角范畴公理
-- ============================================

class TriangulatedCategory (C : Type*) [Category C] [HasShift C ℤ] where
  distinguishedTriangles : Set (Triangle C)
  isomorphic_distinguished : ∀ T ∈ distinguishedTriangles, ∀ T' ≅ T, T' ∈ distinguishedTriangles
  contractible_distinguished : ∀ X, contractibleTriangle X ∈ distinguishedTriangles
  distinguished_cocone_triangle : ∀ (f : X ⟶ Y), ∃ (Z : C) (g : Y ⟶ Z) (h : Z ⟶ X⟦1⟧),
    Triangle.mk f g h ∈ distinguishedTriangles
  rotate_distinguished : ∀ T ∈ distinguishedTriangles, T.rotate ∈ distinguishedTriangles
  -- ...
```
