# 层上同调

**拓扑与几何的上同调工具 — 从局部到整体的统一**

---

## 1. 概念深度解析

### 1.1 代数直观

**层上同调** $H^i(X, \mathcal{F})$ 度量层 $\mathcal{F}$ 的整体截面的"障碍"：

- $H^0$：整体截面
- $H^1$：将局部截面粘贴成整体的障碍
- 更高维：高阶障碍

### 1.2 几何背景

层论提供了统一的语言：

- 连续函数层
- 微分形式层
- 代数结构层

### 1.3 形式定义

#### 定义 1.1 (层上同调)

设X是拓扑空间，$\mathcal{F}$ 是Abel群层。
$$H^i(X, \mathcal{F}) = R^i\Gamma(X, \mathcal{F})$$
其中 $\Gamma(X, -)$ 是整体截面函子。

#### 定义 1.2 (Čech上同调)

对开覆盖 $\mathcal{U} = \{U_i\}$：
$$\check{H}^i(\mathcal{U}, \mathcal{F}) = H^i(C^\bullet(\mathcal{U}, \mathcal{F}))$$
其中 $C^p = \prod_{i_0 < ... < i_p} \mathcal{F}(U_{i_0} \cap ... \cap U_{i_p})$。

---

## 2. 属性与关系

### 2.1 Leray定理

**定理 2.1 (Leray)**
若 $\mathcal{U}$ 是 $X$ 的Leray覆盖（即对所有有限交，$\check{H}^i = 0$ 对 $i > 0$），则：
$$\check{H}^i(\mathcal{U}, \mathcal{F}) \cong H^i(X, \mathcal{F})$$

### 2.2 Leray谱序列

**定理 2.2**
设 $f: X \to Y$，则：
$$E_2^{p,q} = H^p(Y, R^q f_* \mathcal{F}) \Rightarrow H^{p+q}(X, \mathcal{F})$$

### 2.3 de Rham定理

**定理 2.3**
对光滑流形X：
$$H^i_{dR}(X) \cong H^i(X, \mathbb{R})$$

### 3.1 习题

## 4. 形式化实现 (Lean 4)

```lean4
import Mathlib.AlgebraicGeometry.Scheme
import Mathlib.Algebra.Category.ModuleCat.Abelian

-- ============================================
-- 层上同调
-- ============================================

variable {X : Scheme} (F : SheafAb X)

def SheafCohomology (i : ℕ) : Type _ :=
  F.cohomology i

-- ============================================
-- Čech上同调
-- ============================================

def CechCohomology {I : Type*} (U : I → Opens X) (i : ℕ) : Type _ :=
  F.ČechCohomology U i
```
