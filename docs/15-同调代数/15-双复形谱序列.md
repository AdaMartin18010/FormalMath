# 双复形谱序列

## 引言

双复形（double complex）是带有水平和垂直两种微分的代数结构，其总复形将双复形压缩为单复形。总复形的上同调通常难以直接计算，但双复形的两种标准谱序列——分别对应于先算垂直方向再算水平方向，或反之——提供了强大的间接计算工具。在许多实际应用中，一种谱序列容易计算且退化，从而"免费"给出总复形同调的信息。这种"从两边夹逼"的策略是谱序列技术的精髓，也是同调代数中最具计算威力的方法之一。

本教程从双复形的严格定义出发，构造两种滤过谱序列，分析其收敛性与退化条件，并以Čech-de Rham双复形和Künneth公式为例展示应用。

---

## 1. 双复形与总复形

### 1.1 定义

**定义 1.1**。$R$-模的 **双复形** $(C^{\bullet,\bullet}, d_h, d_v)$ 由双分次模 $C^{p,q}$（$p,q \in \mathbb{Z}$）和两个微分组成：

- **水平微分** $d_h: C^{p,q} \to C^{p+1,q}$，满足 $d_h^2 = 0$；
- **垂直微分** $d_v: C^{p,q} \to C^{p,q+1}$，满足 $d_v^2 = 0$；
- **交换性**：$d_h d_v + d_v d_h = 0$（或带符号的严格交换）。

**定义 1.2（总复形）**。总复形 $\operatorname{Tot}^n(C)$ 定义为
$$\operatorname{Tot}^n(C) := \bigoplus_{p+q=n} C^{p,q},$$
微分 $D = d_h + d_v: \operatorname{Tot}^n \to \operatorname{Tot}^{n+1}$。由交换性，$D^2 = d_h^2 + d_h d_v + d_v d_h + d_v^2 = 0$。

---

## 2. 滤过与谱序列

### 2.1 第一滤过（水平优先）

在总复形上定义滤过 $'F^p \operatorname{Tot}^n = \bigoplus_{i \geq p} C^{i, n-i}$。

**定理 2.1**。该滤过给出的谱序列满足
$$'E_0^{p,q} = C^{p,q}, \quad 'd_0 = d_v,$$
$$'E_1^{p,q} = H^q_v(C^{p,\bullet}), \quad 'd_1 = d_h^*,$$
$$'E_2^{p,q} = H^p_h(H^q_v(C)) \Rightarrow H^{p+q}(\operatorname{Tot}(C)).$$

### 2.2 第二滤过（垂直优先）

类似地，$''F^q \operatorname{Tot}^n = \bigoplus_{j \geq q} C^{n-j, j}$ 给出
$$''E_2^{p,q} = H^q_v(H^p_h(C)) \Rightarrow H^{p+q}(\operatorname{Tot}(C)).$$

---

## 3. 退化条件

### 3.1 行正合与列正合

**命题 3.1**。若双复形的所有行正合（每行 $C^{\bullet,q}$ 作为水平复形零调），则 ''E_2 退化：$''E_2^{p,q} = 0$ 对 $p > 0$，且 $H^n(\operatorname{Tot}) \cong H^n_v(\ker d_h)$。

**命题 3.2**。若所有列正合，则 'E_2 退化。

### 3.2 第一象限双复形

若 $C^{p,q} = 0$ 对 $p < 0$ 或 $q < 0$，称双复形为 **第一象限** 的。此时谱序列强收敛到总复形上同调。

---

## 4. 应用例子

### 4.1 Čech-de Rham谱序列

设 $M$ 为光滑流形，$\mathcal{U} = \{U_i\}$ 为开覆盖。定义双复形
$$C^{p,q} = \check{C}^p(\mathcal{U}, \Omega^q) = \prod_{i_0 < \dots < i_p} \Omega^q(U_{i_0} \cap \dots \cap U_{i_p}).$$

- 垂直微分 $d_v$：de Rham外微分 $d$；
- 水平微分 $d_h$：Čech上边缘 $\delta$。

**第一谱序列**：$'E_1^{p,q} = \check{C}^p(\mathcal{U}, \mathcal{H}^q)$，其中 $\mathcal{H}^q$ 为 de Rham上同调层（局部常值）。若覆盖是 **好覆盖**（good cover，即有限交可缩），则 $\mathcal{H}^q$ 仅在 $q=0$ 非零（为常值层 $\underline{\mathbb{R}}$）。故 $'E_2^{p,q} = 0$ 对 $q > 0$，$'E_2^{p,0} = \check{H}^p(\mathcal{U}, \mathbb{R})$。

**第二谱序列**：$''E_1^{p,q} = \prod_{i_0 < \dots < i_p} H^q_{dR}(U_{i_0} \cap \dots \cap U_{i_p})$。对好覆盖，非空交可缩，$H^q_{dR} = 0$ 对 $q > 0$，$H^0_{dR} = \mathbb{R}$。故 $''E_1$ 只在 $q=0$ 非零，$''E_2^{p,0} = H^p_{dR}(M)$（由Poincaré引理）。

两种谱序列收敛到同一目标，故
$$\check{H}^p(\mathcal{U}, \mathbb{R}) \cong H^p_{dR}(M).$$
这是 **de Rham定理** 的一种证明。

### 4.2 Künneth公式的谱序列证明

设 $A^\bullet, B^\bullet$ 为 cochain complexes。定义双复形 $C^{p,q} = A^p \otimes B^q$，$d_h = d_A \otimes \mathrm{id}$，$d_v = (-1)^p \mathrm{id} \otimes d_B$。

总复形即 $A \otimes B$ 的张量积复形。

**第一谱序列**：$'E_2^{p,q} = H^p(A) \otimes H^q(B)$（当 $R$ 为域）。微分 $d_2$ 为零（因次数原因）。故谱序列在 $E_2$ 退化：
$$H^n(A \otimes B) \cong \bigoplus_{p+q=n} H^p(A) \otimes H^q(B).$$
此即 **Künneth公式**。
