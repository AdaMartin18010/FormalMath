# 15-同调代数 内容索引

**Homological Algebra — 从链复形到导出范畴的完整理论体系**

---

## 📋 分支概览

同调代数是20世纪数学最重要的创造之一，为研究代数、几何、拓扑结构提供统一框架。本分支涵盖从基础理论到前沿应用的完整体系。

**难度等级**: ⭐⭐⭐⭐⭐（研究级）
