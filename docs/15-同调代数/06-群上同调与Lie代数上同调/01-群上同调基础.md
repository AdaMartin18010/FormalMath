# 群上同调基础

**群论的同调工具 — 从标准复形到低维不变量**

---

## 1. 概念深度解析

### 1.1 代数直观

**群上同调** $H^n(G, M)$ 度量群G在模M上的"扭曲"：

- $n=1$：交叉同态模主交叉同态
- $n=2$：群扩张的分类
- $n=3$：障碍类

### 1.2 范畴论语境

群上同调是函子 $H^n(G, -) = R^n(-)^G$：

- 固定子群函子的右导出
- 或 $\text{Ext}^n_{\mathbb{Z}[G]}(\mathbb{Z}, M)$

### 1.3 形式定义

#### 定义 1.1 (群上同调)

设G是群，M是G-模。
$$H^n(G, M) = \text{Ext}^n_{\mathbb{Z}[G]}(\mathbb{Z}, M)$$
其中$\mathbb{Z}$是平凡G-模。

#### 定义 1.2 (标准复形)

$$C^n(G, M) = \text{Fun}(G^n, M)$$
$$(\delta f)(g_0, ..., g_n) = g_0 f(g_1, ...) + \sum (-1)^i f(..., g_i g_{i+1}, ...) + (-1)^{n+1} f(..., g_{n-1})$$

---

## 2. 属性与关系

### 2.1 低维解释

**定理 2.1**

- $H^0(G, M) = M^G$（固定子群）
- $H^1(G, M) = \text{交叉同态}/\text{主交叉同态}$
- $H^2(G, M) = \{\text{扩张 } 0 \to M \to E \to G \to 1\}$

### 2.2 Hochschild-Serre谱序列

**定理 2.2**
设 $H \triangleleft G$，则：
$$E_2^{p,q} = H^p(G/H, H^q(H, M)) \Rightarrow H^{p+q}(G, M)$$
