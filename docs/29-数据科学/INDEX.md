## 学习路径

### 基础路径
```
线性回归 → 逻辑回归 → 正则化 → 核方法 → 神经网络
```

### 进阶路径
```
统计学习理论 → 高维统计 → 优化理论 → 深度学习 → 强化学习
```

### 应用路径
- **数据挖掘**: 特征工程 + 模型选择 + 评估指标
- **计算机视觉**: CNN + 目标检测 + 分割
- **自然语言处理**: 序列模型 + Transformer + 预训练
- **推荐系统**: 协同过滤 + 矩阵分解 + 深度学习

## 核心定理速查

| 定理/概念 | 描述 | 应用 |
|-----------|------|------|
| **无免费午餐** | 没有 universally 最好的算法 | 算法选择 |
| **大数定律** | 经验风险收敛于期望风险 | 泛化保证 |
| **VC界** | $R(h) \leq \hat{R}(h) + O(\sqrt{d/n})$ | 复杂度控制 |
| **表示定理** | 最优解在核张成的空间中 | 核方法 |
| **通用近似** | 神经网络可逼近任意连续函数 | 深度学习 |

## 算法复杂度

| 算法 | 训练复杂度 | 预测复杂度 | 特点 |
|------|------------|------------|------|
| 线性回归 | $O(nd^2)$ | $O(d)$ | 解析解 |
| 逻辑回归 | $O(nd \cdot \text{iter})$ | $O(d)$ | 凸优化 |
| SVM | $O(n^2d)$ to $O(n^3d)$ | $O(sd)$ | s支持向量数 |
| 随机森林 | $O(ntnd)$ | $O(tn)$ | n树数 |
| 神经网络 | $O(nd \cdot \text{epoch})$ | $O(d)$ | 并行友好 |
