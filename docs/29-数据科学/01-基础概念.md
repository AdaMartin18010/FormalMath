## 一、统计学习理论

### 1.1 学习问题的形式化

**定义 1.1** (统计学习框架)
给定：

- 输入空间 $\mathcal{X}$
- 输出空间 $\mathcal{Y}$
- 联合概率分布 $P(X, Y)$
- 假设空间 $\mathcal{H}$（函数集合 $f: \mathcal{X} \to \mathcal{Y}$）

目标：从训练样本 $S = \{(x_i, y_i)\}_{i=1}^n$ 学习函数 $f \in \mathcal{H}$ 使得期望风险最小：

$$R(f) = \mathbb{E}_{(X,Y) \sim P}[L(Y, f(X))]$$

其中 $L$ 是损失函数。

**经验风险**: $\hat{R}(f) = \frac{1}{n}\sum_{i=1}^n L(y_i, f(x_i))$

**经验风险最小化(ERM)**: $\hat{f} = \arg\min_{f \in \mathcal{H}} \hat{R}(f)$

### 1.2 泛化与过拟合

**定义 1.2** (泛化误差)
泛化误差 = 期望风险 - 经验风险：

$$R(f) - \hat{R}(f)$$

**过拟合**: 当假设空间过大时，ERM可能导致训练误差小但测试误差大。

**正则化**: 在目标函数中加入复杂度惩罚：

$$\hat{f} = \arg\min_{f \in \mathcal{H}} \hat{R}(f) + \lambda \Omega(f)$$

其中 $\Omega(f)$ 是复杂度度量（如参数范数）。

### 1.3 偏差-方差分解

**定理 1.1** (偏差-方差分解)
对平方损失，学习算法的期望预测误差可分解为：

$$\mathbb{E}[(Y - \hat{f}(X))^2] = \text{Bias}^2(\hat{f}) + \text{Var}(\hat{f}) + \sigma^2$$

其中：

- **偏差**: $\text{Bias}(\hat{f}) = \mathbb{E}[\hat{f}(X)] - f^*(X)$（模型欠拟合程度）
- **方差**: $\text{Var}(\hat{f}) = \mathbb{E}[(\hat{f}(X) - \mathbb{E}[\hat{f}(X)])^2]$（模型敏感度）
- **不可约误差** $\sigma^2$: 数据噪声

**权衡**: 复杂模型偏差小但方差大，简单模型方差小但偏差大。

### 1.4 模型选择

**交叉验证**: 将数据分为训练集和验证集，评估泛化性能。

**k折交叉验证**: 数据分为k份，轮流用k-1份训练、1份验证。

**AIC/BIC准则**:

- AIC = $2k - 2\ln(\hat{L})$
- BIC = $k\ln(n) - 2\ln(\hat{L})$

其中 $k$ 是参数个数，$\hat{L}$ 是最大似然。

## 二、降维方法

### 2.1 主成分分析(PCA)

**定义 2.1** (PCA)
给定中心化数据矩阵 $X \in \mathbb{R}^{n \times d}$，寻找低维表示 $Z = XW$，$W \in \mathbb{R}^{d \times k}$。

**优化目标**: 最大化投影方差或最小化重构误差

$$\max_{W^TW = I_k} \text{Tr}(W^T X^T X W)$$

**解**: $W$ 的列是 $X^T X$（协方差矩阵）的前 $k$ 个特征向量。

**奇异值分解(SVD)**:

$$X = U\Sigma V^T$$

主成分：$V$ 的前 $k$ 列。

### 2.2 核PCA

**定义 2.2** (核PCA)
通过核函数 $K(x, x') = \langle \phi(x), \phi(x') \rangle$ 隐式映射到高维空间。

步骤：

1. 计算核矩阵 $K_{ij} = K(x_i, x_j)$
2. 中心化核矩阵
3. 对 $K$ 特征分解
4. 投影到新特征空间

### 2.3 流形学习

**定义 2.3** (流形假设)
高维数据位于低维流形上。

**Isomap**: 保持测地距离

**LLE (Locally Linear Embedding)**: 保持局部线性关系

**t-SNE**: 保持局部邻域结构，适合可视化

### 2.4 随机投影

**Johnson-Lindenstrauss引理**: 高维数据可投影到低维空间，同时保持点间距离。

**随机投影矩阵**: $R \in \mathbb{R}^{d \times k}$，元素随机采样。

投影：$Y = XR/\sqrt{k}$

## 三、深度学习数学

### 3.1 神经网络基础

**定义 3.1** (前馈神经网络)

$$f(x) = W_L \sigma(W_{L-1} \sigma(\cdots \sigma(W_1 x + b_1) \cdots) + b_{L-1}) + b_L$$

其中：

- $W_l$: 第 $l$ 层权重矩阵
- $b_l$: 偏置向量
- $\sigma$: 激活函数（ReLU、sigmoid、tanh）

### 3.2 反向传播

**链式法则应用**:

损失函数 $L$ 对参数 $\theta$ 的梯度：

$$\frac{\partial L}{\partial \theta} = \frac{\partial L}{\partial z_L} \frac{\partial z_L}{\partial z_{L-1}} \cdots \frac{\partial z_1}{\partial \theta}$$

**计算图**: 自动微分的基础

### 3.3 优化算法

**梯度下降**: $\theta_{t+1} = \theta_t - \eta \nabla L(\theta_t)$

**随机梯度下降(SGD)**: 用小批量数据估计梯度

**动量法**:
$$v_{t+1} = \beta v_t + (1-\beta)\nabla L(\theta_t)$$
$$\theta_{t+1} = \theta_t - \eta v_{t+1}$$

**Adam**: 自适应学习率，结合动量和二阶矩估计

### 3.4 正则化技术

**L2正则化(权重衰减)**: $L + \lambda \|\theta\|^2$

**Dropout**: 训练时随机置零神经元

**批归一化**: 标准化层输入

**早停**: 根据验证集性能停止训练

## 总结

本章节介绍了数据科学的基础概念：

- **统计学习理论**: 学习框架、泛化、偏差-方差分解、模型选择
- **降维方法**: PCA、核PCA、流形学习、随机投影
- **深度学习数学**: 神经网络结构、反向传播、优化算法、正则化

这些概念构成了现代数据科学的理论基础。
