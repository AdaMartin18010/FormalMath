# 导出代数几何

## 题目

**(a)** 定义导出概形并讨论其与经典概形的关系。

**(b)** 证明拟凝聚层复形范畴的导出性质。

**(c)** 讨论 Donaldson-Thomas 理论与 PT 稳定对的联系。

## 解答

### (a) 导出概形

**动机**：经典代数几何中，相交理论、形变理论需要更高阶的信息。导出几何将 "nilpotent" 推广到 "高阶同伦"。

**Simplicial 交换环**：函子 $A: \Delta^{\mathrm{op}} \to \mathbf{CRing}$，满足 Kan 条件（适当的 horn filling）。

等价：dg 交换代数（特征零）或 $E_\infty$-环谱。

**导出概形**：$X = (X^{\mathrm{cl}}, \mathcal{O}_X)$，其中：

- $X^{\mathrm{cl}}$ 是经典拓扑空间（或 $\infty$-topos）
- $\mathcal{O}_X$ 是 $X^{\mathrm{cl}}$ 上的 simplicial 交换环层（或 structured sheaf of $E_\infty$-环）
- $\pi_0(\mathcal{O}_X) = \mathcal{O}_{X^{\mathrm{cl}}}$

**与经典概形的关系**：

导出概形 $X$ 有 **经典截断** $t_0(X) = (X^{\mathrm{cl}}, \pi_0(\mathcal{O}_X))$，是经典概形。

**例**：两个子概形 $Y, Z \subset X$ 的导出交：

$$Y \times_X^{\mathrm{derived}} Z = (Y \times_X Z, \mathcal{O}_Y \otimes_{\mathcal{O}_X}^{\mathbf{L}} \mathcal{O}_Z)$$

经典交的维数可能过大，导出交的虚拟维数正确。

**Lurie 的谱代数几何**：用 $E_\infty$-环谱代替交换环，更一般的导出几何。

---

### (b) 拟凝聚层复形

**定义**：导出概形 $X$ 上的拟凝聚层是 $dg$-$\mathcal{O}_X$-模（或谱模），其同调层是经典拟凝聚层。

**范畴**：$\operatorname{QCoh}(X)$ 是稳定 $\infty$-范畴（dg-范畴的增强）。

**性质**：

1. **$t$-结构**：$\operatorname{QCoh}(X)^{\heartsuit} = \operatorname{Coh}(t_0(X))$（经典 heart）
2. **推出/拉回**：导出推出 $f_!$、导出拉回 $f^*$ 满足通常的伴随关系
3. **Tensor 积**：$\otimes^{\mathbf{L}}_{\mathcal{O}_X}$ 是导出张量积

**Toën-Vezzosi 定理**：导出 Artin 堆的拟凝聚层范畴有 compact generator，从而等价于某 dg-代数的模范畴。

**应用**：
- **矩阵因子化**：$W: X \to \mathbb{A}^1$ 的奇异纤维由 $W^{-1}(0)$ 的导出范畴描述
- ** Landau-Ginzburg 模型**：B-model 的导出几何实现

### (c) Donaldson-Thomas 理论

**背景**：Calabi-Yau 3-fold $X$ 上的枚举几何。

**经典 DT 不变量**：对稳定凝聚层 $E$（Gieseker 稳定），模空间 $\mathcal{M}$。

因 $X$ 是 CY3，$\operatorname{Ext}^i(E, E)$ 有 Serre 对偶：$\operatorname{Ext}^i \cong \operatorname{Ext}^{3-i}$。

故 $T_{[E]}\mathcal{M} = \operatorname{Ext}^1(E, E)$，$\operatorname{Ob}_{[E]} = \operatorname{Ext}^2(E, E) \cong \operatorname{Ext}^1(E, E)^*$。

**对称 obstruction theory**：$\operatorname{Ob} = T^*$，故虚拟基本类有特别性质。

**Behrend 定理**：$\deg [\mathcal{M}]^{\mathrm{vir}} = \chi(\mathcal{M}, \nu_\mathcal{M})$，其中 $\nu_\mathcal{M}$ 是 Behrend 函数（由 Milnor 纤维的 Euler 特征定义）。

**PT 稳定对（Pandharipande-Thomas, 2009）**：

对纯 1 维层 $F$ 和截面 $s: \mathcal{O}_X \to F$，$(F, s)$ 是 **PT 稳定对**，若：

1. $F$ 是纯 1 维（无 0 维子层）
2. $\operatorname{coker}(s)$ 是 0 维

**模空间**：$P_n(X, \beta)$，参数化 $\chi(F) = n$，$[F] = \beta$ 的 PT 稳定对。

**MNOP 猜想**：

$$Z_{DT}(q, Q) = Z_{PT}(q, Q) \cdot Z_{0}(q)$$

其中 $Z_{DT}$ 是 DT 配分函数，$Z_{PT}$ 是 PT 配分函数。

**证明**：Bridgeland 通过 Hall 代数方法和 wall-crossing 证明（对 Calabi-Yau 3-fold）。

## 常见错误

- **导出概形的截断**：$t_0(X)$ 丢失同伦信息，但相交数在导出层次正确。
- **$\infty$-范畴的技术**：导出几何需在 $(\infty, 1)$-范畴中工作，经典范畴论不足。
- **DT vs PT 的区别**：DT 计数层，PT 计数带截面的层，两者通过 wall-crossing 联系。
