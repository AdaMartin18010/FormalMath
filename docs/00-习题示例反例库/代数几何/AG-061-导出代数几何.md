# 导出代数几何

## 题目

**(a)** 定义导出概形并讨论几何。

**(b)** 证明拟凝聚层复形的范畴。

**(c)** 讨论 Donaldson-Thomas 理论。

## 解答

### (a) 导出概形

X = (X^{cl}, O_X)，O_X 是 simplicial 交换环层。

π₀(O_X) = O_{X^{cl}}。

### (b) 拟凝聚层

QCoh(X)：复形层，上同调拟凝聚。

Toën-Vezzosi、Lurie 的导出代数几何。

### (c) DT 理论

X 光滑射影三维簇，稳定层 E 的模空间 M。

虚拟基本类 [M]^{vir}，DT 不变量 = ∫_{[M]^{vir}} 1。

MNOP 猜想：DT = PT（稳定对）。
