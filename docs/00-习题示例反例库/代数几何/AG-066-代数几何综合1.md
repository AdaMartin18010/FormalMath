# 代数几何综合专题 I

## 题目

**(a)** 综合模空间、导出范畴和算术几何的方法。

**(b)** 讨论 Langlands 纲领与代数几何的联系。

**(c)** 证明 Hodge 理论和 motive 的关系。

## 解答

### (a) 综合方法

模空间的导出范畴（Arinkin、Bezrukavnikov）。

算术应用：p-进 Hodge 理论中的比较定理。

### (b) Langlands 纲领

函数域 Langlands：l-adic 层 ↔ 自守形式。

几何 Langlands：D-模 ↔ 局部系统。

### (c) Hodge 与 Motive

Hodge 猜想：H^{p,p} ∩ H^{2p} 是 motive 的 realization。

混合 Hodge 结构 ↔ 混合 motive。
