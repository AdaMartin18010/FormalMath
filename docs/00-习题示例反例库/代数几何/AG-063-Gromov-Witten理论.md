# Gromov-Witten 理论

## 题目

**(a)** 证明 WDVV 方程（Witten-Dijkgraaf-Verlinde-Verlinde）。

**(b)** 讨论 Frobenius 流形结构。

**(c)** 证明 Virasoro 约束。

## 解答

### (a) WDVV 方程

量子积的结合性：
$$\sum_e \Phi_{abe}g^{ef}\Phi_{fcd} = \sum_e \Phi_{ace}g^{ef}\Phi_{fbd}$$

Gromov-Witten 势的混合偏导。

### (b) Frobenius 流形

流形 M，切丛上 Frobenius 代数结构。

量子上同圆：M = H^*(X)，量子积。

### (c) Virasoro 约束

L_n（n ≥ -1）算子作用在 GW 生成函数上为零。

Eguchi-Hori-Xiong，Konstevich 证明 P¹ 情形。
