# 算术几何基础

## 题目

**(a)** 证明 Hasse 原理的陈述和失效例子。

**(b)** 讨论局部-整体原理的 Brauer-Manin 阻碍。

**(c)** 证明弱逼近和强逼近。

## 解答

### (a) Hasse 原理

Q 上齐次方程：有 ℚ_p 解 ∀p 且有 ℝ 解 ⟹ 有 ℚ 解？

Selmer 的 3x³ = 4y³ + 5 是反例。

### (b) Brauer-Manin 阻碍

Manin：Brauer 群给出有理点集的闭包约束。

若 X(ℚ)^{Br} = ∅，则 X(ℚ) = ∅。

### (c) 弱逼近

弱逼近：X(ℚ) 在 X(A_ℚ) 中稠密。

强逼近：考虑无穷远点。
