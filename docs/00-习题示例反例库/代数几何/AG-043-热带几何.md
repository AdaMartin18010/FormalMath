# 热带几何

## 题目

**(a)** 定义热带半环 $(\mathbb{R} \cup \{\infty\}, \min, +)$。

**(b)** 证明Maslov去量子化。

**(c)** 计算代数簇的热带化。

---

## 解答

### (a) 热带半环

#### 定义

**热带半环**（tropical semiring）$\mathbb{T} = (\mathbb{R} \cup \{\infty\}, \oplus, \otimes)$ 定义为：

- **热带加法**：$a \oplus b = \min(a, b)$
- **热带乘法**：$a \otimes b = a + b$
- **加法单位元**：$\infty$（因为 $\min(a, \infty) = a$）
- **乘法单位元**：$0$（因为 $a + 0 = a$）

#### 半环性质验证

**加法交换律**：$a \oplus b = \min(a,b) = \min(b,a) = b \oplus a$ $\\checkmark$

**加法结合律**：$\min(a, \min(b,c)) = \min(a,b,c) = \min(\min(a,b), c)$ $\\checkmark$

**乘法交换律**：$a \otimes b = a + b = b + a = b \otimes a$ $\\checkmark$

**分配律**：$a \otimes (b \oplus c) = a + \min(b,c) = \min(a+b, a+c) = (a \otimes b) \oplus (a \otimes c)$ $\\checkmark$

**注意**：$\mathbb{T}$ 是**半环**而非环——加法没有逆元（不存在 $b$ 使得 $\min(a,b) = \infty$ 除非 $a = \infty$）。

#### 与max-plus半环的关系

有时也使用**max-plus半环**：$a \oplus' b = \max(a,b)$，$a \otimes' b = a + b$。

两者通过映射 $a \mapsto -a$ 同构（$\min(a,b) \mapsto -\max(-a,-b)$）。

#### 热带多项式

热带多项式：
$$f(x_1, \ldots, x_n) = \bigoplus_{\alpha \in A} c_\alpha \otimes x_1^{\otimes \alpha_1} \otimes \cdots \otimes x_n^{\otimes \alpha_n}$$

展开为经典记号：
$$f(x) = \min_{\alpha \in A} (c_\alpha + \alpha \cdot x)$$

即**仿射线性函数的逐点最小值**。

**例子**：$f(x) = 1 \otimes x^{\otimes 2} \oplus 2 \otimes x \oplus 3$（热带意义下）
$$= \min(1 + 2x, 2 + x, 3)$$

在 $\mathbb{R}$ 上这是分段线性凹函数（若用min；若用max则是凸函数）。

### (b) Maslov去量子化

#### 经典背景

量子力学中，物理可观测量由非交换算子代数描述。当Planck常数 $\hbar \to 0$ 时，量子力学退化为经典力学（对应交换代数）。

Maslov的去量子化是这一极限的代数几何版本。

#### 非Archimedean绝对值

设 $K$ 是代数闭非Archimedean域（如 $p$-adic数域的代数闭包 $\mathbb{C}_p$，或Puiseux级数域 $\mathbb{C}\{\{t\}\}$）。

**赋值**（valuation）$\text{val}: K^* \to \mathbb{R}$ 满足：

- $\text{val}(ab) = \text{val}(a) + \text{val}(b)$
- $\text{val}(a + b) \geq \min(\text{val}(a), \text{val}(b))$（非Archimedean三角不等式）

对 $K = \mathbb{C}\{\{t\}\}$，$\text{val}(\sum c_i t^{r_i}) = \min\{r_i : c_i \neq 0\}$。

#### Maslov去量子化映射

对 $a, b \in K^*$，考虑其赋值的极限行为：

$$\text{val}(a + b) = \min(\text{val}(a), \text{val}(b)) \quad \text{（若 } \text{val}(a) \neq \text{val}(b)\text{）}$$
$$\text{val}(ab) = \text{val}(a) + \text{val}(b)$$

这正是热带半环的运算！

**形式化表述**：

设 $K$ 带非Archimedean绝对值 $|\cdot|$，定义**对数极限集**：
$$\text{Log}_t: (K^*)^n \to \mathbb{R}^n, \quad (z_1, \ldots, z_n) \mapsto (\log_t |z_1|, \ldots, \log_t |z_n|)$$

当 $t \to \infty$ 时，$(K^*, \cdot)$ 的代数结构"退化为" $(\mathbb{R}, \min, +)$。

**Maslov原理**：代数簇 $V \subseteq (K^*)^n$ 的图像在对数极限下收敛于热带簇 $\text{Trop}(V) \subseteq \mathbb{R}^n$。

#### 与amoeba的关系

对代数簇 $V \subseteq (\mathbb{C}^*)^n$，其**amoeba**：
$$\mathcal{A}(V) = \text{Log}(V) = \{(\log|z_1|, \ldots, \log|z_n|) : z \in V\} \subseteq \mathbb{R}^n$$

热带簇是amoeba的"骨架"（spine）——非Archimedean类比下更精确的概念。

### (c) 代数簇的热带化

#### 定义

设 $V \subseteq (K^*)^n$ 是代数簇，$I(V) \subseteq K[x_1^{\pm 1}, \ldots, x_n^{\pm 1}]$ 是其理想。

**Kapranov热带化**：
$$\text{Trop}(V) = \overline{\{(\text{val}(z_1), \ldots, \text{val}(z_n)) : z \in V(K)\}} \subseteq \mathbb{R}^n$$

**初等热带化**（利用多项式）：
对 $f = \sum c_\alpha x^\alpha \in K[x^{\pm 1}]$，其**热带化**：
$$\text{trop}(f)(w) = \min_\alpha (\text{val}(c_\alpha) + \alpha \cdot w)$$

则
$$\text{Trop}(V) = \bigcap_{f \in I(V)} \{w \in \mathbb{R}^n : \text{trop}(f)(w) \text{ 在某点取到最小值至少两次}\}$$

即：$w \in \text{Trop}(V)$ 当且仅当对每个 $f \in I(V)$，$\text{trop}(f)(w)$ 的最小值由至少两项同时达到。

#### 计算示例

**例1：直线**

设 $V \subseteq (K^*)^2$ 由 $f(x,y) = x + y + 1 = 0$ 定义。

$$\text{trop}(f)(w_1, w_2) = \min(w_1, w_2, 0)$$

$\text{trop}(f)$ 取最小值至少两次的区域：

- $w_1 = w_2 \leq 0$：直线 $w_1 = w_2$，$w_1 \leq 0$
- $w_1 = 0 \leq w_2$：射线 $w_1 = 0$，$w_2 \geq 0$
- $w_2 = 0 \leq w_1$：射线 $w_2 = 0$，$w_1 \geq 0$

这是 $\mathbb{R}^2$ 中从原点出发的三条射线，夹角各 $120^\circ$——一条"热带直线"。

**例2：平面曲线**

设 $f(x,y) = Axy + Bx + Cy + D = 0$（一般二次曲线）。

$$\text{trop}(f) = \min(\text{val}(A) + w_1 + w_2, \text{val}(B) + w_1, \text{val}(C) + w_2, \text{val}(D))$$

Newton多边形是 $(1,1), (1,0), (0,1), (0,0)$ 的凸包（正方形）。

热带曲线 $\text{Trop}(V)$ 是分段线性图，其边方向由Newton多边形的边法向量决定。

**一般规则**：

- 热带曲线的边 $ot$ Newton多边形的边
- 顶点对应Newton多边形的对偶剖分
- 边的重数 = 对应Newton多边形边的格点长度

#### 基本定理

**结构定理**（Bieri-Groves）：$\text{Trop}(V)$ 是有限个有理多面体的并，构成纯 $\dim(V)$-维多面复形。

**完备性**（Kapranov）：若 $K$ 代数闭且赋值满射到 $\mathbb{R}$，则 $\text{Trop}(V) \neq \emptyset$ 当且仅当 $V \neq \emptyset$。

**度保持**：若 $V$ 是 $d$ 次曲线，则 $\text{Trop}(V)$ 满足热带Bézout定理：两条次数为 $d_1, d_2$ 的热带曲线的交点数为 $d_1 d_2$（计重数）。

#### 应用

1. **计数几何**：Kontsevich通过热带方法计算有理曲线数目
2. **组合优化**：线性规划的对偶性 = 热带行列式的消失
3. **生物学**：系统发育树的距离 = 热带Grassmannian的点
4. **经济学**：拍卖理论中的最优分配 = 热带线性方程组的解
