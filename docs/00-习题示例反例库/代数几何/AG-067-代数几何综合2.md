# 代数几何综合专题 II

## 题目

**(a)** 综合双有理几何与复几何的方法。

**(b)** 讨论 Hodge 理论的现代发展。

**(c)** 展望代数几何的前沿问题。

## 解答

### (a) 双有理与复几何

Kähler-Ricci 流与 MMP。

极小模型存在性的分析方法（复维 2）。

### (b) 现代 Hodge 理论

混合 Hodge 模（Saito）。

非交换 Hodge 理论（Kontsevich-Soibelman）。

### (c) 前沿问题

- 标准猜想和 motive 理论
- 几何 Langlands 纲领
- p-进 Langlands 和 perfectoid 空间
- 高维簇的算术
- 导出代数几何的应用
