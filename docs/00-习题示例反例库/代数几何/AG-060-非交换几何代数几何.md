# 非交换代数几何

## 题目

**(a)** 定义非交换射影概形 Artin-Zhang。

**(b)** 讨论 Sklyanin 代数和量子 P²。

**(c)** 证明非交换曲面的分类。

## 解答

### (a) 非交换 Proj

分次代数 A，范畴 qgr(A) = gr(A)/tors(A)。

(A, O(1)) 是非交换射影概形。

### (b) Sklyanin 代数

生成元 x,y,z，关系：
ayz + bzy + cx² = 0（循环）

对应椭圆曲线上的点，非交换 P²。

### (c) 非交换曲面

Artin-Van den Bergh：非交换 P² 的分类。

非交换直纹面、del Pezzo 曲面。
