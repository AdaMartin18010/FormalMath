# 导出范畴基础

## 题目

**(a)** 定义三角范畴和 t-结构。

**(b)** 证明 $D^b(Coh(X))$ 的 hearts。

**(c)** 讨论导出范畴的粘合。

---

## 解答

### (a) 三角范畴与t-结构

#### 三角范畴

已在ALG-153中详细讨论。核心要点：

- 对象 + 平移 $[1]$ + 特异三角形
- TR1-TR4公理
- 导出范畴 $D(\mathcal{A})$ 是三角范畴

#### t-结构

**定义**：$(\mathcal{D}^{\leq 0}, \mathcal{D}^{\geq 0})$ 满足：
1. $\mathcal{D}^{\leq 0}[1] \subseteq \mathcal{D}^{\leq 0}$
2. $\text{Hom}(\mathcal{D}^{\leq 0}, \mathcal{D}^{\geq 1}) = 0$
3. 任意 $X$ 有三角形 $A \to X \to B \to A[1]$，$A \in \mathcal{D}^{\leq 0}$，$B \in \mathcal{D}^{\geq 1}$

**心** $\mathcal{A} = \mathcal{D}^{\leq 0} \cap \mathcal{D}^{\geq 0}$ 是Abel范畴。

---

### (b) $D^b(Coh(X))$ 的hearts

#### 标准t-结构

对光滑射影簇 $X$，$D^b(Coh(X))$ 有标准t-结构：
- $\mathcal{D}^{\leq 0} = \{K : H^i(K) = 0, i > 0\}$
- $\mathcal{D}^{\geq 0} = \{K : H^i(K) = 0, i < 0\}$

**心** = $Coh(X)$（凝聚层范畴）。

#### 非标准t-结构

**perverse t-结构**（代数几何）：

对 proper 态射 $f: X \to Y$，定义：
- ${}^p\mathcal{D}^{\leq 0} = \{K : \dim \text{supp } H^i(K) \leq -i\}$
- ${}^p\mathcal{D}^{\geq 0} = \{K : \dim \text{supp } H^i(\mathbb{D}K) \leq -i\}$

**心** = **perverse sheaves**。

#### 倾斜（Tilting）

**倾斜t-结构**：给定**倾斜对象** $T \in D^b(Coh(X))$（$\text{Ext}^i(T,T) = 0$，$i \neq 0$；$T$ 生成 $D^b(Coh(X))$）。

定义：
- $\mathcal{D}^{\leq 0}_T = \{K : \text{Hom}(K, T[i]) = 0, i > 0\}$
- $\mathcal{D}^{\geq 0}_T = \{K : \text{Hom}(T[i], K) = 0, i < 0\}$

**心** = $\text{mod-}\text{End}(T)$。

**例子**：Beilinson的 $T = \bigoplus_{i=0}^n \mathcal{O}(-i)$ 在 $\mathbb{P}^n$ 上。

### (c) 导出范畴的粘合

#### 开闭分解

设 $X$ 是概形，$Z \subseteq X$ 闭，$U = X \setminus Z$ 开。

**粘合**（Recollement）：
$$D(U) \xleftarrow{j^*} D(X) \xleftarrow{i_*} D(Z)$$

其中 $j: U \hookrightarrow X$，$i: Z \hookrightarrow X$。

#### 粘合的公理

1. $i_*$ fully faithful，有左伴随 $i^*$ 和右伴随 $i^!$
2. $j^*$ fully faithful，有左伴随 $j_!$ 和右伴随 $j_*$
3. $j^* i_* = 0$
4. 对 $K \in D(X)$，有 distinguished triangle：
$$j_! j^* K \to K \to i_* i^* K \to j_! j^* K[1]$$

#### 应用

1. **Perverse sheaves的构造**：通过粘合局部perverse sheaves
2. **Mixed Hodge modules**：Saito用粘合构造mixed Hodge modules
3. **Stratified spaces**：按strata粘合导出范畴
