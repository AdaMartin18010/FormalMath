# 导出范畴基础

## 题目

**(a)** 定义三角范畴和 t-结构。

**(b)** 证明 D^b(Coh(X)) 的 hearts。

**(c)** 讨论导出范畴的粘合。

## 解答

### (a) 三角范畴

平移函子 [1]，好三角形 A → B → C → A[1]。

t-结构：D^{≤0}, D^{≥0}，满足公理（平移、正合对、正交）。

### (b) Hearts

heart A = D^{≤0} ∩ D^{≥0} 是 Abel 范畴。

标准 t-结构：heart 是凝聚层。

### (c) 粘合

开闭分解 U → X ← Z，导出范畴粘合：
D(U) ← D(X) ← D(Z)。
