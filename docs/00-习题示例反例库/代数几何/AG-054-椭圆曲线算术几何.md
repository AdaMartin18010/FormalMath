# 椭圆曲线的算术几何

## 题目

**(a)** 证明 Mordell-Weil 定理的下降法。

**(b)** 讨论椭圆曲线的 Galois 表示。

**(c)** 证明 Tate 模上的 Weil 配对。

## 解答

### (a) 下降法

弱 Mordell-Weil：E(ℚ)/nE(ℚ) 有限。

高度下降：若 P ∈ E(ℚ)，则存在 Q，nQ = P，h(Q) < h(P)（近似）。

### (b) Galois 表示

ρ_{E,ℓ}: G_ℚ → GL₂(ℤ_ℓ)。

Serre 开像定理：非 CM 曲线像开（在 GL₂(ℤ̂) 中）。

### (c) Weil 配对

e_n: E[n] × E[n] → μ_n。

交错、Galois 等变、非退化。
