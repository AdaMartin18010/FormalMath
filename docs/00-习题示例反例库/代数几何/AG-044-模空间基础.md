# 模空间基础

## 题目

**(a)** 定义粗模空间和细模空间。

**(b)** 证明 $M_g$（亏格 $g$ 曲线模空间）的存在性。

**(c)** 讨论 Hilbert 概形和 Quot 概形的构造。

## 解答

### (a) 粗模空间与细模空间

**模函子**：设 $\mathcal{F}: (\mathrm{Sch}/S)^{\mathrm{op}} \to \mathrm{Set}$ 是模函子（如参数化曲线的同构类）。

**细模空间**：$M$ 是 $\mathcal{F}$ 的 **细模空间**，若 $M$ 表示 $\mathcal{F}$：

$$\mathcal{F}(T) \cong \operatorname{Hom}(T, M)$$

对所有 $T$ 自然。等价：存在 **万有族** $\mathcal{U} \to M$ 使得任意族 $X \to T$ 由唯一的 $f: T \to M$ 拉回得到。

**粗模空间**：$M$ 是 $\mathcal{F}$ 的 **粗模空间**，若：

1. 存在自然变换 $\Phi: \mathcal{F} \to h_M$
2. $\Phi$ 在代数闭域上双射：$\mathcal{F}(\operatorname{Spec}(k)) \cong M(k)$
3. 对任意 $N$ 和自然变换 $\Psi: \mathcal{F} \to h_N$，存在唯一的 $f: M \to N$ 使 $\Psi = f \circ \Phi$

**区别**：细模空间要求万有族，粗模空间仅分类闭点的同构类。

### (b) $M_g$ 的存在性

**问题**：参数化光滑亏格 $g$ 曲线的同构类。

**Deligne-Mumford 定理（1969）**：存在 **Deligne-Mumford 栈** $\mathcal{M}_g$，其粗模空间 $M_g$ 是不可约拟射影簇。

*证明概要*：

1. **Hilbert 概形嵌入**：利用 $m$-canonical 嵌入 $\omega_C^{\otimes m}: C \hookrightarrow \mathbb{P}^N$（$m \geq 3$）。

2. **Hilbert 多项式**：$P(t) = (2m(g-1))t + (1-g)$。

3. **Chow 变体**：$C$ 在 $\mathbb{P}^N$ 中的 Chow 点满足某些条件。

4. **稳定约化**：Mumford 的 GIT 构造。$\overline{M}_g$（稳定曲线模空间）是射影的。

**维数**：$\dim M_g = 3g - 3$（$g \geq 2$）。

### (c) Hilbert 概形与 Quot 概形

**Hilbert 概形** $\operatorname{Hilb}_P(X)$：参数化 $X$ 中 Hilbert 多项式为 $P$ 的闭子概形。

**Grothendieck 定理**：$X$ 射影时 $\operatorname{Hilb}_P(X)$ 存在且射影。

**应用**：
- 曲线模空间（通过嵌入）
- 曲面模空间
- 稳定映射模空间（Kontsevich）

**Quot 概形** $\operatorname{Quot}_{E/X/P}$：参数化凝聚层 $E$ 的商 $E \to F$，$F$ 的 Hilbert 多项式为 $P$。

**Grothendieck 定理**：$X$ 射影时 Quot 概形存在且射影。

**应用**：
- 向量丛模空间
- Donaldson-Thomas 理论中的稳定层模空间

## 常见错误

- **粗模空间的存在性**：不是所有模函子都有粗模空间（需某种紧性条件）。
- **$M_g$ 的维数**：$g = 0, 1$ 时需特别处理（$M_0 = pt$，$M_1 = \mathbb{A}^1$）。
- **Hilbert 概形的光滑性**：一般 Hilbert 概形奇异，仅在特殊情形（如 $\mathbb{P}^2$ 中的点）光滑。
