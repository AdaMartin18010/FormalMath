# 模空间基础

## 题目

**(a)** 定义粗模空间和细模空间。

**(b)** 证明 M_g（亏格 g 曲线模空间）的存在性。

**(c)** 讨论 Hilbert 概形和 Quot 概形。

## 解答

### (a) 模空间定义

粗模空间：M 表示函子 F: Sch → Set，存在自然变换 F → h_M 在域点上双射。

细模空间：F ≅ h_M，存在万有族。

### (b) M_g 的存在

Deligne-Mumford：M_g 是 Deligne-Mumford 栈。

几何不变量理论构造，或 Teichmüller 理论。

### (c) Hilbert 概形

Hilb_{P}(P^n)：P^n 中具有 Hilbert 多项式 P 的子概形。

Grothendieck：射影概形存在。
