# 算术对偶性

## 题目

**(a)** 证明局部域的 Tate 对偶。

**(b)** 讨论整体域的 Poitou-Tate 对偶。

**(c)** 证明 étale 上同调的算术对偶性。

## 解答

### (a) Tate 对偶

K 局部域，M 有限 G_K-模。

H^i(K, M) × H^{2-i}(K, M^∨) → H²(K, G_m) ≅ ℚ/ℤ。

完美配对。

### (b) Poitou-Tate

K 数域，9 项正合列：
$$0 → H⁰ → P⁰ → H³^∨ → H¹ → P¹ → H²^∨ → H² → P² → H¹^∨ → H³ → 0$$

### (c) Étale 对偶

Milne：X 在有限域上，l-进层。

Ext^i(F, G_m) × H^{3-i}_c(X, F) → ℚ/ℤ。
