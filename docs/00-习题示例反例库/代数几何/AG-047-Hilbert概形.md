# Hilbert 概形深入

## 题目

**(a)** 证明 Hilbert 概形代表 Hilbert 函子。

**(b)** 讨论 Hilbert 概形的切空间。

**(c)** 计算 Hilb^n(A²) 的几何。

## 解答

### (a) 可表性

Grothendieck：对射影概形 X，Hilb_P(X) 存在且射影。

构造：嵌入 X → P^N，子概形 ↔ 饱和理想。

### (b) 切空间

Z ⊂ X，T_{[Z]} Hilb = H⁰(Z, N_{Z/X})。

阻碍在 H¹(Z, N_{Z/X})。

### (c) Hilb^n(A²)

维数 2n，光滑。

Hilbert-Chow 态射：Hilb^n(A²) → Sym^n(A²)。

例外纤维：C²/(1/n)(1,-1) 型 ALE 空间。
