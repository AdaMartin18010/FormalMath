# 完美胚空间

## 题目

**(a)** 定义 perfectoid 代数并证明基本性质。

**(b)** 证明 almost 数学和 almost 纯正合列。

**(c)** 讨论倾斜对应（Tilting equivalence）。

## 解答

### (a) Perfectoid 代数

R：p-完备、一致、p 是伪一致元、Frobenius 满。

特征 0 完美胚代数的例子：ℂ_p 的有限型扩张。

### (b) Almost 数学

固定伪一致元 π，模的 almost 性质。

几乎纯正合：核和余核被 π^{1/p^∞} 零化。

### (c) 倾斜对应

R（特征 0）↔ R^b（特征 p）。

等价：Perf_{R} ≅ Perf_{R^b}。

Scholze：p-进 Hodge 理论的革命性工具。
