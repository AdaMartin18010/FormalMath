# 矩阵因子分解

## 题目

**(a)** 定义超曲面奇点的矩阵因子化。

**(b)** 证明矩阵因子化范畴等价于奇点范畴。

**(c)** 讨论 Orlov 的奇点定理。

## 解答

### (a) 矩阵因子化

W: X → ℂ，奇点集 {dW = 0}。

矩阵因子化：A ⊗ B → B ⊗ A → ... 满足 d² = W·id。

### (b) 等价性

Eisenbud：矩阵因子化 ↔ 极大 Cohen-Macaulay 模。

Orlov：MF(W) ≅ D_{sg}(X_0)，X_0 = W⁻¹(0)。

### (c) Orlov 定理

X 光滑射影，W 超曲面，奇点范畴与 D(X) 的关系：

半直积分解：D(X_0) = ⟨D(X), D_{sg}(X_0)⟩。
