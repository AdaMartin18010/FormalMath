# 层上同调进阶

## 题目

**(a)** 证明Grothendieck消失定理：$X$ 是 $n$ 维Noether拓扑空间，$H^i(X, \mathcal{F}) = 0$ 对 $i > n$。

**(b)** 证明Serre对偶：光滑射影簇 $X$，$H^i(X, \mathcal{F})^\vee \cong H^{n-i}(X, \omega_X \otimes \mathcal{F}^\vee)$。

**(c)** 计算 $H^i(\mathbb{P}^n, \mathcal{O}(k))$。

## 解答

### (a) Grothendieck消失

Godement分解和维数归纳。

### (b) Serre对偶

Ext版本和迹映射。

### (c) 射影空间上同调

Bott公式：$H^i(\mathbb{P}^n, \mathcal{O}(k))$ 的具体计算。
