# 模空间与几何不变量

## 题目

**(a)** 定义GIT商和稳定点。

**(b)** 证明曲线的模空间 $\bar{M}_g$ 的存在性。

**(c)** 讨论稳定约化和紧化。

---

## 解答

### (a) GIT商与稳定点

#### 设置

设 $G$ 是约化代数群，作用在射影簇 $X \subseteq \mathbb{P}^N$ 上，$L$ 是 $G$-线性化的 ample line bundle。

#### 稳定点的定义

**Mumford稳定性**（Geometric Invariant Theory, GIT）：

对 $x \in X$，考虑Hilbert-Mumford权重：对任意1-参数子群 $\lambda: \mathbb{G}_m \to G$：
$$\mu^L(x, \lambda) = -\min\{\text{weight of } \lambda \text{ on } \tilde{x}\}$$

其中 $\tilde{x}$ 是 $x$ 在仿射锥 $\mathbb{A}^{N+1}$ 上的提升。

**定义**：
- **稳定**（Stable）：$\mu^L(x, \lambda) > 0$ 对所有非平凡 $\lambda$
- **半稳定**（Semistable）：$\mu^L(x, \lambda) \geq 0$ 对所有 $\lambda$
- **不稳定**（Unstable）：存在 $\lambda$ 使 $\mu^L(x, \lambda) < 0$
- **严格半稳定**（Strictly semistable）：半稳定但不稳定

**稳定点集** $X^s$ 和**半稳定点集** $X^{ss}$ 都是 $X$ 的 Zariski 开子集。

#### GIT商

**几何商**（Geometric quotient）：$X // G$ 满足：
1. 点 = $G$-轨道
2. 拓扑 = 商拓扑
3. 结构层 = $G$-不变函数层

**定理**（Mumford）：存在射影概形 $X //_L G$ 和开子集 $X^s // G \subseteq X //_L G$ 使得：
- $X^s // G$ 是 $X^s$ 的几何商
- $X //_L G = \text{Proj}\left(\bigoplus_{n \geq 0} H^0(X, L^{\otimes n})^G\right)$ 是 $X^{ss}$ 的"范畴商"
- $X //_L G$ 的点是 $X^{ss}$ 中闭轨道的等价类（在 $X^{ss}$ 中的闭包相交）

**关键观察**：半稳定点的轨道不一定闭。GIT商将轨道闭包相交的点等同。

#### 例子

**例1：$	ext{SL}(2)$ 作用在二元形式**

$G = \text{SL}(2)$ 作用在 $V_d = \text{Sym}^d(\mathbb{C}^2)$（二元 $d$ 次形式）上。

- $d = 3$（三次曲线）：稳定点 = 光滑三次曲线（判别式 $\neq 0$）
- $d = 4$（四次曲线）：稳定点 = 无三重根的四次式

**例2：Grassmannian的Plücker嵌入**

$G = \text{SL}(n)$ 作用在 $Gr(k, n)$ 上。Plücker坐标给出自然的线性化。

所有点都是稳定的（因为 $G$ 在 $Gr(k,n)$ 上传递作用，且稳定子是抛物子群）。

### (b) 曲线模空间 $\bar{M}_g$ 的存在性

#### 构造思路

**步骤1：参数化光滑曲线**

对 $g \geq 2$，考虑Hilbert方案 $\text{Hilb}^{P(m)}(\mathbb{P}^N)$，其中 $P(m) = (2m-1)(g-1)$ 对应 $m$-canonical 嵌入 $\phi_{mK_C}: C \hookrightarrow \mathbb{P}^{(2m-1)(g-1)-1}$。

**定理**（Mumford）：对 $m \geq 3$，$m$-canonical嵌入的 Hilbert 点稳定。

**步骤2：GIT商**

取 $G = \text{PGL}((2m-1)(g-1))$ 作用在 Hilbert 方案上，GIT商给出粗模空间：
$$M_g = \text{Hilb}^{P(m), s} // G$$

$M_g$ 是拟射影簇，参数化光滑 genus $g$ 曲线的同构类。

#### Deligne-Mumford紧化

**稳定曲线**：$(g, n)$-稳定曲线是连通 nodal 曲线，每个有理分量至少有3个特殊点（节点或标记点）。

**定理**（Deligne-Mumford, 1969）：存在紧模空间 $\bar{M}_g$，参数化稳定曲线的同构类。

**性质**：
- $\bar{M}_g$ 是紧的、不可约的 $3g-3$ 维Deligne-Mumford栈（$g \geq 2$）
- $M_g \subseteq \bar{M}_g$ 是稠密开子集
- 边界 $\Delta = \bar{M}_g \setminus M_g$ 是除子，参数化 singular 稳定曲线
- 边界分层：$\Delta = \Delta_0 \cup \Delta_1 \cup \cdots \cup \Delta_{[g/2]}$
  - $\Delta_0$：不可约 nodal 曲线
  - $\Delta_i$（$i > 0$）：两个分量在一点相交，亏格分别为 $i$ 和 $g-i$

#### $\bar{M}_g$ 的几何

**Picard群**（Harer, Arbarello-Cornalba, Faber）：
$$\text{Pic}(\bar{M}_g) \otimes \mathbb{Q} = \mathbb{Q}\lambda \oplus \mathbb{Q}\delta_0 \oplus \cdots \oplus \mathbb{Q}\delta_{[g/2]}$$

- $\lambda = c_1(\mathbb{E})$：Hodge bundle 的陈类
- $\delta_i = [\Delta_i]$：边界除子类

**有效锥与 ample 锥**（Harris-Mumford-Eisenbud, Farkas）：

对 $g \geq 24$，$\bar{M}_g$ 是一般型的（$K_{\bar{M}_g}$ big）。这是通过计算 $\bar{M}_g$ 上的 divisor class 并证明 $K_{\bar{M}_g}$ 在有效锥内部来实现的。

**Harris-Mumford定理**：$\bar{M}_g$ 对 $g \geq 24$ 是一般型的。

### (c) 稳定约化与紧化

#### 稳定约化定理

**定理**（Deligne-Mumford）：设 $C \to \Delta^*$ 是 punctured disk 上的光滑曲线族，则存在有限 base change $\Delta^* \to \Delta^*$ 和填充（filling）使得：

$$\xymatrix{C' \ar[r] \ar[d] & \bar{C} \ar[d] \\ \Delta^* \ar[r] & \Delta}$$

其中 $\bar{C} \to \Delta$ 是 nodal 曲线族，中心纤维是稳定曲线。

**意义**：任何退化（degeneration）都可以通过有限 base change 稳定化。

**证明要点**：
1. 先 semi-stable 约化（节点交叉）
2. 然后收缩不稳定的有理分量（Castelnuovo收缩）
3. 得到 stable 填充

#### 其他紧化

| 紧化 | 参数化对象 | 边界性质 | 优点 | 缺点 |
|------|------------|----------|------|------|
| **DM紧化** $\bar{M}_g$ | 稳定曲线 | nodal | 典范的、模解释清晰 | 高维时 singular |
| **Voronoi紧化** | 主极化Abel簇 | 半稳定约化 | 光滑 | 不是模空间 |
| **Schottky紧化** | 曲线+theta特征 | theta null | 与theta函数联系 | 不完全 |
| **Hassett紧化** | 加权稳定曲线 | 带权重点 | 更灵活的标记点 | 需要选择权重 |
| **Alexeev紧化** | 稳定对 $(X, D)$ | semi-log-canonical | 对数推广 | 非常复杂 |

#### 高维推广

对曲面（$K3$、Abel 曲面等），模空间的紧化更复杂：
- **KSBA紧化**（Kollár-Shepherd-Barron-Alexeev）：稳定对 $(X, D)$ 的模空间
- **GIT紧化**：依赖于具体嵌入的选择
- **Torelli紧化**：通过周期映射的闭包

**极化选择问题**：高维模空间的紧化通常依赖于极化的选择（ample line bundle），不同的选择给出不同的 birational 模型（flip、divisorial contraction 的关系）。
