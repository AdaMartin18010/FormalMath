# Gromov-Witten理论

## 题目

**(a)** 定义稳定映射模空间 $\bar{M}_{g,n}(X,\beta)$。

**(b)** 证明虚拟基本类存在。

**(c)** 计算镜像对称的预测。

---

## 解答

### (a) 稳定映射模空间

#### 稳定曲线

**定义**：$(g, n)$-**稳定曲线**是连通 nodal 曲线 $C$（仅允许普通双重点奇点），满足：
- $g(C) = g$（算术亏格）
- $C$ 有 $n$ 个标记点 $p_1, \ldots, p_n \in C$（互不相同，不在节点上）
- **稳定性**：$\text{Aut}(C, p_1, \ldots, p_n)$ 有限，等价于每条有理分量至少有3个特殊点（标记点或节点）

**Deligne-Mumford模空间**：$\bar{M}_{g,n}$ 是 $(g,n)$-稳定曲线的粗模空间（$2g - 2 + n > 0$ 时紧）。

$$\dim \bar{M}_{g,n} = 3g - 3 + n$$

#### 稳定映射

**定义**：设 $X$ 是光滑射影簇，$\beta \in H_2(X, \mathbb{Z})$。一个**稳定映射**是态射 $f: (C, p_1, \ldots, p_n) \to X$ 使得：
- $(C, p_1, \ldots, p_n)$ 是稳定曲线（可能不连通但每个分量稳定）
- $f_*[C] = \beta$
- **稳定性**：$f$ 的非平凡形变有限。等价：对 $C$ 的每个不可约有理分量 $C_i$，若 $f|_{C_i}$ 常值，则 $C_i$ 至少有3个特殊点

#### 模空间 $\bar{M}_{g,n}(X, \beta)$

**定理**（Kontsevich）：$\bar{M}_{g,n}(X, \beta)$ 是Deligne-Mumford栈，紧的（proper），若 $X$ 凸（如 $\mathbb{P}^r$、旗簇、齐性空间）则是轨道空间。

**期望维数**（Expected dimension）：
$$\text{vir.dim} = (\dim X - 3)(1 - g) + \int_\beta c_1(T_X) + n$$

**推导**：由 deformation theory，稳定映射的形变由 $H^0(C, f^*T_X)$ 参数化，阻碍由 $H^1(C, f^*T_X)$ 参数化。由Riemann-Roch：
$$\chi(f^*T_X) = \text{rank}(T_X) \cdot \chi(\mathcal{O}_C) + \int_C c_1(f^*T_X) = (\dim X)(1 - g) + \int_\beta c_1(T_X)$$

实际维数 = $h^0 - h^1 = \chi + 3g - 3 + n$（$3g-3+n$ 来自曲线的模），整理得期望维数公式。

#### 评估映射与遗忘映射

**评估映射**：
$$\text{ev}_i: \bar{M}_{g,n}(X, \beta) \to X, \quad [f: C \to X, p_1, \ldots, p_n] \mapsto f(p_i)$$

**遗忘映射**：
$$\pi: \bar{M}_{g,n}(X, \beta) \to \bar{M}_{g,n}$$
（当 $2g - 2 + n > 0$ 时，遗忘映射和稳定化）

### (b) 虚拟基本类

#### 问题

$\bar{M}_{g,n}(X, \beta)$ 的实际维数可能大于期望维数（存在"多余维数"，excess dimension）。例如：
- $g = 0$，$X = \mathbb{P}^3$，$\beta = d[\text{line}]$，$n = 0$：期望维数 = 0，但实际可能有正维数分量（常值映射到点）
- 高亏格时 $h^1(f^*T_X) \neq 0$ 导致 obstructed

#### 虚拟基本类的构造

**方法1：Kontsevich的代数构造**

设 $X$ 是凸簇（$H^1(\mathbb{P}^1, f^*T_X) = 0$ 对所有 $f: \mathbb{P}^1 \to X$）。此时 $\bar{M}_{0,n}(X, \beta)$ 是轨道空间，期望维数 = 实际维数，$[\bar{M}_{0,n}(X, \beta)]^{vir} = [\bar{M}_{0,n}(X, \beta)]$（通常基本类）。

**方法2：Li-Tian / Behrend-Fantechi 的完全相交构造**

一般情形：由 perfect obstruction theory（POT）定义虚拟基本类。

**Perfect Obstruction Theory**：在 $\bar{M}_{g,n}(X, \beta)$ 上，存在 perfect complex $\mathcal{E}^\bullet$ 使得：
- $h^0(\mathcal{E}^\bullet) = $ 形变层
- $h^1(\mathcal{E}^\bullet) = $ 阻碍层
- $\mathcal{E}^\bullet$ 局部拟同构于2项复形 $[\mathcal{E}^0 \to \mathcal{E}^1]$（向量丛的态射）

**虚拟维数**：$\text{rank}(\mathcal{E}^0) - \text{rank}(\mathcal{E}^1) = $ 期望维数

**虚拟基本类**：
$$[\bar{M}_{g,n}(X, \beta)]^{vir} = 0_{\mathcal{E}^1}^!([\mathcal{C}]) \in A_{\text{vir.dim}}(\bar{M}_{g,n}(X, \beta))$$

其中 $\mathcal{C} \subseteq \mathcal{E}^1$ 是intrinsic normal cone，$0^!$ 是 refined Gysin pullback。

**Behrend-Fantechi定理**：虚拟基本类满足：
1. 形变不变性
2. 与通常基本类兼容（无阻碍时一致）
3. 满足各种 functoriality（pushforward、pullback等）

#### Gromov-Witten不变量

**定义**：对 $\gamma_1, \ldots, \gamma_n \in H^*(X)$，
$$\langle \gamma_1, \ldots, \gamma_n \rangle_{g, n, \beta}^X = \int_{[\bar{M}_{g,n}(X, \beta)]^{vir}} \text{ev}_1^*\gamma_1 \smile \cdots \smile \text{ev}_n^*\gamma_n$$

**性质**：
- 满足WDVV方程（associativity）
- 满足弦方程和膨胀方程（String and Dilaton equations）
- 高亏格满足Virasoro约束（猜想，部分证明）

### (c) 镜像对称

#### 陈述

**镜像对称**（Mirror Symmetry）预测：Calabi-Yau三维fold $X$ 的复几何与"镜像" $\check{X}$ 的辛几何对偶。

具体地，$X$ 上的**A-模型**（辛几何）Gromov-Witten势函数等于 $\check{X}$ 上的**B-模型**（复几何）周期积分：
$$\mathcal{F}^A_X(q) = \mathcal{F}^B_{\check{X}}(q)$$

#### 五边形Calabi-Yau的预测

**Candelas-de la Ossa-Green-Parkes计算**（1991）：

设 $X$ 是Quintic threefold（$\mathbb{P}^4$ 中一般5次超曲面）。

**A-模型**：
- 度 $d$ 的有理曲线数目 $n_d$（Gromov-Witten不变量经多重覆盖修正）
- 生成函数：$K(q) = 5 + \sum_{d=1}^\infty n_d d^3 \frac{q^d}{1 - q^d}$

**B-模型**（镜像 $\check{X}$）：
- Picard-Fuchs方程（描述周期）：
$$\mathcal{D} \varpi = 0, \quad \mathcal{D} = \theta^4 - 5z(5\theta + 1)(5\theta + 2)(5\theta + 3)(5\theta + 4)$$
其中 $\theta = z \frac{d}{dz}$

- 在 $z = 0$ 附近，解有形式：
$$\varpi_0(z) = 1 + \sum_{n=1}^\infty \frac{(5n)!}{(n!)^5} z^n$$

- 镜像映射：$q = \exp(\varpi_1 / \varpi_0)$，其中 $\varpi_1$ 是对数解

- Yukawa耦合：$K(q) = \frac{5}{(1 - 5^5 z) \varpi_0^2(z)} \left(\frac{q}{z} \frac{dz}{dq}\right)^3$

**预测值**（通过 $q$-展开比较）：

| $d$ | $n_d$ |
|-----|-------|
| 1 | 2875 |
| 2 | 609250 |
| 3 | 317206375 |
| 4 | 242467530000 |
| 5 | 229305888887625 |

**验证**：
- $n_1 = 2875$：古典结果（$	ext{Gr}(2,5)$ 中直线数）
- $n_2 = 609250$：Katz（1986）用退化技术计算
- 更高 $n_d$：最初无直接证明，是镜像对称最强证据
- 后来Givental、Lian-Liu-Yau等给出数学证明（Mirror Theorem）

#### Mirror Theorem

**Givental定理**：对 toric complete intersection Calabi-Yau，$J$-函数（生成GW不变量）等于 $I$-函数（B-模型周期），经镜像映射。

**Lian-Liu-Yau定理**：对Quintic等具体情形，证明了CDGP预测。

#### 现代发展

1. **SYZ猜想**（Strominger-Yau-Zaslow）：镜像对由对偶特殊Lagrange纤维化给出
2. **Homological Mirror Symmetry**（Kontsevich）：$D^b Fuk(X) \cong D^b Coh(\check{X})$
3. **Wall-crossing**：Bridgeland稳定性条件下的不变量跳跃公式
4. **log GW理论**：Abramovich-Chen-Gross-Siebert的logarithmic Gromov-Witten理论，统一了退化技巧和Berkovich几何
