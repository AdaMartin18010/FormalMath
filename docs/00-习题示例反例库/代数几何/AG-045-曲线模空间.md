# 曲线模空间

## 题目

**(a)** 证明 M_g 的维数计算。

**(b)** 讨论 M_g 的紧致化（Deligne-Mumford）。

**(c)** 计算 M₁ = A¹ 的 j-线。

## 解答

### (a) 维数计算

光滑射影曲线：3g-3 个模（g ≥ 2）。

变形理论：H¹(C, T_C)，Riemann-Roch 给出 h¹ = 3g-3。

### (b) DM 紧致化

M̄_g：稳定曲线（结点、有限自同构）。

边界 Δ = M̄_g \ M_g 是除子。

### (c) M₁

椭圆曲线：y² = x³ + ax + b，判别式 Δ ≠ 0。

j-不变量：j = 1728·4a³/(4a³ + 27b²)。

M₁ ≅ A¹_j。
