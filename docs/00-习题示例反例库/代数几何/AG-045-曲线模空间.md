# 曲线模空间

## 题目

**(a)** 证明 $M_g$ 的维数计算。

**(b)** 讨论 $M_g$ 的紧致化（Deligne-Mumford）。

**(c)** 计算 $M_1 = \mathbb{A}^1$ 的 $j$-线。

---

## 解答

### (a) $M_g$ 的维数

#### 变形理论

光滑射影曲线 $C$ 的变形由 $H^1(C, T_C)$ 参数化。

由Riemann-Roch：
$$h^0(T_C) - h^1(T_C) = \deg(T_C) + 1 - g = 2 - 2g + 1 - g = 3 - 3g$$

$h^0(T_C) = 0$（$g \geq 2$ 时无非零向量场），故：
$$h^1(T_C) = 3g - 3$$

**定理**：$\dim M_g = 3g - 3$（$g \geq 2$）。

$g = 0$：$M_0 = \{pt\}$（$\mathbb{P}^1$ 唯一）。

$g = 1$：$M_1 = \mathbb{A}^1$（$j$-线）。

---

### (b) Deligne-Mumford紧致化

#### 稳定曲线

**定义**：$n$-点标记的稳定曲线 $(C, p_1, \ldots, p_n)$：
- $C$ 是连通 nodal 曲线
- 算术亏格 $g$
- 稳定性：$\text{Aut}(C, p_1, \ldots, p_n)$ 有限

等价：每个有理分量至少有3个特殊点（节点或标记点）。

#### $\bar{M}_g$ 的结构

**定理**（Deligne-Mumford）：$\bar{M}_g$ 是紧的、不可约的 $3g-3$ 维Deligne-Mumford栈。

**边界**：$\Delta = \bar{M}_g \setminus M_g$ 是除子，分层为：
- $\Delta_0$：不可约 nodal 曲线
- $\Delta_i$（$1 \leq i \leq [g/2]$）：两个分量在一点相交，亏格分别为 $i$ 和 $g-i$

#### 模空间的几何

**Picard群**（Harer, Arbarello-Cornalba）：
$$\text{Pic}(\bar{M}_g) \otimes \mathbb{Q} = \mathbb{Q}\lambda \oplus \mathbb{Q}\delta_0 \oplus \cdots \oplus \mathbb{Q}\delta_{[g/2]}$$

- $\lambda = c_1(\mathbb{E})$：Hodge bundle
- $\delta_i = [\Delta_i]$：边界除子

### (c) $M_1$ 与 $j$-线

#### Weierstrass方程

椭圆曲线：$y^2 = x^3 + ax + b$，判别式 $\Delta = 4a^3 + 27b^2 \neq 0$。

#### $j$-不变量

$$j = 1728 \cdot \frac{4a^3}{4a^3 + 27b^2}$$

**定理**：$j$ 完全分类椭圆曲线的同构类（在代数闭域上）。

$$M_1 \cong \mathbb{A}^1_j$$

#### 证明

- $j$ 在同构下不变
- $j(E) = j(E') \Rightarrow E \cong E'$
- 对任意 $j \in k$，存在椭圆曲线 $E$ 使 $j(E) = j$：
  - $j = 0$：$y^2 = x^3 + 1$
  - $j = 1728$：$y^2 = x^3 + x$
  - 一般 $j$：$y^2 + xy = x^3 - \frac{36}{j-1728}x - \frac{1}{j-1728}$

$\square$

#### 模形式

$M_1$ 的坐标 $j$ 不是模形式（有极点）。

模形式：$E_4$，$E_6$，$\Delta = (E_4^3 - E_6^2)/1728$。

$j = E_4^3 / \Delta$。
