# 双有理几何深入

## 题目

**(a)** 证明 BCHM 定理（一般型典范模型）。

**(b)** 讨论 Sarkisov 程序和双有理rigidity。

**(c)** 证明 Flip 的存在性。

## 解答

### (a) BCHM

Birkar-Cascini-Hacon-McKernan：

对数一般型对 (X, D)，典范模型存在。

有限生成性 + MMP。

### (b) Sarkisov 程序

双有理映射分解为 links。

rigidity：无其他 Mori 纤维空间结构。

### (c) Flip 存在性

Shokurov：flip 存在性约化到 pl-flip。

Hacon-McKernan：扩展理想层和限制定理。
