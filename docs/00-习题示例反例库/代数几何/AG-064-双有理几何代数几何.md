### (a) BCHM定理

已在AG-037中详细讨论。核心要点：

**定理**（Birkar-Cascini-Hacon-McKernan, 2006）：对数一般型对 $(X, \Delta)$，典范模型存在。

**关键**：
- 有限生成性：$R(X, K_X + \Delta)$ 有限生成
- MMP：extremal ray contraction + flip
- 归纳维数 + extension theorem

### (b) Sarkisov程序与双有理rigidity

#### Sarkisov程序

**目标**：将两个Mori纤维空间 $X \to S$ 和 $X' \to S'$ 之间的双有理映射分解为简单的**Sarkisov links**。

**Sarkisov links的类型**：

1. **Type I**：$X \dashrightarrow X_1$ 是 divsorial contraction + 接着 isomorphism
2. **Type II**：$X \dashrightarrow X_1$ 是 flip/flop
3. **Type III**：$X \dashrightarrow X_1$ 是 inverse of Type I
4. **Type IV**：$X$ 和 $X_1$ 是同一底空间 $S$ 上的不同纤维空间

#### 双有理rigidity

**定义**：$X$ 是**双有理刚性的**（birationally rigid），若任何到另一Mori纤维空间的双有理映射实际上是由 $X$ 的自同构给出的。

等价：$X$ 有唯一的Mori纤维空间结构（在双有理等价意义下）。

**例子**：
- 一般的三次超曲面 $X_3 \subseteq \mathbb{P}^4$（Iskovskikh-Manin, Clemens-Griffiths）
- 一般的四次超曲面 $X_4 \subseteq \mathbb{P}^4$
- 某些del Pezzo纤维化

**证明rigidity的方法**：
1. **Noether-Fano不等式**：控制移动线性系的奇点
2. **极大奇点**（maximal singularities）：不存在除已知之外的
3. **Sarkisov程序**：所有links回到自身

### (c) Flip存在性

已在AG-037中详细讨论。核心要点：

**定理**：flip存在且终止（termination of flips）。

**Shokurov的约化**：
- flip存在性 $\Rightarrow$ **pl-flip**存在性（purely log terminal flip）
- pl-flip存在性 $\Rightarrow$ **有限生成性**

**Hacon-McKernan的扩展定理**：

$H^0(X, m(K_X + S + B)) \to H^0(S, m(K_S + B_S))$ 满射。

这是证明flip存在性的关键技术。

#### 最新进展

- **Birkar（2018, Fields奖）**：flip的终止性（某种意义下）
- **Shokurov的complements理论**：有界性 $\Rightarrow$ 终止性
