# 向量丛模空间

## 题目

**(a)** 定义稳定向量丛和半稳定向量丛。

**(b)** 证明 M(r, d)（曲线上的稳定丛模空间）的存在性。

**(c)** 讨论模空间的辛结构。

## 解答

### (a) Mumford 稳定性

斜率 μ(E) = deg(E)/rank(E)。

稳定：对所有真子丛 F，μ(F) < μ(E)。

半稳定：μ(F) ≤ μ(E)。

### (b) 模空间存在性

Mumford：GIT 构造，商 Quot 概形。

维数：r²(g-1) + 1（与 r, d 互素时）。

### (c) 辛结构

Hitchin 系统：余切丛 T*M 有 Hyperkähler 结构。

模空间上的自然辛形式。
