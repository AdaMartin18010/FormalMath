# 向量丛模空间

## 题目

**(a)** 定义稳定向量丛和半稳定向量丛。

**(b)** 证明 $M(r,d)$（曲线上的稳定丛模空间）的存在性。

**(c)** 讨论模空间的辛结构。

## 解答

### (a) Mumford 稳定性

**背景**：设 $X$ 为光滑射影曲线（紧 Riemann 面），亏格 $g \geq 0$。$X$ 上的全纯向量丛 $E$ 的拓扑不变量由秩 $r = \text{rank}(E)$ 和度 $d = \deg(E) = \deg(\det E)$ 决定。

对向量丛的分类，最简单的情形是线丛（$r=1$）。此时由 Jacobi 簇（Jacobian variety）$J^d(X)$ 参数化，这是一个 $g$ 维 Abel 簇。对于秩 $r \geq 2$，分类问题更加复杂，因为向量丛的模空间中存在"退化"的丛，其行为类似于轨道的 GIT 商中的不稳定点。

**定义（斜率）**：向量丛 $E$ 的**斜率**（slope）定义为：
$$\mu(E) = \frac{\deg(E)}{\text{rank}(E)}$$
斜率是稳定性理论的核心数值不变量。

**定义（稳定性）**：设 $E$ 为 $X$ 上的非零全纯向量丛。
- $E$ 称为**稳定**（stable）的，如果对所有真子丛 $0 \subsetneq F \subsetneq E$，有：
  $$\mu(F) < \mu(E)$$
- $E$ 称为**半稳定**（semistable）的，如果对所有真子丛 $0 \subsetneq F \subsetneq E$，有：
  $$\mu(F) \leq \mu(E)$$

**基本性质**：

1. **线丛的稳定性**：每个线丛 $L$ 自动稳定（因为不存在非平凡真子丛）。

2. **斜率的加法性**：对正合列 $0 \to E_1 \to E \to E_2 \to 0$：
   $$\mu(E) = \frac{r_1 \mu(E_1) + r_2 \mu(E_2)}{r_1 + r_2}$$
   因此 $\mu(E)$ 是 $\mu(E_1)$ 和 $\mu(E_2)$ 的加权平均。$E$ 稳定意味着它不能分解为两个斜率更大或相等的子丛。

3. **Harder-Narasimhan 滤过**：每个向量丛 $E$ 都有唯一的滤过：
   $$0 = E_0 \subset E_1 \subset \cdots \subset E_k = E$$
   使得每个商 $E_i/E_{i-1}$ 半稳定，且 $\mu(E_i/E_{i-1}) > \mu(E_{i+1}/E_i)$。

4. **Jordan-Hölder 定理**：对半稳定丛 $E$，存在合成列（composition series），其因子是稳定丛且斜率相同。这些因子在同构和重排意义下唯一。

**例子**：
- 在 $\mathbb{P}^1$ 上，Grothendieck 定理说每个全纯丛分解为线丛直和 $E = \bigoplus_i \mathcal{O}(a_i)$。此时 $E$ 稳定当且仅当 $E$ 本身就是线丛（因为不同线丛的斜率不同，直和不满足严格不等式）。
- 在椭圆曲线（$g=1$）上，Atiyah 分类了秩 $r$、度 $d$ 的丛。当 $(r,d)=1$ 时，稳定丛与半稳定丛一致。

**注**：稳定性是**开条件**——在平坦族中，稳定丛的集合构成 Zariski 开子集。

### (b) 模空间 $M(r,d)$ 的存在性

**模函子**：定义模函子 $\mathcal{M}_{r,d}$：对任意概形 $S$，$\mathcal{M}_{r,d}(S)$ 是 $S \times X$ 上平坦族向量丛（秩 $r$，度 $d$）的同构类集合，模去 $S$-同构。

**问题**：这个函子通常**不可表示**（not representable），因为稳定丛有自同构群（当 $(r,d) \neq 1$ 时，标量乘法给出 $\mathbb{C}^*$ 自同构）。因此需要构造"粗模空间"（coarse moduli space）或利用某种等价关系。

**定理（Mumford, Seshadri, Narasimhan）**：设 $X$ 为光滑射影曲线，$(r,d)=1$（互素）。则存在光滑射影簇 $M(r,d)$，它是稳定丛的精细模空间（fine moduli space）。

对一般的 $(r,d)$，存在粗模空间 $M(r,d)$，参数化 S-等价类（S-equivalence classes）的半稳定丛。

*构造方法（GIT 构造）*：

**步骤1：Quot 概形**。固定一个足够丰沛（ample）的线丛 $\mathcal{O}_X(1)$。对 $N$ 充分大，每个半稳定丛 $E$ 满足 $H^1(X, E(N)) = 0$，且 $E(N)$ 由整体截面生成。令 $V = \mathbb{C}^{P(N)}$，其中 $P(N) = rN + d + r(1-g)$ 是 Hilbert 多项式。则 $E$ 对应于 Quot 概形中的一个点：
$$\text{Quot}_{r,d} = \{q: V \otimes \mathcal{O}_X(-N) \twoheadrightarrow E \to 0\}$$
其中 $E$ 有 Hilbert 多项式 $P$，且 $V \xrightarrow{\cong} H^0(X, E(N))$。

**步骤2：群作用**。$GL(V)$ 自然作用在 $\text{Quot}_{r,d}$ 上，改变 $V$ 的基。由于标量矩阵作用平凡，实际作用群是 $PGL(V)$。

**步骤3：Mumford 的稳定性判据**。利用 Hilbert 判据，Mumford 证明了：向量丛 $E$ 的稳定性等价于对应 Quot 概形点在 GIT 意义下的稳定性。具体地，设 $E$ 对应点 $[q] \in \text{Quot}$。则：
- $E$ 稳定 $\Leftrightarrow$ $[q]$ 是 GIT 稳定点
- $E$ 半稳定 $\Leftrightarrow$ $[q]$ 是 GIT 半稳定点

GIT 稳定点的意义是：对任意1参数子群 $\lambda: \mathbb{C}^* \to PGL(V)$，当 $t \to 0$ 时，$\lambda(t) \cdot [q]$ 的极限行为的 Hilbert-Mumford 权为正。

**步骤4：取商**。由 GIT 的主要定理，半稳定点的商存在且是射影簇：
$$M(r,d) = \text{Quot}_{r,d}^{ss} /\!/ PGL(V)$$
稳定点的商是拟射影的，当 $(r,d)=1$ 时所有半稳定点都稳定，故 $M(r,d)$ 是射影簇。

**维数公式**：当 $(r,d)=1$ 时：
$$\dim M(r,d) = r^2(g-1) + 1$$
这是因为稳定丛 $E$ 的自同构群仅为标量 $\mathbb{C}^*$（已模去），其切空间为 $H^1(X, \text{End}(E))$。由 Riemann-Roch：
$$\chi(\text{End}(E)) = r^2(1-g)$$
对稳定丛，$H^0(X, \text{End}(E)) = \mathbb{C}$（仅有标量自同态，由 Schur 引理），故 $h^1 = r^2(g-1) + 1$。

**Narasimhan-Seshadri 定理**：设 $X$ 为紧 Riemann 面。秩 $r$、度 $d$（$(r,d)=1$）的全纯向量丛 $E$ 稳定当且仅当 $E$ 对应于 $X$ 的基本群的**不可约酉表示**：
$$\rho: \pi_1(X) \to U(r)$$
满足 $\det \rho = e^{2\pi i d/r} \cdot \text{id}$（在中心上）。

这建立了代数几何的稳定性与微分几何的平坦酉联络之间的深刻对应：
$$\{\text{稳定全纯丛}\} \xleftrightarrow{1:1} \{\text{不可约酉表示}\}$$
对应映射由 $E \mapsto (\bar{\partial}_E, h)$ 的 Hermitian-Einstein 联络给出，其中 $h$ 是 $E$ 上的 Hermitian 度量。

### (c) 模空间的辛结构

模空间 $M(r,d)$ 具有丰富的几何结构，其中辛结构（symplectic structure）和 Hyperkähler 结构尤为重要。

**辛形式**：设 $(r,d)=1$，$M = M(r,d)$。在 $[E] \in M$ 处，切空间为：
$$T_{[E]} M \cong H^1(X, \text{End}_0(E))$$
其中 $\text{End}_0(E)$ 是迹零自同态丛。

利用 Serre 对偶：
$$H^1(X, \text{End}_0(E))^* \cong H^0(X, K_X \otimes \text{End}_0(E)^*) \cong H^0(X, K_X \otimes \text{End}_0(E))$$
（因为 $\text{End}_0(E)$ 是自对偶的），我们可以定义2-形式：
$$\omega(\alpha, \beta) = \int_X \text{tr}(\alpha \cup \beta)$$
其中 $\alpha, \beta \in H^1(X, \text{End}_0(E))$，$\cup$ 是楔积与迹的组合，给出 $K_X$ 值2-形式，积分后得到复数。

**定理（Narasimhan）**：上述 $\omega$ 是 $M(r,d)$ 上的**全纯辛形式**（holomorphic symplectic form），即：
- $\omega$ 非退化
- $d\omega = 0$（闭的）
- $\omega$ 是全纯的

实际上，$M(r,d)$ 的维数 $r^2(g-1)+1$ 总是偶数（因为 $r^2(g-1)$ 的奇偶性），故辛形式有意义。

**Hitchin 系统（1987）**：模空间 $M(r,d)$ 可以推广到**Higgs 丛模空间** $\mathcal{M}_{r,d}^{Higgs}$。Higgs 丛是一对 $(E, \Phi)$，其中 $E$ 是全纯向量丛，$\Phi \in H^0(X, K_X \otimes \text{End}(E))$ 是 Higgs 场。稳定性条件要求对 $\Phi$-不变子丛 $F \subset E$，斜率不等式成立。

**定理（Hitchin）**：$\mathcal{M}_{r,d}^{Higgs}$ 具有**Hyperkähler 结构**，即它同时具有三个复结构 $I, J, K = IJ$，满足四元数关系，且每个复结构对应一个 Kähler 形式，组合为全纯辛形式。

在复结构 $I$ 下，$\mathcal{M}_{r,d}^{Higgs}$ 是 Higgs 丛的模空间（代数结构）。在复结构 $J$ 下，由 Narasimhan-Seshadri-Hitchin 对应，它同构于 $GL(r, \mathbb{C})$-特征的模空间：
$$\mathcal{M}_{r,d}^{Higgs} \cong \{(A, \Phi) : F_A + [\Phi, \Phi^*] = 0\} / \mathcal{G}$$
这是平坦联络模空间（复版本）。

**Hitchin 纤维化**：定义映射（Hitchin map）：
$$h: \mathcal{M}_{r,d}^{Higgs} \to \bigoplus_{i=2}^r H^0(X, K_X^{\otimes i})$$
$$h(E, \Phi) = (\text{tr}(\Phi^2), \text{tr}(\Phi^3), \ldots, \text{tr}(\Phi^r))$$
这是特征多项式的系数。

**定理**：$h$ 是 proper 的、全纯的满射，其一般纤维是 Abel 簇（具体地，是谱曲线 $X_\Phi$ 的 Picard 簇）。这使得 $\mathcal{M}_{r,d}^{Higgs}$ 成为一个**代数完全可积系统**（algebraic completely integrable system）。

**谱曲线**：对 $\Phi \in H^0(X, K_X \otimes \text{End}(E))$，定义在总空间 $|K_X|$ 中的曲线：
$$X_\Phi = \{(x, \lambda) \in K_X : \det(\lambda \cdot \text{id} - \Phi(x)) = 0\}$$
这是 $X$ 的 $r$ 重分支覆盖。Higgs 丛 $(E, \Phi)$ 对应于谱曲线 $X_\Phi$ 上的线丛，这解释了纤维的 Abel 簇结构。

这些结构（辛、Hyperkähler、可积系统）使模空间成为连接代数几何、微分几何、表示论和数学物理的枢纽。例如，Hitchin 系统在几何 Langlands 纲领、镜像对称和拓扑场论中都扮演核心角色。
