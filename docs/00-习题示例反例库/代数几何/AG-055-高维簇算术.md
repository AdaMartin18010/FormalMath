# 高维簇的算术

## 题目

**(a)** 证明 Faltings 定理：高亏格曲线的有理点有限。

**(b)** 讨论 Manin-Mumford 猜想。

**(c)** 证明 André-Oort 猜想（陈述）。

## 解答

### (a) Faltings 定理

Mordell 猜想：g ≥ 2 曲线 C，C(ℚ) 有限。

Faltings 方法：Arakelov 理论 + Parshin 构造 + 高度理论。

或 Vojta：Diophantine 逼近。

### (b) Manin-Mumford

C 嵌入 Jac(C)，C ∩ Jac(C)_{tors} 有限。

Raynaud 证明，Hindry 推广到任意簇。

### (c) André-Oort

Shimura 簇中特殊点的 Zariski 闭包是特殊子簇。

Klingler-Yafaev、Ullmo 证明。
