# 枚举几何

## 题目

**(a)** 计算 P² 中 d 次曲线的 Gromov-Witten 不变量。

**(b)** 证明镜面对称的 BCOV 理论。

**(c)** 讨论稳定映射模空间。

## 解答

### (a) GW 不变量

Kontsevich 公式：N_d = Σ N_{d₁}N_{d₂}d₁²d₂²(d₁²d₂²-2)/(d²-1)。

递归计算，N₁ = 1，N₂ = 1，N₃ = 12，...

### (b) BCOV

Bershadsky-Cecotti-Ooguri-Vafa：高亏格势函数。

镜面对称计算 B 模型。

### (c) 稳定映射

M̄_{g,n}(X, β)：亏格 g，n 个标记点，类 β 的稳定映射。

Kontsevich 紧化，虚拟基本类。
