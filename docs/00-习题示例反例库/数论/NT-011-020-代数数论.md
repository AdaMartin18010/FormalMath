# 代数数论习题 (NT-011 ~ NT-020)

## NT-011 ⭐⭐⭐

**题目**：设 $K = \mathbb{Q}(\sqrt{d})$ 是二次域，其中 $d$ 是无平方因子整数。

**(a)** 确定 $K$ 的整数环 $\mathcal{O}_K$。

**(b)** 计算判别式 $d_K$。

**(c)** 当 $d = -5$ 时，证明 $\mathcal{O}_K$ 不是唯一分解整环。

**来源**：代数数论基础

**解答思路**：
1. (a) 当 $d \equiv 2, 3 \pmod{4}$ 时，$\mathcal{O}_K = \mathbb{Z}[\sqrt{d}]$；当 $d \equiv 1 \pmod{4}$ 时，$\mathcal{O}_K = \mathbb{Z}[\frac{1+\sqrt{d}}{2}]$
2. (b) 判别式公式：$d_K = d$ 若 $d \equiv 1 \pmod{4}$；$d_K = 4d$ 若 $d \equiv 2, 3 \pmod{4}$
3. (c) 考虑 $6 = 2 \cdot 3 = (1+\sqrt{-5})(1-\sqrt{-5})$，证明不可约元不唯一

**关键步骤**：
- 极小多项式的整数性判别
- 范数计算：$N(a + b\sqrt{-5}) = a^2 + 5b^2$

## NT-012 ⭐⭐⭐⭐

**题目**：设 $K$ 是数域，$\mathcal{O}_K$ 是其整数环。

**(a)** 证明 $\mathcal{O}_K$ 是 Dedekind 整环。

**(b)** 证明 $\mathcal{O}_K$ 的理想类群 $\text{Cl}(\mathcal{O}_K)$ 是有限的（类数有限定理）。

**来源**：理想论基础

**解答思路**：
1. (a) 验证三条：Noether 环、整闭、非零素理想极大
2. 整闭性：$\mathcal{O}_K$ 是 $\mathbb{Z}$ 在 $K$ 中的整闭包
3. (b) 利用 Minkowski 定理：每个理想类包含代表元满足范数界
4. 证明范数小于 Minkowski 界的理想只有有限个

**关键步骤**：
- Minkowski 界：$N(\mathfrak{a}) \leq \frac{n!}{n^n} \left(\frac{4}{\pi}\right)^{r_2} \sqrt{|d_K|}$
- 格的体积与 covolume 计算

## NT-013 ⭐⭐⭐⭐

**题目**：设 $L/K$ 是数域的 Galois 扩张，$\mathfrak{p}$ 是 $\mathcal{O}_K$ 的素理想，在 $\mathcal{O}_L$ 中分解为：
$$\mathfrak{p}\mathcal{O}_L = \mathfrak{P}_1^{e_1} \cdots \mathfrak{P}_g^{e_g}$$

**(a)** 证明 Galois 群 $G = \text{Gal}(L/K)$ 传递地作用在 $\{\mathfrak{P}_i\}$ 上。

**(b)** 由此推出所有分歧指数相等：$e_1 = \cdots = e_g = e$。

**(c)** 定义分解群 $D_{\mathfrak{P}} = \{\sigma \in G : \sigma(\mathfrak{P}) = \mathfrak{P}\}$，证明 $|D_{\mathfrak{P}}| = ef$，其中 $f = [\mathcal{O}_L/\mathfrak{P} : \mathcal{O}_K/\mathfrak{p}]$。

**来源**：Galois 扩张的素理想分解

**解答思路**：
1. (a) 用中国剩余定理和 Galois 不变性
2. 若存在 $\mathfrak{P}_i$ 不能达到，构造矛盾元素
3. (b) 传递性直接推出分解的均匀性
4. (c) 轨道-稳定子定理：$|G| = g \cdot |D_{\mathfrak{P}}|$，结合 $[L:K] = efg$

**关键步骤**：
- 扩张次数与局部次数的关系
- 惯性群的进一步结构

## NT-014 ⭐⭐⭐⭐⭐

**题目**：（局部域与完备化）设 $K$ 是数域，$\mathfrak{p}$ 是 $\mathcal{O}_K$ 的素理想，$v_{\mathfrak{p}}$ 是相应的离散赋值。

**(a)** 定义 $K_{\mathfrak{p}}$ 为 $K$ 关于 $v_{\mathfrak{p}}$ 的完备化。证明 $K_{\mathfrak{p}}$ 是局部紧域。

**(b)** 设 $L/K$ 是有限扩张，$\mathfrak{P} | \mathfrak{p}$。证明局部扩张次数满足：
$$[L_{\mathfrak{P}} : K_{\mathfrak{p}}] = e(\mathfrak{P}/\mathfrak{p}) \cdot f(\mathfrak{P}/\mathfrak{p})$$

**来源**：局部域理论

**解答思路**：
1. (a) 构造绝对值 $|x|_{\mathfrak{p}} = N(\mathfrak{p})^{-v_{\mathfrak{p}}(x)}$
2. 验证 $K_{\mathfrak{p}}$ 是完备赋值域，剩余类域有限
3. (b) 利用基的提升：选取 $K$ 上 $L$ 的基，在完备化后仍保持基性质
4. 或者通过张量积：$L \otimes_K K_{\mathfrak{p}} \cong \prod_{\mathfrak{P}|\mathfrak{p}} L_{\mathfrak{P}}$

**关键步骤**：
- Hensel 引理的应用
- 分歧与惯性在局部的表现

## NT-015 ⭐⭐⭐⭐⭐

**题目**：（Kronecker-Weber 定理的证明框架）

**(a)** 证明：$\mathbb{Q}(\zeta_m)/\mathbb{Q}$ 是 Abel 扩张，且 $\text{Gal}(\mathbb{Q}(\zeta_m)/\mathbb{Q}) \cong (\mathbb{Z}/m\mathbb{Z})^\times$。

**(b)** （Kronecker-Weber 定理）设 $K/\mathbb{Q}$ 是有限 Abel 扩张，证明存在正整数 $m$ 使得 $K \subseteq \mathbb{Q}(\zeta_m)$。

**来源**：类域论入门

**解答思路**：
1. (a) 自同构由 $\sigma_a: \zeta_m \mapsto \zeta_m^a$ 给出，$(a,m)=1$
2. (b) 证明框架：
   - 第一步：约化到 $K/\mathbb{Q}$ 是素数幂次循环扩张
   - 第二步：局部分析，利用分歧理论
   - 第三步：构造适当的分圆域包含给定扩张
3. 关键工具：局部类域论、导子（conductor）的概念

**关键步骤**：
- 分圆域的判别式计算
- 导子的上界估计

**(a)** 证明 $\mathbb{A}_K$ 是局部紧拓扑环。

**(b)** 证明 $K$ 在 $\mathbb{A}_K$ 中离散，且 $\mathbb{A}_K/K$ 紧（即 $K$ 是 $\mathbb{A}_K$ 的 cocompact 格）。

**(c)** 计算 $\mathbb{A}_K/K$ 的体积（Tate 的 Riemann-Roch 定理）。

**关键步骤**：
- 在 $\mathbb{A}_K$ 上选取自对偶测度

## NT-017 ⭐⭐⭐⭐

**题目**：设 $\rho: G_{\mathbb{Q}} \to \text{GL}_2(\overline{\mathbb{Q}}_\ell)$ 是 Galois 群的二维 $\ell$-adic 表示，其中 $G_{\mathbb{Q}} = \text{Gal}(\overline{\mathbb{Q}}/\mathbb{Q})$。

**(a)** 定义 $L(\rho, s) = \prod_p \det(I - \rho(\text{Frob}_p) p^{-s})^{-1}$（对好的素数）。证明该 Euler 乘积在 $\text{Re}(s) > 1$ 时收敛。

**(b)** 设 $E/\mathbb{Q}$ 是椭圆曲线，$\rho_{E,\ell}$ 是 Tate 模表示。证明 $L(\rho_{E,\ell}, s)$ 与 $E$ 的 Hasse-Weil $L$ 函数一致。

**来源**：Galois 表示基础

**解答思路**：
1. (a) 利用 Weil 猜想（Deligne 定理）：$|\lambda_i| \leq p^{w/2}$
2. 对于二维表示，$L$ 因子形式为 $(1 - a_p p^{-s} + \epsilon(p) p^{k-1-2s})^{-1}$
3. (b) Tate 模：$T_\ell(E) = \varprojlim E[\ell^n]$，$\rho_{E,\ell}$ 作用于其上
4. 比较局部 $L$ 因子的定义

**关键步骤**：
- Frobenius 元素与约化曲线的关系
- Hasse 界 $|a_p| \leq 2\sqrt{p}$

## NT-018 ⭐⭐⭐⭐⭐

**题目**：（模 $p$ 与模 $\ell$ Galois 表示的提升）

设 $\bar{\rho}: G_{\mathbb{Q}} \to \text{GL}_2(\mathbb{F}_p)$ 是 Galois 群的模 $p$ 表示。

**(a)** 什么是 $\bar{\rho}$ 的形变（deformation）？定义形变函子 $D_{\bar{\rho}}$。

**(b)** 设 $\bar{\rho}$ 是绝对不可约的。证明 $D_{\bar{\rho}}$ 是可表函子（Mazur 定理）。

**(c)** 讨论 Fermat 大定理的 Galois 表示证明思路（Wiles 定理）。

**来源**：Galois 表示形变理论

**解答思路**：
1. (a) 形变：提升 $\rho: G_{\mathbb{Q}} \to \text{GL}_2(A)$ 使得与 $\bar{\rho}$ 模极大理想相容
2. (b) 利用 Schlessinger 判别准则验证形变函子的可表性
3. (c) Wiles 证明框架：
   - 将 Fermat 方程与椭圆曲线 $E: y^2 = x(x-a^p)(x+b^p)$ 联系
   - 证明该曲线对应的 Galois 表示是模的
   - 利用 Serre 猜想（现为定理）

**关键步骤**：
- 形变环的结构
- 最低水平的提升

## NT-019 ⭐⭐⭐⭐⭐

**题目**：设 $K$ 是虚二次域，$H$ 是 $K$ 的 Hilbert 类域。

**(a)** 证明 $H/K$ 是 $K$ 的最大非分歧 Abel 扩张。

**(b)** 证明 $\text{Gal}(H/K) \cong \text{Cl}(\mathcal{O}_K)$（类群与 Galois 群的同构）。

**(c)** 当 $K = \mathbb{Q}(\sqrt{-23})$ 时，计算类数并确定 $H$。

**来源**：复乘理论

3. (c) 计算 Minkowski 界，枚举范数小的理想
   - $\mathcal{O}_K = \mathbb{Z}[\frac{1+\sqrt{-23}}{2}]$，$d_K = -23$
   - Minkowski 界 $\approx 3.05$，检查素数 2, 3
   - 证明类数为 3

**关键步骤**：
- 二元二次型的约化理论
- 判别式与类数的关系

## NT-020 ⭐⭐⭐⭐⭐

**题目**：（Iwasawa 理论初步）设 $p$ 是素数，$K = \mathbb{Q}(\mu_p)$，$K_\infty = \bigcup_{n \geq 1} \mathbb{Q}(\mu_{p^n})$ 是分圆 $\mathbb{Z}_p$-扩张。

**(a)** 定义 $K_\infty/K$ 的分支群和惯性群结构。

**(b)** 设 $\Gamma = \text{Gal}(K_\infty/K) \cong \mathbb{Z}_p$，$\Lambda = \mathbb{Z}_p[[\Gamma]]$ 是 Iwasawa 代数。描述 $\Lambda$ 的代数结构。

**(c)** 叙述 Iwasawa 主猜想：分圆域的 $p$-进 $L$ 函数与类群的特征理想之间的关系。

**来源**：Iwasawa 理论

**解答思路**：
1. (a) 唯一分歧的素数是 $p$，完全分歧
2. (b) $\Lambda \cong \mathbb{Z}_p[[T]]$（选取拓扑生成元 $\gamma \mapsto 1+T$）
3. (c) Iwasawa 主猜想（Mazur-Wiles 定理证明）：
   - 设 $X$ 是 $K_\infty$ 的最大非分歧 Abel $p$-扩张的 Galois 群
   - $X$ 是有限生成扭 $\Lambda$-模
   - 特征理想 $\text{char}(X)$ 与 Kubota-Leopoldt $p$-进 $L$ 函数生成相同理想

**关键步骤**：
- 伪同构与特征理想
- $p$-进解析插值
