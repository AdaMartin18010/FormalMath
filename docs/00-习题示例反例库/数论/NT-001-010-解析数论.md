# 解析数论习题 (NT-001 ~ NT-010)

## NT-001 ⭐⭐⭐

**题目**：设 $\pi(x)$ 为不超过 $x$ 的素数个数。利用切比雪夫方法证明：存在正常数 $c_1, c_2$ 使得对于充分大的 $x$，有
$$c_1 \frac{x}{\log x} \leq \pi(x) \leq c_2 \frac{x}{\log x}$$

**来源**：解析数论基础，切比雪夫定理

**解答思路**：
1. 考虑 $n!$ 的素因子分解
2. 利用 Legendre 公式计算素数 $p$ 在 $n!$ 中的指数
3. 建立 $\psi(x) = \sum_{p^k \leq x} \log p$ 与 $\pi(x)$ 的关系
4. 通过积分估计得到上下界

**关键步骤**：
- 证明 $\psi(x) = O(x)$ 和 $\psi(x) \gg x$
- 利用 Abel 求和公式转换

**参考**：Apostol, Introduction to Analytic Number Theory, Ch. 4

## NT-002 ⭐⭐⭐⭐

**题目**：证明素数定理的等价形式：
$$\psi(x) \sim x \quad \Longleftrightarrow \quad \pi(x) \sim \frac{x}{\log x}$$

其中 $\psi(x) = \sum_{n \leq x} \Lambda(n)$ 是第二切比雪夫函数，$\Lambda(n)$ 是 von Mangoldt 函数。

**来源**：素数定理证明

**解答思路**：
1. ($\Rightarrow$) 利用 Abel 求和：$\pi(x) = \frac{\psi(x)}{\log x} + \int_2^x \frac{\psi(t)}{t \log^2 t} dt$
2. 由 $\psi(x) \sim x$ 代入得渐近公式
3. ($\Leftarrow$) 利用分部求和反向推导
4. 控制误差项的阶

**关键步骤**：
- 验证两个方向的误差相容性
- 注意余项的精细估计

## NT-003 ⭐⭐⭐⭐⭐

**题目**：利用复变函数方法证明：
$$\sum_{n=1}^{\infty} \frac{\mu(n)}{n} = 0$$

其中 $\mu(n)$ 是 Möbius 函数。并由此推出素数定理。

**来源**：解析数论高级技巧

**解答思路**：
1. 考虑 Dirichlet 级数 $\sum_{n=1}^{\infty} \frac{\mu(n)}{n^s} = \frac{1}{\zeta(s)}$，$\text{Re}(s) > 1$
2. 证明 $\zeta(s)$ 在 $\text{Re}(s) = 1$ 上无零点（Hadamard-de la Vallée Poussin 定理）
3. 利用 Tauber 型定理（如 Ikehara-Wiener 定理）
4. 由 $\frac{1}{\zeta(s)}$ 在 $s=1$ 处解析推出求和为0

**关键步骤**：
- 构造适当的围道积分
- 处理 $\zeta$ 函数的零自由区域

## NT-004 ⭐⭐⭐⭐

**题目**：设 $\chi$ 是模 $q$ 的 Dirichlet 特征标，$L(s, \chi)$ 是相应的 Dirichlet $L$ 函数。

**(a)** 证明当 $\chi \neq \chi_0$（主特征标）时，$L(1, \chi) \neq 0$。

**(b)** 利用 (a) 证明 Dirichlet 定理：对 $(a,q)=1$，算术级数 $a, a+q, a+2q, \ldots$ 中含有无穷多个素数。

**来源**：Dirichlet 定理证明

**解答思路**：
1. (a) 分特征标为实特征标和复特征标两种情况
2. 实特征标情形利用类数公式或 Landau 方法证明非零
3. (b) 考虑 $\sum_{p \equiv a \pmod{q}} \frac{1}{p^s}$ 的 Dirichlet 密度
4. 利用特征标的正交性关系

**关键步骤**：
- 实原特征标 $L(1, \chi) > 0$ 的证明
- 密度公式中的极点分析

## NT-005 ⭐⭐⭐⭐⭐

**题目**：（Riemann $\zeta$ 函数的函数方程）

证明：对于所有 $s \in \mathbb{C}$，有
$$\pi^{-s/2} \Gamma\left(\frac{s}{2}\right) \zeta(s) = \pi^{-(1-s)/2} \Gamma\left(\frac{1-s}{2}\right) \zeta(1-s)$$

**来源**：Riemann 假设基础

**解答思路**：
1. 从 Poisson 求和公式出发
2. 考虑 theta 函数 $\theta(t) = \sum_{n \in \mathbb{Z}} e^{-\pi n^2 t}$
3. 利用 Mellin 变换建立 $\zeta(s)$ 与 theta 函数的联系
4. 通过 theta 函数的模变换性质推导函数方程

**关键步骤**：
- Poisson 求和：$\theta(1/t) = \sqrt{t} \theta(t)$
- 围道积分与解析延拓
- Gamma 函数的反射公式应用

## NT-006 ⭐⭐⭐⭐

**题目**：设 $N(T)$ 表示 $\zeta(s)$ 在区域 $0 < \text{Im}(s) < T$，$0 \leq \text{Re}(s) \leq 1$ 内的零点个数。

证明 Riemann-von Mangoldt 公式：
$$N(T) = \frac{T}{2\pi} \log \frac{T}{2\pi} - \frac{T}{2\pi} + O(\log T)$$

**来源**：零点分布理论

**解答思路**：
1. 应用辐角原理于 $\xi(s) = \frac{1}{2} s(s-1) \pi^{-s/2} \Gamma(s/2) \zeta(s)$
2. 计算 $\Delta_{\mathcal{R}} \arg \xi(s)$ 沿矩形围道的变化
3. 利用 Stirling 公式估计 Gamma 函数的贡献
4. 估计 $\zeta(s)$ 的幅角变化

**关键步骤**：
- 围道选取：顶点为 $2, 2+iT, -1+iT, -1$
- 函数 $\xi(s)$ 是整函数且阶为1

## NT-007 ⭐⭐⭐⭐⭐

**题目**：（显式公式）设 $\rho = \beta + i\gamma$ 遍历 $\zeta(s)$ 的所有非平凡零点（计重数）。

对于 $x > 1$ 且 $x$ 不是素数幂，证明：
$$\psi_0(x) = x - \sum_{\rho} \frac{x^{\rho}}{\rho} - \frac{\zeta'(0)}{\zeta(0)} - \frac{1}{2} \log(1-x^{-2})$$

其中 $\psi_0(x) = \frac{1}{2}(\psi(x^+) + \psi(x^-))$。

**来源**：素数分布精细理论

**解答思路**：
1. 从 Perron 公式出发：$\frac{1}{2\pi i} \int_{c-iT}^{c+iT} \frac{x^s}{s} ds$
2. 将围道向左移动，经过 $\zeta'(s)/\zeta(s)$ 的极点
3. 计算各极点的留数贡献
4. 估计围道积分余项

**关键步骤**：
- 极点位于 $s = 1$（来自 $\zeta$ 的简单极点）、$s = \rho$（非平凡零点）、$s = -2n$（平凡零点）
- 水平线积分的估计需要零自由区域

## NT-008 ⭐⭐⭐⭐

**题目**：设 $p_n$ 表示第 $n$ 个素数。利用素数定理证明：
$$p_n \sim n \log n \quad (n \to \infty)$$

并进一步证明更精细的渐近公式：
$$p_n = n(\log n + \log \log n - 1 + o(1))$$

**来源**：素数分布渐近分析

**解答思路**：
1. 由 $\pi(p_n) = n$ 和 $\pi(x) \sim x/\log x$ 得 $n \sim p_n/\log p_n$
2. 取对数得 $\log n \sim \log p_n - \log \log p_n$
3. 迭代求解渐近展开
4. 利用 $\pi(x) = \text{li}(x) + O(x e^{-c\sqrt{\log x}})$ 获得更精细估计

**关键步骤**：
- 反函数渐近技巧
- 迭代法求高阶渐近项

## NT-009 ⭐⭐⭐⭐⭐

**题目**：（算术级数中的素数分布）

设 $(a,q) = 1$，定义 $\pi(x; q, a) = \sum_{\substack{p \leq x \\ p \equiv a \pmod{q}}} 1$。

证明 Siegel-Walfisz 定理：对于任意 $A > 0$，当 $q \leq (\log x)^A$ 时，
$$\pi(x; q, a) = \frac{\text{li}(x)}{\varphi(q)} + O_A(x e^{-c\sqrt{\log x}})$$

**来源**：算术级数素数分布

**解答思路**：
1. 建立 $\psi(x; q, a)$ 的显式公式
2. 利用特征标分解：$\psi(x; q, a) = \frac{1}{\varphi(q)} \sum_{\chi \pmod{q}} \bar{\chi}(a) \psi(x, \chi)$
3. 需要 $L(s, \chi)$ 的零自由区域（Siegel 定理）
4. 处理例外零点（Siegel 零点）的影响

**关键步骤**：
- Siegel 定理：$L(1, \chi) \gg_\epsilon q^{-\epsilon}$
- 有效与无效估计的区分

## NT-010 ⭐⭐⭐⭐⭐

**题目**：设 $\zeta(s)$ 的非平凡零点为 $\rho = \beta + i\gamma$。假设 Riemann 假设成立（即所有 $\beta = 1/2$），证明素数定理的误差项可以改进为：
$$\psi(x) = x + O(\sqrt{x} \log^2 x)$$

并讨论：反过来是否成立？即由误差项 $O(x^{1/2+\epsilon})$ 能否推出 Riemann 假设？

**来源**：Riemann 假设与误差项

**解答思路**：
1. 由显式公式：$\psi(x) = x - \sum_{|\gamma| \leq T} \frac{x^{\rho}}{\rho} + O\left(\frac{x \log^2 x}{T}\right)$
2. RH 下 $\rho = 1/2 + i\gamma$，故 $|x^{\rho}| = \sqrt{x}$
3. 利用 $N(T) \sim \frac{T}{2\pi} \log T$ 估计零点个数
4. 选取 $T = x$ 平衡各项
5. 反向需要 Lindelöf 假设等更强条件

**关键步骤**：
- 零点求和的绝对收敛性
- 误差项与零点实部的等价关系（Littlewood 定理）
