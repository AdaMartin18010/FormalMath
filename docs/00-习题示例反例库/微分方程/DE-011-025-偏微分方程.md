# 偏微分方程习题 (DE-011 ~ DE-025)

## DE-011 ⭐⭐⭐⭐

**题目**：设 $\Omega \subset \mathbb{R}^n$ 是有界光滑区域，考虑椭圆边值问题
$$-\Delta u = f \text{ in } \Omega, \quad u = 0 \text{ on } \partial\Omega$$

**(a)** 用 Lax-Milgram 定理证明当 $f \in L^2(\Omega)$ 时存在唯一弱解 $u \in H_0^1(\Omega)$。

**(b)** 证明椭圆正则性：若 $f \in H^k(\Omega)$，则 $u \in H^{k+2}(\Omega)$。

**(c)** 叙述极大值原理：若 $f \geq 0$，则 $u \geq 0$。

**来源**：椭圆型方程基础

**解答思路**：
1. (a) 变分形式：$a(u, v) = \int_\Omega \nabla u \cdot \nabla v = \int_\Omega f v$
   - 强制性由 Poincaré 不等式给出
2. (b) 差商方法或 Fourier 分析
   - 内正则性和边界正则性分别处理
3. (c) 弱极大值原理：$-\Delta u \geq 0$ $\Rightarrow$ $\min_{\bar{\Omega}} u = \min_{\partial\Omega} u = 0$

**关键步骤**：
- Sobolev 空间的紧嵌入
- Schauder 估计

## DE-012 ⭐⭐⭐⭐

**题目**：考虑非线性椭圆方程
$$-\Delta u = |u|^{p-1}u \text{ in } \Omega, \quad u = 0 \text{ on } \partial\Omega$$

**(a)** 当 $1 < p < \frac{n+2}{n-2}$（次临界），用变分法证明存在正解。

**(b)** 当 $p = \frac{n+2}{n-2}$（临界 Sobolev 指数），讨论紧性的缺失与爆破现象。

**(c)** 对于临界情形，叙述 Brezis-Nirenberg 关于低阶扰动的存在性结果。

**来源**：非线性椭圆方程

**解答思路**：
1. (a) 极小化泛函 $J(u) = \frac{1}{2}\int |\nabla u|^2 - \frac{1}{p+1}\int |u|^{p+1}$
   - Palais-Smale 条件由紧嵌入保证
2. (b) $H^1 \hookrightarrow L^{2^*}$ 不紧，极小化序列可能爆破
   - Talenti 函数给出最佳 Sobolev 常数
3. (c) 添加低阶项 $-\lambda u$ 可恢复紧性

**关键步骤**：
- Mountain Pass 定理
- 集中紧性原理

## DE-013 ⭐⭐⭐⭐⭐

**题目**：（特征值问题）设 $\Omega \subset \mathbb{R}^n$ 是有界区域。

**(a)** 证明 Laplace 算子第一特征值 $\lambda_1$ 是单的，对应特征函数不变号。

**(b)** 叙述 Courant-Fischer 极小极大原理：
$$\lambda_k = \min_{\dim V = k} \max_{u \in V, u \neq 0} \frac{\int_\Omega |\nabla u|^2}{\int_\Omega u^2}$$

**(c)** 证明 Faber-Krahn 不等式：在所有体积相同的区域中，球的 $\lambda_1$ 最小。

**来源**：谱理论

**解答思路**：
1. (a) 第一特征函数是泛函的极小元
   - 若 $u$ 变号，则 $|u|$ 也是极小元，由强极大值原理 $u$ 不变号
2. (b) 极小极大原理的变分刻画
3. (c) Schwarz 对称化：$\lambda_1(\Omega) \geq \lambda_1(\Omega^\star)$
   - 等周不等式的谱版本

**关键步骤**：
- 重排不等式
- 热核比较

## DE-014 ⭐⭐⭐⭐⭐

**题目**：（De Giorgi-Nash-Moser 理论）考虑散度型椭圆方程
$$-\text{div}(A(x)\nabla u) = 0 \text{ in } \Omega$$

其中 $A(x)$ 满足一致椭圆条件：$\lambda |\xi|^2 \leq A_{ij}(x)\xi_i\xi_j \leq \Lambda |\xi|^2$。

**(a)** 叙述 De Giorgi 的局部有界性定理。

**(b)** 叙述 Nash 的 H"older 连续性结果。

**(c)** 证明 Moser 的 Harnack 不等式：对正解 $u$，有
$$\sup_{B_r} u \leq C \inf_{B_r} u$$

**来源**：椭圆正则性理论

**解答思路**：
1. (a) De Giorgi：弱解是局部有界的，$u \in L_{loc}^\infty$
   - Caccioppoli 不等式 + 迭代
2. (b) Nash：弱解是 H"older 连续的
   - 关键估计：振荡衰减
3. (c) Moser 迭代：测试函数 $u^p$ 的选择
   - 对数变换后应用均值不等式

**关键步骤**：
- 尺度不变估计
- John-Nirenberg 不等式

## DE-015 ⭐⭐⭐⭐⭐

**题目**：（抛物型方程）考虑热方程初边值问题
$$\partial_t u - \Delta u = f \text{ in } \Omega \times (0, T), \quad u = 0 \text{ on } \partial\Omega \times (0, T), \quad u(0) = u_0$$

**(a)** 用 Galerkin 方法证明存在唯一弱解 $u \in L^2(0, T; H_0^1) \cap L^\infty(0, T; L^2)$。

**(b)** 证明能量估计：$\frac{1}{2}\frac{d}{dt}\|u\|_{L^2}^2 + \|\nabla u\|_{L^2}^2 = \int fu$。

**(c)** 当 $u_0 \geq 0$，证明 $u(t) \geq 0$（抛物型极大值原理）。

**来源**：抛物型方程基础

**解答思路**：
1. (a) Galerkin：取有限维基 $\{w_k\}$，求解 ODE 系统
   - 先验估计 + 紧性论证
2. (b) 用 $u$ 乘方程并积分
3. (c) 弱极大值原理：设 $u^- = \max(-u, 0)$，证明 $\frac{d}{dt}\|u^-\|^2 \leq 0$

**关键步骤**：
- Aubin-Lions 紧性引理
- 半群理论（$e^{t\Delta}$ 是正算子）

## DE-016 ⭐⭐⭐⭐

**题目**：考虑半线性热方程
$$\partial_t u - \Delta u = u^p \text{ in } \mathbb{R}^n \times (0, T), \quad u(0) = u_0 \geq 0$$

**(a)** 当 $p > 1$，证明小初值存在全局解，大初值解在有限时间爆破。

**(b)** 定义 Fujita 临界指数 $p_F = 1 + \frac{2}{n}$，证明：
   - $1 < p \leq p_F$：所有非负解爆破
   - $p > p_F$：小初值存在全局解

**(c)** 计算爆破时间和爆破速率。

**来源**：爆破理论

**解答思路**：
1. (a) 比较原理 + 构造上下解
   - 上解：常数 $M \geq u_0$，$M \geq M^p$（当 $M$ 小）
2. (b) Fujita 临界现象：
   - 线性热核衰减 $t^{-n/2}$ 与非线性增长 $u^p$ 的平衡
   - 测试函数方法证明爆破
3. (c) 自相似爆破：$u(x,t) \sim (T-t)^{-\frac{1}{p-1}}$

**关键步骤**：
- Kaplan 特征函数方法
- 尺度分析与自相似解

## DE-017 ⭐⭐⭐⭐

**题目**：（非线性扩散）考虑多孔介质方程（PME）
$$\partial_t u = \Delta(u^m), \quad m > 1$$

**(a)** 推导 Barenblatt 自相似解（ZKB 解）：
$$U(x,t) = t^{-\alpha} \left[1 - \frac{(m-1)\beta}{2m} \frac{|x|^2}{t^{2\beta}}\right]_+^{\frac{1}{m-1}}$$
其中 $\alpha = \frac{n}{n(m-1)+2}$，$\beta = \frac{1}{n(m-1)+2}$。

**(b)** 证明有限传播速度：若 $u_0$ 具有紧支集，则 $u(t)$ 对所有 $t > 0$ 具有紧支集。

**(c)** 叙述 PME 解的渐近行为：$u(t) \sim U(t)$ 当 $t \to \infty$。

**来源**：退化抛物方程

**解答思路**：
1. (a) 尺度不变性分析：设 $u(x,t) = t^{-\alpha} f(x/t^\beta)$
   - 求解 ODE 得 Barenblatt 解
2. (b) 界面 $s(t)$ 满足 $s(t) \sim t^\beta$
   - 与热方程（$m=1$）的无限传播速度对比
3. (c) 熵方法或 Lyapunov 泛函

**关键步骤**：
- 弱解与能量解的定义
- 熵耗散不等式

## DE-018 ⭐⭐⭐⭐⭐

**题目**：（双曲守恒律）考虑 Burgers 方程
$$\partial_t u + \partial_x \left(\frac{u^2}{2}\right) = 0, \quad u(x, 0) = u_0(x)$$

**(a)** 用特征线方法求解光滑初值问题，证明光滑解在有限时间产生激波当且仅当 $u_0$ 在某处递减。

**(b)** 定义弱解，叙述 Rankine-Hugoniot 激波条件。

**(c)** 当 $u_0(x) = \begin{cases} 1 & x < 0 \\ -1 & x > 0 \end{cases}$，构造包含激波的熵解。

**来源**：守恒律基础

**解答思路**：
1. (a) 特征线：$x = x_0 + u_0(x_0)t$
   - 特征线相交时产生激波：$t_s = -1/\min u_0'$
2. (b) 弱解：分布意义满足方程
   - R-H 条件：$s = \frac{f(u_R) - f(u_L)}{u_R - u_L}$（激波速度）
3. (c) 激波速度 $s = 0$，解为
   $$u(x,t) = \begin{cases} 1 & x < 0 \\ -1 & x > 0 \end{cases}$$

**关键步骤**：
- Oleinik 熵条件
- Lax 几何熵条件

## DE-019 ⭐⭐⭐⭐⭐

**题目**：（双曲方程组）考虑一维等熵 Euler 方程
$$\partial_t \rho + \partial_x(\rho u) = 0, \quad \partial_t(\rho u) + \partial_x(\rho u^2 + p(\rho)) = 0$$

其中 $p(\rho) = K\rho^\gamma$，$\gamma > 1$。

**(a)** 将方程组写成守恒形式 $\partial_t U + \partial_x F(U) = 0$。

**(b)** 计算 Jacobian $A(U) = DF(U)$ 的特征值，证明系统是双曲的。

**(c)** 叙述 Riemann 问题的解结构：激波、稀疏波、接触间断。

**来源**：可压缩流

**解答思路**：
1. (a) $U = (\rho, \rho u)^T$，$F(U) = (\rho u, \rho u^2 + p)^T$
2. (b) 特征值：$\lambda_{1,2} = u \mp c$，$c = \sqrt{p'(\rho)}$（声速）
   - 两个不同的实特征值 $\Rightarrow$ 严格双曲
3. (c) Riemann 问题初值 $U(x,0) = \begin{cases} U_L & x < 0 \\ U_R & x > 0 \end{cases}$
   - 中间波可能是激波或稀疏波

**关键步骤**：
-  genuinely nonlinear 与 linearly degenerate 场
- Lax 激波条件

## DE-020 ⭐⭐⭐⭐⭐

**题目**：（波动方程）考虑非线性波动方程
$$\partial_t^2 u - \Delta u = |u|^{p-1}u \text{ in } \mathbb{R}^n \times \mathbb{R}$$

**(a)** 定义能量：$E(u) = \int \left(\frac{1}{2}|\partial_t u|^2 + \frac{1}{2}|\nabla u|^2 - \frac{1}{p+1}|u|^{p+1}\right)dx$，证明能量守恒。

**(b)** 当 $p < p_c = \frac{n+2}{n-2}$（次临界），用 Strichartz 估计证明局部适定性。

**(c)** 叙述 Strauss 猜想：临界指数 $p_0(n)$ 使得小初值全局存在当且仅当 $p > p_0(n)$。

**来源**：非线性波动方程

**解答思路**：
1. (a) 形式计算：$\frac{dE}{dt} = \int (u_{tt} u_t + \nabla u \cdot \nabla u_t - |u|^{p-1}u u_t) = 0$
2. (b) Strichartz 估计：$\|u\|_{L^q L^r} \lesssim \|u_0\|_{\dot{H}^s} + \|u_1\|_{\dot{H}^{s-1}} + \|F\|_{L^{\tilde{q}'}L^{\tilde{r}'}}$
   - 选取适当的指标，用不动点定理
3. (c) Strauss 指数是方程 $p^2 - (n+1)p - 2 = 0$ 的正根

**关键步骤**：
- 散射理论
- 集中爆破机制

## DE-021 ⭐⭐⭐⭐

**题目**：（Navier-Stokes 方程）考虑不可压缩 Navier-Stokes 方程
$$\partial_t u - \nu \Delta u + (u \cdot \nabla)u + \nabla p = 0, \quad \text{div } u = 0$$

**(a)** 用 Leray 投影消去压力，写成算子形式。

**(b)** 叙述 Leray 弱解的存在性定理（1934）。

**(c)** 讨论弱解的正则性和唯一性：二维情形唯一，三维情形 Millennium Prize 问题。

**来源**：流体力学 PDE

**解答思路**：
1. (a) Leray 投影 $P: L^2 \to \{u : \text{div } u = 0\}$
   - $u_t - \nu \Delta u + P(u \cdot \nabla)u = 0$
2. (b) Leray：存在满足能量不等式的弱解
   - $u \in L^\infty(0,T; L^2) \cap L^2(0,T; H^1)$
3. (c) 二维：enstrophy 估计给出正则性
   - 三维：部分正则性（Caffarelli-Kohn-Nirenberg）

**关键步骤**：
- Helmholtz 分解
- Serrin 正则性准则

## DE-022 ⭐⭐⭐⭐⭐

**题目**：（薛定谔方程）考虑非线性薛定谔方程（NLS）
$$i\partial_t u + \Delta u = \pm |u|^{p-1}u, \quad u(0) = u_0$$

**(a)** 证明质量守恒 $M(u) = \int |u|^2$ 和能量守恒 $E(u) = \int (\frac{1}{2}|\nabla u|^2 \pm \frac{1}{p+1}|u|^{p+1})$。

**(b)** 对于聚焦情形（负号），证明驻波解的存在性：$u(x,t) = e^{i\omega t}Q(x)$。

**(c)** 叙述轨道稳定性/不稳定性结果（Weinstein, Grillakis-Shatah-Strauss）。

**来源**：色散方程

**解答思路**：
1. (a) 形式计算：$\frac{dM}{dt} = 2\text{Re}\int \bar{u} u_t = 0$
2. (b) $Q$ 满足 $-\Delta Q + \omega Q = Q^p$，用变分法（极小化 Gagliardo-Nirenberg 泛函）
3. (c) 稳定性判据与线性化算子的谱有关
   - 基态解通常是轨道稳定的

**关键步骤**：
- 驻波的变分刻画
- 调制稳定性分析

## DE-023 ⭐⭐⭐⭐⭐

**题目**：（边界层理论）考虑粘性 Burgers 方程
$$\partial_t u^\varepsilon + u^\varepsilon \partial_x u^\varepsilon = \varepsilon \partial_x^2 u^\varepsilon, \quad u^\varepsilon(0) = u_0$$

**(a)** 当 $\varepsilon \to 0$，形式推导无粘 Burgers 方程。

**(b)** 在激波附近构造边界层展开：$u^\varepsilon(x,t) = U\left(\frac{x-st}{\varepsilon}, t\right) + O(\varepsilon)$。

**(c)** 证明边界层方程的解存在唯一，且满足匹配条件。

**来源**：奇异摄动

**解答思路**：
1. (a) 形式极限得 $\partial_t u + u\partial_x u = 0$
2. (b) 内变量 $\xi = (x-st)/\varepsilon$
   - 边界层方程：$-s U' + U U' = U''$
3. (c) 求解 ODE：$U(\xi) = s - A \tanh(A\xi/2)$
   - 匹配条件：$U(-\infty) = u_L$，$U(+\infty) = u_R$

**关键步骤**：
- Prandtl 边界层方程
- 匹配渐近展开

## DE-024 ⭐⭐⭐⭐⭐

**题目**：（变分法与 Gamma 收敛）设 $\Omega \subset \mathbb{R}^n$ 有界，$F_\varepsilon(u) = \int_\Omega \left(\frac{\varepsilon}{2}|\nabla u|^2 + \frac{1}{\varepsilon}W(u)\right)dx$

其中 $W$ 是双阱势（如 $W(u) = (u^2-1)^2$）。

**(a)** 叙述 Modica-Mortola 定理：$F_\varepsilon$ 在 $L^1$ 中 Gamma 收敛于面积泛函 $c_0 \text{Per}(E)$。

**(b)** 证明下界不等式：$\liminf F_\varepsilon(u_\varepsilon) \geq c_0 \text{Per}(E)$。

**(c)** 讨论 Allen-Cahn 方程 $\partial_t u = \Delta u - W'(u)/\varepsilon^2$ 与平均曲率流的关系。

**来源**：变分法与几何流

**解答思路**：
1. (a) Gamma 收敛：liminf 不等式 + 恢复序列
   - 过渡层厚度 $\sim \varepsilon$，能量 $\sim \varepsilon^{-1} \cdot \varepsilon = O(1)$
2. (b) 一维最优剖面：$q'(t) = \sqrt{2W(q(t))}$
   - 常数 $c_0 = \int_{-1}^1 \sqrt{2W(s)} ds$
3. (c) 形式展开：界面运动由平均曲率驱动

**关键步骤**：
-  Young 测度
- 布拉克相变动力学

## DE-025 ⭐⭐⭐⭐⭐

**题目**：（最优传输与 Monge-Ampère）设 $\mu, \nu$ 是 $\mathbb{R}^n$ 上的概率测度，具有密度 $f, g$。

**(a)** 叙述 Kantorovich 最优传输问题：
$$\inf_{\pi \in \Pi(\mu,\nu)} \int |x-y|^2 d\pi(x,y)$$

**(b)** 当 $\mu$ 绝对连续，证明存在唯一的最优传输映射 $T$，且 $T = \nabla \phi$ 是凸函数的梯度。

**(c)** 推导 Monge-Ampère 方程：$\det(D^2\phi) = \frac{f}{g \circ \nabla\phi}$。

**来源**：最优传输理论

**解答思路**：
1. (a) Kantorovich 松弛：允许分拆传输
2. (b) Brenier 定理：
   - Kantorovich 对偶问题
   - $c(x,y) = |x-y|^2/2$ 时，最优传输是梯度映射
3. (c) 由测度保持：$\int_A f = \int_{T(A)} g = \int_A g(\nabla\phi) \det(D^2\phi)$
   - 得 Monge-Ampère 方程

**关键步骤**：
- 凸分析（Legendre 变换）
- Caffarelli 正则性理论
