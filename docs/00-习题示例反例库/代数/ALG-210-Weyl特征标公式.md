# Weyl 特征标公式

## 题目

**(a)** 证明 Weyl 特征标公式：ch L(λ) = Σ_{w∈W} ε(w) e^{w(λ+ρ)} / e^ρ ∏_{α>0}(1-e^{-α})。

**(b)** 证明 Weyl 维数公式。

**(c)** 计算经典群表示的维数。

## 解答

### (a) Weyl 特征标

证明：Weyl 特征标公式。关键：特征标在 Weyl 群下反对称，分母公式。

### (b) Weyl 维数公式

$$\dim L(\lambda) = \prod_{\alpha > 0} \frac{\langle \lambda + \rho, \alpha^\vee \rangle}{\langle \rho, \alpha^\vee \rangle}$$

### (c) 经典群

sl(n)：V(mω₁) 维数 C(m+n-1, m)。

so(2n+1)：旋量表示维数 2^n。
