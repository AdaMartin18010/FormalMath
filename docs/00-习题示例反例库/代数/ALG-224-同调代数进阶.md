# 同调代数进阶

## 题目

**(a)** 证明导出函子的长正合列。

**(b)** 讨论谱序列及其收敛性。

**(c)** 证明 Grothendieck 谱序列定理。

## 解答

### (a) 导出函子长正合列

0 → A → B → C → 0 短正合 ⟹ ··· → R^iF(A) → R^iF(B) → R^iF(C) → R^{i+1}F(A) → ···

构造：内射分解 + 蛇引理。

### (b) 谱序列

E_r^{p,q}，微分 d_r: E_r^{p,q} → E_r^{p+r,q-r+1}。

收敛：E_∞^{p,q} 是 H^{p+q} 的滤过商。

### (c) Grothendieck 谱序列

F: A → B, G: B → C，F 保内射。

则 E_2^{p,q} = R^pG(R^qF(A)) ⇒ R^{p+q}(GF)(A)。
