# 李群表示

## 题目

**(a)** 定义李群的表示并证明有限维表示与 Lie 代数表示的对应。

**(b)** 证明 Peter-Weyl 定理。

**(c)** 讨论紧李群的极大环面和 Weyl 积分公式。

## 解答

### (a) 李群与 Lie 代数

G 连通单连通，则 G 的表示 ↔ g 的表示。

指数映射联系：ρ(exp(X)) = exp(dρ(X))。

### (b) Peter-Weyl 定理

L²(G) ≅ ⊕_{π∈Ĝ} V_π ⊗ V_π^*，矩阵元 {π_{ij}} 是 L² 的正交基。

### (c) 极大环面

紧连通 G 的极大环面 T，W = N(T)/T。

Weyl 积分：∫_G f(g)dg = (1/|W|) ∫_T f(t) |Δ(t)|² dt。
