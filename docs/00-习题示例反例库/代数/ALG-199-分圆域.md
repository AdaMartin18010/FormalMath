# 分圆域

## 题目

**(a)** 证明 Q(ζ_n)/Q 的 Galois 群同构于 (ℤ/nℤ)^×。

**(b)** 证明分圆域的判别式公式。

**(c)** 讨论分圆域的类数与 Fermat 最后定理的关系。

## 解答

### (a) Galois 群

ζ_n ↦ ζ_n^a，a ∈ (ℤ/nℤ)^× 给出 Galois 作用。

次数 φ(n) = |(ℤ/nℤ)^×|，故 Gal ≅ (ℤ/nℤ)^×。

### (b) 判别式

$$d_{\mathbb{Q}(\zeta_n)} = (-1)^{\frac{\varphi(n)}{2}} \frac{n^{\varphi(n)}}{\prod_{p|n} p^{\frac{\varphi(n)}{p-1}}}$$

### (c) Kummer 理论

正则素数：p ∤ h_{Q(ζ_p)}。Kummer：Fermat 对正则素数成立。
