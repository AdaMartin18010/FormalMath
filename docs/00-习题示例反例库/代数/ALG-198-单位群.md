# 单位群

## 题目

**(a)** 证明 Dirichlet 单位定理：$\mathcal{O}_K^\times \cong \mu(K) \times \mathbb{Z}^{r_1+r_2-1}$。

**(b)** 计算 $\mathbb{Q}(\sqrt[3]{2})$ 和 $\mathbb{Q}(\sqrt{2}, \sqrt{3})$ 的单位群秩。

**(c)** 讨论分圆域的单位群和分圆单位。

## 解答

### (a) Dirichlet 单位定理的证明

**设定**：设 $K$ 是数域，$[K:\mathbb{Q}] = n$。设 $r_1$ 是实嵌入个数，$2r_2$ 是复嵌入对个数（$n = r_1 + 2r_2$）。$\mathcal{O}_K$ 是整数环，$\mathcal{O}_K^\times$ 是其单位群。$\mu(K)$ 是 $K$ 中的单位根群。

**Dirichlet 单位定理**：
$$\mathcal{O}_K^\times \cong \mu(K) \times \mathbb{Z}^{r_1 + r_2 - 1}$$
即单位群是有限循环群（单位根）和自由 Abel 群的直积，自由部分的秩为 $r = r_1 + r_2 - 1$。

**证明**：

**步骤 1：对数嵌入（logarithmic embedding）**

设 $\sigma_1, \ldots, \sigma_{r_1}$ 是实嵌入，$\sigma_{r_1+1}, \bar{\sigma}_{r_1+1}, \ldots, \sigma_{r_1+r_2}, \bar{\sigma}_{r_1+r_2}$ 是复嵌入。定义映射：
$$\lambda: \mathcal{O}_K^\times \to \mathbb{R}^{r_1 + r_2}$$
$$\lambda(\alpha) = (\log|\sigma_1(\alpha)|, \ldots, \log|\sigma_{r_1}(\alpha)|, 2\log|\sigma_{r_1+1}(\alpha)|, \ldots, 2\log|\sigma_{r_1+r_2}(\alpha)|)$$

**引理 1**：$\ker \lambda = \mu(K)$。

*证明*：若 $\lambda(\alpha) = 0$，则对所有 $i$，$|\sigma_i(\alpha)| = 1$。于是 $\alpha$ 的所有 Galois 共轭的绝对值均为 1。这推出 $\alpha$ 的极小多项式的系数有界，故满足该条件的 $\alpha$ 只有有限多个。因此 $\alpha$ 生成有限乘法群，即单位根。反之，单位根显然满足 $|\sigma(\zeta)| = 1$。∎

**步骤 2：像位于超平面中**

**引理 2**：$\lambda(\mathcal{O}_K^\times) \subset H$，其中 $H = \{x \in \mathbb{R}^{r_1+r_2} : \sum_{i=1}^{r_1+r_2} x_i = 0\}$。

*证明*：由范数公式 $N_{K/\mathbb{Q}}(\alpha) = \prod_{i=1}^{r_1} \sigma_i(\alpha) \prod_{j=1}^{r_2} \sigma_{r_1+j}(\alpha)\bar{\sigma}_{r_1+j}(\alpha)$。若 $\alpha \in \mathcal{O}_K^\times$，则 $N(\alpha) = \pm 1$，故 $|N(\alpha)| = 1$。取对数：
$$\sum_{i=1}^{r_1} \log|\sigma_i(\alpha)| + \sum_{j=1}^{r_2} 2\log|\sigma_{r_1+j}(\alpha)| = \log|N(\alpha)| = 0$$
这正是 $\sum_{i=1}^{r_1+r_2} x_i = 0$。∎

**步骤 3：像的离散性与秩**

**引理 3**：$\lambda(\mathcal{O}_K^\times)$ 是 $H$ 中的完备格（complete lattice），即秩为 $\dim H = r_1 + r_2 - 1$ 的自由 Abel 群。

*证明概要*：利用 Minkowski 格点定理。对 $H$ 中任意有界区域，只有有限多个 $\alpha \in \mathcal{O}_K^\times$ 的像落在其中（因系数有界 $\Rightarrow$ 有限个代数整数）。这说明 $\lambda(\mathcal{O}_K^\times)$ 离散。

为证其生成 $H$，构造足够多的单位。对 $H$ 中任意非零线性泛函，通过选择适当的嵌入上的凸对称区域，利用 Minkowski 定理找到非零 $\alpha \in \mathcal{O}_K$ 满足特定的范数条件。重复此过程构造 $r$ 个乘法无关的单位。

**步骤 4：结论**

由引理 1，$\mathcal{O}_K^\times / \mu(K) \cong \lambda(\mathcal{O}_K^\times) \cong \mathbb{Z}^r$，其中 $r = r_1 + r_2 - 1$。故：
$$\mathcal{O}_K^\times \cong \mu(K) \times \mathbb{Z}^{r_1+r_2-1}$$

### (b) 单位群秩的计算

**(i) $K = \mathbb{Q}(\sqrt[3]{2})$**

$K$ 由 $x^3 - 2$ 的根生成。嵌入：
- $\sigma_1(\sqrt[3]{2}) = \sqrt[3]{2}$（实嵌入）
- $\sigma_2(\sqrt[3]{2}) = \zeta_3 \sqrt[3]{2}$，$\sigma_3(\sqrt[3]{2}) = \zeta_3^2 \sqrt[3]{2}$（复嵌入对）

因此 $r_1 = 1$，$r_2 = 1$，$n = 3$。单位秩：
$$r = r_1 + r_2 - 1 = 1 + 1 - 1 = 1$$

即 $\mathcal{O}_K^\times$ 的挠部分乘以一个无限循环群。基本单位可取 $\varepsilon = \sqrt[3]{2} - 1$（或其逆），因为 $N(\sqrt[3]{2} - 1) = 1 - 2 = -1$（在 $x^3 - 2 = (x - \sqrt[3]{2})(x - \zeta_3\sqrt[3]{2})(x - \zeta_3^2\sqrt[3]{2})$ 中代入 $x = 1$）。

**(ii) $K = \mathbb{Q}(\sqrt{2}, \sqrt{3})$**

这是双二次扩张，$[K:\mathbb{Q}] = 4$。嵌入由 $\sqrt{2} \mapsto \pm\sqrt{2}$，$\sqrt{3} \mapsto \pm\sqrt{3}$ 给出，共 4 个嵌入，均为实的：
$$(\sqrt{2}, \sqrt{3}) \mapsto (\sqrt{2}, \sqrt{3}), (-\sqrt{2}, \sqrt{3}), (\sqrt{2}, -\sqrt{3}), (-\sqrt{2}, -\sqrt{3})$$

故 $r_1 = 4$，$r_2 = 0$。单位秩：
$$r = 4 + 0 - 1 = 3$$

一组基本单位系可取：$\varepsilon_1 = 1 + \sqrt{2}$，$\varepsilon_2 = 2 + \sqrt{3}$，$\varepsilon_3 = \sqrt{2} + \sqrt{3}$。

### (c) 分圆域的单位群和分圆单位

**设定**：设 $\zeta_n = e^{2\pi i/n}$，$K = \mathbb{Q}(\zeta_n)$ 是第 $n$ 个分圆域。

**单位秩**：$[K:\mathbb{Q}] = \varphi(n)$。嵌入由 $\zeta_n \mapsto \zeta_n^a$（$(a,n) = 1$）给出，均为复的（除 $n = 1, 2$ 外）。故 $r_1 = 0$，$2r_2 = \varphi(n)$，即 $r_2 = \varphi(n)/2$（$\varphi(n)$ 为偶数当 $n > 2$）。

单位秩：
$$r = r_1 + r_2 - 1 = \frac{\varphi(n)}{2} - 1$$

**分圆单位（Cyclotomic Units）**：对 $a \in (\mathbb{Z}/n\mathbb{Z})^\times$，定义：
$$\eta_a = \frac{1 - \zeta_n^a}{1 - \zeta_n}$$

**定理**：当 $n = p$ 为素数时，分圆单位群 $C_p = \langle \pm\zeta_p, \eta_2, \eta_3, \ldots, \eta_{(p-1)/2} \rangle$ 在 $\mathcal{O}_{\mathbb{Q}(\zeta_p)}^\times$ 中有限指数。且：
$$[\mathcal{O}_{\mathbb{Q}(\zeta_p)}^\times : C_p] = h_p^+$$
其中 $h_p^+$ 是 $\mathbb{Q}(\zeta_p + \zeta_p^{-1})$ 的类数（极大实子域的类数）。

**Kummer 证明**：分圆单位群的 regulator 与 $L$-函数在 $s = 0$ 的值直接相关。由解析类数公式：
$$\text{Res}_{s=1} \zeta_K(s) = \frac{2^{r_1}(2\pi)^{r_2} R_K h_K}{w_K \sqrt{|d_K|}}$$
对 $K = \mathbb{Q}(\zeta_p + \zeta_p^{-1})$，比较分圆单位的 regulator 与完整 regulator 即得指数公式。

**常见错误**：
- 误认为分圆单位生成整个单位群：实际上只生成有限指数子群，指数即类数 $h_p^+$。
- 混淆 $r_2$ 与复嵌入总数：$r_2$ 是复嵌入**对**数，不是复嵌入数。
