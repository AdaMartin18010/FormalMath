# 单位群

## 题目

**(a)** 证明 Dirichlet 单位定理：O_K^× ≅ μ(K) × ℤ^{r₁+r₂-1}。

**(b)** 计算 Q(∛2) 和 Q(√2, √3) 的单位群秩。

**(c)** 讨论分圆域的单位群和分圆单位。

## 解答

### (a) Dirichlet 单位定理

对数嵌入：λ: O_K^× → ℝ^{r₁+r₂}，λ(α) = (log|σ₁(α)|, ..., log|σ_{r₁+r₂}(α)|)。

像位于超平面 ∑ x_i = 0，是格。

### (b) 计算

Q(∛2)：r₁=1, r₂=1，秩1。

Q(√2, √3)：r₁=4, r₂=0，秩3。

### (c) 分圆域单位

Q(ζ_n)^× 秩 φ(n)/2 - 1。分圆单位由 (1-ζ_n^a)/(1-ζ_n) 生成。
