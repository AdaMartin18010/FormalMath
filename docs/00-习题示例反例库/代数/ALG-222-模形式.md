# 模形式

## 题目

**(a)** 定义模形式并证明其 Fourier 展开。

**(b)** 证明 Hecke 算子的交换性和特征形式。

**(c)** 讨论模形式与椭圆曲线的对应（Eichler-Shimura）。

## 解答

### (a) 模形式

f(γz) = (cz+d)^k f(z)，γ ∈ SL₂(ℤ)。尖点形式：Fourier 展开 a_n q^n，a_0=0。

### (b) Hecke 算子

T_n：双重陪集 Γ(1)\(1 n; 0 1)/Γ(1)。

性质：T_m T_n = T_{mn} (m,n)=1，T_{p^{r+1}} = T_p T_{p^r} - p^{k-1} T_{p^{r-1}}。

### (c) Eichler-Shimura

f = Σ a_n q^n 权2 本原 Hecke 特征形式 ⟹ 存在椭圆曲线 E_f，a_p = p+1 - |E_f(F_p)|。
