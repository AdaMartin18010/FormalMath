# 枚举几何

## 题目

**(a)** 计算 $\mathbb{P}^2$ 中 $d$ 次曲线经过 $3d-1$ 个一般点的个数（Kontsevich 公式）。

**(b)** 证明镜面对称关于 Gromov-Witten 不变量的预测。

**(c)** 讨论稳定映射的模空间 $\overline{M}_{g,n}(X,\beta)$。

## 解答

### (a) Kontsevich 公式

**问题**：设 $N_d$ 为 $\mathbb{P}^2$ 中经过 $3d-1$ 个一般点的有理 $d$ 次曲线的数目。求 $N_d$ 的递推公式。

**定理（Kontsevich, 1994）**：
$$N_d = \sum_{d_1+d_2=d} N_{d_1}N_{d_2}d_1^2d_2^2\left(d_1d_2\binom{3d-4}{3d_1-2} - d_1^2\binom{3d-4}{3d_1-1}\right)$$

等价且更对称的形式为：
$$N_d = \sum_{d_1+d_2=d} N_{d_1}N_{d_2}d_1^2d_2^2\frac{(3d-4)!}{(3d_1-2)!(3d_2-2)!}$$

初值：$N_1 = 1$（直线过 2 点），$N_2 = 1$（圆锥曲线过 5 点），$N_3 = 12$，$N_4 = 620$，$N_5 = 87304$，$\ldots$

**证明思路**：

考虑模空间 $\overline{M}_{0,3d}(\mathbb{P}^2, d)$。固定 $3d-1$ 个一般点 $p_1, \ldots, p_{3d-1} \in \mathbb{P}^2$。定义除子：
$$D_i = ev_i^{-1}(p_i), \quad i = 1, \ldots, 3d-1$$

$N_d$ 是这些除子的交点数（在虚拟基本类上）：
$$N_d = \int_{[\overline{M}_{0,3d}(\mathbb{P}^2,d)]^{vir}} \prod_{i=1}^{3d-1} ev_i^*[p_i]$$

利用 forgetful 映射 $ft: \overline{M}_{0,3d}(\mathbb{P}^2,d) \to \overline{M}_{0,4}$。在 $\overline{M}_{0,4} \cong \mathbb{P}^1$ 上，选取边界除子 $D(12|34)$ 和 $D(13|24)$。它们线性等价，拉回后交积相等。

对 $D(12|34)$，曲线分裂为两分支，度数为 $d_1, d_2$（$d_1+d_2=d$）。第一分支带标记 $\{1,2, 3d_1-1 \text{个一般点}\}$，第二分支带剩余标记。分裂点附加一个约束（在 $\mathbb{P}^2$ 中相交），由 $d_1d_2$ 计数。结合各标记点分配的 combinatorial 因子，经整理即得 Kontsevich 公式。

### (b) 镜面对称的 GW 预测

**镜面对称（Candelas-de la Ossa-Green-Parkes, 1991）**：对 quintic 三维簇 $X \subset \mathbb{P}^4$（5 次超曲面），其镜像伴侣是另一个 Calabi-Yau 三维簇 $Y$，其复模空间上的周期积分给出 $X$ 的 GW 不变量。

**$n_d$ 的计算**：设 $n_d$ 是 $X$ 中度 $d$ 有理曲线的数目（适当的 virtual 计数）。引入生成函数：
$$F(q) = \frac{5}{6}(\log q)^3 + \sum_{d \geq 1} n_d q^d$$

镜面对称预言：$n_d$ 可通过 Picard-Fuchs 方程的解得到。

**Picard-Fuchs 方程**：镜像 $Y$ 的复模参数为 $\psi$，周期 $\varpi(\psi)$ 满足：
$$\left(\theta^4 - 5^5 z\left(\theta + \frac{1}{5}\right)\left(\theta + \frac{2}{5}\right)\left(\theta + \frac{3}{5}\right)\left(\theta + \frac{4}{5}\right)\right)\varpi = 0$$
其中 $\theta = z\frac{d}{dz}$，$z = (5\psi)^{-5}$。

**解与 GW 不变量的联系**：设 $\varpi_0, \varpi_1$ 是基本解。定义 mirror map：
$$q = e^{2\pi i \tau}, \quad \tau = \frac{\varpi_1}{\varpi_0}$$

则 Yukawa coupling：
$$K_{zzz} = \frac{5}{y_0^2(1-5^5z)}\left(\frac{dq}{dz}\right)^{-3}$$

的 $q$-展开给出：
$$K_{zzz} = 5 + \sum_{d \geq 1} \frac{n_d d^3 q^d}{1-q^d}$$

由 $n_1 = 2875$（Clemens 的经典结果），$n_2 = 609250$，$n_3 = 317206375$，$\ldots$

**证明概要（Givental, Lian-Liu-Yau）**：

1. 建立 $X$ 上 GW 势的 $I$-函数和 $J$-函数。
2. 证明 mirror theorem：$I_X = J_Y$（在适当坐标变换下）。
3. 由 $J$-函数的渐近展开提取 GW 不变量。

### (c) 稳定映射的模空间

**定义**：设 $X$ 是光滑射影簇，$\beta \in H_2(X, \mathbb{Z})$。**稳定映射** $(C, p_1, \ldots, p_n; f)$ 由以下组成：
- $(C, p_1, \ldots, p_n)$ 是 $n$-标记 nodal 曲线（算术亏格 $g$）
- $f: C \to X$ 是态射，$f_*[C] = \beta$
- **稳定性**：$f$ 在 $C$ 的每个不可约分量上非平凡，或该分量有足够标记点/节点使其自同构有限

**定理（Kontsevich）**：$\overline{M}_{g,n}(X, \beta)$ 是紧 Deligne-Mumford 栈（proper DM stack），具有虚拟基本类 $[\overline{M}_{g,n}(X, \beta)]^{vir}$，维数为：
$$\dim^{vir} = (\dim X - 3)(1-g) + c_1(X) \cdot \beta + n$$

**紧化机制**：允许曲线 $C$ 退化到有节点的曲线（stable reduction 的类比）。当映射从某分量上塌缩（映射为常值），该分量必须有足够多的特殊点（标记或节点）以保证稳定性。

**虚拟基本类的构造**：

由 Behrend-Fantechi 的内在正常锥（intrinsic normal cone）：

1. 形变理论给出 obstruction sheaf $\mathcal{O}b \to \overline{M}_{g,n}(X, \beta)$。
2. 虚拟维数由形变空间的期望维数减去 obstruction 的维数得到。
3. $[\overline{M}_{g,n}(X, \beta)]^{vir}$ 是 normal cone 到 obstruction bundle 的 intersection 类。

**常见错误**：
- 混淆 $N_d$ 和 $n_d$：$N_d$ 是 $\mathbb{P}^2$ 中的 rational curve 计数；$n_d$ 一般是 Calabi-Yau 三维簇中的 curve 计数（可能涉及 multiple covers 和 bubbling）。
- 忽视 virtual fundamental class：模空间可能有大于期望维数的分支，需用虚拟类才能得到正确数值。
- 将 mirror map 与变量 $q$ 混淆：mirror map 是从 complex modulus $z$ 到 Kähler modulus $q$ 的坐标变换，非线性。
