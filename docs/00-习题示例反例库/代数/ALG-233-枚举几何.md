# 枚举几何

## 题目

**(a)** 计算 P² 中 d 次曲线经过 3d-1 个一般点的个数（Kontsevich 公式）。

**(b)** 证明镜面对称关于 Gromov-Witten 不变量的预测。

**(c)** 讨论稳定映射的模空间 M̄_g,n(X,β)。

## 解答

### (a) Kontsevich 公式

N_d = Σ_{d₁+d₂=d} N_{d₁}N_{d₂}d₁²d₂²(d₁²d₂² - 2)/(d²-1)。

N₁=1, N₂=1, N₃=12, N₄=620, ...

### (b) 镜面对称

 quintic 三维簇的 GW 不变量通过 B 模型（周期积分）计算。

Candelas-de la Ossa-Green-Parkes 公式。

### (c) 稳定映射模空间

Kontsevich 紧化：允许节点曲线，映射稳定（有限自同构）。

虚拟基本类 [M̄_g,n(X,β)]^{vir}。
