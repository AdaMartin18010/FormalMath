# Noether正规化定理

## 题目

**定理**：设 $k$ 是域，$A = k[x_1, \ldots, x_n]/I$ 是有限生成 $k$-代数。

则存在代数无关元 $y_1, \ldots, y_d \in A$ 使得 $A$ 在 $k[y_1, \ldots, y_d]$ 上是整的。

(a) 证明 $A = k[x_1, \ldots, x_n]$ 情形（多项式环，$I = 0$）。

(b) 证明一般情形（用变量替换技巧）。

(c) 应用：证明仿射簇的维数良定义。

## 解答

### (a) 多项式环情形

**证明：**

$A = k[x_1, \ldots, x_n]$，取 $d = n$，$y_i = x_i$。

显然代数无关，且 $A = k[y_1, \ldots, y_n]$（显然整）。$\square$

### (b) 一般情形证明

**归纳法**：

**基础**：$n = 0$，$A = k$，$d = 0$。

**归纳步**：设对 $< n$ 个变量成立。

若 $x_1, \ldots, x_n$ 代数无关，证毕。

否则，存在非零多项式 $f \in I$ 使得 $f(x_1, \ldots, x_n) = 0$。

设 $f = \sum_\alpha c_\alpha x^\alpha$，$\deg f = m$。

**变量替换**：

令 $x_i' = x_i - x_n^{m^{n-i}}$，$i = 1, \ldots, n-1$，$x_n' = x_n$。

代入 $f$：
$$0 = f(x_1' + x_n^{m^{n-1}}, \ldots, x_{n-1}' + x_n^m, x_n)$$

展开后，$x_n$ 的最高次项来自 $x_1^{\alpha_1} \cdots x_n^{\alpha_n}$ 中：

$x_n$ 的贡献为 $\alpha_1 m^{n-1} + \alpha_2 m^{n-2} + \cdots + \alpha_n$。

由 $m$ 进制表示的唯一性，不同单项式给出不同的 $x_n$ 次数。

因此存在唯一的最高次项，$x_n$ 在 $k[x_1', \ldots, x_{n-1}']$ 上整。

由归纳假设，$k[x_1', \ldots, x_{n-1}']$ 有正规化子环。

故 $A$ 整于某个 $k[y_1, \ldots, y_d]$。$\square$

### (c) 维数的良定义性

**定义**：仿射簇 $X$ 的维数是超越次数。

**证明**：

设 $X$ 有两次嵌入，给出两个正规化：

$A$ 整于 $k[y_1, \ldots, y_d]$ 和 $k[z_1, \ldots, z_{d'}]$。

整扩张保持维数（素理想链长度）。

$k[y_1, \ldots, y_d]$ 的维数是 $d$（显然）。

由整扩张的Lying over和Incomparability，$\dim A = d$。

因此 $d = d'$，维数良定义。$\square$

## 证明技巧

1. **变量替换**：巧妙消去最高次项的技巧
2. **$m$ 进制唯一性**：保证最高次项唯一
3. **整扩张性质**：Lying over, Incomparability, Going up

## 常见错误

- ❌ 变量替换中指数选择不当
- ❌ 忘记验证新变量生成的子环
- ❌ 整扩张维数定理的条件检查
