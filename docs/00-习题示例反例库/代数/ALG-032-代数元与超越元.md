## 题目

**(a)** 证明 $\alpha = \sqrt{2} + \sqrt{3}$ 是代数元，并求其极小多项式。

**(b)** 证明 $e$ 和 $\pi$ 中至少有一个是超越数（ Lindemann-Weierstrass 定理的推论）。

**(c)** 设 $F \subset K$ 是域扩张，$\alpha \in K$。证明：$\alpha$ 在 $F$ 上代数当且仅当 $[F(\alpha) : F] < \infty$。

**(d)** 设 $\alpha$ 在 $F$ 上超越，证明 $F(\alpha) \cong F(x)$（有理函数域）。

## 详细解答

### (a) 代数元的极小多项式

**证明：**

设 $\alpha = \sqrt{2} + \sqrt{3}$。

**步骤1：** 消去根号

$$\alpha - \sqrt{2} = \sqrt{3}$$

$$(\alpha - \sqrt{2})^2 = 3$$

$$\alpha^2 - 2\sqrt{2}\alpha + 2 = 3$$

$$\alpha^2 - 1 = 2\sqrt{2}\alpha$$

**步骤2：** 再平方消去剩余根号

$$(\alpha^2 - 1)^2 = 8\alpha^2$$

$$\alpha^4 - 2\alpha^2 + 1 = 8\alpha^2$$

$$\alpha^4 - 10\alpha^2 + 1 = 0$$

**步骤3：** 验证不可约性

$p(x) = x^4 - 10x^2 + 1$。

检查无有理根（有理根只能是 $\pm 1$，验证不成立）。

检查是否可分解为两个二次式：

设 $x^4 - 10x^2 + 1 = (x^2 + ax + b)(x^2 + cx + d)$。

比较系数：$a + c = 0$，$b + d + ac = -10$，$ad + bc = 0$，$bd = 1$。

由 $bd = 1$，设 $b = d = 1$ 或 $b = d = -1$。

若 $b = d = 1$：$2 + a(-a) = -10$，$a^2 = 12$，$a \notin \mathbb{Q}$。

若 $b = d = -1$：$-2 - a^2 = -10$，$a^2 = 8$，$a \notin \mathbb{Q}$。

故不可约。

**结论：** 极小多项式为 $x^4 - 10x^2 + 1$，$[\mathbb{Q}(\alpha) : \mathbb{Q}] = 4$。

**证毕。**

### (b) $e$ 和 $\pi$ 的超越性

**定理（Lindemann-Weierstrass）：** 若 $\alpha_1, ..., \alpha_n$ 是不同的代数数，则 $e^{\alpha_1}, ..., e^{\alpha_n}$ 在 $\mathbb{Q}$ 上线性无关。

**推论：** $e$ 和 $\pi$ 中至少有一个超越。

**证明：**

假设 $e$ 和 $\pi$ 都是代数数。

则 $i\pi$ 也是代数数（$i$ 代数，代数数对乘法封闭）。

但 $e^{i\pi} = -1$，根据Lindemann-Weierstrass，$e^{i\pi}$ 应该是超越数（或至少是超越的），矛盾！

实际上更精确的结论是：$e$ 是超越数（Hermite，1873），$\pi$ 是超越数（Lindemann，1882）。

**证毕。**

### (c) 代数元的等价刻画

**证明：**

**($\Rightarrow$)** 设 $\alpha$ 代数，极小多项式为 $p(x)$，$\deg(p) = n$。

则 $\{1, \alpha, ..., \alpha^{n-1}\}$ 是 $F(\alpha)$ 作为 $F$-向量空间的基。

故 $[F(\alpha) : F] = n < \infty$。

**($\Leftarrow$)** 设 $[F(\alpha) : F] = n < \infty$。

考虑 $1, \alpha, \alpha^2, ..., \alpha^n$，共 $n+1$ 个元素。

因 $\dim_F F(\alpha) = n$，这些元素在 $F$ 上线性相关。

存在不全为零的 $c_0, ..., c_n \in F$ 使：
$$c_0 + c_1\alpha + ... + c_n\alpha^n = 0$$

故 $\alpha$ 是多项式 $c_0 + c_1x + ... + c_nx^n$ 的根，代数。

**证毕。**

### (d) 超越元的有理函数域

**证明：**

设 $\alpha$ 在 $F$ 上超越。

**步骤1：** 同态的构造

定义 $\phi: F[x] \to F[\alpha]$，$\phi(f(x)) = f(\alpha)$。

**步骤2：** 核的分析

$\ker(\phi) = \{f \in F[x] : f(\alpha) = 0\}$。

因 $\alpha$ 超越，$\ker(\phi) = \{0\}$，$\phi$ 是单射。

**步骤3：** 局部化到有理函数域

$F[x]$ 的分式域是 $F(x)$，$F[\alpha]$ 的分式域是 $F(\alpha)$。

$\phi$ 诱导同构 $F(x) \to F(\alpha)$。

**结论：** $F(\alpha) \cong F(x)$。

**证毕。**

## 证明技巧

1. **消去根号：** 逐步分离根号后平方
2. **超越性证明：** 通常是反证法，利用已知超越数定理
3. **域同构的构造：** 从多项式环的同态开始，扩展到分式域

## 常见错误

| 错误 | 说明 |
|------|------|
| 找到多项式后不验证不可约 | 必须是极小多项式 |
| 混淆代数闭包与代数扩张 | 代数闭包是代数扩张的极大元 |
| 超越元的域扩张描述错误 | $F(\alpha) \cong F(x)$ 而非 $F[x]$ |

## 变式练习

**变式1：** 证明 $\sqrt{2} + \sqrt[3]{3}$ 在 $\mathbb{Q}$ 上的次数为6。

**变式2：** 证明 $e^{\sqrt{2}}$ 是超越数（Gelfond-Schneider定理）。

**变式3：** 设 $\alpha$ 超越，$\beta$ 代数，证明 $\alpha + \beta$ 超越。
