# 相交理论

## 题目

**(a)** 定义 Chow 环 A*(X) 并证明其基本性质。

**(b)** 证明 Bézout 定理：P^n 中 d 次和 e 次超曲面横截相交于 de 个点。

**(c)** 讨论相交积的投影公式和自交公式。

## 解答

### (a) Chow 环

A_k(X) = k-维环面类的群，模有理等价。

相交积：A^i(X) × A^j(X) → A^{i+j}(X)，由横截相交定义后延拓。

### (b) Bézout 定理

H ∈ A^1(P^n) 是超平面类。V(f) ∼ dH，V(g) ∼ eH。

[V(f)]·[V(g)] = de H^n = de [pt]。

### (c) 投影公式

f: X → Y，f_*(α·f^*β) = f_*α · β。自交：D² = deg(N_{D/X})。
