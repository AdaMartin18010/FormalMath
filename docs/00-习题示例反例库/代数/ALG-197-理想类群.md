# 理想类群

## 题目

**(a)** 证明理想类群 $\text{Cl}(K)$ 有限（Minkowski 界）。

**(b)** 计算 $\mathbb{Q}(\sqrt{-23})$ 的类数和类群结构。

**(c)** 讨论类数公式和解析方法。

## 解答

### (a) 类数有限性的证明

**定义**：设 $K$ 是数域，$[K:\mathbb{Q}] = n$。$\mathcal{O}_K$ 的**分式理想**是 $\mathcal{O}_K$-子模 $\mathfrak{a} \subset K$，使得存在非零 $d \in \mathcal{O}_K$ 满足 $d\mathfrak{a} \subset \mathcal{O}_K$。

两个非零分式理想 $\mathfrak{a}, \mathfrak{b}$ **等价**（记作 $\mathfrak{a} \sim \mathfrak{b}$），若存在非零 $\alpha \in K^\times$ 使得 $\mathfrak{a} = \alpha\mathfrak{b}$。等价类构成**理想类群** $\text{Cl}(K)$，其运算由理想乘法诱导。

**定理（Minkowski）**：$\text{Cl}(K)$ 是有限群。其阶 $h_K = |\text{Cl}(K)|$ 称为 $K$ 的**类数**。

**证明**：

**步骤 1：Minkowski 界**

**定理**：每个理想类包含一个整理想 $\mathfrak{a}$（即 $\mathfrak{a} \subset \mathcal{O}_K$），满足范数界：
$$N(\mathfrak{a}) \leq \frac{n!}{n^n}\left(\frac{4}{\pi}\right)^{r_2}\sqrt{|d_K|} =: M_K$$

其中 $d_K$ 是 $K$ 的判别式，$r_2$ 是复嵌入对数，$M_K$ 称为 **Minkowski 界**。

*证明*：设 $\mathfrak{a}$ 是整理想。考虑嵌入 $\sigma: K \to \mathbb{R}^{r_1} \times \mathbb{C}^{r_2} \cong \mathbb{R}^n$。像 $\sigma(\mathfrak{a})$ 是 $\mathbb{R}^n$ 中的格，其基本区域的体积为 $2^{-r_2}N(\mathfrak{a})\sqrt{|d_K|}$。

考虑中心对称凸集：
$$S = \{(x_1, \ldots, x_{r_1}, z_1, \ldots, z_{r_2}) : \sum_{i=1}^{r_1}|x_i| + 2\sum_{j=1}^{r_2}|z_j| \leq t\}$$

其体积为 $\text{Vol}(S) = 2^{r_1}\left(\frac{\pi}{2}\right)^{r_2}\frac{t^n}{n!}$。

由 Minkowski 格点定理，若 $\text{Vol}(S) \geq 2^n \cdot \text{covolume}$，则 $S$ 含非零格点。取：
$$t^n = n!\left(\frac{4}{\pi}\right)^{r_2}N(\mathfrak{a})\sqrt{|d_K|}$$

则存在非零 $\alpha \in \mathfrak{a}$ 满足上述不等式。计算其范数：
$$|N_{K/\mathbb{Q}}(\alpha)| = \prod_{i=1}^{r_1}|\sigma_i(\alpha)| \prod_{j=1}^{r_2}|\sigma_{r_1+j}(\alpha)|^2$$

由 AM-GM 不等式及 $\alpha$ 满足的约束：
$$|N(\alpha)|^{1/n} \leq \frac{1}{n}\left(\sum_{i=1}^{r_1}|\sigma_i(\alpha)| + 2\sum_{j=1}^{r_2}|\sigma_{r_1+j}(\alpha)|\right) \leq \frac{t}{n}$$

故 $|N(\alpha)| \leq t^n/n^n = M_K \cdot N(\mathfrak{a})$。

令 $\mathfrak{b} = \alpha\mathfrak{a}^{-1}$，这是整理想（因 $\alpha \in \mathfrak{a}$），且 $N(\mathfrak{b}) = |N(\alpha)|/N(\mathfrak{a}) \leq M_K$。∎

**步骤 2：有限性**

因范数 $N(\mathfrak{a})$ 是正整数，且对每个 $m$，满足 $N(\mathfrak{a}) = m$ 的整理想只有有限多个（由素理想的唯一分解），故界 $M_K$ 以下的整理想只有有限个。因此 $\text{Cl}(K)$ 有限。∎

### (b) $\mathbb{Q}(\sqrt{-23})$ 的类群

**设定**：$K = \mathbb{Q}(\sqrt{-23})$，$n = 2$，$r_1 = 0$，$r_2 = 1$。判别式 $d_K = -23$（因 $-23 \equiv 1 \pmod 4$）。

**Minkowski 界**：
$$M_K = \frac{2!}{2^2}\left(\frac{4}{\pi}\right)^1\sqrt{23} = \frac{1}{2} \cdot \frac{4}{\pi} \cdot \sqrt{23} = \frac{2\sqrt{23}}{\pi} \approx \frac{2 \times 4.796}{3.1416} \approx 3.05$$

故只需检查素数 $p = 2, 3$。

**素数 2 的分解**：因 $-23 \equiv 1 \pmod 8$，由 Dedekind 分解定理，$x^2 - x + 6 \equiv x^2 + x \pmod 2$ 有根（$x = 0, 1$），故：
$$(2) = \mathfrak{p}_2 \bar{\mathfrak{p}}_2, \quad N(\mathfrak{p}_2) = 2$$

具体地，$\mathfrak{p}_2 = (2, \frac{1+\sqrt{-23}}{2})$，因 $\frac{1+\sqrt{-23}}{2} \cdot \frac{1-\sqrt{-23}}{2} = \frac{1+23}{4} = 6$。

**素数 3 的分解**：$-23 \equiv 1 \pmod 3$，故 $x^2 + 23 \equiv x^2 + 2 \pmod 3$。检验：$0^2 = 0, 1^2 = 1, 2^2 = 1$，无根。故 $(3)$ 是惰性的（inert），$\mathfrak{p}_3 = (3)$，$N(\mathfrak{p}_3) = 9 > M_K$，不贡献新的理想类。

**类群结构**：理想类由 $\mathfrak{p}_2$ 生成。计算 $\mathfrak{p}_2$ 的阶：
- $\mathfrak{p}_2^1$：范数 2，非主理想（因 $N(a + b\frac{1+\sqrt{-23}}{2}) = a^2 + ab + 6b^2 = 2$ 无整数解）。
- $\mathfrak{p}_2^2$：范数 4。方程 $a^2 + ab + 6b^2 = 4$：$b = 0$ 时 $a^2 = 4$，$a = \pm 2$。故 $(2) = \mathfrak{p}_2\bar{\mathfrak{p}}_2$，但 $\mathfrak{p}_2^2$ 不是主理想（若 $\mathfrak{p}_2^2 = (\alpha)$，则 $\alpha/2$ 给出矛盾）。实际上 $a^2 + ab + 6b^2 = 4$ 的解 $(\pm 2, 0)$ 对应 $(2)$，而 $\mathfrak{p}_2^2 \neq (2)$。

更精确计算：利用 $2 = \mathfrak{p}_2\bar{\mathfrak{p}}_2$，且 $\mathfrak{p}_2 \neq \bar{\mathfrak{p}}_2$。

尝试 $a^2 + ab + 6b^2 = 8$（$\mathfrak{p}_2^3$ 的范数）：$b = 1$ 时 $a^2 + a + 6 = 8$，$a^2 + a - 2 = 0$，$a = 1$ 或 $-2$。$a = 1, b = 1$：$1 + 1 + 6 = 8$。故 $\mathfrak{p}_2^3 = (\frac{3+\sqrt{-23}}{2})$ 是主理想？验证：$(\frac{3+\sqrt{-23}}{2})(\frac{3-\sqrt{-23}}{2}) = \frac{9+23}{4} = 8$。

因此 $\mathfrak{p}_2$ 的阶为 3，$\text{Cl}(K) \cong \mathbb{Z}/3\mathbb{Z}$，类数 $h_K = 3$。

### (c) 解析类数公式

**Dedekind zeta 函数**：
$$\zeta_K(s) = \sum_{\mathfrak{a} \subset \mathcal{O}_K} \frac{1}{N(\mathfrak{a})^s} = \prod_{\mathfrak{p}}\left(1 - \frac{1}{N(\mathfrak{p})^s}\right)^{-1}, \quad \text{Re}(s) > 1$$

**解析类数公式**：
$$\lim_{s \to 1}(s-1)\zeta_K(s) = \frac{2^{r_1}(2\pi)^{r_2} R_K h_K}{w_K\sqrt{|d_K|}}$$

其中：
- $R_K$ 是调节子（regulator），由基本单位的对数嵌入生成
- $h_K = |\text{Cl}(K)|$ 是类数
- $w_K = |\mu(K)|$ 是单位根个数
- $d_K$ 是判别式

**对虚二次域的简化**：若 $K$ 是虚二次域（$r_1 = 0, r_2 = 1$），则 $R_K = 1$（无自由单位），$w_K$ 通常为 2（$K \neq \mathbb{Q}(i), \mathbb{Q}(\zeta_3)$）。公式简化为：
$$\text{Res}_{s=1}\zeta_K(s) = \frac{2\pi h_K}{w_K\sqrt{|d_K|}}$$

**Dirichlet 的 $L$-函数表达**：对二次域 $K = \mathbb{Q}(\sqrt{D})$，
$$\zeta_K(s) = \zeta(s)L(s, \chi_D)$$
其中 $\chi_D$ 是 Kronecker 符号。因 $\text{Res}_{s=1}\zeta(s) = 1$，$L(1, \chi_D) = \frac{2\pi h_K}{w_K\sqrt{|d_K|}}$。

对 $D = -23$：
$$L(1, \chi_{-23}) = \sum_{n=1}^\infty \left(\frac{-23}{n}\right)\frac{1}{n} = \frac{2\pi \cdot 3}{2 \cdot \sqrt{23}} = \frac{3\pi}{\sqrt{23}}$$

**常见错误**：
- 在实二次域中忽略调节子 $R_K$，认为 $h_K$ 仅由 $L(1,\chi)$ 决定。
- 对虚二次域使用错误的 $w_K$：$\mathbb{Q}(i)$ 有 $w = 4$，$\mathbb{Q}(\zeta_3)$ 有 $w = 6$。
