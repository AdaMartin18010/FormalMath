# 理想类群

## 题目

**(a)** 证明理想类群 Cl(K) 有限（Minkowski 界）。

**(b)** 计算 Q(√-23) 的类数和类群结构。

**(c)** 讨论类数公式和解析方法。

## 解答

### (a) 类数有限性

Minkowski 界：每个理想类包含整理想 a，满足
$$N(a) \leq \frac{n!}{n^n}\left(\frac{4}{\pi}\right)^{r_2}\sqrt{|d_K|}$$

有限个可能，故 Cl(K) 有限。

### (b) Q(√-23)

判别式 -23，Minkowski 界 < 4。检查素数 2, 3。

类数3：Cl(K) ≅ ℤ/3ℤ。

### (c) 类数公式

$$
\lim_{s\to 1}(s-1)\zeta_K(s) = \frac{2^{r_1}(2\pi)^{r_2}R_K h_K}{w\sqrt{|d_K|}}$$
