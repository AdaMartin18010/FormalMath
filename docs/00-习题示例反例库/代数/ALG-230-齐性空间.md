# 齐性空间

## 题目

**(a)** 证明 Grassmann 簇 G(k,n) 的 Plücker 嵌入。

**(b)** 讨论旗簇和 Schubert 簇。

**(c)** 证明 Bott 公式：旗簇上线丛的上同调消失。

## 解答

### (a) Plücker 嵌入

G(k,n) = {ℂ^n 的 k-维子空间}。

Plücker：V ↦ [e₁∧···∧e_k] ∈ P(Λ^kℂ^n)。

像满足 Plücker 关系。

### (b) 旗簇

F(k₁,...,k_r; n) = {0 ⊂ V₁ ⊂ ··· ⊂ V_r ⊂ ℂ^n, dim V_i = k_i}。

Schubert 簇：固定标志的 Schubert 条件定义的子簇。

### (c) Bott 公式

对非支配权 λ，H^i(G/B, L_λ) = 0 对所有 i。

对支配权，仅一个 i 使得 H^i ≠ 0。
