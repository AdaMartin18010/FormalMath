# 层上同调理论

## 题目

**(a)** 定义层上同调 H^i(X, F) 并证明其基本性质。

**(b)** 证明 Čech 上同调与导出函子上同调的关系。

**(c)** 讨论 Serre 对偶定理。

## 解答

### (a) 层上同调

导出函子：H^i(X, F) = R^iΓ(X, F)。长正合列、维数移动。

### (b) Čech 上同调

对仿紧空间和好覆盖，Ȟ^q(X, F) ≅ H^q(X, F)。

Leray 定理：若覆盖是 F-无环的，则 Čech 上同调正确。

### (c) Serre 对偶

光滑射影簇 X 维 n，F 局部自由层：
$$H^i(X, F) \cong H^{n-i}(X, F^* \otimes K_X)^*$$
