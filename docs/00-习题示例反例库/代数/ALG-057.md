# Tor函子与张量积

## 题目

设 $R$ 是环，$M$ 是右 $R$-模，$N$ 是左 $R$-模。

**定义**：$\text{Tor}_n^R(M, N) = H_n(P_\bullet \otimes_R N)$，其中 $P_\bullet \to M$ 是投射分解。

(a) 证明 $\text{Tor}_0^R(M, N) \cong M \otimes_R N$。

(b) 若 $0 \to A \to B \to C \to 0$ 是右 $R$-模的短正合列，构造长正合列：
$$\cdots \to \text{Tor}_1(C, N) \to A \otimes N \to B \otimes N \to C \otimes N \to 0$$

(c) 证明 $N$ 是平坦模当且仅当对所有 $n \geq 1$ 和所有 $M$，$\text{Tor}_n^R(M, N) = 0$。

(d) 计算 $\text{Tor}_1^{\mathbb{Z}}(\mathbb{Z}/m\mathbb{Z}, \mathbb{Z}/n\mathbb{Z})$。

## 解答

### (a) $\text{Tor}_0$ 的计算

**证明：**

设 $P_\bullet: \cdots \to P_1 \xrightarrow{d_1} P_0 \to M \to 0$。

张量积复形：
$$\cdots \to P_1 \otimes N \xrightarrow{d_1 \otimes \text{id}} P_0 \otimes N \to 0$$

$$\text{Tor}_0(M, N) = H_0 = \text{coker}(d_1 \otimes \text{id}) = \text{coker}(P_1 \otimes N \to P_0 \otimes N)$$

由张量积的右正合性：
$$P_1 \otimes N \to P_0 \otimes N \to M \otimes N \to 0$$

故 $\text{Tor}_0(M,N) \cong M \otimes N$。 $\square$

### (b) 长正合列

**证明概要：**

由于每个 $P_n$ 投射（从而平坦），$P_n \otimes -$ 保持正合性。

因此 $0 \to P_\bullet \otimes A \to P_\bullet \otimes B \to P_\bullet \otimes C \to 0$ 是复形的短正合列。

由同调的长正合列即得。连接同态：
$$\partial: \text{Tor}_1(C, N) \to A \otimes N$$

的构造通过蛇引理在复形层面追踪。 $\square$

### (c) 平坦模的Tor判别法

**证明：**

**($\Rightarrow$)：** 设 $N$ 平坦，取 $M$ 的投射分解 $P_\bullet$。

由于 $N$ 平坦，$P_\bullet \otimes N$ 正合（除 $P_0 \otimes N \to M \otimes N$ 外）。

因此 $H_n = 0$ 对 $n \geq 1$。

**($\Leftarrow$)：** 设 $\text{Tor}_1(M, N) = 0$ 对所有 $M$。

对任意单射 $A \to B$，由(b)：
$$\text{Tor}_1(B/A, N) = 0 \to A \otimes N \to B \otimes N$$

故 $A \otimes N \to B \otimes N$ 单，$N$ 平坦。 $\square$

### (d) Tor的具体计算

**解答：**

$\mathbb{Z}/m\mathbb{Z}$ 的投射分解：
$$0 \to \mathbb{Z} \xrightarrow{\cdot m} \mathbb{Z} \to \mathbb{Z}/m\mathbb{Z} \to 0$$

张量 $N = \mathbb{Z}/n\mathbb{Z}$：
$$0 \to \mathbb{Z} \otimes N \xrightarrow{m \otimes \text{id}} \mathbb{Z} \otimes N \to 0$$

即：
$$0 \to \mathbb{Z}/n\mathbb{Z} \xrightarrow{\cdot m} \mathbb{Z}/n\mathbb{Z} \to 0$$

因此：
$$\text{Tor}_1 = \ker(\cdot m) = \{x \in \mathbb{Z}/n\mathbb{Z} : mx \equiv 0 \pmod{n}\}$$

$$= \frac{n}{\gcd(m,n)}\mathbb{Z}/n\mathbb{Z} \cong \mathbb{Z}/\gcd(m,n)\mathbb{Z}$$

$\square$

## 证明技巧

1. **张量积的右正合性**：Tor测量了张量积左正合性的失败
2. **具体计算**：用周期为2的投射分解计算Abel群的Tor
3. **平坦=正合**：Tor消失等价于张量积保持正合

## 常见错误

- ❌ 混淆左右模的张量积定义域
- ❌ 在(d)中错误计算核的结构
- ❌ 忘记Tor对两个变量都是协变的

## 变式练习

**变式1：** 证明 $\text{Tor}_n^R(M, N) \cong \text{Tor}_n^R(N, M)$ 当 $R$ 交换时。

**变式2：** 若 $R$ 是主理想整环，证明 $\text{Tor}_n^R = 0$ 对 $n \geq 2$。
