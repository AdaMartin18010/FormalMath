# 谱序列基础

## 题目

**(a)** 定义谱序列 $(E^r, d^r)$，其中 $d^r: E^r_{p,q} \to E^r_{p-r, q+r-1}$ 满足 $d^r \circ d^r = 0$，且 $E^{r+1} = H(E^r, d^r)$。

**(b)** 设 $C_{\bullet, \bullet}$ 是第一象限双复形，说明如何用两个谱序列计算 $H_n(\text{Tot}(C))$。

**(c)** 用谱序列证明Grothendieck谱序列：设 $F: \mathcal{A} \to \mathcal{B}$，$G: \mathcal{B} \to \mathcal{C}$ 是函子，$F$ 映内射对象为 $G$-零调对象，则
$$E^2_{p,q} = (L_p G)((L_q F)(A)) \Rightarrow L_{p+q}(GF)(A)$$

## 解答

### (a) 谱序列定义

**解答：**

谱序列是复形的序列，每页由前一页的同调定义。

收敛：若对所有 $(p,q)$，当 $r$ 大时 $E^r_{p,q}$ 稳定，记为 $E^\infty_{p,q}$。

滤过：$H_n$ 有滤过 $0 = F_{-1} \subset F_0 \subset \cdots \subset F_n = H_n$ 使
$$E^\infty_{p,q} \cong F_p / F_{p-1}$$
其中 $p+q = n$。

### (b) 双复形的两个谱序列

**解答：**

谱序列1（先垂直）：
$$E^1_{p,q} = H_q^v(C_{p, \bullet}), \quad E^2_{p,q} = H_p^h(H_q^v(C))$$

谱序列2（先水平）：
$$'E^1_{p,q} = H_p^h(C_{\bullet, q}), \quad 'E^2_{p,q} = H_q^v(H_p^h(C))$$

两谱序列都收敛到 $H_{p+q}(\text{Tot}(C))$。

### (c) Grothendieck谱序列

**证明概要：**

取 $A$ 的内射分解 $I^\bullet$，$F(I^\bullet)$ 是 $G$-零调分解。

构造Cartan-Eilenberg分解，得双复形。

用比较两个谱序列得到结果。
