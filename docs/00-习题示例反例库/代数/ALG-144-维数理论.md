# Krull维数与正则局部环

## 题目

设 $R$ 是Noether环。

**(a)** 定义Krull维数 $\dim R$ 为素理想链长度的上确界。证明 $\dim R[x] = \dim R + 1$。

**(b)** 设 $(R, \mathfrak{m})$ 是Noether局部环，证明 $\dim R \leq \dim_{R/\mathfrak{m}} \mathfrak{m}/\mathfrak{m}^2$。

**(c)** 定义正则局部环：等号成立。证明正则局部环是整环、唯一分解整环。

---

## 解答

### (a) 多项式环维数

**证明：**

**上界**：设 $\mathfrak{p}_0 \subset \cdots \subset \mathfrak{p}_n$ 是 $R[x]$ 中链。

$\mathfrak{p}_i \cap R$ 是 $R$ 中素理想链，长度 $\leq \dim R$。

严格上升需纤维维数增加，总长度 $\leq \dim R + 1$。

**详细展开**：

**上界证明**：设 $\mathfrak{P}_0 \subset \mathfrak{P}_1 \subset \cdots \subset \mathfrak{P}_n$ 是 $R[x]$ 中的素理想严格链。

考虑与 $R$ 的交：$\mathfrak{p}_i = \mathfrak{P}_i \cap R$。这是 $R$ 中的素理想链：
$$\mathfrak{p}_0 \subseteq \mathfrak{p}_1 \subseteq \cdots \subseteq \mathfrak{p}_n$$

**关键引理**：若 $\mathfrak{P} \subsetneq \mathfrak{Q}$ 是 $R[x]$ 中的素理想，且 $\mathfrak{P} \cap R = \mathfrak{Q} \cap R = \mathfrak{p}$，则 $\mathfrak{Q}$ 对应于 $(R/\mathfrak{p})[x]$ 中的非零素理想。

由于 $(R/\mathfrak{p})[x]$ 是PID上的多项式环（当 $R/\mathfrak{p}$ 是整环时），其在 $0$ 以上的素理想链长度至多为1。

因此，在 $\mathfrak{P}_0 \subset \cdots \subset \mathfrak{P}_n$ 中，与 $R$ 交相同的相邻素理想对至多有一组。

故 $n \leq \dim R + 1$。

**下界**：$R$ 中有长为 $\dim R$ 的链，$\mathfrak{p}_i[x] \subset \mathfrak{p}_i[x] + (x)$ 更长。

**详细证明**：设 $\mathfrak{p}_0 \subset \cdots \subset \mathfrak{p}_d$ 是 $R$ 中长度为 $d = \dim R$ 的素理想链。

在 $R[x]$ 中：
$$\mathfrak{p}_0[x] \subset \mathfrak{p}_1[x] \subset \cdots \subset \mathfrak{p}_d[x] \subset \mathfrak{p}_d[x] + (x)$$

验证这是严格链：
- $\mathfrak{p}_i[x] \subsetneq \mathfrak{p}_{i+1}[x]$ 由 $\mathfrak{p}_i \subsetneq \mathfrak{p}_{i+1}$ 保证
- $\mathfrak{p}_d[x] \subsetneq \mathfrak{p}_d[x] + (x)$：$x \notin \mathfrak{p}_d[x]$

长度 $d + 1 = \dim R + 1$，故 $\dim R[x] \geq \dim R + 1$。

综合上下界，$\dim R[x] = \dim R + 1$。$\square$

### (b) 维数不等式

**证明：**

$\mathfrak{m}$ 可由 $d = \dim_{k} \mathfrak{m}/\mathfrak{m}^2$ 个元生成。

Krull主理想定理：$d$ 个元生成的理想的极小素理想高度 $\leq d$。

$\mathfrak{m}$ 是唯一的，故 $\dim R = \text{ht}(\mathfrak{m}) \leq d$。

**详细展开**：

**Zariski切空间**：设 $k = R/\mathfrak{m}$。向量空间 $\mathfrak{m}/\mathfrak{m}^2$ 称为 $R$ 的**Zariski切空间**（对偶于 $k$-导子空间）。其维数 $d$ 是 $\mathfrak{m}$ 的极小生成元个数（由Nakayama引理）。

**Krull主理想定理**：设 $R$ 是Noether环，$x \in R$。则 $x$ 生成的理想的任一极小素理想的高度至多为1。

推广：由 $d$ 个元生成的理想，其极小素理想的高度至多为 $d$。

**应用到本题**：$\mathfrak{m}$ 由 $d$ 个元 $x_1, ..., x_d$ 生成。由Krull主理想定理的推广，$\text{ht}(\mathfrak{m}) \leq d$。

但 $\dim R = \text{ht}(\mathfrak{m})$（局部环的维数等于极大理想的高度）。

因此 $\dim R \leq d = \dim_k \mathfrak{m}/\mathfrak{m}^2$。$\square$

### (c) 正则局部环性质

**证明：**

**整环性**：用归纳和关联素理想论证。

**详细证明**：设 $(R, \mathfrak{m})$ 是 $d$-维正则局部环，$x_1, ..., x_d$ 是 $\mathfrak{m}$ 的极小生成集（正则参数系）。

**步骤1：$x_1$ 是非零因子**

由正则性，$R$ 是Cohen-Macaulay的（深度 = 维数 = $d$）。因此 $x_1, ..., x_d$ 是 $R$-正则序列。

特别地，$x_1$ 不是零因子。

**步骤2：归纳**

$R/(x_1)$ 是 $(d-1)$-维正则局部环。由归纳，$R/(x_1)$ 是整环。

**步骤3：$R$ 是整环**

设 $ab = 0$。在 $R/(x_1)$ 中，$\bar{a}\bar{b} = 0$。由于 $R/(x_1)$ 是整环，不妨设 $\bar{a} = 0$。

则 $a \in (x_1)$，即 $a = x_1 a'$。

$x_1 a' b = 0$，由于 $x_1$ 不是零因子，$a'b = 0$。

重复此过程。若 $a \neq 0$，则得到无限递降序列 $a, a', a'', ...$ 使得 $a^{(n)} \in \mathfrak{m}^n$。

由Krull交定理，$\bigcap_n \mathfrak{m}^n = 0$，故 $a = 0$。

因此 $R$ 是整环。

**UFD性（Auslander-Buchsbaum）**：

正则局部环的同调维数有限（等于维数）。模有有限投射分解，故任意高度1素理想自由，主理想。

**详细解释**：

**Auslander-Buchsbaum定理**：正则局部环是唯一分解整环（UFD）。

证明思路：
1. 正则局部环的**整体维数**等于Krull维数 $d$
2. 对高度1的素理想 $\mathfrak{p}$，$R/\mathfrak{p}$ 有有限投射分解
3. 由**Bass猜想**的相关结果，$\mathfrak{p}$ 是自由模，因此是主理想
4. 所有高度1素理想都是主理想 $\Rightarrow$ $R$ 是UFD（Nagata判据）

$\square$

## 维数理论的图景

### 维数的多种定义

对Noether局部环 $(R, \mathfrak{m})$，以下定义等价：
1. **Krull维数**：素理想链的最大长度
2. **Chevalley维数**：$\inf\{n \mid \exists x_1, ..., x_n \in \mathfrak{m}, \mathfrak{m}^N \subseteq (x_1, ..., x_n)\}$
3. **Hilbert-Samuel维数**：Hilbert-Samuel多项式的次数

### 正则局部环的几何意义

正则局部环对应于**光滑代数簇**的局部。$(R, \mathfrak{m})$ 正则意味着对应的概形在点 $\mathfrak{m}$ 处是光滑的——没有奇点。

- **维数0**：域 $k$
- **维数1**：DVR（离散赋值环），如 $\mathbb{Z}_{(p)}$, $k[[t]]$
- **维数2**：二维正则局部环，如 $k[[x, y]]$
