# ALG-001: 群的定义验证

**题号**: ALG-001
**难度**: ⭐⭐ (Level 2)
**类型**: 证明型 (PRF)
**来源**: MIT 18.701 Chapter 2.1
**主题**: 群的定义验证

---

## 题目

**群的定义**: 群 $(G, \cdot)$ 是一个集合 $G$ 配上一个二元运算 $\cdot: G \times G \to G$，满足：
- **G1 (封闭性)**: 对所有 $a, b \in G$，有 $a \cdot b \in G$
- **G2 (结合律)**: 对所有 $a, b, c \in G$，有 $(a \cdot b) \cdot c = a \cdot (b \cdot c)$
- **G3 (单位元)**: 存在 $e \in G$，使得对所有 $a \in G$，有 $a \cdot e = e \cdot a = a$
- **G4 (逆元)**: 对每个 $a \in G$，存在 $a^{-1} \in G$，使得 $a \cdot a^{-1} = a^{-1} \cdot a = e$

---

### 问题

判定下列集合与运算是否构成群。若是，指出单位元与逆元；若不是，说明不满足哪条公理。

**(a)** $(\mathbb{Z}, +)$ —— 整数集与普通加法

**(b)** $(\mathbb{Z}, \cdot)$ —— 整数集与普通乘法

**(c)** $(\mathbb{Q}^*, \cdot)$ —— 非零有理数集与乘法

**(d)** $(\mathbb{Z}_n, +)$ —— 模 $n$ 剩余类与模 $n$ 加法

**(e)** 设 $G = \mathbb{R} \setminus \{-1\}$，定义运算 $a * b = a + b + ab$。$(G, *)$ 是否构成群？

---

## 解答

### (a) $(\mathbb{Z}, +)$ 是群

**验证**:

| 公理 | 验证 |
|------|------|
| 封闭性 | $m, n \in \mathbb{Z} \Rightarrow m + n \in \mathbb{Z}$ ✓ |
| 结合律 | $(m + n) + p = m + (n + p)$，整数加法结合律 ✓ |
| 单位元 | $e = 0$，因为 $n + 0 = 0 + n = n$ ✓ |
| 逆元 | $n^{-1} = -n$，因为 $n + (-n) = 0$ ✓ |

**结论**: $(\mathbb{Z}, +)$ 是群（无限循环群）。 $\square$

---

### (b) $(\mathbb{Z}, \cdot)$ 不是群

**验证**:

| 公理 | 验证 |
|------|------|
| 封闭性 | $m \cdot n \in \mathbb{Z}$ ✓ |
| 结合律 | 整数乘法结合律 ✓ |
| 单位元 | $e = 1$ ✓ |
| **逆元** | **不满足** |

**反例**: $2 \in \mathbb{Z}$，但不存在 $n \in \mathbb{Z}$ 使得 $2 \cdot n = 1$。

**结论**: $(\mathbb{Z}, \cdot)$ 不是群。它构成一个幺半群（monoid）。 $\square$

---

### (c) $(\mathbb{Q}^*, \cdot)$ 是群

其中 $\mathbb{Q}^* = \mathbb{Q} \setminus \{0\}$

**验证**:

| 公理 | 验证 |
|------|------|
| 封闭性 | $p/q \cdot r/s = pr/qs$，若 $p, r \neq 0$，则 $pr \neq 0$ ✓ |
| 结合律 | 有理数乘法结合律 ✓ |
| 单位元 | $e = 1 = 1/1$ ✓ |
| 逆元 | $(p/q)^{-1} = q/p \in \mathbb{Q}^*$ ✓ |

**结论**: $(\mathbb{Q}^*, \cdot)$ 是群（乘法阿贝尔群）。 $\square$

---

### (d) $(\mathbb{Z}_n, +)$ 是群

其中 $\mathbb{Z}_n = \{0, 1, 2, \ldots, n-1\}$，运算为模 $n$ 加法。

**验证**:

| 公理 | 验证 |
|------|------|
| 封闭性 | $\bar{a} + \bar{b} = \overline{a+b \pmod n} \in \mathbb{Z}_n$ ✓ |
| 结合律 | 由整数加法结合律继承 ✓ |
| 单位元 | $e = \bar{0}$ ✓ |
| 逆元 | $\bar{k}^{-1} = \overline{n-k}$，因为 $\bar{k} + \overline{n-k} = \bar{n} = \bar{0}$ ✓ |

**结论**: $(\mathbb{Z}_n, +)$ 是 $n$ 阶循环群。 $\square$

---

### (e) $(G, *)$ 是群

其中 $G = \mathbb{R} \setminus \{-1\}$，$a * b = a + b + ab$

**验证**:

**Step 1: 封闭性**

需验证 $a, b \neq -1 \Rightarrow a * b \neq -1$。

若 $a * b = -1$，则 $a + b + ab = -1$，即 $a + b + ab + 1 = 0$，$(1+a)(1+b) = 0$。

这意味着 $a = -1$ 或 $b = -1$，矛盾。因此封闭性 ✓

**Step 2: 结合律**

$$
\begin{aligned}
(a * b) * c &= (a + b + ab) * c \\ &= (a + b + ab) + c + (a + b + ab)c \\ &= a + b + c + ab + ac + bc + abc
\end{aligned}
$$

$$
\begin{aligned}
a * (b * c) &= a * (b + c + bc) \\ &= a + (b + c + bc) + a(b + c + bc) \\ &= a + b + c + ab + ac + bc + abc
\end{aligned}
$$

两者相等，结合律 ✓

**Step 3: 单位元**

找 $e$ 使得 $a * e = a$ 对所有 $a$ 成立：

$$a + e + ae = a \Rightarrow e(1 + a) = 0$$

由于 $a \neq -1$，必须有 $e = 0$。验证：$a * 0 = a + 0 + a \cdot 0 = a$ ✓

**Step 4: 逆元**

对 $a \in G$，找 $b$ 使得 $a * b = 0$：

$$a + b + ab = 0 \Rightarrow b(1 + a) = -a \Rightarrow b = -\frac{a}{1+a}$$

由于 $a \neq -1$，$b$ 良定义。还需验证 $b \neq -1$：

若 $-\frac{a}{1+a} = -1$，则 $a = 1+a$，矛盾。因此 $b \in G$。 $\square$

## 教学要点

### 证明技巧总结

| 技巧 | 应用位置 | 说明 |
|------|----------|------|
| 逐条验证 | 全部 | 群的四个公理必须全部满足 |
| 反例法 | (b)部分 | 证明不是群只需找出一个违反公理的例子 |
| 公式变形 | (e)部分 | $a+b+ab = (1+a)(1+b) - 1$，此形式便于计算 |
| 待定系数 | (e)部分 | 设单位元/逆元为未知数，解方程确定 |

### 关键洞察

1. **运算形式转换**: 令 $\phi(x) = 1 + x$，则 $\phi(a * b) = \phi(a) \cdot \phi(b)$，即 $(G, *) \cong (\mathbb{R}^*, \cdot)$
2. **零元排除**: 在(c)中必须排除0，在(e)中必须排除-1（对应标准乘法中的"零"）

## 常见错误

### 错误1: 忘记验证封闭性

❌ **错误做法**:
> "$(\mathbb{Z}, +)$ 显然是群，因为加法有结合律、单位元、逆元"

✅ **正确理解**:
> 封闭性也需验证！虽然整数加法封闭性显然，但在(e)这类非标准运算中至关重要。

### 错误2: 误认为乘法单位元是0

❌ **错误做法**:
> "$(\mathbb{Z}, \cdot)$ 的单位元是0"

✅ **正确理解**:
> 乘法单位元是1：$a \cdot 1 = a$。0是加法单位元。

### 错误3: 忽略逆元存在的条件

❌ **错误做法**:
> "在$\mathbb{Z}$中，2的逆元是1/2"

✅ **正确理解**:
> $1/2 \notin \mathbb{Z}$，所以2在$(\mathbb{Z}, \cdot)$中没有逆元。
