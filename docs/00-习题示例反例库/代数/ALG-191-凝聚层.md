# 凝聚层

## 题目

**(a)** 定义凝聚层并证明仿射概形上凝聚层由有限生成模给出。

**(b)** 证明凝聚层在正合序列、推前、拉回下的性质。

**(c)** 讨论 Serre 的 GAGA 原理。

## 解答

### (a) 凝聚层

O_X-模 F 凝聚：局部有限展示 ∃ O_X^m → O_X^n → F → 0。

X = Spec A 仿射：F ≅ M̃，M = Γ(X,F) 有限生成 A-模。

### (b) 函子性

ker, coker, im 保持凝聚性。有限态射 f_* 保持凝聚；平坦拉回 f^* 保持凝聚。

### (c) GAGA

Serre：复射影簇上，凝聚解析层 ↔ 凝聚代数层，上同调对应。
