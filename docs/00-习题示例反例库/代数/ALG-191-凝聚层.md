# 凝聚层

## 题目

**(a)** 定义凝聚层并证明仿射概形上凝聚层由有限生成模给出。

**(b)** 证明凝聚层在正合序列、推前、拉回下的性质。

**(c)** 讨论 Serre 的 GAGA 原理。

## 解答

### (a) 凝聚层的定义与仿射对应

**定义（$\mathcal{O}_X$-模）**：设 $(X, \mathcal{O}_X)$ 为环化空间（ringed space）。$X$ 上的一个 **$\mathcal{O}_X$-模** 是 $X$ 上的层 $\mathcal{F}$，使得对每个开集 $U \subset X$，$\mathcal{F}(U)$ 是 $\mathcal{O}_X(U)$-模，且限制映射与模结构相容。

**定义（拟凝聚层与凝聚层）**：设 $X$ 为概形（scheme），$\mathcal{F}$ 为 $\mathcal{O}_X$-模。
- $\mathcal{F}$ 称为 **拟凝聚**（quasi-coherent）的，如果 $X$ 可以被仿射开子集 $U_i = \text{Spec } A_i$ 覆盖，使得对每个 $i$，$\mathcal{F}|_{U_i} \cong \widetilde{M_i}$，其中 $M_i$ 是某个 $A_i$-模，$\widetilde{M_i}$ 是 $M_i$ 的关联层（associated sheaf）。
- $\mathcal{F}$ 称为 **凝聚**（coherent）的，如果它是拟凝聚的，且 $X$ 可以被仿射开子集 $U_i = \text{Spec } A_i$ 覆盖，使得对每个 $i$，$M_i$ 是**有限生成** $A_i$-模，且对任意仿射开子集 $\text{Spec } B \subset U_i$，$\mathcal{F}|_{\text{Spec } B} \cong \widetilde{N}$ 其中 $N$ 是有限生成 $B$-模。

等价地，凝聚层可以通过局部有限展示（finite presentation）来刻画：$\mathcal{F}$ 凝聚当且仅当对每个点 $x \in X$，存在开邻域 $U$ 和正合列：
$$\mathcal{O}_U^m \to \mathcal{O}_U^n \to \mathcal{F}|_U \to 0$$
其中 $m, n$ 为非负整数。这里 $\mathcal{O}_U^n$ 表示 $\mathcal{O}_U$ 的 $n$ 次直和，称为秩 $n$ 的自由层。

**定理（仿射对应）**：设 $X = \text{Spec } A$ 为仿射概形。则函子
$$M \mapsto \widetilde{M}$$
建立了 $A$-模范畴与 $X$ 上拟凝聚层范畴之间的等价。在此等价下：
- $M$ 有限生成 $A$-模 $\Leftrightarrow$ $\widetilde{M}$ 是凝聚层。

*证明*：

**步骤1：$\widetilde{M}$ 的定义**。对 $A$-模 $M$，定义预层：
$$\widetilde{M}(D(f)) = M_f = M[1/f]$$
其中 $D(f) = \{p \in \text{Spec } A : f \notin p\}$ 是主开集。对一般开集 $U$，定义为：
$$\widetilde{M}(U) = \{(m_p) \in \prod_{p \in U} M_p : \forall p \in U, \exists f \notin p, m \in M_f \text{ s.t. } q \in D(f) \Rightarrow m_q = m\}$$
这满足层的粘合公理。

**步骤2：函子性质**。
- $(M \oplus N)^\sim \cong \widetilde{M} \oplus \widetilde{N}$
- $(M \otimes_A N)^\sim \cong \widetilde{M} \otimes_{\mathcal{O}_X} \widetilde{N}$
- $(\text{Hom}_A(M, N))^\sim \cong \mathcal{H}om_{\mathcal{O}_X}(\widetilde{M}, \widetilde{N})$（$M$ 有限生成时）
- 局部化：$\widetilde{M}|_{D(f)} \cong \widetilde{M_f}$

**步骤3：拟凝聚层的反方向**。设 $\mathcal{F}$ 为 $X = \text{Spec } A$ 上的拟凝聚层。令 $M = \Gamma(X, \mathcal{F}) = \mathcal{F}(X)$。则 $M$ 有自然的 $A$-模结构：对 $a \in A = \mathcal{O}_X(X)$，$m \in M$，$a \cdot m$ 由 $\mathcal{O}_X$-模结构定义。

构造典范映射 $\alpha: \widetilde{M} \to \mathcal{F}$。在 $D(f)$ 上，$\widetilde{M}(D(f)) = M_f$，而 $\mathcal{F}(D(f))$ 是 $\mathcal{O}_X(D(f)) = A_f$-模。由 $
$\mathcal{F}$ 的拟凝聚性，$\mathcal{F}(D(f)) \cong M_f$。因此 $\alpha$ 是层的同构。

**步骤4：凝聚性**。若 $M$ 有限生成，设 $m_1, \ldots, m_n$ 为生成元。则有满射 $A^n \to M \to 0$，诱导层的满射 $\mathcal{O}_X^n \to \widetilde{M} \to 0$，故 $\widetilde{M}$ 局部有限生成。

反之，若 $\mathcal{F}$ 凝聚，取有限仿射覆盖（单点覆盖即可），$\mathcal{F} \cong \widetilde{M}$。由 $\mathcal{F}$ 的局部有限展示条件，存在开覆盖和正合列 $\mathcal{O}_X^m|_U \to \mathcal{O}_X^n|_U \to \mathcal{F}|_U \to 0$。在 $X = \text{Spec } A$ 上，这全局化为 $A^m \to A^n \to M \to 0$，故 $M$ 有限展示，从而有限生成。

**例子**：
- $\mathcal{O}_X$ 本身是凝聚层（取 $M = A$）。
- 若 $X = \text{Spec } k[x_1, \ldots, x_n]$，理想层 $\widetilde{I}$ 是凝聚层（$I$ 有限生成，由 Noether 性质）。
- 层 $\bigoplus_{i=1}^\infty \mathcal{O}_X$ 是拟凝聚但非凝聚的（无限直和）。

### (b) 凝聚层的函子性

凝聚层在多种代数操作下保持凝聚性。以下设 $X, Y$ 为 Noether 概形（以保证凝聚层的良好性质）。

**命题1（核、余核、像）**：设 $\phi: \mathcal{F} \to \mathcal{G}$ 为凝聚层之间的同态。则 $\ker \phi$, $\text{im } \phi$, $\text{coker } \phi$ 都是凝聚层。

*证明*：问题局部，设 $X = \text{Spec } A$ 仿射，$\mathcal{F} = \widetilde{M}$, $\mathcal{G} = \widetilde{N}$，$M, N$ 有限生成。则 $\phi$ 对应 $A$-模同态 $f: M \to N$。由伴随层函子的正合性：
$$\ker \phi = \widetilde{\ker f}, \quad \text{im } \phi = \widetilde{\text{im } f}, \quad \text{coker } \phi = \widetilde{\text{coker } f}$$
由于 $A$ Noether，有限生成模的子模和商模仍有限生成，故三者均为凝聚。

**命题2（张量积与内部 Hom）**：设 $\mathcal{F}, \mathcal{G}$ 为凝聚层。则：
- $\mathcal{F} \otimes_{\mathcal{O}_X} \mathcal{G}$ 是凝聚层。
- 若 $\mathcal{F}$ 局部自由（或 $X$ 仿射），则 $\mathcal{H}om_{\mathcal{O}_X}(\mathcal{F}, \mathcal{G})$ 是凝聚层。

*证明*：局部地，$\widetilde{M} \otimes \widetilde{N} \cong \widetilde{M \otimes N}$，$M \otimes N$ 有限生成若 $M, N$ 有限生成。对内部 Hom，局部地 $\mathcal{H}om(\widetilde{M}, \widetilde{N}) \cong \widetilde{\text{Hom}(M, N)}$，当 $M$ 有限生成时 $\text{Hom}(M, N)$ 有限生成（$A$ Noether）。

**命题3（推前，Proper Pushforward）**：设 $f: X \to Y$ 为**真**（proper）态射（紧集的原像紧，分离且有限型）。若 $\mathcal{F}$ 为 $X$ 上的凝聚层，则 $f_* \mathcal{F}$ 是 $Y$ 上的凝聚层。

*证明*：这是 **Grothendieck 的相干定理**（coherence theorem）。核心步骤：
1. 问题局部于 $Y = \text{Spec } B$，需证 $\Gamma(X, \mathcal{F})$ 是有限生成 $B$-模。
2. 由 Chow 引理或直接分析，利用 $f$ 的 proper 性证明 $\Gamma(X, \mathcal{F})$ 的有限生成性。
3. 对于射影态射 $X = \text{Proj } S \to \text{Spec } B$（$S$ 为分次 $B$-代数），利用 Serre 的消没定理和 Hilbert 多项式的性质，可以归纳证明凝聚层的整体截面有限生成。

注：若 $f$ 不是 proper 的，$f_* \mathcal{F}$ 一般只是拟凝聚的。例如，设 $j: \mathbb{A}^1 \setminus \{0\} \hookrightarrow \mathbb{A}^1$ 为开嵌入，$\mathcal{F} = \mathcal{O}$，则 $j_* \mathcal{O}$ 的整体截面是 $k[x, x^{-1}]$，作为 $k[x]$-模不是有限生成的。

**命题4（拉回）**：设 $f: X \to Y$ 为概形态射，$\mathcal{G}$ 为 $Y$ 上的凝聚层。
- $f^* \mathcal{G}$ 总是 $X$ 上的凝聚层。
- 若 $f$ 平坦（flat），则 $f^*$ 是正合函子。

*证明*：局部地，设 $X = \text{Spec } B$, $Y = \text{Spec } A$，$f$ 对应环同态 $A \to B$。则 $f^* \widetilde{M} = \widetilde{M \otimes_A B}$。若 $M$ 有限生成，$M \otimes_A B$ 作为 $B$-模有限生成（由 $M$ 的生成元张成）。故 $f^* \mathcal{G}$ 凝聚。

**命题5（局部自由分解）**：在正则 Noether 概形上，每个凝聚层具有局部自由分解（即**局部自由消解**）：
$$0 \to \mathcal{E}_n \to \cdots \to \mathcal{E}_1 \to \mathcal{E}_0 \to \mathcal{F} \to 0$$
其中每个 $\mathcal{E}_i$ 是局部自由凝聚层（即向量丛）。

*证明*：这是 **Hilbert 合冲定理**（syzygy theorem）的整体版本。局部地，$A$ 是正则局部环，有限生成模 $M$ 具有有限自由消解。通过粘合，得到层的局部自由分解。

### (c) Serre 的 GAGA 原理

**GAGA**（Géométrie Algébrique et Géométrie Analytique）是 Jean-Pierre Serre 于1956年发表的奠基性论文，建立了复代数几何与复解析几何之间的深刻联系。

**设定**：设 $X$ 为 $\mathbb{C}$ 上的**射影概形**（projective scheme）。则 $X$ 自然地赋予复解析空间结构 $X^{an}$（通过局部将多项式方程的复解集视为复解析子集）。

**函子 $(\cdot)^{an}$**：存在自然函子（解析化，analytification）：
$$\text{Coh}(X) \to \text{Coh}^{an}(X^{an})$$
$$\mathcal{F} \mapsto \mathcal{F}^{an}$$
其中 $\text{Coh}(X)$ 是 $X$ 上凝聚代数层的范畴，$\text{Coh}^{an}(X^{an})$ 是 $X^{an}$ 上凝聚解析层的范畴。

**定理（Serre GAGA）**：设 $X$ 为 $\mathbb{C}$ 上的射影概形。则：

1. **等价性**：函子 $(\cdot)^{an}$ 是范畴等价。即每个凝聚解析层 $\mathcal{G}$ 在 $X^{an}$ 上都是某个凝聚代数层 $\mathcal{F}$ 的解析化，且：
   $$\text{Hom}_X(\mathcal{F}_1, \mathcal{F}_2) \xrightarrow{\cong} \text{Hom}_{X^{an}}(\mathcal{F}_1^{an}, \mathcal{F}_2^{an})$$

2. **上同调对应**：对任意凝聚层 $\mathcal{F}$，自然映射
   $$H^i(X, \mathcal{F}) \to H^i(X^{an}, \mathcal{F}^{an})$$
   是复向量空间的同构，对所有 $i \geq 0$。

*证明概要*：

**步骤1：$X = \mathbb{P}^n_\mathbb{C}$ 的情形**。利用标准仿射覆盖 $\{U_i = \{x_i \neq 0\}\}$，以及 Čech 上同调计算。对于 $\mathcal{O}(m)$ 层，Serre 已经计算了 $H^i(\mathbb{P}^n, \mathcal{O}(m))$：
- $H^0(\mathbb{P}^n, \mathcal{O}(m))$ 是 $m$ 次齐次多项式空间（$m \geq 0$）
- $H^n(\mathbb{P}^n, \mathcal{O}(m))$ 对偶于 $H^0(\mathbb{P}^n, \mathcal{O}(-n-1-m))$
- 中间上同调为零

解析版本的计算给出相同结果（由 Dolbeault 定理和 Kodaira 消没定理）。

**步骤2：一般射影簇**。对任意闭浸入 $X \hookrightarrow \mathbb{P}^n$，利用归纳法和长正合列：若 $X$ 由方程 $f=0$ 定义，则有余法层正合列：
$$0 \to \mathcal{O}_X(-d) \to \mathcal{O}_X \to \mathcal{O}_Y \to 0$$
其中 $Y = X \cap V(f)$。比较代数版本和解析版本的上同调长正合列，由5-引理归纳推出同构。

**步骤3：全忠实性**。由 Ext 的对应和 Hilbert 多项式的守恒，证明 Hom 集合同构。

**步骤4：本质满性**。对凝聚解析层 $\mathcal{G}$，利用 Cartan 定理A（$\mathbb{P}^n$ 上的凝聚解析层由整体截面生成）和适当的 twist，构造一个凝聚代数层映射到 $\mathcal{G}$，然后通过上同调同构和归纳法证明这是同构。

**GAGA 的意义与限制**：

- **射影性是必要的**：对于仿射簇（如 $\mathbb{A}^1$），GAGA 不成立。例如，$\mathbb{A}^1$ 上的指数函数 $e^z$ 定义了一个非代数的解析函数，对应解析层 $\mathcal{O}^{an}$ 的自同态，它不是代数的。
- **凝聚性是必要的**：对于非凝聚层（如常值层 $\mathbb{C}$），代数与解析上同调不同。$H^1_{Zar}(X, \mathbb{C}) = 0$（Zariski 拓扑不连通），而 $H^1_{an}(X^{an}, \mathbb{C}) = H^1_{sing}(X^{an}; \mathbb{C})$ 可能非零。
- **应用**：GAGA 允许在解析范畴内使用代数工具（如 Chow 定理：紧复解析子流形必为代数子簇），反之亦然（如利用 Hodge 理论研究代数簇的上同调）。

**Chow 定理**：作为 GAGA 的推论，$\mathbb{P}^n(\mathbb{C})$ 中的闭复解析子集必为代数子集。这是因为解析子集的理想层是凝聚解析层，由 GAGA 等价性，它来自某个凝聚代数层的理想，故子集是代数的。

GAGA 原理是现代数学的里程碑之一，它统一了代数几何与复几何两个领域，使得两个范畴的工具可以相互转化使用。
