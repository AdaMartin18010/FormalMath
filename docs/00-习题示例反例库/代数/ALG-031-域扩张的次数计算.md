## 题目

计算下列域扩张的次数 $[K : F]$：

**(a)** $K = \mathbb{Q}(\sqrt{2})$，$F = \mathbb{Q}$

**(b)** $K = \mathbb{Q}(\sqrt{2}, \sqrt{3})$，$F = \mathbb{Q}$

**(c)** $K = \mathbb{Q}(\sqrt[3]{2})$，$F = \mathbb{Q}$

**(d)** $K = \mathbb{Q}(\sqrt{2}, \sqrt[3]{2})$，$F = \mathbb{Q}$

**(e)** $K = \mathbb{Q}(e^{2\pi i/3})$，$F = \mathbb{Q}$

## 详细解答

### (a) 二次扩张 $\mathbb{Q}(\sqrt{2})/\mathbb{Q}$

**解：**

$\sqrt{2}$ 的极小多项式是 $p(x) = x^2 - 2$（Eisenstein判别法，$p=2$）。

该多项式在 $\mathbb{Q}$ 上不可约，次数为2。

因此：
$$[\mathbb{Q}(\sqrt{2}) : \mathbb{Q}] = \deg(p) = 2$$

基：$\{1, \sqrt{2}\}$

### (b) 双二次扩张 $\mathbb{Q}(\sqrt{2}, \sqrt{3})/\mathbb{Q}$

**解：**

用塔律（Tower Law）：
$$[\mathbb{Q}(\sqrt{2}, \sqrt{3}) : \mathbb{Q}] = [\mathbb{Q}(\sqrt{2}, \sqrt{3}) : \mathbb{Q}(\sqrt{2})] \cdot [\mathbb{Q}(\sqrt{2}) : \mathbb{Q}]$$

已知 $[\mathbb{Q}(\sqrt{2}) : \mathbb{Q}] = 2$。

**计算 $[\mathbb{Q}(\sqrt{2}, \sqrt{3}) : \mathbb{Q}(\sqrt{2})]$：**

$\sqrt{3}$ 在 $\mathbb{Q}(\sqrt{2})$ 上的极小多项式？

设 $x = \sqrt{3}$，则 $x^2 = 3$，即 $x^2 - 3 = 0$。

验证 $x^2 - 3$ 在 $\mathbb{Q}(\sqrt{2})$ 上不可约：

若可约，则 $\sqrt{3} \in \mathbb{Q}(\sqrt{2})$，即 $\sqrt{3} = a + b\sqrt{2}$，$a, b \in \mathbb{Q}$。

平方：$3 = a^2 + 2b^2 + 2ab\sqrt{2}$

若 $ab \neq 0$，则 $\sqrt{2} \in \mathbb{Q}$，矛盾。

若 $a = 0$：$3 = 2b^2$，$b^2 = 3/2$，$b \notin \mathbb{Q}$。

若 $b = 0$：$3 = a^2$，$a \notin \mathbb{Q}$。

故不可约，$[\mathbb{Q}(\sqrt{2}, \sqrt{3}) : \mathbb{Q}(\sqrt{2})] = 2$。

因此：
$$[\mathbb{Q}(\sqrt{2}, \sqrt{3}) : \mathbb{Q}] = 2 \times 2 = 4$$

基：$\{1, \sqrt{2}, \sqrt{3}, \sqrt{6}\}$

### (c) 三次扩张 $\mathbb{Q}(\sqrt[3]{2})/\mathbb{Q}$

**解：**

$\sqrt[3]{2}$ 的极小多项式是 $p(x) = x^3 - 2$（Eisenstein，$p=2$）。

不可约，次数为3。

$$[\mathbb{Q}(\sqrt[3]{2}) : \mathbb{Q}] = 3$$

基：$\{1, \sqrt[3]{2}, \sqrt[3]{4}\}$

### (d) 复合扩张 $\mathbb{Q}(\sqrt{2}, \sqrt[3]{2})/\mathbb{Q}$

**解：**

用塔律：
$$[\mathbb{Q}(\sqrt{2}, \sqrt[3]{2}) : \mathbb{Q}] = [\mathbb{Q}(\sqrt{2}, \sqrt[3]{2}) : \mathbb{Q}(\sqrt{2})] \cdot 2$$

或
$$= [\mathbb{Q}(\sqrt{2}, \sqrt[3]{2}) : \mathbb{Q}(\sqrt[3]{2})] \cdot 3$$

**方法：** 证明 $\sqrt{2} \notin \mathbb{Q}(\sqrt[3]{2})$。

$[\mathbb{Q}(\sqrt[3]{2}) : \mathbb{Q}] = 3$。

若 $\sqrt{2} \in \mathbb{Q}(\sqrt[3]{2})$，则 $\mathbb{Q}(\sqrt{2}) \subset \mathbb{Q}(\sqrt[3]{2})$。

但 $[\mathbb{Q}(\sqrt{2}) : \mathbb{Q}] = 2$，而 $2 \nmid 3$，矛盾！

因此 $\sqrt{2} \notin \mathbb{Q}(\sqrt[3]{2})$。

**计算扩张次数：**

$\sqrt[3]{2}$ 在 $\mathbb{Q}(\sqrt{2})$ 上的次数：

$[\mathbb{Q}(\sqrt{2}, \sqrt[3]{2}) : \mathbb{Q}(\sqrt{2})]$ 整除 $[\mathbb{Q}(\sqrt[3]{2}) : \mathbb{Q}] = 3$。

故为1或3。不为1（因 $\sqrt[3]{2} \notin \mathbb{Q}(\sqrt{2})$，否则 $\sqrt[3]{2} = a + b\sqrt{2}$，验证不可能）。

因此：
$$[\mathbb{Q}(\sqrt{2}, \sqrt[3]{2}) : \mathbb{Q}] = 3 \times 2 = 6$$

### (e) 分圆扩张 $\mathbb{Q}(\omega)/\mathbb{Q}$

其中 $\omega = e^{2\pi i/3} = \frac{-1 + i\sqrt{3}}{2}$。

**解：**

$\omega$ 是 primitive 3次单位根，极小多项式是分圆多项式 $\Phi_3(x) = x^2 + x + 1$。

验证：$\omega^2 + \omega + 1 = 0$，且该多项式在 $\mathbb{Q}$ 上不可约（无有理根）。

$$[\mathbb{Q}(\omega) : \mathbb{Q}] = 2$$

基：$\{1, \omega\}$ 或 $\{1, \sqrt{-3}\}$

## 证明技巧

1. **扩张次数计算：** 找代数元的极小多项式，次数即扩张次数
2. **塔律（Tower Law）：** $[K:F] = [K:L][L:F]$，用于复合扩张
3. **不可约性验证：** Eisenstein判别法、有理根定理、反证法

## 常见错误

| 错误 | 说明 |
|------|------|
| 忽略极小多项式需不可约 | 必须验证不可约性 |
| 假设不同根式生成不同扩张 | 可能有包含关系，如$\sqrt{8} \in \mathbb{Q}(\sqrt{2})$ |
| 塔律应用时忘记验证中间域 | 需确保$L$确实是中间域 |
