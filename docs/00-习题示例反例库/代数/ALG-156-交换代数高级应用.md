# 混合特征环与完美胚

## 题目

**(a)** 定义完美胚（perfectoid algebras）。

**(b)** 证明Tilting等价：特征 $p$ 与混合特征的范畴等价。

**(c)** 讨论在交换代数中的应用。

---

## 解答

### (a) 完美胚的定义

#### 动机

$p$-进Hodge理论比较三种上同调：
- $p$-进平展上同调 $H^i_{\text{\'et}}(X_{\bar{K}}, \mathbb{Q}_p)$
- de Rham上同调 $H^i_{dR}(X/K)$
- 晶体上同调 $H^i_{\text{cris}}(X_k/W(k))$

经典的比较定理（Tsuje, Faltings）要求良好的约化条件。完美胚理论由Scholze（2011）引入，彻底改变了这一领域。

#### 完美胚的定义

设 $K$ 是特征0的完全非Archimedean域，$\mathcal{O}_K$ 是其整数环，$\varpi \in \mathcal{O}_K$ 满足 $|p| \leq |\varpi| < 1$。

**定义**：$K$ 是**完美胚域**，若：
1. $K$ 是完全的（关于秩1赋值）
2. 剩余域 $k = \mathcal{O}_K/\mathfrak{m}$ 的特征 $p > 0$
3. **Frobenius几乎是满射**：$\mathcal{O}_K/p$ 的Frobenius是满射
4. $|p| < 1$（即 $K$ 是**混合特征** $(0, p)$）

等价表述（更常用）：Frobenius诱导 $\mathcal{O}_K/\varpi \xrightarrow{\sim} \mathcal{O}_K/\varpi^p$ 是同构。

#### 完美胚代数

**定义**：$\mathcal{O}_K$-代数 $A$ 是**完美胚代数**，若：
1. $A$ 是 $\varpi$-adically 完备的
2. $A$ 是**均匀**的（uniform）：$A^\circ = \{f \in A[\frac{1}{\varpi}] : f^n \in A \text{ for } n \gg 0\}$ 是 $\varpi$-adically 分离的
3. Frobenius $A/\varpi \to A/\varpi^p$ 是同构

**例子**：
- $K = \widehat{\mathbb{Q}_p(p^{1/p^\infty})}$：$p$-进数添加所有 $p$-幂次根后完备化
- $K = \mathbb{C}_p$ 的某些完全子域
- $K\langle T_1^{1/p^\infty}, \ldots, T_n^{1/p^\infty}\rangle$：Tate代数的完美胚版本

### (b) Tilting等价

#### Tilting函子

对完美胚域 $K$，定义其**tilt** $K^\flat$：

$$K^\flat = \varprojlim_{x \mapsto x^p} K$$

即：$K^\flat = \{(x^{(0)}, x^{(1)}, x^{(2)}, \ldots) \in K^{\mathbb{N}} : (x^{(i+1)})^p = x^{(i)}\}$

**性质**：
- $K^\flat$ 是特征 $p$ 的完美胚域
- 赋值 $|(x^{(i)})|_{K^\flat} = |x^{(0)}|_K$
- 整数环 $(K^\flat)^\circ = \varprojlim_{\text{Frob}} \mathcal{O}_K/p$

**关键观察**：虽然 $K$ 是特征0，$K^\flat$ 是特征 $p$，但两者有深刻的联系。

#### 基本定理

**定理**（Scholze的Tilting等价）：

存在范畴等价：
$$\{\text{完美胚 } K\text{-代数}\} \xrightarrow{\sim} \{\text{完美胚 } K^\flat\text{-代数}\}$$
$$A \mapsto A^\flat = \varprojlim_{\text{Frob}} A/\varpi$$

**逆函子**（untilting）：给定特征 $p$ 的完美胚 $K^\flat$-代数 $B$，构造特征0的 perfectoid $A$ 使得 $A^\flat \cong B$。

**构造**：取 $W(B^\circ)$（$B^\circ$ 的Witt向量），然后通过某个关键元 $\xi$ 商去得到 $A$。

具体地，选择提升 $\varpi^\flat \in K^\flat$ 使得 $(\varpi^\flat)^\sharp = \varpi$。则：
$$A = W(B^\circ) \otimes_{W((K^\flat)^\circ)} \mathcal{O}_K$$

#### 几乎数学

Tilting等价的证明依赖**几乎数学**（almost mathematics，Faltings-Gabber-Ramero）。

**定义**：设 $A$ 是 $\varpi$-adically完备环。$A$-模 $M$ 是**几乎零**，若 $M$ 被 $\varpi$ 的任意幂零。

几乎数学研究"modulo almost zero"的范畴。关键工具：
- **Almost flatness**：某些态射在几乎意义下是平坦的
- **Almost finite \'etale morphism**：在几乎意义下的有限平展态射

**定理**（Gabber）：几乎数学满足好的同调性质。

#### 比较同构

**定理**（Scholze）：设 $X$ 是定义在完美胚域 $K$ 上的光滑刚解析簇，$X^\flat$ 是其tilt，则：

$$H^i(X_{\text{\'et}}, \mathbb{Z}/p^n) \cong H^i(X^\flat_{\text{\'et}}, \mathbb{Z}/p^n)$$

这比较了特征0和特征 $p$ 的平展上同调！

### (c) 应用

#### 应用1：$p$-进Hodge理论

**定理**（Scholze）：设 $X$ 是定义在完美胚域 $K$ 上的 proper smooth rigid analytic 簇，则：

$$H^i(X_{\text{\'et}}, \mathbb{Z}_p) \otimes_{\mathbb{Z}_p} B_{dR} \cong H^i_{dR}(X/K) \otimes_K B_{dR}$$

其中 $B_{dR}$ 是Fontaine的de Rham周期环。

**意义**：
- 将平展上同调与de Rham上同调比较
- 推广了Tsuje-Faltings的比较定理到更一般的基
- 不需要良好的约化条件

#### 应用2：近接覆盖（Nearby Cycles）

对 scheme $X \to \text{Spec}(\mathcal{O}_K)$，近接函子 $R\psi$ 联系 generic fiber 和 special fiber 的上同调。

**定理**：在完美胚条件下，近接函子可以"几乎"用特征 $p$ 的几何描述。

#### 应用3：Berkovich空间的拓扑

**定理**（Hochster）：$p$-进域上光滑簇的Berkovich解析空间的拓扑维数等于代数维数。

完美胚理论提供了新的证明，并推广到更一般的基。

#### 应用4：局部Langlands对应

**定理**（Caraiani-Scholze）：对 $p$-adic 域上的 Shimura 簇，局部Langlands对应可以通过 perfectoid 空间的 $p$-进 统一化实现。

这给出了局部Langlands的新的几何实现，联系了：
- $p$-进 上同调
- moduli of $p$-divisible groups
- Rapoport-Zink 空间

#### 应用5：有理同伦论

Bhatt-Morrow-Scholze 将完美胚理论与拓扑循环同调（TC）、代数K理论联系：

$$TC^-(R; \mathbb{Z}_p) \cong THH(R; \mathbb{Z}_p)^{hS^1}$$

对 perfectoid $R$，$THH(R; \mathbb{Z}_p)$ 有简单的结构（几乎完美的de Rham-Witt复形）。

#### 现代发展

1. **Prismatic cohomology**（Bhatt-Scholze）：统一了de Rham、晶体、平展上同调的 new cohomology theory
2. **Condensed mathematics**（Clausen-Scholze）：将完美胚理论纳入更一般的拓扑框架
3. **Liquid tensor products**：解决Banach空间上同调的古老问题
4. **Analytic geometry**（Bambozzi-Ben-Bassat-Kremnizer, Clausen-Scholze）：将刚解析几何推广到更一般的环
