# 导出代数几何

## 题目

**(a)** 定义导出概形（derived scheme）并给出例子。

**(b)** 讨论拟凝聚层复形的范畴。

**(c)** 证明导出不变量（Donaldson-Thomas 理论）。

## 解答

### (a) 导出概形

环空间 (X, O_X)，O_X 是 sheaf of simplicial 交换环，π₀(O_X) = O_{X^{cl}}。

例子：导出截断，导出纤维积。

### (b) 拟凝聚层复形

QCoh(X)：复形层，上同调拟凝聚。导出范畴 D(QCoh(X))。

### (c) DT 不变量

X 光滑射影三维簇，稳定层 E 的模空间 M。

DT 不变量 = 虚拟基本类的积分，计数曲线/层。

与 Gromov-Witten、Pandharipande-Thomas 不变量联系。
