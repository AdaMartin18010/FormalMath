# 特征标理论

## 题目

**(a)** 证明特征标的正交关系（第一和第二正交关系）。

**(b)** 证明 Burnside 定理：p^aq^b 阶群可解。

**(c)** 讨论 Frobenius 群的性质。

## 解答

### (a) 正交关系

第一：⟨χ_i, χ_j⟩ = δ_{ij}，内积 (1/|G|) Σ_g χ_i(g)χ_j(g⁻¹)。

第二：Σ_χ χ(g)χ(h⁻¹) = |C_G(g)| 若 g ∼ h，否则 0。

### (b) Burnside 定理

关键：非平凡不可约特征标次数整除群阶，且 |G|/χ(1)² 是代数整数。

若 χ(1) 是 p^a 或 q^b，利用特征标零点定理。

### (c) Frobenius 群

G 传递作用于 X，仅有恒等元固定多点。结构：G = K ⋊ H，K Frobenius 核。
