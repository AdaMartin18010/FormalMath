## 题目

设 $R$ 是含幺环，$\{M_i\}_{i \in I}$ 是一族 $R$-模。

**(a)** 定义外直积 $\prod_{i \in I} M_i$ 和外直和 $\bigoplus_{i \in I} M_i$，比较二者的区别。

**(b)** 证明：当 $I$ 有限时，$\prod_{i \in I} M_i = \bigoplus_{i \in I} M_i$。

**(c)** 设 $M$ 是 $R$-模，$N_1, N_2$ 是子模。证明：$M = N_1 \oplus N_2$（内直和）当且仅当：
1. $N_1 + N_2 = M$
2. $N_1 \cap N_2 = \{0\}$

**(d)** 证明：$\mathbb{Z}$ 作为 $\mathbb{Z}$-模不是任何真子模的直和项。

## 详细解答

### (a) 外直积与外直和

**定义：**

**外直积：**
$$\prod_{i \in I} M_i = \{(m_i)_{i \in I} : m_i \in M_i\}$$

所有分量都有定义的序列。

**外直和：**
$$\bigoplus_{i \in I} M_i = \{(m_i)_{i \in I} : m_i \in M_i, \text{仅有有限个 } m_i \neq 0\}$$

几乎所有分量为0的序列。

**区别：**
- 直积包含所有序列
- 直和只含"有限支撑"序列
- 直和是直积的子模
- $I$ 无限时，二者不等

### (b) 有限指标集的情形

**证明：**

设 $I = \{1, ..., n\}$ 有限。

对 $(m_1, ..., m_n) \in \prod M_i$，只有有限个（$n$ 个）分量。

故自动满足直和的条件，$\prod M_i \subset \bigoplus M_i$。

反向包含显然，故相等。

**证毕。**

### (c) 内直和的判定

**证明：**

**($\Rightarrow$)** 设 $M = N_1 \oplus N_2$（内直和）。

**条件1：** $N_1 + N_2 = M$ 由定义。

**条件2：** 设 $x \in N_1 \cap N_2$。

则 $x = x + 0 = 0 + x$ 是 $x$ 的两种分解。

由分解唯一性，$x = 0$。

**($\Leftarrow$)** 设条件1和2成立。

**条件1**保证任何 $m \in M$ 可写 $m = n_1 + n_2$。

**条件2**保证唯一性：若 $m = n_1 + n_2 = n_1' + n_2'$，则：
$$n_1 - n_1' = n_2' - n_2 \in N_1 \cap N_2 = \{0\}$$

故 $n_1 = n_1'$，$n_2 = n_2'$。

**证毕。**

### (d) $\mathbb{Z}$ 的非分裂性

**证明：**

设 $\mathbb{Z} = N \oplus K$，$N, K$ 是真子模。

**步骤1：** $\mathbb{Z}$ 的子模形如 $n\mathbb{Z}$。

**步骤2：** 若 $\mathbb{Z} = m\mathbb{Z} \oplus n\mathbb{Z}$，$m, n \geq 1$。

由直和条件：$m\mathbb{Z} \cap n\mathbb{Z} = \{0\}$。

但 $m\mathbb{Z} \cap n\mathbb{Z} = \text{lcm}(m,n)\mathbb{Z} \neq \{0\}$，矛盾。

**结论：** $\mathbb{Z}$ 不能分解为真子模的直和。

**注：** 这说明 $\mathbb{Z}$ 是"不可分解"模，但不是单模。

## 证明技巧

1. **直和与直积的区别：** 有限支撑 vs 任意序列
2. **内直和的判定：** 交为零 + 生成整个模
3. **不可分解的证明：** 假设分解导出矛盾

## 常见错误

| 错误 | 说明 |
|------|------|
| 混淆内外直和 | 内直和是子模关系，外直和是构造 |
| 无限直和中混淆条件 | 直和要求有限支撑 |
| 认为不可分解即单 | $\mathbb{Z}$是反例 |

## 变式练习

**变式1：** 证明$\mathbb{Q}$作为$\mathbb{Z}$-模不可分解。

**变式2：** 设$M = \bigoplus_{i \in I} M_i$，证明$M$有限生成当且仅当$I$有限且每个$M_i$有限生成。

**变式3：** 研究半单模：等价于单子模的直和。
