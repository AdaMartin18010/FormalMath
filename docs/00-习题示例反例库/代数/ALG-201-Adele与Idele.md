# Adele 与 Idele

## 题目

**(a)** 定义 Adele 环 A_K 和 Idele 群 I_K，证明其拓扑性质。

**(b)** 证明强逼近定理。

**(c)** 讨论 Tate 关于 ζ 函数的论文及adelic方法。

## 解答

### (a) Adele 与 Idele

A_K = ∏'_v K_v：限制积，几乎所有分量在 O_v 中。

I_K = ∏'_v K_v^×：Idele。

K 对角嵌入离散，商 K\A_K 紧致。

### (b) 强逼近

对 SL_n，SL_n(K) 在 SL_n(A_K^S) 中稠密（S 有限）。

### (c) Tate 论文

ζ_K(s) = ∫_{I_K} f(x)|x|^s d^×x，Poisson 求和给出函数方程。
