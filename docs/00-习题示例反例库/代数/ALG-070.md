# Abel范畴基础

## 题目

**定义**：Abel范畴是预Abel范畴（有加法、核和余核）且：
- (AB1) 每个态射有核和余核
- (AB2) 每个单态射是其余核的核，每个满态射是其核的余核
- (AB3) 有限双积存在

等价地：有限Limit和Colimit存在，且每个态射可分解为 $f = i \circ p$，$i$ 单，$p$ 满。

(a) 证明在Abel范畴中，态射 $f: A \to B$ 单当且仅当 $\ker f = 0$。

(b) 证明短正合列 $0 \to A \xrightarrow{f} B \xrightarrow{g} C \to 0$ 等价于：
- $f$ 单且 $\text{coker } f = g$（在同构意义下）
- $g$ 满且 $\ker g = f$（在同构意义下）

(c) 证明蛇引理（Snake Lemma）：给定交换图
```
0 → A → B → C → 0
    |     |     |
    | f   | g   | h
    v     v     v
0 → A' → B' → C' → 0
```
行正合，存在连接同态 $\partial: \ker h \to \text{coker } f$ 使六边形正合。

## 解答

### (a) 单态射的核刻画

**证明：**

**($\Rightarrow$)：** 设 $f$ 单。$\ker f \to A$ 满足 $f \circ k = 0 = f \circ 0$。

由单性，$k = 0$，故 $\ker f = 0$。

**($\Leftarrow$)：** 设 $\ker f = 0$。设 $f \circ u = f \circ v$。

考虑 $u - v: X \to A$，则 $f \circ (u - v) = 0$。

故 $u - v$ 通过 $\ker f = 0$，即 $u - v = 0$，$u = v$。

$\square$

### (b) 短正合列的等价刻画

**证明：**

**$0 \to A \xrightarrow{f} B \xrightarrow{g} C \to 0$ 正合意味着：**
- $f$ 单（$\ker f = 0$）
- $\text{im } f = \ker g$（在中间 $B$ 正合）
- $g$ 满（$\text{coker } g = 0$）

在Abel范畴，$\text{coker } f = B / \text{im } f = B / \ker g \cong C$（由 $g$ 诱导的同构）。

反之，若 $f$ 单且 $\text{coker } f \cong g: B \to C$，则：
- $\text{im } f = \ker(\text{coker } f) = \ker g$（由AB2）
- $g$ 满（余核满）

$\square$

### (c) 蛇引理

**证明概要：**

**Step 1**：构造 $\partial: \ker h \to \text{coker } f$。

对 $c \in \ker h$（即 $C$ 中满足 $h(c) = 0$ 的元素）：
- 取 $b \in B$ 使 $g(b) = c$（$g$ 满）
- $g'(f'(b)) = h(g(b)) = h(c) = 0$
- 故 $f'(b) \in \ker g' = \text{im } f$，即存在 $a' \in A'$ 使 $f(a') = f'(b)$
- 定义 $\partial(c) = [a'] \in \text{coker } f$

**Step 2**：验证良定义性。

- $b$ 的选择：若 $g(b_1) = g(b_2) = c$，则 $b_1 - b_2 \in \ker g = \text{im } f$
- 故 $f'(b_1) - f'(b_2) = f'(b_1 - b_2) = f'(f(a)) = f(f'(a)) \in \text{im } f$
- 所以 $a'_1 - a'_2 \in \text{im } f$，在余核中等价

**Step 3**：验证正合性。

需验证六边形：
$$\ker f \to \ker g \to \ker h \xrightarrow{\partial} \text{coker } f \to \text{coker } g \to \text{coker } h$$

每处正合，即像等于核。

例如 $\text{im}(\ker g \to \ker h) = \ker(\partial)$：
- 若 $c \in \ker g$ 提升自 $b \in \ker g$，则 $f'(b) = 0$，故 $\partial(c) = 0$
- 反之若 $\partial(c) = 0$，则 $a' \in \text{im } f$，调整 $b$ 可使 $f'(b) = 0$，即 $b \in \ker g' = \text{im } f$，矛盾（需细致分析）

$\square$

## 证明技巧

1. **核-余核对偶**：Abel范畴的自对偶性
2. **图表追踪**：蛇引理的经典证明方法
3. **构造性定义**：连接同态的显式构造

## 常见错误

- ❌ 忘记验证 $\partial$ 的良定义性
- ❌ 在图表追踪中混淆态射方向
- ❌ 忽略蛇引理所需要的Abel范畴条件

## 变式练习

**变式1：** 用蛇引理证明长正合列：短正合列复形诱导同调长正合列。

**变式2：** 证明五引理（Five Lemma）作为蛇引理的推论。
