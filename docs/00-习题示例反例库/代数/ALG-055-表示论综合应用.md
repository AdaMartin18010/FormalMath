## 题目

设 $S_n$ 是 $n$ 次对称群。

**(a)** 证明：$S_n$ 的不可约复表示与 $n$ 的**分拆**（partition）$\lambda \vdash n$ 一一对应。

**(b)** （Young对称化子）对Young图 $\lambda$，定义 $c_\lambda \in \mathbb{C}[S_n]$。证明 $V_\lambda = \mathbb{C}[S_n] c_\lambda$ 是不可约表示（Specht模）。

**(c)** 证明 **Hook长度公式**：$\dim V_\lambda = \frac{n!}{\prod_{\text{cells}} h(c)}$。

**(d)** 证明 **分支规则**：$\text{Res}_{S_{n-1}}^{S_n} V_\lambda = \bigoplus_{\mu} V_\mu$，其中 $\mu$ 遍历从 $\lambda$ 去掉一个方格得到的分拆。

## 详细解答

### (a) 不可约表示与分拆的对应

**证明：**

$S_n$ 的共轭类与分拆一一对应（轮换型）。

不可约表示个数 = 共轭类个数 = 分拆个数。

故不可约表示可用分拆标记。

**构造性证明概要：**

对每个分拆 $\lambda \vdash n$，构造表示 $V_\lambda$（Specht模）。

证明 $V_\lambda$ 不可约，且 $\lambda \neq \mu$ 时 $V_\lambda \not\cong V_\mu$。

**证毕。**

### (b) Young对称化子与Specht模

**定义：**

对Young表 $T$（形状 $\lambda$），定义：
- $P_T = \{g \in S_n : g \text{ 保持各行}\}$（行稳定子）
- $Q_T = \{g \in S_n : g \text{ 保持各列}\}$（列稳定子）

$$a_T = \sum_{p \in P_T} p, \quad b_T = \sum_{q \in Q_T} \text{sgn}(q) q$$

**Young对称化子：** $c_T = a_T b_T \in \mathbb{C}[S_n]$

**Specht模：** $V_\lambda = \mathbb{C}[S_n] c_T$（同构类与 $T$ 无关）

**证明不可约：**

**步骤1：** $c_T^2 = n_\lambda c_T$（某标量 $n_\lambda \neq 0$）

**步骤2：** $V_\lambda$ 的任意非零子模含 $c_T$

**步骤3：** $V_\lambda$ 作为左理想，是极小左理想，故不可约

**证毕。**

### (c) Hook长度公式

**Hook长度：** 方格 $c$ 的hook是同行的右方和同列的下方（含自身）。
$h(c)$ 是hook中方格数。

**公式：**
$$f^\lambda = \dim V_\lambda = \frac{n!}{\prod_{c \in \lambda} h(c)}$$

**例子：**

$\lambda = (3, 2) \vdash 5$

Young图：
```
□ □ □
□ □
```

Hook长度：
```
4 3 1
2 1
```

$$f^\lambda = \frac{5!}{4 \cdot 3 \cdot 1 \cdot 2 \cdot 1} = \frac{120}{24} = 5$$

**证明概要：**

用分支规则和归纳法。

$\dim V_\lambda = \sum_{\mu} \dim V_\mu$（分支规则）

结合Hook长度的组合性质。

### (d) 分支规则

**定理：**
$$\text{Res}_{S_{n-1}}^{S_n} V_\lambda = \bigoplus_{\mu = \lambda \setminus \square} V_\mu$$

其中 $\mu$ 是从 $\lambda$ 去掉一个角格得到的分拆。

**证明概要：**

**步骤1：** $S_{n-1}$ 嵌入 $S_n$（固定 $n$）

**步骤2：** Young基（Gelfand-Tsetlin基）的限制

**步骤3：** 对Young表 $T$，去掉含 $n$ 的方格，得形状 $\mu$ 的表

**步骤4：** 不同位置给出不同 $\mu$，且正交

**组合推论：**

$\dim V_\lambda = \sum_{\mu} \dim V_\mu$（标准Young表的计数）

**例子：**

$\lambda = (3, 1) \vdash 4$

去掉角格得：$(2, 1)$ 或 $(3)$

$\text{Res} V_{(3,1)} = V_{(2,1)} \oplus V_{(3)}$

维数：$3 = 2 + 1$ ✓

## 证明技巧

1. **Specht模的构造：** Young对称化子是核心
2. **Hook长度公式：** 组合论证 + 归纳
3. **分支规则：** 从子群的嵌入理解表示限制

## 常见错误

| 错误 | 说明 |
|------|------|
| 混淆行稳定子和列稳定子 | $a_T$用正号，$b_T$用交错号 |
| 忘记Young图与分拆的对应 | 分拆是行长度 |
| Hook长度计算遗漏 | 包括自身和同行右、同列下 |

## 变式练习

**变式1：** 证明Murnaghan-Nakayama规则：计算对称群特征标。

**变式2：** 研究Littlewood-Richardson规则：表示诱导的分解。

**变式3：** 证明Robinson-Schensted-Knuth对应：排列与Young表的双射。
