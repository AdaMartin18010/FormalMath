# 向量丛与 K-理论

## 题目

**(a)** 证明 Serre-Swan 定理：紧空间 X 上向量丛 ↔ 有限生成 C(X)-模。

**(b)** 定义拓扑 K-群 K⁰(X) 并证明 Bott 周期性。

**(c)** 讨论代数 K-群与拓扑 K-群的关系。

## 解答

### (a) Serre-Swan

Γ: Vect(X) → C(X)-mod，整体截面函子。

逆：局部化构造向量丛。等价：有限生成投射 C(X)-模 ↔ 向量丛。

### (b) Bott 周期性

K⁰(S²) ≅ ℤ²， Bott 元 β = [H] - 1。

K⁻ⁿ(X) ≅ K⁻ⁿ⁻²(X)，8 周期对实 K-群。

### (c) 代数 K-群

K₀(A)：投射模的 Grothendieck 群。K_i^alg 与 K_i^top 比较（Swan、Karoubi）。
