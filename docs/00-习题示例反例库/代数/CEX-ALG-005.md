# CEX-ALG-005: 非交换的群（S₃是最小例子）

## 反例构造

设 $S_3$ 为3元对称群，包含6个元素：
$$S_3 = \{e, (12), (13), (23), (123), (132)\}$$

其中：
- $e$：单位元
- $(12), (13), (23)$：对换（2阶）
- $(123), (132)$：3-循环（3阶）

## 反例分析

**证明 $S_3$ 非交换：**

计算 $(12)$ 与 $(13)$ 的乘积：

$$(12)(13): \begin{cases} 1 \mapsto 3 \mapsto 3 \\ 2 \mapsto 2 \mapsto 1 \\ 3 \mapsto 1 \mapsto 2 \end{cases} = (132)$$

$$(13)(12): \begin{cases} 1 \mapsto 2 \mapsto 2 \\ 2 \mapsto 1 \mapsto 3 \\ 3 \mapsto 3 \mapsto 1 \end{cases} = (123)$$

因此：
$$(12)(13) = (132) \neq (123) = (13)(12)$$

**验证这是最小阶的非交换群：**

| 阶 | 群 | 是否交换 |
|----|-----|---------|
| 1 | $\{e\}$ | 是 |
| 2 | $\mathbb{Z}_2$ | 是 |
| 3 | $\mathbb{Z}_3$ | 是 |
| 4 | $\mathbb{Z}_4, \mathbb{Z}_2 \times \mathbb{Z}_2$ | 是 |
| 5 | $\mathbb{Z}_5$ | 是 |
| 6 | $\mathbb{Z}_6, S_3$ | $\mathbb{Z}_6$ 是，$S_3$ **否** |

由拉格朗日定理，1,2,3,5阶群必循环（素数阶），4阶群都是阿贝尔群（$p^2$ 阶群必交换）。因此6阶的 $S_3$ 是最小阶非交换群。

## 直观解释

**置换的"顺序敏感性"：**

想象3个位置的标记 $\{1, 2, 3\}$。

- 操作A：交换位置1和2
- 操作B：交换位置1和3

**先做A再做B：** 
- 位置1的元素 → 到位置2 → 留在位置2
- 最终：位置2有原来的1

**先做B再做A：**
- 位置1的元素 → 到位置3 → 留在位置3
- 最终：位置3有原来的1

**几何解释（正三角形对称群）：**

$S_3$ 同构于正三角形的对称群（二面体群 $D_3$）：
- 3个旋转：恒等、120°、240°
- 3个反射：关于3条对称轴的翻转

先做反射A再做反射B，与先做B再做A，结果不同！

**非交换的本质：** 群元素代表"操作"，操作的顺序会影响最终结果。这在几何变换、量子力学算符中都有深刻体现。

## 相关定理

**拉格朗日定理推论**
> 若 $|G| = p$（素数），则 $G \cong \mathbb{Z}_p$ 是循环群，因而交换。

说明：解释了为什么1,2,3,5阶群都是交换的。

**$p^2$ 阶群定理**
> 若 $|G| = p^2$（$p$ 素数），则 $G$ 是阿贝尔群。

说明：解释了为什么4阶群都是交换的。

**中心非平凡性**
> 若 $|G| = p^n$（$p$ 素数），则 $Z(G) \neq \{e\}$。

说明：$p$-群有非平凡中心。$S_3$ 的阶是 $6 = 2 \cdot 3$，不是素数幂，中心平凡：$Z(S_3) = \{e\}$。

**换位子群**
> 群 $G$ 交换当且仅当换位子群 $[G,G] = \{e\}$。

说明：$[S_3, S_3] = A_3 = \{e, (123), (132)\} \neq \{e\}$，因此 $S_3$ 不交换。

**凯莱定理**
> 任何群都同构于某个置换群的子群。

说明：$S_3$ 本身就是置换群，任何非交换群都可以"实现"为置换群的子群。

## 避免此反例的条件

| 条件 | 结论 | 验证方法 |
|------|------|---------|
| $|G| \leq 5$ | 必交换 | 拉格朗日定理 |
| $|G| = p$（素数） | 必循环交换 | 群作用分析 |
| $|G| = p^2$ | 必交换 | 类方程 |
| $G/Z(G)$ 循环 | $G$ 交换 | 换位子论证 |
| $G$ 是阿贝尔群 | 元素可交换 | 直接验证 $ab=ba$ |

**交换群的等价刻画：**

以下条件等价：
1. $G$ 是阿贝尔群
2. 映射 $g \mapsto g^{-1}$ 是自同构
3. 映射 $g \mapsto g^n$ 对任意 $n$ 是自同态
4. 换位子群 $[G,G] = \{e\}$

**教学提示：** $S_3$ 是理解非交换性的最佳入门例子。它同时具有：
- 简单的结构（仅6个元素）
- 丰富的子群结构（3个2阶子群，1个3阶子群）
- 几何直观（正三角形对称）
- 作为 $S_n$（$n \geq 3$）的子群代表

**更高阶例子：** 对于任何 $n \geq 3$，$S_n$ 都是非交换群。更一般地，任何包含 $S_3$ 作为子群的群都是非交换的。
