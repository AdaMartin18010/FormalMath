# 半单 Lie 代数

## 题目

**(a)** 证明 Cartan 判别：Killing 形式非退化 ⟺ Lie 代数半单。

**(b)** 证明 Weyl 定理：半单 Lie 代数的表示完全可约。

**(c)** 讨论根系的结构和 Weyl 群。

## 解答

### (a) Cartan 判别

κ(x,y) = tr(ad_x ad_y)。rad(κ) 是可解理想。

L 半单 ⟺ 无交换理想 ⟺ κ 非退化。

### (b) Weyl 定理

Casimir 元 C ∈ U(L) 与所有元交换，在不可约表示上是标量。

利用 C 构造投影到不变子空间。

### (c) 根系

Cartan 分解：L = H ⊕ ⊕_α L_α。根 α: H → ℂ，根系 Φ 的公理。

Weyl 群 W = N_G(H)/H，由反射生成。
