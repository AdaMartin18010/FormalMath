# 代数综合专题 III

## 题目

**(a)** 证明 motive 理论的构造和问题（Grothendieck）。

**(b)** 讨论 Voevodsky 的三角 motive 范畴。

**(c)** 证明 Bloch-Kato 猜想和 Milnor 猜想。

## 解答

### (a) Grothendieck motive

光滑射影簇，对应是代数闭链模数值/同调等价。

范畴 M_k，是半单 Abel 范畴（标准猜想）。

### (b) Voevodsky motive

DM(k)：有限对应，Nisnevich 拓扑，A¹-同伦。

三角范畴，有紧生成元和对偶。

### (c) Bloch-Kato

Milnor K-理论与 Galois 上同调的联系：
$$K_n^M(k)/m \cong H^n(k, μ_m^{\otimes n})$$

Voevodsky 证明（2003年 Fields 奖工作）。
