## 题目

设 $(C_{\bullet,\bullet}, d^h, d^v)$ 是第一象限双复形（$p < 0$ 或 $q < 0$ 时 $C_{p,q} = 0$）。

(1) 定义**全复形** $\operatorname{Tot}(C)_n = \bigoplus_{p+q=n} C_{p,q}$，证明 $d = d^h + (-1)^p d^v$ 满足 $d^2 = 0$；

(2) 构造与双复形相关的**谱序列**，说明 $E^0$, $E^1$, $E^2$ 页；

(3) 叙述**谱序列的比较定理**并解释其用途。

## 详细解答

### (1) 全复形

**定义**：$\operatorname{Tot}(C)_n = \bigoplus_{p+q=n} C_{p,q}$。

微分 $d: \operatorname{Tot}(C)_n \to \operatorname{Tot}(C)_{n-1}$ 在 $C_{p,q}$ 上为：

$$d = d^h + (-1)^p d^v$$

其中 $d^h: C_{p,q} \to C_{p-1,q}$，$d^v: C_{p,q} \to C_{p,q-1}$。

**验证 $d^2 = 0$**：

对 $x \in C_{p,q}$：

$$d^2(x) = d(d^h x + (-1)^p d^v x)$$
$$= d^h(d^h x) + (-1)^{p-1}d^v(d^h x) + (-1)^p d^h(d^v x) + (-1)^{2p}d^v(d^v x)$$

利用 $d^h \circ d^h = 0$，$d^v \circ d^v = 0$，以及双复形条件 $d^h \circ d^v = -d^v \circ d^h$：

$$= (-1)^{p-1}d^v d^h x + (-1)^p d^h d^v x = (-1)^{p-1}(-d^h d^v) + (-1)^p d^h d^v = 0$$

### (2) 谱序列构造

**过滤**：全复形有两种自然过滤：
- **第一过滤**：$F^p_I \operatorname{Tot}_n = \bigoplus_{i+j=n, i \geq p} C_{i,j}$
- **第二过滤**：$F^q_{II} \operatorname{Tot}_n = \bigoplus_{i+j=n, j \geq q} C_{i,j}$

**$E^0$ 页**：

$$E^0_{p,q} = F^p \operatorname{Tot}_{p+q} / F^{p+1} \operatorname{Tot}_{p+q} = C_{p,q}$$

微分 $d^0$ 由 $d^v$ 诱导（第一过滤）。

**$E^1$ 页**：

$$E^1_{p,q} = H_q^v(C_{p,\bullet})$$

即对固定 $p$，取竖直方向的同调。

**$E^2$ 页**：

$$E^2_{p,q} = H_p^h(H_q^v(C))$$

即对 $E^1$ 取水平方向的同调。

### (3) 比较定理

**定理**：设 $f: C \to D$ 是双复形的映射。若 $f$ 在某页 $E^r$ 诱导同构，则在 $E^\infty$ 也诱导同构，从而 $f_*: H(\operatorname{Tot}(C)) \to H(\operatorname{Tot}(D))$ 是同构。

**用途**：
- 比较不同分解计算的同调
- 证明不同构造给出同构结果
- 简化复杂复形的计算

## 证明技巧

1. **过滤选择**：根据计算方便选择第一或第二过滤
2. **页间转换**：理解 $d^r: E^r \to E^r$ 的次数
3. **收敛分析**：第一象限双复形保证谱序列收敛

## 常见错误

| 错误类型 | 错误表现 | 正确做法 |
|---------|---------|---------|
| 符号错误 | $(-1)^p$ 位置 | 仔细跟踪水平/竖直次数 |
| 收敛混淆 | 认为所有谱序列都收敛 | 需要适当有界条件 |
| 页的顺序 | $E^0$ vs $E^1$ vs $E^2$ | 从微分商的同调逐层计算 |

## 变式练习

**变式1（难度⭐⭐⭐⭐）**：用谱序列证明Tor的平衡性：$\operatorname{Tor}_n(M,N)$ 可用任一模的投射分解计算。

**变式2（难度⭐⭐⭐⭐⭐）**：用Grothendieck谱序列定理推导Leray谱序列。

**变式3（难度⭐⭐⭐⭐）**：计算特定双复形（如群上同调中的Hochschild-Serre谱序列）。
