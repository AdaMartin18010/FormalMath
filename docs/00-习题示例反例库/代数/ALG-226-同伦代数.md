# 同伦代数

## 题目

**(a)** 定义 A-无穷代数并给出例子。

**(b)** 讨论 DG 范畴和导出范畴的关系。

**(c)** 证明同伦转移定理。

## 解答

### (a) A-无穷代数

向量空间 V 上运算 m_n: V^{⊗n} → V，次数 2-n。

关系：Σ (-1)^{...} m_k(id^{⊗i} ⊗ m_j ⊗ id^{⊗...}) = 0。

例子：上同调代数，Massey 积。

### (b) DG 范畴

态射复形，合成是链映射。

导出范畴：H⁰(DG-范畴) 得到三角范畴。

### (c) 同伦转移

(V,d) 到 (W,d') 的收缩给出 A-无穷 结构转移：
$$m_n' = p \circ m_n \circ (i \otimes \cdots \otimes i) + \cdots$$
