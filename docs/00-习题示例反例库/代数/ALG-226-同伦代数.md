# 同伦代数

## 题目

**(a)** 定义 $A_\infty$-代数并给出具体例子。

**(b)** 讨论 DG 范畴和导出范畴的关系。

**(c)** 证明同伦转移定理（Homotopy Transfer Theorem）。

## 解答

### (a) $A_\infty$-代数

**定义**：$
$ \mathbb{Z}$-分次向量空间 $V$，运算 $m_n: V^{\otimes n} \to V$（$n \geq 1$），$|m_n| = 2-n$，满足 **$A_\infty$-关系**：

$$\sum_{\substack{n = r+s+t \\ r,t \geq 0, s \geq 1}} (-1)^{r+st} m_{r+1+t}(\operatorname{id}^{\otimes r} \otimes m_s \otimes \operatorname{id}^{\otimes t}) = 0$$

**低阶关系**：

- $n=1$：$m_1^2 = 0$（$m_1$ 是微分）
- $n=2$：$m_1(m_2(a,b)) + m_2(m_1(a),b) + (-1)^{|a|}m_2(a,m_1(b)) = 0$（$m_2$ 对 $m_1$ 是结合的 up to homotopy）
- $n=3$：$m_2$ 的结合性由 $m_3$ 控制

**例子 1：DGA**：微分分次代数 $(A, d, \cdot)$ 是 $A_\infty$-代数，$m_1 = d$，$m_2 = \cdot$，$m_n = 0$（$n \geq 3$）。

**例子 2：Massey 积**：拓扑空间 $X$ 的上同调 $H^*(X)$ 有 $A_\infty$-结构，$m_3$ 是 Massey 积。

---

### (b) DG 范畴与导出范畴

**DG 范畴**：范畴 $\mathcal{C}$，态射 $
$ \operatorname{Hom}(X, Y)$ 是链复形，复合是链映射。

**例**：$\operatorname{Ch}(\mathcal{A})$（链复形范畴），$\operatorname{Hom}^n(A, B) = \prod_i \operatorname{Hom}(A^i, B^{i+n})$。

**导出范畴**：

$$H^0(\mathcal{C}) = \text{同调范畴}$$

- 对象与 $\mathcal{C}$ 相同
- 态射：$\operatorname{Hom}_{H^0(\mathcal{C})}(X, Y) = H^0(\operatorname{Hom}_\mathcal{C}(X, Y))$

**与三角范畴的关系**：$H^0(\mathcal{C})$ 有自然的三角结构（由 DG 范畴的 mapping cone 给出）。

**增强（Enhancement）**：三角范畴 $\mathcal{T}$ 有 DG 增强，若存在 DG 范畴 $\mathcal{C}$ 使 $H^0(\mathcal{C}) \cong \mathcal{T}$。

**唯一性（Lunts-Orlov, Canonaco-Orlov）**：在一定条件下，DG 增强唯一。

### (c) 同伦转移定理

**定理**：设 $(V, m_1)$ 和 $(W, d)$ 是链复形，有链映射 $i: W \to V$，$p: V \to W$，$p \circ i = \operatorname{id}_W$，以及同伦 $h: V \to V$，$i \circ p - \operatorname{id}_V = m_1 h + h m_1$。

若 $(V, \{m_n\})$ 是 $A_\infty$-代数，则 $W$ 上有诱导的 $A_\infty$-结构 $\{m'_n\}$：

$$m'_n = \sum_{\text{trees } T} \pm \; p \circ m_{|T|} \circ (i^{\otimes |T|})$$

更精确地，由 rooted planar trees 的和给出。

**特例（Kadeishvili）**：$W = H^*(V)$（上同调），$i$ 是代表元选取，$p$ 是投影，$h$ 是 homotopy。

则 $H^*(V)$ 上有 $A_\infty$-结构，$m_1 = 0$，$m_2$ 是通常的乘积，$m_3$ 是 Massey 积。

**应用**：

- **镜像对称**：$A_\infty$-结构编码 Fukaya 范畴的 higher products
- **形变理论**：$A_\infty$-结构的形变由 Hochschild 上同调控制

## 常见错误

- **$A_\infty$ 的符号**：不同文献对 $A_\infty$-关系的符号约定不同（ suspended 或 unsuspended）。
- **DG 增强的非唯一性**：一般三角范畴可能有多个 DG 增强（Schlichting 的反例）。
- **同伦转移的树求和**：不是所有树都贡献，只有 planar rooted trees 对应 $A_\infty$。
