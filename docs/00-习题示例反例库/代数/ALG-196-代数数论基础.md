# 代数数论基础

## 题目

**(a)** 证明代数整数构成环。

**(b)** 证明数域的整数环是 Dedekind 整环。

**(c)** 计算 Q(√-5) 的整数环和类数。

## 解答

### (a) 代数整数环

α, β 代数整数 ⟹ α+β, αβ 代数整数。

证明：Z[α,β] 是有限生成 Z-模，其子模 Z[α+β] 和 Z[αβ] 也是。

### (b) 整数环性质

O_K 是：整闭、Noether、维数1 ⟹ Dedekind 整环。理想唯一分解为素理想积。

### (c) Q(√-5)

O_K = Z[√-5]。类数2：(2, 1+√-5) 是非主理想。
