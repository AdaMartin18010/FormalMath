# 谱序列基础

## 题目

**定义**：谱序列 $(E_r, d_r)$ 是一族双分次模 $E_r = \bigoplus_{p,q} E_r^{p,q}$ 和微分 $d_r: E_r^{p,q} \to E_r^{p+r, q-r+1}$ 满足 $d_r^2 = 0$，且 $E_{r+1} = H(E_r, d_r)$。

设 $C^{\bullet, \bullet}$ 是第一象限双复形（$p < 0$ 或 $q < 0$ 时 $C^{p,q} = 0$）。

(a) 证明存在两个谱序列：
- ${}^I E_2^{p,q} = H_h^p(H_v^q(C)) \Rightarrow H^{p+q}(\text{Tot}(C))$
- ${}^{II} E_2^{p,q} = H_v^p(H_h^q(C)) \Rightarrow H^{p+q}(\text{Tot}(C))$

(b) 设 $F: \mathcal{A} \to \mathcal{B}$ 和 $G: \mathcal{B} \to \mathcal{C}$ 是加性函子，$\mathcal{A}, \mathcal{B}$ 有足够内射，$F$ 保持内射。证明Grothendieck谱序列：
$$E_2^{p,q} = R^pG(R^qF(A)) \Rightarrow R^{p+q}(GF)(A)$$

## 解答

### (a) 双复形的谱序列

**证明概要：**

**过滤构造**：

全复形 $\text{Tot}(C)^n = \bigoplus_{p+q=n} C^{p,q}$，微分 $D = d_h + (-1)^p d_v$。

**第一过滤**：$F_I^p(\text{Tot})^n = \bigoplus_{i \geq p} C^{i, n-i}$

这诱导谱序列，$E_0^{p,q} = F^p/F^{p+1} = C^{p,q}$。

$E_1^{p,q} = H(D|_{E_0})$。由于 $D = d_h + (-1)^p d_v$ 在 $E_0$ 上，且 $d_h$ 保持 $p$，

$E_1$ 上微分是 $d_v$（纵向）：
$$E_1^{p,q} = H_v^q(C^{p, \bullet})$$

$E_2$ 上微分是 $d_h$（横向）诱导的：
$$E_2^{p,q} = H_h^p(H_v^q(C))$$

**第二过滤**类似，交换行列角色。

收敛性由第一象限假设保证：对每个 $(p,q)$，当 $r$ 大时 $d_r$ 进出都为零。 $\square$

### (b) Grothendieck谱序列

**证明概要：**

**Step 1：** 取 $A$ 的内射分解 $0 \to A \to I^\bullet$。

**Step 2：** 对每个 $F(I^q)$，取 $\mathcal{B}$ 中的内射分解 $F(I^q) \to J^{\bullet, q}$。

由 $F$ 保持内射，$F(I^q)$ 是 $G$-适应的。

**Step 3：** 构造双复形 $C^{p,q} = G(J^{p,q})$。

**Step 4（第一谱序列）**：
先算纵向：$H_v^q(C^{p, \bullet}) = H^q(G(J^{p, \bullet})) = R^qG(F(I^p))$

由于 $I^p$ 内射，$F(I^p)$ 是 $G$-适应的，故 $R^qG(F(I^p)) = G(F(I^p))$ 若 $q=0$，否则需调整。

实际上对一般对象，$R^qG(F(I^p))$ 非零。

再算横向：$E_2^{p,q} = H^p(R^qG(F(I^\bullet))) = R^pG(R^qF(A))$。

**Step 5（第二谱序列）**：
先算横向：$H_h^p(C^{\bullet, q}) = H^p(G(J^{\bullet, q})) = G(H^p(J^{\bullet, q}))$（$G$ 左正合）

由于 $J^{\bullet, q}$ 是 $F(I^q)$ 的内射分解，$H^p(J^{\bullet, q}) = R^pF(I^q) = 0$（$p > 0$，内射）。

故 ${}^{II} E_2$ 只在 $p=0$ 非零：$H_v^q(G(F(I^\bullet))) = R^q(GF)(A)$。

**Step 6**：两谱序列都收敛到 $H(\text{Tot})$，故：
$$E_2^{p,q} = R^pG(R^qF(A)) \Rightarrow R^{p+q}(GF)(A)$$

$\square$

## 证明技巧

1. **过滤方法**：谱序列来自复形的过滤
2. **双复形技巧**：用两个方向计算得到不同信息
3. **收敛条件**：第一象限保证谱序列"稳定"

## 常见错误

- ❌ 混淆 $E_r$ 页的不同指标约定
- ❌ 忘记验证收敛条件
- ❌ 在Grothendieck谱序列中忽略 $F$ 保持内射的条件

## 变式练习

**变式1：** 用Lyndon-Hochschild-Serre谱序列计算群扩张的上同调。

**变式2：** 证明Leray谱序列：对拓扑空间映射 $f: X \to Y$，有 $E_2^{p,q} = H^p(Y, R^qf_*\mathcal{F}) \Rightarrow H^{p+q}(X, \mathcal{F})$。
