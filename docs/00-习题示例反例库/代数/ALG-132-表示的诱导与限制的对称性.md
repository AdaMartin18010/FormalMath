# 表示的诱导与限制的对称性

## 题目

设 $G$ 是有限群，$H \subset G$ 是子群。

**(a) Nakayama关系**：
$$\text{Hom}_G(\text{Ind}_H^G V, W) \cong \text{Hom}_H(V, \text{Res}_H^G W)$$
$$\text{Hom}_G(V, \text{Ind}_H^G W) \cong \text{Hom}_H(\text{Res}_H^G V, W)$$

**(b) 弗罗贝尼乌斯互反性的推论**：
证明 $\text{Ind}_H^G$ 和 $\text{Res}_H^G$ 是伴随函子。

**(c) 具体计算**：设 $G = S_3$，$H = \langle(12)\rangle$，$V$ 是 $H$ 的符号表示。计算 $\text{Ind}_H^G V$ 并验证互反性。

## 解答

### (a) Nakayama关系

**证明**：

**第一式**（Frobenius互反性）：

已在ALG-123中证明。

**第二式**（伴随形式）：

考虑映射：
$$\Phi: \text{Hom}_G(V, \text{Ind}_H^G W) \to \text{Hom}_H(\text{Res}_H^G V, W)$$

$$\Phi(\varphi)(v) = \text{pr}_1(\varphi(v)(1))$$

或更自然：
$$\varphi \mapsto (v \mapsto \varphi(v)(e))$$

逆映射：给定 $\psi: \text{Res}_H^G V \to W$。

$$\tilde{\psi}(v)(g) = \psi(g^{-1}v)$$

验证是诱导表示中的元素。

两者互逆。$\square$

### (b) 伴随函子

**证明**：

由(a)，存在自然的同构：
$$\text{Hom}_G(\text{Ind}_H^G V, W) \cong \text{Hom}_H(V, \text{Res}_H^G W)$$

这正是左伴随的定义：$\text{Ind}_H^G$ 是 $\text{Res}_H^G$ 的左伴随。

类似地，第二式表明在适当范畴下，它们也是右伴随。

具体：$(\text{Res}, \text{Coind})$ 和 $(\text{Ind}, \text{Res})$ 是伴随对。

对于有限群，$\text{Ind} = \text{Coind}$（因 $[G:H] < \infty$）。$\square$

### (c) $S_3$ 的具体计算

**设置**：

$H = \langle(12)\rangle \cong \mathbb{Z}/2$，$V = \text{sgn}$。

$\chi_V((12)) = -1$，$\chi_V(e) = 1$。

**诱导表示**：

$\text{Ind}_H^G V$，维数 $= [G:H] \cdot \dim V = 3$。

**特征标**（用公式）：

| $g$ | $\chi_{\text{Ind}}(g)$ |
|-----|----------------------|
| $e$ | 3 |
| $(12)$ | -1 |
| $(13)$ | 0 |
| $(23)$ | 0 |
| $(123)$ | 0 |
| $(132)$ | 0 |

**分解**：

与不可约特征标内积：

$\langle \chi_{\text{Ind}}, \chi_{\text{triv}} \rangle = 0$

$\langle \chi_{\text{Ind}}, \chi_{\text{sgn}} \rangle = 1$

$\langle \chi_{\text{Ind}}, \chi_{\text{std}} \rangle = 1$

因此：$\text{Ind}_H^G V = \text{sgn} \oplus \text{std}$。

**验证互反性**：

取 $W = \text{std}$（$S_3$ 的标准表示）。

$\text{Hom}_G(\text{Ind}_H^G V, W) \cong \mathbb{C}$（由分解）。

$\text{Res}_H^G W = \text{sgn} \oplus \text{triv}$（作为 $H$-表示）。

$\text{Hom}_H(V, \text{Res}_H^G W) = \text{Hom}_H(\text{sgn}, \text{sgn}) \cong \mathbb{C}$。

同构成立。$\square$

## 证明技巧

1. **函子伴随性**：Hom集的自然同构
2. **诱导表示的显式构造**：函数空间的理解
3. **特征标验证**：计算与理论的对照

## 常见错误

- ❌ 混淆两种Nakayama关系
- ❌ 诱导表示维数计算
- ❌ 限制表示的分解错误
