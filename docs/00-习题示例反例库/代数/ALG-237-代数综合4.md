# 代数综合专题 IV

## 题目

**(a)** 陈述 perfectoid 空间的基本性质（Scholze）。

**(b)** 讨论 p-进 Langlands 纲领的核心问题。

**(c)** 证明 p-进 Hodge 理论的现代发展。

## 解答

### (a) Perfectoid 空间

**动机**：$p$-进域上的代数几何受困于拓扑问题（完全不连续）。Perfectoid 理论通过"倾斜"（tilting）将特征 0 与特征 $p$ 联系起来。

**Perfectoid 代数**：完备 Banach 代数 $R$（$\mathbb{Q}_p$ 上），满足：

1. **一致**：$R^\circ = \{x : |x| \leq 1\}$ 有界
2. **$p$-完备**：$R$ 是 $p$-adically 完备的
3. **Frobenius 满**：$R^\circ/p$ 上的 Frobenius 是满的

**倾斜（Tilting）**：$R \mapsto R^\flat = \varprojlim_{x \mapsto x^p} R$，是完美 $
$ \mathbb{F}_p$-代数。

**定理（Scholze）**：

$$
\{\text{perfectoid 代数 over } \mathbb{Q}_p\} \xrightarrow{\sim} \{\text{perfect 完备代数 over } \mathbb{F}_p((t^{1/p^\infty}))\}
$$

**Perfectoid 空间**：局部是 $\operatorname{Spa}(R, R^+)$ 的解析空间，$R$ perfectoid。

**应用**：
- **几乎纯性**：perfectoid 代数上的有限étale 覆盖几乎是 étale
- **$p$-进自守形式**：Hilbert 模形式的 $p$-进族

---

### (b) p-进 Langlands 纲领

**经典 Langlands**：复表示 $
$ \leftrightarrow$ 复 Galois 表示。

**p-进 Langlands**：$p$-进 Banach 空间表示 $
$ \leftrightarrow$ $p$-进 Galois 表示。

**$GL_2(\mathbb{Q}_p)$ 情形**：

**定理（Colmez, Breuil, Emerton, Paskunas）**：存在函子ial 对应：

$$
\{\text{admissible } p\text{-adic Banach reps of } GL_2(\mathbb{Q}_p)\} \leftrightarrow \{\text{2-dim } p\text{-adic Galois reps}\}
$$

**构造**：
- Colmez 的 Montreal 函子：从 Galois 表示构造 $GL_2$ 表示
- $p$-进局部 Langlands：$\Pi(V)$ 是 Galois 表示 $V$ 对应的 $GL_2$ 表示

**困难**：
- $p$-进表示通常不可约但非不可约分支
- Banach 空间表示的分类复杂（无完全约化性）

**高秩情形**：Calegari-Geraghty 的 patching 方法，对 $GL_n$（$n \geq 3$）有猜想但远未解决。

---

### (c) 现代 p-进 Hodge 理论

**经典 p-进 Hodge（Tate, Sen, Fontaine）**：

对 $p$-adic Galois 表示 $V$，比较函子：

- $D_{HT}(V) = (V \otimes_{\mathbb{Q}_p} \mathbb{C}_p)^{G_K}$（Hodge-Tate）
- $D_{dR}(V) = (V \otimes_{\mathbb{Q}_p} B_{dR})^{G_K}$（de Rham）
- $D_{cris}(V) = (V \otimes_{\mathbb{Q}_p} B_{cris})^{G_K}$（crystalline）

**现代发展**：

1. **Perfectoid 方法**：Scholze 用 perfectoid 空间证明几乎纯性，构造比较同构

2. **Prismatic 上同调（Bhatt-Scholze, 2019）**：统一 crystalline 和 étale 上同调的框架
   
   对形式概形 $X$，prismatic site $(X)_\Prism$ 上的层给出 $A_{\mathrm{inf}}$-系数上同调。

3. **相对 p-进 Hodge**：对光滑态射 $f: X \to Y$，$R^if_* \mathbb{Q}_p$ 是局部系统，有 $p$-进 Hodge 分解。

---

## 常见错误

- **Perfectoid 的一致条件**：仅 $p$-完备不够，需一致且 Frobenius 满。
- **p-进 Langlands 的非对称性**：从 Galois 到 $GL_2$ 的构造比逆方向更明确。
- **$B_{dR}$ 的幂零元**：$B_{dR}$ 是 DVR，剩余域 $\mathbb{C}_p$，不是代数闭的。
