# 最高权理论

## 题目

**(a)** 定义最高权模 Verma 模 M(λ) 并讨论其性质。

**(b)** 证明最高权不可约表示 L(λ) 的分类定理。

**(c)** 计算 s̄l(n, ℂ) 的基本表示和权重格。

## 解答

### (a) Verma 模

M(λ) = U(g) ⊗_{U(b)} ℂ_λ，通用最高权模。

性质：有唯一极大子模，商为不可约 L(λ)。

### (b) 分类定理

{L(λ) : λ 支配整权} 是 g 的有限维不可约表示的完全集。

### (c) s̄l(n, ℂ)

Cartan：对角迹零矩阵。基本权 ω_i：前 i 个对角和为1，其余0。

V(ω_i) = Λ^i(ℂ^n)，维数 C(n,i)。
