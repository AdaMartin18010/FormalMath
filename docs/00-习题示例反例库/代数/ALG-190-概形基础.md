# 概形基础

## 题目

**(a)** 定义局部环层空间、仿射概形和概形。

**(b)** 证明 $\text{Spec } A$ 的层 $\mathcal{O}_X$ 是局部环层，茎同构于 $A_{\mathfrak{p}}$。

**(c)** 讨论概形的纤维积及其几何意义。

## 解答

### (a) 概形的定义

**局部环层空间**：一个**环化空间**（ringed space）是一对 $(X, \mathcal{O}_X)$，其中 $X$ 是拓扑空间，$\mathcal{O}_X$ 是 $X$ 上的环层（sheaf of rings）。若对每个点 $x \in X$，茎（stalk）$\mathcal{O}_{X,x}$ 是局部环，则称 $(X, \mathcal{O}_X)$ 为**局部环层空间**（locally ringed space）。

局部环的极大理想记为 $\mathfrak{m}_x \subset \mathcal{O}_{X,x}$，剩余域为 $k(x) = \mathcal{O}_{X,x} / \mathfrak{m}_x$。

**仿射概形**：设 $A$ 为交换环（含单位元）。$A$ 的**素谱**（prime spectrum）定义为：
$$\text{Spec } A = \{\mathfrak{p} \subset A : \mathfrak{p} \text{ 是素理想}\}$$
配备**Zariski 拓扑**：闭集形如 $V(I) = \{\mathfrak{p} : I \subset \mathfrak{p}\}$，其中 $I \subset A$ 是理想。

在 $\text{Spec } A$ 上定义结构层 $\mathcal{O}_X$：对主开集 $D(f) = \{\mathfrak{p} : f \notin \mathfrak{p}\}$（$f \in A$），令：
$$\mathcal{O}_X(D(f)) = A_f$$
其中 $A_f = A[1/f]$ 是 $A$ 在 $f$ 处的局部化。对一般开集，通过层的粘合公理定义。

对 $(X, \mathcal{O}_X) = (\text{Spec } A, \mathcal{O}_{\text{Spec } A})$，称为**仿射概形**。

**概形**：一个**概形**（scheme）是局部环层空间 $(X, \mathcal{O}_X)$，使得 $X$ 可以被仿射开子集覆盖：存在开覆盖 $\{U_i\}$，使得每个 $(U_i, \mathcal{O}_X|_{U_i})$ 同构于某个仿射概形 $\text{Spec } A_i$。

**态射**：概形态射 $\varphi: (X, \mathcal{O}_X) \to (Y, \mathcal{O}_Y)$ 是连续映射 $f: X \to Y$ 加上环层同态 $f^\#: \mathcal{O}_Y \to f_* \mathcal{O}_X$，使得在茎上诱导的映射 $f^\#_x: \mathcal{O}_{Y, f(x)} \to \mathcal{O}_{X,x}$ 是**局部同态**，即 $(f^\#_x)^{-1}(\mathfrak{m}_x) = \mathfrak{m}_{f(x)}$。

**例子**：
- $\text{Spec } k$（$k$ 为域）是单点，结构层为 $k$。
- $\text{Spec } \mathbb{Z} = \{(0), (2), (3), (5), \ldots\}$。闭点是素数理想 $(p)$， generic 点是 $(0)$。
- $\mathbb{A}^1_k = \text{Spec } k[x]$。点是 $(0)$（generic 点）和 $(x-a)$（闭点，对应 $a \in k$，若 $k$ 代数闭）。
- $\text{Spec } k[\varepsilon]/(\varepsilon^2)$（**对偶数**）是单点，但结构层含幂零元。这是"无穷小箭头"，切向量的代数模型。

**与经典代数簇的关系**：若 $A$ 是有限生成 $k$-代数且约化（无幂零元），$k$ 代数闭，则 $\text{Spec } A$ 的经典点（闭点）对应于 $A$ 的极大理想，即 $k^n$ 中代数方程的解集。概形比簇多了 generic 点和幂零结构。

### (b) 局部环层与茎的结构

**定理**：设 $X = \text{Spec } A$，$\mathcal{O}_X$ 为上述结构层。则：

1. $(X, \mathcal{O}_X)$ 是局部环层空间。
2. 对每个 $x = \mathfrak{p} \in X$，茎 $\mathcal{O}_{X,x}$ 同构于局部化 $A_{\mathfrak{p}}$。
3. 极大理想 $\mathfrak{m}_x \subset \mathcal{O}_{X,x}$ 对应于 $A_{\mathfrak{p}}$ 的极大理想 $\mathfrak{p} A_{\mathfrak{p}}$。

*证明*：

**步骤1：层的良定性**。需验证预层 $D(f) \mapsto A_f$ 满足层的粘合公理。

设 $\{D(f_i)\}$ 覆盖 $D(f)$，$s_i \in A_{f_i}$ 满足在重叠处一致：$s_i|_{D(f_i f_j)} = s_j|_{D(f_i f_j)}$。需证存在唯一的 $s \in A_f$ 限制为 $s_i$。

由拟紧性，可设覆盖有限。设 $s_i = a_i / f_i^{n_i} \in A_{f_i}$。一致条件意味着：在 $A_{f_i f_j}$ 中，$a_i f_j^{n_j} / (f_i f_j)^{n_i+n_j} = a_j f_i^{n_i} / (f_i f_j)^{n_i+n_j}$，即：
$$(f_i f_j)^N (a_i f_j^{n_j} - a_j f_i^{n_i}) = 0 \in A$$
对某个 $N$。

构造 $s = \sum_i b_i a_i / f^{m}$，利用 $D(f) = \bigcup D(f_i)$ 意味着 $f^k = \sum_i c_i f_i$ 对某 $k$。通过标准的"partition of unity"论证（在代数中利用理想的生成），可以粘合 $s_i$ 为全局截面。

**步骤2：茎的计算**。由定义：
$$\mathcal{O}_{X,x} = \varinjlim_{U \ni x} \mathcal{O}_X(U)$$
极限取遍 $x$ 的所有开邻域。由于 $\{D(f) : f \notin \mathfrak{p}\}$ 构成 $x$ 的邻域基，且 $A_f$ 的归纳系统对应于 $A$ 在乘法集 $S = A \setminus \mathfrak{p}$ 上的局部化：
$$\mathcal{O}_{X,x} = \varinjlim_{f \notin \mathfrak{p}} A_f = A_{\mathfrak{p}} = S^{-1}A$$

**步骤3：局部环性质**。$A_{\mathfrak{p}}$ 的极大理想是 $\mathfrak{p} A_{\mathfrak{p}} = \{a/s : a \in \mathfrak{p}, s \notin \mathfrak{p}\}$。这是因为：
- 若 $a/s \notin \mathfrak{p} A_{\mathfrak{p}}$，则 $a \notin \mathfrak{p}$，故 $a/s$ 可逆（逆为 $s/a$）
- 反之，$\mathfrak{p} A_{\mathfrak{p}}$ 中的元素都不是单位

因此 $A_{\mathfrak{p}}$ 是局部环，极大理想为 $\mathfrak{p} A_{\mathfrak{p}}$，剩余域为：
$$k(x) = A_{\mathfrak{p}} / \mathfrak{p} A_{\mathfrak{p}} \cong \text{Frac}(A/\mathfrak{p})$$

当 $\mathfrak{p}$ 是极大理想且 $A$ 是有限生成 $k$-代数时，$k(x) \cong k$（若 $k$ 代数闭）。

**步骤4：结构层的整体截面**。一个重要结果是：
$$\Gamma(X, \mathcal{O}_X) = A$$
即仿射概形的整体截面环就是原始环。这由层的定义直接得到（取 $f=1$，$D(1) = X$）。

**函子性**：映射 $\varphi: A \to B$ 诱导概形态射 $f: \text{Spec } B \to \text{Spec } A$：
- 拓扑上：$f(\mathfrak{q}) = \varphi^{-1}(\mathfrak{q})$（素理想的原像是素的）
- 层上：$\mathcal{O}_A \to f_* \mathcal{O}_B$ 由 $A_f \to B_{\varphi(f)}$ 给出

这建立了范畴等价：
$$(\text{交换环})^{op} \xrightarrow{\cong} (\text{仿射概形})$$
$$A \mapsto \text{Spec } A$$

### (c) 纤维积及其几何意义

**范畴论中的纤维积**：在任意范畴中，对象 $X, Y$  over $S$（即配备到 $S$ 的态射）的**纤维积** $X \times_S Y$ 是对象 $W$ 加上投影 $p_X: W \to X$, $p_Y: W \to Y$，使得图表交换：
$$\begin{array}{ccc}
W & \xrightarrow{p_Y} & Y \\
p_X \downarrow & & \downarrow \\
X & \longrightarrow & S
\end{array}$$
且满足泛性质：对任意 $Z \to X$, $Z \to Y$ 使得合成到 $S$ 一致，存在唯一的 $Z \to W$ 使图表交换。

**定理**：概形范畴具有任意纤维积。

*构造*：

**仿射情形**：设 $X = \text{Spec } A$, $Y = \text{Spec } B$, $S = \text{Spec } C$，态射对应于环同态 $C \to A$ 和 $C \to B$。则：
$$X \times_S Y = \text{Spec } (A \otimes_C B)$$
这是因为张量积 $A \otimes_C B$ 满足环的推出（pushout）泛性质，与概形的纤维积（pullback）对偶。

**一般情形**：对一般概形，取 $S$ 的仿射开覆盖 $\{S_i = \text{Spec } C_i\}$。设 $X_i, Y_i$ 为其原像的仿射开覆盖。在每个仿射片上构造 $X_{ij} \times_{S_i} Y_{kl} = \text{Spec}(A_{ij} \otimes_{C_i} B_{kl})$，然后利用层的粘合公理将这些仿射概形粘合成全局的 $X \times_S Y$。

**几何意义**：

1. **纤维**：设 $f: X \to S$ 为态射，$s \in S$ 为点。则**纤维** $X_s$ 定义为：
   $$X_s = X \times_S \text{Spec } k(s)$$
   其中 $k(s) = \mathcal{O}_{S,s} / \mathfrak{m}_s$ 是剩余域。

   在仿射情形，$X = \text{Spec } A$, $S = \text{Spec } C$, $s = \mathfrak{p} \in S$。则：
   $$X_s = \text{Spec}(A \otimes_C k(\mathfrak{p})) = \text{Spec}(A_{\mathfrak{p}} / \mathfrak{p} A_{\mathfrak{p}})$$
   这是 $X$ 在 $s$ 上的"几何纤维"——即将方程系数在点 $s$ 处取值后的解集。

   **例子**：设 $X = \text{Spec } \mathbb{Z}[x]/(x^2 - 2)$，$S = \text{Spec } \mathbb{Z}$。则：
   - 在 $s = (0)$（generic 点）：$X_s = \text{Spec } \mathbb{Q}[x]/(x^2 - 2) = \text{Spec } \mathbb{Q}(\sqrt{2})$，两点（若取代数闭包）。
   - 在 $s = (p)$（$p$ 为奇素数）：若 $2$ 是模 $p$ 二次剩余，$X_s = \text{Spec } \mathbb{F}_p \times \mathbb{F}_p$（两点）；若否，$X_s = \text{Spec } \mathbb{F}_{p^2}$（一点，域扩张）。
   - 在 $s = (2)$：$X_s = \text{Spec } \mathbb{F}_2[x]/(x^2) = \text{Spec } \mathbb{F}_2[\varepsilon]/\varepsilon^2$（对偶数，非约化）。

2. **基变换**：纤维积允许任意**基变换**（base change）。若 $S' \to S$ 是态射，则 $X' = X \times_S S'$ 是 $X$ 拉回到新基上的概形。这是研究概形族的重要工具。

   **例子**：设 $X = \text{Spec } \mathbb{R}[x]/(x^2 + 1)$（一点，对应复数域 $\mathbb{C}$）。基变换到 $\mathbb{C}$：
   $$X_{\mathbb{C}} = \text{Spec } \mathbb{C}[x]/(x^2+1) = \text{Spec } \mathbb{C} \times \text{Spec } \mathbb{C}$$
   两点！这说明基扩张可以"分裂"原先不可约的几何对象。

3. **对角与分离性**：态射 $f: X \to S$ 是**分离的**（separated），如果对角映射 $\Delta: X \to X \times_S X$ 是闭浸入。这是 Hausdorff 条件的概形类比。仿射概形在仿射基上总是分离的。

4. **族的交**：设 $X, Y \subset Z$ 为 $S$ 上的子概形。则 $X \cap Y = X \times_Z Y$ 给出正确的概形论交（可能含幂零元，反映"非横截"交的信息）。

   **例子**：在 $\mathbb{A}^2 = \text{Spec } k[x,y]$ 中，直线 $y=0$ 和抛物线 $y=x^2$ 的交：
   $$\text{Spec } k[x,y]/(y) \times_{\mathbb{A}^2} \text{Spec } k[x,y]/(y-x^2) = \text{Spec } k[x]/(x^2)$$
   这是 $\mathbb{A}^1$ 的原点上的对偶数，反映了两条曲线在原点处**切触**（tangent intersection），相交重数为2。

纤维积是概形理论中最重要的构造之一，它将几何直观（纤维、交、基变换）统一在范畴论的框架中，使得代数几何能够处理算术（不同特征）、形变（幂零元）和几何的统一问题。
