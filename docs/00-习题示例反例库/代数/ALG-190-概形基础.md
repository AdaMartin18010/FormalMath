# 概形基础

## 题目

**(a)** 定义局部环层空间、仿射概形和概形。

**(b)** 证明 Spec A 的层 O_X 是局部环层，茎同构于 A_p。

**(c)** 讨论概形的纤维积及其几何意义。

## 解答

### (a) 概形定义

局部环层空间 (X, O_X) 被仿射开覆盖。

仿射概形：X = Spec A，O_X(D(f)) = A_f。

### (b) 局部环层

对 x = p ∈ Spec A，茎 O_{X,x} = lim→ O_X(U) ≅ A_p。

极大理想 m_p 是局部环的唯一极大理想。

### (c) 纤维积

X ×_S Y 存在且唯一（范畴论）。几何：纤维的积。

A ⊗_C B 是 B 代数的推出。
