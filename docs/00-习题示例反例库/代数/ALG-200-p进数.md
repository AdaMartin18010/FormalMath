# p-进数

## 题目

**(a)** 构造 p-进数域 Q_p 并证明其拓扑性质。

**(b)** 证明 Hensel 引理及其应用。

**(c)** 讨论局部域 Galois 理论（局部类域论）。

## 解答

### (a) Q_p 的构造

Q 关于 |·|_p 完备化。元素：x = ∑_{n≥N} a_n p^n，a_n ∈ {0,...,p-1}。

拓扑：非阿基米德，Z_p 紧致开子环。

### (b) Hensel 引理

f ∈ Z_p[x]，若 f(a) ≡ 0 (mod p) 且 f'(a) ≢ 0 (mod p)，则存在 α ∈ Z_p，f(α)=0，α ≡ a (mod p)。

### (c) 局部类域论

局部 Artin 映射：rec_K: K^× → Gal(K^{ab}/K)，开子群 ↔ 有限 Abel 扩张。
