# 算术几何综合

## 题目

**(a)** 证明 Faltings 定理：曲线有理点有限（Mordell 猜想）。

**(b)** 讨论 Arakelov 几何和算术相交理论。

**(c)** 证明 p-进上同调理论比较（Crystalline-étale-de Rham）。

## 解答

### (a) Faltings 定理

方法：p-进 Hodge 理论 + 曲面的Kodaira-Parshin 构造。

或 Vojta 的方法：Diophantine 逼近。

### (b) Arakelov 几何

算术曲面 X → Spec O_K，纤维在无穷远点加 Hermitian 度规。

算术相交数：(D₁, D₂) = (D₁, D₂)_fin + (D₁, D₂)_∞。

### (c) p-进比较

Crystalline-étale：Berkovich 几何或 Faltings 的几乎纯分支。

de Rham-étale：Fontaine 的 B_{dR}。
