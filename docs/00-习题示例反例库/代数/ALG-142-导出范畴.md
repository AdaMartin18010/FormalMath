# 导出范畴初步

## 题目

设 $\mathcal{A}$ 是Abel范畴，$\mathcal{C}(\mathcal{A})$ 是链复形范畴。

**(a)** 定义同伦范畴 $\mathcal{K}(\mathcal{A})$：态射是链映射的同伦类。证明同调 $H: \mathcal{K}(\mathcal{A}) \to \mathcal{A}$ 是良定义的函子。

**(b)** 定义导出范畴 $D(\mathcal{A})$：对拟同构（诱导同调同构的链映射）取局部化。

**(c)** 证明：$A \in \mathcal{A}$ 的投射分解 $P_\bullet$ 在 $D(\mathcal{A})$ 中唯一确定 $A$。

---

## 解答

### (a) 同伦范畴

**证明：**

若 $f \sim g$（链同伦），则 $H_n(f) = H_n(g)$。

同调函子通过商范畴分解。

**详细展开**：

设 $f, g: C_\bullet \to D_\bullet$ 是链同伦的链映射，即存在同伦算子 $h_n: C_n \to D_{n+1}$ 使得：
$$f_n - g_n = \partial_{n+1}^D h_n + h_{n-1} \partial_n^C$$

对任意闭链 $z \in Z_n(C)$（即 $\partial z = 0$），有：
$$f_n(z) - g_n(z) = \partial_{n+1}^D(h_n(z))$$

因此 $f_n(z)$ 与 $g_n(z)$ 相差一个边缘，即 $[f_n(z)] = [g_n(z)] \in H_n(D)$。

这说明 $H_n(f) = H_n(g)$ 作为同调群上的同态，因此 $H_n$ 可以良定义地作用于链同伦等价类。

**函子性的验证**：
- **恒等**：$H_n(\text{id}_{C}) = \text{id}_{H_n(C)}$（显然）
- **复合**：$H_n(g \circ f) = H_n(g) \circ H_n(f)$（由同调的函子性）

因此 $H: \mathcal{K}(\mathcal{A}) \to \mathcal{A}$（严格地说，$H_n$ 对每个 $n$ 是一个函子到 $\mathcal{A}$）是良定义的函子。$\square$

---

### (b) 导出范畴

**构造：**

对拟同构类 $W$，$D(\mathcal{A}) = \mathcal{K}(\mathcal{A})[W^{-1}]$。

态射是 roofs：$X \leftarrow Z \to Y$，左边是拟同构。

或：$\text{Hom}_{D}(X,Y) = \varinjlim_{X' \xrightarrow{\sim} X} \text{Hom}_{\mathcal{K}}(X', Y)$。

**直观解释**：在导出范畴中，拟同构（在同调上诱导同构的链映射）被形式地变为同构。这类似于在拓扑中将弱等价（weak equivalence）转化为同伦等价。

**构造细节**：

1. **对象**：与 $\mathcal{K}(\mathcal{A})$ 相同，即链复形
2. **态射**：从 $X$ 到 $Y$ 的态射由等价类 of roofs 表示：
   ```
       Z
      / \
   qis  f
    /     \
   X       Y
   ```
   其中 $Z \to X$ 是拟同构。两个 roofs 等价，如果存在更大的 roof 使两图交换。

3. **复合**：通过拉回（homotopy pullback）构造。给定：
   ```
       Z_1          Z_2
      /  \          /  \
     /    \        /    \
    X      Y ---- Y      W
   ```
   取 $Z_1$ 和 $Z_2$ 在 $Y$ 上的同伦纤维积得到复合 roof。

**关键性质**：
- 局部化函子 $Q: \mathcal{K}(\mathcal{A}) \to D(\mathcal{A})$ 满足泛性质
- $Q(s)$ 对所有拟同构 $s$ 是同构
- 导出函子 $RF$ 通过 $D(\mathcal{A})$ 定义

### (c) 投射分解的唯一性

**证明：**

两投射分解 $P_\bullet$，$Q_\bullet$ 间有比较定理给出链同伦等价。

故在 $\mathcal{K}(\mathcal{A})$ 中同构，在 $D(\mathcal{A})$ 中亦然。

投射分解给出 $A$ 在 $D(\mathcal{A})$ 中的标准代表。

**详细展开**：

**比较定理**：设 $P_\bullet \to A$ 和 $Q_\bullet \to B$ 是投射分解，$f: A \to B$ 是态射。则存在链映射 $\tilde{f}: P_\bullet \to Q_\bullet$ 使下图交换：
```
P_\bullet → A
↓          ↓ f
Q_\bullet → B
```
且 $\tilde{f}$ 在链同伦意义下唯一。

**应用到 $A = B$，$f = \text{id}_A$**：
- 存在 $\tilde{\text{id}}: P_\bullet \to Q_\bullet$ 和 $\tilde{\text{id}}': Q_\bullet \to P_\bullet$
- 复合 $\tilde{\text{id}}' \circ \tilde{\text{id}} \sim \text{id}_{P_\bullet}$（由唯一性）
- 复合 $\tilde{\text{id}} \circ \tilde{\text{id}}' \sim \text{id}_{Q_\bullet}$（由唯一性）

因此 $P_\bullet$ 和 $Q_\bullet$ 在 $\mathcal{K}(\mathcal{A})$ 中链同伦等价，进而在 $D(\mathcal{A})$ 中同构。

**在 $D(\mathcal{A})$ 中的意义**：

对象 $A \in \mathcal{A}$ 可以等同于 $D(\mathcal{A})$ 中的对象 $A[0]$（集中在0次的复形）。投射分解 $P_\bullet \to A[0]$ 是拟同构，因此在 $D(\mathcal{A})$ 中：
$$A[0] \cong P_\bullet$$

这意味着在导出范畴中，$A$ 的"同调信息"完全由 $P_\bullet$ 编码。$\square$

## 拓展思考

**与Ext的关系**：对 $A, B \in \mathcal{A}$，
$$\text{Ext}^n_\mathcal{A}(A, B) = \text{Hom}_{D(\mathcal{A})}(A[0], B[n])$$

这说明Ext群可以纯粹在导出范畴中定义，无需提及投射/内射分解。

**三角结构**：导出范畴是三角范畴。对任意链映射 $f: C \to D$，有**映射锥**（mapping cone）给出正合三角：
$$C \xrightarrow{f} D \to \text{Cone}(f) \to C[1]$$

这一结构与拓扑中的 Puppe 序列完全平行。
