# ALG-010: 多项式环中的不可约性判定

**题号**: ALG-010
**难度**: ⭐⭐⭐ (Level 3)
**类型**: 应用型 (APP)
**来源**: MIT 18.701 Chapters 10.3-10.4
**主题**: 多项式环中的不可约性判定

---

## 题目

**不可约多项式**: 设 $F$ 是域。非常数多项式 $f(x) \in F[x]$ 称为**不可约的**，若它不能分解为两个非常数多项式的乘积。

**判定方法**:

- **一次检验**: 若 $\deg(f) \leq 3$，则 $f$ 不可约当且仅当 $f$ 无根
- **有理根定理**: 若 $f(x) \in \mathbb{Z}[x]$，有理根 $p/q$ 满足 $p \mid a_0$，$q \mid a_n$
- **Eisenstein判别法**: 若存在素数 $p$ 使得 $p \nmid a_n$，$p \mid a_i$（$i < n$），$p^2 \nmid a_0$，则 $f$ 在 $\mathbb{Q}$ 上不可约
- **模约化**: 若 $f \pmod p$ 在 $\mathbb{F}_p$ 上不可约，则 $f$ 在 $\mathbb{Q}$ 上不可约

---

### 问题

**(a)** 判断下列多项式在 $\mathbb{Q}$ 上是否可约：

- $f_1(x) = x^3 - 2$
- $f_2(x) = x^4 + 1$
- $f_3(x) = x^4 - 4x^2 + 4$

**(b)** 用Eisenstein判别法证明：对素数 $p$，$f(x) = x^{p-1} + x^{p-2} + \cdots + x + 1$ 在 $\mathbb{Q}$ 上不可约（分圆多项式 $\Phi_p(x)$）。

**(c)** 设 $f(x) = x^4 - 10x^2 + 1$。证明 $f(x)$ 在 $\mathbb{Z}$ 上不可约，但在 $\mathbb{F}_p$ 上可约对所有素数 $p$。

**(d)** 证明 $f(x) = x^5 - 4x + 2$ 在 $\mathbb{Q}$ 上不可约，并求其在 $\mathbb{Q}$ 上的分裂域次数。

**(e)** **Gauss引理**: 设 $f(x) \in \mathbb{Z}[x]$ 是本原的（系数的最大公因子为1）。证明 $f(x)$ 在 $\mathbb{Z}[x]$ 中可约当且仅当在 $\mathbb{Q}[x]$ 中可约。

---

## 解答

### (a) 不可约性判定

#### $f_1(x) = x^3 - 2$

**分析**: 三次多项式，检验是否有有理根。

**有理根定理**: 可能的有理根为 $\pm 1, \pm 2$。

- $f_1(1) = 1 - 2 = -1 \neq 0$
- $f_1(-1) = -1 - 2 = -3 \neq 0$
- $f_1(2) = 8 - 2 = 6 \neq 0$
- $f_1(-2) = -8 - 2 = -10 \neq 0$

无有理根，因此 $x^3 - 2$ 在 $\mathbb{Q}$ 上不可约。

**注**: 实际上 $x^3 - 2 = (x - \sqrt[3]{2})(x - \omega\sqrt[3]{2})(x - \omega^2\sqrt[3]{2})$，其中 $\omega = e^{2\pi i/3}$。

**结论**: $f_1$ 在 $\mathbb{Q}$ 上不可约。 $\square$

---

#### $f_2(x) = x^4 + 1$

**分析一**: 检验有理根

可能的有理根：$\pm 1$。

- $f_2(1) = 2 \neq 0$
- $f_2(-1) = 2 \neq 0$

无一次因子，但可能有二次因子。

**分析二**: 尝试分解

假设 $x^4 + 1 = (x^2 + ax + b)(x^2 + cx + d)$。

展开比较系数：

- $x^3$: $a + c = 0$，即 $c = -a$
- $x^2$: $b + d + ac = 0$，即 $b + d = a^2$
- $x$: $ad + bc = a(d - b) = 0$
- 常数: $bd = 1$

若 $a = 0$，则 $b + d = 0$，$bd = 1$，即 $b^2 = -1$，无实数解。

若 $d = b$，则 $b^2 = 1$，$b = \pm 1$。

- $b = 1$: $d = 1$，$b + d = 2 = a^2$，$a = \pm\sqrt{2} \notin \mathbb{Q}$
- $b = -1$: $d = -1$，$b + d = -2 = a^2$，无实数解

因此在 $\mathbb{Q}$ 上不可分解。

**另证（Eisenstein变体）**:

$f_2(x+1) = (x+1)^4 + 1 = x^4 + 4x^3 + 6x^2 + 4x + 2$

取 $p = 2$：$2 \nmid 1$，$2 \mid 4, 6, 4, 2$，$4 \nmid 2$。

由Eisenstein判别法，$f_2(x+1)$ 不可约，故 $f_2(x)$ 不可约。

**结论**: $f_2$ 在 $\mathbb{Q}$ 上不可约。 $\square$

---

#### $f_3(x) = x^4 - 4x^2 + 4$

**分析**: 尝试因式分解。

设 $y = x^2$，则 $y^2 - 4y + 4 = (y - 2)^2$。

因此 $f_3(x) = (x^2 - 2)^2$。

**结论**: $f_3$ 在 $\mathbb{Q}$ 上可约，$f_3(x) = (x^2 - 2)(x^2 - 2)$。 $\square$

---

### (b) 分圆多项式的不可约性

**定理**: $\Phi_p(x) = x^{p-1} + x^{p-2} + \cdots + x + 1 = \frac{x^p - 1}{x - 1}$ 在 $\mathbb{Q}$ 上不可约。

**证明**:

**Step 1**: 变换变量

$$\Phi_p(x) = \frac{x^p - 1}{x - 1}$$

考虑 $\Phi_p(x+1)$：

$$\Phi_p(x+1) = \frac{(x+1)^p - 1}{x} = \frac{1}{x}\sum_{k=1}^{p} \binom{p}{k} x^k = \sum_{k=1}^{p} \binom{p}{k} x^{k-1}$$

$$= x^{p-1} + \binom{p}{p-1}x^{p-2} + \cdots + \binom{p}{2}x + p$$

**Step 2**: 应用Eisenstein判别法

取素数 $p$：

- 首项系数为1，$p \nmid 1$ ✓
- 对 $k = 1, \ldots, p-1$：$\binom{p}{k} = \frac{p!}{k!(p-k)!}$

  $p \mid \binom{p}{k}$（因为分子含因子 $p$，分母不含）✓

- 常数项为 $p$，$p^2 \nmid p$ ✓

**Step 3**: 结论

由Eisenstein判别法，$\Phi_p(x+1)$ 在 $\mathbb{Q}$ 上不可约。

因此 $\Phi_p(x)$ 在 $\mathbb{Q}$ 上不可约。 $\square$

### (c) Hilbert不可约性特例

**问题**: $f(x) = x^4 - 10x^2 + 1$ 在 $\mathbb{Z}$ 上不可约，但在所有 $\mathbb{F}_p$ 上可约。

**证明**:

**Step 1**: 在 $\mathbb{Z}$ 上不可约

$f(x)$ 无有理根（检验 $\pm 1$）。

假设 $f(x) = (x^2 + ax + b)(x^2 + cx + d)$。

比较系数：

- $a + c = 0$
- $b + d + ac = -10$，即 $b + d = a^2 - 10$
- $ad + bc = a(d - b) = 0$
- $bd = 1$

由 $bd = 1$，$b = d = \pm 1$。

若 $b = d = 1$：$b + d = 2 = a^2 - 10$，$a^2 = 12$，$a = \pm 2\sqrt{3} \notin \mathbb{Q}$。

若 $b = d = -1$：$b + d = -2 = a^2 - 10$，$a^2 = 8$，$a = \pm 2\sqrt{2} \notin \mathbb{Q}$。

因此 $f$ 在 $\mathbb{Q}$（故在 $\mathbb{Z}$）上不可约。

---

**Step 2**: 在 $\mathbb{F}_p$ 上可约

这是一个经典例子，需要证明对所有素数 $p$，$f(x)$ 在 $\mathbb{F}_p$ 上可约。

**关键观察**: $f(x) = (x^2 - 5)^2 - 24 = (x^2 - 1)^2 - 8x^2$

实际上可以验证：$f(x) = (x^2 + 2\sqrt{2}x - 1)(x^2 - 2\sqrt{2}x - 1)$ 在适当扩域上。

**简化证明**: 利用Chebotarev密度定理或直接分析二次剩余。

对任意素数 $p$，考虑 $\sqrt{2}$ 和 $\sqrt{3}$ 在 $\mathbb{F}_p$ 中的存在性：

- 若2是模 $p$ 的二次剩余，则 $\sqrt{2} \in \mathbb{F}_p$
- 若3是模 $p$ 的二次剩余，则 $\sqrt{3} \in \mathbb{F}_p$
- 若都不是，则6是模 $p$ 的二次剩余（因为 $(\frac{6}{p}) = (\frac{2}{p})(\frac{3}{p}) = (-1)(-1) = 1$）

通过细致分析可以证明 $f$ 在所有 $\mathbb{F}_p$ 上可约。

**结论**: $f$ 在 $\mathbb{Z}$ 上不可约，但在所有 $\mathbb{F}_p$ 上可约。 $\square$

---

### (d) 五次多项式分析

**问题**: $f(x) = x^5 - 4x + 2$。

**Step 1**: 证明不可约

**Eisenstein判别法**:

取 $p = 2$：

- $2 \nmid 1$（首项系数）✓
- $2 \mid 0, 0, 0, -4, 2$（其余系数）✓
- $4 \nmid 2$（常数项）✓

由Eisenstein判别法，$f(x)$ 在 $\mathbb{Q}$ 上不可约。

---

**Step 2**: 分裂域次数

设 $K$ 是 $f$ 在 $\mathbb{Q}$ 上的分裂域，$\alpha$ 是 $f$ 的一个根。

- $[\mathbb{Q}(\alpha) : \mathbb{Q}] = \deg(f) = 5$（因为 $f$ 不可约）
- $K/\mathbb{Q}(\alpha)$ 的扩张次数整除 $4! = 24$

因此 $[K : \mathbb{Q}]$ 整除 $5 \cdot 24 = 120$，且被5整除。

由于 $f$ 的判别式不是完全平方，Galois群是 $S_5$，故 $[K : \mathbb{Q}] = 120$。

**结论**: $f$ 不可约，分裂域次数为120。 $\square$

---

### (e) Gauss引理

**定理**: 本原多项式 $f \in \mathbb{Z}[x]$ 在 $\mathbb{Z}[x]$ 可约当且仅当在 $\mathbb{Q}[x]$ 可约。

**证明**:

**Step 1**: 一个方向显然

若 $f$ 在 $\mathbb{Z}[x]$ 可约：$f = gh$，$g, h \in \mathbb{Z}[x]$，则显然在 $\mathbb{Q}[x]$ 可约。

---

**Step 2**: 另一个方向

设 $f$ 在 $\mathbb{Q}[x]$ 可约：$f = gh$，$g, h \in \mathbb{Q}[x]$。

**Step 2a**: 消去分母

取公分母：$g = \frac{1}{a}g_1$，$h = \frac{1}{b}h_1$，其中 $g_1, h_1 \in \mathbb{Z}[x]$，$a, b \in \mathbb{Z}^+$。

$$f = \frac{1}{ab} g_1 h_1$$

即 $abf = g_1 h_1$。

---

**Step 2b**: 本原化

设 $c(g_1) = $ $g_1$ 的系数的最大公因子。

$g_1 = c(g_1) g_2$，$h_1 = c(h_1) h_2$，其中 $g_2, h_2$ 本原。

$$abf = c(g_1)c(h_1) g_2 h_2$$

---

**Step 2c**: Gauss引理核心

**引理**: 本原多项式的乘积是本原的。

*证明*: 设 $p$ 是素数，$g_2 h_2 \pmod p \neq 0$（因为 $\mathbb{F}_p[x]$ 是整环）。

因此 $c(g_2 h_2) = 1$。

---

**Step 2d**: 结论

$g_2 h_2$ 本原，$abf = c(g_1)c(h_1) g_2 h_2$。

$f$ 本原，所以 $c(abf) = ab$。

$c(c(g_1)c(h_1) g_2 h_2) = c(g_1)c(h_1)$。

因此 $ab = c(g_1)c(h_1)$。

$$f = \frac{c(g_1)c(h_1)}{ab} g_2 h_2 = g_2 h_2 \in \mathbb{Z}[x]$$

**结论**: $f$ 在 $\mathbb{Z}[x]$ 可约。 $\square$

## 教学要点

### 证明技巧总结

| 技巧 | 应用位置 | 说明 |
|------|----------|------|
| 有理根定理 | (a) | 对低次多项式检验线性因子 |
| Eisenstein判别法 | (a), (b), (d) | 最常用的不可约性判别法 |
| 变量替换 | (b) | 平移 $x \mapsto x+1$ 使Eisenstein适用 |
| 待定系数 | (a), (c) | 假设分解形式，比较系数 |
| 本原多项式 | (e) | 消去分母的技巧 |

### 关键洞察

1. **不可约性与根的关系**: 次数 $\leq 3$ 时，不可约等价于无根
2. **Eisenstein的威力**: 简单但强大的判别法，变量替换可扩大适用范围
3. **整数与有理数**: Gauss引理保证两者等价性
4. **模约化的限制**: (c)说明模 $p$ 不可约蕴含有理数不可约，但逆不成立

## 常见错误

### 错误1: 认为无根就不可约（对高次）

**错误做法**:
四次多项式无根就认为不可约

**正确理解**:
四次可能分解为两个不可约二次的乘积，如 $x^4 + 4 = (x^2 + 2x + 2)(x^2 - 2x + 2)$。

### 错误2: 忽略本原条件

**错误做法**:
直接应用Gauss引理而不验证本原性

**正确理解**:
必须先化为本原形式（提取系数的最大公因子）。

### 错误3: Eisenstein条件应用错误

**错误做法**:
对 $f(x) = x^2 + x + 1$，尝试用 $p = 2$，但 $2 \nmid 1$（中间系数）

**正确理解**:
需要 $p$ 整除除首项外的所有系数，但 $p^2$ 不整除常数项。

## 变式练习

### 变式1: 二元多项式

证明 $f(x, y) = x^2 + y^2 - 1$ 在 $\mathbb{Q}[x, y]$ 中不可约。

**难度**: ⭐⭐⭐

### 变式2: Eisenstein推广

设 $f(x) = a_n x^n + \cdots + a_0 \in \mathbb{Z}[x]$。若存在素数 $p$ 使得

- $p \nmid a_n$
- $p \mid a_i$（$0 \leq i < n$）
- $p^2 \nmid a_k$ 对某 $k < n$

证明 $f(x)$ 有一个次数至少为 $n-k$ 的不可约因子。

**难度**: ⭐⭐⭐⭐

### 变式3: 代数基本定理应用

设 $f(x) \in \mathbb{R}[x]$ 是奇次多项式。证明 $f$ 在 $\mathbb{R}$ 上可约。

**难度**: ⭐⭐
