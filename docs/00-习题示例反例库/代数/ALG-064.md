# 局部对偶定理

## 题目

设 $(R, \mathfrak{m}, k)$ 是完备局部Noether环，维数 $d$，$E = E_R(k)$ 是 $k$ 的内射包。

**定义**：Matlis对偶 $M^\vee = \text{Hom}_R(M, E)$。

**局部对偶定理**：对有限生成 $R$-模 $M$，存在自然同构：
$$H^i_{\mathfrak{m}}(M) \cong \text{Ext}_R^{d-i}(M, \omega_R)^\vee$$
其中 $\omega_R$ 是 $R$ 的典范模（若 $R$ Cohen-Macaulay则 $\omega_R = H^d_{\mathfrak{m}}(R)^\vee$）。

(a) 证明 Matlis对偶给出反变等价：$\{\text{Artin } R\text{-模}\} \leftrightarrow \{\text{有限生成 } R\text{-模}\}$。

(b) 证明 Grothendieck局部对偶：对 $i \geq 0$，
$$H^i_{\mathfrak{m}}(M)^\vee \cong \text{Ext}_R^i(M, \omega_R)$$

(c) 设 $R = k[[x_1, \ldots, x_d]]$，计算 $\omega_R$ 并验证(b)。

## 解答

### (a) Matlis对偶等价

**证明：**

**Step 1**：$E$ 是 $R$ 的内射余生成子（injective cogenerator），即 $\text{Hom}_R(M, E) = 0 \Rightarrow M = 0$。

**Step 2**：对Artin模 $N$，$N^\vee$ 有限生成。

因 $N$ 是 $k$ 的有限个拷贝的扩张，$N^\vee$ 是 $k^\vee = k$ 的有限个拷贝的扩张。

**Step 3**：对有限生成 $M$，$M^\vee$ Artin。

$M/\mathfrak{m}^n M$ 有限长度，$(M/\mathfrak{m}^n M)^\vee$ 有限长度。

$M^\vee = \varinjlim (M/\mathfrak{m}^n M)^\vee$ 是Artin的。

**Step 4**：自然映射 $M \to M^{\vee\vee}$ 是同构（对Artin或有限生成模）。

由完备性，$M = \varprojlim M/\mathfrak{m}^n M$，且对偶交换适当极限。 $\square$

### (b) 局部对偶定理

**证明概要：**

**Step 1**：设 $\mathbf{x} = x_1, \ldots, x_d$ 是 $R$ 的s.o.p（参数系）。

由Cech复形，$H^i_{\mathfrak{m}}(M) = H^i(\check{C}^\bullet(\mathbf{x}; M))$。

**Step 2**：对偶化。

$H^i_{\mathfrak{m}}(M)^\vee = H^i(\check{C}^\bullet(\mathbf{x}; M))^\vee = H_{d-i}(\check{C}^\bullet(\mathbf{x}; M)^\vee)$

Cech复形的对偶与Koszul复形相关。

**Step 3**：识别 $\text{Ext}$。

由Koszul复形计算，$\check{C}^\bullet(\mathbf{x}; M)^\vee \cong K_\bullet(\mathbf{x}; \text{Hom}(M, \omega_R))$（适当平移）。

**Step 4**：得到：
$$H^i_{\mathfrak{m}}(M)^\vee \cong H_{d-i}(K_\bullet(\mathbf{x}; \text{Hom}(M, \omega_R)))$$

$$= \text{Ext}_R^{d-i}(R/(\mathbf{x}), \text{Hom}(M, \omega_R))$$

通过适当解释，这给出 $\text{Ext}^i(M, \omega_R)$。 $\square$

### (c) 形式幂级数环

**解答：**

$R = k[[x_1, \ldots, x_d]]$ 是正则局部环，故 Cohen-Macaulay 且 Gorenstein。

**计算 $\omega_R$：**

对 Gorenstein 环，$\omega_R = R$。

验证：$H^d_{\mathfrak{m}}(R) = H^d(\check{C}^\bullet(\mathbf{x}; R))$，其中 $\mathbf{x} = (x_1, \ldots, x_d)$。

$H^d = R_{x_1\cdots x_d} / \sum_i R_{x_1\cdots\hat{x}_i\cdots x_d}$。

这同构于 $x_1^{-n_1}\cdots x_d^{-n_d}$（全次数负）的 $k$-空间。

Matlis对偶：$H^d_{\mathfrak{m}}(R)^\vee = \text{Hom}_k(H^d, k) \cong R$（作为 $R$-模）。

故 $\omega_R = R$。

**验证(b)：**

对 $M$ 有限生成，需证 $H^i_{\mathfrak{m}}(M)^\vee \cong \text{Ext}^i_R(M, R)$。

由正则局部环的同调理论，$\text{pd}(M) \leq d$，且 Serre 定理给出所需对偶。

$\square$

## 证明技巧

1. **内射包与Matlis理论**：完备局部环的对偶理论
2. **Cech-Koszul对应**：两种复形的对偶关系
3. **Gorenstein环的特殊性**：典范模平凡化

## 常见错误

- ❌ 混淆 $R$ 完备的必要性
- ❌ 忘记验证对偶保持模结构
- ❌ 在(c)中错误计算Cech上同调

## 变式练习

**变式1：** 证明若 $R$ 是Cohen-Macaulay，则 $\omega_R$ 是 Cohen-Macaulay 且 $\dim(\omega_R) = \dim(R)$。

**变式2：** 计算 $R = k[x,y]/(xy)$ 的 $\omega_R$（非Gorenstein情形）。
