# 算术几何基础

## 题目

**(a)** 证明Mordell-Weil定理：椭圆曲线有理点群有限生成。

**(b)** 定义Tate模和Galois表示。

**(c)** 讨论BSD猜想。

---

## 解答

### (a) Mordell-Weil定理

#### 陈述

**定理**（Mordell 1922, Weil 1928）：设 $E$ 是定义在数域 $K$ 上的椭圆曲线，则Mordell-Weil群 $E(K)$ 是有限生成Abel群：
$$E(K) \cong E(K)_{\text{tors}} \oplus \mathbb{Z}^r$$

其中 $E(K)_{\text{tors}}$ 是有限挠子群，$r = \text{rank}(E(K))$ 称为**秩**。

#### 证明框架

Mordell-Weil定理的证明分两部分：

**I. 弱Mordell-Weil定理**：$E(K)/mE(K)$ 对某个 $m \geq 2$ 有限。

**II. 高度下降论证**：利用高度函数（height function）从有限商 $E(K)/mE(K)$ 推出 $E(K)$ 有限生成。

#### I. 弱Mordell-Weil定理

**核心工具**：Galois上同调与Kummer序列。

对 $m \geq 2$，Kummer正合列：
$$0 \to E[m] \to E(\bar{K}) \xrightarrow{m} E(\bar{K}) \to 0$$

取Galois上同调，得长正合列：
$$0 \to E(K)/mE(K) \xrightarrow{\delta} H^1(G_K, E[m]) \to H^1(G_K, E(\bar{K}))[m] \to 0$$

其中 $G_K = \text{Gal}(\bar{K}/K)$。

**关键步骤**：证明 $\text{Im}(\delta)$ 有限。

$E[m] \cong (\mathbb{Z}/m\mathbb{Z})^2$（作为Abel群），故 $H^1(G_K, E[m])$ 是大的。但 $\text{Im}(\delta)$ 中的上同调类满足好的局部条件——它们在几乎所有位上非分歧。

**具体论证**：

1. $E$ 在几乎所有素理想 $\mathfrak{p}$ 上有好约化
2. 在好约化位上，$E(K_\mathfrak{p})/mE(K_\mathfrak{p})$ 有限（由形式群或Hensel引理）
3. 由Chevalley-Weil定理，$\delta$ 的像落在**Selmer群** $S^{(m)}(E/K)$ 中
4. Selmer群由局部条件定义，且通过Hilbert 90和有限性定理证明其有限

$$0 \to E(K)/mE(K) \to S^{(m)}(E/K) \to Ш(E/K)[m] \to 0$$

其中 $Ш(E/K)$ 是**Tate-Shafarevich群**（猜想有限但未证）。

#### II. 高度函数与下降

**Weil高度**：对 $P = [x_0 : \cdots : x_n] \in \mathbb{P}^n(K)$，
$$H_K(P) = \prod_{v} \max_i |x_i|_v$$

**对数高度**：$h(P) = \log H_K(P)$。

**Néron-Tate高度**（典范高度）：
$$\hat{h}: E(K) \to \mathbb{R}_{\geq 0}$$

满足：

- $\hat{h}(mP) = m^2 \hat{h}(P)$
- $\hat{h}(P) = 0 \Leftrightarrow P \in E(K)_{\text{tors}}$
- $\hat{h}$ 是 $E(K) \otimes \mathbb{R}$ 上的正定二次型

**下降论证**：

1. 取有限代表元集 $\{Q_1, \ldots, Q_n\}$ 代表 $E(K)/mE(K)$
2. 令 $C = \max_i \hat{h}(Q_i)$
3. 若 $P \in E(K)$，写 $P = mP_1 + Q_{i_1}$
4. 由高度性质：$\hat{h}(P_1) \leq \frac{\hat{h}(P) + C}{m^2}$
5. 迭代得到序列 $P = mP_1 + Q_{i_1} = m(mP_2 + Q_{i_2}) + Q_{i_1} = \cdots$
6. 当 $\hat{h}(P_k) \leq \frac{2C}{m^2 - 1}$ 时停止
7. 这样的 $P_k$ 有限（高度有界集有限）
8. 故 $E(K)$ 由有限生成集 $\{Q_i\} \cup \{P : \hat{h}(P) \leq \frac{2C}{m^2-1}\}$ 生成

$\square$

### (b) Tate模与Galois表示

#### $\ell$-进Tate模

设 $E$ 是椭圆曲线，$\ell$ 是素数。对 $n \geq 1$，$\ell^n$-挠子群：
$$E[\ell^n] = \{P \in E(\bar{K}) : \ell^n P = O\} \cong (\mathbb{Z}/\ell^n\mathbb{Z})^2$$

**Tate模**：
$$T_\ell(E) = \varprojlim_n E[\ell^n]$$

逆向极限通过乘法映射 $E[\ell^{n+1}] \xrightarrow{\times \ell} E[\ell^n]$。

**性质**：

- $T_\ell(E)$ 是秩2自由 $\mathbb{Z}_\ell$-模
- $T_\ell(E) \cong \mathbb{Z}_\ell \oplus \mathbb{Z}_\ell$
- $V_\ell(E) = T_\ell(E) \otimes_{\mathbb{Z}_\ell} \mathbb{Q}_\ell$ 是2维 $\mathbb{Q}_\ell$-向量空间

#### Galois表示

Galois群 $G_K = \text{Gal}(\bar{K}/K)$ 作用在 $E[\ell^n]$ 上（因为 $E[\ell^n]$ 是Galois不变的）。

这诱导**$\ell$-进Galois表示**：
$$\rho_{E,\ell}: G_K \to \text{Aut}(T_\ell(E)) \cong GL_2(\mathbb{Z}_\ell)$$

**关键性质**：

- $\rho_{E,\ell}$ 在几乎所有位（好约化位）非分歧
- 对不整除 $\ell$ 的好约化位 $\mathfrak{p}$，Frobenius共轭类 $\text{Frob}_\mathfrak{p}$ 满足
  $$\det \rho_{E,\ell}(\text{Frob}_\mathfrak{p}) = N\mathfrak{p}, \quad \text{tr } \rho_{E,\ell}(\text{Frob}_\mathfrak{p}) = a_\mathfrak{p} = N\mathfrak{p} + 1 - \#E(\mathbb{F}_\mathfrak{p})$$
- 由Weil猜想，$|a_\mathfrak{p}| \leq 2\sqrt{N\mathfrak{p}}$

#### Serre开映射定理

**定理**（Serre 1972）：若 $E$ 无复乘（CM），则 $\rho_{E,\ell}(G_K)$ 在 $GL_2(\mathbb{Z}_\ell)$ 中开（对几乎所有 $\ell$）。

**推论**：非CM椭圆曲线的Galois像是大的——$\rho_{E,\ell}$ 的像"几乎"是整个 $GL_2(\mathbb{Z}_\ell)$。

对CM椭圆曲线，像的可交换（Abelian），大小由类域论控制。

### (c) BSD猜想

#### 陈述

设 $E$ 是定义在 $\mathbb{Q}$ 上的椭圆曲线。

**Hasse-Weil $L$-函数**：
$$L(E,s) = \prod_{p \text{ good}} \frac{1}{1 - a_p p^{-s} + p^{1-2s}} \prod_{p \mid N_E} \frac{1}{L_p(p^{-s})}$$

其中 $N_E$ 是导子（conductor）。

**BSD猜想**（Birch-Swinnerton-Dyer）：

1. **解析秩 = 代数秩**：$\text{ord}_{s=1} L(E,s) = \text{rank}(E(\mathbb{Q}))$
2. **精细BSD公式**：
$$\lim_{s \to 1} \frac{L(E,s)}{(s-1)^r} = \frac{\Omega_E \cdot \text{Reg}(E) \cdot \#Ш(E/\mathbb{Q}) \cdot \prod_p c_p}{\#E(\mathbb{Q})_{\text{tors}}^2}$$

其中：

- $\Omega_E = \int_{E(\mathbb{R})} |\omega_E|$（实周期）
- $\text{Reg}(E) = \det(\langle P_i, P_j \rangle)$（Néron-Tate配对的Regulator）
- $c_p = [E(\mathbb{Q}_p) : E_0(\mathbb{Q}_p)]$（Tamagawa数）
- $Ш(E/\mathbb{Q})$ = Tate-Shafarevich群（猜想有限）

#### 已知结果

**Coates-Wiles定理**（1977）：若 $E$ 有CM且 $L(E,1) \neq 0$，则 $E(\mathbb{Q})$ 有限。

**Gross-Zagier定理**（1986）：对模椭圆曲线，$L'(E,1)$ 与Heegner点的高度相关：
$$L'(E,1) = C \cdot \hat{h}(P_K)$$

其中 $P_K$ 是Heegner点。若 $L'(E,1) \neq 0$，则 $\text{rank} \geq 1$。

**Kolyvagin定理**（1988）：若 $L(E,1) \neq 0$，则 $E(\mathbb{Q})$ 有限且 $Ш$ 有限；若 $L'(E,1) \neq 0$ 且Heegner点非挠，则 $\text{rank} = 1$。

**Wiles等**（模性定理）：所有定义在 $\mathbb{Q}$ 上的椭圆曲线都是模的（$E$ 对应权2模形式）。

**Skinner-Urban**（2014）等：对一大类椭圆曲线，BSD秩部分成立（$\text{ord}_{s=1} L(E,s) \leq 1$ 推出等式）。

#### 数值验证

对 $\text{rank} \leq 1$ 的所有椭圆曲线，BSD的秩部分已验证。

Millennium Prize问题之一：证明完整的BSD猜想（包括 $Ш$ 的有限性和精细公式）。

#### 应用与意义

- **密码学**：椭圆曲线离散对数问题的难度与 $E(\mathbb{F}_p)$ 的群结构相关
- **同余数问题**：$n$ 是同余数当且仅当 $E_n: y^2 = x^3 - n^2 x$ 有正秩
- **ABC猜想**：Szpiro猜想（椭圆曲线的判别式与导子的关系）与ABC猜想等价
