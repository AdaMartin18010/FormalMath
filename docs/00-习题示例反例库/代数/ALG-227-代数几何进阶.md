# 代数几何进阶专题

## 题目

**(a)** 证明光滑态射的开性和形式光滑的等价性。

**(b)** 讨论 étale 上同调和 l-adic 上同调。

**(c)** 证明 Deligne 的 Weil 猜想（概览）。

## 解答

### (a) 光滑态射

形式光滑：提升性质。光滑：局部标准形 Spec B[x₁,...,x_n]/(f₁,...,f_c)。

Grothendieck：形式光滑 + 有限展示 = 光滑。

### (b) étale 上同调

Grothendieck 为 Weil 猜想构造。

l-adic：H^i(X, ℚ_l) = (lim← H^i(X, ℤ/l^n)) ⊗ ℚ_l。

### (c) Weil 猜想

Deligne 证明（1974）：
- 有限域上光滑射影簇的 Zeta 函数有理性、函数方程
- Riemann 假设类比：|α_i| = q^{i/2}

方法：l-adic 上同调 +  hard Lefschetz +  Fourier 变换。
