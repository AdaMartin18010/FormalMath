## 题目

**(a)** 求 $f(x) = x^3 - 2$ 在 $\mathbb{Q}$ 上的分裂域，并计算其次数。

**(b)** 证明：域扩张 $K/F$ 是正规的当且仅当 $K$ 是 $F$ 上一族多项式的分裂域。

**(c)** 证明 $\mathbb{Q}(\sqrt[3]{2})/\mathbb{Q}$ 不是正规扩张。

**(d)** 设 $K/F$ 是有限扩张，证明存在有限扩张 $L/K$ 使得 $L/F$ 正规。

## 详细解答

### (a) $x^3 - 2$ 的分裂域

**解：**

$f(x) = x^3 - 2$ 的根为：
$$\sqrt[3]{2}, \quad \omega\sqrt[3]{2}, \quad \omega^2\sqrt[3]{2}$$

其中 $\omega = e^{2\pi i/3} = \frac{-1 + i\sqrt{3}}{2}$ 是本原三次单位根。

分裂域必须包含所有根，故包含：
- $\sqrt[3]{2}$
- $\omega = \frac{\omega\sqrt[3]{2}}{\sqrt[3]{2}}$

分裂域 $K = \mathbb{Q}(\sqrt[3]{2}, \omega)$。

**计算次数：**

$$[K : \mathbb{Q}] = [\mathbb{Q}(\sqrt[3]{2}, \omega) : \mathbb{Q}(\sqrt[3]{2})] \cdot [\mathbb{Q}(\sqrt[3]{2}) : \mathbb{Q}]$$

已知 $[\mathbb{Q}(\sqrt[3]{2}) : \mathbb{Q}] = 3$。

$\omega$ 的极小多项式是 $\Phi_3(x) = x^2 + x + 1$，在 $\mathbb{Q}(\sqrt[3]{2})$ 上仍不可约（因 $\mathbb{Q}(\sqrt[3]{2}) \subset \mathbb{R}$，而 $\omega \notin \mathbb{R}$）。

故 $[K : \mathbb{Q}(\sqrt[3]{2})] = 2$。

$$[K : \mathbb{Q}] = 2 \times 3 = 6$$

### (b) 正规扩张的等价刻画

**证明：**

**($\Rightarrow$)** 设 $K/F$ 正规，$[K:F] = n$。

设 $\alpha_1, ..., \alpha_n$ 是 $K$ 作为 $F$-向量空间的基。

设 $p_i(x)$ 是 $\alpha_i$ 在 $F$ 上的极小多项式。

因 $K/F$ 正规，每个 $p_i$ 在 $K$ 中分裂（有一个根在 $K$ 中则全在）。

$K = F(\alpha_1, ..., \alpha_n)$ 是 $\{p_1, ..., p_n\}$ 的分裂域。

**($\Leftarrow$)** 设 $K$ 是 $\{f_i\}_{i \in I}$ 的分裂域。

设 $p(x) \in F[x]$ 不可约，有根 $\alpha \in K$。

设 $\beta$ 是 $p$ 在代数闭包中的另一根，需证 $\beta \in K$。

因 $K$ 由 $\{f_i\}$ 的根生成，$F$-同构 $\sigma: F(\alpha) \to F(\beta)$，$\alpha \mapsto \beta$ 可延拓到 $K$。

但 $K$ 是 $\{f_i\}$ 的分裂域，$\sigma(K) = K$。

故 $\beta = \sigma(\alpha) \in K$。

**证毕。**

### (c) 非正规扩张的例子

**证明：**

$K = \mathbb{Q}(\sqrt[3]{2})$，$[K : \mathbb{Q}] = 3$。

$x^3 - 2$ 在 $K$ 中有根 $\sqrt[3]{2}$，但其他两个根 $\omega\sqrt[3]{2}, \omega^2\sqrt[3]{2}$ 不在 $K$ 中（它们不是实数，而 $K \subset \mathbb{R}$）。

因此 $x^3 - 2$ 在 $K$ 上不分裂，$K/\mathbb{Q}$ 不是正规扩张。

**证毕。**

### (d) 正规闭包的存在性

**证明：**

设 $K/F$ 有限，$\alpha_1, ..., \alpha_n$ 是 $K$ 在 $F$ 上的生成元。

设 $p_i(x)$ 是 $\alpha_i$ 的极小多项式，$f(x) = p_1(x) \cdots p_n(x)$。

设 $L$ 是 $f(x)$ 在 $K$ 上的分裂域（也是 $F$ 上的分裂域）。

则 $L/F$ 是分裂域扩张，故正规。

且 $L/K$ 有限（由有限个代数元生成）。

**证毕。**

## 证明技巧

1. **分裂域的构造：** 包含多项式所有根的最小域扩张
2. **正规扩张的验证：** 不可约多项式有一个根则有所有根
3. **反证法证非正规：** 找一个不可约多项式，它有一部分根在域中，一部分不在

## 常见错误

| 错误 | 说明 |
|------|------|
| 认为所有有限扩张都正规 | 需要验证所有不可约多项式的行为 |
| 混淆分裂域与代数闭包 | 分裂域针对特定多项式，代数闭包包含所有多项式的根 |
| 计算次数时忘记验证不可约性 | 如$\omega$在$\mathbb{Q}(\sqrt[3]{2})$上仍不可约 |
