# 交换代数综合挑战

## 题目

**(a)** 设 $(R, \mathfrak{m})$ 是完备Noether局部环，证明Cohen结构定理：$R$ 是正则局部环被理想商，或形式幂级数环模。

**(b)** 证明Auslander-Buchsbaum公式：对有限生成 $R$-模 $M$，若 $\text{pd}(M) < \infty$，则
$$\text{pd}(M) + \text{depth}(M) = \text{depth}(R)$$

**(c)** 证明Serre对正则性的判据：$R$ 是正则局部环当且仅当 $\text{gldim}(R) < \infty$。

## 解答

### (a) Cohen结构定理

**证明概要：**

设 $k = R/\mathfrak{m}$，$\Lambda$ 是系数环（等特征或不等特征）。

**等特征**：$R \cong k[[x_1, \ldots, x_n]]/I$。

选取 $\mathfrak{m}$ 的极小生成组，映射形式幂级数环。

**不等特征**：用Cohen环（特征0的完备DV提升）。

### (b) Auslander-Buchsbaum公式

**证明：**

对 $\text{pd}(M) = 0$（投射），$\text{depth}(M) = \text{depth}(R)$（忠实平坦）。

对短正合列 $0 \to K \to P \to M \to 0$，$P$ 投射。

$\text{pd}(K) = \text{pd}(M) - 1$，深度关系由正则序列保持。

归纳得公式。

### (c) Serre判据

**证明概要：**

**($\Rightarrow$)**：正则局部环 $\text{gldim}(R) = \dim(R) < \infty$。

**($\Leftarrow$)**：设 $\text{gldim}(R) = n < \infty$。

证 $\dim(R/\mathfrak{m}^2) = n + 1$，由AB公式 $\text{pd}(k) = n$，$\text{depth}(R) = n$。

故 $\dim(R) = n$，正则。
