# 代数综合专题 II

## 题目

**(a)** 证明 Langlands 纲领的函子性原理（陈述）。

**(b)** 讨论朗兰兹对实约化群的分类。

**(c)** 证明 Shimura 簇和自守形式的联系。

## 解答

### (a) Langlands 函子性

G, H 约化群，^LG, ^LH L-群。

^L-同态 φ: ^LH → ^LG 诱导自守形式的转移。

### (b) 实约化群

Knapp-Zuckerman 分类：Langlands 商，Vogan 图。

Harish-Chandra 模，(g, K)-模。

### (c) Shimura 簇

(G, X) Shimura 数据，Sh_K = G(ℚ)\G(𝔸_f)×X/K。

自守形式在 étale 上同调中实现，与 Galois 表示对应。
