# 交换代数与代数几何的联系

## 题目

本题探讨交换代数与代数几何的深刻联系。

**(a) Hilbert零点定理（弱形式）**：

设 $k$ 是代数闭域，$I \subset k[x_1, \ldots, x_n]$ 是真理想。

证明 $V(I) \neq \emptyset$（公共零点存在）。

**(b) 仿射簇的坐标环**：

设 $X \subset \mathbb{A}^n_k$ 是仿射簇，证明 $k[X] = k[x_1, \ldots, x_n]/I(X)$ 满足：
$$\text{Spec}(k[X]) \cong X \text{ (拓扑空间)}$$

**(c) 非奇异点的Jacobian判别法**：

设 $X = V(f_1, \ldots, f_r) \subset \mathbb{A}^n$，$p \in X$。

证明 $p$ 是非奇异点当且仅当 $\text{rank}\left(\frac{\partial f_i}{\partial x_j}(p)\right) = n - \dim(X)$。

## 解答

### (a) 弱零点定理

**证明：**

设 $I$ 是真理想，$I \subset \mathfrak{m}$（极大理想）。

$R = k[x_1, \ldots, x_n]/\mathfrak{m}$ 是域，有限生成 $k$-代数。

由Zariski引理：有限生成 $k$-代数是域 $\Rightarrow$ 是 $k$ 的有限扩张。

$k$ 代数闭，故 $R = k$。

因此 $x_i \equiv a_i \pmod{\mathfrak{m}}$，$a_i \in k$。

$\mathfrak{m} = (x_1 - a_1, \ldots, x_n - a_n)$。

$(a_1, \ldots, a_n) \in V(I)$。$\square$

### (b) 坐标环的谱

**证明：**

$k[X]$ 的素理想 $\leftrightarrow$ $k[x_1, \ldots, x_n]$ 中包含 $I(X)$ 的素理想。

$\leftrightarrow$ $X$ 的不可约闭子集（素理想的零点集）。

映射：$\mathfrak{p} \mapsto V(\mathfrak{p}) \cap X$。

逆：$Y \mapsto I(Y)/I(X)$。

Zariski拓扑对应：$V(J) \leftrightarrow V(J/I(X))$。

因此 $\text{Spec}(k[X]) \cong X$（作为拓扑空间）。$\square$

### (c) Jacobian判别法

**证明：**

**切空间**：$T_p X = \{v \in k^n : df_i(p)(v) = 0\}$。

$$= \ker\left(\frac{\partial f_i}{\partial x_j}(p)\right)$$

**非奇异性**：$\dim(T_p X) = \dim(X)$。

由切空间的定义：
$$\dim(T_p X) = n - \text{rank}\left(\frac{\partial f_i}{\partial x_j}(p)\right)$$

$\dim(T_p X) = \dim(X)$ 当且仅当：
$$\text{rank}\left(\frac{\partial f_i}{\partial x_j}(p)\right) = n - \dim(X)$$
$\square$

## 证明技巧

1. **Zariski引理**：代数闭域上多项式环的极大理想分类
2. **理想-簇对应**：Nullstellensatz的核心
3. **切空间的代数定义**：导数作为线性泛函

## 常见错误

- ❌ 忘记代数闭域的必要性
- ❌ 混淆$\text{Spec}$与极大谱
- ❌ Jacobian秩与余维数的关系

**变式1：** 证明射影Nullstellensatz。
