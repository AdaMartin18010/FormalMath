# 对称群表示

## 题目

**(a)** 证明 S_n 的不可约表示由分拆 λ ⊢ n 分类。

**(b)** 证明 Young 对称化子和 Specht 模的不可约性。

**(c)** 讨论 Murnaghan-Nakayama 规则。

## 解答

### (a) 分拆分类

λ ⊢ n：λ₁ ≥ λ₂ ≥ ... ≥ 0，Σλ_i = n。

共轭类 ↔ 轮换型 ↔ 分拆。不可约表示 ↔ 分拆。

### (b) Specht 模

Young 表 t，行对称化 a_t，列反对称化 b_t。

c_t = b_t a_t 是 Young 对称化子，S^λ = ℂ[S_n]c_t 不可约。

### (c) Murnaghan-Nakayama

特征标递归公式：
$$\chi^\lambda(\pi) = \sum_{\text{rim hook } \xi} (-1)^{ht(\xi)} \chi^{\lambda \setminus \xi}(\pi')$$
