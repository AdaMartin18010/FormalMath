# CEX-ALG-003: 非唯一分解整环（经典例子）

## 反例构造

考虑二次整数环 $R = \mathbb{Z}[\sqrt{-5}] = \{a + b\sqrt{-5} : a, b \in \mathbb{Z}\}$，配备通常的加法和乘法。

**在 $R$ 中考察元素6：**

$$6 = 2 \cdot 3 = (1 + \sqrt{-5})(1 - \sqrt{-5})$$

## 反例分析

**验证这是两种不同的不可约分解：**

**步骤1：验证范数**

定义范数 $N: R \to \mathbb{Z}_{\geq 0}$ 为：
$$N(a + b\sqrt{-5}) = a^2 + 5b^2$$

范数性质：$N(\alpha\beta) = N(\alpha)N(\beta)$

计算各元素的范数：

- $N(2) = 4$
- $N(3) = 9$
- $N(1 + \sqrt{-5}) = 1 + 5 = 6$
- $N(1 - \sqrt{-5}) = 1 + 5 = 6$

**步骤2：验证不可约性**

**$2$ 是不可约的：**

若 $2 = \alpha\beta$，则 $N(2) = 4 = N(\alpha)N(\beta)$。

可能情况：

- $N(\alpha) = 1$：则 $\alpha = \pm 1$ 是单位
- $N(\alpha) = 4, N(\beta) = 1$：同理
- $N(\alpha) = 2$：需要 $a^2 + 5b^2 = 2$，无整数解

故2不可约。类似可证3、$1 \pm \sqrt{-5}$ 都不可约。

**步骤3：验证两种分解不等价**

假设两种分解等价，则存在单位 $u$ 使得：
$$2 = u(1 + \sqrt{-5})$$

但 $N(2) = 4$ 而 $N(1 + \sqrt{-5}) = 6$，范数不匹配，矛盾！

$R$ 中的单位只有 $\pm 1$，都无法使等式成立。

## 直观解释

**几何视角：** $\mathbb{Z}[\sqrt{-5}]$ 对应复平面上形如 $a + bi\sqrt{5}$ 的点格。范数 $N(z) = |z|^2$ 是到原点的距离平方。

**因式分解失效的原因：** 在这个数格中，"素数"的概念变得模糊。元素6可以看作由两组不同的"不可约向量"组合而成——就像同一个面积可以用不同的长方形组合表示。

**与 $\mathbb{Z}[i]$ 对比：** 高斯整数环 $\mathbb{Z}[i] = \{a + bi : a, b \in \mathbb{Z}\}$ 是唯一分解整环。区别在于 $-5 \equiv 3 \pmod{4}$ 而 $-1 \equiv 3 \pmod{4}$ 不满足特定条件。

**类数解释：** $\mathbb{Z}[\sqrt{-5}]$ 的理想类数为2，意味着"理想分解"是唯一的，但"元素分解"不唯一。这引出了代数数论中理想理论的发展。

## 相关定理

**唯一分解整环（UFD）定义**
> 整环 $R$ 是UFD，如果每个非零非单位元都可唯一地（在相伴意义下）分解为不可约元的乘积。

**高斯整数环是UFD**
> $\mathbb{Z}[i]$ 是唯一分解整环。

**二次整数环的UFD判定（不完全）**
> 对于 $\mathbb{Z}[\sqrt{d}]$（$d < 0$ 无平方因子），仅当 $d = -1, -2, -3, -7, -11, -19, -43, -67, -163$ 时可能是UFD。

说明：$d = -5$ 不在此列，因此 $\mathbb{Z}[\sqrt{-5}]$ 不是UFD。

**戴德金定理**
> 代数整数环中，任何理想都可唯一分解为素理想的乘积。

说明：这是UFD失效后的补救措施，理想理论由此诞生。

## 避免此反例的条件

| 条件 | 结论 | 例子 |
|------|------|------|
| $R$ 是主理想整环（PID） | PID $\Rightarrow$ UFD | $\mathbb{Z}, \mathbb{Z}[i], F[x]$（$F$ 域） |
| $R$ 是欧几里得整环 | 欧几里得 $\Rightarrow$ PID $\Rightarrow$ UFD | $\mathbb{Z}, \mathbb{Z}[i], \mathbb{Z}[\omega]$ |
| $d = -1, -2, -3, -7, -11$ | $\mathbb{Z}[\sqrt{d}]$ 是UFD（虚二次情形） | $\mathbb{Z}[i]$（$d=-1$）|
| $R = \mathbb{Z}$ | 整数环是UFD | 算术基本定理 |
| $d > 0$ 且类数为1 | 实二次整数环可能是UFD | $\mathbb{Z}[\sqrt{2}]$ |

**重要提醒：**
$$\text{欧几里得整环} \subsetneq \text{PID} \subsetneq \text{UFD} \subsetneq \text{整环}$$

所有包含关系都是真包含。本例中 $\mathbb{Z}[\sqrt{-5}]$ 是整环但不是UFD。

**识别UFD的方法：**

1. 验证是否PID（检查所有理想是否主理想）
2. 使用范数证明不可约元都是素元
3. 验证因子分解的存在性和唯一性
