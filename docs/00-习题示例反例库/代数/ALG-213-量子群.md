# 量子群

## 题目

**(a)** 定义量子群 U_q(g) 并证明其 Hopf 代数结构。

**(b)** 讨论量子群的表示理论（半经典极限）。

**(c)** 证明 R-矩阵和 Yang-Baxter 方程。

## 解答

### (a) 量子群定义

U_q(sl₂)：生成元 E, F, K，关系
$$KE = q^2 EK, \quad KF = q^{-2}FK, \quad [E,F] = \frac{K-K^{-1}}{q-q^{-1}}$$

Hopf 代数：余乘、余单位、对极。

### (b) 表示理论

q 非单位根：类似经典，最高权理论成立。

q → 1 极限恢复经典表示。

### (c) R-矩阵

R ∈ U_q(g)^{⊗2}，满足 YBE：R_{12}R_{13}R_{23} = R_{23}R_{13}R_{12}。

量子杨-Baxter 方程的解。
