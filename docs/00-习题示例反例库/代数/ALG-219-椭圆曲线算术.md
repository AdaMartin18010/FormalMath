# 椭圆曲线的算术

## 题目

**(a)** 证明 Mordell-Weil 定理：椭圆曲线有理点群 E(Q) 有限生成。

**(b)** 定义高度函数并证明其在 E(Q) 上的性质。

**(c)** 讨论弱 Mordell-Weil 定理的证明（下降法）。

## 解答

### (a) Mordell-Weil 定理

E(Q) ≅ ℤ^r ⊕ E(Q)_{tors}，秩 r 和挠子群。

证明：高度下降 + 弱 Mordell-Weil + 挠子群有限。

### (b) 高度函数

Weil 高度：h(P) = (1/2)log H(x(P))。

性质：h(2P) = 4h(P) + O(1)，h(P+Q) + h(P-Q) = 2h(P) + 2h(Q) + O(1)。

### (c) 弱 Mordell-Weil

E(Q)/2E(Q) 有限。利用齐次空间、局部-整体原理。
