# 伴随函子理论

## 题目

**定义**：函子 $L: \mathcal{C} \to \mathcal{D}$ 和 $R: \mathcal{D} \to \mathcal{C}$ 是**伴随**的（记 $L \dashv R$），若存在自然同构：
$$\text{Hom}_{\mathcal{D}}(L(X), Y) \cong \text{Hom}_{\mathcal{C}}(X, R(Y))$$

(a) 证明自由函子 $F: \mathbf{Set} \to \mathbf{Grp}$（自由群）是遗忘函子 $U: \mathbf{Grp} \to \mathbf{Set}$ 的左伴随。

(b) 证明伴随对给出单位 $\eta: \text{id}_{\mathcal{C}} \Rightarrow RL$ 和余单位 $\epsilon: LR \Rightarrow \text{id}_{\mathcal{D}}$，满足三角恒等式。

(c) 证明左伴随保持Colimit，右伴随保持Limit。

## 解答

### (a) 自由-遗忘伴随

**证明：**

需证自然双射：
$$\text{Hom}_{\mathbf{Grp}}(F(S), G) \cong \text{Hom}_{\mathbf{Set}}(S, U(G))$$

对集合 $S$ 和群 $G$，映射 $f: S \to U(G)$ 唯一决定群同态 $\tilde{f}: F(S) \to G$：
- $F(S)$ 是 $S$ 上自由群，元素是 $S$ 上简化字
- $\tilde{f}(s_1^{\epsilon_1} \cdots s_n^{\epsilon_n}) = f(s_1)^{\epsilon_1} \cdots f(s_n)^{\epsilon_n}$

这给出双射，且自然性由构造保证。$\square$

### (b) 单位与余单位

**证明：**

**单位 $\eta$**：对 $X \in \mathcal{C}$，取 $Y = L(X)$：
$$\text{Hom}(L(X), L(X)) \cong \text{Hom}(X, RL(X))$$

$\text{id}_{L(X)}$ 对应 $\eta_X: X \to RL(X)$。

**余单位 $\epsilon$**：对 $Y \in \mathcal{D}$，取 $X = R(Y)$：
$$\text{Hom}(LR(Y), Y) \cong \text{Hom}(R(Y), R(Y))$$

$\text{id}_{R(Y)}$ 对应 $\epsilon_Y: LR(Y) \to Y$。

**三角恒等式**：

$$(\epsilon L) \circ (L \eta) = \text{id}_L: L \to LRL \to L$$

验证：对 $X$，需 $L(X) \xrightarrow{L\eta_X} LRL(X) \xrightarrow{\epsilon_{L(X)}} L(X)$ 是恒等。

由自然性和伴随定义，这等价于 $\eta$ 和 $\epsilon$ 的互逆构造。$\square$

### (c) 伴随与极限

**证明（左伴随保持Colimit）：**

设 $\{X_i\}$ 是 $\mathcal{C}$ 中的图表，$\text{colim } X_i$ 存在。

需证 $L(\text{colim } X_i) \cong \text{colim } L(X_i)$。

即证 $L(\text{colim } X_i)$ 满足 $\{L(X_i)\}$ 的余泛性质。

对任意 $Y \in \mathcal{D}$ 和余锥 $\{L(X_i) \to Y\}$：

$$\text{Hom}(L(\text{colim } X_i), Y) \cong \text{Hom}(\text{colim } X_i, R(Y))$$

$$\cong \lim \text{Hom}(X_i, R(Y)) \cong \lim \text{Hom}(L(X_i), Y)$$

这正是 $\text{colim } L(X_i)$ 的定义性质。$\square$

## 证明技巧

1. **泛性质翻译**：伴随等价于某种泛问题
2. **单位-余单位**：三角恒等式完全刻画伴随
3. **极限保持**：Hom与极限交换的自然结果

## 常见错误

- ❌ 混淆左右伴随的方向
- ❌ 忘记验证单位/余单位的自然性
- ❌ 在(c)中混淆协变和逆变的极限行为

## 变式练习

**变式1：** 证明张量积 $- \otimes M$ 是 $\text{Hom}(M, -)$ 的左伴随。

**变式2：** 证明若 $L \dashv R$ 且 $L$ 是完全忠实嵌入，则 $R$ 是局部化。
