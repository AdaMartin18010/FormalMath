# 舒尔正交关系与张量积

## 题目

设 $G$ 是有限群，$V, W$ 是复表示。

(a) 证明第二正交关系：对共轭类 $C, C'$：
$$\sum_\chi \chi(C) \overline{\chi(C')} = \frac{|G|}{|C|} \delta_{C,C'}$$
其中求和跑遍所有不可约特征标。

(b) 证明：$\chi_{V \otimes W}(g) = \chi_V(g) \cdot \chi_W(g)$。

(c) 计算对称幂和交错幂的特征标公式。

## 解答

### (a) 第二正交关系

**证明**：

特征标表是方阵，第一正交关系给出：
$$U^* U = I$$

其中 $U_{\chi, C} = \sqrt{\frac{|C|}{|G|}} \chi(C)$。

因此 $UU^* = I$，即：
$$\sum_C \frac{|C|}{|G|} \chi(C) \overline{\chi'(C)} = \delta_{\chi, \chi'}$$

转置得第二正交关系。$\square$

### (b) 张量积的特征标

**证明**：

$(\rho_V \otimes \rho_W)(g) = \rho_V(g) \otimes \rho_W(g)$。

$$\text{tr}(A \otimes B) = \text{tr}(A) \cdot \text{tr}(B)$$

因此 $\chi_{V \otimes W}(g) = \chi_V(g) \chi_W(g)$。$\square$

### (c) 对称幂与交错幂

**对称幂 $S^n V$**：

$$\chi_{S^n V}(g) = h_n(\lambda_1, \ldots, \lambda_d)$$

其中 $\lambda_i$ 是 $\rho_V(g)$ 的特征值，$h_n$ 是完全对称多项式。

**生成函数**：
$$\sum_{n \geq 0} \chi_{S^n V}(g) t^n = \frac{1}{\det(1 - t \rho_V(g))}$$

**交错幂 $\Lambda^n V$**：

$$\chi_{\Lambda^n V}(g) = e_n(\lambda_1, \ldots, \lambda_d)$$

$e_n$ 是初等对称多项式。

**生成函数**：
$$\sum_{n \geq 0} \chi_{\Lambda^n V}(g) t^n = \det(1 + t \rho_V(g))$$
$\square$

## 证明技巧

1. **特征标表的正交性**：矩阵的酉性质
2. **张量积的迹**：Kronecker积的迹公式
3. **对称多项式**：表示论的生成函数

## 常见错误

- ❌ 混淆两种正交关系的求和顺序
- ❌ 张量积特征标公式中的维度错误
- ❌ 对称/交错幂生成函数的符号

## 变式练习

**变式1：** 用特征标证明 $V \otimes V^* \cong \text{End}(V)$。

**变式2：** 计算 $S^2 V$ 和 $\Lambda^2 V$ 的分解（Clebsch-Gordan）。
