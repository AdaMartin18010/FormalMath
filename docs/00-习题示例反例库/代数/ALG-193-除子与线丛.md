# 除子与线丛

## 题目

**(a)** 定义 Weil 除子、Cartier 除子并证明光滑簇上二者等价。

**(b)** 证明除子类群 Cl(X) 与 Picard 群 Pic(X) 的关系。

**(c)** 计算 P^n 的 Picard 群。

## 解答

### (a) Weil 与 Cartier 除子

Weil：余维1子簇的形式和。Cartier：局部由有理函数定义。

X 正则 ⟹ 两者等价（Krull 维数1正则局部环是 DVR）。

### (b) 对应关系

Cartier 除子 D ↔ 线丛 O(D)。D ~ D' ⟺ O(D) ≅ O(D')。

Pic(X) = {线丛} ≅ Cartier 除子 / 线性等价。

### (c) Pic(P^n)

Cl(P^n) ≅ ℤ，生成元是超平面 H。O(k) = O(kH)，Pic(P^n) ≅ ℤ。
