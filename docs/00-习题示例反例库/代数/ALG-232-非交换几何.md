# 非交换几何

## 题目

**(a)** 定义非交换射影概形 Proj A 并讨论其几何。

**(b)** 证明 Artin-Zhang 维数理论。

**(c)** 讨论 Sklyanin 代数及其几何。

## 解答

### (a) 非交换 Proj

分次代数 A，范畴 qgr A = gr A / tors A。

Proj_{nc} A = (qgr A, O(1))。

### (b) Artin-Zhang

非交换射影概形的维数理论：
- 增长 GK 维数
- 上同维数
- 对偶性

### (c) Sklyanin 代数

S(a,b,c)：生成元 x,y,z，关系
$$ayz + bzy + cx² = 0$$
（循环置换）

对应椭圆曲线上的点，非交换 P²。
