# 交换代数进阶综合

## 题目

**(a)** 证明Nagata定理：Noether环 $R$ 的整闭包在有限生成代数下仍Noether。

**(b)** 证明Zariski主定理：双有理态射 $f: X \to Y$ 的纤维要么有限要么含正维数变体。

**(c)** 讨论excellent环的定义和在代数几何中的重要性。

## 解答

### (a) Nagata定理

日本环（Nagata环）的整闭包有限性。

### (b) Zariski主定理

双有理态射的结构定理。

### (c) Excellent环

Grothendieck定义，保证几何性质的环类。
