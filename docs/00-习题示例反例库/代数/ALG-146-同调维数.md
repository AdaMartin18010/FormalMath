# 同调维数与整体维数

## 题目

设 $R$ 是环，$M$ 是 $R$-模。

**(a)** 定义投射维数 $\text{pd}(M) = \inf\{n : \text{存在投射分解 } 0 \to P_n \to \cdots \to P_0 \to M \to 0\}$。

证明 $\text{pd}(M) \leq n$ 当且仅当 $\text{Ext}^{n+1}(M, N) = 0$ 对所有 $N$。

**(b)** 定义整体维数 $\text{gldim}(R) = \sup_M \text{pd}(M)$。证明
$$\text{gldim}(R) = \sup_M \text{id}(M) = \sup_{M,N} \inf\{n : \text{Ext}^{n+1}(M,N) = 0\}$$

**(c)** 证明：$\text{gldim}(R[x]) = \text{gldim}(R) + 1$（Hilbert合冲定理）。

## 解答

### (a) 投射维数的刻画

**证明：**

**($\Rightarrow$)**：设 $\text{pd}(M) \leq n$，取有限投射分解。

$\text{Ext}^{n+1}(M, N)$ 用此分解计算，$P_{n+1} = 0$，故为0。

**($\Leftarrow$)**：设 $\text{Ext}^{n+1}(M, -) = 0$。

取投射分解，在第 $n$ 步截断，核 $K$ 满足 $\text{Ext}^1(K, -) = 0$，故 $K$ 投射。

因此 $\text{pd}(M) \leq n$。

### (b) 整体维数的等价性

**证明：**

由内射维数的对偶刻画和对称性。

$\text{Ext}^n$ 对第一或第二变元的消失性等价。

### (c) Hilbert合冲定理

**证明概要：**

设 $\text{gldim}(R) = n$，证 $\text{gldim}(R[x]) = n+1$。

**上界**：$R[x]$-模 $M$ 有由 $R$-模结构提升的投射分解。

用变化环的谱序列，长度增1。

**下界**：$R$ 有模 $M$ 使 $\text{Ext}^n(M, N) \neq 0$。

$M$ 视为 $R[x]$-模（$x$ 零化），则 $\text{pd}_{R[x]}(M) \geq n+1$。
