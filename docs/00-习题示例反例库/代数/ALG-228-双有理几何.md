# 双有理几何

## 题目

**(a)** 证明维数 ≥ 2 时双有理等价类不唯一极小模型。

**(b)** 讨论翻转（flip）和flop的几何。

**(c)** 证明典范模型存在性（BCHM）。

## 解答

### (a) 非唯一性

直纹面：不同的直纹结构给出不同的极小模型。

维数 ≥ 2：需要 MMP 处理。

### (b) Flip 和 Flop

Flip：K_X 负的 extremal 收缩后，沿曲线 C 翻转。
$$(K_X.C) < 0 \to (K_{X^+}.C^+) > 0$$

Flop：K_X 平凡的 surgery。

### (c) BCHM 定理

Birkar-Cascini-Hacon-McKernan：对数一般型对 (X,D)，典范模型存在。

方法：有限生成性 + 下降 MMP。
