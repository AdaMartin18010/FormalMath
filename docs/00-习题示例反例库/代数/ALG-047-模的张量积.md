## 题目

设 $R$ 是含幺环，$M$ 是右 $R$-模，$N$ 是左 $R$-模，$A$ 是Abel群。

**(a)** 定义 $R$-**平衡映射** $f: M \times N \to A$，并给出张量积 $M \otimes_R N$ 的泛性质定义。

**(b)** 证明张量积的存在性（构造：自由Abel群商去平衡关系）。

**(c)** 证明：$\mathbb{Z}/m\mathbb{Z} \otimes_\mathbb{Z} \mathbb{Z}/n\mathbb{Z} \cong \mathbb{Z}/\gcd(m,n)\mathbb{Z}$。

**(d)** 设 $R$ 是交换环，$M, N$ 是 $R$-模。证明 $M \otimes_R N$ 有自然的 $R$-模结构。

## 详细解答

### (a) 张量积的定义

**定义（平衡映射）：** 映射 $f: M \times N \to A$ 是 $R$-平衡的，如果：
1. $f(m_1 + m_2, n) = f(m_1, n) + f(m_2, n)$
2. $f(m, n_1 + n_2) = f(m, n_1) + f(m, n_2)$
3. $f(mr, n) = f(m, rn)$（对所有 $r \in R$）

**定义（张量积）：** 张量积 $(M \otimes_R N, \otimes)$ 是Abel群配备平衡映射 $\otimes: M \times N \to M \otimes_R N$，满足泛性质：

对任何平衡映射 $f: M \times N \to A$，存在唯一的同态 $\tilde{f}: M \otimes_R N \to A$ 使 $f = \tilde{f} \circ \otimes$。

### (b) 张量积的存在性

**构造：**

**步骤1：** 令 $F$ 是以 $M \times N$ 为基的自由Abel群。

**步骤2：** 令 $K$ 是由以下元素生成的子群：
- $(m_1 + m_2, n) - (m_1, n) - (m_2, n)$
- $(m, n_1 + n_2) - (m, n_1) - (m, n_2)$
- $(mr, n) - (m, rn)$

**步骤3：** 定义 $M \otimes_R N = F/K$。

记 $m \otimes n = (m,n) + K$。

**步骤4：** 验证泛性质

定义 $\otimes: M \times N \to F/K$，$\otimes(m,n) = m \otimes n$。

平衡性由 $K$ 的定义保证。

对平衡映射 $f$，定义 $\tilde{f}(m \otimes n) = f(m,n)$，线性延拓。

良定性和唯一性由构造保证。

**证毕。**

### (c) 具体计算

**证明：**

设 $M = \mathbb{Z}/m\mathbb{Z}$，$N = \mathbb{Z}/n\mathbb{Z}$。

**生成元：** $M \otimes N$ 由 $\{\bar{a} \otimes \bar{b}\}$ 生成。

**关系：**
- $m \cdot (\bar{1} \otimes \bar{b}) = \bar{m} \otimes \bar{b} = \bar{0} \otimes \bar{b} = 0$
- $n \cdot (\bar{a} \otimes \bar{1}) = \bar{a} \otimes \bar{n} = \bar{a} \otimes \bar{0} = 0$

故 $M \otimes N$ 由 $\bar{1} \otimes \bar{1}$ 生成，阶整除 $\gcd(m,n)$。

**构造同态：**

定义 $f: M \times N \to \mathbb{Z}/\gcd(m,n)\mathbb{Z}$，$f(\bar{a}, \bar{b}) = ab \mod \gcd(m,n)$。

验证平衡性：
$$f(\bar{a} + m\mathbb{Z}, \bar{b}) = (a+m) b = ab + mb \equiv ab \pmod{\gcd(m,n)}$$

诱导 $\tilde{f}: M \otimes N \to \mathbb{Z}/\gcd(m,n)\mathbb{Z}$，$\tilde{f}(\bar{1} \otimes \bar{1}) = 1$。

**满射：** $\tilde{f}$ 满射。

**单射：** $|M \otimes N| \leq \gcd(m,n)$，且 $\tilde{f}$ 满射，故 $|M \otimes N| = \gcd(m,n)$，$\tilde{f}$ 是同构。

**证毕。**

### (d) 交换环上的模结构

**证明：**

设 $R$ 交换，$M, N$ 是 $R$-模（既是左也是右模）。

定义 $r \cdot (m \otimes n) = (rm) \otimes n = m \otimes (rn)$。

**良定性：**

$(rm) \otimes n = (mr) \otimes n = m \otimes (rn)$（$R$ 交换）。

**模公理：** 由张量积的定义验证。

**证毕。**

## 证明技巧

1. **泛性质的应用：** 定义映射到/从张量积
2. **具体计算：** 找生成元和关系
3. **交换环的简化：** 左右模结构一致

## 常见错误

| 错误 | 说明 |
|------|------|
| 认为$m \otimes n = 0$仅当$m=0$或$n=0$ | 可以有非平凡关系 |
| 忘记张量积是Abel群 | 一般不是$R$-模（除非$R$交换） |
| 混淆$\otimes_R$与$\otimes_\mathbb{Z}$ | 标量环不同 |

## 变式练习

**变式1：** 证明$\mathbb{Q} \otimes_\mathbb{Z} \mathbb{Q} \cong \mathbb{Q}$。

**变式2：** 设$R$是交换环，$S \subset R$是乘性子集。证明$S^{-1}R \otimes_R M \cong S^{-1}M$。

**变式3：** 证明张量积的右正合性：$M \otimes -$保持右正合列。
