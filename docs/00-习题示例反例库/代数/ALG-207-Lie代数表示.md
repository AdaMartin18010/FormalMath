# Lie 代数表示

## 题目

**(a)** 证明 Engel 定理和 Lie 定理。

**(b)** 定义 Cartan 子代数并证明其共轭性。

**(c)** 计算 sl(2, ℂ) 的有限维不可约表示。

## 解答

### (a) Engel 与 Lie 定理

Engel：L 幂零 ⟺ ad_x 幂零 ∀x。

Lie：可解 Lie 代数在代数闭域上的表示可三角化。

### (b) Cartan 子代数

自正规幂次子代数。任意两个 Cartan 子代数通过内自同构共轭。

### (c) sl(2, ℂ) 表示

V_m：m+1 维，基 {v_0,...,v_m}，h·v_k = (m-2k)v_k，e·v_k = (m-k+1)v_{k-1}，f·v_k = (k+1)v_{k+1}。
