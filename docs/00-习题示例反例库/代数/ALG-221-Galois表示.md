# Galois 表示

## 题目

**(a)** 定义 ℓ-adic Galois 表示并证明其基本性质。

**(b)** 证明 Tate 模上的 Galois 作用。

**(c)** 讨论 Fontaine-Laffaille 理论和 p-进 Hodge 理论。

## 解答

### (a) ℓ-adic 表示

G_K = Gal(K̄/K) 在有限维 ℚ_ℓ-向量空间上的连续表示。

例子：Tate 模、étale 上同调。

### (b) Tate 模

对椭圆曲线 E：T_ℓ(E) = lim← E[ℓ^n] ≅ ℤ_ℓ²。

Galois 作用：ρ_E: G_K → GL₂(ℤ_ℓ)。

### (c) p-进 Hodge 理论

de Rham 表示、半稳定表示、晶体表示。

Fontaine 的环 B_{dR}, B_{st}, B_{crys} 比较定理。
