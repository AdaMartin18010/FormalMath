# 曲面的分类

## 题目

**(a)** 证明 Castelnuovo 的有理性判别：P² 的有理性等价于 P¹ × P¹。

**(b)** 定义曲面的 Kodaira 维数并给出 Enriques-Kodaira 分类。

**(c)** 讨论极小模型的存在唯一性。

## 解答

### (a) Castelnuovo 判别

曲面的有理性 ⟺ 无奇点且 q = P₂ = 0。

### (b) Kodaira 维数

κ(X) = trdeg ⊕_{n≥0} H⁰(X, K_X^{⊗n}) - 1，取值 -∞, 0, 1, 2。

分类：有理/直纹 (κ=-∞)、K3/Enriques/阿贝尔/双曲椭圆 (κ=0)、椭圆曲面 (κ=1)、一般型 (κ=2)。

### (c) 极小模型

收缩 (-1)-曲线得到极小模型。κ ≥ 0 时唯一，κ = -∞ 时有多个（直纹面）。
