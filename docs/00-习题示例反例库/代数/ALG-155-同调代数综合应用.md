# 同调代数与代数几何

## 题目

**(a)** 证明Beilinson定理：$D^b(\mathbb{P}^n)$ 由 $\mathcal{O}, \mathcal{O}(-1), \ldots, \mathcal{O}(-n)$ 生成。

**(b)** 讨论Fourier-Mukai变换。

**(c)** 陈述Bondal-Orlov定理。

## 解答

Beilinson用monad分解任何复。

Fourier-Mukai变换：$\Phi_{\mathcal{E}}(F) = Rp_{2*}(\mathcal{E} \otimes^L p_1^*F)$。
