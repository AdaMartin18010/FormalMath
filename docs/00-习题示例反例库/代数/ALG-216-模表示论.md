# 模表示论

## 题目

**(a)** 定义模表示并证明块分解定理。

**(b)** 证明 Brauer 特征标和 Brauer 互反律。

**(c)** 讨论 Cartan 矩阵和分解矩阵。

## 解答

### (a) 模表示

特征 p > 0 域上的表示，p | |G| 时 Maschke 定理失效。

块分解：群代数 kG = ⊕ B_i，本原中心幂等元分解。

### (b) Brauer 特征标

p'-元素上定义，取值在 Witt 向量环。Brauer 互反：分解 d: R_K(G) → R_k(G)。

### (c) Cartan 矩阵

C = D^t D，D 分解矩阵。PIMS 在 G_0(kG) 中的展开。
