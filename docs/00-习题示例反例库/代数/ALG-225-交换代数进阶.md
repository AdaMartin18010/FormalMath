# 交换代数进阶

## 题目

**(a)** 证明 Cohen 结构定理：完备 Noether 局部环的结构。

**(b)** 讨论正则序列和 Koszul 复形。

**(c)** 证明 Matlis 对偶和局部对偶定理。

## 解答

### (a) Cohen 结构定理

等特征：R ≅ k[[x₁,...,x_n]]/I。

不等特征：R 是完备 DVR 的有限型扩张。

### (b) 正则序列

x₁,...,x_n 是 M-正则序列若 x_i 在 M/(x₁,...,x_{i-1})M 上非零因子。

Koszul 复形：计算 Tor 和 Ext。

### (c) 局部对偶

(A,m) Cohen-Macaulay，维数 d：
$$H^i_m(M) \cong \text{Ext}_A^{d-i}(M, K_A)^\vee$$
