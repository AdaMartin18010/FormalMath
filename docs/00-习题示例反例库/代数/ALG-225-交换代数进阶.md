# 交换代数进阶

## 题目

**(a)** 证明 Cohen 结构定理：完备 Noether 局部环的结构分类。

**(b)** 讨论正则序列、Koszul 复形与完全交的关系。

**(c)** 证明 Matlis 对偶和局部对偶定理。

## 解答

### (a) Cohen 结构定理

**定理**：设 $(R, \mathfrak{m})$ 是完备 Noether 局部环（$\mathfrak{m}$-adic 完备）。

**等特征情形**（$\operatorname{char}(R) = \operatorname{char}(R/\mathfrak{m})$）：

$$R \cong k[[x_1, \ldots, x_n]] / I$$

其中 $k = R/\mathfrak{m}$ 是剩余域，$I$ 是形式幂级数环的理想。

**不等特征情形**（$\operatorname{char}(R) = 0, \operatorname{char}(k) = p$）：

$R$ 是某个完备 DVR $V$ 的有限型商：

$$R \cong V[[x_1, \ldots, x_n]] / I$$

其中 $V$ 是特征零的完备 DVR，极大理想 $(p)$，剩余域 $k$。

*证明概要*：

1. **系数域**：由 Zorn 引理，存在域 $k \subset R$ 映同构到 $R/\mathfrak{m}$（当等特征）。

2. **正则参数**：取 $\mathfrak{m}$ 的生成元 $x_1, \ldots, x_n$（$n = \dim_k \mathfrak{m}/\mathfrak{m}^2$）。

3. **完备性**：$R = \varprojlim R/\mathfrak{m}^k$，每个 $R/\mathfrak{m}^k$ 是 Artin 局部环。

4. **形式幂级数映射**：$k[[x_1, \ldots, x_n]] \to R$，$x_i \mapsto \text{generator}$，由完备性这是满射。

**推论**：
- 正则局部环 $
$ \Leftrightarrow$ $I = 0$，$R \cong k[[x_1, \ldots, x_n]]$
- 完备化 $
$ \hat{R}$ 保持维数、深度、正则性

---

### (b) 正则序列与完全交

**正则序列**：$x_1, \ldots, x_n \in \mathfrak{m}$ 是 $M$-正则序列，若 $x_i$ 在 $M/(x_1, \ldots, x_{i-1})M$ 上非零因子。

**完全交**：局部环 $(R, \mathfrak{m})$ 是**完全交**，若：

$$\hat{R} \cong S / (f_1, \ldots, f_c)$$

其中 $S$ 是正则局部环，$f_1, \ldots, f_c$ 是 $S$-正则序列，$c = \dim S - \dim R$。

**等价**：$R$ 的极小自由分解长度（投射维数）等于余维数 $c$。

**Koszul 复形与完全交**：若 $x_1, \ldots, x_n$ 是 $R$-正则序列，$I = (x_1, \ldots, x_n)$，则 $K(x; R)$ 给出 $R/I$ 的极小自由分解。

完全交环的 Tor 代数是外代数：

$$\operatorname{Tor}_*^R(R/I, k) = \Lambda^*(I/\mathfrak{m}I)$$

---

### (c) Matlis 对偶与局部对偶

**Matlis 对偶**：设 $(R, \mathfrak{m})$ 是 Noether 局部环，$E = E_R(R/\mathfrak{m})$ 是 $R/\mathfrak{m}$ 的内射包。

**Matlis 对偶函子**：$D(-) = \operatorname{Hom}_R(-, E)$。

**定理**：
- $D$ 给出 Artinian $R$-模与 Noetherian $\hat{R}$-模之间的反等价
- $D(D(M)) \cong M$（对 Artinian 或有限生成模）

**局部对偶定理**（Grothendieck）：设 $(R, \mathfrak{m})$ 是 $d$ 维 Cohen-Macaulay 局部环，$M$ 是有限生成模。则：

$$
H^i_{\mathfrak{m}}(M) \cong \operatorname{Ext}_R^{d-i}(M, K_R)^\vee$$

其中 $H^i_{\mathfrak{m}}$ 是局部上同调，$K_R$ 是典范模，$(-)^\vee = \operatorname{Hom}_R(-, E)$ 是 Matlis 对偶。

*证明概要*：对 Cohen-Macaulay 环，$H^i_{\mathfrak{m}}(R) = 0$ 对 $i \neq d$，$H^d_{\mathfrak{m}}(R) \cong E$（当 Gorenstein）。

由 Grothendieck 对偶的一般理论（残复形），局部对偶是特殊情形。

**应用**：
- **Serre 对偶的局部版本**：$H^i(X, \mathcal{F}) \cong H^{n-i}(X, \mathcal{F}^* \otimes \omega_X)^*$
- **深度与局部上同调**：$\operatorname{depth}(M) = \inf\{i : H^i_{\mathfrak{m}}(M) \neq 0\}$

---

## 常见错误

- **Cohen 定理的完备性要求**：不完备的局部环没有系数域，结构更复杂。
- **完全交 vs Gorenstein**：完全交 $
$ \Rightarrow$ Gorenstein，反之不成立（如 $k[x,y,z]/(xy, xz, yz)$ 是 Gorenstein 但非完全交）。
- **Matlis 对偶的模类限制**：仅对 Artinian 模给出完全对偶，一般有限生成模需取完备化。
