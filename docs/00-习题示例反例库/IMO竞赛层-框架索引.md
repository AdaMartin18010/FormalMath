# IMO 竞赛层框架索引

## 1. 项目概述

IMO（国际数学奥林匹克）竞赛层旨在建立 FormalMath 项目与竞赛数学之间的桥梁，将银层核心课程中的概念、定理和方法映射到 IMO 真题和竞赛技巧中。

## 2. 难度分级体系

| 级别 | 描述 | 对应银层课程 |
|------|------|-------------|
| L1 | 基础概念与直接应用 | MIT 18.02 前5章 |
| L2 | 标准技巧与变形 | MIT 18.06 前8章 |
| L3 | 综合应用与构造 | MIT 18.06 全册, MIT 18.100A |
| L4 | 高级技巧与证明 | MIT 18.701, Harvard 232br |
| L5 | 研究级问题与开放问题 | Stanford FOAG, 金层内容 |

## 3. 主题分类

### 3.1 代数

- 方程与不等式（L1-L3）
- 多项式理论（L2-L4）
- 数列与递推（L2-L4）
- 函数方程（L3-L5）

### 3.2 几何

- 平面几何（L1-L4）
- 解析几何（L2-L3）
- 向量几何（L2-L3）
- 射影几何（L4-L5）

### 3.3 数论

- 整除与同余（L1-L3）
- 素数与算术函数（L2-L4）
- 不定方程（L3-L5）
- 代数数论（L5）

### 3.4 组合数学

- 计数与组合恒等式（L2-L3）
- 图论（L3-L4）
- 极值组合（L4-L5）

## 6. 形式化目标

每道 IMO 真题均配备：

- 自然语言解答（中文）
- Lean4 形式化证明（如适用）
- 银层知识点引用链接
- 难度评级与标签
