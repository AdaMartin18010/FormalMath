# 概率论习题 (ST-001 ~ ST-015)

## ST-001 ⭐⭐⭐

**题目**：设 $(B_t)_{t \geq 0}$ 是标准布朗运动。

**(a)** 证明 $B_t$ 是鞅，且二次变差 $\langle B \rangle_t = t$。

**(b)** 计算 $E[B_t^4]$ 和 $E[B_s^2 B_t^2]$（$s < t$）。

**(c)** 证明 $B_t^3 - 3t B_t$ 是鞅。

**来源**：布朗运动基础

**解答思路**：
1. (a) 增量独立、高斯性、$E[B_t - B_s | \mathcal{F}_s] = 0$
   - 二次变差：$\sum (B_{t_i} - B_{t_{i-1}})^2 \to t$（均方收敛）
2. (b) 高斯矩：$E[B_t^4] = 3t^2$
   - 分解 $B_t = B_s + (B_t - B_s)$
3. (c) Itô 公式：$d(B_t^3) = 3B_t^2 dB_t + 3B_t dt$
   - $d(B_t^3 - 3tB_t) = 3B_t^2 dB_t$（局部鞅）

**关键步骤**：
- 高斯过程性质
- 伊藤等距

## ST-002 ⭐⭐⭐⭐

**题目**：设 $M_t$ 是连续局部鞅。

**(a)** 叙述 Doob-Meyer 分解：$M_t^2 = N_t + \langle M \rangle_t$，其中 $N_t$ 是鞅。

**(b)** 对于 $M_t = \int_0^t H_s dB_s$，证明 $\langle M \rangle_t = \int_0^t H_s^2 ds$。

**(c)** 叙述 Levy 特征刻画：若 $M_t$ 是连续局部鞅，$M_0 = 0$，$\langle M \rangle_t = t$，则 $M_t$ 是布朗运动。

**来源**：鞅论基础

**解答思路**：
1. (a) Doob-Meyer：可料增过程 $\langle M \rangle_t$ 的存在唯一性
2. (b) Itô 等距：$E[(\int H dB)^2] = E[\int H^2 ds]$
3. (c) Levy 定理：验证 $E[e^{i\xi(M_t-M_s)} | \mathcal{F}_s] = e^{-\xi^2(t-s)/2}$
   - 对 $f(M_t, \langle M \rangle_t) = e^{i\xi M_t + \xi^2 t/2}$ 用 Itô 公式

**关键步骤**：
- 可料二次变差
- 指数鞅

## ST-003 ⭐⭐⭐⭐

**题目**：（随机积分）设 $H_t$ 是适应过程，满足 $E[\int_0^T H_s^2 ds] < \infty$。

**(a)** 定义伊藤积分 $\int_0^T H_s dB_s$ 作为 $L^2$ 极限。

**(b)** 证明伊藤等距：$E\left[\left(\int_0^T H_s dB_s\right)^2\right] = E\left[\int_0^T H_s^2 ds\right]$。

**(c)** 计算 $\int_0^T B_s dB_s$。

**来源**：随机微积分

**解答思路**：
1. (a) 简单过程逼近：$H_t^{(n)} = \sum H_{t_i} 1_{[t_i, t_{i+1})}(t)$
   - 证明是 $L^2$ 中的 Cauchy 列
2. (b) 对简单过程直接验证，再取极限
3. (c) $\int_0^T B_s dB_s = \frac{1}{2}(B_T^2 - T)$（伊藤校正项）

**关键步骤**：
- 伊藤过程定义
- 随机积分与路径积分区别

## ST-004 ⭐⭐⭐⭐

**题目**：（伊藤公式）设 $X_t$ 是伊藤过程：$dX_t = \mu_t dt + \sigma_t dB_t$。

**(a)** 叙述伊藤公式：对 $f \in C^{1,2}$，
$$df(t, X_t) = \partial_t f dt + \partial_x f dX_t + \frac{1}{2}\partial_{xx} f (dX_t)^2$$

**(b)** 应用：设 $S_t$ 满足几何布朗运动 $dS_t = \mu S_t dt + \sigma S_t dB_t$，求 $d(\log S_t)$。

**(c)** 证明 $S_t = S_0 \exp\left((\mu - \sigma^2/2)t + \sigma B_t\right)$。

**来源**：伊藤微积分

**解答思路**：
1. (a) 伊藤公式（链式法则带校正项）
   - $(dX_t)^2 = \sigma_t^2 dt$（伊藤乘积表）
2. (b) $f(x) = \log x$，$f'(x) = 1/x$，$f''(x) = -1/x^2$
   - $d(\log S_t) = \frac{dS_t}{S_t} - \frac{1}{2}\frac{(dS_t)^2}{S_t^2} = (\mu - \sigma^2/2)dt + \sigma dB_t$
3. (c) 积分指数形式

**关键步骤**：
- 伊藤乘积表：$dt \cdot dt = 0$，$dt \cdot dB = 0$，$dB \cdot dB = dt$
- 多维伊藤公式

## ST-005 ⭐⭐⭐⭐⭐

**题目**：（Girsanov 定理）设 $(\Omega, \mathcal{F}, P)$ 是概率空间，$B_t$ 是 $P$-布朗运动。

**(a)** 对适应过程 $\theta_t$，定义指数鞅 $Z_t = \exp\left(-\int_0^t \theta_s dB_s - \frac{1}{2}\int_0^t \theta_s^2 ds\right)$。

在 Novikov 条件 $E[\exp(\frac{1}{2}\int_0^T \theta_s^2 ds)] < \infty$ 下，定义新测度 $dQ = Z_T dP$。

**(b)** 叙述 Girsanov 定理：$\tilde{B}_t = B_t + \int_0^t \theta_s ds$ 是 $Q$-布朗运动。

**(c)** 应用：求使带漂移布朗运动变为鞅的等价鞅测度。

**来源**：测度变换

**解答思路**：
1. (a) Novikov 条件保证 $Z_t$ 是真鞅
2. (b) Levy 准则：验证 $\tilde{B}_t$ 是 $Q$-连续鞅，二次变差为 $t$
   - $E_Q[e^{i\xi \tilde{B}_t}] = E_P[Z_t e^{i\xi(B_t + \int \theta)}] = e^{-\xi^2 t/2}$
3. (c) Black-Scholes 模型中的风险中性测度

**关键步骤**：
- Cameron-Martin 定理
- 鞅表示定理

## ST-006 ⭐⭐⭐⭐⭐

**题目**：（鞅表示定理）设 $(\mathcal{F}_t)_{t \geq 0}$ 是布朗运动生成的自然滤波。

**(a)** 叙述鞅表示定理：任何 $(\mathcal{F}_t)$-局部鞅 $M_t$ 可表示为
$$M_t = M_0 + \int_0^t H_s dB_s$$
对于某个适应过程 $H_t$。

**(b)** 证明 Brownian 滤波的鞅都有连续修正。

**(c)** 应用：证明在完备市场中，任何可达权益都可对冲。

**来源**：鞅表示理论

**解答思路**：
1. (a) 鞅表示定理的证明框架：
   - 验证简单鞅的稠密性
   - Itô 等距保证表示唯一性
2. (b) Brownian 滤波是拟左连续的，鞅有右连左极修正
   - 进一步证明连续性
3. (c) Black-Scholes 模型完备性证明

**关键步骤**：
- 可料表示性
- 市场完备性

## ST-007 ⭐⭐⭐⭐

**题目**：（随机微分方程）考虑 SDE
$$dX_t = b(X_t)dt + \sigma(X_t)dB_t, \quad X_0 = x$$

**(a)** 定义强解：适应过程 $X_t$ 满足积分方程。

**(b)** 叙述存在唯一性定理：若 $b, \sigma$ 是 Lipschitz，则存在唯一强解。

**(c)** 对于 Ornstein-Uhlenbeck 过程 $dX_t = -\alpha X_t dt + \sigma dB_t$，求显式解和不变测度。

**来源**：SDE 基础

**解答思路**：
1. (a) 强解定义：$X_t$ 适应于 $B$ 生成的滤波
2. (b) Picard 迭代：$X_t^{(n+1)} = x + \int b(X_s^{(n)})ds + \int \sigma(X_s^{(n)})dB_s$
   - Gronwall 不等式 + 压缩映射
3. (c) O-U 过程：
   - 解：$X_t = e^{-\alpha t}x + \sigma \int_0^t e^{-\alpha(t-s)}dB_s$
   - 不变测度：$N(0, \sigma^2/(2\alpha))$

**关键步骤**：
- Yamada-Watanabe 定理
- Feller 性质与强马氏性

## ST-008 ⭐⭐⭐⭐⭐

**题目**：（Feynman-Kac 公式）设 $u(t,x)$ 是终值问题的解：
$$\partial_t u + \frac{1}{2}\sigma^2 \partial_{xx} u + b \partial_x u - Vu = 0, \quad u(T, x) = f(x)$$

**(a)** 叙述 Feynman-Kac 公式：
$$u(t, x) = E\left[\exp\left(-\int_t^T V(X_s)ds\right) f(X_T) \Big| X_t = x\right]$$
其中 $dX_s = b(X_s)ds + \sigma(X_s)dB_s$。

**(b)** 当 $V = 0$，证明该公式给出热方程的概率解。

**(c)** 讨论 Feynman-Kac 公式在期权定价中的应用。

**来源**：PDE 与概率联系

**解答思路**：
1. (a) 对 $M_s = u(s, X_s) \exp(-\int_t^s V(X_r)dr)$ 用 Itô 公式
   - $dM_s$ 的漂移项由 PDE 消去，$M_s$ 是鞅
2. (b) 热方程：$b = 0$，$\sigma = 1$，$u(t,x) = E[f(x + B_{T-t})]$
3. (c) 利率衍生品定价中的折现因子

**关键步骤**：
- Kolmogorov 后向方程
- 生成元与半群

## ST-009 ⭐⭐⭐⭐

**题目**：（反射原理与首中时）设 $B_t$ 是标准布朗运动，$T_a = \inf\{t > 0 : B_t = a\}$。

**(a)** 对 $a > 0$，用反射原理证明 $P(T_a \leq t) = 2P(B_t \geq a)$。

**(b)** 求 $T_a$ 的密度函数。

**(c)** 证明 $P(\sup_{0 \leq s \leq t} B_s \geq a) = 2P(B_t \geq a)$。

**来源**：布朗运动深入性质

**解答思路**：
1. (a) 反射原理：对到达 $a$ 的路径在首次到达时反射
   - $P(B_t \geq a) = P(B_t \geq a | T_a \leq t) P(T_a \leq t) = \frac{1}{2} P(T_a \leq t)$
2. (b) 对 $P(T_a \leq t)$ 求导：
   $$f_{T_a}(t) = \frac{a}{\sqrt{2\pi t^3}} e^{-a^2/(2t)}$$
3. (c) 直接由反射原理

**关键步骤**：
- 反正弦律
- 局部时

## ST-010 ⭐⭐⭐⭐⭐

**题目**：（局部时与 Tanaka 公式）设 $B_t$ 是标准布朗运动。

**(a)** 定义局部时 $L_t(a)$ 为 $B$ 在 $a$ 处停留时间的密度。

**(b)** 叙述 Tanaka 公式：
$$|B_t - a| = |B_0 - a| + \int_0^t \text{sgn}(B_s - a)dB_s + L_t(a)$$

**(c)** 证明 Occupation Time 公式：
$$\int_0^t f(B_s)ds = \int_{-\infty}^{\infty} f(a)L_t(a)da$$

**来源**：布朗运动局部时

**解答思路**：
1. (a) 局部时定义：$L_t(a) = \lim_{\epsilon \to 0} \frac{1}{2\epsilon} \int_0^t 1_{|B_s - a| < \epsilon} ds$
2. (b) Tanaka 公式：对 $|x-a|$ 用凸函数的伊藤公式
   - 梯度 $\text{sgn}$ 不连续，产生局部时项
3. (c) 由 Fubini 定理和局部时定义

**关键步骤**：
- Meyer-Tanaka 公式
- Trotter 定理（局部时的连续性）

## ST-011 ⭐⭐⭐⭐⭐

**题目**：（随机控制）考虑控制问题：最小化
$$J(u) = E\left[\int_0^T L(X_s, u_s)ds + g(X_T)\right]$$
约束于 $dX_t = b(X_t, u_t)dt + \sigma(X_t)dB_t$。

**(a)** 定义值函数 $V(t,x) = \inf_u J(u)$。

**(b)** 叙述 Hamilton-Jacobi-Bellman (HJB) 方程：
$$-\partial_t V = \inf_u \left[L(x,u) + b(x,u)\partial_x V + \frac{1}{2}\sigma^2(x)\partial_{xx} V\right]$$

**(c)** 对线性二次调节器（LQR）问题显式求解。

**来源**：随机最优控制

**解答思路**：
1. (a) 值函数是最优成本的动态规划表示
2. (b) 动态规划原理：最优性原理导出 HJB
   - 验证定理：HJB 的解给出最优控制
3. (c) LQR：$L = x^T Q x + u^T R u$，$b = Ax + Bu$
   - 解为 $V(t,x) = x^T P(t) x$，Riccati 方程

**关键步骤**：
- 验证定理
- 粘性解

## ST-012 ⭐⭐⭐⭐⭐

**题目**：（滤波理论）考虑信号-观测模型：
$$dX_t = b(X_t)dt + \sigma(X_t)dW_t \quad \text{(信号)}$$
$$dY_t = h(X_t)dt + dV_t \quad \text{(观测)}$$
其中 $W, V$ 是独立布朗运动。

**(a)** 定义滤波问题：估计 $E[f(X_t) | \mathcal{F}_t^Y]$。

**(b)** 叙述 Kushner-Stratonovich 方程。

**(c)** 对于线性高斯情形（Kalman-Bucy 滤波），推导滤波均值和协方差的 ODE。

**来源**：随机滤波

**解答思路**：
1. (a) 滤波：基于观测历史估计隐藏信号
2. (b) Kushner 方程：后验密度的随机 PDE
3. (c) Kalman-Bucy：
   - $d\hat{X}_t = A\hat{X}_t dt + P_t H^T (dY_t - H\hat{X}_t dt)$
   - Riccati 方程：$\dot{P} = AP + PA^T + Q - P H^T H P$

**关键步骤**：
- 创新过程
- Zakai 方程

## ST-013 ⭐⭐⭐⭐

**题目**：（大偏差理论）设 $(X_n)$ 是 i.i.d. 随机变量，$S_n = X_1 + \cdots + X_n$。

**(a)** 叙述 Cramér 定理：$P(S_n/n \in A) \approx \exp(-n \inf_{x \in A} I(x))$，其中 $I$ 是速率函数。

**(b)** 证明 $I(x) = \sup_\lambda (\lambda x - \Lambda(\lambda))$，其中 $\Lambda(\lambda) = \log E[e^{\lambda X}]$。

**(c)** 计算标准正态分布的速率函数。

**来源**：大偏差理论

**解答思路**：
1. (a) Cramér 定理：独立和的大偏差原理
   - 上界：Chernoff 界 + 优化
   - 下界：测度变换（Cramér 变换）
2. (b) Legendre-Fenchel 变换给出速率函数
3. (c) 正态分布：$\Lambda(\lambda) = \lambda^2/2$，$I(x) = x^2/2$

**关键步骤**：
- Varadhan 引理
- Sanov 定理

## ST-014 ⭐⭐⭐⭐⭐

**题目**：（Malliavin 微积分）设 $F$ 是 Wiener 空间上的泛函。

**(a)** 定义 Malliavin 导数 $D_t F$：方向导数沿 $t$ 处的白噪声。

**(b)** 叙述分部积分公式：对光滑 $F, G$，
$$E[F \int_0^T h_t dB_t] = E\left[\int_0^T D_t F \cdot h_t dt\right]$$

**(c)** 应用 Malliavin 微积分证明密度的存在性与光滑性。

**来源**：Malliavin 微积分

**解答思路**：
1. (a) Malliavin 导数：$D_t F = \sum \partial_i f(B_{t_1}, \ldots, B_{t_n}) 1_{[0, t_i]}(t)$
   - 闭算子，定义域 $\mathbb{D}^{1,2}$
2. (b) 分部积分：Ornstein-Uhlenbeck 算子的对偶
3. (c) Hormander 条件：Lie 括号张成 $\Rightarrow$ 密度存在

**关键步骤**：
- Skorohod 积分
- 积分表示公式

## ST-015 ⭐⭐⭐⭐⭐

**题目**：（粗糙路径理论）设 $(X_t)$ 是路径，$\mathbf{X}_{s,t}$ 是提升的粗糙路径。

**(a)** 叙述 Lyons 粗糙路径理论的核心思想：恢复被忽略的 iterated integrals。

**(b)** 对于 $p$-粗糙路径，定义 $p$-变差条件。

**(c)** 叙述 Itô 映射的连续性：粗糙微分方程的解连续依赖于驱动路径。

**来源**：粗糙路径理论

**解答思路**：
1. (a) 粗糙路径：给定路径 $X$ 和其迭代积分 $\mathbb{X}_{s,t} = \int_s^t (X_r - X_s) \otimes dX_r$
2. (b) $p$-变差：$\|X\|_{p-var} = \sup_{\mathcal{P}} \left(\sum |X_{t_i} - X_{t_{i-1}}|^p\right)^{1/p} < \infty$
   - 粗糙度：$\lfloor p \rfloor$ 级迭代积分需要给定
3. (c) Lyons 连续性定理：Itô-Lyons 映射是局部 Lipschitz 的

**关键步骤**：
- 签名的唯一性
- 受控粗糙路径
