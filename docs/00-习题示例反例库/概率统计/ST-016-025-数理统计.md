# 数理统计习题 (ST-016 ~ ST-025)

## ST-016 ⭐⭐⭐

**题目**：设 $X_1, \ldots, X_n \overset{i.i.d.}{\sim} N(\mu, \sigma^2)$。

**(a)** 求 $\mu$ 和 $\sigma^2$ 的矩估计。

**(b)** 求最大似然估计（MLE），并证明其相合性和渐近正态性。

**(c)** 证明样本均值 $\bar{X}$ 是 $\mu$ 的一致最小方差无偏估计（UMVUE）。

**来源**：估计理论基础

**解答思路**：
1. (a) 矩估计：$\hat{\mu} = \bar{X}$，$\hat{\sigma}^2 = \frac{1}{n}\sum (X_i - \bar{X})^2$
2. (b) MLE：
   - 似然函数：$L(\mu, \sigma^2) = (2\pi\sigma^2)^{-n/2} \exp\left(-\frac{1}{2\sigma^2}\sum (X_i-\mu)^2\right)$
   - $\hat{\mu}_{MLE} = \bar{X}$，$\hat{\sigma}^2_{MLE} = \frac{1}{n}\sum (X_i - \bar{X})^2$
   - 渐近理论：$\sqrt{n}(\hat{\theta} - \theta) \xrightarrow{d} N(0, I(\theta)^{-1})$
3. (c) Lehmann-Scheffé：$\bar{X}$ 是完备充分统计量的函数

**关键步骤**：
- Cramér-Rao 下界
- Fisher 信息矩阵

## ST-017 ⭐⭐⭐⭐

**题目**：（指数族）设 $f(x|\theta) = h(x) \exp\left(\eta(\theta) \cdot T(x) - A(\theta)\right)$ 是指数族。

**(a)** 证明 $T(X)$ 是充分完备统计量。

**(b)** 计算 $E[T(X)]$ 和 $\text{Var}(T(X))$。

**(c)** 对于泊松分布 $Pois(\lambda)$，求 $\lambda$ 的 UMVUE。

**来源**：指数族理论

**解答思路**：
1. (a) 因子分解定理直接给出充分性
   - 完备性：指数族的自然参数空间有内点
2. (b) $E[T] = \nabla_\eta A$，$\text{Var}(T) = \nabla^2_\eta A$
3. (c) $T(X) = X$，$E[X] = \lambda$，$X$ 是 UMVUE
   - 更一般：$e^{-\lambda}$ 的 UMVUE 是 $(1 - 1/n)^{n\bar{X}}$

**关键步骤**：
- 自然参数化
- 共轭先验

## ST-018 ⭐⭐⭐⭐

**题目**：（贝叶斯估计）设 $X|\theta \sim f(x|\theta)$，先验 $\theta \sim \pi(\theta)$。

**(a)** 定义后验分布 $\pi(\theta|x)$ 和贝叶斯估计量（后验均值、后验中位数、MAP）。

**(b)** 对于 $X_i \overset{iid}{\sim} N(\theta, 1)$，$\theta \sim N(0, \tau^2)$，求后验分布和贝叶斯估计。

**(c)** 讨论 shrinkage 效应与 James-Stein 估计的联系。

**来源**：贝叶斯统计

**解答思路**：
1. (a) 后验：$\pi(\theta|x) \propto f(x|\theta)\pi(\theta)$
   - 贝叶斯估计：$\hat{\theta} = E[\theta|X]$（二次损失下）
2. (b) 共轭先验：后验 $\theta|X \sim N\left(\frac{n\tau^2}{1+n\tau^2}\bar{X}, \frac{\tau^2}{1+n\tau^2}\right)$
   - 估计：$\hat{\theta} = \frac{n\tau^2}{1+n\tau^2}\bar{X}$（向0收缩）
3. (c) James-Stein：$p \geq 3$ 维正态均值，收缩估计优于 MLE

**关键步骤**：
- 后验一致性
- 层次贝叶斯

## ST-019 ⭐⭐⭐⭐

**题目**：（假设检验基础）设样本 $X \sim f(x|\theta)$，检验 $H_0: \theta \in \Theta_0$ vs $H_1: \theta \in \Theta_1$。

**(a)** 定义检验函数 $\phi(x)$、第一类错误、第二类错误、功效。

**(b)** 叙述 Neyman-Pearson 引理：对于简单假设，似然比检验是 UMP。

**(c)** 对于 $X_i \overset{iid}{\sim} N(\mu, 1)$，检验 $H_0: \mu = 0$ vs $H_1: \mu > 0$，求 UMP 检验。

**来源**：假设检验基础

**解答思路**：
1. (a) 定义：
   - 第一类错误：$P_{\theta_0}(\text{拒绝 } H_0)$
   - 功效：$P_{\theta_1}(\text{拒绝 } H_0)$
2. (b) Neyman-Pearson：检验 $\Lambda(x) = f(x|\theta_1)/f(x|\theta_0) > c$
   - 对于给定水平 $\alpha$，选择 $c$ 控制第一类错误
3. (c) UMP 检验：拒绝域 $\bar{X} > z_\alpha/\sqrt{n}$
   - 单调似然比性质保证 UMP

**关键步骤**：
- 一致最优势检验
- p 值计算

## ST-020 ⭐⭐⭐⭐⭐

**题目**：（似然比检验）设 $\Lambda_n = \frac{\sup_{\theta \in \Theta_0} L(\theta)}{\sup_{\theta \in \Theta} L(\theta)}$ 是似然比统计量。

**(a)** 叙述 Wilks 定理：在正则条件下，$-2\log \Lambda_n \xrightarrow{d} \chi^2_k$，其中 $k = \dim\Theta - \dim\Theta_0$。

**(b)** 对于多项分布的拟合优度检验，推导卡方统计量。

**(c)** 应用：检验两个正态总体均值相等（方差未知）。

**来源**：大样本检验

**解答思路**：
1. (a) Wilks 定理：似然比统计量的渐近分布
   - 基于 MLE 的渐近正态性
2. (b) 多项分布：Pearson 卡方
   $$\chi^2 = \sum \frac{(O_i - E_i)^2}{E_i} = -2\log \Lambda_n + o_p(1)$$
3. (c) 两样本 t 检验可视为似然比检验

**关键步骤**：
- Wald 检验、Score 检验
- 模型选择（AIC、BIC）

## ST-021 ⭐⭐⭐⭐

**题目**：（非参数估计）设 $X_1, \ldots, X_n \overset{iid}{\sim} f$，其中 $f$ 是未知密度。

**(a)** 定义核密度估计：$\hat{f}_h(x) = \frac{1}{nh}\sum_{i=1}^n K\left(\frac{x-X_i}{h}\right)$。

**(b)** 计算偏差-方差分解，求最优带宽 $h$。

**(c)** 证明在一定条件下，$\hat{f}_h(x)$ 依 $L^2$ 收敛到 $f(x)$。

**来源**：非参数统计

**解答思路**：
1. (a) 核估计：局部加权平均
   - 常用核：高斯核、Epanechnikov 核
2. (b) MSE 分解：
   - 偏差$^2$ $\sim h^4$，方差 $\sim (nh)^{-1}$
   - 最优 $h \sim n^{-1/5}$
   - 收敛速率：$n^{-4/5}$
3. (c) 一致性： bias $\to 0$，variance $\to 0$

**关键步骤**：
- 带宽选择（交叉验证）
- 高阶核

## ST-022 ⭐⭐⭐⭐⭐

**题目**：（回归分析）考虑线性模型 $Y = X\beta + \varepsilon$，$\varepsilon \sim N(0, \sigma^2 I)$。

**(a)** 求最小二乘估计 $\hat{\beta} = (X^T X)^{-1}X^T Y$，并推导其分布。

**(b)** 证明高斯-马尔可夫定理：在 BLUE 类中，LS 估计方差最小。

**(c)** 对于 $H_0: A\beta = 0$，推导 F 检验统计量。

**来源**：线性模型

**解答思路**：
1. (a) 正规方程：$X^T X \hat{\beta} = X^T Y$
   - $\hat{\beta} \sim N(\beta, \sigma^2 (X^T X)^{-1})$
2. (b) Gauss-Markov：$\hat{\beta}$ 是最佳线性无偏估计
   - 任何其他线性无偏估计方差不小于 $\hat{\beta}$
3. (c) F 统计量：$F = \frac{(RSS_R - RSS_{UR})/q}{RSS_{UR}/(n-p)} \sim F_{q, n-p}$

**关键步骤**：
- 投影矩阵
- 岭回归、Lasso

## ST-023 ⭐⭐⭐⭐⭐

**题目**：（经验过程）设 $X_1, \ldots, X_n \overset{iid}{\sim} P$，经验分布 $P_n = \frac{1}{n}\sum \delta_{X_i}$。

**(a)** 定义经验过程 $\mathbb{G}_n(f) = \sqrt{n}(P_n f - P f)$。

**(b)** 叙述 Donsker 定理：对于 P-Donsker 类 $\mathcal{F}$，$\mathbb{G}_n \xrightarrow{d} \mathbb{G}$（$P$-布朗桥）。

**(c)** 应用：证明 Kolmogorov-Smirnov 统计量的极限分布。

**来源**：经验过程理论

**解答思路**：
1. (a) 经验过程：随机函数在函数类上的行为
   - Glivenko-Cantelli：一致大数律
2. (b) Donsker 定理：函数中心极限定理
   - 熵条件保证 Donsker 性质
3. (c) K-S 统计量：$\sup_t |\mathbb{G}_n(1_{(-\infty, t]})|$
   - 极限：$\sup_{0 \leq u \leq 1} |B(u)|$，$B$ 是布朗桥

**关键步骤**：
- Vapnik-Chervonenkis 维
- 括号熵

## ST-024 ⭐⭐⭐⭐⭐

**题目**：（因果推断）设 $Y_i(1), Y_i(0)$ 是潜在结果，$D_i$ 是处理指示，$Y_i = D_i Y_i(1) + (1-D_i)Y_i(0)$。

**(a)** 定义平均处理效应（ATE）：$\tau = E[Y(1) - Y(0)]$。

**(b)** 在无混淆性（unconfoundedness）$D \perp (Y(0), Y(1)) | X$ 下，证明
$$\tau = E_X[E[Y|D=1, X] - E[Y|D=0, X]]$$

**(c)** 讨论倾向得分匹配、逆概率加权等估计方法。

**来源**：因果推断

**解答思路**：
1. (a) ATE：处理组和对照组的期望差异
2. (b) 识别：条件期望调整控制混杂
   - $E[Y|D=1] - E[Y|D=0]$ 有偏（选择偏差）
3. (c) 估计方法：
   - 倾向得分：$e(X) = P(D=1|X)$
   - IPW：加权消除选择偏差

**关键步骤**：
- Rubin 因果模型
- 工具变量

## ST-025 ⭐⭐⭐⭐⭐

**题目**：（高维统计）设 $Y = X\beta + \varepsilon$，$X \in \mathbb{R}^{n \times p}$，$p \gg n$。

**(a)** 叙述 Lasso 估计：$\hat{\beta} = \arg\min_\beta \left\{\|Y - X\beta\|_2^2 + \lambda \|\beta\|_1\right\}$。

**(b)** 讨论 Lasso 的变量选择一致性（Irrepresentable 条件）。

**(c)** 比较 Lasso、Ridge、Elastic Net 在高维回归中的适用性。

**来源**：高维统计

**解答思路**：
1. (a) Lasso：$\ell_1$ 惩罚产生稀疏解
   - 几何解释：约束集的角点在坐标轴上
2. (b) 选择一致性：在 Irrepresentable 条件下，Lasso 以高概率选对变量集
3. (c) 比较：
   - Ridge：$\ell_2$ 惩罚，收缩但不稀疏
   - Elastic Net：组合 $\ell_1 + \ell_2$，处理组效应

**关键步骤**：
- 受限特征值条件
- 偏差-方差权衡
