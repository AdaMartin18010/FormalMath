# 常见错误: 混淆逐点收敛与一致收敛

## 所属领域

分析

## 错误描述

初学者常将逐点收敛(pointwise convergence)与一致收敛(uniform convergence)混为一谈，认为"函数列在每点都收敛"就足以保证极限函数继承各项的良好性质（连续性、可积性、可微性）。这是一个极其普遍且危害甚大的错误。

### 形式化表述

设 $(f_n)$ 是定义在集合 $E \subseteq \mathbb{R}$ 上的函数列，$f: E \to \mathbb{R}$。

**逐点收敛**：对每个 $x \in E$，数列 $(f_n(x))$ 收敛于 $f(x)$，即
$$\forall x \in E, \forall \varepsilon > 0, \exists N = N(x, \varepsilon) \text{ s.t. } n \geq N \Rightarrow |f_n(x) - f(x)| < \varepsilon$$

**一致收敛**：$f_n$ 在 $E$ 上一致收敛于 $f$，即
$$\forall \varepsilon > 0, \exists N = N(\varepsilon) \text{ s.t. } \forall x \in E, n \geq N \Rightarrow |f_n(x) - f(x)| < \varepsilon$$

关键区别在于：一致收敛中的 $N$ 仅依赖于 $\varepsilon$ 而不依赖于 $x$，而逐点收敛中的 $N$ 可以随 $x$ 变化。

## 正确理解

### 极限函数的连续性

**定理**（一致收敛保持连续性）：若 $f_n \rightrightarrows f$ 在 $E$ 上一致收敛，且每个 $f_n$ 在 $x_0 \in E$ 连续，则 $f$ 也在 $x_0$ 连续。

**证明**：给定 $\varepsilon > 0$，由一致收敛，存在 $N$ 使得对所有 $x \in E$：
$$|f_N(x) - f(x)| < \frac{\varepsilon}{3}$$

由 $f_N$ 在 $x_0$ 连续，存在 $\delta > 0$ 使得 $|x - x_0| < \delta$ 时：
$$|f_N(x) - f_N(x_0)| < \frac{\varepsilon}{3}$$

于是当 $|x - x_0| < \delta$ 时：
$$|f(x) - f(x_0)| \leq |f(x) - f_N(x)| + |f_N(x) - f_N(x_0)| + |f_N(x_0) - f(x_0)| < \frac{\varepsilon}{3} + \frac{\varepsilon}{3} + \frac{\varepsilon}{3} = \varepsilon$$

$\square$

**注意**：一致收敛是**充分条件**而非必要条件。例如，$f_n(x) = x^n$ 在 $[0,1]$ 上逐点收敛于 $f(x) = \begin{cases} 0 & x \in [0,1) \\ 1 & x = 1 \end{cases}$，不一致收敛，且极限不连续——这正是下面反例的核心。

### 极限函数的积分

**定理**（一致收敛与积分）：若 $f_n \rightrightarrows f$ 在 $[a,b]$ 上一致收敛，且每个 $f_n$ 可积，则 $f$ 可积且
$$\lim_{n \to \infty} \int_a^b f_n(x)\, dx = \int_a^b f(x)\, dx$$

### 极限函数的微分

**定理**（一致收敛与微分）：若 $f_n \to f$ 逐点收敛，$f_n'$ 在 $[a,b]$ 上一致收敛于 $g$，且每个 $f_n'$ 连续，则 $f$ 可微且 $f' = g$。

注意：这里要求**导数列一致收敛**，而非函数列本身。

## 反例

### 反例1：连续函数列的逐点极限不连续

设 $f_n(x) = x^n$，定义域 $[0,1]$。

- 每个 $f_n$ 都是 $[0,1]$ 上的连续函数
- 逐点极限为 $f(x) = \begin{cases} 0 & 0 \leq x < 1 \\ 1 & x = 1 \end{cases}$
- 极限函数 $f$ 在 $x = 1$ 处不连续

**验证不一致收敛**：取 $\varepsilon = \frac{1}{2}$，对任意 $N$，取 $x = (\frac{1}{2})^{1/N} \in [0,1)$，则 $x^N = \frac{1}{2}$，于是 $|f_N(x) - f(x)| = \frac{1}{2} \not< \varepsilon$。

### 反例2：积分与极限不可交换

构造"滑动驼峰"函数：
$$f_n(x) = \begin{cases} n^2 x & 0 \leq x \leq \frac{1}{n} \\ 2n - n^2 x & \frac{1}{n} < x \leq \frac{2}{n} \\ 0 & \frac{2}{n} < x \leq 1 \end{cases}$$

- 每个 $f_n$ 在 $[0,1]$ 上连续
- $f_n \to 0$ 逐点（对任意固定的 $x > 0$，当 $n > \frac{2}{x}$ 时 $f_n(x) = 0$）
- 但 $\int_0^1 f_n(x)\, dx = 1$（三角形面积 $= \frac{1}{2} \cdot \frac{2}{n} \cdot n = 1$）
- 因此 $\lim_{n \to \infty} \int_0^1 f_n(x)\, dx = 1 \neq 0 = \int_0^1 \lim_{n \to \infty} f_n(x)\, dx$

### 反例3：导数极限不等于极限的导数

设 $f_n(x) = \frac{x}{1 + nx^2}$，定义域 $\mathbb{R}$。

- $f_n(0) = 0$，对 $x \neq 0$：$f_n(x) \to 0$，故 $f_n \to 0$ 逐点
- 计算导数：$f_n'(x) = \frac{1 - nx^2}{(1 + nx^2)^2}$
- $f_n'(0) = 1$，故 $\lim_{n \to \infty} f_n'(0) = 1$
- 但极限函数 $f(x) = 0$ 的导数 $f'(0) = 0$
- 因此 $\lim_{n \to \infty} f_n'(0) = 1 \neq 0 = \left(\lim_{n \to \infty} f_n\right)'(0)$

## 避免方法

1. **验证一致收敛性**：在交换极限与积分/微分/连续性之前，务必验证是否一致收敛。常用判别法：
   - **Weierstrass M-判别法**：若 $|f_n(x)| \leq M_n$ 且 $\sum M_n < \infty$，则 $\sum f_n$ 一致收敛
   - **Dini定理**：在紧集上，单调连续函数列的逐点极限若连续，则一致收敛

2. **检查控制条件**：对于积分，Lebesgue控制收敛定理只需逐点收敛加上可积控制函数 $g \geq |f_n|$：
   $$\text{若 } f_n \to f \text{ a.e.}, |f_n| \leq g \in L^1, \text{ 则 } \int f_n \to \int f$$

3. **构造反例加深理解**：遇到"极限保持性质"的命题时，先尝试用 $x^n$、滑动驼峰等经典反例检验直觉。

4. **注意收敛的拓扑**：逐点收敛对应乘积拓扑（弱拓扑），一致收敛对应上确界范数拓扑（强拓扑）。后者是Banach空间中的自然拓扑，性质更好。


## 相关定理与概念
- **Dini定理**：紧集上单调连续函数列的逐点极限若连续，则一致收敛
- **Weierstrass M-判别法**：$|f_n(x)| \leq M_n$，$\sum M_n < \infty$ $\Rightarrow$ $\sum f_n$ 一致收敛
- **Arzelà-Ascoli定理**：一致有界且等度连续的函数列有 uniformly convergent 子列
- **Egorov定理**：有限测度空间上，a.e.收敛的函数列在去掉小测度集后一致收敛
- **Lebesgue控制收敛定理（DCT）**：只需逐点收敛加上可积控制函数
