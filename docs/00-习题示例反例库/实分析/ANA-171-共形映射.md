# 共形映射

## 题目

**(a)** 证明 Riemann 映射定理：单连通真子集 $\Omega \subsetneq \mathbb{C}$ 与单位圆盘双全纯等价。

**(b)** 显式构造上半平面到单位圆盘的共形映射。

**(c)** 求将单位圆盘映射到自身的所有分式线性变换。

## 解答

### (a) Riemann 映射定理概要

利用正规族和单叶函数序列的极限。唯一性由 Schwarz 引理保证。

### (b) Cayley 变换

$$f(z) = \frac{z-i}{z+i}$$

将 $i$ 映射到 $0$，实轴映射到单位圆周。

### (c) 自同构群

$$\text{Aut}(\mathbb{D}) = \left\{e^{i\theta} \frac{z-a}{1-\bar{a}z} : |a| < 1, \theta \in \mathbb{R}\right\}$$
