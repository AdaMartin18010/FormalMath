# 随机矩阵论

## 题目

**(a)** 证明半圆律：Wigner矩阵的特征值分布收敛于半圆分布。

**(b)** 证明Marcenko-Pastur律：样本协方差矩阵的极限分布。

**(c)** 讨论随机矩阵的普适性。

---

## 解答

### (a) Wigner半圆律

#### Wigner矩阵

**定义**：$n \times n$ **Wigner矩阵** $W_n = (X_{ij})$ 满足：
- $X_{ij} = X_{ji}$（对称/Hermitian）
- $\{X_{ij}\}_{i \leq j}$ 独立
- $\mathbb{E}[X_{ij}] = 0$，$\mathbb{E}[X_{ij}^2] = \sigma^2$
- **Lindeberg条件**：$\frac{1}{n^2} \sum_{i,j} \mathbb{E}[X_{ij}^2 \mathbf{1}_{|X_{ij}| > \epsilon \sqrt{n}}] \to 0$

#### 经验谱分布

$W_n$ 的特征值 $\lambda_1 \leq \lambda_2 \leq \cdots \leq \lambda_n$，**经验谱分布**（ESD）：
$$\mu_n = \frac{1}{n} \sum_{i=1}^n \delta_{\lambda_i / \sqrt{n}}$$

#### 半圆律

**定理**（Wigner, 1955-1958）：当 $n \to \infty$ 时，$\mu_n$ 弱收敛于**半圆分布**：
$$\mu_{sc}(dx) = \frac{1}{2\pi\sigma^2} \sqrt{4\sigma^2 - x^2} \, \mathbf{1}_{|x| \leq 2\sigma} dx$$

#### 证明：矩方法

**步骤1：矩的计算**

$m_k^{(n)} = \int x^k \, d\mu_n = \frac{1}{n} \sum_{i=1}^n \left(\frac{\lambda_i}{\sqrt{n}}\right)^k = \frac{1}{n^{k/2+1}} \text{Tr}(W_n^k)$

**步骤2：展开迹**

$$\text{Tr}(W_n^k) = \sum_{i_1, \ldots, i_k} X_{i_1 i_2} X_{i_2 i_3} \cdots X_{i_k i_1}$$

由独立性和零均值，只有每个因子至少出现两次的项才非零。

**步骤3：图表示**

将 $(i_1, \ldots, i_k)$ 对应到图：顶点 = 不同的指标，边 = 矩阵元 $X_{ij}$。

非零项对应**闭合路径**，每条边经过至少两次。

**步骤4：计数**

对 $k = 2m$（奇数矩为0由对称性）：
- 不同顶点数 $\leq m + 1$
- 若不同顶点数 $< m + 1$，贡献 $O(n^m)$，除以 $n^{m+1}$ 后趋于0
- 若不同顶点数 $= m + 1$，则图是**树**（或更准确地说，是某种特定的闭合路径）

Catalan数出现：$m_{2m} = \sigma^{2m} C_m = \sigma^{2m} \frac{1}{m+1}\binom{2m}{m}$

**步骤5：矩确定分布**

半圆分布的矩恰好是Catalan数：
$$\int_{-2\sigma}^{2\sigma} x^{2m} \frac{\sqrt{4\sigma^2 - x^2}}{2\pi\sigma^2} dx = \sigma^{2m} C_m$$

由矩问题的唯一性（Carleman条件），$\mu_n \to \mu_{sc}$。

$\square$

#### 证明：Stieltjes变换

**Stieltjes变换**：对概率测度 $\mu$，
$$m_\mu(z) = \int \frac{1}{x - z} \mu(dx), \quad z \in \mathbb{C}^+$$

**步骤1**：对 $\mu_n$，$m_n(z) = \frac{1}{n} \text{Tr}(W_n - zI)^{-1}$

**步骤2**：由**留数恒等式**或**Schur补**，导出 $m_n$ 的方程。

**步骤3**：当 $n \to \infty$，$m_n \to m$ 满足二次方程：
$$m(z) = \frac{1}{-z - \sigma^2 m(z)}$$

解得：
$$m(z) = \frac{-z + \sqrt{z^2 - 4\sigma^2}}{2\sigma^2}$$

**步骤4**：由Stieltjes反演公式，$\mu$ 有密度 $\frac{1}{\pi} \text{Im}\, m(x + i0) = \frac{\sqrt{4\sigma^2 - x^2}}{2\pi\sigma^2}$。

### (b) Marcenko-Pastur律

#### 样本协方差矩阵

设 $X$ 是 $p \times n$ 随机矩阵，列独立同分布，$\mathbb{E}[X_{ij}] = 0$，$\mathbb{E}[X_{ij}^2] = 1$。

**样本协方差矩阵**：
$$S_n = \frac{1}{n} X X^T$$

$\lambda_1, \ldots, \lambda_p$ 是 $S_n$ 的特征值，ESD：
$$\mu_n = \frac{1}{p} \sum_{i=1}^p \delta_{\lambda_i}$$

#### Marcenko-Pastur定理

**定理**（Marcenko-Pastur, 1967）：设 $p, n \to \infty$，$p/n \to y \in (0, \infty)$，则 $\mu_n \to \mu_{MP}$：

- 若 $0 < y \leq 1$：
$$\mu_{MP}(dx) = \frac{\sqrt{(b - x)(x - a)}}{2\pi x y} \mathbf{1}_{[a,b]} dx$$
其中 $a = (1 - \sqrt{y})^2$，$b = (1 + \sqrt{y})^2$

- 若 $y > 1$：加上在0处的原子 $1 - \frac{1}{y}$

#### 证明概要

**Stieltjes变换方法**：

$m_n(z) = \frac{1}{p} \text{Tr}(S_n - zI)^{-1}$

由**留数恒等式**：
$$(S_n - zI)^{-1} = \frac{1}{\frac{1}{n} X X^T - zI}$$

用Sherman-Morrison公式或**Schur补**，导出固定点方程。

当 $n \to \infty$，$m(z)$ 满足：
$$m(z) = \frac{1}{-z + y - 1 - z y m(z)}$$

解这个二次方程，得到Marcenko-Pastur密度。

#### 应用

1. **主成分分析**（PCA）：大样本时，样本协方差矩阵的特征值分布由MP律描述
2. **信号检测**：信号特征值超出MP支撑 $[a,b]$，可作为检测准则
3. **无线通信**：MIMO信道的容量分析

### (c) 普适性

#### 陈述

**普适性猜想**：Wigner矩阵的局部谱统计（特征值间距、关联函数等）仅依赖于对称性类（实对称、复Hermitian、四元数自对偶），而与具体分布无关。

#### 三大家族

Wigner-Dyson按照对称性分类：

| 类 | 对称性 | 随机矩阵系综 | 例子 |
|----|--------|--------------|------|
| **GOE** | 时间反演 + 自旋旋转 | 实对称Wigner | 重原子核（慢中子） |
| **GUE** | 破缺时间反演 | 复HermitianWigner | 复杂量子系统 |
| **GSE** | 时间反演 + 半整数自旋 | 四元数自对偶 | 无自旋-轨道耦合 |

#### 局部统计

**特征值间距分布**：设 $s = \frac{\lambda_{i+1} - \lambda_i}{\text{平均间距}}$，则：

- **GOE**（$\beta = 1$）：$p_1(s) \approx \frac{\pi s}{2} e^{-\pi s^2/4}$
- **GUE**（$\beta = 2$）：$p_2(s) \approx \frac{32 s^2}{\pi^2} e^{-4s^2/\pi}$
- **GSE**（$\beta = 4$）：$p_4(s) \approx \frac{2^{18} s^4}{3^6 \pi^3} e^{-64s^2/9\pi}$

**关键特征**：小 $s$ 时 $p_\beta(s) \sim s^\beta$（**能级排斥**），与Poisson分布 $p(s) = e^{-s}$（无排斥）截然不同。

#### 普适性定理

**定理**（Erdős-Schlein-Yau, Tao-Vu, 2010s）：

对Wigner矩阵（满足适当矩条件），局部谱统计收敛于相应的随机矩阵系综（GOE/GUE/GSE）。

**技术**：
1. **Lindeberg替换**：逐步替换矩阵元为高斯元
2. **局部半圆律**：在微观尺度（$O(1/n)$）上控制Stieltjes变换
3. **Dyson Brownian motion**：特征值的随机微分方程

#### 应用

1. **量子混沌**：能级间距的GOE/GUE统计是量子混沌的标志
2. **数论**：Riemann $\zeta$ 函数的零点间距与GUE相同（Montgomery-Odlyzko定律）
3. **神经网络**：权重矩阵的谱分析
4. **生态学**：物种丰度分布的随机矩阵模型
