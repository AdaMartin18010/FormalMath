# ANA-016: 夹逼定理进阶应用

**题号**: ANA-016
**难度**: ⭐⭐⭐ (Level 3)
**类型**: 计算型 (CALC)
**来源**: MIT 18.100A Chapter 2.3
**主题**: 夹逼定理的创造性应用

---

## 题目

利用夹逼定理计算下列极限：

**(a)** 计算 $\displaystyle\lim_{n\to\infty} \sum_{k=1}^{n^2} \frac{n}{n^3 + k}$

**(b)** 计算 $\displaystyle\lim_{n\to\infty} \frac{1}{n^2} \sum_{k=1}^{n} k e^{k/n^2}$

**(c)** 设 $a_n = \sum_{k=1}^{n} \frac{1}{\sqrt{n^2 + k}}$，求 $\lim_{n\to\infty} (a_n - 1) \cdot n$

---

## 解答

### (a) 求 $\displaystyle\lim_{n\to\infty} \sum_{k=1}^{n^2} \frac{n}{n^3 + k}$

**分析**: 和式有 $n^2$ 项，每项量级为 $\frac{n}{n^3} = \frac{1}{n^2}$。

**解**:

对 $k = 1, 2, \ldots, n^2$：
$$n^3 + 1 \leq n^3 + k \leq n^3 + n^2$$

因此：
$$\frac{n}{n^3 + n^2} \leq \frac{n}{n^3 + k} \leq \frac{n}{n^3 + 1}$$

对 $k$ 从 $1$ 到 $n^2$ 求和：
$$\frac{n \cdot n^2}{n^3 + n^2} \leq \sum_{k=1}^{n^2} \frac{n}{n^3 + k} \leq \frac{n \cdot n^2}{n^3 + 1}$$

即：
$$\frac{n^3}{n^3 + n^2} \leq S_n \leq \frac{n^3}{n^3 + 1}$$

计算边界：
$$\frac{n^3}{n^3 + n^2} = \frac{1}{1 + 1/n} \to 1$$
$$\frac{n^3}{n^3 + 1} = \frac{1}{1 + 1/n^3} \to 1$$

由夹逼定理：
$$\lim_{n\to\infty} \sum_{k=1}^{n^2} \frac{n}{n^3 + k} = 1 \quad \square$$

---

### (b) 求 $\displaystyle\lim_{n\to\infty} \frac{1}{n^2} \sum_{k=1}^{n} k e^{k/n^2}$

**分析**: 指数项 $e^{k/n^2}$ 当 $k \leq n$ 时接近1。

**解**:

对 $k \in [1, n]$，有 $0 < \frac{k}{n^2} \leq \frac{1}{n}$。

利用 $e^x = 1 + x + \frac{x^2}{2}e^{\xi}$（$\xi$ 在 $0$ 与 $x$ 之间）：
$$1 + \frac{k}{n^2} \leq e^{k/n^2} \leq 1 + \frac{k}{n^2} + O\left(\frac{1}{n^2}\right)$$

更简单的估计：
$$1 \leq e^{k/n^2} \leq e^{1/n} = 1 + O\left(\frac{1}{n}\right)$$

因此：
$$\frac{1}{n^2} \sum_{k=1}^{n} k \leq \frac{1}{n^2} \sum_{k=1}^{n} k e^{k/n^2} \leq \frac{e^{1/n}}{n^2} \sum_{k=1}^{n} k$$

计算 $\sum_{k=1}^{n} k = \frac{n(n+1)}{2}$：
$$\frac{n(n+1)}{2n^2} \leq S_n \leq \frac{e^{1/n} \cdot n(n+1)}{2n^2}$$

即：
$$\frac{n+1}{2n} \leq S_n \leq e^{1/n} \cdot \frac{n+1}{2n}$$

边界都收敛于 $\frac{1}{2}$。

由夹逼定理：
$$\lim_{n\to\infty} \frac{1}{n^2} \sum_{k=1}^{n} k e^{k/n^2} = \frac{1}{2} \quad \square$$

---

### (c) 渐进展开

**分析**: 先求主项，再分析余项。

**解**:

**Step 1: 主项估计**

$$a_n = \sum_{k=1}^{n} \frac{1}{\sqrt{n^2 + k}} = \frac{1}{n} \sum_{k=1}^{n} \frac{1}{\sqrt{1 + k/n^2}}$$

对 $k \in [1, n]$，有 $0 < \frac{k}{n^2} \leq \frac{1}{n}$。

利用 $(1+x)^{-1/2} = 1 - \frac{x}{2} + O(x^2)$：
$$\frac{1}{\sqrt{1 + k/n^2}} = 1 - \frac{k}{2n^2} + O\left(\frac{1}{n^2}\right)$$

**Step 2: 求和**
$$a_n = \frac{1}{n} \sum_{k=1}^{n} \left(1 - \frac{k}{2n^2} + O\left(\frac{1}{n^2}\right)\right)$$
$$= \frac{1}{n} \cdot n - \frac{1}{2n^3} \cdot \frac{n(n+1)}{2} + O\left(\frac{1}{n^2}\right)$$
$$= 1 - \frac{n+1}{4n^2} + O\left(\frac{1}{n^2}\right)$$
$$= 1 - \frac{1}{4n} + O\left(\frac{1}{n^2}\right)$$

**Step 3: 求目标极限**
$$(a_n - 1) \cdot n = -\frac{1}{4} + O\left(\frac{1}{n}\right) \to -\frac{1}{4}$$

因此：
$$\lim_{n\to\infty} (a_n - 1) \cdot n = -\frac{1}{4} \quad \square$$

| 技巧 | 应用位置 | 说明 |

| 统一放缩 | (a) | 对分母进行统一估计 |
| Taylor展开 | (b)(c) | 对小量进行展开 |
| 主项分析 | (c) | 分离主导项和余项 |
| 渐进估计 | (c) | 计算下一阶修正 |

### 关键洞察

1. **夹逼的精度**: 边界的精度决定了能否得到正确的极限
2. **项数的影响**: 项数与单项大小的权衡决定主导行为
3. **渐进展开**: 对于接近1的量，一阶展开往往足够

## 常见错误

### 错误1: 放缩过松

❌ **错误做法**:
> 对(a)直接写 $0 \leq S_n \leq n^2 \cdot \frac{n}{n^3} = 1$，虽然结论对但估计太松

✅ **正确做法**:
> 应该利用求和项数进行更精细的估计

### 错误2: 忽略高阶项累积

❌ **错误做法**:
> 在(c)中，认为 $O(1/n^2)$ 乘以 $n$ 项后仍为 $O(1/n^2)$

✅ **正确做法**:
> $n$ 个 $O(1/n^2)$ 项累加为 $O(1/n)$，需要仔细估计

### 错误3: 边界选择不当

❌ **错误做法**:
> 在夹逼时，上下界收敛速度差异大

✅ **正确做法**:
> 上下界应该"夹得足够紧"，收敛于同一值

## 变式练习

### 变式1（难度⭐⭐⭐）

计算 $\displaystyle\lim_{n\to\infty} \sum_{k=1}^{n} \frac{k}{n^3 + k^2}$

**答案**: $\frac{1}{2}$

### 变式2（难度⭐⭐⭐⭐）

计算 $\displaystyle\lim_{n\to\infty} \frac{1}{n^3} \sum_{k=1}^{n} k^2 e^{-k/n}$

**答案**: $2 - 5e^{-1}$

**提示**: 将求和转化为Riemann和或利用分部求和

### 变式3（难度⭐⭐⭐⭐）

设 $a_n = \sum_{k=n}^{2n} \frac{1}{k}$，求 $\lim_{n\to\infty} n(a_n - \ln 2)$

**答案**: $-\frac{1}{4}$

**提示**: 利用 $H_n = \ln n + \gamma + \frac{1}{2n} + O(1/n^2)$
