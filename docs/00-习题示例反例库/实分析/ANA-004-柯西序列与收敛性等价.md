# ANA-004: 柯西序列与收敛性等价

## 题目陈述

**定义（柯西序列）**: 序列 $(a_n)$ 称为柯西序列，如果对任意 $\varepsilon > 0$，存在 $N \in \mathbb{N}$ 使得
$$|a_m - a_n| < \varepsilon \quad \text{对所有 } m, n \geq N \text{ 成立}$$

**定理**: 实数序列 $(a_n)$ 收敛当且仅当它是柯西序列。

请给出该定理的完整证明，并讨论该定理在一般度量空间中的成立条件。

## 详细解答

### 第一部分：收敛 ⇒ 柯西

**定理**: 若 $(a_n)$ 收敛，则 $(a_n)$ 是柯西序列。

**证明**:

设 $\lim_{n \to \infty} a_n = L$。对任意 $\varepsilon > 0$，取 $N$ 使得
$$|a_n - L| < \frac{\varepsilon}{2} \quad \text{对所有 } n \geq N$$

则对任意 $m, n \geq N$：

$$|a_m - a_n| = |a_m - L + L - a_n| \leq |a_m - L| + |L - a_n| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon$$

故 $(a_n)$ 是柯西序列。 $\square$

---

### 第二部分：柯西 ⇒ 收敛

**定理**: 若 $(a_n)$ 是柯西序列，则 $(a_n)$ 收敛。

**证明**:

**步骤1：柯西序列有界**

取 $\varepsilon = 1$，存在 $N$ 使得 $|a_m - a_n| < 1$ 对所有 $m, n \geq N$ 成立。

特别地，$|a_n - a_N| < 1$ 对所有 $n \geq N$ 成立，即
$$|a_n| < |a_N| + 1 \quad \text{对所有 } n \geq N$$

令 $M = \max\{|a_1|, |a_2|, \ldots, |a_{N-1}|, |a_N| + 1\}$，则 $|a_n| \leq M$ 对所有 $n$ 成立。 $\checkmark$

**步骤2：应用Bolzano-Weierstrass定理**

由于 $(a_n)$ 有界，由Bolzano-Weierstrass定理，存在收敛子列 $(a_{n_k})$。

设 $\lim_{k \to \infty} a_{n_k} = L$。

**步骤3：证明原序列收敛于 $L$**

对任意 $\varepsilon > 0$：

- 由柯西条件，存在 $N_1$ 使得 $|a_m - a_n| < \frac{\varepsilon}{2}$ 对所有 $m, n \geq N_1$ 成立
- 由子列收敛，存在 $K$ 使得 $|a_{n_k} - L| < \frac{\varepsilon}{2}$ 对所有 $k \geq K$ 成立
- 取 $k$ 足够大使得 $n_k \geq N_1$，则对任意 $n \geq N_1$：

$$|a_n - L| \leq |a_n - a_{n_k}| + |a_{n_k} - L| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon$$

故 $\lim_{n \to \infty} a_n = L$。 $\square$

---

### 第三部分：一般度量空间的情形

**定义**: 度量空间 $(X, d)$ 称为**完备**的，如果其中每个柯西序列都收敛。

**重要结论**:

| 空间 | 是否完备 | 说明 |
|------|----------|------|
| $\mathbb{R}$ | ✓ | 本题定理 |
| $\mathbb{R}^n$ | ✓ | 有限维赋范空间 |
| $\mathbb{Q}$ | ✗ | 柯西序列可能收敛于无理数 |
| $(0, 1)$ | ✗ | 序列可能收敛于0或1（不在空间内） |
| $C([a,b])$（一致度量） | ✓ | 一致收敛保持连续性 |
| $C([a,b])$（$L^1$度量） | ✗ | 极限可能是可积函数而非连续函数 |
| $\ell^p$（$1 \leq p \leq \infty$） | ✓ | 经典的Banach空间 |

**不完备空间的柯西序列例子**:

在 $\mathbb{Q}$ 中，设 $a_n$ 为 $\sqrt{2}$ 的有理逼近（如连分数截断）。$(a_n)$ 是柯西序列，但在 $\mathbb{Q}$ 中不收敛。

## 证明技巧总结

### 柯西条件的核心思想

```
自收敛性: 序列的项彼此"越来越接近"
          不需要知道极限，只比较序列内部

对比: 收敛定义    |a_n - L| < ε  (需要知道L)
      柯西定义    |a_m - a_n| < ε  (不需要知道L)
```

### 证明柯西⇒收敛的关键步骤

```
柯西序列 ──→ 有界序列 ──→ 有收敛子列 ──→ 原序列收敛
           (取ε=1)      (BW定理)       (利用柯西条件"拖拽")
```

### 技巧：用柯西条件"拖拽"子列极限

这是分析学中的经典技巧：

1. 柯西性保证：任意两项充分靠后时距离很小
2. 子列收敛：子列中某项充分靠后时接近极限
3. 组合：任意项与极限的距离 ≤ (与该项接近的子列项的距离) + (该子列项与极限的距离)

## 常见错误分析

### 错误1：混淆柯西序列与单调序列

```
❌ 错误：认为柯西序列一定是单调的

✅ 正确：柯西序列不必单调，如 a_n = (-1)^n / n

💡 记忆：柯西序列 = 自收敛，单调序列 = 方向一致
```

### 错误2：柯西⇒收敛证明中跳过有界性

```
❌ 错误：直接应用BW定理而不证明有界性

✅ 正确：BW定理要求有界性，必须先证

💡 技巧：柯西序列取ε=1立即得有界性
```

### 错误3：收敛⇒柯西证明中N的选择

```
❌ 错误："存在N使得|a_n - L| < ε"

✅ 正确："存在N使得|a_n - L| < ε/2"

💡 原因：三角不等式会累加误差，需要预留余量
```

### 错误4：认为所有度量空间都完备

```
❌ 错误："柯西序列必收敛是普遍真理"

✅ 正确：完备性是空间的性质，需要验证

💡 反例：(0,1)不完备，a_n = 1/n 柯西但不收敛
```

### 错误5：混淆柯西序列与子列柯西

```
❌ 错误："若所有子列都柯西，则原序列柯西"

✅ 正确：原序列柯西 ⇒ 所有子列柯西，反之不成立

💡 反例：a_n = (-1)^n，所有真子列都可柯西（实际上都收敛），但原序列不柯西
```

## 变式练习

### 变式1（难度⭐⭐）

设 $(a_n)$ 满足 $|a_{n+1} - a_n| \leq \frac{1}{2^n}$ 对所有 $n$ 成立。证明 $(a_n)$ 收敛。

<details>
<summary>提示</summary>
证明 $(a_n)$ 是柯西序列：

对 $m > n$，
$$|a_m - a_n| \leq \sum_{k=n}^{m-1} |a_{k+1} - a_k| \leq \sum_{k=n}^{m-1} \frac{1}{2^k} < \frac{1}{2^{n-1}}$$

当 $n$ 充分大时，上式可任意小。 $\square$
</details>

### 变式2（难度⭐⭐⭐）

设 $(a_n)$ 满足对任意 $\varepsilon > 0$，存在 $N$ 使得 $|a_{n+1} - a_n| < \varepsilon$ 对所有 $n \geq N$ 成立（即相邻项距离趋于0）。问：$(a_n)$ 是否必为柯西序列？

<details>
<summary>提示</summary>
<strong>答案：否。</strong>

反例：$a_n = \sum_{k=1}^n \frac{1}{k}$（调和级数）

- 相邻项距离：$|a_{n+1} - a_n| = \frac{1}{n+1} \to 0$
- 但不是柯西序列（事实上发散到$+\infty$）

<strong>对比</strong>：变式1的条件更强（几何级数衰减），保证了柯西性。
</details>

### 变式3（难度⭐⭐⭐⭐）

设 $K \subset \mathbb{R}$ 是紧集，$(f_n)$ 是 $C(K)$ 中的一致有界等度连续函数序列。证明对任意 $\varepsilon > 0$，存在子列 $(f_{n_k})$ 使得 $\|f_{n_k} - f_{n_j}\|_{\infty} < \varepsilon$ 对所有 $k, j$ 充分大成立。

<details>
<summary>提示</summary>
这是Arzelà-Ascoli定理的核心内容。关键步骤：

1. 取 $K$ 的有限 $\delta$-网 $\{x_1, \ldots, x_m\}$
2. 在每个 $x_i$ 处，$(f_n(x_i))$ 有界，用对角线法得子列在所有 $x_i$ 收敛
3. 由等度连续性，该子列在 $K$ 上一致柯西

</details>
