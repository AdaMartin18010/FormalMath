# 层与上同调

## 题目

**(a)** 定义预层、层，并构造层化 (sheafification)。

**(b)** 证明 Čech 上同调与导出函子上同调的关系。

**(c)** 计算 $\mathbb{P}^1$ 上全纯线丛的上同调群。

## 解答

### (a) 层化

对预层 $\mathcal{F}$，层化 $\mathcal{F}^+$ 的茎与 $\mathcal{F}$ 相同，但截面满足粘接条件。

### (b) Čech 上同调

对仿紧空间和好覆盖，$\check{H}^q(X, \mathcal{F}) \cong H^q(X, \mathcal{F})$。

### (c) $\mathbb{P}^1$ 上同调

由 Borel-Weil-Bott 定理或直接计算：$H^0(\mathbb{P}^1, \mathcal{O}(k)) = \mathbb{C}^{k+1}$ 对 $k \geq 0$，$H^1(\mathbb{P}^1, \mathcal{O}(-k-2)) = \mathbb{C}^{k+1}$。
