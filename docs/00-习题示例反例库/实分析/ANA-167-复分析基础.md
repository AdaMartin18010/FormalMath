# 复分析基础

## 题目

**(a)** 设 $f(z) = u(x,y) + iv(x,y)$ 在区域 $\Omega \subset \mathbb{C}$ 上连续可微。证明：$f$ 全纯当且仅当 $u, v$ 满足 Cauchy-Riemann 方程。

**(b)** 计算积分 $\displaystyle\oint_{|z|=1} \frac{e^z}{z^2+1} dz$。

**(c)** 证明最大模原理：若 $f$ 在区域 $\Omega$ 上全纯且非常数，则 $|f|$ 不能在 $\Omega$ 内部取得最大值。

## 解答

### (a) Cauchy-Riemann 方程

利用方向导数在水平和垂直方向相等，得到：
$$\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}, \quad \frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}$$

### (b) 围道积分

被积函数在 $z = \pm i$ 有奇点，只有 $z = i$ 在单位圆内（若 $|i|<1$ 不成立，需重新考虑）。实际上 $|i|=1$，应使用极限或变形围道。

$$\text{Res}_{z=i} \frac{e^z}{z^2+1} = \frac{e^i}{2i}$$

### (c) 最大模原理

利用均值性质：$f(z_0) = \frac{1}{2\pi} \int_0^{2\pi} f(z_0 + re^{i\theta}) d\theta$。
