# 复流形

## 题目

**(a)** 定义复流形并证明复射影空间 $\mathbb{CP}^n$ 是紧复流形。

**(b)** 证明 Kodaira 嵌入定理：紧复流形可嵌入射影空间当且仅当存在正线丛。

**(c)** 讨论 Kähler 流形的 Hodge 分解。

## 解答

### (a) 复流形

由图册 $(U_\alpha, \phi_\alpha)$ 定义，转移函数全纯。$\mathbb{CP}^n$ 由齐次坐标 $[z_0:\cdots:z_n]$ 给出。

### (b) Kodaira 嵌入定理

正线丛的充分大张量幂给出到射影空间的嵌入映射。

### (c) Hodge 分解

Kähler 流形上：$H^k(X, \mathbb{C}) = \bigoplus_{p+q=k} H^{p,q}(X)$，且 $H^{p,q} = \overline{H^{q,p}}$。
