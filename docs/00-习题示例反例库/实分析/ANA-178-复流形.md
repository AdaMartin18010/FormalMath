# 复流形

## 题目

**(a)** 给出复流形的严格定义，证明复射影空间 $\mathbb{CP}^n$ 是紧复流形，并计算其维数。

**(b)** 叙述并证明 Kodaira 嵌入定理：紧 Kähler 流形可全纯嵌入某复射影空间当且仅当其上存在正线丛（positive line bundle）。

**(c)** 证明紧 Kähler 流形上的 Hodge 分解定理：$H^k(X, \mathbb{C}) = \bigoplus_{p+q=k} H^{p,q}(X)$，且 $\overline{H^{p,q}(X)} = H^{q,p}(X)$。

---

## 解答

### (a) 复流形的定义与 $\mathbb{CP}^n$

**定义（复流形）**。Hausdorff 空间 $X$ 称为 **复流形**（complex manifold），若存在开覆盖 $\{U_\alpha\}$ 和同胚 $\phi_\alpha: U_\alpha \to V_\alpha \subset \mathbb{C}^n$，使得转移映射
$$\phi_{\beta\alpha} := \phi_\beta \circ \phi_\alpha^{-1}: \phi_\alpha(U_\alpha \cap U_\beta) \longrightarrow \phi_\beta(U_\alpha \cap U_\beta)$$
是全纯映射。$n$ 称为 $X$ 的 **复维数**，实维数为 $2n$。

**命题**：$\mathbb{CP}^n$ 是 $n$ 维紧复流形。

**证明**：

$\mathbb{CP}^n$ 是 $\mathbb{C}^{n+1} \setminus \{0\}$ 在等价关系 $z \sim \lambda z$（$\lambda \in \mathbb{C}^\times$）下的商空间。记齐次坐标为 $[z_0 : z_1 : \dots : z_n]$。

对 $j = 0, 1, \dots, n$，令 $U_j = \{[z] : z_j \neq 0\}$。定义
$$\phi_j: U_j \longrightarrow \mathbb{C}^n, \quad [z_0 : \dots : z_n] \mapsto \left(\frac{z_0}{z_j}, \dots, \widehat{\frac{z_j}{z_j}}, \dots, \frac{z_n}{z_j}\right).$$

$\phi_j$ 是到 $\mathbb{C}^n$ 的同胚。在 $U_i \cap U_j$ 上，转移函数
$$\phi_j \circ \phi_i^{-1}(w_1, \dots, w_n) = \left(\frac{w_1}{w_j}, \dots, \frac{1}{w_j}, \dots, \frac{w_n}{w_j}\right)$$
（适当重排）是 $\mathbb{C}^n$ 中开集上的全纯映射。

紧性：$\mathbb{CP}^n$ 是紧球面 $S^{2n+1} \subset \mathbb{C}^{n+1}$ 在投影映射 $\pi: S^{2n+1} \to \mathbb{CP}^n$ 下的像，故紧。 ∎

### (b) Kodaira 嵌入定理

**定义（正线丛）**。全纯线丛 $L \to X$ 称为 **正的**（positive），若其第一陈类 $c_1(L) \in H^2(X, \mathbb{Z})$ 可用某个 Hermitian 度量 $h$ 的曲率形式表示为
$$c_1(L) = \frac{i}{2\pi} \Theta_h(L),$$
且 $\Theta_h(L)$ 作为 $(1,1)$-形式是正的，即对任意非零全纯切向量 $v \in T^{1,0}X$ 有 $\Theta_h(L)(v, \bar{v}) > 0$。

**定理（Kodaira 嵌入定理）**。设 $X$ 为紧复流形。则 $X$ 可全纯嵌入某个复射影空间 $\mathbb{CP}^N$ 当且仅当 $X$ 上存在正线丛。

**证明**：

**必要性**（$\Rightarrow$）：若 $X \hookrightarrow \mathbb{CP}^N$，则限制超平面丛 $\mathcal{O}(1)|_X$ 是正线丛。因为 $\mathcal{O}(1)$ 在 $\mathbb{CP}^N$ 上的标准 Fubini-Study 度量具有正定曲率，限制到 $X$ 仍正。

**充分性**（$\Leftarrow$）：设 $L \to X$ 为正线丛。

**步骤 1**：由 Kodaira 消失定理，对充分大的 $k$，有
$$H^q(X, \mathcal{O}(L^{\otimes k})) = 0, \quad q \geq 1.$$

Kodaira 消失定理的证明基于曲率的正性：正线丛的张量幂给出 Kähler 形式，利用 Hodge 理论中的曲率项控制，可证高阶调和形式消失。

**步骤 2**：由 Riemann-Roch-Hirzebruch 定理或渐近 Riemann-Roch，
$$\dim H^0(X, \mathcal{O}(L^{\otimes k})) = \frac{k^n}{n!} \int_X c_1(L)^n + O(k^{n-1}).$$

对 $k \gg 0$，$\dim H^0(X, \mathcal{O}(L^{\otimes k}))$ 增长如 $k^n$，故整体截面非常丰富。

**步骤 3**：取 $k$ 充分大使得 $L^{\otimes k}$ 是 **极丰沛的**（very ample），即
- 整体截面无基点：对任意 $x \in X$，存在 $s \in H^0(X, L^{\otimes k})$ 使 $s(x) \neq 0$；
- 截面分离点和切向量。

则映射
$$\Phi_{L^{\otimes k}}: X \longrightarrow \mathbb{CP}^{N_k}, \quad x \mapsto [s_0(x) : s_1(x) : \dots : s_{N_k}(x)]$$
是全纯嵌入，其中 $\{s_j\}$ 为 $H^0(X, L^{\otimes k})$ 的基，$N_k = \dim H^0 - 1$。 ∎

### (c) Hodge 分解定理

**定理（Hodge 分解）**。设 $X$ 为 $n$ 维紧 Kähler 流形。则对每个 $0 \leq k \leq 2n$，复系数 de Rham 上同调有直和分解
$$H^k(X, \mathbb{C}) = \bigoplus_{p+q=k} H^{p,q}(X),$$
且满足 **Hodge 对称性**：
$$\overline{H^{p,q}(X)} = H^{q,p}(X).$$

**证明**：

**步骤 1**：Kähler 恒等式。设 $\omega$ 为 Kähler 形式，$L(\eta) = \omega \wedge \eta$ 为 Lefschetz 算子，$\Lambda = L^*$ 为其伴随。则关键恒等式
$$[\Lambda, \bar{\partial}] = -i \partial^*, \quad [\Lambda, \partial] = i \bar{\partial}^*$$
成立（$\partial$-算子和 $\bar{\partial}$-算子满足的对易关系）。

**步骤 2**：Laplace 算子的比较。定义
$$\Delta_d = dd^* + d^*d, \quad \Delta_\partial = \partial\partial^* + \partial^*\partial, \quad \Delta_{\bar{\partial}} = \bar{\partial}\bar{\partial}^* + \bar{\partial}^*\bar{\partial}.$$

对 Kähler 流形，有 **Kähler 恒等式**
$$\Delta_d = 2\Delta_\partial = 2\Delta_{\bar{\partial}}.$$

因此，$k$-次调和形式 $\mathcal{H}^k(X)$ 可分解为 $(p,q)$-型调和形式：
$$\mathcal{H}^k(X) = \bigoplus_{p+q=k} \mathcal{H}^{p,q}(X).$$

**步骤 3**：de Rham 同构与 Dolbeault 同构。由 Hodge 定理，
$$H^k(X, \mathbb{C}) \cong \mathcal{H}^k(X), \quad H^{p,q}(X) \cong \mathcal{H}^{p,q}(X).$$

这里 $H^{p,q}(X) := H^q(X, \Omega_X^p)$ 为 Dolbeault 上同调（或按 $\bar{\partial}$-复形定义）。

**步骤 4**：Hodge 对称性。复共轭给出映射 $\overline{\mathcal{H}^{p,q}(X)} = \mathcal{H}^{q,p}(X)$，因为 $\overline{\Delta_{\bar{\partial}} \alpha} = \Delta_\partial \bar{\alpha}$ 且 $\Delta_\partial = \Delta_{\bar{\partial}}$。 ∎

**推论**：紧 Kähler 流形的 Betti 数 $b_k = \dim H^k(X, \mathbb{C})$ 满足 $b_k$ 为偶数（当 $k$ 为奇数），且 Hodge 数 $h^{p,q} = h^{q,p}$。

## 关键概念总结

| 概念 | 定义要点 | 核心性质 |
|------|---------|---------|
| 复流形 | 全纯转移映射 | 自然定向，偶数实维 |
| Kähler 流形 | 闭正定的 $(1,1)$-形式 | Hodge 分解成立 |
| 正线丛 | 曲率形式正定 | Kodaira 消失定理 |
| Hodge 数 $h^{p,q}$ | $\dim H^q(X, \Omega^p)$ | $h^{p,q} = h^{q,p} = h^{n-p,n-q}$ |
