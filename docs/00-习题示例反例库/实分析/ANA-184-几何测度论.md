# 几何测度论

## 题目

**(a)** 定义 Hausdorff 测度 $\mathcal{H}^s$ 并证明其基本性质。

**(b)** 证明 Federer 降维定理：可求长集的支撑维数不超过集本身的维数。

**(c)** 讨论极小曲面的正则性理论。

## 解答

### (a) Hausdorff 测度

$$\mathcal{H}^s(E) = \lim_{\delta \to 0} \inf\left\{\sum_{i} (\text{diam } U_i)^s : E \subset \bigcup_i U_i, \text{diam } U_i < \delta\right\}$$

### (b) Federer 降维

可求长集的切空间几乎处处存在且与测地凸性相关。

### (c) 极小曲面正则性

Simons 定理：$n$-维极小锥在 $n \leq 6$ 时平坦。Almgren 的 ${\varepsilon}$-正则性定理。
