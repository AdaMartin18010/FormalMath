# ANA-002: 单调收敛定理的应用

## 题目陈述

**定理（单调收敛定理）**: 设 $(a_n)$ 是实数序列。

- 若 $(a_n)$ 单调递增且有上界，则 $(a_n)$ 收敛于 $\sup\{a_n : n \in \mathbb{N}\}$
- 若 $(a_n)$ 单调递减且有下界，则 $(a_n)$ 收敛于 $\inf\{a_n : n \in \mathbb{N}\}$

**问题**: 设序列 $(a_n)$ 定义为：
$$a_1 = 1, \quad a_{n+1} = \sqrt{2 + a_n}, \quad n \geq 1$$

证明：

1. $(a_n)$ 单调递增
2. $(a_n)$ 有上界
3. 求 $\lim_{n \to \infty} a_n$

## 详细解答

### 第一步：证明有界性（数学归纳法）

**断言**: 对所有 $n \in \mathbb{N}$，有 $a_n < 2$。

**证明**:

- **基础步骤**: $a_1 = 1 < 2$，成立。
- **归纳假设**: 假设 $a_k < 2$ 对某个 $k \geq 1$ 成立。
- **归纳步骤**:
  $$a_{k+1} = \sqrt{2 + a_k} < \sqrt{2 + 2} = \sqrt{4} = 2$$

由数学归纳法，对所有 $n \in \mathbb{N}$，$a_n < 2$。 $\square$

### 第二步：证明单调性

**断言**: 对所有 $n \in \mathbb{N}$，有 $a_{n+1} > a_n$。

**证明**: 我们需要证明 $\sqrt{2 + a_n} > a_n$，即 $2 + a_n > a_n^2$，即 $a_n^2 - a_n - 2 < 0$。

因式分解：$(a_n - 2)(a_n + 1) < 0$

由于 $a_n > 0$（显然）且 $a_n < 2$（已证），所以 $a_n - 2 < 0$ 且 $a_n + 1 > 0$。

因此 $(a_n - 2)(a_n + 1) < 0$，即 $a_{n+1} > a_n$。 $\square$

### 第三步：应用单调收敛定理求极限

由前两步，$(a_n)$ 单调递增且有上界，根据单调收敛定理，$(a_n)$ 收敛。

设 $L = \lim_{n \to \infty} a_n$。在递推公式两边取极限：

$$L = \sqrt{2 + L}$$

两边平方：$L^2 = 2 + L$，即 $L^2 - L - 2 = 0$

因式分解：$(L - 2)(L + 1) = 0$

解得 $L = 2$ 或 $L = -1$。

由于 $a_n > 0$，由极限保号性，$L \geq 0$，故 $L = 2$。

**结论**: $\displaystyle\lim_{n \to \infty} a_n = 2$ $\square$

## 证明技巧总结

### 核心技巧

1. **递推序列的极限求法**
   - 假设极限存在，设为 $L$
   - 在递推公式两边取极限
   - 解关于 $L$ 的方程
   - 利用序列性质排除不合理解

2. **单调性的证明方法**
   - 直接比较 $a_{n+1}$ 与 $a_n$
   - 分析函数 $f(x) = \sqrt{2+x}$ 的单调性
   - 数学归纳法

3. **有界性的证明方法**
   - 猜测上界（可通过计算前几项）
   - 数学归纳法验证

### 关键观察

- 递推函数 $f(x) = \sqrt{2+x}$ 的不动点决定了极限值
- 单调递增序列的极限就是其上确界

## 常见错误分析

### 错误1：未证明极限存在就求极限

```
❌ 错误做法：直接假设极限存在，解方程得到 L = 2

⚠️ 危险：如果序列发散，"极限值"毫无意义

✅ 正确做法：先证有界性和单调性，确保极限存在
```

### 错误2：未排除不合理的极限值

```
❌ 错误做法：得到 L = 2 或 L = -1 后不做选择

✅ 正确做法：利用 a_n > 0 排除 L = -1
```

### 错误3：有界性猜测错误

```
❌ 错误做法：猜测上界为 3（虽然正确但不够精确）

💡 建议：通过解不动点方程预判极限值，以此作为界
```

### 错误4：单调性证明不完整

```
❌ 错误做法：仅验证前几项就断言单调性

✅ 正确做法：给出严格的归纳证明或函数分析
```

## 变式练习

### 变式1（难度⭐⭐）

设 $a_1 = 2$，$a_{n+1} = \frac{1}{2}(a_n + \frac{2}{a_n})$。证明 $(a_n)$ 收敛并求极限。

<details>
<summary>提示</summary>
这是著名的巴比伦方法（求平方根的牛顿迭代法）。证明：
1. $a_n > \sqrt{2}$ 对所有 $n \geq 2$ 成立
2. $a_{n+1} < a_n$ 对 $n \geq 2$ 成立
3. 极限为 $\sqrt{2}$
</details>

### 变式2（难度⭐⭐⭐）

设 $a_1 = 1$，$a_{n+1} = 1 + \frac{1}{a_n}$。证明 $(a_n)$ 收敛并求极限。

<details>
<summary>提示</summary>
注意此序列不是单调的！考虑子序列 $(a_{2n})$ 和 $(a_{2n+1})$：
- 偶数项子序列单调递减有下界
- 奇数项子序列单调递增有上界
- 两子序列收敛于同一极限 $\frac{1+\sqrt{5}}{2}$
</details>

### 变式3（难度⭐⭐⭐⭐）

设 $a_1 = 1$，$a_{n+1} = \sin(a_n)$。研究 $(a_n)$ 的收敛性。

<details>
<summary>提示</summary>
1. 证明 $a_n \in (0, 1]$ 对所有 $n$ 成立
2. 利用不等式 $\sin x < x$（$x > 0$）证明序列单调递减
3. 极限为 0（唯一的不动点）
4. 进一步研究收敛速度：$a_{n+1} \approx a_n - \frac{a_n^3}{6}$
</details>
