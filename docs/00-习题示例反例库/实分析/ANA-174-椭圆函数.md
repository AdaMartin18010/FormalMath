# 椭圆函数

## 题目

**(a)** 定义 Weierstrass $\wp$ 函数并证明它是双周期的。

**(b)** 证明椭圆函数在周期平行四边形内的留数之和为零。

**(c)** 推导 $\wp$ 函数满足的微分方程：$\wp'^2 = 4\wp^3 - g_2\wp - g_3$。

## 解答

### (a) Weierstrass $\wp$ 函数

$$\wp(z; \omega_1, \omega_2) = \frac{1}{z^2} + \sum_{(m,n)\neq(0,0)} \left[\frac{1}{(z+m\omega_1+n\omega_2)^2} - \frac{1}{(m\omega_1+n\omega_2)^2}\right]$$

双周期性：$\wp(z+\omega_1) = \wp(z+\omega_2) = \wp(z)$。

### (b) 留数定理

周期函数沿周期平行四边形边界积分为零。

### (c) 微分方程

比较 $\wp$ 和 $\wp'$ 在原点附近的 Laurent 展开。
