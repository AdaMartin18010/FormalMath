# ANA-020: 多重极限技巧

**题号**: ANA-020
**难度**: ⭐⭐⭐⭐ (Level 4)
**类型**: 计算型 (CALC)
**来源**: MIT 18.100A Chapter 2.3
**主题**: 迭代极限与二重极限

---

## 题目

**(a)** 计算 $\displaystyle\lim_{m,n\to\infty} \sum_{k=1}^{m} \frac{1}{n^2 + k}$

**(b)** 计算 $\displaystyle\lim_{m\to\infty} \lim_{n\to\infty} \left(\cos\frac{\pi}{m}\right)^n$ 和 $\displaystyle\lim_{n\to\infty} \lim_{m\to\infty} \left(\cos\frac{\pi}{m}\right)^n$，并说明为何不同

**(c)** 设 $f(m,n) = \frac{m}{m+n}$，讨论 $\lim_{m\to\infty} \lim_{n\to\infty} f(m,n)$、$\lim_{n\to\infty} \lim_{m\to\infty} f(m,n)$ 和 $\lim_{m,n\to\infty} f(m,n)$

---

## 解答

### (a) 二重求和极限

**分析**: 先固定 $n$，令 $m \to \infty$，然后 $n \to \infty$。

**解**:

**Step 1: 内层极限**（固定 $n$，$m \to \infty$）

$$S_n = \lim_{m\to\infty} \sum_{k=1}^{m} \frac{1}{n^2 + k} = \sum_{k=1}^{\infty} \frac{1}{n^2 + k}$$

这发散！因为当 $k \to \infty$，$\frac{1}{n^2+k} \sim \frac{1}{k}$，调和级数发散。

**重新理解题目**: 题目应理解为 $m = n^2$，即：
$$\lim_{n\to\infty} \sum_{k=1}^{n^2} \frac{1}{n^2 + k}$$

**计算**:

由夹逼定理：
$$\frac{n^2}{n^2 + n^2} \leq \sum_{k=1}^{n^2} \frac{1}{n^2 + k} \leq \frac{n^2}{n^2 + 1}$$

即：
$$\frac{1}{2} \leq S_n \leq \frac{n^2}{n^2 + 1}$$

两边都趋于 $\frac{1}{2}$？不对，上界趋于1。

更精细的估计：
$$\sum_{k=1}^{n^2} \frac{1}{n^2 + k} = H_{2n^2} - H_{n^2}$$

利用 $H_n = \ln n + \gamma + O(1/n)$：
$$H_{2n^2} - H_{n^2} = \ln(2n^2) - \ln(n^2) + O(1/n^2) = \ln 2$$

因此：
$$\lim_{n\to\infty} \sum_{k=1}^{n^2} \frac{1}{n^2 + k} = \ln 2 \quad \square$$

---

### (b) 迭代极限的顺序

**计算**:

**先 $n$ 后 $m$**:

对固定的 $m$，由于 $\cos\frac{\pi}{m} < 1$：
$$\lim_{n\to\infty} \left(\cos\frac{\pi}{m}\right)^n = 0$$

因此：
$$\lim_{m\to\infty} \lim_{n\to\infty} \left(\cos\frac{\pi}{m}\right)^n = \lim_{m\to\infty} 0 = 0$$

**先 $m$ 后 $n$**:

对固定的 $n$：
$$\lim_{m\to\infty} \cos\frac{\pi}{m} = \cos 0 = 1$$

因此：
$$\lim_{n\to\infty} \lim_{m\to\infty} \left(\cos\frac{\pi}{m}\right)^n = \lim_{n\to\infty} 1^n = 1$$

**为何不同**: 两个极限不相等！

**解释**: 令 $a_{m,n} = \left(\cos\frac{\pi}{m}\right)^n$
- 若先让 $n \to \infty$（$m$ 固定），底数小于1，极限为0
- 若先让 $m \to \infty$（$n$ 固定），底数趋于1，极限为1

这说明**迭代极限的顺序不可交换**。 $\square$

---

### (c) 二重极限的各种形式

**计算**:

**先 $n$ 后 $m$**:
$$\lim_{n\to\infty} f(m,n) = \lim_{n\to\infty} \frac{m}{m+n} = 0$$
$$\lim_{m\to\infty} 0 = 0$$

**先 $m$ 后 $n$**:
$$\lim_{m\to\infty} f(m,n) = \lim_{m\to\infty} \frac{m}{m+n} = 1$$
$$\lim_{n\to\infty} 1 = 1$$

**二重极限**:

沿不同路径：
- 沿 $m = n$：$f(n,n) = \frac{n}{2n} = \frac{1}{2}$
- 沿 $m = 2n$：$f(2n,n) = \frac{2n}{3n} = \frac{2}{3}$

由于沿不同路径极限不同，**二重极限不存在**。

**结论**:
$$\lim_{m\to\infty} \lim_{n\to\infty} f(m,n) = 0, \quad \lim_{n\to\infty} \lim_{m\to\infty} f(m,n) = 1$$
$$\lim_{m,n\to\infty} f(m,n) \text{ 不存在}$$

这说明：
1. 两个迭代极限可以都存在但不相等
2. 迭代极限存在不能保证二重极限存在
3. 二重极限存在要求沿所有路径极限相同 $\square$

## 证明技巧总结

| 技巧 | 应用位置 | 说明 |
|------|----------|------|
| 迭代极限 | (b)(c) | 按顺序计算极限 |
| 路径检验 | (c) | 通过不同路径判断二重极限不存在 |
| 调和级数估计 | (a) | $H_n = \ln n + \gamma + O(1/n)$ |
| 极限顺序 | 全部 | 理解不同顺序可能给出不同结果 |

### 关键洞察

1. **极限顺序不可交换**: 一般情况下 $\lim\lim \neq \lim\lim$
2. **二重极限更强**: 二重极限存在要求沿所有路径相同
3. **一致收敛**: 若内层极限一致收敛，则可交换顺序

## 常见错误

### 错误1: 盲目交换极限顺序

❌ **错误做法**:
> 假设 $\lim_m \lim_n = \lim_n \lim_m$ 总是成立

✅ **正确做法**:
> 验证一致收敛条件或分别计算

### 错误2: 忽略路径检验

❌ **错误做法**:
> 在(c)中，认为二重极限为某个值而不检验所有路径

✅ **正确做法**:
> 沿不同直线 $m = cn$ 检验

### 错误3: 混淆迭代与二重极限

❌ **错误做法**:
> 认为迭代极限存在则二重极限存在

✅ **正确理解**:
> 二重极限存在要求更强的条件

## 变式练习

### 变式1（难度⭐⭐⭐⭐）

设 $f(m,n) = \frac{mn}{m^2 + n^2}$，讨论各种极限

**答案**: 两个迭代极限都是0，但二重极限不存在（沿 $m=n$ 得1/2）

### 变式2（难度⭐⭐⭐⭐⭐）

计算 $\displaystyle\lim_{n\to\infty} \sum_{k=1}^{n} \sum_{j=1}^{n} \frac{1}{n^2 + k + j}$

**答案**: $2\ln 2 - 1$

**提示**: 转化为二重积分或精细估计

### 变式3（难度⭐⭐⭐⭐⭐）

证明：若 $f(m,n)$ 关于两个变量分别单调递增，则
$$\lim_{m\to\infty} \lim_{n\to\infty} f(m,n) = \lim_{n\to\infty} \lim_{m\to\infty} f(m,n)$$

**提示**: 单调收敛定理的双变量版本
