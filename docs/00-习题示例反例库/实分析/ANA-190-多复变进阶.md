# 多复变进阶

## 题目

**(a)** 证明强凸域的边界球化定理（Cheng-Yau）。

**(b)** 讨论 Bergman 核的渐近展开（Fefferman）。

**(c)** 证明伪凸域上 $\bar{\partial}$-Neumann 问题的次椭圆估计。

## 解答

### (a) 边界球化定理

**设置**：设 $\Omega \subset \mathbb{C}^n$ 是有界强拟凸域，带 $C^\infty$ 边界。强拟凸性意味着边界定义函数 $\rho$（满足 $\Omega = \{z : \rho(z) < 0\}$，$\nabla \rho \neq 0$ on $\partial\Omega$）的复 Hessian（Levi 形式）在复切空间上正定：
$$\sum_{j,k=1}^n \frac{\partial^2\rho}{\partial z_j \partial \bar{z}_k}(p) w_j \bar{w}_k > 0, \quad \forall p \in \partial\Omega, \forall w \in T_p^{\mathbb{C}}(\partial\Omega), w \neq 0$$

**Kähler-Einstein 度规的存在性**：

**定理（Cheng-Yau, 1980）**：对任何有界强拟凸域 $\Omega \subset \mathbb{C}^n$，存在唯一的（至多差常数）完备 Kähler-Einstein 度规 $\omega_{KE}$，满足：
$$\text{Ric}(\omega_{KE}) = -(n+1)\omega_{KE}$$

*证明概要*：Cheng-Yau 将 Calabi 的复 Monge-Ampère 边值问题方法推广到非紧边界情形。考虑 Dirichlet 问题：
$$\det(g_{j\bar{k}} + \varphi_{j\bar{k}}) = e^{F}\det(g_{j\bar{k}})$$

在完备度量上利用极值原理和 a priori 估计（$C^0, C^1, C^2$ 估计），证明解的存在性。强拟凸性保证边界附近的正则性。

**边界渐近（Fefferman, Cheng-Yau）**：

在边界附近，若 $\rho$ 是定义函数，则 Kähler-Einstein 度规有渐近展开：
$$\omega_{KE} = -\partial\bar{\partial}\log(-\rho) + \text{(光滑到边界的项)}$$

更精确地，取 Fefferman 的近似解：
$$\varphi = -\log(-\rho) + \psi$$
其中 $\psi$ 满足 $J(\rho) = 1$ 到高阶（$J$ 是 Monge-Ampère 算子）。则：
$$\omega_{KE} = \frac{i}{2\pi}\left(\frac{\partial\bar{\partial}\rho}{-\rho} + \frac{\partial\rho \wedge \bar{\partial}\rho}{\rho^2} + \partial\bar{\partial}\psi\right)$$

**球化定理**：设 $p \in \partial\Omega$。在 $p$ 的切空间上，考虑伸缩（blow-up）序列：对 $\epsilon > 0$，定义 dilation $D_\epsilon(z) = p + \epsilon^{-1}(z - p)$（在适当局部坐标下）。则缩放后的域 $\Omega_\epsilon = D_\epsilon(\Omega)$ 收敛到模型域：
$$\Omega_p = \{w \in \mathbb{C}^n : \text{Im}(w_n) > |w'|^2\}$$
（Siegel 上半空间，双全纯等价于球 $\mathbb{B}^n$）。

在此缩放极限下，Kähler-Einstein 度规 $\omega_{KE}$ 的 rescaled 版本收敛到复双曲空间 $\mathbb{B}^n$ 的 Bergman 度规（曲率为 $-1$ 的常全纯截面曲率度规）。这就是**边界球化**（boundary spherification）的精确含义。

*证明要点*：局部坐标下将边界展开到二阶：$\rho(z) = \text{Re}(z_n) - |z'|^2 + O(|z|^3)$。在 blow-up 下高阶项消失，Levi 形式的主项给出 Siegel 域。Kähler-Einstein 方程的尺度不变性保证极限度规满足同样的方程，由唯一性即得 Bergman 度规。

### (b) Bergman 核的渐近展开

**Bergman 核的定义**：设 $\Omega \subset \mathbb{C}^n$ 是有界域。Bergman 空间 $A^2(\Omega) = L^2(\Omega) \cap \mathcal{O}(\Omega)$ 是平方可积全纯函数空间。Bergman 核 $K(z, w)$ 是 $A^2(\Omega)$ 的再生核：
$$f(z) = \int_\Omega K(z, w) f(w) dV_w, \quad \forall f \in A^2(\Omega)$$

由 Riesz 表示定理，$K(z, w) = \sum_j \phi_j(z)\overline{\phi_j(w)}$，其中 $\{\phi_j\}$ 是 $A^2(\Omega)$ 的标准正交基。

**Fefferman 展开定理（1974）**：设 $\Omega \subset \mathbb{C}^n$ 是有界强拟凸域，$\partial\Omega \in C^\infty$。则 Bergman 核在对角线上有渐近展开：
$$K(z, z) = \frac{\phi(z)}{\rho(z)^{n+1}} + \psi(z) \log(-\rho(z))$$

其中 $\phi, \psi \in C^\infty(\overline{\Omega})$，且 $\phi|_{\partial\Omega} > 0$。

**证明的核心步骤**：

**步骤 1：奇异性分析**。对球 $\mathbb{B}^n$，直接计算 Bergman 核：
$$K_{\mathbb{B}^n}(z, w) = \frac{n!}{\pi^n}\frac{1}{(1 - z \cdot \bar{w})^{n+1}}$$
故 $K(z,z) = \frac{n!}{\pi^n}\frac{1}{(1-|z|^2)^{n+1}}$，在边界有 $(n+1)$-阶极点。对一般强拟凸域，边界局部双全纯等价于球的扰动，故预期同样的奇异性阶。

**步骤 2：复 Monge-Ampère 方程**。Fefferman 证明了 $K(z,z)^{-1/(n+1)}$ 近似满足复 Monge-Ampère 方程。设 $u = K(z,z)^{-1/(n+1)}$，则 $u$ 在边界附近近似满足 $J(u) = 1$，其中 $J(u) = (-1)^n \det\begin{pmatrix} u & u_{\bar{j}} \\ u_i & u_{i\bar{j}} \end{pmatrix}$ 是 Fefferman 的 Monge-Ampère 算子。

**步骤 3：对数项的不变性**。Fefferman 证明若对数项 $\psi$ 在某边界点为零，则通过所有边界点的测地流上恒为零。这推出对数项是**边界全纯不变量**：若两个强拟凸域双全纯等价，则其对数项在对应边界点一致。

**应用**：
- **边界不变量的计算**：对数项系数给出强拟凸域的局部双全纯不变量，可用于区分不等价的域。
- **Bergman 度规的完备性**：由渐近展开，$K(z,z) \to \infty$ 当 $z \to \partial\Omega$，故 Bergman 度规 $ds^2 = \sum \frac{\partial^2 \log K}{\partial z_j \partial \bar{z}_k} dz_j d\bar{z}_k$ 在边界处完备。

### (c) $\bar{\partial}$-Neumann 问题的次椭圆估计

**问题设置**：设 $\Omega \subset \mathbb{C}^n$ 是有界强拟凸域。对 $(0,q)$-形式 $u$，考虑 **$\bar{\partial}$-Neumann 问题**：求 $N u$ 使得
$$\Box N u = u \text{ on } \Omega, \quad N u \in \text{Dom}(\bar{\partial}^*)$$
其中 $\Box = \bar{\partial}\bar{\partial}^* + \bar{\partial}^*\bar{\partial}$ 是复 Laplacian（Kohn Laplacian），$\bar{\partial}^*$ 是 $\bar{\partial}$ 的 $L^2$-伴随。

**Kohn-Morrey 公式**：对 $u \in \text{Dom}(\bar{\partial}^*) \cap C^1_{(0,q)}(\overline{\Omega})$：
$$Q(u, u) := \|\bar{\partial}u\|^2 + \|\bar{\partial}^*u\|^2 = \sum_{|I|=q} \sum_{j,k} \int_\Omega \frac{\partial^2\rho}{\partial z_j \partial \bar{z}_k} u_{jI}\overline{u_{kI}} + \text{(低阶项 + 边界项)}$$

其中 $u = \sum_{|I|=q} u_I d\bar{z}_I$，$u_{jI}$ 是经适当反对称化后的分量。

**定理（Kohn, 1963；次椭圆估计）**：设 $\Omega$ 是强拟凸域。则存在 $\varepsilon > 0$（实际上对强拟凸域 $\varepsilon = 1/2$）和常数 $C > 0$，使得对所有 $u \in \text{Dom}(\bar{\partial}) \cap \text{Dom}(\bar{\partial}^*)$：
$$\|u\|_\varepsilon^2 \leq C(\|\bar{\partial}u\|^2 + \|\bar{\partial}^*u\|^2 + \|u\|^2)$$

其中 $\|\cdot\|_\varepsilon$ 是 Sobolev $\varepsilon$-范数。

**证明要点**：

**步骤 1：强拟凸性的作用**。因 Levi 形式在复切空间上正定，存在 $c > 0$ 使边界项满足：
$$\sum_{j,k} \int_{\partial\Omega} \frac{\partial^2\rho}{\partial z_j \partial \bar{z}_k} u_{jI}\overline{u_{kI}} \geq c \int_{\partial\Omega} |u|^2$$

（对 $u \in \text{Dom}(\bar{\partial}^*)$，法向分量为零，故 $u$ 位于复切空间。）

**步骤 2：Friedrichs 引理与 tangential 微分**。利用 Fourier 变换在边界法向方向的分解，将估计分为切向和法向。切向微分由边界项控制（步骤 1），法向微分由 $\bar{\partial}$ 和 $\bar{\partial}^*$ 的结构控制。

**步骤 3：Ehrling 不等式与插值**。由内部椭圆估计（$\bar{\partial} \oplus \bar{\partial}^*$ 是椭圆的），有：
$$\|u\|_1^2 \leq C'(\|\bar{\partial}u\|^2 + \|\bar{\partial}^*u\|^2 + \|u\|^2) + \text{(边界误差)}$$

结合边界控制与插值理论（$\|u\|_{1/2}^2 \leq \delta\|u\|_1^2 + C_\delta\|u\|^2$），选取 $\delta$ 小使得吸收法成立，最终得到 $\varepsilon = 1/2$ 估计。

**正则性推论**：次椭圆估计推出：若 $\Box u = f \in W^s(\Omega)$，则 $u \in W^{s+\varepsilon}(\Omega)$。由 bootstrap 论证，$f \in C^\infty \Rightarrow u \in C^\infty$（在强拟凸域上）。

**有限型（Finite Type）与弱拟凸域**：

Catlin（1987）将次椭圆估计推广到**有限型**域：$p \in \partial\Omega$ 的型 $m$ 是复解析曲线与边界接触的最大阶数。对有限型 $m$ 的域，次椭圆估计中的 $\varepsilon = 1/(2m)$。

**常见错误**：
- **强拟凸 vs 强凸**：强拟凸仅要求 Levi 形式在复切空间上正定；强凸要求实 Hessian 在所有切方向上正定。强凸 $\Rightarrow$ 强拟凸，反之不成立（如 $\{z : |z_1|^2 + |z_2|^4 < 1\}$ 是强拟凸但非强凸）。
- **Bergman 核的对数项**：对一般强拟凸域对数项非零。仅当域双全纯等价于球时（由 Fefferman 定理）对数项才消失。
- **次椭圆估计的指数**：强拟凸时 $\varepsilon = 1/2$，但在弱拟凸域（如有限型 $m > 2$）中 $\varepsilon$ 更小，且正则性 bootstrap 步数增加。
