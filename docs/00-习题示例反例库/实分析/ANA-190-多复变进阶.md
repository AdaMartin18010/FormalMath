# 多复变进阶

## 题目

**(a)** 证明强凸域的边界球化定理（Cheng-Yau）。

**(b)** 讨论 Bergman 核的渐近展开（Fefferman）。

**(c)** 证明伪凸域上 ∂̄-Neumann 问题的次椭圆估计。

## 解答

### (a) 边界球化

强凸域的完备 Einstein-Kähler 度规在边界附近有渐近展开，边界切空间球化。

### (b) Bergman 核渐近

$$K(z,z) = \frac{\phi(z)}{\rho(z)^{n+1}} + \psi(z) \log \rho(z)$$
其中 ρ 是边界定义函数。

### (c) 次椭圆估计

Kohn 估计：在强伪凸域上，∂̄-Neumann 算子 N 满足次椭圆估计，增益 1/2 导数。
