# 算子半群理论

## 题目

设 $X$ 是Banach空间，$\{T(t)\}_{t \geq 0} \subset \mathcal{B}(X)$ 是**强连续半群**：
- $T(0) = I$
- $T(t+s) = T(t)T(s)$
- $\lim_{t \to 0^+} T(t)x = x$（强收敛）

定义生成元：$Ax = \lim_{t \to 0^+} \frac{T(t)x - x}{t}$，$D(A) = \{x : \text{极限存在}\}$。

**(a)** 证明：存在 $\omega, M$ 使 $\|T(t)\| \leq Me^{\omega t}$。

**(b)** 证明：$x \in D(A) \Rightarrow T(t)x \in D(A)$ 且 $\frac{d}{dt}T(t)x = AT(t)x = T(t)Ax$。

**(c)** 证明Hille-Yosida定理：闭稠定线性算子 $A$ 生成收缩半群当且仅当：
- $(0, \infty) \subset \rho(A)$
- $\|R(\lambda, A)\| \leq \frac{1}{\lambda}$ 对 $\lambda > 0$

## 解答

### (a) 半群增长界

**证明：**

由一致有界原理，$\sup_{0 \leq t \leq 1} \|T(t)\| < \infty$。

否则存在 $t_n \to 0$，$\|T(t_n)\| \to \infty$，与强连续性矛盾。

设 $M = \sup_{[0,1]} \|T(t)\|$。

对 $t \geq 0$，写 $t = n + r$，$n \in \mathbb{N}$，$r \in [0,1)$。

$$\|T(t)\| = \|T(1)^n T(r)\| \leq M^{n+1} = M \cdot M^n$$

取 $\omega = \log M$，$\|T(t)\| \leq Me^{\omega t}$。$\square$

### (b) 生成元与微分

**证明：**

对 $x \in D(A)$：
$$\frac{T(t+h)x - T(t)x}{h} = T(t) \frac{T(h)x - x}{h} \to T(t)Ax$$

故右导数 $\frac{d^+}{dt}T(t)x = T(t)Ax$。

对 $h < 0$：
$$\frac{T(t+h)x - T(t)x}{h} = T(t+h) \frac{T(-h)x - x}{-h} \to T(t)Ax$$

因 $\|T(t+h) - T(t)\| \to 0$。

另一等式：$T(t)Ax = AT(t)x$ 由$A$定义。$\square$

### (c) Hille-Yosida定理

**证明概要：**

**($\Rightarrow$)**：设 $A$ 生成收缩半群 $T(t)$。

预解式：$R(\lambda, A)x = \int_0^\infty e^{-\lambda t} T(t)x dt$。

$$\|R(\lambda, A)x\| \leq \int_0^\infty e^{-\lambda t} \|x\| dt = \frac{\|x\|}{\lambda}$$

**($\Leftarrow$)**：构造Yosida逼近：$A_\lambda = \lambda AR(\lambda, A) = \lambda^2 R(\lambda, A) - \lambda I$。

$A_\lambda$ 有界，生成 $T_\lambda(t) = e^{tA_\lambda}$。

证明 $\{T_\lambda(t)\}$ 是Cauchy列，极限 $T(t)$ 是所需半群。$\square$
