# 变分法进阶

## 题目

**(a)** 证明 mountain pass 定理及其在多解问题中的应用。

**(b)** 讨论 Γ-收敛和均质化问题。

**(c)** 证明极小曲面的正则性理论（Almgren, De Giorgi）。

## 解答

### (a) Mountain Pass 定理

Ambrosetti-Rabinowitz：泛函 J 满足山路几何，则存在临界点 c ≥ α > 0。

应用：超线性椭圆方程的多解。

### (b) Γ-收敛

De Giorgi 的收敛概念：保证极小化子的收敛。均质化：快速振荡系数极限。

### (c) 极小曲面正则性

De Giorgi 的递减振荡论证：维数 ≤ 7 时极小锥平坦，进而边界正则。
