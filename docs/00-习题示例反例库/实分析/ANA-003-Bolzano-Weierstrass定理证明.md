# ANA-003: Bolzano-Weierstrass定理证明

## 题目陈述

**Bolzano-Weierstrass定理**: 有界实数序列必有收敛子列。

**形式化表述**: 设 $(a_n)$ 是实数序列，若存在 $M > 0$ 使得 $|a_n| \leq M$ 对所有 $n \in \mathbb{N}$ 成立，则存在严格递增的指标序列 $n_1 < n_2 < n_3 < \cdots$ 使得子序列 $(a_{n_k})$ 收敛。

请给出该定理的完整证明。

## 详细解答

### 证法一：区间二分法（经典证明）

**核心思想**: 通过不断二分区间，构造一个区间套，利用区间套原理得到收敛子列。

**证明**:

**步骤1：初始设置**

设 $|a_n| \leq M$ 对所有 $n$ 成立。令 $I_0 = [-M, M]$。

**步骤2：二分构造**

将 $I_0$ 二等分为 $[-M, 0]$ 和 $[0, M]$。至少有一个区间包含 $(a_n)$ 的无穷多项，记该区间为 $I_1$。

归纳地，若 $I_{k-1} = [a, b]$ 已定义，将其二等分为 $[a, \frac{a+b}{2}]$ 和 $[rac{a+b}{2}, b]$。选择包含无穷多项的区间为 $I_k$。

**步骤3：区间套性质**

我们得到区间套：

- $I_0 \supseteq I_1 \supseteq I_2 \supseteq \cdots$
- $|I_k| = \frac{2M}{2^k} = \frac{M}{2^{k-1}} \to 0$ 当 $k \to \infty$

**步骤4：构造子序列**

- 在 $I_1$ 中选取 $a_{n_1}$
- 在 $I_2$ 中选取 $a_{n_2}$ 且 $n_2 > n_1$
- 依此类推，在 $I_k$ 中选取 $a_{n_k}$ 且 $n_k > n_{k-1}$

这样得到子序列 $(a_{n_k})$，其中 $a_{n_k} \in I_k$。

**步骤5：证明收敛**

由区间套定理，存在唯一的 $L \in \bigcap_{k=1}^{\infty} I_k$。

对任意 $\varepsilon > 0$，取 $K$ 使得 $|I_K| < \varepsilon$。则对所有 $k \geq K$：
$$|a_{n_k} - L| \leq |I_k| \leq |I_K| < \varepsilon$$

故 $a_{n_k} \to L$。 $\square$

---

### 证法二：单调子列法

**核心思想**: 任何序列都含有单调子列，再结合单调收敛定理。

**定义（峰值点）**: 称 $n$ 为峰值点，如果 $a_n \geq a_m$ 对所有 $m > n$ 成立。

**证明**:

**情形1：有无穷多个峰值点**

设峰值点为 $n_1 < n_2 < n_3 < \cdots$。

由峰值点定义：$a_{n_1} \geq a_{n_2} \geq a_{n_3} \geq \cdots$

即 $(a_{n_k})$ 是单调递减序列。又因 $(a_n)$ 有界，故 $(a_{n_k})$ 有下界。

由单调收敛定理，$(a_{n_k})$ 收敛。 $\checkmark$

**情形2：只有有限个峰值点**

设最大峰值点为 $N$。取 $n_1 = N + 1$。

由于 $n_1$ 不是峰值点，存在 $n_2 > n_1$ 使得 $a_{n_2} > a_{n_1}$。

由于 $n_2$ 不是峰值点，存在 $n_3 > n_2$ 使得 $a_{n_3} > a_{n_2}$。

依此类推，得到严格递增子序列 $a_{n_1} < a_{n_2} < a_{n_3} < \cdots$

由单调收敛定理，$(a_{n_k})$ 收敛。 $\checkmark$

两种情形必居其一，证毕。 $\square$

## 证明技巧总结

### 核心技巧

| 技巧 | 证法一（二分法） | 证法二（单调子列） |
|------|------------------|-------------------|
| **核心工具** | 区间套定理 | 单调收敛定理 |
| **关键构造** | 区间套 | 峰值点概念 |
| **优点** | 构造性，可计算 | 简洁优雅 |
| **缺点** | 较繁琐 | 非构造性 |

### 二分法的精髓

```
无限 → 有限: 将有界区间转化为长度趋于0的区间套
分散 → 集中: 每一步都保留"足够多"的项
```

### 峰值点法的精髓

```
全局单调 ← 局部单调: 通过峰值点提取天然单调子列
分类讨论: 穷尽所有可能情形
```

## 常见错误分析

### 错误1：子列构造不严格

```
❌ 错误："从每个区间中随便选一个"

✅ 正确：必须确保指标严格递增 $n_1 < n_2 < n_3 < \cdots$

⚠️ 关键点：每个区间有无穷多项，所以总能找到更大指标
```

### 错误2：混淆"有无穷多项"与"包含所有项"

```
❌ 错误：认为未被选中的区间不包含序列的任何项

✅ 正确：只要求至少一个区间有无穷多项，另一个可能有有限或无穷多项
```

### 错误3：峰值点定义错误

```
❌ 错误："$n$ 是峰值点如果 $a_n > a_{n+1}$"

✅ 正确："$n$ 是峰值点如果 $a_n \geq a_m$ 对所有 $m > n$"

💡 区别：后者要求 $a_n$ 是"尾部最大值"，而不仅仅是大于下一项
```

### 错误4：忽略有界性条件

```
❌ 错误：认为任意序列都有收敛子列

✅ 正确：反例：$a_n = n$ 无收敛子列

💡 有界性是本质条件，不可省略
```

## 变式练习

### 变式1（难度⭐⭐）

设 $(a_n)$ 是有界序列，$\lim_{n \to \infty} (a_{n+1} - a_n) = 0$。证明 $(a_n)$ 的极限点集是一个连通集（即区间）。

<details>
<summary>提示</summary>
利用Bolzano-Weierstrass定理证明：若 $L_1 < L_2$ 都是极限点，则 $[L_1, L_2]$ 中任意点也是极限点。

关键步骤：对任意 $c \in (L_1, L_2)$ 和 $\varepsilon > 0$，找到 $N$ 使得 $|a_{n+1} - a_n| < \varepsilon$ 对所有 $n \geq N$ 成立。然后利用 $a_{n_k} \to L_1$ 和 $a_{m_k} \to L_2$ 构造趋近于 $c$ 的子列。
</details>

### 变式2（难度⭐⭐⭐）

证明 $\mathbb{R}^n$ 中的有界序列必有收敛子列。

<details>
<summary>提示</summary>
对维度 $n$ 归纳：
- $n=1$：即原定理
- 归纳步骤：设 $(x_k) = (x_k^{(1)}, \ldots, x_k^{(n)})$，先对第一坐标用Bolzano-Weierstrass，再对剩余坐标用归纳假设。
</details>

### 变式3（难度⭐⭐⭐⭐）

称序列 $(a_n)$ 是**等度连续**的，如果对任意 $\varepsilon > 0$，存在 $\delta > 0$ 使得 $|a_n(x) - a_n(y)| < \varepsilon$ 对所有 $n$ 和 $|x-y| < \delta$ 成立。

设 $(f_n)$ 是 $[0,1]$ 上的一致有界且等度连续的函数序列。证明存在一致收敛的子列（Arzelà-Ascoli定理的序列版本）。

<details>
<summary>提示</summary>
1. 取 $[0,1]$ 的可数稠密子集 $\{q_1, q_2, \ldots\}$
2. 用对角线法：先在 $q_1$ 点用Bolzano-Weierstrass得子列，再在该子列中在 $q_2$ 点提取子列，依此类推
3. 证明对角线子列在所有有理点收敛
4. 利用等度连续性证明在 $[0,1]$ 上一致收敛
</details>
