# 偏微分方程进阶

## 题目

**(a)** 证明 De Giorgi-Nash 定理：散度型椭圆方程解的 Hölder 连续性。

**(b)** 证明 Krylov-Safonov 的 Harnack 不等式。

**(c)** 讨论黏性解理论及其在完全非线性方程中的应用。

## 解答

### (a) De Giorgi-Nash 定理

**方程**：散度型椭圆方程：

$$-\partial_i(a^{ij}(x)\partial_j u) = 0$$

其中 $\lambda |\xi|^2 \leq a^{ij}\xi_i\xi_j \leq \Lambda |\xi|^2$（一致椭圆）。

**定理（De Giorgi, 1957; Nash, 1958）**：弱解 $u \in H^1$ 是 Hölder 连续的：$u \in C^{0,\alpha}$ 对某 $\alpha > 0$。

*证明概要*（De Giorgi 方法）：

1. **Caccioppoli 估计**：对 $k \in \mathbb{R}$，令 $A(k, r) = \{x \in B_r : u(x) > k\}$。
   
   $$\int_{A(k,r)} |\nabla u|^2 \leq \frac{C}{(R-r)^2} \int_{A(k,R)} (u-k)^2$$

2. **迭代**：定义 $U(r) = \operatorname{osc}_{B_r} u = \sup_{B_r} u - \inf_{B_r} u$。

3. **De Giorgi 类**：证明 $u$ 属于 De Giorgi 类，满足：
   $$U(\rho) \leq C\left(\frac{\rho}{R}\right)^\alpha U(R)$$

4. **Hölder 连续性**：振荡衰减给出 Hölder 估计。

---

### (b) Krylov-Safonov Harnack 不等式

**方程**：非散度型一致椭圆方程：

$$a^{ij}(x)\partial_{ij} u = 0$$

**Harnack 不等式**：对 $u \geq 0$ 在 $B_R$ 中：

$$\sup_{B_{R/2}} u \leq C \inf_{B_{R/2}} u$$

*证明*（Aleksandrov-Bakelman-Pucci 估计 + 覆盖论证）：

1. **ABP 估计**：对 $Lu = a^{ij}\partial_{ij}u$，$u|_{\partial B_R} = 0$：
   $$\sup_{B_R} u \leq C R \|Lu\|_{L^n}$$

2. **弱 Harnack**：对 $u \geq 0$，$Lu = 0$：
   $$\|u\|_{L^p(B_{R/2})} \leq C \inf_{B_{R/2}} u$$

3. **局部有界性 + Harnack**：由迭代和覆盖定理得 Harnack 不等式。

---

### (c) 黏性解理论

**动机**：完全非线性方程（如 Monge-Ampère、Hamilton-Jacobi）的古典解可能不存在。

**定义（Crandall-Lions, 1983）**：$u$ 是 **黏性下解**，若对任意 $\varphi \in C^2$，$u - \varphi$ 在 $x_0$ 处取局部极大，则：

$$F(D^2\varphi(x_0), D\varphi(x_0), u(x_0), x_0) \leq 0$$

类似定义黏性上解。$u$ 是 **黏性解** 若同时是上下解。

**比较原理**：$u$ 下解，$v$ 上解，$u \leq v$ 在边界 $
$ \Rightarrow$ $u \leq v$ 在内部。

**应用**：

1. **Hamilton-Jacobi 方程**：$u_t + H(Du) = 0$，黏性解给出唯一的弱解。
2. **Monge-Ampère**：$\det(D^2u) = f$，Alexandrov 解 = 黏性解。
3. **Isaacs 方程**：随机微分博弈的值函数是黏性解。

---

## 常见错误

- **De Giorgi-Nash 的系数条件**：需 $a^{ij} \in L^\infty$，不连续系数也可能有 Hölder 解。
- **Krylov-Safonov 的散度型限制**：非散度型 Harnack 不等式比散度型更难，需 ABP 估计。
- **黏性解的唯一性**：比较原理需 $F$ 满足适当结构条件（如单调性、椭圆性）。
