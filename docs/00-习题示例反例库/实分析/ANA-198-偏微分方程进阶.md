# 偏微分方程进阶

## 题目

**(a)** 证明 De Giorgi-Nash 定理：散度型椭圆方程解的 Hölder 连续性。

**(b)** 证明 Krylov-Safonov 的 Harnack 不等式。

**(c)** 讨论黏性解理论及其在完全非线性方程中的应用。

## 解答

### (a) De Giorgi-Nash

通过 Caccioppoli 估计和迭代论证，证明振荡衰减：
$$\text{osc}_{B_r} u \leq C r^\alpha$$

### (b) Krylov-Safonov Harnack

非散度型方程的弱 Harnack 不等式，利用 Aleksandrov-Bakelman-Pucci 估计。

### (c) 黏性解

Crandall-Lions 理论：通过检验函数定义，适用于 Hamilton-Jacobi 和完全非线性方程。
