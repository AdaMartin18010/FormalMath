# 实分析综合挑战

## 题目

本题集涵盖Fourier分析、分布理论、算子理论与遍历理论的交叉内容。

**(a)** 设 $T$ 是 $\mathbb{R}^n$ 上的Calderón-Zygmund算子，证明 $T$ 可延拓为 $H^1(\mathbb{R}^n) \to H^1(\mathbb{R}^n)$ 和 $L^\infty(\mathbb{R}^n) \to BMO(\mathbb{R}^n)$ 的有界算子。

**(b)** 设 $P$ 是上半空间 $\mathbb{R}^{n+1}_+$ 的Poisson积分算子，证明非切向极大函数
$$P^*f(x) = \sup_{|x-y| < t} |P_t * f(y)|$$
满足 $\|P^*f\|_{L^p} \leq C_p \|f\|_{L^p}$（$1 < p \leq \infty$）。

**(c)** 设 $T$ 是保测遍历变换，$f \in L \log L$（Orlicz空间）。证明加权遍历定理：对a.e. $x$，
$$\lim_{n \to \infty} \frac{1}{n} \sum_{k=0}^{n-1} f(T^k x) = \int f d\mu$$

## 解答

### (a) C-Z算子的端点估计

**证明概要：**

**$H^1 \to H^1$有界性：**

只需证对任意原子 $a$，$Ta$ 是 $H^1$分子。

设 $a$ 支撑于 $Q$，$\|a\|_\infty \leq |Q|^{-1}$，$\int a = 0$。

**Step 1**：$L^2$估计。
$$\|Ta\|_{L^2} \leq C\|a\|_{L^2} \leq C|Q|^{-1/2}$$

**Step 2**：远离支撑估计。

对 $x \notin Q^* = 2\sqrt{n}Q$：
$$|Ta(x)| = \left|\int (K(x,y) - K(x, x_Q)) a(y) dy\right|$$
$$\leq C \int_Q \frac{|y - x_Q|^\delta}{|x - x_Q|^{n+\delta}} |a(y)| dy \leq \frac{C|Q|^{\delta/n}}{|x - x_Q|^{n+\delta}}$$

**Step 3**：分子条件验证。

积分 $\int_{(Q^*)^c} |Ta(x)| |x - x_Q|^{n\varepsilon} dx$ 可控。

**$L^\infty \to BMO$：**

设 $f \in L^\infty$，$Q$ 是任意方体。

$$Tf - (Tf)_Q = T(f - f_Q) + \text{误差}$$

第一项由C-Z理论，第二项由核的光滑性。

$$\frac{1}{|Q|}\int_Q |Tf - (Tf)_Q| \leq C\|f\|_\infty$$
$\square$

### (b) 非切向极大函数

**证明概要：**

Poisson核：$P_t(x) = c_n \frac{t}{(|x|^2 + t^2)^{(n+1)/2}}$。

**Step 1**：点wise控制。

$$P_t * f(y) = \int P_t(y-z) f(z) dz$$

对 $|x-y| < t$：
$$P_t(y-z) \leq C \frac{t}{(t + |y-z|)^{n+1}} \leq C' \frac{t}{(t + |x-z|)^{n+1}}$$

**Step 2**：与Hardy-Littlewood极大函数比较。

$$P^*f(x) \leq C Mf(x)$$

因 $P_t$ 是径向递减的$L^1$函数。

**Step 3**：应用极大函数定理。
$$\|P^*f\|_{L^p} \leq C \|Mf\|_{L^p} \leq C_p \|f\|_{L^p}$$
$\square$

### (c) $L \log L$遍历定理

**证明概要：**

**Step 1**：分解。

对 $f \geq 0$，$\lambda > 0$，令 $f = f \chi_{f \leq \lambda} + f \chi_{f > \lambda} = f_1 + f_2$。

**Step 2**：好函数。

$f_1 \in L^\infty$，由经典Birkhoff定理：
$$\frac{1}{n} \sum_{k=0}^{n-1} f_1(T^k x) \to \int f_1$$

**Step 3**：坏函数的弱控制。

对 $f_2$，用极大遍历不等式：
$$\mu\left(\sup_n A_n f_2 > \alpha\right) \leq \frac{C}{\alpha} \int f_2 \leq \frac{C}{\alpha} \int_{f > \lambda} f$$

**Step 4**：$L \log L$条件。

$$\int_{f > \lambda} f \leq \frac{1}{\log \lambda} \int f \log^+ f$$

选择 $\lambda$ 适当依赖于$\alpha$，用Borel-Cantelli完成证明。$\square$

## 证明技巧

1. **原子-分子理论**：Hardy空间的算子有界性
2. **径向控制**：Poisson积分与极大函数
3. **截断技巧**：Orlicz空间的遍历理论

## 常见错误

- ❌ C-Z算子与$H^1$分子定义不匹配
- ❌ 非切向区域与锥形区域混淆
- ❌ $L\log L$与$L^1$遍历定理的区别
