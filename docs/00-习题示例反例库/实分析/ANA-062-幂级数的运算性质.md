## 题目

设幂级数 $S(x) = \sum_{n=0}^{\infty} a_n x^n$ 的收敛半径为 $R > 0$。

**(a)** 证明逐项求导后的级数 $\sum_{n=1}^{\infty} n a_n x^{n-1}$ 收敛半径仍为 $R$。

**(b)** 证明逐项积分后的级数 $\sum_{n=0}^{\infty} \frac{a_n}{n+1} x^{n+1}$ 收敛半径仍为 $R$。

**(c)** 已知 $\frac{1}{1-x} = \sum_{n=0}^{\infty} x^n$ ($|x| < 1$)，求 $\ln(1-x)$ 和 $\frac{1}{(1-x)^2}$ 的幂级数展开。

**(d)** 证明 $S(x)$ 在 $(-R, R)$ 内无穷次可微。

## 详细解答

### (a) 逐项求导保持收敛半径

**证明：**

设原级数收敛半径为 $R$，则 $\limsup |a_n|^{1/n} = \frac{1}{R}$。

求导后级数系数为 $b_n = (n+1)a_{n+1}$。

$$\limsup |b_n|^{1/n} = \limsup |(n+1)a_{n+1}|^{1/n} = \limsup |a_{n+1}|^{1/n}$$

（因 $\lim (n+1)^{1/n} = 1$）

$$= \limsup |a_{n+1}|^{\frac{1}{n+1} \cdot \frac{n+1}{n}} = \left(\frac{1}{R}\right)^1 = \frac{1}{R}$$

故求导后收敛半径仍为 $R$。

**证毕。**

### (b) 逐项积分保持收敛半径

**证明：**

积分后级数系数为 $c_n = \frac{a_{n-1}}{n}$（$n \geq 1$）。

$$\limsup |c_n|^{1/n} = \limsup \left|\frac{a_{n-1}}{n}\right|^{1/n} = \limsup |a_{n-1}|^{1/n}$$

$$= \limsup |a_{n-1}|^{\frac{1}{n-1} \cdot \frac{n-1}{n}} = \frac{1}{R}$$

故积分后收敛半径仍为 $R$。

**证毕。**

### (c) 几何级数的运算

**求 $\ln(1-x)$：**

$$-\ln(1-x) = \int_0^x \frac{1}{1-t} dt = \int_0^x \sum_{n=0}^{\infty} t^n dt = \sum_{n=0}^{\infty} \frac{x^{n+1}}{n+1} = \sum_{n=1}^{\infty} \frac{x^n}{n}$$

故：
$$\ln(1-x) = -\sum_{n=1}^{\infty} \frac{x^n}{n}, \quad |x| < 1$$

**求 $\frac{1}{(1-x)^2}$：**

$$\frac{1}{(1-x)^2} = \frac{d}{dx}\left(\frac{1}{1-x}\right) = \frac{d}{dx}\sum_{n=0}^{\infty} x^n = \sum_{n=1}^{\infty} n x^{n-1} = \sum_{n=0}^{\infty} (n+1) x^n$$

**验证：** 在 $x=0$ 处，左边=1，右边系数 $a_0 = 1$。✓

### (d) 无穷次可微性

**证明：**

由(a)，逐项求导后收敛半径不变，故在 $(-R, R)$ 内可导。

新级数仍可逐项求导，归纳可得 $S(x)$ 有任意阶导数。

第 $k$ 阶导数：
$$S^{(k)}(x) = \sum_{n=k}^{\infty} n(n-1)\cdots(n-k+1) a_n x^{n-k}$$

**证毕。**

## 证明技巧

1. **收敛半径不变性的本质：** 系数的多项式增长/衰减不影响指数级行为
2. **幂级数运算的合法性：** 在收敛区间内可逐项积分、求导
3. **常见函数的幂级数：** 从几何级数出发，通过运算得到其他函数

## 常见错误

| 错误 | 说明 |
|------|------|
| 认为端点收敛性也不变 | 逐项运算可能改变端点行为 |
| 积分时忘记常数项 | 不定积分有常数，定积分从0开始 |
| 收敛半径为0时强行运算 | $R=0$ 时级数仅在一点收敛 |
