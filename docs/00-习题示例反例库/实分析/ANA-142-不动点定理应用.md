# 不动点定理及其应用

## 题目

**(a)** 证明Banach不动点定理：完备度量空间 $(X, d)$ 上的压缩映射 $T: X \to X$（即存在 $0 \leq k < 1$ 使 $d(Tx, Ty) \leq k d(x, y)$）有唯一不动点。

**(b)** 证明Schauder不动点定理：Banach空间中凸紧集 $K$ 上的连续映射 $T: K \to K$ 有不动点。

**(c)** 用Banach不动点定理证明Picard存在唯一性定理：初值问题 $y' = f(x,y)$，$y(x_0) = y_0$ 在 $f$ Lipschitz时有唯一解。

## 解答

### (a) Banach不动点定理

**证明：**

任取 $x_0 \in X$，定义 $x_{n+1} = T x_n$。

$$d(x_{n+1}, x_n) = d(Tx_n, Tx_{n-1}) \leq k d(x_n, x_{n-1}) \leq \cdots \leq k^n d(x_1, x_0)$$

对 $m > n$：
$$d(x_m, x_n) \leq \sum_{j=n}^{m-1} d(x_{j+1}, x_j) \leq d(x_1, x_0) \sum_{j=n}^\infty k^j = \frac{k^n}{1-k} d(x_1, x_0) \to 0$$

故 $\{x_n\}$ 是Cauchy列，收敛于某 $x^*$。

$$x^* = \lim x_{n+1} = \lim Tx_n = Tx^*$$

**唯一性**：若 $Tx = x$，$Ty = y$：
$$d(x,y) = d(Tx, Ty) \leq k d(x,y)$$

$(1-k)d(x,y) \leq 0$，故 $x = y$。$\square$

### (b) Schauder不动点定理

**证明概要：**

**Step 1**：有限维逼近。

对 $\varepsilon > 0$，取有限 $\varepsilon$-网 $\{x_1, \ldots, x_n\} \subset K$。

定义Schauder投影：
$$P_\varepsilon(x) = \frac{\sum_i \max(0, \varepsilon - \|x - x_i\|) x_i}{\sum_i \max(0, \varepsilon - \|x - x_i\|)}$$

$P_\varepsilon: K \to K_\varepsilon \subset \text{conv}\{x_i\}$ 连续。

**Step 2**：有限维情形。

$T_\varepsilon = P_\varepsilon \circ T: K_\varepsilon \to K_\varepsilon$。

$K_\varepsilon$ 是有限维凸紧集，由Brouwer不动点定理，$T_\varepsilon$ 有不动点 $x_\varepsilon$。

**Step 3**：极限。

$\varepsilon \to 0$，$x_\varepsilon$ 有子列收敛于 $x \in K$，$Tx = x$。$\square$

### (c) Picard定理

**证明：**

初值问题等价于积分方程：
$$y(x) = y_0 + \int_{x_0}^x f(t, y(t)) dt$$

定义算子：$(Ty)(x) = y_0 + \int_{x_0}^x f(t, y(t)) dt$。

设 $f$ 关于 $y$ 是 $L$-Lipschitz。

在 $C([x_0 - \delta, x_0 + \delta])$ 上：
$$|Ty_1(x) - Ty_2(x)| \leq \int_{x_0}^x L|y_1(t) - y_2(t)| dt \leq L\delta \|y_1 - y_2\|_\infty$$

取 $\delta < 1/L$，$T$ 是压缩映射。

由Banach定理，存在唯一不动点，即唯一解。$\square$
