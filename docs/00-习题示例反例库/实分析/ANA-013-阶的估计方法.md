# ANA-013: 阶的估计方法

**题号**: ANA-013
**难度**: ⭐⭐⭐ (Level 3)
**类型**: 计算型 (CALC)

**主题**: 阶的比较与估计

---

## 题目

使用阶的估计方法计算或证明下列极限：

**(a)** 求 $\displaystyle\lim_{n\to\infty} \frac{n!}{n^n}$

**(b)** 求 $\displaystyle\lim_{n\to\infty} \frac{\sqrt[n]{n!}}{n}$

**(c)** 设 $a_n = 1 + \frac{1}{2} + \cdots + \frac{1}{n} - \ln n$，证明 $\{a_n\}$ 收敛

---

## 解答

### 阶的符号回顾

- **大O记号**: $f(n) = O(g(n))$ 表示 $|f(n)| \leq C|g(n)|$（$n$ 充分大）
- **小o记号**: $f(n) = o(g(n))$ 表示 $\lim_{n\to\infty} \frac{f(n)}{g(n)} = 0$
- **渐近等价**: $f(n) \sim g(n)$ 表示 $\lim_{n\to\infty} \frac{f(n)}{g(n)} = 1$

---

### (a) 求 $\displaystyle\lim_{n\to\infty} \frac{n!}{n^n}$

**分析**: 使用Stirling公式进行阶的估计。

**解**:

**Stirling公式**: 
$$n! \sim \sqrt{2\pi n} \left(\frac{n}{e}\right)^n$$

即 $n! = \sqrt{2\pi n} \left(\frac{n}{e}\right)^n (1 + o(1))$

估计：
$$\frac{n!}{n^n} \sim \frac{\sqrt{2\pi n} \cdot n^n \cdot e^{-n}}{n^n} = \sqrt{2\pi n} \cdot e^{-n}$$

由于 $e^{-n}$ 衰减远快于 $\sqrt{n}$ 增长：
$$\sqrt{2\pi n} \cdot e^{-n} = \sqrt{2\pi} \cdot \frac{\sqrt{n}}{e^n} \to 0$$

因此：
$$\lim_{n\to\infty} \frac{n!}{n^n} = 0 \quad \square$$

---

### (b) 求 $\displaystyle\lim_{n\to\infty} \frac{\sqrt[n]{n!}}{n}$

**分析**: 利用对数转化为求和问题。

**解**:

令 $L = \displaystyle\lim_{n\to\infty} \frac{\sqrt[n]{n!}}{n}$

取对数：
$$\ln L = \lim_{n\to\infty} \left(\frac{\ln(n!)}{n} - \ln n\right) = \lim_{n\to\infty} \frac{\ln(n!) - n\ln n}{n}$$

利用Stirling公式：
$$\ln(n!) = n\ln n - n + \frac{1}{2}\ln(2\pi n) + O\left(\frac{1}{n}\right)$$

因此：
$$\ln(n!) - n\ln n = -n + \frac{1}{2}\ln(2\pi n) + O\left(\frac{1}{n}\right)$$

于是：
$$\frac{\ln(n!) - n\ln n}{n} = -1 + \frac{\ln(2\pi n)}{2n} + O\left(\frac{1}{n^2}\right) \to -1$$

所以：
$$\ln L = -1 \Rightarrow L = e^{-1} = \frac{1}{e}$$

即：
$$\lim_{n\to\infty} \frac{\sqrt[n]{n!}}{n} = \frac{1}{e} \quad \square$$

---

### (c) Euler-Mascheroni常数

**分析**: 证明单调有界。

**证明**:

设 $a_n = H_n - \ln n$，其中 $H_n = 1 + \frac{1}{2} + \cdots + \frac{1}{n}$。

**Step 1: 证明递减**

$$a_{n+1} - a_n = \frac{1}{n+1} - \ln(n+1) + \ln n = \frac{1}{n+1} - \ln\left(1 + \frac{1}{n}\right)$$

利用不等式 $\ln(1+x) > \frac{x}{1+x}$（$x > 0$）：
$$\ln\left(1 + \frac{1}{n}\right) > \frac{\frac{1}{n}}{1 + \frac{1}{n}} = \frac{1}{n+1}$$

因此 $a_{n+1} - a_n < 0$，即 $\{a_n\}$ 递减。

**Step 2: 证明有下界**

利用不等式 $\ln(1+x) < x$（$x > 0$）：
$$\int_k^{k+1} \frac{1}{x}\, dx = \ln(k+1) - \ln k < \frac{1}{k}$$

对 $k = 1$ 到 $n-1$ 求和：
$$\ln n < 1 + \frac{1}{2} + \cdots + \frac{1}{n-1} = H_n - \frac{1}{n}$$

因此：
$$a_n = H_n - \ln n > \frac{1}{n} > 0$$

**Step 3: 结论**

$\{a_n\}$ 递减有下界，故收敛。

极限值 $\gamma = \lim_{n\to\infty} (H_n - \ln n) \approx 0.5772$ 称为**Euler-Mascheroni常数**。 $\square$

| 技巧 | 应用位置 | 说明 |
|------|----------|------|
| Stirling公式 | (a)(b) | 阶乘的渐近估计 |
| 对数转换 | (b) | 将乘积转化为求和 |
| 积分比较 | (c) | 调和级数与对数的联系 |
| 单调有界 | (c) | 收敛的经典判定 |

### 关键洞察

1. **阶乘的增长**: $n!$ 比任何指数 $a^n$ ($a > 0$) 增长都快，但比 $n^n$ 慢
2. **Stirling公式的精度**: $\ln(n!) = n\ln n - n + O(\ln n)$
3. **调和级数的渐近**: $H_n = \ln n + \gamma + O(1/n)$

## 常见错误

### 错误1: Stirling公式误用

❌ **错误做法**:
> $n! = n^n$（忽略 $e^{-n}$ 因子）

✅ **正确理解**:
> $n! \sim \sqrt{2\pi n}(n/e)^n$，指数因子至关重要

### 错误2: 忽略余项

❌ **错误做法**:
> $\ln(n!) = n\ln n - n$（完全相等）

✅ **正确做法**:
> 这是渐近等价，实际有 $O(\ln n)$ 余项

### 错误3: 积分方向错误

❌ **错误做法**:
> 对递减函数 $f$，错误使用 $\int_k^{k+1} f(x)\, dx > f(k)$

✅ **正确做法**:
> 递减函数满足 $\int_k^{k+1} f(x)\, dx < f(k)$

计算 $\displaystyle\lim_{n\to\infty} \frac{(n!)^{1/n}}{\sqrt{n}}$

**答案**: $\sqrt{2\pi}/e$

**提示**: 使用完整Stirling公式

**提示**: 将乘积写为 $\frac{(2n)!}{n!}$，再用Stirling公式

求 $\displaystyle\lim_{n\to\infty} \sum_{k=1}^{n} \frac{1}{\sqrt{n^2 + k}}$

**答案**: $1$

**提示**: 阶的估计 + 夹逼定理
