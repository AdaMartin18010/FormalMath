# 复几何中的分析

## 题目

**(a)** 证明 Calabi-Yau 定理：紧 Kähler 流形上给定体积形式，存在唯一的 Ricci 平坦 Kähler 度规。

**(b)** 证明 Hörmander 的 L² 估计：在拟凸域上，∂̄-方程有解且满足估计。

**(c)** 讨论全纯线丛上的 Hermitian-Einstein 度规。

## 解答

### (a) Calabi-Yau 定理

Yau 的证明：解复 Monge-Ampère 方程 det(gᵢⱼ + ∂ᵢ∂ⱼ̄φ) = eᶠ det(gᵢⱼ)。连续性方法结合先验估计。

### (b) Hörmander L² 估计

对强拟凸域，利用权函数和 Morrey-Kohn-Hörmander 公式。

### (c) Hermitian-Einstein 度规

√-1 Λ Fₕ = λ I，与稳定性相关（Donaldson-Uhlenbeck-Yau）。
