# ANA-007: 中值定理的应用

## 题目陈述

**微分中值定理（MVT）**: 设 $f: [a, b] \to \mathbb{R}$ 连续，在 $(a, b)$ 可导，则存在 $c \in (a, b)$ 使得
$$f'(c) = \frac{f(b) - f(a)}{b - a}$$

**Rolle定理**: MVT 在 $f(a) = f(b)$ 时的特例，即存在 $c$ 使得 $f'(c) = 0$。

**问题**:

1. 证明：若 $f'(x) = 0$ 对所有 $x \in (a, b)$，则 $f$ 为常数
2. 证明：若 $f'(x) > 0$ 对所有 $x \in (a, b)$，则 $f$ 严格递增
3. 设 $f$ 在 $[0, 1]$ 上二阶可导，$f(0) = f(1) = 0$，$\min_{x \in [0,1]} f(x) = -1$。证明存在 $\xi \in (0, 1)$ 使得 $f''(\xi) \geq 8$。

## 详细解答

### 例1：导数为零则函数为常数

**定理**: 设 $f$ 在 $(a, b)$ 可导且 $f'(x) = 0$ 对所有 $x \in (a, b)$，则 $f$ 为常数。

**证明**:

任取 $x_1, x_2 \in (a, b)$，$x_1 < x_2$。

$f$ 在 $[x_1, x_2]$ 上连续，在 $(x_1, x_2)$ 上可导，由 MVT，存在 $c \in (x_1, x_2)$ 使得
$$f(x_2) - f(x_1) = f'(c)(x_2 - x_1) = 0 \cdot (x_2 - x_1) = 0$$

故 $f(x_1) = f(x_2)$。由 $x_1, x_2$ 的任意性，$f$ 为常数。 $\square$

**推论**: 若 $f' = g'$ on $(a, b)$，则 $f - g = \text{const}$。

---

### 例2：导数符号与单调性

**定理**: 设 $f$ 在 $(a, b)$ 可导。

- 若 $f'(x) > 0$ 对所有 $x \in (a, b)$，则 $f$ 严格递增
- 若 $f'(x) \geq 0$ 对所有 $x \in (a, b)$，则 $f$ 单调递增

**证明**（$f' > 0$ 情形）：

对 $x_1 < x_2$，由 MVT，存在 $c \in (x_1, x_2)$ 使得
$$f(x_2) - f(x_1) = f'(c)(x_2 - x_1) > 0 \cdot (x_2 - x_1) = 0$$

故 $f(x_2) > f(x_1)$，$f$ 严格递增。 $\square$

**注意**: 逆命题不成立！$f(x) = x^3$ 严格递增，但 $f'(0) = 0$。

---

### 例3：二阶导数的下界估计

**问题**: $f \in C^2[0, 1]$，$f(0) = f(1) = 0$，$\min_{[0,1]} f = -1$。证明 $\exists \xi: f''(\xi) \geq 8$。

**分析**: 最小值点 $x_0$ 满足 $f(x_0) = -1$，$f'(x_0) = 0$（Fermat定理）。

**证明**:

设 $f$ 在 $x_0 \in (0, 1)$ 处取最小值 $-1$，则 $f'(x_0) = 0$。

对 $f$ 分别在 $[0, x_0]$ 和 $[x_0, 1]$ 上用 Taylor 展开（带 Lagrange 余项）：

**在 $[0, x_0]$ 上**：
$$f(0) = f(x_0) + f'(x_0)(0 - x_0) + \frac{f''(\xi_1)}{2}(0 - x_0)^2$$

即
$$0 = -1 + 0 + \frac{f''(\xi_1)}{2} x_0^2$$

解得
$$f''(\xi_1) = \frac{2}{x_0^2}$$

**在 $[x_0, 1]$ 上**：
$$f(1) = f(x_0) + f'(x_0)(1 - x_0) + \frac{f''(\xi_2)}{2}(1 - x_0)^2$$

即
$$0 = -1 + 0 + \frac{f''(\xi_2)}{2}(1 - x_0)^2$$

解得
$$f''(\xi_2) = \frac{2}{(1 - x_0)^2}$$

**综合分析**:

若 $x_0 \leq \frac{1}{2}$，则 $f''(\xi_1) = \frac{2}{x_0^2} \geq \frac{2}{(1/2)^2} = 8$。

若 $x_0 \geq \frac{1}{2}$，则 $f''(\xi_2) = \frac{2}{(1-x_0)^2} \geq \frac{2}{(1/2)^2} = 8$。

总之，存在 $\xi \in (0, 1)$ 使得 $f''(\xi) \geq 8$。 $\square$

**最优性**: $f(x) = 4x^2 - 4x = 4x(x-1)$ 在 $x = \frac{1}{2}$ 处取最小值 $-1$，$f''(x) = 8$。

---

## 证明技巧总结

### MVT 的核心应用场景

```
1. 函数差值的估计
   f(b) - f(a) = f'(c)(b - a)

2. 导数信息 ⇒ 函数性质
   f' > 0 ⇒ f 递增
   f' = 0 ⇒ f 常数

3. 函数信息 ⇒ 导数存在性
   f(a) = f(b) ⇒ 存在 c: f'(c) = 0
```

### 高阶导数问题的策略

| 问题类型 | 工具 | 关键步骤 |
|----------|------|---------|
| 二阶导数估计 | Taylor展开到二阶 | 在关键点展开 |
| 多中值问题 | 多次MVT或Cauchy MVT | 构造辅助函数 |
| 存在性问题 | Rolle定理或反证法 | 构造满足端点条件的函数 |

### Taylor展开的选取原则

```
展开点的选择：
- 已知函数值/导数值的点
- 极值点（导数为0）
- 区间中点（对称性）
- 待定参数，后优化
```

## 常见错误分析

### 错误1：忽视连续性或可导性条件

```
❌ 错误：对不连续函数用MVT

✅ 正确：验证[a,b]连续，(a,b)可导

💡 反例：f(x) = {0, x=0; 1, x∈(0,1]}，f(0)=f(1)但无f'=0的点
```

### 错误2：混淆严格递增与导数正

```
❌ 错误："f严格递增 ⇒ f' > 0"

✅ 正确：f'(x) ≥ 0，且不在任何区间上恒为0

💡 反例：f(x) = x³，严格递增但f'(0) = 0
```

### 错误3：Taylor展开阶数不足

```
❌ 错误：对二阶导数问题用一阶Taylor

✅ 正确：展开到所需阶数，保留Lagrange余项

💡 标准形式：f(x) = f(a) + f'(a)(x-a) + f''(ξ)(x-a)²/2!
```

### 错误4：多点展开后处理不当

```
❌ 错误：在a和b分别展开后不知如何联系

✅ 正确：利用中间点（如极值点）作为"桥梁"

💡 常见模式：a ← x₀ → b，其中x₀是关键点
```

## 变式练习

### 变式1（难度⭐⭐）

设 $f$ 在 $[a, b]$ 上可导，$f(a) = f(b) = 0$。证明对任意实数 $\lambda$，存在 $c \in (a, b)$ 使得
$$f'(c) = \lambda f(c)$$

<details>
<summary>提示</summary>
考虑辅助函数 $g(x) = e^{-\lambda x} f(x)$。

则 $g(a) = g(b) = 0$，由Rolle定理，存在 $c$ 使得 $g'(c) = 0$。

计算：$g'(c) = e^{-\lambda c}(f'(c) - \lambda f(c)) = 0$

故 $f'(c) = \lambda f(c)$。 $\square$
</details>

### 变式2（难度⭐⭐⭐）

设 $f$ 在 $[0, 1]$ 上二阶可导，$|f(x)| \leq 1$，$|f''(x)| \leq 1$ 对所有 $x \in [0, 1]$。证明 $|f'(x)| \leq 2$ 对所有 $x \in [0, 1]$。

<details>
<summary>提示</summary>
对任意 $x \in (0, 1)$，用Taylor展开：

在 $x$ 点展开到0和1：
$$f(0) = f(x) - xf'(x) + \frac{x^2}{2}f''(\xi_1)$$
$$f(1) = f(x) + (1-x)f'(x) + \frac{(1-x)^2}{2}f''(\xi_2)$$

两式相减消去 $f(x)$：
$$f(1) - f(0) = f'(x) + \frac{(1-x)^2}{2}f''(\xi_2) - \frac{x^2}{2}f''(\xi_1)$$

解出 $f'(x)$ 并估计：
$$|f'(x)| \leq |f(1)| + |f(0)| + \frac{(1-x)^2 + x^2}{2} \leq 2 + \frac{1}{2} = 2.5$$

进一步优化：利用 $f(0)$ 或 $f(1)$ 的展开直接估计，可得 $|f'(x)| \leq 2$。
</details>

### 变式3（难度⭐⭐⭐⭐）

设 $f$ 在 $[0, 1]$ 上连续，在 $(0, 1)$ 可导，$f(0) = 0$，$f(1) = 1$。证明：

1. 存在 $\xi_1, \xi_2 \in (0, 1)$，$\xi_1 \neq \xi_2$ 使得 $f'(\xi_1) + f'(\xi_2) = 2$
2. 若还满足 $f(x) \neq x$ 对所有 $x \in (0, 1)$，则存在 $c$ 使得 $f'(c) = 2$

<details>
<summary>提示</summary>
<strong>第一部分</strong>：由MVT，存在 $c \in (0,1)$ 使得 $f'(c) = 1$。

分别在 $[0, c]$ 和 $[c, 1]$ 上用MVT：

- 存在 $\xi_1 \in (0, c)$：$f'(\xi_1) = \frac{f(c)}{c}$
- 存在 $\xi_2 \in (c, 1)$：$f'(\xi_2) = \frac{1 - f(c)}{1 - c}$

注意到 $f'(\xi_1) \cdot c + f'(\xi_2) \cdot (1-c) = f(c) + (1-f(c)) = 1$。

若 $f(c) = c$，则 $f'(\xi_1) = f'(\xi_2) = 1$，和为2。

若 $f(c) \neq c$，适当选择 $c$ 使得...

<strong>第二部分</strong>：由条件，$f(x) > x$ 对所有 $x$ 或 $f(x) < x$ 对所有 $x$。

不妨设 $f(x) > x$。则 $g(x) = f(x) - x$ 在 $(0,1)$ 为正，$g(0) = g(1) = 0$。

设 $g$ 在 $c$ 取最大值，则 $g'(c) = 0$ 即 $f'(c) = 1$。

进一步分析可得更强的结论...
</details>
