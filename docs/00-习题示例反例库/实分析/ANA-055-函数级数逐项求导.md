## 题目

**(a)** 证明函数级数 $S(x) = \sum_{n=1}^{\infty} \frac{\sin(nx)}{n^3}$ 在 $\mathbb{R}$ 上可导，并求 $S'(x)$。

**(b)** 研究级数 $\sum_{n=1}^{\infty} \frac{\sin(nx)}{n}$ 的逐项可导性。

**(c)** 设 $f(x) = \sum_{n=0}^{\infty} \frac{x^n}{n!}$（指数函数的定义），证明 $f'(x) = f(x)$。

## 详细解答

### (a) 高阶衰减级数的逐项求导

**定理回顾：** 若 $\sum f_n'$ 一致收敛且 $\sum f_n$ 在某点收敛，则 $(\sum f_n)' = \sum f_n'$。

**证明：**

原级数：$f_n(x) = \frac{\sin(nx)}{n^3}$

形式求导：$f_n'(x) = \frac{n\cos(nx)}{n^3} = \frac{\cos(nx)}{n^2}$

**步骤1：** 验证 $\sum f_n$ 收敛
- 由M-判别法：$|f_n(x)| \leq \frac{1}{n^3}$，$\sum \frac{1}{n^3} < \infty$
- 故原级数一致收敛

**步骤2：** 验证 $\sum f_n'$ 一致收敛
- $|f_n'(x)| = \left|\frac{\cos(nx)}{n^2}\right| \leq \frac{1}{n^2}$
- $\sum \frac{1}{n^2} < \infty$
- 由M-判别法，$\sum f_n'$ 一致收敛

**步骤3：** 应用逐项求导定理

因此 $S(x)$ 可导，且：
$$S'(x) = \sum_{n=1}^{\infty} \frac{\cos(nx)}{n^2}$$

同理，$S(x)$ 实际上无穷次可导。

**证毕。**

### (b) 临界情况分析

级数 $\sum \frac{\sin(nx)}{n}$（Fourier级数相关）。

**分析：**

形式求导得 $\sum \cos(nx)$，这级数**不收敛**（通项不趋于0）。

**结论：**
- 原级数在 $(0, 2\pi)$ 上收敛（到 $\frac{\pi-x}{2}$）
- 但不能逐项求导
- 实际上，和函数在 $x=0$ 有跳跃间断，不可导

**注：** 这说明逐项求导需要比逐项积分更强的条件。

### (c) 指数函数的微分性质

**证明：**

对于任意有限区间 $[-M, M]$，考虑级数 $f(x) = \sum_{n=0}^{\infty} \frac{x^n}{n!}$。

**步骤1：** 原级数收敛性

由比值判别法，收敛半径 $R = \infty$，在任意有界区间上一致收敛。

**步骤2：** 导函数级数

形式求导：
$$\frac{d}{dx}\left(\frac{x^n}{n!}\right) = \frac{nx^{n-1}}{n!} = \frac{x^{n-1}}{(n-1)!}$$

导函数级数：$\sum_{n=1}^{\infty} \frac{x^{n-1}}{(n-1)!} = \sum_{m=0}^{\infty} \frac{x^m}{m!} = f(x)$

**步骤3：** 验证一致收敛

在 $[-M, M]$ 上：
$$\left|\frac{x^{n-1}}{(n-1)!}\right| \leq \frac{M^{n-1}}{(n-1)!}$$

由比值判别法，$\sum \frac{M^{n-1}}{(n-1)!}$ 收敛，故导函数级数一致收敛。

因此 $f'(x) = f(x)$。

**证毕。**

## 证明技巧

1. **逐项求导的充分条件：**
   - $\sum f_n$ 在某点收敛
   - 每个 $f_n$ 可导
   - $\sum f_n'$ **一致收敛**

2. **一致收敛的验证策略：**
   - 有界区间上用Weierstrass M-判别法
   - 注意收敛半径的概念

3. **反例构造：** 降低衰减阶数（如 $\frac{1}{n}$ 代替 $\frac{1}{n^2}$）往往破坏逐项求导性

## 常见错误

| 错误 | 说明 |
|------|------|
| 仅验证原级数收敛就逐项求导 | 需要导函数级数一致收敛 |
| 忽略定义域的限制 | 一致收敛是局部性质，需在紧集上验证 |
| 对条件收敛级数盲目求导 | 条件收敛级数求导后可能发散 |

## 变式练习

**变式1：** 证明 $\sum_{n=0}^{\infty} x^n = \frac{1}{1-x}$ 在 $(-1,1)$ 内可逐项求导，得到 $\sum nx^{n-1} = \frac{1}{(1-x)^2}$。

**变式2：** 研究 $\zeta(s) = \sum_{n=1}^{\infty} \frac{1}{n^s}$ 在 $s > 1$ 时的可导性。

**变式3：** 证明幂级数在其收敛区间内可任意次逐项求导。
