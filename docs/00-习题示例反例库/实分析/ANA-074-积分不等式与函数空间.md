## 题目

设 $(X, \mathcal{M}, \mu)$ 是测度空间，$1 \leq p, q, r \leq \infty$ 满足 $\frac{1}{r} = \frac{1}{p} + \frac{1}{q}$。

(1) 证明 **Hölder不等式的一般形式**：若 $f \in L^p$，$g \in L^q$，则 $fg \in L^r$ 且

$$\|fg\|_r \leq \|f\|_p \|g\|_q$$

(2) 设 $1 \leq p < q < \infty$，$f \in L^p \cap L^q$，证明 **插值不等式**：

$$\|f\|_s \leq \|f\|_p^{\alpha} \|f\|_q^{1-\alpha}$$

其中 $\frac{1}{s} = \frac{\alpha}{p} + \frac{1-\alpha}{q}$，$\alpha \in (0,1)$。

(3) 应用：设 $f \in L^1[0,1] \cap L^3[0,1]$，$\|f\|_1 = 2$，$\|f\|_3 = 1$，求 $\|f\|_2$ 的最佳上界估计。

## 详细解答

### (1) Hölder不等式的一般形式

**标准Hölder回顾**：若 $\frac{1}{p} + \frac{1}{q} = 1$，则 $\|fg\|_1 \leq \|f\|_p \|g\|_q$。

**证明一般形式**：

设 $F = |f|^r$，$G = |g|^r$，我们需要估计 $\|FG\|_1 = \|fg\|_r^r$。

寻找共轭指数：设 $\frac{1}{a} + \frac{1}{b} = 1$，对 $F \in L^a$，$G \in L^b$ 用标准Hölder。

$$F \in L^a \Leftrightarrow |f|^{ra} \in L^1 \Leftrightarrow f \in L^{ra}$$

需要 $ra = p$，即 $a = \frac{p}{r}$。同理 $b = \frac{q}{r}$。

验证：$\frac{1}{a} + \frac{1}{b} = \frac{r}{p} + \frac{r}{q} = r\left(\frac{1}{p} + \frac{1}{q}\right) = r \cdot \frac{1}{r} = 1$。✓

应用标准Hölder：

$$\|FG\|_1 \leq \|F\|_a \|G\|_b = \||f|^r\|_{p/r} \||g|^r\|_{q/r}$$

$$= \left(\int |f|^p\right)^{r/p} \left(\int |g|^q\right)^{r/q} = \|f\|_p^r \|g\|_q^r$$

因此：

$$\|fg\|_r^r \leq \|f\|_p^r \|g\|_q^r$$

取 $r$ 次方根得证。

### (2) 插值不等式

**证明**：我们需要将 $|f|^s$ 表示为适当的乘积形式。

设 $\frac{1}{s} = \frac{\alpha}{p} + \frac{1-\alpha}{q}$，改写为：

$$1 = \frac{s\alpha}{p} + \frac{s(1-\alpha)}{q}$$

令 $a = \frac{p}{s\alpha}$，$b = \frac{q}{s(1-\alpha)}$，则 $\frac{1}{a} + \frac{1}{b} = 1$。

将 $|f|^s = |f|^{s\alpha} \cdot |f|^{s(1-\alpha)}$。

对指数 $\frac{p}{s\alpha}$ 和 $\frac{q}{s(1-\alpha)}$ 用Hölder：

$$\|f\|_s^s = \int |f|^s = \int |f|^{s\alpha} \cdot |f|^{s(1-\alpha)}$$

$$\leq \left(\int |f|^{s\alpha \cdot \frac{p}{s\alpha}}\right)^{\frac{s\alpha}{p}} \left(\int |f|^{s(1-\alpha) \cdot \frac{q}{s(1-\alpha)}}\right)^{\frac{s(1-\alpha)}{q}}$$

$$= \left(\int |f|^p\right)^{\frac{s\alpha}{p}} \left(\int |f|^q\right)^{\frac{s(1-\alpha)}{q}}$$

$$= \|f\|_p^{s\alpha} \|f\|_q^{s(1-\alpha)}$$

取 $s$ 次方根：

$$\|f\|_s \leq \|f\|_p^{\alpha} \|f\|_q^{1-\alpha}$$

### (3) 应用计算

**设定**：$p = 1$，$q = 3$，求 $s = 2$ 的估计。

由插值条件：$\frac{1}{2} = \frac{\alpha}{1} + \frac{1-\alpha}{3}$

解得：$\frac{1}{2} = \alpha + \frac{1}{3} - \frac{\alpha}{3} = \frac{2\alpha}{3} + \frac{1}{3}$

$$\frac{1}{2} - \frac{1}{3} = \frac{2\alpha}{3} \Rightarrow \frac{1}{6} = \frac{2\alpha}{3} \Rightarrow \alpha = \frac{1}{4}$$

因此：

$$\|f\|_2 \leq \|f\|_1^{1/4} \|f\|_3^{3/4} = 2^{1/4} \cdot 1^{3/4} = 2^{1/4} = \sqrt[4]{2}$$

**最优性验证**：这个上界是紧的。考虑测度空间 $[0,1]$ 上的函数：

设 $f = c \cdot \chi_{[0,a]}$ 对某常数 $c$ 和 $a \in (0,1]$。

则：$\|f\|_1 = ca = 2$，$\|f\|_3 = c a^{1/3} = 1$。

由第二式：$c = a^{-1/3}$，代入第一式：$a^{2/3} = 2$，$a = 2^{3/2} = 2\sqrt{2} > 1$。

这不可行。尝试两水平函数。

实际上在有限测度空间，常数函数可以达到极值。设 $f = c$：

$\|f\|_1 = |c| = 2$，$\|f\|_3 = |c| = 1$，矛盾。

因此最优函数非常数。可以证明 $\sqrt[4]{2}$ 是可达的上确界但不可达。

**估计结果**：$\|f\|_2 \leq \sqrt[4]{2} \approx 1.189$。

## 证明技巧

1. **指数变换**：将乘积范数转化为和的形式，利用凸性
2. **插值参数**：通过调整 $\alpha$ 得到不同范数之间的连续过渡
3. **对偶表示**：$\|f\|_p = \sup_{\|g\|_{p'}=1} |\int fg|$，其中 $\frac{1}{p} + \frac{1}{p'} = 1$

## 常见错误

| 错误类型 | 错误表现 | 正确做法 |
|---------|---------|---------|
| 指数混淆 | Hölder中指数相加为1，但用于乘积 | 一般形式中是 $\frac{1}{r} = \frac{1}{p} + \frac{1}{q}$ |
| 插值条件 | 误用 $s = \alpha p + (1-\alpha)q$ | 正确是 $\frac{1}{s}$ 的凸组合 |
| 最优性 | 认为不等式总可取等 | 需要验证极值函数是否存在 |

## 变式练习

**变式1（难度⭐⭐⭐）**：证明若 $\mu(X) < \infty$ 且 $p < q$，则 $L^q \subset L^p$。

**变式2（难度⭐⭐⭐⭐）**：设 $f \in L^1 \cap L^\infty$，证明 $\|f\|_p \to \|f\|_\infty$ 当 $p \to \infty$。

**变式3（难度⭐⭐⭐⭐）**：在 $\mathbb{R}^n$ 上，设 $f \in L^1 \cap L^2$，$\hat{f}$ 是Fourier变换，估计 $\|\hat{f}\|_{L^\infty}$。
