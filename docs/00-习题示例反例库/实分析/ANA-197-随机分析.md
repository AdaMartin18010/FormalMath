# 随机分析基础

## 题目

**(a)** 证明 Itô 等距：E[|∫₀ᵀ Hₛ dWₛ|²] = E[∫₀ᵀ |Hₛ|² ds]。

**(b)** 证明鞅表示定理：任何 Brown 鞅可表示为随机积分。

**(c)** 讨论 Girsanov 定理及其在金融数学中的应用。

## 解答

### (a) Itô 等距

简单过程直接计算，一般过程通过逼近。

### (b) 鞅表示

对 Brown 滤子，任何平方可积鞅 M 有 $M_t = M_0 + \int_0^t H_s dW_s$。

### (c) Girsanov 定理

测度变换：dQ = Z_T dP，其中 $Z_t = \exp(\int_0^t \theta_s dW_s - \frac{1}{2}\int_0^t \theta_s^2 ds)$。

在新测度下，$\tilde{W}_t = W_t - \int_0^t \theta_s ds$ 是 Brown 运动。
