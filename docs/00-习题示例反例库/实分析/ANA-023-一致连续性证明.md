# ANA-023: 一致连续性证明

**题号**: ANA-023
**难度**: ⭐⭐⭐ (Level 3)
**类型**: 证明型 (PRF)
**来源**: MIT 18.100A Chapter 4.3
**主题**: 一致连续性的判定与证明

---

## 题目

判断下列函数在给定区间上是否一致连续，并证明你的结论：

**(a)** $f(x) = x^2$ 在 $[0,1]$ 和 $[0, +\infty)$ 上

**(b)** $f(x) = \sqrt{x}$ 在 $[0, +\infty)$ 上

**(c)** $f(x) = \sin\frac{1}{x}$ 在 $(0, 1]$ 上

---

## 解答

### 回顾：一致连续的定义

$f$ 在区间 $I$ 上**一致连续**，如果：
$$\forall \varepsilon > 0, \exists \delta > 0, \forall x,y \in I: |x-y| < \delta \Rightarrow |f(x)-f(y)| < \varepsilon$$

关键是$\delta$只依赖于$\varepsilon$，不依赖于点的位置。

---

### (a) $f(x) = x^2$ 的一致连续性

**在 $[0,1]$ 上**:

**结论**: 一致连续。

**证明**:

对 $x, y \in [0,1]$：
$$|f(x) - f(y)| = |x^2 - y^2| = |x+y||x-y| \leq 2|x-y|$$

给定 $\varepsilon > 0$，取 $\delta = \varepsilon/2$。

当 $|x-y| < \delta$：
$$|f(x) - f(y)| \leq 2|x-y| < 2 \cdot \frac{\varepsilon}{2} = \varepsilon$$

因此 $f$ 在 $[0,1]$ 上一致连续。 $\checkmark$

**在 $[0, +\infty)$ 上**:

**结论**: 不一致连续。

**证明**:

取 $\varepsilon = 1$。对任意 $\delta > 0$，令 $x = n$，$y = n + \frac{\delta}{2}$（$n$ 充分大）。

则 $|x-y| = \frac{\delta}{2} < \delta$。

但：
$$|f(x) - f(y)| = |n^2 - (n + \delta/2)^2| = |n\delta + \delta^2/4|$$

当 $n > \frac{1 - \delta^2/4}{\delta}$，有 $|f(x) - f(y)| > 1 = \varepsilon$。

因此 $f$ 在 $[0, +\infty)$ 上不一致连续。 $\square$

---

### (b) $f(x) = \sqrt{x}$ 的一致连续性

**结论**: 在 $[0, +\infty)$ 上一致连续。

**证明**:

**方法1: 分段估计**

**在 $[0, 1]$ 上**: $f$ 连续，$[0,1]$ 紧致，故一致连续。

**在 $[1, +\infty)$ 上**: 对 $x, y \geq 1$：
$$|\sqrt{x} - \sqrt{y}| = \frac{|x-y|}{\sqrt{x} + \sqrt{y}} \leq \frac{|x-y|}{2}$$

由Lipschitz条件，一致连续。

由于两个区间重叠于1，整体上一致连续。

**方法2: 直接证明**

对任意 $x, y \geq 0$：
$$|\sqrt{x} - \sqrt{y}|^2 \leq |\sqrt{x} - \sqrt{y}|(\sqrt{x} + \sqrt{y}) = |x-y|$$

因此 $|\sqrt{x} - \sqrt{y}| \leq \sqrt{|x-y|}$。

给定 $\varepsilon > 0$，取 $\delta = \varepsilon^2$。

当 $|x-y| < \delta$：
$$|\sqrt{x} - \sqrt{y}| \leq \sqrt{|x-y|} < \sqrt{\varepsilon^2} = \varepsilon$$

一致连续得证。 $\square$

---

### (c) $f(x) = \sin\frac{1}{x}$ 的一致连续性

**结论**: 在 $(0, 1]$ 上不一致连续。

**证明**:

取两个序列：
$$x_n = \frac{1}{2n\pi + \pi/2}, \quad y_n = \frac{1}{2n\pi}$$

则：
- $f(x_n) = \sin(2n\pi + \pi/2) = 1$
- $f(y_n) = \sin(2n\pi) = 0$

**计算距离**:
$$|x_n - y_n| = \frac{1}{2n\pi} - \frac{1}{2n\pi + \pi/2} = \frac{\pi/2}{2n\pi(2n\pi + \pi/2)} \to 0$$

但 $|f(x_n) - f(y_n)| = 1$ 不趋于0。

因此 $f$ 在 $(0, 1]$ 上不一致连续。 $\square$

---

## 证明技巧总结

| 技巧 | 应用位置 | 说明 |
|------|----------|------|
| Lipschitz条件 | (a)(b) | $|f(x)-f(y)| \leq L|x-y|$ 蕴含一致连续 |
| 紧致集性质 | (a) | 连续函数在紧致集上一致连续 |
| 反例构造 | (a)(c) | 找序列使距离趋于0但函数值距离不趋于0 |
| Hölder连续 | (b) | $|\sqrt{x}-\sqrt{y}| \leq |x-y|^{1/2}$ |

### 关键洞察

1. **有界区间**: 有界闭区间上的连续函数必一致连续
2. **无穷区间**: 需要增长条件（如Lipschitz或缓慢变化）
3. **振荡函数**: 高频振荡可能导致不一致连续

## 常见错误

### 错误1: 混淆连续与一致连续

❌ **错误做法**:
> $x^2$ 在$\mathbb{R}$上连续，故一致连续

✅ **正确理解**:
> 连续不一定一致连续，一致连续要求更强的全局控制

### 错误2: $\delta$ 依赖于点

❌ **错误做法**:
> 证明$x^2$时，取$\delta = \varepsilon/(2|x|)$，依赖于$x$

✅ **正确做法**:
> 一致连续要求$\delta$仅依赖于$\varepsilon$

### 错误3: 序列选取不当

❌ **错误做法**:
> 在(c)中，选$x_n = 1/n$，$y_n = 1/(n+1)$，但这可能不够

✅ **正确做法**:
> 选择使函数值差异最大化的序列
