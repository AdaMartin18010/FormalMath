# 奇异积分算子

## 题目

设 $K$ 是 $\mathbb{R}^n \setminus \{0\}$ 上的函数，满足：
- **光滑性**：$|\partial^\alpha K(x)| \leq C_\alpha |x|^{-n-|\alpha|}$
- **消失性**：$\int_{r < |x| < R} K(x) dx = 0$（对所有 $0 < r < R$）

定义奇异积分算子（主值意义）：
$$Tf(x) = \text{p.v.} \int K(x-y) f(y) dy = \lim_{\varepsilon \to 0} \int_{|x-y| > \varepsilon} K(x-y) f(y) dy$$

**(a)** 证明：对 $f \in C_c^\infty(\mathbb{R}^n)$，主值积分存在。

**(b)** 证明 $L^2$ 有界性：$\|Tf\|_{L^2} \leq C \|f\|_{L^2}$。

**(c)** 证明Hilbert变换 $Hf(x) = \text{p.v.} \frac{1}{\pi} \int \frac{f(y)}{x-y} dy$ 的 $L^p$ 有界性（$1 < p < \infty$）。

## 解答

### (a) 主值积分的存在性

**证明：**

写：
$$\int_{|x-y| > \varepsilon} K(x-y)f(y)dy = \int_{|z| > \varepsilon} K(z)f(x-z)dz$$

$$= \int_{\varepsilon < |z| < 1} K(z)[f(x-z) - f(x)]dz + \int_{|z| \geq 1} K(z)f(x-z)dz$$
（利用消失性加常数项）

第一项：$|f(x-z) - f(x)| \leq \|\nabla f\|_\infty |z|$，故
$$\left|\int_{\varepsilon < |z| < 1} K(z)[f(x-z) - f(x)]dz\right| \leq C \|\nabla f\|_\infty \int_{|z| < 1} |z|^{1-n} dz < \infty$$

第二项因 $f$ 紧支集，积分域有界。

因此主值极限存在。$\square$

### (b) $L^2$ 有界性

**证明概要：**

**Step 1**：Fourier变换。

形式计算：$\widehat{Tf}(\xi) = m(\xi) \hat{f}(\xi)$，其中 $m = \hat{K}$（分布意义）。

**Step 2**：乘子有界性。

由 $K$ 的齐次性和消失条件，$m$ 是零次齐次函数。

计算：对 $\lambda > 0$，$K(\lambda x) = \lambda^{-n} K(x)$，故 $m$ 齐次。

$m$ 在球面 $S^{n-1}$ 光滑，故有界。

**Step 3**：由Plancherel：
$$\|Tf\|_{L^2} = \|m \hat{f}\|_{L^2} \leq \|m\|_\infty \|\hat{f}\|_{L^2} = \|m\|_\infty \|f\|_{L^2}$$
$\square$

### (c) Hilbert变换的$L^p$有界性

**证明概要：**

Hilbert变换的核 $K(x) = \frac{1}{\pi x}$。

**Step 1**：C-Z分解。

对 $f \in L^p$，$\alpha > 0$，做Calderón-Zygmund分解 $f = g + b$。

**Step 2**：好函数估计。

$\|g\|_{L^2}^2 \leq C\alpha^{2-p}\|f\|_{L^p}^p$，$\|g\|_{L^\infty} \leq C\alpha$。

$$|\{x : |Hg(x)| > \alpha/2\}| \leq \frac{4}{\alpha^2}\|Hg\|_{L^2}^2 \leq \frac{C}{\alpha^2}\|g\|_{L^2}^2 \leq \frac{C}{\alpha^p}\|f\|_{L^p}^p$$

**Step 3**：坏函数估计。

$b = \sum_j b_j$，支撑于不交方体 $Q_j$，$\int b_j = 0$。

$$Hb(x) = \sum_j \int K(x-y) b_j(y) dy$$

对 $x \notin Q_j^*$（$Q_j$ 的适当扩张）：
$$\int_{Q_j} K(x-y)b_j(y)dy = \int_{Q_j} [K(x-y) - K(x-y_j)]b_j(y)dy$$

由Hörmander条件：
$$\int_{(Q_j^*)^c} |Hb_j| \leq C\|b_j\|_{L^1}$$

因此 $|\{Hb > \alpha/2\}|$ 可控。

**Step 4**：Marcinkiewicz插值得 $L^p$ 有界性。$\square$

## 证明技巧

1. **消失性利用**：主值积分收敛的关键
2. **Fourier乘子**：$L^2$理论的代数工具
3. **C-Z分解**：$L^p$估计的标准方法

## 常见错误

- ❌ 主值积分与Lebesgue积分混淆
- ❌ Hörmander条件的验证疏漏
- ❌ 插值端点$p=1,\infty$的排除
