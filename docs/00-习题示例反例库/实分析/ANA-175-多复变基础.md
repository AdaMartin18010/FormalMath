# 多复变基础

## 题目

**(a)** 证明 Hartogs 现象：在 $\mathbb{C}^n$ ($n \geq 2$) 中，存在可以全纯延拓到更大区域上的函数。

**(b)** 证明多复变情形的 Cauchy 积分公式。

**(c)** 讨论多圆盘与球域的全纯不等价性。

## 解答

### (a) Hartogs 现象

例子：$f(z_1, z_2) = \frac{1}{z_1 z_2 - 1}$ 在 $H = \{(z_1,z_2): |z_1|<1, |z_2|<1, |z_1z_2|>1\}$ 上全纯，可延拓到整个双圆盘。

### (b) 多复变 Cauchy 公式

对多圆盘 $P = \prod_{j=1}^n D(z_j^0, r_j)$：
$$f(z) = \frac{1}{(2\pi i)^n} \int_{|\zeta_j-z_j^0|=r_j} \frac{f(\zeta)}{\prod_{j=1}^n (\zeta_j-z_j)} d\zeta_1 \cdots d\zeta_n$$

### (c) 全纯不等价

球域具有光滑边界，多圆盘边界不光滑，两者双全纯不等价。
