# ANA-011: 夹逼定理求极限

**题号**: ANA-011
**难度**: ⭐⭐ (Level 2)
**类型**: 计算型 (CALC)
**来源**: MIT 18.100A Chapter 2.3
**主题**: 夹逼定理应用

---

## 题目

利用夹逼定理计算下列极限：

**(a)** 计算 $\displaystyle\lim_{n\to\infty} \sum_{k=1}^{n} \frac{1}{n^2 + k}$

**(b)** 计算 $\displaystyle\lim_{n\to\infty} \frac{1}{n} \sum_{k=1}^{n} \sin\left(\frac{k\pi}{n}\right)$

**(c)** 设 $a > 0$，计算 $\displaystyle\lim_{n\to\infty} \sqrt[n]{a^n + b^n}$（其中 $0 < b \leq a$）

---

## 解答

### (a) 求 $\displaystyle\lim_{n\to\infty} \sum_{k=1}^{n} \frac{1}{n^2 + k}$

**分析**: 这是 $n$ 项求和，每项都很小，需要找到合适的上下界。

**证明**:

对 $k = 1, 2, \ldots, n$，有：
$$n^2 + 1 \leq n^2 + k \leq n^2 + n$$

因此：
$$\frac{1}{n^2 + n} \leq \frac{1}{n^2 + k} \leq \frac{1}{n^2 + 1}$$

对 $k = 1$ 到 $n$ 求和：
$$\sum_{k=1}^{n} \frac{1}{n^2 + n} \leq \sum_{k=1}^{n} \frac{1}{n^2 + k} \leq \sum_{k=1}^{n} \frac{1}{n^2 + 1}$$

即：
$$\frac{n}{n^2 + n} \leq \sum_{k=1}^{n} \frac{1}{n^2 + k} \leq \frac{n}{n^2 + 1}$$

**计算边界**:

- 下界：$\displaystyle\frac{n}{n^2 + n} = \frac{1}{n + 1} \to 0$（当 $n \to \infty$）
- 上界：$\displaystyle\frac{n}{n^2 + 1} = \frac{1/n}{1 + 1/n^2} \to 0$（当 $n \to \infty$）

由夹逼定理：
$$\lim_{n\to\infty} \sum_{k=1}^{n} \frac{1}{n^2 + k} = 0 \quad \square$$

---

### (b) 求 $\displaystyle\lim_{n\to\infty} \frac{1}{n} \sum_{k=1}^{n} \sin\left(\frac{k\pi}{n}\right)$

**分析**: 这是Riemann和的形式，可以用积分估计。

**证明**:

注意到对 $x \in [0, \pi]$，函数 $\sin x$ 满足：
$$0 \leq \sin x \leq 1$$

更精确的估计：当 $\frac{k-1}{n}\pi \leq x \leq \frac{k}{n}\pi$ 时，由积分中值定理：
$$\int_{\frac{(k-1)\pi}{n}}^{\frac{k\pi}{n}} \sin x \, dx = \frac{\pi}{n} \sin\xi_k$$

其中 $\xi_k \in \left[\frac{(k-1)\pi}{n}, \frac{k\pi}{n}\right]$。

利用不等式（对 $x \in [0, \frac{\pi}{2}]$）：
$$\frac{2}{\pi}x \leq \sin x \leq x$$

对求和进行估计：
$$\frac{1}{n} \sum_{k=1}^{n} \sin\left(\frac{k\pi}{n}\right)$$

这是积分 $\displaystyle\int_0^1 \sin(\pi x)\, dx$ 的Riemann和：
$$\int_0^1 \sin(\pi x)\, dx = \left[-\frac{\cos(\pi x)}{\pi}\right]_0^1 = \frac{2}{\pi}$$

更严格地，由于 $\sin x$ 在 $[0, \pi]$ 上连续，Riemann和收敛于积分：
$$\lim_{n\to\infty} \frac{1}{n} \sum_{k=1}^{n} \sin\left(\frac{k\pi}{n}\right) = \int_0^1 \sin(\pi x)\, dx = \frac{2}{\pi} \quad \square$$

---

### (c) 求 $\displaystyle\lim_{n\to\infty} \sqrt[n]{a^n + b^n}$（$0 < b \leq a$）

**分析**: 提取主导项是关键技巧。

**证明**:

提取 $a^n$:
$$\sqrt[n]{a^n + b^n} = a \cdot \sqrt[n]{1 + \left(\frac{b}{a}\right)^n}$$

由于 $0 < \frac{b}{a} \leq 1$，有 $\left(\frac{b}{a}\right)^n \to 0$（当 $n \to \infty$）。

因此：
$$\sqrt[n]{1 + \left(\frac{b}{a}\right)^n} \to 1$$

严格用夹逼定理验证：

**下界**：
$$a^n \leq a^n + b^n \Rightarrow a \leq \sqrt[n]{a^n + b^n}$$

**上界**：
$$a^n + b^n \leq a^n + a^n = 2a^n \Rightarrow \sqrt[n]{a^n + b^n} \leq a \cdot \sqrt[n]{2}$$

由于 $\sqrt[n]{2} \to 1$，对任意 $\varepsilon > 0$，存在 $N$ 使得 $n \geq N$ 时：
$$a \leq \sqrt[n]{a^n + b^n} \leq a(1 + \varepsilon)$$

由夹逼定理：
$$\lim_{n\to\infty} \sqrt[n]{a^n + b^n} = a \quad \square$$

## 证明技巧总结

| 技巧 | 应用位置 | 说明 |
|------|----------|------|
| 寻找紧的上下界 | 全部 | 边界要同时收敛到同一极限 |
| 提取主导项 | (c)部分 | 对指数形式极限特别有效 |
| Riemann和转换 | (b)部分 | 求和转化为积分计算 |
| 分母放缩 | (a)部分 | 统一分母便于求和 |

### 关键洞察

1. **夹逼定理的核心**: 找到两个收敛于同一极限的序列作为边界
2. **放缩原则**: 放缩不能过度，上下界要"夹得住"
3. **主导项识别**: 在多项式或指数和式中，找出增长最快的项

## 常见错误

### 错误1: 上下界选择过松

❌ **错误做法**:
> 对(a)部分，直接写 $0 \leq \sum_{k=1}^{n} \frac{1}{n^2 + k} \leq n \cdot \frac{1}{n^2} = \frac{1}{n}$

✅ **正确理解**:
> 虽然结论正确，但估计 $n^2 + k \geq n^2$ 不够紧，应该利用求和项数

### 错误2: 忽视边界收敛验证

❌ **错误做法**:
> 找到上下界后，不验证它们是否收敛于同一极限

✅ **正确做法**:
> 必须验证 $\lim$ 下界 $= \lim$ 上界，否则夹逼定理不适用

### 错误3: Riemann和识别错误

❌ **错误做法**:
> 在(b)部分，直接得出极限为0

✅ **正确理解**:
> $\frac{1}{n}$ 因子与 $n$ 项求和抵消，极限非零，需要用积分计算

## 变式练习

### 变式1（难度⭐⭐）

计算 $\displaystyle\lim_{n\to\infty} \sum_{k=1}^{n} \frac{k}{n^2 + n + k}$

**答案**: $\frac{1}{2}$

**提示**: $\sum_{k=1}^n k = \frac{n(n+1)}{2}$，用上下界估计

### 变式2（难度⭐⭐⭐）

计算 $\displaystyle\lim_{n\to\infty} \frac{1}{n^2} \sum_{k=1}^{n} \sqrt{n^2 - k^2}$

**答案**: $\frac{\pi}{4}$

**提示**: 转化为积分 $\displaystyle\int_0^1 \sqrt{1-x^2}\, dx$

### 变式3（难度⭐⭐⭐）

设 $a_1, a_2, \ldots, a_m > 0$，计算 $\displaystyle\lim_{n\to\infty} \sqrt[n]{a_1^n + a_2^n + \cdots + a_m^n}$

**答案**: $\max\{a_1, a_2, \ldots, a_m\}$

**提示**: 提取最大项
