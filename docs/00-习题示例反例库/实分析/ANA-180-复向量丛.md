# 复向量丛

## 题目

**(a)** 定义全纯向量丛并证明其转移函数满足上循环条件。

**(b)** 证明 Oka-Grauert 原理：Stein 流形上的全纯向量丛与连续向量丛分类相同。

**(c)** 讨论全纯向量丛的陈类。

## 解答

### (a) 全纯向量丛

局部平凡化 $E|_{U_\alpha} \cong U_\alpha \times \mathbb{C}^r$，转移函数 $g_{\alpha\beta}: U_\alpha \cap U_\beta \to GL(r, \mathbb{C})$ 全纯且满足 $g_{\alpha\beta}g_{\beta\gamma}g_{\gamma\alpha} = I$。

### (b) Oka-Grauert 原理

$H^1(X, GL(r, \mathcal{O})) \to H^1(X, GL(r, \mathcal{C}))$ 是同构对 Stein 流形。

### (c) 陈类

通过曲率形式 $c_k(E) = \left(\frac{i}{2\pi}\right)^k \sigma_k(\Theta)$ 定义，其中 $\Theta$ 是陈联络的曲率。
