# 微局部分析

## 题目

**(a)** 定义波前集 WF(u) 并证明其与奇点支集的关系。

**(b)** 证明 Hörmander 的奇点传播定理。

**(c)** 讨论 Fourier 积分算子和典则变换的量化。

## 解答

### (a) 波前集

$(x_0, \xi_0) \notin WF(u)$ 若存在 φ ∈ C_c^∞，φ(x_0)≠0，使得 φu 的 Fourier 变换在 ξ_0 锥邻域速降。

$\pi(WF(u)) = \text{sing supp}(u)$。

### (b) 奇点传播

对主型微分算子 P，WF(u)\WF(Pu) 在次特征流形上不变。

### (c) Fourier 积分算子

与典则变换 χ: T*X → T*Y 关联，形式：$Au(x) = \int e^{i\phi(x,y,\theta)} a(x,y,\theta) u(y) dy d\theta$。
