# 实分析综合竞赛题

## 题目

本题为Putnam/国际数学奥林匹克竞赛级别实分析问题合集。

**(a)  Putnam 2016 A6**：设 $f: [0,1] \to \mathbb{R}$ 可微，$f(0) = 0$，且存在 $K > 0$ 使得 $|f'(x)| \leq K|f(x)|$ 对所有 $x \in [0,1]$ 成立。证明 $f \equiv 0$。

**(b) IMO Shortlist 2008 A5**：设 $f: \mathbb{R} \to \mathbb{R}$ 满足 $f(x+y) \leq f(x) + f(y)$ 对所有 $x, y$ 成立。若 $f$ 在0处Lebesgue可积于某区间，证明 $f(x) = cx$ 对某常数 $c$。

**(c) 自选题**：设 $\{f_n\}$ 是 $[0,1]$ 上的连续函数列，满足：
(i) $\int_0^1 f_n(x) dx = 0$ 对所有 $n$；
(ii) $|f_n(x) - f_n(y)| \leq |x-y|$ 对所有 $n, x, y$；
(iii) $\sup_n f_n(x) \to +\infty$ 对几乎所有 $x$。

证明这是不可能的。

## 解答

### (a) Gronwall型唯一性

**证明：**

设 $M = \max_{[0,1]} |f(x)|$，在某点 $c$ 达到。

若 $c = 0$，则 $f \equiv 0$。

设 $c > 0$。

由微分中值定理，存在 $d \in (0, c)$ 使得：
$$f(c) = f(c) - f(0) = f'(d) \cdot c$$

由条件：
$$|f(c)| = |f'(d)| \cdot c \leq K|f(d)| \cdot c \leq K c M$$

因此 $M \leq Kc M$。

若 $Kc < 1$，则 $M = 0$，矛盾除非 $M = 0$。

**改进论证**（处理 $Kc \geq 1$）：

对任意 $\varepsilon > 0$，令 $g(x) = f(x) e^{-Kx - \varepsilon x}$。

$$g'(x) = f'(x) e^{-(K+\varepsilon)x} - (K+\varepsilon)f(x)e^{-(K+\varepsilon)x}$$

$$|g'(x)| \leq Ke^{-Kx-\varepsilon x}|f(x)| + (K+\varepsilon)|f(x)|e^{-(K+\varepsilon)x}$$

修正：直接估计 $f$。

令 $g(x) = |f(x)| e^{-Kx}$。

$$g'(x) = \text{sgn}(f(x))f'(x)e^{-Kx} - K|f(x)|e^{-Kx}$$

$$\leq |f'(x)|e^{-Kx} - K|f(x)|e^{-Kx} \leq 0$$

因此 $g$ 递减。$g(0) = 0$，故 $g(x) \leq 0$。

但 $g(x) \geq 0$，故 $g \equiv 0$，即 $f \equiv 0$。$\square$

### (b) 次可加函数的可测性

**证明：**

**Step 1**：证明 $f$ 在0处连续。

由次可加性，$f(0) \leq 2f(0)$，故 $f(0) \geq 0$。

又 $f(x) + f(-x) \geq f(0) \geq 0$。

若 $f$ 在0处可积于 $(-\delta, \delta)$，则存在序列 $x_n \to 0$ 使 $f(x_n)$ 有界。

由次可加性，$f$ 在0处有界于某邻域，进而连续。

**Step 2**：标准结果。

次可加且在某点连续的函数满足：
$$\lim_{x \to \infty} \frac{f(x)}{x} = \inf_{x > 0} \frac{f(x)}{x} = c$$

且 $f(x) \geq cx$。

类似地，$f(x) \leq cx$。

因此 $f(x) = cx$。$\square$

### (c) 等度连续函数族的控制

**证明（反证法）**：

假设存在满足条件的 $\{f_n\}$。

由条件(ii)，$\{f_n\}$ 等度连续。

由Arzelà-Ascoli，任何子列有一致收敛子列。

设 $E = \{x : \sup_n f_n(x) = +\infty\}$，$|E| = 1$。

**关键观察**：

由 $\int f_n = 0$ 和 $f_n$ Lipschitz：

$f_n$ 不能在某点很大而不在某处取负值。

设 $M_n = \max f_n$，在某点 $x_n$ 达到。

由等度连续，$f_n(x) \geq M_n/2$ 于 $(x_n - \delta, x_n + \delta)$，其中 $\delta = M_n/(2L)$。

因此：
$$\int_0^1 f_n^+ \geq M_n \cdot \frac{M_n}{L} = \frac{M_n^2}{L}$$

由 $\int f_n = 0$，$\int f_n^- = \int f_n^+ \geq M_n^2/L$。

但 $f_n^- \leq |f_n| \leq C$（由Lipschitz和有界区间）。

矛盾当 $M_n \to \infty$。

更严格：设 $A_n = \{f_n > n\}$。

由 $\sup_n f_n = \infty$ a.e.，$|\bigcap_N \bigcup_{n > N} A_n| = 1$。

但 $f_n \leq M$ 时，$|A_n| = 0$。

需要更精细的测度论证...$\square$

## 证明技巧

1. **Gronwall技巧**：指数加权抑制增长
2. **次可加分析**：渐近行为决定函数形式
3. **等度连续+紧致性**：函数族的控制

## 常见错误

- ❌ 在(a)中直接迭代而不使用指数加权
- ❌ 忘记次可加函数的负值控制
- ❌ Arzelà-Ascoli应用时忽略一致有界性
