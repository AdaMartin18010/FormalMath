# 双曲几何

## 题目

**(a)** 证明单位圆盘的 Poincaré 度规是 Aut(𝔻)-不变的完备度规，曲率 -1。

**(b)** 计算双曲距离并证明 Schwarz-Pick 引理的几何形式。

**(c)** 讨论双曲 Riemann 面的分类（Koebe 统一化）。

## 解答

### (a) Poincaré 度规

$$ds^2 = \frac{4|dz|^2}{(1-|z|^2)^2}$$
曲率 $K = -1$，完备性：沿半径积分发散。

### (b) Schwarz-Pick

全纯映射 f: 𝔻 → 𝔻 是双曲度规的压缩：$f^*ds^2 \leq ds^2$。

### (c) Koebe 统一化

单连通 Riemann 面双全纯于 ℂ、𝔻 或 ℂ̂。
