# Riesz-Thorin插值定理

## 题目

设 $(X, \mu)$，$(Y, \nu)$ 是$\sigma$-有限测度空间，$T$ 是线性算子。

**(a)** 证明Riesz-Thorin插值定理：若
$$T: L^{p_0}(X) \to L^{q_0}(Y), \quad \|T\| \leq M_0$$
$$T: L^{p_1}(X) \to L^{q_1}(Y), \quad \|T\| \leq M_1$$
则对 $0 < \theta < 1$，$\frac{1}{p} = \frac{1-\theta}{p_0} + \frac{\theta}{p_1}$，$\frac{1}{q} = \frac{1-\theta}{q_0} + \frac{\theta}{q_1}$：
$$T: L^p(X) \to L^q(Y), \quad \|T\| \leq M_0^{1-\theta} M_1^\theta$$

**(b)** 用插值定理证明Hausdorff-Young不等式：对 $1 \leq p \leq 2$，$\frac{1}{p} + \frac{1}{q} = 1$：
$$\|\hat{f}\|_{L^q} \leq \|f\|_{L^p}$$

**(c)** 证明Young卷积不等式：对 $1 \leq p, q, r \leq \infty$，$\frac{1}{r} + 1 = \frac{1}{p} + \frac{1}{q}$：
$$\|f * g\|_{L^r} \leq \|f\|_{L^p} \|g\|_{L^q}$$

## 解答

### (a) Riesz-Thorin定理

**证明概要（三线定理）：**

设 $f = \sum_j a_j \chi_{E_j}$，$g = \sum_k b_k \chi_{F_k}$ 是简单函数，$\|f\|_p = \|g\|_{q'} = 1$。

定义：
$$f_z = \sum_j |a_j|^{\alpha(z)} \text{sgn}(a_j) \chi_{E_j}$$
$$g_z = \sum_k |b_k|^{\beta(z)} \text{sgn}(b_k) \chi_{F_k}$$

其中 $\alpha(z) = p(\frac{1-z}{p_0} + \frac{z}{p_1})$，$\beta(z) = q'(\frac{1-z}{q_0'} + \frac{z}{q_1'})$。

设 $F(z) = \int (Tf_z) g_z = \sum_{j,k} |a_j|^{\alpha(z)} |b_k|^{\beta(z)} c_{jk}$。

$F$ 在 $0 < \text{Re}(z) < 1$ 有界解析。

**边界估计**：
- $\text{Re}(z) = 0$：$|F(iy)| \leq M_0$
- $\text{Re}(z) = 1$：$|F(1+iy)| \leq M_1$

由三线定理：$|F(\theta)| \leq M_0^{1-\theta} M_1^\theta$。

$F(\theta) = \int (Tf)g$，取上确界得算子范数界。$\square$

### (b) Hausdorff-Young不等式

**证明：**

Fourier变换：
- $L^1 \to L^\infty$：$\|\hat{f}\|_\infty \leq \|f\|_1$
- $L^2 \to L^2$：$\|\hat{f}\|_2 = \|f\|_2$（Plancherel）

插值：$\frac{1}{p} = \frac{1-\theta}{1} + \frac{\theta}{2} = 1 - \frac{\theta}{2}$。

$\frac{1}{q} = \frac{\theta}{2} = 1 - \frac{1}{p}$，即 $\frac{1}{p} + \frac{1}{q} = 1$。

范数：$\|T\| \leq 1^{1-\theta} \cdot 1^\theta = 1$。$\square$

### (c) Young卷积不等式

**证明：**

固定 $g \in L^q$，考虑 $Tf = f * g$。

- $L^1 \to L^q$：$\|f * g\|_q \leq \|f\|_1 \|g\|_q$
- $L^{q'} \to L^\infty$：$\|f * g\|_\infty \leq \|f\|_{q'} \|g\|_q$

插值：$\frac{1}{p} = \frac{1-\theta}{1} + \frac{\theta}{q'} = 1 - \frac{\theta}{q}$。

$\frac{1}{r} = \frac{1-\theta}{q} + 0 = \frac{1}{q}(1 - \frac{q(1-p^{-1})}{1-q^{-1}}) = \frac{1}{p} + \frac{1}{q} - 1$。$\square$
