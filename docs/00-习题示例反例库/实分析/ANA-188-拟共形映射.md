# 拟共形映射

## 题目

**(a)** 定义 $K$-拟共形映射并证明其分析定义与几何定义的等价性。

**(b)** 证明拟共形映射的紧性定理。

**(c)** 讨论 Beltrami 方程的可解性。

## 解答

### (a) 拟共形映射

几何定义：$K^{-1} \leq \frac{M(\Gamma)}{M(f(\Gamma))} \leq K$。

分析定义：Sobolev 空间 $W^{1,2}_{loc}$ 且 $|\partial_{\bar{z}} f| \leq k |\partial_z f|$，$k = \frac{K-1}{K+1}$。

### (b) 紧性

一致 $K$-拟共形映射族是正规的（Montel 定理推广）。

### (c) Beltrami 方程

$\partial_{\bar{z}} f = \mu(z) \partial_z f$，$\|\mu\|_\infty < 1$。解由 Ahlfors-Bers 理论给出。
