## 题目

设 $a_n = \frac{(-1)^n}{\sqrt{n}} + \frac{1}{n}$，考虑级数 $\sum_{n=1}^{\infty} a_n$。

(1) 判断 $\sum a_n$ 的收敛性（绝对收敛、条件收敛或发散）；

(2) 设 $S = \sum_{n=1}^{\infty} a_n$，$S_N = \sum_{n=1}^{N} a_n$，估计 $|S - S_N|$；

(3) 将级数 $\sum a_n$ 重排，使得重排后的级数发散到 $+\infty$，描述你的构造方法。

## 详细解答

### (1) 收敛性判别

**分解**：$a_n = b_n + c_n$，其中 $b_n = \frac{(-1)^n}{\sqrt{n}}$，$c_n = \frac{1}{n}$。

**分析 $\sum b_n$**：
- 这是交错级数，$\frac{1}{\sqrt{n}}$ 单调递减趋于 $0$
- 由莱布尼茨判别法，$\sum b_n$ **收敛**
- 但 $\sum |b_n| = \sum \frac{1}{\sqrt{n}}$ 发散（$p$-级数，$p=\frac{1}{2}<1$）
- 故 $\sum b_n$ **条件收敛**

**分析 $\sum c_n$**：
- $\sum \frac{1}{n}$ 是调和级数，**发散**到 $+\infty$

**综合**：

$$\sum a_n = \sum b_n + \sum c_n$$

由于 $\sum b_n$ 收敛而 $\sum c_n$ 发散，故 $\sum a_n$ **发散**到 $+\infty$。

**验证**：部分和 $S_N = \sum_{n=1}^N \frac{(-1)^n}{\sqrt{n}} + \sum_{n=1}^N \frac{1}{n}$

- 第一项有界（交错级数部分和有界）
- 第二项 $\sim \ln N \to +\infty$

故 $S_N \to +\infty$。

### (2) 余项估计（修正理解）

实际上 $\sum a_n$ 发散，所以(2)应理解为对截断误差的分析。

设 $T_N = \sum_{n=1}^N b_n$（收敛级数的部分和），$U_N = \sum_{n=1}^N c_n = H_N$（调和数）。

则 $S_N = T_N + H_N$。

由于 $T_N \to T$（某个有限值），而 $H_N \sim \ln N + \gamma$：

$$S_N \sim T + \ln N + \gamma$$

发散速度是对数级的。

若考虑形式上的"余项" $R_N = \sum_{n=N+1}^{\infty} a_n$，这实际上是 $-\infty$（发散）。

**合理解读**：比较 $S_N$ 与 $\ln N$ 的差异。

$$S_N - \ln N = T_N + (H_N - \ln N) \to T + \gamma$$

其中 $\gamma$ 是欧拉常数。

因此 $|S_N - \ln N - (T+\gamma)| \to 0$，收敛速度由 $T_N$ 和 $H_N - \ln N$ 的收敛速度决定。

**$T_N$ 的收敛速度**：对于交错级数，$|T - T_N| \leq |b_{N+1}| = \frac{1}{\sqrt{N+1}}$。

**$H_N$ 的渐近**：$H_N = \ln N + \gamma + \frac{1}{2N} + O(\frac{1}{N^2})$。

因此：

$$S_N = \ln N + (T + \gamma) + O\left(\frac{1}{\sqrt{N}}\right)$$

### (3) 重排构造

虽然原级数已发散，但考虑形式上的重排：将正项提前可以加速发散。

**原级数结构**：

$$a_n = \begin{cases} \frac{1}{\sqrt{n}} + \frac{1}{n} & n \text{ 偶} \\ -\frac{1}{\sqrt{n}} + \frac{1}{n} & n \text{ 奇} \end{cases}$$

对于大 $n$，正项（$n$ 偶）$\approx \frac{1}{\sqrt{n}}$，负项（$n$ 奇）$\approx -\frac{1}{\sqrt{n}}$。

**重排策略**：将所有"正主导"的项排在前面。

观察：当 $n$ 偶，$a_n = \frac{1}{\sqrt{n}} + \frac{1}{n} > 0$

当 $n$ 奇，$a_n = -\frac{1}{\sqrt{n}} + \frac{1}{n} = \frac{1-\sqrt{n}}{n} < 0$（对 $n \geq 2$）

所以：$a_1 = 0$，$a_2 = \frac{1}{\sqrt{2}} + \frac{1}{2} > 0$，$a_3 < 0$，$a_4 > 0$，...

**重排方法**：按照正项优先的原则，每次取 $k$ 个正项后跟 $1$ 个负项。

具体地，设正项下标为 $E = \{2, 4, 6, ...\}$，负项下标为 $O = \{3, 5, 7, ...\}$（$a_1=0$ 可忽略）。

重排：取 $a_2, a_4, ..., a_{2k}$（$k$ 个正项），然后 $a_3$（1个负项），再 $a_{2k+2}, ..., a_{4k}$，然后 $a_5$，依此类推。

**验证发散**：

每组（$k$ 正 + 1 负）的和：

$$\sum_{j=1}^{k} \left(\frac{1}{\sqrt{2j}} + \frac{1}{2j}\right) - \left(\frac{1}{\sqrt{2m+1}} - \frac{1}{2m+1}\right)$$

$\approx \frac{1}{\sqrt{2}} \cdot 2\sqrt{k} + \frac{1}{2}\ln k - \frac{1}{\sqrt{2m}}$

对于足够大的 $k$，每组贡献 $\sim \sqrt{2k}$，趋于无穷。

因此重排级数发散到 $+\infty$，且比原级数更快。

## 证明技巧

1. **分解策略**：将复杂通项分解为易分析的部分
2. **渐近分析**：利用已知级数的渐近行为（如 $H_n \sim \ln n$）
3. **重排构造**：通过控制正负项比例控制级数和的行为

## 常见错误

| 错误类型 | 错误表现 | 正确做法 |
|---------|---------|---------|
| 莱布尼茨滥用 | 直接对 $a_n$ 用莱布尼茨判别法 | 验证单调性：$|a_n|$ 不单调 |
| 忽略符号变化 | 认为正项级数 $\sum\frac{1}{n}$ 决定一切 | 仔细分析各项的实际符号 |
| 重排定理误用 | 认为条件收敛级数可重排到任意值 | 原级数实际发散，需先确认收敛性 |

## 变式练习

**变式1（难度⭐⭐）**：判断 $\sum\left(\frac{(-1)^n}{n} + \frac{1}{n^2}\right)$ 的收敛性。

**变式2（难度⭐⭐⭐）**：设 $a_n = \frac{(-1)^n}{n^{1/3}} + \frac{(-1)^n}{n^{2/3}}$，分析收敛性。

**变式3（难度⭐⭐⭐⭐）**：证明对任何条件收敛级数，存在重排使其发散到 $+\infty$。
