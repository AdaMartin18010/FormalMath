# 奇异积分算子

## 题目

**Calderón-Zygmund核**：$K(x)$ 满足：
(i) $|K(x)| \leq \frac{C}{|x|^n}$；
(ii) $|\nabla K(x)| \leq \frac{C}{|x|^{n+1}}$；
(iii) $\int_{r_1 < |x| < r_2} K(x) dx = 0$。

定义奇异积分：
$$Tf(x) = \text{p.v.} \int_{\mathbb{R}^n} K(x-y) f(y) dy = \lim_{\varepsilon \to 0} \int_{|x-y| > \varepsilon} K(x-y) f(y) dy$$

(a) 证明主值积分对 $f \in C_c^\infty$ 存在。

(b) 证明 $T$ 的 $L^2$ 有界性（Plancherel定理）。

(c) 证明 $T$ 的弱(1,1)有界性，从而 $(p,p)$ 有界性。

## 解答

### (a) 主值存在性

**证明：**

对 $f \in C_c^\infty$，写：
$$\int_{|x-y| > \varepsilon} K(x-y) f(y) dy = \int_{|x-y| > 1} K(x-y) f(y) dy + \int_{\varepsilon < |x-y| < 1} K(x-y) f(y) dy$$

**第一项**：由 $|K(x-y)| \leq C$，积分绝对收敛。

**第二项**：用消失条件：
$$\int_{\varepsilon < |z| < 1} K(z) f(x-z) dz = \int_{\varepsilon < |z| < 1} K(z) (f(x-z) - f(x)) dz$$

（因 $\int_{\varepsilon < |z| < 1} K(z) dz = 0$）

由 $f$ 光滑，$|f(x-z) - f(x)| \leq C|z|$。

$$\left|\int_{\varepsilon < |z| < 1} K(z) (f(x-z) - f(x)) dz\right| \leq \int_{|z| < 1} \frac{C}{|z|^n} \cdot C|z| dz < \infty$$

故主值存在。$\square$

### (b) L²有界性

**证明：**

由Plancherel：$\|Tf\|_2 = \|\widehat{Tf}\|_2$。

形式地：$\widehat{Tf}(\xi) = \hat{K}(\xi) \hat{f}(\xi)$。

需证 $\hat{K} \in L^\infty$。

$$\hat{K}(\xi) = \lim_{\varepsilon \to 0, R \to \infty} \int_{\varepsilon < |x| < R} K(x) e^{-ix\cdot\xi} dx$$

由消失条件，可减 $K(x)$：
$$= \lim \int_{\varepsilon < |x| < R} K(x) (e^{-ix\cdot\xi} - 1) dx$$

分区域估计：$|x| < 1/|\xi|$ 和 $|x| > 1/|\xi|$。

用光滑性条件，可得 $|\hat{K}(\xi)| \leq C$。

因此 $\|Tf\|_2 \leq C\|f\|_2$。$\square$

### (c) 弱(1,1)与(p,p)有界性

**证明概要：**

用CZ分解：$f = g + b$，其中 $g$（好函数）和 $b$（坏函数）。

**好函数**：$|g| \leq 2\lambda$，$\|g\|_2^2 \leq C\lambda\|f\|_1$。

$$|\{Tg > \lambda\}| \leq \frac{\|Tg\|_2^2}{\lambda^2} \leq \frac{C\|g\|_2^2}{\lambda^2} \leq \frac{C\|f\|_1}{\lambda}$$

**坏函数**：$b = \sum b_j$，$\int b_j = 0$，支集在方体 $Q_j$。

用Hörmander条件（光滑性）证明：
$$\int_{(2Q_j)^c} |Tb_j| \leq C\|b_j\|_1$$

求和得 $|\{Tb > \lambda\}| \leq \frac{C\|f\|_1}{\lambda}$。

插值得 $(p,p)$。$\square$

## 证明技巧

1. **消失条件**：保证奇性可积
2. **Fourier变换**：$L^2$ 理论的基石
3. **CZ分解**：坏函数的精细处理

## 常见错误

- ❌ 忘记验证主值积分的存在性
- ❌ Fourier变换计算中忽略主值处理
- ❌ Hörmander条件的应用条件
