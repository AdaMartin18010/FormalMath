# ANA-039: 中值定理综合应用

**题号**: ANA-039
**难度**: ⭐⭐⭐⭐ (Level 4)
**类型**: 应用型 (APP)
**来源**: MIT 18.100A Chapter 5.2
**主题**: 中值定理的综合运用

---

## 题目

**(a)** 设 $f$ 在 $[a,b]$ 上连续，在 $(a,b)$ 内可导。证明存在 $c \in (a,b)$ 使得
$$f'(c) = \frac{f(c) - f(a)}{b - c}$$

**(b)** 设 $f$ 在 $[0,1]$ 上可导，$f(0) = 0$，$f(1) = 1$。对任意正整数 $n$，证明存在互不相同的 $c_1, \ldots, c_n \in (0,1)$ 使得
$$\sum_{k=1}^n \frac{1}{f'(c_k)} = n$$

**(c)** 设 $f$ 在 $[0,1]$ 上二阶可导，$|f''(x)| \leq 1$，$f(0) = f(1) = 0$。证明 $|f'(0)| + |f'(1)| \leq 1$

---

## 解答

### (a) 比值形式的零点

**证明**:

构造 $g(x) = (b-x)(f(x) - f(a))$。

**验证**:
- $g(a) = (b-a)(f(a) - f(a)) = 0$
- $g(b) = 0 \cdot (f(b) - f(a)) = 0$

由Rolle定理，存在 $c \in (a,b)$ 使得 $g'(c) = 0$。

**计算导数**:
$$g'(x) = -(f(x) - f(a)) + (b-x)f'(x)$$

由 $g'(c) = 0$：
$$(b-c)f'(c) = f(c) - f(a)$$

即：
$$f'(c) = \frac{f(c) - f(a)}{b - c} \quad \square$$

---

### (b) 多点中值定理

**证明**:

**方法**: 构造$n$个区间分别应用中值定理。

取分点 $0 = x_0 < x_1 < \cdots < x_n = 1$ 使得：
$$f(x_k) = \frac{k}{n}, \quad k = 0, 1, \ldots, n$$

（由介值定理，这样的分点存在）

在每个区间 $[x_{k-1}, x_k]$ 上应用LMVT：

存在 $c_k \in (x_{k-1}, x_k)$ 使得：
$$f'(c_k) = \frac{f(x_k) - f(x_{k-1})}{x_k - x_{k-1}} = \frac{1/n}{x_k - x_{k-1}}$$

因此：
$$\frac{1}{f'(c_k)} = n(x_k - x_{k-1})$$

求和：
$$\sum_{k=1}^n \frac{1}{f'(c_k)} = n\sum_{k=1}^n (x_k - x_{k-1}) = n(x_n - x_0) = n \quad \square$$

---

### (c) 边界导数估计

**证明**:

在0处Taylor展开：
$$f(x) = f(0) + f'(0)x + \frac{f''(\xi)}{2}x^2 = f'(0)x + \frac{f''(\xi)}{2}x^2$$

取 $x = 1$：
$$0 = f(1) = f'(0) + \frac{f''(\xi_1)}{2}$$

因此 $f'(0) = -\frac{f''(\xi_1)}{2}$，故 $|f'(0)| \leq \frac{1}{2}$。

类似地，在1处展开：
$$f(x) = f(1) + f'(1)(x-1) + \frac{f''(\xi)}{2}(x-1)^2$$

取 $x = 0$：
$$0 = f(0) = f'(1)(-1) + \frac{f''(\xi_2)}{2}$$

因此 $f'(1) = \frac{f''(\xi_2)}{2}$，故 $|f'(1)| \leq \frac{1}{2}$。

所以：
$$|f'(0)| + |f'(1)| \leq \frac{1}{2} + \frac{1}{2} = 1 \quad \square$$

## 证明技巧总结

| 技巧 | 应用位置 | 说明 |
|------|----------|------|
| 辅助函数 | (a) | 构造$g$使Rolle条件满足 |
| 区间分割 | (b) | 将$[0,1]$分成$n$等函数值区间 |
| Taylor展开 | (c) | 在边界点展开获取导数信息 |
| 反推法 | (b) | 从目标等式反推需要的条件 |

### 关键洞察

1. **辅助函数构造**: 目标是使$g(a) = g(b)$以便用Rolle
2. **多点中值**: 通过适当分割实现多点控制
3. **边界估计**: Taylor展开在边界点的应用

## 常见错误

### 错误1: 辅助函数构造错误

❌ **错误做法**:
> 尝试直接对$f$用Rolle定理

✅ **正确做法**:
> 必须构造新函数满足端点相等

### 错误2: 分点存在性未证

❌ **错误做法**:
> 直接断言等函数值分点存在

✅ **正确做法**:
> 用介值定理说明$\{x: f(x) = k/n\}$非空

### 错误3: 展开点选择错误

❌ **错误做法**:
> 在(c)中在中点展开

✅ **正确做法**:
> 在边界展开才能得到边界导数
