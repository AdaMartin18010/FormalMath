# 调和分析进阶

## 题目

**(a)** 证明T1定理：$T$ 是 $L^2$ 有界的Calderón-Zygmund算子当且仅当 $T(1), T^*(1) \in BMO$ 且 $T$ 弱有界。

**(b)** 证明多项式序列的decoupling定理。

**(c)** 讨论Fourier限制性猜想的最新进展。

---

## 解答

### (a) T1定理

#### Calderón-Zygmund算子

**定义**：线性算子 $T: \mathcal{D} \to \mathcal{D}'$ 是**标准Calderón-Zygmund算子**（CZO），若：

1. **积分核**：存在 $K(x,y)$（$x \neq y$）使得：
$$Tf(x) = \int K(x,y) f(y) dy$$
（在适当意义下）

2. **大小条件**：$|K(x,y)| \leq \frac{C}{|x-y|^n}$

3. **光滑条件**：$|K(x,y) - K(x',y)| \leq C \frac{|x-x'|^\delta}{|x-y|^{n+\delta}}$（$|x-x'| < |x-y|/2$）

且对 $K(y,x)$ 类似。

#### T1定理

**定理**（David-Journé, 1984）：

标准CZO $T$ 可延拓为 $L^2(\mathbb{R}^n)$ 有界算子，当且仅当：

1. **弱有界性**（WBP）：对所有标准化的 bump functions $\phi, \psi$：
$$|\langle T\phi_R^x, \psi_R^x \rangle| \leq C R^n$$
其中 $\phi_R^x(y) = R^{-n}\phi((y-x)/R)$。

2. **$T(1) \in BMO$**

3. **$T^*(1) \in BMO$**

#### BMO空间

**定义**：$f \in BMO(\mathbb{R}^n)$（有界平均振荡），若：
$$\|f\|_{BMO} = \sup_B \frac{1}{|B|} \int_B |f - f_B| < \infty$$
其中 $f_B = \frac{1}{|B|} \int_B f$。

**Fefferman对偶**：$BMO = (H^1)^*$（Hardy空间的对偶）。

#### 证明概要

**步骤1：Haar小波分解**

将 $T$ 用Haar基（或小波基）表示。CZO在小波基下有**几乎对角**的矩阵表示。

**步骤2：Schur引理**

对几乎对角矩阵，用Schur引理证明 $L^2$ 有界性。

**步骤3：$T(1)$ 和 $T^*(1)$ 的处理**

若 $T(1) = 0$，$T^*(1) = 0$，则 $T$ 是**paraproduct**的扰动，可用Littlewood-Paley理论控制。

一般情形：通过paraproduct分解，将 $T$ 写为：
$$T = \pi_{T(1)} + \pi_{T^*(1)}^* + T_0$$

其中 $T_0(1) = T_0^*(1) = 0$，$\pi_b$ 是paraproduct：
$$\pi_b(f) = \sum_Q \langle f, \phi_Q \rangle \psi_Q b_Q$$

paraproduct的 $L^2$ 有界性由 $b \in BMO$ 保证（Carleson测度论证）。

$\square$

#### 应用

T1定理统一了大量已有结果：
- Calderón的交换子 $[A, T]$ 的 $L^2$ 有界性
- Cauchy积分在Lipschitz曲线上的有界性（Coifman-McIntosh-Meyer）
- 边界值问题中的双层位势

### (b) Decoupling定理

#### 设置

设 $\Gamma = \{(t, t^2, \ldots, t^n) : t \in [0,1]\} \subseteq \mathbb{R}^n$ 是**moment curve**。

将 $\Gamma$ 分成 $N$ 段弧 $\theta$，每段长度 $\sim 1/N$。

对 $f_\theta$ Fourier支撑在 $\theta$ 的 $\delta = N^{-n}$-邻域上，问：
$$\left\|\sum_\theta f_\theta\right\|_{L^p(B)} \leq C_\varepsilon N^\varepsilon \left(\sum_\theta \|f_\theta\|_{L^p(w_B)}^2\right)^{1/2}$$

对什么 $p$ 成立？

#### Bourgain-Demeter定理

**定理**（Bourgain-Demeter, 2015）：对 $2 \leq p \leq n(n+1)$：
$$\left\|\sum_\theta f_\theta\right\|_{L^p(w_B)} \leq C_\varepsilon N^\varepsilon \left(\sum_\theta \|f_\theta\|_{L^p(w_B)}^2\right)^{1/2}$$

其中 $B$ 是 $\mathbb{R}^n$ 中的球，半径 $N^n$。

**关键指数**：$p_{crit} = n(n+1)$ 是 **critical exponent**，由 **dimension counting** 决定：
- 弧的切空间是1维的
- 在 $L^p$ 中，需要 $p \leq \frac{2 \cdot \text{codim}}{\text{dim}} + 2 = n(n+1)$

#### 证明思路

**步骤1：归纳**

对抛物面（moment curve的扩张），用**induction on dimension**和**induction on scale**。

**步骤2：平移相互作用**

不同弧 $\theta_i$ 的Fourier支撑几乎是分离的（在 $N^{-n}$ 尺度上）。

用**Kakeya-type estimate**或**bilinear restriction**控制不同弧之间的相互作用。

**步骤3：$l^2$ 到 $L^p$**

对 $p = 2$：由Plancherel，等式成立。

对 $p = \infty$：由三角不等式，$\|\sum f_\theta\|_\infty \leq \sum \|f_\theta\|_\infty$。

对中间 $p$：用插值。

**步骤4：抛物面分解**

对高维抛物面 $P^{n-1} = \{(
\xi, |\xi|^2) : \xi \in [0,1]^{n-1}\}$，类似定理成立，critical exponent $p = \frac{2(n+1)}{n-1}$。

这是Stein-Tomas限制定理的最优形式。

### (c) Fourier限制性猜想

#### 陈述

设 $S \subseteq \mathbb{R}^n$ 是光滑超曲面（非零Gauss曲率），$\sigma$ 是表面测度。

**限制性猜想**：对 $p < \frac{2n}{n+1}$：
$$\|\hat{f}|_S\|_{L^q(S, d\sigma)} \leq C \|f\|_{L^p(\mathbb{R}^n)}$$

其中 $q = \frac{n-1}{n+1} p'$，$p' = p/(p-1)$。

#### 已知结果

**Stein-Tomas定理**（$p \leq \frac{2(n+1)}{n+3}$）：

用Oscillatory integral估计和TT*方法：
$$\|\hat{f}|_S\|_{L^2(S)}^2 = \langle \hat{f}, \hat{f} \rangle_S = \langle f, f * \check{\sigma} \rangle$$

$\check{\sigma}(x)$ 的衰减由 stationary phase 给出：$|\check{\sigma}(x)| \leq C |x|^{-(n-1)/2}$。

故 $\check{\sigma} \in L^r$ 对 $r > \frac{2n}{n-1}$，由Young不等式得限制定理。

#### Bourgain的突破

**Bourgain (1991)**：对 $n = 2$，证明限制定理对 $p < \frac{4}{3} - \varepsilon$ 成立。

用Kakeya极大函数的估计和**multiscale analysis**。

**Wolff (2001)**：$n = 3$，$p < \frac{3}{2} - \varepsilon$。

**Guth (2016)**：多项式方法，$n = 3$，$p < \frac{3}{2}$（几乎最优）。

#### 与Decoupling的联系

**Decoupling $
\Rightarrow$ Restriction**：

对 $f$ 的Fourier变换支撑在 $S$ 的 $\delta$-邻域上，decoupling给出：
$$\|f\|_{L^p} \leq C_\varepsilon \delta^{-\varepsilon} \left(\sum_\theta \|f_\theta\|_{L^p}^2\right)^{1/2}$$

当 $\delta \to 0$（尺度细化），这趋近于限制定理。

**Bourgain-Demeter (2015)**：Decoupling for the paraboloid 推出限制定理在 $p < \frac{2(n+1)}{n+3}$ 成立（即Stein-Tomas范围）。

对更优范围，需要**refined decoupling**或**Kakeya估计**的进一步突破。
