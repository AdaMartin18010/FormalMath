# 遍历理论：熵与混合

## 题目

设 $(X, \mathcal{B}, \mu, T)$ 是保测动力系统。

**(a)** 定义测度熵 $h_\mu(T)$：对有限可测分割 $\mathcal{P} = \{P_1, \ldots, P_k\}$，
$$H(\mathcal{P}) = -\sum_{i=1}^k \mu(P_i) \log \mu(P_i)$$
$$h_\mu(T, \mathcal{P}) = \lim_{n \to \infty} \frac{1}{n} H\left(\bigvee_{i=0}^{n-1} T^{-i}\mathcal{P}\right)$$
证明该极限存在。

**(b)** 证明Bernoulli移位 $T$ 于 $(\{0,1\}^\mathbb{N}, \mu_p)$（$\mu_p$ 是乘积测度，$p$ 概率）的熵为
$$h_\mu(T) = -p\log p - (1-p)\log(1-p)$$

**(c)** 定义**强混合**：$\mu(T^{-n}A \cap B) \to \mu(A)\mu(B)$ 对所有可测 $A, B$。

证明Bernoulli移位是强混合的。

## 解答

### (a) 熵极限的存在性

**证明：**

设 $a_n = H\left(\bigvee_{i=0}^{n-1} T^{-i}\mathcal{P}\right)$。

**次可加性**：$a_{m+n} \leq a_m + a_n$。

因：
$$\bigvee_{i=0}^{m+n-1} T^{-i}\mathcal{P} = \left(\bigvee_{i=0}^{m-1} T^{-i}\mathcal{P}\right) \vee T^{-m}\left(\bigvee_{i=0}^{n-1} T^{-i}\mathcal{P}\right)$$

由熵的单调性和保测性：
$$H(\mathcal{Q} \vee T^{-m}\mathcal{R}) \leq H(\mathcal{Q}) + H(T^{-m}\mathcal{R}) = H(\mathcal{Q}) + H(\mathcal{R})$$

**Fekete引理**：次可加序列 $\frac{a_n}{n}$ 收敛于 $\inf \frac{a_n}{n}$。$\square$

### (b) Bernoulli移位的熵

**解答：**

设 $\mathcal{P} = \{[0], [1]\}$ 是柱集分割（第0坐标）。

对 $\bigvee_{i=0}^{n-1} T^{-i}\mathcal{P}$，每个原子是长度为$n$的柱集 $[a_0, \ldots, a_{n-1}]$。

$$\mu([a_0, \ldots, a_{n-1}]) = p^{k}(1-p)^{n-k}$$

其中 $k = \sum a_i$。

$$H\left(\bigvee_{i=0}^{n-1} T^{-i}\mathcal{P}\right) = -\sum_{\text{所有柱集}} \mu(C) \log \mu(C)$$

$$= -\sum_{k=0}^n \binom{n}{k} p^k(1-p)^{n-k} \log(p^k(1-p)^{n-k})$$

$$= -\sum_{k=0}^n \binom{n}{k} p^k(1-p)^{n-k} [k\log p + (n-k)\log(1-p)]$$

$$= -n p \log p - n(1-p)\log(1-p) = nH(p)$$

因此：
$$h_\mu(T, \mathcal{P}) = \lim_{n \to \infty} \frac{nH(p)}{n} = H(p) = -p\log p - (1-p)\log(1-p)$$

$\mathcal{P}$ 是生成元，故 $h_\mu(T) = H(p)$。$\square$

### (c) Bernoulli移位的强混合性

**证明：**

只需证柱集情形（因柱集生成$\sigma$-代数）。

设 $A = [a_0, \ldots, a_{m-1}]$，$B = [b_0, \ldots, b_{k-1}]$。

对 $n > m$：
$$T^{-n}A = \{(x_i) : x_{n+i} = a_i, i = 0, \ldots, m-1\}$$

$$T^{-n}A \cap B = \{(x_i) : x_i = b_i (i < k), x_{n+i} = a_i (i < m)\}$$

由乘积测度的独立性：
$$\mu(T^{-n}A \cap B) = \mu(B) \cdot \mu(A) = \mu(A)\mu(B)$$

对 $n \geq k$，坐标集 $\{0, \ldots, k-1\}$ 和 $\{n, \ldots, n+m-1\}$ 不交，故测度乘积。

因此 $\mu(T^{-n}A \cap B) = \mu(A)\mu(B)$ 对所有 $n \geq k$，极限显然成立。$\square$

## 证明技巧

1. **次可加性论证**：熵序列的核心性质
2. **生成元利用**：熵计算的简化技巧
3. **独立性**：乘积测度的混合性质来源

## 常见错误

- ❌ 熵定义中底数选取（2或e）不一致
- ❌ 柱集测度计算时忽略二项分布
- ❌ 强混合与遍历性的关系混淆
