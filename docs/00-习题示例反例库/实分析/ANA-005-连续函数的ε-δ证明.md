# ANA-005: 连续函数的ε-δ证明

## 题目陈述

**定义**: 函数 $f: D \to \mathbb{R}$ 在点 $c \in D$ 连续，如果对任意 $\varepsilon > 0$，存在 $\delta > 0$ 使得
$$|f(x) - f(c)| < \varepsilon \quad \text{对所有 } x \in D \text{ 满足 } |x - c| < \delta$$

**问题**: 用 $\varepsilon$-$\delta$ 方法证明下列函数的连续性：

1. $f(x) = 3x + 2$ 在 $\mathbb{R}$ 上连续
2. $f(x) = x^2$ 在 $\mathbb{R}$ 上连续
3. $f(x) = \frac{1}{x}$ 在 $(0, \infty)$ 上连续

并讨论逐点连续与一致连续的区别。

## 详细解答

### 例1：$f(x) = 3x + 2$

**目标**: 对任意 $c \in \mathbb{R}$，证明 $f$ 在 $c$ 连续。

**分析**:

$$|f(x) - f(c)| = |(3x + 2) - (3c + 2)| = 3|x - c|$$

要使 $|f(x) - f(c)| < \varepsilon$，只需 $3|x - c| < \varepsilon$，即 $|x - c| < \frac{\varepsilon}{3}$。

**证明**:

对任意 $\varepsilon > 0$，取 $\delta = \frac{\varepsilon}{3} > 0$。

当 $|x - c| < \delta$ 时：
$$|f(x) - f(c)| = 3|x - c| < 3 \cdot \frac{\varepsilon}{3} = \varepsilon$$

故 $f$ 在 $c$ 连续。由 $c$ 的任意性，$f$ 在 $\mathbb{R}$ 上连续。 $\square$

**关键观察**: $\delta$ 只依赖于 $\varepsilon$，与 $c$ 无关 $\Rightarrow$ **一致连续**

---

### 例2：$f(x) = x^2$

**目标**: 证明 $f$ 在任意 $c \in \mathbb{R}$ 连续。

**分析**:

$$|f(x) - f(c)| = |x^2 - c^2| = |x + c| \cdot |x - c|$$

困难：$|x + c|$ 随 $c$ 变化，可能很大。

**技巧**: 先限制 $|x - c| < 1$（或任何固定数），则 $x \in (c-1, c+1)$，故
$$|x + c| \leq |x| + |c| < |c| + 1 + |c| = 2|c| + 1$$

于是 $|f(x) - f(c)| < (2|c| + 1)|x - c|$。

要使上式 $< \varepsilon$，需要 $|x - c| < \frac{\varepsilon}{2|c| + 1}$。

**综合**: 取 $\delta = \min\{1, \frac{\varepsilon}{2|c| + 1}\}$

**证明**:

对任意 $c \in \mathbb{R}$ 和 $\varepsilon > 0$，取
$$\delta = \min\left\{1, \frac{\varepsilon}{2|c| + 1}\right\}$$

当 $|x - c| < \delta$ 时：

- 由 $\delta \leq 1$，有 $|x| < |c| + 1$，故 $|x + c| \leq |x| + |c| < 2|c| + 1$
- 因此
$$|f(x) - f(c)| = |x + c| \cdot |x - c| < (2|c| + 1) \cdot \frac{\varepsilon}{2|c| + 1} = \varepsilon$$

故 $f$ 在 $c$ 连续。 $\square$

**关键观察**: $\delta$ 依赖于 $\varepsilon$ **和** $c$ $\Rightarrow$ 只是**逐点连续**

---

### 例3：$f(x) = \frac{1}{x}$ on $(0, \infty)$

**目标**: 证明 $f$ 在任意 $c > 0$ 连续。

**分析**:

$$|f(x) - f(c)| = \left|\frac{1}{x} - \frac{1}{c}\right| = \frac{|x - c|}{xc}$$

困难：分母 $xc$ 可以任意小（当 $c$ 或 $x$ 接近0时）。

**技巧**: 先限制 $x$ 远离0。设 $|x - c| < \frac{c}{2}$，则 $x > \frac{c}{2}$，故 $xc > \frac{c^2}{2}$。

于是
$$|f(x) - f(c)| < \frac{|x - c|}{\frac{c^2}{2}} = \frac{2|x - c|}{c^2}$$

要使上式 $< \varepsilon$，需要 $|x - c| < \frac{\varepsilon c^2}{2}$。

**综合**: 取 $\delta = \min\{\frac{c}{2}, \frac{\varepsilon c^2}{2}\}$

**证明**:

对任意 $c > 0$ 和 $\varepsilon > 0$，取
$$\delta = \min\left\{\frac{c}{2}, \frac{\varepsilon c^2}{2}\right\}$$

当 $|x - c| < \delta$ 时：

- 由 $\delta \leq \frac{c}{2}$，有 $x > c - \frac{c}{2} = \frac{c}{2}$
- 因此 $xc > \frac{c^2}{2}$
- 故
$$|f(x) - f(c)| = \frac{|x - c|}{xc} < \frac{\delta}{\frac{c^2}{2}} \leq \frac{\frac{\varepsilon c^2}{2}}{\frac{c^2}{2}} = \varepsilon$$

故 $f$ 在 $c$ 连续。 $\square$

**关键观察**: $\delta = O(c^2 \varepsilon)$，当 $c \to 0^+$ 时 $\delta$ 必须趋于0。

---

## 逐点连续 vs 一致连续

### 定义对比

| | 逐点连续 | 一致连续 |
|---|---------|---------|
| **定义** | $\forall c, \forall \varepsilon > 0, \exists \delta > 0$ | $\forall \varepsilon > 0, \exists \delta > 0, \forall c$ |
| **$\delta$ 依赖** | 可依赖于 $c$ 和 $\varepsilon$ | 只能依赖于 $\varepsilon$ |
| **顺序** | $\forall c$, 后 $\exists \delta$ | $\exists \delta$, 后 $\forall c$ |

### 函数分类

| 函数 | 逐点连续 | 一致连续 | 说明 |
|------|---------|---------|------|
| $3x + 2$ | ✓ | ✓ | 线性函数 |
| $x^2$ on $\mathbb{R}$ | ✓ | ✗ | 增长太快 |
| $x^2$ on $[0, 1]$ | ✓ | ✓ | 紧集上连续 |
| $\frac{1}{x}$ on $(0, 1]$ | ✓ | ✗ | 在0附近震荡剧烈 |
| $\frac{1}{x}$ on $[a, \infty), a > 0$ | ✓ | ✓ | 远离0后"变平" |
| $\sin x$ on $\mathbb{R}$ | ✓ | ✓ | Lipschitz连续 |

### 核心定理

**定理**: 若 $f$ 在紧集 $K$ 上连续，则 $f$ 在 $K$ 上一致连续。

## 证明技巧总结

### ε-δ证明的标准流程

```
1. 分析阶段: |f(x) - f(c)| = ... = (含|x-c|的表达式)
2. 放缩技巧: 限制|x-c| < 某常数，控制其他因子
3. 求解δ: 从放缩后的不等式解出δ关于ε的表达式
4. 综合取值: δ = min{限制常数, ε的函数}
5. 严格书写: 按定义格式写出完整证明
```

### 常见放缩技巧

| 技巧 | 适用场景 | 例子 |
|------|---------|------|
| **限制 $|x-c|$** | 控制 $x$ 的范围 | 例2、例3 |
| **因式分解** | 多项式函数 | $x^2 - c^2 = (x+c)(x-c)$ |
| **有理化** | 根式函数 | $\sqrt{x} - \sqrt{c} = \frac{x-c}{\sqrt{x}+\sqrt{c}}$ |
| **有界性** | 三角函数 | $|\sin x - \sin c| \leq |x - c|$ |

### 判定一致连续的实用方法

```
一致连续 ⇔ 不会"无限变陡"

检验方法:
1. 若 f' 有界，则 f 一致连续（Lipschitz）
2. 若在无穷远处"变平"，可能一致连续
3. 若存在使 |f'|→∞ 的点，通常不一致连续
```

## 常见错误分析

### 错误1：δ选取不依赖c

```
❌ 错误：对 f(x)=x²，取 δ = ε/100

✅ 正确：δ 必须依赖于 c，如 δ = min{1, ε/(2|c|+1)}

💡 反例：当 c=100 时，x=100.01 满足|x-c|<δ但|f(x)-f(c)|>ε
```

### 错误2：未限制|x-c|的范围

```
❌ 错误：对 f(x)=1/x，直接用 |f(x)-f(c)| = |x-c|/(xc)

✅ 正确：必须先限制 x 的范围以控制 1/(xc)

💡 危险：若 x→0，则 1/(xc)→∞，无法直接控制
```

### 错误3：混淆存在与任意

```
❌ 错误："对所有δ，存在ε使得..."

✅ 正确："对任意ε，存在δ使得..."

💡 记忆：ε是挑战者给出的"误差容忍度"，δ是我们回应的"输入精度"
```

### 错误4：δ依赖于x而非c

```
❌ 错误：在证明中让 δ 随 x 变化

✅ 正确：对固定的 c，δ 只能是 ε 和 c 的函数

💡 δ定义中的"存在"是在 x 变化之前
```

## 变式练习

### 变式1（难度⭐⭐）

用 $\varepsilon$-$\delta$ 方法证明 $f(x) = \sqrt{x}$ 在 $[0, \infty)$ 上连续。

<details>
<summary>提示</summary>
对 $c = 0$：直接估计 $\sqrt{x} < \varepsilon$ 当 $x < \varepsilon^2$

对 $c > 0$：有理化
$$|\sqrt{x} - \sqrt{c}| = \frac{|x - c|}{\sqrt{x} + \sqrt{c}} \leq \frac{|x - c|}{\sqrt{c}}$$

取 $\delta = \varepsilon\sqrt{c}$（对 $c>0$）或 $\delta = \varepsilon^2$（对 $c=0$）
</details>

### 变式2（难度⭐⭐⭐）

证明 $f(x) = x^2$ 在 $\mathbb{R}$ 上**不一致连续**。

<details>
<summary>提示</summary>
<strong>反证法</strong>：假设一致连续，取 $\varepsilon = 1$。

则存在 $\delta > 0$ 使得 $|x - y| < \delta \Rightarrow |x^2 - y^2| < 1$。

取 $x = n$，$y = n + \frac{\delta}{2}$，其中 $n$ 充分大。

则 $|x - y| = \frac{\delta}{2} < \delta$，但
$$|x^2 - y^2| = |n^2 - (n + \frac{\delta}{2})^2| = n\delta + \frac{\delta^2}{4} > n\delta$$

当 $n > 1/\delta$ 时，$|x^2 - y^2| > 1 = \varepsilon$，矛盾！ $\square$
</details>

### 变式3（难度⭐⭐⭐⭐）

设 $f: \mathbb{R} \to \mathbb{R}$ 满足 Lipschitz 条件：存在 $L > 0$ 使得
$$|f(x) - f(y)| \leq L|x - y| \quad \text{对所有 } x, y \in \mathbb{R}$$

证明：

1. $f$ 在 $\mathbb{R}$ 上一致连续
2. 若 $0 < L < 1$，则 $f$ 有唯一不动点（Banach不动点定理的特殊情形）

<details>
<summary>提示</summary>
1. 对任意 $\varepsilon > 0$，取 $\delta = \varepsilon / L$（若 $L>0$）或任意 $\delta > 0$（若 $L=0$，即 $f$ 常值）

1. 取任意 $x_0$，定义 $x_{n+1} = f(x_n)$。证明 $(x_n)$ 是柯西序列：
$$|x_{n+1} - x_n| \leq L^n |x_1 - x_0|$$
由完备性，$x_n \to x^*$。由连续性，$f(x^*) = x^*$。

唯一性：若 $f(a) = a$，$f(b) = b$，则 $|a - b| = |f(a) - f(b)| \leq L|a - b|$，故 $a = b$。
</details>
