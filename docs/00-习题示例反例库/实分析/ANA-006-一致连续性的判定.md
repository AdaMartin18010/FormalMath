# ANA-006: 一致连续性的判定

## 题目陈述

**定义**: 函数 $f: D \to \mathbb{R}$ 称为**一致连续**，如果对任意 $\varepsilon > 0$，存在 $\delta > 0$ 使得
$$|f(x) - f(y)| < \varepsilon \quad \text{对所有 } x, y \in D \text{ 满足 } |x - y| < \delta$$

**问题**:
1. 判断下列函数的一致连续性：
   - $f(x) = x^2$ on $[0, 1]$ 和 on $\mathbb{R}$
   - $f(x) = \frac{1}{x}$ on $(0, 1]$ 和 on $[1, \infty)$
   - $f(x) = \sin(x^2)$ on $\mathbb{R}$
   - $f(x) = \sqrt{x}$ on $[0, \infty)$

2. 证明：若 $f$ 在 $[a, \infty)$ 上连续且 $\lim_{x \to \infty} f(x)$ 存在，则 $f$ 在 $[a, \infty)$ 上一致连续。

3. 讨论 Lipschitz 条件与一致连续的关系。

## 详细解答

### 第一部分：函数判定

#### 1. $f(x) = x^2$

**在 $[0, 1]$ 上：一致连续 ✓**

**证明**：由 Heine-Cantor 定理，紧集上的连续函数必一致连续。$[0,1]$ 是紧集，$f$ 连续，故一致连续。 $\square$

**直接证明**：对 $x, y \in [0,1]$，
$$|x^2 - y^2| = |x + y| \cdot |x - y| \leq 2|x - y|$$

对任意 $\varepsilon > 0$，取 $\delta = \frac{\varepsilon}{2}$，则 $|x - y| < \delta$ 蕴含 $|f(x) - f(y)| < \varepsilon$。 $\square$

---

**在 $\mathbb{R}$ 上：不一致连续 ✗**

**反证**：假设一致连续，取 $\varepsilon = 1$。

存在 $\delta > 0$ 使得 $|x - y| < \delta \Rightarrow |x^2 - y^2| < 1$。

取 $x_n = n + \frac{\delta}{2}$，$y_n = n$，则 $|x_n - y_n| = \frac{\delta}{2} < \delta$。

但 $|x_n^2 - y_n^2| = |2n\cdot\frac{\delta}{2} + \frac{\delta^2}{4}| > n\delta$。

当 $n > \frac{1}{\delta}$ 时，$|f(x_n) - f(y_n)| > 1 = \varepsilon$，矛盾！ $\square$

**直观**：抛物线"越来越陡"，任意固定的 $\delta$ 在远处都会失效。

---

#### 2. $f(x) = \frac{1}{x}$

**在 $(0, 1]$ 上：不一致连续 ✗**

**证明**：取 $x_n = \frac{1}{n}$，$y_n = \frac{1}{n+1}$。

则 $|x_n - y_n| = \frac{1}{n(n+1)} \to 0$，但
$$|f(x_n) - f(y_n)| = |n - (n+1)| = 1$$

若一致连续，则 $|f(x_n) - f(y_n)| \to 0$，矛盾。 $\square$

---

**在 $[1, \infty)$ 上：一致连续 ✓**

**证明**：对 $x, y \geq 1$：
$$\left|\frac{1}{x} - \frac{1}{y}\right| = \frac{|x - y|}{xy} \leq |x - y|$$

取 $\delta = \varepsilon$ 即可。 $\square$

**关键**：远离0后，函数"变平"了。

---

#### 3. $f(x) = \sin(x^2)$ on $\mathbb{R}$

**判定：不一致连续 ✗**

**证明**：取 $x_n = \sqrt{\frac{\pi}{2} + 2\pi n}$，$y_n = \sqrt{2\pi n}$。

计算距离：
$$|x_n - y_n| = \frac{|x_n^2 - y_n^2|}{x_n + y_n} = \frac{\pi/2}{\sqrt{\frac{\pi}{2} + 2\pi n} + \sqrt{2\pi n}} \to 0$$

但 $f(x_n) = \sin(\frac{\pi}{2} + 2\pi n) = 1$，$f(y_n) = \sin(2\pi n) = 0$。

故 $|f(x_n) - f(y_n)| = 1$，不趋于0。 $\square$

**直观**：振荡越来越快，频率无限增大。

---

#### 4. $f(x) = \sqrt{x}$ on $[0, \infty)$

**判定：一致连续 ✓**

**证明**：注意 $\sqrt{x}$ 在 $[0, 1]$ 上一致连续（紧集），在 $[1, \infty)$ 上：
$$|\sqrt{x} - \sqrt{y}| = \frac{|x - y|}{\sqrt{x} + \sqrt{y}} \leq \frac{|x - y|}{2}$$

也是一致连续。

对任意 $\varepsilon > 0$，存在 $\delta_1$（对 $[0,2]$）和 $\delta_2$（对 $[1,\infty)$）。

取 $\delta = \min\{\delta_1, \delta_2, 1\}$，则对任意 $|x - y| < \delta$：
- 若 $x, y \in [0, 2]$：用 $\delta_1$
- 若 $x, y \in [1, \infty)$：用 $\delta_2$
- 若一者在 $[0,1]$，一者在 $[1,2]$：两者都在 $[0,2]$ 中

综合得一致连续。 $\square$

**替代证明**：$\sqrt{x}$ 是 $[0,1]$-Hölder 连续的（指数 $\frac{1}{2}$）。

---

### 第二部分：无穷远处极限存在

**定理**: 设 $f$ 在 $[a, \infty)$ 上连续，$\lim_{x \to \infty} f(x) = L$ 存在，则 $f$ 在 $[a, \infty)$ 上一致连续。

**证明**:

**步骤1：控制无穷远处**

由极限定义，对 $\varepsilon > 0$，存在 $M > a$ 使得
$$|f(x) - L| < \frac{\varepsilon}{2} \quad \text{对所有 } x \geq M$$

因此，对 $x, y \geq M$：
$$|f(x) - f(y)| \leq |f(x) - L| + |L - f(y)| < \varepsilon$$

只要 $|x - y|$ 任意（不需要小于某个 $\delta$）！

**步骤2：控制有界区间**

在 $[a, M+1]$ 上，$f$ 连续且区间紧致，故一致连续。

存在 $\delta_1 > 0$ 使得 $x, y \in [a, M+1]$，$|x - y| < \delta_1$ 蕴含 $|f(x) - f(y)| < \varepsilon$。

**步骤3：综合**

取 $\delta = \min\{\delta_1, 1\}$。

对 $|x - y| < \delta$，有三种情形：
1. $x, y \in [a, M]$：由步骤2，$|f(x) - f(y)| < \varepsilon$
2. $x, y \geq M$：由步骤1，$|f(x) - f(y)| < \varepsilon$
3. $x < M < y$：则 $y < M + \delta \leq M + 1$，故 $x, y \in [a, M+1]$，由步骤2

综上，$f$ 一致连续。 $\square$

---

### 第三部分：Lipschitz与一致连续

**定义（Lipschitz连续）**: $f$ 是 Lipschitz 的，如果存在 $L > 0$ 使得
$$|f(x) - f(y)| \leq L|x - y| \quad \text{对所有 } x, y \in D$$

**关系**: 
$$\text{Lipschitz} \Rightarrow \text{一致连续} \Rightarrow \text{连续}$$

且逆命题不成立。

**例1：一致连续但非 Lipschitz**

$f(x) = \sqrt{x}$ on $[0, 1]$。

- 一致连续：已证
- 非 Lipschitz：假设 $|\sqrt{x} - \sqrt{y}| \leq L|x - y|$，取 $y = 0$，则 $\sqrt{x} \leq Lx$，即 $1 \leq L\sqrt{x}$，当 $x \to 0$ 时矛盾。

**例2：Lipschitz但不可导**

$f(x) = |x|$ on $\mathbb{R}$。

- Lipschitz：$||x| - |y|| \leq |x - y|$（取 $L = 1$）
- 在0点不可导

## 证明技巧总结

### 判定一致连续的实用方法

| 方法 | 条件 | 例子 |
|------|------|------|
| **Heine-Cantor** | 紧集 + 连续 | $x^2$ on $[0,1]$ |
| **Lipschitz** | 导数有界 | $\sin x$ on $\mathbb{R}$ |
| **无穷极限** | $\lim_{\pm\infty} f$ 存在 | $\frac{1}{x}$ on $[1,\infty)$ |
| **Hölder** | $|f(x)-f(y)| \leq C|x-y|^\alpha$, $\alpha \in (0,1]$ | $\sqrt{x}$ |

### 判定不一致连续的方法

```
方法：构造序列对 (x_n), (y_n) 使得
  1. |x_n - y_n| → 0
  2. |f(x_n) - f(y_n)| ↛ 0

常见模式：
- 多项式增长过快：x_n = n, y_n = n + 1/n
- 奇点处震荡：x_n = 1/n, y_n = 1/(n+1) 对 1/x
- 振荡频率递增：对 sin(x²), x_n = √(2πn + π/2), y_n = √(2πn)
```

## 常见错误分析

### 错误1：混淆逐点与一致连续
```
❌ 错误："f在R上每点连续，故一致连续"

✅ 正确：逐点连续不能推出一致连续

💡 反例：x² on R
```

### 错误2：错误使用Heine-Cantor
```
❌ 错误：对(0,1]用Heine-Cantor定理

✅ 正确：Heine-Cantor要求紧集，(0,1]非紧

💡 (0,1] = [0,1] \ {0}，缺少极限点
```

### 错误3：忽视无穷远处的行为
```
❌ 错误："f在[a,∞)连续，故一致连续"

✅ 正确：需要额外条件（如极限存在或有界导数）

💡 反例：x² on [0,∞)，sin(x²) on [0,∞)
```

### 错误4：错误估计Lipschitz常数
```
❌ 错误：由f'(x)→∞断定非Lipschitz

✅ 正确：Lipschitz要求|f(x)-f(y)|/|x-y|有界

💡 f(x)=√x的导数在0处无穷，但仍一致连续
```

## 变式练习

### 变式1（难度⭐⭐）
证明：若 $f$ 在 $(a, b)$ 上一致连续，则 $f$ 可以连续延拓到 $[a, b]$。

<details>
<summary>提示</summary>
1. 证明 $\lim_{x \to a^+} f(x)$ 存在：用柯西条件
2. 对任意 $\varepsilon > 0$，存在 $\delta > 0$ 使得 $|x - y| < \delta$ 蕴含 $|f(x) - f(y)| < \varepsilon$
3. 特别地，对 $x, y \in (a, a + \delta)$，$|f(x) - f(y)| < \varepsilon$，故极限存在
4. 类似处理 $b$ 点
5. 定义 $f(a) = \lim_{x \to a^+} f(x)$，$f(b) = \lim_{x \to b^-} f(x)$
</details>

### 变式2（难度⭐⭐⭐）
设 $f: \mathbb{R} \to \mathbb{R}$ 满足：
$$|f(x) - f(y)| \leq |x - y|^2 \quad \text{对所有 } x, y$$

证明 $f$ 是常数函数。

<details>
<summary>提示</summary>
对任意 $x \neq y$，
$$\left|\frac{f(x) - f(y)}{x - y}\right| \leq |x - y|$$

令 $y \to x$，得 $f'(x) = 0$ 对所有 $x$。

故 $f$ 为常数。 $\square$

<strong>注</strong>：这是比Lipschitz更强的条件，称为指数为2的Hölder连续。
</details>

### 变式3（难度⭐⭐⭐⭐）
设 $f: [0, \infty) \to \mathbb{R}$ 连续，且对任意 $\varepsilon > 0$，存在 $M$ 使得 $|f(x) - f(y)| < \varepsilon$ 对所有 $x, y \geq M$ 成立（Cauchy at infinity）。

证明 $f$ 在 $[0, \infty)$ 上一致连续。

<details>
<summary>提示</summary>
这推广了"极限存在"的条件。

证明框架与第二部分相同：
1. 无穷远处：由条件直接控制
2. 有界区间 $[0, M+1]$：用Heine-Cantor
3. 综合：取 $\delta = \min\{\delta_1, 1\}$

<strong>应用</strong>：$f(x) = \sin(\sqrt{x})$ 满足条件（虽无极限，但振荡衰减）。
</details>
