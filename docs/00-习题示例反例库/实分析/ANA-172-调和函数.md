# 调和函数

## 题目

**(a)** 证明调和函数的均值性质：若 $u$ 在 $\Omega$ 上调和，则 $u(z_0) = \frac{1}{2\pi} \int_0^{2\pi} u(z_0 + re^{i\theta}) d\theta$。

**(b)** 证明 Dirichlet 问题在单位圆盘上的解由 Poisson 积分公式给出。

**(c)** 证明 Harnack 不等式：正调和函数满足 $\frac{1-r}{1+r} u(0) \leq u(re^{i\theta}) \leq \frac{1+r}{1-r} u(0)$。

## 解答

### (a) 均值性质

由 Cauchy 积分公式应用于全纯函数 $f = u + iv$。

### (b) Poisson 积分

$$u(re^{i\theta}) = \frac{1}{2\pi} \int_0^{2\pi} \frac{1-r^2}{1-2r\cos(\theta-\phi)+r^2} u(e^{i\phi}) d\phi$$

### (c) Harnack 不等式

利用 Poisson 核的估计：$\frac{1-r}{1+r} \leq P_r(\theta) \leq \frac{1+r}{1-r}$。
