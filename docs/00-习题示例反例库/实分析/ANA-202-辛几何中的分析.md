# 辛几何中的分析

## 题目

**(a)** 证明 Gromov 的拟全纯曲线紧致性定理。

**(b)** 讨论 Arnold 猜想和 Floer 同调的证明。

**(c)** 证明 Gromov 的非挤压定理。

## 解答

### (a) 拟全纯曲线紧致性

能量估计 + 椭圆正则性 + 泡泡分析。Gromov 紧化：主分量 + 泡泡树。

### (b) Arnold 猜想

辛微分同胚 φ 的 Fix(φ) ≥ cuplength + 1。Floer：通过 Hamilton 作用泛函的 Morse 理论。

### (c) 非挤压定理

若 B²ⁿ(R) 辛嵌入到 Z²ⁿ(r) = B²(r) × ℝ²ⁿ⁻²，则 R ≤ r。

证明：Gromov 宽度的辛不变性或拟全纯曲线阻碍。
