# 位势论

## 题目

**(a)** 证明对数位势的连续性原理和支配原理。

**(b)** 定义能量和容量，证明极集是零容量集。

**(c)** 讨论细拓扑和细极限。

## 解答

### (a) 位势原理

连续性原理：紧支撑测度 μ 的位势 U^μ 在支撑上连续则在全空间连续。

支配原理：若 U^μ ≤ U^ν 在 supp(μ) 上，则该不等式处处成立。

### (b) 容量

$$\text{cap}(E) = \sup\{\mu(E) : I(\mu) \leq 1\}$$
极集：对某超调和函数 u，u|E = +∞，则 cap(E) = 0。

### (c) 细拓扑

由细开集生成的拓扑，u 细调和当且仅当在细拓扑下连续。
