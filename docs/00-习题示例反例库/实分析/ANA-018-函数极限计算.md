# ANA-018: 函数极限计算

**题号**: ANA-018
**难度**: ⭐⭐⭐ (Level 3)
**类型**: 计算型 (CALC)
**来源**: MIT 18.100A Chapter 3.1
**主题**: 函数极限的ε-δ证明

---

## 题目

利用 $\varepsilon$-$\delta$ 定义证明下列极限：

**(a)** 证明 $\displaystyle\lim_{x\to 2} x^2 = 4$

**(b)** 证明 $\displaystyle\lim_{x\to 0} \frac{\sin x}{x} = 1$

**(c)** 证明 $\displaystyle\lim_{x\to 1} \frac{x^2-1}{x-1} = 2$

---

## 解答

### (a) 证明 $\displaystyle\lim_{x\to 2} x^2 = 4$

**证明**:

给定 $\varepsilon > 0$，需要找到 $\delta > 0$ 使得当 $0 < |x-2| < \delta$ 时，$|x^2 - 4| < \varepsilon$。

**分析**:
$$|x^2 - 4| = |x+2||x-2|$$

我们需要控制 $|x+2|$。先限制 $|x-2| < 1$，即 $1 < x < 3$，则 $3 < x+2 < 5$。

**选择 $\delta$**:

取 $\delta = \min\left(1, \frac{\varepsilon}{5}\right)$。

当 $0 < |x-2| < \delta$ 时：
- $|x-2| < 1$ 保证 $|x+2| < 5$
- $|x-2| < \frac{\varepsilon}{5}$

因此：
$$|x^2 - 4| = |x+2||x-2| < 5 \cdot \frac{\varepsilon}{5} = \varepsilon$$

由定义，$\displaystyle\lim_{x\to 2} x^2 = 4$。 $\square$

---

### (b) 证明 $\displaystyle\lim_{x\to 0} \frac{\sin x}{x} = 1$

**证明**:

**引理**: 对 $0 < |x| < \frac{\pi}{2}$，有 $\cos x < \frac{\sin x}{x} < 1$。

**引理证明**:

考虑单位圆，设圆心角为 $x$（弧度）。
- 三角形面积：$\frac{1}{2}\sin x$
- 扇形面积：$\frac{1}{2}x$
- 大三角形面积：$\frac{1}{2}\tan x$

由面积关系：$\sin x < x < \tan x$。

因此 $\cos x < \frac{\sin x}{x} < 1$。

**主证明**:

由夹逼定理，由于 $\lim_{x\to 0} \cos x = 1$，得 $\lim_{x\to 0} \frac{\sin x}{x} = 1$。

**严格的 $\varepsilon$-$\delta$ 证明**:

给定 $\varepsilon > 0$（设 $\varepsilon < 1$），取 $\delta = \arccos(1-\varepsilon)$。

当 $0 < |x| < \delta$：
$$\left|\frac{\sin x}{x} - 1\right| < 1 - \cos x < 1 - \cos \delta = \varepsilon$$

因此极限为 $1$。 $\square$

---

### (c) 证明 $\displaystyle\lim_{x\to 1} \frac{x^2-1}{x-1} = 2$

**证明**:

**简化**:
对 $x \neq 1$：
$$\frac{x^2-1}{x-1} = \frac{(x-1)(x+1)}{x-1} = x+1$$

**$\varepsilon$-$\delta$ 证明**:

给定 $\varepsilon > 0$，需要 $|x+1 - 2| = |x-1| < \varepsilon$。

取 $\delta = \varepsilon$。

当 $0 < |x-1| < \delta$：
$$\left|\frac{x^2-1}{x-1} - 2\right| = |x+1-2| = |x-1| < \varepsilon$$

因此极限为 $2$。 $\square$

## 证明技巧总结

| 技巧 | 应用位置 | 说明 |
|------|----------|------|
| 因式分解 | (a)(c) | 将 $|f(x)-L|$ 分解为可控的因子 |
| 局部有界 | (a) | 先在 $x_0$ 附近限制，得到界 |
| 夹逼定理 | (b) | 利用已知不等式控制目标函数 |
| 消去奇点 | (c) | 代数化简消除可去间断点 |

### 关键洞察

1. **$\varepsilon$-$\delta$ 证明的结构**: 分析 → 找关系 → 选 $\delta$ → 验证
2. **先限制后放缩**: 常在 $x_0$ 的小邻域内先得到局部控制
3. **几何直观**: 如(b)中利用单位圆的几何性质

## 常见错误

### 错误1: $\delta$ 依赖 $x$

❌ **错误做法**:
> "取 $\delta = \frac{\varepsilon}{|x+2|}$"（$\delta$ 不能依赖于 $x$）

✅ **正确做法**:
> 先用 $|x-2| < 1$ 限制得到 $|x+2| < 5$，再取 $\delta = \min(1, \varepsilon/5)$

### 错误2: 忽略 $x \neq x_0$

❌ **错误做法**:
> 在(c)中，不说明 $x \neq 1$ 就约分

✅ **正确做法**:
> 极限过程中 $x \neq 1$，所以可以约分

### 错误3: 证明方向混乱

❌ **错误做法**:
> 从结论倒推，写"由 $|x^2-4| < \varepsilon$ 得..."

✅ **正确做法**:
> 从 $|x-x_0| < \delta$ 出发，推导 $|f(x)-L| < \varepsilon$

## 变式练习

### 变式1（难度⭐⭐⭐）

证明 $\displaystyle\lim_{x\to 3} x^3 = 27$

**提示**: $|x^3-27| = |x-3||x^2+3x+9|$，先限制 $|x-3| < 1$

### 变式2（难度⭐⭐⭐⭐）

证明 $\displaystyle\lim_{x\to a} \sqrt{x} = \sqrt{a}$（$a > 0$）

**提示**: $|\sqrt{x}-\sqrt{a}| = \frac{|x-a|}{\sqrt{x}+\sqrt{a}} < \frac{|x-a|}{\sqrt{a}}$

### 变式3（难度⭐⭐⭐⭐）

证明 $\displaystyle\lim_{x\to 0^+} x\ln x = 0$

**提示**: 令 $x = e^{-t}$，转化为 $t \to +\infty$
