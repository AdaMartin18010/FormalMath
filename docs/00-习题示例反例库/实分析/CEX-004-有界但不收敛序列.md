# 有界但不收敛的序列

## 反例构造

### 经典例子：交错序列

**定义**：

$$a_n = (-1)^n, \quad n = 0, 1, 2, 3, \ldots$$

展开形式：

$$1, -1, 1, -1, 1, -1, \ldots$$

### 变体例子

**正弦序列**：

$$b_n = \sin\left(\frac{n\pi}{2}\right), \quad n = 0, 1, 2, 3, \ldots$$

展开形式：

$$0, 1, 0, -1, 0, 1, 0, -1, \ldots$$

## 反例分析

### 1. 有界性证明

**定理**：序列 $a_n = (-1)^n$ 有界。

**证明**：

对于所有 $n \in \mathbb{N}$：

$$|a_n| = |(-1)^n| = 1 \leq 1$$

因此，$\{a_n\}$ 有界（上界为1，下界为-1）。 $\square$

### 2. 发散性证明

**定理**：序列 $a_n = (-1)^n$ 不收敛。

**证明（反证法）**：

假设 $\lim_{n \to \infty} a_n = L$ 存在。

取 $\varepsilon = \frac{1}{2}$，根据极限定义，存在 $N \in \mathbb{N}$ 使得当 $n > N$ 时：

$$|a_n - L| < \frac{1}{2}$$

考虑 $n > N$ 且为偶数：$a_n = 1$，则 $|1 - L| < \frac{1}{2}$

考虑 $n > N$ 且为奇数：$a_n = -1$，则 $|-1 - L| < \frac{1}{2}$

由三角不等式：

$$|1 - (-1)| = |1 - L + L - (-1)| \leq |1 - L| + |L - (-1)| < \frac{1}{2} + \frac{1}{2} = 1$$

但 $|1 - (-1)| = 2$，矛盾！

因此假设不成立，序列不收敛。 $\square$

**证明（子列法）**：

考虑偶数项子列：$a_{2k} = (-1)^{2k} = 1$，故 $\lim_{k \to \infty} a_{2k} = 1$

考虑奇数项子列：$a_{2k+1} = (-1)^{2k+1} = -1$，故 $\lim_{k \to \infty} a_{2k+1} = -1$

若原序列收敛，则所有子列应收敛到同一极限。但 $1 \neq -1$，故原序列发散。 $\square$

## 直观解释

### 为什么"来回跳动"导致不收敛？

```
序列值分布示意：

a_n ↑
  1 ┤   ●       ●       ●       ●
    │   │       │       │       │
  0 ┼───┼───────┼───────┼───────┼──→ n
    │   │       │       │       │
 -1 ┤   ●       ●       ●       ●
    │
    0   1   2   3   4   5   6   7

序列在 1 和 -1 之间无限振荡，不趋向任何定点
```

**关键洞察**：

- 收敛要求"最终稳定"在极限值附近
- 有界只要求"不跑太远"
- 这个序列虽然范围受限（在[-1,1]内），但永远不会"安定下来"

### 与收敛序列的对比

**收敛序列的特征**：

```
收敛序列：
  L+ε├────────╮
   L ├─────●───●───●───●──→ n
  L-ε├────╯

最终所有项都落在 (L-ε, L+ε) 内
```

**本反例**：无论 $n$ 多大，总有项落在1和-1，不可能都落入任意小的邻域。

## 图形示意

```
详细的序列值分布：

      a_n
        ↑
    1.5 ┤
        │
      1 ┼──●────●────●────●────●────●────●
        │  │    │    │    │    │    │
    0.5 ┤  │    │    │    │    │    │
        │  │    │    │    │    │    │
      0 ┼──┼────┼────┼────┼────┼────┼────→ n
        │  │    │    │    │    │    │
   -0.5 ┤  │    │    │    │    │    │
        │  │    │    │    │    │    │
     -1 ┼──●────●────●────●────●────●
        │
   -1.5 ┤
        └────┬────┬────┬────┬────┬────┬
             1    2    3    4    5    6

收敛的必要条件：序列必须有唯一的聚点（极限点）
本序列有两个聚点：1 和 -1
```

**极限点分析**：

- 聚点（极限点）集：$\{1, -1\}$
- 上极限：$\limsup_{n \to \infty} a_n = 1$
- 下极限：$\liminf_{n \to \infty} a_n = -1$
- 由于上极限 $\neq$ 下极限，序列发散

## 相关定理

### Bolzano-Weierstrass定理

**定理**：在 $\mathbb{R}^n$ 中，每个有界序列必有收敛子列。

**本反例的验证**：

- $\{a_n\}$ 有界 ✓
- 存在收敛子列：偶数项子列 $a_{2k} \to 1$，奇数项子列 $a_{2k+1} \to -1$ ✓

**重要启示**：

Bolzano-Weierstrass定理只保证**存在**收敛子列，不要求原序列收敛。

### 单调收敛定理

**定理**：单调有界序列必收敛。

**本反例违反的条件**：序列 $a_n = (-1)^n$ 不是单调的。

```
单调递增序列：      单调递减序列：
  ↑                  ↑
  │        ╭──→      │╭──→
  │    ╭───╯         ││
  │╭───╯              │╰───╮
  ││                  │    ╰───╮
  └┴──────→          └┴────────→
```

### 极限点唯一性判定

**定理**：序列收敛 $\Longleftrightarrow$ 所有子列收敛到同一极限 $\Longleftrightarrow$ 上极限 = 下极限

**本反例**：

$$\limsup_{n \to \infty} (-1)^n = 1 \neq -1 = \liminf_{n \to \infty} (-1)^n$$

因此发散。

### Cauchy收敛准则

**定理**：序列收敛 $\Longleftrightarrow$ 它是Cauchy序列。

**Cauchy序列定义**：$\forall \varepsilon > 0, \exists N \in \mathbb{N}, \forall m,n > N: |a_m - a_n| < \varepsilon$

**本反例不是Cauchy序列**：

取 $\varepsilon = 1$，对于任意 $N$，取 $m = 2N$（偶数），$n = 2N+1$（奇数）：

$$|a_m - a_n| = |1 - (-1)| = 2 > 1 = \varepsilon$$

因此不是Cauchy序列，故不收敛。

## 避免此反例的条件

### 保证收敛的充分条件

| 条件 | 定理 | 说明 |
|-----|------|------|
| **单调+有界** | 单调收敛定理 | 递增有上界或递减有下界 |
| **Cauchy列** | Cauchy完备性 | 适用于完备度量空间 |
| **压缩映射** | Banach不动点定理 | 特殊迭代序列 |
| **递推关系** | 线性递推理论 | 特征根分析 |

### 修正为有界收敛序列

将本反例修改为收敛序列：

1. **指数衰减振荡**：
   $$b_n = \frac{(-1)^n}{n}$$
   - 有界：$|b_n| \leq 1$
   - 收敛：$\lim_{n \to \infty} b_n = 0$（由夹逼定理）

2. **几何衰减振荡**：
   $$c_n = \left(-\frac{1}{2}\right)^n$$
   - 有界：$|c_n| \leq 1$
   - 收敛：$\lim_{n \to \infty} c_n = 0$

3. **衰减交错序列**：
   $$d_n = \frac{(-1)^n}{2^n}$$
   - 绝对收敛（几何级数）

**关键区别**：衰减因子使"振荡幅度"趋于零，从而整体收敛。

## 拓展与变体

### 更复杂的振荡序列

1. **多值振荡**：
   $$a_n = \sin\left(\frac{2\pi n}{3}\right)$$
   取值：$0, \frac{\sqrt{3}}{2}, -\frac{\sqrt{3}}{2}, 0, \ldots$（三个聚点）

2. **稠密振荡**：
   $$b_n = \sin(\ln n)$$
   在 $[-1, 1]$ 上稠密分布

3. **周期序列**：
   $$c_n = n \mod 3$$
   取值：$0, 1, 2, 0, 1, 2, \ldots$（三个聚点）

### 应用中常见的有界发散序列

1. **周期信号采样**：
   - 数字信号处理中的正弦采样
   - 若采样频率与信号频率不成比例，可能产生有界非周期序列

2. **经济学模型**：
   - 某些周期经济模型的离散时间采样
   - 商业周期的简化模型

3. **动力系统**：
   - 周期轨道的状态序列
   - 准周期运动（如天体轨道）

### 收敛性判定的实用方法

```
判定流程：

序列 {a_n}
    │
    ├─→ 是否单调？
    │       ├─→ 是 → 检查有界性 → 有界则收敛
    │       └─→ 否 → 继续
    │
    ├─→ 计算 limsup 和 liminf
    │       ├─→ 相等 → 收敛
    │       └─→ 不等 → 发散
    │
    └─→ 验证 Cauchy 条件
            ├─→ 满足 → 收敛
            └─→ 不满足 → 发散
```

## 学习要点

1. **概念澄清**：有界性是收敛的**必要**条件，但**不充分**
2. **定理边界**：准确理解 Bolzano-Weierstrass 定理的"存在性"本质
3. **子列方法**：利用子列是判定发散的强有力工具
4. **极限点视角**：理解上、下极限的几何意义
