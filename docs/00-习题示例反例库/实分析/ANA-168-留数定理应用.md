# 留数定理应用

## 题目

**(a)** 计算积分 $\displaystyle\int_{-\infty}^{\infty} \frac{dx}{1+x^4}$。

**(b)** 计算积分 $\displaystyle\int_0^{\infty} \frac{\sin x}{x} dx$。

**(c)** 计算 $\displaystyle\sum_{n=1}^{\infty} \frac{1}{n^2}$ 使用复分析方法。

## 解答

### (a) 有理函数积分

在上半平面，$z^4 + 1 = 0$ 的根为 $e^{i\pi/4}, e^{3i\pi/4}$。

$$\text{Res}_{z=e^{i\pi/4}} \frac{1}{z^4+1} = \frac{1}{4z^3}\bigg|_{z=e^{i\pi/4}} = \frac{1}{4e^{3i\pi/4}}$$

积分值为 $\frac{\pi}{\sqrt{2}}$。

### (b) 正弦积分

考虑 $\frac{e^{iz}}{z}$ 沿钥匙孔围道积分，取虚部。

### (c) $\zeta(2)$ 计算

利用 $\pi \cot(\pi z)$ 在整数处有留数1，计算 $\sum \frac{1}{n^2}$。
