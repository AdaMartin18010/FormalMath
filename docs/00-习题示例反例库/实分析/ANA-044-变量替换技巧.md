# ANA-044: 变量替换技巧

**题号**: ANA-044
**难度**: ⭐⭐⭐ (Level 3)
**类型**: 计算型 (CALC)
**来源**: MIT 18.100A Chapter 6.2
**主题**: 积分的变量替换方法

---

## 题目

**(a)** 叙述并证明定积分的变量替换公式

**(b)** 计算 $\displaystyle\int_0^{\pi/2} \frac{\sin x}{\sin x + \cos x}\,dx$

**(c)** 计算 $\displaystyle\int_0^1 \frac{\ln(1+x)}{1+x^2}\,dx$

---

## 解答

### (a) 变量替换公式

**定理**: 设 $f$ 在 $[a,b]$ 上连续，$\varphi$ 在 $[\alpha,\beta]$ 上连续可导，$\varphi(\alpha) = a$，$\varphi(\beta) = b$。则
$$\int_a^b f(x)\,dx = \int_\alpha^\beta f(\varphi(t))\varphi'(t)\,dt$$

**证明**:

设 $F$ 是 $f$ 的原函数，则 $F'(x) = f(x)$。

考虑 $G(t) = F(\varphi(t))$。

由链式法则：
$$G'(t) = F'(\varphi(t))\varphi'(t) = f(\varphi(t))\varphi'(t)$$

因此：
$$\int_\alpha^\beta f(\varphi(t))\varphi'(t)\,dt = G(\beta) - G(\alpha)$$
$$= F(\varphi(\beta)) - F(\varphi(\alpha)) = F(b) - F(a) = \int_a^b f(x)\,dx \quad \square$$

---

### (b) 对称积分

**解**:

设 $I = \displaystyle\int_0^{\pi/2} \frac{\sin x}{\sin x + \cos x}\,dx$

令 $x = \frac{\pi}{2} - t$，则 $dx = -dt$：
$$I = \int_{\pi/2}^0 \frac{\sin(\pi/2 - t)}{\sin(\pi/2-t) + \cos(\pi/2-t)}(-dt)$$
$$= \int_0^{\pi/2} \frac{\cos t}{\cos t + \sin t}\,dt$$

因此：
$$2I = \int_0^{\pi/2} \frac{\sin x + \cos x}{\sin x + \cos x}\,dx = \int_0^{\pi/2} 1\,dx = \frac{\pi}{2}$$

$$I = \frac{\pi}{4} \quad \square$$

---

### (c) 巧妙换元

**解**:

设 $I = \displaystyle\int_0^1 \frac{\ln(1+x)}{1+x^2}\,dx$

令 $x = \tan\theta$，$dx = \sec^2\theta\,d\theta$：
$$I = \int_0^{\pi/4} \frac{\ln(1+\tan\theta)}{1+\tan^2\theta} \sec^2\theta\,d\theta$$
$$= \int_0^{\pi/4} \ln(1+\tan\theta)\,d\theta$$

利用 $\tan(\frac{\pi}{4} - \theta) = \frac{1-\tan\theta}{1+\tan\theta}$：

令 $\theta = \frac{\pi}{4} - t$：
$$I = \int_0^{\pi/4} \ln\left(1 + \frac{1-\tan t}{1+\tan t}\right)\,dt$$
$$= \int_0^{\pi/4} \ln\left(\frac{2}{1+\tan t}\right)\,dt$$
$$= \int_0^{\pi/4} [\ln 2 - \ln(1+\tan t)]\,dt$$
$$= \frac{\pi}{4}\ln 2 - I$$

因此 $2I = \frac{\pi}{4}\ln 2$：
$$I = \frac{\pi}{8}\ln 2 \quad \square$$

## 常见错误

### 错误1: 积分限遗漏

❌ **错误做法**:
> 换元后忘记改变积分限

✅ **正确做法**:
> 明确写出新的积分限

### 错误2: 雅可比行列式遗漏

❌ **错误做法**:
> 忘记乘以$\varphi'(t)$

✅ **正确做法**:
> $dx = \varphi'(t)dt$

### 错误3: 方向错误

❌ **错误做法**:
> $\alpha > \beta$时积分限处理错误

✅ **正确做法**:
> 确保上限大于下限或正确处理符号

## 变式练习

### 变式1（难度⭐⭐⭐）

计算 $\displaystyle\int_0^a \frac{dx}{x + \sqrt{a^2 - x^2}}$（$a > 0$）

**答案**: $\frac{\pi}{4}$

### 变式2（难度⭐⭐⭐⭐）

计算 $\displaystyle\int_0^1 \frac{\arctan x}{1+x}\,dx$

**答案**: $\frac{\pi}{8}\ln 2$

### 变式3（难度⭐⭐⭐⭐）

计算 $\displaystyle\int_{-\pi}^{\pi} \frac{x\sin x}{1+\cos^2 x}\,dx$

**答案**: $\frac{\pi^2}{2}$
