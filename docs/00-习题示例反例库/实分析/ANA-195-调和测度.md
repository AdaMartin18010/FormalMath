# 调和测度

## 题目

**(a)** 定义区域 $D \subset \mathbb{C}$ 关于 $z \in D$ 的调和测度 $\omega(z, E, D)$。

**(b)** 证明 Beurling 投影定理。

**(c)** 讨论调和测度的绝对连续性及 McMillan 定理。

## 解答

### (a) 调和测度的定义与基本性质

**概率定义**：设 $D \subset \mathbb{C}$ 是区域（连通开集），$z \in D$。考虑从 $z$ 出发的二维 Brown 运动 $B_t$。令 $\tau = \inf\{t > 0 : B_t \notin D\}$ 是首次离开 $D$ 的时间。

对 Borel 集 $E \subset \partial D$，定义 **调和测度**：
$$\omega(z, E, D) = \mathbb{P}_z(B_\tau \in E)$$
即 Brown 运动从 $z$ 出发，首次触及边界时落在 $E$ 中的概率。

**解析定义**：调和测度也可通过 Dirichlet 问题定义。若 $D$ 是 Dirichlet 区域（边界每个点正则），对边界数据 $f = \chi_E$（$E$ 的特征函数），令 $u_f$ 是 Dirichlet 问题的解：
$$\begin{cases} \Delta u = 0 & \text{in } D \\ u = f & \text{on } \partial D \end{cases}$$

则：
$$\omega(z, E, D) = u_{\chi_E}(z) = \int_{\partial D} \chi_E(\zeta) \, d\omega(z, \zeta, D)$$

**基本性质**：

1. **对固定的 $z, D$**，$\omega(z, \cdot, D)$ 是 $\partial D$ 上的概率测度：
   $$\omega(z, \partial D, D) = 1, \quad \omega(z, \emptyset, D) = 0$$
   可数可加性由 Dirichlet 解的线性性保证。

2. **对固定的 $E$**，$\omega(\cdot, E, D)$ 是 $D$ 中的调和函数（因它是 Dirichlet 问题的解）。

3. **共形不变性**：若 $f: D \to D'$ 是共形映射，则：
   $$\omega(z, E, D) = \omega(f(z), f(E), D')$$

4. **单调性**：若 $D_1 \subset D_2$ 且 $z \in D_1$，$E \subset \partial D_1 \cap \partial D_2$，则：
   $$\omega(z, E, D_1) \leq \omega(z, E, D_2)$$

**单位圆盘中的显式公式**：对 $D = \mathbb{D} = \{z : |z| < 1\}$，$E \subset \partial\mathbb{D}$：
$$\omega(z, E, \mathbb{D}) = \frac{1}{2\pi} \int_E \frac{1-|z|^2}{|z - e^{i\theta}|^2} d\theta = \int_E P(z, e^{i\theta}) d\theta$$

其中 $P(z, \zeta) = \frac{1-|z|^2}{|z-\zeta|^2}$ 是 Poisson 核。

### (b) Beurling 投影定理

**设定**：设 $E \subset \partial\mathbb{D}$ 是单位圆周上的闭集（或弧）。$E^*$ 表示 $E$ 的**圆周投影**：若 $E$ 是从原点可见的弧集，$E^*$ 是包含 $E$ 的最短闭弧（对称化的一种形式）。更一般地，对 $0 \in D$ 的单连通区域，考虑从原点出发的射线与边界的交点。

**定理（Beurling 投影定理）**：设 $E \subset \partial\mathbb{D}$ 是闭集，$E^* = \{e^{i\theta} : |\theta| \leq \alpha\}$ 是与 $E$ 有同样测度的对称弧（即 $|E^*| = |E|$，弧长测度）。则对 $z = 0$：
$$\omega(0, E, \mathbb{D}) \leq \omega(0, E^*, \mathbb{D})$$

等号成立当且仅当 $E$ 与 $E^*$ 相差一个旋转（即 $E$ 本身是弧）。

**证明**：

**步骤 1：显式计算**

对 $z = 0$，$\omega(0, E, \mathbb{D}) = \frac{|E|}{2\pi}$（由 Poisson 核在 $z=0$ 处为 1）。Wait，这是弧长比。实际上对一般 $E$，$\omega(0, E, \mathbb{D}) = \frac{1}{2\pi}\int_E d\theta = \frac{|E|}{2\pi}$，仅依赖于测度。

Beurling 定理的更强形式是关于一般点 $z \in \mathbb{D}$ 的。正确的表述：

**定理（Beurling）**：设 $E \subset \partial\mathbb{D}$，$z = r > 0$（实轴上）。设 $E^* = \{e^{i\theta} : \theta_1 \leq \theta \leq \theta_2\}$ 是包含 $E$ 的最短弧。则：
$$\omega(r, E, \mathbb{D}) \leq \omega(r, E^*, \mathbb{D})$$

**步骤 2：极大化原理**

固定 $r \in (0, 1)$。考虑泛函 $E \mapsto \omega(r, E, \mathbb{D})$。需证在所有与 $E$ 同测度的集合中，弧给出最大值。

Poisson 核 $P(r, e^{i\theta}) = \frac{1-r^2}{1 - 2r\cos\theta + r^2}$ 关于 $\theta$ 在 $[0, \pi]$ 递减（对 $r > 0$）。由重排不等式（rearrangement inequality），对递减函数和固定测度的集合，积分在集合"集中在 0 附近"时最大。即弧 $E^*$（关于实轴对称）给出最大调和测度。

**步骤 3：等号条件**

若 $E$ 给出等号，由重排不等式的等号条件，$E$ 必须是水平集，即弧。∎

**一般形式（Extremal Length）**：Beurling 定理是 extremal length 方法的特例。对一般单连通区域 $D$ 和 $z_0 \in D$，调和测度的极值问题可转化为 extremal distance 的计算。

### (c) 绝对连续性与 McMillan 定理

**绝对连续性**：设 $D$ 是 Jordan 区域（由 Jordan 曲线界定的区域），$\varphi: \mathbb{D} \to D$ 是 Riemann 映射。

**问题**：调和测度 $\omega(z_0, \cdot, D)$ 与弧长测度（或 Hausdorff 测度）$\mathcal{H}^1$ 在 $\partial D$ 上的关系？

**定理（McMillan, 1969）**：设 $D$ 是 Jordan 区域。则：

1. 在 $\partial D$ 的**切点集**（points of tangency，即边界有切线的点）上，调和测度与弧长测度相互绝对连续。

2. 更精确地，令 $T \subset \partial D$ 是存在切线的点集，$NT = \partial D \setminus T$ 是无切线点集。则：
   $$\omega(z_0, NT, D) = 0$$
   即无切线点集的调和测度为零。

**推论**：对 Jordan 曲线，调和测度集中在"几何好"的点上（有切线的点）。

**证明的核心思想**：

**步骤 1：共形映射的角导数**

Riemann 映射 $\varphi: \mathbb{D} \to D$ 可延拓到边界（Carathéodory 定理）。在点 $\zeta \in \partial\mathbb{D}$，若 $\varphi$ 有非零角导数，则 $\varphi(\zeta)$ 是 $\partial D$ 的切点。

**步骤 2：Plessner 定理与径向极限**

对 $\varphi'$, Plessner 定理说在 $\partial\mathbb{D}$ 上几乎处处（关于 Lebesgue 测度）径向极限存在且非零（在切点集对应处），或在某例外集上行为混沌。

**步骤 3：调和测度的像**

由共形不变性，$\omega(z_0, E, D) = \omega(\varphi^{-1}(z_0), \varphi^{-1}(E), \mathbb{D})$。在 $\mathbb{D}$ 中，调和测度与 Lebesgue 测度有显式关系（通过 Poisson 核）。

McMillan 证明无切线点对应于 $\varphi'$ 的径向极限为 0 或 $\infty$ 的点，而这些点的 Lebesgue 测度为 0（由 F. 和 M. Riesz 定理的推广）。

**Makarov 定理（1985）**：对任意单连通区域 $D$，调和测度 $\omega$ 是** Hausdorff 维数 1 的测度**。即：
- 对任意 $\alpha > 1$，$\mathcal{H}^\alpha(E) = 0 \Rightarrow \omega(E) = 0$
- 存在 $D$ 使得 $\omega$ 与 $\mathcal{H}^1$ 互相奇异（即支撑在零长度集上）

这 precise 了 McMillan 的结果：调和测度总是 "1-维"的，但不必与 Lebesgue 测度绝对连续。

**常见错误**：
- 认为调和测度总是与弧长测度绝对连续：反例由 Lavrentiev 等构造，存在 Jordan 曲线使得调和测度支撑在零长度集上。
- 忽视 Dirichlet 正则性的条件：对非正则边界点，调和测度的定义需通过 Perron 方法或 Wiener 解，此时概率解释仍成立但对边界点的质量分配可能奇异。
- 混淆共形映射的导数与调和测度密度：在好的边界点，$d\omega/d\mathcal{H}^1 \propto 1/|\varphi'|$，但 $|\varphi'|$ 可能为 0 或 $\infty$。
