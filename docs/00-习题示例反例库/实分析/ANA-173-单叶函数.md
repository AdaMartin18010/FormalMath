# 单叶函数

## 题目

**(a)** 证明 de Branges 定理（Bieberbach 猜想）：$|a_n| \leq n$。

**(b)** 证明 Koebe 1/4 定理。

**(c)** 计算 Koebe 函数的系数并讨论极值性质。

## 解答

### (a) Bieberbach 猜想 / de Branges 定理

**S 类函数**：$f(z) = z + a_2 z^2 + a_3 z^3 + \cdots$ 在 $\mathbb{D}$ 上单叶全纯。

**Bieberbach 猜想（1916）**：$|a_n| \leq n$，等号当且仅当 Koebe 函数及其旋转。

**de Branges 证明（1984）**：

1. **Loewner 方程**：对 $f \in S$，存在链 $f_t(z) = e^t z + \cdots$ 满足：
   $$\frac{\partial f_t}{\partial t} = z \frac{\partial f_t}{\partial z} \cdot \frac{1 + \kappa(t)z}{1 - \kappa(t)z}$$
   其中 $|\kappa(t)| = 1$。

2. **de Branges 函数**：构造特殊函数系统 $\tau_{k,n}(t)$，满足特定的微分方程。

3. **单调性**：$\sum_{k=1}^n (k|a_k(t)|^2 - k^{-1}|b_k(t)|^2) \tau_{k,n}(t)$ 关于 $t$ 单调。

4. **渐近**：$t \to \infty$ 时 $f_t \to e^t z$，给出 $n$ 的界。

**已知特例**：
- $n = 2$：Bieberbach（面积定理）
- $n = 3$：Löwner
- $n = 4$：Garabedian-Schiffer
- $n = 5, 6$：Pederson, Ozawa

---

### (b) Koebe 1/4 定理

**定理**：$f \in S$，则 $f(\mathbb{D}) \supset B(0, 1/4)$。

*证明*：

设 $w \notin f(\mathbb{D})$。考虑：

$$g(z) = \frac{w f(z)}{w - f(z)} = z + (a_2 + \frac{1}{w})z^2 + \cdots$$

$g \in S$（单叶性保持）。由 $|a_2| \leq 2$（$n=2$ 的 Bieberbach）：

$$\left|a_2 + \frac{1}{w}\right| \leq 2$$

若 $|w| < 1/4$，则 $|1/w| > 4$，与 $|a_2| \leq 2$ 矛盾。故 $|w| \geq 1/4$。

**尖锐性**：Koebe 函数 $k(z) = z/(1-z)^2$ 的像为 $\mathbb{C} \setminus (-\infty, -1/4]$。

---

### (c) Koebe 函数

$$k(z) = \frac{z}{(1-z)^2} = z + 2z^2 + 3z^3 + \cdots = \sum_{n=1}^\infty n z^n$$

**性质**：

- $a_n = n$（达到 Bieberbach 界）
- 像：$\mathbb{C} \setminus (-\infty, -1/4]$
- 是 $\mathbb{D}$ 到 slit plane 的共形映射

**旋转**：$k_\theta(z) = e^{-i\theta}k(e^{i\theta}z)$ 也达到界。

**极值性质**：Koebe 函数和旋转是 S 类中多个极值问题的解：

- $|a_n|$ 的最大值
- 遗漏值的最小模
- 像区域的最小凸包

---

## 常见错误

- **Bieberbach 的等号条件**：仅 Koebe 函数及其旋转达到等号，不是任意极值函数。
- **Koebe 1/4 的覆盖半径**：$1/4$ 是最大保证半径，实际覆盖可能更大。
- **S 类的归一化**：$f(0) = 0, f'(0) = 1$ 是必要的归一化条件。
