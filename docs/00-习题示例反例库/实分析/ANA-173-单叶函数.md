# 单叶函数

## 题目

**(a)** 设 $f(z) = z + a_2 z^2 + a_3 z^3 + \cdots$ 在单位圆盘上单叶全纯。证明 Bieberbach 猜想（de Branges 定理）：$|a_n| \leq n$。

**(b)** 证明 Koebe 1/4 定理：$f(\mathbb{D})$ 包含圆盘 $B(0, 1/4)$。

**(c)** 计算 Koebe 函数 $k(z) = \frac{z}{(1-z)^2}$ 的系数。

## 解答

### (a) Bieberbach 猜想

de Branges 证明利用 Loewner 方程和特殊函数。$n=2$ 时由面积定理可得。

### (b) Koebe 1/4 定理

若 $w \notin f(\mathbb{D})$，则 $g(z) = \frac{wf(z)}{w-f(z)}$ 单叶，应用面积定理。

### (c) Koebe 函数

$$k(z) = \sum_{n=1}^{\infty} n z^n, \quad a_n = n$$
