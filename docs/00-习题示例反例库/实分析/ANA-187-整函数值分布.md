# 整函数值分布

## 题目

**(a)** 证明 Picard 小定理：非常数整函数取遍所有复数值，至多一个例外。

**(b)** 证明 Nevanlinna 第一基本定理。

**(c)** 讨论 Julia 方向和 Borel 方向。

## 解答

### (a) Picard 小定理

由模函数或 Bloch 定理证明。例外值例子：$e^z$ 不取0。

### (b) Nevanlinna 第一基本定理

$$m(r, f) + N(r, f) = T(r, f) + O(1)$$
其中 $T(r,f)$ 是特征函数，$m$ 是接近函数，$N$ 是计数函数。

### (c) Julia 方向

若 $f$ 超越整函数，则存在方向 $\arg z = \theta$ 使得在此角域内 $f$ 取所有值无穷多次。
