# ANA-038: 高阶中值定理

**题号**: ANA-038
**难度**: ⭐⭐⭐⭐ (Level 4)
**类型**: 证明型 (PRF)
**来源**: MIT 18.100A Chapter 5.4
**主题**: 高阶导数的中值性质

---

## 题目

**(a)** 设 $f$ 在 $[a,b]$ 上有 $n$ 阶导数，且 $f(a) = f'(a) = \cdots = f^{(n-1)}(a) = 0$，$f(b) = 0$。证明存在 $c \in (a,b)$ 使得 $f^{(n)}(c) = 0$

**(b)** 设 $f$ 在 $[a,b]$ 上三阶可导。证明存在 $c \in (a,b)$ 使得
$$f(b) = f(a) + f'(a)(b-a) + \frac{f''(a)}{2}(b-a)^2 + \frac{f'''(c)}{6}(b-a)^3$$

**(c)** 设 $f$ 在 $[0,1]$ 上二阶可导，$|f''(x)| \leq M$，$f(0) = f(1) = 0$，且 $\max_{x\in[0,1]} |f(x)| = 1$。证明 $M \geq 8$

---

## 解答

### (a) 高阶Rolle定理

**证明**:

**归纳法**。

**$n = 1$**: 标准Rolle定理。

**归纳步骤**: 设对 $n-1$ 成立。

由Rolle定理，存在 $c_1 \in (a,b)$ 使得 $f'(c_1) = 0$。

$f'$ 满足：$f'(a) = (f')'(a) = \cdots = (f')^{(n-2)}(a) = 0$，且 $f'(c_1) = 0$。

由归纳假设，存在 $c \in (a, c_1) \subset (a,b)$ 使得 $(f')^{(n-1)}(c) = f^{(n)}(c) = 0$。 $\square$

---

### (b) 带Lagrange余项的Taylor展开

**证明**:

构造：
$$g(x) = f(b) - f(x) - f'(x)(b-x) - \frac{f''(x)}{2}(b-x)^2 - K\frac{(b-x)^3}{6}$$

其中 $K$ 选取使 $g(a) = 0$。

**计算 $g(b)$**:
$$g(b) = f(b) - f(b) - 0 - 0 - 0 = 0$$

**计算 $g'(x)$**:
$$g'(x) = -f'(x) - [f''(x)(b-x) - f'(x)] - [f'''(x)(b-x)^2/2 - f''(x)(b-x)] + K(b-x)^2/2$$

简化：
$$= -f'(x) - f''(x)(b-x) + f'(x) - f'''(x)(b-x)^2/2 + f''(x)(b-x) + K(b-x)^2/2$$
$$= (K - f'''(x))(b-x)^2/2$$

由Rolle定理，存在 $c \in (a,b)$ 使得 $g'(c) = 0$。

因此 $K = f'''(c)$。

由 $g(a) = 0$：
$$f(b) = f(a) + f'(a)(b-a) + \frac{f''(a)}{2}(b-a)^2 + \frac{f'''(c)}{6}(b-a)^3 \quad \square$$

---

### (c) 极值估计

**证明**:

设 $|f(x_0)| = 1$（最大值在某点取得）。不妨设 $f(x_0) = 1$。

由Taylor展开在 $x_0$ 处：

**在0处展开**:
$$0 = f(0) = f(x_0) + f'(x_0)(0-x_0) + \frac{f''(\xi_1)}{2}(0-x_0)^2$$

**在1处展开**:
$$0 = f(1) = f(x_0) + f'(x_0)(1-x_0) + \frac{f''(\xi_2)}{2}(1-x_0)^2$$

由Fermat定理，$f'(x_0) = 0$（$x_0$是内点极值）。

因此：
$$0 = 1 + \frac{f''(\xi_1)}{2}x_0^2 \Rightarrow f''(\xi_1) = -\frac{2}{x_0^2}$$

$$0 = 1 + \frac{f''(\xi_2)}{2}(1-x_0)^2 \Rightarrow f''(\xi_2) = -\frac{2}{(1-x_0)^2}$$

取绝对值：
$$|f''(\xi_1)| = \frac{2}{x_0^2}, \quad |f''(\xi_2)| = \frac{2}{(1-x_0)^2}$$

设 $g(x) = \max\left(\frac{2}{x^2}, \frac{2}{(1-x)^2}\right)$，求其最小值。

当 $\frac{2}{x^2} = \frac{2}{(1-x)^2}$，即 $x = 1/2$ 时，$g(1/2) = 8$。

因此 $M \geq 8$。 $\square$

## 常见错误

### 错误1: 归纳基础错误

❌ **错误做法**:
> 高阶Rolle归纳时忽略导数在端点的条件

✅ **正确做法**:
> 验证$f'$在$a$处直到$n-2$阶导数都为零

### 错误2: Taylor展开点选择错误

❌ **错误做法**:
> 在(c)中只在0或1处展开

✅ **正确做法**:
> 在极值点展开以利用$f'=0$

### 错误3: 符号处理错误

❌ **错误做法**:
> 忽略$f''$可能为正或负

✅ **正确做法**:
> 用绝对值处理，考虑最坏情况

## 变式练习

### 变式1（难度⭐⭐⭐⭐）

设 $f$ 在 $[a,b]$ 上 $n$ 阶可导，且 $f$ 在 $n+1$ 个不同点取零值。证明存在 $c$ 使得 $f^{(n)}(c) = 0$

### 变式2（难度⭐⭐⭐⭐）

设 $f$ 在 $[0,1]$ 上二阶可导，$f(0) = f(1) = 0$，$\min f(x) = -1$。求 $f''$ 的最小上界

**答案**: $M \geq 8$

### 变式3（难度⭐⭐⭐⭐⭐）

证明：若 $f$ 在 $\mathbb{R}$ 上无穷可导，且对所有 $n$，$f^{(n)}(0) = 0$，$f$ 不一定恒为零

**提示**: 考虑$f(x) = e^{-1/x^2}$（$x \neq 0$），$f(0) = 0$
