# 极值长度与模

## 题目

**(a)** 定义曲线族的极值长度并证明其共形不变性。

**(b)** 计算四边形的模并证明 Grötzsch 定理。

**(c)** 讨论极值拟共形映射的存在性。

## 解答

### (a) 极值长度

曲线族 $\Gamma$ 的极值长度：$\lambda(\Gamma) = \sup_\rho \frac{\inf_{\gamma \in \Gamma} \left(\int_\gamma \rho |dz|\right)^2}{\iint \rho^2 dxdy}$，共形不变。

### (b) Grötzsch 定理

矩形 $R_1, R_2$ 之间存在 $K$-拟共形映射当且仅当 $K^{-1} \leq \frac{m(R_1)}{m(R_2)} \leq K$。

### (c) 极值拟共形映射

Teichmüller 映射：由全纯二次微分诱导的极值映射。
