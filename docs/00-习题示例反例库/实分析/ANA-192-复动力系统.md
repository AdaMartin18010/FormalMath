# 复动力系统

## 题目

**(a)** 证明 Fatou-Julia 集的基本性质：Fillou 集开且不变，Julia 集是周期排斥点的闭包。

**(b)** 讨论 Mandelbrot 集与二次多项式族的联系。

**(c)** 证明多项式 Julia 集连通当且仅当临界点轨道有界。

## 解答

### (a) Fatou-Julia 理论

Fatou 集：迭代族正规点集。Julia 集：混沌区域，排斥周期点稠密。

### (b) Mandelbrot 集

$$\mathcal{M} = \{c \in \mathbb{C} : J(f_c) \text{ 连通}\}$$
其中 $f_c(z) = z^2 + c$。双曲分支对应吸引周期轨。

### (c) Julia 集连通性

Douady-Hubbard：临界点0轨道有界 ⟺ Julia 集连通。
