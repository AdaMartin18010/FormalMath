# 几何偏微分方程

## 题目

**(a)** 证明 Ricci 流的短时间存在性（Hamilton）。

**(b)** 讨论平均曲率流的奇点形成和类型 I/II。

**(c)** 证明 Yang-Mills 联络的正则性（Uhlenbeck）。

## 解答

### (a) Ricci 流

$$\frac{\partial g}{\partial t} = -2\text{Ric}(g)$$

DeTurck 技巧：引入向量场使得方程严格抛物，应用标准存在性理论。

### (b) 平均曲率流

收缩球：Type I 奇点（自相似解）。其他情形可能出现 Type II。

Huisken-Sinestrari 的凸解分类。

### (c) Uhlenbeck 定理

Yang-Mills 方程 $D^*F = 0$ 的解在适当规范下有内估计，可去奇点。
