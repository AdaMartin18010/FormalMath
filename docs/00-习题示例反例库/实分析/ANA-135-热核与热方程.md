# 热核与热方程

## 题目

热方程 $\partial_t u = \Delta u$ 于 $\mathbb{R}^n$，初值 $u(0,x) = f(x)$。

**(a)** 证明热核 $W_t(x) = (4\pi t)^{-n/2} e^{-|x|^2/(4t)}$ 满足热方程，且 $\int W_t = 1$。

**(b)** 证明解公式：$u(t,x) = (W_t * f)(x) = \int W_t(x-y) f(y) dy$。

**(c)** 证明半群性质：$W_{t+s} = W_t * W_s$，并推导 $e^{t\Delta} e^{s\Delta} = e^{(t+s)\Delta}$。

## 解答

### (a) 热核性质

**验证：**

$$\partial_t W_t = -\frac{n}{2}(4\pi t)^{-n/2-1} \cdot 4\pi e^{-|x|^2/(4t)} + (4\pi t)^{-n/2} e^{-|x|^2/(4t)} \cdot \frac{|x|^2}{4t^2}$$

$$= W_t \left(\frac{|x|^2}{4t^2} - \frac{n}{2t}\right)$$

$$\partial_{x_i} W_t = W_t \cdot \left(-\frac{x_i}{2t}\right)$$

$$\partial_{x_i}^2 W_t = W_t \cdot \left(\frac{x_i^2}{4t^2} - \frac{1}{2t}\right)$$

$$\Delta W_t = W_t \cdot \left(\frac{|x|^2}{4t^2} - \frac{n}{2t}\right) = \partial_t W_t$$

**积分归一**：
$$\int W_t = (4\pi t)^{-n/2} \int e^{-|x|^2/(4t)} dx = (4\pi t)^{-n/2} (4\pi t)^{n/2} = 1$$
$\square$

### (b) 解公式

**证明：**

$$u(t,x) = \int W_t(x-y) f(y) dy$$

$$\partial_t u = \int \partial_t W_t(x-y) f(y) dy = \int \Delta_x W_t(x-y) f(y) dy$$

$$= \Delta_x \int W_t(x-y) f(y) dy = \Delta u$$

初值：$W_t \to \delta$（分布意义），故 $u(t,x) \to f(x)$。$\square$

### (c) 半群性质

**证明：**

Fourier变换：$\widehat{W}_t(\xi) = e^{-4\pi^2 t |\xi|^2}$。

$$\widehat{W_t * W_s} = \widehat{W}_t \cdot \widehat{W}_s = e^{-4\pi^2 t|\xi|^2} e^{-4\pi^2 s|\xi|^2} = e^{-4\pi^2(t+s)|\xi|^2} = \widehat{W}_{t+s}$$

因此 $W_t * W_s = W_{t+s}$。

算子形式：$e^{t\Delta} f = W_t * f$，故
$$e^{t\Delta} e^{s\Delta} f = W_t * (W_s * f) = (W_t * W_s) * f = W_{t+s} * f = e^{(t+s)\Delta} f$$
$\square$
