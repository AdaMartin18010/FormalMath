# 复分析综合专题

## 题目

**(a)** 证明 Bieberbach 猜想的 de Branges 证明概要：利用 Loewner 方程和特殊函数。

**(b)** 讨论算术动力系统：有理函数的整数点轨道和 Diophantine 问题。

**(c)** 证明几何函数论中的极值问题与 Schiffer 变分。

## 解答

### (a) de Branges 定理

$|a_n| \leq n$ 对单叶函数 $f(z) = z + \sum_{n=2}^\infty a_n z^n$。

证明：Milin 猜想 ⇒ Robertson 猜想 ⇒ Bieberbach 猜想。

利用 Loewner 方程和 Legendre 函数的渐近性质。

### (b) 算术动力系统

Siegel 定理：椭圆曲线上整点有限。Dynamical Manin-Mumford 猜想。

### (c) Schiffer 变分

极值区域的边界满足微分方程，特征由 Schwarz 函数描述。
