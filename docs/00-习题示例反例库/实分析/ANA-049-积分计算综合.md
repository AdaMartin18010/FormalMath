# ANA-049: 积分计算综合

**题号**: ANA-049
**难度**: ⭐⭐⭐⭐ (Level 4)
**类型**: 计算型 (CALC)
**来源**: MIT 18.100A Chapter 6.2
**主题**: 综合积分技巧

---

## 题目

**(a)** 计算 $\displaystyle\int_0^{\pi} \frac{x\sin x}{1+\cos^2 x}\,dx$

**(b)** 计算 $\displaystyle\int_0^{\infty} \frac{dx}{1+x^4}$

**(c)** 设 $I_n = \displaystyle\int_0^{\pi/4} \tan^n x\,dx$。建立递推关系并计算 $I_n$

---

## 解答

### (a) 对称性技巧

**解**:

设 $I = \displaystyle\int_0^{\pi} \frac{x\sin x}{1+\cos^2 x}\,dx$

令 $x = \pi - t$，$dx = -dt$：
$$I = \int_{\pi}^0 \frac{(\pi-t)\sin(\pi-t)}{1+\cos^2(\pi-t)}(-dt)$$
$$= \int_0^{\pi} \frac{(\pi-t)\sin t}{1+\cos^2 t}\,dt$$
$$= \pi\int_0^{\pi} \frac{\sin t}{1+\cos^2 t}\,dt - I$$

因此：
$$2I = \pi\int_0^{\pi} \frac{\sin x}{1+\cos^2 x}\,dx$$

令 $u = \cos x$，$du = -\sin x\,dx$：
$$\int_0^{\pi} \frac{\sin x}{1+\cos^2 x}\,dx = \int_1^{-1} \frac{-du}{1+u^2} = \int_{-1}^{1} \frac{du}{1+u^2}$$
$$= \arctan u\Big|_{-1}^{1} = \frac{\pi}{4} + \frac{\pi}{4} = \frac{\pi}{2}$$

因此：
$$I = \frac{\pi}{2} \cdot \frac{\pi}{2} = \frac{\pi^2}{4} \quad \square$$

---

### (b) 代数技巧

**解**:

因式分解：$1+x^4 = (1+\sqrt{2}x+x^2)(1-\sqrt{2}x+x^2)$

**方法1: 复变函数**

极点在 $x = e^{i\pi/4}, e^{i3\pi/4}$ 等。

**方法2: 换元**

令 $x = \frac{1}{t}$，$dx = -\frac{1}{t^2}dt$：
$$I = \int_{\infty}^0 \frac{-dt/t^2}{1+1/t^4} = \int_0^{\infty} \frac{t^2}{1+t^4}\,dt$$

因此：
$$2I = \int_0^{\infty} \frac{1+x^2}{1+x^4}\,dx$$

除以 $x^2$：
$$= \int_0^{\infty} \frac{1/x^2 + 1}{1/x^2 + x^2}\,dx = \int_0^{\infty} \frac{d(x-1/x)}{(x-1/x)^2 + 2}$$

令 $u = x - 1/x$：
$$= \int_{-\infty}^{\infty} \frac{du}{u^2 + 2} = \frac{1}{\sqrt{2}}\arctan\frac{u}{\sqrt{2}}\Big|_{-\infty}^{\infty} = \frac{\pi}{\sqrt{2}}$$

因此：
$$I = \frac{\pi}{2\sqrt{2}} = \frac{\pi\sqrt{2}}{4} \quad \square$$

---

### (c) 递推积分

**解**:

设 $I_n = \displaystyle\int_0^{\pi/4} \tan^n x\,dx$

**递推关系**（$n \geq 2$）：

$$I_n = \int_0^{\pi/4} \tan^{n-2}x \cdot \tan^2 x\,dx = \int_0^{\pi/4} \tan^{n-2}x(\sec^2 x - 1)\,dx$$
$$= \int_0^{\pi/4} \tan^{n-2}x\sec^2 x\,dx - I_{n-2}$$

第一项：令 $u = \tan x$，$du = \sec^2 x\,dx$：
$$= \int_0^1 u^{n-2}\,du = \frac{1}{n-1}$$

因此：
$$I_n = \frac{1}{n-1} - I_{n-2}$$

**初值**:
$$I_0 = \frac{\pi}{4}, \quad I_1 = \int_0^{\pi/4} \tan x\,dx = -\ln\cos x\Big|_0^{\pi/4} = \frac{1}{2}\ln 2$$

**通项**:

对偶数 $n = 2m$：
$$I_{2m} = (-1)^m\frac{\pi}{4} + (-1)^{m-1}\left(1 - \frac{1}{3} + \frac{1}{5} - \cdots + \frac{(-1)^{m-1}}{2m-1}\right)$$

对奇数类似。 $\square$

## 证明技巧总结

| 技巧 | 应用位置 | 说明 |
|------|----------|------|
| 对称换元 | (a) | $x \to \pi - x$产生方程 |
| 倒数换元 | (b) | $x \to 1/x$配对简化 |
| 递降法 | (c) | 分部积分产生递推 |
| 代数变形 | (b) | 凑微分形式 |

### 关键洞察

1. **对称性**: 积分区间对称时考虑换元配对
2. **自指技巧**: 换元后得到关于$I$的方程
3. **递推关系**: 复杂积分化为简单积分序列

## 常见错误

### 错误1: 换元后积分限错误

❌ **错误做法**:
> 在(a)中$x = \pi - t$后积分限不变

✅ **正确做法**:
> 新积分限为$t$从$\pi$到$0$

### 错误2: 代数变形方向错误

❌ **错误做法**:
> 在(b)中凑不出合适的微分形式

✅ **正确做法**:
> 观察$x^2 + 1/x^2 = (x-1/x)^2 + 2$

### 错误3: 递推初值错误

❌ **错误做法**:
> $I_0$或$I_1$计算错误

✅ **正确做法**:
> $I_0 = \int_0^{\pi/4} 1\,dx = \pi/4$

## 变式练习

### 变式1（难度⭐⭐⭐⭐）

计算 $\displaystyle\int_0^{2\pi} \frac{dx}{1+a\cos x}$（$|a| < 1$）

**答案**: $\frac{2\pi}{\sqrt{1-a^2}}$

### 变式2（难度⭐⭐⭐⭐⭐）

计算 $\displaystyle\int_0^{\pi/2} \ln\sin x\,dx$

**答案**: $-\frac{\pi}{2}\ln 2$

### 变式3（难度⭐⭐⭐⭐⭐）

计算 $\displaystyle\int_0^{\infty} \frac{\cos x}{1+x^2}\,dx$

**答案**: $\frac{\pi}{2e}$
