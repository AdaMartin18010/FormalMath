# 导数存在但不连续的函数

## 反例构造

### 经典例子

**定义**：设函数

$$f(x) = \begin{cases} x^2 \sin\left(\frac{1}{x}\right), & x \neq 0 \\ 0, & x = 0 \end{cases}$$

### 变体例子

更简单的形式（用于说明原理）：

$$g(x) = \begin{cases} x^2 \sin\left(\frac{1}{x^2}\right), & x \neq 0 \\ 0, & x = 0 \end{cases}$$

或带参数的广义形式：

$$h_\alpha(x) = \begin{cases} |x|^\alpha \sin\left(\frac{1}{x}\right), & x \neq 0 \\ 0, & x = 0 \end{cases}, \quad \alpha > 0$$

## 反例分析

### 1. 函数在原点外的可微性

**定理**：$f(x)$ 在 $x \neq 0$ 处可微。

**证明**：

当 $x \neq 0$ 时，$f(x) = x^2 \sin(1/x)$ 是初等函数的乘积与复合，故可微。

由乘积法则和链式法则：

$$f'(x) = 2x \sin\left(\frac{1}{x}\right) + x^2 \cdot \cos\left(\frac{1}{x}\right) \cdot \left(-\frac{1}{x^2}\right)$$

$$= 2x \sin\left(\frac{1}{x}\right) - \cos\left(\frac{1}{x}\right), \quad x \neq 0 \quad \square$$

### 2. 函数在原点的可微性

**定理**：$f(x)$ 在 $x = 0$ 处可微，且 $f'(0) = 0$。

**证明**：

计算差商极限：

$$f'(0) = \lim_{x \to 0} \frac{f(x) - f(0)}{x - 0} = \lim_{x \to 0} \frac{x^2 \sin(1/x)}{x} = \lim_{x \to 0} x \sin\left(\frac{1}{x}\right)$$

由于 $|\sin(1/x)| \leq 1$，有：

$$|x \sin(1/x)| \leq |x| \to 0 \quad (x \to 0)$$

由夹逼定理：

$$\lim_{x \to 0} x \sin\left(\frac{1}{x}\right) = 0$$

因此 $f'(0) = 0$ 存在。 $\square$

### 3. 导数的不连续性

**定理**：$f'(x)$ 在 $x = 0$ 处不连续。

**证明**：

导函数为：

$$f'(x) = \begin{cases} 2x \sin\left(\frac{1}{x}\right) - \cos\left(\frac{1}{x}\right), & x \neq 0 \\ 0, & x = 0 \end{cases}$$

考察 $x \to 0$ 时 $f'(x)$ 的极限：

- 第一项：$2x \sin(1/x) \to 0$（有界函数乘以无穷小）
- 第二项：$-\cos(1/x)$ 在 $x \to 0$ 时**振荡无极限**

具体地，取两个序列：

- $x_n = \frac{1}{2n\pi}$：$\cos(1/x_n) = \cos(2n\pi) = 1$
- $y_n = \frac{1}{(2n+1)\pi}$：$\cos(1/y_n) = \cos((2n+1)\pi) = -1$

则：

$$\lim_{n \to \infty} f'(x_n) = 0 - 1 = -1$$
$$\lim_{n \to \infty} f'(y_n) = 0 - (-1) = 1$$

两子列极限不同，故 $\lim_{x \to 0} f'(x)$ 不存在。

因此 $f'(x)$ 在 $x = 0$ 处不连续。 $\square$

**间断点类型**：第二类间断点（振荡间断点）

## 直观解释

### 为什么"导数存在"但"不连续"？

**几何直观**：

```
函数 f(x) = x² sin(1/x) 在原点附近的行为：

f(x)↑
    │  ╭╮    ╭╮    ╭╮    振幅被 x² 限制，
    │ ╱  ╲  ╱  ╲  ╱  ╲   随着 x→0，
    │╱    ╲╱    ╲╱    ╲  振荡越来越"扁"
  0 ┼────────────────────→ x
    │
    │

导数 f'(x) = 2x sin(1/x) - cos(1/x) 在原点附近：

f'(x)↑
   1┤╲  ╱╲  ╱╲  ╱╲  ╱╲
    │ ╲╱  ╲╱  ╲╱  ╲╱  振荡不衰减，
   0┼──────────────────→ x  因为 cos(1/x) 的振幅
    │╱╲  ╱╲  ╱╲  ╱╲  ╱      不随 x 变化
  -1┤   ╲╱  ╲╱  ╲╱  ╲╱
    │

注意：f'(0) = 0，但 f'(x) 在 x→0 时振荡于 [-1,1]
```

**关键洞察**：

- 原函数 $f(x)$ 的"振幅衰减因子" $x^2$ 使函数在原点"平坦"
- 但导数的 $\cos(1/x)$ 项继承了 $\sin(1/x)$ 的振荡频率，没有衰减
- 在原点处的导数值 $f'(0) = 0$ 被定义为极限值，与附近导数值无连续性联系

### 连续可微 vs 可微

**连续可微（$C^1$）**：

```
C^1 函数：
  f(x) 光滑
  f'(x) 连续

例如：多项式、sin(x)、e^x
```

**本反例**：

```
仅可微但非 C^1：
  f(x) 处处光滑（视觉上）
  f'(x) 在某点剧烈振荡（不可视但数学上存在）
```

### 数值观察

| x | f(x) | f'(x) |
|---|------|-------|
| 0.1 | 0.0054 | ≈ -0.83 |
| 0.01 | 0.00005 | ≈ -0.98 |
| 0.001 | 5×10⁻⁷ | ≈ -0.999 |
| 1/(2π) | 0.0025 | -1 |
| 1/(π) | 0 | 1 |

注意：$f(x) \to 0$ 但 $f'(x)$ 不趋于确定值。

## 相关定理

### Darboux定理（导函数的介值性）

**定理**：若函数 $f$ 在区间 $I$ 上可微，则 $f'$ 具有**介值性**：

对于任意 $[a, b] \subseteq I$ 和任意 $y$ 介于 $f'(a)$ 与 $f'(b)$ 之间，存在 $c \in (a, b)$ 使得 $f'(c) = y$。

**重要推论**：

导函数不可能有**第一类间断点**（可去或跳跃间断点）。

**本反例的间断点**：

$f'(x)$ 在 $x=0$ 处的间断是**第二类**（振荡间断），与Darboux定理相容。

### 导数极限定理

**定理**：设 $f$ 在 $x_0$ 连续，在 $x_0$ 的去心邻域内可微，且 $\lim_{x \to x_0} f'(x) = L$ 存在，则：

1. $f$ 在 $x_0$ 可微
2. $f'(x_0) = L$

**逆否命题**：若 $f'$ 在 $x_0$ 不连续，则 $\lim_{x \to x_0} f'(x)$ 不存在。

**本反例验证**：$\lim_{x \to 0} f'(x)$ 不存在，符合定理。

### 连续可微的判定

**定理**：以下等价：

1. $f \in C^1[a,b]$（连续可微）
2. $f'$ 在 $[a,b]$ 上存在且连续
3. $f$ 可微且 $f'$ Riemann可积（在满足条件的区域）

**本反例**：$f$ 可微但 $f \notin C^1$（在含0的区间上）。

### 高阶导数的存在性

**观察**：本反例 $f(x) = x^2 \sin(1/x)$ 的二阶导数在原点不存在。

**更光滑的构造**：

$$f(x) = \begin{cases} x^4 \sin(1/x), & x \neq 0 \\ 0, & x = 0 \end{cases}$$

这个函数的二阶导数存在但三阶导数不连续。

一般地，$f(x) = x^{2n} \sin(1/x)$ 在 $x=0$ 处具有 $n$ 阶连续导数，但 $(n+1)$ 阶导数不连续。

## 避免此反例的条件

### 保证导数连续的充分条件

| 条件 | 结果 | 说明 |
|-----|------|------|
| **f ∈ C²** | f' ∈ C¹ | 二阶连续可微 |
| **f' 的极限存在** | f' 连续 | 导数极限定理 |
| **解析函数** | f' 解析 | 幂级数表示 |
| **凸/凹函数** | f' 单调 | 单调函数几乎处处连续 |

### 修正为导数连续的函数

**方法一**：去除振荡奇点

$$f(x) = x^2 \sin(x)$$

- $f'(x) = 2x \sin(x) + x^2 \cos(x)$ 处处连续

**方法二**：使用更高幂次

$$f(x) = x^3 \sin(1/x), \quad x \neq 0$$

- $f'(x) = 3x^2 \sin(1/x) - x \cos(1/x)$
- 当 $x \to 0$：$f'(x) \to 0 = f'(0)$
- 因此 $f' \in C^0$

**关键区别**：幂次 $> 2$ 时，导数中的振荡项有了衰减因子。

## 拓展与变体

### 更剧烈振荡的例子

$$f(x) = \begin{cases} x \sin(1/x), & x \neq 0 \\ 0, & x = 0 \end{cases}$$

- 在 $x=0$ 处**不可微**（差商振荡于[-1,1]）
- 但**连续**（有界振荡）

### 临界情况：$x^\alpha \sin(1/x^\beta)$

一般形式：

$$f_{\alpha,\beta}(x) = \begin{cases} |x|^\alpha \sin(1/|x|^\beta), & x \neq 0 \\ 0, & x = 0 \end{cases}$$

**分类**：

| 参数条件 | 连续性 | 可微性 | C¹ |
|---------|--------|--------|-----|
| $\alpha > 0$ | ✓ | ？ | ？ |
| $\alpha > 1$ | ✓ | ✓ | ？ |
| $\alpha > \beta + 1$ | ✓ | ✓ | ✓ |

**本反例**：$\alpha = 2, \beta = 1$，满足 $\alpha > 1$（可微）但 $\alpha \not> \beta + 1 = 2$（不C¹）。

### 多元函数推广

在 $\mathbb{R}^n$ 中，类似构造：

$$f(\mathbf{x}) = \begin{cases} \|\mathbf{x}\|^2 \sin(1/\|\mathbf{x}\|^2), & \mathbf{x} \neq \mathbf{0} \\ 0, & \mathbf{x} = \mathbf{0} \end{cases}$$

偏导数在原点存在但不连续。

### 分析中的重要性

1. **测试函数空间**：此类函数用于构造 $C_c^\infty$（紧支光滑函数）的逼近
2. **微分方程**：说明初值问题解的唯一性条件
3. **变分法**：极值函数的正则性研究

## 学习要点

1. **概念区分**：
   - 可微 ≠ 导数连续
   - $C^1$ 是比可微更强的条件

2. **Darboux定理的重要性**：
   - 导函数虽不一定连续，但必有介值性
   - 这限制了导数可能的间断类型

3. **构造技巧**：
   - $x^\alpha \sin(1/x^\beta)$ 是分析学中的标准测试函数
   - 幂次控制光滑性级别

4. **严格验证**：
   - 在原点的可微性必须用定义验证
   - 不能用求导法则（因为公式在原点失效）
