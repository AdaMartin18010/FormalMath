# 00-习题示例反例库

本目录是 FormalMath 项目中规模最大的核心资源库，共收录逾 700 份习题、示例与反例文档，覆盖代数、几何、分析、数论、概率统计、拓扑、离散数学等全部基础数学分支。每份文档均按照统一编码规范命名，既包含经典定理的验证性习题，也涵盖帮助学习者澄清概念边界的反例构造，同时还提供难度分层的解题示例，服务于自学练习、教学备课与认知诊断等多重场景。

## 子目录/文件导航

| 子目录 | 说明 |
|--------|------|
| `代数/` | 群论、环论、域论及线性代数习题与反例 |
| `几何/` | 微分几何、代数几何与拓扑几何示例 |
| `实分析/` | 极限、测度、积分与泛函分析习题 |
| `微分方程/` | 常微分方程与偏微分方程典型例题 |
| `拓扑/` | 点集拓扑与代数拓扑反例库 |
| `数论/` | 初等数论与解析数论练习 |
| `概率统计/` | 概率空间、随机变量与统计推断习题 |
| `离散数学/` | 组合数学、图论与逻辑学示例 |
| `代数几何/` | 代数簇、层论与概形相关习题 |

**代表性文件**：

- `ALG-001-群的定义验证.md`
- `GEO-001-光滑流形与图册.md`
- `GEO-003-微分形式与外微分.md`
