# 辛拓扑

## 题目

**(a)** 证明 Arnold 猜想的弱形式。

**(b)** 讨论辛容量的性质和构造。

**(c)** 证明 Gromov 的 nonsqueezing 定理。

## 解答

### (a) Arnold 猜想

辛微分同胚 φ 的固定点 Fix(φ) ≥ cup-length + 1。

Floer 同调：HF^*(φ) 与 Morse 同调同构。

### (b) 辛容量

c(M, ω)：单调、共形、非平凡。

Gromov 宽度：w(M) = sup{πr² : B(r) ↪ (M, ω)}。

### (c) Nonsqueezing 定理

若 B²ⁿ(R) ↪ Z²ⁿ(r) = B²(r) × ℝ²ⁿ⁻² 辛嵌入，则 R ≤ r。

证明：J-全纯曲线的阻碍。
