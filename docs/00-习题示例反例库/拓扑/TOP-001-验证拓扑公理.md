# TOP-001: 验证拓扑公理

**题号**: TOP-001
**难度**: ⭐⭐ (Level 2)
**类型**: 证明型 (PRF)
**来源**: MIT 18.901 Chapter 2.1
**主题**: 验证拓扑公理

---

## 题目

**拓扑的定义**: 设 $X$ 为非空集合，$\mathcal{T} \subseteq \mathcal{P}(X)$ 称为 $X$ 上的一个**拓扑**，若满足：

- **(T1)** $\emptyset \in \mathcal{T}$ 且 $X \in \mathcal{T}$
- **(T2)** 若 $U_1, U_2 \in \mathcal{T}$，则 $U_1 \cap U_2 \in \mathcal{T}$（有限交封闭）
- **(T3)** 若 $\{U_\alpha\}_{\alpha \in I} \subseteq \mathcal{T}$，则 $\bigcup_{\alpha \in I} U_\alpha \in \mathcal{T}$（任意并封闭）

---

### 问题

判定下列集族是否为对应集合上的拓扑。若是，验证三条公理；若不是，指出违反哪条公理。

**(a)** $X = \{a, b, c\}$，$\mathcal{T}_1 = \{\emptyset, \{a\}, \{b, c\}, X\}$

**(b)** $X = \{a, b, c\}$，$\mathcal{T}_2 = \{\emptyset, \{a\}, \{b\}, X\}$

**(c)** $X = \mathbb{R}$，$\mathcal{T}_3 = \{U \subseteq \mathbb{R} : U = \emptyset \text{ 或 } \mathbb{R} \setminus U \text{ 有限}\}$（**余有限拓扑**）

**(d)** $X = \mathbb{R}$，$\mathcal{T}_4 = \{U \subseteq \mathbb{R} : U = \emptyset \text{ 或 } 0 \in U\}$

**(e)** $X = \{1, 2, 3, 4\}$，$\mathcal{T}_5 = \{\emptyset, \{1\}, \{1, 2\}, \{1, 2, 3\}, X\}$

---

## 解答

### (a) $\mathcal{T}_1$ 是拓扑

**验证**:

| 公理 | 验证 |
|------|------|
| (T1) | $\emptyset \in \mathcal{T}_1$，$X \in \mathcal{T}_1$ ✓ |
| (T2) | 需验证所有两两交集：$\{a\} \cap \{b,c\} = \emptyset \in \mathcal{T}_1$；其他交显然 ✓ |
| (T3) | 任意并：$\{a\} \cup \{b,c\} = X \in \mathcal{T}_1$；其他并显然 ✓ |

**结论**: $\mathcal{T}_1$ 是 $X$ 上的拓扑。$\square$

---

### (b) $\mathcal{T}_2$ 不是拓扑

**验证**:

| 公理 | 验证 |
|------|------|
| (T1) | $\emptyset \in \mathcal{T}_2$，$X \in \mathcal{T}_2$ ✓ |
| (T2) | 交：$\{a\} \cap \{b\} = \emptyset \in \mathcal{T}_2$ ✓ |
| (T3) | **不满足**：$\{a\} \cup \{b\} = \{a,b\} \notin \mathcal{T}_2$ ✗ |

**结论**: $\mathcal{T}_2$ 不是拓扑，违反(T3)。$\square$

---

### (c) 余有限拓扑 $\mathcal{T}_3$ 是拓扑

**验证**:

**(T1)**: $\emptyset \in \mathcal{T}_3$（定义）。$\mathbb{R} \setminus \mathbb{R} = \emptyset$ 有限，故 $\mathbb{R} \in \mathcal{T}_3$ ✓

**(T2)**: 设 $U_1, U_2 \in \mathcal{T}_3$。
- 若 $U_1 = \emptyset$ 或 $U_2 = \emptyset$，则 $U_1 \cap U_2 = \emptyset \in \mathcal{T}_3$
- 否则，$\mathbb{R} \setminus U_1$ 和 $\mathbb{R} \setminus U_2$ 都有限
- 于是 $\mathbb{R} \setminus (U_1 \cap U_2) = (\mathbb{R} \setminus U_1) \cup (\mathbb{R} \setminus U_2)$ 有限 ✓

**(T3)**: 设 $\{U_\alpha\}_{\alpha \in I} \subseteq \mathcal{T}_3$。
- 若所有 $U_\alpha = \emptyset$，则并为空，属于 $\mathcal{T}_3$
- 否则存在某个 $U_{\alpha_0} \neq \emptyset$，则 $\mathbb{R} \setminus U_{\alpha_0}$ 有限
- 于是 $\mathbb{R} \setminus \bigcup_\alpha U_\alpha = \bigcap_\alpha (\mathbb{R} \setminus U_\alpha) \subseteq \mathbb{R} \setminus U_{\alpha_0}$ 有限 ✓

**结论**: 余有限拓扑是拓扑。$\square$

---

### (d) $\mathcal{T}_4$ 是拓扑（特定点拓扑）

**验证**:

**(T1)**: $\emptyset \in \mathcal{T}_4$（定义）。$0 \in \mathbb{R}$，故 $\mathbb{R} \in \mathcal{T}_4$ ✓

**(T2)**: 设 $U_1, U_2 \in \mathcal{T}_4$。
- 若其一为空，交为空
- 否则 $0 \in U_1$ 且 $0 \in U_2$，故 $0 \in U_1 \cap U_2$ ✓

**(T3)**: 设 $\{U_\alpha\} \subseteq \mathcal{T}_4$。
- 若所有 $U_\alpha = \emptyset$，则并为空
- 否则存在 $U_{\alpha_0}$ 含 0，故并含 0 ✓

**结论**: $\mathcal{T}_4$ 是拓扑（称为**特定点拓扑**，以 0 为特定点）。$\square$

---

### (e) $\mathcal{T}_5$ 是拓扑（链拓扑）

**验证**:

**(T1)**: $\emptyset, X \in \mathcal{T}_5$ ✓

**(T2)**: 注意到 $\mathcal{T}_5$ 形成链：$\emptyset \subset \{1\} \subset \{1,2\} \subset \{1,2,3\} \subset X$

链中任意两元素，小的是大的的子集，故交为小的那个，仍在链中。✓

**(T3)**: 链中任意子集的并是其中最大的元素（若子集非空），仍在链中。✓

**结论**: $\mathcal{T}_5$ 是拓扑。$\square$

## 教学要点

### 证明技巧总结

| 技巧 | 应用位置 | 说明 |
|------|----------|------|
| 逐条验证 | 全部 | 三条公理必须全部检查 |
| 德摩根律 | (c)部分 | 余集转换：$X \setminus (U \cap V) = (X \setminus U) \cup (X \setminus V)$ |
| 分类讨论 | (c)(d)部分 | 空集情况需单独处理 |
| 链结构识别 | (e)部分 | 链的交和并仍在链中 |

### 关键洞察

1. **余有限拓扑**是 $T_1$ 空间但不是 Hausdorff 空间的重要例子
2. **特定点拓扑**是常见的非标准拓扑
3. 有限集上的拓扑数量有限（$X$ 有 $n$ 个元素时，不同的拓扑数增长很快）

## 常见错误

### 错误1: 忘记验证空集情况

❌ **错误做法**:
> "在(c)中，$U_1 \cap U_2$ 的余集有限，故交在拓扑中"

✅ **正确理解**:
> 需考虑 $U_1 = \emptyset$ 或 $U_2 = \emptyset$ 的情况！

### 错误2: 混淆交与并的封闭性

❌ **错误做法**:
> "任意多个开集的交仍是开集"

✅ **正确理解**:
> 拓扑只保证**有限交**封闭，**任意并**封闭。无限交不一定开！

**反例**: 在 $\mathbb{R}$ 的标准拓扑中，$\bigcap_{n=1}^{\infty} (-\frac{1}{n}, \frac{1}{n}) = \{0\}$ 不是开集。

### 错误3: 误认为子集封闭即拓扑

❌ **错误做法**:
> "$\mathcal{T}$ 对并封闭，所以是拓扑"

✅ **正确理解**:
> 必须同时满足三条公理，缺一不可。

## 变式练习

### 变式1: 可数补拓扑

设 $X$ 为不可数集，$\mathcal{T} = \{U \subseteq X : U = \emptyset \text{ 或 } X \setminus U \text{ 可数}\}$。

证明 $\mathcal{T}$ 是 $X$ 上的拓扑。

**难度**: ⭐⭐⭐

### 变式2: 排他点拓扑

设 $X$ 为集合，$p \in X$。定义 $\mathcal{T} = \{U \subseteq X : p \notin U\} \cup \{X\}$。

验证 $\mathcal{T}$ 是拓扑。

**难度**: ⭐⭐

### 变式3: Sierpiński空间

设 $X = \{0, 1\}$，$\mathcal{T} = \{\emptyset, \{1\}, X\}$。

**(a)** 验证这是拓扑
**(b)** 写出此拓扑的所有连续自映射

**难度**: ⭐⭐⭐
