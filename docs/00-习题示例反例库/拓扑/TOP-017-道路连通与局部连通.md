## 题目

**(a)** 定义**道路连通**：任意两点有道路连接。证明：道路连通 $\Rightarrow$ 连通，并给出连通但不道路连通的例子。

**(b)** 证明：道路连通空间的连续像是道路连通的。

**(c)** 定义**局部连通**和**局部道路连通**。证明：局部道路连通的道路分支是开集。

**(d)** 证明：开集 $U \subset \mathbb{R}^n$ 是局部道路连通的，其道路分支是开集且等于连通分支。

## 详细解答

### (a) 道路连通与连通

**证明：**

**道路连通 $\Rightarrow$ 连通：**

设 $X$ 道路连通，$f: X \to \{0, 1\}$ 连续。

对 $x, y \in X$，有道路 $\gamma: [0, 1] \to X$，$\gamma(0) = x$，$\gamma(1) = y$。

$f \circ \gamma: [0, 1] \to \{0, 1\}$ 连续。

$[0, 1]$ 连通，$f \circ \gamma$ 常值。

故 $f(x) = f(y)$，$f$ 常值。

**连通但非道路连通的例子：**

**拓扑学家的正弦曲线：**

$$S = \{(x, \sin(1/x)) : 0 < x \leq 1\} \cup \{(0, y) : -1 \leq y \leq 1\}$$

**连通：** $S$ 是连通集 $\{(x, \sin(1/x))\}$ 的闭包。

**非道路连通：** $(0, 0)$ 和 $(1, \sin 1)$ 无道路连接。

任何从 $(0, 0)$ 出发的道路必在 $x=0$ 附近震荡无穷次，不连续。

**证毕。**

### (b) 连续映射保持道路连通性

**证明：**

设 $f: X \to Y$ 连续，$X$ 道路连通。

设 $y_1, y_2 \in f(X)$，$y_i = f(x_i)$。

$X$ 道路连通，有道路 $\gamma: [0, 1] \to X$，$\gamma(0) = x_1$，$\gamma(1) = x_2$。

$f \circ \gamma$ 是 $f(X)$ 中从 $y_1$ 到 $y_2$ 的道路。

**证毕。**

### (c) 局部连通性

**定义：**

- **局部连通：** 每点有连通邻域基
- **局部道路连通：** 每点有道路连通邻域基

**证明：**

设 $X$ 局部道路连通，$P$ 是道路分支。

设 $x \in P$，$U$ 是 $x$ 的道路连通邻域。

$U$ 道路连通，$U \subset P$（$P$ 是极大道路连通集）。

故 $P$ 开。

**证毕。**

### (d) $\mathbb{R}^n$ 开集的连通分支

**证明：**

设 $U \subset \mathbb{R}^n$ 开。

**局部道路连通：** 球形邻域道路连通。

**道路分支开：** 由(c)。

**道路分支 = 连通分支：**

连通分支 $C$ 是道路分支的不交并。

道路分支开，若多于一个则 $C$ 可分割，矛盾。

故 $C$ 是单个道路分支。

**证毕。**

## 证明技巧

1. **道路连通的验证：** 显式构造道路
2. **反例的构造：** 拓扑学家的正弦曲线是经典
3. **局部性质的应用：** 邻域基保证分支开

## 常见错误

| 错误 | 说明 |
|------|------|
| 认为连通开集道路连通 | $\mathbb{R}^2$中成立，一般空间不成立 |
| 混淆分支的类型 | 连通分支、道路分支、拟分支 |
| 忘记验证连续性 | 道路需要连续 |

## 变式练习

**变式1：** 证明：局部连通空间的连通分支是开集。

**变式2：** 构造局部连通但不局部道路连通的空间。

**变式3：** 证明：紧致Hausdorff空间是局部连通的当且仅当分支开。
