# 流形拓扑

## 题目

**(a)** 证明高维流形的 h-配边定理。

**(b)** 讨论光滑结构的唯一性和存在性。

**(c)** 证明 exotic sphere 的存在性。

## 解答

### (a) h-配边定理

Smale：dim ≥ 6，单连通 h-配边 (W; M, M') 微分同胚于 M × I。

关键：Morse 理论 + 流柄分解 + 流柄消去。

### (b) 光滑结构

存在性：PL 流形可光滑化（dim ≥ 5，单连通）。

唯一性： Kirby-Siebenmann 阻碍 KS ∈ H⁴(M; ℤ/2)。

### (c) Exotic sphere

Milnor：S⁷ 上有 28 种微分结构。

E₈ 流形的边界， plumbing 构造。
