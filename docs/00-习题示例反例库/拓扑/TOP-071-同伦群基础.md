# 同伦群基础

## 题目

设 $(X, x_0)$ 是带基点的拓扑空间。

**(a)** 定义高阶同伦群 $\pi_n(X, x_0) = [(S^n, s_0), (X, x_0)]$（同伦类集合）。证明 $n \geq 2$ 时这是交换群。

**(b)** 证明 $\pi_n(X \times Y) \cong \pi_n(X) \times \pi_n(Y)$。

**(c)** 计算 $\pi_n(S^k)$ 对 $n \leq k$。

## 解答

### (a) 高阶同伦群的交换性

**证明：**

$\pi_n(X) = [S^n, X]$。

$S^n = I^n / \partial I^n$，同伦类可用立方体表示。

两个映射的连接：在第一个坐标压缩。

对 $n \geq 2$，可同时在两个不同坐标压缩，两种方式同伦（Eckmann-Hilton论证）。

故群交换。

### (b) 积空间的同伦群

**证明：**

$X \times Y$ 的映射 $f: S^n \to X \times Y$ 等价于对 $(f_X, f_Y)$。

同伦保持此分解，故
$$[S^n, X \times Y] = [S^n, X] \times [S^n, Y]$$
群结构逐点。

### (c) 球面同伦群的初值

**解答：**

- $\pi_n(S^n) = \mathbb{Z}$（由度刻画）
- $\pi_n(S^k) = 0$ 对 $n < k$（胞腔逼近）

$n = k = 1$：$\pi_1(S^1) = \mathbb{Z}$。

$n = 1, k > 1$：$\pi_1(S^k) = 0$（单连通）。
