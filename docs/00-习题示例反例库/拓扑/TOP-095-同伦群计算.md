# 同伦群计算

## 题目

**(a)** 计算球面的前几个稳定同伦群 π_n^S，n ≤ 10。

**(b)** 证明 Adams 谱序列的基本结构。

**(c)** 讨论 J-同态和它的像。

## 解答

### (a) 稳定同伦群

π₀^S = ℤ
π₁^S = ℤ/2（Hopf 映射 η）
π₂^S = ℤ/2（η²）
π₃^S = ℤ/24（Hopf 映射 ν）
π₄^S = 0
π₅^S = 0
π₆^S = ℤ/2
π₇^S = ℤ/240
π₈^S = ℤ/2 ⊕ ℤ/2

### (b) Adams 谱序列

E_2^{s,t} = Ext_{A}^{s,t}(ℤ/p, ℤ/p) ⇒ π_{t-s}^S ⊗ ℤ_p。

A = Steenrod 代数。

### (c) J-同态

J: π_n(SO) → π_n^S。

Adams：im(J) 是 ℤ/(分母) 阶循环群，分母是 Bernoulli 数相关。
