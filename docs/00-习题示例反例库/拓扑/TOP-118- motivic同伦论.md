# Motivic 同伦论

## 题目

**(a)** 定义 motivic 稳定同伦范畴。

**(b)** 证明 motivic Adams 谱序列。

**(c)** 讨论 motivic 同伦与代数 K-理论的关系。

## 解答

### (a) Motivic 稳定同伦

Morel-Voevodsky：SH(k)。

对象： motivic 谱，悬置：S^{1,0} = S¹（拓扑），S^{1,1} = 𝔾_m（代数）。

### (b) Motivic Adams 谱序列

E₂ 项： motivic Steenrod 代数的 Ext。

收敛到 2-完备 stable motivic homotopy groups。

### (c) 与 K-理论联系

Bott 周期性在 motivic 同伦中由映射 f: ℙ¹ → BGL 实现。

代数 K-理论的 motivic 表示。
