# 等变拓扑

## 题目

**(a)** 证明 Borel 构造和等变上同调。

**(b)** 讨论 Smith 理论和 p-群作用。

**(c)** 证明等变配边理论。

## 解答

### (a) Borel 构造

EG → BG 万有主 G-丛。

等变上同调：H^*_G(X) = H^*(EG ×_G X)。

### (b) Smith 理论

P 有限 p-群作用在有限维空间 X。

若 X 模 p 同调于点，则 X^P 亦然。

推论：Sⁿ 的 p-群作用的不动点集非空。

### (c) 等变配边

Ω^G_*：G-流形的配边。

Tom Dieck：本地化定理，等变示性类。
