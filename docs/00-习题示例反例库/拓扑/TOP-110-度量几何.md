# 度量几何

## 题目

**(a)** 证明 Gromov-Hausdorff 距离定义度量。

**(b)** 讨论度量空间列的收敛性和紧致性。

**(c)** 证明 Cheeger 有限性定理。

## 解答

### (a) Gromov-Hausdorff 距离

d_{GH}(X, Y) = inf_Z {d_H^Z(X, Y)}。

满足度量的三公理，是等距类的度量。

### (b) 收敛性

Gromov 预紧致性：一致有界直径和维数 ⇒ 子列 GH 收敛。

### (c) Cheeger 有限性

Ricci ≥ (n-1)k，|sec| ≤ Λ，vol ≥ v，diam ≤ D。

则微分同胚类有限。
