# 度量几何

## 题目

**(a)** 证明 Gromov-Hausdorff 距离定义度量。

**(b)** 讨论度量空间列的收敛性和紧致性。

**(c)** 证明 Cheeger 有限性定理。

---

## 解答

### (a) Gromov-Hausdorff距离

#### Hausdorff距离

设 $(Z, d)$ 是度量空间，$A, B \subseteq Z$ 紧子集。

**Hausdorff距离**：
$$d_H^Z(A, B) = \max\left\{\sup_{a \in A} d(a, B), \sup_{b \in B} d(b, A)\right\}$$

等价：$d_H(A, B) < \varepsilon$ 当且仅当 $A \subseteq B_\varepsilon$ 且 $B \subseteq A_\varepsilon$，其中 $A_\varepsilon = \{z \in Z : d(z, A) < \varepsilon\}$。

#### Gromov-Hausdorff距离

对两个紧度量空间 $(X, d_X)$、$(Y, d_Y)$：

$$d_{GH}(X, Y) = \inf_{Z, f, g} \left\{d_H^Z(f(X), g(Y))\right\}$$

下确界取遍所有度量空间 $Z$ 和等距嵌入 $f: X \hookrightarrow Z$、$g: Y \hookrightarrow Z$。

#### 等价定义：对应

$$d_{GH}(X, Y) = \frac{1}{2} \inf_{\mathcal{R}} \text{dis}(\mathcal{R})$$

其中 $\mathcal{R} \subseteq X \times Y$ 是**对应**（correspondence），满足：
- $\forall x \in X, \exists y \in Y: (x,y) \in \mathcal{R}$
- $\forall y \in Y, \exists x \in X: (x,y) \in \mathcal{R}$

**失真**（distortion）：
$$\text{dis}(\mathcal{R}) = \sup_{(x,y), (x',y') \in \mathcal{R}} |d_X(x,x') - d_Y(y,y')|$$

#### 等价定义：$\varepsilon$-等距

$$d_{GH}(X, Y) < \varepsilon \Leftrightarrow \exists f: X \to Y \text{ s.t. }$$

1. $|d_Y(f(x), f(x')) - d_X(x, x')| < 2\varepsilon$（几乎等距）
2. $f(X)$ 是 $Y$ 的 $\varepsilon$-网

#### 证明 $d_{GH}$ 是度量

**非负性**：由定义 $d_{GH} \geq 0$。

**对称性**：由Hausdorff距离的对称性。

**三角不等式**：设 $X, Y, Z$ 是紧度量空间。对任意 $\varepsilon > 0$，存在 $Z_1$ 使得：
$$d_H^{Z_1}(X, Y) < d_{GH}(X, Y) + \varepsilon$$

存在 $Z_2$ 使得：
$$d_H^{Z_2}(Y, Z) < d_{GH}(Y, Z) + \varepsilon$$

将 $Z_1$ 和 $Z_2$ 沿 $Y$ 粘合（disjoint union + 在 $Y$ 上identify），得到新的度量空间 $W$。

在 $W$ 中：
$$d_H^W(X, Z) \leq d_H^W(X, Y) + d_H^W(Y, Z) = d_H^{Z_1}(X, Y) + d_H^{Z_2}(Y, Z)$$

$$< d_{GH}(X, Y) + d_{GH}(Y, Z) + 2\varepsilon$$

故 $d_{GH}(X, Z) \leq d_{GH}(X, Y) + d_{GH}(Y, Z)$。

**$d_{GH}(X, Y) = 0 \Rightarrow X \cong Y$**：

若 $d_{GH}(X, Y) = 0$，则存在对应序列 $\mathcal{R}_n$ 使得 $\text{dis}(\mathcal{R}_n) \to 0$。

由Arzelà-Ascoli类型的论证，可选取收敛子列得到**等距** $\phi: X \to Y$。

$\square$

### (b) 收敛性与紧致性

#### Gromov-Hausdorff收敛

**定义**：$X_n \xrightarrow{GH} X$ 若 $d_{GH}(X_n, X) \to 0$。

等价：存在度量空间 $Z$ 和等距嵌入 $X_n, X \hookrightarrow Z$ 使得 $d_H^Z(X_n, X) \to 0$。

#### Gromov预紧性定理

**定理**：度量空间族 $\mathcal{X}$ 在GH拓扑中是预紧的，当且仅当：

1. **一致全有界**：存在 $N: (0, \infty) \to \mathbb{N}$ 使得每个 $X \in \mathcal{X}$ 可被 $N(\varepsilon)$ 个半径 $\varepsilon$ 的球覆盖。
2. **一致直径有界**：$\sup_{X \in \mathcal{X}} \text{diam}(X) < \infty$。

**证明概要**：

**必要性**：若 $\mathcal{X}$ 预紧，则任何序列有Cauchy子列。对一致全有界：若否，存在 $X_n$ 需要 $> n$ 个 $\varepsilon$-球覆盖，则任何子列不Cauchy。

**充分性**：对 $X \in \mathcal{X}$，取 $\varepsilon_k = 2^{-k}$。由一致全有界，$X$ 有有限 $\varepsilon_k$-网 $N_k$，$|N_k| \leq N(\varepsilon_k)$。

对序列 $X_n$，用对角线法选取子列使得所有 $N_k$ 的基数和结构收敛。

构造极限空间 $X$ 为网序列的极限。

#### 应用到Riemann流形

**定理**（Gromov）：满足以下条件之一的Riemann流形族是GH预紧的：

- $\text{Ric} \geq (n-1)k$，$\text{diam} \leq D$
- $|K| \leq \Lambda$，$\text{diam} \leq D$，$\text{Vol} \geq v$

**证明**：由Bishop-Gromov体积比较或packing论证，一致全有界成立。

### (c) Cheeger有限性定理

已在GEO-050中详细证明，此处概述要点：

**定理**：给定 $n, D, V, K > 0$，满足以下条件之一的 $n$ 维闭Riemann流形的微分同胚类有限：
- $|K| \leq K$，$\text{diam} \leq D$，$\text{Vol} \geq V$
- $|\text{Ric}| \leq \Lambda$，$\text{inj} \geq i_0$

**关键步骤**：
1. **单射半径下界**（Cheeger引理）
2. **调和坐标**（$C^{1,\alpha}$ 估计）
3. **Gromov紧性** + 调和坐标的 $C^{1,\alpha}$ 紧性
4. **中心质量坐标**构造微分同胚
