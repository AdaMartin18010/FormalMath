# Surgery 流形结构

## 题目

**(a)** 证明 Browder-Novikov 手术理论的基本定理。

**(b)** 讨论同伦球面的分类。

**(c)** 证明高维 Poincaré 猜想的 surgery 证明。

## 解答

### (a) Browder-Novikov 定理

X 单连通 Poincaré 复形，维数 ≥ 5。

X 有流形结构 ⟺ Spivak 法丛存在，且 surgery 阻碍消失。

### (b) 同伦球面

Kervaire-Milnor：Θ_n = {h-球面}/配边。

Θ_n 有限，与稳定同伦群相关。

Θ_n/bP_{n+1} 是 J-同态余核的子群。

### (c) 高维 Poincaré 猜想

n ≥ 5：Smale 的 h-配边定理。

单连通 h-配边是平凡的 ⟹ 同伦球面是球面。
