# 谱序列计算实例

## 题目

**(a)** 用Serre谱序列计算 $H^*(\Omega S^n)$。

**(b)** 用Eilenberg-Moore谱序列计算 $H^*(G/H)$。

**(c)** 证明Adams-Novikov谱序列计算球面同伦群。

---

## 解答

### (a) Serre谱序列计算 $H^*(\Omega S^n; \mathbb{Z})$

#### Serre谱序列回顾

对纤维化 $F \to E \to B$，若 $B$ 单连通，则Serre谱序列：
$$E_2^{p,q} = H^p(B; H^q(F; \mathbb{Z})) \Rightarrow H^{p+q}(E; \mathbb{Z})$$

#### 路径纤维化

考虑路径空间纤维化：
$$\Omega S^n \to PS^n \to S^n$$

其中 $PS^n = \{\gamma: [0,1] \to S^n : \gamma(0) = x_0\}$ 是可缩的（$PS^n \simeq *$）。

#### $n \geq 2$ 的情形

$H^*(S^n; \mathbb{Z}) = \mathbb{Z}[x]/(x^2)$，$|x| = n$。

$E_2$ 页：

- $E_2^{0,q} = H^0(S^n; H^q(\Omega S^n)) = H^q(\Omega S^n)$
- $E_2^{n,q} = H^n(S^n; H^q(\Omega S^n)) = H^q(\Omega S^n)$（由 $x$ 乘）
- 其他 $E_2^{p,q} = 0$

$E_2$ 页只有两列非零（$p = 0$ 和 $p = n$），故所有 $d_r$（$r \neq n$）为零。唯一可能是 $d_n: E_n^{0,q} \to E_n^{n, q-n+1}$。

由于 $E_\infty^{p,q} = 0$（因为 $PS^n$ 可缩），谱序列必须完全消没。这意味着 $d_n$ 必须是同构：
$$d_n: H^q(\Omega S^n) \xrightarrow{\cong} H^{q-n+1}(\Omega S^n)$$

**归纳计算**：

- $q = 0$：$H^0(\Omega S^n) = \mathbb{Z}$（连通）
- $d_n: H^0 \to H^{-n+1}$，若 $n > 1$ 则目标为0
- 由 $E_\infty^{0,0} = 0$（因为 $PS^n$ 可缩，$H^0(PS^n) = \mathbb{Z}$ 但由滤过给出），需要 $d_n: E_n^{0,n-1} \to E_n^{n,0}$ 满

更系统地，设 $a \in H^{n-1}(\Omega S^n)$ 是 $d_n(1)$ 的原像（其中 $1 \in H^0(\Omega S^n)$）。由 $d_n$ 是同构，归纳得：

$$H^*(\Omega S^n; \mathbb{Z}) = \begin{cases} \mathbb{Z}[a] & n \text{ 奇数} \\ \Lambda(a) \otimes \mathbb{Z}[b] & n \text{ 偶数} \end{cases}$$

其中 $|a| = n-1$，$|b| = 2n-2$。

**具体结果**：

- $n$ 奇数：$H^*(\Omega S^n) = \mathbb{Z}[a]$，$|a| = n-1$（多项式代数）
- $n$ 偶数：$H^*(\Omega S^n) = \Lambda(a) \otimes \mathbb{Z}[b]$，$|a| = n-1$，$|b| = 2n-2$（外代数 $  imes$ 多项式）

#### $n = 1$ 的情形

$\Omega S^1 \simeq \mathbb{Z}$（离散），$H^0 = \bigoplus_{\mathbb{Z}} \mathbb{Z}$。

### (b) Eilenberg-Moore谱序列计算 $H^*(G/H)$

#### 设置

设 $G$ 是紧Lie群，$H \subseteq G$ 闭子群。有主纤维化：
$$H \to G \to G/H$$

及万有纤维化：
$$G \to EG \to BG, \quad H \to EH \to BH$$

#### Eilenberg-Moore谱序列

对纤维化 $F \to E \to B$（$B$ 单连通），Eilenberg-Moore谱序列计算 $H^*(F)$：
$$E_2^{*,*} = \text{Tor}_{H^*(B)}^{*,*}(H^*(E), \mathbb{Z}) \Rightarrow H^*(F)$$

#### 应用到 $G/H$

由 $G \to EG \times_G G/H \simeq BH$ 的纤维化（Borel构造），及 $EG \times_G G \simeq EG \simeq *$，可得：

$$H^*(G/H) \cong \text{Tor}_{H^*(BG)}^{*,*}(\mathbb{Z}, H^*(BH))$$

#### 经典结果

**Borel定理**：若 $H^*(G; \mathbb{Q})$ 和 $H^*(H; \mathbb{Q})$ 是外代数（对紧Lie群总成立），则：
$$H^*(G/H; \mathbb{Q}) \cong H^*(BH; \mathbb{Q}) / (\text{im } p^*)$$

其中 $p^*: H^*(BG) \to H^*(BH)$ 由包含诱导。

**具体计算**（以 $G/H = U(n)/U(k) \times U(n-k) = Gr_k(\mathbb{C}^n)$ 为例）：

$H^*(BU(n)) = \mathbb{Z}[c_1, \ldots, c_n]$，$H^*(BU(k) \times BU(n-k)) = \mathbb{Z}[c_1', \ldots, c_k', c_1'', \ldots, c_{n-k}'']$。

由分裂原理，$p^*(1 + c_1 + \cdots + c_n) = (1 + c_1' + \cdots + c_k')(1 + c_1'' + \cdots + c_{n-k}'')$。

Borel定理给出：
$$H^*(Gr_k(\mathbb{C}^n); \mathbb{Z}) = \mathbb{Z}[c_1, \ldots, c_k] / (\bar{c}_{n-k+1}, \ldots, \bar{c}_n)$$

其中 $\bar{c}_i$ 由 $(1 + c_1 + \cdots + c_k)(1 + \bar{c}_1 + \cdots) = 1$ 递推定义。这是Grassmann流形的经典上同调描述。

### (c) Adams-Novikov谱序列与球面同伦群

#### 经典Adams谱序列

对谱 $X$，经典Adams谱序列：
$$E_2^{s,t} = \text{Ext}_{\mathcal{A}}^{s,t}(H^*(X; \mathbb{Z}/2), \mathbb{Z}/2) \Rightarrow \pi_{t-s}(X)_2^\wedge$$

其中 $\mathcal{A}$ 是模2 Steenrod代数。

对 $X = S^0$（球谱），$E_2^{s,t} = \text{Ext}_{\mathcal{A}}^{s,t}(\mathbb{Z}/2, \mathbb{Z}/2)$。

**已知计算**：

- $\pi_0^S = \mathbb{Z}$（由Hurewicz定理）
- $\pi_1^S = \mathbb{Z}/2$（Hopf映射 $\eta: S^3 \to S^2$ 的稳定类）
- $\pi_2^S = \mathbb{Z}/2$（$\eta^2$）
- $\pi_3^S = \mathbb{Z}/24$（由 $d_2$ 消没去 $E_2^{2,5} = \mathbb{Z}/2$ 等）
- 更高群计算极其复杂

#### Adams-Novikov谱序列

Adams-Novikov谱序列（ANSS）使用复配边（complex cobordism）$MU$ 代替 $H\mathbb{Z}/2$：
$$E_2^{s,t} = \text{Ext}_{MU_*MU}^{s,t}(MU_*, MU_*X) \Rightarrow \pi_{t-s}(X)$$

**优势**：

1. $MU_* = \mathbb{Z}[x_1, x_2, \ldots]$（$|x_i| = 2i$）比 $H_*(H\mathbb{Z}/2)$ 结构更清晰
2. 对 $p > 2$ 的素数，ANSS比经典Adams谱序列更有效
3. 与形式群律（formal group laws）深刻联系

#### 计算球面同伦群

**$E_2$ 页结构**：
$$E_2^{*,*} = \text{Ext}_{BP_*BP}(BP_*, BP_*)$$

其中 $BP$ 是Brown-Peterson谱，$BP_* = \mathbb{Z}_{(p)}[v_1, v_2, \ldots]$（$|v_i| = 2(p^i - 1)$）。

**低维计算**：

- $v_1$ 对应 $\alpha$ 族：$\alpha_1 \in \pi_{2p-3}^S$（$p$-adic Hopf不变量1元素）
- $v_2$ 对应 $\beta$ 族
- $v_n$ 对应 $n$-th Greek letter 元素

**关键微分**：

- $d_{2p-1}(v_1^k) = v_1^{k-1} h_0$（$p$ 奇数）
- $d_{2p-1}(v_2) = h_1 v_1^p$（复杂关系）

**历史里程碑**：

- $\pi_n^S$ 对 $n \leq 45$ 已完全计算（Ravenel, Toda, 等）
- $p = 2$ 时，$\pi_{31}^S$ 等群有高度复杂的结构
- $p \geq 5$ 时，Ravenel的nilpotence定理大大简化了结构

**现代发展**：

- Kervaire不变量1问题（Hill-Hopkins-Ravenel定理：除可能有限个维数外，$\theta_j = 0$）
- 拓扑循环同调（TC）与代数K理论的计算
- 等变稳定同伦论的推广

#### 总结

Adams-Novikov谱序列是计算球面稳定同伦群最强大的工具之一，它将困难的同伦论问题转化为（相对）可计算的Ext群问题。与形式群律、椭圆上同调、拓扑模形式谱（tmf）的联系使其成为当代代数拓扑的核心支柱。
