# 同伦论进阶

## 题目

**(a)** 证明 H-空间和上 H-空间的性质。

**(b)** 讨论 James 构造和 loop 空间分解。

**(c)** 证明 Snaith 分裂。

## 解答

### (a) H-空间与上 H-空间

**定义（H-空间）**：拓扑空间 $X$ 带有基点 $e \in X$ 称为 **H-空间**，若存在连续映射（乘法）$\mu: X \times X \to X$ 使得下图在同伦意义下交换：
$$\mu \circ (e \times \text{id}) \simeq \text{id} \simeq \mu \circ (\text{id} \times e)$$
其中 $e: * \to X$ 是基点 inclusion。若还有 $\mu \circ (\mu \times \text{id}) \simeq \mu \circ (\text{id} \times \mu)$（结合性同伦），称为 **同伦结合** H-空间。

**例子**：
- 任何拓扑群是 H-空间。
- $\Omega Y$（$Y$ 的 loop 空间）是 H-空间，乘法为回路连接。
- 球面 $S^1, S^3, S^7$ 是 H-空间（来自复数、四元数、八元数的乘法）。

**Hopf 不变量与 H-空间结构**：

**定理（Adams, 1960）**：$S^{n-1}$ 是 H-空间当且仅当 $n = 1, 2, 4, 8$。

*证明概要*：Adams 利用次级上同调运算（secondary cohomology operations）证明了 Hopf 不变量为 1 的映射 $S^{2n-1} \to S^n$ 仅当 $n = 1, 2, 4, 8$ 存在。而 $S^{n-1}$ 是 H-空间等价于存在 Hopf 不变量 1 的映射。

**定义（上 H-空间）**：空间 $X$ 称为 **上 H-空间**（co-H-space），若存在 **余乘法** $\nu: X \to X \vee X$（楔和）使得：
$$(\text{id} \vee *) \circ \nu \simeq \text{id} \simeq (* \vee \text{id}) \circ \nu$$

**例子**：
- 悬挂 $\Sigma Y$ 是上 H-空间，余乘法由"捏合"$I \to I \vee I$ 诱导。
- 连通 CW 复形若为上 H-空间，则同伦等价于某空间的悬挂。

**定理**：若 $X$ 是 H-空间且 $Y$ 是上 H-空间，则 $[Y, X]$ 上可定义自然群结构。若 $X$ 同伦结合且存在同伦逆，$[Y, X]$ 是群；若还同伦交换，则是 Abel 群。

### (b) James 构造与 Loop 空间分解

**设定**：设 $X$ 是连通 CW 复形，带有基点 $*$。

**James 构造**：$J(X)$ 是 $X$ 上的自由拓扑幺半群：
$$J(X) = \bigsqcup_{n \geq 0} X^n / \sim$$
其中等价关系 $(x_1, \ldots, x_{i-1}, *, x_{i+1}, \ldots, x_n) \sim (x_1, \ldots, x_{i-1}, x_{i+1}, \ldots, x_n)$。

**定理（James, 1955）**：

**(1)** $J(X)$ 是 H-空间，且 $J(X) \simeq \Omega\Sigma X$（当 $X$ 是连通 CW 复形）。

**(2)** 悬挂分裂：
$$\Sigma J(X) \simeq \bigvee_{n \geq 1} \Sigma X^{\wedge n}$$
其中 $X^{\wedge n} = X \wedge \cdots \wedge X$ 是 $n$ 重 smash 积。

**证明概要**：

**(1)** 的构造：James 映射 $J(X) \to \Omega\Sigma X$ 由以下给出：对 $(x_1, \ldots, x_n) \in X^n$，对应回路 $t \mapsto [(x_1, t)] * \cdots * [(x_n, t)]$（连接）。James 证明这是弱同伦等价，从而对 CW 复形是同伦等价。

**(2)** 的证明：考虑 $J(X)$ 的滤过 $J_n(X) = \text{im}(X^n)$。则 $J_n(X)/J_{n-1}(X) \cong X^{\wedge n}$。对 suspension，这些商自然地分裂：
$$\Sigma J(X) \simeq \bigvee_{n \geq 1} \Sigma(J_n(X)/J_{n-1}(X)) \simeq \bigvee_{n \geq 1} \Sigma X^{\wedge n}$$

关键观察是 suspension 消去了连接中的附加信息。

**推论**：对 $X = S^k$：
$$\Sigma\Omega S^{k+1} \simeq \bigvee_{n \geq 1} S^{nk+1}$$

这给出 loop 空间 $\Omega S^{k+1}$ 的 homology 的完整计算（作为 Pontryagin 环）。

### (c) Snaith 分裂

**定理（Snaith, 1974）**：设 $X$ 是连通 CW 复形。则在稳定同伦范畴中：
$$\Sigma^\infty \Omega^\infty \Sigma^\infty X \simeq \bigvee_{n \geq 0} \Sigma^\infty X^{\wedge n}_{h\Sigma_n}$$

其中 $Y_{h\Sigma_n} = E\Sigma_n \times_{\Sigma_n} Y$ 是 homotopy 轨道空间（homotopy orbit）。

等价地，对 infinite loop 空间 $QX = \Omega^\infty\Sigma^\infty X = \varinjlim \Omega^n\Sigma^n X$：
$$\Sigma^\infty QX \simeq \bigvee_{n \geq 0} \Sigma^\infty (X^{\wedge n}/\Sigma_n)$$
（在稳定意义下，需用 homotopy 轨道）。

**证明思路**：

**步骤 1：配置空间模型**

$QX$ 有配置空间模型：$QX$ 的点是 $\mathbb{R}^\infty$ 中有限个带 $X$-标记的点的配置。即：
$$QX \simeq \bigsqcup_{n \geq 0} F(\mathbb{R}^\infty, n) \times_{\Sigma_n} X^n / \sim$$
其中 $F(\mathbb{R}^\infty, n)$ 是配置空间，等价关系来自基点。

**步骤 2：滤过与分裂**

$QX$ 有自然的滤过 $Q_n X$，由点至多 $n$ 个的配置组成。则：
$$Q_n X / Q_{n-1} X \simeq (F(\mathbb{R}^\infty, n)_+ \wedge X^{\wedge n})/\Sigma_n$$

对 suspension 谱，这些商稳定地分裂。

**步骤 3：Barratt-Priddy-Quillen 定理**

作为推论，取 $X = S^0$：
$$QS^0 = \Omega^\infty S^\infty = \bigsqcup_{n \geq 0} B\Sigma_n / \sim$$

Snaith 分裂结合 Barratt-Priddy-Quillen 定理给出：
$$\Sigma^\infty_+ QS^0 \simeq \bigvee_{n \geq 0} \Sigma^\infty B\Sigma_n$$

且 $QS^0$ 的连通分支 $\mathbb{Z} \times B\Sigma_\infty^+$ 的稳定同伦型对应 sphere 谱。

**K-理论中的应用**：对 $X = S^2$，$QS^2 \simeq \mathbb{Z} \times BU$（作为空间，经适当平移）。Snaith 分裂给出 $BU$ 的稳定分裂，与 Bott 周期性相关：
$$\widetilde{KU}(X) \cong [X, \mathbb{Z} \times BU] \cong [X, QS^2]$$

**常见错误**：
- 将不稳定的 James 分裂与稳定的 Snaith 分裂混淆：James 分裂对 suspension 成立，Snaith 分裂对 stable homotopy 成立。
- 忽视 homotopy 轨道与严格轨道的区别：在 stable range 外，$X^{\wedge n}/\Sigma_n$ 与 $X^{\wedge n}_{h\Sigma_n}$ 不同。
