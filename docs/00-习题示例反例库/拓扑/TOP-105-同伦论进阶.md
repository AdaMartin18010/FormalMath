# 同伦论进阶

## 题目

**(a)** 证明 H-空间和上 H-空间的性质。

**(b)** 讨论 James 构造和 loop 空间分解。

**(c)** 证明 Snaith 分裂。

## 解答

### (a) H-空间

乘法 μ: X × X → X，同伦结合、同伦单位。

Hopf 不变量：S^{2n-1} → S^n 的 H-空间结构（n=2,4,8）。

### (b) James 构造

J(X) = ⊔_{n≥0} X^n / ~，X 连通。

ΣJ(X) ≃ ∨_{n≥1} ΣX^{∧n}。

对 X = S^k：ΣΩS^{k+1} ≃ ∨_{n≥1} S^{nk+1}。

### (c) Snaith 分裂

稳定同伦：Σ^∞Ω^∞Σ^∞X ≃ ∨_{n≥0} Σ^∞X_{hΣ_n}^{∧n}。

K-理论中的应用。
