# 谱序列应用

## 题目

**(a)** 证明 Serre 谱序列：纤维丛 F → E → B。

**(b)** 计算 ΩS^n 的同调。

**(c)** 证明 Adams 谱序列的微分。

## 解答

### (a) Serre 谱序列

E^2_{p,q} = H_p(B; H_q(F)) ⇒ H_{p+q}(E)。

微分 d^r: E^r_{p,q} → E^r_{p-r, q+r-1}。

### (b) ΩS^n 的同调

Pontryagin 环 H_*(ΩS^n) 是张量代数。

对 n 偶：H_*(ΩS^n) = Γ[α]（除幂代数）。

对 n 奇：H_*(ΩS^n) = E[x] ⊗ Γ[y]。

### (c) Adams 微分

通过 Steenrod 运算的分解。

d₂ 与 Sq¹、Sq² 相关。高阶微分涉及次级运算。
