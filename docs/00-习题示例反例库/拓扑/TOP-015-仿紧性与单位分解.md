## 题目

**(a)** 定义**仿紧空间**：任何开覆盖有局部有限的开加细。证明：紧致的Hausdorff空间是仿紧的。

**(b)** 证明：度量空间是仿紧的（Stone定理）。

**(c)** 设 $X$ 是仿紧的Hausdorff空间。证明：任何开覆盖有从属于它的**单位分解**。

**(d)** 应用：证明仿紧空间上的向量丛可赋予Riemann度量。

## 详细解答

### (a) 紧致空间的仿紧性

**证明：**

设 $X$ 紧致Hausdorff，$\mathcal{U}$ 是开覆盖。

$X$ 紧致，$\mathcal{U}$ 有有限子覆盖 $\{U_1, ..., U_n\}$。

有限覆盖是局部有限的（每点至多在有限个 $U_i$ 中）。

$\mathcal{U}$ 有局部有限加细。

**证毕。**

### (b) 度量空间的仿紧性

**证明概要（Stone定理）：**

设 $(X, d)$ 是度量空间，$\mathcal{U}$ 是开覆盖。

**步骤1：** 良序覆盖

设 $\mathcal{U} = \{U_\alpha\}_{\alpha < \kappa}$（良序）。

**步骤2：** 构造加细

对每个 $n \in \mathbb{N}$，$\alpha < \kappa$，定义：
$$V_{\alpha, n} = \{x \in X : B(x, 1/2^n) \subset U_\alpha \text{ 且 } x \notin U_\beta \text{ 对 } \beta < \alpha\}$$

**步骤3：** 局部有限性验证

$\{V_{\alpha, n}\}_{\alpha, n}$ 是 $\mathcal{U}$ 的加细。

对 $x \in X$，$B(x, 1/2^{n+1})$ 只与有限个 $V_{\alpha, n}$ 相交。

**证毕。**

### (c) 单位分解的存在性

**定义：** 从属于开覆盖 $\{U_\alpha\}$ 的单位分解是连续函数族 $\{\phi_\alpha\}$ 满足：
1. $\phi_\alpha: X \to [0, 1]$
2. $\text{supp}(\phi_\alpha) \subset U_\alpha$
3. $\{\text{supp}(\phi_\alpha)\}$ 局部有限
4. $\sum_\alpha \phi_\alpha(x) = 1$ 对所有 $x$

**证明：**

设 $X$ 仿紧Hausdorff，$\mathcal{U} = \{U_\alpha\}$ 开覆盖。

**步骤1：** 局部有限加细

设 $\{V_\beta\}$ 是 $\mathcal{U}$ 的局部有限开加细。

**步骤2：** 缩小加细

因 $X$ 仿紧Hausdorff（故正规），可进一步缩小得 $\{W_\beta\}$ 使 $\overline{W_\beta} \subset V_\beta$。

**步骤3：** Urysohn函数

对每个 $\beta$，取 $f_\beta: X \to [0, 1]$ 使 $f_\beta|_{\overline{W_\beta}} = 1$，$f_\beta|_{X \setminus V_\beta} = 0$。

**步骤4：** 归一化

$f = \sum_\beta f_\beta$（局部有限和，连续）。

$f > 0$（因 $\{W_\beta\}$ 覆盖）。

$\phi_\beta = f_\beta / f$。

重新索引到原覆盖 $\{U_\alpha\}$。

**证毕。**

### (d) 向量丛的Riemann度量

**应用：**

设 $E \to X$ 是仿紧空间上的向量丛。

**局部度量：** 每点 $x$ 有邻域 $U$ 使 $E|_U \cong U \times \mathbb{R}^n$ 有标准度量。

**单位分解：** $\{\phi_\alpha\}$ 从属于覆盖 $\{U_\alpha\}$。

**全局度量：** $g = \sum_\alpha \phi_\alpha g_\alpha$。

在每点，有限个 $g_\alpha$ 非零，正定线性组合正定。

**证毕。**

## 证明技巧

1. **仿紧性的验证：** 找局部有限加细
2. **单位分解的构造：** 缩小加细 + Urysohn函数 + 归一化
3. **几何应用：** 局部构造 + 单位分解拼合

## 常见错误

| 错误 | 说明 |
|------|------|
| 忘记Hausdorff条件 | 单位分解需要正规性 |
| 混淆加细与子覆盖 | 加细可以不同标号 |
| 局部有限性验证错误 | 需仔细分析邻域交 |

## 变式练习
