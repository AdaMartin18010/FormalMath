# 高阶同伦论

## 题目

**(a)** 证明 Toda bracket 和 Massey product 的关系。

**(b)** 讨论球面的 v₁-周期族。

**(c)** 证明周期性元素的存在性。

## 解答

### (a) Toda bracket

⟨α, β, γ⟩ ⊂ π_{p+q+r+1}^S，当 αβ = 0, βγ = 0。

Massey product：上同调中的高阶积，spectral sequence 中的微分。

### (b) v₁-周期

im(J) 是 v₁-周期的，与 mod p  Moore 空间上的 Adams 映射相关。

### (c) 周期元素

Adams 的 μ 族，v₁-周期：β 族。

Ravenel 的 nilpotence 和周期性定理。
