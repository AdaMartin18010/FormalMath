# Thom-Pontryagin 构造

## 题目

**(a)** 证明 Thom-Pontryagin 同构：Ω_n^{SO} ≅ π_{n+k}(MSO(k)) (k 大)。

**(b)** 应用此构造计算低维配边群。

**(c)** 讨论 framed 配边和稳定同伦群。

## 解答

### (a) Thom-Pontryagin 同构

M^n ⊂ ℝ^{n+k} 有向子流形，法丛分类映射 → MSO(k)。

流形配边 ⟺ 映射同伦。

### (b) 低维配边

Ω₀^{SO} ≅ ℤ
Ω₁^{SO} = 0（圆周配边于0）
Ω₂^{SO} = 0（曲面分类）
Ω₃^{SO} = 0（Rohlin 定理）
Ω₄^{SO} ≅ ℤ（标志差）

### (c) Framed 配边

Ω_n^{fr} = π_n^S（稳定同伦群）。

Adams 谱序列计算：π₀^S=ℤ, π₁^S=ℤ/2, π₂^S=ℤ/2, π₃^S=ℤ/24。
