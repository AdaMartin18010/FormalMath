# 纽结理论

## 题目

**(a)** 证明 Alexander 多项式的 skein 关系。

**(b)** 讨论 Jones 多项式和 HOMFLY 多项式。

**(c)** 证明 Khovanov 同调 categorifies Jones 多项式。

## 解答

### (a) Alexander 多项式

Skein 关系：Δ(L₊) - Δ(L₋) = (t^{1/2} - t^{-1/2})Δ(L₀)。

Conway 规范化：∇(L₊) - ∇(L₋) = z∇(L₀)。

### (b) Jones 多项式

V(L) ∈ ℤ[t^{±1/2}]， skein：t⁻¹V(L₊) - tV(L₋) = (t^{1/2} - t^{-1/2})V(L₀)。

HOMFLY：P(L₊) - P(L₋) = zP(L₀)。

### (c) Khovanov 同调

立方体分解，链复形 Kh(L)。

Jones 多项式 = Euler 特征：V(L) = Σ (-1)^i q^j dim Kh^{i,j}(L)。
