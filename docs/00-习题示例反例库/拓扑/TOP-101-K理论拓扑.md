# 拓扑 K-理论

## 题目

**(a)** 证明 Bott 周期性：K̃⁰(S²) ≅ ℤ，K̃⁻¹(S²) = 0。

**(b)** 证明 Thom 同构在 K-理论中的形式。

**(c)** 讨论 K-理论的 Chern 特征。

## 解答

### (a) Bott 周期性

K⁰(S²) = ℤ[1] ⊕ ℤ[H]，H = tautological 线丛。

(H-1)² = 0 ⟹ K̃⁰(S²) = ℤ。

Bott 周期：K̃⁻ⁿ(X) ≅ K̃⁻ⁿ⁻²(X)。

### (b) K-理论 Thom 同构

复向量丛 E → X，K⁰(Th(E)) ≅ K⁰(X)。

Thom 类 λ_E ∈ K̃⁰(Th(E))。

### (c) Chern 特征

ch: K⁰(X) → H^{even}(X; ℚ)，环同态。

分裂原理：ch(E) = Σ e^{x_i}，x_i 陈根。
