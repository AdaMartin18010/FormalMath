# 切触拓扑

## 题目

**(a)** 证明 Darboux 定理和 Gray 稳定性。

**(b)** 讨论切触结构和 Stein 可填充性。

**(c)** 证明 Eliashberg 的 overtwisted 切触结构分类。

## 解答

### (a) Darboux 与 Gray

Darboux：所有切触流形局部标准 (ℝ²ⁿ⁺¹, α = dz - Σ yᵢdxᵢ)。

Gray：切触结构的同伦类保持同痕类。

### (b) Stein 可填充

Stein 域 (W, J) 的边界有切触结构 ξ = T∂W ∩ J(T∂W)。

可填充性给出切触不变量。

### (c) Overtwisted 结构

Eliashberg：overtwisted 切触结构由同伦类分类。

包含 overtwisted 圆盘的标准模型。
