# 三维流形

## 题目

**(a)** 证明几何化猜想的陈述和意义。

**(b)** 讨论 Haken 流形和 JSJ 分解。

**(c)** 证明双曲3-流形的 Mostow 刚性。

## 解答

### (a) 几何化猜想

Thurston：每个紧3-流形可分解为带有局部齐性几何的块。

八种几何：S³、ℝ³、H³、S²×ℝ、H²×ℝ、SL̃₂(ℝ)、Nil、Sol。

Perelman 证明（2003）。

### (b) Haken 流形

不可约、P²-不可约、有足够大的不可压缩曲面。

JSJ 分解：沿环面分解为 Seifert 和 atoroidal 块。

### (c) Mostow 刚性

有限体积双曲3-流形：等距类由基本群决定。

Vol(M) 是拓扑不变量。
