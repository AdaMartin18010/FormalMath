# 特征类深入

## 题目

**(a)** 证明陈类、Pontryagin 类和 Euler 类的公理刻画。

**(b)** 证明 Whitney 和公式和分裂原理。

**(c)** 讨论示性数与配边的完整关系。

## 解答

### (a) 公理刻画

陈类：自然性、Whitney 和、规范化 c₁(γ¹) = -e。

Pontryagin：p_i(E) = (-1)^i c_{2i}(E ⊗ ℂ)。

Euler：e(E ⊕ ε¹) = 0，顶维限制是 Thom 类。

### (b) 分裂原理

任意向量丛 E → X，存在 f: F(E) → X 使得 f*E 分裂为线丛。

Whitney 和：c(E ⊕ F) = c(E) ∪ c(F)。

### (c) 示性数与配边

示性数完全分类：Ω_* ⊗ ℚ ≅ ℚ[p₁, p₂, ...]。

整数配边复杂，涉及挠部分。
