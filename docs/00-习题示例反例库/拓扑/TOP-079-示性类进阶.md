# 示性类进阶

## 题目

**(a)** 证明Stiefel-Whitney类 $w_i \in H^i(BO; \mathbb{Z}/2)$ 分类实向量丛至稳定等价。

**(b)** 证明Chern类 $c_i \in H^{2i}(BU; \mathbb{Z})$ 的公理化特征。

**(c)** 计算 $w(T\mathbb{R}P^n)$ 和 $c(T\mathbb{C}P^n)$。

## 解答

### (a) Stiefel-Whitney类

分裂原理，Grothendieck定义。

### (b) Chern类公理

函子性、Whitney和公式、规范化。

### (c) 射影空间切丛

$w(T\mathbb{R}P^n) = (1+x)^{n+1}$，$x = w_1(\gamma^1)$。

$c(T\mathbb{C}P^n) = (1+H)^{n+1}$，$H = c_1(\mathcal{O}(1))$。
