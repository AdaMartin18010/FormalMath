# 纤维丛理论

## 题目

**(a)** 证明纤维丛的分类定理：F → E → B 由 B → BAut(F) 分类。

**(b)** 讨论主丛和伴随丛的构造。

**(c)** 证明示性类是分类映射的上拉。

## 解答

### (a) 分类定理

对结构群 G，主 G-丛由 B → BG 分类。

万有丛 EG → BG，EG 可缩。

### (b) 主丛与伴随丛

P 主 G-丛，F 是 G-空间，P ×_G F 是伴随纤维丛。

逆构造：纤维丛的标架丛是主丛。

### (c) 示性类

万有丛的示性类 c ∈ H^*(BG)。

丛 P = f*EG 的示性类 c(P) = f*c。
