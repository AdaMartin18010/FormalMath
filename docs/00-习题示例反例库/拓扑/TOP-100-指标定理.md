# 指标定理

## 题目

**(a)** 证明 Atiyah-Singer 指标定理的配边证明。

**(b)** 讨论 Dirac 算子的指标。

**(c)** 证明 Hirzebruch-Riemann-Roch 定理作为特例。

## 解答

### (a) Atiyah-Singer 配边证明

椭圆算子 D: Γ(E) → Γ(F) 的指标。

配边不变性：若符号在 S(T*M) 上延拓，则指标为0。

Thom 构造：化归到乘积情形。

### (b) Dirac 算子

Spin 流形上的 Dirac 算子 /∂D。

指标 = Â(M)（A-亏格），紧流形上指标为0。

有边情形：APS 指标 = Â(M) + η(∂M)/2。

### (c) HRR 定理

D = ∂̄ + ∂̄*: Ω^{0,even} → Ω^{0,odd}。

指标 = χ(M, E) = Td(M)∪ch(E)[M]。
