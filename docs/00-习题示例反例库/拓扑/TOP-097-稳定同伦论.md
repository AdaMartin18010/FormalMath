# 稳定同伦论

## 题目

**(a)** 定义谱（spectrum）并证明其稳定同伦范畴。

**(b)** 证明 Spanier-Whitehead 对偶性。

**(c)** 讨论 Brown 可表示定理。

## 解答

### (a) 谱

E = {E_n, ΣE_n → E_{n+1}}，通常 E_n = ΩE_{n+1}。

稳定同伦：π_n(E) = colim π_{n+k}(E_k)。

### (b) Spanier-Whitehead 对偶

X ⊂ S^N，DX 是补的 Spanier-Whitehead 形变。

对偶：H^i(X) ≅ H_{N-i}(DX)。

### (c) Brown 可表示

广义上同调 h^* 由谱 E 表示：h^n(X) = [X, E_n]。

对偶：广义同调由谱表示。
