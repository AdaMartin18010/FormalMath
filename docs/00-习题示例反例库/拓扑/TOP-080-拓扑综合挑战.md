# 代数拓扑综合挑战

## 题目

**(a)** 证明Adams谱序列：$E_2^{s,t} = \text{Ext}^{s,t}_{\mathcal{A}}(H^*(X), H^*(Y)) \Rightarrow \{X, Y\}_{t-s}$。

**(b)** 计算 $\pi_{n+1}(S^n)$ 和 $\pi_{n+2}(S^n)$。

**(c)** 讨论球面同伦群的周期性。

## 解答

### (a) Adams谱序列

由Steenrod代数的上同调计算稳定同伦群。

### (b) 球面稳定同伦群

$\pi_{n+1}(S^n) = \mathbb{Z}/2$（$n \geq 3$），Hopf不变量1。

$\pi_{n+2}(S^n) = \mathbb{Z}/2$（$n \geq 4$）。

### (c) 周期性

稳定同伦群的Bott周期性和非稳定周期性。
