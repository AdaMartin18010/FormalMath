# 代数拓扑综合挑战

## 题目

**(a)** 证明Adams谱序列：$E_2^{s,t} = \text{Ext}^{s,t}_{\mathcal{A}}(H^*(X), H^*(Y)) \Rightarrow \{X, Y\}_{t-s}$。

**(b)** 计算 $\pi_{n+1}(S^n)$ 和 $\pi_{n+2}(S^n)$。

**(c)** 讨论球面同伦群的周期性。

## 解答

### (a) Adams 谱序列

**背景**：球面稳定同伦群 $\pi_*^S = \pi_{*+k}(S^k)$（$k$ 充分大）是代数拓扑中最基本也最困难的问题之一。Serre 使用谱序列计算了前若干同伦群，但稳定同伦群需要更精细的工具。Adams 于 1958 年引入了 Adams 谱序列，以 Steenrod 代数的上同调为 $E_2$ 项，收敛到稳定同伦群。

**稳定同伦范畴**：记 $\{X, Y\}_n = \varinjlim_k [\Sigma^{n+k} X, \Sigma^k Y]$ 为稳定同伦群（映射类）。特别地，$\pi_n^S = \{S^0, S^0\}_n$。

**Steenrod 代数**：设 $\mathcal{A}_p$ 为系数 $\mathbb{Z}/p$ 的 Steenrod 代数。对 $p = 2$，$\mathcal{A}_2$ 由 Steenrod 平方 $Sq^i$（$i \geq 0$）生成，满足 Adem 关系：

$$Sq^a Sq^b = \sum_{j=0}^{\lfloor a/2 \rfloor} \binom{b-j-1}{a-2j} Sq^{a+b-j} Sq^j, \quad a < 2b$$

且 $Sq^0 = 1$，$Sq^1$ 为 Bockstein。

对奇素数 $p$，$\mathcal{A}_p$ 由 Steenrod 幂 $P^i$ 和 Bockstein $\beta$ 生成。

**定理（Adams 谱序列）**：设 $X$、$Y$ 为有限 CW 复形，则存在谱序列：

$$E_2^{s,t} = \text{Ext}_{\mathcal{A}}^{s,t}(H^*(Y; \mathbb{Z}/p), H^*(X; \mathbb{Z}/p)) \Rightarrow \{X, Y\}_{t-s}^\wedge_p$$

收敛到 $p$-完备的稳定同伦群。其中上标 $(s,t)$ 的含义为：$s$ 为同调次数（filtration），$t$ 为全次数。

**构造概要**：

**步骤1：Adams 分解**。构造 $Y$ 的**Adams 分解**：

$$Y = Y_0 \xleftarrow{} Y_1 \xleftarrow{} Y_2 \xleftarrow{} \cdots$$

其中每个 $Y_s \to Y_{s-1}$ 为纤维映射，纤维 $F_s$ 为 Eilenberg-MacLane 空间的积（从而其同伦群为 $\mathbb{Z}/p$ 向量空间）。具体地，取上同调类的映射：

$$Y_{s-1} \to \prod_j K(\mathbb{Z}/p, n_j)$$

将 $H^*(Y_{s-1})$ 的所有类映到 Hurewicz 像中，然后取同伦纤维得 $Y_s$。

**步骤2：导出正合偶**。对 Adams 分解，应用稳定同伦函子 $\{X, -\}_*$，得正合列：

$$\cdots \to \{X, Y_s\}_{t-s} \to \{X, Y_{s-1}\}_{t-s} \to \{X, F_{s-1}\}_{t-s} \to \{X, Y_s\}_{t-s-1} \to \cdots$$

这构成正合偶，导出谱序列。

**步骤3：识别 $E_2$ 项**。关键计算：

$$\{X, F_s\}_* \cong \text{Hom}_{\mathcal{A}}(H^*(F_s), H^*(X))$$

因为 $F_s$ 为 Eilenberg-MacLane 空间的积，$H^*(F_s)$ 为自由 $\mathcal{A}$-模（或投射模）。在正合偶中，$d_1$ 微分对应 $Y_s \to F_s \to \Sigma Y_{s+1}$，在 $\text{Hom}_{\mathcal{A}}$ 层面为 Yoneda Ext 中的复合。

于是：

$$E_2^{s,t} = H^s(\text{Hom}_{\mathcal{A}}(H^*(Y_\bullet), H^*(X)))_{t} = \text{Ext}_{\mathcal{A}}^{s,t}(H^*(Y), H^*(X))$$

**收敛性**：在适当的完备化下（$p$-完备或 Adams 滤过的完备化），谱序列收敛到 $\{X, Y\}_*$ 的伴随分次。

**简单例子**：设 $X = Y = S^0$，$p = 2$。则 $H^*(S^0; \mathbb{Z}/2) = \mathbb{Z}/2$（集中在次数 0）。$E_2^{s,t} = \text{Ext}_{\mathcal{A}_2}^{s,t}(\mathbb{Z}/2, \mathbb{Z}/2) = H^{s,t}(\mathcal{A}_2)$，即 Steenrod 代数的上同调。

低维计算：$H^{0,0}(\mathcal{A}_2) = \mathbb{Z}/2$，对应 $\pi_0^S = \mathbb{Z}$ 的 $p$-完备部分。
$H^{1,t}(\mathcal{A}_2) = 0$ 对 $t \neq 1$（实际上由 Adem 关系，$\text{Ext}^{1,*}$ 由 $h_i = [Sq^{2^i}]$ 生成）。

著名的 $d_2$ 微分 $d_2(h_i) = h_0 h_{i-1}^2$（或类似）导致了 Hopf 不变量 1 问题的解决：仅当 $i = 0, 1, 2, 3$ 时 $h_i$ 为永久循环，对应 Hopf 纤维化 $S^1 \to S^1$（$\eta$）、$S^3 \to S^2$、$S^7 \to S^4$、$S^{15} \to S^8$。

### (b) 球面同伦群 $\pi_{n+1}(S^n)$ 和 $\pi_{n+2}(S^n)$

**稳定范围**：由 Freudenthal 纬垂定理，当 $k \leq 2n - 2$ 时，纬垂同态 $\Sigma: \pi_k(S^n) \to \pi_{k+1}(S^{n+1})$ 为同构。特别地：
- $\pi_{n+1}(S^n)$ 当 $n \geq 3$ 时稳定；
- $\pi_{n+2}(S^n)$ 当 $n \geq 4$ 时稳定。

**计算 $\pi_{n+1}(S^n)$**：

**$n = 1$**：$\pi_2(S^1) = 0$（$S^1$ 为 $K(\mathbb{Z}, 1)$，高阶同伦群消失）。

**$n = 2$**：$\pi_3(S^2) = \mathbb{Z}$，由 Hopf 纤维化 $S^1 \hookrightarrow S^3 \twoheadrightarrow S^2$ 的长正合列：

$$\cdots \to \pi_3(S^1) \to \pi_3(S^3) \to \pi_3(S^2) \to \pi_2(S^1) \to \cdots$$

即 $0 \to \mathbb{Z} \to \pi_3(S^2) \to 0$，故 $\pi_3(S^2) = \mathbb{Z}$。生成元为 **Hopf 映射** $\eta: S^3 \to S^2$。

**$n \geq 3$**：进入稳定范围，$\pi_{n+1}(S^n) \cong \pi_3^S$。由 Adams 谱序列或 Thom-Pontryagin 构造，$\pi_3^S = \mathbb{Z}/24$（不，更正：$\pi_1^S = \mathbb{Z}/2$，生成元为 $\eta$）。

具体而言：$\Sigma \eta: S^4 \to S^3$ 为 $\eta$ 的纬垂。$\pi_4(S^3) = \mathbb{Z}/2$，由 $\Sigma \eta$ 生成。
归纳地，$\pi_{n+1}(S^n) = \mathbb{Z}/2$（$n \geq 3$），生成元为 $\eta$ 的 $(n-2)$ 重纬垂。

**总结**：

$$\pi_{n+1}(S^n) = \begin{cases} 0 & n = 1 \\ \mathbb{Z} & n = 2 \\ \mathbb{Z}/2 & n \geq 3 \end{cases}$$

**计算 $\pi_{n+2}(S^n)$**：

**$n = 1$**：$\pi_3(S^1) = 0$。

**$n = 2$**：$\pi_4(S^2)$。利用 Hopf 纤维化：

$$\cdots \to \pi_4(S^3) \to \pi_4(S^2) \to \pi_4(S^1) = 0$$

故 $\pi_4(S^2) \cong \pi_4(S^3) = \mathbb{Z}/2$（纬垂 $"。

更精确：由长正合列 $0 \to \pi_4(S^2) \to \pi_3(S^1) = 0$，这不对。正确长正合列为：

$$\pi_4(S^1) \to \pi_4(S^3) \to \pi_4(S^2) \to \pi_3(S^1)$$

即 $0 \to \mathbb{Z}/2 \to \pi_4(S^2) \to 0$，故 $\pi_4(S^2) = \mathbb{Z}/2$。

**$n = 3$**：$\pi_5(S^3)$。由 Serre 的谱序列计算，$\pi_5(S^3) = \mathbb{Z}/2$。

**$n \geq 4$**：进入稳定范围，$\pi_{n+2}(S^n) \cong \pi_2^S = \mathbb{Z}/2$。生成元为 $\eta^2$（$"。

**总结**：

$$\pi_{n+2}(S^n) = \begin{cases} 0 & n = 1 \\ \mathbb{Z}/2 & n = 2 \\ \mathbb{Z}/2 & n = 3 \\ \mathbb{Z}/2 & n \geq 4 \end{cases}$$

**Hopf 不变量**：映射 $f: S^{2n-1} \to S^n$ 的 Hopf 不变量 $H(f)$ 定义为：设 $X = S^n \cup_f e^{2n}$，则 $H^n(X; \mathbb{Z}) = \mathbb{Z}\alpha$，$H^{2n}(X; \mathbb{Z}) = \mathbb{Z}\beta$，$\alpha^2 = H(f) \beta$。

Adams 证明：存在 Hopf 不变量 1 的映射仅当 $n = 1, 2, 4, 8$。这对应实可除代数（$\mathbb{R}, \mathbb{C}, \mathbb{H}, \mathbb{O}$）的维数。

### (c) 球面同伦群的周期性

**稳定同伦群的 Bott 周期性**：实 K-理论中的 Bott 周期性给出：

$$\pi_k(O) \cong \pi_{k+8}(O)$$

其中 $O = \varinjlim O(n)$ 为稳定正交群。由 J-同态和 Thom 同构，这蕴含稳定同伦群中的周期现象。

**Bott 周期的具体同构**：

| $k \mod 8$ | $\pi_k(O)$ | 解释 |
|:---|:---|:---|
| 0 | $\mathbb{Z}/2$ | $\pi_1^S$（不，实际 $\pi_0(O) = \mathbb{Z}/2$） |
| 1 | $\mathbb{Z}/2$ | |
| 2 | 0 | |
| 3 | $\mathbb{Z}$ | |
| 4 | 0 | |
| 5 | 0 | |
| 6 | 0 | |
| 7 | $\mathbb{Z}$ | |

实际上 $\pi_k(O)$ 与 $\pi_{k+8}(O)$ 的关系通过 Clifford 代数的模给出。

**Nishida 定理**：稳定同伦群 $\pi_*^S$ 中，所有正次数元素均为幂零的（除了 $\pi_0^S = \mathbb{Z}$）。更精确地，对 $\alpha \in \pi_k^S$（$k > 0$），存在 $n$ 使得 $\alpha^n = 0$。

**Toda 括号与高层周期**：Toda 引入了**Toda 括号**（或 Massey 积的同伦版本）来构造新元素。例如：

$$\nu \in \pi_3^S = \mathbb{Z}/24, \quad \sigma \in \pi_7^S = \mathbb{Z}/240$$

这些是 Hopf 不变量 1 元素的提升。

** chromatic 周期性**：由 Miller-Ravenel-Wilson 的工作，稳定同伦群可按**chromatic 高度**分层。对形式群定律的**高度** $n$，有**Morava K-理论** $K(n)$ 和**Morava E-理论** $E_n$。

**周期族**：对高度 $n$，存在 $v_n$-周期族。最著名的：
- **$v_1$-周期族**：与 image of J 有关，由 Adams 谱序列中的 $v_1$ 自映射生成，周期为 $2(p-1)$（$p$ 为奇素数）或 8（$p = 2$）。
- **$v_2$-周期族**：由 Smith-Toda 复形上的 $v_2$ 自映射生成。

**Hopkins-Smith 定理**：对任意 $n \geq 1$，存在有限谱 $X$ 和自映射 $v: \Sigma^{2(p^n-1)} X \to X$ 诱导 $K(n)$ 上的同构和 $K(m)$（$m \neq n$）上的零映射。这是 thick subcategory 定理的核心。

**总结**：稳定同伦群 $\pi_*^S$ 是一个极为复杂的分次交换环，具有：
1. **Bott 周期**：与 K-理论相关的 8 周期；
2. ** chromatic 周期**：按素数 $p$ 和高度 $n$ 的 $v_n$ 周期现象；
3. **幂零性**：所有正次数元素幂零（Nishida）。
