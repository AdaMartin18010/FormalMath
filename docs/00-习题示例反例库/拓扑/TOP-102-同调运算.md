# 同调运算

## 题目

**(a)** 证明 Steenrod 方运算的基本性质。

**(b)** 证明 Adem 关系。

**(c)** 讨论 Steenrod 运算在球面上的非平凡性。

## 解答

### (a) Steenrod 方

Sq^i: H^n(X; ℤ/2) → H^{n+i}(X; ℤ/2)。

性质：Sq⁰ = id，Sq^n(x) = x² (|x|=n)，Cartan 公式。

### (b) Adem 关系

i < 2j：Sq^i Sq^j = Σ_{k=0}^{[i/2]} C(j-k-1, i-2k) Sq^{i+j-k} Sq^k。

### (c) 球面非平凡性

Sq^i 在 H^*(ℝP^∞; ℤ/2) 上非平凡。

球面上：若 Sq^i 非平凡，则存在 i 维非零同伦群。
