# 稳定同伦群

## 题目

**(a)** 证明稳定同伦群 $\pi_n^S$ 定义良好。

**(b)** 计算 $\pi_0^S, \pi_1^S, \pi_2^S$。

**(c)** 讨论稳定同伦群的周期性和模式。

## 解答

$\pi_n^S = \varinjlim \pi_{n+k}(S^k)$。

Serre计算：$\pi_0^S = \mathbb{Z}$，$\pi_1^S = \mathbb{Z}/2$，$\pi_2^S = \mathbb{Z}/2$。
