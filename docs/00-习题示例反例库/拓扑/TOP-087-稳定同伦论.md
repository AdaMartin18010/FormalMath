# 稳定同伦群

## 题目

**(a)** 证明稳定同伦群 $\pi_n^S$ 定义良好。

**(b)** 计算 $\pi_0^S, \pi_1^S, \pi_2^S$。

**(c)** 讨论稳定同伦群的周期性和模式。

---

## 解答

### (a) 稳定同伦群的定义

#### Freudenthal纬垂定理

**定理**（Freudenthal, 1937）：设 $X$ 是 $k$-连通的CW复形（$k \geq 0$），则纬垂同态：
$$\Sigma: \pi_n(X) \to \pi_{n+1}(\Sigma X)$$

在 $n \leq 2k$ 时是同构，在 $n = 2k + 1$ 时是满射。

**推论**：对球面 $S^r$（$(r-1)$-连通），
$$\pi_{n+r}(S^n) \xrightarrow{\Sigma} \pi_{n+r+1}(S^{n+1})$$

在 $n \geq r + 2$ 时是同构。

#### 稳定同伦群

**定义**：球面的第 $k$ 个**稳定同伦群**：
$$\pi_k^S = \varinjlim_{n \to \infty} \pi_{n+k}(S^n)$$

由Freudenthal定理，正向极限稳定化（$n \geq k + 2$ 后不再变化）。

#### 谱的视角

球谱 $\mathbb{S}$：$E_n = S^n$，结构映射 $\Sigma S^n \xrightarrow{\cong} S^{n+1}$。

$$\pi_k^S = \pi_k(\mathbb{S}) = \varinjlim_n \pi_{n+k}(S^n)$$

这是稳定同伦范畴中最基本的对象——球谱是单位对象（tensor unit）。

### (b) 低维稳定同伦群的计算

#### $\pi_0^S = \mathbb{Z}$

$\pi_n(S^n) = \mathbb{Z}$（由Hurewicz定理，$S^n$ 是 $(n-1)$-连通）。

纬垂 $\	ext{Sigma}: \pi_n(S^n) \to \pi_{n+1}(S^{n+1})$ 是同构（度保持）。

故 $\pi_0^S = \mathbb{Z}$，由映射度（mapping degree）参数化。

#### $\pi_1^S = \mathbb{Z}/2\mathbb{Z}$

**Hopf映射** $\eta: S^3 \to S^2$：
$$\eta(z_1, z_2) = [z_1 : z_2] \in \mathbb{C}P^1 \cong S^2$$

其中 $(z_1, z_2) \in S^3 \subseteq \mathbb{C}^2$，$|z_1|^2 + |z_2|^2 = 1$。

**稳定化**：$\Sigma^k \eta: S^{3+k} \to S^{2+k}$。

**Serre计算**（1951）：
- $\pi_4(S^3) = \mathbb{Z}/2$，由 $\eta$ 生成
- $\pi_{n+1}(S^n) = \mathbb{Z}/2$（$n \geq 3$），由 $\Sigma^{n-2}\eta$ 生成

故 $\pi_1^S = \mathbb{Z}/2$，由 $\eta$ 的稳定类生成。

**证明 $\eta$ 非零**：Hopf不变量 $H(\eta) = 1$。若 $\eta \simeq *$，则Hopf纤维化 $S^1 \to S^3 \to S^2$ 平凡，但 $S^3 \not\simeq S^1 \times S^2$（上同调环不同）。

#### $\pi_2^S = \mathbb{Z}/2\mathbb{Z}$

**生成元**：$\eta^2 = \eta \circ \Sigma\eta$。

**Serre定理**：$\pi_5(S^3) = \mathbb{Z}/2$，由 $\eta^2$ 生成。

对 $n \geq 3$：$\pi_{n+2}(S^n) = \mathbb{Z}/2$。

**证明**：用Serre谱序列或Adams谱序列。

- Serre：对纤维化 $S^3 \to K(\mathbb{Z}, 3) \to K(\mathbb{Z}, 4)$，用谱序列计算
- Adams：$E_2^{s,t} = \text{Ext}_{\mathcal{A}}^{s,t}(\mathbb{Z}/2, \mathbb{Z}/2)$，$d_2(\eta^2) = 0$

#### $\pi_3^S = \mathbb{Z}/24\mathbb{Z}$

这是第一个非平凡的稳定同伦群：

**生成元**：
- $\nu: S^7 \to S^4$（Hopf纤维化 $S^3 \to S^7 \to S^4$ 的稳定化）
- $\eta^3 = \eta \circ \Sigma\eta \circ \Sigma^2\eta$

关系：$\eta^3 = 12\nu$（在 $\pi_3^S$ 中）。

由Adams谱序列：$E_2$ 页有生成元 $h_0$（对应2）、$h_1$（对应$\eta$）、$h_2$（对应$\nu$）。

$d_2(h_0 h_2) = h_0^2 h_1^2$ 等关系使 $E_\infty$ 给出 $\pi_3^S = \mathbb{Z}/24$。

### (c) 周期性与模式

#### 经典模式

| $k$ | $\pi_k^S$ | 生成元 | 备注 |
|-----|-----------|--------|------|
| 0 | $\mathbb{Z}$ | $\iota$ | 单位 |
| 1 | $\mathbb{Z}/2$ | $\eta$ | Hopf |
| 2 | $\mathbb{Z}/2$ | $\eta^2$ | |
| 3 | $\mathbb{Z}/24$ | $\nu$ | Hopf |
| 4 | $0$ | — | |
| 5 | $0$ | — | |
| 6 | $\mathbb{Z}/2$ | $\nu^2$ | |
| 7 | $\mathbb{Z}/240$ | $\sigma$ | Hopf |
| 8 | $\mathbb{Z}/2 \oplus \mathbb{Z}/2$ | $\eta\sigma$, $\bar{\nu}$ | |

#### $v_n$-周期性

**Adams-Novikov谱序列**（用 $MU$ 或 $BP$）揭示周期性结构：

**Morava $K$-理论** $K(n)$ 对应形式群律的高度 $n$。$K(n)$-局部稳定同伦范畴有周期性：

**定理**（Hopkins-Smith, Ravenel）：
- $v_n$ 是 $2(p^n - 1)$ 周期的元素
- $v_n$-局部化后的稳定同伦群是**周期性的**，周期 $2(p^n - 1)$

**例子**：
- $v_1$ 周期 = $2(p-1)$（与复K理论 $KU$ 联系）
- $v_2$ 周期 = $2(p^2 - 1)$（与椭圆上同调联系）
- $v_n$ 周期 = $2(p^n - 1)$（与更高Morava $E$-理论联系）

#### 图像模式

在Adams谱序列的 $E_2$ 页上，稳定同伦群呈现"vanishing line" + "periodic families"的结构：

- **vanishing line**：$E_2^{s,t} = 0$ for $s > m(t - s) + c$（Cohen-Moore-Neisendorfer）
- **Greek letter元素**：$\alpha_n$（$v_1$族）、$\beta_n$（$v_2$族）、$\gamma_n$、$\delta_n$、...

#### 现代计算

**已知结果**：
- $k \leq 45$：完全计算（Ravenel, Toda, 等）
- $k \leq 60$：大部分已知
- $k \geq 60$：越来越困难，计算不完整

**关键工具**：
1. Adams谱序列（经典 + Novikov）
2. 拓扑循环同调（THH/TC）
3. 等变稳定同伦论
4. 计算机辅助（Bruner的程序）

#### 与数学其他领域的联系

1. **代数K理论**：$K_i(\mathbb{Z})$ 与 $\pi_i^S$ 通过Hurewicz映射联系
2. **微分拓扑**：球面上 exotic smooth structure 的存在性（Kervaire-Milnor）
3. **几何拓扑**：h-cobordism定理、手术理论
4. **数学物理**：弦论中的D-膜分类与K理论
