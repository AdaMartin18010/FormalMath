# 配边理论

## 题目

**(a)** 定义配边环 Ω_* = ⊕_n Ω_n 并证明其分次环结构。

**(b)** 证明配边类由示性数完全决定。

**(c)** 讨论复配边 MU 的形式群律。

## 解答

### (a) 配边环

[M] + [N] = [M ⊔ N]，[M]·[N] = [M × N]。

Thom：Ω_*^{SO} ≅ π_*^{SO} = π_{*+k}(MSO(k))。

### (b) 示性数

流形 M^{4n} 的 Pontryagin 数 p_I[M] = ⟨p_{i₁}···p_{i_r}, [M]⟩。

两个流形配边 ⟺ 所有示性数相等。

### (c) 复配边

MU^*(pt) = ℤ[x₁, x₂, ...]，|x_i| = 2i。

形式群律：Quillen 定理，MU 上的万有形式群律。
