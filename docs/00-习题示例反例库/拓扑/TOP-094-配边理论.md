# 配边理论

## 题目

**(a)** 定义配边环 $\Omega_* = \oplus_n \Omega_n$ 并证明其分次环结构。

**(b)** 证明配边类由示性数完全决定。

**(c)** 讨论复配边 $MU$ 的形式群律。

## 解答

### (a) 配边环的定义与分次环结构

**定义（配边关系）**：设 $M^n, N^n$ 是两个紧致无边（closed）的 $n$ 维光滑流形。称 $M$ 与 $N$ **配边**（cobordant），记作 $M \sim N$，若存在一个紧致带边的 $(n+1)$ 维光滑流形 $W^{n+1}$，使得其边界 $\partial W \cong M \sqcup N$（允许定向情形为 $M \sqcup (-N)$）。

**定理**：配边关系 $\sim$ 是等价关系。

*证明*：自反性：取 $W = M \times [0,1]$，则 $\partial W = M \sqcup M$。对称性由定义直接得到。传递性：设 $M \sim N$ 通过 $W_1$，$N \sim P$ 通过 $W_2$，则沿 $N$ 粘合 $W_1$ 与 $W_2$ 得到 $W = W_1 \cup_N W_2$，满足 $\partial W = M \sqcup P$。∎

记 $\Omega_n$ 为 $n$ 维流形配边类的集合。定义加法与乘法：
- **加法**：$[M] + [N] = [M \sqcup N]$（无交并）
- **乘法**：$[M] \cdot [N] = [M \times N]$（笛卡尔积）

**定理**：$(\Omega_*, +, \cdot)$ 构成交换分次环。

*证明*：
1. $(\Omega_n, +)$ 是 Abel 群：零元为 $[\varnothing]$，$[M]$ 的逆元在定向配边环中为 $[-M]$（非定向情形 $2[M] = [M \sqcup M] = [\partial(M \times [0,1])] = 0$，故每个元都是2阶的）。
2. 乘法良定：若 $M \sim M'$ 通过 $W$，则 $W \times N$ 给出 $M \times N \sim M' \times N$。
3. 分配律：$(M \sqcup M') \times N \cong (M \times N) \sqcup (M' \times N)$。
4. 分次性：$\Omega_m \cdot \Omega_n \subset \Omega_{m+n}$。∎

**Thom 定理**：定向配边环 $\Omega_*^{SO}$ 同构于 Thom 谱 $MSO$ 的稳定同伦群：
$$\Omega_*^{SO} \cong \pi_*^{SO} = \pi_{*+k}(MSO(k))$$
其中 $MSO(k)$ 是 $BSO(k)$ 的 Thom 空间。Thom 利用此同构结合谱序列计算，证明了 $\Omega_*^{SO}$ 是有限生成自由 Abel 群（除4的倍数维外），从而完全确定了配边环的结构。

### (b) 示性数完全决定配边类

**定义（Pontryagin 数）**：设 $M^{4n}$ 是 $4n$ 维定向闭流形，$p_i(M) \in H^{4i}(M; \mathbb{Z})$ 是其 $i$ 阶 Pontryagin 类。对分拆 $I = (i_1, \ldots, i_r)$ 满足 $i_1 + \cdots + i_r = n$，定义 Pontryagin 数：
$$p_I[M] = \langle p_{i_1}(M) \smile \cdots \smile p_{i_r}(M), [M] \rangle \in \mathbb{Z}$$
其中 $[M] \in H_{4n}(M; \mathbb{Z})$ 是基本类，$\langle \cdot, \cdot \rangle$ 表示 Kronecker 配对。

**定理（Thom-Pontryagin）**：两个 $4n$ 维定向闭流形 $M, N$ 定向配边当且仅当它们的所有 Pontryagin 数相等。即：
$$[M] = [N] \in \Omega_{4n}^{SO} \iff p_I[M] = p_I[N] \text{ 对所有 } I \text{ 成立}$$

*证明概要*：
- ($\Rightarrow$)：若 $M \sim N$，设 $\partial W = M \sqcup (-N)$。由于 Pontryagin 类是自然的且在带边流形上可定义，由 Stokes 型论证，$p_I[M] - p_I[N] = p_I[\partial W] = 0$。
- ($\Leftarrow$)：Thom 的关键步骤。利用 $\Omega_*^{SO} \otimes \mathbb{Q} \cong \mathbb{Q}[\mathbb{C}P^2, \mathbb{C}P^4, \ldots]$，即有理配边环由复射影空间的生成。由于 Pontryagin 数在 $MSO$ 的 Hurewicz 像上给出单射，且通过 Hirzebruch 符号差定理可验证这些数完全分类了生成元。

对于非定向配边环 $\Omega_*^O$，Stiefel-Whitney 数起类似作用：$M \sim N$ 当且仅当所有 Stiefel-Whitney 数相等（Thom 定理）。

**常见错误**：
- 误认为示性数在非定向情形由 Pontryagin 数决定；实际上非定向情形需要 Stiefel-Whitney 数。
- 忽视定向的重要性：$M$ 与 $-M$ 的 Pontryagin 数相同，但定向配边类差一个符号仅在高维有微妙影响。

### (c) 复配边与形式群律

**定义（复配边谱 $MU$）**：$MU$ 是 Thom 谱，其中 $MU_{2n} = MU(n)$ 是 $BU(n)$ 上的万有复向量丛的 Thom 空间。复配边群 $MU^*(X)$ 定义了广义上同调理论。

**定理（Quillen）**：$MU^*(pt) = MU^{-*}(pt)$ 是形式群律的万有分类环。具体地：
$$MU^*(pt) \cong \mathbb{Z}[x_1, x_2, \ldots], \quad |x_i| = 2i$$

复配边上同调与形式群律的联系如下：对复线丛 $L_1, L_2$，第一陈类在 $MU$ 中的形式群律为：
$$c_1^{MU}(L_1 \otimes L_2) = F(c_1^{MU}(L_1), c_1^{MU}(L_2))$$
其中 $F(x,y) = x + y + \sum_{i,j \geq 1} a_{ij} x^i y^j$ 是 $MU^*(pt)$ 上的万有形式群律。

**Quillen 定理**：$(MU^*(pt), F)$ 是 Lazard 环上的万有形式群律，即对任意形式群律 $(R, f)$，存在唯一的环同态 $\phi: MU^*(pt) \to R$ 使得 $\phi_*F = f$。

**证明思路**：Lazard 证明了形式群律的万有环 $L$ 同构于多项式环 $\mathbb{Z}[b_1, b_2, \ldots]$。Quillen 通过计算 $MU^*(\mathbb{C}P^\infty \times \mathbb{C}P^\infty)$ 及其对角映射，证明了 $MU^*(pt)$ 满足同样的泛性质。这建立了代数形式群律与拓扑复配边之间深刻的桥梁。

**例子**：在通常上同调 $H^*$ 中，$c_1(L_1 \otimes L_2) = c_1(L_1) + c_1(L_2)$，对应加法形式群律 $F(x,y) = x + y$。在 $K$ 理论中，$F(x,y) = x + y + \beta xy$，其中 $\beta$ 是 Bott 周期元，对应乘法形式群律。
