# 奇异同调基础

## 题目

**定义**：$n$-维奇异单形是连续映射 $\sigma: \Delta^n \to X$，其中：
$$\Delta^n = \{(t_0, \ldots, t_n) \in \mathbb{R}^{n+1} : t_i \geq 0, \sum t_i = 1\}$$

链复形 $C_n(X) = \mathbb{Z}\{\sigma: \Delta^n \to X\}$，边缘 $\partial: C_n \to C_{n-1}$：
$$\partial(\sigma) = \sum_{i=0}^n (-1)^i \sigma|_{[e_0, \ldots, \hat{e}_i, \ldots, e_n]}$$

(a) 验证 $\partial^2 = 0$。

(b) 计算 $H_0(X)$ 并解释其几何意义。

(c) 证明若 $f: X \to Y$ 同伦等价，则 $f_*: H_n(X) \to H_n(Y)$ 是同构。

## 解答

### (a) 边缘平方为零

**证明：**

设 $\sigma: \Delta^n \to X$。

$$\partial(\sigma) = \sum_{i=0}^n (-1)^i \sigma \circ d_i$$

其中 $d_i: \Delta^{n-1} \to \Delta^n$ 是第 $i$ 面嵌入。

$$\partial^2(\sigma) = \sum_{i=0}^n (-1)^i \sum_{j=0}^{n-1} (-1)^j \sigma \circ d_i \circ d_j$$

在 $\Delta^n$ 中，先跳过 $i$ 再跳过 $j$（$j < i$）等于先跳过 $j$ 再跳过 $i+1$（调整索引）。

即 $d_i \circ d_j = d_j \circ d_{i-1}$（$j < i$）。

计算 $(-1)^i(-1)^j + (-1)^j(-1)^{i-1} = 0$。

两项抵消，$\partial^2 = 0$。$\square$

### (b) $H_0(X)$ 的计算

**解答：**

$C_0(X) = \mathbb{Z}\{X\text{ 的点}\}$。

$C_1(X) = \mathbb{Z}\{X\text{ 中的道路}\}$。

$\partial: C_1 \to C_0$：对道路 $\gamma: [0,1] \to X$：
$$\partial(\gamma) = \gamma(1) - \gamma(0)$$

故：
- $\text{im}(\partial) = \{\sum n_i x_i : \sum n_i = 0\}$（边界为系数和零的形式和）
- $\ker(\partial) = C_0(X)$（在0维无边缘）

$$H_0(X) = C_0(X)/\text{im}(\partial) \cong \mathbb{Z}\{X\text{ 的道路连通分支}\}$$

**几何意义**：$H_0(X)$ 计数道路连通分支数。

$\square$

### (c) 同伦不变性

**证明：**

设 $f \simeq g: X \to Y$ 同伦，$H: X \times I \to Y$。

构造链同伦 $h: C_n(X) \to C_{n+1}(Y)$ 使：
$$\partial h + h \partial = g_* - f_*$$

这证明 $f_*$ 和 $g_*$ 在同调中相等。

具体：对 $\sigma: \Delta^n \to X$，定义棱柱算子。

$H \circ (\sigma \times \text{id}): \Delta^n \times I \to Y$。

将 $\Delta^n \times I$ 分划为 $n+1$ 个 $(n+1)$-单形，定义 $h(\sigma)$ 为它们的和。

若 $f$ 是同伦等价，有逆 $g$，则 $f_* \circ g_* = (f \circ g)_* = \text{id}_*$（同伦等于恒等）。

故 $f_*$ 是同构。$\square$

## 证明技巧

1. **符号计算**：边缘算子的组合验证
2. **0维解释**：连通分支的代数计数
3. **链同伦**：同伦的代数实现

## 常见错误

- ❌ 在(b)中忘记道路连通与单点同调的区别
- ❌ 链同伦的符号错误
- ❌ 忘记验证同伦逆的诱导映射
