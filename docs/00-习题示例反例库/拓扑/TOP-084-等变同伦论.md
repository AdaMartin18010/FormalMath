# 等变同伦论

## 题目

**(a)** 定义G-CW复形和G-同伦群。

**(b)** 证明Tom Dieck分裂：$\pi^G_*(\Sigma^\infty X) \otimes \mathbb{Q} \cong \bigoplus_{(H)} \pi_*(\Sigma^\infty X^H_{hW H}) \otimes \mathbb{Q}$。

**(c)** 讨论Segal猜想。

---

## 解答

### (a) G-CW复形与G-同伦群

#### G-空间

**定义**：拓扑群 $G$ 作用在空间 $X$ 上，若存在连续映射 $G \times X \to X$ 满足群作用公理。

**等变映射**：$f: X \to Y$ 满足 $f(gx) = gf(x)$。

#### G-CW复形

**定义**：$G$-CW复形是按维数归纳构造的空间：
- $X^{(-1)} = \emptyset$
- $X^{(n)}$ 由 $X^{(n-1)}$ 粘附 $G$-胞腔得到

**$G$-胞腔**：形如 $G/H \times D^n$，其中 $H \leq G$ 是闭子群，$G/H$ 是齐性空间。

粘附映射 $G/H \times S^{n-1} \to X^{(n-1)}$ 是 $G$-等变的。

**例子**：
- $S^1$ 带 $\mathbb{Z}/2$ 作用（对径映射）：CW结构有2个 $G/e$-胞腔（0维）和1个 $G/(\mathbb{Z}/2)$-胞腔（1维）
- $S(V)$（表示 $V$ 的单位球面）：$G$-CW结构由isotypic分解给出

#### 不动点与轨道

**不动点集**：$X^H = \{x \in X : hx = x, \forall h \in H\}$

**轨道空间**：$X/G$

**重要性质**：$(G/H)^K = $ 双陪集 $N_G(K) \backslash G / H$（当 $K$ 是子群时）

#### 等变同伦群

**定义**：对带基点的 $G$-空间 $X$、$Y$，**等变同伦群**：
$$[X, Y]^G = \pi_0(Map^G(X, Y))$$

更一般，对 $G$-表示 $V$：
$$\pi_V^G(X) = [S^V, X]^G$$

其中 $S^V$ 是 $V$ 的一点紧化。

**稳定等变同伦群**：
$$\pi_k^G(X) = \varinjlim_V [S^{V+k}, S^V \wedge X]^G$$

极限取遍所有 $G$-表示 $V$（在有限维子表示的有向集上）。

#### 正交G-谱

现代框架（Mandell-May）：$G$-谱是函子 $E: \mathbf{Orth}_G \to G\text{-Top}_*$，其中 $\mathbf{Orth}_G$ 是 $G$-内积空间范畴。

**稳定同伦范畴** $\mathbf{Sp}^G$ 有：
- 层结构（level model structure）
- 稳定结构（stable model structure）
- 正结构（positive model structure）

### (b) Tom Dieck分裂

#### 陈述

**定理**（Tom Dieck, 1975）：设 $G$ 是紧Lie群，$X$ 是 $G$-CW复形，则：

$$\pi^G_*(\Sigma^\infty X) \otimes \mathbb{Q} \cong \bigoplus_{(H)} \pi_*(\Sigma^\infty (X^H_{hW H})) \otimes \mathbb{Q}$$

其中：
- $(H)$ 跑遍 $G$ 的共轭类代表元
- $W H = N_G H / H$ 是Weyl群
- $X^H_{hW H} = (EWH \times X^H) / WH$ 是同伦轨道（homotopy orbit）

#### 证明概要

**步骤1：等变Spanier-Whitehead对偶**

对 $G$-空间 $X$，等变Spanier-Whitehead对偶 $D_G(X)$ 满足：
$$[S^V, X \wedge Y]^G \cong [D_G(X) \wedge S^V, Y]^G$$

**步骤2：等变换向定理（Equivariant Transfers）**

对有限 $G$-集 $S = G/H_1 \sqcup \cdots \sqcup G/H_k$，存在 transfer 映射：
$$\tau: X \to X \wedge S_+$$

使得复合 $X \xrightarrow{\tau} X \wedge S_+ \xrightarrow{\text{proj}} X$ 是乘法 by $|S|$（计重数）。

**步骤3：Mayer-Vietoris分解**

对 $G$-CW复形 $X$，用胞腔滤过和等变换向，将 $\pi^G_*(X)$ 分解为各轨道型的贡献。

**步骤4：有理化**

在有理化后，挠部分消失，各轨道型的贡献完全分离。

关键观察：$[G/H_+, X]^G \cong [S^0, X^H]^{WH} = \pi_0^{WH}(X^H)$。

**步骤5：归纳**

对 $G$-CW复形的骨架维数归纳，证明分裂在同伦范畴中成立。

#### 意义

Tom Dieck分裂将复杂的等变问题约化为：
1. 各不动点子集 $X^H$ 的非等变问题
2. Weyl群 $WH$ 的作用（已较简单）

这类似于表示论中"Mackey分解"或"轨道积分"的思想。

### (c) Segal猜想

#### 陈述

**Segal猜想**（1970s，由Carlsson、Laitinen、May、McClure、Adams等证明）：

设 $G$ 是有限群，$BG$ 是其分类空间，$\mathbb{S}$ 是球谱。则自然映射：
$$\pi^*_G(\mathbb{S}) \to \pi^*(BG_+)$$

在完备化后是同构。

更精确地：
$$\pi^G_*(\mathbb{S})^\wedge_{I(G)} \cong \pi^*(BG_+)$$

其中 $I(G)$ 是增广理想（augmentation ideal），$A^\wedge_I = \varprojlim_n A/I^n$。

#### 证明历史

**Carlsson定理**（1984，对 $G = \mathbb{Z}/2$）：

用Adams谱序列和 $G$-空间的cellular分解，证明 Segal映射诱导 $I(G)$-adic 完备化后的同构。

**Adams-Haeberly-Jackowski-May**（1988，一般有限群）：

用**等变稳定同伦的范畴论框架**和**单纯空间**技术证明一般情形。

**关键步骤**：
1. 将 $\pi^G_*(\mathbb{S})$ 表示为 $\varprojlim_V [S^V, S^V]^G$
2. 用 $G$-CW结构和Tom Dieck分裂分析各层
3. 证明 $I(G)$-adic 滤过与 skeleton 滤过在某意义下等价
4. 在完备化极限下，两者收敛到同一目标

#### 意义与应用

1. **等变上同调 ring 的计算**：$\pi^*_G(\mathbb{S})$ 是等变上同调环的类比，Segal猜想给出其与 $H^*(BG)$ 的联系

2. **Atiyah-Segal完备化定理**：对 $K$-理论：
$$K^*_G(X)^\wedge_{I(G)} \cong K^*(EG \times_G X)$$
这是Segal猜想在 $K$-理论中的类比。

3. **等变配边理论**：
$$\Omega^G_* \otimes \mathbb{Q} \cong \bigoplus_{(H)} \Omega_*^{WH} \otimes \mathbb{Q}$$

4. **现代发展**：
- Hill-Hopkins-Ravenel的等变谱序列（解决Kervaire不变量1问题）
- $C_2$-等变同伦论与实代数K理论（$KR$-理论）
- cyclotomic spectra 与拓扑循环同调（TC）
- equivariant elliptic cohomology 和 string theory
