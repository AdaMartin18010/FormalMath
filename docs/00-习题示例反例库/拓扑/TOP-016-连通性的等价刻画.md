## 题目

**(a)** 证明下列条件的等价性（连通性的定义）：
1. $X$ 不能分割为两个非空不交开集的并
2. $X$ 中既开又闭的子集只有 $\emptyset$ 和 $X$
3. 任何连续映射 $f: X \to \{0, 1\}$（离散拓扑）是常值映射

**(b)** 证明：$[0, 1]$ 连通。

**(c)** 证明：连续映射保持连通性（连通集的像连通）。

**(d)** 证明：连通空间的并（有公共点）连通。

## 详细解答

### (a) 连通性的等价刻画

**证明：**

**(1 $\Leftrightarrow$ 2)**

假设 $X = U \cup V$，$U, V$ 非空不交开集。

则 $U$ 开，$U = X \setminus V$ 闭，既开又闭，非平凡。

反之，若 $A$ 既开又闭，非平凡，则 $X = A \cup (X \setminus A)$ 是分割。

**(1 $\Leftrightarrow$ 3)**

假设 $f: X \to \{0, 1\}$ 连续，非常值。

则 $f^{-1}(0)$ 和 $f^{-1}(1)$ 是非空不交开集，分割 $X$。

反之，若 $X = U \cup V$ 分割，定义 $f|_U = 0$，$f|_V = 1$，连续非常值。

**证毕。**

### (b) $[0, 1]$ 的连通性

**证明：**

**反证：** 假设 $[0, 1] = A \cup B$ 分割，$0 \in A$。

设 $c = \sup\{x : [0, x] \subset A\}$。

**情形1：** $c \in A$。

$c < 1$（否则 $B = \emptyset$）。

$A$ 开，存在 $(c - \varepsilon, c + \varepsilon) \subset A$，矛盾于 $c$ 定义。

**情形2：** $c \in B$。

$B$ 开，存在 $(c - \varepsilon, c + \varepsilon) \subset B$。

但 $[0, c - \varepsilon/2] \subset A$，某点既在 $A$ 又在 $B$，矛盾。

**证毕。**

### (c) 连续映射保持连通性

**证明：**

设 $f: X \to Y$ 连续，$X$ 连通。

设 $g: f(X) \to \{0, 1\}$ 连续。

$g \circ f: X \to \{0, 1\}$ 连续。

$X$ 连通，$g \circ f$ 常值。

故 $g$ 常值，$f(X)$ 连通。

**证毕。**

### (d) 连通空间的并

**证明：**

设 $\{A_\alpha\}$ 连通，$\bigcap_\alpha A_\alpha \neq \emptyset$。

设 $f: \bigcup A_\alpha \to \{0, 1\}$ 连续。

每个 $A_\alpha$ 连通，$f|_{A_\alpha}$ 常值。

公共点处取值相同，故 $f$ 在所有 $A_\alpha$ 上取值相同。

$f$ 常值，$\bigcup A_\alpha$ 连通。

**证毕。**

## 证明技巧

1. **连通性的转换：** 分割 $\leftrightarrow$ 既开又闭集 $\leftrightarrow$ 到离散空间的映射
2. **区间的连通性：** 用上确界论证
3. **保持性的证明：** 复合映射到离散空间

## 常见错误

| 错误 | 说明 |
|------|------|
| 混淆连通与道路连通 | 道路连通更强 |
| 忘记既开又闭集的验证 | 需要两边都验证 |
| 并集的连通性条件 | 需要公共点 |

## 变式练习

**变式1：** 证明：$\mathbb{R}^n$ 连通。

**变式2：** 证明：连通集的闭包连通。

**变式3：** 构造$\mathbb{R}^2$中连通但不道路连通的子集（拓扑学家的正弦曲线）。
