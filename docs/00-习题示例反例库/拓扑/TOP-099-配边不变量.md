# 配边不变量

## 题目

**(a)** 证明 Stiefel-Whitney 数是未定向配边不变量。

**(b)** 证明 Pontryagin 数是有向配边不变量。

**(c)** 讨论标志差作为配边不变量及 Hirzebruch 符号差定理。

## 解答

### (a) Stiefel-Whitney 数

**定义**：对闭 $n$-流形 $M$，分区 $I = (i_1, \ldots, i_r)$，$\sum i_j = n$：

$$w_I[M] = \langle w_{i_1}(TM) \smile \cdots \smile w_{i_r}(TM), [M] \rangle \in \mathbb{Z}/2$$

其中 $w_i(TM) \in H^i(M; \mathbb{Z}/2)$ 是 Stiefel-Whitney 类。

**定理（Thom）**：$M^n$ 是未定向配边边界（$M = \partial W$）当且仅当所有 Stiefel-Whitney 数为零。

*证明*：

**必要性**：$W^{n+1}$，$M = \partial W$。切丛 $TW|_M = TM \oplus \nu$，$\nu$ 是法丛（平凡，因 $M = \partial W$）。

$w(TM) = w(TW|_M) = i^*w(TW)$。由 $H^{n+1}(W, M) \cong H^{n+1}(W/\partial W)$ 和长正合列，$w_I[M] = \langle i^*(w_I(TW)), [M] \rangle = 0$。

**充分性**：Thom 的构造，$MO$ 是 Eilenberg-MacLane 谱的 wedge 和。

---

### (b) Pontryagin 数

**定义**：对定向闭 $4n$-流形 $M$，分区 $I = (i_1, \ldots, i_r)$，$\sum i_j = n$：

$$p_I[M] = \langle p_{i_1}(TM) \smile \cdots \smile p_{i_r}(TM), [M] \rangle \in \mathbb{Z}$$

其中 $p_i(TM) \in H^{4i}(M; \mathbb{Z})$ 是 Pontryagin 类。

**定理（Thom）**：$M^{4n}$ 是有向配边边界当且仅当所有 Pontryagin 数为零。

*证明*：类似 Stiefel-Whitney 数，利用定向丛的 Euler 类和平行性。

**配边环**：

$$\Omega_*^O \otimes \mathbb{Z}/2 \cong \mathbb{Z}/2[x_i : i \neq 2^k-1]$$
$$\Omega_*^{SO} \otimes \mathbb{Q} \cong \mathbb{Q}[\mathbb{CP}^{2i}]$$

---

### (c) 标志差与 Hirzebruch 定理

**标志差**：对定向 $4n$-流形 $M$，

$$\sigma(M) = \dim H^{2n}(M; \mathbb{R})^+ - \dim H^{2n}(M; \mathbb{R})^-$$

由 cup 积的对称性定义。

**配边不变性**：若 $M = \partial W$，则 $\sigma(M) = 0$。

*证明*：$H^{2n}(M)$ 上的相交形式是 $H^{2n+1}(W, M) \to H^{2n+1}(W)$ 的边界的限制，故迷向子空间维数 = $b_{2n}/2$。

**Hirzebruch 符号差定理**：

$$\sigma(M) = L(p_1, \ldots, p_n)[M]$$

其中 $L$ 是 Hirzebruch $L$-genus：

$$L = \prod_i \frac{x_i/2}{\tanh(x_i/2)} = 1 + \frac{p_1}{3} + \frac{7p_2 - p_1^2}{45} + \cdots$$

**低维**：

- $n = 1$（$\dim = 4$）：$\sigma(M) = \frac{p_1[M]}{3}$
- $n = 2$（$\dim = 8$）：$\sigma(M) = \frac{7p_2 - p_1^2}{45}[M]$

---

## 常见错误

- **Stiefel-Whitney 数的系数**：是 $\mathbb{Z}/2$，非整数。
- **Pontryagin 数的维数**：仅对 $4n$ 维有定义（$p_i \in H^{4i}$）。
- **标志差的符号**：依赖定向，反向定向变号。
