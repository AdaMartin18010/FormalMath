# 配边不变量

## 题目

**(a)** 证明 Stiefel-Whitney 数是配边不变量。

**(b)** 证明 Pontryagin 数是有向配边不变量。

**(c)** 讨论标志差是配边不变量。

## 解答

### (a) Stiefel-Whitney 数

w_I[M] = ⟨w_{i₁}∪···∪w_{i_r}, [M]⟩，Σi_j = dim M。

W ∩ M = ∂W ⟹ 所有 Stiefel-Whitney 数为0。

Thom：示性数完全分类非有向配边。

### (b) Pontryagin 数

p_I[M^{4n}] = ⟨p_{i₁}∪···, [M]⟩。

W 有向，∂W = M ⟹ p_I[M] = 0。

### (c) 标志差

sign: Ω_{4n}^{SO} → ℤ 是满同态。

Hirzebruch 符号差定理：sign(M) = L(p₁,...,p_n)[M]。
