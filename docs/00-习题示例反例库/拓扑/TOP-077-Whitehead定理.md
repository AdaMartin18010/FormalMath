# Whitehead定理

## 题目

**(a)** 证明Whitehead定理：若 $f: X \to Y$ 是CW复形间的映射，诱导同伦群同构，则 $f$ 是同伦等价。

**(b)** 陈述并证明胞腔逼近定理。

**(c)** 讨论非CW空间的反例。

## 解答

### (a) Whitehead定理

**证明：**

用同伦群长正合列和归纳。

锥形构造扩张同伦逆。

### (b) 胞腔逼近

任意映射同伦于胞腔映射。

归纳于骨架维数。

### (c) 反例

拓扑学家的正弦曲线：弱同伦等价但非同伦等价。
