# Ricci 流

## 题目

**(a)** 证明 Ricci 流的短时间存在性。

**(b)** 讨论 Hamilton 的3维球定理。

**(c)** 概述 Perelman 对 Poincaré 猜想的证明。

## 解答

### (a) 短时间存在

Hamilton：Ricci 流 $\frac{\partial g}{\partial t} = -2\text{Ric}$。

DeTurck 技巧：引入向量场使方程严格抛物。

### (b) 球定理

3 维 Ricci 正流形：Ricci 流保持正曲率，收敛于常曲率度量。

### (c) Perelman 证明

手术 Ricci 流：处理奇点形成。

W-泛函、约化体积的单调性。

有限时间消没 ⟹ 几何化猜想 ⟹ Poincaré 猜想。
