## 题目

设 $(X, \mathcal{T})$ 是拓扑空间。

**(a)** 证明下列条件的等价性（紧致性的定义）：
1. 任何开覆盖有有限子覆盖
2. 具有有限交性质的闭集族有非空交
3. 任何网有收敛子网

**(b)** 证明：紧致的Hausdorff空间是正规空间。

**(c)** 证明：连续映射将紧致集映为紧致集。

**(d)** 证明：$[0, 1]$ 在标准拓扑下紧致（Heine-Borel定理的特殊情形）。

## 详细解答

### (a) 紧致性的等价刻画

**证明：**

**(1 $\Leftrightarrow$ 2)** 开覆盖与闭集的对应

开覆盖 $\{U_\alpha\}$ $\Leftrightarrow$ 闭集族 $\{F_\alpha = X \setminus U_\alpha\}$

$\{U_\alpha\}$ 覆盖 $X$ $\Leftrightarrow$ $\bigcap F_\alpha = \emptyset$

$\{U_\alpha\}$ 有有限子覆盖 $\Leftrightarrow$ 某有限子族 $F_{\alpha_1} \cap ... \cap F_{\alpha_n} = \emptyset$

逆否：紧致 $\Leftrightarrow$ 若任何有限交非空，则全交非空。

**(1 $\Rightarrow$ 3)** 用网刻画

设 $(x_\alpha)$ 是网。设 $A_\alpha = \{x_\beta : \beta \geq \alpha\}$。

$\{\overline{A_\alpha}\}$ 有有限交性质（因共尾）。

$\bigcap \overline{A_\alpha} \neq \emptyset$，取 $x$ 在此交中。

$x$ 是子网的极限。

**(3 $\Rightarrow$ 1)** 反证

设开覆盖 $\{U_\alpha\}$ 无有限子覆盖。

对有限子族 $F$，选 $x_F \notin \bigcup_{\alpha \in F} U_\alpha$。

这形成网，任何子网不收敛（因不进入某 $U_\alpha$）。

矛盾。

**证毕。**

### (b) 紧致Hausdorff空间的正规性

**证明：**

已知：紧致Hausdorff空间是正则的。

**步骤1：** 证明正规性

设 $A, B$ 是不交闭集。

对每个 $a \in A$，$b \in B$，有不交开邻域（Hausdorff）。

固定 $a$，$\{U_{a,b}\}_{b \in B}$ 覆盖 $B$。

$B$ 紧致（闭子集），有有限子覆盖 $U_{a,b_1}, ..., U_{a,b_n}$。

$V_a = \bigcap_{i=1}^n U_{a,b_i}$ 开，含 $a$，$\overline{V_a} \cap B = \emptyset$。

$\{V_a\}_{a \in A}$ 覆盖 $A$，$A$ 紧致，有有限子覆盖。

取并得开集 $U \supset A$，$\overline{U} \cap B = \emptyset$。

**证毕。**

### (c) 连续映射保持紧致性

**证明：**

设 $f: X \to Y$ 连续，$K \subset X$ 紧致。

设 $\{V_\alpha\}$ 是 $f(K)$ 的开覆盖。

$\{f^{-1}(V_\alpha)\}$ 是 $K$ 的开覆盖。

$K$ 紧致，有有限子覆盖 $f^{-1}(V_{\alpha_1}), ..., f^{-1}(V_{\alpha_n})$。

则 $V_{\alpha_1}, ..., V_{\alpha_n}$ 覆盖 $f(K)$。

**证毕。**

### (d) $[0, 1]$ 的紧致性

**证明：**

**方法：** 用有限交性质

设 $\{F_\alpha\}$ 是 $[0, 1]$ 的闭集族，有限交非空。

设 $a = \sup\{\inf(\bigcap_{i=1}^n F_{\alpha_i}) : \text{有限交}\}$。

验证 $a \in \bigcap F_\alpha$。

**另一方法：** 用序列紧致性（度量空间）

$[0, 1]$ 中任何序列有收敛子列（Bolzano-Weierstrass）。

度量空间中序列紧致 = 紧致。

**证毕。**

## 证明技巧

1. **紧致性的转换：** 开覆盖 $\leftrightarrow$ 闭集有限交
2. **分离性的提升：** Hausdorff $\to$ 正则 $\to$ 正规（需紧致）
3. **子网的应用：** 处理非度量空间的收敛

## 常见错误

| 错误 | 说明 |
|------|------|
| 混淆序列紧致与紧致 | 度量空间等价，一般空间不等价 |
| 忘记紧致集的闭子集紧致 | 需要空间Hausdorff保证闭 |
| 开覆盖的有限子覆盖构造错误 | 需仔细提取 |

## 变式练习

**变式1：** 证明$\mathbb{R}$中的紧致子集是有界闭集。

**变式2：** 证明：乘积空间紧致当且仅当各分量紧致（Tychonoff定理，$I$有限时）。

**变式3：** 证明：紧致度量空间是第二可数的。
