# Yamabe 问题

## 题目

**(a)** 证明 Yamabe 泛函的共形不变性。

**(b)** 讨论 Aubin-Schoen 的完备解。

**(c)** 证明紧致情况下的解的存在性。

## 解答

### (a) Yamabe 泛函

$$Q(g) = \frac{\int R_g dV_g}{\left(\int dV_g\right)^{(n-2)/n}}$$

共形变换下：g̃ = u^{4/(n-2)}g，Q 不变。

### (b) Aubin-Schoen

Yamabe 方程：-Δu + (n-2)/4(n-1)Ru = u^{(n+2)/(n-2)}。

Aubin：Y(M,g) ≤ Y(S^n) 时解存在。

Schoen：等号情形通过正质量定理解决。

### (c) 紧致存在性

任意紧致黎曼流形有常标量曲率度规。
