# 极小曲面

## 题目

**(a)** 证明极小曲面的第一变分公式。

**(b)** 讨论 Plateau 问题的 Douglas-Rado 解。

**(c)** 证明 Bernstein 定理。

## 解答

### (a) 第一变分

面积泛函 A(Σ) = ∫_Σ dA。

第一变分：δA = -∫ H ⟨X, N⟩ dA。

极小 ⟺ H = 0（平均曲率为零）。

### (b) Plateau 问题

给定边界 Γ，求面积极小曲面。

Douglas-Rado：Jordan 曲线存在 disk 型解。

### (c) Bernstein 定理

全平面极小图是线性函数（维数 ≤ 7）。

反例：Bombieri-De Giorgi-Giusti（维数 ≥ 8）。
