# 几何分析

## 题目

**(a)** 证明 Ricci 流在 Kähler 情形保持 Kähler 性质。

**(b)** 讨论 Kähler-Ricci 流的收敛性。

**(c)** 证明 Perelman 的熵单调性。

## 解答

### (a) Kähler-Ricci 流

$$\frac{\partial \omega}{\partial t} = -Ric(\omega)$$

Kähler 形式保持，复结构不变。

### (b) 收敛性

c₁ < 0：收敛到 Kähler-Einstein。

c₁ = 0：收敛到 Ricci 平坦（Cao）。

c₁ > 0：可能形成奇点（复维 ≥ 2）。

### (c) Perelman 熵

W(g, f, τ) = ∫ [τ(R + |∇f|²) + f - n](4πτ)^{-n/2}e^{-f}dV。

dW/dt ≥ 0，等号当且仅当 Ricci 孤立子。
