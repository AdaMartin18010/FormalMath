# 代数几何中的几何方法

## 题目

**(a)** 证明代数曲线的 Riemann-Roch 定理。

**(b)** 讨论 Jacobian 簇的几何。

**(c)** 证明 Torelli 定理。

## 解答

### (a) Riemann-Roch

对曲线 X 上线丛 L：
$$h^0(L) - h^1(L) = deg(L) + 1 - g$$

或 Serre 对偶：h¹(L) = h⁰(K - L)。

### (b) Jacobian

$$Jac(X) = H^0(X, K)^*/H_1(X, ℤ) \cong Pic^0(X)$$

主极化 Abel 簇。

### (c) Torelli 定理

曲线 X 由其 Jacobian (Jac(X), Θ) 决定。

证明：从 Θ 除子恢复曲线。
