# Lie导数与Cartan公式

## 题目

设 $M$ 是光滑流形，$X$ 是向量场，$\omega$ 是微分形式。

**定义**：
- **Lie导数** $\mathcal{L}_X\omega = \frac{d}{dt}|_{t=0} \phi_t^*\omega$，其中 $\phi_t$ 是 $X$ 的流
- **内积** $i_X\omega(Y_1, \ldots, Y_{k-1}) = \omega(X, Y_1, \ldots, Y_{k-1})$

(a) 证明 Cartan魔法公式：
$$\mathcal{L}_X = d \circ i_X + i_X \circ d$$

(b) 证明 $\mathcal{L}_X(\alpha \wedge \beta) = \mathcal{L}_X\alpha \wedge \beta + \alpha \wedge \mathcal{L}_X\beta$。

(c) 若 $X, Y$ 是Killing向量场（保持度量），证明 $[X, Y]$ 也是。

## 解答

### (a) Cartan魔法公式

**证明：**

验证两边在 $k$-形式上相等。

**对0-形式（函数）**：
- 左边：$\mathcal{L}_X f = X(f)$
- 右边：$i_X(df) + d(i_X f) = df(X) + 0 = X(f)$ ✓

**对1-形式**：

设 $\omega = f dg$，验证两边作用在向量场 $Y$ 上相等。

由线性性和局部性，推广到一般形式。

**归纳**：假设对 $(k-1)$-形式成立。

对 $\omega = \alpha \wedge \beta$，$\alpha$ 1-形式，$\beta$ $(k-1)$-形式：

由Leibniz规则和归纳假设，两边满足相同乘积公式，故相等。

**替代证明**：验证 $d i_X + i_X d$ 满足Lie导数的刻画（与流的关系）。$\square$

### (b) Lie导数的Leibniz规则

**证明：**

由Cartan公式和 $d$、$i_X$ 的Leibniz规则：

$$\mathcal{L}_X(\alpha \wedge \beta) = d(i_X(\alpha \wedge \beta)) + i_X(d(\alpha \wedge \beta))$$

$$= d(i_X\alpha \wedge \beta + (-1)^{|\alpha|}\alpha \wedge i_X\beta) + i_X(d\alpha \wedge \beta + (-1)^{|\alpha|}\alpha \wedge d\beta)$$

展开，用 $d$ 和 $i_X$ 的Leibniz规则，交叉项抵消，得：
$$= (d i_X + i_X d)\alpha \wedge \beta + \alpha \wedge (d i_X + i_X d)\beta$$

$$= \mathcal{L}_X\alpha \wedge \beta + \alpha \wedge \mathcal{L}_X\beta$$

$\square$

### (c) Killing向量场的李括号

**证明：**

$X$ 是Killing场若 $\mathcal{L}_X g = 0$（$g$ 是度量）。

需证 $\mathcal{L}_{[X,Y]} g = 0$。

由Jacobi恒等式的变体：$\mathcal{L}_{[X,Y]} = [\mathcal{L}_X, \mathcal{L}_Y]$。

验证：
$$\mathcal{L}_{[X,Y]}g = \mathcal{L}_X\mathcal{L}_Y g - \mathcal{L}_Y\mathcal{L}_X g = 0 - 0 = 0$$

或用：$[X,Y]$ 的流是 $\phi_X^t \circ \phi_Y^s \circ \phi_X^{-t} \circ \phi_Y^{-s}$ 的极限。

每个 $
\phi$ 保持度量，故复合保持度量。$\square$

## 证明技巧

1. **局部验证+归纳**：张量等式的标准证明
2. **Leibniz规则**：微分算子的特征性质
3. **几何意义**：Killing场的李代数结构

## 常见错误

- ❌ 忘记符号 $(-1)^{|\alpha|}$
- ❌ 在归纳中基底情形不完整
- ❌ 混淆 $
\mathcal{L}_{[X,Y]}$ 和 $[\mathcal{L}_X, \mathcal{L}_Y]$ 的关系

## 变式练习

**变式1：** 证明 $i_{[X,Y]} = [\mathcal{L}_X, i_Y]$。

**变式2：** 用Cartan公式证明de Rham上同调的同伦不变性。
