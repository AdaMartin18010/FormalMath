# Riemann曲率张量

## 题目

设 $(M, g)$ 是黎曼流形，$\nabla$ 是Levi-Civita联络。

**(a)** 定义Riemann曲率张量：
$$R(X,Y)Z = \nabla_X\nabla_Y Z - \nabla_Y\nabla_X Z - \nabla_{[X,Y]}Z$$
证明其代数性质：反对称、第一Bianchi恒等式。

**(b)** 定义Ricci曲率和标量曲率。证明 $\text{Ric}$ 是对称 $(0,2)$-张量。

**(c)** 证明常截面曲率空间满足 $R_{ijkl} = K(g_{ik}g_{jl} - g_{il}g_{jk})$。

---

## 解答

### (a) 曲率张量性质

**反对称：**
$$R(X,Y) = -R(Y,X)$$
由定义显然。

**详细证明**：
$$R(Y,X)Z = \nabla_Y\nabla_X Z - \nabla_X\nabla_Y Z - \nabla_{[Y,X]}Z$$
$$= -(\nabla_X\nabla_Y Z - \nabla_Y\nabla_X Z) - \nabla_{-[X,Y]}Z$$
$$= -(\nabla_X\nabla_Y Z - \nabla_Y\nabla_X Z - \nabla_{[X,Y]}Z)$$
$$= -R(X,Y)Z$$

其中用到 $[Y,X] = -[X,Y]$。

**第一Bianchi：**
$$R(X,Y)Z + R(Y,Z)X + R(Z,X)Y = 0$$

由Jacobi恒等式和挠率自由。

**详细证明**：

由于Levi-Civita联络无挠：$\nabla_X Y - \nabla_Y X = [X,Y]$。

展开 $R(X,Y)Z + R(Y,Z)X + R(Z,X)Y$：

第一项 $R(X,Y)Z = \nabla_X\nabla_Y Z - \nabla_Y\nabla_X Z - \nabla_{[X,Y]}Z$

第二项 $R(Y,Z)X = \nabla_Y\nabla_Z X - \nabla_Z\nabla_Y X - \nabla_{[Y,Z]}X$

第三项 $R(Z,X)Y = \nabla_Z\nabla_X Y - \nabla_X\nabla_Z Y - \nabla_{[Z,X]}Y$

利用无挠性 $[X,Y] = \nabla_X Y - \nabla_Y X$，有：
$$\nabla_{[X,Y]}Z = \nabla_{\nabla_X Y - \nabla_Y X}Z = \nabla_{\nabla_X Y}Z - \nabla_{\nabla_Y X}Z$$

将所有项重新组合，利用Jacobi恒等式 $[[X,Y],Z] + [[Y,Z],X] + [[Z,X],Y] = 0$，可以验证所有项相消。$\square$

---

### (b) Ricci曲率

**定义：**
$$\text{Ric}(X,Y) = \text{tr}(Z \mapsto R(Z,X)Y)$$

在标准正交基 $\{e_i\}$：
$$R_{ij} = R^k_{\,ikj}$$

**对称性：**
由曲率的对称性和Bianchi恒等式得 $R_{ij} = R_{ji}$。

**详细证明**：

在局部坐标下，Ricci张量：
$$R_{ij} = g^{kl} R_{kilj}$$

利用曲率张量的对称性 $R_{kilj} = R_{ljki}$（由度量相容性和第一Bianchi恒等式导出）：

$$R_{ij} = g^{kl} R_{kilj} = g^{kl} R_{ljki} = g^{lk} R_{ljki} = R_{ji}$$

其中最后一步使用了哑指标重命名 $k \leftrightarrow l$。

**标量曲率：**
$$R = g^{ij}R_{ij}$$

标量曲率是Ricci张量的迹（关于度量的缩并）。在二维，标量曲率与高斯曲率的关系为 $R = 2K$。

### (c) 常曲率空间

**证明：**

截面曲率 $K(\Pi) = \frac{\langle R(e_1, e_2)e_2, e_1\rangle}{|e_1|^2|e_2|^2 - \langle e_1, e_2\rangle^2}$ 常数。

张量 $T_{ijkl} = R_{ijkl} - K(g_{ik}g_{jl} - g_{il}g_{jk})$ 满足所有曲率对称性。

对任意平面，$T = 0$，故张量恒零。

**详细证明**：

**步骤1：验证 $K(g_{ik}g_{jl} - g_{il}g_{jk})$ 满足曲率对称性**

设 $S_{ijkl} = K(g_{ik}g_{jl} - g_{il}g_{jk})$。

- 反对称1：$S_{ijkl} = -S_{jikl}$：
  $$S_{jikl} = K(g_{jk}g_{il} - g_{jl}g_{ik}) = -K(g_{ik}g_{jl} - g_{il}g_{jk}) = -S_{ijkl}$$

- 反对称2：$S_{ijkl} = -S_{ijlk}$：
  $$S_{ijlk} = K(g_{il}g_{jk} - g_{ik}g_{jl}) = -S_{ijkl}$$

- 第一Bianchi：$S_{ijkl} + S_{kijl} + S_{jkil} = 0$：
  直接展开验证即可。

**步骤2：证明 $T = R - S$ 在截面曲率意义下为零**

对任意二维平面 $\Pi = \text{span}\{e_1, e_2\}$（标准正交）：

$$T(e_1, e_2, e_2, e_1) = R(e_1, e_2, e_2, e_1) - K(g_{12}g_{21} - g_{11}g_{22})$$
$$= K \cdot 1 - K(0 - 1) = K - K = 0$$

（这里 $R(e_1, e_2, e_2, e_1) = \langle R(e_1, e_2)e_2, e_1\rangle = K$ 由截面曲率定义）

**步骤3：由代数引理得 $T \equiv 0$**

**引理**：若张量 $T_{ijkl}$ 满足所有曲率对称性，且对所有标准正交对 $(e_1, e_2)$ 有 $T(e_1, e_2, e_2, e_1) = 0$，则 $T \equiv 0$。

证明：利用极化恒等式。对任意向量 $X, Y, Z, W$：
$$T(X,Y,Z,W) = \frac{1}{4}[T(X+Z, Y+W, ...) - ...]$$

通过适当的线性组合，可以将任意分量表示为截面曲率形式的组合。由于所有截面曲率形式为零，$T \equiv 0$。

因此 $R_{ijkl} = K(g_{ik}g_{jl} - g_{il}g_{jk})$。$\square$

## 几何意义

### 截面曲率

对二维平面 $\Pi \subset T_p M$，截面曲率 $K(\Pi)$ 是过 $p$ 点沿 $\Pi$ 方向的测地曲面的高斯曲率。

- $K > 0$：球面型几何（三角形内角和 $> \pi$）
- $K = 0$：欧几里得几何（三角形内角和 $= \pi$）
- $K < 0$：双曲几何（三角形内角和 $< \pi$）

### 常曲率空间

常曲率空间是**均匀**的——每点沿每个方向的弯曲程度相同。经典例子：
- $K = 1$：球面 $S^n$
- $K = 0$：欧氏空间 $\mathbb{R}^n$
- $K = -1$：双曲空间 $\mathbb{H}^n$

**单值化定理**（二维）：任意闭黎曼曲面容许常曲率度量。
