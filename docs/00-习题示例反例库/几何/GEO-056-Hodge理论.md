# Hodge 理论深入

## 题目

**(a)** 证明 Lefschetz 分解定理。

**(b)** 讨论 Hodge-Riemann 双线性关系。

**(c)** 证明 hard Lefschetz 定理。

## 解答

### (a) Lefschetz 分解

$$H^k(X, ℂ) = \bigoplus_{j \geq 0} L^j P^{k-2j}$$

P = ker(Λ) 本原上同调，L = ω∧。

### (b) Hodge-Riemann 关系

Q(α, β) = i^{p-q} ∫_X α ∧ β̄ 在 H^{p,q} 上定号。

本原形式：(-1)^{k(k+1)/2} i^{p-q} Q(α, ᾱ) > 0。

### (c) Hard Lefschetz

L^{n-k}: H^k → H^{2n-k} 同构。

证明：表示论 sl₂(ℂ) = ⟨L, Λ, H⟩。
