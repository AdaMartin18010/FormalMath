# 辛流形基础

## 题目

**(a)** 设 $(M, \omega)$ 是辛流形，$\omega \in \Omega^2(M)$ 闭且非退化。证明 $\dim M$ 是偶数。

**(b)** 定义Hamiltonian向量场：对 $H \in C^\infty(M)$，$X_H$ 满足 $\iota_{X_H}\omega = dH$。证明 $\mathcal{L}_{X_H}\omega = 0$。

**(c)** 证明Darboux定理：任意辛流形局部辛同胚于 $(\mathbb{R}^{2n}, \omega_0)$，其中 $\omega_0 = \sum dx_i \wedge dy_i$。

---

## 解答

### (a) 偶数维数

**证明：**

$\omega$ 非退化 $\Leftrightarrow$ $\omega^{\wedge n} \neq 0$，$2n = \dim M$。

反对称双线性型的最大秩为偶数。

**详细展开**：

**线性代数引理**：设 $V$ 是有限维向量空间，$\omega: V \times V \to \mathbb{R}$ 是反对称双线性型（即 $\omega(v, w) = -\omega(w, v)$）。若 $\omega$ 非退化（即 $\omega(v, w) = 0$ 对所有 $w$ 蕴含 $v = 0$），则 $\dim V$ 是偶数。

**证明引理**：设 $\dim V = m$。取 $V$ 的一组基，$\omega$ 的矩阵表示 $A$ 满足 $A^T = -A$。

非退化意味着 $\det(A) \neq 0$。

但：
$$\det(A) = \det(A^T) = \det(-A) = (-1)^m \det(A)$$

若 $m$ 为奇数，则 $\det(A) = -\det(A)$，即 $\det(A) = 0$，与非退化矛盾。

因此 $m$ 必须是偶数。$\square$

**辛流形的情形**：在每点 $p \in M$，$\omega_p: T_pM \times T_pM \to \mathbb{R}$ 是切空间上的非退化反对称双线性型。由引理，$\dim T_pM = \dim M$ 是偶数。

---

### (b) Hamiltonian流保持辛结构

**证明：**
$$\mathcal{L}_{X_H}\omega = d\iota_{X_H}\omega + \iota_{X_H}d\omega = d(dH) + 0 = 0$$

**详细展开**：

**Cartan magic formula**：对任意向量场 $X$ 和微分形式 $\alpha$：
$$\mathcal{L}_X \alpha = d\iota_X \alpha + \iota_X d\alpha$$

应用到 $X_H$ 和 $\omega$：
$$\mathcal{L}_{X_H}\omega = d(\iota_{X_H}\omega) + \iota_{X_H}(d\omega)$$

**第一项**：由Hamiltonian向量场的定义，$\iota_{X_H}\omega = dH$。因此：
$$d(\iota_{X_H}\omega) = d(dH) = 0$$

（$d^2 = 0$）

**第二项**：$\omega$ 是辛形式，故 $d\omega = 0$。因此：
$$\iota_{X_H}(d\omega) = 0$$

综合：$\mathcal{L}_{X_H}\omega = 0 + 0 = 0$。$\square$

**物理意义**：$\mathcal{L}_{X_H}\omega = 0$ 意味着Hamiltonian流（即由 $X_H$ 生成的单参数变换群）**保持辛结构**。在经典力学中，这对应于**Liouville定理**——相空间体积（由 $\omega^n$ 给出）在Hamiltonian流下保持不变。

**等价表述**：设 $\phi_t$ 是 $X_H$ 的流，则 $\phi_t^*\omega = \omega$ 对所有 $t$ 成立。

### (c) Darboux定理

**证明概要**：

Moser技巧：设 $\omega_t = (1-t)\omega_0 + t\omega$，证存在 $\phi_t$ 使 $\phi_t^*\omega_t = \omega_0$。

需解同痕方程，用Poincaré引理和辛结构。

**详细展开**：

**Darboux定理的核心信息**：与Riemann几何不同，辛几何**没有局部不变量**。所有 $2n$-维辛流形在每点附近都"看起来一样"。

**Moser技巧的证明**：

**步骤1：在点 $p$ 附近标准化**

设 $p \in M$。由线性代数，存在 $T_pM$ 的基 $\{e_1, f_1, ..., e_n, f_n\}$ 使得：
$$\omega_p(e_i, e_j) = \omega_p(f_i, f_j) = 0, \quad \omega_p(e_i, f_j) = \delta_{ij}$$

取以 $p$ 为中心的局部坐标 $(x_1, y_1, ..., x_n, y_n)$，使得在该坐标下：
$$\omega(p) = \sum_{i=1}^n dx_i \wedge dy_i$$

**步骤2：构造族 $\omega_t$**

在 $p$ 的邻域 $U$ 上，定义：
$$\omega_t = (1-t)\omega_0 + t\omega$$

其中 $\omega_0 = \sum dx_i \wedge dy_i$。

**关键观察**：对足够小的 $U$，$\omega_t$ 对所有 $t \in [0,1]$ 都是非退化的（非退化是开条件）。

**步骤3：解同痕方程**

寻找向量场族 $X_t$ 使得：
$$\mathcal{L}_{X_t}\omega_t + \frac{d\omega_t}{dt} = 0$$

即：
$$d\iota_{X_t}\omega_t = \omega_0 - \omega$$

由Poincaré引理（在足够小的邻域上），$\omega_0 - \omega$ 是恰当的：$\omega_0 - \omega = d\alpha$。

由于 $\omega_t$ 非退化，存在唯一的 $X_t$ 使 $\iota_{X_t}\omega_t = -\alpha$。

**步骤4：积分得到辛同胚**

设 $\phi_t$ 是 $X_t$ 的流（$\phi_0 = \text{id}$）。则：
$$\frac{d}{dt}\phi_t^*\omega_t = \phi_t^*(\mathcal{L}_{X_t}\omega_t + \frac{d\omega_t}{dt}) = 0$$

因此 $\phi_t^*\omega_t = \text{const} = \omega_0$。

特别地，$\phi_1^*\omega = \omega_0$，即 $\phi_1$ 是从 $(U, \omega)$ 到 $(U, \omega_0)$ 的辛同胚。$\square$

**与Riemann几何的对比**：

在Riemann几何中，曲率是局部不变量。球面和平面在每点都有不同的高斯曲率，因此不能局部等距。但在辛几何中，Darboux定理表明**没有曲率类的局部不变量**——所有辛流形局部都等价于标准辛空间。

## 拓展联系

### 辛几何与经典力学

Hamiltonian力学可以天然地用辛几何描述：
- **构型空间** $Q$ 的**相空间**是余切丛 $T^*Q$
- $T^*Q$ 有**典范辛结构** $\omega = d\theta$，其中 $\theta$ 是Liouville 1-形式
- **Hamilton方程** $\dot{q} = \frac{\partial H}{\partial p}$, $\dot{p} = -\frac{\partial H}{\partial q}$ 等价于 $\iota_X \omega = dH$

### 与复几何的联系

Kähler流形同时是：
1. 辛流形（带有辛形式 $\omega$）
2. 复流形（带有复结构 $J$）
3. Riemann流形（带有度量 $g$）

三者满足相容条件：$\omega(v, w) = g(Jv, w)$。

Kähler几何是代数几何、表示论和弦理论的交汇点。
