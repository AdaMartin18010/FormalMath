## 题目

设 $(M, g)$ 是黎曼流形。

**(a)** 定义**黎曼度量**：光滑地对每点 $p$ 给 $T_p M$ 内积 $g_p$。证明任何光滑流形可赋予黎曼度量。

**(b)** 定义**仿射联络**：$\nabla: \mathfrak{X}(M) \times \mathfrak{X}(M) \to \mathfrak{X}(M)$，满足线性、Leibniz等。给出Christoffel符号表示。

**(c)** 证明**Levi-Civita定理**：存在唯一的无挠联络与度量相容。

**(d)** 计算 $(\mathbb{R}^n, g_{ij} = \delta_{ij})$ 的Levi-Civita联络。

## 详细解答

### (a) 黎曼度量的存在性

**证明：**

**单位分解方法：**

设 $\{(U_\alpha, \phi_\alpha)\}$ 是可数局部有限坐标覆盖。

在 $U_\alpha$ 上，用拉回标准度量：$g_\alpha = \phi_\alpha^* (\sum (dx^i)^2)$。

设 $\{\rho_\alpha\}$ 是从属于 $\{U_\alpha\}$ 的单位分解。

定义：$g = \sum_\alpha \rho_\alpha g_\alpha$。

**验证正定：**

在每点 $p$，某 $\rho_\alpha(p) > 0$，$g_\alpha(p)$ 正定。

凸组合保持正定性。

**证毕。**

### (b) 仿射联络与Christoffel符号

**定义：**

$\nabla_X Y$（$X$ 沿 $Y$ 的协变导数）满足：
1. $\nabla_{fX + gZ} Y = f\nabla_X Y + g\nabla_Z Y$
2. $\nabla_X (aY + bZ) = a\nabla_X Y + b\nabla_X Z$
3. $\nabla_X (fY) = X(f) Y + f\nabla_X Y$（Leibniz）

**Christoffel符号：**

在坐标 $\{x^i\}$ 中，$\nabla_{\partial_i} \partial_j = \sum_k \Gamma_{ij}^k \partial_k$。

对 $X = \sum X^i \partial_i$，$Y = \sum Y^j \partial_j$：
$$\nabla_X Y = \sum_k \left(\sum_i X^i \frac{\partial Y^k}{\partial x^i} + \sum_{i,j} X^i Y^j \Gamma_{ij}^k\right) \partial_k$$

### (c) Levi-Civita定理

**定理：** 存在唯一的联络满足：
1. **无挠：** $T(X, Y) = \nabla_X Y - \nabla_Y X - [X, Y] = 0$
2. **与度量相容：** $X\langle Y, Z\rangle = \langle\nabla_X Y, Z\rangle + \langle Y, \nabla_X Z\rangle$

**证明：**

**Koszul公式（由条件导出）：**

$$2\langle\nabla_X Y, Z\rangle = X\langle Y, Z\rangle + Y\langle Z, X\rangle - Z\langle X, Y\rangle$$
$$- \langle X, [Y, Z]\rangle + \langle Y, [Z, X]\rangle + \langle Z, [X, Y]\rangle$$

**唯一性：** 右边由 $g$ 和 $[\cdot, \cdot]$ 决定，$\nabla_X Y$ 唯一。

**存在性：** 验证由上式定义的 $\nabla$ 满足联络公理和无挠、相容条件。

**Christoffel符号显式公式：**
$$\Gamma_{ij}^k = \frac{1}{2}\sum_l g^{kl}\left(\frac{\partial g_{jl}}{\partial x^i} + \frac{\partial g_{il}}{\partial x^j} - \frac{\partial g_{ij}}{\partial x^l}\right)$$

**证毕。**

### (d) 欧氏空间的联络

**计算：**

$g_{ij} = \delta_{ij}$，常数。

$\frac{\partial g_{ij}}{\partial x^k} = 0$。

Christoffel符号：$\Gamma_{ij}^k = 0$。

Levi-Civita联络：$\nabla_X Y = \sum_k \left(\sum_i X^i \frac{\partial Y^k}{\partial x^i}\right) \partial_k$。

即方向导数！

**证毕。**

## 证明技巧

1. **度量的存在：** 单位分解拼合局部度量
2. **联络的唯一性：** Koszul公式显式解
3. **Christoffel符号：** 度量分量的导数组合

## 常见错误

| 错误 | 说明 |
|------|------|
| 忘记验证正定性 | 凸组合保持正定 |
| 混淆张量变换律 | Christoffel不张量 |
| 挠率的计算错误 | 需要$[X,Y]$项 |
