# Dirac算子与指标

## 题目

**(a)** 定义Clifford代数和旋量表示。

**(b)** 证明Lichnerowicz公式：$D^2 = \nabla^*\nabla + \frac{1}{4}R$。

**(c)** 用Bochner技巧证明正标量曲率障碍。

## 解答

### (a) Clifford代数

$Cl(V, Q) = T(V) / \langle v \otimes v - Q(v) \rangle$。

### (b) Lichnerowicz公式

Dirac算子的Weitzenböck公式。

曲率项来自旋联络。

### (c) Bochner技巧

正标量曲率时调和旋量消失。
