# 比较几何

## 题目

**(a)** 陈述Toponogov比较定理：截面曲率 $\geq k$ 的空间中，三角形角度 $\geq$ 模型空间。

**(b)** 证明Gromov预紧性定理：截面曲率有下界、直径有上界的黎曼流形序列有收敛子列。

**(c)** 讨论Alexandrov空间的坍塌理论。

## 解答

### (a) Toponogov定理

曲率下界蕴含角度比较。

### (b) Gromov预紧性

Gromov-Hausdorff收敛，有界几何条件的紧致性。

### (c) 坍塌

纤维丛结构的极限，F-结构理论。
