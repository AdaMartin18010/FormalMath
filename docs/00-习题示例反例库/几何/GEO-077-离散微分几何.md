# 离散微分几何

## 题目

**(a)** 定义离散曲面的高斯曲率。

**(b)** 讨论离散极小曲面的性质。

**(c)** 证明离散可积系统。

## 解答

### (a) 离散高斯曲率

多面体顶点：K(v) = 2π - Σθ_i，θ_i 是面角。

离散 Gauss-Bonnet：ΣK(v) = 2πχ。

### (b) 离散极小曲面

面积变分为零。

平均曲率流：cotangent 权重的 Laplacian。

### (c) 离散可积

圆丛 pattern：每四边共圆。

离散 KdV、离散 Schrödinger 流。
