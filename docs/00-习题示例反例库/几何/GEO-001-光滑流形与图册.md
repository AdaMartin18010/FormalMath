## 题目

**(a)** 定义**拓扑流形**：局部欧几里得的Hausdorff第二可数空间。证明 $S^n$ 是 $n$ 维拓扑流形。

**(b)** 定义**光滑图册**：坐标卡集合覆盖流形，过渡映射光滑。给出 $S^2$ 的光滑图册。

**(c)** 证明：$\mathbb{R}P^n$ 有自然的光滑结构。

**(d)** 定义**光滑映射** $f: M \to N$。证明：光滑映射的复合光滑。

## 详细解答

### (a) 球面是流形

**证明：**

$S^n = \{x \in \mathbb{R}^{n+1} : |x| = 1\}$。

**Hausdorff：** 作为度量空间的子空间。

**第二可数：** $\mathbb{R}^{n+1}$ 子空间。

**局部欧几里得：** 球极投影。

北极投影 $\phi_N: S^n \setminus \{N\} \to \mathbb{R}^n$：
$$\phi_N(x_1, ..., x_{n+1}) = \frac{(x_1, ..., x_n)}{1 - x_{n+1}}$$

同理南极投影 $\phi_S$。

$\{(S^n \setminus \{N\}, \phi_N), (S^n \setminus \{S\}, \phi_S)\}$ 覆盖 $S^n$。

**证毕。**

### (b) $S^2$ 的光滑图册

**球极投影坐标：**

$(U_N, \phi_N)$：$U_N = S^2 \setminus \{(0,0,1)\}$，
$$\phi_N(x, y, z) = \left(\frac{x}{1-z}, \frac{y}{1-z}\right)$$

$(U_S, \phi_S)$：$U_S = S^2 \setminus \{(0,0,-1)\}$，
$$\phi_S(x, y, z) = \left(\frac{x}{1+z}, \frac{y}{1+z}\right)$$

**过渡映射：**

$\phi_S \circ \phi_N^{-1}: \mathbb{R}^2 \setminus \{0\} \to \mathbb{R}^2 \setminus \{0\}$

$\phi_N^{-1}(u, v) = \frac{(2u, 2v, u^2+v^2-1)}{u^2+v^2+1}$

计算得：
$$\phi_S \circ \phi_N^{-1}(u, v) = \left(\frac{u}{u^2+v^2}, \frac{v}{u^2+v^2}\right)$$

在 $\mathbb{R}^2 \setminus \{0\}$ 上光滑。

**证毕。**

### (c) $\mathbb{R}P^n$ 的光滑结构

**定义：**

$\mathbb{R}P^n = S^n / (x \sim -x)$（对径点等同）。

商映射 $\pi: S^n \to \mathbb{R}P^n$。

**图册构造：**

对 $i = 0, ..., n$，定义：
$$U_i = \{[x_0:...:x_n] : x_i \neq 0\}$$

$\phi_i: U_i \to \mathbb{R}^n$，去掉第 $i$ 个坐标并归一化：
$$\phi_i([x_0:...:x_n]) = \left(\frac{x_0}{x_i}, ..., \widehat{\frac{x_i}{x_i}}, ..., \frac{x_n}{x_i}\right)$$

**过渡映射：**

$\phi_j \circ \phi_i^{-1}$ 是有理函数，在定义域上光滑。

**证毕。**

### (d) 光滑映射

**定义：** $f: M \to N$ 是**光滑的**，如果对任意坐标卡 $(U, \phi)$ 于 $M$，$(V, \psi)$ 于 $N$，
$$\psi \circ f \circ \phi^{-1}: \phi(U \cap f^{-1}(V)) \to \psi(V)$$
是光滑的（欧几里得空间之间的映射）。

**复合的光滑性：**

设 $f: M \to N$，$g: N \to P$ 光滑。

对 $M, P$ 的坐标卡，插入 $N$ 的中间坐标卡。

$\psi_P \circ g \circ f \circ \phi_M^{-1} = (\psi_P \circ g \circ \psi_N^{-1}) \circ (\psi_N \circ f \circ \phi_M^{-1})$

欧几里得空间光滑映射的复合光滑。

**证毕。**

## 证明技巧

1. **流形的验证：** Hausdorff + 第二可数 + 局部坐标
2. **图册的构造：** 球极投影是标准工具
3. **光滑性的验证：** 用坐标卡转换到欧几里得空间

## 常见错误

| 错误 | 说明 |
|------|------|
| 忘记Hausdorff条件 | 流形定义需要 |
| 忘记第二可数条件 | 排除病态例子 |
| 过渡映射计算错误 | 需仔细验证光滑性 |
