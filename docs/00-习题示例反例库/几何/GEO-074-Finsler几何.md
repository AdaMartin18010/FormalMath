# Finsler 几何

## 题目

**(a)** 定义 Finsler 度规并证明其基本性质。

**(b)** 讨论 Finsler 曲率和 Flag 曲率。

**(c)** 证明 Finsler 流形的体积形式。

## 解答

### (a) Finsler 度规

F: TM → ℝ，正齐次、光滑（除零截面）、正定型。

Riemann：F(x,v) = √g_x(v,v)。

Randers 度规：F = α + β。

### (b) Flag 曲率

曲面的 Gauss 曲率推广。

Flag (P, y)，P ⊂ T_xM 平面，y ∈ P。

### (c) 体积形式

Busemann-Hausdorff 体积、Holmes-Thompson 体积。

与 Riemann 体积的关系。
