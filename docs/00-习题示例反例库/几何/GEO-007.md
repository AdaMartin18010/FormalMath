# 外微分与de Rham上同调

## 题目

设 $M$ 是光滑流形，$\Omega^k(M)$ 是 $k$-形式空间。

**外微分** $d: \Omega^k(M) \to \Omega^{k+1}(M)$ 满足：
- $d(\alpha \wedge \beta) = d\alpha \wedge \beta + (-1)^k \alpha \wedge d\beta$
- $d^2 = 0$
- $df(X) = X(f)$ 对函数 $f$

(a) 证明 $d^2 = 0$。

(b) 证明Poincare引理：若 $\omega \in \Omega^k(\mathbb{R}^n)$ 闭（$d\omega = 0$），则恰当（$\omega = d\eta$）。

(c) 计算 $H_{dR}^*(S^1)$。

## 解答

### (a) $d^2 = 0$

**证明：**

对函数 $f$：$d^2f = d(df)$。

$df = \sum \frac{\partial f}{\partial x^i} dx^i$。

$d(df) = \sum_{i,j} \frac{\partial^2 f}{\partial x^j \partial x^i} dx^j \wedge dx^i$。

由Clairaut定理（混合偏导相等）和 $dx^j \wedge dx^i = -dx^i \wedge dx^j$：

$d(df) = \sum_{i < j} \left(\frac{\partial^2 f}{\partial x^j \partial x^i} - \frac{\partial^2 f}{\partial x^i \partial x^j}\right) dx^j \wedge dx^i = 0$。

对一般 $k$-形式，由Leibniz规则和函数情形，$d^2 = 0$。$\square$

### (b) Poincare引理

**证明：**

构造同伦算子 $H: \Omega^k(\mathbb{R}^n) \to \Omega^{k-1}(\mathbb{R}^n)$。

对 $\omega$，定义：
$$(H\omega)_x = \int_0^1 t^{k-1} i_{\vec{x}}(\omega_{tx}) dt$$

其中 $i_{\vec{x}}$ 是与位置向量场内缩。

**Cartan魔法公式**：$\mathcal{L}_X = di_X + i_Xd$。

计算 $dH + Hd$ 给出恒等（在同伦意义上）。

对闭形式 $\omega$（$d\omega = 0$）：
$$\omega = d(H\omega) + H(d\omega) = d(H\omega)$$

故 $\omega$ 恰当。$\square$

### (c) $S^1$ 的 de Rham 上同调

**解答：**

$\Omega^0(S^1) = C^\infty(S^1)$，$\Omega^1(S^1) = \{f(\theta)d\theta\}$，$\Omega^k = 0$（$k \geq 2$）。

**$H^0$**：$\ker(d: \Omega^0 \to \Omega^1) = \{\text{常值函数}\} \cong \mathbb{R}$。

**$H^1$**：$\Omega^1$ 中闭形式 = 所有形式（因 $d: \Omega^1 \to 0$）。

恰当形式：$df = f'(\theta)d\theta$，其中 $\int_0^{2\pi} f'(\theta)d\theta = 0$。

故恰当形式是积分零的1-形式。

$$H^1(S^1) = \frac{\Omega^1}{\text{im}(d)} \cong \mathbb{R}$$

生成元 $d\theta$ 的类。

**更高维**：$H^k = 0$（$k \geq 2$）。$\square$

## 证明技巧

1. **反对称性+Schwarz**：$d^2=0$ 的来源
2. **同伦算子**：Poincare引理的构造性证明
3. **积分判据**：1-形式的恰当性检验

## 常见错误

- ❌ 忘记验证 $H$ 的光滑性
- ❌ 在(c)中混淆闭和恰当
- ❌ 忽视 $S^1$ 的紧性对积分的影响
