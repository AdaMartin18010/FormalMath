# 测地线与变分

## 题目

**(a)** 定义测地线：$\nabla_{\dot{\gamma}}\dot{\gamma} = 0$。证明这是能量泛函 $E(\gamma) = \int_0^1 |\dot{\gamma}|^2 dt$ 的临界点。

**(b)** 陈述Jacobi场方程：沿测地线 $\gamma$ 的变分向量场 $J$ 满足
$$\frac{D^2 J}{dt^2} + R(J, \dot{\gamma})\dot{\gamma} = 0$$

**(c)** 证明Bonnet-Myers定理：若Ricci曲率 $\geq (n-1)k > 0$，则直径 $\leq \pi/\sqrt{k}$，基本群有限。

## 解答

### (a) 测地线变分

**证明：**

对变分 $\gamma_s$，$\gamma_0 = \gamma$，计算 $\frac{d}{ds}E(\gamma_s)|_{s=0}$。

$$\frac{dE}{ds} = 2\int \langle \frac{D}{dt}\frac{\partial\gamma}{\partial s}, \dot{\gamma} \rangle dt$$

分部积分：
$$= -2\int \langle \frac{\partial\gamma}{\partial s}, \frac{D\dot{\gamma}}{dt} \rangle dt$$

临界点条件：$\frac{D\dot{\gamma}}{dt} = 0$。

### (b) Jacobi场

**方程：**

二阶变分导出Jacobi方程。

共轭点：非平凡Jacobi场消失处。

### (c) Bonnet-Myers

**证明概要：**

设 $\gamma$ 是最短测地线，长度 $L > \pi/\sqrt{k}$。

取平行标架，构造变分场 $f(t) = \sin(\pi t/L)e_i(t)$。

第二变分公式：
$$\delta^2 E = \int_0^L \sum_{i=1}^{n-1} [|\dot{f}|^2 - \langle R(f e_i, \dot{\gamma})\dot{\gamma}, f e_i\rangle] dt$$

计算得 $\delta^2 E < 0$，矛盾。

故直径有限，万有覆叠紧，基本群有限。
