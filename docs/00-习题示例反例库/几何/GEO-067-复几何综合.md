# 复几何综合

## 题目

**(a)** 综合 Kähler、Calabi-Yau 和代数几何的方法。

**(b)** 讨论非 Kähler 复流形的几何。

**(c)** 证明 Hodge 猜想的几何意义。

## 解答

### (a) 综合方法

Yau 定理：PDE + 复几何 + 代数几何。

镜像对称：SYZ 猜想（特殊 Lagrange 纤维）。

### (b) 非 Kähler 复流形

Hopf 曲面：S³ × S¹，非 Kähler。

Kodaira 曲面：复结构、非代数。

### (c) Hodge 猜想

H^{p,p}(X) ∩ H^{2p}(X, ℚ) 由代数闭链生成。

几何意义：拓扑定义的子空间有代数几何实现。
