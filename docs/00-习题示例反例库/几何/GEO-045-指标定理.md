# Atiyah-Singer指标定理

## 题目

**(a)** 陈述Atiyah-Singer指标定理：对紧流形 $M$ 上的椭圆微分算子 $D$，
$$\text{ind}(D) = \dim \ker D - \dim \ker D^* = \int_M \hat{A}(TM) \wedge \text{ch}(E)$$

**(b)** 用热核方法证明：对Dirac算子，$\text{ind}(D) = \text{Tr}(e^{-tD^*D}) - \text{Tr}(e^{-tDD^*})$。

**(c)** 计算 $S^2$ 上Dirac算子的指标。

## 解答

### (a) 指标定理陈述

**说明：**

$\hat{A}$-genus：$\hat{A}(M) = \det^{1/2}\left(\frac{R/2}{\sinh(R/2)}\right)$。

Chern特征：$\text{ch}(E) = \text{tr}(e^{F/2\pi i})$。

特例：Gauss-Bonnet、Riemann-Roch、符号差定理。

### (b) 热核证明

**证明概要：**

$e^{-t\Delta}$ 的核 $K_t(x,y)$，$\Delta = D^*D$ 或 $DD^*$。

$$\text{Tr}(e^{-tD^*D}) - \text{Tr}(e^{-tDD^*}) = \int_M (K_t^{D^*D} - K_t^{DD^*})(x,x) dx$$

McKean-Singer公式：该差与 $t$ 无关。

$t \to \infty$：$e^{-t\Delta} \to P_{\ker}$，投影到核。

$t \to 0$：渐近展开给出局部指标密度，积分得拓扑指标。

### (c) $S^2$ 上Dirac算子

**计算：**

$S^2$ 旋量丛分解为 $\pm$ 本征丛。

Dirac算子与Cauchy-Riemann算子关联。

指标 = 全纯截面空间维数 = 1（常数）。

由Riemann-Roch：$\text{ind} = 1 - g = 1$。
