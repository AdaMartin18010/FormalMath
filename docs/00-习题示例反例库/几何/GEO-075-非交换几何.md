# 非交换几何

## 题目

**(a)** 定义谱三元组并证明其几何解释。

**(b)** 讨论 Connes 的重建定理。

**(c)** 证明距离公式的推广。

## 解答

### (a) 谱三元组

(A, H, D)：代数 A 在 Hilbert 空间 H 上表示，Dirac 算子 D。

交换情形：A = C^∞(M)，D = /∂D。

### (b) 重建定理

Connes：满足公理的谱三元组对应紧自旋流形。

公理：维数、正则性、定向性、实结构、一阶条件。

### (c) 距离公式

非交换距离：
$$d(φ, ψ) = \sup\{|φ(a) - ψ(a)| : \|[D,a]\| ≤ 1\}$$

交换情形恢复测地距离。
