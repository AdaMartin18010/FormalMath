# 正质量定理

## 题目

**(a)** 陈述并证明 Schoen-Yau 的正质量定理。

**(b)** 讨论 Witten 的旋量证明。

**(c)** 证明 Penrose 不等式（概述）。

## 解答

### (a) 正质量定理

渐近平坦流形，ADM 质量 m_ADM ≥ 0，等号当且仅当欧氏空间。

Schoen-Yau：极小曲面论证，维数归纳。

### (b) Witten 证明

Dirac 算子，Lichnerowicz 公式。

Σ⁺ 和 Σ⁻ 之间的联络，证明存在调和旋量。

### (c) Penrose 不等式

m_ADM ≥ √(A/16π)，A 是最外表面的面积。

Huisken-Ilmanen：逆平均曲率流。
