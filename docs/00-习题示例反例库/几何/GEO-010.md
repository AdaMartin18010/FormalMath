# de Rham定理与Hodge理论

## 题目

**de Rham定理**：对光滑流形 $M$，有同构：
$$H_{dR}^k(M) \cong H^k(M; \mathbb{R})$$

**Hodge定理**：设 $M$ 紧可定向Riemann流形，则：
$$\Omega^k(M) = \mathcal{H}^k(M) \oplus \text{im}(d) \oplus \text{im}(d^*)$$

其中 $\mathcal{H}^k = \ker(\Delta)$ 是调和形式。

(a) 证明 de Rham 映射 $I: H_{dR}^k(M) \to H^k(M; \mathbb{R})$，$[\omega] \mapsto \left(\sigma \mapsto \int_\sigma \omega\right)$ 是良定义的。

(b) 证明 $\Delta = dd^* + d^*d$ 的核恰是既闭又余闭的形式。

(c) 用Hodge定理证明 $H_{dR}^k(M) \cong \mathcal{H}^k(M)$。

## 解答

### (a) de Rham映射

**证明：**

对奇异 $k$-单形 $\sigma: \Delta^k \to M$，定义：
$$I(\omega)(\sigma) = \int_{\Delta^k} \sigma^*\omega$$

**良定义性**：

若 $\omega = d\eta$，由Stokes：
$$\int_\sigma d\eta = \int_{\partial\sigma} \eta$$

在链复形中，$\partial\sigma$ 是边缘，故 $I(d\eta)$ 是 $C_{k-1}$ 上的上边缘。

因此 $I$ 将恰当形式映到上边缘，诱导上同调映射。

**同构证明概要**：用Mayer-Vietoris和5-引理，从 $\mathbb{R}^n$ 开始归纳。$\square$

### (b) 调和形式

**证明：**

$\Delta\omega = 0$ 意味着：
$$0 = \langle \Delta\omega, \omega \rangle = \langle dd^*\omega + d^*d\omega, \omega \rangle$$

$$= \|d^*\omega\|^2 + \|d\omega\|^2$$

故 $d\omega = 0$ 且 $d^*\omega = 0$。

反之，若 $d\omega = d^*\omega = 0$，则 $\Delta\omega = 0$。$\square$

### (c) Hodge分解

**证明：**

**分解的正交性**：

对 $\alpha \in \mathcal{H}^k$，$d\beta$，$d^*\gamma$：
$$(\alpha, d\beta) = (d^*\alpha, \beta) = 0$$
$$(\alpha, d^*\gamma) = (d\alpha, \gamma) = 0$$
$$(d\beta, d^*\gamma) = (d^2\beta, \gamma) = 0$$

故三者正交。

**同调同构**：

由分解，任意闭形式 $\omega$（$d\omega = 0$）：
$$\omega = h + d\alpha + d^*\beta$$

由 $d\omega = 0$：$dd^*\beta = 0$，即 $(d^*\beta, d^*\beta) = (\beta, dd^*\beta) = 0$。

故 $d^*\beta = 0$，$\omega = h + d\alpha$。

$[\omega] = [h]$，且 $h$ 唯一（因若 $h = d\alpha$，则 $h = 0$）。

故 $\mathcal{H}^k \cong H_{dR}^k$。$\square$

2. **自伴算子**：Hodge理论的核心
