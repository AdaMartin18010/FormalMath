## 题目

设 $M$ 是光滑流形。

**(a)** 定义**微分形式**：$\omega \in \Omega^k(M)$ 是光滑的交错多重线性映射。给出局部坐标表示。

**(b)** 定义**外微分** $d: \Omega^k(M) \to \Omega^{k+1}(M)$，证明 $d^2 = 0$。

**(c)** 证明Poincaré引理：在 $\mathbb{R}^n$ 上，闭形式是恰当的（$k > 0$）。

**(d)** 定义**de Rham上同调** $H_{dR}^k(M) = \ker(d_k)/\text{im}(d_{k-1})$，计算 $H_{dR}^k(S^1)$。

## 详细解答

### (a) 微分形式的局部表示

**定义：**

$\omega \in \Omega^k(M)$：对每点 $p$，$\omega_p: (T_p M)^k \to \mathbb{R}$ 是交错的 $k$-线性。

光滑性：对向量场 $X_1, ..., X_k$，$\omega(X_1, ..., X_k) \in C^\infty(M)$。

**局部坐标：**

设 $(U, x^1, ..., x^n)$ 是坐标卡。

$\{dx^i|_p\}$ 是 $\{\frac{\partial}{\partial x^i}|_p\}$ 的对偶基。

$$\omega|_U = \sum_{i_1 < ... < i_k} \omega_{i_1...i_k} dx^{i_1} \wedge ... \wedge dx^{i_k}$$

其中 $\omega_{i_1...i_k} = \omega(\frac{\partial}{\partial x^{i_1}}, ..., \frac{\partial}{\partial x^{i_k}}) \in C^\infty(U)$。

### (b) 外微分与 $d^2 = 0$

**定义：**

对 $\omega = \sum f_I dx^I$，定义：
$$d\omega = \sum df_I \wedge dx^I = \sum_{i,I} \frac{\partial f_I}{\partial x^i} dx^i \wedge dx^I$$

**验证 $d^2 = 0$：**

对 $f \in \Omega^0(M)$：
$$d(df) = d\left(\sum_j \frac{\partial f}{\partial x^j} dx^j\right) = \sum_{i,j} \frac{\partial^2 f}{\partial x^i \partial x^j} dx^i \wedge dx^j$$

对称部分抵消（二阶偏导交换），反对称部分 $dx^i \wedge dx^j = -dx^j \wedge dx^i$，故 $d(df) = 0$。

一般形式类似。

**证毕。**

### (c) Poincaré引理

**定理：** 在 $\mathbb{R}^n$ 上，若 $d\omega = 0$（$\omega \in \Omega^k$，$k > 0$），则存在 $\eta$ 使 $\omega = d\eta$。

**证明概要：**

**同伦算子：**

定义 $H: \Omega^k(\mathbb{R}^n) \to \Omega^{k-1}(\mathbb{R}^n)$。

对 $k$-形式，沿径向积分。

**同伦公式：**

设 $i_0: \{0\} \to \mathbb{R}^n$，$i_1 = \text{id}$。

$H d + d H = i_1^* - i_0^* = \text{id} - 0$（在 $k > 0$ 上）。

若 $d\omega = 0$，则 $d(H\omega) = \omega$。

**证毕。**

### (d) de Rham上同调

**定义：**

$$H_{dR}^k(M) = \frac{\ker(d: \Omega^k \to \Omega^{k+1})}{\text{im}(d: \Omega^{k-1} \to \Omega^k)}$$

闭形式 / 恰当形式

**计算 $H_{dR}^k(S^1)$：**

**$k = 0$：**

$\ker(d_0) = \{f : df = 0\} = \{\text{常数函数}\} \cong \mathbb{R}$

$\text{im}(d_{-1}) = \{0\}$

$H_{dR}^0(S^1) = \mathbb{R}$

**$k = 1$：**

$\Omega^1(S^1) = \{f(\theta) d\theta\}$（$\theta$ 模 $2\pi$）

$\ker(d_1) = \Omega^1$（因无2-形式）

$\text{im}(d_0) = \{df = f'(\theta) d\theta\}$：恰当形式是周期积分为零的形式。

$$
\int_0^{2\pi} f'(\theta) d\theta = 0$$

闭形式 $g(\theta) d\theta$ 恰当当且仅当 $\int_0^{2\pi} g(\theta) d\theta = 0$。

$H_{dR}^1(S^1) \cong \mathbb{R}$，由 $\omega \mapsto \int_{S^1} \omega$ 给出同构。

**$k \geq 2$：**

$H_{dR}^k(S^1) = 0$（维数原因）。

**证毕。**

## 证明技巧

1. **外微分的计算：** 用坐标，注意反对称性
2. **Poincaré引理：** 同伦算子构造原像
3. **上同调计算：** 找闭形式，判断恰当性

## 常见错误

| 错误 | 说明 |
|------|------|
| 忘记反对称化 | 微分形式是交错的 |
| $d^2=0$的计算错误 | 注意混合偏导交换 |
| de Rham上同调与系数混淆 | 默认实系数 |

## 变式练习

**变式1：** 计算$H_{dR}^k(S^n)$：$k=0,n$时为$\mathbb{R}$，否则为0。

**变式2：** 证明Stokes定理：$\int_M d\omega = \int_{\partial M} \omega$。

**变式3：** 证明de Rham定理：$H_{dR}^k(M) \cong H^k(M; \mathbb{R})$（奇异上同调）。
