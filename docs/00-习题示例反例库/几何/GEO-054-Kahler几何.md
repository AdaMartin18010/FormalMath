# Kähler 几何基础

## 题目

**(a)** 证明 Kähler 度规的等价刻画。

**(b)** 计算 P^n 的 Fubini-Study 度规的曲率。

**(c)** 证明 Kähler 流形的 Hodge 分解。

## 解答

### (a) Kähler 等价条件

1. dω = 0（Kähler 条件）
2. ∇J = 0（复结构平行）
3. 局部存在 Kähler 势 ω = i∂∂̄φ

### (b) Fubini-Study 度规

$$\omega_{FS} = i∂∂̄ \log(1 + |z|^2)$$

曲率：常全纯截面曲率 1。

### (c) Hodge 分解

紧 Kähler 流形：
$$H^k(X, ℂ) = \bigoplus_{p+q=k} H^{p,q}(X)$$

其中 H^{p,q} = H^q(X, Ω^p)，且 H^{p,q} = \overline{H^{q,p}}。
