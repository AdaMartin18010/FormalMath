# Calabi-Yau 流形

## 题目

**(a)** 证明 Calabi-Yau 定理：Ricci 平坦 Kähler 度规的存在性。

**(b)** 讨论 Calabi-Yau 流形的拓扑性质。

**(c)** 计算 quintic 三维簇的 Hodge 数。

## 解答

### (a) Calabi-Yau 定理

Yau（1978）：紧 Kähler 流形，c₁ = 0 ⟹ 存在唯一 Ricci 平坦 Kähler 度规。

证明：复 Monge-Ampère 方程，连续性方法。

### (b) 拓扑性质

K_X 平凡 ⟹ h^{n,0} = 1。

holonomy ⊂ SU(n)，若不可约则 holonomy = SU(n)。

Euler 示性数 = 2(h^{1,1} - h^{2,1})。

### (c) Quintic 三维簇

$X = \{z_0^5 + ··· + z_4^5 = 0\} \subset \mathbb{P}^4$。

Hodge 菱形：h^{1,1} = 1，h^{2,1} = 101，χ = -200。
