# 辛约化进阶

## 题目

**(a)** 证明Kirwan映射 $\kappa: H^*_G(M) \to H^*(M // G)$ 是满射。

**(b)** 讨论GIT商与辛商的等价性。

**(c)** 计算 $T^*G // G$（余切丛约化）。

## 解答

### (a) Kirwan满射

等变上同调到商的上同调。

Atiyah-Bott引理和Morse理论。

### (b) Kempf-Ness定理

GIT商 $M // G$ 与辛商 $M /\!/ G$ 的等价。

### (c) 余切丛约化

$T^*G // G \cong T^*(G/G) = \{pt\}$ 但需细致分析。
