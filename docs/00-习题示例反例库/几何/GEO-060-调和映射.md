# 调和映射

## 题目

**(a)** 证明调和映射的热流存在性。

**(b)** 讨论 Eells-Sampson 定理。

**(c)** 证明 Siu 的强刚性定理。

## 解答

### (a) 调和映射热流

$$\frac{\partial u}{\partial t} = \tau(u)$$

τ(u) = tension field。短时间存在（Hamilton）。

### (b) Eells-Sampson

目标流形具有非正截面曲率 ⟹ 热流收敛到调和映射。

任意映射同伦于调和映射。

### (c) Siu 刚性

紧 Kähler 流形间的调和映射，若某点全纯双截面曲率正且 rank ≥ 4，则全纯或反全纯。

应用于 Mostow 刚性。
