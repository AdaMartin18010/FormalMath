# 伪黎曼几何

## 题目

**(a)** 证明 Lorentz 流形上测地线的因果结构。

**(b)** 讨论时空的奇点定理（Hawking-Penrose）。

**(c)** 证明引力能量的正性。

## 解答

### (a) 因果结构

时序、因果、零曲线。

未来/过去光锥、时序未来 I⁺(p)。

### (b) 奇点定理

能量条件 + 因果条件 ⟹ 测地不完备。

Hawking：大爆炸奇点。

Penrose：黑洞奇点。

### (c) 引力能量

Landau-Lifshitz 赝张量，ADM 能量。

Schoen-Yau：ADM 能量 ≥ 0。
