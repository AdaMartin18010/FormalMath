# 极值度量

## 题目

**(a)** 定义极值 Kähler 度规并证明其变分刻画。

**(b)** 讨论 K-稳定性和极值度量的存在性。

**(c)** 证明 Donaldson-Uhlenbeck-Yau 定理。

## 解答

### (a) 极值度量

标量曲率是 Killing 势：∇^{1,0}s_ω = K_ω^#。

Calabi 泛函的临界点。

### (b) K-稳定性

Tian、Donaldson：存在极值度量 ⟹ K-稳定。

Chen-Donaldson-Sun：K-稳定 ⟹ 存在 Kähler-Einstein（c₁ < 0）。

### (c) DUY 定理

全纯丛 E 稳定 ⟺ 存在 Hermitian-Einstein 度规。

证明：Uhlenbeck-Yau（连续性方法）、Donaldson（代数方法）。
