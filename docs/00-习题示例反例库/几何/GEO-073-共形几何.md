# 共形几何

## 题目

**(a)** 证明 Fefferman-Graham 渐近展开。

**(b)** 讨论共形不变量（Q-曲率、Paneitz 算子）。

**(c)** 证明共形边界的结构及其在 AdS/CFT 中的应用。

## 解答

### (a) Fefferman-Graham 渐近展开

**Poincaré-Einstein 度规**：对带共形边界 $(M, [g])$ 的流形，内部度规：

$$g_+ = \frac{dr^2 + g_r}{r^2}$$

其中 $r > 0$ 是边界定义坐标，$g_0 = \hat{g}$ 是边界上的代表度规。

**Einstein 条件**：$\operatorname{Ric}(g_+) = -n g_+$（$n+1$ 维）。

**Fefferman-Graham 展开**：$g_r$ 有形式展开：

$$g_r = g^{(0)} + r^2 g^{(2)} + r^4 g^{(4)} + \cdots + r^n g^{(n)} + r^n \log r \cdot h + \cdots$$

- $g^{(0)} = \hat{g}$
- $g^{(2)} = -P$，$P = \frac{1}{n-2}(\operatorname{Ric} - \frac{R}{2(n-1)}g)$ 是 Schouten 张量
- 对偶数维，$\log r$ 项出现（共形障碍）
- 对奇数维，无对数项，展开到任意阶

*证明*：将 Einstein 方程写成 $r$ 的递推方程，逐阶求解。

---

### (b) 共形不变量

**Q-曲率**（$n = 4$ 维）：

$$Q = \frac{1}{6}(-\Delta R + R^2 - 3|\operatorname{Ric}|^2)$$

**共形变换**：$\tilde{g} = e^{2w}g$，

$$\tilde{Q} = e^{-4w}(Q + Pw)$$

其中 $P$ 是 **Paneitz 算子**：

$$P = \Delta^2 + \delta\left(\frac{2}{3}R g - 2\operatorname{Ric}\right)dw$$

**性质**：

1. $P$ 是自伴共形协变算子，$P_{\tilde{g}} = e^{-(n+4)w/2} P_g e^{(n-4)w/2}$（$n=4$）
2. $\int Q \, dV$ 是共形不变量（共形类上的泛函临界点）

**总 Q-曲率**：

$$\int_M Q \, dV = 8\pi^2 \chi(M) - \int_M |W|^2 \, dV$$

其中 $W$ 是 Weyl 张量。

---

### (c) 共形边界与 AdS/CFT

**渐近双曲流形**：$(X^{n+1}, g_+)$，$\partial X = M$，$g_+$ 在 $M$ 附近有 Poincaré 型奇点。

**共形无穷远**：$M$ 上的共形类 $[g]$ 由 $g_+$ 决定（不依赖代表元）。

**AdS/CFT 对应**：

- **体（Bulk）**：$(n+1)$-维渐近 Anti-de Sitter 引力理论
- **边界（Boundary）**：$n$-维共形场论（CFT）

**Graham-Witten 的重整化体积**：

$$\operatorname{Vol}_R(X) = \operatorname{FP}_{\epsilon=0} \int_{r > \epsilon} dV_{g_+}$$

与边界 CFT 的异常（Weyl anomaly）相关。

**全息原理**：体中的几何信息编码在边界的共形场论中。

## 常见错误

- **Fefferman-Graham 的偶数维障碍**：$n$ 偶时 $g^{(n)}$ 非唯一确定，对数项是共形不变量。
- **Q-曲率的维数依赖**：$Q$ 的定义随维数变化，$n=4$ 是最常见情形。
- **AdS/CFT 的重整化**：体积发散，需引入 cutoff 和重整化。
