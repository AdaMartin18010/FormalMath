# 共形几何

## 题目

**(a)** 证明 Fefferman-Graham 渐近展开。

**(b)** 讨论共形不变量（Q-曲率、 Paneitz 算子）。

**(c)** 证明共形边界的结构。

## 解答

### (a) Fefferman-Graham

共形无穷远：Poincaré-Einstein 度规

$$g = \frac{1}{r^2}(dr^2 + g_r), \quad g_0 = \hat{g}$$

渐近展开：g_r = g⁽⁰⁾ + r²g⁽²⁾ + ··· + rⁿg⁽ⁿ⁾ + ···

### (b) 共形不变量

Q-曲率：n=4 时，
$$Q = \frac{1}{6}(-ΔR + R² - 3|Ric|²)$$

共形变换：e^{2w}g 的 Q̃ = e^{-4w}(Q + Pw)。

### (c) 共形边界

渐近双曲流形的共形无穷远。

AdS/CFT 对应的几何基础。
