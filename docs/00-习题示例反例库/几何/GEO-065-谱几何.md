# 谱几何

## 题目

**(a)** 证明 Laplacian 特征值的 Weyl 渐近公式。

**(b)** 讨论等谱问题（Can you hear the shape of a drum?）。

**(c)** 证明热核展开和 Minakshisundaram-Pleijel 渐近。

## 解答

### (a) Weyl 渐近

N(λ) = #{特征值 ≤ λ} ~ (2π)^{-n} ω_n vol(M) λ^{n/2}。

体积由谱决定。

### (b) 等谱问题

Milnor：16 维环面等谱但不等距。

Gordon-Webb-Wolpert：平面区域等谱不等距。

### (c) 热核展开

$$\sum_i e^{-\lambda_i t} \sim (4\pi t)^{-n/2} \sum_{k=0}^\infty a_k t^k$$

a₀ = vol(M)，a₁ = (1/6)∫ R。
