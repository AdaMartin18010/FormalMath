# 可视化

本目录为项目的辅助性可视化资源集合，包含TikZ图形、几何变换、数据分析、概念图及概率统计可视化内容，通过图形化手段降低数学概念的认知门槛。

## 内容导航

- **TikZ图形** — 精确绘制的数学图形
- **几何变换** — 变换群与对称性可视化
- **数据分析** — 统计与数据科学图表
- **概念图** — 知识节点关联图示
- **概率统计** — 概率分布与统计推断可视化
