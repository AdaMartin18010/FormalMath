📊 泊松分布 vs 二项分布

当n很大、p很小时，二项分布B(n,p)可以用泊松分布Pois(λ=np)近似。

当前λ对应的近似：

n=30, p=0.1 或 n=60, p=0.05 等

🔗 可加性

若 X ~ Pois(λ₁)，Y ~ Pois(λ₂)，且X、Y独立，则

X + Y ~ Pois(λ₁ + λ₂)

这是泊松过程的重要性质。

🎯 泊松分布的核心性质

  • 均值等于方差：E[X] = Var(X) = λ（散度指标）
  • 矩母函数：M(t) = exp(λ(e^t - 1))
  • 特征函数：φ(t) = exp(λ(e^(it) - 1))
  • 递推关系：P(X=k+1) = (λ/(k+1)) · P(X=k)
  • 稀有事件定律：描述单位时间或单位空间内随机事件发生次数
  • 无记忆性：泊松过程中，等待时间的剩余分布与已等待时间无关
