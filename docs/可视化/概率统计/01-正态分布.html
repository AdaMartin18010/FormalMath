📚 正态分布的核心性质

  • 对称性：关于均值μ对称，偏度为0
  • 单峰性：在x = μ处取得最大值 $\frac{1}{\sigma\sqrt{2\pi}}$
  • 渐近性：当x → ±∞时，f(x) → 0
  • 经验法则：68-95-99.7规则描述了数据落在不同标准差范围内的概率
  • 线性变换：若X ~ N(μ, σ²)，则aX + b ~ N(aμ + b, a²σ²)
  • 可加性：独立正态变量之和仍服从正态分布
  • 中心极限定理：大量独立随机变量之和趋近于正态分布

🔧 交互说明

  • 拖动均值滑块：观察分布曲线的左右平移
  • 拖动标准差滑块：观察分布的"胖瘦"变化（σ越大，曲线越平坦）
  • 选择填充范围：可视化不同区间内的概率面积
  • 鼠标悬停在曲线上可查看具体坐标值
