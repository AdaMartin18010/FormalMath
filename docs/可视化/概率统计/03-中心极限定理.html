📐 中心极限定理 (Central Limit Theorem)

设 $X_1, X_2, \ldots, X_n$ 是独立同分布的随机变量，$E[X_i] = \mu$，$Var(X_i) = \sigma^2 < \infty$。 令 $S_n = \sum_{i=1}^n X_i$，则当 $n \to \infty$ 时：

$$\frac{S_n - n\mu}{\sigma\sqrt{n}} \xrightarrow{d} N(0,1)$$

🔍 CLT的关键洞察

  • 分布无关性：无论原始分布是什么形状（只要方差有限），标准化和的极限分布都是正态分布
  • 收敛速度：收敛速度取决于原始分布的性质。对称、单峰的分布收敛较快
  • Berry-Esseen定理：给出了收敛速度的定量估计，与三阶矩有关
  • 实际意义：解释了为什么正态分布在自然界中如此普遍

📚 历史与应用

  • 历史：由棣莫弗(1733)首次证明二项分布情形，拉普拉斯推广，李雅普诺夫(1901)给出一般证明
  • 统计推断：为大样本统计检验和置信区间提供理论基础
  • 随机模拟：蒙特卡洛方法的理论基础
  • 误差分析：测量误差的正态分布假设的理论依据
  • 金融工程：资产收益率建模、风险度量
