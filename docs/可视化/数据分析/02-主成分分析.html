PCA核心思想

max Var(wᵀx) s.t. ||w|| = 1
  • 寻找数据方差最大的方向
  • 主成分是协方差矩阵的特征向量
  • 特征值表示各主成分解释的方差

计算步骤

  • 1. 标准化：零均值，单位方差
  • 2. 协方差矩阵：Σ = XᵀX/(n-1)
  • 3. 特征分解：Σw = λw
  • 4. 投影：Z = XW

应用场景

  • 高维数据可视化
  • 噪声过滤
  • 特征提取
  • 图像压缩
  • 人脸识别(Eigenfaces)
