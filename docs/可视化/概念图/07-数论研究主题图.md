## 关键概念说明

### 数论主要分支
| 分支 | 主要工具 | 核心问题 |
|-----|---------|---------|
| 初等数论 | 同余、素数理论 | 整除性、丢番图方程 |
| 代数数论 | 代数、伽罗瓦理论 | 数域结构、类域论 |
| 解析数论 | 复分析、傅里叶分析 | 素数分布、ζ函数 |
| 算术几何 | 代数几何 | 椭圆曲线、 motives |
| 超越数论 | 分析方法 | 数的代数独立性 |

### 千禧年难题相关
- **黎曼假设**: ζ函数非平凡零点实部均为1/2
- **BSD猜想**: 椭圆曲线秩与L函数零点阶数的关系

### 费马大定理的证明路径
```
谷山-志村猜想
    ↓
椭圆曲线是模的
    ↓
弗雷椭圆曲线不能是模的
    ↓
费马大定理
```

### 椭圆曲线的重要性
- **数论**: 莫德尔-韦伊定理、BSD猜想
- **密码学**: ECC提供更高安全性/密钥比
- **朗兰兹纲领**: 连接数论与表示论

## 与其他领域联系

- **代数几何**: 算术几何是现代数论的核心
- **表示论**: 自守表示、朗兰兹对应
- **物理学**: 弦论中的卡拉比-丘流形、镜像对称
- **密码学**: RSA、椭圆曲线密码的现代基础
