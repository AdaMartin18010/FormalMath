# 拓扑空间分类图 (Topological Space Classification)

## 概述

本概念图展示拓扑空间的层次结构和分类体系，从基础定义到各类拓扑性质，包括分离公理、紧致性、连通性以及拓扑不变量。

## 关键概念说明

### 分离公理层次
```
T₄ (正规) ⇒ T₃½ (吉洪诺夫) ⇒ T₃ (正则) 
  ⇒ T₂½ (乌雷松) ⇒ T₂ (豪斯多夫) 
  ⇒ T₁ (弗雷歇) ⇒ T₀ (柯尔莫戈洛夫)
```

### 紧致性关系
- 度量空间中：紧致 ⟺ 序列紧致 ⟺ 极限点紧致 ⟺ 完备且完全有界
- 一般拓扑空间：紧致 ⟹ 序列紧致 ⟹ 极限点紧致

### 连通性层次
- **道路连通 ⟹ 连通**（但逆不成立，如拓扑学家的正弦曲线）
- **局部道路连通 + 连通 ⟹ 道路连通**

### 可数性公理的重要性
- **第二可数 ⟹ 第一可数 ⟹ 序列确定闭包**
- **第二可数 + T₃ ⟹ 可度量化**（乌雷松度量化定理）

## 与其他领域联系

- **分析学**: 紧致性在极值定理中的应用
- **代数几何**: 扎里斯基拓扑（非豪斯多夫）
- **泛函分析**: 弱拓扑、弱*拓扑
- **动力系统**: 拓扑动力系统研究
