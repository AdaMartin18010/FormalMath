## 关键概念说明

### 表示论基本问题
```
给定群 G，研究所有可能的线性表示
    ↓
1. 分类所有不可约表示
2. 分解可约表示为不可约的直和
3. 计算表示之间的同态
4. 确定特征标表
```

### 舒尔引理
- 不可约表示之间的同态要么为0，要么为同构
- 不可约表示的自同态是标量乘法
- **推论**: 阿贝尔群的不可约表示都是1维的

### 半单李代数表示分类
```
不可约有限维表示
    ↓
对应于支配整权 λ ∈ P⁺
    ↓
最高权向量 v_λ
    ↓
由 lowering operators 生成整个表示
```

### 重要定理
| 定理 | 内容 | 意义 |
|-----|------|------|
| Maschke定理 | 有限群表示完全可约 | 表示分类的基础 |
| Peter-Weyl | 紧群的L²理论 | 调和分析基础 |
| Borel-Weil-Bott | 用层上同调实现表示 | 几何表示论基石 |
| Kazhdan-Lusztig | 特征标公式 | 深层组合数学 |
