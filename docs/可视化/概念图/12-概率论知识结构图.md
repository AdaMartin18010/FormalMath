## 关键概念说明

### 概率论公理化 (Kolmogorov, 1933)
概率测度 P 满足：
1. **非负性**: P(A) ≥ 0
2. **规范性**: P(Ω) = 1
3. **σ-可加性**: 对不交可列并，P(∪Aᵢ) = ΣP(Aᵢ)

### 收敛模式关系
```
几乎必然收敛
    ↓
依概率收敛
    ↓
依分布收敛

Lᵖ收敛 ⟹ 依概率收敛
```

### 核心极限定理
| 定理 | 条件 | 结论 |
|-----|------|------|
| 大数定律 | i.i.d., E|X|<∞ | X̄ₙ → μ |
| 中心极限定理 | i.i.d., Var(X)<∞ | √n(X̄ₙ-μ)/σ → N(0,1) |

### 随机过程分类
```
按时间: 离散时间 / 连续时间
按状态: 离散状态 / 连续状态
按性质: 马尔可夫 / 鞅 / 平稳 / 高斯
```
