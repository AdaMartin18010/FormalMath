## 关键概念说明

### 动力系统分类
```
按时间: 离散系统 (迭代) / 连续系统 (流)
按空间: 拓扑 / 光滑 / 保测
按结构: 可积 / 遍历 / 混沌
```

### 混沌的特征
1. **初值敏感依赖**: 邻近轨道指数分离
2. **拓扑传递**: 存在稠密轨道
3. **周期点稠密**: 规则行为处处逼近

### 双曲性层次
```
双曲不动点
    ↓
双曲集合
    ↓
一致双曲 (Anosov, Axiom A)
    ↓
非一致双曲 (Pesin理论)
    ↓
部分双曲
```

### KAM定理
在可积系统的微小扰动下，**大部分**不变环面保持。条件是频率满足**丢番图条件**（远离有理数）。
