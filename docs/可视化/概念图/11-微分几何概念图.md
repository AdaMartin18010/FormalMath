## 关键概念说明

### 微分几何核心结构
```
流形 (拓扑 + 光滑结构)
    ↓
切空间 / 余切空间 (局部线性化)
    ↓
张量场 / 微分形式 (几何对象)
    ↓
联络 (微分结构)
    ↓
曲率 (可积性障碍)
```

### 曲率层次
```
黎曼曲率张量 Rᵐᵢⱼₖ
    ↓
里奇曲率 Rᵢⱼ = Rᵏᵢₖⱼ
    ↓
标量曲率 R = gⁱʲRᵢⱼ
    ↓
截面曲率 K(Π) = ⟨R(e₁,e₂)e₂,e₁⟩
```

### 列维-奇维塔联络性质
唯一满足以下条件的联络：
1. **无挠**: T(X,Y) = ∇_XY - ∇_YX - [X,Y] = 0
2. **度量相容**: ∇g = 0

### 爱因斯坦方程
```
G_μν + Λg_μν = 8πG/c⁴ T_μν

其中 G_μν = R_μν - ½Rg_μν (爱因斯坦张量)
     T_μν (能量-动量张量)
     Λ (宇宙学常数)
```
