# 范畴论基本概念图 (Category Theory Fundamentals)

## 概述

本概念图展示范畴论的基本概念和核心结构，范畴论作为"数学的数学"，为现代数学提供了统一的语言和框架。

## 关键概念说明

### Yoneda引理
```
函子 Nat(Hom(A,-), F) ≅ F(A)

意义: 对象完全由其表示的函子决定
     "对象由其与其他对象的关系决定"
```

### 伴随函子基本对
| 左伴随 F | 右伴随 G | 范畴 |
|---------|---------|------|
| 自由群 | 遗忘函子 | Set → Grp |
| 离散拓扑 | 遗忘函子 | Set → Top |
| 左Kan扩张 | 限制 | 函子范畴 |
| 张量积 ⊗ | Hom | 模范畴 |
| 存在量词 ∃ | 弱化 | 逻辑 |

### 极限 vs 余极限
| 概念 | 类型 | 例子 |
|-----|------|------|
| 极限 | 投影到图表 | 积、等化子、拉回、核 |
| 余极限 | 从图表包含 | 余积、余等化子、推出、余核 |

### 范畴论的意义
1. **统一语言**: 跨领域的共同框架
2. **结构主义**: 关注关系而非元素
3. **对偶原理**: 任何概念有其对偶
4. **抽象层次**: 揭示深层模式
