# 格洛腾迪克理论体系图 (Grothendieck Theory System)

## 概述

本概念图展示亚历山大·格洛腾迪克的数学理论体系，这位20世纪最具影响力的数学家彻底改变了代数几何的面貌，创立了概形理论、层上同调等现代数学基础。

## 关键概念说明

### 格洛腾迪克数学哲学
```
几何学 = 关于“空间”的科学
    ↓
空间的本质在于其上定义的函数层
    ↓
相对观点: 研究 X → S 而非孤立的 X
    ↓
普遍性质定义对象，而非具体构造
```

### 概形 vs 古典代数簇
| 方面 | 古典代数簇 | 概形 |
|-----|-----------|------|
| 定义域 | 代数闭域 | 任意交换环 |
| 点 | 闭点 | 所有素理想 |
| 结构 | 拓扑空间 | 环化空间 |
| 几何 | 绝对几何 | 相对几何 |

### 核心贡献时间线
- **1955**: 核空间理论 (函数分析)
- **1957**: 格洛腾迪克群 (K理论先驱)
- **1958-1970**: 代数几何革命
- **1960-1967**: EGA出版
- **1961-1967**: SGA讨论班
- **1969**: 离开数学界

### 未完成愿景
1. **Motives理论**: 所有上同调理论的共同来源
2. **标准猜想**: 关于代数闭链的深刻猜想
3. **拓扑斯**: 几何与逻辑的统合

## 与其他数学领域联系

- **数论**: 韦伊猜想、算术几何基础
- **代数拓扑**: K理论、拓扑斯
- **逻辑**: 构造性数学、高阶类型论
- **数学物理**: 弦论、镜像对称的几何基础
