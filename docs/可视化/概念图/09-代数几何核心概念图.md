## 关键概念说明

### 代数几何发展里程碑
```
古典代数几何 (19世纪)
    ↓
扎里斯基拓扑、层论 (1940s-1950s)
    ↓
概形理论 格洛腾迪克 (1960s)
    ↓
算术几何、韦伊猜想证明 (1970s)
    ↓
导出范畴、镜像对称 (1990s-2000s)
```

### 核心对应关系
```
几何              代数
仿射簇 V        ⟷ 坐标环 k[V]
射影簇 X        ⟷ 分次坐标环 S(X)
态射 φ: V→W     ⟷ k-代数同态 φ*: k[W]→k[V]
点 p ∈ V        ⟷ 极大理想 𝔪_p ⊂ k[V]
仿射概形        ⟷ 任意交换环
```

### 上同调理论对比
| 上同调 | 定义 | 适用对象 |
|-------|------|---------|
| 层上同调 | 导出函子 | 任意概形 |
| 德拉姆 | 微分形式 | 光滑簇/ℂ |
| étale | 层范畴 | 任意概形 |
| 晶体 | 形变理论 | 特征p |

### 重要定理
- **希尔伯特零点定理**: 代数集与根理想对应
- **黎曼-罗赫定理**: 向量丛的欧拉示性数公式
- **韦伊猜想**: 有限域上代数簇的ζ函数性质（德利涅证明）
- **广中平佑奇点消解**: 特征零域上的奇点消解

## 与其他领域联系

- **数论**: 算术几何、椭圆曲线、模形式
- **物理学**: 弦论、镜像对称、规范场论
- **表示论**: 几何表示论（D-模、反常层）
- **组合数学**: 枚举几何、格罗布纳基
