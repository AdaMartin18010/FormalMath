# 几何学分支图 (Geometry Branches)

## 概述

本概念图展示几何学的完整分支体系，从经典欧几里得几何到现代微分几何、代数几何，以及它们之间的深刻联系。

## 关键概念说明

### 几何学发展时间线
```
欧几里得几何 (公元前300年)
    ↓
解析几何 (笛卡尔, 1637)
    ↓
微分几何 (高斯, 1827; 黎曼, 1854)
    ↓
黎曼几何 + 张量分析
    ↓
广义相对论 (爱因斯坦, 1915)
    ↓
代数几何现代化 (格洛腾迪克, 1960s)
```

### 空间曲率分类
| 几何类型 | 高斯曲率 | 模型 | 三角形内角和 |
|---------|---------|------|------------|
| 双曲几何 | K < 0 | 庞加莱圆盘 | < 180° |
| 欧几里得几何 | K = 0 | 平面 | = 180° |
| 椭圆几何 | K > 0 | 球面 | > 180° |

### 微分几何核心对象
- **黎曼度量**: 定义流形上的距离和角度
- **联络**: 定义向量场的微分
- **曲率**: 度量空间偏离平坦的程度

### 代数几何核心对应
```
代数:                    几何:
环 R        ⟷        仿射概形 Spec R
理想 I      ⟷        闭子概形 V(I)
模 M        ⟷        层 M̃
局部化 R_f  ⟷        主开集 D(f)
```

## 与其他领域联系

- **物理学**: 广义相对论(黎曼几何)、规范场论(纤维丛)、弦论(复几何)
- **数论**: 算术几何(费马大定理证明)
- **拓扑学**: 示性类连接几何与拓扑
- **表示论**: 几何表示论(D-模、反常层)
