# 逻辑体系结构图 (Logic System Architecture)

## 概述

本概念图展示数理逻辑的完整体系，从命题逻辑到一阶逻辑、模型论、证明论、递归论和集合论，以及它们在数学基础中的作用。

## 关键概念说明

### 逻辑层次结构
```
命题逻辑 (PL)
    ↓
一阶逻辑 (FOL) - 可判定性：半可判定
    ↓
高阶逻辑 / 模态逻辑
    ↓
类型论 / 集合论
```

### 哥德尔不完备定理
1. **第一不完备定理**: 任何足够强的一致形式系统都存在不可判定命题
2. **第二不完备定理**: 足够强的一致形式系统不能证明自身的一致性

### 逻辑主要分支
| 分支 | 研究对象 | 核心问题 |
|-----|---------|---------|
| 模型论 | 结构与真值 | 可满足性、范畴性 |
| 证明论 | 证明结构 | 一致性、证明复杂性 |
| 递归论 | 可计算性 | 算法、可判定性 |
| 集合论 | 数学基础 | 无穷、公理系统 |

### 数学基础流派
- **柏拉图主义**: 数学对象客观存在
- **形式主义**: 数学是符号游戏
- **直觉主义/构造主义**: 数学对象必须可构造
- **同伦类型论**: 新的基础框架

## 与其他领域联系

- **计算机科学**: 类型论、程序验证、自动推理
- **语言学**: 形式语义学、蒙太古语法
- **哲学**: 分析哲学、语言哲学
- **人工智能**: 知识表示、自动定理证明
