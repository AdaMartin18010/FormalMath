# 国际权威数学内容深度对齐

本节系统对齐 FormalMath 与全球顶尖大学数学课程及权威数学资源，为每个核心主题提供严格的数学定义、定理与证明。

## 1. 实分析

### 1.1 极限与连续性

**定义 1.1（极限）**. $\lim_{x \to a} f(x) = L$ 当且仅当对任意 $\epsilon > 0$，存在 $\delta > 0$ 使 $0 < |x-a| < \delta$ 蕴含 $|f(x) - L| < \epsilon$。

**定理 1.2（连续性保持）**. 连续函数的复合连续，连续函数在紧集上有界且达到最值。

### 1.2 微分与积分

**定理 1.3（微积分基本定理）**. 若 $F' = f$ 连续，则 $\int_a^b f(x)dx = F(b) - F(a)$。

## 2. 线性代数

### 2.1 向量空间

**定义 2.1（向量空间）**. 集合 $V$ 配备加法和数乘，满足八条公理（交换群、分配律、结合律等）。

**定理 2.2（维数定理）**. 有限维向量空间的任何两组基含同样多个元素。

### 2.2 特征值与对角化

**定理 2.3（谱定理）**. Hermitian 矩阵可酉对角化，特征值为实数。

## 3. 抽象代数

### 3.1 群论

**定理 3.1（Lagrange）**. 有限群 $G$ 的子群 $H$ 满足 $|H| \mid |G|$。

### 3.2 环论

**定理 3.2（Hilbert 基定理）**. $R$ Noether 蕴含 $R[x]$ Noether。

## 4. 拓扑学

### 4.1 基本群

**定义 4.1（基本群）**. $\pi_1(X, x_0)$ 为基于 $x_0$ 的环路同伦类群。

**定理 4.2（Seifert-van Kampen）**. $X = U \cup V$，$U, V, U \cap V$ 道路连通，则：

$$\pi_1(X) \cong \pi_1(U) *_{\pi_1(U \cap V)} \pi_1(V).$$

## 5. 例子

### 5.1 例子：基本群计算

$\pi_1(S^1) = \mathbb{Z}$，$\pi_1(S^n) = 0$（$n \geq 2$）。

### 5.2 例子：覆盖空间

$S^1$ 的万有覆盖为 $\mathbb{R} \to S^1$，$t \mapsto e^{2\pi i t}$。
