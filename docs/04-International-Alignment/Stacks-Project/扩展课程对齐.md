# Stacks Project 深度对齐：代数几何核心内容

Stacks Project 是当代最全面的代数几何参考文献，由 Aise Johan de Jong 发起并维护。本节从 Stacks Project 的核心框架出发，对齐 FormalMath 的代数几何内容，并提供严格的数学定义、定理证明与例子。

## 1. 概形理论基础

### 1.1 局部环层空间

**定义 1.1（局部环层空间）**. 环层空间 $(X, \mathcal{O}_X)$ 称为局部环层空间，若对所有 $x \in X$，茎 $\mathcal{O}_{X,x}$ 为局部环。

**定义 1.2（仿射概形）**. 对交换环 $A$，仿射概形 $\operatorname{Spec}(A)$ 为素理想集配备 Zariski 拓扑与结构层。

**定理 1.3（Stacks Project 01HR）**. 结构层 $\mathcal{O}_{\operatorname{Spec}(A)}$ 是层，且 $\mathcal{O}_{\operatorname{Spec}(A), \mathfrak{p}} = A_\mathfrak{p}$。

### 1.2 概形

**定义 1.4（概形）**. 概形是局部同构于仿射概形的局部环层空间。

**定理 1.5（Stacks Project 01ID）**. 态射 $(f, f^\sharp): (X, \mathcal{O}_X) \to (Y, \mathcal{O}_Y)$ 由连续映射 $f: X \to Y$ 和层映射 $f^\sharp: \mathcal{O}_Y \to f_*\mathcal{O}_X$ 组成，使得茎上的诱导映射为局部同态。

## 2. 拟凝聚层

### 2.1 定义与性质

**定义 2.1（拟凝聚层）**. $\mathcal{O}_X$-模层 $\mathcal{F}$ 称为拟凝聚的，若存在仿射开覆盖 $X = \bigcup U_i$，$U_i = \operatorname{Spec}(A_i)$，使得 $\mathcal{F}|_{U_i} \cong \widetilde{M_i}$ 对某 $A_i$-模 $M_i$。

**定理 2.2（Stacks Project 01LA）**. 对仿射概形 $X = \operatorname{Spec}(A)$，函子 $M \mapsto \widetilde{M}$ 给出 $A$-模范畴与 $X$ 上拟凝聚层范畴的等价。

*证明*. 构造逆函子：$\mathcal{F} \mapsto \Gamma(X, \mathcal{F})$。验证 $\widetilde{\Gamma(X, \mathcal{F})} \cong \mathcal{F}$：对 $D(f)$，$\widetilde{\Gamma(X, \mathcal{F})}(D(f)) = \Gamma(X, \mathcal{F})_f$，而由层的性质 $\mathcal{F}(D(f)) = \Gamma(X, \mathcal{F})_f$。$\square$

### 2.2 凝聚层

**定义 2.3（凝聚层）**. 设 $X$ 为 Noether 概形。拟凝聚层 $\mathcal{F}$ 称为凝聚的，若每个 $M_i$ 为有限生成模。

**定理 2.4（Stacks Project 01Y0）**. 在 Noether 概形上，凝聚层的核、余核与像仍是凝聚的。

## 3. 上同调理论

### 3.1 层上同调

**定理 3.1（Stacks Project 01E0）**. 设 $X$ 为环层空间。$\mathsf{Mod}(\mathcal{O}_X)$ 中对象有内射消解，故可定义右导出函子 $H^i(X, -) = R^i\Gamma(X, -)$。

**定理 3.2（Serre）**. 设 $X$ 为 Noether 概形。以下等价：
1. $X$ 仿射；
2. $H^q(X, \mathcal{F}) = 0$ 对所有 $q > 0$ 和拟凝聚层 $\mathcal{F}$。

### 3.2 Čech 上同调

**定理 3.3（Stacks Project 01EO）**. 对概形 $X$ 的仿射开覆盖 $\mathcal{U}$ 和拟凝聚层 $\mathcal{F}$，Čech 上同调与导出函子上同调一致：$\check{H}^q(\mathcal{U}, \mathcal{F}) = H^q(X, \mathcal{F})$。

## 4. 例子

### 4.1 例子：射影空间的层上同调

$X = \mathbb{P}^n_k$，$\mathcal{O}(m)$ 为 $m$ 次扭层。

**定理 4.1**. 
- $H^0(\mathbb{P}^n, \mathcal{O}(m)) = k[x_0, \dots, x_n]_m$（$m$ 次齐次多项式），维数 $\binom{n+m}{n}$（$m \geq 0$）；
- $H^q(\mathbb{P}^n, \mathcal{O}(m)) = 0$（$0 < q < n$）；
- $H^n(\mathbb{P}^n, \mathcal{O}(m)) = H^0(\mathbb{P}^n, \mathcal{O}(-m-n-1))^*$。

*证明*. 用标准仿射覆盖 $U_i = \{x_i \neq 0\} \cong \mathbb{A}^n$ 计算 Čech 上同调。$\square$

### 4.2 例子：曲线上的 Riemann-Roch

设 $X$ 为光滑射影曲线（亏格 $g$）。对除子 $D$：

$$\chi(\mathcal{O}_X(D)) = \deg D + 1 - g.$$

由 Serre 对偶，$H^1(X, \mathcal{O}(D)) \cong H^0(X, \mathcal{O}(K-D))^*$，故：

$$\ell(D) - \ell(K-D) = \deg D + 1 - g.$$
