# 术语标准化对照表

本文档记录FormalMath项目中代数术语符号的统一标准，确保项目内符号使用的一致性和专业性。

## 符号统一标准

| 术语 | 标准符号 | 说明 | 涉及分支 | 备注 |
|------|----------|------|----------|------|
| 同构 | ≅ | 代数结构同构（群、环、域、模等） | 代数、拓扑、分析 | 统一替代 ≃ |
| 同伦等价 | ≃ | 拓扑空间的同伦等价 | 代数拓扑、同伦论 | 保留原符号 |
| 映射同伦 | ≃ | 连续映射之间的同伦关系 | 代数拓扑 | 保留原符号 |
| 态射 | → | 范畴论中的映射/态射 | 范畴论 | 标准映射符号 |
| 自然变换 | ⇒ | 范畴论中的自然变换 | 范畴论 | 双横线箭头 |
| 张量积 | ⊗ | 模、向量空间的张量积 | 代数、几何 | 使用Unicode符号 |
| 子集 | ⊂ 或 ⊆ | 集合包含关系 | 全部分支 | 根据语境选择 |
| 真子集 | ⊊ | 真包含关系 | 集合论 | 严格包含 |
| 同调 | Hₙ | 第n阶同调群 | 代数拓扑、同调代数 | 下标表示阶数 |
| 上同调 | Hⁿ | 第n阶上同调群 | 代数拓扑、同调代数 | 上标表示阶数 |
| 群同构 | ≅* 或 ≃* | Lean4代码中的群同构 | 形式化证明 | Lean语法保留 |
| 环同构 | ≅+* 或 ≃+* | Lean4代码中的环同构 | 形式化证明 | Lean语法保留 |
| 模同构 | ≅ₗ 或 ≃ₗ | Lean4代码中的模同构 | 形式化证明 | Lean语法保留 |
| 等价类型 | ≃ | 同伦类型论中的等价 | 类型论 | HoTT标准符号 |
| 范畴等价 | ≅ 或 ≃ | 范畴之间的等价 | 范畴论 | 根据语境选择 |

## 替换规则详解

### 1. 同构符号统一

**统一前混用情况：**
- `≅` - 传统同构符号
- `≃` - 有时被用作同构

**统一后标准：**
- 所有代数结构同构统一使用 `≅`
- `≃` 仅保留用于同伦等价和类型论等价

**涉及替换示例：**
```markdown
# 替换前
G ⧸ f.ker ≃* f.range
R ≃ MatrixRing D n

# 替换后  
G ⧸ f.ker ≅* f.range
R ≅ MatrixRing D n
```

### 2. 张量积符号

**标准：**
- 数学公式中使用 `$\otimes$`（LaTeX语法）
- 纯文本中使用 `⊗`（Unicode符号）

**示例：**
```markdown
# LaTeX数学公式
$V \otimes W$

# 纯文本
M ⊗ N 表示模的张量积
```

### 3. 同调/上同调符号

**标准：**
- 同调群：`H_n(X)` 或 `Hₙ(X)` - 同调（homology）
- 上同调群：`H^n(X)` 或 `Hⁿ(X)` - 上同调（cohomology）

**记忆口诀：**
- "下homology，上cohomology"
- 下标为同调，上标为上同调

### 4. 态射符号区分

| 符号 | 含义 | 使用场景 |
|------|------|----------|
| → | 映射/态射 | f: X → Y |
| ⇒ | 自然变换 | α: F ⇒ G |
| ↦ | 元素映射 | x ↦ f(x) |

## 目录特定规范

### docs/02-代数结构

- 群同构：`G ≅ H`
- 环同构：`R ≅ S`
- 模同构：`M ≅ N`
- 张量积：`M ⊗_R N`

### docs/05-拓扑学

- 同伦等价：`X ≃ Y`（保留）
- 映射同伦：`f ≃ g`（保留）
- 基本群同构：`π₁(X) ≅ π₁(Y)`
- 同调群：`H_n(X)`
- 上同调群：`H^n(X)`

### docs/11-高级数学

- 范畴等价：`C ≅ D` 或 `C ≃ D`（根据语境）
- 类型等价（HoTT）：`A ≃ B`（保留）
- 张量积：`V ⊗ W`

## Lean4形式化代码

Lean4代码文件(`.lean`)中的符号遵循Lean语法规范：

| Lean符号 | 含义 | 说明 |
|----------|------|------|
| `≃*` | 群同构 | `MulEquiv` |
| `≃+*` | 环同构 | `RingEquiv` |
| `≃ₗ` | 模同构 | `LinearEquiv` |
| `≃` | 类型等价 | `Equiv` |

**注意：** Lean代码中的符号是语法的一部分，不应随意修改。
