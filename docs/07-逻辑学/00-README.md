**国际对标**: ACM/IEEE CS2023, AMS MSC2020

---

## 📋 目录概述

本目录涵盖数理逻辑与形式逻辑主要分支：命题逻辑、谓词逻辑、模态逻辑、直觉逻辑、多值逻辑。内容包括语法、语义、公理化、可证性、完全性及形式化实现。

## 🗺️ 学习路径建议

1. **入门**: 01-命题逻辑 → 02-谓词逻辑  
2. **经典逻辑深化**: 01-命题逻辑-增强版  
3. **非经典逻辑**: 03-模态逻辑 → 04-直觉逻辑 → 05-多值逻辑（可并行）  
4. **形式化**: 与 [09-形式化证明](../09-形式化证明/) 模块衔接
