## 📋 目录概述

逻辑学是研究推理有效性的数学分支，是现代数学、计算机科学与人工智能的理论基石。本目录系统涵盖数理逻辑的五大核心领域：**命题逻辑**研究命题间的真值关系与推理规则，是逻辑系统的入门与基础；**谓词逻辑**引入量词与谓词，构建一阶逻辑系统，支撑数学形式化表达；**模态逻辑**扩展经典逻辑，研究必然性与可能性等模态概念，在知识表示与程序验证中至关重要；**直觉逻辑**摒弃排中律，发展构造性证明方法，是类型论与函数式编程的理论基础；**多值逻辑**突破二值限制，探索模糊性与不确定性推理，广泛应用于人工智能与控制理论。从亚里士多德的三段论到哥德尔不完备定理，从布尔代数到现代类型论，逻辑学不仅是纯粹数学的核心，更是计算机科学、语言学、哲学与认知科学的交汇点。

## 🗺️ 学习路径

### 入门阶段（建议时长：3-4周）

**推荐学习顺序：**

1. **起点**：[01-命题逻辑](01-命题逻辑.md) — 掌握命题联结词、真值表、范式转换
2. **深化**：[01-命题逻辑-增强版](01-命题逻辑-增强版.md) — 理解自然演绎、希尔伯特系统、完备性定理
3. **基础**：[02-谓词逻辑](02-谓词逻辑.md) — 学习量词、一阶语言、模型论初步

**配套练习：**

- 构建复杂命题的真值表
- 实现SAT求解器（DPLL算法）
- 练习自然演绎证明
- 分析经典悖论（理发师悖论、罗素悖论）

### 进阶方向

#### 方向A：经典逻辑深化（理论方向）

**学习序列：**

1. [02-谓词逻辑](02-谓词逻辑.md) — 一阶逻辑完备性、紧致性定理
2. 深入：模型论、递归论、集合论

**先修要求：**

- 数学分析基础
- 抽象代数（群论）
- [01-命题逻辑](01-命题逻辑.md)基础

#### 方向B：非经典逻辑与应用（AI/CS方向）

**学习序列：**

1. [03-模态逻辑](03-模态逻辑.md) — Kripke语义、公理系统、时态逻辑
2. [04-直觉逻辑](04-直觉逻辑.md) — 构造性证明、Curry-Howard对应
3. [05-多值逻辑](05-多值逻辑.md) — 模糊逻辑、不确定性推理

**先修要求：**

- 编程基础（Haskell/OCaml推荐）
- [02-谓词逻辑](02-谓词逻辑.md)基础
- 类型论基础（用于直觉逻辑）

#### 方向C：形式化与证明助手（实践方向）

**学习序列：**

1. [01-命题逻辑-增强版](01-命题逻辑-增强版.md) — 形式化证明系统
2. [04-直觉逻辑](04-直觉逻辑.md) — 类型论与证明即程序
3. 实践：Lean 4/Coq/Agda形式化证明

**衔接模块：**

- [09-形式化证明](../09-形式化证明/) — 形式化证明系统
- [10-语义模型](../10-语义模型/) — 模型论与范畴语义
