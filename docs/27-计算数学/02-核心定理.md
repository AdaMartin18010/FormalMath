## 一、Lax等价定理

### 1.1 适定性

**定义 1.1** (适定问题)
数学问题称为适定的（well-posed），如果：

1. **存在性**: 解存在
2. **唯一性**: 解唯一
3. **稳定性**: 解连续依赖于数据

Hadamard例子：Laplace方程的Cauchy问题是不适定的。

### 1.2 Lax等价定理

**定理 1.1** (Lax等价定理)
对线性初值问题的相容有限差分格式：

**稳定性 $\Leftrightarrow$ 收敛性**

即：相容的数值格式收敛当且仅当它是稳定的。

**关键概念**:

- **相容性**: 局部截断误差趋于零（当网格细化时）
- **稳定性**: 数值解有界（不指数增长）
- **收敛性**: 数值解趋于真解

**证明概要**:

**(稳定性 $\Rightarrow$ 收敛性)**:

设 $u$ 是真解，$u_h$ 是数值解，$\tau_h$ 是截断误差。

$$L_h u_h = f, \quad L_h u = f + \tau_h$$

相减：$L_h(u - u_h) = -\tau_h$

由稳定性：$\|u - u_h\| \leq C\|\tau_h\| \to 0$（相容性）

**(收敛性 $\Rightarrow$ 稳定性)**（反证法）:

假设不稳定，则存在有界初值使解无界增长，与收敛矛盾。∎

### 1.3 应用到差分格式

**例**：热方程 $u_t = u_{xx}$ 的显式格式

$$\frac{u_j^{n+1} - u_j^n}{\Delta t} = \frac{u_{j+1}^n - 2u_j^n + u_{j-1}^n}{\Delta x^2}$$

**相容性**: 截断误差 $O(\Delta t) + O(\Delta x^2)$

**稳定性条件** (von Neumann分析)：
$$r = \frac{\Delta t}{\Delta x^2} \leq \frac{1}{2}$$

由Lax等价定理，$r \leq 1/2$ 时格式收敛。

## 二、收敛性与误差估计

### 2.1 有限元误差估计

**定理 2.1** (Aubin-Nitsche对偶论证)
设 $u$ 是Poisson方程的解，$u_h$ 是P1有限元解，则

$$\|u - u_h\|_{L^2} \leq Ch^2\|u\|_{H^2}$$

比 $H^1$ 误差高一阶。

**证明概要**:

考虑对偶问题：求 $w$ 使得 $a(v, w) = (u - u_h, v)$ 对所有 $v$。

取 $v = u - u_h$：
$$\|u - u_h\|_{L^2}^2 = a(u - u_h, w) = a(u - u_h, w - w_h)$$

$$
\leq M\|u - u_h\|_{H^1}\|w - w_h\|_{H^1} \leq M\|u - u_h\|_{H^1} \cdot Ch\|w\|_{H^2}$$

利用正则性 $\|w\|_{H^2} \leq C\|u - u_h\|_{L^2}$：

$$\|u - u_h\|_{L^2} \leq Ch\|u - u_h\|_{H^1} \leq Ch^2\|u\|_{H^2}$$ ∎

### 2.2 谱方法收敛率

**定理 2.2** (谱收敛)
设 $u \in H^s(-1, 1)$，$u_N$ 是 $N$ 阶Chebyshev逼近，则

$$\|u - u_N\|_{L^2} \leq C N^{-s}\|u\|_{H^s}$$

若 $u$ 解析，则
$$\|u - u_N\|_{L^2} \leq Ce^{-cN}$$

**证明要点**: 
Chebyshev系数 $\hat{u}_n = O(n^{-s})$ 对 $H^s$ 函数。

### 2.3 迭代法收敛速度

**定理 2.3** (CG收敛率)
对对称正定矩阵 $A$，条件数 $\kappa = \lambda_{max}/\lambda_{min}$，CG法满足：

$$\|x_k - x^*\|_A \leq 2\left(\frac{\sqrt{\kappa} - 1}{\sqrt{\kappa} + 1}\right)^k \|x_0 - x^*\|_A$$

其中 $\|x\|_A = \sqrt{x^T A x}$ 是能量范数。

**证明**: 基于Chebyshev多项式的极小极大性质。∎

## 三、条件数与稳定性

### 3.1 矩阵条件数

**定义 3.1** (条件数)
矩阵 $A$ 的条件数：

$$\kappa(A) = \|A\| \cdot \|A^{-1}\|$$

对2-范数：$\kappa_2(A) = \sigma_{max}/\sigma_{min}$

对对称正定矩阵：$\kappa_2(A) = \lambda_{max}/\lambda_{min}$

**定理 3.1** (扰动分析)
设 $Ax = b$，$(A + \Delta A)(x + \Delta x) = b + \Delta b$，则

$$\frac{\|\Delta x\|}{\|x\|} \leq \kappa(A)\left(\frac{\|\Delta A\|}{\|A\|} + \frac{\|\Delta b\|}{\|b\|}\right)$$

条件数衡量问题的敏感性。

### 3.2 后向稳定性

**定义 3.2** (后向稳定性)
算法称为后向稳定的，如果计算的解 $\tilde{x}$ 是附近问题的精确解：

$$(A + \Delta A)\tilde{x} = b, \quad \|\Delta A\| \leq C\epsilon_{machine}\|A\|$$

**定理 3.2** (Wilkinson)
带部分主元的LU分解是后向稳定的。

Householder QR分解是后向稳定的。

Gram-Schmidt正交化不是后向稳定的（需修正版本）。

## 四、迭代法收敛理论

### 4.1 不动点迭代

**定义 4.1** (不动点迭代)
求解 $x = G(x)$ 的迭代：$x^{(k+1)} = G(x^{(k)})$

**定理 4.1** (Banach不动点定理)
设 $G: D \to D$ 是压缩映射，$\|G(x) - G(y)\| \leq L\|x - y\|$，$L < 1$。

则存在唯一不动点 $x^*$，且

$$\|x^{(k)} - x^*\| \leq \frac{L^k}{1-L}\|x^{(1)} - x^{(0)}\|$$

**应用于线性系统**：
对 $Ax = b$，分裂 $A = M - N$，迭代：

$$Mx^{(k+1)} = Nx^{(k)} + b$$

收敛当且仅当 $\rho(M^{-1}N) < 1$，其中 $\rho$ 是谱半径。

### 4.2 谱半径与收敛

**定理 4.2** (收敛判据)
迭代 $x^{(k+1)} = Bx^{(k)} + c$ 收敛当且仅当 $\rho(B) < 1$。

渐近收敛率：$-\log_{10}\rho(B)$ 位/迭代

**Jacobi迭代**: $B_J = D^{-1}(L + U)$
**Gauss-Seidel**: $B_{GS} = (D - L)^{-1}U$

**定理 4.3** (Stein-Rosenberg)
若 $A$ 是不可约弱对角占优矩阵，则

$$\rho(B_{GS}) \leq \rho(B_J) < 1$$

或两者都 $\geq 1$。

即Gauss-Seidel不劣于Jacobi。

## 总结

本章节介绍了计算数学的核心定理：

- **Lax等价定理**: 稳定性与收敛性的等价关系
- **误差估计**: 有限元和谱方法的收敛率
- **条件数**: 问题敏感性和后向稳定性

这些定理为数值方法的可靠性提供了理论保证。
