## 问题1：LU分解求解

### 问题描述

用LU分解求解线性系统 $Ax = b$：

$$A = \begin{pmatrix} 2 & 1 & 1 \\ 4 & 3 & 3 \\ 8 & 7 & 9 \end{pmatrix}, \quad b = \begin{pmatrix} 4 \\ 10 \\ 26 \end{pmatrix}$$

1. 进行Doolittle LU分解
2. 前代求解 $Ly = b$
3. 回代求解 $Ux = y$

### 解答

**1. LU分解**

$k = 1$:
- $u_{1j} = a_{1j}$: $u_{11} = 2$, $u_{12} = 1$, $u_{13} = 1$
- $l_{i1} = a_{i1}/u_{11}$: $l_{21} = 4/2 = 2$, $l_{31} = 8/2 = 4$

$k = 2$:
- $u_{22} = a_{22} - l_{21}u_{12} = 3 - 2(1) = 1$
- $u_{23} = a_{23} - l_{21}u_{13} = 3 - 2(1) = 1$
- $l_{32} = (a_{32} - l_{31}u_{12})/u_{22} = (7 - 4(1))/1 = 3$

$k = 3$:
- $u_{33} = a_{33} - l_{31}u_{13} - l_{32}u_{23} = 9 - 4(1) - 3(1) = 2$

结果：
$$L = \begin{pmatrix} 1 & 0 & 0 \\ 2 & 1 & 0 \\ 4 & 3 & 1 \end{pmatrix}, \quad U = \begin{pmatrix} 2 & 1 & 1 \\ 0 & 1 & 1 \\ 0 & 0 & 2 \end{pmatrix}$$

**2. 前代求解 $Ly = b$**

- $y_1 = 4$
- $y_2 = 10 - 2(4) = 2$
- $y_3 = 26 - 4(4) - 3(2) = 26 - 16 - 6 = 4$

$y = (4, 2, 4)^T$

**3. 回代求解 $Ux = y$**

- $x_3 = 4/2 = 2$
- $x_2 = 2 - 1(2) = 0$
- $x_1 = (4 - 1(0) - 1(2))/2 = 1$

解：$x = (1, 0, 2)^T$

验证：$Ax = (2(1) + 0 + 2, 4(1) + 0 + 6, 8(1) + 0 + 18) = (4, 10, 26)$ ✓

## 问题2：迭代法比较

### 问题描述

求解 $Ax = b$：

$$A = \begin{pmatrix} 4 & -1 & 0 \\ -1 & 4 & -1 \\ 0 & -1 & 4 \end{pmatrix}, \quad b = \begin{pmatrix} 3 \\ 2 \\ 3 \end{pmatrix}$$

初值 $x^{(0)} = (0, 0, 0)^T$，比较Jacobi和Gauss-Seidel迭代3步。

### 解答

真解：$x = (1, 1, 1)^T$（可验证）

**Jacobi迭代**: $x_i^{(k+1)} = (b_i - \sum_{j \neq i} a_{ij}x_j^{(k)})/a_{ii}$

$k = 0$: $x^{(0)} = (0, 0, 0)$

$k = 1$:
- $x_1^{(1)} = 3/4 = 0.75$
- $x_2^{(1)} = 2/4 = 0.5$
- $x_3^{(1)} = 3/4 = 0.75$

$k = 2$:
- $x_1^{(2)} = (3 + 0.5)/4 = 0.875$
- $x_2^{(2)} = (2 + 0.75 + 0.75)/4 = 0.875$
- $x_3^{(2)} = (3 + 0.5)/4 = 0.875$

$k = 3$:
- $x_1^{(3)} = (3 + 0.875)/4 = 0.969$
- $x_2^{(3)} = (2 + 0.875 + 0.875)/4 = 0.938$
- $x_3^{(3)} = (3 + 0.875)/4 = 0.969$

误差：$\|x^{(3)} - x\|_\infty \approx 0.062$

**Gauss-Seidel迭代**:

$k = 1$:
- $x_1^{(1)} = 3/4 = 0.75$
- $x_2^{(1)} = (2 + 0.75)/4 = 0.688$
- $x_3^{(1)} = (3 + 0.688)/4 = 0.922$

$k = 2$:
- $x_1^{(2)} = (3 + 0.688)/4 = 0.922$
- $x_2^{(2)} = (2 + 0.922 + 0.922)/4 = 0.961$
- $x_3^{(2)} = (3 + 0.961)/4 = 0.990$

$k = 3$:
- $x_1^{(3)} = (3 + 0.961)/4 = 0.990$
- $x_2^{(3)} = (2 + 0.990 + 0.990)/4 = 0.995$
- $x_3^{(3)} = (3 + 0.995)/4 = 0.999$

误差：$\|x^{(3)} - x\|_\infty \approx 0.01$

**结论**: Gauss-Seidel收敛更快（符合理论预期）。

## 问题3：CG法求解

### 问题描述

用CG法求解：

$$\begin{pmatrix} 2 & -1 \\ -1 & 2 \end{pmatrix} \begin{pmatrix} x_1 \\ x_2 \end{pmatrix} = \begin{pmatrix} 1 \\ 0 \end{pmatrix}$$

初值 $x_0 = (0, 0)^T$。

### 解答

**迭代0**:
- $r_0 = b - Ax_0 = (1, 0)^T$
- $p_0 = r_0 = (1, 0)^T$

**迭代1**:
- $Ap_0 = (2, -1)^T$
- $\alpha_0 = \frac{r_0^T r_0}{p_0^T Ap_0} = \frac{1}{2}$
- $x_1 = x_0 + \alpha_0 p_0 = (0.5, 0)^T$
- $r_1 = r_0 - \alpha_0 Ap_0 = (1, 0) - 0.5(2, -1) = (0, 0.5)^T$
- $\beta_0 = \frac{r_1^T r_1}{r_0^T r_0} = \frac{0.25}{1} = 0.25$
- $p_1 = r_1 + \beta_0 p_0 = (0, 0.5) + 0.25(1, 0) = (0.25, 0.5)^T$

**迭代2**:
- $Ap_1 = (0, 0.75)^T$
- $\alpha_1 = \frac{0.25}{0.375} = \frac{2}{3}$
- $x_2 = (0.5, 0) + \frac{2}{3}(0.25, 0.5) = (\frac{2}{3}, \frac{1}{3})^T$
- $r_2 = (0, 0.5) - \frac{2}{3}(0, 0.75) = (0, 0)$

$r_2 = 0$，CG在2步内收敛（$n=2$ 系统理论最大步数）。

解：$x = (2/3, 1/3)^T$

验证：$Ax = (4/3 - 1/3, -2/3 + 2/3) = (1, 0)$ ✓

## 问题4：条件数分析

### 问题描述

Hilbert矩阵 $H_{ij} = 1/(i + j - 1)$，$n = 3$：

$$H = \begin{pmatrix} 1 & 1/2 & 1/3 \\ 1/2 & 1/3 & 1/4 \\ 1/3 & 1/4 & 1/5 \end{pmatrix}$$

1. 计算 $\kappa_2(H)$
2. 若右端项有 $10^{-6}$ 扰动，估计解的相对误差

### 解答

**1. 计算条件数**

特征值计算（数值）：
- $\lambda_1 \approx 1.408$
- $\lambda_2 \approx 0.122$
- $\lambda_3 \approx 0.003$

$$\kappa_2(H) = \frac{\lambda_{max}}{\lambda_{min}} \approx \frac{1.408}{0.003} \approx 469.5$$

**2. 误差估计**

$$\frac{\|\Delta x\|}{\|x\|} \leq \kappa(H) \frac{\|\Delta b\|}{\|b\|} = 469.5 \times 10^{-6} \approx 4.7 \times 10^{-4}$$

尽管输入扰动很小，解可能有 $0.05\%$ 的相对误差。

注：$n = 10$ 时 $\kappa_2(H_{10}) \approx 10^{13}$，完全病态！

## 问题5：有限差分稳定性

### 问题描述

热方程 $u_t = u_{xx}$，$x \in [0, 1]$，$t \in [0, T]$。

显式格式：
$$u_j^{n+1} = u_j^n + r(u_{j+1}^n - 2u_j^n + u_{j-1}^n)$$

其中 $r = \Delta t / \Delta x^2$。

1. 用von Neumann分析推导稳定性条件
2. 若 $\Delta x = 0.01$，求最大允许 $\Delta t$

### 解答

**1. von Neumann稳定性分析**

设 Fourier模式 $u_j^n = G^n e^{ikj\Delta x}$，代入格式：

$$G = 1 + r(e^{ik\Delta x} - 2 + e^{-ik\Delta x}) = 1 - 4r\sin^2\left(\frac{k\Delta x}{2}\right)$$

稳定性要求 $|G| \leq 1$ 对所有 $k$：

$$-1 \leq 1 - 4r\sin^2(\theta) \leq 1$$

右不等式自动满足（$r > 0$）。左不等式：

$$1 - 4r\sin^2(\theta) \geq -1$$
$$4r\sin^2(\theta) \leq 2$$

最坏情况 $\sin^2(\theta) = 1$：

$$r \leq \frac{1}{2}$$

**2. 最大时间步长**

$$\Delta t \leq \frac{\Delta x^2}{2} = \frac{0.0001}{2} = 5 \times 10^{-5}$$

非常严格的时间步长限制！

## 问题6：有限元刚度矩阵

### 问题描述

一维Poisson方程：$-u'' = f$，$u(0) = u(1) = 0$

均匀网格，$h = 1/4$，P1元。

1. 构造刚度矩阵 $A$
2. 对 $f = 1$，求解有限元系统

### 解答

**1. 刚度矩阵**

P1元局部刚度（参考元）：
$$A_K = \frac{1}{h}\begin{pmatrix} 1 & -1 \\ -1 & 1 \end{pmatrix}$$

组装全局矩阵（3个内部节点）：

$$A = \frac{1}{h}\begin{pmatrix} 2 & -1 & 0 \\ -1 & 2 & -1 \\ 0 & -1 & 2 \end{pmatrix} = 4\begin{pmatrix} 2 & -1 & 0 \\ -1 & 2 & -1 \\ 0 & -1 & 2 \end{pmatrix}$$

**2. 右端向量**

局部载荷：$F_K = h/2 \cdot (1, 1)^T$

组装：$F = h(1, 1, 1)^T = (0.25, 0.25, 0.25)^T$

**3. 求解**

系统：$4(2u_1 - u_2) = 0.25$，$4(-u_1 + 2u_2 - u_3) = 0.25$，$4(-u_2 + 2u_3) = 0.25$

由对称性：$u_1 = u_3$

$$8u_1 - 4u_2 = 0.25$$
$$-4u_1 + 8u_2 - 4u_1 = 0.25 \Rightarrow -8u_1 + 8u_2 = 0.25$$

相加：$4u_2 = 0.5$，$u_2 = 0.125$

$u_1 = (0.25 + 4(0.125))/8 = 0.75/8 = 0.09375$

数值解：$u = (0.09375, 0.125, 0.09375)$

真解 $u(x) = x(1-x)/2$ 在节点：$(0.09375, 0.125, 0.09375)$，有限元解精确！（P1元对二次精确解）

## 问题7：谱方法微分

### 问题描述

函数 $u(x) = \sin(\pi x)$，$x \in [-1, 1]$。

用Chebyshev谱方法近似 $u'(x)$，取 $N = 4$。

1. 计算Chebyshev系数
2. 构造微分矩阵
3. 计算 $u'(0)$ 的近似值

### 解答

**1. Chebyshev系数**

Chebyshev点：$x_j = \cos(j\pi/N)$，$j = 0, 1, 2, 3, 4$

$x = (1, 0.707, 0, -0.707, -1)$

$u = (0, 1, 0, -1, 0)$（$\sin(\pi x_j)$）

用Chebyshev变换计算系数...

**2. 微分矩阵**

Chebyshev微分矩阵 $D$（$N = 4$）：

$$D_{ij} = \frac{c_i}{c_j}\frac{(-1)^{i+j}}{x_i - x_j}, \quad i \neq j$$

其中 $c_0 = c_4 = 2$，$c_1 = c_2 = c_3 = 1$

$$D_{ii} = -\frac{x_i}{2(1-x_i^2)}, \quad i = 1, 2, 3$$
$$D_{00} = \frac{2N^2 + 1}{6}, \quad D_{NN} = -\frac{2N^2 + 1}{6}$$

**3. 计算导数**

$u' = Du$，在 $x = 0$（$j = 2$）：

$u'(0) \approx \pi \cos(0) = \pi \approx 3.14159$

谱方法在此光滑函数上指数收敛。

## 问题8：FFT应用

### 问题描述

用FFT计算周期函数 $f(x) = \sin(4\pi x) + 0.5\sin(8\pi x)$ 在 $N = 8$ 个均匀点上的DFT。

1. 确定采样点
2. 计算DFT系数
3. 验证Parseval等式

### 解答

**1. 采样点**

$x_j = j/N = j/8$，$j = 0, \ldots, 7$

$f_j = \sin(\pi j/2) + 0.5\sin(\pi j)$

值：$f = (0, 1, 0, -1, 0, 1, 0, -1) + (0, 0.5, 0, -0.5, 0, 0.5, 0, -0.5)$（第二项全零）

实际：$f = (0, 1, 0, -1, 0, 1, 0, -1)$

**2. DFT**

$$\hat{f}_k = \sum_{j=0}^{7} f_j e^{-2\pi i j k / 8}$$

计算得：
- $\hat{f}_2 = -4i$，$\hat{f}_6 = 4i$（对应 $\sin(4\pi x)$）
- 其余为零

**3. Parseval等式**

$$\sum_{j=0}^{7} |f_j|^2 = 4 \times 1 = 4$$

$$\frac{1}{8}\sum_{k=0}^{7} |\hat{f}_k|^2 = \frac{1}{8}(16 + 16) = 4$$

验证成立。

## 问题9：插值误差

### 问题描述

Runge函数 $f(x) = \frac{1}{1 + 25x^2}$，$x \in [-1, 1]$。

比较等距点Lagrange插值和Chebyshev点插值在 $x = 0.5$ 处的误差，$n = 10$。

### 解答

**等距点插值**:

$x_j = -1 + 2j/n$，$j = 0, \ldots, 10$

在 $x = 0.5$ 附近，等距点Lagrange插值振荡严重（Runge现象）。

误差 $|f(0.5) - p_{10}(0.5)| \approx O(1)$（很大！）

**Chebyshev点插值**:

$x_j = \cos(j\pi/n)$

Chebyshev插值最小化最大误差：

$$|f(0.5) - p_{10}(0.5)| \approx 0.01$$

Chebyshev点避免了Runge现象。

## 问题10：后向误差分析

### 问题描述

求解 $Ax = b$，计算解 $\tilde{x}$ 满足残差 $\|b - A\tilde{x}\| = \epsilon$。

证明存在 $\Delta A$ 使得 $(A + \Delta A)\tilde{x} = b$ 且估计 $\|\Delta A\|$。

### 解答

**构造**:

设 $\Delta A = r\tilde{x}^T / \|\tilde{x}\|^2$，其中 $r = b - A\tilde{x}$。

验证：
$$(A + \Delta A)\tilde{x} = A\tilde{x} + \frac{r\tilde{x}^T\tilde{x}}{\|\tilde{x}\|^2} = A\tilde{x} + r = b$$

**范数估计**:

$$\|\Delta A\| = \frac{\|r\|}{\|\tilde{x}\|}$$

相对后向误差：

$$\frac{\|\Delta A\|}{\|A\|} = \frac{\|r\|}{\|A\|\|\tilde{x}\|} \leq \frac{\|r\|}{\|b\|} \kappa(A)$$

若残差小且条件数适中，则后向误差小，算法后向稳定。
