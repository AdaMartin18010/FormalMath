## 三、质量自测清单

在提交或审校前，请确认：

- [ ] frontmatter 包含 `title`、`level`、`course`、`chapter`、`msc_primary`
- [ ] `external_ids` 至少包含课程主页和具体讲座/习题链接
- [ ] `references.textbooks` 包含 ISBN 和精确章节
- [ ] 正文包含严格定义，且与权威教材等价
- [ ] 核心定理附带完整证明或详细证明思路
- [ ] 至少 2 个典型例题，含详细解答
- [ ] 至少 5 道习题，含解答或提示
- [ ] 列出至少 2 个常见误区及纠正
- [ ] 包含与其他概念的交叉引用
- [ ] 文末包含「参考与延伸阅读」节
