# 可视化内容

本目录为项目的辅助性可视化资源专区，按概念、定理、证明和应用四个维度组织可视化内容，通过图形、图表和交互式展示帮助理解数学知识的结构与逻辑。

## 内容导航

- **01-概念可视化** — 核心概念的图形化阐释
- **02-定理可视化** — 定理陈述与推论图示
- **03-证明可视化** — 证明步骤的动态/静态展示
- **04-应用可视化** — 数学模型在实际场景中的呈现
- **可视化总览** — 全目录导览与使用指南
