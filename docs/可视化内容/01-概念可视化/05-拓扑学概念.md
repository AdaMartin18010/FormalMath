## 一、同伦概念可视化

### 1.1 同伦的定义

```mermaid
graph TB
    subgraph 同伦 H: X×I → Y
    F[映射 f: X → Y] --> H[同伦]
    G[映射 g: X → Y] --> H

    H --> D[H(x,0) = f(x)<br/>H(x,1) = g(x)]

    D --> C[连续形变]
    C --> V[可视: f 连续变为 g]

    H --> E[同伦等价<br/>X ≃ Y]
    E --> HE[存在 f: X→Y, g: Y→X<br/>g∘f ≃ id_X, f∘g ≃ id_Y]

    HE --> R[形变收缩<br/>X 收缩到 A ⊆ X]
    end

    style F fill:#e1e5ff
    style G fill:#e1e5ff
    style H fill:#e1f5e1
    style E fill:#ffe1e1

```

### 1.2 同伦的直观图示

```

映射 f: X → Y 到 g: X → Y 的同伦:

时间 t=0:                时间 t=1/2:              时间 t=1:
   f(X)                    H(X,1/2)                 g(X)
    │                        │                        │
    ●───                     ●~──                     ╭─╮
   ╱│                        │~╲                      │ │
  ╱ │                        │ ~╲                     │ │
 ●  │                        ●  ~●                    ╰─╯
    │                           ~~

同伦等价 (X ≃ Y) 的例子:

1. ℝ^n ≃ {点} (可缩空间)
   收缩映射: H(x,t) = tx

2. 穿孔平面 ℝ²\{0} ≃ S¹
   收缩: H(x,t) = (1-t)x + t·x/|x|

3. Möbius带 ≃ S¹
   收缩到中心圆

4. 实心环 ≃ S¹ ∨ S¹ (两个圆 wedge)

非同伦等价的例子:
• S¹ 不等价于 {点} (有洞)
• S² 不等价于 T² (不同亏格)
• ℝ^m 不等价于 ℝ^n (m≠n, 维数不变量)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 二、基本群可视化

### 2.1 基本群的定义

```mermaid
graph TB
    subgraph 基本群 π₁(X,x₀)
    X[拓扑空间 X] --> P[基点 x₀ ∈ X]
    P --> L[环路<br/>γ: I → X, γ(0)=γ(1)=x₀]

    L --> M[乘法: 环路连接<br/>(γ·δ)(t) = ...]
    L --> I[逆元: 反向环路<br/>γ⁻¹(t) = γ(1-t)]
    L --> E[单位元: 常值环路]

    M --> H[同伦等价类<br/>[γ] = {δ | δ ≃ γ}]

    I --> H
    E --> H

    H --> G[π₁(X,x₀) = {[γ]}<br/>群运算: [γ]·[δ] = [γ·δ]]

    G --> C[若X道路连通,<br/>不同基点同构]
    end

    style X fill:#e1e5ff
    style L fill:#e1f5e1
    style G fill:#ffe1e1

```

## 三、覆叠空间可视化

### 3.1 覆叠空间的定义

```mermaid
graph TB

    L --> U[∀b∈B, ∃邻域 U<br/>p⁻¹(U) = ⊔Vₐ]

    F --> D[覆叠的叶数]
    D --> N[可以是有限的或无限的]

    P --> G[覆叠变换群<br/>Deck(E/B)]
```

## 四、同调群概念

### 4.1 同调群的构造

```mermaid
graph TB
    subgraph 同调群 H_n(X)
    X[空间 X] --> S[奇异单形<br/>σ: Δⁿ → X]

    S --> C[链群 C_n(X)<br/>形式线性组合]

    C --> D[边缘算子<br/>∂: C_n → C_{n-1}]
    D --> P[∂∘∂ = 0<br/>∂² = 0]

    C --> Z[闭链群<br/>Z_n = ker(∂: C_n→C_{n-1})]
    C --> B[边缘链群<br/>B_n = im(∂: C_{n+1}→C_n)]

    P --> I[B_n ⊆ Z_n]

    Z --> H[同调群<br/>H_n = Z_n / B_n]
    B --> H

    H --> R[同调类 [z]<br/>z 是闭链, 模去边缘]
    end

    style C fill:#e1e5ff
    style Z fill:#e1f5e1
    style H fill:#ffe1e1

```

### 4.2 同调群的计算

```

同调群的计算实例
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

空间            H₀    H₁      H₂      H₃      ...
────────────────────────────────────────────────
点              ℤ     0       0       0       ...
ℝ^n             ℤ     0       0       0       ...
S¹              ℤ     ℤ       0       0       ...
S²              ℤ     0       ℤ       0       ...
S^n             ℤ     0  ...  0  ℤ    0       ...
T² = S¹×S¹      ℤ     ℤ²      ℤ       0       ...
T^n             ℤ     ℤ^(n)   ℤ^(C(n,2)) ...
ℝP²             ℤ     ℤ/2ℤ    0       0       ...
ℝP^n            ℤ     ℤ/2ℤ or 0 交替  ...
Klein瓶         ℤ     ℤ⊕ℤ/2ℤ  0       0       ...

几何解释:
• H₀: 道路连通分支的数目
  H₀(X) = ℤ^{#连通分支}

• H₁: "1维洞"的数目
  H₁(S¹) = ℤ (圆有一个1维洞)
  H₁(T²) = ℤ² (环面有两个独立环路)

• H₂: "2维洞"的数目
  H₂(S²) = ℤ (球面包围一个2维洞)
  H₂(T²) = ℤ (环面内部)

Euler示性数:
χ(X) = Σ (-1)^n rank H_n(X)
     = Σ (-1)^n #n维胞腔

Mayer-Vietoris序列:
计算 X = U ∪ V 的同调的长正合列
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

## 五、上同调环

### 5.1 上同调的定义

```mermaid
graph TB
    subgraph 上同调 H^*(X)
    X[空间 X] --> C[上链群 C^n(X)<br/>C_n(X) → ℤ 的同态]

    C --> D[上边缘算子<br/>δ: C^n → C^{n+1}]
    D --> P[δ∘δ = 0]

    C --> Z[上闭链<br/>Z^n = ker(δ)]
    C --> B[上边缘<br/>B^n = im(δ)]

    Z --> H[上同调群<br/>H^n = Z^n / B^n]
    B --> H

    H --> U[上积 ∪<br/>H^p × H^q → H^{p+q}]
    U --> R[上同调环<br/>H^*(X) = ⊕ H^n]

    R --> G[分级交换环]
    end

    style C fill:#e1e5ff
    style H fill:#e1f5e1
    style R fill:#ffe1e1

```

### 5.2 上同调环的结构

```

上同调环的结构
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

上积 (Cup Product):
[α] ∪ [β] = [α∪β]

性质:
• 分次交换性: α∪β = (-1)^{pq} β∪α
• 结合性
• 有单位元 1 ∈ H^0

例子:

1. H^*(S^n) = ℤ[x]/(x²)  deg x = n
   只有 1 和 x 两个生成元

2. H^*(T²) = ℤ[a,b]/(a²,b²)  deg a = deg b = 1
   H^0 = ℤ·1
   H^1 = ℤ·a ⊕ ℤ·b
   H^2 = ℤ·(a∪b)

   a∪a = 0, b∪b = 0
   a∪b = -b∪a (分次反对易)

3. H^*(ℂP^n) = ℤ[x]/(x^{n+1})  deg x = 2
   截断多项式环

Künneth公式:
H^*(X×Y) ≅ H^*(X) ⊗ H^*(Y) (在域上)

应用: 区分同调群相同但环结构不同的空间
• S¹∨S¹∨S² vs T²: H_n 相同, H^* 不同
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

## 六、纤维丛与示性类

### 6.1 示性类的概念

```mermaid
graph TB
    subgraph 示性类
    B[底空间 B] --> F[纤维丛 E → B]

    F --> C[示性类<br/>c(E) ∈ H^*(B)]

    C --> T[类型]
    T --> S[Stiefel-Whitney<br/>w_i ∈ H^i(B;ℤ/2)]
    T --> CH[Chern类<br/>c_i ∈ H^{2i}(B;ℤ)]
    T --> P[Pontryagin类<br/>p_i ∈ H^{4i}(B;ℤ)]
    T --> E[Euler类<br/>e ∈ H^n(B;ℤ)]

    C --> P1[性质]
    P1 --> N1[自然性]
    P1 --> W[Whitney和公式]
    P1 --> N2[规范化]

    S --> C1[w(TM) 决定可定向性]
    CH --> C2[c(复向量丛)]
    E --> C3[e(TM) = χ(M)·[M]]
    end

    style F fill:#e1e5ff
    style C fill:#e1f5e1
    style T fill:#ffe1e1

```

### 6.2 示性类的应用

```

示性类的应用
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Stiefel-Whitney类 w_i(TM):

   w₁ = 0 ⟺ M 可定向
   w₂ = 0 ⟺ M 有Spin结构 (可以定义旋量场)

   例子:
   • S^n: w = 1 (所有 w_i = 0 for i>0)
   • ℝP^n: w = (1+a)^{n+1}, a ∈ H¹
   • Möbius带: w₁ ≠ 0 (不可定向)

2. Chern类 c_i(E) (复向量丛):

   总Chern类: c(E) = 1 + c₁ + c₂ + ...
   Whitney公式: c(E⊕F) = c(E)·c(F)

   例子: 复射影空间 ℂP^n
   c(TℂP^n) = (1+x)^{n+1}
   其中 x = c₁(γ) 是超平面丛的Chern类

3. Euler类 e(TM):

   对于可定向闭流形:
   ∫_M e(TM) = χ(M) (Euler示性数)

   Poincaré-Hopf定理:
   χ(M) = Σ index(p_i) (向量场零点指标的和)

4. Pontryagin类 p_i(E):

   对于实向量丛, p_i(E) = (-1)^i c_{2i}(E⊗ℂ)

   Hirzebruch符号差定理:
   sign(M) = ∫ L(p₁,p₂,...) (L-亏格)

示性类完全分类:
复线丛 ↔ c₁ ∈ H²(B;ℤ)
(由第一Chern类完全决定)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

## 七、Morse理论

### 7.1 Morse函数

```mermaid
graph TB
    subgraph Morse理论
    M[流形 M] --> F[Morse函数<br/>f: M → ℝ]

    F --> C[临界点<br/>df_p = 0]
    C --> N[非退化<br/>Hessian非奇异]

    N --> I[指标 ind(p)<br/>Hessian负特征值数]
    I --> D[ind(p) = 经过p的下降方向维数]

    F --> H[水平集 M_a = f⁻¹(-∞,a]]
    H --> T[拓扑变化]
    T --> A[a 经过临界点c时<br/>M_a 附加 ind(c)维胞腔]

    T --> B[Morse不等式]
    B --> MI[C_k ≥ R_k (Betti数)]

    MI --> E[Euler示性数<br/>χ = Σ(-1)^k C_k]
    end

    style F fill:#e1e5ff
    style C fill:#e1f5e1
    style I fill:#ffe1e1

```

### 7.2 Morse理论的应用

```

Morse理论的直观与应用
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Morse函数的地形比喻:

高度 f                    拓扑变化
  │
max├─╮   ╭─╮              附加 2-胞腔 (指标2)
   │  ╲ ╱  ╲
   │   ●     ● 鞍点        附加 1-胞腔 (指标1)
   │  ╱ ╲   ╱ ╲
   │ ╱   ╲ ╱   ╲
   │●     ●     ● 极小点    附加 0-胞腔 (指标0)
   └──────────────────→

例子: 环面 T² 站立放置
• 底部: 极小点 (指标0)
• 两个鞍点 (指标1)
• 顶部: 极大点 (指标2)

Morse不等式:
C_k ≥ R_k = rank H_k(M)
其中 C_k = 指标k临界点个数

完美Morse函数: C_k = R_k 对所有k
(如环面站立放置)

Morse同调:
链复形: C_k = ℤ^{C_k}
边缘算子: 计算梯度流线数

定理: Morse同调 ≅ 奇异同调

应用:
• 证明LS畴数下界
• 证明Bott周期性
• Floer同调的雏形
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

## 八、同伦论高级概念

### 8.1 高阶同伦群

```mermaid
graph TB
    subgraph 高阶同伦群 π_n(X,x₀)
    X[空间 X] --> N[n ≥ 2]
```
