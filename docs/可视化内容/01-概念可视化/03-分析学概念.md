## 一、极限过程可视化

### 1.1 极限的epsilon-delta定义

```mermaid
graph TB
    subgraph 极限 lim x→a f(x) = L
    E[ε > 0] --> D[存在 δ > 0]
    D --> C[条件: |x-a| < δ]
    C --> R[结果: |f(x)-L| < ε]

    E --> V[ε-邻域<br/>V_ε(L)]
    D --> V2[δ-邻域<br/>V_δ(a)]

    V2 --> MAP[f: V_δ(a) → V_ε(L)]
    end

    style E fill:#ffe1e1
    style D fill:#e1f5e1
    style R fill:#e1e5ff

```

### 1.2 极限过程的动态图

```

极限过程: f(x) → L 当 x → a
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    y
    │                    ┌──────────┐
 L+ε├────────────────────┤          │
    │                    │    ε     │
  L ├────────────────────┼────●─────┼───────
    │                    │    带    │
 L-ε├────────────────────┤          │
    │     ┌────────┐     └──────────┘
    │     │  δ带   │
    │     └───┬────┘
    └─────┬───┼───┬──────────────────────→ x
          a-δ a a+δ

动态过程:
• 先给定 ε > 0 (目标精度)
• 找到 δ > 0 (输入精度)
• 使得 x 进入 δ-邻域 ⟹ f(x) 进入 ε-邻域

数学表达:
∀ε > 0, ∃δ > 0, 使得 |x-a| < δ ⟹ |f(x)-L| < ε

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

## 三、收敛性可视化

### 3.1 序列收敛

```mermaid
graph TB
    subgraph 序列收敛 xₙ → L
    S[序列 {xₙ}] --> C[收敛定义]

    C --> D[∀ε>0, ∃N, 使得 n>N ⟹ |xₙ-L|<ε]

    D --> G[几何解释]
    G --> G1[从某项起所有项<br/>都落入ε-邻域]

    S --> T[收敛类型]
    T --> A[绝对收敛<br/>Σ|xₙ|收敛]

    T --> C2[条件收敛<br/>收敛但不绝对收敛]
    T --> D2[发散]
    end

    style S fill:#e1e5ff
    style C fill:#e1f5e1

```

### 3.2 一致收敛

```

点态收敛 vs 一致收敛
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

点态收敛 (fₙ → f):
对每个固定的 x, ∀ε>0, ∃N(x), n>N ⟹ |fₙ(x)-f(x)|<ε

        ↑
   N 依赖于 x

一致收敛 (fₙ ⇉ f):
∀ε>0, ∃N (与x无关), 对所有x, n>N ⟹ |fₙ(x)-f(x)|<ε

可视化:

一致收敛:                    点态收敛但不一致:
┌─────────────────┐         ┌─────────────────┐
│  f+ε ────────   │         │  f+ε ───╱────   │
│     ╱ fₙ  ╲     │         │     ╱    ╲      │
│  f ━━━━━━━━     │         │  f ━━━━━━━━     │
│     ╲     ╱     │         │     ╲    ╱      │
│  f-ε ────────   │         │  f-ε ───╲────   │
│       ↑         │         │        ↑        │
│    整体进入     │         │     局部进入    │
└─────────────────┘         └─────────────────┘

关键区别:
• 一致收敛保持连续性
• 一致收敛允许逐项积分
• 一致收敛不一定能逐项微分
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

## 四、微分概念可视化

### 4.1 导数的几何意义

```mermaid
graph TB
    subgraph 导数 f'(a)
    C[曲线 y=f(x)] --> P[点 (a, f(a))]
    P --> T[切线<br/>斜率 = f'(a)]

    T --> D[定义<br/>f'(a) = lim h→0 (f(a+h)-f(a))/h]

    D --> G[几何意义]
    G --> G1[瞬时变化率]
    G --> G2[切线斜率]
    G --> G3[线性逼近<br/>f(x) ≈ f(a) + f'(a)(x-a)]
    end

    style C fill:#e1e5ff
    style T fill:#ffe1e1

```

### 4.2 微分与线性逼近

```

微分的线性逼近
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

        y
        │
   f(x) ┼──────╱╱╱╱╱╱╱╱ 曲线 y=f(x)
        │     ╱│
   f(a) ┼────●─┼──────── 切点
        │   ╱  │
        │  ╱   │ Δf = f(x)-f(a)
        │ ╱    │
        │╱ df  │
        └──────┼────────────────→ x
               a    x
               ↑____↑
                  dx

微分定义:
df = f'(a) · dx

当 dx → 0 时:
Δf ≈ df = f'(a)·dx

即: f(x) ≈ f(a) + f'(a)(x-a)
   这是函数在 a 点附近的最佳线性逼近
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

## 五、积分概念可视化

### 5.1 Riemann积分与Darboux和

```mermaid
graph TB
    subgraph Riemann积分
    F[函数 f 在 [a,b]] --> R[Riemann可积]

    R --> L[下和 L(f,P)]
    R --> U[上和 U(f,P)]

    L --> I[下积分<br/>sup L(f,P)]
    U --> S[上积分<br/>inf U(f,P)]

    I --> C[可积条件<br/>上积分 = 下积分]
    S --> C

    C --> V[积分值 ∫ₐᵇ f]
    end

    style R fill:#e1f5e1
    style C fill:#ffe1e1

```

### 5.2 积分逼近过程

```

Riemann和逼近
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

函数 f(x) 在 [a,b] 上的积分:

1. 分割区间:
   a = x₀ < x₁ < x₂ < ... < xₙ = b

2. 取样本点 ξᵢ ∈ [xᵢ₋₁, xᵢ]

3. Riemann和:
   S = Σᵢ₌₁ⁿ f(ξᵢ) · Δxᵢ

可视化:

y │    ┌─┐
  │    │ │┌─┐
  │   ┌┘ ││ │┌─┐      ┌─┐
  │  ┌┘  ││ ││ │┌─┐  ┌┘ │
  │ ┌┘   ││ ││ ││ │┌─┘  │
  │─┘    ││ ││ ││ ││    │
  └──────┴┴─┴┴─┴┴─┴┴────┴────→ x
  a     分割点              b

当分割加细 (max Δxᵢ → 0):
  S → ∫ₐᵇ f(x)dx

上和 U(f,P) 与下和 L(f,P):
   U(f,P) = Σ Mᵢ·Δxᵢ  (Mᵢ = sup f on [xᵢ₋₁,xᵢ])
   L(f,P) = Σ mᵢ·Δxᵢ  (mᵢ = inf f on [xᵢ₋₁,xᵢ])

   L(f,P) ≤ ∫ₐᵇ f ≤ U(f,P)

   f 可积 ⟺ U(f,P) - L(f,P) 可任意小
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

## 八、测度与可测集

### 8.1 测度空间结构

```mermaid
graph TB
    subgraph 测度空间 (X, A, μ)
    X[全集 X] --> A[σ-代数 A<br/>可测集族]
    A --> S[可测集]

    S --> C1[对补集封闭<br/>E∈A ⟹ E^c∈A]
    S --> C2[对可数并封闭<br/>Eₙ∈A ⟹ ∪Eₙ∈A]

    X --> M[测度 μ: A → [0,∞]]
    M --> P1[非负性<br/>μ(E) ≥ 0]
    M --> P2[空集为零<br/>μ(∅) = 0]
    M --> P3[可数可加性<br/>μ(∪Eₙ) = Σμ(Eₙ) 对不交并]

    A --> L[Lebesgue测度空间<br/>(ℝ^n, L, λ)]
    end

    style X fill:#e1e5ff
    style A fill:#e1f5e1
    style M fill:#ffe1e1

```

### 8.2 Lebesgue测度的构造

```

Lebesgue测度构造
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步骤1: 区间测度
  λ((a,b)) = b - a

步骤2: 外测度
  λ*(E) = inf{Σ(bᵢ-aᵢ) : E ⊆ ∪(aᵢ,bᵢ)}

步骤3: 可测集 (Caratheodory条件)
  E 可测 ⟺ ∀A⊆ℝ: λ*(A) = λ*(A∩E) + λ*(A∩E^c)

步骤4: Lebesgue测度
  λ(E) = λ*(E) 对可测集 E

可测集的性质:
┌─────────────────────────────────────────────┐
│                                             │
│ • Borel集都是Lebesgue可测的                │
│ • 可测集构成σ-代数                          │

│ • 可数集                                    │
```

## 十、希尔伯特空间结构

### 10.1 Hilbert空间定义

```mermaid
graph TB
    subgraph Hilbert空间 H
    V[向量空间] --> I[内积 ⟨·,·⟩: H×H → ℂ]

    I --> P1[共轭对称<br/>⟨u,v⟩ = ⟨v,u⟩的共轭]
    I --> P2[线性性<br/>⟨αu+βv,w⟩ = α⟨u,w⟩+β⟨v,w⟩]
    I --> P3[正定性<br/>⟨v,v⟩ ≥ 0, =0 ⟺ v=0]

    I --> N[诱导范数<br/>||v|| = √⟨v,v⟩]

    N --> C[完备性]
    C --> H2[Hilbert空间<br/>完备内积空间]

    I --> O[正交性<br/>u⊥v ⟺ ⟨u,v⟩=0]
    O --> P[正交投影]
    end

    style I fill:#e1f5e1
    style H2 fill:#ffe1e1
    style O fill:#e1e5ff

```

### 10.2 Hilbert空间的正交结构

```

Hilbert空间的正交结构
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

核心定理:

1. 正交投影定理:
   对闭子空间 M ⊆ H, 每个 v∈H 有唯一分解:
   v = v_M + v_M⊥
   其中 v_M∈M, v_M⊥∈M⊥

        H
        │
    v_M ┼────●─── v
        │   ╱
        │  ╱ v_M⊥
        │ ╱
   ─────┼╱───────── M
        │
       M⊥

2. 正交基展开:
   若 {eₙ} 是标准正交基, 则:
   v = Σ⟨v,eₙ⟩eₙ

   ||v||² = Σ|⟨v,eₙ⟩|² (Parseval等式)

3. Riesz表示定理:
   每个连续线性泛函 f: H → ℂ 可表示为:
   f(v) = ⟨v, u_f⟩ 对某个 u_f∈H

经典Hilbert空间:
• ℝ^n, ℂ^n (有限维)
• ℓ^2 (平方可和序列)
• L^2[a,b] (平方可积函数)
• H^s (Sobolev空间)
• Hardy空间 H²
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```
