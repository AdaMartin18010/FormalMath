## 一、Lagrange定理

### 定理陈述

**定理**: 若 $H$ 是有限群 $G$ 的子群，则 $|H|$ 整除 $|G|$，且 $[G:H] = |G|/|H|$。

### 证明流程图

```mermaid
graph TD
    A[H ≤ G] --> B[左陪集分解]
    B --> C[G = ∪ gᵢH]
    C --> D[陪集互不相交]
    D --> E[|gH| = |H|]
    E --> F[|G| = [G:H] · |H|]
    F --> G[结论: |H| 整除 |G|]

    style A fill:#e1e5ff
    style F fill:#ffe1e1
    style G fill:#e1f5e1

```

## 二、第一同构定理

### 定理陈述

**定理**: 设 $\varphi: G \to H$ 是群同态，则 $G/\ker(\varphi) \cong \text{im}(\varphi)$。

### 证明结构图

```mermaid
graph TB
    G[群G] -->|同态 φ| H[群H]

    G --> K[ker(φ) ⊲ G]
    H --> I[im(φ) ≤ H]

    K --> Q[商群 G/ker(φ)]
    Q --> ISO{≅}
    I --> ISO

    ISO --> R[G/ker(φ) ≅ im(φ)]

    Q -->|自然投影 π| G
    I -->|包含 ι| H

    style G fill:#e1e5ff
    style H fill:#e1e5ff
    style ISO fill:#ffe1e1
    style R fill:#e1f5e1

```

### 定理关系

```

同构定理的层次
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第一同构定理:
  G/ker(φ) ≅ im(φ)

第二同构定理 (钻石定理):
  H/(H∩N) ≅ HN/N  (H≤G, N⊲G)

第三同构定理:
  (G/N)/(H/N) ≅ G/H  (N≤H≤G, N,H⊲G)

应用:
• 证明群的结构
• 构造同构
• 计算商群

典型例子:
GL(n,ℝ)/SL(n,ℝ) ≅ ℝ* (乘法群)
S_n/A_n ≅ ℤ/2ℤ
```

## 三、Sylow定理

### 定理陈述

**定理**: 设 $G$ 是有限群，$|G| = p^n \cdot m$，$p \nmid m$，则：

1. 存在 $p^n$ 阶子群 (Sylow $p$-子群)
2. 所有Sylow $p$-子群共轭
3. 数目 $n_p \equiv 1 \pmod{p}$ 且 $n_p \mid m$

### 证明策略图

```mermaid
graph TD
    A[|G| = pⁿm] --> B[考虑G在自身上的作用]

    B --> C[计数轨道]
    C --> D[轨道公式]
    D --> E[存在大小为pⁿ的轨道]
    E --> F[稳定子群是Sylow p-子群]

    F --> G[定理1: 存在性]
    F --> H[定理2: 共轭性]
    F --> I[定理3: 计数]

    style A fill:#e1e5ff
    style F fill:#ffe1e1
    style G fill:#e1f5e1
    style H fill:#e1f5e1
    style I fill:#e1f5e1

```

### 应用实例

```

群 |G| = 60 = 2²·3·5 的分类:

Sylow 2-子群: 阶4, n₂ | 15, n₂ ≡ 1 (mod 2)

  可能: 1, 3, 5, 15

Sylow 3-子群: 阶3, n₃ | 20, n₃ ≡ 1 (mod 3)

  可能: 1, 4, 10

  可能: 1, 6

若 n₅ = 6, 则有 6×4 = 24 个5阶元

共44个元，剩余16个

重要推论:
• p²q 阶群可解
```

## 四、结构定理

### 有限生成Abel群结构定理

**定理**: 有限生成Abel群 $A$ 同构于：
$$A \cong \mathbb{Z}^r \oplus \mathbb{Z}_{d_1} \oplus \cdots \oplus \mathbb{Z}_{d_k}$$
其中 $d_1 | d_2 | \cdots | d_k$ (不变因子分解)

```mermaid
graph TB
    A[有限生成Abel群] --> D[分解定理]
    D --> F[自由部分 ℤʳ]
    D --> T[挠部分 ⊕ℤ_{dᵢ}]

    F --> R[秩 rank(A) = r]
    T --> I[不变因子 d₁|d₂|...|dₖ]

    T --> P[初等因子 pᵢ^{eᵢ}]

    R --> U[唯一性定理]
    I --> U

    U --> C[分类完成]

    style A fill:#e1e5ff
    style D fill:#e1f5e1
    style U fill:#ffe1e1

```

## 五、Jordan-Hölder定理

### 合成列的唯一性

**定理**: 群的任意两个合成列等价 (有相同的合成因子，可能排列不同)。

```mermaid
graph TD
    G[群G] --> C1[合成列1<br/>G = G₀ ⊳ G₁ ⊳ ... ⊳ Gₙ = {e}]
    G --> C2[合成列2<br/>G = H₀ ⊳ H₁ ⊳ ... ⊳ Hₘ = {e}]

    C1 --> F[合成因子<br/>Gᵢ/Gᵢ₊₁ 单群]
    C2 --> F

    F --> E[两个合成列等价]
    E --> U[n = m]
    E --> P[合成因子相同(重排)]

    style G fill:#e1e5ff
    style E fill:#ffe1e1

```

## 六、Hilbert基定理

### 诺特环的多项式环

**定理**: 若 $R$ 是Noether环，则 $R[x]$ 也是Noether环。

```mermaid
graph TB
    R[Noether环 R] --> P[R[x]]
    P --> I[R[x]的理想 I]
    I --> L[首项理想 lead(I)]
    L --> F[lead(I) ⊆ R 有限生成]
    F --> G[I 有限生成]
    G --> C[R[x] 是Noether环]

    C --> I2[推论: R[x₁,...,xₙ] Noether]
```

```mermaid
graph TB
```

## 八、Wedderburn定理

### 有限除环是域

**定理**: 有限除环必为域 (乘法交换)。

```mermaid
graph TD
    D[有限除环 D] --> C[中心 Z(D)]
    C --> F[有限域 F_q]

    D --> V[D 是 Z(D)上的向量空间]
    V --> N[|D| = qⁿ]

    N --> M[D* 乘法群]
    M --> E[类方程]
    E --> P[多项式 x^{qⁿ} - x = 0]
    P --> R[所有元与中心交换]
    R --> T[D = Z(D)]
    T --> CO[D 是域]

    style D fill:#e1e5ff
    style T fill:#ffe1e1
    style CO fill:#e1f5e1
```

## 十、Nakayama引理

### 局部环的生成元

**定理**: 设 $M$ 是局部环 $(R, \mathfrak{m})$ 上的有限生成模，若 $M = \mathfrak{m}M$，则 $M = 0$。

```mermaid
graph TB
    R[局部环 R, 极大理想 m] --> M[有限生成模 M]
    M --> A[M = mM]

    A --> N[Nakayama引理]
    N --> Z[M = 0]

    N --> C1[推论1: 生成元极小集大小 = dim_{R/m} M/mM]
    N --> C2[推论2: 若 M/mM → N/mN 满, 则 M → N 满]

    style R fill:#e1e5ff
    style A fill:#e1f5e1
    style Z fill:#ffe1e1

```
