## 一、Brouwer不动点定理

### 定理陈述

**定理**: 连续映射 $f: D^n \to D^n$ 必有不动点，即存在 $x$ 使 $f(x) = x$。

### 证明思路

```mermaid
graph TD
    F[f: Dⁿ→Dⁿ 连续] --> A[反设: f 无不动点]
    A --> R[构造收缩 r: Dⁿ→Sⁿ⁻¹]
    R --> I[r∘i = id_{Sⁿ⁻¹}}]
    I --> H[诱导同调同态]
    H --> C[i_*: H_{n-1}(Sⁿ⁻¹)→H_{n-1}(Dⁿ)]
    C --> Z[H_{n-1}(Sⁿ⁻¹) = ℤ, H_{n-1}(Dⁿ) = 0]
    Z --> E[id_* = 0, 矛盾]
    E --> R2[故 f 有不动点]

    style F fill:#e1e5ff
    style E fill:#ffe1e1
    style R2 fill:#e1f5e1

```

### 几何解释

```

Brouwer不动点定理的几何意义
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

n=1 (区间):

    f(a) ──────────→ f(b)
    a ──────────────── b

    若 f(a)>a 且 f(b)<b, 必有点满足 f(x)=x

n=2 (圆盘):

    搅拌一杯咖啡，
    总有一点回到原位！

    形式化: 连续变形圆盘到自身，
    必有一点不动

应用:
• 经济学: 市场均衡存在性
• 博弈论: Nash均衡存在性
• 微分方程: 解的存在性
• 代数: 代数基本定理证明
```

## 二、Gauss-Bonnet定理

### 定理陈述

**定理**: 紧致定向曲面 $M$：
$$\int_M K dA + \int_{\partial M} k_g ds + \sum(\pi - \alpha_i) = 2\pi \chi(M)$$

```mermaid
graph TB
    G[紧致定向曲面 M] --> K[Gauss曲率 K]
    G --> KG[测地曲率 k_g]
    G --> A[外角 αᵢ]

    K --> I[∫ K dA]
    KG --> I2[∫ k_g ds]
    A --> I3[Σ(π-αᵢ)]

    I --> E[Euler示性数 χ(M)]
    I2 --> E
    I3 --> E

    E --> R[右边 = 2πχ(M)]

    G --> C[闭曲面: ∫K dA = 2πχ(M)]

    style G fill:#e1e5ff
    style E fill:#e1f5e1
    style R fill:#ffe1e1

```

## 三、Stokes定理

### 定理陈述

**定理**: 设 $M$ 是有边定向 $n$ 维流形，$\omega$ 是 $(n-1)$-形式，则：
$$\int_M d\omega = \int_{\partial M} \omega$$

### 定理的统一性

```mermaid
graph TB
    S[Stokes定理 ∫_M dω = ∫_{∂M} ω] --> F[Fundamental定理]
    S --> G[Green定理]
    S --> D[Gauss散度定理]

    F --> N[n=1: ∫ₐᵇ f' = f(b)-f(a)]
    G --> N2[n=2: ∮Pdx+Qdy = ∬(∂Q/∂x-∂P/∂y)]
    D --> N3[n=3: ∯F·n dS = ∭∇·F dV]

    S --> C[微积分基本定理的高维推广]

    style S fill:#ffe1e1
    style C fill:#e1f5e1

```

### 几何意义

```

Stokes定理的几何意义
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

边界算子 ∂ 与外微分 d 的对偶:

    Ω^{k}(M) ──d──→ Ω^{k+1}(M)
       │                │
       │∫               │∫
       ↓                ↓
    C_k(M) ──∂──→ C_{k-1}(M)

    ∫_M dω = ∫_{∂M} ω

    即: 外微分在整体上的积分
       等于原形式在边界上的积分

直观理解:
• 区域内部的"旋度"总和 = 边界上的"环流"
• 场在区域内部的"源" = 通过边界的"通量"

应用:
• 电磁学 (Maxwell方程)
• 流体力学
• 微分几何
```

## 四、de Rham定理

### 定理陈述

**定理**: 光滑流形 $M$ 的de Rham上同调 $H_{dR}^*(M)$ 与奇异上同调 $H^*(M; \mathbb{R})$ 同构。

```mermaid
graph TB
    M[流形 M] --> DR[de Rham复形]
    M --> SI[奇异复形]

    DR --> C[闭形式 / 恰当形式]
    SI --> C2[上闭链 / 上边缘]

    C --> HDR[H_{dR}^*(M)]
    C2 --> HS[H^*(M;ℝ)]

    HDR --> I[同构]
    HS --> I

    I --> S[Stokes定理建立配对]

    style M fill:#e1e5ff
    style HDR fill:#e1f5e1
    style HS fill:#e1f5e1
    style I fill:#ffe1e1

```

## 五、Hodge分解定理

### 定理陈述

**定理**: 紧致定向Riemann流形上，每个微分形式可唯一分解：
$$\omega = d\alpha + \delta\beta + \gamma$$
其中 $\gamma$ 是调和形式 ($\Delta \gamma = 0$)。

```mermaid
graph TB
    M[紧致定向Riemann流形] --> F[微分形式 Ωᵏ(M)]
    F --> H[Hodge分解]

    H --> E[恰当形式 dα]
    H --> CO[余恰当形式 δβ]
    H --> HA[调和形式 γ, Δγ=0]

    HA --> ISO[Hᵏ_{dR}(M) ≅ ℋᵏ(M)]

    E --> O[正交分解]
    CO --> O
    HA --> O

    style M fill:#e1e5ff
    style H fill:#ffe1e1
    style ISO fill:#e1f5e1

```

## 六、Poincaré对偶

### 定理陈述

**定理**: 紧致定向 $n$ 维流形 $M$，$H^k(M) \cong H_{n-k}(M)$。

```mermaid
graph TB
    M[紧致定向n维流形] --> C[配边相交理论]

    C --> P[Poincaré对偶]
    P --> I[Hᵏ(M) ≅ H_{n-k}(M)]
    P --> PD[ caps with fundamental class [M]]

    I --> U[杯积配对 Hᵏ × Hⁿ⁻ᵏ → Hⁿ ≅ ℤ]

    I --> E[Euler示性数 χ = Σ(-1)ᵏbₖ]

    M --> K[Kunneth公式]
    K --> P2[H^*(M×N) ≅ H^*(M)⊗H^*(N)]

    style M fill:#e1e5ff
    style P fill:#ffe1e1
    style I fill:#e1f5e1

```

## 七、Atiyah-Singer指标定理

### 定理陈述

**定理**: 椭圆算子 $D$ 的解析指标等于拓扑指标：
$$\text{ind}(D) = \text{dim ker}D - \text{dim ker}D^* = \int_M \text{ch}(\sigma_D) \wedge \text{Td}(M)$$

```mermaid
graph TB
    D[椭圆算子 D] --> A[解析指标]
    D --> S[符号 σ_D]

    A --> I[ind(D) = dim ker D - dim ker D*]
    S --> T[拓扑指标]
    T --> F[示性类积分]

    I --> AT[Atiyah-Singer]
    F --> AT
    AT --> E[等式成立]

    D --> E2[例: Dirac算子, Dolbeault算子, 符号差算子]
    E2 --> G[Gauss-Bonnet, Hirzebruch-Riemann-Roch]

    style D fill:#e1e5ff
    style AT fill:#ffe1e1
    style E fill:#e1f5e1

```

## 八、Morse理论基本定理

### 定理陈述

**定理**: Morse函数的水平集在临界点处拓扑变化，附加指标维数的胞腔。

```mermaid
graph TB
    M[流形 M] --> F[Morse函数 f: M→ℝ]
    F --> C[临界点 p, Hessian非退化]
    C --> I[指标 ind(p)]

    I --> H[M_a = f⁻¹(-∞,a]]
    H --> T[拓扑变化]
    T --> A[M_{c+ε} ≃ M_{c-ε} ∪ e^{ind(p)}]

    A --> CW[CW复形结构]
    CW --> B[Betti数 bₖ ≤ Cₖ]
    B --> E[Euler示性数 χ = Σ(-1)ᵏCₖ]

    style M fill:#e1e5ff
    style T fill:#ffe1e1
    style E fill:#e1f5e1

```

## 九、Seifert-van Kampen定理

### 定理陈述

**定理**: $X = U \cup V$，$U, V$ 开，$U \cap V$ 道路连通，则：
$$\pi_1(X) = \pi_1(U) *_{\pi_1(U \cap V)} \pi_1(V)$$

```mermaid
graph TB
    X[X = U ∪ V] --> U[U 开]
    X --> V2[V 开]
    U --> I[U∩V 道路连通]
    V2 --> I

    U --> PU[π₁(U)]
    V2 --> PV[π₁(V)]
    I --> PI[π₁(U∩V)]

    PU --> FP[自由积 π₁(U) * π₁(V)]
    PV --> FP
    PI --> AM[合并关系]

    FP --> AM
    AM --> R[π₁(X) = π₁(U) *_{π₁(U∩V)} π₁(V)]

    style X fill:#e1e5ff
    style R fill:#ffe1e1

```

## 十、Poincaré引理

### 定理陈述

**定理**: 可缩开集上的闭形式是恰当的，$H_{dR}^k(U) = 0$ ($k > 0$)。

```mermaid
graph TB
    U[可缩开集 U ≃ {pt}] --> F[闭形式 ω, dω=0]
    F --> H[同伦算子]
    H --> I[d∘H + H∘d = id - i*∘π*]
    I --> P[Poincaré引理]
    P --> E[ω = d(Hω)]
    E --> R[ω 是恰当形式]

    U --> Z[H_{dR}^k(U) = 0, k>0]
    R --> Z

    style U fill:#e1e5ff
    style P fill:#ffe1e1
    style Z fill:#e1f5e1

```

### 几何意义

```

Poincaré引理的几何意义
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

局部与整体的差异:

闭形式: dω = 0
• 局部: 总是恰当的 (Poincaré引理)
• 整体: 可能不恰当 (有拓扑障碍)

可缩区域:

    U ≃ {pt}

    任何闭形式都可以写成 dη

反例 (有洞区域):

    ℝ²\{0} 上的形式:
    ω = (-y dx + x dy)/(x²+y²)

    dω = 0 (闭)
    但 ω ≠ dη (不恰当)
    因为 ∮_{S¹} ω = 2π ≠ 0

拓扑解释:
闭形式/恰当形式 = de Rham上同调
反映空间的"洞"的结构
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```
