# 非欧几何革命

## 历史背景

### 欧几里得第五公设

```

《几何原本》第五公设（平行公设）：

"如果一条直线与两条直线相交，
在同一侧的内角之和小于两直角，
则这两条直线在这一侧延长后必相交"

等价表述（普莱费尔公理）：
"过直线外一点，有且仅有一条直线与之平行"

```

### 两千年的困扰

```

第五公设的特殊性：
- 比其他公设复杂
- 不像"自明"
- 很多数学家试图证明它

证明策略：
假设第五公设不成立，导出矛盾
但导出的"矛盾"并不显然

```

## 先驱者的工作

### 萨凯里（1667-1733）

```

《免除所有污点的欧几里得》（1733年）

方法：
考虑四边形ABCD：
- ∠A = ∠B = 90°
- AD = BC

三种假设：
1. 锐角假设：∠C = ∠D < 90°
2. 直角假设：∠C = ∠D = 90°（欧几里得）
3. 钝角假设：∠C = ∠D > 90°

萨凯里试图证明锐角和钝角假设导致矛盾
钝角假设确实导出矛盾（在欧几里得框架中）
锐角假设导出奇怪但不矛盾的结论

他称锐角假设为"与直线的本质相悖"
但实际上放弃了非欧几何的发现

```

### 兰伯特（1728-1777）

```

《平行线理论》（1766年，1786年出版）

继续萨凯里的工作：
- 锐角假设下，三角形内角和 < 180°
- 面积与角亏（180°-内角和）成正比
- 存在绝对长度单位

发现与球面几何的类比：
- 球面：内角和 > 180°，角盈
- 双曲：内角和 < 180°，角亏

但认为双曲几何不能在现实中实现

```

### 勒让德（1752-1833）

```

试图证明第五公设

错误证明1：
从任意一点向直线作垂线，存在唯一性
→ 平行公设

错误：唯一性等价于平行公设

错误证明2：
三角形内角和不能小于180°

错误：论证中隐含了平行公设

```

### 高斯的未发表工作

**高斯（1777-1855）**

```

"反欧几里得几何"的独立发现者

证据：
- 与朋友的通信
- 未发表的笔记
- 对鲍耶的回应

高斯认识到：
- 非欧几何是逻辑自洽的
- 可能描述真实空间
- 欧几里得几何不是唯一的真理

但未发表：
担心"黄蜂的嗡嗡"（庸俗的批评）

高斯在1817年写信给奥尔伯斯：
"我越来越确信我们不能证明欧几里得几何的必然性"

```

## 非欧几何的创立

### 鲍耶·亚诺什（1802-1860）

**独立发现双曲几何**

```

匈牙利数学家

1823年写信给父亲：
"我发现了如此美妙的东西，让我感到震惊：
从虚无中创造了一个奇怪的新世界"

结果发表在父亲的书中（1832年）：
《为好学青年的数学原理附录》

主要内容：
- 绝对几何（与平行公设无关的部分）
- 双曲几何的展开
- 面积与角亏的关系

高斯的评价：
称赞结果，但说自己早就知道
这使鲍耶沮丧

```

**鲍耶的几何**

```

基本设定：
过直线外一点，有无穷多条直线与之不相交

主要结论：
1. 三角形内角和 < 180°
2. 不存在相似三角形（不全等则不同）
3. 存在绝对长度单位
4. 圆的周长与直径之比不是常数
5. 面积 = k²(π - A - B - C)

```

### 罗巴切夫斯基（1792-1856）

**系统的双曲几何**

```

俄罗斯喀山大学教授

1826年：在喀山大学宣读论文
1829年：《论几何基础》（俄文）
1840年：《平行线理论的几何研究》（德文）

比鲍耶更早发表，更系统

被同事嘲笑，被称为"几何学的哥白尼"

```

**罗巴切夫斯基几何**

```

基本公理：
过直线外一点，至少有两条直线与之平行

等价于：
存在三角形内角和 < 180°

主要定理：
1. 平行角：
   设点P到直线l的距离为d
   存在角度Π(d)，使得过P的直线
   - 与l相交，如果与垂线夹角 < Π(d)
   - 与l平行，如果夹角 = Π(d)
   
2. 平行角公式：
   tan(Π(d)/2) = e^(-d/k)
   k为常数

3. 三角形面积：
   S = k²(π - α - β - γ)

```

### 双曲几何模型

**贝尔特拉米模型（1868年）**

```

 Eugenio Beltrami（1835-1900）

伪球面模型：
- 常负高斯曲率的曲面
- 测地线扮演"直线"
- 实现了双曲几何的一部分

意义：
证明了双曲几何的无矛盾性
（相对于欧几里得几何）

```

**克莱因模型（1871年）**

```

 Felix Klein（1849-1925）

射影圆盘模型：
- 在单位圆内部
- "直线"是圆的弦
- 距离用交比定义

统一观点：
- 欧氏几何：射影几何+度量
- 双曲几何：射影几何+双曲度量
- 椭圆几何：射影几何+椭圆度量

```

**庞加莱模型**

```

 Henri Poincaré（1854-1912）

上半平面模型：
- 复平面的上半部分 Im(z) > 0
- "直线"是与实轴正交的半圆或射线
- 距离：ds² = (dx² + dy²)/y²

单位圆盘模型：
- 单位圆内部 |z| < 1

- "直线"是与单位圆正交的圆弧

庞加莱圆盘模型最美：
- 角度保持（共形）
- 图形美观
- 埃舍尔的作品《圆极限》基于此

```

## 椭圆几何

### 黎曼的洞察

**黎曼（1826-1866）**

```

《关于几何基础中的假设》（1854年）

突破性观点：
空间的几何由度量决定

g = ds² = Σ gᵢⱼ dxᵢ dxⱼ

三种常曲率空间：
- K > 0：椭圆几何
- K = 0：欧几里得几何  
- K < 0：双曲几何

```

**椭圆几何**

```

基本特征：
- 没有平行线（任何两直线都相交）
- 三角形内角和 > 180°
- 类似于球面几何

球面几何：
- "直线"是大圆
- 三角形内角和 = 180° + 面积/R²
- 没有平行线

椭圆几何（修正的球面几何）：
- 对径点视为同一点
- 射影平面上的几何

```

## 几何学的统一

### 克莱因的埃尔朗根纲领

**克莱因（1849-1925）**

```

《埃尔朗根纲领》（1872年）

核心思想：
几何学由变换群决定

定义：
给定集合S和作用于S的变换群G
几何学研究在G下不变的性质

例子：
- 欧氏几何：刚体运动群
- 射影几何：射影变换群
- 双曲几何：双曲运动群
- 椭圆几何：椭圆运动群

统一了看似不同的几何学

```

### 希尔伯特的公理化

**《几何基础》（1899年）**

```

大卫·希尔伯特（1862-1943）

公理化方法：
1. 点、线、面的未定义概念
2. 五组公理：
   - 关联公理
   - 顺序公理
   - 合同公理
   - 平行公理
   - 连续公理

3. 独立性证明：
   平行公理独立于其他公理

意义：
- 形式化了几何学
- 证明了几何的逻辑结构
- 为非欧几何提供了严格基础

```

## 影响与意义

### 对数学的影响

**公理化方法**

```

希尔伯特之后：
- 各分支的公理化
- 形式系统的研究
- 元数学的兴起

```

**几何学扩展**

```

- 微分几何：黎曼几何
- 拓扑学：连续变形
- 代数几何：代数曲线
- 辛几何：力学

```

### 对物理学的影响

**广义相对论（1915年）**

```

爱因斯坦：
"引力不是力，而是时空的弯曲"

数学基础：
- 黎曼几何
- 张量分析
- 四维时空

物理现实：
- 质量-能量弯曲时空
- 光线沿测地线传播
- 水星近日点进动
- 引力红移

非欧几何从"纯粹的思维构造"
成为描述宇宙的语言

```

**宇宙学**

```

宇宙的整体形状：
- 正曲率：封闭宇宙
- 零曲率：平坦宇宙
- 负曲率：开放宇宙

观测表明：
宇宙接近平坦
（总曲率接近零）

```

### 哲学意义

**康德先验论的颠覆**

```

康德认为：
欧几里得几何是"先天综合判断"
是人类认识能力的必然形式

非欧几何证明：
空间的几何性质是经验的
可以被科学检验

几何学从"先验真理"
变为"经验科学"

```

**数学真理的本质**

```

转变：
- 从"发现"真理
- 到"创造"结构

非欧几何显示：
- 公理是假设
- 定理是逻辑推论
- 数学是符号游戏？

希尔伯特：
"我们可以用桌子、椅子、啤酒杯
来代替点、线、面"

```

## 结语

非欧几何的发现是数学史上最深刻的革命之一。经过两千年的努力，数学家们终于认识到欧几里得几何不是唯一的真理，平行公设可以有不同的选择。这一发现不仅开创了几何学的新纪元，更改变了人类对空间、真理和数学本质的认识。从黎曼的微分几何到爱因斯坦的广义相对论，非欧几何的影响延续至今，展示了纯粹数学思维的惊人力量。
