# 分析的严格化

## 背景：18世纪分析的危机

### 直觉与矛盾的冲突

```

18世纪分析学的成就巨大但基础不牢

问题：
1. 无穷小量是什么？
   - 牛顿："最终的比"
   - 莱布尼茨："不可分量"
   - 贝克莱主教："消逝量的鬼魂"

2. 级数收敛性
   欧拉：1 - 1 + 1 - 1 + ... = 1/2
   当时认为所有级数都有"和"

3. 函数的连续性
   直观理解，缺乏精确定义

4. 导数和积分的存在性
   假设总是存在

```

### 贝克莱的批评

```

《分析学家》（1734年）

贝克莱攻击微积分的基础：

"这些流数是什么？
是消逝增量的速度。
这些消逝增量是什么？
既非有限量，也非无穷小量，
还不是无。
难道我们不能称它们为消逝量的鬼魂吗？"

虽然动机是神学的，
但批评击中要害

```

## 严格化的开始

### 达朗贝尔（1717-1783）

```

极限思想的先驱

《百科全书》"微分"条目：
"微分学的基础...在于极限概念"

观点：
- 导数是差商的极限
- 强调极限而非无穷小

但达朗贝尔没有给出极限的严格定义

```

### 拉格朗日（1736-1813）

```

《解析函数论》（1797年）

试图不用无穷小和极限建立分析：
"函数f(x)的导数是f(x+h)展开式中h的系数"

基于泰勒展开：
f(x+h) = f(x) + f'(x)h + f''(x)h²/2! + ...

问题：
- 假设函数总能展开
- 忽略了收敛性问题

但强调了代数化方法

```

### 傅里叶级数的冲击

```

傅里叶（1807年，1822年出版）
《热的解析理论》

任意函数可以展开为三角级数：
f(x) = a₀/2 + Σ(aₙcosnx + bₙsinnx)

这挑战了当时的函数概念：
- 不连续函数也可以有解析表示
- 三角级数的收敛性复杂

推动了对函数、连续性、收敛性的深入研究

```

## 柯西的严格化

### 柯西（1789-1857）

**数学分析的奠基人**

```

柯西的著作：
《分析教程》（1821年）
《无穷小计算教程》（1823年）
《微分计算教程》（1829年）

目标：
"使分析学脱离几何直观，回到代数运算"

```

**极限的严格定义**

```

柯西定义（1821年）：

"当属于一个变量的相继值无限趋近于
某个固定值时，
如果最终与该固定值的差要多小有多小，
那么这个固定值就称为所有其他值的极限"

用现代ε-语言：
∀ε > 0, ∃N, 当n > N时, |aₙ - L| < ε

柯西几乎达到了现代定义，
但缺少"任意"（∀）和"存在"（∃）的明确使用

```

**连续性定义**

```

柯西定义：

"如果f(x)的无穷小增量导致
f(x)本身产生无穷小增量，
则称f(x)是连续的"

现代表述：
∀ε > 0, ∃δ > 0, 

|x - a| < δ ⇒ |f(x) - f(a)| < ε

柯西的定义本质上是正确的，
但"无穷小"的表述仍有歧义

```

**收敛准则**

```

柯西收敛准则：

数列{aₙ}收敛
⇔ 
∀ε > 0, ∃N, 
当m,n > N时, |aₙ - aₘ| < ε

这个准则不依赖极限值本身
可以判断收敛性

柯西认为这个条件是充分的
但证明需要实数的完备性

```

**导数和积分**

```

导数：
f'(x) = lim[h→0] (f(x+h)-f(x))/h

定积分：
∫[a,b] f(x)dx = lim Σ f(ξᵢ)(xᵢ - xᵢ₋₁)

柯西将积分定义为和的极限
而非反导数

微积分基本定理：
若F'(x) = f(x)，则∫[a,b] f(x)dx = F(b) - F(a)

```

## 实数理论的建立

### 问题的提出

```

柯西的工作仍有漏洞：

1. 实数是什么？
   柯西用有理数序列的极限定义实数
   但这会循环定义！

2. 完备性公理
   柯西收敛准则的充分性需要证明
   这需要实数的完备性

3. 存在性证明
   上确界存在吗？
   最大下界存在吗？

需要严格建立实数理论

```

### 戴德金分割（1872年）

**理查德·戴德金（1831-1916）**

```

《连续性与无理数》（1872年）

核心思想：
用有理数的分割定义实数

戴德金分割：
将有理数集Q分成两部分A和B，满足：
1. A ∪ B = Q, A ∩ B = ∅
2. A中每个元素小于B中每个元素
3. A无最大元素

分割（A,B）定义一个实数：
- 若B有最小元素r，则对应有理数r
- 若B无最小元素，则对应无理数

例：√2对应的分割
A = {r∈Q | r≤0 或 r²<2}
B = {r∈Q | r>0 且 r²>2}

```

**完备性**

```

戴德金定理：
实数集的任何分割都对应一个实数

这保证了实数的完备性：
- 任何有上界的集合有上确界
- 柯西序列收敛
- 区间套定理成立

```

### 康托尔的基本序列（1872年）

**格奥尔格·康托尔（1845-1918）**

```

用柯西序列定义实数

基本序列（柯西序列）：
有理数序列{aₙ}，满足
∀ε>0, ∃N, m,n>N ⇒ |aₙ - aₘ| < ε

实数定义：
基本序列的等价类
两个序列等价如果它们的差趋于0

例：√2 = [1, 1.4, 1.41, 1.414, ...]

```

### 魏尔斯特拉斯的贡献

**卡尔·魏尔斯特拉斯（1815-1897）**

```

"分析学的奠基人"

魏尔斯特拉斯的方法：
- 用递增有界序列定义实数
- 完全算术化的分析

名言：
"一个变量趋于一个极限"
必须改为
"如果存在一个数N，使得..."

完全避免无穷小和"运动"的语言

```

**魏尔斯特拉斯的例子**

```

处处连续但处处不可微的函数：

f(x) = Σ[n=0→∞] bⁿcos(aⁿπx)

其中a是奇数，0<b<1，ab>1+3π/2

这震惊了数学界：
直观上认为连续函数"大部分"可微
魏尔斯特拉斯证明这是错误的

促进了对函数概念的严格研究

```

## 分析的进一步发展

### 一致收敛

**问题的发现**

```

连续函数序列的极限是否连续？

例：fₙ(x) = xⁿ 在[0,1]上
每个fₙ都连续
但极限函数f(x) = {0, x<1; 1, x=1}不连续

问题在哪里？
收敛不是"一致"的

```

**魏尔斯特拉斯的定义**

```

一致收敛：
∀ε>0, ∃N, 当n>N时, 
对所有x，|fₙ(x) - f(x)| < ε

关键：N不依赖于x

定理：
一致收敛的连续函数序列的极限连续
一致收敛的可微函数序列的极限可微
（在一定条件下）

```

### 一致连续性

```

普通连续：
对每个x，给定ε，存在δ(x)

一致连续：
给定ε，存在δ，对所有x都适用

例：
f(x) = 1/x 在(0,1)上连续但非一致连续
f(x) = x² 在R上连续但非一致连续
f(x) = x² 在[0,1]上一致连续

紧集上的连续函数必一致连续

```

### 黎曼积分

**伯恩哈德·黎曼（1826-1866）**

```

《用三角级数表示函数的可能性》（1854年）

黎曼和：
∫[a,b] f(x)dx = lim Σ f(ξᵢ)Δxᵢ

黎曼可积的判别条件：
函数的不连续点集的测度为零

推广了柯西的积分概念

```

### 集合论与测度论

**康托尔的集合论（1870年代起）**

```

《关于无穷线性点集》（1872年起）

- 建立了集合论基础
- 研究了无穷集合
- 证明了实数不可数
- 建立了超限数理论

为测度论和勒贝格积分奠定基础

```

**勒贝格积分（1902年）**

```

亨利·勒贝格（1875-1941）
《积分、长度与面积》

核心思想：
划分值域而非定义域

优点：
- 更广泛的函数可积
- 积分与极限可交换条件更弱
- 建立了现代实变函数论

黎曼可积 ⇒ 勒贝格可积
反之不成立

```

## 严格化的意义

### 数学内部

**消除矛盾**

```

- 无穷小概念的澄清
- 级数收敛性的严格判别
- 函数概念的确立
- 积分的严格定义

```

**新的发展**

```

严格化催生了新领域：
- 实变函数论
- 泛函分析
- 点集拓扑学
- 抽象测度论

```

### 对科学的影响

**物理学的数学基础**

```

严格分析为物理学提供了：
- 严格的数学语言
- 存在性定理（保证解的存在）
- 逼近方法的理论基础

```

**计算数学**

```

数值分析的理论基础：
- 收敛性证明
- 误差估计
- 算法稳定性

```

## 结语

19世纪分析的严格化是数学史上最重要的发展之一。从柯西的极限概念到魏尔斯特拉斯的ε-δ语言，从戴德金和康托尔的实数理论到勒贝格的测度论，数学家们建立了分析的坚实基础。这一过程不仅消除了悖论和混乱，更催生了实变函数论、泛函分析、拓扑学等新领域，为20世纪数学的繁荣奠定了根基。
