# 群论的诞生

## 历史背景

### 代数方程求解的困境
```
16世纪解决了三次、四次方程

问题：五次及以上方程能否用根式求解？
即：解能否用系数通过有限次加减乘除和开方表示？

欧拉、拉格朗日、鲁菲尼尝试解决
但没有成功
```

### 拉格朗日的洞察（1770年）
```
《关于代数方程解的思考》

关键思想：
方程的解与根的排列有关

例：二次方程 x² + px + q = 0
根：x₁, x₂

对称多项式：
x₁ + x₂ = -p
x₁x₂ = q

拉格朗日发现：
解方程等价于寻找根的有理函数
使其在根的某些排列下保持不变

这启发了置换群的研究
```

## 阿贝尔的工作

### 尼尔斯·阿贝尔（1802-1829）

**五次方程不可解**
```
1824年：自费出版小册子
1826年：详细论文发表

证明：
一般五次方程（及以上）
不能用根式求解

方法：
研究根式求解的可能性条件
分析方程根的置换性质

例：x⁵ - x + 1 = 0
不能用根式求解
```

**阿贝尔的工作特点**
```
- 严格、清晰
- 发展了无穷级数理论
- 椭圆函数理论
- 交换群（阿贝尔群）的概念

阿贝尔群：
运算满足交换律的群
即：ab = ba

悲剧：
- 贫困中去世，年仅26岁
- 死后才获得认可
- 柏林大学聘书寄到时已去世
```

## 伽罗瓦的革命

### 埃瓦里斯特·伽罗瓦（1811-1832）

**短暂而辉煌的一生**
```
1811年：生于巴黎附近
1829年：第一次提交论文（被柯西遗失）
1830年：第二次提交（被傅里叶遗失）
1831年：入狱
1832年：决斗身亡，年仅20岁

决斗前夜：
写下群论思想，托付给夏瓦利埃
```

**群论的诞生**
```
1830年论文（1846年由刘维尔发表）：
《关于方程根式可解性的条件》

核心思想：
1. 每个方程对应一个群（伽罗瓦群）
2. 方程可根式求解
   ⇔ 
   伽罗瓦群可解

"群"（groupe）一词的引入
```

### 伽罗瓦理论

**群的概念**
```
伽罗瓦定义的群：
- 置换的集合
- 包含恒等置换
- 对复合运算封闭
- 包含每个置换的逆

例：三次方程的伽罗瓦群
根：x₁, x₂, x₃
群S₃包含6个置换
```

**可解群**
```
群G可解：
存在子群列
G = G₀ ⊃ G₁ ⊃ G₂ ⊃ ... ⊃ Gₙ = {e}
使得每个Gᵢ₊₁是Gᵢ的正规子群
且商群Gᵢ/Gᵢ₊₁是阿贝尔群

五次方程的伽罗瓦群S₅不可解
因此一般五次方程不可根式解
```

**具体例子**
```
方程 x⁵ - x + 1 = 0

伽罗瓦群 = S₅（5次对称群）

S₅的正规子群只有：
S₅ ⊃ A₅ ⊃ {e}

A₅是单群（没有非平凡正规子群）
A₅不是阿贝尔群

因此S₅不可解
该方程不可根式求解
```

### 伽罗瓦的遗产

**迟来的认可**
```
1846年：刘维尔发表伽罗瓦论文
1843年：柯西发展置换群理论
1870年：若尔当《论置换与代数方程》
系统阐述伽罗瓦理论
```

**超越方程论**
```
伽罗瓦群概念的应用：
- 代数数论
- 代数几何
- 拓扑学（基本群）
- 微分方程
- 物理学（晶体学、量子力学）
```

## 群论的发展

### 置换群理论

**柯西（1789-1857）**
```
1840年代系统研究置换群

贡献：
- 置换的记号
- 阶的概念
- 子群的研究
- 拉格朗日定理：子群阶整除群阶
```

**若尔当（1838-1922）**
```
《论置换与代数方程》（1870年）

- 系统阐述伽罗瓦理论
- 研究线性群
- 若尔当-赫尔德定理
```

### 抽象群论

**从置换到抽象**
```
凯莱（Cayley，1821-1895）：

1854年：首次定义抽象群
"群是由符号和运算规则定义的"

凯莱表：
群的乘法表表示

但凯莱的工作当时未受重视
```

**弗罗贝尼乌斯和施密特**
```
1890年代：
- 群表示论的发展
- 特征标理论
- 有限群的分类开始
```

**群的公理化**
```
20世纪初：
- 韦伯（Weber）
- 亨廷顿（Huntington）
- 莫尔（Moore）

群的公理：
1. 封闭性：a,b∈G ⇒ ab∈G
2. 结合律：(ab)c = a(bc)
3. 单位元：存在e，ae = ea = a
4. 逆元：对a存在a⁻¹，aa⁻¹ = e
```

### 李群与李代数

**索菲斯·李（1842-1899）**
```
挪威数学家

1870年代研究：
- 微分方程的对称性
- 连续变换群

李群：
既是群又是流形的数学对象
群运算可微

例：
- 旋转群SO(3)
- 一般线性群GL(n,R)
- 洛伦兹群

应用：
- 微分方程
- 物理学
- 几何学
```

**李代数**
```
李群在单位元处的切空间
配备李括号运算

李群 ↔ 李代数
（局部对应关系）

例：
SO(3)的李代数是三维向量空间
李括号是叉积
```

### 有限单群分类

**20世纪最宏大的数学工程**
```
有限单群：群的"原子"
任何有限群都由单群构成

分类定理（1981年完成）：
任何有限单群属于以下之一：
1. 素数阶循环群
2. 交错群Aₙ（n≥5）
3. 李型单群（16个无穷族）
4. 26个散在单群

散在单群：
- 最小的：M₁₁（阶7920）
- 最大的：魔群（Monster）
  阶 ≈ 8×10⁵³
```

## 结语

群论的诞生是19世纪代数学最伟大的成就。从求解方程的具体问题出发，伽罗瓦创造了群的概念，揭示了代数方程与群结构之间的深刻联系。群论不仅解决了方程可解性的经典问题，更发展成为现代数学的核心分支，渗透到几乎所有数学领域以及物理学、化学、计算机科学等学科。群的概念——研究对称性的数学工具——已成为人类理解自然的基本语言。
