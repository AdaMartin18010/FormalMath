# 集合论的创立

## 历史背景

### 分析的严格化需求

```

19世纪分析严格化过程中：

- 需要处理函数的间断点集
- 需要定义积分的范围
- 需要处理无穷集合

康托尔在研究三角级数唯一性时
开始系统研究无穷集合

```

### 三角级数问题

```

问题：
若三角级数Σ(aₙcosnx + bₙsinnx)收敛于0，
是否所有系数都为0？

海涅、康托尔的研究：
- 收敛点集的性质
- 例外点集的构造
- 导致对实数集合的深入研究

```

## 康托尔的革命

### 格奥尔格·康托尔（1845-1918）

**生平**

```

1845年：生于圣彼得堡
1856年：移居德国
1862年：入苏黎世大学
1863年：转入柏林大学
1869年：任哈雷大学教授
1874年：发表第一篇集合论论文
1884年：第一次精神崩溃
1897年：创立德国数学家协会
1918年：卒于哈雷

```

**主要著作**

```

《关于无穷线性点集》（6篇论文，1872-1884）
《超限数理论基础》（1895-1897）

```

### 无穷的概念革命

**伽利略的悖论**

```

伽利略《两门新科学》（1638年）：

"自然数与完全平方数可以一一对应：
n ↔ n²

但完全平方数是自然数的一部分！
部分怎么能等于整体？"

伽利略放弃了对无穷的比较
认为无穷是不可理解的

```

**康托尔的突破**

```

康托尔接受"部分等于整体"作为无穷的特征：

定义：
两个集合有相同的基数（势），
如果它们的元素可以建立一一对应

结论：
自然数集与完全平方数集等势
（尽管后者是前者的真子集）

这是无穷集合的本质特征！

```

### 可数集

**定义**

```

可数集（可列集）：
能与自然数集建立一一对应的集合

例：
- 自然数集ℕ
- 整数集ℤ
- 有理数集ℚ
- 代数数集

```

**有理数可数**

```

康托尔的证明：

按对角线枚举：
    1   2   3   4   ...
1  1/1 1/2 1/3 1/4  ...
2  2/1 2/2 2/3 2/4  ...
3  3/1 3/2 3/3 3/4  ...
4  4/1 4/2 4/3 4/4  ...
...  ...  ...  ...  ...

沿对角线遍历，跳过重复
得到有理数的枚举

因此ℚ是可数集

```

**代数数可数**

```

代数数：整系数多项式的根

证明思路：
1. 整系数多项式可数
   （按次数和系数大小枚举）
   
2. 每个多项式有有限个根

3. 可数个有限集的并是可数的

因此代数数集可数

```

### 实数不可数

**康托尔对角线论证（1891年）**

```

定理：[0,1]区间不可数

证明（反证法）：
假设[0,1]可数，
则所有实数可列成序列：
r₁ = 0.d₁₁d₁₂d₁₃...
r₂ = 0.d₂₁d₂₂d₂₃...
r₃ = 0.d₃₁d₃₂d₃₃...
...

构造新数r = 0.e₁e₂e₃...
其中eᵢ ≠ dᵢᵢ（例如eᵢ = (dᵢᵢ+1) mod 10）

则r与每个rᵢ都不同（第i位不同）
所以r不在列表中

矛盾！因此[0,1]不可数

```

**第一次对角线论证（1874年）**

```

更早期的证明：

利用区间套定理
1. 假设[0,1] = {r₁, r₂, r₃, ...}
2. 取不包含r₁的子区间I₁
3. 取I₁中不包含r₂的子区间I₂
4. 继续下去，得到区间套I₁ ⊃ I₂ ⊃ ...
5. 由区间套定理，存在点在所有Iₙ中
6. 该点不等于任何rₙ

矛盾！

```

### 超限数

**基数（势）**

```

ℵ₀（阿列夫零）：自然数集的基数
                    可数无穷
                    
c（连续统）：实数集的基数
             连续统的势
             
关系：
ℵ₀ < c

```

**基数运算**

```

对于无穷基数：

ℵ₀ + ℵ₀ = ℵ₀
ℵ₀ × ℵ₀ = ℵ₀
ℵ₀ⁿ = ℵ₀（n有限）

但：
2^ℵ₀ = c

康托尔证明：
对任何集合A，|A| < |P(A)|

（P(A)是A的幂集）

因此：
ℵ₀ < 2^ℵ₀ < 2^(2^ℵ₀) < ...

```

**超限序数**

```

良序集的序型：

有限：0, 1, 2, 3, ...

超限：
ω：自然数集的序型
ω+1, ω+2, ...
ω·2, ω·3, ...
ω², ω³, ...
ω^ω, ...
ε₀（满足ω^ε₀ = ε₀的最小序数）

超限归纳：
可以推广到任意良序集

```

### 连续统假设

**康托尔猜想**

```

连续统假设（CH）：
不存在集合S，使得ℵ₀ < |S| < c

等价表述：
c = ℵ₁
（ℵ₁是大于ℵ₀的最小基数）

广义连续统假设（GCH）：
对任意无穷基数κ，
2^κ = κ⁺
（κ⁺是大于κ的最小基数）

```

**后续发展**

```

1940年：哥德尔证明
CH与ZFC公理系统相容

1963年：科恩证明
CH的否定也与ZFC相容

结论：
CH独立于ZFC
既不能证明也不能否定

这是哥德尔不完备定理的重要实例

```

## 集合论的公理化

### 悖论的出现

**布拉利-福尔蒂悖论（1897年）**

```

考虑所有序数的集合Ω

Ω本身是一个良序集
所以Ω有一个序数α

但Ω包含所有序数
所以α ∈ Ω

作为序数，α < α+1 ≤ α
矛盾！

```

**罗素悖论（1902年）**

```

考虑集合R = {x | x ∉ x}

问：R ∈ R 吗？

如果R ∈ R，则根据定义R ∉ R
如果R ∉ R，则根据定义R ∈ R

矛盾！

通俗表述：
"所有不包含自身的集合的集合"

理发师悖论：
"给且只给不给自己理发的人理发的理发师"

```

**康托尔悖论**

```

考虑所有集合的集合V

则V应比P(V)小
但P(V)又是V的子集
（因为V包含所有集合）

矛盾！

```

### ZFC公理系统

**策梅洛（1908年）**

```

恩斯特·策梅洛提出公理化集合论

避免悖论的方法：
- 限制概括公理
- 不允许"所有集合的集合"

```

**ZFC公理**

```

1. 外延公理：集合由元素决定
2. 空集公理：存在空集∅
3. 配对公理：{a,b}是集合
4. 并集公理：∪A是集合
5. 幂集公理：P(A)是集合
6. 无穷公理：存在无穷集
7. 分离公理：{x∈A | P(x)}是集合

8. 替换公理：函数像保持集合性
9. 正则公理：禁止x∈x
10. 选择公理（AC）：任意集族有选择函数

```

**选择公理的争议**

```

选择公理：
对任何非空集合的集合，
存在选择函数

等价命题：
- 良序定理：任何集合可良序
- 佐恩引理
- 豪斯多夫极大原理

应用：
- 巴拿赫-塔斯基悖论
- 不可测集的存在
- 线性基的存在

争论：
直觉主义者反对
大多数数学家接受

```

```
第0层：个体

```

## 集合论的影响

### 数学基础

**数学的集合论化**

```

20世纪数学：
所有数学对象都可定义为集合

自然数：
0 = ∅
1 = {∅}
2 = {∅, {∅}}
...

实数：
戴德金分割或柯西序列

函数：
有序对的集合

一切都是集合！

```

**公理化运动**

```

希尔伯特的公理化纲领：
各数学分支的公理化

- 几何（1899年）
- 算术（佩亚诺，1889年）
- 集合论（策梅洛，1908年）

```

```
这对希尔伯特纲领是重大打击
也显示了形式系统的局限性
```

## 结语

集合论是数学史上最革命性的创造之一。康托尔敢于面对无穷，建立了超限数的理论，开辟了数学的新大陆。尽管经历了悖论的冲击，集合论通过公理化获得了坚实基础，成为现代数学的基础语言。从连续统假设的独立性到哥德尔的不完备定理，集合论深刻揭示了数学的局限性，也展示了抽象思维的惊人力量。
