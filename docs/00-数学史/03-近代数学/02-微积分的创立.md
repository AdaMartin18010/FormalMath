# 微积分的创立

## 历史背景

### 科学需求
17世纪科学革命带来了大量需要解决的数学问题：
1. **天文学**：行星轨道计算、开普勒定律的数学基础
2. **力学**：瞬时速度、加速度、曲线运动
3. **光学**：切线问题、曲率
4. **几何**：曲线长度、曲线下面积、体积

### 关键问题

```

四大核心问题：

1. 切线问题
   给定曲线，求某点处的切线
   → 微分学的核心

2. 瞬时变化率
   求物体在某一瞬时的速度
   → 导数的物理意义

3. 极值问题
   求函数的最大值、最小值
   → 费马方法

4. 求积问题
   求曲线下的面积、曲线长度、体积
   → 积分学的核心

```

## 先驱者的工作

### 开普勒（1571-1630）

```

《酒桶新立体几何》（1615年）

问题：奥地利酒商的酒桶容量

方法：
- 将曲线图形分割为无穷多小切片
- 用几何方法求和

结果：
- 引进了无穷小概念
- 为积分学铺路

```

### 卡瓦列里（1598-1647）

```

《不可分量几何学》（1635年）

卡瓦列里原理：
"两个立体如果在等高处截面积相等，
则体积相等"

这与中国祖暅原理完全相同！

不可分量法：
- 面积由无穷多条线段（不可分量）组成
- 体积由无穷多个平面组成

虽然缺乏严格性，但计算有效

```

### 费马（1601-1665）

```

求极值的方法：

问题：求f(x)的极大值或极小值

费马的算法：
1. 用x+e代替x
2. 令f(x+e) ≈ f(x)
3. 除以e
4. 令e = 0，解方程

这实际上是令f'(x) = 0！

例：求x²(a-x)的极大值
费马得到x = 2a/3（正确）

```

### 沃利斯（1616-1703）

```

《无穷算术》（1655年）

将卡瓦列里的几何方法算术化

重要结果：
∫₀¹ xⁿ dx = 1/(n+1)

对n=1/2的猜测：
通过插值得到
∫₀¹ x^(1/2) dx = 2/3

这启发了牛顿的研究

```

### 巴罗（1630-1677）

```

牛顿的老师
《几何讲义》（1670年）

重要发现：
切线问题与求积问题的互逆关系

基本定理的雏形：
如果曲线下的面积函数是A(x)
则曲线的方程是y = A'(x)

巴罗几乎发现了微积分基本定理！

```

### 流数术（Method of Fluxions）

**基本概念**

```

牛顿的术语：
- 流量（Fluent）：变化的量，相当于函数
- 流数（Fluxion）：流量的变化率，相当于导数

记号：
- x的流量记为x
- x的流数记为ẋ（点号表示）

例：
如果y = xⁿ
则ẏ = n xⁿ⁻¹ ẋ

```

**微积分基本定理**

```

牛顿认识到：

1. 微分与积分是互逆运算
   如果A(x) = ∫₀ˣ f(t) dt
   则 dA/dx = f(x)

2. 切线问题与求积问题的统一
   求曲线下的面积可以通过求反导数得到

这是微积分的核心！

```

**无穷级数**

```

牛顿广泛使用无穷级数：

二项式定理的推广：
(1+x)^α = 1 + αx + α(α-1)x²/2! + ...
对任意实数α成立

应用：
- 计算π
- 计算对数
- 函数展开

```

### 《自然哲学的数学原理》（1687年）

**数学与物理的统一**

```

《原理》的结构：

定义：质量、动量、惯性、力

公理（牛顿运动定律）：
1. 惯性定律
2. F = ma
3. 作用力与反作用力

数学推论：
- 使用流数术推导
- 证明了开普勒定律
- 万有引力定律

这是用数学方法研究自然的典范

```

**流数术的应用**

```

牛顿用流数术：

1. 证明行星轨道是椭圆
2. 计算彗星轨道
3. 研究潮汐
4. 计算地球形状

展示了微积分的强大威力

```

### 微分与积分的创立

**关键年份：1675-1676**

```

莱布尼茨的关键发现：

1. 积分符号∫
   来自拉丁文"summa"（和）的首字母s的拉长

2. 微分符号d
   表示"differentia"（差）

3. 基本关系：
   ∫dy = y
   d∫y = y

4. 微分规则：
   d(xy) = x dy + y dx
   d(x/y) = (y dx - x dy)/y²

```

**符号系统**

```

莱布尼茨符号的优点：

导数：dy/dx
- 明确表示变化率
- 可视为分数运算
- 链式法则自然：dy/dx = (dy/du)(du/dx)

积分：∫f(x)dx
- 明确表示面积元
- 积分限：∫ₐᵇf(x)dx

这些符号一直使用到今天！

```

**微分规则**

```

莱布尼茨给出的规则：

幂函数：d(xⁿ) = n xⁿ⁻¹ dx

和差：d(u±v) = du ± dv

乘积：d(uv) = u dv + v du

商：d(u/v) = (v du - u dv)/v²

链式法则：d(f(g(x))) = f'(g(x))g'(x)dx

```


## 优先权之争

### 争议的起因

```

时间线：

1665-1666年：牛顿发明流数术（未发表）
1675-1676年：莱布尼茨发明微积分（未发表）
1676年：牛顿给莱布尼茨的两封信
1684年：莱布尼茨发表论文
1687年：牛顿出版《原理》
1704年：牛顿首次发表流数术

争议焦点：
- 莱布尼茨是否独立发明？
- 是否受到牛顿的影响？
- 谁的贡献更大？

```

### 调查与结论

```

1712年：皇家学会成立调查委员会
1713年：报告认为牛顿是发明者

现代观点：
- 两人独立发明
- 牛顿更早，但莱布尼茨先发表
- 莱布尼茨的符号更优越
- 两人都对微积分有重大贡献

悲剧：
争论破坏了两位伟大数学家的关系
阻碍了英国数学的发展（英国人坚持用牛顿符号）

```

## 微积分的完善

### 18世纪的进展

**洛必达（1661-1704）**

```

《用于理解曲线的无穷小分析》（1696年）
第一本微积分教科书

洛必达法则：
lim(f(x)/g(x)) = lim(f'(x)/g'(x))
当f,g都趋于0或∞时

```

**泰勒（1685-1731）**

```

泰勒展开（1715年）：
f(x) = f(a) + f'(a)(x-a) + f''(a)(x-a)²/2! + ...

将函数展开为幂级数
统一了函数表示

```

**麦克劳林（1698-1746）**

```

麦克劳林展开：
f(x) = f(0) + f'(0)x + f''(0)x²/2! + ...

泰勒展开的特例（a=0）

```

**欧拉（1707-1783）**

```

《无穷分析引论》（1748年）

贡献：
- 函数概念的明确
- e, π, i 的关系：e^(iπ) + 1 = 0
- 微积分的系统化和推广
- 变分法

欧拉使微积分成为系统的学科

```

## 结语

微积分的创立是人类智力最伟大的成就之一。牛顿和莱布尼茨各自独立地发现了这一强大的数学工具，统一了切线问题和求积问题，建立了微分与积分的互逆关系。微积分不仅解决了古希腊人无法解决的众多问题，更为近代科学和技术的发展提供了数学基础。从行星运动到工程计算，从经济学到生物学，微积分无处不在，其影响延续至今。
