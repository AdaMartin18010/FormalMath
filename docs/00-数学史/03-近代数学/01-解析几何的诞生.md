# 解析几何的诞生

## 历史背景

### 时代背景
17世纪初，欧洲正经历科学革命。天文学的新发现（伽利略望远镜观测）、物理学的新理论（伽利略落体定律）需要新的数学工具。代数方法的成熟和几何学的发展，为两者的结合创造了条件。

### 双重发明
- **笛卡尔**（1596-1650）：从几何问题出发，用代数方法解决
- **费马**（1601-1665）：从代数方程出发，研究其几何图形

## 笛卡尔的贡献

### 生平简介
```
勒内·笛卡尔（René Descartes，1596-1650）

1596年：生于法国拉海耶
1604-1612年：拉弗莱什耶稣会学校
1616年：获法学学位
1618-1628年：游历欧洲，参军服役
1628年：定居荷兰
1637年：出版《方法论》及附录
1649年：赴瑞典宫廷
1650年：卒于斯德哥尔摩
```

### 《几何学》（1637年）
《几何学》是《方法论》的三个附录之一，另外两个是《折光学》和《气象学》。

**核心思想**
```
笛卡尔的基本思想：

1. 任何问题都可以转化为数学问题
2. 任何数学问题都可以转化为代数问题
3. 任何代数问题都可以化为解方程

几何问题的代数化：
- 用坐标表示点
- 用方程表示曲线
- 用代数运算解决几何问题
```

### 坐标系的建立

**基本设定**
```
给定一个点O（原点）
两条互相垂直的直线（坐标轴）

点P的位置由两个数(x, y)确定：
- x：P到y轴的距离
- y：P到x轴的距离

这建立了平面点与实数对的一一对应
```

**距离公式**
```
两点P₁(x₁, y₁)和P₂(x₂, y₂)之间的距离：

d = √[(x₂-x₁)² + (y₂-y₁)²]

这是从毕达哥拉斯定理直接得到的
```

### 曲线的方程表示

**直线方程**
```
一般形式：ax + by + c = 0

斜截式：y = kx + b
- k：斜率
- b：y轴截距

点斜式：y - y₀ = k(x - x₀)
过点(x₀, y₀)，斜率为k
```

**圆的方程**
```
圆心(a, b)，半径r：
(x-a)² + (y-b)² = r²

展开：x² + y² - 2ax - 2by + (a²+b²-r²) = 0
```

**圆锥曲线**
```
标准方程：

抛物线：y² = 4ax 或 x² = 4ay
椭圆：x²/a² + y²/b² = 1
双曲线：x²/a² - y²/b² = 1

阿波罗尼奥斯的成果现在可以用代数统一处理
```

### 笛卡尔的创新方法

**几何作图问题**
```
问题：给定单位长度，作出两线段的乘积

笛卡尔的方法：
设OA = 1，OB = a，OC = b
作直线l过A，与OA成任意角
在l上取点D使AD = b
连接BD，过C作BD的平行线交l于E

则 AE = a × b
（由相似三角形）

这给出了几何作图乘法的代数意义
```

**高次曲线的研究**
```
笛卡尔研究了三类曲线：

1. 几何曲线（可用代数方程表示）
   - 多项式方程定义
   - 包括圆锥曲线

2. 机械曲线（超越曲线）
   - 如摆线、对数曲线
   - 当时无法用代数方程表示

3. 根曲线
   - 与解方程有关的曲线
```

### 笛卡尔的影响

1. **统一代数和几何**
   - 几何问题可以用代数方法解决
   - 代数方程可以可视化

2. **变量概念**
   - x, y作为变量
   - 函数概念的萌芽

3. **坐标方法的建立**
   - 直角坐标系
   - 为微积分奠定基础


### 费马的解析几何

**研究路径**
```
与笛卡尔相反，费马从代数出发：

给定方程 F(x,y) = 0
研究其几何图形

《平面与立体轨迹引论》（1629年写，1679年出版）
```

**直线方程**
```
费马的直线方程：

D in A e → N in B
（现代：y = kx + b）

其中：
- D, N 是变量
- A, e, B 是常数
```

**圆锥曲线统一处理**
```
费马证明了阿波罗尼奥斯的圆锥曲线
都可以用二次方程表示：

Ax² + Bxy + Cy² + Dx + Ey + F = 0

根据判别式 Δ = B² - 4AC 分类：
- Δ < 0：椭圆（包括圆）
- Δ = 0：抛物线  
- Δ > 0：双曲线
```

### 费马大定理

**著名批注**
```
在丢番图《算术》的页边，费马写道：

"不可能将一个立方数写成两个立方数之和，
或者将一个四次幂写成两个四次幂之和，
或者一般地，将任何高于二次的幂
写成两个同次幂的和。

我发现了一个真正美妙的证明，
但这里的空白太小，写不下。"

即：对于n > 2，方程 xⁿ + yⁿ = zⁿ
没有正整数解。
```

**证明历程**
```
费马证明了n=4的情况
欧拉证明了n=3的情况
库默尔证明了大量情况
怀尔斯最终证明（1994年）

历时350多年！
```

## 解析几何的意义

### 数学上的革命

1. **几何代数化**
   ```
   几何问题 → 代数方程 → 代数运算 → 几何解释
   
   例：证明三条高交于一点
   可用坐标法计算验证
   ```

2. **代数可视化**
   ```
   代数方程 → 几何曲线 → 直观理解
   
   例：方程 x² + y² = 1
   立即知道是单位圆
   ```

3. **新的曲线研究**
   ```
   可以用方程定义新曲线：
   - 笛卡尔叶形线
   - 环索线
   - 蔓叶线
   ```

### 对科学的影响

**伽利略力学**
```
抛物运动：
x = v₀t (水平匀速)
y = -½gt² (垂直匀加速)

消去t：y = -gx²/(2v₀²)
即抛物线！

几何与物理的统一
```

**开普勒定律**
```
椭圆轨道：
x²/a² + y²/b² = 1

解析几何提供了研究行星运动的数学工具
```

### 为微积分奠基

```
解析几何提供了：
- 函数概念的图形表示
- 曲线切线的几何问题
- 曲线下面积的几何问题

这直接导致了：
- 微分学（切线斜率）
- 积分学（面积计算）
- 微积分基本定理
```

## 结语

解析几何的发明是数学史上的里程碑。它统一了代数和几何，开创了数学研究的新范式。通过坐标系，几何问题可以转化为代数问题，代数方程获得了几何解释。这不仅解决了古希腊人无法解决的许多问题，更为微积分的发明和科学革命提供了不可或缺的数学工具。笛卡尔和费马的工作，标志着近代数学的真正开端。
