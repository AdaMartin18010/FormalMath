# 17-18世纪分析学发展

## 概述

### 时代特征
17-18世纪是分析学蓬勃发展的时期。微积分发明后，数学家们将其应用于各个领域，发展了微分方程、变分法、无穷级数等众多分支。这一时期的特点是计算技巧的精湛和应用范围的广泛，但理论基础尚不严密。

### 主要成就
- 微分方程理论的建立
- 变分法的发明
- 无穷级数理论
- 复变函数的初步研究
- 特殊函数的系统化

## 微分方程

### 常微分方程

**早期问题**

```

悬链线问题（1690年）：
求悬挂的链条形成的曲线

雅各布·伯努利提出
莱布尼茨、惠更斯、约翰·伯努利解答

微分方程：dy/dx = s/a
其中s为弧长

解：y = a cosh(x/a) = a(e^(x/a)+e^(-x/a))/2

```

**分离变量法**

```

莱布尼茨和约翰·伯努利发展

例：dy/dx = f(x)g(y)
解法：∫dy/g(y) = ∫f(x)dx

```

**线性方程**

```

一阶线性方程：
dy/dx + P(x)y = Q(x)

积分因子法：
μ(x) = e^(∫P(x)dx)
通解：y = e^(-∫Pdx)[∫Qe^(∫Pdx)dx + C]

莱布尼茨和伯努利兄弟的贡献

```

**二阶常系数方程**

```

ay'' + by' + cy = 0

欧拉（1743年）系统解决：
特征方程：ar² + br + c = 0

根据判别式：
- Δ > 0：y = C₁e^(r₁x) + C₂e^(r₂x)
- Δ = 0：y = (C₁ + C₂x)e^(rx)
- Δ < 0：y = e^(αx)(C₁cosβx + C₂sinβx)

```

### 偏微分方程

**弦振动方程**

```

达朗贝尔（1746年）：
∂²u/∂t² = c² ∂²u/∂x²

解：u(x,t) = f(x-ct) + g(x+ct)
表示向左和向右传播的行波

引发了关于函数概念的争论

```

**位势方程**

```

拉普拉斯方程（1780年代）：
∇²u = ∂²u/∂x² + ∂²u/∂y² + ∂²u/∂z² = 0

描述引力势、静电势等
拉格朗日、拉普拉斯、泊松的研究

```

**热传导方程**

```

傅里叶（1807年，发表于1822年）：
∂u/∂t = α ∂²u/∂x²

开创了傅里叶分析

```

## 变分法

### 最速降线问题

**问题的提出**

```

1696年，约翰·伯努利提出：

"给定竖直平面内两点A和B，
求连接A和B的曲线，
使质点在重力作用下从A滑到B
所需时间最短"

牛顿、莱布尼茨、洛必达、伯努利解答

答案：摆线（旋轮线）

```

**变分法的诞生**

```

这类问题与普通的极值问题不同：
- 普通极值：求函数在某点的极值
- 变分问题：求使积分取极值的函数

例：求使∫F(x,y,y')dx取极值的y(x)

欧拉-拉格朗日方程：
∂F/∂y - d/dx(∂F/∂y') = 0

```

**欧拉的贡献（1744年）**

```

《求具有某种极大或极小性质的曲线的方法》

系统发展了变分法：
- 建立了欧拉方程
- 解决了大量具体问题
- 等周问题、测地线问题等

```

**拉格朗日的改进（1755年）**

```

19岁的拉格朗日提出δ方法

用符号δ表示变分
统一了变分法的处理

欧拉对此高度评价，称之为"变分法"
（calculus of variations）

```

## 无穷级数

### 幂级数

**泰勒级数**

```

泰勒（1715年）：
f(x) = Σ[n=0→∞] f⁽ⁿ⁾(a)(x-a)ⁿ/n!

特例（麦克劳林级数）：
eˣ = 1 + x + x²/2! + x³/3! + ...
sin x = x - x³/3! + x⁵/5! - ...
cos x = 1 - x²/2! + x⁴/4! - ...
ln(1+x) = x - x²/2 + x³/3 - ...

```

**二项式级数**

```

牛顿发现：
(1+x)^α = 1 + αx + α(α-1)x²/2! + ...

欧拉证明对|x|<1收敛

```

### 收敛性问题

**级数求和的矛盾**

```

欧拉曾认为：
1 - 1 + 1 - 1 + ... = 1/2

哥德巴赫问：
1 - 2 + 4 - 8 + ... = ?

这些引发了关于级数收敛性的研究

```

**达朗贝尔比值判别法**

```

达朗贝尔（1768年）：
对于级数Σaₙ，如果
lim |aₙ₊₁/aₙ| = L

则：
- L < 1 时收敛
- L > 1 时发散
- L = 1 时不确定

```

**柯西的贡献（19世纪）**

```

收敛的严格定义
级数理论的严密化
（将在现代数学部分详述）

```

## 特殊函数

### 伽马函数

**阶乘的推广**

```

欧拉（1729年）：
Γ(n) = ∫₀^∞ tⁿ⁻¹e⁻ᵗ dt

性质：
- Γ(n+1) = nΓ(n)
- Γ(n+1) = n! （n为正整数）
- Γ(1/2) = √π

连接了离散和连续

```

### 贝塔函数

```

B(m,n) = ∫₀¹ xᵐ⁻¹(1-x)ⁿ⁻¹ dx

与伽马函数的关系：
B(m,n) = Γ(m)Γ(n)/Γ(m+n)

```

### 椭圆积分

**历史背景**

```

计算椭圆弧长引发：

椭圆x²/a² + y²/b² = 1的弧长
导致积分∫√(1-k²sin²φ)dφ

这是椭圆积分的标准形式

```

**勒让德的研究**

```

勒让德（1786年起）：
《椭圆函数研究》

分类：
- 第一类：∫dφ/√(1-k²sin²φ)
- 第二类：∫√(1-k²sin²φ)dφ
- 第三类：∫dφ/[(1+nsin²φ)√(1-k²sin²φ)]

为19世纪椭圆函数理论奠基

```

## 复变函数的萌芽

### 早期工作

**莱布尼茨和约翰·伯努利的争论**

```

问题：ln(-1) = ?

约翰·伯努利认为：
ln(-x) = ln(x)

莱布尼茨认为：
ln(-x) 是虚数

欧拉后来证明：
ln(-1) = iπ

```

**棣莫弗公式**

```

棣莫弗（1707年）：
(cos θ + i sin θ)ⁿ = cos(nθ) + i sin(nθ)

欧拉公式的特例

```

**欧拉公式（1748年）**

```

e^(iθ) = cos θ + i sin θ

特例（欧拉恒等式）：
e^(iπ) + 1 = 0

被许多人认为是数学中最美的公式
包含了五个最重要的数学常数

```

### 复函数的积分

**达朗贝尔和欧拉**

```

研究复函数的积分：
∫f(z)dz

为柯西积分理论铺路

```

## 函数概念的发展

### 从公式到对应关系

**约翰·伯努利的定义（1718年）**

```

"函数是变量和常量以任意方式组成的量"

强调：
- 解析表达式
- 代数运算和分析运算

```

**欧拉的定义（1748年）**

```

《无穷分析引论》：

"变量的函数是一个解析表达式，
由该变量和一些数或常量以任意方式构成"

欧拉区分：
- 代数函数（有限次代数运算）
- 超越函数（需要无限次运算）

```

**函数概念的扩展**

```

弦振动问题的争论：

达朗贝尔：解必须是光滑函数（可微）
欧拉：允许"不连续"函数（分段定义）
丹尼尔·伯努利：解可以表示为三角级数

这推动了函数概念向更一般的方向发展

```

## 结语

17-18世纪是分析学的黄金时代。在这一时期，微积分从发明走向成熟，发展出微分方程、变分法、无穷级数等众多分支。数学家们以惊人的计算技巧解决了大量科学和工程问题，为近代科学的发展奠定了数学基础。虽然这一时期的分析学在严格性方面存在不足（如无限小量的模糊、级数收敛性的忽视），但其成就为19世纪分析的严格化提供了丰富的素材和明确的方向。
