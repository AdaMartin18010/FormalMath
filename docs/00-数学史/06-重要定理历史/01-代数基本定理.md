# 代数基本定理的历史

## 定理陈述

### 现代形式

```

每个非常数的复系数多项式
在复数域中至少有一个根

等价表述：
n次多项式恰好有n个根（计重数）
复数域是代数闭域

```

### 历史意义

```

连接代数与分析的桥梁
复数本质重要性的证明
多项式理论的基石

```

## 早期尝试

### 18世纪前的认识

```

线性方程：ax + b = 0
总有解 x = -b/a

二次方程：ax² + bx + c = 0
判别式 b² - 4ac < 0 时出现"虚数"
卡尔达诺、邦贝利使用复数

三次、四次方程：
16世纪解法涉及复数
即使实根也需要通过复数表示

疑问：
五次及以上方程呢？
复数是否足够？

```

### 达朗贝尔（1746年）

```

《对积分学的研究》

第一个尝试证明

方法：
假设p(z)无根
则1/p(z)是全纯函数
根据某种极值原理
导出矛盾

缺陷：
极值原理未严格证明
需要分析的严格化

```

## 欧拉的贡献

### 欧拉的研究（1749年）

```

《关于方程的虚根》

目标：
证明每个实系数多项式
可分解为实系数的一次或二次因式

方法：
1. 任何奇次多项式有实根（中间值定理）
2. 尝试用归纳法
3. 提出各种分解策略

问题：
证明不完整
某些步骤依赖直觉

```

### 欧拉的洞察

```

欧拉观察到：
如果p(z) = zⁿ + ...
则 p(z) ≈ zⁿ 当|z|很大

p(0) = a₀

如果p(z)无根
则p将复平面映射到
避开0的区域

这暗示了某种矛盾
但未完成严格证明

```

## 拉格朗日和拉普拉斯

### 拉格朗日（1772年）

```

《关于方程代数解的思考》

虽然主要研究可解性
但涉及根的置换

拉格朗日知道：
根的存在性是基础问题
但他的方法更侧重根之间的关系

```

### 拉普拉斯（1795年）

```

《概率分析理论》中的证明

方法：
利用多项式的对称函数
和判别式

思路：
设多项式p(x)无根
考虑根的幂和
...

证明仍不完整
但接近现代方法

```

## 高斯的证明

### 高斯的博士论文（1799年）

```

《每个单变量的整有理代数函数
都可分解为一次和二次实因式
的新证明》

赫尔姆斯特大学博士论文
年仅22岁！

高斯对前人批评：
"所有之前的证明
在我看来都可以反驳"

他的方法：
几何方法 + 代数方法

```

### 高斯的几何方法

```

核心思想：

设 p(z) = zⁿ + a_{n-1}z^{n-1} + ... + a₀

考虑 z = r(cos θ + i sin θ)

当θ从0变到2π时
p(z)描绘一条曲线

关键观察：
对于小r，曲线绕a₀点
对于大r，曲线像zⁿ绕原点n圈

由连续性：
存在某个r使得曲线过原点
即存在z使得p(z) = 0

几何直观清晰
但需要连续性的严格处理

```

### 高斯的后续证明

**1815年的证明**

```

更代数化的方法
使用结式（resultant）

```

**1816年的两个证明**

```

证明1：
基于积分的论证

证明2：
基于幂级数的论证

这些证明更接近现代复分析

```

## 后续发展

### 柯西（1821年）

```

使用复分析的方法

基于：
刘维尔定理的雏形
解析函数的性质

严格化的分析
为定理提供坚实基础

```

### 阿甘（1806年）和沃塞尔（1799年）

```

复平面的表示

几何直观：
复数作为平面上的点
多项式映射的几何

为代数基本定理
提供可视化理解

```

### 刘维尔（1844年）

```

刘维尔定理：
有界整函数必为常数

证明：
若p(z)无根，则1/p(z)是全纯的
当|z|→∞，|1/p(z)|→0

所以1/p(z)有界
因此1/p(z)是常数
矛盾！

这是最简单优雅的证明之一

```

## 现代证明方法

### 拓扑证明

```

映射度方法：

p(z) 诱导球面到自身的映射
（通过球极投影）

多项式映射的度为n > 0
因此必须是满射

代数拓扑的优雅应用

```

### 复分析证明

```

刘维尔定理法（最简单）

或：
柯西积分公式
辐角原理
留数定理

标准复分析课程的内容

```

### 代数证明

```

利用：
实闭域理论
伽罗瓦理论

 Galois群的作用
 复共轭

证明复数域无真代数扩张

```

### 构造性证明？

```

上述证明都是存在性的
不提供求根的具体方法

问题：
是否存在构造性证明？

答案：
不是递归构造的
求根不能是连续的过程
（依赖根的选择）

```

## 影响与意义

### 对代数学的影响

```

1. 多项式完全可解
   n次方程有n个复根

2. 特征值存在
   矩阵总有特征值（复数域上）

3. 代数闭域概念
   复数域是代数闭域的典范

```

### 对分析学的影响

```

1. 复分析的基石
   整函数理论
   多项式逼近

2. 谱理论
   算子的谱非空

3. 泛函分析
   巴拿赫代数的Gelfand-Mazur定理

```

### 对数学基础的启示

```

1. 存在性证明的价值
   即使非构造性也有意义

2. 多证明的重要性
   不同角度理解同一真理

3. 严格化的历程
   从几何直观到严格证明

```

## 历史评价

### 高斯的地位

```

虽然不完全严格
高斯的证明是转折点

标志着：
- 对复数本质的接受
- 几何与代数的结合
- 存在性证明的方法

高斯本人说：
他1799年的证明
"非常严格"
尽管需要分析的完善

```

### 严格化的历程

```

1799年：高斯第一证明
↓
19世纪初：复分析发展
↓
1844年：刘维尔定理
↓
19世纪末：严格分析
↓
20世纪：抽象代数证明

一个定理的200年完善

```

## 结语

代数基本定理的历史展示了数学发展的典型模式：从几何直观到代数方法，从不严格到严格，从单一到多元。高斯在22岁时的博士论文开启了这一历程，而后来的数学家们不断完善和深化对这一基本真理的理解。这个定理不仅是代数学的基石，也深刻影响了分析学、几何学和现代数学的各个分支。
