# 费马大定理的历史

## 定理陈述

### 费马的批注

```

1637年左右，费马在阅读
丢番图《算术》时写下：

"不可能将一个立方数写成两个立方数之和，
或者将一个四次幂写成两个四次幂之和，
或者一般地，将任何高于二次的幂
写成两个同次幂的和。

我发现了一个真正美妙的证明，
但这里的空白太小，写不下。"

用现代符号：
对于整数n > 2，方程
xⁿ + yⁿ = zⁿ
没有正整数解。

```

### 历史意义

```

最著名的数学问题之一
350多年未解决
吸引了无数数学家
推动了数论的整体发展

```

## 费马本人的贡献

### n = 4 的情形

```

费马确实证明了n=4的情形

方法：无穷递降法

证明思路：
假设x⁴ + y⁴ = z²有解
可构造出更小的解
无限递降，矛盾

因此x⁴ + y⁴ = z⁴无解

这是费马唯一留下的证明

```

### 无穷递降法

```

费马的独门绝技

原理：
若P(n)对某n成立
则可构造m < n使P(m)成立

则P(n)对所有n不成立
（否则无限递降）

这是一种反证法
也成为数论的重要工具

```

## 18世纪的发展

### 欧拉与n = 3

```

欧拉（1770年左右）：

证明了n=3的情形
x³ + y³ = z³无解

方法：
引入形式a + b√-3
类似于整数

问题：
欧拉假设了类似唯一分解
实际上需要更精细的论证

勒让德后来完善了证明

```

### 索菲·热尔曼

```

Sophie Germain（1776-1831）

热尔曼素数：
若p是素数，2p+1也是素数

定理：
若p是热尔曼素数
则xᵖ + yᵖ = zᵖ没有解
使得p不整除xyz

（第一种情形的部分结果）

贡献：
在女性被排斥于学术界的时代
做出重要工作
与高斯、勒让德通信

```

## 19世纪的突破

### 狄利克雷与勒让德

```

1825年：
狄利克雷证明n=5
勒让德补充证明

方法：
分圆整数
Q(ζ₅)的算术

虽然Q(ζ₅)是唯一分解整环
但论证仍复杂

```

### 拉梅的尝试（1847年）

```

Gabriel Lamé宣布证明一般情形

方法：
分圆整数
Q(ζₙ) = Q(e^{2πi/n})

拉梅假设：
分圆整数有唯一分解

刘维尔指出问题：
库默尔已证明
唯一分解在n=23时不成立

危机：
拉梅的证明失败
但这催生了代数数论

```

### 库默尔的革命

**恩斯特·库默尔（1810-1893）**

```

1844年起的工作：

发现：
分圆域Q(ζₚ)中
唯一分解不一定成立

解决方案：
引入理想数（理想）
恢复唯一分解

理想类群：
测量与唯一分解的偏差
类数为1当且仅当唯一分解

正则素数：
若素数p不整除Q(ζₚ)的类数
则称p为正则的

库默尔定理：
对正则素数p
费马大定理成立

结果：
证明了对所有正则素数成立
100以内唯一非正则：37, 59, 67

```

### 代数数论的建立

```

库默尔的工作开创了代数数论

后续发展：
戴德金：理想理论
希尔伯特：类域论

费马大定理推动了
整个现代数论的发展

```

## 20世纪的进展

### 早期计算机验证

```

1950-70年代：
用计算机验证n的大值

Vandiver等人：
验证了n < 4000

方法：
改进库默尔方法
处理非正则素数

后来扩展到n < 4,000,000

```

### 法尔廷斯定理（1983年）

```

Gerd Faltings（1986年菲尔兹奖）：

莫德尔猜想：
代数数域上亏格≥2的曲线
只有有限个有理点

法尔廷斯证明了这个猜想

对费马大定理的推论：
对每个n ≥ 3
方程xⁿ + yⁿ = zⁿ
只有有限个本原整数解

虽然不是完全解决
但重大进展

```

## 谷山-志村猜想

### 从椭圆曲线到模形式

**椭圆曲线**

```

方程：y² = x³ + ax + b
判别式非零保证非奇异

有理点构成阿贝尔群
莫德尔定理：
有理点群是有限生成的

L函数：
L(E, s) = ∏(1 - aₚp⁻ˢ + p¹⁻²ˢ)⁻¹

```

**模形式**

```

上半平面上的全纯函数
满足：
f((az+b)/(cz+d)) = (cz+d)ᵏf(z)

傅里叶展开：
f(z) = Σ aₙqⁿ，q = e^{2πiz}

L函数：
L(f, s) = Σ aₙn⁻ˢ

```

**谷山-志村猜想（1955-1960）**

```

谷山丰（1927-1958）
志村五郎（1930-2019）

猜想：
每条定义在有理数上的椭圆曲线
都是模的

即：
存在模形式f使得
L(E, s) = L(f, s)

等价表述：
椭圆曲线可被模曲线参数化

```

### 弗雷和里贝特

**赫尔伯特·弗雷（1984年）**

```

关键洞察：

假设费马大定理不成立：
aᵖ + bᵖ = cᵖ有解

构造椭圆曲线：
y² = x(x - aᵖ)(x + bᵖ)

这条曲线具有非常特殊的性质
似乎不可能是模的

弗雷曲线：
如果谷山-志村猜想成立
则费马大定理成立

```

**肯·里贝特（1986年）**

```

证明ε猜想：

弗雷曲线确实不是模的

技术上：
使用了伽罗瓦表示
水平和提升

结果：
谷山-志村猜想 ⇒ 费马大定理

将问题转化为证明
椭圆曲线的模性

```

## 怀尔斯的证明

### 安德鲁·怀尔斯

```

1953年生于剑桥
1970年代：在剑桥开始研究
1980年代：普林斯顿大学

1986年得知里贝特结果后
决定证明谷山-志村猜想

秘密工作7年
不与数学界交流
1993年宣布结果

```

### 1993年6月

```

剑桥大学牛顿研究所
连续三天演讲

演讲标题：
"模形式、椭圆曲线和伽罗瓦表示"

结束时宣布：
证明了谷山-志村猜想的一部分
足够推出费马大定理

世界轰动！
纽约时报等媒体报道

```

### 发现漏洞

```

审稿过程中：
卡兹发现一个问题

欧拉系统论证中的缺陷
一个关键的同态
可能不存在

危机：
证明不完整
数学界关注

```

### 修复证明

```

怀尔斯与理查德·泰勒合作

泰勒是怀尔斯的学生

一年的努力：
绕过欧拉系统
改用其他方法

1994年9月：
成功修复证明

论文：
《模椭圆曲线和费马大定理》
《某些Hecke代数的环论性质》
（与泰勒合著）

发表于《数学年刊》1995年

```

### 证明的核心

```

 Galois表示的变形理论：

1. 椭圆曲线E的p进Galois表示
2. 与模形式相关的Galois表示
3. 证明这两个表示相同

关键工具：
- 塞尔默群
- 岩泽理论
- Hecke代数
- 类数公式

技术的巅峰：
代数数论、代数几何、表示论的综合

```

## 影响与意义

### 数学发展

```

1. 证明了谷山-志村猜想的一部分
   推动了朗兰兹纲领

2. 发展了Galois变形理论
   成为数论的核心工具

3. 椭圆曲线理论的巨大进步

4. 代数数论与表示论的结合

```

### 费马真的有证明吗？

```

几乎可以肯定：
费马没有正确的证明

原因：
1. 无穷递降法只适用于n=4
2. 椭圆曲线、模形式理论
   直到20世纪才发展
3. 怀尔斯的证明使用了
   费马时代的数学无法想象的工具

费马可能是：
- 犯了错误
- 对某个特殊情况有证明
- 留下了这个著名的挑战

```

## 结语

费马大定理的历史是数学史上最动人的篇章之一。从费马1637年的批注到怀尔斯1994年的证明，350多年的历程见证了数论从初等算术到现代代数几何的演变。库默尔的理想理论、谷山-志村的深刻猜想、怀尔斯的最终突破——这一历程展示了数学的整体性和深刻性。费马大定理不仅是一个定理的证明，更是人类追求真理的见证，数学美的体现。
