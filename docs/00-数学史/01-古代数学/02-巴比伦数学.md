# 巴比伦数学

## 历史背景

### 文明概况
巴比伦文明发源于美索不达米亚平原（今伊拉克境内），幼发拉底河和底格里斯河之间。经历了苏美尔、阿卡德、巴比伦和亚述等时期，数学传统延续近两千年。

### 数学需求
1. **商业贸易**：复杂的货币交换和利息计算
2. **土地管理**：灌溉系统的设计和土地分配
3. **天文学**：编制历法、预测月食
4. **建筑工程**：宫殿、神庙和城墙的建造

## 主要成就

### 1. 位值制数字系统

**六十进制（Sexagesimal）**
- 世界上最早的位值制数字系统
- 以60为基数，但采用十进制的辅助符号
- 符号：
  - `Y` 表示 1
  - `<` 表示 10

**数字表示示例**

```

1 = Y
10 = <
23 = <<YYY
59 = <<<<<YYYYYYYYY
60 = Y (位值制，在六十位)

```

**位值示例**

```

1,25 = 1×60 + 25 = 85
2,30 = 2×60 + 30 = 150
1,0,0 = 1×60² = 3600

```

**分数表示**
- 六十进制小数（实际上是分数）
- 例：0;30 = 30/60 = 1/2
- 例：0;45 = 45/60 = 3/4
- 例：1,25;30 = 85 + 1/2 = 85.5

### 2. 算术运算

**乘法表**
- 学生需要背诵1-59的乘法表
- 主要背诵1, 1;15(=1¼), 1;30(=1½), 2, 3, 4, 5, 6, 8, 9, 10, 12, 15, 16, 18, 20, 24, 25, 27, 30, 32, 36, 40, 45, 48, 50, 54等数的倍数

**倒数表**
- 由于除法通过乘以倒数完成
- 倒数表列出1-60的倒数（以六十进制表示）
- 例：1/2 = 0;30, 1/3 = 0;20, 1/4 = 0;15

**开方算法**
- 已知求√2的近似值：1;24,51,10
- 转换为十进制：1 + 24/60 + 51/3600 + 10/216000 ≈ 1.414213...
- 精确到小数点后6位！
- 可能使用迭代法：x_{n+1} = (x_n + a/x_n)/2

### 3. 代数成就

**二次方程**
巴比伦人能够解形如以下的方程：
- x² + px = q
- x² - px = q
- x² + q = px
- x² = px + q

**解法示例（现代符号表示）**

```

问题：已知正方形面积与边长之和为 3/4
方程：x² + x = 3/4

巴比伦算法（配方法）：
1. 取系数的一半：1/2 × 1 = 1/2
2. 平方：(1/2)² = 1/4
3. 加到右边：3/4 + 1/4 = 1
4. 开方：√1 = 1
5. 减去一半：1 - 1/2 = 1/2

答案：x = 1/2

```

**方程组**
能够解二元一次方程组，如：

```

x + y = a
x - y = b

```

和二元二次方程组，如：

```

x + y = a
xy = b

```

### 4. 数论

**勾股数**
- 普林顿322号泥板列出了15组勾股数
- 最大的：(12709, 13500, 18541)
- 生成方法可能与公式 a²-b², 2ab, a²+b² 有关

|------|---|---|---|-------|
| 1 | 119 | 120 | 169 | 1.983 |
| 2 | 3367 | 3456 | 4825 | 1.949 |
| ... | ... | ... | ... | ... |
| 15 | 106 | 153 | 185 | 1.461 |

**等比数列**
- 了解等比数列求和
- 例：1 + 2 + 4 + ... + 2⁹ = 2¹⁰ - 1 = 1023

### 5. 几何学成就

**圆和圆周率**
- 圆周率取 π ≈ 3
- 但更精确的计算显示他们知道 π ≈ 3⅛ = 3.125

**体积计算**
- 柱体、锥体体积
- 截头锥体积公式与埃及人类似

**π的近似值**
- 泥板YBC 7304给出圆面积公式：S = c²/12
  （c为周长）
- 这相当于 π = 3

### 6. 天文学与数学

**天文表**
- 系统记录行星运动
- 预测月食和日食
- 编制历法（阴阳历）

**内插法**
- 线性内插法
- 用于天文数据的计算

## 关键人物

由于巴比伦数学多为匿名泥板，难以确定具体人物，但我们可以确定几个重要时期：

### 1. 苏美尔时期（约前2000年）
- 建立了基本的数字系统
- 创造了六十进制的雏形

### 2. 古巴比伦时期（前1800-前1600年）
- 数学发展的黄金时期
- 普林顿322、YBC 7289等重要泥板

### 3. 新巴比伦和波斯时期（前600-前300年）
- 天文学与数学结合
- 预测性天文学的发展

## 影响分析

### 对后世的影响

1. **六十进制传承**
   - 时间计量（60秒=1分，60分=1时）
   - 角度计量（360°圆周）
   - 这些一直使用到今天

2. **代数方法**
   - 二次方程的配方法后来被希腊人和阿拉伯人继承
   - 代数问题的系统处理方法

3. **天文学数学**
   - 希腊天文学继承了巴比伦的数学方法
   - 托勒密体系使用了巴比伦的六十进制计算

### 与埃及数学的比较

| 方面 | 巴比伦数学 | 埃及数学 |
|------|-----------|----------|
| 数字系统 | 位值制六十进制 | 非位值制十进制 |
| 代数 | 系统、复杂 | 简单、具体 |
| 几何 | 代数化 | 测量导向 |
| 天文 | 数学天文发达 | 实用历法 |
| 保存 | 泥板保存完好 | 纸草易损 |

## 结语

巴比伦数学代表了古代近东数学的最高成就。他们创造的六十进制位值系统、系统的代数方法和精确的计算技术，在数学史上占有重要地位。特别是他们对二次方程的完整解决和勾股数的深入研究，显示了高度的数学抽象能力。这些成就后来通过希腊和伊斯兰数学家中介，成为现代数学的重要源头。
