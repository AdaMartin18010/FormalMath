# 埃及数学

## 历史背景

### 文明概况
古埃及文明发源于尼罗河流域，约公元前3100年统一。尼罗河的定期泛滥带来了肥沃的土壤，也催生了土地测量和天文观测的需求，这是埃及数学发展的主要动力。

### 数学需求
1. **土地测量**：尼罗河泛滥后需要重新丈量土地（几何学的词源"geometry"即"测地术"）
2. **建筑工程**：金字塔、神庙的建造需要精确的计算
3. **天文历法**：制定历法、预测尼罗河泛滥时间
4. **行政管理**：税收计算、粮食分配

## 主要成就

### 1. 数字系统

**象形数字（约公元前3000年）**
- 十进制系统，但没有位值概念
- 符号：

  | 数字 | 符号 | 说明 |
  |------|------|------|
  | 1 | 单竖线 | 叠加表示 |
  | 10 | 牛轭 | ∧形状 |
  | 100 | 绳卷 | 螺旋形 |
  | 1000 | 莲花 | 花朵图案 |
  | 10000 | 手指 | 弯曲手指 |
  | 100000 | 蝌蚪 | 蝌蚪形状 |
  | 1000000 | 举臂人 | 举臂人像 |

**示例**：325 = 3个绳卷 + 2个牛轭 + 5个竖线

### 2. 算术运算

**分数表示**
- 主要使用单位分数（分子为1的分数）
- 记法：在数字上方画椭圆表示倒数
- 例：1/3 = 椭圆内写3
- 特殊符号：2/3 和 3/4 有专门符号

**乘法方法：倍乘法**

```

例：13 × 24

步骤：
  1    24
  2    48
  4    96
  8   192
──────────
13 = 1 + 4 + 8
结果：24 + 96 + 192 = 312

```

**除法方法**
- 将除法转化为乘以倒数
- 或采用与乘法类似的倍减方法

### 3. 代数问题

**莱因德纸草书（约公元前1650年）**
- 发现于底比斯，现存大英博物馆
- 包含85道数学问题
- 是了解埃及数学的主要来源

**典型问题**

问题24：某数加上它的七分之一等于19，求该数。
- 设该数为x
- 方程：x + x/7 = 19
- 埃及解法：取7的倍数试探
  - 尝试7：7 + 1 = 8 ≠ 19
  - 尝试7×(19/8) = 133/8 = 16.625
- 答案：16 + 5/8

### 4. 几何学成就

**面积计算**

| 图形 | 公式 | 现代等价 |
|------|------|----------|
| 三角形 | S = (b × h)/2 | 正确 |
| 矩形 | S = a × b | 正确 |
| 梯形 | S = (a + b) × h/2 | 正确 |
| 圆 | S = (8d/9)² | ≈ 3.16r² (π≈3.16) |

**圆面积计算**
- 使用直径d的(8/9)作为边长构造正方形
- 这相当于取 π ≈ 256/81 ≈ 3.16
- 误差约为0.6%

**体积计算**
- 柱体体积：V = S × h
- 截头方锥体积（莫斯科纸草书第14题）：
  ```

  V = h/3 × (a² + ab + b²)
  ```

  其中a、b为上下底边长，h为高
- 这是人类历史上最早的复杂体积公式

### 5. 金字塔建造中的数学

**胡夫金字塔（约公元前2560年）**
- 原高：146.5米
- 底边：230.4米
- 斜率：51°50'（对应seked 5½）

**Seked（斜率）概念**
- 定义：底边一半与高的比，以手掌为单位（1手掌≈7指宽）
- 例：seked 5½ 表示每上升1肘尺（约52.5cm），水平移动5½手掌
- 相当于现代的正切函数概念

## 关键人物

### 1. 伊姆霍特普（Imhotep，约公元前2650年）
- 左塞尔金字塔的设计者
- 被认为是世界上第一位留名的建筑师
- 后来被神化为医药之神

### 2. 阿默斯（Ahmes，约公元前1650年）
- 莱因德纸草书的抄写者
- 非原创数学家，但保存了更早的数学知识
- 记录了约公元前1800年的数学问题

## 影响分析

### 对后世的影响
1. **实用数学传统**：强调计算技巧，而非抽象证明
2. **单位分数传统**：影响了后来的希腊和伊斯兰数学
3. **几何方法**：土地测量技术传播到希腊

### 局限性
1. **缺乏抽象证明**：只给出具体问题的解法，无一般性证明
2. **符号系统局限**：没有位值制，大数表示繁琐
3. **知识传承**：多为实用技能，缺乏理论发展

## 结语

埃及数学代表了人类文明早期数学发展的最高成就。虽然在抽象性和系统性上不及后来的希腊数学，但埃及人在实用计算方面达到了很高的水平。他们的成就证明了数学起源于人类解决实际问题的需要，这一实用传统至今仍在工程技术和应用数学中延续。
