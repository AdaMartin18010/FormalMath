# 中国数学

## 历史背景

### 文明概况
中国数学有着悠久而独立的发展历史，从商周时期的萌芽到宋元时期的高峰，形成了独具特色的算法传统。中国数学注重实用计算和算法构造，在代数、算术方面成就卓著。

### 发展时期
- **先秦时期**（前1000-前221年）：数学萌芽
- **汉唐时期**（前206-907年）：数学体系形成
- **宋元时期**（960-1368年）：数学高峰
- **明清时期**（1368-1911年）：中西数学交汇

## 主要成就

### 1. 数字系统与计算工具

**十进制位值制**
- 甲骨文已有十进制数字
- 算筹表示法创造了世界上最早的十进位值制
- 纵式横式交替表示多位数

**算筹记数**

```

纵式：| || ||| |||| 丅 丅| 丅|| 丅||| 丅|||| 丶

数字：1  2   3    4    5   6   7    8     9     0

横式：—  =  ≡  ≡  丅  丅  丅  丅  丅  
数字：1  2  3   4   5   6   7   8   9

例：6728 = 丅 ||| || 丅|||

```

**算盘**
- 珠算起源于宋代
- 明代《算法统宗》完善
- 至今仍在使用

### 2. 《九章算术》

**成书背景**
- 西汉时期（前206-公元8年）编订
- 收录了先秦至汉代的数学成就
- 中国最重要的数学经典

**内容结构**

| 章 | 内容 | 问题数 |
|----|------|--------|
| 方田 | 分数运算、面积计算 | 38 |
| 粟米 | 比例交换 | 46 |
| 衰分 | 比例分配 | 20 |
| 少广 | 开方术 | 24 |
| 商功 | 体积计算 | 28 |
| 均输 | 赋税计算 | 28 |
| 盈不足 | 双假设法 | 20 |

**重要算法**

**开方术**

```

相当于现代的开平方算法
例：√55225 = ?

步骤：
1. 分段：5'52'25
2. 2² ≤ 5 < 3²，首位为2
3. (5-4)×100 + 52 = 152
4. 20×2 = 40，找x使(40+x)x ≤ 152
   取x=3，43×3=129
5. (152-129)×100 + 25 = 2325
6. 230×2 = 460，找x使(460+x)x ≤ 2325
   取x=5，465×5=2325

结果：235

```

**方程术（线性方程组）**

```

例：
3x + 2y + z = 39
2x + 3y + z = 34
x + 2y + 3z = 26

用算筹表示（类似增广矩阵）：
┌──┬──┬──┬──┐
│1 │2 │3 │26│
│2 │3 │2 │34│
│3 │1 │1 │39│
└──┴──┴──┴──┘

通过初等行变换求解
这是世界上最早的矩阵算法

```

**正负术**
- 系统的正负数运算法则
- "同名相除，异名相益"
- 相当于现代符号法则

**盈不足术**

```

问题：若干人买物，每人出8钱，盈3钱；每人出7钱，不足4钱。问人数、物价各多少？

设人数为x，物价为y
8x = y + 3
7x = y - 4

盈不足术：
x = (盈+不足)/(两次出钱差) = (3+4)/(8-7) = 7人
y = 8×7 - 3 = 53钱

这是世界上最早的双假设法

```

### 3. 代数成就

**天元术**
- 宋代创立
- 设未知数（天元一）列方程
- 用算筹位置表示幂次
- 相当于现代的代数方程

**四元术（朱世杰，1303年）**
- 天、地、人、物四元
- 可表示四个未知数的方程组
- 最高成就：四元四次方程组

**增乘开方法**
- 贾宪（约1050年）提出
- 可求高次方程数值解
- 相当于霍纳算法（1819年）的雏形

**贾宪三角（杨辉三角）**

```

        1
      1   1
    1   2   1
  1   3   3   1
1   4   6   4   1

用于开方和展开二项式

```

### 4. 几何学成就

**勾股定理**
- 《周髀算经》已有记载（约前100年）
- 商高与周公对话中提到
- "勾三股四弦五"
- 给出了一般性证明

**刘徽割圆术**

```

刘徽（约263年）计算π：

从正六边形开始，不断倍增边数：
6边 → 12边 → 24边 → ... → 3072边

得到：
π ≈ 3927/1250 = 3.1416

"割之弥细，所失弥少，
 割之又割，以至于不可割，
 则与圆合体而无所失矣"

这是极限思想的早期表达

```

**祖暅原理**

```

幂势既同，则积不容异
（等高处截面积相等，则体积相等）

用于求球体积：
V = (π/6)d³ ≈ (11/21)d³

这是卡瓦列里原理（1635年）的先声

```

### 5. 圆周率计算

| 数学家 | 年代 | 结果 | 精度 |
|--------|------|------|------|
| 刘徽 | 263年 | 3.1416 | 4位 |
| 祖冲之 | 480年 | 355/113 | 6位 |
| | | 3.1415926 < π < 3.1415927 | 7位 |

**祖冲之（429-500年）**
- 密率：355/113 ≈ 3.1415929
- 约率：22/7 ≈ 3.142857
- 保持世界记录近千年

### 6. 大衍求一术

**秦九韶（约1202-1261）**
- 《数书九章》（1247年）
- 大衍求一术：一次同余式组解法
- 相当于中国剩余定理

```

问题（物不知数）：
今有物不知其数，
三三数之剩二，
五五数之剩三，
七七数之剩二，
问物几何？

即解：
x ≡ 2 (mod 3)
x ≡ 3 (mod 5)
x ≡ 2 (mod 7)

答案：x = 23 + 105k
最小正整数解：23

```

## 影响分析

### 特色与贡献

1. **算法传统**
   - 注重算法构造
   - 机械化计算程序
   - 对现代计算机科学有启发

2. **十进位值制**
   - 世界上最先进、最方便的记数法
   - 经印度、阿拉伯传入欧洲

3. **代数成就**
   - 高次方程数值解
   - 线性方程组解法
   - 同余理论

### 衰落与复兴

**明清之际的衰落**
- 科举制度不重视数学
- 缺乏符号系统
- 与西方数学差距拉大

**中西数学交汇**
- 明末利玛窦传入欧洲数学
- 《几何原本》翻译（1607年）
- 康熙帝对数学的重视

### 现代意义

- 吴文俊院士的"数学机械化"研究
- 中国古代数学算法的现代价值
- 东西方数学传统的互补

## 结语

中国数学是世界数学的重要组成部分，其独特的算法传统和实用取向，与希腊的演绎传统形成互补。在宋元时期达到高峰后，中国数学一度衰落，但其思想和方法仍具有重要的历史价值和现代意义。今天，随着对中国古代数学文献的深入研究，世界对中国数学成就的认识正在不断深化。
