# 莱昂哈德·欧拉（Leonhard Euler，1707-1783）

## 生平

### 早年

```

1707年4月15日：生于瑞士巴塞尔
父亲：保罗·欧拉，加尔文教派牧师
母亲：玛格丽特·布吕克

父亲希望他成为牧师
但伯努利家族发现了他的数学天赋

```

### 教育

```

1720年：13岁，入巴塞尔大学
学习：神学、医学、数学

1723年：获硕士学位
论文主题：笛卡尔哲学 vs 牛顿哲学

1726年：完成博士学业
题目：De Sono（论声音）

约翰·伯努利的指导：
- 私下授课
- 推荐欧拉到圣彼得堡

```

### 圣彼得堡时期（1727-1741）

```

1727年：抵达俄罗斯
最初在医学部
后转入数学部

1730年：成为教授

1733年：丹尼尔·伯努利离开后
成为数学部主任

1734年：与凯瑟琳娜结婚
育有13个孩子，5个活到成年

1741年：应腓特烈大帝邀请
离开圣彼得堡前往柏林

在圣彼得堡期间：
- 奠定了分析学基础
- 解决了很多难题
- 写了大量论文

```

### 柏林时期（1741-1766）

```

1741-1766年：在柏林科学院

与腓特烈大帝的关系：
- 初期和谐
- 后来因哲学观点分歧
- 欧拉是虔诚的基督徒
- 大帝倾向启蒙哲学

1766年：接受叶卡捷琳娜二世邀请
返回圣彼得堡

柏林期间成就：
- 《无穷分析引论》（1748年）
- 《微分学原理》（1755年）
- 《积分学原理》（1768-1770年）

```

### 圣彼得堡晚年（1766-1783）

```

1766年：返回圣彼得堡
受到盛大欢迎

1771年：房屋火灾
几乎丧命
手稿部分被毁

1773年：妻子去世
1776年：再婚

失明：
1735年：右眼几乎失明（观察太阳）
1766年：左眼白内障
1771年：完全失明

但失明后仍保持高产：
通过口述，由助手记录

```

### 逝世

```

1783年9月18日

当天：
上午：计算气球上升定律
下午：与孙子玩耍
傍晚：突发脑溢血
去世

临终遗言：
"我停止了"
（I am dying）

俄罗斯科学院：
"欧拉停止了计算，停止了生命"

```

## 数学贡献

### 分析学

**《无穷分析引论》（1748年）**

```

Introductio in Analysin Infinitorum

里程碑式的著作：
- 第一本现代分析学教科书
- 系统使用函数概念
- 统一了初等函数理论

内容：
- 无穷级数
- 连分数
- 指数和对数函数
- 三角函数
- 曲线的解析理论

影响：
培训了几代数学家
定义了分析学的标准

```

**欧拉公式**

```

最著名的公式：
e^(iθ) = cos θ + i sin θ

特例：
e^(iπ) + 1 = 0

被誉为"数学中最美的公式"
联系五个最重要的常数：
- e：自然对数的底
- i：虚数单位
- π：圆周率
- 1：乘法单位元
- 0：加法单位元

```

**无穷级数**

```

欧拉求出了许多级数的和：

Σ 1/n² = π²/6
（巴塞尔问题）

Σ 1/n⁴ = π⁴/90

Σ (-1)^(n+1)/n = ln 2

方法：
- 比较无穷乘积与无穷级数
-  clever的变换技巧
- 有时不太严格但结果正确

```

**Γ函数**

```

阶乘的推广：
Γ(n) = (n-1)! （n为正整数）

定义：
Γ(z) = ∫₀^∞ t^(z-1)e^(-t)dt

性质：
Γ(z+1) = zΓ(z)
Γ(1/2) = √π

应用广泛：
- 数论
- 概率论
- 物理学

```

### 数论

**费马小定理的首次证明**

```

1736年：
给出费马小定理的第一个证明

定理：
若p是素数，a不被p整除
则a^(p-1) ≡ 1 (mod p)

方法：
数学归纳法
组合论证

```

**欧拉函数**

```

φ(n)：小于n且与n互素的正整数个数

性质：
- 乘性函数
- n = Σ φ(d)，d|n

欧拉定理：
a^φ(n) ≡ 1 (mod n)
（a与n互素）

推广了费马小定理

```

**二次互反律的猜想**

```

欧拉发现了二次互反律
但未给出完整证明

勒让德命名
高斯首次证明（1796年）

这是代数数论的起点

```

**对费马大定理的贡献**

```

证明了n=3的情形
（不完整，需勒让德完善）

发展了分圆整数的方法
为库默尔的工作奠基

```

### 图论

**柯尼斯堡七桥问题（1736年）**

```

问题：
柯尼斯堡城（今加里宁格勒）
普雷格尔河穿过城市
有七座桥连接两岸和河中的岛

问：能否一次走遍所有桥
每座桥只走一次？

欧拉的解法：
1. 抽象为图：
   - 陆地为顶点
   - 桥为边
   
2. 关键观察：
   除起点和终点外
   每个顶点必须有偶数条边
   （进入和离开的次数相等）

3. 结论：
   柯尼斯堡七桥不可能走遍

这是图论的诞生！

欧拉路径：
经过每条边恰好一次的路径

欧拉回路：
起点和终点相同的欧拉路径

存在条件：
图连通且顶点度数为偶数
（回路情形）

```

### 变分法

**欧拉方程**

```

变分法的核心方程

问题：
求使泛函J[y] = ∫F(x,y,y')dx
取极值的函数y(x)

欧拉方程：
∂F/∂y - d/dx(∂F/∂y') = 0

这是变分法的奠基性工作
后来的拉格朗日做了改进

```

### 微分方程

**欧拉方法**

```

数值解常微分方程的基本方法

y_{n+1} = y_n + hf(x_n, y_n)

简单但有效
是更复杂方法的基础

```

**线性微分方程**

```

系统研究常微分方程

解法：
- 常系数方程
- 欧拉方程（变系数）
- 级数解法

特殊函数：
- 贝塞尔函数
- 勒让德函数
- 椭圆积分

```

### 几何学

**欧拉线（1765年）**

```

三角形的重要结果：

垂心H、重心G、外心O
三点共线

且HG = 2GO

这是三角形几何的经典定理

```

**多面体公式**

```

欧拉示性数：
V - E + F = 2

V：顶点数
E：边数
F：面数

适用于凸多面体
拓扑学的不变量

```

**地图投影**

```

设计了多种地图投影方法
包括：
- 等距方位投影
- 球面投影的研究

```

### 力学

**刚体转动**

```

欧拉角：
描述刚体转动的三个角度

欧拉方程：
刚体转动的运动方程

动量矩定理：
奠定了刚体动力学的基础

```

**流体力学**

```

欧拉方程：
无粘性流体的运动方程

连续性方程：
质量守恒的表达式

奠定了理论流体力学

```

## 符号创新

### 数学符号

```

```

   - 创造了现代数学语言

## 结语

莱昂哈德·欧拉是数学史上最伟大的数学家之一。他的著作数量和广度前无古人，他的洞察力和创造力令人叹服。从分析学到数论，从几何学到力学，欧拉在每一个领域都做出了奠基性的贡献。即使在失明后，他依然保持了惊人的创造力，展示了人类精神的坚韧力量。欧拉不仅是一位数学家，更是理性思维和勤奋工作的化身，他的名字将永远与数学的光辉历史相连。
