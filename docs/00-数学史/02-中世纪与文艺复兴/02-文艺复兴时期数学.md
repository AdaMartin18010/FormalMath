# 文艺复兴时期数学

## 历史背景

### 时代概况
文艺复兴（约1400-1600年）是欧洲从中世纪向近代过渡的时期，艺术、科学和数学都经历了革命性变化。人文主义思潮推动了对古典知识的重新审视，地理大发现和宗教改革为数学发展提供了新的动力。

### 推动因素
1. **人文主义**：恢复希腊数学传统
2. **印刷术**：知识的广泛传播
3. **商业发展**：复杂的计算需求
4. **艺术需求**：透视法的数学基础
5. **天文学革命**：新的宇宙模型需要新数学

## 主要成就

### 1. 透视法的数学化

**布鲁内莱斯基（Brunelleschi，1377-1446）**
- 佛罗伦萨建筑师
- 透视法的发明者
- 证明了透视的数学原理

**阿尔贝蒂（Alberti，1404-1472）**
- 《论绘画》（1435年）
- 建立透视法的数学基础
- "视觉金字塔"概念

**透视几何原理**

```

基本问题：如何将三维场景投影到二维画面？

阿尔贝蒂的解决方法：
1. 画家站在固定位置（视点）
2. 画面是介于视点和物体之间的平面
3. 所有视线（光线）汇聚于视点
4. 物体与画面的交点即为透视像

数学本质：
- 射影几何的先驱
- 中心投影变换
- 灭点（vanishing point）概念

```

**丢勒（Dürer，1471-1528）**
- 德国画家、数学家
- 《测量指南》（1525年）
- 设计了透视绘图仪器

### 2. 三角学的进步

**雷吉奥蒙塔努斯（Regiomontanus，1436-1476）**
- 德国数学家、天文学家
- 《论各种三角形》（1533年出版）
- 三角学从天文附属变为独立学科

**《论各种三角形》结构**

```

5卷内容：

卷I：三角形的基本性质
- 正弦定理
- 三角形解法

卷II：球面几何
- 球面三角基础

卷III：球面三角
- 球面正弦定理
- 球面三角形的解法

卷IV：天文应用
- 黄道坐标变换
- 升起与落下计算

卷V：更复杂的天文问题

```

**三角函数表**
- 制作精确的正弦表（间隔1'）
- 使用10位小数
- 为天文计算提供工具

### 3. 代数与方程

**三次、四次方程**
- 16世纪意大利数学家的竞争
- 卡尔达诺《大术》（1545年）
- 复数的首次系统使用

**复数的发现**

```

邦贝利的洞察：

方程 x³ = 15x + 4
实数解：x = 4

卡尔达诺公式给出：
x = ³√(2+√-121) + ³√(2-√-121)

邦贝利发现：
³√(2+√-121) = 2+√-1
³√(2-√-121) = 2-√-1

所以：
x = (2+√-1) + (2-√-1) = 4 ✓

这是复数代数有效性的第一个证明

```

**邦贝利《代数学》（1572年）**
- 系统处理复数运算
- 建立了复数的代数规则
- 认识到复数在解三次方程中的必要性

### 4. 对数的发明

**历史背景**
- 16世纪天文计算日益复杂
- 需要简化乘除运算
- 将乘法转化为加法

**纳皮尔（John Napier，1550-1617）**
- 苏格兰数学家
- 《奇妙的对数表的描述》（1614年）

**对数原理**

```

纳皮尔的对数定义：

设点P以速度v从A出发向B运动
点Q以速度v从C出发向无穷远运动
P的速度与PB成正比
Q保持匀速

当P到达某位置时，Q到达某位置
这两个位置的关系定义了对数

现代理解：
log(N) = 10⁷ × ln(10⁷/N)

```

**对数表**
- 纳皮尔计算了正弦的对数
- 间隔1'
- 大大简化了天文计算

**布里格斯（Henry Briggs，1561-1630）**
- 与纳皮尔合作改进对数
- 发展了常用对数（以10为底）
- 《对数算术》（1624年）

**对数的影响**

```

对数将：
- 乘法 → 加法
- 除法 → 减法
- 开方 → 除法
- 乘方 → 乘法

天文学家的福音：
开普勒："对数的发明延长了天文学家的寿命"

```

### 5. 代数符号化

**韦达（Viète，1540-1603）**
- 《分析术入门》（1591年）
- 系统使用代数符号
- 区分未知数和已知数

**符号演变**

```

韦达之前（修辞代数）：
"一个平方加八倍的东西等于二十"

韦达（缩写代数）：
"A quadratum + 8 A aequatur 20"

现代（符号代数）：
x² + 8x = 20

```

**韦达的重要贡献**
1. 元音表示未知数，辅音表示已知数
2. 系统使用+、-等运算符号
3. 用A、E等字母表示幂次
4. 将代数提升为"分析术"

### 6. 计算方法的改进

**斯蒂文（Simon Stevin，1548-1620）**
- 佛兰芒数学家、工程师
- 《十进算术》（1585年）
- 推广十进制小数

**小数表示**

```

斯蒂文的表示法：
3.1415 = 3⓪1①4②1③5④
或 3|1415

现代表示：
3.1415

斯蒂文主张：
"十进制的统一使用"
- 货币
- 度量衡
- 计算

```

## 影响分析

### 为近代数学奠基

1. **符号系统**
   - 代数符号化
   - 便于表达和操作
   - 为解析几何和微积分做准备

2. **计算方法**
   - 对数的发明
   - 小数的推广
   - 计算工具的发展

3. **几何与视觉**
   - 透视几何
   - 图形表示方法
   - 解析几何的先驱

4. **数的扩展**
   - 复数的发现
   - 代数基本定理的雏形
   - 数系概念的扩展

### 科学革命的数学准备

```

文艺复兴数学 → 科学革命

- 开普勒行星运动定律需要新的数学
- 伽利略落体定律需要函数概念
- 笛卡尔解析几何统一代数与几何
- 微积分的发明解决了变化率问题

```

### 数学教育的转变

**印刷术的影响**
- 欧几里得《几何原本》广泛传播
- 教科书的出现
- 数学知识普及

**语言的变化**
- 从拉丁语到各国语言
- 数学术语的标准化
- 国际数学交流

## 结语

文艺复兴时期的数学是连接中世纪和近代的桥梁。在这一时期，数学家们不仅解决了许多具体的技术问题（如透视法、对数计算），更重要的是，他们发展了新的数学思想和方法（符号代数、复数、坐标思想）。这些成就为17世纪的科学革命和数学革命铺平了道路，开启了人类认识自然的新纪元。
