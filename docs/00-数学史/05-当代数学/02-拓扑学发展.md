# 拓扑学发展

## 概述

### 拓扑学的本质

```

拓扑学研究在连续变形下保持不变的性质

"橡皮几何"：
- 拉伸、压缩、弯曲保持的性质
- 撕裂、粘合改变性质

基本问题：
- 两个空间是否同胚？
- 空间的基本结构是什么？

```

### 历史发展

```

18-19世纪：拓扑思想萌芽
  欧拉：七桥问题、多面体公式
  高斯：纽结、链环

1900-1940：组合拓扑
  庞加莱：基本群、同调
  布劳威尔：不动点定理

1940-1970：代数拓扑黄金时代
  同伦论、同调论、K理论
  示性类、谱序列

1970至今：低维拓扑与几何拓扑
  四维庞加莱猜想
  瑟斯顿几何化纲领
  规范场理论、量子拓扑

```

## 组合拓扑学的创立

### 庞加莱的贡献

**亨利·庞加莱（1854-1912）**

```

《位置分析》（1895年）

创立组合拓扑学（代数拓扑）

主要贡献：
- 基本群（第一同伦群）
- 贝蒂数、挠系数
- 庞加莱对偶
- 庞加莱猜想

```

**基本群**

```

π₁(X, x₀)：基于x₀的环路同伦类

运算：环路连接

例：
π₁(S¹) = Z（整数）
π₁(Sⁿ) = 0，n≥2
π₁(T²) = Z × Z（环面）

应用：
- 区分不同伦型的空间
- 覆叠空间理论
- 代数几何（代数基本定理）

```

**贝蒂数和欧拉示性数**

```

贝蒂数bₙ：n维洞的个数

b₀：连通分支数
b₁：一维洞（环路）数
b₂：二维洞（空腔）数

欧拉示性数：
χ = b₀ - b₁ + b₂ - ...

对于多面体：
χ = V - E + F

同胚的空间有相同的χ

```

**庞加莱对偶**

```

对n维闭流形M：
Hₖ(M) ≅ Hⁿ⁻ᵏ(M)

即：k维同调与(n-k)维上同调同构

几何意义：
k维洞与(n-k)维洞配对

例：三维球面S³
H₀ ≅ H³ ≅ Z
H₁ ≅ H₂ = 0

```

**庞加莱猜想**

```

原初形式（1900年）：
若三维闭流形同调平凡，
则同胚于S³

反例：
庞加莱构造了庞加莱同调球
同调平凡但非单连通

修正形式：
单连通的三维闭流形同胚于S³

这是七大千禧年难题之一
佩雷尔曼2003年证明

```

## 同调论的发展

### 单纯同调

**组合方法**

```

将空间分解为单形（点、线段、三角形、四面体...）

链：单形的形式和
边界算子：∂

同调群：
Hₙ = ker(∂ₙ) / im(∂_{n+1})

几何意义：
- 闭链：无边缘的循环
- 边缘链：某区域的边界
- 同调类：不同伦型的洞

```

### 奇异同调

**勒夫谢茨（1930年代）**

```

不依赖三角剖分

奇异单形：
从标准单形到空间的连续映射

奇异链：奇异单形的形式和

结果：
拓扑不变量
可计算性

```

### 上同调

**德·拉姆上同调**

```

用微分形式定义上同调

微分形式的外微分：d
闭形式：dω = 0
恰当形式：ω = dη

德·拉姆上同调群：
Hⁿ_{dR}(M) = {闭n形式}/{恰当n形式}

德·拉姆定理：
Hⁿ_{dR}(M) ≅ Hⁿ(M; R)

联系分析与拓扑

```

**层上同调**

```

格罗滕迪克（1955年）

层的上同调群Hⁿ(X, F)

应用：
- 代数几何
- 复分析
- 代数数论

塞尔对偶：
紧致复流形上的对偶定理

```

### 示性类

**斯蒂费尔-惠特尼类**

```

描述向量丛的扭曲

wᵢ ∈ Hⁱ(X; Z/2Z)

应用：
- 流形可定向性（w₁=0）
- 自旋结构（w₂=0）
- 浸入、嵌入问题

```

**庞特里亚金类和陈类**

```

庞特里亚金类：实向量丛
pᵢ ∈ H⁴ⁱ(X; Z)

陈类：复向量丛
cᵢ ∈ H²ⁱ(X; Z)

示性数：
示性类在基本类上的取值

拓扑不变量
与解析不变量联系（指标定理）

```

## 同伦论

### 同伦群

**高阶同伦群**

```

赫维茨（1935年）：

πₙ(X, x₀)：球面Sⁿ到X的映射同伦类

对于n≥2，πₙ是阿贝尔群

基本问题：
计算球面的同伦群πₙ(Sᵐ)

已知结果：
- πₙ(Sⁿ) = Z
- πₙ(S¹) = 0，n≥2
- π₃(S²) = Z（霍普夫纤维化）
- 一般情形极复杂

```

**霍普夫纤维化**

```

霍普夫（1931年）：

映射 S³ → S²
纤维：S¹

历史意义：
第一个非平凡的高阶同伦群

几何构造：
用复射影直线CP¹ ≅ S²
S³ ⊂ C²通过 (z₁,z₂) ↦ [z₁:z₂]

```

### 同伦论的工具

**纤维丛**

```

局部直积，整体可能扭曲

例子：
- 莫比乌斯带（圆上的线丛）
- 切丛、余切丛
- 主丛

示性类描述扭曲
同伦论研究分类

```

**谱序列**

```

勒雷（1946年）、塞缪尔·艾伦伯格：

计算同调/同伦的近似方法

逐阶逼近：
E² ⇒ E³ ⇒ ... ⇒ E∞

应用：
- 纤维丛的谱序列
- 德拉姆上同调的谱序列
- 稳定同伦群

```

### 稳定同伦论

**配边理论**

```

庞特里亚金、托姆：

配边：流形的等价关系
两个流形配边如果它们构成某流形的边界

配边环：
Ω_* = ⊕ Ωₙ

托姆定理：
配边环同构于某稳定同伦群

应用：
示性类的计算
存在性定理

```

**广义上同调**

```

艾伦伯格-麦克莱恩空间：
K(G,n)：πₙ = G，其他为0

布朗表示性定理：
广义上同调由谱表示

例子：
- K理论（格罗滕迪克、阿蒂亚-希策布鲁赫）
- 配边理论
- 稳定上同伦

```

## K理论

### 拓扑K理论

**阿蒂亚-希策布鲁赫（1961年）**

```

用向量丛构造广义上同调

K⁰(X)：X上向量丛的Grothendieck群

周期性：
K⁻ⁿ(X) ≅ K⁻ⁿ⁻²(X)

Bott周期性（1959年）：
πₖ(U) ≅ πₖ₊₂(U)
深刻联系代数与拓扑

```

**指标定理**

```

阿蒂亚-辛格（1963年）：

椭圆算子的解析指标 = 拓扑指标

解析指标：
dim ker D - dim coker D

拓扑指标：
用示性类表达

统一了：
- 黎曼-罗赫定理
- 希策布鲁赫符号差定理
- 指标定理的推广

```

### 代数K理论

**格罗滕迪克（1957年）**

```

从代数簇上的凝聚层出发

K₀：投射模的Grothendieck群

奎伦（1973年）：
高阶代数K理论
Kₙ(R)，n≥0

与数论的联系：
代数整数的K群
包含深刻算术信息

```

## 低维拓扑

### 纽结理论

**经典纽结理论**

```

琼斯多项式（1984年）：
V_L(t) ∈ Z[t^{±1/2}]

来自冯·诺伊曼代数
区分手性

 HOMFLY多项式：
推广了琼斯和阿历山大多项式

考夫曼括号：
状态求和模型

与统计力学的联系：
杨-巴克斯特方程

```

**纽结不变量的范畴化**

```

克霍万诺夫（2000年）：
琼斯多项式的同调理论

从多项式到分次同调群
更强的不变量

联系：
规范场理论
表示论
代数几何

```

### 三维流形

**瑟斯顿几何化纲领（1980年代）**

```

威廉·瑟斯顿（1946-2012）：

任何三维流形
可以分解为几何片

八种几何：
- S³, R³, H³（常曲率）
- S²×R, H²×R
- Nil, Sol
- SL(2,R)

庞加莱猜想的证明：
里奇流方法（汉密尔顿-佩雷尔曼）
是几何化的特例

```

**双曲三维流形**

```

大多数三维流形是双曲的

双曲结构：
局部像双曲空间H³

体积是拓扑不变量
瑟斯顿证明：
体积谱是良序的

```

### 四维流形

**唐纳森理论（1983年）**

```

西蒙·唐纳森：
四维流形的微分结构

杨-米尔斯方程的解：
瞬子（anti-self-dual联络）

模空间给出不变量

惊人结果：
R⁴上有不可数多个不同的光滑结构
（与其他维数不同！）

与物理学的深刻联系

```

**赛伯格-威滕理论（1994年）**

```

更简单的不变量

旋量方程的解

证明了唐纳森的一些结果
但更容易计算

物理学启发数学的典范

```

## 结语

拓扑学从庞加莱的位置分析发展到今天，已成为数学最丰富、最深奥的分支之一。从基本的同调和同伦理论到高层的代数K理论，从低维流形的几何化到四维流形的规范场理论，拓扑学不仅深刻改变了数学的面貌，也与物理学建立了深刻的联系。拓扑不变量的物理诠释、弦论的拓扑方法，展示了纯粹数学与理论物理之间美妙的相互作用。
