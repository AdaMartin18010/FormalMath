# 20世纪抽象代数结构

## 概述

### 抽象代数的兴起

```

20世纪初，代数学经历了一次彻底的抽象化

从研究具体对象（方程、数、矩阵）
转向研究抽象结构（群、环、域、模）

核心思想：
公理化方法 + 结构研究

布尔巴基学派的影响：
强调结构、统一和严格性

```

### 主要结构

```

群（Group）：一种运算，可逆
环（Ring）：两种运算（加、乘）
域（Field）：可除的环
模（Module）：环上的向量空间
格（Lattice）：偏序结构

```

## 群论的发展

### 有限群

**表示论**

```

弗罗贝尼乌斯（1896年）、伯恩赛德（1911年）：

用线性变换研究群
群的矩阵表示
特征标理论

伯恩赛德定理：
p^a q^b阶群是可解群
（p,q为素数）

```

**有限单群分类**

```

20世纪最宏大的数学工程
历时100多年，数百位数学家参与

分类定理（1981年完成）：
任何有限单群属于：
1. 素数阶循环群Z_p
2. 交错群A_n（n≥5）
3. 李型单群（16个无穷族）
4. 26个散在单群

散在单群：
- 最小的：M₁₁，阶7920
- 最大的：魔群（Fischer-Griess Monster）
  阶：2⁴⁶ × 3²⁰ × 5⁹ × 7⁶ × 11² × 13³ × 17 × 19 × 23 × 29 × 31 × 41 × 47 × 59 × 71
  ≈ 8 × 10⁵³

证明总页数：约10000页

```

## 环论与交换代数

### 诺特的理想理论

**埃米·诺特（1882-1935）**

```

"现代代数之母"

1921年《环中的理想论》：
抽象方法研究交换环

核心贡献：
- 升链条件（ACC）
  任何理想升链稳定

- 诺特环的性质
  理想有限生成

- 准素分解
  理想分解为准素理想的交

影响深远：
代数几何、代数数论的基础

```

**升链条件**

```

诺特环定义：
环R满足理想升链条件

等价于：
每个理想都是有限生成的

例：
- 域上的多项式环
- 整数环
- 诺特环的商环

非例：
多项式环k[x₁,x₂,x₃,...]（无穷多变量）

```

### 交换代数

**希尔伯特基定理**

```

若R是诺特环，则R[x]也是诺特环

推论：
k[x₁,...,xₙ]是诺特环
（k是域）

意义：
代数集由有限个多项式定义

```

**局部化**

```

模仿从整数构造有理数

对乘法封闭集S，
构造环S⁻¹R

例：
在素理想p处局部化R_p
在元素f处局部化R_f

代数几何中的关键操作

```

## 同调代数

### 起源

**拓扑学的影响**

```

同调群的代数化

从拓扑不变量到代数工具

1940-1950年代形成独立学科

```

### 基本概念

**链复形**

```

一系列模和映射：
... → C_{n+1} → C_n → C_{n-1} → ...
满足：∂ₙ ∘ ∂_{n+1} = 0

同调群：
H_n = ker(∂ₙ) / im(∂_{n+1})

```

**导出函子**

```

左正合函子的右导出：Ext, Tor

Extⁿ_R(M,N)：
测量模M,N之间的"扩张"

Torⁿ_R(M,N)：
张量积的导出函子

应用：
代数拓扑、代数几何、表示论

```

### 范畴论

**埃伦贝格和麦克莱恩（1945年）**

```

《自然等价的一般理论》

范畴论的诞生

基本思想：
不仅研究对象，更研究对象之间的关系（态射）

核心概念：
- 范畴（对象+态射）
- 函子（范畴间的映射）
- 自然变换（函子间的映射）

影响：
统一了数学各分支
提供了高层抽象语言

```

## 域论与伽罗瓦理论

### 伽罗瓦理论的现代形式

**基本定理**

```

伽罗瓦对应：

L/K有限伽罗瓦扩张
{中间域} ⟷ {Gal(L/K)的子群}

F ⟼ Gal(L/F)
L^H ⟼ H

这是反序同构

```

**逆伽罗瓦问题**

```

问题：
给定有限群G，是否存在伽罗瓦扩张L/Q
使得Gal(L/Q) ≅ G？

状态：
对可解群已解决
对单群部分解决
一般情形仍开放

```

### 代数闭域

**代数几何的基础**

```

代数闭域：每个多项式都有根

例：复数域C

希尔伯特零点定理：
k代数闭，I是k[x₁,...,xₙ]的理想
则V(I) = ∅ ⇔ I = (1)

联系代数与几何

```

### 结构理论


## 结语

20世纪的抽象代数彻底改变了数学的面貌。从诺特的理想理论到范畴论的高层抽象，从有限单群分类到非交换几何，代数学家们展示了抽象思维的力量。这些发展不仅深化了代数本身，也为代数几何、代数数论、拓扑学、物理学等领域提供了强大的工具。抽象代数的精神——研究结构而非具体对象——已成为现代数学的核心方法论。
