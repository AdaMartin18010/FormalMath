# 精细模空间

## 引言

模空间（moduli space）是代数几何中参数化同构类几何对象的空间。一个"好的"模空间不仅参数化对象，还应反映对象如何随参数变化——即对象的"族"。精细模空间（fine moduli space）正是这一理想的实现：它是可表函子（representable functor）的几何化身，存在泛族（universal family）使得任意其他族都由它经唯一 pullback 得到。

然而，并非所有模问题都有精细模空间。自同构群的存在常常阻碍泛族的存在，这时需要引入粗糙模空间（coarse moduli space）或叠（stack）的框架。本教程介绍精细模空间的定义、可表函子判据以及经典例子。

---

## 1. 可表函子

### 1.1 Yoneda引理与可表性

**定义 1.1（可表函子）**。设 $\mathcal{F}: \mathbf{Sch}^{\mathrm{op}} \to \mathbf{Set}$ 为函子。$\mathcal{F}$ 称为**可表的**（representable），如果存在概形 $M$ 和元素 $\xi \in \mathcal{F}(M)$（**泛对象**），使得对任意概形 $S$ 和 $x \in \mathcal{F}(S)$，存在唯一的态射 $f: S \to M$ 使得 $\mathcal{F}(f)(\xi) = x$。

等价地，存在自然同构 $h_M := \operatorname{Hom}(-, M) \xrightarrow{\cong} \mathcal{F}$。

**定理 1.2（Yoneda引理）**。自然变换 $h_M \to \mathcal{F}$ 一一对应于 $\mathcal{F}(M)$ 的元素。

### 1.2 泛族

**定义 1.3**。在上述可表情形，$M$ 上的对象 $\xi$ 称为**泛族**（universal family）。它使得任意 $S$ 上的族都是 $\xi$ 的 pullback。

## 2. 经典可表函子

### 2.1 Grassmann函子

**定义 2.1**。Grassmann函子 $\mathcal{G}(r, n)$ 定义为：对概形 $S$，

$$\mathcal{G}(r, n)(S) := \{V \subseteq \mathcal{O}_S^{\oplus n} \mid V \text{ 秩 } r \text{ 局部直和子丛}\}.$$

**定理 2.2**。$\mathcal{G}(r, n)$ 由 Grassmann 概形 $\operatorname{Gr}(r, n)$ 表示。

*证明*。经典Grassmann簇的泛性质直接推广到概形情形。$\square$

### 2.2 Hilbert函子

**定义 2.3**。固定射影概形 $X$ 和多项式 $P$。Hilbert函子 $\mathcal{Hilb}_{X}^{P}$ 定义为：

$$\mathcal{Hilb}_{X}^{P}(S) := \{Z \subseteq X \times S \mid Z \text{ 在 } S \text{ 上平坦，且每纤维的Hilbert多项式为 } P\}.$$

**定理 2.4（Grothendieck）**。$\mathcal{Hilb}_{X}^{P}$ 由射影概形 $\operatorname{Hilb}_{X}^{P}$ 表示。

## 3. 不可表的例子与障碍

### 3.1 自同构的障碍

**命题 3.1**。若模函子 $\mathcal{F}$ 中对象有非平凡自同构，则 $\mathcal{F}$ 通常不可表。

**例子 3.2**。椭圆曲线的模函子 $\mathcal{M}_{1,1}$：

$$\mathcal{M}_{1,1}(S) := \{E \to S \text{ 光滑椭圆曲线族}\} / \cong.$$

由于椭圆曲线有 $\mathbb{Z}/2$ 自同构（$[-1]$ 映射），此函子不可表。取而代之的是 $j$-不变量给出的粗糙模空间 $\mathbb{A}^1_j$（参见 [19-粗糙模空间](19-粗糙模空间.md)）或精细模叠 $\mathcal{M}_{1,1}$。
