# 代数几何具体例子与计算

## 例子1：仿射平面曲线

### 例子：$y^2 = x^3$（尖点三次曲线）

**奇点分析**：
- 偏导数：$\frac{\partial f}{\partial x} = -3x^2 = 0$，$\frac{\partial f}{\partial y} = 2y = 0$
- 在$(0,0)$处两者都为0，所以$(0,0)$是奇点
- 这是尖点（cusp）

**参数化**：$t \mapsto (t^2, t^3)$

**Python可视化**（概念性）：
```python
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-2, 2, 400)
y_pos = np.sqrt(x**3)
y_neg = -np.sqrt(x**3)

# 只绘制x>=0的部分（实数解）
x_valid = x[x >= 0]
y_pos_valid = np.sqrt(x_valid**3)
y_neg_valid = -np.sqrt(x_valid**3)

plt.figure(figsize=(8, 6))
plt.plot(x_valid, y_pos_valid, 'b-', linewidth=2)
plt.plot(x_valid, y_neg_valid, 'b-', linewidth=2)
plt.plot(0, 0, 'ro', markersize=10, label='Cusp singularity')
plt.xlabel('x')
plt.ylabel('y')
plt.title('Cubic Curve: $y^2 = x^3$')
plt.legend()
plt.grid(True)
plt.axis('equal')
plt.savefig('cusp_curve.png')
plt.show()
```

## 例子2：椭圆曲线

### $y^2 = x^3 - x$（亏格1曲线）

**性质**：
- 非奇异（判别式$\Delta = 64 \neq 0$）
- 有理点：$(0,0), (1,0), (-1,0)$
- 群结构：椭圆曲线上的点构成阿贝尔群

**Python可视化**：
```python
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-2, 2, 400)
# y^2 = x^3 - x
y_sq = x**3 - x

# 筛选y^2 >= 0的区域
x_valid = x[y_sq >= 0]
y_sq_valid = y_sq[y_sq >= 0]
y_pos = np.sqrt(y_sq_valid)
y_neg = -np.sqrt(y_sq_valid)

plt.figure(figsize=(10, 6))
plt.plot(x_valid, y_pos, 'b-', linewidth=2)
plt.plot(x_valid, y_neg, 'b-', linewidth=2)
plt.plot([0, 1, -1], [0, 0, 0], 'ro', markersize=8, label='Rational points')
plt.xlabel('x')
plt.ylabel('y')
plt.title('Elliptic Curve: $y^2 = x^3 - x$')
plt.legend()
plt.grid(True)
plt.axis('equal')
plt.savefig('elliptic_curve.png')
plt.show()
```

## 例子3：射影空间

### $\mathbb{P}^1$（射影直线）

**齐次坐标**：$[X : Y]$，$[X : Y] = [\lambda X : \lambda Y]$（$\lambda \neq 0$）

**仿射覆盖**：
- $U_0 = \{[X : Y] : X \neq 0\} \cong \mathbb{A}^1$（通过$[X : Y] \mapsto Y/X$）
- $U_1 = \{[X : Y] : Y \neq 0\} \cong \mathbb{A}^1$（通过$[X : Y] \mapsto X/Y$）

**无穷远点**：在$\mathbb{P}^1$中，点$[1 : 0]$对应仿射直线$\mathbb{A}^1$的无穷远点。

## 习题

### 习题1
判断曲线$y^2 = x^2(x+1)$的奇点。

**解答**：$f(x,y) = y^2 - x^2(x+1) = y^2 - x^3 - x^2$

偏导数：
$\frac{\partial f}{\partial x} = -3x^2 - 2x = -x(3x+2) = 0 \Rightarrow x = 0$或$x = -2/3$
$\frac{\partial f}{\partial y} = 2y = 0 \Rightarrow y = 0$

验证：
- $(0,0)$：$f(0,0) = 0$，是奇点（结点）
- $(-2/3, 0)$：$f(-2/3, 0) = 4/27 - 4/9 = -8/27 \neq 0$，不是曲线上的点

所以唯一奇点是$(0,0)$（结点）。

### 习题2
计算椭圆曲线$y^2 = x^3 + 1$上的点加法：$P = (0,1)$，$Q = (2,3)$。

**解答**：
弦斜率：$s = \frac{3-1}{2-0} = 1$

$R = P + Q$：
$x_R = s^2 - x_P - x_Q = 1 - 0 - 2 = -1$
$y_R = s(x_P - x_R) - y_P = 1(0 - (-1)) - 1 = 0$

所以$P + Q = (-1, 0)$。

验证：$(-1, 0)$在曲线上：$0^2 = (-1)^3 + 1 = 0$ ✓

### 习题3
证明：射影平面$\mathbb{P}^2$中任意两条不同直线必相交于一点。

**证明**：设两条直线为
$L_1: a_1X + b_1Y + c_1Z = 0$
$L_2: a_2X + b_2Y + c_2Z = 0$

这是关于$X, Y, Z$的齐次线性方程组。系数矩阵是$2 \times 3$，秩最多为2，所以解空间维数至少为1，即存在非零解$[X : Y : Z]$。

在$\mathbb{P}^2$中，这个解对应唯一的交点。∎
