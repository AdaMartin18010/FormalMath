# Stacks Project与概形理论的函子观点

## 1. 仿射概形的函子观点

**定义 1.1（可表函子）**. 函子 $F: \mathbf{Ring} \to \mathbf{Set}$ 可表，若存在环 $A$ 使 $F \cong h^A = \operatorname{Hom}(A, -)$。

**定理 1.2（Yoneda）**. $h^A$ 完全刻画 $A$：$\operatorname{Nat}(h^A, h^B) \cong \operatorname{Hom}(B, A)$。

## 2. 概形的函子定义

**定义 2.1（ Zariski 层）**. 函子 $F: \mathbf{Ring} \to \mathbf{Set}$ 为 Zariski 层，若对环的 Zariski 开覆盖满足粘接条件。

**定义 2.2（概形）**. 概形是局部可表的 Zariski 层 $F: \mathbf{Ring} \to \mathbf{Set}$。

## 3. 代数叠

**定义 3.1（纤维范畴）**. 范畴 $\mathcal{X}$ 配备函子 $p: \mathcal{X} \to \mathbf{Sch}$，对象在 $S$ 上的纤维为 $p^{-1}(S)$。

**定义 3.2（代数叠）**. 代数叠是满足下降条件的纤维范畴，且对角线态射和覆盖态射满足一定有限性条件。

## 4. 例子

### 4.1 例子：模空间作为叠

曲线模空间 $\mathcal{M}_g$ 是 Deligne-Mumford 叠（对 $g \geq 2$）。

### 4.2 例子：商叠

$[X/G]$：对象为 $G$-torsor $P \to S$ 和等变映射 $P \to X$。
