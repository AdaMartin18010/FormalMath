# 概形的粘接与一般概形定义

> **对标 Stacks Tags**: 01HS – 01HV
> **原始文献**: EGA I, §2; Hartshorne II.2; Vakil FOAG Ch. 5

---

## 1. 从仿射概形到一般概形：粘接的必要性

仿射概形 $\operatorname{Spec} A$ 虽然承载了全部的交换代数信息，但大多数几何对象（如射影空间 $\mathbb{P}^n$、椭圆曲线、Grassmann 簇）本身并不是仿射的。Grothendieck 的关键洞见是：

> **一般概形（scheme）是局部同构于仿射概形的局部环化空间。**

换言之，概形可以通过在一族仿射概形上指定"粘接数据"（gluing data）来构造。这与微分几何中流形（manifold）通过图册（atlas）粘接的构造完全类似，但代数的严格性要求更高。

---

## 2. 局部环化空间与概形的定义

### 2.1 局部环化空间

**定义 2.1**（Stacks Tag 01H6; Hartshorne II.2, p. 72）。
一个 **局部环化空间**（locally ringed space）是一个对 $(X, \mathcal{O}_X)$，其中：

- $X$ 是一个拓扑空间；
- $\mathcal{O}_X$ 是 $X$ 上的一个环层；
- 对每个点 $x \in X$，茎（stalk）$\mathcal{O}_{X, x}$ 是一个局部环，其唯一极大理想记为 $\mathfrak{m}_x \subset \mathcal{O}_{X, x}$。

**定义 2.2**（Stacks Tag 01HR; Hartshorne II.2, p. 74）。
一个局部环化空间 $(X, \mathcal{O}_X)$ 称为 **仿射概形**（affine scheme），如果它同构于某个 $(\operatorname{Spec} A, \widetilde{\mathcal{O}}_{\operatorname{Spec} A})$。

### 2.2 一般概形的定义

**定义 2.3**（Stacks Tag 01HS; Hartshorne II.2, p. 74; EGA I, 2.1.2）。
一个 **概形**（scheme）是一个局部环化空间 $(X, \mathcal{O}_X)$，满足：$X$ 的每一点 $x \in X$ 都有一个开邻域 $U \subset X$，使得限制 $(U, \mathcal{O}_X|_U)$ 是仿射概形。

**注记 2.4**。这个定义体现了几何与代数的对偶性：

- **几何侧**：$X$ 是一个可以用仿射开集覆盖的空间；
- **代数侧**：每个仿射开集 $U \cong \operatorname{Spec} A$ 对应一个环 $A = \mathcal{O}_X(U)$，而这些环通过限制映射在相交处"兼容"。

## 3. 概形的粘接：数据与泛性质

### 3.1 粘接数据

设 $\{X_i\}_{i \in I}$ 为一族概形。要在它们上面粘接出一个新的概形 $X$，需要以下数据：

**定义 3.1**（Stacks Tag 01HT; Hartshorne Ex. II.2.12）。
一组 **粘接数据**（gluing data）由以下要素构成：

1. 对每个 $i, j \in I$，给出 $X_i$ 中的开子概形 $U_{ij} \subset X_i$；
2. 给出同构 $\varphi_{ij}: U_{ij} \xrightarrow{\sim} U_{ji}$，满足：
   - **自反性**：$U_{ii} = X_i$，且 $\varphi_{ii} = \operatorname{id}_{X_i}$；
   - **对称性**：$\varphi_{ji} = \varphi_{ij}^{-1}$；
   - **余圈条件**（cocycle condition）：在 $U_{ij} \cap U_{ik}$ 上，$\varphi_{kj} \circ \varphi_{ij} = \varphi_{ik}$。

### 3.2 粘接定理

**定理 3.2**（概形的粘接定理; Stacks Tag 01HU; Hartshorne Ex. II.2.12）。
给定如上所述的粘接数据，存在一个概形 $X$ 和开浸入（open immersions）$\psi_i: X_i \hookrightarrow X$，使得：

1. $X = \bigcup_{i \in I} \psi_i(X_i)$；
2. $\psi_i(U_{ij}) = \psi_i(X_i) \cap \psi_j(X_j)$；
3. 在 $U_{ij}$ 上，$\psi_j \circ \varphi_{ij} = \psi_i$。

此外，$X$ 在同构意义下唯一满足上述泛性质。

**证明**.
证明分为三个步骤：构造拓扑空间、构造结构层、验证局部环化空间公理。

*步骤 1：拓扑空间的构造*。
令 $Y = \bigsqcup_{i \in I} X_i$ 为拓扑不交并。在 $Y$ 上定义等价关系 $\sim$：
$$x \in U_{ij} \subset X_i \sim \varphi_{ij}(x) \in U_{ji} \subset X_j.$$
由自反性、对称性和余圈条件，$\sim$ 确实是等价关系。令 $X = Y / \sim$，赋予商拓扑。自然映射 $\psi_i: X_i \to X$ 是开映射（因为粘接同构是开映射的复合），因此 $X$ 被 $\{\psi_i(X_i)\}$ 覆盖。

*步骤 2：结构层的构造*。
在 $Y$ 上，每个 $X_i$ 已有结构层 $\mathcal{O}_{X_i}$。我们需要在 $X$ 上定义层 $\mathcal{O}_X$，使得 $\psi_i^*(\mathcal{O}_X) \cong \mathcal{O}_{X_i}$。对 $X$ 中的开集 $V$，定义
$$\mathcal{O}_X(V) := \left\{(s_i) \in \prod_i \mathcal{O}_{X_i}(\psi_i^{-1}(V)) \;\middle|\; \forall i, j,\; s_i|_{U_{ij} \cap \psi_i^{-1}(V)} = \varphi_{ij}^\#(s_j|_{U_{ji} \cap \psi_j^{-1}(V)})\right\}.$$
这里 $\varphi_{ij}^\#$ 是同构 $\varphi_{ij}$ 在结构层上诱导的层同构的反向映射。由层的粘接性质，$\mathcal{O}_X$ 是 $X$ 上的环层。

*步骤 3：局部环化空间的验证*。
对 $x \in X$，设 $x = \psi_i(x_i)$ 对某个 $x_i \in X_i$。则
$$\mathcal{O}_{X, x} = \varinjlim_{V \ni x} \mathcal{O}_X(V) \cong \varinjlim_{U \ni x_i} \mathcal{O}_{X_i}(U) = \mathcal{O}_{X_i, x_i},$$
因为 $\psi_i$ 是开浸入。由于 $X_i$ 是局部环化空间，$\mathcal{O}_{X_i, x_i}$ 是局部环，故 $\mathcal{O}_{X, x}$ 也是局部环。

*唯一性*：由构造的泛性质，任意满足条件 (1)-(3) 的概形 $X'$ 都必须与 $X$ 典范同构，因为存在唯一的态射 $X \to X'$ 和 $X' \to X$ 使得所有三角图交换。 ∎

## 4. 典型例子：射影直线的粘接

**例 4.1**（射影直线 $\mathbb{P}^1_k$; Stacks Tag 01HV; Hartshorne I.1; EGA II, 4.1）。
设 $k$ 为域。射影直线可以通过两个仿射直线 $U_0 = \operatorname{Spec} k[x]$ 和 $U_1 = \operatorname{Spec} k[y]$ 粘接而成：

- 取 $U_{01} = D(x) = \operatorname{Spec} k[x, x^{-1}] \subset U_0$；
- 取 $U_{10} = D(y) = \operatorname{Spec} k[y, y^{-1}] \subset U_1$；
- 粘接同构 $\varphi_{01}: U_{01} \to U_{10}$ 由环同态
  $$k[y, y^{-1}] \longrightarrow k[x, x^{-1}], \qquad y \longmapsto x^{-1}$$
  诱导。

由定理 3.2，这唯一地粘接出一个概形 $\mathbb{P}^1_k$。

**几何解释**：

- $U_0 = \operatorname{Spec} k[x]$ 对应"有限远点"的坐标卡；
- $U_1 = \operatorname{Spec} k[y]$ 对应"无穷远点"附近的坐标卡；
- 粘接 $y = x^{-1}$ 说明两个坐标卡在非零点处是一致的。

这与复分析中黎曼球 $\mathbb{C} \cup \{\infty\}$ 的构造完全一致，但发生在纯粹的代数框架中。

## 5. 与流形理论的比较

| 特征 | 微分流形 | 概形 |
|------|---------|------|
| 局部模型 | $\mathbb{R}^n$ 中的开集 | $\operatorname{Spec} A$（仿射概形） |
| 坐标变换 | 光滑映射 $C^\infty$ | 环同态诱导的 LRS 态射 |
| 粘接条件 | 光滑同胚 | 概形同构（带层结构） |
| 整体不变量 | de Rham 上同调 | 层上同调 $H^i(X, \mathcal{F})$ |
| 函数环 | $C^\infty(M)$ | 整体截面环 $\Gamma(X, \mathcal{O}_X)$ |

**关键差异**：流形的局部模型是"平坦的"（欧氏空间），而概形的局部模型 $\operatorname{Spec} A$ 可以非常"崎岖"（如 nilpotent 元、非约化结构、奇点）。这使得概形能够 encode 比流形更丰富的几何和算术信息。

## 6. Lean4 形式化片段

Mathlib4 的 `AlgebraicGeometry` 模块已实现了概形的基本定义。以下代码展示了如何通过 `Scheme` 类型类和 `OpenImmersion` 来形式化粘接构造的泛性质：

```lean
import Mathlib.AlgebraicGeometry.Scheme
import Mathlib.AlgebraicGeometry.OpenImmersion

universe u v

namespace SchemeGluing

open AlgebraicGeometry CategoryTheory

-- 概形的定义检查
#check Scheme  -- 局部环化空间，且每点有仿射开邻域

-- 开浸入的定义
#check OpenImmersion  -- 概形态射，拓扑上是开嵌入，层映射是同构

-- 射影直线 P¹ 的构造（Mathlib4 中已实现）
#check ProjectiveSpace.Scheme 1

-- 粘接数据的泛性质（概念性框架）
structure GluingData (I : Type u) where
  U : I → Scheme.{v}
  V : ∀ i j, Opens (U i)
  f : ∀ i j, (V i j).toScheme ⟶ (V j i).toScheme
  cocycle : ∀ i j k,
    (f i j).restrict _ _ ≫ (f j k).restrict _ _ = (f i k).restrict _ _

end SchemeGluing
```

**说明**：Mathlib4 目前对概形粘接的直接 API 仍在完善中。`ProjectiveSpace` 的实现采用了 Proj 构造（`Spec` 的相对版本），这是比直接粘接更一般的构造方法。

## 7. 习题

**习题 7.1**（Hartshorne II.2.12）。验证例 4.1 中的 $\mathbb{P}^1_k$ 粘接确实满足余圈条件：在 $U_0 \cap U_1 \cap U_0 = U_{01}$ 上，$\varphi_{10} \circ \varphi_{01} = \operatorname{id}$，以及更高阶的相容性。

**习题 7.2**（Stacks Tag 01HV）。设 $X_1 = \operatorname{Spec} k[x]$，$X_2 = \operatorname{Spec} k[y]$，$U_{12} = D(x) \subset X_1$，$U_{21} = D(y) \subset X_2$，粘接同构由 $y \mapsto x$ 给出。证明粘接后的概形 $X$ 同构于去掉一点（原点）的仿射平面 $\mathbb{A}^2_k \setminus \{(0,0)\}$。这与例 4.1 有何本质区别？

**习题 7.3**（EGA I, 2.3.2）。证明：若概形 $X$ 有两个仿射开覆盖 $\{U_i = \operatorname{Spec} A_i\}$ 和 $\{V_j = \operatorname{Spec} B_j\}$，则对任意开集 $W \subset X$，
$$\mathcal{O}_X(W) = \varprojlim_i A_i = \varprojlim_j B_j,$$
即整体截面环不依赖于仿射覆盖的选择。

**习题 7.4**（综合）。设 $k$ 为代数闭域。在 $\mathbb{P}^1_k$ 中，点 $[1:0]$（即 $U_1$ 中的 $y=0$）对应于哪个素理想？请分别在 $U_0 = \operatorname{Spec} k[x]$ 和 $U_1 = \operatorname{Spec} k[y]$ 的坐标下描述该点的局部环。

## 8. 总结

本节完成了从仿射概形到一般概形的关键跨越：

1. **概形的定义**：局部同构于仿射概形的局部环化空间（Stacks Tag 01HS）；
2. **粘接定理**：给定满足余圈条件的粘接数据，存在唯一概形实现粘接（Stacks Tag 01HT–01HU）；
3. **射影直线 $\mathbb{P}^1_k$**：通过两个仿射直线的粘接，展示了非仿射概形的具体构造（Stacks Tag 01HV）。

这一理论与 EGA I §2 完全一致，并为后续讨论层上同调、模空间、以及 Stacks Project 中更深层的内容奠定了基础。
