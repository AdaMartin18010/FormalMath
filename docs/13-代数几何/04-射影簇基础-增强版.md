## 13.4.1 引言

射影簇是代数几何中最基本也是最重要的研究对象之一。与仿射簇相比，射影簇具有更好的紧性性质，使得许多在仿射情形下不成立的定理在射影情形下得以成立。射影几何的起源可以追溯到文艺复兴时期的透视画法，而其严格的代数基础则由19世纪的数学家们建立。

**The foundations of projective varieties are among the most fundamental and important objects of study in algebraic geometry. Compared to affine varieties, projective varieties possess better compactness properties, enabling many theorems that fail in the affine case to hold in the projective setting.**

## 13.4.2 射影空间的基础理论

### 射影空间的定义

**定义 13.4.1** (射影空间 / Projective Space)

设 $k$ 是一个域，$n$ 是非负整数。$n$ 维射影空间 $\mathbb{P}^n(k)$ 定义为 $k^{n+1}$ 中所有一维子空间的集合：

$$\mathbb{P}^n(k) = \frac{k^{n+1} \setminus \{0\}}{\sim}$$

其中等价关系 $\sim$ 定义为：$(x_0, \ldots, x_n) \sim (\lambda x_0, \ldots, \lambda x_n)$ 对所有 $\lambda \in k^\times$ 成立。

齐次坐标 $[x_0 : x_1 : \cdots : x_n]$ 表示 $\mathbb{P}^n$ 中的点。

### 标准仿射覆盖

**命题 13.4.1** (射影空间的标准仿射覆盖)

射影空间 $\mathbb{P}^n$ 可以被 $n+1$ 个仿射开集覆盖：

$$U_i = \{[x_0 : \cdots : x_n] \in \mathbb{P}^n : x_i \neq 0\} \cong \mathbb{A}^n$$

对于每个 $i$，同构 $\varphi_i: U_i \to \mathbb{A}^n$ 定义为：
$$\varphi_i([x_0 : \cdots : x_n]) = \left(\frac{x_0}{x_i}, \ldots, \frac{x_{i-1}}{x_i}, \frac{x_{i+1}}{x_i}, \ldots, \frac{x_n}{x_i}\right)$$

**证明**: 需要验证 $\varphi_i$ 是双射且双有理。

1. **单射性**: 若 $\varphi_i([x]) = \varphi_i([y])$，则对所有 $j \neq i$ 有 $x_j/x_i = y_j/y_i$，因此 $[x] = [y]$。

2. **满射性**: 对任意 $(a_0, \ldots, a_{i-1}, a_{i+1}, \ldots, a_n) \in \mathbb{A}^n$，点 $[a_0 : \cdots : a_{i-1} : 1 : a_{i+1} : \cdots : a_n]$ 映射到它。

3. **过渡映射**: 在 $U_i \cap U_j$ 上，过渡映射 $\varphi_j \circ \varphi_i^{-1}$ 是有理函数，因此是双有理映射。

### Zariski拓扑

**定义 13.4.2** (射影空间中的Zariski拓扑)

$\mathbb{P}^n$ 上的Zariski拓扑由以下闭集基生成：
$$V(f_1, \ldots, f_m) = \{[x] \in \mathbb{P}^n : f_i(x_0, \ldots, x_n) = 0 \text{ 对所有 } i\}$$

其中 $f_1, \ldots, f_m$ 是齐次多项式。

## 13.4.3 射影簇的齐次坐标环

### 齐次理想的定义

**定义 13.4.3** (齐次理想 / Homogeneous Ideal)

多项式环 $S = k[x_0, \ldots, x_n]$ 的一个理想 $I$ 称为齐次的，如果它由齐次多项式生成，等价地，如果：
$$I = \bigoplus_{d \geq 0} (I \cap S_d)$$

其中 $S_d$ 是 $d$ 次齐次多项式构成的子空间。

### 射影簇与齐次理想

**定理 13.4.1** (射影零点定理 / Projective Nullstellensatz)

设 $k$ 是代数闭域，$I \subseteq S = k[x_0, \ldots, x_n]$ 是齐次理想。

(i) 若 $V(I) = \emptyset$，则存在 $d > 0$ 使得 $S_d \subseteq I$。

(ii) 若 $V(I) \neq \emptyset$，则 $I(V(I)) = \sqrt{I}$。

**证明**: 这是仿射零点定理的射影版本。

对于(i)：若 $V(I) = \emptyset$，则在仿射空间 $\mathbb{A}^{n+1}$ 中，$V^a(I) \subseteq \{0\}$。由仿射零点定理，$\sqrt{I} \supseteq (x_0, \ldots, x_n)$。因此存在 $m$ 使得 $x_i^m \in I$ 对所有 $i$，于是 $S_{m(n+1)} \subseteq I$。

对于(ii)：由于 $V(I) \neq \emptyset$，有 $I(V(I)) = I(V^a(I)) = \sqrt{I}$。

### 齐次坐标环

**定义 13.4.4** (齐次坐标环 / Homogeneous Coordinate Ring)

设 $X \subseteq \mathbb{P}^n$ 是射影簇，其齐次坐标环定义为：
$$S(X) = \frac{k[x_0, \ldots, x_n]}{I(X)}$$

这是一个分次环，继承 $S$ 的分次结构。

## 13.4.4 射影态射与线性系统

### 射影态射的定义

**定义 13.4.5** (射影态射 / Projective Morphism)

设 $X$ 是概形。一个射影态射 $f: X \to Y$ 是满足以下条件的态射：存在局部自由层 $\mathcal{E}$ 在 $Y$ 上使得 $f$ 分解为闭浸入 $X \hookrightarrow \mathbb{P}(\mathcal{E})$ 后接投影 $\mathbb{P}(\mathcal{E}) \to Y$。

### 线性系统

**定义 13.4.6** (线性系统 / Linear System)

设 $X$ 是射影簇，$\mathcal{L}$ 是 $X$ 上的线丛。与 $\mathcal{L}$ 相关的完全线性系统定义为：
$$|\mathcal{L}| = \mathbb{P}(H^0(X, \mathcal{L}))$$

一个线性系统是 $|\mathcal{L}|$ 的线性子空间。

## 13.4.5 射影簇的维数与次数

### 射影簇的维数

**定义 13.4.7** (射影簇的维数)

射影簇 $X \subseteq \mathbb{P}^n$ 的维数定义为其作为拓扑空间的维数，等于其函数域在 $k$ 上的超越次数。

### Hilbert多项式与次数

**定理 13.4.2** (Hilbert-Serre定理)

设 $X \subseteq \mathbb{P}^n$ 是射影簇，$S(X)$ 是其齐次坐标环。Hilbert函数定义为：
$$H_X(d) = \dim_k S(X)_d$$

存在唯一的多项式 $P_X(t) \in \mathbb{Q}[t]$（称为Hilbert多项式），使得对于充分大的 $d$，有 $H_X(d) = P_X(d)$。

**定义 13.4.8** (射影簇的次数 / Degree)

设 $X \subseteq \mathbb{P}^n$ 是 $r$ 维射影簇。$X$ 的次数定义为：
$$\deg(X) = r! \cdot (P_X \text{ 的首项系数})$$

## 13.4.6 详细示例：射影曲线的分类

### 例：平面射影曲线

**示例 13.4.1** (Fermat曲线的射影完备化)

考虑仿射Fermat曲线 $C^a: x^n + y^n = 1$ 在 $\mathbb{A}^2$ 中。其射影完备化定义为：
$$C: x^n + y^n = z^n$$
在 $\mathbb{P}^2$ 中，其中 $[x : y : z]$ 是齐次坐标。

**详细分析**:

1. **无穷远点**: 当 $z = 0$ 时，有 $x^n + y^n = 0$，即 $[1 : \zeta : 0]$，其中 $\zeta^n = -1$。共有 $n$ 个无穷远点。

2. **亏格计算**: 由亏格公式，光滑平面 $n$ 次曲线的亏格为：
$$g = \frac{(n-1)(n-2)}{2}$$

3. **验证光滑性**: 梯度向量 $\nabla F = (nx^{n-1}, ny^{n-1}, -nz^{n-1})$。在特征不整除 $n$ 的域上，$C$ 是光滑的。

**具体计算（$n=3$ 情形）**:

对于 $n=3$（Fermat三次曲线）：
- 齐次方程：$x^3 + y^3 = z^3$
- 无穷远点：$[1 : \zeta : 0]$，其中 $\zeta^3 = -1$，即 $\zeta = -1, \frac{1}{2}(1 \pm i\sqrt{3})$
- 亏格：$g = \frac{2 \cdot 1}{2} = 1$（椭圆曲线）

### 例：Veronese嵌入

**示例 13.4.2** (二次Veronese嵌入)

Veronese嵌入 $v_2: \mathbb{P}^2 \hookrightarrow \mathbb{P}^5$ 定义为：
$$[x : y : z] \mapsto [x^2 : y^2 : z^2 : xy : xz : yz]$$

设齐次坐标为 $[X_0 : X_1 : X_2 : X_3 : X_4 : X_5]$，像满足方程组：
$$\begin{cases}
X_0 X_1 = X_3^2 \\
X_0 X_2 = X_4^2 \\
X_1 X_2 = X_5^2 \\
X_3 X_4 = X_0 X_5 \\
X_3 X_5 = X_1 X_4 \\
X_4 X_5 = X_2 X_3
\end{cases}$$

这是一个2维射影簇，次数为4（计算 $\mathbb{P}^2$ 中一般二次曲线的像）。

## 13.4.7 形式化实现

```lean
-- 射影簇基础的形式化实现 (Lean 4风格)
structure ProjectiveSpace (k : Type*) [Field k] (n : ℕ) where
  carrier : Type*
  equiv : carrier ≃ {v : Fin (n+1) → k // v ≠ 0} / (Units k • ·)

-- 射影簇的定义
def ProjectiveVariety (k : Type*) [Field k] {n : ℕ} 
    (I : HomogeneousIdeal (MvPolynomial (Fin (n+1)) k)) : 
    Set (ProjectiveSpace k n) := 
  {P | ∀ f ∈ I.homogeneous_parts, homogeneousEval f P = 0}

-- Veronese嵌入的形式化
def VeroneseEmbedding {k : Type*} [Field k] (n d : ℕ) :
    ProjectiveSpace k n → ProjectiveSpace k (Nat.choose (n+d) d - 1) :=
  fun P => ⟨monomialsOfDegree d P⟩
```
