# 黎曼-罗赫定理

## 引言

黎曼-罗赫定理（Riemann-Roch theorem）是代数曲线理论中最深刻、最有影响力的定理之一。它建立了曲线上线丛（或除子）的数值不变量——次数（degree）——与其上同调维数之间的精确关系。这一定理由Bernhard Riemann在1857年提出初步形式，随后Gustav Roch在1865年补齐了余项，形成了完整的定理。

Riemann-Roch定理不仅是曲线分类的基石（通过它可确定线性系的维数、判断 Very Ampleness、嵌入射影空间），也是更高维Riemann-Roch-Grothendieck定理和指标定理的原型。在模空间理论中，Riemann-Roch公式是计算Picard群和相交数的起点。

本教程详细介绍曲线上的Riemann-Roch定理，包括多种证明思路、丰富例子和经典应用。

---

## 1. 除子与线丛的回顾

### 1.1 Weil除子

**定义 1.1**。设 $C$ 为光滑射影曲线。$C$ 上的**Weil除子**（Weil divisor）是有限形式和

$$D = \sum_{P \in C} n_P \cdot P, \quad n_P \in \mathbb{Z}, \, n_P = 0 \text{ 对几乎所有 } P.$$

$D$ 的**次数**定义为 $\deg D := \sum n_P$。若所有 $n_P \geq 0$，称 $D$ 为**有效除子**（effective divisor），记 $D \geq 0$。

除子群 $\operatorname{Div}(C)$ 配备次数同态 $\deg: \operatorname{Div}(C) \to \mathbb{Z}$。

### 1.2 主除子与线性等价

**定义 1.2**。对非零有理函数 $f \in k(C)^*$，其**主除子**为

$$\operatorname{div}(f) := \sum_{P \in C} v_P(f) \cdot P,$$

其中 $v_P$ 为 $P$ 处的离散赋值。主除子构成子群 $\operatorname{Prin}(C) \subseteq \operatorname{Div}(C)$。

**定义 1.3**。除子 $D, D'$ 称为**线性等价**（linearly equivalent），记 $D \sim D'$，如果 $D - D' = \operatorname{div}(f)$ 对某个 $f \in k(C)^*$。**除子类群** $\operatorname{Cl}(C) := \operatorname{Div}(C) / \operatorname{Prin}(C)$。

**命题 1.4**。对光滑射影曲线，$\operatorname{Cl}(C) \cong \operatorname{Pic}(C)$（线丛的Picard群），对应关系为 $D \mapsto \mathcal{O}_C(D)$。

## 2. Riemann-Roch定理的陈述

### 2.1 核心定理

**定理 2.1（Riemann-Roch）**。设 $C$ 为光滑射影曲线，亏格为 $g$。对任意除子 $D \in \operatorname{Div}(C)$，

$$h^0(C, \mathcal{O}_C(D)) - h^1(C, \mathcal{O}_C(D)) = \deg D + 1 - g,$$

其中 $h^i := \dim_k H^i(C, \mathcal{O}_C(D))$。

等价地，利用Serre对偶 $H^1(C, \mathcal{O}_C(D)) \cong H^0(C, \mathcal{O}_C(K_C - D))^*$（参见 [06-层上同调基本定理](06-层上同调基本定理.md)），其中 $K_C$ 为典范除子：

$$h^0(D) - h^0(K_C - D) = \deg D + 1 - g.$$

### 2.2 Riemann不等式与Roch的补偿

**推论 2.2（Riemann不等式）**。对任意除子 $D$，$h^0(D) \geq \deg D + 1 - g$。

*证明*。由 $h^0(K_C - D) \geq 0$ 直接得到。$\square$

Riemann原始证明给出了这一不等式。Roch的贡献是识别出余项 $h^0(K_C - D)$，从而将不等式提升为等式。

---

## 3. 证明思路

### 3.1 归纳法证明

*证明概要*（归纳于 $\deg D$）。

**基础**：$D = 0$。则 $h^0(0) = 1$（常值函数），$h^1(0) = g$（由Serre对偶，$h^1(0) = h^0(K_C) = g$）。等式为 $1 - g = 0 + 1 - g$，成立。

**归纳步骤**：设 $D$ 为任意除子，$P \in C$ 为点。考虑短正合列

$$0 \longrightarrow \mathcal{O}_C(D) \longrightarrow \mathcal{O}_C(D+P) \longrightarrow k(P) \longrightarrow 0,$$

其中 $k(P)$ 为在 $P$ 处支撑的摩天大楼层。上同调长正合列给出

$$0 \to H^0(D) \to H^0(D+P) \to k \to H^1(D) \to H^1(D+P) \to 0.$$

由秩-零化度定理，$h^0(D+P) - h^1(D+P) = h^0(D) - h^1(D) + 1$。由于右侧由归纳假设等于 $\deg D + 1 - g + 1 = \deg(D+P) + 1 - g$，归纳完成。$\square$

### 3.2 利用Serre对偶的简洁证明

**证明**（基于Serre对偶）。考虑Euler示性数 $\chi(D) := h^0(D) - h^1(D)$。上述归纳实际上证明了 $\chi(D) = \deg D + \chi(0)$。而 $\chi(0) = 1 - g$，故定理成立。

Serre对偶的作用是将 $h^1(D)$ 转化为 $h^0(K_C - D)$，从而得到对称形式的Riemann-Roch公式。$\square$

## 4. 经典应用

### 4.1 亏格计算

**命题 4.1**。设 $C \subseteq \mathbb{P}^2$ 为 $d$ 次光滑平面曲线。则

$$g = \frac{(d-1)(d-2)}{2}.$$

*证明*。由adjunction公式，$K_C = (d-3)H|_C$，故 $\deg K_C = d(d-3)$。由Riemann-Roch对 $D = K_C$：

$$h^0(K_C) - h^0(0) = d(d-3) + 1 - g,$$

即 $g - 1 = d(d-3) + 1 - g$，解得 $g = (d-1)(d-2)/2$。$\square$

### 4.2 超椭圆曲线判定

**定义 4.2**。亏格 $g \geq 2$ 的曲线 $C$ 称为**超椭圆的**（hyperelliptic），如果存在二重覆盖 $C \to \mathbb{P}^1$。

**命题 4.3**。$C$ 超椭圆当且仅当存在 $g^1_2$（次数 $2$、维数 $1$ 的线性系），等价于 $|K_C|$ 不是 very ample。

*证明要点*。若 $C \to \mathbb{P}^1$ 为二重覆盖，拉回 $

$\mathbb{P}^1$ 上的点给出一个 $g^1_2$。反之，$g^1_2$ 给出到 $\mathbb{P}^1$ 的态射。由Riemann-Roch，$h^0(K_C) = g$，且 $K_C$ 是非常丰沛的当且仅当 $C$ 非超椭圆。$\square$

## 5. 重要例子

### 例子 1：椭圆曲线

**例 5.1**。设 $E$ 为椭圆曲线（$g = 1$）。对任意点 $P \in E$，Riemann-Roch给出：

$$h^0(P) - h^0(K_E - P) = 1 + 1 - 1 = 1.$$

由于 $\deg K_E = 0$（由 $\deg K_C = 2g - 2$），$K_E \sim 0$，故 $h^0(K_E - P) = h^0(-P) = 0$。因此 $h^0(P) = 1$，即存在唯一（差常数倍）的亚纯函数以 $P$ 为单极点。这给出了 $E$ 的群结构（参见 [11-椭圆曲线群结构](11-椭圆曲线群结构.md)）。

### 例子 2：亏格2曲线

**例 5.2**。设 $C$ 为亏格 $2$ 曲线。则 $\deg K_C = 2$。

- $h^0(K_C) = 2$，故 $|K_C|$ 给出 $C \to \mathbb{P}^1$ 的二重覆盖，即所有亏格 $2$ 曲线都是超椭圆的。
- 对一般点 $P$，$h^0(P) = 1$（仅有常值函数），$h^0(2P) = 2$（由Riemann-Roch：$2 + 1 - 2 = 1$，加上常值共 $2$）。

### 例子 3：典范嵌入

**例 5.3**。设 $C$ 为非超椭圆曲线，$g \geq 3$。则 $|K_C|$ 是 very ample，给出**典范嵌入**

$$\phi_{K_C}: C \hookrightarrow \mathbb{P}^{g-1}.$$

*验证*。由Riemann-Roch，$h^0(K_C) = g$，故映射到 $\mathbb{P}^{g-1}$。Very ampleness 需要验证对任意 $P, Q \in C$（可相同），$h^0(K_C - P - Q) = g - 2$。由Riemann-Roch：

$$h^0(K_C - P - Q) - h^0(P + Q) = 2g - 4 + 1 - g = g - 3.$$

若 $h^0(P+Q) = 1$（无 $g^1_2$），则 $h^0(K_C - P - Q) = g - 2$，恰为very ample条件。$\square$

## 练习

1. 对椭圆曲线 $E$ 上的除子 $D$，证明若 $\deg D > 0$ 则 $h^0(D) = \deg D$，并由此推导 $E$ 上任意次数 $d > 0$ 的线丛给出一个 $d:1$ 映射到 $\mathbb{P}^{d-1}$。
2. 证明亏格 $g=3$ 的光滑曲线要么是平面四次曲线（非超椭圆），要么是超椭圆的。
3. 计算 $h^0(nK_C)$ 对所有 $n \geq 0$，并讨论其增长速率（pluricanonical map）。
