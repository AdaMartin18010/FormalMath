# 直纹面与有理曲面

## 引言

曲面的双有理分类是代数几何的里程碑成就之一。与曲线不同，每个双有理等价类中包含无穷多条不同构的光滑射影曲面。然而，通过对曲面的极小模型（minimal model）的研究，Enriques、Castelnuovo、Kodaira等人建立了系统的分类理论。

在这个分类体系中，**有理曲面**（rational surface）和**直纹面**（ruled surface）占据基础地位。有理曲面双有理等价于 $\mathbb{P}^2$（或 $\mathbb{P}^1 \times \mathbb{P}^1$），而直纹面则是曲线上的 $\mathbb{P}^1$-丛。Blow-up和blow-down操作是连接同一双有理等价类中不同曲面的基本工具。

本教程介绍直纹面和有理曲面的结构、分类以及它们在曲面理论中的角色。

---

## 1. 直纹面

### 1.1 定义与结构

**定义 1.1（直纹面）**。曲面 $S$ 称为**直纹面**（ruled surface），如果它双有理等价于 $C \times \mathbb{P}^1$，其中 $C$ 为光滑曲线。等价地，$S$ 容许一个到曲线 $C$ 的满射 $p: S \to C$，使得一般纤维同构于 $\mathbb{P}^1$。

**定理 1.2**。任何直纹面 $p: S \to C$ 是 $C$ 上的射影丛：$S \cong \mathbb{P}(\mathcal{E})$，其中 $\mathcal{E}$ 是 $C$ 上的秩 $2$ 局部自由层（向量丛）。

*证明*。由Tsen定理（或Noether-Enriques定理），规则纤维的曲面在其一般纤维上给出 $k(C)$ 上的圆锥曲线，而 $k(C)$ 是 $C_1$ 域，故该圆锥曲线有有理点，从而同构于 $\mathbb{P}^1_{k(C)}$。这意味着 $S$  birational 于 $C \times \mathbb{P}^1$，再通过相对极小模型的分析得到 $S = \mathbb{P}(\mathcal{E})$。$\square$

### 1.2 不变量

**命题 1.3**。设 $S = \mathbb{P}(\mathcal{E})$，$C$ 亏格 $g$。
- $q(S) = h^1(S, \mathcal{O}_S) = g$。
- $p_g(S) = 0$。
- $K_S^2 = 8(1 - g)$。

## 2. 有理曲面

### 2.1 定义与例子

**定义 2.1（有理曲面）**。曲面 $S$ 称为**有理的**（rational），如果它双有理等价于 $\mathbb{P}^2$。

**例子 2.2**。有理曲面包括：
- $\mathbb{P}^2$ 本身。
- $\mathbb{P}^1 \times \mathbb{P}^1$（Hirzebruch曲面 $F_0$）。
- Hirzebruch曲面 $F_n = \mathbb{P}(\mathcal{O}_{\mathbb{P}^1} \oplus \mathcal{O}_{\mathbb{P}^1}(n))$（$n \geq 0$）。
- $\mathbb{P}^2$ 在有限个点的 blow-up。

### 2.2 Castelnuovo有理性判据

**定理 2.3（Castelnuovo）**。光滑射影曲面 $S$ 是有理的当且仅当 $q(S) = p_g(S) = P_2(S) = 0$，其中 $P_2 = h^0(2K_S)$ 为二重亏格。

*证明概要*。必要性：有理曲面的双有理不变量 $q, p_g$ 为零，且 $P_2$ 也是双有理不变量。

充分性：由 $q = p_g = 0$ 和Noether公式，$K_S^2 + e(S) = 12$。由 $P_2 = 0$，$|2K_S| = \varnothing$，故 $K_S$ 不是伪有效的。通过曲线锥的分析，可以找到使 $K_S \cdot C < 0$ 的曲线 $C$，从而进行blow-down，最终得到极小模型。在有理情形，极小模型必为 $\mathbb{P}^2$ 或 $F_n$。$\square$

## 3. Blow-up与Blow-down

### 3.1 Blow-up的几何

**定义 3.1（Blow-up）**。设 $S$ 为光滑曲面，$p \in S$ 为点。$S$ 在 $p$ 处的**blow-up**是态射 $\pi: \widetilde{S} \to S$，其中 $\widetilde{S}$ 光滑，$E = \pi^{-1}(p) \cong \mathbb{P}^1$ 是例外曲线，$\pi$ 在 $S \setminus \{p\}$ 上是同构。

**定理 3.2**。$\widetilde{S}$ 由 $S$ 在 $p$ 处切空间的射影化构造：局部上，若 $S = \operatorname{Spec} k[x,y]$，则

$$\widetilde{S} = \{(x,y; [u:v]) \in \mathbb{A}^2 \times \mathbb{P}^1 \mid xv = yu\}.$$

### 3.2 Castelnuovo收缩判据

**定理 3.3（Castelnuovo收缩判据）**。设 $S$ 为光滑曲面，$E \subseteq S$ 为曲线。则 $E$ 可收缩为光滑点（即存在 blow-up 的逆）当且仅当 $E \cong \mathbb{P}^1$ 且 $E^2 = -1$。

这样的曲线 $E$ 称为**例外曲线**（exceptional curve）或 **$(-1)$-曲线**。

## 4. 重要例子

### 例子 1：Hirzebruch曲面

**例 4.1**。$F_n = \mathbb{P}(\mathcal{O}_{\mathbb{P}^1} \oplus \mathcal{O}_{\mathbb{P}^1}(n))$（$n \geq 0$）。

- $F_0 = \mathbb{P}^1 \times \mathbb{P}^1$。
- $F_1$ 是 $\mathbb{P}^2$ 在一点的 blow-up。
- $F_n$（$n \geq 1$）有唯一的负截面 $C_-$ 满足 $C_-^2 = -n$。
- $F_n \not\cong F_m$ 对 $n \neq m$。

### 例子 2：Cremona变换

**例 4.2**。$

$\mathbb{P}^2$ 的 Cremona 变换是双有理自同构

$$[x:y:z] \mapsto [yz: xz: xy] = [1/x : 1/y : 1/z].$$

它可分解为 blow-up 三个一般点（坐标顶点）后再 blow-down 三条通过两点的直线。

### 例子 3：Del Pezzo曲面

**例 4.3**。光滑曲面 $S$ 称为**Del Pezzo曲面**，如果 $-K_S$ 是丰富的。它们恰好是 $\mathbb{P}^2$ 在至多 $8$ 个一般点的 blow-up 或 $\mathbb{P}^1 \times \mathbb{P}^1$。

- 次数 $d = K_S^2$ 从 $9$（$\mathbb{P}^2$）递减到 $1$（8个点的blow-up）。
- $d = 3$：三次曲面（27条直线）。
- $d = 2$：双曲面，$|-K_S|$ 给出到 $\mathbb{P}^2$ 的 $2:1$ 覆盖。
- $d = 1$：$|-K_S|$ 给出到 $\mathbb{P}^1$ 的有理映射，一般纤维为椭圆曲线。
