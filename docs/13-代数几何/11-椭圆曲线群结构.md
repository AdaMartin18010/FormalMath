# 椭圆曲线群结构

## 引言

椭圆曲线（elliptic curve）是亏格为一的光滑射影曲线，配备了指定的有理点作为群单位元。它是代数几何、数论和密码学交汇的璀璨明珠。椭圆曲线的群结构来源于其 Jacobian 簇与自身的同构：任何椭圆曲线都是一维Abel簇，其上存在自然的Abel群结构。

从几何上看，椭圆曲线的群法则可以通过"三点共线"的优美几何构造来描述：三点之和为零当且仅当它们共线。从代数上看，Weierstrass方程提供了具体的群运算公式。从算术上看，椭圆曲线上的有理点构成有限生成Abel群（Mordell-Weil定理），其结构深刻反映了底域的算术性质。

本教程系统介绍椭圆曲线的群法则、$j$-不变量、Weierstrass方程以及复乘理论。

---

## 1. 群法则的定义

### 1.1 几何定义

**定义 1.1（椭圆曲线）**。域 $k$ 上的**椭圆曲线**是亏格为 $1$ 的光滑射影曲线 $E$，配备了一个 $k$-有理点 $O \in E(k)$（单位元）。

**定理 1.2（群法则）**。存在唯一的 Abel 群结构 $(E, +)$ 满足：
1. $O$ 是零元；
2. 三点 $P, Q, R \in E$ 满足 $P + Q + R = O$ 当且仅当 $P, Q, R$ 共线（作为 $

$\mathbb{P}^2$ 中的点，在Weierstrass嵌入下）。

*构造*。给定 $P, Q$，连接 $P, Q$ 的直线交 $E$ 于第三点 $R$（计重数）。则 $P + Q = -R$，其中 $-R$ 是 $R$ 关于 $x$-轴的对称点（在Weierstrass模型中）。

### 1.2 代数公式

**定理 1.3（Weierstrass方程）**。特征不为 $2, 3$ 时，任何椭圆曲线可写成

$$y^2 = x^3 + ax + b, \quad \Delta = -16(4a^3 + 27b^2) \neq 0.$$

**群加法公式**：设 $P = (x_1, y_1)$，$Q = (x_2, y_2)$。若 $P \neq -Q$：
- 若 $P \neq Q$：斜率 $\lambda = \frac{y_2 - y_1}{x_2 - x_1}$
- 若 $P = Q$：$\lambda = \frac{3x_1^2 + a}{2y_1}$

则 $P + Q = (x_3, y_3)$，其中

$$x_3 = \lambda^2 - x_1 - x_2, \quad y_3 = \lambda(x_1 - x_3) - y_1.$$

## 2. j-不变量

### 2.1 定义与分类

**定义 2.1**。椭圆曲线 $E: y^2 = x^3 + ax + b$ 的 **$j$-不变量**定义为

$$j(E) := 1728 \frac{4a^3}{4a^3 + 27b^2}.$$

**定理 2.2**。设 $k$ 为代数闭域。两条椭圆曲线 $E, E'$ 同构（作为带单位元的曲线）当且仅当 $j(E) = j(E')$。

*证明*。同构保持Weierstrass形式，而 $j$ 恰好参数化Weierstrass方程在同构下的轨道。$\square$

### 2.2 $j$-不变量的模解释

**命题 2.3**。$j$-不变量给出模空间同构

$$j: \mathcal{M}_{1,1} \xrightarrow{\cong} \mathbb{A}^1,$$

其中 $\mathcal{M}_{1,1}$ 是带一个标定点的椭圆曲线的粗模空间。

**注记 2.4**。$j = 0$ 对应 $E: y^2 = x^3 + 1$（有额外自同构 $\mathbb{Z}/6\mathbb{Z}$）。$j = 1728$ 对应 $E: y^2 = x^3 + x$（自同构群 $\mathbb{Z}/4\mathbb{Z}$）。一般 $j$ 的自同构群为 $\mathbb{Z}/2\mathbb{Z}$。

## 3. 复乘理论

### 3.1 自同态环

**定义 3.1**。椭圆曲线 $E$ 的**自同态环** $\operatorname{End}(E)$ 是保持 $O$ 的态射 $E \to E$ 在加法下构成的环，乘法为复合。

**定理 3.2**。$\operatorname{End}(E)$ 要么同构于 $\mathbb{Z}$（**无复乘**），要么是虚二次域中的序（**有复乘**）。

*证明*。$\operatorname{End}(E) \otimes \mathbb{Q}$ 是可除代数，作用于 $H^1(E, \mathbb{Q})$。对复椭圆曲线 $E \cong \mathbb{C}/\Lambda$，$\operatorname{End}(E) = \{\alpha \in \mathbb{C} \mid \alpha \Lambda \subseteq \Lambda\}$。若 $\alpha \notin \mathbb{R}$，则 $\mathbb{Q}(\alpha)$ 是虚二次域。$\square$

### 3.2 复乘椭圆曲线的性质

**定理 3.3**。设 $E$ 有复乘于虚二次域 $K$。则：
1. $j(E)$ 是代数整数。
2. $K(j(E))$ 是 $K$ 的Hilbert类域。
3. $E$ 的模 $p$ 约化性质与 $K$ 中的素理想分解相关。

## 4. 重要例子

### 例子 1：实点群结构

**例 4.1**。$E: y^2 = x^3 - x$（$j = 1728$）在 $\mathbb{R}$ 上的实点群 $E(\mathbb{R}) \cong \mathbb{Z}/2\mathbb{Z} \times \mathbb{R}/\mathbb{Z}$（两个连通分支，每个同胚于 $S^1$）。

### 例子 2：有限域上的点数

**例 4.2**。设 $E/\mathbb{F}_q$。Hasse定理断言

$$| \#E(\mathbb{F}_q) - (q + 1) | \leq 2\sqrt{q}.$$

对 $E: y^2 = x^3 + 1$ 过 $\mathbb{F}_7$，$x = 0, 1, 2, 3, 4, 5, 6$ 时 $x^3 + 1 \equiv 1, 2, 2, 0, 2, 0, 0 \pmod{7}$。二次剩余为 $0, 1, 2, 4$，故点数（含无穷远点）为 $1 + 2 + 0 + 2 + 0 + 0 + 2 + 1 = 8$？需仔细重算：对每个 $x$，若 $x^3+1 = 0$ 则一个 $y$ 值；若为二次剩余则两个 $y$ 值；若为非剩余则无。$x=0:1$（剩余，2点）；$x=1:2$（剩余，2点）；$x=2:9\equiv2$（剩余，2点）；$x=3:28\equiv0$（1点）；$x=4:65\equiv2$（2点）；$x=5:126\equiv0$（1点）；$x=6:217\equiv0$（1点）。总计 $2+2+2+1+2+1+1=11$ 有限点，加无穷远点 $O$，共 $12$ 点。$q+1=8$，偏差 $4 \leq 2\sqrt{7} \approx 5.29$，符合Hasse界。

### 例子 3：椭圆曲线的扭点

**例 4.3**。$E: y^2 = x^3 - x$。扭点包括：
- $2$-扭点：$E[2] = \{O, (0,0), (1,0), (-1,0)\} \cong (\mathbb{Z}/2\mathbb{Z})^2$。
- $E[2]$ 恰好是拐点（inflection points）在Weierstrass嵌入下。

## 练习

1. 验证椭圆曲线群法则的结合律（可通过Weierstrass方程直接计算，或用Riemann-Roch证明三点共线条件等价于除子线性等价）。
2. 证明 $j$-不变量确实分类椭圆曲线：若 $j(E) = j(E')$，构造显式同构。
3. 对 $E/\mathbb{Q}: y^2 = x^3 + 1$，计算 $E[3]$（3-扭点）并证明 $\operatorname{End}(E) \cong \mathbb{Z}[\omega]$，$\omega = e^{2\pi i/3}$。
