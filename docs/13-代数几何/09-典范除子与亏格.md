# 典范除子与亏格

## 引言

在代数几何的分类问题中，数值不变量扮演着指南针的角色。它们将复杂的几何对象浓缩为可计算、可比较的整数，从而使分类成为可能。在这些不变量中，**典范除子**（canonical divisor）和**亏格**（genus）是最基本、最深刻的两个。

典范除子 $K_X$ 来源于微分形式：在光滑簇上，它是最高次全纯微分形式的零点/极点除子。它不仅控制着Serre对偶中的对偶化层，也是判断丰富性、研究双有理几何的关键工具。Kodaira维数的定义就直接依赖于典范除子的倍数线性系的行为。

亏格则是曲线论的核心不变量，刻画了曲线的"洞数"或"复杂度"。从拓扑上看，它是曲线的第一Betti数的一半；从代数上看，它是正则微分形式空间的维数。本教程将系统介绍这些不变量及其相互关系。

## 1. 典范除子的定义

### 1.1 微分层与典范层

**定义 1.1（Kähler微分层）**。设 $X$ 为 $k$-概形。$\Omega_{X/k}$ 为满足以下泛性质的 $\mathcal{O}_X$-模层：

$$\operatorname{Hom}_{\mathcal{O}_X}(\Omega_{X/k}, \mathcal{F}) \cong \operatorname{Der}_k(\mathcal{O}_X, \mathcal{F})$$

对所有 $\mathcal{O}_X$-模层 $\mathcal{F}$。

**定义 1.2（典范层）**。设 $X$ 为 $n$ 维光滑簇。定义**典范层**（canonical sheaf）

$$\omega_X := \bigwedge^n \Omega_{X/k}.$$

这是可逆层（线丛），对应某个Cartier除子类，称为**典范除子**，记 $K_X$。

### 1.2 Adjunction公式

**定理 1.3（Adjunction公式）**。设 $X$ 为光滑簇，$Y \subseteq X$ 为光滑除子（余维数 $1$ 光滑子簇）。则

$$\omega_Y = \omega_X(Y)|_Y = \omega_X \otimes \mathcal{O}_X(Y)|_Y,$$

等价地 $K_Y = (K_X + Y)|_Y$。

*证明*。由法丛的短正合列

$$0 \longrightarrow \mathcal{I}_Y/\mathcal{I}_Y^2 \longrightarrow \Omega_X|_Y \longrightarrow \Omega_Y \longrightarrow 0$$

取最高外幂，利用 $\mathcal{I}_Y/\mathcal{I}_Y^2 \cong \mathcal{O}_X(-Y)|_Y$ 即得。$\square$

## 2. 曲线的亏格

### 2.1 多种等价定义

设 $C$ 为光滑射影曲线。

**定义 2.1（几何亏格）**。

$$p_g(C) := h^0(C, \omega_C) = h^1(C, \mathcal{O}_C) = g.$$

**定义 2.2（拓扑亏格）**。若 $k = \mathbb{C}$，$C$ 是亏格 $g$ 的紧Riemann面，拓扑上同胚于带 $g$ 个环柄的球面。

**定义 2.3（算术亏格）**。对任意射影曲线 $C$，$p_a(C) := 1 - \chi(\mathcal{O}_C) = 1 - h^0(\mathcal{O}_C) + h^1(\mathcal{O}_C)$。

**命题 2.4**。对光滑射影曲线，$g = p_g = p_a$。

### 2.2 典范除子的次数

**命题 2.5**。对光滑射影曲线 $C$，$\deg K_C = 2g - 2$。

*证明*。由Riemann-Roch（参见 [07-黎曼-罗赫定理](07-黎曼-罗赫定理.md)）对 $D = K_C$：

$$h^0(K_C) - h^0(0) = \deg K_C + 1 - g,$$

即 $g - 1 = \deg K_C + 1 - g$，故 $\deg K_C = 2g - 2$。$\square$

## 3. 高维簇的数值不变量

### 3.1 Hodge数

**定义 3.1**。设 $X$ 为光滑射影 $n$ 维簇。**Hodge数**

$$h^{p,q}(X) := \dim H^q(X, \Omega^p_X).$$

**命题 3.2（Hodge对称性）**。$h^{p,q} = h^{q,p} = h^{n-p,n-q}$。

### 3.2 不规则性与几何亏格

**定义 3.3**。$q(X) := h^1(X, \mathcal{O}_X)$ 称为**不规则性**（irregularity）。

**定义 3.4**。$p_g(X) := h^n(X, \mathcal{O}_X) = h^0(X, \omega_X)$ 称为**几何亏格**（geometric genus）。

**定义 3.5**。$\chi(\mathcal{O}_X) := \sum_i (-1)^i h^i(X, \mathcal{O}_X)$ 称为**Holomorphic Euler示性数**。

## 4. Kodaira维数

### 4.1 定义

**定义 4.1（Kodaira维数）**。设 $X$ 为光滑射影簇。定义**Kodaira维数**

$$\kappa(X) := \limsup_{m \to \infty} \frac{\log h^0(X, \omega_X^{\otimes m})}{\log m} \in \{-\infty, 0, 1, \dots, \dim X\}.$$

等价地，$\kappa(X)$ 是 $mK_X$ 的线性系给出的有理映射的像的维数（对充分大 $m$），若所有 $h^0(mK_X) = 0$ 则 $\kappa = -\infty$。

### 4.2 分类意义

**命题 4.2**。Kodaira维数是双有理不变量。

**例子 4.3**。曲线分类：

- $g = 0$：$\kappa = -\infty$（$\mathbb{P}^1$，有理曲线）
- $g = 1$：$\kappa = 0$（椭圆曲线）
- $g \geq 2$：$\kappa = 1$（一般型曲线）

## 5. 重要例子

### 例子 1：超曲面的典范除子

**例 5.1**。设 $X \subseteq \mathbb{P}^n$ 为 $d$ 次光滑超曲面。由adjunction公式，

$$K_X = (K_{\mathbb{P}^n} + X)|_X = ((-n-1)H + dH)|_X = (d - n - 1)H|_X,$$

其中 $H$ 为超平面类。特别地：

- $d = n + 1$：$K_X = 0$，$X$ 为Calabi-Yau簇（如三次曲面、四次三维fold等）。
- $d < n + 1$：$K_X$ 丰富，$X$ 为一般型。
- $d > n + 1$：$-K_X$ 丰富，$X$ 为Fano簇。

### 例子 2：曲线的典范映射

**例 5.2**。亏格 $g = 2$ 曲线：$K_C$ 给出一个 $2:1$ 映射到 $\mathbb{P}^1$（超椭圆）。

亏格 $g = 3$：

- 非超椭圆：$|K_C|$ 给出嵌入 $C \hookrightarrow \mathbb{P}^2$ 为四次曲线。
- 超椭圆：$|K_C|$ 给出 $2:1$ 映射到 $\mathbb{P}^2$ 中的二次曲线（有理正规曲线）。

### 例子 3：曲面的数值不变量

**例 5.3**。设 $S$ 为光滑射影曲面。

- 由Noether公式：$\chi(\mathcal{O}_S) = \frac{K_S^2 + e(S)}{12}$，其中 $e(S)$ 为拓扑Euler示性数。
- 对K3曲面：$K_S = 0$，$q = 0$，$p_g = 1$。由Noether公式，$e(S) = 24$。
- 对Enriques曲面：$K_S \not\sim 0$ 但 $2K_S \sim 0$，$q = 0$，$p_g = 0$。
