## 1. 概念深度解析

### 1.1 几何直观

**向量丛模空间**参数化给定拓扑类型的向量丛：

- **稳定性条件**：排除可约丛，保证好的模空间
- **斜率稳定性**：μ(E) = deg(E)/rank(E)
- **模空间维数**：依赖于秩、度、亏格

**关键洞见**：稳定丛的模空间是拟射影的，紧化用半稳定丛的 S-等价类。

### 1.2 形式定义

**定义 1.1** (斜率 / Slope)
对曲线 C 上向量丛 E：
μ(E) := deg(E) / rank(E)

**定义 1.2** (稳定性 / Stability)
E 称为**(半)稳定的**，若对所有真子丛 F ⊂ E：
μ(F) < (≤) μ(E)

**定义 1.3** (模空间 M(r,d))
M_C(r,d) = {秩 r，度 d 的稳定丛}/同构

**Seshadri 紧化**：
\overline{M}_C(r,d) = {半稳定丛}/S-等价

### 1.3 代数表述

**GIT 构造**：
对充分大 n，考虑 Quot 概形：
Quot = {O(-n)^N → E → 0}

稳定条件对应 GIT 稳定性，模空间为商：
M(r,d) = Quot^{ss} // PGL(N)

## 2. 属性与关系

### 2.1 核心定理

**定理 2.1** (Narasimhan-Seshadri)
M_C(r,d) 是光滑拟射影簇，维数 r^2(g-1) + 1（当 g ≥ 2）。

**定理 2.2** (紧化)
\overline{M}_C(r,d) 是射影的，边界对应可约半稳定丛。

**定理 2.3** (维数公式)
dim M(r,d) = r^2(g-1) + 1（当 r,d 互素）

### 2.2 完整证明

**定理 2.1 (维数) 的证明**

**步骤1**：稳定丛 E 的形变由 H^1(C, End(E)) 控制。

**步骤2**：由 Serre 对偶：
h^1(End(E)) = h^0(End(E) ⊗ K)

**步骤3**：E 稳定 ⇒ End(E) = C（标量），故：
χ(End(E)) = 1 - r^2(g-1)

**步骤4**：由稳定性，h^0(End(E)) = 1，故：
h^1(End(E)) = r^2(g-1) + 1 - 1 = r^2(g-1)。

光滑性由 Ext^2 = 0（曲线维数1）得到。□

## 3. 示例与习题

### 3.1 具体示例

**示例 3.1** (秩1)
M_C(1,d) = Pic^d(C)，维数 g。

与稳定丛条件：线丛自动稳定。

**示例 3.2** (椭圆曲线)
设 E 为椭圆曲线，M_E(r,d)（r,d 互素）：

- dim = 1（与 E 同构）
- 由 Atiyah 分类

**示例 3.3** (g=2, r=2)
M_C(2,1)，dim = 4·1 + 1 = 5。

可用经典不变量理论描述。

### 3.2 反例

**反例 3.4** (非互素)
r,d 不互素时，稳定丛可能不存在，模空间奇异。

### 3.3 习题

**习题 1**
证明：稳定丛是简单的（End(E) = C）。

**习题 2**
计算 M_C(2,0) 的维数。

**解答**：= 4(g-1) + 1 = 4g - 3。

**习题 3**
设 L 为线丛，证明 E 稳定 ⇔ E ⊗ L 稳定。

**习题 4**
描述 \overline{M}_C(2,1) 的边界。

**习题 5**
证明：g=0 时 M_{P^1}(r,d) 为空或单点。

## 4. 形式化实现

import Mathlib

-- 斜率
def Slope {C : Curve} (E : VectorBundle C) : Q :=
  E.degree / E.rank

-- 稳定性
def IsStable {C : Curve} (E : VectorBundle C) : Prop :=
  ∀ (F : Subbundle E), F ≠ 0 → F ≠ E → Slope F < Slope E

-- 模空间
def ModuliSpaceOfBundles (C : Curve) (r d : ℕ) (h : r > 0) : Type :=
  {E : VectorBundle C // E.rank = r ∧ E.degree = d ∧ IsStable E} ⧸
