## 1. 概念深度解析

### 1.1 几何直观

**曲线模空间 M_g** 是亏格 g 光滑曲线的"参数空间"：

- **点 ↔ 曲线**：每个点代表同构类 [C]
- **维数**：dim M_g = 3g - 3（g ≥ 2）
- **边界**：包含稳定曲线（带节点）的紧化

**模问题的函子性**：
对概形 S，考虑族 F → S（纤维为亏格 g 曲线）。
M_g 表示此函子。

### 1.2 形式定义

**定义 1.1** (稳定曲线 / Stable Curve)
连通 nodal 曲线 C 称为**稳定的**，若：

- 只有寻常二重奇点（节点）
- 每个光滑有理分支与曲线的交点数 ≥ 3
- 算术亏格 pa(C) = g

**定义 1.2** (模函子 / Moduli Functor)
M_g: {概形}^op → {集合}

M_g(S) = {稳定曲线 C → S}/~（S-同构）

**定义 1.3** (粗模空间 / Coarse Moduli Space)
**粗模空间** (M_g, Φ) 满足：

- M_g 为概形
- Φ: M_g(-) → Hom(-, M_g) 为自然变换
- 在代数闭域上为双射
- 泛性质：任何其他 (N, Ψ) 唯一通过 M_g

### 1.3 代数表述

**GIT 构造**：
用 Hilbert 概形或 Chow 簇的商构造：
M_g = H^0 // PGL(N+1)

其中 H^0 为三典范嵌入曲线的 Hilbert 点。

**Deligne-Mumford 堆栈**：
为获得细模空间，考虑带自同构的 stack：
M_g^{DM} = {稳定曲线 + 标记}

## 2. 属性与关系

### 2.1 核心定理

**定理 2.1** (Deligne-Mumford)
M_g（g ≥ 2）是不可约拟射影概形，维数 3g - 3。

**定理 2.2** (紧化)
\overline{M}_g（稳定曲线模空间）是射影的。

**定理 2.3** (维数计算)

- g = 0: M_0 = Spec(Z)（唯一点）
- g = 1: M_1 = A^1（j-线）
- g ≥ 2: dim M_g = 3g - 3

### 2.2 完整证明

**定理 2.1 (维数) 的证明**

**步骤1**：曲线 C 的形变由 Ext^1(Ω_C, O_C) = H^1(C, T_C) 控制。

**步骤2**：由 Serre 对偶，h^1(T_C) = h^0(K_C^{⊗2})。

**步骤3**：由 Riemann-Roch：
h^0(K^{⊗2}) - h^1(K^{⊗2}) = 2(2g-2) + 1 - g = 3g - 3

**步骤4**：对 g ≥ 2，h^1(K^{⊗2}) = 0（deg > 2g-2）。□

## 3. 示例与习题

### 3.1 具体示例

**示例 3.1** (椭圆曲线)
M_1 = A^1_j，其中 j = j(E) 为 j-不变量。

\overline{M}_1 = P^1_j，边界点为 nodal cubic。

**示例 3.2** (g = 2)
M_2 可用双覆盖 P^1 的 6 个分歧点描述：
M_2 = (Sym^6(P^1) - Δ)/PGL_2

维数 = 6 - 3 = 3 = 3·2 - 3。

**示例 3.3** (超椭圆轨迹)
H_g ⊂ M_g：超椭圆曲线轨迹，维数 2g - 1。

### 3.2 反例

**反例 3.4** (细模空间不存在)
M_g 没有细模空间，因曲线有自同构（generic 情形除外）。

### 3.3 习题

**习题 1**
证明：g ≥ 3 的一般曲线非超椭圆。

**解答**：dim M_g = 3g - 3 > 2g - 1 = dim H_g。

**习题 2**
计算 M_3 的维数并描述超椭圆轨迹。

**习题 3**
证明 \overline{M}_g 的边界 divisor 对应 nodal 曲线退化。

**习题 4**
描述 M_{1,1}（带标记点的椭圆曲线模空间）。

**习题 5**
计算 \overline{M}_{0,n} 的维数（n 标记点的稳定有理曲线）。

  total : Scheme

def ModuliFunctor (g : ℕ) : ContravariantFunctor Scheme Type where

## 5. 应用与拓展

### 5.1 数论联系

**Shafarevich猜想**：Faltings 证明高亏格曲线在数域上有限，用 moduli 的高度理论。

### 5.2 物理应用

**弦理论**：bosonic string 的 moduli 空间维数：
M_{g,0} 维数 = 3g - 3（共形结构）

### 5.3 前沿方向

**Gromov-Witten理论**：稳定映射的 moduli 空间 \overline{M}_{g,n}(X, β)。
