## 1. 概念深度解析

### 1.1 几何直观

**相交积** [V] · [W] 是子簇 V 和 W 的交：

- **横截相交**：dim(V ∩ W) = dim V + dim W - dim X
- **自相交**：V ∩ V 需要法丛信息（ excess intersection）
- **相交重数**：在交点的局部代数数据

**关键洞察**：相交积在 Chow 环中是良定的，不依赖具体代表。

### 1.2 形式定义

**定义 1.1** (横截相交 / Transversal Intersection)
子簇 V, W ⊆ X **横截相交**，若：
T_p(V) + T_p(W) = T_p(X), ∀ p ∈ V ∩ W

此时 [V] · [W] = [V ∩ W]。

**定义 1.2** (相交积 / Intersection Product)
对光滑射影簇 X，相交积：
A^i(X) × A^j(X) → A^{i+j}(X)

由以下公理刻画：

- 对横截情形：[V] · [W] = [V ∩ W]
- 结合律、交换律
- 与 proper 推进/平坦 pullback 相容

**定义 1.3** (相交重数 / Intersection Multiplicity)
对孤立交点 p ∈ V ∩ W：
m_p(V, W) = dim_k O_{X,p}/(I_V + I_W)

### 1.3 代数表述

**陈类与相交**：
对向量丛 E，第 i 陈类 c_i(E) ∈ A^i(X)。

**分裂原理**：计算时可假设 E 分裂为线丛直和。

**总陈类**：c(E) = 1 + c_1(E) + ... + c_r(E)

## 2. 属性与关系

### 2.1 核心定理

**定理 2.1** (Bézout定理的一般形式)
设 X ⊆ P^n 为 d 次光滑射影簇，H 为超平面截面，则：
H^{dim X} = d · [pt]

**定理 2.2** (自相交公式)
设 V ⊆ X 光滑，N_{V/X} 为法丛，则：
[V] · [V] = c_{top}(N_{V/X}) ∩ [V]

**定理 2.3** (Riemann-Roch代数)
对向量丛 E，有：
ch(E) = rank(E) + c_1(E) + (c_1^2 - 2c_2)/2 + ...

### 2.2 完整证明

**定理 2.1 的证明** (Bézout)

**步骤1**：P^n 的 Chow 环：A^*(P^n) = Z[h]/(h^{n+1})。

**步骤2**：X = V(f) 为 d 次超曲面，则 [X] = d·h。

**步骤3**：X 的 hyperplane class 为 h|_X，故：
(h|_X)^{n-1} = d·h^n ∩ [X] = d·[pt]。□

## 3. 示例与习题

### 3.1 具体示例

**示例 3.1** (P^2 中两直线)
两条不同直线 L_1, L_2 交于一点：
[L_1] · [L_2] = [pt]

同一直线自交：
[L] · [L] = [L]^2 = [pt]（通过形变论证）

**示例 3.2** (P^3 中的扭三次曲线)
设 C ⊂ P^3 为扭三次曲线，deg C = 3。

C 是一般超平面的交线：
[H_1] · [H_2] · [H_3] = 3[pt]

**示例 3.3** (曲面上的曲线)
设 S 为曲面，C ⊂ S 曲线：
C^2 = C · C = self-intersection number

- P^2 中直线：L^2 = 1
- 爆破后的例外曲线：E^2 = -1

### 3.2 反例

**反例 3.4** (非横截相交)
P^2 中两曲线在原点切触，相交重数 > 1。

### 3.3 习题

**习题 1**
计算 P^1 × P^1 中对角 Δ 的自交数。

**解答**：设 h_1, h_2 为 pullbacks，则 [Δ] = h_1 + h_2。
[Δ]^2 = (h_1 + h_2)^2 = 2h_1h_2 = 2[pt]。

**习题 2**
设 C ⊂ P^2 为光滑 d 次曲线，计算 C^2。

**习题 3**
用相交理论证明 P^n 中 n 个一般超曲面有 d_1...d_n 个交点。

**习题 4**
设 E 为椭圆曲线，描述 E × E 的相交形式。

**习题 5**
计算 G(2,4) 中 Schubert 类的相交数。
