## 1. 概念深度解析

### 1.1 几何直观

**层 (Sheaf)** 是"局部定义，全局粘合"的数据结构。几何上：

- **预层**：每开集 U 赋予数据 F(U)，带限制映射
- **层公理**：局部相等 ⇒ 全局相等；局部存在 ⇒ 全局存在（可粘合）
- **茎 (Stalk)**：在某点 x 的局部数据，F_x = colim_{U∋x} F(U)

### 1.2 形式定义

**定义 1.1** (预层 / Presheaf)
设 X 为拓扑空间。预层 F 包含：

- 对每个开集 U ⊆ X，对象 F(U)
- 限制映射 ρ_UV: F(U) → F(V) 对 V ⊆ U

满足：ρ_UU = id，ρ_UW = ρ_VW ∘ ρ_UV

**定义 1.2** (层 / Sheaf)
预层 F 称为层，若对任意开覆盖 U = ⋃_i U_i：

**(唯一性)** 若 s, t ∈ F(U) 且 s|_{U_i} = t|_{U_i} 对所有 i，则 s = t。

**(存在性)** 若 s_i ∈ F(U_i) 满足相容条件，则存在 s ∈ F(U) 使 s|_{U_i} = s_i。

## 2. 属性与关系

### 2.1 核心定理

**定理 2.1** (层的正合性)
层序列 0 → F' → F → F'' → 0 正合 ⇔ 对所有 x ∈ X，茎序列 0 → F'_x → F_x → F''_x → 0 正合。

**定理 2.2** (整体截面的左正合性)
函子 Γ(X, -) 左正合：若 0 → F' → F → F'' → 0 正合，则：
0 → Γ(X, F') → Γ(X, F) → Γ(X, F'')
正合（最后一箭头不一定满）。

### 2.2 完整证明

**定理 2.2 的证明** (整体截面的左正合性)

**步骤1**（单射）：设 φ: F' → F 单。若 φ_U(s) = 0 ∈ F(U)，则对所有 x ∈ U，φ_x(s_x) = 0 ∈ F_x。因 φ_x 单，s_x = 0。由层公理，s = 0。

**步骤2**（核的像）：由层序列正合，im(φ) = ker(ψ) 作为层。对 s ∈ Γ(X, F)，ψ(s) = 0 ⇔ s ∈ Γ(X, ker(ψ)) = Γ(X, im(φ))。但 t ∈ Γ(X, im(φ)) 不一定来自 Γ(X, F')，因提升可能仅局部存在。□

## 4. 形式化实现

import Mathlib

-- 预层结构
structure Presheaf (X : Type) [TopologicalSpace X] (C : Type) [Category C] where
  obj : Opens X → C
  map : ∀ {U V : Opens X}, V ≤ U → obj U ⟶ obj V
  map_id : ∀ U, map (le_refl U) = 𝟙 (obj U)
  map_comp : ∀ {U V W : Opens X} (hUV : V ≤ U) (hVW : W ≤ V),
    map (hVW.trans hUV) = map hUV ≫ map hVW

-- 层公理（简化）
