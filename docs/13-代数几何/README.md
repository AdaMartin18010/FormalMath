# 13-代数几何

**多项式方程的几何 — 从古典代数几何到概形理论**

---

## 📋 分支概览

代数几何是现代数学的核心分支，研究多项式方程组的解集的几何性质：

- **代数曲线**：Riemann曲面、Jacobian
- **代数曲面**：双有理分类、相交理论
- **概形理论**：Grothendieck的概形语言
- **层与上同调**：Sheaf cohomology
- **除子与线丛**：线丛分类、Riemann-Roch
- **动机理论**：混合 motive、标准猜想
- **p进Hodge理论**：p进方法

**难度等级**: ⭐⭐⭐⭐⭐（研究级）
