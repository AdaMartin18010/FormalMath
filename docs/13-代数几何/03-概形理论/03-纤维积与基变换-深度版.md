## 1. 概念深度解析

### 1.1 几何直观

**纤维积 X ×_S Y** 是"同时在 X 和 Y 上的点，且在 S 上一致"：

- 几何点：{(x, y) : f(x) = g(y) ∈ S}
- 纤维：X_s = X ×_S Spec(k(s))
- 基变换：沿 S' → S 拉回族

**直观**：将 X 和 Y 在 S 上"粘合"。

### 1.2 形式定义

**定义 1.1** (纤维积 / Fiber Product)
概形态射 X → S, Y → S 的**纤维积** X ×_S Y 是概形，带投影 p_1: X ×_S Y → X, p_2: X ×_S Y → Y，满足：

- f ∘ p_1 = g ∘ p_2
- 泛性质：对任意 Z → X, Z → Y 使图表交换，存在唯一 Z → X ×_S Y

**定义 1.2** (基变换 / Base Change)
态射 f: X → S 沿 g: S' → S 的**基变换**为：
f': X ×_S S' → S'

### 1.3 代数表述

**仿射情形**：
Spec(A) ×_{Spec(R)} Spec(B) = Spec(A ⊗_R B)

**纤维**：
对 s ∈ S，几何纤维：
X_{\bar{s}} = X ×_S Spec(\overline{k(s)})

## 2. 属性与关系

### 2.1 核心定理

**定理 2.1** (纤维积的存在性)
概形范畴中纤维积总存在。

**定理 2.2** (基变换与性质保持)
以下性质在基变换下保持：

- 分离、proper、有限、仿射、有限型
- 平坦、光滑、平展

**定理 2.3** (维数公式)
设 f: X → Y 为有限型态射，则：
dim X ≤ dim Y + sup_{y ∈ Y} dim X_y

等号在某些条件下成立。

### 2.2 完整证明

**定理 2.2 (光滑性保持) 的证明**

**步骤1**：光滑 = 平坦 + 几何纤维光滑。

**步骤2**：平坦性在基变换下保持（张量积的平坦性）。

**步骤3**：几何纤维：X'_{s'} = (X ×_S S')_{s'} = X_{g(s')} ×_{k(g(s'))} k(s')。

**步骤4**：光滑簇的基扩张仍光滑。□

## 3. 示例与习题

### 3.1 具体示例

**示例 3.1** (仿射直线的纤维积)
A^1 ×_{Spec(k)} A^1 = A^2

**示例 3.2** (爆破的基变换)
设 Bl_0(A^2) → A^2 为原点爆破，沿原点外子集基变换给出同构。

**示例 3.3** (算术概形的纤维)
X = Spec(Z[x]/(x^2 + 1)) → Spec(Z)

- 一般纤维：Q(i)（虚二次域）
- p = 2：F_2[x]/(x+1)^2（分歧）
- p ≡ 1 (mod 4)：F_p × F_p（分裂）
- p ≡ 3 (mod 4)：F_{p^2}（惰性）

### 3.2 反例

**反例 3.4** (非平坦基变换)
某些性质如"约化"在基变换下不保持。

### 3.3 习题

**习题 1**
计算 Spec(C) ×_{Spec(R)} Spec(C)。

**解答**：= Spec(C ⊗_R C) = Spec(C × C) = Spec(C) ⊔ Spec(C)。

**习题 2**
设 X → S 为光滑态射，证明所有几何纤维光滑且等维。

**习题 3**
描述 A^1 ×_{G_m} A^1（用映射 x ↦ x^2）。

**习题 4**
证明：若 f: X → Y 有限，则 f 的基变换也有限。

**习题 5**
设 E → Spec(Q) 为椭圆曲线，描述 E ×_{Spec(Q)} Spec(Q(i))。

## 4. 形式化实现

```lean4
import Mathlib

-- 纤维积的泛性质
structure IsFiberProduct {X Y S P : Scheme} (f : X ⟶ S) (g : Y ⟶ S)
    (p1 : P ⟶ X) (p2 : P ⟶ Y) : Prop where
  comm : p1 ≫ f = p2 ≫ g
  lift : ∀ (Z : Scheme) (q1 : Z ⟶ X) (q2 : Z ⟶ Y),
    q1 ≫ f = q2 ≫ g → ∃! u : Z ⟶ P, u ≫ p1 = q1 ∧ u ≫ p2 = q2

-- 纤维积存在性
theorem fiberProductExists (X Y S : Scheme) (f : X ⟶ S) (g : Y ⟶ S) :
    ∃ (P : Scheme) (p1 : P ⟶ X) (p2 : P ⟶ Y), IsFiberProduct f g p1 p2 :=
  sorry

-- 基变换
def baseChange {X Y S S' : Scheme} (f : X ⟶ S) (g : S' ⟶ S) :
    Scheme :=
  (fiberProductExists X S' S f g).choose
```
