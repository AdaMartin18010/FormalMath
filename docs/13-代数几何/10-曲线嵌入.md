# 曲线嵌入

## 引言

将抽象定义的代数曲线具体地嵌入射影空间，是代数曲线几何的核心问题之一。一个除子 $D$ 的完全线性系 $|D|$ 若能给出闭嵌入 $C \hookrightarrow \mathbb{P}^n$，则不仅将曲线的抽象结构可视化，还使我们能够用齐次坐标方程描述曲线，从而应用相交理论、Hilbert函数等工具。

本教程探讨曲线嵌入的基本判据、典范嵌入（canonical embedding）的优美理论、超椭圆曲线的特殊性质，以及高亏格曲线的一般嵌入行为。这些理论紧密依赖于Riemann-Roch定理和Serre对偶。

---

## 1. Very Ample除子的判据

### 1.1 Very Ample的定义

**定义 1.1**。设 $C$ 为光滑射影曲线，$D$ 为除子。$D$ 称为**非常丰富**（very ample），如果完全线性系 $|D|$ 给出一个闭嵌入 $\phi_D: C \hookrightarrow \mathbb{P}^{h^0(D)-1}$。

**命题 1.2**。$D$ very ample 当且仅当对任意 $P, Q \in C$（可相同），$h^0(D - P - Q) = h^0(D) - 2$。

*证明*。$\phi_D$ 是嵌入当且仅当：

1. 分离点：对 $P \neq Q$，存在 $s \in H^0(D)$ 使得 $s(P) = 0, s(Q) \neq 0$，等价于 $h^0(D-P) = h^0(D) - 1$ 且 $h^0(D-P-Q) = h^0(D-P) - 1$。
2. 分离切向量：对每点 $P$，存在 $s$ 在 $P$ 处有单零点，等价于 $h^0(D-2P) = h^0(D) - 2$。

由Riemann-Roch，$h^0(D-P) = h^0(D) - 1$ 当 $\deg D > 2g - 2$ 自动成立（因 $h^1(D-P) = h^0(K_C - D + P) = 0$）。$\square$

### 1.2 一般嵌入定理

**定理 1.3**。设 $D$ 为度 $d$ 的除子。

1. 若 $d \geq 2g$，则 $D$ 是 base-point-free 的。
2. 若 $d \geq 2g + 1$，则 $D$ 是 very ample 的。

*证明*。由Riemann-Roch：

- $h^0(D) = d + 1 - g$（当 $d > 2g - 2$）。
- $h^0(D - P) = (d-1) + 1 - g = h^0(D) - 1$。
- $h^0(D - P - Q) = h^0(D) - 2$。

故 very ample 条件满足。$\square$

## 2. 典范嵌入

### 2.1 非超椭圆曲线的典范嵌入

**定理 2.1**。设 $C$ 为亏格 $g \geq 2$ 的非超椭圆光滑曲线。则 $K_C$ 是 very ample，给出**典范嵌入**

$$\phi_{K_C}: C \hookrightarrow \mathbb{P}^{g-1}.$$

*证明*。由 [09-典范除子与亏格](09-典范除子与亏格.md) 的例 5.3，very ample条件等价于 $C$ 非超椭圆。$\square$

**定理 2.2（Petri）**。典范曲线 $C \subseteq \mathbb{P}^{g-1}$ 的理想层由二次型生成，除非 $C$ 是 trigonal 或光滑平面五次曲线（$g = 6$）。

### 2.2 超椭圆曲线

**定理 2.3**。设 $C$ 为亏格 $g \geq 2$ 的超椭圆曲线。则 $|K_C|$ 给出一个有理映射

$$\phi_{K_C}: C \longrightarrow \mathbb{P}^{g-1},$$

它是到有理正规曲线 $R_{g-1} \subseteq \mathbb{P}^{g-1}$ 的 $2:1$ 覆盖。

*证明*。超椭圆曲线有 $g^1_2$，即 $h^0(P+Q) = 2$ 对某 $P+Q$。对一般点 $R$，$h^0(K_C - R) = g - 1$，$h^0(K_C - R - S) = g - 2$ 当 $S$ 不与 $R$ 在同一个 $g^1_2$ 的纤维中；若 $S$ 在同一纤维，$h^0(K_C - R - S) = g - 1$。故 $\phi_{K_C}$ 将每对超椭圆对映到同一点。$\square$

## 3. 高亏格曲线的一般性质

### 3.1 Trigonal曲线

**定义 3.1**。亏格 $g \geq 3$ 的曲线称为**trigonal**的，如果它有三重覆盖 $C \to \mathbb{P}^1$ 但不是超椭圆的。

**命题 3.2**。$C$ trigonal 当且仅当存在 $g^1_3$（次数 $3$、维数 $1$ 的线性系）。

### 3.2 Clifford定理

**定理 3.3（Clifford）**。设 $C$ 为光滑射影曲线，$D$ 为特殊除子（即 $h^0(K_C - D) > 0$）。则

$$\dim |D| \leq \frac{1}{2} \deg D.$$

等号成立当且仅当 $D = 0$，$D = K_C$，或 $C$ 超椭圆且 $D$ 是 $g^1_2$ 的倍数。

*证明*。利用 $h^0(D) + h^0(K_C - D) \leq g + 1$（由Riemann-Roch和 $h^1(D) = h^0(K_C - D)$）。$\square$

## 4. 重要例子

### 例子 1：平面曲线

**例 4.1**。$\mathbb{P}^2$ 中 $d$ 次光滑曲线 $C$ 有亏格 $g = (d-1)(d-2)/2$。超平面截痕给出 $g^1_d$。

- $d = 3$（$g = 1$）：椭圆曲线，$K_C = H|_C$（$H$ 为直线），$\phi_{K_C}$ 是同构 $C \to \mathbb{P}^1$？不，$K_C = 0$ 对椭圆曲线。实际上平面三次曲线的 $K_C = 0$，故不通过典范映射嵌入。
- $d = 4$（$g = 3$）：$K_C = H|_C$，故平面四次曲线是其自身的典范嵌入！
- $d = 5$（$g = 6$）：$K_C = 2H|_C$，$\phi_{K_C}$ 将 $C$ 嵌入 $\mathbb{P}^5$。

### 例子 2：完全交曲线

**例 4.2**。$C = \mathbb{P}^3$ 中二次与三次曲面的完全交。由adjunction，$g = 4$。$K_C = \mathcal{O}_C(1)$，故 $C$ 是其自身的典范曲线（嵌入 $\mathbb{P}^3$）。

### 例子 3：超椭圆曲线的具体方程

**例 4.3**。亏格 $g$ 超椭圆曲线有仿射方程

$$y^2 = f(x),$$

其中 $f(x)$ 为次数 $2g + 1$ 或 $2g + 2$ 的无重根多项式。两个无穷远点（当 $\deg f = 2g+2$）或一个无穷远点（当 $\deg f = 2g+1$）完备化为光滑射影曲线。

$g^1_2$ 由 $x$ 坐标给出：$(x) = P + Q - P_\infty - Q_\infty$（适当情形）。
