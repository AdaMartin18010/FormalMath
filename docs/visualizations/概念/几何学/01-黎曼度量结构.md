# 黎曼度量结构可视化

## 描述

本可视化展示黎曼流形上的度量张量结构，这是微分几何的核心概念，定义了流形上的距离和角度。

## 数学概念

黎曼度量g是流形上光滑变化的内积，在每点p给出切空间T_pM上的正定双线性形式。

## 可视化代码

```mermaid
graph TB
    subgraph 黎曼度量结构
    A[流形 M] --> B[切丛 TM]
    B --> C[每点 p 的切空间 T_pM]
    C --> D[度量 g_p: T_pM × T_pM → ℝ]

    D --> E[长度: |v| = √(g(v,v))]
    D --> F[角度: cos θ = g(v,w)/(|v||w|)]

    D --> G[体积元: dV = √det(g) dx¹∧...∧dxⁿ]

    H[坐标表示] --> I[g = g_ij dx^i ⊗ dx^j]
    I --> J[g_ij = g(∂/∂x^i, ∂/∂x^j)]
    end

    style A fill:#e3f2fd
    style D fill:#fff3e0
    style E fill:#e8f5e9

```

```

黎曼度量的几何意义
═══════════════════════════════════════════════════════════════

度量张量分量:
───────────────────────────────────────────────────────────────
g_ij = g(∂/∂x^i, ∂/∂x^j)

度量矩阵表示:
┌                     ┐
│ g_11  g_12  ... g_1n│
│ g_21  g_22  ... g_2n│
│  ...   ...  ...  ...│
│ g_n1  g_n2  ... g_nn│
└                     ┘

正定性: 对任意非零向量 v, g(v,v) > 0

弧长公式:
ds² = g_ij dx^i dx^j

例子:
• 欧氏空间: g_ij = δ_ij (单位矩阵)
• 球面: ds² = dθ² + sin²θ dφ²
• 双曲空间: ds² = (dx² + dy²)/y²

```
