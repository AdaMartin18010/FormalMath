# 黎曼度量结构可视化

## 1. 流形与切空间

**定义 1.1（光滑流形）**. $n$ 维光滑流形 $M$ 是 Hausdorff、第二可数的拓扑空间，配备极大光滑图册 $\{(U_\alpha, \varphi_\alpha)\}$，其中 $\varphi_\alpha: U_\alpha \to \mathbb{R}^n$ 为同胚，转移映射 $\varphi_\beta \circ \varphi_\alpha^{-1}$ 为光滑映射。

**定义 1.2（切空间）**. 点 $p \in M$ 的切空间 $T_pM$ 可定义为在该点的导子空间：满足 Leibniz 法则的线性映射 $X: C^\infty(M) \to \mathbb{R}$，即 $X(fg) = f(p)X(g) + g(p)X(f)$。在局部坐标 $(x^1, \dots, x^n)$ 下，切空间的基为 $\{\partial/\partial x^i\}_{i=1}^n$。

## 2. 黎曼度量

**定义 2.1（黎曼度量）**. 黎曼度量 $g$ 是流形 $M$ 上光滑变化的内积，即在每点 $p \in M$ 给出对称正定双线性形式
$$g_p: T_pM \times T_pM \to \mathbb{R},$$
且关于 $p$ 光滑变化。$(M, g)$ 称为黎曼流形。

在局部坐标下，度量表示为
$$g = g_{ij}\,dx^i \otimes dx^j, \quad g_{ij} = g\left(\frac{\partial}{\partial x^i}, \frac{\partial}{\partial x^j}\right).$$
矩阵 $(g_{ij})$ 对称正定。

**例子 2.2**.
- **欧氏空间**：$M = \mathbb{R}^n$，$g_{ij} = \delta_{ij}$，$ds^2 = (dx^1)^2 + \cdots + (dx^n)^2$。
- **球面** $S^2$：用球坐标 $(\theta, \phi)$，$ds^2 = d\theta^2 + \sin^2\theta\,d\phi^2$。
- **双曲平面** $\mathbb{H}^2$：上半平面模型 $ds^2 = \frac{dx^2 + dy^2}{y^2}$（$y > 0$）。

## 3. 度量的几何量

### 3.1 长度、角度与体积

对切向量 $v, w \in T_pM$：
- **长度**：$|v| = \sqrt{g(v, v)}$；
- **角度**：$\cos\theta = \frac{g(v, w)}{|v||w|}$；
- **体积元**：$dV = \sqrt{\det(g)}\,dx^1 \wedge \cdots \wedge dx^n$。

曲线 $\gamma: [a,b] \to M$ 的弧长为
$$L(\gamma) = \int_a^b \sqrt{g_{ij}(\gamma(t))\dot{\gamma}^i(t)\dot{\gamma}^j(t)}\,dt.$$

### 3.2 Levi-Civita 联络

**定理 3.1（Levi-Civita）**. 对任意黎曼流形 $(M, g)$，存在唯一的无挠、与度量相容的仿射联络 $\nabla$，称为 Levi-Civita 联络。

*证明概要*. 由无挠条件 $\nabla_X Y - \nabla_Y X = [X, Y]$ 和度量相容条件 $X(g(Y,Z)) = g(\nabla_X Y, Z) + g(Y, \nabla_X Z)$，通过轮换三式相加减，得 Koszul 公式：
$$\begin{aligned}
2g(\nabla_X Y, Z) = &X(g(Y,Z)) + Y(g(Z,X)) - Z(g(X,Y)) \\
&- g(X,[Y,Z]) + g(Y,[Z,X]) + g(Z,[X,Y]).
\end{aligned}$$
右端仅依赖 $g$ 与李括号，故 $\nabla$ 唯一存在。$\square$

在坐标下，Christoffel 符号为
$$\Gamma^k_{ij} = \frac{1}{2}g^{kl}\left(\partial_i g_{jl} + \partial_j g_{il} - \partial_l g_{ij}\right).$$

### 3.3 曲率张量

**定义 3.2（Riemann 曲率张量）**.
$$R(X,Y)Z = \nabla_X\nabla_Y Z - \nabla_Y\nabla_X Z - \nabla_{[X,Y]}Z.$$

在坐标下，$R^l_{\,ijk} = \partial_i\Gamma^l_{jk} - \partial_j\Gamma^l_{ik} + \Gamma^l_{im}\Gamma^m_{jk} - \Gamma^l_{jm}\Gamma^m_{ik}$。

**定义 3.3（截面曲率）**. 对 $2$ 维平面 $\Pi = \operatorname{span}\{v, w\} \subset T_pM$，截面曲率为
$$K(\Pi) = \frac{g(R(v,w)w, v)}{g(v,v)g(w,w) - g(v,w)^2}.$$

**定义 3.4（Ricci 曲率与数量曲率）**. Ricci 张量 $R_{ij} = R^k_{\,ikj}$。数量曲率 $S = g^{ij}R_{ij}$。

## 4. 测地线

**定义 4.1（测地线）**. 测地线是弧长泛函的临界点，满足测地线方程
$$\frac{d^2 x^k}{dt^2} + \Gamma^k_{ij}\frac{dx^i}{dt}\frac{dx^j}{dt} = 0.$$

**定理 4.2（指数映射）**. 对 $p \in M$，存在邻域 $U \subset T_pM$，使得指数映射 $\exp_p: U \to M$，$\exp_p(v) = \gamma_v(1)$（$\gamma_v$ 为从 $p$ 出发初速为 $v$ 的测地线）是微分同胚。此邻域上的坐标称为法坐标。
