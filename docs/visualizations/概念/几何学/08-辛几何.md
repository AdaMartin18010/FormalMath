# 辛几何基础可视化

## 概念定义

设 $M$ 为 $2n$ 维光滑流形。一个**辛形式** $\omega$ 是 $M$ 上的一个闭的（$\mathrm{d}\omega = 0$）、非退化的 2-形式。非退化意味着在每点 $p \in M$，映射
$$v \mapsto \omega(v, \cdot): T_p M \to T_p^* M$$
是线性同构。装备辛形式 $\omega$ 的流形 $(M, \omega)$ 称为**辛流形**。

由非退化性，辛形式在每点 $p$ 的切空间上给出一个标准型：存在 Darboux 坐标 $(x^1, \ldots, x^n, y_1, \ldots, y_n)$，使得
$$\omega = \sum_{i=1}^n \mathrm{d}x^i \wedge \mathrm{d}y_i。$$

## 关键公式

**Darboux 定理**：辛流形上不存在局部的辛不变量——任何 $2n$ 维辛流形在每点附近都辛同胚于 $(\mathbb{R}^{2n}, \omega_0)$，其中
$$\omega_0 = \sum_{i=1}^n \mathrm{d}x^i \wedge \mathrm{d}y_i = \mathrm{d}\left(\sum_{i=1}^n x^i \, \mathrm{d}y_i\right)。$$
这意味着与黎曼几何不同，辛流形没有曲率之类的局部不变量。

**哈密顿向量场**：对任意光滑函数 $H: M \to \mathbb{R}$（哈密顿量），存在唯一的向量场 $X_H$ 使得
$$\iota_{X_H}\omega = \mathrm{d}H，$$
即 $\omega(X_H, \cdot) = \mathrm{d}H$。哈密顿方程为
$$\dot{z} = X_H(z)，$$
在 Darboux 坐标下即为经典形式：
$$\dot{x}^i = \frac{\partial H}{\partial y_i}, \quad \dot{y}_i = -\frac{\partial H}{\partial x^i}。$$

**刘维尔定理**：辛形式 $\omega$（因而相空间体积 $\omega^n / n!$）沿哈密顿流保持不变：
$$\mathcal{L}_{X_H} \omega = 0。$$

**泊松括号**：函数 $f, g \in C^\infty(M)$ 的泊松括号为
$$\{f, g\} := \omega(X_f, X_g) = \sum_{i=1}^n \left(\frac{\partial f}{\partial x^i}\frac{\partial g}{\partial y_i} - \frac{\partial f}{\partial y_i}\frac{\partial g}{\partial x^i}\right)。$$
它满足反对称性、Jacobi 恒等式和 Leibniz 法则，使 $(C^\infty(M), \{\cdot, \cdot\})$ 成为李代数。

## 可视化说明

**相空间的"面积守恒"图**：以 $n=1$（二维相空间）为例，$(x, p)$ 平面上的辛形式就是 $\omega = \mathrm{d}x \wedge \mathrm{d}p$，即通常的有向面积元。

- 画一个矩形区域 $R_0$ 在 $(x, p)$ 平面中。跟踪该区域内大量代表点随哈密顿流 $\Phi_t$ 的演化。例如取谐振子 $H = \frac{1}{2}(p^2 + x^2)$，其轨道是圆周。
- **关键可视化**：尽管 $R_0$ 的形状会扭曲、拉伸、旋转，但它所包围的面积始终保持不变。用动画帧展示 $t = 0, 1, 2, 3$ 时刻的区域 $R_t = \Phi_t(R_0)$——区域变得越来越"细长"但面积不变。这直观地解释了刘维尔定理：相空间的"不可压缩性"。
- 对比一个耗散系统（如阻尼振子 $\dot{x} = p, \dot{p} = -x - \gamma p$），展示其相空间面积如何收缩至零，突出辛结构所编码的保守性。

**Darboux 坐标的"标准辛蜂窝"图**：

- 在 $\mathbb{R}^{2n}$ 中，将 $x^i$ 轴与 $y_i$ 轴配对。对 $n=2$，画出四维空间在 $(x^1, y_1)$ 和 $(x^2, y_2)$ 两个平面上的投影。
- 在每个 $(x^i, y_i)$ 平面上，画出一组单位正方形网格。辛形式 $\omega$ 在正方形 $[0,1] \times [0,1]$ 上的积分等于 $1$。用颜色梯度标注这些正方形的"辛面积"（红色表示正值 $+1$，蓝色表示负值）。
- 展示一个非线性的辛映射（如标准的 Chirikov-Taylor 标准映射）：将正方形网格映射为扭曲的平行四边形网格，但每个网格单元的辛面积仍保持为 $1$。

**拉格朗日子流形的"截面"图**：

- 辛流形中，$n$ 维子流形 $L \subset M$ 若满足 $\omega|_L = 0$（即辛形式在 $L$ 上为零），则称为**拉格朗日子流形**。
- 在 $T^*\mathbb{R} = \{(q, p)\}$ 上，画出零截面 $L_0 = \{(q, 0)\}$（水平线，显然拉格朗日）和函数 $S(q)$ 的图像 $L_S = \{(q, S'(q))\}$（也是拉格朗日，因为 $p = S'(q)$ 给出 $\mathrm{d}x \wedge \mathrm{d}p = \mathrm{d}x \wedge S''(x)\mathrm{d}x = 0$）。
- 对 $S(q) = q^3/3 - q$，画出 $L_S$ 的曲线（三次曲线）。在曲线与垂直线相交三次的区域，展示多值性与经典的"旋节分解"（spinodal decomposition）的联系。

## 直观解释

辛几何是经典力学的天然数学语言。在牛顿力学中，我们关注位形空间中的运动轨迹；在哈密顿力学中，我们将位形与其共轭动量组合为**相空间**中的点，运动方程由一个标量函数 $H$（能量）通过辛结构生成。

辛非退化条件 $\omega(v, \cdot) = 0 \Rightarrow v = 0$ 的几何意义是：相空间中不存在"与所有方向都辛正交"的非零方向。这使得每个能量函数 $H$ 唯一确定一个动力学生成元 $X_H$——能量唯一决定了"运动的方向场"。

Darboux 定理的深刻含义是：**辛几何没有局部的曲率。** 在黎曼几何中，度量 $g$ 在每点附近可以通过法坐标化为标准欧氏度量仅到一阶；曲率张量度量了二阶偏差。而在辛几何中，辛形式 $\omega$ 可以通过 Darboux 坐标化为标准型到所有阶。这意味着辛几何的丰富性完全是整体的（global）：辛流形的分类依赖于整体的拓扑不变量（如 Gromov-Witten 不变量、Fukaya 范畴），而非局部曲率。

## 例子

1. **余切丛 $T^*Q$**：任何流形 $Q$ 的余切丛 $T^*Q$ 都有标准的精确辛形式
   $$\omega = -\mathrm{d}\theta = \sum_i \mathrm{d}q^i \wedge \mathrm{d}p_i，$$
   其中 $\theta = \sum_i p_i \, \mathrm{d}q^i$ 是 Liouville 1-形式。这是力学中相空间的最一般形式：$Q$ 是位形空间，$T^*Q$ 是相空间。

2. **凯勒流形**：任何凯勒流形 $(M, \omega, J, g)$ 都是辛流形，其中 $\omega(\cdot, \cdot) = g(J\cdot, \cdot)$。例如 $(\mathbb{CP}^n, \omega_{\mathrm{FS}})$ 是紧致的辛流形。Gromov 的非挤压定理说：若存在辛嵌入 $B^{2n}(R) \hookrightarrow Z^{2n}(r) = B^2(r) \times \mathbb{C}^{n-1}$，则 $R \leq r$。这是辛拓扑（区别于体积拓扑）存在性的标志性结果。

3. **二维球面 $S^2$**：$S^2$ 上的标准面积形式（归一化为总面积 $4\pi$）是一个辛形式。每个光滑函数 $H: S^2 \to \mathbb{R}$ 的哈密顿流就是绕 $H$ 的等值线（纬线）的刚体旋转。这直接对应于 $SO(3)$ 的 Euler 方程描述的刚体转动。

4. **Arnold 猜想**：对辛流形 $(M, \omega)$ 上的非退化哈密顿微分同胚 $\phi$，其不动点个数不小于 $\sum_i \dim H_i(M; \mathbb{R})$。Floer 通过引入无限维 Morse 理论（Floer 同调）证明了特殊情形。这一定理将辛几何与代数拓扑深刻联系起来，是现代辛拓扑的起点。
