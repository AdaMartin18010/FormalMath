# 流形局部坐标卡可视化

## 描述

本可视化展示微分流形的核心概念——局部坐标卡（chart）以及坐标卡之间的转移映射，帮助理解流形如何局部等同于欧氏空间。

## 数学概念

n维光滑流形M是一个局部类似于ℝⁿ的拓扑空间，通过一族坐标卡{(Uₐ, φₐ)}覆盖，其中φₐ: Uₐ → ℝⁿ是同胚映射。坐标卡之间的兼容性由转移映射描述。

## 可视化代码

### 流形坐标卡结构

```mermaid
graph TB
    subgraph 流形M上的坐标卡
    A[流形 M] --> B[坐标卡 (U, φ)]
    A --> C[坐标卡 (V, ψ)]
    
    B --> D[U ⊂ M<br/>开集]
    B --> E[φ: U → ℝⁿ<br/>同胚映射]
    
    C --> F[V ⊂ M]
    C --> G[ψ: V → ℝⁿ]
    
    H[交集 U ∩ V] --> I[转移映射<br/>ψ ∘ φ⁻¹: φ(U∩V) → ψ(U∩V)]
    end
    
    style A fill:#e3f2fd
    style B fill:#fff3e0
    style C fill:#e8f5e9
    style I fill:#fce4ec

```

### ASCII几何图解

```

流形的局部坐标卡
═══════════════════════════════════════════════════════════════

        流形 M (二维曲面)                坐标平面 ℝ²
        
              ●P                          φ(P)
             ╱│╲                           ●─────→ x¹
           ╱  │  ╲                         │
         ╱  U │    ╲                       │
        │     │     │                      ↓ x²
        │     │     │                      
         ╲    │    ╱
           ╲  │  ╱
             ╲│╱
              
    坐标卡 (U, φ): 
    • U 是M的开子集
    • φ: U → ℝ² 是同胚
    • P点的局部坐标: (x¹, x²) = φ(P)

坐标卡重叠与转移映射:
───────────────────────────────────────────────────────────────

        流形 M                    ℝ² 的像
        
    ┌─────┐┌─────┐              φ(U)   ψ(V)
    │  U  ││  V  │             ┌───┐ ┌───┐
    │    ═╪═    │             │   │⁄│   │
    │    ═╪═    │             │  ═╪═│   │
    └─────┘└─────┘             └───┘ └───┘
      U ∩ V 重叠区域              转移映射
                                   ψ∘φ⁻¹
    
    相容性条件: 转移映射必须是光滑的 (C^∞)

典型的流形例子:
═══════════════════════════════════════════════════════════════
┌─────────────────┬───────────────────────────────────────────┐
│ 流形            │ 坐标卡描述                                │
├─────────────────┼───────────────────────────────────────────┤
│ S¹ (圆)         │ θ ↦ (cos θ, sin θ), θ ∈ (0, 2π)          │
│ S² (球面)       │ 球极投影: 南北两极坐标卡                 │
│ T² (环面)       │ (θ, φ) ↦ 嵌入 ℝ³ 的参数化               │
│ ℝP² (实射影平面)│ 齐次坐标 [x:y:z]                          │
│ GL(n,ℝ)        │ 矩阵分量作为坐标 (开子集 of ℝ^{n²})       │
└─────────────────┴───────────────────────────────────────────┘

图册 (Atlas):
───────────────────────────────────────────────────────────────
    一个图册是坐标卡的集合 { (Uₐ, φₐ) } 满足:
    
    1. 覆盖:  ⋃ Uₐ = M
    2. 相容:  任意两个坐标卡的转移映射是光滑的
    
    微分结构 = 极大图册 (包含所有相容的坐标卡)

```
