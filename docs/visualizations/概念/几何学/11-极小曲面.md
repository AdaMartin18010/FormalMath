# 极小曲面

## 描述

极小曲面是平均曲率为零的曲面，是面积泛函的临界点。

## 可视化

```mermaid
graph TD
    A[曲面 Σ ⊂ ℝ³] --> B[平均曲率 H = 0]
    B --> C[面积变分 δA = 0]
    C --> D[悬链面、螺旋面]
end

```
