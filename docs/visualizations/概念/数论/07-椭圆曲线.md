# 椭圆曲线可视化

## 概念定义

**椭圆曲线** $E$ 是亏格为 1 的光滑射影代数曲线， equipped with a specified point $O$（原点）。在 Weierstrass 形式下，$E$ 由方程
$$y^2 = x^3 + ax + b$$
给出（特征 $\neq 2, 3$），要求判别式 $\Delta = -16(4a^3 + 27b^2) \neq 0$ 以保证光滑性。

射影完备化：在 $\mathbb{P}^2$ 中，齐次方程为
$$Y^2 Z = X^3 + aXZ^2 + bZ^3，$$
无穷远点为 $O = [0:1:0]$。

## 关键公式

**群定律**：$E$ 上的点构成 Abel 群，运算几何定义为：
- 过 $P, Q$ 的直线与 $E$ 交于第三点 $R$（计重数）。
- $P + Q$ 定义为 $R$ 关于 $x$-轴的对称点（即 $-R$）。
- $O$ 为单位元。

**显式加法公式**：设 $P = (x_1, y_1)$，$Q = (x_2, y_2)$，$P \neq Q$。直线斜率
$$\lambda = \frac{y_2 - y_1}{x_2 - x_1}，$$
则 $P + Q = (x_3, y_3)$，其中
$$x_3 = \lambda^2 - x_1 - x_2, \quad y_3 = \lambda(x_1 - x_3) - y_1。$$

对 $P = Q$（倍点），用切线斜率
$$\lambda = \frac{3x_1^2 + a}{2y_1}。$$

**Mordell-Weil 定理**：设 $E$ 为定义在数域 $K$ 上的椭圆曲线。则 $E(K)$ 是有限生成 Abel 群：
$$E(K) \cong E(K)_{\mathrm{tors}} \oplus \mathbb{Z}^r，$$
其中 $E(K)_{\mathrm{tors}}$ 为有限挠子群，$r$ 为**秩（rank）**。

**Hasse 定理**：对有限域 $\mathbb{F}_q$ 上的椭圆曲线，
$$|\#E(\mathbb{F}_q) - (q + 1)| \leq 2\sqrt{q}。$$

## 可视化说明

**实点上的群运算图**：

- 在 $\mathbb{R}^2$ 中画出椭圆曲线 $y^2 = x^3 - x$（一条有两个连通分支的曲线：一个卵形环在左侧，一个开口向右无限延伸的分支）。
- 取两点 $P, Q$ 在右侧分支上。画出过 $P, Q$ 的直线（割线），它与曲线的第三个交点 $R$ 也在右侧分支上。
- **关键可视化**：将 $R$ 关于 $x$-轴反射得到 $-R$（因若 $(x,y)$ 在曲线上，则 $(x,-y)$ 也在）。$P + Q = -R$。用不同颜色标注：$P$ 为红点，$Q$ 为蓝点，$R$ 为绿点，$P+Q$ 为紫点（$R$ 的反射）。
- 对倍点 $2P$，画出 $P$ 处的切线。切线与曲线交于 $R$，然后 $2P = -R$。展示切线如何"轻微偏离"割线，但仍保持群运算的几何一致性。
- 对左侧环上的点 $P'$ 与右侧分支上的点 $Q$，割线穿过曲线于 $P', Q$ 和第三个点 $R$（在右侧），展示不同分支上的加法。

**挠点的"正多边形"图**：

- 复椭圆曲线 $E(\mathbb{C}) \cong \mathbb{C}/\Lambda$ 是复环面（torus）。画出环面的"正方形展开"图：将 $\mathbb{C}/\Lambda$ 表示为带对边粘合的单位正方形。
- $n$-挠点 $E[n] = \{P \in E : nP = O\}$ 构成群 $\cong (\mathbb{Z}/n\mathbb{Z})^2$，共 $n^2$ 个点。
- 对 $n = 2$，画出四个 2-挠点：$O$（无穷远）和三个 $x$-轴截距（$y=0$ 的解）。在环面上，这四个点形成一个矩形格子的顶点。
- 对 $n = 3$，画出九个 3-挠点在环面上形成的 $3 \times 3$ 格点。标注它们的加法结构：每行、每列、每条对角线（在环面的粘合意义下）的和为零。
- 对 $n = 5$，展示 25 个 5-挠点在环面上如何形成 intricate 的几何图案，与正五边形的对称性关联。

**BSD 猜想的"解析-算术桥梁"图**：

- 画一条复平面上的曲线代表 $L(E, s)$（椭圆曲线的 Hasse-Weil $L$-函数）。
- $L(E, s)$ 在 $s = 1$ 处的行为与 $E(\mathbb{Q})$ 的秩密切相关。BSD 猜想说：
  $$\mathrm{ord}_{s=1} L(E, s) = \mathrm{rank}\,E(\mathbb{Q})。$$
- 画出三种情形：
  - $L(E, 1) \neq 0$：曲线在 $s=1$ 处非零，秩为 0；
  - $L(E, 1) = 0, L'(E, 1) \neq 0$：曲线在 $s=1$ 处过一阶零点，秩为 1；
  - $L(E, 1) = L'(E, 1) = 0, L''(E, 1) \neq 0$：二阶零点，秩为 2。
- 用箭头连接解析侧的零点阶数与算术侧的生成元个数。

## 直观解释

椭圆曲线是数论、代数几何和密码学的交汇点。它们是"最简单的非平凡代数簇"——亏格为 1，既不像有理曲线（亏格 0）那样平凡，又不像更高亏格曲线那样复杂。

椭圆曲线上的群结构是其最非凡的性质。从纯代数角度看，$y^2 = x^3 + ax + b$ 只是三次曲线；但几何上，任意两点确定一条直线，直线与三次曲线必交于第三点（Bezout 定理），这为群运算提供了自然的合成法则。无穷远点 $O$ 作为单位元的角色源于：所有垂直线（$x = \mathrm{const}$）在射影平面上都经过 $[0:1:0]$，因此 $P + (-P) = O$ 的几何意义就是过 $P$ 和 $-P$ 的垂直线交曲线于 $O$。

在密码学中，椭圆曲线的优势在于：与乘法群 $\mathbb{F}_q^\times$ 相比，椭圆曲线离散对数问题（ECDLP）目前没有被指数时间算法破解，因此可以用更短的密钥达到同等安全性。

## 例子

1. **$y^2 = x^3 - x$**：判别式 $\Delta = 64 \neq 0$。实点有两个连通分支。$E(\mathbb{Q})_{\mathrm{tors}} \cong \mathbb{Z}/2\mathbb{Z} \times \mathbb{Z}/2\mathbb{Z}$，由点 $(0,0), (1,0), (-1,0)$ 生成。秩为 0，故 $E(\mathbb{Q})$ 恰为这四个挠点。

2. **$y^2 = x^3 + 1$**：$E(\mathbb{Q})_{\mathrm{tors}} = \{O, (-1,0), (0,\pm 1)\} \cong \mathbb{Z}/6\mathbb{Z}$。挠点构成六阶循环群。

3. ** congruent number 问题**：正整数 $n$ 是 congruent number 当且仅当椭圆曲线 $y^2 = x^3 - n^2 x$ 的正秩。这是连接古代数学问题（寻找边长为有理数的面积为 $n$ 的直角三角形）与现代椭圆曲线理论的桥梁。Tunnell 定理在 BSD 猜想假设下给出了 congruent number 的完全判别法。

4. **椭圆曲线密码学（ECC）**：定义在 $\mathbb{F}_p$ 上的椭圆曲线用于密钥交换（ECDH）和数字签名（ECDSA）。例如 Curve25519 由 $y^2 = x^3 + 486662x^2 + x$ 在 $\mathbb{F}_{2^{255}-19}$ 上定义，其设计保证了 $p$-挠子群的阶为大素数，从而抵抗小步大步（baby-step giant-step）和 Pollard $\rho$ 攻击。
