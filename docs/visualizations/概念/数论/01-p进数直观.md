# p进数直观可视化

p进数是数论的完备化，与实数不同，p进数的"大小"由p的幂次决定。

## 可视化

`
p进绝对值:

|p^n * (a/b)|_p = p^(-n)

即: p^n 整除得越多，绝对值越小！

p进整数 Z_p:
  无穷级数展开: a_0 + a_1*p + a_2*p^2 + ...

几何: p进"圆盘"既是开集又是闭集
`
