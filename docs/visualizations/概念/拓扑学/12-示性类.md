# 示性类可视化

## 概念定义

设 $G$ 为拓扑群（如 $U(n), O(n), SU(n), SO(n)$），$EG \to BG$ 为万有主 $G$-丛（$EG$ 可缩）。对任意主 $G$-丛 $P \to X$，存在（在同伦意义下唯一的）分类映射 $f: X \to BG$ 使得 $P \cong f^*EG$。

**示性类（Characteristic Class）** 是丛的拓扑不变量，定义为分类映射在上同调中的 pullback：
$$c(P) := f^*(c) \in H^*(X; R)，$$
其中 $c \in H^*(BG; R)$ 是万有示性类，$R$ 为系数环（通常为 $\mathbb{Z}, \mathbb{Q}, \mathbb{Z}/2\mathbb{Z}$）。

主要示性类包括：
- **陈类（Chern classes）** $c_k(E) \in H^{2k}(X; \mathbb{Z})$：复向量丛（$G = U(n)$）；
- **Pontryagin 类** $p_k(E) \in H^{4k}(X; \mathbb{Z})$：实向量丛（$G = O(n)$）；
- **Euler 类** $e(E) \in H^n(X; \mathbb{Z})$：可定向实向量丛（$G = SO(n)$）；
- **Stiefel-Whitney 类** $w_k(E) \in H^k(X; \mathbb{Z}/2\mathbb{Z})$：实向量丛（$G = O(n)$，模 2 系数）。

## 关键公式

**陈类的公理化定义**：
1. $c_k(E) \in H^{2k}(X; \mathbb{Z})$，$c_0(E) = 1$，$c_k(E) = 0$（$k > \mathrm{rank}\,E$）。
2. **自然性**：$c(f^*E) = f^*c(E)$。
3. **Whitney 和公式**：$c(E \oplus F) = c(E) \smile c(F)$，即
   $$c_k(E \oplus F) = \sum_{i=0}^k c_i(E) \smile c_{k-i}(F)。$$
4. **规范化**：对 $\mathcal{O}(-1) \to \mathbb{CP}^1$，$c_1 = -1$（$H^2(\mathbb{CP}^1) \cong \mathbb{Z}$ 的生成元）。

**全陈类与陈根**：形式地，将全陈类写为
$$c(E) = \prod_{i=1}^n (1 + x_i)，$$
其中 $x_i$ 为形式陈根（first Chern classes of line bundle splittings）。则
$$c_k(E) = e_k(x_1, \ldots, x_n) \quad \text{（第 } k \text{ 个初等对称多项式）。}$$

**Gauss-Bonnet-Chern 定理**：设 $M$ 为 $2n$ 维紧致定向黎曼流形，则
$$\chi(M) = \int_M e(TM) = \frac{1}{(2\pi)^n} \int_M \mathrm{Pf}(\Omega)，$$
其中 $\Omega$ 为曲率形式，$\mathrm{Pf}$ 为 Pfaffian。

## 可视化说明

**陈类的"曲率积叠"图**：

- 取复线丛 $L \to \mathbb{CP}^1$（如 tautological bundle）。$\mathbb{CP}^1 \cong S^2$ 可画为球面。
- $c_1(L)$ 是 $H^2(S^2; \mathbb{Z}) \cong \mathbb{Z}$ 中的元素。对 tautological bundle $\mathcal{O}(-1)$，$c_1 = -1$；对对偶丛 $\mathcal{O}(1)$（超平面丛），$c_1 = +1$。
- **关键可视化**：在球面上画出切丛的向量场。$c_1$ 的几何意义是"扭转的总量"。取球面上一个闭环路，沿线平行移动切向量一周后，向量相对于原方向旋转的角度就是 $c_1$ 在该环路围成区域上的积分。
- 对 $\mathcal{O}(1)$，画出一个只有一个零点的全纯截面（如 $[z_0:z_1] \mapsto z_0/z_1$），零点位于北极。Poincaré-Hopf 定理说零点指标之和等于 $c_1$。

**向量丛的"分裂旗流形"图**：

- 对一个秩 2 复向量丛 $E \to X$，在旗流形 $F(E)$ 上 $E$ 分裂为线丛直和 $L_1 \oplus L_2$。
- 画出底空间 $X$ 中的一条曲线，在其上方画出向量丛的全空间——一个由二维复平面（实四维）纤维组成的"粗管"。
- 在管中画出子丛 $L_1$（一维线纤维），用红色标注。商丛 $E/L_1 \cong L_2$ 用蓝色标注。Whitney 和公式 $c(E) = c(L_1)c(L_2)$ 意味着：整体丛的扭曲等于各分量丛扭曲的乘积。
- 对 $E = TM$（复流形的切丛），$c_1(TM)$ 与 Ricci 曲率相关：$c_1 = [\frac{i}{2\pi}\mathrm{tr}\,\Theta]$，其中 $\Theta$ 为曲率矩阵。

**示性数的"积分计数"图**：

- 画一个闭曲面 $M$（如 $T^2$ 或 $S^2$）。对陈类（或 Euler 类）的最高次幂 $c_n(E)$（或 $e(TM)$），可积分得到示性数。
- 在 $S^2$ 上，画出切向量场的流线。根据 Poincaré-Hopf 定理，向量场零点的指标和等于 $\chi(S^2) = 2$。
- 画出两个源（指标 +1）类型的零点（如球面的北极和南极，经线从北极出发汇聚到南极）。标注：$e(TS^2)[S^2] = \chi(S^2) = 2$。
- 对环面 $T^2$，画出无零点的平行向量场（$\chi(T^2) = 0$）。标注：Euler 类为零，因为可以在整个曲面上定义处处非零的切向量场。

## 直观解释

示性类是拓扑学中最深刻的不变量之一，它们测量了向量丛"从整体到局部无法一致化"的障碍。

- **陈类**测量复向量丛偏离平凡丛的程度。$c_1$ 是一阶障碍：线丛 $L$ 是拓扑平凡的当且仅当 $c_1(L) = 0$。高阶陈类测量了高维的扭曲。
- **Stiefel-Whitney 类**是模 2 的障碍：$w_1(E) = 0$ 当且仅当 $E$ 可定向；$w_2(E) = 0$（对定向丛）当且仅当 $E$ 可 spin 结构。这些条件是物理中定义旋量场的必要前提。
- **Euler 类**直接测量了向量场零点的"必然性"。 hairy ball theorem 说 $S^2$ 上不存在处处非零的连续切向量场，其根源正是 $e(TS^2) \neq 0$。

Atiyah-Singer 指标定理将示性类的积分与椭圆算子的解析指标联系起来，统一了拓扑、分析与几何，是 20 世纪数学最辉煌的成就之一。

## 例子

1. **切丛的陈类**：$\mathbb{CP}^n$ 的切丛满足 $T\mathbb{CP}^n \oplus \mathcal{O} = \mathcal{O}(1)^{\oplus (n+1)}$，故
   $$c(T\mathbb{CP}^n) = (1 + H)^{n+1}，$$
   其中 $H = c_1(\mathcal{O}(1))$ 是超平面类。因此 $c_k(T\mathbb{CP}^n) = \binom{n+1}{k} H^k$。

2. **曲面的分类**：紧致定向曲面由亏格 $g$ 分类。$\chi = 2 - 2g = \int_M e(TM)$。当 $g=0$（球面），$e = 2$；$g=1$（环面），$e = 0$；$g \geq 2$，$e < 0$。这与曲面上是否存在处处非零向量场直接相关。

3. **Yang-Mills 瞬子**：在 $S^4$ 上的 $SU(2)$-主丛由 $c_2 \in H^4(S^4; \mathbb{Z}) \cong \mathbb{Z}$ 分类。Atiyah-Drinfeld-Hitchin-Manin (ADHM) 构造给出了所有自对偶 Yang-Mills 联络（瞬子）的显式参数化，在 $c_2 = k$ 时模空间维数为 $8k - 3$。

4. **指标定理**：对 Dirac 算子 $D$ 在 spin 流形 $M$ 上，
   $$\mathrm{ind}(D) = \hat{A}(TM)[M] = \int_M \hat{A}(p_1, p_2, \ldots)，$$
   其中 $\hat{A}$-genus 由 Pontryagin 类构造。当 $M$ 为四维时，$\hat{A} = 1 - \frac{p_1}{24}$。这联系了拓扑不变量与量子场论中的手征反常。
