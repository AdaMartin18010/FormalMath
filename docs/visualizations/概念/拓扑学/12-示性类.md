# 示性类理论

## 描述

示性类是将向量丛的拓扑信息映射到上同调类的函子性构造。

## 可视化

```mermaid
graph TD
    A[向量丛 E → M] --> B[示性类]
    B --> C[陈类 cᵢ(E) ∈ H²ⁱ(M;ℤ)]
    B --> D[庞特里亚金类 pᵢ(E)]
    B --> E[欧拉类 e(E)]
end

```
