# 层论基础

## 描述

层是拓扑空间上的一种数据结构，将代数对象系统地附着在空间的每一点上。

## 可视化

```mermaid
graph TD
    A[拓扑空间 X] --> B[层 F]
    B --> C[茎 F_x]
    B --> D[截面 F(U)]
    
    E[预层] --> F[满足粘合条件]
    F --> G[成为层]
end

```

Hartshorne, R. (1977). Algebraic Geometry.
