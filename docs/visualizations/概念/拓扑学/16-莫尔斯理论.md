# 莫尔斯理论可视化

## 概念定义

设 $M$ 为 $n$ 维光滑闭流形，$f: M \to \mathbb{R}$ 为光滑函数。点 $p \in M$ 称为 $f$ 的**临界点**，若其微分在该点为零：
$$\mathrm{d}f_p = 0 \in T_p^* M。$$

在临界点 $p$ 处，$f$ 的**黑塞矩阵（Hessian）**是切空间上的对称双线性形式
$$H_f(p)(v, w) := \tilde{v}(\tilde{w}(f))(p), \quad v, w \in T_p M,$$
其中 $\tilde{v}, \tilde{w}$ 是 $v, w$ 的任意光滑延拓。若 $H_f(p)$ 非退化（即作为矩阵行列式非零），则称 $p$ 为**非退化临界点**。

**莫尔斯指标（Morse index）** $\lambda(p)$ 定义为黑塞矩阵 $H_f(p)$ 的负特征值个数（计重数）。莫尔斯函数是指所有临界点均非退化的光滑函数。

## 关键公式

**莫尔斯引理**：设 $p$ 为 $f$ 的非退化临界点，指标为 $\lambda$。则存在 $p$ 附近的局部坐标 $(x^1, \ldots, x^n)$，使得
$$f(x) = f(p) - (x^1)^2 - \cdots - (x^\lambda)^2 + (x^{\lambda+1})^2 + \cdots + (x^n)^2。$$
这意味着在临界点邻域内，$f$ 可被标准二次型完全刻画。

**莫尔斯不等式**：设 $c_\lambda$ 为指标 $\lambda$ 的临界点个數，$b_\lambda = \dim H_\lambda(M; \mathbb{R})$ 为第 $\lambda$ 个 Betti 数，则
$$c_\lambda \geq b_\lambda \quad \text{(弱莫尔斯不等式)}；$$
$$\sum_{i=0}^\lambda (-1)^{\lambda-i} c_i \geq \sum_{i=0}^\lambda (-1)^{\lambda-i} b_i \quad \text{(强莫尔斯不等式)}。$$

**莫尔斯复形与同调**：由临界点生成的自由 Abel 群 $C_\lambda$ 配上由梯度流定义的边界算子 $\partial: C_\lambda \to C_{\lambda-1}$ 构成链复形，其同调恰好等于 $M$ 的奇异同调 $H_*(M; \mathbb{Z})$。

## 可视化说明

**环面高度函数图**：这是莫尔斯理论最经典的可视化。将环面 $T^2$ 竖直放置（像汽车轮胎），考虑高度函数 $f: T^2 \to \mathbb{R}$。

- 图形应展示环面的侧视图，标注四个临界点：
  - **底部点**（$f_{\min}$）：指标 $\lambda = 0$（极小值），黑塞矩阵正定。视觉上是一个"碗底"，所有方向都向上弯曲。
  - **两个鞍点**（中间高度）：指标 $\lambda = 1$。图形中应画出通过鞍点的两条相交曲线——一条是局部极大方向（沿环面内侧圆周），另一条是局部极小方向（沿环面外侧圆周）。鞍点形状像"马鞍"或"山坳"。
  - **顶部点**（$f_{\max}$）：指标 $\lambda = 2$（极大值），黑塞矩阵负定。视觉上是一个"碗顶"，所有方向都向下弯曲。

**胞腔粘贴过程动画帧**：
- **帧 0**（$a < f_{\min}$）：$M^a = \{f \leq a\} = \varnothing$。
- **帧 1**（$f_{\min} < a < $ 鞍点高度）：$M^a$ 是一个拓扑圆盘 $D^2$，由粘贴一个 0-胞腔和一个 2-胞腔组成（实际上是从空集粘贴一个 0-胞腔得到点，再粘贴 2-胞腔填充）。
- **帧 2**（鞍点高度 $< a <$ 顶部）：每次经过一个鞍点，沿其不稳定流形（维数 = 指标 = 1）粘贴一个 1-胞腔 $D^1 = [-1, 1]$。图形应展示两个圆柱段如何被一根"横梁"（1-胞腔）连接，逐渐变成带柄的曲面。
- **帧 3**（$f_{\max} < a$）：最后粘贴一个 2-胞腔封顶，得到完整环面。

**梯度流与 Morse-Smale 复形**：在环面上画出若干条梯度流线（最速上升/下降曲线），它们连接不同临界点。用蓝色箭头表示 $-\nabla f$ 的流向，从鞍点分别流向底部和顶部。稳定流形与不稳定流形横截相交的条件（Morse-Smale 条件）保证了胞腔分解的相容性。图形中可以填充由流线围成的区域，每个区域对应 Morse 复形中的一个生成元。

## 直观解释

莫尔斯理论的核心洞见是：**流形的整体拓扑可以通过研究一个恰当函数（莫尔斯函数）的局部临界点来重构。** 这类似于用等高线地图重建三维地形——只要知道山峰、谷底和鞍点的位置与类型，就能推断出山脉的整体连通结构。

指标 $\lambda$ 的几何意义是：在临界点处，$f$ 在 $\lambda$ 个独立方向上"向下开口"（像山坳），在其余 $n-\lambda$ 个方向上"向上开口"。粘贴一个 $\lambda$-胞腔 $D^\lambda$ 正是在拓扑上添加一个"$\lambda$ 维的洞"。

莫尔斯理论揭示了分析与拓扑之间的深刻桥梁：一个纯分析的对象（光滑函数的临界点）完全编码了流形的纯拓扑不变量（同调群）。这一思想在无限维流形（如环路空间）上的推广——**莫尔斯同调**与**Floer 同调**——是现代辛几何与低维拓扑的基石。

## 例子

1. **球面 $S^n$**：高度函数有两个临界点——南极（指标 0）和北极（指标 $n$）。胞腔分解为一个 0-胞腔和一个 $n$-胞腔，同调群为 $H_0 = H_n = \mathbb{Z}$，其余为零。

2. **实射影空间 $\mathbb{RP}^n$**：可以构造莫尔斯函数使其在每个维数 $0 \leq k \leq n$ 恰好有一个临界点（指标 $k$）。这给出最优的 CW 分解，同调群为
   $$H_k(\mathbb{RP}^n; \mathbb{Z}) = \begin{cases} \mathbb{Z} & k=0 \text{ 或 } k=n \text{（$n$ 奇）} \\ \mathbb{Z}/2\mathbb{Z} & 0 < k < n \text{（$k$ 奇）} \\ 0 & \text{否则} \end{cases}$$

3. **复 Grassmannian** $G_k(\mathbb{C}^n)$：经典的莫尔斯函数（如与射影算子相关的函数）给出 Schubert 胞腔分解。临界点对应 Young 图，指标由图的权重决定。这从分析角度重新解释了代数几何中的 Schubert 演算。

4. **环路空间上的莫尔斯理论**：设 $M$ 为黎曼流形，考虑能量泛函 $E(\gamma) = \int_0^1 \|\dot{\gamma}(t)\|^2 \, \mathrm{d}t$ 在自由环路空间 $\Lambda M$ 上的临界点。$E$ 的临界点恰好是闭测地线。莫尔斯不等式在此情形下给出了流形上闭测地线数量的下界估计，是整体微分几何的经典结果。
