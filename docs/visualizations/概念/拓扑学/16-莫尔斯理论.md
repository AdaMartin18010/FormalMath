# 莫尔斯理论

## 描述

莫尔斯理论研究流形上光滑函数的临界点与流形拓扑的关系。

## 可视化

```mermaid
graph TD
    A[光滑函数 f: M → ℝ] --> B[临界点 ∇f = 0]
    B --> C[黑塞矩阵非退化]
    C --> D[莫尔斯指标]
    D --> E[胞腔粘贴]
end

```
