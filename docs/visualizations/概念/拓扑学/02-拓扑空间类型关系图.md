# 拓扑空间类型关系图

## 描述

本可视化展示各种拓扑空间类型之间的包含关系与蕴含关系，帮助理解从一般拓扑空间到特殊拓扑空间（如度量空间、紧致空间）的分类体系。

## 数学概念

拓扑空间是几何学和代数学的交汇点，研究在连续变形下保持不变的性质。不同类型的拓扑空间具有不同的分离性、紧致性和连通性。
