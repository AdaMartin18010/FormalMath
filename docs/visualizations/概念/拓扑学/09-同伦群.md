# 同伦群序列可视化

## 描述

高阶同伦群 πₙ(X,x₀) 描述了n维球面到空间的映射同伦类。

## 可视化代码

```mermaid
graph LR
    A[π₁] --> B[基本群]
    C[π₂] --> D[二维洞]
    E[πₙ] --> F[n维洞]

    B --> G[非交换]
    D --> H[交换]
    F --> I[交换 n≥2]
end

```
