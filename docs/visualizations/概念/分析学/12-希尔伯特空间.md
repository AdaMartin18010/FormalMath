# 希尔伯特空间理论可视化

## 概念定义

设 $H$ 为复（或实）向量空间。映射 $\langle \cdot, \cdot \rangle: H \times H \to \mathbb{C}$ 称为**内积**，若满足：
1. **共轭对称性**：$\langle x, y \rangle = \overline{\langle y, x \rangle}$；
2. **第一变元线性**：$\langle \alpha x + \beta y, z \rangle = \alpha \langle x, z \rangle + \beta \langle y, z \rangle$；
3. **正定性**：$\langle x, x \rangle \geq 0$，且等号成立当且仅当 $x = 0$。

由内积诱导的范数为 $\|x\| := \sqrt{\langle x, x \rangle}$。若 $(H, \|\cdot\|)$ 作为赋范空间是完备的（所有 Cauchy 列收敛），则称 $H$ 为**希尔伯特空间**。

## 关键公式

**柯西-施瓦茨不等式**：
$$|\langle x, y \rangle| \leq \|x\| \cdot \|y\|，$$
等号成立当且仅当 $x$ 与 $y$ 线性相关。

**平行四边形法则**：希尔伯特空间的范数满足
$$\|x + y\|^2 + \|x - y\|^2 = 2(\|x\|^2 + \|y\|^2)，$$
这是内积空间区别于一般 Banach 空间的特征恒等式。

**正交投影定理**：设 $M \subseteq H$ 为闭子空间，则任意 $x \in H$ 可唯一分解为
$$x = Px + (x - Px)，\quad Px \in M, \quad x - Px \in M^\perp，$$
其中 $P: H \to M$ 是到 $M$ 的正交投影，满足 $\|x - Px\| = \inf_{y \in M} \|x - y\|$。

**Parseval 等式**：若 $\{e_n\}_{n=1}^\infty$ 是标准正交基，则
$$x = \sum_{n=1}^\infty \langle x, e_n \rangle e_n, \quad \|x\|^2 = \sum_{n=1}^\infty |\langle x, e_n \rangle|^2。$$

**Riesz 表示定理**：对每个连续线性泛函 $\varphi \in H^*$，存在唯一的 $y \in H$，使得
$$\varphi(x) = \langle x, y \rangle \quad \text{对所有 } x \in H，$$
且 $\|\varphi\|_{H^*} = \|y\|_H$。这给出了 $H \cong H^*$ 的共轭线性等距同构。

## 可视化说明

**无限维正交基展开图**：想象希尔伯特空间 $H$ 为一个"无限维的欧几里得空间"。

- 在二维示意图中，用一组互相垂直的坐标轴表示标准正交基 $\{e_1, e_2, e_3, \ldots\}$。虽然无法画出无穷多条轴，但可以用箭头标注 $e_1, e_2, e_3$，并以虚线暗示无限延伸。
- 画一个向量 $x \in H$，将其分解为各坐标轴上的投影分量 $\langle x, e_n \rangle e_n$。每个投影分量画为从原点出发、沿 $e_n$ 方向的彩色线段，长度等于 Fourier 系数 $|\langle x, e_n \rangle|$。
- **Parseval 等式的几何可视化**：以这些投影线段为边构造一个"广义直角三角形"的斜边。直角三角形的勾股定理在无限维版本下变为 $\|x\|^2 = \sum |\langle x, e_n \rangle|^2$。图形中用不同颜色标记各 $|\langle x, e_n \rangle|^2$ 的矩形块，它们的总面积等于以 $\|x\|$ 为边长的正方形面积。

**正交投影与最近点图**：

- 画一个平面代表闭子空间 $M \subset H$，平面外有一点 $x$。
- 从 $x$ 向平面 $M$ 作垂线，垂足为 $Px$。在平面 $M$ 上另取任意一点 $y \neq Px$，连接 $x$ 到 $y$。
- 用直角标记强调 $(x - Px) \perp M$，即残差向量垂直于整个子空间。用虚线圆弧标注 $\|x - Px\| < \|x - y\|$，直观展示 $Px$ 是 $M$ 中距 $x$ 最近的点。
- 对 $M$ 为有限维子空间 $\mathrm{span}\{e_1, \ldots, e_n\}$ 的情形，画出 Gram-Schmidt 正交化过程的动画帧：将一组斜交的基向量逐步"掰直"为正交基，同时保持张成的子空间不变。

**$L^2[0,1]$ 中的 Fourier 基图**：

- 取 $H = L^2[0,1]$，标准正交基为 $e_0(t) = 1$，$e_n(t) = \sqrt{2}\cos(2\pi n t)$，$f_n(t) = \sqrt{2}\sin(2\pi n t)$。
- 在 $[0,1]$ 区间上画出前几个基函数的波形图：$e_0$ 是水平直线，$e_1$ 是一个完整周期的余弦波，$e_2$ 是两个周期的余弦波，等等。
- 取一个具体函数如方波 $s(t) = \mathrm{sgn}(\sin(2\pi t))$，画出它的 Fourier 部分和
  $$S_N(t) = \sum_{n=1}^N \frac{2\sqrt{2}}{(2n-1)\pi} \sin(2\pi (2n-1)t)。$$
  随着 $N = 1, 3, 5, 11$ 增大，展示部分和如何逐渐逼近方波，同时注意 Gibbs 现象（在间断点附近出现的约 9%  overshoot）。

## 直观解释

希尔伯特空间是欧几里得几何向无限维的直接推广，保留了"勾股定理"和"正交投影"这两个核心几何直觉。

在有限维中，我们知道任何向量空间都有基，任何向量都可唯一表示为基向量的线性组合。在无限维中，Completeness（完备性）保证了同样的结论成立——只要允许无限级数（并要求按范数收敛）。正交基的存在使得希尔伯特空间的结构与 $\ell^2$ 空间完全等价："所有可分无限维希尔伯特空间彼此等距同构"。这一结论被称为**希尔伯特空间分类定理**，其威力在于：无论你的希尔伯特空间是由函数、序列、还是随机变量构成的，只要它是可分且无限维的，其几何结构与 $\ell^2$ 或 $L^2[0,1]$ 完全相同。

Riesz 表示定理的深刻含义是：在希尔伯特空间中，**对偶空间与空间本身可以视为同一**（通过反线性同构 $y \mapsto \langle \cdot, y \rangle$）。这使得希尔伯特空间上的算子理论（如自伴算子、酉算子、谱分解）具有与有限维矩阵理论极为相似的丰富结构，这是 Banach 空间一般理论所不具备的。

## 例子

1. **序列空间 $\ell^2$**：
   $$\ell^2 = \left\{x = (x_n)_{n=1}^\infty : \sum_{n=1}^\infty |x_n|^2 < \infty\right\}，\quad \langle x, y \rangle = \sum_{n=1}^\infty x_n \overline{y_n}。$$
   标准基 $e_n = (0, \ldots, 0, 1, 0, \ldots)$ 构成正交基。

2. **$L^2$ 空间**：对测度空间 $(X, \mathcal{M}, \mu)$，
   $$L^2(X, \mu) = \left\{f : X \to \mathbb{C} : \int_X |f|^2 \, \mathrm{d}\mu < \infty\right\} / \sim，$$
   内积为 $\langle f, g \rangle = \int_X f \bar{g} \, \mathrm{d}\mu$。当 $X = [0,1]$ 且 $\mu$ 为 Lebesgue 测度时，Fourier 基给出标准正交基。

3. **Sobolev 空间 $H^s(\mathbb{R}^n)$**：
   $$H^s = \{f \in \mathcal{S}' : (1 + |\xi|^2)^{s/2} \hat{f}(\xi) \in L^2(\mathbb{R}^n_\xi)\}。$$
   这是研究偏微分方程的核心空间。Sobolev 嵌入定理描述了 $H^s$ 与连续函数空间之间的关系。

4. **再生核希尔伯特空间（RKHS）**：设 $H$ 为 $\mathbb{C}$ 上区域 $\Omega$ 上全纯函数的希尔伯特空间。若对任意 $w \in \Omega$，赋值泛函 $f \mapsto f(w)$ 连续，则 Riesz 表示定理给出**再生核** $K(z, w)$，满足 $f(w) = \langle f, K(\cdot, w) \rangle$。Bergman 核与 Szegő 核是复分析与几何中的核心对象，其渐近行为与域的边界几何密切相关。
