# 极限的ε-δ定义可视化

## 描述

本可视化直观展示微积分中极限的ε-δ定义的数学含义，通过几何图形和动态过程帮助理解这一核心概念。

## 数学概念

极限是分析学的基础概念。函数f(x)在x→a时的极限为L，其严格定义为：

$$\forall \varepsilon > 0, \exists \delta > 0, \text{使得} 0 < |x - a| < \delta \Rightarrow |f(x) - L| < \varepsilon$$
