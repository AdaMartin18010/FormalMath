# 极限的ε-δ定义可视化

## 1. 严格的极限定义

**定义 1.1（函数极限）**. 设 $f: D \to \mathbb{R}$，$a$ 为 $D$ 的聚点。称 $f$ 在 $x \to a$ 时的极限为 $L$，记作 $\lim_{x \to a} f(x) = L$，若
$$\forall \varepsilon > 0, \exists \delta > 0: 0 < |x - a| < \delta \Rightarrow |f(x) - L| < \varepsilon.$$

**定义 1.2（序列极限）**. 序列 $\{a_n\}$ 的极限为 $L$，记作 $\lim_{n \to \infty} a_n = L$，若
$$\forall \varepsilon > 0, \exists N \in \mathbb{N}: n > N \Rightarrow |a_n - L| < \varepsilon.$$

## 2. ε-δ 定义的几何解释

极限定义的精髓在于**"任意接近"**的精确化：

- **挑战者**给出任意小的误差容限 $\varepsilon > 0$（水平带的半宽）；
- **回应者**找到一个距离 $\delta > 0$（垂直带的半宽），使得当 $x$ 落在 $(a-\delta, a+\delta)$ 内时，$f(x)$ 必定落在 $(L-\varepsilon, L+\varepsilon)$ 内。

这意味着：**无论对方要求多高的精度，我总能保证在足够接近 $a$ 时达到该精度。**

```
    垂直带: x ∈ (a-δ, a+δ)    水平带: f(x) ∈ (L-ε, L+ε)
```

## 3. 极限的运算性质

**定理 3.1（极限四则运算）**. 设 $\lim_{x \to a} f(x) = L$，$\lim_{x \to a} g(x) = M$，则：
1. $\lim_{x \to a} (f(x) + g(x)) = L + M$；
2. $\lim_{x \to a} (f(x) \cdot g(x)) = L \cdot M$；
3. 若 $M \neq 0$，$\lim_{x \to a} \frac{f(x)}{g(x)} = \frac{L}{M}$。

*证明（加法）*. 给定 $\varepsilon > 0$，取 $\delta_1$ 使 $|x-a|<\delta_1 \Rightarrow |f(x)-L|<\varepsilon/2$，取 $\delta_2$ 使 $|x-a|<\delta_2 \Rightarrow |g(x)-M|<\varepsilon/2$。令 $\delta = \min(\delta_1, \delta_2)$，则
$$|(f(x)+g(x))-(L+M)| \leq |f(x)-L| + |g(x)-M| < \varepsilon.$$
$\square$

**定理 3.2（夹逼定理）**. 若 $g(x) \leq f(x) \leq h(x)$ 在 $a$ 的某去心邻域内成立，且 $\lim_{x \to a} g(x) = \lim_{x \to a} h(x) = L$，则 $\lim_{x \to a} f(x) = L$。

*证明*. 给定 $\varepsilon > 0$，存在 $\delta$ 使得 $0 < |x-a| < \delta$ 时 $g(x), h(x) \in (L-\varepsilon, L+\varepsilon)$。由夹逼，$f(x)$ 亦在此区间。$\square$

## 4. 连续性与一致连续

**定义 4.1（连续性）**. $f$ 在 $a$ 连续，若 $\lim_{x \to a} f(x) = f(a)$。等价地：
$$\forall \varepsilon > 0, \exists \delta > 0: |x - a| < \delta \Rightarrow |f(x) - f(a)| < \varepsilon.$$

**定义 4.2（一致连续）**. $f$ 在集合 $D$ 上一致连续，若
$$\forall \varepsilon > 0, \exists \delta > 0: \forall x, y \in D, |x - y| < \delta \Rightarrow |f(x) - f(y)| < \varepsilon.$$

关键区别：一致连续中 $\delta$ 只依赖 $\varepsilon$ 而不依赖点的位置。

**定理 4.3**. 紧集上的连续函数必一致连续。

*证明概要*. 对每点用连续性得局部 $\delta$，紧集有有限子覆盖，取最小 $\delta$。$\square$

## 5. 重要例子

**例子 5.1**. 证明 $\lim_{x \to 0} \frac{\sin x}{x} = 1$。

对 $0 < x < \pi/2$，由单位圆几何，$\sin x < x < \tan x$。取倒数并乘以 $\sin x$：
$$\cos x < \frac{\sin x}{x} < 1.$$
由 $\lim_{x \to 0} \cos x = 1$ 和夹逼定理即得。对 $x < 0$，由偶性同样成立。

**例子 5.2**. 证明 $\lim_{n \to \infty} \left(1 + \frac{1}{n}\right)^n = e$。

记 $a_n = (1 + 1/n)^n$。由二项式定理，
$$a_n = \sum_{k=0}^n \binom{n}{k}\frac{1}{n^k} = \sum_{k=0}^n \frac{1}{k!}\prod_{j=0}^{k-1}\left(1-\frac{j}{n}\right).$$
可证 $\{a_n\}$ 单调递增有上界（如 $a_n < \sum_{k=0}^n \frac{1}{k!} < 3$），故收敛。其极限定义为 $e$。
