# 自由群构造可视化

## 描述

自由群是由生成元生成的没有任何关系的群，是群论中最基本的构造。

## 可视化代码

```mermaid
graph TD
    A[集合 S] --> B[自由群 F(S)]
    B --> C[字 word]
    C --> D[约化字]
    D --> E[群运算: 连接+约化]

    B --> F[泛性质]
    F --> G[任意映射 S→G 唯一延拓为同态]
end

```
