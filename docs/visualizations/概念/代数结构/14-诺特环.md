# 诺特环

## 描述

诺特环满足升链条件，是代数几何的基础。

## 可视化

```mermaid
graph TD
    A[升链 I₁ ⊂ I₂ ⊂ I₃ ⊂ ...] --> B[升链条件]
    B --> C[存在N, I_N = I_{N+1} = ...]
    
    D[等价条件] --> E[理想有限生成]
end

```
