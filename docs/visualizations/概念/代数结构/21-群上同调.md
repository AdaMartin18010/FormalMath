# 群上同调

## 描述

群上同调是研究群作用的重要工具，定义为导出函子。

## 可视化

```mermaid
graph TD
    A[群 G] --> B[G-模 M]
    B --> C[不变量 M^G]
    C --> D[导出函子 H^n(G,M)]
    
    E[H²(G,M)] --> F[群扩张分类]
end

```

## 参考

Brown, K. S. (1982). Cohomology of Groups.
