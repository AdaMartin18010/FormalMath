# 西罗定理结构

## 描述

西罗定理给出了有限群中p子群的存在性、共轭性和计数公式。

## 可视化

```mermaid
graph TD
    A[|G| = pⁿ·m] --> B[Sylow p子群]

    B --> C[存在性: n_p ≥ 1]
    B --> D[共轭性: 所有Sylow p子群共轭]
    B --> E[n_p ≡ 1 mod p]
end

```
