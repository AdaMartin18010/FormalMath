# visualizations

本目录汇集了FormalMath项目的辅助性可视化资源，包含定理、应用、思维导图、概念和证明等多个维度的图表与图形化内容，旨在通过直观的视觉呈现帮助理解抽象的数学结构与逻辑关系。

## 内容导航

- **定理** — 数学定理的可视化呈现
- **应用** — 数学模型与实际应用的图示
- **思维导图** — 知识结构与发展脉络导图
- **概念** — 核心概念的图形化解释
- **证明** — 经典证明过程的步骤可视化
