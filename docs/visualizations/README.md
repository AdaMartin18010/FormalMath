# FormalMath 可视化文档集

本目录汇集了 FormalMath 项目中数学概念、定理、证明与应用的可视化解释文档。每篇文档不仅给出严格的数学定义与公式，更通过文字描述的图像/图表设计，帮助读者建立抽象数学结构的几何直觉。

## 内容导航

### 概念可视化

通过图形化方式解释核心数学概念的结构与性质：

- **数论**：p进数的树状拓扑、椭圆曲线的群结构等
- **几何学**：复几何的 Hodge 菱形、辛几何的相空间面积守恒、凯勒流形的兼容结构等
- **拓扑学**：层论的拼图粘合、CW 复形的胞腔粘贴、Morse 理论的临界点分层、同伦群的球面映射等
- **代数结构**：群上同调的扩张分类、自由群的生成树、模论的线性表示、诺特环的理想升链等
- **分析学**：哈代空间的边界值重构、希尔伯特空间的正交投影、巴拿赫空间的几何等

### 定理可视化

用文字详细描述经典定理的图示化呈现方式：

- **代数**：若尔当标准型的广义特征向量塔、希尔伯特基定理的首项锥、同构定理的对应格、西罗定理的子群计数等
- **分析**：单调收敛与控制收敛的积分区域逼近、里斯表示定理的对偶同构等
- **几何拓扑**：黎曼-罗赫定理的除子天平、高斯-博内定理的角度亏量等

### 证明可视化

描述经典证明策略的结构化图示：

- 归纳法的多米诺骨牌、极大原理的链上界、反证法的矛盾爆炸图等

### 应用可视化

展示数学理论在自然科学和工程中的直观图像：

- **物理学**：麦克斯韦方程组的场线结构、量子力学的希尔伯特空间演化等
- **计算机科学**：支持向量机的最大间隔超平面、核技巧的特征空间提升等

### 思维导图

以层次化导图形式呈现逻辑数学概念的知识结构。

## 文档标准

每篇可视化文档包含：

1. **严格定义**：概念的精确数学定义
2. **关键公式**：核心定理与恒等式的 LaTeX 表述
3. **可视化说明**：用文字详细描述应绘制的图像/动画内容，包括颜色、布局、标注和动态演化
4. **直观解释**：将抽象概念转化为几何直觉和物理类比
5. **数学例子**：具体计算和经典实例
6. **参考文献**：经典教材与原始文献
