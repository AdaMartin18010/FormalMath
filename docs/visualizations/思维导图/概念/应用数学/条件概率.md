# 条件概率 (Conditional Probability)

## 中心概念精确定义

**条件概率**是概率论中描述"在已知某些信息条件下"事件发生可能性的核心概念。给定概率空间 $(\Omega, \mathcal{F}, P)$，对于事件 $B \in \mathcal{F}$ 且 $P(B) > 0$，事件 $A$ 在 $B$ 发生的条件下的条件概率定义为：

$$P(A|B) = \frac{P(A \cap B)}{P(B)}$$

这一定义可以直观理解为：将样本空间限制在 $B$ 内，重新归一化概率测度。

**条件概率测度**：对于固定的 $B$，$P(\cdot|B)$ 本身构成一个概率测度，满足Kolmogorov公理。

---

## 核心要素

### 1. Bayes定理 (Bayes' Theorem)

**基本形式**：
$$P(A|B) = \frac{P(B|A)P(A)}{P(B)}$$

其中：
- $P(A)$：先验概率（Prior）
- $P(A|B)$：后验概率（Posterior）
- $P(B|A)$：似然（Likelihood）

- $P(B)$：证据（Evidence）

**推广形式**：若 $\{A_i\}_{i=1}^n$ 是 $\Omega$ 的一个划分，则
$$P(A_j|B) = \frac{P(B|A_j)P(A_j)}{\sum_{i=1}^n P(B|A_i)P(A_i)}$$

**Bayesian推断框架**：
$$\text{Posterior} \propto \text{Likelihood} \times \text{Prior}$$

### 2. 条件期望 (Conditional Expectation)

**初等定义**：给定事件 $B$，$E[X|B] = \frac{E[X \cdot 1_B]}{P(B)}$

**现代定义**：给定子$\sigma$-代数 $\mathcal{G} \subseteq \mathcal{F}$，条件期望 $E[X|\mathcal{G}]$ 是满足以下条件的随机变量：
- $E[X|\mathcal{G}]$ 是 $\mathcal{G}$-可测的
- $\forall G \in \mathcal{G}: \int_G E[X|\mathcal{G}] dP = \int_G X dP$

**关键性质**：
- **塔性质（Tower Property）**：若 $\mathcal{H} \subseteq \mathcal{G}$，则 $E[E[X|\mathcal{G}]|\mathcal{H}] = E[X|\mathcal{H}]$
- **全期望公式**：$E[E[X|\mathcal{G}]] = E[X]$
- **线性性**：$E[aX + bY|\mathcal{G}] = aE[X|\mathcal{G}] + bE[Y|\mathcal{G}]$
- **提取已知量**：若 $Y$ 是 $\mathcal{G}$-可测的，则 $E[XY|\mathcal{G}] = Y E[X|\mathcal{G}]$

### 3. 鞅 (Martingale)

**定义**：随机过程 $\{M_n, \mathcal{F}_n\}_{n \geq 0}$ 称为鞅，如果：
1. $E[|M_n|] < \infty$ 对所有 $n$

2. $M_n$ 是 $\mathcal{F}_n$-可测的（适应性）
3. $E[M_{n+1}|\mathcal{F}_n] = M_n$ a.s.

**类型**：
- **上鞅（Supermartingale）**：$E[M_{n+1}|\mathcal{F}_n] \leq M_n$（公平游戏中的不利玩家）
- **下鞅（Submartingale）**：$E[M_{n+1}|\mathcal{F}_n] \geq M_n$（公平游戏中的有利玩家）

**重要例子**：
- 简单随机游走（对称时）
- Wald鞅：$M_n = \exp(\theta S_n - n\psi(\theta))$
- Doob鞅：$M_n = E[X|\mathcal{F}_n]$

### 4. 停时与可选停时定理 (Stopping Time & Optional Stopping)

**停时定义**：随机变量 $T: \Omega \to \{0, 1, 2, ...\} \cup \{\infty\}$ 称为关于 $\{\mathcal{F}_n\}$ 的停时，如果
$$\{T = n\} \in \mathcal{F}_n, \quad \forall n \geq 0$$

直观解释：根据到时刻 $n$ 的信息，可以决定是否停止。

**可选停时定理**：在适当条件下（如有界停时、鞅有界等），
$$E[M_T] = E[M_0]$$

**应用**：赌徒破产问题、 Wald等式、金融衍生品定价。

### 5. 条件独立 (Conditional Independence)

事件 $A$ 和 $B$ 在给定 $C$ 条件下独立，如果：
$$P(A \cap B|C) = P(A|C)P(B|C)$$

**条件独立与Bayesian网络**：
- 用于简化联合概率分布的表示
- 关键概念：d-分离、Markov性质
- 应用：概率图模型、因果推断

### 6. 条件方差与方差分解

**条件方差**：$\text{Var}(X|\mathcal{G}) = E[(X - E[X|\mathcal{G}])^2|\mathcal{G}]$

**方差分解公式（Law of Total Variance）**：
$$\text{Var}(X) = E[\text{Var}(X|\mathcal{G})] + \text{Var}(E[X|\mathcal{G}])$$

解释：总方差 = 组内方差的期望 + 组间方差

## 性质与定理

### 定理1：全概率公式

设 $\{B_i\}_{i=1}^n$ 是样本空间 $\Omega$ 的一个划分（即互不相交且并集为$\Omega$），则对任意事件 $A$：
$$P(A) = \sum_{i=1}^n P(A|B_i)P(B_i)$$

连续版本：若 $X$ 有密度 $f_X$，则
$$P(A) = \int_{-\infty}^{\infty} P(A|X=x)f_X(x)dx$$

### 定理2：Bayesian更新的一致性

在序列观测 $D_1, D_2, ...$ 下，后验分布可以序贯更新：
$$P(\theta|D_1, D_2) \propto P(D_2|\theta, D_1)P(\theta|D_1)$$

这保证了Bayesian推断的一致性。

### 定理3：鞅收敛定理

**Doob收敛定理**：设 $\{M_n\}$ 是上鞅且 $\sup_n E[M_n^-] < \infty$，则 $M_n \to M_\infty$ a.s.

**L²鞅收敛**：若 $\{M_n\}$ 是鞅且 $\sup_n E[M_n^2] < \infty$，则 $M_n$ 在 $L^2$ 和 a.s. 意义下收敛。

### 定理4：Doob分解定理

任何可积的适应过程 $\{X_n, \mathcal{F}_n\}$ 可以唯一分解为：
$$X_n = M_n + A_n$$
其中 $\{M_n\}$ 是鞅，$\{A_n\}$ 是可料（$A_n$ 是 $\mathcal{F}_{n-1}$-可测）且可积的递增过程。

### 定理5：条件期望的收敛性

若 $X_n \xrightarrow{L^1} X$，则 $E[X_n|\mathcal{G}] \xrightarrow{L^1} E[X|\mathcal{G}]$。

若 $X_n \uparrow X$ a.s. 且 $X_n \geq 0$，则 $E[X_n|\mathcal{G}] \uparrow E[X|\mathcal{G}]$ a.s.

## 典型例子

### 例子1：Monty Hall问题

**问题设置**：三扇门后有一辆汽车和两只山羊。参赛者选一扇门后，主持人（知道门后情况）打开另一扇有山羊的门，问参赛者是否应该换门？

**分析**：设 $C_i$ 表示汽车在第 $i$ 扇门后，$H_j$ 表示主持人打开第 $j$ 扇门。

$$P(C_1|H_3) = \frac{P(H_3|C_1)P(C_1)}{P(H_3)} = \frac{\frac{1}{2} \cdot \frac{1}{3}}{\frac{1}{2}} = \frac{1}{3}$$

$$P(C_2|H_3) = \frac{P(H_3|C_2)P(C_2)}{P(H_3)} = \frac{1 \cdot \frac{1}{3}}{\frac{1}{2}} = \frac{2}{3}$$

**结论**：换门获胜概率为 2/3，不换为 1/3。

### 例子2：随机游走与赌徒破产

**问题**：赌徒有 $i$ 元，每次以概率 $p$ 赢1元，以概率 $q=1-p$ 输1元，直到破产（0元）或达到目标 $N$ 元。求破产概率。

**解**：设 $u_i$ 为从 $i$ 元开始的破产概率。由全期望公式：
$$u_i = p \cdot u_{i+1} + q \cdot u_{i-1}, \quad u_0 = 1, u_N = 0$$

解得：
- 当 $p = q = 1/2$：$u_i = 1 - \frac{i}{N}$
- 当 $p \neq q$：$u_i = \frac{(q/p)^i - (q/p)^N}{1 - (q/p)^N}$

**鞅的应用**：当 $p = q$ 时，$X_n$ 是鞅，用可选停时定理 $E[X_T] = E[X_0] = i$。

### 例子3：Bayesian参数估计

**场景**：估计一枚硬币正面朝上的概率 $\theta$，观测到 $n$ 次抛掷中有 $k$ 次正面。

**先验**：取共轭先验 $\theta \sim \text{Beta}(\alpha, \beta)$

**似然**：$P(k|\theta) = \binom{n}{k}\theta^k(1-\theta)^{n-k}$

**后验**：
$$\theta|k \sim \text{Beta}(\alpha + k, \beta + n - k)$$

**后验期望**：$E[\theta|k] = \frac{\alpha + k}{\alpha + \beta + n}$

当 $n \to \infty$，$E[\theta|k] \to \frac{k}{n}$（MLE估计）。

## 关联概念

### 上游概念
- **概率测度**：条件概率的基础
- **$\sigma$-代数**：信息结构的数学表示
- **Lebesgue积分**：条件期望的严格定义

### 下游概念
- **随机过程**：Markov链、鞅、Brown运动
- **数理统计**：Bayesian统计、序贯分析
- **金融数学**：风险中性测度、衍生品定价
- **信息论**：条件熵、互信息
- **机器学习**：概率图模型、EM算法

### 应用领域
- **金融工程**：期权定价（Black-Scholes）、风险度量
- **人工智能**：贝叶斯网络、隐马尔可夫模型
- **生物统计**：临床试验、生存分析
- **信号处理**：卡尔曼滤波、粒子滤波
- **博弈论**：不完全信息博弈
