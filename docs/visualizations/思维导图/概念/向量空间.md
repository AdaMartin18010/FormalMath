# 向量空间

## 1. 定义与例子

**定义 1.1（向量空间）**. 域 $\mathbb{F}$ 上的向量空间是四元组 $(V, \mathbb{F}, +, \cdot)$，其中 $V$ 为集合，$+$ 为向量加法，$\cdot$ 为数乘，满足八条公理：
- $(V, +)$ 为 Abel 群；
- $1 \cdot v = v$；
- $a(bv) = (ab)v$；
- $(a+b)v = av + bv$；
- $a(v+w) = av + aw$。

**例子 1.2**.
- $\mathbb{F}^n$：坐标空间；
- $M_{m \times n}(\mathbb{F})$：$m \times n$ 矩阵空间，维数 $mn$；
- $\mathbb{F}[x]$：多项式空间，无限维，基 $\{1, x, x^2, \dots\}$；
- $C([0,1])$：连续函数空间，无限维；
- $\ell^p = \{(x_n) : \sum |x_n|^p < \infty\}$：序列空间（$p \geq 1$）。

## 2. 基与维数

**定义 2.1（线性无关与生成）**.
- 向量集 $\{v_1, \dots, v_n\}$ **线性无关**，若 $\sum a_i v_i = 0 \Rightarrow a_i = 0$；
- **生成集**：$\operatorname{span}\{v_i\} = \{\sum a_i v_i\}$；
- **基**：既是线性无关又是生成集。

**定理 2.2（基的扩张定理）**. 任何线性无关集可扩张为基；任何生成集可约化为基。

**定理 2.3（维数不变性）**. 有限维向量空间的所有基含同样多个元素。此数称为**维数** $\dim V$。

*证明*. 设 $\{u_1, \dots, u_m\}$ 和 $\{v_1, \dots, v_n\}$ 为基。将 $v_1$ 表为 $u_i$ 的线性组合，某系数非零，替换对应 $u_i$ 后仍为基。归纳替换所有 $v_j$，得 $n \leq m$。对称地 $m \leq n$。$\square$

**推论 2.4**. $\dim \mathbb{F}^n = n$；$\dim M_{m \times n}(\mathbb{F}) = mn$。

## 3. 线性映射

**定义 3.1（线性映射）**. $T: V \to W$ 为线性映射，若 $T(av + bw) = aT(v) + bT(w)$。

**定义 3.2（核与像）**.
- $\ker T = \{v \in V : T(v) = 0\}$；
- $\operatorname{im} T = \{T(v) : v \in V\}$。

**定理 3.3（秩-零化度定理）**. 对有限维 $V$，
$$\dim V = \dim \ker T + \dim \operatorname{im} T.$$

*证明*. 取 $\ker T$ 的基 $\{v_1, \dots, v_k\}$，扩张为 $V$ 的基 $\{v_1, \dots, v_k, w_1, \dots, w_r\}$。则 $\{T(w_1), \dots, T(w_r)\}$ 为 $\operatorname{im} T$ 的基。$\square$

## 4. 矩阵表示

**定义 4.1（矩阵表示）**. 取 $V$ 的基 $\mathcal{B} = \{v_1, \dots, v_n\}$，$W$ 的基 $\mathcal{C} = \{w_1, \dots, w_m\}$。$T$ 的矩阵表示 $[T]_{\mathcal{C}}^{\mathcal{B}}$ 的第 $j$ 列为 $T(v_j)$ 在 $\mathcal{C}$ 下的坐标。

**定理 4.2（基变换）**. 对 $V$ 的两组基 $\mathcal{B}, \mathcal{B}'$，基变换矩阵 $P = [I]_{\mathcal{B}'}^{\mathcal{B}}$ 可逆，且
$$[T]_{\mathcal{B}'} = P^{-1}[T]_{\mathcal{B}}P.$$

## 5. 特征值与对角化

**定义 5.1（特征值）**. $T: V \to V$ 的特征值 $\lambda$ 满足 $T(v) = \lambda v$ 对某 $v \neq 0$。

**定义 5.2（特征多项式）**. $p_T(\lambda) = \det(\lambda I - T)$。特征值为 $p_T$ 的根。

**定理 5.3（Cayley-Hamilton）**. $p_T(T) = 0$（零变换）。

**定理 5.4（谱定理）**. 有限维复内积空间上，正规算子（$T^*T = TT^*$）可酉对角化。Hermitian 算子的特征值为实数；酉算子的特征值模为 1。

## 6. 对偶空间与张量积

**定义 6.1（对偶空间）**. $V^* = \{\varphi: V \to \mathbb{F} \text{ 线性}\}$。若 $\dim V = n$，则 $\dim V^* = n$。

**定义 6.2（张量积）**. $V \otimes W$ 是由 $
\{v \otimes w\}$ 生成的空间，满足双线性关系。

**定理 6.3**. $\dim(V \otimes W) = (\dim V)(\dim W)$。$(V \otimes W)^* \cong V^* \otimes W^*$（有限维）。
