# 积分

## 1. Riemann 积分

**定义 1.1（分划与 Riemann 和）**. 对 $[a,b]$ 的分划 $P = \{x_0, x_1, \dots, x_n\}$（$a = x_0 < x_1 < \cdots < x_n = b$），取样点 $\xi_i \in [x_{i-1}, x_i]$。Riemann 和为
$$S(f, P, \xi) = \sum_{i=1}^n f(\xi_i) \Delta x_i, \quad \Delta x_i = x_i - x_{i-1}.$$

**定义 1.2（Riemann 可积）**. $f$ 在 $[a,b]$ 上 Riemann 可积，若存在 $I \in \mathbb{R}$ 使得对任意 $\varepsilon > 0$，存在 $\delta > 0$，当 $\max \Delta x_i < \delta$ 时，$|S(f,P,\xi) - I| < \varepsilon$。此 $I$ 记为 $\int_a^b f(x)\,dx$。

**定理 1.3（可积性判别）**. 有界函数 $f$ Riemann 可积当且仅当不连续点集为零测集（Lebesgue 判别准则）。

特别地，连续函数、单调函数、仅有有限个间断点的有界函数均 Riemann 可积。

## 2. 微积分基本定理

**定理 2.1（微积分基本定理，第一部分）**. 设 $f$ 在 $[a,b]$ 上连续，定义
$$F(x) = \int_a^x f(t)\,dt.$$
则 $F$ 在 $[a,b]$ 上可导，且 $F'(x) = f(x)$。

*证明*. 
$$\frac{F(x+h) - F(x)}{h} = \frac{1}{h}\int_x^{x+h} f(t)\,dt.$$
由积分中值定理，存在 $\xi$ 在 $x$ 与 $x+h$ 之间使上式等于 $f(\xi)$。当 $h \to 0$，$\xi \to x$，由连续性 $f(\xi) \to f(x)$。$\square$

**定理 2.2（微积分基本定理，第二部分）**. 若 $F' = f$ 在 $[a,b]$ 上连续，则
$$\int_a^b f(x)\,dx = F(b) - F(a).$$

## 3. 积分技巧

**定理 3.1（换元法）**. 设 $g$ 在 $[\alpha, \beta]$ 上连续可导，$f$ 在 $g([\alpha, \beta])$ 上连续。则
$$\int_\alpha^\beta f(g(x))g'(x)\,dx = \int_{g(\alpha)}^{g(\beta)} f(u)\,du.$$

**定理 3.2（分部积分）**. $u, v$ 连续可导，则
$$\int_a^b u(x)v'(x)\,dx = u(x)v(x)\big|_a^b - \int_a^b u'(x)v(x)\,dx.$$

*证明*. 由 $(uv)' = u'v + uv'$ 积分即得。$\square$

## 4. Lebesgue 积分

**定义 4.1（Lebesgue 积分）**. 对简单函数 $\varphi = \sum_{i=1}^n a_i \chi_{E_i}$（$E_i$ 可测），定义 $\int \varphi = \sum a_i \mu(E_i)$。对非负可测函数 $f$，定义
$$\int f\,d\mu = \sup\left\{\int \varphi : 0 \leq \varphi \leq f, \varphi \text{ 简单}\right\}.$$
对一般可测函数，分解为正部负部 $f = f^+ - f^-$。

**定理 4.2（单调收敛定理）**. 若 $0 \leq f_1 \leq f_2 \leq \cdots$ 且 $f_n \to f$ 逐点，则
$$\lim_{n \to \infty} \int f_n\,d\mu = \int f\,d\mu.$$

**定理 4.3（控制收敛定理）**. 若 $f_n \to f$ 逐点，且 $|f_n| \leq g$ 对某可积 $g$，则
$$\lim_{n \to \infty} \int f_n\,d\mu = \int f\,d\mu.$$

## 5. 重积分与 Fubini 定理

**定理 5.1（Fubini）**. 若 $f: X \times Y \to \mathbb{R}$ 可积（关于乘积测度），则
$$\int_{X \times Y} f(x,y)\,d(\mu \times \nu) = \int_X \left(\int_Y f(x,y)\,d\nu\right) d\mu = \int_Y \left(\int_X f(x,y)\,d\mu\right) d\nu.$$

**定义 5.2（Jordan 可测）**. 有界集 $E \subset \mathbb{R}^n$ 为 Jordan 可测，若其特征函数 $\chi_E$ Riemann 可积。等价地，边界 $\partial E$ 为零测集。

## 6. 曲线与曲面积分

**定义 6.1（曲线积分）**. 对曲线 $\gamma: [a,b] \to \mathbb{R}^n$，
$$\int_\gamma f\,ds = \int_a^b f(\gamma(t))|\gamma'(t)|\,dt.$$

**定义 6.2（曲面积分）**. 对参数曲面 $r: D \to \mathbb{R}^3$，
$$\iint_S f\,dS = \iint_D f(r(u,v))|r_u \times r_v|\,du\,dv.$$
