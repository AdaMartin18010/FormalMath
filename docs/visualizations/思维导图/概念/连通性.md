# 连通性

## 1. 连通空间

**定义 1.1（连通）**. 拓扑空间 $X$ 称为**连通的**，若它不能表示为两个非空不交开集的并。等价地，$X$ 中既开又闭的子集只有 $\varnothing$ 和 $X$。

**定义 1.2（道路连通）**. $X$ 称为**道路连通的**，若对任意 $x, y \in X$，存在连续映射 $\gamma: [0,1] \to X$ 使 $\gamma(0) = x$，$\gamma(1) = y$。

**定理 1.3**. 道路连通蕴含连通，但逆不成立。

*证明*. 若 $X = U \cup V$（不交非空开集），取 $x \in U$，$y \in V$。道路 $\gamma: [0,1] \to X$ 连续，$\gamma^{-1}(U)$ 和 $\gamma^{-1}(V)$ 将 $[0,1]$ 分离，矛盾（因 $[0,1]$ 连通）。$\square$

**例子 1.4（连通但非道路连通）**. **拓扑学家的正弦曲线**：
$$S = \{(x, \sin(1/x)) : 0 < x \leq 1\} \cup \{(0, y) : -1 \leq y \leq 1\}.$$
$S$ 连通（曲线加极限纵轴段），但 $(0,0)$ 与曲线上任一点无道路相连。

## 2. 连通分支

**定义 2.1（连通分支）**. $x$ 的**连通分支**是包含 $x$ 的最大连通子集：
$$C(x) = \bigcup\{A \subseteq X : x \in A, A \text{ 连通}\}.$$

**性质 2.2**.
- 连通分支是闭集（但未必开）；
- 不同的连通分支互不相交；
- $X$ 为连通分支的不交并。

**定义 2.3（局部连通）**. $X$ 为**局部连通的**，若每点有由连通集组成的邻域基。

## 3. 道路连通分支与同伦

**定义 3.1（同伦）**. 映射 $f, g: X \to Y$ 称为**同伦的**（$f \simeq g$），若存在连续 $H: X \times [0,1] \to Y$ 使 $H(x,0) = f(x)$，$H(x,1) = g(x)$。

**定义 3.2（单连通）**. $X$ 为**单连通的**，若道路连通且每条环路同伦于常值环路（可连续缩为一点）。

等价地：基本群 $\pi_1(X, x_0) = 0$。

**例子 3.3**.
- $\mathbb{R}^n$ 单连通；
- $S^1$ 道路连通但非单连通（$\pi_1(S^1) = \mathbb{Z}$）；
- $S^n$（$n \geq 2$）单连通。

## 4. 连通性的应用

**定理 4.1（介值定理）**. 设 $X$ 连通，$f: X \to \mathbb{R}$ 连续。若 $a, b \in f(X)$ 且 $a < c < b$，则 $c \in f(X)$。

*证明*. $f(X)$ 为 $\mathbb{R}$ 的连通子集，即为区间。$\square$

**定理 4.2（不动点定理，一维 Brouwer）**. 连续 $f: [0,1] \to [0,1]$ 有不动点。

*证明*. 令 $g(x) = f(x) - x$。$g(0) \geq 0$，$g(1) \leq 0$。由介值定理，存在 $c$ 使 $g(c) = 0$。$\square$

## 5. 高维连通：同调与同伦

**定义 5.1（高阶同伦群）**. $\pi_n(X, x_0)$ 为基于 $x_0$ 的 $n$-维球面映射 $S^n \to X$ 的同伦类群。

**定理 5.2（Hurewicz）**. 若 $X$ 道路连通且 $\pi_k(X) = 0$（$k < n$），则 $\pi_n(X) \cong H_n(X)$。
