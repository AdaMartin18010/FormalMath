# 分布理论 思维导图

## 中心概念
分布理论（广义函数论）是微分方程和数学物理的革命性工具，它扩展了函数概念，使所有分布都可任意次微分。Dirac δ函数等奇异对象在此框架下获得严格数学基础。

### 定义与公理
- **测试函数空间** $\mathcal{D}(\Omega)$: $C_c^\infty(\Omega)$ 配备归纳极限拓扑
- **分布**: 连续线性泛函 $T: \mathcal{D}(\Omega) \to \mathbb{C}$
- **Schwartz空间** $\mathcal{S}$: 速降光滑函数空间
- **缓增分布** $\mathcal{S}'$: $\mathcal{S}$ 上的连续线性泛函

### 基本性质
- **任意可微**: 所有分布有任意阶导数
- **局部化**: 分布可限制到开集，可粘贴
- **支集**: 分布作用为零的最大开集的补集
- **结构定理**: 每个分布局部是连续函数的导数

### 重要例子
- **Dirac δ**: $\langle \delta, \phi \rangle = \phi(0)$
- **主值分布**: $\text{p.v.}\frac{1}{x}$
- **有限部分**: $\text{Pf.}\frac{1}{x^2}$
- **层状分布**: 沿超曲面的δ函数
- **Heaviside函数**: $H(x)$，$H' = \delta$

### 核心定理
- **结构定理**: 紧支分布是连续函数导数的有限和
- **Schwartz核定理**: 分布核表示连续线性映射（证明思路：抽象核定理）
- ** Paley-Wiener定理**: 紧支分布的Fourier变换刻画
- **基本解存在性**: 常系数偏微分算子总有基本解（Ehrenpreis-Malgrange）
- **Hörmander定理**: 解的奇性传播

### 相关概念
- **父概念**: 泛函分析、测度论、偏微分方程
- **子概念**: 超函数、微局部分析、Colombeau代数
- **相邻概念**: Sobolev空间、Fourier分析、算子理论

### 应用领域
- **偏微分方程**: 弱解、基本解、格林函数
- **信号处理**: 脉冲响应、系统函数
- **量子场论**: 传播子、重整化
- **概率论**: 随机过程的广义函数描述

### 历史发展
- **前驱**: Dirac (1920s) δ函数的物理应用
- **创立者**: Laurent Schwartz (1945-1950)，系统理论
- **关键发展**:
  - 1936：Sobolev弱导数概念
  - 1945：Schwartz分布理论
  - 1950年代：Hörmander、Malgrange PDE理论
  - 1970年代：微局部分析
- **现代研究**: Colombeau代数（非线性分布乘法）
