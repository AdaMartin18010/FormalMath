# 级数

## 1. 级数的基本概念

**定义 1.1（级数）**. 对序列 $\{a_n\}$，其形式无穷和
$$\sum_{n=1}^\infty a_n = a_1 + a_2 + a_3 + \cdots$$
称为级数。部分和 $S_N = \sum_{n=1}^N a_n$。若 $\lim_{N \to \infty} S_N = S$ 存在，则称级数**收敛**，和为 $S$；否则**发散**。

**定理 1.2（Cauchy 准则）**. 级数 $\sum a_n$ 收敛当且仅当
$$\forall \varepsilon > 0, \exists N: m > n > N \Rightarrow \left|\sum_{k=n+1}^m a_k\right| < \varepsilon.$$

**推论 1.3（必要条件）**. 若 $\sum a_n$ 收敛，则 $\lim_{n \to \infty} a_n = 0$。逆否命题是判断发散的常用工具。

## 2. 正项级数判别法

设 $a_n \geq 0$。则部分和单调递增，收敛当且仅当有上界。

**定理 2.1（比较判别法）**. 若 $0 \leq a_n \leq b_n$ 且 $\sum b_n$ 收敛，则 $\sum a_n$ 收敛。若 $\sum a_n$ 发散，则 $\sum b_n$ 发散。

**定理 2.2（比值判别法）**. 设 $\lim_{n \to \infty} |a_{n+1}/a_n| = L$：
- $L < 1$：绝对收敛；
- $L > 1$：发散；
- $L = 1$：不确定。

*证明（$L < 1$）*. 取 $r$ 使 $L < r < 1$。存在 $N$ 使 $n > N$ 时 $|a_{n+1}/a_n| < r$。则 $|a_{N+k}| < r^k |a_N|$，与几何级数比较。$\square$

**定理 2.3（根值判别法）**. 设 $\limsup_{n \to \infty} \sqrt[n]{|a_n|} = L$，结论同比值判别法。

**定理 2.4（积分判别法）**. 设 $f$ 在 $[1, \infty)$ 上非负递减。则 $\sum_{n=1}^\infty f(n)$ 与 $\int_1^\infty f(x)\,dx$ 同敛散。

*证明*. 由 $f(n+1) \leq \int_n^{n+1} f(x)\,dx \leq f(n)$，求和即得。$\square$

**例子 2.5**. $p$-级数 $\sum_{n=1}^\infty \frac{1}{n^p}$：$\int_1^\infty \frac{dx}{x^p}$ 在 $p > 1$ 时收敛，$p \leq 1$ 时发散。

## 3. 任意项级数

**定义 3.1（绝对收敛）**. $\sum a_n$ **绝对收敛**，若 $\sum |a_n|$ 收敛。

**定理 3.2**. 绝对收敛蕴含收敛。

**定义 3.3（条件收敛）**. $\sum a_n$ 收敛但不绝对收敛，称为条件收敛。

**定理 3.4（交错级数判别法，Leibniz）**. 若 $a_n > 0$ 单调递减趋于 0，则 $\sum (-1)^n a_n$ 收敛。

*证明*. 部分和 $S_{2n} = (a_1 - a_2) + \cdots + (a_{2n-1} - a_{2n})$ 单调递增有上界 $a_1$。$S_{2n+1} = S_{2n} + a_{2n+1} \to \lim S_{2n}$。$\square$

**定理 3.5（Riemann 重排定理）**. 条件收敛级数可通过重排收敛到任意实数，或发散到 $\pm\infty$。

## 4. 幂级数

**定义 4.1（幂级数）**. $\sum_{n=0}^\infty c_n (x-a)^n$。

**定理 4.2（收敛半径）**. 存在 $R \in [0, \infty]$（收敛半径）使级数在 $|x-a| < R$ 绝对收敛，在 $|x-a| > R$ 发散。且
$$\frac{1}{R} = \limsup_{n \to \infty} \sqrt[n]{|c_n|} = \lim_{n \to \infty} \left|\frac{c_{n+1}}{c_n}\right|$$
（若后者存在）。

**定理 4.3**. 幂级数在收敛开区间 $(a-R, a+R)$ 内可逐项求导和逐项积分，且收敛半径不变。

## 5. Fourier 级数

**定义 5.1（Fourier 级数）**. 对周期 $2\pi$ 的可积函数 $f$，
$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^\infty (a_n \cos nx + b_n \sin nx),$$
其中
$$a_n = \frac{1}{\pi}\int_{-\pi}^\pi f(x)\cos nx\,dx, \quad b_n = \frac{1}{\pi}\int_{-\pi}^\pi f(x)\sin nx\,dx.$$

**定理 5.2（逐点收敛）**. 若 $f$ 分段 $C^1$，则 Fourier 级数在每点收敛到 $\frac{f(x^+) + f(x^-)}{2}$。

**定理 5.3（L² 收敛）**. Fourier 级数在 $L^2$ 意义下收敛：
$$\lim_{N \to \infty} \int_{-\pi}^\pi \left|f(x) - S_N(f)(x)\right|^2 dx = 0.$$
