# 极限

## 1. 序列极限

**定义 1.1**. 序列 $\{a_n\}$ 收敛于 $L$，记 $\lim_{n \to \infty} a_n = L$，若
$$\forall \varepsilon > 0, \exists N \in \mathbb{N}: n > N \Rightarrow |a_n - L| < \varepsilon.$$

**定理 1.2（极限的唯一性）**. 收敛序列的极限唯一。

*证明*. 设 $a_n \to L$ 且 $a_n \to M$。对任意 $\varepsilon > 0$，存在 $N_1, N_2$ 使 $n > N_1 \Rightarrow |a_n - L| < \varepsilon/2$，$n > N_2 \Rightarrow |a_n - M| < \varepsilon/2$。对 $n > \max(N_1, N_2)$，
$$|L - M| \leq |L - a_n| + |a_n - M| < \varepsilon.$$
故 $L = M$。$\square$

**定理 1.3（单调收敛）**. 单调有界序列必收敛。

*证明*. 设 $\{a_n\}$ 递增有上界。令 $L = \sup\{a_n\}$。对 $\varepsilon > 0$，存在 $N$ 使 $a_N > L - \varepsilon$。由单调性，$n > N$ 时 $L - \varepsilon < a_N \leq a_n \leq L$，故 $|a_n - L| < \varepsilon$。$\square$

## 2. 函数极限

**定义 2.1**. $\lim_{x \to a} f(x) = L$ 若
$$\forall \varepsilon > 0, \exists \delta > 0: 0 < |x - a| < \delta \Rightarrow |f(x) - L| < \varepsilon.$$

**定理 2.2（Heine）**. $\lim_{x \to a} f(x) = L$ 当且仅当对任意序列 $x_n \to a$（$x_n \neq a$），有 $f(x_n) \to L$。

**定理 2.3（夹逼定理）**. 若 $g(x) \leq f(x) \leq h(x)$ 且 $\lim_{x \to a} g(x) = \lim_{x \to a} h(x) = L$，则 $\lim_{x \to a} f(x) = L$。

## 3. 上极限与下极限

**定义 3.1**.
$$\limsup_{n \to \infty} a_n = \lim_{n \to \infty} \sup_{k \geq n} a_k, \quad \liminf_{n \to \infty} a_n = \lim_{n \to \infty} \inf_{k \geq n} a_k.$$

**定理 3.2**.
- $\limsup a_n$ 是 $\{a_n\}$ 所有收敛子序列极限的最大值；
- $\liminf a_n$ 是最小值；
- $\lim a_n$ 存在当且仅当 $\limsup a_n = \liminf a_n$。

## 4. Cauchy 准则

**定义 4.1（Cauchy 序列）**. $\{a_n\}$ 为 Cauchy 序列，若
$$\forall \varepsilon > 0, \exists N: m, n > N \Rightarrow |a_m - a_n| < \varepsilon.$$

**定理 4.2**. 在 $\mathbb{R}$ 中，序列收敛当且仅当它是 Cauchy 列。

*证明*. $(\Rightarrow)$ 由 $|a_m - a_n| \leq |a_m - L| + |L - a_n|$。$(\Leftarrow)$ 先证 Cauchy 列有界，再用 Bolzano-Weierstrass 定理取收敛子列，最后证整个序列收敛到同一极限。$\square$

**定义 4.3（完备性）**. 度量空间称为完备的，若所有 Cauchy 列都收敛。$\mathbb{R}$ 完备；$\mathbb{Q}$ 不完备。

## 5. 无穷极限与渐近分析

**定义 5.1**.
- $\lim_{x \to \infty} f(x) = L$：$\forall \varepsilon > 0, \exists M: x > M \Rightarrow |f(x) - L| < \varepsilon$；
- $\lim_{x \to a} f(x) = \infty$：$\forall M > 0, \exists \delta > 0: 0 < |x - a| < \delta \Rightarrow f(x) > M$。

**定义 5.2（Landau 记号）**.
- $f = O(g)$：$|f(x)| \leq C|g(x)|$（ eventually）；
- $f = o(g)$：$\lim f(x)/g(x) = 0$；
- $f \sim g$：$\lim f(x)/g(x) = 1$。

## 6. 拓扑极限

**定义 6.1（网与滤子）**. 在拓扑空间 $X$ 中：
- **网**：从有向集到 $X$ 的映射，推广了序列；
- **滤子**：$X$ 的子集族，对有限交封闭，向上封闭，不含空集。

**定理 6.2**. 拓扑空间 $X$ 中 $x$ 是 $S$ 的极限点当且仅当存在 $S \setminus \{x\}$ 中的网收敛到 $x$。
