# 变分法 (Calculus of Variations)

## 中心概念精确定义

**变分法**是研究泛函极值的数学分支，是经典力学、几何学和最优控制理论的基础。

> **泛函**：从函数空间到实数的映射 $J: \mathcal{F} \to \mathbb{R}$，通常表示为积分形式：
> $$J[y] = \int_a^b F(x, y(x), y'(x))\,dx$$

> **Euler-Lagrange方程**：泛函 $J[y]$ 在 $y$ 处取极值的必要条件：
> $$\frac{\partial F}{\partial y} - \frac{d}{dx}\frac{\partial F}{\partial y'} = 0$$

## 核心要素详解

### 1. Euler-Lagrange 方程推导

**问题**：求 $y(x)$ 使 $J[y] = \int_a^b F(x, y, y')\,dx$ 取极值，边界条件 $y(a) = A$，$y(b) = B$。

**推导步骤**：

1. **变分**：考虑 $y + \varepsilon \eta$，其中 $\eta(a) = \eta(b) = 0$

2. **一阶变分**：
$$\delta J = \left.\frac{d}{d\varepsilon}J[y + \varepsilon\eta]\right|_{\varepsilon=0} = \int_a^b \left(\frac{\partial F}{\partial y}\eta + \frac{\partial F}{\partial y'}\eta'\right)dx = 0$$

3. **分部积分**：
$$\int_a^b \frac{\partial F}{\partial y'}\eta'\,dx = \left[\frac{\partial F}{\partial y'}\eta\right]_a^b - \int_a^b \frac{d}{dx}\frac{\partial F}{\partial y'}\eta\,dx = -\int_a^b \frac{d}{dx}\frac{\partial F}{\partial y'}\eta\,dx$$

4. **基本引理**：由 $\int_a^b \left(\frac{\partial F}{\partial y} - \frac{d}{dx}\frac{\partial F}{\partial y'}\right)\eta\,dx = 0$ 对所有 $\eta$ 成立，得：
$$\frac{\partial F}{\partial y} - \frac{d}{dx}\frac{\partial F}{\partial y'} = 0$$

### 2. Euler-Lagrange 方程的特殊形式

**情况1：$F$ 不显含 $y$**

方程化为：$\frac{d}{dx}\frac{\partial F}{\partial y'} = 0$ $\Rightarrow$ $\frac{\partial F}{\partial y'} = C$（首次积分）

**情况2：$F$ 不显含 $x$**

存在守恒量：$F - y'\frac{\partial F}{\partial y'} = C$

**情况3：$F$ 不显含 $y'$**

方程化为：$\frac{\partial F}{\partial y} = 0$（代数方程）

### 3. 经典变分问题

**最速降线问题 (Brachistochrone)**：

质点从 $(0,0)$ 沿曲线滑到 $(a,b)$，求时间最短的曲线。

**解**：最小化 $T[y] = \int_0^a \sqrt{\frac{1 + (y')^2}{2gy}}\,dx$

Euler-Lagrange方程的解是**摆线（旋轮线）**：
$$x = r(\theta - \sin\theta), \quad y = r(1 - \cos\theta)$$

**测地线问题**：

曲面上两点间的最短路径。

**极小曲面问题 (Plateau)**：

给定边界曲线，求张在其上面积最小的曲面。

Euler-Lagrange方程导出**极小曲面方程**：
$$(1 + f_y^2)f_{xx} - 2f_x f_y f_{xy} + (1 + f_x^2)f_{yy} = 0$$

**等周问题**：

给定周长求面积最大的图形（或反之）。

**解**：圆（由变分法可严格证明）。

### 4. 带约束的变分问题

**等周约束**：在 $\int_a^b G(x, y, y')\,dx = L$ 约束下求 $J[y]$ 的极值。

**解法**：引入Lagrange乘子 $\lambda$，构造：
$$\mathcal{L}[y] = J[y] - \lambda\left(\int_a^b G\,dx - L\right)$$

对新泛函求Euler-Lagrange方程：
$$\frac{\partial(F - \lambda G)}{\partial y} - \frac{d}{dx}\frac{\partial(F - \lambda G)}{\partial y'} = 0$$

### 5. 最优控制与Pontryagin原理

**问题**：控制系统 $\dot{\mathbf{x}} = \mathbf{f}(\mathbf{x}, \mathbf{u}, t)$，求控制 $\mathbf{u}(t)$ 使代价泛函最小：
$$J = \int_{t_0}^{t_f} L(\mathbf{x}, \mathbf{u}, t)\,dt + \Phi(\mathbf{x}(t_f))$$

**Pontryagin极大值原理**：定义Hamiltonian：
$$H(\mathbf{x}, \mathbf{u}, \mathbf{p}, t) = \mathbf{p} \cdot \mathbf{f}(\mathbf{x}, \mathbf{u}, t) - L(\mathbf{x}, \mathbf{u}, t)$$

最优控制满足：
- 状态方程：$\dot{\mathbf{x}} = \frac{\partial H}{\partial \mathbf{p}}$
- 协态方程：$\dot{\mathbf{p}} = -\frac{\partial H}{\partial \mathbf{x}}$
- 极值条件：$H(\mathbf{x}^*, \mathbf{u}^*, \mathbf{p}^*, t) = \max_{\mathbf{u}} H(\mathbf{x}^*, \mathbf{u}, \mathbf{p}^*, t)$

## 关键性质与定理

### 定理1：Noether 定理

**定理**：若Lagrange函数 $L$ 在某个单参数变换群下不变，则存在相应的守恒量。

**常见对应**：

| 对称性 | 守恒量 |
|--------|--------|
| 时间平移 $t \to t + \varepsilon$ | 能量 $E = \sum \dot{q}_i \frac{\partial L}{\partial \dot{q}_i} - L$ |
| 空间平移 $q_i \to q_i + \varepsilon$ | 动量 $p_i = \frac{\partial L}{\partial \dot{q}_i}$ |
| 旋转 | 角动量 |

### 定理2：Legendre 条件

**定理**：若 $y$ 使 $J[y]$ 取极小值，则：
$$\frac{\partial^2 F}{\partial (y')^2} \geq 0$$

这是极值的二阶必要条件。

### 定理3：Jacobi 条件

**定理**：设 $y$ 满足Euler-Lagrange方程和Legendre条件，且Jacobi方程：
$$-\frac{d}{dx}\left(\frac{\partial^2 F}{\partial (y')^2}\dot{\eta}\right) + \left(\frac{\partial^2 F}{\partial y^2} - \frac{d}{dx}\frac{\partial^2 F}{\partial y \partial y'}\right)\eta = 0$$

在 $(a,b]$ 上无非零解满足 $\eta(a) = \eta(c) = 0$，则 $y$ 给出严格极小值。

### 定理4：Hamilton-Jacobi 方程

**定理**：若 $S(q, t)$ 满足：
$$\frac{\partial S}{\partial t} + H\left(q, \frac{\partial S}{\partial q}, t\right) = 0$$

则通过 $\frac{\partial S}{\partial q} = p$ 可得到经典轨道的解。

## 典型例子

### 例子1：最短路径问题

**问题**：平面上两点间最短路径。

**解**：最小化 $J[y] = \int_a^b \sqrt{1 + (y')^2}\,dx$

$F$ 不显含 $x$ 和 $y$，得 $F - y'F_{y'} = C$，即：
$$\sqrt{1+(y')^2} - \frac{(y')^2}{\sqrt{1+(y')^2}} = C \Rightarrow (y')^2 = \text{const}$$

故 $y = mx + b$（直线）。

### 例子2：悬链线问题

**问题**：均匀链条在重力作用下悬挂的形状。

**解**：最小化势能 $J[y] = \int_a^b y\sqrt{1+(y')^2}\,dx$（约束：固定长度）

解是**悬链线**：$y = a \cosh\frac{x-b}{a}$

### 例子3：几何光学中的Fermat原理

**原理**：光沿耗时最短的路径传播。

**推导Snell定律**：
$$\frac{\sin\theta_1}{v_1} = \frac{\sin\theta_2}{v_2}$$

其中 $v_i = c/n_i$ 是介质中的光速。

## 总结

变分法是连接分析、几何和物理的桥梁，通过Euler-Lagrange方程将泛函极值问题转化为微分方程。从经典的最速降线、等周问题到现代的最优控制理论，变分法展示了数学的统一力量。Noether定理揭示了守恒律与对称性的深刻联系，是理论物理的基石。理解变分法的核心思想，是掌握现代数学物理的关键一步。
