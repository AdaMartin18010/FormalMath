# 幂级数 (Power Series)

## 中心概念精确定义

**幂级数**是以某点为中心的无穷多项式，是解析函数的核心表示工具。

> **定义**：形如 $\displaystyle\sum_{n=0}^{\infty} a_n (x - c)^n$ 的函数项级数称为以 $c$ 为中心的**幂级数**，其中 $a_n \in \mathbb{R}$（或 $\mathbb{C}$）为系数，$c$ 为中心点。

当中心 $c = 0$ 时，简写为 $\sum_{n=0}^{\infty} a_n x^n$。

## 核心要素详解

### 1. 收敛半径 (Radius of Convergence)

**Cauchy-Hadamard 定理**：幂级数 $\sum a_n (x-c)^n$ 的收敛半径：
$$R = \frac{1}{\displaystyle\limsup_{n \to \infty} \sqrt[n]{|a_n|}}$$

**收敛行为**：
- $|x - c| < R$：绝对收敛
- $|x - c| > R$：发散
- $|x - c| = R$：可能收敛也可能发散（需单独判别）

**实用计算方法**：
- **比值法**：$R = \displaystyle\lim_{n \to \infty} \left|\frac{a_n}{a_{n+1}}\right|$（若极限存在）
- **根值法**：$R = \displaystyle\frac{1}{\limsup |a_n|^{1/n}}$

**例子**：$\sum \frac{x^n}{n!}$，$R = \lim \frac{(n+1)!}{n!} = \lim (n+1) = \infty$

### 2. Taylor 级数

**定义**：设 $f$ 在 $c$ 点无限可微，则：
$$f(x) \sim \sum_{n=0}^{\infty} \frac{f^{(n)}(c)}{n!}(x-c)^n$$

**收敛到原函数的条件**：
$$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(c)}{n!}(x-c)^n \Leftrightarrow \lim_{n \to \infty} R_n(x) = 0$$

**余项形式**：
- **Lagrange余项**：$R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!}(x-c)^{n+1}$
- **Cauchy余项**：$R_n(x) = \frac{f^{(n+1)}(\xi)}{n!}(x-\xi)^n(x-c)$
- **积分余项**：$R_n(x) = \frac{1}{n!}\int_c^x f^{(n+1)}(t)(x-t)^n dt$

### 3. 解析函数 (Analytic Functions)

**定义**：函数 $f$ 在开集 $U$ 上称为**解析的**，如果对每个 $c \in U$，存在邻域使 $f$ 可表示为收敛的幂级数。

**等价刻画**（实情形）：
- $f \in C^\infty(U)$
- Taylor级数在 $c$ 的某邻域收敛到 $f$

**等价刻画**（复情形）：
- $f$ 全纯（holomorphic）
- 满足 Cauchy-Riemann 方程

**光滑但非解析的例子**：
$$f(x) = \begin{cases} e^{-1/x^2} & x \neq 0 \\ 0 & x = 0 \end{cases}$$

在 $x = 0$ 处所有导数为0，Taylor级数为0，但 $f(x) \neq 0$（$x \neq 0$）。

### 4. 逐项运算

**逐项微分定理**：若 $\sum a_n (x-c)^n$ 收敛半径为 $R$，则：
$$\frac{d}{dx}\sum_{n=0}^{\infty} a_n (x-c)^n = \sum_{n=1}^{\infty} n a_n (x-c)^{n-1}$$

且微分后级数的收敛半径仍为 $R$。

**逐项积分定理**：
$$\int \sum_{n=0}^{\infty} a_n (x-c)^n dx = \sum_{n=0}^{\infty} \frac{a_n}{n+1}(x-c)^{n+1} + C$$

收敛半径仍为 $R$。

**意义**：幂级数在其收敛区间内可以像多项式一样自由微分和积分。

### 5. 重要幂级数展开

| 函数 | 展开式 | 收敛域 |
|------|--------|--------|
| $\frac{1}{1-x}$ | $\sum_{n=0}^{\infty} x^n$ | $|x| < 1$ |
| $e^x$ | $\sum_{n=0}^{\infty} \frac{x^n}{n!}$ | $x \in \mathbb{R}$ |
| $\sin x$ | $\sum_{n=0}^{\infty} \frac{(-1)^n}{(2n+1)!}x^{2n+1}$ | $x \in \mathbb{R}$ |
| $\cos x$ | $\sum_{n=0}^{\infty} \frac{(-1)^n}{(2n)!}x^{2n}$ | $x \in \mathbb{R}$ |
| $\ln(1+x)$ | $\sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n}x^n$ | $-1 < x \leq 1$ |
| $(1+x)^\alpha$ | $\sum_{n=0}^{\infty} \binom{\alpha}{n} x^n$ | $|x| < 1$ |
| $\arctan x$ | $\sum_{n=0}^{\infty} \frac{(-1)^n}{2n+1}x^{2n+1}$ | $|x| \leq 1$ |

其中 $\binom{\alpha}{n} = \frac{\alpha(\alpha-1)\cdots(\alpha-n+1)}{n!}$ 为广义二项式系数。

## 关键性质与定理

### 定理1：Abel 定理

**第一定理**：若 $\sum a_n$ 收敛，则 $\sum a_n x^n$ 在 $[0,1]$ 上一致收敛。

**第二定理（Abel连续性）**：若 $\sum a_n$ 收敛于 $S$，则：
$$\lim_{x \to 1^-} \sum_{n=0}^{\infty} a_n x^n = S$$

**应用**：利用Abel定理可以求某些数值级数的和，如：
$$\ln 2 = \sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n} = \lim_{x \to 1^-} \sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n}x^n = \lim_{x \to 1^-} \ln(1+x)$$

### 定理2：幂级数的唯一性

**定理**：若 $\sum a_n (x-c)^n = \sum b_n (x-c)^n$ 在 $c$ 的某邻域成立，则 $a_n = b_n$ 对所有 $n$ 成立。

**推论**：解析函数的Taylor展开唯一，系数 $a_n = \frac{f^{(n)}(c)}{n!}$。

### 定理3：解析延拓

**定理**：设 $f$ 在区域 $D$ 上解析，$E \subset D$ 有聚点在 $D$ 内，若 $f|_E = 0$，则 $f \equiv 0$ 在 $D$ 上。

**意义**：解析函数由其在任意小区域上的值唯一确定（刚性）。

### 定理4：Cauchy 估计

**定理**：若 $f$ 在 $|z - c| \leq R$ 上解析，$|f(z)| \leq M$，则：
$$|f^{(n)}(c)| \leq \frac{n! M}{R^n}$$

**推论**：Liouville 定理（有界整函数必为常数）。

## 典型例子

### 例子1：计算收敛半径

**问题**：求 $\sum_{n=0}^{\infty} \frac{n!}{n^n} x^n$ 的收敛半径。

**解**：用比值法：
$$R = \lim_{n \to \infty} \left|\frac{a_n}{a_{n+1}}\right| = \lim_{n \to \infty} \frac{n!}{n^n} \cdot \frac{(n+1)^{n+1}}{(n+1)!} = \lim_{n \to \infty} \left(\frac{n+1}{n}\right)^n = e$$

### 例子2：Taylor展开的应用

**问题**：求 $\arcsin x$ 的Maclaurin展开。

**解**：由 $(1-x^2)^{-1/2} = \sum_{n=0}^{\infty} \binom{-1/2}{n}(-x^2)^n = \sum_{n=0}^{\infty} \frac{(2n)!}{4^n(n!)^2} x^{2n}$

逐项积分：
$$\arcsin x = \int_0^x \frac{dt}{\sqrt{1-t^2}} = \sum_{n=0}^{\infty} \frac{(2n)!}{4^n(n!)^2(2n+1)} x^{2n+1}$$

### 例子3：利用幂级数求和

**问题**：计算 $\sum_{n=1}^{\infty} \frac{n}{2^n}$。

**解**：考虑 $f(x) = \sum_{n=1}^{\infty} n x^n = x \frac{d}{dx}\sum_{n=0}^{\infty} x^n = x \frac{d}{dx}\left(\frac{1}{1-x}\right) = \frac{x}{(1-x)^2}$

代入 $x = \frac{1}{2}$：$\sum_{n=1}^{\infty} \frac{n}{2^n} = \frac{1/2}{(1/2)^2} = 2$

## 总结

幂级数是连接代数与分析的强大工具，它将复杂的函数表示为简单的多项式无穷和。收敛半径决定了幂级数的有效区域，而Taylor级数提供了在一点附近展开函数的通用方法。解析函数的刚性性质（唯一延拓）使得幂级数在复分析和物理应用中具有不可替代的价值。
