# 导数

## 1. 导数的定义

**定义 1.1（导数）**. 函数 $f$ 在 $a$ 点可导，若极限
$$f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h}$$
存在。等价地，
$$f'(a) = \lim_{x \to a} \frac{f(x) - f(a)}{x - a}.$$

**几何意义**：$f'(a)$ 是曲线 $y = f(x)$ 在点 $(a, f(a))$ 处切线的斜率。

**定义 1.2（高阶导数）**. 递归定义 $f^{(n)} = (f^{(n-1)})'$，$f^{(0)} = f$。

## 2. 求导法则

**定理 2.1（基本法则）**. 设 $f, g$ 可导：
- **线性**：$(af + bg)' = af' + bg'$；
- **乘积法则**：$(fg)' = f'g + fg'$；
- **商法则**：$(f/g)' = (f'g - fg')/g^2$（$g \neq 0$）；
- **链式法则**：$(f \circ g)'(x) = f'(g(x)) \cdot g'(x)$。

*证明（乘积法则）*.
$$\frac{f(x+h)g(x+h) - f(x)g(x)}{h} = \frac{f(x+h)-f(x)}{h}g(x+h) + f(x)\frac{g(x+h)-g(x)}{h} \to f'(x)g(x) + f(x)g'(x).$$
$\square$

**定理 2.2（反函数求导）**. 若 $f$ 连续严格单调，$f^{-1}$ 可导且
$$(f^{-1})'(y) = \frac{1}{f'(f^{-1}(y))}.$$

## 3. 重要定理

**定理 3.1（Fermat）**. 若 $f$ 在 $a$ 处取极值且可导，则 $f'(a) = 0$。

**定理 3.2（Rolle）**. $f$ 在 $[a,b]$ 连续，$(a,b)$ 可导，$f(a) = f(b)$，则存在 $c \in (a,b)$ 使 $f'(c) = 0$。

**定理 3.3（Lagrange 中值定理）**. 在同样条件下，存在 $c \in (a,b)$ 使
$$f'(c) = \frac{f(b) - f(a)}{b - a}.$$

*证明*. 令 $g(x) = f(x) - \frac{f(b)-f(a)}{b-a}(x-a)$，则 $g(a) = g(b) = f(a)$。由 Rolle 定理，$g'(c) = 0$，即 $f'(c) = \frac{f(b)-f(a)}{b-a}$。$\square$

**定理 3.4（Cauchy 中值定理）**. $f, g$ 连续，可导，$g' \neq 0$，则存在 $c$ 使
$$\frac{f'(c)}{g'(c)} = \frac{f(b) - f(a)}{g(b) - g(a)}.$$

**推论 3.5（L'Hôpital 法则）**. 若 $\lim_{x \to a} f(x) = \lim_{x \to a} g(x) = 0$ 且 $\lim_{x \to a} f'(x)/g'(x) = L$，则 $\lim_{x \to a} f(x)/g(x) = L$。

## 4. Taylor 定理

**定理 4.1（Taylor）**. 若 $f$ 在 $a$ 的邻域有 $n+1$ 阶导数，则
$$f(x) = \sum_{k=0}^n \frac{f^{(k)}(a)}{k!}(x-a)^k + R_n(x),$$
其中 Lagrange 余项
$$R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!}(x-a)^{n+1}$$
对某 $\xi$ 在 $a$ 与 $x$ 之间。

*证明*. 对 $g(t) = f(x) - \sum_{k=0}^n \frac{f^{(k)}(t)}{k!}(x-t)^k$ 应用中值定理或 Rolle 定理。$\square$

## 5. 多元函数的导数

**定义 5.1（偏导数）**.
$$\frac{\partial f}{\partial x_i}(a) = \lim_{h \to 0} \frac{f(a_1, \dots, a_i+h, \dots, a_n) - f(a)}{h}.$$

**定义 5.2（可微性）**. $f: \mathbb{R}^n \to \mathbb{R}^m$ 在 $a$ 可微，若存在线性映射 $Df(a): \mathbb{R}^n \to \mathbb{R}^m$ 使
$$\lim_{h \to 0} \frac{|f(a+h) - f(a) - Df(a)h|}{|h|} = 0.$$

**定理 5.3（Jacobian 矩阵）**. $Df(a)$ 的矩阵表示为
$$(Df(a))_{ij} = \frac{\partial f_i}{\partial x_j}(a).$$

**定理 5.4（逆函数定理）**. 若 $Df(a)$ 可逆，则 $f$ 在 $a$ 的某邻域上为局部微分同胚。

**定理 5.5（隐函数定理）**. 设 $F: \mathbb{R}^{n+m} \to \mathbb{R}^m$ 连续可微，$F(a,b) = 0$，且 $\partial F/\partial y(a,b)$ 可逆。则存在局部定义的隐函数 $y = g(x)$ 使 $F(x, g(x)) = 0$。
