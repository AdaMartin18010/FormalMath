# 连续统假设

## 1. 基数理论

**定义 1.1（基数）**. 集合 $A$ 的基数 $|A|$ 是在等势关系下的等价类。有限基数即自然数。第一个无穷基数 $\aleph_0 = |\mathbb{N}|$。

**定理 1.2（Cantor）**. $|\mathbb{N}| < |\mathbb{R}|$。实际上，$|\mathbb{R}| = |\mathcal{P}(\mathbb{N})| = 2^{\aleph_0}$。

*证明*. 对角线论证证明 $[0,1]$ 不可数。二进制展开给出 $|\mathcal{P}(\mathbb{N})| = 2^{\aleph_0}$（子集对应特征序列），且 $|\mathbb{R}| = |\mathcal{P}(\mathbb{N})|$。$\square$

**定义 1.3（连续统）**. $2^{\aleph_0} = |\mathbb{R}|$ 称为连续统的基数，也记作 $\mathfrak{c}$。

## 2. 连续统假设

**定义 2.1（CH）**. **连续统假设**断言：不存在基数 $\kappa$ 使得 $\aleph_0 < \kappa < 2^{\aleph_0}$。等价地，$2^{\aleph_0} = \aleph_1$。

**广义连续统假设（GCH）**. 对所有无穷基数 $\kappa$，$2^\kappa = \kappa^+$（$\kappa^+$ 为 $\kappa$ 的后继基数）。

## 3. 独立性定理

**定理 3.1（Gödel，1940）**. 若 ZFC 相容，则 ZFC + CH 相容。

*证明概要*. Gödel 构造了**可构造宇宙** $L = \bigcup_{\alpha \in \mathrm{Ord}} L_\alpha$：
- $L_0 = \varnothing$；
- $L_{\alpha+1} = \operatorname{Def}(L_\alpha)$（$L_\alpha$ 上可定义子集的集合）；
- 极限步：$L_\lambda = \bigcup_{\alpha < \lambda} L_\alpha$。

$L$ 是 ZFC 的传递内模型，且 $L \models \text{"}V = L\text{"} \wedge \text{CH}$。由绝对性，若 ZFC 有模型则 $L$ 有模型。$\square$

**定理 3.2（Cohen，1963）**. 若 ZFC 相容，则 ZFC + $\neg$CH 相容。

*证明概要*. Cohen 发明了**力迫法**（forcing）：
1. 从 ZFC 模型 $M$ 出发；
2. 定义偏序集 $\mathbb{P}$（forcing 条件）和 $M$- generic 滤子 $G \subseteq \mathbb{P}$；
3. 构造泛型扩张 $M[G]$，仍为 ZFC 模型；
4. 选取适当的 $\mathbb{P}$ 使 $M[G] \models \neg$CH（如添加 $\aleph_2$ 个 Cohen 实数）。

**推论 3.3**. CH 在 ZFC 中既不可证明也不可否证。

## 4. CH 的等价形式与推论

**命题 4.1**. 以下等价于 CH：
- $\mathbb{R}^2$ 的每个子集都是 ($F_\sigma \times G_\delta$)-可测的；
- 存在 $\mathbb{R}$ 的 Well-ordering 使其初始段均可数（Hausdorff 悖论的相关形式）；
- 笛卡尔平面上每个集合要么是可数的，要么包含一个完美子集（完美集性质）。

**定理 4.2（CH 的推论）**.
- 存在 $\mathbb{R}$ 上的病态函数：$\mathbb{R}$ 可划分为两个 Lebesgue 测度为零的集合的并（Sierpiński 定理）；
- 存在非主超滤子在 $\mathbb{N}$ 上的显式构造；
- 多个分析学和拓扑学中的存在性定理可加强为构造性定理。

## 5. 大基数与 CH

**定义 5.1（可测基数）**. 不可数基数 $\kappa$ 称为可测的，若存在 $\kappa$-完备的非主超滤子在 $\kappa$ 上。

**定理 5.2（Scott）**. 若存在可测基数，则 $V \neq L$。

大基数公理提供了超越 ZFC 的框架。一些学者（如 Woodin）认为适当的大基数公理可能决定 CH 的真假。Woodin 证明：若存在一类适当的 Woodin 基数且某内模型假设成立，则 $2^{\aleph_0} = \aleph_2$（即 $\neg$CH 的某种形式）在某种意义下是"正确"的。
