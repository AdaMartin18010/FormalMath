# 黎曼度量 (Riemannian Metric)

## 中心概念精确定义

**黎曼度量**是流形上光滑、对称、正定的$(0,2)$型张量场，赋予流形内蕴的几何结构。对光滑流形$M$，**黎曼度量**$g$是满足以下条件的张量：

$$g: \mathfrak{X}(M) \times \mathfrak{X}(M) \to C^\infty(M)$$

1. **$C^\infty$-双线性**：$g(fX, Y) = fg(X, Y)$
2. **对称性**：$g(X, Y) = g(Y, X)$
3. **正定性**：$g(X, X) \geq 0$，等号当且仅当$X = 0$

**局部表达式**：在坐标$(x^i)$下，$g = g_{ij}dx^i \otimes dx^j$，其中$g_{ij} = g(\partial_i, \partial_j)$是对称正定矩阵值函数。

**几何意义**：每点$T_pM$成为内积空间，长度和角度得以定义，流形获得"弯曲空间"的内蕴结构。

## 核心要素

### 1. 度量张量

**度量分量**：$g_{ij} = \langle \partial_i, \partial_j \rangle_g$

**逆度量**：$g^{ij} = (g^{-1})_{ij}$，满足$g^{ik}g_{kj} = \delta^i_j$

**指标升降**：
- 降指标：$v_i = g_{ij}v^j$
- 升指标：$v^i = g^{ij}v_j$

**线元**：$ds^2 = g_{ij}dx^i dx^j$

### 2. 距离结构

**曲线长度**：对光滑曲线$\gamma: [a, b] \to M$：
$$L(\gamma) = \int_a^b \sqrt{g_{\gamma(t)}(\dot{\gamma}, \dot{\gamma})} \, dt$$

**距离函数**：
$$d(p, q) = \inf\{L(\gamma) : \gamma(a) = p, \gamma(b) = q\}$$

**度量空间**：$(M, d)$成为度量空间，其拓扑与流形拓扑一致（Hopf-Rinow定理保证完备性条件）。

### 3. 体积元

**Riemann体积形式**：对定向坐标：
$$d\text{vol}_g = \sqrt{\det g_{ij}} \, dx^1 \wedge \cdots \wedge dx^n$$

**体积计算**：
$$\text{Vol}(M) = \int_M d\text{vol}_g$$

**子流形体积**：$k$-维子流形诱导度量，体积为：
$$\text{Vol}_k(N) = \int_N \sqrt{\det(g|_{TN})} \, d^k x$$

### 4. 正交标架

**正交标架丛**：$F_O(M) = \{(p, e_1, \ldots, e_n) : \{e_i\}\text{是}T_pM\text{的标准正交基}\}$

**局部正交标架**：在每点邻域存在光滑向量场$\{e_a\}$使得$g(e_a, e_b) = \delta_{ab}$。

**对偶余标架**：$\{\theta^a\}$满足$\theta^a(e_b) = \delta^a_b$，则：
$$g = \sum_{a=1}^n \theta^a \otimes \theta^a$$

### 5. 等距映射

**等距**：微分同胚$\phi: (M, g) \to (N, h)$满足$\phi^*h = g$。

**局部等距**：浸入$f$满足$f^*h = g$。

**Killing向量场**：流产生单参数等距群，满足：
$$\mathcal{L}_X g = 0 \Leftrightarrow \nabla_i X_j + \nabla_j X_i = 0$$

### 6. 曲率与度量

**Levi-Civita联络**：由度量唯一确定的挠率自由、度量相容联络：
$$\nabla g = 0, \quad T^\nabla = 0$$

**Christoffel符号**：
$$\Gamma^k_{ij} = \frac{1}{2}g^{kl}(\partial_i g_{jl} + \partial_j g_{il} - \partial_l g_{ij})$$

## 性质与定理

### 定理1：Nash嵌入定理

任意Riemann流形$(M^n, g)$可等距嵌入到$(\mathbb{R}^N, \delta)$中，其中$N = \frac{n(n+1)(3n+11)}{2}$。

### 定理2：Hopf-Rinow定理

对Riemann流形，以下等价：
- $(M, d)$完备度量空间
- 闭有界集紧
- 测地线可无限延长
- 任两点有极小测地线连接

### 定理3：Gauss引理

指数映射$\exp_p: T_pM \to M$保持径向测地线正交于测地球面。

### 定理4：度量存在性

任意光滑流形（第二可数、Hausdorff）容许Riemann度量。

**证明**：利用单位分解将欧氏度量拼接到局部坐标上。

### 定理5：度量空间的刚性

若$(M, g)$和$(N, h)$作为度量空间等距，则存在光滑等距（Myers-Steenrod定理）。

## 典型例子

### 例子1：欧氏空间$(\mathbb{R}^n, \delta)$

$$g = \sum_{i=1}^n (dx^i)^2$$

- 曲率为零
- 测地线为直线
- 等距群为$\mathbb{R}^n \rtimes O(n)$

### 例子2：球面$(S^n, g_{\text{round}})$

$$g_{\text{round}} = \frac{4}{(1 + |x|^2)^2} \sum_{i=1}^n (dx^i)^2$$

（球极投影坐标）
- 截面曲率$K = 1$
- 测地线为大圆
- 体积$\text{Vol}(S^n) = \frac{2\pi^{(n+1)/2}}{\Gamma((n+1)/2)}$

### 例子3：双曲空间$(\mathbb{H}^n, g_{\text{hyp}})$

**Poincaré圆盘模型**：
$$g_{\text{hyp}} = \frac{4}{(1 - |x|^2)^2} \sum_{i=1}^n (dx^i)^2$$

- 截面曲率$K = -1$
- 测地线为与边界正交的圆弧
- 体积无限，面积指数增长
