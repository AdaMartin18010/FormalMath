# 曲率张量 (Curvature Tensor)

## 中心概念精确定义

**曲率张量**是Riemann几何的核心不变量，度量空间的"弯曲"程度。对Riemann流形$(M, g)$配Levi-Civita联络$\nabla$，**Riemann曲率张量**定义为：

$$R(X, Y)Z = \nabla_X \nabla_Y Z - \nabla_Y \nabla_X Z - \nabla_{[X,Y]}Z$$

**类型(1,3)张量**：$R: \mathfrak{X}(M) \times \mathfrak{X}(M) \times \mathfrak{X}(M) \to \mathfrak{X}(M)$

**局部坐标**：$R = R^l_{ijk} \partial_l \otimes dx^i \otimes dx^j \otimes dx^k$，其中：

$$R^l_{ijk} = \partial_i \Gamma^l_{jk} - \partial_j \Gamma^l_{ik} + \Gamma^l_{im}\Gamma^m_{jk} - \Gamma^l_{jm}\Gamma^m_{ik}$$

**协变形式**：$R_{ijkl} = g_{im}R^m_{jkl}$

---

## 核心要素

### 1. Riemann曲率张量的代数性质

**反对称性**：
- $R(X, Y)Z = -R(Y, X)Z$
- $R_{ijkl} = -R_{jikl} = -R_{ijlk}$

**第一Bianchi恒等式**：
$$R(X, Y)Z + R(Y, Z)X + R(Z, X)Y = 0$$

$$R_{ijkl} + R_{iklj} + R_{iljk} = 0$$

**配对对称性**：$R_{ijkl} = R_{klij}$

**代数独立分量**：$n$维流形上独立分量数为$\frac{n^2(n^2-1)}{12}$

### 2. 截面曲率

**定义**：对2-平面$\sigma = \text{span}\{X, Y\} \subset T_pM$：
$$K(\sigma) = \frac{g(R(X, Y)Y, X)}{g(X, X)g(Y, Y) - g(X, Y)^2}$$

**几何意义**：曲面在$\sigma$方向的Gauss曲率。

**常曲率空间**：$K = \text{const}$当且仅当：
$$R_{ijkl} = K(g_{ik}g_{jl} - g_{il}g_{jk})$$

### 3. Ricci曲率与标量曲率

**Ricci张量**：$R_{ij} = R^k_{ikj} = g^{kl}R_{ikjl}$

- 对称：$R_{ij} = R_{ji}$
- 几何意义：$n-1$个正交截面曲率的平均

**标量曲率**：$R = g^{ij}R_{ij}$

- 每点曲率的平均
- 广义相对论中的作用量密度

**Einstein张量**：$G_{ij} = R_{ij} - \frac{1}{2}Rg_{ij}$（无散：$\nabla^i G_{ij} = 0$）

### 4. Weyl张量

**定义**：$n \geq 4$时的共形不变曲率部分：

$$W_{ijkl} = R_{ijkl} - \frac{1}{n-2}(g_{ik}R_{jl} - g_{il}R_{jk} + g_{jl}R_{ik} - g_{jk}R_{il}) + \frac{R}{(n-1)(n-2)}(g_{ik}g_{jl} - g_{il}g_{jk})$$

**性质**：
- 迹为零（缩并为零）
- 共形不变：若$\tilde{g} = e^{2f}g$，则$\tilde{W} = e^{2f}W$
- $W = 0 \Leftrightarrow$ 局部共形平坦

### 5. Ricci流

**方程**：
$$\frac{\partial g_{ij}}{\partial t} = -2R_{ij}$$

**Hamilton-Perelman理论**：
- 2维：收敛到常曲率度量
- 3维：奇点分析导出Poincaré猜想证明
- 手术技术控制奇点形成

**归一化Ricci流**：
$$\frac{\partial g_{ij}}{\partial t} = -2R_{ij} + \frac{2r}{n}g_{ij}$$

保持体积，收敛到Einstein度量。

### 6. 曲率与拓扑

**Gauss-Bonnet定理**：
$$\int_M \text{Pf}(\Omega) = (2\pi)^n \chi(M)$$

对$2n$维定向Riemann流形。

**正曲率拓扑**：
- Synge定理：偶维正曲率定向流形单连通
- 球面定理：$1/4 < K \leq 1$的单连通流形同胚于球面

**负曲率几何**：
- Mostow刚性：高维双曲流形由基本群决定
- 测地流、遍历理论

## 性质与定理

### 定理1：Schur定理

若$\dim M \geq 3$且$K(\sigma) = f(p)$仅依赖于点，则$f = \text{const}$。

### 定理2：第二Bianchi恒等式

$$\nabla_i R_{jkla} + \nabla_j R_{kila} + \nabla_k R_{ijla} = 0$$

导出$\nabla^i G_{ij} = 0$（Einstein张量守恒）。

### 定理3：Ricci分解

$$R_{ijkl} = W_{ijkl} + \frac{1}{n-2}(g_{ik}R_{jl} - \cdots) - \frac{R}{(n-1)(n-2)}(\cdots)$$

正交分解为Weyl、无迹Ricci、标量曲率部分。

### 定理4：Cheeger-Gromoll分裂定理

若完备非紧Riemann流形有$\text{Ric} \geq 0$且包含直线，则等距分裂$M = \mathbb{R} \times N$。

### 定理5：Hamilton-Perelman定理

闭合3-流形上的Ricci流带手术分类了3-流形的拓扑，证明了Poincaré猜想和几何化猜想。

## 典型例子

### 例子1：常曲率空间形式

- $K = 0$：欧氏空间$\mathbb{R}^n$
- $K = 1$：球面$S^n$
- $K = -1$：双曲空间$\mathbb{H}^n$

统一为：$R_{ijkl} = K(g_{ik}g_{jl} - g_{il}g_{jk})$

### 例子2：乘积流形$M \times N$

$$R^{M \times N} = R^M \oplus R^N$$

混合分量为零，若$M$和$N$曲率不同则非常曲率。

### 例子3：Schwarzschild度量

广义相对论的真空球对称解：
$$ds^2 = -\left(1 - \frac{2m}{r}\right)dt^2 + \left(1 - \frac{2m}{r}\right)^{-1}dr^2 + r^2 d\Omega^2$$

- $r = 2m$：事件视界
- $r = 0$：曲率奇点
