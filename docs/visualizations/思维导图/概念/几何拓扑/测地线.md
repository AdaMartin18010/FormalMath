# 测地线 (Geodesics)

## 中心概念精确定义

**测地线**是Riemann流形上"直线"的自然推广，是局部距离极小的曲线。曲线$\gamma: I \to M$称为**测地线**，若其切向量沿曲线平行：

$$\nabla_{\dot{\gamma}} \dot{\gamma} = 0$$

**局部坐标方程**：$\gamma(t) = (x^1(t), \ldots, x^n(t))$满足**测地线方程**：
$$\ddot{x}^k + \Gamma^k_{ij}\dot{x}^i\dot{x}^j = 0$$

这是二阶常微分方程组，由存在唯一性定理，对初值$(p, v) \in TM$存在唯一测地线$\gamma_v$满足$\gamma_v(0) = p$，$\dot{\gamma}_v(0) = v$。

**等价刻画**：
- 速度恒定的临界曲线（弧长泛函的临界点）
- 局部最短路径
- 曲率为零的曲线（Frenet意义下）

## 核心要素

### 1. 指数映射

**定义**：$\exp_p: T_pM \to M$，$\exp_p(v) = \gamma_v(1)$（若$\gamma_v$定义到$t=1$）。

**性质**：
- $d(\exp_p)_0 = \text{id}_{T_pM}$（Gauss引理）
- 在$0 \in T_pM$附近是微分同胚
- $\exp_p(tv) = \gamma_v(t)$

**定义域**：**.injectivity radius* inj$(p)$是$\exp_p$为微分同胚的最大球半径。

### 2. 法坐标

**法坐标系**：选取$T_pM$的标准正交基$\{e_i\}$，通过$\exp_p$将坐标移到$M$上：

$$x^i(\exp_p(v^j e_j)) = v^i$$

**正规性质**：在$p$点：
- $g_{ij}(p) = \delta_{ij}$
- $\Gamma^k_{ij}(p) = 0$
- $\partial_k g_{ij}(p) = 0$

**测地极坐标**：在$T_pM$上用极坐标$(r, \theta)$，测地线为径向线$\theta = \text{const}$。

### 3. 变分公式

**弧长泛函**：$L(\gamma) = \int_a^b \sqrt{g(\dot{\gamma}, \dot{\gamma})} dt$

**第一变分公式**：设$\gamma_s$是变分，$V = \partial_s\gamma_s|_{s=0}$：
$$\left.\frac{d}{ds}\right|_{s=0} L(\gamma_s) = -\int_a^b g(V, \nabla_{\dot{\gamma}}\dot{\gamma}) dt + [g(V, \dot{\gamma})]_a^b$$

**结论**：固定端点的测地线是弧长的临界点。

**第二变分公式**：
$$\left.\frac{d^2}{ds^2}\right|_{s=0} L(\gamma_s) = \int_a^b \left(g(\nabla_{\dot{\gamma}}V^\perp, \nabla_{\dot{\gamma}}V^\perp) - g(R(V^\perp, \dot{\gamma})\dot{\gamma}, V^\perp)\right) dt$$

其中$V^\perp = V - \frac{g(V, \dot{\gamma})}{g(\dot{\gamma}, \dot{\gamma})}\dot{\gamma}$。

### 4. Jacobi场

**定义**：沿测地线$\gamma$的向量场$J$满足**Jacobi方程**：
$$\nabla_{\dot{\gamma}}\nabla_{\dot{\gamma}}J + R(J, \dot{\gamma})\dot{\gamma} = 0$$

**几何意义**：Jacobi场对应测地变分，$J(t) = \partial_s \gamma_s(t)|_{s=0}$。

**共轭点**：$p = \gamma(0)$和$q = \gamma(t_0)$共轭若存在非零Jacobi场在两端为零。

### 5. 完备性

**测地完备**：所有测地线可延拓到$\mathbb{R}$。

**Hopf-Rinow等价性**：
- $(M, d)$完备度量空间
- 测地完备
- 有界闭集紧
- 任两点有最短测地线连接

### 6. 距离函数与割迹

**距离函数**：$r(p) = d(p_0, p)$在$p_0$的割迹外光滑。

**割迹**$C(p_0)$：测地线不再是极短路径的点集。

**割点**：两种情形：
- 两个不同测地线到达同一点
- 测地线上$p_0$的第一个共轭点

## 性质与定理

### 定理1：Gauss引理

指数映射$\exp_p$保持径向测地线与测地球面正交。

### 定理2：测地线的局部极小性

测地线段在其定义域的充分小子区间上是弧长最短的。

### 定理3：Synge定理

偶维紧致定向正曲率Riemann流形是单连通的。

### 定理4：Bonnet-Myers直径定理

若Ricci曲率$\text{Ric} \geq (n-1)k > 0$，则$\text{diam}(M) \leq \pi/\sqrt{k}$，且$M$紧、基本群有限。

### 定理5：Rauch比较定理

比较曲率条件下的Jacobi场比较，导出拓扑-几何联系。

## 典型例子

### 例子1：欧氏空间的测地线

$\ddot{x}^i = 0$，解为$x^i(t) = a^i + tb^i$。
- 测地线是直线
- 指数映射为恒等（$\mathbb{R}^n$与切空间典范认同）
- 无割点、无共轭点

### 例子2：球面的测地线

$S^n$上测地线是大圆：
- 周期为$2\pi$（曲率归一化为1）
- 对径点是共轭点
- 割迹为对径点

### 例子3：双曲空间的测地线

**Poincaré圆盘**：测地线是正交于边界的圆弧（包括直径）。
- 无限延伸，无周期性
- 无共轭点、无割点
- 指数体积增长
