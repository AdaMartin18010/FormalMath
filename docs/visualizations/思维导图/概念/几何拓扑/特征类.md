# 特征类 (Characteristic Classes)

## 中心概念精确定义

**特征类**是向量丛和主丛的拓扑不变量，实现了从几何到拓扑的核心映射。对结构群$G$，特征类是函子性的对应：

$$\{\text{主}G\text{-丛}\} \longrightarrow H^*(M; R)$$

对每个同伦等价类$[f: M \to BG]$，特征类$\chi(f^*EG) = f^*\chi(EG) \in H^*(M)$。

**特征类的公理化**（Steenrod公理）：
1. **函子性**：$f^*\chi(E) = \chi(f^*E)$
2. **Whitney和**：$\chi(E \oplus F) = \chi(E) \cup \chi(F)$（对乘性类）
3. **归一化**：在万有丛上的取值决定一切

特征类量化了丛的"扭曲程度"，是指标定理和拓扑量子场论的核心工具。

## 核心要素

### 1. Stiefel-Whitney类

**定义**：实向量丛$E \to M$的$\mathbb{Z}_2$特征类：
$$w_i(E) \in H^i(M; \mathbb{Z}_2), \quad i = 0, 1, \ldots, \text{rank}E$$

**公理**：
- $w_0 = 1$，$w_i = 0$对$i > \text{rank}$
- **Whitney和**：$w(E \oplus F) = w(E) \cup w(F)$，其中$w = 1 + w_1 + w_2 + \cdots$
- **自然性**：$f^*w_i(E) = w_i(f^*E)$
- **非平凡性**：$w_1(\gamma^1) \neq 0$，$\gamma^1 \to \mathbb{R}P^\infty$

**应用**：
- $w_1(E) = 0 \Leftrightarrow E$可定向
- $w_2(E) = 0$（对定向丛）与spin结构相关

### 2. Chern类

**定义**：复向量丛$E \to M$的整系数特征类：
$$c_i(E) \in H^{2i}(M; \mathbb{Z}), \quad c(E) = 1 + c_1 + c_2 + \cdots$$

**全Chern类与分裂**：通过分裂原理，设$E = L_1 \oplus \cdots \oplus L_n$（线丛）：
$$c(E) = \prod_{i=1}^n (1 + x_i), \quad x_i = c_1(L_i)$$

**Chern特征**：
$$\text{ch}(E) = \sum_{i=1}^n e^{x_i} = \text{rank} + c_1 + \frac{c_1^2 - 2c_2}{2} + \cdots$$

是环同态$\text{ch}: K(M) \otimes \mathbb{Q} \to H^{\text{ev}}(M; \mathbb{Q})$。

### 3. Pontryagin类

**定义**：实向量丛$E$的$E \otimes \mathbb{C}$的Chern类导出：
$$p_i(E) = (-1)^i c_{2i}(E \otimes \mathbb{C}) \in H^{4i}(M; \mathbb{Z})$$

**分裂原理**：设$E \otimes \mathbb{C} = L_1 \oplus \bar{L}_1 \oplus \cdots$：
$$p(E) = \prod_{i=1}^{[n/2]} (1 + y_i^2), \quad y_i = c_1(L_i)$$

**全Pontryagin类**：
$$p(E) = 1 + p_1 + p_2 + \cdots$$

### 4. Euler类

**定义**：定向实秩$2k$丛的$e(E) \in H^{2k}(M; \mathbb{Z})$。

**性质**：
- $e(E \oplus F) = e(E) \cup e(F)$
- $e(E^*) = (-1)^{\text{rank}}e(E)$
- 对切丛：$\int_M e(TM) = \chi(M)$

**与Chern类关系**：$e(E) = c_k(E)$当$E$是复丛看作定向实丛。

### 5. Chern-Weil理论

**曲率表示**：对向量丛$E$配联络$\nabla$，曲率$R^\nabla \in \Omega^2(M; \text{End}E)$。

**不变多项式**：$GL(n, \mathbb{C})$-不变对称多项式$P$定义闭形式：
$$P(R^\nabla) \in \Omega^{2k}(M), \quad d P(R^\nabla) = 0$$

**特征形式**：
- **Chern形式**：$c_k = \left[\sigma_k\left(\frac{iR}{2\pi}\right)\right]$
- **Pontryagin形式**：$p_k = \left[\sigma_{2k}\left(\frac{R}{2\pi}\right)\right]$

### 6. 分裂原理

**定理**：对任意复向量丛$E \to M$，存在$f: F(E) \to M$使得$f^*E$分裂为线丛直和。

**推论**：证明关于Chern类的恒等式只需验证线丛情形。

**旗流形**：$F(E)$是E的旗流形（fiber = 完全旗流形$GL(n, \mathbb{C})/B$）。

## 性质与定理

### 定理1：Thom同构与Euler类

对定向秩$2k$丛$E$：
$$\Phi: H^i(M) \xrightarrow{\cong} H^{i+2k}(E, E_0), \quad \Phi(1) = \tau$$

Thom类$\tau$满足$i^*\tau = e(E)$，$i: M \hookrightarrow E$是零截面。

### 定理2：Hirzebruch符号差定理

对紧定向$4k$-维流形：
$$\sigma(M) = \int_M L(p_1, \ldots, p_k)$$

其中$L$是Hirzebruch$L$-亏格。

### 定理3：Atiyah-Singer指标定理

椭圆复形$E^\bullet$的解析指标：
$$\text{ind}(D) = \int_M \text{ch}(E^\bullet) \wedge \text{Td}(TM)$$

### 定理4：Grothendieck-Riemann-Roch

对proper映射$f: X \to Y$：
$$\text{ch}(f_!E) \cdot \text{Td}(Y) = f_*(\text{ch}(E) \cdot \text{Td}(X))$$

### 定理5：Wu公式

Stiefel-Whitney类满足：
$$\text{Sq}^k(w) = w_k$$

其中$\text{Sq}$是Steenrod平方。

## 典型例子

### 例子1：切丛$T\mathbb{C}P^n$的Chern类

$$c(T\mathbb{C}P^n) = (1 + H)^{n+1}$$

其中$H = c_1(\mathcal{O}(1))$是超平面类。
- $c_1 = (n+1)H$
- $\int_{\mathbb{C}P^n} H^n = 1$
- Euler示性数$\chi(\mathbb{C}P^n) = n + 1$

### 例子2：球面的稳定切丛

$$T S^n \oplus \underline{\mathbb{R}} = S^n \times \mathbb{R}^{n+1}$$

因此$p(TS^n) = 1$（稳定平凡），但$e(TS^n) = 2$（对偶球面）。

### 例子3：K3曲面的特征类

复$K3$曲面（复二维，$b_2 = 22$）：
- $c_1 = 0$（Calabi-Yau）
- $c_2 = \chi = 24$
- 符号差$\sigma = -16$
