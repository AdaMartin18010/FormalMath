# 黎曼-罗赫定理可视化

## 概念定义

设 $C$ 为亏格 $g$ 的紧致连通黎曼面（复代数曲线）。$C$ 上的**除子（divisor）**是有限形式和
$$D = \sum_{p \in C} n_p \cdot p，\quad n_p \in \mathbb{Z}, \text{ 几乎全为零}。$$
除子的次数为 $\deg(D) = \sum n_p$。

除子 $D$ 称为**有效的**，若所有 $n_p \geq 0$。对除子 $D$，定义**完全线性系**
$$|D| := \{D' \geq 0 : D' \sim D\}，$$
其中 $D' \sim D$ 表示线性等价（$D' - D = (f)$ 为某亚纯函数 $f$ 的除子）。

**亚纯函数与全纯 1-形式空间**：
- $\ell(D) := \dim H^0(C, \mathcal{O}(D)) = \dim \{f \text{ 亚纯} : (f) + D \geq 0\} \cup \{0\}$；
- $i(D) := \dim H^0(C, \mathcal{O}(K - D)) = \dim \{\omega \text{ 全纯 1-形式} : (\omega) \geq D\} \cup \{0\}$，称为 $D$ 的**特殊指数**。

## 关键公式

**黎曼-罗赫定理**：对任意除子 $D$，
$$\ell(D) - i(D) = \deg(D) + 1 - g。$$

等价形式：利用 Serre 对偶 $H^1(C, \mathcal{O}(D)) \cong H^0(C, \mathcal{O}(K-D))^*$，有
$$\chi(\mathcal{O}(D)) := h^0(\mathcal{O}(D)) - h^1(\mathcal{O}(D)) = \deg(D) + 1 - g。$$

**Riemann 不等式**：总有 $\ell(D) \geq \deg(D) + 1 - g$。当 $\deg(D) > 2g - 2$ 时 $i(D) = 0$，此时等号精确成立：$\ell(D) = \deg(D) + 1 - g$。

**典范除子**：全纯 1-形式构成的线丛 $K$（典范丛）的次数为 $\deg(K) = 2g - 2$。

**推论——亏格决定嵌入**：若 $\deg(D) \geq 2g + 1$，则 $|D|$ 给出 $C$ 到 $\mathbb{CP}^{\deg(D) - g}$ 的射影嵌入。

## 可视化说明

**除子与线性系的"珠子串"图**：将黎曼面 $C$ 画成一个带柄的闭曲面（如环面 $g=1$ 或双柄曲面 $g=2$）。

- 在 $C$ 上标注若干点 $p_1, p_2, p_3$，用不同大小的彩色珠子表示除子 $D = 2p_1 - p_2 + 3p_3$：大蓝珠在 $p_1$（系数 $+2$），小红珠在 $p_2$（系数 $-1$，表示"允许极点"），大绿珠在 $p_3$（系数 $+3$）。
- 线性系 $|D|$ 的可视化：画出所有有效除子 $D' \sim D$。对 $g=0$（$C = \mathbb{CP}^1$），$D = n \cdot \infty$，则 $|D|$ 是所有次数 $n$ 的齐次多项式（即 $\mathbb{P}^n$ 的点）。在图中画出 $\mathbb{P}^1$ 上 $n$ 个可变点的运动轨迹。
- **关键可视化**：当 $\deg(D)$ 从 $0$ 增加到 $2g-2$ 时，$i(D)$ 从 $g$ 下降到 $0$，而 $\ell(D)$ 的曲线呈"阶梯状"上升。画出 $\ell(D)$ 随 $\deg(D)$ 变化的图像：一条斜率为 $1$ 的直线 $\deg(D) + 1 - g$ 上下波动，波动的幅度由 $i(D)$ 给出。

**典范除子的"全纯形式分布"图**：

- 在亏格 $g$ 的曲面上，典范除子 $K$ 是全纯 1-形式 $\omega$ 的零点除子。全纯 1-形式无极点，故 $(\omega) \geq 0$。
- 对 $g=1$（环面），$\omega = dz$ 无零点，$K = 0$。画出环面上均匀分布的"平行线场"代表 $dz$ 的实部（或虚部）。
- 对 $g=2$（双曲曲线），任意非零全纯 1-形式 $\omega$ 恰有 $2g - 2 = 2$ 个零点（计重数）。画出曲面上两条相交的零点轨迹，附近的方向场呈螺旋收敛状。

**Riemann-Roch 的"簿记平衡"图**：

- 画一个天平。左盘放置 $\ell(D)$（"允许极点的函数空间维数"），右盘放置 $i(D)$（"在 $D$ 处有零点的全纯形式空间维数"）。
- 天平的支点标注 $\deg(D) + 1 - g$。黎曼-罗赫定理说天平总是平衡的。
- 对 $D = 0$：$\ell(0) = 1$（常函数），$i(0) = g$（所有全纯 1-形式），平衡方程 $1 - g = 0 + 1 - g$ ✓。
- 对 $D = K$：$\ell(K) = g$，$i(K) = 1$（常数倍 $K$ 本身），平衡方程 $g - 1 = (2g-2) + 1 - g = g - 1$ ✓。
- 当 $\deg(D) < 0$ 时，$\ell(D) = 0$（无全局亚纯函数有负阶极点），天平向右倾斜，$i(D) = g - 1 - \deg(D)$。

## 直观解释

黎曼-罗赫定理是代数曲线理论的中心结果，它精确量化了**"在指定位置允许极点的亚纯函数空间"与"在指定位置必须为零的全纯微分形式空间"之间的对偶关系。**

核心直觉来自两个方面：
1. **拓扑约束**：曲线 $C$ 的亏格 $g$ 决定了它的"洞"的个数。每个洞都给全纯形式提供一个自由度，也给函数的构造增加约束。
2. **解析自由度**：每增加一次极点的允许（$\deg(D)$ 增加 1），原则上函数空间应多一个自由度——但全纯形式的约束（$i(D)$）可能抵消这一增长。

当 $\deg(D)$ 足够大（$> 2g-2$）时，全纯形式的约束消失（$i(D) = 0$），函数空间的维数完全由次数和亏格决定。这类似于射影空间中"足够丰富"的线丛给出射影嵌入：足够多的自由度让你足以"分辨"曲线上的每一点。

## 例子

1. **$g = 0$（$C = \mathbb{CP}^1$）**：对 $D = n \cdot \infty$（$n \geq 0$），$\ell(D) = n + 1$（基 $1, z, z^2, \ldots, z^n$），$i(D) = 0$。Riemann-Roch 给出 $(n+1) - 0 = n + 1 - 0$。这说明 $\mathbb{CP}^1$ 可被 $1$ 次映射嵌入 $\mathbb{CP}^1$（恒等），被 $n$ 次映射为 $\mathbb{CP}^n$ 中的有理正规曲线。

2. **$g = 1$（椭圆曲线）**：设 $C = \mathbb{C}/\Lambda$。典范丛 $K$ 平凡。对 $D = p$（单点），Riemann-Roch 给出 $\ell(p) - i(p) = 1 + 1 - 1 = 1$。因 $i(p) = 0$（无非零全纯 1-形式在 $p$ 处为零），故 $\ell(p) = 1$。唯一的（计常数倍）亚纯函数在 $p$ 处有一阶极点、无其他极点，就是 Weierstrass $\wp$-函数。$\wp$ 的导数给出椭圆曲线到 $\mathbb{CP}^2$ 的嵌入 $w^2 = 4z^3 - g_2 z - g_3$。

3. **超椭圆曲线**：亏格 $g$ 的超椭圆曲线 $y^2 = P_{2g+2}(x)$ 有自然的 $2:1$ 映射到 $\mathbb{CP}^1$（遗忘 $y$ 坐标）。典范映射由全纯 1-形式 $x^k dx/y$（$k = 0, \ldots, g-1$）给出。当 $g \geq 3$ 时，超椭圆曲线的典范映射是 $2:1$ 到 $\mathbb{CP}^{g-1}$ 中的有理正规曲线，而非嵌入——这是区分超椭圆曲线与一般曲线的基本不变量。

4. **高维推广（Hirzebruch-Riemann-Roch）**：对 $n$ 维光滑射影簇 $X$ 上的向量丛 $E$，
   $$\chi(X, E) = \int_X \mathrm{ch}(E) \wedge \mathrm{td}(TX)。$$
   这是指标定理在复几何中的应用，连接了代数几何与拓扑。Atiyah-Singer 指标定理进一步将其推广到任意椭圆算子。
