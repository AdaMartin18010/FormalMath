# 黎曼-罗赫定理

## 描述

紧致黎曼曲面上除子的维数公式。

## 定理

l(D) - l(K-D) = deg(D) + 1 - g

## 可视化

```mermaid
graph TD
    A[除子 D] --> B[l(D) = dim H⁰(D)]
    B --> C[deg(D)]
    C --> D[亏格 g]
    D --> E[黎曼-罗赫公式]
end

```
