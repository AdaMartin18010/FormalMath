# 斯托克斯定理 - 流形边界积分可视化

## 描述

本可视化展示斯托克斯定理的核心思想：流形上的外微分形式的积分等于其在边界上的积分。这是微积分基本定理在高维的推广。

## 数学概念

**斯托克斯定理**: 设M是有边界∂M的n维定向紧致流形，ω是(n-1)-形式，则

$$\int_M d\omega = \int_{\partial M} \omega$$

这是微积分基本定理、格林定理、高斯定理的统一推广。

## 可视化代码

### 斯托克斯定理关系图

```mermaid
graph TB
    subgraph 斯托克斯定理统一框架
    A[斯托克斯定理<br/>∫ₘ dω = ∫∂ₘ ω] --> B[微积分基本定理<br/>∫ₐᵇ F'dx = F(b)-F(a)]
    A --> C[格林定理<br/>∬ᴅ (∂Q/∂x - ∂P/∂y)dA = ∮ᴄ Pdx+Qdy]
    A --> D[斯托克斯定理(向量)<br/>∬ₛ (∇×F)·dS = ∮ᴄ F·dr]
    A --> E[高斯定理<br/>∭ᴠ (∇·F)dV = ∯ₛ F·dS]

    F[n维流形 M] --> G[边界 ∂M<br/>(n-1)维]
    H[n-形式 dω] --> I[在M上积分]
    J[n-1形式 ω] --> K[在∂M上积分]
    end

    style A fill:#e3f2fd

    style D fill:#fce4ec

```
