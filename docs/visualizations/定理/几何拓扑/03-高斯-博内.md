# 高斯-博内定理可视化

## 概念定义

设 $M$ 为二维紧致定向黎曼流形（无边）。$M$ 的**Gauss 曲率** $K$ 是内蕴几何不变量，度量了曲面在每点的局部弯曲程度。对参数化曲面 $\mathbf{r}(u, v)$，
$$K = \frac{LN - M^2}{EG - F^2}，$$
其中 $E, F, G$ 为第一基本形式系数，$L, M, N$ 为第二基本形式系数。

**高斯-博内定理（Gauss-Bonnet Theorem）**：
$$\int_M K \, \mathrm{d}A = 2\pi \chi(M) = 2\pi (2 - 2g)，$$
其中 $\chi(M)$ 为 Euler 示性数，$g$ 为亏格。

对带边曲面，设 $\partial M$ 有测地曲率 $k_g$，则
$$\int_M K \, \mathrm{d}A + \int_{\partial M} k_g \, \mathrm{d}s = 2\pi \chi(M)。$$

## 关键公式

**Euler 示性数**：对三角剖分，$\chi = V - E + F$。对闭可定向曲面，$\chi = 2 - 2g$。

**局部 Gauss-Bonnet**：对测地三角形（边为测地线），内角和
$$\alpha + \beta + \gamma = \pi + \int_\Delta K \, \mathrm{d}A。$$
在平坦空间 $K=0$ 时退化为欧几里得几何（内角和 $= \pi$）；在球面 $K > 0$ 时内角和 $> \pi$；在双曲曲面 $K < 0$ 时内角和 $< \pi$。

**Chern 的推广**：对 $2n$ 维紧致定向黎曼流形，
$$\int_M \mathrm{Pf}(\Omega) = (2\pi)^n \chi(M)，$$
其中 $\mathrm{Pf}$ 为 Pfaffian，$\Omega$ 为曲率形式。这是 Gauss-Bonnet-Chern 定理。

## 可视化说明

**曲率分布的"地形染色"图**：

- 画出三种闭曲面，按 Gauss 曲率染色：
  - **球面 $S^2$（$g=0$）**：$K = 1/R^2 > 0$ 处处相等。用统一红色染色。$\int K \, \mathrm{d}A = 4\pi = 2\pi \cdot 2$。
  - **环面 $T^2$（$g=1$）**：标准嵌入的环面，外侧面 $K > 0$（红色），内侧面 $K < 0$（蓝色），顶部和底部的抛物线 $K = 0$（白色/绿色）。画出彩色环面，标注正负曲率区域面积恰好抵消，总积分 $\int K = 0 = 2\pi \cdot 0$。
  - **双柄曲面（$g=2$）**：嵌入 $\mathbb{R}^3$ 后，大部分区域 $K < 0$（蓝色为主），少量凸起处 $K > 0$（红色斑点）。总积分 $\int K = -4\pi = 2\pi \cdot (-2)$。

**测地三角形的"角度盈余/亏量"图**：

- 在三种曲面上各画一个测地三角形：
  - **球面**：从北极到赤道取三个等距点，画出大圆弧连接。每个角为 $\pi/2$，内角和 $3\pi/2 = \pi + \pi/2$。角度盈余 $\pi/2$ 等于球面三角形的面积（球面 $K=1$，$\int_\Delta K = \text{面积}$）。
  - **欧氏平面**：标准三角形，内角和 $= \pi$。无盈余无亏量。
  - **双曲圆盘（Poincaré 模型）**：画三条测地线（垂直于边界的圆弧或直径）围成的三角形。内角和明显小于 $\pi$。角度亏量等于双曲面积（$K = -1$）。
- 用饼图展示内角和与 $\pi$ 的差异，标注 $\int_\Delta K \, \mathrm{d}A$ 的值。

**向量场与指标定理的"奇点花园"图**：

- Poincaré-Hopf 定理说：$\chi(M) = \sum_{\text{零点 } p} \mathrm{ind}(V, p)$，其中 $V$ 是向量场。
- 在球面上画一个"源"型向量场（所有流线从南极发出，汇聚到北极）。南极指标 $+1$，北极指标 $+1$，总和 $2 = \chi(S^2)$。
- 在环面上画出无零点的平行向量场（如纬线方向）。零个零点的指标和为 $0 = \chi(T^2)$。
- 在双柄曲面上画一个有两个鞍点的向量场。每个鞍点指标 $-1$，总和 $-2 = \chi(\Sigma_2)$。
- 将向量场图与曲率染色图并置，展示两种不同的拓扑不变量（曲率积分与向量场指标）给出同一个 Euler 示性数。

## 直观解释

高斯-博内定理是微分几何中最深刻的结果之一，它揭示了**局部几何（曲率）与全局拓扑（Euler 示性数）之间的精确联系。**

Gauss 的卓越贡献在于证明了 $K$ 是**内蕴的**——它只依赖于曲面上的度量（第一基本形式），而不依赖于曲面在三维空间中的具体嵌入方式。这意味着：一只生活在曲面上的二维生物，通过测量三角形内角和与 $\pi$ 的偏差，就可以"发现"自己所在空间的弯曲，而无需跳到三维空间中去观察。

Bonnet 的贡献是将局部的曲率积分与整体的拓扑不变量联系起来。定理说：无论曲面的具体形状如何变化（拉伸、弯曲但不撕裂），曲率在全曲面上的总积分保持不变——它只由曲面的"洞"的个数（亏格）决定。

这一定理是 Atiyah-Singer 指标定理的二维原型，也是现代几何中"局部不变量决定全局拓扑"这一哲学最早的范例。

## 例子

1. **球面三角形**：单位球面上边长为 $\pi/2$ 的等边测地三角形（由三个坐标平面截取八分之一球面）。每个角为 $\pi/2$，面积 $= \pi/2 = 3\pi/2 - \pi$。

2. **双曲八边形**：亏格 2 的曲面可由双曲平面上一个正八边形（内角 $\pi/4$）通过对边粘合得到。Poincaré 多边形定理保证了这种双曲度量存在。$\int K = -\text{面积} = -4\pi$。

3. **宇宙的形状**：若宇宙是紧致的二维曲面，高斯-博内定理暗示宇宙的总曲率（能量密度积分）与宇宙拓扑相关。在三维中，这一想法推广为 Hamilton 的 Ricci 流与 Perelman 对 Poincaré 猜想的证明。

4. **离散 Gauss-Bonnet**：对多面体表面，在每个顶点 $v$ 处定义离散曲率 $K(v) = 2\pi - \sum_{\text{角}} \alpha_i$。则 $\sum_v K(v) = 2\pi \chi$。这是 Regge 微积分和计算几何中的基本公式。
