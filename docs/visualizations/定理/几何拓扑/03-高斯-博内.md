# 高斯-博内定理

## 描述

曲面的全曲率等于欧拉示性数的2π倍，连接了微分几何与拓扑。

## 定理

∫_M K dA = 2π χ(M)

## 可视化

```mermaid
graph TD
    A[高斯曲率 K] --> B[全曲率 ∫KdA]
    B --> C[2πχ(M)]
    C --> D[欧拉示性数]
    
    E[球面 K=1] --> F[χ=2, ∫KdA=4π]
    G[环面 K=0] --> H[χ=0, ∫KdA=0]
end

```
