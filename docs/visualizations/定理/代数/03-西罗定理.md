# 西罗定理完整版

## 描述

西罗三定理是有限群论中关于p子群的核心结果。

## 定理

1. 存在性: |G| = pⁿm，则存在pⁿ阶子群

2. 共轭性: 所有Sylow p子群共轭
3. n_p ≡ 1 mod p, n_p | m
