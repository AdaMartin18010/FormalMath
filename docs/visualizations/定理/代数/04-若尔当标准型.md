# 若尔当标准型定理

## 描述

任何复方阵都相似于唯一的若尔当标准型。

## 定理

A = PJP⁻¹, J为若尔当块对角阵。

## 可视化

```mermaid
graph TD
    A[矩阵 A] --> B[特征值 λ₁, λ₂, ...]
    B --> C[若尔当块 Jᵢ]
    C --> D[对角元 λ, 上次对角元 1]
end

```
