# 希尔伯特基定理

## 描述

诺特环上的多项式环仍是诺特环。

## 定理

R 诺特 ⟹ R[x] 诺特。

## 可视化

```mermaid
graph TD
    A[诺特环 R] --> B[多项式环 R[x]]
    B --> C[升链条件]
    C --> D[理想有限生成]
end
```
