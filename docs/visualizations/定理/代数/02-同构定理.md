# 同构定理 - 群同态分解

## 描述

群同态基本定理（第一同构定理）建立了群同态与商群之间的联系。

## 定理陈述

若 φ: G → H 是群同态，则 G/ker(φ) ≅ im(φ)。

## 可视化代码

```mermaid
graph TD
    A[G] -->|φ| B[H]

    E[kerφ ⊲ G] --> F[正规子群]
end
```
