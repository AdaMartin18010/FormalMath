# 拉格朗日定理 - 陪集划分可视化

## 描述

本可视化展示群论中的拉格朗日定理：有限群G的子群H的阶整除G的阶。通过左陪集划分展示定理的几何直观。

## 数学概念

**拉格朗日定理**: 若G是有限群，H ≤ G是子群，则|H|整除|G|，且 [G:H] = |G|/|H| 称为H在G中的指数。

**左陪集**: 对g ∈ G，gH = {gh : h ∈ H}

关键观察:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 陪集互不相交: gᵢH ∩ gⱼH = ∅ (i≠j)
2. 陪集大小相等: |gH| = |H| 对所有 g ∈ G

3. 完全覆盖: G = ⨆ gᵢH
4. 因此: |G| = Σ|gᵢH| = [G:H] · |H|

几何类比:
───────────────────────────────────────────────────────────────
想象群G是一个蛋糕，子群H是切蛋糕的模具。
拉格朗日定理说：你可以把蛋糕切成 [G:H] 块，每块大小都是 |H|。

具体例子: S₃ 和 H = {e, (123), (132)}
═══════════════════════════════════════════════════════════════
G = S₃ = {e, (12), (13), (23), (123), (132)}  |G| = 6
H = A₃ = {e, (123), (132)}                   |H| = 3

陪集划分:
  eH = {e, (123), (132)} = H
  (12)H = {(12), (13), (23)}

所以 [S₃:A₃] = 2, 验证了 6 = 2 × 3

重要推论:
───────────────────────────────────────────────────────────────
• 元素阶整除群阶: 若 g ∈ G, 则 |g| 整除 |G|
• 素数阶群必循环: 若 |G| = p (素数), 则 G ≅ ℤ/pℤ
• 指数计算: [G:H] = |G|/|H|
