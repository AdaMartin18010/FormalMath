# 柯西积分公式 - 复平面积分可视化

## 1. 柯西积分定理回顾

**定理 1.1（Cauchy 积分定理）**. 设 $f$ 在单连通区域 $\Omega \subseteq \mathbb{C}$ 上全纯，$\gamma$ 为 $\Omega$ 中的分段光滑闭曲线，则
$$\oint_\gamma f(z)\,dz = 0.$$

*证明*. 写 $f(z)\,dz = (u\,dx - v\,dy) + i(v\,dx + u\,dy)$。由 Green 定理和 Cauchy-Riemann 方程 $u_x = v_y$, $u_y = -v_x$，两部分被积函数均为零：
$$\oint_\gamma f(z)\,dz = \iint_D (-v_x - u_y)\,dx\,dy + i\iint_D (u_x - v_y)\,dx\,dy = 0.$$
$\square$

## 2. 柯西积分公式

**定理 2.1（Cauchy 积分公式）**. 设 $f$ 在闭圆盘 $\overline{D}(z_0, R)$ 的某邻域上全纯，则对 $|z - z_0| < R$：
$$f(z) = \frac{1}{2\pi i} \oint_{|\zeta - z_0| = R} \frac{f(\zeta)}{\zeta - z}\, d\zeta.$$

*证明*. 对固定 $z$（$|z - z_0| < R$），在 $\zeta = z$ 处挖去小圆盘 $D(z, \varepsilon)$。函数 $g(\zeta) = \frac{f(\zeta)}{\zeta - z}$ 在环形区域 $\{R' < |\zeta - z_0| < R\} \setminus D(z, \varepsilon)$ 上全纯（取 $R'$ 使 $|z - z_0| < R' < R$）。由 Cauchy 定理，
$$\oint_{|\zeta - z_0| = R} g(\zeta)\,d\zeta = \oint_{|\zeta - z| = \varepsilon} g(\zeta)\,d\zeta.$$
在小圆上，$f(\zeta) = f(z) + O(\varepsilon)$，故
$$\oint_{|\zeta - z| = \varepsilon} \frac{f(\zeta)}{\zeta - z}\,d\zeta = f(z)\oint_{|\zeta - z| = \varepsilon} \frac{d\zeta}{\zeta - z} + O(\varepsilon) = 2\pi i \, f(z) + O(\varepsilon).$$
令 $\varepsilon \to 0$ 即得。$\square$

## 3. 高阶导数公式

**定理 3.1（高阶导数公式）**. 在同样条件下，$f$ 在 $D(z_0, R)$ 上无穷可微，且
$$f^{(n)}(z) = \frac{n!}{2\pi i} \oint_{|\zeta - z_0| = R} \frac{f(\zeta)}{(\zeta - z)^{n+1}}\, d\zeta.$$

*证明*. 对 $n=1$，在积分号下求导：
$$\frac{d}{dz}\left(\frac{1}{2\pi i}\oint \frac{f(\zeta)}{\zeta - z}\,d\zeta\right) = \frac{1}{2\pi i}\oint \frac{f(\zeta)}{(\zeta - z)^2}\,d\zeta.$$
由被积函数关于 $(z, \zeta)$ 的一致连续性，求导与积分可交换。归纳即得一般公式。$\square$

**推论 3.2（Liouville 定理）**. 有界整函数必为常数。

*证明*. 设 $|f(z)| \leq M$ 对所有 $z \in \mathbb{C}$。对任意 $z_0$，取半径 $R$ 的大圆：
$$|f'(z_0)| = \left|\frac{1}{2\pi i}\oint_{|\zeta - z_0| = R} \frac{f(\zeta)}{(\zeta - z_0)^2}\,d\zeta\right| \leq \frac{1}{2\pi} \cdot \frac{M}{R^2} \cdot 2\pi R = \frac{M}{R} \to 0$$
当 $R \to \infty$。故 $f' \equiv 0$，$f$ 为常数。$\square$

**推论 3.3（代数学基本定理）**. 每个非常值复多项式 $P(z)$ 在 $\mathbb{C}$ 中有根。

*证明*. 若 $P(z) \neq 0$ 对所有 $z$，则 $1/P(z)$ 为整函数。当 $|z| \to \infty$，$|1/P(z)| \to 0$，故有界。由 Liouville 定理，$1/P$ 为常数，矛盾。$\square$

## 4. 均值性质与最大模原理

**定理 4.1（均值性质）**. $f$ 在 $\overline{D}(z_0, R)$ 上全纯，则
$$f(z_0) = \frac{1}{2\pi} \int_0^{2\pi} f(z_0 + Re^{i\theta})\, d\theta.$$

*证明*. 在 Cauchy 积分公式中取 $z = z_0$，参数化 $\zeta = z_0 + Re^{i\theta}$，$d\zeta = iRe^{i\theta}\,d\theta$：
$$f(z_0) = \frac{1}{2\pi i}\int_0^{2\pi} \frac{f(z_0 + Re^{i\theta})}{Re^{i\theta}} \cdot iRe^{i\theta}\,d\theta = \frac{1}{2\pi}\int_0^{2\pi} f(z_0 + Re^{i\theta})\,d\theta.$$
$\square$

**定理 4.2（最大模原理）**. 设 $f$ 在有界区域 $\Omega$ 上全纯且在 $\overline{\Omega}$ 连续。则 $|f|$ 在 $\overline{\Omega}$ 上的最大值在边界 $\partial\Omega$ 上达到。若 $|f|$ 在内部某点达到最大值，则 $f$ 为常数。

*证明*. 设 $|f|$ 在 $z_0 \in \Omega$ 达到最大值 $M$。由均值性质，对充分小 $R$，
$$M = |f(z_0)| \leq \frac{1}{2\pi}\int_0^{2\pi} |f(z_0 + Re^{i\theta})|\,d\theta \leq M.$$
等号成立要求 $|f(z_0 + Re^{i\theta})| = M$ 对所有 $\theta$ 成立。由 $R$ 的任意性，$|f|$ 在 $z_0$ 的邻域上恒为 $M$。由连通性，$|f|$ 在 $\Omega$ 上恒为 $M$，再由 Cauchy-Riemann 方程，$f$ 为常数。$\square$
