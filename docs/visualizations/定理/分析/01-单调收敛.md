# 单调收敛定理

## 描述

单调收敛定理是勒贝格积分的基石，说明单调递增可测函数列的积分与极限可交换。

## 定理陈述

若 0 ≤ f₁ ≤ f₂ ≤ ... → f，则 ∫fₙ → ∫f。
