# 单调收敛定理可视化

## 概念定义

设 $(X, \mathcal{M}, \mu)$ 为测度空间，$\{f_n\}$ 为可测函数列。

**单调收敛定理（Monotone Convergence Theorem, MCT）**：若 $0 \leq f_1 \leq f_2 \leq f_3 \leq \cdots$ 为单调递增收敛到 $f$（逐点，即 $f_n(x) \nearrow f(x)$ 对所有 $x \in X$），则
$$\lim_{n \to \infty} \int_X f_n \, \mathrm{d}\mu = \int_X f \, \mathrm{d}\mu。$$

注意 $f$ 可以是取值为 $+\infty$ 的扩展实值函数，且等式两边都可能为 $+\infty$。

## 关键公式

**MCT 的积分等式**：
$$\int \left(\lim_{n \to \infty} f_n\right) \, \mathrm{d}\mu = \lim_{n \to \infty} \int f_n \, \mathrm{d}\mu = \sup_n \int f_n \, \mathrm{d}\mu。$$

**证明概要**：因 $f_n \leq f$，有 $\int f_n \leq \int f$，故 $\limsup \int f_n \leq \int f$。对任意简单函数 $s \leq f$ 和 $0 < c < 1$，令 $E_n = \{f_n \geq cs\}$。则 $E_n \nearrow X$，且
$$\int f_n \geq \int_{E_n} f_n \geq c \int_{E_n} s \to c \int s。$$
取 $c \to 1$ 和 $s \nearrow f$ 得上界。

**与 Fatou 引理的关系**：Fatou 引理说
$$\int \liminf f_n \leq \liminf \int f_n。$$
MCT 是 Fatou 引理在单调递增非负情形的等号版本。

## 可视化说明

**"阶梯攀登"图**：

- 在 $X = [0, 1]$ 上，画出函数列 $f_n(x) = x^n$（这不是 MCT 的直接例子，因不单调递增），改为 $f_n(x) = \min(nx, 1)$。
- 画出 $f_1$（斜率为 1 的折线）、$f_2$（斜率为 2 的折线，更早达到 1）、$f_5$、$f_{10}$。随着 $n$ 增大，折线越来越接近阶梯函数 $f(x) = \mathbf{1}_{(0,1]}(x)$。
- **关键可视化**：每条 $f_n$ 下方的面积（积分）是一个梯形加矩形，面积为 $1 - 1/(2n)$，单调递增收敛到 $1$。极限函数 $f$ 下方的面积也是 $1$。用颜色填充各 $f_n$ 与 $f$ 之间的差异区域，展示差异面积趋于零。
- 标注：单调性保证了"增量不丢失"——后一个函数总在前一个之上，故积分只增不减。

**函数图像的"层叠信封"图**：

- 画出若干条单调递增的函数曲线 $f_1, f_2, f_3, \ldots$，像一叠逐渐增厚的信封。最底层是 $f_1$，最上层是极限 $f$。
- 用渐变色从深蓝（$f_1$）到浅蓝（$f_n$）再到红色（$f$）标注各层。
- 对每一点 $x$，画出竖直切片：$f_1(x) \leq f_2(x) \leq \cdots \leq f(x)$。这形成一个单调数列，其极限为 $f(x)$。
- 积分 $\int f_n$ 是每层曲面包围的体积。MCT 说这些体积的极限等于极限曲面包围的体积。用三维柱状图展示：横轴为 $x$，纵轴为函数值，高度轴可忽略；每层的"体积柱"单调增高，趋于极限柱。

**非负性的必要"断裂"图**：

- 展示为什么 MCT 要求 $f_n \geq 0$（或至少 $f_n \geq g$ 对某个可积 $g$）。取 $f_n = -\frac{1}{n} \chi_{[0,n]}$（在 $\mathbb{R}$ 上）。
- 画出 $f_n$：在 $[0,n]$ 上为高度 $-1/n$ 的矩形。$f_n \nearrow 0$ 逐点，但 $\int f_n = -1$ 不趋于 $\int 0 = 0$。
- 标注：负值部分的"质量逃逸"到无穷远，导致积分不收敛到极限积分。用动画展示矩形如何变矮变宽，总面积保持 $-1$，但极限函数恒为零。

## 直观解释

单调收敛定理是 Lebesgue 积分理论中第一个也是最基本的极限定理。它的核心思想极其简单：**如果你有一列越来越大的区域，它们并集的面积就是各自面积的上确界。**

与 Riemann 积分不同，Lebesgue 积分对极限更为宽容。MCT 允许函数列的收敛是点态的、非一致的，甚至极限函数可以无界——只要单调性和非负性满足，积分与极限总可交换。

非负性条件的本质在于防止"正负质量相互抵消后逃逸"。若允许函数取负值，一列函数可以在保持总积分不变的同时，将正质量推向 $+\infty$、负质量推向 $-\infty$，使得逐点极限为零但总积分非零。这正是上面 $-\frac{1}{n}\chi_{[0,n]}$ 反例的几何机制。

## 例子

1. **Borel-Cantelli 引理的证明辅助**：设 $\{A_n\}$ 为可测集列，$\sum \mu(A_n) < \infty$。令 $f_n = \sum_{k=1}^n \chi_{A_k}$，则 $f_n \nearrow f = \sum_k \chi_{A_k}$。由 MCT，$\int f = \sum \mu(A_n) < \infty$，故 $f < \infty$ a.e.，即几乎所有点只属于有限个 $A_n$。

2. **级数与积分的交换**：设 $g_k \geq 0$ 可测。则
   $$\int \sum_{k=1}^\infty g_k = \sum_{k=1}^\infty \int g_k。$$
   证明：令 $f_n = \sum_{k=1}^n g_k$，则 $f_n \nearrow \sum_{k=1}^\infty g_k$，应用 MCT。

3. **Gamma 函数**：$\Gamma(s) = \int_0^\infty t^{s-1} e^{-t} \, \mathrm{d}t$。对 $s > 0$，令 $f_n(t) = t^{s-1} e^{-t} \chi_{[1/n, n]}(t)$，则 $f_n \nearrow t^{s-1}e^{-t}$，由 MCT 可得 $\Gamma(s) = \lim \int_{1/n}^n t^{s-1}e^{-t}\,\mathrm{d}t$。

4. **概率期望**：设 $X \geq 0$ 为随机变量。定义 $X_n = \sum_{k=0}^{n2^n} \frac{k}{2^n} \mathbf{1}_{\{k/2^n \leq X < (k+1)/2^n\}}$，则 $X_n \nearrow X$。由 MCT，$\mathbb{E}[X] = \lim \mathbb{E}[X_n]$，这是非负随机变量期望的标准定义方式。
