# 里斯表示定理

## 描述

希尔伯特空间上的连续线性泛函都可由内积表示。

## 定理

∀φ ∈ H*, ∃! y ∈ H, φ(x) = ⟨x,y⟩。

## 可视化

```mermaid
graph TD
    A[对偶空间 H*] --> B[同构 H]
    B --> C[φ ↔ y_φ]
    C --> D[φ(x) = ⟨x, y_φ⟩]
end
```
