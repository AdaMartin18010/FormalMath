# 中值定理 - 几何证明可视化

## 描述

本可视化展示微分中值定理的几何证明，通过罗尔定理和辅助函数构造，直观展示定理的几何意义。

## 数学概念

**拉格朗日中值定理**: 若f在[a,b]连续，在(a,b)可导，则存在c∈(a,b)使得
$$f'(c) = \frac{f(b)-f(a)}{b-a}$$

几何意义: 曲线上某点的切线平行于连接端点的弦。

```
证明步骤:
═══════════════════════════════════════════════════════════════

步骤1: 弦的方程
───────────────────────────────────────────────────────────────
过(a,f(a))和(b,f(b))的直线:
L(x) = f(a) + (f(b)-f(a))/(b-a) * (x-a)

步骤2: 构造辅助函数
───────────────────────────────────────────────────────────────
g(x) = f(x) - L(x)

性质:
• g(a) = f(a) - L(a) = 0
• g(b) = f(b) - L(b) = 0
• g在[a,b]连续，在(a,b)可导

步骤3: 应用罗尔定理
───────────────────────────────────────────────────────────────
罗尔定理: g(a)=g(b)=0 ⇒ 存在c∈(a,b), g'(c)=0

步骤4: 计算g'(c)
───────────────────────────────────────────────────────────────
g'(x) = f'(x) - L'(x) = f'(x) - (f(b)-f(a))/(b-a)

g'(c) = 0 ⇒ f'(c) = (f(b)-f(a))/(b-a)

证毕！

几何解释:
┌─────────────────────────────────────────────────────────────┐
│  曲线上存在一点c，其切线斜率等于连接端点弦的斜率             │
│                                                             │
│  直观理解: 如果你从(a,f(a))开车到(b,f(b))，                  │
│  那么必定有一刻你的瞬时速度等于平均速度                      │
└─────────────────────────────────────────────────────────────┘

重要推论:
───────────────────────────────────────────────────────────────
• f' ≡ 0 ⇒ f 为常数
• f' > 0 ⇒ f 严格递增
• 柯西中值定理
• 洛必达法则
```
