# 中值定理 - 几何证明可视化

## 1. Rolle 定理

**定理 1.1（Rolle）**. 设 $f$ 在 $[a,b]$ 上连续，在 $(a,b)$ 上可导，且 $f(a) = f(b)$。则存在 $c \in (a,b)$ 使得 $f'(c) = 0$。

*证明*. 由最值定理，$f$ 在 $[a,b]$ 上达到最大值 $M$ 和最小值 $m$。若 $M = m$，则 $f$ 为常数，$f' \equiv 0$。否则 $M > f(a)$ 或 $m < f(a)$。不妨设 $M > f(a)$，最大值在某点 $c \in (a,b)$ 达到。对 $h > 0$ 充分小，
$$\frac{f(c+h)-f(c)}{h} \leq 0, \quad \frac{f(c-h)-f(c)}{-h} \geq 0.$$
令 $h \to 0^+$，得 $f'(c) \leq 0$ 且 $f'(c) \geq 0$，故 $f'(c) = 0$。$\square$

## 2. Lagrange 中值定理

**定理 2.1（Lagrange 中值定理）**. 设 $f$ 在 $[a,b]$ 上连续，在 $(a,b)$ 上可导。则存在 $c \in (a,b)$ 使得
$$f'(c) = \frac{f(b) - f(a)}{b - a}.$$

*几何意义*. 曲线上某点 $c$ 的切线平行于连接端点 $(a, f(a))$ 和 $(b, f(b))$ 的弦。

*证明*. 构造辅助函数
$$g(x) = f(x) - \left[f(a) + \frac{f(b)-f(a)}{b-a}(x-a)\right].$$
则 $g(a) = g(b) = 0$，$g$ 满足 Rolle 定理条件。故存在 $c \in (a,b)$ 使 $g'(c) = 0$，即
$$f'(c) - \frac{f(b)-f(a)}{b-a} = 0.$$
$\square$

## 3. Cauchy 中值定理

**定理 3.1（Cauchy 中值定理）**. 设 $f, g$ 在 $[a,b]$ 上连续，在 $(a,b)$ 上可导，且 $g'(x) \neq 0$ 对所有 $x \in (a,b)$。则存在 $c \in (a,b)$ 使得
$$\frac{f'(c)}{g'(c)} = \frac{f(b) - f(a)}{g(b) - g(a)}.$$

*证明*. 首先 $g(a) \neq g(b)$（否则由 Rolle 定理存在 $g' = 0$ 的点，矛盾）。构造
$$h(x) = f(x) - f(a) - \frac{f(b)-f(a)}{g(b)-g(a)}(g(x) - g(a)).$$
则 $h(a) = h(b) = 0$，应用 Rolle 定理得 $h'(c) = 0$，即
$$f'(c) - \frac{f(b)-f(a)}{g(b)-g(a)}g'(c) = 0.$$
$\square$

## 4. 重要推论

**推论 4.1**. 设 $f$ 在区间 $I$ 上可导。
- 若 $f' \equiv 0$，则 $f$ 为常数；
- 若 $f' > 0$，则 $f$ 严格递增；
- 若 $f' < 0$，则 $f$ 严格递减。

*证明*. 对任意 $x_1 < x_2$，由 Lagrange 中值定理，存在 $c \in (x_1, x_2)$ 使 $f(x_2) - f(x_1) = f'(c)(x_2 - x_1)$。由 $f'(c)$ 的符号即得。$\square$

**推论 4.2（L'Hôpital 法则）**. 设 $f, g$ 在 $(a,b)$ 上可导，$g' \neq 0$，且 $\lim_{x \to a^+} f(x) = \lim_{x \to a^+} g(x) = 0$。若 $\lim_{x \to a^+} \frac{f'(x)}{g'(x)} = L$，则
$$\lim_{x \to a^+} \frac{f(x)}{g(x)} = L.$$

*证明*. 延拓 $f(a) = g(a) = 0$。对 $x > a$，由 Cauchy 中值定理，存在 $c_x \in (a,x)$ 使
$$\frac{f(x)}{g(x)} = \frac{f(x)-f(a)}{g(x)-g(a)} = \frac{f'(c_x)}{g'(c_x)}.$$
当 $x \to a^+$，$c_x \to a^+$，故极限为 $L$。$\square$

## 5. 几何图示

```
       y │
         │          ╭──────────╮
    f(b) ├─────────●           ╲
         │       ╱ │            ╲
         │     ╱   │             ╲
         │   ╱     │              ╲
    f(a) ├─●───────┼───────────────╲──────→ x
         │ a       │   c           b
         │         │   ↑
         │         │  切线//弦
         │       ╱────────────────── 弦
         │     ╱
         └─────
```

**直观理解**：如果你从 $(a, f(a))$ 开车到 $(b, f(b))$，那么必定有一刻你的瞬时速度（切线斜率）等于平均速度（弦的斜率）。
