# 康托尔对角线证明可视化

## 1. 集合的势

**定义 1.1（等势）**. 集合 $A$ 与 $B$ 等势（记 $|A| = |B|$），若存在双射 $f: A \to B$。

**定义 1.2（Cantor-Bernstein-Schröder）**. 若 $|A| \leq |B|$ 且 $|B| \leq |A|$，则 $|A| = |B|$。

*证明概要*. 设 $f: A \to B$ 和 $g: B \to A$ 为单射。构造 $A$ 的子集 $C$ 为 $g$ 的像的迭代原像的并，在 $C$ 上用 $g^{-1}$，在 $A \setminus C$ 上用 $f$ 构造双射。$\square$

## 2. 对角线论证

**定理 2.1（Cantor）**. 实数区间 $[0,1]$ 是不可数集。

*证明*. 假设 $[0,1]$ 可数，则可将其元素枚举为序列 $r_1, r_2, r_3, \dots$。将每个实数写成十进制展开（避免无限循环 9 的表示）：
$$\begin{aligned}
r_1 &= 0.d_{11} d_{12} d_{13} d_{14} \dots \\
r_2 &= 0.d_{21} d_{22} d_{23} d_{24} \dots \\
r_3 &= 0.d_{31} d_{32} d_{33} d_{34} \dots \\
&\vdots
\end{aligned}$$

构造新数 $x = 0.x_1 x_2 x_3 \dots$，其中
$$x_n = \begin{cases} 1 & \text{若 } d_{nn} \neq 1, \\ 2 & \text{若 } d_{nn} = 1. \end{cases}$$

则 $x \in [0,1]$，但对每个 $n$，$x_n \neq d_{nn}$，故 $x \neq r_n$。这与枚举包含所有 $[0,1]$ 中实数矛盾！$\square$

**关键洞察**：无论怎么列举 $[0,1]$ 中的元素，总可以通过改变对角线上的数字构造一个不在列表中的新数。这种方法称为**对角线论证**（diagonal argument）。

## 3. 幂集定理

**定理 3.1（Cantor 幂集定理）**. 对任意集合 $A$，$|A| < |\mathcal{P}(A)|$。

*证明*. 单射 $a \mapsto \{a\}$ 给出 $|A| \leq |\mathcal{P}(A)|$。假设存在满射 $f: A \to \mathcal{P}(A)$。构造
$$B = \{a \in A : a \notin f(a)\} \in \mathcal{P}(A).$$
若 $B = f(b)$ 对某 $b \in A$，则
$$b \in B \iff b \notin f(b) = B,$$
矛盾！故不存在满射，$|A| < |\mathcal{P}(A)|$。$\square$

**推论 3.2**. 存在无穷多个不同的无穷基数：
$$|\mathbb{N}| < |\mathcal{P}(\mathbb{N})| < |\mathcal{P}(\mathcal{P}(\mathbb{N}))| < \cdots$$

注意到 $|\mathbb{R}| = |\mathcal{P}(\mathbb{N})| = 2^{\aleph_0}$（由二进制展开）。

## 4. 连续统假设

**定义 4.1（连续统假设，CH）**. 不存在基数 $\kappa$ 使得 $\aleph_0 < \kappa < 2^{\aleph_0}$。等价地，$|\mathbb{R}| = \aleph_1$。

**定理 4.2（Gödel; Cohen）**. CH 在 ZFC 公理系统中既不可证也不可否证：
- Gödel (1940)：CH 与 ZFC 相容（若 ZFC 相容则 ZFC+CH 相容）；
- Cohen (1963)：$\neg$CH 与 ZFC 相容（引入力迫法）。

## 5. 对角线方法的威力

对角线方法不仅是证明不可数性的工具，也是计算理论中证明不可判定性的核心：

**定理 5.1（停机问题的不可判定性）**. 不存在通用算法 $H(P, x)$ 能判定任意程序 $P$ 在输入 $x$ 上是否停机。

*证明概要*. 假设 $H$ 存在。构造程序 $D(P)$：若 $H(P, P)$ 输出"停机"，则 $D$ 进入死循环；否则 $D$ 停机。考虑 $D(D)$：若 $D(D)$ 停机，则 $H(D,D)$ 为真，$D$ 应进入死循环；若 $D(D)$ 不停机，则 $H(D,D)$ 为假，$D$ 应停机。矛盾！$\square$

## 6. 可视化：对角线构造

```
康托尔对角线论证
═══════════════════════════════════════════════════════════════

假设 [0,1] 可数，则可枚举为 r₁, r₂, r₃, ...

写成小数展开:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   r₁ = 0 . d₁₁ d₁₂ d₁₃ d₁₄ d₁₅ ...                         │
│            ↓                                                │
│   r₂ = 0 . d₂₁ d₂₂ d₂₃ d₂₄ d₂₅ ...                         │
│                 ↓                                           │
│   r₃ = 0 . d₃₁ d₃₂ d₃₃ d₃₄ d₃₅ ...                         │
│                      ↓                                      │
│   r₄ = 0 . d₄₁ d₄₂ d₄₃ d₄₄ d₄₅ ...                         │
│                           ↓                                 │
│   r₅ = 0 . d₅₁ d₅₂ d₅₃ d₅₄ d₅₅ ...                         │
│                                                             │
│   ...                                                       │
│                                                             │
│   ↓ 对角线元素: d₁₁, d₂₂, d₃₃, d₄₄, d₅₅, ...               │
│                                                             │
└─────────────────────────────────────────────────────────────┘

构造新数 x = 0 . x₁ x₂ x₃ x₄ x₅ ...

规则: xₙ ≠ dₙₙ (对角线元素)

例如: xₙ = { 1  if dₙₙ ≠ 1
           { 2  if dₙₙ = 1

关键结论:
═══════════════════════════════════════════════════════════════

   x ≠ r₁ 因为 x₁ ≠ d₁₁  (第一位不同)
   x ≠ r₂ 因为 x₂ ≠ d₂₂  (第二位不同)
   x ≠ r₃ 因为 x₃ ≠ d₃₃  (第三位不同)
   ...
   x ≠ rₙ 因为 xₙ ≠ dₙₙ  (第n位不同)
   ...

   因此 x 不在枚举中！

   ┌───────────────────────────────────────────────────────┐
   │  这与"枚举包含所有实数"矛盾！                          │
   │  故 [0,1] 不可数。                                     │
   └───────────────────────────────────────────────────────┘
```
