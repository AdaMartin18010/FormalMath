# 数学归纳法原理

## 描述

归纳法是基于自然数良序原理的证明方法。

## 原理

基础步骤 + 归纳步骤 ⇒ 对所有n成立

## 可视化

```mermaid

    B --> C[∴ P(n) 对所有n成立]

    D[良序原理] --> E[归纳法]
end
```
