# 量子力学的数学结构可视化

## 概念定义

量子力学的数学框架建立在希尔伯特空间与自伴算子理论之上。

**量子态**：物理系统的状态由复可分希尔伯特空间 $\mathcal{H}$ 中的**单位射线**（ray）描述，即非零向量 $\psi \in \mathcal{H}$ 模去整体相位因子 $e^{i\theta}\psi$。等价地，状态由投影算子 $P_\psi = |\psi\rangle\langle\psi|$ 描述。

**可观测量**：物理可观测量对应 $\mathcal{H}$ 上的**自伴算子**（或无界自伴算子）$A = A^\dagger$。其谱 $\sigma(A) \subseteq \mathbb{R}$ 给出测量可能得到的所有取值。

**测量公设**：对处于状态 $\psi$ 的系统测量可观测量 $A$，得到特征值 $\lambda$ 的概率为
$$\mathbb{P}(A = \lambda) = \|P_\lambda \psi\|^2，$$
其中 $P_\lambda$ 是到 $\lambda$-特征空间的谱投影。测量后状态坍缩为
$$\psi \mapsto \frac{P_\lambda \psi}{\|P_\lambda \psi\|}。$$

**薛定谔方程**：封闭系统的演化由
$$i\hbar \frac{\partial}{\partial t} \psi(t) = H \psi(t)$$
给出，其中 $H = H^\dagger$ 为哈密顿算子（能量可观测量）。形式解为 $\psi(t) = e^{-iHt/\hbar} \psi(0)$。

## 关键公式

**海森堡不确定性原理**：对任意态 $\psi \in D(A) \cap D(B)$，
$$\sigma_\psi(A) \cdot \sigma_\psi(B) \geq \frac{1}{2} |\langle \psi, [A, B] \psi \rangle|，$$
其中 $\sigma_\psi(A) = \sqrt{\langle A^2 \rangle_\psi - \langle A \rangle_\psi^2}$ 为标准差，$[A, B] = AB - BA$。特别地，
$$\Delta x \cdot \Delta p \geq \frac{\hbar}{2}。$$

**谐振子**：一维量子谐振子 $H = \frac{p^2}{2m} + \frac{1}{2}m\omega^2 x^2$ 的谱为
$$E_n = \hbar\omega\left(n + \frac{1}{2}\right), \quad n = 0, 1, 2, \ldots$$
本征态为 Hermite 函数 $\psi_n(x) = \frac{1}{\sqrt{2^n n!}} \left(\frac{m\omega}{\pi\hbar}\right)^{1/4} H_n\left(\sqrt{\frac{m\omega}{\hbar}}x\right) e^{-m\omega x^2/2\hbar}$。

**纠缠与 Bell 不等式**：对 bipartite 系统 $\mathcal{H}_A \otimes \mathcal{H}_B$，态 $|\Psi\rangle$ 称为可分的若 $|\Psi\rangle = |\psi_A\rangle \otimes |\psi_B\rangle$；否则称为纠缠态。CHSH 不等式说任何局域隐变量理论满足 $|S| \leq 2$，而量子力学允许 $|S| = 2\sqrt{2}$（Tsirelson bound）。

## 可视化说明

**希尔伯特空间中的态演化"球面轨迹"图**：

- 对两能级系统（qubit），$\mathcal{H} = \mathbb{C}^2$，状态空间是 Bloch 球 $S^2$（单位射线空间）。
- 画一个三维球面，北极代表 $|0\rangle$，南极代表 $|1\rangle$，赤道上的点代表等概率叠加态 $(|0\rangle + e^{i\phi}|1\rangle)/\sqrt{2}$。
- **薛定谔演化**：哈密顿量 $H$ 生成酉演化，对应 Bloch 球上的刚体旋转。画出从 $|0\rangle$ 出发的轨迹：若 $H = \hbar\omega \sigma_z/2$，则态在纬线上匀速转动；若 $H$ 含 $\sigma_x$ 分量，则轨迹为经线或大圆。
- **测量坍缩**：在点 $\psi$ 处测量 $\sigma_z$。以概率 $\cos^2(\theta/2)$ 坍缩到北极 $|0\rangle$，以概率 $\sin^2(\theta/2)$ 坍缩到南极 $|1\rangle$，其中 $\theta$ 是 $\psi$ 的余纬。画出从 $\psi$ 到两极的两条虚线箭头，标注概率。

**波函数的"概率云"图**：

- 对一维谐振子基态 $\psi_0(x)$，画出高斯曲线 $|\psi_0(x)|^2 = \sqrt{\frac{m\omega}{\pi\hbar}} e^{-m\omega x^2/\hbar}$。用颜色填充曲线下方区域表示概率密度。
- 对激发态 $\psi_n(x)$，画出带有 $n$ 个节点的振荡曲线。用竖直虚线标出经典转折点 $x = \pm \sqrt{(2n+1)\hbar/m\omega}$，展示量子隧穿效应：在经典禁区（$|\psi_n|^2$ 的指数衰减尾部），粒子仍有非零出现概率。
- **不确定性可视化**：画出波包 $\psi(x) = (2\pi\sigma_x^2)^{-1/4} e^{-x^2/4\sigma_x^2}$ 及其 Fourier 变换 $\tilde{\psi}(p)$。当 $\sigma_x$ 变窄（位置确定），$\sigma_p$ 变宽（动量不确定）；反之亦然。动画展示 $\sigma_x \sigma_p = \hbar/2$ 的跷跷板关系。

**纠缠态的"关联测量"图**：

- 对 Bell 态 $|\Phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$，画出两个空间分离的测量装置（Alice 和 Bob）。
- 当 Alice 测量她的粒子得 $|0\rangle$ 时，Bob 的粒子瞬时（非定域地）坍缩到 $|0\rangle$。用闪电状虚线连接两个测量事件，但标注"无超光速信号传递"——因为 Alice 无法控制自己得到 $|0\rangle$ 还是 $|1\rangle$（各 $50\%$），她无法利用这种关联发送信息。
- 画出 CHSH 实验的四种设置组合（$A_1B_1, A_1B_2, A_2B_1, A_2B_2$）及其量子关联值。用柱状图对比经典界 $|S| \leq 2$ 与量子值 $2\sqrt{2} \approx 2.828$，突出量子纠缠超越局域实在论的范围。

## 直观解释

量子力学与经典力学的根本区别在于**态的线性叠加**与**测量的概率性**。在经典力学中，粒子的状态由相空间中的点 $(q, p)$ 完全确定；在量子力学中，状态是希尔伯特空间中的向量，可以同时"处于"多个经典状态的叠加中。

叠加原理的直接后果是**干涉**：概率不是相加，而是振幅（波函数值）相加后再取模平方。这解释了双缝实验——粒子通过两条缝的概率不等于各自概率之和，而是包含交叉干涉项 $2\mathrm{Re}(\psi_1^* \psi_2)$。

测量公设引入了量子力学最具争议的方面——**波包坍缩**。从数学上看，这是 von Neumann 将投影算子与谱分解联系起来的结果；从物理哲学上看，它引发了关于测量问题、退相干和多世界诠释的持续争论。

纠缠态的奇妙之处在于：两个粒子可以处于一个无法分解为各自状态乘积的联合态中。这种关联是**非定域的**——即使粒子相距光年，对一个粒子的测量会瞬间影响另一个粒子的状态。然而，这种非定域性不能用于超光速通信（no-signaling theorem），因为它不携带可操控的信息。

## 例子

1. **自由粒子**：$H = p^2/2m$，平面波 $\psi_k(x) = e^{ikx}$ 是能量本征态（$E = \hbar^2 k^2/2m$），但不可归一化。物理态由波包叠加构成：
   $$\psi(x, t) = \int \hat{\psi}(k) e^{ikx - i\omega(k)t} \, \mathrm{d}k，$$
   波包中心以群速度 $v_g = \mathrm{d}\omega/\mathrm{d}k = \hbar k_0/m$ 运动，同时扩散（波包变宽）。

2. **氢原子**：$H = \frac{p^2}{2m} - \frac{e^2}{4\pi\varepsilon_0 r}$。能量本征值为
   $$E_n = -\frac{m e^4}{2(4\pi\varepsilon_0)^2 \hbar^2} \frac{1}{n^2} = -\frac{13.6\ \mathrm{eV}}{n^2}，$$
   简并度为 $n^2$（计自旋则为 $2n^2$）。基态波函数为 $\psi_{100}(r) = \frac{1}{\sqrt{\pi a_0^3}} e^{-r/a_0}$，$a_0$ 为 Bohr 半径。

3. **量子隧穿**：粒子能量 $E$ 小于势垒高度 $V_0$ 时，经典力学预言完全反射；量子力学中，波函数在势垒区内指数衰减但非零，故在势垒另一侧有非零透射概率
   $$T \approx \exp\left(-\frac{2}{\hbar} \int_{x_1}^{x_2} \sqrt{2m(V(x)-E)} \, \mathrm{d}x\right)。$$
   这是扫描隧道显微镜（STM）和核聚变中量子隧穿的理论基础。

4. **路径积分表述**：Feynman 给出了与薛定谔方程等价的表述：
   $$\langle x_f, t_f | x_i, t_i \rangle = \int_{x(t_i)=x_i}^{x(t_f)=x_f} e^{iS[x(t)]/\hbar} \, \mathcal{D}x(t)，$$
   其中 $S = \int L \, \mathrm{d}t$ 为经典作用量。在 $\hbar \to 0$ 极限下，驻相近似给出经典轨道（最小作用量原理）。
