# 量子力学的希尔伯特空间形式

## 描述

量子态是希尔伯特空间中的向量，可观测量是自伴算子。

## 可视化

```mermaid
graph TD
    A[量子态 |ψ⟩ ∈ H] --> B[归一化 ⟨ψ|ψ⟩ = 1]

    B --> C[可观测量 A = A†]
    C --> D[谱分解 A = Σ λᵢ |eᵢ⟩⟨eᵢ|]

end

```
