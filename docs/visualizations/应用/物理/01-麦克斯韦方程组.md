# 麦克斯韦方程组 - 微分形式可视化

## 描述

麦克斯韦方程组统一描述电磁场，微分形式揭示其深刻的几何结构。

## 可视化

`
微分形式表述:

dF = 0       (法拉第定律 + 无磁单极)
d*F = J      (安培定律 + 高斯定律)

其中:
F = 电磁场2形式 = E∧dt + B
*F = Hodge对偶 = H∧dt - D

统一形式: 电磁场是时空上的联络
`

Jackson, J. D. (1999). Classical Electrodynamics.
