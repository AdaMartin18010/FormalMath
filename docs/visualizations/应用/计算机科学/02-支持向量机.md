# 支持向量机原理

## 描述

SVM通过寻找最大间隔超平面进行分类，可推广到核方法处理非线性问题。

## 可视化

```mermaid
graph TD
    A[训练数据] --> B[最大间隔超平面]
    B --> C[支持向量]
    C --> D[w·x + b = ±1]

    E[核技巧] --> F[高维特征空间]
end

```
