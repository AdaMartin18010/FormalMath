# 支持向量机原理可视化

## 概念定义

设给定训练数据集 $\{(x_i, y_i)\}_{i=1}^N \subset \mathbb{R}^d \times \{-1, +1\}$，其中 $x_i$ 为特征向量，$y_i$ 为类别标签。

若数据**线性可分**，则存在超平面 $H = \{x \in \mathbb{R}^d : w \cdot x + b = 0\}$ 使得
$$y_i(w \cdot x_i + b) > 0 \quad \text{对所有 } i。$$

**几何间隔**定义为数据点到超平面的最小有符号距离：
$$\gamma = \min_i \frac{y_i(w \cdot x_i + b)}{\|w\|}。$$

**支持向量机（SVM）** 寻找使几何间隔最大的超平面，即求解
$$\max_{w, b} \gamma \quad \text{s.t.} \quad y_i(w \cdot x_i + b) \geq \|w\| \gamma。$$
标准化 $\|w\| = 1/\gamma$ 后，等价于
$$\min_{w, b} \frac{1}{2}\|w\|^2 \quad \text{s.t.} \quad y_i(w \cdot x_i + b) \geq 1。$$

## 关键公式

**拉格朗日对偶**：引入乘子 $\alpha_i \geq 0$，SVM 的 Wolfe 对偶为
$$\max_\alpha \sum_{i=1}^N \alpha_i - \frac{1}{2}\sum_{i,j} \alpha_i \alpha_j y_i y_j (x_i \cdot x_j)$$
$$\text{s.t.} \quad \sum_i \alpha_i y_i = 0, \quad \alpha_i \geq 0。$$

**KKT 条件与稀疏性**：在最优解处，
$$\alpha_i \left(y_i(w \cdot x_i + b) - 1\right) = 0。$$
这意味着：若 $x_i$ 严格位于间隔带之外（$y_i(w \cdot x_i + b) > 1$），则 $\alpha_i = 0$。只有满足 $y_i(w \cdot x_i + b) = 1$ 的**支持向量**才对 $w = \sum_i \alpha_i y_i x_i$ 有贡献。

**核技巧**：对非线性问题，引入特征映射 $\Phi: \mathbb{R}^d \to \mathcal{H}$（高维甚至无限维希尔伯特空间），将数据映射到高维空间后再求线性超平面。若核函数 $K(x, x') = \langle \Phi(x), \Phi(x') \rangle_\mathcal{H}$ 可高效计算，则无需显式知道 $\Phi$。常见核：

- **多项式核**：$K(x, x') = (x \cdot x' + c)^p$；
- **高斯/RBF 核**：$K(x, x') = \exp(-\|x - x'\|^2 / 2\sigma^2)$；
- **Sigmoid 核**：$K(x, x') = \tanh(\kappa x \cdot x' + \theta)$。

**软间隔 SVM**：对非严格可分数据，引入松弛变量 $\xi_i \geq 0$：
$$\min_{w, b, \xi} \frac{1}{2}\|w\|^2 + C \sum_i \xi_i \quad \text{s.t.} \quad y_i(w \cdot x_i + b) \geq 1 - \xi_i。$$
参数 $C > 0$ 控制间隔宽度与分类误差之间的权衡。

## 可视化说明

**最大间隔超平面的"道路选择"图**：

- 在二维平面上画出两组点：蓝色圆点（类 $+1$）集中在左上，红色方块（类 $-1$）集中在右下。两组点之间有清晰的线性可分间隙。
- 画出三条平行直线：中间实线为决策边界 $w \cdot x + b = 0$，两侧虚线分别为 $w \cdot x + b = +1$ 和 $w \cdot x + b = -1$（间隔边界）。
- **关键可视化**：在两侧虚线上标注恰好落在其上的数据点——这些就是支持向量。用放大镜效果圈出：即使移除所有非支持向量，最优超平面也不变。这直观展示了 SVM 解的稀疏性。
- 对比一个"错误"的超平面（比如更靠近蓝色类的某条直线），用阴影标注其几何间隔更小，解释为什么最大间隔准则选择中央超平面——它具有最好的泛化能力（VC 维理论）。

**核技巧的特征空间映射图**：

- 在原始空间 $\mathbb{R}^2$ 中画出两类数据交错分布成同心圆状（内圆红、外环蓝），显然线性不可分。
- 画出映射后的三维空间 $\mathbb{R}^3$：横轴为 $x_1$，纵轴为 $x_2$，竖轴为 $z = x_1^2 + x_2^2 = r^2$（径向距离平方）。在三维空间中，红色点位于低 $z$ 处，蓝色点位于高 $z$ 处，现在线性可分。
- 画一个水平平面 $z = c$ 作为分离超平面。将其投影回原始二维空间，得到决策边界是一个圆 $x_1^2 + x_2^2 = c$。这直观展示了核方法如何将线性分类器推广到非线性决策边界。
- 对 RBF 核，画出其隐式特征空间的"局部感受野"：每个支持向量 $x_i$ 对应一个以它为中心的高斯"小山丘"，决策函数 $f(x) = \sum_i \alpha_i y_i K(x_i, x) + b$ 是这些山丘的带权叠加。

**软间隔的"容错带"图**：

- 在可分但含噪声的数据集上，画出硬间隔 SVM 的最优超平面——它可能被一个异常点（outlier）严重偏移。
- 对比软间隔 SVM：允许少量数据点穿过间隔边界甚至决策边界，用 $\xi_i$ 标记它们的"违规程度"。画出几个红色方块轻微侵入间隔带，旁边标注其 $\xi_i > 0$ 但很小。
- 用不同 $C$ 值的对比图展示：$C$ 很大时（严格惩罚错误），间隔窄，对异常点敏感；$C$ 很小时（宽松惩罚），间隔宽，可能包容更多训练误差但泛化更好。

## 直观解释

SVM 的核心哲学是**结构风险最小化**：不仅要在训练数据上表现好，更要通过最大化间隔来降低模型的复杂度，从而获得更好的泛化性能。

几何间隔的倒数 $\|w\|$ 与模型的 VC 维密切相关。VC 维衡量了模型的"表达能力"——能力越强的模型越容易过拟合。SVM 通过约束 $\|w\|$ 来控制模型复杂度，这等价于在由支持向量张成的子空间中寻找"最平坦"的分类边界。

核技巧的深刻之处在于：许多高维（甚至无限维）特征空间中内积的计算可以绕开显式映射，直接通过核函数完成。这就是著名的**核技巧（kernel trick）**。Mercer 定理给出了核函数的完整刻画：$K$ 是某个希尔伯特空间上的有效内积，当且仅当它对任意有限点集都是正定核。这一理论框架使 SVM 成为连接泛函分析、优化理论与机器学习的典范。

## 例子

1. **二维线性可分数据**：设数据为 $x_i \in \mathbb{R}^2$，$y_i = \mathrm{sgn}(2x_{i1} - x_{i2} + 0.5) + \text{noise}$。SVM 学到的 $w \approx (2, -1)^T$，$b \approx 0.5$。支持向量通常只占训练集的 $5\%$–$20\%$。

2. **XOR 问题与多项式核**：XOR 数据 $(0,0)\to -1, (0,1)\to +1, (1,0)\to +1, (1,1)\to -1$ 在 $\mathbb{R}^2$ 中线性不可分。用二次核 $K(x, x') = (x \cdot x' + 1)^2$，隐式映射到特征 $(x_1, x_2, x_1x_2, x_1^2, x_2^2, 1)$。此时线性超平面 $w_5 z_5 + w_3 z_3 + b = 0$（$z_5 = x_1^2, z_3 = x_1x_2$ 等）对应于原始空间中的二次曲线，可以完美分离 XOR。

3. **高维文本分类**：在文本分类中，特征维度 $d$ 可达 $10^4$–$10^6$（词袋模型）。SVM 通过线性核即可达到极好效果，因为高维空间中数据往往线性可分（覆盖定理）。同时 SVM 的稀疏性使得预测时只需存储少量支持向量，大大降低了存储和计算开销。

4. **SVM 回归（SVR）**：支持向量机可推广到回归问题。给定数据 $\{(x_i, y_i)\}$，SVR 寻找函数 $f(x) = w \cdot x + b$ 使得 $|y_i - f(x_i)| \leq \varepsilon$ 对大多数点成立，同时最小化 $\|w\|^2$。落在 $\varepsilon$-带之外的点成为支持向量。这与分类 SVM 的几何结构完全类似。
