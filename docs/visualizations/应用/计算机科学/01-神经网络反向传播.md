# 神经网络反向传播可视化

## 描述

本可视化展示深度学习中反向传播算法的数学原理，展示如何通过链式法则高效计算梯度并更新网络参数。

## 数学概念

反向传播利用链式法则计算损失函数对各层参数的梯度，核心是多变量微积分中的梯度下降。

## 可视化代码

```mermaid
graph TB
    subgraph 反向传播流程
    A[输入 x] --> B[前向传播]
    B --> C[计算预测 ŷ]
    C --> D[计算损失 L = L(y, ŷ)]

    D --> E[反向传播]
    E --> F[计算 ∂L/∂W⁽ˡ⁾]
    E --> G[计算 ∂L/∂b⁽ˡ⁾]

    F --> H[参数更新<br/>W := W - η∇W]
    G --> H

    H --> I[重复直到收敛]
    end

    style A fill:#e3f2fd
    style D fill:#fff3e0
    style E fill:#e8f5e9
    style H fill:#fce4ec

```

```

反向传播的数学原理
═══════════════════════════════════════════════════════════════

网络结构:
───────────────────────────────────────────────────────────────
x → [W₁,b₁] → z₁ → σ → a₁ → [W₂,b₂] → z₂ → σ → a₂ → ... → ŷ

前向传播:
z⁽ˡ⁾ = W⁽ˡ⁾a⁽ˡ⁻¹⁾ + b⁽ˡ⁾
a⁽ˡ⁾ = σ(z⁽ˡ⁾)

损失函数 (例如MSE):
L = 1/2 Σ(y - ŷ)²

反向传播 (链式法则):
───────────────────────────────────────────────────────────────
输出层误差:
δ⁽ᴸ⁾ = ∂L/∂z⁽ᴸ⁾ = (a⁽ᴸ⁾ - y) ⊙ σ'(z⁽ᴸ⁾)

隐藏层误差 (反向传播):
δ⁽ˡ⁾ = ((W⁽ˡ⁺¹⁾)ᵀ δ⁽ˡ⁺¹⁾) ⊙ σ'(z⁽ˡ⁾)

梯度计算:
∂L/∂W⁽ˡ⁾ = δ⁽ˡ⁾ (a⁽ˡ⁻¹⁾)ᵀ
∂L/∂b⁽ˡ⁾ = δ⁽ˡ⁾

参数更新:
W⁽ˡ⁾ := W⁽ˡ⁾ - η ∂L/∂W⁽ˡ⁾
b⁽ˡ⁾ := b⁽ˡ⁾ - η ∂L/∂b⁽ˡ⁾

链式法则的几何意义:
┌─────────────────────────────────────────────────────────────┐
│  损失的变化 = 各层变化的乘积                                 │
│                                                             │
│  ∂L/∂W⁽ˡ⁾ = ∂L/∂a⁽ᴸ⁾ · ∂a⁽ᴸ⁾/∂z⁽ᴸ⁾ · ... · ∂z⁽ˡ⁾/∂W⁽ˡ⁾   │
│                                                             │
│  就像一条链条，误差从输出端逐层传回输入端                  │
└─────────────────────────────────────────────────────────────┘

应用:
• 图像识别 (CNN)
• 自然语言处理 (RNN/Transformer)
• 生成模型 (GAN, VAE)
• 强化学习

```
