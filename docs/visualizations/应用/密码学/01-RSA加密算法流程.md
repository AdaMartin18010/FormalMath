# RSA加密算法流程可视化

## 描述

本可视化展示RSA公钥加密算法的完整流程，包括密钥生成、加密和解密过程，以及背后的数论原理。

## 数学概念

RSA基于大整数分解的困难性。核心数论事实：若n=pq，e与φ(n)互素，则存在d使得ed ≡ 1 (mod φ(n))，且对任意m < n，有(m^e)^d ≡ m (mod n)。

## 可视化代码

### RSA完整流程

```mermaid
graph TB
    subgraph RSA算法流程
    A[选择大素数 p, q] --> B[计算 n = pq, φ(n) = (p-1)(q-1)]
    B --> C[选择 e, gcd(e, φ(n)) = 1]
    C --> D[计算 d, ed ≡ 1 mod φ(n)]

    D --> E[公钥 (n, e)]
    D --> F[私钥 (n, d)]

    E --> G[加密 c = m^e mod n]
    F --> H[解密 m = c^d mod n]

    G --> I[密文 c]
    I --> H
    end
```

### ASCII详细流程

```

RSA加密算法详细流程
═══════════════════════════════════════════════════════════════

密钥生成 (Bob执行):
───────────────────────────────────────────────────────────────

步骤1: 选择素数
┌─────────────────────────────────────────────────────────────┐
│  选择两个大素数 p 和 q                                      │
│  例如: p = 61, q = 53 (实际使用数百位大数)                  │
└─────────────────────────────────────────────────────────────┘

步骤2: 计算模数
┌─────────────────────────────────────────────────────────────┐
│  n = p * q = 61 * 53 = 3233                                 │
│  φ(n) = (p-1)(q-1) = 60 * 52 = 3120                         │
│  (欧拉函数)                                                 │
└─────────────────────────────────────────────────────────────┘

步骤3: 选择公钥指数
┌─────────────────────────────────────────────────────────────┐
│  选择 e 使得:                                               │
│  • 1 < e < φ(n)                                             │
│  • gcd(e, φ(n)) = 1                                         │
│  例如: e = 17                                               │
└─────────────────────────────────────────────────────────────┘

步骤4: 计算私钥指数
┌─────────────────────────────────────────────────────────────┐
│  计算 d 使得:                                               │
│  e * d ≡ 1 (mod φ(n))                                       │
│  即: 17 * d ≡ 1 (mod 3120)                                  │
│  解得: d = 2753                                             │
│  (使用扩展欧几里得算法)                                     │
└─────────────────────────────────────────────────────────────┘

最终密钥:
═══════════════════════════════════════════════════════════════

┌─────────────────────────┬─────────────────────────┐
│      公钥 (公开)        │      私钥 (保密)        │
├─────────────────────────┼─────────────────────────┤
│      (n, e)             │      (n, d)             │
│   (3233, 17)            │   (3233, 2753)          │
└─────────────────────────┴─────────────────────────┘

加密解密过程:
───────────────────────────────────────────────────────────────

Alice要发送消息 m = 123 给 Bob:

加密 (使用Bob的公钥):
┌─────────────────────────────────────────────────────────────┐
│  c = m^e mod n                                              │
│  c = 123^17 mod 3233                                        │
│  c = 855  (密文)                                            │
└─────────────────────────────────────────────────────────────┘

解密 (Bob使用私钥):
┌─────────────────────────────────────────────────────────────┐
│  m = c^d mod n                                              │
│  m = 855^2753 mod 3233                                      │
│  m = 123  (原始消息)                                        │
└─────────────────────────────────────────────────────────────┘

数学原理:
═══════════════════════════════════════════════════════════════

欧拉定理: 若 gcd(m, n) = 1, 则 m^φ(n) ≡ 1 (mod n)

因此: (m^e)^d = m^(ed) = m^(kφ(n)+1) = (m^φ(n))^k * m ≡ m (mod n)

安全性基础:
───────────────────────────────────────────────────────────────
• 已知 (n, e) 和密文 c，求 m 困难
• 除非知道 d，而计算 d 需要知道 φ(n) = (p-1)(q-1)
• 这需要分解 n = pq，对大数来说计算不可行

实际应用:
───────────────────────────────────────────────────────────────
• HTTPS/SSL 协议
• 数字签名
• 安全电子邮件 (PGP/GPG)
• 软件签名验证

```
