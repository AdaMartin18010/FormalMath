# 应用数学与综合练习题

本文档包含应用数学习题（30道）和跨学科综合练习题（30道），涵盖基础、进阶和挑战三个难度级别。

---

## 第一部分：应用数学习题（30道）

### 一、基础练习题（10道）

#### 1.1 运筹学基础

**题1** 某工厂生产A、B两种产品，每件A产品利润40元，每件B产品利润60元。生产A需要2小时机器时间和1小时人工时间，生产B需要3小时机器时间和3小时人工时间。每天可用机器时间为60小时，人工时间为40小时。建立线性规划模型求最大利润。

<details>
<summary>提示</summary>

设 $x_1, x_2$ 分别为A、B的产量，目标：$\max Z = 40x_1 + 60x_2$

约束：$2x_1 + 3x_2 \leq 60$，$x_1 + 3x_2 \leq 40$，$x_1, x_2 \geq 0$

</details>

**题2** 用图解法求解线性规划问题：
$$\max Z = 3x_1 + 2x_2$$
$$\text{s.t. } x_1 + x_2 \leq 4, \quad x_1 - x_2 \leq 2, \quad x_1, x_2 \geq 0$$

**题3** 求解运输问题：3个供应地（供应量分别为20, 30, 50）和4个需求地（需求量分别为15, 25, 35, 25），用最小元素法求初始可行解。

**题4** 某项目各工序的最早开始时间和最迟开始时间如下表，确定关键路径。

| 工序 | 最早开始 | 最迟开始 |
|------|----------|----------|
| A    | 0        | 0        |
| B    | 0        | 2        |
| C    | 3        | 3        |
| D    | 5        | 5        |

#### 1.2 控制论基础

**题5** 给定系统 $\dot{x} = ax + bu$，其中 $a = -1$，$b = 1$。设计状态反馈控制律 $u = kx$ 使闭环系统极点位于 $-2$。

<details>
<summary>提示</summary>

闭环系统：$\dot{x} = (a + bk)x = (-1 + k)x$，令 $-1 + k = -2$

</details>

**题6** 离散系统 $x_{k+1} = 0.8x_k + u_k$，性能指标 $J = \sum_{k=0}^{\infty}(x_k^2 + u_k^2)$。求最优控制律。

**题7** 判断系统 $\dot{x}_1 = x_2$，$\dot{x}_2 = -2x_1 - 3x_2 + u$ 的能控性。

**题8** 求传递函数 $G(s) = \frac{1}{s(s+1)}$ 的状态空间实现。

#### 1.3 金融数学基础

**题9** 某股票当前价格100元，无风险利率5%，一年后可能上涨到120元或下跌到80元。求该股票的一年期欧式看涨期权（执行价105元）的价格。

<details>
<summary>提示</summary>

使用二叉树模型，风险中性概率 $q = \frac{e^{rT} - d}{u - d}$

</details>

**题10** 计算本金10000元，年利率6%，按年复利5年后的本利和。

### 二、进阶练习题（12道）

#### 2.1 优化理论

**题11** 用单纯形法求解：
$$\max Z = 2x_1 + 3x_2 + x_3$$
$$\text{s.t. } x_1 + x_2 + x_3 \leq 10, \quad 2x_1 + x_2 \leq 8, \quad x_1, x_2, x_3 \geq 0$$

**题12** 求解整数规划问题（分支定界法）：
$$\max Z = 5x_1 + 4x_2$$
$$\text{s.t. } x_1 + x_2 \leq 5, \quad 10x_1 + 6x_2 \leq 45, \quad x_1, x_2 \geq 0 \text{ 且为整数}$$

**题13** 用动态规划求解资源分配问题：将5台设备分配给3个工厂，各工厂获得不同设备数时的利润如下表，求最优分配方案。

| 设备数 | 工厂1 | 工厂2 | 工厂3 |
|--------|-------|-------|-------|
| 0      | 0     | 0     | 0     |
| 1      | 3     | 5     | 4     |
| 2      | 7     | 10    | 6     |
| 3      | 9     | 11    | 11    |

**题14** 用拉格朗日乘子法求解约束优化：
$$\min f(x,y) = x^2 + y^2 \quad \text{s.t. } x + y = 1$$

#### 2.2 现代控制理论

**题15** 给定系统 $\dot{x} = Ax + Bu$，其中 $A = \begin{bmatrix} 0 & 1 \\ -2 & -3 \end{bmatrix}$，$B = \begin{bmatrix} 0 \\ 1 \end{bmatrix}$。设计LQR控制器使性能指标 $J = \int_0^{\infty}(x^TQx + u^TRu)dt$ 最小，其中 $Q = I$，$R = 1$。

<details>
<summary>提示</summary>

求解Riccati方程：$A^TP + PA - PBR^{-1}B^TP + Q = 0$

</details>

**题16** 系统 $\dot{x} = \begin{bmatrix} 1 & 1 \\ 0 & 1 \end{bmatrix}x + \begin{bmatrix} 0 \\ 1 \end{bmatrix}u$，$y = \begin{bmatrix} 1 & 0 \end{bmatrix}x$。设计状态观测器使观测误差极点位于 $-3, -4$。

**题17** 证明：线性系统 $(A, B)$ 能控的充要条件是能控性矩阵 $\mathcal{C} = [B, AB, A^2B, ..., A^{n-1}B]$ 满秩。

**题18** 考虑MIMO系统 $G(s) = \begin{bmatrix} \frac{1}{s+1} & \frac{1}{s+2} \\ \frac{2}{s+1} & \frac{3}{s+3} \end{bmatrix}$，分析其极点、零点和相对阶。

#### 2.3 金融工程

**题19** 用Black-Scholes公式计算欧式看涨期权价格。已知：$S_0 = 100$，$K = 100$，$r = 0.05$，$\sigma = 0.2$，$T = 1$。

<details>
<summary>提示</summary>

$d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2)T}{\sigma\sqrt{T}}$，$C = S_0\Phi(d_1) - Ke^{-rT}\Phi(d_2)$

</details>

**题20** 构建由两只股票组成的投资组合，股票A期望收益12%，标准差15%；股票B期望收益8%，标准差10%。相关系数$\rho = 0.3$。求最小方差组合。

**题21** 用蒙特卡洛方法模拟几何布朗运动 $dS_t = rS_t dt + \sigma S_t dW_t$，计算亚式看涨期权的价格。

**题22** 证明在无套利条件下，欧式看涨-看跌期权平价关系：$C - P = S_0 - Ke^{-rT}$。

### 三、挑战练习题（8道）

#### 3.1 高级优化

**题23** 证明KKT条件在最优性中的必要性：对于凸优化问题，满足Slater条件的可行点是全局最优当且仅当存在拉格朗日乘子使KKT条件成立。

<details>
<summary>提示</summary>

利用凸分析中的分离超平面定理和对偶理论。

</details>

**题24** 用内点法求解二次规划问题，推导原始-对偶算法的迭代格式，并分析其收敛性。

**题25** 考虑半定规划(SDP)问题：证明SDP可以表示为线性矩阵不等式约束下的优化问题，讨论SDP在组合优化和鲁棒控制中的应用。

#### 3.2 最优控制与博弈

**题26** 求解最优控制问题：最小化 $J = \int_0^T (x^2 + u^2)dt$ 约束于 $\dot{x} = ax + u$，$x(0) = x_0$。用Pontryagin极大值原理推导最优控制律。

**题27** 考虑双人零和微分博弈，状态方程为 $\dot{x} = f(x, u, v)$，支付泛函为 $J = \int_0^T L(x, u, v)dt + \Phi(x(T))$。推导HJI方程并讨论其解的存在性。

**题28** 设计H∞鲁棒控制器：对于系统 $\dot{x} = Ax + B_1w + B_2u$，$z = C_1x + D_{12}u$，求控制律使闭环系统从$w$到$z$的H∞范数小于$\gamma$。

#### 3.3 金融数学高级主题

**题29** 在随机波动率模型(Heston模型)下：$dS_t = rS_t dt + \sqrt{v_t}S_t dW_t^1$，$dv_t = \kappa(\theta - v_t)dt + \sigma\sqrt{v_t}dW_t^2$，推导特征函数并讨论期权定价的傅里叶变换方法。

**题30** 考虑利率期限结构模型(Vasicek模型)：$dr_t = a(b - r_t)dt + \sigma dW_t$。推导零息债券价格的闭式解和利率衍生品的定价公式。

## 第二部分：综合练习题（30道）

### 一、跨分支综合题（15道）

#### 代数-分析综合

**综合1** 证明：对于任意$n \times n$实对称矩阵$A$，存在正交矩阵$Q$使得$Q^TAQ = \Lambda$（对角矩阵）。利用此结论求解二次型$Q(x) = x^TAx$在约束$x^Tx = 1$下的极值。

<details>
<summary>涉及知识点</summary>

线性代数（谱定理）、多元微积分（拉格朗日乘子）、实分析

</details>

**综合2** 设$f: \mathbb{R}^n \to \mathbb{R}$是二阶连续可微函数，$x^*$是其临界点。证明：若Hessian矩阵$\nabla^2 f(x^*)$正定，则$x^*$是严格局部极小值点。

**综合3** 考虑线性常微分方程组$\dot{x} = Ax$，其中$A$是常数矩阵。证明：解可以表示为$x(t) = e^{At}x_0$，并讨论$e^{At}$的计算方法（Jordan标准形、Cayley-Hamilton定理）。

#### 概率-分析综合

**综合4** 设$\{X_n\}$是独立同分布的随机变量，$E[X_1] = 0$，$E[X_1^2] = 1$。定义$S_n = \sum_{k=1}^n X_k$。证明：$\frac{S_n}{\sqrt{n}}$的分布弱收敛于标准正态分布，并讨论此收敛的速度（Berry-Esseen定理）。

<details>
<summary>涉及知识点</summary>

概率论（中心极限定理）、特征函数、傅里叶分析

</details>

**综合5** 布朗运动$B_t$的几乎必然不可微性的证明思路是什么？这与函数论中Weierstrass函数的分析性质有何联系？

**综合6** 考虑随机微分方程$dX_t = b(X_t)dt + \sigma(X_t)dB_t$。讨论其强解和弱解的存在性条件，以及解的马氏性和生成元。

#### 几何-拓扑-分析综合

**综合7** 证明Gauss-Bonnet定理：对于紧致的二维黎曼流形$M$，有$\int_M K dA = 2\pi \chi(M)$，其中$K$是高斯曲率，$\chi(M)$是欧拉示性数。

<details>
<summary>涉及知识点</summary>

微分几何（曲率）、拓扑学（欧拉示性数）、分析（Stokes定理）

</details>

**综合8** 设$M$是紧致的$n$维黎曼流形，$\Delta$是Laplace-Beltrami算子。证明：$\Delta$的特征值构成离散的非负数列，且特征函数构成$L^2(M)$的完备正交基。

**综合9** 用Morse理论证明：若$f: M \to \mathbb{R}$是紧流形$M$上的Morse函数，则$M$同伦等价于一个CW复形，其中$k$维胞腔的个数等于$f$的指标为$k$的临界点的个数。

#### 数论-代数综合

**综合10** 设$K/\mathbb{Q}$是有限Galois扩张，$\mathcal{O}_K$是$K$的整数环。讨论素数$p$在$\mathcal{O}_K$中的分解规律，以及Frobenius元素的作用。

<details>
<summary>涉及知识点</summary>

代数数论（Dedekind整环）、Galois理论、类域论

</details>

**综合11** 证明Dirichlet算术级数定理：若$a, d$互素，则算术级数$a, a+d, a+2d, ...$中包含无穷多个素数。

#### 多领域综合

**综合12** 在量子力学中，粒子的波函数$\psi(t,x)$满足Schrödinger方程$i\hbar \frac{\partial \psi}{\partial t} = H\psi$。讨论：
(a) Hamilton算子$H$的自伴性与其谱的关系
(b) 时间演化算子的幺正性
(c) 与经典力学的对应关系（Ehrenfest定理）

<details>
<summary>涉及知识点</summary>

泛函分析（自伴算子）、偏微分方程、李群李代数、辛几何

</details>

**综合13** 信息论中，信道容量$C = \max_{p(x)} I(X;Y)$。证明Shannon信道编码定理：对于任意$R < C$，存在编码方案使得传输速率可达$R$且错误概率任意小。

**综合14** 在机器学习中，支持向量机(SVM)求解优化问题：$\min_{w,b} \frac{1}{2}\|w\|^2$ s.t. $y_i(w^Tx_i + b) \geq 1$。推导其对偶问题并讨论核方法的数学基础（再生核Hilbert空间）。

**综合15** 考虑Navier-Stokes方程：$\frac{\partial u}{\partial t} + (u \cdot \nabla)u = -\nabla p + \nu \Delta u$，$\nabla \cdot u = 0$。讨论：
(a) 解的存在唯一性（三维情形仍是开放问题）
(b) 能量不等式和解的正则性
(c) 湍流现象的数学描述

### 二、实际应用场景题（15道）

#### 工程应用

**应用1** 【信号处理】设计数字滤波器将模拟信号转换为数字信号。给定截止频率$\omega_c$，设计Butterworth低通滤波器，并分析其频率响应特性。

<details>
<summary>涉及知识点</summary>

复分析、傅里叶变换、数值分析

</details>

**应用2** 【图像处理】图像去噪问题可以建模为变分问题：$\min_u \int_{\Omega}(u - f)^2 dx + \lambda \int_{\Omega}|\nabla u|dx$，其中$f$是噪声图像。讨论该模型的数学性质和数值求解方法。

**应用3** 【结构力学】悬臂梁的挠度$w(x)$满足Euler-Bernoulli方程：$EI w^{(4)} = q(x)$。用有限元方法求解并分析收敛性。

#### 自然科学应用

**应用4** 【流体力学】用复势方法求解理想流体绕圆柱的流动问题。证明达朗贝尔悖论并讨论其物理意义。

<details>
<summary>涉及知识点</summary>

复变函数（共形映射）、偏微分方程、向量分析

</details>

**应用5** 【电磁学】从Maxwell方程组出发，推导波动方程并讨论电磁波在介质中的传播特性（色散关系、折射率等）。

**应用6** 【热传导】一维热方程$\frac{\partial u}{\partial t} = \alpha \frac{\partial^2 u}{\partial x^2}$的初边值问题。用分离变量法求解并分析解的渐近行为。

**应用7** 【量子化学】Hartree-Fock方法中的自洽场计算。讨论Fock算子的构造、基组选择以及计算复杂度的数学分析。

#### 经济与金融应用

**应用8** 【宏观经济学】Solow增长模型：$\dot{k} = sf(k) - (n + \delta)k$。分析平衡增长路径的存在性和稳定性，讨论黄金律储蓄率。

<details>
<summary>涉及知识点</summary>

常微分方程、动力系统、优化理论

</details>

**应用9** 【博弈论】拍卖理论中的最优机制设计。证明Myerson最优拍卖定理，并讨论其在实际拍卖设计中的应用。

**应用10** 【风险管理】计算投资组合在99%置信水平下的风险价值(VaR)。讨论参数方法和历史模拟法的优缺点，以及尾部风险建模（极值理论）。

#### 数据科学与人工智能应用

**应用11** 【推荐系统】矩阵分解在协同过滤中的应用。讨论ALS（交替最小二乘）算法的收敛性分析和正则化的作用。

<details>
<summary>涉及知识点</summary>

线性代数（SVD）、优化理论、统计学习理论

</details>

**应用12** 【自然语言处理】词嵌入模型Word2Vec的数学原理。讨论Skip-gram模型的目标函数、负采样技巧以及向量空间的线性结构。

**应用13** 【深度学习】神经网络的通用逼近定理：证明具有一个隐藏层的前馈神经网络可以任意精度逼近紧集上的连续函数。讨论激活函数的作用。

#### 生物医学应用

**应用14** 【流行病学】SIR传染病模型：$\frac{dS}{dt} = -\beta SI$，$\frac{dI}{dt} = \beta SI - \gamma I$，$\frac{dR}{dt} = \gamma I$。分析基本再生数$R_0$和疫情爆发的条件。

<details>
<summary>涉及知识点</summary>

动力系统、稳定性理论、随机过程

</details>

**应用15** 【医学成像】CT成像中的Radon变换和滤波反投影算法。讨论采样定理对图像重建质量的影响。

## 参考答案要点

### 应用数学答案要点

**题1** 最优解：$x_1 = 20$，$x_2 = \frac{20}{3} \approx 6.67$，最大利润 $Z = 1200$

**题2** 最优解在 $(3, 1)$，$Z = 11$

**题5** $k = -1$，控制律 $u = -x$

**题6** 解Riccati方程：$P = \frac{1 + \sqrt{1.64}}{1.6} \approx 1.45$，$u_k = -0.725x_k$

**题9** $q = \frac{e^{0.05} - 0.8}{1.2 - 0.8} \approx 0.628$，期权价格 $C = e^{-0.05}(0.628 \times 15 + 0.372 \times 0) \approx 8.96$

**题11** 最优解：$x_1 = 0$，$x_2 = 8$，$x_3 = 2$，$Z = 26$

**题19** $d_1 = 0.35$，$d_2 = 0.15$，$C \approx 10.45$

### 综合题答案要点

**综合1** 极值为$A$的最大和最小特征值

**综合4** Berry-Esseen界：$|F_n(x) - \Phi(x)| \leq \frac{C\rho}{\sigma^3\sqrt{n}}$

**综合7** 高斯曲率积分等于$2\pi$乘以拓扑不变量

**综合14** 对偶问题涉及核函数$K(x_i, x_j) = \phi(x_i)^T\phi(x_j)$

## 学习路径建议

### 基础知识巩固
1. 线性代数：矩阵运算、特征值、正交变换
2. 多元微积分：梯度、Hessian、约束优化
3. 概率统计：分布理论、估计与检验

### 进阶能力提升
1. 泛函分析：Hilbert空间、算子理论
2. 微分方程：ODE/PDE理论、数值方法
3. 优化理论：凸分析、对偶理论

### 应用能力培养
1. 建模训练：将实际问题转化为数学问题
2. 数值实现：编程求解、算法实现
3. 交叉学习：了解应用领域的基本知识
