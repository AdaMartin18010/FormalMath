# 变分法基础

## Euler-Lagrange方程

### 推导
寻找使泛函$J[y] = \int_a^b F(x, y, y') dx$取极值的函数$y(x)$。

考虑变分$y(x) + \epsilon \eta(x)$，$\eta(a) = \eta(b) = 0$。

$$\frac{dJ}{d\epsilon}\bigg|_{\epsilon=0} = 0$$

得Euler-Lagrange方程：
$$\frac{\partial F}{\partial y} - \frac{d}{dx}\frac{\partial F}{\partial y'} = 0$$

### 例子：最速降线
$F = \sqrt{\frac{1+y'^2}{2gy}}$

解为摆线（旋轮线）。
