## 1. 引言

应用数学是连接纯粹数学与现实世界的桥梁，通过数学建模、数值计算和理论分析，解决自然科学、工程技术和社会科学中的实际问题。本模块系统涵盖**数值分析**、**优化理论**、**微分方程建模**和**数学物理**等核心领域。

## 2. 数学建模基础

### 2.1 建模方法论

**定义 2.1** (数学模型): 数学模型是对现实系统的抽象数学描述，通常表示为：
$$\mathcal{M}: \text{输入} \mapsto \text{输出}$$

**建模流程**:
```
实际问题 → 简化假设 → 数学表述 → 求解分析 → 验证修正
```

**定理 2.1** (量纲分析定理 / Buckingham π定理): 设物理问题涉及 $n$ 个变量，基本量纲数为 $k$，则存在 $n-k$ 个独立的无量纲量。

---

### 2.2 模型分类

| 模型类型 | 数学结构 | 典型应用 |
|----------|----------|----------|
| **确定性模型** | 常微分/偏微分方程 | 物理定律、工程系统 |
| **随机模型** | 随机过程、概率分布 | 金融风险、生物统计 |
| **优化模型** | 目标函数 + 约束条件 | 资源配置、路径规划 |
| **离散模型** | 差分方程、图论 | 网络分析、组合优化 |
| **混合模型** | 上述组合 | 复杂系统建模 |

## 3. 数值分析 (Numerical Analysis)

### 3.1 数值线性代数

**定义 3.1** (矩阵条件数): 方阵 $A$ 的条件数为：
$$\kappa(A) = \|A\| \cdot \|A^{-1}\|$$
条件数越大，求解线性方程组 $Ax = b$ 的数值稳定性越差。

**定理 3.1** (LU分解存在性): 若方阵 $A$ 的所有顺序主子式非零，则 $A$ 存在唯一的LU分解 $A = LU$，其中 $L$ 是单位下三角矩阵，$U$ 是上三角矩阵。

**定义 3.2** (迭代法收敛): 线性迭代 $x^{(k+1)} = Gx^{(k)} + c$ 收敛当且仅当谱半径 $\rho(G) < 1$。

---

### 3.2 数值微积分

**定理 3.2** (Newton-Cotes公式误差): $n$ 点Newton-Cotes求积公式的代数精度为 $n+1$（$n$ 偶数）或 $n$（$n$ 奇数）。

**定义 3.3** (Richardson外推): 设数值方法满足
$$I_h = I + Ch^p + O(h^{p+1})$$
则外推公式
$$I_{h}^* = \frac{2^p I_{h/2} - I_h}{2^p - 1} = I + O(h^{p+1})$$
将精度提高一阶。

---

### 3.3 微分方程数值解

**定义 3.4** (ODE数值方法阶数): 单步法 $y_{n+1} = y_n + h\Phi(t_n, y_n; h)$ 的阶数为 $p$，若局部截断误差满足：
$$y(t_{n+1}) - y(t_n) - h\Phi(t_n, y(t_n); h) = O(h^{p+1})$$

**定理 3.3** (Runge-Kutta方法阶条件): $s$ 级Runge-Kutta方法可达的最高阶数为：
- $s = 1, 2, 3, 4$: 阶数 $p = s$
- $s = 5, 6, 7$: 阶数 $p = s-1$
- $s \geq 8$: 阶数 $p \leq s-2$

**定义 3.5** (稳定性区域): 方法应用于测试方程 $y' = \lambda y$ 时，$|R(z)| \leq 1$ 的区域称为稳定性区域，其中 $z = h\lambda$。

## 4. 优化理论 (Optimization Theory)

### 4.1 无约束优化

**定义 4.1** (凸函数): 函数 $f: \mathbb{R}^n \to \mathbb{R}$ 是凸函数，若对 $\forall x, y \in \mathbb{R}^n$, $\lambda \in [0, 1]$：
$$f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y)$$

**定理 4.1** (凸优化全局最优): 若 $f$ 是凸函数，则任何局部极小点都是全局极小点。

**定理 4.2** (梯度下降收敛): 设 $f$ 是 $L$-光滑且 $\mu$-强凸的函数，步长 $\alpha \leq 1/L$，则梯度下降满足：
$$f(x_k) - f(x^*) \leq (1 - \alpha\mu)^k (f(x_0) - f(x^*))$$

---

### 4.2 约束优化

**定义 4.2** (拉格朗日函数): 约束优化问题
$$\min f(x) \quad \text{s.t.} \quad g_i(x) \leq 0, h_j(x) = 0$$
的拉格朗日函数为：
$$\mathcal{L}(x, \lambda, \mu) = f(x) + \sum_i \lambda_i g_i(x) + \sum_j \mu_j h_j(x)$$

**定理 4.3** (KKT条件): 在约束规范条件下，局部最优解 $x^*$ 满足：
1. **平稳性**: $\nabla_x \mathcal{L}(x^*, \lambda^*, \mu^*) = 0$
2. **原始可行性**: $g_i(x^*) \leq 0$, $h_j(x^*) = 0$
3. **对偶可行性**: $\lambda_i^* \geq 0$
4. **互补松弛**: $\lambda_i^* g_i(x^*) = 0$

---

### 4.3 线性规划

**定义 4.3** (线性规划标准型):
$$\min c^T x \quad \text{s.t.} \quad Ax = b, \quad x \geq 0$$

**定理 4.4** (单纯形法基本定理): 若线性规划存在最优解，则存在顶点（基本可行解）为最优解。

**定理 4.5** (强对偶定理): 若原问题或对偶问题有有限最优值，则两者最优值相等。

## 5. 微分方程建模

### 5.1 常微分方程模型

**定义 5.1** (初值问题): 
$$\frac{dy}{dt} = f(t, y), \quad y(t_0) = y_0$$

**定理 5.1** (Picard-Lindelöf存在唯一性): 若 $f$ 在矩形区域上关于 $y$ 满足Lipschitz条件，则初值问题存在唯一局部解。

**定义 5.2** (稳定性): 平衡点 $y^*$（满足 $f(t, y^*) = 0$）是：
- **Lyapunov稳定**: 解对初值扰动保持有界
- **渐近稳定**: Lyapunov稳定且扰动解收敛到 $y^*$

**定理 5.2** (线性化稳定性): 对于自治系统 $y' = f(y)$，若 $Df(y^*)$ 的所有特征值实部为负，则 $y^*$ 渐近稳定。

---

### 5.2 偏微分方程模型

**定义 5.3** (二阶线性PDE分类): 方程 $au_{xx} + 2bu_{xy} + cu_{yy} + \cdots = 0$ 的类型由判别式 $\Delta = b^2 - ac$ 决定：
- $\Delta < 0$: 椭圆型（如Laplace方程 $u_{xx} + u_{yy} = 0$）
- $\Delta = 0$: 抛物型（如热方程 $u_t = u_{xx}$）
- $\Delta > 0$: 双曲型（如波动方程 $u_{tt} = u_{xx}$）

**定理 5.3** (分离变量法): 对于线性齐次PDE和齐次边界条件，可设 $u(x,t) = X(x)T(t)$，将PDE分解为ODE。

## 8. 应用案例

**示例 8.1** (种群动力学): Logistic模型
$$\frac{dN}{dt} = rN\left(1 - \frac{N}{K}\right)$$
其中 $r$ 为内禀增长率，$K$ 为环境承载力。解为：
$$N(t) = \frac{K}{1 + \left(\frac{K}{N_0} - 1\right)e^{-rt}}$$

**示例 8.2** (热传导): 一维热方程初边值问题
$$u_t = \alpha u_{xx}, \quad u(0,t) = u(L,t) = 0, \quad u(x,0) = f(x)$$
分离变量得解：
$$u(x,t) = \sum_{n=1}^{\infty} b_n \sin\left(\frac{n\pi x}{L}\right) \exp\left(-\alpha\frac{n^2\pi^2}{L^2}t\right)$$

**示例 8.3** (投资组合优化): Markowitz均值-方差模型
$$\min_w \frac{1}{2}w^T\Sigma w \quad \text{s.t.} \quad \mu^T w \geq R_{target}, \quad \mathbf{1}^T w = 1$$
其中 $\Sigma$ 是协方差矩阵，$\mu$ 是预期收益率向量。
