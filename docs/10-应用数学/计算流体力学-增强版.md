## 1. 控制方程

### 1.1 Navier-Stokes方程

**不可压缩Navier-Stokes方程**：

$$\frac{\partial \mathbf{u}}{\partial t} + (\mathbf{u} \cdot \nabla)\mathbf{u} = -\frac{1}{\rho}\nabla p + \nu\nabla^2\mathbf{u}$$
$$\nabla \cdot \mathbf{u} = 0$$

其中：
- $\mathbf{u}$：速度场
- $p$：压力
- $\rho$：密度
- $\nu$：运动粘度

**无量纲化**：

引入特征长度 $L$ 和特征速度 $U$，定义雷诺数：

$$Re = \frac{UL}{\nu}$$

### 1.2 Euler方程

**可压缩Euler方程**：

$$\frac{\partial \mathbf{U}}{\partial t} + \nabla \cdot \mathbf{F} = 0$$

守恒变量：

$$\mathbf{U} = \begin{pmatrix} \rho \\ \rho u \\ \rho v \\ \rho w \\ \rho E \end{pmatrix}$$

### 1.3 边界条件

**壁面边界**：
- 无滑移：$\mathbf{u} = \mathbf{0}$
- 滑移：$\mathbf{u} \cdot \mathbf{n} = 0$

**入口边界**：
- 速度入口：指定速度剖面
- 压力入口：指定总压

**出口边界**：
- 压力出口：$p = p_{out}$
- 对流边界：$\frac{\partial \mathbf{u}}{\partial t} + U_c\frac{\partial \mathbf{u}}{\partial n} = 0$

## 2. 有限体积法

### 2.1 基本思想

对控制体 $\Omega$ 积分控制方程：

$$\int_\Omega \frac{\partial \mathbf{U}}{\partial t} dV + \oint_{\partial\Omega} \mathbf{F} \cdot \mathbf{n} dS = 0$$

**离散形式**：

$$\frac{d\mathbf{U}_i}{dt} = -\frac{1}{V_i}\sum_{f} \mathbf{F}_f \cdot \mathbf{n}_f A_f$$

### 2.2 通量计算

**中心格式**（二阶精度）：

$$\mathbf{F}_f = \frac{1}{2}(\mathbf{F}_L + \mathbf{F}_R)$$

**迎风格式**：

```python
def roe_flux(UL, UR, gamma=1.4):
    """Roe通量计算"""
    # 计算Roe平均
    rhoL, uL, pL = UL
    rhoR, uR, pR = UR
    
    HL = gamma/(gamma-1) * pL/rhoL + 0.5*uL**2
    HR = gamma/(gamma-1) * pR/rhoR + 0.5*uR**2
    
    sqrt_rhoL = np.sqrt(rhoL)
    sqrt_rhoR = np.sqrt(rhoR)
    
    rho_roe = sqrt_rhoL * sqrt_rhoR
    u_roe = (sqrt_rhoL*uL + sqrt_rhoR*uR) / (sqrt_rhoL + sqrt_rhoR)
    H_roe = (sqrt_rhoL*HL + sqrt_rhoR*HR) / (sqrt_rhoL + sqrt_rhoR)
    a_roe = np.sqrt((gamma-1)*(H_roe - 0.5*u_roe**2))
    
    # 特征值
    lam1 = abs(u_roe)
    lam2 = abs(u_roe + a_roe)
    lam3 = abs(u_roe - a_roe)
    
    # 计算通量
    FL = np.array([rhoL*uL, rhoL*uL**2 + pL, uL*(gamma/(gamma-1)*pL + 0.5*rhoL*uL**2)])
    FR = np.array([rhoR*uR, rhoR*uR**2 + pR, uR*(gamma/(gamma-1)*pR + 0.5*rhoR*uR**2)])
    
    # 耗散项
    diss = 0.5 * (lam2 - lam1) * (pR - pL) / a_roe**2
    
    return 0.5 * (FL + FR) - 0.5 * diss
```

### 2.3 时间离散

**显式格式**（RK4）：

```python
def rk4_step(U, dt, compute_rhs):
    """四阶Runge-Kutta"""
    k1 = compute_rhs(U)
    k2 = compute_rhs(U + 0.5*dt*k1)
    k3 = compute_rhs(U + 0.5*dt*k2)
    k4 = compute_rhs(U + dt*k3)
    return U + dt/6 * (k1 + 2*k2 + 2*k3 + k4)
```

**隐式格式**：

$$(I + \theta dt J)\Delta U = -dt R(U^n)$$

## 3. 湍流模型

### 3.1 RANS模型

**$k-\varepsilon$模型**：

湍动能 $k$ 和耗散率 $\varepsilon$ 的输运方程：

$$\frac{\partial k}{\partial t} + u_j\frac{\partial k}{\partial x_j} = P_k - \varepsilon + \frac{\partial}{\partial x_j}\left(\frac{\nu_t}{\sigma_k}\frac{\partial k}{\partial x_j}\right)$$

$$\frac{\partial \varepsilon}{\partial t} + u_j\frac{\partial \varepsilon}{\partial x_j} = C_{\varepsilon1}\frac{\varepsilon}{k}P_k - C_{\varepsilon2}\frac{\varepsilon^2}{k} + \frac{\partial}{\partial x_j}\left(\frac{\nu_t}{\sigma_\varepsilon}\frac{\partial \varepsilon}{\partial x_j}\right)$$

涡粘系数：

$$\nu_t = C_\mu \frac{k^2}{\varepsilon}$$

### 3.2 LES模型

**Smagorinsky模型**：

$$\nu_{sgs} = (C_s\Delta)^2 |\bar{S}|$$

其中：

$$|\bar{S}| = \sqrt{2\bar{S}_{ij}\bar{S}_{ij}}, \quad \bar{S}_{ij} = \frac{1}{2}\left(\frac{\partial \bar{u}_i}{\partial x_j} + \frac{\partial \bar{u}_j}{\partial x_i}\right)$$

## 4. 可压缩流动

**MUSCL格式**：

```python
def muscl_reconstruct(u, limiter='minmod'):
    """MUSCL重构"""
    # 计算梯度
    du = u[2:] - u[1:-1]
    du_l = u[1:-1] - u[:-2]
    
    if limiter == 'minmod':
        phi = np.where(np.abs(du) < np.abs(du_l), 
                       np.where(du*du_l > 0, du, 0),
                       np.where(du*du_l > 0, du_l, 0))
    
    # 重构界面值
    uL = u[1:-1] + 0.5 * phi
    uR = u[2:] - 0.5 * phi[1:]
    
    return uL, uR
```

## 5. 计算示例

### 5.1 二维层流圆柱绕流

**问题设置**：
- 雷诺数：$Re = 100$
- 计算域：$[-10, 20] \times [-10, 10]$
- 圆柱半径：$R = 0.5$

```python
def cylinder_flow_simulation(Re=100, nx=200, ny=150):
    """二维圆柱绕流模拟"""
    # 网格生成（O型网格）
    r = np.linspace(0.5, 10, nx)
    theta = np.linspace(0, 2*np.pi, ny)
    R, Theta = np.meshgrid(r, theta)
    X = R * np.cos(Theta)
    Y = R * np.sin(Theta)
    
    # 初始化
    u = np.ones_like(X)  # x方向速度
    v = np.zeros_like(X)  # y方向速度
    p = np.zeros_like(X)  # 压力
    
    dt = 0.001
    nu = 1.0 / Re
    
    # 时间推进
    for step in range(10000):
        # 计算中间速度（忽略压力梯度）
        u_star = u + dt * (-u*np.gradient(u)[1] - v*np.gradient(u)[0] + 
                           nu*(np.gradient(np.gradient(u)[1])[1] + 
                               np.gradient(np.gradient(u)[0])[0]))
        v_star = v + dt * (-u*np.gradient(v)[1] - v*np.gradient(v)[0] + 
                           nu*(np.gradient(np.gradient(v)[1])[1] + 
                               np.gradient(np.gradient(v)[0])[0]))
        
        # 求解压力Poisson方程
        rhs = (np.gradient(u_star)[1] + np.gradient(v_star)[0]) / dt
        # ... 求解Poisson方程
        
        # 修正速度
        u = u_star - dt * np.gradient(p)[1]
        v = v_star - dt * np.gradient(p)[0]
        
        # 应用边界条件
        u[0, :] = 0  # 圆柱表面
        v[0, :] = 0
        u[-1, :] = 1  # 远场
        v[-1, :] = 0
    
    return X, Y, u, v, p
```

**结果分析**：
- 斯特劳哈尔数：$St = fD/U \approx 0.165$（与文献一致）
- 阻力系数：$C_d \approx 1.35$
- 升力系数振幅：$C_l \approx 0.33$

### 5.2 顶盖驱动方腔流

**问题设置**：
- 雷诺数：$Re = 1000$
- 计算域：单位正方形
- 顶盖速度：$u = 1$

```python
def lid_driven_cavity(Re=1000, N=129, max_iter=50000):
    """顶盖驱动方腔流"""
    # 交错网格
    h = 1.0 / (N - 1)
    u = np.zeros((N, N-1))  # x方向速度（位于垂直面）
    v = np.zeros((N-1, N))  # y方向速度（位于水平面）
    p = np.zeros((N-1, N-1))  # 压力（位于单元中心）
    
    dt = 0.001
    nu = 1.0 / Re
    
    for iter in range(max_iter):
        # 求解u动量方程
        for i in range(1, N-1):
            for j in range(N-1):
                ue = 0.5 * (u[i,j] + u[i+1,j]) if i < N-2 else 0
                uw = 0.5 * (u[i,j] + u[i-1,j]) if i > 0 else 0
                vn = 0.5 * (v[i,j] + v[i,j+1]) if j < N-2 else 0
                vs = 0.5 * (v[i-1,j] + v[i-1,j+1]) if i > 0 and j < N-2 else 0
                
                # 离散动量方程
                conv = (ue*u[i+1,j] - uw*u[i-1,j] + vn*u[i,j+1] - vs*u[i,j-1]) / (2*h)
                diff = nu * (u[i+1,j] - 2*u[i,j] + u[i-1,j] + u[i,j+1] - 2*u[i,j] + u[i,j-1]) / h**2
                u[i,j] = u[i,j] + dt * (-conv + diff - (p[i,j] - p[i-1,j]) / h)
        
        # 顶盖速度
        u[-1, :] = 1.0
        
        # 求解v动量方程（类似）
        # ...
        
        # 求解压力Poisson方程
        for _ in range(50):  # SOR迭代
            for i in range(N-1):
                for j in range(N-1):
                    div = ((u[i+1,j] - u[i,j]) + (v[i,j+1] - v[i,j])) / h / dt
                    p[i,j] = 0.25 * (p[(i+1)%(N-1),j] + p[i-1,j] + 
                                     p[i,(j+1)%(N-1)] + p[i,j-1] - h**2 * div)
        
        # 压力修正
        u[1:-1,:] -= dt * (p[1:,:] - p[:-1,:]) / h
        v[:,1:-1] -= dt * (p[:,1:] - p[:,:-1]) / h
    
    return u, v, p
```

**结果验证**：

与Ghia等人(1982)基准解对比：
- 主涡中心位置：$(0.53, 0.56)$ vs $(0.53, 0.56)$
- 速度剖面吻合良好

### 5.3 激管问题

**问题设置**：
- Sod激管
- 初始条件：
  - 左侧：$\rho_L = 1$, $p_L = 1$, $u_L = 0$
  - 右侧：$\rho_R = 0.125$, $p_R = 0.1$, $u_R = 0$

```python
def sod_shock_tube(nx=200, gamma=1.4, t_end=0.2):
    """Sod激管问题"""
    x = np.linspace(0, 1, nx)
    dx = x[1] - x[0]
    
    # 初始条件
    rho = np.where(x < 0.5, 1.0, 0.125)
    u = np.zeros(nx)
    p = np.where(x < 0.5, 1.0, 0.1)
    
    # 计算守恒变量
    E = p / (gamma - 1) + 0.5 * rho * u**2
    U = np.array([rho, rho*u, E])
    
    dt = 0.0001
    n_steps = int(t_end / dt)
    
    for step in range(n_steps):
        # 计算通量（使用Roe格式）
        F = np.zeros((3, nx-1))
        for i in range(nx-1):
            UL = U[:, i]
            UR = U[:, i+1]
            F[:, i] = roe_flux_1d(UL, UR, gamma)
        
        # 更新
        U[:, 1:-1] -= dt/dx * (F[:, 1:] - F[:, :-1])
    
    # 恢复原始变量
    rho = U[0, :]
    u = U[1, :] / rho
    p = (gamma - 1) * (U[2, :] - 0.5 * rho * u**2)
    
    return x, rho, u, p

def roe_flux_1d(UL, UR, gamma):
    """一维Roe通量"""
    rhoL, momL, EL = UL
    rhoR, momR, ER = UR
    uL, uR = momL/rhoL, momR/rhoR
    pL = (gamma-1)*(EL - 0.5*rhoL*uL**2)
    pR = (gamma-1)*(ER - 0.5*rhoR*uR**2)
    HL = (EL + pL) / rhoL
    HR = (ER + pR) / rhoR
    
    # Roe平均
    sL, sR = np.sqrt(rhoL), np.sqrt(rhoR)
    rho = sL * sR
    u = (sL*uL + sR*uR) / (sL + sR)
    H = (sL*HL + sR*HR) / (sL + sR)
    a = np.sqrt((gamma-1)*(H - 0.5*u**2))
    
    # 计算通量
    FL = np.array([rhoL*uL, rhoL*uL**2 + pL, uL*(EL + pL)])
    FR = np.array([rhoR*uR, rhoR*uR**2 + pR, uR*(ER + pR)])
    
    return 0.5 * (FL + FR)
```
