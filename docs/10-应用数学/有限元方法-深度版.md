## 1. 数学基础

### 1.1 Sobolev空间

**定义 1.1** (Sobolev空间 $H^k$)

$$H^k(\Omega) = \{u \in L^2(\Omega) : D^\alpha u \in L^2(\Omega), \forall |\alpha| \leq k\}$$

范数定义为：

$$\|u\|_{H^k} = \left(\sum_{|\alpha| \leq k} \|D^\alpha u\|_{L^2}^2\right)^{1/2}$$

**Sobolev嵌入定理**：

对于 $\Omega \subset \mathbb{R}^n$，有：

$$H^k(\Omega) \hookrightarrow C^m(\bar{\Omega}), \quad k > m + \frac{n}{2}$$

### 1.2 变分原理

**模型问题**：Poisson方程

$$-\Delta u = f \quad \text{in } \Omega$$
$$u = 0 \quad \text{on } \partial\Omega$$

**弱形式**：

寻找 $u \in H_0^1(\Omega)$ 使得：

$$a(u, v) = F(v), \quad \forall v \in H_0^1(\Omega)$$

其中：

$$a(u, v) = \int_\Omega \nabla u \cdot \nabla v \, dx$$
$$F(v) = \int_\Omega fv \, dx$$

### 1.3 Lax-Milgram定理

**定理 1.1** (Lax-Milgram)

设 $V$ 是Hilbert空间，$a: V \times V \to \mathbb{R}$ 是有界且强制的双线性形式，$F \in V'$。则存在唯一解 $u \in V$ 满足：

$$a(u, v) = F(v), \quad \forall v \in V$$

且满足稳定性估计：

$$\|u\|_V \leq \frac{1}{\alpha}\|F\|_{V'}$$

## 2. 有限元离散化

### 2.1 网格生成

**Delaunay三角化**：

```python
def delaunay_triangulation(points):
    """Delaunay三角化"""
    from scipy.spatial import Delaunay
    tri = Delaunay(points)
    return tri.simplices
```

**质量度量**：

单元质量因子：

$$Q = \frac{4\sqrt{3}A}{h_1^2 + h_2^2 + h_3^2}$$

其中 $A$ 是面积，$h_i$ 是边长。

### 2.2 单元构造

**参考单元映射**：

$$x = \sum_{i=1}^n x_i \phi_i(\xi)$$

其中 $\xi$ 是参考单元坐标。

**Jacobian矩阵**：

$$J = \frac{\partial x}{\partial \xi}$$

### 2.3 形函数

**线性三角形元**：

在参考单元上：

$$\phi_1 = 1 - \xi - \eta$$
$$\phi_2 = \xi$$
$$\phi_3 = \eta$$

**二次三角形元**：

增加边中点节点，共6个节点。

## 3. 高阶有限元

### 3.1 Lagrange元

**定义 3.1** (P_k元)

多项式空间 $P_k$ 的维数：

$$\dim(P_k) = \frac{(k+1)(k+2)}{2} \quad (2D)$$

**节点分布**：

均匀分布或Gauss-Lobatto点分布。

### 3.2 谱元

**Gauss-Lobatto-Legendre点**：

用于多项式插值，具有最优的Lebesgue常数。

**谱精度**：

对于光滑解，误差呈指数衰减：

$$\|u - u_h\| \leq Ce^{-\sigma p}$$

### 3.3 hp自适应

**策略**：

- h细化：细分单元（提高分辨率）
- p提升：增加多项式阶数（提高精度）

**决策准则**：

基于光滑性指示器选择h或p细化。

## 4. 误差分析

### 4.1 先验误差估计

**定理 4.1** (Céa引理)

对于协调有限元：

$$\|u - u_h\|_V \leq \frac{M}{\alpha} \inf_{v_h \in V_h} \|u - v_h\|_V$$

**收敛阶**：

对于 $P_k$ 元：

$$\|u - u_h\|_{H^1} \leq Ch^k|u|_{H^{k+1}}$$
$$\|u - u_h\|_{L^2} \leq Ch^{k+1}|u|_{H^{k+1}}$$

### 4.2 后验误差估计

**残差型估计器**：

$$\eta_T^2 = h_T^2\|f + \Delta u_h\|_{L^2(T)}^2 + \frac{1}{2}h_T\|[\partial_n u_h]\|_{L^2(\partial T)}^2$$

**自适应算法**：

```python
def adaptive_fem_solve(mesh, f, tol):
    """自适应有限元求解"""
    while True:
        # 求解
        u_h = solve_fem(mesh, f)
        
        # 计算后验误差估计
        eta = compute_error_estimator(mesh, u_h, f)
        
        if np.max(eta) < tol:
            break
        
        # 标记和细化
        marked = mark_elements(eta, strategy='Dorfler')
        mesh = refine_mesh(mesh, marked)
    
    return u_h, mesh
```

## 5. 实际应用案例

### 5.1 航空结构分析

**应用场景**：飞机机翼结构分析

**数学模型**：线弹性方程

$$-\nabla \cdot \sigma = f \quad \text{in } \Omega$$
$$\sigma = \mathbb{C} : \varepsilon(u)$$
$$\varepsilon(u) = \frac{1}{2}(\nabla u + \nabla u^T)$$

**实际案例**：某客机机翼

- 网格：四面体单元，200万个节点
- 单元类型：P2二次元
- 自由度：600万
- 求解时间：2小时
- 应力集中分析结果与实验误差 < 5%

**关键结果**：


### 5.2 油藏模拟

**应用场景**：油田开发模拟

**数学模型**：多孔介质流动

$$\frac{\partial(\phi\rho)}{\partial t} - \nabla \cdot \left(\frac{k\rho}{\mu}\nabla p\right) = q$$

**实际案例**：某海上油田

- 网格：结构化网格，500万单元
- 非线性求解：Newton迭代
- 时间步进：隐式格式
- 模拟周期：30年

### 5.3 电磁场仿真

**应用场景**：天线设计

**数学模型**：Maxwell方程

$$\nabla \times \mu^{-1}\nabla \times E - k^2\varepsilon E = J$$

**实际案例**：5G基站天线

- 使用Nédélec棱边元
- 网格：100万单元
