## 一、引言：NISQ时代的应用

### 1.1 NISQ定义与挑战

**NISQ**（Noisy Intermediate-Scale Quantum）：

- **Noisy**: 噪声限制（T1 ~ 100μs，门错误率 ~ 0.1%）
- **Intermediate-Scale**: 50-1000+ 量子比特
- **Quantum**: 量子计算设备

**NISQ挑战**：

1. **有限的相干时间**：电路深度受限
2. **有限的连接性**：需要额外的SWAP门
3. **有限的纠错**：无完全容错保护

### 1.2 变分量子算法（VQA）框架

**核心思想**：利用**经典优化器** + **量子处理器** 的混合计算：

$$\theta^* = \arg\min_\theta \langle\psi(\theta)|H|\psi(\theta)\rangle$$

**VQA优势**：

- 电路深度浅（适合NISQ）
- 容错性较好
- 可扩展性强

### 1.3 应用领域概览

| 应用领域 | 代表算法 | 量子优势潜力 | 成熟度 |
|---------|---------|------------|-------|
| 量子化学 | VQE, ADAPT-VQE | 高 | 实验验证 |
| 组合优化 | QAOA, Quantum Annealing | 中 | 原型应用 |
| 机器学习 | QML, QNN | 待验证 | 研究阶段 |
| 金融建模 | VQE, QAOA, QML | 中 | 原型应用 |
| 材料科学 | VQE, Quantum Phase Estimation | 高 | 实验验证 |
| 密码分析 | Shor算法 | 高（远期） | 理论阶段 |

## 二、变分量子特征值求解器（VQE）

### 2.1 问题定义

**目标**：找到哈密顿量 $H$ 的基态能量和波函数

$$E_0 = \min_{|\psi\rangle} \frac{\langle\psi|H|\psi\rangle}{\langle\psi|\psi\rangle}$$

**应用**：

- 分子基态能量计算
- 化学反应路径模拟
- 材料性质预测

### 2.2 分子哈密顿量的二次量子化

**电子结构问题**：
$$H = \sum_{pq} h_{pq} a_p^\dagger a_q + \frac{1}{2}\sum_{pqrs} h_{pqrs} a_p^\dagger a_q^\dagger a_r a_s$$

**Jordan-Wigner变换**：将费米子算子映射到Pauli算子

$$a_j^\dagger = \frac{1}{2} \left(\prod_{k<j} Z_k\right) (X_j - iY_j)$$

**Pauli字符串表示**：
$$H = \sum_k c_k P_k, \quad P_k \in \{I, X, Y, Z\}^{\otimes n}$$

### 2.3 VQE算法流程

**算法步骤**：

1. **定义试探波函数**（Ansatz）：
   $$|\psi(\theta)\rangle = U(\theta)|0\rangle^{\otimes n}$$

2. **测量能量期望值**：
   $$E(\theta) = \langle\psi(\theta)|H|\psi(\theta)\rangle = \sum_k c_k \langle P_k \rangle$$

3. **经典优化**：
   $$\theta^{(t+1)} = \theta^{(t)} - \eta \nabla_\theta E(\theta)$$

4. **重复直到收敛**

### 2.4 Ansatz设计

**硬件高效Ansatz（HEA）**：
$$U(\theta) = \prod_{l=1}^L \left(\prod_{j=1}^n R_Y(\theta_{j,l})\right) \left(\prod_{j=1}^{n-1} \text{CNOT}_{j,j+1}\right)$$

**UCCSD（单元耦合簇）**：

- 化学启发Ansatz
- 基于激发算子
- 保持粒子数守恒

**ADAPT-VQE**：

- 自适应构建Ansatz
- 每次迭代添加最重要的算子
- 克服贫瘠高原问题

### 2.5 测量问题与分组策略

**测量挑战**：需要测量 $\sim N^4$ 个Pauli字符串（对于N个轨道）。

**Qubit-wise Commuting（QWC）分组**：

- 如果两个Pauli字符串在所有位置都对易或相同
- 可以同时测量

**示例**：
$$Z_1 Z_2 \text{ 和 } Z_1 Z_3 \text{ 可以同时测量}$$

**General Commuting（GC）分组**：

- 使用Bell基测量
- 更高效的测量策略

### 2.6 化学应用实例

**H₂分子（STO-3G基组）**：

- 2个电子，2个轨道 → 2量子比特
- 基态能量：-1.137 Hartree（精确解）
- VQE结果：误差 < 0.001 Hartree

**LiH分子**：

- 4个电子，6个轨道 → 12量子比特（需要约化）
- VQE在IBM量子计算机上实现

**N₂分子**：

- 10个电子，8个轨道 → 20量子比特
- 展示量子计算的化学精度

## 三、量子近似优化算法（QAOA）

### 3.1 组合优化问题

**问题形式**：
$$C(x) = \sum_{\alpha} C_\alpha(x), \quad x \in \{0, 1\}^n$$

**经典难度**：大多数组合优化问题是NP难的。

**代表问题**：

- Max-Cut
- Max-SAT
- 旅行商问题（TSP）
- 图着色

### 3.2 QAOA框架

**目标哈密顿量**（Problem Hamiltonian）：
$$H_C = \sum_{\alpha} C_\alpha(x) \otimes |x\rangle\langle x|$$

**混合哈密顿量**（Mixer Hamiltonian）：
$$H_B = \sum_{j=1}^n X_j$$

**QAOA电路**（p层）：
$$|\psi(\gamma, \beta)\rangle = e^{-i\beta_p H_B} e^{-i\gamma_p H_C} \cdots e^{-i\beta_1 H_B} e^{-i\gamma_1 H_C} |+\rangle^{\otimes n}$$

### 3.3 Max-Cut问题详解

**问题定义**：给定图 $G = (V, E)$，找到划分 $(S, V\setminus S)$ 最大化割边数。

**目标函数**：
$$C = \sum_{(i,j) \in E} \frac{1 - x_i x_j}{2} = \sum_{(i,j) \in E} \frac{1 - Z_i Z_j}{2}$$

**QAOA电路门**：
$$e^{-i\gamma Z_i Z_j} = \text{CNOT}_{i,j} \cdot R_Z(2\gamma)_j \cdot \text{CNOT}_{i,j}$$

**性能**：

- $p=1$ 时，在某些图类上达到 $0.692$ 近似比
- $p \to \infty$ 时，理论上可以达到最优解

### 3.4 参数优化策略

**插值策略**：

- 从 $p$ 层参数插值到 $p+1$ 层
- 加速收敛

**全局优化**：

- 使用模拟退火、遗传算法等
- 避免局部最优

**机器学习辅助**：

- 用神经网络预测好的初始参数
- 加速优化过程

### 3.5 量子退火对比

**量子退火**（D-Wave）：

- 物理实现绝热量子计算
- 实际哈密顿量：
  $$H(t) = A(t) \sum_i \sigma_i^x + B(t) \left(\sum_i h_i \sigma_i^z + \sum_{i<j} J_{ij} \sigma_i^z \sigma_j^z\right)$$

**QAOA vs 量子退火**：

| 特性 | QAOA | 量子退火 |
|-----|------|---------|
| 实现 | 通用门电路 | 专用硬件 |
| 可编程性 | 高 | 中等 |
| 相干时间要求 | 高 | 低（绝热过程） |
| 连接性 | 可任意 | 硬件限制 |
| 目前规模 | 数十量子比特 | 5000+量子比特 |

## 四、量子机器学习基础

### 4.1 量子特征映射

**思想**：用量子电路将经典数据映射到高维Hilbert空间。

**Z-Feature Map**：
$$|\phi(x)\rangle = \bigotimes_{j=1}^n R_Z(x_j)|0\rangle$$

**ZZ-Feature Map**（考虑数据相关性）：
$$|\phi(x)\rangle = \left(\prod_{j<k} R_{ZZ}(2(\pi - x_j)(\pi - x_k))\right) \left(\bigotimes_{j=1}^n R_Z(x_j)|0\rangle\right)$$

### 4.2 量子核方法

**量子核函数**：
$$K(x, x') = |\langle\phi(x)|\phi(x')\rangle|^2$$

**优势**：

- 可能实现经典难以计算的核函数
- 指数级大的特征空间

**挑战**：

- 核估计需要大量量子电路运行
- "贫瘠高原"问题

### 4.3 参数化量子电路（PQC）作为神经网络

**变分量子分类器**：
$$f(x; \theta) = \langle\phi(x)|U^\dagger(\theta) M U(\theta)|\phi(x)\rangle$$

**训练**：
$$\min_\theta \sum_i (f(x_i; \theta) - y_i)^2$$

**表达能力**：

- PQC可以表示复杂的非线性函数
- 但存在**贫瘠高原**问题

### 4.4 量子神经网络训练

**梯度计算**：
$$\frac{\partial \langle O \rangle}{\partial \theta_j} = \frac{1}{2}\left(\langle O \rangle_{\theta_j + \frac{\pi}{2}} - \langle O \rangle_{\theta_j - \frac{\pi}{2}}\right)$$

**参数平移规则**：
$$\frac{\partial}{\partial \theta} \langle\psi|e^{i\theta G} O e^{-i\theta G}|\psi\rangle = \langle\psi|e^{i(\theta+\frac{\pi}{2}) G} O e^{-i(\theta+\frac{\pi}{2}) G}|\psi\rangle - \langle\psi|e^{i(\theta-\frac{\pi}{2}) G} O e^{-i(\theta-\frac{\pi}{2}) G}|\psi\rangle$$

### 4.5 量子优势的理论分析

**定理（数据中的量子优势）**：对于某些特定分布的数据，量子模型可以用比任何经典模型指数级更少的参数达到相同的精度。

**实现挑战**：

- 需要精心构造的数据集
- 实际数据集不一定满足条件
- 训练难度高

## 五、量子模拟应用

### 5.1 量子多体系统

**Hubbard模型**：
$$H = -t \sum_{\langle i,j\rangle, \sigma} (c_{i\sigma}^\dagger c_{j\sigma} + h.c.) + U \sum_i n_{i\uparrow} n_{i\downarrow}$$

**应用**：

- 高温超导
- 量子磁性
- Mott绝缘体

### 5.2 量子场论模拟

**格点QCD**：

- 模拟强相互作用
- 研究夸克禁闭
- 计算强子质量

**标量场论**：

- 模拟相变
- 研究临界现象

### 5.3 量子动力学

**实时演化**：
$$|\psi(t)\rangle = e^{-iHt}|\psi(0)\rangle$$

**Trotter-Suzuki分解**：
$$e^{-iHt} \approx \left(e^{-iH_1 t/n} e^{-iH_2 t/n} \cdots\right)^n$$

**挑战**：

- sign问题（费米子系统）
- 实时演化需要浅电路

## 六、量子金融应用

### 6.1 期权定价

**问题**：计算欧式看涨期权价格
$$C = e^{-rT} \mathbb{E}^Q[\max(S_T - K, 0)]$$

**量子振幅估计**：

- 提供二次加速

$$\min_w \, w^T \Sigma w - \lambda \mu^T w$$
$$\text{s.t.} \, \mathbf{1}^T w = 1, \, w \geq 0$$

**QAOA应用**：

- 将连续问题离散化
- 使用QAOA求解组合优化版本

**风险价值（VaR）**：
$$\text{VaR}_\alpha = \inf\{x : P(L > x) \leq 1-\alpha\}$$

**量子计算**：

- 量子振幅估计加速VaR计算
- 信用风险评估

### 7.1 VQE实现：H₂分子基态能量

```python
"""
量子计算应用案例 - Qiskit实现
VQE, QAOA, 量子机器学习
"""

import numpy as np
import matplotlib.pyplot as plt
from qiskit import QuantumCircuit, QuantumRegister, ClassicalRegister
from qiskit_aer import AerSimulator
from qiskit.circuit.library import TwoLocal, EfficientSU2
from qiskit_algorithms import VQE, QAOA
from qiskit_algorithms.optimizers import COBYLA, SLSQP, L_BFGS_B
from qiskit.quantum_info import Statevector, SparsePauliOp
from qiskit.primitives import Estimator, Sampler
import warnings
warnings.filterwarnings('ignore')


def vqe_h2_molecule():
    """
    VQE求解H₂分子基态能量
    对应理论：变分原理、UCCSD Ansatz
    """
    print("=" * 60)
    print("VQE求解H₂分子基态能量")
    print("=" * 60)

    # H₂分子在STO-3G基组下的哈密顿量（简化2量子比特版本）
    # 参考: O'Malley et al., "Scalable Quantum Simulation of Molecular Energies", 2016

    # 使用简化的H₂哈密顿量（Jordan-Wigner变换后）
    # 在平衡键长（0.735 Å）附近

    print("\n1. 定义H₂分子哈密顿量")
    print("   分子: H-H")
    print("   键长: 0.735 Å")
    print("   基组: STO-3G")
    print("   量子比特: 2")

    # 简化的H₂哈密顿量（截断到活跃空间）
    # H = g0*I⊗I + g1*Z⊗I + g2*I⊗Z + g3*Z⊗Z + g4*Y⊗Y + g5*X⊗X

    # 系数（来自pyscf或openfermion计算）
    g0 = -0.469594
    g1 =  0.242742
    g2 =  0.242742
    g3 =  0.177713
    g4 =  0.044750
    g5 =  0.044750

    # 构建Pauli算子列表
    pauli_list = [
        ('II', g0), ('ZI', g1), ('IZ', g2), ('ZZ', g3),
        ('YY', g4), ('XX', g5)
    ]

    hamiltonian = SparsePauliOp.from_list(pauli_list)
    print(f"\n   哈密顿量项数: {len(pauli_list)}")
    print(f"   Pauli字符串: {[p[0] for p in pauli_list]}")

    # 精确对角化（对比）
    exact_energy = -1.137283  # Hartree
    print(f"\n2. 精确基态能量（FCI）: {exact_energy:.6f} Hartree")

    # 定义Ansatz
    print("\n3. 定义UCCSD-inspired Ansatz")

    def create_ansatz(theta):
        """创建2量子比特UCCSD-inspired电路"""
        qc = QuantumCircuit(2)

        # Hartree-Fock初态 |01⟩ (或 |10⟩ 取决于排序)

        qc.x(0)

        # 激发算子: exp(-iθ X⊗Y)
        # 使用Jordan-Wigner变换后的单激发
        qc.ry(np.pi/2, 0)
        qc.rx(-np.pi/2, 1)
        qc.cx(0, 1)
        qc.rz(theta, 1)
        qc.cx(0, 1)
        qc.ry(-np.pi/2, 0)
        qc.rx(np.pi/2, 1)

        return qc

    # 手动VQE优化（教学目的）
    print("\n4. 变分优化过程")

    def expectation_value(theta):
        """计算能量期望值"""
        qc = create_ansatz(theta)
        sv = Statevector.from_instruction(qc)

        energy = 0
        for pauli, coef in pauli_list:
            # 计算 <ψ|P|ψ>

            op = SparsePauliOp.from_list([(pauli, 1.0)])
            exp = sv.expectation_value(op).real
            energy += coef * exp

        return energy

    # 网格搜索
    theta_range = np.linspace(-np.pi, np.pi, 100)
    energies = [expectation_value(t) for t in theta_range]

    min_idx = np.argmin(energies)
    optimal_theta = theta_range[min_idx]
    vqe_energy = energies[min_idx]

    print(f"   最优参数 θ = {optimal_theta:.4f}")
    print(f"   VQE能量 = {vqe_energy:.6f} Hartree")
    print(f"   绝对误差 = {abs(vqe_energy - exact_energy):.6f} Hartree")
    print(f"   相对误差 = {abs(vqe_energy - exact_energy)/abs(exact_energy)*100:.4f}%")

    # 绘制能量曲面
    fig, ax = plt.subplots(figsize=(10, 6))
    ax.plot(theta_range, energies, 'b-', linewidth=2, label='VQE Energy')
    ax.axhline(y=exact_energy, color='r', linestyle='--', label=f'Exact Energy = {exact_energy:.4f}')
    ax.axvline(x=optimal_theta, color='g', linestyle=':', label=f'Optimal θ = {optimal_theta:.2f}')
    ax.set_xlabel('Parameter θ', fontsize=12)
    ax.set_ylabel('Energy (Hartree)', fontsize=12)
    ax.set_title('VQE Energy Landscape for H₂ Molecule', fontsize=14)
    ax.legend()
    ax.grid(True, alpha=0.3)
    plt.tight_layout()
    plt.savefig('vqe_h2_landscape.png', dpi=150, bbox_inches='tight')
    print("\n5. 能量曲面图已保存到 vqe_h2_landscape.png")

    return vqe_energy, exact_energy
```

## 7.2 QAOA实现：Max-Cut问题

```python
def qaoa_maxcut():
    """
    QAOA求解Max-Cut问题
    对应理论：组合优化、近似算法
    """
    print("\n" + "=" * 60)
    print("QAOA求解Max-Cut问题")
    print("=" * 60)

    # 定义4节点图
    # 0 -- 1
    # |    |

    # 2 -- 3

    print("\n1. 定义图结构")
    print("   节点: 0, 1, 2, 3")
    print("   边: (0,1), (1,3), (3,2), (2,0), (1,2)")

    edges = [(0, 1), (1, 3), (3, 2), (2, 0), (1, 2)]
    n_nodes = 4
    n_edges = len(edges)

    # Max-Cut目标函数
    print("\n2. Max-Cut目标函数")
    print("   C = Σ (1 - Z_i Z_j)/2")
    print("   最大化割边数")

    # 精确解（穷举）
    print("\n3. 精确解（穷举）:")
    max_cut = 0
    best_partitions = []

    for partition in range(2**n_nodes):
        cut = 0
        s = [(partition >> i) & 1 for i in range(n_nodes)]
        for i, j in edges:
            if s[i] != s[j]:
                cut += 1

        if cut > max_cut:
            max_cut = cut
            best_partitions = [s]
        elif cut == max_cut:
            best_partitions.append(s)

    print(f"   最大割值: {max_cut}")
    print(f"   最优划分:")
    for s in best_partitions:
        set0 = [i for i in range(n_nodes) if s[i] == 0]
        set1 = [i for i in range(n_nodes) if s[i] == 1]
        print(f"     S = {set0}, V\\S = {set1}")

    # QAOA电路 (p=1)
    print("\n4. QAOA电路 (p=1)")

    def create_qaoa_circuit(gamma, beta):
        """创建QAOA电路"""
        qc = QuantumCircuit(n_nodes)

        # 初始态: |+>^⊗n

        qc.h(range(n_nodes))

        # 问题哈密顿量演化 e^(-iγH_C)
        for i, j in edges:
            qc.cx(i, j)
            qc.rz(2 * gamma, j)
            qc.cx(i, j)

        # 混合哈密顿量演化 e^(-iβH_B)
        qc.rx(2 * beta, range(n_nodes))

        return qc

    def compute_cut_value(gamma, beta):
        """计算割的期望值"""
        qc = create_qaoa_circuit(gamma, beta)
        sv = Statevector.from_instruction(qc)

        # 采样计算期望值
        probabilities = np.abs(sv.data)**2

        expectation = 0
        for state_idx, prob in enumerate(probabilities):
            s = [(state_idx >> i) & 1 for i in range(n_nodes)]
            cut = sum(1 for i, j in edges if s[i] != s[j])
            expectation += prob * cut

        return expectation

    # 优化参数
    print("\n5. 参数优化")

    # 网格搜索
    gamma_range = np.linspace(0, 2*np.pi, 50)
    beta_range = np.linspace(0, np.pi, 50)

    best_expectation = 0
    best_params = None

    for gamma in gamma_range:
        for beta in beta_range:
            exp = compute_cut_value(gamma, beta)
            if exp > best_expectation:
                best_expectation = exp
                best_params = (gamma, beta)

    print(f"   最优参数: γ = {best_params[0]:.4f}, β = {best_params[1]:.4f}")
    print(f"   期望割值: {best_expectation:.4f}")
    print(f"   近似比: {best_expectation / max_cut:.4f}")

    # 可视化能量曲面
    gamma_grid, beta_grid = np.meshgrid(gamma_range, beta_range)
    expectation_grid = np.zeros_like(gamma_grid)

    for i in range(len(beta_range)):
        for j in range(len(gamma_range)):
            expectation_grid[i, j] = compute_cut_value(gamma_grid[i, j], beta_grid[i, j])

    fig, ax = plt.subplots(figsize=(10, 8))
    contour = ax.contourf(gamma_grid, beta_grid, expectation_grid, levels=20, cmap='viridis')
    ax.scatter([best_params[0]], [best_params[1]], color='red', s=100, marker='*',
              label=f'Optimal (γ={best_params[0]:.2f}, β={best_params[1]:.2f})')
    ax.set_xlabel('γ (gamma)', fontsize=12)
    ax.set_ylabel('β (beta)', fontsize=12)
    ax.set_title(f'QAOA Max-Cut Landscape (p=1)\nBest Expectation = {best_expectation:.3f}', fontsize=14)
    plt.colorbar(contour, ax=ax, label='Expected Cut Value')
    ax.legend()
    plt.tight_layout()
    plt.savefig('qaoa_maxcut_landscape.png', dpi=150, bbox_inches='tight')
    print("\n6. QAOA能量曲面图已保存到 qaoa_maxcut_landscape.png")

    return best_expectation, max_cut
```

```python
def quantum_kernel_method():
    """
    量子核方法演示
    对应理论：量子特征映射、核技巧
    """
    print("\n" + "=" * 60)
    print("量子核方法")
    print("=" * 60)

    print("\n1. 量子特征映射")
    print("   将经典数据 x ∈ ℝⁿ 映射到量子态 |φ(x)⟩")

    # ZZ-Feature Map
    def create_feature_map(x, n_qubits):
        """创建ZZ特征映射"""
        qc = QuantumCircuit(n_qubits)

        # 第一层: 单量子比特旋转
        for j in range(n_qubits):
            qc.ry(x[j % len(x)], j)

        # 第二层: 纠缠门
        for j in range(n_qubits - 1):
            qc.cx(j, j+1)
            qc.rz(x[j % len(x)] * x[(j+1) % len(x)], j+1)
            qc.cx(j, j+1)

        return qc

    # 数据集
    print("\n2. 创建示例数据集")
    np.random.seed(42)

    # 两类数据
    class_0 = np.random.randn(5, 2) + np.array([-1, -1])
    class_1 = np.random.randn(5, 2) + np.array([1, 1])

    X = np.vstack([class_0, class_1])
    y = np.array([0]*5 + [1]*5)

    print(f"   样本数: {len(y)}")
    print(f"   特征维数: 2")
    print(f"   类别数: 2")

    # 计算量子核矩阵
    print("\n3. 计算量子核矩阵 K(x_i, x_j) = |<φ(x_i)|φ(x_j)>|²")

    n_samples = len(y)
    K = np.zeros((n_samples, n_samples))

    for i in range(n_samples):
        for j in range(n_samples):
            # 创建特征映射电路
            qc_i = create_feature_map(X[i], 2)
            qc_j = create_feature_map(X[j], 2)

            # 计算重叠
            sv_i = Statevector.from_instruction(qc_i)
            sv_j = Statevector.from_instruction(qc_j)

            overlap = np.abs(np.vdot(sv_i.data, sv_j.data))**2
            K[i, j] = overlap

    print("   核矩阵:")
    print(np.round(K, 3))

    # 可视化核矩阵
    fig, axes = plt.subplots(1, 2, figsize=(14, 6))

    # 核矩阵热图
    im = axes[0].imshow(K, cmap='viridis', aspect='auto')
    axes[0].set_xlabel('Sample Index', fontsize=12)
    axes[0].set_ylabel('Sample Index', fontsize=12)
    axes[0].set_title('Quantum Kernel Matrix', fontsize=14)
    plt.colorbar(im, ax=axes[0])

    # 添加类别分隔线
    axes[0].axhline(y=4.5, color='red', linestyle='--', alpha=0.5)
    axes[0].axvline(x=4.5, color='red', linestyle='--', alpha=0.5)

    # 数据分布
    axes[1].scatter(class_0[:, 0], class_0[:, 1], c='blue', label='Class 0', s=100)
    axes[1].scatter(class_1[:, 0], class_1[:, 1], c='red', label='Class 1', s=100)
    axes[1].set_xlabel('Feature 1', fontsize=12)
    axes[1].set_ylabel('Feature 2', fontsize=12)
    axes[1].set_title('Data Distribution', fontsize=14)
    axes[1].legend()
    axes[1].grid(True, alpha=0.3)

    plt.tight_layout()
    plt.savefig('quantum_kernel.png', dpi=150, bbox_inches='tight')
    print("\n4. 量子核方法可视化已保存到 quantum_kernel.png")

    # 核矩阵性质分析
    print("\n5. 核矩阵分析")
    eigenvalues = np.linalg.eigvalsh(K)
    print(f"   特征值: {np.round(eigenvalues, 4)}")
    print(f"   条件数: {np.max(eigenvalues) / (np.min(eigenvalues) + 1e-10):.2f}")
    print(f"   是否正定: {np.all(eigenvalues > -1e-10)}")

    return K
```

### 8.2 实际考虑因素

**量子优势的条件**：

1. **问题规模足够大**：量子优势通常在渐近区域显现
2. **电路深度可接受**：NISQ设备限制 ~100-1000门
3. **错误率足够低**：当前 ~0.1%，目标 ~0.01%
4. **经典模拟困难**：问题应该在经典计算机上难以解决

### 8.3 NISQ时代的现实

**当前限制（2024）**：

- 量子比特数：50-1000+
- 相干时间：100μs-1ms
- 门保真度：99.9%（单比特），99%（双比特）
- 连接性：受限（最近邻或网格）

**适用问题**：

- 小规模分子模拟（<50轨道）
- 小规模优化问题（<100变量）
- 概念验证实验
