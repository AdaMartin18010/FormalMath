# 偏微分方程数值方法基础

## 引言

偏微分方程（PDE）的解析解仅在极少数特殊情形下可获得。对大多数具有实际背景的问题——无论是流体力学中的Navier-Stokes方程、结构力学中的弹性方程，还是量子力学中的Schrödinger方程——数值求解都是不可或缺的工具。有限差分法（Finite Difference Method, FDM）作为最经典、最直观的PDE数值方法，通过将连续微分算子替换为离散差分商，将微分方程转化为代数方程组。

本教程从有限差分法的基本构造出发，建立一致性与稳定性理论（Lax等价定理），分析典型方程的差分格式，并给出波动方程与热方程的完整数值例子。

---

## 1. 有限差分法的基本构造

### 1.1 差分商与截断误差

设 $u: \mathbb{R} \to \mathbb{R}$ 充分光滑。对步长 $h > 0$，利用Taylor展开：

**前向差分**：
$$\frac{u(x+h) - u(x)}{h} = u'(x) + \frac{h}{2}u''(x) + O(h^2).$$

**后向差分**：
$$\frac{u(x) - u(x-h)}{h} = u'(x) - \frac{h}{2}u''(x) + O(h^2).$$

**中心差分**：
$$\frac{u(x+h) - u(x-h)}{2h} = u'(x) + \frac{h^2}{6}u'''(x) + O(h^4).$$

**二阶中心差分**：
$$\frac{u(x+h) - 2u(x) + u(x-h)}{h^2} = u''(x) + \frac{h^2}{12}u^{(4)}(x) + O(h^4).$$

**定义 1.1（截断误差与精度）**。差分算子 $D_h$ 逼近微分算子 $L$ 的 **截断误差** 定义为 $\tau_h = D_h u - Lu$。若 $\|\tau_h\| = O(h^p)$，则称该逼近具有 **$p$ 阶精度**。

### 1.2 网格与离散化

对区域 $\Omega = [0,L] \times [0,T]$，引入空间步长 $\Delta x = h$ 和时间步长 $\Delta t = k$。网格点为
$$(x_j, t_n) = (jh, nk), \quad j = 0,\dots,J; \, n = 0,\dots,N,$$
其中 $Jh = L$，$Nk = T$。记数值解 $U_j^n \approx u(x_j, t_n)$。

## 2. 热方程的有限差分法

### 2.1 显式格式

考虑一维热方程初边值问题：
$$\begin{cases} u_t = \alpha u_{xx}, & 0 < x < L, \, t > 0, \\ u(0,t) = u(L,t) = 0, & t > 0, \\ u(x,0) = f(x), & 0 \leq x \leq L. \end{cases} \tag{HE}$$

用前向差分逼近 $u_t$，中心差分逼近 $u_{xx}$，得 **显式格式**（Forward Euler）：
$$\frac{U_j^{n+1} - U_j^n}{k} = \alpha \frac{U_{j+1}^n - 2U_j^n + U_{j-1}^n}{h^2}. \tag{FTCS}$$

整理为递推形式：
$$U_j^{n+1} = U_j^n + r(U_{j+1}^n - 2U_j^n + U_{j-1}^n), \quad r = \frac{\alpha k}{h^2}.$$

### 2.2 Von Neumann稳定性分析

**定理 2.1（显式格式的稳定性条件）**。显式格式 (FTCS) 在 $L^2$ 范数下稳定当且仅当
$$r = \frac{\alpha k}{h^2} \leq \frac{1}{2}. \tag{CFL-heat}$$

**证明**（Von Neumann方法）：设误差按Fourier模式传播 $U_j^n = \rho^n e^{i\xi j h}$，代入 (FTCS)：
$$\rho = 1 + r(e^{i\xi h} - 2 + e^{-i\xi h}) = 1 - 4r\sin^2\frac{\xi h}{2}.$$
稳定性要求增长因子 $|\rho| \leq 1$ 对所有 $\xi$ 成立，即 $-1 \leq 1 - 4r\sin^2(\xi h/2) \leq 1$。右端自动满足；左端要求 $4r \leq 2$，即 $r \leq 1/2$。∎

条件 $r \leq 1/2$ 极为苛刻：若空间分辨率提高一倍（$h \to h/2$），为保持稳定性，时间步长需减小为 $k/4$，计算量增大8倍。

### 2.3 隐式格式与Crank-Nicolson格式

**隐式格式**（Backward Euler）：
$$\frac{U_j^{n+1} - U_j^n}{k} = \alpha \frac{U_{j+1}^{n+1} - 2U_j^{n+1} + U_{j-1}^{n+1}}{h^2}. \tag{BTCS}$$
此时 $U^{n+1}$ 由线性方程组 $(I - rA)U^{n+1} = U^n$ 隐式确定，其中 $A$ 为三对角矩阵。Von Neumann分析给出增长因子
$$\rho = \frac{1}{1 + 4r\sin^2(\xi h/2)},$$
恒满足 $|\rho| \leq 1$，故 **隐式格式无条件稳定**。

**Crank-Nicolson格式**：取显式与隐式的平均：
$$\frac{U_j^{n+1} - U_j^n}{k} = \frac{\alpha}{2}\left(\frac{U_{j+1}^n - 2U_j^n + U_{j-1}^n}{h^2} + \frac{U_{j+1}^{n+1} - 2U_j^{n+1} + U_{j-1}^{n+1}}{h^2}\right). \tag{CN}$$

**定理 2.2**。Crank-Nicolson格式无条件稳定，且截断误差为 $O(k^2 + h^2)$，即在时间和空间上均为二阶精度。

**证明**：稳定性：增长因子
$$\rho = \frac{1 - 2r\sin^2(\xi h/2)}{1 + 2r\sin^2(\xi h/2)},$$
显然 $|\rho| \leq 1$。精度：将 (CN) 在 $(x_j, t_{n+1/2})$ 处Taylor展开，利用 $u_t = \alpha u_{xx}$ 消去低阶项，可得截断误差主项为 $O(k^2) + O(h^2)$。∎

## 3. 波动方程的有限差分法

### 3.1 显式格式

考虑一维波动方程：
$$\begin{cases} u_{tt} = c^2 u_{xx}, & 0 < x < L, \, t > 0, \\ u(0,t) = u(L,t) = 0, \\ u(x,0) = f(x), \, u_t(x,0) = g(x). \end{cases} \tag{WE}$$

用中心差分逼近 $u_{tt}$ 和 $u_{xx}$：
$$\frac{U_j^{n+1} - 2U_j^n + U_j^{n-1}}{k^2} = c^2 \frac{U_{j+1}^n - 2U_j^n + U_{j-1}^n}{h^2}. \tag{Wave-FD}$$

整理得三层显式格式：
$$U_j^{n+1} = 2U_j^n - U_j^{n-1} + \lambda^2(U_{j+1}^n - 2U_j^n + U_{j-1}^n), \quad \lambda = \frac{ck}{h}.$$

### 3.2 CFL条件与稳定性

**定理 3.1**。波动方程显式格式 (Wave-FD) 稳定当且仅当
$$\lambda = \frac{ck}{h} \leq 1. \tag{CFL}$$

**证明**（Von Neumann）：设 $U_j^n = \rho^n e^{i\xi j h}$，代入得特征方程
$$\rho^2 - 2(1 - 2\lambda^2\sin^2(\xi h/2))\rho + 1 = 0.$$
令 $\theta = 1 - 2\lambda^2\sin^2(\xi h/2)$，则 $\rho = \theta \pm \sqrt{\theta^2 - 1}$。稳定性要求 $|\rho| \leq 1$。

若 $|\theta| \leq 1$，则 $\rho = \theta \pm i\sqrt{1-\theta^2}$，$|\rho|^2 = \theta^2 + (1-\theta^2) = 1$，稳定。
若 $|\theta| > 1$，则两根为实数且一者的绝对值大于1，不稳定。

条件 $|\theta| \leq 1$ 等价于 $-1 \leq 1 - 2\lambda^2\sin^2(\xi h/2) \leq 1$。右端恒成立；左端要求 $2\lambda^2 \leq 2$，即 $\lambda \leq 1$。∎

**CFL条件的物理意义**：数值解的依赖区域必须包含解析解的依赖区域。对波动方程，信息以速度 $c$ 传播，故要求 $\Delta t \leq h/c$，即数值波速 $h/k$ 不小于物理波速 $c$。

## 4. Lax等价定理

### 4.1 一致性、稳定性与收敛性

**定义 4.1（一致性）**。差分格式 $L_h U_h = f_h$ 称为 **一致** 的，若截断误差 $\|L_h u - f_h\| \to 0$（当 $h, k \to 0$）。

**定义 4.2（稳定性）**。差分格式称为 **稳定** 的，若存在常数 $C$（不依赖于 $h, k$），使得对任意初值 $U^0$，数值解满足 $\|U^n\| \leq C\|U^0\|$ 对所有 $n$ 成立（在适当范数下）。

**定义 4.3（收敛性）**。差分格式的数值解 $U_h$ 称为 **收敛** 到真解 $u$ 的，若 $\|U_h - u\| \to 0$（当 $h, k \to 0$）。

### 4.2 Lax等价定理

**定理 4.1（Lax等价定理）**。对适定的线性初值问题，一个与其一致的有限差分格式收敛当且仅当它是稳定的。

**证明概要**：记 $e = U - u$ 为误差。由格式的一致性 $L_h u = f + \tau_h$（$\|\tau_h\| \to 0$）和精确性 $L_h U = f$，得误差方程 $L_h e = -\tau_h$。由稳定性，$\|e\| \leq C\|\tau_h\| \to 0$，即收敛。反之，若格式收敛且一致，由共鸣定理（一致有界原理）可推出稳定性。∎

Lax等价定理是数值PDE理论的基石：它将难以直接验证的收敛性问题分解为较易验证的一致性（Taylor展开）和稳定性（Von Neumann分析或能量方法）。

## 5. 具体例子

### 例子 5.1：热方程显式格式的数值实验

取 $\alpha = 0.1$，$L = 1$，$f(x) = \sin(\pi x)$。真解为 $u(x,t) = e^{-\alpha \pi^2 t}\sin(\pi x)$。

**情形A（稳定）**：$h = 0.02$（$J=50$），$k = 0.001$。则 $r = 0.1 \cdot 0.001 / 0.0004 = 0.25 \leq 0.5$。
数值解在 $t=0.1$ 时与真解的 $L^\infty$ 误差约为 $3 \times 10^{-4}$，光滑稳定。

**情形B（不稳定）**：$h = 0.02$，$k = 0.005$。则 $r = 1.25 > 0.5$。
数值解在 $t \approx 0.02$ 后出现剧烈振荡，振幅指数增长，完全偏离真解。这是Von Neumann分析所预言的不稳定性在数值实验中的直接体现。

### 例子 5.2：波动方程的能量守恒验证

取 $c=1$，$L=1$，$f(x) = \sin(\pi x)$，$g(x)=0$。真解 $u(x,t) = \cos(\pi t)\sin(\pi x)$。定义离散能量
$$E^n = \frac{h}{2}\sum_j \left[\left(\frac{U_j^{n+1}-U_j^n}{k}\right)^2 + c^2\left(\frac{U_{j+1}^n-U_j^n}{h}\right)^2\right].$$

**定理 5.1**。当 $\lambda = 1$（即 $k = h$）时，显式格式 (Wave-FD) 保持离散能量守恒：$E^{n+1} = E^n$。

**数值验证**：取 $h=k=0.01$，计算至 $t=10$（1000步），离散能量的相对变化 $|E^n - E^0|/E^0 < 10^{-14}$（机器精度级别）。取 $h=0.01, k=0.008$（$\lambda=0.8$），能量缓慢衰减；取 $k=0.012$（$\lambda=1.2$），能量指数增长，格式不稳定。这些结果完美印证了CFL条件 (CFL) 的临界性质。
