# 偏微分方程习题与数值解

## 习题

### 习题1

用分离变量法求解热方程：
$$\frac{\partial u}{\partial t} = k \frac{\partial^2 u}{\partial x^2}, \quad 0 < x < L, \quad t > 0$$
边界条件：$u(0,t) = u(L,t) = 0$
初始条件：$u(x,0) = f(x)$

**解答**：设$u(x,t) = X(x)T(t)$，分离得：
$$\frac{T'}{kT} = \frac{X''}{X} = -\lambda$$

解得特征值$\lambda_n = (\frac{n\pi}{L})^2$，特征函数$X_n(x) = \sin(\frac{n\pi x}{L})$。

通解：$u(x,t) = \sum_{n=1}^{\infty} B_n \sin(\frac{n\pi x}{L}) e^{-k(\frac{n\pi}{L})^2 t}$

其中$B_n = \frac{2}{L} \int_0^L f(x) \sin(\frac{n\pi x}{L}) dx$。

---

### 习题2

推导波动方程的达朗贝尔解。

**解答**：波动方程$u_{tt} = c^2 u_{xx}$

令$\xi = x + ct$，$\eta = x - ct$，方程化为$u_{\xi\eta} = 0$。

解得：$u(x,t) = F(x+ct) + G(x-ct)$。

### 习题3

证明拉普拉斯方程的解满足极值原理。

**证明**：若$u$在有界区域$\Omega$内调和，则$u$的最大最小值在边界$\partial\Omega$上取得。

**思路**：假设内部有极大值点，由调和函数性质推出矛盾。

## 数值解：有限差分法

### 热方程的显式格式

```python
import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D

# 参数
L = 1.0      # 空间长度
T = 0.1      # 时间长度
k = 0.1      # 热扩散系数
nx = 50      # 空间网格数
nt = 1000    # 时间步数

dx = L / nx
dt = T / nt
r = k * dt / dx**2  # 稳定性条件：r <= 0.5

print(f"稳定性参数 r = {r:.4f}")

# 初始条件
def initial_condition(x):
    return np.sin(np.pi * x)

# 初始化
x = np.linspace(0, L, nx+1)
u = initial_condition(x)
u_new = np.zeros_like(u)

# 边界条件
u[0] = u[-1] = 0

# 存储历史
u_history = [u.copy()]

# 时间推进
for n in range(nt):
    for i in range(1, nx):
        u_new[i] = u[i] + r * (u[i+1] - 2*u[i] + u[i-1])

    u, u_new = u_new, u
    u[0] = u[-1] = 0  # 边界条件

    if n % 50 == 0:
        u_history.append(u.copy())

# 可视化
plt.figure(figsize=(10, 6))
for i, uh in enumerate(u_history[::5]):
    plt.plot(x, uh, label=f't={i*50*dt:.3f}')
plt.xlabel('x')
plt.ylabel('u')
plt.title('Heat Equation Solution (Explicit Method)')
plt.legend()
plt.grid(True)
plt.savefig('heat_equation_fd.png')
plt.show()
```
