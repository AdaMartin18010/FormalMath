# 10-应用数学

本目录聚焦数学在现代科学与技术中的跨学科应用，收录约 90 份深度文档，分为核心内容与多轮补充扩展。文档主题涵盖医学成像、概率建模、统计分析、信息论、优化方法、金融数学及机器学习基础等，旨在展示抽象数学工具如何解决真实世界问题。每份文件通常从实际问题出发，建立数学模型，推导关键公式，并辅以数值实例或算法伪代码，为理工科学习者提供从理论到应用的桥梁。
