# 博弈论基础

## 纳什均衡

### 定义
策略组合$(s_1^*, ..., s_n^*)$是纳什均衡若对任意玩家$i$：
$$u_i(s_i^*, s_{-i}^*) \geq u_i(s_i, s_{-i}^*), \quad \forall s_i$$

### 囚徒困境
|  | C | D |
|--|---|---|
| C | (-1,-1) | (-3,0) |
| D | (0,-3) | (-2,-2) |

纳什均衡：(D,D)
