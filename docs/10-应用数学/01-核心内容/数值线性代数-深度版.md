## 1. 矩阵分解技术

### 1.1 LU分解与选主元

**定义 1.1** (LU分解)

对于非奇异矩阵 $A \in \mathbb{R}^{n \times n}$，LU分解将其分解为下三角矩阵 $L$ 和上三角矩阵 $U$ 的乘积：

$$A = LU$$

其中 $L$ 的对角线元素为1，$U$ 的对角线元素非零。

**算法 1.1** (带部分选主元的LU分解)

```python
def lu_decomposition_pivot(A):
    """带部分选主元的LU分解"""
    n = A.shape[0]
    L = np.eye(n)
    U = A.copy()
    P = np.eye(n)  # 置换矩阵
    
    for k in range(n-1):
        # 选主元
        max_idx = np.argmax(np.abs(U[k:, k])) + k
        if max_idx != k:
            U[[k, max_idx]] = U[[max_idx, k]]
            P[[k, max_idx]] = P[[max_idx, k]]
            if k > 0:
                L[[k, max_idx], :k] = L[[max_idx, k], :k]
        
        # 消元
        for i in range(k+1, n):
            L[i, k] = U[i, k] / U[k, k]
            U[i, k:] -= L[i, k] * U[k, k:]
    
    return P, L, U
```

**复杂度分析**：

- 时间复杂度：$O(n^3)$
- 空间复杂度：$O(n^2)$

**数值稳定性**：

部分选主元保证 $|l_{ij}| \leq 1$，有效控制舍入误差增长。

### 1.2 QR分解

**定义 1.2** (QR分解)

对于矩阵 $A \in \mathbb{R}^{m \times n}$ ($m \geq n$)，QR分解为：

$$A = QR$$

其中 $Q \in \mathbb{R}^{m \times m}$ 是正交矩阵，$R \in \mathbb{R}^{m \times n}$ 是上三角矩阵。

**Householder变换**：

$$H = I - 2\frac{vv^T}{v^Tv}$$

**算法 1.2** (Householder QR分解)

```python
def householder_qr(A):
    """Householder QR分解"""
    m, n = A.shape
    Q = np.eye(m)
    R = A.copy()
    
    for k in range(n):
        x = R[k:, k]
        e1 = np.zeros_like(x)
        e1[0] = np.sign(x[0]) * np.linalg.norm(x)
        v = x + e1
        v = v / np.linalg.norm(v)
        
        # 应用Householder变换
        R[k:, k:] -= 2.0 * np.outer(v, v @ R[k:, k:])
        Q[:, k:] -= 2.0 * np.outer(Q[:, k:] @ v, v)
    
    return Q, R
```

### 1.3 Cholesky分解

**定义 1.3** (Cholesky分解)

对于对称正定矩阵 $A$，Cholesky分解为：

$$A = LL^T$$

其中 $L$ 是下三角矩阵。

**算法 1.3** (Cholesky分解)

```python
def cholesky(A):
    """Cholesky分解"""
    n = A.shape[0]
    L = np.zeros_like(A)
    
    for j in range(n):
        L[j, j] = np.sqrt(A[j, j] - np.sum(L[j, :j]**2))
        for i in range(j+1, n):
            L[i, j] = (A[i, j] - np.sum(L[i, :j] * L[j, :j])) / L[j, j]
    
    return L
```

**复杂度**：$O(n^3/3)$，约为LU分解的一半。

### 1.4 SVD分解

**定义 1.4** (奇异值分解)

对于矩阵 $A \in \mathbb{R}^{m \times n}$，SVD为：

$$A = U\Sigma V^T$$

其中 $U \in \mathbb{R}^{m \times m}$，$V \in \mathbb{R}^{n \times n}$ 是正交矩阵，$\Sigma$ 是对角矩阵。

**应用**：
- 矩阵伪逆
- 主成分分析
- 低秩逼近

## 2. 稀疏矩阵技术

### 2.1 稀疏矩阵存储格式

**CSR (Compressed Sparse Row) 格式**：

```python
class CSRMatrix:
    """CSR格式稀疏矩阵"""
    def __init__(self, data, indices, indptr, shape):
        self.data = data      # 非零元素值
        self.indices = indices  # 列索引
        self.indptr = indptr    # 行指针
        self.shape = shape
    
    def matvec(self, x):
        """矩阵向量乘法"""
        y = np.zeros(self.shape[0])
        for i in range(self.shape[0]):
            for j in range(self.indptr[i], self.indptr[i+1]):
                y[i] += self.data[j] * x[self.indices[j]]
        return y
```

**CSC (Compressed Sparse Column) 格式**：

适合列操作，与CSR格式类似。

### 2.2 稀疏直接法

**最小度排序**：

减少Cholesky分解中的填充元（fill-in）。

**嵌套剖分排序**：

对于二维/三维网格，产生 $O(n^{3/2})$ 或 $O(n^{4/3})$ 的非零元。

## 3. 迭代求解方法

### 3.1 共轭梯度法

**定义 3.1** (共轭梯度法 CG)

用于求解对称正定线性系统 $Ax = b$。

**算法 3.1** (共轭梯度法)

```python
def conjugate_gradient(A, b, x0=None, tol=1e-10, max_iter=None):
    """共轭梯度法"""
    n = len(b)
    if x0 is None:
        x = np.zeros(n)
    else:
        x = x0.copy()
    
    r = b - A @ x
    p = r.copy()
    rs_old = np.dot(r, r)
    
    if max_iter is None:
        max_iter = n
    
    for i in range(max_iter):
        Ap = A @ p
        alpha = rs_old / np.dot(p, Ap)
        x += alpha * p
        r -= alpha * Ap
        rs_new = np.dot(r, r)
        
        if np.sqrt(rs_new) < tol:
            break
        
        p = r + (rs_new / rs_old) * p
        rs_old = rs_new
    
    return x, i
```

**收敛性**：

$$\|x_k - x^*\|_A \leq 2\left(\frac{\sqrt{\kappa} - 1}{\sqrt{\kappa} + 1}\right)^k \|x_0 - x^*\|_A$$

其中 $\kappa = \lambda_{\max}/\lambda_{\min}$ 是条件数。

### 3.2 GMRES方法

**定义 3.2** (GMRES)

广义最小残差法，用于非对称线性系统。

最小化 $\|b - Ax_k\|_2$ 在Krylov子空间 $\mathcal{K}_k(A, r_0)$ 上。

### 3.3 多重网格方法

**原理**：

在不同网格层次上平滑误差分量。

- 细网格：消除高频误差
- 粗网格：消除低频误差

**V循环算法**：

```python
def v_cycle(A_h, b_h, x_h, level, max_levels):
    """多重网格V循环"""
    if level == max_levels:
        return direct_solve(A_h, b_h)
    
    # 前光滑
    x_h = smooth(A_h, b_h, x_h, nu=2)
    
    # 计算残差
    r_h = b_h - A_h @ x_h
    
    # 限制到粗网格
    r_2h = restrict(r_h)
    A_2h = coarse_operator(A_h)
    
    # 粗网格校正
    e_2h = np.zeros_like(r_2h)
    e_2h = v_cycle(A_2h, r_2h, e_2h, level+1, max_levels)
    
    # 延拓到细网格
    x_h += prolongate(e_2h)
    
    # 后光滑
    x_h = smooth(A_h, b_h, x_h, nu=2)
    
    return x_h
```

## 4. 特征值问题

### 4.1 QR算法

**算法 4.1** (带位移的QR算法)

```python
def qr_algorithm(A, max_iter=100, tol=1e-10):
    """QR算法计算特征值"""
    n = A.shape[0]
    T = A.copy()
    
    for k in range(max_iter):
        # 选择位移
        mu = T[n-1, n-1]
        
        # QR分解
        Q, R = np.linalg.qr(T - mu * np.eye(n))
        T = R @ Q + mu * np.eye(n)
        
        # 检查收敛
        if np.abs(T[n-1, n-2]) < tol:
            break
    
    return np.diag(T)
```

## 5. 实际应用案例

### 5.1 结构力学有限元分析

**应用场景**：桥梁结构分析

**数学模型**：

$$Ku = f$$

其中：
- $K$：刚度矩阵（大型稀疏对称正定）
- $u$：位移向量
- $f$：载荷向量

**实际案例**：某大桥有限元模型

```python
# 桥梁有限元分析示例
def bridge_fem_analysis(n_elements):
    """桥梁结构有限元分析"""
    # 组装刚度矩阵（稀疏）
    K = assemble_stiffness_matrix(n_elements)
    
    # 应用边界条件
    K_bc, f_bc = apply_boundary_conditions(K, load_vector)
    
    # 使用预处理共轭梯度法求解
    M = construct_preconditioner(K_bc)  # 不完全Cholesky预处理
    u, iterations = pcg(K_bc, f_bc, M)
    
    # 计算应力
    stress = compute_stress(u)
    
    return u, stress, iterations
```

**性能数据**：
- 自由度：1,000,000
- 求解时间：5分钟（CG+IC预处理）
- 直接法估计时间：> 2小时

### 5.2 电路仿真

**应用场景**：芯片电路仿真

**数学模型**：

$$Gv + C\frac{dv}{dt} = i(t)$$

稳态分析求解：

$$Gx = b$$

其中 $G$ 是电导矩阵（稀疏）。

**实际案例**：

某芯片包含100万个晶体管，电导矩阵维度为500,000，非零元约500万。

### 5.3 图像处理

**应用场景**：图像去模糊

**数学模型**：

$$Ax = b$$

其中 $A$ 是模糊矩阵（块Toeplitz结构）。

**算法选择**：

- 周期性边界：FFT快速求解
- 非周期性边界：迭代方法+预处理器
