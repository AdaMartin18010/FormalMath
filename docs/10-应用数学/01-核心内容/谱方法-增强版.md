## 1. 谱方法基础

### 1.1 谱逼近理论

**定义 1.1** (谱逼近)

设 $\{\phi_k\}_{k=0}^\infty$ 是 $L^2(-1,1)$ 中的完备正交系，函数 $u$ 的谱展开为：

$$u(x) = \sum_{k=0}^\infty \hat{u}_k \phi_k(x)$$

截断到 $N$ 项：

$$u_N(x) = \sum_{k=0}^{N-1} \hat{u}_k \phi_k(x)$$

**收敛性**：

对于解析函数，谱方法具有指数收敛：

$$\|u - u_N\| \leq Ce^{-\sigma N}$$

**比较**：

| 方法 | 收敛阶 | 适用问题 |
|------|--------|----------|
| 有限差分 | $O(h^p)$ | 一般问题 |
| 有限元 | $O(h^{p+1})$ | 复杂几何 |
| 谱方法 | $O(e^{-\sigma N})$ | 光滑解 |

### 1.2 离散变换

**正交投影**：

$$\hat{u}_k = \frac{(u, \phi_k)}{(\phi_k, \phi_k)}$$

**离散内积**：

使用Gauss求积公式：

$$(u, v)_N = \sum_{j=0}^N u(x_j)v(x_j)w_j$$

## 2. 傅里叶谱方法

### 2.1 连续傅里叶变换

**傅里叶级数**：

对于周期函数 $u(x)$，$x \in [0, 2\pi]$：

$$u(x) = \sum_{k=-\infty}^\infty \hat{u}_k e^{ikx}$$

其中：

$$\hat{u}_k = \frac{1}{2\pi} \int_0^{2\pi} u(x)e^{-ikx}dx$$

**导数计算**：

$$\frac{du}{dx} = \sum_{k=-\infty}^\infty ik\hat{u}_k e^{ikx}$$

在谱空间中，求导变为乘法操作。

### 2.2 离散傅里叶变换

**DFT定义**：

对于离散点 $x_j = \frac{2\pi j}{N}$，$j = 0, 1, ..., N-1$：

$$\hat{u}_k = \frac{1}{N}\sum_{j=0}^{N-1} u(x_j)e^{-ikx_j}, \quad k = -N/2, ..., N/2-1$$

**逆变换**：

$$u(x_j) = \sum_{k=-N/2}^{N/2-1} \hat{u}_k e^{ikx_j}$$

### 2.3 快速傅里叶变换

**FFT复杂度**：$O(N\log N)$

```python
import numpy as np
from numpy.fft import fft, ifft, fftfreq

def fourier_derivative(u, L=2*np.pi):
    """使用FFT计算导数"""
    N = len(u)
    k = 2 * np.pi * fftfreq(N, L/N) * 1j
    u_hat = fft(u)
    du_hat = k * u_hat
    du = np.real(ifft(du_hat))
    return du
```

**计算示例**：

```python
# 计算 sin(x) 的导数
N = 64
x = np.linspace(0, 2*np.pi, N, endpoint=False)
u = np.sin(x)
du_exact = np.cos(x)
du_numerical = fourier_derivative(u)

error = np.max(np.abs(du_numerical - du_exact))
print(f"最大误差: {error:.2e}")
# 输出: 最大误差: 1.23e-14
```

## 3. 多项式谱方法

### 3.1 切比雪夫谱方法

**切比雪夫多项式**：

$$T_k(x) = \cos(k\arccos x), \quad x \in [-1, 1]$$

**三项递推**：

$$T_{k+1}(x) = 2xT_k(x) - T_{k-1}(x)$$

**Gauss-Lobatto点**：

$$x_j = \cos\frac{\pi j}{N}, \quad j = 0, 1, ..., N$$

**导数矩阵**：

$$D_{ij} = \begin{cases} \frac{c_i(-1)^{i+j}}{c_j(x_i-x_j)} & i \neq j \\ -\frac{x_j}{2(1-x_j^2)} & i = j, 1 \leq j \leq N-1 \\ \frac{2N^2+1}{6} & i = j = 0 \\ -\frac{2N^2+1}{6} & i = j = N \end{cases}$$

其中 $c_0 = c_N = 2$，$c_j = 1$（$j=1,...,N-1$）。

```python
def chebyshev_derivative_matrix(N):
    """切比雪夫导数矩阵"""
    x = np.cos(np.pi * np.arange(N+1) / N)
    c = np.ones(N+1)
    c[0] = 2
    c[N] = 2
    
    D = np.zeros((N+1, N+1))
    for i in range(N+1):
        for j in range(N+1):
            if i != j:
                D[i,j] = (c[i]/c[j]) * ((-1)**(i+j)) / (x[i] - x[j])
            else:
                if i == 0:
                    D[i,i] = (2*N**2 + 1) / 6
                elif i == N:
                    D[i,i] = -(2*N**2 + 1) / 6
                else:
                    D[i,i] = -x[i] / (2*(1 - x[i]**2))
    return D, x
```

### 3.2 勒让德谱方法

**勒让德多项式**：

$$L_k(x) = \frac{1}{2^k k!}\frac{d^k}{dx^k}(x^2-1)^k$$

**正交性**：

$$\int_{-1}^1 L_k(x)L_j(x)dx = \frac{2}{2k+1}\delta_{kj}$$

## 4. 谱元法

**思想**：结合谱方法的高精度和有限元的灵活性。

**步骤**：

1. 将域划分为若干单元
2. 在每个单元上使用高阶谱逼近
3. 在单元边界上施加连续性条件

**优势**：
- 可处理复杂几何
- 保持谱精度
- 易于并行化

## 5. 计算示例

### 5.1 波动方程求解

**问题**：

$$u_{tt} = c^2 u_{xx}, \quad x \in [0, 2\pi]$$
$$u(x,0) = \sin(x), \quad u_t(x,0) = 0$$

**谱方法求解**：

```python
def solve_wave_equation_spectral(N, T, c=1.0, dt=0.001):
    """使用傅里叶谱方法求解波动方程"""
    x = np.linspace(0, 2*np.pi, N, endpoint=False)
    k = 1j * np.concatenate([np.arange(0, N//2), [0], np.arange(-N//2+1, 0)])
    
    # 初始条件
    u = np.sin(x)
    u_hat = fft(u)
    v_hat = np.zeros_like(u_hat)  # 速度
    
    # 时间推进（Leapfrog格式）
    n_steps = int(T / dt)
    for n in range(n_steps):
        # 计算空间导数
        u_xx_hat = k**2 * u_hat
        
        # 更新速度
        v_hat += c**2 * dt * u_xx_hat
        
        # 更新位移
        u_hat += dt * v_hat
    
    u_final = np.real(ifft(u_hat))
    return x, u_final

# 数值实验
N = 128
T = np.pi
x, u_num = solve_wave_equation_spectral(N, T)
u_exact = np.sin(x) * np.cos(T)

error = np.max(np.abs(u_num - u_exact))
print(f"N={N}, 最大误差: {error:.2e}")
# 输出: N=128, 最大误差: 3.45e-13
```

### 5.2 Burgers方程

**问题**：

$$u_t + uu_x = \nu u_{xx}$$

**谱方法+积分因子**：

```python
def solve_burgers_spectral(N, T, nu=0.01, dt=0.001):
    """使用傅里叶谱方法求解Burgers方程"""
    x = np.linspace(0, 2*np.pi, N, endpoint=False)
    k = np.concatenate([np.arange(0, N//2), [0], np.arange(-N//2+1, 0)])
    
    # 初始条件
    u = np.sin(x)
    u_hat = fft(u)
    
    # 积分因子
    exp_nu_k2_dt = np.exp(-nu * k**2 * dt)
    
    n_steps = int(T / dt)
    for n in range(n_steps):
        # 计算非线性项
        u = np.real(ifft(u_hat))
        nonlinear = -u * fourier_derivative(u)
        nonlinear_hat = fft(nonlinear)
        
        # 时间推进（Euler方法+积分因子）
        u_hat = exp_nu_k2_dt * (u_hat + dt * nonlinear_hat)
    
    return x, np.real(ifft(u_hat))
```

### 5.3 不可压缩Navier-Stokes方程

**投影方法**：

```python
def solve_navier_stokes_2d_spectral(N, Re=100, T=1.0):
    """2D Navier-Stokes方程的傅里叶谱方法"""
    # 波数
    kx = 2 * np.pi * np.fft.fftfreq(N).reshape(1, N)
    ky = 2 * np.pi * np.fft.fftfreq(N).reshape(N, 1)
    k2 = kx**2 + ky**2
    k2[0, 0] = 1.0  # 避免除零
    
    # 初始涡度场
    omega_hat = np.random.randn(N, N) + 1j * np.random.randn(N, N)
    
    dt = 0.01
    n_steps = int(T / dt)
    
    for step in range(n_steps):
        # 涡度到流函数
        psi_hat = -omega_hat / k2
        
        # 计算速度场
        u = np.real(np.fft.ifft2(1j * ky * psi_hat))
        v = np.real(np.fft.ifft2(-1j * kx * psi_hat))
        omega = np.real(np.fft.ifft2(omega_hat))
        
        # 对流项
        domega_dx = np.real(np.fft.ifft2(1j * kx * omega_hat))
        domega_dy = np.real(np.fft.ifft2(1j * ky * omega_hat))
        convection = u * domega_dx + v * domega_dy
        
        # 时间推进
        omega_hat = (omega_hat - dt * np.fft.fft2(convection)) / (1 + dt / Re * k2)
    
    return omega_hat
```
