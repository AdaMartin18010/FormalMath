# 强化学习数学基础

## 引言

强化学习（Reinforcement Learning, RL）是机器学习的一个核心范式，研究智能体（agent）如何在环境中通过试错学习最优决策策略。与监督学习不同，强化学习没有预先标注的数据，智能体仅通过环境反馈的**奖励信号**来评估行为。从数学上看，强化学习的理论框架建立在**马尔可夫决策过程**（Markov Decision Process, MDP）之上，而核心算法（如策略迭代、Q学习）则与动态规划、随机逼近和Bellman最优性原理紧密相连。

本教程从MDP的严格定义出发，建立Bellman方程理论，证明策略迭代与值迭代的收敛性，并分析两个经典例子。

---

## 1. 马尔可夫决策过程

### 1.1 定义

**定义 1.1（马尔可夫决策过程）**。一个**有限马尔可夫决策过程**（finite MDP）是五元组
$$\mathcal{M} = (S, A, P, R, \gamma),$$
其中：
- $S$ 为有限**状态空间**，$|S| = n$；
- $A$ 为有限**动作空间**，$|A| = m$（允许 $A(s) \subseteq A$ 依赖于状态 $s$）；
- $P: S \times A \times S \to [0,1]$ 为**状态转移概率**，$P(s'|s,a)$ 表示在状态 $s$ 采取动作 $a$ 后转移到 $s'$ 的概率，满足 $\sum_{s' \in S} P(s'|s,a) = 1$；
- $R: S \times A \to \mathbb{R}$ 为**期望奖励函数**（有时也定义为 $R: S \times A \times S \to \mathbb{R}$）；
- $\gamma \in [0,1)$ 为**折扣因子**（discount factor）。

**定义 1.2（策略）**。**随机策略**（stochastic policy）是映射 $\pi: S \times A \to [0,1]$，满足 $\sum_{a \in A} \pi(a|s) = 1$ 对所有 $s \in S$。若对每个 $s$ 存在唯一 $a$ 使 $\pi(a|s)=1$，则称 $\pi$ 为**确定性策略**（deterministic policy）。

### 1.2 马尔可夫性

**定义 1.3（马尔可夫性）**。转移概率满足马尔可夫性：对任意历史 $s_0, a_0, \dots, s_{t-1}, a_{t-1}$，
$$\mathbb{P}(S_{t+1}=s' \mid S_t=s, A_t=a, \text{历史}) = P(s'|s,a).$$
即下一状态仅依赖于当前状态和动作，与历史无关。

## 2. 价值函数与Bellman方程

### 2.1 状态价值函数

**定义 2.1（回报）**。在策略 $\pi$ 下，从时刻 $t$ 开始的**折扣回报**（discounted return）定义为随机变量
$$G_t := \sum_{k=0}^{\infty} \gamma^k R_{t+k+1},$$
其中 $R_{t+1}$ 为时刻 $t$ 采取动作后获得的奖励。

**定义 2.2（状态价值函数）**。策略 $\pi$ 的**状态价值函数** $v_\pi: S \to \mathbb{R}$ 定义为
$$v_\pi(s) := \mathbb{E}_\pi\left[ G_t \mid S_t = s \right] = \mathbb{E}_\pi\left[ \sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \mid S_t = s \right].$$
期望 $\mathbb{E}_\pi$ 表示状态转移服从 $P$，动作选择服从 $\pi$。

**定义 2.3（动作价值函数）**。策略 $\pi$ 的**动作价值函数** $q_\pi: S \times A \to \mathbb{R}$ 定义为
$$q_\pi(s,a) := \mathbb{E}_\pi\left[ G_t \mid S_t = s, A_t = a \right].$$

### 2.2 Bellman期望方程

**定理 2.1（Bellman期望方程）**。对任意策略 $\pi$，状态价值函数满足
$$v_\pi(s) = \sum_a \pi(a|s) \left[ R(s,a) + \gamma \sum_{s'} P(s'|s,a) v_\pi(s') \right], \quad \forall s \in S. \tag{BE}$$

**证明**：由定义，$v_\pi(s) = \mathbb{E}_\pi[G_t | S_t=s]$。将 $G_t$ 拆分为即刻奖励与后续回报：
$$G_t = R_{t+1} + \gamma G_{t+1}.$$
取条件期望：
$$\mathbb{E}_\pi[G_t | S_t=s] = \mathbb{E}_\pi[R_{t+1} | S_t=s] + \gamma \mathbb{E}_\pi[G_{t+1} | S_t=s].$$
第一项：$\mathbb{E}_\pi[R_{t+1}|S_t=s] = \sum_a \pi(a|s) R(s,a)$。

第二项：由全期望公式和马尔可夫性，
$$\mathbb{E}_\pi[G_{t+1}|S_t=s] = \sum_a \pi(a|s) \sum_{s'} P(s'|s,a) \mathbb{E}_\pi[G_{t+1}|S_{t+1}=s'] = \sum_a \pi(a|s) \sum_{s'} P(s'|s,a) v_\pi(s').$$
合并即得 (BE)。∎

### 2.3 Bellman最优方程

**定义 2.4（最优价值函数）**。
$$v_*(s) := \max_\pi v_\pi(s), \quad q_*(s,a) := \max_\pi q_\pi(s,a).$$

**定理 2.2（Bellman最优方程）**。最优状态价值函数满足
$$v_*(s) = \max_a \left[ R(s,a) + \gamma \sum_{s'} P(s'|s,a) v_*(s') \right], \quad \forall s \in S. \tag{BOE}$$

**证明**：（概要）对任意策略 $\pi$，由 (BE) 有
$$v_\pi(s) = \sum_a \pi(a|s) q_\pi(s,a) \leq \sum_a \pi(a|s) \max_{a'} q_\pi(s,a') = \max_a q_\pi(s,a).$$
对最优策略 $\pi_*$，由最优性原理，$\pi_*$ 在每个状态选择使右端极大的动作，故等号成立。利用 $q_*(s,a) = R(s,a) + \gamma \sum_{s'} P(s'|s,a)v_*(s')$ 即得 (BOE)。∎

## 3. 压缩映射与收敛性

### 3.1 Bellman算子

**定义 3.1（Bellman期望算子）**。算子 $T_\pi: \mathbb{R}^n \to \mathbb{R}^n$ 定义为
$$(T_\pi v)(s) := \sum_a \pi(a|s)\left[R(s,a) + \gamma \sum_{s'}P(s'|s,a)v(s')\right].$$

**定义 3.2（Bellman最优算子）**。算子 $T_*: \mathbb{R}^n \to \mathbb{R}^n$ 定义为
$$(T_* v)(s) := \max_a \left[R(s,a) + \gamma \sum_{s'}P(s'|s,a)v(s')\right].$$

### 3.2 压缩映射定理

**定义 3.3（$\gamma$-压缩）**。度量空间 $(X,d)$ 上的映射 $T$ 称为 **$\gamma$-压缩**（$0 \leq \gamma < 1$），若
$$d(Tx, Ty) \leq \gamma \, d(x,y) \quad \forall x,y \in X.$$

**定理 3.1（Banach不动点定理）**。完备度量空间上的压缩映射有唯一不动点，且对任意初值，迭代序列收敛到该不动点。

**定理 3.2**。在极大范数 $\|\cdot\|_\infty$ 下，$T_\pi$ 和 $T_*$ 均为 $\gamma$-压缩。

**证明**（对 $T_*$）：设 $v, w \in \mathbb{R}^n$，$s \in S$。令 $a_s^*$ 为在 $(T_*v)(s)$ 中达到极大的动作。则
$$(T_*v)(s) - (T_*w)(s) \leq \gamma \sum_{s'} P(s'|s,a_s^*)[v(s') - w(s')] \leq \gamma \|v - w\|_\infty.$$
交换 $v, w$ 角色得 $|(T_*v)(s) - (T_*w)(s)| \leq \gamma \|v - w\|_\infty$。对 $s$ 取上确界即得 $\|T_*v - T_*w\|_\infty \leq \gamma \|v - w\|_\infty$。∎

**推论 3.1**。由Banach不动点定理，$v_\pi$ 是 $T_\pi$ 的唯一不动点，$v_*$ 是 $T_*$ 的唯一不动点。

## 4. 策略迭代与值迭代

### 4.1 策略评估

**算法（策略评估）**：给定策略 $\pi$，迭代 $v_{k+1} = T_\pi v_k$ 直到收敛。

**定理 4.1**。策略评估迭代满足
$$\|v_k - v_\pi\|_\infty \leq \gamma^k \|v_0 - v_\pi\|_\infty.$$
特别地，$v_k \to v_\pi$ 以几何速率收敛。

### 4.2 策略迭代

**算法（策略迭代）**：
1. **策略评估**：计算当前策略 $\pi_k$ 的价值函数 $v_{\pi_k}$（通过解线性方程组或迭代）。
2. **策略改进**：构造贪婪策略
$$\pi_{k+1}(s) \in \arg\max_a \left[ R(s,a) + \gamma \sum_{s'} P(s'|s,a) v_{\pi_k}(s') \right].$$
3. 若 $\pi_{k+1} = \pi_k$，停止；否则重复。

**定理 4.2（策略改进定理）**。设 $\pi'$ 为关于 $v_\pi$ 的贪婪策略，则 $v_{\pi'} \geq v_\pi$ 逐点成立，且严格不等式至少在一个状态成立除非 $\pi$ 已是最优。

**定理 4.3**。策略迭代在有限步内收敛到最优策略（因策略空间有限）。

### 4.3 值迭代

**算法（值迭代）**：迭代 $v_{k+1} = T_* v_k$，$k=0,1,2,\dots$

**定理 4.4**。值迭代满足
$$\|v_k - v_*\|_\infty \leq \gamma^k \|v_0 - v_*\|_\infty,$$
且贪婪策略 $\pi_k(s) = \arg\max_a q_{v_k}(s,a)$ 满足
$$\|v_{\pi_k} - v_*\|_\infty \leq \frac{2\gamma^k}{1-\gamma}\|v_0 - v_*\|_\infty.$$

## 5. Q学习与随机逼近

### 5.1 Q学习的收敛性

**定义 5.1（Q学习更新）**。对经验样本 $(s, a, r, s')$，Q学习的更新规则为
$$Q_{t+1}(s,a) = Q_t(s,a) + \alpha_t(s,a)\left[ r + \gamma \max_{a'} Q_t(s',a') - Q_t(s,a) \right].$$

**定理 5.1（Watkins-Dayan, 1992）**。若满足
1. $\sum_t \alpha_t(s,a) = \infty$ 且 $\sum_t \alpha_t(s,a)^2 < \infty$（Robbins-Monro条件）；
2. 所有状态-动作对被无限次访问；

则以概率1，$Q_t \to q_*$。

这是**随机逼近**（stochastic approximation）理论在强化学习中的直接应用。

## 6. 具体例子

### 例子 6.1：网格世界（Grid World）

考虑 $3 \times 3$ 网格世界，状态 $S = \{(i,j): i,j \in \{1,2,3\}\}$。智能体可采取动作 $A = \{\text{上}, \text{下}, \text{左}, \text{右}\}$。边界动作保持原地。目标状态 $(3,3)$ 给予奖励 $+1$ 并终止；陷阱状态 $(2,2)$ 给予奖励 $-1$ 并终止；其余转移奖励 $-0.04$（鼓励快速到达目标）。设 $\gamma = 0.9$。

**分析**：负的每步奖励使智能体倾向于尽快到达目标。由Bellman最优方程，对非终止状态 $(i,j)$：
$$v_*(i,j) = -0.04 + \gamma \cdot \frac{1}{4}\sum_{\text{四个方向}} v_*(\text{下一状态}).$$
数值求解（值迭代约20步收敛）得：$v_*(1,1) \approx 0.47$，$v_*(3,3)=1$，$v_*(2,2)=-1$。最优策略在左上角 $(1,1)$ 倾向于向右或向下移动，逐步靠近目标并避开陷阱。

### 例子 6.2：多臂老虎机（Multi-Armed Bandit）

多臂老虎机是状态空间为单点的退化MDP（$|S|=1$）。设 $K$ 个臂，臂 $i$ 的奖励服从 $N(\mu_i, 1)$，$\mu_i$ 未知。策略 $\pi$ 在每一步选择臂 $A_t \in \{1,\dots,K\}$，观测奖励 $R_t \sim N(\mu_{A_t}, 1)$。目标是最大化累计期望奖励（或等价地，最小化**遗憾** regret）。

**遗憾定义**：$T$ 步后的遗憾为
$$\mathrm{Regret}(T) = T\mu^* - \sum_{t=1}^T \mu_{A_t},$$
其中 $\mu^* = \max_i \mu_i$。

**定理 6.1（Lai-Robbins下界）**。任何一致性策略满足
$$\liminf_{T \to \infty} \frac{\mathrm{Regret}(T)}{\log T} \geq \sum_{i: \mu_i < \mu^*} \frac{\Delta_i}{\mathrm{KL}(\mu_i, \mu^*)},$$
其中 $\Delta_i = \mu^* - \mu_i$，$\mathrm{KL}$ 为Kullback-Leibler散度。对高斯奖励，$\mathrm{KL}(\mu_i, \mu^*) = \Delta_i^2/2$，故下界为 $\sum_i 2/\Delta_i$。

**UCB算法**：上置信界（Upper Confidence Bound）算法在每一步选择
$$A_t = \arg\max_i \left( \hat{\mu}_{i,t-1} + \sqrt{\frac{2\log t}{N_{i,t-1}}} \right),$$
其中 $\hat{\mu}_{i,t-1}$ 为臂 $i$ 到时刻 $t-1$ 的经验平均奖励，$N_{i,t-1}$ 为臂 $i$ 被拉动的次数。UCB算法的遗憾满足 $\mathrm{Regret}(T) = O(\sqrt{KT\log T})$，在最小imax意义下接近最优。
