# 强化学习数学基础

## 马尔可夫决策过程

### 定义
$(S, A, P, R, \gamma)$
- $S$：状态空间
- $A$：动作空间
- $P(s'|s,a)$：转移概率
- $R(s,a)$：奖励
- $\gamma$：折扣因子

## Bellman方程

### 价值函数
$$V^\pi(s) = \mathbb{E}_\pi[\sum_{t=0}^{\infty} \gamma^t R_t | S_0 = s]$$

### Bellman最优方程
$$V^*(s) = \max_a [R(s,a) + \gamma \sum_{s'} P(s'|s,a) V^*(s')]$$
