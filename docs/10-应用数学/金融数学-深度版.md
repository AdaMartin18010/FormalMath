## 1. 随机分析基础

### 1.1 布朗运动与随机积分

**定义 1.1** (标准布朗运动)

随机过程 $\{W_t\}_{t \geq 0}$ 是标准布朗运动，如果：
1. $W_0 = 0$
2. 独立增量：$W_t - W_s \perp \mathcal{F}_s$，$t > s$
3. 平稳增量：$W_t - W_s \sim N(0, t-s)$
4. 连续样本路径

**Itô积分**：

对于适应过程 $H_t$，Itô积分定义为：

$$\int_0^T H_t dW_t = \lim_{n \to \infty} \sum_{i=0}^{n-1} H_{t_i}(W_{t_{i+1}} - W_{t_i})$$

**Itô等距性**：

$$\mathbb{E}\left[\left(\int_0^T H_t dW_t\right)^2\right] = \mathbb{E}\left[\int_0^T H_t^2 dt\right]$$

### 1.2 Itô引理

**定理 1.1** (Itô引理)

设 $X_t$ 满足SDE：

$$dX_t = \mu(t, X_t)dt + \sigma(t, X_t)dW_t$$

对于 $C^{2,1}$ 函数 $f(t, x)$：

$$df(t, X_t) = \frac{\partial f}{\partial t}dt + \frac{\partial f}{\partial x}dX_t + \frac{1}{2}\frac{\partial^2 f}{\partial x^2}(dX_t)^2$$

其中 $(dX_t)^2 = \sigma^2 dt$。

**应用示例**：

几何布朗运动 $S_t$：

$$dS_t = \mu S_t dt + \sigma S_t dW_t$$

对 $f(S_t) = \ln S_t$ 应用Itô引理：

$$d\ln S_t = \left(\mu - \frac{\sigma^2}{2}\right)dt + \sigma dW_t$$

解得：

$$S_t = S_0 \exp\left[\left(\mu - \frac{\sigma^2}{2}\right)t + \sigma W_t\right]$$

### 1.3 鞅与Girsanov定理

**定义 1.2** (鞅)

适应过程 $M_t$ 是鞅，如果：
1. $\mathbb{E}[|M_t|] < \infty$
2. $\mathbb{E}[M_t | \mathcal{F}_s] = M_s$，$s \leq t$

**Girsanov定理**：

设 $W_t$ 是测度 $\mathbb{P}$ 下的布朗运动，定义：

$$\tilde{W}_t = W_t + \int_0^t \theta_s ds$$

则在等价测度 $\mathbb{Q}$ 下，$\tilde{W}_t$ 是布朗运动，其中：

$$\frac{d\mathbb{Q}}{d\mathbb{P}}\bigg|_{\mathcal{F}_T} = \exp\left(-\int_0^T \theta_t dW_t - \frac{1}{2}\int_0^T \theta_t^2 dt\right)$$

## 2. 衍生品定价理论

### 2.1 Black-Scholes模型

**模型假设**：
1. 无摩擦市场
2. 无套利机会
3. 标的资产服从几何布朗运动
4. 常数无风险利率和波动率

**Black-Scholes PDE**：

$$\frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} + rS\frac{\partial V}{\partial S} - rV = 0$$

**看涨期权解析解**：

$$C(S, t) = SN(d_1) - Ke^{-r(T-t)}N(d_2)$$

其中：

$$d_1 = \frac{\ln(S/K) + (r + \sigma^2/2)(T-t)}{\sigma\sqrt{T-t}}$$
$$d_2 = d_1 - \sigma\sqrt{T-t}$$

**希腊字母**：

| 希腊字母 | 公式 | 含义 |
|---------|------|------|
| Delta | $\Delta = N(d_1)$ | 价格对标的资产价格的敏感度 |
| Gamma | $\Gamma = \frac{N'(d_1)}{S\sigma\sqrt{T-t}}$ | Delta的变化率 |
| Theta | $\Theta = -\frac{SN'(d_1)\sigma}{2\sqrt{T-t}} - rKe^{-rT}N(d_2)$ | 时间衰减 |
| Vega | $\mathcal{V} = S\sqrt{T-t}N'(d_1)$ | 波动率敏感度 |
| Rho | $\rho = KTe^{-rT}N(d_2)$ | 利率敏感度 |

### 2.2 利率模型

**Vasicek模型**：

$$dr_t = a(b - r_t)dt + \sigma dW_t$$

均值回归特性，解析可处理。

**CIR模型**：

$$dr_t = a(b - r_t)dt + \sigma\sqrt{r_t}dW_t$$

保证利率非负。

**Hull-White模型**：

$$dr_t = (\theta(t) - ar_t)dt + \sigma dW_t$$

可拟合初始期限结构。

**债券定价**：

零息债券价格：

$$P(t, T) = \mathbb{E}^\mathbb{Q}\left[\exp\left(-\int_t^T r_s ds\right)\bigg|\mathcal{F}_t\right]$$

### 2.3 信用风险模型

**结构化模型**（Merton模型）：

违约发生在公司资产价值低于负债时：

$$D_T = D \cdot \mathbb{1}_{V_T < L}$$

**强度模型**：

违约时间 $\tau$ 是随机变量，生存概率：

$$P(\tau > t) = \exp\left(-\int_0^t \lambda_s ds\right)$$

**CDS定价**：

$$\text{CDS Spread} = \frac{(1-R)\int_0^T P(0,t) Q(\tau \in dt) dt}{\int_0^T P(0,t) Q(\tau > t) dt}$$

其中 $R$ 是回收率。

## 3. 风险管理

### 3.1 风险度量

**风险价值(VaR)**：

$$\text{VaR}_\alpha = \inf\{l : P(L > l) \leq 1-\alpha\}$$

计算方法：

1. **历史模拟法**：
   ```python
   def historical_var(returns, confidence=0.95):
       return np.percentile(returns, (1-confidence)*100)
   ```

2. **参数法**（方差-协方差）：
   $$\text{VaR} = \mu - \sigma \cdot z_\alpha$$

3. **蒙特卡洛模拟**：
   ```python
   def monte_carlo_var(portfolio, n_sims=100000):
       # 生成情景
       scenarios = generate_scenarios(n_sims)
       losses = portfolio.evaluate(scenarios)
       return np.percentile(losses, 5)
   ```

**期望损失(ES/CVaR)**：

$$\text{ES}_\alpha = \mathbb{E}[L | L > \text{VaR}_\alpha]$$

满足一致性风险度量的公理。

### 3.2 投资组合优化

**马科维茨均值-方差模型**：

$$\min_w \frac{1}{2}w^T\Sigma w$$
$$\text{s.t. } w^T\mu = R_{target}, \quad w^T\mathbf{1} = 1$$

**有效前沿**：

```python
def efficient_frontier(returns, n_points=100):
    mu = returns.mean()
    Sigma = returns.cov()
    n = len(mu)
    
    frontier = []
    for target_return in np.linspace(mu.min(), mu.max(), n_points):
        # 求解QP
        w = solve_qp(Sigma, target_return, mu)
        risk = np.sqrt(w.T @ Sigma @ w)
        frontier.append((risk, target_return))
    return frontier
```

**Black-Litterman模型**：

结合市场均衡收益和投资者观点：

$$E[R] = [(\tau\Sigma)^{-1} + P^T\Omega^{-1}P]^{-1}[(\tau\Sigma)^{-1}\Pi + P^T\Omega^{-1}Q]$$

## 4. 实际应用案例

### 4.1 期权做市策略

**策略描述**：

做市商通过同时报出买入价(bid)和卖出价(ask)，赚取买卖价差。

**风险管理**：

1. **Delta对冲**：
   ```python
   def delta_hedge(option_position, underlying_price, delta):
       hedge_position = -option_position * delta
       return hedge_position
   ```

2. **Gamma和Vega监控**：
   - Gamma风险：对冲再平衡成本
   - Vega风险：波动率变化风险

**实际案例**：某期权做市商

- 日均交易量：10万手
- 平均买卖价差：0.02元
- 日Delta对冲次数：50次
- 年化收益率：15%
- 最大回撤：3%

### 4.2 利率互换定价

**产品描述**：

固定利率 vs 浮动利率(Libor/SOFR)的交换。

**定价公式**：

$$V = \sum_{i=1}^n (r_{fixed} - F_i) \cdot \tau_i \cdot P(0, T_i)$$

其中 $F_i$ 是远期利率。

**实际案例**：5年期利率互换

- 名义本金：1亿美元
- 固定利率：2.5%
- 浮动利率：3个月SOFR
- 计算：
  ```python
  def irs_pricing(nominal, fixed_rate, float_rates, discount_factors, tau):
      fixed_leg = sum(fixed_rate * tau * df for df in discount_factors)
      float_leg = sum(f * tau * df for f, df in zip(float_rates, discount_factors))
      return nominal * (fixed_leg - float_leg)
  ```

### 4.3 结构化产品设计

**逆向可转换证券(RCN)**：

- 特点：高息票 + 股票挂钩
- 定价：债券部分 + 期权部分

**雪球结构**：

- 敲出条件：标的价格上涨超过障碍
- 敲入条件：标的价格下跌超过障碍
- 定价：使用蒙特卡洛模拟

```python
def snowball_pricing(S0, r, sigma, coupon, knock_out_barrier, 
                     knock_in_barrier, maturity, observation_dates, n_paths=100000):
    """雪球结构化产品定价"""
    dt = 1/252  # 交易日
    n_steps = int(maturity / dt)
    
    payoffs = []
    for _ in range(n_paths):
        S = S0
        knocked_in = False
        knocked_out = False
        knock_out_date = None
        
        for t in observation_dates:
            step = int(t / dt)
            Z = np.random.randn(n_steps)
            for i in range(step):
                S *= np.exp((r - 0.5*sigma**2)*dt + sigma*np.sqrt(dt)*Z[i])
            
            if S >= knock_out_barrier * S0:
                knocked_out = True
                knock_out_date = t
                break
            elif S <= knock_in_barrier * S0:
                knocked_in = True
        
        # 计算收益
        if knocked_out:
            payoff = 1 + coupon * knock_out_date / maturity
        elif not knocked_in:
            payoff = 1 + coupon
        else:
            payoff = S / S0  # 亏损
        
        payoffs.append(payoff)
    
    return np.exp(-r*maturity) * np.mean(payoffs)
```
