# 31.05-极值组合学

极值组合学研究离散结构在满足特定约束条件下的极值问题，是现代组合数学中最活跃的领域之一。

## 🎯 核心主题

### 极值集合论
- **Sperner定理**：反链的最大大小
- **Erdős-Ko-Rado定理**：相交族的极值
- **Kruskal-Katona定理**：阴影的最小大小

### Ramsey理论
- **Ramsey定理**：$R(s, t)$ 的存在性
- **van der Waerden定理**：算术级数
- **Hales-Jewett定理**：组合线

### 概率方法
- **基本计数**：期望与存在性
- **Lovász局部引理**：依赖条件下的存在性
- **二阶矩方法**：方差分析

### Sunflower与Delta系统
- **Sunflower引理**：Erdős-Rado
- **推广与应用**：集合系统的结构
