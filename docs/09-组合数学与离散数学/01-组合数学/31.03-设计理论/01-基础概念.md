# 01-基础概念

## 一、区组设计基础

### 1.1 平衡不完全区组设计 (BIBD)

**定义 1.1** (BIBD)：设 $X$ 为 $v$ 元集，$\mathcal{B}$ 为 $X$ 的 $b$ 个 $k$-子集（称为区组）的族。若满足：
1. 每对元素恰出现在 $\lambda$ 个区组中
2. $k < v$（不完全）

则称 $(X, \mathcal{B})$ 为 $(v, b, r, k, \lambda)$-设计或 $2$-$(v, k, \lambda)$ 设计。

其中 $r$ 为每个元素出现的区组数。

**基本方程**：

**定理 1.2**：BIBD满足：
1. **复制方程**：$bk = vr$
2. **关联方程**：$r(k-1) = \lambda(v-1)$

**证明**：
- 计数 $(x, B)$ 对，$x \in B$：左边 $bk$，右边 $vr$
- 固定 $x$，计数 $(y, B)$，$y \neq x$，$x,y \in B$：左边 $r(k-1)$，右边 $\lambda(v-1)$

### 1.2 对称设计

**定义 1.3** (对称设计)：$b = v$（从而 $r = k$）的BIBD。

**参数**：$(v, k, \lambda)$，满足 $k(k-1) = \lambda(v-1)$。

**定理 1.4**：对称设计中，任意两不同区组恰交于 $\lambda$ 个点。

**证明**：设 $A$ 为关联矩阵，$AA^T = (r-\lambda)I + \lambda J$。

对称设计中 $A$ 可逆，故 $A^T A = AA^T$，即区组交集也为常数。

## 二、Steiner系统

### 2.1 定义

**定义 2.1** ($t$-$(v, k, \lambda)$ 设计)：$v$ 元集上 $k$-子集族，使每 $t$-子集恰含于 $\lambda$ 个区组。

**定义 2.2** (Steiner系统)：$\lambda = 1$ 的 $t$-$(v, k, 1)$ 设计，记为 $S(t, k, v)$。

**特例**：
- $S(2, 3, v)$：Steiner三元系，记为 STS($v$)
- $S(3, 4, v)$：Steiner四元系
- $S(5, 6, 12)$ 和 $S(5, 8, 24)$：Witt设计

### 2.2 Steiner三元系

**定理 2.3** (Kirkman)：STS($v$) 存在当且仅当 $v \equiv 1$ 或 $3 \pmod{6}$。

**必要条件证明**：
- $r = \frac{v-1}{2}$ 为整数，故 $v$ 奇
- $b = \frac{vr}{3} = \frac{v(v-1)}{6}$ 为整数，故 $v \equiv 1, 3 \pmod{6}$

**构作**：
- $v = 3$：平凡
- $v = 7$：Fano平面
- 递归构作：由 STS($v$) 得 STS($2v+1$)

### 2.3 Kirkman三元系

**定义 2.4** (Kirkman三元系)：可分解的STS($v$)，即区组可划分为平行类，每类覆盖所有点。

**Kirkman女学生问题**：15女生每日排成5行3人，7日中每对恰同行一次。

**定理 2.5**：Kirkman三元系存在当且仅当 $v \equiv 3 \pmod{6}$。

## 三、有限几何

### 3.1 射影平面

**定义 3.1** (射影平面)：点和线的有限关联结构，满足：
1. 任意两不同点恰在一条线上
2. 任意两不同线恰交于一点
3. 存在四点，无三点共线

**定理 3.2**：射影平面存在阶 $q$，使得：
- 点数 = 线数 = $q^2 + q + 1$
- 每线点数 = 每点线数 = $q + 1$

**证明**：设某线有 $q+1$ 点。由公理1，其他点通过该线外点与该线点连线确定。计数得点数 $1 + (q+1)q = q^2+q+1$。

**定理 3.3**：$q$ 阶射影平面存在当 $q = p^n$（素数幂），由 $PG(2, q)$ 给出。

**存在性问题**：非素数幂阶射影平面是否存在？（开放问题，$q=10,12$ 不存在）

### 3.2 仿射平面

**定义 3.4** (仿射平面)：点和线的有限关联结构，满足：
1. 任意两不同点恰在一条线上
2. 给定线和线外点，恰有一条平行线（不相交）
3. 存在三点不共线

**定理 3.5**：$q$ 阶仿射平面有 $q^2$ 点，$q^2+q$ 线，每线 $q$ 点。

**构造**：从射影平面删除一条线及其上的点。

### 3.3 射影空间

**定义 3.6** ($PG(n, q)$)：$GF(q)^{n+1}$ 中1维子空间为点，2维子空间为线，等等。

**参数**：
- 点数：$\frac{q^{n+1}-1}{q-1}$
- 超平面数：同点数
- 每超平面点数：$\frac{q^n-1}{q-1}$

## 四、拉丁方

### 4.1 基本定义

**定义 4.1** (拉丁方)：$n \times n$ 数组，每行每列含 $1, 2, \ldots, n$ 各一次。

**定义 4.2** (正交拉丁方)：两个拉丁方 $L_1, L_2$ 正交，如果所有有序对 $(L_1(i,j), L_2(i,j))$ 不同。

**定义 4.3** (MOLS)：Mutually Orthogonal Latin Squares，两两正交的拉丁方组。

**定理 4.4**：$n$ 阶MOLS最多有 $n-1$ 个。

**证明**：不妨设第一行都为 $(1, 2, \ldots, n)$。则其他拉丁方 $(1,1)$ 位置不能为1。$n-1$ 个不同值放入 $n-1$ 个位置。

### 4.2 Euler猜想

**问题**：36军官问题 - 是否存在两个正交的6阶拉丁方？

**Euler猜想**：$n \equiv 2 \pmod{4}$ 时不存在一对正交拉丁方。

**定理 4.5** (Bose-Shrikhande-Parker, 1960)：Euler猜想在 $n > 6$ 时错误。对所有 $n \neq 2, 6$，存在一对正交拉丁方。

**定理 4.6**：$n = p^k$（素数幂）时存在 $n-1$ 个MOLS。

**构造**：利用 $GF(n)$。设 $\alpha_1, \ldots, \alpha_{n-1}$ 为非零元，定义 $L_k(i,j) = i + \alpha_k j$。

## 五、Hadamard矩阵

### 5.1 定义与性质

**定义 5.1** (Hadamard矩阵)：$n \times n$ 矩阵 $H$，元素为 $\pm 1$，满足 $HH^T = nI$。

**定理 5.2**：Hadamard矩阵的阶 $n = 1, 2$ 或 $n \equiv 0 \pmod{4}$。

**证明**：$n \geq 3$ 时，可设第一行全1。其他行正交，故各半1半-1，$n$ 偶。

三行两两正交，分析前三列得 $n \equiv 0 \pmod{4}$。

**Hadamard猜想**：对所有 $n \equiv 0 \pmod{4}$，Hadamard矩阵存在。

**现状**：对所有 $n < 668$，以及许多无穷类已知存在。

### 5.2 与设计的联系

**定理 5.3**：$n = 4t$ 阶Hadamard矩阵存在等价于 $(4t-1, 2t-1, t-1)$-对称设计存在。

**构造**：正规化 $H$（第一行第一列为1），删除第一行第一列，$\frac{1}{2}(J+H')$ 为关联矩阵。
