# 03-实战问题

## 问题集一：基础设计构作

### 问题 1.1

**题目**：构作7阶Steiner三元系（Fano平面），列出所有区组。

**解答**：

点集：$\{0, 1, 2, 3, 4, 5, 6\}$

区组（直线）：
$$\{0, 1, 3\}, \{1, 2, 4\}, \{2, 3, 5\}, \{3, 4, 6\}, \{4, 5, 0\}, \{5, 6, 1\}, \{6, 0, 2\}$$

**几何表示**：

```

      0
     /|\

    6-1-4

    |\|/|

    2-3-5

```

或标准表示：三角形三边中点、重心与顶点连线。

---

### 问题 1.2

**题目**：证明不存在 $(16, 6, 1)$-设计。

**证明**：

若存在，参数满足：
- $r(k-1) = \lambda(v-1) \Rightarrow r \cdot 5 = 15 \Rightarrow r = 3$
- $bk = vr \Rightarrow b \cdot 6 = 16 \cdot 3 = 48 \Rightarrow b = 8$

但由Fisher不等式，$b \geq v = 16$，矛盾。

因此不存在。

---

### 问题 1.3

**题目**：验证 $(15, 8, 4)$-对称设计满足Bruck-Ryser-Chowla条件。

**解答**：

参数：$v = 15, k = 8, \lambda = 4$

检验 $k(k-1) = \lambda(v-1)$：
$$8 \cdot 7 = 56, \quad 4 \cdot 14 = 56 \quad \checkmark$$

BRC条件：$v = 15 \equiv 3 \pmod{4}$，需 $x^2 = 4y^2 - 4z^2 = 4(y^2-z^2)$ 有非零解。

即 $x^2 = 4(y^2-z^2)$，取 $y = 1, z = 0, x = 2$ 即可。

**注**：实际上此设计存在，为 $PG(3, 2)$ 的补。

## 问题集二：有限几何应用

### 问题 2.1

**题目**：证明 $PG(2, 2)$（Fano平面）是唯一的2阶射影平面。

**证明**：

2阶射影平面参数：$v = 2^2 + 2 + 1 = 7$，$k = 3$。

即7点7线，每线3点，每点3线。

固定一点 $P$，它在与3条线。每条线上还有2点，共6点，即所有其他点。

设这三线为 $L_1, L_2, L_3$，$L_i = \{P, A_i, B_i\}$。

线 $A_1 A_2$ 必过第三点，不能是 $P$（否则 $A_1, A_2$ 在 $L_1, L_2$ 上），故必为某 $B_i$。

通过检查，必为 $B_3$。类似确定所有线，结构唯一。

---

### 问题 2.2

**题目**：计算 $PG(3, 2)$ 中的点、线、平面数。

**解答**：

$PG(3, 2)$ 为 $GF(2)^4$ 中1维子空间。

**点数**：$\frac{2^4-1}{2-1} = 15$

**线数**：每条线有 $\frac{2^2-1}{2-1} = 3$ 点，每点与其他14点连线，每线被3点计数。
$$\text{线数} = \frac{15 \cdot 14 / 2}{3} = 35$$

或：2维子空间数 = $[4 \atop 2]_2 = \frac{(2^4-1)(2^3-1)}{(2^2-1)(2-1)} = 35$

**平面数**：3维子空间数 = $[4 \atop 3]_2 = [4 \atop 1]_2 = 15$

每平面为 $PG(2, 2)$，有7点7线。

---

### 问题 2.3

**题目**：证明仿射平面 $AG(2, q)$ 可分解为 $q+1$ 个平行类，每类 $q$ 条线。

**证明**：

$AG(2, q)$ 有 $q^2$ 点，$q^2+q$ 线，每线 $q$ 点。

方向：斜率 $m \in GF(q)$ 或垂直（$\infty$），共 $q+1$ 个方向。

同一方向的线不相交（平行），形成平行类。

每类线数：$\frac{q^2}{q} = q$ 条。

## 问题集三：拉丁方问题

### 问题 3.1

**题目**：构作3阶MOLS完备集（2个）。

**解答**：

利用 $GF(3) = \{0, 1, 2\}$，非零元为 $1, 2$。

$L_1(i,j) = i + j \pmod{3}$：

```

0 1 2
1 2 0
2 0 1

```

$L_2(i,j) = i + 2j \pmod{3}$：

```

0 2 1
1 0 2
2 1 0

```

正交性验证：有序对 $(0,0), (1,2), (2,1), (1,1), (2,0), (0,2), (2,2), (0,1), (1,0)$ 各不相同。

---

### 问题 3.2

**题目**：证明不存在一对正交的6阶拉丁方。

**注**：这是Euler三十六军官问题的否定答案。

**证明概要** (Tarry, 1901)：

穷举所有6阶拉丁方，检查无正交对。

或利用：若存在 $n-1$ 个MOLS，则存在 $n$ 阶射影平面。

对 $n=6$，不存在6阶射影平面（Bruck-Ryser-Chowla），故不存在5个MOLS。

实际上连2个都不存在，需更精细分析。

## 问题集四：高级构作

### 问题 4.1

**题目**：利用Paley构作11阶对称设计。

**解答**：

$GF(11)$ 的二次剩余：$1^2=1, 2^2=4, 3^2=9, 4^2=5, 5^2=3$。

二次剩余集：$QR = \{1, 3, 4, 5, 9\}$，大小 $k = 5 = (11-1)/2$。

差集验证：对任意 $d \neq 0$，方程 $x - y = d$（$x, y \in QR$）恰有 $\lambda = 2$ 解。

例如 $d = 1$：$4-3=1, 5-4=1$。

对称设计参数：$(11, 5, 2)$。

---

### 问题 4.2

**题目**：由Hadamard矩阵构作 $(11, 5, 2)$-设计的对偶 $(11, 6, 3)$-设计。

**解答**：

12阶Hadamard矩阵正规化后，删除第一行第一列得11阶矩阵 $H'$。

$\frac{1}{2}(J + H')$ 为 $(11, 6, 3)$-设计的关联矩阵。

区组大小为6（每行6个1），每对点恰在3个区组中。

## 问题集五：研究前沿

### 问题 5.1 (子平面问题)

**问题**：$q$ 阶射影平面中子平面的可能阶数。

**定理** (Bruck)：若 $q$ 阶射影平面有 $r$ 阶真子平面，则 $r^2 \leq q$ 且 $r^2 = q$ 或 $r^2 + r \leq q$。

**猜想**：$PG(2, q)$ 的 $r$ 阶子平面仅当 $r$ 整除 $q$（对 $q = p^n$）。

---

### 问题 5.2 (存在性问题)

**问题**：是否存在非Desargues有限射影平面？

**答案**：是的。Albert, Veblen-Wedderburn 构作了非Desargues有限平面。

最小例子：阶为9的非Desargues平面存在。
