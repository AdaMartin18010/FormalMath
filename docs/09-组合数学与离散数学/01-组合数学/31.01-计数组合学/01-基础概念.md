# 01-基础概念

## 一、排列与组合

### 1.1 基本定义

**定义 1.1** (排列)：从 $n$ 个不同元素中取出 $k$ 个元素的有序排列数记为 $P(n,k)$ 或 $A_n^k$：

$$P(n,k) = n(n-1)(n-2)\cdots(n-k+1) = \frac{n!}{(n-k)!}$$

**定义 1.2** (组合)：从 $n$ 个不同元素中取出 $k$ 个元素的无序组合数记为 $\binom{n}{k}$ 或 $C_n^k$：

$$\binom{n}{k} = \frac{n!}{k!(n-k)!}$$

### 1.2 有重复排列与组合

**定义 1.3** (多重集排列)：设有 $n$ 个元素，其中第 $i$ 类元素有 $n_i$ 个（$\sum n_i = n$），则这些元素的全排列数为：

$$\frac{n!}{n_1! n_2! \cdots n_k!}$$

**定义 1.4** (有重复组合)：从 $n$ 个不同元素中可重复地选取 $k$ 个元素的组合数：

$$\binom{n+k-1}{k}$$

### 1.3 圆排列

**定义 1.5** (圆排列)：$n$ 个不同元素的圆排列数为 $(n-1)!$。

**证明**：线性排列有 $n!$ 种，每个圆排列对应 $n$ 个线性排列（旋转等价），故圆排列数为 $\frac{n!}{n} = (n-1)!$。

## 二、二项式定理

### 2.1 牛顿二项式定理

**定理 2.1** (二项式定理)：对任意非负整数 $n$，有

$$(x+y)^n = \sum_{k=0}^{n} \binom{n}{k} x^{n-k} y^k$$

**推论 2.2**：
- $\sum_{k=0}^{n} \binom{n}{k} = 2^n$
- $\sum_{k=0}^{n} (-1)^k \binom{n}{k} = 0$ （$n \geq 1$）
- $\sum_{k=0}^{n} k\binom{n}{k} = n2^{n-1}$

### 2.2 多项式定理

**定理 2.3** (多项式定理)：

$$(x_1 + x_2 + \cdots + x_m)^n = \sum_{n_1+\cdots+n_m=n} \binom{n}{n_1,n_2,\ldots,n_m} x_1^{n_1} x_2^{n_2} \cdots x_m^{n_m}$$

其中多重组合系数：

$$\binom{n}{n_1,n_2,\ldots,n_m} = \frac{n!}{n_1! n_2! \cdots n_m!}$$

### 2.3 范德蒙德恒等式

**定理 2.4** (范德蒙德恒等式)：

$$\binom{m+n}{k} = \sum_{j=0}^{k} \binom{m}{j}\binom{n}{k-j}$$

**证明**：考虑从 $m$ 个红球和 $n$ 个蓝球中选取 $k$ 个球的总数。

## 三、容斥原理

### 3.1 基本原理

**定理 3.1** (容斥原理)：设 $A_1, A_2, \ldots, A_n$ 为有限集合，则

$$\left|\bigcup_{i=1}^{n} A_i\right| = \sum_{i}|A_i| - \sum_{i<j}|A_i \cap A_j| + \sum_{i<j<k}|A_i \cap A_j \cap A_k| - \cdots + (-1)^{n-1}\left|\bigcap_{i=1}^{n} A_i\right|$$

等价地：

$$\left|\bigcup_{i=1}^{n} A_i\right| = \sum_{\emptyset \neq I \subseteq [n]} (-1)^{|I|-1}\left|\bigcap_{i \in I} A_i\right|$$

### 3.2 补集形式

**定理 3.2** (补集容斥原理)：设全集为 $U$，则

$$\left|\bigcap_{i=1}^{n} \overline{A_i}\right| = |U| - \sum_{i}|A_i| + \sum_{i<j}|A_i \cap A_j| - \cdots + (-1)^n\left|\bigcap_{i=1}^{n} A_i\right|$$

### 3.3 应用：错位排列

**定义 3.3** (错位排列/derangement)：$n$ 个元素的全排列中，没有任何元素保持原位的排列数记为 $D_n$ 或 $!n$。

**定理 3.4**：

$$D_n = n! \sum_{k=0}^{n} \frac{(-1)^k}{k!} = \left[\frac{n!}{e}\right]$$

其中 $[\cdot]$ 表示最接近的整数。

**证明**：设 $A_i$ 为第 $i$ 个元素在原位的排列集合，由容斥原理：

$$D_n = n! - \binom{n}{1}(n-1)! + \binom{n}{2}(n-2)! - \cdots + (-1)^n\binom{n}{n}0!$$

$$= n!\left(1 - \frac{1}{1!} + \frac{1}{2!} - \cdots + \frac{(-1)^n}{n!}\right)$$

## 四、经典数列

### 4.1 二项式系数

**定义 4.1** (帕斯卡三角形)：
$$\binom{n}{k} = \binom{n-1}{k-1} + \binom{n-1}{k}$$

**定理 4.2** (中心二项式系数渐近)：
$$\binom{2n}{n} \sim \frac{4^n}{\sqrt{\pi n}}$$

### 4.2 Catalan数

**定义 4.3** (Catalan数)：
$$C_n = \frac{1}{n+1}\binom{2n}{n} = \binom{2n}{n} - \binom{2n}{n+1}$$

**组合解释**：
- 有效的括号序列数
- 二叉搜索树的结构数
- 不交叉划分数
- 格路问题：从 $(0,0)$ 到 $(n,n)$ 不越过对角线的路径数

**递推关系**：
$$C_0 = 1, \quad C_{n+1} = \sum_{i=0}^{n} C_i C_{n-i}$$

### 4.3 Stirling数

**定义 4.4** (第一类Stirling数)：$s(n,k)$ 表示将 $n$ 个元素排成 $k$ 个轮换的方式数。

**递推关系**：
$$s(n,k) = s(n-1,k-1) + (n-1)s(n-1,k)$$

**定义 4.5** (第二类Stirling数)：$S(n,k)$ 或 $\left\{n \atop k\right\}$ 表示将 $n$ 个元素划分为 $k$ 个非空子集的方式数。

**递推关系**：
$$S(n,k) = S(n-1,k-1) + kS(n-1,k)$$

**显式公式**：
$$S(n,k) = \frac{1}{k!}\sum_{j=0}^{k}(-1)^{k-j}\binom{k}{j}j^n$$

### 4.4 Bell数

**定义 4.6** (Bell数)：$B_n$ 表示 $n$ 元素集合的划分数。

$$B_n = \sum_{k=0}^{n} S(n,k)$$

**递推关系** (Dobinski公式)：
$$B_n = \frac{1}{e}\sum_{k=0}^{\infty}\frac{k^n}{k!}$$

## 五、形式幂级数基础

### 5.1 定义

**定义 5.1** (形式幂级数)：序列 $(a_n)_{n \geq 0}$ 的形式幂级数定义为

$$A(x) = \sum_{n=0}^{\infty} a_n x^n$$

不考虑收敛性，仅作为代数对象处理。

### 5.2 基本运算

**定义 5.2** (运算)：
- **加法**：$(A+B)(x) = \sum_{n}(a_n + b_n)x^n$
- **乘法 (Cauchy积)**：$(A \cdot B)(x) = \sum_{n}\left(\sum_{k=0}^{n}a_k b_{n-k}\right)x^n$
- **形式导数**：$A'(x) = \sum_{n=1}^{\infty} n a_n x^{n-1}$

### 5.3 几何级数

**定理 5.3**：
$$\frac{1}{1-x} = \sum_{n=0}^{\infty} x^n$$

**推论 5.4**：
$$\frac{1}{(1-x)^k} = \sum_{n=0}^{\infty} \binom{n+k-1}{k-1} x^n$$
