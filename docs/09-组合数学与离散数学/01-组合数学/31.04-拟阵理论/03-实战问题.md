# 03-实战问题

## 问题集一：拟阵基础验证

### 问题 1.1

**题目**：验证均匀拟阵 $U_{2,4}$ 满足独立集公理。

**解答**：

$E = \{1, 2, 3, 4\}$，$\mathcal{I} = \{X \subseteq E : |X| \leq 2\}$。

**(I1)** $\emptyset \in \mathcal{I}$：$|\emptyset| = 0 \leq 2$ $\checkmark$

**(I2)** 遗传性：$X \in \mathcal{I}, Y \subseteq X \Rightarrow |Y| \leq |X| \leq 2 \Rightarrow Y \in \mathcal{I}$ $\checkmark$

**(I3)** 交换性：$X, Y \in \mathcal{I}, |X| < |Y|$。

若 $|X| = 0, |Y| = 1$：取 $Y$ 中元素即可。

若 $|X| = 1, |Y| = 2$：$X = \{x\}, Y = \{y_1, y_2\}$。

若 $x \in Y$，$X \subseteq Y$，但 $|X| < |Y|$ 允许 $X$ 外元素。

若 $x \notin Y$，取 $y_1 \in Y \setminus X$，$|X \cup \{y_1\}| = 2 \leq 2$ $\checkmark$

### 问题 1.2

**题目**：完全图 $K_4$ 的圈拟阵的基、圈、秩函数。

**解答**：

$K_4$ 有6条边，4个顶点。

**基**：生成树，每棵树有 $4-1 = 3$ 条边。

基数为 $\binom{4}{2} = 16$ 个（Cayley：$4^{4-2} = 16$）。

**圈**：3-圈和4-圈。

- 3-圈：4个（每个顶点对应一个）
- 4-圈：3个（对角线配对）

**秩函数**：$r(X) = |V(X)| - c(X)$，其中 $c(X)$ 为 $(V, X)$ 的连通分支数。

### 问题 1.3

**题目**：证明 $U_{2,4}$ 不是图拟阵。

**证明**：

$U_{2,4}$ 有4元素，秩2，所有2-子集为基。

假设 $U_{2,4} = M(G)$，则 $G$ 有4边，生成树2边，故 $|V| = 3$。

$K_3$ 有3边，不够。

添加一条平行边得多重图，但平行边形成2-圈，$U_{2,4}$ 无2-圈。

矛盾。

## 问题集二：对偶与Minors

### 问题 2.1

**题目**：求 $U_{r,n}$ 的对偶。

**解答**：

$U_{r,n}^* = U_{n-r, n}$。

**证明**：$B$ 为 $U_{r,n}$ 的基 $\Leftrightarrow$ $|B| = r$。

$E \setminus B$ 大小为 $n-r$，为 $U_{r,n}^*$ 的基。

故 $U_{r,n}^* = U_{n-r, n}$。

---

### 问题 2.2

**题目**：证明 $M(G)^* = M(G^*)$ 对连通平面图 $G$。

**解答**：

$G^*$ 的边与 $G$ 的边一一对应（对偶边）。

$G^*$ 的圈 $\Leftrightarrow$ $G$ 的割（Jordan曲线定理）。

$M(G^*)$ 的圈 = $G^*$ 的圈 = $G$ 的割 = $M(G)$ 的余圈 = $M(G)^*$ 的圈。

---

### 问题 2.3

**题目**：$K_5$ 的圈拟阵在 $GF(2)$ 上可表示，但在其他哪些域上可表示？

**解答**：

$M(K_5)$ 是图拟阵，故在所有域上可表示。

实际上，图拟阵在任何域上可表示（定向后全单模或一般表示）。

## 问题集三：拟阵交算法

### 问题 3.1

**题目**：用拟阵交求二分图 $K_{2,3}$ 的最大匹配。

**解答**：

$E = \{e_1, e_2, e_3, e_4, e_5, e_6\}$（6条边）。

$M_1$：左侧每点至多一条边（划分拟阵，划分由左侧顶点给出）。

$M_2$：右侧每点至多一条边。

最大公共独立集 = 最大匹配 = $\min(2, 3) = 2$。

**算法执行**：
1. 初始 $I = \emptyset$
2. 找增广路，添加边
3. 直到无法增广

---

### 问题 3.2

**题目**：证明Edmonds拟阵交定理蕴含Hall婚姻定理。

**证明**：

设 $G = (U \cup V, E)$ 为二部图。

$M_1$：$U$ 上划分拟阵（每点至多一条边）。

$M_2$：$V$ 上划分拟阵。

拟阵交定理：
$$\max |I| = \min_{A \subseteq E} (r_1(A) + r_2(E \setminus A))$$

对 $S \subseteq U$，设 $A$ 为与 $S$ 关联的边。

$r_1(A) = |S|$，$r_2(E \setminus A) \geq |N(U \setminus S)|$。

Hall条件：对所有 $S \subseteq U$，$|S| \leq |N(S)|$。

等价于 $\max |I| = |U|$。

## 问题集四：多面体优化

### 问题 4.1

**题目**：均匀拟阵 $U_{r,n}$ 的独立集多面体。

**解答**：

$P_{\mathcal{I}} = \{x \in \mathbb{R}_+^n : \sum_{i=1}^n x_i \leq r, 0 \leq x_i \leq 1\}$

秩函数：$r(S) = \min(|S|, r)$。

不等式：$x(S) \leq \min(|S|, r)$。

对 $|S| \leq r$：$x(S) \leq |S|$ 由 $x_i \leq 1$ 蕴含。

对 $|S| > r$：$x(S) \leq r$ 为有效约束。

---

### 问题 4.2

**题目**：图拟阵的贪婪算法即Kruskal算法。

**证明**：

图拟阵的基 = 生成树。

权重排序后，贪婪选择不构成圈的边，正是Kruskal算法。

由拟阵贪婪定理，此算法最优。

## 问题集五：可表示性

### 问题 5.1

**题目**：验证Fano拟阵 $F_7$ 在 $GF(2)$ 上可表示。

**解答**：

Fano平面关联矩阵（$7 \times 7$，点为行，线为列）：

标准表示：删除一点，得 $GF(2)$ 上 $3 \times 7$ 矩阵，秩3。

每3点共线 $\Leftrightarrow$ 列线性相关。

$F_7$ 在特征2域可表示，但在特征 $\neq 2$ 不可（7个点无法在一般位置）。

---

### 问题 5.2

**题目**：证明 $U_{2,4}$ 不是正则拟阵。

**证明**：

$U_{2,4}$ 在 $GF(2)$ 上不可表示（4点共线需要特征2中 $1+1=0$）。

实际上，$U_{2,4}$ 在所有域上可表示当且仅当域有至少3个元素。

$GF(2)$ 只有2个元素，无法放置4个一般位置点（任意3点不共线需要至少3个斜率）。
