# 组合数学与离散数学基础理论

## 1. 引言

组合数学（Combinatorics）与离散数学（Discrete Mathematics）是研究**有限或可数结构**的数学分支，与计算机科学、信息论、统计物理以及现代优化理论有着深刻的联系。组合数学关注"如何计数与构造"，其核心问题包括排列组合、图的存在性与极值性质、设计理论以及代数组合；离散数学则是更广泛的领域，涵盖逻辑、集合论、图论、算法分析与组合优化。

本文从严格的集合论视角出发，系统阐述组合计数原理、图论基础以及生成函数方法，为后续学习极值组合、概率方法以及代数图论奠定坚实基础。

## 2. 严格数学定义

### 2.1 组合计数基础

**定义 2.1（排列与组合）**
设 $S$ 为 $n$ 元集合。

- $S$ 的**$k$-排列**数（有序选取 $k$ 个不同元素）：
  $$P(n, k) = \frac{n!}{(n-k)!}$$

- $S$ 的**$k$-组合**数（无序选取 $k$ 个不同元素）：
  $$\binom{n}{k} = \frac{n!}{k!(n-k)!}$$

**定义 2.2（生成函数）**
序列 $(a_n)_{n \geq 0}$ 的**普通生成函数**（OGF）定义为形式幂级数
$$A(x) = \sum_{n=0}^\infty a_n x^n$$
**指数生成函数**（EGF）定义为
$$\hat{A}(x) = \sum_{n=0}^\infty a_n \frac{x^n}{n!}$$

### 2.2 图论基础

**定义 2.3（图）**
一个**图** $G$ 是一个二元组 $(V, E)$，其中 $V$ 为有限非空**顶点集**，$E \subseteq \binom{V}{2}$ 为**边集**。若允许 $E$ 为多重集，则称 $G$ 为**多重图**；若 $E$ 包含有序对，则称 $G$ 为**有向图**。

**定义 2.4（度与握手引理）**
顶点 $v \in V$ 的**度** $d(v)$ 为与 $v$ 关联的边数（环计为2）。**握手引理**断言：
$$\sum_{v \in V} d(v) = 2|E|$$

**定义 2.5（树与森林）**
连通无圈图称为**树**。树的等价刻画包括：
1. 连通且 $|E| = |V| - 1$
2. 任意两点间存在唯一简单路径
3. 无圈但添加任一边产生唯一圈

**定义 2.6（二部图）**
图 $G = (V, E)$ 称为**二部图**，若存在顶点划分 $V = A \sqcup B$ 使得每条边连接 $A$ 与 $B$ 中的顶点。等价条件：$G$ 不含奇圈（Kőnig定理）。

## 3. 核心定理与证明

### 定理 3.1（Cayley公式）

$n$ 个标号顶点上的不同树共有 $n^{n-2}$ 棵。

**证明（Prüfer编码法）**：建立树与序列之间的一一对应。

对 $n$ 个标号顶点 $\{1, 2, \ldots, n\}$ 上的树 $T$，构造长度为 $n-2$ 的 **Prüfer序列** $P(T) = (a_1, a_2, \ldots, a_{n-2})$：

**编码算法**：
对 $i = 1, \ldots, n-2$：
1. 找出 $T$ 中编号最小的叶子顶点 $v$
2. 令 $a_i$ 为 $v$ 的唯一邻接点
3. 从 $T$ 中删除 $v$ 及其关联边

**解码算法**：
给定序列 $(a_1, \ldots, a_{n-2})$，令 $S = \{1, \ldots, n\}$：
对 $i = 1, \ldots, n-2$：
1. 找出 $S$ 中最小且不在 $\{a_i, a_{i+1}, \ldots, a_{n-2}\}$ 中的元素 $x$
2. 连接 $x$ 与 $a_i$
3. 从 $S$ 中删除 $x$
最后连接 $S$ 中剩余的两个元素。

编码与解码互为逆操作，故树与长度为 $n-2$、取值于 $\{1, \ldots, n\}$ 的序列一一对应。序列总数为 $n^{n-2}$。$\square$

### 定理 3.2（Kőnig-Egerváry定理）

在二部图 $G = (A \sqcup B, E)$ 中，最大匹配的大小等于最小顶点覆盖的大小：
$$\nu(G) = \tau(G)$$

**证明**：设 $M$ 为最大匹配。构造顶点覆盖 $C$ 如下：

1. 令 $U \subseteq A$ 为在 $M$ 下未被匹配的顶点集
2. 令 $Z$ 为从 $U$ 出发经交错路径（alternating path）可达的所有顶点集
3. 令 $K = (A \setminus Z) \cup (B \cap Z)$

**断言**：$K$ 是顶点覆盖。若存在边 $ab \in E$ 未被 $K$ 覆盖，则 $a \in Z$ 且 $b \notin Z$。但 $a \in Z$ 意味着存在从 $U$ 到 $a$ 的交错路径，此路径可经边 $ab$ 延伸到 $b$，故 $b \in Z$，矛盾。

**断言**：$|K| = |M|$。$A \setminus Z$ 中每个顶点均被 $M$ 匹配（否则属于 $U \subseteq Z$），且其匹配伙伴在 $B \setminus Z$（若在 $B \cap Z$ 则可延伸交错路径）。$B \cap Z$ 中每个顶点均被 $M$ 匹配（否则得到增广路径，与 $M$ 最大性矛盾）。故
$$|K| = |A \setminus Z| + |B \cap Z| = |M|$$
$\square$

### 定理 3.3（二项式定理与组合恒等式）

对任意非负整数 $n$ 和交换元 $x, y$：
$$(x + y)^n = \sum_{k=0}^n \binom{n}{k} x^k y^{n-k}$$

**证明**：展开 $(x+y)^n = (x+y)(x+y)\cdots(x+y)$（$n$ 个因子）。选择 $k$ 个因子取 $x$，其余 $n-k$ 个取 $y$，贡献项 $x^k y^{n-k}$。选择方式恰有 $\binom{n}{k}$ 种。由加法原理求和即得。$\square$

## 4. 详细例子

### 例 4.1：Catalan数的生成函数推导

Catalan数 $C_n$ 计数 $n+1$ 个叶子的有根二叉树、合法的括号序列以及不穿过对角线的格路等多种组合结构。递推关系为：
$$C_0 = 1, \quad C_{n+1} = \sum_{k=0}^n C_k C_{n-k}$$

设生成函数 $C(x) = \sum_{n=0}^\infty C_n x^n$。由递推：
$$C(x) = 1 + x \sum_{n=0}^\infty \sum_{k=0}^n C_k C_{n-k} x^n = 1 + x C(x)^2$$

解二次方程 $x C^2 - C + 1 = 0$：
$$C(x) = \frac{1 - \sqrt{1 - 4x}}{2x}$$

由广义二项式定理：
$$(1 - 4x)^{1/2} = \sum_{n=0}^\infty \binom{1/2}{n} (-4x)^n = 1 - 2x - 2x^2 - 4x^3 - \cdots$$

提取系数得：
$$C_n = \frac{1}{n+1}\binom{2n}{n}$$

前几项：$C_0 = 1, C_1 = 1, C_2 = 2, C_3 = 5, C_4 = 14, C_5 = 42$。

### 例 4.2：Turán定理的极值应用

**Turán定理**：不含 $K_{r+1}$ 的 $n$ 顶点图的最大边数为
$$\text{ex}(n, K_{r+1}) = \left(1 - \frac{1}{r}\right)\frac{n^2}{2}$$
当 $r \mid n$ 时由完全 $r$-部 Turán 图 $T_r(n)$ 达到。

对 $r = 2$（即不含三角形），$\text{ex}(n, K_3) \leq \frac{n^2}{4}$。

**证明（归纳法）**：对 $n$ 归纳。$n = 1, 2$ 显然。设对小于 $n$ 成立。取 $n$ 顶点无三角形图 $G$ 的最大度顶点 $v$，$d(v) = \Delta$。$v$ 的邻域 $N(v)$ 是独立集（否则形成三角形）。故
$$|E(G)| \leq d(v) + |E(G \setminus v)| \leq \Delta + \frac{(n-1)^2}{4}$$

又由平均度 $2|E|/n$，$\Delta \geq 2|E|/n$。令 $e = |E|$，则
$$e \leq \frac{2e}{n} + \frac{(n-1)^2}{4}$$
解得 $e \leq \frac{n^2}{4}$。$\square$

## 5. 进阶主题

### 5.1 Möbius反演与偏序集

有限偏序集 $(P, \leq)$ 上的Möbius函数 $\mu$ 递归定义为
$$\mu(x, x) = 1, \quad \sum_{x \leq z \leq y} \mu(x, z) = 0 \text{（$x < y$）}$$

若 $f(y) = \sum_{x \leq y} g(x)$，则 $g(y) = \sum_{x \leq y} \mu(x, y) f(x)$。此框架统一了数论Möbius反演、容斥原理以及图论中的色多项式计算。

### 5.2 概率方法

Erdős发展的概率方法通过证明随机对象以正概率满足某性质，来证明该性质对象的存在性。经典应用：证明 Ramsey 数 $R(k, k) > 2^{k/2 - 1}$。
