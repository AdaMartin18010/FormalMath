# 组合数学与离散数学基础理论

组合数学（Combinatorics）与离散数学（Discrete Mathematics）是研究有限或可数结构的数学分支，与计算机科学、信息论、统计物理以及现代优化理论有着深刻的联系。组合数学关注"如何计数与构造"，其核心问题包括排列组合、图的存在性与极值性质、设计理论以及代数组合；离散数学则是更广泛的领域，涵盖逻辑、集合论、图论、算法分析与组合优化。

## 1. 组合计数原理

### 1.1 排列与组合

**定义 1.1（排列）**. 从 $n$ 个不同元素中取 $k$ 个的有序选取数为：

$$P(n,k) = \frac{n!}{(n-k)!}.$$

**定义 1.2（组合）**. 从 $n$ 个不同元素中取 $k$ 个的无序选取数为：

$$\binom{n}{k} = \frac{n!}{k!(n-k)!}.$$

**定理 1.3（二项式定理）**. 

$$(x+y)^n = \sum_{k=0}^n \binom{n}{k} x^k y^{n-k}.$$

*证明*. 对 $n$ 归纳，或直接从展开式中每项 $x^k y^{n-k}$ 的系数计数。$\square$

### 1.2 容斥原理

**定理 1.4（容斥原理）**. 对有限集 $A_1, \dots, A_n$：

$$\left|\bigcup_{i=1}^n A_i\right| = \sum_{i} |A_i| - \sum_{i<j} |A_i \cap A_j| + \sum_{i<j<k} |A_i \cap A_j \cap A_k| - \cdots + (-1)^{n-1} |A_1 \cap \cdots \cap A_n|.$$

*证明*. 对任意元素 $x$ 属于恰好 $m$ 个集合，右边计数为 $\sum_{k=1}^m (-1)^{k-1} \binom{m}{k} = 1 - (1-1)^m = 1$。$\square$

## 2. 图论基础

### 2.1 图的基本定义

**定义 2.1（图）**. 图 $G = (V, E)$ 由顶点集 $V$ 和边集 $E \subseteq \binom{V}{2}$ 组成。

**定义 2.2（度）**. 顶点 $v$ 的度 $d(v)$ 为与 $v$ 关联的边数。

**定理 2.3（握手引理）**. $\sum_{v \in V} d(v) = 2|E|$。

### 2.2 Euler 路径与 Hamilton 圈

**定理 2.4（Euler）**. 连通图有 Euler 路径（经过每边恰好一次）当且仅当奇度顶点个数为 0 或 2。

**定理 2.5（Dirac）**. 设 $G$ 为 $n \geq 3$ 个顶点的简单图，若每个顶点度 $\geq n/2$，则 $G$ 有 Hamilton 圈。

*证明*. 假设 $G$ 无 Hamilton 圈，取最长路径 $v_1, \dots, v_k$。由度条件，$v_1$ 与 $v_k$ 的邻域在路径上有足够多重叠，可构造更长路径或圈，矛盾。$\square$

## 3. 生成函数

### 3.1 普通生成函数

**定义 3.1（OGF）**. 序列 $(a_n)_{n \geq 0}$ 的普通生成函数为 $A(x) = \sum_{n=0}^\infty a_n x^n$。

**定理 3.2**. 组合类的笛卡尔积对应生成函数的乘积。

### 3.2 指数生成函数

**定义 3.3（EGF）**. $A(x) = \sum_{n=0}^\infty a_n \frac{x^n}{n!}$。

**定理 3.4**. 标记组合类的集合构造对应指数生成函数的指数。

## 4. 例子

### 4.1 例子：Catalan 数

$n$ 对括号的合法匹配数、二叉树个数、不交叉划分个数均为 Catalan 数：

$$C_n = \frac{1}{n+1}\binom{2n}{n}.$$

生成函数满足 $C(x) = 1 + xC(x)^2$，解得 $C(x) = \frac{1 - \sqrt{1-4x}}{2x}$。

### 4.2 例子：四色定理

**定理 4.1（四色定理）**. 任何平面图可用四种颜色正常着色。

*注*. Appel 与 Haken 于 1976 年借助计算机完成证明，2005 年 Georges Gonthier 在 Coq 中形式化验证。
