# 实例表征完善指南

## 📋 指南概述

### 目标

为FormalMath项目的数学概念提供丰富、准确、易懂的实例表征，增强内容的可理解性和实用性。

### 原则

- **准确性**: 实例必须数学上准确无误
- **典型性**: 选择最具代表性的实例
- **渐进性**: 从简单到复杂，循序渐进
- **多样性**: 提供多种类型的实例
- **实用性**: 实例应具有实际应用价值

## 🎯 实例类型分类

### 1. 基础实例

**特点**: 简单直观，易于理解
**用途**: 帮助初学者建立概念认知
**要求**: 计算简单，结果明确

**示例**:

- 集合论：{1, 2, 3} 的交集、并集运算
- 群论：整数加法群 (ℤ, +)
- 线性代数：2×2矩阵的运算

### 2. 标准实例

**特点**: 经典常见，广泛应用
**用途**: 展示概念的标准应用
**要求**: 具有代表性，计算适中

**示例**:

- 微积分：函数 f(x) = x² 的导数
- 拓扑学：实数集上的标准拓扑
- 代数几何：仿射平面上的曲线

### 3. 反例

**特点**: 说明概念边界和限制
**用途**: 澄清概念，避免误解
**要求**: 简单明确，易于验证

**示例**:

- 连续函数：Dirichlet函数
- 可积函数：非黎曼可积的函数
- 群：非交换群的例子

### 4. 应用实例

**特点**: 来自实际应用领域
**用途**: 展示数学的实际价值
**要求**: 背景清晰，计算可行

**示例**:

- 概率论：抛硬币、掷骰子
- 统计学：正态分布的应用
- 优化理论：线性规划问题

### 5. 历史实例

**特点**: 具有历史意义和背景
**用途**: 展示概念的发展历程
**要求**: 历史准确，数学正确

**示例**:

- 数论：费马大定理的历史
- 几何学：欧几里得几何公理
- 分析学：柯西序列的构造

## 📈 实例质量评估

### 评估维度

#### 1. 数学准确性 (权重: 30%)

- **标准**: 实例在数学上完全正确
- **检查点**:
  - 定义使用正确
  - 计算过程无误
  - 结论准确

#### 2. 教育价值 (权重: 25%)

- **标准**: 实例有助于概念理解
- **检查点**:
  - 概念展示清晰
  - 理解难度适中
  - 启发性强

#### 3. 典型性 (权重: 20%)

- **标准**: 实例具有代表性
- **检查点**:
  - 覆盖核心概念
  - 具有一般性
  - 易于推广

#### 4. 实用性 (权重: 15%)

- **标准**: 实例具有实际应用价值
- **检查点**:
  - 来自实际应用
  - 计算可行
  - 结果有意义

#### 5. 表述质量 (权重: 10%)

- **标准**: 实例表述清晰易懂
- **检查点**:
  - 语言简洁明了
  - 逻辑结构清晰
  - 图表配合恰当

### 评估流程

1. **自评**: 作者进行自我评估
2. **同行评审**: 团队成员交叉评审
3. **专家评审**: 邀请领域专家评审
4. **用户测试**: 实际用户理解测试
5. **综合评估**: 综合各方意见给出最终评分

## 🔄 实例更新维护

### 更新触发条件

- 发现数学错误
- 用户反馈问题
- 内容过时
- 表述不够清晰
- 实例不够典型

### 更新流程

1. **问题识别**: 识别需要更新的实例
2. **更新设计**: 设计新的实例内容
3. **验证测试**: 验证新实例的正确性
4. **用户测试**: 测试新实例的效果
5. **正式更新**: 正式更新实例内容

### 版本控制

- 记录实例的修改历史
- 保留重要版本的备份
- 标注实例的来源和贡献者
- 建立实例的引用关系
