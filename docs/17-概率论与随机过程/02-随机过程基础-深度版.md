# 随机过程基础（深度版）

## 1. 引言与历史背景

随机过程是概率论从静态（随机变量）到动态（随时间演变的随机现象）的自然延伸。从Brown对花粉颗粒运动的观察（1827年）到Wiener的严格数学构造（1923年），从Bachelier的投机理论（1900年）到Itô的随机积分（1944年），随机过程理论经历了深刻的发展。

**核心人物关联**：

- **Robert Brown (1773-1858)**：苏格兰植物学家，1827年观察到花粉的随机运动
- **Louis Bachelier (1870-1946)**：法国数学家，1900年博士论文《投机理论》首次将布朗运动应用于金融
- **Norbert Wiener (1894-1964)**：美国数学家，1923年严格构造布朗运动，证明了Wiener测度的存在
- **Andrey Kolmogorov (1903-1987)**：1931年建立马尔可夫过程的严格理论
- **Paul Lévy (1886-1971)**：法国概率论大师，对布朗运动路径性质有深入研究

## 2. 随机过程的定义与有限维分布

### 2.1 随机过程的测度论定义

**定义 2.1（随机过程）** 设 $(\Omega, \mathcal{F}, \mathbb{P})$ 为概率空间，$T$ 为指标集（通常 $T = \mathbb{N}, \mathbb{R}_+, [0,T]$）。一族随机变量 $\{X_t: t \in T\}$ 称为 **随机过程**。

等价地，随机过程可视为映射 $X: \Omega \times T \to \mathbb{R}$，满足对任意固定的 $t \in T$，$X_t(\cdot)$ 是随机变量。

**定义 2.2（样本路径）** 固定 $\omega \in \Omega$，映射 $t \mapsto X_t(\omega)$ 称为 **样本路径**（或 **轨道**、**实现**）。

### 2.2 有限维分布

**定义 2.3（有限维分布）** 随机过程 $X$ 的 **有限维分布** 为所有有限维的联合分布：
$$\mu_{t_1, \ldots, t_n}(B_1 \times \cdots \times B_n) = \mathbb{P}(X_{t_1} \in B_1, \ldots, X_{t_n} \in B_n)$$
其中 $n \geq 1$，$t_1, \ldots, t_n \in T$，$B_1, \ldots, B_n \in \mathcal{B}(\mathbb{R})$。

**定理 2.4（Kolmogorov扩张/相容性定理）** 设 $\{\mu_{t_1, \ldots, t_n}\}$ 为一族有限维分布，满足：

1. **对称性**：对任意置换 $\pi$：$\mu_{t_{\pi(1)}, \ldots, t_{\pi(n)}} = \mu_{t_1, \ldots, t_n} \circ \pi^{-1}$
2. **相容性**：$\mu_{t_1, \ldots, t_n}(B_1 \times \cdots \times B_{n-1} \times \mathbb{R}) = \mu_{t_1, \ldots, t_{n-1}}(B_1 \times \cdots \times B_{n-1})$

则存在概率空间 $(\Omega, \mathcal{F}, \mathbb{P})$ 和其上的随机过程 $\{X_t\}$，使其有限维分布恰为 $\{\mu_{t_1, \ldots, t_n}\}$。

**证明概要**：取 $\Omega = \mathbb{R}^T$，$\mathcal{F} = \sigma$-代数由柱集生成，利用Carathéodory扩张定理构造概率测度。

### 2.3 过程的版本与修正

**定义 2.5（过程的版本）** 两个随机过程 $X$ 和 $Y$ 称为互为 **版本**（或 **修正**），若对所有 $t \in T$，$\mathbb{P}(X_t = Y_t) = 1$。

**定义 2.6（不可区分）** $X$ 和 $Y$ 称为 **不可区分**，若
$$\mathbb{P}(X_t = Y_t, \forall t \in T) = 1$$

**定理 2.7（Kolmogorov-Čentsov连续性准则）** 设 $X = \{X_t\}_{t \in [0,1]}$ 为实值随机过程，若存在 $\alpha, \beta, C > 0$ 使得
$$\mathbb{E}[|X_t - X_s|^\alpha] \leq C|t-s|^{1+\beta}, \quad \forall s,t \in [0,1]$$
则 $X$ 存在 Hölder 连续的版本，指数为 $\gamma \in (0, \beta/\alpha)$。

## 3. 布朗运动（Wiener过程）

### 3.1 定义与基本性质

**定义 3.1（标准布朗运动）** 随机过程 $W = \{W_t\}_{t \geq 0}$ 称为 **标准布朗运动**（或 **Wiener过程**），若：

1. $W_0 = 0$ a.s.
2. **独立增量**：对 $0 \leq t_1 < t_2 < \cdots < t_n$，增量 $W_{t_2}-W_{t_1}, \ldots, W_{t_n}-W_{t_{n-1}}$ 相互独立
3. **平稳增量**：$W_t - W_s \sim \mathcal{N}(0, t-s)$，对 $0 \leq s < t$
4. **连续性**：样本路径 $t \mapsto W_t(\omega)$ 连续

**历史注记**：Wiener在1923年首次严格证明了满足上述条件的随机过程的存在性，其构造使用随机Fourier级数。

### 3.2 布朗运动的构造

**构造 3.2（Lévy-Ciesielski构造）** 设 $\{Z_n\}_{n \geq 0}$ i.i.d. $\sim \mathcal{N}(0,1)$，定义
$$W_t = t Z_0 + \sum_{n=0}^{\infty} \sum_{k=0}^{2^n-1} f_{n,k}(t) Z_{n,k}$$
其中 $f_{n,k}$ 为Schauder函数（分段线性"帐篷"函数）。

级数在 $[0,1]$ 上一致收敛（a.s.），从而保证连续性。

**定理 3.3（Wiener测度）** 存在唯一的概率测度 $\mathbb{W}$（**Wiener测度**）在 $C([0,1])$ 上，使得坐标过程 $W_t(\omega) = \omega(t)$ 是标准布朗运动。

### 3.3 布朗运动的路径性质

**定理 3.4（路径不可微性）** 对几乎所有的 $\omega$，布朗运动的路径 $t \mapsto W_t(\omega)$ 处处不可微。

**证明概要**：利用 scaling 性质 $W_{ct} \stackrel{d}{=} \sqrt{c} W_t$，可证明变差无限，从而不可微。

**定理 3.5（二次变差）** 设 $\Pi = \{0 = t_0 < t_1 < \cdots < t_n = t\}$ 为 $[0,t]$ 的分割，$|\Pi| = \max_i (t_i - t_{i-1})$，则
$$\lim_{|\Pi| \to 0} \sum_{i=1}^{n} (W_{t_i} - W_{t_{i-1}})^2 = t \quad \text{(依概率收敛)}$$

**重述**：布朗运动的二次变差为 $d\langle W \rangle_t = dt$，这是随机微积分与普通微积分的根本区别。

**定理 3.6（重对数律 - Khinchin, 1933）**
$$\mathbb{P}\left(\limsup_{t \to 0} \frac{W_t}{\sqrt{2t \log\log(1/t)}} = 1\right) = 1$$

## 4. 鞅理论

### 4.1 鞅的定义与例子

**定义 4.1（滤子/信息流）** 一族σ-代数 $\{\mathcal{F}_t\}_{t \geq 0}$ 称为 **滤子**，若 $\mathcal{F}_s \subseteq \mathcal{F}_t \subseteq \mathcal{F}$ 对 $s \leq t$。

**定义 4.2（适应过程）** 随机过程 $X$ 称为 **适应的**（adapted），若对每个 $t$，$X_t$ 是$\mathcal{F}_t$-可测的。

**定义 4.3（鞅）** 适应过程 $M = \{M_t\}$ 称为 **鞅**，若：

1. $\mathbb{E}[|M_t|] < \infty$ 对所有 $t$
2. $\mathbb{E}[M_t | \mathcal{F}_s] = M_s$ a.s.，对 $s \leq t$

若条件2中的等号替换为 $\leq$ 或 $\geq$，则分别称为 **上鞅** 或 **下鞅**。

**历史注记**：Joseph Doob在1953年出版的《随机过程》中系统发展了鞅理论，将其确立为现代概率论的核心工具。

**例 4.4（布朗运动是鞅）** 标准布朗运动 $W_t$ 关于其自然滤子 $\mathcal{F}_t^W = \sigma(W_s: 0 \leq s \leq t)$ 是鞅：
$$\mathbb{E}[W_t | \mathcal{F}_s^W] = \mathbb{E}[W_t - W_s + W_s | \mathcal{F}_s^W] = W_s$$

**例 4.5（指数鞅）** $M_t = \exp(\theta W_t - \frac{1}{2}\theta^2 t)$ 是鞅。

### 4.2 停时与可选停止定理

**定义 4.6（停时）** 映射 $\tau: \Omega \to [0, +\infty]$ 称为 **停时**，若对所有 $t \geq 0$，$\{\tau \leq t\} \in \mathcal{F}_t$。

**例 4.7** 首次到达时间 $\tau_a = \inf\{t \geq 0: W_t = a\}$ 是停时。

**定理 4.8（可选停止定理 - Doob）** 设 $M$ 为鞅，$\tau$ 为有界停时（或满足一定可积性条件），则
$$\mathbb{E}[M_\tau] = \mathbb{E}[M_0]$$
更一般地，对停时 $\sigma \leq \tau$：$\mathbb{E}[M_\tau | \mathcal{F}_\sigma] = M_\sigma$ a.s.

**例 4.9（赌徒破产问题）** 设 $\{S_n = S_0 + \sum_{i=1}^{n} X_i\}$ 为简单随机游走，$X_i = \pm 1$ 等概率。令
$$\tau = \inf\{n \geq 0: S_n = a \text{ 或 } S_n = b\}, \quad a < S_0 < b$$

应用可选停止定理于鞅 $S_n$ 和 $S_n^2 - n$ 可得：
$$\mathbb{P}(S_\tau = a) = \frac{b - S_0}{b - a}, \quad \mathbb{E}[\tau] = (S_0 - a)(b - S_0)$$

### 4.3 Doob不等式与收敛定理

**定理 4.10（Doob极大不等式）** 设 $M$ 为非负下鞅，$\lambda > 0$，则
$$\lambda \mathbb{P}\left(\max_{0 \leq s \leq t} M_s \geq \lambda\right) \leq \mathbb{E}[M_t]$$

**定理 4.11（Doob $L^p$不等式）** 对 $p > 1$，$M$ 为正下鞅，
$$\mathbb{E}\left[\left(\max_{0 \leq s \leq t} M_s\right)^p\right] \leq \left(\frac{p}{p-1}\right)^p \mathbb{E}[M_t^p]$$

**定理 4.12（鞅收敛定理）** 设 $M$ 为上鞅且 $\sup_t \mathbb{E}[M_t^-] < \infty$，则 $M_t$ 几乎必然收敛于可积随机变量 $M_\infty$。

## 5. 马尔可夫过程

### 5.1 马尔可夫性

**定义 5.1（马尔可夫过程）** 适应过程 $X$ 称为 **马尔可夫过程**，若
$$\mathbb{P}(X_t \in B | \mathcal{F}_s) = \mathbb{P}(X_t \in B | X_s), \quad \forall s \leq t, B \in \mathcal{B}(\mathbb{R})$$

**解释**：在给定现在 $X_s$ 的条件下，未来 $X_t$ 与过去 $\mathcal{F}_s$ 独立。

**定理 5.2（布朗运动的马尔可夫性）** 布朗运动是马尔可夫过程，其转移密度为
$$p(t-s, x, y) = \frac{1}{\sqrt{2\pi(t-s)}} \exp\left(-\frac{(y-x)^2}{2(t-s)}\right)$$

### 5.2 转移半群与Chapman-Kolmogorov方程

**定义 5.3（转移核/概率）** 函数 $P(s,t;x,dy)$ 称为 **转移概率**，若：

1. 对每个 $s,t,x$，$P(s,t;x,\cdot)$ 是概率测度
2. 对每个 $s,t,B$，$P(s,t;\cdot,B)$ 可测
3. 满足Chapman-Kolmogorov方程

**定义 5.4（Chapman-Kolmogorov方程）**
$$P(s,t;x,B) = \int_{\mathbb{R}} P(u,t;y,B) P(s,u;x,dy), \quad s < u < t$$

**定义 5.5（转移半群）** 对时齐马尔可夫过程，定义算子半群
$$(P_t f)(x) = \int_{\mathbb{R}} f(y) P(t;x,dy) = \mathbb{E}_x[f(X_t)]$$
满足 $P_{s+t} = P_s P_t$（半群性质）。

### 5.3 生成元与Kolmogorov方程

**定义 5.6（无穷小生成元）**
$$(\mathcal{A}f)(x) = \lim_{t \downarrow 0} \frac{(P_t f)(x) - f(x)}{t}$$
定义域为使该极限存在的函数 $f$。

**例 5.7（布朗运动的生成元）**
$$\mathcal{A} = \frac{1}{2}\frac{d^2}{dx^2}$$
即布朗运动的生成元是二分之一拉普拉斯算子。

**定理 5.8（Kolmogorov向后方程）**
$$\frac{\partial u}{\partial s} + \mathcal{A}u = 0, \quad u(t,x) = f(x)$$
其中 $u(s,x) = \mathbb{E}[f(X_t) | X_s = x]$。

**定理 5.9（Kolmogorov向前方程/Fokker-Planck方程）**
$$\frac{\partial p}{\partial t} = \mathcal{A}^* p$$
其中 $\mathcal{A}^*$ 为生成元的形式伴随，$p(t,y)$ 为 $X_t$ 的密度。

## 6. 综合示例：布朗运动首中时

**例 6.1** 设 $W$ 为标准布朗运动，$a > 0$，定义首次到达 $a$ 的时间：
$$\tau_a = \inf\{t \geq 0: W_t = a\}$$

**定理**：$\tau_a$ 的分布为 Lévy 分布：
$$\mathbb{P}(\tau_a \leq t) = 2\mathbb{P}(W_t \geq a) = 2\left(1 - \Phi\left(\frac{a}{\sqrt{t}}\right)\right)$$
密度函数为：
$$f_{\tau_a}(t) = \frac{a}{\sqrt{2\pi t^3}} \exp\left(-\frac{a^2}{2t}\right), \quad t > 0$$

**证明**：利用指数鞅 $M_t = e^{\theta W_t - \theta^2 t/2}$ 和可选停止定理。

**推论**：$\mathbb{E}[\tau_a] = +\infty$，即布朗运动几乎必然到达任意水平，但期望时间为无穷！
