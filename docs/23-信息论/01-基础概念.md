## 一、熵

### 1.1 香农熵

**定义 1.1** (香农熵)
离散随机变量 $X$ 的熵定义为：
$$H(X) = -\sum_{x \in \mathcal{X}} p(x) \log p(x) = E[\log \frac{1}{p(X)}]$$

其中对数以2为底时，熵的单位是比特（bits）。

### 1.2 熵的性质

**定理 1.1** (熵的基本性质)
1. **非负性**：$H(X) \geq 0$，等号成立当且仅当 $X$ 是确定性变量
2. **上界**：$H(X) \leq \log |\mathcal{X}|$，等号成立当且仅当均匀分布
3. **链式法则**：$H(X, Y) = H(X) + H(Y|X)$

**二元熵函数**：
$$H_b(p) = -p\log p - (1-p)\log(1-p)$$

最大值 $H_b(0.5) = 1$ bit，最小值 $H_b(0) = H_b(1) = 0$。

### 1.3 联合熵与条件熵

**定义 1.2** (联合熵)
$$H(X, Y) = -\sum_{x,y} p(x, y) \log p(x, y)$$

**定义 1.3** (条件熵)
$$H(Y|X) = \sum_x p(x) H(Y|X=x) = -\sum_{x,y} p(x, y) \log p(y|x)$$

**定理 1.2** (熵的链式法则)
$$H(X_1, X_2, \ldots, X_n) = \sum_{i=1}^{n} H(X_i | X_1, \ldots, X_{i-1})$$

### 1.4 微分熵（连续情形）

**定义 1.4** (微分熵)
连续随机变量 $X$ 的微分熵：
$$h(X) = -\int_{-\infty}^{\infty} f(x) \log f(x) dx$$

**注意**：微分熵可以为负，不具有离散熵的非负性。

**例子**：
- 均匀分布 $U(0, a)$：$h(X) = \log a$
- 高斯分布 $\mathcal{N}(\mu, \sigma^2)$：$h(X) = \frac{1}{2}\log(2\pi e \sigma^2)$

## 二、互信息

### 2.1 定义

**定义 2.1** (互信息)
$$I(X; Y) = \sum_{x,y} p(x, y) \log \frac{p(x, y)}{p(x)p(y)}$$

### 2.2 互信息的性质

**定理 2.1** (互信息的基本性质)
1. **对称性**：$I(X; Y) = I(Y; X)$
2. **非负性**：$I(X; Y) \geq 0$，等号成立当且仅当 $X$ 和 $Y$ 独立
3. **与熵的关系**：
   - $I(X; Y) = H(X) - H(X|Y)$
   - $I(X; Y) = H(Y) - H(Y|X)$

   - $I(X; Y) = H(X) + H(Y) - H(X, Y)$

**解释**：互信息表示知道 $Y$ 后 $X$ 的不确定性减少量。

### 2.3 条件互信息

**定义 2.2** (条件互信息)
$$I(X; Y|Z) = H(X|Z) - H(X|Y, Z)$$

**链式法则**：
$$I(X; Y, Z) = I(X; Y) + I(X; Z|Y)$$

### 2.4 数据处理不等式

**定理 2.2** (数据处理不等式)
若 $X \to Y \to Z$ 形成马尔可夫链，则：
$$I(X; Z) \leq I(X; Y)$$

**推论**：处理数据不会增加信息量。

## 三、信道容量

### 3.1 离散无记忆信道

**定义 3.1** (离散无记忆信道)
离散无记忆信道（DMC）由输入字母表 $\mathcal{X}$、输出字母表 $\mathcal{Y}$ 和转移概率 $p(y|x)$ 描述。

**信道容量**：
$$C = \max_{p(x)} I(X; Y)$$

### 3.2 对称信道

**定义 3.2** (对称信道)
信道转移矩阵的行是排列，列是排列（或列的排列相同）。

**定理 3.1** (对称信道容量)
对称信道的容量为：
$$C = \log |\mathcal{Y}| - H(\text{行分布})$$

**例子**：
- 二进制对称信道（BSC）：$C = 1 - H_b(p)$
- 二进制擦除信道（BEC）：$C = 1 - \epsilon$

### 3.3 高斯信道

**加性高斯白噪声（AWGN）信道**：
$$Y = X + Z, \quad Z \sim \mathcal{N}(0, N)$$

**功率约束**：$E[X^2] \leq P$

**定理 3.2** (高斯信道容量)
$$C = \frac{1}{2}\log\left(1 + \frac{P}{N}\right) \text{ bits/channel use}$$

**带宽受限信道**：
$$C = W \log\left(1 + \frac{P}{WN_0}\right) \text{ bits/s}$$

其中 $W$ 是带宽，$N = WN_0$。

### 3.4 信道编码定理

**定理 3.3** (信道编码定理 - Shannon, 1948)
对于码率 $R < C$，存在编码方案使得错误概率任意小；对于 $R > C$，不存在这样的编码方案。

## 四、相对熵

### 4.1 Kullback-Leibler散度

**定义 4.1** (KL散度/相对熵)
$$D_{KL}(P \| Q) = \sum_x p(x) \log \frac{p(x)}{q(x)} = E_P\left[\log \frac{p(X)}{q(X)}\right]$$

### 4.2 性质

**定理 4.1** (KL散度的性质)
1. **非负性**：$D_{KL}(P \| Q) \geq 0$，等号成立当且仅当 $P = Q$
2. **不对称**：$D_{KL}(P \| Q) \neq D_{KL}(Q \| P)$

3. **不满足三角不等式**

### 4.3 与互信息的关系

$$I(X; Y) = D_{KL}(P_{X,Y} \| P_X P_Y)$$

互信息是联合分布与独立分布之间的KL散度。

### 4.4 信息不等式

**定理 4.2** (信息不等式)
对任意两个分布 $P$ 和 $Q$：
$$D_{KL}(P \| Q) \geq 0$$

**推论**（用Jensen不等式证明）：
- $H(X) \leq \log |\mathcal{X}|$
- $H(X|Y) \leq H(X)$（条件减少熵）

## 总结

本章节介绍了信息论的基础概念：
- **熵**：随机变量的不确定性度量
- **互信息**：两个变量之间的信息共享
- **信道容量**：可靠通信的最大速率
- **相对熵**：两个分布之间的差异度量

这些概念构成了现代信息论的理论基础。
