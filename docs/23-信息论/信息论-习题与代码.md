# 信息论习题与代码实现

## 习题

**习题1** 计算二元熵函数 $H(p) = -p\log_2 p - (1-p)\log_2(1-p)$ 在 $p=0.5$ 时的值。

**解答**：$H(0.5) = -0.5\log_2 0.5 - 0.5\log_2 0.5 = 0.5 + 0.5 = 1$ 比特

---

**习题2** 证明：$H(X, Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)$

**证明**：
$$\begin{aligned}
H(X, Y) &= -\sum_{x,y} p(x,y) \log p(x,y) \\
&= -\sum_{x,y} p(x,y) \log [p(x)p(y|x)] \\
&= -\sum_{x,y} p(x,y) [\log p(x) + \log p(y|x)] \\
&= -\sum_x p(x)\log p(x) - \sum_{x,y} p(x,y)\log p(y|x) \\
&= H(X) + H(Y|X) \quad \text{∎}
\end{aligned}$$

---

**习题3** 计算信道容量（二元对称信道，错误概率 $p$）。

**解答**：$C = 1 - H(p)$，其中 $H(p)$ 是二元熵函数。

当 $p = 0$ 或 $p = 1$ 时，$C = 1$；当 $p = 0.5$ 时，$C = 0$。

## Python实现

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import entropy

def binary_entropy(p):
    """二元熵函数"""
    if p == 0 or p == 1:
        return 0
    return -p * np.log2(p) - (1-p) * np.log2(1-p)

# 绘制二元熵函数
p_values = np.linspace(0, 1, 1000)
H_values = [binary_entropy(p) for p in p_values]

plt.figure(figsize=(10, 6))
plt.plot(p_values, H_values, linewidth=2)
plt.xlabel('p')
plt.ylabel('H(p) (bits)')
plt.title('Binary Entropy Function')
plt.grid(True)
plt.savefig('binary_entropy.png')
plt.show()

# 计算联合熵
def joint_entropy(pxy):
    """计算联合熵"""
    pxy_flat = pxy.flatten()
    pxy_flat = pxy_flat[pxy_flat > 0]  # 移除0概率
    return -np.sum(pxy_flat * np.log2(pxy_flat))

# 示例
pxy = np.array([[0.25, 0.25], [0.25, 0.25]])  # 独立均匀分布
print(f"联合熵: {joint_entropy(pxy):.4f} bits")

# 互信息
def mutual_information(pxy):
    """计算互信息 I(X;Y)"""
    px = np.sum(pxy, axis=1)
    py = np.sum(pxy, axis=0)
    
    mi = 0
    for i in range(len(px)):
        for j in range(len(py)):
            if pxy[i,j] > 0:
                mi += pxy[i,j] * np.log2(pxy[i,j] / (px[i] * py[j]))
    return mi

# 示例
pxy = np.array([[0.4, 0.1], [0.1, 0.4]])  # 相关分布
print(f"互信息: {mutual_information(pxy):.4f} bits")
```

## 反例

### 反例：$I(X;Y) \neq I(X;Z)$ 即使 $Y$ 和 $Z$ 同分布
**反例**：设 $X \sim \text{Bernoulli}(0.5)$
- $Y = X$（完全相关），$I(X;Y) = 1$
- $Z$ 独立于 $X$，$Z \sim \text{Bernoulli}(0.5)$，$I(X;Z) = 0$

$Y$ 和 $Z$ 同分布但互信息不同。
