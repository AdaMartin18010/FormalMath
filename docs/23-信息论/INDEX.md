## 核心定理速查

| 定理 | 描述 | 意义 |
|------|------|------|
| **Kraft不等式** | 前缀码存在条件: $\sum D^{-l_i} \leq 1$ | 信源编码理论 |
| **信源编码定理** | 可达率 $R > H(X)$，不可达 $R < H(X)$ | 无损压缩极限 |
| **信道编码定理** | 可达率 $R < C$，不可达 $R > C$ | 可靠通信极限 |
| **率失真定理** | $R(D) = \min I(X;\hat{X})$ | 有损压缩极限 |
| **数据处理不等式** | $I(X;Z) \leq I(X;Y)$ | 信息不增原理 |

## 编码效率对比

| 编码方法 | 压缩比 | 计算复杂度 | 主要应用 |
|----------|--------|------------|----------|
| Huffman | 接近熵限 | O(n log n) | 通用无损 |
| 算术编码 | 更逼近熵限 | O(n) | 现代压缩 |
| LZ77/78 | 依赖重复模式 | O(n) | gzip, zip |
| 霍夫曼+LZ | 优秀 | 中等 | PNG, ZIP |
