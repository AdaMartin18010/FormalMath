## 1. 问题建模

### 1.1 信息的不确定性度量

**问题背景**: 如何量化随机事件的不确定性？

**示例：天气预测**

| 天气 | 概率 $p$ | 自信息 $-\log p$ |
|------|----------|-------------------|
| 晴 | 0.7 | 0.51 bits |
| 雨 | 0.2 | 2.32 bits |
| 雪 | 0.1 | 3.32 bits |

**直观**: 小概率事件携带更多信息。

### 1.2 Shannon熵的公理化推导

**Shannon提出**，不确定性度量 $H(p_1, \ldots, p_n)$ 应满足：

1. **连续性**: $H$ 是概率的连续函数
2. **对称性**: $H$ 对概率排列不变
3. **可加性**: $H(p, 1-p) + (1-p)H(q, 1-q) = H(pq, p(1-q), (1-p)q, (1-p)(1-q))$
4. **极值性**: 均匀分布时熵最大

**定理 1.1** (Shannon): 唯一满足上述条件的函数（差一常数倍）为：
$$H(X) = -\sum_{i=1}^n p_i \log p_i$$

## 2. 数学理论

### 2.1 熵的定义与性质

**定义 2.1** (Shannon熵): 离散随机变量 $X$ 的熵：
$$H(X) = -\sum_{x \in \mathcal{X}} p(x) \log p(x) = \mathbb{E}[-\log p(X)]$$

（对数以2为底，单位为比特bits；以 $e$ 为底，单位为纳特nats）

**定理 2.1** (熵的基本性质):
1. $H(X) \geq 0$（非负性）
2. $H(X) \leq \log |\mathcal{X}|$（极值性），等号当且仅当均匀分布
3. $H(X) = 0$ 当且仅当 $X$ 是确定性变量

*证明* (极值性): 用Jensen不等式
$$H(X) - \log|\mathcal{X}| = \sum p(x) \log\frac{1}{p(x)|\mathcal{X}|} \leq \log\sum\frac{1}{|\mathcal{X}|} = 0$$

### 2.2 联合熵与条件熵

**定义 2.2** (联合熵): 
$$H(X, Y) = -\sum_{x,y} p(x,y) \log p(x,y)$$

**定义 2.3** (条件熵): 
$$H(Y|X) = \sum_x p(x) H(Y|X=x) = -\sum_{x,y} p(x,y) \log p(y|x)$$

**定理 2.2** (链式法则):
$$H(X, Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)$$

**定理 2.3** (条件作用减少熵):
$$H(Y|X) \leq H(Y)$$
等号当且仅当 $X$ 与 $Y$ 独立。

### 2.3 互信息

**定义 2.4** (互信息): 
$$I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X) = H(X) + H(Y) - H(X,Y)$$

**解释**: 知道 $Y$ 后 $X$ 不确定性的减少量，即 $Y$ 包含关于 $X$ 的信息量。

**定理 2.4** (互信息性质):
1. $I(X;Y) = I(Y;X)$（对称性）
2. $I(X;Y) \geq 0$（非负性），等号当且仅当独立
3. $I(X;X) = H(X)$（自信息等于熵）

### 2.4 相对熵 (KL散度)

**定义 2.5** (Kullback-Leibler散度): 
$$D_{KL}(p \| q) = \sum_x p(x) \log\frac{p(x)}{q(x)}$$

**定理 2.5** (信息不等式): $D_{KL}(p \| q) \geq 0$，等号当且仅当 $p = q$。

**关系**:
$$I(X;Y) = D_{KL}(p(x,y) \| p(x)p(y))$$

## 3. 算法与方法

### 3.1 熵的估计

**直方图估计**:
```python
def estimate_entropy(samples, bins='auto'):
    """
    用直方图法估计离散分布的熵
    """
    counts, _ = np.histogram(samples, bins=bins)
    probs = counts / len(samples)
    probs = probs[probs > 0]  # 避免log(0)
    return -np.sum(probs * np.log2(probs))
```

**KNN估计** (连续分布):
$$\hat{H}(X) = \psi(N) - \psi(k) + \log c_d + \frac{d}{N} \sum_{i=1}^N \log \epsilon(i)$$

其中 $\psi$ 是digamma函数，$\epsilon(i)$ 是第 $k$ 近邻距离。

### 3.2 互信息估计

**直方图法**: 将数据离散化后计算
**KSG估计** (Kraskov-Stögbauer-Grassberger): 更精确的KNN方法

## 4. 具体案例

### 4.1 案例：英文文本熵分析

**数据集**: 英文文本样本

**计算**:
- 字母级熵: $H_{\text{letter}} \approx 4.03$ bits/letter
- 单词级熵: $H_{\text{word}} \approx 11.82$ bits/word

**比较**: 
- 随机字母（26个）: $\log_2 26 \approx 4.7$ bits
- 英文文本冗余度: $1 - 4.03/4.7 \approx 14\%$

### 4.2 案例：基因序列信息分析

**问题**: DNA序列中四种碱基A、C、G、T的分布

**数据**: 人类基因组某区域
- $p(A) = 0.295$, $p(C) = 0.205$
- $p(G) = 0.205$, $p(T) = 0.295$

**熵计算**:
$$H = -2 \times 0.295 \log_2 0.295 - 2 \times 0.205 \log_2 0.205 \approx 1.97 \text{ bits}$$

**最大熵**: $\log_2 4 = 2$ bits

**结论**: 基因组并非完全随机，存在一定的结构和规律。

## 5. 代码实现

### 5.1 熵与互信息计算

```python
import numpy as np
from scipy.stats import entropy as scipy_entropy

def compute_entropy(p, base=2):
    """
    计算离散分布的熵
    
    参数:
        p: 概率分布（数组，和为1）
        base: 对数底数，2为bits，e为nats
    """
    p = np.array(p)
    p = p[p > 0]  # 移除零概率
    return -np.sum(p * np.log(p)) / np.log(base)

def compute_joint_entropy(joint_prob, base=2):
    """
    计算联合熵
    
    参数:
        joint_prob: 联合概率矩阵 p(x,y)
    """
    p = np.array(joint_prob).flatten()
    p = p[p > 0]
    return -np.sum(p * np.log(p)) / np.log(base)

def compute_conditional_entropy(joint_prob, marginal_x, base=2):
    """
    计算条件熵 H(Y|X)
    """
    joint = np.array(joint_prob)
    marginal = np.array(marginal_x).reshape(-1, 1)
    
    # p(y|x) = p(x,y) / p(x)
    conditional = joint / marginal
    conditional[joint == 0] = 0  # 避免0/0
    
    return -np.sum(joint * np.log(conditional + 1e-12)) / np.log(base)

def compute_mutual_info(joint_prob, marginal_x, marginal_y, base=2):
    """
    计算互信息 I(X;Y)
    """
    joint = np.array(joint_prob)
    mx = np.array(marginal_x).reshape(-1, 1)
    my = np.array(marginal_y).reshape(1, -1)
    
    # I(X;Y) = sum p(x,y) log(p(x,y) / (p(x)p(y)))
    mi_matrix = joint * np.log(joint / (mx @ my + 1e-12) + 1e-12)
    return np.sum(mi_matrix) / np.log(base)

# 示例：二元对称信道
p = 0.1  # 误码率
px = [0.5, 0.5]  # 输入分布
py_given_x = [[1-p, p], [p, 1-p]]  # 信道矩阵

# 计算各种熵
joint = np.array([[px[i] * py_given_x[i][j] for j in range(2)] for i in range(2)])
py = [sum(joint[:, j]) for j in range(2)]

H_x = compute_entropy(px)
H_y = compute_entropy(py)
H_xy = compute_joint_entropy(joint)
I_xy = compute_mutual_info(joint, px, py)

print(f"H(X) = {H_x:.4f} bits")
print(f"H(Y) = {H_y:.4f} bits")
print(f"H(X,Y) = {H_xy:.4f} bits")
print(f"I(X;Y) = {I_xy:.4f} bits")
print(f"信道容量 C = {1 - compute_entropy([p, 1-p]):.4f} bits")
```

### 5.2 文本熵分析

```python
import numpy as np
from collections import Counter

def analyze_text_entropy(text):
    """
    分析文本的字符级和单词级熵
    """
    # 字符级分析
    char_counts = Counter(text.lower())
    total_chars = sum(char_counts.values())
    char_probs = [c/total_chars for c in char_counts.values()]
    char_entropy = -sum(p * np.log2(p) for p in char_probs if p > 0)
    
    # 单词级分析
    words = text.lower().split()
    word_counts = Counter(words)
    total_words = sum(word_counts.values())
    word_probs = [c/total_words for c in word_counts.values()]
    word_entropy = -sum(p * np.log2(p) for p in word_probs if p > 0)
    
    return {
        'char_entropy': char_entropy,
        'char_count': len(char_counts),
        'word_entropy': word_entropy,
        'word_count': len(word_counts),
        'avg_word_length': total_chars / total_words
    }

# 示例文本
text = """
Information theory is a mathematical framework for quantifying information.
It was developed by Claude Shannon in 1948. The central concept is entropy,
which measures the uncertainty or randomness of a random variable.
"""

result = analyze_text_entropy(text)
print(f"字符熵: {result['char_entropy']:.3f} bits/char")
print(f"不同字符数: {result['char_count']}")
print(f"单词熵: {result['word_entropy']:.3f} bits/word")
print(f"不同单词数: {result['word_count']}")
```

- gzip: 通常接近熵限

## 7. 习题

### 习题 7.1 (熵的基本性质)

证明：
1. $H(X) \leq \log |\mathcal{X}|$，等号当且仅当 $X$ 均匀分布
2. $H(X, Y) \leq H(X) + H(Y)$，等号当且仅当独立
3. $H(Y|X) \leq H(Y)$，条件作用减少熵

### 习题 7.2 (互信息)

设 $X, Y$ 是两个二元随机变量，联合分布为：

|  | Y=0 | Y=1 |
|--|-----|-----|
| X=0 | 0.25 | 0.25 |
| X=1 | 0.25 | 0.25 |

1. 计算 $H(X)$, $H(Y)$, $H(X,Y)$
2. 计算 $I(X;Y)$
3. 若联合分布变为对角阵（如 $p(0,0)=0.5, p(1,1)=0.5$），重新计算

### 习题 7.3 (Fano不等式)

证明Fano不等式：设 $P_e = P(X \neq \hat{X})$ 是估计错误概率，则
$$H(X|\hat{X}) \leq H(P_e) + P_e \log(|\mathcal{X}| - 1)$$

并解释其信息论意义。

### 习题 7.4 (数据处理不等式)

证明：若 $X \to Y \to Z$ 形成马尔可夫链，则
$$I(X;Z) \leq I(X;Y)$$

这说明信息在数据处理过程中不会增加。
