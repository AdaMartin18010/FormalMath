## 问题1：熵的计算

### 问题描述
计算以下随机变量的熵：

a) 公平骰子的结果

b) 随机变量 $X$ 的分布：$P(X=0) = 0.5, P(X=1) = 0.25, P(X=2) = 0.25$

c) 二元随机变量 $Y$，$P(Y=1) = p$

### 解答

**a) 公平骰子**：

均匀分布，$|X| = 6$

$$H(X) = \log_2 6 = \frac{\ln 6}{\ln 2} \approx 2.585 \text{ bits}$$

**b) 离散随机变量**：

$$H(X) = -0.5\log_2 0.5 - 0.25\log_2 0.25 - 0.25\log_2 0.25$$
$$= 0.5 + 0.5 + 0.5 = 1.5 \text{ bits}$$

**c) 二元熵函数**：

$$H(Y) = H_b(p) = -p\log_2 p - (1-p)\log_2 (1-p)$$

- 当 $p = 0.5$ 时，$H(Y) = 1$ bit（最大熵）
- 当 $p = 0.1$ 时，$H(Y) \approx 0.469$ bits
- 当 $p = 0.9$ 时，$H(Y) \approx 0.469$ bits（对称性）

## 问题2：信道容量计算

### 问题描述
计算以下信道的容量：

a) 二进制对称信道（BSC），交叉概率 $p = 0.1$

b) 二进制擦除信道（BEC），擦除概率 $\epsilon = 0.2$

c) 高斯信道，SNR = 10 dB

### 解答

**a) BSC信道**：

$$C = 1 - H_b(p) = 1 - H_b(0.1)$$

$$H_b(0.1) = -0.1\log_2 0.1 - 0.9\log_2 0.9 \approx 0.469$$

$$C \approx 1 - 0.469 = 0.531 \text{ bits/channel use}$$

**b) BEC信道**：

$$C = 1 - \epsilon = 1 - 0.2 = 0.8 \text{ bits/channel use}$$

**c) 高斯信道**：

SNR = 10 dB = $10^{10/10} = 10$（线性值）

$$C = \frac{1}{2}\log_2(1 + \text{SNR}) = \frac{1}{2}\log_2(11) \approx 1.73 \text{ bits/channel use}$$

若带宽 $W = 1$ MHz：
$$C = W\log_2(1 + \text{SNR}) = 10^6 \times \log_2(11) \approx 3.46 \text{ Mbits/s}$$

## 问题3：Huffman编码

### 问题描述
为以下概率分布构造Huffman编码：

| 符号 | A | B | C | D | E |
|------|---|---|---|---|---|
| 概率 | 0.4 | 0.2 | 0.2 | 0.15 | 0.05 |

计算期望码长和编码效率。

### 解答

**Huffman编码构造**：

步骤1：将符号按概率降序排列
步骤2：合并概率最小的两个符号
步骤3：重复直到只剩一个节点

**编码树构造**：

```

       1.0
      /   \
    0.4    0.6(A)

    |     /   \

    A   0.2    0.4(B)

        |     /   \

        B   0.2    0.2(C)

            |     /   \

            C   0.15   0.05(D)

                |      |

                D      E

```

**编码表**：

| 符号 | 概率 | 码字 | 码长 |
|------|------|------|------|
| A | 0.4 | 0 | 1 |
| B | 0.2 | 10 | 2 |
| C | 0.2 | 110 | 3 |
| D | 0.15 | 1110 | 4 |
| E | 0.05 | 1111 | 4 |

**期望码长**：
$$L = 0.4 \times 1 + 0.2 \times 2 + 0.2 \times 3 + 0.15 \times 4 + 0.05 \times 4 = 2.2 \text{ bits}$$

**信源熵**：
$$H(X) = -0.4\log_2 0.4 - 0.2\log_2 0.2 - 0.2\log_2 0.2 - 0.15\log_2 0.15 - 0.05\log_2 0.05$$
$$\approx 2.04 \text{ bits}$$

**编码效率**：
$$\eta = \frac{H(X)}{L} = \frac{2.04}{2.2} \approx 92.7\%$$

## 问题4：互信息应用

### 问题描述
设 $X$ 和 $Y$ 的联合分布为：

| | Y=0 | Y=1 |
|---|---|---|
| X=0 | 0.3 | 0.2 |
| X=1 | 0.1 | 0.4 |

计算：
a) $H(X), H(Y)$
b) $H(X, Y)$
c) $H(X|Y), H(Y|X)$

d) $I(X; Y)$

### 解答

**边缘分布**：
- $P(X=0) = 0.5, P(X=1) = 0.5$
- $P(Y=0) = 0.4, P(Y=1) = 0.6$

**a) 边缘熵**：

$$H(X) = H_b(0.5) = 1 \text{ bit}$$

$$H(Y) = -0.4\log_2 0.4 - 0.6\log_2 0.6 \approx 0.971 \text{ bits}$$

**b) 联合熵**：

$$H(X, Y) = -0.3\log_2 0.3 - 0.2\log_2 0.2 - 0.1\log_2 0.1 - 0.4\log_2 0.4$$
$$\approx 1.846 \text{ bits}$$

**c) 条件熵**：

$$H(X|Y) = H(X, Y) - H(Y) = 1.846 - 0.971 = 0.875 \text{ bits}$$

$$H(Y|X) = H(X, Y) - H(X) = 1.846 - 1 = 0.846 \text{ bits}$$

**d) 互信息**：

$$I(X; Y) = H(X) - H(X|Y) = 1 - 0.875 = 0.125 \text{ bits}$$

验证：
$$I(X; Y) = H(X) + H(Y) - H(X, Y) = 1 + 0.971 - 1.846 = 0.125 \text{ bits}$$ ✓

**结论**：$X$ 和 $Y$ 不是独立的（$I(X;Y) > 0$），也不是完全相关的（$I(X;Y) < \min(H(X), H(Y))$）。

## 问题5：率失真函数

### 问题描述

a) 计算伯努利信源（$p=0.5$）在汉明失真下的率失真函数

b) 计算高斯信源（$\sigma^2 = 1$）在平方误差失真下的率失真函数，$D = 0.1$

### 解答

**a) 伯努利信源**：

对于二元信源和汉明失真，率失真函数为：

$$R(D) = \begin{cases} H_b(p) - H_b(D) & 0 \leq D \leq \min(p, 1-p) \\ 0 & D > \min(p, 1-p) \end{cases}$$

当 $p = 0.5$：
$$R(D) = \begin{cases} 1 - H_b(D) & 0 \leq D \leq 0.5 \\ 0 & D > 0.5 \end{cases}$$

计算几个点：
- $D = 0$: $R(0) = 1$ bit
- $D = 0.1$: $R(0.1) = 1 - 0.469 = 0.531$ bits
- $D = 0.5$: $R(0.5) = 0$

**b) 高斯信源**：

$$R(D) = \frac{1}{2}\log_2\frac{\sigma^2}{D}, \quad 0 \leq D \leq \sigma^2$$

当 $\sigma^2 = 1, D = 0.1$：

$$R(0.1) = \frac{1}{2}\log_2\frac{1}{0.1} = \frac{1}{2}\log_2 10 \approx 1.66 \text{ bits}$$

**解释**：
- 允许失真 $D = 0.1$ 时，最小码率约为 1.66 bits
- 对比无损编码（$D = 0$）需要无穷大码率
- 当 $D = 1 = \sigma^2$ 时，$R(D) = 0$（直接估计为0即可）
