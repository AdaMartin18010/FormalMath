## 1. 问题建模

### 1.1 公钥密码学原理

**问题**: 在不安全信道上建立安全通信，无需预先共享密钥。

**解决方案**:
- **公钥**: 公开，用于加密
- **私钥**: 保密，用于解密
- **数学基础**: 单向陷门函数

### 1.2 RSA加密系统

**密钥生成**:
1. 选择两个大素数 $p, q$
2. 计算 $N = pq$, $\phi(N) = (p-1)(q-1)$
3. 选择 $e$ 使 $\gcd(e, \phi(N)) = 1$
4. 计算 $d \equiv e^{-1} \pmod{\phi(N)}$

**公钥**: $(N, e)$  
**私钥**: $(N, d)$

**加密**: $c = m^e \pmod{N}$  
**解密**: $m = c^d \pmod{N}$

**正确性** (由欧拉定理):
$$c^d = (m^e)^d = m^{ed} = m^{1 + k\phi(N)} \equiv m \pmod{N}$$

## 2. 数学理论

### 2.1 数论基础

**定理 2.1** (欧拉定理): 若 $\gcd(a, n) = 1$，则
$$a^{\phi(n)} \equiv 1 \pmod{n}$$

**定理 2.2** (中国剩余定理): 设 $n_1, \ldots, n_k$ 两两互素，则同余方程组
$$x \equiv a_i \pmod{n_i}, \quad i = 1, \ldots, k$$
有唯一解模 $N = n_1 \cdots n_k$。

### 2.2 RSA安全性分析

**攻击方式**:
1. **因数分解攻击**: 分解 $N$ 得到 $p, q$，从而计算私钥
2. **低加密指数攻击**: 小 $e$ 可能导致安全问题
3. **选择密文攻击**: 需要填充方案(如OAEP)

**安全参数**:
- 当前推荐: 2048-bit RSA ( $p, q$ 各约1024位)
- 量子攻击: Shor算法可在多项式时间分解

### 2.3 椭圆曲线密码学 (ECC)

**椭圆曲线方程**: $y^2 = x^3 + ax + b$，判别式 $\Delta = -16(4a^3 + 27b^2) \neq 0$

**点加法**: 几何定义，形成阿贝尔群

**椭圆曲线离散对数问题 (ECDLP)**: 
- 给定 $P, Q = kP$，求 $k$ 是困难的
- 比整数分解更难，因此ECC可用更短密钥达到同等安全

**密钥长度对比**:
| 安全级别 | RSA | ECC |
|----------|-----|-----|
| 80-bit | 1024 | 160 |
| 128-bit | 3072 | 256 |
| 256-bit | 15360 | 512 |

## 3. 算法与方法

### 3.1 快速模幂运算

**平方-乘算法**:
```
输入: 底数 a, 指数 e, 模数 n
输出: a^e mod n

result = 1
while e > 0:
    if e是奇数:
        result = (result * a) mod n
    a = (a * a) mod n
    e = e // 2
return result
```

**复杂度**: $O(\log e)$ 次模乘

### 3.2 素数测试

**Miller-Rabin测试**: 
- 概率性测试
- 经过 $k$ 轮后，错误概率 $\leq 4^{-k}$

### 3.3 密钥派生

**PKCS#1 v1.5填充**:
```
EB = 0x00 || 0x02 || PS || 0x00 || M
```

**OAEP填充** (更安全的填充方案):
- 使用哈希函数和掩码生成函数
- 可证明安全

## 4. 具体案例

### 4.1 案例：RSA密钥生成与加密

**参数** (小数值便于演示):
- $p = 61$, $q = 53$
- $N = 3233$, $\phi(N) = 3120$
- $e = 17$ (常用值)
- $d = 2753$ (因为 $17 \times 2753 = 46801 = 15 \times 3120 + 1$)

**加密消息** $m = 123$:
$$c = 123^{17} \mod 3233 = 855$$

**解密**:
$$m = 855^{2753} \mod 3233 = 123$$

### 4.2 案例：ECC密钥交换 (ECDH)

**曲线**: $y^2 = x^3 + 2x + 2$ over $\mathbb{F}_{17}$

**基点**: $G = (5, 1)$，阶为 $n = 19$

**协议流程**:
1. Alice选择私钥 $a = 7$，发送 $A = 7G = (10, 6)$ 给Bob
2. Bob选择私钥 $b = 13$，发送 $B = 13G = (7, 11)$ 给Alice
3. Alice计算共享密钥: $S = aB = 7 \times 13G = 91G = (2, 7)$
4. Bob计算共享密钥: $S = bA = 13 \times 7G = 91G = (2, 7)$

**安全性**: 基于ECDLP的困难性

## 5. 代码实现

### 5.1 RSA实现 (Python)

```python
import random
from math import gcd

def mod_exp(base, exp, mod):
    """快速模幂运算"""
    result = 1
    base = base % mod
    while exp > 0:
        if exp & 1:
            result = (result * base) % mod
        base = (base * base) % mod
        exp >>= 1
    return result

def extended_gcd(a, b):
    """扩展欧几里得算法"""
    if a == 0:
        return b, 0, 1
    gcd_val, x1, y1 = extended_gcd(b % a, a)
    x = y1 - (b // a) * x1
    y = x1
    return gcd_val, x, y

def mod_inverse(a, m):
    """计算模逆元"""
    gcd_val, x, _ = extended_gcd(a, m)
    if gcd_val != 1:
        return None
    return (x % m + m) % m

def is_prime(n, k=10):
    """Miller-Rabin素数测试"""
    if n < 2: return False
    if n == 2 or n == 3: return True
    if n % 2 == 0: return False
    
    # 写成 n-1 = 2^r * d
    r, d = 0, n - 1
    while d % 2 == 0:
        r += 1
        d //= 2
    
    for _ in range(k):
        a = random.randrange(2, n - 1)
        x = mod_exp(a, d, n)
        
        if x == 1 or x == n - 1:
            continue
            
        for _ in range(r - 1):
            x = mod_exp(x, 2, n)
            if x == n - 1:
                break
        else:
            return False
    return True

def generate_prime(bits):
    """生成指定位数的素数"""
    while True:
        n = random.getrandbits(bits) | 1 | (1 << (bits - 1))
        if is_prime(n):
            return n

class RSA:
    def __init__(self, key_bits=512):
        """生成RSA密钥对"""
        # 生成两个素数
        p = generate_prime(key_bits // 2)
        q = generate_prime(key_bits // 2)
        
        self.n = p * q
        phi = (p - 1) * (q - 1)
        
        # 选择公钥指数
        self.e = 65537  # 常用值
        while gcd(self.e, phi) != 1:
            self.e = random.randrange(3, phi, 2)
        
        # 计算私钥
        self.d = mod_inverse(self.e, phi)
        
        print(f"公钥: (e={self.e}, n={self.n})")
        print(f"私钥: (d={self.d}, n={self.n})")
    
    def encrypt(self, message):
        """加密 (消息必须小于n)"""
        if message >= self.n:
            raise ValueError("Message too large")
        return mod_exp(message, self.e, self.n)
    
    def decrypt(self, ciphertext):
        """解密"""
        return mod_exp(ciphertext, self.d, self.n)

# 测试
print("=" * 50)
print("RSA加密演示")
print("=" * 50)

rsa = RSA(key_bits=256)  # 小位数便于演示

message = 12345
print(f"\n原始消息: {message}")

ciphertext = rsa.encrypt(message)
print(f"加密后: {ciphertext}")

decrypted = rsa.decrypt(ciphertext)
print(f"解密后: {decrypted}")

assert decrypted == message, "解密失败"
print("\n✓ RSA加密解密成功！")
```

### 5.2 椭圆曲线实现

```python
class EllipticCurve:
    def __init__(self, a, b, p):
        """
        椭圆曲线 y^2 = x^3 + ax + b (mod p)
        """
        self.a = a
        self.b = b
        self.p = p
        
        # 验证非奇异
        assert (4 * a**3 + 27 * b**2) % p != 0, "曲线奇异"
    
    def is_on_curve(self, point):
        """检查点是否在曲线上"""
        if point is None:
            return True
        x, y = point
        return (y**2 - x**3 - self.a * x - self.b) % self.p == 0
    
    def add(self, P, Q):
        """椭圆曲线点加法"""
        if P is None:
            return Q
        if Q is None:
            return P
        
        x1, y1 = P
        x2, y2 = Q
        
        if x1 == x2 and y1 != y2:
            return None  # 无穷远点
        
        if x1 == x2:
            # 点加倍
            m = (3 * x1**2 + self.a) * mod_inverse(2 * y1, self.p) % self.p
        else:
            # 点加法
            m = (y2 - y1) * mod_inverse(x2 - x1, self.p) % self.p
        
        x3 = (m**2 - x1 - x2) % self.p
        y3 = (m * (x1 - x3) - y1) % self.p
        
        return (x3, y3)
    
    def scalar_mult(self, k, P):
        """标量乘法 k*P"""
        result = None
        addend = P
        
        while k:
            if k & 1:
                result = self.add(result, addend)
            addend = self.add(addend, addend)
            k >>= 1
        
        return result

# 测试ECC
print("\n" + "=" * 50)
print("椭圆曲线密码演示")
print("=" * 50)

# secp256k1参数简化版
p = 2**256 - 2**32 - 977
a, b = 0, 7

# 使用小曲线测试
curve = EllipticCurve(2, 3, 97)
G = (3, 6)  # 基点

print(f"曲线: y^2 = x^3 + {curve.a}x + {curve.b} (mod {curve.p})")
print(f"基点 G = {G}")

# 密钥生成
private_key = 7
public_key = curve.scalar_mult(private_key, G)
print(f"\n私钥: {private_key}")
print(f"公钥: {public_key}")

# 验证点在曲线上
assert curve.is_on_curve(public_key), "公钥不在曲线上"
print("\n✓ ECC密钥生成成功！")
```
