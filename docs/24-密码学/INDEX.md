## 学习路径

### 基础路径
```
数论基础 → 对称加密 → 公钥加密 → 数字签名
```

### 进阶路径
```
公钥加密 → 零知识证明 → 同态加密 → 后量子密码
```

### 应用路径
- **网络安全**: TLS/SSL协议 + 证书体系 + 密钥管理
- **区块链**: 哈希函数 + 数字签名 + 共识机制
- **隐私计算**: 同态加密 + 安全多方计算 + 零知识证明

## 核心概念速查

| 概念 | 描述 | 安全基础 |
|------|------|----------|
| **单向函数** | 易计算、难逆 | 计算复杂性 |
| **陷门函数** | 有秘密信息则易逆 | 数论难题 |
| **语义安全** | 密文不泄露信息 | 概率加密 |
| **IND-CCA2** | 自适应选择密文安全 | 强安全定义 |
| **零知识** | 证明者不泄露知识 | 交互协议 |

---

## 密码体制对比

| 体制 | 密钥类型 | 速度 | 主要用途 |
|------|----------|------|----------|
| AES | 对称 | 快(Gbps) | 数据加密 |
| RSA | 非对称 | 慢(Kbps) | 密钥交换、签名 |
| ECC | 非对称 | 中等 | 移动设备、区块链 |
| ChaCha20 | 对称 | 快 | 现代流加密 |
