# 数论高级习题

本节汇集初等数论与代数数论中的经典难题与证明技巧，涵盖素数分布、二次剩余、Diophantine 方程与算术函数等核心主题。

## 1. 素数与算术级数

### 1.1 Dirichlet 定理的特殊情形

**习题 1.1**. 证明算术级数 $4n + 3$ 中有无穷多个素数。

**证明**. 假设只有有限个这样的素数 $p_1, \dots, p_k$。考虑：

$$N = 4p_1 p_2 \cdots p_k - 1.$$

$N \equiv 3 \pmod{4}$ 且 $N > 1$，故 $N$ 必有素因子。$N$ 的素因子不能全 $\equiv 1 \pmod{4}$（否则 $N \equiv 1 \pmod{4}$），故存在素因子 $p \equiv 3 \pmod{4}$。但 $p \neq p_i$（因 $N \equiv -1 \pmod{p_i}$），矛盾。$\square$

### 1.2 素数定理的初等形式

**习题 1.2（Chebyshev）**. 证明存在常数 $c_1, c_2 > 0$ 使得对所有充分大的 $x$：

$$c_1 \frac{x}{\ln x} \leq \pi(x) \leq c_2 \frac{x}{\ln x},$$

其中 $\pi(x)$ 为不超过 $x$ 的素数个数。

**证明概要**. 考虑 $\binom{2n}{n}$。其素因子均 $\leq 2n$，且每个素数 $p$ 的幂次为 $\sum_{k \geq 1} (\lfloor 2n/p^k \rfloor - 2\lfloor n/p^k \rfloor) \leq \log_p(2n)$。故：

$$\binom{2n}{n} \leq \prod_{p \leq 2n} p^{\log_p(2n)} = (2n)^{\pi(2n)}.$$

又 $\binom{2n}{n} \geq \frac{4^n}{2n+1}$，取对数得下界。上界利用 $n\# = \prod_{p \leq n} p \leq 4^n$（$n\#$ 为素数阶乘）。$\square$

## 2. 二次剩余

### 2.1 Legendre 符号的计算

**习题 2.1**. 对奇素数 $p$，证明：

$$\left(\frac{2}{p}\right) = (-1)^{\frac{p^2-1}{8}} = \begin{cases} 1 & p \equiv \pm 1 \pmod{8}, \\ -1 & p \equiv \pm 3 \pmod{8}. \end{cases}$$

**证明**. 考虑 Gauss 引理：$\left(\frac{a}{p}\right) = (-1)^\mu$，其中 $\mu$ 为 $a, 2a, \dots, \frac{p-1}{2}a$ 模 $p$ 最小正剩余中大于 $p/2$ 的个数。

对 $a = 2$，序列 $2, 4, \dots, p-1$（或 $p-3$）的最小正剩余。当 $p = 8k + 1$ 时，所有剩余均 $< p/2$，$\mu = 0$。当 $p = 8k + 3$ 时，后半部分 $> p/2$，$\mu$ 为奇数。系统计算得公式。$\square$

### 2.2 二次互反律

**习题 2.2（Gauss 二次互反律）**. 设 $p, q$ 为相异奇素数，则：

$$\left(\frac{p}{q}\right)\left(\frac{q}{p}\right) = (-1)^{\frac{p-1}{2} \cdot \frac{q-1}{2}}.$$

**证明概要（Gauss 和法）**. 定义二次 Gauss 和：

$$g_p = \sum_{n=0}^{p-1} \left(\frac{n}{p}\right) e^{2\pi i n/p}.$$

可证 $g_p^2 = \left(\frac{-1}{p}\right) p$。计算 $g_p^{q-1} \pmod{q}$（在环 $\mathbb{Z}[\zeta_p]$ 中）并利用 $g_p^q \equiv g_p \left(\frac{q}{p}\right) \pmod{q}$，比较得互反律。$\square$

## 3. Diophantine 方程

### 3.1 Fermat 大定理的初等情形

**习题 3.1**. 证明 $x^4 + y^4 = z^2$ 无正整数解（从而 Fermat 大定理对 $n=4$ 成立）。

**证明（Fermat 的无穷递降法）**. 假设存在解，取 $z$ 最小。可设 $\gcd(x,y) = 1$ 且 $x$ 偶 $y$ 奇。则 $(x^2, y^2, z)$ 为毕达哥拉斯三元组：

$$x^2 = 2mn, \quad y^2 = m^2 - n^2, \quad z = m^2 + n^2,$$

其中 $m > n$，$\gcd(m,n) = 1$，一奇一偶。由 $y^2 + n^2 = m^2$，$(y, n, m)$ 也是本原毕达哥拉斯三元组，故：

$$n = 2rs, \quad y = r^2 - s^2, \quad m = r^2 + s^2.$$

于是 $x^2 = 2mn = 4rs(r^2 + s^2)$。因 $r, s, r^2 + s^2$ 两两互素，它们均为完全平方：$r = a^2$，$s = b^2$，$r^2 + s^2 = c^2$。即 $a^4 + b^4 = c^2$，且 $c \leq m \leq \sqrt{z} < z$，与 $z$ 的最小性矛盾。$\square$

### 3.2 Pell 方程

**习题 3.2**. 设 $d > 1$ 为非平方正整数。证明 Pell 方程 $x^2 - dy^2 = 1$ 有无穷多组正整数解。

**证明**. 由 Dirichlet 逼近定理，存在无穷多对 $(p,q)$ 使 $|\sqrt{d} - p/q| < 1/q^2$。则：

$$|p^2 - dq^2| = |p - q\sqrt{d}| \cdot |p + q\sqrt{d}| < \frac{1}{q} \cdot (2q\sqrt{d} + 1) < 2\sqrt{d} + 1.$$

故 $|p^2 - dq^2|$ 有界且取整数值，某值 $N$ 出现无穷多次。取两组不同的 $(p_1, q_1), (p_2, q_2)$ 使 $p_i^2 - dq_i^2 = N$。考虑：

$$\frac{p_1 + q_1\sqrt{d}}{p_2 + q_2\sqrt{d}} = \frac{(p_1p_2 - dq_1q_2) + (p_2q_1 - p_1q_2)\sqrt{d}}{N}.$$

其分子给出 $x^2 - dy^2 = 1$ 的解。由连分数理论，最小解可通过 $\sqrt{d}$ 的连分数展开求得，所有解为其幂次。$\square$

## 4. 算术函数

### 4.1 Möbius 反演

**习题 4.1**. 证明 Möbius 反演公式：若 $f(n) = \sum_{d|n} g(d)$，则 $g(n) = \sum_{d|n} \mu(d) f(n/d)$，其中 $\mu$ 为 Möbius 函数。

**证明**. 利用 Dirichlet 卷积：$f = g * 1$，其中 $1(n) = 1$。因 $\mu * 1 = \varepsilon$（单位函数），$g = f * \mu$。$\square$

### 4.2 Euler $\varphi$ 函数的性质

**习题 4.2**. 证明 $\sum_{d|n} \varphi(d) = n$，并由此推出 $\varphi(n) = n \prod_{p|n}(1 - 1/p)$。

**证明**. 考虑分式 $k/n$（$1 \leq k \leq n$），化为最简分数后分母 $d|n$，分子与 $d$ 互素且 $\leq d$，恰有 $\varphi(d)$ 个。故 $n = \sum_{d|n} \varphi(d)$。

由 Möbius 反演：$\varphi(n) = \sum_{d|n} \mu(d) \frac{n}{d} = n \sum_{d|n} \frac{\mu(d)}{d}$。利用积性即得乘积公式。$\square$

## 5. 例子

### 5.1 例子：具体 Pell 方程求解

**习题 5.1**. 求 $x^2 - 2y^2 = 1$ 的最小正整数解及通解公式。

**解答**. $\sqrt{2}$ 的连分数展开为 $[1; \overline{2}]$，其渐近分数 $p_k/q_k$。计算得 $p_1/q_1 = 3/2$，验证 $3^2 - 2 \cdot 2^2 = 1$。故最小解 $(x_1, y_1) = (3, 2)$。

通解由 $(3 + 2\sqrt{2})^k = x_k + y_k\sqrt{2}$ 给出。例如 $(3+2\sqrt{2})^2 = 17 + 12\sqrt{2}$，$(x_2, y_2) = (17, 12)$。

### 5.2 例子：高斯整数中的素数分解

**习题 5.2**. 在 $\mathbb{Z}[i]$ 中分解 $5$ 和 $13$，并判断 $3$ 是否素。

**解答**. $5 = (2+i)(2-i)$，$13 = (3+2i)(3-2i)$。范数 $N(a+bi) = a^2 + b^2$。若 $3 = \alpha\beta$ 非平凡，则 $N(3) = 9 = N(\alpha)N(\beta)$，故 $N(\alpha) = 3$。但 $a^2 + b^2 = 3$ 无整数解，故 $3$ 在 $\mathbb{Z}[i]$ 中不可约（从而是素元，因 $\mathbb{Z}[i]$ 是 UFD）。$\square$
