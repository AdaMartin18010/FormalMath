## 🎯 学习路径

### 进阶路线（研究生高阶）

```

基础数论 → 代数数论/解析数论 → 本目录深度扩展

```

**推荐学习序列：**

1. **起点**：[05-解析数论-深度扩展版](05-解析数论-深度扩展版.md) — 掌握Dirichlet特征、L函数解析延拓、算术级数PNT
2. **深化**：选择以下方向之一
   - **代数方向**：[07-代数数论深化-类域论基础](07-代数数论深化-类域论基础.md) → 类域论 → Langlands纲领
   - **几何方向**：[08-椭圆曲线算术-深度扩展版](08-椭圆曲线算术-深度扩展版.md) → 算术几何
   - **模形式**：[06-模形式-深度扩展版](06-模形式-深度扩展版.md) → 自守形式
