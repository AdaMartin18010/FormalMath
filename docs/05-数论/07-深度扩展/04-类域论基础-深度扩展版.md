## 一、历史背景：Hilbert第12问题

**Hilbert第12问题** 是David Hilbert于1900年在巴黎国际数学家大会上提出的23个问题之一，其陈述为：

> 对于任意代数数域 $K$，如何构造所有阿贝尔扩张的生成元，类似于Kronecker-Weber定理中分圆域对 $\mathbb{Q}$ 的情形？

**Kronecker-Weber定理**（**定理 1.1**）：
$$\text{$\mathbb{Q}$ 的每个阿贝尔扩张都包含在某个分圆域 $\mathbb{Q}(\zeta_n)$ 中}$$

该定理表明，$  ext{Gal}(\mathbb{Q}(\zeta_n)/\mathbb{Q}) \cong (\mathbb{Z}/n\mathbb{Z})^\times$ 的结构完全描述了 $\mathbb{Q}$ 的阿贝尔扩张。

**关键问题**：对于一般数域 $K$，是否存在类似的"分圆"构造？

**历史发展脉络**：

| 年代 | 数学家 | 贡献 |
|------|--------|------|
| 1896-1897 | Hilbert | 提出类域概念，证明主理想定理 |
| 1920s | Takagi | 完成类域论的一般理论 |
| 1927 | Artin | 证明互反律，建立Artin映射 |
| 1930s | Chevalley | 引入idele，简化理论 |
| 1950s | Tate | 上同调方法，统一表述 |

## 二、Hilbert类域

### 2.1 定义与基本性质

**定义 2.1**（Hilbert类域）：设 $K$ 是数域，$H(K)$ 称为 $K$ 的**Hilbert类域**，如果：

1. $H(K)/K$ 是 $K$ 的极大非分歧阿贝尔扩张
2. $\text{Gal}(H(K)/K) \cong \text{Cl}(K)$（类群同构）

**定理 2.2**（Hilbert类域的存在性）：对任意数域 $K$，Hilbert类域 $H(K)$ 存在且唯一。

**证明思路**（概要）：

- 利用全局类域论的存在定理
- 对应于 $\text{Cl}(K)$ 的商结构
- 非分歧条件对应于所有素理想上的温和分歧约束

**例 2.3**（虚二次域的Hilbert类域）：

设 $K = \mathbb{Q}(\sqrt{-23})$，则：

- $d_K = -23$
- 类数 $h_K = 3$
- Hilbert类域 $H(K) = K(\alpha)$，其中 $\alpha$ 是 $x^3 - x - 1 = 0$ 的实根

验证：$x^3 - x - 1$ 的判别式为 $-23$，且 $[H(K):K] = 3 = h_K$。

### 2.2 主理想定理

**定理 2.4**（Hilbert主理想定理）：设 $H$ 是 $K$ 的Hilbert类域，$\mathfrak{a}$ 是 $\mathcal{O}_K$ 的任意理想。则 $\mathfrak{a}\mathcal{O}_H$ 是主理想。

**意义**：$H/K$ 的扩张"解决"了 $K$ 的理想类结构——所有理想在 $H$ 中都成为主理想。

**证明框架**（Artin, 1927）：

1. 构造Artin同构 $\text{Gal}(H/K) \cong \text{Cl}(K)$
2. 利用转移映射（transfer）证明：类群到自身的平凡性
3. 应用群论中的转移原理：$V: G/[G,G] \to H/[H,H]$ 在Gal情形下为平凡

**推论 2.5**：$K$ 的理想类群通过扩张到Hilbert类域被"消解"。

## 三、Artin互反律

### 3.1 Frobenius元素

**定义 3.1**（Frobenius元素）：设 $L/K$ 是伽罗瓦扩张，$\mathfrak{p}$ 是 $\mathcal{O}_K$ 的素理想，$\mathfrak{P}$ 是 $\mathcal{O}_L$ 位于 $\mathfrak{p}$ 上的素理想。定义**Frobenius元素** $\left(\frac{L/K}{\mathfrak{P}}\right) \in \text{Gal}(L/K)$ 为满足：

$$\sigma(\alpha) \equiv \alpha^{N(\mathfrak{p})} \pmod{\mathfrak{P}}, \quad \forall \alpha \in \mathcal{O}_L$$

的唯一元素。

**定理 3.2**（Frobenius元素的性质）：

1. $\left(\frac{L/K}{\mathfrak{P}}\right)$ 的阶等于 $\mathfrak{p}$ 在 $L$ 中的惯性度 $f(\mathfrak{P}|\mathfrak{p})$

2. 若 $L/K$ 阿贝尔，则Frobenius元素仅依赖于 $\mathfrak{p}$，记为 $\left(\frac{L/K}{\mathfrak{p}}\right)$

### 3.2 Artin映射的构造

**定义 3.3**（Artin映射）：设 $L/K$ 是阿贝尔扩张，$\mathfrak{m}$ 是 $K$ 的一个modulus（形式素理想积），定义**Artin映射**：

$$\phi_{L/K}: I_K^{\mathfrak{m}} \to \text{Gal}(L/K)$$
$$\phi_{L/K}\left(\prod_{\mathfrak{p} \nmid \mathfrak{m}} \mathfrak{p}^{n_{\mathfrak{p}}}\right) = \prod_{\mathfrak{p} \nmid \mathfrak{m}} \left(\frac{L/K}{\mathfrak{p}}\right)^{n_{\mathfrak{p}}}$$

其中 $I_K^{\mathfrak{m}}$ 是与 $\mathfrak{m}$ 互素的理想群。

**定理 3.4**（Artin互反律，1927）：设 $L/K$ 是有限阿贝尔扩张，则：

1. Artin映射是满射
2. 存在modulus $\mathfrak{m}$（导子），使得 $\ker(\phi_{L/K}) = \mathfrak{N}_{L/K}(I_L^{\mathfrak{m}}) \cdot P_K^{\mathfrak{m}}$

由此得到同构：
$$\text{Gal}(L/K) \cong I_K^{\mathfrak{m}} / \mathfrak{N}_{L/K}(I_L^{\mathfrak{m}}) P_K^{\mathfrak{m}}$$

**例 3.5**（二次互反律作为Artin互反律的特例）：

设 $K = \mathbb{Q}$，$L = \mathbb{Q}(\sqrt{d})$，$d$ 无平方因子。则对奇素数 $p \nmid d$：
$$\left(\frac{L/K}{p}\right) = \left(\frac{d}{p}\right) \in \{\pm 1\} \cong \text{Gal}(L/K)$$

Artin互反律导出二次互反律：$\left(\frac{p}{q}\right)\left(\frac{q}{p}\right) = (-1)^{\frac{p-1}{2}\cdot\frac{q-1}{2}}$。

## 四、Adele与Idele

### 4.1 Adele环的构造

**定义 4.1**（$p$-进数回顾）：对素数 $p$，$\mathbb{Q}_p$ 是 $\mathbb{Q}$ 关于 $p$-进绝对值 $|\cdot|_p$ 的完备化。

**定义 4.2**（Adele环）：数域 $K$ 的**Adele环**定义为：
$$\mathbb{A}_K = \left\{ (x_v)_v \in \prod_v K_v : x_v \in \mathcal{O}_v \text{ 对几乎所有 } v \right\}$$

其中 $v$ 遍历所有位（place），$K_v$ 是完备化，$\mathcal{O}_v$ 是整数环。

**定理 4.3**（Adele环的拓扑性质）：

1. $\mathbb{A}_K$ 是局部紧拓扑环
2. $K$ 通过对角嵌入 $x \mapsto (x, x, \ldots)$ 在 $\mathbb{A}_K$ 中是离散且余紧的

### 4.2 Idele群与类群

**定义 4.4**（Idele群）：$K$ 的**Idele群**是Adele环中的单位群：
$$\mathbb{I}_K = \mathbb{A}_K^\times = \{(x_v)_v \in \prod_v K_v^\times : x_v \in \mathcal{O}_v^\times \text{ 对几乎所有 } v\}$$

**定义 4.5**（Idele类群）：**Idele类群**定义为：
$$C_K = \mathbb{I}_K / K^\times$$

**定理 4.6**（Idele与理想的联系）：存在典范满射 $\mathbb{I}_K \to I_K$（理想群），诱导同构：
$$C_K / C_K^0 \cong \text{Cl}(K)$$

其中 $C_K^0$ 是范数1的idele类子群。

**定理 4.7**（类域论的Idele表述）：设 $L/K$ 是有限阿贝尔扩张，则Artin互反律可表述为：
$$\text{Gal}(L/K) \cong C_K / \text{N}_{L/K}(C_L)$$

**推论 4.8**（局部-整体原理）：全局Artin映射可由局部Artin映射的乘积构造：
$$\phi_{L/K} = \prod_v \phi_{L_w/K_v}$$

## 五、类域论的主定理

**定理 5.1**（类域论的对应）：设 $K$ 是数域，则：

1. **存在性定理**：对每个有限指数开子群 $H \subseteq C_K$，存在唯一的有限阿贝尔扩张 $L/K$ 使得 $\text{N}_{L/K}(C_L) = H$

2. **同构定理**：$\text{Gal}(L/K) \cong C_K / \text{N}_{L/K}(C_L)$

3. **包含对应**：$L_1 \subseteq L_2$ 当且仅当 $\text{N}_{L_2/K}(C_{L_2}) \subseteq \text{N}_{L_1/K}(C_{L_1})$

4. **分歧理论**：$\mathfrak{p}$ 在 $L$ 中分歧当且仅当 $\mathcal{O}_\mathfrak{p}^\times \not\subseteq \text{N}_{L/K}(C_L)$

**应用 5.2**（导子与判别式的关系）：对阿贝尔扩张 $L/K$，导子 $\mathfrak{f}(L/K)$ 满足：
$$\mathfrak{d}_{L/K} = \prod_{\chi} \mathfrak{f}_\chi$$

其中 $\chi$ 遍历 $\widehat{\text{Gal}(L/K)}$，$\mathfrak{f}_\chi$ 是特征标的导子。

**现代发展**：

| 方向 | 核心问题 | 进展 |
|------|----------|------|
| 非阿贝尔类域论 | Langlands纲领 | 局部Langlands对应已证 |
| 高维类域论 | 高维算术簇 | Kato-Saito理论 |
| 显式类域论 | 构造生成元 | Stark猜想，复乘理论 |
