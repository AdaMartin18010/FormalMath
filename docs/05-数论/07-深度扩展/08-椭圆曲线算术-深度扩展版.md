## 一、Weierstrass方程与判别式

### 1.1 短Weierstrass形式

**定义 1.1**（椭圆曲线）：域 $K$（特征 $\neq 2, 3$）上的**椭圆曲线**是光滑的射影三次曲线，仿射部分可写为**短Weierstrass形式**：
$$E: y^2 = x^3 + ax + b, \quad a, b \in K$$

**射影闭包**：齐次方程
$$Y^2 Z = X^3 + aXZ^2 + bZ^3$$

无穷远点（唯一的无穷远点）：$O = [0 : 1 : 0]$。

**例 1.2**：

**(a)** $E: y^2 = x^3 - x$（**同余数曲线**，与三角形面积问题相关）

**(b)** $E: y^2 = x^3 + 1$（Hasse研究过的曲线）

**(c)** Frey曲线：$y^2 = x(x - a^n)(x + b^n)$（费马大定理证明的核心）

**一般Weierstrass形式**（任意特征）：
$$y^2 + a_1 xy + a_3 y = x^3 + a_2 x^2 + a_4 x + a_6$$

### 1.2 判别式与j-不变量

**定义 1.3**（不变量）：设 $E: y^2 = x^3 + ax + b$，定义

- **判别式**：$\Delta = -16(4a^3 + 27b^2)$
- **$c_4$ 不变量**：$c_4 = -48a$
- **$j$-不变量**：$j(E) = -1728 \frac{(4a)^3}{\Delta} = 1728 \frac{4a^3}{4a^3 + 27b^2}$

**定理 1.4**（光滑性判别）：$E$ 是光滑曲线（即椭圆曲线）当且仅当 $\Delta \neq 0$。

**证明**：曲线在点 $(x_0, y_0)$ 奇异当且仅当
$$f = y^2 - x^3 - ax - b = 0, \quad \frac{\partial f}{\partial x} = \frac{\partial f}{\partial y} = 0$$

即 $2y_0 = 0$ 且 $3x_0^2 + a = 0$。若特征 $\neq 2$，则 $y_0 = 0$，代入得 $x_0^3 + ax_0 + b = 0$ 且 $3x_0^2 + a = 0$。消去 $x_0$ 得 $4a^3 + 27b^2 = 0$，即 $\Delta = 0$。

**例 1.5**：

- $E: y^2 = x^3 - x$：$\Delta = 64$，$j = 1728$
- $E: y^2 = x^3 + 1$：$\Delta = -432$，$j = 0$

**定理 1.6**（$j$-不变量的分类作用）：两个椭圆曲线 $E_1, E_2$ 在代数闭域上同构当且仅当 $j(E_1) = j(E_2)$。

**复乘曲线**：$j = 0$（$\mathbb{Z}[\omega]$，$\omega = e^{2\pi i/3}$）和 $j = 1728$（$\mathbb{Z}[i]$）对应具有复乘的椭圆曲线。

## 二、弦切法与群结构

### 2.1 几何加法法则

**历史背景**：Poincaré (1901) 首次系统阐述椭圆曲线的群结构，Diophantus、Newton、Abel、Jacobi 都有早期贡献。

**几何法则**：设 $P, Q \in E(K)$，定义 $P + Q$ 如下：

1. 画直线 $L$ 通过 $P$ 和 $Q$（若 $P = Q$，取切线）
2. $L$ 与 $E$ 交于第三点 $R$（计重数）
3. $P + Q = -R$（$R$ 关于 $x$ 轴的对称点）

**图示说明**（代数上）：若 $R = (x_R, y_R)$，则 $-R = (x_R, -y_R)$。

**定理 2.1**：上述运算使 $(E(K), +)$ 成为Abel群，单位元为无穷远点 $O$。

**性质**：

- $P + O = P$（对任意 $P$）
- $P + (-P) = O$，其中 $-(x, y) = (x, -y)$
- 结合律（非平凡，需验证）

### 2.2 代数公式

**定理 2.2**（加法公式）：设 $E: y^2 = x^3 + ax + b$，$P = (x_1, y_1)$，$Q = (x_2, y_2)$。

**情形 1**：$P \neq Q$，$x_1 \neq x_2$
$$\lambda = \frac{y_2 - y_1}{x_2 - x_1}, \quad x_3 = \lambda^2 - x_1 - x_2, \quad y_3 = \lambda(x_1 - x_3) - y_1$$

**情形 2**：$P = Q$（倍点公式）
$$\lambda = \frac{3x_1^2 + a}{2y_1}, \quad x_3 = \lambda^2 - 2x_1, \quad y_3 = \lambda(x_1 - x_3) - y_1$$

**例 2.3**：在 $E: y^2 = x^3 - x$ 上，计算 $2P$，$P = (2, \sqrt{6})$。

$$\lambda = \frac{3 \cdot 4 - 1}{2\sqrt{6}} = \frac{11}{2\sqrt{6}} = \frac{11\sqrt{6}}{12}$$

$$x_3 = \frac{121 \cdot 6}{144} - 4 = \frac{726 - 576}{144} = \frac{150}{144} = \frac{25}{24}$$

**挠点（Torsion Points）**：满足 $nP = O$ 的点 $P$ 称为 **$n$-挠点**。全体挠点构成挠子群 $E(K)_{tors}$。

**Mazur定理（1978）**：$E(\mathbb{Q})$ 的挠子群只有15种可能：
$$\mathbb{Z}/n\mathbb{Z}, \ 1 \leq n \leq 10, \ n = 12$$
或
$$\mathbb{Z}/2\mathbb{Z} \times \mathbb{Z}/2n\mathbb{Z}, \ 1 \leq n \leq 4$$

## 三、Mordell-Weil定理

### 3.1 定理陈述与证明思路

**历史背景**：Poincaré猜想椭圆曲线上有理点有限生成，Mordell (1922) 证明，Weil (1928) 推广到Abel簇。

**定理 3.1**（Mordell-Weil）：设 $E$ 是数域 $K$ 上的椭圆曲线，则 $E(K)$ 是有限生成Abel群：
$$E(K) \cong E(K)_{tors} \times \mathbb{Z}^r$$

其中 $r = r(E/K)$ 称为**秩**，$E(K)_{tors}$ 是有限挠子群。

**证明框架**（下降法）：

**步骤 1**：弱Mordell-Weil定理
$$E(K)/mE(K) \text{ 有限} \quad (m \geq 2)$$

**证明要点**：利用Kummer序列
$$0 \to E(K)/mE(K) \to H^1(G_K, E[m]) \to H^1(G_K, E)[m] \to 0$$

以及Hilbert 90和有限性定理。

**步骤 2**：高度函数

**定义 3.2**（典范高度）：对 $P \in E(\mathbb{Q})$，定义**对数高度**
$$h(P) = \frac{1}{2} \log \max(|x(P)|, 1)$$

**典范高度**（Néron-Tate高度）：
$$\hat{h}(P) = \lim_{n \to \infty} \frac{h(2^n P)}{4^n}$$

**性质**：

1. $\hat{h}(P) \geq 0$，等号当且仅当 $P$ 是挠点
2. $\hat{h}(mP) = m^2 \hat{h}(P)$
3. 配对 $\langle P, Q \rangle = \hat{h}(P+Q) - \hat{h}(P) - \hat{h}(Q)$ 是双线性的

**步骤 3**：下降论证

利用高度函数的有界性，从 $E(K)/mE(K)$ 有限推出 $E(K)$ 有限生成。

### 3.2 秩的计算与实例

**秩的意义**：秩 $r$ 测量椭圆曲线有理点的"丰富程度"。$r = 0$ 时只有有限个有理点（挠点）；$r \geq 1$ 时有无穷多个有理点。

**计算方法**：

1. **解析秩**：由BSD猜想，$r = \text{ord}_{s=1} L(s, E)$
2. **代数秩**：通过2-下降、4-下降等算法计算
3. **Heegner点**：构造具有特定高度的点

**例 3.3**（秩0的例子）：$E: y^2 = x^3 - x$

- $j = 1728$，复乘于 $\mathbb{Z}[i]$
- $E(\mathbb{Q})_{tors} = \mathbb{Z}/2\mathbb{Z} \times \mathbb{Z}/2\mathbb{Z} = \{O, (0,0), (1,0), (-1,0)\}$
- $r = 0$

**例 3.4**（秩1的例子）：$E: y^2 = x^3 - 2$

- 点 $P = (3, 5)$ 是无限阶点
- $r = 1$，$E(\mathbb{Q}) \cong \mathbb{Z}$

**例 3.5**（高秩曲线）：截至2024年，已知最高秩的椭圆曲线：

- Elkies (2006)：秩 $\geq 28$ 的曲线
- Noether Institute (2024)：秩 $\geq 29$ 的曲线族

**猜想**（Rank Conjecture）：椭圆曲线的秩可以任意大，但分布稀疏。

## 四、BSD猜想与已知结果

### 4.1 猜想陈述

**历史背景**：Birch和Swinnerton-Dyer (1960s) 基于数值计算提出，是Clay数学研究所七大千禧年难题之一。

**定义 4.1**（Hasse-Weil L函数）：设 $E/\mathbb{Q}$，定义
$$L(s, E) = \prod_p L_p(s, E)^{-1}$$

其中对好素数 $p$，
$$L_p(s, E) = 1 - a_p p^{-s} + p^{1-2s}, \quad a_p = p + 1 - |E(\mathbb{F}_p)|$$

**定理 4.2**（Wiles等）：$L(s, E)$ 可全纯延拓到 $\mathbb{C}$，满足函数方程
$$\Lambda(s, E) = N_E^{s/2} (2\pi)^{-s} \Gamma(s) L(s, E) = \varepsilon(E) \Lambda(2-s, E)$$

**猜想 4.3**（Birch-Swinnerton-Dyer）：

**BSD I（秩的猜想）**：
$$\text{ord}_{s=1} L(s, E) = r(E)$$

**BSD II（精确公式）**：设 $r = \text{ord}_{s=1} L(s, E)$，则
$$\lim_{s \to 1} \frac{L(s, E)}{(s-1)^r} = \frac{\Omega_E \cdot \text{Reg}_E \cdot \#Ш(E/\mathbb{Q}) \cdot \prod_p c_p}{(\#E(\mathbb{Q})_{tors})^2}$$

其中：

- $\Omega_E$：实周期
- $\text{Reg}_E$：调节子（高度配对的行列式）
- $Ш(E/\mathbb{Q})$：Tate-Shafarevich群（测量局部-整体原则失效）
- $c_p$：Tamagawa数

### 4.2 已知结果与2024-2025进展

**定理 4.4**（Coates-Wiles, 1977）：若 $E$ 具有复乘且 $L(1, E) \neq 0$，则 $r = 0$。

**定理 4.5**（Gross-Zagier, 1986；Kolyvagin, 1988）：

- 若 $\text{ord}_{s=1} L(s, E) = 0$，则 $r = 0$
- 若 $\text{ord}_{s=1} L(s, E) = 1$，则 $r = 1$
- 在这些情况下，$Ш(E/\mathbb{Q})$ 有限

**定理 4.6**（Bhargava-Skinner-Zhang, 2014）：至少 $66\%$ 的椭圆曲线满足 BSD I（秩 $\leq 1$ 时）。

**2024-2025年重要进展**：

1. **$p$-adic BSD 进展**：
   - 新的 $p$-adic L函数构造技术
   - 对更广泛的椭圆曲线族证明 $p$-adic BSD

2. **Iwasawa理论**：
   - 主猜想的新证明（基于Euler系统）
   - 对 $\mathbb{Z}_p$-扩张的精细分析

3. **计算验证**：
   - 对秩 $\leq 3$ 的曲线，BSD II 的数值验证扩展到更大范围
   - 高精度计算 $L$ 函数的导数和特殊值

4. **统计与随机矩阵理论**：
   - 秩的分布与随机矩阵模型的对比研究
   - 发现新的统计规律（Katz-Sarnak哲学的深化）

**同余数问题**：

**定义 4.7**：正整数 $n$ 是**同余数**，如果存在有理边长的直角三角形面积为 $n$。

**等价表述**：$n$ 是同余数当且仅当 $E_n: y^2 = x^3 - n^2 x$ 有正秩。

**定理 4.8**（Tunnel, 1983）：BSD I 蕴含：若 $n$ 是无平方因子的奇数，则 $n$ 是同余数当且仅当
$$\#\{(x,y,z) \in \mathbb{Z}^3 : 2x^2 + y^2 + 8z^2 = n\} = 2 \#\{(x,y,z) \in \mathbb{Z}^3 : 2x^2 + y^2 + 32z^2 = n\}$$

（对偶数 $n$ 有类似公式）

**进展**：Smith (2017, 2024) 在同余数分布方面取得突破，证明在特定分布下100%的整数满足Tunnel判别法的预期行为。
