## 一、Dirichlet特征的理论体系

### 1.1 定义与基本性质

**定义 1.1**（Dirichlet特征）：设 $q \geq 1$ 为正整数。模 $q$ 的**Dirichlet特征**是函数 $\chi: \mathbb{Z} \to \mathbb{C}$ 满足：

1. **完全积性**：$\chi(mn) = \chi(m)\chi(n)$ 对所有 $m, n \in \mathbb{Z}$ 成立
2. **周期性**：$\chi(n+q) = \chi(n)$ 对所有 $n \in \mathbb{Z}$ 成立
3. **支集条件**：$\chi(n) = 0$ 当且仅当 $\gcd(n, q) > 1$

**例 1.2**（模4特征）：

| $n$ | 1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 | ... |
|-----|---|---|---|---|---|---|---|---|-----|
| $\chi_0(n)$ | 1 | 0 | 1 | 0 | 1 | 0 | 1 | 0 | ... |
| $\chi_1(n)$ | 1 | 0 | -1 | 0 | 1 | 0 | -1 | 0 | ... |

其中 $\chi_0$ 是主特征，$\chi_1$ 是本原特征，满足 $\chi_1(3) = -1$。

**定义 1.3**（本原特征）：特征 $\chi \pmod{q}$ 称为**本原的**，如果不存在 $q' < q$ 使得 $\chi$ 可诱导自模 $q'$ 的特征。

**定理 1.4**（特征个数）：模 $q$ 的Dirichlet特征恰有 $\varphi(q)$ 个，构成乘法群 $(\mathbb{Z}/q\mathbb{Z})^*$ 的特征群。

**证明思路**：由群表示论，有限交换群 $G$ 的特征群 $\widehat{G} = \text{Hom}(G, \mathbb{C}^*)$ 满足 $|\widehat{G}| = |G|$。取 $G = (\mathbb{Z}/q\mathbb{Z})^*$ 即可。

### 1.2 正交关系及其应用

**定理 1.5**（第一正交关系）：对模 $q$ 的特征，有
$$\frac{1}{\varphi(q)} \sum_{\chi \pmod{q}} \chi(n)\overline{\chi(a)} = \begin{cases} 1 & \text{若 } n \equiv a \pmod{q} \\ 0 & \text{否则} \end{cases}$$

其中求和遍历所有模 $q$ 的Dirichlet特征。

**定理 1.6**（第二正交关系）：对 $(a, q) = 1$，有
$$\frac{1}{\varphi(q)} \sum_{a=1, (a,q)=1}^{q} \chi_1(a)\overline{\chi_2(a)} = \begin{cases} 1 & \text{若 } \chi_1 = \chi_2 \\ 0 & \text{否则} \end{cases}$$

**应用**：正交关系是筛选算术级数的关键工具。对 $\gcd(a, q) = 1$，定义
$$\psi(x; q, a) = \sum_{\substack{n \leq x \\ n \equiv a \pmod{q}}} \Lambda(n)$$

利用第一正交关系：
$$\psi(x; q, a) = \frac{1}{\varphi(q)} \sum_{\chi \pmod{q}} \overline{\chi(a)} \psi(x, \chi)$$

其中 $\psi(x, \chi) = \sum_{n \leq x} \chi(n)\Lambda(n)$。

**历史注记**：Dirichlet（1805-1859）于1837年引入这些特征，首次证明了算术级数中存在无穷多素数。这是解析方法进入数论的标志性工作。

## 二、Dirichlet L函数的解析理论

### 2.1 解析延拓与函数方程

**定义 2.1**（Dirichlet L函数）：对Dirichlet特征 $\chi \pmod{q}$，定义
$$L(s, \chi) = \sum_{n=1}^{\infty} \frac{\chi(n)}{n^s}, \quad \text{Re}(s) > 1$$

**定理 2.2**（Euler乘积）：对 $\text{Re}(s) > 1$，
$$L(s, \chi) = \prod_p \left(1 - \frac{\chi(p)}{p^s}\right)^{-1}$$

**定理 2.3**（解析延拓）：

- 若 $\chi \neq \chi_0$（非主特征），$L(s, \chi)$ 可全纯延拓到整个复平面
- 若 $\chi = \chi_0$（主特征），$L(s, \chi_0) = \zeta(s)\prod_{p|q}(1-p^{-s})$，在 $s=1$ 有单极点

**证明框架**：利用Mellin-Barnes型积分。定义**高斯和**：
$$\tau(\chi) = \sum_{a=1}^{q} \chi(a)e^{2\pi i a/q}$$

对本原特征，$|\tau(\chi)| = \sqrt{q}$（高斯证明，1801）。

**定理 2.4**（函数方程）：设 $\chi$ 是模 $q$ 的本原特征，定义
$$\Lambda(s, \chi) = \left(\frac{\pi}{q}\right)^{-(s+\kappa)/2} \Gamma\left(\frac{s+\kappa}{2}\right) L(s, \chi)$$

其中 $\kappa = 0$（$\chi(-1)=1$）或 $\kappa = 1$（$\chi(-1)=-1$）。则
$$\Lambda(s, \chi) = \varepsilon(\chi) \Lambda(1-s, \overline{\chi})$$

其中 $\varepsilon(\chi) = \tau(\chi)/(i^{\kappa}\sqrt{q})$ 满足 $|\varepsilon(\chi)| = 1$。

### 2.2 非零区域与Siegel定理

**定理 2.5**（L函数的非零区域）：存在常数 $c > 0$，使得对所有 $\chi \pmod{q}$，$L(s, \chi)$ 在区域
$$\sigma > 1 - \frac{c}{\log(q(|t|+2))}$$

内无零点（可能有例外实零点）。

**定义 2.6**（Siegel零点）：对实特征 $\chi$，$L(s, \chi)$ 在区间 $(0, 1)$ 内的实零点称为**Siegel零点**。

**定理 2.7**（Siegel定理，1935）：对任意 $\varepsilon > 0$，存在 $C(\varepsilon) > 0$，使得若 $\beta$ 是 $L(s, \chi)$ 的实零点，则
$$\beta < 1 - \frac{C(\varepsilon)}{q^{\varepsilon}}$$

**注**：Siegel定理是**非有效**的，即不能给出 $C(\varepsilon)$ 的具体值。

## 三、算术级数中的素数定理

### 3.1 显式公式与误差项

**定理 3.1**（算术级数PNT）：设 $\gcd(a, q) = 1$，则
$$\pi(x; q, a) \sim \frac{1}{\varphi(q)} \cdot \frac{x}{\log x} \quad (x \to \infty)$$

等价地，$\psi(x; q, a) \sim \frac{x}{\varphi(q)}$。

**定理 3.2**（显式公式）：对 $x \geq 2$，$T \geq 2$，有
$$\psi(x, \chi) = -\sum_{|\gamma| \leq T} \frac{x^{\rho}}{\rho} + O\left(\frac{x \log^2(qxT)}{T}\right)$$

其中 $\rho = \beta + i\gamma$ 遍历 $L(s, \chi)$ 的非平凡零点。

**推论**：若L函数在 $\text{Re}(s) \geq \sigma_0$ 无零点，则
$$\psi(x; q, a) = \frac{x}{\varphi(q)} + O(x^{\sigma_0} \log^2 x)$$

### 3.2 Page-Siegel-Walfisz定理

**定理 3.3**（Page定理）：设 $q \leq (\log x)^A$（对任意固定 $A > 0$），则
$$\psi(x; q, a) = \frac{x}{\varphi(q)} + O(x \exp(-c\sqrt{\log x}))$$

**定理 3.4**（Siegel-Walfisz定理）：在Page定理条件下，误差项可改进为
$$\psi(x; q, a) = \frac{x}{\varphi(q)} + O_A(x \exp(-c_A\sqrt{\log x}))$$

**应用示例**：计算 $\pi(1000; 4, 1)$（不超过1000且模4余1的素数个数）。

实际值：5, 13, 17, 29, ..., 997，共78个。
渐近公式：$\frac{1000}{2\log 1000} \approx 72.4$。

## 四、Riemann假设：历史、进展与前沿

### 4.1 从Euler到Riemann

**历史脉络**：

- **Euler (1737)**：发现 $\zeta(s)$ 的Euler乘积，建立素数与 $\zeta$ 函数的联系
- **Gauss (1792)**：通过数值计算猜想 $\pi(x) \sim \text{Li}(x)$
- **Dirichlet (1837)**：引入L函数，证明算术级数PNT
- **Riemann (1859)**：发表划时代论文《论小于给定值的素数个数》，提出Riemann假设

**Riemann假设（RH，1859）**：
$$\zeta(s) = 0, \quad 0 < \text{Re}(s) < 1 \implies \text{Re}(s) = \frac{1}{2}$$

**广义Riemann假设（GRH）**：对所有Dirichlet L函数，非平凡零点均位于临界线 $\text{Re}(s) = \frac{1}{2}$。

### 4.2 2024-2025年的突破性进展

**历史验证**：

- **G.H. Hardy (1914)**：证明 $\zeta(s)$ 在临界线上有无穷多零点
- **Levinson (1974)**：证明至少 $1/3$ 的零点在临界线上
- **Conrey (1989)**：改进到 $2/5$
- **Bui-Conrey-Young (2011)**：$41\%$
- **Feng (2012)**：$41.07\%$

**计算验证**：

截至2024年，已数值验证前 $10^{13}$ 个零点均在临界线上（Gourdon, 2004; 后续工作持续推进）。

**2024-2025年重要进展**：

1. **高阶矩的渐进公式**：
   - 利用随机矩阵理论，证明了某些加权高阶矩的渐进行为与Montgomery-Odlyzko定律一致
   - 推进了对零点分布统计规律的理解

2. **临界线上的新估计**：
   - 改进的指数和估计技术应用于 $\zeta(1/2 + it)$
   - 新的subconvexity bound：$\zeta(1/2 + it) \ll_{\varepsilon} t^{13/84 + \varepsilon}$（基于Bourgain的方法）

3. **机器学习辅助研究**：
   - 使用神经网络探索零点相关函数的模式
   - 发现新的启发式联系，但尚未产生严格证明

**开放问题**：

- 证明 $100\%$ 的零点在临界线上
- 证明零点间距分布服从GUE统计
- 证明Lindelöf假设（RH的推论）

**定理 4.1**（RH的等价形式）：以下陈述等价：

1. RH成立
2. $\psi(x) = x + O(x^{1/2}\log^2 x)$
3. $\pi(x) = \text{Li}(x) + O(x^{1/2}\log x)$
4. $\sum_{n \leq x} \mu(n) = O(x^{1/2+\varepsilon})$（Mertens函数）
5. $\sum_{n \leq x} \Lambda(n) = x + O(x^{1/2}\log^2 x)$

其中 $\mu(n)$ 是Möbius函数。
