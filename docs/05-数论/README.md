# 05-数论

本目录系统梳理数论各分支，从初等数论、解析数论到代数数论与算术几何，涵盖椭圆曲线、模形式、p进数、L函数等高级主题，并配有算法实现与习题。

## 内容导航

- **01-初等数论** — 整除、同余、原根与二次剩余
- **02-解析数论** — 素数分布、Riemann ζ函数
- **03-代数数论** — 代数整数、理想类群
- **04-算术几何** — 椭圆曲线、Diophantine方程
- **07-深度扩展** — 高级专题与前沿进展
- **算法实现与计算实践 / 高级习题**
- **L函数 / p进数 / 模形式 / 椭圆曲线算术**
