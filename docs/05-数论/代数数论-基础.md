# 代数数论基础

代数数论（algebraic number theory）研究数域（有理数域的有限扩张）的算术性质，核心对象是代数整数环、理想类群与 $L$-函数。从高斯二次互反律到 Wiles 的费马大定理证明，代数数论始终是数论的主流范式。

## 1. 代数整数

### 1.1 定义与基本性质

**定义 1.1（代数整数）**. 复数 $\alpha \in \mathbb{C}$ 称为代数整数，若它是某个首一整系数多项式的根，即存在 $f(x) = x^n + a_{n-1}x^{n-1} + \cdots + a_0 \in \mathbb{Z}[x]$ 使得 $f(\alpha) = 0$。

**命题 1.2**. $\alpha$ 是代数整数当且仅当 $\mathbb{Z}[\alpha]$ 是有限生成 $\mathbb{Z}$-模。

*证明*. $(\Rightarrow)$ 设 $\alpha$ 满足首一多项式 $f$ 的次数为 $n$。则 $\alpha^n$ 可表为 $1, \alpha, \dots, \alpha^{n-1}$ 的 $\mathbb{Z}$-线性组合，归纳可得 $\mathbb{Z}[\alpha]$ 由这 $n$ 个元素生成。

$(\Leftarrow)$ 设 $\mathbb{Z}[\alpha]$ 由 $v_1, \dots, v_n$ 生成。乘法算子 $m_\alpha: x \mapsto \alpha x$ 是 $\mathbb{Z}[\alpha]$ 的 $\mathbb{Z}$-线性自同态。由 Cayley-Hamilton，$m_\alpha$ 满足某首一多项式 $f$，故 $f(\alpha) = f(m_\alpha)(1) = 0$。$\square$

### 1.2 数域的整数环

**定义 1.3（数域）**. 数域 $K$ 是 $\mathbb{Q}$ 的有限扩张，$[K:\mathbb{Q}] < \infty$。

**定义 1.4（整数环）**. $K$ 的代数整数全体记为 $\mathcal{O}_K$，是 $K$ 的子环。

**定理 1.5**. $\mathcal{O}_K$ 是秩为 $[K:\mathbb{Q}]$ 的自由 $\mathbb{Z}$-模。

## 2. Dedekind 整环

### 2.1 理想的分解

**定义 2.1（Dedekind 整环）**. 整环 $R$ 称为 Dedekind 整环，若：
1. $R$ Noether；
2. $R$ 中每个非零素理想极大；
3. $R$ 整闭（在其分式域中）。

**定理 2.2**. 数域的整数环 $\mathcal{O}_K$ 是 Dedekind 整环。

**定理 2.3（理想的唯一分解）**. Dedekind 整环中每个非零理想唯一分解为素理想的乘积。

*证明概要*. 证明每个非零理想可逆：对理想 $I$，定义 $I^{-1} = \{x \in K : xI \subseteq R\}$。验证 $II^{-1} = R$。由此理想的唯一分解类比整数的唯一分解。$\square$

### 2.2 理想类群

**定义 2.4（分式理想）**. $K$ 的分式理想是 $\mathcal{O}_K$-模 $I \subseteq K$，存在非零 $d \in \mathcal{O}_K$ 使 $dI \subseteq \mathcal{O}_K$。

**定义 2.5（理想类群）**. 分式理想在乘法下构成群，主分式理想构成子群。商群 $\operatorname{Cl}(K) = I(K)/P(K)$ 称为理想类群。

**定理 2.6（类数有限）**. 对任意数域 $K$，$|\operatorname{Cl}(K)| < \infty$。

## 3. Dirichlet 单位定理

**定理 3.1（Dirichlet 单位定理）**. 设 $K$ 有 $r_1$ 个实嵌入和 $r_2$ 对复嵌入（$r_1 + 2r_2 = [K:\mathbb{Q}]$）。则单位群 $\mathcal{O}_K^\times$ 是有限生成 Abel 群：

$$\mathcal{O}_K^\times \cong \mu(K) \times \mathbb{Z}^{r_1 + r_2 - 1},$$

其中 $\mu(K)$ 为 $K$ 中的单位根群（有限循环群）。

*证明概要*. 利用对数嵌入 $\lambda: \mathcal{O}_K^\times \to \mathbb{R}^{r_1 + r_2}$，$\lambda(u) = (\log|\sigma_1(u)|, \dots, \log|\sigma_{r_1}(u)|, 2\log|\sigma_{r_1+1}(u)|, \dots)$。像落在超平面 $\sum x_i = 0$ 中，且为离散子群，故是格。核为单位根。$\square$

## 4. 例子

### 4.1 例子：二次域

设 $K = \mathbb{Q}(\sqrt{d})$，$d$ 无平方因子。

若 $d \equiv 2, 3 \pmod{4}$，$\mathcal{O}_K = \mathbb{Z}[\sqrt{d}]$；若 $d \equiv 1 \pmod{4}$，$\mathcal{O}_K = \mathbb{Z}[\frac{1+\sqrt{d}}{2}]$。

$\mathbb{Q}(\sqrt{-5})$：$\mathcal{O}_K = \mathbb{Z}[\sqrt{-5}]$。理想 $(2, 1+\sqrt{-5})$ 非主理想，故类数 $> 1$（实际为 2）。这说明 $\mathbb{Z}[\sqrt{-5}]$ 不是 UFD：$6 = 2 \cdot 3 = (1+\sqrt{-5})(1-\sqrt{-5})$ 是两种不同的不可约元分解。

### 4.2 例子：分圆域

$K = \mathbb{Q}(\zeta_p)$，$\zeta_p = e^{2\pi i/p}$。$[K:\mathbb{Q}] = p-1$，$\mathcal{O}_K = \mathbb{Z}[\zeta_p]$。

**定理 4.1（Kummer）**. $p$ 为正则素数（不整除 $\mathbb{Q}(\zeta_p)$ 的类数）当且仅当 $p$ 不整除 Bernoulli 数 $B_2, B_4, \dots, B_{p-3}$ 的分子。

对正则素数 $p$，Fermat 大定理 $x^p + y^p = z^p$ 无正整数解（Kummer，1850）。
