# 代数数论基础

## 代数整数

### 定义

复数$\alpha$是代数整数若它是首一整数系数多项式的根。

### 例子

- 普通整数：$\alpha \in \mathbb{Z}$
- $\sqrt{2}$：根为$x^2 - 2 = 0$
- $\omega = e^{2\pi i/3}$：根为$x^2 + x + 1 = 0$

### 代数整数环

数域$K$的代数整数环$\mathcal{O}_K$是$K$中所有代数整数的集合。

### 理想类群

衡量$\mathcal{O}_K$与UFD差距的群。
