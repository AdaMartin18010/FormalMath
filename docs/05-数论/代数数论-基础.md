# 代数数论基础

代数数论（algebraic number theory）研究数域（有理数域的有限扩张）的算术性质，核心对象是代数整数环、理想类群与 $L$-函数。从高斯二次互反律到 Wiles 的费马大定理证明，代数数论始终是数论的主流范式。

## 1. 代数整数

### 1.1 定义与基本性质

**定义 1.1（代数整数）**. 复数 $\alpha \in \mathbb{C}$ 称为代数整数，若它是某个首一整系数多项式的根，即存在 $f(x) = x^n + a_{n-1}x^{n-1} + \cdots + a_0 \in \mathbb{Z}[x]$ 使得 $f(\alpha) = 0$。

**命题 1.2**. $\alpha$ 是代数整数当且仅当 $\mathbb{Z}[\alpha]$ 是有限生成 $\mathbb{Z}$-模。

*证明*. $(\Rightarrow)$ 设 $\alpha$ 满足首一多项式 $f$ 的次数为 $n$。则 $\alpha^n$ 可表为 $1, \alpha, \dots, \alpha^{n-1}$ 的 $\mathbb{Z}$-线性组合，归纳可得 $\mathbb{Z}[\alpha]$ 由这 $n$ 个元素生成。

$(\Leftarrow)$ 设 $\mathbb{Z}[\alpha]$ 由 $v_1, \dots, v_n$ 生成。乘法算子 $m_\alpha: x \mapsto \alpha x$ 是 $\mathbb{Z}[\alpha]$ 的 $\mathbb{Z}$-线性自同态。由 Cayley-Hamilton，$m_\alpha$ 满足某首一多项式 $f$，故 $f(\alpha) = f(m_\alpha)(1) = 0$。$\square$

**定理 1.3**. 代数整数全体构成 $\mathbb{C}$ 的子环。

*证明*. 设 $\alpha, \beta$ 为代数整数。$\mathbb{Z}[\alpha, \beta]$ 作为 $\mathbb{Z}$-模有限生成（因 $\mathbb{Z}[\alpha]$ 和 $\mathbb{Z}[\beta]$ 均有限生成）。$\alpha + \beta$ 和 $\alpha\beta$ 属于 $\mathbb{Z}[\alpha, \beta]$，故由命题 1.2，它们也是代数整数。$\square$

### 1.2 数域的整数环

**定义 1.4（数域）**. 有限扩张 $K/\mathbb{Q}$ 称为数域（number field），$[K:\mathbb{Q}] = n < \infty$。

**定义 1.5（代数整数环）**. $K$ 中代数整数全体记为 $\mathcal{O}_K$，称为 $K$ 的代数整数环（ring of integers）。

**定理 1.6**. $\mathcal{O}_K$ 是 Dedekind 整环：Noether 环、整闭、且每个非零素理想极大（即 Krull 维数为 1）。

*证明概要*. Noether 性：$\mathcal{O}_K$ 是秩 $n$ 的自由 $\mathbb{Z}$-模，故理想作为 $\mathbb{Z}$-子模有限生成。

整闭性：$K$ 中在 $\mathbb{Z}$ 上整的元素恰为 $\mathcal{O}_K$ 的元素。

维数 1：设 $\mathfrak{p}$ 为非零素理想。取非零 $a \in \mathfrak{p}$，$N_{K/\mathbb{Q}}(a) \in \mathfrak{p} \cap \mathbb{Z} = p\mathbb{Z}$。故 $p \in \mathfrak{p}$，$\mathcal{O}_K/\mathfrak{p}$ 为有限整环，从而是域。$\square$

## 2. 理想的唯一分解

### 2.1 理想的分解

**定理 2.1**. 在 Dedekind 整环 $\mathcal{O}_K$ 中，每个非零理想 $I$ 可唯一分解为素理想的乘积：

$$I = \mathfrak{p}_1^{e_1} \cdots \mathfrak{p}_g^{e_g}.$$

*注*. 元素层面的唯一分解一般不成立（即 $\mathcal{O}_K$ 未必是 UFD），但理想层面始终成立。

### 2.2 素理想的分裂

设 $L/K$ 为数域的有限扩张，$\mathfrak{p}$ 为 $\mathcal{O}_K$ 的素理想。素理想 $\mathfrak{p}\mathcal{O}_L$ 在 $\mathcal{O}_L$ 中分解为：

$$\mathfrak{p}\mathcal{O}_L = \mathfrak{P}_1^{e_1} \cdots \mathfrak{P}_g^{e_g}.$$

**定义 2.2**. $e_i$ 称为分歧指数（ramification index），$f_i = [\mathcal{O}_L/\mathfrak{P}_i : \mathcal{O}_K/\mathfrak{p}]$ 称为剩余次数（residue degree）。

**定理 2.3**. $$\sum_{i=1}^g e_i f_i = [L:K].$$

## 3. 理想类群

### 3.1 分式理想与类群

**定义 3.1（分式理想）**. $K$ 的分式理想是 $\mathcal{O}_K$-子模 $I \subseteq K$，存在非零 $d \in \mathcal{O}_K$ 使 $dI \subseteq \mathcal{O}_K$。

**定义 3.2（理想类群）**. 分式理想在乘法下构成群，主分式理想构成子群。理想类群定义为：

$$\operatorname{Cl}(K) = \{\text{分式理想}\} / \{\text{主分式理想}\}.$$

**定理 3.3（Minkowski）**. $\operatorname{Cl}(K)$ 是有限群。

*证明概要*. 利用 Minkowski 界：每个理想类包含整理想 $I$ 满足：

$$N(I) \leq \frac{n!}{n^n}\left(\frac{4}{\pi}\right)^{r_2} \sqrt{|d_K|},$$

其中 $n = [K:\mathbb{Q}]$，$r_2$ 为复嵌入对数，$d_K$ 为判别式。因范数有界的理想有限，类群有限。$\square$

### 3.2 类数

$|\operatorname{Cl}(K)|$ 称为 $K$ 的类数（class number），记 $h_K$。$h_K = 1$ 当且仅当 $\mathcal{O}_K$ 是主理想整环（PID），对 Dedekind 整环等价于 UFD。

## 4. 例子

### 4.1 例子：二次域

设 $K = \mathbb{Q}(\sqrt{d})$，$d$ 为无平方因子整数。

**定理 4.1**. $\mathcal{O}_K = \mathbb{Z}[\omega]$，其中：

$$\omega = \begin{cases} \sqrt{d} & d \equiv 2, 3 \pmod{4}, \\ \frac{1+\sqrt{d}}{2} & d \equiv 1 \pmod{4}. \end{cases}$$

对 $d = -1$（Gauss 整数），$\mathcal{O}_K = \mathbb{Z}[i]$，为 Euclidean 整环，$h_K = 1$。

对 $d = -5$，$\mathcal{O}_K = \mathbb{Z}[\sqrt{-5}]$。理想 $(2, 1+\sqrt{-5})$ 非主理想：

$$(2, 1+\sqrt{-5})^2 = (2),$$

故 $h_K > 1$（实际上 $h_K = 2$）。元素层面：$6 = 2 \cdot 3 = (1+\sqrt{-5})(1-\sqrt{-5})$ 给出两种不同的不可约分解。

### 4.2 例子：分圆域

设 $\zeta_n = e^{2\pi i/n}$，$K = \mathbb{Q}(\zeta_n)$。则 $[K:\mathbb{Q}] = \varphi(n)$，$\mathcal{O}_K = \mathbb{Z}[\zeta_n]$。

**定理 4.2（Kummer）**. 若 $p$ 为奇素数且 $p \nmid h_{\mathbb{Q}(\zeta_p)}$，则 Fermat 大定理对指数 $p$ 成立。
