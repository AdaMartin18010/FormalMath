## 使用指南

### 如何选择决策树

1. **明确当前需求**：是评估基础、选择资源、制定计划，还是解决问题？
2. **找到对应分类**：根据需求选择相应的决策树类别
3. **阅读决策树**：按照决策树的引导逐步分析
4. **执行建议**：根据决策结果采取相应行动

### 决策树使用流程

```
识别问题 → 选择决策树 → 跟随引导 → 获得建议 → 执行行动 → 反馈调整
```

### 组合使用建议

- **开始学习新领域**：前置知识评估 → 教材选择 → 长期学习路径
- **准备考试**：考试准备计划 → 练习资源选择 → 时间管理诊断
- **遇到困难**：困难诊断 → 学习资源选择 → 短期学习计划
- **规划职业**：研究准备计划 → 长期学习路径 → 动机问题诊断
