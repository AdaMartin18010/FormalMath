## 核心计算策略

### 曲线曲率

**平面曲线**：

```

弧长参数：κ = |dT/ds| = |d²r/ds²|

一般参数 r(t) = (x(t), y(t))：
κ = |x'y'' - x''y'| / (x'² + y'²)^(3/2)

显式 y = f(x)：
κ = |f''| / (1 + f'²)^(3/2)

```

**空间曲线（Frenet标架）**：

```

弧长参数 r(s)：
- T = dr/ds （单位切向量）
- N = dT/ds / |dT/ds| （主法向量）

- B = T × N （副法向量）

曲率：κ = |dT/ds| = |r' × r''| / |r'|³

挠率：τ = -N · dB/ds

Frenet公式：
dT/ds = κN
dN/ds = -κT + τB
dB/ds = -τN

```

### 曲面曲率

**第一基本形式**：

```

I = E du² + 2F dudv + G dv²
其中：
E = r_u · r_u
F = r_u · r_v
G = r_v · r_v

```

**第二基本形式**：

```

II = L du² + 2M dudv + N dv²
其中：
L = r_uu · n = (r_u × r_v) · r_uu / |r_u × r_v|

M = r_uv · n
N = r_vv · n

n = (r_u × r_v) / |r_u × r_v| （单位法向量）

```

**曲率计算**：

```

Gauss曲率：K = κ₁κ₂ = (LN - M²) / (EG - F²)

平均曲率：H = (κ₁ + κ₂)/2 = (EN + GL - 2FM) / 2(EG - F²)

主曲率：κ = H ± √(H² - K)

Weingarten映射 W = -dn 的特征值

```

**Gauss绝妙定理**：

```

K 仅依赖于第一基本形式（内蕴量）

计算：
K = -1/(2√(EG)) [∂_u(G_u/√(EG)) + ∂_v(E_v/√(EG))]
（当 F=0 时）

```

### Riemann曲率

**Christoffel符号**：

```

Γᵏᵢⱼ = (1/2) gᵏˡ (∂ᵢ gⱼₗ + ∂ⱼ gᵢₗ - ∂ₗ gᵢⱼ)

第二类：Γᵢⱼ,ₖ = gₖₗ Γˡᵢⱼ = (1/2)(∂ᵢ gⱼₖ + ∂ⱼ gᵢₖ - ∂ₖ gᵢⱼ)

```

**Riemann曲率张量**：

```

R(X,Y)Z = ∇ₓ∇ᵧZ - ∇ᵧ∇ₓZ - ∇_{[X,Y]}Z

分量：
Rᵏₗᵢⱼ = ∂ᵢ Γᵏⱼₗ - ∂ⱼ Γᵏᵢₗ + Γᵏᵢₘ Γᵐⱼₗ - Γᵏⱼₘ Γᵐᵢₗ

协变形式：
Rₖₗᵢⱼ = gₖₘ Rᵐₗᵢⱼ

```

**对称性**：

```

Rₖₗᵢⱼ = -Rₖₗⱼᵢ = -Rₗₖᵢⱼ = Rᵢⱼₖₗ
Rₖₗᵢⱼ + Rₖᵢⱼₗ + Rₖⱼₗᵢ = 0 (Bianchi第一恒等式)

独立分量数：n²(n²-1)/12

```

### 导出曲率

**Ricci曲率**：

```

Rᵢⱼ = Rᵏᵢₖⱼ = gᵏˡ Rₖᵢₗⱼ

收缩的Riemann张量

标量曲率：R = gⁱʲ Rᵢⱼ

```

**截面曲率**：

```

平面 Π = span{X,Y} 截面：
Sec(Π) = ⟨R(X,Y)Y, X⟩ / (|X|²|Y|² - ⟨X,Y⟩²)

二维情形：Sec = K (Gauss曲率)

常截面曲率：Sec = K (常数)
- K > 0：球面
- K = 0：欧氏空间
- K < 0：双曲空间

```

**Einstein流形**：

```

Rᵢⱼ = λ gᵢⱼ （λ 为常数）

等价：Ricci曲率与度量成比例

性质：
- 自动有常标量曲率
- 真空Einstein方程的解
- 重要例子：球面、双曲空间、Ricci平坦流形

```

### 常曲率空间

**常曲率判定**：

```

截面曲率常数：
R(X,Y,Z,W) = K(⟨X,Z⟩⟨Y,W⟩ - ⟨X,W⟩⟨Y,Z⟩)

等价于：
Rᵢⱼₖₗ = K(gᵢₖ gⱼₗ - gᵢₗ gⱼₖ)

```

**标准模型**：

```

K > 0：半径为 1/√K 的球面 Sⁿ
  度量：ds² = dr² + (sin²(√K r)/K) dΩ²

K = 0：欧氏空间 ℝⁿ
  度量：ds² = dr² + r² dΩ²

K < 0：双曲空间 Hⁿ
  度量：ds² = dr² + (sinh²(√(-K) r)/(-K)) dΩ²

```

## 计算流程模板

### 曲面曲率计算

```

给定参数曲面 r(u,v)：

步骤1：计算偏导数
  r_u, r_v, r_uu, r_uv, r_vv

步骤2：计算第一基本形式系数
  E = r_u·r_u, F = r_u·r_v, G = r_v·r_v

步骤3：计算单位法向量
  n = (r_u × r_v) / |r_u × r_v|

步骤4：计算第二基本形式系数
  L = r_uu·n, M = r_uv·n, N = r_vv·n

步骤5：计算曲率
  K = (LN - M²) / (EG - F²)
  H = (EN + GL - 2FM) / 2(EG - F²)
  κ = H ± √(H² - K)

```

### Riemann流形曲率计算

```

给定度量 gᵢⱼ：

步骤1：计算度量逆 gⁱʲ

步骤2：计算Christoffel符号
  Γᵏᵢⱼ = (1/2) gᵏˡ (∂ᵢ gⱼₗ + ∂ⱼ gᵢₗ - ∂ₗ gᵢⱼ)

步骤3：计算Riemann张量
  Rᵏₗᵢⱼ = ∂ᵢ Γᵏⱼₗ - ∂ⱼ Γᵏᵢₗ + Γᵏᵢₘ Γᵐⱼₗ - Γᵏⱼₘ Γᵐᵢₗ

步骤4：计算Ricci张量
  Rᵢⱼ = Rᵏᵢₖⱼ

步骤5：计算标量曲率
  R = gⁱʲ Rᵢⱼ

步骤6：分析曲率性质
  - Einstein条件？
  - 常曲率？
  - 截面曲率范围？

```

## 常见问题解决策略

| 问题 | 策略 |
|------|------|
| 计算曲线曲率 | Frenet公式或直接公式 |
| 计算曲面曲率 | 第一/二基本型方法 |
| 计算Riemann张量 | Christoffel → Riemann |
| 判断Einstein | 验证 Rᵢⱼ = λgᵢⱼ |
| 计算常曲率 | 截面曲率公式或标准模型比对 |
| 极小曲面 | H = 0 |
| 等距嵌入 | Nash嵌入定理保证存在性 |
| 测地线 | 解测地线方程 ∇_T T = 0 |
