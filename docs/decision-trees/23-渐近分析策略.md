## 核心分析策略

### 极限计算方法

**L'Hôpital法则**：

```

0/0 或 ∞/∞ 型：
lim f(x)/g(x) = lim f'(x)/g'(x)
（右侧极限存在或为∞）

高阶应用：
若仍为不定式，继续求导直到确定

```

**Taylor展开法**：

```

展开到足够高的阶数，保留主项

例：求 lim_{x→0} (sin x - x)/x³
sin x = x - x³/6 + O(x⁵)
(sin x - x)/x³ = -1/6 + O(x²) → -1/6

```

**等价无穷小替换**：

```

x→0 时的常用等价：
sin x ∼ x          tan x ∼ x
arcsin x ∼ x       arctan x ∼ x
1 - cos x ∼ x²/2   e^x - 1 ∼ x
ln(1+x) ∼ x        (1+x)^a - 1 ∼ ax

注意：加减法中不能直接用！

```

**增长率比较**（x→∞）：

```

对于任意 a>0, b>0：
(ln x)^b << x^a << e^x << x! << x^x

即：
lim (ln x)^b / x^a = 0
lim x^a / e^x = 0
lim e^x / x! = 0

```

### 渐近展开方法

**Taylor/Maclaurin展开**：

```

f(x) = f(0) + f'(0)x + f''(0)x²/2! + ... + f^(n)(0)xⁿ/n! + Rₙ(x)

Peano余项：Rₙ(x) = o(xⁿ)
Lagrange余项：Rₙ(x) = f^(n+1)(ξ)x^(n+1)/(n+1)!

```

**渐近级数（Poincaré展开）**：

```

f(x) ∼ a₀ + a₁/x + a₂/x² + ...  (x→∞)

含义：f(x) - Σ_{k=0}^n a_k/x^k = o(1/xⁿ)

注意：渐近级数可能不收敛，但仍有用！

```

**常见函数的渐近展开**：

```

x→0:
  e^x = 1 + x + x²/2! + x³/3! + ...
  ln(1+x) = x - x²/2 + x³/3 - ...
  (1+x)^a = 1 + ax + a(a-1)x²/2! + ...
  sin x = x - x³/3! + x⁵/5! - ...
  cos x = 1 - x²/2! + x⁴/4! - ...

x→∞:
  Γ(x) ∼ √(2π/x) (x/e)^x  (Stirling)
  erfc(x) ∼ e^{-x²}/(x√π)
  ln x! ∼ x ln x - x

```

**匹配渐近展开**：

```

用于边界层问题、奇异摄动

步骤：
1. 外展开（outer expansion）：远离边界
2. 内展开（inner expansion）：边界层内
3. 匹配：内外展开在中间区域一致
4. 合成展开：得到一致有效展开

变量缩放：ξ = x/ε （边界层坐标）

```

### Landau记号体系

**定义与关系**：

```

f = O(g): ∃C>0, |f| ≤ C|g| （大O）

f = o(g): f/g → 0 （小o）
f ∼ g: f/g → 1 （渐近等价）
f = Ω(g): g = O(f) （大Ω）
f = Θ(g): f = O(g) 且 g = O(f) （紧界）
f = ω(g): g = o(f) （小ω）

```

**运算规则**：

```

O(f) + O(g) = O(|f|+|g|)

O(f) · O(g) = O(fg)
O(f) + o(f) = O(f)
O(o(f)) = o(O(f)) = o(f)

c·O(f) = O(f)  (c≠0)
O(O(f)) = O(f)

```

**常用渐近阶**：

```

对数多项式 < 多项式 < 指数 < 阶乘 < 双指数

具体：
(log n)^k = o(n^ε)  对任意 k, ε>0
n^k = o(a^n)        对任意 k, a>1
a^n = o(n!)
n! = o(n^n)

```

### 特殊函数渐近

**Stirling公式**（Gamma函数）：

```

n! ∼ √(2πn) (n/e)^n

更精确：
n! = √(2πn) (n/e)^n (1 + 1/(12n) + 1/(288n²) - ...)

Gamma函数：
Γ(x) ∼ √(2π/x) (x/e)^x

```

**误差函数**：

```

erf(x) = (2/√π) ∫₀ˣ e^{-t²} dt

erfc(x) = 1 - erf(x) ∼ e^{-x²}/(x√π)  (x→∞)

```

**Bessel函数**（x→∞）：

```

J_ν(x) ∼ √(2/πx) cos(x - νπ/2 - π/4)
Y_ν(x) ∼ √(2/πx) sin(x - νπ/2 - π/4)

I_ν(x) ∼ e^x/√(2πx)
K_ν(x) ∼ √(π/2x) e^{-x}

```

**Airy函数**（x→∞）：

```

Ai(x) ∼ e^(-2x^(3/2)/3) / (2√π x^(1/4))
Bi(x) ∼ e^(2x^(3/2)/3) / (√π x^(1/4))

```

### 渐近分析方法

**主项提取法**：

```

找出表达式的主导项，忽略低阶项

例：分析 √(n²+n) - n
= n√(1+1/n) - n
= n(1 + 1/(2n) + O(1/n²)) - n
= 1/2 + O(1/n) → 1/2

```

**积分渐近（Laplace方法）**：

```

∫ₐᵇ e^{Mf(x)} g(x) dx  (M→∞)

若 f 在 c 取最大值：
∼ e^{Mf(c)} g(c) √(2π/(M|f''(c)|))

应用：大偏差理论、统计物理

```

**最速下降法**（驻相法）：

```

∫ e^{if(x)} g(x) dx

主要贡献来自驻点 f'(x)=0

原理：非驻点处快速振荡，相互抵消

```

## 渐近分析模板

### 函数渐近行为分析模板

```

分析 f(x) 当 x→x₀（或 x→∞）的渐近行为

1. 主导项识别
   - 提取主要增长/衰减因子
   - 确定主导阶

2. 渐近展开
   - 选择合适的展开点
   - 展开到所需精度
   - 确定余项阶

3. 验证
   - 数值验证（如有条件）
   - 一致性检验
   - 特殊情形验证

4. 应用
   - 极限计算
   - 算法复杂度分析
   - 物理模型近似

```

### 复杂度分析模板

```

算法复杂度渐近分析

1. 确定基本操作
2. 建立递推关系（如需要）
3. 求解渐近阶
   - 主定理（分治递推）
   - 展开法
   - 生成函数
4. 结果表示
   - 时间复杂度：T(n) = O(...)
   - 空间复杂度：S(n) = O(...)

```

## 常见错误与注意

1. **渐近等价不保持加减**：f∼g, h∼k ⇏ f-h∼g-k
2. **展开阶数不足**：需展开到能区分主项的阶数
3. **Landau记号误用**：O(f)-O(f) ≠ 0
4. **收敛≠渐近**：渐近级数可能发散
5. **一致性问题**：渐近展开可能在某些区域不一致有效

## 重要公式速查

```

Stirling: n! ∼ √(2πn)(n/e)^n

调和数: H_n = ln n + γ + 1/(2n) + O(1/n²)

素数计数: π(x) ∼ x/ln x

Euler-Maclaurin: Σf(k) ∼ ∫f + (f(a)+f(b))/2 + ...

```
