## 核心构造策略

### 尺规作图基础

**基本操作**：

```

给定两点，可构造：
1. 过两点的直线
2. 以一点为圆心过另一点的圆
3. 两直线的交点
4. 直线与圆的交点
5. 两圆的交点

可构造数：
从 {0,1} 出发，通过加减乘除和开平方得到的数
即：ℚ 的二次扩域塔中的数

```

**正多边形可构造性**（Gauss-Wantzel定理）：

```

正 n 边形可尺规构造 ⇔ 
n = 2ᵏ · p₁ · p₂ · ... · pₘ

其中 pᵢ 是不同的Fermat素数
Fermat素数：Fᵣ = 2^(2ʳ) + 1
已知：F₀=3, F₁=5, F₂=17, F₃=257, F₄=65537

例：
- n=3,4,5,6,8,10,12,15,16,17 可构造
- n=7,9,11,13,14 不可构造

```

**古典不可能问题**：

```

1. 三等分任意角（一般情形）
   - 需要解三次方程 x³ - 3x - 2cos(3θ) = 0
   - 一般不可约，需三次扩张

2. 倍立方（Delian问题）
   - 构造 ³√2
   - 需要三次扩张 ℚ(³√2)/ℚ

3. 化圆为方
   - 构造 √π
   - Lindemann证明 π 超越

结论：这三个问题用尺规不可解

```

### 射影几何构造

**射影平面模型**：

```

P² = (𝔸³\{0}) / ~
(x,y,z) ~ (λx,λy,λz), λ≠0

仿射坐标：
当 z≠0，取 (x/z, y/z, 1) = (X, Y, 1)
无穷远线：z=0

```

**对偶原理**：

```

P² 中：
- 点 ↔ 直线
- 两点确定一线 ↔ 两线确定一点
- 点在线上 ↔ 线过点

Desargues定理与其对偶等价

```

**交比**：

```

共线四点 A,B,C,D 的交比：
(A,B;C,D) = (CA/CB) / (DA/DB)
          = (C-A)(D-B) / (C-B)(D-A)

性质：
- 射影变换保持交比
- 调和分割：(A,B;C,D) = -1

```

**配极对应**：

```

对非退化二次曲线 C：
点 P → 极线 p（过 P 的切线的切点连线）
直线 l → 极点 L

性质：P 在 l 上 ⇔ L 在 p 上

```

### 双曲几何构造

**Poincaré圆盘模型**：

```

单位圆盘 D = {z ∈ ℂ : |z| < 1}
度量：ds² = 4|dz|² / (1-|z|²)²

测地线：
- 与边界正交的圆弧
- 过圆心的直径

等距变换：
Möbius变换：f(z) = e^iθ (z-a)/(1-āz), |a|<1

```

**Poincaré半平面模型**：

```

上半平面 H = {z ∈ ℂ : Im(z) > 0}
度量：ds² = |dz|² / (Im z)²

测地线：
- 垂直于实轴的半圆
- 垂直线

等距变换：
PSL(2,ℝ): z ↦ (az+b)/(cz+d), a,b,c,d∈ℝ, ad-bc=1

```

**双曲圆**：

```

欧氏圆完全在 D 中 ⇒ 双曲圆
但圆心不重合

双曲圆心：到边界"等距"的点（欧氏意义不同）

```

### 代数几何构造

**仿射簇**：

```

仿射空间 𝔸ⁿ = kⁿ （k 代数闭域）

代数集：V(I) = {x ∈ 𝔸ⁿ : f(x)=0, ∀f ∈ I}
其中 I ⊂ k[x₁,...,xₙ] 是理想

不可约代数集 = 代数簇

Hilbert零点定理：
I(V(I)) = √I （根理想）

```

**射影簇**：

```

射影空间 ℙⁿ = (𝔸ⁿ⁺¹\{0}) / ~

齐次坐标 [x₀:x₁:...:xₙ]

射影代数集：由齐次理想定义的零点集

仿射锥：射影簇 ↔ 仿射锥（原点的并）

```

**态射与有理映射**：

```

正则映射（态射）：
f: X → Y 局部由多项式给出

有理映射：
f: X ⇢ Y 由有理函数给出
可能有不定义点

双有理等价：
存在互逆的有理映射
（比同构更弱的等价）

```

### 圆反演

**定义与性质**：

```

关于圆 C（中心 O，半径 r）的反演：
P ↦ P' 使得 OP · OP' = r²，且 O,P,P' 共线

性质：
- 圆过 O ↦ 直线（不过 O）
- 直线过 O ↦ 自身
- 圆不过 O ↦ 圆（不过 O）
- 角度保持（保角）
- 正交圆不变

```

**应用**：

```

Apollonius问题（作圆与三给定圆相切）：
用反演将问题简化，再反演回去

反演将圆变为直线或圆，简化相切条件

```

### 复数方法

**欧氏几何的复表示**：

```

平面点 ↔ 复数
- 旋转：乘 e^iθ
- 平移：加复数
- 缩放：乘实数
- 反射：取共轭

直线/圆：
a z z̄ + b z + b̄ z̄ + c = 0
其中 a,c ∈ ℝ, b ∈ ℂ

```

**共形映射**：

```

解析函数 f: U → ℂ
保角（保持角度）

重要例子：
- Möbius变换
- 指数、对数
- Schwarz-Christoffel映射

```

## 构造流程模板

### 尺规作图判定

```

问题：判断 α 是否可尺规构造

步骤1：确定 α 的极小多项式
  找 f ∈ ℚ[x] 使 f(α) = 0，f 不可约

步骤2：分析域扩张次数
  [ℚ(α):ℚ] = deg f

步骤3：应用判定准则
  α 可构造 ⇔ [ℚ(α):ℚ] = 2ᵏ

步骤4：构造域塔（若可构造）
  ℚ = F₀ ⊂ F₁ ⊂ ... ⊂ Fₖ = ℚ(α)
  每步 [Fᵢ₊₁:Fᵢ] = 2

```

### 双曲几何构造

```

在Poincaré圆盘中构造：

步骤1：理解测地线
  - 与边界正交的圆弧
  - 过原点的直径

步骤2：构造测地线段
  给定两点 A,B：
  - 若 A,B,O 共线：用直径
  - 否则：找过 A,B 且与边界正交的圆

步骤3：构造垂线
  过 P 作 l 的垂线：
  - 找过 P 和 l 的极点的测地线

步骤4：构造距离
  双曲距离公式：
  d(z,w) = arctanh |(z-w)/(1-z̄w)|

```

## 常见问题解决策略

| 问题 | 策略 |
|------|------|
| 判断可构造性 | 域扩张次数分析 |
| 射影定理证明 | 对偶原理转化 |
| 交比计算 | 坐标公式或射影不变性 |
| 双曲构造 | Poincaré模型转换 |
| 代数簇性质 | 理想与坐标环对应 |
| 圆的反演应用 | 选择合适反演圆简化 |
| 复几何问题 | 复数表示与解析函数 |
| 正多边形 | Gauss-Wantzel判定 |
