# 长期学习路径

## 决策目标
制定系统的长期（3个月-2年）数学学习路径，实现从入门到精通的进阶。

## 年度学习计划模板

### 模板：本科数学第一年

| 学期 | 核心课程 | 辅助学习 | 目标 |
|-----|---------|---------|------|
| 秋季 | 数学分析I、线性代数 | 数学软件入门 | 建立分析思维 |
| 冬季 | 数学分析II | 证明训练 | 掌握ε-δ语言 |
| 春季 | 数学分析III、常微分方程 | 参加讨论班 | 多元分析基础 |
| 夏季 | 复习+预习 | 科研项目/阅读 | 巩固+拓展 |

### 模板：研究生第一年

| 阶段 | 课程学习 | 研究准备 | 资格考试 |
|-----|---------|---------|---------|
| 秋季 | 3-4门核心课 | 找导师、定方向 | 了解要求 |
| 冬季 | 2-3门课+阅读 | 读论文、提问题 | 开始准备 |
| 春季 | 2门课+研讨 | 初步研究 | 部分考试 |
| 夏季 | 专注研究 | 写报告/论文 | 完成考试 |

## 里程碑设置

### 基础阶段里程碑（每3个月）
- [ ] 完成一本核心教材的学习
- [ ] 独立完成课后80%的习题
- [ ] 能讲授核心概念给他人
- [ ] 通过阶段性自测

### 进阶阶段里程碑（每6个月）
- [ ] 完成一个方向的系统学习
- [ ] 读懂本领域的经典论文
- [ ] 完成一个小研究项目
- [ ] 参加学术报告并提问

### 专家阶段里程碑（每年）
- [ ] 发表研究论文
- [ ] 在学术会议做报告
- [ ] 指导低年级学生
- [ ] 拓展新的研究方向

## 学习资源规划

### 时间分配建议

```

核心教材学习：     40%
习题练习：         25%
扩展阅读：         15%
讨论交流：         10%
复习总结：         10%

```

### 资源获取策略

| 类型 | 获取方式 | 成本 |
|-----|---------|------|
| 教材 | 图书馆/二手/电子版 | 低 |
| 论文 | arXiv/学校数据库 | 免费 |
| 课程 | MOOC/校内旁听 | 低 |
| 辅导 | 讨论班/学长指导 | 免费 |
| 会议 | 在线/学生资助 | 中 |

## 常见陷阱与规避

| 陷阱 | 症状 | 解决方案 |
|-----|------|---------|
| 贪多嚼不烂 | 同时学多门课，都没学好 | 一次聚焦1-2个核心主题 |
| 只看不练 | 看懂了但做题不会 | 强制自己做题时间 |
| 完美主义 | 一个点卡太久 | 标记后前进，回头再看 |
| 孤独学习 | 没有反馈 | 找学习伙伴/导师 |
| 缺乏复习 | 学完就忘 | 定期回顾机制 |
| 目标模糊 | 不知道学到什么程度 | 设定明确里程碑 |
