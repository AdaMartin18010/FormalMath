## 核心求解策略

### 理想理论

**理想判别法**：
```
I ⊆ R 是理想当且仅当：
1. (I,+) 是 (R,+) 的子群
2. ∀r ∈ R, ∀i ∈ I, ri ∈ I 且 ir ∈ I

左理想：只要求 ri ∈ I
右理想：只要求 ir ∈ I
双边理想：同时要求
```

**生成理想**：
```
由 S ⊆ R 生成的理想：
<S> = {Σrᵢsᵢ | rᵢ ∈ R, sᵢ ∈ S, 有限和}

主理想（由一个元素生成）：
<a> = {ra | r ∈ R}
```

**素理想与极大理想**：
```
素理想 P：
  ab ∈ P ⇒ a ∈ P 或 b ∈ P
  ⇔ R/P 是整环

极大理想 M：
  M ⊂ I ⊆ R ⇒ I = M 或 I = R
  ⇔ R/M 是域

关系：极大理想 ⇒ 素理想
（域是整环，逆不成立）
```

### 环同态

**基本性质**：
```
φ: R → S 是环同态：
- φ(a+b) = φ(a) + φ(b)
- φ(ab) = φ(a)φ(b)
- φ(1_R) = 1_S （保单位元）

ker φ = {r ∈ R | φ(r) = 0} 是 R 的理想
im φ 是 S 的子环
```

**同构定理**：
```
第一同构定理：
R/ker φ ≅ im φ

对应定理：
{包含 ker φ 的理想} ↔ {im φ 的理想}
```

### 整环层次

**关系链**：
```
域 ⊃ Euclid整环(ED) ⊃ 主理想整环(PID) ⊃ 唯一分解整环(UFD) ⊃ 整环

性质传递：
- ED：有带余除法
- PID：每个理想是主理想
- UFD：每个元唯一分解为素元
```

**PID的性质**：
```
1. 每个理想由单个元素生成
2. 任意两元素有最大公因子
3. ⟨a,b⟩ = ⟨gcd(a,b)⟩
4. 升链条件（ACC）
5. 素理想 = 极大理想（除零理想）
```

**UFD的判定**：
```
等价条件：
1. 每个非零非单位元可唯一分解
2. 每个不可约元是素元
3. 满足ACC，且任意两元有最大公因子
```

### 因式分解

**Eisenstein判别法**：
```
f(x) = aₙxⁿ + ... + a₀ ∈ ℤ[x]

若存在素数 p 满足：
1. p ∤ aₙ
2. p | aᵢ (i=0,...,n-1)
3. p² ∤ a₀

则 f 在 ℚ 上不可约（在 ℤ 上不可约）
```

**模p约化**：
```
f ∈ ℤ[x]，若 f mod p 在 𝔽ₚ[x] 中不可约
则 f 在 ℚ 上不可约

注意：逆不成立！
```

**有理根定理**：
```
f(x) = aₙxⁿ + ... + a₀ ∈ ℤ[x]

若有理根 r/s（既约），则：
r | a₀ 且 s | aₙ

应用：检验一次因子
```

### 多项式环

**多变量多项式**：
```
R[x₁,...,xₙ] 的性质：
- R UFD ⇒ R[x] UFD （Gauss引理）
- Hilbert基定理：R Noetherian ⇒ R[x] Noetherian
- 不可约性判定更复杂
```

**形式导数与重根**：
```
f ∈ R[x] 有重根 ⇔ gcd(f, f') ≠ 1

应用：
- 判别多项式是否有重根
- 计算根的重数
```

### 环扩张

**整扩张**：
```
α ∈ S 在 R 上整：
存在首一多项式 f ∈ R[x]，使得 f(α) = 0

整闭包：所有整元素构成的子环
```

**代数扩张**：
```
S = R[α]，α 在 R 上代数：
S ≅ R[x]/(f(x))

其中 f 是 α 的极小多项式
```

**局部化**：
```
S 是 R 的乘法子集，局部化 S⁻¹R：
- 元素：r/s 形式（r∈R, s∈S）
- 将 S 中元素变为单位

应用：研究环在素理想附近的性质
```

## 证明模板

### 证明理想是素理想

```
证明 P 是素理想：

方法1（定义）：
设 ab ∈ P，证 a ∈ P 或 b ∈ P

方法2（商环）：
证 R/P 是整环（无零因子）

方法3（逆否）：
若 a ∉ P 且 b ∉ P，证 ab ∉ P
```

### 证明多项式不可约

```
证明 f ∈ ℤ[x] 在 ℚ 上不可约：

方法1：Eisenstein判别
找素数 p 满足Eisenstein条件

方法2：模p约化
找素数 p 使 f mod p 不可约

方法3：次数分析
- 次数≤3：只需检验一次因子
- 高次：分析可能分解的形式

方法4：有理根检验（仅一次因子）
用有理根定理检验
```

## 常见问题解决策略

| 问题 | 策略 |
|------|------|
| 找所有理想 | 从生成元入手，分析主理想 |
| 判定UFD | 检验分解唯一性或证明是PID |
| 因式分解 | Eisenstein、模p、公式法 |
| 证明域结构 | 找极大理想，证商环是域 |
| 计算商环 | 利用同构定理简化 |
| 扩张结构 | 找极小多项式，构造同构 |
