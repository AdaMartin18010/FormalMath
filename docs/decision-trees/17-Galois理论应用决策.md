## Galois理论核心

### Galois群定义

```

E/F是Galois扩张（正规且可分）
Gal(E/F) = Aut(E/F) = {σ∈Aut(E) : σ|_F = id_F}

```

### Galois基本定理

**对应关系**：

```

{中间域 K : F⊆K⊆E} ↔ {Gal(E/F)的子群 H}
K ↔ Gal(E/K)
E^H ↔ H

```

**性质**：
- [E:K] = |Gal(E/K)|

- H◁Gal(E/F) ⇔ K/F是正规扩张
- 若H◁G，则Gal(K/F) ≅ G/H

## 应用场景详解

### 1. 方程可解性判定

**根式可解定义**：

```

f(x)=0根式可解 ⇔ 存在根塔
F = F₀ ⊆ F₁ ⊆ ... ⊆ F_m = E
其中F_{i+1} = F_i(ⁿ√a_i)

```

**Galois判据**：

```

f根式可解 ⇔ Gal(f)是可解群

```

**可解群链**：

```

G = G₀ ⊵ G₁ ⊵ ... ⊵ G_n = {e}
其中G_i/G_{i+1}是Abel群

```

### 2. 具体方程分析

**二次方程**：

```

x² + bx + c = 0
Gal ≅ C₂（可解）
求根公式存在

```

**三次方程**：

```

x³ + px + q = 0
Gal ⊆ S₃
判别式D = -4p³ - 27q²

D是完全平方 ⇔ Gal ≅ A₃ ≅ C₃
否则 Gal ≅ S₃

两者都可解

```

**四次方程**：

```

Gal ⊆ S₄
S₄可解（可解列：S₄ ⊵ A₄ ⊵ V₄ ⊵ {e}）
总有根式解

```

**五次及以上**：

```

一般五次多项式：Gal ≅ S₅
S₅不可解（A₅是单群）

Abel-Ruffini定理：
一般五次及以上方程无根式解

```

### 3. Galois群计算

**分圆扩张**：

```

E = ℚ(ζ_n), ζ_n = e^(2πi/n)
Gal(E/ℚ) ≅ (ℤ/nℤ)*

```

**二次扩张**：

```

E = ℚ(√d)
Gal(E/ℚ) ≅ C₂

```

**双二次扩张**：

```

E = ℚ(√a, √b)
Gal(E/ℚ) ≅ C₂ × C₂ 或 C₂

```

**一般n次多项式**：

```

Gal ⊆ S_n
判别式为完全平方 ⇔ Gal ⊆ A_n

```

### 4. 尺规作图问题

**可作图性判据**：

```

α可作图 ⇔ [ℚ(α):ℚ] = 2^k

```

**经典问题**：

| 问题 | 数学表述 | 可解性 | 原因 |
|------|---------|-------|------|
| 三等分角 | cos(θ/3) | 一般不可 | 3次扩张 |
| 倍立方 | ³√2 | 不可 | 3次扩张 |
| 化圆为方 | √π | 不可 | π超越 |
| 正n边形 | ζ_n | 有时可 | 取决于n |

**正n边形可作图**：

```

⇔ φ(n) = 2^k
⇔ n = 2^m·p₁·...·p_r
其中p_i是不同的Fermat素数

```

### 5. 域扩张结构

**单扩张定理**：

```

E/F有限可分 ⇒ E = F(α)（单扩张）

```

**正规扩张**：

```

E/F正规 ⇔ E是某多项式的分裂域
⇔ ∀σ: E→F̄, σ(E) = E

```

**可分扩张**：

```

E/F可分 ⇔ ∀α∈E, irr(α,F)无重根
特征0时自动可分

```

## 使用Galois理论的步骤

1. **确定扩张类型**：Galois？可分？正规？
2. **计算扩张次数**：[E:F]
3. **分析Galois群**：结构、生成元
4. **应用Galois对应**：中间域↔子群
5. **判断可解性**：群的可解性

## 可解性判定流程

```

多项式f
    ↓
计算判别式
    ↓
确定Gal(f) ⊆ S_n
    ↓
Gal(f)可解? 
    ├─是→根式可解
    └─否→无根式解

```
