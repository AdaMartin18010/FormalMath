## 核心计算策略

### 古典概型

**计数方法**：
```
基本公式：P(A) = |A| / |Ω|

排列：
- 全排列：n!
- 选排列：P(n,k) = n!/(n-k)!

组合：
- 组合数：C(n,k) = n!/(k!(n-k)!)
- 可重复组合：C(n+k-1,k)

常用技巧：
- 隔板法（分配问题）
- 圆排列：(n-1)!
- 错位排列：!n = n! Σ(-1)^k/k!
```

**容斥原理**：
```
|A₁ ∪ A₂ ∪ ... ∪ Aₙ| = 
Σ|Aᵢ| - Σ|Aᵢ ∩ Aⱼ| + Σ|Aᵢ ∩ Aⱼ ∩ Aₖ| - ... + (-1)^(n+1)|A₁ ∩ ... ∩ Aₙ|

概率形式：
P(∪Aᵢ) = ΣP(Aᵢ) - ΣP(AᵢAⱼ) + ...
```

### 条件概率

**基本公式**：
```
P(A|B) = P(AB) / P(B), P(B) > 0

乘法公式：
P(A₁A₂...Aₙ) = P(A₁)P(A₂|A₁)P(A₃|A₁A₂)...P(Aₙ|A₁...Aₙ₋₁)
```

**全概率公式**：
```
{Aᵢ} 是样本空间的分划：
P(B) = Σ P(B|Aᵢ) P(Aᵢ)

应用：分情况讨论，各情况互斥且穷尽
```

**Bayes公式**：
```
P(Aᵢ|B) = P(B|Aᵢ) P(Aᵢ) / P(B)
        = P(B|Aᵢ) P(Aᵢ) / Σ P(B|Aⱼ) P(Aⱼ)

先验概率 → 后验概率
```

### 随机变量分布

**离散分布**：
```
Bernoulli(p)：P(X=1)=p, P(X=0)=1-p
  E=p, Var=p(1-p)

二项 B(n,p)：P(X=k)=C(n,k)p^k(1-p)^(n-k)
  E=np, Var=np(1-p)

Poisson(λ)：P(X=k)=λ^k e^(-λ)/k!
  E=Var=λ

几何 G(p)：P(X=k)=(1-p)^(k-1)p
  E=1/p, Var=(1-p)/p²

负二项：
  P(X=k)=C(k+r-1,k)(1-p)^k p^r
```

**连续分布**：
```
均匀 U(a,b)：f(x)=1/(b-a), x∈[a,b]
  E=(a+b)/2, Var=(b-a)²/12

指数 Exp(λ)：f(x)=λe^(-λx), x>0
  E=1/λ, Var=1/λ²
  无记忆性：P(X>s+t|X>s)=P(X>t)

正态 N(μ,σ²)：f(x)=e^(-(x-μ)²/(2σ²))/√(2πσ²)
  E=μ, Var=σ²
  标准正态：Z=(X-μ)/σ ~ N(0,1)

Gamma(α,λ)：
  f(x)=λ^α x^(α-1)e^(-λx)/Γ(α)
  E=α/λ, Var=α/λ²

Beta(α,β)：
  f(x)=x^(α-1)(1-x)^(β-1)/B(α,β), x∈[0,1]
```

### 数字特征

**期望计算**：
```
定义：
E[X] = Σ xᵢ pᵢ （离散）
E[X] = ∫ x f(x) dx （连续）

性质：
- 线性：E[aX+bY] = aE[X] + bE[Y]
- 单调性：X≤Y ⇒ E[X]≤E[Y]
- 独立时：E[XY]=E[X]E[Y]

条件期望：
E[X] = E[E[X|Y]] （全期望公式）
```

**方差计算**：
```
定义：Var(X) = E[(X-EX)²]
计算：Var(X) = E[X²] - (E[X])²

性质：
- Var(aX+b) = a² Var(X)
- 独立时：Var(X+Y) = Var(X) + Var(Y)
- 一般：Var(X+Y) = Var(X) + Var(Y) + 2Cov(X,Y)

协方差：
Cov(X,Y) = E[(X-EX)(Y-EY)] = E[XY] - EX·EY
```

**生成函数**：
```
矩母函数：M(t) = E[e^(tX)]
  M^(k)(0) = E[X^k]

特征函数：φ(t) = E[e^(itX)]
  总是存在，唯一确定分布

概率母函数（离散）：G(z) = E[z^X]
  G^(k)(1) = E[X(X-1)...(X-k+1)]
```

### 极限定理

**大数定律**：
```
弱大数定律（WLLN）：
Xᵢ i.i.d., E[Xᵢ]=μ
则 (X₁+...+Xₙ)/n →ᵖ μ

强大数定律（SLLN）：
同上条件下，(X₁+...+Xₙ)/n → a.s. μ

应用：频率→概率，蒙特卡洛方法
```

**中心极限定理**：
```
经典CLT：
Xᵢ i.i.d., E[Xᵢ]=μ, Var(Xᵢ)=σ²
则 (ΣXᵢ - nμ)/(σ√n) →ᵈ N(0,1)

等价：
√n(X̄ₙ - μ)/σ →ᵈ N(0,1)

应用：
- 大样本统计推断
- 二项分布正态近似
- 误差分析
```

**De Moivre-Laplace**：
```
二项分布的正态近似：
X ~ B(n,p)，n大时
(X-np)/√(np(1-p)) ≈ N(0,1)

连续性修正：
P(X=k) ≈ P(k-0.5 < Y < k+0.5)
其中 Y ~ N(np, np(1-p))
```

### 高级技巧

**指示随机变量**：
```
I_A = 1 若 A 发生，否则 0

E[I_A] = P(A)

应用：
- 计数问题转化为求和
- 线性期望：E[ΣXᵢ] = ΣE[Xᵢ]

例：记录数期望
X = 记录发生次数
X = Σ Iᵢ，其中 Iᵢ 表示第 i 个是记录
E[X] = Hₙ = 1 + 1/2 + ... + 1/n
```

**对称性利用**：
```
对称分布：
- 利用对称性简化计算
- 配对技巧

随机排列：
- 对称位置同分布
- 期望相等
```

**递推方法**：
```
建立递推关系：
Eₙ = f(Eₙ₋₁, Eₙ₋₂, ...)

例：赌徒破产问题
Eₖ = pEₖ₊₁ + (1-p)Eₖ₋₁
解差分方程
```

## 计算流程模板

### 复杂概率计算

```
给定复杂随机事件：

步骤1：定义随机变量
- 明确问题的随机变量
- 确定取值范围

步骤2：确定分布类型
- 离散/连续
- 识别标准分布
- 或推导分布

步骤3：选择计算工具
- 古典：计数
- 条件：Bayes/全概率
- 渐近：极限定理

步骤4：计算数字特征
- 期望：定义或性质
- 方差：Var(X)=E[X²]-(EX)²
- 协方差（如需要）

步骤5：验证
- 概率之和为1
- 期望合理性
- 特殊情形检验
```

## 常见问题解决策略

| 问题 | 策略 |
|------|------|
| 古典概率 | 计数原理+容斥 |
| 条件概率 | 定义或Bayes公式 |
| 分布识别 | 匹配标准分布特征 |
| 期望计算 | 线性性质或条件期望 |
| 方差计算 | 定义或E[X²]-(EX)² |
| 大样本近似 | 中心极限定理 |
| 复杂事件 | 指示变量分解 |
| 递推问题 | 建立递推关系 |
