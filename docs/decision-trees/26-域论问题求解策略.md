## 核心求解策略

### 域扩张基础

**扩张次数与塔律**：

```

[E:F] = dim_F E （E作为F上向量空间的维数）

塔律：
F ⊆ E ⊆ K ⇒ [K:F] = [K:E] · [E:F]

应用：逐步计算扩张次数

```

**单代数扩张**：

```

α 在 F 上代数，f = irred_F(α)：
F(α) ≅ F[x]/(f)

性质：
- [F(α):F] = deg f
- {1, α, α², ..., αⁿ⁻¹} 是 F-基
- 每个元可唯一表示为 F 上次数 < deg f 的多项式

```

**单超越扩张**：

```

α 在 F 上超越：
F(α) ≅ F(x) （有理函数域）

性质：
- [F(α):F] = ∞
- F[α] ≅ F[x] （多项式环）

```

### 极小多项式

**求解方法**：

```

找 α 在 F 上的极小多项式：

1. 猜测或构造以 α 为根的多项式 f ∈ F[x]
2. 证 f 在 F 上不可约
3. 证 f 首一

不可约性判定：
- Eisenstein判别法
- 模p约化
- 次数分析

```

**共轭元**：

```

α 的 F-共轭元：
irred_F(α) 在代数闭包中的其他根

性质：
- F-同构 F(α) → F(β) 对应共轭元
- 共轭元具有相同的极小多项式

```

### Galois理论

**Galois群定义**：

```

E/F 是Galois扩张：
Gal(E/F) = Aut(E/F) = {σ: E→E | σ|_F = id}

性质：
- |Gal(E/F)| = [E:F]

- Gal(E/F) 作用于 f 的根

```

**Galois基本定理**：

```

E/F Galois，G = Gal(E/F)：

{E/F 的中间域} ↔ {G 的子群}

K ↦ Gal(E/K)
E^H := {x∈E | σ(x)=x, ∀σ∈H} ↤ H

性质对应：
- 包含反序
- [E:K] = |Gal(E/K)|

- [K:F] = [G:Gal(E/K)]
- H ◁ G ⇔ E^H/F Galois
- H ◁ G ⇒ Gal(E^H/F) ≅ G/H

```

**Galois群计算**：

```

1. 确定扩张次数 [E:F]
2. 找所有 F-自同构
3. 分析自同构在生成元上的作用
4. 确定群结构

技巧：
- 自同构由生成元的像唯一确定
- 像必须是共轭元

```

### 有限域

**基本定理**：

```

1. 存在性：对每个素数幂 q = pⁿ，存在 q 元域 𝔽_q
2. 唯一性：q 元域在同构意义下唯一
3. 结构：𝔽_{pⁿ} 是 𝔽_p 的 n 次扩张

```

**乘法群**：

```

𝔽_q^× 是 q-1 阶循环群

本原元：𝔽_q^× 的生成元

性质：
- 存在 φ(q-1) 个本原元
- 𝔽_q = 𝔽_p(α)，α 是本原元

```

**Frobenius自同构**：

```

σ: x ↦ x^p 是 𝔽_{pⁿ}/𝔽_p 的生成元

性质：
- σⁿ = id
- Gal(𝔽_{pⁿ}/𝔽_p) = <σ> ≅ ℤ/nℤ
- 是循环群

```

**有限域构造**：

```

𝔽_{pⁿ} ≅ 𝔽_p[x]/(f(x))

其中 f 是 𝔽_p 上 n 次不可约多项式

找不可约多项式的方法：
- 试错法
- 本原多项式表

```

### 可分扩张

**重根判别**：

```

f 有重根 ⇔ gcd(f, f') ≠ 1

应用：
- 检验多项式是否可分
- 确定可分扩张

```

**完全域**：

```

F 是完全域：每个代数扩张都可分

特征0域：都是完全域
特征p域：F 完全 ⇔ F = F^p

有限域：都是完全域

```

**可分扩张性质**：

```

- 可分元的和、积、商（分母≠0）可分
- 可分扩张的复合可分
- 代数扩张中可分元构成子域
- 本原元定理：有限可分扩张是单扩张

```

### 正规扩张与分裂域

**分裂域**：

```

f ∈ F[x] 的分裂域：
包含 f 所有根的最小域扩张

性质：
- 在同构意义下唯一
- 次数 ≤ (deg f)!
- f 在分裂域上分解为一次因子

```

**正规扩张**：

```

E/F 正规：
每个不可约 f ∈ F[x] 在 E 中有一根
则 f 在 E 上完全分裂

等价：E 是某族多项式的分裂域

```

**Galois扩张**：

```

E/F Galois ⇔ E/F 正规且可分

性质：
- |Gal(E/F)| = [E:F]

- Galois对应成立

```

### 方程可解性

**根式可解**：

```

f ∈ F[x] 根式可解：
存在根式塔 F = F₀ ⊆ F₁ ⊆ ... ⊆ Fₙ = E
使得 f 在 E 上分裂

根式塔：F_{i+1} = Fᵢ(ⁿ√aᵢ) 对某 aᵢ ∈ Fᵢ

```

**Galois可解性准则**：

```

f 根式可解 ⇔ Gal(f) 是可解群

可解群：存在正规列
G = G₀ ▷ G₁ ▷ ... ▷ Gₙ = {e}
使得 Gᵢ/G_{i+1} 交换

```

**Abel-Ruffini定理**：

```

一般 n 次方程（n ≥ 5）在 ℚ 上不是根式可解的

原因：Sₙ (n≥5) 不是可解群

```

## 证明模板

### 证明扩张是Galois扩张

```

证明 E/F 是Galois扩张：

方法1：证正规+可分
- 证 E 是某多项式的分裂域（正规）
- 证 F 完全或 E/F 可分（可分）

方法2：证 |Aut(E/F)| = [E:F]

方法3：证 E^Aut(E/F) = F

```

### 计算Galois群

```

计算 Gal(E/F)：

步骤1：确定 [E:F]
步骤2：找所有 F-嵌入 E → F̄
  - 确定生成元的共轭元
  - 构造自同构
步骤3：分析群结构
  - 元素的阶
  - 群的关系
  - 确定群类型（循环、对称等）

```

## 常见问题解决策略

| 问题 | 策略 |
|------|------|
| 找极小多项式 | 猜测+验证不可约性 |
| 计算扩张次数 | 塔律逐步计算 |
| 确定Galois群 | 分析自同构在根上的作用 |
| 证明正规性 | 验证是某多项式的分裂域 |
| 证明可分性 | 特征0自动可分；特征p用导数检验 |
| 构造有限域 | 找不可约多项式取商 |
| 证明不可解 | 证Galois群含非交换单群 |
