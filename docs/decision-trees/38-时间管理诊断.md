## 时间审计方法

### 步骤1：记录时间
连续记录3-7天的时间使用情况：

| 时间 | 活动 | 类别 | 效率(1-5) |
|-----|------|------|----------|
| 8:00-8:30 | 刷手机 | 娱乐 | 1 |
| 8:30-10:00 | 数学分析 | 学习 | 4 |
| ... | ... | ... | ... |

### 步骤2：分类统计

**类别**：
- 深度学习（高价值）
- 浅层学习（低价值）
- 必要活动（吃饭、通勤）
- 娱乐休闲
- 无效时间

### 步骤3：分析优化

**问题识别**：
- 时间花在哪里最多？
- 高效时间段利用如何？
- 有多少时间被浪费？

**优化策略**：
- 增加深度学习时间
- 减少无效时间
- 优化时间分配

## 效率提升工具

### 番茄工作法
```
25分钟专注
  ↓
5分钟休息
  ↓
重复4次
  ↓
15-30分钟长休息
```

**适用场景**：
- 适合：需要深度思考的任务
- 不适合：已经进入心流状态（不要打断）

### 时间块方法
将一天分为几个大块：
```
8:00-10:00 数学分析（核心块1）
10:00-10:30 休息
10:30-12:00 线性代数（核心块2）
...
```

### 艾森豪威尔矩阵
```
              紧急          不紧急
         ┌──────────┬──────────┐
  重要   │ 立即做   │ 计划做   │
         │ (考试)   │ (深度学习)│
         ├──────────┼──────────┤
不重要   │ 委托他人 │ 尽量不做 │
         │ (部分作业)│ (刷手机) │
         └──────────┴──────────┘
```

## 精力管理

### 精力曲线识别
记录一周每天不同时段的精力水平，找到自己的：
- **黄金时间**：效率最高时段（做最难任务）
- **白银时间**：效率中等（做常规任务）
- **低谷时间**：效率低下（做机械任务或休息）

### 精力恢复
- **微休息**：每25-50分钟休息5-10分钟
- **午间休息**：20-30分钟小憩
- **晚间休息**：保证7-8小时睡眠
- **运动**：每天适度运动提升精力

## 常见时间陷阱

| 陷阱 | 表现 | 对策 |
|-----|------|------|
| 计划膨胀 | 计划过多完不成 | 80%原则，留缓冲 |
| 社交绑架 | 被他人打断计划 | 学会说"不" |
| 完美准备 | 花太多时间准备 | 直接开始，边做边学 |
| 虚假忙碌 | 看起来很忙但没产出 | 以结果为导向 |
| 报复性熬夜 | 白天低效，晚上加班 | 固定作息时间 |
| 信息过载 | 学太多不消化 | 少即是多，注重复习 |

- [ ] 今日三大任务完成了吗？
- [ ] 高效时段利用如何？
- [ ] 是否有不必要的打断？
- [ ] 明天计划准备好了吗？

- [ ] 周目标完成度？
- [ ] 时间分配是否合理？
- [ ] 效率有提升吗？
