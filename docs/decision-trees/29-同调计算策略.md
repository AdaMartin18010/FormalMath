# 同调计算策略决策树

## 概述

本决策树为代数拓扑中的同调群计算提供系统化的策略选择，涵盖奇异同调、胞腔同调、上同调、Mayer-Vietoris序列等核心计算工具。

## 核心计算策略

### 胞腔同调

**胞腔链复形**：

```

X 的 CW 复形结构：
- Xⁿ：n-骨架
- n-胞腔：eₙᵅ

链群：
Cₙ^cell(X) = Hₙ(Xⁿ, Xⁿ⁻¹) ≅ ℤ^{#n-胞腔}

边缘算子：
∂ₙ: Cₙ → Cₙ₋₁
dₙᵅᵝ = deg(Sᵅᵝ: Sⁿ⁻¹→Sⁿ⁻¹)

```

**计算步骤**：

```

1. 给 CW 结构
2. 确定胞腔数
3. 计算边缘算子（附着映射的度）
4. 计算 Hₙ = ker ∂ₙ / im ∂ₙ₊₁

```

**常见空间的胞腔同调**：

```

Sⁿ：
- 1个0-胞腔，1个n-胞腔
- H₀=ℤ, Hₙ=ℤ, 其余0

ℂPⁿ：
- 1个0-胞腔, 1个2-胞腔, ..., 1个2n-胞腔
- H_{2k}=ℤ, k=0,...,n

ℝPⁿ：
- 1个k-胞腔，k=0,...,n
- Hₖ(ℝPⁿ;ℤ) = ℤ (k=0,n奇)
                    ℤ/2 (0<k<n, k奇)
                    0 (其他)

```

### 长正合序列

**对序列**：

```

A ⊂ X 的子空间对：
... → Hₙ(A) → Hₙ(X) → Hₙ(X,A) → Hₙ₋₁(A) → ...

应用：
- 已知两个求第三个
- 相对同调→绝对同调

```

**Mayer-Vietoris序列**：

```

X = U ∪ V，U∩V 非空：
... → Hₙ(U∩V) → Hₙ(U)⊕Hₙ(V) → Hₙ(X) → Hₙ₋₁(U∩V) → ...

简化版（约化同调）：
X = A ∪ B，A∩B 可缩：
H̃ₙ(X) ≅ H̃ₙ₋₁(A∩B)

应用：
- 球面：Sⁿ = Dⁿ₊ ∪ Dⁿ₋，交为 Sⁿ⁻¹
- 透镜空间
- 连通和

```

### Künneth公式

**同调Künneth**：

```

Hₙ(X×Y) ≅ ⊕ᵢ Hᵢ(X)⊗Hₙ₋ᵢ(Y) ⊕ ⊕ᵢ Tor(Hᵢ(X), Hₙ₋ᵢ₋₁(Y))

当系数域：
Hₙ(X×Y; F) ≅ ⊕ᵢ Hᵢ(X; F)⊗F Hₙ₋ᵢ(Y; F)

```

**上同调Künneth**：

```

H*(X×Y) ≅ H*(X)⊗H*(Y) （域系数）

杯积结构：
(a⊗b)∪(c⊗d) = (-1)^{|b||c|}(a∪c)⊗(b∪d)

```

### 上同调与杯积

**上同调定义**：

```

Cⁿ(X) = Hom(Cₙ(X), ℤ)
δ: Cⁿ → Cⁿ⁺¹, (δf)(σ) = f(∂σ)

Hⁿ(X) = ker δ / im δ

```

**万有系数定理**：

```

上同调：
0 → Ext(Hₙ₋₁(X), G) → Hⁿ(X; G) → Hom(Hₙ(X), G) → 0

分裂（非典范）：
Hⁿ(X; G) ≅ Hom(Hₙ(X), G) ⊕ Ext(Hₙ₋₁(X), G)

```

**杯积**：

```

∪: Hᵖ(X) × Hᑫ(X) → Hᵖ⁺ᑫ(X)

性质：
- 分次交换：a∪b = (-1)^{pq} b∪a
- 结合律
- 有单位元 1 ∈ H⁰(X)

计算：
- 对角映射 Δ: X → X×X
- a∪b = Δ*(a×b)

```

### 基本计算技巧

**归纳法**：

```

用 MV 序列归纳计算：
- Sⁿ：Dⁿ₊ ∪ Dⁿ₋
- ℂPⁿ：ℂPⁿ⁻¹ ∪ D²ⁿ
- ℝPⁿ：ℝPⁿ⁻¹ ∪ Dⁿ

```

**切除定理**：

```

Z ⊂ A ⊂ X，且 Z̄ ⊂ A°：
Hₙ(X, A) ≅ Hₙ(X\Z, A\Z)

应用：简化相对同调计算

```

**维数公理**：

```

Hₙ(pt) = ℤ (n=0), 0 (n>0)

推论：
- 可缩空间同调同构于点
- 道路连通：H₀ ≅ ℤ

```

## 证明模板

### 计算球面同调

```

计算 Hₙ(Sᵏ)：

用 MV 序列：
Sᵏ = Dᵏ₊ ∪ Dᵏ₋，交为 Sᵏ⁻¹

序列：
... → H̃ₙ(Sᵏ⁻¹) → H̃ₙ(Dᵏ₊)⊕H̃ₙ(Dᵏ₋) → H̃ₙ(Sᵏ) → H̃ₙ₋₁(Sᵏ⁻¹) → ...

因 Dᵏ 可缩：
H̃ₙ(Dᵏ) = 0

故：
H̃ₙ(Sᵏ) ≅ H̃ₙ₋₁(Sᵏ⁻¹)

归纳基础：
H̃₀(S⁰) = ℤ（两点不连通）

归纳得：
H̃ₙ(Sᵏ) = ℤ (n=k), 0 (n≠k)

```

### 计算实射影空间同调

```

计算 Hₙ(ℝPᵏ)：

CW 结构：
1个0-胞腔, 1个1-胞腔, ..., 1个k-胞腔

边缘算子：
∂ₙ: ℤ → ℤ
对 n 奇：×2
对 n 偶：×0

链复形：
... → ℤ →ᶻˣ² ℤ →ᶻˣ⁰ ℤ →ᶻˣ² ℤ →ᶻˣ⁰ ℤ → 0

计算同调：
Hₙ(ℝPᵏ) = 
  ℤ       (n=0)
  ℤ/2     (0<n<k, n奇)
  ℤ       (n=k, k奇)
  0       (其他)

```

## 常见问题解决策略

| 问题 | 策略 |
|------|------|
| 计算同调 | 胞腔结构 → 链复形 → 同调 |
| 乘积空间 | Künneth公式 |
| 并空间 | Mayer-Vietoris序列 |
| 相对同调 | 长正合序列 |
| 系数变换 | 万有系数定理 |
| 上同调环 | 杯积计算 |
| 判断同调类 | 用配对或映射到球面 |
| 计算Ext/Tor | 自由分解或标准公式 |
