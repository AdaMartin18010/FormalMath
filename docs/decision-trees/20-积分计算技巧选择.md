## 核心计算策略

### 不定积分基础技巧

**换元积分法**：
```
第一类换元（凑微分）：∫f(φ(x))φ'(x)dx = ∫f(u)du
第二类换元：令 x = φ(t)，dx = φ'(t)dt

常用换元：
- √(a²-x²): x = a sin t 或 a cos t
- √(a²+x²): x = a tan t
- √(x²-a²): x = a sec t
- n√(ax+b): t = n√(ax+b)
```

**分部积分法（LIATE法则）**：
```
∫u dv = uv - ∫v du

选择 u 的优先级（LIATE）：
L - 对数函数 (Logarithmic)
I - 反三角函数 (Inverse trigonometric)
A - 代数函数 (Algebraic)
T - 三角函数 (Trigonometric)
E - 指数函数 (Exponential)
```

### 有理函数积分

**部分分式分解**：
```
P(x)/Q(x) 分解为：
- A/(x-a) —— 单重实根
- A/(x-a)^k —— k重实根
- (Ax+B)/(x²+px+q) —— 单重复根
- (Ax+B)/(x²+px+q)^k —— k重复根
```

### 定积分特殊技巧

**对称性利用**：
```
∫₋ₐ^a f(x)dx = 0, 若 f 奇函数
∫₋ₐ^a f(x)dx = 2∫₀^a f(x)dx, 若 f 偶函数

区间再现：∫₀^a f(x)dx = ∫₀^a f(a-x)dx
```

**Wallis公式**：
```
∫₀^(π/2) sinⁿx dx = ∫₀^(π/2) cosⁿx dx =
  (n-1)!!/n!! · π/2,  n偶
  (n-1)!!/n!!,        n奇
```

**Feynman积分技巧**（积分号下求导）：
```
I(a) = ∫f(x,a)dx
则 I'(a) = ∫∂f/∂a dx
通过求导简化积分，再积分回去
```

### 重积分坐标变换

**极坐标**：
```
x = r cos θ, y = r sin θ
∬f(x,y)dxdy = ∬f(r cos θ, r sin θ) r dr dθ

适用：圆形区域、含 x²+y² 的被积函数
```

**柱坐标**：
```
x = r cos θ, y = r sin θ, z = z
dV = r dr dθ dz

适用：柱形区域
```

**球坐标**：
```
x = r sin φ cos θ
y = r sin φ sin θ
z = r cos φ
dV = r² sin φ dr dφ dθ

适用：球形区域、含 x²+y²+z² 的被积函数
```

### 曲线曲面积分

**Green公式**：
```
∮_L Pdx + Qdy = ∬_D (∂Q/∂x - ∂P/∂y) dxdy

应用：将闭曲线积分转为二重积分
条件：L 正向、P,Q 有连续偏导
```

**Gauss公式（散度定理）**：
```
∯_Σ Pdy dz + Qdz dx + Rdx dy = ∭_Ω (∂P/∂x + ∂Q/∂y + ∂R/∂z) dV

或：∯_Σ F·n dS = ∭_Ω ∇·F dV

应用：将闭曲面积分转为三重积分
```

**Stokes公式**：
```
∮_L F·dr = ∬_Σ (∇×F)·n dS

应用：将曲线积分转为曲面积分（反之亦然）
```

### 特殊函数积分

**Gamma函数**：
```
Γ(s) = ∫₀^∞ t^(s-1) e^(-t) dt,  Re(s) > 0

性质：
- Γ(n+1) = n! （n∈ℕ）
- Γ(1/2) = √π
- Γ(s+1) = sΓ(s)
- 余元公式：Γ(s)Γ(1-s) = π/sin(πs)
```

**Beta函数**：
```
B(p,q) = ∫₀^1 t^(p-1) (1-t)^(q-1) dt,  Re(p),Re(q) > 0

与Gamma关系：B(p,q) = Γ(p)Γ(q)/Γ(p+q)

变形：∫₀^(π/2) sin^(2p-1)θ cos^(2q-1)θ dθ = B(p,q)/2
```

## 积分次序选择

### 重积分次序

**选择原则**：
1. 被积函数先积容易的变量
2. 积分区域描述简单的方向
3. 避免分段积分

**常见情形**：
- 含 e^(±x²), sin(x)/x, cos(x)/x：后积该变量
- 含 x^a e^(-x)：可考虑Gamma函数

### 累次积分交换

```
∫_a^b dx ∫_{φ₁(x)}^{φ₂(x)} f(x,y) dy = ∫_c^d dy ∫_{ψ₁(y)}^{ψ₂(y)} f(x,y) dx

适用：交换后积分更简单，或原积分无法直接计算
```

## 计算流程总结

```
1. 识别积分类型
2. 观察被积函数特征
3. 选择合适坐标系/技巧
4. 执行计算
5. 验证结果（求导检验、对称性验证等）
```
