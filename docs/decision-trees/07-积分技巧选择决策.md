## 核心方法详解

### 换元积分法

**第一类换元（凑微分）**：

```

∫f(g(x))g'(x)dx = ∫f(u)du, u=g(x)

```

常见凑微分：
- ∫f(ax+b)dx = (1/a)∫f(u)du
- ∫f'(x)/f(x)dx = ln|f(x)|

- ∫f(x^n)x^(n-1)dx = (1/n)∫f(u)du

**第二类换元**：

```

∫f(x)dx = ∫f(φ(t))φ'(t)dt

```

常用换元：
- 三角换元：去根号
- 倒代换：x=1/t，适用于分母次数高
- Euler换元：处理特殊根式

### 分部积分法

**公式**：

```

∫udv = uv - ∫vdu

```

**LIATE法则**（选择u的优先级）：
1. **L**ogarithmic（对数函数）
2. **I**nverse trig（反三角函数）
3. **A**lgebraic（代数函数）
4. **T**rigonometric（三角函数）
5. **E**xponential（指数函数）

**循环积分**：分部积分后出现原积分，解方程求出

### 有理函数积分

**部分分式分解**：

```

P(x)/Q(x) = 多项式 + 简单分式

```

分解类型：
- A/(x-a)：简单实根
- A/(x-a)^k：重实根
- (Bx+C)/(x²+px+q)：不可约二次因子

### 三角函数积分

**万能代换**：

```

t = tan(x/2), dx = 2dt/(1+t²)
sin x = 2t/(1+t²), cos x = (1-t²)/(1+t²)

```

**特殊技巧**：
- sin^n x：n奇数，凑d(cos x)；n偶数，降幂
- 积化和差：sin A cos B = ½[sin(A+B)+sin(A-B)]

### 定积分技巧

**对称性利用**：

```

∫[-a,a] f(x)dx = 0 (f奇函数)
∫[-a,a] f(x)dx = 2∫[0,a] f(x)dx (f偶函数)

```

**区间再现**：

```

∫[a,b] f(x)dx = ∫[a,b] f(a+b-x)dx

```

**Wallis公式**：

```

∫[0,π/2] sin^n x dx = ∫[0,π/2] cos^n x dx = 
(n-1)!!/n!! · (π/2) (n偶) 或 (n-1)!!/n!! (n奇)

```

## 积分策略速查

| 被积函数 | 推荐方法 | 注意点 |
|---------|---------|-------|
| x^n·e^x | 分部积分 | 多次分部，表格法 |
| e^ax·sin(bx) | 分部积分 | 循环出现，解方程 |
| √(a²-x²) | 三角换元x=asinθ | 画三角形回代 |
| 1/(x²+a²) | 直接公式或x=atanθ | arctan形式 |
| sin^m x·cos^n x | 看m,n奇偶 | 奇数凑微分 |
| (px+q)/(x²+bx+c) | 拆分为两部分 | 分子配成分母的导数 |
