## 分科准备策略

### 数学分析准备

| 阶段 | 重点内容 | 习题来源 | 时间分配 |
|-----|---------|---------|---------|
| 基础 | 极限、连续、微分 | 课后习题 | 30% |
| 积分 | 定积分、反常积分 | 吉米多维奇 | 25% |
| 级数 | 数项级数、函数项级数 | 裴礼文 | 25% |
| 多元 | 重积分、曲线曲面积分 | 真题 | 20% |

**重点题型**：
- 极限计算（15种方法）
- 一致收敛证明
- 积分不等式
- 级数求和

### 线性代数准备

| 主题 | 核心考点 | 常见题型 |
|-----|---------|---------|
| 矩阵 | 运算、逆、秩 | 计算题 |
| 行列式 | 性质、计算 | 证明题 |
| 向量空间 | 基、维数、坐标 | 概念题 |
| 线性变换 | 矩阵表示、特征值 | 综合题 |
| 内积空间 | 正交化、标准形 | 计算+证明 |

### 概率统计准备

| 阶段 | 内容 | 重点 |
|-----|------|------|
| 基础 | 概率公理、随机变量 | 分布计算 |
| 进阶 | 多维分布、数字特征 | 协方差矩阵 |
| 统计 | 估计、检验 | 方法选择 |
| 应用 | 回归、方差分析 | 实际应用 |

## 应试技巧

### 时间分配

```

选择题/填空题：  30%时间
计算题：          40%时间
证明题：          25%时间
检查：            5%时间

```

### 答题策略
1. **先易后难**：快速拿到基础分
2. **分步得分**：不会也要写相关公式
3. **审题仔细**：避免看错条件
4. **书写规范**：证明要有逻辑性
5. **合理放弃**：太难的先跳过

### 检查重点
- [ ] 计算结果是否合理
- [ ] 单位/符号是否正确
- [ ] 证明是否完整
- [ ] 是否漏题
- [ ] 填涂/书写是否清晰

## 心态调整

### 考前焦虑应对
1. **充分准备**：最好的减压方式
2. **积极暗示**："我已经准备好了"
3. **放松技巧**：深呼吸、冥想
4. **适度运动**：释放压力
5. **充足睡眠**：保证状态

### 考试中紧张
- 先做几道简单题建立信心
- 遇到难题深呼吸，暂时跳过
- 专注当下，不想结果
- 相信自己的能力
