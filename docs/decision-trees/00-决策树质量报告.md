## 使用建议

### 快速开始
1. 阅读 `00-决策树使用指南.md` 了解基本概念
2. 根据需求选择相应的索引文件
3. 按照决策树指引逐步分析
4. 使用检查清单验证决策和执行

### 组合使用
- **学习新领域**：前置知识评估 → 学习路径 → 资源选择 → 学习计划
- **解决问题**：问题诊断 → 方法选择 → 执行 → 验证
