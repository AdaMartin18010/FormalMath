## 核心求解策略

### 线性方程组

**解的判定**：

```

Ax = b

r = rank(A), r̃ = rank([A|b])

- r < r̃：无解（不相容）
- r = r̃ = n：唯一解
- r = r̃ < n：无穷多解（n-r个自由变量）

```

**求解方法**：

```

1. Gauss消元法：
   [A|b] → 行阶梯形 → 回代

2. 解的结构：
   x = x_p + x_h
   - x_p：特解
   - x_h：齐次方程通解（零空间）

```

**Cramer法则**：

```

当 det(A) ≠ 0：
xᵢ = det(Aᵢ) / det(A)

其中 Aᵢ 是用 b 替换 A 的第 i 列

局限：计算量大，仅理论价值

```

### 矩阵计算

**行列式计算**：

```

方法1：行变换化为上三角
  - 行交换：变号
  - 行倍乘：提出倍数
  - 行倍加：不变

方法2：Laplace展开（按行/列）
  det(A) = Σ aᵢⱼ Cᵢⱼ

方法3：特征值
  det(A) = λ₁ · λ₂ · ... · λₙ

```

**逆矩阵计算**：

```

方法1：伴随矩阵
  A⁻¹ = A* / det(A)
  （A*是伴随矩阵，A* = (cof(A))ᵀ）

方法2：初等行变换
  [A|I] → [I|A⁻¹]

方法3：分块矩阵（特殊结构）

```

**秩的计算**：

```

方法1：行阶梯形非零行数
方法2：最高阶非零子式阶数
方法3：列（行）向量组的极大无关组大小

```

### 特征问题

**特征值与特征向量**：

```

特征多项式：p(λ) = det(λI - A)
特征值：p(λ) = 0 的根
特征向量：(λI - A)v = 0 的非零解

```

**代数重数与几何重数**：

```

alg(λ)：特征多项式中 λ 的重数
geo(λ)：特征子空间维数 = n - rank(λI - A)

性质：geo(λ) ≤ alg(λ)

```

**矩阵对角化**：

```

A 可对角化 ⇔ 以下条件等价：
1. 有 n 个线性无关特征向量
2. 对每个特征值 λ，geo(λ) = alg(λ)
3. 极小多项式无重根

对角化：A = PDP⁻¹
  - P = [v₁, v₂, ..., vₙ]（特征向量矩阵）
  - D = diag(λ₁, λ₂, ..., λₙ)

```

**谱分解（对称矩阵）**：

```

A 实对称：A = QΛQᵀ

- Q 正交矩阵（特征向量标准正交）
- Λ 对角阵（特征值）

性质：
- 特征值都是实数
- 不同特征值的特征向量正交

```

### 标准形

**Jordan标准形**：

```

任意复方阵 A：A = PJP⁻¹

J = diag(J_{λ₁}, J_{λ₂}, ..., J_{λₖ})
Jordan块：
  J_λ = λI + N =
  [λ 1 0 ... 0]
  [0 λ 1 ... 0]
  [...       ]
  [0 0 0 ... λ]

算法：
1. 求特征值
2. 对每个 λ，求广义特征向量链
3. 构造 P

```

**有理标准形**：

```

基于不变因子的标准形
由友矩阵块组成

优点：在有理数域上计算

```

### 内积空间

**Gram-Schmidt正交化**：

```

给定向量组 {v₁, v₂, ..., vₙ}：

u₁ = v₁
u₂ = v₂ - proj_{u₁}(v₂)
u₃ = v₃ - proj_{u₁}(v₃) - proj_{u₂}(v₃)
...

eᵢ = uᵢ / ||uᵢ|| （单位化）

其中 proj_u(v) = <v,u>/<u,u> · u

```

**正交投影**：

```

到子空间 W 的投影：
proj_W(v) = Σ <v,eᵢ> eᵢ

其中 {eᵢ} 是 W 的标准正交基

矩阵形式：P = AA⁺ （A的列是W的基）

```

**伴随算子**：

```

有限维：A* = Āᵀ （共轭转置）

性质：
<Ax, y> = <x, A*y>

```

### 二次型

**标准化**：

```

方法1：配方法
  逐步配方消去交叉项

方法2：正交变换
  对对称阵 A，找正交 Q 使 QᵀAQ = Λ
  令 x = Qy，则 xᵀAx = yᵀΛy

```

**正定性判定**：

```

A 实对称矩阵：

正定 ⇔ 以下条件等价：
1. ∀x ≠ 0, xᵀAx > 0
2. 所有特征值 > 0
3. 所有顺序主子式 > 0
4. 存在可逆 C，使 A = CᵀC

半正定：将 > 改为 ≥
负定：A 正定 ⇔ -A 负定

```

**惯性定律**：

```

实二次型的标准形中：
- 正平方项个数（正惯性指数）
- 负平方项个数（负惯性指数）
- 零平方项个数

在可逆线性变换下不变

```

## 证明模板

### 证明矩阵可对角化

```

证明 A 可对角化：

方法1：证有 n 个线性无关特征向量
- 找所有特征值
- 对每个 λ，找 geo(λ) 个线性无关特征向量
- 验证总和为 n

方法2：证极小多项式无重根
- 找极小多项式 m_A(x)
- 验证 m_A 无重因式

方法3：证 A 正规（复）或对称（实）
- 复：证 A*A = AA*
- 实：证 A = Aᵀ

```

### 证明二次型正定性

```

证明 A 正定：

方法1（定义）：
∀x ≠ 0，证 xᵀAx > 0

方法2（特征值）：
证 A 的所有特征值 > 0

方法3（主子式）：
证所有顺序主子式 > 0
  Δ₁ = a₁₁ > 0
  Δ₂ = |a₁₁ a₁₂| > 0
       |a₂₁ a₂₂|

  ...

方法4（分解）：
找可逆 C，使 A = CᵀC

```

## 常见问题解决策略

| 问题 | 策略 |
|------|------|
| 求行列式 | 行变换化三角或Laplace展开 |
| 求逆矩阵 | 初等行变换 [A\|I]→[I\|A⁻¹] |
| 解方程组 | Gauss消元或矩阵求逆 |
| 对角化 | 找特征值和特征向量，验证可对角化条件 |
| Jordan形 | 求特征值，找广义特征向量链 |
| 正交化 | Gram-Schmidt过程 |
| 正定性 | 特征值、主子式或定义 |
| 最小多项式 | 找使 m(A)=0 的次数最低首一多项式 |
