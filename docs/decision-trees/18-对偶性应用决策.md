## 对偶性类型详解

### 1. 线性规划对偶

**原问题（Primal）**：
```
minimize    cᵀx
subject to  Ax ≥ b
            x ≥ 0
```

**对偶问题（Dual）**：
```
maximize    bᵀy
subject to  Aᵀy ≤ c
            y ≥ 0
```

**对偶定理**：
- **弱对偶**：cᵀx ≥ bᵀy（对任意可行解）
- **强对偶**：若原问题有最优解，则对偶问题也有，且最优值相等
- **互补松弛**：最优解满足互补条件

**应用**：
- 原问题约束多时，对偶变量少，更易求解
- 灵敏度分析：对偶变量表示约束的影子价格
- 整数规划：提供下界

### 2. Lagrange对偶（凸优化）

**原问题**：
```
minimize    f₀(x)
subject to  fᵢ(x) ≤ 0, i=1,...,m
            hⱼ(x) = 0, j=1,...,p
```

**Lagrange函数**：
```
L(x,λ,ν) = f₀(x) + Σλᵢfᵢ(x) + Σνⱼhⱼ(x)
```

**对偶函数**：
```
g(λ,ν) = infₓ L(x,λ,ν)
```

**对偶问题**：
```
maximize    g(λ,ν)
subject to  λ ≥ 0
```

**KKT条件**：
- 原可行性
- 对偶可行性
- 互补松弛
- 梯度条件

### 3. 线性代数对偶空间

**对偶空间定义**：
```
V* = Hom(V, F) = {线性泛函 φ: V → F}
```

**自然配对**：
```
⟨φ, v⟩ = φ(v)
```

**双重对偶**：
```
V → V**（自然映射）
v ↦ v̂, 其中v̂(φ) = φ(v)
```

**有限维**：V ≅ V* ≅ V**

**无限维**：需要自反性条件

### 4. Hilbert空间对偶

**Riesz表示定理**：
```
H是Hilbert空间
∀φ∈H*, ∃!y∈H, 使得 φ(x) = ⟨x,y⟩
且 ||φ|| = ||y||
```

**推论**：H* ≅ H（共轭线性同构）

### 5. Pontryagin对偶

**定义**：局部紧Abel群G的对偶群
```
Ĝ = Hom(G, 𝕋) = {连续同态 χ: G → 𝕋}
其中𝕋 = {z∈ℂ : |z|=1}
```

**Pontryagin对偶定理**：
```
Ĝ̂ ≅ G
```

**经典对偶对**：

| 群G | 对偶群Ĝ | 说明 |
|-----|--------|------|
| ℝ | ℝ | 自对偶 |
| 𝕋 = ℝ/ℤ | ℤ | 圆对偶整数 |
| ℤ | 𝕋 | 整数对偶圆 |
| ℤₙ | ℤₙ | 有限自对偶 |
| ℚₚ | ℚₚ | p进数自对偶 |

**Fourier变换**：
```
F: L²(G) → L²(Ĝ)
(Ff)(χ) = ∫_G f(x)χ̄(x)dx
```

### 6. 范畴对偶

**对偶范畴C^op**：
- 对象：与C相同
- 态射：C中态射反向

**反变Hom函子**：
```
Hom(-, X): C^op → Set
Y ↦ Hom(Y, X)
```

**对偶对象**：
- 向量空间：V* = Hom(V, k)
- 拓扑空间：Stone对偶
- 代数几何：Serre对偶、Grothendieck对偶

### 7. 分布对偶

**测试函数空间D(U)**：
- 紧支撑光滑函数
- 拓扑：归纳极限

**分布空间D'(U)**：
- D(U)的连续对偶
- 广义函数

**Sobolev空间对偶**：
```
H^{-s}(ℝⁿ) = (H^s(ℝⁿ))*
```

## 对偶性应用时机

| 问题特征 | 对偶类型 | 应用效果 |
|---------|---------|---------|
| 约束多变量少 | LP对偶 | 简化问题 |
| 凸优化问题 | Lagrange对偶 | 分解问题 |
| 无限维空间 | 对偶空间 | 泛函分析工具 |
| 调和分析 | Pontryagin对偶 | Fourier理论 |
| 代数几何 | Serre对偶 | 上同调计算 |
| PDE理论 | 分布对偶 | 弱解框架 |

## 对偶性思维

**核心思想**：每个对象都有一个"对偶"对象，
- 原问题的困难可能在对偶问题中简化
- 对偶提供新的视角和工具
- 对称性往往通过对偶体现
