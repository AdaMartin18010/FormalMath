# 证明困难诊断

## 决策目标
诊断数学证明过程中遇到的困难，分析原因并提供针对性解决方案。

| 策略 | 适用场景 | 示例 |

## 证明写作检查表

### 内容层面
- [ ] 假设和结论明确
- [ ] 每一步都有依据
- [ ] 逻辑链条完整
- [ ] 特殊情况覆盖
- [ ] 结论正确得出

### 形式层面
- [ ] 符号定义清楚
- [ ] 量词使用正确（∀∃顺序）
- [ ] 逻辑连接词准确
- [ ] 段落结构清晰
- [ ] 数学表达规范
- [ ] 证明有明确结束标记

### 风格层面
- [ ] 叙述流畅自然
- [ ] 不过度依赖"显然"
- [ ] 关键步骤有解释
- [ ] 适当使用图表

## 证明训练进阶

### Level 1：模仿证明
- 阅读标准证明
- 理解每一步
- 尝试复现
- 对比差异

### Level 2：变式证明
- 改变定理条件
- 证明类似命题
- 使用不同方法
- 证明逆命题

### Level 3：独立证明
- 不看提示自己尝试
- 允许多次失败
- 总结失败原因
- 建立个人策略库

### Level 4：创造证明
- 自己提出问题
- 尝试证明新结果
- 阅读研究论文中的证明
- 改进现有证明

## 常见证明陷阱

### 逻辑陷阱

| 陷阱 | 示例 | 避免方法 |
|-----|------|---------|
| 循环论证 | 用结论证明中间步骤 | 反向检查 |
| 否定错误 | ¬(∀x,P(x)) ≠ ∀x,¬P(x) | 牢记De Morgan |
| 逆命题错误 | A⇒B ≠ B⇒A | 举反例检验 |
| 偷换概念 | 改变定义使用 | 对照原始定义 |

### 技术陷阱

| 陷阱 | 示例 | 避免方法 |
|-----|------|---------|
| 除以零 | 消去含变量分母 | 检查分母非零 |
| 无穷操作 | 交换极限/求和 | 验证一致收敛 |
| 几何直观 | 用图画代替证明 | 代数化验证 |

## 求助指南

### 自助解决不了时
- 卡在同一问题超过2小时
- 多次尝试同一策略失败
- 不知道自己的漏洞在哪

### 有效求助方式
1. **向老师求助**：
   - 带着已尝试的方法
   - 指出具体卡住的地方
   - 询问解题思路而非答案

2. **与同学讨论**：
   - 轮流讲解自己的思路
   - 互相质疑每一步
   - 共同寻找突破口

3. **参考资源**：
   - 查阅类似定理的证明
   - 寻找提示而非完整解答
   - 学习证明技巧书
