## 离散分布详解

### 二项分布 Binomial(n,p)
- **场景**：n次独立试验，每次成功概率p
- **PMF**：P(X=k) = C(n,k)p^k(1-p)^(n-k)
- **期望**：np，**方差**：np(1-p)

### Poisson分布 Poisson(λ)
- **场景**：单位时间/空间内随机事件发生次数
- **PMF**：P(X=k) = e^(-λ)λ^k/k!
- **期望**=**方差**：λ
- **性质**：可加性

### 几何分布 Geometric(p)
- **场景**：首次成功所需的试验次数
- **PMF**：P(X=k) = (1-p)^(k-1)p
- **性质**：无记忆性
- **期望**：1/p

### 负二项分布 NB(r,p)
- **场景**：第r次成功所需的试验次数
- **与几何关系**：r=1时为几何分布

### 超几何分布 Hypergeometric(N,K,n)
- **场景**：不放回抽样
- **与二项关系**：N大时近似二项

## 连续分布详解

### 正态分布 Normal(μ,σ²)
- **PDF**：f(x) = (1/√(2π)σ)exp(-(x-μ)²/(2σ²))
- **性质**：中心极限定理的基础
- **应用**：测量误差、自然现象

### 指数分布 Exponential(λ)
- **PDF**：f(x) = λe^(-λx), x≥0
- **性质**：无记忆性
- **应用**：等待时间、寿命

### Gamma分布 Gamma(α,β)
- **PDF**：f(x) = β^α x^(α-1)e^(-βx)/Γ(α)
- **特例**：α=1时为指数分布
- **应用**：等待多个事件的总时间

### Beta分布 Beta(α,β)
- **支撑集**：[0,1]
- **应用**：比例、概率的建模
- **共轭先验**：二项分布的共轭先验

### Weibull分布 Weibull(λ,k)
- **应用**：可靠性分析、寿命数据
- **形状参数k**：
  - k<1：失效率递减
  - k=1：指数分布（常数失效率）
  - k>1：失效率递增

### Pareto分布 Pareto(α,xm)
- **特性**：厚尾分布
- **应用**：收入分布、损失建模
- **80-20法则**：α=log(4)/log(5)时

### 对数正态分布 LogNormal(μ,σ²)
- **性质**：若log(X)~Normal，则X~LogNormal
- **应用**：股价、粒子大小、收入

## 分布选择速查表

| 问题特征 | 推荐分布 | 替代分布 |
|---------|---------|---------|
| 固定次数成功计数 | 二项 | 超几何（不放回） |
| 单位时间事件发生 | Poisson | 负二项（过度离散） |
| 等待首次事件 | 几何 | 指数（连续时间） |
| 连续测量误差 | 正态 | t分布（小样本） |
| 等待多个事件 | Gamma | Erlang（整数α） |
| 寿命/可靠性 | Weibull | 指数（特殊情况） |
| 比例/概率 | Beta | 均匀（α=β=1） |
| 极端事件 | GEV | Pareto（尾部） |

## 分布关系图

```

                    正态分布
                   ↗    ↖
           中心极限定理   对数变换
             ↗              ↖
    独立同分布和               对数正态
         ↗                       ↖
    二项分布 ← n→∞,p→0 → Poisson
         ↓                        ↓
    负二项 ← 等待r次成功 ← 几何分布
         
    指数 ← 等待时间 → Gamma

```

## 选择分布的步骤

1. **确定变量类型**：离散/连续/混合
2. **分析支撑集**：取值范围
3. **考虑物理意义**：问题背景
4. **检查分布性质**：对称性、尾部、峰度
5. **数据拟合检验**：QQ图、假设检验
