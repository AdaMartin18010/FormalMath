## 核心求解策略

### 表示构造

**置换表示**：
```
群 G 作用在集合 X 上：
V = ℂ[X] （以 X 为基的形式向量空间）

ρ(g)(Σaₓ·x) = Σaₓ·(g·x)

特征标：χ(g) = |X^g| （g的不动点个数）
```

**正则表示**：
```
G 通过左乘作用在自身：
V = ℂ[G]

性质：
- dim V = |G|
- 包含所有不可约表示
- 每个不可约表示 Vᵢ 出现 dim Vᵢ 次
- 特征标：χ_reg(g) = |G|·δ_{g,e}
```

**矩阵表示**：
```
ρ: G → GL_n(ℂ)

构造方法：
1. 定义生成元的像
2. 验证关系保持
3. 延拓到整个群
```

### 特征标理论

**基本性质**：
```
χ_V(g) = tr(ρ_V(g))

性质：
- χ(e) = dim V
- χ(hgh⁻¹) = χ(g) （类函数）
- χ(g⁻¹) = χ(g)̄ （复表示）
- χ_{V⊕W} = χ_V + χ_W
- χ_{V⊗W} = χ_V · χ_W
```

**正交关系**：
```
第一正交关系（表示正交）：
⟨χᵢ, χⱼ⟩ = (1/|G|) Σ_g χᵢ(g) χⱼ(g)̄ = δᵢⱼ

第二正交关系（共轭类正交）：
Σᵢ χᵢ(g) χᵢ(h)̄ = |G|/|C(g)| · δ_{C(g),C(h)}
```

**特征标内积**：
```
⟨χ, ψ⟩ = (1/|G|) Σ_g χ(g) ψ(g)̄

应用：
- 不可约判定：⟨χ,χ⟩=1 ⇔ χ不可约
- 重数计算：Vᵢ 在 V 中出现 mᵢ 次
  mᵢ = ⟨χ_V, χᵢ⟩
```

### 不可约表示

**Maschke定理**：
```
条件：char(ℂ) ∤ |G| （自动满足）

结论：任何表示完全可约
即：V ≅ V₁^{⊕m₁} ⊕ V₂^{⊕m₂} ⊕ ... ⊕ Vₖ^{⊕mₖ}

应用：只需研究不可约表示
```

**维数公式**：
```
|G| = Σᵢ₌₁ᵏ (dim Vᵢ)²

其中 V₁, ..., Vₖ 是所有不可约表示

应用：
- 检验不可约表示是否找全
- 计算不可约表示的维数
```

**不可约表示个数**：
```
不可约表示个数 = 共轭类个数

应用：确定不可约表示的数量
```

**投影算子**：
```
到同型分量 Vᵢ^{⊕mᵢ} 的投影：
Pᵢ = (dim Vᵢ / |G|) Σ_g χᵢ(g⁻¹) ρ(g)

应用：从任意表示中提取不可约分支
```

### 张量积表示

**定义与性质**：
```
V ⊗ W 作为向量空间：
- 基：{vᵢ ⊗ wⱼ}
- dim(V ⊗ W) = dim V · dim W

G作用：g·(v ⊗ w) = (g·v) ⊗ (g·w)

特征标：χ_{V⊗W} = χ_V · χ_W
```

**Clebsch-Gordan分解**：
```
Vᵢ ⊗ Vⱼ = ⊕ₖ Nᵢⱼᵏ Vₖ

Clebsch-Gordan系数 Nᵢⱼᵏ：
Nᵢⱼᵏ = ⟨χᵢχⱼ, χₖ⟩ = (1/|G|) Σ_g χᵢ(g)χⱼ(g)χₖ(g)̄

计算：用特征标表计算张量积分解
```

### 诱导与限制

**限制表示**：
```
H ≤ G，V 是 G-表示：
Res_H^G V = V 作为 H-表示

特征标：χ_{Res V}(h) = χ_V(h) （h ∈ H）
```

**诱导表示**：
```
H ≤ G，W 是 H-表示：
Ind_H^G W = ℂ[G] ⊗_{ℂ[H]} W

特征标公式：
χ_{Ind W}(g) = Σ_{x∈G/H, x⁻¹gx∈H} χ_W(x⁻¹gx)
         = (1/|H|) Σ_{x∈G, x⁻¹gx∈H} χ_W(x⁻¹gx)
```

**Frobenius互反**：
```
⟨Ind_H^G W, V⟩_G = ⟨W, Res_H^G V⟩_H

应用：
- 计算诱导表示的重数
- 限制表示的分解
```

### 表示分类

**Abel群的表示**：
```
G Abel：所有不可约表示都是一维的

数目：|G| 个

构造：
G ≅ ℤ_{n₁} × ... × ℤ_{nₖ}
对每个因子选特征标 χⱼ(g) = e^{2πi mⱼ/nⱼ}
```

**对称群的表示**：
```
Sₙ 的不可约表示 ↔ n 的分划

Specht模 S^λ：对应分划 λ 的表示
维数：钩长公式
```

**特征标表的构造**：
```
1. 列出共轭类 C₁, ..., Cₖ
2. 列出不可约特征标 χ₁, ..., χₖ
3. 利用：
   - 正交关系
   - 维数公式
   - 特征标性质
4. 填充表格
```

## 证明模板

### 证明表示不可约

```
证明 V 不可约：

方法1（特征标）：
⟨χ_V, χ_V⟩ = (1/|G|) Σ_g |χ_V(g)|² = 1

方法2（子表示）：
证 V 没有非平凡子表示
- 设 W ⊂ V 子表示
- 证 W = 0 或 W = V

方法3（交换子代数）：
证 End_G(V) = ℂ·id
（Schur引理）
```

### 计算特征标表

```
构造 G 的特征标表：

步骤1：找共轭类
- 确定共轭类代表元
- 计算各类大小

步骤2：确定表示数
- 不可约表示数 = 共轭类数

步骤3：找已知表示
- 平凡表示（全1）
- 一维表示（Abel化）
- 置换表示分解

步骤4：利用正交关系
- 行正交（第一正交关系）
- 列正交（第二正交关系）
- 维数公式

步骤5：确定剩余特征标
```

## 常见问题解决策略

| 问题 | 策略 |
|------|------|
| 构造表示 | 从群作用、正则表示或定义同态 |
| 计算特征标 | 取矩阵表示的迹或利用特征标性质 |
| 判断不可约 | 计算 ⟨χ,χ⟩ 是否等于1 |
| 分解表示 | 用特征标投影或计算重数 |
| 张量积分解 | Clebsch-Gordan系数 |
| 诱导表示 | Frobenius公式或特征标公式 |
| 构造特征标表 | 共轭类+正交关系+已知表示 |
| 确定所有不可约表示 | 维数公式检验是否完整 |
