## 中值定理体系

### Rolle定理

**条件**：
1. f在[a,b]上连续
2. f在(a,b)内可导
3. f(a) = f(b)

**结论**：存在ξ∈(a,b)，使f'(ξ)=0

**几何意义**：水平弦必有水平切线

### Lagrange中值定理

**条件**：
1. f在[a,b]上连续
2. f在(a,b)内可导

**结论**：存在ξ∈(a,b)，使
```
f'(ξ) = (f(b) - f(a)) / (b - a)
```

**几何意义**：任意弦必有平行切线

**等价形式**：
```
f(b) - f(a) = f'(ξ)(b - a)
f(x) = f(x₀) + f'(ξ)(x - x₀)
```

### Cauchy中值定理

**条件**：
1. f,g在[a,b]上连续
2. f,g在(a,b)内可导
3. g'(x)≠0在(a,b)内

**结论**：存在ξ∈(a,b)，使
```
(f(b) - f(a)) / (g(b) - g(a)) = f'(ξ) / g'(ξ)
```

**与Lagrange关系**：g(x)=x时即为Lagrange

## 应用场景

### 1. 证明方程根的存在性

**策略**：
- 构造F(x)，使F'(x)=f(x)
- 找到F(a)=F(b)的点
- 应用Rolle定理得f(ξ)=0

**示例**：证明e^x = ax + b有解

### 2. 证明不等式

**Lagrange估计**：
```
|f(b) - f(a)| = |f'(ξ)|·|b - a| ≤ M·|b - a|
```
其中M = sup|f'(x)|

**常用不等式**：
- |sin x - sin y| ≤ |x - y|
- |arctan x - arctan y| ≤ |x - y|
- e^x ≥ 1 + x（x≥0时）

### 3. 求极限（L'Hôpital法则）

**条件**：
- lim f(x) = lim g(x) = 0 或 ∞
- f,g在点附近可导
- lim f'(x)/g'(x) 存在

**结论**：
```
lim f(x)/g(x) = lim f'(x)/g'(x)
```

### 4. 高阶导数零点

**Rolle推广**：
- f有n+1个零点 → f'至少有n个零点
- 递推：f^(k)至少有n+1-k个零点

### 5. 泰勒展开的余项

**Lagrange余项**：
```
f(x) = Σ(f^(k)(x₀)/k!)(x-x₀)^k + (f^(n+1)(ξ)/(n+1)!)(x-x₀)^(n+1)
```

## 辅助函数构造技巧

| 目标等式 | 辅助函数构造 | 定理 |
|---------|-------------|------|
| f'(ξ)=0 | F(x)=f(x) | Rolle |
| f'(ξ)+f(ξ)=0 | F(x)=e^x f(x) | Rolle |
| f'(ξ)g(ξ)=f(ξ)g'(ξ) | F(x)=f(x)/g(x) | Rolle |
| f(b)-f(a)=f'(ξ)(b-a) | F(x)=f(x) | Lagrange |
| (f(b)-f(a))g'(ξ)=(g(b)-g(a))f'(ξ) | F=f,G=g | Cauchy |

## 判断使用时机

**使用Rolle定理的信号**：
- 证明f'(ξ)=0或某等式
- 函数在两点取值相同
- 高阶导数零点问题

**使用Lagrange的信号**：
- 涉及函数差值f(b)-f(a)
- 需要估计函数变化
- 证明不等式

**使用Cauchy的信号**：
- 两个函数的比值问题
- 参数方程表示的曲线
- L'Hôpital法则证明

## 常见错误

1. **忽略条件**：不验证连续性和可导性
2. **ξ不独立**：错误认为多个中值点相同
3. **滥用L'Hôpital**：导数比的极限不存在时
4. **辅助函数构造错误**：求导后不能得到目标式
