## 核心分析策略

### Markov链

**离散时间Markov链**：

```

Markov性：
P(Xₙ₊₁=j | Xₙ=i, Xₙ₋₁, ..., X₀) = P(Xₙ₊₁=j | Xₙ=i) = pᵢⱼ

转移矩阵 P = [pᵢⱼ]：
- pᵢⱼ ≥ 0
- Σⱼ pᵢⱼ = 1

n步转移：P⁽ⁿ⁾ = Pⁿ
Chapman-Kolmogorov：P⁽ᵐ⁺ⁿ⁾ = P⁽ᵐ⁾ P⁽ⁿ⁾

```

**状态分类**：

```

常返性：
- 常返：Pᵢ(返回i) = 1，Eᵢ[Tᵢ] < ∞（正常返）或 = ∞（零常返）
- 暂态：Pᵢ(返回i) < 1

周期性：
d(i) = gcd{n: pᵢᵢ⁽ⁿ⁾ > 0}
- 非周期：d(i) = 1
- 周期：d(i) > 1

遍历状态：正常返且非周期

```

**极限行为**：

```

遍历定理：
不可约、非周期、正常返：
- lim pᵢⱼ⁽ⁿ⁾ = πⱼ
- π 是唯一不变分布

不变分布：πP = π, Σπᵢ = 1

遍历性：
(1/n) Σₖ₌₀ⁿ⁻¹ f(Xₖ) → E_π[f(X)]

```

**连续时间Markov链**：

```

生成元（Q矩阵）：
qᵢⱼ = lim_{h→0} (pᵢⱼ(h) - δᵢⱼ)/h  (i≠j)
qᵢᵢ = -Σⱼ≠ᵢ qᵢⱼ

Kolmogorov方程：
- 前向：P'(t) = P(t)Q
- 后向：P'(t) = QP(t)

解：P(t) = e^(Qt) = Σ (Qt)ⁿ/n!

不变分布：πQ = 0

```

### Poisson过程

**基本性质**：

```

计数过程 N(t) = [0,t]内事件发生次数

定义（等价形式）：
1. N(0) = 0
2. 独立增量
3. N(t+s) - N(s) ~ Poisson(λt)
4. 轨道右连续

性质：
- E[N(t)] = Var(N(t)) = λt
- 到达时间 Tₙ ~ Gamma(n, λ)
- 等待时间 τₙ = Tₙ - Tₙ₋₁ ~ Exp(λ)

```

**叠加与分解**：

```

叠加：
独立Poisson过程 N₁(t;λ₁), N₂(t;λ₂)
⇒ N₁ + N₂ ~ Poisson(λ₁+λ₂)

分解（稀疏化）：
事件以概率 p 被"标记"
⇒ 标记事件 ~ Poisson(λp)
  未标记事件 ~ Poisson(λ(1-p))
  且独立

```

**复合Poisson过程**：

```

X(t) = Σᵢ₌₁^{N(t)} Yᵢ

其中 Yᵢ i.i.d.，与 N(t) 独立

特征函数：
φ_{X(t)}(u) = exp(λt(φ_Y(u) - 1))

期望：E[X(t)] = λt E[Y₁]
方差：Var(X(t)) = λt E[Y₁²]

```

### 鞅论

**定义与例子**：

```

鞅：E[Xₙ₊₁ | Fₙ] = Xₙ
下鞅：E[Xₙ₊₁ | Fₙ] ≥ Xₙ
上鞅：E[Xₙ₊₁ | Fₙ] ≤ Xₙ

例子：
- 随机游走的部分和（零均值）
- 赌徒财富（公平游戏）
- Wald鞅：exp(θSₙ)/φ(θ)ⁿ
- Brown运动 Bₜ

```

**停时定理**：

```

停时 τ：{τ ≤ n} ∈ Fₙ

Doob可选停时定理：
若 Xₙ 一致可积鞅，τ 停时：
E[X_τ] = E[X₀]

应用：
- 赌徒破产问题
- 首次通过时间
- 期权定价

```

**鞅收敛定理**：

```

下鞅收敛定理：
若 sup E[Xₙ⁺] < ∞，则 Xₙ → X∞ a.s.

一致可积鞅：
Xₙ → X∞ a.s. 且 L¹

应用：证明极限存在

```

### 布朗运动

**定义与性质**：

```

标准布朗运动 {Bₜ}ₜ≥₀：
1. B₀ = 0
2. 独立增量
3. Bₜ - Bₛ ~ N(0, t-s)
4. 轨道连续

性质：
- 鞅：E[Bₜ | Fₛ] = Bₛ

- Markov过程
- 自相似：B_{ct} ~ √c Bₜ
- 轨道无处可微

```

**二次变差**：

```

<B>ₜ = lim_{|π|→0} Σ(B_{tᵢ} - B_{tᵢ₋₁})² = t

Itô等距：
E[(∫₀ᵗ Hₛ dBₛ)²] = E[∫₀ᵗ Hₛ² ds]

```

**Itô积分与公式**：

```

Itô积分：∫₀ᵗ Hₛ dBₛ
- 被积过程 H 适应且满足可积条件

Itô公式（链式法则）：
df(t, Xₜ) = ∂f/∂t dt + ∂f/∂x dXₜ + ½ ∂²f/∂x² d<X>ₜ

对 Bₜ：
df(Bₜ) = f'(Bₜ)dBₜ + ½ f''(Bₜ)dt

乘积法则：
d(XY) = XdY + YdX + d<X,Y>

```

### 平稳过程

**定义**：

```

严平稳：
(X_{t₁},...,X_{tₙ}) 与 (X_{t₁+h},...,X_{tₙ+h}) 同分布

宽平稳：
- E[Xₜ] = μ （常数）
- Cov(Xₜ, Xₛ) = γ(t-s) （仅依赖时差）

严平稳 + 有限二阶矩 ⇒ 宽平稳
逆一般不成立

```

**自协方差与谱密度**：

```

自协方差函数：
γ(h) = Cov(Xₜ, Xₜ₊ₕ)

性质：
- γ(0) = Var(Xₜ)
- γ(-h) = γ(h) （偶函数）
- |γ(h)| ≤ γ(0)

谱密度：
若 Σ|γ(h)| < ∞，存在 f(λ)：

f(λ) = (1/2π) Σ γ(h) e^{-ihλ}

Wiener-Khinchin定理：
γ(h) = ∫_{-π}^{π} e^{ihλ} f(λ) dλ

```

**ARMA过程**：

```

AR(p)自回归：
Xₜ = φ₁Xₜ₋₁ + ... + φₚXₜ₋ₚ + εₜ

MA(q)滑动平均：
Xₜ = εₜ + θ₁εₜ₋₁ + ... + θₚεₜ₋ₚ

ARMA(p,q)：
Xₜ - ΣφᵢXₜ₋ᵢ = εₜ + Σθⱼεₜ₋ⱼ

平稳条件：
AR特征方程在单位圆外

```

### 随机微分方程

**Itô SDE**：

```

dXₜ = μ(t, Xₜ)dt + σ(t, Xₜ)dBₜ

解的定义：
Xₜ = X₀ + ∫₀ᵗ μ(s,Xₛ)ds + ∫₀ᵗ σ(s,Xₛ)dBₛ

存在唯一性（Lipschitz条件）：

|μ(t,x)-μ(t,y)| + |σ(t,x)-σ(t,y)| ≤ K|x-y|

```

**线性SDE**：

```

几何布朗运动（Black-Scholes模型）：
dSₜ = μSₜdt + σSₜdBₜ

解：Sₜ = S₀ exp((μ-σ²/2)t + σBₜ)

Ornstein-Uhlenbeck过程：
dXₜ = -θXₜdt + σdBₜ
解：Xₜ = X₀e^{-θt} + σ∫₀ᵗ e^{-θ(t-s)}dBₛ

```

**Fokker-Planck方程**：

```

Xₜ 的转移密度 p(t,x|s,y)：

∂p/∂t = -∂(μp)/∂x + ½ ∂²(σ²p)/∂x²

前向方程，描述概率密度演化

稳态解（当 t→∞）：
-∂(μp)/∂x + ½ ∂²(σ²p)/∂x² = 0

```

## 分析流程模板

### Markov链完整分析

```

给定Markov链：

步骤1：确定转移机制
- 转移矩阵 P 或生成元 Q
- 画出状态转移图

步骤2：状态分类
- 计算互通类
- 判断常返/暂态
- 计算周期

步骤3：求不变分布（如存在）
- 解 πP = π（离散）
- 或 πQ = 0（连续）
- 归一化 Σπᵢ = 1

步骤4：分析极限行为
- 遍历？lim pᵢⱼ⁽ⁿ⁾ = πⱼ
- 周期？子序列极限

步骤5：计算感兴趣的量
- 首达时间期望
- 吸收概率
- 长期平均

```

### SDE求解流程

```

给定SDE：dXₜ = μdt + σdBₜ

步骤1：识别类型
- 线性？几何Brown运动？
- 系数依赖时间/状态？

步骤2：尝试解法
- 猜测解的形式
- 应用Itô公式验证
- 或使用积分因子

步骤3：解的存在唯一性
- 验证Lipschitz条件

步骤4：分析解的性质
- 矩计算
- 平稳分布（如存在）
- 渐近行为

步骤5：模拟验证（如需要）
- Euler-Maruyama离散
- 蒙特卡洛模拟

```

## 常见问题解决策略

| 问题 | 策略 |
|------|------|
| Markov链极限 | 找不变分布+遍历性检验 |
| Poisson过程 | 利用独立增量和计数分布 |
| 鞅验证 | 检查条件期望性质 |
| 停时期望 | 可选停时定理（验证条件） |
| Itô积分 | Itô等距计算矩 |
| SDE求解 | Itô公式+猜测验证 |
| 平稳过程 | 谱分析或ARMA建模 |
| 首达时间 | 建立微分方程或鞅方法 |
