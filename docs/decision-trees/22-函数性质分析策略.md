## 核心分析策略

### 连续性分析

**点连续性定义**：

```

f 在 x₀ 连续：
∀ε>0, ∃δ>0, |x-x₀|<δ ⇒ |f(x)-f(x₀)|<ε

或等价：lim_{x→x₀} f(x) = f(x₀)

```

**一致连续性**：

```

f 一致连续：
∀ε>0, ∃δ>0, |x-y|<δ ⇒ |f(x)-f(y)|<ε

（δ 不依赖于点的位置）

判别：
- 紧集上的连续函数必一致连续（Cantor定理）
- Lipschitz连续 ⇒ 一致连续
- 导数有界 ⇒ Lipschitz连续

```

**连续性保持运算**：

```

连续函数的四则运算（除数非零）：连续
连续函数的复合：连续
连续函数的反函数（严格单调）：连续

```

### 可微性分析

**点可微性**：

```

f 在 x₀ 可微：
lim_{h→0} [f(x₀+h)-f(x₀)]/h 存在

几何意义：切线存在且唯一

```

**可微性层次**：

| 类别 | 定义 | 性质 |
|------|------|------|
| C⁰ | 连续 | - |
| C¹ | 连续可微 | f' 连续 |
| C^k | k 阶连续可微 | f^(k) 连续 |
| C^∞ | 光滑函数 | 任意阶可微 |
| 解析 | 实解析 | 局部可展开为幂级数 |

**多元函数可微性**：

```

f 在 x₀ 可微：
f(x₀+h) = f(x₀) + ∇f(x₀)·h + o(|h|)

偏导存在 + 偏导连续 ⇒ 可微

```

### 单调性分析

**导数判别法**：

```

f'(x) > 0 on (a,b) ⇒ f 在 [a,b] 严格单调增
f'(x) ≥ 0 on (a,b) ⇔ f 在 [a,b] 单调增

注意：f 严格增 ⇏ f' > 0 （如 f(x)=x³）

```

**单调性应用**：
- 不等式证明
- 反函数存在性
- 方程根的唯一性

### 极值分析

**局部极值必要条件**（Fermat定理）：

```

f 在 x₀ 可微且取极值 ⇒ f'(x₀) = 0

临界点：f'(x₀) = 0 或 f'(x₀) 不存在

```

**局部极值充分条件**：

| 条件 | 结论 |
|------|------|
| f'(x₀)=0, f''(x₀)>0 | 局部极小 |
| f'(x₀)=0, f''(x₀)<0 | 局部极大 |
| f'(x₀)=0, f''(x₀)=0, f'''(x₀)≠0 | 拐点（非极值） |
| 最低阶非零导数为偶数阶 | 极值 |
| 最低阶非零导数为奇数阶 | 拐点 |

**全局最值求解**：

```

闭区间 [a,b] 上：
1. 找临界点（f'=0 或不存在的内点）
2. 计算端点值 f(a), f(b)
3. 比较所有值

开区间 (a,b) 上：
需额外考虑极限 lim_{x→a⁺} f(x) 和 lim_{x→b⁻} f(x)

```

**条件极值（Lagrange乘数法）**：

```

求 f(x) 在约束 g(x)=0 下的极值

构造 L = f - λg
解方程组：
∇f = λ∇g
g = 0

```

### 凸性分析

**凸函数定义**：

```

f 凸：∀x,y, ∀λ∈[0,1]
f(λx+(1-λ)y) ≤ λf(x)+(1-λ)f(y)

几何：函数图像在弦下方

```

**凸函数判别法**：

| 方法 | 判别条件 | 适用 |
|------|---------|------|
| 一阶 | f(y) ≥ f(x) + f'(x)(y-x) | 可微函数 |
| 二阶 | f''(x) ≥ 0 | 二阶可微 |
| 多元 | ∇²f 半正定 | 多元函数 |

**凸性应用**：
- Jensen不等式
- 优化问题（局部极小=全局极小）
- Legendre-Fenchel变换

### 渐近分析

**Landau记号**：

```

f = O(g): |f| ≤ C|g| （大O）

f = o(g): f/g → 0 （小o）
f ∼ g: f/g → 1 （渐近等价）
f = Θ(g): f = O(g) 且 g = O(f)

```

**渐近展开**：

```

f(x) = a₀ + a₁x + a₂x² + ... + O(xⁿ)

求法：Taylor展开、递推、待定系数

```

### 积分性质

**Riemann可积条件**：

```

f Riemann可积 ⇔ f 有界且间断点集测度为0

充分条件：
- 连续 ⇒ 可积
- 单调 ⇒ 可积
- 有界+有限个间断点 ⇒ 可积

```

**绝对连续性**：

```

f 绝对连续：
∀ε>0, ∃δ>0, Σ|bᵢ-aᵢ|<δ ⇒ Σ|f(bᵢ)-f(aᵢ)|<ε

等价：f(x) = f(a) + ∫ₐˣ g(t)dt，其中 g∈L¹

```

## 分析流程模板

### 函数全面分析模板

```

对于函数 f(x) = ...，定义域 D = ...

1. 基本性质
   - 定义域、值域
   - 奇偶性、周期性
   - 连续性分析

2. 可微性
   - 可微点、不可微点
   - 导函数表达式
   - 高阶可微性

3. 单调性与极值
   - 单调区间
   - 临界点分析
   - 极值点和最值

4. 凸性与拐点
   - 凸/凹区间
   - 拐点位置

5. 渐近行为
   - 极限值
   - 渐近线（水平、垂直、斜）
   - 渐近展开

6. 图像特征
   - 关键点坐标
   - 变化趋势
   - 草图绘制

```

## 重要定理总结

| 定理 | 内容 | 应用 |
|------|------|------|
| 介值定理 | 连续函数取中间值 | 根的存在性 |
| 最值定理 | 紧集上连续函数有最值 | 极值存在性 |
| Rolle定理 | 端点相等⇒内点导数为0 | 中值定理基础 |
| Lagrange中值 | f(b)-f(a)=f'(ξ)(b-a) | 估计、单调性 |
| Cauchy中值 | 两函数的中值关系 | L'Hôpital法则 |
| Taylor定理 | 带余项的展开 | 近似、估计 |
| 隐函数定理 | 方程确定函数的条件 | 隐函数求导 |
| 反函数定理 | 局部可逆的条件 | 变量替换 |
