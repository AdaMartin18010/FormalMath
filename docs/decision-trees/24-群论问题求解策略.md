## 核心求解策略

### 子群相关

**子群判别法（两步法）**：

```

H ⊆ G 是子群当且仅当：
1. ∀a,b ∈ H, ab ∈ H （运算封闭）
2. ∀a ∈ H, a⁻¹ ∈ H （逆元封闭）

或等价（一步法）：
∀a,b ∈ H, ab⁻¹ ∈ H

```

**生成子群**：

```

<S> = {s₁^{e₁} s₂^{e₂} ... sₖ^{eₖ} | sᵢ ∈ S, eᵢ = ±1, k ∈ ℕ}

性质：
- <S> 是包含 S 的最小子群
- 若 <S> = G，称 S 生成 G

```

**Lagrange定理及应用**：

```

H ≤ G, |G| < ∞:
|G| = |H| · [G:H]

推论：
- 子群阶整除群阶
- 元素阶整除群阶
- |G| 素数 ⇒ G 循环

```

### 正规子群

**等价判定**：

```

H ◁ G 当且仅当以下任一成立：
1. ∀g ∈ G, gHg⁻¹ = H
2. ∀g ∈ G, h ∈ H, ghg⁻¹ ∈ H
3. ∀g ∈ G, gH = Hg （左右陪集相等）
4. H 是某同态的核

```

**常见正规子群**：

```

- 交换群的所有子群
- 指数2的子群
- 群的中心 Z(G)
- 换位子群 [G,G]
- 任意同态的核
- Sylow子群（在特定条件下）

```

### 同态与同构

**第一同构定理**：

```

φ: G → H 是同态，则：
G/ker φ ≅ im φ

构造同构：
φ̄: G/ker φ → im φ
φ̄(g·ker φ) = φ(g)

```

**同态基本性质**：

```

- ker φ ◁ G （核是正规子群）
- im φ ≤ H （像的子群）
- φ 单射 ⇔ ker φ = {e}
- φ 满射 ⇔ im φ = H

```

**对应定理**：

```

φ: G → H 满同态
则 {G 的包含 ker φ 的子群} ↔ {H 的子群}
保持：包含关系、指数、正规性

```

### Sylow定理

**三个定理**：

```

设 |G| = pⁿ·m, p∤m

第一（存在性）：G 有阶为 pⁿ 的子群（Sylow p-子群）

第二（共轭性）：所有 Sylow p-子群互相共轭

第三（计数）：n_p ≡ 1 (mod p) 且 n_p | m

其中 n_p 是 Sylow p-子群的个数

```

**Sylow定理应用**：

```

1. 证明某阶群非单
2. 确定群的正规子群
3. 群结构分析
4. 小阶群分类

例：证明 30 阶群非单
- n₅ ≡ 1 (mod 5), n₅ | 6 ⇒ n₅ = 1 或 6
- n₃ ≡ 1 (mod 3), n₃ | 10 ⇒ n₃ = 1 或 10

- 若 n₅=6, n₃=10，则元素太多（矛盾）
- 故 n₅=1 或 n₃=1，有正规子群

```

### 群作用

**轨道-稳定子定理**：

```

G 作用在 X 上，x ∈ X:

|Gx| · |Stab(x)| = |G|

其中：
- Gx = {gx | g ∈ G} （轨道）
- Stab(x) = {g ∈ G | gx = x} （稳定子群）

```

**类方程**：

```

G 通过共轭作用在自身：

|G| = |Z(G)| + Σ|G:C(xᵢ)|

其中求和取非中心共轭类的代表元

应用：p-群的中心非平凡

```

**Cayley定理**：

```

任意群 G 同构于对称群 S_{|G|} 的子群

构造：G 通过左乘作用在自身

```

### 有限生成Abel群结构

**基本定理**：

```

G 有限生成Abel群，则：
G ≅ ℤʳ ⊕ ℤ_{n₁} ⊕ ℤ_{n₂} ⊕ ... ⊕ ℤ_{nₖ}

其中：
- r ≥ 0 （自由秩）
- n₁ | n₂ | ... | nₖ （不变因子）

或初等因子分解：
G ≅ ℤʳ ⊕ ⊕_{i,j} ℤ_{pᵢ^{eᵢⱼ}}}

```

**不变因子算法**：

```

1. 将群写成循环群的直积
2. 对每个素数 p，收集 p-分量
3. 按大小重新组合得到不变因子

```

## 群分类策略

### 小阶群分类

| 阶数 | 群结构 | 备注 |
|------|--------|------|
| 素数 p | ℤₚ | 循环 |
| p² | ℤ_{p²} 或 ℤₚ × ℤₚ | 交换 |
| 2p | ℤ_{2p} 或 Dₚ | p奇素数 |
| pq (p<q, p∤(q-1)) | ℤ_{pq} | 循环 |
| 8 | 5个群 | 3交换+2非交换 |

### 单群判定

```

群 G 是单群：{e} 和 G 是仅有的正规子群

常见单群：
- ℤₚ （p素数）
- Aₙ （n≥5）
- PSL(2,p) 等 Lie 型单群
- 26个散在单群

非单群判定：
- 交换群且非素数阶
- 有非平凡正规子群

```

## 证明模板

### 证明子群是正规子群

```

证明 H ◁ G：

方法1（定义）：
∀g ∈ G, ∀h ∈ H，证 ghg⁻¹ ∈ H

方法2（陪集）：
证 ∀g ∈ G, gH = Hg

方法3（同态）：
构造同态 φ: G → K，使 H = ker φ

```

### 证明群同构

```

证明 G ≅ H：

步骤1：构造映射 φ: G → H
步骤2：验证 φ 是同态
步骤3：验证 φ 是双射
  - 单射：ker φ = {e}
  - 满射：im φ = H

或使用同构定理：
找 K ◁ G，证 G/K ≅ H

```

## 常见问题解决策略

| 问题 | 策略 |
|------|------|
| 找所有子群 | 从生成元、循环子群入手 |
| 找正规子群 | 从中心、换位子群、Sylow子群入手 |
| 确定群结构 | 用Sylow定理、直积分解 |
| 证明非单 | 找非平凡正规子群 |
| 计算元素阶 | 利用Lagrange定理约束 |
| 找自同构群 | 分析生成元的像 |
