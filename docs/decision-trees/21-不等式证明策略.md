## 核心证明策略

### 经典不等式工具箱

**均值不等式链**：

```

调和平均 HM ≤ 几何平均 GM ≤ 算术平均 AM ≤ 平方平均 QM
n/(∑1/aᵢ) ≤ (∏aᵢ)^(1/n) ≤ (∑aᵢ)/n ≤ √((∑aᵢ²)/n)

等号成立当且仅当所有 aᵢ 相等

```

**Cauchy-Schwarz不等式**：

```

(∑aᵢbᵢ)² ≤ (∑aᵢ²)(∑bᵢ²)

等号成立当且仅当 a 与 b 线性相关

积分形式：
(∫fg)² ≤ (∫f²)(∫g²)

```

**Hölder不等式**：

```

∑|aᵢbᵢ| ≤ (∑|aᵢ|^p)^(1/p) · (∑|bᵢ|^q)^(1/q)

其中 1/p + 1/q = 1

是Cauchy-Schwarz的推广（p=q=2时）

```

**Minkowski不等式**（三角不等式）：

```

(∑|aᵢ+bᵢ|^p)^(1/p) ≤ (∑|aᵢ|^p)^(1/p) + (∑|bᵢ|^p)^(1/p)

L^p空间中的三角不等式

```

**Jensen不等式**（凸函数）：

```

f 凸函数: f(∑λᵢxᵢ) ≤ ∑λᵢf(xᵢ), 其中 ∑λᵢ=1, λᵢ≥0

积分形式：
f(∫g dμ) ≤ ∫f(g) dμ

等号成立条件：g 几乎处处为常数，或 f 线性

```

### 函数不等式证明

**单调性法**：

```

要证 f(x) ≥ g(x) on [a,b]：
1. 令 h(x) = f(x) - g(x)
2. 证 h(a) ≥ 0
3. 证 h'(x) ≥ 0 on [a,b]
4. 结论：h(x) ≥ h(a) ≥ 0

```

**极值法**：

```

要证 f(x) ≥ C：
1. 求 f 的临界点
2. 比较临界值与边界值
3. 证 min f(x) ≥ C

```

**凸性法**：

```

f 凸函数 + f 在某点切线 ≥ f
利用凸函数的几何性质

例：e^x ≥ 1+x (e^x 在 0 处切线)

```

**Taylor展开估计**：

```

f(x) = f(a) + f'(a)(x-a) + f''(ξ)(x-a)²/2

利用Lagrange余项进行估计
适用于：需要更精细估计的情形

```

### 积分不等式

**积分形式的Cauchy-Schwarz**：

```

(∫ₐᵇ f(x)g(x)dx)² ≤ (∫ₐᵇ f²(x)dx)(∫ₐᵇ g²(x)dx)

证明：考虑 ∫(f+tg)² ≥ 0 对任意 t

```

**Chebyshev积分不等式**：

```

若 f,g 同单调（同增或同减）：
∫fg ≥ (∫f)(∫g)/(b-a)

若 f,g 反单调：
∫fg ≤ (∫f)(∫g)/(b-a)

```

**Gronwall不等式**：

```

u(t) ≤ a + ∫₀ᵗ b(s)u(s)ds
⇒ u(t) ≤ a exp(∫₀ᵗ b(s)ds)

应用：ODE解的唯一性、稳定性分析

```

### 序列与级数不等式

**数学归纳法**：

```

1. 验证 n=1（基础情形）
2. 假设 n=k 成立（归纳假设）
3. 证 n=k+1 成立（归纳步骤）
4. 结论：对所有 n∈ℕ 成立

变体：强归纳法、反向归纳法

```

**Abel变换**（分部求和）：

```

∑ₖ₌ₘⁿ aₖbₖ = Aₙbₙ - Aₘ₋₁bₘ - ∑ₖ₌ₘⁿ⁻¹ Aₖ(bₖ₊₁-bₖ)
其中 Aₖ = ∑ᵢ₌ₘᵏ aᵢ

应用：Dirichlet判别法、Abel判别法证明

```

**Stolz-Cesàro定理**：

```

{bₙ} 严格单调且趋于无穷：
lim aₙ/bₙ = lim (aₙ₊₁-aₙ)/(bₙ₊₁-bₙ)

离散版的L'Hôpital法则

```

### 概率不等式

**Markov不等式**：

```

X ≥ 0, a > 0:
Pr(X ≥ a) ≤ E[X]/a

证明：E[X] ≥ E[X·1_{X≥a}] ≥ a·Pr(X≥a)

```

**Chebyshev不等式**：

```

Pr(|X-μ| ≥ kσ) ≤ 1/k²

证明：对 (X-μ)² 应用Markov不等式

```

**Chernoff界**：

```

Pr(X ≥ a) = Pr(e^(tX) ≥ e^(ta)) ≤ E[e^(tX)]/e^(ta) = M_X(t)/e^(ta)

对 t>0 取最优：Pr(X ≥ a) ≤ inf_{t>0} M_X(t)e^(-ta)

```

**Hoeffding不等式**：

```

Xᵢ ∈ [aᵢ,bᵢ] 独立，Sₙ = ∑Xᵢ:
Pr(|Sₙ-E[Sₙ]| ≥ t) ≤ 2exp(-2t²/∑(bᵢ-aᵢ)²)

有界随机变量和的集中不等式

```

### 几何不等式

**凸集分离定理**：

```

两个不相交凸集可用超平面分离
应用：支撑超平面、对偶性

```

**Brunn-Minkowski不等式**：

```

Vol(A+B)^(1/n) ≥ Vol(A)^(1/n) + Vol(B)^(1/n)

等周不等式的推广

```

## 不等式证明模板

### 函数不等式模板

```

证明：令 f(x) = LHS - RHS

第一步：边界值
f(a) = ... ≥ 0

第二步：导数分析
f'(x) = ...
f''(x) = ...

第三步：单调性或凸性
[证明 f' ≥ 0 或 f 凸且在某点≥0]

第四步：结论
f(x) ≥ f(a) ≥ 0，即 LHS ≥ RHS

```

### 对称不等式模板（n元）

```

对于对称不等式，可假设：
a₁ ≥ a₂ ≥ ... ≥ aₙ 或 a₁ ≤ a₂ ≤ ... ≤ aₙ

或使用替换：
令 a₁ = x+y, a₂ = x-y 等

或使用齐次化：假设 ∑aᵢ = 1 或 ∏aᵢ = 1

```

## 常用技巧总结

| 技巧 | 适用场景 | 关键步骤 |
|------|---------|---------|
| 作差法 | 一般不等式 | LHS - RHS 分析 |
| 作商法 | 正数不等式 | LHS/RHS 与1比较 |
| 分析法 | 复杂不等式 | 从结论倒推 |
| 综合法 | 已知条件多 | 从条件正向推导 |
| 放缩法 | 需要估计 | 适当放大或缩小 |
| 构造函数 | 涉及函数 | 求导分析性质 |
| 几何解释 | 有几何意义 | 利用几何直观 |

## 注意事项

1. **等号条件**：证明后验证等号何时成立
2. **定义域**：注意变量的取值范围
3. **齐次性**：利用齐次性简化问题
4. **对称性**：利用对称性减少变量
5. **边界检验**：检验边界情形验证不等式紧性
