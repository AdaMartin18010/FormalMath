## 证明训练阶段规划

### 阶段1：基础证明（4-6周）
**目标**：掌握基本证明格式和方法

**训练内容**：
- 直接证明练习（20题）
- 反证法入门（15题）
- 数学归纳法（15题）
- 定义理解练习（每天2-3个）

**推荐资源**：
- 《The Book of Proof》
- 教材课后证明题

### 阶段2：技巧拓展（6-8周）
**目标**：熟练运用多种证明技巧

**训练内容**：
- 反证法深入（20题）
- 存在性证明（15题）
- 构造性证明（15题）
- 分析证明专题（20题）
- 代数证明专题（20题）

**推荐资源**：
- 《Understanding Analysis》
- 《Algebra》（Artin）
- 裴礼文中的证明题

### 阶段3：综合提升（持续）
**目标**：形成证明直觉，能独立完成复杂证明

**训练内容**：
- 研究级证明题
- 竞赛难度证明
- 证明写作训练
- 参与数学讨论班

**推荐资源**：
- Putnam竞赛题
- 研究级教材习题
- 学术论文证明

## 证明类型专项训练

### 类型1：存在性证明
**方法**：
- 构造法：直接给出满足条件的对象
- 非构造法：证明存在而不具体构造

**练习资源**：
- 《数学分析》中连续函数的存在性
- 《代数》中群的存在性
- Brouwer不动点定理相关练习

### 类型2：唯一性证明
**方法**：
- 假设存在两个，证明它们相等

**练习资源**：
- 极限的唯一性
- 代数闭包的唯一性
- 标准形定理

### 类型3：等价性证明
**方法**：
- 循环证明：A⇒B⇒C⇒A
- 两两证明：A⇔B, A⇔C

**练习资源**：
- 紧性的等价刻画
- 可积性的等价条件
- 矩阵可逆的等价条件

### 类型4：反证法
**适用场景**：
- 命题以否定形式出现
- 直接证明困难
- 有限 vs 无限

**经典练习**：
- √2的无理性
- 素数无限
- 实数不可数

## 证明写作检查清单

完成证明后，检查：

### 内容层面
- [ ] 假设和结论明确
- [ ] 每一步都有依据
- [ ] 没有循环论证
- [ ] 特殊情况已考虑
- [ ] 结论正确得出

### 形式层面
- [ ] 符号使用规范
- [ ] 逻辑连接词正确
- [ ] 段落结构清晰
- [ ] 数学表达式准确
- [ ] 证明完整结束（□或QED）

## 常见证明错误分析

| 错误类型 | 示例 | 纠正 |
|---------|------|------|
| 循环论证 | 用结论证明中间步骤 | 检查每一步的前提 |
| 过度推广 | 从特例推广到一般 | 确认条件是否充分 |
| 符号混乱 | 量词顺序错误 | 练习∀∃的规范使用 |
| 偷换概念 | 改变定义使用 | 严格对照定义 |
| 忽略条件 | 遗漏定理的前提 | 列出定理所有条件 |
| 跳跃推导 | 省略关键推理 | 补充中间步骤 |

## 讨论班参与指南

### 准备阶段
1. 提前阅读材料和问题
2. 尝试独立解决
3. 记录卡住的地方
4. 准备提问

### 参与阶段
1. 认真听讲解
2. 积极提问
3. 尝试白板证明
4. 记录关键思路

### 复习阶段
1. 整理笔记
2. 补充遗漏证明
3. 过几天重做
4. 与同学讨论
