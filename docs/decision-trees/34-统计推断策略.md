## 概述

本决策树为统计学中的推断问题提供系统化的策略选择，涵盖点估计、区间估计、假设检验、贝叶斯推断等核心方法。

## 核心推断策略

### 点估计

**矩估计（MOM）**：

```
步骤：
1. 计算总体矩 μ_k = E[X^k] （用参数表示）
2. 令 μ_k = M_k （样本矩）
3. 解方程组得参数估计

优点：简单易算
缺点：不一定最优，可能不在参数空间

例：正态分布
μ₁ = μ, μ₂ - μ₁² = σ²
μ̂ = X̄, σ̂² = (1/n)Σ(Xᵢ-X̄)²
```

**极大似然估计（MLE）**：

```
似然函数：
L(θ|x) = ∏ f(xᵢ;θ) （i.i.d.情形）

对数似然：
l(θ) = ln L(θ) = Σ ln f(xᵢ;θ)

求解：
∂l/∂θ = 0 （得分方程）
验证：Hessian矩阵负定（极大值）

不变性：
若 θ̂ 是 θ 的MLE，则 g(θ̂) 是 g(θ) 的MLE
```

**估计量评价标准**：

```
无偏性：E[θ̂] = θ
有效性：Var(θ̂) 最小
一致性：θ̂ →ᵖ θ （n→∞）
充分性：T(X) 包含 θ 的全部信息

Cramér-Rao下界：
Var(θ̂) ≥ 1/I(θ)
其中 I(θ) = E[(∂ln f/∂θ)²] 是Fisher信息
```

### 区间估计

**枢轴量法**：

```
枢轴量 G(X,θ)：
- 依赖数据和参数
- 分布已知且不依赖参数

构造步骤：
1. 找枢轴量 G(X,θ)
2. 确定 a,b 使 P(a<G<b)=1-α
3. 反解不等式得 θ 的范围

例：正态均值（方差已知）
G = (X̄-μ)/(σ/√n) ~ N(0,1)
置信区间：X̄ ± z_{α/2}·σ/√n
```

**大样本置信区间**：

```
基于MLE的渐近正态性：
√n(θ̂-θ) →ᵈ N(0, I(θ)⁻¹)

置信区间：
θ̂ ± z_{α/2} / √(nI(θ̂))

或利用观测Fisher信息：
θ̂ ± z_{α/2} / √(-∂²l/∂θ²|θ̂)
```

**Bootstrap方法**：

```
当理论分布复杂时：

步骤：
1. 从原始样本中有放回抽取B个bootstrap样本
2. 对每个样本计算估计量 θ̂*
3. 用 θ̂* 的分布近似 θ̂ 的分布
4. 构造置信区间

区间类型：
- 百分位法
- 偏差修正法
- Bootstrap-t
```

### 假设检验

**基本要素**：

```
原假设 H₀ vs 备择假设 H₁
检验统计量 T(X)
拒绝域 W = {T(X) > c}
显著性水平 α = P(T∈W | H₀)
p-value = P(T ≥ t_obs | H₀)

两类错误：
- 第一类：拒真，概率 α
- 第二类：取伪，概率 β
- 功效：1-β
```

**似然比检验（LRT）**：

```
似然比统计量：
Λ = sup_{θ∈Θ₀} L(θ) / sup_{θ∈Θ} L(θ)

或：
λ = -2 ln Λ

渐近分布（Wilks定理）：
λ →ᵈ χ²_k （k=dim(Θ)-dim(Θ₀)）

拒绝域：λ > χ²_{k,α}
```

**三大检验**：

```
似然比检验（LR）：
  -2lnΛ → χ²

Wald检验：
  W = (θ̂-θ₀)' [Var(θ̂)]⁻¹ (θ̂-θ₀) → χ²

Score检验（Rao）：
  S = U(θ₀)' I(θ₀)⁻¹ U(θ₀) → χ²
  其中 U(θ) = ∂l/∂θ 是得分函数

等价性：三者在一阶渐近等价
```

**检验选择策略**：

```
参数模型：
- 似然比检验（最常用）
- Wald检验（方便，用估计量）
- Score检验（不需估计备择）

非参数：
- χ²拟合优度检验
- Kolmogorov-Smirnov检验
- 秩检验（Wilcoxon等）
```

### 贝叶斯推断

**核心公式**：

```
先验：π(θ)
似然：f(x|θ)
后验：π(θ|x) = f(x|θ)π(θ) / ∫f(x|θ)π(θ)dθ
     ∝ 似然 × 先验
```

**先验选择**：

```
共轭先验：
后验与先验同分布族

例：
- 正态-正态
- 二项- Beta
- Poisson-Gamma

无信息先验：
- 均匀先验
- Jeffreys先验：π(θ) ∝ √I(θ)
- 参考先验
```

**贝叶斯估计**：

```
后验均值：E[θ|x]
后验中位数：median(π(θ|x))
后验众数（MAP）：argmax π(θ|x)

贝叶斯风险：
最小化后验期望损失
```

**可信区间**：

```
HPD（最高后验密度）区间：
{θ: π(θ|x) ≥ c}
其中 c 满足 P(θ∈区间|x)=1-α

等尾区间：
分位数 α/2 和 1-α/2
```

**贝叶斯预测**：

```
后验预测分布：
f(x̃|x) = ∫ f(x̃|θ) π(θ|x) dθ

预测区间：
从 f(x̃|x) 构造
```

### 模型选择

**信息准则**：

```
AIC（Akaike）：
AIC = -2l(θ̂) + 2k
选最小值

AICc（小样本修正）：
AICc = AIC + 2k(k+1)/(n-k-1)

BIC（Bayesian）：
BIC = -2l(θ̂) + k ln n

比较：
- AIC：预测最优
- BIC：模型一致选择
```

**交叉验证**：

```
K折交叉验证：
1. 数据分为K份
2. K-1份训练，1份验证
3. 轮换验证集
4. 平均验证误差

留一法：K=n
```

## 推断流程模板

### 完整统计推断

```
给定数据和模型：

步骤1：探索性分析
- 描述统计
- 可视化
- 初步假设

步骤2：点估计
- 选择方法（矩/MLE/贝叶斯）
- 计算估计值
- 评估性质（无偏、一致等）

步骤3：区间估计
- 选择方法（枢轴量/渐近/Bootstrap）
- 构建置信/可信区间
- 解释区间含义

步骤4：假设检验（如需要）
- 设定H₀和H₁
- 选择检验统计量
- 计算p-value
- 做出决策

步骤5：模型诊断
- 残差分析
- 模型假设检验
- 敏感性分析

步骤6：报告结果
- 点估计 ± 标准误
- 置信区间
- p-value和实际显著性
```

## 常见问题解决策略

| 问题 | 策略 |
|------|------|
| 参数估计 | MLE或矩估计 |
| 构建置信区间 | 枢轴量或渐近方法 |
| 假设检验 | 似然比或Wald检验 |
| 模型比较 | AIC/BIC或交叉验证 |
| 小样本 | 精确方法或Bootstrap |
| 复杂模型 | 贝叶斯MCMC |
| 检验功效 | 计算功效函数 |
| 多重检验 | Bonferroni或FDR校正 |
