# decision-trees

本目录收录了 FormalMath 项目的结构化决策树与智能学习路径导航文档，共计近 60 份。决策树以“问题—条件—分支—建议”的逻辑链组织，帮助学习者在面对复杂的数学知识网络时快速定位起点、选择学习路线或确定解题策略。内容涵盖分析学、代数学、几何学、应用数学方向的入门路径决策，以及极限求解、高级问题求解等具体方法论决策，是项目自适应学习系统的重要组成部分。
