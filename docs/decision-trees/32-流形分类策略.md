## 核心分类策略

### 低维流形分类

**一维流形**：

```

连通1维流形（无边界）：
- 紧：S¹（圆周）
- 非紧：ℝ

带边界：
- 紧：[0,1]（闭区间）
- 非紧：[0,∞) 或 ℝ

```

**二维流形（曲面）分类**：

```

闭可定向曲面：
- 亏格 g = 0：S²（球面）
- 亏格 g = 1：T²（环面）
- 亏格 g ≥ 2：gT² = T² # ... # T²（连通和）

闭不可定向曲面：
- 亏格 k：kℝP² = ℝP² # ... # ℝP²

带边界曲面：
由闭曲面挖去圆盘得到
边界为圆周的不交并

```

**曲面不变量**：

```

Euler示性数：
χ = 2 - 2g （可定向）
χ = 2 - k   （不可定向）

基本群：
π₁(gT²) = ⟨a₁,b₁,...,a_g,b_g | Π[aᵢ,bᵢ]=1⟩
π₁(kℝP²) = ⟨a₁,...,a_k | a₁²...a_k²=1⟩

万有覆盖：
- g=0：S²
- g=1：ℝ²
- g≥2：双曲平面 ℍ²

```

**三维流形几何化**：

```

Thurston几何化猜想（Perelman证明）：
任何3维流形可分解为几何块的连通和

8种几何：
1. S³（球面几何）
2. ℝ³（欧氏几何）
3. ℍ³（双曲几何）
4. S² × ℝ
5. ℍ² × ℝ
6. SL(2,ℝ)̃
7. Nil
8. Sol

应用：
- Poincaré猜想（作为推论）
- 3维流形结构的深入理解

```

### 四维流形

**拓扑分类（Freedman）**：

```

单连通闭4维流形：
由相交形式 Q_X: H²(X;ℤ) × H²(X;ℤ) → ℤ 拓扑分类

相交形式性质：
- 对称双线性形式
- 幺模（det=±1）

Freedman定理：
- 奇形式：唯一实现
- 偶形式：由signature和Rokhlin不变量决定

```

**光滑分类（Donaldson）**：

```

光滑结构复杂得多：

- ℝ⁴：存在不可数多个互不平滑微分同胚的结构
- 紧流形：有限个光滑结构

Donaldson不变量：
从Yang-Mills方程定义的不变量
区分不同光滑结构

Seiberg-Witten不变量：
更易于计算的替代不变量

```

### 配边理论

**配边定义**：

```

Mⁿ 和 Nⁿ 配边：
存在紧 (n+1) 维流形 W 使得 ∂W = M ⊔ N

配边类构成环（不交并和笛卡尔积）

```

**Thom配边理论**：

```

配边群：
- MOₙ：非定向配边
- MSOₙ：定向配边
- MUₙ：复配边

Thom谱：
MO, MSO, MU

同伦解释：
MOₙ ≅ πₙ^S (MO) （稳定同伦群）

```

**配边不变量**：

```

Stiefel-Whitney数：
w_I[M] = ⟨w_{i₁}∪...∪w_{iₖ}, [M]⟩
决定非定向配边类

Pontryagin数：
p_I[M] = ⟨p_{i₁}∪...∪p_{iₖ}, [M]⟩
决定定向配边类（配合边公式）

符号差：
σ(M) = signature(Q_M)
配边不变量

```

### 示性类

**Stiefel-Whitney类**：

```

wᵢ ∈ Hⁱ(M; ℤ/2)

性质：
- w₀ = 1
- w(M⊕N) = w(M)∪w(N)
- w(TM) 全Stiefel-Whitney类

应用：
- w₁ = 0 ⇔ M 可定向
- w₁ = w₂ = 0 ⇔ M 有spin结构
- 浸入/嵌入的阻碍

```

**Pontryagin类**：

```

pᵢ ∈ H⁴ⁱ(M; ℤ)

定义：对实丛 E，复化后取Chern类
pᵢ(E) = (-1)ⁱ c₂ᵢ(E⊗ℂ)

应用：
- Hirzebruch符号差公式
- 配边理论
- 组合流形的阻碍

```

**Euler类**：

```

e ∈ Hⁿ(M; ℤ) （对定向n维丛）

性质：
- e(ξ) = 0 若 ξ 有非零截面
- ⟨e(TM), [M]⟩ = χ(M) （Euler示性数）

应用：
- Poincaré-Hopf定理
- 向量场奇点

```

**Chern类（复丛）**：

```

cᵢ ∈ H²ⁱ(M; ℤ)

分裂原理：
复丛形式上分拆为线丛直和
c(E) = ∏(1+xᵢ)

全Chern类：
c(E) = 1 + c₁ + c₂ + ...
c(E⊕F) = c(E)c(F)

```

### 手术理论

**手术操作**：

```

index-k 手术：
切除 Sᵏ × Dⁿ⁻ᵏ
粘贴 Dᵏ⁺¹ × Sⁿ⁻ᵏ⁻¹

要求：Sᵏ 在 M 中有平凡法丛

效果：
- 改变同调
- 构造新流形

```

**h-配边定理**：

```

W 是 M 和 N 的 h-配边：
- ∂W = M ⊔ N
- 包含 M,W 和 N,W 都是同伦等价

Smale h-配边定理：
dim W ≥ 6 且单连通 ⇒ W ≅ M × [0,1]

推论：M 和 N 微分同胚

```

**Wall阻碍群**：

```

手术障碍位于 Wall 群 Lₙ(π)

取决于：
- 维数 n (mod 4)
- 基本群 π = π₁(M)

Lₙ(1)（单连通）：
L₄ₖ(1) = ℤ （符号差）
L₄ₖ₊₂(1) = ℤ/2 （Kervaire不变量）
L_{奇数}(1) = 0

```

## 分类流程模板

### 闭曲面分类

```

给定闭曲面 M：

步骤1：判断可定向性
- 计算 H₂(M) = ℤ 或 0
- 或检查 w₁(M) = 0?

步骤2：计算Euler示性数 χ(M)

步骤3：确定亏格
- 可定向：χ = 2 - 2g
- 不可定向：χ = 2 - k

步骤4：写出标准形式
- 可定向：gT²
- 不可定向：kℝP²

步骤5：验证不变量
- 基本群
- 同调群
- 万有覆盖

```

### 4维流形拓扑分类

```

给定单连通闭4维流形 X：

步骤1：计算相交形式 Q_X
- H²(X;ℤ) 的挠自由部分
- 双线性形式 (a,b) ↦ ⟨a∪b, [X]⟩

步骤2：判定形式类型
- 偶形式：Q(x,x) 恒偶
- 奇形式：存在 Q(x,x) 奇

步骤3：计算不变量
- signature(Q) = b₂⁺ - b₂⁻
- （偶形式）Rokhlin不变量 μ(Q) ∈ ℤ/16

步骤4：应用Freedman定理
- 奇形式：唯一拓扑流形
- 偶形式：(Q, μ) 决定拓扑流形

步骤5：（若需要）区分光滑结构
- 计算Donaldson或Seiberg-Witten不变量

```

## 常见问题解决策略

| 问题 | 策略 |
|------|------|
| 曲面分类 | Euler示性数+可定向性 |
| 判断同胚 | 计算完整不变量集 |

| 配边判定 | 示性数或配边群计算 |

| 手术构造 | 验证障碍群元素为零 |
| 3维流形 | 几何化分解 |
