## 积分中值定理体系

### 积分第一中值定理

**基本形式**：
```
f在[a,b]连续 ⇒ ∃ξ∈[a,b], ∫[a,b]f(x)dx = f(ξ)(b-a)
```

**几何意义**：曲边梯形面积等于以f(ξ)为高的矩形面积

**推广形式**（f可积）：
```
∫[a,b]f(x)dx = μ(b-a), 其中m ≤ μ ≤ M
m = inf f(x), M = sup f(x)
```

### 积分第一中值定理（推广）

**条件**：
- f在[a,b]连续
- g在[a,b]可积且不变号

**结论**：
```
∃ξ∈[a,b], ∫[a,b]f(x)g(x)dx = f(ξ)∫[a,b]g(x)dx
```

**特殊情形**：g(x)≡1时为基本形式

### 积分第二中值定理

**条件**：
- f在[a,b]单调
- g在[a,b]可积

**结论**：
```
∃ξ∈[a,b], ∫[a,b]f(x)g(x)dx = f(a)∫[a,ξ]g(x)dx + f(b)∫[ξ,b]g(x)dx
```

**Bonnet形式**（f非负单调递减）：
```
∃ξ∈[a,b], ∫[a,b]f(x)g(x)dx = f(a)∫[a,ξ]g(x)dx
```

## 积分不等式

### Cauchy-Schwarz不等式

```
(∫[a,b]f(x)g(x)dx)² ≤ ∫[a,b]f²(x)dx · ∫[a,b]g²(x)dx
```

**等号成立**：f与g线性相关

### Hölder不等式

```
∫|fg| ≤ (∫|f|^p)^(1/p) · (∫|g|^q)^(1/q), 1/p + 1/q = 1
```

**p=q=2时为Cauchy-Schwarz**

### Minkowski不等式

```
(∫|f+g|^p)^(1/p) ≤ (∫|f|^p)^(1/p) + (∫|g|^p)^(1/p)
```

## 应用场景

### 1. 积分估计

**利用第一中值**：
```
|∫[a,b]f(x)dx| ≤ M(b-a), M = max|f(x)|
```

**利用Cauchy-Schwarz**：
```
|∫fg| ≤ ||f||₂ · ||g||₂
```

### 2. 极限计算

**例**：求lim[n→∞]∫[0,1]x^n/(1+x)dx

**解**：由积分第一中值定理
```
∫[0,1]x^n/(1+x)dx = 1/(1+ξ)∫[0,1]x^ndx = 1/[(1+ξ)(n+1)] → 0
```

### 3. 证明积分等式

**例**：设f在[0,1]连续，∫[0,1]f(x)dx = 0，证明∃ξ, f(ξ)=f(ξ+1/2)

**解**：构造F(x)=f(x)-f(x+1/2)，应用中值定理

### 4. Riemann和与定积分

```
lim[n→∞](1/n)Σf(k/n) = ∫[0,1]f(x)dx
```

**应用**：将求和极限转化为积分

## 使用时机判断

| 问题特征 | 推荐定理 | 说明 |
|---------|---------|------|
| 单函数积分估计 | 第一中值 | f(ξ)(b-a) |
| f·g积分，g定号 | 第一中值推广 | f(ξ)∫g |
| f·g积分，g变号 | 第二中值 | 分部形式 |
| 积分乘积估计 | Cauchy-Schwarz | L²范数 |
| 一般p范数估计 | Hölder | L^p范数 |
| 含参积分极限 | 中值+控制收敛 | 一致收敛 |

## 与微分中值定理的联系

**Newton-Leibniz公式**：
```
∫[a,b]f'(x)dx = f(b) - f(a)
```

**积分-微分关系**：
```
F(x) = ∫[a,x]f(t)dt ⇒ F'(x) = f(x)
```

**Lagrange与积分**：
```
(f(b)-f(a))/(b-a) = f'(ξ) = (1/(b-a))∫[a,b]f'(x)dx
```

## 常见技巧

1. **拆分积分区间**：针对不同区间用不同估计
2. **变量替换**：简化被积函数
3. **分部积分**：产生递推或简化
4. **控制函数**：用简单函数控制复杂积分
