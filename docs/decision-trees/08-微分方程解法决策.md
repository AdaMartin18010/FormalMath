## ODE核心方法

### 一阶ODE

**可分离变量**：
```
dy/dx = f(x)g(y) → ∫dy/g(y) = ∫f(x)dx
```

**一阶线性方程**：
```
y' + p(x)y = q(x)
积分因子: μ = exp(∫p(x)dx)
解: y = (∫μq dx + C)/μ
```

**恰当方程**：
```
Mdx + Ndy = 0, ∂M/∂y = ∂N/∂x
解: ∫Mdx + ∫(N-∂/∂y∫Mdx)dy = C
```

**Bernoulli方程**：
```
y' + p(x)y = q(x)y^n
换元: v = y^(1-n) → 线性方程
```

### 高阶线性ODE

**常系数齐次**：
```
y^(n) + a_{n-1}y^(n-1) + ... + a_0y = 0
特征方程: r^n + a_{n-1}r^{n-1} + ... + a_0 = 0
```

| 特征根类型 | 对应解 |
|-----------|-------|
| 单实根 r | e^(rx) |
| k重实根 r | e^(rx), xe^(rx), ..., x^(k-1)e^(rx) |
| 单复根 α±iβ | e^(αx)cos(βx), e^(αx)sin(βx) |
| k重复根 | 乘以1, x, ..., x^(k-1) |

**非齐次方程**：
- 待定系数法：特殊形式右端项
- 常数变易法：通用方法

### Euler方程

```
x^n y^(n) + ... + a_0 y = 0
换元: x = e^t → 常系数方程
```

## PDE核心方法

### 分离变量法

**基本步骤**：
1. 设 u(x,t) = X(x)T(t)
2. 代入PDE，分离变量得常数
3. 解ODE特征值问题
4. 叠加得级数解
5. 用初始条件定系数

**适用**：线性PDE + 规则区域

### 积分变换法

**Fourier变换**：
- 适用：初值问题，全空间
- 优点：降维，化PDE为ODE

**Laplace变换**：
- 适用：初边值问题，t≥0
- 优点：处理初始条件自然

### Green函数法

**思想**：点源解的叠加
```
Lu = f → u = ∫G(x,ξ)f(ξ)dξ
```

Green函数满足：
```
LG = δ(x-ξ) + 齐次边界条件
```

## 方程类型速查

| 方程形式 | 分类 | 解法 |
|---------|------|------|
| y'=f(x)g(y) | 可分离 | 分离变量 |
| y'=f(y/x) | 齐次 | 换元u=y/x |
| y'+p(x)y=q(x) | 一阶线性 | 积分因子 |
| Pdx+Qdy=0, ∂P/∂y=∂Q/∂x | 恰当 | 求原函数 |
| y^(n)+...+a₀y=0 | 常系数线性 | 特征方程 |
| u_tt=a²u_xx | 波动方程 | d'Alembert |
| u_t=a²u_xx | 热方程 | 分离变量/Fourier |
| Δu=0 | Laplace方程 | Green函数 |
