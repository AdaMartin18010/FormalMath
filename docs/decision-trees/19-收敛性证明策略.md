# 收敛性证明策略决策树

## 概述

本决策树为分析学中各类收敛性证明提供系统化的策略选择指导，涵盖数列收敛、函数收敛、级数收敛和积分收敛等多种情形。

## 核心证明策略

### 数列收敛

**ε-N定义法**（最基础）：

```

要证 aₙ → L：
∀ε>0, 找到 N(ε)∈ℕ, 使得 n≥N 时 |aₙ-L|<ε

技巧：
1. 先对 |aₙ-L| 进行放缩

2. 解不等式求出 N 关于 ε 的表达式
3. 验证 n≥N 时不等式成立

```

**柯西准则**（不知极限值时）：

```

数列收敛 ⇔ 柯西列
∀ε>0, ∃N, m,n≥N → |aₘ-aₙ|<ε

适用：证明抽象数列收敛而不求极限

```

**单调有界定理**（实数完备性）：

```

单调递增 + 有上界 → 收敛
单调递减 + 有下界 → 收敛

适用：递推数列、单调函数生成的数列

```

### 函数收敛

**点态 vs 一致收敛**：

| 类型 | 定义 | N 依赖 |
|------|------|--------|
| 点态 | ∀x, ∀ε>0, ∃N, n≥N→|fₙ(x)-f(x)|<ε | N 依赖于 x |
| 一致 | ∀ε>0, ∃N, ∀x, n≥N→|fₙ(x)-f(x)|<ε | N 不依赖于 x |

**一致收敛判别**：
- **上确界范数**：‖fₙ-f‖∞ → 0
- **M判别法（Weierstrass）**：|fₙ(x)| ≤ Mₙ, ΣMₙ 收敛

- **Dini定理**：紧集 + 连续 + 单调 → 一致收敛

### 级数收敛

**正项级数判别法**：

1. **比较判别法**：
   - 0 ≤ aₙ ≤ bₙ, Σbₙ 收敛 → Σaₙ 收敛
   - aₙ ≥ bₙ ≥ 0, Σbₙ 发散 → Σaₙ 发散

2. **比值判别法（d'Alembert）**：
   ```

   lim |aₙ₊₁/aₙ| = ρ

   ρ < 1: 收敛；ρ > 1: 发散；ρ = 1: 不确定
   ```

3. **根值判别法（Cauchy）**：
   ```

   lim sup |aₙ|^(1/n) = ρ

   ρ < 1: 收敛；ρ > 1: 发散；ρ = 1: 不确定
   ```

4. **积分判别法**：
   ```

   f 单调递减非负，则 Σf(n) 与 ∫f(x)dx 同敛散
   ```

**任意项级数**：
- 先判绝对收敛（用正项级数判别法）
- 条件收敛：Leibniz判别法（交错级数）、Dirichlet判别法、Abel判别法

### 积分收敛

**反常积分判别**：
- **比较判别**：|f| ≤ g, ∫g 收敛 → ∫f 绝对收敛

- **p-积分比较**：∫₁^∞ 1/xᵖ dx: p>1 收敛；∫₀¹ 1/xᵖ dx: p<1 收敛

**含参积分一致收敛**：
- **Weierstrass M判别**：|f(x,t)| ≤ M(t), ∫M 收敛

- **Dirichlet判别**：积分有界 + 函数单调趋于0
- **Abel判别**：积分收敛 + 函数单调有界

### 随机收敛

| 收敛类型 | 记号 | 定义 |
|----------|------|------|
| 几乎必然 | Xₙ → X a.s. | Pr(lim Xₙ = X) = 1 |
| 依概率 | Xₙ →ᵖ X | Pr(|Xₙ-X|>ε) → 0 |
| L^p收敛 | Xₙ → X in L^p | E[|Xₙ-X|^p] → 0 |
| 依分布 | Xₙ ⇒ X | E[f(Xₙ)] → E[f(X)] |

**关系链**：

```

L^p 收敛 → 依概率收敛 → 依分布收敛
     ↓
几乎必然收敛（需子列）

```

## 证明模板

### 数列收敛证明模板

```

证明：设 ε>0 任意给定。

第一步：分析 |aₙ - L|
|aₙ - L| = ... [代数变形]

         ≤ ... [放缩]

第二步：求 N
令 ... < ε，解得 n > φ(ε)
取 N = ⌈φ(ε)⌉ + 1

第三步：验证
当 n ≥ N 时：

|aₙ - L| ≤ ... < ε

因此 aₙ → L (n→∞)。

```

### 一致收敛证明模板

```

证明：估计 |fₙ(x) - f(x)|

|fₙ(x) - f(x)| = ... 

               ≤ Mₙ （Mₙ 不依赖于 x）

且 Mₙ → 0 (n→∞)

因此 supₓ |fₙ(x)-f(x)| ≤ Mₙ → 0

故 fₙ 一致收敛于 f。

```

## 常见错误与注意

1. **混淆点态与一致收敛**：点态收敛的 N 依赖于 x，一致收敛的 N 对所有 x 一致
2. **忽略条件检验**：使用判别法前验证条件（如单调性、非负性）
3. **极限交换随意**：一致收敛是极限交换（极限、求和、积分、求导）的充分条件
4. **柯西准则误用**：柯西准则只能证明收敛，不能求出极限值

## 相关定理

- **Weierstrass M判别法**：函数项级数一致收敛
- **Lebesgue控制收敛定理**：积分与极限交换
- **Fubini定理**：重积分交换次序
- **Prokhorov定理**：依分布收敛的刻画
