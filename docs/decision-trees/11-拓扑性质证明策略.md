## 核心证明策略

### 开集与闭集

**开集证明**（定义法）：

```

要证A是开集：
∀x∈A, ∃ε>0, B(x,ε)⊂A

```

**闭集证明**（三种方法）：

1. **补集法**：
   ```

   证A是闭集 ⇔ 证A^c是开集
   ```

2. **极限点法**：
   ```

   A是闭集 ⇔ A包含其所有极限点
   ```

3. **序列法**（度量空间）：
   ```

   A是闭集 ⇔ A中收敛序列的极限在A中
   ```

### 连续性证明

**开集原像法**（最一般）：

```

f连续 ⇔ ∀开集V, f⁻¹(V)是开集

```

**ε-δ法**（度量空间）：

```

∀ε>0, ∃δ>0, d(x,x₀)<δ → d(f(x),f(x₀))<ε

```

**邻域法**：

```

对f(x₀)的任意邻域V，存在x₀的邻域U使得f(U)⊂V

```

**序列法**（度量空间）：

```

x_n→x₀ ⇒ f(x_n)→f(x₀)

```

### 紧致性证明

**开覆盖定义**：

```

K紧致 ⇔ K的任意开覆盖有有限子覆盖

```

**常用技巧**：

1. **序列紧致性**（度量空间等价）：
   ```

   K紧致 ⇔ K中任意序列有收敛子列且极限在K中
   ```

2. **有限交性质**：
   ```

   闭集族有有限交性质 → 全交非空
   ```

3. **连续映射保持**：
   ```

   f连续，K紧致 ⇒ f(K)紧致
   ```

4. **乘积紧致**（Tychonoff）：
   ```

   X,Y紧致 ⇒ X×Y紧致
   ```

### 连通性证明

**连通性**（定义法）：

```

X连通 ⇔ X不能写成两个非空不交开集的并

```

**道路连通性**（更强的性质）：

```

∀x,y∈X, ∃连续道路γ:[0,1]→X, γ(0)=x, γ(1)=y

```

**重要定理**：
- 道路连通 ⇒ 连通（逆不成立）
- 连通集的连续像连通
- ℝ的子集连通 ⇔ 是区间

```

   ↓                    ↓

   ↓

```
