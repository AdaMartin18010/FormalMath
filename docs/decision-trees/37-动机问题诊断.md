## 动机类型分析

### 内在动机 vs 外在动机

| 内在动机 | 外在动机 |
|---------|---------|
| 对数学本身的好奇 | 考试成绩 |
| 解决问题的满足感 | 家长/老师的期望 |
| 理解深层结构的愉悦 | 奖学金/荣誉 |
| 智力挑战的兴奋 | 就业竞争力 |

**策略**：培养内在动机为主，合理利用外在动机

### 动机水平自评

1-5分评估：
- 我期待学习数学的时间
- 数学让我感到兴奋
- 我愿意为解决难题付出额外努力
- 我对数学有持久的兴趣
- 我主动寻找额外的数学内容

**总分**：
- 20-25：动机很强
- 15-19：动机中等，需要维护
- 10-14：动机较弱，需要干预
- 5-9：动机严重不足，需要深度分析

## 动机提升策略

### 策略1：目标设定

**SMART目标**：
- **S**pecific：具体（如"掌握链式法则"）
- **M**easurable：可测量（如"能解10道题"）
- **A**chievable：可实现
- **R**elevant：相关
- **T**ime-bound：有时限（如"本周内"）

**长期-短期目标连接**：
```
长期目标（5年）：成为数学家
    ↓
中期目标（1年）：完成硕士课程
    ↓
短期目标（1月）：通过期末考试
    ↓
今日任务：完成习题5.3
```

### 策略2：意义建构

**找到数学对你的意义**：
- 智力探索的乐趣
- 解决问题的工具
- 职业发展的基础
- 思维训练的方式
- 理解世界的语言

**应用连接**：
- 了解数学在AI/金融/物理中的应用
- 看数学如何解释自然现象
- 体验数学之美（对称、简洁）

### 策略3：环境设计

**物理环境**：
- 整洁的桌面
- 良好的光线
- 远离干扰源

**社会环境**：
- 寻找学习伙伴
- 加入学习小组
- 找到榜样/导师

**数字环境**：
- 使用专注APP
- 屏蔽社交媒体
- 使用学习追踪工具

### 策略4：习惯养成

**微习惯策略**：
- 从每天1道题开始
- 固定时间学习
- 建立仪式感

**习惯链条**：
```
已有习惯 → 新习惯
吃早餐 → 看1个数学视频
晚饭后 → 做30分钟习题
```

### 策略5：奖励机制

**即时奖励**：
- 完成目标后的小休息
- 喜欢的零食
- 看一集喜欢的剧

**延迟奖励**：
- 周目标达成后的外出
- 月目标达成后的购买
- 里程碑的大奖励

## 倦怠应对

### 倦怠信号
- 持续的疲惫感
- 对数学产生厌恶
- 效率明显下降
- 睡眠/食欲变化

### 应对策略
1. **短期休息**：1-2天完全不碰数学
2. **降低强度**：减少学习时间和任务量
3. **改变方式**：尝试不同的学习方法
4. **寻求支持**：与朋友、导师交流
5. **专业帮助**：严重时寻求心理咨询

### 恢复计划
```
Week 1: 休息+轻度活动
Week 2: 恢复性学习（感兴趣的轻松内容）
Week 3: 渐进增加强度
Week 4: 回到正常节奏
```

## 自我对话调整

### 消极自我对话 → 积极自我对话

| 消极 | 积极 |
|-----|------|
| "我数学不好" | "我正在进步" |
| "这太难了" | "这是一个挑战" |
| "我永远学不会" | "我需要更多时间" |
| "别人都比我强" | "我有自己的节奏" |
| "我肯定要失败" | "我会尽力而为" |

### 成长型思维培养
- 能力是可以发展的
- 困难是成长的机会
- 努力比天赋更重要
- 从错误中学习
