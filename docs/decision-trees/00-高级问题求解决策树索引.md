## 使用建议

### 1. 问题求解流程

```
1. 识别问题所属数学领域
2. 确定问题类型（证明/计算/分类/推断）
3. 查阅对应决策树
4. 按决策树分支逐步分析
5. 选择合适策略实施
6. 验证结果
```

### 2. 交叉引用

复杂问题可能需要多个决策树：

- **分析+代数**：收敛性证明可能涉及线性代数的特征值分析
- **几何+代数**：流形分类涉及同调计算（代数拓扑）
- **概率+分析**：随机过程分析需要渐近分析技巧

### 3. 学习路径建议

**分析学进阶**：

```
19(收敛性) → 20(积分) → 21(不等式) → 22(函数分析) → 23(渐近)
```

**代数学进阶**：

```
24(群论) → 25(环论) → 26(域论) → 27(线性代数) → 28(表示论)
```

**几何拓扑进阶**：

```
11(拓扑) → 29(同调) → 30(曲率) → 31(构造) → 32(分类)
```

**概率统计进阶**：

```
33(概率计算) → 13(检验选择) → 34(统计推断) → 35(随机过程)
```

| 颜色 | 含义 |
