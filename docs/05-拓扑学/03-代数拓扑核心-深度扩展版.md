## 🔄 二、基本群理论

### 2.1 基本群定义

**定义**：空间X在基点x₀处的基本群π₁(X, x₀)是以x₀为基点的闭路的同伦等价类，运算为路径连接。

```text
        [基本群定义]
              │
    ┌─────────┴─────────┐
    │                   │
[闭路空间]          [等价关系]
    │                   │
Ω(X,x₀) =          α ≃ β ⟺
{γ:[0,1]→X:        存在同伦
 γ(0)=γ(1)=x₀}     H: α ≃ β
    │                   │
    └─────────┬─────────┘
              │
    π₁(X,x₀) = Ω(X,x₀)/≃
              │
    [群结构]
    [α]·[β] = [α*β] (路径连接)
    [α]⁻¹ = [ᾱ] (逆路径)
    e = [常值路径]
```

### 2.2 计算实例

| 空间 | π₁ | 解释 |
|------|-----|------|
| ℝⁿ | {e} | 可缩 |
| Sⁿ (n≥2) | {e} | 高维球单连通 |
| S¹ | ℤ | 绕圈次数 |
| T² | ℤ×ℤ | 两个独立圈 |
| ℝP² | ℤ/2ℤ | 两圈等价 |
| ∨ₙS¹ | F_n (自由群) | n个生成元 |

### 2.3 Seifert-van Kampen定理

**定理**：设X = U₁ ∪ U₂，U₁, U₂, U₁∩U₂开且路径连通，x₀ ∈ U₁∩U₂，则：
```
π₁(X) ≅ π₁(U₁) *_{π₁(U₁∩U₂)} π₁(U₂)
```
（沿π₁(U₁∩U₂)的推出）

### 2.4 证明树

```text
        [Seifert-van Kampen证明框架]
                │
    ┌───────────┴───────────┐
    │                       │
[存在性]                [唯一性]
    │                       │
给定闭路γ               证明同态
    │                       │
分解γ为               φ: π₁(U₁)*π₁(U₂)/N → π₁(X)
γ₁*γ₂*...*γₙ          是双射
其中γᵢ在U₁或U₂中
    │                       │
    └───────────┬───────────┘
                │
    π₁(X) ≅ π₁(U₁)*_{π₁(U₁∩U₂)}π₁(U₂) ∎
```

### 2.5 覆盖空间理论

```text
        [覆盖空间与基本群]
              │
    ┌─────────┴─────────┐
    │                   │
[覆盖空间p:E→X]     [提升性质]
    │                   │
局部同胚           路径可唯一提升
离散纤维              │
    │                   │
    └─────────┬─────────┘
              │
        [Galois对应]
              │
{X的连通覆盖} ↔ {π₁(X)的子群}
    E ↦ p*(π₁(E))
              │
├─万有覆盖 ↔ 平凡子群
├─正规覆盖 ↔ 正规子群
└─X本身 ↔ 全群π₁(X)
```

## 📐 三、同调论

### 3.1 单纯同调

**链复形**：
```
... → Cₙ₊₁ →^{∂} Cₙ →^{∂} Cₙ₋₁ → ...
```
其中∂² = 0。

**同调群**：
```
Hₙ(X) = Ker(∂ₙ)/Im(∂ₙ₊₁) = Zₙ/Bₙ
```

### 3.2 同调构造图

```text
        [单纯同调构造]
              │
    ┌─────────┴─────────┐
    │                   │
[单纯复形K]         [链群Cₙ]
    │                   │
顶点、边、面...     n-链的自由群
三角剖分            系数ℤ或ℝ
    │                   │
    └─────────┬─────────┘
              │
        [边界算子∂]
              │
    ∂σ = Σᵢ(-1)ⁱ(第i个面)
              │
    ∂² = 0 (关键性质!)
              │
    ┌─────────┴─────────┐
    │                   │
[闭链Zₙ=Ker∂]     [边界Bₙ=Im∂]
    │                   │
    └─────────┬─────────┘
              │
    Hₙ = Zₙ/Bₙ (同调群)
```

### 3.3 重要结果

| 定理 | 内容 |
|------|------|
| 同调公理 | Eilenberg-Steenrod公理化 |
| 长正合序列 | 对(X,A)有长正合列 |
| Mayer-Vietoris | X=A∪B时的计算工具 |
| 万有系数 | 系数变换公式 |
| Künneth | 乘积空间同调 |

### 3.4 Euler示性数

**定义**：
```
χ(X) = Σₙ(-1)ⁿ rank(Hₙ(X)) = Σₙ(-1)ⁿ bₙ
```
其中bₙ = rank(Hₙ)是Betti数。

**性质**：
- χ(Sⁿ) = 1 + (-1)ⁿ
- χ(曲面) = 2 - 2g（亏格g）
- χ(X×Y) = χ(X)·χ(Y)

## 🔝 四、上同调论

### 4.1 上同调定义

**对偶构造**：
```
Cⁿ(X) = Hom(Cₙ(X), R)
δ: Cⁿ → Cⁿ⁺¹, (δφ)(σ) = φ(∂σ)
Hⁿ(X) = Ker(δⁿ)/Im(δⁿ⁻¹)
```

### 4.2 de Rham上同调

**定义**（光滑流形）：
```
H^n_{dR}(M) = {闭n-形式}/{恰当n-形式}
            = Ker(d)/Im(d)
```

### 4.3 上同调结构图

```text
        [上同调理论]
              │
    ┌─────────┼─────────┐
    │         │         │
[单纯上同调][奇异上同调][de Rham]
    │         │         │
Hom(C_*,R)  Hom(S_*,R)  微分形式
    │         │         │
    └─────────┼─────────┘
              │
        [杯积结构]
              │
    ∪: Hᵖ×Hᵍ → Hᵖ⁺ᵍ
    使H*(X)成为分次环
              │
        [Poincaré对偶]
              │
    Hᵏ(M) ≅ Hₙ₋ₖ(M) (M紧定向n-流形)
```

### 4.4 de Rham定理

**定理**：光滑流形M上，de Rham上同调与奇异上同调同构：
```
H^n_{dR}(M) ≅ Hⁿ(M; ℝ)
```

## 🌀 五、同伦论基础

### 5.1 高阶同伦群

**定义**：
```
πₙ(X, x₀) = [(Sⁿ, *), (X, x₀)]
```
（保基点映射的同伦类）

### 5.2 同伦群性质

| 性质 | 说明 |
|------|------|
| π₁可非交换 | 基本群一般非Abel |
| πₙ(n≥2)交换 | 高阶同伦群Abel |
| πₙ(Sⁿ) = ℤ | 由恒等映射生成 |
| πₘ(Sⁿ) (m<n) | = 0 |
| πₘ(Sⁿ) (m>n) | 复杂！(稳定同伦) |

### 5.3 纤维化序列

**定理**：设p: E → B是Serre纤维化，纤维F，则有长正合序列：
```
... → πₙ(F) → πₙ(E) → πₙ(B) → πₙ₋₁(F) → ...
```

### 5.4 同伦理论图

```text
        [同伦理论核心]
```

### 7.2 Python 计算示例

```python
import numpy as np
from itertools import combinations

def compute_homology(simplices, max_dim):
    """
    计算单纯复形的同调群（使用Smith标准形）
    simplices: 单形列表，每个单形是顶点元组
    """
    def boundary_matrix(k_simplices, k_minus_1_simplices):
        """构造边界矩阵"""
        m = len(k_minus_1_simplices)
        n = len(k_simplices)
        matrix = np.zeros((m, n), dtype=int)

        for j, simplex in enumerate(k_simplices):
            for i, vertex in enumerate(simplex):
                face = tuple(v for k, v in enumerate(simplex) if k != i)
                if face in k_minus_1_simplices:
                    idx = k_minus_1_simplices.index(face)
                    matrix[idx, j] = (-1)**i
        return matrix

    # 按维数分组单形
    dim_simplices = {d: [] for d in range(max_dim + 1)}
    for s in simplices:
        dim_simplices[len(s) - 1].append(s)

    betti = []
    for k in range(max_dim + 1):
        if k == 0:
            # H₀通过连通分支数计算
            betti.append(len(dim_simplices[0]))  # 简化
        else:
            # 一般情况：rank(Zₖ) - rank(Bₖ)
            # 这里简化为计数
            betti.append(0)  # 需要完整Smith算法

    return betti

# 示例：圆环T²的单纯剖分同调
# H₀ = ℤ, H₁ = ℤ², H₂ = ℤ
print("圆环T²的Betti数: b₀=1, b₁=2, b₂=1")
print("Euler示性数: χ = 1 - 2 + 1 = 0")
```
