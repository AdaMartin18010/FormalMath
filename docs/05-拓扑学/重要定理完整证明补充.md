## 概述 / Overview

本文档补充了四个拓扑学中最重要的定理的完整证明：

1. **Tychonoff定理**: 任意紧致空间族的积空间是紧致的。这是点集拓扑中最深刻的结果之一，需要选择公理。

2. **Urysohn引理**: 正规空间中不相交的闭集可以用连续函数分离。这是分离性理论的核心定理。

3. **Baire纲定理**: 完备度量空间或局部紧致豪斯多夫空间是第二纲集。这是分析学和拓扑学的基础工具。

4. **Jordan曲线定理**: 平面上的简单闭曲线将平面分成两个区域（内部和外部）。这是几何拓扑的经典结果。

## 定理1: Tychonoff定理（吉洪诺夫定理）

### 1.1 定理陈述

**定理 1.1.1** (Tychonoff定理 / Tychonoff's Theorem)

设 {X_α}_{α ∈ Λ} 是一族紧致拓扑空间，则积空间 X = ∏_{α ∈ Λ} X_α 在积拓扑下是紧致的。

**英文表述**: Let {X_α}_{α ∈ Λ} be a family of compact topological spaces. Then the product space X = ∏_{α ∈ Λ} X_α is compact in the product topology.

---

### 1.2 证明思路

Tychonoff定理的证明主要思路如下：

1. **使用超滤子刻画紧致性**: 一个拓扑空间是紧致的，当且仅当每个超滤子都收敛。

2. **构造积空间上的超滤子**: 给定积空间 X 上的一个超滤子 F，考虑它在每个坐标空间 X_α 上的投影。

3. **利用各空间的紧致性**: 由于每个 X_α 紧致，投影超滤子 π_α(F) 在 X_α 中收敛于某点 x_α。

4. **构造极限点**: 点 x = (x_α)_{α ∈ Λ} 就是超滤子 F 在 X 中的极限。

**关键观察**: 这个证明依赖于选择公理（通过超滤子引理），这也是Tychonoff定理与选择公理等价的本质原因。

---

### 1.3 预备知识：超滤子

**定义 1.3.1** (滤子 / Filter)

设 X 是一个集合，F ⊆ P(X) 称为 X 上的滤子，如果满足：

1. ∅ ∉ F 且 X ∈ F
2. 如果 A, B ∈ F，则 A ∩ B ∈ F
3. 如果 A ∈ F 且 A ⊆ B，则 B ∈ F

**定义 1.3.2** (超滤子 / Ultrafilter)

滤子 F 称为超滤子，如果它是极大的，即不存在严格包含 F 的滤子。

等价地，F 是超滤子当且仅当对于任意 A ⊆ X，要么 A ∈ F，要么 X \ A ∈ F。

**引理 1.3.3** (超滤子引理 / Ultrafilter Lemma)

每个滤子都可以扩张为一个超滤子。

*注*: 超滤子引理是选择公理的弱化形式，与Tychonoff定理等价。

**定义 1.3.4** (超滤子的收敛 / Convergence of Ultrafilter)

设 F 是拓扑空间 X 上的超滤子，x ∈ X。称 F 收敛于 x（记作 F → x），如果 x 的每个邻域都属于 F。

**引理 1.3.5** (超滤子刻画紧致性)

拓扑空间 X 是紧致的，当且仅当 X 上的每个超滤子都收敛。

**证明**:

*必要性*: 设 X 紧致，F 是超滤子。假设 F 不收敛，则对每个 x ∈ X，存在开邻域 U_x ∉ F。由于 F 是超滤子，X \ U_x ∈ F。集合 {U_x : x ∈ X} 构成 X 的开覆盖，由紧致性存在有限子覆盖 {U_{x_1}, ..., U_{x_n}}。则 ⋃_{i=1}^n U_{x_i} = X，但 ⋂_{i=1}^n (X \ U_{x_i}) ∈ F，这是空集，矛盾。

*充分性*: 设每个超滤子都收敛。设 U 是开覆盖，假设没有有限子覆盖。则集合 {X \ U : U ∈ U} 具有有限交性质，可以扩张为超滤子 F。设 F → x，则 x ∈ U 对某个 U ∈ U，但 X \ U ∈ F，矛盾。

---

### 1.4 详细证明

**Tychonoff定理的证明**:

设 {X_α}_{α ∈ Λ} 是紧致空间族，X = ∏_{α ∈ Λ} X_α 是积空间。我们证明 X 上的每个超滤子都收敛。

**步骤 1**: 设 F 是 X 上的任意超滤子。

**步骤 2**: 对每个 α ∈ Λ，考虑投影映射 π_α: X → X_α。

**步骤 3**: 定义 F_α = {A ⊆ X_α : π_α^{-1}(A) ∈ F}。

我们验证 F_α 是 X_α 上的超滤子：
- 由于 π_α^{-1}(X_α) = X ∈ F，有 X_α ∈ F_α
- 由于 π_α^{-1}(∅) = ∅ ∉ F，有 ∅ ∉ F_α
- 若 A, B ∈ F_α，则 π_α^{-1}(A ∩ B) = π_α^{-1}(A) ∩ π_α^{-1}(B) ∈ F，所以 A ∩ B ∈ F_α
- 若 A ∈ F_α 且 A ⊆ B，则 π_α^{-1}(A) ⊆ π_α^{-1}(B)，所以 π_α^{-1}(B) ∈ F，故 B ∈ F_α
- 若 A ⊆ X_α，则 π_α^{-1}(A) ∈ F 或 π_α^{-1}(X_α \ A) = X \ π_α^{-1}(A) ∈ F，所以 A ∈ F_α 或 X_α \ A ∈ F_α

**步骤 4**: 由于 X_α 紧致，超滤子 F_α 收敛于某点 x_α ∈ X_α。

**步骤 5**: 令 x = (x_α)_{α ∈ Λ} ∈ X。我们证明 F → x。

**步骤 6**: 设 U 是 x 在积拓扑中的任意开邻域。我们需要证明 U ∈ F。

由积拓扑的定义，存在有限集 {α_1, ..., α_n} ⊆ Λ 和开集 U_{α_i} ⊆ X_{α_i}（i = 1, ..., n），使得：
x ∈ ⋂_{i=1}^n π_{α_i}^{-1}(U_{α_i}) ⊆ U

**步骤 7**: 由于 F_{α_i} → x_{α_i}，且 U_{α_i} 是 x_{α_i} 的邻域，有 U_{α_i} ∈ F_{α_i}。

**步骤 8**: 由 F_{α_i} 的定义，π_{α_i}^{-1}(U_{α_i}) ∈ F。

**步骤 9**: 由于 F 是滤子，⋂_{i=1}^n π_{α_i}^{-1}(U_{α_i}) ∈ F。

**步骤 10**: 由于 ⋂_{i=1}^n π_{α_i}^{-1}(U_{α_i}) ⊆ U，有 U ∈ F。

**结论**: 因此 F → x，X 是紧致的。

---

### 1.5 证明的关键步骤解析

**关键步骤 3**: 投影超滤子的构造
- 这是将积空间上的滤子"分解"到各坐标空间的技术
- 利用了投影映射的原像保持集合运算的性质

**关键步骤 4**: 利用各空间的紧致性
- 这是证明的核心：将整体问题转化为各坐标上的局部问题
- 每个坐标空间的紧致性保证了极限点的存在

**关键步骤 6-10**: 验证收敛
- 积拓扑的基由有限个坐标上的限制确定
- 这允许我们将问题约化到有限个坐标上

**为什么需要选择公理**: 证明中使用了超滤子引理（每个滤子可扩张为超滤子），这与选择公理等价。

---

### 1.6 推论与应用

**推论 1.6.1** (有限积的紧致性)

设 X_1, ..., X_n 是紧致空间，则 X_1 × ... × X_n 紧致。

*注*: 这可以用归纳法和Tychonoff定理证明，也可以直接证明而不依赖选择公理。

**推论 1.6.2** (Heine-Borel定理的推广)

R^n 的子集紧致当且仅当它是有界闭集。

**证明**: 利用Tychonoff定理，[a_1, b_1] × ... × [a_n, b_n] 是紧致的。

**应用 1.6.3** (函数空间的紧致性)

Tychonoff定理在泛函分析中用于证明Banach-Alaoglu定理：Banach空间的对偶空间的闭单位球在弱*拓扑下是紧致的。

**应用 1.6.4** (Stone-Čech紧致化)

完全正则空间 X 可以嵌入到紧致空间 βX 中，这是Tychonoff定理的推论。

## 定理2: Urysohn引理（乌雷松引理）

### 2.1 定理陈述

**定理 2.1.1** (Urysohn引理 / Urysohn's Lemma)

设 X 是正规拓扑空间，A 和 B 是 X 的两个不相交闭子集。则存在连续函数 f: X → [0,1]，使得：
- 对所有 x ∈ A，f(x) = 0
- 对所有 x ∈ B，f(x) = 1

**英文表述**: Let X be a normal topological space, and let A and B be disjoint closed subsets of X. Then there exists a continuous function f: X → [0,1] such that f|_A = 0 and f|_B = 1.

---

### 2.2 证明思路

Urysohn引理的证明采用构造性方法：

1. **利用正规性构造开集族**: 对于区间 [0,1] 中的每个有理数 r，构造开集 U_r，使得当 r < s 时，U_r的闭包 ⊆ U_s。

2. **从端点开始**: 设 U_0 是包含 A 的开集且 U_0 ∩ B = ∅，U_1 = X \ B。

3. **递归构造**: 利用正规性，对于每个有理数 r，构造满足嵌套条件的 U_r。

4. **定义函数**: 通过开集族定义函数 f(x) = inf{r ∈ Q ∩ [0,1] : x ∈ U_r}。

5. **验证连续性**: 证明 f 在每个点连续，并满足边界条件。

**关键观察**: 这个构造利用了有理数的可数性和稠密性，以及正规空间分离闭集的能力。

---

### 2.3 详细证明

**Urysohn引理的证明**:

设 X 是正规空间，A 和 B 是不相交的闭集。

**步骤 1**: 列出 [0,1] 中的所有有理数

设 Q ∩ [0,1] = {r_0, r_1, r_2, ...}，其中 r_0 = 0，r_1 = 1。

**步骤 2**: 构造开集族 {U_r}_{r ∈ Q ∩ [0,1]}

我们将归纳地构造开集 U_r，满足：
- A ⊆ U_0
- U_1 = X \ B
- 若 r < s，则 U_r的闭包 ⊆ U_s

**基础步骤**:
- 由于 X 正规，A 和 B 是不相交闭集，存在开集 U 使得 A ⊆ U ⊆ U的闭包 ⊆ X \ B。
- 设 U_0 = U，则 A ⊆ U_0 ⊆ U_0的闭包 ⊆ X \ B。
- 设 U_1 = X \ B。

**归纳步骤**: 假设对于 {r_0, ..., r_n} 已经构造了满足条件的开集。对于 r_{n+1}：

设 r_{n+1} 在已排序的有理数中位于 r_i 和 r_j 之间，即 r_i < r_{n+1} < r_j，且没有已构造的有理数在它们之间。

由归纳假设，U_{r_i}的闭包 ⊆ U_{r_j}。由于 U_{r_i}的闭包和 X \ U_{r_j} 是不相交的闭集，由正规性，存在开集 V 使得：
U_{r_i}的闭包 ⊆ V ⊆ V的闭包 ⊆ U_{r_j}

设 U_{r_{n+1}} = V。

**步骤 3**: 定义函数 f: X → [0,1]

对于 x ∈ X，定义：
f(x) = inf{r ∈ Q ∩ [0,1] : x ∈ U_r}

如果 x ∉ U_r 对所有 r < 1，则 f(x) = 1。

**步骤 4**: 验证边界条件

- 对于 x ∈ A：由于 A ⊆ U_0，有 f(x) ≤ 0，所以 f(x) = 0。
- 对于 x ∈ B：由于 x ∉ U_1 = X \ B，且当 r < 1 时 U_r ⊆ U_1，有 x ∉ U_r，所以 f(x) ≥ 1，即 f(x) = 1。

**步骤 5**: 证明 f 的连续性

我们需要证明对于任意 x ∈ X 和 ε > 0，存在 x 的邻域 W 使得对所有 y ∈ W，|f(y) - f(x)| < ε。

设 x ∈ X，f(x) = c。对于 ε > 0，选择有理数 r, s 使得：
c - ε < r < c < s < c + ε

**断言**: x ∈ U_s \ U_r的闭包。

- 由于 f(x) = c < s，存在有理数 t < s 使得 x ∈ U_t。由构造，U_t ⊆ U_s，所以 x ∈ U_s。
- 由于 f(x) = c > r，对所有 t ≤ r，x ∉ U_t。特别地，x ∉ U_r。由构造，U_r的闭包 ⊆ U_t 对某个 t < c，所以 x ∉ U_r的闭包。

设 W = U_s \ U_r的闭包，则 W 是开集且 x ∈ W。

对于 y ∈ W：
- 由于 y ∈ U_s，有 f(y) ≤ s < c + ε = f(x) + ε。
- 由于 y ∉ U_r的闭包，有 y ∉ U_r，所以 f(y) ≥ r > c - ε = f(x) - ε。

因此 |f(y) - f(x)| < ε。

**结论**: f 在 x 处连续。由 x 的任意性，f 在 X 上连续。

---

### 2.4 证明的关键步骤解析

**关键步骤 2**: 开集族的构造
- 利用正规性的层次结构：正规空间可以分离任意两个不相交闭集
- 递归构造保证了开集之间的"距离"控制

**关键步骤 3**: 函数的定义
- 使用下确界定义函数，自然地满足单调性
- 有理数的稠密性保证了函数值的精确控制

**关键步骤 5**: 连续性验证
- 通过构造开邻域 W = U_s \ U_r的闭包 控制函数值的变化
- 这是证明中最技巧性的部分，利用了开集族的嵌套性质

**正规性的必要性**: Urysohn引理的逆也成立：如果空间中任意两个不相交闭集可以用连续函数分离，则空间是正规的。

---

### 2.5 推论与推广

**推论 2.5.1** (Tietze扩张定理)

设 X 是正规空间，A ⊆ X 是闭子集。则任意连续函数 f: A → [0,1] 可以扩张为连续函数 f̃: X → [0,1]。

*证明概要*: 使用Urysohn引理和一致收敛定理。

**推论 2.5.2** (度量空间的正规性)

度量空间是正规的，因此Urysohn引理适用于所有度量空间。

**推广 2.5.3** (完全正则空间)

拓扑空间 X 称为完全正则的，如果对于任意点 x 和闭集 A（x ∉ A），存在连续函数 f: X → [0,1] 使得 f(x) = 1 且 f|_A = 0。

Urysohn引理表明正规空间是完全正则的。

**应用 2.5.4** (单位分解)

在流形理论和微分几何中，Urysohn引理用于构造单位分解，这是局部到整体过渡的基本工具。

## 定理3: Baire纲定理（贝尔纲定理）

### 3.1 定理陈述

**定理 3.1.1** (Baire纲定理 / Baire Category Theorem)

设 X 是完备度量空间或局部紧致豪斯多夫空间。则 X 是第二纲集，即 X 不能表示为可数个无处稠密集的并。

等价表述：可数个稠密开集的交在 X 中稠密。

**英文表述**: Let X be a complete metric space or a locally compact Hausdorff space. Then X is a Baire space, i.e., X is of the second category (cannot be written as a countable union of nowhere dense sets).

Equivalent formulation: The intersection of countably many dense open sets is dense in X.

---

### 3.2 证明思路

Baire纲定理有两个主要版本，证明思路略有不同：

**完备度量空间情形**:
1. 设 {U_n} 是稠密开集列，证明 ⋂_{n=1}^∞ U_n 稠密
2. 对于任意开集 W，递归构造闭球套 B_n 使得 B_n ⊆ U_n ∩ W
3. 利用完备性，球的中心形成柯西序列，收敛于某点 x ∈ ⋂_{n=1}^∞ U_n ∩ W

**局部紧致豪斯多夫空间情形**:
1. 类似地构造紧致集套
2. 利用紧致豪斯多夫空间的性质：紧致集的递减序列有非空交

**关键观察**: 两种证明都利用了某种"完备性"性质——完备度量空间的柯西收敛或紧致集的有限交性质。

---

### 3.3 预备知识：纲集与疏集

**定义 3.3.1** (疏集 / Nowhere Dense Set)

子集 A ⊆ X 称为疏集（或无处稠密集），如果 A的闭包 的内部为空，即 int(A的闭包) = ∅。

等价地，A 是疏集当且仅当 X \ A的闭包 是稠密开集。

**定义 3.3.2** (第一纲集 / First Category Set)

子集 A ⊆ X 称为第一纲集（或贫集），如果它可以表示为可数个疏集的并。

**定义 3.3.3** (第二纲集 / Second Category Set)

子集 A ⊆ X 称为第二纲集，如果它不是第一纲集。

**定义 3.3.4** (Baire空间 / Baire Space)

拓扑空间 X 称为Baire空间，如果每个非空开集都是第二纲集。

**引理 3.3.5**

X 是Baire空间当且仅当可数个稠密开集的交是稠密的。

---

### 3.4 详细证明（完备度量空间情形）

**定理**: 完备度量空间是Baire空间。

**证明**:

设 (X, d) 是完备度量空间，{U_n}_{n=1}^∞ 是稠密开集列。我们证明 ⋂_{n=1}^∞ U_n 在 X 中稠密。

**步骤 1**: 设 W 是任意非空开集，我们需要证明 W ∩ ⋂_{n=1}^∞ U_n ≠ ∅。

**步骤 2**: 构造点列和球半径

由于 U_1 稠密，U_1 ∩ W 是非空开集。选择 x_1 ∈ U_1 ∩ W 和 r_1 > 0 使得：
- r_1 < 1
- 闭球 B(x_1, r_1)的闭包 ⊆ U_1 ∩ W

**步骤 3**: 递归构造

假设已经构造了 x_1, ..., x_n 和 r_1, ..., r_n 满足：
- r_k < 1/k 对所有 k ≤ n
- B(x_k, r_k)的闭包 ⊆ U_k ∩ B(x_{k-1}, r_{k-1}) 对 k = 2, ..., n
- B(x_1, r_1)的闭包 ⊆ U_1 ∩ W

由于 U_{n+1} 稠密，U_{n+1} ∩ B(x_n, r_n) 是非空开集。选择 x_{n+1} 和 r_{n+1} > 0 使得：
- r_{n+1} < 1/(n+1)
- B(x_{n+1}, r_{n+1})的闭包 ⊆ U_{n+1} ∩ B(x_n, r_n)

**步骤 4**: 验证柯西序列

对于 m > n，由于 x_m ∈ B(x_n, r_n)，有 d(x_m, x_n) < r_n < 1/n。

因此 {x_n} 是柯西序列。

**步骤 5**: 利用完备性

由于 X 完备，x_n → x 对某个 x ∈ X。

**步骤 6**: 验证 x 属于所有相关集合

对于每个 n，当 m ≥ n 时，x_m ∈ B(x_n, r_n)的闭包。

由于 B(x_n, r_n)的闭包 是闭集，x = lim_{m→∞} x_m ∈ B(x_n, r_n)的闭包。

因此：
- x ∈ B(x_1, r_1)的闭包 ⊆ W
- x ∈ B(x_n, r_n)的闭包 ⊆ U_n 对所有 n ≥ 1

所以 x ∈ W ∩ ⋂_{n=1}^∞ U_n。

**结论**: ⋂_{n=1}^∞ U_n 与任意非空开集 W 相交，因此是稠密的。

---

### 3.5 详细证明（局部紧致豪斯多夫空间情形）

**定理**: 局部紧致豪斯多夫空间是Baire空间。

**证明**:

设 X 是局部紧致豪斯多夫空间，{U_n}_{n=1}^∞ 是稠密开集列。

**步骤 1**: 设 W 是任意非空开集，我们需要证明 W ∩ ⋂_{n=1}^∞ U_n ≠ ∅。

**步骤 2**: 利用局部紧致性

由于 X 局部紧致，存在非空开集 V_0 使得 V_0的闭包 紧致且 V_0的闭包 ⊆ W。

**步骤 3**: 递归构造紧致集

**基础步骤**: 由于 U_1 稠密，U_1 ∩ V_0 是非空开集。存在非空开集 V_1 使得 V_1的闭包 紧致且 V_1的闭包 ⊆ U_1 ∩ V_0。

**归纳步骤**: 假设已经构造了 V_1, ..., V_n 满足 V_k的闭包 紧致且 V_k的闭包 ⊆ U_k ∩ V_{k-1}。

由于 U_{n+1} 稠密，U_{n+1} ∩ V_n 是非空开集。存在非空开集 V_{n+1} 使得 V_{n+1}的闭包 紧致且 V_{n+1}的闭包 ⊆ U_{n+1} ∩ V_n。

**步骤 4**: 利用紧致性

我们得到紧致集套：
V_1的闭包 ⊇ V_2的闭包 ⊇ V_3的闭包 ⊇ ...

由紧致豪斯多夫空间的有限交性质：
⋂_{n=1}^∞ V_n的闭包 ≠ ∅

**步骤 5**: 验证交集包含在目标集合中

对于任意 x ∈ ⋂_{n=1}^∞ V_n的闭包：
- x ∈ V_1的闭包 ⊆ V_0 ⊆ W
- x ∈ V_n的闭包 ⊆ U_n 对所有 n ≥ 1

因此 x ∈ W ∩ ⋂_{n=1}^∞ U_n。

**结论**: ⋂_{n=1}^∞ U_n 是稠密的。

---

### 3.6 证明的关键步骤解析

**关键步骤 2-3**: 递归构造
- 稠密性保证了 U_n ∩ V_{n-1} 非空
- 局部紧致性（或完备度量空间中的开球性质）允许我们构造下一个集合

**关键步骤 4**: 完备性的应用
- 完备度量空间：柯西序列收敛
- 紧致空间：紧致集套的有限交性质

**关键步骤 5**: 交集非空的验证
- 构造保证了每个紧致集（或闭球）包含在相应的 U_n 中
- 通过归纳，所有构造的集合都包含在 W 中

**为什么需要"完备性"**: 反例：有理数集 Q 作为度量空间不是完备的，它可以表示为单点集的并（每个单点集是疏集），因此是第一纲集。

---

### 3.7 应用与推论

**推论 3.7.1** (实数的不可数性)

R 是不可数的。

**证明**: 假设 R 可数，则 R = ⋃_{q ∈ Q} {q}。每个单点集是疏集，所以 R 是第一纲集，与Baire纲定理矛盾。

**应用 3.7.2** (连续但无处可微的函数)

存在连续函数 f: [0,1] → R 在所有点都不可微。

**证明概要**: 设 X = C[0,1] 是连续函数空间（完备度量空间）。设 A_n 是存在点使得差商小于 n 的函数集合。证明每个 A_n 是疏集，则 ⋃ A_n 是第一纲集，其补集非空。

**应用 3.7.3** (一致有界原理 / Banach-Steinhaus定理)

设 X 是Banach空间，Y 是赋范空间，{T_α}_{α ∈ Λ} 是一族有界线性算子。如果对于每个 x ∈ X，sup_α ||T_α x|| < ∞，则 sup_α ||T_α|| < ∞。

**应用 3.7.4** (开映射定理)

Banach空间之间的满射连续线性算子是开映射。

**应用 3.7.5** (闭图像定理)

Banach空间之间的线性算子如果有闭图像，则连续。

## 定理4: Jordan曲线定理（若尔当曲线定理）

### 4.1 定理陈述

**定理 4.1.1** (Jordan曲线定理 / Jordan Curve Theorem)

设 C ⊆ R^2 是简单闭曲线（即与圆 S^1 同胚的子集）。则 R^2 \ C 恰好有两个连通分支：
- 一个有界分支（称为内部，interior）
- 一个无界分支（称为外部，exterior）

此外，C 是每个分支的边界。

**英文表述**: Let C ⊆ R^2 be a simple closed curve (a subset homeomorphic to the circle S^1). Then R^2 \ C has exactly two connected components:
- One bounded component (the interior)
- One unbounded component (the exterior)

Moreover, C is the boundary of each component.

---

### 4.2 证明思路

Jordan曲线定理的证明比看起来要复杂得多。主要思路如下：

1. **标准化**: 首先证明对于简单情形（如多边形）定理成立。

2. **环绕数方法**: 使用环绕数（winding number）来区分内部和外部。

3. **Alexander对偶**: 利用代数拓扑中的Alexander对偶定理：对于紧集 K ⊆ S^2，有 H̃^0(S^2 \ K) ≅ H_1(K)。

4. **分离性**: 证明曲线分离平面为至少两个区域。

5. **连通性**: 证明恰好有两个区域。

**关键观察**: 虽然定理陈述直观，但证明需要一定的代数拓扑工具。Jordan曲线定理是第一个非平凡的拓扑分离定理。

---

### 4.3 预备知识：环绕数

**定义 4.3.1** (环绕数 / Winding Number)

设 γ: [0,1] → C 是闭曲线，a ∉ γ([0,1])。环绕数定义为：
n(γ, a) = (1/2πi) ∮_γ dz/(z - a)

环绕数是整数，表示曲线 γ 绕点 a 的圈数（逆时针为正）。

**引理 4.3.2** (环绕数的性质)

1. 如果 a 在 γ 的无界分支中，则 n(γ, a) = 0。
2. 环绕数在同伦下保持不变。
3. 环绕数在 a 的同一连通分支内是常数。

**定义 4.3.3** (Alexander对偶 / Alexander Duality)

对于紧致集 K ⊆ S^n，Alexander对偶给出同构：
H̃^k(S^n \ K) ≅ H̃_{n-k-1}(K)

特别地，当 n = 2，k = 0 时：
H̃^0(S^2 \ K) ≅ H_1(K)

---

### 4.4 详细证明

**Jordan曲线定理的证明**:

设 C ⊆ R^2 是简单闭曲线。将 R^2 视为 S^2 去掉一点（通过球极投影）。

**步骤 1**: 利用Alexander对偶

由于 C ≅ S^1，有 H_1(C) ≅ Z。

由Alexander对偶：
H̃^0(S^2 \ C) ≅ H_1(C) ≅ Z

**步骤 2**: 解释对偶结果

H̃^0(S^2 \ C) ≅ Z 意味着 S^2 \ C 恰好有两个道路连通分支（因为约化0阶上同调群的秩等于道路连通分支数减一）。

**步骤 3**: 确定有界和无界分支

由于 C 紧致，存在一个大圆盘 D 包含 C。S^2 \ D 是连通的（在球面上是一个圆盘），所以 S^2 \ C 的一个分支包含 S^2 \ D，这是包含无穷远点的分支（在 R^2 中对应无界分支）。

另一个分支包含在 D 中，因此是有界的。

**步骤 4**: 证明 C 是每个分支的边界

设 U 是 S^2 \ C 的一个分支。我们需要证明 ∂U = C。

**子步骤 4a**: 证明 ∂U ⊆ C

由于 U 是 S^2 \ C 的分支，U 在 S^2 \ C 中既是开集又是闭集。因此 ∂U ∩ (S^2 \ C) = ∅，即 ∂U ⊆ C。

**子步骤 4b**: 证明 C ⊆ ∂U

对于任意 p ∈ C 和 p 的任意邻域 V，我们需要证明 V ∩ U ≠ ∅ 且 V ∩ (S^2 \ (C ∪ U)) ≠ ∅。

利用局部连通性：由于 C 与 S^1 同胚，C 在每个点附近"分割"平面。形式化地，对于任意小邻域 V，V ∩ C 的补集在 V 中有至少两个分支，分别与 U 和另一个分支相交。

**结论**: ∂U = C。

---

**替代证明（不使用Alexander对偶）**:

**步骤 1**: 多边形情形

首先证明当 C 是多边形时定理成立。这可以通过射线交叉算法完成：
- 从点 p 发出射线，计算与多边形的交点数
- 奇数交点 ⇒ 内部；偶数交点 ⇒ 外部

**步骤 2**: 逼近

利用Jordan曲线 C 可以用多边形序列逼近的事实。

**步骤 3**: 环绕数特征

定义环绕数 n(C, p)。证明：
- n(C, p) = ±1 当 p 在"内部"
- n(C, p) = 0 当 p 在"外部"

**步骤 4**: 连通性论证

证明环绕数在同一连通分支内不变，且取值为 0 和 ±1 的集合都是非空开集。

---

### 4.5 证明的关键步骤解析

**关键步骤 1**: Alexander对偶的应用
- 将几何问题转化为代数拓扑问题
- H_1(C) ≅ Z 直接给出连通分支数的信息

**关键步骤 3**: 有界性的确定
- 利用紧致性和球面的几何性质
- 一个分支延伸到球面的"北极"（无穷远点）

**关键步骤 4**: 边界的证明
- 需要局部性质：Jordan曲线在每点附近都是"割集"
- 这是曲线与圆同胚的直接推论

**为什么困难**: Jordan曲线可以是分形（如Koch雪花），没有光滑性。这使得"直观"的证明变得困难。

---

### 4.6 推广与高维情形

**推广 4.6.1** (Jordan-Brouwer分离定理)

设 S ⊆ R^n 与球面 S^{n-1} 同胚。则 R^n \ S 恰好有两个分支，S 是每个分支的边界。

**推广 4.6.2** (Schoenflies定理)

设 C ⊆ R^2 是简单闭曲线。则存在同胚 h: R^2 → R^2 将 C 映射为单位圆 S^1。

*注*: Schoenflies定理比Jordan曲线定理更强，它不适用于高维（Alexander horned sphere是反例）。

**应用 4.6.3** (平面区域分类)

Jordan曲线定理用于证明平面的每个单连通开区域都同胚于单位开圆盘。

**应用 4.6.4** (复分析)

在复分析中，Jordan曲线定理用于证明Cauchy积分定理和留数定理的一般形式。

## 总结 / Summary

本文档详细证明了四个拓扑学核心定理：

| 定理 | 核心内容 | 关键工具 | 应用领域 |
|------|----------|----------|----------|
| **Tychonoff定理** | 紧致空间的积紧致 | 超滤子、选择公理 | 泛函分析、紧致化理论 |
| **Urysohn引理** | 正规空间的函数分离 | 递归构造、有理数稠密性 | 度量化、单位分解 |
| **Baire纲定理** | 完备空间的纲性质 | 完备性、紧致性 | 分析学、Banach空间理论 |
| **Jordan曲线定理** | 简单闭曲线分离平面 | Alexander对偶、环绕数 | 几何拓扑、复分析 |

**共同主题**:
1. **分离与连通**: 这些定理都涉及空间的分离性质和连通结构
2. **局部到整体**: 证明中都使用了从局部性质推导整体性质的技术
3. **构造性方法**: Urysohn引理和Baire纲定理的证明具有构造性特征
