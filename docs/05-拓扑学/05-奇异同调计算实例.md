## 5.5.1 引言 / Introduction

同调论（homology theory）是代数拓扑中研究拓扑空间"洞"的结构的系统工具。与基本群只能捕捉一维洞的信息不同，同调群$H_n(X)$可以描述空间中任意$n$维洞的形状和数量。奇异同调（singular homology）由Eilenberg和Steenrod在1945年公理化，它通过将空间中的几何单形（simplex）映射到空间中，构造出一个代数链复形（chain complex），再通过取闭链模边缘链的方式定义同调群。

奇异同调的优点在于它适用于所有拓扑空间，且具有极强的计算性。Hatcher（2002）《Algebraic Topology》第二章系统阐述了奇异同调的定义、同伦不变性、长正合序列、切除定理和Mayer-Vietoris序列，这些工具使我们能够计算出绝大多数经典空间的同调群。MIT 18.905课程在第二学期深入讲授了这些内容，而Bredon（1993）《Topology and Geometry》则提供了更具几何直觉的视角。

本文档的目标是通过丰富的计算实例，展示如何将这些一般性工具应用于具体空间：从可缩空间和球面，到实射影空间、复射影空间、环面以及克莱因瓶。每个计算都附有详细的推理过程，帮助读者掌握同调计算的核心技巧。

## 5.5.2 奇异链复形与同调群 / Singular Chain Complex and Homology Groups

### 奇异单形与链群 / Singular Simplices and Chain Groups

**定义 2.1**（标准$n$-单形）. $\\mathbb{R}^{n+1}$中的**标准$n$-单形**$\\Delta^n$定义为
\\[
\\Delta^n = \\{(t_0, t_1, \\ldots, t_n) \\in \\mathbb{R}^{n+1} : \\sum_{i=0}^n t_i = 1, \\ t_i \\geq 0\\}.
\\]

**定义 2.2**（奇异$n$-单形）. 拓扑空间$X$中的一个**奇异$n$-单形**（singular $n$-simplex）是指连续映射$\\sigma: \\Delta^n \\to X$。

**定义 2.3**（奇异链群）. $X$的**奇异$n$-链群**$C_n(X)$是由所有奇异$n$-单形生成的自由阿贝尔群。一个元素$\\sum_i n_i \\sigma_i \\in C_n(X)$称为一条**$n$-链**（$n$-chain），其中$n_i \\in \\mathbb{Z}$，$\\sigma_i$为奇异$n$-单形。

### 边缘算子与同调群 / Boundary Operator and Homology Groups

**定义 2.4**（边缘算子）. 边缘算子$\\partial_n: C_n(X) \\to C_{n-1}(X)$在生成元$\\sigma: \\Delta^n \\to X$上定义为
\\[
\\partial_n(\\sigma) = \\sum_{i=0}^n (-1)^i \\sigma|_{[v_0, \\ldots, \\widehat{v_i}, \\ldots, v_n]},

\\]
其中$[v_0, \\ldots, \\widehat{v_i}, \\ldots, v_n]$是$\\Delta^n$去掉第$i$个顶点后的面，$\\sigma|_{\\text{face}}$表示$\\sigma$在该面上的限制。

**定理 2.5**（$\\partial^2 = 0$）. 对任意$n$，$\\partial_{n-1} \\circ \\partial_n = 0$。

*证明思路*：对生成元$\\sigma$，$\\partial^2(\\sigma)$中每个$(n-2)$-维面出现两次，对应于先去掉顶点$v_i$再去掉$v_j$（$i < j$），以及先去掉$v_j$再去掉$v_i$。这两种方式的符号分别为$(-1)^{i+j-1}$和$(-1)^{i+j}$，恰好相反，因此抵消。$\\square$

**定义 2.6**（同调群）. 
- $Z_n(X) = \\ker(\\partial_n: C_n(X) \\to C_{n-1}(X))$称为**$n$-闭链群**（$n$-cycles）；
- $B_n(X) = \\mathrm{im}(\\partial_{n+1}: C_{n+1}(X) \\to C_n(X))$称为**$n$-边缘链群**（$n$-boundaries）；
- 由$\\partial^2 = 0$知$B_n(X) \\subset Z_n(X)$。

$X$的**第$n$阶奇异同调群**定义为商群
\\[
H_n(X) = Z_n(X) / B_n(X).
\\]

同调群中的元素$[z]$称为一个**同调类**（homology class）。若$[z_1] = [z_2]$，称$z_1$与$z_2$**同调的**（homologous），即$z_1 - z_2$是某条$(n+1)$-链的边缘。

### 约化同调 / Reduced Homology

**定义 2.7**（约化同调）. 对非空空间$X$，定义**增广映射**$\\epsilon: C_0(X) \\to \\mathbb{Z}$为$\\epsilon(\\sum_i n_i \\sigma_i) = \\sum_i n_i$。增广链复形为
\\[
\\cdots \\to C_2(X) \\xrightarrow{\\partial_2} C_1(X) \\xrightarrow{\\partial_1} C_0(X) \\xrightarrow{\\epsilon} \\mathbb{Z} \\to 0.
\\]
其同调群称为**约化同调群**$\\widetilde{H}_n(X)$。对$n \\geq 1$，$\\widetilde{H}_n(X) = H_n(X)$；而$\\widetilde{H}_0(X) = \\ker \\epsilon / \\mathrm{im} \\partial_1$度量了$X$的道路连通分支数减1。

## 5.5.3 同伦不变性与长正合序列 / Homotopy Invariance and Long Exact Sequences

### 同伦不变性 / Homotopy Invariance

**定理 3.1**（同伦不变性）. 若$f, g: X \\to Y$是同伦的连续映射，则它们诱导的同调群同态相等：$f_* = g_*: H_n(X) \\to H_n(Y)$对所有$n$成立。

*证明思路*（链同伦法）：设$H: X \\times [0, 1] \\to Y$为$f$到$g$的同伦。构造**棱柱算子**（prism operator）$P: C_n(X) \\to C_{n+1}(Y)$，将$X$中的奇异单形$\\sigma$对应到$Y$中由$H \\circ (\\sigma \\times \\mathrm{id})$定义的棱柱状奇异链。可以验证$\\partial P + P \\partial = g_\\sharp - f_\\sharp$，其中$f_\\sharp, g_\\sharp$为链映射。这表明$f_\\sharp$和$g_\\sharp$是**链同伦的**（chain homotopic），从而诱导相同的同调群同态。$\\square$

**推论 3.2**.
1. 若$X \\simeq Y$（同伦等价），则$H_n(X) \\cong H_n(Y)$对所有$n$成立；
2. 可缩空间（如$\\mathbb{R}^n$、$D^n$、单点空间）的同调群满足$\\widetilde{H}_n(X) = 0$对所有$n$成立。

### 空间对的长正合序列 / Long Exact Sequence of a Pair

**定义 3.3**（空间对）. 拓扑空间对$(X, A)$是指$X$为拓扑空间，$A \\subset X$为子空间。$X$相对于$A$的**相对链群**为$C_n(X, A) = C_n(X) / C_n(A)$。

由短正合序列$0 \\to C_n(A) \\to C_n(X) \\to C_n(X, A) \\to 0$诱导的长正合序列为
\\[
\\cdots \\to H_n(A) \\xrightarrow{i_*} H_n(X) \\xrightarrow{j_*} H_n(X, A) \\xrightarrow{\\partial} H_{n-1}(A) \\to \\cdots \\to H_0(X, A) \\to 0.
\\]

**定理 3.4**. 上述序列是正合的。

相对同调群$H_n(X, A)$度量了$X$中"在$A$的边界上边缘化"的$n$维洞。特别地，$H_n(X, A) \\cong \\widetilde{H}_n(X/A)$当$(X, A)$满足好的条件时（如CW对）。

## 5.5.4 Mayer-Vietoris序列与切除定理 / Mayer-Vietoris and Excision

### 切除定理 / Excision Theorem

**定理 4.1**（切除定理）. 设$Z \\subset A \\subset X$且$\\bar{Z} \\subset \\mathring{A}$（$Z$的闭包包含于$A$的内部）。则包含映射$(X \\setminus Z, A \\setminus Z) \\hookrightarrow (X, A)$诱导同构
\\[
H_n(X \\setminus Z, A \\setminus Z) \\cong H_n(X, A)
\\]
对所有$n$成立。

切除定理是同调论的公理化特征之一（Eilenberg-Steenrod公理）。它将大的相对同调群"切除"一个小区域后保持不变，为计算提供了极大的灵活性。

### Mayer-Vietoris序列 / Mayer-Vietoris Sequence

**定理 4.2**（Mayer-Vietoris序列）. 设$X = A \\cup B$，其中$A, B$为开集（或满足适当条件的子空间）。则存在以下长正合序列：
\\[
\\cdots \\to H_n(A \\cap B) \\xrightarrow{(i_*, j_*)} H_n(A) \\oplus H_n(B) \\xrightarrow{k_* - l_*} H_n(X) \\xrightarrow{\\partial} H_{n-1}(A \\cap B) \\to \\cdots
\\]
其中$i: A \\cap B \\hookrightarrow A$，$j: A \\cap B \\hookrightarrow B$，$k: A \\hookrightarrow X$，$l: B \\hookrightarrow X$为包含映射。

Mayer-Vietoris序列是计算复杂空间同调群的最常用工具，它是同调论中的"分而治之"策略：将$X$分解为已知空间的并，通过交集的信息拼出整体的同调。

## 5.5.5 经典空间的同调计算 / Homology of Classical Spaces

### 球面$S^n$ / Sphere $S^n$

**定理 5.1**. 
\\[
H_k(S^n) = \\begin{cases} \\mathbb{Z} & k = 0, n, \\ 0 & \\text{otherwise}. \\end{cases}
\\]

*证明思路*（归纳法+MV序列）：对$n = 0$，$S^0 = \\{\\pm 1\\}$有两个道路连通分支，故$H_0(S^0) = \\mathbb{Z}^2$，$H_k(S^0) = 0$对$k > 0$。

假设结论对$S^{n-1}$成立。将$S^n$写成上半球$U$和下半球$V$的并，则$U \\simeq V \\simeq \\{pt\\}$可缩，$U \\cap V \\simeq S^{n-1}$。MV序列给出：
\\[
\\cdots \\to H_k(S^{n-1}) \\to H_k(U) \\oplus H_k(V) \\to H_k(S^n) \\to H_{k-1}(S^{n-1}) \\to \\cdots
\\]
对$k \\geq 2$，$H_k(U) = H_k(V) = 0$，故$H_k(S^n) \\cong H_{k-1}(S^{n-1})$。由归纳假设，$H_k(S^n) = \\mathbb{Z}$当且仅当$k - 1 = n - 1$，即$k = n$。对$k = 1$，由约化MV序列可得$H_1(S^n) = 0$对$n \\geq 2$。$\\square$

这一结果完美诠释了同调群捕捉高维洞的能力：$S^n$恰好有一个$n$维洞。

### 实射影空间$\\mathbb{R}P^n$ / Real Projective Space

**定理 5.2**. $\\mathbb{R}P^n$的同调群为
\\[
H_k(\\mathbb{R}P^n; \\mathbb{Z}) = \\begin{cases} \\mathbb{Z} & k = 0, \\\mathbb{Z}/2\\mathbb{Z} & 0 < k < n, \\ k \\text{ 奇}, \\\mathbb{Z} & k = n, \\ n \\text{ 奇}, \\ 0 & \\text{otherwise}. \\end{cases}
\\]

*证明思路*：使用胞腔同调（将在后续文档中详细介绍）。$\\mathbb{R}P^n$有一个CW分解：一个0-胞腔、一个1-胞腔、一个2-胞腔、...、一个$n$-胞腔。胞腔边缘算子在奇数维为乘以2，在偶数维为0。因此同调群出现$\\mathbb{Z}/2\\mathbb{Z}$挠元。$\\square$

### 复射影空间$\\mathbb{C}P^n$ / Complex Projective Space

**定理 5.3**. 
\\[
H_k(\\mathbb{C}P^n; \\mathbb{Z}) = \\begin{cases} \\mathbb{Z} & k = 0, 2, 4, \\ldots, 2n, \\ 0 & \\text{otherwise}. \\end{cases}
\\]

*证明思路*：$\\mathbb{C}P^n$的CW分解只有一个胞腔在每个偶数维$0, 2, 4, \\ldots, 2n$。由于所有胞腔边缘算子为零，胞腔同调直接给出结果。$\\square$

### $n$维环面$T^n$ / $n$-Torus

**定理 5.4**. 
\\[
H_k(T^n; \\mathbb{Z}) = \\mathbb{Z}^{\\binom{n}{k}}.
\\]

*证明思路*（Künneth公式）：$T^n = S^1 \\times \\cdots \\times S^1$。由Künneth公式，$H_*(T^n)$是$H_*(S^1) = \\mathbb{Z} \\oplus \\mathbb{Z}$（分别对应0维和1维生成元）的$n$重张量积。展开后，$k$维同调群的秩恰好为$\\binom{n}{k}$。$\\square$

### 克莱因瓶$K$ / Klein Bottle

**定理 5.5**. 克莱因瓶的同调群为
\\[
H_0(K; \\mathbb{Z}) = \\mathbb{Z}, \\quad H_1(K; \\mathbb{Z}) = \\mathbb{Z} \\oplus \\mathbb{Z}/2\\mathbb{Z}, \\quad H_2(K; \\mathbb{Z}) = 0.
\\]

*证明思路*：克莱因瓶可用一个正方形将两组对边分别按同向和反向粘合得到。其胞腔分解含一个0-胞腔、两个1-胞腔$a, b$和一个2-胞腔$e$。边缘算子为$\\partial_1(a) = \\partial_1(b) = 0$，$\\partial_2(e) = 2a$（因为一条对边反向粘合贡献了$+a + a = 2a$，另一条同向粘合贡献了$b - b = 0$）。因此
\\[
H_1 = \\ker \\partial_1 / \\mathrm{im} \\partial_2 = \\mathbb{Z}^2 / \\langle 2a \\rangle \\cong \\mathbb{Z} \\oplus \\mathbb{Z}/2\\mathbb{Z}. \\square
\\]

克莱因瓶的1维同调群中的挠元$\\mathbb{Z}/2\\mathbb{Z}$反映了其不可定向性：存在一条闭曲线（如$a$），沿它走两圈才构成某个2维区域的边界，而绕一圈则不是任何区域的边界。
