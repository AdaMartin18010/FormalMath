## 5.5.1 概述 / Overview

同调论是代数拓扑的核心分支，通过代数方法研究拓扑空间的不变量。同调群提供了区分不同拓扑空间的重要工具，是现代数学中最重要的概念之一。

## 5.5.2 基本概念 / Basic Concepts

### 5.5.2.1 链复形 / Chain Complex

**定义 5.5.1** (链复形 / Chain Complex)
链复形是一个序列 $(C_*, \partial_*)$，其中：

- $C_n$ 是阿贝尔群（或模）
- $\partial_n: C_n \to C_{n-1}$ 是群同态
- $\partial_{n-1} \circ \partial_n = 0$（边界算子的平方为零）

**形式化定义**：
$$\cdots \to C_{n+1} \xrightarrow{\partial_{n+1}} C_n \xrightarrow{\partial_n} C_{n-1} \to \cdots$$

### 5.5.2.2 同调群 / Homology Groups

**定义 5.5.2** (同调群 / Homology Groups)
第 $n$ 个同调群定义为：
$$H_n(C_*) = \frac{\ker \partial_n}{\operatorname{im} \partial_{n+1}}$$

**性质**：

- $H_n(C_*) = 0$ 当且仅当链复形在 $n$ 维是正合的
- 同调群是拓扑不变量

## 5.5.3 奇异同调 / Singular Homology

### 5.5.3.1 奇异单形 / Singular Simplex

**定义 5.5.3** (奇异单形 / Singular Simplex)
设 $X$ 是拓扑空间，$\Delta^n$ 是标准 $n$ 维单形。奇异 $n$ 单形是从 $\Delta^n$ 到 $X$ 的连续映射 $\sigma: \Delta^n \to X$。

**定义 5.5.4** (奇异链群 / Singular Chain Group)
$$C_n(X) = \mathbb{Z}[\{\sigma: \Delta^n \to X\}]$$

### 5.5.3.2 边界算子 / Boundary Operator

**定义 5.5.5** (边界算子 / Boundary Operator)
对于奇异单形 $\sigma: \Delta^n \to X$，边界算子定义为：
$$\partial_n(\sigma) = \sum_{i=0}^n (-1)^i \sigma \circ d_i$$

其中 $d_i: \Delta^{n-1} \to \Delta^n$ 是第 $i$ 个面映射。

### 5.5.3.3 奇异同调群 / Singular Homology Groups

**定义 5.5.6** (奇异同调群 / Singular Homology Groups)
$$H_n(X) = \frac{\ker \partial_n}{\operatorname{im} \partial_{n+1}}$$

**定理 5.5.1** (同伦不变性 / Homotopy Invariance)
如果 $f, g: X \to Y$ 是同伦的连续映射，则 $f_* = g_*: H_n(X) \to H_n(Y)$。

## 5.5.4 胞腔同调 / Cellular Homology

### 5.5.4.1 CW复形 / CW Complex

**定义 5.5.7** (CW复形 / CW Complex)
CW复形是通过以下方式构建的空间：

1. 从0维骨架（离散点集）开始
2. 通过粘合 $n$ 维胞腔到 $(n-1)$ 维骨架上构建 $n$ 维骨架
3. 取极限得到整个空间

### 5.5.4.2 胞腔链复形 / Cellular Chain Complex

**定义 5.5.8** (胞腔链群 / Cellular Chain Group)
$$C_n^{cell}(X) = H_n(X^n, X^{n-1})$$

**定理 5.5.2** (胞腔同调计算 / Cellular Homology Computation)
对于CW复形 $X$，胞腔同调群可以通过以下方式计算：
$$H_n^{cell}(X) = \frac{\ker \partial_n^{cell}}{\operatorname{im} \partial_{n+1}^{cell}}$$

## 5.5.5 上同调 / Cohomology

### 5.5.5.1 上链复形 / Cochain Complex

**定义 5.5.9** (上链复形 / Cochain Complex)
上链复形是链复形的对偶：
$$\cdots \to C^{n-1} \xrightarrow{d^{n-1}} C^n \xrightarrow{d^n} C^{n+1} \to \cdots$$

### 5.5.5.2 上同调群 / Cohomology Groups

**定义 5.5.10** (上同调群 / Cohomology Groups)
$$H^n(C^*) = \frac{\ker d^n}{\operatorname{im} d^{n-1}}$$

### 5.5.5.3 奇异上同调 / Singular Cohomology

**定义 5.5.11** (奇异上链群 / Singular Cochain Group)
$$C^n(X; G) = \operatorname{Hom}(C_n(X), G)$$

**定义 5.5.12** (奇异上同调群 / Singular Cohomology Groups)
$$H^n(X; G) = \frac{\ker \delta^n}{\operatorname{im} \delta^{n-1}}$$

## 5.5.6 同调序列 / Homology Sequences

### 5.5.6.1 长正合序列 / Long Exact Sequence

**定理 5.5.3** (相对同调长正合序列 / Long Exact Sequence of Relative Homology)
对于空间对 $(X, A)$，存在长正合序列：
$$\cdots \to H_n(A) \to H_n(X) \to H_n(X, A) \to H_{n-1}(A) \to \cdots$$

### 5.5.6.2 Mayer-Vietoris序列 / Mayer-Vietoris Sequence

**定理 5.5.4** (Mayer-Vietoris序列)
设 $X = A \cup B$，$A, B$ 是开集，则存在长正合序列：
$$\cdots \to H_n(A \cap B) \to H_n(A) \oplus H_n(B) \to H_n(X) \to H_{n-1}(A \cap B) \to \cdots$$

## 5.5.7 乘积结构 / Product Structures

### 5.5.7.1 杯积 / Cup Product

**定义 5.5.13** (杯积 / Cup Product)
杯积是上同调环中的乘法运算：
$$\smile: H^p(X; R) \otimes H^q(X; R) \to H^{p+q}(X; R)$$

**性质**：

- 结合律：$(a \smile b) \smile c = a \smile (b \smile c)$
- 分配律：$(a + b) \smile c = a \smile c + b \smile c$
- 反交换律：$a \smile b = (-1)^{pq} b \smile a$

### 5.5.7.2 卡积 / Cap Product

**定义 5.5.14** (卡积 / Cap Product)
卡积是上同调与同调之间的配对：
$$\frown: H^p(X; R) \otimes H_{p+q}(X; R) \to H_q(X; R)$$

## 5.5.9 应用与计算 / Applications and Computations

### 5.5.9.1 球面的同调 / Homology of Spheres

**定理 5.5.5** (球面同调 / Sphere Homology)
对于 $n$ 维球面 $S^n$：
$$
H_k(S^n) = \begin{cases}
\mathbb{Z} & \text{if } k = 0, n \\
0 & \text{otherwise}
\end{cases}
$$

### 5.5.9.2 环面的同调 / Homology of Torus

**定理 5.5.6** (环面同调 / Torus Homology)
对于2维环面 $T^2 = S^1 \times S^1$：
$$
H_k(T^2) = \begin{cases}
\mathbb{Z} & \text{if } k = 0, 2 \\
\mathbb{Z} \oplus \mathbb{Z} & \text{if } k = 1 \\
0 & \text{otherwise}
\end{cases}
$$

### 5.5.9.3 实射影空间 / Real Projective Space

**定理 5.5.7** (实射影空间同调 / Real Projective Space Homology)
对于 $n$ 维实射影空间 $\mathbb{R}P^n$：
$$
H_k(\mathbb{R}P^n) = \begin{cases}
\mathbb{Z} & \text{if } k = 0 \\
\mathbb{Z}/2\mathbb{Z} & \text{if } k \text{ is odd and } k \leq n \\
0 & \text{otherwise}
\end{cases}
$$

## 5.5.10 高级主题 / Advanced Topics

### 5.5.10.1 谱序列 / Spectral Sequences

**定义 5.5.15** (谱序列 / Spectral Sequence)
谱序列是计算同调群的重要工具，通过逐页逼近的方式计算目标同调群。

### 5.5.10.2 稳定同伦论 / Stable Homotopy Theory

**定义 5.5.16** (稳定同伦群 / Stable Homotopy Groups)
稳定同伦群是普通同伦群的稳定化：
$$\pi_n^S(X) = \lim_{k \to \infty} \pi_{n+k}(S^k \wedge X)$$

### 5.5.10.3 广义同调论 / Generalized Homology Theories

**定义 5.5.17** (广义同调论 / Generalized Homology Theory)
满足Eilenberg-Steenrod公理的同调理论称为广义同调论。

## 5.5.11 总结 / Summary

同调论为拓扑学提供了强大的代数工具，通过同调群可以区分不同的拓扑空间。
奇异同调、胞腔同调和上同调是三种主要的同调理论，它们相互补充，为拓扑空间的研究提供了完整的框架。

### 5.5.11.1 主要成果 / Main Results

1. **同调群的不变性**：同调群在同胚下保持不变
2. **同伦不变性**：同伦等价的空间具有同构的同调群
3. **长正合序列**：提供了计算同调群的重要工具
4. **乘积结构**：杯积和卡积提供了丰富的代数结构

### 5.5.11.2 应用领域 / Applications

- **拓扑学**：空间分类和不变量的计算
- **代数几何**：代数簇的拓扑性质研究
- **微分几何**：流形的拓扑性质
- **数学物理**：规范理论和弦论中的应用
