## 一、项目愿景与目标

### 1.1 愿景

**让每一个数学学习者都能找到适合自己的学习路径**

FormalMath 致力于构建一个覆盖从基础数学到现代数学前沿的综合性教育知识库，通过结构化的内容、数学家理念体系的独特视角、与国际权威课程的紧密对齐，以及智能学习系统的支持，为不同背景的学习者提供个性化的数学学习体验。

### 1.2 目标

| 目标层级 | 具体目标 | 完成状态 |
|----------|----------|----------|
| **基础目标** | 建立15个数学分支的完整文档体系 | ✅ 100% |
| **核心目标** | 建设67位数学家的理念体系 | ✅ 100% |
| **对齐目标** | 与九校121+门国际课程建立映射 | ✅ 100% |
| **技术目标** | 开发三个智能学习系统 | ✅ 100% |
| **形式化目标** | 62个Lean4定理形式化 | ✅ 100% |

### 1.3 项目价值

1. **学术价值**：系统梳理数学家思想，建立知识关联网络
2. **教育价值**：提供结构化学习路径，降低数学学习门槛
3. **技术价值**：探索AI辅助数学教育的可能性
4. **国际价值**：与国际顶尖数学课程对齐，促进学术交流

## 二、核心特色

### 2.1 数学家理念体系 ⭐核心特色

项目最具特色的部分是**67位数学家的理念体系**，每位数学家包含以下标准模块：

```

数学家姓名数学理念/
├── 00-文档索引.md              # 该数学家所有文档索引
├── 00-项目状态.md              # 完成度与待办
├── README.md                   # 快速入门
├── START-HERE.md              # 从这里开始
├── 01-核心理论/               # 核心理论贡献
├── 02-数学内容深度分析/       # 深度数学内容（主要）

├── 05-现代应用与拓展/         # 现代数学/物理/CS应用
├── 06-对比研究/               # 与同时代/前人/后继者对比
```

### 2.2 形式化证明（Lean4）

项目包含**62个经过编译验证的Lean4定理**，代码特点：

- ✅ 全部通过 Lean 4.29.0 + Mathlib4 v4.29.0 编译
- ✅ 与Mathlib4建立62个概念映射
- ✅ 代码规范遵循Mathlib4标准
- ✅ 涵盖代数、分析、数论、拓扑、逻辑五大领域

**定理列表**：

| 领域 | 定理 |
|------|------|
| 群论 | 拉格朗日定理、第一同构定理、Cayley定理、Sylow第一定理 |
| 分析学 | 中值定理、均值定理、Bolzano-Weierstrass定理、Heine-Borel定理 |
| 数论 | 费马小定理、素数无穷多、欧几里得算法 |
| 拓扑学 | Brouwer不动点定理、紧致性定理 |
| 逻辑/组合 | Gödel不完备性定理、鸽巢原理、无穷鸽巢原理、Cantor对角线定理 |

### 2.4 智能学习系统

项目开发了三个相互关联的智能学习系统：

#### 认知诊断系统 (T2.1)
- **技术栈**: Python + FastAPI + PostgreSQL + React + TypeScript
- **核心功能**: HCD诊断算法，100+诊断题目
- **诊断维度**: 知识掌握度、认知能力、学习风格

#### 评估系统 (T2.2)
- **技术栈**: Python + FastAPI + SQLAlchemy + React + Recharts
- **核心功能**: 多维度评价、反馈引擎、报告生成
- **评价维度**: 知识、技能、思维、创新

#### 自适应学习系统 (T3.1)
- **技术栈**: Python + FastAPI + React + JavaScript
- **核心功能**: A*路径算法、实时调整、知识图谱
- **适配维度**: 学习进度、认知状态、兴趣偏好

## 四、使用指南

### 4.1 如何选择起点

根据您的数学背景选择合适的学习起点：

#### 初学者路径（高中→大一）

| 阶段 | 文档 | 预计时间 |
|------|------|----------|
| 1 | `docs/01-基础数学/集合论/01-集合论基础-国际标准版.md` | 1周 |
| 2 | `docs/01-基础数学/集合论/02-数系与运算-深度扩展版.md` | 1周 |
| 3 | `docs/02-代数结构/线性代数/线性代数-深度扩展版.md` | 3周 |

#### 中级路径（大二→大三）

| 阶段 | 文档 | 预计时间 |
|------|------|----------|
| 1 | `docs/02-代数结构/群论/群论-深度扩展版.md` | 3周 |
| 2 | `docs/03-分析学/01-实分析/实分析-深度扩展版.md` | 4周 |
| 3 | `docs/05-拓扑学/点集拓扑-深度扩展版.md` | 3周 |

#### 高级路径（研究生→研究者）

| 阶段 | 文档 | 预计时间 |
|------|------|----------|
| 1 | `数学家理念体系/格洛腾迪克数学理念/02-数学内容深度分析/` | 6月 |
| 2 | `docs/13-代数几何/概形理论-深度扩展版.md` | 2月 |
| 3 | `docs/15-同调代数/导出范畴-深度扩展版.md` | 2月 |

### 4.2 如何利用国际课程对齐

1. **查找您的课程**：在 [国际课程对照表](../project/00-国际课程与机构对齐对照表-2026年4月.md) 中找到对应学校/课程
2. **查看映射关系**：了解课程各章节对应的项目文档
3. **按顺序学习**：遵循推荐的学习顺序
4. **参考权威资源**：利用文中引用的Stacks tag、教材章节

### 4.3 如何使用数学家理念体系

1. **选择感兴趣的数学家**：从 数学家索引 开始
2. **阅读START-HERE.md**：快速了解该数学家的核心贡献
3. **深入02-数学内容深度分析**：学习具体的数学理论
4. **探索08-知识关联分析**：理解概念间的联系

### 4.4 如何运行Lean4代码

```bash
# 安装Lean4和Mathlib4
# 参见 docs/09-形式化证明/Lean4入门指南.md

# 编译项目定理
cd docs/09-形式化证明/lean4-examples/
lean --make

```
