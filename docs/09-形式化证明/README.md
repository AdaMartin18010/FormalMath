# 09-形式化证明

**数学的机械化 — 从定理证明器到形式化数学**

---

## 📋 分支概览

形式化证明分支致力于使用计算机辅助证明系统严格地验证数学定理，是数学与计算机科学的交叉领域：

- **证明系统基础**：形式化方法的理论基础
- **自动定理证明**：ATP系统与算法
- **交互式证明助手**：Lean4、Coq、Isabelle/HOL
- **类型论与证明**：Curry-Howard对应
- **Mathlib4**：现代数学的形式化库
- **形式化与AI**：机器学习的应用

**难度等级**: ⭐⭐⭐⭐（高级）
