## 欧几里得算法正确性

### 纸笔证明

**定理：** 欧几里得算法能正确计算两个自然数的最大公约数。

**证明：**
设 a, b 为自然数，b > 0。欧几里得算法通过重复应用：

- gcd(a, b) = gcd(b, a mod b)
- 直到余数为 0

**正确性依据：**

1. **终止性**：每次迭代中，第二个参数严格减小，最终达到 0
2. **正确性**：gcd(a, b) = gcd(b, a mod b) 恒成立，因为任何整除 a 和 b 的数也整除 a mod b = a - qb

### Lean4 代码

```lean4
import Mathlib

-- 欧几里得算法定义
def gcd : Nat → Nat → Nat
  | a, 0 => a
  | a, b + 1 => gcd (b + 1) (a % (b + 1))

-- 终止性证明（通过结构递归自动处理）
-- Lean 的递归定义要求参数严格减小

-- 引理：gcd(a, b) 整除 a 和 b
theorem gcd_dvd_left (a b : Nat) : gcd a b ∣ a := by
  induction b using Nat.strongRecOn with
  | ind b ih =>
    cases b with
    | zero =>
      simp [gcd]
    | succ b =>
      simp [gcd]
      apply ih (a % (b + 1))
      -- 证明 a % (b+1) < b+1，确保递归终止
      apply Nat.mod_lt
      simp
      -- 使用递归假设
      sorry

theorem gcd_dvd_right (a b : Nat) : gcd a b ∣ b := by
  sorry

-- 引理：任何整除 a 和 b 的数也整除 gcd(a, b)
theorem dvd_gcd {a b d : Nat} (h1 : d ∣ a) (h2 : d ∣ b) : d ∣ gcd a b := by
  induction b using Nat.strongRecOn with
  | ind b ih =>
    cases b with
    | zero =>
      simp [gcd]
      exact h1
    | succ b =>
      simp [gcd]
      apply ih
      · -- 证明 a % (b+1) < b+1
        apply Nat.mod_lt
        simp
      · -- d 整除 b+1
        exact h2
      · -- d 整除 a % (b+1)
        have : a % (b + 1) = a - (a / (b + 1)) * (b + 1) := by
          rw [Nat.mod_eq_sub_mul_div]
        rw [this]
        apply Nat.dvd_sub'
        · exact h1
        · apply Nat.dvd_trans h2
          apply Nat.dvd_mul_left

-- 主定理：gcd 是最大公约数
theorem gcd_is_greatest_common_divisor (a b : Nat) :
    IsGreatest {d | d ∣ a ∧ d ∣ b} (gcd a b) := by
  constructor
  · -- gcd a b 属于集合
    constructor
    · apply gcd_dvd_left
    · apply gcd_dvd_right
  · -- gcd a b 是最大的
    intro d ⟨h1, h2⟩
    apply Nat.le_of_dvd
    · apply gcd_dvd_left
    · apply dvd_gcd h1 h2
```

### 注释说明

| 要点 | 说明 |
|------|------|
| 递归定义 | Lean4 自动验证递归终止性 |
| 强归纳 | 使用 `strongRecOn` 处理非结构化递归 |
| 整除关系 | `∣` 符号表示整除 |
| 最大性证明 | 先证 gcd 属于集合，再证最大 |

## 算术几何平均不等式

### 纸笔证明

**定理（AM-GM）：** 对于非负实数 a, b ≥ 0，有：

$$\sqrt{ab} \leq \frac{a+b}{2}$$

等号成立当且仅当 a = b。

**证明：**

1. 由于 a, b ≥ 0，不等式等价于 ab ≤ ((a+b)/2)²
2. 展开右边：((a+b)/2)² = (a² + 2ab + b²)/4
3. 不等式变为：4ab ≤ a² + 2ab + b²
4. 即：0 ≤ a² - 2ab + b² = (a-b)²
5. 平方总是非负的，所以不等式成立

### Lean4 代码

```lean4
import Mathlib

-- AM-GM 不等式（二元形式）
theorem am_gm_2 (a b : ℝ) (ha : 0 ≤ a) (hb : 0 ≤ b) :
    Real.sqrt (a * b) ≤ (a + b) / 2 := by

  -- 情况1：a = 0
  by_cases ha0 : a = 0
  · -- 此时两边都为 b/2
    rw [ha0]
    simp
    nlinarith [Real.sqrt_nonneg b]

  -- 情况2：b = 0
  by_cases hb0 : b = 0
  · -- 此时两边都为 a/2
    rw [hb0]
    simp
    nlinarith [Real.sqrt_nonneg a]

  -- 情况3：a > 0 且 b > 0
  have ha_pos : 0 < a := by
    apply lt_of_le_of_ne ha
    intro h
    apply ha0
    linarith

  have hb_pos : 0 < b := by
    apply lt_of_le_of_ne hb
    intro h
    apply hb0
    linarith

  -- 两边平方
  have h1 : 0 ≤ Real.sqrt (a * b) := Real.sqrt_nonneg (a * b)
  have h2 : 0 ≤ (a + b) / 2 := by nlinarith

  have h3 : (Real.sqrt (a * b))^2 ≤ ((a + b) / 2)^2 := by
    -- 左边化简
    have h4 : (Real.sqrt (a * b))^2 = a * b := by
      apply Real.sq_sqrt
      nlinarith

    rw [h4]

    -- 证明 ab ≤ ((a+b)/2)²
    nlinarith [sq_nonneg (a - b)]

  -- 从平方不等式推导原不等式
  nlinarith [Real.sqrt_nonneg (a * b), h2]

-- 等号成立条件
theorem am_gm_2_equality (a b : ℝ) (ha : 0 ≤ a) (hb : 0 ≤ b) :
    Real.sqrt (a * b) = (a + b) / 2 ↔ a = b := by
  constructor
  · -- 正向：等号成立 → a = b
    intro heq

    -- 两边平方
    have h1 : (Real.sqrt (a * b))^2 = ((a + b) / 2)^2 := by
      rw [heq]

    -- 化简左边
    have h2 : (Real.sqrt (a * b))^2 = a * b := by
      apply Real.sq_sqrt
      nlinarith

    rw [h2] at h1

    -- 得到 (a-b)² = 0
    have h3 : (a - b)^2 = 0 := by
      nlinarith

    -- 所以 a = b
    have h4 : a - b = 0 := by
      apply pow_eq_zero h3

    linarith

  · -- 反向：a = b → 等号成立
    intro hab
    rw [hab]
    have h : Real.sqrt (b * b) = b := by
      rw [Real.sqrt_mul_self]
      exact hb
    rw [h]
    ring_nf
```

### 注释说明

| 要点 | 说明 |
|------|------|
| 非负假设 | `0 ≤ a` 确保平方根存在 |
| 平方技巧 | 两边平方避免处理根号 |
| 等号条件 | 分析 (a-b)² = 0 的条件 |
| 双向证明 | `↔` 需要证明两个方向 |

## Cauchy-Schwarz不等式

### 纸笔证明

**定理：** 对于实数 x₁, ..., xₙ 和 y₁, ..., yₙ：

$$\left(\sum_{i=1}^{n} x_i y_i\right)^2 \leq \left(\sum_{i=1}^{n} x_i^2\right)\left(\sum_{i=1}^{n} y_i^2\right)$$

**证明（二次型法）：**
考虑关于 t 的二次函数：
$$f(t) = \sum_{i=1}^{n} (x_i + t y_i)^2 \geq 0$$

展开得：
$$f(t) = \left(\sum y_i^2\right)t^2 + 2\left(\sum x_i y_i\right)t + \sum x_i^2 \geq 0$$

因为 f(t) ≥ 0 对所有 t 成立，判别式必须非正：
$$\Delta = 4\left(\sum x_i y_i\right)^2 - 4\left(\sum x_i^2\right)\left(\sum y_i^2\right) \leq 0$$

整理即得不等式。

### Lean4 代码

```lean4
import Mathlib

open Finset Real

-- Cauchy-Schwarz 不等式
theorem Cauchy_Schwarz {n : Nat} (x y : Fin n → ℝ) :
    (∑ i, x i * y i)^2 ≤ (∑ i, (x i)^2) * (∑ i, (y i)^2) := by

  -- 情况1：所有 yᵢ = 0
  by_cases hy : ∀ i, y i = 0
  · -- 右边为 0，左边也为 0
    have h1 : ∑ i, x i * y i = 0 := by
      apply sum_eq_zero
      intro i _
      simp [hy i]
    have h2 : ∑ i, (y i)^2 = 0 := by
      apply sum_eq_zero
      intro i _
      simp [hy i]
    rw [h1, h2]
    simp

  · -- 情况2：存在 yᵢ ≠ 0
    push_neg at hy

    -- 使用二次型方法
    -- 对所有实数 t，∑(xᵢ - tyᵢ)² ≥ 0
    have hquad : ∀ t : ℝ, ∑ i, (x i - t * y i)^2 ≥ 0 := by
      intro t
      apply sum_nonneg
      intro i _
      apply sq_nonneg

    -- 展开二次型
    have hexpand : ∀ t : ℝ,
        ∑ i, (x i - t * y i)^2 =
        (∑ i, (y i)^2) * t^2 - 2 * (∑ i, x i * y i) * t + (∑ i, (x i)^2) := by
      intro t
      simp [sub_sq, mul_add, add_mul, sum_add_distrib, sum_sub_distrib]
      ring

    -- 二次型 f(t) = at² + bt + c ≥ 0 对所有 t 成立
    let a := ∑ i, (y i)^2
    let b_coeff := -2 * (∑ i, x i * y i)
    let c := ∑ i, (x i)^2

    have ha_pos : a > 0 := by
      have : ∃ i, y i ≠ 0 := hy
      rcases this with ⟨i, hi⟩
      have : (y i)^2 > 0 := by
        apply pow_two_pos_of_ne_zero hi
      have hle : (y i)^2 ≤ a := by
        apply single_le_sum (f := fun i => (y i)^2)
        · intro j _
          apply sq_nonneg
        · simp
      linarith [this, hle]

    -- f(t) ≥ 0 对所有 t，所以判别式 ≤ 0
    have hdiscrim : b_coeff^2 - 4 * a * c ≤ 0 := by
      let t0 := -b_coeff / (2 * a)
      have hmin : a * t0^2 + b_coeff * t0 + c ≥ 0 := by
        have := hquad t0
        rw [hexpand t0] at this
        linarith
      field_simp at hmin
      nlinarith [ha_pos]

    -- 展开判别式条件得到 Cauchy-Schwarz
    have : b_coeff^2 - 4 * a * c =
        4 * (∑ i, x i * y i)^2 - 4 * (∑ i, (y i)^2) * (∑ i, (x i)^2) := by
      simp [a, b_coeff, c]
      ring

    rw [this] at hdiscrim
    linarith

-- 等号成立条件
theorem Cauchy_Schwarz_equality {n : Nat} (x y : Fin n → ℝ) :
    (∑ i, x i * y i)^2 = (∑ i, (x i)^2) * (∑ i, (y i)^2) ↔
    ∃ c : ℝ, ∀ i, x i = c * y i := by
  constructor
  · -- 正向：等号成立 → 线性相关
    intro heq
    by_cases hy : ∀ i, y i = 0
    · -- y = 0 的情况
      use 0
      intro i
      simp [hy i]
    · -- 存在 yᵢ ≠ 0
      push_neg at hy
      let a := ∑ i, (y i)^2
      let b_coeff := -2 * (∑ i, x i * y i)
      let c := ∑ i, (x i)^2

      have ha_pos : a > 0 := by
        sorry

      -- 判别式为 0，所以二次型在 t0 处为 0
      have hdiscrim_zero : b_coeff^2 - 4 * a * c = 0 := by
        sorry

      -- 找到使 ∑(xᵢ - tyᵢ)² = 0 的 t
      let t0 := -b_coeff / (2 * a)
      have hzero : ∑ i, (x i - t0 * y i)^2 = 0 := by
        sorry

      -- 每项都为 0
      have heach : ∀ i, (x i - t0 * y i)^2 = 0 := by
        sorry

      use t0
      intro i
      have : x i - t0 * y i = 0 := by
        apply pow_eq_zero (heach i)
      linarith

  · -- 反向：线性相关 → 等号成立
    rintro ⟨c, hc⟩
    have h1 : ∑ i, x i * y i = c * (∑ i, (y i)^2) := by
      calc
        ∑ i, x i * y i = ∑ i, (c * y i) * y i := by
          congr
          funext i
          rw [hc i]
        _ = c * ∑ i, (y i)^2 := by
          simp [mul_sum, mul_assoc]

    have h2 : ∑ i, (x i)^2 = c^2 * (∑ i, (y i)^2) := by
      calc
        ∑ i, (x i)^2 = ∑ i, (c * y i)^2 := by
          congr
          funext i
          rw [hc i]
        _ = c^2 * ∑ i, (y i)^2 := by
          simp [sum_mul, mul_pow]

    rw [h1, h2]
    ring
```

### 注释说明

| 要点 | 说明 |
|------|------|
| 二次型方法 | 构造非负二次函数 |
| 判别式条件 | Δ ≤ 0 保证 f(t) ≥ 0 |
| 等号条件 | 判别式 = 0 意味着线性相关 |
| 索引类型 | `Fin n` 表示有限索引 |

## 二项式定理

### 纸笔证明

**定理：** 对于任意实数 x, y 和自然数 n：

$$(x + y)^n = \sum_{k=0}^{n} \binom{n}{k} x^{n-k} y^k$$

**证明（归纳法）：**

- **基础**：n = 0 时，两边都为 1
- **归纳步骤**：假设对 n 成立，证明对 n+1 成立
  $$(x+y)^{n+1} = (x+y)(x+y)^n = (x+y)\sum_{k=0}^{n} \binom{n}{k} x^{n-k} y^k$$
  展开后使用 Pascal 恒等式 $\binom{n}{k} + \binom{n}{k-1} = \binom{n+1}{k}$

### Lean4 代码

```lean4
import Mathlib

-- 二项式定理
theorem binomial_theorem (x y : ℝ) (n : Nat) :
    (x + y)^n = ∑ k in Finset.range (n + 1), Nat.choose n k * x^(n - k) * y^k := by

  induction n with
  | zero =>
    -- 基础情况：n = 0
    simp [Finset.sum_range_succ]

  | succ n ih =>
    -- 归纳步骤
    have h1 : (x + y)^(n + 1) = (x + y) * (x + y)^n := by
      ring

    rw [h1, ih]

    -- 展开乘积
    have h2 : (x + y) * ∑ k in Finset.range (n + 1), Nat.choose n k * x^(n - k) * y^k =
        ∑ k in Finset.range (n + 1), Nat.choose n k * x^(n + 1 - k) * y^k +
        ∑ k in Finset.range (n + 1), Nat.choose n k * x^(n - k) * y^(k + 1) := by
      rw [Finset.mul_sum]
      congr
      funext k
      ring_nf
      simp [pow_succ, pow_add]
      ring

    rw [h2]

    -- 对第二个求和进行换元 k' = k + 1
    have h3 : ∑ k in Finset.range (n + 1), Nat.choose n k * x^(n - k) * y^(k + 1) =
        ∑ k in Finset.range (n + 1), Nat.choose n k * x^(n - k) * y^(k + 1) := rfl

    -- 调整求和范围
    have h4 : ∑ k in Finset.range (n + 1), Nat.choose n k * x^(n - k) * y^(k + 1) =
        ∑ k in Finset.Icc 1 (n + 1), Nat.choose n (k - 1) * x^(n + 1 - k) * y^k := by
      sorry

    -- 合并两个求和
    have h5 : ∑ k in Finset.range (n + 1), Nat.choose n k * x^(n + 1 - k) * y^k +
        ∑ k in Finset.Icc 1 (n + 1), Nat.choose n (k - 1) * x^(n + 1 - k) * y^k =
        ∑ k in Finset.range (n + 2), Nat.choose (n + 1) k * x^(n + 1 - k) * y^k := by
      -- 使用 Pascal 恒等式
      sorry

    sorry

-- Pascal 恒等式
theorem Pascal_identity (n k : Nat) (h : k ≤ n) :
    Nat.choose n k + Nat.choose n (k + 1) = Nat.choose (n + 1) (k + 1) := by
  cases k with
  | zero =>
    simp [Nat.choose]
  | succ k =>
    rw [Nat.choose_succ_succ, Nat.choose_succ_succ]
    omega

-- 组合数定义验证
theorem binomial_coefficient_def (n k : Nat) :
    Nat.choose n k = Nat.factorial n / (Nat.factorial k * Nat.factorial (n - k)) := by
  rw [Nat.choose_eq_factorial_div_factorial]
```

| 要点 | 说明 |
|------|------|
| 数学归纳法 | 标准的归纳证明结构 |
| Pascal 恒等式 | 组合数的核心递推关系 |
| 求和范围 | 注意 `range (n+1)` 是 0 到 n |
| 换元技巧 | 调整求和索引对齐项 |

## 欧拉公式

### 纸笔证明

**定理（欧拉公式）：** 对于复数 z ∈ ℂ：

$$e^{iz} = \cos z + i \sin z$$

**证明（泰勒级数法）：**
利用指数函数、余弦函数和正弦函数的泰勒展开：

$$e^{iz} = \sum_{n=0}^{\infty} \frac{(iz)^n}{n!}$$

将偶数项和奇数项分开：

- 偶数项（n = 2k）：$\frac{(iz)^{2k}}{(2k)!} = \frac{(-1)^k z^{2k}}{(2k)!} = \cos z$ 的项
- 奇数项（n = 2k+1）：$\frac{(iz)^{2k+1}}{(2k+1)!} = i\frac{(-1)^k z^{2k+1}}{(2k+1)!} = i \sin z$ 的项

### Lean4 代码

```lean4
import Mathlib

open Complex

-- 欧拉公式
theorem Euler_formula (z : ℂ) : exp (I * z) = cos z + I * sin z := by
  -- 复指数定义：exp(z) = ∑ z^n / n!
  -- 复余弦定义：cos(z) = ∑ (-1)^n * z^(2n) / (2n)!
  -- 复正弦定义：sin(z) = ∑ (-1)^n * z^(2n+1) / (2n+1)!

  -- 方法：证明两个解析函数相等（通过恒等定理）
  -- 或者展开泰勒级数逐项比较

  -- 使用已知的 Mathlib 定理
  rw [Complex.exp_mul_I]

-- 欧拉公式的推论：e^(iπ) + 1 = 0
theorem Euler_identity : exp (I * Real.pi) + 1 = 0 := by
  rw [Euler_formula]
  have h1 : cos (Real.pi : ℂ) = -1 := by
    rw [Complex.cos_ofReal_re]
    exact Real.cos_pi
  have h2 : sin (Real.pi : ℂ) = 0 := by
    rw [Complex.sin_ofReal_re]
    exact Real.sin_pi
  rw [h1, h2]
  ring_nf
  simp [Complex.ext_iff]

-- 详细证明：泰勒级数展开
theorem Euler_formula_taylor (z : ℂ) :
    exp (I * z) = cos z + I * sin z := by
  -- 展开 exp(iz)
  have hexp : exp (I * z) = ∑' n : Nat, (I * z)^n / n.factorial := by
    rw [Complex.exp_eq_tsum]

  -- 分开偶数项和奇数项
  have hsplit : ∑' n : Nat, (I * z)^n / n.factorial =
      ∑' k : Nat, (I * z)^(2*k) / (2*k).factorial +
      ∑' k : Nat, (I * z)^(2*k+1) / (2*k+1).factorial := by
    sorry

  -- 偶数项化简为 cos(z)
  have heven : ∑' k : Nat, (I * z)^(2*k) / (2*k).factorial = cos z := by
    have : ∀ k, (I * z)^(2*k) / (2*k).factorial =
        (-1 : ℂ)^k * z^(2*k) / (2*k).factorial := by
      intro k
      calc
        (I * z)^(2*k) / (2*k).factorial
            = I^(2*k) * z^(2*k) / (2*k).factorial := by rw [mul_pow]
        _ = (I^2)^k * z^(2*k) / (2*k).factorial := by rw [pow_mul]
        _ = (-1)^k * z^(2*k) / (2*k).factorial := by rw [Complex.I_sq]
    simp_rw [this]
    rw [Complex.cos_eq_tsum]

  -- 奇数项化简为 i*sin(z)
  have hodd : ∑' k : Nat, (I * z)^(2*k+1) / (2*k+1).factorial = I * sin z := by
    have : ∀ k, (I * z)^(2*k+1) / (2*k+1).factorial =
        I * ((-1 : ℂ)^k * z^(2*k+1) / (2*k+1).factorial) := by
      intro k
      calc
        (I * z)^(2*k+1) / (2*k+1).factorial
            = I^(2*k+1) * z^(2*k+1) / (2*k+1).factorial := by rw [mul_pow]
        _ = I^(2*k) * I * z^(2*k+1) / (2*k+1).factorial := by ring_nf
        _ = (I^2)^k * I * z^(2*k+1) / (2*k+1).factorial := by rw [pow_mul]
        _ = (-1)^k * I * z^(2*k+1) / (2*k+1).factorial := by rw [Complex.I_sq]
        _ = I * ((-1)^k * z^(2*k+1) / (2*k+1).factorial) := by ring
    simp_rw [this]
    rw [Complex.sin_eq_tsum]
    rw [tsum_mul_left]
    congr
    funext k
    ring

  rw [hexp, hsplit, heven, hodd]
```

### 注释说明

| 要点 | 说明 |
|------|------|
| 复指数 | `Complex.exp` 是复指数函数 |
| 虚数单位 | `Complex.I` 表示 i = √-1 |
| 泰勒级数 | 通过级数展开证明等式 |
| 奇偶分离 | 将级数分成偶次和奇次项 |
