## 翻译方法论

### 翻译流程

将纸笔证明转换为形式化证明的典型流程：

```
纸笔证明 → 结构化理解 → 形式化策略 → Lean4 代码 → 验证
```

### 步骤1：理解证明结构

**纸笔证明示例：**

> **定理**：对于所有自然数 n，n³ - n 能被 6 整除。
>
> **证明**：注意到 n³ - n = n(n-1)(n+1)。这是三个连续整数的乘积。
> 在任何三个连续整数中，必有一个是 3 的倍数，至少一个是偶数（2 的倍数）。
> 因此乘积能被 6 = 2 × 3 整除。

**结构化分析：**

1. **陈述**：∀ n : ℕ, 6 | (n³ - n)
2. **关键观察**：n³ - n = n(n-1)(n+1) = (n-1)n(n+1)
3. **核心论证**：
   - 三个连续整数中必有一个是 3 的倍数
   - 三个连续整数中至少有一个是偶数
   - 因此乘积能被 6 整除

### 步骤2：识别隐含假设

纸笔证明中常隐含以下假设：

| 隐含内容 | 形式化注意 |
|---------|-----------|
| "显然" | 需要显式引用引理 |
| "不失一般性" | 需要证明 WLOG 的合理性 |
| "类似的" | 需要完整证明所有情况 |
| "容易验证" | 需要具体计算或引用 |

### 步骤3：选择形式化策略

```lean4
-- 定理陈述
example (n : Nat) : 6 ∣ (n^3 - n) := by
  -- 步骤1：因式分解
  have h1 : n^3 - n = (n - 1) * n * (n + 1) := by
    cases n with
    | zero => simp
    | succ n =>
      simp [Nat.pow_succ, Nat.mul_add, Nat.add_mul]
      ring_nf
      omega

  rw [h1]

  -- 步骤2：证明可被2整除
  have h2 : 2 ∣ (n - 1) * n * (n + 1) := by
    have : (n - 1) * n * (n + 1) % 2 = 0 := by
      have h1 : n % 2 = 0 ∨ n % 2 = 1 := by omega
      rcases h1 with (h | h)
      · -- n 是偶数
        simp [h, Nat.mul_mod, Nat.sub_mod_eq_zero_of_mod_eq]
      · -- n 是奇数
        simp [h, Nat.mul_mod, Nat.sub_mod_eq_zero_of_mod_eq]
        omega
    omega

  -- 步骤3：证明可被3整除
  have h3 : 3 ∣ (n - 1) * n * (n + 1) := by
    have : (n - 1) * n * (n + 1) % 3 = 0 := by
      have h1 : n % 3 = 0 ∨ n % 3 = 1 ∨ n % 3 = 2 := by omega
      rcases h1 with (h | h | h)
      · -- n ≡ 0 (mod 3)
        simp [h, Nat.mul_mod]
      · -- n ≡ 1 (mod 3)
        simp [h, Nat.mul_mod, Nat.sub_mod_eq_zero_of_mod_eq]
        all_goals omega
      · -- n ≡ 2 (mod 3)
        simp [h, Nat.mul_mod, Nat.add_mod]
        all_goals omega
    omega

  -- 步骤4：结合结果
  have h4 : Nat.Coprime 2 3 := by norm_num
  apply Nat.lcm_dvd h2 h3
```

## 常见陷阱和解决方案

### 陷阱1：整数 vs 自然数

**问题：** 在 ℕ 中减法可能导致意外结果。

```lean4
-- 错误理解
example (n : Nat) : n - 5 + 5 = n := by
  -- 当 n < 5 时，n - 5 = 0，所以结果是 5 不是 n
  sorry

-- 正确做法
example (n : Nat) (h : n ≥ 5) : n - 5 + 5 = n := by
  omega  -- 在假设 n ≥ 5 下成立

-- 或者使用整数
example (n : Int) : n - 5 + 5 = n := by
  ring
```

### 陷阱2：除法和取整

**问题：** 整数除法和实数除法的区别。

```lean4
-- 自然数除法
#eval 5 / 2  -- 结果是 2（向下取整）

-- 实数除法
#eval (5 : Real) / 2  -- 结果是 2.5

-- 证明时要小心类型
example (n m : Nat) (h : n / m = 2) : n = 2 * m := by
  -- 错误！n / m = 2 并不意味着 n = 2*m
  -- 例如 n = 5, m = 2 时，5 / 2 = 2 但 5 ≠ 4
  sorry

-- 正确做法
example (n m : Nat) (h : m > 0) (h2 : n = 2 * m) : n / m = 2 := by
  rw [h2]
  rw [Nat.mul_div_cancel_left]
  omega
```

### 陷阱3：空集和边界情况

**问题：** 忘记处理空集或边界情况。

```lean4
-- 列表求和的空列表情况
def sumList (l : List Nat) : Nat := l.foldl (· + ·) 0

theorem sumList_nonneg (l : List Nat) : sumList l ≥ 0 := by
  -- 需要处理空列表
  unfold sumList
  induction l with
  | nil => simp
  | cons x xs ih =>
    simp [List.foldl]
    omega
```

### 陷阱4：量词顺序

**问题：** ∀ 和 ∃ 的顺序很重要。

```lean4
-- 不同的含义
example : ∀ n : Nat, ∃ m : Nat, m > n := by
  intro n
  use n + 1
  linarith

-- 相反的顺序是错误的
example : ∃ m : Nat, ∀ n : Nat, m > n := by
  -- 这是错误的陈述！
  sorry
```

### 陷阱5：严格不等式 vs 非严格不等式

```lean4
-- n < m 和 n + 1 ≤ m 等价
example (n m : Int) : n < m ↔ n + 1 ≤ m := by
  constructor
  · intro h
    linarith
  · intro h
    linarith

-- 但在自然数中要小心
example (n m : Nat) : n < m → n + 1 ≤ m := by
  apply Nat.succ_le_of_lt
```

### 陷阱6：函数外延性

```lean4
-- 两个函数相等需要证明对所有输入相等
example (f g : Nat → Nat) (h : ∀ x, f x = g x) : f = g := by
  funext x
  exact h x

-- 不能直接从 f x = g x 推出 f = g
```

### 陷阱7：类型类实例推断失败

```lean4
-- 需要确保类型类实例可用
example (n : Nat) : n + 0 = n := by
  simp  -- 需要 AddMonoid Nat 实例

-- 显式指定
example (n : Nat) : n + 0 = n := by
  rw [Nat.add_zero]  -- 直接引用具体引理
```

## 复杂证明的分解策略

### 策略1：引理分解

将复杂证明分解为多个小引理。

```lean4
-- 原始复杂证明
theorem complex_theorem_bad (n : Nat) :
    ∑ i in Finset.range (n + 1), i^2 = n * (n + 1) * (2 * n + 1) / 6 := by
  induction n with
  | zero => simp
  | succ n ih =>
    -- 很长的证明...
    sorry

-- 分解后的版本
-- 引理1：递推关系
lemma sum_squares_rec (n : Nat) :
    ∑ i in Finset.range (n + 1), i^2 =
    ∑ i in Finset.range n, i^2 + n^2 := by
  rw [Finset.sum_range_succ]

-- 引理2：代数恒等式
lemma algebra_identity (n : Nat) :
    n * (n + 1) * (2 * n + 1) / 6 + (n + 1)^2 =
    (n + 1) * (n + 2) * (2 * (n + 1) + 1) / 6 := by
  ring_nf
  omega

-- 主定理
theorem sum_squares_formula (n : Nat) :
    ∑ i in Finset.range (n + 1), i^2 = n * (n + 1) * (2 * n + 1) / 6 := by
  induction n with
  | zero => simp
  | succ n ih =>
    rw [sum_squares_rec]
    rw [ih]
    apply algebra_identity
```

### 策略2：情况分析

对于需要分情况讨论的证明：

```lean4
-- 原始版本（混乱）
theorem messy_cases (n : Nat) : n % 3 = 0 ∨ n % 3 = 1 ∨ n % 3 = 2 := by
  -- 各种情况混在一起
  sorry

-- 结构化版本
theorem structured_cases (n : Nat) : n % 3 = 0 ∨ n % 3 = 1 ∨ n % 3 = 2 := by
  have h : n % 3 < 3 := Nat.mod_lt n (by norm_num)
  have h2 : n % 3 = 0 ∨ n % 3 = 1 ∨ n % 3 = 2 := by
    omega
  exact h2
```

### 策略3：结构归纳

对于归纳类型：

```lean4
inductive Tree (α : Type)
  | leaf : Tree α
  | node : α → Tree α → Tree α → Tree α

def Tree.size : Tree α → Nat
  | leaf => 0
  | node _ l r => 1 + size l + size r

def Tree.height : Tree α → Nat
  | leaf => 0
  | node _ l r => 1 + max (height l) (height r)

-- 结构归纳
theorem Tree.size_le_height_exp (t : Tree α) :
    t.size ≤ 2^t.height - 1 := by
  induction t with
  | leaf =>
    simp [size, height]
  | node x l r ihl ihr =>
    simp [size, height]
    have h1 : l.size ≤ 2^l.height - 1 := ihl
    have h2 : r.size ≤ 2^r.height - 1 := ihr
    -- 继续证明
    sorry
```

### 策略4：辅助定义

引入辅助定义简化证明：

```lean4
-- 原始版本
theorem no_aux_def (n : Nat) :
    ∑ i in Finset.range n, (2 * i + 1) = n^2 := by
  -- 复杂的直接证明
  sorry

-- 使用辅助定义
def oddSum (n : Nat) := ∑ i in Finset.range n, (2 * i + 1)

lemma oddSum_rec (n : Nat) : oddSum (n + 1) = oddSum n + (2 * n + 1) := by
  simp [oddSum, Finset.sum_range_succ]

lemma oddSum_base : oddSum 0 = 0 := by
  simp [oddSum]

theorem oddSum_formula (n : Nat) : oddSum n = n^2 := by
  induction n with
  | zero => exact oddSum_base
  | succ n ih =>
    rw [oddSum_rec, ih]
    ring
```

## 可读性与严格性的平衡

### 原则1：使用描述性名称

```lean4
-- 不好的命名
theorem thm1 (n : Nat) (h : n > 0) : ∃ m, n = m + 1 := by
  let x := n - 1
  use x
  omega

-- 好的命名
theorem exists_predecessor (n : Nat) (hn : n > 0) :
    ∃ m, n = m + 1 := by
  let predecessor := n - 1
  use predecessor
  omega
```

### 原则2：添加注释

```lean4
theorem sum_of_odds (n : Nat) :
    ∑ i in Finset.range n, (2 * i + 1) = n^2 := by
  -- 使用数学归纳法
  induction n with
  | zero =>
    -- 基本情况：空和为 0
    simp
  | succ k ih =>
    -- 归纳步骤：利用前 k 个奇数和的公式
    rw [Finset.sum_range_succ, ih]
    -- 代数化简：(k+1)^2 = k^2 + 2k + 1
    ring
```

### 原则3：使用结构化证明

```lean4
-- 非结构化（tactic 风格）
theorem Cauchy_Schwarz_unstructured (x y : ℝ^n) :
    (x ⬝ y)^2 ≤ (x ⬝ x) * (y ⬝ y) := by
  sorry

-- 结构化（calc 风格）
theorem Cauchy_Schwarz_structured (x y : ℝ^n) :
    (x ⬝ y)^2 ≤ (x ⬝ x) * (y ⬝ y) := by
  calc
    (x ⬝ y)^2 ≤ ‖x‖^2 * ‖y‖^2 := by
      apply sq_le_sq'
      -- ...
      sorry
    _ = (x ⬝ x) * (y ⬝ y) := by
      simp [norm_sq_eq_inner]
```

### 原则4：使用 show 标记中间目标

```lean4
theorem arithmetic_geometric_mean (a b : ℝ) (ha : 0 ≤ a) (hb : 0 ≤ b) :
    Real.sqrt (a * b) ≤ (a + b) / 2 := by
  show Real.sqrt (a * b) ≤ (a + b) / 2

  have h : (Real.sqrt (a * b))^2 ≤ ((a + b) / 2)^2 := by
    show a * b ≤ ((a + b) / 2)^2
    nlinarith [sq_nonneg (a - b)]

  have h2 : 0 ≤ (a + b) / 2 := by
    show 0 ≤ (a + b) / 2
    linarith

  nlinarith [Real.sq_sqrt (show 0 ≤ a * b by nlinarith)]
```

## 实战案例

### 案例1：证明素数无穷

**纸笔证明：**

> 假设只有有限个素数 p₁, p₂, ..., pₙ。令 N = p₁p₂...pₙ + 1。
> N 不被任何 pᵢ 整除（余数为 1），所以 N 要么是素数，要么有新的素因子。
> 这与假设矛盾。

**形式化证明：**

```lean4
import Mathlib

theorem infinite_primes (n : Nat) : ∃ p > n, Nat.Prime p := by
  -- 令 N = n! + 1
  let N := Nat.factorial n + 1

  -- 证明 N > 1
  have hN_gt_1 : N > 1 := by
    have : Nat.factorial n ≥ 1 := Nat.factorial_pos n
    linarith

  -- N 有素因子
  obtain ⟨p, hp_prime, hp_dvd⟩ := Nat.exists_prime_and_dvd (by linarith : N ≠ 1)

  use p
  constructor
  · -- 证明 p > n
    by_contra h
    push_neg at h

    -- 如果 p ≤ n，则 p 整除 n!
    have hp_dvd_fact : p ∣ Nat.factorial n := by
      apply Nat.dvd_factorial
      · exact Nat.Prime.pos hp_prime
      · exact h

    -- p 也整除 N = n! + 1
    -- 所以 p 整除 N - n! = 1
    have hp_dvd_one : p ∣ 1 := by
      have : p ∣ N - Nat.factorial n := by
        apply Nat.dvd_sub'
        · exact hp_dvd
        · exact hp_dvd_fact
      simp [N] at this
      exact this

    -- p ≤ 1，但素数 ≥ 2，矛盾
    have hp_le_one : p ≤ 1 := by
      apply Nat.le_of_dvd (by norm_num) hp_dvd_one

    have hp_ge_two : p ≥ 2 := by
      apply Nat.Prime.two_le hp_prime

    linarith

  · -- p 是素数
    exact hp_prime
```

### 案例2：Cauchy-Schwarz 不等式

**纸笔证明：**

> 考虑关于 t 的二次函数 f(t) = ∑(xᵢ + tyᵢ)² ≥ 0。
> 展开得 f(t) = (∑yᵢ²)t² + 2(∑xᵢyᵢ)t + ∑xᵢ²。
> 判别式 Δ = 4(∑xᵢyᵢ)² - 4(∑xᵢ²)(∑yᵢ²) ≤ 0。
> 因此 (∑xᵢyᵢ)² ≤ (∑xᵢ²)(∑yᵢ²)。

**形式化证明：**

```lean4
import Mathlib

open Finset Real

theorem Cauchy_Schwarz_inequality {n : Nat} (x y : Fin n → ℝ) :
    (∑ i, x i * y i)^2 ≤ (∑ i, (x i)^2) * (∑ i, (y i)^2) := by

  -- 情况1：所有 yᵢ = 0
  by_cases hy : ∀ i, y i = 0
  · -- 右边为 0，左边也为 0
    have h1 : ∑ i, x i * y i = 0 := by
      apply sum_eq_zero
      intro i _
      simp [hy i]
    have h2 : ∑ i, (y i)^2 = 0 := by
      apply sum_eq_zero
      intro i _
      simp [hy i]
    rw [h1, h2]
    simp

  · -- 情况2：存在某个 yᵢ ≠ 0
    push_neg at hy

    -- 使用二次型方法
    -- 对所有实数 t，∑(xᵢ - tyᵢ)² ≥ 0
    have hquad : ∀ t : ℝ, ∑ i, (x i - t * y i)^2 ≥ 0 := by
      intro t
      apply sum_nonneg
      intro i _
      apply sq_nonneg

    -- 展开二次型
    have hexpand : ∀ t : ℝ,
        ∑ i, (x i - t * y i)^2 =
        (∑ i, (y i)^2) * t^2 - 2 * (∑ i, x i * y i) * t + (∑ i, (x i)^2) := by
      intro t
      simp [sub_sq, mul_add, add_mul, sum_add_distrib, sum_sub_distrib, mul_assoc]
      ring

    -- 二次型 f(t) = at² + bt + c ≥ 0 对所有 t 成立
    -- 其中 a = ∑yᵢ², b = -2∑xᵢyᵢ, c = ∑xᵢ²
    let a := ∑ i, (y i)^2
    let b := -2 * (∑ i, x i * y i)
    let c := ∑ i, (x i)^2

    have ha_pos : a > 0 := by
      have : ∃ i, y i ≠ 0 := hy
      rcases this with ⟨i, hi⟩
      have : (y i)^2 > 0 := by
        apply pow_two_pos_of_ne_zero hi
      have hle : (y i)^2 ≤ a := by
        apply single_le_sum
        · intro j _
          apply sq_nonneg
        · simp
      linarith

    -- f(t) ≥ 0 对所有 t，所以判别式 ≤ 0
    -- b² - 4ac ≤ 0
    have hdiscrim : b^2 - 4 * a * c ≤ 0 := by
      -- 取 t = -b/(2a)，这是最小值点
      let t0 := -b / (2 * a)
      have hmin : a * t0^2 + b * t0 + c ≥ 0 := by
        have := hquad t0
        rw [hexpand t0] at this
        linarith
      -- 计算并证明判别式条件
      field_simp at hmin
      nlinarith [ha_pos]

    -- 展开判别式条件
    have : b^2 - 4 * a * c =
        4 * (∑ i, x i * y i)^2 - 4 * (∑ i, (y i)^2) * (∑ i, (x i)^2) := by
      simp [a, b, c]
      ring

    rw [this] at hdiscrim
    linarith
```

### 案例3：证明 e 是无理数

**纸笔证明概要：**

> 假设 e = p/q。考虑 e 的级数展开和余项估计。
> 利用 q! × e 不是整数的性质导出矛盾。

**形式化证明：**

```lean4
import Mathlib

-- e 的级数定义
def euler_series (n : Nat) : ℝ := ∑ k in Finset.range n, (1 / Nat.factorial k : ℝ)

-- e 是无理数
theorem irrational_e : Irrational Real.exp 1 := by
  -- Mathlib 中已有此定理
  apply irrational_exp_one

-- 手动证明的关键步骤
-- 证明对于足够大的 n，n! × e 接近整数但不等于整数
theorem factorial_mul_e_not_integer (n : Nat) (hn : n > 0) :
    ¬ ∃ m : ℤ, |(n.factorial : ℝ) * Real.exp 1 - m| < 1 / (n + 1) := by
  rintro ⟨m, hm⟩

  -- 将 e 写成部分和加余项
  have hexp : Real.exp 1 = euler_series (n + 1) +
      ∑ k in Finset.Ico (n + 1) (n + 2), (1 / Nat.factorial k : ℝ) +
      ∑ k in Finset.Ici (n + 2), (1 / Nat.factorial k : ℝ) := by
    sorry

  -- 前两项乘以 n! 是整数
  have hinteger : ∃ k : ℤ, (n.factorial : ℝ) * (euler_series (n + 1) +
      ∑ k in Finset.Ico (n + 1) (n + 2), (1 / Nat.factorial k : ℝ)) = k := by
    sorry

  -- 余项估计
  have hremainder : |∑ k in Finset.Ici (n + 2), (n.factorial : ℝ) * (1 / Nat.factorial k : ℝ)|
      < 1 / (n + 1) := by
    sorry

  -- 结合导出矛盾
  sorry
```

## 总结

从纸笔证明到形式化证明的关键要点：

| 方面 | 纸笔证明 | 形式化证明 |
|------|---------|-----------|
| 严格性 | 允许"显然" | 每个步骤必须验证 |
| 命名 | 随意 | 精确、描述性 |
| 结构 | 灵活 | 显式结构 |
| 边界情况 | 常忽略 | 必须处理 |
| 工具 | 直觉 | 策略和类型检查 |

**最佳实践清单：**

- [ ] 明确陈述所有假设
- [ ] 分解复杂证明为引理
- [ ] 处理所有边界情况
- [ ] 使用描述性名称
- [ ] 添加适当注释
- [ ] 验证类型正确性
- [ ] 测试示例
- [ ] 重构以提高可读性
