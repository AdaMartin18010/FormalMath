# 常见错误与调试技巧

## 常见错误

### 1. 类型不匹配

```lean
-- 错误
theorem bad : 1 = "1" := sorry
-- 修正
theorem good : 1 = 1 := rfl
```

### 2. 未解决的证明目标

使用 `show` 查看当前目标。

### 3. 假设不存在

使用 `clear` 和 `rename` 管理假设。

## 调试技巧

1. 使用 `dbg_trace` 打印信息
2. 使用 `show` 明确目标
3. 使用 `suffices` 简化目标
4. 分步证明，避免长策略链

## 性能优化

- 使用 `simp only` 代替 `simp`
- 避免不必要的 `rewrite`
- 使用 `done` 标记完成
