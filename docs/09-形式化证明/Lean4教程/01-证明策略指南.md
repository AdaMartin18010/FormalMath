# 证明策略指南

## 核心策略

### 基础策略

- `exact`: 直接应用假设
- `apply`: 应用蕴含
- `intro`: 引入假设
- `have`: 建立中间结论

### 等式推理

- `rfl`: 自反性
- `rw`: 重写
- `simp`: 简化
- `ring`: 环代数
- `field_simp`: 域代数

### 逻辑策略

- `constructor`: 分解合取
- `cases`: 案例分析
- `left`/`right`: 选择析取分支
- `by_contra`: 反证法

### 量词策略

- `use`: 提供存在量词证据
- `obtain`: 消去存在量词
- `specialize`: 特化全称量词

### 高级自动化

- `linarith`: 线性算术
- `nlinarith`: 非线性算术
- `tauto`: 命题逻辑
- `finish`: 强力自动求解

## 策略组合

```lean
example (P Q R : Prop) (h1 : P → Q) (h2 : Q → R) (hp : P) : R := by
  have hq : Q := h1 hp
  exact h2 hq
```
