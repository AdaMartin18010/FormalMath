# 各证明系统优势分析

## Lean4

### 核心优势

1. **统一的设计**: 编程和证明使用同一语言
2. **高性能**: 编译到C，执行速度快
3. **Mathlib4**: 世界上最大的数学形式化库
4. **元编程**: 强大的宏系统和策略编写能力
5. **现代化**: 最新的类型理论研究成果

### 典型案例

- 球体同伦群计算
- 完全图染色数
- 费马大定理相关结果

## Coq

### 核心优势

1. **历史悠久**: 30+年发展，工具体系成熟
2. **SSReflect**: 强大的小规模反射框架
3. **数学组件**: 深刻的代数形式化
4. **软件验证**: CompCert等工业级项目

### 典型案例

- CompCert编译器验证
- 四色定理
- Feit-Thompson奇阶定理

## Isabelle/HOL

### 核心优势

1. **自动化**: Sledgehammer等强力工具
2. **可读性**: Isar证明语言接近数学文本
3. **LCF架构**: 可靠的核心设计
4. **文档**: 详细的教程和文档

### 典型案例

- seL4操作系统内核
- Java字节码验证器
- 素数定理

## Agda

### 核心优势

1. **纯粹性**: 依赖类型的纯函数式实现
2. **Haskell生态**: 与Haskell紧密集成
3. **交互式**: 高度交互的开发环境
4. **证明即程序**:  Curry-Howard的直接体现

### 典型案例

- 范畴论库
- 类型论形式化
- 编程语言语义
