## 问题1：拉格朗日力学应用

### 问题描述

考虑双摆系统：
- 摆长 $l_1 = l_2 = l$
- 质量 $m_1 = m_2 = m$
- 广义坐标：角度 $\theta_1$, $\theta_2$

1. 写出系统的动能和势能
2. 构造拉格朗日函数
3. 推导运动方程
4. 小角度近似下线性化

### 解答

**1. 动能和势能**

位置：
- 第一摆：$(x_1, y_1) = (l\sin\theta_1, -l\cos\theta_1)$
- 第二摆：$(x_2, y_2) = (l\sin\theta_1 + l\sin\theta_2, -l\cos\theta_1 - l\cos\theta_2)$

速度平方：
$$v_1^2 = l^2\dot{\theta}_1^2$$

$$v_2^2 = l^2\dot{\theta}_1^2 + l^2\dot{\theta}_2^2 + 2l^2\dot{\theta}_1\dot{\theta}_2\cos(\theta_1 - \theta_2)$$

动能：
$$T = \frac{1}{2}ml^2\dot{\theta}_1^2 + \frac{1}{2}m[l^2\dot{\theta}_1^2 + l^2\dot{\theta}_2^2 + 2l^2\dot{\theta}_1\dot{\theta}_2\cos(\theta_1 - \theta_2)]$$

势能：
$$V = -mgl(2\cos\theta_1 + \cos\theta_2)$$

**2. 拉格朗日函数**

$$L = T - V$$

**3. Euler-Lagrange方程**

对 $\theta_1$：
$$2\ddot{\theta}_1 + \ddot{\theta}_2\cos(\theta_1 - \theta_2) + \dot{\theta}_2^2\sin(\theta_1 - \theta_2) + 2\frac{g}{l}\sin\theta_1 = 0$$

对 $\theta_2$：
$$\ddot{\theta}_2 + \ddot{\theta}_1\cos(\theta_1 - \theta_2) - \dot{\theta}_1^2\sin(\theta_1 - \theta_2) + \frac{g}{l}\sin\theta_2 = 0$$

**4. 小角度线性化**

设 $\theta_1, \theta_2 \ll 1$，保留一阶项：

$$2\ddot{\theta}_1 + \ddot{\theta}_2 + 2\frac{g}{l}\theta_1 = 0$$
$$\ddot{\theta}_1 + \ddot{\theta}_2 + \frac{g}{l}\theta_2 = 0$$

矩阵形式求解简正模式。

## 问题2：哈密顿-雅可比方程

### 问题描述

一维自由粒子，$H = p^2/(2m)$。

1. 写出哈密顿-雅可比方程
2. 求完全积分
3. 用生成函数求运动轨迹

### 解答

**1. 哈密顿-雅可比方程**

$$\frac{\partial S}{\partial t} + H(q, \partial S/\partial q) = 0$$

对自由粒子：
$$\frac{\partial S}{\partial t} + \frac{1}{2m}(\partial S/\partial x)^2 = 0$$

**2. 分离变量求完全积分**

设 $S(x,t) = W(x) - Et$

$$-E + \frac{1}{2m}(W')^2 = 0$$

$$W'(x) = \pm\sqrt{2mE} = p$$

完全积分：
$$S(x, p, t) = px - p^2t/(2m)$$

**3. 求运动**

$$\beta = \partial S/\partial E = mx/p - t$$

解得：$x = p(t+\beta)/m$，匀速直线运动。

## 问题3：谐振子代数解

### 问题描述

一维量子谐振子：$\hat{H} = \hat{p}^2/(2m) + m\omega^2\hat{x}^2/2$

1. 引入升降算子
2. 证明对易关系
3. 求能级
4. 求基态波函数

### 解答

**1. 升降算子**

$$\hat{a} = \sqrt{m\omega/(2\hbar)}(\hat{x} + i\hat{p}/(m\omega))$$
$$\hat{a}^\dagger = \sqrt{m\omega/(2\hbar)}(\hat{x} - i\hat{p}/(m\omega))$$

**2. 对易关系**

$$[\hat{a}, \hat{a}^\dagger] = 1$$

**3. 能级**

$$\hat{H} = \hbar\omega(\hat{n} + 1/2)$$

$$E_n = \hbar\omega(n + 1/2), \quad n = 0, 1, 2, ...$$

**4. 基态波函数**

由 $\hat{a}|0\rangle = 0$：

$$\psi_0(x) = (m\omega/(\pi\hbar))^{1/4} e^{-m\omega x^2/(2\hbar)}$$

## 问题4：氢原子能级

### 问题描述

氢原子哈密顿量：$\hat{H} = -\hbar^2\nabla^2/(2m) - e^2/(4\pi\epsilon_0 r)$

1. 分离变量（球坐标）
2. 求能级公式
3. 计算基态能量（eV）

### 解答

**1. 分离变量**

$\psi(r,\theta,\phi) = R(r)Y_l^m(\theta,\phi)$

**2. 能级**

$$E_n = -\frac{m}{2\hbar^2}\left(\frac{e^2}{4\pi\epsilon_0}\right)^2 \frac{1}{n^2} = -\frac{13.6 \text{ eV}}{n^2}$$

**3. 基态能量**

$E_1 = -13.6$ eV

## 问题5：自旋1/2系统

### 问题描述

自旋1/2算子 $S_i = \hbar\sigma_i/2$

1. 写出Pauli矩阵
2. 证明对易关系
3. 在 $|\\uparrow\\rangle$ 态中计算 $\\langle S_x \\rangle$ 和 $\\Delta S_x$

### 解答

**1. Pauli矩阵**

$$\\sigma_x = \\begin{pmatrix} 0 & 1 \\\\ 1 & 0 \\end{pmatrix}, \\quad \\sigma_y = \\begin{pmatrix} 0 & -i \\\\ i & 0 \\end{pmatrix}, \\quad \\sigma_z = \\begin{pmatrix} 1 & 0 \\\\ 0 & -1 \\end{pmatrix}$$

**2. 对易关系**

$$[\\sigma_x, \\sigma_y] = 2i\\sigma_z$$

**3. 测量**

$\\langle S_x \\rangle = 0$，$\\Delta S_x = \\hbar/2$

## 问题6：密度矩阵

### 问题描述

$\\rho = \\text{diag}(p, 1-p)$

1. 验证有效性
2. 计算纯度
3. 当 $p=1/2$ 时，纯态还是混合态？

### 解答

**1. 有效性**

厄米、正半定、迹为1。

**2. 纯度**

$\\text{Tr}(\\rho^2) = p^2 + (1-p)^2$

**3. $p=1/2$**

$\\text{Tr}(\\rho^2) = 1/2 < 1$，混合态。

## 问题7：配分函数计算

### 问题描述

量子谐振子系综：

1. 写出配分函数 $Z$
2. 计算热容 $C$

### 解答

**1. 配分函数**

$$Z = \\frac{e^{-\beta\\hbar\\omega/2}}{1 - e^{-\beta\\hbar\\omega}}$$

**2. 热容**

$$C = k_B \\frac{(\\beta\\hbar\\omega)^2 e^{\\beta\\hbar\\omega}}{(e^{\\beta\\hbar\\omega} - 1)^2}$$

## 问题8：理想气体熵

### 问题描述

$N$ 个单原子理想气体：

1. 用微正则系综计算熵
2. 推导Sackur-Tetrode方程

### 解答

**Sackur-Tetrode方程**

$$S = Nk_B\\left[\\ln\\left(\\frac{V}{N}\\left(\\frac{2\\pi mk_BT}{h^2}\\right)^{3/2}\\right) + \\frac{5}{2}\\right]$$

## 问题9：散射理论

### 问题描述

一维势垒 $V(x) = V_0$ ($0 < x < a$)

1. 写出波函数
2. 计算透射系数 $T$

### 解答

**透射系数**

$$T = \\left[1 + \\frac{V_0^2 \\sin^2(qa)}{4E(E-V_0)}\\right]^{-1}$$

共振时 $T = 1$。

## 问题10：对称性与生成元

### 问题描述

平移变换分析：

1. 证明幺正性
2. 求无穷小生成元
3. 与动量的关系

### 解答

**1. 幺正性**

保持内积不变。

**2. 生成元**

$G = -i\\hbar d/dx = \\hat{p}$

**3. 动量守恒**

若 $[H, \\hat{p}] = 0$，则动量守恒。
