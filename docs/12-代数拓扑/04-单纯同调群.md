# 单纯同调群

## 1. 引言

同调论是代数拓扑中研究拓扑空间"洞"的代数工具。与基本群不同，同调群是交换群（实际上是模），因而在计算上通常更为容易；同时，高维同调群能够检测任意维数的洞，而基本群只能检测一维的洞。单纯同调（simplicial homology）是同调论中最直观、历史最悠久的版本，由Poincaré在其1895年的论文*Analysis Situs*中首次提出。它通过将空间分解为最简单的几何构件（单形：点、线段、三角形、四面体等）来定义代数不变量。本文从单纯复形的基本概念出发，逐步建立链复形、边缘算子和同调群的严格理论。

## 2. 单纯复形

### 2.1 几何单形

**定义 2.1**。设 $v_0, v_1, \ldots, v_n \in \mathbb{R}^N$ 为仿射无关点（即 $v_1 - v_0, \ldots, v_n - v_0$ 线性无关）。它们张成的**闭几何单形**（geometric simplex）为
$$\sigma = [v_0, v_1, \ldots, v_n] = \left\{ \sum_{i=0}^n t_i v_i : t_i \geq 0, \sum_{i=0}^n t_i = 1 \right\}.$$
数 $n$ 称为单形的**维数**（dimension）。

**例 2.2**。0-单形是点，1-单形是闭线段，2-单形是（实心）三角形，3-单形是四面体。

### 2.2 单纯复形

**定义 2.3**。$\mathbb{R}^N$ 中的**单纯复形**（simplicial complex）$K$ 是一组单形的集合，满足：
1. $K$ 中每个单形的所有面也属于 $K$；
2. $K$ 中任意两个单形的交或者是空集，或者是它们的公共面。

$K$ 的**几何实现**（geometric realization）$|K|$ 是 $K$ 中所有单形的并，赋予子空间拓扑。

**定义 2.4**。单纯复形 $K$ 的**维数**为其单形的最大维数。

**例 2.5**。球面 $S^2$ 可实现为四面体的表面：4个顶点、6条边、4个三角形面。这称为四面体的**边界复形**。

### 2.3 定向

**定义 2.6**。$n$-单形 $[v_0, \ldots, v_n]$ 的**定向**（orientation）是其顶点的一个等价类排列：两个排列等价若相差偶置换。定向单形记作 $\langle v_0, \ldots, v_n \rangle$。

**约定**。交换两个顶点改变定向符号：$\langle v_0, v_1, v_2, \ldots \rangle = -\langle v_1, v_0, v_2, \ldots \rangle$。

## 3. 链复形与边缘算子

### 3.1 链群

**定义 3.1**。设 $K$ 为单纯复形。$K$ 的**$n$-维链群** $C_n(K)$ 是由 $K$ 的全体定向 $n$-单形生成的自由交换群，模去关系 $\sigma + \sigma' = 0$（若 $\sigma'$ 为 $\sigma$ 的反向定向）。等价地，$C_n(K)$ 是 $K$ 的 $n$-单形集合上的整数值函数（有限支撑），群运算为逐点加法。

若 $K$ 有 $k_n$ 个 $n$-单形，则 $C_n(K) \cong \mathbb{Z}^{k_n}$。

### 3.2 边缘算子

**定义 3.2**。**边缘算子**（boundary operator）$\partial_n: C_n(K) \to C_{n-1}(K)$ 在生成元上定义为
$$\partial_n(\langle v_0, \ldots, v_n \rangle) = \sum_{i=0}^n (-1)^i \langle v_0, \ldots, \hat{v}_i, \ldots, v_n \rangle,$$
其中 $\hat{v}_i$ 表示删除顶点 $v_i$，并线性延拓到整个 $C_n(K)$。

**定理 3.3**。$\partial_{n-1} \circ \partial_n = 0$（即"边缘的边缘为零"）。

*证明*。对 $n$-单形 $\sigma = \langle v_0, \ldots, v_n \rangle$，
$$\partial_{n-1}\partial_n \sigma = \sum_{j < i} (-1)^{i+j} \langle \ldots, \hat{v}_j, \ldots, \hat{v}_i, \ldots \rangle + \sum_{j > i} (-1)^{i+j-1} \langle \ldots, \hat{v}_i, \ldots, \hat{v}_j, \ldots \rangle.$$
交换指标即见两项相消。$\square$

### 3.3 链复形

**定义 3.4**。序列
$$\cdots \xrightarrow{\partial_{n+2}} C_{n+1}(K) \xrightarrow{\partial_{n+1}} C_n(K) \xrightarrow{\partial_n} C_{n-1}(K) \xrightarrow{\partial_{n-1}} \cdots$$
称为 $K$ 的**链复形**（chain complex），记作 $C_\bullet(K)$。

## 4. 同调群

### 4.1 定义

**定义 4.1**。在 $C_n(K)$ 中：
- **$n$-闭链**（$n$-cycle）：$Z_n(K) = \ker \partial_n = \{c \in C_n : \partial_n c = 0\}$；
- **$n$-边缘链**（$n$-boundary）：$B_n(K) = \operatorname{im} \partial_{n+1} = \{c \in C_n : c = \partial_{n+1} d, d \in C_{n+1}\}$。

由 $\partial^2 = 0$，$B_n(K) \subseteq Z_n(K)$。

**定义 4.2**。**$n$-维单纯同调群**：
$$H_n(K) = Z_n(K) / B_n(K) = \ker \partial_n / \operatorname{im} \partial_{n+1}.$$

元素 $[z] = z + B_n(K)$ 称为**同调类**。

### 4.2 0维同调

**命题 4.3**。$H_0(K) \cong \mathbb{Z}^{c}$，其中 $c$ 为 $|K|$ 的道路连通分支数。

*证明*。$C_0(K)$ 由顶点生成。$\partial_1$ 将边 $\langle v_0, v_1 \rangle$ 映为 $v_1 - v_0$。故 $B_0(K)$ 由 $v_1 - v_0$（相邻顶点之差）生成。$Z_0(K) = C_0(K)$（因 $\partial_0 = 0$）。$H_0 = C_0 / B_0$：同一道路连通分支中的顶点相互等价，不同分支中的顶点不等价。$\square$

### 4.3 1维同调与基本群的关系

**定理 4.4**（Hurewicz定理，$n = 1$ 情形）。若 $|K|$ 道路连通，则
$$H_1(K) \cong \pi_1(|K|)^{\mathrm{ab}} = \pi_1(|K|) / [\pi_1(|K|), \pi_1(|K|)],$$
即 $H_1$ 是 $\pi_1$ 的交换化。

## 5. 计算例子

### 5.1 圆周 $S^1$

将 $S^1$ 实现为三角形边界（3顶点 $v_0, v_1, v_2$，3边 $e_0, e_1, e_2$）。

- $C_2 = 0$，故 $B_1 = 0$。
- $Z_1 = \{n_0 e_0 + n_1 e_1 + n_2 e_2 : \partial_1(\sum n_i e_i) = 0\}$。
  $\partial_1 e_0 = v_1 - v_0$，$\partial_1 e_1 = v_2 - v_1$，$\partial_1 e_2 = v_0 - v_2$。
  $\partial_1(\sum n_i e_i) = (n_2 - n_0)v_0 + (n_0 - n_1)v_1 + (n_1 - n_2)v_2 = 0$ 当且仅当 $n_0 = n_1 = n_2$。
  故 $Z_1 = \mathbb{Z}(e_0 + e_1 + e_2) \cong \mathbb{Z}$。
- $H_1 = Z_1 / B_1 = Z_1 \cong \mathbb{Z}$。
- $H_0 \cong \mathbb{Z}$（连通）。

### 5.2 球面 $S^2$

四面体边界：4顶点，6边，4个三角形面。

- $C_3 = 0$，故 $B_2 = 0$。
- $Z_2 = \ker \partial_2$。四面体表面 $\Sigma$ 满足 $\partial_2 \Sigma = 0$（每条边恰好出现两次，方向相反）。可证 $Z_2 = \mathbb{Z}\Sigma \cong \mathbb{Z}$。
- $H_2 = Z_2 \cong \mathbb{Z}$。
- $H_1$：$Z_1 = \ker \partial_1$，$B_1 = \operatorname{im} \partial_2$。利用 $\partial_2$ 的矩阵可验证 $Z_1 = B_1$，故 $H_1 = 0$。
- $H_0 \cong \mathbb{Z}$。

### 5.3 轮胎面 $T^2$

将 $T^2$ 实现为矩形的剖分（适当三角化），或用胞腔复形：1个顶点，2条边 $a, b$，1个2-胞腔 $f$（附加映射为 $aba^{-1}b^{-1}$）。

- $H_0 \cong \mathbb{Z}$。
- $H_1$：$Z_1$ 由 $a, b$ 生成（无1维边缘），$B_1 = 0$（因 $C_2 = \mathbb{Z}f$，$\partial_2 f = a + b - a - b = 0$）。故 $H_1 = \mathbb{Z}\{a, b\} \cong \mathbb{Z}^2$。
- $H_2$：$Z_2 = \mathbb{Z}f$（$\partial_2 f = 0$），$B_2 = 0$，故 $H_2 \cong \mathbb{Z}$。

## 6. Δ-复形与胞腔同调

**定义 6.1**。$\Delta$-复形是单纯复形的推广：允许单形通过非单射的方式粘合（如将三角形的两边粘合为Möbius带）。

胞腔复形（CW复形）进一步推广：允许通过任意连续映射将 $n$-圆盘的边界粘接到 $(n-1)$-骨架上。胞腔同调的计算与单纯同调类似，但使用的胞腔数通常远少于单形数。

## 7. 与项目其他部分的关联

单纯同调是同调论最古典的形式，奇异同调（见[05-奇异同调群](05-奇异同调群.md)）将其推广到任意拓扑空间，并证明对可剖分空间两者同构。Mayer-Vietoris序列（见[06-Mayer-Vietoris序列](06-Mayer-Vietoris序列.md)）为同调群的计算提供了切割方法。上同调环（见[07-上同调环](07-上同调环.md)）在乘法结构方面比同调群更为丰富。单纯复形的概念在组合拓扑、计算拓扑以及网格生成中有直接应用。在范畴论视角（见`docs/02-代数结构/范畴论/`）下，同调是链复形范畴到交换群范畴的函子。
