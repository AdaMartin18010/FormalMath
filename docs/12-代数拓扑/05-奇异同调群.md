# 奇异同调群

## 1. 引言

单纯同调虽然直观，但其定义依赖于空间的单纯剖分，这带来了两个问题：并非所有拓扑空间都可三角剖分；不同剖分之间的比较也不够自然。奇异同调（singular homology）由Lefshetz和Eilenberg在20世纪40年代系统发展，它将同调论从可剖分空间推广到任意拓扑空间。奇异同调不再要求将空间分解为标准的单形，而是允许从标准单形到空间的"奇异"（即不必嵌入）连续映射作为基本构件。这一观点不仅使同调论成为真正的拓扑不变量理论，还揭示了同调的函子性质：连续映射诱导同调群的同态，同伦等价的映射诱导相同的同态。本文建立奇异同调的严格框架，证明其同伦不变性，并阐明与单纯同调的关系。

## 2. 奇异单形与链复形

### 2.1 标准单形

**定义 2.1**。**标准 $n$-单形**（standard $n$-simplex）为
$$\Delta^n = \{(t_0, t_1, \ldots, t_n) \in \mathbb{R}^{n+1} : t_i \geq 0, \sum_{i=0}^n t_i = 1\}.$$
这是 $\mathbb{R}^{n+1}$ 中的 $n$-维几何单形，顶点为 $e_0 = (1,0,\ldots,0)$, $e_1 = (0,1,0,\ldots,0)$, ..., $e_n = (0,\ldots,0,1)$。

### 2.2 奇异单形

**定义 2.2**。拓扑空间 $X$ 中的**奇异 $n$-单形**（singular $n$-simplex）是连续映射 $\sigma: \Delta^n \to X$。

**注记**。之所以称为"奇异"，是因为 $\sigma$ 不必是嵌入——它可以将整个 $\Delta^n$ 映为一点，或自相交。

### 2.3 奇异链群

**定义 2.3**。$X$ 的**奇异 $n$-链群** $S_n(X)$ 是由 $X$ 中所有奇异 $n$-单形生成的自由交换群。元素为形式和
$$c = \sum_{i} n_i \sigma_i, \quad n_i \in \mathbb{Z}, \sigma_i: \Delta^n \to X \text{ 连续},$$
其中只有有限多个 $n_i$ 非零。

### 2.4 面映射与边缘算子

**定义 2.4**。对 $i = 0, 1, \ldots, n$，**第 $i$ 个面映射** $F_i^n: \Delta^{n-1} \to \Delta^n$ 为将 $\Delta^{n-1}$ 仿射线性地映到 $\Delta^n$ 中删去顶点 $e_i$ 的面：
$$F_i^n(t_0, \ldots, t_{n-1}) = (t_0, \ldots, t_{i-1}, 0, t_i, \ldots, t_{n-1}).$$

**定义 2.5**。**边缘算子** $\partial_n: S_n(X) \to S_{n-1}(X)$ 在生成元上定义为
$$\partial_n(\sigma) = \sum_{i=0}^n (-1)^i \sigma \circ F_i^n,$$
并线性延拓。

**定理 2.6**。$\partial_{n-1} \circ \partial_n = 0$。

*证明*。与单纯情形类似，利用面映射的复合关系 $F_i^n \circ F_j^{n-1} = F_j^n \circ F_{i-1}^{n-1}$（$j < i$）以及符号的交错抵消。$\square$

### 2.5 奇异链复形

**定义 2.7**。
$$\cdots \to S_{n+1}(X) \xrightarrow{\partial_{n+1}} S_n(X) \xrightarrow{\partial_n} S_{n-1}(X) \to \cdots \to S_0(X) \to 0$$
称为 $X$ 的**奇异链复形**。

## 3. 同调群的定义

**定义 3.1**。奇异**$n$-闭链群**：$Z_n(X) = \ker \partial_n$。**$n$-边缘链群**：$B_n(X) = \operatorname{im} \partial_{n+1}$。**$n$-维奇异同调群**：
$$H_n(X) = Z_n(X) / B_n(X).$$

对 $A \subseteq X$，相对链复形 $S_n(X, A) = S_n(X) / S_n(A)$ 诱导**相对同调群** $H_n(X, A)$。

## 4. 函子性质与同伦不变性

### 4.1 诱导同态

**定理 4.1**。连续映射 $f: X \to Y$ 诱导链映射 $f_\sharp: S_n(X) \to S_n(Y)$，$f_\sharp(\sigma) = f \circ \sigma$，从而诱导同调群的同态
$$f_*: H_n(X) \to H_n(Y), \quad f_*([z]) = [f_\sharp(z)].$$
满足 $(g \circ f)_* = g_* \circ f_*$，$\operatorname{id}_* = \operatorname{id}$。

*证明*。需验证 $f_\sharp$ 与边缘算子交换：$\partial(f_\sharp(\sigma)) = \partial(f \circ \sigma) = \sum (-1)^i f \circ \sigma \circ F_i = f_\sharp(\partial \sigma)$。故 $f_\sharp$ 将闭链映为闭链、边缘映为边缘，从而良定义于商群。$\square$

### 4.2 同伦不变性

**定理 4.2**（同伦不变性）。若 $f \simeq g: X \to Y$，则 $f_* = g_*: H_n(X) \to H_n(Y)$。

*证明概要*。设 $H: X \times [0,1] \to Y$ 为 $f$ 到 $g$ 的同伦。构造**棱柱算子**（prism operator）$P: S_n(X) \to S_{n+1}(Y)$，将奇异单形 $\sigma: \Delta^n \to X$ 映为 $Y$ 中一个 $(n+1)$-链，对应于 $\sigma(\Delta^n) \times [0,1]$ 在 $H$ 下的像的"三角剖分"。具体地，将 $\Delta^n \times [0,1]$ 剖分为 $(n+1)$-单形，再与 $H$ 复合。

验证 $P$ 满足**链同伦**关系：
$$\partial P + P \partial = g_\sharp - f_\sharp.$$
对闭链 $z$（$\partial z = 0$），$\partial P(z) = g_\sharp(z) - f_\sharp(z)$，故 $g_\sharp(z) - f_\sharp(z)$ 为边缘，$[g_\sharp(z)] = [f_\sharp(z)]$。$\square$

**推论 4.3**。同伦等价的空间具有同构的同调群。

**推论 4.4**。可缩空间的同调群：$H_0(X) \cong \mathbb{Z}$，$H_n(X) = 0$（$n > 0$）。

### 4.3 切除定理

**定理 4.5**（切除）。设 $U \subseteq A \subseteq X$，$\bar{U} \subseteq \mathring{A}$（$U$ 的闭包在 $A$ 的内部中）。则包含映射 $(X \setminus U, A \setminus U) \hookrightarrow (X, A)$ 诱导同调群同构
$$H_n(X \setminus U, A \setminus U) \cong H_n(X, A).$$

直观上，可以"切除"内部的开集而不改变相对同调。

## 5. 计算例子

### 5.1 球面 $S^n$

**定理 5.1**。
$$H_k(S^n) = \begin{cases} \mathbb{Z}, & k = 0, n, \\ 0, & \text{其他}. \end{cases}$$

*证明*。对 $n \geq 1$，$S^n$ 道路连通，$H_0 \cong \mathbb{Z}$。对 $k > n$，$H_k = 0$（无 $k$-单形）。$H_n$：可用Mayer-Vietoris或胞腔分解证明。$\square$

### 5.2 实射影空间 $\mathbb{R}P^n$

**定理 5.2**。
$$H_k(\mathbb{R}P^n; \mathbb{Z}) = \begin{cases} \mathbb{Z}, & k = 0, \\ \mathbb{Z}/2\mathbb{Z}, & 0 < k < n, \, k \text{ 奇}, \\ 0, & \text{其他}. \end{cases}$$
且 $H_n = \mathbb{Z}$ 当 $n$ 奇，$0$ 当 $n$ 偶。

**注记**。使用 $\mathbb{Z}/2\mathbb{Z}$ 系数时，$H_k(\mathbb{R}P^n; \mathbb{Z}/2) = \mathbb{Z}/2$ 对所有 $0 \leq k \leq n$。

### 5.3 复射影空间 $\mathbb{C}P^n$

**定理 5.3**。胞腔结构为每偶数维一胞腔，故
$$H_k(\mathbb{C}P^n) = \begin{cases} \mathbb{Z}, & k = 0, 2, 4, \ldots, 2n, \\ 0, & \text{其他}. \end{cases}$$

## 6. 与单纯同调的比较

**定理 6.1**（唯一性）。对可三角剖分的空间（即 $|K| \cong X$），奇异同调与单纯同调同构：$H_n^{\text{sing}}(X) \cong H_n^{\text{simp}}(K)$。

*证明思路*。存在自然的链映射 $C_n(K) \to S_n(|K|)$（将定向单形映为特征映射），诱导同调同构。利用五项引理等工具。$\square$

## 7. 与项目其他部分的关联

奇异同调是上同调理论（见[07-上同调环](07-上同调环.md)）的基础，上同调通过对偶化链复形得到，并具有更丰富的乘法结构。Mayer-Vietoris序列（见[06-Mayer-Vietoris序列](06-Mayer-Vietoris序列.md)）为同调计算提供了有力工具。Poincaré对偶（见[08-庞加莱对偶](08-庞加莱对偶.md)）将流形的 $k$ 维同调与 $(n-k)$ 维上同调联系起来。在层论和代数几何中，层上同调是奇异上同调的深刻推广。范畴论视角（见`docs/02-代数结构/范畴论/`）下，$H_n$ 是从拓扑空间范畴到交换群范畴的协变函子。
