## 📚 概述

最优控制理论研究动态系统的最优控制策略，是现代控制理论的核心分支。它将变分法推广到控制约束情形，在航空航天、机器人、经济学中有广泛应用。

---

## 🏛️ 历史发展

### 变分法时期

**拉格朗日**和**哈密顿**的工作奠定了动力学基础。

### 现代最优控制

**Lev Pontryagin** 和苏联学派（1950s-60s）发展了极大值原理。

**Richard Bellman**（1950s）创立了动态规划方法。

**Rudolf Kalman**（1960s）解决了线性二次问题。

## 📐 问题表述

### Bolza问题

最小化性能指标：
$$J = \Phi(x(t_f), t_f) + \int_{t_0}^{t_f} L(x(t), u(t), t) dt$$

约束：
$$\dot{x} = f(x, u, t), \quad x(t_0) = x_0$$
$$u(t) \in U$$

### 特殊情形

- **Mayer问题**：$L = 0$
- **Lagrange问题**：$\Phi = 0$
- **固定端点时间** vs **自由端点时间**

## 🔍 Pontryagin极大值原理

### 定理陈述

**定理 2.1** (Pontryagin极大值原理)

定义Hamiltonian：
$$H(x, u, p, t) = L(x, u, t) + p^T f(x, u, t)$$

最优控制 $u^*(t)$、状态 $x^*(t)$、协态 $p^*(t)$ 满足：

1. **状态方程**：$\dot{x}^* = \frac{\partial H}{\partial p} = f(x^*, u^*, t)$
2. **协态方程**：$\dot{p}^* = -\frac{\partial H}{\partial x}$
3. **极大值条件**：$H(x^*, u^*, p^*, t) = \max_{u \in U} H(x^*, u, p^*, t)$
4. **横截条件**：$p^*(t_f) = \frac{\partial \Phi}{\partial x}(x^*(t_f), t_f)$

### 证明思路

变分法 + 针状变分（needle variation）证明极大值条件的本质性。

## 📊 动态规划与Hamilton-Jacobi方程

### Bellman最优性原理

**最优策略的子策略也是最优的**。

### Hamilton-Jacobi-Bellman方程

**定理 3.1** (HJB方程)

值函数 $V(x, t)$ 满足：
$$-\frac{\partial V}{\partial t} = \min_u \{L(x, u, t) + \nabla V \cdot f(x, u, t)\}$$

终端条件：$V(x, t_f) = \Phi(x, t_f)$

### 验证定理

若找到HJB的解，则可构造最优控制。

## 🔬 线性二次调节器

### 问题表述

系统：$\dot{x} = Ax + Bu$

性能指标：$J = \int_0^{\infty} (x^T Q x + u^T R u) dt$

### 最优解

**定理 4.1** (LQR解)

最优控制为状态反馈：
$$u^* = -Kx, \quad K = R^{-1}B^T P$$

其中 $P$ 满足**代数Riccati方程**：
$$A^T P + PA - PBR^{-1}B^T P + Q = 0$$

## 💻 形式化实现

```lean
import analysis.ODE
import analysis.normed_space.inner_product

namespace optimal_control

-- 系统动力学
structure control_system (n m : ℕ) :=
  (f : euclidean_space ℝ (fin n) → euclidean_space ℝ (fin m) → ℝ →
       euclidean_space ℝ (fin n))

-- Hamiltonian
def hamiltonian {n m : ℕ} (L : euclidean_space ℝ (fin n) → 
  euclidean_space ℝ (fin m) → ℝ → ℝ)
  (f : euclidean_space ℝ (fin n) → euclidean_space ℝ (fin m) → ℝ →
       euclidean_space ℝ (fin n))
  (x : euclidean_space ℝ (fin n)) (u : euclidean_space ℝ (fin m))
  (p : euclidean_space ℝ (fin n)) (t : ℝ) : ℝ :=
  L x u t + inner p (f x u t)

-- Pontryagin极大值条件
def maximum_principle_condition {n m : ℕ} {L f}
  (x u p : ℝ → euclidean_space ℝ (fin n)) (t : ℝ)
  (U : set (euclidean_space ℝ (fin m))) : Prop :=
  let H := λ u, hamiltonian L f (x t) u (p t) t in
  H (u t) = ⨆ u ∈ U, H u

-- LQR反馈增益
def lqr_gain {n m : ℕ} (A : matrix (fin n) (fin n) ℝ)
  (B : matrix (fin n) (fin m) ℝ)
  (Q : matrix (fin n) (fin n) ℝ)
  (R : matrix (fin m) (fin m) ℝ)
  (P : matrix (fin n) (fin n) ℝ) : matrix (fin m) (fin n) ℝ :=
  R⁻¹ ⬝ Bᵀ ⬝ P

-- 代数Riccati方程
def algebraic_riccati_eq {n m : ℕ} (A B Q R P : matrix (fin n) (fin n) ℝ) : Prop :=
  Aᵀ ⬝ P + P ⬝ A - P ⬝ B ⬝ R⁻¹ ⬝ Bᵀ ⬝ P + Q = 0

end optimal_control
```
