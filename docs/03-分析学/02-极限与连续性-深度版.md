## 📚 概述

极限是分析学的核心概念，连续性则是函数最基本的分析性质之一。本文档从 $\varepsilon$-$\delta$ 定义出发，系统阐述极限与连续性的理论体系，包括各种极限类型的严格定义、极限运算性质、连续函数的深层性质以及间断点分类。

## 📐 极限理论

### 数列极限

**定义 1.1** ($\varepsilon$-$N$ 定义)

数列 $(a_n)$ 收敛于 $L$，记作 $\lim_{n \to \infty} a_n = L$，如果：

$$\forall \varepsilon > 0, \exists N \in \mathbb{N}, \forall n \geq N: |a_n - L| < \varepsilon$$

**定理 1.1** (极限唯一性)

若 $\lim a_n = L_1$ 且 $\lim a_n = L_2$，则 $L_1 = L_2$。

**证明**：
假设 $L_1 \neq L_2$，取 $\varepsilon = |L_1 - L_2|/3 > 0$。

$\exists N_1: \forall n \geq N_1, |a_n - L_1| < \varepsilon$

$\exists N_2: \forall n \geq N_2, |a_n - L_2| < \varepsilon$

对 $n \geq \max(N_1, N_2)$：
$$|L_1 - L_2| \leq |L_1 - a_n| + |a_n - L_2| < 2\varepsilon = \frac{2}{3}|L_1 - L_2|$$

矛盾！故 $L_1 = L_2$。

---

### 函数极限

**定义 1.2** ($\varepsilon$-$\delta$ 定义)

$\lim_{x \to a} f(x) = L$ 定义为：

$$\forall \varepsilon > 0, \exists \delta > 0, \forall x: 0 < |x - a| < \delta \Rightarrow |f(x) - L| < \varepsilon$$

**Heine定理**（序列刻画）：

$\lim_{x \to a} f(x) = L$ 当且仅当对任意满足 $x_n \to a$ 且 $x_n \neq a$ 的序列，有 $f(x_n) \to L$。

**证明**：

($\Rightarrow$) 给定 $\varepsilon > 0$，由函数极限定义，$\exists \delta > 0$ 使得 $|x-a|<\delta \Rightarrow |f(x)-L|<\varepsilon$。

由数列极限，$\exists N: \forall n \geq N, |x_n - a| < \delta$，故 $|f(x_n) - L| < \varepsilon$。

($\Leftarrow$) 反证法。假设函数极限不成立，则：
$$\exists \varepsilon > 0, \forall \delta > 0, \exists x: 0 < |x-a| < \delta \land |f(x)-L| \geq \varepsilon$$

取 $\delta = 1/n$，得序列 $x_n \to a$ 但 $f(x_n) \not\to L$，矛盾。

---

### 单侧极限

**定义 1.3** (右极限)

$$\lim_{x \to a^+} f(x) = L \Leftrightarrow \forall \varepsilon > 0, \exists \delta > 0, \forall x: a < x < a + \delta \Rightarrow |f(x) - L| < \varepsilon$$

**定义 1.4** (左极限)

$$\lim_{x \to a^-} f(x) = L \Leftrightarrow \forall \varepsilon > 0, \exists \delta > 0, \forall x: a - \delta < x < a \Rightarrow |f(x) - L| < \varepsilon$$

**定理 1.2** (双侧极限存在准则)

$\lim_{x \to a} f(x)$ 存在当且仅当左右极限存在且相等。

---

### 无穷极限

**定义 1.5** (趋于无穷)

$$\lim_{x \to a} f(x) = +\infty \Leftrightarrow \forall M > 0, \exists \delta > 0, \forall x: 0 < |x-a| < \delta \Rightarrow f(x) > M$$

**定义 1.6** (无穷远处的极限)

$$\lim_{x \to +\infty} f(x) = L \Leftrightarrow \forall \varepsilon > 0, \exists N > 0, \forall x > N: |f(x) - L| < \varepsilon$$

## 🔗 极限运算性质

### 四则运算法则

**定理 2.1** (极限代数运算)

设 $\lim a_n = A$, $\lim b_n = B$，则：

1. $\lim (a_n + b_n) = A + B$
2. $\lim (a_n \cdot b_n) = A \cdot B$
3. $B \neq 0 \Rightarrow \lim (a_n / b_n) = A/B$

**证明 (乘积)**：

$$|a_n b_n - AB| = |a_n b_n - a_n B + a_n B - AB| \leq |a_n||b_n - B| + |B||a_n - A|$$

$(a_n)$ 收敛故有界，设 $|a_n| \leq M$。

给定 $\varepsilon > 0$，取 $N$ 使得 $n \geq N$ 时：
$$|a_n - A| < \frac{\varepsilon}{2(|B|+1)}, \quad |b_n - B| < \frac{\varepsilon}{2M}$$

则 $|a_n b_n - AB| < M \cdot \frac{\varepsilon}{2M} + |B| \cdot \frac{\varepsilon}{2(|B|+1)} < \varepsilon$。

---

### 夹逼定理

**定理 2.2** (夹逼定理/Squeeze定理)

若 $a_n \leq b_n \leq c_n$ 且 $\lim a_n = \lim c_n = L$，则 $\lim b_n = L$。

**证明**：

给定 $\varepsilon > 0$，$\exists N_1: \forall n \geq N_1, |a_n - L| < \varepsilon$，即 $L - \varepsilon < a_n < L + \varepsilon$。

$\exists N_2: \forall n \geq N_2, |c_n - L| < \varepsilon$，即 $L - \varepsilon < c_n < L + \varepsilon$。

对 $n \geq \max(N_1, N_2)$：
$$L - \varepsilon < a_n \leq b_n \leq c_n < L + \varepsilon$$

故 $|b_n - L| < \varepsilon$。

---

### Cauchy收敛准则

**定理 2.3** (Cauchy收敛准则)

数列 $(a_n)$ 收敛当且仅当：
$$\forall \varepsilon > 0, \exists N, \forall m, n \geq N: |a_m - a_n| < \varepsilon$$

**证明**：

($\Rightarrow$) 设 $\lim a_n = L$。给定 $\varepsilon > 0$，$\exists N: \forall n \geq N, |a_n - L| < \varepsilon/2$。

则 $|a_m - a_n| \leq |a_m - L| + |L - a_n| < \varepsilon$。

($\Leftarrow$) 由Cauchy条件，$(a_n)$ 有界。由Bolzano-Weierstrass定理，存在收敛子列 $a_{n_k} \to L$。

给定 $\varepsilon > 0$，$\exists N_1: \forall m, n \geq N_1, |a_m - a_n| < \varepsilon/2$。

$\exists K: \forall k \geq K, |a_{n_k} - L| < \varepsilon/2$ 且 $n_K \geq N_1$。

对 $n \geq N_1$：$|a_n - L| \leq |a_n - a_{n_K}| + |a_{n_K} - L| < \varepsilon$。

## 🔄 连续性理论

### 点连续定义

**定义 3.1** ($\varepsilon$-$\delta$ 连续)

$f$ 在点 $a$ 连续，如果：
$$\forall \varepsilon > 0, \exists \delta > 0, \forall x: |x - a| < \delta \Rightarrow |f(x) - f(a)| < \varepsilon$$

**等价刻画**：
1. $\lim_{x \to a} f(x) = f(a)$
2. $f(a^+) = f(a^-) = f(a)$
3. 任意 $x_n \to a$ 蕴含 $f(x_n) \to f(a)$

---

### 一致连续性

**定义 3.2** (一致连续)

$f$ 在区间 $I$ 上一致连续，如果：
$$\forall \varepsilon > 0, \exists \delta > 0, \forall x, y \in I: |x - y| < \delta \Rightarrow |f(x) - f(y)| < \varepsilon$$

**与点连续的区别**：$\delta$ 不依赖于点，对所有点通用。

**定理 3.1** (Cantor定理)

闭区间 $[a,b]$ 上的连续函数必一致连续。

**证明**：

反证法。设 $f$ 不一致连续，则：
$$\exists \varepsilon > 0, \forall \delta > 0, \exists x, y: |x-y| < \delta \land |f(x)-f(y)| \geq \varepsilon$$

取 $\delta = 1/n$，得序列 $x_n, y_n$ 满足 $|x_n - y_n| < 1/n$ 但 $|f(x_n) - f(y_n)| \geq \varepsilon$。

由Bolzano-Weierstrass，$x_{n_k} \to c \in [a,b]$。由 $|x_n - y_n| \to 0$，$y_{n_k} \to c$。

由连续性，$f(x_{n_k}) \to f(c)$，$f(y_{n_k}) \to f(c)$，矛盾。

---

### Lipschitz连续性

**定义 3.3** (Lipschitz连续)

$f$ 满足Lipschitz条件，如果：
$$\exists L > 0, \forall x, y: |f(x) - f(y)| \leq L|x - y|$$

**性质**：
- Lipschitz连续 $\Rightarrow$ 一致连续 $\Rightarrow$ 连续
- Lipschitz常数 $L$ 是变化率的上界
- 压缩映射：$L < 1$ 时用于Banach不动点定理

---

## 📊 连续函数性质

### 介值定理

**定理 4.1** (介值定理 / IVT)

设 $f$ 在 $[a,b]$ 连续，$f(a) < k < f(b)$（或反之），则 $\exists c \in (a,b): f(c) = k$。

**证明**（二分法）：

不妨设 $f(a) < k < f(b)$。构造区间套 $[a_n, b_n]$：

- $[a_0, b_0] = [a, b]$
- 令 $m = (a_n + b_n)/2$
  - 若 $f(m) \geq k$，则 $[a_{n+1}, b_{n+1}] = [a_n, m]$
  - 否则 $[a_{n+1}, b_{n+1}] = [m, b_n]$

则 $f(a_n) < k \leq f(b_n)$，$b_n - a_n \to 0$。

由区间套定理，$\exists! c \in \bigcap [a_n, b_n]$。

由连续性，$f(c) = \lim f(a_n) \leq k \leq \lim f(b_n) = f(c)$。

故 $f(c) = k$。

---

### 最值定理

**定理 4.2** (最值定理 / EVT)

闭区间 $[a,b]$ 上的连续函数必有最大值和最小值。

**证明**：

先证有界性。反证：设 $f$ 无上界，则 $\forall n, \exists x_n \in [a,b]: f(x_n) > n$。

由Bolzano-Weierstrass，$x_{n_k} \to c \in [a,b]$。

由连续性，$f(x_{n_k}) \to f(c)$，但 $f(x_{n_k}) > n_k \to \infty$，矛盾。

再证最大值存在。设 $M = \sup f([a,b])$。由上确界性质，$\forall n, \exists x_n: f(x_n) > M - 1/n$。

取收敛子列 $x_{n_k} \to c$，则 $f(c) = \lim f(x_{n_k}) = M$。

---

### 反函数连续性

**定理 4.3** (反函数定理)

设 $f: [a,b] \to \mathbb{R}$ 严格单调连续，则反函数 $f^{-1}$ 在其值域上连续。

**证明**（以严格递增为例）：

设 $y_0 = f(x_0)$，给定 $\varepsilon > 0$。取 $\delta_1 = f(x_0) - f(x_0 - \varepsilon) > 0$，$\delta_2 = f(x_0 + \varepsilon) - f(x_0) > 0$。

令 $\delta = \min(\delta_1, \delta_2)$。则 $|y - y_0| < \delta$ 时：
$$f(x_0 - \varepsilon) < y < f(x_0 + \varepsilon)$$

由严格递增，$x_0 - \varepsilon < f^{-1}(y) < x_0 + \varepsilon$，即 $|f^{-1}(y) - f^{-1}(y_0)| < \varepsilon$。

---

## 🧠 不连续点分类

### 第一类间断点

**定义 5.1** (第一类间断点/跳跃间断点)

左右极限都存在但不相等：
$$\lim_{x \to a^-} f(x) \neq \lim_{x \to a^+} f(x)$$

**例子**：符号函数 $\text{sgn}(x)$ 在 $x=0$ 处。

### 第二类间断点

**定义 5.2** (第二类间断点)

至少一个单侧极限不存在（为无穷或振荡）。

**例子**：$f(x) = \sin(1/x)$ 在 $x=0$ 处振荡。

### 可去间断点

**定义 5.3** (可去间断点)

极限存在但不等于函数值（或函数无定义）。

**例子**：$f(x) = \frac{\sin x}{x}$ 在 $x=0$ 处，极限为1但原函数无定义。

## 💻 形式化实现

### Lean 4实现

```lean
import topology.basic
import topology.instances.real
import analysis.normed_space.basic

namespace limit_continuity

-- 数列极限定义
def seq_limit (a : ℕ → ℝ) (L : ℝ) : Prop :=
  ∀ ε > 0, ∃ N : ℕ, ∀ n ≥ N, |a n - L| < ε

-- 函数极限定义
def func_limit (f : ℝ → ℝ) (a L : ℝ) : Prop :=
  ∀ ε > 0, ∃ δ > 0, ∀ x : ℝ, 0 < |x - a| ∧ |x - a| < δ → |f x - L| < ε

-- 连续性定义
def continuous_at (f : ℝ → ℝ) (a : ℝ) : Prop :=
  ∀ ε > 0, ∃ δ > 0, ∀ x : ℝ, |x - a| < δ → |f x - f a| < ε

-- 一致连续性定义
def uniformly_continuous (f : ℝ → ℝ) (I : set ℝ) : Prop :=
  ∀ ε > 0, ∃ δ > 0, ∀ x y ∈ I, |x - y| < δ → |f x - f y| < ε

-- 介值定理
theorem intermediate_value_theorem {f : ℝ → ℝ} {a b k : ℝ}
  (hf : continuous_on f (Icc a b)) (hle : a ≤ b)
  (hka : f a < k) (hkb : k < f b) :
  ∃ c ∈ Icc a b, f c = k :=
intermediate_value_Icc hle hf (by simp [hka]) (by simp [hkb])

-- 最值定理
theorem extreme_value_theorem {f : ℝ → ℝ} {a b : ℝ}
  (hf : continuous_on f (Icc a b)) (hle : a ≤ b) :
  ∃ xₘ xₘₐₓ ∈ Icc a b, ∀ x ∈ Icc a b, f xₘ ≤ f x ∧ f x ≤ f xₘₐₓ :=
begin
  have compact : compact (Icc a b) := is_compact_Icc,
  obtain ⟨xₘ, hxₘ, hmin⟩ := compact.exists_is_min_on (nonempty_Icc.mpr hle) hf.norm,
  obtain ⟨xₘₐₓ, hxₘₐₓ, hmax⟩ := compact.exists_is_max_on (nonempty_Icc.mpr hle) hf.norm,
  exact ⟨xₘ, hxₘ, xₘₐₓ, hxₘₐₓ, λ x hx, ⟨hmin hx, hmax hx⟩⟩,
end

end limit_continuity
```
