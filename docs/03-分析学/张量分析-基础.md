# 张量分析基础

## 张量定义

### 逆变与协变张量

逆变张量：$T^{i_1...i_p}$在坐标变换下按
$$\bar{T}^{i} = \frac{\partial \bar{x}^i}{\partial x^j} T^j$$
变换。

协变张量：$T_{i_1...i_p}$按
$$\bar{T}_i = \frac{\partial x^j}{\partial \bar{x}^i} T_j$$
变换。

### 度量张量

$g_{ij}$定义内积：$ds^2 = g_{ij} dx^i dx^j$
