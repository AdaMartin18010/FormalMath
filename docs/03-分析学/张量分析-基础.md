# 张量分析基础

张量分析是研究多重线性映射及其在坐标变换下行为的数学分支，在微分几何、广义相对论、连续介质力学与理论物理中具有核心地位。其现代观点由 Ricci 与 Levi-Civita 的绝对微分学（1900）奠定，后经 Einstein 求和约定普及。

## 1. 多重线性代数基础

### 1.1 张量的定义

**定义 1.1（张量）**. 设 $V$ 为域 $\mathbb{F}$ 上的有限维向量空间，$V^*$ 为其对偶空间。$(r,s)$ 型张量（tensor of type $(r,s)$）是多重线性映射：

$$T: \underbrace{V^* \times \cdots \times V^*}_{r} \times \underbrace{V \times \cdots \times V}_{s} \to \mathbb{F}.$$

全体 $(r,s)$ 型张量构成向量空间，记为 $T^r_s(V)$，维数为 $(\dim V)^{r+s}$。

**定义 1.2（张量积）**. 设 $S \in T^{r_1}_{s_1}(V)$，$T \in T^{r_2}_{s_2}(V)$。其张量积 $S \otimes T \in T^{r_1+r_2}_{s_1+s_2}(V)$ 定义为：

$$(S \otimes T)(\omega^1, \dots, \omega^{r_1+r_2}, v_1, \dots, v_{s_1+s_2})$$

$$= S(\omega^1, \dots, \omega^{r_1}, v_1, \dots, v_{s_1}) \cdot T(\omega^{r_1+1}, \dots, \omega^{r_1+r_2}, v_{s_1+1}, \dots, v_{s_1+s_2}).$$

**命题 1.3**. 张量积满足结合律与分配律，但一般不满足交换律：$S \otimes T \neq T \otimes S$。

### 1.2 基与分量表示

设 $\{e_i\}_{i=1}^n$ 为 $V$ 的基，$\{\varepsilon^i\}_{i=1}^n$ 为 $V^*$ 的对偶基（即 $\varepsilon^i(e_j) = \delta^i_j$）。则 $T^r_s(V)$ 的基为：

$$\{ e_{i_1} \otimes \cdots \otimes e_{i_r} \otimes \varepsilon^{j_1} \otimes \cdots \otimes \varepsilon^{j_s} \},$$

任意 $(r,s)$ 型张量 $T$ 可唯一展开为：

$$T = T^{i_1 \dots i_r}_{j_1 \dots j_s} \, e_{i_1} \otimes \cdots \otimes e_{i_r} \otimes \varepsilon^{j_1} \otimes \cdots \otimes \varepsilon^{j_s}.$$

系数 $T^{i_1 \dots i_r}_{j_1 \dots j_s}$ 称为 $T$ 在基 $\{e_i\}$ 下的**分量**（components）。

## 2. 坐标变换与指标升降

### 2.1 逆变与协变变换

设 $\{\bar{e}_i\}$ 为另一组基，变换矩阵为 $A = (a^i_j)$，即 $\bar{e}_j = a^i_j e_i$。设逆矩阵为 $A^{-1} = (b^i_j)$，则对偶基变换为 $\bar{\varepsilon}^j = b^j_i \varepsilon^i$。

**定理 2.1（变换法则）**. 张量分量在新基下的变换为：

$$\bar{T}^{i_1 \dots i_r}_{j_1 \dots j_s} = b^{i_1}_{k_1} \cdots b^{i_r}_{k_r} \, a^{l_1}_{j_1} \cdots a^{l_s}_{j_s} \, T^{k_1 \dots k_r}_{l_1 \dots l_s}.$$

**定义 2.2**. 上指标（contravariant indices）按 $b^i_j$（逆矩阵）变换，称为**逆变指标**；下指标（covariant indices）按 $a^i_j$（原矩阵）变换，称为**协变指标**。

*证明*. 直接计算：

$$T(\bar{\varepsilon}^{i_1}, \dots, \bar{\varepsilon}^{i_r}, \bar{e}_{j_1}, \dots, \bar{e}_{j_s})$$

$$= T(b^{i_1}_{k_1}\varepsilon^{k_1}, \dots, b^{i_r}_{k_r}\varepsilon^{k_r}, a^{l_1}_{j_1}e_{l_1}, \dots, a^{l_s}_{j_s}e_{l_s})$$

$$= b^{i_1}_{k_1} \cdots b^{i_r}_{k_r} a^{l_1}_{j_1} \cdots a^{l_s}_{j_s} T(\varepsilon^{k_1}, \dots, \varepsilon^{k_r}, e_{l_1}, \dots, e_{l_s}).$$

$\square$

### 2.2 度量张量与指标升降

**定义 2.3（度量张量）**. 非退化对称双线性型 $g: V \times V \to \mathbb{F}$ 称为度量张量。在基 $\{e_i\}$ 下，$g_{ij} = g(e_i, e_j)$。

非退化性等价于矩阵 $(g_{ij})$ 可逆，其逆记为 $(g^{ij})$，满足 $g^{ik}g_{kj} = \delta^i_j$。

**定义 2.4（指标升降）**. 利用度量张量可定义指标的升降：

- 升指标：$T^{i \dots} = g^{ij} T_{j \dots}$
- 降指标：$T_{i \dots} = g_{ij} T^{j \dots}$

**命题 2.5**. 指标升降保持张量类型。例如，若 $v \in V$ 的分量为 $v^i$，则 $v_i = g_{ij}v^j$ 定义了 $V^*$ 中的元素。

## 3. 缩并与对称化

### 3.1 缩并

**定义 3.1（缩并）**. 对 $(r,s)$ 型张量 $T$，选定一个上指标和一个下指标进行缩并（contraction），得到 $(r-1, s-1)$ 型张量。例如：

$$(\operatorname{tr}_{(1,1)} T)^{i_2 \dots i_r}_{j_2 \dots j_s} = T^{k i_2 \dots i_r}_{k j_2 \dots j_s}.$$

**定理 3.2**. 缩并的结果与坐标系的选择无关，即缩并是良定义的 $GL(V)$-等变映射 $T^r_s(V) \to T^{r-1}_{s-1}(V)$。

*证明*. 在新基下：

$$\bar{T}^{k i_2 \dots}_{k j_2 \dots} = b^k_{l_1} a^{m_1}_k b^{i_2}_{l_2} \cdots a^{n_2}_{j_2} \cdots T^{l_1 l_2 \dots}_{m_1 n_2 \dots}$$

$$= \delta^{m_1}_{l_1} b^{i_2}_{l_2} \cdots a^{n_2}_{j_2} \cdots T^{l_1 l_2 \dots}_{m_1 n_2 \dots} = b^{i_2}_{l_2} \cdots a^{n_2}_{j_2} \cdots T^{l_1 l_2 \dots}_{l_1 n_2 \dots}.$$

恰为缩并张量的正确变换律。$\square$

### 3.2 对称化与交错化

**定义 3.3**. 对协变张量 $T_{i_1 \dots i_s}$，对称化（symmetrization）与交错化（alternation）定义为：

$$\operatorname{Sym}(T)_{i_1 \dots i_s} = \frac{1}{s!} \sum_{\sigma \in S_s} T_{i_{\sigma(1)} \dots i_{\sigma(s)}},$$

$$\operatorname{Alt}(T)_{i_1 \dots i_s} = \frac{1}{s!} \sum_{\sigma \in S_s} \operatorname{sgn}(\sigma) T_{i_{\sigma(1)} \dots i_{\sigma(s)}}.$$

## 4. 微分流形上的张量场

### 4.1 切丛与张量丛

设 $M$ 为光滑流形，$p \in M$。切空间 $T_p M$ 的余切空间为 $T_p^* M$。

**定义 4.1（张量场）**. $(r,s)$ 型张量场是光滑映射 $T: M \to \bigsqcup_{p \in M} T^r_s(T_p M)$ 使得 $T(p) \in T^r_s(T_p M)$。

局部坐标 $(x^1, \dots, x^n)$ 下，张量场表示为：

$$T = T^{i_1 \dots i_r}_{j_1 \dots j_s}(x) \, \frac{\partial}{\partial x^{i_1}} \otimes \cdots \otimes \frac{\partial}{\partial x^{i_r}} \otimes dx^{j_1} \otimes \cdots \otimes dx^{j_s}.$$

### 4.2 协变导数

**定义 4.2（仿射联络）**. 光滑流形 $M$ 上的仿射联络（affine connection）是映射 $\nabla: \mathfrak{X}(M) \times \mathfrak{X}(M) \to \mathfrak{X}(M)$，记 $\nabla_X Y$，满足：
1. $\nabla_{fX + gZ} Y = f\nabla_X Y + g\nabla_Z Y$（$C^\infty$-线性于 $X$）；
2. $\nabla_X (Y + Z) = \nabla_X Y + \nabla_X Z$；
3. $\nabla_X (fY) = X(f) Y + f\nabla_X Y$（Leibniz 法则）。

**定义 4.3（Christoffel 符号）**. 在坐标系下，$\nabla_{\partial_i} \partial_j = \Gamma^k_{ij} \partial_k$。系数 $\Gamma^k_{ij}$ 称为 Christoffel 符号。

**定理 4.4（Levi-Civita 联络）**. 设 $(M, g)$ 为 Riemann 流形，则存在唯一的无挠、与度量相容的联络，称为 Levi-Civita 联络，其 Christoffel 符号为：

$$\Gamma^k_{ij} = \frac{1}{2} g^{kl} \left( \partial_i g_{jl} + \partial_j g_{il} - \partial_l g_{ij} \right).$$

*证明*. 无挠性要求 $\Gamma^k_{ij} = \Gamma^k_{ji}$。度量相容性要求 $\nabla_k g_{ij} = 0$，即：

$$\partial_k g_{ij} = \Gamma^l_{ki} g_{lj} + \Gamma^l_{kj} g_{il}.$$

轮换指标得三个方程，适当加减可解出上述公式。唯一性由方程组的解唯一确定。$\square$

## 5. 例子

### 5.1 例子：应力张量

在连续介质力学中，Cauchy 应力张量 $\sigma$ 为 $(2,0)$ 型（或 $(1,1)$ 型）张量，描述材料内部的面力分布。在正交坐标系下：

$$\sigma = \sigma^{ij} e_i \otimes e_j,$$

其中 $\sigma^{ij}$ 为应力分量。平衡方程（无体力）为：

$$\nabla_j \sigma^{ij} = 0.$$

对各向同性线弹性材料，广义 Hooke 定律给出：

$$\sigma^{ij} = \lambda \delta^{ij} \varepsilon^k_k + 2\mu \varepsilon^{ij},$$

其中 $\varepsilon$ 为应变张量，$\lambda, \mu$ 为 Lamé 常数。

### 5.2 例子：Riemann 曲率张量

Riemann 曲率张量 $R$ 是 $(1,3)$ 型张量：

$$R(X, Y)Z = \nabla_X \nabla_Y Z - \nabla_Y \nabla_X Z - \nabla_{[X,Y]} Z.$$

在坐标下：

$$R^l_{ijk} = \partial_i \Gamma^l_{jk} - \partial_j \Gamma^l_{ik} + \Gamma^l_{im}\Gamma^m_{jk} - \Gamma^l_{jm}\Gamma^m_{ik}.$$

第一 Bianchi 恒等式：

$$R^l_{ijk} + R^l_{jki} + R^l_{kij} = 0.$$

Ricci 张量由缩并得到：$R_{ij} = R^k_{ikj}$，标量曲率为 $R = g^{ij}R_{ij}$。
