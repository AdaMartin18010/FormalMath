# 张量分析基础

张量分析是研究多重线性映射及其在坐标变换下行为的数学分支，在微分几何、广义相对论、连续介质力学与理论物理中具有核心地位。其现代观点由 Ricci 与 Levi-Civita 的绝对微分学（1900）奠定，后经 Einstein 求和约定普及。本节从多重线性代数出发，经协变导数到 Riemann 几何，建立完整的张量计算框架。

## 1. 多重线性代数基础

### 1.1 张量的定义

**定义 1.1（张量）**. 设 $V$ 为域 $\mathbb{F}$ 上的有限维向量空间，$V^*$ 为其对偶空间。$(r,s)$ 型张量（tensor of type $(r,s)$）是多重线性映射：

$$T: \underbrace{V^* \times \cdots \times V^*}_{r} \times \underbrace{V \times \cdots \times V}_{s} \to \mathbb{F}.$$

全体 $(r,s)$ 型张量构成向量空间，记为 $T^r_s(V)$，维数为 $(\dim V)^{r+s}$。

**定义 1.2（张量积）**. 对 $S \in T^{r_1}_{s_1}(V)$，$T \in T^{r_2}_{s_2}(V)$，其张量积 $S \otimes T \in T^{r_1+r_2}_{s_1+s_2}(V)$ 定义为：

$$(S \otimes T)(\omega^1, \dots, \omega^{r_1+r_2}, v_1, \dots, v_{s_1+s_2}) = S(\omega^1, \dots, v_{s_1}) T(\omega^{r_1+1}, \dots, v_{s_1+s_2}).$$

### 1.2 基变换

设 $\{e_i\}$ 为 $V$ 的基，$\{\varepsilon^i\}$ 为 $V^*$ 的对偶基（$\varepsilon^i(e_j) = \delta^i_j$）。$(r,s)$ 型张量 $T$ 的分量为：

$$T^{i_1 \dots i_r}_{j_1 \dots j_s} = T(\varepsilon^{i_1}, \dots, \varepsilon^{i_r}, e_{j_1}, \dots, e_{j_s}).$$

在新基 $\tilde{e}_i = A^j_i e_j$ 下：

$$\tilde{T}^{i_1 \dots i_r}_{j_1 \dots j_s} = (A^{-1})^{i_1}_{k_1} \cdots (A^{-1})^{i_r}_{k_r} A^{l_1}_{j_1} \cdots A^{l_s}_{j_s} T^{k_1 \dots k_r}_{l_1 \dots l_s}.$$

## 2. 协变导数与联络

### 2.1 仿射联络

**定义 2.1（仿射联络）**. 流形 $M$ 上的仿射联络是映射 $\nabla: \mathfrak{X}(M) \times \mathfrak{X}(M) \to \mathfrak{X}(M)$，$(X, Y) \mapsto \nabla_X Y$，满足：
1. $\nabla_{fX+gY} Z = f\nabla_X Z + g\nabla_Y Z$；
2. $\nabla_X(Y+Z) = \nabla_X Y + \nabla_X Z$；
3. $\nabla_X(fY) = X(f)Y + f\nabla_X Y$（Leibniz 法则）。

**定义 2.2（Christoffel 符号）**. 在坐标基 $\{\partial_i\}$ 下，$\nabla_{\partial_i} \partial_j = \Gamma^k_{ij} \partial_k$。

### 2.2 张量的协变导数

**定义 2.3**. 对 $(r,s)$ 型张量场 $T$，其协变导数 $\nabla T$ 为 $(r,s+1)$ 型张量：

$$(\nabla T)(\omega^1, \dots, \omega^r, v_1, \dots, v_s; X) = \nabla_X T(\omega^1, \dots, v_s).$$

对分量：

$$\nabla_k T^{i_1 \dots i_r}_{j_1 \dots j_s} = \partial_k T^{i_1 \dots i_r}_{j_1 \dots j_s} + \sum_{m=1}^r \Gamma^{i_m}_{kl} T^{i_1 \dots l \dots i_r}_{j_1 \dots j_s} - \sum_{m=1}^s \Gamma^l_{k j_m} T^{i_1 \dots i_r}_{j_1 \dots l \dots j_s}.$$

## 3. Riemann 几何中的曲率

### 3.1 Riemann 曲率张量

**定义 3.1（Riemann 曲率张量）**. 对联络 $\nabla$，曲率张量为：

$$R(X, Y)Z = \nabla_X \nabla_Y Z - \nabla_Y \nabla_X Z - \nabla_{[X,Y]} Z.$$

在坐标下：$R^l_{ijk} = \partial_i \Gamma^l_{jk} - \partial_j \Gamma^l_{ik} + \Gamma^l_{im}\Gamma^m_{jk} - \Gamma^l_{jm}\Gamma^m_{ik}$。

### 3.2 对称性与 Bianchi 恒等式

**定理 3.2（代数对称性）**. $R_{lijk} = g_{lm}R^m_{ijk}$ 满足：
1. $R_{lijk} = -R_{lijk}$（前两个指标反对称）；
2. $R_{lijk} = -R_{likj}$（后两个指标反对称）；
3. $R_{lijk} = R_{jkli}$（前后对交换）；
4. $R_{lijk} + R_{lkij} + R_{ljki} = 0$（第一 Bianchi 恒等式）。

**定理 3.3（第二 Bianchi 恒等式）**. $\nabla_m R^l_{ijk} + \nabla_j R^l_{imk} + \nabla_k R^l_{ijm} = 0$。

*证明*. 在正规坐标系（$\Gamma$ 在某点为零）下直接计算：

$$\nabla_m R^l_{ijk} = \partial_m \partial_i \Gamma^l_{jk} - \partial_m \partial_j \Gamma^l_{ik}.$$

轮换 $m, i, j$ 并相加，混合偏导数相消。$\square$

### 3.3 Ricci 曲率与标量曲率

**定义 3.4**. Ricci 曲率张量 $R_{ij} = R^k_{ikj}$，标量曲率 $R = g^{ij}R_{ij}$。

**定理 3.5（Ricci 恒等式）**. $\nabla_i R^i_j = \frac{1}{2}\nabla_j R$。

*证明*. 由第二 Bianchi 恒等式，缩并 $l$ 与 $i$：

$$\nabla_m R_{jk} - \nabla_j R_{mk} + \nabla_k R^l_{jlm} = 0.$$

再缩并 $m$ 与 $k$：$\nabla^k R_{jk} - \nabla_j R + \nabla^l R_{lj} = 0$，即 $2\nabla^i R_{ij} = \nabla_j R$。$\square$

## 4. 例子

### 4.1 例子：球面 $S^2$ 的曲率

半径为 $a$ 的球面，度规 $ds^2 = a^2(d\theta^2 + \sin^2\theta \, d\phi^2)$。

非零 Christoffel 符号：$\Gamma^\theta_{\phi\phi} = -\sin\theta\cos\theta$，$\Gamma^\phi_{\theta\phi} = \cot\theta$。

Riemann 曲率：$R^\theta_{\phi\theta\phi} = \sin^2\theta$，$R_{\theta\phi\theta\phi} = a^2\sin^2\theta$。

截面曲率：$K = \frac{R_{\theta\phi\theta\phi}}{g_{\theta\theta}g_{\phi\phi} - g_{\theta\phi}^2} = \frac{a^2\sin^2\theta}{a^2 \cdot a^2\sin^2\theta} = \frac{1}{a^2}$。

Ricci 曲率：$R_{\theta\theta} = 1/a^2$，$R_{\phi\phi} = \sin^2\theta/a^2$。标量曲率 $R = 2/a^2$。

### 4.2 例子：Schwarzschild 度规

广义相对论中球对称质量 $M$ 外部的度规：

$$ds^2 = -\left(1 - \frac{2GM}{r}\right)dt^2 + \left(1 - \frac{2GM}{r}\right)^{-1}dr^2 + r^2(d\theta^2 + \sin^2\theta \, d\phi^2).$$

计算非零 Christoffel 符号（如 $\Gamma^r_{tt} = \frac{GM}{r^2}(1-\frac{2GM}{r})$），进而可得 Ricci 张量 $R_{\mu\nu} = 0$（真空 Einstein 方程）。
