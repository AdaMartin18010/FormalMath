## 📚 概述

小波分析是20世纪80年代兴起的数学工具，它克服了傅里叶分析在时频局部化方面的局限。小波变换能同时在时间和频率上提供局部化信息，在信号处理、图像压缩、数值分析等领域有革命性应用。

---

## 🏛️ 历史发展

### 傅里叶分析的局限

**约瑟夫·傅里叶**的变换提供全局频率信息，但无法定位时间/空间上的局部特征。

### 窗口傅里叶变换

**丹尼斯·加博**（Dennis Gabor, 1946）引入了高斯窗，实现了初步的时频局部化。

### 小波的诞生

- **1981**: Jean Morlet 在地震信号分析中提出"wavelet"
- **1984**: Alex Grossmann 与 Morlet 建立数学理论
- **1986**: Yves Meyer 构造出光滑正交小波
- **1988**: Ingrid Daubechies 构造出紧支集正交小波
- **1989**: Stéphane Mallat 建立多分辨分析框架

## 📐 连续小波变换

### 母小波

**定义 1.1** (容许小波)

函数 $\psi \in L^2(\mathbb{R})$ 称为**母小波**，若满足**容许条件**：
$$C_\psi = \int_{-\infty}^{\infty} \frac{|\hat{\psi}(\omega)|^2}{|\omega|} d\omega < \infty$$

这意味着 $\hat{\psi}(0) = 0$，即 $\int_{-\infty}^{\infty} \psi(t) dt = 0$。

### 小波变换定义

**定义 1.2** (连续小波变换)

$$W_\psi f(a, b) = \frac{1}{\sqrt{|a|}} \int_{-\infty}^{\infty} f(t) \overline{\psi\left(\frac{t-b}{a}\right)} dt$$

其中 $a \neq 0$（尺度参数），$b \in \mathbb{R}$（平移参数）。

### 重构公式

**定理 1.1** (逆小波变换)

$$f(t) = \frac{1}{C_\psi} \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} W_\psi f(a, b) \psi_{a,b}(t) \frac{da db}{a^2}$$

## 🔍 离散小波变换

### 二进小波

取 $a = 2^j$，$b = k \cdot 2^j$（$j, k \in \mathbb{Z}$）：
$$\psi_{j,k}(t) = 2^{-j/2} \psi(2^{-j}t - k)$$

### 框架与Riesz基

**定义 2.1** (小波框架)

若存在 $A, B > 0$ 使得：
$$A\|f\|^2 \leq \sum_{j,k} |\langle f, \psi_{j,k} \rangle|^2 \leq B\|f\|^2$$

则 $\{\psi_{j,k}\}$ 构成框架。若 $A = B = 1$ 且正交，则为标准正交基。

## 📊 多分辨分析

### 定义

**定义 3.1** (多分辨分析, MRA)

闭子空间序列 $\{V_j\}_{j \in \mathbb{Z}}$ 满足：
1. $V_j \subseteq V_{j+1}$
2. $\overline{\bigcup_j V_j} = L^2(\mathbb{R})$
3. $\bigcap_j V_j = \{0\}$
4. $f(t) \in V_j \Leftrightarrow f(2t) \in V_{j+1}$
5. $f(t) \in V_0 \Leftrightarrow f(t-k) \in V_0$
6. 存在 $\phi \in V_0$ 使得 $\{\phi(t-k)\}$ 构成 $V_0$ 的Riesz基

### 尺度函数与小波

**尺度函数** $\phi$ 满足**双尺度方程**：
$$\phi(t) = \sum_{k} h_k \phi(2t - k)$$

**小波函数**：
$$\psi(t) = \sum_{k} g_k \phi(2t - k)$$

其中 $g_k = (-1)^k \overline{h_{1-k}}$。

## 🔬 重要小波

### Haar小波

最早、最简单的小波：
$$\psi(t) = \begin{cases} 1 & 0 \leq t < 1/2 \\ -1 & 1/2 \leq t < 1 \\ 0 & \text{其他} \end{cases}$$

### Daubechies小波

紧支集正交小波，具有最大光滑度。记为 dbN（N为消失矩阶数）。

### Meyer小波

频率域定义的无限可微小波。

### Morlet小波

复值小波：$\psi(t) = e^{i\omega_0 t} e^{-t^2/2}$

## 💻 形式化实现

```lean
import measure_theory.function.l2_space
import analysis.fourier

namespace wavelet_analysis

-- 连续小波变换
def cwt (ψ : ℝ → ℂ) (f : ℝ → ℂ) (a b : ℝ) : ℂ :=
  let ψ_ab := λ t, real.sqrt (|a|)⁻¹ * ψ ((t - b) / a) in
  ∫ t, f t * conj (ψ_ab t)

-- 容许条件
def admissible (ψ : ℝ → ℂ) : Prop :=
  let ψ_hat := fourier_transform ψ in
  ∫ ω, ‖ψ_hat ω‖^2 / |ω| < ∞

-- 离散小波变换（Mallat算法）
def dwt_decomposition {j : ℕ} (c_j : ℤ → ℝ) (h g : ℤ → ℝ) :
  (ℤ → ℝ) × (ℤ → ℝ) :=
  let c_j1 := λ k, ∑ n, h (n - 2*k) * c_j n in
  let d_j1 := λ k, ∑ n, g (n - 2*k) * c_j n in
  (c_j1, d_j1)

def dwt_reconstruction (c_j1 d_j1 : ℤ → ℝ) (h g : ℤ → ℝ) : ℤ → ℝ :=
  λ n, ∑ k, h (n - 2*k) * c_j1 k + ∑ k, g (n - 2*k) * d_j1 k

-- Haar小波
def haar_wavelet : ℝ → ℝ :=
  λ t, if 0 ≤ t ∧ t < 1/2 then 1 else if 1/2 ≤ t ∧ t < 1 then -1 else 0

end wavelet_analysis
```
