# 凸分析基础

## 凸集与凸函数

### 凸集

集合$C$是凸的若对任意$x, y \in C$，$\lambda \in [0,1]$：
$$\lambda x + (1-\lambda)y \in C$$

### 凸函数

$f: C \to \mathbb{R}$是凸的若：
$$f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y)$$

### 次梯度

凸函数$f$在$x$处的次梯度$g$满足：
$$f(y) \geq f(x) + g^T(y-x), \quad \forall y$$
