# 变分法 - 深度版

## 1. 引言

### 1.1 什么是变分法？

**变分法**（Calculus of Variations）研究**泛函**的极值问题。与经典微积分中求函数 $f: \mathbb{R}^n \to \mathbb{R}$ 的极值不同，变分法处理的是定义在函数空间上的泛函 $J: \mathcal{F} \to \mathbb{R}$ 的极值。

**核心问题**：给定泛函
$$J[y] = \int_a^b F(x, y(x), y'(x))\, dx$$
求使 $J[y]$ 取极值的函数 $y(x)$，满足边界条件 $y(a) = A$，$y(b) = B$。

### 1.2 历史发展

| 年份 | 数学家 | 贡献 |
|------|--------|------|
| 1696 | 约翰·伯努利 | 提出最速降线问题 |
| 1744 | 欧拉 | 导出极值的必要条件（Euler-Lagrange方程） |
| 1755 | 拉格朗日 | 引入变分方法，被欧拉命名 |

## 2. Euler-Lagrange方程

### 2.1 基本引理

**引理 2.1**（变分法基本引理）
设 $f \in C([a,b])$，若对任意满足 $\eta(a) = \eta(b) = 0$ 的 $\eta \in C^\infty([a,b])$ 都有
$$\int_a^b f(x)\eta(x)\, dx = 0$$
则 $f(x) \equiv 0$ 在 $[a,b]$ 上。

**证明**：
假设存在 $x_0 \in (a,b)$ 使 $f(x_0) > 0$。由连续性，存在区间 $[c,d] \subset (a,b)$ 使得 $f(x) > 0$ 对所有 $x \in [c,d]$ 成立。

构造测试函数：
$$\eta(x) = \begin{cases} \exp\left(-\frac{1}{(x-c)(d-x)}\right) & x \in (c,d) \\ 0 & \text{否则} \end{cases}$$

则 $\eta \in C^\infty([a,b])$，$\eta(a) = \eta(b) = 0$，且
$$\int_a^b f(x)\eta(x)\, dx = \int_c^d f(x)\eta(x)\, dx > 0$$
与假设矛盾。故 $f \equiv 0$。 $\square$

### 2.2 Euler-Lagrange方程的推导

**定理 2.1**（Euler-Lagrange方程）
设 $F \in C^2([a,b] \times \mathbb{R} \times \mathbb{R})$，若 $y \in C^2([a,b])$ 使泛函
$$J[y] = \int_a^b F(x, y, y')\, dx$$
在固定端点 $y(a) = A$，$y(b) = B$ 下取局部极值，则 $y$ 满足：
$$\frac{\partial F}{\partial y} - \frac{d}{dx}\frac{\partial F}{\partial y'} = 0$$

**证明**：
设 $y$ 是极值函数，考虑单参数变分 $y_\epsilon(x) = y(x) + \epsilon\eta(x)$，其中 $\eta(a) = \eta(b) = 0$。

定义 $\Phi(\epsilon) = J[y_\epsilon]$。由于 $y$ 是极值，$\Phi'(0) = 0$。

计算：
$$\Phi'(\epsilon) = \int_a^b \left[\frac{\partial F}{\partial y}\eta + \frac{\partial F}{\partial y'}\eta'\right] dx$$

对第二项分部积分：
$$\int_a^b \frac{\partial F}{\partial y'}\eta'\, dx = \left.\frac{\partial F}{\partial y'}\eta\right|_a^b - \int_a^b \frac{d}{dx}\frac{\partial F}{\partial y'}\eta\, dx = -\int_a^b \frac{d}{dx}\frac{\partial F}{\partial y'}\eta\, dx$$

因此：
$$\Phi'(0) = \int_a^b \left[\frac{\partial F}{\partial y} - \frac{d}{dx}\frac{\partial F}{\partial y'}\right]\eta\, dx = 0$$

由变分法基本引理，方括号内恒为零：
$$\frac{\partial F}{\partial y} - \frac{d}{dx}\frac{\partial F}{\partial y'} = 0 \quad \square$$

### 2.3 首次积分

**定理 2.2**（首次积分）
若 $F$ 不显含 $x$，则Euler-Lagrange方程有首次积分：
$$F - y'\frac{\partial F}{\partial y'} = C \quad \text{(常数)}$$

**证明**：
计算 $\frac{d}{dx}\left(F - y'F_{y'}\right)$：
$$\frac{dF}{dx} = F_x + F_y y' + F_{y'}y''$$
$$\frac{d}{dx}(y'F_{y'}) = y''F_{y'} + y'\frac{d}{dx}F_{y'}$$

相减：
$$\frac{d}{dx}\left(F - y'F_{y'}\right) = F_x + y'\left(F_y - \frac{d}{dx}F_{y'}\right) = F_x$$

当 $F_x = 0$ 时，上式为零，故 $F - y'F_{y'}$ 为常数。 $\square$

## 3. 经典变分问题

### 3.1 最速降线问题

**问题**：求连接 $(0,0)$ 和 $(a,b)$（$b > 0$）的曲线，使质点在重力作用下沿其下滑时间最短。

**解**：
时间泛函：
$$T[y] = \int_0^a \frac{\sqrt{1 + (y')^2}}{\sqrt{2gy}}\, dx$$

这里 $F(y, y') = \frac{\sqrt{1+(y')^2}}{\sqrt{2gy}}$ 不显含 $x$，使用首次积分：
$$F - y'F_{y'} = \frac{1}{\sqrt{2gy}\sqrt{1+(y')^2}} = C$$

解得：
$$y(1+(y')^2) = \frac{1}{2gC^2} = 2R \quad \text{(设)}$$

参数化解（令 $y' = \cot\theta$）：
$$\begin{cases} x = R(\theta - \sin\theta) \\ y = R(1 - \cos\theta) \end{cases}$$

这是**摆线**（旋轮线），$R$ 由端点条件确定。 $\square$

### 3.2 等周问题

**问题**：在固定周长 $L$ 下，求围成最大面积的平面闭曲线。

**解**：
用变分法（Lagrange乘子），面积 $A = \frac{1}{2}\oint(x\,dy - y\,dx)$，约束 $\oint\sqrt{(dx)^2+(dy)^2} = L$。

**结论**：解为**圆**，最大面积 $A_{max} = \frac{L^2}{4\pi}$。

## 4. 二阶变分与极值充分条件

### 4.1 Legendre条件

**定理 4.1**（Legendre必要条件）
若 $y$ 使 $J[y]$ 取弱极小值，则在极值曲线上：
$$F_{y'y'}(x, y(x), y'(x)) \geq 0$$

### 4.2 Jacobi条件

**定义 4.1**（共轭点）
设 $y(x, \alpha, \beta)$ 是Euler-Lagrange方程依赖于两个参数的解族。若存在 $x^* > a$ 使得
$$\left.\frac{\partial y}{\partial\alpha}\right|_{\alpha=\alpha_0, \beta=\beta_0}(x^*) = 0$$
则称 $x^*$ 为 $a$ 的**共轭点**。

**定理 4.2**（Jacobi必要条件）
若 $y$ 使 $J[y]$ 取弱极小值，则开区间 $(a,b)$ 内不存在 $a$ 的共轭点。

## 5. 约束变分问题

### 5.1 Lagrange乘子法

**定理 5.1**
在等周约束 $K[y] = \int_a^b G(x,y,y')\,dx = c$ 下求 $J[y]$ 的极值，构造：
$$\mathcal{L}[y] = J[y] - \lambda K[y] = \int_a^b (F - \lambda G)\,dx$$

则极值函数满足带乘子的Euler-Lagrange方程：
$$\frac{\partial(F-\lambda G)}{\partial y} - \frac{d}{dx}\frac{\partial(F-\lambda G)}{\partial y'} = 0$$

## 6. 应用示例

**例 1**：两点间最短路径

最小化 $J[y] = \int_a^b \sqrt{1+(y')^2}\,dx$。

Euler-Lagrange方程：$\frac{d}{dx}\frac{y'}{\sqrt{1+(y')^2}} = 0$，即 $y'' = 0$。

解：$y = c_1 x + c_2$，**直线**为最短路径。

**例 2**：极小旋转曲面

求曲线 $y(x) \geq 0$ 绕 $x$ 轴旋转所得曲面面积最小。

面积泛函：$S[y] = 2\pi\int_a^b y\sqrt{1+(y')^2}\,dx$。

首次积分给出：$y = c\cosh\frac{x-x_0}{c}$，**悬链线**。 $\square$
