## 📚 概述

变分法研究泛函的极值问题，是连接微积分、微分方程和物理学的桥梁。它起源于最速降线问题，发展成为现代数学物理和最优控制的基础理论。

---

## 🏛️ 历史发展

### 最速降线问题

**约翰·伯努利**（Johann Bernoulli, 1696）向欧洲数学家提出挑战：求连接两点的曲线，使质点在其上仅受重力作用下用时最短。

**牛顿、莱布尼茨、洛必达、伯努利家族**都给出了解答——摆线（cycloid）。

### Euler-Lagrange方程

**莱昂哈德·欧拉**（1744）发展了系统方法，导出极值的必要条件。

**约瑟夫-路易·拉格朗日**（1755，年仅19岁）给出了优雅的变分方法，被欧拉称赞并推广。

### Hamilton原理

**威廉·罗文·哈密顿**（1834-1835）建立了分析力学的变分基础：
$$\delta \int_{t_1}^{t_2} L dt = 0$$

## 📐 Euler-Lagrange方程

### 基本问题

求泛函 $J[y] = \int_a^b F(x, y, y') dx$ 的极值，固定端点 $y(a) = A$，$y(b) = B$。

### Euler-Lagrange方程

**定理 1.1** (Euler-Lagrange)

若 $y$ 是极值曲线，则满足：
$$\frac{\partial F}{\partial y} - \frac{d}{dx}\frac{\partial F}{\partial y'} = 0$$

### 首次积分

若 $F$ 不显含 $x$：
$$F - y'\frac{\partial F}{\partial y'} = C$$

## 🔍 经典变分问题

### 最速降线问题

最小化：$T = \int_0^a \frac{\sqrt{1+y'^2}}{\sqrt{2gy}} dx$

**解**：摆线（旋轮线）

### 等周问题

在固定周长下最大化面积。**解**：圆

### 测地线问题

曲面上两点间的最短路径。

## 📊 二阶变分

### Legendre条件

对于极小值，需要：
$$\frac{\partial^2 F}{\partial y'^2} \geq 0$$

### Jacobi条件

考虑Jacobi方程。若存在共轭点，则不是极小值。

---

## 🔬 约束变分

### 等周约束

**方法**：Lagrange乘子法
$$\mathcal{L} = F + \lambda G$$

## 💻 形式化实现

```lean
namespace calculus_of_variations

-- Euler-Lagrange方程条件
def euler_lagrange_equation {F : ℝ → ℝ → ℝ → ℝ}
  (y : ℝ → ℝ) (x : ℝ) : Prop :=
  let F_y := deriv (λ y, F x y (deriv y x)) (y x) in
  let F_y' := deriv (λ y', F x (y x) y') (deriv y x) in
  deriv (λ x, F_y') x - F_y = 0

-- 最速降线问题的解
def cycloid (r θ : ℝ) : ℝ × ℝ :=
  (r * (θ - real.sin θ), r * (1 - real.cos θ))

end calculus_of_variations
```
