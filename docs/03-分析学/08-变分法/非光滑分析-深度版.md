## 📚 概述

非光滑分析将微分学推广到非光滑函数，是优化理论、控制论和变分法的重要工具。它处理不可微但具有某种"正则性"的函数，填补了经典分析与凸分析之间的空白。

---

## 🏛️ 历史发展

### 早期探索

**Weierstrass** (1872) 构造了处处连续但处处不可微的函数，挑战了经典微积分的局限性。

### 现代发展

**Francis Clarke** (1970s) 创立了**广义梯度理论**（Nonsmooth Analysis），为Lipschitz函数建立了完整的微分理论。

**Rockafellar**、**Wets**等人发展了变分分析和集值分析。

## 📐 广义导数

### Dini导数

**定义 1.1** (Dini导数)

$$D^+f(x) = \limsup_{h \to 0^+} \frac{f(x+h) - f(x)}{h}$$

$$D_+f(x) = \liminf_{h \to 0^+} \frac{f(x+h) - f(x)}{h}$$

### 方向导数

**定义 1.2** (方向导数)

$$f'(x; d) = \lim_{t \to 0^+} \frac{f(x+td) - f(x)}{t}$$

### 广义方向导数(Clarke)

**定义 1.3** (Clarke广义方向导数)

对Lipschitz函数 $f$：
$$f^\circ(x; v) = \limsup_{y \to x, t \to 0^+} \frac{f(y+tv) - f(y)}{t}$$

## 🔍 Clarke次微分

### 定义

**定义 2.1** (Clarke次微分/广义梯度)

$$\partial_C f(x) = \{\xi : f^\circ(x; v) \geq \langle \xi, v \rangle, \forall v\}$$

### 性质

- 非空、凸、紧集
- 上半连续性
- 若 $f$ 凸：$\partial_C f = \partial f$（凸次微分）
- 若 $f$ 光滑：$\partial_C f = \{\nabla f\}$

### 链式法则

$$\partial_C (f \circ g)(x) \subseteq \partial_C f(g(x)) \circ \partial_C g(x)$$

## 📊 非光滑优化

### 最优性条件

**定理 3.1** (Fermat法则的推广)

若 $x^*$ 是局部极小点，则：
$$0 \in \partial_C f(x^*)$$

### 下降方法

**最速下降方向**：$-g$，其中 $g = \arg\min_{\xi \in \partial_C f(x)} \|\xi\|$

### 束方法

利用历史次梯度信息构造模型函数。

## 🔬 应用

### 最优控制

Pontryagin极大值原理涉及非光滑Hamiltonian。

### 变分不等式

互补问题和障碍问题。

### 机器学习

正则化项（如L1范数）导致的非光滑性。

## 💻 形式化实现

```lean
import analysis.calculus.fderiv
import analysis.normed_space.inner_product

namespace nonsmooth_analysis

-- Dini导数
def dini_upper_derivative (f : ℝ → ℝ) (x : ℝ) : ℝ :=
  limsup (λ h, (f (x + h) - f x) / h) (𝓝[>] 0)

-- 方向导数
def directional_derivative {E : Type*} [normed_add_comm_group E]
  (f : E → ℝ) (x : E) (v : E) : ℝ :=
  lim (λ t : ℝ, (f (x + t • v) - f x) / t) (𝓝[>] 0)

-- Clarke广义方向导数
def clarke_generalized_derivative {E : Type*} [normed_add_comm_group E]
  (f : E → ℝ) (x : E) (v : E) : ℝ :=
  limsup (λ p : E × ℝ, (f (p.1 + p.2 • v) - f p.1) / p.2) 
    (𝓝 x ×ˢ 𝓝[>] 0)

-- Clarke次微分
def clarke_subdifferential {E : Type*} [normed_add_comm_group E] 
  [inner_product_space ℝ E] (f : E → ℝ) (x : E) : set E :=
  {ξ : E | ∀ v, clarke_generalized_derivative f x v ≥ inner ξ v}

end nonsmooth_analysis
```
