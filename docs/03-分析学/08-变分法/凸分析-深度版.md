## 📚 概述

凸分析研究凸集和凸函数的性质，是优化理论、经济学和凸几何的数学基础。其核心洞见在于：凸问题具有良好的结构性质——局部最优即全局最优，对偶理论简洁优美。

---

## 🏛️ 历史发展

### 早期起源

**约翰·詹森**（Johan Jensen, 1906）系统研究了凸函数不等式。

**闵可夫斯基**（Hermann Minkowski, 1896-1910）创建了凸几何基础。

### 现代凸分析

**Werner Fenchel**（1949, 1953）创立了对偶理论。

**R. Tyrrell Rockafellar**（1970）的《Convex Analysis》确立了现代理论框架。

## 📐 凸集与凸函数

### 凸集

**定义 1.1** (凸集)

集合 $C \subseteq \mathbb{R}^n$ 是**凸的**，如果：
$$\forall x, y \in C, \forall \lambda \in [0,1]: \lambda x + (1-\lambda)y \in C$$

### 凸函数

**定义 1.2** (凸函数)

函数 $f: \mathbb{R}^n \to \mathbb{R} \cup \{+\infty\}$ 是**凸的**，如果：
$$f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y)$$

**二阶条件**：$\nabla^2 f(x) \succeq 0$（Hessian半正定）

## 🔍 分离定理

### Hahn-Banach几何形式

**定理 2.1** (分离定理)

设 $C, D$ 为不相交凸集，$C$ 开，则存在超平面严格分离 $C$ 和 $D$。

**定理 2.2** (严格分离)

设 $C$ 为闭凸集，$x_0 \notin C$，则存在超平面严格分离 $x_0$ 和 $C$。

## 📊 次微分

### 定义

**定义 3.1** (次梯度)

$g$ 是 $f$ 在 $x$ 的**次梯度**，如果：
$$f(y) \geq f(x) + g^T(y-x), \quad \forall y$$

### 性质

- $f$ 在 $x$ 可微 $\Leftrightarrow$ $\partial f(x) = \{\nabla f(x)\}$
- $\partial f(x)$ 是闭凸集

---

## 🔬 对偶理论

### 共轭函数

**定义 4.1** (Fenchel共轭)

$$f^*(y) = \sup_x \{y^T x - f(x)\}$$

### 优化对偶

**原始问题**：$\min_x f(x) + g(Ax)$

**对偶问题**：$\max_y -f^*(A^T y) - g^*(-y)$

## 💻 形式化实现

```lean
import analysis.convex.basic
import analysis.normed_space.inner_product

namespace convex_analysis

-- 凸函数定义
def convex_function {E : Type*} [normed_add_comm_group E] [inner_product_space ℝ E]
  (f : E → ℝ) : Prop :=
  ∀ x y : E, ∀ θ : ℝ, 0 ≤ θ → θ ≤ 1 →
    f (θ • x + (1 - θ) • y) ≤ θ * f x + (1 - θ) * f y

-- 次微分
def subdifferential {E : Type*} [normed_add_comm_group E] [inner_product_space ℝ E]
  (f : E → ℝ) (x : E) : set E :=
  {g : E | ∀ y, f y ≥ f x + inner g (y - x)}

-- 共轭函数
def fenchel_conjugate {E : Type*} [normed_add_comm_group E] [inner_product_space ℝ E]
  (f : E → ℝ) : E → ℝ :=
  λ y, ⨆ x, inner y x - f x

end convex_analysis
```
