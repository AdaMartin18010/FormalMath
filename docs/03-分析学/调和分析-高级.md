# 调和分析高级主题

调和分析（harmonic analysis）研究函数在其对称群作用下的分解，核心工具是 Fourier 变换、奇异积分与小波分析。本节聚焦 Hardy 空间、BMO 与 Calderón-Zygmund 理论，展示调和分析在复分析、偏微分方程与概率论中的深刻应用。从经典 Fourier 级数到现代时间-频率分析，调和分析始终是连接抽象分析与具体问题的桥梁。

## 1. Hardy 空间

### 1.1 $H^p$ 空间的定义

**定义 1.1（单位圆上的 Hardy 空间）**. 设 $0 < p \leq \infty$。单位圆盘 $\mathbb{D} = \{z \in \mathbb{C} : |z| < 1\}$ 上的 Hardy 空间 $H^p(\mathbb{D})$ 由满足以下条件的全纯函数 $f$ 组成：

$$\|f\|_{H^p} = \sup_{0 < r < 1} \left( \frac{1}{2\pi} \int_0^{2\pi} |f(re^{i\theta})|^p d\theta \right)^{1/p} < \infty.$$

对 $p = \infty$，$H^\infty(\mathbb{D})$ 为 $\mathbb{D}$ 上的有界全纯函数空间。

**定义 1.2（上半平面的 Hardy 空间）**. 上半平面 $\mathbb{H} = \{z : \operatorname{Im} z > 0\}$ 上的 $H^p(\mathbb{H})$ 由全纯函数 $f$ 组成：

$$\|f\|_{H^p(\mathbb{H})} = \sup_{y > 0} \left( \int_{-\infty}^{\infty} |f(x+iy)|^p dx \right)^{1/p} < \infty.$$

### 1.2 边界值与 Poisson 积分

**定理 1.3（Fatou）**. 设 $f \in H^p(\mathbb{D})$，$1 \leq p \leq \infty$。则对几乎处处的 $\theta$，径向极限 $f^*(e^{i\theta}) = \lim_{r \to 1^-} f(re^{i\theta})$ 存在，且 $f^* \in L^p(\mathbb{T})$。

*证明*. 对 $p > 1$，$H^p$ 函数可表示为 Poisson 积分 $f(re^{i\theta}) = P_r * f^*(e^{i\theta})$，其中 $P_r$ 为 Poisson 核。由 Hardy-Littlewood 极大函数的有界性，径向极限几乎处处存在。$\square$

## 2. BMO 与 John-Nirenberg 不等式

### 2.1 有界平均振动函数

**定义 2.1（BMO）**. $f \in L^1_{\text{loc}}(\mathbb{R}^n)$ 称为有界平均振动（BMO），若：

$$\|f\|_{\text{BMO}} = \sup_{Q} \frac{1}{|Q|} \int_Q |f(x) - f_Q| dx < \infty,$$

其中上确界取遍所有方体 $Q$，$f_Q = \frac{1}{|Q|} \int_Q f$ 为平均值。

**定理 2.2（John-Nirenberg）**. 存在常数 $c_1, c_2 > 0$ 使得对所有 $f \in \text{BMO}$、方体 $Q$ 和 $\lambda > 0$：

$$|\{x \in Q : |f(x) - f_Q| > \lambda\}| \leq c_1 |Q| \exp\left(-\frac{c_2 \lambda}{\|f\|_{\text{BMO}}}\right).$$

*证明概要*. 对 $f$ 进行 Calderón-Zygmund 分解。设 $a = \|f\|_{\text{BMO}}$，递归构造方体族：在 $Q$ 上，若 $|f - f_Q| > 2a$ 的集合非空，则分解为子方体；在每个子方体上重复。利用 BMO 定义控制每一层的测度，得到指数衰减。$\square$

**推论 2.3**. BMO $\subset \bigcap_{p < \infty} L^p_{\text{loc}}$，且 BMO 是 $L^\infty$ 的真子集（$\ln|x| \in \text{BMO}$ 但 $\notin L^\infty$）。

## 3. Calderón-Zygmund 理论

### 3.1 奇异积分算子

**定义 3.1（Calderón-Zygmund 核）**. 函数 $K(x)$ 称为 CZ 核，若：
1. $K \in C^1(\mathbb{R}^n \setminus \{0\})$；
2. **大小条件**：$|K(x)| \leq C/|x|^n$；
3. **光滑性条件**：$|\nabla K(x)| \leq C/|x|^{n+1}$；
4. **消没条件**：$\int_{R_1 < |x| < R_2} K(x) dx = 0$（$0 < R_1 < R_2 < \infty$）。

**定义 3.2（奇异积分算子）**. 对 CZ 核 $K$，奇异积分算子定义为（主值意义）：

$$Tf(x) = \text{p.v.} \int_{\mathbb{R}^n} K(x-y) f(y) dy = \lim_{\epsilon \to 0} \int_{|x-y| > \epsilon} K(x-y) f(y) dy.$$

### 3.2 $L^2$ 有界性

**定理 3.3（Plancherel）**. 若 $\widehat{K} \in L^\infty$，则 $T$ 在 $L^2$ 上有界。

*证明*. 由 Fourier 变换，$\widehat{Tf}(\xi) = \widehat{K}(\xi) \widehat{f}(\xi)$。故：

$$\|Tf\|_{L^2} = \|\widehat{Tf}\|_{L^2} = \|\widehat{K} \widehat{f}\|_{L^2} \leq \|\widehat{K}\|_{L^\infty} \|\widehat{f}\|_{L^2} = \|\widehat{K}\|_{L^\infty} \|f\|_{L^2}.$$

$\square$

### 3.3 $L^p$ 有界性

**定理 3.4（Calderón-Zygmund）**. 奇异积分算子 $T$ 可延拓为 $L^p(\mathbb{R}^n)$（$1 < p < \infty$）上的有界算子，且是弱 $(1,1)$ 型的：

$$|\{x : |Tf(x)| > \lambda\}| \leq \frac{C}{\lambda} \|f\|_{L^1}.$$

*证明概要*. 对 $f \in L^1$，进行 Calderón-Zygmund 分解 $f = g + b$，其中 $g \in L^2$ 且 $\|g\|_{L^2}^2 \leq C\lambda \|f\|_{L^1}$，$b$ 支在不交方体上且每个方体上积分为零。$Tg$ 用 $L^2$ 估计，$Tb$ 利用核的光滑性和 $b$ 的消没性质。$\square$

## 4. 例子

### 4.1 例子：Hilbert 变换

Hilbert 变换是经典的奇异积分算子：

$$Hf(x) = \text{p.v.} \frac{1}{\pi} \int_{-\infty}^{\infty} \frac{f(y)}{x-y} dy.$$

核 $K(x) = 1/(\pi x)$ 满足 CZ 条件。Fourier 变换：$\widehat{Hf}(\xi) = -i \operatorname{sgn}(\xi) \widehat{f}(\xi)$，故 $\|\widehat{K}\|_{L^\infty} = 1$，$H$ 在 $L^2$ 上等距（相差常数）。

### 4.2 例子：Riesz 变换

$n$ 维 Riesz 变换：

$$R_j f(x) = c_n \text{ p.v.} \int_{\mathbb{R}^n} \frac{x_j - y_j}{|x-y|^{n+1}} f(y) dy, \quad j = 1, \dots, n.$$

满足 $\sum_{j=1}^n R_j^2 = -I$（在 $L^2$ 上）。在 PDE 中，Riesz 变换联系调和函数的共轭系统。
