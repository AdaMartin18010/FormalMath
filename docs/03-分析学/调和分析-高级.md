# 调和分析高级主题

调和分析（harmonic analysis）研究函数在其对称群作用下的分解，核心工具是 Fourier 变换、奇异积分与小波分析。本节聚焦 Hardy 空间、BMO 与 Calderón-Zygmund 理论，展示调和分析在复分析、偏微分方程与概率论中的深刻应用。

## 1. Hardy 空间

### 1.1 $H^p$ 空间的定义

**定义 1.1（单位圆上的 Hardy 空间）**. 设 $0 < p \leq \infty$。单位圆盘 $\mathbb{D} = \{z \in \mathbb{C} : |z| < 1\}$ 上的 Hardy 空间 $H^p(\mathbb{D})$ 由满足以下条件的全纯函数 $f$ 组成：

$$\|f\|_{H^p} = \sup_{0 < r < 1} \left( \frac{1}{2\pi} \int_0^{2\pi} |f(re^{i\theta})|^p d\theta \right)^{1/p} < \infty.$$

对 $p = \infty$，$H^\infty(\mathbb{D})$ 为 $\mathbb{D}$ 上的有界全纯函数空间。

**定义 1.2（上半平面的 Hardy 空间）**. 上半平面 $\mathbb{H} = \{z : \operatorname{Im} z > 0\}$ 上的 $H^p(\mathbb{H})$ 由全纯函数 $f$ 组成：

$$\|f\|_{H^p}^p = \sup_{y > 0} \int_{-\infty}^{\infty} |f(x+iy)|^p dx < \infty.$$

### 1.2 边值与 Poisson 积分

**定理 1.3**. 对 $f \in H^p(\mathbb{D})$（$0 < p \leq \infty$），径向极限

$$f^*(e^{i\theta}) = \lim_{r \to 1^-} f(re^{i\theta})$$

对几乎处处的 $\theta$ 存在，且 $f^* \in L^p(\mathbb{T})$。映射 $f \mapsto f^*$ 是 $H^p$ 到 $L^p(\mathbb{T})$ 的等距嵌入（对 $1 \leq p \leq \infty$）。

**定理 1.4（Poisson 积分表示）**. 对 $f \in H^p(\mathbb{D})$（$1 \leq p \leq \infty$），

$$f(re^{i\theta}) = \frac{1}{2\pi} \int_0^{2\pi} P_r(\theta - \phi) f^*(e^{i\phi}) d\phi,$$

其中 $P_r(\theta) = \frac{1-r^2}{1-2r\cos\theta+r^2}$ 为 Poisson 核。

### 1.3 $H^1$ 与 BMO 的对偶

**定义 1.5（BMO）**. 局部可积函数 $f$ 属于 BMO（bounded mean oscillation）空间，若：

$$\|f\|_{\text{BMO}} = \sup_{Q} \frac{1}{|Q|} \int_Q |f - f_Q| dx < \infty,$$

其中 $Q$ 遍历 $\mathbb{R}^n$ 中的方体，$f_Q = \frac{1}{|Q|}\int_Q f dx$。

**定理 1.6（Fefferman-Stein）**. $(H^1(\mathbb{R}^n))^* \cong \text{BMO}(\mathbb{R}^n)$。

*证明概要*. 利用原子分解：$H^1$ 中每个函数可写为 $f = \sum \lambda_j a_j$，其中 $a_j$ 为原子（支集在方体 $Q_j$ 上，$\|a_j\|_\infty \leq |Q_j|^{-1}$，$\int a_j = 0$）。对 $g \in \text{BMO}$，定义泛函 $\Lambda_g(f) = \int fg$（对光滑 $f$），利用 BMO 条件证明其在原子上的一致有界性。$\square$

## 2. Calderón-Zygmund 理论

### 2.1 奇异积分算子

**定义 2.1（Calderón-Zygmund 核）**. 函数 $K: \mathbb{R}^n \setminus \{0\} \to \mathbb{C}$ 称为 Calderón-Zygmund 核，若：

1. **尺寸条件**：$|K(x)| \leq \frac{C}{|x|^n}$；
2. **光滑性条件**：$|K(x) - K(x-y)| \leq \frac{C|y|^\delta}{|x|^{n+\delta}}$ 对某 $\delta > 0$ 和 $|y| < |x|/2$；
3. **消失条件**：$\int_{r < |x| < R} K(x) dx = 0$ 对所有 $0 < r < R < \infty$。

**定义 2.2（奇异积分算子）**. 对 $f \in \mathcal{S}(\mathbb{R}^n)$，定义：

$$Tf(x) = \text{p.v.} \int_{\mathbb{R}^n} K(x-y) f(y) dy = \lim_{\epsilon \to 0} \int_{|x-y| > \epsilon} K(x-y) f(y) dy.$$

### 2.2 $L^2$ 有界性

**定理 2.3（Plancherel）**. 若 $\hat{K} \in L^\infty(\mathbb{R}^n)$，则 $T$ 可延拓为 $L^2(\mathbb{R}^n)$ 上的有界算子。

*证明*. 由 Fourier 变换的性质，$\widehat{Tf}(\xi) = \hat{K}(\xi)\hat{f}(\xi)$，故：

$$\|Tf\|_{L^2} = \|\widehat{Tf}\|_{L^2} = \|\hat{K}\hat{f}\|_{L^2} \leq \|\hat{K}\|_\infty \|\hat{f}\|_{L^2} = \|\hat{K}\|_\infty \|f\|_{L^2}.$$

$\square$

### 2.3 $L^p$ 有界性与弱 $(1,1)$

**定理 2.4（Calderón-Zygmund）**. 设 $T$ 为奇异积分算子，$L^2$ 有界。则：

1. $T$ 是弱 $(1,1)$ 型：$|\{x : |Tf(x)| > \alpha\}| \leq \frac{C}{\alpha}\|f\|_{L^1}$；
2. 对 $1 < p < \infty$，$T$ 可延拓为 $L^p$ 上的有界算子。

*证明 (1)*. 对 $f \in L^1$ 和 $\alpha > 0$ 做 Calderón-Zygmund 分解：$f = g + b$，其中 $|g| \leq \alpha$，$b = \sum b_j$ 支集在互不相交的方体 $Q_j$ 上，$\int b_j = 0$，$\sum |Q_j| \leq \frac{C}{\alpha}\|f\|_{L^1}$。

$g$ 项用 $L^2$ 有界性：$|\{Tg > \alpha/2\}| \leq \frac{C}{\alpha^2}\|g\|_{L^2}^2 \leq \frac{C}{\alpha}\|f\|_{L^1}$。

$b$ 项利用核的光滑性：对每个 $b_j$，$Tb_j$ 在 $Q_j$ 的适当放大 $\tilde{Q}_j$ 外有小的 $L^1$ 范数。$\square$

## 3. 例子

### 3.1 例子：Hilbert 变换

一维 Hilbert 变换是最重要的奇异积分：

$$Hf(x) = \text{p.v.} \frac{1}{\pi} \int_{-\infty}^{\infty} \frac{f(y)}{x-y} dy.$$

核 $K(x) = \frac{1}{\pi x}$ 满足所有 Calderón-Zygmund 条件。Fourier 乘子为 $-i\operatorname{sgn}(\xi)$，故 $\|H\|_{L^2 \to L^2} = 1$。

**定理 3.1**. Hilbert 变换满足：

- 弱 $(1,1)$：$|\{Hf > \alpha\}| \leq \frac{C}{\alpha}\|f\|_{L^1}$；
- $L^p$ 有界（$1 < p < \infty$）：$\|Hf\|_{L^p} \leq C_p \|f\|_{L^p}$；
- 在 $L^1$ 和 $L^\infty$ 上无界。

### 3.2 例子：Riesz 变换

$n$ 维 Riesz 变换：

$$R_j f(x) = c_n \, \text{p.v.} \int_{\mathbb{R}^n} \frac{x_j - y_j}{|x-y|^{n+1}} f(y) dy, \quad j = 1, \dots, n.$$

Fourier 乘子为 $-i\xi_j/|\xi|$。它们满足：

$$\sum_{j=1}^n R_j^2 = -I,$$

并用于刻画 Hardy 空间：$f \in H^1(\mathbb{R}^n)$ 当且仅当 $f \in L^1$ 且所有 $R_j f \in L^1$。
