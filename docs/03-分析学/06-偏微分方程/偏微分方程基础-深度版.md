## 📚 概述

偏微分方程(PDE)是含有未知多元函数及其偏导数的方程。作为描述自然现象的基本语言，PDE在物理学、工程学和金融学中无处不在。本文档介绍PDE的基础理论和经典解法。

---

## 🏛️ 历史发展

### 18世纪的起源

**达朗贝尔**（d'Alembert, 1746）研究振动弦，导出波动方程。

**欧拉**和**伯努利家族**发展了分离变量法。

**拉普拉斯**（1780s）引入位势理论和Laplace方程。

### 19世纪的严格化

**傅里叶**（1807）发展了热传导方程的级数解法。

**柯西**和**Kovalevskaya**建立了存在性理论。

**黎曼**在几何和物理问题中应用PDE。

### 现代发展

**希尔伯特第19-23问题**与PDE相关。

**Sobolev**（1930s）引入广义函数理论。

**Leray、Schauder**等发展了非线性理论。

## 📐 基本分类

### 按阶数分类

**一阶PDE**：最高阶导数为一阶，如输运方程。

**二阶PDE**：最常见，如波动方程、热方程、Poisson方程。

**高阶PDE**：双调和方程、Korteweg-de Vries方程等。

### 二阶线性方程的分类

一般形式：
$$Au_{xx} + 2Bu_{xy} + Cu_{yy} + Du_x + Eu_y + Fu = G$$

判别式 $\Delta = B^2 - AC$：
- $\Delta < 0$：**椭圆型**（Laplace方程）
- $\Delta = 0$：**抛物型**（热方程）
- $\Delta > 0$：**双曲型**（波动方程）

**分类的物理意义**：
- 椭圆型：稳态问题（平衡）
- 抛物型：扩散问题（不可逆）
- 双曲型：波动问题（可逆）

## 🔍 一阶方程

### 线性输运方程

$$u_t + c u_x = 0$$

**通解**：$u(x,t) = f(x - ct)$（行波解）

### 特征线法

对于 $a(x,y)u_x + b(x,y)u_y = c(x,y,u)$：

沿特征曲线 $\frac{dx}{a} = \frac{dy}{b} = \frac{du}{c}$ 求解。

---

## 📊 二阶线性方程

### 波动方程

$$u_{tt} = c^2 u_{xx}$$

**d'Alembert公式**：
$$u(x,t) = \frac{1}{2}[f(x+ct) + f(x-ct)] + \frac{1}{2c}\int_{x-ct}^{x+ct} g(s) ds$$

其中 $u(x,0) = f(x)$，$u_t(x,0) = g(x)$。

### 热方程

$$u_t = k u_{xx}$$

**基本解**（热核）：
$$\Phi(x,t) = \frac{1}{\sqrt{4\pi k t}} e^{-x^2/(4kt)}$$

**Poisson公式**：
$$u(x,t) = \int_{-\infty}^{\infty} \Phi(x-y, t) f(y) dy$$

### Laplace方程

$$\Delta u = u_{xx} + u_{yy} = 0$$

**基本性质**：
- 平均值性质
- 最大值原理
- 解析性（解是实解析的）

## 🔬 经典问题

### 初值问题(Cauchy问题)

给定初始条件，求未来演化。

### 边值问题

**Dirichlet问题**：给定边界上的函数值。

**Neumann问题**：给定边界上的法向导数。

**Robin问题**：混合边界条件。

### 适定性

Hadamard适定性要求：
1. **存在性**：解存在
2. **唯一性**：解唯一
3. **稳定性**：解连续依赖于数据
