## 📚 概述

留数定理是复分析中最强大的计算工具之一。它将围道积分转化为留数的求和，为计算实积分、级数求和提供了优雅的统一方法。本文档系统阐述留数计算的理论和技巧。

---

## 🏛️ 历史发展

### 柯西的奠基

**奥古斯丁-路易·柯西**在19世纪20年代发展了围道积分理论，为留数定理奠定了基础。

### 留数概念的引入

柯西引入"留数"(résidu)概念，用以刻画函数在孤立奇点处的"残留"信息。

### 现代形式

留数定理的现代形式由**柯西**、**刘维尔**等人完善，成为复分析的核心定理。

## 📐 留数定理

### 留数的定义

**定义 1.1** (留数)

设 $f$ 在 $z_0$ 有孤立奇点，Laurent展开为：
$$f(z) = \sum_{n=-\infty}^{\infty} a_n (z-z_0)^n$$

留数定义为：
$$\text{Res}(f, z_0) = a_{-1} = \frac{1}{2\pi i} \oint_{|z-z_0|=r} f(z) dz$$

### 留数定理

**定理 1.1** (留数定理)

设 $f$ 在区域 $D$ 内除有限个奇点 $z_1, \ldots, z_n$ 外全纯，$\gamma$ 为 $D$ 内围绕所有奇点的正向简单闭曲线，则：
$$\oint_\gamma f(z) dz = 2\pi i \sum_{k=1}^{n} \text{Res}(f, z_k)$$

## 🔍 留数计算方法

### 可去奇点

若 $z_0$ 为可去奇点，则 $\text{Res}(f, z_0) = 0$。

### 单极点

**定理 2.1** (单极点留数)

$$\text{Res}(f, z_0) = \lim_{z \to z_0} (z-z_0)f(z)$$

若 $f(z) = \frac{g(z)}{h(z)}$，$g(z_0) \neq 0$，$h(z_0) = 0$，$h'(z_0) \neq 0$：
$$\text{Res}(f, z_0) = \frac{g(z_0)}{h'(z_0)}$$

### m阶极点

**定理 2.2** (m阶极点留数)

$$\text{Res}(f, z_0) = \frac{1}{(m-1)!} \lim_{z \to z_0} \frac{d^{m-1}}{dz^{m-1}}[(z-z_0)^m f(z)]$$

### 本性奇点

需计算Laurent展开的 $a_{-1}$ 系数。

### 无穷远点

**定义 2.1** (无穷远点留数)

$$\text{Res}(f, \infty) = -\frac{1}{2\pi i} \oint_{|z|=R} f(z) dz$$

**定理 2.3** (全平面留数和)

$$\sum_{k} \text{Res}(f, z_k) + \text{Res}(f, \infty) = 0$$

## 📊 实积分计算

### 类型I：三角函数积分

$$I = \int_0^{2\pi} R(\cos\theta, \sin\theta) d\theta$$

**方法**：令 $z = e^{i\theta}$，$d\theta = \frac{dz}{iz}$

$$I = \oint_{|z|=1} R\left(\frac{z+z^{-1}}{2}, \frac{z-z^{-1}}{2i}\right) \frac{dz}{iz}$$

**例 3.1**：$I = \int_0^{2\pi} \frac{d\theta}{1+a\cos\theta}$，$|a| < 1$

$$I = \oint_{|z|=1} \frac{2}{a z^2 + 2z + a} dz = \frac{2\pi}{\sqrt{1-a^2}}$$

### 类型II：有理函数积分

$$I = \int_{-\infty}^{\infty} \frac{P(x)}{Q(x)} dx$$

**条件**：$\deg Q \geq \deg P + 2$，$Q$ 无实根。

**方法**：上半平面围道，留数和乘以 $2\pi i$。

**例 3.2**：$I = \int_{-\infty}^{\infty} \frac{1}{1+x^2} dx = \pi$

上半平面极点 $z = i$，留数 $\frac{1}{2i}$。

### 类型III：含指数因子的积分

$$I = \int_{-\infty}^{\infty} f(x) e^{iax} dx$$

**方法**：Jordan引理 + 上半平面围道。

**例 3.3**：$I = \int_0^{\infty} \frac{\sin x}{x} dx = \frac{\pi}{2}$

考虑 $f(z) = \frac{e^{iz}}{z}$，围道绕过原点。

## 🔬 其他应用

### 级数求和

**定理 4.1** (Mittag-Leffler求和)

$$\sum_{n=-\infty}^{\infty} f(n) = -\sum_{k} \text{Res}(\pi \cot(\pi z) f(z), z_k)$$

**例 4.1**：$\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6}$

### 反演Laplace变换

$$f(t) = \frac{1}{2\pi i} \int_{c-i\infty}^{c+i\infty} F(s) e^{st} ds$$
