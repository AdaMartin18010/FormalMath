## 🔄 二、Cauchy积分理论

### 2.1 Cauchy积分定理

**定理**：设f在单连通区域D上解析，γ是D内任意闭曲线，则：

```text
∮_γ f(z) dz = 0
```

**证明树**：

```text
        [Cauchy积分定理证明]
                │
    ┌───────────┴───────────┐
    │                       │
[Green公式法]           [同伦法]
    │                       │
∮f dz = ∮(u+iv)(dx+idy)   γ ≃ 点(D单连通)
    │                       │
= ∮(udx-vdy)+i∮(vdx+udy)   积分沿同伦
    │                       │
Green公式:               不变
∬(∂v/∂x+∂u/∂y)dA=0
∬(∂u/∂x-∂v/∂y)dA=0
    │                       │
由C-R方程成立              │
    │                       │
    └───────────┬───────────┘
                │
        ∮_γ f(z) dz = 0 ∎
```

### 2.2 Cauchy积分公式

**定理**：设f在闭圆盘D̄(a,r)上解析，则对z∈D(a,r)：

```text
f(z) = (1/2πi) ∮_{|ζ-a|=r} f(ζ)/(ζ-z) dζ
```

**推论（高阶导数）**：

```text
f^(n)(z) = (n!/2πi) ∮ f(ζ)/(ζ-z)^{n+1} dζ
```

### 2.3 Cauchy不等式

**定理**：若f在D̄(a,R)上解析且|f(z)|≤M，则：

```text
|f^(n)(a)| ≤ n! M / R^n
```

**应用**：证明Liouville定理（有界整函数为常数）

## 📐 三、留数定理

### 3.1 Laurent级数

**定理**：f在环域r < |z-a| < R内解析，则：

```text
f(z) = Σ_{n=-∞}^{∞} c_n (z-a)^n
```

其中：

```text
c_n = (1/2πi) ∮ f(ζ)/(ζ-a)^{n+1} dζ
```

### 3.2 奇点分类

```text
        [孤立奇点分类]
              │
    ┌─────────┼─────────┐
    │         │         │
[可去奇点] [极点]    [本性奇点]
    │         │         │
Laurent级数  有限个   无穷个
无负幂项     负幂项   负幂项
    │         │         │
lim存在    lim=∞     lim不存在
且有限                 (Picard)
    │         │         │
例: sin z/z  例: 1/z   例: e^{1/z}
```

### 3.3 留数定理

**定理**：设f在区域D内除孤立奇点a₁,...,aₙ外解析，γ是围绕这些奇点的简单闭曲线，则：

```text
∮_γ f(z) dz = 2πi Σₖ Res(f, aₖ)
```

**留数计算方法**：

| 奇点类型 | 计算公式 |
|----------|----------|
| 单极点 | Res(f,a) = lim_{z→a}(z-a)f(z) |
| m阶极点 | Res(f,a) = (1/(m-1)!) lim_{z→a} d^{m-1}/dz^{m-1}[(z-a)^m f(z)] |
| 本性奇点 | 展开Laurent级数取c_{-1} |

### 3.4 留数定理证明树

```text
        [留数定理证明]
              │
    ┌─────────┴─────────┐
    │                   │
[分离奇点]          [局部计算]
    │                   │
在每个aₖ周围      对每个小圆Cₖ:
取小圆Cₖ           ∮_{Cₖ}f dz = ?
    │                   │
γ与Σ Cₖ围成      Laurent展开:
无奇点区域        f = Σc_n(z-aₖ)^n
    │                   │
由Cauchy定理:     ∮(z-aₖ)^n dz =
∮_γ = Σ∮_{Cₖ}     {2πi, n=-1
                  {0,   n≠-1
    │                   │
    └─────────┬─────────┘
              │
    ∮_γ f dz = 2πi Σ c_{-1}^(k)
             = 2πi Σ Res(f,aₖ) ∎
```

### 3.5 应用：实积分计算

**类型1：有理函数积分**

```text
∫_{-∞}^{∞} P(x)/Q(x) dx = 2πi Σ(上半平面留数)
```

**类型2：三角函数积分**

```text
∫_0^{2π} R(cos θ, sin θ) dθ  (令z=e^{iθ})
```

**类型3：围道积分**

```text
选择适当围道，利用Jordan引理
```

## 🔗 四、解析延拓

### 4.1 解析延拓定理

**定理（唯一性）**：设f₁和f₂在连通区域D上解析，若在D的某子集E（有聚点）上f₁=f₂，则在整个D上f₁=f₂。

**推论**：解析函数由边界值唯一决定。

### 4.2 单值性定理

```text
        [解析延拓唯一性]
              │
    ┌─────────┴─────────┐
    │                   │
[恒等定理]          [延拓唯一]
    │                   │
f=g在E上          沿不同路径
E有聚点           延拓结果相同
    │                   │
f-g的零点集       (在单连通域上)
有聚点                 │
    │                   │
零点孤立性矛盾         │
    │                   │
f=g在整个D上           │
    │                   │
    └─────────┬─────────┘
              │
        [多连通域]
        可能多值(如log z)
```

### 4.3 Schwarz反射原理

**定理**：设f在上半平面解析，在实轴连续且取实值，则可延拓到整个平面：

```text
F(z) = { f(z),      Im z > 0
       { f(z̄)̄,     Im z < 0
```

## 🗺️ 五、共形映射

### 5.1 共形映射基本性质

**定义**：f共形 ⟺ f解析且f'(z)≠0

**几何意义**：

- 保角：两曲线夹角不变
- 保向：方向保持
- 局部保形：无穷小圆映为无穷小圆

### 5.2 Riemann映射定理

**定理**：设D是ℂ中真子集的单连通区域，则存在双全纯映射f: D → 𝔻（单位圆盘）。

**证明思路**：

```text
        [Riemann映射定理证明]
                │
    ┌───────────┴───────────┐
    │                       │
[存在性]                [唯一性]
    │                       │
考虑函数族            固定z₀∈D
F = {f: D→𝔻单射     和f(z₀)=0
    全纯}              f'(z₀)>0
    │                       │
F非空(存在            唯一确定f
单射全纯函数)              │
    │                       │
考虑sup|f'(z₀)|           │
    │                       │
Montel定理:              │
正规族有收敛子列          │
    │                       │
极限函数满足条件          │
    │                       │
    └───────────┬───────────┘
                │
        存在唯一f: D ≃ 𝔻 ∎
```

### 5.3 基本共形映射

| 映射 | 公式 | 效果 |
|------|------|------|
| Möbius变换 | (az+b)/(cz+d) | 圆→圆，保交比 |
| Joukowski | z + 1/z | 圆→翼型 |
| Koebe函数 | z/(1-z)² | 单叶函数极值 |
| 指数映射 | e^z | 带→圆环 |

### 7.2 Python 数值实现

```python
import numpy as np
from scipy import integrate

def residue_at_pole(f, z0, order=1, eps=1e-8):
    """计算f在z0处的留数（假设为极点）"""
    if order == 1:
        # 单极点: Res = lim (z-z0)*f(z)
        return (f(z0 + eps) * eps).real
    else:
        # 高阶极点: 使用数值微分
        from scipy.misc import derivative
        g = lambda z: (z - z0)**order * f(z)
        return derivative(g, z0, n=order-1, dx=eps) / np.math.factorial(order-1)

def contour_integral(f, center, radius, n_points=1000):
    """计算围道积分 ∮f(z)dz"""
    theta = np.linspace(0, 2*np.pi, n_points)
    z = center + radius * np.exp(1j * theta)
    dz = 1j * radius * np.exp(1j * theta)
    integrand = f(z) * dz
    return np.trapz(integrand, theta)

# 示例：计算 ∫_{-∞}^{∞} 1/(1+x²) dx = π
def rational_integral():
    f = lambda z: 1 / (1 + z**2)
    # 上半平面极点: z = i，留数 = 1/(2i)
    residue = 1 / (2j)
    result = 2 * np.pi * 1j * residue
    print(f"∫1/(1+x²)dx = {result.real:.6f} (理论值: π = {np.pi:.6f})")

rational_integral()
```
