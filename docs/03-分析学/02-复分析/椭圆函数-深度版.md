## 📚 概述

椭圆函数是双周期亚纯函数，是三角函数到复平面的深刻推广。它们是椭圆积分的反函数，在代数几何（椭圆曲线）、数论（模形式）和数学物理中有核心地位。

**核心特征**：

- 双周期性：存在两个周期 $\omega_1, \omega_2$（线性无关）
- 亚纯性：在整个复平面上亚纯
- 椭圆曲线与椭圆函数一一对应

## 🏛️ 历史发展

### 欧拉的发现

**莱昂哈德·欧拉**（1753年）研究了椭圆弧长计算问题，发现了椭圆积分的加法定理。

### 阿贝尔的革命

**尼尔斯·阿贝尔**（1827年）认识到椭圆积分的反函数（椭圆函数）比积分本身更自然。他证明了：

- 椭圆函数的双周期性
- 加法定理的代数形式

### 雅可比与Weierstrass

**卡尔·古斯塔夫·雅可比**发展了椭圆函数的Theta函数表示和模变换理论。

**卡尔·魏尔斯特拉斯**引入了标准形式 $\wp(z)$，统一了椭圆函数理论。

## 📐 椭圆积分的反演

### 椭圆积分的起源

椭圆积分最初来自椭圆弧长的计算：

$$s = a \int_0^{\phi} \sqrt{1 - k^2 \sin^2 \theta} d\theta$$

Legendre标准形式：

- **第一类**：$F(k, \phi) = \int_0^{\phi} \frac{d\theta}{\sqrt{1-k^2\sin^2\theta}}$
- **第二类**：$E(k, \phi) = \int_0^{\phi} \sqrt{1-k^2\sin^2\theta} d\theta$
- **第三类**：$\Pi(n, k, \phi) = \int_0^{\phi} \frac{d\theta}{(1-n\sin^2\theta)\sqrt{1-k^2\sin^2\theta}}$

### 反演思想

**关键洞察**：不是研究积分本身，而是研究其反函数。

若 $u = F(k, \phi)$，定义**振幅函数**：$\phi = \text{am}(u, k)$

由此定义Jacobi椭圆函数：
$$\text{sn}(u, k) = \sin \phi, \quad \text{cn}(u, k) = \cos \phi, \quad \text{dn}(u, k) = \sqrt{1-k^2\sin^2\phi}$$

## 🔍 Weierstrass椭圆函数

### 定义

**定义 1.1** (Weierstrass $\wp$函数)

给定周期格 $\Lambda = \{m\omega_1 + n\omega_2 : m,n \in \mathbb{Z}\}$：

$$\wp(z; \Lambda) = \frac{1}{z^2} + \sum_{\omega \in \Lambda \setminus \{0\}} \left(\frac{1}{(z-\omega)^2} - \frac{1}{\omega^2}\right)$$

### 基本性质

**定理 1.1** (双周期性)

$$\wp(z + \omega_1) = \wp(z), \quad \wp(z + \omega_2) = \wp(z)$$

**定理 1.2** (微分方程)

$$\wp'(z)^2 = 4\wp(z)^3 - g_2\wp(z) - g_3$$

其中：
$$g_2 = 60 \sum_{\omega \neq 0} \frac{1}{\omega^4}, \quad g_3 = 140 \sum_{\omega \neq 0} \frac{1}{\omega^6}$$

**定理 1.3** (加法定理)

$$\wp(z+w) = -\wp(z) - \wp(w) + \frac{1}{4}\left(\frac{\wp'(z) - \wp'(w)}{\wp(z) - \wp(w)}\right)^2$$

## 📊 Jacobi椭圆函数

### 定义

**定义 2.1** (Jacobi椭圆函数)

$$\text{sn}(u, k) = \sin(\text{am}(u, k))$$
$$\text{cn}(u, k) = \cos(\text{am}(u, k))$$
$$\text{dn}(u, k) = \sqrt{1 - k^2 \text{sn}^2(u, k)}$$

### 基本关系

$$\text{sn}^2(u) + \text{cn}^2(u) = 1$$
$$k^2 \text{sn}^2(u) + \text{dn}^2(u) = 1$$

### 微分方程

$$\frac{d}{du}\text{sn}(u) = \text{cn}(u)\text{dn}(u)$$
$$\frac{d}{du}\text{cn}(u) = -\text{sn}(u)\text{dn}(u)$$
$$\frac{d}{du}\text{dn}(u) = -k^2\text{sn}(u)\text{cn}(u)$$

### 周期性

- $\text{sn}(u)$：周期为 $4K$ 和 $2iK'$
- $\text{cn}(u)$：周期为 $4K$ 和 $2K + 2iK'$
- $\text{dn}(u)$：周期为 $2K$ 和 $4iK'$

其中 $K = K(k)$ 为第一类完全椭圆积分，$K' = K(k')$，$k' = \sqrt{1-k^2}$。

## 🔬 基本性质

### 椭圆函数的分类

**定理 3.1** (Liouville定理)

非常数椭圆函数必有极点。

**定理 3.2** (留数和为零)

椭圆函数在一个周期平行四边形内的留数之和为零。

**定理 3.3** (零点与极点)

椭圆函数在一个周期平行四边形内的零点数等于极点数（计重数）。

### 模形式联系

椭圆函数与模形式深刻联系：

$$j(\tau) = 1728 \frac{g_2(\tau)^3}{g_2(\tau)^3 - 27g_3(\tau)^2}$$

是模群 $\text{SL}(2, \mathbb{Z})$ 的基本不变量。
