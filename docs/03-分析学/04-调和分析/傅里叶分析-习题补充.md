# 傅里叶分析习题补充

## 习题1

计算$f(x) = x$在$[-\pi, \pi]$上的傅里叶级数。

**解答**：
$$a_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} x dx = 0$$

$$a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} x \cos(nx) dx = 0$$（奇函数）

$$b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} x \sin(nx) dx = \frac{2(-1)^{n+1}}{n}$$

所以：$f(x) = \sum_{n=1}^{\infty} \frac{2(-1)^{n+1}}{n} \sin(nx)$

## 习题2

证明Parseval等式：$\|f\|^2 = \sum_{n=-\infty}^{\infty} |\hat{f}(n)|^2$

**证明**：利用傅里叶基$\{e^{inx}\}$的正交性。

$$\|f\|^2 = \langle f, f \rangle = \sum_{m,n} \hat{f}(m)\overline{\hat{f}(n)} \langle e^{imx}, e^{inx} \rangle = \sum_n |\hat{f}(n)|^2$$ ∎
