# 傅里叶分析习题补充

Fourier 分析研究函数在三角函数系下的展开，是调和分析、偏微分方程、信号处理与概率论的基础工具。本节通过精选习题展示 Fourier 级数的收敛性、Parseval 等式与 Poisson 求和公式的核心技巧。

## 1. Fourier 级数的计算

### 1.1 方波函数的 Fourier 级数

**习题 1.1**. 计算 $f(x) = x$ 在 $[-\pi, \pi]$ 上的 Fourier 级数。

**解答**. 因 $f$ 为奇函数，所有 $a_n = 0$。

$$b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} x \sin(nx) dx = \frac{2}{\pi} \int_0^{\pi} x \sin(nx) dx$$

$$= \frac{2}{\pi} \left[ -\frac{x\cos(nx)}{n} \right]_0^{\pi} + \frac{2}{\pi n} \int_0^{\pi} \cos(nx) dx$$

$$= \frac{2}{\pi} \cdot \frac{-\pi(-1)^n}{n} + 0 = \frac{2(-1)^{n+1}}{n}.$$

故：

$$f(x) = \sum_{n=1}^{\infty} \frac{2(-1)^{n+1}}{n} \sin(nx), \quad x \in (-\pi, \pi).$$

在 $x = \pm\pi$ 处级数收敛到 $0$（左右极限的平均值）。$\square$

### 1.2 全波整流器的 Fourier 级数

**习题 1.2**. 计算 $f(x) = |\cos x|$ 在 $[-\pi, \pi]$ 上的 Fourier 级数。

**解答**. $f$ 为偶函数，$b_n = 0$。

$$a_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} |\cos x| dx = \frac{2}{\pi} \int_{-\pi/2}^{\pi/2} \cos x dx = \frac{4}{\pi}.$$

对 $n \geq 1$：

$$a_n = \frac{2}{\pi} \int_0^{\pi} |\cos x| \cos(nx) dx = \frac{2}{\pi} \left[ \int_0^{\pi/2} + \int_{\pi/2}^{\pi} \right].$$

计算得：当 $n$ 为奇数时 $a_n = 0$；当 $n = 2k$ 时：

$$a_{2k} = \frac{4}{\pi} \cdot \frac{(-1)^{k+1}}{4k^2 - 1}.$$

故：

$$|\cos x| = \frac{2}{\pi} + \frac{4}{\pi} \sum_{k=1}^{\infty} \frac{(-1)^{k+1}}{4k^2 - 1} \cos(2kx).$$

令 $x = 0$ 得：$1 = \frac{2}{\pi} + \frac{4}{\pi} \sum_{k=1}^{\infty} \frac{(-1)^{k+1}}{4k^2 - 1}$，可由此求级数和。$\square$

## 2. 收敛性与求和

### 2.1 Dirichlet 核与点态收敛

**习题 2.1**. 证明 Fourier 级数的部分和可写为：

$$S_N f(x) = \frac{1}{2\pi} \int_{-\pi}^{\pi} f(x-t) D_N(t) dt,$$

其中 Dirichlet 核 $D_N(t) = \sum_{n=-N}^{N} e^{int} = \frac{\sin((N+1/2)t)}{\sin(t/2)}$。

**证明**.

$$S_N f(x) = \sum_{n=-N}^{N} \hat{f}(n) e^{inx} = \sum_{n=-N}^{N} \left(\frac{1}{2\pi} \int_{-\pi}^{\pi} f(y) e^{-iny} dy\right) e^{inx}$$

$$= \frac{1}{2\pi} \int_{-\pi}^{\pi} f(y) \sum_{n=-N}^{N} e^{in(x-y)} dy = \frac{1}{2\pi} \int_{-\pi}^{\pi} f(x-t) D_N(t) dt.$$

求和公式：$\sum_{n=-N}^{N} e^{int} = e^{-iNt} \frac{1 - e^{i(2N+1)t}}{1 - e^{it}} = \frac{\sin((N+1/2)t)}{\sin(t/2)}$。$\square$

### 2.2 Parseval 等式

**习题 2.2**. 证明 Parseval 等式：对 $f \in L^2(\mathbb{T})$，

$$\frac{1}{2\pi} \int_{-\pi}^{\pi} |f(x)|^2 dx = \sum_{n=-\infty}^{\infty} |\hat{f}(n)|^2.$$

**证明**. 三角多项式在 $L^2$ 中稠密。对三角多项式 $f$，直接计算：

$$\|f\|_2^2 = \langle f, f \rangle = \sum_{m,n} \hat{f}(m)\overline{\hat{f}(n)} \langle e^{imx}, e^{inx} \rangle = \sum_n |\hat{f}(n)|^2.$$

一般 $f \in L^2$ 用 $S_N f \to f$（$L^2$ 中）及范数的连续性。$\square$

## 3. Fourier 变换

### 3.1 基本性质

**习题 3.1**. 设 $f \in L^1(\mathbb{R})$，$\hat{f}(\xi) = \int_{-\infty}^{\infty} f(x) e^{-2\pi i x \xi} dx$。证明：

1. $\widehat{f * g} = \hat{f} \cdot \hat{g}$；
2. $\widehat{f'}(\xi) = 2\pi i \xi \hat{f}(\xi)$（若 $f'$ 存在且可积）。

**证明**.

1. $$\widehat{f * g}(\xi) = \int\left(\int f(y)g(x-y)dy\right) e^{-2\pi i x \xi} dx$$
   $$= \int f(y) e^{-2\pi i y \xi} \left(\int g(x-y) e^{-2\pi i (x-y)\xi} dx\right) dy = \hat{f}(\xi)\hat{g}(\xi).$$
2. 分部积分：$\int f'(x) e^{-2\pi i x \xi} dx = [f(x)e^{-2\pi i x \xi}] + 2\pi i \xi \int f(x) e^{-2\pi i x \xi} dx = 2\pi i \xi \hat{f}(\xi)$（边界项消失）。$\square$

### 3.2 Poisson 求和公式

**习题 3.2**. 证明 Poisson 求和公式：对充分好的函数 $f$，

$$\sum_{n=-\infty}^{\infty} f(n) = \sum_{k=-\infty}^{\infty} \hat{f}(k).$$

**证明概要**. 令 $F(x) = \sum_{n} f(x+n)$，$F$ 为周期 1 函数。其 Fourier 系数：

$$\hat{F}(k) = \int_0^1 F(x) e^{-2\pi i k x} dx = \sum_n \int_0^1 f(x+n) e^{-2\pi i k x} dx$$

$$= \int_{-\infty}^{\infty} f(y) e^{-2\pi i k y} dy = \hat{f}(k).$$

故 $F(x) = \sum_k \hat{f}(k) e^{2\pi i k x}$。取 $x = 0$ 即得。$\square$

## 4. 例子

### 4.1 例子：热核的 Fourier 变换

**习题 4.1**. 求热方程基本解 $G_t(x) = \frac{1}{\sqrt{4\pi t}} e^{-x^2/(4t)}$ 的 Fourier 变换。

**解答**. 利用 Gaussian 的 Fourier 变换：$\widehat{e^{-\pi x^2}} = e^{-\pi \xi^2}$。尺度变换得：

$$\hat{G}_t(\xi) = e^{-4\pi^2 t \xi^2}.$$

验证：$\widehat{\partial_t G_t} = \partial_t \hat{G}_t = -4\pi^2 \xi^2 \hat{G}_t = \widehat{\partial_x^2 G_t}$，与热方程一致。$\square$

### 4.2 例子：$oldsymbol{\zeta(2)}$ 的计算

**习题 4.2**. 利用 Fourier 级数计算 $\zeta(2) = \sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6}$。

**解答**. 对 $f(x) = x$ 在 $[-\pi, \pi]$ 上，Parseval 等式：

$$\frac{1}{2\pi} \int_{-\pi}^{\pi} x^2 dx = \frac{\pi^2}{3} = \sum_{n \neq 0} |\hat{f}(n)|^2 = 2\sum_{n=1}^{\infty} \left(\frac{(-1)^{n+1}}{n}\right)^2 = 2\sum_{n=1}^{\infty} \frac{1}{n^2}.$$

（注意：直接用 $f(x) = x$ 的展开得系数 $b_n = 2(-1)^{n+1}/n$，Parseval 给出 $\frac{\pi^2}{3} = \sum b_n^2/2 = 2\sum 1/n^2$。）整理得 $\zeta(2) = \pi^2/6$。$\square$
