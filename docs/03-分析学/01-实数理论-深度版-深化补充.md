## 一、证明完善与证明思路

### 1.1 Dedekind分割完备性 - 详细证明

**定理** (Dedekind分割的完备性)：Dedekind分割的集合 $R$ 构成完备有序域。

**完整证明**：

**第一步：定义序关系**

对两个Dedekind分割 $(A_1, B_1)$ 和 $(A_2, B_2)$，定义：
$$(A_1, B_1)   rianglelefteq (A_2, B_2)   ext{ 当且仅当 } A_1   rianglelefteq A_2$$

**验证序公理**：

- 三歧性：对任意两个分割，$A_1   rianglelefteq A_2$ 或 $A_2   rianglelefteq A_1$ 必有一个成立（由集合包含的三分性）
- 传递性：$A_1   rianglelefteq A_2$ 且 $A_2   rianglelefteq A_3$ 则 $A_1   rianglelefteq A_3$

**第二步：定义加法**

$$(A_1, B_1) + (A_2, B_2) = (A_1 + A_2, B_1 + B_2)$$

其中 $A_1 + A_2 = \{a_1 + a_2 : a_1 \in A_1, a_2 \in A_2\}$

**验证加法性质**：

- 封闭性：$A_1 + A_2$ 非空（因 $A_1, A_2$ 非空），$B_1 + B_2$ 非空
- $A_1 + A_2$ 无最大元：若 $a_1 + a_2 \in A_1 + A_2$，则 $\exists a_1' > a_1$ 于 $A_1$，故 $a_1' + a_2 > a_1 + a_2$ 仍在 $A_1 + A_2$
- 分割性质：$\forall a \in A_1 + A_2, \forall b \in B_1 + B_2$，有 $a = a_1 + a_2 < b_1 + b_2 = b$

**第三步：定义乘法（正数情形）**

对正分割 $(A, B)$（即 $A$ 含正有理数），定义：
$$(A_1, B_1) \cdot (A_2, B_2) = (A_1 \cdot A_2, B_1 \cdot B_2)$$

其中 $A_1 \cdot A_2 = \{a_1 \cdot a_2 : a_1 \in A_1, a_2 \in A_2, a_1 > 0, a_2 > 0\} \cup \{q \in \mathbb{Q} : q \leq 0\}$

**第四步：证明完备性（确界原理）**

设 $S \subseteq R$ 非空有上界。令：
$$A = \bigcup_{(A_i, B_i) \in S} A_i$$

**验证 $(A, \mathbb{Q} \setminus A)$ 是Dedekind分割**：

1. $A \neq \emptyset$（因 $S$ 非空）
2. $A \neq \mathbb{Q}$（因 $S$ 有上界，存在上界 $(A_0, B_0)$，$A \subseteq A_0 \neq \mathbb{Q}$）
3. $A$ 对小于关系封闭：若 $q \in A$，则 $\exists (A_i, B_i) \in S$ 使 $q \in A_i$，故 $\forall q' < q$，$q' \in A_i \subseteq A$
4. $A$ 无最大元：若 $q \in A$，则 $\exists (A_i, B_i)$ 使 $q \in A_i$，因 $A_i$ 无最大元，$\exists q' > q$ 于 $A_i \subseteq A$

**证明 $(A, \mathbb{Q} \setminus A)$ 是最小上界**：

- 上界性：$\forall (A_i, B_i) \in S$，$A_i \subseteq A$，故 $(A_i, B_i) \trianglelefteq (A, \mathbb{Q} \setminus A)$
- 最小性：设 $(C, D)$ 是另一上界，则 $\forall (A_i, B_i) \in S$，$A_i \subseteq C$，故 $A = \bigcup A_i \subseteq C$，即 $(A, \mathbb{Q} \setminus A) \trianglelefteq (C, D)$

**证明思路**：将实数视为有理数的"分割"，利用有理数的完备性来构造实数的完备性。关键思想是把实数的"界限"本身作为实数。

### 1.2 Cauchy序列完备性 - 详细ε-N证明

**定理**：Cauchy序列等价类的集合构成完备度量空间。

**证明Cauchy条件**：

设 $(q_n)$ 是Cauchy序列，证明：
$$\forall \varepsilon > 0, \exists N \in \mathbb{N}, \forall m, n \geq N: |q_m - q_n| < \varepsilon$$

**详细推导**：

**步骤1：证明Cauchy序列有界**

取 $\varepsilon = 1$，$\exists N_1$ 使 $\forall n \geq N_1$，$|q_n - q_{N_1}| < 1$

则 $\forall n \geq N_1$，$|q_n| \leq |q_{N_1}| + 1$

令 $M = \max\{|q_1|, |q_2|, ..., |q_{N_1-1}|, |q_{N_1}| + 1\}$，则 $\forall n$，$|q_n| \leq M$

**步骤2：证明收敛子列存在**（Bolzano-Weierstrass对有界序列）

**步骤3：证明Cauchy序列收敛于子列极限**

设子列 $q_{n_k} \to L$。给定 $\varepsilon > 0$：

- $\exists K: \forall k \geq K$，$|q_{n_k} - L| < \varepsilon/2$
- $\exists N: \forall m, n \geq N$，$|q_m - q_n| < \varepsilon/2$

取 $k$ 使得 $n_k \geq \max(N, n_K)$，则对 $\forall n \geq N$：
$$|q_n - L| \leq |q_n - q_{n_k}| + |q_{n_k} - L| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon$$

**证明思路**：利用有理数稠密性和序列有界性，通过构造收敛子列来证明Cauchy序列收敛。

### 1.3 区间套定理 - 详细ε证明

**定理**：设 $[a_1, b_1] \supseteq [a_2, b_2] \supseteq ...$ 为闭区间套，且 $\lim_{n \to \infty} (b_n - a_n) = 0$，则存在唯一的 $c \in \bigcap_{n=1}^{\infty} [a_n, b_n]$。

**证明**：

**唯一存在性**：

序列 $\{a_n\}$ 单调递增：$a_1 \leq a_2 \leq ... \leq a_n \leq ...$
序列 $\{b_n\}$ 单调递减：$b_1 \geq b_2 \geq ... \geq b_n \geq ...$

且 $a_n \leq b_n$ 对所有 $n$ 成立。

因此 $\{a_n\}$ 有上界 $b_1$，$\{b_n\}$ 有下界 $a_1$。

由单调有界定理：

- $\exists a = \lim_{n \to \infty} a_n$
- $\exists b = \lim_{n \to \infty} b_n$

**证明 $a = b$**：

由 $b_n - a_n \to 0$，且 $b_n - a_n \geq 0$，得：
$$b - a = \lim_{n \to \infty} (b_n - a_n) = 0$$

故 $a = b = c$。

**证明 $c \in [a_n, b_n]$ 对所有 $n$**：

对任意固定的 $n$，$\forall m \geq n$：
$$a_n \leq a_m \leq b_m \leq b_n$$

令 $m \to \infty$，由保序性：
$$a_n \leq c \leq b_n$$

**唯一性证明**：

设 $c_1, c_2 \in \bigcap_{n=1}^{\infty} [a_n, b_n]$，则：
$$|c_1 - c_2| \leq b_n - a_n \text{ 对所有 } n$$

由 $b_n - a_n \to 0$，得 $|c_1 - c_2| = 0$，即 $c_1 = c_2$。

**证明思路**：利用单调有界原理和夹逼定理，关键是证明区间端点序列收敛于同一点。

## 二、习题扩充（共15道）

### 2.1 基础计算题（5道）

**题1**：用Dedekind分割表示实数 $\sqrt{2}$，写出其左集 $A$ 和右集 $B$ 的明确形式。

**解答**：
$$A = \{q \in \mathbb{Q} : q \leq 0 \text{ 或 } q^2 < 2\}$$
$$B = \{q \in \mathbb{Q} : q > 0 \text{ 且 } q^2 > 2\}$$

**题2**：验证序列 $a_n = \frac{n}{n+1}$ 是Cauchy序列，并求其极限。

**解答**：
$$|a_m - a_n| = \left|\frac{m}{m+1} - \frac{n}{n+1}\right| = \frac{|m-n|}{(m+1)(n+1)}$$

对 $\varepsilon > 0$，取 $N > \frac{1}{\varepsilon}$，则 $\forall m, n \geq N$：
$$|a_m - a_n| < \frac{1}{N} < \varepsilon$$

极限为 $\lim_{n \to \infty} \frac{n}{n+1} = 1$。

**题3**：求区间套 $[a_n, b_n] = [0, 1 + \frac{1}{n}]$ 的交集。

**解答**：
$\{a_n\} = \{0, 0, 0, ...\}$，$\lim a_n = 0$
$\{b_n\} = \{2, 1.5, 1.33..., ...\}$，$\lim b_n = 1$

但 $a_n = 0$ 不对所有 $n$ 属于 $[0, 1]$，实际上：
$$\bigcap_{n=1}^{\infty} [0, 1 + \frac{1}{n}] = [0, 1]$$

**题4**：给定分划 $P = \{0, 0.3, 0.7, 1\}$ 和 $f(x) = x^2$，计算Darb上和与Darb下和。

**解答**：

- $\Delta x_1 = 0.3$，$M_1 = 0.09$，$m_1 = 0$
- $\Delta x_2 = 0.4$，$M_2 = 0.49$，$m_2 = 0.09$
- $\Delta x_3 = 0.3$，$M_3 = 1$，$m_3 = 0.49$

$U(f, P) = 0.3 \times 0.09 + 0.4 \times 0.49 + 0.3 \times 1 = 0.027 + 0.196 + 0.3 = 0.523$
$L(f, P) = 0.3 \times 0 + 0.4 \times 0.09 + 0.3 \times 0.49 = 0 + 0.036 + 0.147 = 0.183$

**题5**：将十进制数 $3.14159...$ 表示为Dedekind分割。

**解答**：
$$A = \{q \in \mathbb{Q} : q < 3.14159...\}$$

更明确地，$A$ 包含所有小于 $\pi$ 的有理数，可通过截断小数来逼近。

### 2.2 概念理解题（5道）

**题6**（判断题）：所有有界序列都是Cauchy序列。

**答案**：**错误**

**解释**：反例 $a_n = (-1)^n$ 有界但不收敛，故不是Cauchy序列。

**题7**（选择题）：下列哪个条件可推出序列收敛？

A. 有界
B. 单调
C. 是Cauchy序列
D. 子列收敛

**答案**：**C**

**解释**：在完备空间中，Cauchy序列等价于收敛。A、B需结合，D需要所有子列收敛于同一极限。

**题8**（简答题）：解释为什么Dedekind分割要求"左集无最大元"。

**答案**：
若不满足此条件，存在有理数对应的分割会有两种表示：

- 作为有理数 $r$ 本身的分割
- 作为左集为 $\{q : q < r\}$ 的分割

为避免重复，要求左集无最大元，使每个实数有唯一表示。

**题9**（判断题）：若 $\{[a_n, b_n]\}$ 是区间套且 $b_n - a_n \to 0$，则交集非空。

**答案**：**正确**（在实数系中）

**解释**：这是区间套定理的结论，实数完备性保证了唯一公共点存在。

**题10**（简答题）：描述构造主义与经典数学对"存在性"的不同理解。

**答案**：

- **构造主义**：存在性必须给出具体的构造方法，排斥纯存在性证明
- **经典数学**：接受排中律和反证法证明的存在性，不必然要求构造

### 2.3 证明题（5道）

**题11**（简单）：证明两个Cauchy序列的和仍是Cauchy序列。

**证明**：
设 $(a_n)$, $(b_n)$ 为Cauchy序列。给定 $\varepsilon > 0$：

- $\exists N_1: \forall m, n \geq N_1$，$|a_m - a_n| < \varepsilon/2$
- $\exists N_2: \forall m, n \geq N_2$，$|b_m - b_n| < \varepsilon/2$

取 $N = \max(N_1, N_2)$，则 $\forall m, n \geq N$：
$$|(a_m + b_m) - (a_n + b_n)| \leq |a_m - a_n| + |b_m - b_n| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon$$

**题12**（简单）：证明单调递增有上界的序列必为Cauchy序列。

**证明**：
设 $(a_n)$ 单调递增有上界，则 $\lim a_n = L = \sup\{a_n\}$ 存在。

给定 $\varepsilon > 0$，$\exists N: a_N > L - \varepsilon$。

对 $\forall m, n \geq N$（设 $m \geq n$）：
$$|a_m - a_n| = a_m - a_n \leq L - a_N < \varepsilon$$

**题13**（中等）：证明Dedekind分割集合上的加法满足结合律。

**证明**：
设 $\alpha = (A_1, B_1)$，$\beta = (A_2, B_2)$，$\gamma = (A_3, B_3)$。

$(\alpha + \beta) + \gamma$ 的左集为：
$$(A_1 + A_2) + A_3 = \{(a_1 + a_2) + a_3 : a_i \in A_i\}$$

$\alpha + (\beta + \gamma)$ 的左集为：
$$A_1 + (A_2 + A_3) = \{a_1 + (a_2 + a_3) : a_i \in A_i\}$$

由有理数加法结合律，$(a_1 + a_2) + a_3 = a_1 + (a_2 + a_3)$，故两分割相等。

**题14**（中等）：设 $S \subseteq \mathbb{R}$ 非空有下界，证明 $S$ 有最大下界（下确界）。

**证明**：
令 $T = \{-x : x \in S\}$，则 $T$ 非空有上界。

由上确界原理，$\sup T$ 存在。

令 $L = -\sup T$。

**验证**：

1. **下界性**：$\forall x \in S$，$-x \leq \sup T$，故 $x \geq -\sup T = L$
2. **最大性**：设 $M > L$ 为另一下界，则 $-M < -L = \sup T$。但 $-M$ 不是 $T$ 的上界，$\exists x \in S$ 使 $-x > -M$，即 $x < M$，矛盾。

**题15**（困难）：证明完备性的六种等价形式（确界原理、单调有界定理、区间套定理、Bolzano-Weierstrass定理、Cauchy收敛准则、Heine-Borel定理）的循环等价性。

**证明**（完成循环 $(5) \Rightarrow (6) \Rightarrow (1)$）：

**$(5) \Rightarrow (6)$**（Cauchy收敛 $\Rightarrow$ Heine-Borel）：

设 $[a, b]$ 有开覆盖 $\mathcal{U}$。假设无有限子覆盖。

二分 $[a, b]$，至少一半无有限子覆盖，记为 $[a_1, b_1]$。

继续二分，得区间套 $\{[a_n, b_n]\}$，$b_n - a_n = (b-a)/2^n \to 0$。

由区间套定理，$\exists! c \in \bigcap [a_n, b_n]$。

因 $\mathcal{U}$ 覆盖 $[a,b]$，$\exists U \in \mathcal{U}$ 使 $c \in U$。

$U$ 开，$\exists \delta > 0: (c-\delta, c+\delta) \subseteq U$。

当 $n$ 足够大，$[a_n, b_n] \subseteq (c-\delta, c+\delta) \subseteq U$，矛盾。

**$(6) \Rightarrow (1)$**（Heine-Borel $\Rightarrow$ 确界原理）：

设 $S$ 非空有上界。假设 $S$ 无最小上界。

令 $\mathcal{U} = \{(-\infty, s) : s \in S\} \cup \{(b, +\infty) : b \text{ 是 } S \text{ 的上界}\}$。

可证 $\mathcal{U}$ 覆盖 $\mathbb{R}$ 但任何有限子集不覆盖 $[\inf S, \sup S + 1]$，矛盾。

## 三、示例扩充（每个概念≥5个示例）

### 3.1 Dedekind分割示例

**示例1**：表示 $\sqrt{2}$
$$A = \{q \in \mathbb{Q} : q \leq 0 \text{ 或 } q^2 < 2\}, \quad B = \{q \in \mathbb{Q} : q > 0, q^2 > 2\}$$

**示例2**：表示有理数 $3/4$
$$A = \{q \in \mathbb{Q} : q < 3/4\}, \quad B = \{q \in \mathbb{Q} : q \geq 3/4\}$$

**示例3**：表示 $e$（自然对数的底）
$$A = \{q \in \mathbb{Q} : q < e\}, \quad B = \{q \in \mathbb{Q} : q > e\}$$
可用 $e = \sum_{n=0}^{\infty} \frac{1}{n!}$ 逼近。

**示例4**：表示 $-\sqrt{3}$
$$A = \{q \in \mathbb{Q} : q < 0, q^2 > 3\} \cup \{q \in \mathbb{Q} : q \leq 0\}$$

**示例5**：表示 $\pi$
$$A = \{q \in \mathbb{Q} : q < \pi\}, \quad B = \{q \in \mathbb{Q} : q > \pi\}$$
可用多边形逼近圆的周长来定义。

### 3.2 Cauchy序列示例

**示例1**：$a_n = \frac{1}{n}$
$$|a_m - a_n| = \frac{|m-n|}{mn} < \frac{1}{\min(m,n)} \to 0$$

**示例2**：$a_n = \sum_{k=1}^{n} \frac{1}{k^2}$
对 $m > n$：
$$|a_m - a_n| = \sum_{k=n+1}^{m} \frac{1}{k^2} < \sum_{k=n+1}^{\infty} \frac{1}{k^2} < \frac{1}{n}$$

**示例3**：$a_n = \frac{n+1}{n} = 1 + \frac{1}{n}$
$$|a_m - a_n| = \left|\frac{1}{m} - \frac{1}{n}\right| < \frac{1}{m} + \frac{1}{n}$$

**示例4**：$a_n = \sin(n)/n$
$$|a_m - a_n| \leq \frac{|\sin m|}{m} + \frac{|\sin n|}{n} \leq \frac{1}{m} + \frac{1}{n}$$

**示例5**（边界情况）：$a_n = \sum_{k=1}^{n} \frac{(-1)^{k-1}}{k}$（交错调和级数部分和）

$$|a_m - a_n| \leq \frac{1}{n+1} \text{（对 } m > n\text{）}$$

### 3.3 区间套示例

**示例1**：$[0, 1 + 1/n]$，交集 $[0, 1]$

**示例2**：$[1 - 1/n, 1 + 1/n]$，交集 $\{1\}$

**示例3**：$[0, 1/n]$，交集 $\{0\}$

**示例4**：$[1/2^n, 1/2^{n-1}]$，交集为空？

**注意**：此例不满足区间套条件（非嵌套），修改为 $[0, 1/2^n]$，交集 $\{0\}$。

**示例5**（边界）：$[n, n + 1/n]$ 在 $n \to \infty$ 时交集为空，但此区间套不收敛于有限区间。

## 四、反例专节（每个概念≥3个反例）

### 4.1 Cauchy序列相关反例

**反例1**：有界但不Cauchy的序列

序列 $a_n = (-1)^n$。

- **有界性**：$|a_n| = 1$ 对所有 $n$
- **非Cauchy**：取 $\varepsilon = 1$，$\forall N$，$|a_{N+1} - a_{N+2}| = |(-1)^{N+1} - (-1)^{N+2}| = 2 > 1$

**反例2**：Cauchy但在有理数中不收敛的序列

令 $a_n = 1 + \sum_{k=1}^{n} \frac{1}{k!}$，这是 $e$ 的有理逼近序列。

- 在 $\mathbb{R}$ 中收敛于 $e$
- 在 $\mathbb{Q}$ 中不收敛（因 $e$ 无理）

**反例3**：子列收敛但序列本身不Cauchy

$a_n = \sin(\frac{n\pi}{4})$。子列 $a_{8k} = 0$ 收敛，但序列在 $\{-1, -\frac{\sqrt{2}}{2}, 0, \frac{\sqrt{2}}{2}, 1\}$ 中振荡，不是Cauchy序列。

### 4.2 区间套定理的反例

**反例1**：开区间套

$(0, 1/n)$，交集 $\bigcap_{n=1}^{\infty} (0, 1/n) = \emptyset$。

**错在哪里**：开区间不包含端点，$0$ 不属于任何 $(0, 1/n)$。

**反例2**：长度不趋于零

$[0, 1 + 1/n]$，交集 $[0, 1]$（非单点）。

**错在哪里**：长度不趋于零，交集不是一个点。

**反例3**：非嵌套区间

$[n, n+1]$，无交集。

**错在哪里**：不满足嵌套条件 $[a_n, b_n] \supseteq [a_{n+1}, b_{n+1}]$。

### 4.3 Dedekind分割的反例

**反例1**：左集有最大元的分割

若允许，$r$ 对应的分割不唯一：

- $(A_1, B_1)$ 其中 $A_1 = \{q : q \leq r\}$，$B_1 = \{q : q > r\}$
- $(A_2, B_2)$ 其中 $A_2 = \{q : q < r\}$，$B_2 = \{q : q \geq r\}$

**错在哪里**：违反Dedekind分割定义的唯一性要求。

```
区间套示意：

第1层: [===================]  长度 = b₁ - a₁
第2层:    [=============]     长度 = b₂ - a₂
第3层:       [========]        长度 = b₃ - a₃
第4层:          [====]          长度 = b₄ - a₄
...               ...
第n层:              [•]          长度 → 0
                    ↑
                  唯一公共点
```

**几何解释**：想象一个无限套娃，每一层比上一层更小，但都包含前一层。当层数趋于无穷时，所有层收缩到一个点。

### 5.4 Bolzano-Weierstrass定理的几何直观

**图示描述**：

```
二分法找聚点：

原始区间:    [========================] 含无限点
第1次二分:   [========]                (右半无无限点)
第2次二分:       [====]                (左半无无限点)
第3次二分:         [==]                (右半无无限点)
第4次二分:           []                收缩到聚点
             ↑
           聚点
```

**几何解释**：在有限区间内有无限多个点，无论怎么细分，总有一个小区间包含无限多个点，这个收缩过程确定一个聚点。
