# 偏微分方程高级主题简介

偏微分方程（PDE）的现代理论涵盖非线性波动、自由边界、几何演化方程等深刻主题。本节介绍孤立子理论、Navier-Stokes 方程的正则性等核心问题，展示 PDE 与几何、拓扑、可积系统的深刻联系。

## 1. 可积系统与孤立子

### 1.1 KdV 方程

**定义 1.1（KdV 方程）**. Korteweg-de Vries 方程为：

$$u_t + 6uu_x + u_{xxx} = 0, \quad x \in \mathbb{R},\ t > 0. \tag{1.1}$$

它描述浅水长波的运动，于 1895 年由 Korteweg 与 de Vries 导出。

**定理 1.2（单孤立子解）**. 对任意 $c > 0$ 和 $x_0 \in \mathbb{R}$，

$$u(x,t) = \frac{c}{2} \operatorname{sech}^2\left( \frac{\sqrt{c}}{2}(x - ct - x_0) \right)$$

是 (1.1) 的解，称为速度为 $c$ 的单孤立子（soliton）。

*验证*. 直接计算：设 $\xi = x - ct - x_0$，$\phi(\xi) = \frac{c}{2}\operatorname{sech}^2(\frac{\sqrt{c}}{2}\xi)$。则：

$$\phi' = -\frac{c^{3/2}}{2} \operatorname{sech}^2 \tanh, \quad \phi'' = \frac{c^2}{2}\operatorname{sech}^2(3\tanh^2 - 1), \quad \phi''' = \dots$$

代入 $-c\phi' + 6\phi\phi' + \phi'''$ 并利用 $\operatorname{sech}^2 + \tanh^2 = 1$ 可验证为零。$\square$

**定理 1.3（逆散射变换）**. KdV 方程可通过逆散射变换（Inverse Scattering Transform, IST）求解：将非线性 PDE 转化为线性谱问题。

*概要*. 考虑 Schrödinger 算子 $L = -\partial_x^2 + u(x,t)$。KdV 方程等价于 Lax 对：

$$L_t = [P, L] = PL - LP, \quad P = -4\partial_x^3 + 3u\partial_x + 3\partial_x u.$$

由此 $L$ 的谱不变，散射数据随时间线性演化，通过逆散射重构 $u(x,t)$。

### 1.2 非线性 Schrödinger 方程

**定义 1.4（NLS）**. 聚焦型非线性 Schrödinger 方程为：

$$i\psi_t + \psi_{xx} + 2|\psi|^2\psi = 0. \tag{1.2}$$

**命题 1.5**. 方程 (1.2) 具有守恒量：
1. **质量**：$M = \int_{-\infty}^{\infty} |\psi|^2 dx$；
2. **能量**：$E = \int_{-\infty}^{\infty} (|\psi_x|^2 - |\psi|^4) dx$；
3. **动量**：$P = \operatorname{Im} \int_{-\infty}^{\infty} \bar{\psi} \psi_x dx$。

## 2. 自由边界问题

### 2.1 Stefan 问题

**定义 2.1（一相 Stefan 问题）**. 求区域 $Q = \{(x,t) : 0 < x < s(t),\ 0 < t < T\}$ 中的 $u(x,t)$ 和自由边界 $x = s(t)$ 使得：

$$\begin{cases} u_t = u_{xx} & 0 < x < s(t),\ 0 < t < T, \\ u(0,t) = f(t) \geq 0 & 0 < t < T, \\ u(s(t), t) = 0 & 0 < t < T, \\ u_x(s(t), t) = -\dot{s}(t) & 0 < t < T, \\ s(0) = b. \end{cases} \tag{2.1}$$

最后一个条件为 **Stefan 条件**，表示相变潜热与热流的平衡。

**定理 2.2（自相似解）**. 对 $f(t) \equiv 1$，Stefan 问题存在自相似解 $u(x,t) = v(\xi)$，$s(t) = 2\lambda\sqrt{t}$，其中 $\xi = x/(2\sqrt{t})$，$\lambda$ 由超越方程确定。

*推导*. 设 $u(x,t) = v(\xi)$，$\xi = x/(2\sqrt{t})$，则：

$$u_t = -\frac{\xi}{2t} v'(\xi), \quad u_{xx} = \frac{1}{4t} v''(\xi).$$

方程 $u_t = u_{xx}$ 化为 $v'' + 2\xi v' = 0$，解为 $v(\xi) = A \operatorname{erf}(\xi) + B$。由边界条件确定 $A, B, \lambda$。$\square$

## 3. 几何 PDE

### 3.1 平均曲率流

**定义 3.1（平均曲率流）**. 设 $M_0 \subset \mathbb{R}^{n+1}$ 为超曲面。平均曲率流（Mean Curvature Flow, MCF）是单参数超曲面族 $M_t$ 满足：

$$\frac{\partial x}{\partial t} = H(x,t) \nu(x,t), \quad x \in M_t,$$

其中 $H$ 为平均曲率，$\nu$ 为单位内法向量。

**定理 3.2（球面收缩）**. 半径为 $R_0$ 的球面在 MCF 下收缩为：

$$R(t) = \sqrt{R_0^2 - 2nt},$$

在有限时间 $T = R_0^2/(2n)$ 内收缩为一点。

*证明*. 球面的平均曲率为 $H = n/R$，法向运动方程为 $\dot{R} = -n/R$，积分得 $R^2 = R_0^2 - 2nt$。$\square$

**定理 3.3（Huisken, 1984）**. 若 $M_0 \subset \mathbb{R}^{n+1}$ 为凸闭超曲面，则 MCF 在有限时间内收缩为"round point"：适当缩放后趋于球面。

## 4. Navier-Stokes 方程

### 4.1 方程与正则性

**定义 4.1（Navier-Stokes）**. 不可压 Navier-Stokes 方程为：

$$\begin{cases} \partial_t u + (u \cdot \nabla) u + \nabla p = \nu \Delta u + f & \text{in } \mathbb{R}^3 \times (0, T), \\ \nabla \cdot u = 0 & \text{in } \mathbb{R}^3 \times (0, T), \\ u(x, 0) = u_0(x). \end{cases} \tag{4.1}$$

**定理 4.2（Leray-Hopf 弱解）**. 对 $u_0 \in L^2(\mathbb{R}^3)$，$\nabla \cdot u_0 = 0$，存在弱解 $u \in L^\infty(0,T; L^2) \cap L^2(0,T; H^1)$ 满足能量不等式：

$$\frac{1}{2}\|u(t)\|_{L^2}^2 + \nu \int_0^t \|\nabla u\|_{L^2}^2 ds \leq \frac{1}{2}\|u_0\|_{L^2}^2.$$

**定理 4.3（Caffarelli-Kohn-Nirenberg）**. Navier-Stokes 方程的 suitable weak solution 的一维 Hausdorff 测度奇点集为零。

### 4.2 例子：Taylor-Green 涡

二维 Taylor-Green 涡是 Navier-Stokes 的显式解：

$$u(x,y,t) = (\sin x \cos y, -\cos x \sin y) F(t),$$

代入方程得 $F'(t) = -2\nu F(t)$，故 $F(t) = e^{-2\nu t}$。此解展示了粘性耗散机制。
