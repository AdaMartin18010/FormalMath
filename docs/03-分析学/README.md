## 📋 分支概览

分析学是数学的核心分支，研究变化、极限和连续性的数学理论。本分支涵盖：

- **实分析**：极限、连续性、微分、积分、级数
- **复分析**：解析函数、复积分、留数理论
- **泛函分析**：无穷维空间、算子理论
- **微分方程**：ODE与PDE理论与应用
- **调和分析**：Fourier分析、偏微分方程
- **测度论**：现代积分理论的基础

**难度等级**: ⭐⭐⭐（进阶级）
