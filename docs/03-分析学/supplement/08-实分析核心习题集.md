# 实分析核心习题集

实分析是研究 $\mathbb{R}^n$ 上测度、积分、极限与函数空间的数学分支，由 Lebesgue、Borel、Radon 与 Nikodym 等人于 20 世纪初创立。本节通过精选习题展示测度构造、收敛定理、$L^p$ 空间与微分理论的核心技巧。

## 1. 测度论基础

### 1.1 Lebesgue 外测度

**习题 1.1**. 证明 Lebesgue 外测度 $m^*$ 满足：

1. $m^*(\emptyset) = 0$；
2. 单调性：$A \subseteq B \Rightarrow m^*(A) \leq m^*(B)$；
3. 次可数可加性：$m^*(\bigcup_n A_n) \leq \sum_n m^*(A_n)$。

**证明**.

1. 对任意 $\varepsilon > 0$，$\emptyset \subseteq (-\varepsilon/2, \varepsilon/2)$，故 $m^*(\emptyset) \leq \varepsilon$。令 $\varepsilon \to 0$。
2. $A$ 的任何区间覆盖也是 $B$ 的覆盖，取下确界即得。
3. 对任意 $\varepsilon > 0$，取覆盖 $\{I_{n,k}\}_k$ 使 $\sum_k |I_{n,k}| < m^*(A_n) + \varepsilon/2^n$。则 $\{I_{n,k}\}_{n,k}$ 覆盖 $\bigcup_n A_n$，且：
   $$m^*(\bigcup_n A_n) \leq \sum_{n,k} |I_{n,k}| < \sum_n m^*(A_n) + \varepsilon.$$
   令 $\varepsilon \to 0$。$\square$

### 1.2 Carathéodory 可测性

**习题 1.2**. 证明 $E$ 可测当且仅当对任意 $A \subseteq \mathbb{R}$：

$$m^*(A) = m^*(A \cap E) + m^*(A \cap E^c).$$

**证明**.

$(\Rightarrow)$ 设 $E$ 可测。$\leq$ 由次可加性。对 $\geq$，取开集 $G \supseteq A$ 使 $m(G) < m^*(A) + \varepsilon$。则：

$$m^*(A \cap E) + m^*(A \cap E^c) \leq m(G \cap E) + m(G \cap E^c) = m(G) < m^*(A) + \varepsilon.$$

$(\Leftarrow)$ 对任意 $A$，取 $G \supseteq E$ 开集使 $m(G) < m^*(E) + \varepsilon$。对 $A = G$ 用条件：

$$m^*(G) = m^*(E) + m^*(G \setminus E) \Rightarrow m^*(G \setminus E) = m^*(G) - m^*(E) < \varepsilon.$$

故 $E$ 可测。$\square$

## 2. Lebesgue 积分与收敛定理

### 2.1 单调收敛定理

**定理 2.1（单调收敛, MCT）**. 设 $0 \leq f_1 \leq f_2 \leq \dots \to f$ a.e.，则：

$$\int f = \lim_{n \to \infty} \int f_n.$$

**习题 2.1**. 计算 $\lim_{n \to \infty} \int_0^1 \frac{n \sin x}{1 + n^2 x^2} dx$。

**解答**. 令 $f_n(x) = \frac{n \sin x}{1 + n^2 x^2}$。

- 逐点极限：$x > 0$ 时 $f_n(x) \sim \frac{nx}{n^2x^2} = \frac{1}{nx} \to 0$；$f_n(0) = 0$。
- 控制函数：$|f_n(x)| \leq \frac{n}{1+n^2x^2}$。令 $t = nx$，$\int_0^1 \frac{n}{1+n^2x^2}dx = \int_0^n \frac{dt}{1+t^2} = \arctan n \to \frac{\pi}{2}$。

由 DCT，极限 $= \int_0^1 0 = 0$。$\square$

### 2.2 Fatou 引理与反例

**定理 2.2（Fatou）**. 对非负可测函数列：

$$\int \liminf f_n \leq \liminf \int f_n.$$

**习题 2.2**. 构造严格不等式的例子。

**解答**. 令 $f_n = \mathbf{1}_{[n, n+1]}$。则 $\liminf f_n = 0$，$\int f_n = 1$。故：

$$\int \liminf f_n = 0 < 1 = \liminf \int f_n. \quad \square$$

## 3. $L^p$ 空间

### 3.1 Hölder 不等式

**习题 3.1**. 设 $f \in L^p$，$g \in L^q$，$\frac{1}{p} + \frac{1}{q} = \frac{1}{r}$。证明 $fg \in L^r$ 且 $\|fg\|_r \leq \|f\|_p \|g\|_q$。

**证明**. 令 $F = |f|^r \in L^{p/r}$，$G = |g|^r \in L^{q/r}$，且 $\frac{r}{p} + \frac{r}{q} = 1$。由 Hölder：

$$\int |fg|^r = \int FG \leq \|F\|_{p/r} \|G\|_{q/r} = \|f\|_p^r \|g\|_q^r.$$

取 $r$ 次方根即得。$\square$

### 3.2 完备性

**习题 3.2**. 证明 $L^p$（$1 \leq p < \infty$）完备。

**证明概要**. 设 $\{f_n\}$ 为 Cauchy 列。取子列使 $\|f_{n_{k+1}} - f_{n_k}\|_p < 2^{-k}$。令 $g = \sum_k |f_{n_{k+1}} - f_{n_k}|$，由 MCT 和 Fatou，$g \in L^p$。故 $\sum_k (f_{n_{k+1}} - f_{n_k})$ 绝对收敛 a.e.，$f_{n_k} \to f$ a.e.。再由 Fatou，$f \in L^p$ 且 $f_n \to f$ 于 $L^p$。$\square$

## 4. 微分与不定积分

### 4.1 绝对连续性

**习题 4.1**. 证明 $f$ 绝对连续 $\Leftrightarrow$ $f$ 是某可积函数的不定积分。

**证明**.

$(\Rightarrow)$ $f$ 绝对连续 $\Rightarrow$ 有界变差 $\Rightarrow$ $f'$ 存在 a.e. 且可积。对任意 $[a,b]$：

$$f(b) - f(a) = \int_a^b f'(x) dx$$

（由 Lebesgue 微分定理与 Vitali 覆盖引理）。

$(\Leftarrow)$ 设 $f(x) = f(a) + \int_a^x g(t)dt$，$g \in L^1$。由积分的绝对连续性，$m(E) < \delta \Rightarrow \int_E |g| < \varepsilon$。对不交区间 $\{(a_k, b_k)\}$ 满足 $\sum (b_k - a_k) < \delta$：

$$\sum |f(b_k) - f(a_k)| = \sum \left|\int_{a_k}^{b_k} g\right| \leq \int_{\bigcup (a_k, b_k)} |g| < \varepsilon. \quad \square$$

### 4.2 Riemann-Lebesgue 引理

**习题 4.2**. 证明 $f \in L^1(\mathbb{R}) \Rightarrow \hat{f}(\xi) \to 0$（$|\xi| \to \infty$）。

**证明**.

**步骤 1**：对 $f = \mathbf{1}_{[a,b]}$，$\hat{f}(\xi) = \frac{e^{-i\xi b} - e^{-i\xi a}}{-i\xi} \to 0$。

**步骤 2**：对阶梯函数，由线性性成立。

**步骤 3**：对一般 $f \in L^1$，取阶梯函数 $g$ 使 $\|f - g\|_1 < \varepsilon$。则：

$$|\hat{f}(\xi)| \leq |\widehat{(f-g)}(\xi)| + |\hat{g}(\xi)| \leq \|f-g\|_1 + |\hat{g}(\xi)| < \varepsilon + |\hat{g}(\xi)|.$$

对足够大的 $\xi$，$|\hat{g}(\xi)| < \varepsilon$。$\square$

## 5. 例子

### 5.1 例子：Fatou 严格不等式

已见 $f_n = \mathbf{1}_{[n,n+1]}$，$\int \liminf f_n = 0 < 1 = \liminf \int f_n$。

### 5.2 例子：Dirichlet 函数的积分

$f(x) = \mathbf{1}_{\mathbb{Q} \cap [0,1]}(x)$ 在 Riemann 意义下不可积，但 $f = 0$ a.e.，故 $\int f = 0$（Lebesgue 积分）。
