# 复分析习题精解

## 1. 复变函数基础

### 习题1：Cauchy-Riemann方程

**题目**：设 $f(z) = u(x,y) + iv(x,y)$，证明 $f$ 可微 ⟺ $u, v$ 满足Cauchy-Riemann方程：
$$u_x = v_y, \quad u_y = -v_x$$

**解答**：

**必要性**：设 $f$ 在 $z_0$ 可微，导数为 $f'(z_0)$

$$f'(z_0) = \lim_{h \to 0} \frac{f(z_0 + h) - f(z_0)}{h}$$

**沿实轴**：$h = \Delta x$
$$f'(z_0) = \lim_{\Delta x \to 0} \frac{u(x+\Delta x, y) - u(x,y)}{\Delta x} + i \frac{v(x+\Delta x, y) - v(x,y)}{\Delta x} = u_x + iv_x$$

**沿虚轴**：$h = i\Delta y$
$$f'(z_0) = \lim_{\Delta y \to 0} \frac{u(x, y+\Delta y) - u(x,y)}{i\Delta y} + i \frac{v(x, y+\Delta y) - v(x,y)}{i\Delta y} = -iu_y + v_y$$

比较得：$u_x = v_y$，$v_x = -u_y$

**充分性**：设C-R方程成立，验证导数存在

∎

### 习题2：解析函数的幂级数展开

**题目**：求 $f(z) = \frac{1}{1-z}$ 在 $|z| < 1$ 的Taylor展开。

**解答**：

**几何级数**：
$$\frac{1}{1-z} = \sum_{n=0}^\infty z^n, \quad |z| < 1$$

**验证收敛半径**：

由Cauchy-Hadamard，$R = \frac{1}{\limsup |a_n|^{1/n}} = 1$

∎

## 2. Cauchy积分理论

### 习题3：Cauchy积分公式

**题目**：设 $f$ 在简单闭曲线 $C$ 及其内部解析，证明：
$$f^{(n)}(z_0) = \frac{n!}{2\pi i} \oint_C \frac{f(z)}{(z-z_0)^{n+1}} dz$$

**解答**：

**归纳法**：

**基础** ($n=0$)：即Cauchy积分公式本身

**归纳步骤**：假设对 $n$ 成立，证 $n+1$

对参数 $z_0$ 求导（在积分号下）：
$$f^{(n+1)}(z_0) = \frac{d}{dz_0} f^{(n)}(z_0) = \frac{n!}{2\pi i} \oint_C f(z) \frac{d}{dz_0} \frac{1}{(z-z_0)^{n+1}} dz$$

$$= \frac{n!}{2\pi i} \oint_C f(z) \frac{(n+1)}{(z-z_0)^{n+2}} dz = \frac{(n+1)!}{2\pi i} \oint_C \frac{f(z)}{(z-z_0)^{n+2}} dz$$

∎

### 习题4：留数计算

**题目**：计算 $\oint_{|z|=2} \frac{e^z}{z^2 + 1} dz$

**解答**：

**奇点分析**：分母 $z^2 + 1 = 0$ ⟹ $z = \pm i$

均在圆 $|z| = 2$ 内。

**留数计算**：

在 $z = i$（一阶极点）：
$$\text{Res}_{z=i} \frac{e^z}{z^2+1} = \lim_{z \to i} (z-i) \frac{e^z}{(z-i)(z+i)} = \frac{e^i}{2i}$$

在 $z = -i$：
$$\text{Res}_{z=-i} = \frac{e^{-i}}{-2i} = -\frac{e^{-i}}{2i}$$

**留数定理**：
$$\oint = 2\pi i \left(\frac{e^i}{2i} - \frac{e^{-i}}{2i}\right) = 2\pi i \cdot \frac{\sin 1}{i} = 2\pi \sin 1$$

∎

## 3. 级数展开

### 习题5：Laurent级数

**题目**：求 $f(z) = \frac{1}{z(z-1)}$ 在 $0 < |z| < 1$ 的Laurent展开。

**解答**：

**部分分式**：
$$f(z) = \frac{1}{z-1} - \frac{1}{z}$$

**展开**：对 $0 < |z| < 1$
$$\frac{1}{z-1} = -\frac{1}{1-z} = -\sum_{n=0}^\infty z^n$$

**Laurent级数**：
$$f(z) = -\frac{1}{z} - \sum_{n=0}^\infty z^n = -\sum_{n=-1}^\infty z^n$$

主部：$-\frac{1}{z}$，解析部：$-\sum_{n=0}^\infty z^n$

∎

## 4. 共形映射

### 习题6：Möbius变换

**题目**：求将上半平面映到单位圆盘的Möbius变换。

**解答**：

**一般形式**：$f(z) = e^{i\theta} \frac{z - z_0}{z - \bar{z}_0}$，其中 $\text{Im}(z_0) > 0$

**验证**：
- 当 $z$ 在实轴上：$|f(z)| = \left|\frac{z-z_0}{z-\bar{z}_0}\right| = 1$（实轴映到单位圆）
- 当 $z = z_0$：$f(z_0) = 0$（$z_0$ 映到圆心）

**标准形式**（取 $z_0 = i$，$\theta = 0$）：
$$f(z) = \frac{z-i}{z+i}$$

∎

## 5. 最大模原理

### 习题7：最大模原理应用

**题目**：设 $f$ 在有界区域 $D$ 内解析，在 $\bar{D}$ 连续。证明 $|f|$ 的最大值在边界 $\partial D$ 上达到。

**解答**：

**反证法**：

假设 $|f|$ 在内点 $z_0 \in D$ 达到最大值 $M$。

由最大模原理，若 $f$ 非常数，则 $|f|$ 不能在内部达到最大值。

故若 $|f(z_0)| = M$ 最大，则 $f$ 必为常数。

**结论**：非常数解析函数的模最大值必在边界达到。∎
