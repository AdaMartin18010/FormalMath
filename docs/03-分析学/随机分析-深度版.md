# 随机分析 - 深度版

## 1. 引言

### 1.1 什么是随机分析？

**随机分析**（Stochastic Analysis）是概率论与现代分析学的交叉领域，研究随机过程（特别是布朗运动）的微积分理论。其核心是**伊藤清**（Kiyoshi Itô）于1940年代创立的**随机积分**和**随机微分方程**理论。

**应用范围**: 金融数学、物理学、生物学、控制论、滤波理论等。

### 1.2 历史发展

| 时期 | 数学家 | 贡献 |
|------|--------|------|
| 1827 | 布朗 | 发现花粉的随机运动 |
| 1900 | 巴舍利耶 | 将布朗运动应用于股票价格建模 |
| 1905 | 爱因斯坦 | 从物理角度解释布朗运动 |
| 1923 | 维纳 | 严格定义布朗运动的数学模型 |
| 1944 | 伊藤清 | 创立随机积分理论 |
| 1949 | 费曼-卡茨 | 建立随机过程与偏微分方程的联系 |
| 1976 | 马利亚文 | 创立随机变分学（Malliavin Calculus） |

## 2. 随机积分

### 2.1 布朗运动

**定义 2.1** (标准布朗运动):
随机过程 $\{W_t\}_{t \geq 0}$ 称为**标准布朗运动**，如果：

1. $W_0 = 0$ a.s.
2. 独立增量：对 $0 \leq s < t$，$W_t - W_s$ 与 $\{W_u : u \leq s\}$ 独立
3. 平稳增量：$W_t - W_s \sim N(0, t-s)$
4. 轨道连续：$t \mapsto W_t$ 几乎必然连续

**性质**:
- $E[W_t] = 0$，$\text{Var}(W_t) = t$
- 自相似性：$W_{ct} \stackrel{d}{=} \sqrt{c} W_t$
- 二次变差：$\langle W \rangle_t = t$（即 $\sum (W_{t_i} - W_{t_{i-1}})^2 \to t$）

### 2.2 伊藤积分

**定义 2.2** (伊藤积分):
设 $f(t, \omega)$ 是**适应的**、**平方可积**的随机过程，其伊藤积分定义为：

$$\int_0^T f(t, \omega) \, dW_t = \lim_{\Delta t \to 0} \sum_{i} f(t_i, \omega) (W_{t_{i+1}} - W_{t_i})$$

其中极限在 $L^2$ 意义下。

**关键特征**: 被积函数在**左端点**取值，使积分具有**鞅性**。

**定理 2.1** (伊藤等距):
$$E\left[\left(\int_0^T f_t \, dW_t\right)^2\right] = E\left[\int_0^T f_t^2 \, dt\right]$$

### 2.3 伊藤公式

**定理 2.2** (伊藤公式/链式法则):
设 $X_t$ 是伊藤过程：$dX_t = \mu_t dt + \sigma_t dW_t$

对 $C^2$ 函数 $f(t, x)$：

$$df(t, X_t) = \frac{\partial f}{\partial t} dt + \frac{\partial f}{\partial x} dX_t + \frac{1}{2} \frac{\partial^2 f}{\partial x^2} (dX_t)^2$$

其中 $(dX_t)^2 = \sigma_t^2 dt$（伊藤乘积法则：$dt \cdot dt = 0$，$dt \cdot dW_t = 0$，$dW_t \cdot dW_t = dt$）

**示例**: 
计算 $d(W_t^2)$：
$$d(W_t^2) = 2W_t dW_t + dt$$
即 $W_t^2 = 2\int_0^t W_s dW_s + t$

## 3. 随机微分方程

### 3.1 SDE的定义

**定义 3.1** (随机微分方程):
**随机微分方程**（SDE）的形式为：

$$dX_t = b(t, X_t) dt + \sigma(t, X_t) dW_t$$

或积分形式：

$$X_t = X_0 + \int_0^t b(s, X_s) ds + \int_0^s \sigma(s, X_s) dW_s$$

其中 $b$ 称为**漂移系数**，$\sigma$ 称为**扩散系数**。

### 3.2 解的存在唯一性

**定理 3.1** (存在唯一性):
若 $b$ 和 $\sigma$ 满足**全局Lipschitz条件**：
$$|b(t,x) - b(t,y)| + |\sigma(t,x) - \sigma(t,y)| \leq K|x-y|$$

且**线性增长条件**：
$$|b(t,x)| + |\sigma(t,x)| \leq K(1+|x|)$$

则SDE存在唯一的强解。

### 3.3 示例

**几何布朗运动** (Black-Scholes模型):
$$dS_t = \mu S_t dt + \sigma S_t dW_t$$

**解**: $S_t = S_0 \exp\left((\mu - \frac{\sigma^2}{2})t + \sigma W_t\right)$

**Ornstein-Uhlenbeck过程**:
$$dX_t = -\alpha X_t dt + \sigma dW_t$$

**解**: $X_t = X_0 e^{-\alpha t} + \sigma \int_0^t e^{-\alpha(t-s)} dW_s$

## 4. 费曼-卡茨公式

### 4.1 与偏微分方程的联系

**定理 4.1** (费曼-卡茨公式):
设 $u(t,x)$ 满足偏微分方程：
$$\frac{\partial u}{\partial t} + \frac{1}{2}\sigma^2 \frac{\partial^2 u}{\partial x^2} + b \frac{\partial u}{\partial x} - Vu = 0$$

终值条件 $u(T,x) = f(x)$，则：

$$u(t,x) = E\left[\exp\left(-\int_t^T V(s, X_s) ds\right) f(X_T) \bigg| X_t = x\right]$$

其中 $X_t$ 是漂移为 $b$、扩散为 $\sigma$ 的扩散过程。

**意义**: 将抛物型PDE的解表示为随机表示，是数值计算的重要工具。

## 5. Girsanov定理

### 5.1 测度变换

**定理 5.1** (Girsanov定理):
设 $W_t$ 是概率空间 $(\Omega, \mathcal{F}, P)$ 上的布朗运动，$\theta_t$ 满足Novikov条件：
$$E\left[\exp\left(\frac{1}{2}\int_0^T \theta_s^2 ds\right)\right] < \infty$$

定义新测度 $Q$：
$$\frac{dQ}{dP}\bigg|_{\mathcal{F}_T} = \exp\left(-\int_0^T \theta_s dW_s - \frac{1}{2}\int_0^T \theta_s^2 ds\right)$$

则 $\tilde{W}_t = W_t + \int_0^t \theta_s ds$ 在 $Q$ 下是标准布朗运动。

**应用**: 金融数学中的风险中性测度变换。

## 6. 鞅表示定理

### 6.1 表示定理

**定理 6.1** (鞅表示定理):
设 $M_t$ 是关于由布朗运动生成的滤链 $\{\mathcal{F}_t^W\}$ 的平方可积鞅，则存在唯一的适应过程 $\phi_t$ 使得：

$$M_t = M_0 + \int_0^t \phi_s dW_s$$

## 7. 应用

### 7.1 Black-Scholes期权定价公式

**定理 7.1** (Black-Scholes, 1973):
欧式看涨期权的价格为：

$$C = S_0 N(d_1) - K e^{-rT} N(d_2)$$

其中：
$$d_{1,2} = \frac{\ln(S_0/K) + (r \pm \sigma^2/2)T}{\sigma\sqrt{T}}$$

### 7.2 滤波理论

**卡尔曼-布西滤波**: 从噪声观测中估计随机过程的状态。
