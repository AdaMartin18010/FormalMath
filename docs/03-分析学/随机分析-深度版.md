# 随机分析 - 深度版

> **"伊藤清的随机积分是20世纪概率论最深刻的成就，它将微积分推广到了随机世界。"**

---

## 1. 布朗运动

### 1.1 定义与存在性

**定义 1.1**（标准布朗运动）
随机过程 $\{W_t\}_{t \geq 0}$ 称为**标准布朗运动**，如果：

1. $W_0 = 0$ a.s.
2. **独立增量**：对 $0 \leq s < t$，$W_t - W_s$ 与 $\{W_u : u \leq s\}$ 独立
3. **平稳增量**：$W_t - W_s \sim N(0, t-s)$
4. **轨道连续**：$t \mapsto W_t$ 几乎必然连续

**定理 1.1**（Wiener，1923）
标准布朗运动存在。

**证明概要**：
在 $C([0,1])$ 上构造Wiener测度：

**步骤1**：对二进有理点 $t_k = k/2^n$，定义 $W_{t_k}$ 通过正态增量：
$$W_{t_{k+1}} - W_{t_k} \sim N(0, 1/2^n)$$

**步骤2**：对一般 $t \in [0,1]$，用二进展开插值。由Kolmogorov连续性准则（$\beta$阶矩控制：$E|W_t - W_s|^4 = 3(t-s)^2$），过程有Hölder连续修正。 $\square$

### 1.2 基本性质

**性质**：
- $E[W_t] = 0$，$\text{Var}(W_t) = t$
- **自相似性**：$W_{ct} \stackrel{d}{=} \sqrt{c}W_t$（对 $c > 0$）
- **二次变差**：$\langle W \rangle_t = t$，即
$$\lim_{\|\Pi\| \to 0} \sum_{i=1}^n (W_{t_i} - W_{t_{i-1}})^2 = t \quad \text{(在 } L^2 \text{中)}$$

**定理 1.2**
布朗运动的轨道几乎必然无处可微。

**证明概要**：
若 $W_t$ 在 $t_0$ 可微，则存在 $C > 0$ 使 $|W_t - W_{t_0}| \leq C|t - t_0|$ 对 $|t - t_0|$ 小成立。

但对二进区间，由增量独立性和正态分布尾部衰减，可证这种局部Lipschitz性概率为0。 $\square$

## 2. 伊藤积分

### 2.1 定义

**定义 2.1**（伊藤积分）
设 $f(t,\omega)$ 是**适应的**、满足 $E\int_0^T f_t^2\,dt < \infty$ 的随机过程。其**伊藤积分**定义为：
$$\int_0^T f_t\,dW_t = \lim_{\Delta t \to 0} \sum_i f_{t_i}(W_{t_{i+1}} - W_{t_i})$$

其中极限在 $L^2$ 意义下。

**关键特征**：被积函数在**左端点**取值，使积分具有**鞅性**。

**注**：若用Stratonovich积分（中点取值），公式更简单但失去鞅性：
$$\int_0^T f_t \circ dW_t = \lim \sum_i f_{(t_i+t_{i+1})/2}(W_{t_{i+1}} - W_{t_i})$$

### 2.2 伊藤等距

**定理 2.1**（伊藤等距）
$$E\left[\left(\int_0^T f_t\,dW_t\right)^2\right] = E\left[\int_0^T f_t^2\,dt\right]$$

**证明**：
对简单过程 $f_t = \sum_i \xi_i \mathbf{1}_{[t_i, t_{i+1})}(t)$，其中 $\xi_i$ 是 $\mathcal{F}_{t_i}$-可测的：

$$E\left[\left(\sum_i \xi_i \Delta W_i\right)^2\right] = E\left[\sum_i \xi_i^2 (\Delta W_i)^2 + 2\sum_{i<j} \xi_i\xi_j \Delta W_i \Delta W_j\right]$$

对 $i < j$，$\xi_i\xi_j\Delta W_i$ 是 $\mathcal{F}_{t_j}$-可测的，与 $\Delta W_j$ 独立，故交叉项期望为0。

而 $E[\xi_i^2(\Delta W_i)^2] = E[\xi_i^2]E[(\Delta W_i)^2] = E[\xi_i^2]\Delta t_i$。

因此：
$$E\left[\left(\int f_t\,dW_t\right)^2\right] = \sum_i E[\xi_i^2]\Delta t_i = E\left[\int f_t^2\,dt\right]$$

一般情形通过简单过程逼近。 $\square$

## 3. 伊藤公式

### 3.1 一维伊藤公式

**定理 3.1**（伊藤公式/链式法则）
设 $X_t$ 是**伊藤过程**：
$$dX_t = \mu_t\,dt + \sigma_t\,dW_t$$

对 $f \in C^{2,1}([0,T] \times \mathbb{R})$，有：
$$df(t, X_t) = \frac{\partial f}{\partial t}dt + \frac{\partial f}{\partial x}dX_t + \frac{1}{2}\frac{\partial^2 f}{\partial x^2}(dX_t)^2$$

其中伊藤乘积法则：$dt \cdot dt = 0$，$dt \cdot dW_t = 0$，$dW_t \cdot dW_t = dt$。

即：
$$df = \left(\frac{\partial f}{\partial t} + \mu_t\frac{\partial f}{\partial x} + \frac{\sigma_t^2}{2}\frac{\partial^2 f}{\partial x^2}\right)dt + \sigma_t\frac{\partial f}{\partial x}dW_t$$

**证明概要**：
对 $[0,T]$ 的分割 $0 = t_0 < t_1 < \cdots < t_n = T$，由Taylor展开：

$$f(t_{i+1}, X_{t_{i+1}}) - f(t_i, X_{t_i}) = f_t\Delta t + f_x\Delta X + \frac{1}{2}f_{xx}(\Delta X)^2 + R$$

其中 $R$ 为高阶项。关键是 $(\Delta X)^2 \approx \sigma^2 \Delta t$（由二次变差），而 $O((\Delta t)^{3/2})$ 项趋于0。 $\square$

### 3.2 例子

**例 1**：计算 $d(W_t^2)$

取 $f(x) = x^2$，$X_t = W_t$（$\mu = 0$，$\sigma = 1$）：
$$d(W_t^2) = 0 \cdot dt + 2W_t\,dW_t + \frac{1}{2} \cdot 2 \cdot dt = 2W_t\,dW_t + dt$$

即：
$$W_t^2 = 2\int_0^t W_s\,dW_s + t$$

因此：
$$\int_0^t W_s\,dW_s = \frac{W_t^2 - t}{2}$$

这与普通微积分 $\int x\,dx = x^2/2$ 相比多出了 $-t/2$ 项，体现了伊藤修正。 $\square$

**例 2**：几何布朗运动

设 $S_t$ 满足 $dS_t = \mu S_t\,dt + \sigma S_t\,dW_t$。

取 $f(x) = \log x$，则：
$$d(\log S_t) = \frac{1}{S_t}dS_t - \frac{1}{2S_t^2}(dS_t)^2 = (\mu - \frac{\sigma^2}{2})dt + \sigma\,dW_t$$

积分：
$$\log S_t = \log S_0 + (\mu - \frac{\sigma^2}{2})t + \sigma W_t$$

故：
$$S_t = S_0 \exp\left((\mu - \frac{\sigma^2}{2})t + \sigma W_t\right)$$ $\square$

## 4. 随机微分方程

### 4.1 存在唯一性

**定理 4.1**（SDE的存在唯一性）
考虑SDE：
$$dX_t = b(t, X_t)\,dt + \sigma(t, X_t)\,dW_t, \quad X_0 = x_0$$

若 $b$ 和 $\sigma$ 满足：
1. **全局Lipschitz条件**：$|b(t,x) - b(t,y)| + |\sigma(t,x) - \sigma(t,y)| \leq K|x-y|$
2. **线性增长条件**：$|b(t,x)| + |\sigma(t,x)| \leq K(1+|x|)$

则存在唯一的强解（适应于 $W_t$ 生成的滤链）。

**证明概要**：
用Picard迭代：$X^{(0)}_t = x_0$，
$$X^{(n+1)}_t = x_0 + \int_0^t b(s, X^{(n)}_s)\,ds + \int_0^t \sigma(s, X^{(n)}_s)\,dW_s$$

由Lipschitz条件和伊藤等距，证明这是 $L^2$ 中的Cauchy列。 $\square$

### 4.2 例子

**例 1**：Ornstein-Uhlenbeck过程
$$dX_t = -\alpha X_t\,dt + \sigma\,dW_t$$

**解**：用积分因子 $e^{\alpha t}$：
$$d(e^{\alpha t}X_t) = \alpha e^{\alpha t}X_t\,dt + e^{\alpha t}dX_t = \sigma e^{\alpha t}\,dW_t$$

故：
$$X_t = X_0 e^{-\alpha t} + \sigma\int_0^t e^{-\alpha(t-s)}\,dW_s$$

$X_t$ 是高斯过程，$E[X_t] = X_0 e^{-\alpha t}$，平稳分布为 $N(0, \sigma^2/2\alpha)$。 $\square$

## 5. Girsanov定理

**定理 5.1**（Girsanov定理）
设 $W_t$ 是概率空间 $(\Omega, \mathcal{F}, P)$ 上的布朗运动，$\theta_t$ 满足Novikov条件：
$$E\left[\exp\left(\frac{1}{2}\int_0^T \theta_s^2\,ds\right)\right] < \infty$$

定义**指数鞅**：
$$Z_t = \exp\left(-\int_0^t \theta_s\,dW_s - \frac{1}{2}\int_0^t \theta_s^2\,ds\right)$$

和新测度 $Q$：$\frac{dQ}{dP}\big|_{\mathcal{F}_T} = Z_T$。

则 $\tilde{W}_t = W_t + \int_0^t \theta_s\,ds$ 在 $Q$ 下是标准布朗运动。

**应用**：金融数学中的**风险中性测度变换**——将带漂移的股价过程转换为鞅。 $\square$

## 6. Black-Scholes期权定价

**定理 6.1**（Black-Scholes-Merton，1973）
设股价服从几何布朗运动：$dS_t = \mu S_t\,dt + \sigma S_t\,dW_t$。

欧式看涨期权（执行价 $K$，到期 $T$）的价格为：
$$C = S_0 N(d_1) - Ke^{-rT}N(d_2)$$

其中：
$$d_{1,2} = \frac{\ln(S_0/K) + (r \pm \sigma^2/2)T}{\sigma\sqrt{T}}$$

$N(\cdot)$ 是标准正态CDF，$r$ 是无风险利率。

**推导概要**：

**步骤1**：在风险中性测度 $Q$ 下，$dS_t = rS_t\,dt + \sigma S_t\,d\tilde{W}_t$。

**步骤2**：期权价格 = 折现期望收益：
$$C = e^{-rT}E^Q[(S_T - K)^+]$$

**步骤3**：在 $Q$ 下，$S_T = S_0\exp((r-\sigma^2/2)T + \sigma\tilde{W}_T)$，$\tilde{W}_T \sim N(0,T)$。

**步骤4**：计算正态分布的截断期望，得到 $N(d_1)$ 和 $N(d_2)$。 $\square$

## 7. 鞅表示定理

**定理 7.1**（鞅表示定理）
设 $M_t$ 是关于由布朗运动生成的滤链 $\{\mathcal{F}_t^W\}$ 的平方可积鞅，则存在唯一的适应过程 $\phi_t$ 使得：
$$M_t = M_0 + \int_0^t \phi_s\,dW_s$$

**意义**：任何鞅都可表示为伊藤积分，这是完备市场的数学基础。
