## 1. 引言与历史

### 1.1 什么是分析学？

**分析学**（Analysis）是数学的核心分支，研究极限、连续性、微分、积分以及无穷级数等概念。分析学源于17世纪牛顿和莱布尼茨创立的微积分，经过19世纪柯西、魏尔斯特拉斯等人的严格化，发展成为现代数学的基石。

### 1.2 发展历程

| 时期 | 代表人物 | 主要贡献 |
|------|----------|----------|
| 17世纪 | 牛顿、莱布尼茨 | 创立微积分 |
| 18世纪 | 欧拉、拉格朗日 | 分析方法的发展 |
| 19世纪 | 柯西、魏尔斯特拉斯 | 分析的严格化 |
| 20世纪 | 勒贝格、巴拿赫 | 测度论与泛函分析 |

## 2. 核心概念

### 2.1 极限 (Limit)

**定义 2.1** ($\varepsilon$-$\delta$ 定义): 
设函数 $f$ 在点 $a$ 的某去心邻域内有定义。若存在实数 $L$，使得对于任意 $\varepsilon > 0$，存在 $\delta > 0$，当 $0 < |x - a| < \delta$ 时，有 $|f(x) - L| < \varepsilon$，则称 $f(x)$ 当 $x \to a$ 时的极限为 $L$，记作：

$$\lim_{x \to a} f(x) = L$$

**性质**:
1. **唯一性**: 若极限存在，则唯一
2. **保号性**: 若 $\lim_{x \to a} f(x) > 0$，则在 $a$ 的某邻域内 $f(x) > 0$
3. **夹逼定理**: 若 $g(x) \leq f(x) \leq h(x)$ 且 $\lim g = \lim h = L$，则 $\lim f = L$

### 2.2 连续性 (Continuity)

**定义 2.2**: 函数 $f$ 在点 $a$ 连续，当且仅当：
$$\lim_{x \to a} f(x) = f(a)$$

**定理 2.1** (介值定理): 
设 $f$ 在 $[a, b]$ 上连续，且 $f(a) \neq f(b)$，则对于 $f(a)$ 与 $f(b)$ 之间的任意值 $c$，存在 $\xi \in (a, b)$ 使得 $f(\xi) = c$。

**证明**: 
考虑集合 $S = \{x \in [a, b] : f(x) < c\}$，令 $\xi = \sup S$。
由连续性，$f(\xi) = c$。 $\square$

### 2.3 微分 (Differentiation)

**定义 2.3**: 函数 $f$ 在点 $a$ 可微，若极限存在：
$$f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h}$$

**基本求导法则**:

| 法则 | 公式 |
|------|------|
| 线性性 | $(af + bg)' = af' + bg'$ |
| 乘积法则 | $(fg)' = f'g + fg'$ |
| 商法则 | $\left(\frac{f}{g}\right)' = \frac{f'g - fg'}{g^2}$ |
| 链式法则 | $(f \circ g)' = (f' \circ g) \cdot g'$ |

**定理 2.2** (中值定理):
若 $f$ 在 $[a, b]$ 上连续，在 $(a, b)$ 内可微，则存在 $\xi \in (a, b)$ 使得：
$$f'(\xi) = \frac{f(b) - f(a)}{b - a}$$

### 2.4 积分 (Integration)

**定义 2.4** (黎曼积分):
设 $f$ 在 $[a, b]$ 上有界，分割 $P: a = x_0 < x_1 < \cdots < x_n = b$，取 $\xi_i \in [x_{i-1}, x_i]$，黎曼和为：
$$S(f, P) = \sum_{i=1}^{n} f(\xi_i) \Delta x_i$$
若 $\lim_{\|P\| \to 0} S(f, P)$ 存在且与分割和取点无关，则称 $f$ 黎曼可积。

**定理 2.3** (微积分基本定理):
设 $f$ 在 $[a, b]$ 上连续，定义 $F(x) = \int_a^x f(t) \, dt$，则：
1. $F$ 在 $[a, b]$ 上可微，且 $F'(x) = f(x)$
2. 若 $G$ 是 $f$ 的任一原函数，则 $\int_a^b f(x) \, dx = G(b) - G(a)$

## 3. 分析学的主要分支

### 3.1 实分析 (Real Analysis)

**研究对象**: 实数、实函数、测度与积分

**核心内容**:
- 实数系的构造（戴德金分割、柯西序列）
- 测度论与勒贝格积分
- $L^p$ 空间理论

### 3.2 复分析 (Complex Analysis)

**研究对象**: 复变函数的全纯性质

**核心定理**:

**定理 3.1** (柯西积分公式):
设 $f$ 在区域 $D$ 内全纯，$\gamma$ 是 $D$ 内可求长简单闭曲线，$z$ 在 $\gamma$ 内部，则：
$$f(z) = \frac{1}{2\pi i} \oint_{\gamma} \frac{f(\zeta)}{\zeta - z} \, d\zeta$$

**定理 3.2** (留数定理):
$$\oint_{\gamma} f(z) \, dz = 2\pi i \sum_{k=1}^{n} \text{Res}(f, z_k)$$

### 3.3 泛函分析 (Functional Analysis)

**研究对象**: 无穷维空间上的函数（泛函）与算子

**核心概念**:
- **巴拿赫空间**: 完备的赋范线性空间
- **希尔伯特空间**: 带有内积的完备空间
- **线性算子**: 空间之间的线性映射

**定理 3.3** (哈恩-巴拿赫定理):
设 $X$ 是赋范线性空间，$Y \subseteq X$ 是子空间，$f$ 是 $Y$ 上的有界线性泛函，则 $f$ 可保范延拓到整个 $X$ 上。

## 4. 重要不等式

| 不等式 | 表述 | 适用条件 |
|--------|------|----------|
| 柯西不等式 | $|\langle x, y \rangle| \leq \|x\| \|y\|$ | 内积空间 |
| 三角不等式 | $\|x + y\| \leq \|x\| + \|y\|$ | 赋范空间 |
| 霍尔德不等式 | $\int |fg| \leq \|f\|_p \|g\|_q$ | $\frac{1}{p} + \frac{1}{q} = 1$ |
| 闵可夫斯基不等式 | $\|f + g\|_p \leq \|f\|_p + \|g\|_p$ | $p \geq 1$ |
| 詹森不等式 | $f(E[X]) \leq E[f(X)]$ | $f$ 凸函数 |
