## 📚 概述

函数序列的收敛性是分析学中的核心问题。点态收敛虽然直观，但不足以保证极限运算的交换性。**一致收敛**的引入解决了这一关键问题，它要求收敛速度在整个定义域上"一致"，从而确保了连续性、可积性和可微性的保持。

本文档系统阐述一致收敛的理论，包括判别法、极限运算交换定理，以及函数项级数的分析性质。

## 🏛️ 历史发展

### 柯西的错误与修正

1821年，柯西在他的《分析教程》中给出了一个著名但错误的断言：

> **柯西的错误定理**：收敛的连续函数序列的极限函数也是连续的。

这个错误在1850年代被阿贝尔和后来的塞德尔指出。问题出在柯西对"收敛"的理解——他只考虑了**点态收敛**，而点态收敛不足以保持连续性。

**反例（Abel, 1826）**：
$$f_n(x) = \sum_{k=0}^{n} (-1)^k \frac{x^k}{k!}$$

在 $x = 1$ 处，虽然每一项都连续，但极限函数在 $x = 1$ 处不连续。

更经典的反例：
$$f_n(x) = x^n, \quad x \in [0, 1]$$

每个 $f_n$ 都连续，但极限函数：
$$f(x) = \begin{cases} 0 & 0 \leq x < 1 \\ 1 & x = 1 \end{cases}$$

在 $x = 1$ 处不连续。

### 魏尔斯特拉斯的一致收敛概念

**卡尔·魏尔斯特拉斯**（Karl Weierstrass）在1840年代提出了**一致收敛**（Gleichmäßige Konvergenz）的概念，彻底解决了柯西的错误。

魏尔斯特拉斯的洞察：
- 点态收敛允许收敛速度依赖于点
- 一致收敛要求收敛速度在整个区间上一致
- 一致收敛保持了连续性、可积性和可微性

> "一致收敛是函数序列分析性质的守护者。" —— 魏尔斯特拉斯

### 与黎曼积分理论的联系

**伯恩哈德·黎曼**在1854年的博士论文中研究了函数的可积性。黎曼积分理论的一个关键问题是：**何时可以交换积分与极限**？

黎曼的洞见：
- 一致收敛保证了积分号下取极限的合法性
- 这一性质是黎曼积分理论的核心之一
- 黎曼还发现了比一致收敛更弱的条件（控制收敛思想的萌芽）

## 📐 点态收敛与一致收敛

### 点态收敛的定义

**定义 1.1** (点态收敛)

函数序列 $(f_n)$ 在集合 $E$ 上**点态收敛**于 $f$，记作 $f_n \xrightarrow{p.w.} f$，如果：

$$\forall x \in E, \forall \varepsilon > 0, \exists N = N(x, \varepsilon), \forall n \geq N: |f_n(x) - f(x)| < \varepsilon$$

**关键特征**：$N$ 依赖于点 $x$ 和 $\varepsilon$。

### 一致收敛的定义

**定义 1.2** (一致收敛)

函数序列 $(f_n)$ 在集合 $E$ 上**一致收敛**于 $f$，记作 $f_n \rightrightarrows f$ 或 $f_n \xrightarrow{u} f$，如果：

$$\forall \varepsilon > 0, \exists N = N(\varepsilon), \forall n \geq N, \forall x \in E: |f_n(x) - f(x)| < \varepsilon$$

**关键特征**：$N$ 仅依赖于 $\varepsilon$，对所有 $x \in E$ 通用。

**等价表述**：
$$\lim_{n \to \infty} \sup_{x \in E} |f_n(x) - f(x)| = 0$$

### 一致范数与距离

**定义 1.3** (上确界范数/一致范数)

$$\|f\|_{\infty} = \sup_{x \in E} |f(x)|$$

**定理 1.1** (一致收敛的范数刻画)

$$f_n \rightrightarrows f \Leftrightarrow \|f_n - f\|_{\infty} \to 0$$

**证明**：

($\Rightarrow$) 设 $f_n \rightrightarrows f$。给定 $\varepsilon > 0$，$\exists N: \forall n \geq N, \forall x, |f_n(x) - f(x)| < \varepsilon$。

故 $\sup_x |f_n(x) - f(x)| \leq \varepsilon$，即 $\|f_n - f\|_{\infty} \to 0$。

($\Leftarrow$) 设 $\|f_n - f\|_{\infty} \to 0$。给定 $\varepsilon > 0$，$\exists N: \forall n \geq N, \sup_x |f_n(x) - f(x)| < \varepsilon$。

故 $\forall x, |f_n(x) - f(x)| < \varepsilon$，即 $f_n \rightrightarrows f$。

## 🔍 一致收敛判别法

### Weierstrass M-判别法

**定理 2.1** (Weierstrass M-判别法)

设函数项级数 $\sum_{n=1}^{\infty} u_n(x)$ 满足：
1. $|u_n(x)| \leq M_n$ 对所有 $x \in E$
2. $\sum_{n=1}^{\infty} M_n$ 收敛

则 $\sum_{n=1}^{\infty} u_n(x)$ 在 $E$ 上一致收敛。

**证明**：

由 $\sum M_n$ 收敛，根据Cauchy准则：
$$\forall \varepsilon > 0, \exists N, \forall n > m \geq N: \sum_{k=m+1}^{n} M_k < \varepsilon$$

对任意 $x \in E$：
$$\left|\sum_{k=m+1}^{n} u_k(x)\right| \leq \sum_{k=m+1}^{n} |u_k(x)| \leq \sum_{k=m+1}^{n} M_k < \varepsilon$$

由Cauchy一致收敛准则，级数一致收敛。

**例 2.1**：$\sum_{n=1}^{\infty} \frac{\sin(nx)}{n^2}$ 在 $\mathbb{R}$ 上一致收敛。

因 $|\frac{\sin(nx)}{n^2}| \leq \frac{1}{n^2}$，而 $\sum \frac{1}{n^2}$ 收敛。

### Cauchy一致收敛准则

**定理 2.2** (Cauchy一致收敛准则)

函数序列 $(f_n)$ 在 $E$ 上一致收敛当且仅当：

$$\forall \varepsilon > 0, \exists N, \forall m, n \geq N, \forall x \in E: |f_m(x) - f_n(x)| < \varepsilon$$

**证明**：

($\Rightarrow$) 设 $f_n \rightrightarrows f$。给定 $\varepsilon > 0$，$\exists N: \forall n \geq N, \forall x, |f_n(x) - f(x)| < \varepsilon/2$。

对 $m, n \geq N$：
$$|f_m(x) - f_n(x)| \leq |f_m(x) - f(x)| + |f(x) - f_n(x)| < \varepsilon$$

($\Leftarrow$) 固定 $x$，$(f_n(x))$ 是Cauchy数列，故收敛。定义 $f(x) = \lim f_n(x)$。

由条件，$\forall \varepsilon > 0, \exists N: \forall m, n \geq N, \forall x, |f_m(x) - f_n(x)| < \varepsilon$。

令 $m \to \infty$：$|f(x) - f_n(x)| \leq \varepsilon$ 对 $n \geq N$ 成立。

故 $f_n \rightrightarrows f$。

### Abel判别法

**定理 2.3** (Abel一致收敛判别法)

设：
1. $\sum_{n=1}^{\infty} a_n(x)$ 在 $E$ 上一致收敛
2. $(b_n(x))$ 对每个 $x$ 单调
3. $(b_n(x))$ 在 $E$ 上一致有界：$|b_n(x)| \leq M$

则 $\sum_{n=1}^{\infty} a_n(x)b_n(x)$ 在 $E$ 上一致收敛。

**证明概要**：

利用Abel变换（分部求和）：
$$\sum_{k=m+1}^{n} a_k b_k = A_n b_n - A_m b_{m+1} + \sum_{k=m+1}^{n-1} A_k(b_k - b_{k+1})$$

其中 $A_k = \sum_{j=1}^{k} a_j$。由一致收敛和一致有界性，可证余项一致趋于零。

### Dirichlet判别法

**定理 2.4** (Dirichlet一致收敛判别法)

设：
1. $\sum_{n=1}^{\infty} a_n(x)$ 的部分和在 $E$ 上一致有界
2. $(b_n(x))$ 对每个 $x$ 单调趋于0
3. $b_n(x) \rightrightarrows 0$ 在 $E$ 上

则 $\sum_{n=1}^{\infty} a_n(x)b_n(x)$ 在 $E$ 上一致收敛。

## 📊 极限运算的交换

### 连续性定理

**定理 3.1** (一致收敛保持连续性)

设 $f_n \in C(E)$（$E$ 为度量空间中的集合），且 $f_n \rightrightarrows f$ 在 $E$ 上，则 $f \in C(E)$。

**证明**：

设 $x_0 \in E$，证明 $f$ 在 $x_0$ 连续。

给定 $\varepsilon > 0$：
1. 由一致收敛，$\exists N: \forall n \geq N, \forall x, |f_n(x) - f(x)| < \varepsilon/3$
2. $f_N$ 连续，$\exists \delta > 0: d(x, x_0) < \delta \Rightarrow |f_N(x) - f_N(x_0)| < \varepsilon/3$

对 $d(x, x_0) < \delta$：
$$|f(x) - f(x_0)| \leq |f(x) - f_N(x)| + |f_N(x) - f_N(x_0)| + |f_N(x_0) - f(x_0)| < \varepsilon$$

**注**：这是魏尔斯特拉斯对柯西错误的修正。

### 积分号下取极限

**定理 3.2** (一致收敛与积分)

设 $f_n \in C[a,b]$，$f_n \rightrightarrows f$ 在 $[a,b]$ 上，则：

$$\lim_{n \to \infty} \int_a^b f_n(x) dx = \int_a^b f(x) dx = \int_a^b \lim_{n \to \infty} f_n(x) dx$$

**证明**：

$$\left|\int_a^b f_n(x) dx - \int_a^b f(x) dx\right| \leq \int_a^b |f_n(x) - f(x)| dx$$

$$\leq (b-a) \cdot \sup_{x \in [a,b]} |f_n(x) - f(x)| = (b-a) \|f_n - f\|_{\infty} \to 0$$

**推论**：函数项级数可逐项积分：
$$\int_a^b \sum_{n=1}^{\infty} u_n(x) dx = \sum_{n=1}^{\infty} \int_a^b u_n(x) dx$$

### 微分号下取极限

**定理 3.3** (一致收敛与微分)

设：
1. $f_n \in C^1[a,b]$
2. $f_n \to f$ 在某点 $x_0 \in [a,b]$
3. $f_n' \rightrightarrows g$ 在 $[a,b]$ 上

则 $f \in C^1[a,b]$ 且 $f' = g$，即：
$$\left(\lim_{n \to \infty} f_n\right)' = \lim_{n \to \infty} f_n'$$

**证明**：

由微积分基本定理：
$$f_n(x) = f_n(x_0) + \int_{x_0}^x f_n'(t) dt$$

令 $n \to \infty$：
- $f_n(x_0) \to f(x_0)$
- 由积分号下取极限定理，$\int_{x_0}^x f_n'(t) dt \to \int_{x_0}^x g(t) dt$

故 $f(x) = f(x_0) + \int_{x_0}^x g(t) dt$，求导得 $f' = g$。

**注**：微分需要更强的条件——导数序列一致收敛，而非原函数序列一致收敛。

### Dini定理

**定理 3.4** (Dini定理)

设：
1. $K$ 为紧集
2. $f_n \in C(K)$，$f \in C(K)$
3. $f_n \to f$ 点态
4. $(f_n)$ 单调（对每个 $x$，$(f_n(x))$ 单调）

则 $f_n \rightrightarrows f$ 在 $K$ 上。

**证明**（以 $f_n \geq f_{n+1} \geq f$ 为例）：

令 $g_n = f_n - f \geq 0$，$g_n \searrow 0$ 点态。

给定 $\varepsilon > 0$，令 $E_n = \{x \in K : g_n(x) \geq \varepsilon\}$。

- $E_n$ 闭（因 $g_n$ 连续）
- $E_{n+1} \subseteq E_n$（因 $g_{n+1} \leq g_n$）
- $\bigcap E_n = \emptyset$（因 $g_n(x) \to 0$）

由紧集的有限交性质，$\exists N: E_N = \emptyset$。

故 $\forall n \geq N, \forall x: g_n(x) < \varepsilon$，即一致收敛。

## 🔄 函数项级数的性质

### 逐项积分

**定理 4.1** (逐项积分定理)

设 $u_n \in C[a,b]$，$\sum u_n(x)$ 在 $[a,b]$ 上一致收敛于 $S(x)$，则：

$$\int_a^b S(x) dx = \sum_{n=1}^{\infty} \int_a^b u_n(x) dx$$

**证明**：

部分和 $S_N(x) = \sum_{n=1}^{N} u_n(x) \rightrightarrows S(x)$。

由积分号下取极限：
$$\int_a^b S(x) dx = \lim_{N \to \infty} \int_a^b S_N(x) dx = \lim_{N \to \infty} \sum_{n=1}^{N} \int_a^b u_n(x) dx = \sum_{n=1}^{\infty} \int_a^b u_n(x) dx$$

### 逐项求导

**定理 4.2** (逐项求导定理)

设：
1. $u_n \in C^1[a,b]$
2. $\sum u_n(x)$ 在某点 $x_0 \in [a,b]$ 收敛
3. $\sum u_n'(x)$ 在 $[a,b]$ 上一致收敛

则 $\sum u_n(x)$ 在 $[a,b]$ 上一致收敛，且：
$$\left(\sum_{n=1}^{\infty} u_n(x)\right)' = \sum_{n=1}^{\infty} u_n'(x)$$

**证明**：

令 $S_N(x) = \sum_{n=1}^{N} u_n(x)$，$g(x) = \sum_{n=1}^{\infty} u_n'(x)$。

由微分号下取极限定理，应用于 $S_N$：
- $S_N(x_0)$ 收敛
- $S_N'(x) = \sum_{n=1}^{N} u_n'(x) \rightrightarrows g(x)$

故 $S_N \rightrightarrows S$，且 $S' = g$。

### 幂级数的特殊性质

**定理 4.3** (幂级数的一致收敛性)

设幂级数 $\sum_{n=0}^{\infty} a_n (x-x_0)^n$ 的收敛半径为 $R$，则：

1. 在任意 $[x_0-r, x_0+r]$（$r < R$）上一致收敛
2. 和函数 $S(x)$ 在 $(x_0-R, x_0+R)$ 内连续
3. 可逐项积分、逐项求导，且新级数收敛半径仍为 $R$

**证明**：

取 $r < R$，则 $\sum |a_n| r^n$ 收敛。

对 $x \in [x_0-r, x_0+r]$：$|a_n (x-x_0)^n| \leq |a_n| r^n$。

由M-判别法，一致收敛。

def uniform_converges {α β : Type*} [topological_space α] [metric_space β]
  (f : ℕ → α → β) (g : α → β) : Prop :=
  tendsto (λ n, ⨆ x, dist (f n x) (g x)) at_top (𝓝 0)

  (u : ℕ → α → ℝ) (M : ℕ → ℝ)

  (hsum : summable M) :
  ∃ S : α → ℝ, uniform_converges (λ n x, ∑ k in finset.range n, u k x) S :=
begin
  use λ x, ∑' n, u n x,
  intros ε hε,
  rcases hsum.tendsto_cofinite_zero.eventually_lt hε with ⟨N, hN⟩,
  use N,
  intros n hn x,
  have h : |∑ k in finset.range n, u k x - ∑' k, u k x| = |∑' k : {k // k ≥ n}, u k x|,
  { sorry }, -- 需要处理级数余项
  sorry
end

-- 连续性保持定理
theorem uniform_limit_continuous {α β : Type*} [topological_space α] [metric_space β]
  (f : ℕ → α → β) (g : α → β)
  (hf : ∀ n, continuous (f n))
  (hu : uniform_converges f g) :
  continuous g :=
begin
  apply continuous_iff_continuous_at.mpr,
  intro x,
  apply tendsto_iff_dist_tendsto_zero.mpr,
  have : ∀ ε > 0, ∃ N, ∀ n ≥ N, ∀ y, dist (f n y) (g y) < ε / 3,
  { sorry }, -- 利用一致收敛
  sorry
end

-- 积分号下取极限
theorem integral_uniform_limit {a b : ℝ} (f : ℕ → ℝ → ℝ) (g : ℝ → ℝ)
  (hf : ∀ n, continuous_on (f n) (set.Icc a b))
  (hu : uniform_converges f g) :
  tendsto (λ n, ∫ x in a..b, f n x) at_top (𝓝 (∫ x in a..b, g x)) :=
begin
  have hg : continuous_on g (set.Icc a b),
  { apply uniform_limit_continuous f g hf hu },

end

## 🧠 深入分析

### 一致收敛与等度连续性

**定义 5.1** (等度连续性)

函数族 $\mathcal{F} = \{f_\alpha\}$ 在 $E$ 上**等度连续**，如果：
$$\forall \varepsilon > 0, \exists \delta > 0, \forall f \in \mathcal{F}, \forall x, y \in E: |x-y| < \delta \Rightarrow |f(x) - f(y)| < \varepsilon$$

**定理 5.1** 

若连续函数序列 $f_n \rightrightarrows f$ 在紧集 $K$ 上，则 $(f_n)$ 等度连续。

这一概念通向更一般的**Arzelà-Ascoli定理**。

### Arzelà-Ascoli定理预览

**定理 5.2** (Arzelà-Ascoli)

设 $K$ 为紧度量空间，$\mathcal{F} \subseteq C(K)$。则 $\mathcal{F}$ 在一致范数下相对紧当且仅当：
1. $\mathcal{F}$ 一致有界
2. $\mathcal{F}$ 等度连续

这一定理在微分方程、变分法中有重要应用。
