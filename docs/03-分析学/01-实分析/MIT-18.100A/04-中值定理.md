# 中值定理（Mean Value Theorem）

## 一、定理陈述

**定理（拉格朗日中值定理 / Lagrange Mean Value Theorem）**：设函数 $f: [a, b] \to \mathbb{R}$ 满足：

1. $f$ 在闭区间 $[a, b]$ 上**连续**
2. $f$ 在开区间 $(a, b)$ 内**可导**

则存在 $c \in (a, b)$ 使得：

$$f'(c) = \frac{f(b) - f(a)}{b - a}$$

**等价表述**：
$$f(b) - f(a) = f'(c) \cdot (b - a)$$

**几何意义**：曲线 $y = f(x)$ 上存在一点 $c$，使得该点切线的斜率等于连接端点 $(a, f(a))$ 和 $(b, f(b))$ 的割线斜率。

---

## 二、证明思路

中值定理的证明核心在于构造一个**辅助函数**，将问题转化为**罗尔定理**（Rolle's Theorem）的情形：

1. **观察**：若 $f(a) = f(b)$，则中值定理退化为罗尔定理（存在 $c$ 使 $f'(c) = 0$）
2. **构造辅助函数**：将 $f$ 减去连接两端点的直线，得到一个新函数 $g$，满足 $g(a) = g(b) = 0$
3. **应用罗尔定理**：存在 $c \in (a, b)$ 使得 $g'(c) = 0$
4. **回推**：$g'(c) = 0$ 等价于 $f'(c) = \dfrac{f(b) - f(a)}{b - a}$

---

## 三、详细证明

**给定**：$f: [a, b] \to \mathbb{R}$ 在 $[a, b]$ 上连续，在 $(a, b)$ 内可导。

**目标**：证明存在 $c \in (a, b)$ 使得 $f'(c) = \dfrac{f(b) - f(a)}{b - a}$。

---

### 步骤 1：构造辅助函数

设连接 $(a, f(a))$ 和 $(b, f(b))$ 的直线方程为：
$$L(x) = f(a) + \frac{f(b) - f(a)}{b - a}(x - a)$$

定义辅助函数 $g: [a, b] \to \mathbb{R}$ 为：
$$g(x) = f(x) - L(x) = f(x) - f(a) - \frac{f(b) - f(a)}{b - a}(x - a)$$

---

### 步骤 2：验证 $g$ 满足罗尔定理的条件

**连续性**：$f$ 连续，$L$ 是多项式（连续），故 $g = f - L$ 在 $[a, b]$ 上连续。

**可导性**：$f$ 在 $(a, b)$ 内可导，$L$ 处处可导，故 $g$ 在 $(a, b)$ 内可导。

**端点值**：

- $g(a) = f(a) - f(a) - \dfrac{f(b) - f(a)}{b - a}(a - a) = 0$
- $g(b) = f(b) - f(a) - \dfrac{f(b) - f(a)}{b - a}(b - a) = f(b) - f(a) - (f(b) - f(a)) = 0$

因此 $g(a) = g(b) = 0$。

---

### 步骤 3：应用罗尔定理

由**罗尔定理**，存在 $c \in (a, b)$ 使得：
$$g'(c) = 0$$

---

### 步骤 4：计算 $g'(c)$ 并得出结论

对 $g(x)$ 求导：
$$g'(x) = f'(x) - \frac{f(b) - f(a)}{b - a}$$

令 $x = c$：
$$g'(c) = f'(c) - \frac{f(b) - f(a)}{b - a} = 0$$

因此：
$$f'(c) = \frac{f(b) - f(a)}{b - a}$$

**结论**：存在 $c \in (a, b)$ 满足上述等式。∎

---

## 四、双语对照：自然语言证明 ↔ Lean4 代码

### 定理陈述

**数学表述**：设 $f$ 在 $[a, b]$ 连续、在 $(a, b)$ 可导，则存在 $c \in (a, b)$ 使得 $f'(c) = \dfrac{f(b) - f(a)}{b - a}$。

```lean4
-- 拉格朗日中值定理
theorem lagrange_mean_value {f : ℝ → ℝ} {a b : ℝ} (hab : a < b)
    (hcont : ContinuousOn f (Icc a b))
    (hdiff : ∀ x ∈ Ioo a b, HasDerivAt f (f' x) x) :
    ∃ c ∈ Ioo a b, f' c = (f b - f a) / (b - a) := by
```

### 步骤 1：构造辅助函数

**数学**：$g(x) = f(x) - \dfrac{f(b) - f(a)}{b - a}(x - a) - f(a)$

```lean4
  let g : ℝ → ℝ := fun x => f x - ((f b - f a) / (b - a)) * (x - a) - f a
```

### 步骤 2：验证端点条件

**数学**：$g(a) = 0$ 且 $g(b) = 0$。

```lean4
  have h_ga : g a = 0 := by
    simp [g]
  have h_gb : g b = 0 := by
    simp [g]
    field_simp
    ring
```

### 步骤 3：验证连续性

**数学**：$g$ 在 $[a, b]$ 上连续（因为 $f$ 连续，线性函数连续）。

```lean4
  have h_g_cont : ContinuousOn g (Icc a b) := by
    apply ContinuousOn.sub
    · apply ContinuousOn.sub
      · exact hcont
      · apply ContinuousOn.mul
        · exact continuousOn_const
        · apply continuousOn_id.sub
          exact continuousOn_const
    · exact continuousOn_const
```

### 步骤 4：验证可导性

**数学**：$g'(x) = f'(x) - \dfrac{f(b) - f(a)}{b - a}$。

```lean4
  have h_g_diff : ∀ x ∈ Ioo a b, HasDerivAt g (f' x - (f b - f a) / (b - a)) x := by
    intro x hx
    simp [g]
    apply HasDerivAt.sub
    · apply HasDerivAt.sub
      · exact hdiff x hx
      · apply HasDerivAt.mul_const
        apply HasDerivAt.sub
        · exact hasDerivAt_id x
        · exact hasDerivAt_const x a
    · exact hasDerivAt_const x (f a)
```

### 步骤 5：应用罗尔定理

**数学**：由罗尔定理，存在 $c \in (a, b)$ 使得 $g'(c) = 0$。

```lean4
  rcases exists_hasDerivAt_eq_zero hab h_g_cont
      (fun x hx => h_g_diff x hx) (h_ga.trans h_gb.symm) with ⟨c, hc, hderiv⟩
```

### 步骤 6：得出结论

**数学**：由 $g'(c) = 0$ 推出 $f'(c) = \dfrac{f(b) - f(a)}{b - a}$。

```lean4
  use c, hc
  simp at hderiv
  linarith
```

## 五、典型例题

### 例题 1：证明当 $x > 0$ 时，$\sin x < x$

**证明**：令 $f(t) = t - \sin t$。

- $f$ 在 $[0, x]$ 上连续，在 $(0, x)$ 内可导
- $f(0) = 0 - 0 = 0$

由中值定理，存在 $c \in (0, x)$ 使得：
$$f(x) - f(0) = f'(c) \cdot (x - 0)$$

计算导数：
$$f'(t) = 1 - \cos t$$

因此：
$$f(x) = (1 - \cos c) \cdot x$$

由于 $c \in (0, x)$ 且 $c > 0$，有 $\cos c < 1$（因为 $\cos t = 1$ 仅当 $t = 2k\pi$，而 $c \in (0, x)$ 且可取 $x < 2\pi$；对于一般 $x > 0$，$1 - \cos c \geq 0$，且等号仅在 $c = 2k\pi$ 时成立。若 $x > 0$ 足够小，则 $c \in (0, x)$ 保证 $\cos c \neq 1$，故 $1 - \cos c > 0$）。

更严格地：对任意 $x > 0$，存在 $c \in (0, x)$ 使得：
$$x - \sin x = (1 - \cos c) \cdot x \geq 0$$

若 $x - \sin x = 0$，则 $1 - \cos c = 0$，即 $c = 2k\pi$。但 $c \in (0, x)$，若 $x \leq 2\pi$ 则不可能。若 $x > 2\pi$，可在 $[0, 2\pi]$ 上先应用中值定理得到 $\sin(2\pi) < 2\pi$，然后 $\sin x \leq 1 < x$ 对 $x > 2\pi$ 显然成立。

因此 $x - \sin x > 0$，即 $\sin x < x$ 对所有 $x > 0$ 成立。∎

---

### 例题 2：证明当 $x > 0$ 时，$e^x > 1 + x$

**证明**：令 $f(t) = e^t$。

在 $[0, x]$ 上应用中值定理，存在 $c \in (0, x)$ 使得：
$$e^x - e^0 = e^c \cdot (x - 0) = e^c \cdot x$$

由于 $c > 0$，有 $e^c > e^0 = 1$。因此：
$$e^x - 1 = e^c \cdot x > 1 \cdot x = x$$

即 $e^x > 1 + x$。∎

---

## 六、常见误区分析

### ❌ 误区：忽视端点连续性和内部可导性的条件

**错误陈述**："只要 $f$ 在 $(a, b)$ 内可导，中值定理就成立。"

**纠正**：中值定理要求 $f$ 在**闭区间** $[a, b]$ 上**连续**，且在**开区间** $(a, b)$ 内可导。两个条件缺一不可。

**反例**：设 $f(x) = \begin{cases} x & 0 \leq x < 1 \\ 0 & x = 1 \end{cases}$。

- $f$ 在 $(0, 1)$ 内可导（$f'(x) = 1$）
- 但 $f$ 在 $x = 1$ 处不连续

此时 $\dfrac{f(1) - f(0)}{1 - 0} = \dfrac{0 - 0}{1} = 0$，但 $f'(x) = 1$ 对所有 $x \in (0, 1)$，不存在 $c$ 使 $f'(c) = 0$。

**另一个反例**：$f(x) = |x|$ 在 $[-1, 1]$ 上。

- $f$ 在 $[-1, 1]$ 上连续
- 但 $f$ 在 $x = 0$ 处不可导

此时 $\dfrac{f(1) - f(-1)}{1 - (-1)} = \dfrac{1 - 1}{2} = 0$，但 $f'(x) = \pm 1$ 对所有 $x \neq 0$，不存在 $c$ 使 $f'(c) = 0$。

---

## 七、关键洞察

1. **局部与全局的桥梁**：中值定理是微分学中连接**局部性质**（导数在某点的值）与**全局性质**（函数在区间上的整体变化）的最重要的桥梁。没有中值定理，导数就只能描述局部行为，无法推断函数的单调性、凹凸性等全局特征。

2. **不等式证明的利器**：中值定理是证明分析学中各种不等式的核心工具。通过估计中值定理中的 $f'(c)$，可以将函数值的比较转化为导数值的比较。

3. **与其他定理的关系**：
   - **罗尔定理**：中值定理的特例（$f(a) = f(b)$）
   - **柯西中值定理**：中值定理的推广（两个函数的情形）
   - **Taylor 定理**：中值定理的高阶推广
   - **洛必达法则**：柯西中值定理的应用

## 习题

**习题 1.1**。证明 $|\\arctan x - \\arctan y| \\leq |x-y|$ 对所有 $x,y\\in\\mathbb{R}$。

*解答*：$(\\arctan x)' = \\frac{1}{1+x^2} \\leq 1$。由中值定理 $|\\arctan x - \\arctan y| = \\frac{1}{1+c^2}|x-y| \\leq |x-y|$。$\\square$

---

**习题 1.2**。设 $f$ 在 $[a,b]$ 上二阶可微，$f(a)=f(b)=0$，且存在 $c\\in(a,b)$ 使 $f(c)>0$。证明存在 $\\xi\\in(a,b)$ 使 $f''(\\xi)<0$。

*解答*：由中值定理，存在 $\\eta_1\\in(a,c)$ 使 $f'(\\eta_1) = \\frac{f(c)}{c-a} > 0$；存在 $\\eta_2\\in(c,b)$ 使 $f'(\\eta_2) = \\frac{-f(c)}{b-c} < 0$。再对 $f'$ 在 $[\\eta_1,\\eta_2]$ 上用中值定理，存在 $\\xi$ 使 $f''(\\xi) = \\frac{f'(\\eta_2)-f'(\\eta_1)}{\\eta_2-\\eta_1} < 0$。$\\square$
