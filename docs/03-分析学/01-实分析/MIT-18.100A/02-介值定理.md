# 介值定理（Intermediate Value Theorem）

## 定理陈述

**定理（介值定理 / Bolzano 介值定理）**：设函数 $f: [a, b] \to \mathbb{R}$ 在闭区间 $[a, b]$ 上**连续**，且 $y$ 是介于 $f(a)$ 与 $f(b)$ 之间的任意实数（即 $f(a) < y < f(b)$ 或 $f(b) < y < f(a)$），则存在 $c \in (a, b)$ 使得：

$$f(c) = y$$

**等价表述**：闭区间上的连续函数的值域是一个区间。即若 $f$ 在 $[a, b]$ 上连续，则 $f([a, b])$ 是包含端点值 $f(a)$ 和 $f(b)$ 的区间。

**推论（零点存在定理）**：若 $f: [a, b] \to \mathbb{R}$ 连续，且 $f(a) \cdot f(b) < 0$（即 $f(a)$ 与 $f(b)$ 异号），则存在 $c \in (a, b)$ 使得 $f(c) = 0$。

---

## 证明思路

介值定理是连续函数"不会跳过任何值"的严格数学表述。证明的核心方法是**二分法**（Bisection Method），这是分析学中构造性证明的经典范例：

1. **初始区间**：$[a_0, b_0] = [a, b]$，满足 $f(a_0)$ 与 $f(b_0)$ 在 $y$ 的两侧
2. **反复二分**：取中点 $c_n = \dfrac{a_n + b_n}{2}$
   - 若 $f(c_n) = y$，得证
   - 若 $f(c_n)$ 与 $f(a_n)$ 在 $y$ 同侧，则令 $[a_{n+1}, b_{n+1}] = [c_n, b_n]$
   - 否则令 $[a_{n+1}, b_{n+1}] = [a_n, c_n]$
3. **区间套收敛**：得到区间套 $[a_n, b_n]$，长度趋于 0，且 $f(a_n)$ 与 $f(b_n)$ 始终在 $y$ 的两侧
4. **应用连续性**：由区间套定理，存在唯一的 $c \in \bigcap [a_n, b_n]$。由连续性，$f(c) = y$

---

## 详细证明（二分法）

不失一般性，假设 $f(a) < y < f(b)$。（若 $f(b) < y < f(a)$，可对 $-f$ 应用同样的论证。）

---

### 步骤 1：初始化

令 $a_0 = a$，$b_0 = b$。

**性质**：$f(a_0) < y < f(b_0)$，且 $a_0 < b_0$。

---

### 步骤 2：递归构造区间套

假设已构造 $[a_n, b_n]$ 满足：

- $a_n < b_n$
- $f(a_n) \leq y \leq f(b_n)$

令 $c_n = \dfrac{a_n + b_n}{2}$。

**分三种情形**：

1. **若 $f(c_n) = y$**：令 $c = c_n$，定理得证。

2. **若 $f(c_n) < y$**：令 $a_{n+1} = c_n$，$b_{n+1} = b_n$。
   - 此时 $f(a_{n+1}) = f(c_n) < y$，且 $f(b_{n+1}) = f(b_n) > y$
   - 故 $f(a_{n+1}) < y < f(b_{n+1})$

3. **若 $f(c_n) > y$**：令 $a_{n+1} = a_n$，$b_{n+1} = c_n$。
   - 此时 $f(a_{n+1}) = f(a_n) < y$，且 $f(b_{n+1}) = f(c_n) > y$
   - 故 $f(a_{n+1}) < y < f(b_{n+1})$

若在某一步得到 $f(c_n) = y$，证明结束。否则，得到无穷区间序列：
$$[a_0, b_0] \supseteq [a_1, b_1] \supseteq [a_2, b_2] \supseteq \cdots$$

---

### 步骤 3：区间套的性质

**长度计算**：
$$b_n - a_n = \frac{b - a}{2^n} \to 0 \quad (n \to \infty)$$

由**区间套定理**，存在唯一的 $c \in \mathbb{R}$ 使得：
$$\bigcap_{n=0}^{\infty} [a_n, b_n] = \{c\}$$

且 $c \in [a, b]$。

---

### 步骤 4：利用连续性证明 $f(c) = y$

**关键不等式**：由构造，对所有 $n$：
$$f(a_n) \leq y \leq f(b_n)$$

**连续性应用**：

- 由于 $a_n \to c$ 且 $f$ 在 $c$ 连续，有 $f(a_n) \to f(c)$
- 由于 $b_n \to c$ 且 $f$ 在 $c$ 连续，有 $f(b_n) \to f(c)$

**夹逼定理**：
$$f(a_n) \leq y \leq f(b_n)$$
令 $n \to \infty$：
$$f(c) \leq y \leq f(c)$$

因此 $f(c) = y$。

**结论**：存在 $c \in [a, b]$ 使得 $f(c) = y$。进一步，由于 $f(a) \neq y$ 且 $f(b) \neq y$（严格不等式），必有 $c \in (a, b)$。∎

## 双语对照：自然语言证明 ↔ Lean4 代码

### 定理陈述

**数学表述**：设 $f: [a, b] \to \mathbb{R}$ 连续，$y$ 介于 $f(a)$ 与 $f(b)$ 之间，则存在 $c \in [a, b]$ 使得 $f(c) = y$。

```lean4
-- 介值定理：连续函数取得端点值之间的所有值
theorem intermediate_value {f : ℝ → ℝ} {a b : ℝ} (hab : a ≤ b)
    (hf : ContinuousOn f (Icc a b)) {y : ℝ}
    (hy : f a ≤ y ∧ y ≤ f b ∨ f b ≤ y ∧ y ≤ f a) :
    ∃ (c : ℝ), c ∈ Icc a b ∧ f c = y := by
```

### 步骤 1：利用连通性

**数学**：闭区间 $[a, b]$ 是 $\mathbb{R}$ 中的连通集。

```lean4
  let S := f '' Icc a b
  have h_connected : IsPreconnected (Icc a b) := connected_Icc hab
```

### 步骤 2：连续像保持连通性

**数学**：连续函数将连通集映为连通集，因此 $f([a, b])$ 是连通的。

```lean4
  have h_image_connected : IsPreconnected S := by
    apply IsPreconnected.image h_connected hf
```

### 步骤 3：像集包含端点值

**数学**：$f(a) \in f([a, b])$ 且 $f(b) \in f([a, b])$。

```lean4
  have ha_in : f a ∈ S := by
    use a
    constructor
    · exact ⟨by linarith, by linarith⟩
    · rfl
  have hb_in : f b ∈ S := by
    use b
    constructor
    · exact ⟨by linarith, by linarith⟩
    · rfl
```

### 步骤 4：连通区间包含中间值

**数学**：$\mathbb{R}$ 中的连通集恰好是区间。因 $f([a, b])$ 是包含 $f(a)$ 和 $f(b)$ 的区间，故包含它们之间的所有值，包括 $y$。

```lean4
  have h_interval : IsPreconnectedOrd S := by
    rw [← isPreconnected_iff_isPreconnectedOrd]
    exact h_image_connected
  have hy_in : y ∈ S := by
    cases hy with
    | inl h =>
        exact h_interval.mem_interval ha_in hb_in h.1 h.2
    | inr h =>
        exact h_interval.mem_interval hb_in ha_in h.1 h.2
```

### 步骤 5：得到存在性结论

**数学**：$y \in f([a, b])$ 意味着存在 $c \in [a, b]$ 使得 $f(c) = y$。

```lean4
  rcases hy_in with ⟨c, hc, hfc⟩
  use c
  constructor
  · exact hc
  · exact hfc
```

## 典型例题

### 例题 1：证明方程 $x^3 - x - 1 = 0$ 在 $(1, 2)$ 内有根

**证明**：令 $f(x) = x^3 - x - 1$。

- $f$ 是多项式函数，在 $[1, 2]$ 上连续
- $f(1) = 1 - 1 - 1 = -1 < 0$
- $f(2) = 8 - 2 - 1 = 5 > 0$

由**零点存在定理**，存在 $c \in (1, 2)$ 使得 $f(c) = 0$。∎

---

### 例题 2：一维 Brouwer 不动点定理

**问题**：设 $f: [0, 1] \to [0, 1]$ 连续，证明 $f$ 必有不动点，即存在 $c \in [0, 1]$ 使得 $f(c) = c$。

**证明**：构造辅助函数 $g(x) = f(x) - x$。

- $f$ 连续，$x$ 连续，故 $g$ 在 $[0, 1]$ 上连续
- $g(0) = f(0) - 0 = f(0) \geq 0$（因为 $f(0) \in [0, 1]$）
- $g(1) = f(1) - 1 \leq 0$（因为 $f(1) \in [0, 1]$）

**分情形**：

- 若 $g(0) = 0$，则 $c = 0$ 是不动点
- 若 $g(1) = 0$，则 $c = 1$ 是不动点
- 若 $g(0) > 0$ 且 $g(1) < 0$，由零点存在定理，存在 $c \in (0, 1)$ 使得 $g(c) = 0$，即 $f(c) = c$

综上，不动点必存在。∎

## 常见误区分析

### ❌ 误区：忽视定义域必须是闭区间

**错误陈述**："开区间上的连续函数也满足介值性质。"

**纠正**：介值定理**确实**对任意区间（开、闭、半开半闭、无限）上的连续函数都成立。但我们的标准证明（二分法）依赖于区间的**连通性**和**完备性**。更常见的错误是：

**真正的误区：将介值定理用于不连续函数**

**反例**：设 $f(x) = \begin{cases} x & x \neq 1 \\ 0 & x = 1 \end{cases}$ 在 $[0, 2]$ 上。$f(0) = 0$，$f(2) = 2$，但 $f$ 在 $x = 1$ 处不连续，且 $f$ 不取到 $1.5$（实际上 $f(1.5) = 1.5$，这个反例不太好）。

更好的反例：设 $f(x) = \begin{cases} -1 & x \in [0, 1) \\ 1 & x \in [1, 2] \end{cases}$。$f(0) = -1$，$f(2) = 1$，但 $f$ 在 $x = 1$ 处不连续，且 $f$ 不取到 $0$。

**核心教训**：**连续性**是介值定理的关键前提，不能省略。

---

## 关键洞察

1. **二分法是构造性证明**：介值定理的证明不仅给出了存在性，还提供了一种**算法**（二分法）来逼近根 $c$。每步迭代误差减半：$|c_n - c| \leq \dfrac{b - a}{2^n}$。

2. **拓扑视角**：从拓扑学角度看，介值定理的本质是**连续函数保持连通性**。$[a, b]$ 是连通的，因此 $f([a, b])$ 也是连通的；而 $\mathbb{R}$ 中的连通集恰好是区间。

3. **与极值定理的关系**：介值定理和极值定理共同刻画了闭区间上连续函数的优良性质——值域是**闭区间** $[m, M]$，其中 $m$ 是最小值，$M$ 是最大值。

## 习题

**习题 1.1**。证明：奇数次实系数多项式至少有一个实根。

*解答*：设 $p(x) = a_n x^n + \\cdots + a_0$，$n$ 奇数。当 $x \\to +\\infty$，$p(x) \\to \\operatorname{sgn}(a_n)\\infty$；当 $x \\to -\\infty$，$p(x) \\to -\\operatorname{sgn}(a_n)\\infty$。故 $p$ 变号，由介值定理存在根。$\\square$

---

**习题 1.2**。设 $f:[0,1]\\to[0,1]$ 连续且 $f(0)=0, f(1)=1$。证明存在 $c$ 使 $f(c)=c^2$。

*解答*：令 $g(x) = f(x) - x^2$，则 $g(0)=0, g(1)=0$。若 $g\\equiv 0$ 则结论显然；否则存在 $x_0$ 使 $g(x_0)\\neq 0$，由介值定理在 $(0,1)$ 内存在零点。$\\square$
