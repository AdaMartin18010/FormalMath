# Princeton 复分析学习诊断手册

## 常见错误模式

### 错误1: 混淆解析与可微

**错误认知**: 复可微 ⇒ 实可微，但实可微 ⇏ 复可微
**正确理解**: Cauchy-Riemann方程是复可微的充要条件。f(z)=u(x,y)+iv(x,y)复可微当且仅当u,v满足CR方程且连续可微。
**检测方法**: 检查学生是否能构造反例 f(z)=z̄（实可微但非复可微）

### 错误2: 忽视Cauchy定理的条件

**错误认知**: 任何闭曲线上的解析函数积分为0
**正确理解**: 需要：1) 函数在单连通区域内解析；2) 曲线在区域内可缩。
**检测方法**: 检查学生是否能计算 ∮_{|z|=1} 1/z dz = 2πi ≠ 0

### 错误3: 错误应用留数定理

**错误认知**: 留数定理适用于所有孤立奇点
**正确理解**: 需要：1) 函数在区域内除有限个极点外解析；2) 极点在积分围道内部。
**检测方法**: 检查学生是否知道分支点（如√z在0处）不能用留数定理

### 错误4: 混淆Taylor与Laurent展开

**错误认知**: 所有解析函数都可Taylor展开
**正确理解**: Taylor展开要求函数在圆盘内解析。环形区域需要Laurent展开，允许负幂项。

## 能力层级

| 层级 | 表现 | 建议 |
|------|------|------|
| L1 | 能计算基本复积分 | 加强CR方程理解 |
| L2 | 能应用Cauchy定理 | 练习留数计算 |
| L3 | 能计算实积分 | 学习共形映射 |
| L4 | 理解Riemann映射 | 研究整函数理论 |
