# 极限与连续性 - 深度版内容深化补充

## 一、证明完善与ε-δ详细证明

### 1.1 函数极限的ε-δ证明（详细步骤）

**定理**：$\lim_{x \to a} f(x) = L$ 当且仅当
$$\forall \varepsilon > 0, \exists \delta > 0, \forall x: 0 < |x - a| < \delta \Rightarrow |f(x) - L| < \varepsilon$$

**示例**：证明 $\lim_{x \to 2} (3x + 1) = 7$

**证明**：

给定 $\varepsilon > 0$，需要找到 $\delta > 0$ 使得：
$$0 < |x - 2| < \delta \Rightarrow |(3x + 1) - 7| < \varepsilon$$

**分析不等式**：
$$|3x + 1 - 7| = |3x - 6| = 3|x - 2|$$

**选择δ**：
需要 $3|x - 2| < \varepsilon$，即 $|x - 2| < \frac{\varepsilon}{3}$。

取 $\delta = \frac{\varepsilon}{3}$。

**验证**：
当 $0 < |x - 2| < \delta = \frac{\varepsilon}{3}$ 时：
$$|3x + 1 - 7| = 3|x - 2| < 3 \cdot \frac{\varepsilon}{3} = \varepsilon$$

**证明思路**：通过代数变形将 $|f(x) - L|$ 表示为 $|x - a|$ 的函数，然后反解出 $\delta$。

---

### 1.2 二次函数的ε-δ证明

**定理**：证明 $\lim_{x \to 3} x^2 = 9$

**证明**：

给定 $\varepsilon > 0$（设 $\varepsilon < 1$），分析：
$$|x^2 - 9| = |x - 3||x + 3|$$

**控制 $|x + 3|$**：
先限制 $|x - 3| < 1$，则 $2 < x < 4$，故 $|x + 3| < 7$。

**选择δ**：
取 $\delta = \min\{1, \frac{\varepsilon}{7}\}$。

**验证**：
当 $0 < |x - 3| < \delta$ 时：

- 若 $\delta = 1$，则 $|x - 3| < \frac{\varepsilon}{7}$，故 $|x^2 - 9| < \frac{\varepsilon}{7} \cdot 7 = \varepsilon$
- 若 $\delta = \frac{\varepsilon}{7}$，则 $|x^2 - 9| < \frac{\varepsilon}{7} \cdot 7 = \varepsilon$

**证明思路**：对于非线性函数，先限制 $x$ 的范围以控制增长因子（此处为 $|x+3|$），再选择适当的 $\delta$。

---

### 1.3 连续性ε-δ证明

**定理**：证明 $f(x) = x^3$ 在 $x = 2$ 处连续

**证明**：

给定 $\varepsilon > 0$（设 $\varepsilon < 1$），分析：
$$|x^3 - 8| = |x - 2||x^2 + 2x + 4|$$

**控制 $|x^2 + 2x + 4|$**：
限制 $|x - 2| < 1$，则 $1 < x < 3$。

在此区间：

- $x^2 < 9$
- $2x < 6$
- 故 $|x^2 + 2x + 4| < 9 + 6 + 4 = 19$

**选择δ**：
取 $\delta = \min\{1, \frac{\varepsilon}{19}\}$。

**验证**：
当 $|x - 2| < \delta$ 时：
$$|x^3 - 8| = |x - 2||x^2 + 2x + 4| < \frac{\varepsilon}{19} \cdot 19 = \varepsilon$$

**证明思路**：立方函数在局部有界，利用多项式的有界性因子来选取 $\delta$。

### 1.4 Cantor一致连续性定理详细证明

**定理**：闭区间 $[a,b]$ 上的连续函数必一致连续。

**证明**（反证法）：

假设 $f$ 不一致连续，则：
$$\exists \varepsilon > 0, \forall \delta > 0, \exists x, y \in [a,b]: |x - y| < \delta \land |f(x) - f(y)| \geq \varepsilon$$

**构造序列**：

取 $\delta_n = \frac{1}{n}$，则 $\exists x_n, y_n \in [a,b]$ 使得：
$$|x_n - y_n| < \frac{1}{n} \text{ 且 } |f(x_n) - f(y_n)| \geq \varepsilon$$

**应用Bolzano-Weierstrass**：

$(x_n) \subseteq [a,b]$ 有界，存在收敛子列 $x_{n_k} \to c \in [a,b]$。

由 $|x_{n_k} - y_{n_k}| < \frac{1}{n_k} \to 0$，得 $y_{n_k} \to c$。

**导出矛盾**：

由 $f$ 在 $c$ 连续：

- $f(x_{n_k}) \to f(c)$，故 $\exists K_1: \forall k \geq K_1$，$|f(x_{n_k}) - f(c)| < \frac{\varepsilon}{2}$
- $f(y_{n_k}) \to f(c)$，故 $\exists K_2: \forall k \geq K_2$，$|f(y_{n_k}) - f(c)| < \frac{\varepsilon}{2}$

对 $k \geq \max(K_1, K_2)$：
$$|f(x_{n_k}) - f(y_{n_k})| \leq |f(x_{n_k}) - f(c)| + |f(c) - f(y_{n_k})| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon$$

这与 $|f(x_{n_k}) - f(y_{n_k})| \geq \varepsilon$ 矛盾！

**证明思路**：不一致连续性可构造两个收敛于同一点的序列，但函数值差保持正下界，这与连续性矛盾。

## 二、习题扩充（15道）

### 2.1 基础计算题（5道）

**题1**：用ε-δ定义证明 $\lim_{x \to 1} (2x - 3) = -1$

**解答**：
给定 $\varepsilon > 0$，取 $\delta = \frac{\varepsilon}{2}$。

当 $0 < |x - 1| < \delta$ 时：
$$|(2x - 3) - (-1)| = |2x - 2| = 2|x - 1| < 2 \cdot \frac{\varepsilon}{2} = \varepsilon$$

**题2**：求下列极限：
$$(a) \lim_{x \to 0} \frac{\sin x}{x} \quad (b) \lim_{x \to \infty} \frac{3x^2 + 2x - 1}{x^2 + 5}$$

**解答**：
(a) 重要极限，值为 $1$

(b) 分子分母同除 $x^2$：
$$\lim_{x \to \infty} \frac{3 + \frac{2}{x} - \frac{1}{x^2}}{1 + \frac{5}{x^2}} = \frac{3 + 0 - 0}{1 + 0} = 3$$

**题3**：计算 $\lim_{x \to 2} \frac{x^2 - 4}{x - 2}$

**解答**：
$$\lim_{x \to 2} \frac{x^2 - 4}{x - 2} = \lim_{x \to 2} \frac{(x-2)(x+2)}{x-2} = \lim_{x \to 2} (x+2) = 4$$

**题4**：判断函数 $f(x) = \begin{cases} x\sin\frac{1}{x}, & x \neq 0 \\ 0, & x = 0 \end{cases}$ 在 $x=0$ 处的连续性。

**解答**：
$$|f(x) - f(0)| = |x\sin\frac{1}{x}| \leq |x| \to 0 \text{ (当 } x \to 0\text{)}$$

故 $\lim_{x \to 0} f(x) = 0 = f(0)$，函数在 $x=0$ 连续。

**题5**：求函数 $f(x) = x^2$ 在区间 $[0, 2]$ 上的Lipschitz常数（如果存在）。

**解答**：
$$|f(x) - f(y)| = |x^2 - y^2| = |x + y||x - y|$$

在 $[0, 2]$ 上，$|x + y| \leq 4$，故Lipschitz常数为 $4$。

### 2.2 概念理解题（5道）

**题6**（判断题）：若 $f$ 在 $a$ 点连续，则 $f$ 在 $a$ 的某邻域内有界。

**答案**：**错误**

**解释**：连续性只保证局部有界。反例：$f(x) = \frac{1}{x}$ 在 $x=1$ 连续，但在任何包含 $0$ 的区间无界。实际上，若 $f$ 在 $a$ 连续，则存在 $a$ 的某个邻域使 $f$ 有界。

修正后答案：**正确**（在某邻域内有界）

**题7**（选择题）：下列哪个函数在 $x=0$ 处不连续？

A. $f(x) = |x|$
B. $f(x) = \begin{cases} \frac{\sin x}{x}, & x \neq 0 \\ 1, & x = 0 \end{cases}$
C. $f(x) = \begin{cases} 1, & x \geq 0 \\ -1, & x < 0 \end{cases}$
D. $f(x) = x^2$

**答案**：**C**

**解释**：C在$x=0$处有跳跃间断（左右极限不相等）。A、B、D都在$x=0$连续。

**题8**（简答题）：解释"一致连续"与"点连续"的本质区别。

**答案**：

- **点连续**：$\delta$ 依赖于点 $x$ 和 $\varepsilon$，即 $\delta = \delta(x, \varepsilon)$
- **一致连续**：$\delta$ 只依赖于 $\varepsilon$，即 $\delta = \delta(\varepsilon)$，对所有点统一适用

一致连续保证了函数在整个区间上"均匀"连续，没有局部的"突变"。

**题9**（判断题）：单调函数只有第一类间断点。

**答案**：**正确**

**解释**：单调函数在每点的左右极限都存在（单调有界定理），故间断点只能是跳跃间断（第一类）。

**题10**（简答题）：描述Heine定理的作用和意义。

**答案**：Heine定理建立了函数极限与数列极限的联系：
$$\lim_{x \to a} f(x) = L \Leftrightarrow \forall x_n \to a, x_n \neq a: \lim_{n \to \infty} f(x_n) = L$$

**意义**：

1. 将函数极限问题转化为数列极限问题
2. 为证明函数极限不存在提供方法（找两个收敛于 $a$ 但函数值极限不同的数列）

### 2.3 证明题（5道）

**题11**（简单）：用ε-δ定义证明常数函数 $f(x) = c$ 在所有点连续。

**证明**：
给定 $\varepsilon > 0$ 和任意点 $a$，取任意 $\delta > 0$。

当 $|x - a| < \delta$ 时：
$$|f(x) - f(a)| = |c - c| = 0 < \varepsilon$$

故 $f$ 在 $a$ 连续。由 $a$ 任意性，$f$ 处处连续。

**题12**（简单）：证明若 $f$ 和 $g$ 都在 $a$ 连续，则 $f + g$ 也在 $a$ 连续。

**证明**：
给定 $\varepsilon > 0$：

- 因 $f$ 在 $a$ 连续，$\exists \delta_1 > 0: |x - a| < \delta_1 \Rightarrow |f(x) - f(a)| < \frac{\varepsilon}{2}$
- 因 $g$ 在 $a$ 连续，$\exists \delta_2 > 0: |x - a| < \delta_2 \Rightarrow |g(x) - g(a)| < \frac{\varepsilon}{2}$

取 $\delta = \min\{\delta_1, \delta_2\}$，当 $|x - a| < \delta$ 时：
$$|(f+g)(x) - (f+g)(a)| \leq |f(x) - f(a)| + |g(x) - g(a)| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon$$

**题13**（中等）：证明 $f(x) = \sqrt{x}$ 在 $[0, \infty)$ 上一致连续。

**证明**：
对 $x, y \geq 0$：
$$|\sqrt{x} - \sqrt{y}| = \frac{|x - y|}{\sqrt{x} + \sqrt{y}}$$

**情况1**：$x, y \in [0, 1]$，则 $\sqrt{x} + \sqrt{y} \geq 0$（需另处理）

实际上用Lipschitz条件：对 $x, y \geq 1$，$\sqrt{x} + \sqrt{y} \geq 2$，故：
$$|\sqrt{x} - \sqrt{y}| \leq \frac{|x - y|}{2}$$

对 $[0, 1]$ 用Cantor定理（闭区间连续必一致连续）。

**题14**（中等）：证明介值定理的特殊情形：若 $f$ 在 $[0, 1]$ 连续，$f(0) < 0$，$f(1) > 0$，则 $\exists c \in (0, 1): f(c) = 0$。

**证明**：
令 $S = \{x \in [0, 1] : f(x) < 0\}$，则 $S$ 非空（$0 \in S$）且有上界。

令 $c = \sup S$，则 $c \in [0, 1]$。

**证明 $f(c) = 0$**：

- 若 $f(c) < 0$，由连续性，$\exists \delta > 0$ 使 $f(x) < 0$ 对所有 $x \in (c - \delta, c + \delta) \cap [0,1]$ 成立，这与 $c = \sup S$ 矛盾。
- 若 $f(c) > 0$，同理存在左邻域使 $f(x) > 0$，与 $c = \sup S$ 矛盾。

故 $f(c) = 0$。

**题15**（困难）：证明有界闭区间上的连续函数必能达到最大值和最小值。

**证明**（仅证最大值）：

设 $M = \sup\{f(x) : x \in [a,b]\}$（由有界性，$M$ 有限或由最值定理存在性部分）。

由确界定义，$\forall n \in \mathbb{N}$，$\exists x_n \in [a,b]$ 使得：
$$M - \frac{1}{n} < f(x_n) \leq M$$

序列 $(x_n) \subseteq [a,b]$ 有界，由Bolzano-Weierstrass定理，存在收敛子列 $x_{n_k} \to c \in [a,b]$。

由 $f$ 在 $c$ 连续：
$$f(c) = \lim_{k \to \infty} f(x_{n_k})$$

由夹逼定理，$\lim f(x_{n_k}) = M$，故 $f(c) = M$。

## 三、示例扩充

### 3.1 极限计算示例

**示例1**：$\lim_{x \to 0} \frac{\sin x}{x} = 1$（重要极限）

**几何解释**：单位圆中，弧长 $x$ 与弦长 $2\sin(x/2)$ 的比值趋于1。

**示例2**：$\lim_{x \to \infty} (1 + \frac{1}{x})^x = e$

**典型应用**：连续复利公式，人口增长模型。

**示例3**：$\lim_{x \to 0} \frac{1 - \cos x}{x^2} = \frac{1}{2}$

**计算**：$\frac{1 - \cos x}{x^2} = \frac{2\sin^2(x/2)}{x^2} = \frac{1}{2} \cdot \frac{\sin^2(x/2)}{(x/2)^2} \to \frac{1}{2}$

**示例4**（边界情况）：$\lim_{x \to 0} x\sin\frac{1}{x} = 0$

**分析**：有界量 $\times$ 无穷小量 = 无穷小量。

**示例5**：$\lim_{x \to 1} \frac{x^n - 1}{x - 1} = n$

**计算**：$\frac{x^n - 1}{x - 1} = 1 + x + x^2 + ... + x^{n-1} \to n$（当 $x \to 1$）

### 3.2 连续性示例

**示例1**：$f(x) = x^2$ 在 $\mathbb{R}$ 上连续

**示例2**：$f(x) = |x|$ 在 $\mathbb{R}$ 上连续（但 $x=0$ 不可导）

**示例3**：$f(x) = \begin{cases} x\sin\frac{1}{x}, & x \neq 0 \\ 0, & x = 0 \end{cases}$ 在 $x=0$ 连续

**示例4**（边界）：$f(x) = \begin{cases} \frac{\sin x}{x}, & x \neq 0 \\ 1, & x = 0 \end{cases}$ 在 $x=0$ 连续

**示例5**：$f(x) = \begin{cases} \frac{1}{x}, & x \neq 0 \\ 0, & x = 0 \end{cases}$ 在 $x=0$ 不连续（第二类间断点）

### 3.3 一致连续性示例

**示例1**：$f(x) = x$ 在 $\mathbb{R}$ 上一致连续（Lipschitz连续，$L=1$）

**示例2**：$f(x) = x^2$ 在 $[0, 1]$ 上一致连续（闭区间连续）

**示例3**（边界）：$f(x) = x^2$ 在 $\mathbb{R}$ 上**不**一致连续

**分析**：取 $x_n = n + \frac{1}{n}$，$y_n = n$，则 $|x_n - y_n| = \frac{1}{n} \to 0$，但 $|f(x_n) - f(y_n)| = |2 + \frac{1}{n^2}| \to 2 \neq 0$。

**示例4**：$f(x) = \sqrt{x}$ 在 $[0, \infty)$ 上一致连续

**示例5**：$f(x) = \sin x$ 在 $\mathbb{R}$ 上一致连续（Lipschitz连续，$L=1$）

## 四、反例专节

### 4.1 极限不存在反例

**反例1**：$\lim_{x \to 0} \sin\frac{1}{x}$ 不存在

**说明**：取 $x_n = \frac{1}{n\pi} \to 0$，则 $\sin\frac{1}{x_n} = 0$；取 $y_n = \frac{2}{(4n+1)\pi} \to 0$，则 $\sin\frac{1}{y_n} = 1$。

两个子列极限不同，故极限不存在。

**反例2**：$\lim_{x \to \infty} \sin x$ 不存在

**说明**：函数在无穷远处振荡，不趋于任何确定值。

**反例3**：$f(x) = \frac{|x|}{x}$ 在 $x \to 0$ 时极限不存在

**说明**：左极限为 $-1$，右极限为 $1$。

### 4.2 连续性反例

**反例1**：Dirichlet函数
$$D(x) = \begin{cases} 1, & x \in \mathbb{Q} \\ 0, & x \notin \mathbb{Q} \end{cases}$$

在每一点都不连续（处处间断）。

**反例2**：Thomae函数（修正Dirichlet函数）
$$f(x) = \begin{cases} \frac{1}{q}, & x = \frac{p}{q} \in \mathbb{Q} \text{（最简分数）} \\ 0, & x \notin \mathbb{Q} \end{cases}$$

在有理点间断，在无理点连续。

**反例3**：$f(x) = \begin{cases} \sin\frac{1}{x}, & x \neq 0 \\ 0, & x = 0 \end{cases}$ 在 $x=0$ 不连续（振荡间断）

### 4.3 一致连续性反例

**反例1**：$f(x) = x^2$ 在 $\mathbb{R}$ 上不一致连续

**分析**：虽然每点连续，但随着 $|x|$ 增大，变化率增大，无法找到统一的 $\delta$。

**反例2**：$f(x) = \frac{1}{x}$ 在 $(0, 1)$ 上不一致连续

**分析**：当 $x \to 0^+$ 时，函数值趋于无穷，变化率无界。

**反例3**：$f(x) = \sin(x^2)$ 在 $\mathbb{R}$ 上不一致连续

**分析**：取 $x_n = \sqrt{2n\pi + \frac{\pi}{2}}$，$y_n = \sqrt{2n\pi}$，则 $|x_n - y_n| \to 0$，但 $|f(x_n) - f(y_n)| = 1$。

## 五、几何直观与图示描述

### 5.1 ε-δ定义的几何直观

**图示描述**：

```
函数极限的几何意义：

y
|
  L+ε -------+     +     +-----------
        |    / \   / \    |
  L ----+---/---★---\----+-----------
        |  /     |     \  |
  L-ε --+-/------+------\-+-----------
        |/       |       \|
        +--------+--------+-----→ x
                a-δ  a  a+δ

★ = 极限点 (a, L)

对于任意给定的ε-带状区域（水平方向），
存在δ-区间（垂直方向），使得函数图像
在该区间内落入ε-带状区域。
```

**几何解释**：极限定义的几何意义是：当 $x$ 足够接近 $a$（在 $\delta$ 范围内），函数值 $f(x)$ 就落入 $L$ 的 $\varepsilon$ 邻域内。

### 5.2 一致连续性的几何直观

**图示描述**：

```
一致连续 vs 点连续：

点连续（δ依赖于位置）：
  在a点:   δ_a        在b点:     δ_b
     |<--->|             |<--->|
     +-----+-----+       +-----+-----+
    a-δ_a a a+δ_a       b-δ_b b b+δ_b

一致连续（统一δ）：
  所有点使用相同的δ
     |<--->|
  +--+--+--+--+--+--+--+--+--→ x
    a  b  c  d  e  f
```

**几何解释**：一致连续性要求在整个区间上，相同的 "$\delta$ 精度" 能保证相同的 "$\varepsilon$ 精度"。如果函数在某处"变化太快"，就无法找到统一的 $\delta$。

### 5.3 间断点分类的几何直观

**图示描述**：

```
各类间断点示意：

可去间断点：       跳跃间断点：
    |    ●              |    ●
    |   /               |    |
    |  /                |    |
----●----→ x         ----●----●----→ x
    a                 a

第二类间断点（无穷）： 第二类间断点（振荡）：
    |                   |  ~ ~ ~
    |    ↑              | / \ / \
    |   /|\            ●/   \   \
----●---+----→ x     --●----------→ x
    a                 a
    |
   ↓ -∞
```

**几何解释**：

- **可去间断点**：左右极限存在且相等，但不等于函数值（或函数无定义）
- **跳跃间断点**：左右极限存在但不相等，形成"跳跃"
- **无穷间断点**：至少一侧趋于无穷
- **振荡间断点**：至少一侧无限振荡，无极限

### 5.4 介值定理的几何直观

**图示描述**：

```
介值定理几何解释：

y
|
  f(b)●                     /●  f(b)
     |\                   /
     | \                 /
  k  +--\---------------/--+  ← 水平线 y=k
     |   \             /
     |    \           /
  f(a)●----\---------/----●  f(a)
     |      \       /
     |       \     /
     |        \   /
     |         \ /
     +----------●-----------→ x
               c

连续曲线从 y=f(a) 到 y=f(b) 必须
穿过水平线 y=k。
```

**几何解释**：连续函数的图像是一条"不间断的曲线"，如果曲线从 $f(a)$ 变化到 $f(b)$，则必须经过 $f(a)$ 和 $f(b)$ 之间的所有值。
