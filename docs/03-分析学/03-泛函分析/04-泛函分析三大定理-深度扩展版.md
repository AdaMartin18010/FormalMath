## 🔧 二、Hahn-Banach定理

### 2.1 解析形式

**定理**：设X是ℝ上的向量空间，p: X → ℝ是次可加正齐次泛函。设M⊂X是子空间，f: M → ℝ是线性泛函，且f(x) ≤ p(x) ∀x∈M。则存在线性泛函F: X → ℝ使得：

1. F|_M = f（F是f的延拓）

2. F(x) ≤ p(x) ∀x∈X

### 2.2 几何形式

**定理（分离定理）**：设X是赋范空间，A, B是非空不交凸集。

- **弱分离**：若A开，则存在x*∈ X*, α ∈ ℝ使得：

  ```text
  ⟨x*, a⟩ < α ≤ ⟨x*, b⟩, ∀a∈A, b∈B
  ```

- **强分离**：若A紧，B闭，则存在x*∈ X*, α, β ∈ ℝ使得：

  ```text
  ⟨x*, a⟩ < α < β < ⟨x*, b⟩
  ```

### 2.3 证明树

```text
        [Hahn-Banach定理证明]
                │
    ┌───────────┴───────────┐
    │                       │
[有限维情形]            [无限维情形]
    │                       │
归纳延拓               Zorn引理
一维 → n维                 │
    │               考虑所有延拓
    │               (f',M')其中f'|M=f

    │                       │
    │               偏序: (f₁,M₁)≤(f₂,M₂)
    │               ⟺ M₁⊂M₂且f₂|M₁=f₁

    │                       │
    │               链有上界(取并)
    │                       │
    │               Zorn ⟹ 存在极大元(F,N)
    │                       │
    │               证明N=X(否则可再延拓)
    │                       │
    └───────────┬───────────┘
                │
        F是满足条件的延拓 ∎

```

### 2.4 一维延拓关键步骤

```text
        [一维延拓]
              │
设M⊂X, x₀∉M, 需延拓f到M+ℝx₀
              │
定义F(m+tx₀) = f(m) + tc, 需选择c
              │
        [约束条件]
              │
对m₁,m₂∈M: f(m₁)-p(m₁-x₀) ≤ c ≤ p(m₂+x₀)-f(m₂)
              │
        [存在性]
              │
需证: sup(f(m₁)-p(m₁-x₀)) ≤ inf(p(m₂+x₀)-f(m₂))
              │
由p次可加和f≤p可得 ✓

```

### 2.5 重要推论

| 推论 | 内容 |
|------|------|
| 范数达到 | X完备时，∀x∈X ∃x*∈X*: ‖x*‖=1, x*(x)=‖x‖ |
| 对偶嵌入 | 自然嵌入j: X → X**: x ↦ (x*↦ x*(x))是等距 |
| 分离定理 | 凸集可被超平面分离 |

## ⚡ 三、Banach-Steinhaus定理

### 3.1 定理陈述

**定理（一致有界原理）**：设X是Banach空间，Y是赋范空间，{Tα}α∈A ⊂ B(X,Y)是有界线性算子族。若对每个x∈X，sup_α‖Tαx‖ < ∞（逐点有界），则sup_α‖Tα‖ < ∞（一致有界）。

### 3.2 证明树

```text
        [Banach-Steinhaus定理证明]
                │
        使用Baire纲定理
                │
    ┌───────────┴───────────┐
    │                       │
[构造闭集]              [应用Baire]
    │                       │
Fn = {x: sup_α‖Tαx‖≤n}   X = ∪Fn (因逐点有界)
    │                       │
Fn是闭集               X是完备的
(由算子连续性)         Banach空间
    │                       │
    └───────────┬───────────┘
                │
        Baire: 某Fₙ内部非空
                │
    存在x₀和r>0: B(x₀,r)⊂Fₙ
                │
        [范数估计]
                │
对‖x‖≤1: x₀+rx ∈ Fₙ
    ⟹ ‖Tα(x₀+rx)‖ ≤ n
    ⟹ ‖Tαx‖ ≤ (n+‖Tαx₀‖)/r ≤ (n+n)/r = 2n/r
                │
    sup_α‖Tα‖ ≤ 2n/r < ∞ ∎

```

### 3.3 应用

**应用1：Fourier级数发散**:

```text
存在连续函数f，其Fourier级数在某点发散

证明：设Sn(f)(0) = Σ_{k=-n}^n ĉ_k(f)
     定义Ln: C[0,2π] → ℂ, Ln(f) = Sn(f)(0)

若对所有f Fourier级数收敛于f(0)
⟹ sup_n|Ln(f)| < ∞ ∀f

⟹ sup_n‖Ln‖ < ∞ (Banach-Steinhaus)

但‖Ln‖ = ∫|Dn(t)|dt → ∞ (Lebesgue常数)

矛盾！

```

**应用2：算子序列收敛**:

若Tnx → Tx ∀x，且X完备，则T有界且‖T‖ ≤ liminf‖Tn‖

## 🚀 四、开映射定理与闭图像定理

### 4.1 开映射定理

**定理**：设X, Y是Banach空间，T: X → Y是有界线性满射，则T是开映射（开集的像是开集）。

**等价形式**：存在c > 0使得B_Y(0,c) ⊂ T(B_X(0,1))

### 4.2 证明树

```text
        [开映射定理证明]
                │
        [第一步: 稠密性]
                │
Y = T(X) = ∪_n T(nB_X) = ∪_n nT(B_X)
                │
由Baire定理: 某nT(B_X)内部非空
                │
⟹ T(B_X)的闭包有内点
                │
⟹ 存在ε>0: B_Y(0,2ε) ⊂ cl(T(B_X))
                │
        [第二步: 从闭包到像]
                │
目标: B_Y(0,ε) ⊂ T(B_X)
                │
取y ∈ B_Y(0,ε)
                │
├─存在x₁∈B_X: ‖y-Tx₁‖ < ε/2
│ (因y ∈ cl(T(B_X)))
├─存在x₂∈B_X/2: ‖y-Tx₁-Tx₂‖ < ε/4
├─...
├─存在xₙ∈B_X/2^{n-1}: ‖y-Σ_{k=1}^n Txₖ‖ < ε/2^n
                │
令x = Σxₖ (级数收敛因X完备)
‖x‖ < 2, Tx = y
                │
    B_Y(0,ε) ⊂ T(2B_X) = 2T(B_X)
                │
    ⟹ B_Y(0,ε/2) ⊂ T(B_X) ∎

```

### 4.3 闭图像定理

**定理**：设X, Y是Banach空间，T: X → Y是线性算子。则T有界 ⟺ Graph(T) = {(x,Tx): x∈X}在X×Y中闭。

### 4.4 证明（闭图像⟹有界）

```text
        [闭图像定理证明]
                │
Graph(T)闭 ⟹ (Graph(T), ‖·‖₁)是Banach空间
其中‖(x,y)‖₁ = ‖x‖+‖y‖
                │
考虑投影π₁: Graph(T) → X
π₁(x,Tx) = x
                │
π₁是双射(因T是函数)
π₁连续且‖π₁‖ ≤ 1
                │
由开映射定理: π₁⁻¹连续
                │
⟹ 存在C>0: ‖x‖+‖Tx‖ ≤ C‖x‖
                │
⟹ ‖Tx‖ ≤ (C-1)‖x‖
                │
T有界 ∎

```

### 4.5 重要推论

| 推论 | 内容 |
|------|------|
| 有界逆定理 | 双射T有界⟹T⁻¹有界 |
| 等价范数 | 同一向量空间上完备范数等价 |
| 商空间 | X/M是Banach空间(M闭子空间) |

## 💻 六、技术实现

### 6.1 Lean 4 形式化

```lean
-- 泛函分析三大定理形式化（Mathlib风格）
import Mathlib.Analysis.NormedSpace.HahnBanach
import Mathlib.Analysis.NormedSpace.OperatorNorm

-- Hahn-Banach定理
theorem hahn_banach {E : Type*} [NormedAddCommGroup E] [NormedSpace ℝ E]
    {p : Submodule ℝ E} {f : p →ₗ[ℝ] ℝ} (hf : ∀ x : p, f x ≤ ‖(x : E)‖) :
    ∃ F : E →ₗ[ℝ] ℝ, (∀ x : p, F x = f x) ∧ (∀ x : E, F x ≤ ‖x‖) := by
  exact exists_extension_norm_le p f hf

-- 一致有界原理
theorem uniform_boundedness {E F : Type*} [NormedAddCommGroup E]
    [NormedSpace ℝ E] [CompleteSpace E]
    [NormedAddCommGroup F] [NormedSpace ℝ F]
    {T : ι → E →L[ℝ] F} (h : ∀ x, ∃ C, ∀ i, ‖T i x‖ ≤ C) :
    ∃ C, ∀ i, ‖T i‖ ≤ C := by
  exact BoundedLinearMap.isBoundedBilinear_norm_bound_of_pointwise_bound T h

-- 开映射定理
theorem open_mapping {E F : Type*} [NormedAddCommGroup E]
    [NormedSpace ℝ E] [CompleteSpace E]
    [NormedAddCommGroup F] [NormedSpace ℝ F] [CompleteSpace F]
    {T : E →L[ℝ] F} (hT : Function.Surjective T) :
    IsOpenMap T := by
  exact T.isOpenMap_of_surjective hT

```

### 6.2 Python 示例

```python
import numpy as np
from scipy.linalg import norm

def demonstrate_uniform_boundedness():
    """
    演示一致有界原理
    构造算子序列，验证逐点有界⟹一致有界
    """
    # 在有限维空间ℝⁿ上演示
    n = 10

    # 构造有界算子序列Tₖ (投影到前k个坐标)
    def T(k, x):
        result = np.zeros_like(x)
        result[:min(k,len(x))] = x[:min(k,len(x))]
        return result

    # 检验逐点有界
    x = np.random.randn(n)
    for k in range(1, n+1):
        print(f"‖T_{k}x‖ = {norm(T(k, x)):.4f}")

    # 算子范数(对于投影算子，范数为1)
    print("\n算子范数均为1，验证一致有界原理")

demonstrate_uniform_boundedness()

```
