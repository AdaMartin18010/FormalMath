# 泛函分析习题补充

泛函分析研究无穷维向量空间上的分析结构，是量子力学、偏微分方程、概率论与逼近论的数学基础。本节通过精选习题展示 Banach 空间、Hilbert 空间、算子理论与谱分析的核心技巧。

## 1. Banach 空间

### 1.1 $\ell^p$ 空间的完备性

**习题 1.1**. 证明 $\ell^p$（$1 \leq p < \infty$）是 Banach 空间。

**证明**. 设 $\{x^{(n)}\}$ 为 $\ell^p$ 中的 Cauchy 列，$x^{(n)} = (x^{(n)}_1, x^{(n)}_2, \dots)$。对每个 $k$，$\{x^{(n)}_k\}_n$ 为 $\mathbb{C}$ 中的 Cauchy 列（因 $|x^{(n)}_k - x^{(m)}_k| \leq \|x^{(n)} - x^{(m)}\|_p$），故收敛到某 $x_k$。

令 $x = (x_1, x_2, \dots)$。对任意 $\epsilon > 0$，存在 $N$ 使得 $n, m \geq N$ 时：

$$\sum_{k=1}^K |x^{(n)}_k - x^{(m)}_k|^p \leq \epsilon^p, \quad \forall K.$$

令 $m \to \infty$，$\sum_{k=1}^K |x^{(n)}_k - x_k|^p \leq \epsilon^p$。再令 $K \to \infty$，$\|x^{(n)} - x\|_p \leq \epsilon$。故 $x^{(n)} - x \in \ell^p$，从而 $x \in \ell^p$，且 $x^{(n)} \to x$。$\square$

### 1.2 Hölder 与 Minkowski 不等式

**习题 1.2**. 证明 Hölder 不等式：设 $p, q > 1$，$\frac{1}{p} + \frac{1}{q} = 1$，则对 $x \in \ell^p$，$y \in \ell^q$：

$$\sum_{n=1}^{\infty} |x_n y_n| \leq \|x\|_p \|y\|_q.$$

**证明**. 先证 Young 不等式：对 $a, b \geq 0$，$ab \leq \frac{a^p}{p} + \frac{b^q}{q}$。由 $\ln$ 的凹性：

$$\ln\left(\frac{a^p}{p} + \frac{b^q}{q}\right) \geq \frac{1}{p}\ln(a^p) + \frac{1}{q}\ln(b^q) = \ln(ab).$$

对 $|x_n|/\|x\|_p$ 和 $|y_n|/\|y\|_q$ 应用 Young：

$$\frac{|x_n y_n|}{\|x\|_p \|y\|_q} \leq \frac{1}{p}\frac{|x_n|^p}{\|x\|_p^p} + \frac{1}{q}\frac{|y_n|^q}{\|y\|_q^q}.$$

对 $n$ 求和，右边 $= \frac{1}{p} + \frac{1}{q} = 1$。$\square$

## 2. Hilbert 空间

### 2.1 正交分解

**习题 2.1**. 设 $H$ 为 Hilbert 空间，$M \subseteq H$ 为闭子空间。证明 $H = M \oplus M^\perp$。

**证明**. $M \cap M^\perp = \{0\}$ 显然。对任意 $x \in H$，考虑极小化问题：求 $m \in M$ 使 $\|x - m\| = \inf_{y \in M} \|x - y\|$。

由平行四边形法则，极小化序列 $\{y_n\}$ 为 Cauchy 列：

$$\|y_n - y_m\|^2 = 2\|y_n - x\|^2 + 2\|y_m - x\|^2 - 4\left\|\frac{y_n + y_m}{2} - x\right\|^2 \to 0.$$

因 $M$ 闭，$y_n \to m \in M$。对任意 $z \in M$，$t \in \mathbb{R}$：

$$\|x - m\|^2 \leq \|x - m - tz\|^2 = \|x - m\|^2 - 2t\operatorname{Re}\langle x - m, z \rangle + t^2\|z\|^2.$$

故 $\operatorname{Re}\langle x - m, z \rangle = 0$。同理 $\operatorname{Im}$ 部分，$x - m \in M^\perp$。$\square$

### 2.2 Riesz 表示定理

**习题 2.2**. 设 $H$ 为 Hilbert 空间。对任意连续线性泛函 $f \in H^*$，存在唯一的 $y \in H$ 使 $f(x) = \langle x, y \rangle$ 对所有 $x \in H$，且 $\|f\| = \|y\|$。

**证明**. 若 $f = 0$，取 $y = 0$。否则，$\ker(f)$ 为闭真子空间，取非零 $z \in \ker(f)^\perp$。对任意 $x \in H$：

$$x - \frac{f(x)}{f(z)} z \in \ker(f),$$

故 $\langle x - \frac{f(x)}{f(z)}z, z \rangle = 0$，即 $f(x) = \frac{f(z)}{\|z\|^2} \langle x, z \rangle$。令 $y = \frac{\overline{f(z)}}{\|z\|^2} z$。$\square$

## 3. 算子理论

### 3.1 有界线性算子

**习题 3.1**. 证明有界线性算子 $T: X \to Y$ 的范数：

$$\|T\| = \sup_{\|x\|=1} \|Tx\| = \sup_{x \neq 0} \frac{\|Tx\|}{\|x\|} = \inf\{ M : \|Tx\| \leq M\|x\|, \forall x \}.$$

**证明**. 由齐次性，$\|Tx\| = \|x\| \cdot \|T(x/\|x\|)\|$，故 $\sup_{x \neq 0} \frac{\|Tx\|}{\|x\|} = \sup_{\|x\|=1} \|Tx\|$。设 $M_0 = \inf\{M : \|Tx\| \leq M\|x\|\}$。显然 $M_0 \geq \sup_{\|x\|=1}\|Tx\|$。反之，$\|Tx\| \leq \sup_{\|x\|=1}\|Tx\| \cdot \|x\|$，故 $M_0 \leq \sup_{\|x\|=1}\|Tx\|$。$\square$

### 3.2 开映射定理

**习题 3.2（开映射定理）**. 设 $T: X \to Y$ 为 Banach 空间之间的满射有界线性算子。证明 $T$ 为开映射。

**证明概要**. 先证 $T(B_X)$ 的闭包含某球 $B_Y(0, \delta)$。由 Baire 纲定理，$Y = \bigcup_n n\overline{T(B_X)}$，某 $n\overline{T(B_X)}$ 有内点，故 $\overline{T(B_X)}$ 含球。

再用尺度论证：对 $y \in B_Y(0, \delta)$，存在 $x_1 \in B_X(0, 1/2)$ 使 $\|y - Tx_1\| < \delta/2$。归纳得 $x_n \in B_X(0, 2^{-n})$ 使 $\|y - T(x_1 + \dots + x_n)\| < \delta 2^{-n}$。令 $x = \sum x_n$，$\|x\| < 1$ 且 $Tx = y$。故 $B_Y(0, \delta) \subseteq T(B_X)$，$T$ 开。$\square$

## 4. 谱理论

### 4.1 自伴算子的谱

**习题 4.1**. 设 $T$ 为 Hilbert 空间上的自伴算子（$T = T^*$）。证明 $\sigma(T) \subseteq \mathbb{R}$。

**证明**. 设 $\lambda = a + bi \in \mathbb{C}$，$b \neq 0$。对任意 $x$：

$$\|(T - \lambda I)x\|^2 = \langle (T - aI)x - bix, (T - aI)x - bix \rangle$$

$$= \|(T - aI)x\|^2 + b^2\|x\|^2 \geq b^2\|x\|^2.$$

故 $T - \lambda I$ 下有界，单射且有闭值域。类似证 $(T - \lambda I)^* = T - \bar{\lambda}I$ 单射，故 $T - \lambda I$ 满射，从而可逆。$\square$

## 5. 例子

### 5.1 例子：有限维空间范数等价

**习题 5.1**. 证明有限维赋范空间上的所有范数等价。

**证明**. 设 $X$ 为 $n$ 维空间，基 $\{e_1, \dots, e_n\}$。定义 $\|x\|_2 = (\sum |\alpha_i|^2)^{1/2}$（$x = \sum \alpha_i e_i$）。对任意范数 $\|\cdot\|$：

$$\|x\| \leq \sum |\alpha_i| \|e_i\| \leq \left(\sum \|e_i\|^2\right)^{1/2} \|x\|_2 = C\|x\|_2.$$

故 $\|\cdot\|$ 关于 $\|\cdot\|_2$ 连续。单位球面 $S = \{x : \|x\|_2 = 1\}$ 紧致，$\|\cdot\|$ 在 $S$ 上达到正最小值 $c > 0$。故 $\|x\| \geq c\|x\|_2$。$\square$

### 5.2 例子：紧算子的谱

**习题 5.2**. 证明 Hilbert 空间上紧自伴算子的非零谱由特征值组成，且仅能以 $0$ 为聚点。

**证明**. 由谱定理（紧算子版本），$T = \sum_n \lambda_n \langle \cdot, e_n \rangle e_n$，其中 $\{e_n\}$ 为标准正交系，$\lambda_n \to 0$。对 $\mu \neq 0, \lambda_n$，$(T - \mu I)^{-1} = -\frac{1}{\mu}I + \sum_n \frac{\lambda_n}{\mu(\lambda_n - \mu)}\langle \cdot, e_n \rangle e_n$ 有界，故 $\mu \in \rho(T)$。$\square$
