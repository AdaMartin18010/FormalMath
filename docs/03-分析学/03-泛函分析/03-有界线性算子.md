# 有界线性算子

## 1. 引言

在泛函分析中，线性算子是研究无穷维空间之间映射的核心对象。与有限维情形中所有线性算子自动连续不同，无穷维空间中存在大量不连续的线性算子。本文聚焦于**有界**（即连续）线性算子，系统阐述算子范数、有界线性算子空间 $\mathcal{B}(X,Y)$ 的结构，以及泛函分析三大基本定理之一——开映射定理。这些理论构成了算子谱分析、紧算子理论以及偏微分方程算子方法的基础。

## 2. 有界线性算子的基本概念

### 2.1 定义与等价刻画

**定义 2.1**。设 $X, Y$ 为赋范线性空间。线性映射 $T: X \to Y$ 称为**有界**的，若存在 $M \geq 0$ 使得
$$\|Tx\| \leq M\|x\|, \quad \forall x \in X.$$

**命题 2.2**（有界性与连续性的等价）。设 $T: X \to Y$ 为线性映射，则以下条件等价：

1. $T$ 有界；
2. $T$ 连续；
3. $T$ 在某一点连续；
4. $T$ 将有界集映为有界集。

*证明*。(1) $\Rightarrow$ (2)：设 $x_n \to x$，则 $\|Tx_n - Tx\| = \|T(x_n - x)\| \leq M\|x_n - x\| \to 0$。

(2) $\Rightarrow$ (3)：显然。

(3) $\Rightarrow$ (1)：设 $T$ 在 $x_0$ 连续。存在 $\delta > 0$ 使得当 $\|x\| \leq \delta$ 时，$\|T(x_0 + x) - Tx_0\| = \|Tx\| \leq 1$。对任意 $y \neq 0$，$\|\frac{\delta y}{\|y\|}\| = \delta$，故 $\|T(\frac{\delta y}{\|y\|})\| \leq 1$，即 $\|Ty\| \leq \frac{1}{\delta}\|y\|$。

(1) $\Leftrightarrow$ (4)：线性算子将有界集映为有界集当且仅当单位球 $B_X = \{x : \|x\| \leq 1\}$ 的像有界，即 $\sup_{\|x\| \leq 1} \|Tx\| < \infty$。$\square$

### 2.2 算子范数

**定义 2.3**。有界线性算子 $T: X \to Y$ 的**算子范数**定义为
$$\|T\| = \sup_{x \neq 0} \frac{\|Tx\|}{\|x\|} = \sup_{\|x\| = 1} \|Tx\| = \sup_{\|x\| \leq 1} \|Tx\|.$$

**命题 2.4**。$\|\cdot\|$ 确实构成范数，且满足 $\|Tx\| \leq \|T\| \|x\|$ 对所有 $x \in X$ 成立。此外，$\|ST\| \leq \|S\| \|T\|$（当复合有意义时），称此不等式为次可乘性。

## 3. 有界线性算子空间 $\mathcal{B}(X,Y)$

### 3.1 空间结构

**定义 3.1**。记 $\mathcal{B}(X,Y)$（或 $\mathcal{L}(X,Y)$）为从 $X$ 到 $Y$ 的全体有界线性算子构成的线性空间，配备算子范数。

**定理 3.2**。若 $Y$ 为Banach空间，则 $\mathcal{B}(X,Y)$ 也是Banach空间（不论 $X$ 是否完备）。

*证明*。设 $(T_n)$ 为 $\mathcal{B}(X,Y)$ 中的Cauchy列。对每个 $x \in X$，$\|T_n x - T_m x\| \leq \|T_n - T_m\| \|x\| \to 0$，故 $(T_n x)$ 是 $Y$ 中的Cauchy列。由 $Y$ 完备，定义 $Tx = \lim_{n\to\infty} T_n x$。

验证 $T$ 线性：$T(\alpha x + \beta y) = \lim T_n(\alpha x + \beta y) = \alpha \lim T_n x + \beta \lim T_n y = \alpha Tx + \beta Ty$。

验证 $T$ 有界：由Cauchy条件，存在 $K$ 使得 $\|T_n\| \leq K$ 对所有 $n$ 成立（Cauchy列有界）。则 $\|Tx\| = \lim \|T_n x\| \leq K\|x\|$，故 $\|T\| \leq K$。

验证 $T_n \to T$：对 $\varepsilon > 0$，存在 $N$ 使得当 $n, m \geq N$ 时，$\|T_n - T_m\| < \varepsilon$。则对 $\|x\| \leq 1$，$\|T_n x - T_m x\| < \varepsilon$。令 $m \to \infty$，得 $\|T_n x - Tx\| \leq \varepsilon$，即 $\|T_n - T\| \leq \varepsilon$。$\square$

### 3.2 对偶空间

**定义 3.3**。$X^* = \mathcal{B}(X, \mathbb{K})$ 称为 $X$ 的**对偶空间**（或共轭空间），其中的元素称为**连续线性泛函**。

由定理3.2，$X^*$ 总是Banach空间（因 $\mathbb{K}$ 完备）。

**例 3.4**（$l^p$ 的对偶）。对 $1 \leq p < \infty$，设 $q$ 为 $p$ 的共轭指数（$\frac{1}{p} + \frac{1}{q} = 1$）。则 $(l^p)^* \cong l^q$，等距同构由
$$f_y(x) = \sum_{n=1}^\infty x_n y_n, \quad y \in l^q$$
给出，且 $\|f_y\| = \|y\|_q$。

*证明概要*。由Hölder不等式，$|f_y(x)| \leq \|x\|_p \|y\|_q$，故 $\|f_y\| \leq \|y\|_q$。对有限支撑序列取 $x_n = |y_n|^{q-1} \operatorname{sgn}(y_n) / \|y\|_q^{q/p}$ 可证等号。$\square$

**例 3.5**（$L^p$ 的对偶）。对 $1 \leq p < \infty$，$(L^p(\Omega, \mu))^* \cong L^q(\Omega, \mu)$，同构由
$$f_g(\varphi) = \int_\Omega \varphi \bar{g} \, d\mu$$
给出。$L^\infty$ 和 $l^\infty$ 的对偶更复杂，包含绝对连续有界可加集函数（Yosida-Hewitt分解）。

### 3.3 自反空间

**定义 3.6**。存在自然嵌入 $J: X \to X^{**}$，$(Jx)(f) = f(x)$。若 $J$ 是满射，称 $X$ 为**自反的**（reflexive）。

**定理 3.7**。$l^p$ 和 $L^p$（$1 < p < \infty$）自反。$l^1$、$L^1$、$l^\infty$、$L^\infty$、$C[0,1]$ 不自反。

## 4. 一致有界原理（共鸣定理）

**定理 4.1**（一致有界原理/Banach-Steinhaus）。设 $X$ 为Banach空间，$Y$ 为赋范空间，$\{T_\alpha\}_{\alpha \in I} \subseteq \mathcal{B}(X,Y)$。若对每个 $x \in X$，$\sup_{\alpha \in I} \|T_\alpha x\| < \infty$，则
$$\sup_{\alpha \in I} \|T_\alpha\| < \infty.$$

*证明*。设 $E_n = \{x \in X : \sup_\alpha \|T_\alpha x\| \leq n\} = \bigcap_\alpha \{x : \|T_\alpha x\| \leq n\}$。每个 $\{x : \|T_\alpha x\| \leq n\}$ 闭（因 $T_\alpha$ 连续），故 $E_n$ 闭。由假设，$X = \bigcup_{n=1}^\infty E_n$。由Baire纲定理（$X$ 完备），某 $E_{n_0}$ 含内点，即存在 $x_0$ 和 $r > 0$ 使得 $B(x_0, r) \subseteq E_{n_0}$。

对任意 $\|x\| \leq r$，有 $x_0 + x \in B(x_0, r)$，故 $\|T_\alpha(x_0 + x)\| \leq n_0$。于是
$$\|T_\alpha x\| \leq \|T_\alpha(x_0 + x)\| + \|T_\alpha x_0\| \leq 2n_0.$$
对任意 $y \neq 0$，$\|T_\alpha y\| = \frac{\|y\|}{r} \|T_\alpha(\frac{ry}{\|y\|})\| \leq \frac{2n_0}{r} \|y\|$。故 $\|T_\alpha\| \leq \frac{2n_0}{r}$。$\square$

**推论 4.2**。设 $X$ 为Banach空间，$Y$ 赋范，$(T_n) \subseteq \mathcal{B}(X,Y)$。若对每个 $x \in X$，$\lim_{n\to\infty} T_n x$ 存在，则 $Tx = \lim T_n x$ 定义了一个有界线性算子，且 $\|T\| \leq \liminf \|T_n\|$。

## 5. 开映射定理

### 5.1 开映射与闭图像

**定义 5.1**。映射 $f: X \to Y$ 称为**开的**，若它将开集映为开集。线性映射 $T$ 是开映射当且仅当 $T(B_X)$ 包含 $Y$ 中0的某邻域。

**定理 5.2**（开映射定理/Banach-Schauder）。设 $X, Y$ 为Banach空间，$T \in \mathcal{B}(X,Y)$ 为满射。则 $T$ 是开映射。

*证明*。分三步：

**第一步**：证明 $\overline{T(B_X)}$ 包含0的邻域。令 $B_n = B_X(0, n)$。由 $T$ 满射，$Y = \bigcup_{n=1}^\infty T(B_n) = \bigcup_{n=1}^\infty \overline{T(B_n)}$。由Baire纲定理，某 $\overline{T(B_{n_0})}$ 有内点。由对称性，$0$ 是 $\overline{T(B_{n_0})}$ 的内点（若 $y$ 为内点，则 $-y$ 也是，由凸性 $0 = \frac{y + (-y)}{2}$ 也是）。故存在 $\delta > 0$ 使 $B_Y(0, \delta) \subseteq \overline{T(B_{n_0})} = n_0 \overline{T(B_X)}$，即 $B_Y(0, \delta/n_0) \subseteq \overline{T(B_X)}$。

**第二步**：证明 $T(B_X)$ 包含0的邻域。设 $y \in B_Y(0, \delta/2n_0)$。由第一步，存在 $x_1 \in B_X(0, 1/2)$ 使 $\|y - Tx_1\| < \delta/4n_0$。归纳地，取 $x_n \in B_X(0, 1/2^n)$ 使 $\|y - \sum_{k=1}^n Tx_k\| < \delta/2^{n+1}n_0$。令 $x = \sum_{k=1}^\infty x_k$，则 $\|x\| \leq \sum_{k=1}^\infty 2^{-k} = 1$，且 $Tx = y$。故 $B_Y(0, \delta/2n_0) \subseteq T(B_X)$。

**第三步**：设 $U \subseteq X$ 开，$x \in U$。存在 $r > 0$ 使 $B_X(x, r) \subseteq U$。则 $T(B_X(x,r)) = Tx + rT(B_X) \supseteq Tx + rB_Y(0, \delta/2n_0) = B_Y(Tx, r\delta/2n_0)$。故 $T(U)$ 开。$\square$

**推论 5.3**（逆算子定理）。设 $X, Y$ 为Banach空间，$T \in \mathcal{B}(X,Y)$ 为双射。则 $T^{-1} \in \mathcal{B}(Y,X)$。

*证明*。$T^{-1}$ 线性显然。$T$ 为开映射意味着 $(T^{-1})^{-1}(U) = T(U)$ 开，故 $T^{-1}$ 连续。$\square$

### 5.2 范数等价性

**推论 5.4**。设 $X$ 为线性空间，$\|\cdot\|_1$ 和 $\|\cdot\|_2$ 均为Banach空间范数。若存在 $C > 0$ 使 $\|x\|_2 \leq C\|x\|_1$（即恒等映射 $I: (X, \|\cdot\|_1) \to (X, \|\cdot\|_2)$ 连续），则两范数等价。

*证明*。由逆算子定理，$I^{-1}$ 也连续。$\square$

## 6. 闭图像定理

**定义 6.1**。线性映射 $T: X \to Y$ 的**图像**为
$$\mathcal{G}(T) = \{(x, Tx) : x \in X\} \subseteq X \times Y.$$
称 $T$ 有**闭图像**，若 $\mathcal{G}(T)$ 是 $X \times Y$ 的闭子空间（装备范数 $\|(x,y)\| = \|x\| + \|y\|$ 或 $\max\{\|x\|, \|y\|\}$）。

**定理 6.2**（闭图像定理）。设 $X, Y$ 为Banach空间，$T: X \to Y$ 为线性算子。若 $T$ 有闭图像，则 $T$ 有界（连续）。

*证明*。$\mathcal{G}(T)$ 是Banach空间 $X \times Y$ 的闭子空间，故自身是Banach空间。投影 $\pi_1: \mathcal{G}(T) \to X$，$\pi_1(x, Tx) = x$ 是有界双射线性算子。由逆算子定理，$\pi_1^{-1}: X \to \mathcal{G}(T)$，$\pi_1^{-1}(x) = (x, Tx)$ 有界。故 $\|Tx\| \leq \|\pi_1^{-1}\| \|x\|$，$T$ 有界。$\square$

**注记 6.3**。闭图像定理的实用价值在于：要证明线性算子 $T$ 有界，只需验证若 $x_n \to x$ 且 $Tx_n \to y$，则 $y = Tx$（不必预先知道 $Tx_n \to Tx$）。这在验证微分算子等无先验连续性信息的算子时特别有用。

## 7. 紧算子初步

**定义 7.1**。设 $X, Y$ 为Banach空间，$T \in \mathcal{B}(X,Y)$ 称为**紧算子**（compact operator），若 $T$ 将 $X$ 中的有界集映为 $Y$ 中的相对紧集（闭包紧）。等价地，$\overline{T(B_X)}$ 在 $Y$ 中紧。

**命题 7.2**。全体紧算子 $\mathcal{K}(X,Y)$ 构成 $\mathcal{B}(X,Y)$ 的闭子空间。若 $T$ 紧且 $S$ 有界，则 $ST$ 和 $TS$（复合有意义时）紧。

**例 7.3**（有限秩算子）。值域为有限维的算子称为**有限秩算子**。有限秩算子必紧（因有限维空间中有界闭集列紧）。在Hilbert空间中，紧算子可表为有限秩算子的一致极限。

**例 7.4**（积分算子）。设 $K \in C([0,1] \times [0,1])$，定义
$$(Tf)(x) = \int_0^1 K(x,y) f(y) \, dy.$$
则 $T: C[0,1] \to C[0,1]$ 是紧算子（由Arzelà-Ascoli定理）。

## 8. 与项目其他部分的关联

有界线性算子的理论是泛函分析的主干，与谱理论（见[04-谱理论基础](04-谱理论基础.md)）和紧算子理论（见[05-紧算子](05-紧算子.md)）直接衔接。开映射定理、闭图像定理和一致有界原理并称为"泛函分析三大定理"，它们深刻依赖于空间的完备性。在分布论（见[09-分布论入门](09-分布论入门.md)）中，微分算子作为无界算子的研究也离不开这些框架。对偶空间的理论在Sobolev空间 $W^{-k,q}$（对偶于 $W_0^{k,p}$）的研究中至关重要（见[10-Sobolev空间](10-Sobolev空间.md)）。
