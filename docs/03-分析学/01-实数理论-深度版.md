## 📚 概述

实数理论是现代分析学的基石，为微积分提供了严格的逻辑基础。本文档从构造主义角度深入探讨实数的三种经典构造方法（Dedekind分割、Cauchy序列、十进制展开），详细阐述实数公理系统，并给出完备性六大等价定理的完整证明。

## 🏗️ 实数构造理论

### Dedekind分割构造

**定义 1.1** (Dedekind分割)

有理数集 $\mathbb{Q}$ 的一个分割 $(A, B)$ 满足：

1. $A \neq \emptyset$, $B \neq \emptyset$
2. $A \cup B = \mathbb{Q}$
3. $\forall a \in A, \forall b \in B: a < b$
4. $A$ 中无最大元

**定理 1.1** (实数完备性)

Dedekind分割的集合 $R$ 构成完备有序域。

**证明**：

*有序性*：定义 $(A_1, B_1) \leq (A_2, B_2)$ 当且仅当 $A_1 \subseteq A_2$

*域运算*：
- 加法：$A_1 + A_2 = \{a_1 + a_2 : a_1 \in A_1, a_2 \in A_2\}$
- 乘法：对正分割定义 $A_1 \cdot A_2 = \{a_1 \cdot a_2 : a_1 \in A_1, a_2 \in A_2, a_1 > 0, a_2 > 0\} \cup \{q \in \mathbb{Q} : q \leq 0\}$

*完备性*：设 $S \subseteq R$ 有上界，令 $A = \bigcup_{(A_i, B_i) \in S} A_i$，则 $(A, \mathbb{Q} \setminus A)$ 为最小上界。

```lean
theorem dedekind_complete (S : set ℝ) (hne : S.nonempty) (hbdd : bdd_above S) :
  ∃ x : ℝ, is_lub S x :=
begin
  let A := ⋃ y ∈ S, {q : ℚ | q < y},
  let B := ℚ \ A,
  use mk (A, B),
  split,
  { -- 证明是上界
    intros y hy q hq,
    exact subset_Union₂ q y hy hq },
  { -- 证明是最小上界
    intros z hz q hq,
    obtain ⟨y, hy, hqy⟩ := mem_Union₂.mp hq,
    exact lt_of_lt_of_le hqy (hz y hy) }
end
```

### Cauchy序列构造

**定义 1.2** (Cauchy序列)

有理数序列 $(q_n)$ 称为Cauchy序列，如果：

$$\forall \varepsilon > 0, \exists N \in \mathbb{N}, \forall m, n \geq N: |q_m - q_n| < \varepsilon$$

**定义 1.3** (序列等价)

两个Cauchy序列 $(q_n) \sim (r_n)$ 当且仅当：

$$\lim_{n \to \infty} |q_n - r_n| = 0$$

**定理 1.2** (实数作为等价类)

实数 $\mathbb{R}$ 定义为Cauchy序列等价类的集合，构成完备有序域。

**证明要点**：

1. *良定义性*：验证等价关系保持运算
2. *域结构*：分量定义加法和乘法
3. *完备性*：Cauchy序列的极限仍为Cauchy序列

```lean
def cauchy_equiv (x y : ℕ → ℚ) : Prop :=
  ∀ ε > 0, ∃ N, ∀ n ≥ N, |x n - y n| < ε

theorem cauchy_equiv_is_equivalence : equivalence cauchy_equiv :=
⟨λ x ε hε, by simp [hε.lt_iff_ne],
 λ x y hxy ε hε, by simpa using hxy ε hε,
 λ x y z hxy hyz ε hε, begin
   obtain ⟨N₁, h₁⟩ := hxy (ε/2) (by linarith),
   obtain ⟨N₂, h₂⟩ := hyz (ε/2) (by linarith),
   use max N₁ N₂,
   intros n hn,
   have : |x n - z n| ≤ |x n - y n| + |y n - z n| := abs_sub_le _ _ _,
   linarith [h₁ n (le_of_max_le_left hn), h₂ n (le_of_max_le_right hn)]
 end⟩
```

### 十进制展开构造

**定义 1.4** (十进制展开)

实数由整数部分 $a_0 \in \mathbb{Z}$ 和小数部分 $0.a_1a_2a_3...$ 表示，其中 $a_i \in \{0,1,...,9\}$

**规范形式**：不允许从某一位起全为9（避免 $0.999... = 1$ 的双重表示）

**定理 1.3** (十进制展开的完备性)

十进制展开与Dedekind分割构造的实数存在保序同构。

## 📐 实数公理系统

### 域公理

$(\mathbb{R}, +, \cdot)$ 满足：

| 公理 | 加法 | 乘法 |
|------|------|------|
| 结合律 | $(a+b)+c = a+(b+c)$ | $(ab)c = a(bc)$ |
| 交换律 | $a+b = b+a$ | $ab = ba$ |
| 单位元 | $a+0 = a$ | $a \cdot 1 = a$ |
| 逆元 | $a+(-a) = 0$ | $a \cdot a^{-1} = 1$ (a≠0) |
| 分配律 | $a(b+c) = ab + ac$ | — |

### 序公理

$< $ 关系满足：

1. **三歧性**：$\forall a,b: a < b \lor a = b \lor b < a$
2. **传递性**：$a < b \land b < c \Rightarrow a < c$
3. **加法保序**：$a < b \Rightarrow a + c < b + c$
4. **乘法保序**：$a < b \land c > 0 \Rightarrow ac < bc$

### 完备性公理

**最小上界原理**：非空有上界的集合必有最小上界。

$$\forall S \subseteq \mathbb{R}, S \neq \emptyset, \exists M: \forall x \in S, x \leq M \Rightarrow \exists \sup S \in \mathbb{R}$$

## 🔬 核心定理与证明

### 确界原理

**定理 2.1** (确界原理)

非空有上界的实数集必有上确界。

**证明**：

设 $S \subseteq \mathbb{R}$ 非空有上界。构造：

$$A = \{q \in \mathbb{Q} : \exists s \in S, q < s\}$$
$$B = \mathbb{Q} \setminus A$$

则 $(A, B)$ 构成Dedekind分割，且 $\sup S = (A, B)$。

---

### 区间套定理

**定理 2.2** (区间套定理 / Cantor区间套原理)

设 $[a_1, b_1] \supseteq [a_2, b_2] \supseteq ...$ 为闭区间套，且 $\lim_{n \to \infty} (b_n - a_n) = 0$，则存在唯一的 $c \in \bigcap_{n=1}^{\infty} [a_n, b_n]$。

**证明**：

1. $\{a_n\}$ 单调递增有上界，$\{b_n\}$ 单调递减有下界
2. 由单调有界定理，$\exists \lim a_n = a$, $\exists \lim b_n = b$
3. 由 $b_n - a_n \to 0$，得 $a = b = c$
4. 唯一性：若 $c_1, c_2 \in \bigcap [a_n, b_n]$，则 $|c_1 - c_2| \leq b_n - a_n \to 0$

---

### Bolzano-Weierstrass定理

**定理 2.3** (Bolzano-Weierstrass定理)

有界无限实数集必有聚点。

**证明**（二分法）：

设 $S \subseteq [a, b]$ 无限。重复二分区间，选取含无限多点的子区间：

$$[a_0, b_0] = [a, b]$$
$$[a_{n+1}, b_{n+1}] = \begin{cases} [a_n, \frac{a_n+b_n}{2}] & S \cap [a_n, \frac{a_n+b_n}{2}] \text{ 无限} \\ [\frac{a_n+b_n}{2}, b_n] & \text{否则} \end{cases}$$

由区间套定理，$\exists! c \in \bigcap [a_n, b_n]$，此 $c$ 即为聚点。

---

### Heine-Borel定理

**定理 2.4** (Heine-Borel定理)

$[a, b]$ 是紧集（任意开覆盖有有限子覆盖）。

**证明**（反证法）：

假设存在无有限子覆盖的开覆盖 $\mathcal{U}$。二分法构造区间套 $[a_n, b_n]$，每个都不可有限覆盖。

由区间套定理，$\exists c \in \bigcap [a_n, b_n]$。因 $\mathcal{U}$ 覆盖 $[a,b]$，$\exists U \in \mathcal{U}, c \in U$。

$U$ 开，$\exists \delta > 0: (c-\delta, c+\delta) \subseteq U$。当 $n$ 足够大，$[a_n, b_n] \subseteq (c-\delta, c+\delta) \subseteq U$，矛盾。

## 🔄 等价性证明

### 完备性的六种等价形式

**定理 3.1** 下列命题等价：

1. **确界原理**：非空有上界集有上确界
2. **单调有界定理**：单调有界序列收敛
3. **区间套定理**：长度趋于0的闭区间套有唯一公共点
4. **Bolzano-Weierstrass定理**：有界无限集有聚点
5. **Cauchy收敛准则**：Cauchy序列收敛
6. **Heine-Borel定理**：闭区间是紧集

**证明循环**：$(1) \Rightarrow (2) \Rightarrow (3) \Rightarrow (4) \Rightarrow (5) \Rightarrow (6) \Rightarrow (1)$

**$(1) \Rightarrow (2)$**：
设 $(x_n)$ 单调递增有上界。令 $S = \{x_n : n \in \mathbb{N}\}$，则 $\sup S$ 存在。
$\forall \varepsilon > 0, \exists N: x_N > \sup S - \varepsilon$。由单调性，$\forall n \geq N: |x_n - \sup S| < \varepsilon$。

**$(5) \Rightarrow (1)$**：
设 $S$ 非空有上界。构造序列：$x_0 \in S$，$y_0$ 为上界。
$$x_{n+1} = \begin{cases} \frac{x_n+y_n}{2} & \text{若 } \frac{x_n+y_n}{2} \text{ 非上界} \\ x_n & \text{否则} \end{cases}$$
$$y_{n+1} = \begin{cases} y_n & \text{若 } \frac{x_n+y_n}{2} \text{ 非上界} \\ \frac{x_n+y_n}{2} & \text{否则} \end{cases}$$

则 $(x_n)$ 递增，$(y_n)$ 递减，$y_n - x_n \to 0$。$(x_n)$ 为Cauchy序列，收敛于 $\sup S$。

## 🧠 哲学与认知分析

### 连续统假设

**连续统假设 (CH)**：不存在基数严格介于 $|\mathbb{N}|$ 和 $|\mathbb{R}|$ 之间的集合。

**历史地位**：
- Hilbert第一问题（1900年）
- Gödel（1940年）：CH与ZFC一致
- Cohen（1963年）：CH的否定与ZFC一致
- **结论**：CH独立于ZFC公理系统

### 构造主义vs经典数学

| 观点 | 构造主义 | 经典数学 |
|------|----------|----------|
| 存在性 | 必须给出构造 | 反证法可证 |
| 排中律 | 限制使用 | 自由使用 |
| 实数定义 | 可计算序列 | 任意序列 |
| 代表性学者 | Brouwer, Bishop | Hilbert, Gödel |

## 💻 形式化实现

### Lean 4实现

```lean
import data.real.basic
import analysis.specific_limits

-- 实数完备性定理
namespace real_completeness

-- 确界原理
theorem supremum_principle (S : set ℝ) (hne : S.nonempty) (hbdd : bdd_above S) :
  ∃ x : ℝ, is_lub S x :=
real.exists_supremum_real hne hbdd

-- 单调有界定理
theorem monotone_convergence (f : ℕ → ℝ) (hf : monotone f) (hbdd : bdd_above (set.range f)) :
  ∃ L : ℝ, tendsto f at_top (𝓝 L) :=
begin
  obtain ⟨L, hL⟩ := real.exists_supremum_real (set.range_nonempty f) hbdd,
  use L,
  exact tendsto_at_top_is_lub hf hL,
end

-- Cauchy收敛准则
theorem cauchy_convergence (f : ℕ → ℝ) (hcauchy : cauchy_seq f) :
  ∃ L : ℝ, tendsto f at_top (𝓝 L) :=
complete_space.complete hcauchy

end real_completeness
```

### Isabelle/HOL实现

```isabelle
theory Real_Completeness
  imports Complex_Main
begin

(* 确界原理 *)
theorem Supremum_principle:
  assumes "S ≠ {}" and "bdd_above S"
  shows "Sup S ∈ S" ∨ "(∀x ∈ S. x ≤ Sup S) ∧ (∀y. (∀x ∈ S. x ≤ y) ⟶ Sup S ≤ y)"
  using assms by (auto simp: cSup_eq_maximum cSup_least)

(* 区间套定理 *)
theorem Nested_Intervals:
  assumes "⋀n. closed_segment (a n) (b n) ⊇ closed_segment (a (Suc n)) (b (Suc n))"
    and "(λn. b n - a n) ⇢ 0"
  shows "∃!c. ∀n. c ∈ closed_segment (a n) (b n)"
proof -
  have "(λn. b n - a n) ⇢ 0" using assms(2) .
  then obtain c where "a ⇢ c" "b ⇢ c" using assms(1) by (metis Lim_transform)
  then show ?thesis using assms by (auto simp: closed_segment_def)
qed

end
```
