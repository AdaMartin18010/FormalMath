# 随机分析基础

## Itô积分

### 定义
$$\int_0^t H_s dX_s = \lim_{\Delta \to 0} \sum H_{t_i}(X_{t_{i+1}} - X_{t_i})$$

### Itô公式
$$df(t, X_t) = \frac{\partial f}{\partial t}dt + \frac{\partial f}{\partial x}dX_t + \frac{1}{2}\frac{\partial^2 f}{\partial x^2}d[X]_t$$
