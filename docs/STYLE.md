# 文档风格规范

## 核心约定

- **语言**：简体中文为主，英文术语首次出现时给出
- **数学**：行间 `\[ ... \]`、行内 `\( ... \)`
- **标题**：`##` 为二级起，避免 `#` 过大
- **代码块**：使用 ```text 标注命令/脚本
- **文件引用**：用反引号 `filename.md`

## 数学记号

- 内积/范数默认欧氏；Lorentz 以内角标 L
- 齐次坐标列向量；点/直线对偶按 \(A, A^{-1}\)
- 变量/矩阵/群按正文统一（见主文 158 节）

## 导出与发布

- 使用 Pandoc 导出（见导出说明）
- 静态站点导航在 `mkdocs.yml` 中维护
