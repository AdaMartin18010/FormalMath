## 使用指南

### 如何查阅对齐文档

1. **按大学查阅**: 选择对应大学的Markdown文档
2. **按课程查阅**: 在文档内查看具体课程的对齐分析
3. **按概念查阅**: 通过概念映射表找到FormalMath对应文档

### 如何利用对齐结果

1. **内容创建优先级**: 根据"待补充概念清单"确定创建优先级
2. **学习路径规划**: 参考各大学的"建议学习路径"部分
3. **概念对比**: 通过"关键概念定义对比"了解不同体系的表述差异
