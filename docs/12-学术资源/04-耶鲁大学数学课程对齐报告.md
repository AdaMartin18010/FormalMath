## 一、耶鲁大学数学教育体系概述

### 1.1 机构背景

**耶鲁大学 (Yale University)** 是美国最古老的高等教育机构之一，其数学系有着悠久的历史和卓越的成就：

| 历史贡献 | 描述 | 代表人物 |
|---------|------|---------|
| **分析学** | 调和分析与泛函分析 | Antoni Zygmund, Raphy Coifman |
| **代数几何** | 现代代数几何发展 | Serge Lang, Oscar Zariski |
| **数论** | 解析数论与算术几何 | Serge Lang, Gregg Zuckerman |
| **动力系统** | 遍历理论与混沌 | Benjamin Weiss, Hillel Furstenberg |
| **逻辑** | 模型论与递归论 | Abraham Robinson, Angus Macintyre |
| **几何学** | 微分几何与辛几何 | F. Reese Harvey, H. Blaine Lawson |
| **表示论** | 李群与表示论 | Roger Howe, Gregg Zuckerman |

### 1.3 本科课程结构

Yale数学系提供灵活的课程体系：

| 类别 | 课程编号 | 内容描述 |
|------|----------|----------|
| **基础课程** | 112, 115, 120, 121 | 微积分、线性代数、多变量微积分 |
| **过渡课程** | 225, 255 | 实分析入门、抽象代数入门 |
| **核心课程** | 301, 310, 350, 370 | 实分析、复分析、抽象代数 |
| **专题课程** | 430-480 | 数论、几何、拓扑、概率等 |
