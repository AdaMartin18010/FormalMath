> 本清单系统整理了数学研究和教学中常用的软件工具，涵盖符号计算、数值计算、几何绘图和形式化证明等领域。

## 一、符号计算工具

### 1.1 Mathematica

| 属性 | 详情 |
|------|------|
| **开发商** | Wolfram Research |
| **官网** | www.wolfram.com/mathematica |
| **价格** | 商业软件（个人/教育版优惠） |
| **平台** | Windows, macOS, Linux |

#### 核心功能

- **符号计算**：任意精度计算、符号积分/微分
- **可视化**：2D/3D图形、动画、交互式图表
- **数据处理**：统计分析、图像处理
- **编程**：Wolfram语言，函数式编程
- **知识库**：集成Wolfram Knowledgebase

#### 代码示例

```mathematica
(* 符号积分 *)
Integrate[Sin[x]^2, x]

(* 求解方程 *)
Solve[x^2 - 3 x + 2 == 0, x]

(* 矩阵运算 *)
A = {{1, 2}, {3, 4}};
Eigenvalues[A]

(* 3D绘图 *)
Plot3D[Sin[x y], {x, -3, 3}, {y, -3, 3}]

(* 级数展开 *)
Series[Exp[x], {x, 0, 5}]
```

#### 适用场景

- 复杂符号计算
- 高质量可视化
- 教学演示
- 快速原型验证

#### 代码示例

```lean
-- 证明 sqrt(2) 是无理数
import Mathlib

theorem irrational_sqrt_two : Irrational (Real.sqrt 2) := by
  native_decide

-- 自然数归纳法
example (n : ℕ) : 2 * ∑ i in Finset.range n, i = n * (n - 1) := by
  induction n with
  | zero => simp
  | succ n ih =>
    rw [Finset.sum_range_succ, mul_add, ih]
    ring

-- 群论基础
import Mathlib.GroupTheory.Basic

example (G : Type*) [Group G] (a b : G) :
    (a * b)⁻¹ = b⁻¹ * a⁻¹ := by
  simp
```

#### 代码示例

```coq
(* 证明加法交换律 *)
Theorem plus_comm : forall n m : nat, n + m = m + n.
Proof.
  intros n m.
  induction n as [| n' IHn'].
  - simpl. rewrite <- plus_n_O. reflexivity.
  - simpl. rewrite IHn'. rewrite plus_n_Sm. reflexivity.
Qed.

(* 列表反转 *)
Fixpoint rev {X:Type} (l:list X) : list X :=
  match l with
  | nil => nil
  | cons h t => app (rev t) (cons h nil)
  end.
```
