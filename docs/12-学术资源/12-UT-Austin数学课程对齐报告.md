## 一、UT Austin数学系概况

**德克萨斯大学奥斯汀分校 (UT Austin)** 是美国顶尖的公立研究型大学，其数学系在拓扑学、数论和分析领域具有卓越声誉：

| 历史贡献 | 描述 | 代表人物 |
|---------|------|---------|
| 拓扑学 | 代数拓扑与几何拓扑研究 | Karen Uhlenbeck (Abel奖), Cameron Gordon |
| 动力系统 | 遍历理论与混沌理论 | Giovanni Forni, Lewis Bowen |
| 数论 | 代数数论与算术几何 | Fernando Rodriguez Villegas |
| 分析学 | 调和分析与PDE | William Beckner, Alessio Figalli |
| 计算数学 | 数值分析与科学计算 | Björn Engquist |

### 2.4 拓扑性质一览

```
分离公理层级:
T₀ (Kolmogorov) → T₁ → T₂ (Hausdorff) → T₂½ → T₃ (正则) → T₃½ → T₄ (正规)

紧性相关概念:
紧 → 序列紧 → 可数紧 → 伪紧 → 局部紧 → σ-紧 → Lindelöf

连通性相关概念:
连通 → 道路连通 → 局部连通 → 局部道路连通
```

### 3.4 理想分解定理

**Dedekind唯一分解定理**: 在Dedekind整环中，每个非零理想可唯一分解为素理想的乘积。

**分解模式**: 对于素数p在数域K中的分解：
- pO_K = 𝔭₁^{e₁}...𝔭_g^{e_g}
- Σeᵢfᵢ = [K:ℚ]
