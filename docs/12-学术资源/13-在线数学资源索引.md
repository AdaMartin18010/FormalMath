## 一、arXiv数学分类导航

### 1.1 arXiv概述

**arXiv** (https://arxiv.org) 是数学和理论科学领域最重要的预印本服务器，研究者可以在此免费获取最新的研究进展。

| 特性 | 详情 |
|------|------|
| **成立时间** | 1991年 |
| **覆盖领域** | 数学、物理、计算机科学等 |
| **投稿量** | 每月约15000篇 |
| **费用** | 完全免费 |
| **版权** | 作者保留版权 |

### 1.3 arXiv使用技巧

#### 搜索策略
```
基本搜索:
- 按分类浏览: arxiv.org/list/math.AG/recent
- 按作者搜索: "author:Deligne"
- 按标题搜索: "title:moduli space"

高级搜索:
- 组合条件: "math.AG AND moduli"
- 日期范围: submittedDate:[20230101 TO 20231231]
- 排除关键词: "math.GT NOT knot"
```

### 2.3 热门问题类型

#### 经典问题示例
```
1. 概念澄清类
   "What is the precise definition of X?"
   标签: specific-reference, definitions

2. 定理寻找类
   "Is there a theorem that says X?"
   标签: reference-request

3. 证明技术类
   "How to prove X without using Y?"
   标签: alternative-proof

4. 反例寻找类
   "Is there a counterexample to X?"
   标签: counterexamples

5. 历史背景类
   "What is the history of X?"
   标签: ho.history-overview
```

### 2.4 使用建议

#### 提问技巧
1. **提供背景**：说明你的研究背景
2. **展示尝试**：说明你已尝试的方法
3. **明确定义**：给出所有专业术语的定义
4. **使用LaTeX**：数学公式使用LaTeX

#### 搜索技巧
```
按标签搜索:
[tag]ag.algebraic-geometry[/tag] moduli space

按分数筛选:
score:10..  (10分以上的问题)

按时间筛选:
created:2023..  (2023年以后)

组合搜索:
[tag]nt.number-theory[/tag] is:question score:20..
```

## 三、zbMATH检索指南

### 3.1 zbMATH概述

**zbMATH** (https://zbmath.org) 是数学领域最权威、最全面的文献数据库。

| 特性 | 详情 |
|------|------|
| **历史** | 1931年创刊（Zentralblatt） |
| **覆盖** | 超过400万条文献记录 |
| **更新** | 每日更新 |
| **语言** | 支持多语言检索 |
| **索引** | 超过3000种期刊 |

### 3.2 检索方式

#### 基本检索
```
作者检索:
- 作者: "Tao, Terence"
- 作者ID: 08313

标题检索:
- 标题包含: "prime number"

全文检索:
- 任意字段: "Riemann hypothesis"
```

#### 高级检索
```
组合检索:
- 作者 AND 标题: "Tao" AND "green"
- 时间范围: 2020-2024
- 文献类型: 仅文章/书籍/综述

MSC分类检索:
- 11M06: Riemann zeta函数
- 14Fxx: 代数几何上同调
- 53Cxx: 整体微分几何
```

## 四、MathSciNet使用指南

### 4.1 MathSciNet概述

**MathSciNet** (https://mathscinet.ams.org) 是美国数学学会(AMS)提供的数学文献数据库。

| 特性 | 详情 |
|------|------|
| **文献评论** | 专家撰写的文献评论 |
| **作者数据库** | 详细的作者档案 |
| **引用数据** | 引用分析工具 |
| **期刊评价** | MCQ (Math Citation Quotient) |

### 4.2 检索功能

#### 基本检索
```
出版物检索:
- 作者: Author="Tao, Terence"
- 标题: Title="prime"
- 期刊: Journal="Annals of Mathematics"

MSC检索:
- Primary MSC: 11M06
- Any MSC: 14Fxx
```

#### 高级检索
```
布尔组合:
- (Author="Wiles" OR Author="Taylor") AND MSC="11Fxx"

时间范围:
- Publication Year: 2020-2024

文献类型:
- Books / Articles / Proceedings
```

### 4.3 特色工具

#### 合作距离 (Collaboration Distance)
- 查找两个作者的合作路径
- 数学界的"Erdős数"计算

#### 引用工具
```
引用统计:
- 总被引次数
- 年度引用趋势
- 引用文献列表

引用导出:
- EndNote格式
- BibTeX格式
- 自定义格式
```

#### 期刊信息
```
期刊评价指标:
- MCQ (Math Citation Quotient)
- 影响因子
- 发文量统计

期刊搜索:
- 按标题搜索
- 按ISSN搜索
- 按出版社筛选
```

### 4.4 与zbMATH对比

| 特性 | MathSciNet | zbMATH |
|------|------------|--------|
| **评论** | 有 | 有 |
| **覆盖** | 北美偏重 | 欧洲偏重 |
| **作者档案** | 详细 | 详细 |
| **期刊评价** | MCQ | JFM |
| **历史** | 1940年起 | 1931年起 |

## 六、资源使用建议

### 6.1 日常研究流程

```
1. 跟踪最新进展
   - 订阅arXiv相关分类
   - 浏览MathOverflow新讨论
   
2. 深入文献检索
   - 使用zbMATH/MathSciNet
   - 查找原始文献
   
3. 背景知识学习
   - 阅读MathWorld/nLab
   - 参考PlanetMath
   
4. 数据验证
   - 使用OEIS验证序列
   - 使用软件验证计算
```
