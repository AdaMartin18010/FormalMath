## 一、UCLA数学系概况

### 1.1 机构背景

**加州大学洛杉矶分校 (UCLA)** 是世界顶尖的公立研究型大学，其数学系在分析和应用数学领域享有盛誉：

| 历史贡献 | 描述 | 代表人物 |
|---------|------|---------|
| 调和分析 | 现代调和分析发展 | John Garnett, Terence Tao |
| 偏微分方程 | 非线性波方程研究 | Sergiu Klainerman |
| 泛函分析 | 算子代数与Banach空间 | Edward Effros |
| 数论 | 解析数论与自守形式 | Chandrashekhar Khare |
| 应用数学 | 流体力学与计算数学 | Andrea Bertozzi, Stanley Osher |

### 1.2 课程体系结构

```
┌─────────────────────────────────────────────────────────────────┐
│                    UCLA数学课程体系                              │
├─────────────────────────────────────────────────────────────────┤
│  学术传统                                                        │
│  ├── 强调分析与PDE的严格训练                                      │
│  ├── 纯数学与应用数学并重                                        │
│  └── 参与IPAM (纯与应用数学研究所) 跨学科研究                    │
├─────────────────────────────────────────────────────────────────┤
│  课程特色                                                        │
│  ├── MATH 131A/B: 实分析 - 度量空间与Lebesgue积分                │
│  ├── MATH 235A: 泛函分析 - Banach空间与算子理论                  │
│  └── MATH 266A: 偏微分方程 - 椭圆/抛物/双曲方程                  │
└─────────────────────────────────────────────────────────────────┘
```
