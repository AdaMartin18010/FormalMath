# 06-概率论与统计

本目录是 FormalMath 项目中概率论与数理统计的专门知识库，共收录 40 余份文档，按概率空间、随机变量、随机过程、数理统计、极限定理、统计推断与回归分析等主题分层组织。文档既包含经典理论的严格陈述（如大数定律、中心极限定理、贝叶斯推断），也涵盖现代统计方法的应用拓展，部分文件提供增强版与深度版，以满足不同层次读者的需求。整体目标是构建从概率直觉到统计建模的完整学习闭环。
