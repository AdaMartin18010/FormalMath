## 问题1：生日问题（含实际数据案例）⭐⭐☆☆☆

### 问题描述

在一个房间里至少需要多少人，才能使至少两人生日相同的概率超过50%？

### 实际案例

**Facebook数据分析**：
在Facebook的7.5亿用户中，有约70%的用户 birthday 是唯一的，但仍有大量重复。

对于随机选择的n个人，计算至少两人生日相同的概率。

### 解答

**理论计算**：
P(全部不同) = (365/365) × (364/365) × ... × ((365-n+1)/365)

P(至少一对相同) = 1 - P(全部不同)

计算结果：
- n = 22: P ≈ 0.476
- n = 23: P ≈ 0.507 ✓

**答案**：只需要23人！

### 数据驱动分析

**真实生日分布不均**：
实际上，9-10月出生率较高，这会略微增加重复概率。

**Python验证**：
```python
import numpy as np

def birthday_probability(n, simulations=100000):
    matches = 0
    for _ in range(simulations):
        birthdays = np.random.randint(0, 365, n)
        if len(set(birthdays)) < n:
            matches += 1
    return matches / simulations

for n in [20, 23, 30, 50]:
    print(f"n={n}: P ≈ {birthday_probability(n):.3f}")
```

### 应用场景

1. **密码学**：生日攻击（哈希碰撞）
2. **数据库**：索引设计
3. **流行病学**：疾病爆发模型

## 问题2：蒙特霍尔问题（多解法对比）⭐⭐⭐☆☆

### 问题描述

三扇门后分别是一辆车和两只山羊。你选择一扇门后，主持人（知道车在哪）打开另一扇有山羊的门。你应该换门吗？

### 多种解法

#### 解法一：条件概率（贝叶斯方法）

设 C_i = "车在门i后"，O_j = "主持人开门j"

不换策略：P(赢|不换) = P(C_1) = 1/3

换策略：P(赢|换) = P(C_2|O_3) + P(C_3|O_2) = 2/3

**结论**：应该换门！

#### 解法二：枚举法

| 车在 | 你选 | 主持人开 | 不换 | 换 |
|:---:|:---:|:---:|:---:|:---:|
| 1 | 1 | 2或3 | 赢 | 输 |
| 2 | 1 | 3 | 输 | 赢 |
| 3 | 1 | 2 | 输 | 赢 |

换的胜率：2/3

#### 解法三：直观解释

初始选择正确的概率是1/3，错误的概率是2/3。
换门相当于选择"初始选择错误"的情况。

### 实际案例

**实验验证**：
Marilyn vos Savant在Parade杂志提出此问题后，收到数万封反对信，包括众多数学家。

计算机模拟验证了换门策略的优越性。

## 问题3：赌徒破产问题 ⭐⭐⭐⭐☆

### 问题描述

赌徒初始有i元，每次赌局以概率p赢1元，以概率q=1-p输1元。在达到N元或破产时停止，求破产概率。

### 解答

**递推关系**：
P_i = p·P_{i+1} + q·P_{i-1}

边界条件：P_0 = 1，P_N = 0

**解**：

情况1：p = q = 1/2
P_i = 1 - i/N

情况2：p ≠ q
P_i = [(q/p)^i - (q/p)^N] / [1 - (q/p)^N]

### 实际应用

**例子**：i = 50, N = 100, p = 0.48
P_50 ≈ 0.92

即使只是略微劣势，破产概率也高达92%！

### 现代应用

1. **金融风险管理**：止损策略设计
2. **生物统计**：种群灭绝概率
3. **网络科学**：随机游走模型

## 问题4：优惠券收集问题 ⭐⭐⭐☆☆

### 问题描述

收集一套n种不同的优惠券，每次随机获得一种，平均需要收集多少次才能集齐整套？

### 解答

**分解**：T = T_1 + T_2 + ... + T_n

其中T_i是从已有i-1种到获得第i种新优惠券所需的次数。

**分析**：
E[T_i] = n/(n-i+1)

**期望**：
E[T] = n(1 + 1/2 + ... + 1/n) = n·H_n ≈ n(ln n + γ)

其中H_n是第n个调和数，γ ≈ 0.577是Euler-Mascheroni常数。

### 实际案例

**麦当劳玩具收集**：
n = 10种玩具，E[T] ≈ 29.3次

n = 52张扑克牌，E[T] ≈ 236次

## 问题5：医疗检测的准确率（贝叶斯推断）⭐⭐⭐☆☆

### 问题描述

某种疾病患病率1%。检测灵敏度95%，特异度90%。若检测阳性，实际患病概率是多少？

### 数据驱动解答

**已知**：
- P(D) = 0.01（患病率）
- P(T+|D) = 0.95（灵敏度）
- P(T-|D^c) = 0.90（特异度）

**贝叶斯计算**：

P(D|T+) = P(T+|D)·P(D) / P(T+)

P(T+) = P(T+|D)P(D) + P(T+|D^c)P(D^c)
      = 0.95×0.01 + 0.10×0.99
      = 0.0095 + 0.099 = 0.1085

P(D|T+) = 0.0095 / 0.1085 ≈ 0.0876

**结论**：即使检测阳性，实际患病概率仅约8.76%！

### 直观解释

这个结果看似反直觉，但原因在于：
1. 疾病本身罕见（患病率仅1%）
2. 健康人群基数大（99%），即使假阳性率只有10%，也会产生大量假阳性

### 现代应用

**COVID-19检测**：
在低患病率人群中，即使高灵敏度/特异度检测，阳性预测值也可能不高。

这强调了多次检测和症状综合判断的重要性。

## 问题6：置信区间计算 ⭐⭐⭐☆☆

### 问题描述

从正态总体N(μ, σ²)中抽取n=25的样本，样本均值x̄=10.5，样本标准差s=2.0。求μ的95%置信区间。

### 解答

由于σ未知，使用t分布：

置信区间：x̄ ± t_{α/2,n-1} · s/√n

计算：
- t_{0.025,24} = 2.064
- 标准误 = 2/5 = 0.4
- 边际误差 = 2.064 × 0.4 = 0.826

**95%置信区间**：(9.674, 11.326)

### Python实现

```python
from scipy import stats
import numpy as np

n = 25
x_bar = 10.5
s = 2.0
confidence = 0.95

# 计算置信区间
alpha = 1 - confidence
t_crit = stats.t.ppf(1 - alpha/2, df=n-1)
margin = t_crit * s / np.sqrt(n)

ci_lower = x_bar - margin
ci_upper = x_bar + margin

print(f"95%置信区间: ({ci_lower:.3f}, {ci_upper:.3f})")
```

## 问题7：假设检验 ⭐⭐⭐☆☆

### 问题描述

某工厂零件长度服从N(μ, 0.5²)。抽取25个零件，平均长度10.2cm。检验H₀: μ=10 vs H₁: μ>10（α=0.05）。

### 解答

**检验统计量**：
Z = (x̄ - μ₀)/(σ/√n) = (10.2-10)/(0.5/5) = 2.0

**临界值**：z_{0.05} = 1.645

**决策**：
Z = 2.0 > 1.645，拒绝H₀

**p值**：P(Z > 2.0) = 0.0228 < 0.05

**结论**：有充分证据表明平均长度大于10cm。

## 问题8：回归分析 ⭐⭐⭐☆☆

### 问题描述

研究学习时间（小时）与考试成绩的关系：

| 学习时间 | 2 | 3 | 4 | 5 | 6 | 7 | 8 |
|---------|---|---|---|---|---|---|---|
| 成绩    | 65| 70| 75| 80| 85| 88| 92|

求线性回归方程并计算决定系数R²。

### 数据驱动解答

**计算**：
- x̄ = 5, ȳ = 79.29
- Σ(x-x̄)² = 28
- Σ(x-x̄)(y-ȳ) = 115

**回归系数**：
β̂₁ = 115/28 = 4.107
β̂₀ = 79.29 - 4.107×5 = 58.75

**回归方程**：ŷ = 58.75 + 4.107x

**解释**：每增加1小时学习时间，成绩平均提高约4.1分。

**决定系数**：
R² = 0.970

97%的成绩变异可以由学习时间解释。

## 问题9：中心极限定理应用 ⭐⭐⭐☆☆

### 问题描述

某工厂产品重量服从μ=100g, σ=5g的分布。随机抽取64个产品，求样本均值落在99g到101g之间的概率。

### 解答

**应用CLT**：
x̄ ≈ N(μ, σ²/n) = N(100, 25/64)

标准误：σ/√n = 5/8 = 0.625

**标准化**：
Z = (x̄ - 100)/0.625 ~ N(0,1)

**计算概率**：
P(99 ≤ x̄ ≤ 101) = P(-1.6 ≤ Z ≤ 1.6)
                = 2Φ(1.6) - 1
                = 2×0.9452 - 1
                = 0.8904

**结论**：概率约为89%。

## 问题10：最大值分布 ⭐⭐⭐☆☆

### 问题描述

设X₁, X₂, ..., Xₙ是i.i.d.的U(0,1)随机变量。求Mₙ = max(X₁,...,Xₙ)的分布、期望和方差。

### 解答

**分布函数**：
F_{Mₙ}(x) = P(Mₙ ≤ x) = P(所有Xᵢ ≤ x) = xⁿ, 0 ≤ x ≤ 1

**密度函数**：
f_{Mₙ}(x) = nx^{n-1}

**期望**：
E[Mₙ] = n/(n+1)

**方差**：
Var(Mₙ) = n/[(n+1)²(n+2)]

### 渐近行为

当n → ∞：
- E[Mₙ] → 1
- Var(Mₙ) → 0

这说明随着样本量增加，最大值越来越接近1。
