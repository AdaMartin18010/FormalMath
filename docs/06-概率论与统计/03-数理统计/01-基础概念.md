## 一、概率空间

### 1.1 定义

**定义 1.1** (概率空间)
概率空间是一个三元组 $(\Omega, \mathcal{F}, P)$，其中：
- $\Omega$ 是样本空间（所有可能结果的集合）
- $\mathcal{F}$ 是 $\sigma$-代数（事件的集合）
- $P: \mathcal{F} \to [0,1]$ 是概率测度，满足：
  1. $P(\Omega) = 1$（规范性）
  2. 对可数个互不相交的事件 $A_1, A_2, \ldots$，有 $P(\bigcup_{i=1}^{\infty} A_i) = \sum_{i=1}^{\infty} P(A_i)$（可数可加性）

### 1.2 基本性质

**定理 1.1** (概率的基本性质)
设 $(\Omega, \mathcal{F}, P)$ 为概率空间，则：

1. $P(\emptyset) = 0$
2. $P(A^c) = 1 - P(A)$（补事件公式）
3. $P(A \cup B) = P(A) + P(B) - P(A \cap B)$（加法公式）
4. 若 $A \subseteq B$，则 $P(A) \leq P(B)$（单调性）

### 1.3 条件概率

**定义 1.2** (条件概率)
设 $P(B) > 0$，事件 $A$ 在事件 $B$ 发生的条件下的条件概率为：
$$P(A|B) = \frac{P(A \cap B)}{P(B)}$$

**定理 1.2** (全概率公式)
设 $B_1, B_2, \ldots, B_n$ 是 $\Omega$ 的一个划分（即互不相交且并集为 $\Omega$），则对任意事件 $A$：
$$P(A) = \sum_{i=1}^{n} P(A|B_i)P(B_i)$$

**定理 1.3** (贝叶斯公式)
设 $B_1, B_2, \ldots, B_n$ 是 $\Omega$ 的一个划分，$P(A) > 0$，则：
$$P(B_j|A) = \frac{P(A|B_j)P(B_j)}{\sum_{i=1}^{n} P(A|B_i)P(B_i)}$$

## 二、随机变量

### 2.1 定义

**定义 2.1** (随机变量)
设 $(\Omega, \mathcal{F}, P)$ 为概率空间，随机变量 $X$ 是从 $\Omega$ 到 $\mathbb{R}$ 的可测函数，即对任意 Borel 集 $B \subseteq \mathbb{R}$，有 $\{\omega \in \Omega : X(\omega) \in B\} \in \mathcal{F}$。

### 2.2 分布函数

**定义 2.2** (累积分布函数)
随机变量 $X$ 的累积分布函数（CDF）定义为：
$$F_X(x) = P(X \leq x), \quad x \in \mathbb{R}$$

**性质**:
1. $F_X$ 单调不减
2. $\lim_{x \to -\infty} F_X(x) = 0$，$\lim_{x \to +\infty} F_X(x) = 1$
3. $F_X$ 右连续

### 2.3 离散型与连续型

**定义 2.3** (离散型随机变量)
若随机变量 $X$ 至多取可数个值，则称 $X$ 为离散型随机变量。其概率质量函数（PMF）为：
$$p_X(x) = P(X = x)$$

**定义 2.4** (连续型随机变量)
若存在非负可积函数 $f_X$，使得：
$$F_X(x) = \int_{-\infty}^{x} f_X(t) dt$$
则称 $X$ 为连续型随机变量，$f_X$ 称为概率密度函数（PDF）。

## 三、期望与方差

### 3.1 期望

**定义 3.1** (期望)
- 离散型：$E[X] = \sum_{x} x \cdot p_X(x)$
- 连续型：$E[X] = \int_{-\infty}^{\infty} x \cdot f_X(x) dx$

**定理 3.1** (期望的线性性)
$$E[aX + bY] = aE[X] + bE[Y]$$

### 3.2 方差

**定义 3.2** (方差)
$$\text{Var}(X) = E[(X - E[X])^2] = E[X^2] - (E[X])^2$$

**定理 3.2** (方差的性质)
1. $\text{Var}(aX + b) = a^2 \text{Var}(X)$
2. 若 $X, Y$ 独立，则 $\text{Var}(X + Y) = \text{Var}(X) + \text{Var}(Y)$

**定义 3.3** (标准差)
$$\sigma_X = \sqrt{\text{Var}(X)}$$

### 3.3 协方差与相关系数

**定义 3.4** (协方差)
$$\text{Cov}(X, Y) = E[(X - E[X])(Y - E[Y])] = E[XY] - E[X]E[Y]$$

**定义 3.5** (相关系数)
$$\rho_{X,Y} = \frac{\text{Cov}(X, Y)}{\sigma_X \sigma_Y}$$

## 四、重要分布

### 4.1 离散分布

**伯努利分布** $X \sim \text{Bernoulli}(p)$:
- $P(X=1) = p$，$P(X=0) = 1-p$
- $E[X] = p$，$\text{Var}(X) = p(1-p)$

**二项分布** $X \sim \text{Binomial}(n, p)$:
- $P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}$
- $E[X] = np$，$\text{Var}(X) = np(1-p)$

**泊松分布** $X \sim \text{Poisson}(\lambda)$:
- $P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$
- $E[X] = \lambda$，$\text{Var}(X) = \lambda$

### 4.2 连续分布

**均匀分布** $X \sim \text{Uniform}(a, b)$:
- $f_X(x) = \frac{1}{b-a}$，$x \in [a, b]$
- $E[X] = \frac{a+b}{2}$，$\text{Var}(X) = \frac{(b-a)^2}{12}$

**正态分布** $X \sim \mathcal{N}(\mu, \sigma^2)$:
- $f_X(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$
- $E[X] = \mu$，$\text{Var}(X) = \sigma^2$

**指数分布** $X \sim \text{Exponential}(\lambda)$:
- $f_X(x) = \lambda e^{-\lambda x}$，$x \geq 0$
- $E[X] = \frac{1}{\lambda}$，$\text{Var}(X) = \frac{1}{\lambda^2}$

## 总结

本章节介绍了概率统计的基础概念：
- **概率空间**：严格定义随机实验的数学框架
- **随机变量**：将随机实验结果数量化
- **期望方差**：描述随机变量的数字特征
- **重要分布**：常见的离散型和连续型分布
