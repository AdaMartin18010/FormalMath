# 概率统计基础概念

数理统计学是基于概率论的数据分析科学，研究如何从样本推断总体、估计参数、检验假设以及建立统计模型。Fisher、Neyman、Pearson 与 Wald 的工作奠定了现代统计推断的数学基础。

## 1. 概率空间与随机变量

### 1.1 概率空间的公理化

**定义 1.1（概率空间）**. 概率空间是三元组 $(\Omega, \mathcal{F}, P)$，其中：

- $\Omega$ 为样本空间（所有可能结果的集合）；
- $\mathcal{F} \subseteq \mathcal{P}(\Omega)$ 为 $\sigma$-代数（事件的集合），满足：(i) $\Omega \in \mathcal{F}$；(ii) $A \in \mathcal{F} \Rightarrow A^c \in \mathcal{F}$；(iii) 对可数交封闭；
- $P: \mathcal{F} \to [0,1]$ 为概率测度，满足 $P(\Omega) = 1$ 与可数可加性：对互不相交的 $\{A_n\}$，$P(\bigcup_n A_n) = \sum_n P(A_n)$。

**定理 1.2（概率的基本性质）**. 对任意 $A, B \in \mathcal{F}$：

1. $P(\emptyset) = 0$；
2. $P(A^c) = 1 - P(A)$；
3. $P(A \cup B) = P(A) + P(B) - P(A \cap B)$；
4. 若 $A \subseteq B$，则 $P(A) \leq P(B)$（单调性）。

### 1.2 条件概率与 Bayes 公式

**定义 1.3（条件概率）**. 对 $P(B) > 0$，事件 $A$ 在 $B$ 发生条件下的条件概率为：

$$P(A|B) = \frac{P(A \cap B)}{P(B)}.$$

**定理 1.4（全概率公式）**. 设 $\{B_1, \dots, B_n\}$ 为 $\Omega$ 的划分（$B_i$ 互不相交且 $\bigcup B_i = \Omega$），则：

$$P(A) = \sum_{i=1}^n P(A|B_i)P(B_i).$$

**定理 1.5（Bayes 公式）**. 在相同条件下，若 $P(A) > 0$：

$$P(B_j|A) = \frac{P(A|B_j)P(B_j)}{\sum_i P(A|B_i)P(B_i)}.$$

*证明*. 由 $P(B_j|A) = \frac{P(A \cap B_j)}{P(A)} = \frac{P(A|B_j)P(B_j)}{P(A)}$，代入全概率公式即得。$\square$

## 2. 随机变量与分布

### 2.1 分布函数

**定义 2.1（随机变量）**. 随机变量 $X$ 是可测函数 $X: \Omega \to \mathbb{R}$，即对任意 Borel 集 $B$，$X^{-1}(B) \in \mathcal{F}$。

**定义 2.2（累积分布函数）**. $F_X(x) = P(X \leq x)$，满足：

1. 单调不减；
2. $\lim_{x \to -\infty} F(x) = 0$，$\lim_{x \to +\infty} F(x) = 1$；
3. 右连续。

### 2.2 期望与方差

**定义 2.3（期望）**.

- 离散型：$E[X] = \sum_x x \cdot P(X = x)$；
- 连续型：$E[X] = \int_{-\infty}^{\infty} x f_X(x) dx$。

**定理 2.4（期望的线性性）**. $E[aX + bY] = aE[X] + bE[Y]$（无论 $X, Y$ 是否独立）。

**定义 2.5（方差）**. $\operatorname{Var}(X) = E[(X - E[X])^2] = E[X^2] - (E[X])^2$。

**定理 2.6**. 若 $X, Y$ 独立，则 $\operatorname{Var}(X + Y) = \operatorname{Var}(X) + \operatorname{Var}(Y)$。

## 3. 重要分布

### 3.1 离散分布

**定义 3.1（二项分布）**. $X \sim \text{Binomial}(n, p)$：

$$P(X = k) = \binom{n}{k} p^k (1-p)^{n-k}, \quad k = 0, \dots, n.$$

$E[X] = np$，$\operatorname{Var}(X) = np(1-p)$。

**定义 3.2（Poisson 分布）**. $X \sim \text{Poisson}(\lambda)$：

$$P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}, \quad k = 0, 1, 2, \dots$$

$E[X] = \operatorname{Var}(X) = \lambda$。

### 3.2 连续分布

**定义 3.3（正态分布）**. $X \sim \mathcal{N}(\mu, \sigma^2)$：

$$f_X(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}.$$

**定理 3.4（中心极限定理）**. 设 $\{X_n\}$ 为 i.i.d.，$E[X_i] = \mu$，$\operatorname{Var}(X_i) = \sigma^2 < \infty$。则：

$$\frac{\sum_{i=1}^n X_i - n\mu}{\sigma\sqrt{n}} \xrightarrow{d} \mathcal{N}(0, 1).$$

*证明概要*. 特征函数法：令 $\phi(t) = E[e^{itX_1}]$。标准化后 $Y_i = (X_i - \mu)/\sigma$，$\phi_Y(t) = 1 - t^2/2 + o(t^2)$。则：

$$E\left[\exp\left(it \frac{\sum Y_i}{\sqrt{n}}\right)\right] = \phi_Y(t/\sqrt{n})^n \to e^{-t^2/2},$$

即标准正态的特征函数。$\square$

## 4. 例子

### 4.1 例子：Bayes 推断

某疾病患病率 $0.1\%$，检测灵敏度 $99\%$（患者阳性率），特异度 $99\%$（健康者阴性率）。若某人检测阳性，求其患病概率。

**解答**. 令 $D$ = "患病"，$T$ = "阳性"。$P(D) = 0.001$，$P(T|D) = 0.99$，$P(T|D^c) = 0.01$。

$$P(D|T) = \frac{P(T|D)P(D)}{P(T|D)P(D) + P(T|D^c)P(D^c)} = \frac{0.99 \times 0.001}{0.99 \times 0.001 + 0.01 \times 0.999} \approx 9\%.$$

即使检测准确率很高，因患病率很低，阳性者的实际患病概率仍不高。$\square$

### 4.2 例子：正态近似

掷公平硬币 100 次，求正面次数在 45 到 55 之间的概率。

**解答**. $X \sim \text{Binomial}(100, 0.5)$，$E[X] = 50$，$\operatorname{Var}(X) = 25$。由 CLT：

$$P(45 \leq X \leq 55) \approx P\left(\frac{44.5 - 50}{5} \leq Z \leq \frac{55.5 - 50}{5}\right)$$

$$= P(-1.1 \leq Z \leq 1.1) \approx 2\Phi(1.1) - 1 \approx 0.7286.$$

（连续性修正：用 44.5 和 55.5。）$\square$
