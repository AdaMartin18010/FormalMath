# 马尔可夫链 - 增强版

## 一、引言

马尔可夫链（Markov Chain）是以俄罗斯数学家Andrey Markov（1856-1922）命名的随机过程，其核心特征是"无记忆性"——未来状态只依赖于当前状态，而与过去无关。这一简洁而深刻的性质使马尔可夫链成为物理学、生物学、计算机科学、经济学等领域中最广泛应用的随机模型。

**历史里程碑**：
- **1906年**：Markov提出并研究了链式依赖的随机变量序列
- **1913年**：Markov将马尔可夫链应用于分析普希金诗歌中元音与辅音的交替规律
- **20世纪中叶**：Kolmogorov、Doob等建立了完整的理论体系
- **1990年代至今**：MCMC方法革命性地改变了贝叶斯计算

**数学学科分类（MSC）**: 60J10, 60J20, 60J22

## 二、离散时间马尔可夫链

### 2.1 基本定义

**定义 2.1**（马尔可夫链）随机过程 $\{X_n : n = 0, 1, 2, \ldots\}$，状态空间为可数集 $S$，称为**马尔可夫链**，若满足**马尔可夫性**：
$$\mathbb{P}(X_{n+1} = j \mid X_n = i, X_{n-1}, \ldots, X_0) = \mathbb{P}(X_{n+1} = j \mid X_n = i)$$

对所有 $n \geq 0$ 和 $i, j \in S$ 成立。

---

### 2.2 转移概率

**定义 2.2**（一步转移概率）
$$p_{ij} = \mathbb{P}(X_{n+1} = j \mid X_n = i)$$

矩阵 $P = (p_{ij})_{i,j \in S}$ 称为**转移概率矩阵**，满足：
1. $p_{ij} \geq 0$
2. $\sum_{j \in S} p_{ij} = 1$（对所有 $i \in S$）

**定义 2.3**（$n$步转移概率）
$$p_{ij}^{(n)} = \mathbb{P}(X_n = j \mid X_0 = i)$$

矩阵 $P^{(n)} = (p_{ij}^{(n)})$。

---

### 2.3 Chapman-Kolmogorov方程

**定理 2.4**（Chapman-Kolmogorov方程）对所有 $m, n \geq 0$：
$$p_{ij}^{(m+n)} = \sum_{k \in S} p_{ik}^{(m)} p_{kj}^{(n)}$$

矩阵形式：$P^{(m+n)} = P^{(m)} P^{(n)}$，特别地 $P^{(n)} = P^n$。

**证明**：由马尔可夫性和全概率公式：
$$\begin{aligned}
p_{ij}^{(m+n)} &= \mathbb{P}(X_{m+n} = j \mid X_0 = i) \\
&= \sum_{k} \mathbb{P}(X_{m+n} = j, X_m = k \mid X_0 = i) \\
&= \sum_{k} \mathbb{P}(X_{m+n} = j \mid X_m = k, X_0 = i) \cdot \mathbb{P}(X_m = k \mid X_0 = i) \\
&= \sum_{k} p_{kj}^{(n)} \cdot p_{ik}^{(m)}
\end{aligned}$$

## 三、状态分类

### 3.1 可达与互通

**定义 3.1**（可达）状态 $j$ **可达**于 $i$（记 $i \to j$），若存在 $n \geq 0$ 使 $p_{ij}^{(n)} > 0$。

**定义 3.2**（互通）若 $i \to j$ 且 $j \to i$，称 $i$ 与 $j$ **互通**（记 $i \leftrightarrow j$）。

**定理 3.3** 互通是等价关系，将状态空间划分为**等价类**。

---

### 3.2 常返与暂态

设 $f_i = \mathbb{P}(\exists n \geq 1: X_n = i \mid X_0 = i)$ 为返回 $i$ 的概率。

**定义 3.4**（常返与暂态）
- 若 $f_i = 1$，称 $i$ **常返**（Recurrent）
- 若 $f_i < 1$，称 $i$ **暂态**（Transient）

**定理 3.5**（常返判据）状态 $i$ 常返当且仅当：
$$\sum_{n=0}^{\infty} p_{ii}^{(n)} = \infty$$

**证明**：设 $N_i = \sum_{n=0}^{\infty} \mathbf{1}_{\{X_n = i\}}$ 为访问 $i$ 的总次数。

$$\mathbb{E}[N_i \mid X_0 = i] = \sum_{n=0}^{\infty} \mathbb{P}(X_n = i \mid X_0 = i) = \sum_{n=0}^{\infty} p_{ii}^{(n)}$$

若 $i$ 暂态，每次返回后以概率 $1-f_i$ 不再返回，故 $N_i \sim \text{Geometric}(1-f_i)$，期望有限。

若 $i$ 常返，必然无限次返回，期望为无穷。

---

### 3.3 正递归与零递归

**定义 3.6**（平均返回时间）$\mu_i = \mathbb{E}[T_i \mid X_0 = i]$，其中 $T_i = \min\{n \geq 1 : X_n = i\}$。

**定义 3.7**（正递归与零递归）
- 若 $i$ 常返且 $\mu_i < \infty$，称 $i$ **正递归**
- 若 $i$ 常返且 $\mu_i = \infty$，称 $i$ **零递归**

---

### 3.4 周期性

**定义 3.8**（周期）状态 $i$ 的**周期**定义为：
$$d(i) = \gcd\{n \geq 1 : p_{ii}^{(n)} > 0\}$$

若 $d(i) = 1$，称 $i$ **非周期**。

**定理 3.9** 同一等价类中所有状态具有相同的周期。

---

## 四、遍历理论

### 4.1 平稳分布

**定义 4.1**（平稳分布）概率分布 $\pi = (\pi_j)_{j \in S}$ 称为**平稳分布**，若：
$$\pi_j = \sum_{i \in S} \pi_i p_{ij}, \quad \forall j \in S$$

矩阵形式：$\pi = \pi P$。

**解释**：若 $X_0 \sim \pi$，则对所有 $n$，$X_n \sim \pi$。

---

### 4.2 遍历定理

**定义 4.2**（遍历链）马尔可夫链称为**遍历**的，若它是不可约、非周期、正递归的。

**定理 4.3**（遍历定理）设马尔可夫链遍历，则：

1. 存在唯一平稳分布 $\pi$
2. $\lim_{n \to \infty} p_{ij}^{(n)} = \pi_j$（与初始状态 $i$ 无关）
3. $\pi_j = \frac{1}{\mu_j}$（其中 $\mu_j$ 为平均返回时间）

**证明概要**（耦合方法）：
考虑两个独立的链 $\{X_n\}$ 和 $\{Y_n\}$，分别始于 $i$ 和平稳分布 $\pi$。构造耦合时间 $T = \min\{n : X_n = Y_n\}$。由不可约性和非周期性，$\mathbb{P}(T < \infty) = 1$。故：
$$|\mathbb{P}(X_n = j) - \pi_j| = |\mathbb{P}(X_n = j) - \mathbb{P}(Y_n = j)| \leq \mathbb{P}(T > n) \to 0$$

---

### 4.3 细致平衡条件

**定义 4.4**（细致平衡）概率分布 $\pi$ 满足**细致平衡条件**，若：
$$\pi_i p_{ij} = \pi_j p_{ji}, \quad \forall i, j \in S$$

**定理 4.5** 满足细致平衡条件的分布必为平稳分布。

**证明**：
$$\sum_i \pi_i p_{ij} = \sum_i \pi_j p_{ji} = \pi_j \sum_i p_{ji} = \pi_j$$

---

## 五、典型示例

### 5.1 简单随机游走

**例 5.1**（一维简单随机游走）$X_n = \sum_{i=1}^n \xi_i$，其中 $\xi_i$ i.i.d.，$\mathbb{P}(\xi_i = 1) = p$，$\mathbb{P}(\xi_i = -1) = q = 1-p$。

- 转移概率：$p_{i,i+1} = p$，$p_{i,i-1} = q$
- 周期：$d = 2$
- 当 $p = q = 1/2$（对称随机游走）：
  - 常返性：常返（一维、二维），暂态（三维及以上）
  - **Polya定理**（1921）：对称随机游走在 $\mathbb{Z}^d$ 中常返当且仅当 $d \leq 2$

---

### 5.2 生灭过程

**例 5.2**（生灭链）状态空间 $S = \{0, 1, 2, \ldots\}$，转移概率：
- $p_{i,i+1} = p_i$（出生）
- $p_{i,i-1} = q_i$（死亡）
- $p_{i,i} = r_i$

其中 $p_i + q_i + r_i = 1$，$q_0 = 0$。

**平稳分布的存在性**：定义 $\rho_0 = 1$，$\rho_i = \frac{p_0 p_1 \cdots p_{i-1}}{q_1 q_2 \cdots q_i}$。则平稳分布存在当且仅当 $\sum_i \rho_i < \infty$，且：
$$\pi_i = \frac{\rho_i}{\sum_j \rho_j}$$

---

### 5.3 Ehrenfest模型

**例 5.3**（Ehrenfest扩散模型）$N$ 个分子分布在两个容器 $A$ 和 $B$ 中。每步随机选择一个分子移到另一个容器。

状态：$A$ 中分子数 $X_n \in \{0, 1, \ldots, N\}$

转移概率：
$$p_{i,i+1} = \frac{N-i}{N}, \quad p_{i,i-1} = \frac{i}{N}$$

**平稳分布**：$\pi_i = \binom{N}{i} 2^{-N}$（二项分布）

验证细致平衡：$\pi_i p_{i,i+1} = \binom{N}{i} 2^{-N} \cdot \frac{N-i}{N} = \binom{N}{i+1} 2^{-N} \cdot \frac{i+1}{N} = \pi_{i+1} p_{i+1,i}$

---

## 六、吸收概率与平均吸收时间

### 6.1 吸收概率

设 $A$ 为吸收态集合（$p_{ii} = 1$ 对 $i \in A$）。

**定理 6.1** 从状态 $i$ 出发最终被吸收于 $a \in A$ 的概率 $u_i$ 满足：
$$u_i = \sum_{j \in S} p_{ij} u_j$$

边界条件：$u_a = 1$（$a \in A$），$u_b = 0$（$b \in A$，$b \neq a$）。

---

### 6.2 赌徒破产问题

**例 6.2**（赌徒破产）赌徒有 $i$ 元，目标 $N$ 元。每局以概率 $p$ 赢1元，以概率 $q = 1-p$ 输1元。求破产概率。

**解**：设 $u_i$ 为从 $i$ 元出发最终破产的概率。递推方程：
$$u_i = p \cdot u_{i+1} + q \cdot u_{i-1}, \quad 1 \leq i \leq N-1$$

边界：$u_0 = 1$，$u_N = 0$。

**解得**：
- 若 $p \neq q$：$u_i = \frac{(q/p)^i - (q/p)^N}{1 - (q/p)^N}$
- 若 $p = q = 1/2$：$u_i = 1 - \frac{i}{N}$

---

## 七、连续时间马尔可夫链

### 7.1 定义

**定义 7.1**（连续时间马尔可夫链）连续时间随机过程 $\{X_t : t \geq 0\}$，状态空间 $S$ 可数，满足：
$$\mathbb{P}(X_{t+s} = j \mid X_t = i, X_u : u < t) = \mathbb{P}(X_{t+s} = j \mid X_t = i) = p_{ij}(s)$$

---

### 7.2 转移速率矩阵

**定义 7.2**（转移速率矩阵）定义：
$$q_{ij} = \lim_{h \to 0} \frac{p_{ij}(h) - \delta_{ij}}{h}$$

矩阵 $Q = (q_{ij})$ 满足 $q_{ij} \geq 0$（$i \neq j$），$\sum_j q_{ij} = 0$。

---

### 7.3 Kolmogorov方程

**定理 7.3**（Kolmogorov前向与后向方程）
$$\text{前向：} \frac{d}{dt} P(t) = P(t) Q$$
$$\text{后向：} \frac{d}{dt} P(t) = Q P(t)$$

形式解：$P(t) = e^{Qt} = \sum_{n=0}^{\infty} \frac{t^n Q^n}{n!}$

---

## 八、MCMC方法简介

### 8.1 Metropolis-Hastings算法

**算法**：
1. 当前状态 $X_n = i$
2. 从提议分布 $q(\cdot|i)$ 采样 $j$
3. 计算接受概率 $\alpha = \min\left(1, \frac{\pi_j q(i|j)}{\pi_i q(j|i)}\right)$
4. 以概率 $\alpha$ 接受 $X_{n+1} = j$，否则 $X_{n+1} = i$

**定理 8.2** 生成的链以 $\pi$ 为平稳分布。
