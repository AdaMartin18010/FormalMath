# 布朗运动与随机微积分 - 深度扩展版

## 一、引言与历史背景

布朗运动（Brownian Motion）是随机过程理论中最基本、最重要的连续时间随机过程，它既是物理现象的理想化模型，也是现代金融数学的理论基石。

1827年，英国植物学家Robert Brown在显微镜下观察到悬浮于水中的花粉颗粒呈现无规则、永不停息的运动。1905年，Albert Einstein从分子动理论出发，首次给出Brown运动的定量解释，证明了粒子的均方位移与时间成正比。Louis Bachelier在1900年的博士论文《投机理论》中，率先将Brown运动应用于股票价格建模，开创了数理金融学的先河。

1923年，Norbert Wiener构造了Brown运动的严格数学模型，因此Brown运动在数学文献中也常被称为**Wiener过程**。1940年代，日本数学家Kiyoshi Itô创立了随机微积分（Stochastic Calculus），定义了随机积分并建立了Itô公式，使处理含随机微分项的微分方程成为可能。这一理论框架后来在Black、Scholes和Merton的期权定价模型中发挥了关键作用，并为他们赢得了1997年的诺贝尔经济学奖。

**数学学科分类（MSC）**: 60H05, 60H10

## 二、布朗运动的数学定义

### 2.1 标准布朗运动的定义

**定义 2.1**（标准布朗运动）随机过程 $\{B_t : t \geq 0\}$ 称为**标准布朗运动**（Standard Brownian Motion），若满足：

1. **初始条件**：$B_0 = 0$（a.s.）
2. **独立增量**：对任意 $0 \leq t_1 < t_2 < \cdots < t_n$，增量 $B_{t_2}-B_{t_1}, \ldots, B_{t_n}-B_{t_{n-1}}$ 相互独立
3. **平稳正态增量**：$B_{t+s} - B_s \sim \mathcal{N}(0, t)$，即服从均值为0、方差为 $t$ 的正态分布
4. **样本路径连续性**：$t \mapsto B_t(\omega)$ 几乎必然连续

---

### 2.2 存在性与基本性质

**定理 2.2**（Wiener存在定理）标准布朗运动存在。

**证明思路**（Kolmogorov扩张 + Kolmogorov-Chentsov连续性判据）：
1. 利用Brown运动的高斯性质定义相容的有限维分布族
2. 应用Kolmogorov扩张定理得到过程的存在性
3. 由Kolmogorov-Chentsov定理（指数 $\alpha = 4, \beta = 1$）证明存在连续修正

**定理 2.3**（协方差结构）对 $s, t \geq 0$：
$$\text{Cov}(B_s, B_t) = \min(s, t)$$

**证明**：不妨设 $s \leq t$，则：
$$\begin{aligned}
\text{Cov}(B_s, B_t) &= \text{Cov}(B_s, B_s + (B_t - B_s)) \\
&= \text{Cov}(B_s, B_s) + \text{Cov}(B_s, B_t - B_s) \\
&= s + 0 = s = \min(s, t)
\end{aligned}$$

---

### 2.3 布朗运动的不可微性

**定理 2.4**（Paley-Wiener-Zygmund，1933）Brown运动的样本路径几乎必然处处不可微。

这一性质深刻反映了Brown运动的"粗糙"本质，也是Itô积分必须采用特殊定义的根本原因——传统Riemann-Stieltjes积分要求被积函数对有界变差函数可积，而Brown运动的路径具有无限变差。

---

## 三、Itô积分

### 3.1 构造动机

考虑形如以下的"随机微分方程"：
$$dX_t = \mu(X_t, t)dt + \sigma(X_t, t)dB_t$$

其中第二项涉及对Brown运动路径的积分。由于Brown运动路径几乎必然具有无限变差，传统Riemann-Stieltjes积分 $\int f dB_t$ 无法按经典意义定义。Itô的突破性思想是：利用Brown运动的鞅性质，在均方意义（$L^2$）下定义随机积分。

---

### 3.2 简单过程的Itô积分

**定义 3.1**（简单过程）过程 $H_t$ 称为**简单过程**，若存在分割 $0 = t_0 < t_1 < \cdots < t_n$ 和随机变量 $\xi_i \in \mathcal{F}_{t_i}$，使得：
$$H_t = \sum_{i=0}^{n-1} \xi_i \mathbf{1}_{[t_i, t_{i+1})}(t)$$

**定义 3.2**（简单过程的Itô积分）定义：
$$\int_0^T H_t dB_t = \sum_{i=0}^{n-1} \xi_i (B_{t_{i+1}} - B_{t_i})$$

---

### 3.3 一般可料过程的扩展

设 $\mathcal{H}^2$ 为满足以下条件的可料过程空间：
$$\|H\|_{\mathcal{H}^2}^2 = \mathbb{E}\left[\int_0^T H_t^2 dt\right] < \infty$$

**定理 3.3**（Itô等距）对简单过程 $H$：
$$\mathbb{E}\left[\left(\int_0^T H_t dB_t\right)^2\right] = \mathbb{E}\left[\int_0^T H_t^2 dt\right]$$

**证明**：设 $H_t = \sum_{i} \xi_i \mathbf{1}_{[t_i, t_{i+1})}(t)$，则：
$$\begin{aligned}
\mathbb{E}\left[\left(\sum_i \xi_i \Delta B_i\right)^2\right] &= \sum_i \mathbb{E}[\xi_i^2 \Delta B_i^2] + 2\sum_{i<j} \mathbb{E}[\xi_i \xi_j \Delta B_i \Delta B_j] \\
&= \sum_i \mathbb{E}[\xi_i^2] \Delta t_i + 0 = \mathbb{E}\left[\int_0^T H_t^2 dt\right]
\end{aligned}$$

其中利用了 $\xi_i \in \mathcal{F}_{t_i}$ 与 $\Delta B_i = B_{t_{i+1}} - B_{t_i} \perp \mathcal{F}_{t_i}$ 的独立性。

**定理 3.4**（扩展到一般过程）Itô等距允许我们将积分等距地扩展到整个 $\mathcal{H}^2$ 空间。

---

### 3.4 Itô过程与二次变差

**定义 3.5**（Itô过程）过程 $X_t$ 称为**Itô过程**，若可表示为：
$$X_t = X_0 + \int_0^t \mu_s ds + \int_0^t \sigma_s dB_s$$

或用微分形式：$dX_t = \mu_t dt + \sigma_t dB_t$。

**定义 3.6**（二次变差）过程 $X$ 的**二次变差**定义为：
$$[X]_t = \lim_{|\Pi| \to 0} \sum_{i} (X_{t_{i+1}} - X_{t_i})^2$$

**定理 3.7**（Brown运动的二次变差）$[B]_t = t$（a.s.）。

## 四、Itô公式

### 4.1 一维Itô公式

**定理 4.1**（Itô公式）设 $X_t$ 为Itô过程，$f \in C^{2,1}(\mathbb{R} \times [0,\infty))$，则 $Y_t = f(X_t, t)$ 也是Itô过程，且：
$$df(X_t, t) = \frac{\partial f}{\partial t}dt + \frac{\partial f}{\partial x}dX_t + \frac{1}{2}\frac{\partial^2 f}{\partial x^2}(dX_t)^2$$

其中 $(dX_t)^2 = \sigma_t^2 dt$（由 $dt \cdot dt = 0$，$dt \cdot dB_t = 0$，$dB_t \cdot dB_t = dt$ 的Itô乘法表）。

若 $dX_t = \mu_t dt + \sigma_t dB_t$，则：
$$df(X_t, t) = \left(\frac{\partial f}{\partial t} + \mu_t \frac{\partial f}{\partial x} + \frac{\sigma_t^2}{2}\frac{\partial^2 f}{\partial x^2}\right)dt + \sigma_t \frac{\partial f}{\partial x}dB_t$$

---

### 4.2 Itô公式的应用示例

**示例 4.2**（几何Brown运动）求解 $dS_t = \mu S_t dt + \sigma S_t dB_t$。

令 $Y_t = \ln S_t$，应用Itô公式：
$$dY_t = \frac{1}{S_t}dS_t - \frac{1}{2S_t^2}(dS_t)^2 = \left(\mu - \frac{\sigma^2}{2}\right)dt + \sigma dB_t$$

积分得：
$$\ln S_t = \ln S_0 + \left(\mu - \frac{\sigma^2}{2}\right)t + \sigma B_t$$

因此：
$$S_t = S_0 \exp\left(\left(\mu - \frac{\sigma^2}{2}\right)t + \sigma B_t\right)$$

这就是Black-Scholes模型中股票价格的标准形式。

**示例 4.3**（Ornstein-Uhlenbeck过程）求解 $dX_t = -\alpha X_t dt + \sigma dB_t$，$\alpha > 0$。

令 $Y_t = e^{\alpha t} X_t$，由乘积法则：
$$dY_t = \alpha e^{\alpha t} X_t dt + e^{\alpha t} dX_t = \sigma e^{\alpha t} dB_t$$

积分得：
$$X_t = e^{-\alpha t} X_0 + \sigma \int_0^t e^{-\alpha(t-s)} dB_s$$

---

### 4.3 多维Itô公式

**定理 4.4**（多维Itô公式）设 $X_t = (X_t^1, \ldots, X_t^n)$ 为 $n$ 维Itô过程，$f \in C^{2,1}(\mathbb{R}^n \times [0,\infty))$，则：
$$df(X_t, t) = \frac{\partial f}{\partial t}dt + \sum_{i=1}^n \frac{\partial f}{\partial x_i}dX_t^i + \frac{1}{2}\sum_{i,j=1}^n \frac{\partial^2 f}{\partial x_i \partial x_j}dX_t^i dX_t^j$$

---

## 五、随机微分方程

### 5.1 一般形式与解的存在性

**定义 5.1**（随机微分方程，SDE）形如以下的方程称为SDE：
$$dX_t = \mu(X_t, t)dt + \sigma(X_t, t)dB_t, \quad X_0 = x_0$$

或积分形式：
$$X_t = x_0 + \int_0^t \mu(X_s, s)ds + \int_0^t \sigma(X_s, s)dB_s$$

**定理 5.2**（存在唯一性）设 $\mu$ 和 $\sigma$ 满足**全局Lipschitz条件**和**线性增长条件**：
$$|\mu(x,t) - \mu(y,t)| + |\sigma(x,t) - \sigma(y,t)| \leq L|x-y|$$
$$|\mu(x,t)|^2 + |\sigma(x,t)|^2 \leq K(1+|x|^2)$$

则存在唯一的强解 $\{X_t\}$（适应于Brown运动生成的滤子）。

---

### 5.2 常用SDE及其解

| SDE名称 | 方程 | 显式解 |
|--------|------|--------|
| **几何Brown运动** | $dS_t = \mu S_t dt + \sigma S_t dB_t$ | $S_t = S_0 \exp((\mu-\sigma^2/2)t + \sigma B_t)$ |
| **Ornstein-Uhlenbeck** | $dX_t = -\alpha X_t dt + \sigma dB_t$ | $X_t = e^{-\alpha t}X_0 + \sigma\int_0^t e^{-\alpha(t-s)}dB_s$ |
| **CIR过程** | $dr_t = a(b-r_t)dt + \sigma\sqrt{r_t}dB_t$ | 无闭式解，但存在拉普拉斯变换 |
| **Brown桥** | $dX_t = \frac{c-X_t}{T-t}dt + dB_t$ | $X_t = \frac{T-t}{T}a + \frac{t}{T}c + (T-t)\int_0^t \frac{dB_s}{T-s}$ |

---

### 5.3 Feynman-Kac公式

**定理 5.3**（Feynman-Kac公式）设 $u(x,t)$ 满足PDE：
$$\frac{\partial u}{\partial t} + \mu(x)\frac{\partial u}{\partial x} + \frac{\sigma^2(x)}{2}\frac{\partial^2 u}{\partial x^2} - r(x)u = 0, \quad u(x,T) = g(x)$$

则解可表示为条件期望：
$$u(x,t) = \mathbb{E}\left[\exp\left(-\int_t^T r(X_s)ds\right)g(X_T) \Big| X_t = x\right]$$

其中 $X_t$ 满足 $dX_t = \mu(X_t)dt + \sigma(X_t)dB_t$。

**应用**：这是Black-Scholes期权定价偏微分方程的概率表示，将PDE问题转化为Monte Carlo模拟问题。

## 六、在金融中的应用：Black-Scholes模型

### 6.1 模型设定

在Black-Scholes框架下，股票价格 $\{S_t\}$ 服从几何Brown运动：
$$dS_t = \mu S_t dt + \sigma S_t dB_t$$

其中：
- $\mu$：期望收益率
- $\sigma$：波动率
- $B_t$：标准Brown运动

设无风险利率为 $r$，则折现后价格 $\tilde{S}_t = e^{-rt}S_t$ 在风险中性测度 $\mathbb{Q}$ 下是鞅：
$$d\tilde{S}_t = \sigma \tilde{S}_t d\tilde{B}_t$$

### 6.2 Black-Scholes公式

欧式看涨期权在风险中性定价下的价值：
$$C(S_0, K, T, r, \sigma) = S_0 N(d_1) - Ke^{-rT}N(d_2)$$

其中：
$$d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2)T}{\sigma\sqrt{T}}, \quad d_2 = d_1 - \sigma\sqrt{T}$$

## 七、总结与参考文献

本讲系统阐述了Brown运动与随机微积分的核心理论：

- **布朗运动**：具有独立平稳增量的连续高斯过程，样本路径处处连续但几乎必然不可微
- **Itô积分**：对适应过程关于Brown运动的均方极限定义，满足Itô等距
- **Itô公式**：随机分析的核心工具，包含二阶修正项
- **SDE理论**：解的存在唯一性条件，常见方程的显式解
- **Feynman-Kac公式**：连接随机分析与偏微分方程的桥梁

随机微积分为现代金融工程提供了坚实的数学基础，期权定价、风险管理、利率建模等领域都依赖于这一理论框架。
