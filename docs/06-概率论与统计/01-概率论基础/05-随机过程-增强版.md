# 随机过程 - 增强版

## 一、引言

随机过程（Stochastic Process）是概率论中研究随时间演变的随机现象的核心分支。从1827年Brown发现花粉的无规则运动，到1900年Bachelier开创金融数学，再到现代机器学习中的随机算法，随机过程理论始终处于数学与科学的交叉前沿。

**数学学科分类（MSC）**: 60G05, 60G07, 60G40

---

## 二、随机过程的基本概念

### 2.1 形式化定义

**定义 2.1**（随机过程）设 $(\Omega, \mathcal{F}, \mathbb{P})$ 为概率空间，$T$ 为指标集（通常表示时间）。一族随机变量 $\{X_t : t \in T\}$ 称为**随机过程**，其中 $X_t: \Omega \to E$ 取值于状态空间 $E$。

等价视角：
- 对每个固定的 $t \in T$，$X_t(\cdot)$ 是随机变量
- 对每个固定的 $\omega \in \Omega$，$X(\cdot, \omega): T \to E$ 是**样本路径**
- $X$ 可视为映射 $X: T \times \Omega \to E$

**分类**：
| 维度 | 类型 | 描述 |
|------|------|------|
| 时间 | 离散时间 | $T = \{0, 1, 2, \ldots\}$ |
| | 连续时间 | $T = [0, \infty)$ |
| 状态 | 离散状态 | $E$ 可数（如整数） |
| | 连续状态 | $E = \mathbb{R}^d$ |

---

### 2.2 有限维分布

**定义 2.2**（有限维分布族）对任意 $t_1 < t_2 < \cdots < t_n$，定义：
$$F_{t_1, \ldots, t_n}(x_1, \ldots, x_n) = \mathbb{P}(X_{t_1} \leq x_1, \ldots, X_{t_n} \leq x_n)$$

**相容性条件**：
1. **对称性**：对任意置换 $\pi$：
   $$F_{t_1, \ldots, t_n}(x_1, \ldots, x_n) = F_{t_{\pi(1)}, \ldots, t_{\pi(n)}}(x_{\pi(1)}, \ldots, x_{\pi(n)})$$

2. **一致性**：
   $$F_{t_1, \ldots, t_n, t_{n+1}}(x_1, \ldots, x_n, \infty) = F_{t_1, \ldots, t_n}(x_1, \ldots, x_n)$$

---

### 2.3 Kolmogorov扩张定理

**定理 2.3**（Kolmogorov扩张定理）设 $\{F_{t_1, \ldots, t_n}\}$ 是满足相容性条件的有限维分布族。则存在概率空间 $(\Omega, \mathcal{F}, \mathbb{P})$ 和随机过程 $\{X_t\}$，使其有限维分布恰为 $\{F_{t_1, \ldots, t_n}\}$。

**意义**：该定理保证了随机过程的存在性，是随机过程理论的基石。

---

## 三、Poisson过程

### 3.1 定义与等价刻画

**定义 3.1**（Poisson过程）计数过程 $\{N_t : t \geq 0\}$ 称为强度为 $\lambda > 0$ 的**Poisson过程**，若：

1. $N_0 = 0$（a.s.）
2. **独立增量**：对 $0 \leq t_1 < t_2 < \cdots < t_n$，增量 $N_{t_2}-N_{t_1}, \ldots, N_{t_n}-N_{t_{n-1}}$ 相互独立
3. **平稳增量**：$N_{t+s} - N_s \stackrel{d}{=} N_t$，即分布仅依赖于时间长度
4. **稀有性**：$\mathbb{P}(N_h = 1) = \lambda h + o(h)$，$\mathbb{P}(N_h \geq 2) = o(h)$

**定理 3.2**（Poisson分布）对任意 $t \geq 0$，$N_t \sim \text{Poisson}(\lambda t)$：
$$\mathbb{P}(N_t = k) = \frac{(\lambda t)^k e^{-\lambda t}}{k!}, \quad k = 0, 1, 2, \ldots$$

**证明思路**：将 $[0, t]$ 分为 $n$ 等份，每份发生事件概率为 $\lambda t/n$。由二项分布逼近：
$$\binom{n}{k} \left(\frac{\lambda t}{n}\right)^k \left(1-\frac{\lambda t}{n}\right)^{n-k} \to \frac{(\lambda t)^k e^{-\lambda t}}{k!}$$

---

### 3.2 到达时间与到达间隔

设 $S_n$ 为第 $n$ 个事件的到达时间，$T_n = S_n - S_{n-1}$ 为到达时间间隔（$S_0 = 0$）。

**定理 3.3**（到达时间间隔的分布）$T_1, T_2, \ldots$ 是 i.i.d. 的指数随机变量：
$$T_n \sim \text{Exp}(\lambda), \quad f_{T_n}(t) = \lambda e^{-\lambda t}, \quad t \geq 0$$

**证明**：
$$\mathbb{P}(T_1 > t) = \mathbb{P}(N_t = 0) = e^{-\lambda t}$$

故 $T_1 \sim \text{Exp}(\lambda)$。由独立平稳增量性质，$T_n$ i.i.d.。

**定理 3.4**（到达时间的分布）$S_n \sim \text{Gamma}(n, \lambda)$：
$$f_{S_n}(t) = \frac{\lambda^n t^{n-1} e^{-\lambda t}}{(n-1)!}, \quad t \geq 0$$

**等价刻画**：Poisson过程是到达间隔为 i.i.d. 指数分布的计数过程。

---

### 3.3 叠加与分解

**定理 3.5**（叠加定理）设 $\{N_t^{(1)}\}$ 和 $\{N_t^{(2)}\}$ 是独立的Poisson过程，强度分别为 $\lambda_1$ 和 $\lambda_2$。则 $N_t = N_t^{(1)} + N_t^{(2)}$ 是强度为 $\lambda_1 + \lambda_2$ 的Poisson过程。

**定理 3.6**（分解定理）设 $\{N_t\}$ 是强度为 $\lambda$ 的Poisson过程，每个事件以概率 $p$ 标记为类型1，以概率 $1-p$ 标记为类型2。则类型1事件形成强度为 $\lambda p$ 的Poisson过程，类型2事件形成强度为 $\lambda(1-p)$ 的Poisson过程，且二者独立。

---

## 四、更新过程

### 4.1 定义

**定义 4.1**（更新过程）设 $\{T_n\}$ 是 i.i.d. 的非负随机变量，分布函数为 $F$，且 $F(0) < 1$。令 $S_n = \sum_{i=1}^n T_i$，$N_t = \max\{n : S_n \leq t\}$。则 $\{N_t : t \geq 0\}$ 称为**更新过程**。

Poisson过程是 $T_n \sim \text{Exp}(\lambda)$ 的特例。

---

### 4.2 更新函数与更新方程

**定义 4.2**（更新函数）$m(t) = \mathbb{E}[N_t]$ 称为**更新函数**。

**定理 4.3**（更新方程）更新函数满足：
$$m(t) = F(t) + \int_0^t m(t-s) dF(s)$$

**证明**：
$$m(t) = \mathbb{E}[N_t] = \int_0^{\infty} \mathbb{E}[N_t|T_1 = s] dF(s)$$

$$= \int_0^t \mathbb{E}[1 + N_{t-s}] dF(s) = F(t) + \int_0^t m(t-s) dF(s)$$

---

### 4.3 基本更新定理

**定理 4.4**（基本更新定理）
$$\lim_{t \to \infty} \frac{m(t)}{t} = \frac{1}{\mu}$$

其中 $\mu = \mathbb{E}[T_1] \leq \infty$。

**定理 4.5**（Blackwell更新定理）若 $F$ 非格点分布，则对任意 $h > 0$：
$$\lim_{t \to \infty} [m(t+h) - m(t)] = \frac{h}{\mu}$$

---

## 五、平稳过程

### 5.1 定义

**定义 5.1**（严平稳过程）随机过程 $\{X_t : t \in T\}$ 称为**严平稳**的，若对任意 $n$，任意 $t_1, \ldots, t_n \in T$ 和 $h$（使得 $t_i + h \in T$）：
$$(X_{t_1}, \ldots, X_{t_n}) \stackrel{d}{=} (X_{t_1+h}, \ldots, X_{t_n+h})$$

**定义 5.2**（宽平稳过程）二阶矩存在的随机过程称为**宽平稳**的，若：
1. $\mathbb{E}[X_t] = \mu$（常数）
2. $\text{Cov}(X_t, X_{t+h}) = \gamma(h)$（仅依赖于时滞 $h$）

函数 $\gamma(h)$ 称为**自协方差函数**。

---

### 5.2 遍历性

**定义 5.3**（均值遍历性）宽平稳过程 $\{X_t\}$ 称为**均值遍历**的，若：
$$\frac{1}{T}\int_0^T X_t dt \xrightarrow{L^2} \mu \quad (T \to \infty)$$

**定理 5.4**（均值遍历定理）宽平稳过程均值遍历当且仅当：
$$\lim_{T \to \infty} \frac{1}{T}\int_0^T \gamma(t) dt = 0$$

---

## 六、鞅初步

### 6.1 定义

**定义 6.1**（离散时间鞅）适应于滤子 $\{\mathcal{F}_n\}$ 的随机过程 $\{M_n\}$ 称为**鞅**，若：
1. $\mathbb{E}[|M_n|] < \infty$
2. $\mathbb{E}[M_{n+1} | \mathcal{F}_n] = M_n$（a.s.）

若将等号替换为 $\geq$（$\leq$），分别称为**下鞅**（Submartingale）和**上鞅**（Supermartingale）。

---

### 6.2 鞅收敛定理

**定理 6.2**（Doob鞅收敛定理）设 $\{M_n\}$ 为下鞅且 $\sup_n \mathbb{E}[M_n^+] < \infty$。则 $M_n \to M_{\infty}$ a.s.，且 $\mathbb{E}[|M_{\infty}|] < \infty$。

**定理 6.3**（Doob停时定理）设 $\{M_n\}$ 为鞅，$T$ 为有界停时，则：
$$\mathbb{E}[M_T] = \mathbb{E}[M_0]$$
