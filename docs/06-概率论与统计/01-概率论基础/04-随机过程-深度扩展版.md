# 随机过程 - 深度扩展版

## 一、引言与历史背景

随机过程（Stochastic Process）是现代概率论的核心分支，研究随时间演变的随机现象。其历史可追溯至1827年英国植物学家Robert Brown观察到的花粉颗粒无规则运动——Brown运动。1900年，Louis Bachelier在其博士论文中首次用数学方法研究股票价格，开创了金融数学的先河。

20世纪30年代，Andrey Kolmogorov建立了概率论的公理化体系，为随机过程的严格数学理论奠定了基础。Norbert Wiener（1923）构造了Brown运动的严格数学模型，Joseph Doob发展了鞅理论，而Kiyoshi Itô在1940年代创立了随机微积分。这些里程碑式的工作使随机过程成为连接纯粹数学与应用科学的桥梁。

**数学学科分类（MSC）**: 60G05, 60G07, 60G42

## 二、随机过程的基本概念

### 2.1 形式化定义

**定义 2.1**（随机过程）设 $(\Omega, \mathcal{F}, \mathbb{P})$ 为概率空间，$T$ 为指标集（通常表示时间）。一族随机变量 $\{X_t : t \in T\}$ 称为**随机过程**（Stochastic Process），其中 $X_t: \Omega \to E$ 取值于状态空间 $E$（通常为 $\mathbb{R}^d$）。

等价地，随机过程可视为映射 $X: T \times \Omega \to E$，其中：
- 对每个固定的 $t \in T$，$X_t(\cdot)$ 是随机变量
- 对每个固定的 $\omega \in \Omega$，$X(\cdot, \omega)$ 是样本路径（Sample Path）

**定义 2.2**（有限维分布）随机过程的**有限维分布**（Finite-Dimensional Distributions）定义为：
$$\mu_{t_1, \ldots, t_n}(B_1 \times \cdots \times B_n) = \mathbb{P}(X_{t_1} \in B_1, \ldots, X_{t_n} \in B_n)$$

**定理 2.3**（Kolmogorov扩张定理）设 $\{\mu_{t_1, \ldots, t_n}\}$ 满足相容性条件的概率测度族，则存在概率空间 $(\Omega, \mathcal{F}, \mathbb{P})$ 和随机过程 $\{X_t\}$，使其有限维分布恰为 $\{\mu_{t_1, \ldots, t_n}\}$。

---

### 2.2 随机过程的分类

| 分类依据 | 类型 | 特征 |
|---------|------|------|
| **时间参数** | 离散时间 | $T = \{0, 1, 2, \ldots\}$，如Markov链 |
| | 连续时间 | $T = [0, \infty)$，如Brown运动 |
| **状态空间** | 离散状态 | $E$ 可数，如Poisson过程 |
| | 连续状态 | $E = \mathbb{R}^d$，如扩散过程 |
| **增量性质** | 独立增量 | $X_t - X_s \perp \sigma(X_u : u \leq s)$ |
| | 平稳增量 | $X_{t+s} - X_s \stackrel{d}{=} X_t - X_0$ |

## 三、Poisson过程

### 3.1 定义与基本性质

**定义 3.1**（Poisson过程）计数过程 $\{N_t : t \geq 0\}$ 称为强度为 $\lambda > 0$ 的**Poisson过程**，若满足：
1. $N_0 = 0$（a.s.）
2. 独立增量：对 $0 \leq t_1 < t_2 < \cdots < t_n$，增量 $N_{t_2}-N_{t_1}, \ldots, N_{t_n}-N_{t_{n-1}}$ 相互独立
3. 平稳增量：$N_{t+s} - N_s \sim \text{Poisson}(\lambda t)$

**定理 3.2**（Poisson分布）对任意 $t \geq 0$ 和 $k \in \mathbb{N}_0$：
$$\mathbb{P}(N_t = k) = \frac{(\lambda t)^k e^{-\lambda t}}{k!}$$

**证明思路**：利用独立平稳增量性质，将 $[0,t]$ 分为 $n$ 等份，每份发生事件概率为 $\lambda t/n$。由二项分布逼近，当 $n \to \infty$ 时收敛到Poisson分布。

---

### 3.2 到达时间间隔与等待时间

设 $S_n$ 为第 $n$ 个事件的到达时间，$T_n = S_n - S_{n-1}$ 为到达时间间隔。

**定理 3.3**（到达时间间隔的分布）$T_1, T_2, \ldots$ 是独立同分布的指数随机变量：
$$T_n \sim \text{Exp}(\lambda), \quad f_{T_n}(t) = \lambda e^{-\lambda t}, \quad t \geq 0$$

**定理 3.4**（到达时间的分布）$S_n \sim \text{Gamma}(n, \lambda)$：
$$f_{S_n}(t) = \frac{\lambda^n t^{n-1} e^{-\lambda t}}{(n-1)!}, \quad t \geq 0$$

---

### 3.3 示例与应用

**示例 3.5**（放射性衰变）某放射性物质的衰变计数服从 $\lambda = 2.3$ 次/秒的Poisson过程。则：
- 10秒内恰好检测到20次衰变的概率：$\mathbb{P}(N_{10} = 20) = \frac{23^{20}e^{-23}}{20!} \approx 0.062$
- 平均等待时间：$\mathbb{E}[T_1] = 1/\lambda \approx 0.435$ 秒

**应用**：Poisson过程在排队论（到达顾客流）、保险数学（理赔次数）、网络流量建模等领域有广泛应用。

## 四、Markov链

### 4.1 离散时间Markov链

**定义 4.1**（Markov链）随机过程 $\{X_n : n \geq 0\}$ 称为**Markov链**，若满足**Markov性**：
$$\mathbb{P}(X_{n+1} = j \mid X_n = i, X_{n-1}, \ldots, X_0) = \mathbb{P}(X_{n+1} = j \mid X_n = i) = p_{ij}$$

其中 $P = (p_{ij})$ 为**转移概率矩阵**，满足：
$$p_{ij} \geq 0, \quad \sum_{j} p_{ij} = 1$$

**定义 4.2**（$n$步转移概率）定义 $p_{ij}^{(n)} = \mathbb{P}(X_n = j \mid X_0 = i)$。

**定理 4.3**（Chapman-Kolmogorov方程）
$$p_{ij}^{(m+n)} = \sum_{k} p_{ik}^{(m)} p_{kj}^{(n)}$$

**证明**：由Markov性和全概率公式，条件于 $X_m$ 的状态：
$$\begin{aligned}
p_{ij}^{(m+n)} &= \mathbb{P}(X_{m+n} = j \mid X_0 = i) \\
&= \sum_{k} \mathbb{P}(X_{m+n} = j, X_m = k \mid X_0 = i) \\
&= \sum_{k} p_{ik}^{(m)} p_{kj}^{(n)}
\end{aligned}$$

矩阵形式：$P^{(m+n)} = P^{(m)} P^{(n)}$，特别地 $P^{(n)} = P^n$。

---

### 4.2 状态分类与遍历理论

**定义 4.4**（可达与互通）状态 $j$ **可达**于 $i$（记 $i \to j$）若存在 $n \geq 0$ 使 $p_{ij}^{(n)} > 0$。若 $i \to j$ 且 $j \to i$，称 $i$ 与 $j$ **互通**（$i \leftrightarrow j$）。

**定义 4.5**（常返与暂态）设 $f_i = \mathbb{P}(\exists n \geq 1: X_n = i \mid X_0 = i)$：
- 若 $f_i = 1$，称 $i$ **常返**（Recurrent）
- 若 $f_i < 1$，称 $i$ **暂态**（Transient）

**定理 4.6**（常返判据）状态 $i$ 常返当且仅当 $\sum_{n=0}^{\infty} p_{ii}^{(n)} = \infty$。

**定理 4.7**（遍历定理）设Markov链不可约、非周期、正常返，则存在唯一平稳分布 $\pi = (\pi_j)$ 满足：
$$\pi_j = \sum_{i} \pi_i p_{ij}, \quad \lim_{n \to \infty} p_{ij}^{(n)} = \pi_j$$

---

### 4.3 示例：随机游走

**示例 4.8**（一维简单随机游走）$X_n = \sum_{i=1}^{n} \xi_i$，其中 $\xi_i$ i.i.d.，$\mathbb{P}(\xi_i = 1) = p$，$\mathbb{P}(\xi_i = -1) = q = 1-p$。

- 转移概率：$p_{i,i+1} = p$，$p_{i,i-1} = q$
- 当 $p = q = 1/2$（对称随机游走）：所有状态常返（一维、二维），暂态（三维及以上）
- **Polya定理**（1921）：对称随机游走在 $\mathbb{Z}^d$ 中常返当且仅当 $d \leq 2$

## 五、鞅（Martingale）

### 5.1 定义与基本性质

**定义 5.1**（鞅）适应于滤子 $\{\mathcal{F}_n\}$ 的随机过程 $\{M_n\}$ 称为**鞅**，若：
1. $\mathbb{E}[|M_n|] < \infty$（可积性）

2. $\mathbb{E}[M_{n+1} \mid \mathcal{F}_n] = M_n$（a.s.）（鞅性质）

若将等号替换为 $\geq$（$\leq$），分别称为**下鞅**（Submartingale）和**上鞅**（Supermartingale）。

**定义 5.2**（连续时间鞅）适应于滤子 $\{\mathcal{F}_t\}$ 的随机过程 $\{M_t : t \geq 0\}$ 称为鞅，若：
$$\mathbb{E}[M_t \mid \mathcal{F}_s] = M_s, \quad \forall s \leq t$$

---

### 5.2 重要例子

**示例 5.3**（简单随机游走）设 $\xi_i$ i.i.d.，$\mathbb{E}[\xi_i] = 0$，$\mathcal{F}_n = \sigma(\xi_1, \ldots, \xi_n)$。则：
$$M_n = \sum_{i=1}^{n} \xi_i$$
是鞅。若 $\mathbb{E}[\xi_i] = \mu \neq 0$，则 $M_n - n\mu$ 是鞅。

**示例 5.4**（Doob鞅）设 $X$ 为可积随机变量，则 $M_n = \mathbb{E}[X \mid \mathcal{F}_n]$ 是鞅。

**示例 5.5**（指数鞅）设 $B_t$ 为标准Brown运动，则：
$$M_t = \exp\left(\theta B_t - \frac{\theta^2 t}{2}\right)$$
是鞅（对任意 $\theta \in \mathbb{R}$）。

---

### 5.3 鞅收敛定理与停时定理

**定义 5.6**（停时）随机变量 $T: \Omega \to \{0, 1, 2, \ldots\} \cup \{\infty\}$ 称为**停时**，若：
$$\{T = n\} \in \mathcal{F}_n, \quad \forall n \geq 0$$

**定理 5.7**（Doob停时定理）设 $\{M_n\}$ 为鞅，$T$ 为有界停时，则：
$$\mathbb{E}[M_T] = \mathbb{E}[M_0]$$

**定理 5.8**（鞅收敛定理）设 $\{M_n\}$ 为下鞅且 $\sup_n \mathbb{E}[M_n^+] < \infty$，则 $M_n \to M_{\infty}$ a.s.，且 $\mathbb{E}[|M_{\infty}|] < \infty$。

---

### 5.4 鞅在赌博与金融中的应用

**示例 5.9**（公平赌博）设赌徒每局输赢概率各半，$M_n$ 为 $n$ 局后的资本。则 $\{M_n\}$ 是鞅，表示"公平"游戏。停时定理表明：无论采用何种停止策略，期望收益始终等于初始资本。

**示例 5.10**（期权定价）在金融数学中，折现后的股票价格过程在风险中性测度下是鞅，这是Black-Scholes期权定价理论的核心。

## 六、总结与参考文献

本讲系统介绍了随机过程的基础理论：
- **随机过程的一般理论**：Kolmogorov扩张定理、有限维分布
- **Poisson过程**：计数过程的典型代表，独立平稳增量
- **Markov链**：无记忆性、Chapman-Kolmogorov方程、遍历理论
- **鞅**：公平游戏的数学抽象，停时定理与收敛理论

这些理论构成了现代随机分析的基石，在物理学（统计力学）、生物学（种群动力学）、工程学（信号处理）、经济学（金融衍生品定价）等领域有深刻应用。
