# 概率统计练习题

本文档包含概率论与统计学的练习题，涵盖基础、进阶和挑战三个难度级别。

---

## 一、基础练习题（15道）

### 1.1 概率基础

**题1** 掷一枚公平硬币3次，求恰好出现2次正面的概率。

<details>
<summary>提示</summary>

使用二项分布公式 $P(X=k) = C_n^k p^k (1-p)^{n-k}$

</details>

**题2** 从一副52张的标准扑克牌中随机抽取一张，求抽到红心或A的概率。

<details>
<summary>提示</summary>

使用容斥原理：$P(A \cup B) = P(A) + P(B) - P(A \cap B)$

</details>

**题3** 设事件A和B独立，且 $P(A) = 0.4$，$P(B) = 0.5$，求 $P(A \cap B)$ 和 $P(A \cup B)$。

**题4** 某工厂生产的零件中，一等品占60%，二等品占30%，三等品占10%。随机抽取一个零件，求它不是三等品的概率。

### 1.2 随机变量

**题5** 设随机变量 $X$ 服从参数 $\lambda = 2$ 的泊松分布，求 $P(X = 3)$。

<details>
<summary>提示</summary>

泊松分布概率质量函数：$P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$

</details>

**题6** 设 $X \sim N(0, 1)$，求 $P(-1 < X < 1)$。（已知 $\Phi(1) = 0.8413$）

**题7** 随机变量 $X$ 的概率密度函数为 $f(x) = 2x$（$0 \leq x \leq 1$），求 $E[X]$ 和 $Var(X)$。

**题8** 设 $X$ 服从参数为 $n=10, p=0.3$ 的二项分布，求 $E[X]$ 和 $Var(X)$。

### 1.3 统计基础

**题9** 给定样本数据：5, 7, 8, 10, 12，求样本均值和样本方差。

**题10** 从总体中抽取容量为100的样本，样本均值为50，样本标准差为10。求总体均值的95%置信区间。

<details>
<summary>提示</summary>

置信区间公式：$\bar{x} \pm z_{\alpha/2} \cdot \frac{s}{\sqrt{n}}$

</details>

**题11** 设 $X_1, X_2, ..., X_n$ 是来自总体 $N(\mu, \sigma^2)$ 的简单随机样本，证明样本均值 $\bar{X}$ 是 $\mu$ 的无偏估计。

**题12** 掷两枚骰子，设 $X$ 为点数之和，求 $X$ 的分布列和期望值。

### 1.4 条件概率与贝叶斯

**题13** 某疾病在人群中的患病率为1%。检测该疾病的试纸准确率为：患者检测结果为阳性的概率为99%，健康人检测结果为阴性的概率为98%。若某人检测结果为阳性，求其实际患病的概率。

<details>
<summary>提示</summary>

应用贝叶斯公式：$P(D|+) = \frac{P(+|D)P(D)}{P(+)}$

</details>

**题14** 袋中有3个红球和2个白球。不放回地连续抽取两球，已知第一次抽到红球，求第二次抽到白球的概率。

**题15** 一个家庭有两个孩子，已知至少有一个是男孩，求两个孩子都是男孩的概率（假设生男生女概率相等）。

## 二、进阶练习题（15道）

### 2.1 多元分布与相关性

**题16** 设 $(X, Y)$ 服从二维正态分布，$E[X]=E[Y]=0$，$Var(X)=Var(Y)=1$，相关系数 $\rho=0.5$。求 $P(X > 0, Y > 0)$。

<details>
<summary>提示</summary>

利用对称性和相关系数的性质。

</details>

**题17** 设随机变量 $X$ 和 $Y$ 的联合密度函数为 $f(x,y) = 8xy$（$0 < x < y < 1$），求边缘密度 $f_X(x)$ 和条件密度 $f_{Y|X}(y|x)$。

**题18** 设 $X_1, X_2, ..., X_n$ 是独立同分布的随机变量，$E[X_i]=\mu$，$Var(X_i)=\sigma^2$。定义 $S_n = X_1 + ... + X_n$，求 $E[S_n^2]$。

**题19** 证明：对于任意随机变量 $X$ 和 $Y$，有 $|Corr(X,Y)| \leq 1$，等号成立当且仅当存在常数 $a \neq 0$ 和 $b$ 使得 $Y = aX + b$ 几乎必然成立。

**题20** 设 $X \sim Exp(\lambda)$，$Y = \min(X, c)$，其中 $c > 0$ 为常数。求 $E[Y]$。

### 2.2 极限定理

**题21** 设 $X_1, X_2, ..., X_{100}$ 是独立同分布的伯努利随机变量，$P(X_i=1)=0.6$。用中心极限定理近似计算 $P(\sum_{i=1}^{100} X_i > 55)$。

<details>
<summary>提示</summary>

标准化：$Z = \frac{\sum X_i - np}{\sqrt{np(1-p)}}$

</details>

**题22** 设 $X_1, X_2, ...$ 是独立同分布的随机变量，$E[X_i]=0$，$E[X_i^4] < \infty$。证明：$\frac{1}{n}\sum_{i=1}^n X_i^2 \xrightarrow{P} E[X_1^2]$。

**题23** 用特征函数证明：若 $X_n \xrightarrow{d} X$，$Y_n \xrightarrow{P} c$（常数），则 $X_n + Y_n \xrightarrow{d} X + c$。

### 2.3 参数估计

**题24** 设 $X_1, ..., X_n \sim N(\mu, \sigma^2)$，求 $\mu$ 和 $\sigma^2$ 的最大似然估计。

**题25** 设 $X_1, ..., X_n \sim Uniform[0, \theta]$，求 $\theta$ 的矩估计和最大似然估计，并比较它们的优劣。

**题26** 设 $X_1, ..., X_n \sim Poisson(\lambda)$，证明 $\hat{\lambda} = \bar{X}$ 是 $\lambda$ 的UMVUE。

**题27** 给定样本 $X_1, ..., X_n$ 来自密度函数 $f(x;\theta) = \theta x^{\theta-1}$（$0 < x < 1$）的总体，求 $\theta$ 的Fisher信息量。

### 2.4 假设检验

**题28** 某工厂声称其产品平均重量为100g。随机抽取25个产品，样本均值为98g，样本标准差为5g。在显著性水平 $\alpha = 0.05$ 下，检验工厂声明是否可信。

<details>
<summary>提示</summary>

使用t检验：$t = \frac{\bar{x} - \mu_0}{s/\sqrt{n}}$

</details>

**题29** 设 $X_1, ..., X_n \sim N(\mu, \sigma^2)$，其中 $\sigma^2$ 未知。构造检验 $H_0: \mu = \mu_0$ vs $H_1: \mu \neq \mu_0$ 的UMP检验。

**题30** 两个独立样本分别来自两个正态总体，样本容量分别为 $n_1=20$，$n_2=25$，样本方差分别为 $s_1^2=10$，$s_2^2=15$。检验两总体方差是否相等（$\alpha = 0.05$）。

## 三、挑战练习题（10道）

### 3.1 高级概率论

**题31** 设 $\{X_n\}$ 是鞅，$\tau$ 是关于其自然滤波的停时。证明在适当条件下，$E[X_\tau] = E[X_0]$（可选停时定理）。

<details>
<summary>提示</summary>

考虑有界停时 $\tau \wedge n$，利用鞅的收敛性。

</details>

**题32** 设 $B_t$ 是标准布朗运动，计算 $E[B_t^4]$ 和 $E[B_s^2 B_t^2]$（$s < t$）。

**题33** 证明Donsker不变原理：设 $\{X_i\}$ 是i.i.d.随机变量，$E[X_i]=0$，$Var(X_i)=1$。定义 $W_n(t) = \frac{1}{\sqrt{n}}\sum_{i=1}^{[nt]} X_i$，则 $W_n$ 弱收敛于标准布朗运动。

**题34** 设 $X$ 和 $Y$ 是独立的标准正态随机变量，求 $Z = X/Y$ 的分布（柯西分布）。

### 3.2 高级统计

**题35** 设 $X_1, ..., X_n \sim N(\mu, \sigma^2)$，证明样本方差 $S^2$ 与样本均值 $\bar{X}$ 独立。

<details>
<summary>提示</summary>

利用Basu定理或正交变换证明。

</details>

**题36** 设 $\hat{\theta}_n$ 是 $\theta$ 的估计量，满足 $\sqrt{n}(\hat{\theta}_n - \theta) \xrightarrow{d} N(0, \sigma^2(\theta))$。构造 $\theta$ 的渐近置信区间，并比较不同估计量的渐近相对效率。

**题37** 考虑线性回归模型 $Y = X\beta + \varepsilon$，其中 $\varepsilon \sim N(0, \sigma^2 I)$。证明最小二乘估计 $\hat{\beta} = (X^TX)^{-1}X^TY$ 是 $\beta$ 的BLUE（最佳线性无偏估计）。

### 3.3 随机过程

**题38** 设 $\{N_t\}$ 是速率为 $\lambda$ 的泊松过程，证明：
(a) 增量的独立性和平稳性
(b) 到达时间间隔服从指数分布
(c) 等待时间 $S_n$ 服从Gamma分布

**题39** 设 $X_t$ 是Ornstein-Uhlenbeck过程，满足 $dX_t = -\theta X_t dt + \sigma dB_t$。求 $X_t$ 的平稳分布，并计算自相关函数。

**题40** 考虑离散时间马尔可夫链，状态空间为有限集 $\{1, 2, ..., n\}$。证明：若链是不可约且非周期的，则存在唯一的平稳分布 $\pi$，且对任意初始分布，$P(X_n = j) \to \pi_j$。

## 参考答案

### 基础题答案

**题1** $C_3^2 (\frac{1}{2})^2 (\frac{1}{2})^1 = \frac{3}{8}$

**题2** $P(红心) + P(A) - P(红心A) = \frac{13}{52} + \frac{4}{52} - \frac{1}{52} = \frac{16}{52} = \frac{4}{13}$

**题3** $P(A \cap B) = 0.4 \times 0.5 = 0.2$；$P(A \cup B) = 0.4 + 0.5 - 0.2 = 0.7$

**题4** $P(不是三等品) = 1 - 0.1 = 0.9$

**题5** $P(X=3) = \frac{2^3 e^{-2}}{3!} = \frac{8e^{-2}}{6} = \frac{4e^{-2}}{3} \approx 0.180$

**题6** $P(-1 < X < 1) = 2\Phi(1) - 1 = 2 \times 0.8413 - 1 = 0.6826$

**题7** $E[X] = \int_0^1 x \cdot 2x dx = \frac{2}{3}$；$E[X^2] = \int_0^1 x^2 \cdot 2x dx = \frac{1}{2}$；$Var(X) = \frac{1}{2} - (\frac{2}{3})^2 = \frac{1}{18}$

**题8** $E[X] = np = 3$；$Var(X) = np(1-p) = 2.1$

**题9** 均值 $\bar{x} = 8.4$；方差 $s^2 = \frac{1}{4}\sum(x_i - \bar{x})^2 = 7.3$

**题10** $50 \pm 1.96 \times \frac{10}{\sqrt{100}} = 50 \pm 1.96 = [48.04, 51.96]$

**题11** $E[\bar{X}] = E[\frac{1}{n}\sum X_i] = \frac{1}{n}\sum E[X_i] = \frac{1}{n} \cdot n\mu = \mu$

**题12** $E[X] = 7$；分布：$P(X=2)=P(X=12)=\frac{1}{36}$，$P(X=3)=P(X=11)=\frac{2}{36}$，等等

**题13** $P(D|+) = \frac{0.99 \times 0.01}{0.99 \times 0.01 + 0.02 \times 0.99} = \frac{0.0099}{0.0297} = \frac{1}{3}$

**题14** $P(白|第一次红) = \frac{2}{4} = \frac{1}{2}$

**题15** 样本空间为 $\{BB, BG, GB, GG\}$，已知至少一个男孩，排除 $GG$，剩下3种等可能情况，故 $P(BB) = \frac{1}{3}$

### 进阶题答案（简要）

**题16** $P(X>0, Y>0) = \frac{1}{4} + \frac{1}{2\pi}\arcsin(\rho) = \frac{1}{4} + \frac{1}{2\pi}\arcsin(0.5) = \frac{1}{4} + \frac{1}{12} = \frac{1}{3}$

**题17** $f_X(x) = \int_x^1 8xy dy = 4x(1-x^2)$；$f_{Y|X}(y|x) = \frac{2y}{1-x^2}$

**题18** $E[S_n^2] = Var(S_n) + (E[S_n])^2 = n\sigma^2 + n^2\mu^2$

**题21** $E[\sum X_i] = 60$，$Var(\sum X_i) = 24$，$Z = \frac{55.5-60}{\sqrt{24}} \approx -0.92$，$P(Z > -0.92) \approx 0.8212$

**题24** $\hat{\mu} = \bar{X}$，$\hat{\sigma}^2 = \frac{1}{n}\sum(X_i - \bar{X})^2$

**题25** MLE: $\hat{\theta} = X_{(n)}$；矩估计: $\hat{\theta} = 2\bar{X}$。MLE更有效但不无偏。

**题28** $t = \frac{98-100}{5/5} = -2$，$|t| = 2 > t_{0.025,24} = 2.064$（边界），在0.05水平拒绝 $H_0$。

## 学习建议

1. **基础题**：重点掌握概率计算、分布性质和基本统计概念
2. **进阶题**：深入理解多元分布、极限定理和推断方法
3. **挑战题**：需要扎实的测度论基础和现代概率论知识

建议按照难度循序渐进，先确保基础题完全掌握，再挑战高级题目。
