# 随机变量与数字特征

## 一、概念深度解析

### 1.1 直观理解

随机变量是概率论中最核心的概念之一，它将抽象的随机现象转化为可计算的数学对象。直观上，随机变量就像是一个"数值化的随机实验结果"——它将样本空间中的每个结果映射到一个实数（或向量），使得我们可以使用微积分、线性代数等强大工具来研究随机现象。

数字特征则是对随机变量"整体特性"的数值概括。期望描述"长期平均位置"，方差刻画"波动程度"，偏度反映"不对称性"，峰度衡量"尾部厚度"。这些数字特征就像随机变量的"身份证"，在不知道完整分布的情况下也能提供重要信息。

### 1.2 形式化定义

**定义 1.1**（随机变量）：设 $(\Omega, \Sigma, \mathbb{P})$ 为概率空间，映射 $X: \Omega \to \mathbb{R}$ 称为**随机变量**，若对任意 Borel 集 $B \in \mathcal{B}(\mathbb{R})$：

$$X^{-1}(B) = \{\omega \in \Omega : X(\omega) \in B\} \in \Sigma$$

等价地，只需验证对所有 $x \in \mathbb{R}$，$\{X \leq x\} \in \Sigma$。

**定义 1.2**（分布函数）：随机变量 $X$ 的**分布函数**（CDF）定义为：

$$F_X(x) = \mathbb{P}(X \leq x), \quad x \in \mathbb{R}$$

**定义 1.3**（期望）：随机变量 $X$ 的**期望**定义为：
- 离散型：$\mathbb{E}[X] = \sum_i x_i p_X(x_i)$
- 连续型：$\mathbb{E}[X] = \int_{-\infty}^{\infty} x f_X(x) dx$
- 一般形式：$\mathbb{E}[X] = \int_{\Omega} X(\omega) d\mathbb{P}(\omega)$（Lebesgue积分）

**定义 1.4**（方差）：$\text{Var}(X) = \mathbb{E}[(X - \mathbb{E}[X])^2] = \mathbb{E}[X^2] - (\mathbb{E}[X])^2$

### 1.3 等价表述

**等价表述 1.5**（通过特征函数）：随机变量 $X$ 的**特征函数**定义为：

$$\varphi_X(t) = \mathbb{E}[e^{itX}] = \int_{-\infty}^{\infty} e^{itx} dF_X(x)$$

特征函数唯一确定分布，且 $\varphi_X^{(k)}(0) = i^k \mathbb{E}[X^k]$。

**等价表述 1.6**（通过矩母函数）：若存在 $h > 0$ 使得 $\mathbb{E}[e^{tX}] < \infty$ 对 $|t| < h$ 成立，则**矩母函数** $M_X(t) = \mathbb{E}[e^{tX}]$ 存在，且 $M_X^{(k)}(0) = \mathbb{E}[X^k]$。

### 1.4 动机与背景

随机变量的概念源于将赌博问题中的"点数"、"金额"等数量化。Chebyshev在19世纪中叶首次系统使用"随机变量"术语。Kolmogorov在1933年的公理化工作中，将随机变量定义为可测函数，奠定了现代概率论的基础。

期望的概念历史更悠久，从赌博中的"公平价值"问题发展而来。Huygens在1657年的著作中讨论了期望值的计算。方差的概念则由Fisher在20世纪初正式引入。

## 二、属性与关系

### 2.1 核心性质与证明

**定理 2.1**（分布函数的特征性质）：函数 $F: \mathbb{R} \to [0,1]$ 是某个随机变量的分布函数当且仅当：
1. **单调不减**：若 $x_1 < x_2$，则 $F(x_1) \leq F(x_2)$
2. **右连续**：$\lim_{y \to x^+} F(y) = F(x)$，$\forall x \in \mathbb{R}$
3. **极限条件**：$\lim_{x \to -\infty} F(x) = 0$，$\lim_{x \to +\infty} F(x) = 1$

**证明**（必要性）：
1. 若 $x_1 < x_2$，则 $\{X \leq x_1\} \subseteq \{X \leq x_2\}$，由概率单调性得 $F(x_1) \leq F(x_2)$
2. 令 $A_n = \{X \leq x + \frac{1}{n}\}$，则 $A_n \downarrow \{X \leq x\}$，由连续性得右连续
3. 令 $B_n = \{X \leq -n\}$，则 $B_n \downarrow \emptyset$，故 $\lim_{x \to -\infty} F(x) = 0$

**定理 2.2**（期望的线性性）：设 $X, Y$ 为随机变量，$a, b \in \mathbb{R}$，则：

$$\mathbb{E}[aX + bY] = a\mathbb{E}[X] + b\mathbb{E}[Y]$$

**证明**：由Lebesgue积分的线性性直接可得。

**定理 2.3**（方差的性质）：
1. $\text{Var}(aX + b) = a^2 \text{Var}(X)$
2. $\text{Var}(X) \geq 0$，等号成立当且仅当 $X = c$ a.s.
3. $\text{Var}(X) = \mathbb{E}[X^2] - (\mathbb{E}[X])^2$

**定理 2.4**（切比雪夫不等式）：设 $X$ 有有限方差，则对任意 $\epsilon > 0$：

$$\mathbb{P}(|X - \mathbb{E}[X]| \geq \epsilon) \leq \frac{\text{Var}(X)}{\epsilon^2}$$

**证明**：

$$\text{Var}(X) = \mathbb{E}[(X - \mu)^2] \geq \mathbb{E}[(X - \mu)^2 \mathbf{1}_{|X-\mu| \geq \epsilon}] \geq \epsilon^2 \mathbb{P}(|X - \mu| \geq \epsilon)$$

## 三、示例与习题

### 3.1 基础示例

**例 3.1**（伯努利随机变量）：$X \sim \text{Bernoulli}(p)$，$\mathbb{P}(X=1) = p$，$\mathbb{P}(X=0) = 1-p$。

$\mathbb{E}[X] = p$，$\text{Var}(X) = p(1-p)$

**例 3.2**（均匀分布）：$X \sim \text{Uniform}(a, b)$，$f_X(x) = \frac{1}{b-a}$，$x \in [a, b]$。

$\mathbb{E}[X] = \frac{a+b}{2}$，$\text{Var}(X) = \frac{(b-a)^2}{12}$

### 3.2 典型示例

**例 3.3**（正态分布的数字特征）：$X \sim \mathcal{N}(\mu, \sigma^2)$。

$\mathbb{E}[X] = \mu$，$\text{Var}(X) = \sigma^2$，偏度 = 0，峰度 = 3

**例 3.4**（柯西分布的异常性质）：$X$ 有密度 $f(x) = \frac{1}{\pi(1+x^2)}$。

**性质**：期望不存在！（积分发散）

### 3.3 进阶示例

**例 3.5**（Wald等式）：设 $\{X_i\}$ 为i.i.d.，$N$ 为取非负整数值的随机变量且与 $\{X_i\}$ 独立。则：

$$\mathbb{E}\left[\sum_{i=1}^N X_i\right] = \mathbb{E}[N] \cdot \mathbb{E}[X_1]$$

### 3.4 反例

**反例 3.6**（期望不存在的例子）：设 $X$ 的分布为 $\mathbb{P}(X = (-1)^n n) = \frac{6}{\pi^2 n^2}$，$n = 1, 2, \ldots$

$$\sum_{n=1}^{\infty} |x_n| p_n = \frac{6}{\pi^2} \sum_{n=1}^{\infty} \frac{1}{n} = \infty$$

期望不存在（不绝对收敛）。

**反例 3.7**（不相关但不独立）：设 $X \sim \text{Uniform}(-1, 1)$，$Y = X^2$。

$$\text{Cov}(X, Y) = \mathbb{E}[XY] - \mathbb{E}[X]\mathbb{E}[Y] = \mathbb{E}[X^3] - 0 = 0$$

$X$ 与 $Y$ 不相关，但显然不独立。

### 3.5 习题与解答

**习题 3.1**（基础）：设 $X \sim \text{Binomial}(n, p)$，求 $\mathbb{E}[X]$ 和 $\text{Var}(X)$。

**解**：$X = \sum_{i=1}^n X_i$，$X_i \sim \text{Bernoulli}(p)$ i.i.d.

$\mathbb{E}[X] = np$，$\text{Var}(X) = np(1-p)$

**习题 3.2**（中等）：设 $X$ 为非负整数值随机变量，证明：

$$\mathbb{E}[X] = \sum_{n=0}^{\infty} \mathbb{P}(X > n) = \sum_{n=1}^{\infty} \mathbb{P}(X \geq n)$$

**习题 3.3**（进阶）：证明对于任意随机变量 $X$（具有有限期望），

$$\mathbb{E}[X] = \int_0^{\infty} \mathbb{P}(X > t) dt - \int_{-\infty}^0 \mathbb{P}(X < t) dt$$

**习题 3.4**（挑战）：设 $\{X_n\}$ 为i.i.d.，$\mathbb{E}[X_1] = 0$，$\text{Var}(X_1) = 1$。定义 $S_n = \sum_{i=1}^n X_i$。求 $\mathbb{E}[S_n^4]$。

**习题 3.5**（综合）：设 $X$ 和 $Y$ 为随机变量，证明：

$$\text{Var}(X) = \mathbb{E}[\text{Var}(X|Y)] + \text{Var}(\mathbb{E}[X|Y])$$

（方差分解公式/全方差公式）

## 四、形式化实现（Lean 4）

```lean4
import Mathlib

namespace RandomVariable

-- 随机变量的定义（作为可测函数）
abbrev RandomVariable (Ω : Type*) [MeasurableSpace Ω] := Ω → ℝ

-- 期望的定义（使用Lebesgue积分）
noncomputable def expectation {Ω : Type*} [MeasurableSpace Ω] 
    (X : RandomVariable Ω) (ℙ : Measure Ω) : ℝ :=
  ∫ ω, X ω ∂ℙ

notation "𝔼[" X "; " ℙ "]" => expectation X ℙ

-- 方差的定义
noncomputable def variance {Ω : Type*} [MeasurableSpace Ω]
    (X : RandomVariable Ω) (ℙ : Measure Ω) : ℝ :=
  𝔼[(λ ω => (X ω - 𝔼[X; ℙ])^2); ℙ]

notation "Var[" X "; " ℙ "]" => variance X ℙ

-- 期望的线性性
lemma expectation_linear {Ω : Type*} [MeasurableSpace Ω] {ℙ : Measure Ω}
    (X Y : RandomVariable Ω) (a b : ℝ)
    (hX : Integrable X ℙ) (hY : Integrable Y ℙ) :
    𝔼[(λ ω => a * X ω + b * Y ω); ℙ] = a * 𝔼[X; ℙ] + b * 𝔼[Y; ℙ] := by
  simp [expectation]
  rw [integral_add, integral_add]
  · rw [integral_mul_left, integral_mul_left]
  · exact Integrable.mul_const hX a
  · exact Integrable.mul_const hY b
  · exact Integrable.add (Integrable.mul_const hX a) (Integrable.mul_const hY b)

end RandomVariable
```

## 五、应用与拓展

### 5.1 实际应用

**金融工程**：Black-Scholes期权定价模型假设股票价格服从几何布朗运动，其中漂移参数（期望收益率）和波动率（标准差）是核心输入。

**信号处理**：信噪比（SNR）定义为信号功率与噪声功率之比，本质是随机过程二阶矩的比值。

**质量控制**：过程能力指数 $C_p = \frac{\text{USL} - \text{LSL}}{6\sigma}$ 使用标准差衡量生产过程的稳定性。

### 5.2 与其他分支的联系

- **统计学**：样本均值和样本方差是总体参数的估计量
- **信息论**：熵 $\mathbb{E}[-\log p(X)]$ 是随机变量信息量的期望
- **统计物理**：系综平均（期望）连接微观状态和宏观可观测

### 5.3 前沿方向

**高维概率**：研究随机向量在高维空间中的性质，如集中不等式、随机矩阵理论。

**强化学习**：价值函数定义为累积奖励的期望，Bellman方程建立了期望的递归关系。
