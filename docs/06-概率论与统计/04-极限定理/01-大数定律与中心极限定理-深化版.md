# 大数定律与中心极限定理 - 深度深化版

## 一、概念深度解析

### 1.1 直观理解

大数定律（Law of Large Numbers, LLN）和中心极限定理（Central Limit Theorem, CLT）是概率论的两大支柱，它们揭示了随机现象背后的确定性与规律性。

大数定律告诉我们：当我们重复进行随机实验时，样本均值会"收敛"到理论期望值。这就是频率学派概率的数学基础——长期频率稳定于概率。弱大数定律说样本均值"依概率收敛"，而强大数定律则给出更强的"几乎必然收敛"。

中心极限定理则揭示了更深刻的规律：无论原始分布是什么形状（只要满足一定条件），大量独立随机变量之和的标准化形式都会趋近于正态分布。这就是为什么正态分布在自然界和统计学中如此普遍——它是"随机性累积"的普遍结果。

### 1.2 形式化定义

**定义 1.1**（收敛模式）：

- **几乎必然收敛**：$X_n \xrightarrow{\text{a.s.}} X$，若 $\mathbb{P}(\lim_{n \to \infty} X_n = X) = 1$
- **依概率收敛**：$X_n \xrightarrow{\mathbb{P}} X$，若 $\forall \epsilon > 0$，$\lim_{n \to \infty} \mathbb{P}(|X_n - X| > \epsilon) = 0$
- **依分布收敛**：$X_n \xrightarrow{d} X$，若 $F_{X_n}(x) \to F_X(x)$ 在所有连续点成立

**定义 1.2**（弱大数定律 - WLLN）：设 $\{X_n\}$ 为i.i.d.随机变量序列，$\mathbb{E}[X_1] = \mu$，则：

$$\bar{X}_n = \frac{1}{n}\sum_{i=1}^n X_i \xrightarrow{\mathbb{P}} \mu$$

**定义 1.3**（强大数定律 - SLLN）：在相同条件下：

$$\bar{X}_n \xrightarrow{\text{a.s.}} \mu$$

**定义 1.4**（中心极限定理 - CLT）：设 $\{X_n\}$ 为i.i.d.，$\mathbb{E}[X_1] = \mu$，$\text{Var}(X_1) = \sigma^2 < \infty$，则：

$$\frac{\sqrt{n}(\bar{X}_n - \mu)}{\sigma} \xrightarrow{d} \mathcal{N}(0, 1)$$

或等价地：

$$\frac{\sum_{i=1}^n X_i - n\mu}{\sigma\sqrt{n}} \xrightarrow{d} \mathcal{N}(0, 1)$$

### 1.3 等价表述

**等价表述 1.5**（特征函数形式）：CLT等价于：

$$\varphi_{\frac{S_n - n\mu}{\sigma\sqrt{n}}}(t) \to e^{-t^2/2}, \quad \forall t \in \mathbb{R}$$

**等价表述 1.6**（Berry-Esseen定理）：CLT的收敛速度估计：

$$\sup_x \left|\mathbb{P}\left(\frac{S_n - n\mu}{\sigma\sqrt{n}} \leq x\right) - \Phi(x)\right| \leq \frac{C\rho}{\sigma^3\sqrt{n}}$$

其中 $\rho = \mathbb{E}[|X_1 - \mu|^3]$，$C$ 为绝对常数（约0.4748）。

## 二、完整理论证明

### 2.1 弱大数定律的完整证明

**定理 2.1**（Chebyshev弱大数定律）：设 $\{X_n\}$ 两两不相关，$\mathbb{E}[X_i] = \mu$，$\text{Var}(X_i) \leq C < \infty$，则 $\bar{X}_n \xrightarrow{\mathbb{P}} \mu$。

**证明**：

首先计算样本均值的期望和方差：

$$\mathbb{E}[\bar{X}_n] = \mathbb{E}\left[\frac{1}{n}\sum_{i=1}^n X_i\right] = \frac{1}{n}\sum_{i=1}^n \mathbb{E}[X_i] = \frac{1}{n} \cdot n\mu = \mu$$

由于两两不相关：

$$\text{Var}(\bar{X}_n) = \text{Var}\left(\frac{1}{n}\sum_{i=1}^n X_i\right) = \frac{1}{n^2}\sum_{i=1}^n \text{Var}(X_i) \leq \frac{nC}{n^2} = \frac{C}{n}$$

应用Chebyshev不等式：

$$\mathbb{P}(|\bar{X}_n - \mu| \geq \epsilon) \leq \frac{\text{Var}(\bar{X}_n)}{\epsilon^2} \leq \frac{C}{n\epsilon^2}$$

当 $n \to \infty$ 时：

$$\lim_{n \to \infty} \mathbb{P}(|\bar{X}_n - \mu| \geq \epsilon) \leq \lim_{n \to \infty} \frac{C}{n\epsilon^2} = 0$$

因此 $\bar{X}_n \xrightarrow{\mathbb{P}} \mu$。 ∎

**定理 2.2**（Khinchin弱大数定律）：设 $\{X_n\}$ i.i.d.，$\mathbb{E}[|X_1|] < \infty$，$\mathbb{E}[X_1] = \mu$，则 $\bar{X}_n \xrightarrow{\mathbb{P}} \mu$。

**证明**（使用特征函数法）：

令 $\varphi(t) = \mathbb{E}[e^{itX_1}]$ 为 $X_1$ 的特征函数。

由于 $\mathbb{E}[|X_1|] < \infty$，特征函数在 $t=0$ 处可导：

$$\varphi(t) = 1 + it\mu + o(t), \quad t \to 0$$

样本均值 $\bar{X}_n$ 的特征函数：

$$\varphi_{\bar{X}_n}(t) = \mathbb{E}[e^{it\bar{X}_n}] = \mathbb{E}\left[e^{i\frac{t}{n}\sum_{j=1}^n X_j}\right] = \prod_{j=1}^n \mathbb{E}\left[e^{i\frac{t}{n}X_j}\right] = \left[\varphi\left(\frac{t}{n}\right)\right]^n$$

代入展开式：

$$\varphi_{\bar{X}_n}(t) = \left[1 + i\mu\frac{t}{n} + o\left(\frac{1}{n}\right)\right]^n \to e^{it\mu}, \quad n \to \infty$$

$e^{it\mu}$ 是退化分布 $\delta_\mu$ 的特征函数，因此：

$$\bar{X}_n \xrightarrow{d} \mu$$

由于依分布收敛到常数等价于依概率收敛：

$$\bar{X}_n \xrightarrow{\mathbb{P}} \mu \quad \text{∎}$$

### 2.2 强大数定律的完整证明

**定理 2.3**（Kolmogorov强大数定律）：设 $\{X_n\}$ i.i.d.，则：

$$\bar{X}_n \xrightarrow{\text{a.s.}} \mu \text{ 对某个有限 } \mu \iff \mathbb{E}[|X_1|] < \infty$$

当条件满足时，$\mu = \mathbb{E}[X_1]$。

**证明**：

**必要性**（$\Rightarrow$）：假设 $\bar{X}_n \xrightarrow{\text{a.s.}} \mu$。

注意到：

$$\frac{X_n}{n} = \bar{X}_n - \frac{n-1}{n}\bar{X}_{n-1} \to \mu - 1 \cdot \mu = 0 \quad \text{a.s.}$$

由Borel-Cantelli引理：

$$\sum_{n=1}^{\infty} \mathbb{P}(|X_n| \geq n) < \infty$$

由于同分布：

$$\sum_{n=1}^{\infty} \mathbb{P}(|X_1| \geq n) < \infty$$

这等价于：

$$\mathbb{E}[|X_1|] = \int_0^{\infty} \mathbb{P}(|X_1| > t) dt \leq \sum_{n=0}^{\infty} \mathbb{P}(|X_1| > n) < \infty$$

**充分性**（$\Leftarrow$）：设 $\mathbb{E}[|X_1|] < \infty$，$\mu = \mathbb{E}[X_1]$。

**步骤1**：截断。定义 $Y_n = X_n \mathbf{1}_{\{|X_n| \leq n\}}$。

证明 $\sum_{n=1}^{\infty} \mathbb{P}(X_n \neq Y_n) < \infty$：

$$\sum_{n=1}^{\infty} \mathbb{P}(|X_1| > n) \leq \mathbb{E}[|X_1|] < \infty$$

由Borel-Cantelli，$\mathbb{P}(X_n \neq Y_n \text{ i.o.}) = 0$，因此只需证明：

$$\frac{1}{n}\sum_{k=1}^n Y_k \xrightarrow{\text{a.s.}} \mu$$

**步骤2**：应用Kolmogorov三级数定理。

令 $Z_n = \frac{Y_n - \mathbb{E}[Y_n]}{n}$，证明 $\sum_{n=1}^{\infty} Z_n$ a.s.收敛。

验证三级数条件：

1. $\sum_{n=1}^{\infty} \mathbb{P}(|Z_n| > A) < \infty$（对足够大的 $A$ 自动满足）
2. $\sum_{n=1}^{\infty} \mathbb{E}[Z_n \mathbf{1}_{\{|Z_n| \leq A\}}] = 0$（由中心化）
3. $\sum_{n=1}^{\infty} \text{Var}(Z_n) < \infty$

对于条件3：

$$\sum_{n=1}^{\infty} \frac{\text{Var}(Y_n)}{n^2} \leq \sum_{n=1}^{\infty} \frac{\mathbb{E}[Y_n^2]}{n^2} = \sum_{n=1}^{\infty} \frac{\mathbb{E}[X_1^2 \mathbf{1}_{\{|X_1| \leq n\}}]}{n^2}$$

交换求和与期望：

$$= \mathbb{E}\left[X_1^2 \sum_{n=\lceil|X_1|\rceil}^{\infty} \frac{1}{n^2}\right]$$

利用 $\sum_{n=k}^{\infty} \frac{1}{n^2} \leq \frac{C}{k}$：

$$\leq C \cdot \mathbb{E}\left[\frac{X_1^2}{|X_1|}\right] = C \cdot \mathbb{E}[|X_1|] < \infty$$

由Kolmogorov三级数定理，$\sum_{n=1}^{\infty} \frac{Y_n - \mathbb{E}[Y_n]}{n}$ a.s.收敛。

**步骤3**：应用Kronecker引理。

若 $\sum_{n=1}^{\infty} \frac{a_n}{n}$ 收敛，则 $\frac{1}{n}\sum_{k=1}^n a_k \to 0$。

因此：

$$\frac{1}{n}\sum_{k=1}^n (Y_k - \mathbb{E}[Y_k]) \to 0 \quad \text{a.s.}$$

**步骤4**：证明 $\mathbb{E}[Y_n] \to \mu$。

$$\mathbb{E}[Y_n] = \mathbb{E}[X_1 \mathbf{1}_{\{|X_1| \leq n\}}] \to \mathbb{E}[X_1] = \mu$$

由控制收敛定理（$|X_1 \mathbf{1}_{\{|X_1| \leq n\}}| \leq |X_1|$ 且 $\mathbb{E}[|X_1|] < \infty$）。

因此：

$$\frac{1}{n}\sum_{k=1}^n Y_k = \frac{1}{n}\sum_{k=1}^n (Y_k - \mathbb{E}[Y_k]) + \frac{1}{n}\sum_{k=1}^n \mathbb{E}[Y_k] \to 0 + \mu = \mu \quad \text{a.s.}$$

综上，$\bar{X}_n \xrightarrow{\text{a.s.}} \mu$。 ∎

### 2.3 中心极限定理的完整证明

**定理 2.4**（经典中心极限定理 - Lindeberg-Lévy）：设 $\{X_n\}$ i.i.d.，$\mathbb{E}[X_1] = \mu$，$\text{Var}(X_1) = \sigma^2 < \infty$，则：

$$\frac{\sqrt{n}(\bar{X}_n - \mu)}{\sigma} \xrightarrow{d} \mathcal{N}(0, 1)$$

**证明**（特征函数法）：

**步骤1**：标准化。

令 $Y_i = \frac{X_i - \mu}{\sigma}$，则 $\mathbb{E}[Y_i] = 0$，$\text{Var}(Y_i) = 1$。

需证明：

$$\frac{1}{\sqrt{n}}\sum_{i=1}^n Y_i \xrightarrow{d} \mathcal{N}(0, 1)$$

**步骤2**：特征函数展开。

令 $\varphi(t) = \mathbb{E}[e^{itY_1}]$ 为 $Y_1$ 的特征函数。

由于 $\mathbb{E}[Y_1] = 0$，$\mathbb{E}[Y_1^2] = 1$，特征函数在 $t=0$ 处有二阶Taylor展开：

$$\varphi(t) = 1 + it\mathbb{E}[Y_1] - \frac{t^2}{2}\mathbb{E}[Y_1^2] + o(t^2) = 1 - \frac{t^2}{2} + o(t^2), \quad t \to 0$$

**步骤3**：标准化和的特征函数。

令 $S_n^* = \frac{1}{\sqrt{n}}\sum_{i=1}^n Y_i$，其特征函数：

$$\varphi_{S_n^*}(t) = \mathbb{E}[e^{itS_n^*}] = \mathbb{E}\left[e^{i\frac{t}{\sqrt{n}}\sum_{j=1}^n Y_j}\right] = \prod_{j=1}^n \mathbb{E}\left[e^{i\frac{t}{\sqrt{n}}Y_j}\right] = \left[\varphi\left(\frac{t}{\sqrt{n}}\right)\right]^n$$

**步骤4**：极限计算。

代入Taylor展开：

$$\varphi\left(\frac{t}{\sqrt{n}}\right) = 1 - \frac{t^2}{2n} + o\left(\frac{1}{n}\right)$$

因此：

$$\varphi_{S_n^*}(t) = \left[1 - \frac{t^2}{2n} + o\left(\frac{1}{n}\right)\right]^n$$

利用 $(1 + \frac{a}{n} + o(\frac{1}{n}))^n \to e^a$：

$$\lim_{n \to \infty} \varphi_{S_n^*}(t) = e^{-t^2/2}$$

**步骤5**：应用Lévy连续性定理。

$e^{-t^2/2}$ 是标准正态分布的特征函数，且在 $t=0$ 处连续。

由Lévy连续性定理：

$$S_n^* = \frac{\sqrt{n}(\bar{X}_n - \mu)}{\sigma} \xrightarrow{d} \mathcal{N}(0, 1) \quad \text{∎}$$

### 2.4 Lindeberg-Feller定理的完整证明

**定理 2.5**（Lindeberg-Feller CLT）：设 $\{X_{nk}\}$ 为独立随机变量三角阵列，$\mathbb{E}[X_{nk}] = 0$，$\text{Var}(X_{nk}) = \sigma_{nk}^2$。记 $s_n^2 = \sum_{k=1}^{n} \sigma_{nk}^2$。若满足**Lindeberg条件**：

$$\lim_{n \to \infty} \frac{1}{s_n^2} \sum_{k=1}^n \mathbb{E}[X_{nk}^2 \mathbf{1}_{\{|X_{nk}| > \epsilon s_n\}}] = 0, \quad \forall \epsilon > 0$$

则：

$$\frac{\sum_{k=1}^n X_{nk}}{s_n} \xrightarrow{d} \mathcal{N}(0, 1) \quad \text{且} \quad \max_{1 \leq k \leq n} \frac{\sigma_{nk}^2}{s_n^2} \to 0$$

**证明**：

**步骤1**：验证Feller条件的必要性。

若CLT成立，则每个 $X_{nk}/s_n$ 对和的贡献趋于0，这蕴含Feller条件。

**步骤2**：证明Lindeberg条件 $\Rightarrow$ CLT。

令 $Y_{nk} = X_{nk}/s_n$，需证明 $\sum_{k=1}^n Y_{nk} \xrightarrow{d} \mathcal{N}(0, 1)$。

设 $\varphi_{nk}$ 为 $Y_{nk}$ 的特征函数。

**步骤3**：利用特征函数近似。

利用不等式：$|e^{ix} - (1 + ix - \frac{x^2}{2})| \leq \min\{\frac{|x|^3}{6}, x^2\}$

对任意特征函数 $\varphi(t) = \mathbb{E}[e^{itX}]$：

$$\left|\varphi(t) - \left(1 - \frac{t^2\sigma^2}{2}\right)\right| \leq \mathbb{E}\left[\min\left\{\frac{|tX|^3}{6}, (tX)^2\right\}\right]$$

**步骤4**：应用Lindeberg条件。

将上式应用于 $\varphi_{nk}(t)$：

$$\left|\varphi_{nk}(t) - \left(1 - \frac{t^2\sigma_{nk}^2}{2s_n^2}\right)\right| \leq \mathbb{E}\left[\min\left\{\frac{|tY_{nk}|^3}{6}, (tY_{nk})^2\right\}\right]$$

利用Lindeberg条件可以证明：

$$\sum_{k=1}^n \left|\varphi_{nk}(t) - \left(1 - \frac{t^2\sigma_{nk}^2}{2s_n^2}\right)\right| \to 0$$

**步骤5**：比较乘积。

利用 $|\prod a_k - \prod b_k| \leq \sum |a_k - b_k|$（当 $|a_k|, |b_k| \leq 1$）：

$$\left|\prod_{k=1}^n \varphi_{nk}(t) - \prod_{k=1}^n \left(1 - \frac{t^2\sigma_{nk}^2}{2s_n^2}\right)\right| \to 0$$

**步骤6**：标准正态极限。

由于 $\sum_{k=1}^n \frac{\sigma_{nk}^2}{s_n^2} = 1$：

$$\prod_{k=1}^n \left(1 - \frac{t^2\sigma_{nk}^2}{2s_n^2}\right) \approx \exp\left(-\frac{t^2}{2}\sum_{k=1}^n \frac{\sigma_{nk}^2}{s_n^2}\right) = e^{-t^2/2}$$

因此：

$$\prod_{k=1}^n \varphi_{nk}(t) \to e^{-t^2/2}$$

由Lévy连续性定理，CLT成立。 ∎

## 三、Python模拟实验

### 3.1 大数定律模拟实验

```python
"""
大数定律模拟实验
演示样本均值如何收敛到理论期望值
"""

import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

# 设置中文字体
plt.rcParams['font.sans-serif'] = ['SimHei', 'DejaVu Sans']
plt.rcParams['axes.unicode_minus'] = False

def simulate_lln(distribution, n_max=10000, n_trials=100, **dist_params):
    """
    模拟大数定律

    参数:
    - distribution: 分布类型 ('uniform', 'exponential', 'binomial', 'cauchy')
    - n_max: 最大样本量
    - n_trials: 模拟次数
    - dist_params: 分布参数
    """
    sample_sizes = np.unique(np.logspace(1, np.log10(n_max), 100).astype(int))

    # 生成理论均值
    if distribution == 'uniform':
        a, b = dist_params.get('a', 0), dist_params.get('b', 1)
        true_mean = (a + b) / 2
        samples = np.random.uniform(a, b, (n_trials, n_max))
    elif distribution == 'exponential':
        scale = dist_params.get('scale', 1)
        true_mean = scale
        samples = np.random.exponential(scale, (n_trials, n_max))
    elif distribution == 'binomial':
        n, p = dist_params.get('n', 1), dist_params.get('p', 0.5)
        true_mean = n * p
        samples = np.random.binomial(n, p, (n_trials, n_max))
    elif distribution == 'cauchy':
        # Cauchy分布没有均值，作为反例
        true_mean = None
        samples = np.random.standard_cauchy((n_trials, n_max))
    else:
        raise ValueError(f"Unknown distribution: {distribution}")

    # 计算累积均值
    cumulative_means = np.cumsum(samples, axis=1) / np.arange(1, n_max + 1)

    return sample_sizes, cumulative_means[:, sample_sizes-1], true_mean

# 创建模拟图
fig, axes = plt.subplots(2, 2, figsize=(14, 10))

# 1. 均匀分布
ax = axes[0, 0]
sample_sizes, means, true_mean = simulate_lln('uniform', a=0, b=1)
for i in range(min(50, means.shape[0])):
    ax.plot(sample_sizes, means[i], alpha=0.3, color='blue')
ax.axhline(y=true_mean, color='red', linestyle='--', linewidth=2, label=f'True Mean = {true_mean}')
ax.set_xscale('log')
ax.set_xlabel('Sample Size (n)')
ax.set_ylabel('Sample Mean')
ax.set_title('Uniform(0,1): Strong Law of Large Numbers')
ax.legend()
ax.grid(True, alpha=0.3)

# 2. 指数分布
ax = axes[0, 1]
sample_sizes, means, true_mean = simulate_lln('exponential', scale=2)
for i in range(min(50, means.shape[0])):
    ax.plot(sample_sizes, means[i], alpha=0.3, color='green')
ax.axhline(y=true_mean, color='red', linestyle='--', linewidth=2, label=f'True Mean = {true_mean}')
ax.set_xscale('log')
ax.set_xlabel('Sample Size (n)')
ax.set_ylabel('Sample Mean')
ax.set_title('Exponential(λ=0.5): Strong Law of Large Numbers')
ax.legend()
ax.grid(True, alpha=0.3)

# 3. 二项分布
ax = axes[1, 0]
sample_sizes, means, true_mean = simulate_lln('binomial', n=10, p=0.3)
for i in range(min(50, means.shape[0])):
    ax.plot(sample_sizes, means[i], alpha=0.3, color='purple')
ax.axhline(y=true_mean, color='red', linestyle='--', linewidth=2, label=f'True Mean = {true_mean}')
ax.set_xscale('log')
ax.set_xlabel('Sample Size (n)')
ax.set_ylabel('Sample Mean')
ax.set_title('Binomial(10, 0.3): Strong Law of Large Numbers')
ax.legend()
ax.grid(True, alpha=0.3)

# 4. Cauchy分布（反例）
ax = axes[1, 1]
sample_sizes, means, _ = simulate_lln('cauchy')
for i in range(min(20, means.shape[0])):
    ax.plot(sample_sizes, means[i], alpha=0.5)
ax.set_xscale('log')
ax.set_xlabel('Sample Size (n)')
ax.set_ylabel('Sample Mean')
ax.set_title('Cauchy Distribution: NO Convergence (Counter-example)')
ax.grid(True, alpha=0.3)

plt.tight_layout()
plt.savefig('lln_simulation.png', dpi=150, bbox_inches='tight')
plt.show()

print("大数定律模拟完成！")
print("\n观察结果：")
print("1. 均匀分布、指数分布、二项分布：样本均值收敛到理论期望值")
print("2. Cauchy分布：样本均值不收敛，震荡剧烈（期望不存在）")
```

### 3.2 中心极限定理可视化

```python
"""
中心极限定理可视化实验
演示不同分布的标准化和如何收敛到标准正态分布
"""

import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

def clt_visualization(distribution, n_values=[1, 5, 30, 100], n_samples=10000, **dist_params):
    """
    可视化中心极限定理

    参数:
    - distribution: 分布类型
    - n_values: 样本量列表
    - n_samples: 模拟样本数
    """
    fig, axes = plt.subplots(2, 2, figsize=(14, 10))
    axes = axes.flatten()

    # 生成原始分布的理论均值和标准差
    if distribution == 'uniform':
        a, b = dist_params.get('a', 0), dist_params.get('b', 1)
        mu, sigma = (a + b) / 2, (b - a) / np.sqrt(12)
        title = f'Uniform({a},{b})'
        generator = lambda n: np.random.uniform(a, b, (n_samples, n))
    elif distribution == 'exponential':
        scale = dist_params.get('scale', 1)
        mu, sigma = scale, scale
        title = f'Exponential(λ={1/scale:.2f})'
        generator = lambda n: np.random.exponential(scale, (n_samples, n))
    elif distribution == 'binomial':
        n_trials, p = dist_params.get('n', 10), dist_params.get('p', 0.3)
        mu, sigma = n_trials * p, np.sqrt(n_trials * p * (1-p))
        title = f'Binomial({n_trials},{p})'
        generator = lambda n: np.random.binomial(n_trials, p, (n_samples, n))
    elif distribution == 'chi2':
        df = dist_params.get('df', 3)
        mu, sigma = df, np.sqrt(2 * df)
        title = f'Chi-squared({df})'
        generator = lambda n: np.random.chisquare(df, (n_samples, n))

    for idx, n in enumerate(n_values):
        ax = axes[idx]
```

### 3.3 收敛速度分析（Berry-Esseen界）

```python
"""
Berry-Esseen定理验证
分析中心极限定理的收敛速度
"""

import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

def berry_esseen_analysis(distribution, n_max=1000, n_samples=10000, **dist_params):
    """
    分析CLT收敛速度
    """
    n_values = np.logspace(1, np.log10(n_max), 30).astype(int)

    # 设置分布参数
    if distribution == 'uniform':
        a, b = dist_params.get('a', 0), dist_params.get('b', 1)
        mu, sigma, rho = (a + b) / 2, (b - a) / np.sqrt(12), (b - a)**3 / 32
        title = f'Uniform({a},{b})'
        generator = lambda n: np.random.uniform(a, b, (n_samples, n))
    elif distribution == 'exponential':
        scale = dist_params.get('scale', 1)
        mu, sigma = scale, scale
        rho = 2 * scale**3  # E[|X-mu|^3] for exponential
        title = f'Exponential(λ={1/scale:.2f})'
        generator = lambda n: np.random.exponential(scale, (n_samples, n))
    elif distribution == 'bernoulli':
        p = dist_params.get('p', 0.5)
        mu, sigma = p, np.sqrt(p * (1-p))
        rho = p*(1-p)*(p**2 + (1-p)**2)
        title = f'Bernoulli({p})'
        generator = lambda n: np.random.binomial(1, p, (n_samples, n))

    ks_stats = []
    berry_esseen_bounds = []

    for n in n_values:
        samples = generator(n)
        standardized = (np.sum(samples, axis=1) - n * mu) / (sigma * np.sqrt(n))

        # KS统计量
        ks_stat, _ = stats.kstest(standardized, 'norm')
        ks_stats.append(ks_stat)

        # Berry-Esseen理论界
        bound = 0.4748 * rho / (sigma**3 * np.sqrt(n))
        berry_esseen_bounds.append(bound)

    # 绘图
    fig, axes = plt.subplots(1, 2, figsize=(14, 5))

    # 左图：KS统计量随n变化
    ax = axes[0]
    ax.plot(n_values, ks_stats, 'b-o', label='Empirical KS Statistic', markersize=4)
    ax.plot(n_values, berry_esseen_bounds, 'r--', label='Berry-Esseen Bound', linewidth=2)
    ax.set_xscale('log')
    ax.set_yscale('log')
    ax.set_xlabel('Sample Size (n)')
    ax.set_ylabel('KS Statistic / BE Bound')
    ax.set_title(f'{title}: CLT Convergence Rate')
    ax.legend()
    ax.grid(True, alpha=0.3)

    # 右图：实际误差 vs 理论界
    ax = axes[1]
    ratio = np.array(ks_stats) / np.array(berry_esseen_bounds)
    ax.plot(n_values, ratio, 'g-s', label='Empirical / BE Bound', markersize=4)
    ax.axhline(y=1, color='red', linestyle='--', label='y=1 (theoretical limit)')
    ax.set_xscale('log')
    ax.set_xlabel('Sample Size (n)')
    ax.set_ylabel('Ratio')
    ax.set_title(f'{title}: Empirical vs Theory')
    ax.legend()
    ax.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.savefig(f'berry_esseen_{distribution}.png', dpi=150, bbox_inches='tight')
    plt.show()

    return n_values, ks_stats, berry_esseen_bounds

# 运行分析
print("=" * 60)
print("Berry-Esseen收敛速度分析")
print("=" * 60)

for dist_name, params in [
    ('uniform', {'a': 0, 'b': 1}),
    ('exponential', {'scale': 1}),
    ('bernoulli', {'p': 0.3})
]:
    print(f"\n分析分布: {dist_name}")
    n_vals, ks, be = berry_esseen_analysis(dist_name, **params)
```

```r
simulate_lln_r <- function(n_max = 10000, n_paths = 100, distribution = "normal") {

        true_mean = true_mean,
```

**习题 6.1**：设 $X_1, X_2, \ldots, X_n$ 是i.i.d.随机变量，$\mathbb{E}[X_1] = \mu$，$\text{Var}(X_1) = \sigma^2$。使用Chebyshev不等式证明：对于任意 $\epsilon > 0$，

$$\mathbb{P}(|\bar{X}_n - \mu| \geq \epsilon) \leq \frac{\sigma^2}{n\epsilon^2}$$

并由此推出弱大数定律。

**解答**：

由Chebyshev不等式，对任意具有有限方差的随机变量 $Y$：

$$\mathbb{P}(|Y - \mathbb{E}[Y]| \geq \epsilon) \leq \frac{\text{Var}(Y)}{\epsilon^2}$$

取 $Y = \bar{X}_n = \frac{1}{n}\sum_{i=1}^n X_i$，则 $\mathbb{E}[\bar{X}_n] = \mu$，且：

$$\text{Var}(\bar{X}_n) = \frac{1}{n^2}\sum_{i=1}^n \text{Var}(X_i) = \frac{\sigma^2}{n}$$

因此：

$$\mathbb{P}(|\bar{X}_n - \mu| \geq \epsilon) \leq \frac{\sigma^2}{n\epsilon^2} \to 0 \quad (n \to \infty)$$

即 $\bar{X}_n \xrightarrow{\mathbb{P}} \mu$。 ∎

**习题 6.2**：设 $X_1, X_2, \ldots$ 是i.i.d. Bernoulli($p$)随机变量。使用特征函数法证明：

$$\frac{\sum_{i=1}^n X_i - np}{\sqrt{np(1-p)}} \xrightarrow{d} \mathcal{N}(0, 1)$$

**解答**：

Bernoulli($p$)的特征函数为：

$$\varphi(t) = pe^{it} + (1-p) = 1 + p(e^{it} - 1)$$

标准化变量 $Y_i = \frac{X_i - p}{\sqrt{p(1-p)}}$ 的特征函数：

$$\varphi_Y(t) = e^{-itp/\sqrt{p(1-p)}}[pe^{it/\sqrt{p(1-p)}} + (1-p)]$$

Taylor展开：

$$\varphi_Y(t) = 1 - \frac{t^2}{2} + o(t^2)$$

标准化和的特征函数：

$$\varphi_{S_n^*}(t) = \left[\varphi_Y\left(\frac{t}{\sqrt{n}}\right)\right]^n = \left[1 - \frac{t^2}{2n} + o(n^{-1})\right]^n \to e^{-t^2/2}$$

由Lévy连续性定理得证。 ∎

**习题 6.3**：设 $\{X_n\}$ i.i.d.，$\mathbb{E}[X_1] = 0$，$\text{Var}(X_1) = 1$。求最小的 $n$ 使得：

$$\mathbb{P}(|\bar{X}_n| < 0.1) > 0.95$$

**解答**：

由CLT，$\sqrt{n}\bar{X}_n \approx \mathcal{N}(0, 1)$。

$$\mathbb{P}(|\bar{X}_n| < 0.1) = \mathbb{P}(|\sqrt{n}\bar{X}_n| < 0.1\sqrt{n}) \approx 2\Phi(0.1\sqrt{n}) - 1 > 0.95$$

因此：

$$\Phi(0.1\sqrt{n}) > 0.975$$

$$0.1\sqrt{n} > 1.96$$

$$n > \left(\frac{1.96}{0.1}\right)^2 = 384.16$$

最小整数 $n = 385$。 ∎

**习题 6.4**：证明：几乎必然收敛蕴含依概率收敛，但反之不成立。

**解答**：

**（$\Rightarrow$）** 设 $X_n \xrightarrow{\text{a.s.}} X$。对任意 $\epsilon > 0$：

$$\{|X_n - X| > \epsilon\} \subseteq \bigcup_{m=n}^{\infty}\{|X_m - X| > \epsilon\}$$

由于 $X_n \to X$ a.s.，右边集合的概率趋于0（单调收敛）。因此：

$$\mathbb{P}(|X_n - X| > \epsilon) \to 0$$

**（反例）** 设 $\Omega = [0, 1]$ 均匀分布。定义：

$$X_1 = \mathbf{1}_{[0,1]}, X_2 = \mathbf{1}_{[0,1/2]}, X_3 = \mathbf{1}_{[1/2,1]}, X_4 = \mathbf{1}_{[0,1/3]}, \ldots$$

则 $X_n \xrightarrow{\mathbb{P}} 0$（因为 $\mathbb{P}(|X_n| > 0) \to 0$），但对任意 $\omega$，$X_n(\omega) = 1$ 对无穷多个 $n$ 成立，故不a.s.收敛。 ∎

**习题 6.5**：设 $X \sim \text{Binomial}(100, 0.3)$，使用正态近似计算 $\mathbb{P}(25 \leq X \leq 35)$，并与精确值比较。

**解答**：

**精确计算**：

$$\mathbb{P}(25 \leq X \leq 35) = \sum_{k=25}^{35} \binom{100}{k} 0.3^k 0.7^{100-k}$$

**正态近似**：$\mu = 30$，$\sigma^2 = 21$，$\sigma = 4.583$

使用连续性校正：

$$\mathbb{P}(25 \leq X \leq 35) \approx \Phi\left(\frac{35.5 - 30}{4.583}\right) - \Phi\left(\frac{24.5 - 30}{4.583}\right)$$

$$= \Phi(1.20) - \Phi(-1.20) = 2\Phi(1.20) - 1 = 0.7698$$

**Python验证**：

```python
from scipy import stats

# 精确值
exact = stats.binom.cdf(35, 100, 0.3) - stats.binom.cdf(24, 100, 0.3)
print(f"精确值: {exact:.4f}")

# 正态近似
mu, sigma = 30, np.sqrt(21)
approx = stats.norm.cdf(35.5, mu, sigma) - stats.norm.cdf(24.5, mu, sigma)
print(f"正态近似: {approx:.4f}")
print(f"误差: {abs(exact - approx):.4f}")
```

**习题 6.6**：设 $\{X_n\}$ 是独立随机变量，$\mathbb{P}(X_n = n) = \mathbb{P}(X_n = -n) = \frac{1}{2n^2}$，$\mathbb{P}(X_n = 0) = 1 - \frac{1}{n^2}$。验证弱大数定律是否成立。

**解答**：

计算期望和方差：

$$\mathbb{E}[X_n] = n \cdot \frac{1}{2n^2} + (-n) \cdot \frac{1}{2n^2} + 0 = 0$$

$$\mathbb{E}[X_n^2] = n^2 \cdot \frac{1}{2n^2} + n^2 \cdot \frac{1}{2n^2} = 1$$

因此 $\text{Var}(X_n) = 1$。

由于方差一致有界，由Chebyshev弱大数定律：

$$\frac{1}{n}\sum_{i=1}^n X_i \xrightarrow{\mathbb{P}} 0$$

WLLN成立。 ∎

**习题 6.7**：用Monte Carlo方法估计 $\pi$，分析误差收敛速度。

**解答**：

向单位正方形 $[0,1]^2$ 随机投点，落在单位圆内的概率为 $\frac{\pi}{4}$。

设投 $n$ 次，落入圆内的次数为 $S_n$，则：

$$\hat{\pi}_n = 4 \cdot \frac{S_n}{n}$$

由SLLN，$\hat{\pi}_n \xrightarrow{\text{a.s.}} \pi$。

由CLT：

$$\sqrt{n}(\hat{\pi}_n - \pi) \xrightarrow{d} \mathcal{N}(0, 16 \cdot \frac{\pi}{4} \cdot (1 - \frac{\pi}{4}))$$

因此标准差为 $O(n^{-1/2})$，即误差以 $n^{-1/2}$ 速度衰减。

要达到精度 $\epsilon$，需要 $n \approx \frac{16 \cdot \frac{\pi}{4} \cdot (1 - \frac{\pi}{4})}{\epsilon^2} \approx \frac{2.7}{\epsilon^2}$。

**习题 6.8**：设 $X_1, X_2, \ldots$ i.i.d. 具有密度 $f(x) = \frac{1}{2}e^{-|x-\theta|}$（Laplace分布）。

(a) 求 $\theta$ 的MLE。
(b) 证明样本中位数 $\hat{\theta}_n$ 是相合估计。

**解答**：

**(a)** 似然函数：

$$L(\theta) = \prod_{i=1}^n \frac{1}{2}e^{-|X_i - \theta|} = 2^{-n} e^{-\sum_{i=1}^n |X_i - \theta|}$$

对数似然：$\ell(\theta) = -n\ln 2 - \sum_{i=1}^n |X_i - \theta|$

最大化 $\ell(\theta)$ 等价于最小化 $\sum_{i=1}^n |X_i - \theta|$，其解为样本中位数。

**(b)** 设 $m$ 为总体中位数（对于Laplace分布，$m = \theta$）。样本中位数 $\hat{m}_n$ 满足：

$$\hat{m}_n \xrightarrow{\mathbb{P}} m$$

这是由大数定律和顺序统计量的性质保证的。 ∎

### 6.2 中等难度习题（9-14）

**习题 6.9**（Lindeberg条件验证）：设 $\{X_{nk}\}$ 为独立随机变量三角阵列，$\mathbb{E}[X_{nk}] = 0$，$\text{Var}(X_{nk}) = \sigma_{nk}^2$。若 $|X_{nk}| \leq M_n$ a.s. 且 $M_n = o(s_n)$，其中 $s_n^2 = \sum_{k=1}^n \sigma_{nk}^2$，验证Lindeberg条件成立。

**解答**：

Lindeberg条件要求：

$$\frac{1}{s_n^2}\sum_{k=1}^n \mathbb{E}[X_{nk}^2 \mathbf{1}_{\{|X_{nk}| > \epsilon s_n\}}] \to 0, \quad \forall \epsilon > 0$$

由于 $|X_{nk}| \leq M_n$ 且 $M_n = o(s_n)$，对任意固定的 $\epsilon > 0$，当 $n$ 足够大时：

$$M_n < \epsilon s_n$$

因此 $\{|X_{nk}| > \epsilon s_n\} = \emptyset$，即：

$$\mathbb{E}[X_{nk}^2 \mathbf{1}_{\{|X_{nk}| > \epsilon s_n\}}] = 0$$

Lindeberg条件显然满足。 ∎

**习题 6.10**：设 $\{X_n\}$ 是i.i.d.随机变量，$\mathbb{E}[X_1] = 0$，$\text{Var}(X_1) = 1$，$\mathbb{E}[|X_1|^3] = \gamma < \infty$。使用Berry-Esseen定理，求 $n$ 使得：

$$\sup_x \left|\mathbb{P}\left(\frac{\sum_{i=1}^n X_i}{\sqrt{n}} \leq x\right) - \Phi(x)\right| < 0.01$$

**解答**：

Berry-Esseen界：

$$\sup_x |F_n(x) - \Phi(x)| \leq \frac{C\gamma}{\sigma^3\sqrt{n}} = \frac{0.4748\gamma}{\sqrt{n}}$$

要求：

$$\frac{0.4748\gamma}{\sqrt{n}} < 0.01$$

$$n > (47.48\gamma)^2 = 2254.35\gamma^2$$

若 $\gamma = 2$（如指数分布），则 $n > 9017$。 ∎

**习题 6.11**：设 $\{X_n\}$ i.i.d.，$\mathbb{E}[X_1] = \mu$，定义样本方差：

$$S_n^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \bar{X}_n)^2$$

(a) 证明 $S_n^2 \xrightarrow{\mathbb{P}} \sigma^2$。
(b) 推导 $\sqrt{n}(S_n^2 - \sigma^2)$ 的渐近分布。

**解答**：

**(a)** 注意到：

$$S_n^2 = \frac{n}{n-1}\left(\frac{1}{n}\sum_{i=1}^n X_i^2 - \bar{X}_n^2\right)$$

由SLLN：

$$\frac{1}{n}\sum_{i=1}^n X_i^2 \xrightarrow{\text{a.s.}} \mathbb{E}[X_1^2] = \mu^2 + \sigma^2$$

$$\bar{X}_n \xrightarrow{\text{a.s.}} \mu$$

因此：

$$S_n^2 \xrightarrow{\text{a.s.}} (\mu^2 + \sigma^2) - \mu^2 = \sigma^2$$

**(b)** 由Delta方法或直接计算：

$$\sqrt{n}(S_n^2 - \sigma^2) \xrightarrow{d} \mathcal{N}(0, \mu_4 - \sigma^4)$$

其中 $\mu_4 = \mathbb{E}[(X_1 - \mu)^4]$ 是四阶中心矩。 ∎

**习题 6.12**：设 $\{X_n\}$ 是i.i.d.随机变量，矩母函数 $M(t) = \mathbb{E}[e^{tX_1}]$ 在 $|t| < h$ 时存在。

(a) 证明Chernoff界：$\mathbb{P}(\bar{X}_n \geq a) \leq e^{-nI(a)}$，其中 $I(a) = \sup_t (ta - \log M(t))$。

(b) 推导Cramér大偏差定理。

**解答**：

**(a)** 对 $t > 0$：

$$\mathbb{P}(\bar{X}_n \geq a) = \mathbb{P}(e^{t\sum X_i} \geq e^{nta}) \leq \frac{\mathbb{E}[e^{t\sum X_i}]}{e^{nta}} = \frac{M(t)^n}{e^{nta}} = e^{-n(ta - \log M(t))}$$

对 $t$ 取下确界：

$$\mathbb{P}(\bar{X}_n \geq a) \leq e^{-n\sup_t(ta - \log M(t))} = e^{-nI(a)}$$

**(b)** Cramér定理断言上述界是紧的：

$$\lim_{n \to \infty} \frac{1}{n}\log \mathbb{P}(\bar{X}_n \geq a) = -I(a)$$

证明需要精细的分析，包括下界估计（通过变分法）和上界优化。 ∎

**习题 6.13**（Delta方法）：设 $\sqrt{n}(T_n - \theta) \xrightarrow{d} \mathcal{N}(0, \sigma^2)$，$g$ 是可微函数，$g'(\theta) \neq 0$。证明：

$$\sqrt{n}(g(T_n) - g(\theta)) \xrightarrow{d} \mathcal{N}(0, [g'(\theta)]^2\sigma^2)$$

**解答**：

由Taylor展开：

$$g(T_n) = g(\theta) + g'(\theta)(T_n - \theta) + o_p(|T_n - \theta|)$$

因此：

$$\sqrt{n}(g(T_n) - g(\theta)) = g'(\theta)\sqrt{n}(T_n - \theta) + \sqrt{n} \cdot o_p(|T_n - \theta|)$$

由于 $\sqrt{n}(T_n - \theta) = O_p(1)$，有 $T_n - \theta = O_p(n^{-1/2})$。

因此第二项：

$$\sqrt{n} \cdot o_p(n^{-1/2}) = o_p(1)$$

由Slutsky定理：

$$\sqrt{n}(g(T_n) - g(\theta)) \xrightarrow{d} g'(\theta) \cdot \mathcal{N}(0, \sigma^2) = \mathcal{N}(0, [g'(\theta)]^2\sigma^2)$$ ∎

**习题 6.14**：设 $\{X_n\}$ 是平稳遍历序列，$\mathbb{E}[|X_1|] < \infty$。陈述并证明遍历定理。

**解答**：

**遍历定理**：设 $\{X_n\}$ 是遍历的平稳序列，$\mathbb{E}[|X_1|] < \infty$，则：

$$\frac{1}{n}\sum_{i=1}^n X_i \xrightarrow{\text{a.s.}} \mathbb{E}[X_1]$$

**证明概要**：

1. 定义保测变换 $T$ 和不变 $\sigma$-代数 $\mathcal{I}$。

2. 条件期望 $\mathbb{E}[X_1|\mathcal{I}]$ 是极限的候选。

3. 对截断变量应用Birkhoff遍历定理：

$$\frac{1}{n}\sum_{i=0}^{n-1} X_1 \circ T^i \to \mathbb{E}[X_1|\mathcal{I}] \quad \text{a.s.}$$

1. 由遍历性，$\mathcal{I}$ 是平凡的，因此 $\mathbb{E}[X_1|\mathcal{I}] = \mathbb{E}[X_1]$ a.s.。

2. 去掉截断，完成证明。 ∎

### 6.3 进阶与挑战习题（15-20）

**习题 6.15**：设 $\{X_n\}$ i.i.d.，$\mathbb{E}[X_1] = 0$，$\text{Var}(X_1) = 1$。定义部分和 $S_n = \sum_{i=1}^n X_i$。

(a) 证明：$\limsup_{n \to \infty} \frac{S_n}{\sqrt{2n\ln\ln n}} = 1$ a.s.（重对数律的上半部分）

(b) 若 $X_1 \sim \mathcal{N}(0, 1)$，计算 $\mathbb{P}(S_n > \sqrt{2n\ln\ln n})$ 的渐近行为。

**解答**：

**(a)** Hartman-Wintner重对数律的证明需要：

1. 上界：使用指数不等式和Borel-Cantelli引理。

2. 下界：构造特定子序列，证明无穷多次越过边界。

详细证明见Petrov (1995) 或 Durrett (2019)。

**(b)** 若 $X_1 \sim \mathcal{N}(0, 1)$，则 $S_n \sim \mathcal{N}(0, n)$。

$$\mathbb{P}(S_n > \sqrt{2n\ln\ln n}) = 1 - \Phi(\sqrt{2\ln\ln n})$$

利用Mill比率：$1 - \Phi(x) \sim \frac{\phi(x)}{x}$（当 $x \to \infty$）

$$\approx \frac{1}{\sqrt{2\pi} \cdot \sqrt{2\ln\ln n}} \cdot \frac{1}{(\ln n)^{1/\ln n}}$$

对于 $n = e^k$：

$$\sum_{k=1}^{\infty} \mathbb{P}(S_{e^k} > \sqrt{2e^k \ln k}) \approx \sum_{k=1}^{\infty} \frac{1}{\sqrt{4\pi \ln k}} \cdot k^{-1} = \infty$$

由Borel-Cantelli（逆），$S_n > \sqrt{2n\ln\ln n}$ i.o. a.s.。 ∎

**习题 6.16**（多维中心极限定理）：设 $\{\mathbf{X}_n\}$ 是i.i.d.随机向量，$\mathbf{X}_n \in \mathbb{R}^d$，$\mathbb{E}[\mathbf{X}_1] = \boldsymbol{\mu}$，$\text{Cov}(\mathbf{X}_1) = \Sigma$。证明：

$$\frac{1}{\sqrt{n}}\sum_{i=1}^n (\mathbf{X}_i - \boldsymbol{\mu}) \xrightarrow{d} \mathcal{N}_d(\mathbf{0}, \Sigma)$$

**解答**：

使用Cramér-Wold方法：证明对所有 $\mathbf{t} \in \mathbb{R}^d$：

$$\mathbf{t}^T \left(\frac{1}{\sqrt{n}}\sum_{i=1}^n (\mathbf{X}_i - \boldsymbol{\mu})\right) \xrightarrow{d} \mathbf{t}^T \mathbf{Z}, \quad \mathbf{Z} \sim \mathcal{N}_d(\mathbf{0}, \Sigma)$$

左边：

$$\frac{1}{\sqrt{n}}\sum_{i=1}^n \mathbf{t}^T(\mathbf{X}_i - \boldsymbol{\mu}) = \frac{1}{\sqrt{n}}\sum_{i=1}^n Y_i$$

其中 $Y_i = \mathbf{t}^T(\mathbf{X}_i - \boldsymbol{\mu})$ i.i.d.，$\mathbb{E}[Y_i] = 0$，$\text{Var}(Y_i) = \mathbf{t}^T\Sigma\mathbf{t}$。

由一维CLT：

$$\frac{1}{\sqrt{n}}\sum_{i=1}^n Y_i \xrightarrow{d} \mathcal{N}(0, \mathbf{t}^T\Sigma\mathbf{t})$$

这正是 $\mathbf{t}^T\mathbf{Z}$ 的分布。由Cramér-Wold定理，多维CLT得证。 ∎

**习题 6.17**（稳定分布）：设 $\{X_n\}$ i.i.d.，$\mathbb{E}[|X_1|^\alpha] < \infty$ 对某个 $\alpha \in (0, 2)$，但 $\mathbb{E}[X_1^2] = \infty$。证明存在常数 $a_n, b_n$ 使得：

$$\frac{\sum_{i=1}^n X_i - a_n}{b_n}$$

收敛到非正态的稳定分布，并确定 $b_n$ 的量级。

**解答**：

**背景**：稳定分布是独立同分布随机变量和的唯一可能的极限分布（在适当的中心化、标准化下）。

**结果**：

1. 若 $\mathbb{E}[|X_1|^\alpha] < \infty$ 对 $\alpha \in (0, 2)$，且分布属于 $\alpha$-稳定吸引域，则：

$$b_n = n^{1/\alpha} L(n)$$

其中 $L$ 是缓变函数。

1. 极限分布是指数为 $\alpha$ 的稳定分布，其特征函数：

$$\varphi(t) = \exp\left(-|t|^\alpha[1 - i\beta \text{sgn}(t)\tan(\frac{\pi\alpha}{2})]\right)$$

（对 $\alpha \neq 1$）。

**例子**：Cauchy分布是1-稳定分布，标准化因子 $b_n = n$。

若 $X_1$ 服从指数为 $\alpha$ 的Pareto分布尾部，则 $b_n = n^{1/\alpha}$。 ∎

**习题 6.18**（经验过程的CLT）：设 $\{X_n\}$ i.i.d.，分布函数为 $F$。定义经验分布函数：

$$F_n(x) = \frac{1}{n}\sum_{i=1}^n \mathbf{1}_{\{X_i \leq x\}}$$

证明：$\sqrt{n}(F_n(x) - F(x))$ 作为随机过程收敛到Gauss过程。

**解答**：

**Donsker定理**：过程 $\{\sqrt{n}(F_n(x) - F(x)) : x \in \mathbb{R}\}$ 在 $\ell^\infty(\mathbb{R})$ 中弱收敛到**Brownian桥** $\mathbb{B}(F(x))$。

**证明要点**：

1. 有限维分布：对固定 $x_1, \ldots, x_k$，

$$(F_n(x_1), \ldots, F_n(x_k))$$

服从多项分布，由多维CLT收敛到Gauss。

1. 紧性：使用 $\epsilon$-网或bracketing论证证明经验过程是渐近等度连续的。

2. 极限过程是Gauss过程，均值为0，协方差：

$$\text{Cov}(\mathbb{B}(F(x)), \mathbb{B}(F(y))) = F(x)(1 - F(y)), \quad x \leq y$$

这就是Brownian桥。 ∎

**习题 6.19**：设 $\{X_n\}$ 是鞅差序列，$\mathbb{E}[X_n|\mathcal{F}_{n-1}] = 0$，$S_n = \sum_{i=1}^n X_i$。

(a) 陈述鞅中心极限定理的条件。
(b) 若 $\sum_{i=1}^n \mathbb{E}[X_i^2|\mathcal{F}_{i-1}] \xrightarrow{\mathbb{P}} \sigma^2$ 且Lindeberg条件成立，证明：

$$\frac{S_n}{\sqrt{\sum_{i=1}^n \mathbb{E}[X_i^2|\mathcal{F}_{i-1}]}} \xrightarrow{d} \mathcal{N}(0, 1)$$

**解答**：

**(a)** 鞅CLT的条件（Hall & Heyde, 1980）：

1. **条件方差收敛**：$\frac{\langle S \rangle_n}{n} \xrightarrow{\mathbb{P}} \sigma^2$，其中 $\langle S \rangle_n = \sum_{i=1}^n \mathbb{E}[X_i^2|\mathcal{F}_{i-1}]$ 是可预测二次变差。

2. **Lindeberg条件**：对所有 $\epsilon > 0$，

$$\frac{1}{n}\sum_{i=1}^n \mathbb{E}[X_i^2 \mathbf{1}_{\{|X_i| > \epsilon\sqrt{n}\}}|\mathcal{F}_{i-1}] \xrightarrow{\mathbb{P}} 0$$

1. **最大跳跃控制**：$\frac{\max_{1 \leq i \leq n} |X_i|}{\sqrt{n}} \xrightarrow{\mathbb{P}} 0$。

**(b)** 证明使用鞅特征函数方法：

1. 定义 $M_n(t) = \prod_{i=1}^n (1 + itX_i/\sqrt{n})$。

2. 证明 $\mathbb{E}[M_n(t)] \to e^{-t^2\sigma^2/2}$。

3. 使用鞅收敛定理和特征函数连续性。

详细证明需要精细的估计，见Hall & Heyde (1980)。 ∎

**习题 6.20**（高维CLT）：设 $\mathbf{X}_1, \ldots, \mathbf{X}_n \in \mathbb{R}^d$ i.i.d.，$d$ 可能与 $n$ 同阶增长。讨论经典CLT在何种条件下仍成立，并给出反例说明当 $d \gg n$ 时CLT失效。

**解答**：

**经典CLT在高维的问题**：

当 $d \to \infty$ 与 $n$ 同阶时，样本协方差矩阵的估计变得不可靠。

**有效条件**（Portnoy, 1986）：若 $d = o(\sqrt{n})$，经典CLT对边际分布仍成立。

**反例**：设 $d = n$，$\mathbf{X}_i = e_i$（标准基向量）以概率 $1/n$ 各选一个方向。

则：

$$\frac{1}{\sqrt{n}}\sum_{i=1}^n \mathbf{X}_i$$

的各分量是独立的Poisson(1)（近似），不是Gauss。

**现代结果**（随机矩阵理论）：

当 $d/n \to \gamma \in (0, \infty)$，样本协方差矩阵的特征值服从Marchenko-Pastur分布，而非集中在总体特征值。

协方差估计需要正则化（Ridge、Lasso等）。 ∎
