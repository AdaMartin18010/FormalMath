# 数理统计习题补充

## 点估计

### 习题1

设$X_1, ..., X_n \sim N(\mu, \sigma^2)$，求$\mu$和$\sigma^2$的MLE。

**解答**：
似然函数：$L(\mu, \sigma^2) = (2\pi\sigma^2)^{-n/2} \exp\left(-\frac{1}{2\sigma^2}\sum(X_i-\mu)^2\right)$

对数似然：$\ell = -\frac{n}{2}\log(2\pi\sigma^2) - \frac{1}{2\sigma^2}\sum(X_i-\mu)^2$

$\frac{\partial \ell}{\partial \mu} = \frac{1}{\sigma^2}\sum(X_i-\mu) = 0 \Rightarrow \hat{\mu} = \bar{X}$

$\frac{\partial \ell}{\partial \sigma^2} = -\frac{n}{2\sigma^2} + \frac{1}{2(\sigma^2)^2}\sum(X_i-\mu)^2 = 0 \Rightarrow \hat{\sigma}^2 = \frac{1}{n}\sum(X_i-\bar{X})^2$

---

### 习题2

证明样本均值$\bar{X}$是$\mu$的无偏估计。

**证明**：$E[\bar{X}] = E[\frac{1}{n}\sum X_i] = \frac{1}{n}\sum E[X_i] = \frac{1}{n} \cdot n\mu = \mu$ ∎

---

## 假设检验

### 习题3

设$X_1, ..., X_n \sim N(\mu, \sigma^2)$，$\sigma^2$已知，检验$H_0: \mu = \mu_0$ vs $H_1: \mu \neq \mu_0$。

**解答**：检验统计量$Z = \frac{\bar{X} - \mu_0}{\sigma/\sqrt{n}} \sim N(0,1)$（在$H_0$下）

拒绝域：$|Z| > z_{\alpha/2}$

---

### 习题4

解释第一类错误和第二类错误。

**解答**：

- **第一类错误**：$H_0$为真时拒绝$H_0$（假阳性），概率为$\alpha$
- **第二类错误**：$H_0$为假时接受$H_0$（假阴性），概率为$\beta$
- **检验力**：$1-\beta$，正确拒绝$H_0$的概率

---

## 回归分析

### 习题5

证明最小二乘估计使残差平方和最小。

**解答**：$RSS = \sum_{i=1}^n (y_i - \hat{y}_i)^2 = \|y - X\hat{\beta}\|^2$

$\frac{\partial RSS}{\partial \hat{\beta}} = -2X^T(y - X\hat{\beta}) = 0$

$\hat{\beta} = (X^T X)^{-1} X^T y$ ∎

## Python实践

### 参数估计

```python
import numpy as np
from scipy import stats

# 生成样本
np.random.seed(42)
mu_true, sigma_true = 5, 2
n = 100
sample = np.random.normal(mu_true, sigma_true, n)

# 矩估计
mu_mme = np.mean(sample)
sigma_mme = np.std(sample, ddof=1)
print(f"矩估计: μ={mu_mme:.4f}, σ={sigma_mme:.4f}")

# MLE（正态分布的MLE等于样本均值和样本方差）
mu_mle = np.mean(sample)
sigma_mle = np.sqrt(np.mean((sample - mu_mle)**2))
print(f"MLE: μ={mu_mle:.4f}, σ={sigma_mle:.4f}")

# 置信区间
conf_int = stats.t.interval(0.95, n-1, loc=mu_mme, scale=stats.sem(sample))
print(f"95%置信区间: {conf_int}")
```

### 假设检验

```python
# t检验
t_stat, p_value = stats.ttest_1samp(sample, 5.0)
print(f"t统计量: {t_stat:.4f}, p值: {p_value:.4f}")

# 结论
alpha = 0.05
if p_value < alpha:
    print("拒绝原假设")
else:
    print("接受原假设")
```
