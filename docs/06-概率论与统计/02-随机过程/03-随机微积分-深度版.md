# 随机微积分（深度版）

msc_primary: 60xx

## 一、引言与历史背景

随机微积分是现代概率论最具影响力的分支之一，它将微积分运算推广到随机过程，特别是布朗运动。Kiyoshi Itô（伊藤清）在1944年发表的奠基性论文中，创立了对布朗运动进行随机积分的严格理论，并发现了著名的Itô公式——随机分析中的微积分基本定理。这一理论框架后来成为数理金融的数学基石，特别是Black-Scholes-Merton期权定价模型（1973年）的核心数学工具。

**核心人物关联**：

- **Kiyoshi Itô (1915-2008)**：日本数学家，随机分析奠基人，创立Itô积分与Itô公式，2006年获首届Gauss奖
- **Fischer Black (1938-1995)** & **Myron Scholes (1941-)**：Black-Scholes期权定价模型，1997年诺贝尔经济学奖
- **Robert C. Merton (1944-)**：连续时间金融理论奠基人，与Scholes共享1997年诺贝尔经济学奖
- **Paul Malliavin (1925-2010)**：创立Malliavin分析，随机变分计算

## 二、Itô积分的构造

### 2.1 动机：为何需要新的积分理论？

考虑用Riemann-Stieltjes积分定义 $\int_0^t f(W_s) dW_s$。对布朗运动，由于路径的二次变差不为零（$\langle W \rangle_t = t$），传统的有界变差积分理论不适用。

**关键观察**：若尝试用不同方式逼近：
$$\int_0^t W_s dW_s \approx \sum_{i} W_{t_i^*}(W_{t_{i+1}} - W_{t_i})$$

- 取 $t_i^* = t_i$（左端点）：结果为 $\frac{1}{2}(W_t^2 - t)$
- 取 $t_i^* = \frac{t_i + t_{i+1}}{2}$（中点）：结果为 $\frac{1}{2}W_t^2$

积分值依赖于取点方式！Itô选择左端点，使积分具有鞅性质。

### 2.2 可积函数空间 $\mathcal{L}^2$ 的定义

**定义 2.1（可料过程）** 随机过程 $\{H_t\}_{t \geq 0}$ 称为 **可料过程**（predictable），若关于由左连续适应过程生成的σ-代数可测。

**定义 2.2（可积函数空间）** 设 $\mathcal{L}^2$ 为所有可料过程 $H$ 满足
$$\|H\|_{\mathcal{L}^2}^2 = \mathbb{E}\left[\int_0^{\infty} H_s^2 ds\right] < \infty$$

构成的Hilbert空间（模去不可区分过程）。

### 2.3 Itô积分的构造步骤

**步骤 1：简单过程的积分**

设 $H_t = \sum_{i=0}^{n-1} h_i \mathbf{1}_{(t_i, t_{i+1}]}(t)$，其中 $h_i$ 为$\mathcal{F}_{t_i}$-可测且 $\mathbb{E}[h_i^2] < \infty$。定义
$$\int_0^{\infty} H_s dW_s = \sum_{i=0}^{n-1} h_i (W_{t_{i+1}} - W_{t_i})$$

**步骤 2：等距性质（Itô等距）**

**定理 2.3（Itô等距）** 对简单过程 $H$：
$$\mathbb{E}\left[\left(\int_0^{\infty} H_s dW_s\right)^2\right] = \mathbb{E}\left[\int_0^{\infty} H_s^2 ds\right]$$
即映射 $H \mapsto \int H dW$ 是保距嵌入。

**证明**：展开平方，利用布朗运动增量的独立性和 $\mathbb{E}[(W_{t_{i+1}}-W_{t_i})^2] = t_{i+1}-t_i$。

**步骤 3：极限过程**

**定理 2.4** 简单过程在 $\mathcal{L}^2$ 中稠密。

对一般 $H \in \mathcal{L}^2$，取简单过程列 $H^{(n)} \to H$，由Itô等距，$\{\int H^{(n)} dW\}$ 是 $L^2$ 中的Cauchy列。定义
$$\int_0^{\infty} H_s dW_s = \lim_{n \to \infty} \int_0^{\infty} H_s^{(n)} dW_s \quad \text{(}L^2\text{极限)}$$

**定义 2.5（Itô积分）** 上述极限定义了对 $H \in \mathcal{L}^2$ 的 **Itô随机积分**。

**定理 2.6（Itô积分的性质）**

1. **线性性**：$\int (aH + bK) dW = a\int H dW + b\int K dW$
2. **零期望**：$\mathbb{E}[\int H dW] = 0$
3. **鞅性**：$M_t = \int_0^t H_s dW_s$ 是连续鞅
4. **二次变差**：$\langle M \rangle_t = \int_0^t H_s^2 ds$

## 三、Itô公式

### 3.1 一元Itô公式

**定理 3.1（Itô公式 - 经典形式）** 设 $f \in C^{2}(\mathbb{R})$，则
$$f(W_t) = f(0) + \int_0^t f'(W_s) dW_s + \frac{1}{2}\int_0^t f''(W_s) ds$$
微分形式：
$$df(W_t) = f'(W_t) dW_t + \frac{1}{2}f''(W_t) dt$$

**证明概要**：对分割 $0 = t_0 < t_1 < \cdots < t_n = t$，Taylor展开：
$$f(W_{t_{i+1}}) - f(W_{t_i}) = f'(W_{t_i})(W_{t_{i+1}} - W_{t_i}) + \frac{1}{2}f''(W_{t_i})(W_{t_{i+1}} - W_{t_i})^2 + R_i$$
第一项收敛于Itô积分，第二项收敛于 $\frac{1}{2}\int f''(W_s)ds$（利用二次变差），余项趋于0。

**关键区别**：与普通微积分相比，多出 $\frac{1}{2}f'' dt$ 项，这是由 $(dW_t)^2 = dt$ 产生的。

**例 3.2** 取 $f(x) = x^2$：
$$W_t^2 = 2\int_0^t W_s dW_s + t$$
整理得：$\int_0^t W_s dW_s = \frac{1}{2}(W_t^2 - t)$。

### 3.2 Itô过程与一般Itô公式

**定义 3.3（Itô过程）** 过程 $X$ 称为 **Itô过程**，若
$$X_t = X_0 + \int_0^t K_s ds + \int_0^t H_s dW_s$$
其中 $K, H$ 满足可积性条件。微分形式：
$$dX_t = K_t dt + H_t dW_t$$

**定理 3.4（一般Itô公式）** 设 $X$ 为Itô过程，$f \in C^{1,2}([0,\infty) \times \mathbb{R})$，则
$$\begin{aligned}
f(t, X_t) = f(0, X_0) &+ \int_0^t \frac{\partial f}{\partial s}(s, X_s) ds + \int_0^t \frac{\partial f}{\partial x}(s, X_s) dX_s \\
&+ \frac{1}{2}\int_0^t \frac{\partial^2 f}{\partial x^2}(s, X_s) (dX_s)^2
\end{aligned}$$
其中 $(dX_s)^2 = H_s^2 ds$（计算规则：$(dt)^2 = 0$，$dt \cdot dW = 0$，$(dW)^2 = dt$）。

### 3.3 多元Itô公式

**定理 3.5（多元Itô公式）** 设 $W = (W^1, \ldots, W^d)$ 为$d$维布朗运动，$X^i$ 为Itô过程，$f \in C^{1,2}([0,\infty) \times \mathbb{R}^d)$，则
$$\begin{aligned}
f(t, X_t) &= f(0, X_0) + \int_0^t \frac{\partial f}{\partial s} ds + \sum_{i=1}^{d} \int_0^t \frac{\partial f}{\partial x_i} dX_s^i \\
&+ \frac{1}{2}\sum_{i,j=1}^{d} \int_0^t \frac{\partial^2 f}{\partial x_i \partial x_j} d\langle X^i, X^j \rangle_s
\end{aligned}$$

其中 **二次协变差**：$d\langle X^i, X^j \rangle_t = H_t^i H_t^j \delta_{ij} dt$。

**例 3.6（乘积法则）** 对两个Itô过程 $X, Y$：
$$d(X_t Y_t) = X_t dY_t + Y_t dX_t + d\langle X, Y \rangle_t$$

## 四、随机微分方程

### 4.1 SDE的定义

**定义 4.1（随机微分方程）** 形如
$$dX_t = b(t, X_t)dt + \sigma(t, X_t)dW_t, \quad X_0 = x_0$$
的方程称为 **随机微分方程**（SDE），其中：
- $b: [0,\infty) \times \mathbb{R} \to \mathbb{R}$ 称为 **漂移系数**
- $\sigma: [0,\infty) \times \mathbb{R} \to \mathbb{R}$ 称为 **扩散系数**

**定义 4.2（强解）** 适应过程 $X$ 称为SDE的 **强解**，若对所有 $t$：
$$X_t = x_0 + \int_0^t b(s, X_s) ds + \int_0^t \sigma(s, X_s) dW_s \quad \text{a.s.}$$

### 4.2 存在唯一性定理

**定理 4.3（解的存在唯一性）** 设 $b, \sigma$ 满足 **Lipschitz条件** 和 **线性增长条件**：
1. $|b(t,x) - b(t,y)| + |\sigma(t,x) - \sigma(t,y)| \leq L|x-y|$
2. $|b(t,x)| + |\sigma(t,x)| \leq C(1+|x|)$

则SDE存在唯一的强解 $X$，且对任意 $T > 0$：
$$\mathbb{E}\left[\sup_{0 \leq t \leq T} X_t^2\right] < \infty$$

**证明概要**：使用Picard迭代：
$$X_t^{(n+1)} = x_0 + \int_0^t b(s, X_s^{(n)}) ds + \int_0^t \sigma(s, X_s^{(n)}) dW_s$$
利用Lipschitz条件和Itô等距证明迭代在适当范数下收敛。

### 4.3 线性SDE与几何布朗运动

**例 4.4（Ornstein-Uhlenbeck过程）**
$$dX_t = -\alpha X_t dt + \sigma dW_t, \quad \alpha, \sigma > 0$$

解法：乘以积分因子 $e^{\alpha t}$，应用Itô公式：
$$d(e^{\alpha t} X_t) = e^{\alpha t} dX_t + \alpha e^{\alpha t} X_t dt = \sigma e^{\alpha t} dW_t$$
积分得：
$$X_t = e^{-\alpha t} X_0 + \sigma \int_0^t e^{-\alpha(t-s)} dW_s$$

$X_t \sim \mathcal{N}(e^{-\alpha t} X_0, \frac{\sigma^2}{2\alpha}(1 - e^{-2\alpha t}))$，当 $t \to \infty$ 时收敛于稳态分布 $\mathcal{N}(0, \frac{\sigma^2}{2\alpha})$。

**例 4.5（几何布朗运动）**
$$dS_t = \mu S_t dt + \sigma S_t dW_t$$

应用Itô公式于 $\log S_t$：
$$d(\log S_t) = \frac{dS_t}{S_t} - \frac{1}{2}\frac{(dS_t)^2}{S_t^2} = (\mu - \frac{\sigma^2}{2})dt + \sigma dW_t$$
解得：
$$S_t = S_0 \exp\left((\mu - \frac{\sigma^2}{2})t + \sigma W_t\right)$$

**历史注记**：几何布朗运动是Black-Scholes模型的基础，由Paul Samuelson于1965年引入金融建模。

## 五、Black-Scholes模型推导

### 5.1 市场模型

**假设**：
1. **无风险资产**（债券）：$dB_t = rB_t dt$，即 $B_t = e^{rt}$
2. **风险资产**（股票）：服从几何布朗运动
   $$dS_t = \mu S_t dt + \sigma S_t dW_t$$
   其中$\mu$为期望收益率，$\sigma$为波动率

### 5.2 自融资策略

**定义 5.1（投资组合）** 投资组合 $(\alpha_t, \beta_t)$ 表示持有 $\alpha_t$ 份股票和 $\beta_t$ 份债券，其财富过程
$$V_t = \alpha_t S_t + \beta_t B_t$$

**定义 5.2（自融资策略）** 策略 $(\alpha, \beta)$ 称为 **自融资**，若
$$dV_t = \alpha_t dS_t + \beta_t dB_t$$
即财富变化仅来自资产价格变动，无外部资金流入流出。

### 5.3 风险中性定价

**定理 5.3（Girsanov定理）** 存在唯一的等价鞅测度 $\mathbb{Q}$（风险中性测度），在此测度下
$$\tilde{W}_t = W_t + \frac{\mu - r}{\sigma}t$$
是布朗运动，且折现股价 $\tilde{S}_t = e^{-rt}S_t$ 是鞅。

**推导**：在 $\mathbb{Q}$ 下，$dS_t = rS_t dt + \sigma S_t d\tilde{W}_t$（漂移率变为无风险利率）。

### 5.4 Black-Scholes公式

**问题**：求欧式看涨期权在时刻 $t=0$ 的价格，到期 $T$，执行价 $K$。

**定理 5.4（Black-Scholes公式, 1973）** 欧式看涨期权价格
$$C(0, S_0) = S_0 \Phi(d_1) - Ke^{-rT} \Phi(d_2)$$
其中 $\Phi$ 为标准正态分布函数，
$$d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2)T}{\sigma\sqrt{T}}, \quad d_2 = d_1 - \sigma\sqrt{T}$$

**证明概要**：
1. 由风险中性定价，$C(0, S_0) = e^{-rT}\mathbb{E}^\mathbb{Q}[(S_T - K)^+]$
2. 在 $\mathbb{Q}$ 下，$S_T = S_0 \exp((r - \sigma^2/2)T + \sigma \tilde{W}_T)$
3. 直接计算期望（利用对数正态分布性质）即得公式。

**历史意义**：这是第一个将期权价格与标的资产期望收益率$\mu$无关的公式，完全由可观测参数决定。

## 六、扩展话题

### 6.1 Feynman-Kac公式

**定理 6.1（Feynman-Kac）** 设 $u(t,x)$ 满足终值问题
$$\frac{\partial u}{\partial t} + \frac{1}{2}\sigma^2 \frac{\partial^2 u}{\partial x^2} + b \frac{\partial u}{\partial x} - ru = 0, \quad u(T,x) = g(x)$$
则
$$u(t,x) = \mathbb{E}\left[e^{-r(T-t)} g(X_T) | X_t = x\right]$$

其中 $dX_s = b ds + \sigma dW_s$。

**应用**：将抛物型PDE的解表示为随机表示，用于数值计算（Monte Carlo方法）。

### 6.2 Itô表示定理与鞅表示

**定理 6.2（Itô表示定理）** 对任意 $F \in L^2(\mathcal{F}_T)$，存在唯一的可料过程 $H \in \mathcal{L}^2$ 使得
$$F = \mathbb{E}[F] + \int_0^T H_s dW_s$$

**推论**：任意平方可积鞅可表示为Itô积分。
