# 条件概率与贝叶斯定理

## 一、概念深度解析

### 1.1 直观理解

条件概率是概率论中最富有洞察力的概念之一，它回答了"在已知某些信息的情况下，某事件发生的可能性如何变化"这一核心问题。直观上，当我们获得新的信息时，样本空间实际上被"缩小"了——我们只关注那些与新信息相容的结果。条件概率 $\mathbb{P}(A|B)$ 就是在这个"缩小后的世界"中重新度量事件 $A$ 发生的可能性。

贝叶斯定理则提供了一种"逆向推理"的强大工具：它允许我们根据观察到的结果反推原因的概率。这在医学诊断、机器学习、金融预测等领域有着不可替代的作用。贝叶斯推断的核心思想是将先验信念与观测数据相结合，得到更精确的后验信念。

### 1.2 形式化定义

**定义 1.1**（条件概率）：设 $(\Omega, \Sigma, \mathbb{P})$ 为概率空间，$B \in \Sigma$ 且 $\mathbb{P}(B) > 0$。事件 $A$ 在 $B$ 发生条件下的**条件概率**定义为：

$$\mathbb{P}(A|B) = \frac{\mathbb{P}(A \cap B)}{\mathbb{P}(B)}$$

**定义 1.2**（贝叶斯定理）：设 $\{B_1, B_2, \ldots, B_n\}$ 为样本空间的一个划分，即 $B_i$ 两两不交且 $\bigcup_{i=1}^n B_i = \Omega$。若 $\mathbb{P}(A) > 0$，则：

$$\mathbb{P}(B_j|A) = \frac{\mathbb{P}(A|B_j) \cdot \mathbb{P}(B_j)}{\sum_{i=1}^n \mathbb{P}(A|B_i) \cdot \mathbb{P}(B_i)}$$

其中 $\mathbb{P}(B_j)$ 称为**先验概率**，$\mathbb{P}(B_j|A)$ 称为**后验概率**。

### 1.3 等价表述

**等价表述 1.3**（通过期望）：条件概率可等价定义为：
$$\mathbb{P}(A|B) = \mathbb{E}[\mathbf{1}_A | \mathbf{1}_B = 1]$$

**等价表述 1.4**（通过似然比）：设 $L(B|A) = \frac{\mathbb{P}(B|A)}{\mathbb{P}(B|A^c)}$ 为**似然比**，则：
$$\frac{\mathbb{P}(A|B)}{\mathbb{P}(A^c|B)} = L(B|A) \cdot \frac{\mathbb{P}(A)}{\mathbb{P}(A^c)}$$

### 1.4 动机与背景

条件概率的严格定义归功于Kolmogorov，但其思想可以追溯到18世纪。Thomas Bayes在1763年发表的论文《论有关机遇问题的求解》中提出了贝叶斯定理的雏形。这一定理在统计推断中的应用由Laplace进一步发展。

在20世纪，贝叶斯统计学经历了起伏。频率学派在很长一段时间占主导地位，但20世纪下半叶以来，随着计算能力的提升和MCMC等算法的发展，贝叶斯方法在机器学习、人工智能等领域重获新生。

## 二、属性与关系

### 2.1 核心性质与证明

**定理 2.1**（条件概率的规范性）：设 $\mathbb{P}(B) > 0$，则 $\mathbb{P}(\cdot|B)$ 是 $(\Omega, \Sigma)$ 上的概率测度。

**证明**：
1. **非负性**：$\mathbb{P}(A|B) = \frac{\mathbb{P}(A \cap B)}{\mathbb{P}(B)} \geq 0$
2. **规范性**：$\mathbb{P}(\Omega|B) = \frac{\mathbb{P}(\Omega \cap B)}{\mathbb{P}(B)} = \frac{\mathbb{P}(B)}{\mathbb{P}(B)} = 1$
3. **可列可加性**：设 $A_1, A_2, \ldots$ 两两不交，则：
   $$\mathbb{P}\left(\bigcup_{i=1}^{\infty} A_i \big| B\right) = \frac{\mathbb{P}\left(\bigcup_{i=1}^{\infty} (A_i \cap B)\right)}{\mathbb{P}(B)} = \frac{\sum_{i=1}^{\infty} \mathbb{P}(A_i \cap B)}{\mathbb{P}(B)} = \sum_{i=1}^{\infty} \mathbb{P}(A_i|B)$$

**定理 2.2**（乘法公式）：若 $\mathbb{P}(A_1 \cap \cdots \cap A_{n-1}) > 0$，则：
$$\mathbb{P}(A_1 \cap \cdots \cap A_n) = \mathbb{P}(A_1) \cdot \mathbb{P}(A_2|A_1) \cdot \mathbb{P}(A_3|A_1 \cap A_2) \cdots \mathbb{P}(A_n|A_1 \cap \cdots \cap A_{n-1})$$

**证明**：由条件概率定义迭代应用即得。

**定理 2.3**（全概率公式）：设 $\{B_1, B_2, \ldots\}$ 为样本空间的划分，$\mathbb{P}(B_i) > 0$，则：
$$\mathbb{P}(A) = \sum_{i} \mathbb{P}(A|B_i) \cdot \mathbb{P}(B_i)$$

**证明**：$A = A \cap \Omega = A \cap (\bigcup_i B_i) = \bigcup_i (A \cap B_i)$，由可列可加性：
$$\mathbb{P}(A) = \sum_i \mathbb{P}(A \cap B_i) = \sum_i \mathbb{P}(A|B_i) \cdot \mathbb{P}(B_i)$$

**定理 2.4**（贝叶斯定理的推广形式）：设 $\pi$ 为先验分布，$f(x|\theta)$ 为似然函数，则后验分布为：
$$\pi(\theta|x) = \frac{f(x|\theta) \pi(\theta)}{\int f(x|\theta) \pi(\theta) d\theta}$$

## 三、示例与习题

### 3.1 基础示例

**例 3.1**（简单条件概率）：掷一颗公平骰子，已知点数为偶数，求点数大于3的概率。

**解**：设 $A = \{\text{点数} > 3\} = \{4,5,6\}$，$B = \{\text{偶数}\} = \{2,4,6\}$。

$$\mathbb{P}(A|B) = \frac{\mathbb{P}(A \cap B)}{\mathbb{P}(B)} = \frac{\mathbb{P}(\{4,6\})}{\mathbb{P}(\{2,4,6\})} = \frac{2/6}{3/6} = \frac{2}{3}$$

**例 3.2**（全概率公式应用）：某工厂有三台机器 $M_1, M_2, M_3$，产量占比分别为30%、45%、25%，次品率分别为2%、3%、2%。求随机抽取一件产品为次品的概率。

**解**：设 $D$ 为"次品"事件。

$$\mathbb{P}(D) = \sum_{i=1}^3 \mathbb{P}(D|M_i)\mathbb{P}(M_i) = 0.02 \times 0.30 + 0.03 \times 0.45 + 0.02 \times 0.25 = 0.0245$$

### 3.2 典型示例

**例 3.3**（贝叶斯定理经典应用——医学检测）：某疾病患病率为1%，检测的灵敏度（真阳性率）为95%，特异度（真阴性率）为90%。若某人检测结果为阳性，求其真实患病的概率。

**解**：设 $D$ = "患病"，$T^+$ = "检测阳性"。

$$\mathbb{P}(D|T^+) = \frac{\mathbb{P}(T^+|D)\mathbb{P}(D)}{\mathbb{P}(T^+|D)\mathbb{P}(D) + \mathbb{P}(T^+|D^c)\mathbb{P}(D^c)}$$

$$= \frac{0.95 \times 0.01}{0.95 \times 0.01 + 0.10 \times 0.99} = \frac{0.0095}{0.0095 + 0.099} \approx 0.0876$$

即使检测阳性，真实患病概率仍低于9%！这是因为疾病本身很罕见。

**例 3.4**（Monty Hall问题）：三扇门后分别有一辆车和两只山羊。你选一扇门（假设门1），主持人（知道门后情况）打开另一扇有山羊的门（假设门3），问你是否应该换到门2？

**解**：设 $C_i$ 表示车在门 $i$ 后。初始 $\mathbb{P}(C_i) = 1/3$。

若车在门1（你选的），主持人随机开门2或3；若车在门2，主持人必须开门3；若车在门3，主持人必须开门2。

$$\mathbb{P}(C_2|\text{开3}) = \frac{\mathbb{P}(\text{开3}|C_2)\mathbb{P}(C_2)}{\mathbb{P}(\text{开3})} = \frac{1 \times (1/3)}{(1/2) \times (1/3) + 1 \times (1/3) + 0 \times (1/3)} = \frac{2}{3}$$

换门赢得车的概率是2/3，不换是1/3，应该换！

### 3.3 进阶示例

**例 3.5**（Polya urn模型）： urn中有 $r$ 个红球和 $b$ 个蓝球。每次随机取一球，观察颜色后放回，并额外加入 $c$ 个同色球。求第 $n$ 次取到红球的概率。

**解**：令 $R_n$ 表示第 $n$ 次取红球。有趣的是，通过归纳可证 $\mathbb{P}(R_n) = \frac{r}{r+b}$ 对所有 $n$ 成立！这就是著名的"可交换性"现象。

**证明**：对 $n=1$ 显然。假设对 $n$ 成立，考虑 $n+1$：

$$\mathbb{P}(R_{n+1}) = \mathbb{E}[\mathbb{P}(R_{n+1}|\text{前}n\text{次结果})]$$

利用可交换性，可证明 $\mathbb{P}(R_{n+1}|\text{前}n\text{次有}k\text{次红}) = \frac{r+kc}{r+b+nc}$，取期望后恰好等于 $r/(r+b)$。

### 3.4 反例

**反例 3.6**（条件概率的直觉陷阱）：设有两个孩子，已知至少有一个是男孩，求两个孩子都是男孩的概率。

**错误答案**：1/2（认为另一个孩子是男孩的概率）

**正确答案**：1/3。样本空间为 $\{BB, BG, GB, GG\}$，排除 $GG$ 后，$BB$ 占1/3。

**反例 3.7**（辛普森悖论）：某大学两学院录取数据如下：

| 学院 | 男生申请 | 男生录取 | 女生申请 | 女生录取 |
|------|----------|----------|----------|----------|
| A    | 900      | 450 (50%)| 100      | 80 (80%) |
| B    | 100      | 10 (10%) | 900      | 180 (20%)|
| **总计** | **1000** | **460 (46%)** | **1000** | **260 (26%)** |

每个学院女生录取率都更高，但总体男生录取率更高！这是因为更多女生申请了录取率较低的学院B。

### 3.5 习题与解答

**习题 3.1**（基础）：证明若 $A$ 与 $B$ 独立，则 $A^c$ 与 $B^c$ 也独立。

**解**：$\mathbb{P}(A^c \cap B^c) = 1 - \mathbb{P}(A \cup B) = 1 - \mathbb{P}(A) - \mathbb{P}(B) + \mathbb{P}(A)\mathbb{P}(B) = (1-\mathbb{P}(A))(1-\mathbb{P}(B))$。

**习题 3.2**（中等）：设 $A_1, A_2, \ldots$ 为事件，证明：
$$\mathbb{P}\left(\bigcap_{i=1}^n A_i\right) \geq 1 - \sum_{i=1}^n \mathbb{P}(A_i^c)$$
（并的界/Boole不等式的补形式）

**解**：由De Morgan律和次可加性：
$$\mathbb{P}\left(\bigcap A_i\right) = 1 - \mathbb{P}\left(\bigcup A_i^c\right) \geq 1 - \sum \mathbb{P}(A_i^c)$$

**习题 3.3**（进阶）：设 $(X,Y)$ 在三角形 $\{(x,y): 0 \leq x \leq y \leq 1\}$ 上均匀分布，求 $\mathbb{P}(X \leq 1/2 | Y = y)$。

**解**：条件密度 $f_{X|Y}(x|y) = \frac{f_{X,Y}(x,y)}{f_Y(y)}$。联合密度为2（三角形面积为1/2），边缘密度 $f_Y(y) = \int_0^y 2 dx = 2y$。

$$f_{X|Y}(x|y) = \frac{2}{2y} = \frac{1}{y}, \quad 0 \leq x \leq y$$

$$\mathbb{P}(X \leq 1/2 | Y = y) = \begin{cases} 1 & y \leq 1/2 \\ \frac{1}{2y} & y > 1/2 \end{cases}$$

**习题 3.4**（挑战）：证明 Bayes风险的下界（Van Trees不等式）：设 $\theta$ 为参数，$\hat{\theta}$ 为估计量，则：
$$\mathbb{E}[(\hat{\theta} - \theta)^2] \geq \frac{1}{\mathbb{E}[I(\theta)] + I_0}$$
其中 $I(\theta)$ 为Fisher信息，$I_0$ 为先验的Fisher信息。

**解**：利用贝叶斯版本的Cram\'{e}r-Rao下界，对后验分布应用Cauchy-Schwarz不等式。

**习题 3.5**（综合）：设 $n$ 个犯人排成一列，每人头上戴一顶黑帽或白帽（各1/2概率，独立）。每人能看到前面所有人的帽子但看不到自己和后面人的。从最后一人开始，每人猜自己帽子的颜色（只能说"黑"或"白"）。设计策略使存活人数期望最大。

**解**：策略：最后一人（看到前面 $n-1$ 人）报出前面黑帽数的奇偶性（约定黑=1，白=0）。他能存活概率1/2。其他人根据听到的信息和看到的信息可确定自己帽子颜色，全部存活。期望存活人数：$(n-1) + 1/2 = n - 1/2$。

## 四、形式化实现（Lean 4）

```lean4
import Mathlib

namespace ConditionalProbability

variable {Ω : Type*} [MeasurableSpace Ω] (ℙ : Measure Ω) [IsProbabilityMeasure ℙ]

-- 条件概率的定义
noncomputable def condProb (A B : Set Ω) (hB : MeasurableSet B) (hpos : ℙ B > 0) : ℝ :=
  ℙ (A ∩ B) / ℙ B

notation "ℙ[ " A " | " B "; " hB "; " hpos " ]" => condProb _ A B hB hpos

-- 条件概率的基本性质：非负性
lemma condProb_nonneg (A B : Set Ω) (hB : MeasurableSet B) (hpos : ℙ B > 0) :
    ℙ[A | B; hB; hpos] ≥ 0 := by
  apply div_nonneg
  · apply measure_nonneg
  · linarith

-- 条件概率的基本性质：规范性
lemma condProb_univ (B : Set Ω) (hB : MeasurableSet B) (hpos : ℙ B > 0) :
    ℙ[Set.univ | B; hB; hpos] = 1 := by
  simp [condProb, Set.univ_inter]
  field_simp [hpos]

-- 乘法公式
lemma multiplication_rule (A B : Set Ω) (hA : MeasurableSet A) (hB : MeasurableSet B)
    (hpos : ℙ B > 0) :
    ℙ (A ∩ B) = ℙ[A | B; hB; hpos] * ℙ B := by
  simp [condProb]
  field_simp [hpos]
  rw [mul_comm]

-- 全概率公式（有限情形）
lemma total_probability {n : ℕ} (A : Set Ω) (B : Fin n → Set Ω)
    (hB : ∀ i, MeasurableSet (B i))
    (hdisj : Pairwise (Disjoint on B))
    (hcover : ⋃ i, B i = Set.univ)
    (hpos : ∀ i, ℙ (B i) > 0) :
    ℙ A = ∑ i, ℙ[A | B i; hB i; hpos i] * ℙ (B i) := by
  have h : A = ⋃ i, (A ∩ B i) := by
    rw [← Set.inter_iUnion]
    simp [hcover]
  rw [h]
  rw [measure_iUnion]
  · simp [multiplication_rule ℙ A (B _) hA (hB _) (hpos _)]
  · intro i j hij
    apply Disjoint.mono inter_subset_right inter_subset_right
    exact hdisj hij
  · intro i
    exact MeasurableSet.inter hA (hB i)

-- 贝叶斯定理（简单形式）
lemma bayes_theorem_simple (A B : Set Ω) (hA : MeasurableSet A) (hB : MeasurableSet B)
    (hposA : ℙ A > 0) (hposB : ℙ B > 0) :
    ℙ[B | A; hA; hposA] = ℙ[A | B; hB; hposB] * ℙ B / ℙ A := by
  simp [condProb]
  field_simp [hposA, hposB]
  rw [Set.inter_comm]
  ring

-- 贝叶斯定理（划分形式）
lemma bayes_theorem_partition {n : ℕ} (A : Set Ω) (B : Fin n → Set Ω) (j : Fin n)
    (hA : MeasurableSet A)
    (hB : ∀ i, MeasurableSet (B i))
    (hdisj : Pairwise (Disjoint on B))
    (hcover : ⋃ i, B i = Set.univ)
    (hposA : ℙ A > 0)
    (hposB : ∀ i, ℙ (B i) > 0) :
    ℙ[B j | A; hA; hposA] = 
      ℙ[A | B j; hB j; hposB j] * ℙ (B j) / ∑ i, ℙ[A | B i; hB i; hposB i] * ℙ (B i) := by
  rw [bayes_theorem_simple ℙ (B j) A (hB j) hA hposA (hposB j)]
  have h : ℙ A = ∑ i, ℙ[A | B i; hB i; hposB i] * ℙ (B i) := 
    total_probability ℙ A B hB hdisj hcover hposB
  rw [h]

end ConditionalProbability
```

## 五、应用与拓展

### 5.1 实际应用

**医学诊断**：贝叶斯定理是医学诊断决策的核心工具。结合疾病的先验发病率、检测的灵敏度和特异度，计算后验患病概率。这在罕见病诊断中尤为重要。

**垃圾邮件过滤**：朴素贝叶斯分类器基于贝叶斯定理，假设特征（单词出现）在给定类别（垃圾/正常）下条件独立，高效地进行文本分类。

**金融风控**：在信用评分和欺诈检测中，贝叶斯网络建模变量间的条件依赖关系，进行概率推理和风险评估。

**自动驾驶**：贝叶斯滤波（如Kalman滤波、粒子滤波）用于传感器融合和状态估计，在不确定性环境下进行决策。

### 5.2 与其他分支的联系

- **信息论**：条件熵 $H(X|Y)$ 和互信息 $I(X;Y)$ 基于条件概率定义，是信息压缩和传输的基础
- **统计学**：贝叶斯统计学派以贝叶斯定理为核心，与频率学派形成互补
- **机器学习**：变分推断、期望传播等算法都是贝叶斯推断的近似方法
- **认知科学**：贝叶斯大脑假说认为人脑使用贝叶斯推理处理感知信息

### 5.3 前沿方向

**贝叶斯深度学习**：结合神经网络与贝叶斯方法，量化模型不确定性。变分自编码器（VAE）和贝叶斯神经网络是典型代表。

**因果推断**：Pearl的结构因果模型使用条件概率定义干预和反事实，推动AI从关联到因果的跨越。

**概率编程**：Stan、PyMC、Turing.jl等框架允许用户声明式地指定概率模型，自动进行推断，降低贝叶斯分析门槛。

```
├─ 参数已知？
│  ├─ 是 → 直接计算条件概率
│  └─ 否 → 需要估计参数
│     ├─ 有先验信息？
│     │  ├─ 是 → 使用贝叶斯估计（MAP/后验均值）
│     │  └─ 否 → 使用频率学派估计（MLE）
└─ 结构已知？
   ├─ 是（因果图/贝叶斯网络）
   │  ├─ 树结构 → 信念传播算法
   │  ├─ 单连通 → 多树传播
   │  └─ 多连通 →  junction tree / 采样方法
   └─ 否 → 结构学习 + 参数学习
      ├─ 数据充足 → 约束-based方法
      └─ 数据有限 → 贝叶斯结构学习

计算策略选择：
├─ 解析可解？
│  ├─ 是（共轭先验）→ 直接计算后验
│  └─ 否 → 近似方法
│     ├─ 变分推断（速度快，近似）
│     ├─ MCMC采样（精确但慢）
│     └─ 拉普拉斯近似（局部高斯近似）
```
