# 随机过程基础

随机过程（stochastic process）是概率论中研究随时间演化的随机现象的分支，在物理学（Brown 运动）、金融学（期权定价）、生物学（种群动态）与工程学（排队论）中有着广泛应用。Kolmogorov 于 1933 年建立的公理化体系奠定了现代随机过程理论的基础。

## 1. 马尔可夫链

### 1.1 定义与转移矩阵

**定义 1.1（马尔可夫链）**. 设 $(\Omega, \mathcal{F}, P)$ 为概率空间，$S$ 为至多可数的状态空间。随机过程 $\{X_n\}_{n \geq 0}$ 称为离散时间马尔可夫链（Markov chain），若对所有 $n \geq 0$ 和状态 $i_0, \dots, i_{n+1} \in S$：

$$P(X_{n+1} = i_{n+1} \mid X_0 = i_0, \dots, X_n = i_n) = P(X_{n+1} = i_{n+1} \mid X_n = i_n),$$

只要条件概率有定义。称上述条件概率为一步转移概率，记 $p_{ij} = P(X_{n+1} = j \mid X_n = i)$。

**定义 1.2（转移矩阵）**. 矩阵 $P = (p_{ij})_{i,j \in S}$ 称为转移矩阵（transition matrix），满足 $p_{ij} \geq 0$ 且 $\sum_j p_{ij} = 1$。

**定理 1.3（Chapman-Kolmogorov）**. $n$ 步转移概率 $p_{ij}^{(n)} = P(X_n = j \mid X_0 = i)$ 满足：

$$p_{ij}^{(n+m)} = \sum_{k \in S} p_{ik}^{(n)} p_{kj}^{(m)},$$

即 $P^{(n)} = P^n$（矩阵幂）。

### 1.2 平稳分布与遍历性

**定义 1.4（平稳分布）**. 概率分布 $\pi = (\pi_i)_{i \in S}$ 称为平稳分布（stationary distribution），若 $\pi P = \pi$，即：

$$\sum_i \pi_i p_{ij} = \pi_j, \quad \forall j \in S.$$

**定义 1.5（不可约与正常返）**. 马尔可夫链称为不可约（irreducible），若任意两状态互通（即对某 $n$，$p_{ij}^{(n)} > 0$）。状态 $i$ 称为正常返（positive recurrent），若从 $i$ 出发的期望返回时间 $E_i[T_i] < \infty$，其中 $T_i = \min\{n \geq 1 : X_n = i\}$。

**定理 1.6**. 不可约马尔可夫链存在唯一平稳分布当且仅当所有状态正常返。此时：

$$\pi_i = \frac{1}{E_i[T_i]}.$$

**定理 1.7（遍历定理）**. 对有限不可约非周期马尔可夫链，无论初始分布如何：

$$\lim_{n \to \infty} P(X_n = j) = \pi_j, \quad \forall j \in S.$$

且对任意有界函数 $f$：

$$\frac{1}{n} \sum_{k=0}^{n-1} f(X_k) \xrightarrow{a.s.} \sum_i \pi_i f(i).$$

## 2. Brown 运动

### 2.1 定义与存在性

**定义 2.1（Brown 运动）**. 标准 Brown 运动（Wiener 过程）$\{B_t\}_{t \geq 0}$ 是满足以下条件的随机过程：
1. $B_0 = 0$ a.s.；
2. **独立增量**：对 $0 \leq t_1 < t_2 < \dots < t_n$，$B_{t_2} - B_{t_1}, \dots, B_{t_n} - B_{t_{n-1}}$ 独立；
3. **平稳增量**：$B_t - B_s \sim N(0, t-s)$ 对 $0 \leq s < t$；
4. **连续轨道**：$t \mapsto B_t(\omega)$ 连续对几乎处处的 $\omega$。

**定理 2.2（Wiener）**. 标准 Brown 运动在 $(C[0,\infty), \mathcal{B})$ 上存在且分布唯一（Wiener 测度）。

*证明概要*. 利用 Kolmogorov 扩张定理与一致性条件构造有限维分布。由 Kolmogorov-Chentsov 定理，若过程满足 $E|X_t - X_s|^\alpha \leq C|t-s|^{1+\beta}$（$\alpha, \beta > 0$），则存在 Hölder 连续修正。对 Brown 运动，$E|B_t - B_s|^4 = 3(t-s)^2$，满足 $\alpha = 4, \beta = 1$。$\square$

### 2.2 Brown 运动的鞅性质

**定理 2.3**. 标准 Brown 运动具有以下鞅性质：
1. $\{B_t\}$ 是鞅（关于自然 filtration $\mathcal{F}_t = \sigma(B_s : s \leq t)$）；
2. $\{B_t^2 - t\}$ 是鞅；
3. $\{e^{\theta B_t - \theta^2 t/2}\}$ 是鞅（指数鞅）。

*证明 (1)*. $E[B_t \mid \mathcal{F}_s] = E[B_t - B_s + B_s \mid \mathcal{F}_s] = E[B_t - B_s] + B_s = B_s$。$\square$

## 3. Poisson 过程

### 3.1 定义

**定义 3.1（Poisson 过程）**. 计数过程 $\{N_t\}_{t \geq 0}$ 称为强度 $\lambda > 0$ 的 Poisson 过程，若：
1. $N_0 = 0$；
2. 独立增量；
3. $N_t - N_s \sim \text{Poisson}(\lambda(t-s))$ 对 $s < t$；
4. 轨道右连续且具左极限（càdlàg）。

**定理 3.2**. Poisson 过程的到达间隔时间 $\tau_n = T_n - T_{n-1}$（$T_n = \inf\{t : N_t = n\}$）为独立同分布的指数随机变量，$\tau_n \sim \text{Exp}(\lambda)$。

## 4. 例子

### 4.1 例子：随机游走的尺度极限

设 $\{X_n\}$ 为简单对称随机游走，$S_n = X_1 + \dots + X_n$。定义连续过程：

$$B_t^{(n)} = \frac{1}{\sqrt{n}} S_{\lfloor nt \rfloor}.$$

**定理 4.1（Donsker）**. 在 $C[0,1]$ 上，$B^{(n)}$ 弱收敛于标准 Brown 运动 $B$（Wiener 测度）。

这是中心极限定理的函数形式，说明 Brown 运动是随机游走的普适尺度极限。

### 4.2 例子：生灭过程

生灭过程是状态空间 $\mathbb{N}$ 上的连续时间马尔可夫链，从 $n$ 只能转移到 $n+1$（出生，率 $\lambda_n$）或 $n-1$（死亡，率 $\mu_n$）。

对线性生灭过程 $\lambda_n = n\lambda$，$\mu_n = n\mu$，灭绝概率 $q$ 满足：

$$q = \begin{cases} 1 & \lambda \leq \mu, \\ (\mu/\lambda)^{n_0} & \lambda > \mu, \end{cases}$$

其中 $n_0$ 为初始种群大小。
