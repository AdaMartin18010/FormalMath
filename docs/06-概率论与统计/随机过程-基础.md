# 随机过程基础

## 马尔可夫链

### 定义

随机过程$\{X_n\}$是马尔可夫的若：
$$P(X_{n+1} = j | X_n = i, X_{n-1}, ..., X_0) = P(X_{n+1} = j | X_n = i) = p_{ij}$$

### 转移矩阵

$P = (p_{ij})$，$p_{ij} \geq 0$，$\sum_j p_{ij} = 1$

### 平稳分布

$\pi$满足$\pi P = \pi$，$\sum_i \pi_i = 1$。

---

## 布朗运动

### 定义

标准布朗运动$\{B_t\}_{t \geq 0}$满足：

- $B_0 = 0$
- 独立增量
- $B_t - B_s \sim N(0, t-s)$
- 连续路径
