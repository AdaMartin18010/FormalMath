## 系统概述

### 系统简介

FormalMath自适应学习路径系统是一个基于知识图谱和认知诊断的个性化学习平台。系统能够：

- **分析学习者特征**：认知风格、能力水平、学习目标、时间可用性
- **生成个性化路径**：基于A*算法和多目标优化的最优学习路径
- **实时调整**：根据学习进度动态调整路径难度和内容
- **提供学习支持**：资源推荐、伙伴匹配、难点预警、成就激励

### 系统架构

```

自适应学习路径系统
├── 学习者特征分析模块
│   ├── 认知诊断结果导入
│   ├── 学习风格评估
│   ├── 先验知识评估
│   ├── 学习目标设定
│   └── 可用时间评估
├── 个性化学习路径生成模块
│   ├── A*路径规划算法
│   ├── 多目标优化
│   └── 路径多样性保证
├── 实时调整引擎
│   ├── 进度监控
│   ├── 困难检测
│   ├── 路径动态调整
│   ├── 间隔重复复习
│   └── 效果反馈
└── 学习支持系统
    ├── 资源推荐
    ├── 学习伙伴匹配
    ├── 难点预警
    └── 成就系统

```

### 技术栈

- **后端**: Python + FastAPI
- **算法**: NetworkX（图算法）+ scikit-learn（机器学习）
- **数据库**: PostgreSQL（结构化数据）+ Neo4j（知识图谱）
- **前端**: React + CSS3

## 学习者特征分析

### 1. 创建学习者档案

```bash
POST /api/v1/learners
{
    "name": "张三",
    "email": "zhangsan@example.com"
}

```

### 2. 学习风格评估

系统提供5道问卷题目，评估四种学习风格：
- **视觉型**：偏好图表、图像、视频
- **听觉型**：偏好音频、讲解、讨论
- **动手型**：偏好实践、操作、体验
- **阅读型**：偏好文字、阅读、笔记

```bash
POST /api/v1/learners/{learner_id}/learning-style
{
    "answers": {
        "visual_1": "visual",
        "visual_2": "reading",
        ...
    }
}

```

### 3. 先验知识评估

评估学习者对数学概念的熟悉程度：

```bash
POST /api/v1/learners/{learner_id}/prior-knowledge
{
    "concept_checks": {
        "集合": true,
        "函数": false,
        "群": false
    }
}

```

### 4. 设置学习目标

```bash
POST /api/v1/learners/{learner_id}/goals
{
    "target_concepts": ["群", "环", "域"],
    "deadline": "2026"-06-01T00:00:00",
    "target_level": 2,
    "description": "掌握基础代数结构"
}

```

### 5. 设置可用时间

```bash
POST /api/v1/learners/{learner_id}/time-availability
{
    "daily_hours": 2,
    "weekly_days": 5,
    "preferred_time": "evening",
    "max_session": 60
}

```

### 6. 导入认知诊断结果

从T2.1认知诊断系统导入结果：

```bash
POST /api/v1/learners/{learner_id}/diagnosis-import
{
    "knowledge_state": {
        "集合": 0.8,
        "函数": 0.6
    },
    "ability_level": {
        "L0": 0.9,
        "L1": 0.7,
        "L2": 0.4,
        "L3": 0.1
    }
}

```

## 学习路径生成

### 生成学习路径

```bash
POST /api/v1/learning-path/generate
{
    "learner_id": "learner-uuid",
    "goal_concepts": ["群", "环", "域"],
    "max_time_hours": 100,
    "difficulty_range_min": 0.5,
    "difficulty_range_max": 1.5,
    "diversity_weight": 0.2
}

```

响应示例：

```json
{
    "recommendation_id": "rec-uuid",
    "paths": [
        {
            "path_id": "path-uuid",
            "name": "个性化学习路径 - 2026-04-03",
            "description": "基于A*算法生成的学习路径...",
            "total_nodes": 15,
            "estimated_hours": 45.5,
            "match_score": 0.85,
            "reason": "学习内容多样；时间效率高"
        }
    ]
}

```

### 路径生成算法

系统使用**A*算法**结合**多目标优化**生成路径：

1. **知识缺口分析**：识别从当前状态到目标所需的概念
2. **拓扑排序**：根据先修关系确定学习顺序
3. **A*搜索**：寻找最优路径
4. **多目标优化**：平衡时间、深度、广度、平滑度

### 路径评估指标

- **平滑度**：难度曲线的连续性
- **覆盖率**：目标概念的覆盖程度
- **效率**：单位时间的学习效果
- **多样性**：MSC分类的分布多样性

## 实时调整机制

### 进度监控

```bash
POST /api/v1/learning-path/{path_id}/progress
{
    "node_id": "node-uuid",
    "progress": 0.75,
    "mastery": 0.8
}

```

### 困难检测

系统自动检测学习困难：

| 检测指标 | 阈值 | 信号 |
|---------|------|------|
| 时间比例 | > 2.0 | 内容过难 |
| 错误率 | > 50% | 需要复习 |
| 帮助请求 | ≥ 3次 | 需要支持 |

### 路径调整

```bash
GET /api/v1/learning-path/{path_id}/adjust

```

调整策略：
- **难度调整**：根据表现增加或降低难度
- **内容增删**：添加支持内容或跳过已掌握内容
- **时间重分配**：根据进度重新分配时间
- **复习插入**：基于遗忘曲线插入复习节点

### 间隔重复复习

基于艾宾浩斯遗忘曲线的复习计划：

```python
# 掌握度 >= 0.9: [7, 14, 30, 60, 120] 天
# 掌握度 >= 0.8: [3, 7, 14, 30, 60] 天
# 掌握度 >= 0.7: [1, 3, 7, 14, 30] 天
# 掌握度 < 0.7:  [1, 2, 4, 7, 14] 天

```

## 学习支持系统

### 资源推荐

```bash
GET /api/v1/resources/recommend?learner_id={id}&concept_id={id}&count=5

```

推荐算法考虑：
- 学习风格匹配（40%）
- 难度适宜性（30%）
- 先验知识（20%）
- 时间效率（10%）

### 学习伙伴匹配

```bash
POST /api/v1/peers/register
{
    "current_concepts": ["群", "环"],
    "available_times": ["evening"]
}

GET /api/v1/peers/match?learner_id={id}

```

匹配因素：
- 概念重叠（30%）
- 学习风格互补（20%）
- 能力水平相近（30%）
- 时间可用性（20%）

### 难点预警

```bash
GET /api/v1/warnings/analyze?learner_id={id}&concept_id={id}

```

预警类型：
- **先修知识缺失**：需要先学习先修概念
- **复杂度过高**：当前能力不足以学习该概念
- **难度跳跃**：与前一概念难度差距过大
- **时间密集**：需要较多学习时间

### 成就系统

```bash
GET /api/v1/achievements?learner_id={id}
POST /api/v1/achievements/check

```

成就类别：
- 📚 学习成就：完成概念学习
- 📅 坚持成就：连续学习天数
- ⭐ 掌握成就：达到高掌握度
- 🤝 社交成就：与伙伴互动
