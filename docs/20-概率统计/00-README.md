# 概率统计 / Probability and Statistics

**最后更新**: 2026年4月4日

---

## 📋 分支概述

概率统计是研究随机现象数量规律的数学分支，包括概率论和数理统计两大部分。概率论研究随机现象的数学模型和规律，数理统计则研究如何从数据中提取信息和进行推断。本分支系统构建概率统计的完整理论体系，从基础概念到核心定理，再到实际应用。

## 📚 核心人物

- **Andrey Kolmogorov (1903-1987)**: 现代概率论奠基人，建立公理化体系
- **Pierre-Simon Laplace (1749-1827)**: 古典概率论集大成者
- **Carl Friedrich Gauss (1777-1855)**: 正态分布（高斯分布）
- **Ronald A. Fisher (1890-1962)**: 现代统计学奠基人
- **Jerzy Neyman (1894-1981) & Egon Pearson (1895-1980)**: 假设检验理论

## 🔗 与其他分支的联系

- **实分析**: 测度论与积分理论是概率论的严格基础
- **泛函分析**: 随机过程与随机分析需要泛函分析工具
- **线性代数**: 协方差矩阵、多元正态分布
- **优化理论**: 最大似然估计、贝叶斯估计
- **信息论**: 熵、互信息、信道容量

---

## 📖 学习路径建议

1. **先修知识**: 微积分、线性代数、基础集合论
2. **第一阶段**: 概率论基础（概率空间、随机变量、分布）
3. **第二阶段**: 数字特征与极限定理（期望、方差、大数定律、CLT）
4. **第三阶段**: 数理统计（估计、检验、回归）
5. **第四阶段**: 高级专题（随机过程、贝叶斯统计、非参数统计）
