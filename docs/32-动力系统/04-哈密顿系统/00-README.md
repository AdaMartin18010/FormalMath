# 哈密顿系统

## 概述

哈密顿系统描述能量守恒的经典力学系统，是动力系统中最优美、最深刻的分支之一，连接微分几何、辛拓扑与可积系统理论。

### KAM理论
- 无理旋转的保持

- 近可积系统的稳定性

## 数学思想

哈密顿系统揭示：**守恒律与对称性深刻联系**。Noether定理表明对称性即守恒量，可积系统是完美对称的极致表现。

## 历史发展

- **Hamilton (1834)**: 重写牛顿方程
- **Liouville (1855)**: 可积系统理论
- **Poincaré**: 天体力学，混沌发现
- **Kolmogorov (1954)**: KAM定理
- **Arnold & Moser (1960s)**: KAM严格证明
- **Nekhoroshev (1977)**: 有效稳定性

## 与其他主题的联系

- 常微分方程定性理论的特殊情形
- 光滑动力系统中的保体积系统
- 量子力学的经典极限
- 代数几何（代数可积系统）
