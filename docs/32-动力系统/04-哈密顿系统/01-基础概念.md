# 哈密顿系统 - 基础概念

## 一、辛几何基础

### 辛流形

**定义**: 偶数维流形 $M^{2n}$ 配备闭的非退化2-形式 $\omega$：
- **闭**: $d\omega = 0$
- **非退化**: $\omega^n \neq 0$（处处）

**标准例子**: $(\mathbb{R}^{2n}, \omega_0)$
$$\omega_0 = \sum_{i=1}^n dp_i \wedge dq_i = dp \wedge dq$$

### Darboux定理

**定理**: 任意辛流形局部辛同构于 $(\mathbb{R}^{2n}, \omega_0)$。

即存在局部坐标 $(q, p)$ 使得 $\omega = \sum dq_i \wedge dp_i$。

### Hamilton向量场

对函数 $H: M \to \mathbb{R}$，定义**Hamilton向量场** $X_H$：
$$\iota_{X_H}\omega = dH$$

即 $\omega(X_H, Y) = dH(Y)$ 对所有 $Y$。

## 二、Hamilton方程

### 标准形式

在Darboux坐标 $(q, p)$ 下：
$$X_H = \sum_{i=1}^n \left(\frac{\partial H}{\partial p_i}\frac{\partial}{\partial q_i} - \frac{\partial H}{\partial q_i}\frac{\partial}{\partial p_i}\right)$$

轨道 $(q(t), p(t))$ 满足**Hamilton方程**：
$$\dot{q}_i = \frac{\partial H}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial H}{\partial q_i}$$

### 能量守恒

$$\frac{dH}{dt} = \sum_i \left(\frac{\partial H}{\partial q_i}\dot{q}_i + \frac{\partial H}{\partial p_i}\dot{p}_i\right) = 0$$

Hamilton流保持能量曲面 $H = E$。

## 三、泊松括号

### 定义

$$\{F, G\} = \sum_{i=1}^n \left(\frac{\partial F}{\partial q_i}\frac{\partial G}{\partial p_i} - \frac{\partial F}{\partial p_i}\frac{\partial G}{\partial q_i}\right) = \omega(X_F, X_G)$$

### 性质

- **反对称**: $\{F, G\} = -\{G, F\}$
- **Jacobi恒等式**: $\{\{F, G\}, H\} + \{\{G, H\}, F\} + \{\{H, F\}, G\} = 0$
- **Leibniz法则**: $\{FG, H\} = F\{G, H\} + G\{F, H\}$

### 运动方程

$$\dot{F} = \{F, H\}$$

函数 $F$ 是**运动常数**（首次积分）当且仅当 $\{F, H\} = 0$。

## 四、可积系统

### Liouville可积性

**定义**: Hamilton系统 $(M, \omega, H)$ 是**可积的**，如果存在 $n = \frac{1}{2}\dim M$ 个独立的运动常数 $F_1 = H, F_2, ..., F_n$，两两对合：
$$\{F_i, F_j\} = 0, \quad \forall i, j$$

### Arnold-Liouville定理

**定理**: 若系统可积，则：
1. 非奇异等能集（$F_i = c_i$，$dF_i$ 独立）是 $n$ 维环面 $T^n$
2. Hamilton流在此环面上是拟周期运动
3. 存在作用-角变量

## 五、作用-角变量

### 定义

**作用变量** ($I$): 
$$I_i = \frac{1}{2\pi}\oint_{\gamma_i} p \, dq$$

其中 $\gamma_i$ 是环面上的基本循环。

**角变量** ($\theta$): 环面上的角坐标，$\theta_i \in [0, 2\pi)$。

### 变换性质

在此变量下，Hamiltonian 只依赖于 $I$：
$$H = H(I)$$

运动方程：
$$\dot{I}_i = 0, \quad \dot{\theta}_i = \omega_i(I) = \frac{\partial H}{\partial I_i}$$

解：
$$I(t) = I(0), \quad \theta(t) = \theta(0) + \omega(I)t$$

**频率** $\omega(I)$ 决定环面上的运动。

## 六、对称性与约化

### Noether定理

Hamilton版本：若 $F$ 生成单参数对称群，则 $F$ 是运动常数。

### 矩映射

Lie群 $G$ 作用在辛流形上，**矩映射** $\mu: M \to \mathfrak{g}^*$ 满足：
$$d\langle\mu, \xi\rangle = \iota_{\xi_M}\omega$$

### Marsden-Weinstein约化

在 $\mu^{-1}(0)/G$ 上诱导辛结构，降维Hamilton系统。

## 七、近可积系统

### 定义

$$H = H_0(I) + \epsilon H_1(I, \theta)$$

其中 $H_0$ 可积，$\epsilon \ll 1$。

### KAM问题

- 哪些环面在小扰动下保持？
- 需要频率 $\omega(I)$ 满足**Diophantine条件**：
$$|k \cdot \omega| \geq \frac{\gamma}{|k|^\tau}, \quad \forall k \in \mathbb{Z}^n \setminus \{0\}$$

## 九、标准例子

### 谐振子
$$H = \frac{1}{2}(p^2 + q^2)$$
- 作用变量：$I = \frac{1}{2}(p^2 + q^2) = E$
- 角变量：$\theta = \arctan(q/p)$
- 频率：$\omega = 1$

### 开普勒问题
$$H = \frac{|p|^2}{2} - \frac{1}{|q|}$$

- 可积（4个运动常数）
- 轨道：椭圆/抛物线/双曲线

### 刚体旋转 (Euler)
$$H = \frac{I_1^2}{2J_1} + \frac{I_2^2}{2J_2} + \frac{I_3^2}{2J_3}$$
- 一般情形下非可积
