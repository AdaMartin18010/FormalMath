# 无穷维动力系统 - 实战问题

## 问题1: 反应扩散方程的行波

### 问题陈述

Fisher方程：$u_t = u_{xx} + u(1-u)$

(a) 寻找行波解 $u(x,t) = \phi(x - ct)$ 的形式
(b) 确定最小波速
(c) 分析波的稳定性

### 解答

**(a) 行波方程**

令 $\xi = x - ct$，$u(x,t) = \phi(\xi)$：

$$-c\phi' = \phi'' + \phi(1-\phi)$$

即：
$$\phi'' + c\phi' + \phi(1-\phi) = 0$$

ODE系统（设 $\psi = \phi'$）：
$$\phi' = \psi$$
$$\psi' = -c\psi - \phi(1-\phi)$$

**(b) 最小波速**

平衡点：$(0, 0)$ 和 $(1, 0)$

在 $(0, 0)$ 线性化：
$$\begin{pmatrix} 0 & 1 \\ -1 & -c \end{pmatrix}$$

特征值：$\lambda^2 + c\lambda + 1 = 0$

$$\lambda = \frac{-c \pm \sqrt{c^2 - 4}}{2}$$

物理意义：波前从 $u = 1$ 到 $u = 0$ 单调递减。

需要 $(0,0)$ 是稳定结点或鞍点：$c \geq 2$

**最小波速**：$c^* = 2$

**(c) 稳定性**

Fisher方程的行波在 $c \geq 2$ 时是渐近稳定的（在移动坐标系中）。

## 问题2: Navier-Stokes能量估计

### 问题陈述

2D Navier-Stokes方程，区域 $\Omega$：
$$u_t + (u \cdot \nabla)u = \nu\Delta u - \nabla p + f$$
$$\nabla \cdot u = 0$$

(a) 推导能量不等式
(b) 证明存在吸收集
(c) 估计吸引子维数

### 解答

**(a) 能量不等式**

与 $u$ 做 $L^2$ 内积：
$$\frac{1}{2}\frac{d}{dt}|u|^2 + \nu|\nabla u|^2 = (f, u)$$

由Poincaré不等式 $|\nabla u|^2 \geq \lambda_1 |u|^2$：
$$\frac{d}{dt}|u|^2 + \nu\lambda_1|u|^2 \leq \frac{|f|^2}{\nu\lambda_1}$$

**(b) 吸收集**

解微分不等式：
$$|u(t)|^2 \leq |u_0|^2 e^{-\nu\lambda_1 t} + \frac{|f|^2}{\nu^2\lambda_1^2}(1 - e^{-\nu\lambda_1 t})$$

当 $t \to \infty$：
$$\limsup_{t \to \infty} |u(t)|^2 \leq \frac{|f|^2}{\nu^2\lambda_1^2} =: \rho_0^2$$

吸收球：$B = \{u: |u| \leq \rho_0 + 1\}$

**(c) 维数估计**

全局吸引子 $\mathcal{A}$ 的Hausdorff维数：
$$\dim_H \mathcal{A} \leq C \cdot G$$

其中Grashof数：
$$G = \frac{|f|}{\nu^2\lambda_1}$$

## 问题3: Turing模式分析

### 问题陈述

Gray-Scott模型：
$$u_t = D_u \Delta u - uv^2 + F(1-u)$$
$$v_t = D_v \Delta v + uv^2 - (F+k)v$$

分析参数空间中Turing不稳定性的条件。

### 解答

**(a) 常数稳态**

$(u_0, v_0) = (1, 0)$ 是一个平凡稳态。

非平凡稳态需要数值求解。

**(b) 线性稳定性分析**

在 $(u_0, v_0)$ 附近线性化，考虑扰动 $e^{\lambda t + ikx}$：

特征方程：
$$\det\begin{pmatrix} -D_u k^2 - F - \lambda & 0 \\ 0 & -D_v k^2 - (F+k) - \lambda \end{pmatrix} = 0$$

ODE稳定（无扩散）。

**(c) 扩散驱动的Turing不稳定性**

对于非平凡稳态 $(u_s, v_s)$：

Jacobin：
$$J = \begin{pmatrix} -v_s^2 - F & -2u_s v_s \\ v_s^2 & 2u_s v_s - (F+k) \end{pmatrix}$$

Turing条件：

1. $\text{tr}(J) < 0$，$\det(J) > 0$（ODE稳定）
2. $D_v J_{11} + D_u J_{22} > 0$（扩散失稳）
3. $(D_v J_{11} + D_u J_{22})^2 > 4D_u D_v \det(J)$

参数范围：通常在 $D_v \gg D_u$ 时发生。

## 问题4: Galerkin逼近误差

### 问题陈述

Burgers方程：$u_t + uu_x = \nu u_{xx}$，周期边界条件。

(a) 设置Galerkin逼近
(b) 估计逼近误差

### 解答

**(a) Galerkin设置**

Fourier基：$e^{ikx}$，$k = -N, ..., N$

$$u_N(x,t) = \sum_{k=-N}^N \hat{u}_k(t) e^{ikx}$$

投影方程：
$$\frac{d\hat{u}_k}{dt} + \nu k^2 \hat{u}_k + \widehat{(u_N u_{N,x})}_k = 0$$

**(b) 误差估计**

设 $u$ 是精确解，$u_N$ 是Galerkin解，$w = u - u_N$。

标准估计：
$$\|w(t)\|_{L^2} \leq C \frac{\|u\|_{H^s}}{N^{s-1}}$$

对于光滑解（$u \in H^s$，$s$ 大），误差指数衰减。

## 问题5: 惯性流形验证

### 问题陈述

Chafee-Infante方程：
$$u_t = u_{xx} + \lambda u - u^3, \quad x \in (0, \pi)$$
$$u(0) = u(\pi) = 0$$

(a) 验证谱间隙条件
(b) 证明惯性流形存在

### 解答

**(a) 线性算子**

$A = -\frac{\partial^2}{\partial x^2}$，特征值 $\lambda_n = n^2$，$n = 1, 2, ...$

特征函数：$\phi_n(x) = \sin(nx)$

谱间隙：$\lambda_{N+1} - \lambda_N = (N+1)^2 - N^2 = 2N + 1$

**(b) 非线性**

$F(u) = \lambda u - u^3$，在 $L^2(0, \pi)$ 上局部Lipschitz。

通过截断技术可假设整体Lipschitz常数 $K$。

**(c) 谱间隙条件**

需要：
$$2N + 1 > 2K(\sqrt{(N+1)^2} + \sqrt{N^2}) = 2K(2N + 1)$$

即 $K < 1/2$。

若Lipschitz常数足够小，惯性流形存在。

更精细分析（利用Sobolev嵌入）可得对任意 $\lambda$ 存在惯性流形。
