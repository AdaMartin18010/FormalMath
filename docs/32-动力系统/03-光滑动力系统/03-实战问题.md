# 光滑动力系统 - 实战问题

## 问题1: 双曲不动点分析

### 问题陈述

设 Hénon 映射 $H(x, y) = (1 - ax^2 + y, bx)$，$a = 1.4$，$b = 0.3$。

(a) 找到所有不动点
(b) 判定哪些不动点是双曲的
(c) 计算稳定和不稳定子空间的维数

### 解答

**(a) 不动点**

解 $H(x, y) = (x, y)$：
- $x = 1 - ax^2 + y$
- $y = bx$

代入得：$x = 1 - ax^2 + bx$，即 $ax^2 + (1-b)x - 1 = 0$

$$x = \frac{-(1-b) \pm \sqrt{(1-b)^2 + 4a}}{2a}$$

代入 $a = 1.4$，$b = 0.3$：
$$x = \frac{-0.7 \pm \sqrt{0.49 + 5.6}}{2.8} = \frac{-0.7 \pm 2.46}{2.8}$$

- $x_1 = 0.629$，$y_1 = 0.189$
- $x_2 = -1.129$，$y_2 = -0.339$

**(b) Jacobi矩阵**

$$DH = \begin{pmatrix} -2ax & 1 \\ b & 0 \end{pmatrix}$$

**在 $(x_1, y_1)$**：
$$DH = \begin{pmatrix} -1.761 & 1 \\ 0.3 & 0 \end{pmatrix}$$

特征值：$\lambda^2 + 1.761\lambda - 0.3 = 0$
- $\lambda = \frac{-1.761 \pm \sqrt{3.101 + 1.2}}{2} = \frac{-1.761 \pm 2.08}{2}$
- $\lambda_1 = 0.16$，$\lambda_2 = -1.92$

$|\lambda_1| < 1 < |\lambda_2|$，是**鞍点**（双曲）。

**在 $(x_2, y_2)$**：
计算类似，也是鞍点。

**(c) 维数**

每个鞍点：
- 稳定子空间 $\dim E^s = 1$（$|\lambda| < 1$ 的方向）
- 不稳定子空间 $\dim E^u = 1$（$|\lambda| > 1$ 的方向）

## 问题2: Smale马蹄构造

### 问题陈述

设 $f: \mathbb{R}^2 \to \mathbb{R}^2$ 是线性映射与弯曲的复合：
1. 水平压缩因子 $\lambda < 1/2$
2. 竖直扩张因子 $\mu > 2$
3. 水平弯曲形成马蹄形状

(a) 证明存在不变Cantor集
(b) 证明 $f$ 在该集上共轭于移位 $\sigma: \Sigma_2 \to \Sigma_2$

### 解答

**(a) 构造Cantor集**

定义：
- **竖直条带** $V_1, V_2$: $f(Q) \cap Q$ 的两个连通分支
- **水平条带** $H_1, H_2$: $f^{-1}(V_1), f^{-1}(V_2)$

迭代构造：
$$\Lambda = \bigcap_{n=-\infty}^{\infty} f^n(Q)$$

= 所有永不离散 $Q$ 的点的集合。

由压缩映射原理，$\Lambda$ 是Cantor集 $\times$ Cantor集。

**(b) 共轭于移位**

定义编码 $\pi: \Sigma_2 \to \Lambda$:

对 $s = (...s_{-1}s_0s_1...) \in \Sigma_2$，定义：
$$\pi(s) = \bigcap_{n=-\infty}^{\infty} f^{-n}(H_{s_n})$$

由压缩性，此交集为单点。

验证 $f \circ \pi = \pi \circ \sigma$:
$$f(\pi(s)) = f\left(\bigcap_{n} f^{-n}(H_{s_n})\right) = \bigcap_{n} f^{-(n-1)}(H_{s_n}) = \pi(\sigma(s))$$

$\pi$ 是同胚，得证。

## 问题3: Lyapunov指数计算

### 问题陈述

设 $f(x) = 2x \pmod{1}$ 在 $[0, 1]$ 上，Lebesgue测度不变。

(a) 计算Lyapunov指数
(b) 验证Oseledets定理

### 解答

**(a) Lyapunov指数**

$f'(x) = 2$（a.e.）

$$\lambda = \lim_{n \to \infty} \frac{1}{n}\sum_{k=0}^{n-1}\log|f'(f^k(x))| = \log 2$$

对所有非二进有理点（满测度集）成立。

**(b) Oseledets验证**

- 遍历性：$f$ 关于Lebesgue测度遍历
- Lyapunov指数几乎处处为 $\log 2$
- 分解：$E_1 = \mathbb{R}$，平凡分解

符合Oseledets定理。

## 问题4: Anosov环面自同构

### 问题陈述

设 $A = \begin{pmatrix} 2 & 1 \\ 1 & 1 \end{pmatrix}$，诱导环面 $T^2$ 上的映射 $f_A$。

(a) 证明 $f_A$ 是Anosov微分同胚
(b) 计算稳定/不稳定叶层
(c) 证明 $f_A$ 是遍历的

### 解答

**(a) Anosov性**

特征值：$\det(A - \lambda I) = \lambda^2 - 3\lambda + 1 = 0$

$$\lambda = \frac{3 \pm \sqrt{5}}{2}$$

- $\lambda_1 = \frac{3+\sqrt{5}}{2} > 1$（不稳定）
- $\lambda_2 = \frac{3-\sqrt{5}}{2} < 1$（稳定）

特征向量张成不变子空间 $E^u, E^s$，在环面上给出双曲分解。

**(b) 稳定/不稳定叶层**

- **不稳定叶层** $W^u$: 平行于 $E^u$ 特征向量的直线投影
- **稳定叶层** $W^s$: 平行于 $E^s$ 特征向量的直线投影

叶层是稠密的（无理斜率）。

**(c) 遍历性**

由Anosov遍历定理：体积保持的Anosov微分同胚是遍历的。

或直接用Fourier分析：
$$\int f \circ f_A \cdot \bar{f} = 0 \text{ for } f \perp 1$$

## 问题5: 结构稳定性验证

### 问题陈述

设 $f: S^1 \to S^1$，$f(x) = x + \alpha \pmod{1}$，$\alpha \notin \mathbb{Q}$。

(a) $f$ 是结构稳定的吗？
(b) 如果改为 $g(x) = x + \epsilon\sin(2\pi x)$，分析稳定性

### 解答

**(a) 无理旋转的稳定性**

**不是**结构稳定的。

小扰动 $\alpha \to \alpha + \epsilon$，若 $\alpha + \epsilon \in \mathbb{Q}$，则产生周期点，拓扑结构完全不同。

**(b) 扰动分析**

$g(x) = x + \epsilon\sin(2\pi x)$

$g'(x) = 1 + 2\pi\epsilon\cos(2\pi x)$

对 $0 < \epsilon < \frac{1}{2\pi}$：
- $g'(x) > 0$，$g$ 是圆周同胚
- 旋转数 $\rho(g)$ 连续依赖于 $\epsilon$

若 $\rho(g) \notin \mathbb{Q}$，$g$ 共轭于无理旋转（Denjoy定理要求 $g \in C^2$）。
