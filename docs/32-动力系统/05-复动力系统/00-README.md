# 复动力系统

## 概述

复动力系统研究复平面上有理函数或整函数的迭代，连接复分析、几何与动力系统，以Julia集和Mandelbrot集的精美图像闻名于世。

## 关键主题

### Julia集与Fatou集

- Fatou集：正规性区域（稳定动力学）
- Julia集：混沌边界（不稳定动力学）
- 填充Julia集
- 连通常数

### Mandelbrot集

- 连通性轨迹
- 双曲分量的结构
- 重整化与Misiurewicz点
- 局部连通性猜想

## 数学思想

复动力系统揭示：**简单迭代产生极端复杂性**。$z \mapsto z^2 + c$ 的迭代产生分形几何与深刻数学理论。

## 历史发展

- **Fatou & Julia (1918-1920)**: 奠基工作
- **Brolin (1965)**: 平衡测度
- **Douady & Hubbard (1980s)**: Mandelbrot集系统研究
- **Sullivan (1985)**: 无游荡域定理
- **Yoccoz (1990s)**: 线性化与Brjuno条件
- **McMullen (1990s)**: 重整化理论

## 与其他主题的联系

- 单复变函数的几何理论
- Kleinian群（离散群）
- 统计物理（Lee-Yang零点）
- 数论（Diophantine逼近）
