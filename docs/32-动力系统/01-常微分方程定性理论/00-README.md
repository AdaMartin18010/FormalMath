# 常微分方程定性理论

## 概述

定性理论研究微分方程解的长期行为，不依赖于显式求解方程，而是通过分析相空间的几何结构来理解系统动力学。

## 数学思想

定性理论的核心在于：**几何优先于代数**。通过相空间的拓扑结构，我们可以洞察系统的本质行为。

## 历史发展

- **Poincaré (1880s)**: 奠基人，引入相空间概念
- **Lyapunov (1892)**: 稳定性理论的系统发展
- **Birkhoff (1920s)**: 动力系统一般理论
- **Andronov & Pontryagin (1937)**: 结构稳定性

## 与其他主题的联系

- 为拓扑动力系统提供具体例子
- 光滑动力系统的基础
- 哈密顿系统的特例
