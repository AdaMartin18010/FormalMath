## 题目

**问题**：说明由内积 \(\langle \cdot,\cdot \rangle\) 定义的 \(\|x\| = \sqrt{\langle x,x \rangle}\) 满足范数公理；并写出 Cauchy–Schwarz 不等式及其取等条件。

---

## 完整解答（工作示例）

**步骤 1**：非负性、齐次性显然。三角不等式：\(\|x+y\|^2 = \|x\|^2 + 2\mathrm{Re}\langle x,y\rangle + \|y\|^2 \leq \|x\|^2 + 2|\langle x,y\rangle| + \|y\|^2 \leq \|x\|^2 + 2\|x\|\|y\| + \|y\|^2 = (\|x\|+\|y\|)^2\)，其中用了 \(|\langle x,y\rangle| \leq \|x\|\|y\|\)（Cauchy–Schwarz）。故 \(\|x+y\| \leq \|x\|+\|y\|\)。

**步骤 2**：Cauchy–Schwarz：\(|\langle x,y \rangle| \leq \|x\| \|y\|\)。取等当且仅当 \(x,y\) 线性相关（若 \(y \neq 0\)，取等当且仅当 \(x = \lambda y\) 对某 \(\lambda\)）。
