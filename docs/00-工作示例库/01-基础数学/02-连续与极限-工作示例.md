## 题目

设 \(f\) 在 \(x_0\) 的某邻域有定义。说明“\(f\) 在 \(x_0\) 连续”与“\(\lim_{x\to x_0} f(x) = f(x_0)\)”的等价性，并写出 \(\varepsilon\)-\(\delta\) 表述。

---

## 完整解答（工作示例）

**定义回顾**

- **极限**：\(\lim_{x\to x_0} f(x) = L\) 指：对任意 \(\varepsilon > 0\)，存在 \(\delta > 0\)，使得当 \(0 < |x - x_0| < \delta\) 时有 \(|f(x) - L| < \varepsilon\)。
- **连续**：\(f\) 在 \(x_0\) 连续通常定义为：对任意 \(\varepsilon > 0\)，存在 \(\delta > 0\)，使得当 \(|x - x_0| < \delta\) 时有 \(|f(x) - f(x_0)| < \varepsilon\)。

**等价性**

- 若 \(\lim_{x\to x_0} f(x) = f(x_0)\)，则取 \(L = f(x_0)\)，对任意 \(\varepsilon > 0\) 存在 \(\delta > 0\) 使 \(0 < |x-x_0| < \delta \Rightarrow |f(x)-f(x_0)| < \varepsilon\)；当 \(x = x_0\) 时 \(|f(x)-f(x_0)| = 0 < \varepsilon\)，故当 \(|x-x_0| < \delta\) 时均有 \(|f(x)-f(x_0)| < \varepsilon\)，即 \(f\) 在 \(x_0\) 连续。
- 若 \(f\) 在 \(x_0\) 连续，则对任意 \(\varepsilon > 0\) 存在 \(\delta > 0\) 使 \(|x-x_0| < \delta \Rightarrow |f(x)-f(x_0)| < \varepsilon\)，特别地对 \(0 < |x-x_0| < \delta\) 也成立，故 \(\lim_{x\to x_0} f(x) = f(x_0)\)。

**\(\varepsilon\)-\(\delta\) 表述（连续）**

- 对任意 \(\varepsilon > 0\)，存在 \(\delta > 0\)，使得 \(|x - x_0| < \delta \Rightarrow |f(x) - f(x_0)| < \varepsilon\)。

---

## 关键步骤说明

- **区别**：极限定义中常写 \(0 < |x-x_0|\)（去心邻域）；连续定义中可含 \(x = x_0\)（邻域）。

- **等价**：在 \(x_0\) 有定义的前提下，“极限等于函数值”与“连续”等价。
