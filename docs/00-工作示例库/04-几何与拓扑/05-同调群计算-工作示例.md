## 题目

计算下列空间的单纯同调群：

1. 圆 $S^1$
2. 实心三角形（2-单形）
3. 三角形边界（即 $S^1$）

---

## 完整解答（工作示例）

### 同调群定义

$$H_n(X) = \ker(\partial_n) / \text{im}(\partial_{n+1})$$

其中 $\partial_n: C_n \to C_{n-1}$ 是边缘算子。

---

**解答 1**：$S^1$ 的同调

将 $S^1$ 看作一个 1-单形的两端粘合：顶点 $v_0, v_1$，边 $e$。

链复形：
$$0 \to C_1 \xrightarrow{\partial_1} C_0 \to 0$$

其中 $C_1 = \mathbb{Z}[e]$，$C_0 = \mathbb{Z}[v_0] \oplus \mathbb{Z}[v_1]$。

边缘算子：$\partial_1(e) = v_1 - v_0$。

- $\ker(\partial_1) = 0$（无 1-闭链）
- $\text{im}(\partial_1) = \mathbb{Z}(v_1 - v_0)$

因此：
- $H_0 = C_0 / \text{im}(\partial_1) = \mathbb{Z}[v_0, v_1] / \langle v_1 - v_0 \rangle \cong \mathbb{Z}$
- $H_1 = \ker(\partial_1) / 0 = 0$

**等等**，这不对。重新考虑：$S^1$ 应该有一个 1-维洞。

正确构造：两个顶点 $v_0, v_1$，两条边 $e_1, e_2$。

$\partial_1(e_1) = v_1 - v_0$，$\partial_1(e_2) = v_0 - v_1 = -(v_1 - v_0)$。

$\ker(\partial_1) = \mathbb{Z}(e_1 + e_2)$。

$H_1 = \mathbb{Z}(e_1 + e_2) / 0 = \mathbb{Z}$。

**结论**：
- $H_0(S^1) \cong \mathbb{Z}$（连通分支）
- $H_1(S^1) \cong \mathbb{Z}$（一个 1-维洞）
- $H_n(S^1) = 0$（$n \geq 2$）

---

**解答 2**：实心三角形

顶点 $v_0, v_1, v_2$，边 $e_{01}, e_{12}, e_{02}$，面 $f$。

链复形：
$$0 \to C_2 \xrightarrow{\partial_2} C_1 \xrightarrow{\partial_1} C_0 \to 0$$

$\partial_2(f) = e_{01} + e_{12} - e_{02}$。

$\partial_1(e_{ij}) = v_j - v_i$。

计算得：$H_2 = 0$，$H_1 = 0$（有面填充了洞），$H_0 = \mathbb{Z}$。

**结论**：实心三角形可缩，同调与单点相同。

---

**解答 3**：三角形边界

同解答 1 中的 $S^1$，$H_1 = \mathbb{Z}$。

---

## 关键步骤说明

- **链复形**：按维数组织链群和边缘算子
- **闭链与边缘**：闭链是 $\partial$ 的核，边缘是 $\partial$ 的像
- **同调群**：闭链模边缘，测量"洞"
- **Betti数**：$\beta_n = \text{rank}(H_n)$
