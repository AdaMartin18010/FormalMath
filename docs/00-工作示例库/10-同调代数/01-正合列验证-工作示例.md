## 题目

考虑 $\mathbb{Z}$-模（阿贝尔群）的序列：

1. 验证 $0 \to \mathbb{Z} \xrightarrow{\cdot 2} \mathbb{Z} \xrightarrow{\pi} \mathbb{Z}/2\mathbb{Z} \to 0$ 是短正合列
2. 判断该短正合列是否分裂
3. 验证 $0 \to \mathbb{Z} \xrightarrow{\Delta} \mathbb{Z} \oplus \mathbb{Z} \xrightarrow{+} \mathbb{Z} \to 0$ 的正合性，其中 $\Delta(n) = (n, n)$，$+(m, n) = m + n$

---

## 完整解答（工作示例）

### 正合列定义

序列 $A \xrightarrow{f} B \xrightarrow{g} C$ 在 $B$ 处**正合**，如果 $\text{Im}(f) = \ker(g)$。

**短正合列**：$0 \to A \xrightarrow{f} B \xrightarrow{g} C \to 0$
- $f$ 是单射
- $g$ 是满射
- $\text{Im}(f) = \ker(g)$

---

**解答 1**：验证短正合列

序列：$0 \to \mathbb{Z} \xrightarrow{\cdot 2} \mathbb{Z} \xrightarrow{\pi} \mathbb{Z}/2\mathbb{Z} \to 0$

- $\cdot 2$：乘以 2 的映射，$n \mapsto 2n$
- $\pi$：商映射，$n \mapsto n \pmod{2}$

**验证**：

1. **$\cdot 2$ 是单射**：若 $2n = 0$，则 $n = 0$ ✓

2. **$\pi$ 是满射**：$\mathbb{Z}/2\mathbb{Z} = \{\bar{0}, \bar{1}\}$
   - $\pi(0) = \bar{0}$
   - $\pi(1) = \bar{1}$
   满射 ✓

3. **正合性**：$\text{Im}(\cdot 2) = \ker(\pi)$
   - $\text{Im}(\cdot 2) = 2\mathbb{Z} = \{\text{偶数}\}$
   - $\ker(\pi) = \{n : n \equiv 0 \pmod{2}\} = 2\mathbb{Z}$
   - 相等 ✓

**结论**：这是短正合列。

---

**解答 2**：判断是否分裂

**分裂定义**：短正合列分裂，如果存在 $s: C \to B$ 使得 $g \circ s = \text{id}_C$，或存在 $r: B \to A$ 使得 $r \circ f = \text{id}_A$。

等价地：$B \cong A \oplus C$。

**判定**：假设存在 $s: \mathbb{Z}/2\mathbb{Z} \to \mathbb{Z}$ 使得 $\pi \circ s = \text{id}$。

设 $s(\bar{1}) = n \in \mathbb{Z}$。

则 $\pi(s(\bar{1})) = \pi(n) = \bar{n} = \bar{1}$。

这意味着 $n$ 是奇数。

但 $s$ 必须是同态：
$$s(\bar{1} + \bar{1}) = s(\bar{0}) = 0$$
$$s(\bar{1}) + s(\bar{1}) = n + n = 2n$$

需要 $2n = 0$，即 $n = 0$，与 $n$ 是奇数矛盾。

**结论**：该短正合列**不分裂**。

（这与 $\mathbb{Z} \not\cong \mathbb{Z} \oplus \mathbb{Z}/2\mathbb{Z}$ 一致）

---

**解答 3**：验证另一序列

序列：$0 \to \mathbb{Z} \xrightarrow{\Delta} \mathbb{Z} \oplus \mathbb{Z} \xrightarrow{+} \mathbb{Z} \to 0$

- $\Delta(n) = (n, n)$（对角映射）
- $+(m, n) = m + n$（加法映射）

**验证**：

1. **$\Delta$ 是单射**：若 $(n, n) = (0, 0)$，则 $n = 0$ ✓

2. **$+$ 是满射**：对任意 $k \in \mathbb{Z}$，$+(k, 0) = k$ ✓

3. **正合性**：$\text{Im}(\Delta) = \ker(+)$
   - $\text{Im}(\Delta) = \{(n, n) : n \in \mathbb{Z}\}$（对角子群）
   - $\ker(+) = \{(m, n) : m + n = 0\} = \{(m, -m) : m \in \mathbb{Z}\}$
   - 令 $n = m$，则 $\{(n, -n)\}$... 
   
   **等等**，重新计算：
   - $\text{Im}(\Delta) = \{(n, n) : n \in \mathbb{Z}\}$
   - $\ker(+) = \{(m, n) : m + n = 0\} = \{(m, -m)\}$
   
   这两个集合不相等！

**修正**：若要正合，应将 $+$ 改为 $(m, n) \mapsto m - n$，或修改 $\Delta$。

**修正后的序列**：$0 \to \mathbb{Z} \xrightarrow{\Delta} \mathbb{Z} \oplus \mathbb{Z} \xrightarrow{-} \mathbb{Z} \to 0$
其中 $-(m, n) = m - n$。

验证：$\ker(-) = \{(m, n) : m - n = 0\} = \{(n, n)\} = \text{Im}(\Delta)$ ✓

**分裂性**：定义 $s: \mathbb{Z} \to \mathbb{Z} \oplus \mathbb{Z}$，$s(n) = (n, 0)$。

则 $-(s(n)) = -(n, 0) = n$，故 $-\circ s = \text{id}$。

**结论**：修正后的短正合列**分裂**，且 $\mathbb{Z} \oplus \mathbb{Z} \cong \mathbb{Z} \oplus \mathbb{Z}$。

---

## 关键步骤说明

- **正合性验证**：检查像等于核
- **分裂判定**：寻找截面（section）或收缩（retraction）
- **分裂等价条件**：短正合列分裂 $\Leftrightarrow$ 中间项是直和
- **反例**：$0 \to \mathbb{Z} \xrightarrow{\cdot n} \mathbb{Z} \to \mathbb{Z}/n\mathbb{Z} \to 0$（$n > 1$）永不分裂
