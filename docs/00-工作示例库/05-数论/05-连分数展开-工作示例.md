## 题目

1. 将 $\frac{17}{12}$ 展开为有限连分数
2. 将 $\sqrt{2}$ 展开为无限连分数
3. 计算 $\pi$ 的前几个收敛子

---

## 完整解答（工作示例）

### 连分数定义

$$a_0 + \cfrac{1}{a_1 + \cfrac{1}{a_2 + \cfrac{1}{\ddots}}}$$

记为 $[a_0; a_1, a_2, \ldots]$。

---

**解答 1**：$\frac{17}{12}$

$$\frac{17}{12} = 1 + \frac{5}{12} = 1 + \frac{1}{\frac{12}{5}}$$

$$\frac{12}{5} = 2 + \frac{2}{5} = 2 + \frac{1}{\frac{5}{2}}$$

$$\frac{5}{2} = 2 + \frac{1}{2}$$

因此：
$$\frac{17}{12} = [1; 2, 2, 2]$$

---

**解答 2**：$\sqrt{2}$

$$\sqrt{2} = 1 + (\sqrt{2} - 1) = 1 + \frac{1}{\frac{1}{\sqrt{2}-1}} = 1 + \frac{1}{\sqrt{2}+1}$$

$$\sqrt{2} + 1 = 2 + (\sqrt{2} - 1) = 2 + \frac{1}{\sqrt{2}+1}$$

周期出现：
$$\sqrt{2} = [1; \overline{2}] = [1; 2, 2, 2, \ldots]$$

---

**解答 3**：$\pi$ 的收敛子

$$\pi = [3; 7, 15, 1, 292, \ldots]$$

**收敛子**：
- $c_0 = 3 = \frac{3}{1}$
- $c_1 = 3 + \frac{1}{7} = \frac{22}{7} \approx 3.142857$
- $c_2 = \frac{333}{106} \approx 3.141509$
- $c_3 = \frac{355}{113} \approx 3.141593$（密率）

## 关键步骤说明

- **欧几里得算法**：用于计算连分数展开
- **收敛子**：提供最佳有理逼近
- **周期连分数**：二次无理数的特征
- **应用**：历法计算、密码学
