## 题目

在Lean4中：

1. 定义自然数列表类型
2. 定义列表长度函数并证明其性质
3. 定义列表连接函数并证明结合律

---

## 完整解答（工作示例）

### 列表类型定义

```lean
inductive List (α : Type)
| nil : List α
| cons : α → List α → List α

namespace List

-- 中缀表示法
infixr:67 " :: " => cons
syntax "[" term,* "]" => nil
```

---

**解答 1**：长度函数

```lean
def length : List α → Nat
| nil => 0
| cons _ t => 1 + length t

-- 长度性质：length (a :: l) = 1 + length l
theorem length_cons (a : α) (l : List α) : length (a :: l) = 1 + length l := by
  rfl

-- 长度非负
theorem length_nonneg (l : List α) : length l ≥ 0 := by
  induction l with
  | nil => simp [length]
  | cons h t ih => 
    simp [length]
    linarith
```

---

**解答 2**：列表连接

```lean
def append : List α → List α → List α
| nil, bs => bs
| cons a as, bs => cons a (append as bs)

-- 中缀表示法
infixr:65 " ++ " => append

-- 结合律：$(l_1 ++ l_2) ++ l_3 = l_1 ++ (l_2 ++ l_3)$
theorem append_assoc (l₁ l₂ l₃ : List α) : (l₁ ++ l₂) ++ l₃ = l₁ ++ (l₂ ++ l₃) := by
  induction l₁ with
  | nil => 
    simp [append]
  | cons h t ih =>
    simp [append]
    rw [ih]
```

---

**解答 3**：长度与连接的关系

```lean
theorem length_append (l₁ l₂ : List α) : length (l₁ ++ l₂) = length l₁ + length l₂ := by
  induction l₁ with
  | nil => 
    simp [append, length]
  | cons h t ih =>
    simp [append, length]
    rw [ih]
    simp [Nat.add_assoc]
```

## 关键步骤说明

- **归纳类型**：`nil` 和 `cons` 构造子定义列表
- **递归函数**：按结构递归定义操作
- **结构归纳**：证明时按构造子归纳
- **simp**：自动简化定义展开
- **rw**：重写归纳假设
