## 题目

在Lean4中：

1. 定义群结构
2. 证明单位元的唯一性
3. 证明逆元的唯一性

---

## 完整解答（工作示例）

### 群结构定义

```lean
class Group (G : Type) where
  mul : G → G → G
  one : G
  inv : G → G
  mul_assoc : ∀ a b c : G, mul (mul a b) c = mul a (mul b c)
  one_mul : ∀ a : G, mul one a = a
  mul_one : ∀ a : G, mul a one = a
  mul_left_inv : ∀ a : G, mul (inv a) a = one
  mul_right_inv : ∀ a : G, mul a (inv a) = one

infix:70 " * " => Group.mul
postfix:max "⁻¹" => Group.inv
notation "𝟭" => Group.one
```

---

**证明 1**：单位元唯一

```lean
theorem one_unique {G : Type} [Group G] (e : G) (h : ∀ a, e * a = a) : e = 𝟭 := by
  calc
    e = e * 𝟭 := by rw [Group.mul_one]
    _ = 𝟭 := by rw [h 𝟭]
```

---

**证明 2**：逆元唯一

```lean
theorem inv_unique {G : Type} [Group G] (a b : G) (h : b * a = 𝟭) : b = a⁻¹ := by
  calc
    b = b * 𝟭 := by rw [Group.mul_one]
    _ = b * (a * a⁻¹) := by rw [Group.mul_right_inv]
    _ = (b * a) * a⁻¹ := by rw [Group.mul_assoc]
    _ = 𝟭 * a⁻¹ := by rw [h]
    _ = a⁻¹ := by rw [Group.one_mul]
```

---

**证明 3**：逆元的逆元

```lean
theorem inv_inv {G : Type} [Group G] (a : G) : (a⁻¹)⁻¹ = a := by
  apply inv_unique
  apply Group.mul_left_inv
```

## 关键步骤说明

- **类型类**：`Group` 作为类型类，便于实例化
- **符号重载**：定义中缀运算符增强可读性
- **证明策略**：`calc` 用于等式链推导
- **唯一性证明**：利用群的公理进行代数操作
