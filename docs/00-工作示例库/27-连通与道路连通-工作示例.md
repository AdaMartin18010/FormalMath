## 预备知识

### 连通空间

拓扑空间 \(X\) 是**连通**的，如果它不能表示为两个非空不交开集的并。

等价表述：不存在连续满射 \(f: X \to \{0, 1\}\)（离散拓扑）。

### 道路连通空间

拓扑空间 \(X\) 是**道路连通**的，如果对任意 \(x, y \in X\)，存在连续映射 \(\gamma: [0,1] \to X\) 使得 \(\gamma(0) = x, \gamma(1) = y\)。

---

## 题目

说明道路连通空间必连通；并给出连通但非道路连通的例子。

---

## 完整解答

### 第一部分：道路连通 ⇒ 连通

**定理**: 若 \(X\) 道路连通，则 \(X\) 连通。

**证明**:

设 \(X\) 道路连通，假设 \(X\) 不连通。

则存在非空不交开集 \(U, V\) 使得 \(X = U \cup V\)。

取 \(x \in U, y \in V\)。由道路连通性，存在道路 \(\gamma: [0,1] \to X\) 连接 \(x, y\)。

则 \(\gamma^{-1}(U)\) 和 \(\gamma^{-1}(V)\) 是 \([0,1]\) 的非空不交开集，且：
$$[0,1] = \gamma^{-1}(U) \cup \gamma^{-1}(V)$$

这与 \([0,1]\) 连通矛盾！

因此 \(X\) 连通。

---

### 第二部分：连通但非道路连通的例子

**拓扑学家正弦曲线**:

$$S = \{(x, \sin(1/x)) : x > 0\} \cup \{(0, y) : -1 \leq y \leq 1\}$$

**图像描述**:

```
     y
     │
  1 ─┤  ╭─╮   ╭─╮   ╭─╮   ╭─╮
     │ ╱   ╲ ╱   ╲ ╱   ╲ ╱   ╲
  0 ─┼─────────────────────────→ x
     │ ╲   ╱ ╲   ╱ ╲   ╱ ╲   ╱
-1 ─┤  ╰─╯   ╰─╯   ╰─╯   ╰─╯
     │
     └─────┬───────────────────
           │
        x→0⁺时振荡越来越快
        加上y轴上的线段
```

**证明**:

**1. S是连通的**:

- \(A = \{(x, \sin(1/x)) : x > 0\}\) 是连续映射 \(x \mapsto (x, \sin(1/x))\) 的像，连通
- \(S = \overline{A}\)（A的闭包），连通集的闭包连通

**2. S不是道路连通的**:

假设存在道路 \(\gamma: [0,1] \to S\) 从 \((0,0)\) 到 \((1, \sin 1)\)。

设 \(\gamma(t) = (x(t), y(t))\)。

由于 \(\gamma\) 连续且 \(x(0) = 0, x(1) = 1\)，由介值性，\(x(t)\) 取遍 \([0,1]\) 中所有值。

但当 \(t \to 0^+\) 时，若 \(x(t) > 0\)，则 \(y(t) = \sin(1/x(t))\) 在 \([-1, 1]\) 上无限振荡，不可能连续地趋于 \(y(0)\)。

矛盾！因此不存在这样的道路。

## 关系总结

```
道路连通 ────────→ 连通
    │                │
    │                │
    ↓                ↓
  路径连通分量    连通分量
  （可能更细）    （可能更粗）
```

**重要**: 连通性 ≠ 道路连通性，但局部道路连通 + 连通 ⇒ 道路连通。
