## 题目

用数值方法求解初值问题：
$$y' = y, \quad y(0) = 1$$

在 $x = 0.2$ 处的近似值，步长 $h = 0.1$。

---

## 完整解答（工作示例）

### 精确解

$y = e^x$，$y(0.2) = e^{0.2} \approx 1.2214$

---

**Euler法**

$$y_{n+1} = y_n + h f(x_n, y_n)$$

**步骤 1**：$x_0 = 0, y_0 = 1$
$$y_1 = y_0 + h \cdot y_0 = 1 + 0.1 \cdot 1 = 1.1$$

**步骤 2**：$x_1 = 0.1, y_1 = 1.1$
$$y_2 = y_1 + h \cdot y_1 = 1.1 + 0.1 \cdot 1.1 = 1.21$$

**结果**：$y(0.2) \approx 1.21$

**误差**：$|1.2214 - 1.21| = 0.0114$

---

**改进Euler法（Heun法）**

$$k_1 = f(x_n, y_n)$$
$$k_2 = f(x_n + h, y_n + hk_1)$$
$$y_{n+1} = y_n + \frac{h}{2}(k_1 + k_2)$$

**步骤 1**：
- $k_1 = y_0 = 1$
- $k_2 = y_0 + hk_1 = 1 + 0.1 = 1.1$
- $y_1 = 1 + 0.05(1 + 1.1) = 1.105$

**步骤 2**：
- $k_1 = y_1 = 1.105$
- $k_2 = y_1 + hk_1 = 1.105 + 0.1105 = 1.2155$
- $y_2 = 1.105 + 0.05(1.105 + 1.2155) = 1.2210$

**结果**：$y(0.2) \approx 1.2210$

**误差**：$|1.2214 - 1.2210| = 0.0004$（明显更小）

---

## 对比

| 方法 | 近似值 | 误差 | 收敛阶 |
|------|--------|------|--------|
| Euler | 1.2100 | 0.0114 | $O(h)$ |
| 改进Euler | 1.2210 | 0.0004 | $O(h^2)$ |

---

## 关键步骤说明

- **Euler法**：简单显式方法，一阶精度
- **改进Euler法**： predictor-corrector，二阶精度
- **步长选择**：精度与计算量的权衡
- **稳定性**：显式方法有步长限制
