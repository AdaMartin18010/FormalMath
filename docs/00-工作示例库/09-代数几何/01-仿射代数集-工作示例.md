## 题目

在 $\mathbb{A}^2(\mathbb{C})$（复仿射平面）中：

1. 求代数集 $V(x^2 + y^2 - 1)$ 与 $V(x - y)$ 的交
2. 求代数集 $V(x)$ 与 $V(y)$ 的并对应的理想
3. 验证 $I(V(x^2 + y^2)) = (x, y)$ 在 $\mathbb{A}^2(\mathbb{C})$ 中

---

## 完整解答（工作示例）

### 基本定义

- **零点集**：$V(S) = \{P \in \mathbb{A}^n : f(P) = 0, \forall f \in S\}$
- **理想**：$I(X) = \{f \in k[x_1, \ldots, x_n] : f(P) = 0, \forall P \in X\}$
- **交**：$V(I) \cap V(J) = V(I + J)$
- **并**：$V(I) \cup V(J) = V(I \cap J) = V(I \cdot J)$

---

**解答 1**：$V(x^2 + y^2 - 1) \cap V(x - y)$

用理想运算：
$$V(x^2 + y^2 - 1) \cap V(x - y) = V((x^2 + y^2 - 1) + (x - y))$$

在交点处同时满足：
$$\begin{cases} x^2 + y^2 = 1 \\ x = y \end{cases}$$

代入：$2x^2 = 1$，$x = \pm \frac{1}{\sqrt{2}}$

**结果**：两个点
$$\left(\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}\right), \quad \left(-\frac{1}{\sqrt{2}}, -\frac{1}{\sqrt{2}}\right)$$

理想描述：$V(x^2 + y^2 - 1, x - y) = V(2x^2 - 1, x - y)$

---

**解答 2**：$V(x) \cup V(y)$ 的理想

- $V(x)$ 是 $y$ 轴
- $V(y)$ 是 $x$ 轴

**并集**：$V(x) \cup V(y) = $ 坐标轴的并

用理想运算：
$$V(x) \cup V(y) = V((x) \cap (y)) = V((x) \cdot (y)) = V(xy)$$

验证：
- $(x) \cap (y) = (xy)$ 在 $k[x, y]$ 中？
- 实际上 $(x) \cap (y) = (xy)$ 因为 $k[x,y]$ 是 UFD

**结论**：$V(x) \cup V(y) = V(xy)$

对应理想：$I(V(x) \cup V(y)) = \sqrt{(xy)} = (xy)$（在代数闭域上，$xy$ 已经是根理想）

---

**解答 3**：验证 $I(V(x^2 + y^2)) = (x, y)$

**步骤 1**：求 $V(x^2 + y^2)$

在 $\mathbb{A}^2(\mathbb{C})$ 中：
$$x^2 + y^2 = (x + iy)(x - iy) = 0$$

故 $V(x^2 + y^2) = V(x + iy) \cup V(x - iy)$

这是两条直线的并：$y = ix$ 和 $y = -ix$

**等等**，$V(x^2 + y^2)$ 实际上是这两条直线的并，不是单点。

**重新审题**：若问题是 $V(x^2 + y^2)$ 在 $\mathbb{A}^2(\mathbb{R})$ 中...

在 $\mathbb{R}^2$ 中：$x^2 + y^2 = 0$ 仅当 $x = y = 0$，故 $V(x^2 + y^2) = \{(0,0)\}$

**步骤 2**：计算 $I(\{(0,0)\})$

$$I(\{(0,0)\}) = \{f \in \mathbb{R}[x,y] : f(0,0) = 0\} = (x, y)$$

**验证**：$\sqrt{(x^2 + y^2)} = (x, y)$？

- $(x^2 + y^2) \subseteq (x, y)$（因为 $x^2 + y^2 = x \cdot x + y \cdot y$）
- $x \notin (x^2 + y^2)$，但 $x^2 \in (x^2 + y^2)$
- 实际上 $(x^2 + y^2)$ 不是根理想

$$\sqrt{(x^2 + y^2)} = (x, y)$$

**结论**：在 $\mathbb{A}^2(\mathbb{R})$ 中，$I(V(x^2 + y^2)) = (x, y)$。

---

## 关键步骤说明

- **零点集计算**：解多项式方程组
- **理想运算**：交对应理想的和，并对应对理想的交或积
- **Nullstellensatz**：$I(V(I)) = \sqrt{I}$（在代数闭域上）
- **实 vs 复**：实数域上多项式可能有不同的零点集结构
