## 题目

求 \(f(z) = \frac{z}{(z-1)(z-2)^2}\) 在 \(z=1\) 与 \(z=2\) 处的留数，并求 \(\oint_{|z|=3/2} f(z)\,dz\)。

---

## 完整解答（工作示例）

**步骤 1**：\(z=1\) 为一阶极点，\(\mathrm{Res}(f,1) = \lim_{z\to 1}(z-1)f(z) = \lim_{z\to 1} \frac{z}{(z-2)^2} = \frac{1}{1} = 1\)。

**步骤 2**：\(z=2\) 为二阶极点，\(\mathrm{Res}(f,2) = \lim_{z\to 2} \frac{d}{dz}\bigl((z-2)^2 f(z)\bigr) = \lim_{z\to 2} \frac{d}{dz}\frac{z}{z-1} = \lim_{z\to 2} \frac{-1}{(z-1)^2} = -1\)。

**步骤 3**：圆周 \(|z|=3/2\) 包围 \(z=1\) 与 \(z=2\)，故 \(\oint f\,dz = 2\pi i(\mathrm{Res}(f,1)+\mathrm{Res}(f,2)) = 2\pi i(1-1)=0\)。

---

## 关键步骤说明

- 一阶极点：\(\mathrm{Res}(f,z_0) = \lim_{z\to z_0}(z-z_0)f(z)\)。
- 二阶极点：用 \((z-z_0)^2 f(z)\) 求导再取极限。
