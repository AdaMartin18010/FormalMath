## 预备知识

### 幂级数定义
形如 \(\sum_{n=0}^{\infty} a_n (x-c)^n\) 的级数称为幂级数。

### Cauchy-Hadamard公式

收敛半径：
$$R = \frac{1}{\limsup_{n\to\infty} |a_n|^{1/n}}$$

特殊情况：
- 若极限为0，则 \(R = +\infty\)（处处收敛）
- 若极限为\(+\infty\)，则 \(R = 0\)（仅在中心收敛）

### 比值法（实用方法）

若极限存在：
$$R = \lim_{n\to\infty} \left|\frac{a_n}{a_{n+1}}\right|$$

---

## 题目

求 \(\sum_{n=0}^{\infty} n! x^n\) 的收敛半径。

---

## 完整解答

### 方法1：使用Cauchy-Hadamard公式

**步骤1**：识别系数
$$a_n = n!$$

**步骤2**：计算 \(|a_n|^{1/n}\)
$$|a_n|^{1/n} = (n!)^{1/n}$$

**步骤3**：利用Stirling公式或估计
由 Stirling 公式：
$$n! \sim \sqrt{2\pi n} \left(\frac{n}{e}\right)^n$$

取对数：
$$\ln(n!)^{1/n} = \frac{\ln(n!)}{n} \sim \frac{n\ln n - n}{n} = \ln n - 1 \to +\infty$$

或直接估计：
$$n! \geq (n/2)^{n/2} \Rightarrow (n!)^{1/n} \geq (n/2)^{1/2} \to +\infty$$

**步骤4**：求收敛半径
$$\limsup_{n\to\infty} |a_n|^{1/n} = +\infty$$

$$R = \frac{1}{+\infty} = \boxed{0}$$

### 方法2：使用比值法验证

**步骤1**：计算比值
$$\left|\frac{a_{n+1}}{a_n}\right| = \frac{(n+1)!}{n!} = n+1 \to +\infty$$

**步骤2**：求收敛半径
$$R = \lim_{n\to\infty} \frac{1}{n+1} = 0$$

---

## 结论

| 属性 | 结果 |
|-----|------|
| **收敛半径** | R = 0 |
| **收敛域** | 仅 x = 0 |
| **原因** | 系数增长过快（阶乘级） |

**解释**: \(n!\) 增长比任何指数函数都快，因此幂级数仅在 \(x=0\) 处收敛。
