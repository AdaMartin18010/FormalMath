## 题目

用 $\varepsilon$-$\delta$ 定义证明下列函数在给定点的连续性：

1. $f(x) = 3x + 2$ 在 $x = 5$ 处
2. $f(x) = x^2$ 在任意点 $x = a$ 处
3. $f(x) = \sqrt{x}$ 在 $x = 4$ 处

---

## 完整解答（工作示例）

### $\varepsilon$-$\delta$ 定义

$f$ 在 $a$ 处连续：$\forall \varepsilon > 0, \exists \delta > 0$，当 $|x - a| < \delta$ 时，$|f(x) - f(a)| < \varepsilon$。

---

**解答 1**：$f(x) = 3x + 2$ 在 $x = 5$ 处

给定 $\varepsilon > 0$，需要找 $\delta > 0$ 使得：

$$|x - 5| < \delta \Rightarrow |(3x + 2) - 17| < \varepsilon$$

计算：
$$|(3x + 2) - 17| = |3x - 15| = 3|x - 5|$$

要使 $3|x - 5| < \varepsilon$，只需 $|x - 5| < \frac{\varepsilon}{3}$。

**取** $\delta = \frac{\varepsilon}{3}$，则当 $|x - 5| < \delta$ 时：

$$|f(x) - f(5)| = 3|x - 5| < 3 \cdot \frac{\varepsilon}{3} = \varepsilon$$

故 $f$ 在 $x = 5$ 处连续。

---

**解答 2**：$f(x) = x^2$ 在 $x = a$ 处

给定 $\varepsilon > 0$，需要找 $\delta > 0$ 使得：

$$|x - a| < \delta \Rightarrow |x^2 - a^2| < \varepsilon$$

计算：
$$|x^2 - a^2| = |x + a||x - a|$$

**控制** $|x + a|$：先限制 $|x - a| < 1$，则 $|x| < |a| + 1$，故 $|x + a| \leq |x| + |a| < 2|a| + 1$。

要使 $|x^2 - a^2| < \varepsilon$，需要：
$$|x - a| < \frac{\varepsilon}{2|a| + 1}$$

**取** $\delta = \min\left(1, \frac{\varepsilon}{2|a| + 1}\right)$，则当 $|x - a| < \delta$ 时：

$$|x^2 - a^2| = |x + a||x - a| < (2|a| + 1) \cdot \frac{\varepsilon}{2|a| + 1} = \varepsilon$$

故 $f(x) = x^2$ 在任意点 $a$ 处连续。

---

**解答 3**：$f(x) = \sqrt{x}$ 在 $x = 4$ 处

给定 $\varepsilon > 0$（不妨设 $\varepsilon < 1$），需要找 $\delta > 0$ 使得：

$$|x - 4| < \delta \Rightarrow |\sqrt{x} - 2| < \varepsilon$$

计算：
$$|\sqrt{x} - 2| = \frac{|x - 4|}{\sqrt{x} + 2}$$

先限制 $|x - 4| < 1$，则 $3 < x < 5$，故 $\sqrt{x} > \sqrt{3} > 1$，$\sqrt{x} + 2 > 3$。

因此：
$$|\sqrt{x} - 2| = \frac{|x - 4|}{\sqrt{x} + 2} < \frac{|x - 4|}{3}$$

要使 $|\sqrt{x} - 2| < \varepsilon$，只需 $|x - 4| < 3\varepsilon$。

**取** $\delta = \min(1, 3\varepsilon)$，则当 $|x - 4| < \delta$ 时：

$$|\sqrt{x} - 2| < \frac{3\varepsilon}{3} = \varepsilon$$

故 $f(x) = \sqrt{x}$ 在 $x = 4$ 处连续。

---

## 关键步骤说明

- **线性函数**：$\delta$ 与 $\varepsilon$ 成简单比例关系
- **非线性函数**：需要控制额外因子（如 $|x + a|$）
- **技巧**：先限制 $|x - a|$ 在某个范围内，再确定 $\delta$
- **最小值技巧**：$\delta = \min(\delta_1, \delta_2)$ 满足多个条件
