## 题目

判定下列群的可解性：

1. $S_3$（3 元对称群）
2. $A_5$（5 元交错群）
3. 四元数群 $Q_8$

---

## 完整解答（工作示例）

### 可解群定义

群 $G$ **可解**，如果存在子群列：
$$G = G_0 \triangleright G_1 \triangleright \cdots \triangleright G_n = \{e\}$$

使得每个商群 $G_i / G_{i+1}$ 都是阿贝尔群。

等价：导列 $G^{(0)} = G, G^{(k+1)} = [G^{(k)}, G^{(k)}]$ 最终到达 $\{e\}$。

---

**解答 1**：$S_3$ 的可解性

**导列计算**：

$G^{(0)} = S_3$

换位子群 $[S_3, S_3] = A_3 \cong \mathbb{Z}_3$（3 阶循环群）

$G^{(1)} = A_3$

$[A_3, A_3] = \{e\}$（阿贝尔群的换位子群是平凡的）

$G^{(2)} = \{e\}$

导列：$S_3 \triangleright A_3 \triangleright \{e\}$

商群：$S_3/A_3 \cong \mathbb{Z}_2$，$A_3/\{e\} \cong A_3$ 都是阿贝尔群。

**结论**：$S_3$ 是**可解群**。

---

**解答 2**：$A_5$ 的可解性

**关键事实**：$A_5$ 是单群（非平凡正规子群只有自身）。

**导列计算**：

$G^{(0)} = A_5$

换位子群 $[A_5, A_5]$：由于 $A_5$ 是非阿贝尔单群，$[A_5, A_5] \trianglelefteq A_5$ 是非平凡正规子群。

由于 $A_5$ 是单群，$[A_5, A_5] = A_5$。

因此导列为：$A_5 \triangleright A_5 \triangleright A_5 \triangleright \cdots$

永不到达 $\{e\}$。

**结论**：$A_5$ **不是可解群**。

（这也是五次方程没有根式解的群论原因）

---

**解答 3**：四元数群 $Q_8$

$$Q_8 = \{\pm 1, \pm i, \pm j, \pm k\}$$

其中 $i^2 = j^2 = k^2 = ijk = -1$。

**导列计算**：

$G^{(0)} = Q_8$

换位子群 $[Q_8, Q_8] = \{1, -1\} = Z(Q_8)$（中心）

$G^{(1)} = \{1, -1\} \cong \mathbb{Z}_2$

$[\{1, -1\}, \{1, -1\}] = \{1\}$

$G^{(2)} = \{1\}$

导列：$Q_8 \triangleright \{1, -1\} \triangleright \{1\}$

**结论**：$Q_8$ 是**可解群**。

---

## 关键步骤说明

- **换位子**：$[g, h] = g^{-1}h^{-1}gh$
- **导群**：由所有换位子生成的子群
- **单群**：非阿贝尔单群不可解
- **可解列与根式解**：多项式方程可根式解当且仅当Galois群可解
