## 题目

考虑环同态 $\varphi: \mathbb{Z}[x] \to \mathbb{Z}$，定义为 $\varphi(f(x)) = f(2)$（在 $x = 2$ 处求值）。

1. 证明 $\varphi$ 是环同态
2. 求 $\ker(\varphi)$
3. 求 $\text{Im}(\varphi)$
4. 用第一同构定理描述 $\mathbb{Z}[x] / \ker(\varphi)$

---

## 完整解答（工作示例）

### 基本定义

- **环同态**：保持加法和乘法的映射
- **核**：$\ker(\varphi) = \{f \in \mathbb{Z}[x] : \varphi(f) = 0\}$
- **像**：$\text{Im}(\varphi) = \{\varphi(f) : f \in \mathbb{Z}[x]\}$

---

**解答 1**：证明 $\varphi$ 是环同态

**保持加法**：
$$\varphi(f + g) = (f + g)(2) = f(2) + g(2) = \varphi(f) + \varphi(g)$$

**保持乘法**：
$$\varphi(f \cdot g) = (f \cdot g)(2) = f(2) \cdot g(2) = \varphi(f) \cdot \varphi(g)$$

**保持单位元**：
$$\varphi(1) = 1(2) = 1$$

故 $\varphi$ 是环同态。

---

**解答 2**：求 $\ker(\varphi)$

$$\ker(\varphi) = \{f \in \mathbb{Z}[x] : f(2) = 0\}$$

由因式定理，$f(2) = 0$ 当且仅当 $(x - 2) \mid f(x)$。

因此：
$$\ker(\varphi) = \{(x - 2) \cdot g(x) : g(x) \in \mathbb{Z}[x]\} = (x - 2)$$

即由 $x - 2$ 生成的主理想。

---

**解答 3**：求 $\text{Im}(varphi)$

对任意 $n \in \mathbb{Z}$，常数多项式 $f(x) = n$ 满足 $\varphi(f) = n$。

因此：
$$\text{Im}(\varphi) = \mathbb{Z}$$

---

**解答 4**：第一同构定理

**第一同构定理**：$R / \ker(\varphi) \cong \text{Im}(\varphi)$

因此：
$$\mathbb{Z}[x] / (x - 2) \cong \mathbb{Z}$$

同构映射为：$f(x) + (x - 2) \mapsto f(2)$

**意义**：在商环 $\mathbb{Z}[x] / (x - 2)$ 中，$x$ 被等同于 2。

---

## 关键步骤说明

- **赋值同态**：$f(x) \mapsto f(a)$ 总是环同态
- **核的计算**：利用因式定理，核由 $x - a$ 生成
- **第一同构定理**：商环的结构由原环模去核得到
- **几何意义**：$\mathbb{Z}[x] / (x - 2)$ 是 $\mathbb{Z}[x]$ 在"点 $x = 2$"处的纤维
