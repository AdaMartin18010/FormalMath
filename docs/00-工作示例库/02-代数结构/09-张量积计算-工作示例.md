## 题目

设 $V = \mathbb{R}^2$，$W = \mathbb{R}^3$，标准基分别为 $\{e_1, e_2\}$ 和 $\{f_1, f_2, f_3\}$。

1. 确定 $V \otimes W$ 的维数
2. 将 $(2e_1 + 3e_2) \otimes (f_1 - f_2 + 2f_3)$ 展开为基元素的线性组合
3. 验证张量积的双线性性

---

## 完整解答（工作示例）

### 张量积定义

$V \otimes W$ 是由符号 $v \otimes w$ 生成的向量空间，满足：
- $(v_1 + v_2) \otimes w = v_1 \otimes w + v_2 \otimes w$
- $v \otimes (w_1 + w_2) = v \otimes w_1 + v \otimes w_2$
- $(\lambda v) \otimes w = v \otimes (\lambda w) = \lambda(v \otimes w)$

---

**解答 1**：$V \otimes W$ 的维数

若 $\{e_i\}$ 是 $V$ 的基，$\{f_j\}$ 是 $W$ 的基，则 $\{e_i \otimes f_j\}$ 是 $V \otimes W$ 的基。

$$\dim(V \otimes W) = \dim(V) \cdot \dim(W) = 2 \cdot 3 = 6$$

基：$\{e_1 \otimes f_1, e_1 \otimes f_2, e_1 \otimes f_3, e_2 \otimes f_1, e_2 \otimes f_2, e_2 \otimes f_3\}$

---

**解答 2**：展开张量积

$$(2e_1 + 3e_2) \otimes (f_1 - f_2 + 2f_3)$$

$$= 2e_1 \otimes (f_1 - f_2 + 2f_3) + 3e_2 \otimes (f_1 - f_2 + 2f_3)$$

$$= 2e_1 \otimes f_1 - 2e_1 \otimes f_2 + 4e_1 \otimes f_3 + 3e_2 \otimes f_1 - 3e_2 \otimes f_2 + 6e_2 \otimes f_3$$

---

**解答 3**：验证双线性性

设 $B(v, w) = v \otimes w$。

**对第一个变量线性**：
$$B(v_1 + v_2, w) = (v_1 + v_2) \otimes w = v_1 \otimes w + v_2 \otimes w = B(v_1, w) + B(v_2, w)$$

$$B(\lambda v, w) = (\lambda v) \otimes w = \lambda(v \otimes w) = \lambda B(v, w)$$

**对第二个变量线性**：类似验证。

---

## 关键步骤说明

- **基的构造**：$V \otimes W$ 的基是张量积的基
- **维数公式**：$\dim(V \otimes W) = \dim V \cdot \dim W$
- **泛性质**：$\text{Bil}(V \times W, U) \cong \text{Hom}(V \otimes W, U)$
- **应用**：多线性代数、微分几何、量子力学
