## 题目

计算下列域扩张的次数 $[K : F]$：

1. $K = \mathbb{Q}(\sqrt{2})$，$F = \mathbb{Q}$
2. $K = \mathbb{Q}(\sqrt[3]{2})$，$F = \mathbb{Q}$
3. $K = \mathbb{Q}(\sqrt{2}, \sqrt{3})$，$F = \mathbb{Q}$
4. $K = \mathbb{Q}(\sqrt{2} + \sqrt{3})$，$F = \mathbb{Q}$

---

## 完整解答（工作示例）

### 基本定理

- **扩张次数**：$[K : F] = \dim_F K$
- **塔定律**：$[K : F] = [K : E] \cdot [E : F]$（若 $F \subseteq E \subseteq K$）
- **代数元**：若 $\alpha$ 的极小多项式次数为 $n$，则 $[F(\alpha) : F] = n$

---

**解答 1**：$[\mathbb{Q}(\sqrt{2}) : \mathbb{Q}]$

$\sqrt{2}$ 的极小多项式为 $x^2 - 2$（Eisenstein 判别法，$p = 2$）。

次数为 2，故：
$$[\mathbb{Q}(\sqrt{2}) : \mathbb{Q}] = 2$$

基：$\{1, \sqrt{2}\}$

---

**解答 2**：$[\mathbb{Q}(\sqrt[3]{2}) : \mathbb{Q}]$

$\sqrt[3]{2}$ 的极小多项式为 $x^3 - 2$（Eisenstein 判别法，$p = 2$）。

次数为 3，故：
$$[\mathbb{Q}(\sqrt[3]{2}) : \mathbb{Q}] = 3$$

基：$\{1, \sqrt[3]{2}, \sqrt[3]{4}\}$

---

**解答 3**：$[\mathbb{Q}(\sqrt{2}, \sqrt{3}) : \mathbb{Q}]$

**步骤 1**：$[\mathbb{Q}(\sqrt{2}) : \mathbb{Q}] = 2$

**步骤 2**：$[\mathbb{Q}(\sqrt{2}, \sqrt{3}) : \mathbb{Q}(\sqrt{2})]$

$\sqrt{3}$ 在 $\mathbb{Q}(\sqrt{2})$ 上的极小多项式：

假设 $\sqrt{3} \in \mathbb{Q}(\sqrt{2})$，则 $\sqrt{3} = a + b\sqrt{2}$，$a, b \in \mathbb{Q}$。

平方：$3 = a^2 + 2b^2 + 2ab\sqrt{2}$

若 $b \neq 0$，则 $\sqrt{2} = \frac{3 - a^2 - 2b^2}{2ab} \in \mathbb{Q}$，矛盾。

故 $b = 0$，则 $\sqrt{3} = a \in \mathbb{Q}$，矛盾。

因此 $\sqrt{3} \notin \mathbb{Q}(\sqrt{2})$，极小多项式仍为 $x^2 - 3$。

$$[\mathbb{Q}(\sqrt{2}, \sqrt{3}) : \mathbb{Q}(\sqrt{2})] = 2$$

**步骤 3**：由塔定律

$$[\mathbb{Q}(\sqrt{2}, \sqrt{3}) : \mathbb{Q}] = 2 \cdot 2 = 4$$

基：$\{1, \sqrt{2}, \sqrt{3}, \sqrt{6}\}$

---

**解答 4**：$[\mathbb{Q}(\sqrt{2} + \sqrt{3}) : \mathbb{Q}]$

**关键观察**：$\mathbb{Q}(\sqrt{2} + \sqrt{3}) = \mathbb{Q}(\sqrt{2}, \sqrt{3})$

**证明**：
- 显然 $\mathbb{Q}(\sqrt{2} + \sqrt{3}) \subseteq \mathbb{Q}(\sqrt{2}, \sqrt{3})$

- 反向：设 $\alpha = \sqrt{2} + \sqrt{3}$
  $$\alpha^2 = 5 + 2\sqrt{6} \Rightarrow \sqrt{6} = \frac{\alpha^2 - 5}{2} \in \mathbb{Q}(\alpha)$$
  
  $$\alpha \cdot \sqrt{6} = (\sqrt{2} + \sqrt{3})\sqrt{6} = 2\sqrt{3} + 3\sqrt{2}$$
  
  $$\alpha \cdot \sqrt{6} - 2\alpha = (2\sqrt{3} + 3\sqrt{2}) - (2\sqrt{2} + 2\sqrt{3}) = \sqrt{2} \in \mathbb{Q}(\alpha)$$
  
  同理 $\sqrt{3} = \alpha - \sqrt{2} \in \mathbb{Q}(\alpha)$

因此：
$$[\mathbb{Q}(\sqrt{2} + \sqrt{3}) : \mathbb{Q}] = 4$$

极小多项式：$x^4 - 10x^2 + 1$

---

## 关键步骤说明

- **极小多项式**：决定扩张次数的关键
- **塔定律**：分步计算复杂扩张的次数
- **本原元定理**：$\mathbb{Q}(\sqrt{2}, \sqrt{3}) = \mathbb{Q}(\sqrt{2} + \sqrt{3})$ 是本原元的例子
- **基的构造**：乘积构造法
