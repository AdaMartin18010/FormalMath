# 00-工作示例库

本目录汇集了 FormalMath 项目在各数学分支中的规范工作示例，总计 300 余份文档，按照从基础数学到前沿专题的递进结构组织。与“习题示例反例库”侧重练习不同，本目录更强调知识点的示范性讲解：每个文件通常包含概念引入、定理陈述、详细推导、直观解释与形式化对照，帮助学习者建立从直觉理解到严格证明的完整认知链。内容覆盖代数结构、分析学、几何拓扑、数论、概率统计、计算数学、代数几何与同调代数等核心领域。

## 子目录/文件导航

| 子目录 | 说明 |
|--------|------|
| `01-基础数学/` | 集合论、逻辑基础与初等数学示例 |
| `02-代数结构/` | 群、环、域、模与表示论讲解 |
| `03-分析学/` | 实分析、复分析与泛函分析示例 |
| `04-几何与拓扑/` | 微分几何、代数拓扑与流形理论 |
| `05-数论/` | 初等数论与代数数论工作示例 |
| `06-概率统计/` | 概率论与数理统计核心演示 |
| `07-计算数学/` | 数值分析、优化与计算几何 |
| `08-形式化证明/` | Lean4 等形式化工具实践案例 |
| `09-代数几何/` | 代数簇、概形与层论示例 |
| `10-同调代数/` | 导出函子、谱序列与同调技术 |
