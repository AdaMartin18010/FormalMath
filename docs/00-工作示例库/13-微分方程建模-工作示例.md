## 题目

建立人口指数增长模型并求其解；说明何时可忽略容量限制。

---

## 完整解答

### 步骤1：建立微分方程

设 \(P(t)\) 为时刻 \(t\) 的人口数量。

**假设**：人口增长率与当前人口成正比

$$\frac{dP}{dt} = kP$$

其中 \(k > 0\) 为增长率（单位：1/时间）。

**物理意义**：

- 人口越多，新增人口越快
- 无环境容量限制
- 资源无限

### 步骤2：求解微分方程

分离变量：
$$\frac{dP}{P} = k\,dt$$

两边积分：
$$\ln|P| = kt + C_1$$

取指数：
$$P(t) = Ce^{kt}$$

其中 \(C = e^{C_1}\)。

**利用初始条件**：\(P(0) = P_0\)

$$P(t) = P_0 e^{kt}$$

这就是**指数增长模型**。

### 步骤3：解的分析

| 特征 | 说明 |
|-----|------|
| **倍增时间** | \(T = \frac{\ln 2}{k}\)，人口翻倍所需时间 |
| **长期行为** | \(t \to \infty\) 时 \(P(t) \to \infty\) |
| **增长率** | 相对增长率恒为 \(k\) |

### 步骤4：应用场景分析

**适用条件**（可忽略容量限制）：

1. **时间尺度短**：短期内资源相对充足
2. **人口基数小**：相对于环境容量
3. **新物种入侵**：初期无竞争

**不适用情况**：

- 长期预测需用 **Logistic模型**：
  $$\frac{dP}{dt} = kP\left(1 - \frac{P}{K}\right)$$
  其中 \(K\) 为环境容量。

## 应用实例

### 实例1：细菌培养

某细菌每小时增长率为0.3（每小时增长30%）。

- 初始：1000个细菌
- 2小时后：\(1000 \times e^{0.3 \times 2} \approx 1822\) 个
- 倍增时间：\(\ln 2 / 0.3 \approx 2.31\) 小时

### 实例2：放射性衰变

放射性物质以指数方式衰变（\(k < 0\)）。

- 半衰期：\(T_{1/2} = \frac{\ln 2}{|k|}\)
- 碳14测年法基于此原理

| **适用条件** | 短期、小种群、资源充足 |
| **局限性** | 长期需考虑容量限制 |
