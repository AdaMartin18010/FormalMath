## 预备知识

### 部分分式定理

设 \(P(x)/Q(x)\) 是真分式（deg P < deg Q），且 \(Q(x)\) 可分解为：
$$Q(x) = (x-a_1)^{m_1} \cdots (x-a_k)^{m_k} (x^2+b_1x+c_1)^{n_1} \cdots$$

则 \(P(x)/Q(x)\) 可唯一分解为：

**线性因子**:
$$\frac{A_1}{x-a} + \frac{A_2}{(x-a)^2} + \cdots + \frac{A_m}{(x-a)^m}$$

**不可约二次因子**:
$$\frac{B_1x+C_1}{x^2+bx+c} + \frac{B_2x+C_2}{(x^2+bx+c)^2} + \cdots$$

---

## 题目

将 \(\frac{1}{(x-1)(x+2)}\) 部分分式分解并求 \(\int \frac{dx}{(x-1)(x+2)}\)。

---

## 完整解答

### 步骤1：设定分解形式

分母是两个不同线性因子的积：
$$\frac{1}{(x-1)(x+2)} = \frac{A}{x-1} + \frac{B}{x+2}$$

### 步骤2：确定系数

**方法一：通分比较**

通分右边：
$$\frac{A(x+2) + B(x-1)}{(x-1)(x+2)} = \frac{(A+B)x + (2A-B)}{(x-1)(x+2)}$$

比较分子：
$$1 = (A+B)x + (2A-B)$$

比较系数：

- \(x\) 的系数: \(A + B = 0\)
- 常数项: \(2A - B = 1\)

解得：

- 由第一式：\(B = -A\)
- 代入第二式：\(2A - (-A) = 3A = 1\)，得 \(A = \frac{1}{3}\)
- \(B = -\frac{1}{3}\)

**方法二：Heaviside覆盖法（更快捷）**

求 \(A\)：两边乘以 \((x-1)\)，令 \(x = 1\)
$$A = \frac{1}{1+2} = \frac{1}{3}$$

求 \(B\)：两边乘以 \((x+2)\)，令 \(x = -2\)
$$B = \frac{1}{-2-1} = -\frac{1}{3}$$

### 步骤3：写出分解结果

$$\frac{1}{(x-1)(x+2)} = \frac{1/3}{x-1} - \frac{1/3}{x+2}$$

### 步骤4：积分

$$\int \frac{dx}{(x-1)(x+2)} = \int \left(\frac{1/3}{x-1} - \frac{1/3}{x+2}\right) dx$$

$$= \frac{1}{3} \ln|x-1| - \frac{1}{3} \ln|x+2| + C$$

$$= \boxed{\frac{1}{3} \ln\left|\frac{x-1}{x+2}\right| + C}$$

---

## 常见类型总结

| 分母类型 | 分解形式 | 系数求法 |
|---------|---------|---------|
| \((x-a)(x-b)\) | \(\frac{A}{x-a} + \frac{B}{x-b}\) | Heaviside法 |
| \((x-a)^2\) | \(\frac{A}{x-a} + \frac{B}{(x-a)^2}\) | 比较系数 |
| \(x^2+bx+c\)（不可约）| \(\frac{Ax+B}{x^2+bx+c}\) | 比较系数 |

---

## 验证

对结果求导验证：
$$\frac{d}{dx}\left[\frac{1}{3}\ln|x-1| - \frac{1}{3}\ln|x+2|\right] = \frac{1}{3(x-1)} - \frac{1}{3(x+2)}$$

$$= \frac{(x+2) - (x-1)}{3(x-1)(x+2)} = \frac{3}{3(x-1)(x+2)} = \frac{1}{(x-1)(x+2)}$$
