## 摘要

本报告将FormalMath项目与东京大学大学院数理科学研究科（Graduate School of Mathematical Sciences, University of Tokyo）的课程体系进行深度对齐。东京大学数学系代表了日本传统数学教育的最高水准，其课程设计体现了日本数学教育的核心特点：重视计算技巧（計算力重視）、系统性强（系統性重視）、以及与欧美体系的独特差异。

## 一、东京大学数理科学研究科课程概览

### 1.1 课程体系结构

东京大学数理科学研究科采用**学部（本科）+ 大学院（研究生）**的两段制结构：

```
┌─────────────────────────────────────────────────────────────┐
│                    东京大学数学课程体系                       │
├─────────────────────────────────────────────────────────────┤
│  学部（本科）1-4年                                            │
│  ├── 基础课程（1-2年）                                         │
│  │   ├── 解析学I/II/III（微积分→实分析→复分析）               │
│  │   ├── 代数学I/II（线性代数→抽象代数）                      │
│  │   └── 幾何学I/II（微分几何→拓扑学基础）                    │
│  │                                                            │
│  ├── 专业课程（3-4年）                                         │
│  │   ├── 代数学III（体论·Galois理论）                         │
│  │   ├── 代数学特論（代数几何、数论、表示论）                  │
│  │   ├── 解析学特論（泛函分析、偏微分方程）                    │
│  │   └── 幾何学特論（微分几何、代数拓扑）                     │
│  │                                                            │
│  └── 演习课程（1-4年）                                         │
│      └── 代数学/解析学/幾何学特別演習（习题课+研讨）          │
├─────────────────────────────────────────────────────────────┤
│  大学院（研究生）修士2年 + 博士3年                              │
│  ├── 基础课程（修士前期）                                       │
│  │   ├── 代数学概論（环论、模论、同调代数）                    │
│  │   ├── 幾何学概論（拓扑空间、基本群、覆盖空间）              │
│  │   └── 解析学概論（测度论、泛函分析、概率论）                │
│  │                                                            │
│  └── 专门课程（修士后期+博士）                                  │
│      ├── 代数幾何（代数簇、概形、层上同调）                   │
│      ├── 微分幾何（黎曼几何、辛几何、复几何）                  │
│      ├── 代数トポロジー（同调论、同伦论、示性类）              │
│      ├── 表現論（李群、李代数、自守形式）                      │
│      ├── 数論幾何（算术几何、p进数论、Langlands纲领）         │
│      └── 複素解析幾何（多复变函数、复流形、层论）              │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 课程特点

| 特点 | 描述 | 与欧美差异 |
|------|------|------------|
| **計算力重視** | 强调基础计算技巧和手算能力 | 欧美更侧重概念理解和证明 |
| **演习制度** | 必修习题课（演習），强化解题训练 | 欧美习题课多为选修 |
| **系统性强** | 从小学到大学的连贯知识谱系 | 日本数学教育具有独特的系统性 |
| **教材传统** | 寺澤·藤原·黒田等经典教材体系 | 形成了独特的日本数学教材风格 |

| 精確性 | 重视计算精度和严密性 | Lean4形式化证明确保严密性 |

### 5.2 寺澤·藤原·黒田教材传统

日本数学教材形成了独特的传统，以几位著名数学家的教材为代表：

| 教材系列 | 代表作者 | 特点 | FormalMath对齐 |

### 5.3 与欧美体系的差异

| 方面 | 日本体系 | 欧美体系 | FormalMath整合 |
|------|----------|----------|----------------|
| **教学风格** | 循序渐进，重视基础 | 快速进入前沿 | L0-L4层次体系兼容两者 |
| **证明要求** | 强调计算验证 | 强调抽象证明 | 两种视角（集合论+范畴论） |
| **课程组织** | 代数/几何/分析三轨并行 | 交叉融合 | 概念关联网络支持交叉 |
| **术语使用** | 保留汉字术语 | 拉丁字母为主 | 提供日英对照 |
| **习题比重** | 高（演习课程） | 中等 | 提供大量习题资源 |
