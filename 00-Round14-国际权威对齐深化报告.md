## 二、知识梳理深化

### 1. 概念层次结构

**代数结构层次**（已文档化）:

```
原群 → 半群 → 幺半群 → 群 → Abel群 → 环 → 整环 → UFD → PID → 域
```

**可微性层次**:

```
连续 → Lipschitz → 可微 → C¹ → C^k → C^∞ → 解析
```

**收敛性层次**:

```
一致收敛 ⇒ 逐点收敛 ⇐ Lp收敛 ⇐ 几乎处处收敛 (有限测度) ⇒ 依测度收敛
```

### 2. 反例体系

**分析学反例**:

- Dirichlet函数（处处不连续）
- Thomae函数（有理点间断）
- Weierstrass函数（连续不可微）
- Cantor函数（魔鬼楼梯）

**拓扑学反例**:

- Sierpinski空间（T0但非T1）
- 拓扑学家正弦曲线（连通但非道路连通）
- 长线（局部像R但非仿紧）

**代数学反例**:

- ℤ[√-5]（非UFD）
- A4（无6阶子群，Lagrange逆不成立）

### 3. 实例库

**特殊函数**: Gamma, Beta, ζ, Bessel, 等
