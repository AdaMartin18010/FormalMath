# FormalMath PWA 使用指南

## 什么是 PWA？

PWA（Progressive Web App，渐进式 Web 应用）是一种使用现代 Web 技术构建的应用程序，可以提供类似原生应用的用户体验。

## FormalMath PWA 功能

### 1. 添加到主屏幕
将 FormalMath 安装到您的设备主屏幕，像使用原生应用一样快速访问。

#### Android (Chrome)
1. 打开 FormalMath 网站
2. 点击浏览器菜单（三个点）
3. 选择"添加到主屏幕"或"安装应用"
4. 确认安装

#### iOS (Safari)
1. 打开 FormalMath 网站
2. 点击分享按钮（底部中间的方框）
3. 向上滚动，选择"添加到主屏幕"
4. 点击"添加"

#### 桌面 (Chrome/Edge)
1. 打开 FormalMath 网站
2. 地址栏右侧会出现安装图标
3. 点击"安装 FormalMath"

### 2. 离线访问
安装后，即使在没有网络的情况下，也可以：
- 浏览已加载的知识图谱
- 查看已访问过的页面
- 使用基本功能

### 3. 深色模式

#### 手动切换
1. 打开"更多"菜单（底部导航的"更多"按钮）
2. 选择"外观"
3. 选择"浅色模式"、"深色模式"或"跟随系统"

#### 自动深色模式
应用可以根据时间自动切换深色模式：
- 晚上 18:00 后自动切换到深色
- 早上 6:00 后自动切换到浅色

### 4. 推送通知

#### 启用学习提醒
1. 首次访问时，浏览器会请求通知权限
2. 点击"允许"
3. 系统会定期发送学习提醒

#### 通知类型
- **学习提醒**: 定期提醒您继续学习
- **知识更新**: 新内容添加时通知
- **成就解锁**: 完成学习目标时祝贺

### 5. 手势操作

#### 知识图谱页面
- **双指捏合**: 放大/缩小图谱
- **单指拖动**: 平移视图
- **双击**: 快速放大到指定位置
- **长按节点**: 显示节点详情

#### 页面导航
- **左右滑动**: 切换页面（开发中）
- **摇一摇**: 快速打开搜索

### 6. 摇一摇快捷操作

#### 使用方法
1. 确保在移动端使用
2. 握住手机，快速摇动
3. 自动打开搜索功能

#### 自定义
可以在设置中：
- 开启/关闭摇一摇功能
- 设置摇动灵敏度
- 选择摇一摇触发的操作

## 故障排除

### 无法安装 PWA

#### iOS Safari
- 确保使用 Safari 浏览器
- 确保网站已完全加载
- 尝试刷新页面后重试

#### Android Chrome
- 检查 Chrome 是否为最新版本
- 确保有足够的存储空间
- 清除浏览器缓存后重试

### 离线功能不工作
1. 确保已正确安装 PWA
2. 首次访问时需要联网加载资源
3. 检查浏览器是否允许 Service Worker

### 推送通知不显示
1. 检查浏览器通知权限是否开启
2. iOS 需要 16.4+ 版本且已添加到主屏幕
3. 检查系统通知设置

### 深色模式不生效
1. 检查是否选择了"跟随系统"
2. 系统深色模式可能需要在系统设置中开启
3. 尝试手动选择深色模式
