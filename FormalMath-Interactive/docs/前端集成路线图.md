# FormalMath Interactive 前端集成路线图

## 1. 现状

`FormalMath-Interactive/` 目录包含基于 Node.js 的交互式学习平台骨架：

- `src/`：前端源码（React/Vue框架待确认）
- `server/`：后端API服务
- `public/`：静态资源

## 2. 集成目标

将银层 6 门核心课程（61篇文档）+ 金层 Grothendieck 50篇文档接入前端，提供：

1. **在线阅读**：Markdown渲染 + LaTeX公式支持
2. **Lean4代码运行**：WebAssembly或云端Lean4编译器
3. **进度追踪**：用户学习进度、习题完成情况
4. **搜索与导航**：全文搜索 + 交叉引用跳转

## 3. 技术方案

### 3.1 内容接入

- 将银层/金层 Markdown 文件编译为结构化 JSON
- 提取 YAML frontmatter 作为元数据
- 建立文件系统监控，自动同步更新

### 3.2 Lean4在线运行

- 方案A：WebAssembly Lean4（编译慢但无需后端）
- 方案B：云端Lean4服务（响应快但需服务器资源）
- 推荐：方案B作为默认，方案A作为离线备用

### 3.3 搜索系统

- 本地：Lunr.js 或 FlexSearch（轻量）
- 高级：Elasticsearch 或 Meilisearch（需部署）
