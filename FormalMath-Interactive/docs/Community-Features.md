# 学习社区功能文档

## 概述

学习社区是 FormalMath-Interactive 平台的重要组成部分，为数学学习者提供交流、协作和分享的空间。

## 功能模块

### 1. 讨论区 (DiscussionForum)

讨论区是学习者交流的主要场所，支持以下功能：

- **主题发布**: 用户可以发布讨论主题，支持多种分类
  - 综合讨论
  - 问题求助
  - 学习分享
  - 挑战讨论
  - 资源分享
  - 公告

- **回复系统**: 支持嵌套回复，便于深度讨论
- **点赞功能**: 对优质内容进行点赞
- **置顶/锁定**: 管理员可以对重要主题进行置顶或锁定
- **标签系统**: 便于内容分类和检索
- **搜索筛选**: 支持按分类、关键词搜索

### 2. 问答系统 (QASystem)

专业的数学问答平台，特点：

- **悬赏机制**: 用户可以设置积分悬赏，鼓励优质回答
- **难度分级**: 入门、中级、高级、专家四个等级
- **答案采纳**: 提问者可以采纳最佳答案
- **投票系统**: 社区投票决定答案质量
- **标签关联**: 问题可以与数学概念关联

### 3. 学习小组 (StudyGroups)

协作学习的组织形式：

- **小组创建**: 用户可创建公开或私密小组
- **成员管理**: 支持设置管理员、成员角色
- **活动安排**: 支持创建学习活动、设置日程
- **资源共享**: 小组内共享学习资源
- **讨论专区**: 小组专属讨论区

### 4. 资源分享 (ResourceSharing)

学习资料的集中地：

- **资源类型**:
  - 笔记
  - 文章
  - 视频
  - 代码
  - 题目
  - 书籍
  - 外部链接

- **上传下载**: 支持文件上传和下载
- **点赞收藏**: 优质资源推荐
- **评论互动**: 对资源进行讨论

### 5. 积分系统 (PointsSystem)

激励学习者的 gamification 机制：

#### 积分获取方式
- 每日登录: +10 分
- 发布主题: +20 分
- 发布回复: +10 分
- 获得点赞: +5 分
- 答案被采纳: +50 分
- 分享资源: +30 分
- 完成挑战: +100 分
- 帮助他人: +15 分

#### 等级系统
| 等级 | 名称 | 积分要求 | 特权 |
|------|------|----------|------|
| 1 | 数学新手 | 0 | 基础功能 |
| 3 | 探索者 | 300 | 发布讨论、上传资源 |
| 5 | 初级学者 | 1000 | 悬赏提问 |
| 7 | 熟练学者 | 2200 | 小组管理 |
| 10 | 大师 | 5500 | 内容审核 |
| 12 | 传奇 | 10000 | 所有权限 |

#### 徽章系统
- 参与类徽章
- 成就类徽章
- 贡献类徽章
- 专业类徽章
- 社交类徽章

#### 排行榜
- 总积分榜
- 周榜/月榜
- 签到排行榜

## 技术实现

### 前端组件

```
src/components/Community/
├── DiscussionForum.tsx    # 讨论区组件
├── QASystem.tsx           # 问答系统组件
├── StudyGroups.tsx        # 学习小组组件
├── ResourceSharing.tsx    # 资源分享组件
├── PointsSystem.tsx       # 积分系统组件
└── index.ts               # 组件导出
```

### 页面路由

```
src/pages/
├── Community/
│   └── index.tsx          # 社区主页
└── Admin/
    └── CommunityAdmin.tsx # 管理后台
```

### API 接口

```
src/api/community.ts
```

主要 API 模块：
- `discussionApi`: 讨论区接口
- `qaApi`: 问答系统接口
- `groupApi`: 学习小组接口
- `resourceApi`: 资源分享接口
- `pointsApi`: 积分系统接口
- `statsApi`: 统计接口

### 类型定义

```
src/types/community.ts
```

包含所有社区相关的 TypeScript 类型定义。

### 后端 API

```
server/routes/community.js
```

Express 路由，提供 RESTful API。

## 配置说明

### 积分规则配置

在管理后台的"社区设置"中可以调整：
- 各项行为的积分值
- 每日积分上限
- 等级晋升阈值

### 社区规范

建议在社区首页展示：
1. 尊重他人，友善交流
2. 提问前请先搜索
3. 提供完整的上下文
4. 分享优质学习资源
5. 禁止发布不当内容
