## 📊 Project Overview

FormalMath is a comprehensive knowledge base for mathematics education, dedicated to:

- 🎯 **Complete Knowledge System**: Structured content from basic mathematics to modern mathematical frontiers
- 🧠 **Mathematician Ideology System**: In-depth analysis of 62 mathematicians' thoughts, contributions, and modern impact
- 🌍 **International Authority Alignment**: Detailed correspondence with 121+ courses from MIT, Harvard, Stanford, and six other universities
- 💻 **Formalized Proofs**: 20 Lean4 theorem formalizations, aligned with Mathlib4 v4.29.0
- 🤖 **Intelligent Learning System**: Cognitive diagnosis, learning assessment, and adaptive learning path systems

### Quick Browse

No installation required, browse all documents directly via GitHub:

```bash
# Browse project main document
open README.md

# Browse full index
open INDEX.md

# Browse document center
open docs/README.md
```

### Local Clone

```bash
# Clone repository
git clone https://github.com/formalmath/formalmath.git
cd formalmath

# Use any Markdown reader to view documents
# Or use VS Code for better reading experience
```

### Lean4 Formalized Code Execution

```bash
# Enter Lean4 project directory
cd docs/09-形式化证明/Lean4

# Build using Lake
lake build

# Run tests
lake test
```
