## 工具使用说明

### link_checker.py - 链接检查工具

**功能:**
- 递归扫描所有Markdown文件
- 检查内部链接有效性
- 验证锚点引用正确性
- 生成详细检查报告

**用法:**
```bash
python tools/link_checker.py [项目根目录路径]
```

**输出:**
- `output/link_check_report.json` - JSON格式详细报告
- `output/link_check_report.txt` - 文本格式报告

### link_fixer.py - 链接修复工具

**功能:**
- 自动修复常见链接问题
- 建议替代链接方案
- 批量应用修复

**用法:**
```bash
# 干运行模式(预览修复建议)
python tools/link_fixer.py --quick

# 应用修复
python tools/link_fixer.py --quick --apply

# 完整模式(更智能但较慢)
python tools/link_fixer.py --apply
```

**输出:**
- `output/link_fix_report.md` - Markdown格式修复报告
- `output/link_fix_report.json` - JSON格式详细数据
