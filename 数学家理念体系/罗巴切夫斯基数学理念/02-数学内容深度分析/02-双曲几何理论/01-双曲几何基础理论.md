## 一、双曲几何的定义

### 1.1 双曲几何的概念

**双曲几何的定义**：

双曲几何是具有负曲率的非欧几何，由罗巴切夫斯基在1829年建立。

**数学表述**：

**双曲公设**（否定欧几里得第五公设）：
$$\text{过直线外一点，有至少两条直线与已知直线平行}$$

**双曲几何模型**：

1. **Poincaré圆盘模型**：
   - 单位圆盘：$\{z \in \mathbb{C}: |z| < 1\}$
   - 双曲距离：$d(z_1, z_2) = \text{arccosh}\left(1 + \frac{2|z_1 - z_2|^2}{(1-|z_1|^2)(1-|z_2|^2)}\right)$

2. **上半平面模型**：
   - 上半平面：$\{z \in \mathbb{C}: \text{Im}(z) > 0\}$
   - 双曲距离：$d(z_1, z_2) = \text{arccosh}\left(1 + \frac{|z_1 - z_2|^2}{2\text{Im}(z_1)\text{Im}(z_2)}\right)$

### 1.2 双曲几何的例子

**双曲几何的典型例子**：

1. **双曲平面**：
   - Poincaré圆盘模型
   - 上半平面模型

2. **双曲空间**：
   - 三维双曲空间：$\mathbb{H}^3$
   - $n$ 维双曲空间：$\mathbb{H}^n$

**数学表述**：

- **双曲三角形**：内角和小于 $180°$
- **双曲圆**：双曲距离相等的点的集合

## 二、双曲几何的性质

### 2.1 基本性质

**双曲几何的基本性质**：

1. **负曲率**：
   - 双曲几何具有常数负曲率 $K = -1$
   - 与欧几里得几何（$K = 0$）和球面几何（$K > 0$）不同

2. **平行性**：
   - 过直线外一点有无限多条平行线
   - 与欧几里得几何（唯一平行线）不同

3. **一致性**：
   - 双曲几何是一致的（不产生矛盾）
   - 可以通过模型验证

**数学表述**：

**三角形内角和**：
$$\angle A + \angle B + \angle C < 180°$$

**面积公式**：
$$S = \pi - (\angle A + \angle B + \angle C)$$

### 2.2 双曲几何的结构

**双曲几何的结构**：

双曲几何的结构由公理系统决定。

**数学表述**：

1. **公理系统**：
   - 保留欧几里得前4条公设
   - 否定第5条公设（平行公设）

2. **等距群**：
   - 双曲等距：保持双曲距离的变换
   - Möbius变换：在Poincaré模型中
