## 📖 项目概述

本项目致力于全面梳理和呈现**尼古拉·罗巴切夫斯基**（Nikolai Lobachevsky, 1792-1856）的数学理念体系。罗巴切夫斯基是非欧几何的创始人之一，在双曲几何和非欧几何方面做出了重要贡献，对现代几何学和数学基础产生了深远影响。
