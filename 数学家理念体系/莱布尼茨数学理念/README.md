## 📚 核心内容方向

### 1. 微积分

- **历史背景**: 1680年代微分的发现
- **数学内容**: 微分符号、积分符号、基本定理
- **应用**: 数学、物理

### 2. 符号逻辑

- **核心思想**: 用符号表示逻辑推理
- **方法**: 通用符号语言、逻辑演算
- **影响**: 现代逻辑学的基础

### 3. 组合数学

- **贡献**: 组合数学的基础
- **方法**: 排列组合、生成函数
- **应用**: 概率论、离散数学
