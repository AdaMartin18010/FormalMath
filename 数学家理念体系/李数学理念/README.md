## 📚 核心内容方向

### 1. 李群理论

- **历史背景**: 1870年代开始研究连续变换群
- **数学内容**: 群结构与流形结构的结合
- **应用**: 对称性、物理中的规范群、几何

### 2. 李代数理论

- **核心思想**: 李群的无穷小生成元
- **方法**: 线性化、交换子运算
- **应用**: 表示理论、量子力学

### 3. 连续群理论

- **贡献**: 从离散群到连续群的推广
- **方法**: 无穷小方法、微分方程
- **影响**: 现代群论的基础

### 4. 微分几何

- **贡献**: 接触变换、不变式理论
- **方法**: 微分形式、切空间
- **应用**: 几何不变性、对称性
