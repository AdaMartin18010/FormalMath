## 🔑 核心概念

### ZF公理系统

ZF（Zermelo-Fraenkel）公理系统是现代集合论的基础，由Zermelo和Fraenkel共同建立。

**主要公理**：
1. 外延公理：集合由其元素唯一确定
2. 空集公理：存在空集
3. 配对公理：任意两个集合可以配对
4. 并集公理：集合的并集存在
5. 幂集公理：集合的幂集存在
6. 无穷公理：存在无穷集
7. 分离公理：子集存在
8. 替换公理：函数像存在
9. 正则公理：集合的良基性

### 替换公理

替换公理是ZF系统的重要组成部分，它保证了函数像的存在性。

**意义**：
- 允许通过函数构造新集合
- 保证了序数和基数的存在
- 是现代集合论的基础
