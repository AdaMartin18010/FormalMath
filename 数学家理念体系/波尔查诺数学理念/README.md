## 📖 项目概述

本项目致力于全面梳理和呈现**伯纳德·波尔查诺**（Bernard Bolzano, 1781-1848）的数学理念体系。波尔查诺是实分析的先驱，在连续性理论、极限理论和数学基础方面做出了重要贡献，对现代分析学产生了深远影响。
