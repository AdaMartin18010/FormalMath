## 一、引言：Hauptsatz的历史地位

### 1.1 根岑的动机

1934-1935年，格哈德·根岑在他的博士论文《逻辑演绎研究》中提出了两个革命性的证明系统——自然演绎和序列演算。但论文的核心亮点是**切割消除定理**（Hauptsatz）。

根岑设计序列演算的直接动机是解决以下问题：

**问题**：在自然演绎中，证明包含**迂回**（detour）：通过引入规则引入一个公式，又通过消去规则消去它。消除这种迂回的**证明归一化**过程极其复杂。

**解决方案**：序列演算中的"切割"（cut）直接对应自然演绎中的迂回，而切割消除具有清晰的结构。

### 1.2 定理的命名与意义

根岑将切割消除定理命名为 **"Hauptsatz"**（德语：主定理），凸显其核心地位。

**Hauptsatz的重要性**：

| 方面 | 意义 |
|------|------|
| **元理论** | 提供分析证明结构的强大工具 |
| **一致性** | 使一致性证明成为可能 |
| **可判定性** | 某些逻辑片段可判定 |
| **证明搜索** | 证明搜索空间有限 |

## 二、切割规则的形式化定义

### 2.1 切割规则

在LK系统中，**切割规则**定义如下：

$$\frac{\Gamma \vdash A, \Delta \quad \Gamma', A \vdash \Delta'}{\Gamma, \Gamma' \vdash \Delta, \Delta'}(\text{Cut})$$

其中：
- 公式 $A$ 称为**切公式**（cut formula）
- 左侧的子证明称为**左前提**（left premise）
- 右侧的子证明称为**右前提**（right premise）

### 2.2 切割的直观理解

**逻辑直觉**：

如果我们能从假设 $\Gamma$ 推出 $A$（或 $\Delta$ 中的某个），并且能从 $\Gamma'$ 和 $A$ 推出 $\Delta'$，那么从 $\Gamma$ 和 $\Gamma'$ 可以推出 $\Delta$ 或 $\Delta'$。

**类比**：切公式 $A$ 就像证明中的"桥梁"或"中间结论"，连接两个子证明。切割消除定理表明：任何需要桥梁的证明都可以"拉直"，直接从前提到结论。

**示例**：

假设我们要证：$\Gamma, \Gamma' \vdash \Delta, \Delta'$

使用切割：
1. 先证：$\Gamma \vdash A, \Delta$（从 $\Gamma$ 得到中间结论 $A$）
2. 再证：$\Gamma', A \vdash \Delta'$（用 $A$ 推出 $\Delta'$）

切割消除告诉我们：可以直接证明，不需要 $A$ 这个中间步骤。

### 2.3 多重切割与混合规则

**混合规则**（Mix Rule）是切割的变体：

$$\frac{\Gamma \vdash \Delta \quad \Gamma' \vdash \Delta'}{\Gamma, \Gamma' \setminus \{A\} \vdash \Delta \setminus \{A\}, \Delta'}(\text{Mix})$$

条件：$A$ 在 $\Gamma'$ 和 $\Delta$ 中都出现。

根岑原始证明使用了Mix规则，现代证明通常直接使用Cut规则。

---

## 三、Hauptsatz的正式陈述

**定理**（切割消除定理，Gentzen's Hauptsatz，1935）：

> 在序列演算LK（经典逻辑）或LJ（直觉主义逻辑）中，任何使用切割规则的证明都可以转化为不使用切割规则的证明。

形式化：

$$\vdash_{LK+Cut} \Delta \implies \vdash_{LK} \Delta$$

**等价表述**：

- 切割规则在LK/LJ中是**可容许的**（admissible）：添加它不会增加可证序列
- 任何使用切割的证明都可以**消除**切割

**重要说明**：
- 消除切割后，证明长度可能**指数级增长**
- 甚至可能是**非初等增长**（对于包含归纳的系统）
- 但在纯逻辑中，增长是有界的

## 四、证明思路：归纳法框架

切割消除的证明使用**双重归纳法**，这是理解证明结构的关键。

### 4.1 复杂度度量

为了进行归纳，我们需要度量"切割的复杂度"：

**定义**（秩，Rank）：

切公式 $A$ 的**秩** $r(A)$ 定义如下：
- $r(P) = 0$（$P$ 为原子公式）
- $r(A \land B) = r(A \lor B) = r(A \to B) = \max(r(A), r(B)) + 1$
- $r(\neg A) = r(A) + 1$
- $r(\forall x A) = r(\exists x A) = r(A) + 1$

**定义**（度，Degree）：

一个切割的**度**是其切公式的秩。

**定义**（证明的高度）：

证明树的高度 $h(\pi)$ 是从根到叶子的最长路径上的节点数。

### 4.2 主要情况分析

切割消除将切割"向上推"，根据切公式在左右前提中是否是**主公式**（principal formula）分为不同情况。

**主公式**：在某条规则的最后一步中，通过该规则引入的公式。

**副公式**（side formula）：规则中传递下去的公式。

### 4.3 归纳步骤

**归纳假设**：

对于所有满足以下之一的切割：
- 度更小，或
- 度相同但左或右子证明高度更短

都可以被消除。

**目标**：证明当前切割也可消除。

## 五、关键情况的详细分析

### 5.1 公理情况

**情况1**：左前提是公理 $A \vdash A$

$$\frac{A \vdash A \quad \Gamma, A \vdash \Delta}{\Gamma, A \vdash \Delta}$$

**消除**：直接使用右前提 $\Gamma, A \vdash \Delta$（因为左前提只是重言式）。

**情况2**：右前提是公理 $A \vdash A$

$$\frac{\Gamma \vdash A, \Delta \quad A \vdash A}{\Gamma \vdash A, \Delta}$$

**消除**：直接使用左前提 $\Gamma \vdash A, \Delta$。

### 5.2 主公式情况

这是最关键的情况：切公式在左右前提中都是**主公式**（即最后一步引入的公式）。

**示例**：切公式为 $A \land B$

原始证明：

$$\frac{\frac{\vdots}{\Gamma \vdash A, \Delta} \quad \frac{\vdots}{\Gamma \vdash B, \Delta}}{\Gamma \vdash A \land B, \Delta}(\land\text{R}) \quad \frac{\frac{\vdots}{\Gamma', A, B \vdash \Delta'}}{\Gamma', A \land B \vdash \Delta'}(\land\text{L})}{\Gamma, \Gamma' \vdash \Delta, \Delta'}(\text{Cut})$$

**转换**：用两次更简单（度更低）的切割替代：

$$\frac{\frac{\vdots}{\Gamma \vdash A, \Delta} \quad \frac{\frac{\vdots}{\Gamma \vdash B, \Delta} \quad \frac{\vdots}{\Gamma', A, B \vdash \Delta'}}{\Gamma, \Gamma', A \vdash \Delta, \Delta'}(\text{Cut}, B)}{\Gamma, \Gamma, \Gamma' \vdash \Delta, \Delta, \Delta'}(\text{Cut}, A)}{\Gamma, \Gamma' \vdash \Delta, \Delta'}$$

**关键点**：
- 原来的切割：切公式 $A \land B$，秩 = $\max(r(A), r(B)) + 1$
- 新的切割：切公式 $A$ 和 $B$，秩分别为 $r(A)$ 和 $r(B)$
- **度降低**，可以应用归纳假设

### 5.3 副公式情况

**情况**：切公式在一侧（或两侧）不是主公式。

**示例**：在左前提中，切公式 $A$ 是副公式（通过弱化引入）

$$\frac{\frac{\vdots}{\Gamma \vdash \Delta}}{\Gamma \vdash A, \Delta}(\text{WR}) \quad \Gamma', A \vdash \Delta'}{\Gamma, \Gamma' \vdash \Delta, \Delta'}$$

**转换**：直接弱化左子证明的结论

$$\frac{\vdots}{\Gamma \vdash \Delta} \implies \Gamma, \Gamma' \vdash \Delta, \Delta'$$

使用弱化规则添加 $\Gamma'$ 和 $\Delta'$。

**一般策略**：
- 如果切公式在某侧不是主公式，就将切割向上"推"过该规则
- 这样降低子证明的高度，可以应用归纳假设

## 六、子公式性质与一致性证明

### 6.1 子公式性质

**定理**（子公式性质）：

在无切证明中，出现的所有公式都是**最终结论中公式的子公式**。

**证明思路**：

检查所有规则（除切割外）：
- 公理 $A \vdash A$：只涉及 $A$
- 结构规则：只传递已有公式
- 逻辑规则：结论中的公式是前提中公式的子公式

例如：$\land$R 规则

$$\frac{\Gamma \vdash A, \Delta \quad \Gamma \vdash B, \Delta}{\Gamma \vdash A \land B, \Delta}$$

结论中的 $A \land B$ 以子公式形式出现在前提中的 $A$ 和 $B$ 中。

**意义**：
- 证明是"自包含"的
- 不需要引入"外部"公式
- 证明搜索空间有限

### 6.2 一致性证明应用

**定理**：纯逻辑LK（不含归纳）是一致的，即不能证明空序列 $\vdash$（矛盾）。

**证明**：

1. 假设 $\vdash$ 可证
2. 由切割消除，存在无切证明
3. 检查无切证明的最后规则：
   - 不可能是公理（公理形式为 $A \vdash A$）
   - 不可能是结构规则（需要前提有公式）
   - 逻辑规则需要前提有连接词
4. 不可能构造出 $\vdash$ 的无切证明
5. 矛盾！

**对于PA（皮亚诺算术）**：

根岑1936年证明了PA的一致性，使用：
- 将PA证明转化为序列演算
- 超穷归纳到 $\varepsilon_0$
- 切割消除（限制形式）

## 七、切割消除的计算复杂性

切割消除可能显著增加证明长度。

**定理**（Statman, 1979）：

切割消除可以导致证明长度的**非初等增长**。

具体地，存在一序列公式 $A_n$，使得：
- 使用切割的证明长度为 $O(n)$
- 任何无切证明长度至少为 $2^{2^{\cdot^{\cdot^{2}}}}$（$n$ 层指数塔）

**实际意义**：
- 切割消除是重要的理论工具
- 但在实践中直接消除所有切割不现实
- 现代定理证明器使用**受控切割**（controlled cut）

## 八、现代发展与应用

### 8.1 线性逻辑与聚焦

1987年，Jean-Yves Girard 提出**线性逻辑**（Linear Logic），革新了序列演算。

**关键洞见**：将结构规则（弱化、收缩）显式化

| 逻辑 | 结构规则 |
|------|----------|
| 经典/直觉主义 | 隐含在序列中 |
| 线性逻辑 | 需要显式模态（!, ?）|

**聚焦**（Focusing）：Andreoli (1992) 发现证明可以组织为"聚焦阶段"，显著优化证明搜索。

### 8.2 证明网

Girard 的证明网（Proof Nets）是序列演算的几何化：
- 消除证明中的"bureaucracy"（规则应用顺序无关的细节）
- 切割消除对应图的归约
- 完美对应线性逻辑的语义

### 8.3 类型论与Curry-Howard

在Curry-Howard对应中：

| 逻辑 | 计算 |
|------|------|
| 切割 | 函数复合/let绑定 |
| 切割消除 | β归约/程序执行 |
| 子公式性质 | 类型安全性 |

**在Lean4/Coq中**：
- 证明项包含"let定义"（对应切割）
- 类型检查包含隐含的切割消除
- 证明归一化对应程序求值

```lean4
-- 切割对应let绑定
def cut_example : A → C :=
  fun a => 
    let b : B := f a  -- 切割：A ⊢ B
    g b             -- 切割：B ⊢ C
```

## 九、总结

切割消除定理（Hauptsatz）是根岑对证明论最深刻的贡献：

1. **结构清晰**：揭示证明的本质结构，去除"桥梁"公式
2. **元理论强大**：提供一致性证明、可判定性分析的工具
3. **计算解释**：对应程序归约，连接逻辑与计算
4. **现代延续**：线性逻辑、证明网、聚焦证明的发展基础

Hauptsatz不仅解决了希尔伯特纲领中的技术问题，更开创了现代证明论的新纪元。
