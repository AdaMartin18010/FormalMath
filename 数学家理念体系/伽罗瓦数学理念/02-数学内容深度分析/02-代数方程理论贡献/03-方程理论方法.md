## 二、群论方法

### 2.1 伽罗瓦群方法

伽罗瓦群方法是方程理论方法的核心：

**Galois群的定义**：

设 $f(x) \in F[x]$ 是不可约多项式，$E$ 是 $f(x)$ 在 $F$ 上的分裂域，则：

$$\text{Gal}(E/F) = \{\sigma: E \to E : \sigma \text{ 是 } F\text{-自同构}\}$$

**Galois群的性质**：

- **群结构**：Galois群是有限群
- **阶数**：$|\text{Gal}(E/F)| = [E:F]$（如果 $E/F$ 是Galois扩张）
- **结构反映**：群的结构反映方程的结构

**应用**：

- **可解性判断**：通过Galois群的结构判断可解性
- **结构分析**：分析方程的结构
- **问题解决**：解决可解性问题

### 2.2 结构分析方法

结构分析方法是方程理论方法的重要组成部分：

**群的结构分析**：

- **子群分析**：通过子群分析群的结构
- **商群分析**：通过商群简化问题
- **可解性分析**：分析群的可解性

**应用**：

- **可解性判断**：判断Galois群是否是可解群
- **结构理解**：理解方程的结构
- **问题解决**：解决可解性问题

### 2.3 对称性方法

对称性方法是方程理论方法的重要方面：

**根的对称性**：

- **根的关系**：方程的根之间存在对称性
- **置换作用**：置换作用在根上
- **对称群**：这些置换形成群

**应用**：

## 三、域论方法

### 3.1 域扩张方法

域扩张方法是方程理论方法的重要组成部分：

**分裂域构造**：

- **定义**：分裂域是包含方程所有根的最小域
- **构造**：通过添加根构造分裂域
- **性质**：分裂域是有限扩张

**应用**：

- **Galois群**：在分裂域上定义Galois群
- **结构分析**：分析域扩张的结构
- **问题解决**：解决可解性问题

### 3.2 分裂域方法

分裂域方法是方程理论方法的核心：

**分裂域的定义**：

设 $f(x) \in F[x]$ 是多项式，$E$ 是 $f(x)$ 在 $F$ 上的分裂域，如果：
1. $f(x)$ 在 $E$ 中完全分解：$f(x) = a(x - \alpha_1)(x - \alpha_2) \cdots (x - \alpha_n)$
2. $E = F(\alpha_1, \alpha_2, \ldots, \alpha_n)$

**分裂域的性质**：

- **存在性**：任何多项式都有分裂域
- **唯一性**：分裂域在同构意义下唯一
- **有限性**：分裂域是有限扩张

### 3.3 Galois对应方法

Galois对应方法是方程理论方法的核心：

**Galois基本定理**：

设 $E/F$ 是Galois扩张，$G = \text{Gal}(E/F)$，则：

1. **对应关系**：子群 $H \leq G$ 与中间域 $K$（$F \subset K \subset E$）之间存在一一对应：
   $$H \leftrightarrow E^H, \quad K \leftrightarrow \text{Gal}(E/K)$$

2. **包含关系反转**：如果 $H_1 \leq H_2$，则 $E^{H_2} \subset E^{H_1}$

3. **正规性对应**：$H \triangleleft G$ 当且仅当 $E^H/F$ 是Galois扩张

**应用**：

- **结构分析**：通过对应关系分析结构
- **问题解决**：解决可解性问题

## 五、方法的步骤

### 5.1 基本步骤

方程理论方法的基本步骤：

1. **建立多项式**：给定域 $F$ 上的多项式 $f(x)$
2. **构造分裂域**：构造 $f(x)$ 在 $F$ 上的分裂域 $E$
3. **计算Galois群**：计算 $\text{Gal}(E/F)$
4. **分析群结构**：分析Galois群的结构
5. **判断可解性**：判断Galois群是否是可解群
6. **应用判据**：应用可解性判据
7. **得出结论**：判断方程是否可用根式求解

### 5.2 具体操作

**计算Galois群**：

- **置换方法**：通过根的置换计算Galois群
- **不变量方法**：使用不变量区分Galois群
- **数值方法**：使用数值方法计算Galois群

**分析群结构**：

- **子群分析**：分析Galois群的子群
- **商群分析**：分析Galois群的商群
- **可解性分析**：分析Galois群的可解性

### 5.3 应用示例

**例子：二次方程**：

1. **多项式**：$f(x) = x^2 + bx + c \in F[x]$
2. **分裂域**：$E = F(\sqrt{b^2 - 4ac})$
3. **Galois群**：$\text{Gal}(E/F) \cong \mathbb{Z}_2$（可解群）
4. **结论**：二次方程可用根式求解

**例子：一般五次方程**：

1. **多项式**：$f(x) = x^5 + ax^4 + bx^3 + cx^2 + dx + e \in F[x]$
2. **分裂域**：$E$ 是 $f(x)$ 在 $F$ 上的分裂域
3. **Galois群**：$\text{Gal}(E/F) \cong S_5$（不是可解群）
4. **结论**：一般五次方程不可用根式求解
