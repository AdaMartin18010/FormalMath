## 📚 核心内容方向

### 1. 群论

- **历史背景**: 19世纪初群的概念的建立
- **数学内容**: 置换群、群的结构、可解群
- **应用**: 代数、几何、物理

### 2. 伽罗瓦理论

- **核心思想**: 代数方程与群的对应
- **方法**: 域扩张、伽罗瓦群
- **影响**: 现代代数学的基础

### 3. 可解性理论

- **贡献**: 代数方程可解性的判据
- **方法**: 可解群、根式解
- **应用**: 代数方程、数论
