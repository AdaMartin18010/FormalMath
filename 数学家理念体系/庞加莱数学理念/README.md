# 庞加莱数学理念体系

## 项目概述

本项目系统化分析**亨利·庞加莱（Henri Poincaré, 1854-1912）**的数学理念、方法论和具体数学贡献。

**核心理念关键词**：约定论、拓扑学、混沌理论、自守函数、相对论先声

## 🎯 项目特色

1. **约定论哲学**：数学真理的约定性本质
2. **拓扑学奠基**：同调论、基本群、Poincaré猜想
3. **动力系统**：混沌理论、周期轨道、稳定性
4. **自守函数**：Fuchs群、模函数、单值化
5. **批判性中介**：连接形式主义与直觉主义
