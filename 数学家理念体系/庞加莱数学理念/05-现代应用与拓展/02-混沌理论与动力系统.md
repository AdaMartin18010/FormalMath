## 一、庞加莱的定性理论

### 1.1 定性理论的开创

**庞加莱的贡献**（1880s-1890s）：

庞加莱在《天体力学新方法》中开创了动力系统的定性理论。

**核心思想**：

```
定性理论：
- 不求解方程
- 研究解的全局性质
- 相空间方法
```

**关键概念**：

- 相空间
- 轨道
- 稳定性
- 周期解

### 1.2 同宿交错

**同宿交错**（1890s）：

```
同宿轨道：
- 稳定流形与不稳定流形相交
- 复杂动力学
- 混沌先声
```

**意义**：

- 预见混沌
- 现代动力系统
- 应用广泛

## 二、混沌理论的诞生

### 2.1 Lorenz的发现

**Lorenz吸引子**（1963）：

```
Lorenz系统：
ẋ = σ(y - x)
ẏ = x(ρ - z) - y
ż = xy - βz

性质：
- 奇怪吸引子
- 混沌行为
- 应用广泛
```

**与庞加莱的关系**：

- 定性理论
- 混沌先声
- 现代发展

**现代发展**：

