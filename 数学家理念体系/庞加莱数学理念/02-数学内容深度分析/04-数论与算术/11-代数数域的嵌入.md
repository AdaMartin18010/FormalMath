# 代数数域的嵌入：数域的几何观点

## 一、嵌入

### 1.1 定义

**数域的嵌入**：

```
数域K到复数域的嵌入：

σ: K → ℂ

分类：
- 实嵌入：σ(K) ⊆ ℝ
- 复嵌入：σ(K) ⊆ ℂ \ ℝ（成对出现）

性质：
- 几何观点
- 应用广泛
```

---

### 1.2 嵌入数

**嵌入数**：

- r₁：实嵌入数
- r₂：复嵌入对数
- n = r₁ + 2r₂ = [K : ℚ]

数域K的Minkowski嵌入：

K → ℝ^{r₁} × ℂ^{r₂}

性质：
- 几何表示
- 应用广泛
