# 算术函数：庞加莱的数论工作

## 一、算术函数

### 1.1 定义

**算术函数**：

```
定义在正整数上的函数：

f: ℕ → ℂ

例子：
- 除数函数
- Euler函数
- Möbius函数
- 应用广泛
```

---

### 1.2 生成函数

**生成函数**：

```
算术函数的Dirichlet生成函数：

F(s) = ∑_{n≥1} f(n) n^{-s}

性质：
- 解析性质
- 数论应用
- 应用广泛
```
