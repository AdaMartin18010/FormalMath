# 模形式与L函数：数论的核心工具

## 一、模形式

### 1.1 定义

**模形式**：

```
权重k的模形式：

f(γz) = (cz+d)^k f(z)

其中：
- γ ∈ SL(2,ℤ)
- 全纯函数
- 应用广泛
```

---

### 1.2 性质

**性质**：

- 模变换不变
- 数论应用
- 应用广泛

## 二、L函数

### 2.1 定义

**L函数**：

```
模形式的L函数：

L(s, f) = ∑_{n≥1} a_n n^{-s}

其中：
- a_n：傅里叶系数
- 解析延拓
- 应用广泛
```

---

### 2.2 应用

**应用**：

- 数论
- 代数几何
- 现代研究
