# 流体力学：偏微分方程的应用

## 一、Navier-Stokes方程

### 1.1 方程

**Navier-Stokes方程**：

```
流体运动方程：

∂u/∂t + (u·∇)u = -∇p + νΔu + f

其中：
- u：速度场
- p：压力
- ν：粘性

性质：
- 非线性PDE
- 应用广泛
```
