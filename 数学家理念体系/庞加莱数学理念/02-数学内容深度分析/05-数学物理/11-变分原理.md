# 变分原理：数学物理的统一方法

## 一、变分问题

### 1.1 定义

**变分问题**：

```
泛函J[y] = ∫_a^b F(x, y, y') dx

极值问题：
δJ = 0

性质：
- 统一方法
- 应用广泛
```

---

### 1.2 Euler-Lagrange方程

**Euler-Lagrange方程**：

```
变分问题的解满足：

∂F/∂y - d/dx(∂F/∂y') = 0

性质：
- 必要条件
- 应用广泛
```
