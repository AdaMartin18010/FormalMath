# 相对论原理：Lorentz不变性

## 一、Lorentz群

### 1.1 定义

**Lorentz群**：

```
保持Minkowski度规的线性变换：

x' = Λx

其中：
- Λ：Lorentz变换
- 度规不变
- 应用广泛
```

**性质**：

- 群结构
