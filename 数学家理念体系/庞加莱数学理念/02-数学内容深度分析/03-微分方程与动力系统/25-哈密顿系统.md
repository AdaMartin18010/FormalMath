# 哈密顿系统：保守动力系统

## 一、哈密顿系统

### 1.1 定义

**哈密顿系统**：

```
哈密顿方程：

dq/dt = ∂H/∂p
dp/dt = -∂H/∂q

其中：
- H：哈密顿函数
- (q,p)：相空间坐标

性质：
- 保守系统
- 应用广泛
```
