# KAM理论：可积系统的微扰理论

## 一、KAM理论

### 1.1 定理

**Kolmogorov-Arnold-Moser定理**：

```
接近可积系统：

大部分不变环面保持

条件：
- 充分小的扰动
- 非退化
- 应用广泛
```

**意义**：

- 准周期轨道
