# 稳定性理论：动力系统的定性分析

## 一、稳定性

### 1.1 定义

**稳定性**：

```
动力系统：
dx/dt = f(x)

平衡点x*：
- 稳定：小扰动不改变
- 不稳定：小扰动发散
- 应用广泛
```
---

### 1.2 分类

**分类**：

- 稳定结点
- 不稳定结点
- 鞍点
- 焦点
- 中心

---

## 二、Lyapunov方法

### 2.1 Lyapunov函数

**方法**：

```
Lyapunov函数V(x)：
- V(x*) = 0
- V(x) > 0（x ≠ x*）
- dV/dt < 0

意义：
- 稳定性判据
- 应用广泛
```
---

### 2.2 应用

**应用**：

- 控制系统
- 物理系统
- 应用广泛
