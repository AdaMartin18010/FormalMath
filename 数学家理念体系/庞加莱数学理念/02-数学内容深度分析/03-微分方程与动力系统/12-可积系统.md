## 一、可积系统

### 1.1 定义

**n自由度Hamilton系统**：

```
如果有n个独立、对合的首次积分，则系统可积

条件：
- 独立：线性无关
- 对合：{H_i, H_j} = 0
- 应用广泛
```

**历史**：

- Liouville（1855）
- Arnold（1960s）
- 现代发展

### 1.2 例子

**例子**：

```
例子：
- 2体问题
- 谐振子
- 应用广泛
```

**应用**：

- 天体力学
- 统计力学
- 现代应用
