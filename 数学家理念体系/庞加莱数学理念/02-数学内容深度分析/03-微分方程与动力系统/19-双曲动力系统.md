## 一、双曲性

### 1.1 定义

**双曲动力系统**：

```
动力系统(X, f)的双曲性：

对每个点x ∈ X，切空间分解为：
T_xX = E^s_x ⊕ E^u_x

其中：
- E^s：稳定方向（收缩）
- E^u：不稳定方向（扩张）

性质：
- 结构稳定
- 混沌基础
- 应用广泛
```

---

### 1.2 性质

**性质**：

- 结构稳定
- 遍历性
- 混沌性

## 二、稳定流形

### 2.1 定义

**稳定流形**：

```
点x的稳定流形：

W^s(x) = {y : d(f^n(x), f^n(y)) → 0 (n→∞)}

性质：
- 不变流形
- 局部结构
- 应用广泛
```

---

### 2.2 不稳定流形

**不稳定流形**：

```
点x的不稳定流形：

W^u(x) = {y : d(f^{-n}(x), f^{-n}(y)) → 0 (n→∞)}

性质：
- 不变流形
- 局部结构
- 应用广泛
```
