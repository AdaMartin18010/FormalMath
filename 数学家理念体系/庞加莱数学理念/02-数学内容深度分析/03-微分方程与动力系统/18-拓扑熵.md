# 拓扑熵：动力系统复杂度的度量

## 一、拓扑熵

### 1.1 定义

**拓扑熵**：

```
动力系统(X, f)的拓扑熵：

h_top(f) = lim_{n→∞} (1/n) log N(n)

其中：
- N(n)：n步轨道数
- 复杂度度量

性质：
- 拓扑不变量
- 系统复杂度
- 应用广泛
```
