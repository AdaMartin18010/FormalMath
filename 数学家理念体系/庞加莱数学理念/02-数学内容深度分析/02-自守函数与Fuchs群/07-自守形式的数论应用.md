# 自守形式的数论应用：Langlands纲领的基础

## 一、自守形式

### 1.1 定义

**自守形式**：

```
群作用下的不变函数：

f(γz) = j(γ,z)^k f(z)

其中：
- γ ∈ 离散群
- j：因子
- 数论应用
```
---

### 1.2 性质

**性质**：

- 模变换不变

### 2.1 定义

**L函数**：

```
自守形式的L函数：

L(s, f) = ∑_{n≥1} a_n n^{-s}

其中：
- a_n：傅里叶系数

- 应用广泛
```
