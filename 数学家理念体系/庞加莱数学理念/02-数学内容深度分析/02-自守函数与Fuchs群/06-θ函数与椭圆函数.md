# θ函数与椭圆函数：自守函数的特例

## 一、θ函数

### 1.1 定义

**θ函数**：

```
定义在上半平面：

θ(z, τ) = ∑_{n=-∞}^∞ e^{πi n²τ + 2πi n z}

其中：
- τ：模参数
- 准周期
- 应用广泛
```

---

### 1.2 性质

**性质**：

- 准周期
- 模变换
- 应用广泛

---

## 二、椭圆函数

### 2.1 定义

**椭圆函数**：

```
双周期函数：

f(z + mω₁ + nω₂) = f(z)

其中：
- ω₁, ω₂：周期
- 椭圆曲线
- 应用广泛
```

---

### 2.2 与θ函数的关系

**关系**：

```
θ函数
    ↓
椭圆函数
    ↓
自守函数
```
