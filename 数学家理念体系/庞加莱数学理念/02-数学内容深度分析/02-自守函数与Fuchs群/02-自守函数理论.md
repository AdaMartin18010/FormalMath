# 自守函数理论：群作用下的不变函数

## 一、自守函数

### 1.1 定义

**自守函数**：

```
函数f(z)在群G作用下不变：

f(γz) = f(z)（对所有γ ∈ G）

其中：
- G：Fuchs群或Kleinian群
- 应用广泛
```

---

### 1.2 性质

**性质**：

- 群不变性
- 单值化
- 应用广泛
