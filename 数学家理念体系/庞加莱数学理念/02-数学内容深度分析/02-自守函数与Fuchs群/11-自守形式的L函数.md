## 一、L函数

### 1.1 定义

**自守形式的L函数**：

设 $f$ 是自守形式，其Fourier展开为：
$$f(z) = \sum_{n=1}^\infty a_n e^{2\pi i n z}$$

**L函数**定义为：
$$L(s, f) = \sum_{n=1}^\infty \frac{a_n}{n^s}$$

其中 $s \in \mathbb{C}$，$\text{Re}(s) > 1$。

**数学表述**：

L函数：
$$L(s, f) = \sum_{n=1}^\infty \frac{a_n}{n^s} = \prod_{p} \left(1 - \frac{a_p}{p^s} + \frac{\chi(p)}{p^{2s}}\right)^{-1}$$

其中 $\chi$ 是特征，$p$ 遍历素数。

**解析延拓**：

L函数可以解析延拓到整个复平面（除了可能的极点）。

**例子1：Riemann ζ函数**：

Riemann ζ函数是自守形式L函数的特例：
$$\zeta(s) = \sum_{n=1}^\infty \frac{1}{n^s} = \prod_{p} \left(1 - \frac{1}{p^s}\right)^{-1}$$

**例子2：模形式的L函数**：

对于权为 $k$ 的模形式 $f$，其L函数为：
$$L(s, f) = \sum_{n=1}^\infty \frac{a_n}{n^s}$$

可以解析延拓到整个复平面。

### 1.2 函数方程

**函数方程**：

L函数满足函数方程：
$$L(s, f) = \varepsilon(s) L(1-s, \tilde{f})$$

其中 $\tilde{f}$ 是对偶形式，$\varepsilon(s)$ 是epsilon因子。

**数学表述**：

完整的函数方程：
$$\Lambda(s, f) = \varepsilon(s) \Lambda(1-s, \tilde{f})$$

其中：
$$\Lambda(s, f) = \left(\frac{\sqrt{N}}{2\pi}\right)^s \Gamma(s) L(s, f)$$

$\varepsilon(s)$ 的形式：
$$\varepsilon(s) = \varepsilon \cdot N^{1/2-s}$$

其中 $\varepsilon$ 是根数（root number），$|\varepsilon| = 1$。

**例子3：Riemann ζ函数的函数方程**：

$$\Lambda(s) = \pi^{-s/2} \Gamma(s/2) \zeta(s) = \Lambda(1-s)$$

这给出了：
$$\zeta(s) = 2^s \pi^{s-1} \sin\left(\frac{\pi s}{2}\right) \Gamma(1-s) \zeta(1-s)$$

**对称性**：

函数方程表明L函数在 $s \leftrightarrow 1-s$ 下对称（乘以epsilon因子）。

**数学表述**：

函数方程的对称性：
$$L(s, f) = \varepsilon(s) L(1-s, \tilde{f})$$

如果 $\varepsilon(s) = 1$，则：
$$L(s, f) = L(1-s, \tilde{f})$$

**零点研究**：

函数方程帮助我们研究L函数的零点分布。

**Riemann假设**（对于Riemann ζ函数）：

所有非平凡零点都在直线 $\text{Re}(s) = 1/2$ 上。

**数学表述**：

Riemann假设：
$$\zeta(s) = 0 \land 0 < \text{Re}(s) < 1 \Rightarrow \text{Re}(s) = \frac{1}{2}$$
