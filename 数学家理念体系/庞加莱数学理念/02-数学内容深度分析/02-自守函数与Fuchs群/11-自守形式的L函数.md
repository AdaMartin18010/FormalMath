## 一、L函数

### 1.1 定义

**自守形式的L函数**：

```
自守形式f的L函数：

L(s, f) = ∑_{n=1}^∞ a_n n^{-s}

其中：
- a_n：Fourier系数
- Re(s) > 1

性质：
- 解析延拓
- 函数方程
- 应用广泛
```

---

### 1.2 函数方程

**函数方程**：

```
L(s, f) = ε(s) L(1-s, \tilde{f})

其中：
- \tilde{f}：对偶形式
- ε(s)：epsilon因子

性质：
- 对称性
- 零点研究
- 应用广泛
```
