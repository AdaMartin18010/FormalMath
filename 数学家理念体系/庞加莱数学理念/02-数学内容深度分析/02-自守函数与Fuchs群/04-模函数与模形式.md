# 模函数与模形式：自守函数的数论应用

## 一、模函数

### 1.1 定义

**模函数**：

```
SL(2,ℤ)作用下的自守函数：

f(γz) = f(z)（对所有γ ∈ SL(2,ℤ)）

其中：
- SL(2,ℤ)：模群
- 上半平面
- 应用广泛
```

---

### 1.2 性质

**性质**：

- 模变换不变
- 数论应用
- 应用广泛

---

## 二、模形式

### 2.1 定义

**模形式**：

```
权重k的模形式：

f(γz) = (cz+d)^k f(z)

其中：
- γ ∈ SL(2,ℤ)
- 全纯函数
- 应用广泛
```

---

### 2.2 应用

**应用**：

- 数论
- 代数几何
- 现代数学
