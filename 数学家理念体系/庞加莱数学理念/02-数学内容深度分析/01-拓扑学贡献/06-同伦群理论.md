## 一、同伦群

### 1.1 定义

**n维同伦群π_n(X)**：

```
拓扑空间X的n维同伦群：

π_n(X, x₀) = {n维球面到X的映射}/同伦等价

其中：
- 映射保持基点
- 同伦：连续变形
- 群运算：球面粘合

性质：
- 拓扑不变量
- 高阶"孔洞"
- 应用广泛
```

---

### 1.2 基本群的关系

**关系**：

```
π₁(X) = 基本群（已讨论）

π_n(X)（n ≥ 2）：
- 交换群（Hurewicz定理）
- 高阶同伦群
- 应用广泛
```

---

## 二、Hurewicz定理

### 2.1 定理

**Hurewicz定理**：

```
如果π_i(X) = 0（i < n），则：

π_n(X) ≅ H_n(X)

性质：
- 同伦与同调的联系
- 拓扑不变量
- 应用广泛
```

---

### 2.2 应用

**应用**：

- 流形分类
- 几何分析
- 现代发展

## 三、Whitehead定理

### 3.1 定理

**Whitehead定理**：

```
映射f: X → Y

如果f*: π_n(X) → π_n(Y)是同构（所有n），
则f是同伦等价

性质：
- 同伦分类
- 应用广泛
```

## 九、数学公式总结

### 核心公式

1. **同伦群定义**：
   $$\pi_n(X, x_0) = [S^n, X]_*$$

2. **基本群**：
   $$\pi_1(X, x_0) = \text{Hom}(\pi_1(X, x_0), \mathbb{Z})$$

3. **高阶同伦群**：
   $$\pi_n(X, x_0) = [S^n, X]_* / \text{同伦}$$

4. **同伦群的性质**：
   $$\pi_n(X \times Y) \cong \pi_n(X) \times \pi_n(Y)$$

5. **Hurewicz定理**：
   $$\pi_n(X) \cong H_n(X), \quad X \text{ 单连通}, n \\\geq 2$$

6. **纤维丛的长正合序列**：
   $$\cdots \to \pi_n(F) \to \pi_n(E) \to \pi_n(B) \to \pi_{n-1}(F) \to \cdots$$

7. **Whitehead定理**：
   $$f_*: \pi_n(X) \to \pi_n(Y) \text{ 同构} \Rightarrow f \text{ 同伦等价}$$

8. **稳定同伦群**：
   $$\pi_n^s = \lim_{k \to \infty} \pi_{n+k}(S^k)$$

9. **Freudenthal悬垂定理**：
   $$\pi_{n+1}(S^{n+1}) \cong \pi_n(S^n)$$

10. **同伦群的计算**：
    $$\pi_k(S^n) = 0, \quad k < n; \quad \pi_n(S^n) \cong \mathbb{Z}$$

## 十、同伦群的详细理论

### 10.1 同伦群的严格定义

**定义**：
- $n$维同伦群$\pi_n(X, x_0)$是$n$维球面$S^n$到$X$的基点映射的同伦类
- 对于$n \\\geq 2$，同伦群是交换群
- 基本群$\pi_1(X)$一般不是交换群

**数学公式**：
- 同伦群：$$\pi_n(X, x_0) = [S^n, X]_* = \{\text{基点映射 } f: S^n \to X\}/\text{同伦}$$
- 群运算：$$(f + g)(x) = \begin{cases} f(2x) & x \in S^n_+ \\ g(2x) & x \in S^n_- \end{cases}$$

---

### 10.2 同伦群的性质

**性质**：
- $\pi_n(X \times Y) \cong \pi_n(X) \times \pi_n(Y)$
- $\pi_n(X)$是拓扑不变量
- 对于$n \\\geq 2$，$\pi_n(X)$是交换群

**数学公式**：
- 乘积：$$\pi_n(X \times Y) \cong \pi_n(X) \times \pi_n(Y)$$
- 交换性：$$\pi_n(X) \text{ 是交换群 } (n \\\geq 2)$$

## 十一、Hurewicz定理的详细内容

### 11.1 Hurewicz定理

**定理**（Hurewicz）：
- 如果$\pi_i(X) = 0$对所有$i < n$，则$\pi_n(X) \cong H_n(X)$
- 这建立了同伦群与同调群的联系
- 是代数拓扑的基础定理

**数学公式**：
- Hurewicz映射：$$h: \pi_n(X) \to H_n(X)$$
- Hurewicz定理：$$\pi_n(X) \cong H_n(X) \text{（若 } \pi_i(X) = 0 \text{ 对所有 } i < n \text{）}$$

---

### 11.2 应用

**应用**：
- 计算同伦群
- 流形分类
- 拓扑应用

**数学公式**：
- 单连通空间：$$\pi_1(X) = 0 \Rightarrow \pi_2(X) \cong H_2(X)$$

---

## 十二、Whitehead定理的详细内容

### 12.1 Whitehead定理

**定理**（Whitehead）：
- 如果映射$f: X \to Y$诱导同伦群同构$f_*: \pi_n(X) \to \pi_n(Y)$对所有$n$
- 则$f$是同伦等价
- 这是同伦分类的基础

**数学公式**：
- Whitehead定理：$$f_*: \pi_n(X) \to \pi_n(Y) \text{ 同构 } \forall n \Rightarrow f \text{ 是同伦等价}$$
- 同伦等价：$$X \simeq Y \iff \exists f: X \to Y, g: Y \to X, f \circ g \simeq \text{id}_Y, g \circ f \simeq \text{id}_X$$

---

### 12.2 应用

**应用**：
- 同伦分类
- 流形分类
- 拓扑应用

## 十三、稳定同伦论

### 13.1 稳定同伦群

**定义**：
- 稳定同伦群$\pi_n^s = \lim_{k \to \infty} \pi_{n+k}(S^k)$
- 稳定同伦群比同伦群更容易计算
- 是现代同伦论的基础

**数学公式**：
- 稳定同伦群：$$\pi_n^s = \lim_{k \to \infty} \pi_{n+k}(S^k)$$
- Freudenthal悬垂：$$\pi_{n+1}(S^{n+1}) \cong \pi_n(S^n)$$

---

### 13.2 应用

**应用**：
- 同伦论
- 代数拓扑
- 现代数学

**数学公式**：
- 稳定化：$$\pi_n^s(S^0) = \lim_{k \to \infty} \pi_{n+k}(S^k)$$
