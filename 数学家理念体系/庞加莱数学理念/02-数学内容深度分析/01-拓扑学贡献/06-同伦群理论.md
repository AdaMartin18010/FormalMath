# 同伦群理论：高阶拓扑不变量

## 一、同伦群

### 1.1 定义

**n维同伦群π_n(X)**：

```
拓扑空间X的n维同伦群：

π_n(X, x₀) = {n维球面到X的映射}/同伦等价

其中：
- 映射保持基点
- 同伦：连续变形
- 群运算：球面粘合

性质：
- 拓扑不变量
- 高阶"孔洞"
- 应用广泛
```
---

### 1.2 基本群的关系

**关系**：

```
π₁(X) = 基本群（已讨论）

π_n(X)（n ≥ 2）：
- 交换群（Hurewicz定理）
- 高阶同伦群
- 应用广泛
```
---

## 二、Hurewicz定理

### 2.1 定理

**Hurewicz定理**：

```
如果π_i(X) = 0（i < n），则：

π_n(X) ≅ H_n(X)

性质：
- 同伦与同调的联系
- 拓扑不变量
- 应用广泛
```
---

### 2.2 应用

**应用**：

- 流形分类
- 几何分析
- 现代发展

---

## 三、Whitehead定理

### 3.1 定理

**Whitehead定理**：

```
映射f: X → Y

如果f*: π_n(X) → π_n(Y)是同构（所有n），
则f是同伦等价

性质：
- 同伦分类
- 应用广泛
```
---

### 3.2 应用

**应用**：

- 同伦分类
- 流形分类
- 现代发展
