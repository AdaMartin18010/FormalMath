# 指标定理：拓扑与分析的统一

## 一、指标定理

### 1.1 Atiyah-Singer指标定理

**Atiyah-Singer指标定理**：

```
椭圆算子D: Γ(E) → Γ(F)的指标：

ind(D) = ∫_M ch(E - F) ∧ Td(M)

其中：
- ch：Chern特征
- Td：Todd类
- M：底流形

性质：
- 拓扑与分析统一
- 应用广泛
```

---

### 1.2 意义

**意义**：

- 拓扑不变量 = 分析不变量
- 统一框架
- 应用广泛
