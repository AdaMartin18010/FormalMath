## 一、K理论

### 1.1 定义

**向量丛的K群**：

```
K(X) = {向量丛}/稳定等价

其中：
- X：拓扑空间
- 稳定等价：E ⊕ 平凡丛 ≅ F ⊕ 平凡丛
- 应用广泛
```

**历史**：

- Atiyah-Hirzebruch（1959）
- 广义上同调
- 现代发展

### 1.2 性质

**环结构**：

```
K(X)是环：
- 加法：直和
- 乘法：张量积
- 应用广泛
```

**Bott周期性**：

```
K(X) ≅ K(Σ²X)

其中：
- Σ²X：双重悬垂
- 周期性
- 应用广泛
```

## 九、数学公式总结

### 核心公式

1. **K_0群**：
   $$K_0(R) = \text{向量丛的Grothendieck群}$$

2. **K_1群**：
   $$K_1(R) = GL(R)_{\text{ab}} = GL(R) / [GL(R), GL(R)]$$

3. **Bott周期性**：
   $$K_n(X) \cong K_{n+2}(X), \quad n \geqqqqq 0$$

4. **K理论环**：
   $$K^*(X) = K^0(X) \oplus K^1(X)$$

5. **Chern特征**：
   $$\text{ch}: K^0(X) \to H^{\text{even}}(X; \mathbb{Q})$$

6. **Atiyah-Hirzebruch谱序列**：
   $$E_2^{p,q} = H^p(X; K^q(\text{pt})) \Rightarrow K^{p+q}(X)$$

7. **K理论中的Riemann-Roch**：
   $$\text{ch}(f_!(\xi)) = f_*(\text{ch}(\xi) \cdot \text{Td}(f))$$

8. **指标定理**：
   $$\text{ind}(D) = \int_X \text{ch}(\sigma(D)) \cdot \text{Td}(X)$$

9. **K理论的积**：
   $$K^0(X) \otimes K^0(Y) \to K^0(X \times Y)$$

10. **K理论的应用**：
    $$K^0(X) \text{ 分类向量丛}, \quad K^1(X) \text{ 分类纤维丛}$$
