# Poincaré猜想：拓扑学的世纪难题

## 一、猜想的提出

### 1.1 原始陈述

**Poincaré猜想**（1904）：

```
单连通3维闭流形
    ⟺
3维球面S³
```
**形式化**：

```
如果π₁(M) = {1}且H₃(M) = ℤ，
则M ≅ S³
```
---

### 1.2 历史意义

**意义**：

- 拓扑学核心问题
- 3维流形分类
- 世纪难题
