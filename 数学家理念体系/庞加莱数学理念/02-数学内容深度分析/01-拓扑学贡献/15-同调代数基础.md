# 同调代数基础：抽象同调理论

## 一、链复形

### 1.1 定义

**链复形**：

```
... → C_{n+1} → C_n → C_{n-1} → ...

性质：
- 边界映射
- d² = 0
- 应用广泛
```

---

### 1.2 同调

**同调**：

```
H_n = Ker(d_n) / Im(d_{n+1})

性质：
- 同调群
- 拓扑不变量
- 应用广泛
```

---

## 二、导函子

### 2.1 定义

**导函子**：

```
函子F的导函子：

R^nF(X) = H^n(F(I*))

其中：
- I*：内射分解
- 同调代数
- 应用广泛
```

---

### 2.2 应用

**应用**：

- 层上同调
- 同调代数
- 现代发展
