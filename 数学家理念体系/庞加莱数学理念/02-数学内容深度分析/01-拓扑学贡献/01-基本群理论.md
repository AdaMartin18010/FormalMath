## 一、基本群的定义

### 1.1 定义

**基本群π₁(X)**：

```
拓扑空间X的基本群：

π₁(X, x₀) = {闭路径}/同伦等价

其中：
- 闭路径：从x₀到x₀的连续映射
- 同伦：路径的连续变形
- 群运算：路径复合

性质：
- 拓扑不变量
- 捕捉"孔洞"结构
- 应用广泛
```

---

### 1.2 例子

**基本例子**：

```
S¹（圆）：
π₁(S¹) = ℤ

S²（球面）：
π₁(S²) = {1}（平凡群）

T²（环面）：
π₁(T²) = ℤ×ℤ

ℝP²（射影平面）：
π₁(ℝP²) = ℤ/2ℤ
```

## 九、数学公式总结

### 核心公式

1. **基本群定义**：
   $$\pi_1(X, x_0) = \{[f] : f: [0,1] \to X, f(0) = f(1) = x_0\}$$

2. **路径复合**：
   $$(f * g)(t) = \begin{cases} f(2t) & 0 \leq t \leq 1/2 \\ g(2t-1) & 1/2 \leq t \leq 1 \end{cases}$$

3. **同伦等价**：
   $$f \simeq g \iff \exists H: [0,1] \times [0,1] \to X, H(0,t) = f(t), H(1,t) = g(t)$$

4. **基本群例子**：
   - $\pi_1(S^1) = \mathbb{Z}$
   - $\pi_1(S^n) = \{1\}$（$n \geq 2$）
   - $\pi_1(T^2) = \mathbb{Z} \times \mathbb{Z}$

5. **Seifert-van Kampen定理**：
   $$\pi_1(X, x_0) = \pi_1(U, x_0) *_{\pi_1(U \cap V, x_0)} \pi_1(V, x_0)$$

6. **覆盖空间对应**：
   $$\text{覆盖空间} \leftrightarrow \text{子群} \subset \pi_1(X)$$

7. **万有覆盖**：
   $$\tilde{X} = \{\text{从 } x_0 \text{ 出发的路径类}\}$$

8. **基本群同态**：
   $$f_*: \pi_1(X, x_0) \to \pi_1(Y, f(x_0))$$

9. **Hurewicz定理**：
   $$\pi_1(X)_{\text{ab}} \cong H_1(X)$$

10. **基本群与同调群**：
    $$\pi_1(X) \to H_1(X) \text{（Abel化）}$$
