# 上同调理论：同调的代数对偶

## 一、上同调群

### 1.1 定义

**上同调群H^n(X)**：

```
拓扑空间X的上同调群：

H^n(X) = {上闭链} / {上边界链}

其中：
- 上闭链：n维上闭链
- 上边界链：n维上边界链

性质：
- 拓扑不变量
- 代数对偶
- 应用广泛
```

---

```
H^n(X) 与 H_n(X) 对偶

性质：
```
