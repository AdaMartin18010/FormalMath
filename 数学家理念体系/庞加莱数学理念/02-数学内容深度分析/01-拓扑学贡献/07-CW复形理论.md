# CW复形理论：拓扑空间的胞腔分解

## 一、CW复形

### 1.1 定义

**CW复形**：

```
拓扑空间X的胞腔分解：

X = ∪_{n≥0} X^n

其中：
- X^0：0维胞腔（点）
- X^n：n维胞腔（通过n维圆盘粘合）

性质：
- 逐步构造
- 可计算
- 应用广泛
```
