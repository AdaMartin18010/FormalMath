# 同调理论：拓扑的代数工具

## 一、同调群

### 1.1 定义

**同调群H_n(X)**：

```
拓扑空间X的同调群：

H_n(X) = Z_n / B_n

其中：
- Z_n：n维闭链
- B_n：n维边界链

性质：
- 拓扑不变量
- 代数化拓扑
- 应用广泛
```

---

### 1.2 计算

**计算方法**：

- 单纯复形
- CW复形
- 应用广泛

---

## 二、Poincaré对偶

### 2.1 对偶定理

**Poincaré对偶**：

```
n维紧致流形M：

H^k(M) ≅ H_{n-k}(M)

性质：
- 深刻对称性
- 拓扑不变量
- 应用广泛
```

---

### 2.2 应用

**应用**：

- 流形分类
- 几何分析
- 现代发展
