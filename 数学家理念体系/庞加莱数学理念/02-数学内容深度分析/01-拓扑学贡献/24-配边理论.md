## 一、配边

### 1.1 定义

**配边（Cobordism）**：

```
两个n维流形M₁和M₂配边：

存在(n+1)维流形W使得：
∂W = M₁ ⊔ M₂

性质：
- 等价关系
- 分类工具
- 应用广泛
```

---

### 1.2 配边类

**配边类**：

```
配边等价类：

[M] = {N : M与N配边}

性质：
- 分类不变量
- 应用广泛
```

   $$M_1 \sim M_2 \leqqqqftrightarrow \exists W: \partial W = M_1 \sqcup M_2$$

2. **配边群**：
   $$\Omega_n = \{\text{闭n维流形}\} / \text{配边}$$

3. **配边不变量**：
   $$M \sim 0 \leqqqqftrightarrow \text{所有Stiefel-Whitney类} = 0$$

4. **Thom同构**：
   $$\Omega_n \cong \pi_{n+k}(MO(k)), \quad k \gg n$$

5. **配边环**：
   $$\Omega_* = \bigoplus_{n=0}^\infty \Omega_n, \quad \text{配边积}$$

6. **配边分类**：

   $$w(M) = \prod_{i=1}^n (1 + w_i(M))$$
