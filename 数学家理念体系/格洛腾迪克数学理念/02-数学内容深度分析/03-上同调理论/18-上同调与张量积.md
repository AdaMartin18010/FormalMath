## 九、数学公式总结

### 核心公式

1. **张量积定义**：
   $$(\mathcal{F} \otimes_{\mathcal{O}_X} \mathcal{G})(U) = \mathcal{F}(U) \otimes_{\mathcal{O}_X(U)} \mathcal{G}(U)$$

2. **导出张量积**：
   $$\mathcal{F} \otimes_{\mathcal{O}_X}^{\mathbb{L}} \mathcal{G} = \mathcal{F} \otimes_{\mathcal{O}_X} P_\mathcal{G}^\bullet$$

3. **Künneth公式**：
   $$H^n(X \times Y, \mathcal{F} \boxtimes \mathcal{G}) \cong \bigoplus_{p+q=n} H^p(X, \mathcal{F}) \otimes H^q(Y, \mathcal{G}) \text{（某些条件下）}$$

4. **Tor函子**：
   $$\text{Tor}_i^{\mathcal{O}_X}(\mathcal{F}, \mathcal{G}) = H^{-i}(\mathcal{F} \otimes_{\mathcal{O}_X}^{\mathbb{L}} \mathcal{G})$$

5. **张量积上同调**：
   $$H^i(X, \mathcal{F} \otimes \mathcal{G}) \cong \bigoplus_{p+q=i} H^p(X, \mathcal{F}) \otimes H^q(X, \mathcal{G}) \text{（某些条件下）}$$

6. **投影公式**：
   $$Rf_*(\mathcal{F} \otimes f^*\mathcal{G}) \cong Rf_*\mathcal{F} \otimes \mathcal{G}$$

7. **张量积与拉回**：
   $$f^*(\mathcal{F} \otimes \mathcal{G}) \cong f^*\mathcal{F} \otimes f^*\mathcal{G}$$

8. **张量积与推前**：
   $$f_*(\mathcal{F} \otimes \mathcal{G}) \cong f_*\mathcal{F} \otimes f_*\mathcal{G} \text{（某些条件下）}$$

9. **张量积与Ext**：
   $$\text{Ext}^i(\mathcal{F} \otimes \mathcal{G}, \mathcal{H}) \cong \text{Ext}^i(\mathcal{F}, \mathcal{H}om(\mathcal{G}, \mathcal{H}))$$

10. **平坦性与张量积**：
    $$\mathcal{F} \text{ 平坦 } \Rightarrow \text{Tor}_i(\mathcal{F}, \mathcal{G}) = 0 \text{（$i > 0$）}$$
