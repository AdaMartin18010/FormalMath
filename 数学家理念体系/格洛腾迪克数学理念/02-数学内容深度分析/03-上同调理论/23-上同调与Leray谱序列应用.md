## 八、数学公式总结

### 核心公式

1. **Leray谱序列**：
   $$E_2^{p,q} = H^p(Y, R^q f_* \mathcal{F}) \Rightarrow H^{p+q}(X, \mathcal{F})$$

2. **谱序列微分**：
   $$d_r: E_r^{p,q} \to E_r^{p+r,q-r+1}, \quad d_r^2 = 0$$

3. **收敛**：
   $$E_\infty^{p,q} = F^p H^{p+q} / F^{p+1} H^{p+q}$$

4. **边缘项**：
   $$E_2^{p,0} = H^p(Y, f_*\mathcal{F}), \quad E_2^{0,q} = H^0(Y, R^q f_*\mathcal{F})$$

5. **退化条件**：
   $$\text{若 } R^q f_*\mathcal{F} = 0 \text{（$q > 0$），则 } H^n(X, \mathcal{F}) \cong H^n(Y, f_*\mathcal{F})$$

6. **五项序列**：
   $$0 \to H^1(Y, f_*\mathcal{F}) \to H^1(X, \mathcal{F}) \to H^0(Y, R^1 f_*\mathcal{F}) \to H^2(Y, f_*\mathcal{F}) \to H^2(X, \mathcal{F})$$

7. **应用（纤维丛）**：
   $$E_2^{p,q} = H^p(B, H^q(F)) \Rightarrow H^{p+q}(E)$$

8. **应用（覆盖）**：
   $$E_2^{p,q} = H^p(G, H^q(\tilde{X}, \mathcal{F})) \Rightarrow H^{p+q}(X, \mathcal{F})$$

9. **函子性**：
   $$\text{Leray谱序列与态射、基变化等操作相容}$$

10. **计算应用**：
