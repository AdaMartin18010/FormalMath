## 二、平展基本群

### 2.1 定义

**平展基本群π_1^{ét}(X, x)**：

```
连通概形X
几何点x

平展基本群：
π_1^{ét}(X, x) = Aut(F_x)

其中F_x是纤维函子：
F_x: FEt(X) → Set
F_x(Y) = Hom_X(x, Y)
```
---

### 2.2 性质

**平展基本群的性质**：

```
性质：
- 射有限群
- 几何不变量
- 函子性
- 应用广泛
```
