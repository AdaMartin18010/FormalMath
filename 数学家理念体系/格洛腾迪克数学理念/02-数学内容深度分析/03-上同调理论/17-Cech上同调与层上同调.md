# Cech上同调与层上同调：计算的两种方法

## 一、Cech上同调

### 1.1 定义

**Cech上同调Ȟ^i(U, F)**：

```
概形X
开覆盖U = {U_i}
Abel层F

Cech复形：
C^p(U, F) = ∏_{i_0<...<i_p} F(U_{i_0...i_p})

其中U_{i_0...i_p} = U_{i_0} ∩ ... ∩ U_{i_p}

Cech上同调：
Ȟ^i(U, F) = H^i(C^*(U, F))

意义：
- 计算工具
- 应用广泛
```
