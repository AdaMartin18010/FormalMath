## 一、函子关系

### 1.1 基本函子

**基本函子**：

```
概形态射f: X → Y

基本函子：
- f^*: D(Y) → D(X)（拉回）
- f_*: D(X) → D(Y)（推前）
- f_!: D(X) → D(Y)（真推前）
- f^!: D(Y) → D(X)（例外拉回）

意义：
- 六函子理论
- 应用广泛
```

---

### 1.2 伴随关系

**伴随关系**：

```
关系：
- f^* ⊣ f_*
- f_! ⊣ f^!

意义：
- 函子关系
- 应用广泛
```

```
推前上同调：
R^i f_* F = H^i(f_* F)

```

### 2.2 拉回上同调

**拉回上同调**：

```
概形态射f: X → Y
Abel层G在Y上

拉回上同调：
L_i f^* G = H^{-i}(f^* G)

- 相对上同调
- 应用广泛
```

```
意义：
- 推前与拉回
```
