## 二、导出函子

### 2.1 左导出函子

**左导出函子**：

```
左正合函子F: A → B

左导出函子：
L_i F: D(A) → D(B)

定义：
L_i F(A) = H_i(F(P^*))

其中P^*是A的投射分解
```

---

### 2.2 右导出函子

**右导出函子**：

```
右正合函子F: A → B

右导出函子：
R^i F: D(A) → D(B)

定义：
R^i F(A) = H^i(F(I^*))

其中I^*是A的内射分解
```
