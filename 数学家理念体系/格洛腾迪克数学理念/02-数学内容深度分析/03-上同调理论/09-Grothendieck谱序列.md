# Grothendieck谱序列：上同调计算的强大工具

## 一、谱序列基础

### 1.1 定义

**谱序列E^{p,q}_r**：

```
双分次对象E^{p,q}_0

微分：
d_r: E^{p,q}_r → E^{p+r,q-r+1}_r

性质：
- d_r^2 = 0
- E^{p,q}_{r+1} = H(E^{p,q}_r, d_r)

收敛：
E^{p,q}_∞ = E^{p,q}_r（对r >> 0）
```

---

### 1.2 收敛

**谱序列收敛**：

```
谱序列E^{p,q}_r

收敛到H^n，若存在
滤过F^p H^n使得：
E^{p,q}_∞ ≅ F^p H^{p+q} / F^{p+1} H^{p+q}

意义：
- 计算上同调
- 应用广泛
```

## 二、Leray谱序列

### 2.1 定义

**Leray谱序列**：

```
连续映射f: X → Y
Abel层F在X上

Leray谱序列：
E_2^{p,q} = H^p(Y, R^q f_* F)
⇒ H^{p+q}(X, F)

意义：
- 上同调计算
- 应用广泛
```

## 三、Grothendieck谱序列

### 3.1 复合函子谱序列

**复合函子谱序列**：

```
复合函子：
G ∘ F: A → C

若F和G满足条件：
存在谱序列：
E_2^{p,q} = R^p G(R^q F(A))
⇒ R^{p+q}(G ∘ F)(A)

意义：
- 导出函子计算
- 应用广泛
```
