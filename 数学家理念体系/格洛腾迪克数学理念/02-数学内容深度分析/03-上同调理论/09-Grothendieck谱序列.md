## 一、谱序列基础

### 1.1 定义

**谱序列E^{p,q}_r**：

```
双分次对象E^{p,q}_0

微分：
d_r: E^{p,q}_r → E^{p+r,q-r+1}_r

性质：
- d_r^2 = 0
- E^{p,q}_{r+1} = H(E^{p,q}_r, d_r)

收敛：
E^{p,q}_∞ = E^{p,q}_r（对r >> 0）
```

---

### 1.2 收敛

**谱序列收敛**：

```
谱序列E^{p,q}_r

收敛到H^n，若存在
滤过F^p H^n使得：
E^{p,q}_∞ ≅ F^p H^{p+q} / F^{p+1} H^{p+q}

意义：
- 计算上同调
- 应用广泛
```

⇒ H^{p+q}(X, F)

意义：

- 应用广泛

## 九、数学公式总结

### 核心公式

1. **谱序列定义**：
   $$E = \{E_r^{p,q}\}, \quad d_r: E_r^{p,q} \to E_r^{p+r,q-r+1}, \quad d_r^2 = 0$$

2. **谱序列收敛**：
   $$E_r^{p,q} \Rightarrow H^{p+q}, \quad E_\infty^{p,q} = F^p H^{p+q} / F^{p+1} H^{p+q}$$

3. **Leray谱序列**：
   $$E_2^{p,q} = H^p(Y, R^q f_* \mathcal{F}) \Rightarrow H^{p+q}(X, \mathcal{F})$$

4. **Grothendieck谱序列**：
   $$E_2^{p,q} = R^p F(R^q G(A)) \Rightarrow R^{p+q}(F \circ G)(A)$$

5. **Hochschild-Serre谱序列**：
   $$E_2^{p,q} = H^p(G/H, H^q(H, M)) \Rightarrow H^{p+q}(G, M)$$

6. **Serre谱序列**：
   $$E_2^{p,q} = H^p(B, H^q(F)) \Rightarrow H^{p+q}(E)$$

7. **谱序列微分**：
   $$E_{r+1}^{p,q} = H(E_r^{p,q}, d_r) = \ker(d_r) / \text{im}(d_r)$$

8. **边缘项**：
   $$E_2^{p,0} = H^p(Y, f_* \mathcal{F}), \quad E_2^{0,q} = H^0(Y, R^q f_* \mathcal{F})$$

9. **滤过**：
   $$F^p H^n = \text{im}(H^n(X_{\leqqqqqqq p}, \mathcal{F}) \to H^n(X, \mathcal{F}))$$

10. **退化条件**：
    $$\text{若 } E_r^{p,q} = 0 \text{ 对所有 } p + q = n, p \neqqqqqqq p_0, \text{ 则 } H^n \cong E_r^{p_0,n-p_0}$$
