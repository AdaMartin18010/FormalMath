## 一、谱序列的基础

### 1.1 定义

**谱序列E**：

```
谱序列E = {E_r^{p,q}}

数据：
- 页：E_r（r ≥ 1）
- 双分次：E_r^{p,q}
- 微分：d_r: E_r^{p,q} → E_r^{p+r,q-r+1}
- 收敛：E_∞

意义：
- 计算工具
- 复杂上同调
- 逐步逼近
```

---

### 1.2 收敛

**谱序列收敛**：

```
谱序列E收敛到H^*：

E_r^{p,q} ⇒ H^{p+q}

意义：
- 最终同调
- 逐步计算
- 应用广泛
```

## 九、数学公式总结

### 核心公式

1. **谱序列定义**：
   $$E = \{E_r^{p,q}\}, \quad d_r: E_r^{p,q} \to E_r^{p+r,q-r+1}$$

2. **谱序列收敛**：
   $$E_r^{p,q} \Rightarrow H^{p+q}, \quad E_\infty^{p,q} = F^p H^{p+q} / F^{p+1} H^{p+q}$$

3. **Leray谱序列**：
   $$E_2^{p,q} = H^p(Y, R^q f_* \mathcal{F}) \Rightarrow H^{p+q}(X, \mathcal{F})$$

4. **Grothendieck谱序列**：
   $$E_2^{p,q} = R^p F(R^q G(A)) \Rightarrow R^{p+q}(F \circ G)(A)$$

5. **Hochschild-Serre谱序列**：
   $$E_2^{p,q} = H^p(G/H, H^q(H, M)) \Rightarrow H^{p+q}(G, M)$$

6. **Serre谱序列**：
   $$E_2^{p,q} = H^p(B, H^q(F)) \Rightarrow H^{p+q}(E)$$

7. **微分算子**：
   $$d_r^2 = 0, \quad E_{r+1}^{p,q} = H(E_r^{p,q}, d_r)$$

8. **边缘项**：
   $$E_r^{p,0} = \text{im}(H^p) \text{（边缘项）}$$

9. **消失项**：
   $$E_r^{0,q} = \ker(H^q) \text{（消失项）}$$

10. **退化条件**：
    $$\text{若 } E_r^{p,q} = 0 \text{ 对某个 $r$，则 } E_s^{p,q} = 0 \text{ 对所有 } s \geq r$$

11. **谱序列的定义**：
    谱序列的定义：
    $$E_r = \{E_r^{p,q}, d_r: E_r^{p,q} \to E_r^{p+r, q-r+1}\}$$

12. **Leray谱序列的构造**：
    Leray谱序列的构造：
    $$E_2^{p,q} = H^p(Y, R^q f_* \mathcal{F}) \Rightarrow H^{p+q}(X, \mathcal{F})$$

13. **Grothendieck谱序列的构造**：
    Grothendieck谱序列的构造：
    $$E_2^{p,q} = R^p F(R^q G(A)) \Rightarrow R^{p+q}(F \circ G)(A)$$

14. **谱序列的收敛**：
    谱序列的收敛：
    $$E_r^{p,q} \Rightarrow H^{p+q} \text{（收敛到 $H^{p+q}$）}$$

## 十一、谱序列与Leray谱序列的详细数学表述

### 11.1 谱序列的定义

**定义**：

**谱序列**$E_r$由以下数据组成：
$$E_r = \{E_r^{p,q}, d_r: E_r^{p,q} \to E_r^{p+r, q-r+1}\}$$

其中$d_r$是**微分算子**，满足$d_r^2 = 0$。

**收敛**：

谱序列**收敛**到$H^{p+q}$：
$$E_r^{p,q} \Rightarrow H^{p+q} \text{（收敛到 $H^{p+q}$）}$$

**数学公式**：
- 谱序列: $$E_r = \{E_r^{p,q}, d_r: E_r^{p,q} \to E_r^{p+r, q-r+1}\}$$
- 微分算子: $$d_r^2 = 0, \quad E_{r+1}^{p,q} = H(E_r^{p,q}, d_r)$$
- 收敛: $$E_r^{p,q} \Rightarrow H^{p+q}$$

---

### 11.2 Leray谱序列与Grothendieck谱序列

**Leray谱序列**：

对于态射$f: X \to Y$和层$\mathcal{F}$，**Leray谱序列**为：
$$E_2^{p,q} = H^p(Y, R^q f_* \mathcal{F}) \Rightarrow H^{p+q}(X, \mathcal{F})$$

**Grothendieck谱序列**：

对于函子复合$F \circ G$，**Grothendieck谱序列**为：
$$E_2^{p,q} = R^p F(R^q G(A)) \Rightarrow R^{p+q}(F \circ G)(A)$$

**数学公式**：
- Leray谱序列: $$E_2^{p,q} = H^p(Y, R^q f_* \mathcal{F}) \Rightarrow H^{p+q}(X, \mathcal{F})$$
- Grothendieck谱序列: $$E_2^{p,q} = R^p F(R^q G(A)) \Rightarrow R^{p+q}(F \circ G)(A)$$
- 应用: $$\text{计算上同调、研究函子复合}$$

---

### 11.3 谱序列的收敛与应用

**收敛条件**：

谱序列在以下条件下**收敛**：
- 对每个$(p,q)$，存在$r_0$使得$E_r^{p,q} = E_\infty^{p,q}$对所有$r \geq r_0$
- $H^{p+q} = \bigoplus_{p+q=n} E_\infty^{p,q}$

**应用**：

谱序列在以下领域有重要应用：
- **计算上同调**：通过谱序列计算复杂上同调群
- **研究函子复合**：研究导出函子的复合
- **研究群上同调**：通过Hochschild-Serre谱序列研究群上同调

**数学公式**：
- 收敛条件: $$E_r^{p,q} = E_\infty^{p,q} \text{（对所有 $r \geq r_0$）}$$
- 收敛: $$H^{p+q} = \bigoplus_{p+q=n} E_\infty^{p,q}$$
- 应用: $$\text{计算上同调、研究函子复合、研究群上同调}$$
