## 一、谱序列的基础

### 1.1 定义

**谱序列E**：

```
谱序列E = {E_r^{p,q}}

数据：
- 页：E_r（r ≥ 1）
- 双分次：E_r^{p,q}
- 微分：d_r: E_r^{p,q} → E_r^{p+r,q-r+1}
- 收敛：E_∞

意义：
- 计算工具
- 复杂上同调
- 逐步逼近
```

---

### 1.2 收敛

**谱序列收敛**：

```
谱序列E收敛到H^*：

E_r^{p,q} ⇒ H^{p+q}

意义：
- 最终同调
- 逐步计算
- 应用广泛
```

## 二、Leray谱序列

### 2.1 构造

**Leray谱序列**：

```
连续映射f: X → Y
层F在X上

Leray谱序列：
E_2^{p,q} = H^p(Y, R^q f_* F)
⇒ H^{p+q}(X, F)

意义：
- 相对上同调
- 绝对上同调
- 计算工具
```

### 2.2 应用

## 九、数学公式总结

### 核心公式

1. **谱序列定义**：
   $$E = \{E_r^{p,q}\}, \quad d_r: E_r^{p,q} \to E_r^{p+r,q-r+1}$$

2. **谱序列收敛**：
   $$E_r^{p,q} \Rightarrow H^{p+q}, \quad E_\infty^{p,q} = F^p H^{p+q} / F^{p+1} H^{p+q}$$

3. **Leray谱序列**：
   $$E_2^{p,q} = H^p(Y, R^q f_* \mathcal{F}) \Rightarrow H^{p+q}(X, \mathcal{F})$$

4. **Grothendieck谱序列**：
   $$E_2^{p,q} = R^p F(R^q G(A)) \Rightarrow R^{p+q}(F \circ G)(A)$$

5. **Hochschild-Serre谱序列**：
   $$E_2^{p,q} = H^p(G/H, H^q(H, M)) \Rightarrow H^{p+q}(G, M)$$

6. **Serre谱序列**：
   $$E_2^{p,q} = H^p(B, H^q(F)) \Rightarrow H^{p+q}(E)$$

7. **微分算子**：
   $$d_r^2 = 0, \quad E_{r+1}^{p,q} = H(E_r^{p,q}, d_r)$$

8. **边缘项**：
   $$E_r^{p,0} = \text{im}(H^p) \text{（边缘项）}$$

9. **消失项**：
   $$E_r^{0,q} = \ker(H^q) \text{（消失项）}$$

10. **退化条件**：
    $$\text{若 } E_r^{p,q} = 0 \text{ 对某个 $r$，则 } E_s^{p,q} = 0 \text{ 对所有 } s \geq r$$
