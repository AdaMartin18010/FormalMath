## 一、谱序列的基础

### 1.1 定义

**谱序列E**：

```
谱序列E = {E_r^{p,q}}

数据：
- 页：E_r（r ≥ 1）
- 双分次：E_r^{p,q}
- 微分：d_r: E_r^{p,q} → E_r^{p+r,q-r+1}
- 收敛：E_∞

意义：
- 计算工具
- 复杂上同调
- 逐步逼近
```

---

### 1.2 收敛

**谱序列收敛**：

```
谱序列E收敛到H^*：

E_r^{p,q} ⇒ H^{p+q}

意义：
- 最终同调
- 逐步计算
- 应用广泛
```

---

## 二、Leray谱序列

### 2.1 构造

**Leray谱序列**：

```
连续映射f: X → Y
层F在X上

Leray谱序列：
E_2^{p,q} = H^p(Y, R^q f_* F)
⇒ H^{p+q}(X, F)

意义：
- 相对上同调
- 绝对上同调
- 计算工具
```

---

### 2.2 应用

**Leray谱序列的应用**：

```
应用：
- 计算上同调
- 相对上同调
- 几何应用

例子：
- 纤维丛
- 覆盖
- 几何构造
```
