## 二、消失性定理

### 2.1 Serre消失定理

**Serre消失定理**：

```
射影概形X
充分大的可逆层L

Serre消失：
H^i(X, F ⊗ L^n) = 0 (i > 0, n >> 0)

意义：
- 消失性
- 应用广泛
```

---

### 2.2 Grothendieck消失定理

**Grothendieck消失定理**：

```
Noether概形X
维度n

消失定理：
H^i(X, F) = 0 (i > n)

对所有Abel层F

意义：
- 上同调维数
- 应用广泛
```

## 九、数学公式总结

### 核心公式

1. **有限性定理**：
   $$X \text{ Noether}, f: X \to S \text{ 相对完备}, \mathcal{F} \text{ 凝聚} \Rightarrow H^i(X, \mathcal{F}) \text{ 有限型}$$

2. **Serre消失定理**：
   $$X \text{ 射影}, \mathcal{L} \text{ 非常丰富} \Rightarrow H^i(X, \mathcal{F} \otimes \mathcal{L}^n) = 0 \text{（$i > 0$, $n \gg 0$）}$$

3. **Grothendieck消失定理**：
   $$X \text{ Noether}, \dim X = n \Rightarrow H^i(X, \mathcal{F}) = 0 \text{（$i > n$）}$$

4. **上同调维数**：
   $$\text{cd}(X) = \sup\{i : H^i(X, \mathcal{F}) \neq 0 \text{ 对某个 $\mathcal{F}$}\}$$

5. **Serre对偶与消失**：
   $$H^i(X, \mathcal{F}) = 0 \iff H^{n-i}(X, \mathcal{F}^* \otimes \omega_X) = 0$$

6. **Kodaira消失（特征0）**：
   $$X \text{ 光滑射影}, \mathcal{L} \text{ 丰富} \Rightarrow H^i(X, \omega_X \otimes \mathcal{L}) = 0 \text{（$i > 0$）}$$

7. **上同调有限性（相对情况）**：
   $$R^i f_* \mathcal{F} \text{ 凝聚（$f$ 相对完备，$\mathcal{F}$ 凝聚）}$$

8. **局部消失**：
   $$H^i_{\{x\}}(X, \mathcal{F}) = 0 \text{（$i < \text{depth}_x(\mathcal{F})$）}$$

9. **支撑维数**：
   $$\text{Supp}(\mathcal{F}) = \{x : \mathcal{F}_x \neq 0\}, \quad \dim \text{Supp}(\mathcal{F}) \leq \dim X$$

10. **消失性判定**：
    $$H^i(X, \mathcal{F}) = 0 \text{（$i > \dim \text{Supp}(\mathcal{F})$）}$$
