## 二、消失性定理

### 2.1 Serre消失定理

**Serre消失定理**：

```
射影概形X
充分大的可逆层L

Serre消失：
H^i(X, F ⊗ L^n) = 0 (i > 0, n >> 0)

意义：
- 消失性
- 应用广泛
```
---

### 2.2 Grothendieck消失定理

**Grothendieck消失定理**：

```
Noether概形X
维度n

消失定理：
H^i(X, F) = 0 (i > n)

对所有Abel层F

意义：
- 上同调维数
- 应用广泛
```
---
