## 一、层理论

### 1.1 预层

**预层F**：

```
数据：
对每个开集U ⊆ X，集合F(U)
对包含V ⊆ U，限制映射：
res_{U,V}: F(U) → F(V)

公理：
- res_{U,U} = id
- res_{V,W} ∘ res_{U,V} = res_{U,W}
```

---

### 1.2 层条件

**层F**：

```
预层F是层，若满足粘接条件：

对开覆盖U = ∪_i U_i
和截面s_i ∈ F(U_i)

若在交集上一致：
res_{U_i, U_i∩U_j}(s_i) = res_{U_j, U_i∩U_j}(s_j)

则存在唯一s ∈ F(U)使得
res_{U, U_i}(s) = s_i
```

## 四、Čech上同调

### 4.1 构造

**Čech复形**：

```
开覆盖U = {U_i}

Čech复形：
C^p(U, F) = ∏_{i_0<...<i_p} F(U_{i_0...i_p})

微分：
d: C^p → C^{p+1}

上同调：
Ĥ^p(U, F) = H^p(C^•(U, F))
```

---

### 4.2 与层上同调的关系

**比较定理**：

```
对好覆盖U：
Ĥ^p(U, F) ≅ H^p(X, F)

一般：
Ĥ^p(X, F) = lim_{U} Ĥ^p(U, F)
```

## 五、消失定理

### 5.1 仿射情况

**Serre消失**：

```
X仿射概形
F拟凝聚层

则：
H^i(X, F) = 0（i > 0）

意义：
仿射上上同调平凡
```

---

### 5.2 分离概形

**分离情况**：

```
X分离Noether概形
F拟凝聚层

则：
H^i(X, F) = 0（i > dim X）

有界性
```

## 六、Leray谱序列

```
f: X → Y（连续映射）

Leray谱序列：
E_2^{p,q} = H^p(Y, R^q f_* F)
⇒ H^{p+q}(X, F)
```


## 九、数学公式总结

### 核心公式

1. **预层定义**：
   $$\mathcal{F}(U) \text{（开集 $U$ 上的截面）}$$

2. **层条件**：
   $$\text{若 } s_i|_{U_i \cap U_j} = s_j|_{U_i \cap U_j}, \text{ 则存在唯一 } s \text{ 使得 } s|_{U_i} = s_i$$

3. **全局截面函子**：
   $$\Gamma(X, \mathcal{F}) = \mathcal{F}(X)$$

4. **上同调群**：
   $$H^i(X, \mathcal{F}) = R^i \Gamma(X, \mathcal{F})$$

5. **Čech上同调**：
   $$\check{H}^i(\mathcal{U}, \mathcal{F}) = H^i(\check{C}^\bullet(\mathcal{U}, \mathcal{F}))$$

6. **长正合序列**：
   $$0 \to \Gamma(X, \mathcal{F}) \to \Gamma(X, \mathcal{G}) \to \Gamma(X, \mathcal{H}) \to H^1(X, \mathcal{F}) \to \cdots$$

7. **Leray谱序列**：
   $$E_2^{p,q} = H^p(X, R^q f_* \mathcal{F}) \Rightarrow H^{p+q}(Y, \mathcal{F})$$

8. **消失定理（Serre）**：
   $$H^i(X, \mathcal{F}) = 0 \text{ 对所有 } i > 0 \text{（$X$ 仿射，$\mathcal{F}$ 拟凝聚）}$$

9. **上同调维数**：
   $$\text{cd}(X) = \max\{i : H^i(X, \mathcal{F}) \neq 0 \text{ 对某个 } \mathcal{F}\}$$

10. **对偶定理**：
    $$H^i(X, \mathcal{F})^* \cong H^{n-i}(X, \mathcal{F}^* \otimes \omega_X) \text{（Serre对偶）}$$

11. **上同调的长正合列**：
    上同调的长正合列：
    $$0 \to H^0(X, \mathcal{F}) \to H^0(X, \mathcal{G}) \to H^0(X, \mathcal{H}) \to H^1(X, \mathcal{F}) \to \cdots$$

12. **Čech上同调与层上同调的关系**：
    Čech上同调与层上同调的关系：
    $$\check{H}^i(\mathcal{U}, \mathcal{F}) \cong H^i(X, \mathcal{F}) \text{（对好覆盖）}$$

13. **上同调的函子性**：
    上同调的函子性：
    $$f: X \to Y \Rightarrow f^*: H^i(Y, \mathcal{F}) \to H^i(X, f^*\mathcal{F})$$

14. **上同调的消失定理**：
    上同调的消失定理：
    $$H^i(X, \mathcal{F}) = 0 \text{ 对所有 } i > 0 \text{（$X$ 仿射，$\mathcal{F}$ 拟凝聚）}$$

15. **上同调维数**：
    上同调维数：
    $$\text{cd}(X) = \max\{i : H^i(X, \mathcal{F}) \neq 0 \text{ 对某个 } \mathcal{F}\}$$

## 十、层上同调的详细数学表述

### 10.1 上同调的长正合列

**长正合列**：

对于短正合列$0 \to \mathcal{F} \to \mathcal{G} \to \mathcal{H} \to 0$，有**长正合列**：
$$0 \to H^0(X, \mathcal{F}) \to H^0(X, \mathcal{G}) \to H^0(X, \mathcal{H}) \to H^1(X, \mathcal{F}) \to H^1(X, \mathcal{G}) \to H^1(X, \mathcal{H}) \to \cdots$$

**连接同态**：

存在**连接同态**$\delta: H^i(X, \mathcal{H}) \to H^{i+1}(X, \mathcal{F})$使得序列正合。

**数学公式**：
- 长正合列: $$0 \to H^0(X, \mathcal{F}) \to H^0(X, \mathcal{G}) \to H^0(X, \mathcal{H}) \to H^1(X, \mathcal{F}) \to \cdots$$
- 连接同态: $$\delta: H^i(X, \mathcal{H}) \to H^{i+1}(X, \mathcal{F})$$
- 正合性: $$\text{序列正合}$$

---

### 10.2 Čech上同调与层上同调的关系

**Čech上同调**：

对于开覆盖$\mathcal{U} = \{U_i\}$，**Čech上同调**定义为：
$$\check{H}^i(\mathcal{U}, \mathcal{F}) = H^i(\check{C}^\bullet(\mathcal{U}, \mathcal{F}))$$

**关系**：

对于**好覆盖**（所有有限交是仿射的），有：
$$\check{H}^i(\mathcal{U}, \mathcal{F}) \cong H^i(X, \mathcal{F})$$

**数学公式**：
- Čech上同调: $$\check{H}^i(\mathcal{U}, \mathcal{F}) = H^i(\check{C}^\bullet(\mathcal{U}, \mathcal{F}))$$
- 关系: $$\check{H}^i(\mathcal{U}, \mathcal{F}) \cong H^i(X, \mathcal{F}) \text{（对好覆盖）}$$
- 好覆盖: $$\text{所有有限交是仿射的}$$

---

### 10.3 上同调的消失定理与维数

**消失定理（Serre）**：

如果$X$是**仿射概形**，$\mathcal{F}$是**拟凝聚层**，则：
$$H^i(X, \mathcal{F}) = 0 \text{ 对所有 } i > 0$$

**上同调维数**：

**上同调维数**定义为：
$$\text{cd}(X) = \max\{i : H^i(X, \mathcal{F}) \neq 0 \text{ 对某个 } \mathcal{F}\}$$

**数学公式**：
- 消失定理: $$H^i(X, \mathcal{F}) = 0 \text{ 对所有 } i > 0 \text{（$X$ 仿射，$\mathcal{F}$ 拟凝聚）}$$
- 上同调维数: $$\text{cd}(X) = \max\{i : H^i(X, \mathcal{F}) \neq 0\}$$
- 维数性质: $$\text{cd}(X) \leq \dim X$$
