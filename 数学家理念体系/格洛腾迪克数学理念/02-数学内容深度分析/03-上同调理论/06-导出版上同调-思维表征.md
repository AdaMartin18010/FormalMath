## 七、自测要点与复习要点

**自测要点（回忆问题）**：① D(A) 如何由 K(A) 得到？② RΓ、Rf_*、Lf^* 的含义与关系？③ 经典 H^i 与导出版 RΓ 的关系？

供回忆与间隔重复使用；与 [project/00-认知诊断与内容对接说明-2026年2月](../../../../project/00-认知诊断与内容对接说明-2026年2月.md) 对接；建议在 1 天、1 周、1 月 后各回顾一次。

1. **导出范畴**：D(A) 由 K(A) 对拟同构局部化得到，是三角范畴；对象为链复形，态射含拟同构之逆。
2. **右/左导出函子**：R^i F = H^i(F(I^*))（内射消解）、L_i F = H_i(F(P^*))（投射消解）；与消解选取无关。
3. **层上导出版**：RΓ: D^+(X)→D^+(Ab) 满足 H^i(RΓ(F))=H^i(X,F)；Rf_* 为导出推前，与 RΓ 复合得 Leray 谱序列。
4. **何时用导出版**：需要函子性、复合 R(G∘F)、或现代表述时用 RΓ/Rf_*/RHom/⊗^L；仅算 H^i 时可用经典 R^i。
5. **与经典关系**：经典 H^i 是导出版 RΓ 的 cohomology 对象；Grothendieck 谱序列来自 D 中复合。

## 八、最小可运行示例（RΓ 的一个具体计算）

**目的**：用最简情形看清 RΓ 与经典 H^i 的对应，降低认知负荷。

- **设定**：X = Spec(k)（一点概形，k 为域），F = k̃（常层，即 F(X)=k）。则 Sh(X) 上 Γ(X,-) 为「取全局截面」，即 Γ(X,F)=k。
- **经典计算**：k̃ 为内射层（在一点上 Abel 层均内射），故内射消解可取 0 → k̃ → 0 → …，因此 H^0(X,k̃)=k，H^{>0}(X,k̃)=0。
- **导出版**：RΓ(X,k̃) ∈ D^+(Ab) 拟同构于「复形」仅在第 0 位为 k、其余为 0 的对象，即 k[0]。因此 H^i(RΓ(X,k̃)) = H^i(k[0]) = k 当 i=0，否则 0，与 H^i(X,k̃) 一致。
- **要点**：RΓ 把层 F 变成 D^+(Ab) 中一个对象；「取第 i 个上同调」H^i 得到的就是经典 H^i(X,F)。在仿射概形上对拟凝聚层常有 RΓ 拟同构于「集中在 degree 0」的复形（消失定理）。
