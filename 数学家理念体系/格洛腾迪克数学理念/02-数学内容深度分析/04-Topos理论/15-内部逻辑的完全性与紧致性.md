## 九、完全性与紧致性的详细数学表述

### 9.1 完全性的严格定义与证明

**完全性定义**：

Topos $\mathcal{E}$ 的**内部逻辑完全**，如果对任意一阶理论$T$：
$$T \text{ 在 } \mathcal{E} \text{ 中有模型 } \iff T \text{ 一致}$$

**模型存在定理**：

完全性等价于**模型存在定理**：
$$T \text{ 一致 } \Rightarrow \text{存在模型 } M: T \to \mathcal{E}$$

**证明方法**：

使用**分类Topos**和**几何点**：

1. 构造分类Topos $\mathcal{E}_T$ 对应理论$T$
2. 使用几何点证明模型存在性
3. 完全性成立

**数学公式**：

- 完全性定义：$$T \text{ 有模型 } \iff T \text{ 一致}$$
- 模型存在定理：$$T \text{ 一致 } \Rightarrow \exists M: T \to \mathcal{E}$$
- 分类Topos：$$\mathcal{E}_T = \text{分类Topos}(T)$$

---

### 9.2 紧致性的严格定义与证明

**紧致性定义**：

Topos $\mathcal{E}$ 的**内部逻辑紧致**，如果对任意一阶理论$T$：
$$\text{若 } T \text{ 的每个有限子理论有模型，则 } T \text{ 有模型}$$

**等价形式**：

$$T \text{ 有模型 } \iff \forall T_0 \subset T \text{ 有限}, T_0 \text{ 有模型}$$

**拓扑解释**：

紧致性对应拓扑空间的**紧致性**（有限覆盖性质）：

- 逻辑紧致性：有限子理论有模型 $\Rightarrow$ 理论有模型
- 拓扑紧致性：有限子覆盖 $\Rightarrow$ 空间紧致

**Grothendieck Topos的紧致性**：

**Grothendieck Topos**的内部逻辑是紧致的。

**数学公式**：

- 紧致性定义：$$T \text{ 有模型 } \iff \forall T_0 \subset T \text{ 有限}, T_0 \text{ 有模型}$$
- 拓扑对应：$$\text{逻辑紧致性} \leftrightarrow \text{拓扑紧致性}$$
- Grothendieck Topos：$$\text{Grothendieck Topos} \Rightarrow \text{紧致性}$$
