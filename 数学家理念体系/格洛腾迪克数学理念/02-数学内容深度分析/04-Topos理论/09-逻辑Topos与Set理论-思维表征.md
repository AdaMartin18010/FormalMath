## 二、多维概念对比矩阵

### 2.1 逻辑 Topos vs Set

| 概念 | 逻辑 Topos | Set |
|------|------------|-----|
| 对象 | 层、类型 | 集合 |
| 真值 | $\Omega$ | 2 |
| 幂集 | $P(A)$ | $\mathcal{P}(A)$ |
| 逻辑 | 直觉主义（一般） | 经典 |

### 2.2 概念层次（逻辑 Topos 到内部集合论）

| 概念 | 含义 | 与本文关系 |
|------|------|------------|
| 逻辑 Topos = 分类 Topos | 对应几何理论 T，$\mathbf{Set}[\mathbb{T}]$ | 见 08 |
| 内部集合论 | $\Omega$ 为真值，$P(A)$ 为幂集，公理在 E 中解释 | 外延、配对、并等 |
| 与 Set | Set = 初等 Topos，经典逻辑 | 几何点、分类 |
