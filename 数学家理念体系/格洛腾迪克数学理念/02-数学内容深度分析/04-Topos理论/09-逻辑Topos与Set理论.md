# 逻辑Topos与Set理论：集合论基础

## 一、Set范畴

### 1.1 定义

**Set范畴**：

```
Set = 集合范畴

对象：集合
态射：函数

性质：
- Grothendieck Topos
- 内部逻辑：经典逻辑
- 应用广泛
```

---

### 1.2 Topos结构

**Set作为Topos**：

```
Set的性质：
- 有所有（余）极限
- 子对象分类器：2 = {0, 1}
- 幂对象：P(A) = 2^A
- 内部逻辑：经典逻辑

意义：
- 基础Topos
- 应用广泛
```
