## 一、Topos的内部逻辑

### 1.1 内部语言

**Topos的内部语言**：

```
在Topos E中：
- 对象：类型
- 态射：项
- 子对象：命题
- 逻辑运算：范畴运算

意义：
- 在E中做数学
- 使用E的逻辑
- 相对逻辑
```
---

### 1.2 逻辑规则

**Topos逻辑规则**：

```
在Topos E中：
- 合取：A ∧ B = A ×_Ω B
- 析取：A ∨ B = 余积
- 蕴含：A ⇒ B = 指数对象
- 否定：¬A = A ⇒ false

性质：
- 直觉主义逻辑
- 可能不满足排中律
```
---

## 二、排中律

### 2.1 经典逻辑

**Set中的排中律**：

```
在Set中：
对任意命题P：
P ∨ ¬P = true

排中律成立
```
---

### 2.2 Topos逻辑

**排中律在Topos中**：

```
在一般Topos E中：
P ∨ ¬P可能 ≠ true

例子：Sh(X)（X非平凡）
- 局部vs全局
- 排中律失效
- 直觉主义逻辑
```
