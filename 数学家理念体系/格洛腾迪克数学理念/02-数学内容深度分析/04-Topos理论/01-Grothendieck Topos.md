# Grothendieck Topos：广义集合论

## 一、Topos的定义

### 1.1 基本要求

**Grothendieck Topos E**：

```
条件：
1. 有所有有限极限
2. 有所有余极限
3. 有子对象分类器Ω
4. 有幂对象P(A)
5. 是层范畴Sh(C, J)

等价：
E是某个格罗腾迪克位型的层范畴
```

---

### 1.2 例子

**拓扑空间上的层**：

```
X：拓扑空间
Sh(X)：X上的层范畴

性质：
- Grothendieck Topos
- 子对象分类器
- 幂对象
```

---

## 二、子对象分类器

### 2.1 定义

**子对象分类器Ω**：

```
存在对象Ω和真值true: 1 → Ω
使得对任意子对象A ⊆ X
存在唯一特征映射χ_A: X → Ω
使得A是χ_A^{-1}(true)的拉回
```

---

### 2.2 例子

**Set中的Ω**：

```
Ω = {true, false}
true: {*} → {true, false}

对子集A ⊆ X：
χ_A(x) = true（x ∈ A）
χ_A(x) = false（x ∉ A）
```

**Sh(X)中的Ω**：

```
Ω(U) = {U的开子集}

真值：
true(U) = U

意义：
- 局部真值
- 几何逻辑
```

### 3.1 Topos逻辑

**内部逻辑**：

```
在Topos E中：
- 对象：类型
- 态射：项
- 子对象分类器：命题
- 逻辑运算：范畴运算
```

---

### 3.2 逻辑运算

**合取、析取、蕴含**：

```
合取：
A ∧ B = A ×_Ω B

析取：
A ∨ B = 余积

蕴含：
A ⇒ B = 指数对象

否定：
¬A = A ⇒ false
```

## 四、排中律

### 4.1 经典逻辑

**Set中的排中律**：

```
对任意命题P：
P ∨ ¬P = true

排中律成立
```

---

### 4.2 Topos逻辑

**排中律可能失效**：

```
在一般Topos E中：
P ∨ ¬P可能 ≠ true

意义：
- 直觉主义逻辑
- 几何逻辑
- 相对逻辑
```

## 五、层上同调框架

### 5.1 上同调在Topos中

**上同调框架**：

```
Topos E：
- 全局截面函子Γ: E → Set
- 上同调H^i(E, -) = R^i Γ

性质：
- 导出函子
- 长正合列
- 统一框架
```

---

### 5.2 应用

```
概形X：
Sh(X)是Topos

H^i(X, F) = H^i(Sh(X), F)

统一框架
```

## 六、几何形态

### 6.1 定义

**几何形态f: E → F**：

```
函子对：
f*: F → E（拉回）
f_*: E → F（推前）

伴随：
f* ⊣ f_*

性质：
- 保持有限极限
- 有右伴随
```

---

### 6.2 例子

**连续映射诱导**：

```
f: X → Y（连续映射）

诱导几何形态：
f_*: Sh(X) → Sh(Y)
f*: Sh(Y) → Sh(X)

f* ⊣ f_*
```
