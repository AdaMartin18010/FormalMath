## 二、拟凝聚层

### 2.1 定义

**拟凝聚层F**：

```
概形X上的O_X模层F

拟凝聚条件：
对每个仿射开集U = Spec(R)
F|_U ≅ M~（M是R模）

其中M~是M关联的层
```

---

### 2.2 性质

**拟凝聚层的性质**：

```
性质：
- 局部自由（局部）
- 全局可能复杂
- 上同调有限维（Noether情况）
```
