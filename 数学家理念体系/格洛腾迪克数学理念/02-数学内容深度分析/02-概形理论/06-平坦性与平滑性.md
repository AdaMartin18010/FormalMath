## 十一、平坦性与平滑性的详细数学表述

### 11.1 平坦性的等价定义

**等价定义**：

模$M$是**平坦的**，如果张量积函子$M \otimes_A -$是正合的：
$$M \text{ 平坦 } \iff M \otimes_A - \text{ 正合函子}$$

**Tor函子刻画**：

平坦性也可以用Tor函子刻画：
$$M \text{ 平坦 } \iff \text{Tor}_i^A(M, N) = 0 \text{ 对所有 } i > 0 \text{ 和所有 $N$}$$

**数学公式**：
- 等价定义: $$M \text{ 平坦 } \iff M \otimes_A - \text{ 正合函子}$$
- Tor刻画: $$\text{Tor}_i^A(M, N) = 0 \text{ 对所有 } i > 0$$
- 平坦性: $$\text{平坦模保持正合性}$$

---

### 11.2 平坦性的局部判定

**局部判定**：

模$M$是平坦的当且仅当它在所有素理想处局部化是平坦的：
$$M \text{ 平坦 } \iff M_\mathfrak{p} \text{ 平坦（对所有素理想 $\mathfrak{p}$）}$$

**局部性质**：

平坦性是**局部性质**，可以在局部环上检查。

**数学公式**：
- 局部判定: $$M \text{ 平坦 } \iff M_\mathfrak{p} \text{ 平坦（对所有素理想 $\mathfrak{p}$）}$$
- 局部性质: $$\text{平坦性是局部性质}$$
- 局部检查: $$\text{可以在局部环上检查平坦性}$$

---

### 11.3 平滑态射的切空间刻画

**切空间刻画**：

态射$f: X \to Y$是**平滑的**，如果切空间的维数等于相对维数：
$$f: X \to Y \text{ 平滑 } \iff \dim T_{X/Y,x} = \dim(X/Y) \text{（对所有 $x$）}$$

**Kähler微分的性质**：

如果$f$是平滑的，则Kähler微分$\Omega_{X/Y}$是**局部自由的**：
$$\Omega_{X/Y} \text{ 局部自由 } \iff f \text{ 平滑}$$

**相对维数的计算**：

**相对维数**可以计算为：
$$\dim(X/Y) = \dim_x X_{f(x)} = \dim \Omega_{X/Y,x}$$

**数学公式**：
- 切空间刻画: $$\dim T_{X/Y,x} = \dim(X/Y) \text{（对所有 $x$）}$$
- Kähler微分: $$\Omega_{X/Y} \text{ 局部自由 } \iff f \text{ 平滑}$$
- 相对维数: $$\dim(X/Y) = \dim_x X_{f(x)} = \dim \Omega_{X/Y,x}$$
