## 十一、完备概形与紧性的详细数学表述

### 11.1 相对完备的等价定义

**等价定义**：

态射$f: X \to S$是**相对完备的（proper）**，如果：
$$f: X \to S \text{ 相对完备 } \iff f \text{ 分离 + 泛闭}$$

**泛闭性**：

态射$f$是**泛闭的**，如果对所有基变化$S' \to S$，$f_{S'}$是闭映射。

**数学公式**：
- 相对完备: $$f: X \to S \text{ 相对完备 } \iff f \text{ 分离 + 泛闭}$$
- 泛闭性: $$f \text{ 泛闭 } \iff f_{S'} \text{ 闭（对所有基变化）}$$
- 完备性: $$\text{完备 = 分离 + 泛闭}$$

---

### 11.2 Valuative准则

**Valuative准则（分离性）**：

态射$f: X \to S$是**分离的**，如果对每个赋值环$R$和$K = \text{Frac}(R)$，存在**唯一扩张**：
$$\text{若存在唯一扩张，则 } f \text{ 分离}$$

**Valuative准则（完备性）**：

态射$f: X \to S$是**完备的**，如果对每个赋值环$R$和$K = \text{Frac}(R)$，存在**扩张**：
$$\text{若存在扩张，则 } f \text{ 完备}$$

**数学公式**：
- 分离性: $$\text{若存在唯一扩张，则 } f \text{ 分离}$$
- 完备性: $$\text{若存在扩张，则 } f \text{ 完备}$$
- Valuative准则: $$\text{Valuative准则用于判定分离性和完备性}$$

---

### 11.3 射影概形的完备性与上同调有限性

**射影概形的完备性**：

射影概形是**相对完备的**：
$$f: X \to S \text{ 射影 } \Rightarrow f \text{ 相对完备}$$

**上同调有限性**：

对于相对完备的态射$f: X \to S$和凝聚层$\mathcal{F}$，推前层是**凝聚的**：
$$f: X \to S \text{ 相对完备 } \Rightarrow R^i f_* \mathcal{F} \text{ 凝聚（$\mathcal{F}$ 凝聚）}$$

**数学公式**：
- 射影完备: $$f: X \to S \text{ 射影 } \Rightarrow f \text{ 相对完备}$$
- 上同调有限性: $$R^i f_* \mathcal{F} \text{ 凝聚（$\mathcal{F}$ 凝聚）}$$
- 完备性与紧性: $$\text{完备概形是代数几何中的紧性概念}$$
