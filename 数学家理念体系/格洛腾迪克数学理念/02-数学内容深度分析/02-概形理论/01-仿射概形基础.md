## 一、交换环的谱

### 1.1 Spec(R)的定义

**交换环R的谱**：

```
定义：
Spec(R) = {素理想p ⊆ R}

拓扑（Zariski拓扑）：
- 闭集：V(I) = {p | I ⊆ p}（I是理想）
- 开集：D(f) = Spec(R) - V((f))（f ∈ R）

基：
{D(f) | f ∈ R}形成拓扑基
```

---

### 1.2 结构层

**结构层O**：

```
定义：
O(D(f)) = R_f（R在f的局部化）
O(Spec(R)) = R

性质：
- 层结构
- 局部化自然
- 函数观点
```

## 二、基本例子

### 2.1 Spec(ℤ)

**整数环的谱**：

```text
Spec(ℤ) = {(0), (2), (3), (5), ...}

- 点(0)：一般点（稠密）
- 点(p)：闭点（素数p）

几何意义：
- (0)：整数的"一般点"
- (p)：模p约化
```

---

### 2.2 Spec(k[x])

**多项式环的谱**：

```text
k代数闭域：
Spec(k[x]) = {0} ∪ {(x-a) | a ∈ k}

- 点(0)：一般点
- 点(x-a)：闭点，对应a ∈ k

几何：
- 仿射直线A^1_k
- 点对应k中元素
```

**仿射概形**：

```
仿射概形 = (Spec(R), O)

数据：
- 拓扑空间：Spec(R)
- 结构层：O

性质：
- 环化空间
- 局部同构性质
```

---

### 3.2 函数观点

**函数在点上的值**：

```
f ∈ R，点p ∈ Spec(R)

函数值：
f(p) ∈ k(p)（剩余域）

k(p) = Frac(R/p)
```

## 四、态射

### 4.1 环同态与概形态射

**对应关系**：

```
环同态φ: R → S
    ↓
概形态射f: Spec(S) → Spec(R)

定义：
f(p) = φ^{-1}(p)

对偶：
- 环同态 ←→ 概形态射
- 范畴等价
```

---

### 4.2 局部化

**局部化对应开子集**：

```
f ∈ R
局部化R_f
    ↓
开集D(f) ⊆ Spec(R)

性质：
Spec(R_f) ≅ D(f)
```

## 五、纤维积

### 5.1 定义

**仿射概形的纤维积**：

```
Spec(A) → Spec(R) ← Spec(B)

纤维积：
Spec(A) ×_{Spec(R)} Spec(B)
= Spec(A ⊗_R B)

张量积对应纤维积
```

---

### 5.2 例子

**基变化**：

```
Spec(A) → Spec(R)
基变化到Spec(S)：

Spec(A) ×_{Spec(R)} Spec(S)
= Spec(A ⊗_R S)
```

## 十、数学公式总结

### 核心公式

1. **Spec定义**：
   $$\text{Spec}(R) = \{\mathfrak{p} \subseteq R : \mathfrak{p} \text{ 是素理想}\}$$

2. **Zariski拓扑**：
   $$V(I) = \{\mathfrak{p} \in \text{Spec}(R) : I \subseteq \mathfrak{p}\}$$

3. **开集基**：
   $$D(f) = \text{Spec}(R) \setminus V((f))$$

4. **结构层**：
   $$\mathcal{O}_{\text{Spec}(R)}(D(f)) = R_f = R[f^{-1}]$$

5. **局部化**：
   $$R_f = \left\{\frac{a}{f^n} : a \in R, n \geq 0\right\}$$

6. **概形态射**：
   $$f: \text{Spec}(S) \to \text{Spec}(R) \leftrightarrow \varphi: R \to S$$

7. **纤维积**：
   $$X \times_Z Y = \text{Spec}(R \otimes_S T)$$

8. **全局截面**：
   $$\Gamma(\text{Spec}(R), \mathcal{O}) = R$$

9. **函数域**：
   $$K(X) = \mathcal{O}_{X,\eta} \text{（$\eta$ 是一般点）}$$

10. **点与理想对应**：
    $$\text{点} \leftrightarrow \text{素理想}, \quad \text{闭点} \leftrightarrow \text{极大理想}$$
