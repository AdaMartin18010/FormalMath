## 一、交换环的谱

### 1.1 Spec(R)的定义

**交换环R的谱**：

```
定义：
Spec(R) = {素理想p ⊆ R}

拓扑（Zariski拓扑）：
- 闭集：V(I) = {p | I ⊆ p}（I是理想）
- 开集：D(f) = Spec(R) - V((f))（f ∈ R）

基：
{D(f) | f ∈ R}形成拓扑基
```

---

### 1.2 结构层

**结构层O**：

```
定义：
O(D(f)) = R_f（R在f的局部化）
O(Spec(R)) = R

性质：
- 层结构
- 局部化自然
- 函数观点
```

---

## 二、基本例子

### 2.1 Spec(ℤ)

**整数环的谱**：

```text
Spec(ℤ) = {(0), (2), (3), (5), ...}

- 点(0)：一般点（稠密）
- 点(p)：闭点（素数p）

几何意义：
- (0)：整数的"一般点"
- (p)：模p约化
```

---

### 2.2 Spec(k[x])

**多项式环的谱**：

```text
k代数闭域：
Spec(k[x]) = {0} ∪ {(x-a) | a ∈ k}

- 点(0)：一般点
- 点(x-a)：闭点，对应a ∈ k

几何：
- 仿射直线A^1_k
- 点对应k中元素
```

## 四、态射

### 4.1 环同态与概形态射

**对应关系**：

```
环同态φ: R → S
    ↓
概形态射f: Spec(S) → Spec(R)

定义：
f(p) = φ^{-1}(p)

对偶：
- 环同态 ←→ 概形态射
- 范畴等价
```

---

### 4.2 局部化

**局部化对应开子集**：

```
f ∈ R
局部化R_f
    ↓
开集D(f) ⊆ Spec(R)

性质：
Spec(R_f) ≅ D(f)
```

## 五、纤维积

### 5.1 定义

**仿射概形的纤维积**：

```
Spec(A) → Spec(R) ← Spec(B)

纤维积：
Spec(A) ×_{Spec(R)} Spec(B)
= Spec(A ⊗_R B)

张量积对应纤维积
```

**基变化**：

```

Spec(A) ×_{Spec(R)} Spec(S)
= Spec(A ⊗_R S)
```

## 十、数学公式总结

### 核心公式

1. **Spec定义**：
   $$\text{Spec}(R) = \{\mathfrak{p} \subseteq R : \mathfrak{p} \text{ 是素理想}\}$$

2. **Zariski拓扑**：
   $$V(I) = \{\mathfrak{p} \in \text{Spec}(R) : I \subseteq \mathfrak{p}\}$$

3. **开集基**：
   $$D(f) = \text{Spec}(R) \setminus V((f))$$

4. **结构层**：
   $$\mathcal{O}_{\text{Spec}(R)}(D(f)) = R_f = R[f^{-1}]$$

5. **局部化**：
   $$R_f = \left\{\frac{a}{f^n} : a \in R, n \geq 0\right\}$$

6. **概形态射**：
   $$f: \text{Spec}(S) \to \text{Spec}(R) \leftrightarrow \varphi: R \to S$$

7. **纤维积**：
   $$X \times_Z Y = \text{Spec}(R \otimes_S T)$$

8. **全局截面**：
   $$\Gamma(\text{Spec}(R), \mathcal{O}) = R$$

9. **函数域**：
   $$K(X) = \mathcal{O}_{X,\eta} \text{（$\eta$ 是一般点）}$$

10. **点与理想对应**：
    $$\text{点} \leftrightarrow \text{素理想}, \quad \text{闭点} \leftrightarrow \text{极大理想}$$

11. **仿射概形的泛性质**：
    仿射概形的泛性质：
    $$\text{Hom}(\text{Spec}(S), \text{Spec}(R)) \cong \text{Hom}(R, S)$$

12. **结构层的局部化**：
    结构层的局部化：
    $$\mathcal{O}_{X,x} = \varinjlim_{x \in U} \mathcal{O}_X(U) = R_{\mathfrak{p}_x}$$

13. **仿射概形的分离性**：
    仿射概形的分离性：
    $$\text{Spec}(R) \text{ 是分离概形}$$

14. **仿射概形的Noether性**：
    仿射概形的Noether性：
    $$\text{Spec}(R) \text{ Noether } \iff R \text{ Noether环}$$

15. **仿射概形与簇的关系**：
    仿射概形与簇的关系：
    $$\text{仿射簇} \subset \text{仿射概形}, \quad \text{概形理论统一了代数几何}$$

## 十一、仿射概形的详细数学表述

### 11.1 仿射概形的泛性质

**泛性质**：

仿射概形满足**泛性质**：
$$\text{Hom}(\text{Spec}(S), \text{Spec}(R)) \cong \text{Hom}(R, S)$$

即概形态射对应环同态。

**证明思路**：

使用结构层的定义和局部化性质证明。

**数学公式**：
- 泛性质: $$\text{Hom}(\text{Spec}(S), \text{Spec}(R)) \cong \text{Hom}(R, S)$$
- 环同态对应: $$f: \text{Spec}(S) \to \text{Spec}(R) \leftrightarrow \varphi: R \to S$$
- 结构层对应: $$f^\#: \mathcal{O}_{\text{Spec}(R)} \to f_* \mathcal{O}_{\text{Spec}(S)}$$

---

### 11.2 结构层的局部化

**局部化构造**：

结构层在点$x$处的**局部化**定义为：
$$\mathcal{O}_{X,x} = \varinjlim_{x \in U} \mathcal{O}_X(U) = R_{\mathfrak{p}_x}$$

其中$\mathfrak{p}_x$是点$x$对应的素理想。

**局部环**：

局部化$\mathcal{O}_{X,x}$是**局部环**，极大理想是$\mathfrak{p}_x R_{\mathfrak{p}_x}$。

**数学公式**：
- 局部化: $$\mathcal{O}_{X,x} = \varinjlim_{x \in U} \mathcal{O}_X(U) = R_{\mathfrak{p}_x}$$
- 局部环: $$\mathcal{O}_{X,x} \text{ 是局部环}$$
- 极大理想: $$\mathfrak{m}_x = \mathfrak{p}_x R_{\mathfrak{p}_x}$$

---

### 11.3 仿射概形的性质

**分离性**：

仿射概形$\text{Spec}(R)$是**分离概形**：
$$\text{Spec}(R) \text{ 是分离概形}$$

**Noether性**：

仿射概形$\text{Spec}(R)$是**Noether概形**当且仅当$R$是Noether环：
$$\text{Spec}(R) \text{ Noether } \iff R \text{ Noether环}$$

**与簇的关系**：

仿射概形理论统一了代数几何：
$$\text{仿射簇} \subset \text{仿射概形}, \quad \text{概形理论统一了代数几何}$$

**数学公式**：
- 分离性: $$\text{Spec}(R) \text{ 是分离概形}$$
- Noether性: $$\text{Spec}(R) \text{ Noether } \iff R \text{ Noether环}$$
- 与簇关系: $$\text{仿射簇} \subset \text{仿射概形}$$
