   $$\text{emb dim}(O_{X,x}) = \dim_{k(x)}(\mathfrak{m}_x/\mathfrak{m}_x^2)$$

3. **光滑性定义**：
   $$X \text{ 在 $x$ 光滑 } \iff \text{正则} + k(x)/k \text{ 可分扩展}$$

4. **特征0情况**：
   $$\text{char}(k) = 0 \Rightarrow \text{正则} \iff \text{光滑}$$

## 十一、光滑概形与正则概形的详细数学表述

### 11.1 正则性的等价条件

**等价条件**：

局部环$O_{X,x}$是**正则的**，如果：
$$O_{X,x} \text{ 正则 } \iff \text{gl dim}(O_{X,x}) < \infty$$

其中$\text{gl dim}$是整体维数。

**嵌入维数**：

**嵌入维数**定义为：
$$\text{emb dim}(O_{X,x}) = \dim_{k(x)}(\mathfrak{m}_x/\mathfrak{m}_x^2)$$

**数学公式**：
- 正则性: $$O_{X,x} \text{ 正则 } \iff \text{gl dim}(O_{X,x}) < \infty$$
- 嵌入维数: $$\text{emb dim}(O_{X,x}) = \dim_{k(x)}(\mathfrak{m}_x/\mathfrak{m}_x^2)$$
- 维数关系: $$\text{emb dim}(O_{X,x}) \geq \dim O_{X,x}$$

---

### 11.2 光滑性的Kähler微分判定与Jacobian准则

**Kähler微分判定**：

概形$X$是**光滑的**，如果：
$$X \text{ 光滑 } \iff \Omega_{X/k}^1 \text{ 局部自由}$$

**Jacobian准则**：

对于$X = V(f_1, \ldots, f_r)$，**Jacobian准则**断言：
$$X \text{ 光滑 } \iff \text{rank}(\partial f_i/\partial x_j) = r$$

**数学公式**：
- Kähler微分判定: $$X \text{ 光滑 } \iff \Omega_{X/k}^1 \text{ 局部自由}$$
- Jacobian准则: $$X = V(f_1, \ldots, f_r) \text{ 光滑 } \iff \text{rank}(\partial f_i/\partial x_j) = r$$
- 光滑性: $$\text{光滑 = 正则 + 可分扩展}$$

---

### 11.3 特征0与特征p的区别

**特征0情况**：

在**特征0**中，正则性与光滑性等价：
$$\text{char}(k) = 0 \Rightarrow \text{正则} \iff \text{光滑}$$

**特征p情况**：

在**特征p**中，光滑性强于正则性：
$$\text{char}(k) = p > 0 \Rightarrow \text{光滑} \Rightarrow \text{正则}$$

**局部性质**：

光滑性是**局部性质**：
$$X \text{ 光滑 } \iff X|_U \text{ 光滑（对所有仿射开 $U$）}$$

**数学公式**：
- 特征0: $$\text{char}(k) = 0 \Rightarrow \text{正则} \iff \text{光滑}$$
- 特征p: $$\text{char}(k) = p > 0 \Rightarrow \text{光滑} \Rightarrow \text{正则}$$
- 局部性质: $$X \text{ 光滑 } \iff X|_U \text{ 光滑（对所有仿射开 $U$）}$$
