## 一、相对概形的定义

### 1.1 S上的概形

**相对概形X/S**：

```
定义：
概形X是S上的概形，若
存在态射X → S

记作：X/S

意义：
- 相对观点
- 基变化自然
```

---

### 1.2 相对范畴

```
对象：概形态射X → S

性质：

- 纤维积存在
- 相对观点实现
```

## 三、纤维

### 3.1 纤维定义

**纤维X_s**：

```
f: X → S
点s ∈ S

纤维：
X_s = X ×_S {s}

性质：
- 几何直观
- 相对性质
```

---

### 3.2 几何直观

**纤维的几何**：

```
椭圆曲线族E → S

对s ∈ S：
纤维E_s = E ×_S {s}

意义：
- 在每个点上的曲线
- 相对观点
```

## 四、基变化

### 4.1 定义

**基变化**：

```
X → S
基变化到T：

X_T = X ×_S T

性质：
- 相对观点
- 纤维变化
```

---

### 4.2 约化

**模p约化**：

```
X → Spec(ℤ)

约化到Spec(F_p)：
X_p = X ×_{Spec(ℤ)} Spec(F_p)

意义：
- 模p约化
- 算术应用
```

## 五、相对上同调

### 5.1 推前层

**推前层f_*F**：

```
f: X → S
层F在X上

推前层f_*F在S上：
(f_*F)(U) = F(f^{-1}(U))

性质：
- 层结构
- 相对上同调
```

---

### 5.2 相对上同调

f: X → S
