   $$O_{X,x} = \varinjlim_{x \in U} O_X(U)$$

3. **完整化环**：
   $$\hat{O}_{X,x} = \varprojlim_n O_{X,x} / \mathfrak{m}_x^n$$

4. **形式概形**：
   $$\text{Spf}(\hat{O}_{X,x}) = \varinjlim_n \text{Spec}(O_{X,x} / \mathfrak{m}_x^n)$$

5. **完整化与局部化关系**：
   $$\hat{O}_{X,x} = O_{X,x} \otimes_{O_{X,x}} \hat{O}_{X,x}$$

   $$\text{形式邻域 } \hat{X}_x = \text{Spf}(\hat{O}_{X,x})$$

   $$\dim \hat{O}_{X,x} = \dim O_{X,x}$$

8. **完整化与正规性**：
   $$O_{X,x} \text{ 正规 } \iff \hat{O}_{X,x} \text{ 正规}$$
