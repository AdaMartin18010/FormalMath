# 概形定义与构造：从仿射到一般

## 一、概形的定义

### 1.1 环化空间

**环化空间(X, O_X)**：

```
数据：
- 拓扑空间X
- 层O_X（结构层）

条件：
- O_X是交换环层
- 对每个开集U，O_X(U)是环
- 限制映射是环同态
```
---

### 1.2 概形条件

**概形**：

```
环化空间(X, O_X)是概形，若
对每个点x ∈ X，存在开邻域U
使得(U, O_X|_U)同构于仿射概形

即：
局部仿射
```
---

## 二、概形的构造

### 2.1 从仿射粘接

**粘接概形**：

```
开覆盖：
X = ∪_i U_i，U_i仿射

粘接数据：
- 同构φ_{ij}: U_i ∩ U_j → U_j ∩ U_i
- 相容性条件

结果：
概形X
```
---

### 2.2 从环构造

**概形的构造**：

```
方法1：从仿射粘接
方法2：从层构造
方法3：从函子构造

统一：
所有方法等价
```
---

## 三、概形态射

### 3.1 定义

**概形态射f: X → Y**：

```
数据：
- 连续映射f: X → Y
- 层同态f#: O_Y → f_*O_X

条件：
- 相容性
- 局部性质
```
---

### 3.2 局部性质

**局部性质**：

```
对每个x ∈ X，y = f(x)

局部环同态：
f#_y: O_{Y,y} → O_{X,x}

性质：
- 局部化
- 茎的性质
```
---

## 四、纤维积

### 4.1 构造

**概形的纤维积**：

```
X → S ← Y

纤维积：
X ×_S Y

构造：
- 仿射情况：张量积
- 一般情况：粘接
```
---

### 4.2 普遍性质

**纤维积的普遍性质**：

```
对任意T和映射：
T → X, T → Y（在S上相容）

存在唯一：
T → X ×_S Y

唯一性：
满足此性质的唯一
```
---

## 五、基变化

### 5.1 定义

**基变化**：

```
X → S
基变化到T：

X_T = X ×_S T

性质：
- 相对观点
- 纤维变化
```
---

### 5.2 例子

**约化**：

```
X → Spec(ℤ)
约化到Spec(F_p)：

X_p = X ×_{Spec(ℤ)} Spec(F_p)

意义：
- 模p约化
- 算术应用
```
---

## 六、相对概形

### 6.1 S上的概形

**相对概形X → S**：

```
X是S上的概形，若
存在态射X → S

记作：X/S

意义：
- 相对观点
- 基变化
```
---

### 6.2 相对性质

**相对性质**：

```
f: X → S

相对性质：
- 相对分离
- 相对有限型
- 相对有限表示
- 相对平滑

意义：
- 不依赖绝对
- 相对定义
```
---

## 七、特殊概形

### 7.1 分离概形

**分离概形**：

```
概形X分离，若
对角嵌入：
Δ: X → X ×_Z X

是闭嵌入

意义：
- Hausdorff类比
- 几何性质
```
---

### 7.2 Noether概形

**Noether概形**：

```
概形X是Noether的，若
- X拟紧
- 每个开仿射是Noether环的谱

性质：
- 有限性
- 上同调有限维
```
