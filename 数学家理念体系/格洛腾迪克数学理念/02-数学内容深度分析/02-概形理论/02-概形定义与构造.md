## 一、概形的定义

### 1.1 环化空间

**环化空间(X, O_X)**：

```
数据：
- 拓扑空间X
- 层O_X（结构层）

条件：
- O_X是交换环层
- 对每个开集U，O_X(U)是环
- 限制映射是环同态
```

---

### 1.2 概形条件

**概形**：

```
环化空间(X, O_X)是概形，若
对每个点x ∈ X，存在开邻域U
使得(U, O_X|_U)同构于仿射概形

即：
局部仿射
```

---

## 二、概形的构造

### 2.1 从仿射粘接

**粘接概形**：

```
开覆盖：
X = ∪_i U_i，U_i仿射

粘接数据：
- 同构φ_{ij}: U_i ∩ U_j → U_j ∩ U_i
- 相容性条件

结果：
概形X
```

---

### 2.2 从环构造

**概形的构造**：

```
方法1：从仿射粘接
方法2：从层构造
方法3：从函子构造

统一：
所有方法等价
```

## 三、概形态射

### 3.1 定义

**概形态射f: X → Y**：

```
数据：
- 连续映射f: X → Y
- 层同态f#: O_Y → f_*O_X

条件：
- 相容性
- 局部性质
```

---

### 3.2 局部性质

**局部性质**：

```
对每个x ∈ X，y = f(x)

局部环同态：
f#_y: O_{Y,y} → O_{X,x}

性质：
- 局部化
- 茎的性质
```

---

## 四、纤维积

### 4.1 构造

**概形的纤维积**：

```
X → S ← Y

纤维积：
X ×_S Y

构造：
- 仿射情况：张量积
- 一般情况：粘接
```

---

### 4.2 普遍性质

**纤维积的普遍性质**：

```
对任意T和映射：
T → X, T → Y（在S上相容）

存在唯一：
T → X ×_S Y

唯一性：
满足此性质的唯一
```

---

## 五、基变化

### 5.1 定义

**基变化**：

```
X → S
基变化到T：

X_T = X ×_S T

性质：
- 相对观点
- 纤维变化
```

---

### 5.2 例子

**约化**：

```
X → Spec(ℤ)
约化到Spec(F_p)：

X_p = X ×_{Spec(ℤ)} Spec(F_p)

意义：
- 模p约化
- 算术应用
```

## 七、特殊概形

### 7.1 分离概形

**分离概形**：

```
概形X分离，若
对角嵌入：
Δ: X → X ×_Z X

是闭嵌入

意义：
- Hausdorff类比
- 几何性质
```

---

### 7.2 Noether概形

**Noether概形**：

```
概形X是Noether的，若
- X拟紧
- 每个开仿射是Noether环的谱

性质：
- 有限性
- 上同调有限维
```

## 十、概形构造的详细数学表述

### 10.1 概形的粘接构造

**粘接方法**：

概形可以通过**粘接**仿射概形构造：
$$X = \bigcup_i U_i, \quad U_i \cong \text{Spec}(R_i), \quad U_i \cap U_j \text{ 开子概形}$$

**粘接条件**：

粘接需要满足：
- $U_i \cap U_j$是$U_i$和$U_j$的开子概形
- 结构层在交上一致

**数学公式**：
- 粘接: $$X = \bigcup_i U_i, \quad U_i \cong \text{Spec}(R_i)$$
- 开子概形: $$U_i \cap U_j \text{ 是开子概形}$$
- 结构层一致: $$\mathcal{O}_X|_{U_i \cap U_j} \text{ 一致}$$

---

### 10.2 概形的分离性与Noether性

**分离性**：

概形$X$是**分离的**，如果对角映射是闭嵌入：
$$\Delta: X \to X \times_S X \text{ 是闭嵌入} \iff X \text{ 分离}$$

**Noether性**：

概形$X$是**Noether概形**，如果：
$$X = \bigcup_i \text{Spec}(R_i), \quad R_i \text{ Noether环} \iff X \text{ Noether概形}$$

**数学公式**：
- 分离性: $$\Delta: X \to X \times_S X \text{ 是闭嵌入}$$
- Noether性: $$X = \bigcup_i \text{Spec}(R_i), \quad R_i \text{ Noether环}$$
- 条件: $$\text{分离概形} + \text{Noether概形} = \text{好概形}$$

---

### 10.3 相对概形的基变化与纤维

**基变化**：

对于相对概形$f: X \to S$和态射$g: T \to S$，**基变化**定义为：
$$X_T = X \times_S T, \quad f_T: X_T \to T$$

**纤维**：

对于点$s \in S$，**纤维**定义为：
$$X_s = X \times_S \text{Spec}(k(s)) = f^{-1}(s)$$

**数学公式**：
- 基变化: $$X_T = X \times_S T, \quad f_T: X_T \to T$$
- 纤维: $$X_s = X \times_S \text{Spec}(k(s)) = f^{-1}(s)$$
- 基变化保持性质: $$\text{基变化保持分离性、有限型等性质}$$
