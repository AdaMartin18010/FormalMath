# 微分形式与对偶层：概形上的微分几何

## 一、微分形式层

### 1.1 1-形式层

**1-形式层Ω^1_{X/S}**：

```
概形态射f: X → S

1-形式层：
Ω^1_{X/S} = I/I^2

其中I是X ×_S X中对角的理想层

性质：
- O_X模层
- 局部自由（光滑）
- 应用广泛
```

---

### 1.2 高次形式

**p-形式层Ω^p_{X/S}**：

```
概形态射f: X → S

p-形式层：
Ω^p_{X/S} = ∧^p Ω^1_{X/S}

性质：
- O_X模层
- 外微分
- 应用广泛
```

---

## 二、对偶层

### 2.1 对偶模

**对偶模**：

```
O_X模层F

对偶层：
F^∨ = Hom_{O_X}(F, O_X)

性质：
- O_X模层
- 函子性
- 应用广泛
```

---

### 2.2 对偶性

**对偶性**：

```
局部自由层E

对偶：
E^∨ ⊗ E → O_X

意义：
- 对偶理论
- 应用广泛
```

---

## 三、典范层

概形X

ω_X = ∧^n Ω^1_{X/k}

意义：

## 四、Serre对偶

### 4.1 Serre对偶定理

**Serre对偶**：

```
射影概形X
维度n

Serre对偶：
H^i(X, F) ≅ H^{n-i}(X, F^∨ ⊗ ω_X)^∨

意义：
- 对偶理论
- 应用广泛
```

## 五、de Rham复形

### 5.1 定义

**de Rham复形**：

```
概形X

de Rham复形：
Ω^*_X:
O_X → Ω^1_X → Ω^2_X → ...

外微分：
d: Ω^p → Ω^{p+1}

性质：
d^2 = 0
```

---

### 5.2 上同调

**de Rham上同调**：

```
de Rham上同调：
H^*_{dR}(X) = H^*(Ω^*_X)

意义：
- 几何不变量
- 应用广泛
```
