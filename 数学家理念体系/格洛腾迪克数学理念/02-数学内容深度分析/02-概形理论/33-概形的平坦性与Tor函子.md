## 八、数学公式总结

### 核心公式

1. **平坦性定义**：
   $$M \text{ 平坦 } \iff \text{Tor}_i(M, N) = 0 \text{（$i > 0$，对所有 $N$）}$$

2. **平坦态射**：
   $$f: X \to Y \text{ 平坦 } \iff O_X \text{ 是 $f^{-1}O_Y$-平坦模}$$

3. **Tor函子定义**：
   $$\text{Tor}_i^{O_X}(M, N) = H_i(M \otimes P_N^\bullet), \quad P_N^\bullet \text{ 平坦分解}$$

4. **平坦性与Tor**：
   $$M \text{ 平坦 } \iff \text{Tor}_1(M, N) = 0 \text{（对所有 $N$）}$$

5. **平坦性与纤维**：
   $$f \text{ 平坦 } \Rightarrow \dim X_s = \dim X - \dim S \text{（某些条件下）}$$

6. **Tor与张量积**：
   $$\text{Tor}_0(M, N) = M \otimes N$$

7. **Tor长正合列**：
   $$0 \to \text{Tor}_1(M, N) \to M \otimes P \to M \otimes N \to 0 \text{（某些条件下）}$$

8. **平坦性与基变化**：
   $$f \text{ 平坦 } \Rightarrow f_{S'} \text{ 平坦（$S' \to S$）}$$

9. **Tor与拉回**：
   $$f^*(\text{Tor}_i(M, N)) \cong \text{Tor}_i(f^*M, f^*N) \text{（某些条件下）}$$

## 九、平坦性与Tor函子的详细数学表述

### 9.1 平坦性的严格定义

**平坦模定义**：

$\mathcal{O}_X$-模 $M$ 是**平坦的**当且仅当：
$$\text{Tor}_i^{\mathcal{O}_X}(M, N) = 0 \quad \text{对所有 } i > 0, \text{ 所有 } N$$

**平坦态射**：

态射 $f: X \to Y$ 是**平坦的**当且仅当 $\mathcal{O}_X$ 是 $f^{-1}\mathcal{O}_Y$-平坦模。

**数学公式**：
- 平坦模：$$M \text{ 平坦 } \iff \text{Tor}_i(M, N) = 0 \quad (i > 0)$$
- 平坦态射：$$f \text{ 平坦 } \iff \mathcal{O}_X \text{ 是 } f^{-1}\mathcal{O}_Y\text{-平坦}$$
- 等价条件：$$M \text{ 平坦 } \iff \text{Tor}_1(M, N) = 0 \quad \text{（对所有 } N\text{）}$$

### 9.2 Tor函子的严格定义

**Tor函子定义**：

**Tor函子** $\text{Tor}_i^{\mathcal{O}_X}(M, N)$ 定义为：
$$\text{Tor}_i^{\mathcal{O}_X}(M, N) = H_i(M \otimes_{\mathcal{O}_X} P_N^\bullet)$$

其中 $P_N^\bullet$ 是 $N$ 的平坦分解。

**数学公式**：
- Tor函子：$$\text{Tor}_i(M, N) = H_i(M \otimes P_N^\bullet)$$
- Tor_0：$$\text{Tor}_0(M, N) = M \otimes N$$
- 长正合列：$$\cdots \to \text{Tor}_1(M, N) \to M \otimes P \to M \otimes N \to 0$$
