**约化概形**：

```

X约化，若
对所有开集U，O_X(U)约化环

约化环：
没有非零幂零元

意义：
- 几何性质
- 应用广泛
```

## 九、数学公式总结

### 核心公式

1. **约化概形定义**：
   $$X \text{ 约化 } \iff O_{X,x} \text{ 无幂零元（对所有 $x$）}$$

2. **约化概形构造**：
   $$X_{\text{red}} = (|X|, O_X / \sqrt{(0)}), \quad \sqrt{(0)} = \{f : f^n = 0 \text{ 对某个 $n$}\}$$

3. **正规化定义**：
   $$X^\nu = \text{Spec}(\text{整闭包}(O_X(X))) \to X$$

4. **正规性条件**：
   $$O_{X,x} \text{ 正规 } \iff O_{X,x} = \text{整闭包}(O_{X,x} \text{ 在 } K(X))$$

5. **正规化泛性质**：
   $$\text{Hom}(Y, X^\nu) \cong \{\text{在 $X$ 上整的态射 } Y \to X\}$$

6. **约化泛性质**：
   $$\text{Hom}(Y_{\text{red}}, X) \cong \text{Hom}(Y, X) \text{（$X$ 约化）}$$

7. **正规化与维数**：
   $$\dim X^\nu = \dim X$$

8. **约化与上同调**：
   $$H^i(X_{\text{red}}, \mathcal{F}) \cong H^i(X, \mathcal{F}) \text{（某些条件下）}$$

9. **正规化与奇点**：
   $$X^\nu \text{ 正规，$X^\nu \to X$ 双有理且有限}$$

10. **标准化过程**：
    $$X \to X_{\text{red}} \to (X_{\text{red}})^\nu = X^{\text{norm}}$$
