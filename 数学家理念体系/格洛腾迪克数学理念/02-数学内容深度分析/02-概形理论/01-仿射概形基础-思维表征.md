## 二、多维概念对比矩阵

### 2.1 仿射概形 vs 概形 vs 仿射代数簇

| 维度 | 仿射概形 Spec R | 概形 X（一般） | 仿射代数簇 V(I)⊂A^n |
|------|-----------------|----------------|----------------------|
| **局部模型** | 自身即 Spec(R) | 局部同构于 Spec(A) | 仿射代数集 + 正则函数环 |
| **环** | 任一交换环 R | 每开集有环、茎为局部环 | k[x_1,…,x_n]/I，k 代数闭 |
| **点** | 素理想 p | 点 x，茎 O_{X,x} 局部环 | 极大理想（k 代数闭时） |
| **几何** | 环的几何化；含非闭点 | 粘合仿射开集 | 经典代数几何、闭点为主 |

### 2.2 概念层次（仿射概形 → 概形理论）

| 概念 | 含义 | 与其它篇关系 |
|------|------|--------------|
| Spec(R)、结构层 | 环→空间，O(D(f))=R_f | 01 基础；02-概形定义与构造 |
| 态射 Spec B→Spec A | 与环同态 A→B 一一对应 | 03-纤维积与基变化、04-相对概形 |
| 纤维积 Spec(B⊗_A C) | 仿射纤维积 = 张量积谱 | 03-纤维积与基变化 |

## 五、公理–定理–证明树图（核心逻辑链）

```text
[设定] R 交换环（含单位）；Spec(R) = {素理想 p}，Zariski 拓扑；O(D(f))=R_f.

A1. D(f) 形成拓扑基；茎 O_p = R_p（局部环）；(Spec R, O) 为环层空间。
A2. 环同态 φ: A→B 给出态射 (f, f^#): Spec B→Spec A，f(p)=φ^{-1}(p)，f^# 由 φ 诱导。

        |--- 定理 T1 (仿射=环几何): 仿射概形范畴 与 交换环范畴 反等价。
        |         (证明: Spec 与 Γ(Spec R, O)=R 互为拟逆.)
        |
        |--- 定理 T2 (纤维积): Spec B ×_{Spec A} Spec C ≅ Spec(B⊗_A C).
        |         (证明: 泛性质；张量积的泛性质.)
        |
        └--- 定理 T3 (粘合): 若概形 X 有仿射开覆盖 {U_i=Spec A_i} 且粘合相容，
                 则 X 由环 A_i 及粘合数据唯一确定. (证明: 02-概形定义与构造.)
```
