# 射影概形：从射影空间到射影概形

## 一、射影空间

### 1.1 射影空间的构造

**射影空间P^n**：

```
域k上的射影空间：

P^n_k = (k^{n+1} - {0}) / k^*

等价关系：
(x_0, ..., x_n) ~ (λx_0, ..., λx_n)

意义：
- 齐次坐标
- 几何对象
- 基础构造
```
---

### 1.2 射影空间的覆盖

**仿射覆盖**：

```
射影空间P^n有仿射覆盖：

U_i = {[x_0:...:x_n] | x_i ≠ 0}

同构：
U_i ≅ A^n

意义：
- 局部仿射
- 全局射影
- 构造基础
```
---

## 二、射影概形

### 2.1 定义

**射影概形**：

```
概形X

X射影，若存在闭嵌入：
X → P^n_S

其中P^n_S是S上的射影空间

意义：
- 相对射影
- 几何性质
- 分类工具
```

## 三、射影嵌入

### 3.1 非常丰富线丛

**非常丰富线丛**：

```
线丛L在概形X上

L非常丰富，若：
L定义闭嵌入
X → P^n

条件：
- 有限型
- 分离
- 非常丰富
```
---

### 3.2 射影嵌入定理

**射影嵌入**：

```
概形X
非常丰富线丛L

存在闭嵌入：
X → P(H^0(X, L))

意义：
- 射影表示
- 几何构造
- 应用广泛
```
