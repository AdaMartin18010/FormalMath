## 一、纤维积的定义

### 1.1 概形的纤维积

**纤维积X ×_S Y**：

```
给定：
概形态射：X → S ← Y

纤维积：
X ×_S Y满足普遍性质

对任意概形T和映射：
T → X, T → Y（在S上相容）

存在唯一：
T → X ×_S Y
```

---

### 1.2 仿射情况

**仿射概形的纤维积**：

```
X = Spec(A)
Y = Spec(B)
S = Spec(R)

纤维积：
X ×_S Y = Spec(A ⊗_R B)

张量积对应纤维积
```

---

## 二、纤维积的性质

### 2.1 函子性

**纤维积的函子性**：

```
给定交换图：
X' → X
↓    ↓
Y' → Y
    ↓
    S

诱导：
X' ×_S Y' → X ×_S Y

函子性
```

---

### 2.2 结合律

**纤维积的结合律**：

```
X → Y → Z

结合律：
(X ×_Y Z) ×_Z W ≅ X ×_Y (Z ×_Z W)

同构自然
```

## 三、基变化

### 3.1 定义

**基变化**：

```
X → S
基变化到T：

X_T = X ×_S T

性质：
- 相对观点
- 纤维变化
```

---

### 3.2 约化

**模p约化**：

```
X → Spec(ℤ)

约化到Spec(F_p)：
X_p = X ×_{Spec(ℤ)} Spec(F_p)

意义：
- 模p约化
- 算术应用
```

## 四、纤维

### 4.1 纤维的定义

**纤维X_s**：

```
f: X → S
点s ∈ S

纤维：
X_s = X ×_S {s}

性质：
- 几何直观
- 相对性质
```

---

### 4.2 几何直观

**纤维的几何**：

```
椭圆曲线族E → S

对s ∈ S：
纤维E_s = E ×_S {s}

意义：
- 在每个点上的曲线
- 相对观点
```

## 十、纤维积与基变化的详细数学表述

### 10.1 纤维积的泛性质

**泛性质**：

纤维积$X \times_S Y$满足**泛性质**：
$$\text{Hom}(T, X \times_S Y) \cong \text{Hom}(T, X) \times_{\text{Hom}(T, S)} \text{Hom}(T, Y)$$

**唯一性**：

满足泛性质的纤维积是唯一的（在同构意义下）。

**数学公式**：
- 泛性质: $$\text{Hom}(T, X \times_S Y) \cong \text{Hom}(T, X) \times_{\text{Hom}(T, S)} \text{Hom}(T, Y)$$
- 唯一性: $$\text{纤维积在同构意义下唯一}$$
- 存在性: $$\text{纤维积在概形范畴中存在}$$

---

### 10.2 纤维积的结合律与交换律

**结合律**：

纤维积满足**结合律**：
$$(X \times_S Y) \times_S Z \cong X \times_S (Y \times_S Z)$$

**交换律**：

纤维积满足**交换律**：
$$X \times_S Y \cong Y \times_S X$$

**数学公式**：
- 结合律: $$(X \times_S Y) \times_S Z \cong X \times_S (Y \times_S Z)$$
- 交换律: $$X \times_S Y \cong Y \times_S X$$
- 单位元: $$X \times_S S \cong X$$

---

### 10.3 基变化与纤维的性质

**基变化的保持性质**：

基变化保持许多性质：
- **分离性**: $X$分离 $\Rightarrow$ $X_T$分离
- **有限型**: $X \to S$有限型 $\Rightarrow$ $X_T \to T$有限型
- **光滑性**: $X \to S$光滑 $\Rightarrow$ $X_T \to T$光滑

**纤维的几何性质**：

纤维$X_s = f^{-1}(s)$是$k(s)$-概形：
$$X_s = X \times_S \text{Spec}(k(s)) = f^{-1}(s) \text{ 是 $k(s)$-概形}$$

**数学公式**：
- 基变化保持: $$\text{基变化保持分离性、有限型、光滑性等性质}$$
- 纤维: $$X_s = X \times_S \text{Spec}(k(s)) = f^{-1}(s)$$
- 几何性质: $$X_s \text{ 是 $k(s)$-概形}$$
