# 纤维积与基变化：相对观点的实现

## 一、纤维积的定义

### 1.1 概形的纤维积

**纤维积X ×_S Y**：

```
给定：
概形态射：X → S ← Y

纤维积：
X ×_S Y满足普遍性质

对任意概形T和映射：
T → X, T → Y（在S上相容）

存在唯一：
T → X ×_S Y
```
---

### 1.2 仿射情况

**仿射概形的纤维积**：

```
X = Spec(A)
Y = Spec(B)
S = Spec(R)

纤维积：
X ×_S Y = Spec(A ⊗_R B)

张量积对应纤维积
```
---

## 二、纤维积的性质

### 2.1 函子性

**纤维积的函子性**：

```
给定交换图：
X' → X
↓    ↓
Y' → Y
    ↓
    S

诱导：
X' ×_S Y' → X ×_S Y

函子性
```
---

### 2.2 结合律

**纤维积的结合律**：

```
X → Y → Z

结合律：
(X ×_Y Z) ×_Z W ≅ X ×_Y (Z ×_Z W)

同构自然
```
---

## 三、基变化

### 3.1 定义

**基变化**：

```
X → S
基变化到T：

X_T = X ×_S T

性质：
- 相对观点
- 纤维变化
```
---

### 3.2 约化

**模p约化**：

```
X → Spec(ℤ)

约化到Spec(F_p)：
X_p = X ×_{Spec(ℤ)} Spec(F_p)

意义：
- 模p约化
- 算术应用
```
---

## 四、纤维

### 4.1 纤维的定义

**纤维X_s**：

```
f: X → S
点s ∈ S

纤维：
X_s = X ×_S {s}

性质：
- 几何直观
- 相对性质
```
---

### 4.2 几何直观

**纤维的几何**：

```
椭圆曲线族E → S

对s ∈ S：
纤维E_s = E ×_S {s}

意义：
- 在每个点上的曲线
- 相对观点
```
---

## 五、相对性质

### 5.1 相对分离

**相对分离**：

```
f: X → S相对分离，若
对角嵌入：
Δ_{X/S}: X → X ×_S X

是闭嵌入

意义：
- 相对Hausdorff
- 几何性质
```
---

### 5.2 相对有限型

**相对有限型**：

```
f: X → S相对有限型，若
对S的仿射开覆盖
X的拉回有限型

性质：
- 相对定义
- 局部性质
```
---

## 六、应用

```
模空间M → S
```

```
相对上同调
```
