## 八、数学公式总结

### 核心公式

1. **稳定性定义**：
   $$X \text{ 稳定 } \iff X \text{ 满足稳定性条件（几何不变量）}$$

2. **稳定条件（曲线）**：
   $$g \geq 2: \text{ 稳定 } \iff \text{ 仅有节点奇点，自同构群有限}$$

3. **稳定条件（高维）**：
   $$X \text{ 稳定 } \iff K_X \text{ 丰沛}, \quad \text{某些几何条件}$$

4. **稳定与模空间**：
   $$\overline{\mathcal{M}}_g = \{\text{稳定曲线}\} / \sim$$

5. **稳定与形变**：
   $$X \text{ 稳定 } \Rightarrow \text{ 形变空间紧致}$$

6. **稳定与上同调**：
   $$H^i(X, O_X) \text{ 有限维（$X$ 稳定）}$$

7. **稳定与Kodaira维数**：
   $$\kappa(X) \geq 0 \text{（$X$ 稳定）}$$

8. **稳定与除子**：
   $$K_X \text{ 丰沛 } \Rightarrow X \text{ 稳定（某些条件下）}$$

9. **稳定与群作用**：
   $$X/G \text{ 稳定 } \iff X \text{ 稳定且 $G$ 作用适当}$$

10. **稳定与双有理等价**：
    $$X \sim_{\text{bir}} Y, \quad X, Y \text{ 稳定 } \Rightarrow X \cong Y$$
