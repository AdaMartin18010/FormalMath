## 九、数学公式总结

### 核心公式

1. **切空间定义**：
   $$T_x X = (\mathfrak{m}_x / \mathfrak{m}_x^2)^* = \text{Hom}_{k(x)}(\mathfrak{m}_x / \mathfrak{m}_x^2, k(x))$$

2. **切空间维数**：
   $$\dim_{k(x)} T_x X = \text{emb dim}(O_{X,x}) = \dim_{k(x)}(\mathfrak{m}_x / \mathfrak{m}_x^2)$$

3. **切丛定义**：
   $$T_X = \text{Spec}(\text{Sym}(\Omega_{X/k}^1)) = \text{Spec}(\bigoplus_{n \\\geq 0} \text{Sym}^n(\Omega_{X/k}^1))$$

4. **切丛与切空间**：
   $$T_X|_x = T_x X$$

5. **对偶切丛**：
   $$T_X^* = \Omega_{X/k}^1$$

6. **切空间与微分**：
   $$df_x: T_x X \to T_{f(x)} Y, \quad df_x(v)(g) = v(g \circ f)$$

7. **切空间与Kähler微分**：
   $$T_x X \cong \text{Der}_{k(x)}(O_{X,x}, k(x))$$

8. **切空间与形变**：
   $$T_x X \text{ 参数化 $X$ 在 $x$ 的一阶形变}$$

9. **切丛与向量场**：
   $$\Gamma(X, T_X) = \text{Der}_k(O_X, O_X)$$

10. **切空间与正则性**：
    $$X \text{ 在 $x$ 正则 } \iff \dim_{k(x)} T_x X = \dim_x X$$
