## 二、多维概念对比矩阵

### 2.1 稳定 vs 半稳定 vs 不稳定

| 维度 | 稳定 | 半稳定 | 不稳定 |
|------|------|--------|--------|
| **定义** | 轨道闭、维数最大、有限稳定子 | 无坏极限、可纳入商 | 轨道不闭、坏极限 |
| **商** | 商上点对应闭轨道 | 商上点可对应多个轨道 | 商中略去 |
| **典型** | 光滑商、自由作用附近 | 商概形、可能奇异 | 排除于商 |

### 2.2 概念层次（稳定性理论 到 应用）

| 概念 | 含义 | 与其它篇关系 |
|------|------|--------------|
| GIT 稳定/半稳定 | 线性化 L、稳定子群、极限 | 23-概形的群作用与商概形、08-射影概形 |
| 模空间 | 商为粗/细模空间；Proj 不变环 | 23-群作用与商概形、25-平坦族与形变理论 |
| Mumford GIT | 半稳定商、Proj(⊕ Γ(L^{⊗n})^G) | 08-射影概形、07-除子与线丛（线性化） |

## 五、公理–定理–证明树图（核心逻辑链）

```text
A1. 线性化：群 G 作用 X、线丛 L；线性化为 G 作用在 L 上、覆盖 X 上作用。
A2. 半稳定点 X^ss：x 半稳定 当且仅当 存在不变截面 s、x∈D_+(s)、轨道在 D_+(s) 中闭。
T1. GIT 商：X^ss//G = Proj(⊕_{n≥0} Γ(X, L^{⊗n})^G) 为拟射影商概形。(Mumford GIT；23-群作用与商概形)
T2. 稳定点商：X^s/G 为几何商、轨道→点。(23-群作用与商概形)
```
