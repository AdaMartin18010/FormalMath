### 1.1 定义

**概形维数dim X**：

```
概形X

Krull维数：
dim X = sup{链的长度}
P_0 ⊂ P_1 ⊂ ... ⊂ P_n

其中P_i是X的闭点或不可约闭子集

意义：
- 几何维数
```

## 三、维数的计算

### 3.1 仿射概形

**仿射概形的维数**：

```
仿射概形X = Spec A

dim X = dim A

其中dim A是环A的Krull维数

意义：
- 代数计算
- 应用广泛
```

---

### 3.2 一般概形

**一般概形的维数**：

```
概形X

dim X = sup_{x ∈ X} dim O_{X,x}

意义：
- 局部到全局
- 应用广泛
```
