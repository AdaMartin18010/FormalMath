## 十一、概形的维数理论的详细数学表述

### 11.1 Krull维数与局部维数

**Krull维数**：

概形$X$的**Krull维数**定义为：
$$\dim X = \sup\{n : \exists \text{ 链 } Z_0 \subset Z_1 \subset \cdots \subset Z_n, Z_i \text{ 不可约闭}\}$$

**局部维数**：

在点$x$处的**局部维数**定义为：
$$\dim_x X = \dim O_{X,x} = \text{Krull dim}(O_{X,x})$$

**关系**：

全局维数是局部维数的上确界：
$$\dim X = \sup_{x \in X} \dim_x X$$

**数学公式**：
- Krull维数: $$\dim X = \sup\{n : \exists \text{ 链 } Z_0 \subset Z_1 \subset \cdots \subset Z_n\}$$
- 局部维数: $$\dim_x X = \dim O_{X,x}$$
- 关系: $$\dim X = \sup_{x \in X} \dim_x X$$

---

### 11.2 相对维数与维数公式

**相对维数**：

对于态射$f: X \to Y$，**相对维数**定义为：
$$\dim_{f(x)} f = \dim O_{X,x} / \mathfrak{m}_{f(x)} O_{X,x}$$

**维数公式**：

在某些条件下（如$f$平坦且有限型），有**维数公式**：
$$\dim X = \dim Y + \dim_{f(x)} f \text{（$f: X \to Y$ 某些条件下）}$$

**数学公式**：
- 相对维数: $$\dim_{f(x)} f = \dim O_{X,x} / \mathfrak{m}_{f(x)} O_{X,x}$$
- 维数公式: $$\dim X = \dim Y + \dim_{f(x)} f$$
- 条件: $$\text{维数公式在 $f$ 平坦且有限型时成立}$$

---

### 11.3 维数与上同调的关系

**上同调消失**：

对于某些层$\mathcal{F}$，上同调在维数以上消失：
$$H^i(X, \mathcal{F}) = 0 \text{（$i > \dim X$，某些条件下）}$$

**维数与切空间**：

对于正则概形，维数等于切空间的维数：
$$\dim_x X = \dim_{k(x)} T_x X \text{（$X$ 在 $x$ 正则）}$$

**数学公式**：
- 上同调消失: $$H^i(X, \mathcal{F}) = 0 \text{（$i > \dim X$）}$$
- 维数与切空间: $$\dim_x X = \dim_{k(x)} T_x X \text{（$X$ 在 $x$ 正则）}$$
