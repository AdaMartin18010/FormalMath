## 一、闭嵌入

### 1.1 定义

**闭嵌入**：

```
概形态射f: X → Y

f是闭嵌入，若：
- 拓扑嵌入（像闭）
- O_Y → f_* O_X满射

等价：
- 像 = V(I)（I是理想）
- X ≅ Spec(O_Y / I)

意义：
- 闭子概形
- 应用广泛
```

## 二、开嵌入

## 三、浸入

### 3.1 定义

**浸入**：

```
概形态射f: X → Y

f是浸入，若：
f = j ∘ i

其中：
- i: X → Z闭嵌入
- j: Z → Y开嵌入

意义：
- 局部闭子概形
- 应用广泛
```

   $$f: X \to Y \text{ 闭嵌入 } \iff X \cong \text{Spec}(O_Y / I), \quad I \text{ 理想层}$$

2. **开嵌入定义**：
   $$f: U \to X \text{ 开嵌入 } \iff U \subset X \text{ 开}, \quad O_X|_U \cong O_U$$

3. **浸入定义**：
   $$f: X \to Y \text{ 浸入 } \iff f = j \circ i, \quad i \text{ 闭嵌入}, j \text{ 开嵌入}$$

4. **闭嵌入与理想**：
   $$X \hookrightarrow Y \text{ 闭嵌入 } \iff X = V(I), \quad I \subset O_Y \text{ 理想层}$$

   $$U \hookrightarrow X \text{ 开嵌入 } \iff U = D(f), \quad f \in O_X(X)$$

6. **浸入与切空间**：
   $$f: X \to Y \text{ 浸入 } \Rightarrow df_x: T_x X \hookrightarrow T_{f(x)} Y \text{ 单射}$$

7. **闭嵌入与推前**：
   $$i: X \hookrightarrow Y \text{ 闭嵌入 } \Rightarrow i_*: \text{Sh}(X) \to \text{Sh}(Y) \text{ 正合}$$

   $$j: U \hookrightarrow X \text{ 开嵌入 } \Rightarrow j^*: \text{Sh}(X) \to \text{Sh}(U) \text{ 正合}$$

## 十一、概形的嵌入与浸入的详细数学表述

### 11.1 闭嵌入与开嵌入

**闭嵌入**：

态射$f: X \to Y$是**闭嵌入**，如果：
$$f: X \to Y \text{ 闭嵌入 } \iff f \text{ 闭且单射}$$

**闭嵌入与理想**：

闭嵌入对应理想层：
$$X \hookrightarrow Y \text{ 闭嵌入 } \iff X = V(I), \quad I \subset O_Y \text{ 理想层}$$

**开嵌入**：

态射$f: U \to X$是**开嵌入**，如果：
$$f: U \to X \text{ 开嵌入 } \iff U \text{ 是 $X$ 的开子概形}$$

**数学公式**：
- 闭嵌入: $$f: X \to Y \text{ 闭嵌入 } \iff f \text{ 闭且单射}$$
- 理想对应: $$X = V(I), \quad I \subset O_Y \text{ 理想层}$$
- 开嵌入: $$U \hookrightarrow X \text{ 开嵌入 } \iff U \text{ 是开子概形}$$

---

### 11.2 浸入的定义

**浸入**：

态射$f: X \to Y$是**浸入**，如果：
$$f: X \to Y \text{ 浸入 } \iff f = j \circ i, \quad i \text{ 闭嵌入}, j \text{ 开嵌入}$$

**浸入与切空间**：

浸入诱导切空间的单射：
$$f: X \to Y \text{ 浸入 } \Rightarrow df_x: T_x X \hookrightarrow T_{f(x)} Y \text{ 单射}$$

**数学公式**：
- 浸入: $$f: X \to Y \text{ 浸入 } \iff f = j \circ i$$
- 切空间: $$df_x: T_x X \hookrightarrow T_{f(x)} Y \text{ 单射}$$
- 分解: $$f = j \circ i, \quad i \text{ 闭嵌入}, j \text{ 开嵌入}$$

---

### 11.3 嵌入与上同调

**闭嵌入与推前**：

闭嵌入$i: X \hookrightarrow Y$的推前函子是**正合的**：
$$i: X \hookrightarrow Y \text{ 闭嵌入 } \Rightarrow i_*: \text{Sh}(X) \to \text{Sh}(Y) \text{ 正合}$$

**开嵌入与拉回**：

开嵌入$j: U \hookrightarrow X$的拉回函子是**正合的**：
$$j: U \hookrightarrow X \text{ 开嵌入 } \Rightarrow j^*: \text{Sh}(X) \to \text{Sh}(U) \text{ 正合}$$

**上同调关系**：

对于闭嵌入，有：
$$H^i(X, \mathcal{F}) \cong H^i(Y, f_*\mathcal{F}) \text{（$f$ 闭嵌入）}$$

**数学公式**：
- 推前正合: $$i_*: \text{Sh}(X) \to \text{Sh}(Y) \text{ 正合}$$
- 拉回正合: $$j^*: \text{Sh}(X) \to \text{Sh}(U) \text{ 正合}$$
- 上同调: $$H^i(X, \mathcal{F}) \cong H^i(Y, f_*\mathcal{F})$$
