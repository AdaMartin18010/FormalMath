# 概形的嵌入与浸入：几何映射的代数化

## 一、闭嵌入

### 1.1 定义

**闭嵌入**：

```
概形态射f: X → Y

f是闭嵌入，若：
- 拓扑嵌入（像闭）
- O_Y → f_* O_X满射

等价：
- 像 = V(I)（I是理想）
- X ≅ Spec(O_Y / I)

意义：
- 闭子概形
- 应用广泛
```

## 二、开嵌入

### 2.1 定义

**开嵌入**：

```
概形态射f: U → X

f是开嵌入，若：
- 拓扑嵌入（像开）
- O_X|_U → O_U同构

意义：
- 开子概形
- 应用广泛
```

## 三、浸入

### 3.1 定义

**浸入**：

```
概形态射f: X → Y

f是浸入，若：
f = j ∘ i

其中：
- i: X → Z闭嵌入
- j: Z → Y开嵌入

意义：
- 局部闭子概形
- 应用广泛
```
