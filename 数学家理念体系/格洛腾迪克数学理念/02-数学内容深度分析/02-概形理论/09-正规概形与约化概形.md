## 九、数学公式总结

### 核心公式

1. **正规性定义**：
   $$X \text{ 在点 } x \text{ 正规 } \iff O_{X,x} \text{ 是整闭整环}$$

2. **正规化**：
   $$X^\nu = \text{Spec}(\text{整闭包}(O_X(X))) \to X$$

3. **约化性定义**：
   $$X \text{ 约化 } \iff O_{X,x} \text{ 无幂零元（对所有 $x$）}$$

4. **约化概形**：
   $$X_{\text{red}} = (|X|, O_X/\sqrt{(0)})$$

5. **Cohen-Macaulay条件**：
   $$\text{depth}(O_{X,x}) = \dim(O_{X,x})$$

6. **Gorenstein条件**：
   $$\text{Ext}^i(O_{X,x}, O_{X,x}) = \begin{cases} k(x) & i = \dim X \\ 0 & \text{其他} \end{cases}$$

7. **正规化泛性质**：
   $$\text{Hom}(Y, X^\nu) \cong \{\text{在 $X$ 上整的态射 } Y \to X\}$$

8. **Serre条件**：
   $$R_k: \text{正则在余维数 $\leq k$ 的点}, \quad S_k: \text{深度 $\geq \min(k, \dim)$}$$

9. **正规性判据**：
   $$X \text{ 正规 } \iff R_1 + S_2$$

10. **约化性判据**：
    $$X \text{ 约化 } \iff \text{对所有 $x$, } \sqrt{(0)} = (0) \text{ 在 } O_{X,x}$$

11. **正规性的等价定义**：
    正规性的等价定义：
    $$X \text{ 正规 } \iff O_{X,x} \text{ 整闭（对所有 $x$）}$$

12. **正规化的唯一性**：
    正规化的唯一性：
    $$X^\nu \text{ 在同构意义下唯一}$$

13. **Serre条件的应用**：
    Serre条件的应用：
    $$R_k + S_k \text{ 条件用于研究正规性和奇点}$$

14. **约化概形的构造**：
    约化概形的构造：
    $$X_{\text{red}} = (|X|, O_X/\sqrt{(0)}) \text{ 是约化概形}$$

15. **正规性与约化性的关系**：
    正规性与约化性的关系：

## 十一、正规概形与约化概形的详细数学表述

### 11.1 正规性的等价定义

**等价定义**：

概形$X$是**正规的**，如果：
$$X \text{ 正规 } \iff O_{X,x} \text{ 整闭（对所有 $x$）}$$

**整闭性**：

局部环$O_{X,x}$是**整闭的**，如果它在分式域中是整闭的。

**数学公式**：
- 正规性: $$X \text{ 正规 } \iff O_{X,x} \text{ 整闭（对所有 $x$）}$$
- 整闭性: $$\text{整闭 = 在分式域中整闭}$$
- 正规性判据: $$X \text{ 正规 } \iff R_1 + S_2$$

---

### 11.2 正规化的唯一性与Serre条件

**正规化的唯一性**：

**正规化**$X^\nu$在同构意义下**唯一**：
$$X^\nu \text{ 在同构意义下唯一}$$

**Serre条件**：

**Serre条件**用于研究正规性：
- $R_k$：正则在余维数$\leq k$的点
- $S_k$：深度$\geq \min(k, \dim)$

**数学公式**：
- 唯一性: $$X^\nu \text{ 在同构意义下唯一}$$
- Serre条件: $$R_k: \text{正则在余维数 $\leq k$ 的点}, \quad S_k: \text{深度 $\geq \min(k, \dim)$}$$
- 正规性判据: $$X \text{ 正规 } \iff R_1 + S_2$$

---

### 11.3 约化概形的构造与性质

**约化概形的构造**：

**约化概形**$X_{\text{red}}$定义为：
$$X_{\text{red}} = (|X|, O_X/\sqrt{(0)}) \text{ 是约化概形}$$

**正规性与约化性的关系**：

**正规概形是约化的**，但反之不成立：
$$\text{正规概形是约化的，但反之不成立}$$

**数学公式**：
- 约化概形: $$X_{\text{red}} = (|X|, O_X/\sqrt{(0)})$$
- 约化性: $$X \text{ 约化 } \iff \sqrt{(0)} = (0) \text{ 在 } O_{X,x}$$
- 关系: $$\text{正规 } \Rightarrow \text{约化，但反之不成立}$$
