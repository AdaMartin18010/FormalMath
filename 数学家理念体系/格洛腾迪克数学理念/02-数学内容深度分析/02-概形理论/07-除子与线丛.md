# 除子与线丛：概形上的几何对象

## 一、除子的定义

### 1.1 Weil除子

**Weil除子**：

```
概形X上的Weil除子：

D = ∑ n_i Z_i

其中：
- Z_i是X的不可约闭子概形
- n_i是整数
- 和是局部有限的

意义：
- 编码几何信息
- 分类工具
- 几何不变量
```

---

### 1.2 Cartier除子

**Cartier除子**：

```
概形X上的Cartier除子：

D由开覆盖{U_i}和f_i ∈ K(X)^*定义
在交U_i ∩ U_j上：f_i/f_j ∈ O_{X,ij}^*

意义：
- 局部定义
- 更灵活
- 与线丛对应
```

---

## 二、线丛

### 2.1 可逆层

**可逆层（线丛）**：

```
概形X上的可逆层L：

可逆层：
- O_X模层
- 局部同构O_X
- 秩1

对应：
- 线丛
- 除子类
- 几何对象
```

---

### 2.2 除子与线丛对应

**对应关系**：

```
Cartier除子D
    ↓
线丛O(D)

对应：
O(D)的截面 = 有理函数
满足(f) ≥ -D

双射：
Cartier除子类 ↔ Pic(X)
```

---

## 三、主除子

### 3.1 定义

**主除子**：

```
有理函数f ∈ K(X)^*

主除子：
(f) = (f)_0 - (f)_∞

其中：
- (f)_0：零点
- (f)_∞：极点

意义：
- 线性等价
- 分类
```

---

### 3.2 线性等价

**线性等价**：

```
除子D_1和D_2线性等价：
D_1 ~ D_2

若存在有理函数f使得
D_1 - D_2 = (f)

意义：
- 等价关系
- Picard群
- 分类
```

---

## 四、Picard群

### 4.1 定义

**Picard群**：

```
概形X的Picard群：

Pic(X) = 线丛的等价类

群运算：
- 乘法：L_1 ⊗ L_2
- 单位：O_X
- 逆：L^{-1}

意义：
- 分类不变量
- 几何不变量
```

---

### 4.2 计算

**Picard群的计算**：

```
方法：
- 除子类群
- 上同调群
- 长正合列

关系：
Pic(X) ≅ H^1(X, O_X^*)

Cech上同调
```

## 七、Riemann-Roch定理

### 7.1 曲线情况

**Riemann-Roch（曲线）**：

```
曲线C
除子D

χ(C, O(D)) = deg D + 1 - g

其中：
- χ：Euler特征
- deg D：度
- g：亏格
```
