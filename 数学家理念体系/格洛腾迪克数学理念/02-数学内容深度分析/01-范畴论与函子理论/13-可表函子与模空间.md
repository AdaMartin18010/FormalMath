# 可表函子与模空间：Yoneda引理的应用

## 一、可表函子

### 1.1 定义

**可表函子**：

```text
函子F: C^op → Set

F可表，若存在X ∈ C使得：
F ≅ Hom_C(-, X)

意义：
- Yoneda引理
- 应用广泛
```


## 三、Yoneda引理

### 3.1 Yoneda引理

**Yoneda引理**：

```text
范畴C
对象X, Y

Hom_{C^∧}(h_X, h_Y) ≅ Hom_C(X, Y)

意义：
- 嵌入
- 应用广泛
```
