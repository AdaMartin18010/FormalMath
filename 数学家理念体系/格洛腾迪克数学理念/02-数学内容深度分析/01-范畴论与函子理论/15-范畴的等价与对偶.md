## 一、范畴等价

### 1.1 定义

**范畴等价**：

```text
范畴C, D

等价：
存在函子F: C → D, G: D → C
使得G ∘ F ≅ id_C, F ∘ G ≅ id_D

意义：
- 等价关系
- 应用广泛
```

## 二、伴随与等价

### 2.1 伴随

**伴随**：

```text
函子F: C → D, G: D → C

伴随：
F ⊣ G

若：
Hom_D(F(-), -) ≅ Hom_C(-, G(-))

意义：
- 函子关系
- 应用广泛
```
