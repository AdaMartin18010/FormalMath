# Yoneda引理与可表函子：范畴论的基石

## 一、Yoneda引理

### 1.1 陈述

**Yoneda引理**：

```
设C是范畴，X ∈ Ob(C)

对任意函子F: C^op → Set：
Hom(h_X, F) ≅ F(X)

其中：
h_X = Hom(-, X): C^op → Set

同构是自然的（在X和F上）
```
---

### 1.2 证明思路

**Yoneda引理的证明**：

```
构造映射：
φ: Hom(h_X, F) → F(X)
φ(η) = η_X(id_X)

逆映射：
ψ: F(X) → Hom(h_X, F)
(ψ(a))_Y(f) = F(f)(a)

验证：
- 自然性
- 互逆
- 同构
```
---

## 二、Yoneda嵌入

### 2.1 嵌入构造

**Yoneda嵌入**：

```
h: C → [C^op, Set]
h(X) = Hom(-, X) = h_X

性质：
- 全忠实函子
- 嵌入
- 保持所有结构
```
---

### 2.2 完全性

**Yoneda完全性**：

```
对任意范畴C：
- 对象：通过h嵌入预层范畴
- 性质：在预层中自然

意义：
- 范畴可以嵌入Set^C^op
- 统一观点
- 函子观点
```
---

## 三、可表函子

### 3.1 定义

**可表函子**：

```
函子F: C^op → Set

F可表，若存在X ∈ Ob(C)使得
F ≅ h_X = Hom(-, X)

表示对象：
X表示F
```
---

### 3.2 唯一性

**表示对象的唯一性**：

```
Yoneda引理：
表示对象在同构意义下唯一

若F ≅ h_X且F ≅ h_Y
则X ≅ Y

唯一性（在同构意义下）
```
---

## 四、在代数几何中的应用

### 4.1 概形作为函子

**概形的函子观点**：

```
概形X：
h_X: Sch^op → Set
h_X(T) = Hom(T, X)

Yoneda：
X ≅ Y  ⟺  h_X ≅ h_Y

意义：
- 概形由函子决定
- 函子观点
- 相对观点自然
```
---

### 4.2 模空间

**模空间作为可表函子**：

```
模空间M表示函子F：
F(T) = {T上对象的等价类}

若F可表，则
M = 表示F的概形

构造：
- 通过函子
- Yoneda引理
- 分类问题
```
---

## 五、预层范畴

### 5.1 预层范畴

**预层范畴C^∧**：

```
C^∧ = [C^op, Set]

对象：预层F: C^op → Set
态射：自然变换

性质：
- 有所有（余）极限
```
