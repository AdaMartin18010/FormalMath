# 加法范畴与Abel范畴：范畴论的基础结构

## 一、加法范畴

### 1.1 定义

**加法范畴C**：

```
范畴C是加性的，若：
- 有零对象0
- 每对对象有积（=余积）
- Hom集合是Abel群
- 复合双线性

意义：
- Abel范畴的基础
- 加性结构
- 应用广泛
```
---

### 1.2 性质

**加法范畴的性质**：

```
性质：
- 有限积 = 有限余积
- 加性函子
- 良好性质
- 应用广泛
```
---

## 二、Abel范畴

### 2.1 定义

**Abel范畴A**：

```
Abel范畴是：
- 加性范畴
- 有核和余核
- 单射 = 核的核
- 满射 = 余核的余核
- 每个态射有像分解

意义：
- 同调代数
- 应用广泛
- 统一框架
```
---

### 2.2 例子

**Abel范畴的例子**：

```
例子：
- Ab：Abel群范畴
- Mod(R)：R模范畴
- Sh(X)：X上的Abel层范畴
- 链复形范畴

意义：
- 统一框架
- 应用广泛
- 同调代数
```
---

## 三、核与余核

### 3.1 核

**核ker(f)**：

```
Abel范畴中的态射f: A → B

核：
ker(f) → A → B

性质：
- 泛性质
- 应用广泛
```
---

### 3.2 余核

**余核coker(f)**：

```
Abel范畴中的态射f: A → B

余核：
A → B → coker(f)

性质：
- 泛性质
- 应用广泛
```
---

## 四、像与余像

### 4.1 像

**像im(f)**：

```
Abel范畴中的态射f: A → B

像：
im(f) = ker(coker(f))

性质：
- 像分解
- 应用广泛
```
---

### 4.2 余像

**余像coim(f)**：

```
Abel范畴中的态射f: A → B

余像：
coim(f) = coker(ker(f))

性质：
- 像分解
- 应用广泛
```
---

## 五、正合列

### 5.1 定义

**正合列**：

```
Abel范畴中的序列：

... → A → B → C → ...

正合，若在每个对象处：
im = ker

意义：
- 同调代数
- 关键概念
- 应用广泛
```
---

### 5.2 短正合列

**短正合列**：

```
0 → A → B → C → 0

正合 ⟺
- A → B单射
- B → C满射
- im(A → B) = ker(B → C)

意义：
- 扩张
- 分类问题
- 应用广泛
```
