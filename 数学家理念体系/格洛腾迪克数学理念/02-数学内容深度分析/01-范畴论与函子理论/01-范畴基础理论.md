# 范畴基础理论：对象与态射

## 一、范畴的定义

### 1.1 基本结构

**范畴C**：

```
数据：
- 对象集合：Ob(C)
- 态射集合：Mor(C)
- 对每个态射f，定义源和目标：
  dom(f), cod(f) ∈ Ob(C)
- 复合运算：
  ∘: Mor(Y,Z) × Mor(X,Y) → Mor(X,Z)
- 恒等态射：
  对每个X ∈ Ob(C)，id_X ∈ Mor(X,X)

公理：
1. 结合律：(h ∘ g) ∘ f = h ∘ (g ∘ f)
2. 单位律：id_Y ∘ f = f = f ∘ id_X
```
---

### 1.2 例子

**Set（集合范畴）**：

```
对象：集合
态射：函数
复合：函数复合
恒等：恒等函数

验证：
- 结合律：函数复合满足
- 单位律：恒等函数满足
```
**Grp（群范畴）**：

```
对象：群
态射：群同态
复合：同态复合
恒等：恒等同态
```
---

## 二、函子

### 2.1 函子定义

**函子F: C → D**：

```
数据：
- 对象映射：F: Ob(C) → Ob(D)
- 态射映射：F: Mor(C) → Mor(D)
- 保持域和余域：
  dom(F(f)) = F(dom(f))
  cod(F(f)) = F(cod(f))

公理：
1. 保持复合：F(g ∘ f) = F(g) ∘ F(f)
2. 保持恒等：F(id_X) = id_{F(X)}
```
---

### 2.2 协变与反变

**协变函子**：

```
方向不变：
f: X → Y
F(f): F(X) → F(Y)
```
**反变函子**：

```
方向反转：
f: X → Y
F(f): F(Y) → F(X)

例子：
Hom(-, Z): C^op → Set
Hom(f, Z): Hom(Y, Z) → Hom(X, Z)
```
---

## 三、自然变换

### 3.1 定义

**自然变换η: F → G**：

```
数据：
对每个X ∈ Ob(C)，态射η_X: F(X) → G(X)

自然性条件（交换图）：
对任意f: X → Y：

F(X) --F(f)--> F(Y)
 |              |
η_X            η_Y
 |              |
 G(X) --G(f)--> G(Y)

即：η_Y ∘ F(f) = G(f) ∘ η_X
```
---

### 3.2 自然同构

**自然同构**：

```
自然变换η是自然同构，若
对每个X，η_X是同构

记作：F ≅ G
```
---

## 四、Yoneda引理

### 4.1 陈述

**Yoneda引理**：

```
设C是范畴，X ∈ Ob(C)

对任意函子F: C^op → Set：
Hom(h_X, F) ≅ F(X)

其中：
h_X = Hom(-, X): C^op → Set
```
---

### 4.2 应用

**概形作为函子**：

```
概形X定义函子：
h_X: Sch^op → Set
h_X(T) = Hom(T, X)

Yoneda：
X ≅ Y  ⟺  h_X ≅ h_Y

意义：
概形由其"测试"决定
相对观点自然
```
---

## 五、伴随

### 5.1 定义

**伴随对(F, G)**：

```
F: C → D（左伴随）
G: D → C（右伴随）

存在自然同构：
Hom_D(F(X), Y) ≅ Hom_C(X, G(Y))

记作：F ⊣ G
```
---

### 5.2 单位与余单位

**单位η和余单位ε**：

```
η: id_C → G ∘ F
ε: F ∘ G → id_D

满足三角恒等式
```
---

## 六、极限与余极限

### 6.1 极限

**图D: J → C的极限**：

```
lim D满足普遍性质：
对任意对象X和自然变换
η: Δ_X → D（Δ_X是常函子）

存在唯一态射：
X → lim D
```
---

### 6.2 特殊极限

**积**：

```
∏_i X_i满足：
对任意Y和态射f_i: Y → X_i
存在唯一h: Y → ∏_i X_i
使得π_i ∘ h = f_i
```
