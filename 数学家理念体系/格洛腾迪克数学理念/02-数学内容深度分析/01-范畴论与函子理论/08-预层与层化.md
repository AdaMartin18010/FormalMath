## 二、层条件

### 2.1 层公理

**层条件**：

```
预层F是层，若

对覆盖{U_i → X}和
截面s_i ∈ F(U_i)

若在交集上一致：
res_{U_i, U_i×_X U_j}(s_i) = res_{U_j, U_i×_X U_j}(s_j)

则存在唯一s ∈ F(X)使得
res_{X, U_i}(s) = s_i
```
---

### 2.2 意义

**层的意义**：

```
意义：
- 局部到全局
- 几何直观
- 应用广泛
- 上同调基础
```

## 三、层化

### 3.1 层化函子

**层化F^+**：

```
预层F

层化F^+：
F^+(X) = lim_{U} H^0(U, F)

其中U是X的覆盖

性质：
- 左伴随
- 保持余极限
- 层化
```
---

### 3.2 双重层化

**双重层化F^{++}**：

```
预层F

双重层化F^{++}：
F^{++}是层

性质：
- F^{++}满足层条件
- 通用性质
- 应用广泛
```
