## 一、Abel范畴的定义

### 1.1 定义

**Abel范畴A**：

```text
Abel范畴是：
- 加性范畴
- 有核和余核
- 单射 = 核的核
- 满射 = 余核的余核
- 每个态射有像分解

意义：
- 同调代数
- 应用广泛
- 统一框架
```

意义：

## 二、加性范畴

### 2.1 定义

**加性范畴**：

```text
范畴C是加性的，若：
- 有零对象
- 每对对象有积（=余积）
- Hom集合是Abel群
- 复合双线性

意义：
- Abel范畴的基础
- 加性结构
- 应用广泛
```

```text
性质：

```

## 三、正合列

### 3.1 定义

**正合列**：

```text
Abel范畴中的序列：

... → A → B → C → ...

正合，若在每个对象处：
im = ker

意义：
- 同调代数
- 关键概念
- 应用广泛
```

---

### 3.2 短正合列

**短正合列**：

```text
0 → A → B → C → 0

正合 ⟺
- A → B单射
- B → C满射
- im(A → B) = ker(B → C)

意义：
- 扩张
- 分类问题
- 应用广泛
```

## 九、数学公式总结

### 核心公式

1. **Abel范畴定义**：
   $$\text{加性范畴} + \text{核和余核} + \text{像分解}$$

2. **核**：
   $$\ker(f) = \{x \in X : f(x) = 0\}$$

3. **余核**：
   $$\text{coker}(f) = Y / \text{im}(f)$$

4. **像分解**：
   $$f = i \circ \text{coim}(f) \circ p, \quad X \twoheadrightarrow \text{im}(f) \hookrightarrow Y$$

5. **短正合列**：
   $$0 \to A \xrightarrow{f} B \xrightarrow{g} C \to 0$$

6. **同调群**：
   $$H_n = \ker(\partial_n) / \text{im}(\partial_{n+1})$$

7. **导出函子**：
   $$R^i F = H^i(F(I^\bullet)), \quad L^i F = H^{-i}(F(P^\bullet))$$

8. **Ext函子**：
   $$\text{Ext}^i(A, B) = H^i(\text{Hom}(P^\bullet, B))$$

9. **Tor函子**：
   $$\text{Tor}_i(A, B) = H_i(P^\bullet \otimes B)$$

10. **投射分解**：
    $$\cdots \to P_1 \to P_0 \to M \to 0 \text{（$P_i$ 投射）}$$

## 十、Abel范畴与同调代数的详细数学表述

### 10.1 Abel范畴的严格定义

**Abel范畴的严格定义**：

**Abel范畴** $\mathcal{A}$ 是满足以下条件的加性范畴：

1. **核和余核存在**：对任意态射 $f: X \to Y$，核 $\ker(f)$ 和余核 $\text{coker}(f)$ 存在
2. **像分解**：对任意态射 $f$，有分解：
   $$X \twoheadrightarrow \text{im}(f) \hookrightarrow Y$$
3. **像 = 余像**：$\text{im}(f) = \text{coim}(f)$

**数学表述**：

Abel范畴：
$$\mathcal{A} \text{ 是Abel范畴 } \iff \text{核和余核存在，像 = 余像}$$

**例子9：Abel群的范畴**：

Abel群的范畴 $\text{Ab}$ 是Abel范畴。

**例子10：$R$-模的范畴**：

$R$-模的范畴 $\text{Mod}(R)$ 是Abel范畴。

### 10.2 导出函子的严格定义

**导出函子的严格定义**：

设 $F: \mathcal{A} \to \mathcal{B}$ 是左正合函子，$\mathcal{A}$ 有足够内射对象。

**右导出函子** $R^i F$ 定义为：
$$R^i F(X) = H^i(F(I_X^\bullet))$$

其中 $I_X^\bullet$ 是 $X$ 的内射分解。

**数学表述**：

右导出函子：
$$R^i F(X) = H^i(F(I_X^\bullet))$$

**例子11：上同调**：

对于全局截面函子 $\Gamma: \text{Sh}(X) \to \text{Ab}$，上同调为：
$$H^i(X, \mathcal{F}) = R^i \Gamma(\mathcal{F})$$

### 10.3 Ext和Tor函子的严格定义

**Ext函子的严格定义**：

对于 $R$-模 $A, B$，**Ext函子**定义为：
$$\text{Ext}^i_R(A, B) = H^i(\text{Hom}_R(P_A^\bullet, B))$$

其中 $P_A^\bullet$ 是 $A$ 的投射分解。

**数学表述**：

Ext函子：
$$\text{Ext}^i_R(A, B) = H^i(\text{Hom}_R(P_A^\bullet, B))$$

**Tor函子的严格定义**：

对于 $R$-模 $A, B$，**Tor函子**定义为：
$$\text{Tor}_i^R(A, B) = H_i(P_A^\bullet \otimes_R B)$$

其中 $P_A^\bullet$ 是 $A$ 的投射分解。

**数学表述**：

Tor函子：
$$\text{Tor}_i^R(A, B) = H_i(P_A^\bullet \otimes_R B)$$
