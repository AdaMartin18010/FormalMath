# Abel范畴与同调代数：范畴论的应用

## 一、Abel范畴的定义

### 1.1 定义

**Abel范畴A**：

```
Abel范畴是：
- 加性范畴
- 有核和余核
- 单射 = 核的核
- 满射 = 余核的余核
- 每个态射有像分解

意义：
- 同调代数
- 应用广泛
- 统一框架
```

---

### 1.2 例子

**Abel范畴的例子**：

```
例子：
- Ab：Abel群范畴
- Mod(R)：R模范畴
- Sh(X)：X上的Abel层范畴
- 链复形范畴

意义：
- 统一框架
- 应用广泛
- 同调代数
```

---

## 二、加性范畴

### 2.1 定义

**加性范畴**：

```
范畴C是加性的，若：
- 有零对象
- 每对对象有积（=余积）
- Hom集合是Abel群
- 复合双线性

意义：
- Abel范畴的基础
- 加性结构
- 应用广泛
```

---

### 2.2 性质

**加性范畴的性质**：

```
性质：
- 有限积 = 有限余积
- 加性函子
- 良好性质
- 应用广泛
```

---

## 三、正合列

### 3.1 定义

**正合列**：

```
Abel范畴中的序列：

... → A → B → C → ...

正合，若在每个对象处：
im = ker

意义：
- 同调代数
- 关键概念
- 应用广泛
```

---

### 3.2 短正合列

**短正合列**：

```
0 → A → B → C → 0

正合 ⟺
- A → B单射
- B → C满射
- im(A → B) = ker(B → C)

意义：
- 扩张
- 分类问题
- 应用广泛
```

---

## 四、导出函子

### 4.1 定义

**导出函子**：

```
左正合函子F: A → B

导出函子：
R^i F: A → B

定义：
R^i F(A) = H^i(F(I^*))

其中I^*是A的投射分解
```

---

### 4.2 例子

**导出函子的例子**：

```
例子：
- Ext = Hom的导出函子
- Tor = ⊗的导出函子
- 上同调 = Γ的导出函子

意义：
- 同调代数
- 应用广泛
- 统一框架
```

## 六、在代数几何中的应用

### 6.1 层上同调

```

意义：
- 几何不变量
- 应用广泛
```
