# 伴随与等价：范畴间的深层关系

## 一、伴随的定义

### 1.1 伴随对

**伴随F ⊣ G**：

```
F: C → D（左伴随）
G: D → C（右伴随）

存在自然同构：
Hom_D(F(X), Y) ≅ Hom_C(X, G(Y))

记作：F ⊣ G

对每个X ∈ C, Y ∈ D
```

---

### 1.2 单位与余单位

**单位η和余单位ε**：

```
单位：
η: id_C → G ∘ F
η_X: X → G(F(X))

余单位：
ε: F ∘ G → id_D
ε_Y: F(G(Y)) → Y

三角恒等式：
(εF) ∘ (Fη) = id_F
(Gε) ∘ (ηG) = id_G
```

---

## 二、伴随的例子

### 2.1 自由-遗忘伴随

**自由群伴随**：

```
F: Set → Grp（自由群函子）
U: Grp → Set（遗忘函子）

F ⊣ U

对应：
群同态 F(X) → G
↕
函数 X → U(G)

核心：
从集合到群的映射
唯一确定群同态
```

---

### 2.2 拉回-推前伴随

**概形态射的伴随**：

```
f: X → Y（概形态射）

拉回：
f*: Sh(Y) → Sh(X)
f*F = F在X上的拉回

推前：
f_*: Sh(X) → Sh(Y)
f_*F在y的茎 = F在f^{-1}(y)上的截面

伴随：
f* ⊣ f_*
```

---

## 三、伴随的性质

### 3.1 唯一性

**伴随的唯一性**：

```
若F ⊣ G和F ⊣ G'，则
G ≅ G'

类似：
若F ⊣ G和F' ⊣ G，则
F ≅ F'

唯一性（在同构意义下）
```

---

### 3.2 保持极限

**伴随保持极限**：

```
左伴随F保持余极限：
若colim D存在，则
F(colim D) ≅ colim (F ∘ D)

右伴随G保持极限：
若lim D存在，则
G(lim D) ≅ lim (G ∘ D)

应用：
- 自由函子保持余积
- 遗忘函子保持积
```

---

## 四、范畴等价

### 4.1 等价

**范畴等价**：

```
函子F: C → D是等价，若
存在函子G: D → C使得
G ∘ F ≅ id_C
F ∘ G ≅ id_D

记作：C ≃ D

性质：
- 保持所有范畴性质
- 可逆（在同构意义下）
```

---

### 4.2 强等价

**强等价（同构）**：

```
函子F: C → D是同构，若
存在函子G: D → C使得
G ∘ F = id_C
F ∘ G = id_D

记作：C ≅ D

更强条件：
等价的严格版本
```

---

## 五、Yoneda嵌入

### 5.1 嵌入

**Yoneda嵌入**：

```
范畴C：
h: C → [C^op, Set]
h(X) = Hom(-, X)

性质：
- 全忠实函子
- 嵌入
- 保持所有结构
```

---

### 5.2 完全性

**Yoneda完全性**：

```
对任意范畴C：
- 对象：通过h嵌入预层范畴
- 性质：在预层中自然

意义：
- 范畴可以嵌入Set^C^op
- 统一观点
```

## 六、在代数几何中的应用

### 6.1 概形范畴

**概形范畴的等价子范畴**：

```
仿射概形范畴：
AffSch ≃ CommRing^op

对应：
Spec: CommRing^op → AffSch
全局截面：AffSch → CommRing^op

反等价
```

---

### 6.2 函子观点

**概形作为函子**：

```
概形X：
h_X: Sch^op → Set
h_X(T) = Hom(T, X)

Yoneda：
X ≅ Y  ⟺  h_X ≅ h_Y

意义：
- 概形由函子决定
- 函子观点
```
