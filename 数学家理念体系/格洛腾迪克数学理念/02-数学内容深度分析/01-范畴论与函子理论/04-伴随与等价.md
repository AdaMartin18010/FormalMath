## 一、伴随的定义

### 1.1 伴随对

**伴随F ⊣ G**：

```text
F: C → D（左伴随）
G: D → C（右伴随）

存在自然同构：
Hom_D(F(X), Y) ≅ Hom_C(X, G(Y))

记作：F ⊣ G

对每个X ∈ C, Y ∈ D
```

---

### 1.2 单位与余单位

**单位η和余单位ε**：

```text
单位：
η: id_C → G ∘ F
η_X: X → G(F(X))

余单位：
ε: F ∘ G → id_D
ε_Y: F(G(Y)) → Y

三角恒等式：
(εF) ∘ (Fη) = id_F
(Gε) ∘ (ηG) = id_G
```

## 二、伴随的例子

### 2.1 自由-遗忘伴随

**自由群伴随**：

```text
F: Set → Grp（自由群函子）
U: Grp → Set（遗忘函子）

F ⊣ U

对应：
群同态 F(X) → G
↕
函数 X → U(G)

核心：
从集合到群的映射
唯一确定群同态
```

---

### 2.2 拉回-推前伴随

**概形态射的伴随**：

```text
f: X → Y（概形态射）

拉回：
f*: Sh(Y) → Sh(X)
f*F = F在X上的拉回

推前：
f_*: Sh(X) → Sh(Y)
f_*F在y的茎 = F在f^{-1}(y)上的截面

伴随：
f* ⊣ f_*
```

## 三、伴随的性质

### 3.1 唯一性

**伴随的唯一性**：

```text
若F ⊣ G和F ⊣ G'，则
G ≅ G'

类似：
若F ⊣ G和F' ⊣ G，则
F ≅ F'

唯一性（在同构意义下）
```

---

### 3.2 保持极限

**伴随保持极限**：

```text
左伴随F保持余极限：
若colim D存在，则
F(colim D) ≅ colim (F ∘ D)

右伴随G保持极限：
若lim D存在，则
G(lim D) ≅ lim (G ∘ D)

应用：
- 自由函子保持余积
- 遗忘函子保持积
```

## 四、范畴等价

### 4.1 等价

**范畴等价**：

```text
函子F: C → D是等价，若
存在函子G: D → C使得
G ∘ F ≅ id_C
F ∘ G ≅ id_D

记作：C ≃ D

性质：
- 保持所有范畴性质
- 可逆（在同构意义下）
```

---

### 4.2 强等价

**强等价（同构）**：

```text
函子F: C → D是同构，若
存在函子G: D → C使得
G ∘ F = id_C
F ∘ G = id_D

记作：C ≅ D

更强条件：
等价的严格版本
```

## 五、Yoneda嵌入

### 5.1 嵌入

**Yoneda嵌入**：

```text
范畴C：
h: C → [C^op, Set]
h(X) = Hom(-, X)

性质：
- 全忠实函子
- 嵌入
- 保持所有结构
```

---

### 5.2 完全性

**Yoneda完全性**：

```text
对任意范畴C：
- 对象：通过h嵌入预层范畴
- 性质：在预层中自然

意义：
- 范畴可以嵌入Set^C^op
- 统一观点
```

## 九、数学公式总结

### 核心公式

1. **伴随对**：
   $$F \dashv G \iff \text{Hom}(F(X), Y) \cong \text{Hom}(X, G(Y))$$

2. **单位**：
   $$\eta: \text{id}_\mathcal{C} \Rightarrow G \circ F, \quad \eta_X: X \to G(F(X))$$

3. **余单位**：
   $$\varepsilon: F \circ G \Rightarrow \text{id}_\mathcal{D}, \quad \varepsilon_Y: F(G(Y)) \to Y$$

4. **三角恒等式**：
   $$(\varepsilon F) \circ (F \eta) = \text{id}_F, \quad (G \varepsilon) \circ (\eta G) = \text{id}_G$$

5. **伴随保持极限**：
   $$F(\lim D) \cong \lim(F \circ D), \quad G(\text{colim } D) \cong \text{colim}(G \circ D)$$

6. **范畴等价**：
   $$F: \mathcal{C} \simeq \mathcal{D} \iff F \text{ 完全忠实本质满}$$

7. **等价与伴随**：
   $$F \dashv G \text{ 且 } \eta, \varepsilon \text{ 是同构} \Rightarrow F, G \text{ 是等价}$$

8. **Yoneda嵌入**：
   $$h: \mathcal{C} \hookrightarrow [\mathcal{C}^{\text{op}}, \text{Set}], \quad h(X) = \text{Hom}(-, X)$$

9. **自由-遗忘伴随**：
   $$F: \text{Set} \rightleftarrows \text{Grp} :U, \quad F \dashv U$$

10. **拉回-推前伴随**：
    $$f^*: \text{Sh}(Y) \rightleftarrows \text{Sh}(X) :f_*, \quad f^* \dashv f_*$$

## 十、伴随与等价的详细数学表述

### 10.1 伴随的严格定义

**伴随对的严格定义**：

设 $F: \mathcal{C} \to \mathcal{D}$ 和 $G: \mathcal{D} \to \mathcal{C}$ 是函子。

**$F$ 和 $G$ 是伴随对**（$F \dashv G$）当且仅当存在自然同构：
$$\text{Hom}_{\mathcal{D}}(F(X), Y) \cong \text{Hom}_{\mathcal{C}}(X, G(Y))$$

对所有 $X \in \mathcal{C}$ 和 $Y \in \mathcal{D}$ 成立。

**数学表述**：

伴随对：
$$F \dashv G \iff \text{Hom}_{\mathcal{D}}(F(X), Y) \cong \text{Hom}_{\mathcal{C}}(X, G(Y))$$

**例子9：自由-遗忘伴随**：

自由群函子 $F: \text{Set} \to \text{Grp}$ 和遗忘函子 $U: \text{Grp} \to \text{Set}$ 是伴随对：
$$F \dashv U$$

### 10.2 单位的严格定义

**单位的严格定义**：

对于伴随对 $F \dashv G$，**单位** $\eta: \text{id}_{\mathcal{C}} \Rightarrow G \circ F$ 定义为：
$$\eta_X: X \to G(F(X))$$

其中 $\eta_X$ 对应恒等映射 $\text{id}_{F(X)}: F(X) \to F(X)$ 在伴随同构下的像。

**数学表述**：

单位：
$$\eta: \text{id}_{\mathcal{C}} \Rightarrow G \circ F$$

单位映射：
$$\eta_X: X \to G(F(X))$$

**例子10：自由群的单位**：

对于自由群函子，单位 $\eta_X: X \to U(F(X))$ 将集合 $X$ 的元素映射到自由群 $F(X)$ 的生成元。

### 10.3 余单位的严格定义

**余单位的严格定义**：

对于伴随对 $F \dashv G$，**余单位** $\varepsilon: F \circ G \Rightarrow \text{id}_{\mathcal{D}}$ 定义为：
$$\varepsilon_Y: F(G(Y)) \to Y$$

其中 $\varepsilon_Y$ 对应恒等映射 $\text{id}_{G(Y)}: G(Y) \to G(Y)$ 在伴随同构下的像。

**数学表述**：

余单位：
$$\varepsilon: F \circ G \Rightarrow \text{id}_{\mathcal{D}}$$

余单位映射：
$$\varepsilon_Y: F(G(Y)) \to Y$$

**例子11：自由群的余单位**：

对于自由群函子，余单位 $\varepsilon_G: F(U(G)) \to G$ 将自由群 $F(U(G))$ 映射到群 $G$。
