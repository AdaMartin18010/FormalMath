## 一、图的极限

### 1.1 图

**图D: J → C**：

```text
小范畴J：索引范畴
函子D：J → C

例子：
- J离散：积
- J = • → •：等化子
- J = • → • ← •：拉回
```

---

### 1.2 锥

**锥到D**：

```text
对象X和自然变换：
η: Δ_X → D

其中Δ_X是常函子：
Δ_X(j) = X（对所有j）

等价：
对每个j ∈ J，态射η_j: X → D(j)
对每个α: j → k，图交换：
X → D(j)
↓    ↓
X → D(k)
```

## 二、极限的定义

### 2.1 极限

**图D的极限lim D**：

```text
锥(L, {p_j: L → D(j)})

满足普遍性质：
对任意锥(X, {f_j: X → D(j)})
存在唯一态射h: X → L
使得f_j = p_j ∘ h
```

---

### 2.2 特殊极限

**积**：

```text
J离散范畴

积：
∏_i X_i = lim D

投影：
π_i: ∏_i X_i → X_i

普遍性质：
对任意Y和f_i: Y → X_i
存在唯一h: Y → ∏_i X_i
使得π_i ∘ h = f_i
```

## 三、余极限

### 3.1 对偶

**余锥**：

```text
从D的余锥：
对象X和自然变换：
η: D → Δ_X

等价：
对每个j，态射η_j: D(j) → X
满足相容性
```

---

### 3.2 余极限定义

**余极限colim D**：

```text
余锥(C, {i_j: D(j) → C})

满足普遍性质：
对任意余锥(X, {f_j: D(j) → X})
存在唯一态射h: C → X
使得f_j = h ∘ i_j
```

## 四、特殊余极限

### 4.1 余积

**余积（不交并）**：

```text
J离散

余积：
⊔_i X_i = colim D

包含：
ι_i: X_i → ⊔_i X_i

普遍性质：
对任意Y和f_i: X_i → Y
存在唯一h: ⊔_i X_i → Y
使得h ∘ ι_i = f_i
```

---

### 4.2 推出

**推出（余纤维积）**：

```text
图：X → Z ← Y

推出：
X ⊔_Z Y = colim

性质：
对任意W和映射：
X → W ← Y（在Z上相容）
存在唯一X ⊔_Z Y → W
```

## 五、纤维积

### 5.1 定义

**纤维积（拉回）**：

```text
图：X → S ← Y

纤维积：
X ×_S Y = lim

投影：
p_X: X ×_S Y → X
p_Y: X ×_S Y → Y

在S上相容
```

---

### 5.2 普遍性质

**纤维积的普遍性质**：

```text
对任意T和映射：
f: T → X, g: T → Y

若在S上相容：
h_X ∘ f = h_Y ∘ g

则存在唯一：
h: T → X ×_S Y
使得f = p_X ∘ h, g = p_Y ∘ h
```

## 九、数学公式总结

### 核心公式

1. **极限定义**：
   $$\lim D = (L, \{p_j: L \to D(j)\}_{j \in J})$$

2. **积**：
   $$\prod_{i \in I} X_i = \lim D \text{（$D$ 是离散图）}$$

3. **等化子**：
   $$\text{eq}(f, g) = \{x \in X : f(x) = g(x)\}$$

4. **拉回（纤维积）**：
   $$X \times_Z Y = \{(x, y) \in X \times Y : f(x) = g(y)\}$$

5. **余极限定义**：
   $$\text{colim } D = (C, \{i_j: D(j) \to C\}_{j \in J})$$

6. **余积（和）**：
   $$\coprod_{i \in I} X_i = \text{colim } D$$

7. **推出**：
   $$X \sqcup_Z Y = X \sqcup Y / \sim \text{（$f(z) \sim g(z)$）}$$

8. **终对象**：
   $$\lim \emptyset = 1 \text{（终对象）}$$

9. **始对象**：
   $$\text{colim } \emptyset = 0 \text{（始对象）}$$

10. **保持极限**：
    $$F(\lim D) \cong \lim(F \circ D) \text{（$F$ 保持极限）}$$

## 十、极限与余极限的详细数学表述

### 10.1 极限的严格定义

**极限的严格定义**：

设 $J$ 是小范畴，$D: J \to \mathcal{C}$ 是函子（图）。

**图 $D$ 的极限** $\lim D$ 是对象 $L$ 和态射族 $\{p_j: L \to D(j)\}_{j \in \text{Ob}(J)}$，满足：

**普遍性质**：
对任意对象 $X$ 和态射族 $\{f_j: X \to D(j)\}_{j \in \text{Ob}(J)}$，若对任意态射 $u: i \to j$ 有 $D(u) \circ f_i = f_j$，则存在唯一态射 $h: X \to L$ 使得 $p_j \circ h = f_j$ 对所有 $j$。

**数学表述**：

极限：
$$\lim D = (L, \{p_j: L \to D(j)\}_{j \in J})$$

普遍性质：
$$\forall X, \{f_j: X \to D(j)\}, \exists! h: X \to L, p_j \circ h = f_j$$

**例子9：积的严格定义**：

积 $\prod_{i \in I} X_i$ 是离散图 $D: I \to \mathcal{C}$ 的极限。

### 10.2 余极限的严格定义

**余极限的严格定义**：

设 $J$ 是小范畴，$D: J \to \mathcal{C}$ 是函子。

**图 $D$ 的余极限** $\text{colim } D$ 是对象 $C$ 和态射族 $\{i_j: D(j) \to C\}_{j \in \text{Ob}(J)}$，满足：

**普遍性质**：
对任意对象 $X$ 和态射族 $\{g_j: D(j) \to X\}_{j \in \text{Ob}(J)}$，若对任意态射 $u: i \to j$ 有 $g_j \circ D(u) = g_i$，则存在唯一态射 $h: C \to X$ 使得 $h \circ i_j = g_j$ 对所有 $j$。

**数学表述**：

余极限：
$$\text{colim } D = (C, \{i_j: D(j) \to C\}_{j \in J})$$

普遍性质：
$$\forall X, \{g_j: D(j) \to X\}, \exists! h: C \to X, h \circ i_j = g_j$$

**例子10：余积的严格定义**：

余积 $\coprod_{i \in I} X_i$ 是离散图 $D: I \to \mathcal{C}$ 的余极限。

### 10.3 极限与余极限的存在性

**极限的存在性**：

**定理**：如果范畴 $\mathcal{C}$ 有所有小极限，则 $\mathcal{C}$ 是**完备的**。

**余极限的存在性**：

**定理**：如果范畴 $\mathcal{C}$ 有所有小余极限，则 $\mathcal{C}$ 是**余完备的**。

**数学表述**：

完备性：
$$\mathcal{C} \text{ 完备 } \iff \mathcal{C} \text{ 有所有小极限}$$

余完备性：
$$\mathcal{C} \text{ 余完备 } \iff \mathcal{C} \text{ 有所有小余极限}$$

**例子11：Set的完备性**：

Set 是完备且余完备的。

**例子12：Grp的完备性**：

Grp 是完备且余完备的。

## 十一、极限与余极限在概形理论中的应用

### 11.1 概形的纤维积

**概形纤维积的构造**：

概形的纤维积使用极限构造。

**构造方法**：

对于概形态射 $X \to S \leftarrow Y$，纤维积 $X \times_S Y$ 是图：
$$X \to S \leftarrow Y$$

的极限。

**数学表述**：

纤维积：
$$X \times_S Y = \lim(X \to S \leftarrow Y)$$

**例子13：仿射概形的纤维积**：

对于仿射概形 $\text{Spec}(A) \to \text{Spec}(R) \leftarrow \text{Spec}(B)$，纤维积为：
$$\text{Spec}(A) \times_{\text{Spec}(R)} \text{Spec}(B) = \text{Spec}(A \otimes_R B)$$

### 11.2 概形的推出

**概形推出的构造**：

概形的推出使用余极限构造。

**构造方法**：

对于概形态射 $S \to X, S \to Y$，推出 $X \sqcup_S Y$ 是图：
$$S \to X, S \to Y$$

的余极限。

**数学表述**：

推出：
$$X \sqcup_S Y = \text{colim}(S \to X, S \to Y)$$

**例子14：概形推出的应用**：

概形推出在代数几何中有重要应用，特别是在粘接构造中。
