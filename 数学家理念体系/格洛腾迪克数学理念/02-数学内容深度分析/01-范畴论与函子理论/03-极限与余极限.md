# 极限与余极限：普遍性质

## 一、图的极限

### 1.1 图

**图D: J → C**：

```
小范畴J：索引范畴
函子D：J → C

例子：
- J离散：积
- J = • → •：等化子
- J = • → • ← •：拉回
```

---

### 1.2 锥

**锥到D**：

```
对象X和自然变换：
η: Δ_X → D

其中Δ_X是常函子：
Δ_X(j) = X（对所有j）

等价：
对每个j ∈ J，态射η_j: X → D(j)
对每个α: j → k，图交换：
X → D(j)
↓    ↓
X → D(k)
```

---

## 二、极限的定义

### 2.1 极限

**图D的极限lim D**：

```
锥(L, {p_j: L → D(j)})

满足普遍性质：
对任意锥(X, {f_j: X → D(j)})
存在唯一态射h: X → L
使得f_j = p_j ∘ h
```

---

### 2.2 特殊极限

**积**：

```
J离散范畴

积：
∏_i X_i = lim D

投影：
π_i: ∏_i X_i → X_i

普遍性质：
对任意Y和f_i: Y → X_i
存在唯一h: Y → ∏_i X_i
使得π_i ∘ h = f_i
```

## 三、余极限

### 3.1 对偶

**余锥**：

```
从D的余锥：
对象X和自然变换：
η: D → Δ_X

等价：
对每个j，态射η_j: D(j) → X
满足相容性
```

---

### 3.2 余极限定义

**余极限colim D**：

```
余锥(C, {i_j: D(j) → C})

满足普遍性质：
对任意余锥(X, {f_j: D(j) → X})
存在唯一态射h: C → X
使得f_j = h ∘ i_j
```

---

## 四、特殊余极限

### 4.1 余积

**余积（不交并）**：

```
J离散

余积：
⊔_i X_i = colim D

包含：
ι_i: X_i → ⊔_i X_i

普遍性质：
对任意Y和f_i: X_i → Y
存在唯一h: ⊔_i X_i → Y
使得h ∘ ι_i = f_i
```

---

### 4.2 推出

**推出（余纤维积）**：

```
图：X → Z ← Y

推出：
X ⊔_Z Y = colim

性质：
对任意W和映射：
X → W ← Y（在Z上相容）
存在唯一X ⊔_Z Y → W
```

---

## 五、纤维积

### 5.1 定义

**纤维积（拉回）**：

```
图：X → S ← Y

纤维积：
X ×_S Y = lim

投影：
p_X: X ×_S Y → X
p_Y: X ×_S Y → Y

在S上相容
```

---

### 5.2 普遍性质

**纤维积的普遍性质**：

```
对任意T和映射：
f: T → X, g: T → Y

若在S上相容：
h_X ∘ f = h_Y ∘ g

则存在唯一：
h: T → X ×_S Y
使得f = p_X ∘ h, g = p_Y ∘ h
```

## 六、在概形中的应用

### 6.1 概形的纤维积

**概形的纤维积存在**：

```
概形态射：
X → S ← Y

纤维积X ×_S Y存在

构造：
- 仿射情况：张量积
- 一般情况：粘接
```

---

### 6.2 基变化

**基变化作为纤维积**：

```
X → S
基变化到T：

X_T = X ×_S T

性质：
- 相对观点
- 纤维变化
- 统一框架
```

## 七、极限的保持

### 7.1 函子保持极限

**保持极限的函子**：

```
函子F: C → D保持极限，若
对任意极限L = lim D
F(L) = lim(F ∘ D)

性质：
- 左伴随保持余极限
- 右伴随保持极限
```
