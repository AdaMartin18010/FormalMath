# 范畴的局部化：商范畴与同伦范畴

```text
- 泛性质

```

## 二、局部化的构造

### 2.1 定义

**局部化C[W^{-1}]**：

```text
范畴C
一类态射W

局部化：
C[W^{-1}]

性质：
- 函子C → C[W^{-1}]
- 将W映为同构
- 泛性质
```

---

### 2.2 泛性质

**局部化的泛性质**：

```text
局部化的泛性质：

对所有函子F: C → D
若F将W映为同构
则存在唯一F': C[W^{-1}] → D
使得F = F' ∘ q

其中q: C → C[W^{-1}]
```

```text
范畴M是模型范畴，若：

意义：
- 同伦理论

- 应用广泛
```

### 3.2 同伦范畴

```text
Ho(M) = M[W^{-1}]
```

## 四、导出范畴

### 4.1 构造

**导出范畴D(A)**：

```text
Abel范畴A

导出范畴：
D(A) = K(A)[Q^{-1}]

其中：
- K(A) = 链复形同伦范畴
- Q = 拟同构

意义：
- 同调代数
- 应用广泛
```
