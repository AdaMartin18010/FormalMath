# 范畴的局部性质与整体性质：Grothendieck的观点

## 一、局部性质

### 1.1 定义

**局部性质**：

```
范畴C

性质P是局部的，若：
对所有对象X ∈ C

X满足P ⟺ 存在覆盖{U_i → X}
使得每个U_i满足P

意义：
- 局部性质
- 应用广泛
```
---

### 1.2 例子

**局部性质的例子**：

```
例子：
- 平坦性

- 几何性质

```

## 二、整体性质

### 2.1 定义

**整体性质**：

```
范畴C

性质P是整体的，若：
不能由局部性质推出

意义：
- 整体性质
- 应用广泛
```
