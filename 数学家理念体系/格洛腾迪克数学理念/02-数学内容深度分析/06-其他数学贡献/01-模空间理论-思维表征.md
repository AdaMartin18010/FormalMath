## 一、术语与符号表（中英对照）

| 中文 | 英文 | 符号/备注 |
|------|------|-----------|
| 模空间 | moduli space | 参数化等价类的几何空间 |
| 模函子 | moduli functor | $F: \text{Sch}^{\text{op}} \to \text{Set}$，$F(T)$ = $T$ 上族/等价类 |
| 精细模空间 | fine moduli space | 概形 $M$ 可表 $F$ 且存在**万有族**（拉回得任意族） |
| 粗模空间 | coarse moduli space | $M(k)$ 与等价类双射、万有映射性质，可无万有族 |
| 万有族 | universal family | 在 $M$ 上的族，经唯一拉回得到任意族 |
| 可表函子 | representable functor | $h_M \cong F$，$h_M(T)=\operatorname{Hom}(T,M)$ |
| 模栈 / 精细模栈 | moduli stack / fine moduli stack | 群胚值函子可表为代数栈；常替代「精细模空间」当无概形可表时 |
| Hilbert 概形 | Hilbert scheme | 参量化闭子概形；FGA 可表性基础 |
| Quot 概形 | Quot scheme | 参量化商层；构造向量丛等模空间 |
| GIT 商 | GIT quotient | $X/\!/G$；Mumford 几何不变量理论，稳定点商 |
| 曲线模空间 | moduli space of curves | $\mathcal{M}_g$（栈）、粗模空间、$\overline{\mathcal{M}}_{g,n}$（稳定曲线） |

## 三、与形变理论的关系

模空间的**局部存在性与维数**（切空间、障碍、光滑性）由形变理论控制：

- 形变函子 $\text{Def}_Y$、Schlessinger 准则（hull/pro-可表）→ 形式邻域存在；
- $T^1 = H^1(Y,T_Y)$、$T^2 = H^2(Y,T_Y)$ → 模空间在对应点的切空间与光滑性；
- 曲线 $\mathcal{M}_g$：$\dim T^1 = 3g-3$，$T^2=0$，紧致化 $\overline{\mathcal{M}}_g$。

详见 [13-代数几何中的形变理论](./13-代数几何中的形变理论.md) 及 [13-形变理论-思维表征](./13-形变理论-思维表征.md)。
