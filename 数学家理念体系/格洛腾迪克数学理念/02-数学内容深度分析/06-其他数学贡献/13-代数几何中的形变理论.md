## 八、形变理论的详细数学表述

### 8.1 形变的严格定义

**形变**：

设 $Y$ 是概形，$S$ 是概形，$s_0 \in S$ 是点。

**形变**是概形态射：
$$f: X \to S$$

使得 $X_{s_0} = Y$（纤维在 $s_0$ 处等于 $Y$）。

**形变空间**：

**形变空间** $\text{Def}_Y$ 是形变类的集合，模去同构。

**数学公式**：
- 形变：$$f: X \to S, \quad X_{s_0} = Y$$
- 形变空间：$$\text{Def}_Y = \{\text{形变类}\} / \cong$$
- 纤维：$$X_s = f^{-1}(s)$$

---

### 8.2 切空间与障碍的严格表述

**切空间**：

设 $Y$ 是概形，$T_Y$ 是切丛。

**形变的切空间** $T^1$ 定义为：
$$T^1 = H^1(Y, T_Y)$$

**障碍**：

**形变的障碍** $T^2$ 定义为：
$$T^2 = H^2(Y, T_Y)$$

**形变理论**：

- 如果 $T^2 = 0$，则形变空间光滑
- 如果 $T^1 = 0$，则 $Y$ 是刚性的

**数学公式**：
- 切空间：$$T^1 = H^1(Y, T_Y)$$
- 障碍：$$T^2 = H^2(Y, T_Y)$$
- 形变空间：$$\dim \text{Def}_Y = \dim T^1 \quad \text{（如果 } T^2 = 0\text{）}$$

---

### 8.3 形变函子的严格定义

**形变函子**：

**形变函子** $\text{Def}_Y: \text{Art}_k \to \text{Set}$ 定义为：
$$\text{Def}_Y(A) = \{\text{在 } \text{Spec}(A) \text{ 上的形变}\} / \cong$$

其中 $\text{Art}_k$ 是Artin $k$-代数的范畴。

**Schlessinger准则**：

形变函子可表当且仅当满足Schlessinger条件。

**数学公式**：
- 形变函子：$$\text{Def}_Y: \text{Art}_k \to \text{Set}$$
- Schlessinger条件：$$\text{Def}_Y \text{ 可表 } \iff \text{Schlessinger条件}$$
