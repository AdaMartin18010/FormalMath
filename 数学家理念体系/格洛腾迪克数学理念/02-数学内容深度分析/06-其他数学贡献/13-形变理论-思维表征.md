## 一、术语与符号表（中英对照）

便于引用与检索；正文符号表见 [13-代数几何中的形变理论](./13-代数几何中的形变理论.md) 目录下「主要符号与术语」。

| 中文 | 英文 | 符号/备注 |
|------|------|-----------|
| 形变 | deformation | $f: X \to S$, $X_{s_0} \cong Y$ |
| 形变类 | deformation class | $[X \to S]$ 模同构 |
| 形变函子 | deformation functor | $\text{Def}_Y: \text{Art}_k \to \text{Set}$ |
| 形变空间 | deformation space | $\text{Def}_Y$（形变类集合） |
| 局部 Artin k-代数 | Artin local k-algebra | $\text{Art}_k$，剩余域 $k$ |
| 切空间 | tangent space / Zariski tangent space | $T^1 = H^1(Y, T_Y)$ |
| 障碍 | obstruction | $T^2 = H^2(Y, T_Y)$；Ob: $H^1 \to H^2$ |
| 可表 | representable | 概形 $M$ 使 $h_M \cong \text{Def}_Y$ |
| Pro-可表 / Hull | pro-representable / hull | 完备局部环 $R$，$\operatorname{Hom}(R,-) \to \text{Def}_Y$；(H1)–(H4) |
| Versal 形变 | versal deformation | 任一形变可由该族基变换得到 |
| 余切复形 | cotangent complex | $L_{B/A}$；Quillen |
| 形式模问题 | formal moduli problem | Lurie DAG X ↔ DGLA |
| 微分分次李代数 | differential graded Lie algebra (DGLA) | 特征零时控制形式邻域 |

## 二、多维概念对比矩阵

### 2.1 形变理论流派与框架对比

| 维度 | Kodaira–Spencer (复几何) | Grothendieck (概形/FGA) | Schlessinger (函子) | Lurie/导出 (DGLA) |
|------|--------------------------|--------------------------|----------------------|--------------------|
| **基空间** | 复流形/解析空间 B | 概形 S，常取 Spec A, A∈Art_k | Artin 局部 k-代数 A | 导出环/E∞-环，char k=0 |
| **切空间** | KS: T_0 B → H^1(M,T_M) | T^1 = H^1(Y,T_Y) | 函子 F 在 k[ε] 上的值 | H^1(DGLA) / 余切复形 H^1 |
| **障碍** | Ob: H^1 → H^2 | T^2 = H^2(Y,T_Y) | 提升到 k[ε]/ε^3 等 | H^2(DGLA)，Massey 括号 |
| **可表性** | Versal 形变存在性 | 下降+可表（EGA/SGA） | Hull / pro-可表 | 形式模问题 ↔ DGLA 等价 |
| **主要工具** | 全纯结构、上同调 | 层、上同调、下降 | 函子条件 (H1)–(H4) | L∞/DGLA，Koszul 对偶 |
| **典型文献** | KS 系列论文 | FGA, EGA | Schlessinger 1968 | DAG IV, DAG X |

### 2.2 概念层次对比（形变 vs 模空间 vs 可表性）

| 概念 | 含义 | 与形变理论关系 |
|------|------|----------------|
| 形变 (deformation) | X→S，X_{s_0}≅Y | 基本对象 |
| 形变类 | 模同构的形变 | Def_Y(S) 的元素 |
| 形变函子 Def_Y | Art_k ∋ A ↦ Def_Y(A) | 将「形变」提升为函子 |
| Versal 形变 | 任一形变可由基变换从该族得到 | 存在性定理（KS、Artin） |
| Pro-可表 / Hull | Def_Y ≅ Hom(R,−) 或逼近 R | Schlessinger 准则 |
| 模空间 M | h_M ≅ Def_Y（或粗模空间） | 应用：M 存在时 Def_Y 可表 |

│   ├── 形变 (X→S, X_{s_0}=Y)

## 五、公理–定理–证明树图（核心逻辑链）

```text
[设定] k 域，Y 为 k 上概形，T_Y = Hom(Ω_Y, O_Y)

A1. 形变定义: f: X→S 平坦，s_0∈S，X_{s_0} ≅ Y.
A2. 形变函子: Def_Y(A) = {A 上形变}/≅, A ∈ Art_k.
A3. 切空间定义: T^1 = H^1(Y, T_Y).
A4. 障碍定义: T^2 = H^2(Y, T_Y).

        |--- 定理 T1 (一阶形变): 一阶形变双射于 T^1.
        |         (证明: 双数 k[ε]/ε^2 上形变 ↔ H^1(T_Y).)
        |
        |--- 定理 T2 (障碍): 从 A_n 到 A_{n+1} 的提升障碍在 T^2 中.
        |         (证明: 上同调长正合列 / 障碍类.)
        |
        |--- 定理 T3 (维数): dim Def_Y ≤ dim T^1; 若 T^2=0 则局部等式.
        |         (证明: 切空间即 T^1; 无障碍则光滑.)
        |
        |--- 定理 T4 (Schlessinger): Def_Y 有 hull ⟺ (H1)–(H4).
        |         (证明: Schlessinger 1968.)
        |
        |--- 定理 T5 (曲线): 光滑曲线 C, g≥2 ⇒ T^1 维数 3g−3, T^2=0.
        |         (证明: H^2(C,T_C)=0 因 dim C=1.)
        |
        └--- 定理 T6 (Versal): 在适当条件下存在 versal 形变 (Artin/KS).
                  (证明: Artin 逼近 / 复几何 KS.)
```
