# 群概形理论：代数群的一般化

## 一、群概形的定义

### 1.1 基本定义

**群概形**：

```
概形G

群概形结构：
- 乘法：m: G × G → G
- 单位：e: S → G
- 逆：i: G → G

满足群公理（在概形范畴中）
```
---

### 1.2 相对群概形

**相对群概形**：

```
基概形S

S上群概形G：
- G → S（概形态射）
- 群结构在S上

意义：
- 相对观点
- 灵活构造
- 统一框架
```
---

## 二、例子

### 2.1 代数群

**代数群作为群概形**：

```
代数闭域k上的群概形G

例子：
- GL_n = 一般线性群
- SL_n = 特殊线性群
- 交换代数群

对应：
- 经典代数群
- 群概形
```
---

### 2.2 加法群与乘法群

**基本例子**：

```
加法群G_a:
G_a(T) = O_T(T)（加群）

乘法群G_m:
G_m(T) = O_T(T)^*（乘群）

意义：
- 基本群概形
- 构造基础
```
---

## 三、群概形的性质

### 3.1 仿射群概形

**仿射群概形**：

```
群概形G是仿射的，若
G = Spec(A)

Hopf代数结构：
- 余乘法：A → A ⊗ A
- 余单位：A → k
- 对极：A → A

群结构 ↔ Hopf代数结构
```
---

### 3.2 光滑群概形

**光滑群概形**：

```
群概形G是光滑的，若
G → S是平滑态射

性质：
- 几何光滑
- 良好性质
- 应用广泛
```
