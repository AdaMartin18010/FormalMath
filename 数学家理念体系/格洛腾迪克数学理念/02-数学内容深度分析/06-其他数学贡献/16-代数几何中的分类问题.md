## 一、分类问题

### 1.1 问题

**分类问题**：

分类问题是代数几何的核心问题之一：给定一类几何对象，如何系统地分类它们？

**核心问题**：

- **分类目标**：将几何对象按同构类分类
- **分类工具**：使用模空间和不变量
- **分类方法**：函子观点和可表函子

**数学表述**：

分类问题可以表示为：
$$\text{分类} = \{\text{几何对象}\} / \cong$$

其中 $\cong$ 是同构关系。

**例子1：曲线的分类**：

分类亏格 $g$ 的曲线：
$$\mathcal{M}_g = \{\text{亏格 } g \text{ 的曲线}\} / \cong$$

---

### 1.2 Grothendieck的方法

**Grothendieck的方法**：

格洛腾迪克使用函子观点解决分类问题：

1. **函子观点**：将分类问题转化为函子可表问题
2. **可表函子**：模空间是可表函子对应的概形
3. **模空间**：分类几何对象的空间

**数学表述**：

分类函子：
$$F: \text{Sch}^{\text{op}} \to \text{Set}, \quad F(S) = \{\text{几何对象在 } S \text{ 上的族}\} / \cong$$

如果 $F$ 可表，则模空间 $M$ 满足：
$$F \cong h_M$$

**例子2：向量丛的分类**：

向量丛的分类函子：
$$F(S) = \{\text{相对向量丛 } E \to X \times S\} / \cong$$

### 1.3 历史与渊源

**分类问题**通过**模空间**与**可表函子**系统化：Grothendieck FGA、Mumford 模空间与稳定曲线。参见 [01-模空间理论](./01-模空间理论.md)、FGA、Mumford。

## 二、模空间

### 2.1 定义

**模空间M**：

**模空间**是表示分类函子的概形。

**严格定义**：

设 $\mathcal{M}$ 是一类几何对象，**模函子** $F: \text{Sch}^{\text{op}} \to \text{Set}$ 定义为：
$$F(S) = \{\text{几何对象在 } S \text{ 上的族}\} / \cong$$

**模空间** $M$ 是概形，使得：
$$F \cong h_M$$

其中 $h_M(S) = \text{Hom}(S, M)$ 是可表函子。

**数学表述**：

- 模函子：$$F(S) = \{\text{几何对象在 } S \text{ 上的族}\} / \cong$$
- 模空间：$$F \cong h_M$$
- 分类：$$M(k) = \{\text{几何对象}\} / \cong$$

**例子3：曲线模空间**：

曲线模空间 $\mathcal{M}_g$ 满足：
$$\mathcal{M}_g(k) = \{\text{亏格 } g \text{ 的曲线}\} / \cong$$

---

### 2.2 性质

**模空间的性质**：

模空间具有以下重要性质：

1. **分类性质**：模空间的点对应几何对象的同构类
2. **函子性**：模空间是函子
3. **相对性**：模空间参数化相对族

**数学表述**：

- 分类性质：$$M(k) = \{\text{几何对象}\} / \cong$$
- 函子性：$$M(S) = \{\text{几何对象在 } S \text{ 上的族}\} / \cong$$
- 万有族：如果存在精细模空间，则有万有族 $\mathcal{U} \to M$

**例子4：精细模空间**：

精细模空间 $M$ 存在万有族：
$$\mathcal{U} \to M$$

使得对每个几何对象 $X$，存在唯一 $x \in M$ 使得 $\mathcal{U}_x \cong X$。
