## 一、枚举几何

### 1.1 定义

**枚举几何**：

枚举几何是计数几何对象的数学分支，使用模空间和几何不变量进行计数。

**核心问题**：

给定几何条件，计算满足条件的几何对象的数量。

**数学表述**：

枚举问题可以表示为：
$$N = \#\{X \in \mathcal{M} : X \text{ 满足条件 } P\}$$

其中 $\mathcal{M}$ 是模空间。

**例子1：曲线的计数**：

计算亏格 $g$、度 $d$ 的曲线数量：
$$N_{g,d} = \#\{C \subset \mathbb{P}^n : \text{亏格 } g, \text{度 } d\}$$

---

### 1.2 方法

**枚举方法**：

枚举几何使用以下方法：

1. **模空间**：几何对象的分类空间
2. **不变量**：Gromov-Witten不变量等
3. **虚拟类**：模空间上的虚拟基本类

**数学表述**：

- 模空间：$$\mathcal{M} = \{\text{几何对象}\} / \cong$$
- Gromov-Witten不变量：$$\langle \gamma_1, \ldots, \gamma_k \rangle_{g,\beta}$$
- 虚拟类：$$[\mathcal{M}]^{\text{vir}} \in H_*(\mathcal{M})$$

**例子2：Gromov-Witten不变量**：

对于光滑射影概形 $X$，Gromov-Witten不变量为：
$$\langle \gamma_1, \ldots, \gamma_k \rangle_{g,\beta} = \int_{[\overline{M}_{g,k}(X,\beta)]^{\text{vir}}} \prod_{i=1}^k \text{ev}_i^*(\gamma_i)$$

## 七、枚举几何的详细数学表述

### 7.1 Gromov-Witten不变量的严格定义

**Gromov-Witten不变量**：

设 $X$ 是光滑射影概形，$\beta \in H_2(X, \mathbb{Z})$ 是曲线类，$g, k \geq 0$。

**Gromov-Witten不变量** $\langle \gamma_1, \ldots, \gamma_k \rangle_{g,\beta}$ 定义为：
$$\langle \gamma_1, \ldots, \gamma_k \rangle_{g,\beta} = \int_{[\overline{M}_{g,k}(X,\beta)]^{\text{vir}}} \prod_{i=1}^k \text{ev}_i^*(\gamma_i)$$

其中 $\overline{M}_{g,k}(X,\beta)$ 是稳定映射的模空间。

**数学公式**：

- Gromov-Witten不变量：$$\langle \gamma_1, \ldots, \gamma_k \rangle_{g,\beta} = \int_{[\overline{M}_{g,k}(X,\beta)]^{\text{vir}}} \prod_{i=1}^k \text{ev}_i^*(\gamma_i)$$
- 模空间：$$\overline{M}_{g,k}(X,\beta) = \{\text{稳定映射 } f: C \to X\}$$
- 虚拟类：$$[\overline{M}_{g,k}(X,\beta)]^{\text{vir}} \in H_*(\overline{M}_{g,k}(X,\beta))$$
