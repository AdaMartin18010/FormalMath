### 主要符号与术语

| 符号/术语 | 含义 | 英文 |
|-----------|------|------|
| $\mathcal{F}$, $F$ | 模函子；$\mathcal{F}(T)$ = $T$ 上族模等价 | moduli functor |

**精细模空间与粗模空间**（fine / coarse moduli space）：若概形 $M$ 可表模函子 $\mathcal{F}$ 且存在**万有族**（在 $M$ 上的族，使任意 $T$ 上的族由唯一态射 $T \to M$ 拉回得到），则称 $M$ 为**精细模空间**。当精细模空间不存在（例如对象有非平凡自同构）时，常用**粗模空间**：概形 $M$ 满足（1）$M(k)$ 与等价类双射，（2）对任意从其他概形到该点集的态射有万有因子化，但**不必有万有族**。曲线模空间 $\mathcal{M}_g$ 的**精细**模空间是**代数栈**（见第五节），其**粗模空间**为概形。

形变理论为模空间的存在性与局部结构提供工具：形变函子 $\text{Def}_Y$、切空间 $T^1$、障碍 $T^2$ 及 Schlessinger 准则（hull / pro-可表）等，详见 [13-代数几何中的形变理论](./13-代数几何中的形变理论.md)。

曲线模空间：
$$\mathcal{M}_g(T) = \{\text{相对光滑曲线 } C \to T \text{ 使得 } C_t \text{ 亏格 } g\} / \cong$$

$$\dim \mathcal{M}_g = 3g - 3 \quad \text{（}g \geq 2\text{）}$$

## 八、模空间理论的数学细节

### 8.1 函子观点的严格数学表述

**模问题的函子表述**：

设 $\mathcal{F}: \text{Sch}^{\text{op}} \to \text{Set}$ 是一个函子，表示某个分类问题。

**模空间**是概形 $M$ 使得：
$$h_M \cong \mathcal{F}$$

其中 $h_M$ 是 $M$ 的**可表函子**：
$$h_M(T) = \text{Hom}_{\text{Sch}}(T, M)$$

**精细模空间与粗模空间**：若上述 $M$ 存在且存在**万有族**（在 $M$ 上的族 $\mathcal{U} \to M$，使任意族由唯一 $T \to M$ 拉回得到），则称 $M$ 为**精细模空间**（fine moduli space）。等价地，$M$ 为精细模空间当且仅当 $h_M \cong \mathcal{F}$ 且存在 $\mathcal{U} \to M$ 使得对任意 $T$ 与族 $\mathcal{E} \in \mathcal{F}(T)$，存在唯一 $f: T \to M$ 满足 $\mathcal{E} = f^*\mathcal{U}$。**粗模空间**（coarse moduli space）是概形 $M$ 满足：$M(k)$ 与 $\mathcal{F}(\operatorname{Spec} k)$ 双射，且对任意从概形到该点集的态射有万有因子化；粗模空间**不必**有万有族。可表性常需层条件（如 fppf/étale 下降）；**Hilbert 概形**（参量化闭子概形）与 **Quot 概形**（参量化商层）在 FGA 中证明可表，为曲线、向量丛等模空间的可表性提供技术基础。

**Yoneda引理的应用**：

如果模空间存在，则它是**唯一的**（在同构意义下），因为：
$$\text{Hom}(h_M, h_{M'}) \cong \text{Hom}(M, M')$$

**数学公式**：

- 可表函子：$$h_M(T) = \text{Hom}(T, M)$$
- 模空间条件：$$h_M \cong \mathcal{F}$$
- Yoneda引理：$$\text{Nat}(h_M, h_{M'}) \cong \text{Hom}(M, M')$$

---

### 8.2 曲线模空间的详细构造

**曲线的模函子**：

对亏格 $g \geq 2$，定义函子：
$$\mathcal{M}_g(T) = \left\{\text{相对光滑曲线 } C \to T \text{ 使得 } C_t \text{ 亏格 } g \text{ 的等价类}\right\}$$

**Deligne-Mumford紧化**（1969）：

$\mathcal{M}_g$ 不是紧致的，需要添加**稳定曲线**得到紧化 $\overline{\mathcal{M}}_g$。曲线模空间在每点的**局部维数与光滑性**由形变理论控制：切空间 $T^1 = H^1(C, T_C)$、障碍 $T^2 = H^2(C, T_C)=0$，故 $\dim \mathcal{M}_g = 3g-3$（$g \geq 2$）。详见 [13-代数几何中的形变理论](./13-代数几何中的形变理论.md) 第三节与第七节。

**稳定曲线**：

- 节点：允许简单节点
- 稳定性：$\text{Aut}(C)$ 有限
- 条件：$2g - 2 + n > 0$（$n$ 是标记点数）

**数学公式**：

- 模空间：$$\mathcal{M}_g = \{\text{亏格 } g \text{ 光滑曲线的模空间}\}$$
- 紧化：$$\overline{\mathcal{M}}_g = \{\text{亏格 } g \text{ 稳定曲线的模空间}\}$$
- 维数：$$\dim \mathcal{M}_g = 3g - 3 \text{（}g \geq 2\text{）}$$

---

### 8.3 向量丛模空间的构造

**向量丛的模函子**：

设 $X$ 是光滑射影曲线，定义：
$$\mathcal{M}_{X,r,d}(T) = \left\{\text{相对向量丛 } E \to X \times T \text{ 秩 } r \text{ 度 } d \text{ 的等价类}\right\}$$

**稳定性条件**（Mumford, 1965）：

向量丛 $E$ 是**稳定的**，如果对所有子丛 $F \subset E$：
$$\frac{\deg(F)}{\text{rk}(F)} < \frac{\deg(E)}{\text{rk}(E)}$$

**模空间构造**：

稳定向量丛的模空间 $\mathcal{M}_{X,r,d}^{\text{st}}$ 存在且是**拟射影概形**。

**数学公式**：

- 稳定性：$$\mu(F) = \frac{\deg(F)}{\text{rk}(F)} < \mu(E) = \frac{\deg(E)}{\text{rk}(E)}$$
- 模空间：$$\mathcal{M}_{X,r,d}^{\text{st}} = \{\text{稳定向量丛的模空间}\}$$
- 维数：$$\dim \mathcal{M}_{X,r,d}^{\text{st}} = r^2(g-1) + 1$$

### 8.4 GIT商的详细理论

**几何不变量理论（GIT）**（Mumford, 1965）：

设 $G$ 是约化群作用在射影概形 $X$ 上。

**稳定点**：
$$X^{\text{st}} = \{x \in X : \text{轨道 } G \cdot x \text{ 闭且 } \dim G_x = 0\}$$

**半稳定点**：
$$X^{\text{ss}} = \{x \in X : \text{存在不变截面 } s \text{ 使得 } s(x) \neq 0\}$$

**GIT商**：
$$X // G = \text{Proj} \left(\bigoplus_{n \geq 0} \Gamma(X, \mathcal{L}^{\otimes n})^G\right)$$

**数学公式**：

- 稳定点：$$X^{\text{st}} = \{x : G \cdot x \text{ 闭}, \dim G_x = 0\}$$
- 半稳定点：$$X^{\text{ss}} = \{x : \exists s \in \Gamma(X, \mathcal{L})^G, s(x) \neq 0\}$$
- GIT商：$$X // G = \text{Proj} \left(\bigoplus_n \Gamma(X, \mathcal{L}^n)^G\right)$$

### 8.5 栈理论的数学基础

**群胚值函子**：

**栈**是群胚值函子 $\mathcal{X}: \text{Sch}^{\text{op}} \to \text{Grpd}$ 满足：

1. **下降条件**：对覆盖 $\{U_i \to U\}$，$\mathcal{X}(U)$ 由 $\mathcal{X}(U_i)$ 和 $\mathcal{X}(U_i \times_U U_j)$ 决定
2. **层条件**：$\mathcal{X}$ 是**层**（sheaf）

**精细模空间**：

如果精细模空间存在，则：
$$\mathcal{M}_g^{\text{fine}} = \text{栈} \quad \text{（当 } g \geq 23 \text{ 时）}$$

**数学公式**：

- 栈：$$\mathcal{X}: \text{Sch}^{\text{op}} \to \text{Grpd}$$
- 下降条件：$$\mathcal{X}(U) \cong \lim \left[\prod_i \mathcal{X}(U_i) \rightrightarrows \prod_{i,j} \mathcal{X}(U_i \times_U U_j)\right]$$
- 精细模空间：$$\mathcal{M}_g^{\text{fine}} = \{\text{带标记的曲线的栈}\}$$

### 9.2 数论中的应用

**椭圆曲线的模空间**：

**模曲线** $Y_0(N)$ 参数化带水平结构 $N$ 的椭圆曲线：
$$Y_0(N) = \Gamma_0(N) \backslash \mathbb{H}$$

其中 $\Gamma_0(N) = \left\{\begin{pmatrix} a & b \\ c & d \end{pmatrix} \in \text{SL}(2, \mathbb{Z}) : c \equiv 0 \pmod{N}\right\}$。

**应用**：

- 模形式理论
- Langlands纲领
- 费马大定理（Wiles, 1994）

**数学公式**：

- 模曲线：$$Y_0(N) = \Gamma_0(N) \backslash \mathbb{H}$$
- 紧化：$$X_0(N) = Y_0(N) \cup \{\text{尖点}\}$$
- 模形式：$$f \in M_k(\Gamma_0(N))$$

## 十一、数学公式总结

### 核心公式

1. **模空间的函子定义**：
   $$h_M \cong \mathcal{F}$$
   其中 $\mathcal{F}: \text{Sch}^{\text{op}} \to \text{Set}$ 是模函子

2. **可表函子**：
   $$h_M(T) = \text{Hom}_{\text{Sch}}(T, M)$$

3. **Yoneda引理**：
   $$\text{Hom}(h_M, h_{M'}) \cong \text{Hom}(M, M')$$

4. **曲线的模函子**：
   $$\mathcal{M}_g(T) = \left\{\text{相对光滑曲线 } C \to T \text{ 使得 } C_t \text{ 亏格 } g \text{ 的等价类}\right\}$$

5. **曲线模空间的维数**：
   $$\dim \mathcal{M}_g = 3g - 3 \quad \text{（}g \geq 2\text{）}$$

6. **稳定曲线的条件**：
   $$2g - 2 + n > 0$$
   其中 $n$ 是标记点数

7. **向量丛的模函子**：
   $$\mathcal{M}_{X,r,d}(T) = \left\{\text{相对向量丛 } E \to X \times T \text{ 秩 } r \text{ 度 } d \text{ 的等价类}\right\}$$

8. **向量丛的稳定性条件**：
   $$\mu(F) = \frac{\deg(F)}{\text{rk}(F)} < \mu(E) = \frac{\deg(E)}{\text{rk}(E)}$$

9. **稳定向量丛模空间的维数**：
   $$\dim \mathcal{M}_{X,r,d}^{\text{st}} = r^2(g-1) + 1$$

10. **GIT稳定点**：
    $$X^{\text{st}} = \{x \in X : \text{轨道 } G \cdot x \text{ 闭且 } \dim G_x = 0\}$$

11. **GIT半稳定点**：
    $$X^{\text{ss}} = \{x \in X : \text{存在不变截面 } s \text{ 使得 } s(x) \neq 0\}$$

12. **GIT商**：
    $$X // G = \text{Proj} \left(\bigoplus_{n \geq 0} \Gamma(X, \mathcal{L}^{\otimes n})^G\right)$$

13. **栈的定义**：
    $$\mathcal{X}: \text{Sch}^{\text{op}} \to \text{Grpd}$$

14. **栈的下降条件**：
    $$\mathcal{X}(U) \cong \lim \left[\prod_i \mathcal{X}(U_i) \rightrightarrows \prod_{i,j} \mathcal{X}(U_i \times_U U_j)\right]$$

15. **稳定映射的模空间**：
    $$\overline{\mathcal{M}}_{g,n}(X, \beta) = \{\text{亏格 } g \text{ 带 } n \text{ 个标记的稳定映射 } f: C \to X \text{ 度 } \beta\}$$

16. **Gromov-Witten不变量**：
    $$\langle \gamma_1, \ldots, \gamma_n \rangle_{g,\beta} = \int_{[\overline{\mathcal{M}}_{g,n}(X,\beta)]^{\text{vir}}} \prod_{i=1}^n \text{ev}_i^*(\gamma_i)$$

17. **模曲线**：
    $$Y_0(N) = \Gamma_0(N) \backslash \mathbb{H}$$

18. **导出模空间**：
    $$\mathcal{M}^{\text{der}} = \text{导出栈}$$

19. **∞-栈**：
    $$\mathcal{M}^{\infty} = \text{∞-群胚值函子}$$
