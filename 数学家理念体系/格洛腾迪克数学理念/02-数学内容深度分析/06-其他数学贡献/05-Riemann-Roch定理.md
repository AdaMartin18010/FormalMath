# Riemann-Roch定理：格洛腾迪克的推广

## 一、经典Riemann-Roch

### 1.1 曲线情况

**Riemann-Roch（曲线）**：

```
曲线C
除子D

χ(C, O(D)) = deg D + 1 - g

其中：
- χ：Euler特征
- deg D：度
- g：亏格
```

---

### 1.2 意义

**经典定理的意义**：

```
意义：
- 分类工具
- 存在性定理
- 几何应用
- 历史重要性
```

---

## 二、Hirzebruch-Riemann-Roch

### 2.1 陈述

**HRR定理**：

```
射影概形X
向量丛E

χ(X, E) = ∫_X ch(E) · td(X)

其中：
- ch：Chern特征
- td：Todd类
```

---

### 2.2 意义

**HRR的意义**：

```
意义：
- 高维推广
- 拓扑方法
- 应用广泛
```

## 三、Grothendieck-Riemann-Roch

### 3.1 相对版本

**GRR定理**：

```
概形态射f: X → Y
向量丛E在X上

ch(R f_* E) · td(Y) = f_*(ch(E) · td(X))

意义：
- 相对版本
- 函子性
- 统一框架
```

应用：
