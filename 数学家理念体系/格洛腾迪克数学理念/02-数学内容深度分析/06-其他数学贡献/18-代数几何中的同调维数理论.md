## 七、同调维数的详细数学表述

### 7.1 投射维数的严格定义

**投射维数**：

设 $A$ 是环，$M$ 是 $A$-模。

**投射维数** $\text{pd}_A(M)$ 定义为：
$$\text{pd}_A(M) = \inf\{n \geq 0 \mid \exists \text{投射分解 } 0 \to P_n \to \cdots \to P_0 \to M \to 0\}$$

**内射维数**：

**内射维数** $\text{id}_A(M)$ 定义为：
$$\text{id}_A(M) = \inf\{n \geq 0 \mid \exists \text{内射分解 } 0 \to M \to I^0 \to \cdots \to I^n \to 0\}$$

**数学公式**：
- 投射维数：$$\text{pd}_A(M) = \inf\{n \geq 0 \mid \exists \text{投射分解}\}$$
- 内射维数：$$\text{id}_A(M) = \inf\{n \geq 0 \mid \exists \text{内射分解}\}$$
- 同调维数：$$\text{cohdim}(M) = \sup\{i \mid \text{Ext}^i(M, -) \neq 0\}$$

---

### 7.2 全局维数的严格定义

**全局维数**：

设 $A$ 是环。

**左全局维数** $\text{l.gl.dim}(A)$ 定义为：
$$\text{l.gl.dim}(A) = \sup\{\text{pd}_A(M) \mid M \text{ 是左 } A\text{-模}\}$$

**右全局维数** $\text{r.gl.dim}(A)$ 类似定义。

**数学公式**：
- 左全局维数：$$\text{l.gl.dim}(A) = \sup\{\text{pd}_A(M) \mid M \text{ 是左 } A\text{-模}\}$$
- 右全局维数：$$\text{r.gl.dim}(A) = \sup\{\text{pd}_A(M) \mid M \text{ 是右 } A\text{-模}\}$$
- 全局维数：$$\text{gl.dim}(A) = \max\{\text{l.gl.dim}(A), \text{r.gl.dim}(A)\}$$

---

### 7.3 概形的同调维数

**概形的同调维数**：

设 $X$ 是概形，$\mathcal{F}$ 是 $\mathcal{O}_X$-模层。

**同调维数** $\text{cohdim}_X(\mathcal{F})$ 定义为：
$$\text{cohdim}_X(\mathcal{F}) = \sup\{i \mid \text{Ext}^i_{\mathcal{O}_X}(\mathcal{F}, -) \neq 0\}$$

**数学公式**：
- 同调维数：$$\text{cohdim}_X(\mathcal{F}) = \sup\{i \mid \text{Ext}^i_{\mathcal{O}_X}(\mathcal{F}, -) \neq 0\}$$
- 上同调维数：$$\text{coh.dim}(X) = \sup\{\text{cohdim}_X(\mathcal{F}) \mid \mathcal{F} \in \text{Coh}(X)\}$$
