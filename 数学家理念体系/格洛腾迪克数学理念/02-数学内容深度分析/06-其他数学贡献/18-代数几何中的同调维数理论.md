## 一、同调维数

### 1.1 定义

**同调维数**：

设 $X$ 是概形，$\mathcal{F}$ 是 $\mathcal{O}_X$-模层。

**同调维数** $\text{cohdim}_X(\mathcal{F})$ 定义为：
$$\text{cohdim}_X(\mathcal{F}) = \sup\{i \geq 0 \mid \text{Ext}^i_{\mathcal{O}_X}(\mathcal{F}, -) \neq 0\}$$

**数学表述**：

同调维数衡量层的"复杂性"：
- 如果 $\text{cohdim}_X(\mathcal{F}) = 0$，则 $\mathcal{F}$ 是投射的
- 如果 $\text{cohdim}_X(\mathcal{F}) = d$，则需要 $d$ 步投射分解

**例子1：线丛的同调维数**：

对于线丛 $\mathcal{L}$，如果 $X$ 是正则概形，则：
$$\text{cohdim}_X(\mathcal{L}) = 0$$

**例子2：凝聚层的同调维数**：

对于凝聚层 $\mathcal{F}$，如果 $X$ 是 $n$ 维正则概形，则：
$$\text{cohdim}_X(\mathcal{F}) \leq n$$

---

### 1.2 性质

**同调维数的性质**：

同调维数具有以下重要性质：

1. **几何不变量**：同调维数是几何不变量
2. **上同调控制**：控制上同调群
3. **分类工具**：用于分类概形

**数学表述**：

- 同调维数：$$\text{cohdim}_X(\mathcal{F}) = \sup\{i \mid \text{Ext}^i(\mathcal{F}, -) \neq 0\}$$
- 上同调维数：$$\text{coh.dim}(X) = \sup\{\text{cohdim}_X(\mathcal{F}) \mid \mathcal{F} \in \text{Coh}(X)\}$$
- 正则性：$$X \text{ 正则} \iff \text{coh.dim}(X) = \dim X$$

### 1.3 历史与渊源

**同调维数**（投射维数、整体维数）源于同调代数与 **Serre**、**Grothendieck** 的正则性理论（EGA IV）。参见 EGA IV、Stacks Project、[11-导出范畴与同调代数](./11-导出范畴与同调代数.md)。

## 七、同调维数的详细数学表述

### 7.1 投射维数的严格定义

**投射维数**：

设 $A$ 是环，$M$ 是 $A$-模。

**投射维数** $\text{pd}_A(M)$ 定义为：
$$\text{pd}_A(M) = \inf\{n \geq 0 \mid \exists \text{投射分解 } 0 \to P_n \to \cdots \to P_0 \to M \to 0\}$$

**内射维数**：

**内射维数** $\text{id}_A(M)$ 定义为：
$$\text{id}_A(M) = \inf\{n \geq 0 \mid \exists \text{内射分解 } 0 \to M \to I^0 \to \cdots \to I^n \to 0\}$$

**数学公式**：
- 投射维数：$$\text{pd}_A(M) = \inf\{n \geq 0 \mid \exists \text{投射分解}\}$$
- 内射维数：$$\text{id}_A(M) = \inf\{n \geq 0 \mid \exists \text{内射分解}\}$$
- 同调维数：$$\text{cohdim}(M) = \sup\{i \mid \text{Ext}^i(M, -) \neq 0\}$$

---

### 7.2 全局维数的严格定义

**全局维数**：

设 $A$ 是环。

**左全局维数** $\text{l.gl.dim}(A)$ 定义为：
$$\text{l.gl.dim}(A) = \sup\{\text{pd}_A(M) \mid M \text{ 是左 } A\text{-模}\}$$

**右全局维数** $\text{r.gl.dim}(A)$ 类似定义。

**数学公式**：
- 左全局维数：$$\text{l.gl.dim}(A) = \sup\{\text{pd}_A(M) \mid M \text{ 是左 } A\text{-模}\}$$
- 右全局维数：$$\text{r.gl.dim}(A) = \sup\{\text{pd}_A(M) \mid M \text{ 是右 } A\text{-模}\}$$
- 全局维数：$$\text{gl.dim}(A) = \max\{\text{l.gl.dim}(A), \text{r.gl.dim}(A)\}$$

---

### 7.3 概形的同调维数

**概形的同调维数**：

设 $X$ 是概形，$\mathcal{F}$ 是 $\mathcal{O}_X$-模层。

## 九、数学公式总结

### 核心公式

1. **同调维数定义**：
   $$\text{cohdim}_X(\mathcal{F}) = \sup\{i \geq 0 \mid \text{Ext}^i_{\mathcal{O}_X}(\mathcal{F}, -) \neq 0\}$$

2. **上同调维数定义**：
   $$\text{coh.dim}(X) = \sup\{\text{cohdim}_X(\mathcal{F}) \mid \mathcal{F} \in \text{Coh}(X)\}$$

3. **正则概形条件**：
   $$X \text{ 正则} \iff \text{coh.dim}(X) = \dim X$$

4. **线丛的同调维数**：
   $$\text{cohdim}_X(\mathcal{L}) = 0 \quad \text{（当 } X \text{ 正则时）}$$

5. **凝聚层的同调维数上界**：
   $$\text{cohdim}_X(\mathcal{F}) \leq n \quad \text{（当 } X \text{ 是 } n \text{ 维正则概形时）}$$

6. **曲线上的上同调维数**：
   $$\text{coh.dim}(C) = 1 = \dim C \quad \text{（当 } C \text{ 是光滑曲线时）}$$

7. **n维光滑概形**：
   $$\text{coh.dim}(X) = n \quad \text{（当 } X \text{ 是 } n \text{ 维光滑概形时）}$$

8. **投射维数**：
   $$\text{pd}_A(M) = \inf\{n \geq 0 \mid \exists \text{投射分解 } 0 \to P_n \to \cdots \to P_0 \to M \to 0\}$$

9. **内射维数**：
   $$\text{id}_A(M) = \inf\{n \geq 0 \mid \exists \text{内射分解 } 0 \to M \to I^0 \to \cdots \to I^n \to 0\}$$

10. **左全局维数**：
    $$\text{l.gl.dim}(A) = \sup\{\text{pd}_A(M) \mid M \text{ 是左 } A\text{-模}\}$$

11. **全局维数**：
    $$\text{gl.dim}(A) = \max\{\text{l.gl.dim}(A), \text{r.gl.dim}(A)\}$$

12. **形变空间维数**：
    $$\dim \text{Def}_Y \leq \dim H^1(Y, T_Y) = \text{cohdim}_Y(T_Y)$$

13. **导出同调维数**：
    $$\text{cohdim}^{\text{der}}(X) = \text{导出同调维数}$$

14. **∞-同调维数**：
    $$\text{cohdim}^{\infty}(X) = \text{∞-同调维数}$$
