# 从簇到概形：代数几何的现代化转变

## 一、传统代数几何

### 1.1 簇理论

**代数簇**：

```
定义：
- 定义在代数闭域k上
- 仿射簇：V(I) ⊆ A^n_k
- 射影簇：齐次理想

例子：
椭圆曲线：y² = x³ + ax + b
- 点：(x, y) ∈ k²
- k = ℂ或F_p^alg
```
---

### 1.2 局限

**簇理论的局限**：

```
1. 依赖基域
   - 不同域需要重新定义
   - 无法统一处理

2. 无法处理算术
   - 整数解问题困难
   - 数论几何难以进行

3. 局部结构不清楚
   - 奇点处理困难
   - 不统一
```
---

## 二、概形理论

### 2.1 从环到几何

**交换环的谱**：

```
环R的谱Spec(R)：
- 点：素理想p
- 拓扑：Zariski拓扑
- 结构层：局部化

例子：
Spec(ℤ) = {(0), (2), (3), ...}
- 素数点：算术
- 一般点：几何
```
---

### 2.2 概形定义

**概形**：

```
概形 = 环化空间(X, O_X)

条件：
- 局部同构于仿射概形

意义：
- 统一几何与算术
- 相对观点
- 范畴框架
```
---
