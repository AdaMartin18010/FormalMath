# 函子观点与相对观点：格洛腾迪克的方法论

## 一、函子观点

### 1.1 Yoneda引理

**Yoneda引理的应用**：

```
概形X：
h_X: Sch^op → Set
h_X(T) = Hom(T, X)

Yoneda：
X ≅ Y  ⟺  h_X ≅ h_Y

意义：
- 概形由函子决定
- 函子观点
- 相对观点自然
```

---

### 1.2 可表函子

**可表函子的应用**：

```
模空间M表示函子F：
F(T) = {T上对象的等价类}

若F可表，则
M = 表示F的概形

构造：
- 通过函子
- Yoneda引理
- 分类问题
```
