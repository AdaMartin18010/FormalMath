# 格洛腾迪克数学理念体系 - 从这里开始

## 🔑 核心概念速览

| 概念 | 一句话解释 |
|------|----------|
| **范畴** | 对象的集合+态射的集合，满足组合律 |
| **函子** | 范畴之间的"映射" |
| **概形** | 代数几何的现代化对象，局部像交换环的谱 |
| **层** | 在拓扑空间上定义的数据 |
| **上同调** | 用代数方法测量"洞"的数量 |
| **Topos** | 广义集合论，可以改变逻辑规则 |
| **相对观点** | 不在孤立中研究对象，而在相对语境中 |
