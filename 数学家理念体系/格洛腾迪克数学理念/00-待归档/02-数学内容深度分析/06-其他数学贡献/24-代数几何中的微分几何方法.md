## 一、微分几何的基础

### 1.1 切空间

**切空间T_p X**：

设 $X$ 是概形，$p \in X$ 是点，$\mathfrak{m}_p$ 是 $p$ 处的极大理想。

**Zariski切空间** $T_p X$ 定义为：
$$T_p X = (\mathfrak{m}_p / \mathfrak{m}_p^2)^\vee$$

**数学表述**：

切空间是余切空间的对偶：
$$T_p X = \text{Hom}_k(\mathfrak{m}_p / \mathfrak{m}_p^2, k)$$

**例子1：仿射概形的切空间**：

对于仿射概形 $X = \text{Spec}(A)$，点 $p$ 对应素理想 $\mathfrak{p}$，切空间为：
$$T_p X = (\mathfrak{p} / \mathfrak{p}^2)^\vee$$

**例子2：曲线的切空间**：

对于曲线 $C$，在点 $p$ 处，切空间 $T_p C$ 是1维向量空间。

### 1.2 切丛

**切丛TX**：

**切丛** $T_X$ 是切空间的并：
$$T_X = \bigcup_{p \in X} T_p X$$

**数学表述**：

切丛是向量丛：
$$T_X = \mathcal{H}om(\Omega_X, \mathcal{O}_X)$$

其中 $\Omega_X$ 是微分形式层。

**例子3：曲线的切丛**：

对于曲线 $C$，切丛 $T_C$ 是线丛，度数为：
$$\deg(T_C) = 2 - 2g(C)$$

这是Riemann-Roch定理的应用。

### 1.3 历史与渊源

**微分几何方法**（切丛、余切丛、Kähler、**Hodge 理论**）与代数几何的交叉；**Chern 类**与 **Riemann-Roch**（[05-Riemann-Roch定理](./05-Riemann-Roch定理.md)）统一。参见 [05-Riemann-Roch定理](./05-Riemann-Roch定理.md)、Hodge 理论、Chern 类。

## 二、概形的切空间

### 2.1 Zariski切空间

**Zariski切空间**：

**Zariski切空间**是概形切空间的标准定义。

**定义**：

对于概形 $X$ 在点 $p$ 处，**Zariski切空间**为：
$$T_p X = (\mathfrak{m}_p / \mathfrak{m}_p^2)^\vee$$

其中 $\mathfrak{m}_p$ 是局部环 $\mathcal{O}_{X,p}$ 的极大理想。

**数学表述**：

- 切空间：$$T_p X = (\mathfrak{m}_p / \mathfrak{m}_p^2)^\vee$$
- 维数：$$\dim T_p X = \dim \mathfrak{m}_p / \mathfrak{m}_p^2$$
- 正则性：$$X \text{ 在 } p \text{ 正则} \iff \dim T_p X = \dim \mathcal{O}_{X,p}$$

**例子4：正则点**：

如果 $X$ 在 $p$ 处正则，则：
$$\dim T_p X = \dim \mathcal{O}_{X,p}$$

### 2.2 应用

**几何分析**：

切空间用于几何分析：

- **奇点分析**：研究奇点的性质
- **维数计算**：计算概形的维数
- **分类理论**：分类概形

**数学表述**：

- 奇点：$$\dim T_p X > \dim \mathcal{O}_{X,p}$$
- 正则点：$$\dim T_p X = \dim \mathcal{O}_{X,p}$$

## 三、联络与曲率

### 3.1 联络

**联络∇**：

**联络**（或协变导数）是向量丛上的微分算子。

**定义**：

设 $E$ 是 $X$ 上的向量丛，**联络** $\nabla$ 是映射：
$$\nabla: \Gamma(E) \to \Gamma(E \otimes \Omega_X)$$

满足：

- **线性性**：$\nabla(fs) = f\nabla(s) + s \otimes df$
- **Leibniz法则**：$\nabla(s_1 + s_2) = \nabla(s_1) + \nabla(s_2)$

**数学表述**：

联络满足：
$$\nabla(fs) = f\nabla(s) + s \otimes df$$

**例子5：切丛的联络**：

对于切丛 $T_X$，联络 $\nabla$ 称为**仿射联络**。

### 3.2 曲率

**曲率R**：

**曲率** $R$ 是联络的曲率张量。

**定义**：

对于联络 $\nabla$，**曲率** $R$ 定义为：
$$R(X, Y) = [\nabla_X, \nabla_Y] - \nabla_{[X,Y]}$$

**数学表述**：

曲率张量：
$$R \in \Gamma(\text{End}(E) \otimes \Omega_X^2)$$

**例子6：平坦联络**：

如果 $R = 0$，则联络是**平坦**的。

## 五、现代发展

### 5.1 导出方法

**导出代数几何**：

微分几何方法可以推广到导出几何。

**导出切空间**：

导出切空间是切空间的导出推广：

- **导出切空间**：$T_p^{\text{der}} X$ 是导出切空间
- **导出结构**：导出切空间具有导出结构
- **应用**：在导出几何中的应用

**数学表述**：

导出切空间：
$$T_p^{\text{der}} X = \text{导出切空间}$$

导出切丛：
$$T_X^{\text{der}} = \text{导出切丛}$$

**例子5：导出切空间的应用**：

导出切空间在导出几何中有重要应用。

**导出切空间的严格定义**：

导出切空间 $T_p^{\text{der}} X$ 定义为：
$$T_p^{\text{der}} X = \text{Spec}(\text{Sym}(\mathfrak{m}_p / \mathfrak{m}_p^2)^{\text{der}})$$

其中 $\text{Sym}^{\text{der}}$ 是导出对称代数。

**数学表述**：

导出切空间：
$$T_p^{\text{der}} X = \text{Spec}(\text{Sym}(\mathfrak{m}_p / \mathfrak{m}_p^2)^{\text{der}})$$

导出切丛：
$$T_X^{\text{der}} = \bigcup_{p \in X} T_p^{\text{der}} X$$

**∞-切空间**：

∞-切空间是切空间的∞-推广：

- **∞-切空间**：$T_p^{\infty} X$ 是∞-切空间
- **∞-结构**：∞-切空间具有∞-结构
- **应用**：在∞-几何中的应用

**数学表述**：

∞-切空间：
$$T_p^{\infty} X = \text{colim}_n T_p^{(n)} X$$

其中 $T_p^{(n)} X$ 是 $n$ 阶切空间。

∞-切丛：
$$T_X^{\infty} = \bigcup_{p \in X} T_p^{\infty} X$$

**例子6：∞-切空间的应用**：

∞-切空间在∞-几何中有重要应用，特别是在∞-模空间理论中。

### 5.2 应用

**现代应用**：

微分几何方法在现代数学中有广泛应用，特别是在导出几何、∞-几何和同伦理论等领域。

1. **导出切空间理论**：
   - 导出切空间是切空间的导出推广

   - 为导出几何提供几何工具

**导出切空间的构造**：

设 $X$ 是导出概形，$p \in X$ 是点。**导出切空间** $T_p^{\text{der}} X$ 定义为：
$$T_p^{\text{der}} X = \text{Spec}(\text{Sym}(\mathfrak{m}_p / \mathfrak{m}_p^2)^{\text{der}})$$

其中 $\text{Sym}^{\text{der}}$ 是导出对称代数。

**性质**：

- 导出切空间是导出向量空间
- 维数可能为负数（导出维数）
- 在奇点研究中重要

**例子7：导出切空间的应用**：

对于有奇点的概形 $X$：

- **经典切空间**：$\dim T_p X > \dim X$（奇点）

设 $E^{\text{der}}$ 是导出向量丛，**导出联络** $\nabla^{\text{der}}$ 是映射：
$$\nabla^{\text{der}}: \Gamma(E^{\text{der}}) \to \Gamma(E^{\text{der}} \otimes \Omega_X^{\text{der}})$$

满足Leibniz规则：
$$\nabla^{\text{der}}(fs) = f\nabla^{\text{der}}(s) + s \otimes d^{\text{der}}f$$

其中 $d^{\text{der}}$ 是导出微分。

**例子8：导出联络的应用**：

**导出曲率的定义**：

对于导出联络 $\nabla^{\text{der}}$，**导出曲率** $R^{\text{der}}$ 定义为：
$$R^{\text{der}}(X, Y) = [\nabla_X^{\text{der}}, \nabla_Y^{\text{der}}] - \nabla_{[X,Y]}^{\text{der}}$$

**数学表述**：

$$T_p^{\text{der}} X = \text{Spec}(\text{Sym}(\mathfrak{m}_p / \mathfrak{m}_p^2)^{\text{der}})$$

$$\nabla^{\text{der}}: \Gamma(E^{\text{der}}) \to \Gamma(E^{\text{der}} \otimes \Omega_X^{\text{der}})$$

### 核心公式

   $$T_p X = (\mathfrak{m}_p / \mathfrak{m}_p^2)^\vee$$

   $$T_p X = \text{Hom}_k(\mathfrak{m}_p / \mathfrak{m}_p^2, k)$$

   $$T_p X = (\mathfrak{p} / \mathfrak{p}^2)^\vee$$

4. **切丛定义**：
   $$T_X = \bigcup_{p \in X} T_p X$$

5. **切丛的层表示**：
   $$T_X = \mathcal{H}om(\Omega_X, \mathcal{O}_X)$$

   $$\dim T_p X = \dim \mathfrak{m}_p / \mathfrak{m}_p^2$$

   $$X \text{ 在 } p \text{ 正则} \iff \dim T_p X = \dim \mathcal{O}_{X,p}$$

9. **奇点条件**：
   $$\dim T_p X > \dim \mathcal{O}_{X,p} \Rightarrow p \text{ 是奇点}$$

    $$\nabla: \Gamma(E) \to \Gamma(E \otimes \Omega_X)$$
