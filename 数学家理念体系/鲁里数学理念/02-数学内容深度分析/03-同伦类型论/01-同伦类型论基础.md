## 一、同伦类型论的定义与性质

### 1.1 基本定义

**同伦类型论（Homotopy Type Theory, HoTT）**：

同伦类型论是类型论与同伦理论的结合，它将类型视为空间，将类型之间的等价视为同伦等价，从而在类型论中自然地表达同伦理论的概念。

**核心思想**：

1. **类型即空间**：每个类型对应一个拓扑空间（或更一般地，一个∞-群胚）
2. **等价即同伦等价**：类型之间的等价对应空间的同伦等价
3. **证明即路径**：两个项之间的相等性证明对应空间中的路径
4. **单值性公理**：等价类型是相等的

**数学意义**：

同伦类型论统一了：

- **类型论**：作为数学基础的形式化系统
- **同伦理论**：研究拓扑空间的同伦性质
- **范畴论**：通过∞-范畴的语言

**历史背景**：

同伦类型论的发展可以追溯到：

- **2000s**：Voevodsky提出单值性公理（Univalence Axiom）
- **2010s**：Lurie等数学家系统发展同伦类型论
- **2013**：《同伦类型论》一书出版，系统阐述理论

### 1.2 类型与同伦

**类型作为空间**：

在同伦类型论中，每个类型A对应一个空间|A|，类型中的项对应空间中的点。

```text
类型A ⟷ 空间|A|
项a: A ⟷ 点a ∈ |A|
相等性a = b ⟷ 路径p: a → b
```

**路径空间**：

对于类型A和项a, b: A，路径类型a = b对应从a到b的所有路径的空间。

**性质**：

- **自反性**：对于任何a: A，存在自反路径refl_a: a = a
- **对称性**：路径p: a = b可以反转得到p⁻¹: b = a
- **传递性**：路径p: a = b和q: b = c可以复合得到p·q: a = c

**高阶路径**：

路径之间也有路径，形成高阶同伦结构：

```text
路径: a = b (一阶)
路径的路径: p = q (二阶)
路径的路径的路径: α = β (三阶)
...
```

这对应了空间的同伦群和高阶同伦群。

### 1.3 同伦等价

**等价的定义**：

对于类型A和B，等价f: A ≃ B是一个函数，满足：

1. **单射性**：对于任何x, y: A，如果f(x) = f(y)，则x = y
2. **满射性**：对于任何y: B，存在x: A使得f(x) = y
3. **同伦性质**：存在逆函数和同伦

**等价与同伦等价**：

在同伦类型论中，类型之间的等价对应空间的同伦等价：

```text
等价f: A ≃ B ⟷ 同伦等价f: |A| ≃ |B|
```

**单值性公理（Univalence Axiom）**：

单值性公理是同伦类型论的核心公理：

```text
对于类型A和B，等价类型A ≃ B与相等类型A = B是等价的：
(A ≃ B) ≃ (A = B)
```

**意义**：

- 等价类型被视为相等的
- 这允许我们在类型论中自然地表达同伦理论
- 连接了类型论和同伦理论

## 二、同伦类型论的构造

### 2.1 从类型论到同伦类型论

**基础类型论**：

同伦类型论建立在依赖类型论（Dependent Type Theory）的基础上：

1. **基本类型**：自然数类型ℕ，布尔类型Bool等
2. **函数类型**：A → B
3. **积类型**：A × B
4. **和类型**：A + B
5. **依赖类型**：Π(x: A) B(x)，Σ(x: A) B(x)

**同伦类型论的扩展**：

在同伦类型论中，我们添加：

1. **路径类型**：a = b
2. **高阶路径类型**：p = q（路径的路径）
3. **单值性公理**：等价类型是相等的
4. **高阶归纳类型**：可以定义更复杂的类型

**构造过程**：

```text
基础类型论
    ↓
添加路径类型
    ↓
添加单值性公理
    ↓
添加高阶归纳类型
    ↓
同伦类型论
```

### 2.2 从同伦理论到同伦类型论

**同伦理论的基础**：

同伦理论研究拓扑空间的同伦性质：

- **同伦等价**：两个空间如果同伦等价，则具有相同的同伦性质
- **同伦群**：πₙ(X)测量空间X的n维"洞"
- **纤维化**：研究空间的纤维结构

**类型论中的同伦**：

在同伦类型论中，同伦理论的概念自然地出现：

```text
拓扑空间 ⟷ 类型
同伦等价 ⟷ 类型等价
同伦群 ⟷ 高阶路径类型
纤维化 ⟷ 依赖类型
```

**构造方法**：

1. **类型作为空间**：每个类型对应一个∞-群胚
2. **路径作为同伦**：路径类型对应同伦
3. **等价作为同伦等价**：类型等价对应同伦等价

### 2.3 同伦类型论的模型

**语义模型**：

同伦类型论有多种语义模型：

1. **∞-群胚模型**：
   - 类型对应∞-群胚
   - 路径对应同伦
   - 这是最自然的模型

2. **单纯集合模型**：
   - 类型对应单纯集合
   - 路径对应单纯集合中的路径
   - 这是计算友好的模型

3. **立方集合模型**：
   - 类型对应立方集合
   - 路径对应立方集合中的路径
   - 这是另一种计算模型

**模型的意义**：

不同的模型提供了理解同伦类型论的不同视角：

- **∞-群胚模型**：强调同伦理论的观点
- **单纯集合模型**：强调计算的观点
- **立方集合模型**：强调构造性的观点

## 三、同伦类型论的性质

### 3.1 单值性公理

**公理陈述**：

单值性公理（Univalence Axiom）是同伦类型论的核心公理：

```text
对于类型A和B，等价类型A ≃ B与相等类型A = B是等价的：
univalence: (A ≃ B) ≃ (A = B)
```

**具体形式**：

对于等价f: A ≃ B，单值性公理给出：

- 从等价f到相等性idtoequiv⁻¹(f): A = B的映射
- 从相等性p: A = B到等价equiv(p): A ≃ B的映射
- 这两个映射是互逆的

**意义**：

1. **类型等价即类型相等**：等价类型被视为相等的
2. **自然性**：允许我们在类型论中自然地表达同伦理论
3. **计算性**：提供了计算类型等价的方法

**应用**：

单值性公理在以下方面有重要应用：

- **类型同构**：两个同构类型是相等的
- **结构传输**：可以将结构从一个类型传输到等价类型
- **同伦理论**：自然地表达同伦理论的概念

### 3.2 高阶归纳类型

**归纳类型**：

在类型论中，归纳类型是通过构造子和归纳原理定义的：

```text
例子: 自然数类型ℕ
构造子:
  zero: ℕ
  succ: ℕ → ℕ

归纳原理:
  对于任何类型P: ℕ → Type，
  如果P(zero)和对于任何n: ℕ，P(n) → P(succ(n))，
  则对于任何n: ℕ，P(n)
```

**高阶归纳类型（HIT）**：

高阶归纳类型允许我们定义路径：

```text
例子: 圆S¹
构造子:
  base: S¹
  loop: base = base

这定义了圆，其中base是基点，loop是从base到base的路径。
```

**高阶归纳类型的例子**：

1. **圆S¹**：一个点和一条自环
2. **球面Sⁿ**：n维球面
3. **环面T²**：两个生成元和关系
4. **商类型**：通过等价关系定义的商

**应用**：

高阶归纳类型在以下方面有重要应用：

- **同伦理论**：定义同伦类型
- **代数拓扑**：定义拓扑空间
- **范畴论**：定义∞-范畴

### 3.3 同伦理论

**同伦群**：

在同伦类型论中，同伦群可以定义为：

```text
πₙ(A, a) = (base = base)ₙ

其中:
- base: A是基点
- (base = base)ₙ是n阶路径类型
```

**纤维化**：

纤维化在同伦类型论中对应依赖类型：

```text
纤维化p: E → B ⟷ 依赖类型E: B → Type

对于b: B，纤维E(b)对应类型E(b)
```

**同伦纤维序列**：

同伦纤维序列在同伦类型论中可以自然地表达：

```text
... → ΩB → F → E → B → ...

其中:
- ΩB是B的环路空间
- F是纤维
- E是全空间
- B是底空间
```

## 四、主要定理

### 4.1 同伦类型论的存在性定理

**模型存在性**：

同伦类型论有多种模型，证明了理论的一致性：

1. **∞-群胚模型**：类型对应∞-群胚
2. **单纯集合模型**：类型对应单纯集合
3. **立方集合模型**：类型对应立方集合

**一致性定理**：

同伦类型论（包括单值性公理）是一致的，即不会导致矛盾。

**证明方法**：

一致性通过构造模型来证明：

- 在∞-群胚中解释类型
- 验证所有公理都成立
- 这证明了理论的一致性

### 4.2 同伦类型论的构造定理

**类型构造**：

在同伦类型论中，我们可以构造各种类型：

1. **基本类型**：ℕ, Bool等
2. **函数类型**：A → B
3. **路径类型**：a = b
4. **高阶归纳类型**：S¹, Sⁿ等

**等价构造**：

我们可以构造类型之间的等价：

1. **恒等等价**：id: A ≃ A
2. **复合等价**：如果f: A ≃ B和g: B ≃ C，则g ∘ f: A ≃ C
3. **逆等价**：如果f: A ≃ B，则f⁻¹: B ≃ A

**同伦构造**：

我们可以构造同伦：

1. **恒等同伦**：id: f ~ f
2. **复合同伦**：如果α: f ~ g和β: g ~ h，则β · α: f ~ h
3. **逆同伦**：如果α: f ~ g，则α⁻¹: g ~ f

### 4.3 同伦类型论的应用定理

**类型等价定理**：

如果两个类型等价，则它们具有相同的同伦性质：

```text
如果f: A ≃ B，则:
- πₙ(A) ≃ πₙ(B)
- Hₙ(A) ≃ Hₙ(B)
- 其他同伦不变量也相同
```

**结构传输定理**：

如果两个类型等价，可以将结构从一个类型传输到另一个：

```text
如果f: A ≃ B，且A有结构S，
则可以通过f将S传输到B，得到f*(S)
```

**同伦理论定理**：

同伦类型论中的许多定理对应经典同伦理论的定理：

1. **Whitehead定理**：如果两个类型的所有同伦群都同构，则类型等价
2. **Hurewicz定理**：连接同伦群和同调群
3. **纤维化定理**：关于纤维化的各种定理
