## 一、概率论问题

### 问题 1.1：拉普拉斯继承法则（基础）

**问题陈述**:
假设某现象已经连续发生了 `n` 次，使用拉普拉斯继承法则计算该现象在下一次发生的概率。

**具体情形**:
太阳已经连续升起了5000年（约1825000天），明天太阳升起的概率是多少？

**解答**:
根据拉普拉斯继承法则：
$$P = \\frac{n+1}{n+2} = \\frac{1825000+1}{1825000+2} \\approx 0.99999945$$

**讨论**:
1. 这一结果反映的是归纳推理的置信度，而非物理概率
2. 拉普拉斯指出，对于自然规律，我们不应仅依赖统计归纳
3. 牛顿力学提供了太阳升起的确定性解释

### 问题 1.2：贝叶斯推断（中级）

**问题陈述**:
某医疗检测的敏感性为95%（真阳性率），特异性为90%（真阴性率）。人群中该疾病的患病率为1%。若某人检测呈阳性，他实际患病的概率是多少？

**解答**:
使用贝叶斯定理：

设：
- D: 患病事件
- T+: 检测阳性

已知：
- P(T+|D) = 0.95
- P(T-|¬D) = 0.90 → P(T+|¬D) = 0.10
- P(D) = 0.01

计算：
$$P(D|T+) = \\frac{P(T+|D)P(D)}{P(T+|D)P(D) + P(T+|¬D)P(¬D)}$$

$$= \\frac{0.95 \\times 0.01}{0.95 \\times 0.01 + 0.10 \\times 0.99} = \\frac{0.0095}{0.0095 + 0.099} \\approx 0.0876$$

**结论**:
即使检测阳性，实际患病概率仅约8.76%。这说明了即使检测准确性较高，对于罕见疾病，假阳性的数量可能超过真阳性。

### 问题 1.3：中心极限定理应用（中级）

**问题陈述**:
掷一枚公平骰子100次，求点数总和在320到380之间的概率。

**解答**:
设 `X_i` 为第 `i` 次掷骰子的点数，则：
- E[X_i] = 3.5
- Var(X_i) = 35/12 ≈ 2.917

总和 `S = ΣX_i`：
- E[S] = 100 × 3.5 = 350
- Var(S) = 100 × 35/12 ≈ 291.7
- σ_S ≈ 17.08

标准化：
$$Z = \\frac{S - 350}{17.08}$$

$$P(320 \\leq S \\leq 380) = P\\left(\\frac{320-350}{17.08} \\leq Z \\leq \\frac{380-350}{17.08}\\right)$$

$$= P(-1.756 \\leq Z \\leq 1.756) \\approx 0.921$$

### 问题 1.4：拉普拉斯分布的稳健性（高级）

**问题陈述**:
证明：对于拉普拉斯分布，最大似然估计等价于最小绝对偏差（LAD）估计，并解释其相对于正态分布（最小二乘）的稳健性。

**解答**:

**拉普拉斯分布的密度**:
$$f(x|\\mu, b) = \\frac{1}{2b}\\exp\\left(-\\frac{|x-\\mu|}{b}\\right)$$

**似然函数**:
$$L(\\mu) = \\prod_{i=1}^n \\frac{1}{2b}\\exp\\left(-\\frac{|x_i-\\mu|}{b}\\right)$$

**对数似然**:
$$\\ln L(\\mu) = -n\\ln(2b) - \\frac{1}{b}\\sum_{i=1}^n |x_i - \\mu|$$

**最大化**:
等价于最小化 `Σ|x_i - μ|`，即：
$$\\hat{\\mu}_{MLE} = \\text{median}(x_1, ..., x_n)$$

**稳健性分析**:
- **正态分布/最小二乘**: 目标函数 `Σ(x_i - μ)²`，对大偏差敏感
- **拉普拉斯分布/LAD**: 目标函数 `Σ|x_i - μ|`，对大偏差线性惩罚

**崩溃点**:
- 中位数的崩溃点为50%
- 均值的崩溃点为0%

## 二、天体力学问题

### 问题 2.1：引力势计算（基础）

**问题陈述**:
计算质量为 `M` 的均匀球体外部的引力势，并验证其满足拉普拉斯方程。

**解答**:

由对称性，势仅依赖于径向距离 `r`：

**外部** (r > R):
$$\\Phi(r) = -\\frac{GM}{r}$$

验证：
$$\\nabla² \\Phi = \\frac{1}{r²}\\frac{d}{dr}\\left(r²\\frac{d\\Phi}{dr}\\right) = \\frac{1}{r²}\\frac{d}{dr}\\left(r² \\cdot \\frac{GM}{r²}\\right) = 0$$

**内部** (r < R):
$$\\Phi(r) = -\\frac{GM}{2R³}(3R² - r²)$$

$$\\nabla² \\Phi = 4\\pi G \\rho$$

（泊松方程）

---

### 问题 2.2：开普勒轨道摄动（中级）

**问题陈述**:
考虑一个受微弱中心摄动力作用的行星轨道。设未摄动轨道为椭圆，半长轴为 `a`，偏心率为 `e`。若引入径向摄动力 `F_r = ε·(GMm/r³)`（`ε << 1`），求轨道近心点的进动速率。

**解答**:

**比耐方程**:
$$\\frac{d²u}{d\\theta²} + u = \\frac{GM}{L²} + \\frac{F_r(1/u)}{mL²u²}$$

其中 `u = 1/r`，`L` 为单位质量的角动量。

**代入摄动力**:
$$\\frac{d²u}{d\\theta²} + u = \\frac{GM}{L²} + \\frac{\\epsilon GM}{L²}u$$

**整理**:
$$\\frac{d²u}{d\\theta²} + (1-\\epsilon)\\frac{GM}{L²} = \\frac{GM}{L²}$$

**解**:
设 `k² = 1 - ε`，则轨道为：
$$u = \\frac{GM/L²}{k²}[1 + e\\cos(k(\\theta - \\theta_0))]$$

**进动**:
当 `θ` 增加 `2π/k` 时完成一个径向周期。进动角度：
$$\\delta\\theta = 2\\pi\\left(\\frac{1}{k} - 1\\right) \\approx \\pi\\epsilon$$

**相对论修正**:
广义相对论预言的进动：`ε = 3GM/(c²a(1-e²))`

对于水星：`δθ ≈ 43''/世纪`

### 问题 2.3：潮汐锁定（高级）

**问题陈述**:
推导月球潮汐锁定时间的数量级估计。已知：月球质量 `m = 7.3×10²² kg`，地球质量 `M = 6.0×10²⁴ kg`，地月距离 `a = 3.8×10⁸ m`，月球半径 `R = 1.7×10⁶ m`。

**解答**:

**潮汐力矩**:
$$\\tau \\approx \\frac{3}{2}\\frac{G M² R⁵}{a⁶}k_2 \\sin(2\\epsilon)$$

其中 `k₂` 是 love 数，`ε` 是潮汐滞后角。

**角动量变化**:
$$\\frac{dL}{dt} = \\tau$$

**自转角动量**:
$$L_{spin} = \\frac{2}{5}m R² \\omega$$

**锁定时间估计**:
$$T \\sim \\frac{L_{spin}}{\\tau} \\sim \\frac{4 m R² \\omega a⁶}{15 G M² R⁵ k_2 \\sin(2\\epsilon)}$$

代入数值（假设 `k₂ ≈ 0.03`，`ε ≈ 0.1°`）：
$$T \\sim 10^{16} \\text{s} \\sim 3\\times 10⁸ \\text{年}$$

与月球年龄（约45亿年）同数量级，说明有足够时间完成锁定。

## 三、拉普拉斯变换问题

### 问题 3.1：基础变换（基础）

**问题陈述**:
计算下列函数的拉普拉斯变换：
(a) `f(t) = t²e⁻³ᵗ`
(b) `f(t) = sin(2t)cos(t)`

**解答**:

**(a)**:
已知 `L{tⁿ} = n!/sⁿ⁺¹`，由频移性质：
$$\\mathcal{L}\\{t²e^{-3t}\\} = \\frac{2}{(s+3)³}$$

**(b)**:
使用积化和差：
$$\\sin(2t)\\cos(t) = \\frac{1}{2}[\\sin(3t) + \\sin(t)]$$

$$\\mathcal{L}\\{\\sin(at)\\} = \\frac{a}{s²+a²}$$

$$\\mathcal{L}\\{f(t)\\} = \\frac{1}{2}\\left[\\frac{3}{s²+9} + \\frac{1}{s²+1}\\right]$$

---

### 问题 3.2：解微分方程（中级）

**问题陈述**:
用拉普拉斯变换解初值问题：
$$y'' + 4y' + 5y = e^{-2t}, \\quad y(0) = 1, \\quad y'(0) = -2$$

**解答**:

**变换**:
$$s²Y - s + 2 + 4(sY - 1) + 5Y = \\frac{1}{s+2}$$

**整理**:
$$(s² + 4s + 5)Y = \\frac{1}{s+2} + s + 2$$

$$Y = \\frac{s² + 4s + 5}{(s+2)(s² + 4s + 5)} = \\frac{1}{s+2}$$

**逆变换**:
$$y(t) = e^{-2t}$$

**验证**:
- `y(0) = 1` ✓
- `y'(t) = -2e⁻²ᵗ`, `y'(0) = -2` ✓
- 代入微分方程：
  $$4e^{-2t} - 8e^{-2t} + 5e^{-2t} = e^{-2t}$$ ✓

---

### 问题 3.3：传递函数分析（中级）

**问题陈述**:
某系统的传递函数为：
$$H(s) = \\frac{s+3}{s²+3s+2}$$

(a) 求系统的冲激响应
(b) 判断系统稳定性
(c) 求对输入 `u(t)` 的响应

**解答**:

**(a) 冲激响应**:
$$H(s) = \\frac{s+3}{(s+1)(s+2)} = \\frac{2}{s+1} - \\frac{1}{s+2}$$

$$h(t) = (2e^{-t} - e^{-2t})u(t)$$

**(b) 稳定性**:
极点：`s = -1, -2`
都在左半平面，系统BIBO稳定。

**(c) 阶跃响应**:
$$Y(s) = H(s)\\cdot\\frac{1}{s} = \\frac{s+3}{s(s+1)(s+2)}$$

$$= \\frac{3/2}{s} - \\frac{2}{s+1} + \\frac{1/2}{s+2}$$

$$y(t) = \\left(\\frac{3}{2} - 2e^{-t} + \\frac{1}{2}e^{-2t}\\right)u(t)$$

### 问题 3.4：卷积计算（高级）

**问题陈述**:
设 `f(t) = e⁻ᵗu(t)`，`g(t) = t·u(t)`，用卷积定理求 `(f * g)(t)`。

**解答**:

**拉普拉斯变换**:
$$F(s) = \\frac{1}{s+1}, \\quad G(s) = \\frac{1}{s²}$$

**乘积**:
$$F(s)G(s) = \\frac{1}{s²(s+1)} = \\frac{1}{s+1} - \\frac{1}{s} + \\frac{1}{s²}$$

**逆变换**:
$$(f * g)(t) = e^{-t} - 1 + t, \\quad t \\geq 0$$

**直接验证**:
$$(f * g)(t) = \\int_0^t e^{-\\tau}(t-\\tau)d\\tau$$

$$= t\\int_0^t e^{-\\tau}d\\tau - \\int_0^t \\tau e^{-\\tau}d\\tau$$

$$= t(1-e^{-t}) - [1-(t+1)e^{-t}]$$

$$= t - te^{-t} - 1 + (t+1)e^{-t} = t - 1 + e^{-t}$$ ✓

## 四、微分方程问题

### 问题 4.1：分离变量法（基础）

**问题陈述**:
用分离变量法解矩形区域 `[0,a]×[0,b]` 上的拉普拉斯方程，边界条件为：
- `u(x,0) = u(x,b) = 0`
- `u(0,y) = 0`
- `u(a,y) = f(y)`

**解答**:

设 `u(x,y) = X(x)Y(y)`，得到：
$$\\frac{X''}{X} = -\\frac{Y''}{Y} = -\\lambda$$

**Y方程**:
$$Y'' + \\lambda Y = 0, \\quad Y(0) = Y(b) = 0$$

特征值： `λ_n = (nπ/b)²`, `Y_n(y) = sin(nπy/b)`, `n = 1,2,3,...`

**X方程**:
$$X'' - (n\\pi/b)²X = 0$$

$$X_n(x) = A_n \\sinh(n\\pi x/b) + B_n \\cosh(n\\pi x/b)$$

由 `X(0) = 0`，得 `B_n = 0`：
$$X_n(x) = \\sinh(n\\pi x/b)$$

**通解**:
$$u(x,y) = \\sum_{n=1}^\\infty c_n \\sinh\\left(\\frac{n\\pi x}{b}\\right)\\sin\\left(\\frac{n\\pi y}{b}\\right)$$

**确定系数**:
由 `u(a,y) = f(y)`：
$$c_n \\sinh\\left(\\frac{n\\pi a}{b}\\right) = \\frac{2}{b}\\int_0^b f(y)\\sin\\left(\\frac{n\\pi y}{b}\\right)dy$$

### 问题 4.2：格林函数（中级）

**问题陈述**:
构造上半空间 `z > 0` 的格林函数，并求解：
$$\\nabla² u = 0, \\quad z > 0$$
$$u(x,y,0) = g(x,y)$$

**解答**:

**格林函数**:
$$G(x,y,z; x',y',z') = \\frac{1}{4\\pi}\\left[\\frac{1}{\\sqrt{(x-x')²+(y-y')²+(z-z')²}} - \\frac{1}{\\sqrt{(x-x')²+(y-y')²+(z+z')²}}\\right]$$

**验证**:
- 满足 `∇²G = δ(x-x')`
- 在 `z=0` 上 `G = 0`

**解的表示**:
$$u(x,y,z) = -\\int_{-\\infty}^\\infty\\int_{-\\infty}^\\infty \\frac{\\partial G}{\\partial z'}|_{z'=0} g(x',y')dx'dy'$$

$$= \\frac{z}{2\\pi}\\int_{-\\infty}^\\infty\\int_{-\\infty}^\\infty \\frac{g(x',y')}{[(x-x')²+(y-y')²+z²]^{3/2}}dx'dy'$$

这是上半空间的**泊松积分公式**。

### 问题 4.3：特征值问题（高级）

**问题陈述**:
求单位圆盘上拉普拉斯算子的特征值和特征函数：
$$-\\nabla² u = \\lambda u, \\quad x²+y² < 1$$
$$u = 0, \\quad x²+y² = 1$$

**解答**:

**极坐标**:
$$\\frac{1}{r}\\frac{\\partial}{\\partial r}\\left(r\\frac{\\partial u}{\\partial r}\\right) + \\frac{1}{r²}\\frac{\\partial² u}{\\partial \\theta²} + \\lambda u = 0$$

设 `u(r,θ) = R(r)Θ(θ)`，得：
$$\\frac{r²R'' + rR'}{R} + \\lambda r² = -\\frac{\\Theta''}{\\Theta} = m²$$

**角度部分**:
$$\\Theta'' + m²\\Theta = 0$$

周期边界条件要求 `m = 0, ±1, ±2, ...`

**径向部分**:
$$r²R'' + rR' + (\\lambda r² - m²)R = 0$$

设 `ρ = √λ·r`，得**贝塞尔方程**：
$$\\rho² \\frac{d²R}{d\\rho²} + \\rho \\frac{dR}{d\\rho} + (\\rho² - m²)R = 0$$

**解**:
$$R(r) = J_m(\\sqrt{\\lambda}r)$$

**边界条件**:
$$J_m(\\sqrt{\\lambda}) = 0$$

设 `j_{m,n}` 是 `J_m` 的第 `n` 个零点，则：
$$\\lambda_{m,n} = j_{m,n}²$$

**特征函数**:
$$u_{m,n}(r,\\theta) = J_m(j_{m,n}r)\\cos(m\\theta) \\text{ 或 } \\sin(m\\theta)$$

**前几个特征值**:
- `λ₀,₁ = j₀,₁² ≈ (2.405)² ≈ 5.78`
- `λ₁,₁ = j₁,₁² ≈ (3.832)² ≈ 14.68`
- `λ₀,₂ = j₀,₂² ≈ (5.520)² ≈ 30.47`

### 问题 5.1：卫星轨道设计（研究级）

**问题陈述**:
设计一颗地球同步卫星的轨道，使其在考虑地球扁率（J₂项）摄动下，轨道倾角保持为零，轨道周期等于地球自转周期。

**解答要点**:

**1. 地球引力势**:
$$\\Phi = -\\frac{GM}{r}\\left[1 - J_2\\left(\\frac{R_E}{r}\\right)²P_2(\\cos\\theta) + ...\\right]$$

**2. J₂摄动效应**:
- 轨道平面进动
- 近地点进动
- 周期修正

**3. 地球同步轨道条件**:
- 半长轴：`a = (GMT²/4π²)^(1/3) ≈ 42164 km`
- 偏心率：`e ≈ 0`
- 倾角：`i = 0`

**4. 轨道保持**:
需要定期修正以补偿：
- 太阳和月球引力
- 太阳辐射压
- 地球非球形摄动

### 问题 5.3：期权定价模型（研究级）

**问题陈述**:
用拉普拉斯变换方法推导Black-Scholes模型中欧式看涨期权的定价公式。

**解答要点**:

**1. Black-Scholes PDE**:
$$\\frac{\\partial V}{\\partial t} + \\frac{1}{2}\\sigma²S²\\frac{\\partial² V}{\\partial S²} + rS\\frac{\\partial V}{\\partial S} - rV = 0$$

**2. 变量替换**:
设 `S = Keˣ`, `t = T - 2τ/σ²`, `V = Kw(x,τ)`

**3. 变换后方程**:
$$\\frac{\\partial w}{\\partial \\tau} = \\frac{\\partial² w}{\\partial x²} + (k-1)\\frac{\\partial w}{\\partial x} - kw$$

其中 `k = 2r/σ²`

**4. 拉普拉斯变换**:
对 `τ` 取拉普拉斯变换，求解ODE。

**5. Black-Scholes公式**:
$$C = S_0N(d_1) - Ke^{-rT}N(d_2)$$

其中：
$$d_1 = \\frac{\\ln(S_0/K) + (r + \\sigma²/2)T}{\\sigma\\sqrt{T}}$$
$$d_2 = d_1 - \\sigma\\sqrt{T}$$
