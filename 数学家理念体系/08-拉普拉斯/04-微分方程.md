## 一、历史背景

### 1.1 18世纪的分析学发展

**牛顿与莱布尼茨**:
微积分的创立为微分方程的研究奠定了基础。

**欧拉** (1707-1783):

- 系统发展了常微分方程理论
- 研究了弦振动方程等偏微分方程
- 变分法的奠基人

**拉普拉斯** (1749-1827):
将微分方程应用于天体物理学，发展了位势理论。

### 1.2 从天体力学到数学物理

拉普拉斯在研究天体引力问题时发现：

- 引力势满足特定的偏微分方程
- 这一方程在静电学、流体力学等领域也有广泛应用
- 由此建立了**位势理论**这一数学分支

## 二、拉普拉斯方程

### 2.1 定义

**三维拉普拉斯方程**:
$$\\nabla² u = \\frac{\\partial² u}{\\partial x²} + \\frac{\\partial² u}{\\partial y²} + \\frac{\\partial² u}{\\partial z²} = 0$$

**二维拉普拉斯方程**:
$$\\nabla² u = \\frac{\\partial² u}{\\partial x²} + \\frac{\\partial² u}{\\partial y²} = 0$$

**算子表示**:
$$\\Delta u = 0, \\quad \\Delta = \\nabla² = \\sum_{i=1}^n \\frac{\\partial²}{\\partial x_i²}$$

### 2.2 导出与物理意义

**引力场**:
由质量分布 `ρ(x)` 产生的引力势 `Φ` 满足：

- 无质量区： `∇²Φ = 0`
- 有质量区： `∇²Φ = 4πGρ`

**静电场**:
电荷分布产生的电势满足类似方程：

- 无电荷区： `∇²φ = 0`
- 有电荷区： `∇²φ = -ρ/ε₀`

**稳态热传导**:
稳态温度分布满足：
$$\\nabla² T = 0$$

**不可压缩无旋流动**:
速度势满足：
$$\\nabla² \\phi = 0$$

### 2.3 调和函数

**定义**:
满足拉普拉斯方程的函数称为**调和函数**。

**基本性质**:

**1. 平均值性质**:
调和函数在球心的值等于球面上的平均值：
$$u(x_0) = \\frac{1}{4\\pi R²} \\oint_{|x-x_0|=R} u(x) dS$$

**2. 极值原理**:
非恒定调和函数在区域内部不能取得最大值或最小值。

**3. 解析性**:
调和函数是无限可微的，甚至是实解析的。

**4. 唯一性**:
给定边界值，狄利克雷问题的解唯一。

### 2.4 基本解

**三维基本解**:
$$\\Phi(x, y) = \\frac{1}{4\\pi|x-y|}$$

满足：
$$-\\nabla² \\Phi = \\delta(x-y)$$

**二维基本解**:
$$\\Phi(x, y) = -\\frac{1}{2\\pi} \\ln|x-y|$$

### 2.5 分离变量法

**直角坐标系**:
设 `u(x,y,z) = X(x)Y(y)Z(z)`，得到：
$$\\frac{X''}{X} + \\frac{Y''}{Y} + \\frac{Z''}{Z} = 0$$

设：
$$\\frac{X''}{X} = -k_x², \\quad \\frac{Y''}{Y} = -k_y², \\quad \\frac{Z''}{Z} = k_x² + k_y²$$

**柱坐标系**:
$$\\frac{1}{r}\\frac{\\partial}{\\partial r}\\left(r\\frac{\\partial u}{\\partial r}\\right) + \\frac{1}{r²}\\frac{\\partial² u}{\\partial \\theta²} + \\frac{\\partial² u}{\\partial z²} = 0$$

径向方程给出**贝塞尔函数**。

**球坐标系**:
$$\\frac{1}{r²}\\frac{\\partial}{\\partial r}\\left(r²\\frac{\\partial u}{\\partial r}\\right) + \\frac{1}{r²\\sin\\theta}\\frac{\\partial}{\\partial \\theta}\\left(\\sin\\theta\\frac{\\partial u}{\\partial \\theta}\\right) + \\frac{1}{r²\\sin²\\theta}\\frac{\\partial² u}{\\partial \\phi²} = 0$$

角度部分给出**球谐函数**。

## 三、泊松方程

### 3.1 定义

$$-\\nabla² u = f$$

或：
$$\\nabla² u = -f$$

其中 `f` 是给定的源函数。

### 3.2 物理解释

| 物理领域 | u | f | 意义 |
|----------|---|---|------|
| 静电学 | 电势 | 电荷密度/ε₀ | 静电场方程 |
| 引力 | 引力势 | 质量密度·4πG | 引力场方程 |
| 热传导 | 温度 | 热源 | 稳态热方程 |
| 弹性力学 | 位移 | 体力 | 平衡方程 |

### 3.3 解的存在性与唯一性

**变分形式**:
寻找 `u ∈ H₀¹(Ω)` 使得：
$$\\int_\\Omega \\nabla u \\cdot \\nabla v dx = \\int_\\Omega fv dx, \\quad \\forall v \\in H₀¹(Ω)$$

**存在唯一性定理**:
若 `f ∈ L²(Ω)`，则泊松方程在 `H₀¹(Ω)` 中存在唯一弱解。

### 3.4 基本解表示

$$u(x) = \\int_\\Omega \\Phi(x,y)f(y)dy$$

其中 `Φ(x,y)` 是基本解。

## 四、边值问题

### 4.1 狄利克雷问题

**问题陈述**:
求 `u` 满足：
$$\\begin{cases}
\\nabla² u = 0 & \\text{在} \\Omega \\text{内} \\
u = g & \\text{在} \\partial\\Omega \\text{上}
\\end{cases}$$

**存在唯一性**:
- 解存在（对于足够光滑的边界和边界数据）
- 解唯一（由极值原理）

**稳定性**:
$$\\max_\\Omega |u| \\leq \\max_{\\partial\\Omega} |g|$$

### 4.2 诺伊曼问题

**问题陈述**:
求 `u` 满足：
$$\\begin{cases}
\\nabla² u = 0 & \\text{在} \\Omega \\text{内} \\
\\frac{\\partial u}{\\partial n} = h & \\text{在} \\partial\\Omega \\text{上}
\\end{cases}$$

**相容性条件**:
$$\\oint_{\\partial\\Omega} h dS = 0$$

**唯一性**:
解唯一确定到常数。

### 4.3 罗宾问题

**混合边界条件**:
$$\\frac{\\partial u}{\\partial n} + \\alpha u = \\beta, \\quad \\alpha > 0$$

### 4.4 复变函数方法（二维）

**柯西-黎曼方程**:
若 `f(z) = u(x,y) + iv(x,y)` 解析，则：
$$\\frac{\\partial u}{\\partial x} = \\frac{\\partial v}{\\partial y}, \\quad \\frac{\\partial u}{\\partial y} = -\\frac{\\partial v}{\\partial x}$$

**性质**:
- `u` 和 `v` 都是调和函数
- `v` 称为 `u` 的共轭调和函数

**保角映射**:
通过解析函数将复杂区域映射到简单区域（如单位圆），简化边值问题求解。

## 五、球谐函数

### 5.1 定义

球谐函数 `Y_l^m(θ, φ)` 是球坐标下拉普拉斯方程的角度解：

$$Y_l^m(\\theta, \\varphi) = (-1)^m \\sqrt{\\frac{(2l+1)(l-m)!}{4\\pi(l+m)!}} P_l^m(\\cos\\theta) e^{im\\varphi}$$

其中：
- `l = 0, 1, 2, ...` 是角量子数
- `m = -l, -l+1, ..., l-1, l` 是磁量子数
- `P_l^m` 是连带勒让德函数

### 5.2 正交归一性

$$\\int_0^{2\\pi} \\int_0^\\pi Y_l^m(\\theta, \\varphi) \\overline{Y_{l'}^{m'}(\\theta, \\varphi)} \\sin\\theta d\\theta d\\varphi = \\delta_{ll'} \\delta_{mm'}$$

### 5.3 完备性

球谐函数构成球面上的完备正交系，任意平方可积函数可展开：
$$f(\\theta, \\varphi) = \\sum_{l=0}^\\infty \\sum_{m=-l}^l a_{lm} Y_l^m(\\theta, \\varphi)$$

### 5.4 应用

**量子力学**:
- 角动量本征函数
- 原子轨道形状

**地球物理学**:
- 地球重力场模型
- 地磁场建模

**计算机图形学**:
- 环境光遮蔽
- 预计算辐射度传递

## 六、格林函数方法

### 6.1 格林函数定义

对于区域 `Ω`，格林函数 `G(x,y)` 满足：
$$\\begin{cases}
-\\nabla²_x G(x,y) = \\delta(x-y) & x \\in \\Omega \\
G(x,y) = 0 & x \\in \\partial\\Omega
\\end{cases}$$

### 6.2 解的表示

狄利克雷问题的解可表示为：
$$u(x) = \\int_\\Omega G(x,y)f(y)dy - \\int_{\\partial\\Omega} \\frac{\\partial G}{\\partial n_y}(x,y)g(y)dS_y$$

### 6.3 镜像法

对于特殊区域（半空间、球、圆），格林函数可用镜像法构造。

**半空间** (`z > 0`):
$$G(x,y) = \\frac{1}{4\\pi|x-y|} - \\frac{1}{4\\pi|x-y^*|}$$

其中 `y*` 是 `y` 关于 `z=0` 平面的镜像。

**球** (半径 `R`):
$$G(x,y) = \\frac{1}{4\\pi|x-y|} - \\frac{R/|y|}{4\\pi|x-y^*|}$$

其中 `y* = R²y/|y|²` 是反演点。

## 七、变分原理

### 7.1 狄利克雷原理

**能量泛函**:
$$J[u] = \\frac{1}{2}\\int_\\Omega |\\nabla u|² dx$$

**变分原理**:
调和函数使狄利克雷能量取极小值（在固定边界条件下）。

**欧拉-拉格朗日方程**:
$$\\frac{\\delta J}{\\delta u} = -\\nabla² u = 0$$

### 7.2 瑞利商

$$R[u] = \\frac{\\int_\\Omega |\\nabla u|² dx}{\\int_\\Omega u² dx}$$

拉普拉斯算子的第一特征值是瑞利商的最小值：
$$\\lambda_1 = \\min_{u \\neq 0} R[u]$$

## 八、数值方法

### 8.1 有限差分法

**二维五点格式**:
$$\\frac{u_{i+1,j} + u_{i-1,j} + u_{i,j+1} + u_{i,j-1} - 4u_{i,j}}{h²} = 0$$

**截断误差**: `O(h²)`

### 8.2 有限元方法

**弱形式**:
寻找 `u_h ∈ V_h` 使得：
$$\\int_\\Omega \\nabla u_h \\cdot \\nabla v_h dx = \\int_\\Omega f v_h dx, \\quad \\forall v_h \\in V_h$$

**误差估计**:
$$\\|u - u_h\\|_{H¹} \\leq Ch\\|u\\|_{H²}$$

### 8.3 边界元方法

利用格林恒等式将问题转化为边界积分方程：
$$\\frac{1}{2}u(x) + \\int_{\\partial\\Omega} \\frac{\\partial \\Phi}{\\partial n_y}(x,y)u(y)dS_y = \\int_{\\partial\\Omega} \\Phi(x,y)\\frac{\\partial u}{\\partial n}(y)dS_y$$

**优势**:
- 降维（三维问题化为二维）
- 适合无限域问题

### 8.4 多重网格法

利用不同尺度的网格加速迭代求解，收敛速度与网格尺寸无关。

## 九、应用领域

### 9.1 电磁学

**静电场**:
$$\\nabla² \\phi = -\\frac{\\rho}{\\varepsilon_0}$$

**静磁场**:
$$\\nabla² \\mathbf{A} = -\\mu_0 \\mathbf{J}$$

### 9.2 流体力学

**势流**:
不可压缩无旋流动的速度势满足拉普拉斯方程。

**应用**:
- 翼型设计
- 水波理论
- 多孔介质流动

### 9.3 地球物理学

**重力勘探**:
通过地表重力异常反演地下密度分布。

**地磁学**:
地磁场的球谐分析。

### 9.4 生物医学

**脑电图 (EEG)**:
头部电势分布的建模。

**电阻抗断层成像 (EIT)**:
通过边界测量重建内部电导率分布。

### 9.5 计算机科学

**图像处理**:
- 图像修复（inpainting）
- 边缘检测
- 形状插值

**几何处理**:
- 网格参数化
- 曲面平滑
