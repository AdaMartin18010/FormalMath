## 八、核心公式速查

### 8.1 概率论

```
贝叶斯定理:        P(H|E) = P(E|H)P(H) / P(E)

拉普拉斯继承:      P = (n+1)/(n+2)

拉普拉斯分布:      f(x) = (1/2b)exp(-|x-μ|/b)

中心极限定理:      (S_n - nμ)/(σ√n) → N(0,1)
```

### 8.2 天体力学

```
拉普拉斯方程:      ∇²Φ = 0

泊松方程:          ∇²Φ = 4πGρ

球谐函数:          Y_l^m(θ,φ) ∝ P_l^m(cosθ)e^{imφ}

引力势:            Φ = -GM/r (点质量)
```

### 8.3 拉普拉斯变换

```
定义:              F(s) = ∫_0^∞ e^{-st}f(t)dt

逆变换:            f(t) = (1/2πi)∫_{γ-i∞}^{γ+i∞} e^{st}F(s)ds

微分性质:          L{f'} = sF(s) - f(0)

卷积定理:          L{f*g} = F(s)G(s)
```
