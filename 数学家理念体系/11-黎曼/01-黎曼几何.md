# 黎曼几何

## 概述

黎曼几何是19世纪数学最伟大的成就之一，由伯恩哈德·黎曼在1854年的就职演讲《关于几何基础中的假设》中创立。它彻底改变了人类对空间的理解，从欧几里得几何的刚性框架中解放出来，为现代物理学（特别是广义相对论）奠定了数学基础。

---

## 历史背景

### 欧几里得几何的局限

**第五公设问题**

欧几里得《几何原本》中的第五公设（平行公设）长期以来被认为不如其他公设直观。数学家们试图证明它可以从其他公设导出，但这些尝试最终导致了非欧几何的发现。

**高斯的内蕴几何**

卡尔·弗里德里希·高斯在1827年的论文《曲面的一般研究》（Disquisitiones Generales circa Superficies Curvas）中提出了曲面的**内蕴几何**概念：
- 曲面的几何性质可以在曲面本身上测量，而不需要参考外部空间
- 高斯曲率是内蕴性质，可以通过曲面上的度量计算
- 高斯-博内定理：曲面的拓扑与几何之间的深刻联系

高斯的工作为黎曼的推广奠定了基础，但高斯只考虑了嵌入在三维欧氏空间中的二维曲面。

### 黎曼的革命性推广

**n维流形的概念**

黎曼将高斯的思想推广到任意维度的抽象流形：
- 空间本身可以具有任意维度（n维流形）
- 空间的几何由度规张量决定
- 欧氏几何只是无穷多种可能几何中的一种

**就职演讲的背景**

1854年6月10日，黎曼在哥廷根大学哲学系发表就职演讲。高斯出席了这次演讲——这是他罕见的公开露面——并对黎曼的工作给予了高度赞扬。

## 数学基础

### 1. 微分流形

**定义**

一个n维**微分流形**M是满足以下条件的拓扑空间：
- 局部同胚于n维欧氏空间R^n
- 具有光滑的坐标变换

**局部坐标**

流形上的每一点p可以用局部坐标(x¹, x², ..., xⁿ)描述。不同坐标系之间的变换是光滑的（无限可微）。

**切空间**

在流形上的每一点p，可以定义**切空间**T_pM，它是n维向量空间，由该点所有可能的切向量组成。

### 2. 黎曼度规

**度规张量的定义**

黎曼度规g是在流形M上定义的对称正定(0,2)型张量场：

$$g = \sum_{i,j=1}^{n} g_{ij}(x) dx^i \otimes dx^j$$

其中$g_{ij} = g_{ji}$，且对于任何非零向量v，有$g(v,v) > 0$。

**度规的物理意义**
- **长度**：曲线长度 $L = \int \sqrt{g_{ij}\dot{x}^i\dot{x}^j} dt$
- **角度**：两向量夹角由度规内积决定
- **体积**：体积元 $dV = \sqrt{\det(g_{ij})} dx^1 \wedge ... \wedge dx^n$

**例子**

**欧氏空间**：在R^n中，标准度规为
$$g = \sum_{i=1}^{n} (dx^i)^2$$
度规张量为单位矩阵。

**球面**：S²上的标准度规在球坐标中为
$$g = d\theta^2 + \sin^2\theta \, d\phi^2$$

**双曲空间**：双曲平面上的庞加莱圆盘模型
$$g = \frac{4}{(1-r^2)^2}(dr^2 + r^2 d\theta^2)$$

### 3. 克里斯托费尔符号与协变导数

**克里斯托费尔符号**

为了对向量场进行微分，需要定义**协变导数**。克里斯托费尔符号（联络系数）定义为：

$$\Gamma^k_{ij} = \frac{1}{2}g^{kl}\left(\partial_i g_{jl} + \partial_j g_{il} - \partial_l g_{ij}\right)$$

**协变导数**

向量场V的协变导数为：
$$\nabla_i V^j = \partial_i V^j + \Gamma^j_{ik} V^k$$

**协变导数的性质**
- 满足莱布尼茨法则
- 与度规相容：$\nabla_k g_{ij} = 0$
- 无挠性：$\Gamma^k_{ij} = \Gamma^k_{ji}$

### 4. 黎曼曲率张量

**定义**

黎曼曲率张量是描述空间弯曲程度的核心几何量：

$$R^l_{ijk} = \partial_j \Gamma^l_{ik} - \partial_k \Gamma^l_{ij} + \Gamma^l_{jm}\Gamma^m_{ik} - \Gamma^l_{km}\Gamma^m_{ij}$$

**几何意义**

曲率张量测量向量沿无穷小环路平行移动后的变化：
- 平坦空间：R = 0
- 弯曲空间：R ≠ 0

**曲率张量的性质**
- 反对称性：$R_{ijkl} = -R_{jikl} = -R_{ijlk}$
- 第一比安基恒等式：$R_{ijkl} + R_{iklj} + R_{iljk} = 0$
- 第二比安基恒等式：$\nabla_m R_{ijkl} + \nabla_k R_{ijlm} + \nabla_l R_{ijmk} = 0$

**缩并曲率**

- **里奇张量**：$R_{ij} = R^k_{ikj}$
- **标量曲率**：$R = g^{ij}R_{ij}$
- **爱因斯坦张量**：$G_{ij} = R_{ij} - \frac{1}{2}g_{ij}R$

### 5. 测地线

**定义**

**测地线**是局部最短的曲线，是欧氏空间中直线的推广。

**测地线方程**

测地线满足二阶常微分方程：
$$\frac{d^2x^k}{dt^2} + \Gamma^k_{ij}\frac{dx^i}{dt}\frac{dx^j}{dt} = 0$$

**变分原理**

测地线也可以通过变分原理得到：它是使弧长泛函取极值的曲线。

**例子**
- **欧氏空间**：测地线是直线
- **球面**：测地线是大圆（如经线和赤道）
- **双曲空间**：测地线是与边界垂直的圆弧或直径

### 6. 平行移动

**概念**

在弯曲空间中，"平行"的概念需要通过**平行移动**来定义。给定一条曲线和曲线起点的向量，可以将该向量沿曲线"平行移动"到终点。

**数学定义**

向量V沿曲线γ(t)平行移动满足：
$$\frac{DV^i}{dt} = \frac{dV^i}{dt} + \Gamma^i_{jk}\frac{dx^j}{dt}V^k = 0$$

**和乐群**

向量沿闭合曲线平行移动一周后，一般不与原向量重合。这种变换构成**和乐群**，是空间整体几何的重要特征。

## 核心定理

### 高斯-博内定理

**定理陈述**

对于紧致二维黎曼流形M（无边）：
$$\int_M K \, dA = 2\pi \chi(M)$$

其中K是高斯曲率，χ(M)是欧拉示性数。

**意义**
- 将局部几何量（曲率）与全局拓扑量（欧拉示性数）联系
- 证明曲率是内蕴性质
- 是指标定理的原型

### 黎曼几何基本定理

**存在唯一性**

在黎曼流形上存在唯一的无挠联络与度规相容（Levi-Civita联络）。

**证明思路**
- 利用度规相容性条件$\nabla_k g_{ij} = 0$
- 利用无挠条件$\Gamma^k_{ij} = \Gamma^k_{ji}$
- 解出克里斯托费尔符号的唯一表达式

### 霍奇分解定理

**定理陈述**

在紧致黎曼流形上，任意微分形式可以唯一分解为：
$$\omega = d\alpha + \delta\beta + \gamma$$

其中γ是调和形式（$\Delta \gamma = 0$）。

**推论**
- 德拉姆上同调与调和形式之间的联系
- 贝蒂数的计算

## 特殊类型的黎曼流形

### 常曲率空间

**定义**

曲率张量满足：
$$R_{ijkl} = K(g_{ik}g_{jl} - g_{il}g_{jk})$$

其中K是常数。

**分类**
- **K > 0**：球面几何（椭圆几何）
- **K = 0**：欧氏几何（抛物几何）
- **K < 0**：双曲几何（双曲几何）

**统一性**

这三种几何是完备的、单连通的常曲率空间的唯三种可能。

### 爱因斯坦流形

**定义**

满足爱因斯坦方程的黎曼流形：
$$R_{ij} = \lambda g_{ij}$$

其中λ是常数（爱因斯坦常数）。

**例子**
- 球面
- 双曲空间
- 卡拉比-丘流形（Ricci平坦，λ=0）

**物理意义**

爱因斯坦流形是真空爱因斯坦场方程的解，在广义相对论中具有核心地位。

### 凯勒流形

**定义**

具有相容复结构的黎曼流形，满足：
- 复结构J与度规相容：$g(JX, JY) = g(X, Y)$
- 基本形式$\omega(X,Y) = g(JX, Y)$是闭的

**例子**
- 复射影空间CP^n
- 复环面
- 卡拉比-丘流形

**重要性**
- 代数几何的核心对象
- 弦理论的紧化空间
- 镜像对称的研究对象

## 广义相对论中的应用

### 时空的几何化

**四维伪黎曼流形**

在广义相对论中，时空是四维**伪黎曼流形**（度规符号为(−,+,+,+)）：
$$ds^2 = g_{\mu\nu}dx^\mu dx^\nu$$

**爱因斯坦场方程**

$$G_{\mu\nu} + \Lambda g_{\mu\nu} = \frac{8\pi G}{c^4}T_{\mu\nu}$$

其中：
- $G_{\mu\nu} = R_{\mu\nu} - \frac{1}{2}g_{\mu\nu}R$ 是爱因斯坦张量
- Λ是宇宙学常数
- $T_{\mu\nu}$ 是应力-能量张量

### 引力作为几何

**等效原理**
- 惯性质量等于引力质量
- 局部惯性系中物理定律与狭义相对论相同

**弯曲时空中的自由落体**
- 自由粒子沿测地线运动
- 引力不是"力"，而是时空弯曲的表现

### 经典解

**史瓦西解**（1916）

描述球对称静态引力场的外部解：
$$ds^2 = -\left(1-\frac{2GM}{c^2r}\right)c^2dt^2 + \left(1-\frac{2GM}{c^2r}\right)^{-1}dr^2 + r^2(d\theta^2 + \sin^2\theta \, d\phi^2)$$

- 描述恒星外部的引力场
- 预言黑洞的存在

**弗里德曼-勒梅特-罗伯逊-沃克度规**

描述均匀各向同性宇宙：
$$ds^2 = -c^2dt^2 + a(t)^2\left[\frac{dr^2}{1-kr^2} + r^2(d\theta^2 + \sin^2\theta \, d\phi^2)\right]$$

其中a(t)是尺度因子，k是曲率参数。

## 现代发展

### 黎曼几何的分支

**大范围黎曼几何**
- 研究流形的整体几何性质
- 比较几何（Toponogov定理、Gromov紧性定理）
- 曲率与拓扑的关系

**几何分析**
- 偏微分方程在几何中的应用
- 里奇流（Ricci Flow）：$\partial_t g_{ij} = -2R_{ij}$
- 庞加莱猜想的证明（佩雷尔曼，2003）

**指标定理**
- Atiyah-Singer指标定理
- 将分析指标与拓扑指标联系
- 推广了高斯-博内定理

### 在理论物理中的应用

**弦理论**
- 额外维度的紧化使用卡拉比-丘流形
- 镜像对称：两个不同的卡拉比-丘流形产生相同的物理

**规范场论**
- 主纤维丛上的联络
- 曲率对应场强
- 陈-西蒙斯理论

**量子引力**
- 圈量子引力
- 渐进安全引力
- 因果集理论

## 总结

黎曼几何从1854年的就职演讲发展到今天，已成为现代数学和物理学的核心支柱。它不仅是广义相对论的数学基础，也是微分几何、代数几何、复几何、几何分析等多个数学分支的基础工具。黎曼关于空间内蕴几何的深刻洞见，至今仍在指引着数学和物理学的前沿探索。
