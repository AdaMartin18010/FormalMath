## 📑 项目概述

本项目旨在系统化分析**斯里尼瓦瑟·拉马努金（Srinivasa Ramanujan, 1887-1920）**的数学理念、方法论和具体数学贡献，聚焦其数学思想、数论方法和对现代数学的影响。

**参照标准**：格洛腾迪克数学理念项目（已完成90%+）

**核心定位**：

- ✅ **数学思想**：直觉数学、数论直觉、模形式理论
- ✅ **具体数学内容**：拉马努金猜想、模形式、q-级数、连分数、椭圆函数
- ✅ **历史影响**：对Hardy、Littlewood、现代数论的影响
- ✅ **现代验证**：在现代数论、模形式理论、Langlands纲领中的应用
