## 一、导数理论

### 1.1 导数的定义

**牛顿的流数定义**：

牛顿将变量的变化率称为"流数"（fluxion）。

**流数的定义**：

设 $x$ 是一个流（变量），$x$ 的流数 $\dot{x}$ 定义为：

$$\dot{x} = \lim_{\Delta t \to 0} \frac{\Delta x}{\Delta t}$$

**现代导数的定义**：

设 $y = f(x)$，则 $y$ 关于 $x$ 的导数定义为：

$$f'(x) = \lim_{\Delta x \to 0} \frac{f(x+\Delta x) - f(x)}{\Delta x}$$

**对应关系**：

- **流数** $\dot{x}$ 对应 **导数** $\frac{dx}{dt}$
- **流数比** $\dot{y}/\dot{x}$ 对应 **导数** $\frac{dy}{dx}$

**几何意义**：

- **切线斜率**：$\dot{y}/\dot{x}$ 表示曲线 $y = f(x)$ 在点 $(x, y)$ 处的切线斜率
- **瞬时变化率**：$\dot{x}$ 表示 $x$ 的瞬时变化率

### 1.2 导数的性质

**基本性质**：

1. **线性性**：$(af + bg)' = af' + bg'$
2. **乘积法则**：$(fg)' = f'g + fg'$
3. **商法则**：$\left(\frac{f}{g}\right)' = \frac{f'g - fg'}{g^2}$
4. **链式法则**：$(f \circ g)' = (f' \circ g) \cdot g'$

**高阶导数**：

- **一阶导数**：$f'(x)$ 或 $\dot{x}$
- **二阶导数**：$f''(x)$ 或 $\ddot{x}$
- **$n$阶导数**：$f^{(n)}(x)$

### 1.3 导数的计算

**基本函数的导数**：

1. **幂函数**：$(x^n)' = nx^{n-1}$
2. **指数函数**：$(e^x)' = e^x$，$(a^x)' = a^x \ln a$
3. **对数函数**：$(\ln x)' = \frac{1}{x}$，$(\log_a x)' = \frac{1}{x \ln a}$
4. **三角函数**：
   - $(\sin x)' = \cos x$
   - $(\cos x)' = -\sin x$
   - $(\tan x)' = \sec^2 x$

## 二、积分理论

### 2.1 积分的定义

**不定积分**：

如果 $F'(x) = f(x)$，则 $F$ 是 $f$ 的原函数，$f$ 的不定积分为：

$$\int f(x)dx = F(x) + C$$

**定积分**：

函数 $f$ 在区间 $[a, b]$ 上的定积分定义为：

$$\int_a^b f(x)dx = \lim_{n \to \infty} \sum_{i=1}^n f(\xi_i)\Delta x_i$$

**几何意义**：

- **不定积分**：表示 $f$ 的所有原函数（相差常数）
- **定积分**：表示曲线 $y = f(x)$ 从 $a$ 到 $b$ 的面积

### 2.2 积分的性质

**基本性质**：

1. **线性性**：$\int [\alpha f + \beta g]dx = \alpha\int f dx + \beta\int g dx$
2. **区间可加性**：$\int_a^b f dx + \int_b^c f dx = \int_a^c f dx$
3. **积分不等式**：如果 $f \\leq g$，则 $\int_a^b f dx \\leq \int_a^b g dx$

### 2.3 积分的计算

**基本积分公式**：

1. **幂函数**：$\int x^n dx = \frac{x^{n+1}}{n+1} + C$（$n \neqqq -1$）
2. **指数函数**：$\int e^x dx = e^x + C$
3. **三角函数**：$\int \sin x dx = -\cos x + C$，$\int \cos x dx = \sin x + C$

**计算方法**：

- **换元积分法**：$\int f(g(x))g'(x)dx = \int f(u)du$
- **分部积分法**：$\int u dv = uv - \int v du$

## 三、微积分基本定理

### 3.1 基本定理的内容

**第一部分**：

如果 $F(x) = \int_a^x f(t)dt$，则 $F'(x) = f(x)$

**第二部分**：

如果 $F$ 是 $f$ 的任意原函数，则：

$$\int_a^b f(x)dx = F(b) - F(a)$$

### 3.2 基本定理的证明

**第一部分的证明**：

根据导数的定义和积分中值定理。

**第二部分的证明**：

使用第一部分和原函数的性质。

### 3.3 基本定理的应用

**计算定积分**：

使用基本定理第二部分计算定积分。

**求原函数**：

使用基本定理第一部分，通过积分求原函数。

## 四、牛顿的贡献

### 4.1 流数法（导数）

**概念建立**：

- 定义了流（变量）和流数（变化率）的概念
- 建立了流数的记号系统

**规则建立**：

- 建立了流数的基本计算规则
- 发现了复合函数的流数（链式法则）

### 4.2 反流数法（积分）

**概念建立**：

- 定义了反流数（积分的逆运算）的概念
- 建立了积分与面积的关系

**方法建立**：

- 使用级数展开计算积分
- 建立了积分的基本方法

### 4.3 基本定理的发现

**发现过程**：

牛顿发现微分与积分是互逆运算，建立了微积分基本定理。

**历史意义**：

- 统一了微分学和积分学
- 建立了微积分的完整理论
- 为现代分析学奠定了基础

## 五、理论的发展

### 5.1 严格化过程

**18-19世纪**：

- **欧拉**：发展了微积分的符号系统
- **拉格朗日**：发展了分析方法
- **柯西**：建立了严格的极限理论
- **魏尔斯特拉斯**：建立了 $\varepsilon-\delta$ 极限定义

### 5.2 推广与发展

**多元微积分**：

- 推广到多元函数
- 偏导数、多重积分

**向量微积分**：

- 梯度、散度、旋度
- 线积分、面积分

### 5.3 现代理论

**勒贝格积分**：

- 推广了黎曼积分
- 应用到更一般的函数

**分布理论**：

- 推广了函数概念
- 应用到广义函数
