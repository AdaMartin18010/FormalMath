## 一、无穷级数的概念

### 1.1 级数的定义

**无穷级数**：

无穷级数是无穷多个数的和：

$$\sum_{n=0}^\infty a_n = a_0 + a_1 + a_2 + \cdots$$

**部分和**：

前 $n$ 项的和称为部分和：

$$S_n = \sum_{k=0}^{n-1} a_k = a_0 + a_1 + \cdots + a_{n-1}$$

**级数的和**：

如果部分和序列 $\{S_n\}$ 收敛，则级数收敛，其和为：

$$S = \lim_{n \to \infty} S_n$$

### 1.2 级数的收敛性

**收敛定义**：

级数 $\sum_{n=0}^\infty a_n$ 收敛，如果部分和序列 $\{S_n\}$ 收敛。

**发散**：

如果部分和序列不收敛，则级数发散。

**绝对收敛**：

如果 $\sum_{n=0}^\infty |a_n|$ 收敛，则级数绝对收敛。

**条件收敛**：

如果级数收敛但不绝对收敛，则条件收敛。

### 1.3 级数的和

**计算方法**：

1. **直接求和**：对于某些简单级数可以直接求和
2. **极限方法**：使用极限计算和
3. **积分方法**：使用积分计算和

**例子**：

**几何级数**：

$$\sum_{n=0}^\infty r^n = \frac{1}{1-r} \quad (|r| < 1)$$

## 三、级数的类型

### 3.1 几何级数

**定义**：

$$\sum_{n=0}^\infty ar^n = a + ar + ar^2 + \cdots$$

**和**：

当 $|r| < 1$ 时：

$$\sum_{n=0}^\infty ar^n = \frac{a}{1-r}$$

**应用**：

- 计算无限和
- 近似计算
- 其他应用

### 3.2 幂级数

**定义**：

$$\sum_{n=0}^\infty a_n x^n = a_0 + a_1 x + a_2 x^2 + \cdots$$

**收敛半径**：

幂级数在 $|x| < R$ 时收敛，其中 $R$ 是收敛半径。

**应用**：

- 函数展开
- 函数研究
- 数值计算

### 3.3 泰勒级数

**定义**：

如果函数 $f$ 在 $x=a$ 处无限可微，则：

$$f(x) = \sum_{n=0}^\infty \frac{f^{(n)}(a)}{n!}(x-a)^n$$

**应用**：

- 函数展开
- 近似计算
- 函数研究
