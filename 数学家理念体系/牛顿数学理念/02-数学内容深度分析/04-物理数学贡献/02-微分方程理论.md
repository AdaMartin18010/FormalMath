## 一、微分方程的建立

### 1.1 历史背景

**17世纪的数学问题**：

在17世纪，数学家面临描述变化过程的问题：

- 如何描述物体的运动？
- 如何描述变化过程？
- 如何求解变化问题？

**传统方法**：

- 主要使用几何方法
- 缺乏代数工具
- 难以描述变化

**需要**：

- 描述变化的数学工具
- 求解变化问题的方法
- 统一的变化理论

### 1.2 牛顿的贡献

**微分方程的建立**：

牛顿建立了微分方程理论，用于描述物理过程。

**核心贡献**：

1. **运动方程**：
   - 建立了运动微分方程
   - 描述了物体运动
   - 解决了运动问题

2. **求解方法**：
   - 发展了求解方法
   - 使用级数求解
   - 应用到实际问题

3. **应用扩展**：
   - 应用到物理问题
   - 应用到几何问题
   - 扩展到其他问题

### 1.3 方程的类型

**常微分方程**：

只涉及一个自变量的微分方程：

$$\frac{dy}{dx} = f(x, y)$$

**偏微分方程**：

涉及多个自变量的微分方程：

$$\frac{\partial u}{\partial t} = \frac{\partial^2 u}{\partial x^2}$$

**方程阶数**：

方程中最高阶导数的阶数。

## 二、常微分方程

### 2.1 一阶方程

**一般形式**：

$$\frac{dy}{dx} = f(x, y)$$

**可分离变量方程**：

$$\frac{dy}{dx} = g(x)h(y)$$

**线性方程**：

$$\frac{dy}{dx} + P(x)y = Q(x)$$

**齐次方程**：

$$\frac{dy}{dx} = f\left(\frac{y}{x}\right)$$

### 2.2 二阶方程

**一般形式**：

$$\frac{d^2y}{dx^2} = f\left(x, y, \frac{dy}{dx}\right)$$

**线性齐次方程**：

$$\frac{d^2y}{dx^2} + p(x)\frac{dy}{dx} + q(x)y = 0$$

**常系数方程**：

$$\frac{d^2y}{dx^2} + a\frac{dy}{dx} + by = 0$$

**例子**：

**简谐运动**：

$$m\frac{d^2x}{dt^2} + kx = 0$$

解为：

$$x(t) = A\sin(\omega t + \phi)$$

其中 $\omega = \sqrt{k/m}$。

### 2.3 高阶方程

**一般形式**：

$$\frac{d^ny}{dx^n} = f\left(x, y, \frac{dy}{dx}, \ldots, \frac{d^{n-1}y}{dx^{n-1}}\right)$$

**线性方程**：

$$\sum_{k=0}^n a_k(x)\frac{d^ky}{dx^k} = f(x)$$

## 三、微分方程的求解

### 3.1 解析方法

**直接积分**：

对于某些简单方程，可以直接积分。

**变量分离**：

对于可分离变量方程，分离变量后积分。

**积分因子**：

对于线性方程，使用积分因子。

### 3.2 级数方法

**级数展开**：

将解展开为级数：

$$y(x) = \sum_{n=0}^\infty a_n x^n$$

**系数确定**：

代入方程确定系数。

**应用**：

- 求解复杂方程
- 近似求解
- 理论研究

### 3.3 数值方法

**欧拉法**：

$$y_{n+1} = y_n + hf(x_n, y_n)$$

**龙格-库塔法**：

使用更高阶方法提高精度。

**应用**：

- 数值求解
- 计算机计算
- 实际应用

## 四、微分方程的应用

### 4.1 在物理中的应用

**运动方程**：

描述物体运动：

$$m\frac{d^2x}{dt^2} = F(x, t)$$

**波动方程**：

描述波动现象：

$$\frac{\partial^2u}{\partial t^2} = c^2\frac{\partial^2u}{\partial x^2}$$

**热传导方程**：

描述热传导：

$$\frac{\partial T}{\partial t} = \alpha\frac{\partial^2T}{\partial x^2}$$

### 4.2 在几何中的应用

**曲线方程**：

使用微分方程描述曲线。

**例子**：

- **悬链线**：$\frac{d^2y}{dx^2} = \frac{1}{a}\sqrt{1 + \left(\frac{dy}{dx}\right)^2}$
- **最速降线**：使用变分法求解

### 4.3 在其他领域的应用

**生物学**：

- 人口增长模型
- 生物系统模型

**经济学**：

- 经济增长模型
- 市场模型

**工程**：

- 控制系统
- 信号处理
