## 一、曲线的分类

### 1.1 代数曲线

**定义**：

代数曲线是可以用代数方程表示的曲线。

**例子**：

1. **直线**：$ax + by + c = 0$
2. **圆**：$(x-a)^2 + (y-b)^2 = r^2$
3. **椭圆**：$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$
4. **双曲线**：$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$
5. **抛物线**：$y = ax^2 + bx + c$

**次数**：

曲线的次数是方程的最高次数。

### 1.2 超越曲线

**定义**：

超越曲线是不能用代数方程表示的曲线。

**例子**：

1. **指数曲线**：$y = e^x$
2. **对数曲线**：$y = \ln x$
3. **三角函数曲线**：$y = \sin x$, $y = \cos x$
4. **摆线**：参数方程表示

### 1.3 参数曲线

**定义**：

参数曲线用参数方程表示：

$$x = x(t), \quad y = y(t)$$

**例子**：

1. **圆**：$x = r\cos t$, $y = r\sin t$
2. **椭圆**：$x = a\cos t$, $y = b\sin t$
3. **摆线**：$x = r(t - \sin t)$, $y = r(1 - \cos t)$

## 二、曲线的性质

### 2.1 切线

**切线定义**：

曲线在点 $P$ 处的切线是过 $P$ 点且与曲线在该点方向相同的直线。

**切线斜率**：

对于曲线 $y = f(x)$，在点 $(a, f(a))$ 处的切线斜率为：

$$m = f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h}$$

**切线方程**：

$$y - f(a) = f'(a)(x - a)$$

### 2.2 曲率

**曲率定义**：

曲率表示曲线的弯曲程度：

$$\kappa = \frac{|y''|}{(1 + (y')^2)^{3/2}}$$

**曲率半径**：

$$R = \frac{1}{\kappa}$$

### 2.3 渐近线

**垂直渐近线**：

如果 $\lim_{x \to a} f(x) = \pm\infty$，则 $x = a$ 是垂直渐近线。

**水平渐近线**：

如果 $\lim_{x \to \pm\infty} f(x) = L$，则 $y = L$ 是水平渐近线。

**斜渐近线**：

如果 $\lim_{x \to \pm\infty} [f(x) - (mx + b)] = 0$，则 $y = mx + b$ 是斜渐近线。

摆线是圆在直线上滚动时，圆上一点的轨迹。

**参数方程**：

$$x = r(t - \sin t), \quad y = r(1 - \cos t)$$

- 最速降线问题
- 钟摆问题
