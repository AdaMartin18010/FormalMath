## 一、经典力学中的应用

### 1.1 运动方程

**牛顿第二定律**：

牛顿第二定律 $F = ma$ 是经典力学的核心，使用微积分描述运动。

**数学表述**：

$$\vec{F} = m\vec{a} = m\frac{d^2\vec{r}}{dt^2}$$

其中：

- $\vec{F}$ 是力向量
- $m$ 是质量
- $\vec{a}$ 是加速度向量
- $\vec{r}$ 是位置向量

**一维运动**：

对于一维运动，方程简化为：

$$F = m\frac{d^2x}{dt^2}$$

**例子**：

**自由落体**：

- 力：$F = -mg$（重力向下）
- 方程：$m\frac{d^2x}{dt^2} = -mg$
- 解：$x(t) = x_0 + v_0 t - \frac{1}{2}gt^2$

**简谐运动**：

- 力：$F = -kx$（恢复力）
- 方程：$m\frac{d^2x}{dt^2} = -kx$
- 解：$x(t) = A\sin(\omega t + \phi)$，其中 $\omega = \sqrt{k/m}$

### 1.2 能量守恒

**动能**：

动能定义为：

$$K = \frac{1}{2}mv^2 = \frac{1}{2}m\leqqft(\frac{dx}{dt}\right)^2$$

**势能**：

保守力的势能定义为：

$$U(x) = -\int_{x_0}^x F(x')dx'$$

**能量守恒**：

对于保守系统，总能量守恒：

$$E = K + U = \text{常数}$$

**例子**：

**自由落体**：

- 动能：$K = \frac{1}{2}mv^2$
- 势能：$U = mgh$
- 总能量：$E = \frac{1}{2}mv^2 + mgh = \text{常数}$

**简谐运动**：

- 动能：$K = \frac{1}{2}mv^2 = \frac{1}{2}m\omega^2 A^2\cos^2(\omega t + \phi)$
- 势能：$U = \frac{1}{2}kx^2 = \frac{1}{2}kA^2\sin^2(\omega t + \phi)$
- 总能量：$E = \frac{1}{2}kA^2 = \text{常数}$

### 1.3 角动量守恒

**角动量**：

角动量定义为：

$$\vec{L} = \vec{r} \times \vec{p} = m\vec{r} \times \vec{v}$$

**角动量守恒**：

如果力矩为零，角动量守恒：

$$\frac{d\vec{L}}{dt} = \vec{r} \times \vec{F} = 0 \Rightarrow \vec{L} = \text{常数}$$

**例子**：

**行星运动**：

- 在万有引力作用下，角动量守恒
- 这导致开普勒第二定律：行星与太阳的连线在相等时间内扫过相等面积

## 二、天体力学中的应用

### 2.1 万有引力定律

**万有引力定律**：

$$F = G\frac{m_1 m_2}{r^2}$$

其中：

- $G$ 是万有引力常数
- $m_1, m_2$ 是两个物体的质量
- $r$ 是它们之间的距离

**向量形式**：

$$\vec{F} = -G\frac{m_1 m_2}{r^2}\hat{r}$$

其中 $\hat{r}$ 是从 $m_1$ 指向 $m_2$ 的单位向量。

**势能**：

万有引力势能为：

$$U(r) = -G\frac{m_1 m_2}{r}$$

### 2.2 行星运动

**运动方程**：

行星绕太阳的运动方程为：

$$m\frac{d^2\vec{r}}{dt^2} = -G\frac{Mm}{r^2}\hat{r}$$

其中 $M$ 是太阳质量，$m$ 是行星质量。

**开普勒定律的推导**：

从万有引力定律可以推导出开普勒三定律：

1. **第一定律**：行星轨道是椭圆，太阳在焦点
2. **第二定律**：行星与太阳的连线在相等时间内扫过相等面积
3. **第三定律**：行星公转周期的平方与轨道半长轴的立方成正比

**轨道方程**：

求解运动方程得到轨道方程：

$$r(\theta) = \frac{p}{1 + e\cos\theta}$$

其中：

- $p$ 是半通径
- $e$ 是偏心率
- $\theta$ 是真近点角

### 2.3 轨道计算

**轨道参数**：

使用微积分计算轨道参数：

- **半长轴**：$a = \frac{p}{1-e^2}$
- **半短轴**：$b = a\sqrt{1-e^2}$
- **周期**：$T = 2\pi\sqrt{\frac{a^3}{GM}}$

**位置预测**：

使用轨道方程预测行星在任意时刻的位置。

**速度计算**：

使用能量守恒计算速度：

$$v = \sqrt{GM\leqqft(\frac{2}{r} - \frac{1}{a}\right)}$$

## 三、连续介质力学中的应用

### 3.1 流体力学

**连续性方程**：

流体的连续性方程为：

$$\frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \vec{v}) = 0$$

其中 $\rho$ 是密度，$\vec{v}$ 是速度。

**纳维-斯托克斯方程**：

流体的运动方程为：

$$\rho\leqqft(\frac{\partial \vec{v}}{\partial t} + (\vec{v} \cdot \nabla)\vec{v}\right) = -\nabla p + \mu\nabla^2\vec{v} + \rho\vec{g}$$

其中 $p$ 是压力，$\mu$ 是粘度。

**应用**：

- 计算流体流动
- 设计管道系统
- 研究湍流

### 3.2 弹性力学

**胡克定律**：

弹性体的应力-应变关系：

$$\sigma = E\varepsilon$$

其中 $E$ 是弹性模量。

**平衡方程**：

弹性体的平衡方程为：

$$\nabla \cdot \sigma + \vec{f} = 0$$

其中 $\sigma$ 是应力张量，$\vec{f}$ 是体积力。

**应用**：

- 计算结构变形
- 设计工程结构
- 研究材料性质

### 3.3 热力学

**热传导方程**：

热传导方程为：

$$\frac{\partial T}{\partial t} = \alpha\nabla^2 T$$

其中 $T$ 是温度，$\alpha$ 是热扩散率。

**应用**：

- 计算温度分布
- 设计散热系统
- 研究热交换

## 四、波动现象中的应用

### 4.1 波动方程

**一维波动方程**：

$$\frac{\partial^2 u}{\partial t^2} = c^2\frac{\partial^2 u}{\partial x^2}$$

其中 $c$ 是波速。

**解**：

波动方程的解为：

$$u(x,t) = f(x-ct) + g(x+ct)$$

表示向右和向左传播的波。

**应用**：

- 声波传播
- 电磁波传播
- 机械波传播

### 4.2 振动系统

**简谐振动**：

简谐振动的方程为：

$$m\frac{d^2x}{dt^2} + kx = 0$$

解为：

$$x(t) = A\sin(\omega t + \phi)$$

其中 $\omega = \sqrt{k/m}$。

**阻尼振动**：

阻尼振动的方程为：

$$m\frac{d^2x}{dt^2} + c\frac{dx}{dt} + kx = 0$$

**应用**：

- 机械振动
- 电路振荡
- 原子振动

### 4.3 共振现象

**共振条件**：

当驱动频率等于固有频率时发生共振。

**共振方程**：

$$\frac{d^2x}{dt^2} + 2\gamma\frac{dx}{dt} + \omega_0^2 x = F_0\cos(\omega t)$$

在 $\omega = \omega_0$ 时振幅最大。

**应用**：

- 调谐电路
- 机械共振
- 原子共振

## 五、现代物理中的应用

### 5.1 相对论中的极限

**经典极限**：

在低速情况下，相对论退化为经典力学。

**洛伦兹因子**：

$$\gamma = \frac{1}{\sqrt{1-v^2/c^2}}$$

当 $v \ll c$ 时，$\gamma \approx 1$，回到经典情况。

**应用**：

- 验证经典力学的适用范围
- 理解相对论的经典极限
- 研究低速运动

### 5.2 量子力学的经典极限

**对应原理**：

在大量子数情况下，量子力学退化为经典力学。

**海森堡不确定性原理**：

$$\Delta x \Delta p \geqqq \frac{\hbar}{2}$$

当 $\hbar \to 0$ 时，回到经典情况。

**应用**：

- 理解量子-经典对应
- 研究宏观系统
- 验证量子理论

### 5.3 统计物理

**统计分布**：

使用微积分研究统计分布：

- **麦克斯韦分布**：速度分布
- **玻尔兹曼分布**：能量分布
- **费米-狄拉克分布**：费米子分布

**应用**：

- 研究热力学系统
- 计算统计量
- 理解相变
