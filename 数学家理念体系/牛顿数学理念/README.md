## 📚 核心内容方向

### 1. 微积分

- **历史背景**: 1660年代流数法的发现
- **数学内容**: 微积分基本定理、导数、积分
- **应用**: 物理、几何、优化

### 2. 流数法

- **核心思想**: 变化率、瞬时速度
- **方法**: 无穷小方法、极限思想
- **影响**: 现代微积分的基础

### 3. 数学物理

- **贡献**: 用数学方法解决物理问题
- **方法**: 微分方程、数学建模
- **应用**: 经典力学、天体力学
