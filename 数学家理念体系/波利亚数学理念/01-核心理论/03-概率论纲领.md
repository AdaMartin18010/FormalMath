## 一、概率论理论的建立

### 1.1 历史背景

**概率论理论建立的背景**：

波利亚在研究随机问题时，建立了概率论理论，特别是在随机游走方面做出了重要贡献。

**历史意义**：

- **1921年论文**：《Über eine Aufgabe der Wahrscheinlichkeitsrechnung betreffend die Irrfahrt im Straßennetz》
- **核心贡献**：研究随机游走的性质
- **影响**：为现代概率论奠定基础

### 1.2 理论的建立

**理论的建立**：

波利亚建立了概率论理论的基础，特别是随机游走理论。

**数学表述**：

1. **随机游走的定义**：
   $$S_0 = 0, \quad S_n = \sum_{i=1}^n X_i$$
   其中 $X_i$ 是独立同分布的随机变量。

2. **Pólya定理**（1921）：
   对于 $d$ 维随机游走：
   $$P_0(d) = \begin{cases}
   1 & d = 1, 2 \\
   < 1 & d \geq 3
   \end{cases}$$
   其中 $P_0(d)$ 是返回原点的概率。

**例子**：

1. **一维随机游走**：$P_0(1) = 1$（几乎必然返回）
2. **二维随机游走**：$P_0(2) = 1$（几乎必然返回）
3. **三维随机游走**：$P_0(3) < 1$（不一定返回）

## 二、随机游走理论

### 2.1 游走的形式

**随机游走的形式**：

随机游走是概率论的重要形式，用于研究随机过程的性质。

**数学表述**：

1. **一维随机游走**：
   $$S_n = \sum_{i=1}^n X_i, \quad X_i \in \{-1, +1\}$$
   其中 $P(X_i = 1) = p$，$P(X_i = -1) = 1-p$。

2. **对称随机游走**：
   当 $p = \frac{1}{2}$ 时，称为对称随机游走：
   $$P(X_i = 1) = P(X_i = -1) = \frac{1}{2}$$

3. **多维随机游走**：
   $$S_n = (S_n^{(1)}, \ldots, S_n^{(d)}), \quad S_n^{(i)} = \sum_{j=1}^n X_j^{(i)}$$

**例子**：

1. **一维随机游走**：在数轴上，每一步以概率 $\frac{1}{2}$ 向左或向右移动
2. **二维随机游走**：在平面上，每一步随机选择一个方向
3. **三维随机游走**：在空间中，每一步随机选择一个方向

### 2.2 游走的意义

**游走的意义**：

随机游走揭示了随机过程的深层结构，是概率论的重要模型。

**数学表述**：

1. **马尔可夫性**：
   $$P(S_{n+1} = j \mid S_n = i, S_{n-1}, \ldots, S_0) = P(S_{n+1} = j \mid S_n = i)$$

2. **中心极限定理**：
   $$\frac{S_n}{\sqrt{n}} \xrightarrow{d} N(0, \sigma^2)$$
   其中 $\sigma^2 = \text{Var}(X_i)$。

**例子**：

1. **布朗运动**：随机游走的连续极限是布朗运动
2. **扩散过程**：随机游走模型描述扩散过程
3. **统计物理**：随机游走用于研究统计物理系统
