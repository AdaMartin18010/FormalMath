## 一、生成函数的定义

### 1.1 函数的概念

**生成函数的定义**：

生成函数是组合数学的核心工具，用于研究序列和组合问题。

**数学定义**：

设序列 $(a_n)_{n \geq 0}$，其**普通生成函数**（OGF）定义为：

$$A(x) = \sum_{n=0}^{\infty} a_n x^n$$

**指数生成函数**（EGF）定义为：

$$A(x) = \sum_{n=0}^{\infty} \frac{a_n}{n!} x^n$$

**例子**：

1. **二项式系数**：$(a_n) = (\binom{n}{k})_{n \geq 0}$
   - 生成函数：$A(x) = (1+x)^k$

2. **斐波那契数列**：$F_0 = 0, F_1 = 1, F_n = F_{n-1} + F_{n-2}$
   - 生成函数：$F(x) = \frac{x}{1-x-x^2}$

### 1.2 函数的意义

**函数的意义**：

生成函数揭示了组合问题的深层结构。

**数学表述**：

生成函数将组合问题转化为函数问题：

- **计数问题**：序列的生成函数
- **组合恒等式**：生成函数的恒等式
- **递推关系**：生成函数的函数方程

## 二、函数的性质

### 2.1 基本性质

**函数的基本性质**：

1. **生成性**：函数具有生成性
   - 生成函数生成序列：$a_n = [x^n] A(x)$
   - 其中 $[x^n]$ 表示提取 $x^n$ 的系数

2. **计数性**：函数具有计数性
   - 生成函数的系数给出计数
   - 例如：$[x^n] (1+x)^k = \binom{k}{n}$

3. **结构性**：函数具有结构性
   - 生成函数的运算对应序列的运算
   - 例如：$A(x) \cdot B(x)$ 对应序列的卷积

**数学表述**：

**卷积性质**：

如果 $A(x) = \sum a_n x^n$，$B(x) = \sum b_n x^n$，则：

$$A(x) \cdot B(x) = \sum_{n=0}^{\infty} \leqft(\sum_{k=0}^n a_k b_{n-k}\right) x^n$$

**递推关系**：

如果序列满足递推关系 $a_n = c_1 a_{n-1} + \cdots + c_k a_{n-k}$，则生成函数满足：

$$A(x) = \frac{P(x)}{1 - c_1 x - \cdots - c_k x^k}$$

其中 $P(x)$ 是初始条件的多项式。

### 2.2 函数的结构

**函数的结构**：

生成函数具有重要的数学结构。

- **环结构**：生成函数构成环
