## 🎯 核心概念

### λ演算

**λ演算**是一种形式系统，用于研究函数定义、函数应用和递归。基本语法包括：

- **变量**：$x, y, z, \ldots$
- **抽象**：$\lambda x.M$（函数定义）
- **应用**：$M N$（函数应用）

### Church编码

**Church编码**将自然数和基本运算编码为λ项：

- **自然数**：$n = \lambda f.\lambda x.f^n(x)$
- **加法**：$+ = \lambda m.\lambda n.\lambda f.\lambda x.m f (n f x)$
- **乘法**：$\times = \lambda m.\lambda n.\lambda f.m (n f)$

### Church-Turing论题

**Church-Turing论题**断言：所有可计算函数都是递归函数（或Turing可计算函数）。
