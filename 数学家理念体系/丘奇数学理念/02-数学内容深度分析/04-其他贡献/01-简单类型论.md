## 一、历史背景与动机

### 1.1 Russell悖论与类型论的起源

20世纪初，集合论成为数学的基础。然而，1901年Bertrand Russell发现的**Russell悖论**动摇了这一基础：

> **Russell悖论**：考虑集合 $R = \{x \mid x \notin x\}$。那么 $R \in R$ 当且仅当 $R \notin R$。矛盾！

Russell的解决方案是**类型论**（1908）：

- 每个对象都有一个**类型**
- 集合只能包含**更低类型**的对象
- $x \in x$ 这样的表达式在类型上是不合法的

### 1.2 Church的贡献（1940）

Alonzo Church在1940年的论文 *"A Formulation of the Simple Theory of Types"* 中为λ演算引入了类型系统，主要动机是：

1. **避免悖论**：类型系统阻止了类似Russell悖论的自指构造
2. **逻辑一致性**：为逻辑系统提供一致的语义基础
3. **表达力控制**：限制λ演算的表达力以获得更好的性质

**与Russell类型论的区别**：

- Russell的类型论是**逻辑主义**的，旨在为数学提供基础
- Church的类型论是**计算性**的，关注函数的类型和计算行为

## 二、简单类型λ演算的语法

### 2.1 类型语法

**定义 2.1**（简单类型）：

给定可数无穷的类型变量集合 $\mathcal{V}_T = \{\alpha, \beta, \gamma, \ldots\}$，简单类型的集合 $\mathbb{T}$ 由以下BNF定义：

$$\sigma, \tau ::= \alpha \mid \iota \mid o \mid \tau \to \sigma$$

其中：

- $\alpha$：**类型变量**（可实例化为具体类型）
- $\iota$：**个体类型**（自然数、布尔值等基本类型）
- $o$：**命题类型**（真值类型，对应于逻辑命题）
- $\tau \to \sigma$：**函数类型**（从 $\tau$ 到 $\sigma$ 的函数）

**约定**：

- 函数类型是**右结合**的：$\tau_1 \to \tau_2 \to \tau_3$ 表示 $\tau_1 \to (\tau_2 \to \tau_3)$
- 这与Currying对应：多参数函数表示为高阶函数

### 2.2 项语法

**定义 2.2**（带类型的λ项）：

$$M, N ::= x \mid \lambda x:\tau.M \mid MN \mid c^\tau$$

其中：

- $x$：**变量**，带有隐式类型
- $\lambda x:\tau.M$：**带类型的抽象**，参数 $x$ 具有类型 $\tau$
- $MN$：**应用**（与无类型λ演算相同）
- $c^\tau$：**常量**，具有固定类型 $\tau$

**语境（Context）**：

类型推导在**语境**（或类型环境）中进行：
$$\Gamma ::= \emptyset \mid \Gamma, x:\tau$$

语境是变量到类型的映射，表示"在假设 $x$ 具有类型 $\tau$ 的条件下"。

## 三、类型推导规则

### 3.1 类型判断

**类型判断**的形式为：
$$\Gamma \vdash M : \tau$$

读作："在语境 $\Gamma$ 下，项 $M$ 具有类型 $\tau$"。

### 3.2 推导规则

**公理（变量规则）**：
$$\frac{}{\Gamma, x:\tau \vdash x : \tau} \text{(VAR)}$$

**规则 3.1**（抽象规则）：
$$\frac{\Gamma, x:\tau \vdash M : \sigma}{\Gamma \vdash \lambda x:\tau.M : \tau \to \sigma} \text{(ABS)}$$

**规则 3.2**（应用规则）：
$$\frac{\Gamma \vdash M : \tau \to \sigma \quad \Gamma \vdash N : \tau}{\Gamma \vdash MN : \sigma} \text{(APP)}$$

### 3.3 常量的类型规则

对于逻辑常量，Church定义了：

| 常量 | 类型 | 含义 |
|------|------|------|
| $\top$ | $o$ | 真 |
| $\bot$ | $o$ | 假 |
$\neg$ | $o \to o$ | 否定 |
$\wedge, \vee$ | $o \to o \to o$ | 合取、析取 |
$\forall_\tau, \exists_\tau$ | $(\tau \to o) \to o$ | 全称/存在量词 |

**量词的直观理解**：

- $\forall_\tau$ 接受一个谓词 $P: \tau \to o$，返回命题
- $\forall_\tau P$ 表示"对所有 $x:\tau$，$Px$ 成立"

## 四、类型系统的基本定理

### 4.1 类型唯一性（Uniqueness of Types）

**定理 4.1**（类型唯一性）：

如果 $\Gamma \vdash M : \tau$ 且 $\Gamma \vdash M : \sigma$，则 $\tau = \sigma$。

**证明思路**（对 $M$ 的结构归纳）：

- **基本情况** $M = x$：由VAR规则，类型由语境唯一确定
- **归纳情况** $M = \lambda x:\tau_1.N$：由ABS规则，$N$ 的类型唯一，因此函数类型唯一
- **归纳情况** $M = NP$：由APP规则，$N$ 的类型唯一确定输出类型

### 4.2 类型保持（Type Preservation / Subject Reduction）

**定理 4.2**（类型保持）：

如果 $\Gamma \vdash M : \tau$ 且 $M \to_\beta N$，则 $\Gamma \vdash N : \tau$。

**证明概要**：

对归约关系 $M \to_\beta N$ 进行归纳分析：

- **基本情况** $(\lambda x:\tau.M)N \to_\beta M[N/x]$：
  - 前提：$\Gamma \vdash \lambda x:\tau.M : \tau \to \sigma$ 且 $\Gamma \vdash N : \tau$
  - 由ABS：$\Gamma, x:\tau \vdash M : \sigma$
  - 替换引理：$\Gamma \vdash M[N/x] : \sigma$

- **归纳情况**（兼容闭包）：保持类型结构

### 4.3 强规范化（Strong Normalization）

**定理 4.3**（强规范化，Girard 1971, Tait 1967）：

所有良类型的项都是**强规范化**的：任何归约序列都终止于唯一的范式。

**推论**：

- 简单类型λ演算不是图灵完备的
- 无类型λ演算中的非终止项（如 $\Omega = (\lambda x.xx)(\lambda x.xx)$）无法在简单类型系统中类型化

**证明方法**：

- Tait的可计算性方法（Reducibility Candidates）
- Girard的候选者方法（Girard's Candidates）

## 五、类型推导示例

### 5.1 恒等函数的推导

**例 5.1**：推导 $\vdash \lambda x:\tau.x : \tau \to \tau$

$$
\frac{\frac{}{x:\tau \vdash x : \tau} \text{VAR}}{\vdash \lambda x:\tau.x : \tau \to \tau} \text{ABS}
$$

### 5.2 组合子K的推导

**例 5.2**：推导 $\vdash \lambda x:\tau.\lambda y:\sigma.x : \tau \to \sigma \to \tau$

$$
\frac{\frac{\frac{}{x:\tau, y:\sigma \vdash x : \tau} \text{VAR}}{x:\tau \vdash \lambda y:\sigma.x : \sigma \to \tau} \text{ABS}}{\vdash \lambda x:\tau.\lambda y:\sigma.x : \tau \to \sigma \to \tau} \text{ABS}
$$

### 5.3 函数复合的推导

**例 5.3**：推导复合组合子 $\circ$ 的类型

$$\circ = \lambda f:\beta \to \gamma.\lambda g:\alpha \to \beta.\lambda x:\alpha.f(gx)$$

推导过程：

$$
\frac{
  \frac{
    \frac{
      \frac{f:\beta \to \gamma, g:\alpha \to \beta, x:\alpha \vdash f : \beta \to \gamma}{\text{VAR}}
      \quad
      \frac{f:\beta \to \gamma, g:\alpha \to \beta, x:\alpha \vdash g : \alpha \to \beta}{\text{VAR}}
      \quad
      \frac{f:\beta \to \gamma, g:\alpha \to \beta, x:\alpha \vdash x : \alpha}{\text{VAR}}
    }{f:\beta \to \gamma, g:\alpha \to \beta, x:\alpha \vdash g x : \beta} \text{APP}
  }{f:\beta \to \gamma, g:\alpha \to \beta, x:\alpha \vdash f(g x) : \gamma} \text{APP}
}{f:\beta \to \gamma, g:\alpha \to \beta \vdash \lambda x:\alpha.f(g x) : \alpha \to \gamma} \text{ABS}
}{f:\beta \to \gamma \vdash \lambda g:\alpha \to \beta.\lambda x:\alpha.f(g x) : (\alpha \to \beta) \to \alpha \to \gamma} \text{ABS}
}{\vdash \circ : (\beta \to \gamma) \to (\alpha \to \beta) \to \alpha \to \gamma} \text{ABS}
$$

**最终类型**：
$$\circ : (\beta \to \gamma) \to (\alpha \to \beta) \to (\alpha \to \gamma)$$

### 5.4 自应用的不可能性

**例 5.4**：证明 $\lambda x.xx$ 无法在简单类型系统中类型化。

**反证**：假设存在 $\tau, \sigma$ 使得：

- $x:\tau \vdash x : \tau \to \sigma$（对应用项 $xx$ 的第一个 $x$）
- $x:\tau \vdash x : \tau$（对第二个 $x$）

由VAR规则，$\tau = \tau \to \sigma$。

但简单类型不允许**递归类型**（$\tau$ 出现在自身的定义中），因此矛盾。

**结论**：自应用 $\lambda x.xx$ 和 $\Omega$ 组合子在简单类型λ演算中无类型。

## 六、Curry-Howard对应

### 6.1 命题与类型的同构

**Curry-Howard对应**（命题即类型，证明即程序）：

| 逻辑 | 类型论 |
|------|--------|
| 命题 $A$ | 类型 $A$ |
| 证明 $p : A$ | 项 $p : A$ |
| $A \Rightarrow B$ | 函数类型 $A \to B$ |
| $A \wedge B$ | 积类型 $A \times B$ |
| $A \vee B$ | 和类型 $A + B$ |
| $\forall x:A.B(x)$ | 依赖类型 $\Pi x:A.B(x)$ |
| $\exists x:A.B(x)$ | 依赖类型 $\Sigma x:A.B(x)$ |

### 6.2 推导规则对应

**蕴含引入** ↔ **函数抽象**：

$$
\frac{\Gamma, A \vdash B}{\Gamma \vdash A \Rightarrow B} \Rightarrow\text{-intro}
\quad \leftrightarrow \quad
\frac{\Gamma, x:A \vdash M:B}{\Gamma \vdash \lambda x.M : A \to B} \text{ABS}
$$

**蕴含消去** ↔ **函数应用**：

$$
\frac{\Gamma \vdash A \Rightarrow B \quad \Gamma \vdash A}{\Gamma \vdash B} \Rightarrow\text{-elim}
\quad \leftrightarrow \quad
\frac{\Gamma \vdash M:A \to B \quad \Gamma \vdash N:A}{\Gamma \vdash MN:B} \text{APP}
$$

### 6.3 简单类型对应直觉主义命题逻辑

**定理 6.1**：

简单类型λ演算对应于**直觉主义命题逻辑**的蕴涵片段（IPC$_\to$）。

- 类型 $  au$ 对应命题 $|\tau|$
- 良类型项 $M : \tau$ 对应于命题 $|\tau|$ 的证明
- β归约对应证明归约（Normalization）

## 七、现代应用：从Haskell到Lean 4

### 7.1 Haskell中的类型系统

Haskell基于**Hindley-Milner类型系统**（简单类型的扩展）：

```haskell
-- 恒等函数（多态）
id :: a -> a
id x = x

-- 组合子K
const :: a -> b -> a
const x y = x

-- 函数复合
(.) :: (b -> c) -> (a -> b) -> (a -> c)
(f . g) x = f (g x)

-- 类型推导：Haskell自动推导类型
-- \x -> x 的类型是 a -> a
-- \x y -> x 的类型是 a -> b -> a
```

**类型类**（Type Classes）是简单类型的进一步扩展，实现ad-hoc多态。

### 7.2 Lean 4中的依赖类型

Lean 4使用**依赖类型**（Martin-Löf类型论），是简单类型的强大扩展：

```lean4
-- 简单函数类型（与Church的记法一致）
def id (α : Type) : α → α :=
  λ x => x

-- 依赖函数类型（Π类型）
def vec_map (α β : Type) (n : Nat) :
  (α → β) → Vec α n → Vec β n :=
  λ f v => ...

-- 命题即类型，证明即程序
theorem idempotent_and (p q : Prop) :
  (p ∧ q) → (p ∧ q) :=
  λ h => h  -- 恒等函数即证明

-- 类型检查确保正确性
#check id Nat    -- Nat → Nat
#check id Bool   -- Bool → Bool
```

### 7.3 类型推导算法

**Hindley-Milner类型推导**（Algorithm W）：

1. **生成约束**：从语法树生成类型等式
2. **合一（Unification）**：求解约束系统（Robinson合一算法）
3. **泛化**：将自由类型变量泛化为多态类型

**示例推导**：

```
λf.λg.λx.f(g x)
- f : β → γ
- g : α → β
- x : α
- gx : β
- f(gx) : γ
- 结果类型：(β → γ) → (α → β) → α → γ
```
