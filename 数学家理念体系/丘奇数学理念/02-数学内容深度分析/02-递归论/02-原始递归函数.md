## 一、引言：原始递归的概念

### 1.1 历史背景

**Gödel的贡献（1931）**：

- 原始递归函数的定义
- 为不完备定理提供基础
- 形式化可计算性

**丘奇的贡献**：

- 连接λ可定义与递归函数
- 证明等价性
- 建立可计算性理论

---

### 1.2 原始递归的意义

**核心思想**：

- 使用归纳定义函数
- 从基础情况开始
- 逐步构造复杂函数

---

## 二、原始递归模式

### 2.1 定义

**原始递归模式**：

给定函数 $g$ 和 $h$，定义函数 $f$：

- $f(0, \bar{x}) = g(\bar{x})$
- $f(n+1, \bar{x}) = h(n, f(n, \bar{x}), \bar{x})$

**直观**：

- 基础情况：$f(0, \bar{x}) = g(\bar{x})$
- 递归情况：$f(n+1, \bar{x})$ 依赖于 $f(n, \bar{x})$

---

### 2.2 例子

**加法**：

- $+(0, y) = y$
- $+(n+1, y) = S(+(n, y))$

**乘法**：

- $\times(0, y) = 0$
- $\times(n+1, y) = +(\times(n, y), y)$

**指数**：

- $\exp(0, y) = 1$
- $\exp(n+1, y) = \times(\exp(n, y), y)$

---

## 三、原始递归函数的性质

### 3.1 全函数性

**性质**：

所有原始递归函数都是全函数（total），即对所有输入都有定义。

**证明**：

- 使用数学归纳法
- 基础情况和递归情况都有定义

---

### 3.2 封闭性

**封闭性**：

原始递归函数类在以下运算下封闭：

- 复合
- 原始递归

**意义**：

- 可以从简单函数构造复杂函数
- 形成函数类

---

## 四、原始递归函数的例子

### 4.1 算术函数

**基本算术运算**：

所有基本算术运算都可以通过原始递归定义。

**前驱函数（Predecessor）**：

前驱函数 $\text{pred}: \mathbb{N} \to \mathbb{N}$ 定义为：
- $\text{pred}(0) = 0$
- $\text{pred}(n+1) = n$

**数学表述**：

前驱函数：
$$\text{pred}(0) = 0, \quad \text{pred}(n+1) = P_1^2(n, \text{pred}(n)) = n$$

**截断减法（Monus）**：

截断减法 $\dot{-}$ 定义为：
- $x \dot{-} 0 = x$
- $x \dot{-} (n+1) = \text{pred}(x \dot{-} n)$

**数学表述**：

截断减法：
$$x \dot{-} 0 = x, \quad x \dot{-} (n+1) = \text{pred}(x \dot{-} n)$$

**例子9：截断减法的计算**：

- $5 \dot{-} 2 = \text{pred}(5 \dot{-} 1) = \text{pred}(\text{pred}(5 \dot{-} 0)) = \text{pred}(\text{pred}(5)) = \text{pred}(4) = 3$ ✓
- $2 \dot{-} 5 = 0$（截断，不出现负数）

**绝对值差**：

绝对值差 $|x - y|$ 可以通过截断减法定义：
$$|x - y| = (x \dot{-} y) + (y \dot{-} x)$$

**截断除法**：

截断除法可以通过原始递归定义，但定义较复杂。

---

### 4.2 逻辑函数

**特征函数（Characteristic Function）**：

集合 $A \subseteq \mathbb{N}$ 的**特征函数** $\chi_A$ 定义为：
$$\chi_A(x) = \begin{cases} 1 & \text{如果 } x \in A \\ 0 & \text{如果 } x \notin A \end{cases}$$

**数学表述**：

特征函数：
$$\chi_A(x) = \begin{cases} 1 & x \in A \\ 0 & x \notin A \end{cases}$$

**布尔运算**：

通过特征函数可以定义布尔运算：
- **逻辑与**：$\text{and}(x, y) = x \cdot y$（如果 $x, y \in \{0, 1\}$）
- **逻辑或**：$\text{or}(x, y) = \min(x + y, 1)$
- **逻辑非**：$\text{not}(x) = 1 \dot{-} x$

**条件函数**：

条件函数 $\text{if-then-else}$ 可以定义为：
$$\text{ite}(c, x, y) = c \cdot x + (1 \dot{-} c) \cdot y$$

**数学表述**：

条件函数：
$$\text{ite}(c, x, y) = \begin{cases} x & \text{如果 } c = 1 \\ y & \text{如果 } c = 0 \end{cases}$$

**例子10：逻辑函数的应用**：

- $\text{and}(1, 1) = 1$，$\text{and}(1, 0) = 0$
- $\text{or}(0, 1) = 1$，$\text{or}(0, 0) = 0$
- $\text{ite}(1, 5, 3) = 5$，$\text{ite}(0, 5, 3) = 3$

## 五、原始递归的局限性

### 5.1 Ackermann函数

**Ackermann函数的定义**：

Ackermann函数 $A: \mathbb{N}^2 \to \mathbb{N}$ 是增长极快的函数，定义为：
- $A(0, n) = n+1$
- $A(m+1, 0) = A(m, 1)$
- $A(m+1, n+1) = A(m, A(m+1, n))$

**数学表述**：

Ackermann函数：
$$A(0, n) = n+1$$
$$A(m+1, 0) = A(m, 1)$$
$$A(m+1, n+1) = A(m, A(m+1, n))$$

**例子11：Ackermann函数的计算**：

- $A(0, 0) = 1$
- $A(0, 1) = 2$
- $A(1, 0) = A(0, 1) = 2$
- $A(1, 1) = A(0, A(1, 0)) = A(0, 2) = 3$
- $A(2, 0) = A(1, 1) = 3$
- $A(2, 1) = A(1, A(2, 0)) = A(1, 3) = A(0, A(1, 2)) = A(0, A(0, A(1, 1))) = A(0, A(0, 3)) = A(0, 4) = 5$

**增长速度**：

Ackermann函数增长极快：
- $A(1, n) = n+2$
- $A(2, n) = 2n+3$
- $A(3, n) = 2^{n+3} - 3$
- $A(4, n)$ 增长更快，$A(4, 2)$ 就已经是一个巨大的数

**定理（Ackermann, 1928）**：

Ackermann函数**不是原始递归的**。

**证明思路**：

可以证明，任何原始递归函数 $f$ 都存在常数 $c$ 使得对足够大的 $n$，$f(n) < A(c, n)$。但 $A(m+1, n)$ 的增长速度超过任何 $A(m, \cdot)$，因此 $A$ 本身不能是原始递归的。

**意义**：

1. **原始递归的局限性**：原始递归不足以表达所有可计算函数
2. **一般递归的必要性**：需要更一般的递归概念（μ递归）
3. **计算理论的发展**：推动了可计算性理论的发展

---

### 5.2 一般递归

**一般递归函数**：

**一般递归函数**（或**μ递归函数**）通过添加**μ算子**（最小化算子）扩展了原始递归函数类。

**μ算子**：

给定函数 $g$，定义函数 $f$：
$$f(\bar{x}) = \mu y[g(\bar{x}, y) = 0]$$

表示最小的 $y$ 使得 $g(\bar{x}, y) = 0$，如果不存在则 $f(\bar{x})$ 未定义。

**数学表述**：

μ算子：
$$f(\bar{x}) = \begin{cases}
\min\{y \mid g(\bar{x}, y) = 0\} & \text{如果存在} \\
\text{未定义} & \text{否则}
\end{cases}$$

**一般递归函数的定义**：

一般递归函数是：
1. 初始函数
2. 通过复合、原始递归和μ算子得到的函数

**性质**：

- **包含部分函数**：μ算子可能产生未定义的函数
- **计算完备性**：等价于图灵可计算函数
- **Church-Turing论题**：支持Church-Turing论题

**数学表述**：

一般递归函数：
$$\text{一般递归函数} = \text{图灵可计算函数}$$

## 九、总结

原始递归函数展示了Gödel和丘奇在计算理论中的核心贡献：

1. **原始递归模式**：提供了函数定义的方法
2. **全函数性**：所有原始递归函数都是全函数
3. **封闭性**：形成函数类
4. **现代影响**：为计算理论和复杂度理论提供基础

这些成果不仅深化了我们对计算的理解，也为现代计算机科学提供了重要的理论基础。
