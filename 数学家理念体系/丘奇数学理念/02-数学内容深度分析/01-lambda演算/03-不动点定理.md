## 一、引言：不动点的意义

### 1.1 不动点的概念

**定义**：

函数 $f$ 的不动点是满足 $f(x) = x$ 的点 $x$。

**在λ演算中**：

- 项 $M$ 是 $F$ 的不动点，如果 $FM = M$
- 不动点组合子可以构造不动点

---

### 1.2 不动点的应用

**应用**：

- 定义递归函数
- 实现循环
- 计算理论

---

## 二、不动点组合子

### 2.1 Y组合子

**Y组合子（Curry, 1942）**：

Y组合子是最著名的不动点组合子，由Haskell Curry发现。

**定义**：

$$Y = \lambda f.(\lambda x.f(xx))(\lambda x.f(xx))$$

**数学表述**：

Y组合子：
$$Y = \lambda f.(\lambda x.f(xx))(\lambda x.f(xx))$$

**性质**：

对任意λ项 $F$，$YF$ 是 $F$ 的不动点：
$$YF = F(YF)$$

**证明**：

$$YF = (\lambda f.(\lambda x.f(xx))(\lambda x.f(xx)))F$$
$$= (\lambda x.F(xx))(\lambda x.F(xx))$$
$$= F((\lambda x.F(xx))(\lambda x.F(xx)))$$
$$= F(YF)$$

**例子1：Y组合子的应用**：

考虑恒等函数 $I = \lambda x.x$：
$$YI = I(YI) = YI$$

因此 $YI$ 是 $I$ 的不动点（实际上任何项都是 $I$ 的不动点）。

**例子2：使用Y组合子定义递归函数**：

定义阶乘函数：
$$\text{fact} = Y(\lambda g.\lambda n.\text{if } (\text{iszero} n) \overline{1} (\text{mult} n (g (\text{pred} n))))$$

**Y组合子的结构**：

Y组合子使用了**自应用** $xx$，这在无类型λ演算中是合法的，但在类型化λ演算中会导致类型问题。

---

### 2.2 其他不动点组合子

**Turing组合子（Turing, 1937）**：

$$\Theta = (\lambda xy.y(xxy))(\lambda xy.y(xxy))$$

**数学表述**：

Turing组合子：
$$\Theta = AA \text{ 其中 } A = \lambda xy.y(xxy)$$

**性质**：

Turing组合子也是不动点组合子：
$$\Theta F = F(\Theta F)$$

**证明**：

$$\Theta F = AA F = (\lambda xy.y(xxy))A F = F(AA) = F(\Theta F)$$

**例子3：Turing组合子的应用**：

$$\Theta F = F(\Theta F) = F(F(\Theta F)) = \ldots$$

**其他不动点组合子**：

1. **Z组合子**：用于严格求值语言
   $$Z = \lambda f.(\lambda x.f(\lambda v.xxv))(\lambda x.f(\lambda v.xxv))$$

2. **Böhm组合子**：使用不同的构造方法

**不动点组合子的存在性**：

不动点定理保证了不动点组合子的存在，但构造方法不唯一。

**数学表述**：

存在性定理：
$$\forall F \exists M: FM = M$$

构造性证明：$M = YF$ 或 $M = \Theta F$。

## 三、不动点定理

### 3.1 定理陈述

**不动点定理（Fixed Point Theorem）**：

这是λ演算中的基本定理，由Church和Turing独立发现。

**定理陈述**：

对任意λ项 $F$，存在λ项 $M$ 使得：
$$FM =_\beta M$$

**数学表述**：

不动点存在性：
$$\forall F \in \Lambda \exists M \in \Lambda: FM =_\beta M$$

**构造性证明**：

给定 $F$，可以构造不动点：
$$M = YF = (\lambda f.(\lambda x.f(xx))(\lambda x.f(xx)))F$$

**验证**：

$$YF = F(YF)$$

因此 $YF$ 确实是 $F$ 的不动点。

**例子4：不动点的构造**：

- $F = \lambda x.x$：任何项都是不动点
- $F = \lambda x.y$：不动点是 $y$
- $F = \lambda f.\lambda n.n$：不动点是 $\lambda n.n$

**不动点定理的意义**：

1. **递归定义**：允许在λ演算中定义递归函数
2. **计算完备性**：证明了λ演算的表达力
3. **理论意义**：连接了递归理论与λ演算

---

### 3.2 不动点的唯一性

**唯一性问题**：

不动点通常**不唯一**。例如，对恒等函数 $I = \lambda x.x$，任何项都是不动点。

**数学表述**：

唯一性：
$$\text{一般地，} |\{M \mid FM =_\beta M\}| \geq 1$$

**例子5：多个不动点**：

考虑 $F = \lambda x.y$：
- $y$ 是不动点：$Fy = y$
- 实际上，任何项 $M$ 如果 $M =_\beta y$，则都是不动点

**最小不动点**：

在某些情况下，可以定义"最小"不动点，这通常对应于递归函数的"最小定义域"。

**数学表述**：

最小不动点：
$$\text{fix}(F) = \bigcap \{M \mid FM =_\beta M\}$$

**不动点的选择**：

在实际应用中，通常选择：
- **最小不动点**：对应递归函数的最小定义域
- **最大不动点**：对应递归函数的最大定义域
- **Y组合子给出的不动点**：通常是最小不动点

## 四、递归函数的定义

### 4.1 使用不动点

**递归定义的挑战**：

在λ演算中，函数没有名字，因此无法直接定义递归函数。不动点组合子解决了这个问题。

**方法**：

要定义递归函数 $f$，首先将其写成非递归形式 $F$，其中 $F$ 接受函数 $g$ 作为参数，返回函数体。然后使用不动点组合子：
$$f = YF$$

**数学表述**：

递归函数定义：
$$f = Y(\lambda g.\lambda x.\text{函数体使用 } g)$$

**例子6：阶乘函数**：

定义：
$$\text{fact} = Y(\lambda g.\lambda n.\text{if } (\text{iszero} n) \overline{1} (\text{mult} n (g (\text{pred} n))))$$

**验证**：

$$\text{fact} = YF = F(YF) = F(\text{fact})$$

因此：
$$\text{fact} n = \text{if } (\text{iszero} n) \overline{1} (\text{mult} n (\text{fact} (\text{pred} n)))$$

这正是阶乘的递归定义。

**例子7：斐波那契函数**：

定义：
$$\text{fib} = Y(\lambda g.\lambda n.\text{if } (n \leq 1) n (\text{plus} (g (n-1)) (g (n-2))))$$

**例子8：列表长度函数**：

定义：
$$\text{length} = Y(\lambda g.\lambda l.\text{if } (\text{isnil} l) \overline{0} (\text{succ} (g (\text{tail} l))))$$

---

### 4.2 递归与循环

**递归与循环的等价性**：

在计算理论中，递归和循环是等价的。不动点提供了统一的框架。

**数学表述**：

递归函数：
$$f(x) = \begin{cases}
\text{base case} & \text{if condition} \\
\text{recursive case using } f & \text{otherwise}
\end{cases}$$

对应不动点：
$$f = \text{fix}(F) \text{ 其中 } F(g)(x) = \begin{cases}
\text{base case} & \text{if condition} \\
\text{recursive case using } g & \text{otherwise}
\end{cases}$$

**循环的λ演算表示**：

循环可以通过递归表示：
$$\text{while } C \text{ do } B = Y(\lambda w.\lambda s.\text{if } C(s) \text{ then } w(B(s)) \text{ else } s)$$

**例子9：循环的递归表示**：

计算 $n!$ 的循环：
$$\text{fact\_loop} = Y(\lambda w.\lambda (n, acc).\text{if } (n = 0) acc \text{ else } w(n-1, n \cdot acc))$$

## 五、不动点的计算

### 5.1 不动点的求值

**方法**：

- 使用归约
- $YF \to F(YF) \to F(F(YF)) \to \ldots$
- 可能不终止

---

### 5.2 类型化λ演算

**问题**：

- 无类型λ演算中，$Y$ 不能有类型
- 需要特殊处理

**解决**：

- 使用递归类型
- 或者使用特殊的递归构造

## 八、总结

不动点定理展示了丘奇在计算理论中的核心贡献：

1. **不动点组合子**：可以构造任意函数的不动点
2. **递归定义**：为递归函数提供基础
3. **计算模型**：展示了λ演算的表达力

这些成果不仅深化了我们对计算的理解，也为现代计算机科学提供了重要的理论基础。
