## 一、历史背景：从Russell到Curry

### 1.1 自指的悖论与突破

20世纪初，**自指**（self-reference）导致了逻辑危机。Russell悖论就是集合论中的自指问题：

> $R = \{x \mid x \notin x\}$，那么 $R \in R$ 当且仅当 $R \notin R$。

然而，在计算理论中，自指成为了强大的工具。1930年代，Church、Turing和Curry发现：

- **自应用**（$xx$）是λ演算的核心
- **不动点组合子**可以构造任意函数的不动点

### 1.2 Curry的发现（1942）

Haskell Curry在1942年发现了著名的**Y组合子**，这是λ演算中最优雅的构造之一。

**关键洞察**：通过**自应用**，可以构造递归定义的解。

### 1.3 与图灵的等价性

Turing独立发现了类似的结果（**Turing组合子**，1937年）。两人证明了：

- λ演算可以表达所有递归函数
- 不动点提供了递归的统一框架

## 二、不动点的数学概念

### 2.1 不动点的定义

**定义 2.1**（不动点）：

对于函数 $f: X \to X$，元素 $x \in X$ 是 $f$ 的**不动点**，如果：
$$f(x) = x$$

**在λ演算中**：

- 函数是λ项
- 项 $M$ 是 $F$ 的不动点，如果 $FM =_\beta M$

### 2.2 不动点的例子

**例 2.2**（数学中的不动点）：

| 函数 | 不动点 | 说明 |
|------|--------|------|
| $f(x) = x$ | 所有 $x$ | 恒等函数 |
| $f(x) = x^2$ | 0, 1 | 幂函数 |
| $f(x) = \cos(x)$ | ≈0.739 | 余弦函数 |

**例 2.3**（λ演算中的不动点）：

- 对 $I = \lambda x.x$，**任何项** $M$ 都是不动点：$IM = M$
- 对 $K_y = \lambda x.y$，**$y$** 是不动点：$K_y y = y$

### 2.3 不动点与递归

**关键观察**：

递归函数的定义形如：
$$f = F(f)$$

其中 $F$ 是一个高阶函数（函数变换器）。

这恰好是不动点的定义！因此：
$$f \text{ 是递归方程的解} \Leftrightarrow f \text{ 是 } F \text{ 的不动点}$$

## 三、Y组合子的构造与推导

### 3.1 问题：如何在λ演算中定义递归？

在λ演算中，函数是**匿名**的，没有名字可以自我引用。如何定义递归函数如阶乘？

**传统定义**：
$$fact(n) = \begin{cases} 1 & n = 0 \\ n \times fact(n-1) & n > 0 \end{cases}$$

**目标**：在λ演算中表示 $fact$，而不直接使用 $fact$ 本身。

### 3.2 高阶函数的思路

**关键思想**：将递归函数写成**非递归的高阶函数**，然后寻找不动点。

定义：
$$F = \lambda g.\lambda n.\text{if } (n = 0) \text{ then } 1 \text{ else } n \times g(n-1)$$

如果我们能找到 $f$ 使得 $Ff = f$，则 $f$ 就是阶乘函数！

### 3.3 自应用的魔法

**构造策略**：

我们需要构造一个项 $Y$，使得对任意 $F$：
$$YF = F(YF)$$

**推导过程**：

设 $Y$ 形如 $\lambda f.\ldots$，我们希望：
$$YF = F(YF)$$

考虑自应用。令 $W = \lambda x.f(xx)$，则：
$$WW = (\lambda x.f(xx))(\lambda x.f(xx)) = f((\lambda x.f(xx))(\lambda x.f(xx))) = f(WW)$$

因此 $WW$ 是 $f$ 的不动点！

### 3.4 Y组合子的定义

**定义 3.1**（Y组合子）：

$$Y = \lambda f.(\lambda x.f(xx))(\lambda x.f(xx))$$

### 3.5 Y组合子的性质证明

**定理 3.2**：对任意λ项 $F$，$YF$ 是 $F$ 的不动点：
$$YF =_\beta F(YF)$$

**证明**：

$$
\begin{aligned}
YF &= (\lambda f.(\lambda x.f(xx))(\lambda x.f(xx)))F \\
&=_\beta (\lambda x.F(xx))(\lambda x.F(xx)) \\
&=_\beta F((\lambda x.F(xx))(\lambda x.F(xx))) \\
&=_\beta F(YF) \quad \blacksquare
\end{aligned}
$$

### 3.6 推导步骤详解

让我们更详细地展示推导：

**步骤 1**：应用 $Y$ 到 $F$
$$YF = (\lambda f.(\lambda x.f(xx))(\lambda x.f(xx)))F$$

**步骤 2**：β归约（将 $f$ 替换为 $F$）
$$\to_\beta (\lambda x.F(xx))(\lambda x.F(xx))$$

**步骤 3**：再次β归约（将 $x$ 替换为 $(\lambda x.F(xx))$）
$$\to_\beta F((\lambda x.F(xx))(\lambda x.F(xx)))$$

**步骤 4**：观察到 $(\lambda x.F(xx))(\lambda x.F(xx)) = YF$
$$=_\beta F(YF)$$

## 四、不动点定理

### 4.1 不动点存在性定理

**定理 4.1**（不动点定理）：

对于任意λ项 $F$，存在λ项 $M$ 使得：
$$FM =_\beta M$$

**构造性证明**：取 $M = YF$。

### 4.2 不动点的非唯一性

**注意**：不动点通常**不唯一**。

**例 4.2**：对 $I = \lambda x.x$，任何项 $M$ 都是不动点。

**最小不动点**：在指称语义中，可以定义"最小不动点"，它对应于递归函数的最小定义域。

### 4.3 不动点与计算

**观察**：$YF$ 的展开过程：
$$YF = F(YF) = F(F(YF)) = F(F(F(YF))) = \ldots$$

这恰好对应于递归调用的展开！

## 五、递归函数的λ定义

### 5.1 阶乘函数的λ定义

**定义 5.1**（阶乘）：

$$
fact = Y(\lambda g.\lambda n.\text{if } (n = 0) \overline{1} (\text{mult } n (g (\text{pred } n))))
$$

**验证**：

$$
\begin{aligned}
fact \, \overline{n} &= YF \overline{n} \\
&= F(YF) \overline{n} \\
&= (\lambda g.\lambda n.\ldots)(YF) \overline{n} \\
&= \text{if } (n = 0) \overline{1} (\text{mult } \overline{n} (fact \, (\overline{n-1})))
\end{aligned}
$$

**计算示例**（$fact \, \overline{2}$）：

$$
\begin{aligned}
fact \, \overline{2} &= F(fact) \overline{2} \\
&= \text{mult } \overline{2} (fact \, \overline{1}) \\
&= \text{mult } \overline{2} (\text{mult } \overline{1} (fact \, \overline{0})) \\
&= \text{mult } \overline{2} (\text{mult } \overline{1} \overline{1}) \\
&= \text{mult } \overline{2} \overline{1} \\
&= \overline{2}
\end{aligned}
$$

### 5.2 斐波那契函数

**定义 5.2**（斐波那契）：

$$
fib = Y(\lambda g.\lambda n.\text{if } (n \leq 1) n (\text{plus } (g (n-1)) (g (n-2))))
$$

### 5.3 列表长度函数

**定义 5.3**（列表长度）：

$$
length = Y(\lambda g.\lambda l.\text{if } (\text{isnil } l) \overline{0} (\text{succ } (g (\text{tail } l))))
$$

### 5.4 不动点与循环的等价

**定理 5.4**：

在λ演算中，递归和循环是等价的。

**循环的表示**：
$$\text{while } C \text{ do } B = Y(\lambda w.\lambda s.\text{if } C(s) \text{ then } w(B(s)) \text{ else } s)$$

## 六、其他不动点组合子

### 6.1 Turing组合子

**定义 6.1**（Turing组合子 $\Theta$，1937）：

$$
\Theta = AA \text{ 其中 } A = \lambda xy.y(xxy)
$$

即：
$$\Theta = (\lambda xy.y(xxy))(\lambda xy.y(xxy))$$

**性质**：
$$\Theta F = F(\Theta F)$$

**证明**：

$$
\begin{aligned}
\Theta F &= AA F \\
&= (\lambda xy.y(xxy)) A F \\
&= F(AA) \\
&= F(\Theta F) \quad \blacksquare
\end{aligned}
$$

### 6.2 Z组合子（严格求值）

在**严格求值**（先求参数值）的语言中，Y组合子会导致无限展开。

**定义 6.2**（Z组合子）：

$$Z = \lambda f.(\lambda x.f(\lambda v.xxv))(\lambda x.f(\lambda v.xxv))$$

**性质**：
$$ZF = F(\lambda v.ZFv)$$

这延迟了递归调用，直到参数被需要。

### 6.3 不同组合子的比较

| 组合子 | 定义 | 适用场景 |
|--------|------|----------|
| $Y$ | $\lambda f.(\lambda x.f(xx))(\lambda x.f(xx))$ | 惰性求值 |
| $\Theta$ | $(\lambda xy.y(xxy))(\lambda xy.y(xxy))$ | 惰性求值 |
| $Z$ | $\lambda f.(\lambda x.f(\lambda v.xxv))(\lambda x.f(\lambda v.xxv))$ | 严格求值 |

## 七、不动点与类型论

### 7.1 类型化λ演算中的问题

在**简单类型λ演算**中：

**定理 7.1**：$Y$ 组合子无法被类型化。

**证明思路**：

- $Y$ 包含 $xx$（自应用）
- 在简单类型系统中，$x$ 不能同时是 $A$ 和 $A \to B$

### 7.2 递归类型

解决方案：**递归类型**。

**定义 7.2**（递归类型）：

类型 $\mu \alpha.\tau$ 满足：
$$\mu \alpha.\tau = \tau[\mu \alpha.\tau / \alpha]$$

**例 7.3**：自然数类型可以定义为：
$$Nat = \mu \alpha.(Unit + \alpha)$$

### 7.3 类型化不动点组合子

在带递归类型的系统中：

$$fix : (\tau \to \tau) \to \tau$$

这与 $Y$ 的类型一致（如果 $Y$ 可以有类型）。

## 八、现代应用：函数式编程中的实现

### 8.1 Haskell中的不动点

Haskell提供内置的 `fix` 函数：

```haskell
-- Y组合子的Haskell实现
fix :: (a -> a) -> a
fix f = let x = f x in x

-- 等价于：
fix f = f (fix f)

-- 阶乘
fact :: Integer -> Integer
fact = fix (\f n -> if n == 0 then 1 else n * f (n-1))

-- 斐波那契
fib :: Integer -> Integer
fib = fix (\f n -> if n <= 1 then n else f (n-1) + f (n-2))

-- 无限列表
ones :: [Integer]
ones = fix (1:)

-- 自然数列表
nats :: [Integer]
nats = fix (\xs -> 0 : map (+1) xs)
```

### 8.2 Scheme/Racket中的实现

```scheme
;; Y组合子
(define Y
  (lambda (f)
    ((lambda (x) (f (lambda (v) ((x x) v))))
     (lambda (x) (f (lambda (v) ((x x) v)))))))

;; 阶乘
(define fact
  (Y (lambda (g)
       (lambda (n)
         (if (zero? n)
             1
             (* n (g (- n 1))))))))

(fact 5)  ;; 120
```

### 8.3 Lean 4中的不动点

```lean4
-- Lean 4中的不动点（部分递归）
partial def fix {α : Type} (f : α → α) : α :=
  f (fix f)

-- 阶乘（使用不动点）
partial def fact (n : Nat) : Nat :=
  fix (λ g m => if m = 0 then 1 else m * g (m - 1)) n

-- 使用结构递归（推荐方式）
def fact' : Nat → Nat
  | 0 => 1
  | n+1 => (n+1) * fact' n

-- 结构递归确保终止性
#eval fact' 5  -- 120
```

### 8.4 不动点与数据类型

**递归数据类型的不动点视角**：

```haskell
-- 列表作为不动点
data ListF a r = Nil | Cons a r

type List a = Fix (ListF a)

-- 自然数作为不动点
data NatF r = Zero | Succ r
type Nat = Fix NatF
```

这展示了不动点在**范畴论**和**代数数据类型**中的普适性。
