## 一、引言：λ演算的创立

### 1.1 历史背景

**丘奇的贡献（1930s）**：

- 1932年：λ演算的提出
- 1936年：不可判定性结果
- 1940年：简单类型论

**λ演算的意义**：

- 函数式编程的基础
- 计算理论的基础
- 类型论的基础

---

### 1.2 核心思想

**基本概念**：

- **函数抽象**：$\lambda x.M$ 表示函数
- **函数应用**：$MN$ 表示应用函数 $M$ 到参数 $N$
- **变量绑定**：$\lambda$ 绑定变量

---

## 二、λ演算语法

### 2.1 项的定义

**项（Term）的递归定义**：

λ演算的语法极其简洁，所有λ项都可以通过以下BNF语法定义：

$$M, N ::= x \mid \lambda x.M \mid MN$$

其中：

- **$x$** 是变量（variable），来自可数无穷的变量集合 $\mathcal{V} = \{x, y, z, x_1, x_2, \ldots\}$
- **$\lambda x.M$** 是抽象（abstraction），表示函数，$x$ 是形式参数，$M$ 是函数体
- **$MN$** 是应用（application），表示将函数 $M$ 应用到参数 $N$

**数学表述**：

λ项的集合 $\Lambda$ 是满足上述语法的最小集合，可以通过归纳定义：

- **基础**：所有变量 $x \in \mathcal{V}$ 都是λ项
- **归纳**：如果 $M, N$ 是λ项，$x$ 是变量，则 $\lambda x.M$ 和 $MN$ 也是λ项
- **闭包**：只有通过上述规则得到的才是λ项

**语法约定**：

1. **应用结合性**：应用是左结合的，$MNP$ 表示 $(MN)P$
2. **抽象结合性**：抽象是右结合的，$\lambda x.\lambda y.M$ 表示 $\lambda x.(\lambda y.M)$
3. **作用域**：抽象的作用域尽可能向右延伸，$\lambda x.MN$ 表示 $\lambda x.(MN)$

**例子1：合法的λ项**：

- $x$：单个变量
- $\lambda x.x$：恒等函数
- $(\lambda x.x) y$：应用恒等函数到 $y$
- $\lambda x.\lambda y.xy$：柯里化的二元函数
- $(\lambda x.xx)(\lambda x.xx)$：自应用（Ω组合子）

**例子2：语法解析**：

- $\lambda x.xy$ 解析为 $\lambda x.(xy)$，不是 $(\lambda x.x)y$
- $xyz$ 解析为 $(xy)z$，不是 $x(yz)$
- $\lambda x.\lambda y.xy$ 解析为 $\lambda x.(\lambda y.(xy))$

---

### 2.2 语法性质

**语法的简洁性**：

λ演算的语法只有三个构造，但具有强大的表达力。这种简洁性使得理论分析变得可能。

**表达力**：

尽管语法简单，λ演算可以表示：

- 所有可计算函数（Church-Turing论题）
- 数据结构（通过Church编码）
- 控制结构（通过组合子）

**数学表述**：

λ演算的表达力：
$$\text{可计算函数} = \text{λ可定义函数}$$

**语法树表示**：

每个λ项都可以表示为语法树：

- 变量：叶子节点
- 抽象：单子节点，子节点是函数体
- 应用：二元节点，左右子节点分别是函数和参数

**例子3：语法树**：

项 $(\lambda x.xy) z$ 的语法树：

```
      @
     / \
    λ   z
   / \
  x   @
     / \
    x   y
```

**语法性质**：

1. **简洁性**：只有三个基本构造
2. **表达力**：可以表示所有可计算函数
3. **递归性**：项的定义是递归的
4. **结构性**：每个项都有唯一的语法树表示

## 三、自由变量与绑定变量

### 3.1 自由变量

**自由变量（FV）**：

- $\text{FV}(x) = \{x\}$
- $\text{FV}(\lambda x.M) = \text{FV}(M) \setminus \{x\}$
- $\text{FV}(MN) = \text{FV}(M) \cup \text{FV}(N)$

---

### 3.2 绑定变量

**绑定变量**：

- 在 $\lambda x.M$ 中，$x$ 在 $M$ 中被绑定
- 绑定变量的重命名不影响语义

---

## 四、替换

### 4.1 替换的定义

**替换操作**：

替换 $M[N/x]$ 表示将项 $M$ 中所有**自由出现**的变量 $x$ 替换为项 $N$。

**替换的递归定义**：

$$x[N/x] = N$$
$$y[N/x] = y \quad \text{（如果 } y \neq x\text{）}$$
$$
(\lambda y.M)[N/x] = \begin{cases}
\lambda y.M & \text{如果 } y = x \\
\lambda y.(M[N/x]) & \text{如果 } y \neq x \text{ 且 } y \notin \text{FV}(N) \\
\lambda z.((M[z/y])[N/x]) & \text{如果 } y \neq x \text{ 且 } y \in \text{FV}(N), z \text{ 新变量}
\end{cases}
$$
$$(PQ)[N/x] = (P[N/x])(Q[N/x])$$

**数学表述**：

替换操作：
$$M[N/x] = \text{将 } M \text{ 中所有自由出现的 } x \text{ 替换为 } N$$

**变量捕获问题**：

在替换 $(\lambda y.xy)[y/x]$ 时，如果直接替换会得到 $\lambda y.yy$，但 $y$ 被错误地绑定了。正确的做法是先进行α转换。

**例子4：替换的例子**：

- $x[y/x] = y$
- $z[y/x] = z$（$z \neq x$）
- $(\lambda x.x)[y/x] = \lambda x.x$（$x$ 被绑定，不替换）
- $(\lambda y.xy)[z/x] = \lambda y.zy$（正确替换）
- $(\lambda y.xy)[y/x] = \lambda z.yz$（需要α转换避免捕获）

---

### 4.2 α转换

**α等价（α-equivalence）**：

两个项是α等价的，如果它们只在绑定变量的名称上不同。

**α转换规则**：

$$\lambda x.M =_\alpha \lambda y.M[y/x]$$

其中 $y$ 不在 $M$ 中自由出现（或通过重命名避免冲突）。

**数学表述**：

α等价关系：

- **反射性**：$M =_\alpha M$
- **对称性**：如果 $M =_\alpha N$，则 $N =_\alpha M$
- **传递性**：如果 $M =_\alpha N$ 且 $N =_\alpha P$，则 $M =_\alpha P$

**例子5：α转换的例子**：

- $\lambda x.x =_\alpha \lambda y.y$（重命名绑定变量）
- $\lambda x.\lambda y.xy =_\alpha \lambda u.\lambda v.uv$（重命名多个变量）
- $(\lambda x.x) y =_\alpha (\lambda z.z) y$（抽象部分的α转换）

**α转换的意义**：

绑定变量的名称不影响语义，因此α等价项在语义上相同。这允许我们在需要时重命名变量以避免冲突。

**Barendregt约定**：

为了避免变量冲突，通常采用**Barendregt约定**：在同一个项中，不同的绑定使用不同的变量名。

## 五、语义

### 5.1 操作语义

**归约关系**：

- β归约：$(\lambda x.M)N \to_\beta M[N/x]$
- η归约：$\lambda x.Mx \to_\eta M$（$x$ 不在 $M$ 中自由）

---

### 5.2 范式

**β范式**：

一个项是β范式，如果它不包含任何β可约式（redex）。

**性质**：

- 不是所有项都有范式
- 如果项有范式，则通过归约可以到达

---

## 六、Church编码

### 6.1 自然数编码

**自然数**：

- $0 = \lambda f.\lambda x.x$
- $1 = \lambda f.\lambda x.fx$
- $2 = \lambda f.\lambda x.f(fx)$
- $n = \lambda f.\lambda x.f^n x$

---

### 6.2 运算编码

**后继函数**：

$$\text{succ} = \lambda n.\lambda f.\lambda x.f(nfx)$$

**加法**：

$$\text{add} = \lambda m.\lambda n.\lambda f.\lambda x.mf(nfx)$$

## 十、数学公式总结

### 核心公式

1. **λ演算语法**：
   $$M, N ::= x \mid \lambda x.M \mid MN$$

2. **自由变量**：
   $$\text{FV}(x) = \{x\}$$
   $$\text{FV}(\lambda x.M) = \text{FV}(M) \setminus \{x\}$$
   $$\text{FV}(MN) = \text{FV}(M) \cup \text{FV}(N)$$

3. **替换操作**：
   $$x[N/x] = N$$
   $$y[N/x] = y \quad \text{（}y \neq x\text{）}$$
   $$(\lambda y.M)[N/x] = \lambda y.(M[N/x]) \quad \text{（避免捕获）}$$

4. **α转换**：
   $$\lambda x.M =_\alpha \lambda y.M[y/x]$$

5. **β归约**：
   $$(\lambda x.M)N \to_\beta M[N/x]$$

6. **η归约**：
   $$\lambda x.Mx \to_\eta M \quad \text{（}x \notin \text{FV}(M)\text{）}$$

7. **Church数**：
   $$\overline{n} = \lambda f.\lambda x.f^n x$$

8. **后继函数**：
   $$\text{succ} = \lambda n.\lambda f.\lambda x.f(nfx)$$

9. **加法**：
   $$\text{add} = \lambda m.\lambda n.\lambda f.\lambda x.mf(nfx)$$
