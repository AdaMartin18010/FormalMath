## 一、数论的定义

### 1.1 数论的概念

**数论的定义**：

数论是研究整数性质的学科。

**数学表述**：

数论研究：
- **整数**：$\mathbb{Z} = \{\ldots, -2, -1, 0, 1, 2, \ldots\}$
- **自然数**：$\mathbb{N} = \{1, 2, 3, \ldots\}$
- **素数**：只能被1和自身整除的正整数
- **整除性**：$a \mid b$ 表示 $a$ 整除 $b$

**欧几里得的贡献**：

- 《几何原本》第7-9卷研究数论
- 欧几里得算法：求最大公约数
- 素数无穷性：证明素数有无穷多个

### 1.2 数论的例子

**数论的典型例子**：

1. **素数**：
   - $2, 3, 5, 7, 11, 13, 17, 19, 23, \ldots$
   - 素数定理：$\pi(x) \sim \frac{x}{\ln x}$

2. **整除性**：
   - $3 \mid 6$（3整除6）
   - $4 \nmid 7$（4不整除7）

3. **最大公约数**：
   - $\gcd(12, 18) = 6$
   - 欧几里得算法：$\gcd(a, b) = \gcd(b, a \bmod b)$

## 二、数论的性质

### 2.1 基本性质

**数论的基本性质**：

1. **整数性**：数论研究整数
   - 所有研究对象都是整数

2. **整除性**：数论研究整除关系
   - $a \mid b$ 表示存在 $c$ 使得 $b = ac$

3. **结构性**：数论具有结构
   - 模运算：$a \equiv b \pmod{m}$
   - 同余类：$[a]_m = \{b: b \equiv a \pmod{m}\}$

**数学表述**：

**整除性质**：
- $a \mid b, b \mid c \implies a \mid c$
- $a \mid b, a \mid c \implies a \mid (bx + cy)$

**同余性质**：
- $a \equiv b \pmod{m}, c \equiv d \pmod{m} \implies a+c \equiv b+d \pmod{m}$
- $a \equiv b \pmod{m}, c \equiv d \pmod{m} \implies ac \equiv bd \pmod{m}$

### 2.2 数论的结构

**数论的结构**：

数论的结构由整数的性质决定。

**数学表述**：

1. **整数环**：$(\mathbb{Z}, +, \cdot)$ 是环
2. **模运算**：$\mathbb{Z}/m\mathbb{Z}$ 是模 $m$ 的剩余类环
3. **素数分解**：每个正整数可以唯一分解为素数的乘积
