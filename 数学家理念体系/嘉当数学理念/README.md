## 📖 项目概述

本项目致力于全面梳理和呈现**埃利·嘉当**（Élie Cartan, 1869-1951）的数学理念体系。嘉当是20世纪最重要的数学家之一，他在李群、李代数、微分几何等领域做出了重大贡献，对现代数学和物理学产生了深远影响。

### 1. 李群理论

### 2. 李代数理论

- 李代数的结构
- 李代数的分类

- 嘉当联络

- 几何结构

### 4. 数学哲学

- 数学的结构性
- 数学的统一性
- 数学的几何直觉
