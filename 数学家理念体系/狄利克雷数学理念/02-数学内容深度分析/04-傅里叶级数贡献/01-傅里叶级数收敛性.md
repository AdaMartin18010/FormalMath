## 一、傅里叶级数的定义

### 1.1 周期函数的傅里叶级数

**周期函数**：

设 $f(x)$ 是周期为 $2\pi$ 的函数，即 $f(x+2\pi) = f(x)$。

**傅里叶级数**：

$f(x)$ 的傅里叶级数为：

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^\infty (a_n\cos nx + b_n\sin nx)$$

其中 $\sim$ 表示"对应"，不一定相等。

### 1.2 傅里叶系数

**傅里叶系数**：

$$a_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\cos nx dx \quad (n = 0, 1, 2, \ldots)$$

$$b_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\sin nx dx \quad (n = 1, 2, 3, \ldots)$$

**复形式**：

使用欧拉公式，傅里叶级数可以写成复形式：

$$f(x) \sim \sum_{n=-\infty}^\infty c_n e^{inx}$$

其中：

$$c_n = \frac{1}{2\pi}\int_{-\pi}^{\pi} f(x)e^{-inx} dx$$

### 1.3 傅里叶级数的形式

**三角形式**：

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^\infty (a_n\cos nx + b_n\sin nx)$$

**复形式**：

$$f(x) \sim \sum_{n=-\infty}^\infty c_n e^{inx}$$

**关系**：

$$
c_n = \begin{cases}
\frac{a_0}{2} & \text{如果 } n = 0 \\
\frac{a_n - ib_n}{2} & \text{如果 } n > 0 \\
\frac{a_{-n} + ib_{-n}}{2} & \text{如果 } n < 0
\end{cases}
$$

## 二、收敛性问题

### 2.1 收敛性问题的提出

**问题**：

傅里叶级数是否收敛到 $f(x)$？这是一个重要问题。

**历史背景**：

- 傅里叶提出了傅里叶级数
- 但收敛性问题没有解决
- 需要严格的理论

**困难**：

- 不是所有函数的傅里叶级数都收敛
- 即使收敛，也不一定收敛到 $f(x)$
- 需要找到收敛的条件

### 2.2 收敛性的困难

**不收敛的例子**：

存在函数的傅里叶级数不收敛。

**收敛但不收敛到函数的例子**：

存在函数的傅里叶级数收敛，但不收敛到函数本身。

**需要条件**：

需要找到函数满足什么条件时，傅里叶级数收敛到函数。

### 2.3 狄利克雷的贡献

**1829年的工作**：

狄利克雷在1829年发表了关于傅里叶级数收敛性的重要工作。

**贡献**：

- 找到了傅里叶级数收敛的充分条件
- 建立了收敛性理论
- 推动了傅里叶级数理论的发展

## 三、狄利克雷条件

### 3.1 条件的表述

**狄利克雷条件**：

如果函数 $f(x)$ 满足以下条件，则其傅里叶级数在 $f(x)$ 的连续点收敛到 $f(x)$，在间断点收敛到左右极限的平均值：

1. **有界性**：$f(x)$ 在 $[-\pi, \pi]$ 上有界
2. **分段单调**：$f(x)$ 在 $[-\pi, \pi]$ 上只有有限个极值点
3. **分段连续**：$f(x)$ 在 $[-\pi, \pi]$ 上只有有限个间断点，且每个间断点都是第一类间断点

**第一类间断点**：

在间断点 $x_0$ 处，左右极限 $f(x_0^-)$ 和 $f(x_0^+)$ 都存在。

### 3.2 条件的意义

**充分条件**：

狄利克雷条件是傅里叶级数收敛的充分条件，不是必要条件。

**实用性**：

大多数实际遇到的函数都满足狄利克雷条件。

**理论意义**：

- 建立了傅里叶级数收敛性理论
- 推动了傅里叶级数理论的发展
- 影响了分析学的发展

### 3.3 条件的应用

**应用范围**：

- 应用到大多数实际函数
- 应用到物理问题
- 应用到工程问题

**应用例子**：

- 方波函数
- 锯齿波函数
- 其他周期函数

## 四、收敛性定理

### 4.1 狄利克雷定理

**定理内容**：

如果函数 $f(x)$ 满足狄利克雷条件，则：

1. **在连续点**：傅里叶级数收敛到 $f(x)$
   $$f(x) = \frac{a_0}{2} + \sum_{n=1}^\infty (a_n\cos nx + b_n\sin nx)$$

2. **在间断点**：傅里叶级数收敛到左右极限的平均值
   $$\frac{f(x_0^-) + f(x_0^+)}{2} = \frac{a_0}{2} + \sum_{n=1}^\infty (a_n\cos nx_0 + b_n\sin nx_0)$$

### 4.2 定理的证明

**证明思路**：

使用狄利克雷核和黎曼-勒贝格引理。

**关键步骤**：

1. **狄利克雷核**：
   $$D_N(x) = \frac{\sin((N+1/2)x)}{2\sin(x/2)}$$

2. **部分和**：
   $$S_N(x) = \frac{1}{\pi}\int_{-\pi}^{\pi} f(t)D_N(x-t)dt$$

3. **收敛性**：使用狄利克雷条件和黎曼-勒贝格引理证明收敛性。

### 4.3 定理的意义
