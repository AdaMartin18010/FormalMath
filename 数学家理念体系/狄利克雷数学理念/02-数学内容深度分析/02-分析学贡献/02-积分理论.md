## 一、黎曼积分理论

### 1.1 积分的定义

**黎曼积分**：

函数 $f$ 在区间 $[a, b]$ 上的黎曼积分定义为：

$$\int_a^b f(x)dx = \lim_{n \to \infty} \sum_{i=1}^n f(\xi_i)\Delta x_i$$

其中：

- $\Delta x_i = x_i - x_{i-1}$ 是子区间的长度
- $\xi_i \in [x_{i-1}, x_i]$ 是子区间中的点
- 极限是对所有分割取上确界和下确界

**上积分和下积分**：

- **上积分**：$\overline{\int_a^b} f(x)dx = \inf U(f, P)$
- **下积分**：$\underline{\int_a^b} f(x)dx = \sup L(f, P)$

其中 $U(f, P)$ 和 $L(f, P)$ 分别是上黎曼和和下黎曼和。

**积分存在**：

如果上积分等于下积分，则函数黎曼可积，积分值为：

$$\int_a^b f(x)dx = \overline{\int_a^b} f(x)dx = \underline{\int_a^b} f(x)dx$$

### 1.2 可积性条件

**必要条件**：

函数必须有界。如果函数无界，则不是黎曼可积的。

**充分条件**：

1. **连续性**：如果函数在 $[a, b]$ 上连续，则黎曼可积
2. **有限间断点**：如果函数只有有限个间断点，则黎曼可积
3. **单调性**：如果函数单调，则黎曼可积

**可积性判据**：

函数 $f$ 在 $[a, b]$ 上黎曼可积，当且仅当对于任意 $\varepsilon > 0$，存在分割 $P$ 使得：

$$U(f, P) - L(f, P) < \varepsilon$$

### 1.3 积分性质

**线性性**：

$$\int_a^b [\alpha f(x) + \beta g(x)]dx = \alpha\int_a^b f(x)dx + \beta\int_a^b g(x)dx$$

**区间可加性**：

$$\int_a^b f(x)dx + \int_b^c f(x)dx = \int_a^c f(x)dx$$

**积分不等式**：

如果 $f(x) \leq g(x)$ 在 $[a, b]$ 上，则：

$$\int_a^b f(x)dx \leq \int_a^b g(x)dx$$

**中值定理**：

如果 $f$ 在 $[a, b]$ 上连续，则存在 $\xi \in [a, b]$ 使得：

$$\int_a^b f(x)dx = f(\xi)(b-a)$$

### 2.2 不可积函数的例子

**狄利克雷函数**：

狄利克雷函数不是黎曼可积的：

$$
D(x) = \begin{cases}
1 & \text{如果 } x \text{ 是有理数} \\
0 & \text{如果 } x \text{ 是无理数}
\end{cases}
$$

**不可积的原因**：

- 在任意区间中，有理数和无理数都稠密
- 上黎曼和和下黎曼和不相等
- 因此不是黎曼可积的

**意义**：

- 展示了函数的复杂性
- 推动了积分理论的发展
- 为勒贝格积分奠定了基础

## 三、积分计算方法

### 3.1 基本积分方法

**换元积分法**：

如果 $u = g(x)$，则：

$$\int f(g(x))g'(x)dx = \int f(u)du$$

**分部积分法**：

$$\int u dv = uv - \int v du$$

**有理函数积分**：

- 部分分式分解
- 计算有理函数的积分
- 应用有理函数积分

### 3.2 积分技巧

**对称性**：

- 利用函数的对称性简化积分
- 奇函数在对称区间上的积分为0
- 偶函数在对称区间上的积分可以简化

**周期性**：

- 利用函数的周期性简化积分
- 周期函数在周期区间上的积分
- 应用周期性计算积分

**其他技巧**：

- 使用各种技巧计算积分
- 简化积分计算
- 提高计算效率

### 3.3 数值积分

**矩形法**：

$$\int_a^b f(x)dx \approx \sum_{i=1}^n f(x_i)\Delta x$$

**梯形法**：

$$\int_a^b f(x)dx \approx \frac{b-a}{2n}[f(a) + 2\sum_{i=1}^{n-1} f(x_i) + f(b)]$$

**辛普森法**：

$$\int_a^b f(x)dx \approx \frac{b-a}{6n}[f(a) + 4\sum_{i=1}^{n} f(x_{i-1/2}) + 2\sum_{i=1}^{n-1} f(x_i) + f(b)]$$

## 四、积分理论的发展

### 4.1 勒贝格积分

**勒贝格积分**：

勒贝格积分是黎曼积分的推广，可以积分更多函数。

**可积性**：

- 黎曼可积函数都是勒贝格可积的
- 勒贝格可积函数不一定是黎曼可积的
- 狄利克雷函数是勒贝格可积的

**优势**：

- 可以积分更多函数
