## 一、级数收敛性理论

### 1.1 收敛性的严格定义

**级数的定义**：

无穷级数是无穷多个数的和：

$$\sum_{n=1}^\infty a_n = a_1 + a_2 + a_3 + \cdots$$

**部分和**：

前 $n$ 项的和称为部分和：

$$S_n = \sum_{k=1}^n a_k = a_1 + a_2 + \cdots + a_n$$

**收敛性定义**：

级数 $\sum_{n=1}^\infty a_n$ 收敛，如果部分和序列 $\{S_n\}$ 收敛，即存在极限：

$$S = \lim_{n \to \infty} S_n$$

此时 $S$ 称为级数的和。

**发散**：

如果部分和序列不收敛，则级数发散。

### 1.2 收敛性判据

**柯西判据**：

级数 $\sum_{n=1}^\infty a_n$ 收敛的充分必要条件是：对于任意 $\varepsilon > 0$，存在 $N$ 使得当 $m > n > N$ 时：

$$\left|\sum_{k=n+1}^m a_k\right| < \varepsilon$$

**比较判据**：

如果 $0 \leq a_n \leq b_n$ 且 $\sum_{n=1}^\infty b_n$ 收敛，则 $\sum_{n=1}^\infty a_n$ 也收敛。

**比值判据**：

如果 $\lim_{n \to \infty} \left|\frac{a_{n+1}}{a_n}\right| = L$，则：

- 如果 $L < 1$，级数绝对收敛
- 如果 $L > 1$，级数发散
- 如果 $L = 1$，判据失效

**根值判据**：

如果 $\lim_{n \to \infty} \sqrt[n]{|a_n|} = L$，则：

- 如果 $L < 1$，级数绝对收敛
- 如果 $L > 1$，级数发散
- 如果 $L = 1$，判据失效

### 1.3 绝对收敛与条件收敛

**绝对收敛**：

如果 $\sum_{n=1}^\infty |a_n|$ 收敛，则级数 $\sum_{n=1}^\infty a_n$ 绝对收敛。

**条件收敛**：

如果级数 $\sum_{n=1}^\infty a_n$ 收敛但不绝对收敛，则条件收敛。

**绝对收敛的性质**：

- 绝对收敛的级数可以重新排列而不改变和
- 绝对收敛的级数的乘积也绝对收敛

**条件收敛的性质**：

- 条件收敛的级数重新排列可能改变和（黎曼重排定理）
- 条件收敛的级数需要小心处理

## 二、狄利克雷级数理论

### 2.1 狄利克雷级数的定义

**定义**：

狄利克雷级数是形如

$$\sum_{n=1}^\infty \frac{a_n}{n^s}$$

的级数，其中 $\{a_n\}$ 是数论函数，$s$ 是复变量。

**特殊情形**：

1. **黎曼ζ函数**：当 $a_n = 1$ 对所有 $n$ 时
   $$\zeta(s) = \sum_{n=1}^\infty \frac{1}{n^s}$$

2. **狄利克雷L函数**：当 $a_n = \chi(n)$ 是特征函数时
   $$L(s, \chi) = \sum_{n=1}^\infty \frac{\chi(n)}{n^s}$$

**收敛域**：

狄利克雷级数在 $\text{Re}(s) > \sigma_a$ 时绝对收敛，其中 $\sigma_a$ 是绝对收敛横坐标。

### 2.2 狄利克雷级数的性质

**欧拉乘积**：

如果 $a_n$ 是积性函数，则：

$$\sum_{n=1}^\infty \frac{a_n}{n^s} = \prod_p \left(1 + \frac{a_p}{p^s} + \frac{a_{p^2}}{p^{2s}} + \cdots\right)$$

其中乘积遍历所有素数 $p$。

**解析延拓**：

狄利克雷级数可以解析延拓到更大的区域。

**函数方程**：

某些狄利克雷级数满足函数方程，将 $s$ 和 $1-s$ 联系起来。

### 2.3 狄利克雷级数的应用

**数论应用**：

1. **素数分布**：
   - 使用狄利克雷级数研究素数分布
   - 证明狄利克雷定理
   - 研究算术级数中的素数

2. **数论函数**：
   - 使用狄利克雷级数研究数论函数
   - 研究数论函数的性质
   - 应用数论函数解决问题

**分析应用**：

- 应用到分析问题
- 研究函数的性质

## 三、傅里叶级数理论

### 3.1 傅里叶级数的收敛性

**傅里叶级数**：

周期为 $2\pi$ 的函数 $f(x)$ 的傅里叶级数为：

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^\infty (a_n\cos nx + b_n\sin nx)$$

其中：

$$a_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\cos nx dx$$

$$b_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\sin nx dx$$

**收敛性问题**：

傅里叶级数是否收敛到 $f(x)$？这是一个重要问题。

### 3.2 狄利克雷条件

**狄利克雷条件**：

如果函数 $f(x)$ 满足以下条件，则其傅里叶级数在 $f(x)$ 的连续点收敛到 $f(x)$，在间断点收敛到左右极限的平均值：

1. **有界性**：$f(x)$ 在 $[-\pi, \pi]$ 上有界
2. **分段单调**：$f(x)$ 在 $[-\pi, \pi]$ 上只有有限个极值点
3. **分段连续**：$f(x)$ 在 $[-\pi, \pi]$ 上只有有限个间断点，且每个间断点都是第一类间断点

**定理表述**：

如果 $f(x)$ 满足狄利克雷条件，则：

$$f(x) = \frac{a_0}{2} + \sum_{n=1}^\infty (a_n\cos nx + b_n\sin nx)$$

在 $f(x)$ 的连续点成立，在间断点 $x_0$ 处：

$$\frac{f(x_0^-) + f(x_0^+)}{2} = \frac{a_0}{2} + \sum_{n=1}^\infty (a_n\cos nx_0 + b_n\sin nx_0)$$

### 3.3 傅里叶级数的应用

**信号处理**：

- 信号分解
- 信号合成
- 信号分析

**物理应用**：

- 波动问题
- 热传导问题
- 其他物理问题

**工程应用**：

## 四、级数求和理论

### 4.1 级数求和的方法

**直接求和**：

对于某些简单级数，可以直接求和。

**例子**：

**几何级数**：

$$\sum_{n=0}^\infty r^n = \frac{1}{1-r} \quad (|r| < 1)$$

**方法技巧**：

1. **部分分式**：将级数分解为部分分式
2. **积分方法**：使用积分求和
3. **微分方法**：使用微分求和

### 4.2 特殊级数的和

**调和级数**：

$$\sum_{n=1}^\infty \frac{1}{n} = \infty$$

（发散）

**交错调和级数**：

$$\sum_{n=1}^\infty \frac{(-1)^{n-1}}{n} = \ln 2$$

**巴塞尔问题**：

$$\sum_{n=1}^\infty \frac{1}{n^2} = \frac{\pi^2}{6}$$

（欧拉解决）

### 4.3 级数求和的应用

**数值计算**：
