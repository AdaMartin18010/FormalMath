## 一、引言：完全性定理的意义

### 1.1 历史背景

**Gödel的完全性定理（1930）**：

- 一阶逻辑的完全性
- 语法与语义的对应
- 为模型论提供基础

**塔斯基的贡献**：

- 模型论形式的完全性定理
- 紧致性定理
- 模型存在性定理

---

### 1.2 完全性的两种形式

**语法完全性**：

所有有效的公式都可以在系统中证明。

**语义完全性**：

所有可证的公式都是有效的。

**关系**：

- 可靠性：可证 $\Rightarrow$ 有效
- 完全性：有效 $\Rightarrow$ 可证

---

## 二、Gödel完全性定理

### 2.1 定理陈述

**定理（Gödel, 1930）**：

一阶逻辑是语法完全的，即：

- 如果 $\models \varphi$（$\varphi$ 在所有模型中为真），则 $\vdash \varphi$（$\varphi$ 可证）

**等价表述**：

- 如果 $\Gamma \models \varphi$，则 $\Gamma \vdash \varphi$
- 如果 $\Gamma$ 一致，则 $\Gamma$ 有模型

---

### 2.2 证明思路

**Gödel完全性定理的证明（1930）**：

这是逻辑学中最重要的定理之一，连接了语法（可证性）与语义（有效性）。

**关键步骤**：

1. **扩展理论**：如果理论一致，则可以通过添加新常数扩展
   - 添加新常数符号 $c_1, c_2, \ldots$
   - 对每个存在语句 $\exists x \varphi(x)$，添加公理 $\varphi(c)$（其中 $c$ 是新常数）

2. **构造极大一致理论**：构造"极大一致"的理论
   - 使用Zorn引理或递归构造
   - 对每个语句 $\varphi$，要么 $\varphi$ 在理论中，要么 $\neg \varphi$ 在理论中
   - 理论是一致的且极大的

3. **构造典范模型**：构造"典范模型"
   - 论域：项的等价类（模理论中的等式）
   - 解释：通过理论定义关系和函数

4. **证明满足性**：证明该模型满足理论
   - 使用归纳法证明对每个语句 $\varphi$，$\mathcal{M} \models \varphi$ 当且仅当 $\varphi$ 在理论中

**数学表述**：

完全性定理：
$$\text{如果 } \Gamma \models \varphi \text{，则 } \Gamma \vdash \varphi$$

等价表述：
$$\text{如果 } \Gamma \text{ 一致，则 } \Gamma \text{ 有模型}$$

**Henkin构造**：

Henkin在1949年给出了一个更直接的证明方法：
1. 添加新常数使理论具有见证性质（witness property）
2. 扩展为极大一致理论
3. 构造典范模型

**例子1：完全性定理的应用**：

如果理论 $T$ 一致，则存在模型 $\mathcal{M} \models T$。这为模型论提供了基础。

## 三、模型论完全性定理

### 3.1 模型存在性定理

**定理（模型论形式）**：

一个一阶理论 $T$ 是一致的，当且仅当 $T$ 有模型。

**意义**：

- 连接语法（一致性）与语义（模型存在）
- 为模型论提供基础

---

### 3.2 证明方法

**Henkin构造**：

- 添加新常数
- 构造极大一致理论
- 构造典范模型

**超积构造**：

- 使用超滤子
- 构造新模型
- 保持理论

---

## 四、紧致性定理

### 4.1 定理陈述

**定理（Tarski, 1930s）**：

一个一阶理论 $T$ 有模型，当且仅当 $T$ 的每个有限子集有模型。

**等价表述**：

- 如果每个有限子集有模型，则整个理论有模型
- 如果理论无模型，则存在有限子集无模型

---

### 4.2 证明

**证明思路**：

- 使用完全性定理
- 如果理论一致，则有模型
- 如果理论不一致，则存在有限子集不一致

---

### 4.3 应用

**非标准分析**：

- 构造非标准实数
- 使用紧致性定理
- 提供新的数学工具

**模型的存在性**：

- 证明某些理论有模型
- 构造特殊模型
- 研究模型的结构

## 五、Löwenheim-Skolem定理

### 5.1 向下Löwenheim-Skolem定理

**定理**：

如果可数理论有无限模型，则它有可数模型。

**意义**：

- 一阶逻辑不能"固定"模型的基数
- 存在不同基数的模型

---

### 5.2 向上Löwenheim-Skolem定理

**定理**：

如果理论有无限模型，则对任意无限基数 $\kappa$，它有基数 $\geq \kappa$ 的模型。

**意义**：

- 一阶理论有"大"模型
- 不能限制模型的基数

## 九、数学公式总结

### 核心公式

1. **Gödel完全性定理**：
   $$\text{如果 } \Gamma \models \varphi \text{，则 } \Gamma \vdash \varphi$$

2. **等价表述**：
   $$\text{如果 } \Gamma \text{ 一致，则 } \Gamma \text{ 有模型}$$

3. **模型存在性定理**：
   $$T \text{ 一致} \iff T \text{ 有模型}$$

4. **紧致性定理**：
   $$T \text{ 有模型} \iff \text{每个有限子集有模型}$$

5. **向下Löwenheim-Skolem定理**：
   $$\text{如果可数理论有无限模型，则它有可数模型}$$

6. **向上Löwenheim-Skolem定理**：
   $$\text{如果理论有无限模型，则对任意无限基数 } \kappa \text{，它有基数 } \geq \kappa \text{ 的模型}$$

7. **可靠性定理**：
   $$\text{如果 } \Gamma \vdash \varphi \text{，则 } \Gamma \models \varphi$$

8. **完全性定理（语义形式）**：
   $$\text{如果 } \Gamma \models \varphi \text{，则 } \Gamma \vdash \varphi$$

9. **Henkin构造**：
   $$\text{通过添加新常数和极大一致理论构造模型}$$

10. **典范模型**：
    $$\mathcal{M} = (M, I) \text{ 其中 } M = \{\text{项的等价类}\}$$
