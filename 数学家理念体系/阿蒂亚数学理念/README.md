## 📚 核心内容方向

### 1. 指标定理（Atiyah-Singer Index Theorem）

- **历史背景**: 1963年与Singer合作证明
- **数学内容**: 椭圆算子的解析指标等于拓扑指标
- **应用**: 几何、拓扑、物理的统一

### 2. K理论

- **拓扑K理论**: 向量丛的稳定等价类
- **代数K理论**: 环的K群
- **应用**: 代数拓扑、代数几何

### 3. 拓扑量子场论

- **几何与物理的统一**: 通过几何方法研究物理
- **规范理论**: 纤维丛与规范场
- **弦理论**: 几何结构在弦理论中的应用
