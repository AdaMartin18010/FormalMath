## 一、核心合作者

### 1.1 Minkowski：最亲密的合作者

**关系**：

- 1880s：柯尼斯堡大学同学
- 终身挚友
- 最密切的数学合作

**合作领域**：

```
1. 数论（1880s-1890s）
   - 二次型理论
   - 几何数论

2. 相对论（1905-1909）
   - Minkowski时空几何
   - 四维时空概念
   - 希尔伯特独立发现场方程

3. 几何学
   - 凸体理论
   - 几何不等式
```

**Minkowski的贡献**：

- 将相对论几何化
- 四维时空概念
- 影响爱因斯坦

**Minkowski去世**（1909）：
希尔伯特悼词：
> "我失去了最好的朋友。Minkowski的几何直觉无人能及。"

---

### 1.2 Emmy Noether：抽象化的继承者

**关系**：

- 1915年：希尔伯特邀请Noether来哥廷根
- 导师-学生关系
- 哲学分歧但相互尊重

**合作内容**：

```
不变量理论：
- 希尔伯特：基底定理（存在性）
- Noether：结构理论（抽象化）

理想理论：
- 希尔伯特：理想概念
- Noether：Noether环理论
```

**希尔伯特的支持**：

```
1915年教授会：
反对者："女性不能任教"

希尔伯特：
"先生们，这是大学，不是澡堂。
我不明白性别与数学能力有何关系。"

结果：
Noether以希尔伯特名义开课
1919年正式获得任教资格
```

**Noether的评价**：
> "希尔伯特给了我研究的自由。他相信抽象化的力量。"

---

### 1.3 Bernays与Ackermann：逻辑合作

**Paul Bernays**（1888-1977）：

**合作**：

- 《数学基础》I & II（1934, 1939）
- 技术细节的实现
- 形式化系统的构建

**关系**：

- 最信任的合作者
- 希尔伯特提供思想，Bernays负责技术
- 终身友谊

**Bernays的贡献**：

- 形式化技术
- NBG集合论（Bernays-Gödel）
- 证明论细节

---

**Wilhelm Ackermann**（1896-1962）：

**合作**：

- 《数理逻辑基础》（1928）
- 可判定性问题
- 递归函数

**Ackermann函数**：

```
A(0,n) = n+1
A(m+1,0) = A(m,1)
A(m+1,n+1) = A(m, A(m+1,n))

性质：
- 原始递归
- 增长极快
- 递归论的重要例子
```

## 二、重要论战

### 2.1 vs Gordan：构造性vs存在性

**背景**（1888）：

- 希尔伯特基底定理（非构造性）
- Gordan："Das ist Theologie!"

**Gordan的立场**：

```
数学证明必须：
1. 证明存在
2. 给出构造
3. 提供算法

希尔伯特只做了1，缺2和3
```

**希尔伯特的回应**（1893）：
> "Gordan先生说这是神学，但我说这是数学。重要的不是**如何构造**，而是**理解为何存在**。"

**历史结果**：

- Gordan后来接受（1899年承认希尔伯特正确）
- 但哲学分歧持续
- 现代：两者都有价值（理论+算法）

---

### 2.2 vs Brouwer：形式主义vs直觉主义

**冲突时间线**：

```
1912：Brouwer发表《直觉主义与形式主义》
      → 公开批判希尔伯特

1920s：Weyl转向直觉主义
      → 希尔伯特感到"背叛"

1928：Math. Ann.编委会冲突
      → 希尔伯特开除Brouwer

1929：Einstein等签名抗议
      → 希尔伯特拒绝妥协
```

**核心分歧**：

| 方面 | 希尔伯特 | Brouwer |
|------|---------|---------|
| 排中律 | ✅ 接受 | ❌ 拒绝 |
| 实无限 | ✅ 理想元素 | ❌ 拒绝 |
| 存在证明 | ✅ 非构造性可 | ❌ 必须构造 |
| 数学本质 | 符号游戏 | 心智构造 |

**希尔伯特的攻击**（1925）：
> "禁止数学家使用排中律，就像禁止天文学家使用望远镜和拳击家使用拳头一样。"

**Brouwer的回应**：
> "希尔伯特的形式系统只是**语言尸体**。没有直觉的数学是僵尸。"

**最终**：

- 个人关系破裂（终身未和解）
- 但方法论都影响后世
- 现代：形式化+构造性（类型论）

---

### 2.3 vs Poincaré：形式主义vs约定论

**关系**：

- 相互尊重，但哲学分歧
- 未直接论战（Poincaré 1912年去世）

**Poincaré的批评**（《科学与假设》）：

```
1. 公理非完全任意（受物理经验约束）
2. 形式化失去"生育性"
3. 忽略数学的物理应用价值
```

**希尔伯特的回应**（间接）：

```
1. 区分纯数学与应用
2. 公理化是验证，不阻碍发现
3. 数学有自主价值
```

**现代综合**：

- 纯数学：希尔伯特对
- 应用数学：Poincaré对
- 两者互补

## 三、与学生的关系

### 3.1 Weyl：最杰出的学生

**关系演变**：

```
1910s：Weyl是希尔伯特最欣赏的学生
1921：Weyl发表《论新基础危机》
      → 支持Brouwer
      → 希尔伯特感到"背叛"

1927：Weyl回归经典立场
      → 与希尔伯特和解

1930s：共同流亡美国
      → 普林斯顿IAS
```

**Weyl的评价**（1944）：
> "希尔伯特是20世纪数学的象征。我们的分歧在于哲学，但数学上我永远是他的学生。"

---

### 3.2 Courant：应用数学的传承

**关系**：

- 学生（1907-1910）
- 继承Klein的应用传统
- 但采用希尔伯特的严格性

**Courant的贡献**：

- 应用数学（偏微分方程）
- Courant研究所（NYU）
- 《数学物理方法》

**希尔伯特的影响**：

- 严格性标准
- 但保留应用导向

## 五、论战的哲学意义

### 5.1 数学哲学的成熟

**三大流派的形成**：

```
形式主义（Hilbert）
    ↓
直觉主义（Brouwer）
    ↓
逻辑主义（Russell）
    ↓
20世纪数学哲学三大传统
```

**现代综合**：

- 方法：形式主义（主流）
- 哲学：多元主义
- 实践：综合使用

---

### 5.2 论战的价值

**推动研究**：

- 基础危机 → 基础研究
- 哥德尔定理的发现
- 递归论的建立

**澄清概念**：

- 什么是数学？
- 什么是证明？
- 什么是存在？

**Weyl评价**（1946）：
> "希尔伯特与Brouwer的论战是**悲剧**，但也是**必要的**。它迫使我们深刻反思数学的本质。"

## 六、总结

### 合作模式

**希尔伯特的特点**：

1. **开放**：欢迎不同观点
2. **严格**：要求逻辑严密
3. **支持**：力挺学生（如Noether）
4. **国际**：吸引全球学者

---

### 论战的影响

**正面影响**：

- **推动数学基础研究**：
  - 促使数学家深入思考数学基础问题
  - 推动集合论、逻辑学的发展
  - 影响Gödel、Tarski等逻辑学家的研究

- **澄清哲学立场**：
  - 明确形式主义、直觉主义、逻辑主义的区别
  - 推动数学哲学的发展
  - 影响现代数学哲学的研究

- **产生新理论**：
  - 证明论（Proof Theory）
  - 递归论（Recursion Theory）
  - 模型论（Model Theory）
  - 类型论（Type Theory）

**负面影响**：

- **个人关系破裂**：
  - 与Brouwer的关系彻底破裂
  - 1928年将Brouwer从《数学年鉴》编委中除名
  - 影响数学界的团结

- **学派分裂**：
  - 形式主义学派与直觉主义学派的分裂
  - 影响数学基础研究的统一
  - 但最终都贡献数学的发展

- **最终评价**：
  - 虽然论战激烈，但都推动了数学的发展
  - 不同观点的交锋促进了数学的进步
  - 现代数学融合了多种观点

---

**最终评价**：
希尔伯特既是**合作者**（Minkowski, Noether），也是**论战者**（Brouwer, Gordan）。他的开放与严格，合作与争论，共同塑造了20世纪数学的面貌。
