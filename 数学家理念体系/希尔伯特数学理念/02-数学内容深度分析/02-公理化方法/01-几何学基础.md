# 希尔伯特《几何基础》：公理化方法的诞生

## 摘要

1899年出版的《几何基础》（Grundlagen der Geometrie）标志着现代公理化方法的诞生。希尔伯特将欧几里得几何重构为五组二十条公理的严格体系，消除了《原本》中对图形直观的依赖。本文系统阐述希尔伯特公理系统的结构，分析其独立性、完备性证明，并与欧几里得方法进行对比，探讨公理化方法对20世纪数学的深远影响。

## 一、历史背景与写作动机

### 1.1 欧几里得《原本》的缺陷

欧几里得《原本》（约公元前300年）两千年来被视为几何学的典范，但19世纪数学家发现其存在严重逻辑漏洞：

**主要问题**：
- **隐含假设**：依赖图形直观（如"介于"关系未定义）
- **公理不足**：某些"显然"的事实实际无法从公设推出
- **逻辑跳跃**：证明中使用未声明的假设（如Pasch公理）

**Pasch的发现**（1882）：
德国数学家Moritz Pasch指出，欧几里得隐含使用了以下命题：
> "若一条直线穿过三角形的一边而不经过顶点，则必穿过另外两边之一。"

这一公理在欧几里得体系中是未声明的假设。

### 1.2 非欧几何的冲击

19世纪初，Bolyai和Lobachevsky发现双曲几何，Riemann发展了椭圆几何。这些发现动摇了"欧氏几何是唯一真几何"的信念。

**核心问题**：
> "如何严格证明非欧几何的一致性？"

Hilbert意识到，需要一种完全不依赖直观的公理化方法。

### 1.3 公理化转向

**Hilbert的宣言**（1899年，Festschrift为Gauss-Weber纪念碑揭幕）：
> "我们必定可以用'桌子、椅子、啤酒杯'来代替'点、线、面'，只要它们满足公理所规定的关系。"

这一声明标志着数学从"研究对象的本质"转向"研究对象间的关系"。

## 二、希尔伯特公理系统

### 2.1 公理的组织结构

希尔伯特将公理分为五组，共20条：

| 组别 | 名称 | 数量 | 核心内容 |
|------|------|------|----------|
| I | 关联公理 | 8条 | 点、线、面的隶属关系 |
| II | 顺序公理 | 4条 | "介于"关系的性质 |
| III | 合同公理 | 5条 | 线段和角的相等关系 |
| IV | 平行公理 | 1条 | 欧几里得平行公设 |
| V | 连续公理 | 2条 | 阿基米德性和完备性 |

### 2.2 第一组：关联公理（Incidence Axioms）

基本词汇：点（$A, B, C, \ldots$）、线（$a, b, c, \ldots$）、面（$\alpha, \beta, \gamma, \ldots$）、关系 $\in$（"在……上"）。

**公理 I-1**（两点定直线——存在性）：
$$\forall A, B \in \mathcal{P}\, (A \neq B \to \exists a \in \mathcal{L}: A \in a \land B \in a)$$

**公理 I-2**（两点定直线——唯一性）：
$$\forall A, B \in \mathcal{P}\, \forall a, b \in \mathcal{L}\, [(A \neq B \land A \in a \land B \in a \land A \in b \land B \in b) \to a = b]$$

**公理 I-3**（直线上至少三点）：
$$\forall a \in \mathcal{L}\, \exists A, B, C \in a: A \neq B \land B \neq C \land C \neq A$$

**公理 I-4**（非共线三点存在）：
$$\exists A, B, C \in \mathcal{P}: \neg \exists a \in \mathcal{L}: (A \in a \land B \in a \land C \in a)$$

**公理 I-5至I-8**：平面的类似公理（三点定平面、平面的唯一性等）。

### 2.3 第二组：顺序公理（Axioms of Order）

引入未定义关系：$B$ **介于** $A$ 和 $C$（记作 $A*B*C$）。

**公理 II-1**（介于的基本性质）：
若 $A*B*C$，则：
- $A, B, C$ 是不同点且共线
- $A*B*C$ 蕴含 $C*B*A$
- 不同时有 $B*A*C$ 或 $A*C*B$

**公理 II-2**（介于的存在性）：
$$\forall A, C \in a\, \exists B \in a: A*B*C$$

**公理 II-3**（Pasch公理）：
设 $\triangle ABC$ 是不在一条线上的三点，$a$ 是不经过 $A, B, C$ 的直线。若 $a$ 与线段 $AB$ 相交，则 $a$ 必与 $AC$ 或 $BC$ 相交（但不同时与两者相交）。

### 2.4 第三组：合同公理（Congruence Axioms）

引入关系：线段合同 $AB \equiv A'B'$、角合同 $\angle ABC \equiv \angle A'B'C'$。

**公理 III-1**（线段移动）：
给定线段 $AB$ 和射线 $A'C'$，在 $A'C'$ 上存在唯一点 $B'$ 使 $A'B' \equiv AB$。

**公理 III-2**（合同传递性）：
若 $AB \equiv A'B'$ 且 $AB \equiv A''B''$，则 $A'B' \equiv A''B''$。

**公理 III-3**（线段加法）：
若 $A*B*C$，$A'*B'*C'$，$AB \equiv A'B'$，$BC \equiv B'C'$，则 $AC \equiv A'C'$。

**公理 III-4**（角移动）：类似于III-1。

**公理 III-5**（SAS全等）：
若 $AB \equiv A'B'$，$AC \equiv A'C'$，$\angle BAC \equiv \angle B'A'C'$，则 $\triangle ABC \equiv \triangle A'B'C'$。

### 2.5 第四组：平行公理

**公理 IV**（欧几里得平行公理）：
设 $a$ 是直线，$A$ 是不在 $a$ 上的点。则存在**唯一**一条经过 $A$ 且与 $a$ 不相交的直线。

### 2.6 第五组：连续公理

**公理 V-1**（阿基米德公理）：
设 $AB$ 和 $CD$ 是任意两线段，则存在自然数 $n$ 使得沿 $AB$ 连续放置 $n$ 次 $CD$ 后，所得线段超过 $AB$：
$$\exists n \in \mathbb{N}: n \cdot CD > AB$$

**公理 V-2**（完备性公理）：
点、线、面系统是在保持所有前述公理的前提下不能再扩充的系统。

*注*：这一公理由Hilbert在后续版本中修正，现代通常用戴德金完备性替代。

## 三、与欧几里得方法的对比

### 3.1 方法论的根本差异

| 方面 | 欧几里得《原本》 | 希尔伯特《几何基础》 |
|------|------------------|----------------------|
| **基本概念** | 点、线、面有"定义"（但循环） | 点、线、面为**未定义项** |
| **公理来源** | "自明真理" | 任意约定的关系 |
| **图形作用** | 证明的核心依据 | **完全不使用** |
| **逻辑基础** | 有隐含假设 | **完全严格** |
| **可解释性** | 固定解释 | 多重模型可能 |
| **证明方式** | 文字+图形 | 纯符号演算 |

### 3.2 定义问题的对比

**欧几里得的"定义"**（《原本》第一卷）：
> "点是没有部分的东西"  
> "线是没有宽度的长度"

Hilbert指出这些"定义"是循环的："部分"、"宽度"本身需要定义。

**希尔伯特的方法**：
不定义"点是什么"，只规定点满足什么关系。正如Hilbert所说，可以用"桌子、椅子"代替"点、线"。

### 3.3 证明严格性的对比

**欧几里得《原本》第一卷命题1**（等边三角形构造）：
问题：证明中使用了两个圆的交点存在，但未证明交点存在。

**希尔伯特的严格处理**：
需要连续公理（V组）来保证圆与直线的交点存在。

## 四、公理的独立性证明

### 4.1 独立性的概念

**定义**：公理 $A$ 关于公理系统 $S$ 是**独立**的，若 $S \setminus \{A\} \not\vdash A$。

等价地：存在 $S \setminus \{A\}$ 的模型，其中 $A$ 不成立。

### 4.2 平行公理的独立性

**定理**（Hilbert）：平行公理（IV）独立于关联、顺序、合同、连续公理。

**证明**：构造双曲几何模型。

**Poincaré圆盘模型**：
- "点"：单位圆盘 $D = \{ z \in \mathbb{C} : |z| < 1 \}$ 内的点
- "线"：与边界正交的圆弧（包括直径）
- 满足I、II、III、V组公理
- 平行公理不成立（过线外一点有无穷多条平行线）

### 4.3 其他公理的独立性

| 公理 | 独立性的证明模型 |
|------|------------------|
| 阿基米德公理（V-1） | 非阿基米德几何（含无穷小） |
| Pasch公理（II-4） | 去掉II-4的特殊模型 |
| SAS全等（III-5） | 扭曲几何（保持其他公理，改变度量） |

### 4.4 独立性方法的意义

Hilbert的模型构造法开创了数学基础的新纪元：
- **相对一致性**：若实数系统一致，则欧氏几何一致
- **非欧几何合法化**：双曲几何与欧氏几何同样"真"
- **公理选择自由**：公理的选择是任意的，只要相容

## 五、完备性定理

### 5.1 完备性的概念

**定义**（范畴性）：公理系统是**完备的**（categorical），若任意两个模型同构。

等价表述：公理系统**完全确定**了研究对象的结构。

### 5.2 希尔伯特完备性定理

**定理**（Hilbert）：满足五组公理的几何系统在同构意义下唯一。

**证明概要**：

**步骤1：坐标化**
利用公理建立点与 $\mathbb{R}^2$ 的一一对应：
$$\Phi: \mathcal{P} \to \mathbb{R}^2, \quad P \mapsto (x_P, y_P)$$

**步骤2：翻译公理为代数**
| 几何公理 | 代数对应 |
|----------|----------|
| I组 | 线性方程 |
| II组 | 凸性、顺序 |
| III组 | 距离公式、余弦定理 |
| IV组 | 平行线斜率相等 |
| V组 | 实数完备性 |

**步骤3：同构证明**
证明任意两个模型都同构于 $(\mathbb{R}^2, \text{标准结构})$，故彼此同构。

### 5.3 完备性的意义

**与第一-order逻辑的区分**：
- Hilbert的完备性是**范畴性**（同构唯一）
- 现代逻辑中的"完备性"是语法-语义对应

**证明力**：
完备性保证任何欧氏几何的真命题都可从20条公理推出。

## 六、公理化方法的影响

### 6.1 对20世纪数学的影响

希尔伯特的公理化方法成为20世纪数学的标准范式：

| 数学领域 | 公理化成果 | 代表人物 |
|----------|------------|----------|
| 概率论 | Kolmogorov公理（1933） | A.N. Kolmogorov |
| 集合论 | ZFC公理系统（1908-1922） | Zermelo, Fraenkel |
| 拓扑学 | 开集公理 | Hausdorff |
| 代数 | 群、环、域的抽象定义 | Noether, van der Waerden |
| 量子力学 | von Neumann公理（1932） | J. von Neumann |

### 6.2 与Brouwer的直觉主义论战

L.E.J. Brouwer批评Hilbert的形式主义：
> "数学对象必须在直观中被构造，而非通过形式规则定义。"

**Hilbert的回应**：
公理化方法不排除直观，而是将直观从证明中剥离，保证逻辑的纯粹性。

### 6.3 与Gödel定理的关系

**Hilbert计划**（1920s）：
1. 将所有数学形式化
2. 用有穷方法证明形式系统的一致性
3. 证明完备性（所有真命题可证）

**Gödel不完备性定理**（1931）：
- 任何足够强的形式系统，若一致则不完备
- 系统不能证明自身的一致性

**影响**：Hilbert计划的某些目标被证明不可能实现，但公理化方法作为数学工具的价值不受影响。

### 6.4 现代数学结构主义

Bourbaki学派（1935-）将公理化方法推向极致：
- 数学是"结构"的研究
- 群、拓扑空间、向量空间都是抽象结构
- 公理定义结构，定理揭示结构间关系

## 七、现代形式化实现

### 7.1 Tarski的简化

Alfred Tarski（1959）简化了希尔伯特系统：
- 仅使用点和介于关系
- 10条公理（而非20条）
- 可判定性：存在算法判定初等几何命题

### 7.2 计算机形式化

**Lean4中的希尔伯特几何**：

```lean
-- 未定义类型
variable (Point Line Plane : Type)
variable (lies_on : Point → Line → Prop)
variable (between : Point → Point → Point → Prop)
variable (congruent_seg : Point → Point → Point → Point → Prop)

-- I-1: 两点定直线（存在性）
axiom line_existence (A B : Point) (h : A ≠ B) :
  ∃ a : Line, lies_on A a ∧ lies_on B a

-- I-2: 两点定直线（唯一性）  
axiom line_uniqueness (A B : Point) (a b : Line)
  (h1 : A ≠ B) (h2 : lies_on A a ∧ lies_on B a)
  (h3 : lies_on A b ∧ lies_on B b) :
  a = b

-- II-4: Pasch公理
axiom pasch (A B C P : Point) (a : Line)
  (h1 : ¬collinear A B C)
  (h2 : between A P B ∧ lies_on P a)
  (h3 : ¬(lies_on A a ∨ lies_on B a ∨ lies_on C a)) :
  (∃ Q, between A Q C ∧ lies_on Q a) ∨
  (∃ R, between B R C ∧ lies_on R a)
```

### 7.3 自动定理证明

基于希尔伯特公理系统，计算机可以：
- 验证几何证明的正确性
- 自动搜索证明
- 发现新的几何定理
