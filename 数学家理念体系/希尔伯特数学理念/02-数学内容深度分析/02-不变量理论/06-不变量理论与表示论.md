## 一、群表示论

### 1.1 线性表示

**定义**：

```
群G的表示：
ρ: G → GL(V)

其中：
- V：向量空间
- GL(V)：一般线性群
- ρ是群同态

性质：
- ρ(gh) = ρ(g)ρ(h)
- ρ(e) = I
```

**例子**：

```
对称群S₃：
- 3维表示（置换矩阵）
- 2维表示（不可约）
- 1维表示（平凡）
```

---

### 1.2 不可约表示

**定义**：

```
表示ρ不可约 ⟺
V没有非平凡不变子空间

Maschke定理：
有限群表示完全可约
    ↓
V = V₁ ⊕ ... ⊕ Vₙ
（不可约表示的直和）
```

**意义**：

- 分类表示
- 结构理解
- 应用广泛

## 二、不变量与表示

### 2.1 不变量环的表示论

**关系**：

```
群G作用在V上
    ↓
不变量环k[V]^G
    ↓
表示论研究

方法：
- 特征标理论
- Molien级数
- 不变量生成元
```

---

### 2.2 Molien定理

**定理**：

```
不变量环的Hilbert级数：

H(k[V]^G, t) = (1/|G|) ∑_{g∈G} 1/det(I-tg)

其中：
- g：表示矩阵
- det(I-tg)：特征多项式
```

**应用**：

- 计算不变量
- 生成元个数
- 度数的界

## 三、Schur-Weyl对偶

### 3.1 对偶性

**Schur-Weyl对偶**（1920s）：

```
GL(n)和Sₖ在(ℂⁿ)^⊗ᵏ上的作用：
- 对偶
- 交换
- 分解对应

结果：
(ℂⁿ)^⊗ᵏ = ⊕_{λ} V_λ ⊗ S_λ

其中：
- V_λ：GL(n)的不可约表示
- S_λ：Sₖ的不可约表示（Specht模）
- λ：分割
```

---

### 3.2 应用

**不变量理论**：

```
应用：
- 张量不变量
- 多线性不变量
- 组合结构
```

**现代发展**：

- 量子群
- 范畴化
- 几何表示论
