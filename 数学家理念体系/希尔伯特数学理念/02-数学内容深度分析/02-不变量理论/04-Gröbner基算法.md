## 二、Gröbner基的定义

### 2.1 单项式序

**定义**：

```
单项式序<：
- 全序
- 乘法保持：若u<v，则uw<vw
- 良序：任意下降链终止
```
**例子：字典序**：

```
x^a y^b < x^c y^d ⟺
a < c 或 (a=c 且 b<d)

例子：
y < x < xy < x² < x²y
```
**其他序**：

- 度字典序
- 反字典序
- 块序

---

### 2.2 首项与约化

**首项**：

```
给定单项式序<

多项式f的首项：
LT(f) = 最大的单项式（按<）

首项系数：
LC(f)

首项单项式：
LM(f) = LT(f)/LC(f)
```
**例子**：

```
f = 3x²y + 2xy² + 5x

字典序x>y：
LT(f) = 3x²y
LC(f) = 3
LM(f) = x²y
```
---

### 2.3 Gröbner基

**定义**：

```
理想I的Gröbner基G：
- G生成I
- {LT(g) | g∈G}生成LT(I)

其中：
LT(I) = {LT(f) | f∈I}
```
**性质**：

```
1. 任意f∈I可被G约化到0
2. 约化结果唯一（相对于G）
3. 可判定f∈I
```
---

## 三、Buchberger算法

### 3.1 S-多项式

**定义**：

```
给定f, g，单项式序<

S-多项式：
S(f,g) = (LCM/LT(f))·f - (LCM/LT(g))·g

其中：
LCM = LCM(LM(f), LM(g))
```
**例子**：

```
f = x²y + xy
g = xy² + x

字典序x>y：
LT(f) = x²y
LT(g) = xy²
LCM = x²y²

S(f,g) = (x²y²/x²y)·f - (x²y²/xy²)·g
       = y·f - x·g
       = y(x²y+xy) - x(xy²+x)
       = x²y² + xy² - x²y² - x²
       = xy² - x²
```
---

### 3.2 Buchberger准则

**定理**：

```
G是Gröbner基 ⟺
对所有f,g∈G，
S(f,g)被G约化到0
```
**算法**：

```
输入：F = {f₁,...,fₘ}
输出：G（Gröbner基）

步骤：
1. G := F
2. 对每对f,g∈G：
   - 计算S(f,g)
   - 约化S(f,g)相对于G → r
   - 若r≠0，添加r到G
3. 重复直到所有S-多项式约化为0
4. 输出G
```
---

### 3.3 约化Gröbner基

**约化基**：

```
G是约化的，若：
1. 所有g∈G的首项系数为1
2. 对g∈G，g-LT(g)不被{LT(h) | h∈G, h≠g}整除
```
**唯一性**：

```
给定理想I和单项式序<：
I有唯一的约化Gröbner基
```
---

## 四、应用

### 4.1 理想成员判定

**问题**：
> 给定f和理想I=(f₁,...,fₘ)，判定f∈I？

**方法**：

```
步骤1：计算I的Gröbner基G
步骤2：约化f相对于G → r
步骤3：f∈I ⟺ r=0
```
**例子**：

```
I = (x²+y, xy+x)
f = x³

Gröbner基G = {x²+y, y²+xy, ...}
约化f → r = y²
r≠0 → f∉I
```
---

### 4.2 方程组求解

**问题**：

```
解方程组：
f₁(x₁,...,xₙ) = 0
f₂(x₁,...,xₙ) = 0
...
fₘ(x₁,...,xₙ) = 0
```
**方法**：

```
步骤1：计算I=(f₁,...,fₘ)的Gröbner基G
步骤2：G的形式揭示解的结构
步骤3：逐步消元求解
```
**例子**：

```
方程组：
x²+y = 0
xy+x = 0

Gröbner基（字典序x>y）：
{x²+y, y²-y}

解：
y²-y = 0 → y=0或y=1
若y=0：x²=0 → x=0
若y=1：x²+1=0 → x=±i

解：(0,0), (i,1), (-i,1)
```
---

### 4.3 零点定理的计算

**问题**：
> 给定理想I，计算V(I)？

**方法**：

```
步骤1：计算I的Gröbner基G
步骤2：G的形式揭示V(I)的结构
步骤3：分解为不可约分支
```
**与希尔伯特零点定理**：

```
零点定理（理论）：
I(V(I)) = √I

Gröbner基（算法）：
- 计算√I
- 分解理想
- 实际计算V(I)
```

## 五、复杂度分析

### 5.1 理论复杂度

**最坏情况**：

```
输入：m个n变元d次多项式
输出：Gröbner基

复杂度：
- 时间：双重指数
- 空间：指数

具体：
O(m^(2^(n)))
```
**实际表现**：

```
大多数情况：
- 多项式时间
- 或单指数时间

但：
- 最坏情况确实双重指数
- 某些问题不可行
```
---

### 5.2 改进算法

**F4/F5算法**（Faugère, 1999-2002）：

```
改进：
- 线性代数方法
- 矩阵计算
- 更高效

但：
- 最坏情况仍双重指数
- 实际改进显著
```
