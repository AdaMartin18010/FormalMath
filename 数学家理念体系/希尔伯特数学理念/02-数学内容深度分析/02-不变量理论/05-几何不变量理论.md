## 一、GIT的起源

### 1.1 不变量理论的几何化

**19世纪问题**：

```
给定群G作用在代数簇X上

问题：
- 商空间X/G的结构？
- 不变量环的结构？
- 何时商空间是代数簇？
```

**困难**：

- 商空间可能不是代数簇
- 需要"好的"商
- 稳定性条件

## 二、基本概念

### 2.1 群作用

**线性作用**：

```
群G作用在向量空间V上：
ρ: G → GL(V)

诱导作用：
G × V → V
(g, v) → ρ(g)v
```

**例子：SL(n)作用**：

```
SL(n)作用在n×n矩阵上：
g·A = gAg⁻¹

不变量：
- 特征值
- 迹
- 行列式
```

---

### 2.2 不变量环

**定义**：

```
不变量环：
k[V]^G = {f∈k[V] | f(g·v) = f(v)对所有g∈G}

性质：
- 子代数
- 有限生成（Hilbert定理）
- 坐标环
```

**几何意义**：

```
不变量环 ↔ 商空间

Spec(k[V]^G) = V//G（范畴商）
```

## 三、稳定性

### 3.1 半稳定点

**定义**（Mumford）：

```
点v∈V是半稳定的 ⟺
存在不变量f使得f(v)≠0且f(0)=0

等价：
轨道闭包不包含0
```

**几何意义**：

- 半稳定点有"好"轨道
- 可构造商空间

---

### 3.2 稳定点

**定义**：

```
点v∈V是稳定的 ⟺
- v半稳定
- 轨道G·v闭
- 稳定化子有限
```

**性质**：

- 稳定点形成开集
- 商空间光滑（局部）

---

**问题**：

```
给定曲线C，构造：
- 向量丛的模空间
- 参数化所有向量丛
```

**GIT方法**：

```
步骤1：嵌入到Grassmannian
步骤2：用GIT构造商空间
步骤3：得到模空间M(r,d)

其中：
- r：秩
- d：度
```

构造：
