# 不变量理论与拓扑：从代数到拓扑的桥梁

## 一、拓扑不变量

### 1.1 基本概念

**拓扑不变量**：

```
拓扑空间X的不变量：
- 在同胚下不变
- 分类工具
- 几何性质

例子：
- Euler示性数
- Betti数
- 同调群
```
**与代数不变量**：

```
代数不变量（Hilbert）：
- 群作用下的不变量
- 代数结构

拓扑不变量：
- 连续变换下的不变量
- 拓扑结构

联系：
- 代数拓扑
- 几何方法
```
---

### 1.2 Euler示性数

**定义**：

```
多面体P：
χ(P) = V - E + F

其中：
- V：顶点数
- E：边数
- F：面数

性质：
- 拓扑不变量
- 与曲率关联（Gauss-Bonnet）
```
---

## 二、同调理论

### 2.1 单纯同调

**单纯复形**：

```
K = {单纯形}

链复形：
Cₖ → C_{k-1} → ...

边界算子：
∂: Cₖ → C_{k-1}

同调群：
Hₖ = ker(∂ₖ)/im(∂_{k+1})
```
**性质**：

- 拓扑不变量
- 可计算
- 分类工具

---

### 2.2 奇异同调

**定义**：

```
奇异单形：
σ: Δᵏ → X

奇异链：
Cₖ(X) = 自由Abel群

同调：
Hₖ(X) = ker(∂ₖ)/im(∂_{k+1})
```
**优势**：

- 任意拓扑空间
- 函子性
- 计算工具

---

## 三、上同调

### 3.1 定义

**上同调群**：

```
上链复形：
Cᵏ = Hom(Cₖ, ℤ)

上边界：
δ: Cᵏ → C^{k+1}

上同调：
Hᵏ(X) = ker(δᵏ)/im(δ^{k-1})
```
**性质**：

- 对偶于同调
- 有环结构
- 几何意义

---

### 3.2 上同调环

**杯积**：

```
∪: Hᵖ(X) × H^q(X) → H^{p+q}(X)

性质：
- 结合
- 交换（上同调）
- 单位元

应用：
- 分类空间
- 特征类
- 几何
```
---

## 四、特征类

### 4.1 Chern类

**向量丛**：

```
E → X

Chern类：
cᵢ(E) ∈ H²ⁱ(X)

性质：
- 拓扑不变量
- 乘法性
- 分类工具
```
**应用**：

- 代数几何
- 微分几何
- 物理

---

### 4.2 Stiefel-Whitney类

**实向量丛**：

```
wᵢ(E) ∈ Hⁱ(X; ℤ/2)

性质：
- 示性类
- 分类
- 应用
```

## 五、K理论

### 5.1 拓扑K理论

**Atiyah-Hirzebruch（1959）**：

```
K⁰(X) = 向量丛的Grothendieck群

K¹(X) = 稳定同伦群

性质：
- 广义上同调
- 周期2
- 计算工具
```
---

### 5.2 应用

**指标定理**：

```
椭圆算子D：
ind(D) = ∫ ch(D) · td(X)

K理论：
- 提供框架
- 计算工具
- 几何应用
```

## 六、量子不变量

```
链环L：
V_L(t) ∈ ℤ[t^(±1/2)]

性质：
- 拓扑不变量
- 量子群构造
- 物理起源
```
