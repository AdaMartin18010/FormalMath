## 一、组合不变量

### 1.1 图的不变量

**图的数值不变量**：

设 $G = (V, E)$ 是图（顶点集 $V$，边集 $E$）。

**基本不变量**：
- **顶点数**：$|V| = n$（阶数）
- **边数**：$|E| = m$
- **度序列**：$(d(v_1), d(v_2), \ldots, d(v_n))$（每个顶点的度数）

**色数**：
$$\chi(G) = \min\{k : G \text{ 可用 } k \text{ 种颜色着色}\}$$

**独立数**：
$$\alpha(G) = \max\{|S| : S \subseteq V, S \text{ 是独立集}\}$$

其中独立集 $S$ 满足：对任意 $u, v \in S$，$uv \notin E$。

**团数**：
$$\omega(G) = \max\{|C| : C \subseteq V, C \text{ 是团}\}$$

其中团 $C$ 满足：对任意 $u, v \in C$（$u \neqqqqqq v$），$uv \in E$。

**具体例子**：

**例1：完全图 $K_n$**

- $|V| = n$，$|E| = \binom{n}{2} = \frac{n(n-1)}{2}$
- $\chi(K_n) = n$（每个顶点需要不同颜色）
- $\alpha(K_n) = 1$（任意两个顶点相邻）
- $\omega(K_n) = n$（所有顶点形成团）

**例2：完全二部图 $K_{m,n}$**

- $|V| = m + n$，$|E| = mn$
- $\chi(K_{m,n}) = 2$（二部图可用2种颜色着色）
- $\alpha(K_{m,n}) = \max(m, n)$（取较大部分的顶点）
- $\omega(K_{m,n}) = 2$（最大团只有2个顶点）

**例3：圈 $C_n$**

- $|V| = n$，$|E| = n$
- $\chi(C_n) = \begin{cases} 2 & n \text{ 偶} \\ 3 & n \text{ 奇} \end{cases}$
- $\alpha(C_n) = \lfloor n/2 \rfloor$
- $\omega(C_n) = 2$（若 $n \\\geq 4$）

**多项式不变量**：

**色多项式**：
$$P_G(\lambda) = \text{用 } \lambda \text{ 种颜色给 } G \text{ 着色的方法数}$$

**递推关系**：
$$P_G(\lambda) = P_{G-e}(\lambda) - P_{G/e}(\lambda)$$

其中 $G-e$ 是删除边 $e$，$G/e$ 是收缩边 $e$。

**Tutte多项式**：
$$T_G(x, y) = \sum_{A \subseteq E} (x-1)^{r(E)-r(A)} (y-1)^{|A|-r(A)}$$

其中 $r(A)$ 是边集 $A$ 的秩（生成森林的边数）。

## 二、生成函数

**定义**：

```
序列{aₙ}的生成函数：
A(x) = ∑_{n≥0} aₙ xⁿ

性质：
- 加法 ↔ 生成函数加法
- 卷积 ↔ 生成函数乘法
- 递推 ↔ 函数方程
```

**应用**：

- 组合计数
- 递推关系
- 渐近分析

### 2.2 指数生成函数

**定义**：

```
A(x) = ∑_{n≥0} aₙ xⁿ/n!

应用：
- 排列问题
- 标签结构
- 组合结构
```

## 六、Ramsey理论

### 6.1 Ramsey定理

**Ramsey定理**（1930）：

> 对任意正整数 $r, s \\\geq 2$，存在正整数 $N = R(r, s)$ 使得：
>
> 对任意 $N$ 个顶点的完全图 $K_N$，用两种颜色（红、蓝）给边着色，则：
> - 要么存在 $r$ 个顶点的红色完全子图 $K_r$（所有边都是红色）
> - 要么存在 $s$ 个顶点的蓝色完全子图 $K_s$（所有边都是蓝色）

**Ramsey数**：
$$R(r, s) = \min\{N : \text{任意 } K_N \text{ 的2-着色都包含红色 } K_r \text{ 或蓝色 } K_s\}$$

**具体例子**：

**例1：$R(3, 3) = 6$**

- **含义**：任意6个人的完全图，用两种颜色给边着色（认识/不认识），则要么存在3个人互相认识，要么存在3个人互不认识
- **证明**：
  - $R(3, 3) > 5$：存在5个顶点的2-着色不包含单色三角形（五边形）
  - $R(3, 3) \\\leq 6$：对任意6个顶点的2-着色，必存在单色三角形（鸽笼原理）

**例2：$R(3, 4) = 9$**

- **含义**：任意9个顶点的完全图，2-着色后要么存在红色三角形，要么存在蓝色 $K_4$
- **已知值**：$R(3, 4) = 9$（Greenwood & Gleason, 1955）

**例3：$R(4, 4) = 18$**

- **含义**：任意18个顶点的完全图，2-着色后必存在单色 $K_4$
- **已知值**：$R(4, 4) = 18$（Greenwood & Gleason, 1955）

**一般Ramsey定理**：

对任意 $k$ 种颜色和任意 $r_1, r_2, \ldots, r_k \\\geq 2$，存在 $N = R(r_1, r_2, \ldots, r_k)$ 使得：
任意 $N$ 个顶点的完全图用 $k$ 种颜色着色，必存在某个 $i$ 使得存在 $r_i$ 个顶点的单色 $K_{r_i}$（第 $i$ 种颜色）。

**Ramsey数**：

- R(r,s) = 最小N
- 计算困难
- 应用广泛

## 九、数学公式总结

### 核心公式

1. **图的色数**：
   $$\chi(G) = \min\{k : G \text{ 可用 } k \text{ 种颜色着色}\}$$

2. **独立数**：
   $$\alpha(G) = \max\{|S| : S \text{ 是独立集}\}$$

3. **团数**：
   $$\omega(G) = \max\{|C| : C \text{ 是团}\}$$

4. **色多项式递推**：
   $$P_G(\lambda) = P_{G-e}(\lambda) - P_{G/e}(\lambda)$$

5. **Tutte多项式**：
   $$T_G(x, y) = \sum_{A \subseteq E} (x-1)^{r(E)-r(A)} (y-1)^{|A|-r(A)}$$

6. **普通生成函数**：
   $$A(x) = \sum_{n \\\geq 0} a_n x^n$$

7. **指数生成函数**：
   $$A(x) = \sum_{n \\\geq 0} a_n \frac{x^n}{n!}$$

8. **对称函数**：
   $$f(x_{\sigma(1)}, \ldots, x_{\sigma(n)}) = f(x_1, \ldots, x_n) \text{ 对所有 } \sigma \in S_n$$

9. **Ramsey数**：
   $$R(r, s) = \min\{N : \text{任意 } K_N \text{ 的2-着色都包含红色 } K_r \text{ 或蓝色 } K_s\}$$

10. **Ramsey定理（一般形式）**：
    $$R(r_1, r_2, \ldots, r_k) = \min\{N : \text{任意 } K_N \text{ 的 } k \text{-着色都包含某个单色 } K_{r_i}\}$$

11. **组合设计参数**：
    $t$-$(\nu, k, \lambda)$ 设计：$\nu$ 个元素，$k$ 元块，每 $t$ 元恰在 $\lambda$ 个块中

12. **线性规划对偶**：
    $$\max c^T x \text{ s.t. } Ax \\\leq b \quad \\\leqftrightarrow \quad \min b^T y \text{ s.t. } A^T y \\\geq c$$
