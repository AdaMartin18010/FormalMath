# 不变量理论与组合数学：从代数到离散

## 一、组合不变量

### 1.1 图的不变量

**图的数值不变量**：

```
G = (V, E)

不变量：
- 顶点数：|V|
- 边数：|E|
- 色数：χ(G)
- 独立数：α(G)
- 团数：ω(G)
```

**多项式不变量**：

```
色多项式：
P_G(λ) = 用λ种颜色着色的方法数

Tutte多项式：
T_G(x,y) = 更一般的不变量
```

---

### 1.2 与代数不变量的关系

**对应**：

```
代数不变量（Hilbert）：
- 群作用
- 多项式环
- 有限生成

组合不变量：
- 图的自同构

- 类似方法
```

## 二、生成函数

### 2.1 普通生成函数

**定义**：

```
序列{aₙ}的生成函数：
A(x) = ∑_{n≥0} aₙ xⁿ

性质：
- 加法 ↔ 生成函数加法
- 卷积 ↔ 生成函数乘法
- 递推 ↔ 函数方程
```

**应用**：

- 组合计数
- 递推关系
- 渐近分析

---

### 2.2 指数生成函数

**定义**：

```
A(x) = ∑_{n≥0} aₙ xⁿ/n!

应用：
- 排列问题
- 标签结构
- 组合结构
```

---

## 三、对称函数

### 3.1 定义

**对称函数**：

```
f(x₁,...,xₙ)对称 ⟺
f(x_{σ(1)},...,x_{σ(n)}) = f(x₁,...,xₙ)
（对所有置换σ）

例子：
- 初等对称函数
- 完全齐次对称函数
- Schur函数
```

---

### 3.2 与表示论

**对应**：

```
对称函数 ↔ 表示论

Schur函数 ↔ GL(n)的不可约表示
    ↓
组合 ↔ 代数
```

## 六、Ramsey理论

### 6.1 Ramsey定理

**定理**：

```
对任意r, s：
存在N使得：
任意N个顶点的完全图
用两种颜色着色边
→ 存在r个顶点的红色完全子图
或s个顶点的蓝色完全子图
```

**Ramsey数**：

- R(r,s) = 最小N
- 计算困难
- 应用广泛

---

### 6.2 应用

**组合结构**：

```
应用：
- 图论
- 数论
- 几何

例子：
- 任意6个人中，要么3人互相认识，要么3人互不认识
- R(3,3) = 6
```
