## 一、不变量理论的历史背景

### 1.1 19世纪的核心问题

**问题起源**：

```
代数方程的分类：
- 二次方程：ax²+bx+c=0（判别式Δ=b²-4ac）
- 三次方程：复杂的根式公式
- 高次方程：Galois理论

核心问题：
变量替换下的不变量
```

**Cayley-Sylvester理论**（1850s）：

- 不变量：坐标变换下不变的多项式
- 例：二元二次型的判别式

---

### 1.2 希尔伯特的介入（1880s）

**问题状态**：

- Gordan证明二元不变量有有限基（1868）
- 高维情况未解决
- 显式计算极其复杂

**希尔伯特的突破**：

- 基底定理（1888）：所有不变量有有限基
- 零点定理（1893）：理想与簇的对应
- 但方法**非构造性**

## 二、不变量的基本概念

### 2.1 线性群作用

**设定**：

```
V = kⁿ：向量空间
G ⊆ GL(n,k)：线性群
G作用于V：g·v = Av（A是g对应的矩阵）

例子：
- 正交群O(n)：保持长度
- 特殊线性群SL(n)：行列式=1
- 酉群U(n)：保持Hermite内积
```

---

### 2.2 不变多项式

**不变多项式的数学定义**：

设 $G \subseteq \text{GL}(n, k)$ 是线性群，$V = k^n$ 是 $G$ 的表示空间。多项式 $f \in k[x_1, \ldots, x_n]$ 是**$G$-不变的**，若：

$$\forall g \in G, \forall v \in V: f(g \cdot v) = f(v)$$

**等价表述**：
$$f(Ax) = f(x) \text{ 对所有 } A \in G$$

其中 $Ax$ 表示矩阵 $A$ 作用在向量 $x = (x_1, \ldots, x_n)^T$ 上。

**具体例子**：

**例1：$SL(2)$ 作用于二元二次型**

- **二次型**：$f(x, y) = ax^2 + 2bxy + cy^2$
- **$SL(2)$ 变换**：
  $$\begin{pmatrix} x' \\ y' \end{pmatrix} = \begin{pmatrix} \alpha & \beta \\ \gamma & \delta \end{pmatrix} \begin{pmatrix} x \\ y \end{pmatrix}$$
  其中 $\alpha\delta - \beta\gamma = 1$（行列式为1）

- **变换后的二次型**：
  $$f'(x', y') = a'x'^2 + 2b'x'y' + c'y'^2$$
  其中 $(a', b', c')$ 是 $(a, b, c)$ 在 $SL(2)$ 作用下的变换

- **不变量**：判别式
  $$I_2 = b^2 - ac$$

- **验证**：
  - 计算 $I_2(a', b', c')$（用 $(a, b, c)$ 和变换矩阵表示）
  - 通过直接计算可得：$I_2(a', b', c') = b^2 - ac = I_2(a, b, c)$
  - 因此 $I_2$ 是 $SL(2)$-不变量

**例2：$SO(2)$ 作用于二元二次型**

- $SO(2)$ 是旋转群：$\begin{pmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{pmatrix}$
- **不变量**：
  - $a + c$（迹）
  - $b^2 - ac$（判别式）
  - $a^2 + 2b^2 + c^2$（Frobenius范数的平方）

**例3：$GL(n)$ 作用于 $n \times n$ 矩阵**

- $V = M_n(k)$（$n \times n$ 矩阵空间）
- $GL(n)$ 共轭作用：$g \cdot A = gAg^{-1}$
- **不变量**：
  - **特征值**：$\lambda_1, \ldots, \lambda_n$（在共轭下不变）
  - **迹**：$\text{tr}(A) = \lambda_1 + \cdots + \lambda_n$
  - **行列式**：$\det(A) = \lambda_1 \cdots \lambda_n$
  - **特征多项式系数**：$\det(\lambda I - A) = \lambda^n - \text{tr}(A)\lambda^{n-1} + \cdots + (-1)^n\det(A)$

---

### 2.3 不变量环

**定义**：

```
k[V]^G = {f∈k[V] | f是G-不变的}

性质：
- k[V]^G是环（封闭于加法、乘法）
- k[V]^G是k[V]的子环
- k[V]^G是有限生成k-代数（希尔伯特基底定理）
```

## 三、希尔伯特基底定理的深化

### 3.1 代数表述

**定理（代数版本）**：
> 若k是域，G是可约代数群，则k[V]^G有限生成

**证明思路**：

```
步骤1：k[V]是Noether环（希尔伯特基底定理原始版）
步骤2：k[V]^G是k[V]的子环
步骤3：Noether环的子环可能不Noether
步骤4：但G可约→k[V]^G有限生成（深层代数事实）
```

---

### 3.2 Hilbert-Noether定理

**Hilbert-Noether定理**（Noether, 1916）：

> 若 $G$ 是有限群，$\text{char}(k)$ 不整除 $|G|$，则 $k[V]^G$ 有限生成。

**Reynolds算子**：

定义**Reynolds算子** $R: k[V] \to k[V]^G$：

$$R(f) = \frac{1}{|G|} \sum_{g \in G} g \cdot f$$

其中 $(g \cdot f)(v) = f(g^{-1} \cdot v)$。

**性质**：

1. **投影性**：$R(f) \in k[V]^G$（对所有 $f \in k[V]$）
   - **验证**：对任意 $h \in G$：
     $$h \cdot R(f) = \frac{1}{|G|} \sum_{g \in G} h \cdot (g \cdot f) = \frac{1}{|G|} \sum_{g \in G} (hg) \cdot f = R(f)$$
     （因为 $\{hg : g \in G\} = G$）

2. **幂等性**：$R^2 = R$
   - **验证**：$R(R(f)) = R(f)$（因为 $R(f) \in k[V]^G$）

3. **线性性**：$R(\lambda f + \mu g) = \lambda R(f) + \mu R(g)$

**具体例子**：

**例：$C_2 = \{1, \sigma\}$ 作用于 $k[x]$**

- $|C_2| = 2$，$\sigma(x) = -x$
- **Reynolds算子**：
  $$R(f)(x) = \frac{1}{2}(f(x) + f(-x))$$
- **不变量**：$k[x]^{C_2} = \{f \in k[x] : f(-x) = f(x)\}$（偶函数）
- **生成元**：$\{1, x^2, x^4, \ldots\}$，或更简洁地：$\{1, x^2\}$（若只考虑多项式）

**应用**：

- 利用 $R$ 可以从 $k[V]$ 的生成元构造 $k[V]^G$ 的生成元
- 若 $k[V] = k[f_1, \ldots, f_m]$，则 $k[V]^G = k[R(f_1), \ldots, R(f_m)]$（可能需要更多生成元）

---

### 3.3 现代观点：商簇

**几何解释**：

```
V：仿射空间
V/G：商空间（轨道空间）

k[V]^G = k[V/G]（商簇的坐标环）

希尔伯特基底定理 ⟺ V/G是仿射簇
```

## 四、具体例子

### 4.1 二元二次型（SL(2)作用）

**完整描述**：

```
f(x,y) = ax² + 2bxy + cy²
↕
(a,b,c)∈k³（系数空间）

基本不变量：
I₂ = b²-ac（度数2）
I₄ = (ac-b²)²（度数4，但实际是I₂²）

实际上：
k[a,b,c]^{SL(2)} = k[I₂]

Cayley-Salmon定理：
所有不变量都是I₂的多项式
```

**模的理论**：
I₄ = I₂²（不是独立生成元）

---

### 4.2 三元三次型（SL(3)作用）

**复杂度跃升**：

```
f(x,y,z) = ∑_{i+j+k=3} a_{ijk}x^i y^j z^k

系数空间：10维

基本不变量：
S（Aronhold不变量，度数4）
T（Hessian不变量，度数6）

关系：
108T² = S³（syzygy关系）

k[三次型]^{SL(3)} = k[S,T]/(108T²-S³)
```

**Gordan的功绩**：
显式计算S和T（极其困难）

---

### 4.3 四元四次型

**希尔伯特的动机**：

```
问题：基本不变量有几个？

Gordan方法：
- 显式计算
- 1880s的超级计算（手工）
- 找到很多不变量，但无法证明完备

希尔伯特方法：
- 非构造性证明有有限基
- 但不给出基的个数
- Gordan："神学！"

现代：
用Gröbner基算法可计算（计算机辅助）
```

## 五、Noether的抽象化

### 5.1 从计算到结构

**Gordan-Hilbert对比**：

| 方面 | Gordan | Hilbert | Noether |
|------|--------|---------|---------|
| 方法 | 显式构造 | 存在性证明 | 抽象结构 |
| 对象 | 多项式 | 多项式环 | 模 |
| 工具 | 符号计算 | 理想理论 | 同调代数 |

**Noether的革命**（1920s）：
> "不要研究**特殊的不变量多项式**，而要研究**不变量环的结构**。"

---

### 5.2 Noether正规化定理

**定理**：
> 若A是有限生成k-代数，则存在代数独立元y₁,...,yₐ使得
> A是k[y₁,...,yₐ]的整扩张

**应用于不变量**：

```
k[V]^G有限生成
    ↓ Noether正规化
存在代数独立不变量f₁,...,fₐ
使得k[V]^G在k[f₁,...,fₐ]上整

几何：
V/G → k^d（有限到一投影）
```

---

### 5.3 Molien公式

**计数不变量**：

**Molien定理**（1897，Noether改进）：

```
Hilbert级数：
H(t) = ∑_{n=0}^∞ dim(k[V]^G_n) t^n

公式：
H(t) = (1/|G|) ∑_{g∈G} 1/det(I-tg)

例子（Z/2作用于k²）：
H(t) = (1+t²)/(1-t²)(1-t)
生成元度数：1,2
```

## 八、数学公式总结

### 核心公式

1. **群作用**：
   $$g \cdot v = \rho(g)v \quad \text{（$g \in G$, $v \in V$）}$$

2. **不变多项式**：
   $$f(g \cdot v) = f(v) \text{ 对所有 } g \in G, v \in V$$

3. **不变量环**：
   $$k[V]^G = \{f \in k[V] : f(g \cdot v) = f(v) \text{ 对所有 } g \in G\}$$

4. **Reynolds算子**：
   $$R(f) = \frac{1}{|G|} \sum_{g \in G} g \cdot f$$

5. **判别式（二元二次型）**：
   $$I_2 = b^2 - ac \quad \text{（对 } f(x,y) = ax^2 + 2bxy + cy^2 \text{）}$$

6. **Noether正规化定理**：
   $$A \text{ 有限生成 } k\text{-代数 } \implies A \text{ 在 } k[y_1, \ldots, y_d] \text{ 上整}$$

7. **Molien公式**：
   $$H(t) = \frac{1}{|G|} \sum_{g \in G} \frac{1}{\det(I - t\rho(g))}$$

8. **Hilbert级数**：
   $$H(t) = \sum_{n=0}^\infty \dim(k[V]^G_n) t^n$$

9. **商簇坐标环**：
   $$k[V]^G = k[V/G]$$

10. **特征值不变量**：
    $$\text{tr}(A) = \lambda_1 + \cdots + \lambda_n, \quad \det(A) = \lambda_1 \cdots \lambda_n$$
