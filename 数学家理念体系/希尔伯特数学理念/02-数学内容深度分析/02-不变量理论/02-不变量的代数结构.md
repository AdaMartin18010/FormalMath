# 不变量的代数结构：从希尔伯特到Noether

## 一、不变量理论的历史背景

### 1.1 19世纪的核心问题

**问题起源**：

```
代数方程的分类：
- 二次方程：ax²+bx+c=0（判别式Δ=b²-4ac）
- 三次方程：复杂的根式公式
- 高次方程：Galois理论

核心问题：
变量替换下的不变量
```

**Cayley-Sylvester理论**（1850s）：

- 不变量：坐标变换下不变的多项式
- 例：二元二次型的判别式

---

### 1.2 希尔伯特的介入（1880s）

**问题状态**：

- Gordan证明二元不变量有有限基（1868）
- 高维情况未解决
- 显式计算极其复杂

**希尔伯特的突破**：

- 基底定理（1888）：所有不变量有有限基
- 零点定理（1893）：理想与簇的对应
- 但方法**非构造性**

---

## 二、不变量的基本概念

### 2.1 线性群作用

**设定**：

```
V = kⁿ：向量空间
G ⊆ GL(n,k)：线性群
G作用于V：g·v = Av（A是g对应的矩阵）

例子：
- 正交群O(n)：保持长度
- 特殊线性群SL(n)：行列式=1
- 酉群U(n)：保持Hermite内积
```

---

### 2.2 不变多项式

**定义**：
f∈k[x₁,...,xₙ]是G-不变的，若：

```
∀g∈G, ∀v∈V: f(g·v) = f(v)

等价：
f(Ax) = f(x)，对所有A∈G
```

**例子：SL(2)作用于二元二次型**

```
f(x,y) = ax² + 2bxy + cy²

SL(2)变换：[x'] = [α β][x]
           [y']   [γ δ][y]，αδ-βγ=1

不变量：
判别式I₂ = b²-ac

验证：
I₂(ax'²+2bx'y'+cy'²) = b²-ac（计算验证）
```

---

### 2.3 不变量环

**定义**：

```
k[V]^G = {f∈k[V] | f是G-不变的}

性质：
- k[V]^G是环（封闭于加法、乘法）
- k[V]^G是k[V]的子环
- k[V]^G是有限生成k-代数（希尔伯特基底定理）
```

---

## 三、希尔伯特基底定理的深化

### 3.1 代数表述

**定理（代数版本）**：
> 若k是域，G是可约代数群，则k[V]^G有限生成

**证明思路**：

```
步骤1：k[V]是Noether环（希尔伯特基底定理原始版）
步骤2：k[V]^G是k[V]的子环
步骤3：Noether环的子环可能不Noether
步骤4：但G可约→k[V]^G有限生成（深层代数事实）
```

---

### 3.2 Hilbert-Noether定理

**Noether的贡献**（1916）：

**定理**：
> 若G有限群，char(k)不整除|G|，则k[V]^G有限生成

**证明（Reynolds算子）**：

```
定义：R: k[V] → k[V]^G
R(f) = (1/|G|) ∑_{g∈G} g·f

性质：
- R是投影：R(f)∈k[V]^G
- R²=R
- R线性

应用：
利用R构造生成元
```

---

### 3.3 现代观点：商簇

**几何解释**：

```
V：仿射空间
V/G：商空间（轨道空间）

k[V]^G = k[V/G]（商簇的坐标环）

希尔伯特基底定理 ⟺ V/G是仿射簇
```

## 四、具体例子

### 4.1 二元二次型（SL(2)作用）

**完整描述**：

```
f(x,y) = ax² + 2bxy + cy²
↕
(a,b,c)∈k³（系数空间）

基本不变量：
I₂ = b²-ac（度数2）
I₄ = (ac-b²)²（度数4，但实际是I₂²）

实际上：
k[a,b,c]^{SL(2)} = k[I₂]

Cayley-Salmon定理：
所有不变量都是I₂的多项式
```

**模的理论**：
I₄ = I₂²（不是独立生成元）

---

### 4.2 三元三次型（SL(3)作用）

**复杂度跃升**：

```
f(x,y,z) = ∑_{i+j+k=3} a_{ijk}x^i y^j z^k

系数空间：10维

基本不变量：
S（Aronhold不变量，度数4）
T（Hessian不变量，度数6）

关系：
108T² = S³（syzygy关系）

k[三次型]^{SL(3)} = k[S,T]/(108T²-S³)
```

**Gordan的功绩**：
显式计算S和T（极其困难）

---

### 4.3 四元四次型

**希尔伯特的动机**：

```
问题：基本不变量有几个？

Gordan方法：
- 显式计算
- 1880s的超级计算（手工）
- 找到很多不变量，但无法证明完备

希尔伯特方法：
- 非构造性证明有有限基
- 但不给出基的个数
- Gordan："神学！"

现代：
用Gröbner基算法可计算（计算机辅助）
```

## 五、Noether的抽象化

### 5.1 从计算到结构

**Gordan-Hilbert对比**：

| 方面 | Gordan | Hilbert | Noether |
|------|--------|---------|---------|
| 方法 | 显式构造 | 存在性证明 | 抽象结构 |
| 对象 | 多项式 | 多项式环 | 模 |
| 工具 | 符号计算 | 理想理论 | 同调代数 |

**Noether的革命**（1920s）：
> "不要研究**特殊的不变量多项式**，而要研究**不变量环的结构**。"

---

### 5.2 Noether正规化定理

**定理**：
> 若A是有限生成k-代数，则存在代数独立元y₁,...,yₐ使得
> A是k[y₁,...,yₐ]的整扩张

**应用于不变量**：

```
k[V]^G有限生成
    ↓ Noether正规化
存在代数独立不变量f₁,...,fₐ
使得k[V]^G在k[f₁,...,fₐ]上整

几何：
V/G → k^d（有限到一投影）
```

---

### 5.3 Molien公式

**计数不变量**：

**Molien定理**（1897，Noether改进）：

```
Hilbert级数：
H(t) = ∑_{n=0}^∞ dim(k[V]^G_n) t^n

公式：
H(t) = (1/|G|) ∑_{g∈G} 1/det(I-tg)

例子（Z/2作用于k²）：
H(t) = (1+t²)/(1-t²)(1-t)
生成元度数：1,2
```

## 六、现代发展

### 6.1 几何不变量理论（GIT）

**Mumford（1965）**：

```
问题：对无限群（如SL(n)），如何定义商？

GIT：
- 稳定点、半稳定点概念
- 商V//G（GIT商）
- 模空间理论

应用：
代数几何的模空间（如曲线模空间M_g）
```

---

### 6.2 拓扑不变量

**Chern类、特征类**：

```
向量丛E → X
Chern类c_i(E) ∈ H^{2i}(X,ℤ)

性质：
- 坐标变换不变
- 拓扑不变量

联系不变量理论：
Chern多项式是GL(n)-不变量
```

---

### 6.3 量子不变量

**Jones多项式**（1984）：

```
纽结K → Jones多项式V_K(t)

性质：
- 环境同痂不变量
- 量子群表示论构造
- 继承不变量理论思想
```
