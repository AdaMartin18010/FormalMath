## 一、有界算子的代数

### 1.1 算子代数

**B(H)**（有界算子代数）：

```
B(H) = {T: H → H | T有界}

运算：
- 加法：(T+S)x = Tx + Sx
- 乘法：(TS)x = T(Sx)
- 共轭：T*（伴随）

性质：
- Banach代数
- C*-代数
- von Neumann代数（弱闭）
```

**例子**：

```
有限维：
B(ℂⁿ) = Mₙ(ℂ)（n×n矩阵）

无限维：
B(ℓ²) = 有界序列算子
```

---

### 1.2 C*-代数

**C*-代数的数学定义**：

C*-代数是满足以下条件的Banach代数 $A$：

1. **对合（Involution）**：存在映射 $*: A \to A$ 满足：
   - $(a^*)^* = a$（对合性）
   - $(ab)^* = b^*a^*$（反同态性）
   - $(\lambda a + \mu b)^* = \overline{\lambda}a^* + \overline{\mu}b^*$（共轭线性）

2. **C*-恒等式**：
   $$\|a^*a\| = \|a\|^2$$
   对所有 $a \in A$ 成立。

**具体例子**：

**例1：有界算子代数 $B(H)$**
- $A = B(H)$（希尔伯特空间 $H$ 上的有界算子）
- 对合：$T^*$ 是 $T$ 的伴随算子
- C*-恒等式：$\|T^*T\| = \|T\|^2$（这是算子理论的基本事实）

**例2：连续函数代数 $C(X)$**
- $X$ 是紧Hausdorff空间
- $A = C(X)$（$X$ 上的连续复值函数）
- 对合：$f^*(x) = \overline{f(x)}$（复共轭）
- C*-恒等式：$\|f^*f\| = \|f\|^2$（因为 $f^*f = |f|^2$）

**例3：矩阵代数 $M_n(\mathbb{C})$**
- $A = M_n(\mathbb{C})$（$n \times n$ 复矩阵）
- 对合：$A^* = \overline{A}^T$（共轭转置）
- C*-恒等式：$\|A^*A\| = \|A\|^2$（矩阵范数）

**C*-代数的特殊元素**：

- **自伴元**：$a = a^*$（对应自伴算子）
- **正元**：$a = b^*b$ 对某个 $b \in A$（对应正算子）
- **投影**：$p = p^* = p^2$（对应投影算子）
- **酉元**：$u^*u = uu^* = 1$（对应酉算子）

**Gelfand-Naimark定理**（1943）：

> 任意C*-代数 $A$ 同构于某个希尔伯特空间 $H$ 上的算子代数 $B(H)$ 的闭*-子代数。
>
> 特别地，若 $A$ 是交换的，则 $A \cong C(X)$（某个紧Hausdorff空间 $X$ 上的连续函数代数）。

**具体例子**：

**例：交换C*-代数**

设 $A = C([0,1])$（$[0,1]$ 上的连续函数）：
- $A$ 是交换C*-代数
- 根据Gelfand-Naimark，$A \cong C(\sigma(A))$
- 其中 $\sigma(A)$ 是 $A$ 的谱空间（极大理想空间）
- 实际上，$\sigma(A) \cong [0,1]$（通过点值泛函）

## 二、von Neumann代数

### 2.1 定义

**von Neumann代数的数学定义**：

设 $M \subseteq B(H)$ 是 $B(H)$ 的子集，则 $M$ 是von Neumann代数，若：

1. ***-子代数**：$M$ 是*-子代数（对加法、乘法、数乘、对合封闭）
2. **包含恒等**：$I \in M$
3. **弱闭性**：$M$ 在弱算子拓扑下闭

**双换位定理**（von Neumann, 1929）：

> $M$ 是von Neumann代数当且仅当 $M = M''$（双换位子），其中：
> - $M' = \{T \in B(H) : TS = ST \text{ 对所有 } S \in M\}$（换位子）
> - $M'' = (M')'$（双换位子）

**具体例子**：

**例1：$B(H)$（最大von Neumann代数）**
- $M = B(H)$（所有有界算子）
- 换位子：$B(H)' = \mathbb{C}I$（只有标量算子与所有算子交换）
- 双换位子：$B(H)'' = B(H)$
- 性质：最大的von Neumann代数

**例2：$\mathbb{C}I$（最小von Neumann代数）**
- $M = \mathbb{C}I = \{\lambda I : \lambda \in \mathbb{C}\}$（标量算子）
- 换位子：$(\mathbb{C}I)' = B(H)$（所有算子与标量算子交换）
- 双换位子：$(\mathbb{C}I)'' = \mathbb{C}I$
- 性质：最小的von Neumann代数

**例3：对角算子代数**

在 $H = \ell^2$ 上，定义：
$$M = \{T \in B(\ell^2) : T \text{ 是对角算子}\}$$

即 $T$ 在标准基 $\{e_n\}$ 下是对角矩阵。

- $M$ 是von Neumann代数
- 换位子：$M' = M$（对角算子与对角算子交换）
- 双换位子：$M'' = M$
- 性质：交换von Neumann代数

---

### 2.2 因子

**中心**：

```
Z(M) = {a ∈ M | ab = ba对所有b ∈ M}

因子：
M是因子 ⟺ Z(M) = ℂI
```

**分类**（Murray-von Neumann）：

```
I型：有极小投影
II型：无极小投影，有迹
III型：无迹
```

## 三、Gelfand理论

### 3.1 交换C*-代数

**Gelfand变换**：

```
交换C*-代数A

谱：
σ(A) = {极大理想} = {非零*同态χ: A → ℂ}

Gelfand变换：
a → â: σ(A) → ℂ
â(χ) = χ(a)

定理：
A ≅ C(σ(A))
```

---

### 3.2 应用

**函数演算**：

```
给定：
- 自伴算子T
- 函数f: σ(T) → ℂ

定义：
f(T) = Gelfand逆变换(f(â))

性质：
- (fg)(T) = f(T)g(T)
- f(T)* = f̄(T)
- ||f(T)|| = sup|f(λ)|
```

C*-代数A

K₀(A) = {投影的Grothendieck群}

- 酉元：u*u = uu* = 1

## 五、应用：指标理论

### 5.1 Atiyah-Singer指标定理

**椭圆算子的数学定义**：

设 $X$ 是紧流形，$E, F$ 是 $X$ 上的向量丛，$D: \Gamma(E) \to \Gamma(F)$ 是椭圆微分算子。

**指标（Index）**：
$$\text{ind}(D) = \dim \ker(D) - \dim \text{coker}(D)$$

其中：
- $\ker(D) = \{s \in \Gamma(E) : Ds = 0\}$（核空间）
- $\text{coker}(D) = \Gamma(F)/\text{Im}(D)$（余核空间）

**关键性质**：
- 指标是有限整数（因为 $D$ 椭圆）
- 指标是拓扑不变量（在同伦下不变）

**Atiyah-Singer指标定理**（1963）：

> 椭圆算子 $D$ 的指标等于拓扑量：
> $$\text{ind}(D) = \int_X \text{ch}(D) \cdot \text{td}(X)$$
>
> 其中：
> - $\text{ch}(D)$ 是 $D$ 的Chern特征类
> - $\text{td}(X)$ 是 $X$ 的Todd类
> - 积分在 $X$ 的顶维上同调类上

**具体例子**：

**例1：Dirac算子**

在紧Riemann流形 $X$ 上：
- $D$ 是Dirac算子
- $\text{ind}(D) = \int_X \hat{A}(X)$（$\hat{A}$-类）
- 应用：证明某些流形上不存在正数量曲率度量

**例2：Dolbeault算子**

在紧复流形 $X$ 上：
- $D = \bar{\partial}$（Dolbeault算子）
- $\text{ind}(\bar{\partial}) = \chi(X, \mathcal{O})$（Euler特征）
- 应用：Riemann-Roch定理

**例3：符号算子**

在 $S^1$ 上：
- $D = -i\frac{d}{dx}$（导数算子）
- $\ker(D) = \mathbb{C}$（常数函数）
- $\text{coker}(D) = 0$
- $\text{ind}(D) = 1$

## 九、数学公式总结

### 核心公式

1. **C*-恒等式**：
   $$\|a^*a\| = \|a\|^2$$

2. **对合性质**：
   - $(a^*)^* = a$
   - $(ab)^* = b^*a^*$
   - $(\lambda a + \mu b)^* = \overline{\lambda}a^* + \overline{\mu}b^*$

3. **von Neumann双换位定理**：
   $$M = M''$$

4. **Gelfand-Naimark定理**：
   $$A \cong C(\sigma(A)) \quad \text{（交换C*-代数）}$$

5. **K₀群定义**：
   $$K_0(A) = \{[p] - [q] : p, q \text{ 是投影}\}$$

6. **Atiyah-Singer指标定理**：
   $$\text{ind}(D) = \int_X \text{ch}(D) \cdot \text{td}(X)$$

7. **椭圆算子指标**：
   $$\text{ind}(D) = \dim \ker(D) - \dim \text{coker}(D)$$

8. **投影性质**：
   $$p = p^* = p^2$$

9. **酉元性质**：
   $$u^*u = uu^* = 1$$

10. **正元定义**：
    $$a = b^*b \quad \text{对某个 } b \in A$$
