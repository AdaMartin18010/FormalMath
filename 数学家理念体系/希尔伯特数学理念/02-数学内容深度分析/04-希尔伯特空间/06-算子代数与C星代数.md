# 算子代数与C*-代数：希尔伯特空间的代数结构

## 一、有界算子的代数

### 1.1 算子代数

**B(H)**（有界算子代数）：

```
B(H) = {T: H → H | T有界}

运算：
- 加法：(T+S)x = Tx + Sx
- 乘法：(TS)x = T(Sx)
- 共轭：T*（伴随）

性质：
- Banach代数
- C*-代数
- von Neumann代数（弱闭）
```
**例子**：

```
有限维：
B(ℂⁿ) = Mₙ(ℂ)（n×n矩阵）

无限维：
B(ℓ²) = 有界序列算子
```
---

### 1.2 C*-代数

**定义**：

```
C*-代数是Banach代数A满足：
1. 有对合*: A → A
2. C*-恒等式：||a*a|| = ||a||²

性质：
- 自伴元：a = a*
- 正元：a = b*b（某个b）
- 投影：p = p* = p²
```
**Gelfand-Naimark定理**：

```
任意C*-代数同构于：
- 某个Hilbert空间上的算子代数
- 或交换C*-代数（函数代数）
```
---

## 二、von Neumann代数

### 2.1 定义

**von Neumann代数**：

```
M ⊆ B(H)是von Neumann代数 ⟺
1. M是*子代数
2. M包含恒等算子
3. M弱闭（或强闭）

等价：
M = M''（双换位子）
```
**例子**：

```
B(H)：
- 最大的von Neumann代数
- 所有有界算子

ℂI：
- 最小的
- 标量算子
```
---

### 2.2 因子

**中心**：

```
Z(M) = {a ∈ M | ab = ba对所有b ∈ M}

因子：
M是因子 ⟺ Z(M) = ℂI
```
**分类**（Murray-von Neumann）：

```
I型：有极小投影
II型：无极小投影，有迹
III型：无迹
```
---

## 三、Gelfand理论

### 3.1 交换C*-代数

**Gelfand变换**：

```
交换C*-代数A

谱：
σ(A) = {极大理想} = {非零*同态χ: A → ℂ}

Gelfand变换：
a → â: σ(A) → ℂ
â(χ) = χ(a)

定理：
A ≅ C(σ(A))
```
---

### 3.2 应用

**函数演算**：

```
给定：
- 自伴算子T
- 函数f: σ(T) → ℂ

定义：
f(T) = Gelfand逆变换(f(â))

性质：
- (fg)(T) = f(T)g(T)
- f(T)* = f̄(T)
- ||f(T)|| = sup|f(λ)|
```
---

## 四、K理论

### 4.1 向量丛的K群

**定义**：

```
紧空间X

K⁰(X) = {向量丛的Grothendieck群}

运算：
- 直和：[E] + [F] = [E⊕F]
- 张量积：[E] · [F] = [E⊗F]
```
**性质**：

- 拓扑不变量
- 与上同调关联
- 计算工具

---

### 4.2 算子代数的K理论

**K₀群**：

```
C*-代数A

K₀(A) = {投影的Grothendieck群}

定义：
- 投影：p = p* = p²
- 等价：p ~ q若存在u使得p = u*u, q = uu*
- 群：K₀(A)
```
**K₁群**：

```
K₁(A) = {酉元的群}

定义：
- 酉元：u*u = uu* = 1
- 同伦类
- Abel群
```
---

## 五、应用：指标理论

### 5.1 Atiyah-Singer指标定理

**椭圆算子**：

```
D: Γ(E) → Γ(F)

其中：
- E, F：向量丛
- D：椭圆微分算子

指标：
ind(D) = dim ker(D) - dim coker(D)
```
**Atiyah-Singer定理**：

```
指标 = 拓扑量

公式：
ind(D) = ∫ ch(D) · td(X)
```
---

### 5.2 应用

**几何**：

- 符号计算
- 拓扑不变量
- 几何结构

**物理**：

- 量子场论
- 规范理论
- 弦理论

## 六、非交换几何

### 6.1 Connes的纲领

**非交换几何**（Connes, 1990s）：

```
交换几何：
- 空间X ↔ C(X)（函数代数）
- 几何 ↔ 代数

非交换：
- "非交换空间" ↔ 非交换C*-代数
- 几何在代数中
```
**例子**：

```
非交换环面：
- C*-代数A_θ
- 参数θ（无理数）
- 几何结构
```
---

### 6.2 应用

**数论**：

- Langlands纲领
- 类域论
- L函数

**物理**：

- 量子场论
- 标准模型
- 弦理论
