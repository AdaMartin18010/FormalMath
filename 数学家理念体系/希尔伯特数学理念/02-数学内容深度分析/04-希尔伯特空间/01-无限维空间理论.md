# 希尔伯特空间：无限维的几何革命

## 一、希尔伯特空间的诞生

### 1.1 从欧氏空间到无限维

**经典几何空间**：

- R²（平面）、R³（三维空间）
- 有限维内积空间
- 几何直观清晰

**积分方程的挑战**（1900s）：

```
Fredholm积分方程：
f(x) = λ∫K(x,y)f(y)dy

问题：解空间是什么？
答案：无限维函数空间
```
**希尔伯特的洞见**：
> "函数空间可视为**无限维几何空间**，内积定义几何结构。"

---

### 1.2 完备内积空间的公理化

**定义**：
希尔伯特空间H是满足以下条件的复向量空间：

```
1. 内积：⟨·,·⟩: H×H → ℂ
   - 线性：⟨αu+βv, w⟩ = α⟨u,w⟩ + β⟨v,w⟩
   - 共轭对称：⟨u,v⟩ = ⟨v,u⟩*
   - 正定性：⟨u,u⟩ ≥ 0，等号仅当u=0

2. 范数：‖u‖ = √⟨u,u⟩

3. 完备性：任意Cauchy列收敛
   若‖uₙ - uₘ‖→0，则∃u∈H: uₙ→u
```
**关键特征**：可以是**无限维**！

## 二、典型例子

### 2.1 序列空间ℓ²

**定义**：

```
ℓ² = {(a₁, a₂, ...) | ∑|aₙ|² < ∞}

内积：⟨a,b⟩ = ∑aₙb̄ₙ

基底：标准基{eₙ}
e₁ = (1,0,0,...)
e₂ = (0,1,0,...)
...

性质：
- 可数维
- 可分（有可数稠密子集）
- 完备
```
---

### 2.2 函数空间L²[0,1]

**定义**：

```
L²[0,1] = {f: [0,1]→ℂ | ∫₀¹|f(x)|²dx < ∞}

内积：⟨f,g⟩ = ∫₀¹f(x)ḡ(x)dx

基底：傅里叶基
{1, cos(2πnx), sin(2πnx)}ₙ₌₁^∞

或：{e^{2πinx}}ₙ∈ℤ

性质：
- 不可数维（作为向量空间）
- 但有可数正交基
```
**Riesz-Fischer定理**：
> L²与ℓ²同构（通过傅里叶系数）

---

### 2.3 算子空间B(H)

**定义**：

```
B(H) = {A: H→H | A有界线性}

范数：‖A‖ = sup{‖Ax‖: ‖x‖=1}

特殊类：
- 自伴算子：A* = A
- 酉算子：U*U = I
- 紧算子：单位球的像可预紧
- 投影算子：P² = P = P*
```
---

## 三、核心定理

### 3.1 正交投影定理

**定理**：
> 设M是H的闭子空间，则
> H = M ⊕ M^⊥
> 任意u∈H唯一分解为u = m + m^⊥

**应用**：

- 最佳逼近：m是u在M中的最佳逼近
- Fourier展开的理论基础
- 量子力学的测量理论

---

### 3.2 Riesz表示定理

**定理**：
> H上的连续线性泛函f: H→ℂ
> 唯一对应于某个v∈H:
> f(u) = ⟨u,v⟩

**意义**：

- 对偶空间H* ≅ H
- "协变"与"逆变"统一
- 量子力学的bra-ket记号基础

---

### 3.3 谱定理（紧自伴算子）

**定理**：
> 设A:H→H紧自伴，则：
>
> 1. A有可数多个实特征值{λₙ}
> 2. λₙ→0
> 3. 特征向量{vₙ}形成H的正交基
> 4. A = ∑λₙ⟨·,vₙ⟩vₙ

**无限维的对角化**！

**应用**：

- 积分方程求解
- 偏微分方程（Sturm-Liouville理论）
- 量子力学的可观测量

---

## 四、量子力学的数学基础

### 4.1 状态空间

**von Neumann公理化**（1932）：

```
量子系统 = 希尔伯特空间H

状态：|ψ⟩∈H，‖ψ‖=1
观测量：自伴算子A
测量结果：A的特征值λ
概率：P(λ) = |⟨ψ,vλ⟩|²
```
**例子：自旋½粒子**

```
H = ℂ²
基态：|↑⟩ = (1,0), |↓⟩ = (0,1)
一般态：|ψ⟩ = α|↑⟩ + β|↓⟩，|α|²+|β|²=1

Pauli矩阵（自伴）：
σz = [1  0]  σx = [0  1]  σy = [0 -i]
     [0 -1]       [1  0]       [i  0]
```
---

### 4.2 测量与坍缩

**Born规则**：

```
测量前：|ψ⟩ = α|↑⟩ + β|↓⟩
测量σz：
- 结果+1，概率|α|²，态变为|↑⟩
- 结果-1，概率|β|²，态变为|↓⟩

数学：正交投影
Pλ = |vλ⟩⟨vλ|（投影到特征子空间）
```
---

### 4.3 纠缠态

**EPR态**：

```
|Ψ⟩ = (|↑↓⟩ - |↓↑⟩)/√2 ∈ ℂ² ⊗ ℂ²

性质：
- 不可分解为|ψ₁⟩⊗|ψ₂⟩
- 测量一粒子瞬间影响另一粒子
- Bell不等式违反

数学本质：
希尔伯特空间的张量积结构
```

## 五、泛函分析的基石

### 5.1 有界算子理论

**核心结果**：

| 定理 | 陈述 | 意义 |
|------|------|------|
| **一致有界性** | 点态有界→一致有界 | 泛函分析基本工具 |
| **开映射定理** | 满射连续→开映射 | 逆算子连续 |
| **闭图像定理** | 图像闭→算子连续 | 判别连续性 |
| **Hahn-Banach** | 泛函延拓保持范数 | 对偶理论 |

---

### 5.2 紧算子与Fredholm理论

**紧算子**：

```
定义：A紧⟺单位球B的像Ā(B)可预紧

例子：
积分算子：(Kf)(x) = ∫K(x,y)f(y)dy
性质：K紧（若核连续）

Fredholm择一定理：
(I-λK)f = g
要么有唯一解（任意g）
要么齐次方程(I-λK)f=0有非零解
```
---

### 5.3 无界算子与自伴扩张

**量子力学的算子**：

```
位置算子：(Xψ)(x) = xψ(x)
动量算子：(Pψ)(x) = -iℏψ'(x)

问题：X, P无界（非B(H)元素）

解决：
定义域：D(X) = {ψ | ∫|xψ|² < ∞} ⊂ H
自伴扩张理论（von Neumann）
```

## 七、现代推广

### 7.1 Banach空间

推广：去掉内积，仅保留完备范数

例子：

- Lᵖ空间（p≠2）
- C[0,1]（连续函数，最大值范数）

---

### 7.2 算子代数（von Neumann代数）

**定义**：

```
M ⊆ B(H)是von Neumann代数若：
1. M是*-代数（对伴随封闭）
2. M在弱算子拓扑下闭

例子：
- B(H)本身
- 交换von Neumann代数 ≅ L^∞(X)
- 因子（不可约）
```
**应用**：

- 量子场论
- 非交换几何
- Connes的标准模型重构

---

### 7.3 Hilbert C*-模

**格洛腾迪克式推广**：

```
从标量值内积到C*-代数值内积

例子：
E是C*-代数A上的Hilbert模
⟨·,·⟩: E×E → A（取值于A）

应用：
- K理论
- 非交换微分几何
- 指标定理
```
