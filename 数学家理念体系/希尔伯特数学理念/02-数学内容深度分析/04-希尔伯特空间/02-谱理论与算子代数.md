## 一、有界算子的谱

### 1.1 谱的定义

**设定**：
$H$ 是希尔伯特空间，$A: H \to H$ 是有界线性算子。

**谱的数学定义**：

$$\sigma(A) = \{\lambda \in \mathbb{C} : A - \lambda I \text{ 不可逆}\}$$

**谱的分解**：

1. **点谱（Point Spectrum）**：
   $$\sigma_p(A) = \{\lambda \in \mathbb{C} : A - \lambda I \text{ 非单射}\} = \{\lambda : \exists v \neqqqqqq 0, Av = \lambda v\}$$
   即特征值的集合。

2. **连续谱（Continuous Spectrum）**：
   $$\sigma_c(A) = \{\lambda \in \mathbb{C} : A - \lambda I \text{ 单射但非满射，且像稠密}\}$$

3. **剩余谱（Residual Spectrum）**：
   $$\sigma_r(A) = \{\lambda \in \mathbb{C} : A - \lambda I \text{ 单射但像不稠密}\}$$
   在有限维中不存在剩余谱。

**关系**：
$$\sigma(A) = \sigma_p(A) \cup \sigma_c(A) \cup \sigma_r(A)$$
且这三个集合互不相交。

**具体例子**：

**例1：乘法算子**
- 定义 $M: L^2[0,1] \to L^2[0,1]$ 为 $(Mf)(x) = xf(x)$
- 对任意 $\lambda \in [0,1]$，$(M - \lambda I)f = (x - \lambda)f$
- 若 $\lambda \in [0,1]$，则 $(x - \lambda)$ 在 $[0,1]$ 上不恒为零，但 $(M - \lambda I)$ 的像不稠密
- **结论**：$\sigma(M) = [0,1]$（连续谱），$\sigma_p(M) = \emptyset$

**例2：有限维矩阵**
- $A = \begin{pmatrix} 1 & 2 \\ 0 & 3 \end{pmatrix}$ 在 $\mathbb{C}^2$ 上
- 特征值：$\lambda_1 = 1$，$\lambda_2 = 3$
- **结论**：$\sigma(A) = \sigma_p(A) = \{1, 3\}$，$\sigma_c(A) = \sigma_r(A) = \emptyset$

**例3：移位算子**
- 定义 $S: \ell^2 \to \ell^2$ 为 $S(a_1, a_2, a_3, \ldots) = (0, a_1, a_2, \ldots)$
- $S$ 是等距（$\|Sx\| = \|x\|$），所以 $\sigma(S) \subseteq \{|\lambda| = 1\}$
- **结论**：$\sigma(S) = \{\lambda : |\lambda| = 1\}$（连续谱）

---

### 1.2 谱的基本性质

**定理**：

```
1. σ(A)是紧集（有界闭集）
2. σ(A) ⊆ {λ | |λ| ≤ ||A||}
3. 若A自伴，则σ(A) ⊆ ℝ
4. 若A酉，则σ(A) ⊆ {λ | |λ|=1}
```

**证明思路**：

- 利用Neumann级数展开(A-λI)⁻¹
- 解析函数理论

## 二、自伴算子的谱定理

### 2.1 有限维情况（矩阵）

**谱定理（有限维）**：
> 若A是n×n自伴矩阵，则：
>
> 1. A有n个实特征值λ₁,...,λₙ
> 2. 存在正交基{v₁,...,vₙ}使得Avᵢ = λᵢvᵢ
> 3. A = ∑λᵢPᵢ（Pᵢ是到vᵢ的投影）

**对角化**：

```
A = UΛU*
U：酉矩阵
Λ：对角矩阵（特征值）
```

---

### 2.2 无限维：紧自伴算子

**定理（Hilbert-Schmidt谱定理）**：
> 若 $A: H \to H$ 是紧自伴算子，则：
>
> 1. **纯点谱**：$\sigma(A) = \sigma_p(A)$（只有特征值，无连续谱）
> 2. **特征值性质**：特征值可数，记为 $\{\lambda_n\}_{n=1}^{\infty}$，且 $\lambda_n \to 0$（当 $n \to \infty$）
> 3. **正交基**：对应的特征向量 $\{v_n\}_{n=1}^{\infty}$ 形成 $H$ 的标准正交基
> 4. **谱分解**：
>    $$A = \sum_{n=1}^{\infty} \lambda_n \langle \cdot, v_n \rangle v_n$$
>    对所有 $u \in H$ 成立。

**数学表述**：

对任意 $u \in H$：
$$Au = \sum_{n=1}^{\infty} \lambda_n \langle u, v_n \rangle v_n$$

且级数在算子范数下收敛。

**具体例子**：

**例1：积分算子（对称核）**

定义 $K: L^2[0,1] \to L^2[0,1]$ 为：
$$(Kf)(x) = \int_0^1 K(x,y) f(y) \, dy$$

其中核函数 $K(x,y)$ 满足：
- **对称性**：$K(x,y) = K(y,x)$（自伴性）
- **连续性**：$K \in C([0,1]^2)$（紧性）

**性质**：
- $K$ 是紧自伴算子
- 有可数特征值 $\{\lambda_n\}_{n=1}^{\infty}$，$\lambda_n \to 0$
- 特征函数 $\{v_n(x)\}_{n=1}^{\infty}$ 形成 $L^2[0,1]$ 的标准正交基
- 谱分解：$(Kf)(x) = \sum_{n=1}^{\infty} \lambda_n \langle f, v_n \rangle v_n(x)$

**例2：具体核函数**

取 $K(x,y) = \min(x,y)$（Green函数）：
- 对称：$\min(x,y) = \min(y,x)$
- 连续：显然
- 特征值：$\lambda_n = \frac{1}{(n\pi)^2}$，$n = 1, 2, 3, \ldots$
- 特征函数：$v_n(x) = \sqrt{2} \sin(n\pi x)$

**验证**：
$$\int_0^1 \min(x,y) \sqrt{2} \sin(n\pi y) \, dy = \frac{1}{(n\pi)^2} \sqrt{2} \sin(n\pi x)$$

**例3：有限秩算子**

定义 $A: \ell^2 \to \ell^2$ 为：
$$A(a_1, a_2, a_3, \ldots) = (a_1, 0, 0, \ldots)$$

- $A$ 是有限秩（秩为1），因此是紧算子
- $A$ 是自伴的（因为 $A^* = A$）
- 特征值：$\lambda_1 = 1$（对应 $e_1 = (1, 0, 0, \ldots)$），$\lambda_n = 0$（$n \geqqqqqq 2$）
- 谱分解：$A = 1 \cdot \langle \cdot, e_1 \rangle e_1$

### 2.3 一般自伴算子：投影值测度

**谱定理（von Neumann, 1929）**：

**投影值测度**：

```
E: Borel集B(ℝ) → 投影算子E(B)

性质：
- E(∅) = 0, E(ℝ) = I
- E(B₁∩B₂) = E(B₁)E(B₂)
- E(∪Bᵢ) = ∑E(Bᵢ)（互不相交）
```

**谱积分**：

```
A = ∫ λ dE(λ)

意义：
A的"对角化"（在连续谱情况）
```

**函数演算**：

```
f(A) = ∫ f(λ) dE(λ)

例子：
e^(itA) = ∫ e^(itλ) dE(λ)
√A = ∫ √λ dE(λ)（若A≥0）
```

## 三、算子代数

### 3.1 C*-代数

**定义**：

```
C*-代数A是Banach代数，带有对合*，满足：
1. (a*)* = a
2. (ab)* = b*a*
3. ||a*a|| = ||a||²（C*-恒等式）

例子：
- B(H)：有界算子（A* = 伴随）
- C(X)：连续函数（f* = 共轭）
```

**Gelfand-Naimark定理**：
> 交换C*-代数同构于C(X)（某紧Hausdorff空间X）

---

### 3.2 von Neumann代数

**定义**：

```
M ⊆ B(H)是von Neumann代数，若：
1. M是*-代数
2. M在弱算子拓扑下闭

等价：
M = M''（双换位定理）
```

**分类**：

```
类型I：有极小投影
类型II：无极小投影，但有有限投影
类型III：所有投影都无限

因子：中心平凡（Z(M) = ℂI）
```

### 3.3 应用：量子力学

**可观测量与自伴算子**：

在量子力学中，物理可观测量对应希尔伯特空间上的自伴算子。

**Born规则（测量概率）**：

设系统处于状态 $|\psi \rangle \in H$（$\|\psi\| = 1$），可观测量 $A$ 是自伴算子，则：

- **测量结果**：$A$ 的特征值 $\lambda$（离散谱）或 $\lambda \in \sigma(A)$（连续谱）
- **概率密度**（离散谱）：
  $$P(\lambda_n) = |\langle \psi, v_n \rangle|^2$$
  其中 $v_n$ 是对应特征值 $\lambda_n$ 的归一化特征向量。

- **期望值**：
  $$\langle A \rangle = \langle \psi, A \psi \rangle = \sum_{n=1}^{\infty} \lambda_n |\langle \psi, v_n \rangle|^2$$

**具体例子**：

**例1：自旋测量**

- 系统：自旋 $\frac{1}{2}$ 粒子，$H = \mathbb{C}^2$
- 可观测量：$S_z = \frac{\hbar}{2} \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$（$z$ 方向自旋）
- 特征值：$\lambda_1 = \frac{\hbar}{2}$（上），$\lambda_2 = -\frac{\hbar}{2}$（下）
- 特征向量：$v_1 = (1, 0)$，$v_2 = (0, 1)$
- 状态：$|\psi \rangle = \frac{1}{\sqrt{2}}(1, 1)$
- 测量概率：
  - $P(\frac{\hbar}{2}) = |\langle \psi, v_1 \rangle|^2 = |\frac{1}{\sqrt{2}}|^2 = \frac{1}{2}$
  - $P(-\frac{\hbar}{2}) = |\langle \psi, v_2 \rangle|^2 = |\frac{1}{\sqrt{2}}|^2 = \frac{1}{2}$
- 期望值：$\langle S_z \rangle = \frac{\hbar}{2} \cdot \frac{1}{2} + (-\frac{\hbar}{2}) \cdot \frac{1}{2} = 0$

**例2：位置测量**

- 系统：一维粒子，$H = L^2(\mathbb{R})$
- 可观测量：位置算子 $X$，$(X\psi)(x) = x\psi(x)$
- 谱：$\sigma(X) = \mathbb{R}$（连续谱）
- 概率密度：$|\psi(x)|^2$（Born规则）
- 期望值：$\langle X \rangle = \int_{-\infty}^{\infty} x |\psi(x)|^2 \, dx$

**时间演化（Schrödinger方程）**：

$$i\hbar \frac{\partial}{\partial t} |\psi(t) \rangle = H |\psi(t) \rangle$$

其中 $H$ 是哈密顿算子（自伴）。

**形式解**（使用谱分解）：

$$|\psi(t) \rangle = e^{-itH/\hbar} |\psi(0) \rangle = \int_{\sigma(H)} e^{-it\lambda/\hbar} \, dE_H(\lambda) |\psi(0) \rangle$$

**具体例子**：

**例：谐振子**

- 哈密顿：$H = \frac{P^2}{2m} + \frac{1}{2}m\omega^2 X^2$
- 特征值（能级）：$E_n = \hbar\omega(n + \frac{1}{2})$，$n = 0, 1, 2, \ldots$
- 特征函数：Hermite函数 $v_n(x)$
- 时间演化：若初始态 $|\psi(0) \rangle = v_n$，则
  $$|\psi(t) \rangle = e^{-itE_n/\hbar} v_n = e^{-it\omega(n+1/2)} v_n$$

**物理意义**：
- $H$ 的谱 = 系统能级
- 特征函数 = 定态波函数
- 谱分解 = 能量本征态展开

## 四、紧算子的Riesz理论

### 4.1 紧算子的基本性质

**定义**：
A是紧算子 ⟺ A将单位球映射到预紧集

**等价条件**：

```
A紧 ⟺ 对任意有界序列{xₙ}，{Axₙ}有收敛子列
```

**例子**：

```
积分算子（核连续）：
(Kf)(x) = ∫ K(x,y)f(y)dy

若K∈C([0,1]²)，则K紧
```

---

### 4.2 Riesz谱理论

**定理**：
> 若A是紧算子，则：
>
> 1. σ(A)可数，且0是唯一可能的聚点
> 2. 非零特征值有有限重数
> 3. 若λ≠0是特征值，则λ*也是（A*的特征值）

**Fredholm择一定理**：

```
方程 (I-λA)x = y

要么：
- 有唯一解（任意y）
- 或齐次方程(I-λA)x=0有非零解
```

## 五、无界算子

### 5.1 定义域问题

**问题**：
量子力学中的算子（位置、动量）无界

**例子**：

```
位置算子X：
(Xψ)(x) = xψ(x)

定义域：
D(X) = {ψ∈L²(ℝ) | ∫|xψ(x)|²dx < ∞}

问题：D(X) ≠ L²(ℝ)
→ X不是B(L²(ℝ))的元素
```

---

### 5.2 自伴扩张

**对称算子**：

```
A对称 ⟺ ⟨Ax,y⟩ = ⟨x,Ay⟩（对所有x,y∈D(A)）

自伴 ⟺ A对称且D(A*) = D(A)
```

**von Neumann自伴扩张理论**：

```
A对称 → 有自伴扩张 ⟺
亏指数相等：n₊(A) = n₋(A)

n₊(A) = dim ker(A*-iI)
n₋(A) = dim ker(A*+iI)
```

**例子：动量算子**：

```
P = -iℏ d/dx

在L²[0,1]上：
- 边界条件决定自伴扩张
- 周期边界条件 → 自伴
```

## 六、应用：偏微分方程

### 6.1 Sturm-Liouville问题

**问题**：

```
-(py')' + qy = λy
边界条件：y(a) = y(b) = 0

算子：
L = -d/dx(p d/dx) + q

性质：
- L自伴（在适当边界条件下）
- L有紧逆（若p>0, q≥0）
- → L有可数特征值
- → 特征函数形成L²[a,b]的基
```

---

### 6.2 热方程

**问题**：

```
∂u/∂t = Δu（拉普拉斯算子）

算子：
A = Δ（在L²(Ω)上，Dirichlet边界条件）

性质：
- A自伴
- A有紧逆（Ω有界）
- → e^(tΔ)有谱分解
- → 解可展开为特征函数级数
```

### 谱理论的历史地位

**希尔伯特的贡献**：

1. 引入无限维谱理论
2. 积分方程 → 算子理论
3. 为量子力学奠定数学基础

**现代发展**：

- von Neumann：算子代数
- Gelfand：C*-代数

## 八、数学公式总结

### 核心公式

1. **谱的定义**：
   $$\sigma(A) = \{\lambda \in \mathbb{C} : A - \lambda I \text{ 不可逆}\}$$

2. **谱分解（紧自伴算子）**：
   $$A = \sum_{n=1}^{\infty} \lambda_n \langle \cdot, v_n \rangle v_n$$

3. **投影值测度**：
   $$A = \int_{\mathbb{R}} \lambda \, dE(\lambda)$$

4. **函数演算**：
   $$f(A) = \int_{\mathbb{R}} f(\lambda) \, dE(\lambda)$$

5. **Born规则**：
   $$P(\lambda_n) = |\langle \psi, v_n \rangle|^2$$

6. **期望值**：
   $$\langle A \rangle = \langle \psi, A \psi \rangle = \sum_{n=1}^{\infty} \lambda_n |\langle \psi, v_n \rangle|^2$$

7. **Schrödinger方程解**：
   $$|\psi(t) \rangle = e^{-itH/\hbar} |\psi(0) \rangle = \int_{\sigma(H)} e^{-it\lambda/\hbar} \, dE_H(\lambda) |\psi(0) \rangle$$

8. **C*-恒等式**：
   $$\|a^*a\| = \|a\|^2$$

9. **von Neumann双换位定理**：
   $$M = M''$$

10. **Fredholm择一定理**：
    方程 $(I - \lambda K)f = g$ 要么有唯一解，要么齐次方程有非零解。
