# 谱理论与算子代数：希尔伯特空间的深层结构

## 一、有界算子的谱

### 1.1 谱的定义

**设定**：
H是希尔伯特空间，A: H→H是有界线性算子

**谱的定义**：

```
σ(A) = {λ∈ℂ | A-λI不可逆}

分解：
- 点谱σₚ(A)：特征值（A-λI非单射）
- 连续谱σc(A)：A-λI单射但非满射
- 剩余谱σᵣ(A)：A-λI满射但非单射（有限维不存在）

关系：
σ(A) = σₚ(A) ∪ σc(A) ∪ σᵣ(A)
```
**例子**：

```
乘法算子(Mf)(x) = xf(x)在L²[0,1]
σ(M) = [0,1]（连续谱）
σₚ(M) = ∅（无特征值）
```
---

### 1.2 谱的基本性质

**定理**：

```
1. σ(A)是紧集（有界闭集）
2. σ(A) ⊆ {λ | |λ| ≤ ||A||}
3. 若A自伴，则σ(A) ⊆ ℝ
4. 若A酉，则σ(A) ⊆ {λ | |λ|=1}
```
**证明思路**：

- 利用Neumann级数展开(A-λI)⁻¹
- 解析函数理论

## 二、自伴算子的谱定理

### 2.1 有限维情况（矩阵）

**谱定理（有限维）**：
> 若A是n×n自伴矩阵，则：
>
> 1. A有n个实特征值λ₁,...,λₙ
> 2. 存在正交基{v₁,...,vₙ}使得Avᵢ = λᵢvᵢ
> 3. A = ∑λᵢPᵢ（Pᵢ是到vᵢ的投影）

**对角化**：

```
A = UΛU*
U：酉矩阵
Λ：对角矩阵（特征值）
```
---

### 2.2 无限维：紧自伴算子

**定理（Hilbert-Schmidt）**：
> 若A是紧自伴算子，则：
>
> 1. σ(A) = σₚ(A)（只有特征值）
> 2. 特征值可数，且λₙ→0
> 3. 特征向量{vₙ}形成H的正交基
> 4. A = ∑λₙ⟨·,vₙ⟩vₙ（谱分解）

**例子：积分算子**：

```
(Kf)(x) = ∫₀¹ K(x,y)f(y)dy

若K(x,y) = K(y,x)（对称核）
则K是自伴紧算子
→ 有可数特征值
→ 特征函数形成L²[0,1]的基
```
---

### 2.3 一般自伴算子：投影值测度

**谱定理（von Neumann, 1929）**：

**投影值测度**：

```
E: Borel集B(ℝ) → 投影算子E(B)

性质：
- E(∅) = 0, E(ℝ) = I
- E(B₁∩B₂) = E(B₁)E(B₂)
- E(∪Bᵢ) = ∑E(Bᵢ)（互不相交）
```
**谱积分**：

```
A = ∫ λ dE(λ)

意义：
A的"对角化"（在连续谱情况）
```
**函数演算**：

```
f(A) = ∫ f(λ) dE(λ)

例子：
e^(itA) = ∫ e^(itλ) dE(λ)
√A = ∫ √λ dE(λ)（若A≥0）
```

## 三、算子代数

### 3.1 C*-代数

**定义**：

```
C*-代数A是Banach代数，带有对合*，满足：
1. (a*)* = a
2. (ab)* = b*a*
3. ||a*a|| = ||a||²（C*-恒等式）

例子：
- B(H)：有界算子（A* = 伴随）
- C(X)：连续函数（f* = 共轭）
```
**Gelfand-Naimark定理**：
> 交换C*-代数同构于C(X)（某紧Hausdorff空间X）

---

### 3.2 von Neumann代数

**定义**：

```
M ⊆ B(H)是von Neumann代数，若：
1. M是*-代数
2. M在弱算子拓扑下闭

等价：
M = M''（双换位定理）
```
**分类**：

```
类型I：有极小投影
类型II：无极小投影，但有有限投影
类型III：所有投影都无限

因子：中心平凡（Z(M) = ℂI）
```
---

### 3.3 应用：量子力学

**可观测量**：

```
自伴算子A ↔ 物理可观测量

测量结果：A的特征值
概率：Born规则
P(λ) = |⟨ψ, vλ⟩|²

期望值：
⟨A⟩ = ⟨ψ, Aψ⟩ = ∫ λ d⟨ψ, E(λ)ψ⟩
```
**时间演化**：

```
Schrödinger方程：
iℏ ∂ψ/∂t = Hψ

解：
ψ(t) = e^(-itH/ℏ)ψ(0)
    = ∫ e^(-itλ/ℏ) dE_H(λ)ψ(0)

H的谱 = 系统能级
```

## 四、紧算子的Riesz理论

### 4.1 紧算子的基本性质

**定义**：
A是紧算子 ⟺ A将单位球映射到预紧集

**等价条件**：

```
A紧 ⟺ 对任意有界序列{xₙ}，{Axₙ}有收敛子列
```
**例子**：

```
积分算子（核连续）：
(Kf)(x) = ∫ K(x,y)f(y)dy

若K∈C([0,1]²)，则K紧
```
---

### 4.2 Riesz谱理论

**定理**：
> 若A是紧算子，则：
>
> 1. σ(A)可数，且0是唯一可能的聚点
> 2. 非零特征值有有限重数
> 3. 若λ≠0是特征值，则λ*也是（A*的特征值）

**Fredholm择一定理**：

```
方程 (I-λA)x = y

要么：
- 有唯一解（任意y）
- 或齐次方程(I-λA)x=0有非零解
```
---

## 五、无界算子

### 5.1 定义域问题

**问题**：
量子力学中的算子（位置、动量）无界

**例子**：

```
位置算子X：
(Xψ)(x) = xψ(x)

定义域：
D(X) = {ψ∈L²(ℝ) | ∫|xψ(x)|²dx < ∞}

问题：D(X) ≠ L²(ℝ)
→ X不是B(L²(ℝ))的元素
```
---

### 5.2 自伴扩张

**对称算子**：

```
A对称 ⟺ ⟨Ax,y⟩ = ⟨x,Ay⟩（对所有x,y∈D(A)）

自伴 ⟺ A对称且D(A*) = D(A)
```
**von Neumann自伴扩张理论**：

```
A对称 → 有自伴扩张 ⟺
亏指数相等：n₊(A) = n₋(A)

n₊(A) = dim ker(A*-iI)
n₋(A) = dim ker(A*+iI)
```
**例子：动量算子**：

```
P = -iℏ d/dx

在L²[0,1]上：
- 边界条件决定自伴扩张
- 周期边界条件 → 自伴
```
---

## 六、应用：偏微分方程

### 6.1 Sturm-Liouville问题

**问题**：

```
-(py')' + qy = λy
边界条件：y(a) = y(b) = 0

算子：
L = -d/dx(p d/dx) + q

性质：
- L自伴（在适当边界条件下）
- L有紧逆（若p>0, q≥0）
- → L有可数特征值
- → 特征函数形成L²[a,b]的基
```
---

### 6.2 热方程

**问题**：

```
∂u/∂t = Δu（拉普拉斯算子）

算子：
A = Δ（在L²(Ω)上，Dirichlet边界条件）

性质：
- A自伴
- A有紧逆（Ω有界）
- → e^(tΔ)有谱分解
- → 解可展开为特征函数级数
```
---

## 七、总结

### 谱理论的历史地位

**希尔伯特的贡献**：

1. 引入无限维谱理论
2. 积分方程 → 算子理论
3. 为量子力学奠定数学基础

**现代发展**：

- von Neumann：算子代数
- Gelfand：C*-代数
- Connes：非交换几何

> "希尔伯特空间理论是20世纪数学与物理最成功的联姻。它证明了抽象数学的**物理实在性**。"
