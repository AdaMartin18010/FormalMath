# 无界算子理论：希尔伯特空间的现代发展

## 一、有界与无界算子

### 1.1 有界算子的局限

**有界算子**：

```
T: H → H有界 ⟺
存在M使得：||Tx|| ≤ M||x||

性质：
- 连续
- 定义域全空间
- 谱有界
```

**局限**：

- 许多重要算子无界
- 微分算子
- 位置、动量算子（量子力学）

---

### 1.2 无界算子的必要性

**例子：微分算子**：

```
D = d/dx: L²[0,1] → L²[0,1]

问题：
- 不连续函数不可微
- 定义域 ≠ 全空间
- 无界
```

**量子力学**：

```
位置算子X：
(Xψ)(x) = x·ψ(x)

动量算子P：
(Pψ)(x) = -iℏ dψ/dx

性质：
- 无界
- 自伴（在适当定义域）
- 谱连续
```

---

## 二、无界算子的定义

### 2.1 定义域

**稠密定义算子**：

```
T: D(T) → H

其中：
- D(T) ⊆ H稠密
- T线性
- D(T)是向量子空间
```

**闭算子**：

```
T是闭的 ⟺
图{(x, Tx) | x∈D(T)}在H×H中闭

等价：
若xₙ → x且Txₙ → y，则x∈D(T)且Tx = y
```

---

### 2.2 对称与自伴

**对称算子**：

```
T对称 ⟺
对任意x,y∈D(T)：
⟨Tx, y⟩ = ⟨x, Ty⟩
```

**自伴算子**：

```
T自伴 ⟺
T对称且D(T*) = D(T)

其中：
D(T*) = {y | 存在z使得⟨Tx,y⟩ = ⟨x,z⟩对所有x∈D(T)}
T*y = z
```

**区别**：

- 对称：D(T) ⊆ D(T*)
- 自伴：D(T) = D(T*)

---

## 三、谱理论

### 3.1 无界算子的谱

**定义**：

```
σ(T) = {λ∈ℂ | T-λI不是双射}

分解：
- σₚ(T)：特征值（点谱）
- σc(T)：连续谱
- σᵣ(T)：剩余谱
```

**性质**：

- σ(T)可能是无界集
- 可能包含整个复平面
- 自伴算子：σ(T) ⊆ ℝ

---

### 3.2 自伴算子的谱定理

**定理**（von Neumann, 1929）：

```
若T是自伴算子，则存在：
- 投影值测度E
- 使得：T = ∫ λ dE(λ)

谱分解：
T = ∫_{σ(T)} λ dE(λ)
```

**应用**：

- 函数演算：f(T) = ∫ f(λ) dE(λ)
- 量子力学：测量理论
- 演化：e^(itT) = ∫ e^(itλ) dE(λ)

---

## 四、Stone定理

### 4.1 单参数酉群

**定义**：

```
U(t): H → H（t∈ℝ）

性质：
- U(0) = I
- U(s+t) = U(s)U(t)
- U(t)* = U(-t)
- t → U(t)强连续
```

**生成元**：

```
A = lim_{t→0} (U(t)-I)/(it)

性质：
- A自伴
- U(t) = e^(itA)
```

---

### 4.2 Stone定理

**定理**（Stone, 1932）：

```
U(t)是强连续单参数酉群
    ⟺
存在唯一自伴算子A使得：
U(t) = e^(itA)

且：
A = lim_{t→0} (U(t)-I)/(it)
```

**应用**：

- 量子力学：时间演化
- Schrödinger方程：U(t) = e^(-itH/ℏ)
- 偏微分方程：半群理论

---

## 五、应用：量子力学

### 5.1 位置与动量算子

**位置算子X**：

```
定义域：
D(X) = {ψ∈L²(ℝ) | x·ψ(x)∈L²(ℝ)}

作用：
(Xψ)(x) = x·ψ(x)

性质：
- 自伴
- σ(X) = ℝ（连续谱）
- 无特征值
```

**动量算子P**：

```
定义域：
D(P) = {ψ∈L²(ℝ) | ψ'∈L²(ℝ)}

作用：
(Pψ)(x) = -iℏ dψ/dx

性质：
- 自伴
- σ(P) = ℝ
- 对易关系：[X,P] = iℏI
```

---

### 5.2 不确定性原理

**数学形式**：

```
对任意ψ∈D(X)∩D(P)：

ΔX · ΔP ≥ ℏ/2

其中：
ΔX = √(⟨X²⟩ - ⟨X⟩²)
ΔP = √(⟨P²⟩ - ⟨P⟩²)
```

**证明**：

- Cauchy-Schwarz不等式
- 对易关系
- 无界算子的性质

---

## 六、应用：偏微分方程

### 6.1 椭圆算子

**例子：Laplace算子**：

```
Δ = -∑ ∂²/∂xᵢ²

定义域：
D(Δ) = H²(Ω)（Sobolev空间）

性质：
- 自伴（在Dirichlet边界条件下）
- σ(Δ)离散
- 特征函数形成基
```

**应用**：

- 热方程：∂u/∂t = Δu
- 波方程：∂²u/∂t² = Δu
- Schrödinger方程：i∂u/∂t = Δu

---

### 6.2 半群理论

**Hille-Yosida定理**：

```
A生成强连续半群e^(tA)
    ⟺
A是闭稠定算子且：
- 存在M,ω使得||(λ-A)⁻¹|| ≤ M/(λ-ω)（λ>ω）

应用：
- 抛物型PDE
- 随机过程
- 控制理论
```

## 八、总结

### 无界算子理论的历史地位

**希尔伯特的贡献**：

- 开创希尔伯特空间理论
- 为无界算子理论奠基

**现代发展**：

- von Neumann的谱定理
- Stone的酉群理论
- 应用：量子力学、PDE

无界算子理论是**希尔伯特空间理论的自然延伸**，是现代数学物理的基础。
