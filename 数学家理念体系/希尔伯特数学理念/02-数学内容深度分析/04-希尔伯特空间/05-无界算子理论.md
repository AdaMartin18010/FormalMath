## 一、有界与无界算子

### 1.1 有界算子的局限

**有界算子**：

```
T: H → H有界 ⟺
存在M使得：||Tx|| ≤ M||x||

性质：
- 连续
- 定义域全空间
- 谱有界
```

**局限**：

- 许多重要算子无界
- 微分算子
- 位置、动量算子（量子力学）

---

### 1.2 无界算子的必要性

**例子：微分算子**：

```
D = d/dx: L²[0,1] → L²[0,1]

问题：
- 不连续函数不可微
- 定义域 ≠ 全空间
- 无界
```

**量子力学**：

```
位置算子X：
(Xψ)(x) = x·ψ(x)

动量算子P：
(Pψ)(x) = -iℏ dψ/dx

性质：
- 无界
- 自伴（在适当定义域）
- 谱连续
```

## 二、无界算子的定义

### 2.1 定义域

**稠密定义算子**：

```
T: D(T) → H

其中：
- D(T) ⊆ H稠密
- T线性
- D(T)是向量子空间
```

**闭算子**：

```
T是闭的 ⟺
图{(x, Tx) | x∈D(T)}在H×H中闭

等价：
若xₙ → x且Txₙ → y，则x∈D(T)且Tx = y
```

---

### 2.2 对称与自伴

**对称算子（数学定义）**：

算子 $T: D(T) \to H$ 是对称的，若对任意 $x, y \in D(T)$：

$$\langle Tx, y \rangle = \langle x, Ty \rangle$$

**具体例子**：

**例1：动量算子（对称但非自伴）**

在 $L^2[0,1]$ 上，定义：
$$D(P) = \{\psi \in C^1[0,1] : \psi(0) = \psi(1) = 0\}$$
$$(P\psi)(x) = -i\hbar \frac{d\psi}{dx}$$

- $P$ 是对称的：$\langle P\psi, \phi \rangle = \langle \psi, P\phi \rangle$（通过分部积分）
- 但 $P$ 不是自伴的：$D(P^*) \supset D(P)$（$P^*$ 的定义域更大）

**例2：动量算子（自伴）**

在 $L^2(\mathbb{R})$ 上，定义：
$$D(P) = \{\psi \in L^2(\mathbb{R}) : \psi' \in L^2(\mathbb{R})\}$$
$$(P\psi)(x) = -i\hbar \frac{d\psi}{dx}$$

- $P$ 是自伴的：$D(P^*) = D(P)$

**自伴算子（数学定义）**：

算子 $T$ 是自伴的，若：
1. $T$ 是对称的：$\langle Tx, y \rangle = \langle x, Ty \rangle$ 对所有 $x, y \in D(T)$
2. $D(T^*) = D(T)$

其中**伴随算子** $T^*$ 的定义域为：
$$D(T^*) = \{y \in H : \exists z \in H, \forall x \in D(T), \langle Tx, y \rangle = \langle x, z \rangle\}$$

且 $T^*y = z$。

**区别**：

| 性质 | 对称算子 | 自伴算子 |
|------|---------|---------|
| **定义域关系** | $D(T) \subseteq D(T^*)$ | $D(T) = D(T^*)$ |
| **谱的性质** | $\sigma(T) \subseteq \mathbb{R}$（可能不闭） | $\sigma(T) \subseteq \mathbb{R}$（闭集） |
| **谱定理** | 不一定成立 | 成立（von Neumann谱定理） |
| **物理意义** | 可能不对应可观测量 | 对应物理可观测量 |

**关键例子**：

**例：位置算子**

在 $L^2(\mathbb{R})$ 上：
$$D(X) = \{\psi \in L^2(\mathbb{R}) : x\psi(x) \in L^2(\mathbb{R})\}$$
$$(X\psi)(x) = x\psi(x)$$

- $X$ 是自伴的：$D(X^*) = D(X)$
- 谱：$\sigma(X) = \mathbb{R}$（连续谱）
- 无特征值（因为 $x\psi(x) = \lambda\psi(x)$ 无 $L^2$ 解）

## 三、谱理论

### 3.1 无界算子的谱

**定义**：

```
σ(T) = {λ∈ℂ | T-λI不是双射}

分解：
- σₚ(T)：特征值（点谱）
- σc(T)：连续谱
- σᵣ(T)：剩余谱
```

**性质**：

- σ(T)可能是无界集
- 可能包含整个复平面
- 自伴算子：σ(T) ⊆ ℝ

---

### 3.2 自伴算子的谱定理

**定理**（von Neumann, 1929）：

```
若T是自伴算子，则存在：
- 投影值测度E
- 使得：T = ∫ λ dE(λ)

谱分解：
T = ∫_{σ(T)} λ dE(λ)
```

**应用**：

- 函数演算：f(T) = ∫ f(λ) dE(λ)
- 量子力学：测量理论
- 演化：e^(itT) = ∫ e^(itλ) dE(λ)

## 四、Stone定理

### 4.1 单参数酉群

**定义**：

```
U(t): H → H（t∈ℝ）

性质：
- U(0) = I
- U(s+t) = U(s)U(t)
- U(t)* = U(-t)
- t → U(t)强连续
```

**生成元**：

```
A = lim_{t→0} (U(t)-I)/(it)

性质：
- A自伴
- U(t) = e^(itA)
```

---

### 4.2 Stone定理

**Stone定理（数学表述）**（Stone, 1932）：

> 设 $\{U(t)\}_{t \in \mathbb{R}}$ 是希尔伯特空间 $H$ 上的强连续单参数酉群，则存在唯一的自伴算子 $A$ 使得：
> $$U(t) = e^{itA}$$
>
> 且生成元 $A$ 由下式给出：
> $$A = \lim_{t \to 0} \frac{U(t) - I}{it}$$
>
> 定义域：
> $$D(A) = \leqqqqqft\{ x \in H : \lim_{t \to 0} \frac{U(t)x - x}{it} \text{ 存在} \right\}$$

**具体例子**：

**例1：量子力学时间演化**

- 单参数酉群：$U(t) = e^{-itH/\hbar}$（$H$ 是哈密顿算子）
- 生成元：$A = -\frac{H}{\hbar}$（自伴，因为 $H$ 自伴）
- 物理意义：时间演化由能量算子生成

**例2：空间平移**

在 $L^2(\mathbb{R})$ 上：
$$(U(a)\psi)(x) = \psi(x - a)$$

- $U(a)$ 是单参数酉群（$a \in \mathbb{R}$）
- 生成元：$A = iP$（其中 $P$ 是动量算子）
- 物理意义：空间平移由动量算子生成

**例3：旋转群**

在 $L^2(\mathbb{R}^2)$ 上：
$$(U(\theta)\psi)(r, \phi) = \psi(r, \phi - \theta)$$

- $U(\theta)$ 是单参数酉群（$\theta \in \mathbb{R}$）
- 生成元：$A = iL_z$（其中 $L_z$ 是角动量算子）
- 物理意义：旋转由角动量算子生成

**应用**：

- **量子力学**：时间演化 $U(t) = e^{-itH/\hbar}$ 由哈密顿算子 $H$ 生成
- **偏微分方程**：演化半群 $e^{tA}$ 由算子 $A$ 生成
- **群表示论**：李群的单参数子群由李代数元素生成

**应用**：

- 量子力学：时间演化
- Schrödinger方程：U(t) = e^(-itH/ℏ)
- 偏微分方程：半群理论

## 五、应用：量子力学

### 5.1 位置与动量算子

**位置算子X**：

```
定义域：
D(X) = {ψ∈L²(ℝ) | x·ψ(x)∈L²(ℝ)}

作用：
(Xψ)(x) = x·ψ(x)

性质：
- 自伴
- σ(X) = ℝ（连续谱）
- 无特征值
```

**动量算子P**：

```
定义域：
D(P) = {ψ∈L²(ℝ) | ψ'∈L²(ℝ)}

作用：
(Pψ)(x) = -iℏ dψ/dx

性质：
- 自伴
- σ(P) = ℝ
- 对易关系：[X,P] = iℏI
```

---

### 5.2 不确定性原理

**数学形式**：

```
对任意ψ∈D(X)∩D(P)：

ΔX · ΔP ≥ ℏ/2

其中：
ΔX = √(⟨X²⟩ - ⟨X⟩²)
ΔP = √(⟨P²⟩ - ⟨P⟩²)
```

**证明**：

- Cauchy-Schwarz不等式
- 对易关系
- 无界算子的性质

## 六、应用：偏微分方程

### 6.1 椭圆算子

**例子：Laplace算子**：

```
Δ = -∑ ∂²/∂xᵢ²

定义域：
D(Δ) = H²(Ω)（Sobolev空间）

性质：
- 自伴（在Dirichlet边界条件下）
- σ(Δ)离散
- 特征函数形成基
```

**应用**：

- 热方程：∂u/∂t = Δu
- 波方程：∂²u/∂t² = Δu
- Schrödinger方程：i∂u/∂t = Δu

---

### 6.2 半群理论

**Hille-Yosida定理**：

```
A生成强连续半群e^(tA)
    ⟺
A是闭稠定算子且：
- 存在M,ω使得||(λ-A)⁻¹|| ≤ M/(λ-ω)（λ>ω）

应用：
- 抛物型PDE
- 随机过程
- 控制理论
```

## 九、数学公式总结

### 核心公式

1. **对称算子**：
   $$\langle Tx, y \rangle = \langle x, Ty \rangle \quad \text{对所有 } x, y \in D(T)$$

2. **自伴算子**：
   $$T \text{ 自伴 } \iff T \text{ 对称且 } D(T^*) = D(T)$$

3. **伴随算子定义域**：
   $$D(T^*) = \{y \in H : \exists z \in H, \forall x \in D(T), \langle Tx, y \rangle = \langle x, z \rangle\}$$

4. **无界算子的谱**：
   $$\sigma(T) = \{\lambda \in \mathbb{C} : T - \lambda I \text{ 不是双射}\}$$

5. **自伴算子的谱定理**：
   $$T = \int_{\sigma(T)} \lambda \, dE(\lambda)$$

6. **函数演算**：
   $$f(T) = \int_{\sigma(T)} f(\lambda) \, dE(\lambda)$$

7. **Stone定理**：
   $$U(t) = e^{itA} \quad \text{其中 } A \text{ 是自伴算子}$$

8. **生成元**：
   $$A = \lim_{t \to 0} \frac{U(t) - I}{it}$$

9. **位置算子**：
   $$(X\psi)(x) = x\psi(x), \quad D(X) = \{\psi : x\psi \in L^2(\mathbb{R})\}$$

10. **动量算子**：
    $$(P\psi)(x) = -i\hbar \frac{d\psi}{dx}, \quad D(P) = \{\psi : \psi' \in L^2(\mathbb{R})\}$$

11. **对易关系**：
    $$[X, P] = i\hbar I$$

12. **不确定性原理**：
    $$\Delta X \cdot \Delta P \geqqqqqq \frac{\hbar}{2}$$
