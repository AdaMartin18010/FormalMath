## 一、von Neumann公理化（1932）

### 1.1 量子力学的数学框架

**von Neumann的贡献**：

```
《量子力学的数学基础》（1932）

核心：
- 状态空间 = 希尔伯特空间H
- 可观测量 = 自伴算子
- 测量 = 谱分解
```

**与希尔伯特的关系**：

- 使用希尔伯特空间理论
- 但赋予物理意义
- 形式主义的"物理实现"

---

### 1.2 基本公理

**公理1：状态空间（数学表述）**：

量子系统的状态由希尔伯特空间 $H$ 中的归一化向量表示：

$$|\psi \rangle \in H, \quad \langle \psi | \psi \rangle = 1$$

**具体例子**：

**例1：自旋 $\frac{1}{2}$ 粒子**
- $H = \mathbb{C}^2$
- 状态：$|\psi \rangle = \begin{pmatrix} \alpha \\ \beta \end{pmatrix}$，其中 $|\alpha|^2 + |\beta|^2 = 1$
- 基态：$|\uparrow \rangle = \begin{pmatrix} 1 \\ 0 \end{pmatrix}$，$|\downarrow \rangle = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$

**例2：一维粒子**
- $H = L^2(\mathbb{R})$
- 状态：$\psi(x) \in L^2(\mathbb{R})$，满足 $\int_{-\infty}^{\infty} |\psi(x)|^2 \, dx = 1$
- 例如：高斯波包 $\psi(x) = \frac{1}{(\pi\sigma^2)^{1/4}} e^{-x^2/(2\sigma^2)}$

**公理2：可观测量（数学表述）**：

物理可观测量对应自伴算子 $A: H \to H$（$A^* = A$）。

**测量结果**：$A$ 的特征值 $\{\lambda_n\}_{n=1}^{\infty}$（离散谱）或 $\lambda \in \sigma(A)$（连续谱）。

**具体例子**：

**例1：自旋算子**
- $S_z = \frac{\hbar}{2} \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$（$z$ 方向自旋）
- 特征值：$\lambda_1 = \frac{\hbar}{2}$，$\lambda_2 = -\frac{\hbar}{2}$
- 特征向量：$v_1 = \begin{pmatrix} 1 \\ 0 \end{pmatrix}$，$v_2 = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$

**例2：位置算子**
- $X: L^2(\mathbb{R}) \to L^2(\mathbb{R})$，$(X\psi)(x) = x\psi(x)$
- 谱：$\sigma(X) = \mathbb{R}$（连续谱）
- 无特征值（因为 $x\psi(x) = \lambda\psi(x)$ 无 $L^2$ 解）

**公理3：测量（Born规则）**：

测量可观测量 $A$ 在状态 $|\psi \rangle$ 的结果：

1. **测量结果**：特征值 $\lambda_n$（离散谱）或 $\lambda \in \sigma(A)$（连续谱）
2. **概率**（离散谱）：
   $$P(\lambda_n) = |\langle \psi | v_n \rangle|^2$$
   其中 $v_n$ 是对应特征值 $\lambda_n$ 的归一化特征向量。
3. **测量后状态**：$|v_n \rangle$（状态坍缩到特征子空间）

**具体例子**：

**例：自旋测量**
- 状态：$|\psi \rangle = \frac{1}{\sqrt{2}} \begin{pmatrix} 1 \\ 1 \end{pmatrix} = \frac{1}{\sqrt{2}}(|\uparrow \rangle + |\downarrow \rangle)$
- 测量 $S_z$：
  - 结果 $+\frac{\hbar}{2}$，概率：$P(+\frac{\hbar}{2}) = |\langle \psi | \uparrow \rangle|^2 = |\frac{1}{\sqrt{2}}|^2 = \frac{1}{2}$
  - 结果 $-\frac{\hbar}{2}$，概率：$P(-\frac{\hbar}{2}) = |\langle \psi | \downarrow \rangle|^2 = |\frac{1}{\sqrt{2}}|^2 = \frac{1}{2}$
- 测量后：若结果为 $+\frac{\hbar}{2}$，则状态变为 $|\uparrow \rangle$

## 二、状态与测量

### 2.1 量子叠加

**叠加原理**：

```
若|ψ₁⟩和|ψ₂⟩是可能状态
则|ψ⟩ = α|ψ₁⟩ + β|ψ₂⟩也是状态

其中：
|α|² + |β|² = 1
```

**例子：自旋½粒子**：

```
状态空间：H = ℂ²

基态：
|↑⟩ = (1,0)
|↓⟩ = (0,1)

一般态：
|ψ⟩ = α|↑⟩ + β|↓⟩

测量σz：
- 结果+1，概率|α|²，态→|↑⟩
- 结果-1，概率|β|²，态→|↓⟩
```

---

### 2.2 Born规则

**Born规则（数学表述）**：

设可观测量 $A$ 是自伴算子，有谱分解：
$$A = \sum_{n=1}^{\infty} \lambda_n |v_n \rangle \langle v_n|$$

其中 $\{v_n\}$ 是标准正交特征向量，$\{\lambda_n\}$ 是特征值。

**测量概率**：
$$P(\lambda_n) = |\langle \psi | v_n \rangle|^2$$

**期望值**：
$$\langle A \rangle = \langle \psi | A | \psi \rangle = \sum_{n=1}^{\infty} \lambda_n |\langle \psi | v_n \rangle|^2 = \sum_{n=1}^{\infty} \lambda_n P(\lambda_n)$$

**具体例子**：

**例1：自旋测量**
- 状态：$|\psi \rangle = \frac{1}{\sqrt{2}} \begin{pmatrix} 1 \\ 1 \end{pmatrix}$
- 可观测量：$S_z = \frac{\hbar}{2} \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$
- 特征值：$\lambda_1 = \frac{\hbar}{2}$，$\lambda_2 = -\frac{\hbar}{2}$
- 特征向量：$v_1 = \begin{pmatrix} 1 \\ 0 \end{pmatrix}$，$v_2 = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$
- 概率：
  - $P(\frac{\hbar}{2}) = |\langle \psi | v_1 \rangle|^2 = |\frac{1}{\sqrt{2}}|^2 = \frac{1}{2}$
  - $P(-\frac{\hbar}{2}) = |\langle \psi | v_2 \rangle|^2 = |\frac{1}{\sqrt{2}}|^2 = \frac{1}{2}$
- 期望值：$\langle S_z \rangle = \frac{\hbar}{2} \cdot \frac{1}{2} + (-\frac{\hbar}{2}) \cdot \frac{1}{2} = 0$

**例2：位置测量（连续谱）**
- 状态：$\psi(x) = \frac{1}{(\pi\sigma^2)^{1/4}} e^{-x^2/(2\sigma^2)}$（高斯波包）
- 可观测量：位置算子 $X$，$(X\psi)(x) = x\psi(x)$
- 谱：$\sigma(X) = \mathbb{R}$（连续谱）
- 概率密度：$|\psi(x)|^2 = \frac{1}{\sqrt{\pi\sigma^2}} e^{-x^2/\sigma^2}$
- 期望值：$\langle X \rangle = \int_{-\infty}^{\infty} x |\psi(x)|^2 \, dx = 0$（对称分布）

**物理意义**：

- **概率性**：量子力学是概率性的，测量结果不确定
- **状态坍缩**：测量后状态从叠加态坍缩到特征态
- **与经典物理不同**：经典物理中，测量不改变系统状态

## 三、时间演化

### 3.1 Schrödinger方程

**方程**：

```
iℏ ∂|ψ⟩/∂t = H|ψ⟩

其中：
- H：Hamiltonian（能量算子）
- ℏ：约化Planck常数
```

**解**：

```
|ψ(t)⟩ = e^(-itH/ℏ)|ψ(0)⟩

其中：
e^(-itH/ℏ) = ∫ e^(-itλ/ℏ) dE_H(λ)
（函数演算）
```

---

### 3.2 能量本征态

**定态**：

```
若|ψ⟩是H的特征态：
H|ψ⟩ = E|ψ⟩

则：
|ψ(t)⟩ = e^(-itE/ℏ)|ψ⟩

性质：
- 概率分布不变
- 仅相位演化
```

**能级**：

```
H的谱 = 系统能级

离散谱：
- 束缚态
- 能级量子化

连续谱：
- 散射态
- 连续能量
```

## 四、不确定性原理

### 4.1 Heisenberg不确定性

**不确定性原理（数学表述）**：

对任意状态 $|\psi \rangle$ 和自伴算子 $A, B$：

$$\Delta A \cdot \Delta B \geq \frac{1}{2} |\langle [A, B] \rangle|$$

其中：
- **标准差**：$\Delta A = \sqrt{\langle A^2 \rangle - \langle A \rangle^2} = \sqrt{\langle (A - \langle A \rangle)^2 \rangle}$
- **对易子**：$[A, B] = AB - BA$
- **期望值**：$\langle [A, B] \rangle = \langle \psi | [A, B] | \psi \rangle$

**位置-动量不确定性**：

**对易关系**：
$$[X, P] = i\hbar I$$

其中 $X$ 是位置算子，$P$ 是动量算子。

**不确定性关系**：
$$\Delta X \cdot \Delta P \geq \frac{\hbar}{2}$$

**具体例子**：

**例1：最小不确定态（相干态）**

对于谐振子，存在最小不确定态（相干态）使得：
$$\Delta X \cdot \Delta P = \frac{\hbar}{2}$$

这些态是高斯波包：
$$\psi(x) = \frac{1}{(\pi\sigma^2)^{1/4}} e^{-x^2/(2\sigma^2)} e^{ip_0 x/\hbar}$$

其中 $\sigma = \sqrt{\frac{\hbar}{2m\omega}}$。

**例2：一般态**

对于任意归一化态 $\psi(x) \in L^2(\mathbb{R})$：
$$\Delta X \cdot \Delta P \geq \frac{\hbar}{2}$$

等号成立当且仅当 $\psi$ 是高斯波包（相干态）。

**物理意义**：

- **位置和动量不能同时精确测量**
- **测量精度受量子限制**
- **这是量子力学的基本特征**

---

### 4.2 数学证明

**Cauchy-Schwarz不等式**：

```
|⟨Aψ|Bψ⟩|² ≤ ⟨Aψ|Aψ⟩⟨Bψ|Bψ⟩

应用：
- A → A - ⟨A⟩I
- B → B - ⟨B⟩I

得到不确定性原理
```

## 五、纠缠与Bell不等式

### 5.1 纠缠态

**定义**：

```
两粒子系统：
H = H₁ ⊗ H₂

纠缠态：
|Ψ⟩不能分解为|ψ₁⟩⊗|ψ₂⟩

例子：Bell态
|Φ⁺⟩ = (|00⟩ + |11⟩)/√2
```

**性质**：

- 非局域关联
- 测量一粒子影响另一粒子
- 违反经典直觉

---

### 5.2 Bell不等式

**经典隐变量理论**：

```
假设：存在隐变量λ决定测量结果
→ 相关性有上界

Bell不等式：
|E(a,b) - E(a,b')| + |E(a',b) + E(a',b')| ≤ 2
```

**量子违反**：

```
量子力学预测：
|E(a,b) - E(a,b')| + |E(a',b) + E(a',b')| = 2√2 > 2

实验验证（Aspect, 1982）：
量子力学正确
→ 非局域性是物理实在
```
