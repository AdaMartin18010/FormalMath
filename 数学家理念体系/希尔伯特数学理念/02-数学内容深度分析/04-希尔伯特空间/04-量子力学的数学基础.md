## 一、von Neumann公理化（1932）

### 1.1 量子力学的数学框架

**von Neumann的贡献**：

```
《量子力学的数学基础》（1932）

核心：
- 状态空间 = 希尔伯特空间H
- 可观测量 = 自伴算子
- 测量 = 谱分解
```

**与希尔伯特的关系**：

- 使用希尔伯特空间理论
- 但赋予物理意义
- 形式主义的"物理实现"

---

### 1.2 基本公理

**公理1：状态空间**：

```
量子系统的状态：
|ψ⟩ ∈ H（希尔伯特空间）

归一化：
⟨ψ|ψ⟩ = 1
```

**公理2：可观测量**：

```
物理可观测量：
A: H → H（自伴算子）

测量结果：
A的特征值{λₙ}
```

**公理3：测量**：

```
测量A在状态|ψ⟩：
- 结果：λₙ（概率P(λₙ)）
- 概率：P(λₙ) = |⟨ψ|vₙ⟩|²
- 测量后状态：|vₙ⟩（坍缩）
```

## 二、状态与测量

### 2.1 量子叠加

**叠加原理**：

```
若|ψ₁⟩和|ψ₂⟩是可能状态
则|ψ⟩ = α|ψ₁⟩ + β|ψ₂⟩也是状态

其中：
|α|² + |β|² = 1
```

**例子：自旋½粒子**：

```
状态空间：H = ℂ²

基态：
|↑⟩ = (1,0)
|↓⟩ = (0,1)

一般态：
|ψ⟩ = α|↑⟩ + β|↓⟩

测量σz：
- 结果+1，概率|α|²，态→|↑⟩
- 结果-1，概率|β|²，态→|↓⟩
```

---

### 2.2 Born规则

**概率解释**：

```
测量A在状态|ψ⟩：

谱分解：
A = ∑λₙ|vₙ⟩⟨vₙ|

概率：
P(λₙ) = |⟨ψ|vₙ⟩|²

期望值：
⟨A⟩ = ⟨ψ|A|ψ⟩ = ∑λₙP(λₙ)
```

**物理意义**：

- 量子力学是概率性的
- 测量改变状态
- 与经典物理根本不同

## 三、时间演化

### 3.1 Schrödinger方程

**方程**：

```
iℏ ∂|ψ⟩/∂t = H|ψ⟩

其中：
- H：Hamiltonian（能量算子）
- ℏ：约化Planck常数
```

**解**：

```
|ψ(t)⟩ = e^(-itH/ℏ)|ψ(0)⟩

其中：
e^(-itH/ℏ) = ∫ e^(-itλ/ℏ) dE_H(λ)
（函数演算）
```

---

### 3.2 能量本征态

**定态**：

```
若|ψ⟩是H的特征态：
H|ψ⟩ = E|ψ⟩

则：
|ψ(t)⟩ = e^(-itE/ℏ)|ψ⟩

性质：
- 概率分布不变
- 仅相位演化
```

**能级**：

```
H的谱 = 系统能级

离散谱：
- 束缚态
- 能级量子化

连续谱：
- 散射态
- 连续能量
```

## 四、不确定性原理

### 4.1 Heisenberg不确定性

**原理**：

```
对任意状态|ψ⟩和算子A, B：

ΔA · ΔB ≥ (1/2)|⟨[A,B]⟩|

其中：
- ΔA = √(⟨A²⟩ - ⟨A⟩²)（标准差）
- [A,B] = AB - BA（对易子）
```

**例子：位置-动量**：

```
A = X（位置）
B = P（动量）

对易关系：
[X,P] = iℏI

不确定性：
ΔX · ΔP ≥ ℏ/2
```

---

### 4.2 数学证明

**Cauchy-Schwarz不等式**：

```
|⟨Aψ|Bψ⟩|² ≤ ⟨Aψ|Aψ⟩⟨Bψ|Bψ⟩

应用：
- A → A - ⟨A⟩I
- B → B - ⟨B⟩I

得到不确定性原理
```

## 五、纠缠与Bell不等式

### 5.1 纠缠态

**定义**：

```
两粒子系统：
H = H₁ ⊗ H₂

纠缠态：
|Ψ⟩不能分解为|ψ₁⟩⊗|ψ₂⟩

例子：Bell态
|Φ⁺⟩ = (|00⟩ + |11⟩)/√2
```

**性质**：

- 非局域关联
- 测量一粒子影响另一粒子
- 违反经典直觉

---

### 5.2 Bell不等式

**经典隐变量理论**：

```
假设：存在隐变量λ决定测量结果
→ 相关性有上界

Bell不等式：
|E(a,b) - E(a,b')| + |E(a',b) + E(a',b')| ≤ 2
```

**量子违反**：

```
量子力学预测：
|E(a,b) - E(a,b')| + |E(a',b) + E(a',b')| = 2√2 > 2

实验验证（Aspect, 1982）：
量子力学正确
→ 非局域性是物理实在
```

## 七、量子场论

### 7.1 场算子的构造

**标量场**：

```
场算子：
φ(x) = ∫ (a(k)e^(ikx) + a*(k)e^(-ikx)) dk

其中：
- a(k)：产生算子
- a*(k)：湮灭算子
- 满足对易关系
```

**Fock空间**：

```
H = ⊕ₙ Hₙ

其中：
- H₀：真空
- Hₙ：n粒子态

基：
|k₁,...,kₙ⟩ = a*(k₁)...a*(kₙ)|0⟩
```
