# 泛函分析基础：希尔伯特空间的推广

## 一、Banach空间

### 1.1 定义

**Banach空间**：

```
赋范向量空间X是Banach空间 ⟺
X关于范数||·||完备

性质：
- 完备性
- 线性结构
- 拓扑结构
```
**例子**：

```
Lᵖ空间（1≤p≤∞）：
- Lᵖ(Ω) = {f | ∫|f|ᵖ < ∞}
- 范数：||f||_p = (∫|f|ᵖ)^(1/p)
- Banach空间

特殊：
- p=2：Hilbert空间
- p=∞：L∞空间
```
---

### 1.2 与Hilbert空间的关系

**包含关系**：

```
Hilbert空间 ⊂ Banach空间

区别：
- Hilbert：有内积
- Banach：只有范数

但：
- 所有Hilbert空间是Banach
- 但Banach空间不一定是Hilbert
```
---

## 二、对偶空间

### 2.1 定义

**对偶空间**：

```
Banach空间X的对偶：
X* = {f: X → ℝ | f有界线性}

范数：
||f|| = sup_{||x||≤1} |f(x)|

性质：
- Banach空间
- 对偶的对偶：X**
```
---

### 2.2 Riesz表示定理

**Hilbert空间情况**：

```
H* ≅ H

对应：
f ↔ y，其中f(x) = ⟨x,y⟩

性质：
- 等距同构
- 自然对应
```
**Banach空间**：

- 一般X* ≠ X
- 需要研究X**
- 自反性

---

## 三、弱拓扑

### 3.1 弱收敛

**定义**：

```
序列{xₙ}弱收敛到x ⟺
对所有f∈X*：
f(xₙ) → f(x)

记号：
xₙ ⇀ x（弱收敛）
```
**性质**：

- 弱极限唯一
- 弱收敛 ⟹ 有界
- 但弱拓扑 ≠ 范数拓扑

---

### 3.2 弱紧性

**定理**：

```
Banach空间X：
- 单位球弱紧（若X自反）
- 弱序列紧（若X可分）

应用：
- 存在性证明
- 变分法
- PDE
```
---

## 四、算子理论

### 4.1 有界算子

**B(X,Y)**：

```
有界线性算子：
T: X → Y

范数：
||T|| = sup_{||x||≤1} ||Tx||

性质：
- Banach空间
- 复合：||TS|| ≤ ||T|| ||S||
```
---

### 4.2 紧算子

**定义**：

```
T是紧算子 ⟺
T将单位球映射到预紧集

性质：
- 有界算子的子空间
- 极限（按范数）仍是紧
- 有限秩算子稠密
```
**应用**：

- 积分方程
- 特征值问题
- Fredholm理论

---

## 五、谱理论

### 5.1 Banach代数的谱

**Banach代数**：

```
A是Banach代数 ⟺
- Banach空间
- 有乘法
- ||ab|| ≤ ||a|| ||b||

谱：
σ(a) = {λ | a-λ不可逆}
```
---

### 5.2 Gelfand理论

**交换Banach代数**：

```
A交换 ⟺
A ≅ C(σ(A))

其中：
σ(A) = 极大理想空间

Gelfand变换：
a → â: σ(A) → ℂ
```
---

## 六、应用：偏微分方程

### 6.1 Sobolev空间

**定义**：

```
W^{k,p}(Ω) = {u | D^αu ∈ Lᵖ(Ω), |α|≤k}

范数：
||u||_{k,p} = (∑_{|α|≤k} ||D^αu||_p^p)^(1/p)

性质：
- Banach空间
- 自反（1<p<∞）
- 紧嵌入
```
---

### 6.2 弱解

**弱解理论**：

```
PDE：
Lu = f

弱解：
u ∈ W^{k,p}(Ω)使得：
对任意v ∈ C₀^∞(Ω)：
∫ u L*v = ∫ fv

优势：
- 存在性
- 正则性
- 数值方法
```
