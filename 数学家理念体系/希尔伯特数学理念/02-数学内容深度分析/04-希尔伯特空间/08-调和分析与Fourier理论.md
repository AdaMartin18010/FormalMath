## 一、Fourier级数

### 1.1 历史起源

**Fourier（1807）**：

```
问题：
热传导方程的解

方法：
- 函数展开为三角函数级数
- Fourier级数

f(x) = a₀/2 + ∑(aₙcos(nx) + bₙsin(nx))
```

**但**：

- 收敛性存疑
- 需要严格化
- Hilbert空间理论

---

### 1.2 L²理论

**Hilbert空间方法**：

```
L²[0,2π]：
- 内积：⟨f,g⟩ = (1/2π)∫ f(x)ḡ(x)dx
- Hilbert空间

正交基：
{e^(inx) : n ∈ ℤ}

Fourier系数：
f̂(n) = ⟨f, e^(inx)⟩
```

**Parseval等式**：

```
||f||² = ∑|f̂(n)|²

意义：
- 能量守恒
- 完备性
- 等距
```

---

## 二、Fourier变换

### 2.1 定义

**Fourier变换**：

```
f̂(ξ) = ∫_ℝ f(x)e^(-2πiξx)dx

逆变换：
f(x) = ∫_ℝ f̂(ξ)e^(2πiξx)dξ

性质：
- 线性
- 卷积 ↔ 乘积
- 微分 ↔ 乘法
```

---

### 2.2 Plancherel定理

**定理**：

```
f ∈ L²(ℝ) ⟹ f̂ ∈ L²(ℝ)

且：
||f||_{L²} = ||f̂||_{L²}

意义：
- L²上的等距
- 可逆
- 完备性
```

## 三、调和函数

### 3.1 定义

**调和函数**：

```
u: Ω → ℝ调和 ⟺
Δu = 0

其中：
Δ = ∑ ∂²/∂xᵢ²（Laplace算子）

性质：
- 平均值性质
- 最大值原理
- 解析性
```

---

### 3.2 应用

**Dirichlet问题**：

```
Δu = 0（在Ω内）
u = f（在∂Ω上）

解：
- 存在唯一
- 变分法
- 积分表示
```

---

## 四、Hardy空间

### 4.1 定义

**H²空间**：

```
单位圆盘D上的解析函数：
H² = {f解析 | sup_{r<1} ∫|f(re^(iθ))|²dθ < ∞}

性质：
- Hilbert空间
- 边界值
- 与L²关联
```

---

### 4.2 应用

**函数论**：

```
应用：
- 插值理论
- 控制理论
- 算子理论
```

## 五、小波分析

### 5.1 小波基

**定义**（1980s）：

```
小波：
ψ_{j,k}(x) = 2^(j/2)ψ(2^jx - k)

性质：
- 局部化
- 多分辨率
- 正交基
```

**优势**：

- 时频局部化
- 比Fourier基灵活
- 应用广泛

---

### 5.2 应用

**信号处理**：

```
应用：
- 图像压缩
- 去噪
- 特征提取

数学：
- 函数空间
- 逼近理论
- 数值分析
```
