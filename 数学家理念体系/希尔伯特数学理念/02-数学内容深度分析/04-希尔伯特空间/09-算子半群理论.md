# 算子半群理论：从Hilbert空间到PDE

## 一、强连续半群

### 1.1 定义

**C₀半群**：

```
{T(t) : t ≥ 0}是C₀半群 ⟺
1. T(0) = I
2. T(s+t) = T(s)T(t)
3. lim_{t→0} T(t)x = x（强连续）

生成元：
A = lim_{t→0} (T(t)-I)/t
```

**例子**：

```
热方程：
T(t) = e^(tΔ)

其中：
Δ：Laplace算子
```

---

### 1.2 Hille-Yosida定理

**定理**：

```
A生成C₀半群 ⟺
A是闭稠定算子且：
存在M,ω使得：
||(λ-A)⁻¹|| ≤ M/(λ-ω)（λ>ω）

半群：
T(t) = lim_{n→∞} (I - tA/n)^(-n)
```

**应用**：

- 存在性证明
- 唯一性
- 正则性

---

## 二、应用：偏微分方程

### 2.1 抛物型方程

**热方程**：

```
u_t = Δu
u(0) = u₀

解：
u(t) = T(t)u₀

其中：
T(t) = e^(tΔ)
```

**性质**：

- 存在唯一
- 正则性
- 渐近行为

---

### 2.2 双曲型方程

**波方程**：

```
u_{tt} = Δu
u(0) = u₀, u_t(0) = v₀

解：
u(t) = cos(t√(-Δ))u₀ + sin(t√(-Δ))/√(-Δ) v₀

半群：
T(t) = e^(it√(-Δ))
```

---

## 三、解析半群

### 3.1 定义

**解析半群**：

```
T(t)可延拓到复平面扇形：
- 解析性
- 更强的正则性
- 应用广泛
```

**生成元**：

```
A生成解析半群 ⟺
A是扇形算子：
- 谱在扇形内
- 有界性
```

---

### 3.2 应用

**椭圆算子**：

```
A = -Δ + V

性质：
- 生成解析半群
- 正则性
- 应用：PDE
```

---

## 四、压缩半群

### 4.1 定义

**压缩半群**：

```
||T(t)|| ≤ 1

生成元：
A是耗散算子：
Re⟨Ax,x⟩ ≤ 0
```

**应用**：

- 概率过程
- 随机PDE
- 物理

---

### 4.2 Lumer-Phillips定理

**定理**：

```
A生成压缩半群 ⟺
A是闭稠定耗散算子且：
ran(λ-A) = H（某个λ>0）
```

---

## 五、分数次幂

### 5.1 定义

**分数次幂**：

```
A^α（0<α<1）：

A^α = (1/Γ(-α))∫₀^∞ t^{-α-1}(e^{-tA}-I)dt

性质：
- 插值
- 正则性
- 应用
```

---

### 5.2 应用

**分数阶PDE**：

```
u_t = -(-Δ)^α u

其中：
(-Δ)^α：分数Laplace算子

应用：
- 非局部扩散
- 金融
- 物理
```

## 六、现代发展

### 6.1 随机PDE

**随机热方程**：

```
du = Δu dt + σ dW

解：
- 随机半群
- Itô积分
- 正则性
```

### 6.2 非自治半群

**演化族**：

{U(t,s) : t≥s}：

- 应用：非自治PDE
