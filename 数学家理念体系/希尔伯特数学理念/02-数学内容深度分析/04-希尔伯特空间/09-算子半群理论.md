## 一、强连续半群

### 1.1 定义

**C₀半群的数学定义**：

设 $H$ 是Banach空间（或Hilbert空间），$\{T(t)\}_{t \\\geq 0}$ 是 $B(H)$ 中的一族有界线性算子。$\{T(t)\}$ 是**强连续半群**（C₀半群），若满足：

1. **单位元**：$T(0) = I$（恒等算子）
2. **半群性质**：$T(s+t) = T(s)T(t)$ 对所有 $s, t \\\geq 0$
3. **强连续性**：$\lim_{t \to 0^+} T(t)x = x$ 对所有 $x \in H$（在强算子拓扑下）

**生成元**：

$$Ax = \lim_{t \to 0^+} \frac{T(t)x - x}{t}$$

定义域：
$$D(A) = \\\leqft\{x \in H : \lim_{t \to 0^+} \frac{T(t)x - x}{t} \text{ 存在}\right\}$$

**具体例子**：

**例1：热方程半群**

在 $H = L^2(\mathbb{R}^n)$ 上：
- **生成元**：$A = \Delta$（Laplace算子）
- **半群**：$T(t) = e^{t\Delta}$（热半群）
- **作用**：$(T(t)u_0)(x) = \int_{\mathbb{R}^n} K_t(x-y) u_0(y) \, dy$
- **热核**：$K_t(x) = \frac{1}{(4\pi t)^{n/2}} e^{-|x|^2/(4t)}$

**验证**：
- $T(0) = I$（显然）
- $T(s+t) = e^{(s+t)\Delta} = e^{s\Delta} e^{t\Delta} = T(s)T(t)$（指数性质）
- 强连续性：$\lim_{t \to 0^+} T(t)u_0 = u_0$（热核的近似恒等性质）

**例2：Schrödinger半群**

在 $H = L^2(\mathbb{R}^n)$ 上：
- **生成元**：$A = i\Delta$（纯虚数倍Laplace算子）
- **半群**：$T(t) = e^{it\Delta}$（Schrödinger半群）
- **作用**：$(T(t)\psi_0)(x) = \int_{\mathbb{R}^n} K_t(x-y) \psi_0(y) \, dy$
- **Schrödinger核**：$K_t(x) = \frac{1}{(4\pi it)^{n/2}} e^{i|x|^2/(4t)}$

**性质**：
- $T(t)$ 是酉算子（保持 $L^2$ 范数）
- $\|T(t)\psi_0\| = \|\psi_0\|$（概率守恒）

**例3：有限维情况**

在 $H = \mathbb{C}^n$ 上：
- **生成元**：$A$ 是 $n \times n$ 矩阵
- **半群**：$T(t) = e^{tA}$（矩阵指数）
- **作用**：$T(t)x = e^{tA}x = \sum_{k=0}^\infty \frac{(tA)^k}{k!} x$

**验证**：
- $T(0) = e^0 = I$
- $T(s+t) = e^{(s+t)A} = e^{sA} e^{tA} = T(s)T(t)$
- 强连续性：$\lim_{t \to 0^+} e^{tA} = I$（矩阵指数连续性）

### 1.2 Hille-Yosida定理

**定理**：

```
A生成C₀半群 ⟺
A是闭稠定算子且：
存在M,ω使得：
||(λ-A)⁻¹|| ≤ M/(λ-ω)（λ>ω）

半群：
T(t) = lim_{n→∞} (I - tA/n)^(-n)
```

**应用**：

- 存在性证明
- 唯一性
- 正则性

## 二、应用：偏微分方程

### 2.1 抛物型方程

**热方程**：

```
u_t = Δu
u(0) = u₀

解：
u(t) = T(t)u₀

其中：
T(t) = e^(tΔ)
```

**性质**：

- 存在唯一
- 正则性
- 渐近行为

---

### 2.2 双曲型方程

**波方程**：

```
u_{tt} = Δu
u(0) = u₀, u_t(0) = v₀

解：
u(t) = cos(t√(-Δ))u₀ + sin(t√(-Δ))/√(-Δ) v₀

半群：
T(t) = e^(it√(-Δ))
```

## 三、解析半群

### 3.1 定义

**解析半群**：

```
T(t)可延拓到复平面扇形：
- 解析性
- 更强的正则性
- 应用广泛
```

**生成元**：

```
A生成解析半群 ⟺
A是扇形算子：
- 谱在扇形内
- 有界性
```

---

### 3.2 应用

**椭圆算子**：

```
A = -Δ + V

性质：
- 生成解析半群
- 正则性
- 应用：PDE
```

## 四、压缩半群

### 4.1 定义

**压缩半群**：

```
||T(t)|| ≤ 1

生成元：
A是耗散算子：
Re⟨Ax,x⟩ ≤ 0
```

**应用**：

- 概率过程
- 随机PDE
- 物理

---

### 4.2 Lumer-Phillips定理

**定理**：

```
A生成压缩半群 ⟺
A是闭稠定耗散算子且：
ran(λ-A) = H（某个λ>0）
```

## 五、分数次幂

### 5.1 定义

**分数次幂**：

```
A^α（0<α<1）：

A^α = (1/Γ(-α))∫₀^∞ t^{-α-1}(e^{-tA}-I)dt

性质：
- 插值
- 正则性
- 应用
```

---

### 5.2 应用

**分数阶PDE**：

```
u_t = -(-Δ)^α u

其中：
(-Δ)^α：分数Laplace算子

应用：
- 非局部扩散
- 金融
- 物理
```

## 九、数学公式总结

### 核心公式

1. **C₀半群定义**：
   - $T(0) = I$
   - $T(s+t) = T(s)T(t)$
   - $\lim_{t \to 0^+} T(t)x = x$（强连续）

2. **生成元**：
   $$Ax = \lim_{t \to 0^+} \frac{T(t)x - x}{t}$$

3. **Hille-Yosida定理**：
   $$A \text{ 生成C₀半群 } \iff \|(\lambda - A)^{-1}\| \\\leq \frac{M}{\lambda - \omega} \text{ 对 } \lambda > \omega$$

4. **Trotter公式**：
   $$T(t) = \lim_{n \to \infty} \\\leqft(I - \frac{tA}{n}\right)^{-n}$$

5. **热方程解**：
   $$u(t) = T(t)u_0 = e^{t\Delta} u_0$$

6. **热核**：
   $$K_t(x) = \frac{1}{(4\pi t)^{n/2}} e^{-|x|^2/(4t)}$$

7. **波方程解**：
   $$u(t) = \cos(t\sqrt{-\Delta}) u_0 + \frac{\sin(t\sqrt{-\Delta})}{\sqrt{-\Delta}} v_0$$

8. **压缩半群**：
   $$\|T(t)\| \\\leq 1 \iff \text{Re}\langle Ax, x \rangle \\\leq 0$$

9. **Lumer-Phillips定理**：
   $$A \text{ 生成压缩半群 } \iff A \text{ 耗散且 } \text{ran}(\lambda - A) = H \text{ 对某个 } \lambda > 0$$

10. **分数次幂**：
    $$A^\alpha = \frac{1}{\Gamma(-\alpha)} \int_0^\infty t^{-\alpha-1}(e^{-tA} - I) \, dt$$
