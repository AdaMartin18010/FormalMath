## 一、积分方程的提出

### 1.1 19世纪末的问题

**Fredholm积分方程**（1900）：

```
第一类：
∫₀¹ K(x,y)f(y)dy = g(x)

第二类：
f(x) = g(x) + λ∫₀¹ K(x,y)f(y)dy

其中：
- K(x,y)：核函数
- f(x)：未知函数
- g(x)：已知函数
- λ：参数
```

**物理背景**：

- 振动问题
- 热传导
- 势理论

---

### 1.2 希尔伯特的介入

**问题**：

- Fredholm给出部分解（1903）
- 但理论不完整
- 需要系统化

**希尔伯特的方法**（1904-1912）：

- 引入无限维空间
- 谱理论
- 算子理论

## 二、从有限维到无限维

### 2.1 矩阵类比

**有限维情况**：

```
线性方程组：
Ax = b

其中A是n×n矩阵
x, b是n维向量

解：
x = A⁻¹b（若A可逆）
```

**无限维推广**：

```
积分方程：
(Kf)(x) = ∫ K(x,y)f(y)dy = g(x)

其中K是"无限矩阵"
f, g是"无限维向量"

问题：
如何定义K⁻¹？
```

---

### 2.2 对称核的特殊情况

**对称核**：

```
K(x,y) = K(y,x)

对应：
自伴矩阵A = A*
```

**谱分解**（Hilbert-Schmidt）：

```
K有可数特征值{λₙ}
特征函数{vₙ(x)}形成L²[0,1]的基

K = ∑λₙ⟨·,vₙ⟩vₙ

解：
f = ∑(1/λₙ)⟨g,vₙ⟩vₙ
```

---

## 三、Hilbert-Schmidt理论

### 3.1 紧算子的性质

**定义**：

```
K是紧算子 ⟺
K将单位球映射到预紧集

等价：
对任意有界序列{fₙ}，
{Kfₙ}有收敛子列
```

**积分算子的紧性**：

```
若K(x,y)连续
    ↓
K是L²[0,1]上的紧算子
    ↓
有可数谱
```

---

### 3.2 特征值理论

**定理**（Hilbert-Schmidt）：

```
若K是紧自伴算子，则：

1. σ(K) = σₚ(K)（只有特征值）
2. 特征值可数：λ₁, λ₂, ...
3. λₙ → 0
4. 特征函数{vₙ}形成H的正交基
5. K = ∑λₙ⟨·,vₙ⟩vₙ
```

**应用**：

```
解积分方程：
f = g + λKf

解：
f = ∑(1/(1-λλₙ))⟨g,vₙ⟩vₙ

条件：λ ≠ 1/λₙ（对所有n）
```

---

## 四、Fredholm择一定理

### 4.1 定理陈述

**Fredholm择一定理**：

```
方程：(I-λK)f = g

要么：
情况1：有唯一解（任意g）
       ⟺ 齐次方程(I-λK)f=0只有零解
       ⟺ λ不是特征值

情况2：齐次方程有非零解
       ⟺ λ是特征值
       ⟺ 非齐次方程有解 ⟺ g与所有特征函数正交
```

---

### 4.2 证明思路

**关键步骤**：

```
步骤1：K紧 → K有可数特征值
步骤2：若λ不是特征值，则(I-λK)可逆
步骤3：若λ是特征值，分析解空间
步骤4：应用Riesz理论
```

**与有限维的类比**：

```
有限维：
Ax = b有解 ⟺ b ∈ Im(A)

无限维：
类似，但需考虑闭性
```

## 五、应用：Sturm-Liouville问题

### 5.1 边值问题

**问题**：

```
-(py')' + qy = λy
边界条件：y(a) = y(b) = 0

其中：
- p(x) > 0
- q(x) ≥ 0
```

**算子形式**：

```
L = -d/dx(p d/dx) + q

问题：Lf = λf
```

---

### 5.2 格林函数方法

**格林函数**：

```
G(x,ξ)满足：
LG(x,ξ) = δ(x-ξ)
边界条件

性质：
- G对称：G(x,ξ) = G(ξ,x)
- 连续核
```

**积分方程形式**：

```
原问题：Lf = λf
    ↓
等价：f = λ∫ G(x,ξ)f(ξ)dξ
    ↓
积分方程（第二类）
```

**应用Hilbert-Schmidt理论**：

```
G是紧自伴算子
    ↓
有可数特征值{λₙ}
    ↓
特征函数{yₙ}形成L²[a,b]的基
    ↓
原问题的解
```

## 六、现代发展

### 6.1 奇异积分算子

**Hilbert变换**：

```
(Hf)(x) = (1/π) P.V. ∫ f(y)/(x-y) dy

性质：
- 非紧算子
- 但有有界性
- 应用：信号处理
```

**Calderón-Zygmund理论**：

```
奇异积分算子的Lᵖ理论
    ↓
现代调和分析
```

---

### 6.2 随机积分方程

**随机积分方程**：

```
f(x,ω) = g(x,ω) + ∫ K(x,y)f(y,ω)dy

其中ω是随机参数

应用：
- 随机过程
- 金融数学
- 物理中的噪声
```
