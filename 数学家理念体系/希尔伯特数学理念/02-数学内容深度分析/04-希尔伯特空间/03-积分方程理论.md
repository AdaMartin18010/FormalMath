## 一、积分方程的提出

### 1.1 19世纪末的问题

**Fredholm积分方程（1900）**：

**第一类积分方程**：
$$\int_0^1 K(x,y) f(y) \, dy = g(x), \quad x \in [0,1]$$

**第二类积分方程**：
$$f(x) = g(x) + \lambda \int_0^1 K(x,y) f(y) \, dy, \quad x \in [0,1]$$

其中：
- $K(x,y)$：核函数（已知）
- $f(x)$：未知函数
- $g(x)$：已知函数
- $\lambda$：参数（通常为复数）

**物理背景**：

**例1：振动问题**
- 弦的振动：$u(x,t)$ 满足波动方程
- 通过分离变量，得到积分方程
- 核函数：Green函数 $G(x,y)$

**例2：热传导问题**
- 热传导方程：$\frac{\partial u}{\partial t} = \Delta u$
- 通过积分变换，得到积分方程
- 核函数：热核 $K(x,y,t) = \frac{1}{\sqrt{4\pi t}} e^{-\frac{(x-y)^2}{4t}}$

**例3：势理论**
- 静电势：$V(x) = \int \frac{\rho(y)}{|x-y|} \, dy$
- 电荷分布 $\rho(y)$ 未知，通过边界条件确定
- 核函数：$\frac{1}{|x-y|}$（Coulomb势）

**问题**：

- 需要系统化

**希尔伯特的方法**（1904-1912）：

## 二、从有限维到无限维

### 2.1 矩阵类比

**有限维情况**：

线性方程组：
$$Ax = b$$

其中：
- $A$ 是 $n \times n$ 矩阵
- $x, b \in \mathbb{C}^n$ 是 $n$ 维向量

**解**：
$$x = A^{-1}b \quad \text{（若 $A$ 可逆）}$$

**具体例子**：

**例1：$2 \times 2$ 系统**
$$\begin{pmatrix} 1 & 2 \\ 3 & 4 \end{pmatrix} \begin{pmatrix} x_1 \\ x_2 \end{pmatrix} = \begin{pmatrix} 5 \\ 6 \end{pmatrix}$$

解：$x_1 = -4$，$x_2 = 4.5$

**例2：对角矩阵**
$$\begin{pmatrix} 2 & 0 \\ 0 & 3 \end{pmatrix} \begin{pmatrix} x_1 \\ x_2 \end{pmatrix} = \begin{pmatrix} 4 \\ 9 \end{pmatrix}$$

解：$x_1 = 2$，$x_2 = 3$

**无限维推广**：

积分方程：
$$(Kf)(x) = \int_0^1 K(x,y) f(y) \, dy = g(x)$$

**类比**：
- $K(x,y)$ 对应矩阵 $A_{ij}$（"无限矩阵"）
- $f(y)$ 对应向量 $x_j$（"无限维向量"）
- $g(x)$ 对应向量 $b_i$（"无限维向量"）
- 积分 $\int_0^1 \cdot \, dy$ 对应求和 $\sum_{j=1}^n$

**问题**：
- 如何定义 $K^{-1}$（逆算子）？
- 何时 $K$ 可逆？
- 如何计算 $(K - \lambda I)^{-1}$？

**希尔伯特的突破**：
- 将积分算子视为 $L^2[0,1]$ 上的有界线性算子
- 使用谱理论研究可逆性
- 建立算子理论框架

---

### 2.2 对称核的特殊情况

**对称核**：

```
K(x,y) = K(y,x)

对应：
自伴矩阵A = A*
```

**谱分解**（Hilbert-Schmidt）：

```
K有可数特征值{λₙ}
特征函数{vₙ(x)}形成L²[0,1]的基

K = ∑λₙ⟨·,vₙ⟩vₙ

解：
f = ∑(1/λₙ)⟨g,vₙ⟩vₙ
```

## 三、Hilbert-Schmidt理论

### 3.1 紧算子的性质

**定义**：

```
K是紧算子 ⟺
K将单位球映射到预紧集

等价：
对任意有界序列{fₙ}，
{Kfₙ}有收敛子列
```

**积分算子的紧性**：

```
若K(x,y)连续
    ↓
K是L²[0,1]上的紧算子
    ↓
有可数谱
```

---

### 3.2 特征值理论

**定理（Hilbert-Schmidt）**：

> 若 $K: L^2[0,1] \to L^2[0,1]$ 是紧自伴积分算子，则：
>
> 1. **纯点谱**：$\sigma(K) = \sigma_p(K)$（只有特征值，无连续谱）
> 2. **特征值可数**：$\{\lambda_n\}_{n=1}^{\infty}$，且 $\lambda_n \to 0$（当 $n \to \infty$）
> 3. **特征函数正交基**：对应的特征函数 $\{v_n(x)\}_{n=1}^{\infty}$ 形成 $L^2[0,1]$ 的标准正交基
> 4. **谱分解**：
>    $$(Kf)(x) = \sum_{n=1}^{\infty} \lambda_n \langle f, v_n \rangle v_n(x)$$
>    对所有 $f \in L^2[0,1]$ 成立。

**具体例子**：

**例1：对称核 $K(x,y) = \min(x,y)$**

特征值问题：
$$\int_0^1 \min(x,y) v(y) \, dy = \lambda v(x)$$

**解**：
- 特征值：$\lambda_n = \frac{1}{(n\pi)^2}$，$n = 1, 2, 3, \ldots$
- 特征函数：$v_n(x) = \sqrt{2} \sin(n\pi x)$

**验证**：
$$\int_0^1 \min(x,y) \sqrt{2} \sin(n\pi y) \, dy = \frac{1}{(n\pi)^2} \sqrt{2} \sin(n\pi x)$$

**例2：核 $K(x,y) = xy$**

特征值问题：
$$\int_0^1 xy \cdot v(y) \, dy = \lambda v(x)$$

**解**：
- 特征值：$\lambda_1 = \frac{1}{3}$，$\lambda_n = 0$（$n \\\geq 2$）
- 特征函数：$v_1(x) = \sqrt{3}x$，其他特征函数任意（在 $\{x\}^{\perp}$ 中）

**应用：解第二类积分方程**：

**问题**：
$$f(x) = g(x) + \lambda \int_0^1 K(x,y) f(y) \, dy$$

**解（使用谱分解）**：

假设 $K$ 是紧自伴算子，有特征值 $\{\lambda_n\}$ 和特征函数 $\{v_n\}$，则：

$$f(x) = \sum_{n=1}^{\infty} \frac{1}{1 - \lambda\lambda_n} \langle g, v_n \rangle v_n(x)$$

**条件**：$\lambda \neqqqqqq \frac{1}{\lambda_n}$ 对所有 $n$ 成立（否则无解或解不唯一）。

**具体计算例子**：

设 $K(x,y) = \min(x,y)$，$g(x) = x$，$\lambda = 1$：

1. 计算 Fourier系数：$\langle g, v_n \rangle = \int_0^1 x \cdot \sqrt{2} \sin(n\pi x) \, dx = \frac{(-1)^{n+1} \sqrt{2}}{n\pi}$
2. 应用公式：$f(x) = \sum_{n=1}^{\infty} \frac{1}{1 - (n\pi)^{-2}} \cdot \frac{(-1)^{n+1} \sqrt{2}}{n\pi} \cdot \sqrt{2} \sin(n\pi x)$
3. 简化：$f(x) = \sum_{n=1}^{\infty} \frac{2(-1)^{n+1}}{n\pi(1 - (n\pi)^{-2})} \sin(n\pi x)$

## 四、Fredholm择一定理

### 4.1 定理陈述

**Fredholm择一定理**：

```
方程：(I-λK)f = g

要么：
情况1：有唯一解（任意g）
       ⟺ 齐次方程(I-λK)f=0只有零解
       ⟺ λ不是特征值

情况2：齐次方程有非零解
       ⟺ λ是特征值
       ⟺ 非齐次方程有解 ⟺ g与所有特征函数正交
```

---

### 4.2 证明思路

**关键步骤**：

```
步骤1：K紧 → K有可数特征值
步骤2：若λ不是特征值，则(I-λK)可逆
步骤3：若λ是特征值，分析解空间
步骤4：应用Riesz理论
```

**与有限维的类比**：

```
有限维：
Ax = b有解 ⟺ b ∈ Im(A)

无限维：
类似，但需考虑闭性
```

## 五、应用：Sturm-Liouville问题

### 5.1 边值问题

**问题**：

```
-(py')' + qy = λy
边界条件：y(a) = y(b) = 0

其中：
- p(x) > 0
- q(x) ≥ 0
```

**算子形式**：

```
L = -d/dx(p d/dx) + q

问题：Lf = λf
```

---

### 5.2 格林函数方法

**格林函数**：

```
G(x,ξ)满足：
LG(x,ξ) = δ(x-ξ)
边界条件

性质：
- G对称：G(x,ξ) = G(ξ,x)
- 连续核
```

**积分方程形式**：

```
原问题：Lf = λf
    ↓
等价：f = λ∫ G(x,ξ)f(ξ)dξ
    ↓
积分方程（第二类）
```

**应用Hilbert-Schmidt理论**：

```
G是紧自伴算子
    ↓
有可数特征值{λₙ}
    ↓
特征函数{yₙ}形成L²[a,b]的基
    ↓
原问题的解
```

## 六、现代发展

### 6.1 奇异积分算子

**Hilbert变换**：

```
(Hf)(x) = (1/π) P.V. ∫ f(y)/(x-y) dy

性质：
- 非紧算子
- 但有有界性
- 应用：信号处理
```

**Calderón-Zygmund理论**：

```
奇异积分算子的Lᵖ理论
    ↓
现代调和分析
```

---

### 6.2 随机积分方程

**随机积分方程**：

```
f(x,ω) = g(x,ω) + ∫ K(x,y)f(y,ω)dy

其中ω是随机参数

应用：
- 随机过程
- 金融数学
- 物理中的噪声
```

## 九、数学公式总结

### 核心公式

1. **第一类积分方程**：
   $$\int_0^1 K(x,y) f(y) \, dy = g(x)$$

2. **第二类积分方程**：
   $$f(x) = g(x) + \lambda \int_0^1 K(x,y) f(y) \, dy$$

3. **积分算子**：
   $$(Kf)(x) = \int_0^1 K(x,y) f(y) \, dy$$

4. **Hilbert-Schmidt谱分解**：
   $$(Kf)(x) = \sum_{n=1}^{\infty} \lambda_n \langle f, v_n \rangle v_n(x)$$

5. **第二类方程的解**：
   $$f(x) = \sum_{n=1}^{\infty} \frac{1}{1 - \lambda\lambda_n} \langle g, v_n \rangle v_n(x)$$

6. **Fredholm择一定理**：
   方程 $(I - \lambda K)f = g$ 要么有唯一解，要么齐次方程 $(I - \lambda K)f = 0$ 有非零解。

7. **Green函数**：
   $$LG(x,\xi) = \delta(x - \xi)$$
   其中 $L$ 是微分算子，$\delta$ 是Dirac delta函数。

8. **Sturm-Liouville特征值问题**：
   $$-(py')' + qy = \lambda y$$
   边界条件：$y(a) = y(b) = 0$。
