## 一、完全解决的问题（10个）

### 第3问：等底等高四面体（1900年解决）

**问题**：等底等高的两个四面体是否必等体积？

**Dehn的否定答案**（希尔伯特提出同年）：

```
Dehn不变量：
D(P) = ∑(边长ᵢ) ⊗ (二面角ᵢ)
（张量积在ℝ ⊗ ℝ/πℚ中）

定理：
D(P₁) = D(P₂) 是P₁≅P₂的必要条件

反例构造：
正方体C：D(C) ≠ 0
正四面体T：D(T) = 0

结论：
即使等底等高，C和T不能剖分等价
```

**深刻意义**：

- 三维与二维的**本质差异**
- 并非所有平面结论都能推广
- 开创**剖分不变量**理论

---

### 第5问：连续群的可微性（1952年解决）

**问题**：拓扑群是否自动有Lie群结构？

**解决历程**：

```
1933：von Neumann
- 证明：紧群情况
- 方法：Haar测度+表示论

1952：Gleason-Montgomery-Zippin
- 证明：局部欧氏连续群
- 结论：自动可微（Hilbert猜想正确）

关键技术：
- Hilbert第五问题 ⟺ 无小子群定理
- 利用：紧致性、逼近定理
```

**现代应用**：

- 李群表示论的基础
- 规范场论的合法性
- 几何化纲领（Perelman）

---

### 第7问：数的超越性（1934年解决）

**问题**：\(2^{\sqrt{2}}\)等数是否超越代数数？

**Gelfond-Schneider定理**（1934）：

```
定理：
若α, β是代数数，α≠0,1，β无理，
则α^β是超越数

特殊情况：
- 2^√2 超越（α=2, β=√2）
- e^π 超越（α=e, β=iπ，用Euler公式）
```

**证明方法**：

- 辅助函数构造
- 超越数论技术
- 贝克方法（后续发展）

---

### 第11问：二次型分类（1920s解决）

**问题**：有理系数二次型的算术理论

**Hasse-Minkowski定理**：

```
局部-全局原理：
二次型在ℚ上有解
    ⟺
在所有局部域ℚp（包括ℚ∞=ℝ）上有解

例子：
方程 x² + y² = 3z²
- 在ℚ₂上无解（2-adic）
- 故在ℚ上无解
```

**意义**：

- 开创局部-全局方法
- 类域论的应用
- 代数数论基础

---

### 第17问：有理函数的平方和表示（1927年解决）

**问题**：非负有理函数能否表为平方和的商？

**Artin定理**：

```
定理：
若f(x₁,...,xₙ)∈ℝ(x₁,...,xₙ)，且f≥0（实点上），
则f = (∑gᵢ²)/(∑hⱼ²)

证明方法：
实闭域理论
Sturm定理的推广

应用：
- 半定规划（SDP）
- 优化理论
- 控制论
```

### 第19问：变分解的解析性（1950s解决）

**问题**：变分问题的解是否总是实解析函数？

```
定理：
椭圆型PDE的弱解是Hölder连续的
进而（在好的条件下）解析

应用：
- PDE正则性理论

```
