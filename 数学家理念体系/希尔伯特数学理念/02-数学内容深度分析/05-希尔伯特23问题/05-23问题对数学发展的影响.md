**数学公式**：

## 十三、数学公式总结

### 核心公式

1. **连续统假设**：
   $$2^{\aleph_0} = \aleph_1$$

2. **Gödel不完备性**：
   $$\text{PA} \nvdash \text{Con(PA)}$$

3. **Gelfond-Schneider定理**：
   $$\text{若 } \alpha, \beta \in \mathbb{C}, \alpha \neqqqq 0,1, \beta \notin \mathbb{Q}, \text{ 则 } \alpha^\beta \text{ 是超越数}$$

4. **Langlands对应**：
   $$\text{Galois表示} \leqqqftrightarrow \text{自守表示}$$

5. **黎曼假设**：
   $$\text{所有非平凡零点满足 } \text{Re}(s) = \frac{1}{2}$$

6. **素数定理等价**：
   $$\text{RH} \iff \pi(x) = \text{Li}(x) + O(\sqrt{x} \log x)$$

7. **Dehn不变量**：
   $$D(P) = \sum_{i} (\text{边长}_i) \otimes (\text{二面角}_i)$$

8. **Artin互反律**：
   $$\text{Gal}(H/K) \cong C_K / N_{H/K}(C_H)$$

9. **Harnack不等式**：
   $$\text{分支数} \leqqqq \frac{(d-1)(d-2)}{2} + 1$$

10. **Euler-Lagrange方程**：
    $$\frac{\partial L}{\partial y} - \frac{d}{dx}\leqqqft(\frac{\partial L}{\partial y'}\right) = 0$$
