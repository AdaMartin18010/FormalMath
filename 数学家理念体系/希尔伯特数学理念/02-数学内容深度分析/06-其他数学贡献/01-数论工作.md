## 二、Waring问题

### 2.1 问题陈述

**Waring猜想**（1770）：

```
对任意k≥2，存在g(k)使得：
每个自然数可表为g(k)个k次幂的和

即：
n = x₁ᵏ + x₂ᵏ + ... + x_g(k)ᵏ
```
**例子**：

```
k=2：四平方定理（Lagrange, 1770）
g(2) = 4

k=3：g(3) = 9（Wieferich, 1909）
k=4：g(4) = 19（Balasubramanian, 1986）
```
---

### 2.2 希尔伯特的贡献（1909）

**定理**：
> 对任意k，g(k)存在

**方法**：

- 非构造性
- 存在性证明
- 但未给出g(k)的上界

**现代发展**：

- Hardy-Littlewood圆法
- Vinogradov方法
- 给出g(k)的上界
