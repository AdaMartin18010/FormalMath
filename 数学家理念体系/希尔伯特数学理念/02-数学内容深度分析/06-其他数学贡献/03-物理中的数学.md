## 一、相对论的数学基础

### 1.1 广义相对论的场方程

**Einstein场方程**（1915）：

```
G_μν = 8πG/c⁴ · T_μν

其中：
- G_μν：Einstein张量
- T_μν：能量-动量张量
```

**希尔伯特的贡献**（1915）：

```
变分原理：
- 作用量S = ∫ R√g d⁴x
- 场方程 = δS/δg_μν = 0

方法：
- 变分法
- 张量分析
- 微分几何
```

**历史**：

- Einstein和Hilbert几乎同时得到
- Hilbert用变分法更优雅
- 但物理解释归Einstein

---

### 1.2 作用量原理

**Hilbert作用量**：

希尔伯特在1915年独立推导出Einstein场方程，使用变分原理：

$$S[g] = \int (R - 2\Lambda) \sqrt{-g} \, d^4x$$

其中：

- $R = g^{\mu\nu} R_{\mu\nu}$ 是标量曲率（Ricci标量）
- $R_{\mu\nu}$ 是Ricci张量
- $\Lambda$ 是宇宙常数
- $g = \det(g_{\mu\nu})$ 是度规的行列式
- $g_{\mu\nu}$ 是度规张量

**变分推导的数学细节**：

**步骤1：度规的变分**

对度规 $g_{\mu\nu} \to g_{\mu\nu} + \delta g_{\mu\nu}$，度规的逆 $g^{\mu\nu}$ 的变分为：
$$\delta g^{\mu\nu} = -g^{\mu\alpha} g^{\nu\beta} \delta g_{\alpha\beta}$$

**步骤2：Christoffel符号的变分**

Christoffel符号：
$$\Gamma^\lambda_{\mu\nu} = \frac{1}{2} g^{\lambda\rho} (\partial_\mu g_{\nu\rho} + \partial_\nu g_{\mu\rho} - \partial_\rho g_{\mu\nu})$$

其变分为：
$$\delta \Gamma^\lambda_{\mu\nu} = \frac{1}{2} g^{\lambda\rho} (\nabla_\mu \delta g_{\nu\rho} + \nabla_\nu \delta g_{\mu\rho} - \nabla_\rho \delta g_{\mu\nu})$$

其中 $\nabla_\mu$ 是协变导数。

**步骤3：Ricci张量的变分**

Ricci张量：
$$R_{\mu\nu} = \partial_\lambda \Gamma^\lambda_{\mu\nu} - \partial_\nu \Gamma^\lambda_{\mu\lambda} + \Gamma^\lambda_{\mu\nu} \Gamma^\sigma_{\lambda\sigma} - \Gamma^\lambda_{\mu\sigma} \Gamma^\sigma_{\nu\lambda}$$

其变分为：
$$\delta R_{\mu\nu} = \nabla_\lambda \delta \Gamma^\lambda_{\mu\nu} - \nabla_\nu \delta \Gamma^\lambda_{\mu\lambda}$$

**步骤4：作用量的变分**

作用量的变分：
$$\delta S = \int \left[\delta R \sqrt{-g} + R \delta(\sqrt{-g}) - 2\Lambda \delta(\sqrt{-g})\right] d^4x$$

其中：

- $\delta(\sqrt{-g}) = -\frac{1}{2} \sqrt{-g} g^{\mu\nu} \delta g_{\mu\nu}$
- $\delta R = g^{\mu\nu} \delta R_{\mu\nu} + R_{\mu\nu} \delta g^{\mu\nu}$

**步骤5：Einstein场方程**

通过分部积分和边界条件，得到：
$$\delta S = \int \left(R_{\mu\nu} - \frac{1}{2} R g_{\mu\nu} + \Lambda g_{\mu\nu}\right) \delta g^{\mu\nu} \sqrt{-g} \, d^4x$$

令 $\delta S = 0$ 对所有 $\delta g^{\mu\nu}$ 成立，得到**Einstein场方程**：
$$R_{\mu\nu} - \frac{1}{2} R g_{\mu\nu} + \Lambda g_{\mu\nu} = 0$$

或等价地：
$$G_{\mu\nu} + \Lambda g_{\mu\nu} = 0$$

其中 $G_{\mu\nu} = R_{\mu\nu} - \frac{1}{2} R g_{\mu\nu}$ 是Einstein张量。

**带物质场的情况**：

如果有能量-动量张量 $T_{\mu\nu}$，作用量变为：
$$S = \int \left[(R - 2\Lambda) + \mathcal{L}_m\right] \sqrt{-g} \, d^4x$$

其中 $\mathcal{L}_m$ 是物质场的拉格朗日密度。变分后得到：
$$G_{\mu\nu} + \Lambda g_{\mu\nu} = 8\pi G T_{\mu\nu}$$

这是完整的Einstein场方程。

**历史意义**：

1. **独立推导**：希尔伯特和Einstein几乎同时（1915年11月）得到场方程
2. **方法优势**：希尔伯特使用变分法，推导更系统、更优雅
3. **物理洞察**：Einstein提供了物理解释（时空弯曲、引力）
4. **数学基础**：为广义相对论建立了严格的数学框架

**现代影响**：

- **量子引力**：作用量原理为量子化提供基础
- **场论**：现代场论（标准模型）都基于作用量原理
- **数学物理**：连接微分几何、变分法、PDE等多个数学分支


## 十三、希尔伯特对现代物理的深远影响

### 13.1 数学物理的统一框架

**Hilbert空间作为统一框架**：

希尔伯特空间理论为现代物理提供了统一的数学框架：

**量子力学**：

- 状态空间：$|\psi\rangle \in \mathcal{H}$（希尔伯特空间）
- 可观测量：$\hat{A}: \mathcal{H} \to \mathcal{H}$（自伴算子）
- 测量结果：$\hat{A}$的特征值 $\{\lambda_n\}$
- 概率：$P(\lambda_n) = |\langle \psi | v_n \rangle|^2$（其中 $v_n$ 是对应特征向量）

**统计力学**：

- 相空间：$\Omega$（可测空间）
- 状态：概率测度 $\mu$ 在 $\Omega$ 上
- 演化：动力系统 $T: \Omega \to \Omega$
- 遍历性：时间平均 = 空间平均（Birkhoff定理）

**场论**：

- 场空间：$\mathcal{F} = \{\phi: M \to \mathbb{R}\}$（函数空间）
- 作用量：$S[\phi] = \int L(\phi, \partial\phi) d^4x$
- 路径积分：$\langle \phi_1, \ldots, \phi_n \rangle = \int \phi_1 \cdots \phi_n e^{iS[\phi]} D[\phi]$

**数学公式**：

- 量子态：$$|\psi\rangle = \sum_n c_n |n\rangle, \quad \sum_n |c_n|^2 = 1$$
- 期望值：$$\langle \hat{A} \rangle = \langle \psi | \hat{A} | \psi \rangle = \sum_n \lambda_n |c_n|^2$$
- 配分函数：$$Z = \text{Tr}(e^{-\beta \hat{H}}) = \sum_n e^{-\beta E_n}$$

### 13.2 变分原理的现代应用

**作用量原理的普遍性**：

希尔伯特的作用量原理不仅适用于广义相对论，还适用于所有现代物理理论：

**经典力学**：
$$S[q] = \int L(q, \dot{q}, t) dt \to \min$$
$$\frac{d}{dt}\frac{\partial L}{\partial \dot{q}} = \frac{\partial L}{\partial q}$$

**经典场论**：
$$S[\phi] = \int \mathcal{L}(\phi, \partial_\mu \phi) d^4x \to \min$$
$$\frac{\partial \mathcal{L}}{\partial \phi} - \partial_\mu \frac{\partial \mathcal{L}}{\partial (\partial_\mu \phi)} = 0$$

**量子场论**：
$$Z = \int e^{iS[\phi]} D[\phi]$$
关联函数：$$\langle \phi(x_1) \cdots \phi(x_n) \rangle = \frac{1}{Z} \int \phi(x_1) \cdots \phi(x_n) e^{iS[\phi]} D[\phi]$$

**数学公式**：

- 作用量：$$S[\phi] = \int \left[\frac{1}{2}(\partial_\mu \phi)(\partial^\mu \phi) - V(\phi)\right] d^4x$$
- 运动方程：$$(\Box + m^2)\phi = -\frac{\partial V}{\partial \phi}$$
- 路径积分：$$Z[J] = \int e^{iS[\phi] + i\int J\phi d^4x} D[\phi]$$

### 13.3 对称性与守恒律的数学基础

**Noether定理的现代形式**：

希尔伯特的变分法为Noether定理提供了数学基础：

**经典Noether定理**（1918）：
若作用量 $S = \int L(q, \dot{q}, t) dt$ 在单参数变换群 $q \to q'(\epsilon)$ 下不变，则存在守恒量。

**现代形式**：
若作用量 $S[\phi] = \int \mathcal{L}(\phi, \partial_\mu \phi) d^4x$ 在Lie群 $G$ 作用下不变：
$$\phi(x) \to \phi'(x) = g \cdot \phi(x), \quad g \in G$$

则存在守恒流：
$$j^\mu = \frac{\partial \mathcal{L}}{\partial (\partial_\mu \phi)} \delta \phi - \mathcal{L} \delta x^\mu$$
满足：$$\partial_\mu j^\mu = 0$$

**数学公式**：

- 守恒流：$$j^\mu = \frac{\partial \mathcal{L}}{\partial (\partial_\mu \phi)} \delta \phi$$
- 守恒律：$$\partial_\mu j^\mu = 0$$
- 守恒量：$$Q = \int j^0 d^3x = \text{常数}$$

**应用**：

- 时间平移对称性 → 能量守恒
- 空间平移对称性 → 动量守恒
- 旋转对称性 → 角动量守恒
- 规范对称性 → 电荷守恒

## 十四、希尔伯特方法的计算实现

### 14.1 数值方法在物理中的应用

**有限元方法**：

希尔伯特的变分方法为有限元方法提供了理论基础：

**变分问题**：
$$J[u] = \int_\Omega \left[\frac{1}{2}|\nabla u|^2 - fu\right] dx \to \min$$

**有限元离散化**：

- 将 $\Omega$ 剖分为单元 $\{K_i\}$
- 在有限维子空间 $V_h \subset H_0^1(\Omega)$ 中求解
- $V_h = \text{span}\{\phi_1, \ldots, \phi_N\}$（基函数）

**Galerkin方法**：
$$\int_\Omega \nabla u_h \cdot \nabla v_h dx = \int_\Omega f v_h dx, \quad \forall v_h \in V_h$$

**数学公式**：

- 有限元解：$$u_h = \sum_{i=1}^N u_i \phi_i$$
- 线性系统：$$A\mathbf{u} = \mathbf{f}, \quad A_{ij} = \int_\Omega \nabla \phi_i \cdot \nabla \phi_j dx$$
- 误差估计：$$\|u - u_h\|_{H^1} \leq Ch^p \|u\|_{H^{p+1}}$$

### 14.2 量子计算的数学基础

**量子计算的Hilbert空间框架**：

量子计算建立在Hilbert空间理论之上：

**量子比特（qubit）**：
$$|\psi\rangle = \alpha |0\rangle + \beta |1\rangle, \quad |\alpha|^2 + |\beta|^2 = 1$$

**量子门**：

- 单量子比特门：$U \in U(2)$（2×2酉矩阵）
- 多量子比特门：$U \in U(2^n)$（$n$ 个量子比特）

**量子算法**：

- Grover算法：搜索问题，$O(\sqrt{N})$ 复杂度
- Shor算法：因子分解，多项式时间

**数学公式**：

- 量子态：$$|\psi\rangle = \sum_{x=0}^{2^n-1} \alpha_x |x\rangle, \quad \sum_x |\alpha_x|^2 = 1$$
- 量子门：$$U|\psi\rangle = \sum_x \alpha_x U|x\rangle$$
- 测量：$$P(x) = |\langle x | \psi \rangle|^2 = |\alpha_x|^2$$
