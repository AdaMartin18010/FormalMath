## 一、Dirichlet问题

### 1.1 问题的提出

**Dirichlet问题**（1850s）：

```
给定：
- 区域Ω ⊆ ℝⁿ
- 边界函数f: ∂Ω → ℝ

求：
- 函数u: Ω → ℝ
- 使得：Δu = 0（在Ω内）
- 且：u|_∂Ω = f
```

**物理意义**：

- 静电势
- 热平衡
- 流体力学

---

### 1.2 Weierstrass的质疑

**Dirichlet原理**（Riemann, 1851）：

```
解u使能量最小：
E[u] = ∫_Ω |∇u|²dx

变分：
δE = 0 → Δu = 0
```

**Weierstrass的批评**（1870）：

```
问题：
- 极小化序列可能不收敛
- 下确界可能达不到
- 需要严格化
```

## 二、希尔伯特的严格化

### 2.1 直接方法

**希尔伯特的方法**（1900）：

```
步骤1：定义函数空间
H¹(Ω) = {u | u, ∇u ∈ L²(Ω)}

步骤2：证明E有下界
inf E[u] > -∞

步骤3：构造极小化序列
uₙ使得E[uₙ] → inf

步骤4：证明弱收敛
uₙ ⇀ u（弱）

步骤5：证明u是解
E[u] = inf且Δu = 0
```

---

### 2.2 Sobolev空间

**定义**：

```
Hᵏ(Ω) = {u | D^αu ∈ L²(Ω), |α| ≤ k}

范数：
||u||_{Hᵏ} = (∑_{|α|≤k} ||D^αu||²_{L²})^(1/2)

性质：
- Banach空间
- 自反（k≥1）
- 可分
```

**紧嵌入**：

```
Hᵏ(Ω) ↪ H^{k-1}(Ω)（紧）

应用：
- 存在性证明
- 正则性
- 数值方法
```

## 三、变分法的一般理论

### 3.1 抽象变分问题

**问题**：

```
最小化：
J[u] = ∫ L(x, u, ∇u) dx

约束：
u|_∂Ω = f
```

**Euler-Lagrange方程**：

```
∂L/∂u - ∇·(∂L/∂(∇u)) = 0

边界条件：
u|_∂Ω = f
```

---

### 3.2 存在性理论

**直接方法**：

```
条件：
1. 强制性：J[u] → ∞当||u|| → ∞
2. 弱下半连续：uₙ ⇀ u → J[u] ≤ lim inf J[uₙ]
3. 凸性：L关于∇u凸

结果：
存在最小化子u
```

**应用**：

- 弹性理论
- 最优控制
- 图像处理

## 四、应用：最小曲面

### 4.1 问题

**Plateau问题**：

```
给定：
- 闭曲线Γ ⊆ ℝ³

求：
- 以Γ为边界的面积最小曲面
```

**变分形式**：

```
面积：
A[S] = ∫_S dS

最小化：
min A[S]
约束：∂S = Γ
```

---

### 4.2 存在性

**Douglas-Rado定理**（1930s）：

```
若Γ是Jordan曲线，则存在最小曲面

方法：
- 变分法
- 共形映射
- Weierstrass表示
```

**现代发展**：

- 几何测度论
- 正则性理论
- 数值方法

## 六、现代发展

### 6.1 Γ-收敛

**定义**（De Giorgi, 1970s）：

```
Jₙ Γ-收敛到J ⟺
1. 下界：lim inf Jₙ[uₙ] ≥ J[u]（若uₙ → u）
2. 恢复序列：存在uₙ → u使得lim Jₙ[uₙ] = J[u]
```

**应用**：

- 均匀化
- 相变
- 材料科学

---

## 九、数学公式总结

### 核心公式

1. **Dirichlet问题**：
   $$\Delta u = 0 \text{ 在 } \Omega, \quad u|_{\partial\Omega} = f$$

2. **Dirichlet能量**：
   $$E[u] = \int_\Omega |\nabla u|^2 dx$$

3. **变分问题**：
   $$J[y] = \int_a^b L(x, y, y') dx \to \min$$

4. **Euler-Lagrange方程**：
   $$\frac{\partial L}{\partial y} - \frac{d}{dx}\\\left(\frac{\partial L}{\partial y'}\right) = 0$$

5. **Sobolev空间**：
   $$H^1(\Omega) = \{u : u, \nabla u \in L^2(\Omega)\}$$

6. **弱解定义**：
   $$\int_\Omega \nabla u \cdot \nabla v dx = \int_\Omega f v dx \quad \text{对所有 } v \in H_0^1(\Omega)$$

7. **极小化序列**：
   $$E[u_n] \to \inf_{u \in H_0^1(\Omega)} E[u]$$

8. **Γ-收敛**：
   $$\Gamma\text{-}\lim_{n \to \infty} J_n = J_0$$

9. **最小曲面方程**：
   $$\text{div}\\\left(\frac{\nabla u}{\sqrt{1 + |\nabla u|^2}}\right) = 0$$

10. **Poisson方程**：
    $$-\Delta u = f \text{ 在 } \Omega, \quad u|_{\partial\Omega} = 0$$
