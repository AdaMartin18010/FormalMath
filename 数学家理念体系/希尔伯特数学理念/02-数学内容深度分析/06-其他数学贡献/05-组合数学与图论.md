## 二、图论基础

### 2.1 图的基本概念

**定义**：

```
图G = (V, E)

其中：
- V：顶点集
- E：边集
- E ⊆ V × V
```

**性质**：

- 简单图：无重边、无自环
- 有向图：边有方向
- 加权图：边有权重

---

### 2.2 图的表示

**邻接矩阵**：

```
A[i,j] = 1（若(i,j)∈E）
       = 0（否则）

性质：
- 对称（无向图）
- 特征值
- 谱理论
```

**关联矩阵**：

```
B[i,e] = 1（若顶点i在边e上）
       = 0（否则）
```

## 三、图的谱理论

### 3.1 图的特征值

**定义**：

```
图G的邻接矩阵A

特征值：
- λ₁ ≥ λ₂ ≥ ... ≥ λₙ
- 图的谱

性质：
- 对称矩阵
- 实特征值
- 正交特征向量
```

### 3.2 应用

```
特征值 ↔ 图的性质：
- 最大特征值 ↔ 度
- 特征值分布 ↔ 结构
- 特征向量 ↔ 划分
```

**应用**：

- 图同构
- 社区检测
- 网络分析

### 4.2 线性规划

```
原问题：
max cᵀx

对偶问题：
min bᵀy
s.t. Aᵀy ≥ c

对偶定理：
原最优值 = 对偶最优值
```

**应用**：

- 组合优化

## 六、Ramsey理论

### 6.1 Ramsey定理

**定理**：

```
对任意r, s：
存在N使得：
- 任意N个顶点的完全图
- 用两种颜色着色边
- 存在r个顶点的红色完全子图
- 或s个顶点的蓝色完全子图
```

**记号**：

- R(r,s) = 最小N
- Ramsey数

---

### 6.2 应用

**组合结构**：

```
应用：
- 图论
- 数论
- 几何

例子：
- 任意6个人中，要么3人互相认识，要么3人互不认识
- R(3,3) = 6
```

## 十、数学公式总结

### 核心公式

1. **图定义**：
   $$G = (V, E), \quad E \subseteq V \times V$$

2. **邻接矩阵**：
   $$A_{ij} = \begin{cases} 1 & \text{若 } (i,j) \in E \\ 0 & \text{否则} \end{cases}$$

3. **图的特征值**：
   $$A \mathbf{v} = \lambda \mathbf{v}$$

4. **生成函数**：
   $$G(x) = \sum_{n=0}^\infty a_n x^n$$

5. **指数生成函数**：
   $$E(x) = \sum_{n=0}^\infty \frac{a_n}{n!} x^n$$

6. **Ramsey数**：
   $$R(k, \ell) = \min\{n : \text{每个 } K_n \text{ 有 } K_k \text{ 或 } K_\ell^c\}$$

7. **二项式系数**：
   $$\binom{n}{k} = \frac{n!}{k!(n-k)!}$$

8. **Catalan数**：
   $$C_n = \frac{1}{n+1}\binom{2n}{n} = \frac{(2n)!}{(n+1)!n!}$$

9. **图的色数**：
   $$\chi(G) = \min\{k : G \text{ 是 } k\text{-可染色的}\}$$

10. **线性规划对偶**：
    $$\min \mathbf{c}^T \mathbf{x} \text{ s.t. } A\mathbf{x} = \mathbf{b}, \mathbf{x} \geqqqqq 0$$
