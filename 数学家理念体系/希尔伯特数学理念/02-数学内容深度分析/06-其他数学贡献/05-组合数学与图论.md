## 二、图论基础

### 2.1 图的基本概念

**定义**：

```
图G = (V, E)

其中：
- V：顶点集
- E：边集
- E ⊆ V × V
```

**性质**：

- 简单图：无重边、无自环
- 有向图：边有方向
- 加权图：边有权重

---

### 2.2 图的表示

**邻接矩阵**：

```
A[i,j] = 1（若(i,j)∈E）
       = 0（否则）

性质：
- 对称（无向图）
- 特征值
- 谱理论
```

**关联矩阵**：

```
B[i,e] = 1（若顶点i在边e上）
       = 0（否则）
```

---

## 三、图的谱理论

### 3.1 图的特征值

**定义**：

```
图G的邻接矩阵A

特征值：
- λ₁ ≥ λ₂ ≥ ... ≥ λₙ
- 图的谱

性质：
- 对称矩阵
- 实特征值
- 正交特征向量
```

---

### 3.2 应用

**图的性质**：

```
特征值 ↔ 图的性质：
- 最大特征值 ↔ 度
- 特征值分布 ↔ 结构
- 特征向量 ↔ 划分
```

**应用**：

- 图同构
- 社区检测
- 网络分析

## 四、组合优化

### 4.1 经典问题

**旅行商问题（TSP）**：

```
给定：
- n个城市
- 距离矩阵

求：
- 最短Hamilton回路

复杂度：
- NP-完全
- 近似算法
```

**匹配问题**：

```
给定：
- 二分图G

求：
- 最大匹配

算法：
- 匈牙利算法
- 多项式时间
```

---

### 4.2 线性规划

**对偶性**：

```
原问题：
max cᵀx
s.t. Ax ≤ b

对偶问题：
min bᵀy
s.t. Aᵀy ≥ c

对偶定理：
原最优值 = 对偶最优值
```

**应用**：

- 组合优化
- 网络流
- 博弈论

## 五、生成函数

### 5.1 普通生成函数

**定义**：

```
序列{aₙ}的生成函数：
A(x) = ∑_{n≥0} aₙ xⁿ

性质：
- 加法 ↔ 生成函数加法
- 卷积 ↔ 生成函数乘法
```

**应用**：

- 递推关系
- 组合计数
- 概率生成函数

---

### 5.2 指数生成函数

**定义**：

```
序列{aₙ}的指数生成函数：
A(x) = ∑_{n≥0} aₙ xⁿ/n!

应用：
- 排列问题
- 组合结构
- 标签结构
```

## 六、Ramsey理论

### 6.1 Ramsey定理

**定理**：

```
对任意r, s：
存在N使得：
- 任意N个顶点的完全图
- 用两种颜色着色边
- 存在r个顶点的红色完全子图
- 或s个顶点的蓝色完全子图
```

**记号**：

- R(r,s) = 最小N
- Ramsey数

---

### 6.2 应用

**组合结构**：

```
应用：
- 图论
- 数论
- 几何

例子：
- 任意6个人中，要么3人互相认识，要么3人互不认识
- R(3,3) = 6
```

## 七、现代发展

### 7.1 极值图论

**问题**：

```
给定性质P：
- 最多多少条边？
- 图仍满足P？

例子：
- 无三角形图的最大边数
- Turán定理
```

---

### 7.2 随机图

**Erdős-Rényi模型**：

```
G(n,p)：
- n个顶点
- 每条边独立存在，概率p

性质：
- 相变
- 阈值函数
- 渐近行为
```

**应用**：

- 网络模型
- 算法分析
- 概率方法
