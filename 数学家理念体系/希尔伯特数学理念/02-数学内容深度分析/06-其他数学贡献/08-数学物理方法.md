## 一、变分原理

### 1.1 最小作用量原理

**经典力学**：

```
作用量：
S = ∫ L(q, q̇, t) dt

其中：
L = T - V（Lagrange函数）

原理：
真实路径使S取极值

Euler-Lagrange方程：
d/dt(∂L/∂q̇) = ∂L/∂q
```

**希尔伯特的影响**：

```
变分法：
- 严格化
- 存在性证明
- 现代方法
```

---

### 1.2 场论

**经典场论**：

```
作用量：
S = ∫ L(φ, ∂φ, x) d⁴x

Euler-Lagrange：
∂L/∂φ - ∂_μ(∂L/∂(∂_μφ)) = 0
```

**应用**：

- 电磁场
- 引力场
- 标准模型
