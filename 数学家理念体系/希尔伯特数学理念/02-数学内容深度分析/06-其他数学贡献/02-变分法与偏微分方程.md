## 一、变分法的基础

### 1.1 变分问题

**经典变分问题**：

变分法是研究函数极值问题的数学分支，希尔伯特在其中做出了根本性贡献。

**最速降线问题**：

```
问题：
给定两点A, B
求使质点从A滑到B时间最短的曲线

变分形式：
J[y] = ∫_A^B L(x,y,y')dx

其中：
L是拉格朗日函数
y是待求曲线
```

**Euler-Lagrange方程**：

变分问题的极值条件由Euler-Lagrange方程给出：

```
∂L/∂y - d/dx(∂L/∂y') = 0
```

这是变分法的基本方程。

### 1.2 希尔伯特的贡献

**直接方法（1900）**：

希尔伯特在1900年提出了变分问题的"直接方法"，这是现代变分法的基础。

**问题的数学表述**：

给定泛函：
$$J[u] = \int_\Omega L(x, u, \nabla u) \, dx$$

其中 $\Omega \subset \mathbb{R}^n$ 是有界区域，$L$ 是拉格朗日函数，$u$ 属于某个函数空间 $V$。

**问题**：证明存在 $u^* \in V$ 使得：
$$J[u^*] = \inf_{u \in V} J[u]$$

**希尔伯特直接方法的数学步骤**：

**步骤1：构造极小化序列**

选择序列 $\{u_n\}_{n=1}^\infty \subset V$ 使得：
$$\lim_{n \to \infty} J[u_n] = \inf_{u \in V} J[u] =: m$$

**步骤2：证明序列有下界**

如果 $J[u] \geq C > -\infty$ 对所有 $u \in V$ 成立，则：
$$m = \inf_{u \in V} J[u] \geq C > -\infty$$

**步骤3：证明序列的紧性（关键步骤）**

需要证明 $\{u_n\}$ 在某个拓扑下是紧的。希尔伯特的关键洞察是：

- **弱紧性**：在Hilbert空间或Banach空间中，有界序列有弱收敛子列
- **能量估计**：如果 $J[u_n]$ 有界，则 $\|u_n\|_V$ 有界（在适当的范数下）

**步骤4：提取弱收敛子列**

根据Banach-Alaoglu定理（在Hilbert空间中更简单），存在子列 $\{u_{n_k}\}$ 和 $u^* \in V$ 使得：
$$u_{n_k} \rightharpoonup u^* \quad \text{（弱收敛）}$$

**步骤5：证明弱极限是解（下连续性）**

关键是要证明 $J$ 是**弱下半连续**的：
$$u_n \rightharpoonup u \implies J[u] \leq \liminf_{n \to \infty} J[u_n]$$

如果这个性质成立，则：
$$J[u^*] \leq \liminf_{k \to \infty} J[u_{n_k}] = m = \inf_{u \in V} J[u]$$

因此 $J[u^*] = m$，即 $u^*$ 是极小化子。

**具体例子**：

**例1：Dirichlet能量**

$$J[u] = \int_\Omega |\nabla u|^2 \, dx$$

在 $H_0^1(\Omega)$ 中，$J$ 是弱下半连续的，因为：

- $u_n \rightharpoonup u$ 在 $H_0^1(\Omega)$ 中
- $\|\nabla u\|_{L^2}^2 \leq \liminf_{n \to \infty} \|\nabla u_n\|_{L^2}^2$（弱下半连续性）

**例2：非线性变分问题**

$$J[u] = \int_\Omega \left(|\nabla u|^2 + |u|^p\right) \, dx$$

其中 $p > 2$。需要证明：

- 序列的有界性（通过能量估计）
- 弱紧性（通过Sobolev嵌入）
- 弱下半连续性（通过凸性）

**希尔伯特方法的历史意义**：

1. **开创性**：首次系统性地解决了变分问题的存在性
2. **方法论**：建立了"直接方法"这一重要工具
3. **影响**：为现代PDE理论、Sobolev空间理论、变分不等式等奠定基础

**历史意义**：

- 开创了现代变分法
- 为PDE理论铺路
- 影响20世纪分析学
- 为弱解理论奠基

## 二、Dirichlet原理

### 2.1 问题背景

**Dirichlet问题**：

```
给定区域Ω和边界函数f
求u使得：
Δu = 0（在Ω内，调和函数）
u = f（在∂Ω上，边界条件）
```

**Dirichlet原理（Riemann, 1851）**：

Riemann提出，解u使能量泛函最小：

```
E[u] = ∫_Ω |∇u|²dx
```

**Weierstrass的质疑（1870）**：

Weierstrass指出Dirichlet原理的缺陷：

- 极小化序列可能不收敛
- 需要严格的数学基础
- 导致Dirichlet原理被质疑

### 2.2 希尔伯特的严格化（1900）

**希尔伯特的方法**：

希尔伯特在1900年严格证明了Dirichlet原理：

```
步骤1：定义Sobolev空间H¹(Ω)
H¹(Ω) = {u | u, ∇u ∈ L²(Ω)}

步骤2：证明E在H¹上有下界
E[u] ≥ 0，且E[u] = 0 ⟺ u = 0

步骤3：构造极小化序列
存在序列{u_n}使得E[u_n] → inf E[u]

步骤4：证明弱收敛
{u_n}在H¹中弱收敛到u₀

步骤5：证明弱极限是解
u₀是Dirichlet问题的解
```

**关键创新**：

1. **弱收敛概念**：引入弱收敛处理序列极限
2. **Sobolev空间**：为PDE提供函数空间框架
3. **紧性方法**：使用紧性提取收敛子列

**历史影响**：

- 严格证明Dirichlet原理
- 开创弱解理论
- 为现代PDE奠基
- 影响20世纪分析学

## 三、积分方程的应用

### 3.1 Green函数方法

**基本思路**：

希尔伯特将PDE问题转化为积分方程：

```
PDE问题
    ↓
转化为积分方程
    ↓
应用Hilbert-Schmidt理论
    ↓
求解
```

**Poisson方程的例子**：

```
问题：
Δu = f（在Ω内）
u = 0（在∂Ω上）

Green函数G(x,ξ)：
ΔG = δ(x-ξ)（在Ω内）
G|_∂Ω = 0

解：
u(x) = ∫_Ω G(x,ξ)f(ξ)dξ
```

**Hilbert-Schmidt理论**：

希尔伯特发展了积分方程的特征值理论：

- 自伴算子的谱理论
- 特征函数的完备性
- 特征值展开

### 3.2 特征值问题

**Sturm-Liouville问题**：

```
-(py')' + qy = λy
边界条件

算子：
L = -d/dx(p d/dx) + q

性质：
- L是自伴算子
- L有紧逆（若p>0, q≥0）
- L有可数特征值 {λ_n}
- 特征函数 {φ_n} 形成L²的完备正交基
```

**应用**：

- 分离变量法
- 特征函数展开
- 谱理论

**定义**：

```
Hᵏ(Ω) = {u | D^αu ∈ L²(Ω), |α| ≤ k}

性质：
- Banach空间（范数：||u||_Hᵏ = Σ_{|α|≤k} ||D^αu||_L²）
- 紧嵌入定理（Hᵏ(Ω) ↪ L²(Ω)）
- 为PDE提供函数空间
```

**应用**：

- 弱解理论

## 九、数学公式总结

### 核心公式

1. **变分问题**：
   $$J[y] = \int_a^b L(x, y, y') dx \to \min$$

2. **Euler-Lagrange方程**：
   $$\frac{\partial L}{\partial y} - \frac{d}{dx}\left(\frac{\partial L}{\partial y'}\right) = 0$$

3. **Dirichlet原理**：
   $$J[u] = \int_\Omega |\nabla u|^2 dx \to \min, \quad u|_{\partial\Omega} = f$$

4. **Poisson方程**：
   $$-\Delta u = f \text{ 在 } \Omega, \quad u|_{\partial\Omega} = 0$$

5. **Green函数**：
   $$-\Delta G(x, \xi) = \delta(x - \xi) \text{ 在 } \Omega, \quad G|_{\partial\Omega} = 0$$

6. **解的表示**：
   $$u(x) = \int_\Omega G(x, \xi) f(\xi) d\xi$$

7. **Sturm-Liouville问题**：
   $$-(py')' + qy = \lambda y, \quad \text{边界条件}$$

8. **Sobolev空间**：
   $$H^k(\Omega) = \{u : D^\alpha u \in L^2(\Omega), |\alpha| \leq k\}$$

9. **Sobolev范数**：
   $$\|u\|_{H^k} = \left(\sum_{|\alpha| \leq k} \|D^\alpha u\|_{L^2}^2\right)^{1/2}$$

10. **弱解定义**：
    $$\int_\Omega \nabla u \cdot \nabla v dx = \int_\Omega f v dx \quad \text{对所有 } v \in H_0^1(\Omega)$$

## 十二、变分法的详细数学表述

### 12.1 弱下半连续性的严格定义

**弱下半连续性**：

设 $V$ 是Banach空间，泛函 $J: V \to \mathbb{R} \cup \{+\infty\}$ 是**弱下半连续**的，如果：
$$u_n \rightharpoonup u \text{ 在 } V \text{ 中 } \implies J[u] \leq \liminf_{n \to \infty} J[u_n]$$

**凸性与弱下半连续性**：

如果 $J$ 是**凸的**且**下半连续**的，则 $J$ 是弱下半连续的。

**数学公式**：

- 弱下半连续性：$$u_n \rightharpoonup u \implies J[u] \leq \liminf_{n \to \infty} J[u_n]$$
- 凸性：$$J[\lambda u + (1-\lambda)v] \leq \lambda J[u] + (1-\lambda) J[v]$$
- 下半连续性：$$u_n \to u \implies J[u] \leq \liminf_{n \to \infty} J[u_n]$$

---

### 12.2 Dirichlet原理的严格证明

**Dirichlet问题的变分形式**：

给定 $f \in H^{-1}(\Omega)$，求 $u \in H_0^1(\Omega)$ 使得：
$$J[u] = \frac{1}{2} \int_\Omega |\nabla u|^2 dx - \langle f, u \rangle \to \min$$

**存在性证明**：

**步骤1：下界性**
$$J[u] = \frac{1}{2} \|\nabla u\|_{L^2}^2 - \langle f, u \rangle \geq \frac{1}{2} \|\nabla u\|_{L^2}^2 - \|f\|_{H^{-1}} \|\nabla u\|_{L^2}$$

使用Poincaré不等式：$\|u\|_{L^2} \leq C \|\nabla u\|_{L^2}$，得到：
$$J[u] \geq \frac{1}{2} \|\nabla u\|_{L^2}^2 - C \|f\|_{H^{-1}} \|\nabla u\|_{L^2} \geq -C' \|f\|_{H^{-1}}^2$$

因此 $J$ 有下界。

**步骤2：极小化序列**
选择 $\{u_n\} \subset H_0^1(\Omega)$ 使得：
$$\lim_{n \to \infty} J[u_n] = \inf_{u \in H_0^1(\Omega)} J[u] =: m$$

**步骤3：有界性**
从下界性得到 $\|\nabla u_n\|_{L^2}$ 有界，因此 $\{u_n\}$ 在 $H_0^1(\Omega)$ 中有界。

**步骤4：弱收敛**
根据Banach-Alaoglu定理，存在子列 $\{u_{n_k}\}$ 和 $u^* \in H_0^1(\Omega)$ 使得：
$$u_{n_k} \rightharpoonup u^* \text{ 在 } H_0^1(\Omega) \text{ 中}$$

**步骤5：弱极限是解**
由于 $J$ 是弱下半连续的（因为它是凸的且下半连续的），得到：
$$J[u^*] \leq \liminf_{k \to \infty} J[u_{n_k}] = m = \inf_{u \in H_0^1(\Omega)} J[u]$$

因此 $J[u^*] = m$，即 $u^*$ 是极小化子。

**步骤6：Euler-Lagrange方程**
对任意 $v \in H_0^1(\Omega)$，计算变分：
$$\frac{d}{d\epsilon} J[u^* + \epsilon v]|_{\epsilon=0} = \int_\Omega \nabla u^* \cdot \nabla v dx - \langle f, v \rangle = 0$$

因此 $u^*$ 满足弱形式的Poisson方程：
$$\int_\Omega \nabla u^* \cdot \nabla v dx = \langle f, v \rangle \quad \text{对所有 } v \in H_0^1(\Omega)$$

**数学公式**：

- Dirichlet能量：$$J[u] = \frac{1}{2} \int_\Omega |\nabla u|^2 dx - \langle f, u \rangle$$
- Poincaré不等式：$$\|u\|_{L^2} \leq C \|\nabla u\|_{L^2} \quad \text{（$u \in H_0^1(\Omega)$）}$$
- 弱形式：$$\int_\Omega \nabla u \cdot \nabla v dx = \langle f, v \rangle \quad \text{对所有 } v \in H_0^1(\Omega)$$

---

### 12.3 积分方程的特征值理论

**Hilbert-Schmidt算子**：

设 $K: L^2(\Omega) \to L^2(\Omega)$ 是积分算子：
$$(Ku)(x) = \int_\Omega k(x, y) u(y) dy$$

其中 $k(x, y)$ 是**Hilbert-Schmidt核**，满足：
$$\int_\Omega \int_\Omega |k(x, y)|^2 dx dy < \infty$$

**Hilbert-Schmidt定理**：

如果 $K$ 是自伴的Hilbert-Schmidt算子，则：

1. $K$ 有可数个特征值 $\{\lambda_n\}_{n=1}^\infty$（重数有限）
2. 特征值满足：$\lambda_n \to 0$（当 $n \to \infty$）
3. 特征函数 $\{\phi_n\}_{n=1}^\infty$ 形成 $L^2(\Omega)$ 的完备正交基
4. $K$ 可以表示为：
$$Ku = \sum_{n=1}^\infty \lambda_n \langle u, \phi_n \rangle \phi_n$$

**应用**：

**例1：Poisson方程的Green函数**

Poisson方程 $\Delta u = f$ 的解可以表示为：
$$u(x) = \int_\Omega G(x, y) f(y) dy$$

其中 $G(x, y)$ 是Green函数。这给出了积分算子：
$$(Kf)(x) = \int_\Omega G(x, y) f(y) dy$$

**例2：特征值问题**

考虑特征值问题：
$$-\Delta u = \lambda u \text{ 在 } \Omega, \quad u|_{\partial\Omega} = 0$$

这等价于积分方程：
$$u = \lambda Ku$$

其中 $K$ 是Green算子。根据Hilbert-Schmidt定理，存在可数个特征值 $\{\lambda_n\}$ 和对应的特征函数 $\{\phi_n\}$。

**数学公式**：

- Hilbert-Schmidt核：$$\int_\Omega \int_\Omega |k(x, y)|^2 dx dy < \infty$$
- 特征值展开：$$Ku = \sum_{n=1}^\infty \lambda_n \langle u, \phi_n \rangle \phi_n$$
- Green函数：$$G(x, y) = \sum_{n=1}^\infty \frac{\phi_n(x) \phi_n(y)}{\lambda_n}$$

## 十三、现代变分法的发展

### 13.1 Γ-收敛理论

**Γ-收敛**（De Giorgi, 1970s）：

设 $\{J_n\}_{n=1}^\infty$ 是一列泛函，$J$ 是另一个泛函。称 $J_n$ **Γ-收敛**到 $J$，如果：

1. **下界条件**：对任意 $u_n \to u$，有：
$$J[u] \leq \liminf_{n \to \infty} J_n[u_n]$$

2. **恢复序列**：对任意 $u$，存在序列 $\{u_n\}$ 使得 $u_n \to u$ 且：
$$J[u] = \lim_{n \to \infty} J_n[u_n]$$

**应用**：

Γ-收敛用于研究：

- **均匀化理论**：研究周期介质的极限行为
- **相变问题**：研究多相系统的能量极限
- **数值方法**：研究离散化方法的收敛性

**数学公式**：

- Γ-收敛：$$J_n \xrightarrow{\Gamma} J$$
- 下界条件：$$u_n \to u \implies J[u] \leq \liminf_{n \to \infty} J_n[u_n]$$
- 恢复序列：$$\forall u, \exists u_n \to u: J[u] = \lim_{n \to \infty} J_n[u_n]$$

### 13.2 集中紧性原理

**问题**：

在某些变分问题中，极小化序列可能**发散**（不紧），例如：
$$J[u] = \int_{\mathbb{R}^N} \left(|\nabla u|^2 - |u|^p\right) dx$$

其中 $p$ 是临界指数。

**集中紧性原理**（Lions, 1980s）：

极小化序列的"能量"可能：

1. **紧性**：集中在有限区域
2. **消失**：分散到无穷远
3. **集中**：集中在有限个点

**应用**：

集中紧性原理用于：

- **临界Sobolev嵌入**：处理临界指数问题
- **非线性PDE**：证明存在性
- **几何分析**：研究流形的几何性质

**数学公式**：

- 集中紧性：$$\text{能量} = \text{紧性} + \text{消失} + \text{集中}$$
- 临界指数：$$p = \frac{2N}{N-2} \text{（$N \geq 3$）}$$

### 13.3 最优传输理论

**Monge问题**（1781）：

给定两个概率测度 $\mu, \nu$，求映射 $T$ 使得：
$$T_* \mu = \nu$$

且最小化传输成本：
$$C[T] = \int |x - T(x)|^2 d\mu(x)$$

**Kantorovich松弛**（1940s）：

将问题松弛为：
$$C[\gamma] = \int |x - y|^2 d\gamma(x, y) \to \min$$

其中 $\gamma$ 是**传输计划**（coupling），满足边际约束。

**Brenier定理**（1987）：

如果 $\mu$ 绝对连续，则存在唯一的**最优传输映射** $T = \nabla \phi$，其中 $\phi$ 是凸函数。

**应用**：

最优传输理论用于：

- **几何分析**：研究流形的几何性质
- **偏微分方程**：Monge-Ampère方程
- **机器学习**：生成模型、GAN

**数学公式**：

- Monge问题：$$T_* \mu = \nu, \quad C[T] = \int |x - T(x)|^2 d\mu(x) \to \min$$
- Kantorovich问题：$$C[\gamma] = \int |x - y|^2 d\gamma(x, y) \to \min$$
- Brenier映射：$$T = \nabla \phi, \quad \phi \text{ 凸函数}$$
