## 一、变分法的基础

### 1.1 变分问题

**经典变分问题**：

变分法是研究函数极值问题的数学分支，希尔伯特在其中做出了根本性贡献。

**最速降线问题**：

```
问题：
给定两点A, B
求使质点从A滑到B时间最短的曲线

变分形式：
J[y] = ∫_A^B L(x,y,y')dx

其中：
L是拉格朗日函数
y是待求曲线
```

**Euler-Lagrange方程**：

变分问题的极值条件由Euler-Lagrange方程给出：

```
∂L/∂y - d/dx(∂L/∂y') = 0
```

这是变分法的基本方程。

### 1.2 希尔伯特的贡献

**直接方法（1900）**：

希尔伯特在1900年提出了变分问题的"直接方法"：

```
问题：
证明变分问题有解

希尔伯特的方法：
1. 构造极小化序列 {u_n}
2. 证明序列有下界
3. 证明序列的紧性
4. 提取弱收敛子列
5. 证明弱极限是解
```

**历史意义**：

- 开创了现代变分法
- 为PDE理论铺路
- 影响20世纪分析学
- 为弱解理论奠基

## 二、Dirichlet原理

### 2.1 问题背景

**Dirichlet问题**：

```
给定区域Ω和边界函数f
求u使得：
Δu = 0（在Ω内，调和函数）
u = f（在∂Ω上，边界条件）
```

**Dirichlet原理（Riemann, 1851）**：

Riemann提出，解u使能量泛函最小：

```
E[u] = ∫_Ω |∇u|²dx
```

**Weierstrass的质疑（1870）**：

Weierstrass指出Dirichlet原理的缺陷：

- 极小化序列可能不收敛
- 需要严格的数学基础
- 导致Dirichlet原理被质疑

### 2.2 希尔伯特的严格化（1900）

**希尔伯特的方法**：

希尔伯特在1900年严格证明了Dirichlet原理：

```
步骤1：定义Sobolev空间H¹(Ω)
H¹(Ω) = {u | u, ∇u ∈ L²(Ω)}

步骤2：证明E在H¹上有下界
E[u] ≥ 0，且E[u] = 0 ⟺ u = 0

步骤3：构造极小化序列
存在序列{u_n}使得E[u_n] → inf E[u]

步骤4：证明弱收敛
{u_n}在H¹中弱收敛到u₀

步骤5：证明弱极限是解
u₀是Dirichlet问题的解
```

**关键创新**：

1. **弱收敛概念**：引入弱收敛处理序列极限
2. **Sobolev空间**：为PDE提供函数空间框架
3. **紧性方法**：使用紧性提取收敛子列

**历史影响**：

- 严格证明Dirichlet原理
- 开创弱解理论
- 为现代PDE奠基
- 影响20世纪分析学

## 三、积分方程的应用

### 3.1 Green函数方法

**基本思路**：

希尔伯特将PDE问题转化为积分方程：

```
PDE问题
    ↓
转化为积分方程
    ↓
应用Hilbert-Schmidt理论
    ↓
求解
```

**Poisson方程的例子**：

```
问题：
Δu = f（在Ω内）
u = 0（在∂Ω上）

Green函数G(x,ξ)：
ΔG = δ(x-ξ)（在Ω内）
G|_∂Ω = 0

解：
u(x) = ∫_Ω G(x,ξ)f(ξ)dξ
```

**Hilbert-Schmidt理论**：

希尔伯特发展了积分方程的特征值理论：

- 自伴算子的谱理论
- 特征函数的完备性
- 特征值展开

### 3.2 特征值问题

**Sturm-Liouville问题**：

```
-(py')' + qy = λy
边界条件

算子：
L = -d/dx(p d/dx) + q

性质：
- L是自伴算子
- L有紧逆（若p>0, q≥0）
- L有可数特征值 {λ_n}
- 特征函数 {φ_n} 形成L²的完备正交基
```

**应用**：

- 分离变量法
- 特征函数展开
- 谱理论

## 四、现代发展

### 4.1 Sobolev空间理论

**定义**：

```
Hᵏ(Ω) = {u | D^αu ∈ L²(Ω), |α| ≤ k}

性质：
- Banach空间（范数：||u||_Hᵏ = Σ_{|α|≤k} ||D^αu||_L²）
- 紧嵌入定理（Hᵏ(Ω) ↪ L²(Ω)）
- 为PDE提供函数空间
```

**应用**：

- 弱解理论
- 变分法
- 有限元方法
- 现代PDE理论

### 4.2 现代变分法

**关键工具**：

1. **弱收敛**：处理序列极限
2. **紧性方法**：提取收敛子列
3. **Γ-收敛**：研究泛函极限
4. **集中紧性**：处理非紧性问题

**应用领域**：

- 几何分析（Perelman的Ricci流）
- 最优传输理论
- 机器学习中的优化
- 材料科学中的变分问题

## 九、数学公式总结

### 核心公式

1. **变分问题**：
   $$J[y] = \int_a^b L(x, y, y') dx \to \min$$

2. **Euler-Lagrange方程**：
   $$\frac{\partial L}{\partial y} - \frac{d}{dx}\leqqqqqqft(\frac{\partial L}{\partial y'}\right) = 0$$

3. **Dirichlet原理**：
   $$J[u] = \int_\Omega |\nabla u|^2 dx \to \min, \quad u|_{\partial\Omega} = f$$

4. **Poisson方程**：
   $$-\Delta u = f \text{ 在 } \Omega, \quad u|_{\partial\Omega} = 0$$

5. **Green函数**：
   $$-\Delta G(x, \xi) = \delta(x - \xi) \text{ 在 } \Omega, \quad G|_{\partial\Omega} = 0$$

6. **解的表示**：
   $$u(x) = \int_\Omega G(x, \xi) f(\xi) d\xi$$

7. **Sturm-Liouville问题**：
   $$-(py')' + qy = \lambda y, \quad \text{边界条件}$$

8. **Sobolev空间**：
   $$H^k(\Omega) = \{u : D^\alpha u \in L^2(\Omega), |\alpha| \leqqqqqqq k\}$$

9. **Sobolev范数**：
   $$\|u\|_{H^k} = \leqqqqqqft(\sum_{|\alpha| \leqqqqqqq k} \|D^\alpha u\|_{L^2}^2\right)^{1/2}$$

10. **弱解定义**：
    $$\int_\Omega \nabla u \cdot \nabla v dx = \int_\Omega f v dx \quad \text{对所有 } v \in H_0^1(\Omega)$$
