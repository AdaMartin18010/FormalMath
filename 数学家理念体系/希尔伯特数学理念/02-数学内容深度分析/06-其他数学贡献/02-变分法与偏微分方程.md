# 变分法与偏微分方程：希尔伯特的应用数学

## 一、变分法的基础

### 1.1 变分问题

**经典问题**：

```
最速降线问题：
给定两点A, B
求使质点从A滑到B时间最短的曲线

变分：
J[y] = ∫ L(x,y,y')dx

其中：
L是拉格朗日函数
```

**Euler-Lagrange方程**：

```
极值条件：
∂L/∂y - d/dx(∂L/∂y') = 0
```

---

### 1.2 希尔伯特的贡献

**直接方法**（1900）：

```
问题：
证明变分问题有解

方法：
1. 构造极小化序列
2. 证明紧性
3. 提取极限
4. 证明极限是解
```

**意义**：

- 开创现代变分法
- 为PDE理论铺路
- 影响20世纪分析学

---

## 二、Dirichlet原理

### 2.1 问题

**Dirichlet问题**：

```
给定区域Ω和边界函数f
求u使得：
Δu = 0（在Ω内）
u = f（在∂Ω上）
```

**Dirichlet原理**（Riemann, 1851）：

```
解u使能量最小：
E[u] = ∫_Ω |∇u|²dx
```

**问题**：

- Weierstrass质疑（1870）
- 极小化序列可能不收敛
- 需要严格化

---

### 2.2 希尔伯特的严格化（1900）

**方法**：

```
步骤1：定义Sobolev空间H¹(Ω)
步骤2：证明E在H¹上有下界
步骤3：构造极小化序列
步骤4：证明弱收敛
步骤5：证明弱极限是解
```

**结果**：

- 严格证明Dirichlet原理
- 开创弱解理论
- 为现代PDE奠基

## 三、积分方程的应用

### 3.1 Green函数方法

**思路**：

```
PDE问题
    ↓
转化为积分方程
    ↓
应用Hilbert-Schmidt理论
    ↓
求解
```

**例子：Poisson方程**：

```
Δu = f（在Ω内）
u = 0（在∂Ω上）

Green函数G(x,ξ)：
ΔG = δ(x-ξ)
G|_∂Ω = 0

解：
u(x) = ∫ G(x,ξ)f(ξ)dξ
```

---

### 3.2 特征值问题

**Sturm-Liouville问题**：

```
-(py')' + qy = λy
边界条件

算子：
L = -d/dx(p d/dx) + q

性质：
- L自伴
- L有紧逆（若p>0, q≥0）
- → L有可数特征值
- → 特征函数形成L²的基
```

## 四、现代发展

### 4.1 Sobolev空间理论

**定义**：

```
Hᵏ(Ω) = {u | D^αu∈L²(Ω), |α|≤k}

性质：
- Banach空间
- 紧嵌入定理
- 为PDE提供函数空间
```

**应用**：

- 弱解理论
- 变分法
- 有限元方法

---


**应用**：
