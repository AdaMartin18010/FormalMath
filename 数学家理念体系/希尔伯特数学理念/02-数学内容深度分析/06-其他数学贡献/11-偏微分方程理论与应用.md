## 一、希尔伯特在偏微分方程中的历史贡献

### 1.1 Dirichlet原理的严格化

**历史背景**：

```
Riemann（1851）：
- 提出Dirichlet原理
- 用于证明存在性

Weierstrass（1870）：
- 指出证明有漏洞
- 极小化序列可能不收敛

Hilbert（1900）：
- 严格证明Dirichlet原理
- 开创现代变分法
```

**Hilbert的方法**：

```
问题：
给定区域Ω和边界函数f
求u使得：Δu = 0（在Ω内），u|_∂Ω = f

方法：
1. 定义能量泛函 E[u] = ∫_Ω |∇u|²dx
2. 在适当函数空间H¹(Ω)上最小化
3. 证明存在最小化子
4. 证明最小化子满足Δu = 0

关键：
- Sobolev空间的引入
- 弱收敛概念
- 紧性论证
```

---

### 1.2 积分方程方法

**Green函数方法**：

```
思路：
PDE问题 → 积分方程 → 求解

例子：Poisson方程
Δu = f（在Ω内）
u = 0（在∂Ω上）

Green函数G(x,ξ)：
Δ_x G = δ(x-ξ)
G|_∂Ω = 0

解：
u(x) = ∫_Ω G(x,ξ)f(ξ)dξ

转化为：
u = Kf（积分算子）
→ 应用Hilbert-Schmidt理论
```

**Hilbert的影响**：

```
积分方程理论
    ↓
PDE求解方法
    ↓
现代数值方法
```

## 二、现代PDE理论中的希尔伯特空间方法

### 2.1 弱解理论

**Sobolev空间**：

```
定义：
H¹(Ω) = {u | u, ∇u ∈ L²(Ω)}

范数：
||u||_{H¹}² = ∫_Ω (|u|² + |∇u|²)dx

性质：
- 完备（Hilbert空间）
- 自反
- 可分
```

**弱解概念**：

```
经典解：
u ∈ C²(Ω)，Δu = f（逐点）

弱解：
u ∈ H¹(Ω)，对所有v ∈ H₀¹(Ω)：
∫_Ω ∇u·∇v dx = ∫_Ω fv dx

优势：
- 允许不光滑解
- 存在性容易证明
- 数值方法友好
```

---

### 2.2 变分方法

**最小化方法**：

```
问题：
-Δu = f（在Ω内）
u = 0（在∂Ω上）

变分形式：
最小化 J[u] = 1/2∫_Ω |∇u|²dx - ∫_Ω fu dx
在 H₀¹(Ω) 上

Euler-Lagrange方程：
δJ/δu = 0 → -Δu = f

存在性：
- J有下界
- 弱下半连续
- 紧性
→ 存在最小化子
```

**Hilbert的影响**：

```
Dirichlet原理的严格化
    ↓
现代变分方法
    ↓
PDE弱解理论
```

## 三、椭圆方程理论

### 3.1 Laplace方程

**经典问题**：

```
Dirichlet问题：
Δu = 0（在Ω内）
u = f（在∂Ω上）

解的存在性：
- Hilbert严格化（1900）
- Perron方法（1923）
- Wiener准则（1924）
```

**能量方法**：

```
能量恒等式：
∫_Ω |∇u|²dx = -∫_∂Ω u(∂u/∂n)dS

应用：
- 唯一性
- 稳定性
- 数值方法
```

---

### 3.2 Poisson方程

**问题**：

```
-Δu = f（在Ω内）
u = 0（在∂Ω上）

弱解：
u ∈ H₀¹(Ω)，对所有v ∈ H₀¹(Ω)：
∫_Ω ∇u·∇v dx = ∫_Ω fv dx
```

**Lax-Milgram引理**：

```
定理：
设a: H×H → ℝ是双线性形式
- 有界：|a(u,v)| ≤ C||u|| ||v||
- 强制：a(u,u) ≥ α||u||²

则对任意F ∈ H*：
存在唯一u ∈ H使得
a(u,v) = F(v)对所有v ∈ H

应用：
Poisson方程弱解存在唯一性
```

---

### 3.3 正则性理论

**Hilbert空间方法**：

```
弱解正则性：
- 如果f光滑 → u光滑
- 内部正则性
- 边界正则性

方法：
- 差分方法
- 先验估计
- 迭代提升

现代工具：
- Sobolev嵌入
- 插值理论
- 正则性理论
```

## 四、抛物方程理论

### 4.1 热方程

**Cauchy问题**：

```
∂u/∂t = Δu（在ℝⁿ×(0,∞)）
u(x,0) = u₀(x)

解：
u(x,t) = ∫_ℝⁿ K(x-y,t)u₀(y)dy

其中：
K(x,t) = (4πt)^{-n/2}e^{-|x|²/(4t)}
（热核）
```

**Hilbert空间方法**：

```
半群方法：
-Δ在L²(ℝⁿ)上生成半群
u(t) = e^{tΔ}u₀

算子半群：
e^{tΔ} = ∫ e^{-tλ} dE_{-Δ}(λ)

应用：
- 存在性
- 唯一性
- 正则性
```

---

### 4.2 算子半群理论

**Hille-Yosida定理**：

```
定理：
A是H上的线性算子

A生成强连续收缩半群
    ⟺
A是闭的、稠定的，且：
- D(A)稠密
- (0,∞) ⊂ ρ(A)
- ||(λI-A)⁻¹|| ≤ 1/λ（λ>0）

应用：
- 抛物方程
- 演化方程
- 随机过程
```

**与Hilbert的关系**：

```
Hilbert空间理论
    ↓
算子理论
    ↓
半群理论
    ↓
PDE演化方程
```

## 五、双曲方程理论

### 5.1 波动方程

**Cauchy问题**：

```
∂²u/∂t² = Δu（在ℝⁿ×(0,∞)）
u(x,0) = u₀(x)
∂u/∂t(x,0) = u₁(x)

解：
- d'Alembert公式（n=1）
- 球面平均方法（n≥2）
- 能量方法
```

**能量守恒**：

```
能量：
E(t) = 1/2∫_ℝⁿ (|∂u/∂t|² + |∇u|²)dx

性质：
dE/dt = 0（能量守恒）

应用：
- 唯一性
- 稳定性
- 有限传播速度
```

---

### 5.2 能量方法

**Hilbert空间框架**：

```
状态空间：
H = H¹(ℝⁿ) × L²(ℝⁿ)
(u, ∂u/∂t) ∈ H

能量范数：
||(u,v)||_H² = ∫ (|∇u|² + |v|²)dx

演化：
d/dt (u, ∂u/∂t) = A(u, ∂u/∂t)

其中A是无界算子
```

## 六、非线性PDE理论

### 6.1 变分方法

**临界点理论**：

```
问题：
-Δu = f(u)（在Ω内）
u = 0（在∂Ω上）

变分形式：
最小化 J[u] = 1/2∫|∇u|²dx - ∫F(u)dx

其中F' = f

方法：
- 直接方法（Hilbert）
- 临界点理论
- 山路引理
```

---

### 6.2 不动点方法

**压缩映射原理**：

```
问题：
u = K(u)（积分方程）

方法：
- 构造迭代u_{n+1} = K(u_n)
- 证明K是压缩映射
- 不动点定理

应用：
- 非线性椭圆方程
- 非线性抛物方程
- 存在性证明
```

## 七、现代数值方法

### 7.1 有限元方法

**变分形式**：

```
弱形式：
∫_Ω ∇u·∇v dx = ∫_Ω fv dx

有限元逼近：
- 将Ω剖分为单元
- 在有限维子空间V_h中求解
- u_h ∈ V_h

Galerkin方法：
∫_Ω ∇u_h·∇v_h dx = ∫_Ω fv_h dx
对所有v_h ∈ V_h
```

**Hilbert空间框架**：

```
理论基础：
- Sobolev空间
- Ritz-Galerkin方法
- Céa引理

误差估计：
||u - u_h||_{H¹} ≤ Ch^p||u||_{H^{p+1}}

其中：
- h：网格尺寸
- p：多项式次数
```

---

### 7.2 谱方法

**Fourier方法**：

```
思路：
用Fourier级数展开

u(x) = ∑_k û_k e^{ikx}

代入PDE：
→ 代数方程组

优势：
- 高精度
- 快速算法（FFT）
- 周期性边界条件
```

**Hilbert空间基础**：

```
L²上的Fourier基
    ↓
Parseval等式
    ↓
谱方法
```

## 八、随机PDE

### 8.1 随机热方程

**问题**：

```
du = Δu dt + σ(u) dW_t
u(x,0) = u₀(x)

其中W_t是Wiener过程
```

**Hilbert空间方法**：

```
状态空间：
H = L²(Ω)

随机演化：
du(t) = Au(t)dt + B(u(t))dW(t)

其中：
- A = Δ（确定性部分）
- B是随机项

理论：
- Itô随机积分
- 随机半群
- 存在唯一性
```

---

### 8.2 Malliavin分析

**随机微积分**：

```
Malliavin导数：
D: L²(Ω) → L²(Ω; H)

应用：
- 随机PDE的正则性
- 随机控制
- 数值方法
```

### 最终评价

从Dirichlet原理到现代有限元方法，从Hilbert空间到随机PDE，希尔伯特的数学工具和方法**持续服务于偏微分方程的研究和应用**。

## 十一、数学公式总结

### 核心公式

1. **Dirichlet原理**：
   $$E[u] = \int_\Omega |\nabla u|^2 dx \to \min, \quad u|_{\partial\Omega} = f$$

2. **Poisson方程**：
   $$-\Delta u = f \text{ 在 } \Omega, \quad u|_{\partial\Omega} = 0$$

3. **Green函数**：
   $$-\Delta_x G(x, \xi) = \delta(x - \xi), \quad G|_{\partial\Omega} = 0$$

4. **解的表示**：
   $$u(x) = \int_\Omega G(x, \xi) f(\xi) d\xi$$

5. **热方程**：
   $$\frac{\partial u}{\partial t} = \Delta u, \quad u(0, x) = u_0(x)$$

6. **算子半群**：
   $$u(t) = e^{t\Delta} u_0$$

7. **波动方程**：
   $$\frac{\partial^2 u}{\partial t^2} = \Delta u$$

8. **能量守恒**：
   $$E(t) = \frac{1}{2}\int_\Omega \leqqqqqqft(|\partial_t u|^2 + |\nabla u|^2\right) dx = \text{常数}$$

9. **Sobolev空间**：
   $$H^k(\Omega) = \{u : D^\alpha u \in L^2(\Omega), |\alpha| \leqqqqqqq k\}$$

10. **弱解定义**：
    $$\int_\Omega \nabla u \cdot \nabla v dx = \int_\Omega f v dx \quad \text{对所有 } v \in H_0^1(\Omega)$$
