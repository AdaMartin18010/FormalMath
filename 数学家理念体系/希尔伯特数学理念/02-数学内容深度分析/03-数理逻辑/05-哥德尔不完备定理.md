## 一、希尔伯特纲领的目标

### 1.1 三大目标

**希尔伯特纲领**（1920s）：

```
目标1：一致性（Consistency）
- 证明数学公理系统一致
- 不存在矛盾

目标2：完备性（Completeness）
- 所有真命题可证
- 对任意φ，要么⊢φ，要么⊢¬φ

目标3：可判定性（Decidability）
- 存在算法判定φ是否可证
- Entscheidungsproblem
```

**元数学方法**：

- 用有限方法研究形式系统
- 不依赖无限
- 可机械验证

### 1.2 早期乐观

**希尔伯特（1928）**：
> "Wir müssen wissen, wir werden wissen!"
> "我们必须知道，我们终将知道！"

**预期**：

- 几年内完成
- 数学基础彻底解决
- 形式主义胜利

## 二、哥德尔的突破

### 2.1 第一不完备定理（1931）

**定理**：
> 若T是包含PA的一致递归可枚举理论，则存在句子G使得：
>
> - T ⊬ G
> - T ⊬ ¬G

**G的构造**：

```
G = "G在T中不可证"

自指：
- G声称自己不可证
- 若G可证，则G假（矛盾）
- 若¬G可证，则G可证（矛盾）
- 故G不可判定
```

---

### 2.2 技术细节：哥德尔配数

**哥德尔配数的数学定义**：

**符号编码**：

将一阶逻辑的每个符号映射到奇数：
- $0 \mapsto 1$
- $S \mapsto 3$（后继函数）
- $+ \mapsto 5$
- $\times \mapsto 7$
- $= \mapsto 9$
- $(\mapsto 11$
- $)\mapsto 13$
- $\forall \mapsto 15$
- $\exists \mapsto 17$
- $\neg \mapsto 19$
- $\land \mapsto 21$
- $\lor \mapsto 23$
- $\to \mapsto 25$
- 变元 $x_i \mapsto 2i+1$（$i \geq 1$）

**公式编码**：

对公式 $\varphi = s_1 s_2 \cdots s_n$（符号序列），定义**哥德尔数**：

$$\text{Gödel}(\varphi) = 2^{c(s_1)} \cdot 3^{c(s_2)} \cdot 5^{c(s_3)} \cdots p_n^{c(s_n)}$$

其中 $c(s_i)$ 是符号 $s_i$ 的编码，$p_n$ 是第 $n$ 个素数。

**具体例子**：

**例1：简单公式**

- 公式：$\varphi = 0 = 0$
- 符号序列：$0, =, 0$
- 编码：$c(0) = 1$，$c(=) = 9$，$c(0) = 1$
- **哥德尔数**：$\text{Gödel}(\varphi) = 2^1 \cdot 3^9 \cdot 5^1 = 2 \cdot 19683 \cdot 5 = 196830$

**例2：带量词的公式**

- 公式：$\varphi = \forall x (x = 0)$
- 符号序列：$\forall, x_1, (, x_1, =, 0, )$
- 编码：$c(\forall) = 15$，$c(x_1) = 3$，$c(() = 11$，$c(x_1) = 3$，$c(=) = 9$，$c(0) = 1$，$c()) = 13$
- **哥德尔数**：$\text{Gödel}(\varphi) = 2^{15} \cdot 3^3 \cdot 5^{11} \cdot 7^3 \cdot 11^9 \cdot 13^1 \cdot 17^{13}$（非常大的数）

**例3：自指构造**

- **关键**：可以构造公式 $\varphi$ 使得 $\text{Gödel}(\varphi)$ 的某些性质在PA中可表示
- **自指句子**：$G = $"我的哥德尔数在PA中不可证"
- **形式化**：$G = \forall x \neg \text{Prov}(x, \ulcorner G \urcorner)$
- 其中 $\ulcorner G \urcorner$ 是 $G$ 的哥德尔数，$\text{Prov}(x, y)$ 表示"$x$ 是公式 $y$ 的证明"

**可表示性**：

```
关系R可表示 ⟺
存在公式φ(x₁,...,xₙ)使得：
- 若R(a₁,...,aₙ)，则PA ⊢ φ(ā₁,...,āₙ)
- 若¬R(a₁,...,aₙ)，则PA ⊢ ¬φ(ā₁,...,āₙ)
```

---

### 2.3 证明的核心

**关键步骤的数学表述**：

**步骤1：定义"可证"关系**

在PA中可以定义关系 $\text{Prov}(x, y)$ 表示"$x$ 是公式 $y$ 的证明"。

**可表示性**：若 $\pi$ 是公式 $\varphi$ 的证明，则：
- $\text{PA} \vdash \text{Prov}(\ulcorner \pi \urcorner, \ulcorner \varphi \urcorner)$

若 $\pi$ 不是 $\varphi$ 的证明，则：
- $\text{PA} \vdash \neg \text{Prov}(\ulcorner \pi \urcorner, \ulcorner \varphi \urcorner)$

**步骤2：构造自指句子**

定义公式：
$$G = \forall x \neg \text{Prov}(x, \ulcorner G \urcorner)$$

其中 $\ulcorner G \urcorner$ 是 $G$ 的哥德尔数。

**自指性**：$G$ 声称"我的哥德尔数在PA中不可证"。

**步骤3：证明 $G$ 不可判定**

**假设** $\text{PA} \vdash G$：

- 则存在证明 $\pi$ 使得 $\text{PA} \vdash \text{Prov}(\ulcorner \pi \urcorner, \ulcorner G \urcorner)$
- 因此 $\text{PA} \vdash \exists x \text{Prov}(x, \ulcorner G \urcorner)$
- 但 $G = \forall x \neg \text{Prov}(x, \ulcorner G \urcorner)$，所以 $\text{PA} \vdash \neg G$
- 这与 $\text{PA} \vdash G$ 矛盾（若PA一致）

**结论**：若PA一致，则 $\text{PA} \nvdash G$。

**步骤4：证明 $\text{PA} \nvdash \neg G$**

**假设** $\text{PA} \vdash \neg G$：

- 则 $\text{PA} \vdash \neg \forall x \neg \text{Prov}(x, \ulcorner G \urcorner)$
- 即 $\text{PA} \vdash \exists x \text{Prov}(x, \ulcorner G \urcorner)$
- 若PA是$\omega$-一致的（更强的条件），则存在 $n$ 使得 $\text{PA} \vdash \text{Prov}(\bar{n}, \ulcorner G \urcorner)$
- 但若 $\text{PA} \vdash G$，则与步骤3矛盾
- 因此 $\text{PA} \nvdash \neg G$（在$\omega$-一致性下）

**最终结论**：若PA一致（或$\omega$-一致），则 $G$ 在PA中不可判定。

## 三、第二不完备定理

### 3.1 定理陈述

**定理**：
> 若T是包含PA的一致递归可枚举理论，则T不能证明自身的一致性

**形式化**：

```
Con(T) = "T是一致的"
    = "不存在证明使得0=1"

定理：
T ⊬ Con(T)
```

---

### 3.2 证明思路

**关键观察**：

```
第一定理的证明：
- 在T内可形式化
- 得到：Con(T) → G不可证

但：
- G不可证 ⟺ G真
- 若T ⊢ Con(T)，则T ⊢ G
- 与第一定理矛盾
```

**结论**：

- 一致性不可在系统内证明
- 需要更强的系统
- 相对一致性

## 四、对希尔伯特纲领的影响

### 4.1 三大目标的失败

**一致性**：

- ❌ 不能在系统内证明
- ✅ 但可相对证明（Gentzen）

**完备性**：

- ❌ 不可能（第一定理）
- ✅ 但语义完备性成立（Gödel, 1930）

**可判定性**：

- ❌ 不可能（Church-Turing, 1936）
- ✅ 但特殊理论可判定

---

### 4.2 希尔伯特的反应

**初始震惊**（1931）：

- 不相信
- 要求检查

**最终接受**（1930s后期）：

- 承认失败
- 但方法论保留

**遗产**：

- 形式化方法成功
- 证明论成为独立学科
- 计算机验证

## 五、现代发展

### 5.1 相对一致性

**Gentzen（1936）**：

```
PA的一致性：
- 不能在PA内证明
- 但可在PA+ε₀内证明

方法：
- 序数分析
- 超限归纳
```

**现代**：

- 序数分析持续发展
- 大基数理论
- 相对一致性层级

---

### 5.2 不完备性的普遍性

**Chaitin不完全性**（1974）：

```
信息论版本：
- 存在常数c使得
- 对任意T，存在n使得
- T不能证明"K(n) > c"

其中：
- K(n)：Kolmogorov复杂度
- 不可压缩性
```

**应用**：

- 算法信息论
- 计算复杂性
- 随机性理论

## 六、哲学意义

### 6.1 数学真理的超越性

**Gödel的柏拉图主义**：

```
不完备性 ⟹
- 数学真理超越形式系统
- 存在不可形式化的真理
- 数学是发现的，不是发明的
```

**形式主义的回应**：

- 承认局限
- 但方法论仍有效
- 实用主义立场

```
可计算性理论：

- 与Gödel定理等价
```

**现代**：

- 计算机辅助证明

- 但人类直觉仍重要

## 九、数学公式总结

### 核心公式

1. **第一不完备定理**：
   $$\text{若 } T \text{ 一致且递归可枚举，则存在 } G \text{ 使得 } T \nvdash G \text{ 且 } T \nvdash \neg G$$

2. **自指句子**：
   $$G = \forall x \neg \text{Prov}(x, \ulcorner G \urcorner)$$

3. **哥德尔配数**：
   $$\text{Gödel}(\varphi) = 2^{c(s_1)} \cdot 3^{c(s_2)} \cdots p_n^{c(s_n)}$$

4. **可证关系**：
   $$\text{Prov}(x, y) \iff x \text{ 是公式 } y \text{ 的证明}$$

5. **第二不完备定理**：
   $$\text{若 } T \text{ 一致且递归可枚举，则 } T \nvdash \text{Con}(T)$$

6. **一致性陈述**：
   $$\text{Con}(T) = \neg \exists x \text{Prov}(x, \ulcorner 0 = 1 \urcorner)$$

7. **相对一致性**：
   $$\text{Con}(T_1) \to \text{Con}(T_2)$$

8. **序数分析**：
   $$|T| = \min\{\alpha : T + \alpha \vdash \text{Con}(T)\}$$

9. **PA的证明论序数**：
   $$|\text{PA}| = \varepsilon_0$$

10. **连续统假设**：
    $$2^{\aleph_0} = \aleph_1$$

11. **Kolmogorov复杂度**：
    $$K(n) = \min\{|p| : U(p) = n\}$$

12. **Chaitin不完全性**：
    $$\exists c \forall T \exists n: T \nvdash K(n) > c$$
