## 一、判定问题（Entscheidungsproblem）的提出

### 1.1 希尔伯特-阿克曼的经典表述（1928）

**判定问题**：

```
问题：是否存在一个**算法**（机械过程），
对于任意一阶逻辑公式φ，
能够判定φ是否可满足（或是否有效）？

即：
给定：一阶逻辑公式φ
输出：是（可满足）或否（不可满足）
```

**等价形式**：

```
形式1（可满足性）：
是否存在算法判定 Sat(φ)？

形式2（有效性）：
是否存在算法判定 Val(φ)？

形式3（可证性）：
是否存在算法判定 Pr(φ)？
```

**希尔伯特-阿克曼的原文**（1928《数理逻辑基础》）：

> "Entscheidungsproblem ist gelöst, wenn man ein Verfahren kennt, das bei einem vorgelegten logischen Ausdruck durch endlich viele Operationen die Entscheidung über die Allgemeingültigkeit bzw. Erfüllbarkeit erlaubt."
>
> "判定问题当且仅当我们知道一个程序，使得对于给定的逻辑表达式，通过有限次运算能够判定其普遍有效性或可满足性时，才算解决。"

---

### 1.2 问题的历史意义

**为什么重要**：

```
希尔伯特纲领：
1. 公理化数学 ✅
2. 证明一致性 ✅
3. 判定所有数学问题 ❓

判定问题 = 纲领的终极目标
→ 能否机械解决所有数学问题？
```

**三种可能答案**：

```
答案1（希尔伯特期望）：✅ 是，存在算法
答案2（Church-Turing）：❌ 否，不可判定
答案3（限制版本）：🟡 部分可判定

实际答案：答案2（负解）
```

## 二、判定问题的特殊情况

### 2.1 命题逻辑的判定

**命题逻辑**：✅ **可判定**

```
真值表方法：
给定命题公式φ
→ 列出所有可能的真值赋值
→ 检查是否有赋值使φ为真

复杂度：O(2ⁿ)，其中n = 变量数
→ 虽然指数时间，但可判定
```

**例子**：

```
公式：(P ∨ Q) ∧ (¬P ∨ R) ∧ (¬Q ∨ R)

真值表：
P  Q  R  |  (P∨Q)  (¬P∨R)  (¬Q∨R)  |  结果
────────────────────────────────────
T  T  T  |    T       T        T     |   T
T  T  F  |    T       F        F     |   F
T  F  T  |    T       T        T     |   T
T  F  F  |    T       F        T     |   F
F  T  T  |    T       T        T     |   T
F  T  F  |    T       T        F     |   F
F  F  T  |    F       T        T     |   F
F  F  F  |    F       T        T     |   F

结论：可满足（有多组真值赋值）
```

---

### 2.2 一阶逻辑的判定：一般情况

**一阶逻辑**：❌ **不可判定**

```
Church定理（1936）：
一阶逻辑的可满足性问题是**不可判定**的

即：
不存在算法判定任意一阶公式是否可满足
```

**证明思路**：

```
步骤1：定义"算法"的精确概念
→ Turing机（Turing, 1936）
→ λ演算（Church, 1936）

步骤2：将判定问题归约到停机问题
→ 停机问题不可判定（Turing）
→ 判定问题也不可判定

步骤3：得出负解
→ 不存在判定算法
```

## 三、Church-Turing定理（1936）

### 3.1 三个等价的负解

**同时发表**（1936）：

```
Church（λ演算）：
"An unsolvable problem of elementary number theory"
→ 一阶逻辑不可判定

Turing（Turing机）：
"On computable numbers, with an application to the Entscheidungsproblem"
→ 停机问题不可判定
→ 判定问题不可判定

Rosser（推广）：
证明两者等价
```

---

### 3.2 停机问题

**问题陈述**：

```
给定：
- Turing机M
- 输入w

问题：
M在输入w上是否会停机？

即：
Halt(M, w) = 是（停机）或否（不停机）
```

**不可判定性**：

```
定理（Turing, 1936）：
停机问题是**不可判定**的

证明（对角线法）：
假设存在算法H判定停机问题
→ 构造自相矛盾的机器D
→ 矛盾
→ 假设错误
```

**证明概要**：

```
假设：存在Halt(M, w)

构造机器D：
D(w) = {
    if Halt(w, w) then
        进入无限循环
    else
        停机
}

矛盾：
D(D)停机 ⟺ Halt(D, D) = 否
         ⟺ D(D)不停机 ❌

结论：Halt不存在
```

---

### 3.3 归约到判定问题

**归约**：

```
停机问题 ≤ 判定问题

即：
如果能判定一阶逻辑
→ 就能判定停机问题
→ 矛盾（停机问题不可判定）
→ 判定问题也不可判定
```

**具体归约**：

```
对任意Turing机M和输入w：
构造一阶公式φ_{M,w}

使得：
φ_{M,w}可满足 ⟺ M在w上停机

如果能判定φ_{M,w}
→ 就能判定M在w上是否停机
→ 与停机问题不可判定矛盾
```

## 四、判定问题的部分正解

### 4.1 可判定的子类

**Monadic一阶逻辑**（1928, Löwenheim）：

```
Monadic谓词：只有一元谓词（无二元及以上）

例：
∀x (P(x) → Q(x))
∃x (P(x) ∧ Q(x))

判定算法：存在
复杂度：双重指数时间
```

**命题逻辑**：✅ 可判定

**等式理论**：✅ 可判定（Ackermann, 1928）

**Presburger算术**（只有加法）：✅ 可判定（1929）

**实数闭域理论**（Tarski）：✅ 可判定（1930s）

### 4.2 希尔伯特第十问题

**问题**（1900）：

```
给定：
- 多项式方程 P(x₁,...,xₙ) = 0
- 系数为整数

问题：
是否存在算法判定该方程是否有整数解？

即：
判定 Diophantine方程的可解性
```

**解答**（1970, Matiyasevich）：

```
答案：❌ **不可判定**

Matiyasevich定理（MRDP定理）：
Diophantine方程的可解性问题是不可判定的

即：
不存在算法判定任意Diophantine方程是否有整数解
```

**历史意义**：

```
希尔伯特第十问题：
→ 23问题的第10个
→ 期待正解
→ 实际负解

Matiyasevich的证明：
→ 使用递归可枚举集
→ 连接数论与逻辑
→ 重要里程碑
```

## 五、可计算性理论的基础

### 5.1 算法的精确定义

**Church-Turing论题**：

```
论题（非定理）：
可计算函数 = Turing可计算函数

即：
直观的"可计算" ↔ 形式化的Turing机
```

**三种等价的模型**：

```
1. Turing机（Turing, 1936）
   - 状态转换
   - 带子读写

2. λ演算（Church, 1936）
   - 函数抽象
   - 函数应用

3. 递归函数（Gödel, 1934）
   - 原始递归
   - μ递归

等价性：三者互相模拟
```

---

### 5.2 可计算性层级

**递归可枚举集（r.e.）**：

```
定义：
A是r.e. ⟺ 存在Turing机M使得
          A = {w | M(w)停机}

性质：
- r.e.集的补不一定是r.e.
- 停机问题是r.e.的
- 停机问题的补不是r.e.
```

**递归集（可判定）**：

```
定义：
A是递归的 ⟺ A和A的补都是r.e.

等价：
A是递归的 ⟺ 存在算法判定w ∈ A

关系：
递归 ⊂ 递归可枚举
```

**不可计算性层级**：

```
0级：递归集（可判定）
    例：偶数集、素数集

1级：递归可枚举但非递归
    例：停机问题

2级：非递归可枚举
    例：停机问题的补
```

## 六、现代发展：SMT求解器

### 6.1 可满足性模理论（SMT）

**SMT问题**：

```
给定：
- 一阶逻辑公式φ
- 背景理论T（如：实数、整数、数组）

问题：
φ在理论T中是否可满足？

即：
判定 φ ∧ T 是否可满足
```

**可判定的理论**：

```
✅ 线性实数算术（LRA）
✅ 线性整数算术（LIA）
✅ 数组理论
✅ 位向量
✅ 未解释函数（EUF）
```

### 6.2 现代SMT求解器

**应用**：

## 七、与希尔伯特纲领的关系

### 7.1 对纲领的影响

**希尔伯特的期望**：

```
判定问题解决 → 机械解决所有数学问题
→ 纲领的终极实现
```

**实际的冲击**：

```
判定问题负解：
→ 不能机械解决所有数学问题
→ 纲领部分失败
→ 但方法论影响深远
```

**辩证关系**：

```
希尔伯特纲领：
- 目标过高 ❌（判定问题、绝对一致性）
- 方法论成功 ✅（公理化、形式化）

结论：
虽然目标未完全实现，
但开创了现代逻辑与计算理论
```

---

### 7.2 现代回响

**形式化数学**：

```
虽然一般判定问题不可解，
但特殊理论可判定 → SMT求解器

应用：
- 程序验证
- 定理证明
- 形式化数学（Lean/Coq）
```

**计算复杂性**：

```
判定问题 → 可计算性
→ 计算复杂性（P vs NP）
→ 现代计算机科学

希尔伯特的遗产：
不仅是数学，更是计算理论的开端
```

## 八、总结

### 判定问题的历史地位

**发展时间线**：

```
1900：希尔伯特第10问题提出
1928：判定问题精确表述
1936：Church-Turing负解
1970：第10问题负解（Matiyasevich）
2000s：SMT求解器（部分正解）
```

**双重意义**：

```
负面结果：
→ 否定希尔伯特的部分期望
→ 揭示计算的极限

正面意义：
→ 开创可计算性理论
→ 定义"算法"概念
→ 现代计算机科学基础
```

---

### 最终评价

**希尔伯特的问题**：

```
问题：是否存在判定算法？
答案：一般情况否，特殊情况是

影响：
→ 可计算性理论
→ 计算机科学基础
→ 现代逻辑学
```

**历史功绩**：

希尔伯特的判定问题虽然得到**负解**，但：
- 推动了可计算性理论的发展
- 促成了Turing机与算法概念的精确化
- 为现代计算机科学奠基
- 部分理论的可判定性催生了SMT求解器等实用工具

**现代启示**：

虽然不能判定**所有**数学问题，但可以：
- 判定**特定理论**中的问题（SMT）
- 用**形式化方法**辅助证明（证明助手）
- 通过**限制范围**获得可判定性

## 九、数学公式总结

### 核心公式

1. **判定问题的形式化表述**：
   $$\text{Entscheidungsproblem}: \exists \text{算法} \text{ 使得 } \forall \varphi \in \text{FO}, \text{ 判定 } \text{Sat}(\varphi)?$$

2. **可满足性定义**：
   $$\text{Sat}(\varphi) \iff \exists \mathcal{M} : \mathcal{M} \models \varphi$$

3. **有效性定义**：
   $$\text{Val}(\varphi) \iff \forall \mathcal{M} : \mathcal{M} \models \varphi$$

4. **Church定理（1936）**：
   $$\text{Entscheidungsproblem} \text{ 不可判定}$$

5. **停机问题形式化**：
   $$\text{Halt}(M, w) = \begin{cases}
   1 & \text{如果 } M(w) \text{ 停机} \\
   0 & \text{如果 } M(w) \text{ 不停机}
   \end{cases}$$

6. **停机问题不可判定性**：
   $$\neqqqxists \text{算法 } H \text{ 使得 } H(M, w) = \text{Halt}(M, w)$$

7. **递归可枚举集定义**：
   $$A \text{ 是 r.e.} \iff \exists \text{Turing机 } M : A = \{w | M(w) \text{ 停机}\}$$

8. **递归集定义**：
   $$A \text{ 是递归的} \iff A \text{ 是 r.e.} \land \overline{A} \text{ 是 r.e.}$$

9. **Matiyasevich定理（MRDP定理，1970）**：
   $$\text{Diophantine方程可解性问题不可判定}$$

10. **SMT问题形式化**：
    $$\text{SMT}(\varphi, T) = \text{Sat}(\varphi \land T)$$

11. **Turing机计算函数**：
    $$f: \Sigma^* \rightarrow \Sigma^*, \quad f(w) = M(w) \text{（如果 } M(w) \text{ 停机）}$$

12. **可计算性层级**：
    $$\text{递归集} \subset \text{递归可枚举集} \subset \mathcal{P}(\Sigma^*)$$
