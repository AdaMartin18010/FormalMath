# 证明论与序数分析：希尔伯特纲领的延续

## 一、Gentzen的突破

### 1.1 一致性证明

**问题**：
> 如何证明PA的一致性？

**Gödel的障碍**：

- 不能在PA内证明
- 需要更强系统

**Gentzen的方法**（1936）：

```
使用：
- 超限归纳
- 序数ε₀
- 证明变换

结果：
PA的一致性在PA+ε₀中可证
```

---

### 1.2 序数分析

**核心思想**：

```
给定理论T

序数分析：
- 找到最小序数α
- 使得T的一致性可在T+α中证明
- α = |T|（证明论序数）
```

**意义**：

- 度量理论强度
- 一致性证明
- 理解数学结构

---

## 二、相继式演算

### 2.1 自然演绎vs相继式

**自然演绎**（Gentzen, 1934）：

```
推理规则：
- 引入规则（I）
- 消去规则（E）

例子：
→I: 从A⊢B得到⊢A→B
→E: 从⊢A→B和⊢A得到⊢B
```

**相继式演算**（Gentzen, 1934）：

```
相继式：
Γ ⊢ Δ

其中：
- Γ：前提（多公式）
- Δ：结论（多公式）

规则：
- 左规则（L）
- 右规则（R）
```

---

### 2.2 切消定理

**切消定理**（Hauptsatz）：

```
若证明使用"切"规则，则存在不使用切的证明

意义：
- 证明的结构化
- 一致性证明
- 计算性质
```

**应用**：

- 子公式性质
- 一致性
- 可判定性（某些情况）

---

## 三、序数分析技术

### 3.1 证明变换

**基本思想**：

```
给定证明π

变换：
1. 消除切
2. 降低复杂度
3. 使用超限归纳

结果：
- 证明长度有界
- 序数度量
```

---

### 3.2 序数记号

**Cantor正规形式**：

```
序数α可表为：
α = ω^α₁ + ... + ω^αₙ

其中：
α₁ ≥ ... ≥ αₙ

例子：
ε₀ = sup{ω, ω^ω, ω^(ω^ω), ...}
```

**应用**：

- 序数分析
- 一致性证明
- 理论强度比较

## 四、主要结果

### 4.1 PA的序数分析

**Gentzen（1936）**：

```
|PA| = ε₀

证明：
- PA的一致性
- 在PA+ε₀中可证
- ε₀是最小的
```

**意义**：

- PA的证明论强度
- 一致性相对证明

## 五、反推数学

### 5.1 Simpson纲领

**问题**：
> 哪些数学定理等价于哪些公理？

**方法**：

```
给定定理T

找到：
- 最小公理系统A
- 使得A ⊢ T
- 且T ⊢ A（在弱系统中）
```

**结果**：

- 理解数学结构
- 公理的等价性
- 理论的层次

---

### 5.2 主要等价性

**Big Five**：

```
RCA₀：递归理解
```
