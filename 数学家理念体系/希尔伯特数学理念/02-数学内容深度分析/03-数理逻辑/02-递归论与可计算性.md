# 递归论与可计算性：希尔伯特判定问题的否定答案

## 一、希尔伯特的Entscheidungsproblem

### 1.1 问题的提出（1928）

**希尔伯特与Ackermann**《数理逻辑基础》：
> "判定问题（Entscheidungsproblem）：是否存在机械程序，判定任意给定的一阶逻辑公式是否可证？"

**具体形式**：

```
输入：一阶逻辑公式φ
输出：φ是否为逻辑真（⊨φ）？

希尔伯特信念：
"应该存在这样的算法。数学中没有ignorabimus（不可知）。"
```

---

### 1.2 问题的哲学背景

**形式主义的要求**：

```
若数学是符号游戏，
则应该有机械规则判定符号串的性质
    ↓
可判定性 = 形式主义的终极目标
```

**与希尔伯特纲领的关系**：

- 纲领核心：有限主义元数学
- 判定问题：终极的有限主义算法

---

## 二、Church-图灵否定（1936）

### 2.1 可计算函数的三种等价定义

**Church的λ-演算**（1936）：

```
λ-可定义函数：
可通过λ-演算表达的函数

例子：
后继：λn.λf.λx.f(nfx)
加法：λm.λn.λf.λx.m f (n f x)
```

**图灵机**（1936）：

```
图灵机：
- 无限纸带
- 有限状态
- 读写头

可计算：存在图灵机在有限步内计算出结果

例子：
判断n是否是素数：可计算
停机问题：不可计算（后证）
```

**Gödel递归函数**（1934）：

```
原始递归函数 + μ-算子

原始递归：
- 初始函数（零、后继、投影）
- 复合、原始递归

μ-算子：
μy[R(x,y)] = 最小的y使得R(x,y)成立
（若不存在则发散）
```

---

### 2.2 Church-图灵论题

**论题**：
> "直觉上可计算的函数 = λ-可定义 = 图灵可计算 = μ-递归"

**性质**：

- 这是**论题**（thesis），非定理
- 无法形式证明（"直觉可计算"非形式概念）
- 但普遍接受（大量证据）

**希尔伯特的反应**：
初期怀疑，后来接受

---

### 2.3 判定问题的否定答案

**Church定理**（1936）：
> 一阶逻辑的可证性问题（Entscheidungsproblem）**不可判定**

**证明思路**：

```
步骤1：将停机问题归约到可证性问题
步骤2：若可证性可判定，则停机问题可判定
步骤3：但停机问题不可判定（图灵证明）
步骤4：矛盾，故可证性不可判定
```

**对希尔伯特的打击**：
> "数学中确实有ignorabimus。某些问题本质上不可机械解决。"

## 三、递归与递归可枚举

### 3.1 递归集与递归可枚举集

**定义**：

```
集合A ⊆ ℕ是递归的（可判定的）
⟺ 特征函数χ_A可计算
χ_A(n) = 1若n∈A，否则0

集合A是递归可枚举的（r.e.）
⟺ A = ∅ 或 A = {f(0), f(1), f(2), ...}，f可计算
等价：A的部分特征函数可计算
```

**关系**：

```
递归 ⊂ 递归可枚举

例子：
- ℕ：递归
- 停机集H = {e | 第e个图灵机在输入e上停机}：r.e.但非递归
- H的补集：非r.e.
```

---

### 3.2 停机问题

**定理（图灵，1936）**：
> 停机集H不可判定

**证明（对角化）**：

```
反设：存在算法halt(e,x)判定"第e个TM在输入x上停机"

构造程序d：
def d(e):
    if halt(e,e):
        loop_forever()
    else:
        return

问题：d(d)是否停机？
- 若d(d)停机 → halt(d,d)=True → d(d)死循环 ⚠️矛盾
- 若d(d)不停 → halt(d,d)=False → d(d)返回（停机）⚠️矛盾

结论：halt不存在
```

**与哥德尔不完备性的联系**：

- 哥德尔：存在真但不可证的命题
- 图灵：存在不可判定的问题
- **统一**：自指导致不可判定性

---

### 3.3 Rice定理

**定理**：
> 任何关于递归函数的非平凡语义性质都不可判定

**形式**：

```
设P是递归函数的性质（如"函数f总是返回偶数"）
若P非平凡（有函数满足P，也有函数不满足P）
则判定"第e个TM计算的函数满足P"不可判定
```

**推论**：

- 无法判定程序是否总终止
- 无法判定两程序是否等价
- 无法判定程序是否无bug

## 四、算术层级

### 4.1 Π, Σ层级

**定义**：

```
Σ₀ = Π₀ = 递归集

Σₙ₊₁ = {A | A = {x | ∃y R(x,y)}，R∈Πₙ}
Πₙ₊₁ = {A | A = {x | ∀y R(x,y)}，R∈Σₙ}

例子：
Σ₁：r.e.集
Π₁：co-r.e.集
Σ₂：存在无穷多个y使得R(x,y)（R∈Π₁）
```

**性质**：

```
Σₙ ∪ Πₙ ⊊ Δₙ₊₁ ⊊ Σₙ₊₁ ∩ Πₙ₊₁

Δₙ = Σₙ ∩ Πₙ（在Σₙ和Πₙ中都的集合）
```

---

### 4.2 应用：分析哥德尔语句

**哥德尔语句G**：

```
"G不可证"（在PA中）

形式：
G ⟺ ∀y ¬Proof(y, ⌜G⌝)

层级：
- Proof(y,x)是递归关系（可机械检查证明）
- ¬Proof(y,⌜G⌝)是Π₁公式
- G是Π₁语句

结论：
G的真值在算术层级的Π₁层
PA无法判定Π₁语句的所有真值
```

## 五、相对可计算性

### 5.1 Oracle图灵机

**定义**：

```
图灵机T^A：带有oracle A的图灵机
- A是集合（"黑盒"）
- T可以查询"n∈A?"（一步得到答案）

A-可计算：存在T^A计算f
```

**例子**：

```
H'（带停机oracle的停机问题）
H' = {e | T_e^H 在输入e上停机}

定理：H'不H-可计算
（对角化论证）
```

---

### 5.2 图灵度

**定义**：

```
A ≤_T B（A图灵归约到B）
⟺ A是B-可计算的

图灵度：
deg(A) = {B | A ≤_T B 且 B ≤_T A}
```

**结构**：

```
图灵度构成偏序（Turing degrees）
- 最小度：0（递归集）
- 最大度：不存在
- 非常复杂的结构（有不可比度）

Post问题（1944）：
存在0 < d < 0'的度d吗？（0'=deg(H)）
答案：是（Friedberg-Muchnik, 1957）
```

## 六、计算复杂性初步

### 6.1 时间复杂性类

**定义**：

```
P = 多项式时间可判定的语言
  = ⋃ₖ DTIME(nᵏ)

NP = 非确定性多项式时间
   = 多项式时间可验证的语言

例子：
- 判断n是否素数：P（AKS算法，2002）
- 布尔可满足性（SAT）：NP
```

**P vs NP问题**（Clay千禧年难题）：

```
P = NP?

信念：P ≠ NP
原因：无人找到SAT的多项式算法（70年失败）
但无人证明不存在
```

---

### 6.2 与递归论的关系

**层次关系**：

```
递归论（可计算性）
    ↓
算法存在性
    ↓
复杂性理论（效率）
    ↓
算法的实际可行性

递归 ⊃ P ⊃ LOGSPACE
不可递归 ⊃ EXP ⊃ NP ⊃ P
```

## 七、希尔伯特遗产

### 7.1 判定问题的部分成功

**可判定片段**：

```
一阶逻辑（一般）：不可判定 ❌
但：
- 命题逻辑：可判定 ✅（NP完全）
- 单子一阶逻辑：可判定 ✅
- Presburger算术（加法）：可判定 ✅
- 实闭域理论：可判定 ✅（Tarski）

策略：
限制表达力 → 获得可判定性
```

---

### 7.2 算法思想的影响

**计算机科学的理论基础**：

```
希尔伯特的Entscheidungsproblem
    ↓
图灵机（1936）
    ↓
冯·诺依曼架构（1945）
    ↓
现代计算机

评价：
虽然判定问题被否定，
但引发了计算理论的诞生
```

---

### 7.3 形式主义的修正

**从绝对到相对**：

```
希尔伯特原信念：
"所有问题可机械解决"

现实：
某些问题不可判定

修正的形式主义：
"尽可能形式化，
承认不可判定性的存在，
研究可判定片段"
```

**现代共识**：

- 形式化方法有价值
- 但不能解决所有问题
- 不可判定性是**本质极限**

## 八、与哥德尔定理的呼应

### 统一视角

**三个不可能性**：

| 定理 | 年份 | 陈述 | 意义 |
|------|------|------|------|
| 哥德尔第一不完备 | 1931 | PA不完备 | 存在不可证的真命题 |
| 哥德尔第二不完备 | 1931 | PA⊬Con(PA) | 无自证一致性 |
| Church-图灵 | 1936 | 一阶逻辑不可判定 | 无通用判定算法 |

**共同模式**：

- 自指导致不可能
- 形式系统的内在极限
- 希尔伯特理想的破灭

**但方法论胜利**：

- 形式化技术成为标准
- 递归论成为独立学科
- 计算机科学诞生
