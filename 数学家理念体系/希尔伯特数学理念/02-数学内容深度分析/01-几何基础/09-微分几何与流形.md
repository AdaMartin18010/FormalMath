## 一、从欧氏几何到流形

### 1.1 流形的概念

**定义**：

```
n维流形M：
- 局部同胚于Rⁿ
- 可微结构
- 光滑变换

例子：
- 球面Sⁿ
- 环面Tⁿ
- 射影空间Pⁿ
```

**与欧氏几何**：

- 局部：类似Rⁿ
- 全局：可能有曲率
- 推广：希尔伯特几何

---

### 1.2 切空间

**定义**：

```
点p∈M的切空间T_pM：
- 切向量的空间
- 维数 = dim M
- 局部坐标表示
```

**切丛**：

```
TM = ∪_{p∈M} T_pM

性质：
- 向量丛
- 2n维流形
- 自然投影
```

---

## 二、Riemann几何

### 2.1 度规

**Riemann度规**：

```
g: TM × TM → ℝ

性质：
- 对称
- 正定
- 光滑

局部：
g = ∑ gᵢⱼ dxᵢ ⊗ dxⱼ
```

**距离**：

```
曲线长度：
L(γ) = ∫ √g(γ'(t), γ'(t)) dt

距离：
d(p,q) = inf L(γ)
（γ连接p和q）
```

---

### 2.2 曲率

**曲率张量**：

```
Riem(g)：
- 度量度规的曲率
- 4阶张量
- 局部性质

标量曲率：
R = g^{ij} Rᵢⱼ
```

**几何意义**：

- 正曲率：球面
- 负曲率：双曲空间
- 零曲率：欧氏空间

---

## 三、Levi-Civita联络

### 3.1 协变导数

**定义**：

```
联络∇：
- 协变导数
- 平行移动
- 测地线

性质：
- 线性
- Leibniz法则
- 无挠（Torsion-free）
```

**Levi-Civita联络**：

```
唯一联络使得：
- 与度规相容：∇g = 0
- 无挠：T = 0
```

---

### 3.2 测地线

**定义**：

```
测地线γ：
∇_{γ'}γ' = 0

性质：
- 局部最短
- 自平行
- 存在唯一（给定初值）
```

**应用**：

- 最短路径
- 物理：自由粒子
- 几何：曲率

---

## 四、Gauss-Bonnet定理

### 4.1 2维情况

**定理**：

```
紧2维Riemann流形M：

∫_M K dA = 2πχ(M)

其中：
- K：Gauss曲率
- χ(M)：Euler示性数
```

**意义**：

- 曲率与拓扑
- 全局与局部
- 几何与拓扑

---

### 4.2 高维推广

**Chern-Gauss-Bonnet**：

```
n维流形：
∫_M Pf(Ω) = (2π)^(n/2) χ(M)

其中：
- Pf：Pfaffian
- Ω：曲率形式
```

---

## 五、Einstein流形

### 5.1 Einstein方程

**定义**：

```
流形(M,g)是Einstein的 ⟺
Ric = λg

其中：
- Ric：Ricci曲率
- λ：常数
```

**物理意义**：

- 真空Einstein方程
- 宇宙学常数
- 引力理论

---

### 5.2 例子

**常曲率空间**：

```
- 球面：正曲率
- 双曲空间：负曲率
- 欧氏空间：零曲率

性质：
- Einstein流形
- 最大对称性
- 分类完整
```
