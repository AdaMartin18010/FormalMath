## 一、从欧氏几何到流形

### 1.1 流形的概念

**定义**：

```
n维流形M：
- 局部同胚于Rⁿ
- 可微结构
- 光滑变换

例子：
- 球面Sⁿ
- 环面Tⁿ
- 射影空间Pⁿ
```

**与欧氏几何**：

- 局部：类似Rⁿ
- 全局：可能有曲率
- 推广：希尔伯特几何

---

### 1.2 切空间

**定义**：

```
点p∈M的切空间T_pM：
- 切向量的空间
- 维数 = dim M
- 局部坐标表示
```

**切丛**：

```
TM = ∪_{p∈M} T_pM

性质：
- 向量丛
- 2n维流形
- 自然投影
```

## 二、Riemann几何

### 2.1 度规

**Riemann度规的数学定义**：

**Riemann度规** $g$ 是切丛 $TM$ 上的（0,2）型张量场：

$$g: TM \times TM \to \mathbb{R}$$

**性质**：

- **对称性**：$g(X, Y) = g(Y, X)$（对所有切向量 $X, Y$）
- **正定性**：$g(X, X) \geq 0$，且 $g(X, X) = 0 \iff X = 0$
- **光滑性**：$g$ 是光滑的（$C^\infty$）

**局部坐标表示**：

在局部坐标 $(x_1, \ldots, x_n)$ 下：

$$g = \sum_{i,j=1}^n g_{ij} dx_i \otimes dx_j$$

其中 $g_{ij} = g\left(\frac{\partial}{\partial x_i}, \frac{\partial}{\partial x_j}\right)$ 是度规矩阵的元素。

**距离的数学定义**：

**曲线长度**：对曲线 $\gamma: [a, b] \to M$：

$$L(\gamma) = \int_a^b \sqrt{g(\gamma'(t), \gamma'(t))} dt$$

**距离**：对 $p, q \in M$：

$$d(p, q) = \inf\{L(\gamma) : \gamma \text{ 是连接 } p \text{ 和 } q \text{ 的光滑曲线}\}$$

**具体例子**：

**例1：欧氏空间**

- **流形**：$M = \mathbb{R}^n$
- **度规**：$g = dx_1^2 + dx_2^2 + \cdots + dx_n^2$（标准欧氏度规）
- **距离**：$d(p, q) = \sqrt{\sum_{i=1}^n (q_i - p_i)^2}$（欧氏距离）

**例2：球面**

- **流形**：$M = S^2 = \{(x, y, z) : x^2 + y^2 + z^2 = 1\}$
- **度规**：从 $\mathbb{R}^3$ 的欧氏度规诱导
- **局部坐标**：球坐标 $(\theta, \phi)$，度规：$g = d\theta^2 + \sin^2\theta d\phi^2$
- **距离**：大圆弧长度

**例3：双曲平面**

- **流形**：$M = \mathbb{H}^2$（Poincaré上半平面）
- **度规**：$g = \frac{dx^2 + dy^2}{y^2}$（$y > 0$）
- **距离**：双曲距离（与欧氏距离不同）

---

### 2.2 曲率

**曲率张量**：

```
Riem(g)：
- 度量度规的曲率
- 4阶张量
- 局部性质

标量曲率：
R = g^{ij} Rᵢⱼ
```

**几何意义**：

- 正曲率：球面
- 负曲率：双曲空间
- 零曲率：欧氏空间

## 三、Levi-Civita联络

### 3.1 协变导数

**定义**：

```
联络∇：
- 协变导数
- 平行移动
- 测地线

性质：
- 线性
- Leibniz法则
- 无挠（Torsion-free）
```

**Levi-Civita联络**：

```
唯一联络使得：
- 与度规相容：∇g = 0
- 无挠：T = 0
```

---

### 3.2 测地线

**测地线的数学定义**：

曲线 $\gamma: I \to M$ 是**测地线**，若：

$$\nabla_{\gamma'(t)} \gamma'(t) = 0$$

对所有 $t \in I$，其中 $\nabla$ 是Levi-Civita联络。

**局部坐标方程**：

在局部坐标 $(x_1, \ldots, x_n)$ 下，测地线方程：

$$\frac{d^2 x_i}{dt^2} + \sum_{j,k=1}^n \Gamma_{jk}^i \frac{dx_j}{dt} \frac{dx_k}{dt} = 0$$

其中 $\Gamma_{jk}^i$ 是Christoffel符号：

$$\Gamma_{jk}^i = \frac{1}{2} \sum_{l=1}^n g^{il} \left(\frac{\partial g_{jl}}{\partial x_k} + \frac{\partial g_{kl}}{\partial x_j} - \frac{\partial g_{jk}}{\partial x_l}\right)$$

**具体例子**：

**例1：欧氏空间中的测地线**

- **流形**：$\mathbb{R}^n$
- **测地线**：直线 $\gamma(t) = p + tv$（$p, v \in \mathbb{R}^n$）
- **验证**：$\nabla_{\gamma'} \gamma' = 0$（因为 $\Gamma_{jk}^i = 0$）

**例2：球面上的测地线**

- **流形**：$S^2$
- **测地线**：大圆（球面与过球心平面的交）
- **性质**：局部最短路径

**例3：双曲平面中的测地线**

- **流形**：$\mathbb{H}^2$（Poincaré上半平面）
- **测地线**：垂直于实轴的半直线或半圆
- **性质**：双曲距离的极值曲线

**应用**：

- 最短路径
- 物理：自由粒子
- 几何：曲率

## 四、Gauss-Bonnet定理

### 4.1 2维情况

**定理**：

```
紧2维Riemann流形M：

∫_M K dA = 2πχ(M)

其中：
- K：Gauss曲率
- χ(M)：Euler示性数
```

**意义**：

- 曲率与拓扑
- 全局与局部
- 几何与拓扑

---

### 4.2 高维推广

**Chern-Gauss-Bonnet**：

```
n维流形：
∫_M Pf(Ω) = (2π)^(n/2) χ(M)

其中：
- Pf：Pfaffian
- Ω：曲率形式
```


## 九、数学公式总结

### 核心公式

1. **Riemann度规**：
   $$g = \sum_{i,j=1}^n g_{ij} dx_i \otimes dx_j$$

2. **曲线长度**：
   $$L(\gamma) = \int_a^b \sqrt{g(\gamma'(t), \gamma'(t))} dt$$

3. **距离**：
   $$d(p, q) = \inf\{L(\gamma) : \gamma \text{ 连接 } p \text{ 和 } q\}$$

4. **测地线方程**：
   $$\frac{d^2 x_i}{dt^2} + \sum_{j,k=1}^n \Gamma_{jk}^i \frac{dx_j}{dt} \frac{dx_k}{dt} = 0$$

5. **Christoffel符号**：
   $$\Gamma_{jk}^i = \frac{1}{2} \sum_{l=1}^n g^{il} \left(\frac{\partial g_{jl}}{\partial x_k} + \frac{\partial g_{kl}}{\partial x_j} - \frac{\partial g_{jk}}{\partial x_l}\right)$$

6. **曲率张量**：
   $$R(X, Y)Z = \nabla_X \nabla_Y Z - \nabla_Y \nabla_X Z - \nabla_{[X, Y]} Z$$

7. **Ricci曲率**：
   $$\text{Ric}(X, Y) = \sum_{i=1}^n g(R(e_i, X)Y, e_i)$$

8. **标量曲率**：
   $$R = \sum_{i,j=1}^n g^{ij} \text{Ric}_{ij}$$

9. **Gauss-Bonnet定理（2维）**：
   $$\int_M K dA = 2\pi \chi(M)$$

10. **Einstein方程**：
    $$\text{Ric} - \frac{1}{2} R g = 8\pi G T$$

11. **切空间**：
    $$T_p M = \{\gamma'(0) : \gamma \text{ 是过 } p \text{ 的曲线}\}$$

12. **切丛**：
    $$TM = \bigcup_{p \in M} T_p M$$
