## 一、拓扑空间的公理化

### 1.1 Hausdorff的公理化（1914）

**拓扑空间**：

```
集合X + 拓扑τ

其中：
τ ⊆ P(X)满足：
1. ∅, X ∈ τ
2. 任意并封闭
3. 有限交封闭

开集：
U ∈ τ

闭集：
X \ U（U开）
```

**与希尔伯特几何**：

```
几何 → 拓扑：
- 点、线、面 → 点、开集
- 关联关系 → 属于关系
- 公理化方法 → 拓扑公理
```

---

### 1.2 连续映射

**定义**：

```
f: X → Y连续 ⟺
对任意开集V ⊆ Y：
f⁻¹(V)是X的开集

等价：
对任意闭集F ⊆ Y：
f⁻¹(F)是X的闭集
```

**性质**：

- 复合连续
- 恒等连续
- 同胚：连续双射，逆也连续

## 二、基本拓扑概念

### 2.1 连通性

**连通空间**：

```
X连通 ⟺
X不能分解为两个非空不相交开集的并

等价：
X和∅是X仅有的既开又闭的子集
```

**道路连通**：

```
X道路连通 ⟺
任意两点可用连续路径连接

性质：
道路连通 ⟹ 连通
但逆不成立
```

---

### 2.2 紧致性

**定义**：

```
X紧致 ⟺
任意开覆盖有有限子覆盖

等价（Hausdorff）：
X的任意序列有收敛子列
```

**性质**：

- 紧致集的闭子集紧致
- 连续像紧致
- 有限积紧致（Tychonoff）

---

## 三、同伦理论

### 3.1 基本群

**定义**：

```
π₁(X, x₀)：
- 基于x₀的闭路径的同伦类
- 群运算：路径复合

性质：
- 拓扑不变量
- 同伦等价下不变
- 计算工具
```

**例子**：

```
π₁(S¹) = ℤ
π₁(Sⁿ) = {e}（n≥2）
π₁(T²) = ℤ²
```

---

### 3.2 高阶同伦群

**定义**：

```
πₙ(X, x₀)：
- n维球面到X的映射的同伦类
- Abel群（n≥2）

性质：
- 拓扑不变量
- 计算困难
- 应用广泛
```

---

## 四、同调理论

### 4.1 单纯同调

**单纯复形**：

```
K = {单纯形σ}

链复形：
... → Cₖ → C_{k-1} → ...

同调群：
Hₖ(K) = ker(∂ₖ)/im(∂_{k+1})
```

**性质**：

- 拓扑不变量
- 可计算
- 分类工具

---

### 4.2 奇异同调

**定义**：

```
奇异单形：
σ: Δᵏ → X

奇异链：
Cₖ(X) = 自由Abel群

同调：
Hₖ(X) = ker(∂ₖ)/im(∂_{k+1})
```

**优势**：

- 任意拓扑空间
- 函子性
- 计算工具

## 五、流形

### 5.1 定义

**n维流形**：

```
M满足：
1. Hausdorff
2. 第二可数
3. 局部同胚于Rⁿ

例子：
- Sⁿ（球面）
- Tⁿ（环面）
- Pⁿ（射影空间）
```

---

### 5.2 分类

**2维流形**：

```
分类：
- 球面S²
- 环面T²
- 亏格g曲面

Euler示性数：
χ = 2 - 2g
```

**3维流形**：

```
Thurston几何化：
- 8种几何类型
- 包括双曲几何
- Perelman证明
```
