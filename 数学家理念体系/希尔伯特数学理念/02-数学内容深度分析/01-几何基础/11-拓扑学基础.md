## 一、拓扑空间的公理化

### 1.1 Hausdorff的公理化（1914）

**拓扑空间的数学定义**：

**拓扑空间** $(X, \tau)$ 由集合 $X$ 和**拓扑** $\tau \subseteq \mathcal{P}(X)$ 组成，满足：

1. **空集和全集**：$\emptyset, X \in \tau$
2. **任意并封闭**：若 $\{U_i\}_{i \in I} \subseteq \tau$，则 $\bigcup_{i \in I} U_i \in \tau$
3. **有限交封闭**：若 $U_1, \ldots, U_n \in \tau$，则 $\bigcap_{i=1}^n U_i \in \tau$

**开集**：$U \in \tau$ 的元素称为**开集**

**闭集**：$F \subseteq X$ 是**闭集**，若 $X \setminus F \in \tau$（即 $X \setminus F$ 是开集）

**具体例子**：

**例1：离散拓扑**

- **集合**：$X$（任意集合）
- **拓扑**：$\tau = \mathcal{P}(X)$（所有子集都是开集）
- **性质**：最细的拓扑，所有点都是孤立的

**例2：平凡拓扑**

- **集合**：$X$（任意集合）
- **拓扑**：$\tau = \{\emptyset, X\}$（只有空集和全集是开集）
- **性质**：最粗的拓扑，没有非平凡的分离

**例3：标准拓扑（$\mathbb{R}$）**

- **集合**：$X = \mathbb{R}$
- **拓扑**：$\tau = \{U \subseteq \mathbb{R} : U \text{ 是开区间的并}\}$
- **开集**：开区间 $(a, b)$ 及其任意并
- **闭集**：闭区间 $[a, b]$ 及其有限并

**例4：度量拓扑**

- **集合**：$X$（度量空间 $(X, d)$）
- **拓扑**：$\tau = \{U \subseteq X : \forall x \in U, \exists \varepsilon > 0, B(x, \varepsilon) \subseteq U\}$
- **开集**：所有开球 $B(x, \varepsilon) = \{y : d(x, y) < \varepsilon\}$ 的并

**与希尔伯特几何**：

```
几何 → 拓扑：
- 点、线、面 → 点、开集
- 关联关系 → 属于关系
- 公理化方法 → 拓扑公理
```

### 1.2 连续映射

**定义**：

```
f: X → Y连续 ⟺
对任意开集V ⊆ Y：
f⁻¹(V)是X的开集

等价：
对任意闭集F ⊆ Y：
f⁻¹(F)是X的闭集
```

**性质**：

- 复合连续
- 恒等连续
- 同胚：连续双射，逆也连续

## 二、基本拓扑概念

### 2.1 连通性

**连通空间**：

```
X连通 ⟺
X不能分解为两个非空不相交开集的并

等价：
X和∅是X仅有的既开又闭的子集
```

**道路连通**：

```
X道路连通 ⟺
任意两点可用连续路径连接

性质：
道路连通 ⟹ 连通
但逆不成立
```

---

### 2.2 紧致性

**定义**：

```
X紧致 ⟺
任意开覆盖有有限子覆盖

等价（Hausdorff）：
X的任意序列有收敛子列
```

**性质**：

- 紧致集的闭子集紧致
- 连续像紧致
- 有限积紧致（Tychonoff）

## 三、同伦理论

### 3.1 基本群

**基本群的数学定义**：

**路径**：连续映射 $\gamma: [0, 1] \to X$ 使得 $\gamma(0) = x_0$，$\gamma(1) = x_1$

**闭路径（回路）**：$\gamma(0) = \gamma(1) = x_0$（起点和终点相同）

**同伦**：两条路径 $\gamma_0, \gamma_1$ 同伦（$\gamma_0 \simeq \gamma_1$），若存在连续映射 $H: [0, 1] \times [0, 1] \to X$ 使得：
- $H(t, 0) = \gamma_0(t)$
- $H(t, 1) = \gamma_1(t)$
- $H(0, s) = H(1, s) = x_0$（对所有 $s$）

**基本群**：$\pi_1(X, x_0)$ 是所有基于 $x_0$ 的闭路径的同伦类，群运算为路径复合：

$$[\gamma_1] \cdot [\gamma_2] = [\gamma_1 * \gamma_2]$$

其中 $(\gamma_1 * \gamma_2)(t) = \begin{cases} \gamma_1(2t) & \text{若 } t \in [0, 1/2] \\ \gamma_2(2t - 1) & \text{若 } t \in [1/2, 1] \end{cases}$

**具体例子**：

**例1：圆周的基本群**

- **空间**：$X = S^1 = \{z \in \mathbb{C} : |z| = 1\}$
- **基本群**：$\pi_1(S^1) \cong \mathbb{Z}$
- **生成元**：绕圆周一圈的路径 $[e^{2\pi it}]$（$t \in [0, 1]$）
- **解释**：$n \in \mathbb{Z}$ 对应绕圆周 $n$ 圈的路径

**例2：球面的基本群**

- **空间**：$X = S^n$（$n \\\geq 2$）
- **基本群**：$\pi_1(S^n) = \{e\}$（平凡群）
- **原因**：所有闭路径都可收缩到点（单连通）

**例3：环面的基本群**

- **空间**：$X = T^2 = S^1 \times S^1$（环面）
- **基本群**：$\pi_1(T^2) \cong \mathbb{Z} \times \mathbb{Z} = \mathbb{Z}^2$
- **生成元**：两个独立的方向（经线和纬线）

## 四、同调理论

### 4.1 单纯同调

**单纯同调的数学定义**：

**单纯复形** $K$：由单纯形（点、线段、三角形、四面体等）组成的集合，满足：
- 若 $\sigma \in K$，则 $\sigma$ 的所有面也在 $K$ 中
- 若 $\sigma_1, \sigma_2 \in K$，则 $\sigma_1 \cap \sigma_2$ 是 $\sigma_1$ 和 $\sigma_2$ 的公共面

**链复形**：

$$\cdots \xrightarrow{\partial_{k+1}} C_k(K) \xrightarrow{\partial_k} C_{k-1}(K) \xrightarrow{\partial_{k-1}} \cdots$$

其中：
- $C_k(K)$ 是 $k$-单纯形的自由Abel群
- $\partial_k: C_k(K) \to C_{k-1}(K)$ 是边界算子

**同调群**：

$$H_k(K) = \frac{\ker(\partial_k)}{\text{im}(\partial_{k+1})} = \frac{Z_k(K)}{B_k(K)}$$

其中：
- $Z_k(K) = \ker(\partial_k)$（$k$-闭链）
- $B_k(K) = \text{im}(\partial_{k+1})$（$k$-边界）

**具体例子**：

**例1：三角形的同调**

- **复形**：$K = \{\text{点}, \text{边}, \text{三角形}\}$（三角形及其边界）
- **$H_0(K) \cong \mathbb{Z}$**：连通分支数
- **$H_1(K) = 0$**：无"洞"
- **$H_2(K) = 0$**：不是闭流形

**例2：环面的同调**

- **复形**：$K$ 是环面的三角剖分
- **$H_0(K) \cong \mathbb{Z}$**：连通
- **$H_1(K) \cong \mathbb{Z}^2$**：两个"洞"（经线和纬线）
- **$H_2(K) \cong \mathbb{Z}$**：2维闭流形

**例3：球面的同调**

- **复形**：$K$ 是 $S^2$ 的三角剖分
- **$H_0(S^2) \cong \mathbb{Z}$**：连通
- **$H_1(S^2) = 0$**：单连通
- **$H_2(S^2) \cong \mathbb{Z}$**：2维闭流形

**性质**：

- 拓扑不变量
- 可计算
- 分类工具

## 九、数学公式总结

### 核心公式

1. **拓扑公理**：
   - $\emptyset, X \in \tau$
   - $\bigcup_{i \in I} U_i \in \tau$（任意并）
   - $\bigcap_{i=1}^n U_i \in \tau$（有限交）

2. **连续映射**：
   $$f: X \to Y \text{ 连续 } \iff \forall V \text{ 开}, f^{-1}(V) \text{ 开}$$

3. **连通性**：
   $$X \text{ 连通 } \iff X \text{ 不能分解为两个非空不相交开集的并}$$

4. **紧致性**：
   $$X \text{ 紧致 } \iff \text{任意开覆盖有有限子覆盖}$$

5. **基本群**：
   $$\pi_1(X, x_0) = \{\text{闭路径的同伦类}\}$$

6. **路径复合**：
   $$(\gamma_1 * \gamma_2)(t) = \begin{cases} \gamma_1(2t) & t \in [0, 1/2] \\ \gamma_2(2t-1) & t \in [1/2, 1] \end{cases}$$

7. **同调群**：
   $$H_k(X) = \frac{\ker(\partial_k)}{\text{im}(\partial_{k+1})} = \frac{Z_k(X)}{B_k(X)}$$

8. **边界算子**：
   $$\partial_k: C_k(X) \to C_{k-1}(X)$$

9. **Euler示性数**：
   $$\chi(X) = \sum_{k=0}^n (-1)^k \dim H_k(X)$$

10. **流形定义**：
    - Hausdorff
    - 第二可数
    - 局部同胚于 $\mathbb{R}^n$

11. **2维流形分类**：
    $$\chi = 2 - 2g \quad \text{（$g$ 是亏格）}$$

12. **基本群例子**：
    - $\pi_1(S^1) \cong \mathbb{Z}$
    - $\pi_1(S^n) = \{e\}$（$n \\\geq 2$）
    - $\pi_1(T^2) \cong \mathbb{Z}^2$
