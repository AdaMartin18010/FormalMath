## 一、Tarski的动机

### 1.1 希尔伯特系统的复杂性

**希尔伯特20条公理**：

```
I组：关联（8条）
II组：顺序（4条）
III组：合同（5条）
IV组：平行（1条）
V组：连续（2条）

问题：
- 公理多
- 依赖关系复杂
- 形式化困难
```

**Tarski的目标**（1959）：
> 简化公理系统，使其更适合形式化和计算机验证

---

### 1.2 Tarski的10条公理

**简化结果**：

```
Tarski系统：
- 10条公理（vs 20条）
- 更简洁
- 等价于希尔伯特系统
```

**优势**：

- 易于形式化
- 适合计算机验证
- 逻辑清晰

## 二、Tarski公理系统

### 2.1 基本概念

**原始概念**：

```
点：P, Q, R, ...

二元关系：
- 介于：B(P, Q, R)（Q在P和R之间）
- 等距：PQ ≡ RS（线段PQ和RS等长）
```

**定义的概念**：

```
共线：
Col(P, Q, R) ⟺ B(P, Q, R) ∨ B(Q, P, R) ∨ B(P, R, Q)

角：
∠PQR（通过等距定义）
```

---

### 2.2 10条公理

**A1（反射性）**：

```
PQ ≡ QP
```

**A2（传递性）**：

```
PQ ≡ RS ∧ PQ ≡ TU → RS ≡ TU
```

**A3（等距可移动）**：

```
PQ ≡ P'Q' → ∃R: B(P, Q, R) ∧ QR ≡ Q'R'
```

**A4（Pasch公理）**：

```
B(A, P, C) ∧ B(B, Q, C) → ∃X: B(P, X, B) ∧ B(Q, X, A)
```

**A5（五线段公理）**：

```
（SAS全等的形式化）
```

**A6（存在性）**：

```
∃Q: B(P, Q, R)
```

**A7（下界）**：

```
∃P: ∀Q, R: ¬B(P, Q, R)
```

**A8（Euclid公理）**：

```
B(A, D, T) ∧ B(B, D, C) ∧ A≠D →
∃X, Y: B(A, B, X) ∧ B(A, C, Y) ∧ B(X, T, Y)
```

**A9（上界）**：

```
∃P: ∀Q, R: B(Q, P, R) → Q = R
```

## 三、与希尔伯特系统的等价性

### 3.1 等价性证明

**定理**：
> Tarski系统与希尔伯特系统等价

**证明思路**：

```
步骤1：在Tarski系统中定义希尔伯特概念
- 直线：通过两点
- 角度：通过等距
- 全等：通过等距

步骤2：证明希尔伯特公理
- 从Tarski公理推出

步骤3：反之亦然
```

---

### 3.2 简化之处

**Tarski的优势**：

```
1. 公理数：10 vs 20
2. 原始概念：2个关系 vs 3类对象
3. 逻辑结构：更清晰
4. 形式化：更容易
```

**但**：

- 某些公理更复杂（如A8）
- 需要更多定义
- 可读性可能降低
