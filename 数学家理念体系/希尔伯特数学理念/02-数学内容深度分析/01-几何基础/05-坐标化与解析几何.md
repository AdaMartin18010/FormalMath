# 坐标化与解析几何：希尔伯特几何的代数实现

## 一、坐标化的历史

### 1.1 Descartes的解析几何

**Descartes的贡献**（1637）：

```
几何问题 ↔ 代数方程

点 ↔ (x,y)
线 ↔ ax+by+c=0
圆 ↔ (x-a)²+(y-b)²=r²
```

**但问题**：

- 依赖直观
- 逻辑不严格
- 未建立严格对应

## 二、坐标系统的构造

### 2.1 基础选择

**任意性**：

```
原点O：任意点
单位长度：任意线段AB（|AB|=1）
坐标轴：通过O的任意两条垂直直线
```

**关键**：

- 选择是**任意的**
- 但一旦选择，对应唯一
- 体现公理的"无意义"性

---

### 2.2 坐标赋值

**点的坐标**：

```
给定点P：

x坐标：
- 过P作y轴平行线
- 与x轴交于Pₓ
- x(P) = |OPₓ|（带符号）

y坐标：
- 类似定义

结果：
P ↔ (x(P), y(P)) ∈ R²
```

**验证**：

- 双射：每个点对应唯一坐标对
- 满射：每个坐标对对应唯一点
- 通过公理保证

## 三、几何对象的代数表示

### 3.1 直线

**两点式**：

```
给定A=(x₁,y₁), B=(x₂,y₂)

直线AB：
- 方程：y-y₁ = m(x-x₁)
- 其中m = (y₂-y₁)/(x₂-x₁)

或一般式：
ax + by + c = 0
```

**验证公理I-1, I-2**：

```
I-1（存在性）：
给定两点，线性方程组有解
→ 直线存在 ✅

I-2（唯一性）：
两点确定唯一斜率
→ 直线唯一 ✅
```

---

### 3.2 圆

**标准方程**：

```
圆心C=(a,b)，半径r

圆：
(x-a)² + (y-b)² = r²

性质：
- 所有点到圆心距离相等
- 通过合同公理定义
```

---

### 3.3 角度

**通过余弦定理**：

```
给定角∠ABC：

cos(∠ABC) = (AB²+BC²-AC²)/(2·AB·BC)

角度：
∠ABC = arccos(...)

验证：
- 满足合同公理III-4, III-5
- 角度可"移动"
```

---

## 四、公理的代数验证

### 4.1 I组（关联）公理

**I-1, I-2（两点定直线）**：

```
代数对应：
给定(x₁,y₁), (x₂,y₂)

唯一直线：
- 若x₁≠x₂：y = m(x-x₁)+y₁
- 若x₁=x₂：x = x₁

验证：✅ 满足I-1, I-2
```

**I-3（直线上至少三点）**：

```
R²中每条直线有无限多个点
→ 满足I-3 ✅
```

---

### 4.2 II组（顺序）公理

**介于关系**：

```
A-B-C ⟺
- A, B, C共线
- B在A, C之间

代数：
若A=(x₁,y₁), B=(x₂,y₂), C=(x₃,y₃)共线
则：x₁ < x₂ < x₃ 或 x₃ < x₂ < x₁
```

**Pasch公理**：

```
通过R²的凸性验证
→ 满足II-4 ✅
```

---

### 4.3 III组（合同）公理

**线段长度**：

```
|AB| = √((x₂-x₁)² + (y₂-y₁)²)

验证：
- 满足III-1（可移动）
- 满足III-2（传递性）
- 满足III-3（加法性）
```

**SAS全等**：

```
通过余弦定理验证
→ 满足III-5 ✅
```

---

### 4.4 IV组（平行）公理

**平行线**：

```
两直线平行 ⟺ 斜率相等

给定线l: y = mx + b
和点P=(x₀,y₀)不在l上

唯一平行线：
y = m(x-x₀) + y₀

验证：满足IV ✅
```

---

### 4.5 V组（连续）公理

**V-1（阿基米德）**：

```
R的阿基米德性：
∀x>0, ∃n∈ℕ: n·x > 1

对应几何：
∀线段AB, CD (AB>0)
∃n: n·AB > CD

验证：✅
```

**V-2（完备性）**：

```
R的Dedekind完备性：
任意切割有分割点

对应几何：
直线与R同构
→ 完备性 ✅
```


---

### 5.2 几何构造的代数化

**例子：中垂线**：

```
几何构造：
给定A, B，构造AB的中垂线

代数对应：
- A=(x₁,y₁), B=(x₂,y₂)
- 中点M=((x₁+x₂)/2, (y₁+y₂)/2)
- 斜率m = -(x₂-x₁)/(y₂-y₁)
- 中垂线：y-y_M = m(x-x_M)

验证：
- 通过M ✅
- 垂直AB ✅
- 唯一 ✅
```

**一般方法**：

```
任意几何构造
    ↓
转化为代数方程
    ↓
R的完备性保证解存在
    ↓
翻译回几何对象
```

## 六、非标准模型

### 6.1 有理数模型

**Q²模型**：

```
使用Q²代替R²
    ↓
满足I-IV公理
    ↓
但不满足V-2（不完备）

例子：
√2不在Q中
→ 某些几何构造不存在
→ V-2独立
```

---

### 6.2 非阿基米德模型

**超实数R*²模型**：

```
使用非标准分析的超实数R*
    ↓
在R*²上定义几何
    ↓
满足I-IV公理
    ↓
但不满足V-1（非阿基米德）

例子：
存在无穷小ε: ∀n, n·ε < 1
→ V-1独立
```
