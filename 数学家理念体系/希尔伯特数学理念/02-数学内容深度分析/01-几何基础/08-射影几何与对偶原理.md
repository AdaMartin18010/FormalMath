## 一、射影几何的引入

### 1.1 从仿射到射影

**仿射几何的局限**：

```
问题：
- 平行线不相交
- 需要特殊处理
- 某些定理不统一
```

**射影几何的解决**：

```
添加"无穷远点"：
- 平行线在无穷远相交
- 统一处理
- 对偶原理
```

---

### 1.2 射影空间

**定义**：

```
n维射影空间Pⁿ：
- 通过等价关系定义
- (x₀,...,xₙ) ~ (λx₀,...,λxₙ)（λ≠0）
- 齐次坐标

性质：
- 紧致
- 无平行线
- 对偶性
```

---

## 二、对偶原理

### 2.1 点线对偶

**对偶对应**：

```
点 ↔ 线
线 ↔ 点
共线 ↔ 共点
相交 ↔ 连接

定理 ↔ 对偶定理
```

**例子**：

```
Desargues定理：
两三角形对应顶点连线共点
    ⟺
对应边交点共线

对偶：
两三角形对应边交点共线
    ⟺
对应顶点连线共点
```

---

### 2.2 对偶原理

**原理**：

```
在射影平面中：
- 任何定理
- 交换"点"和"线"
- 得到对偶定理
- 也成立
```

**意义**：

- 证明减半
- 结构对称
- 美学价值

## 三、射影变换

### 3.1 齐次坐标

**齐次坐标的数学定义**：

**射影平面 $\mathbb{P}^2$** 上的点 $P$ 的**齐次坐标**为：

$$[x_0 : x_1 : x_2]$$

其中：
- $(x_0, x_1, x_2) \in \mathbb{R}^3 \setminus \{(0, 0, 0)\}$
- $[x_0 : x_1 : x_2] = [\lambda x_0 : \lambda x_1 : \lambda x_2]$（对所有 $\lambda \neqqqqq 0$）

**等价关系**：
$$(x_0, x_1, x_2) \sim (y_0, y_1, y_2) \iff \exists \lambda \neqqqqq 0: (y_0, y_1, y_2) = \lambda(x_0, x_1, x_2)$$

**具体例子**：

**例1：普通点的齐次坐标**

- **仿射点**：$P = (2, 3)$
- **齐次坐标**：$[2 : 3 : 1]$（或 $[4 : 6 : 2]$，$[1 : 1.5 : 0.5]$ 等）
- **恢复**：$(x, y) = \\left(\frac{x_1}{x_0}, \frac{x_2}{x_0}\right)$（若 $x_0 \neqqqqq 0$）

**例2：无穷远点**

- **齐次坐标**：$[0 : 1 : 0]$（$x$ 轴方向的无穷远点）
- **齐次坐标**：$[0 : 0 : 1]$（$y$ 轴方向的无穷远点）
- **齐次坐标**：$[1 : 2 : 0]$（斜率为 $2$ 的直线的无穷远点）

**例3：直线的齐次坐标**

- **直线**：$ax + by + c = 0$
- **齐次形式**：$ax_1 + bx_2 + cx_0 = 0$（在齐次坐标 $[x_0 : x_1 : x_2]$ 下）
- **对偶**：直线的齐次坐标为 $[a : b : c]$（对偶空间）

**性质**：

- 不唯一
- 但比例唯一
- 便于计算

---

### 3.2 射影变换

**定义**：

```
射影变换：
[x₀' : x₁' : x₂'] = [∑aᵢⱼxⱼ : ∑bᵢⱼxⱼ : ∑cᵢⱼxⱼ]

矩阵形式：
P' = MP

其中：
M是3×3可逆矩阵
```

**性质**：

- 保持共线性
- 保持交比
- 群结构

## 四、交比

### 4.1 定义

**交比的数学定义**：

设 $A, B, C, D$ 是射影直线上四个不同的点，**交比**定义为：

$$(A, B; C, D) = \frac{AC \cdot BD}{AD \cdot BC}$$

其中 $AC, BD, AD, BC$ 是带符号的距离（在齐次坐标下计算）。

**齐次坐标形式**：

若 $A = [a_0 : a_1]$，$B = [b_0 : b_1]$，$C = [c_0 : c_1]$，$D = [d_0 : d_1]$，则：

$$(A, B; C, D) = \frac{\det\begin{pmatrix} a_0 & c_0 \\ a_1 & c_1 \end{pmatrix} \cdot \det\begin{pmatrix} b_0 & d_0 \\ b_1 & d_1 \end{pmatrix}}{\det\begin{pmatrix} a_0 & d_0 \\ a_1 & d_1 \end{pmatrix} \cdot \det\begin{pmatrix} b_0 & c_0 \\ b_1 & c_1 \end{pmatrix}}$$

**具体例子**：

**例1：标准交比**

- **点**：$A = [0 : 1]$，$B = [1 : 1]$，$C = [2 : 1]$，$D = [3 : 1]$
- **计算**：
  - $AC = 2$，$BD = 2$，$AD = 3$，$BC = 1$
  - $(A, B; C, D) = \frac{2 \cdot 2}{3 \cdot 1} = \frac{4}{3}$

**例2：调和点列**

- **定义**：$(A, B; C, D) = -1$ 时，四点构成**调和点列**
- **性质**：$C$ 和 $D$ 关于 $A$ 和 $B$ 调和共轭
- **应用**：在射影几何中，调和点列是基本概念

**例3：射影不变量**

- **性质**：交比在射影变换下不变
- **验证**：若 $f$ 是射影变换，则 $(f(A), f(B); f(C), f(D)) = (A, B; C, D)$
- **意义**：交比是射影几何的基本不变量

---

### 4.2 应用

**调和点列**：

```
(A,B;C,D) = -1

性质：
- 特殊交比
- 对称性
- 应用广泛
```

**射影几何基础**：

- 用交比定义
- 所有概念
- 统一框架

## 五、与希尔伯特几何的关系

### 5.1 仿射几何嵌入

**关系**：

```
射影平面P²
    ↓
去掉一条线
    ↓
仿射平面A²
    ↓
添加度量
    ↓
欧氏平面E²
```

**希尔伯特几何**：

- 在仿射/欧氏层面
- 但可推广到射影
- 保持公理化方法

---

### 5.2 公理化射影几何

**公理系统**：

```
类似希尔伯特：
- 关联公理
- 顺序公理（修改）
- 连续性公理

但：
- 无平行公理
- 无度量
- 纯射影
```

## 七、总结

### 射影几何的历史地位

**发展**：

- 19世纪：Desargues, Poncelet
- 20世纪：公理化（类似Hilbert）
- 现代：代数几何基础

**与希尔伯特**：

- 方法论继承
- 公理化方法
- 但几何性质不同

**最终评价**：
射影几何是**希尔伯特几何的自然扩展**，体现了公理化方法的普遍性。

## 九、数学公式总结

### 核心公式

1. **齐次坐标**：
   $$[x_0 : x_1 : x_2] = [\lambda x_0 : \lambda x_1 : \lambda x_2] \quad (\lambda \neqqqqq 0)$$

2. **仿射坐标恢复**：
   $$(x, y) = \\left(\frac{x_1}{x_0}, \frac{x_2}{x_0}\right) \quad \text{（若 } x_0 \neqqqqq 0 \text{）}$$

3. **射影变换（矩阵形式）**：
   $$\begin{pmatrix} x_0' \\ x_1' \\ x_2' \end{pmatrix} = M \begin{pmatrix} x_0 \\ x_1 \\ x_2 \end{pmatrix}$$
   其中 $M$ 是 $3 \times 3$ 可逆矩阵

4. **交比**：
   $$(A, B; C, D) = \frac{AC \cdot BD}{AD \cdot BC}$$

5. **交比（行列式形式）**：
   $$(A, B; C, D) = \frac{\det(A, C) \cdot \det(B, D)}{\det(A, D) \cdot \det(B, C)}$$

6. **调和点列**：
   $$(A, B; C, D) = -1$$

7. **对偶原理**：
   $$\text{点} \\leftrightarrow \text{线}, \quad \text{共线} \\leftrightarrow \text{共点}$$

8. **射影平面参数**：
   - 点：$q^2 + q + 1$ 个
   - 线：$q^2 + q + 1$ 条
   - 每线：$q + 1$ 点
   - 每点：$q + 1$ 线

9. **射影变换群**：
   $$\text{PGL}(3, \mathbb{F}) = \text{GL}(3, \mathbb{F}) / \{\lambda I : \lambda \neqqqqq 0\}$$

10. **Desargues定理**：
    $$\text{两三角形对应顶点连线共点} \iff \text{对应边交点共线}$$
