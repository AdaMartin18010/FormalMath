# 射影几何与对偶原理：希尔伯特几何的扩展

## 一、射影几何的引入

### 1.1 从仿射到射影

**仿射几何的局限**：

```
问题：
- 平行线不相交
- 需要特殊处理
- 某些定理不统一
```

**射影几何的解决**：

```
添加"无穷远点"：
- 平行线在无穷远相交
- 统一处理
- 对偶原理
```

---

### 1.2 射影空间

**定义**：

```
n维射影空间Pⁿ：
- 通过等价关系定义
- (x₀,...,xₙ) ~ (λx₀,...,λxₙ)（λ≠0）
- 齐次坐标

性质：
- 紧致
- 无平行线
- 对偶性
```

---

## 二、对偶原理

### 2.1 点线对偶

**对偶对应**：

```
点 ↔ 线
线 ↔ 点
共线 ↔ 共点
相交 ↔ 连接

定理 ↔ 对偶定理
```

**例子**：

```
Desargues定理：
两三角形对应顶点连线共点
    ⟺
对应边交点共线

对偶：
两三角形对应边交点共线
    ⟺
对应顶点连线共点
```

---

### 2.2 对偶原理

**原理**：

```
在射影平面中：
- 任何定理
- 交换"点"和"线"
- 得到对偶定理
- 也成立
```

**意义**：

- 证明减半
- 结构对称
- 美学价值

---

## 三、射影变换

### 3.1 齐次坐标

**定义**：

```
点P的齐次坐标：
[x₀ : x₁ : x₂]

其中：
- (x₀, x₁, x₂) ≠ (0,0,0)
- [x₀ : x₁ : x₂] = [λx₀ : λx₁ : λx₂]（λ≠0）
```

**性质**：

- 不唯一
- 但比例唯一
- 便于计算

---

### 3.2 射影变换

**定义**：

```
射影变换：
[x₀' : x₁' : x₂'] = [∑aᵢⱼxⱼ : ∑bᵢⱼxⱼ : ∑cᵢⱼxⱼ]

矩阵形式：
P' = MP

其中：
M是3×3可逆矩阵
```

**性质**：

- 保持共线性
- 保持交比
- 群结构

---

## 四、交比

### 4.1 定义

**交比**：

```
四点A, B, C, D在直线上：

(A,B;C,D) = (AC/BC)/(AD/BD)

性质：
- 射影不变量
- 唯一确定四点位置
```

---

### 4.2 应用

**调和点列**：

```
(A,B;C,D) = -1

性质：
- 特殊交比
- 对称性
- 应用广泛
```

**射影几何基础**：

- 用交比定义
- 所有概念
- 统一框架

---

## 五、与希尔伯特几何的关系

### 5.1 仿射几何嵌入

**关系**：

```
射影平面P²
    ↓
去掉一条线
    ↓
仿射平面A²
    ↓
添加度量
    ↓
欧氏平面E²
```

**希尔伯特几何**：

- 在仿射/欧氏层面
- 但可推广到射影
- 保持公理化方法

---

### 5.2 公理化射影几何

**公理系统**：

```
类似希尔伯特：
- 关联公理
- 顺序公理（修改）
- 连续性公理

但：
- 无平行公理
- 无度量
- 纯射影
```

## 六、现代应用

**投影变换**：

```
3D → 2D投影：
- 透视投影
- 正交投影

应用：

```

---

### 6.2 代数几何

**射影簇**：

```
射影空间中的代数簇：
- 齐次多项式
- 射影闭集
- 紧致性

优势：
- 更好的相交理论
- 紧化
- 现代代数几何基础
```

---

## 七、总结

### 射影几何的历史地位

**发展**：

- 19世纪：Desargues, Poncelet
- 20世纪：公理化（类似Hilbert）
- 现代：代数几何基础

**与希尔伯特**：

- 方法论继承
- 公理化方法
- 但几何性质不同

**最终评价**：
射影几何是**希尔伯特几何的自然扩展**，体现了公理化方法的普遍性。
