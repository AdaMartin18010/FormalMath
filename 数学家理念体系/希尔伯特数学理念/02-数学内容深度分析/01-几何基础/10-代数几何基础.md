# 代数几何基础：从希尔伯特零点定理到现代

## 一、代数簇

### 1.1 仿射簇

**定义**：

```
仿射空间Aⁿ：
- 点：(a₁,...,aₙ) ∈ kⁿ

代数簇V：
- V = {P ∈ kⁿ | f(P) = 0对所有f ∈ I}
- I：理想

记号：
V = V(I)
```

**Hilbert零点定理**：

```
I(V(I)) = √I

意义：
- 理想 ↔ 簇
- 一一对应（根理想）
- 代数几何基础
```

---

### 1.2 射影簇

**射影空间Pⁿ**：

```
点：[x₀ : ... : xₙ]
- 齐次坐标
- 比例等价

射影簇：
- 齐次理想定义
- 紧致性
- 更好的性质
```

## 三、Riemann-Roch定理

### 3.1 曲线情况

**定理**（Riemann, 1857; Roch, 1865）：

```
曲线C，线丛L：

h⁰(C,L) - h¹(C,L) = deg(L) - g + 1

其中：
- g：亏格
- deg(L)：度
- h⁰, h¹：上同调维数
```

**应用**：

- 分类曲线
- 计算不变量
- 现代代数几何

---

### 3.2 高维推广

**Hirzebruch-Riemann-Roch**：

```
概形X，向量丛E：

χ(X,E) = ∫_X ch(E) · td(X)

其中：
- ch：Chern特征
- td：Todd类
- χ：Euler示性数
```
