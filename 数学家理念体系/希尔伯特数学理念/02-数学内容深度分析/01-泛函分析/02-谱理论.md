# 谱理论：从矩阵特征值到无限维算子

## 摘要

谱理论是泛函分析的核心分支，源于Hilbert对积分方程的研究。本文系统阐述有界线性算子的谱理论、紧算子的Riesz-Schauder理论、自伴算子的谱定理，并深入探讨其在量子力学中的决定性作用。从有限维矩阵特征值问题出发，逐步扩展到无限维希尔伯特空间，揭示谱理论的深刻结构与应用价值。

## 一、历史渊源：从特征值到谱

### 1.1 矩阵特征值问题

线性代数的中心问题之一是矩阵的特征值问题：给定 $n \times n$ 矩阵 $A$，寻找标量 $\lambda$ 和非零向量 $v$ 使得

$$Av = \lambda v$$

这等价于 $(A - \lambda I)v = 0$ 有非零解，即

$$\det(A - \lambda I) = 0$$

### 1.2 Hilbert的关键推广

Hilbert在研究积分方程时发现，虽然积分算子没有行列式，但"特征值"概念仍然有意义。对于积分算子

$$(Kf)(x) = \int_a^b K(x,y)f(y)\,dy$$

特征值方程 $Kf = \lambda f$ 与矩阵特征值问题具有深刻的相似性。

**Hilbert的洞察**（1906）：
> "谱（Spectrum）是特征值概念的适当推广，它包含所有使 $(A - \lambda I)$ 不可逆的 $\lambda$。"

## 二、有界算子的谱

### 2.1 谱的定义与分类

设 $H$ 是复希尔伯特空间，$\mathcal{B}(H)$ 表示 $H$ 上的有界线性算子全体。

**定义 2.1**（预解集与谱）：设 $A \in \mathcal{B}(H)$。

**(i)** **预解集**（Resolvent Set）：
$$\rho(A) = \{ \lambda \in \mathbb{C} : (A - \lambda I)^{-1} \in \mathcal{B}(H) \}$$

**(ii)** **谱**（Spectrum）：
$$\sigma(A) = \mathbb{C} \setminus \rho(A)$$

**(iii)** **预解式**（Resolvent）：
对 $\lambda \in \rho(A)$，定义 $R_\lambda(A) = (A - \lambda I)^{-1}$。

### 2.2 谱的分类

谱可进一步分解为三个互不相交的子集：

**定义 2.2**（谱的分解）：

**(i)** **点谱**（离散谱）：
$$\sigma_p(A) = \{ \lambda \in \mathbb{C} : \ker(A - \lambda I) \neq \{0\} \}$$
即特征值集合。

**(ii)** **连续谱**：
$$\sigma_c(A) = \{ \lambda \in \sigma(A) : \ker(A - \lambda I) = \{0\}, \overline{\text{ran}(A - \lambda I)} = H, (A - \lambda I)^{-1} \text{ 无界} \}$$

**(iii)** **剩余谱**：
$$\sigma_r(A) = \{ \lambda \in \sigma(A) : \ker(A - \lambda I) = \{0\}, \overline{\text{ran}(A - \lambda I)} \neq H \}$$

**定理 2.3**：
$$\sigma(A) = \sigma_p(A) \cup \sigma_c(A) \cup \sigma_r(A) \quad \text{（不交并）}$$

### 2.3 谱的基本性质

**定理 2.4**（谱的紧性）：对 $A \in \mathcal{B}(H)$，$\sigma(A)$ 是 $\mathbb{C}$ 的非空紧子集，且
$$\sigma(A) \subseteq \{ \lambda \in \mathbb{C} : |\lambda| \leq \|A\| \}$$

**定理 2.5**（谱映射定理）：设 $p$ 是多项式，则
$$\sigma(p(A)) = p(\sigma(A)) = \{ p(\lambda) : \lambda \in \sigma(A) \}$$

**证明**（概要）：
若 $\lambda \in \sigma(A)$，则 $A - \lambda I$ 不可逆。因式分解
$$p(z) - p(\lambda) = (z - \lambda)q(z)$$

故 $p(A) - p(\lambda)I = (A - \lambda I)q(A)$ 不可逆。反之类似。

### 2.4 具体例子

**例 2.6**（移位算子）：
设 $H = \ell^2(\mathbb{N})$，定义右移位算子
$$S(x_1, x_2, x_3, \ldots) = (0, x_1, x_2, \ldots)$$

**分析**：
- $\|S\| = 1$，故 $\sigma(S) \subseteq \overline{D(0,1)}$（闭单位圆盘）
- $S$ 无特征值（$Sx = \lambda x$ 推出 $x = 0$），故 $\sigma_p(S) = \emptyset$
- $S^*$（左移位）的特征值为开单位圆盘 $D(0,1)$
- 实际上 $\sigma(S) = \overline{D(0,1)}$（满圆盘），$\sigma_c(S) = \{ \lambda : |\lambda| = 1 \}$

**例 2.7**（乘法算子）：
设 $H = L^2[0,1]$，$\varphi \in C[0,1]$，定义
$$(M_\varphi f)(x) = \varphi(x)f(x)$$

则 $\sigma(M_\varphi) = \text{ran}(\varphi) = \varphi([0,1])$（值域）。

## 三、紧算子的谱理论

### 3.1 紧算子的定义

**定义 3.1**：算子 $K \in \mathcal{B}(H)$ 称为**紧算子**（或全连续算子），若对任意有界集 $B \subseteq H$，$\overline{K(B)}$ 是紧集。

等价地：$K$ 将有界集映为相对紧集。

**等价刻画**：$K$ 是紧算子当且仅当对任意有界序列 $\{x_n\}$，$\{Kx_n\}$ 有收敛子列。

### 3.2 紧算子的基本性质

**定理 3.2**：紧算子全体 $\mathcal{K}(H)$ 是 $\mathcal{B}(H)$ 的闭双边理想。

**定理 3.3**：有限秩算子是紧算子的稠密子集。

**例 3.4**（积分算子）：
设 $K \in L^2([a,b] \times [a,b])$，则积分算子
$$(Kf)(x) = \int_a^b K(x,y)f(y)\,dy$$

是 $L^2[a,b]$ 上的紧算子（Hilbert-Schmidt定理）。

### 3.3 Riesz-Schauder理论

**定理 3.5**（Riesz-Schauder谱定理）：设 $K \in \mathcal{K}(H)$，则

**(i)** $\sigma(K)$ 是至多可数集，$0$ 是唯一可能的聚点；

**(ii)** 每个非零谱点都是特征值；

**(iii)** 对每个非零特征值 $\lambda$，$\dim \ker(K - \lambda I) < \infty$（特征空间有限维）；

**(iv)** 特征值只能以 $0$ 为聚点。

**定理 3.6**（Fredholm择一）：设 $K \in \mathcal{K}(H)$，$\lambda \neq 0$，则方程
$$(K - \lambda I)x = y$$

要么
- 对任意 $y \in H$ 有唯一解；要么
- 齐次方程 $(K - \lambda I)x = 0$ 有非零解（此时 $y$ 必须满足正交条件）。

### 3.4 Hilbert的原始贡献

Hilbert在积分方程研究中证明了紧自伴算子的谱定理（见下文）。他的方法依赖于将积分算子对角化，类似于对称矩阵的正交对角化。

**与Fredholm的联系**：
Erik Ivar Fredholm（1866-1927）在1900年建立了积分方程的基本理论。Hilbert扩展了这一工作，将其提升到抽象的无限维空间。

## 四、自伴算子的谱定理

### 4.1 自伴算子

**定义 4.1**：$A \in \mathcal{B}(H)$ 称为**自伴**（或Hermite），若 $A^* = A$，即
$$\langle Ax, y \rangle = \langle x, Ay \rangle, \quad \forall x,y \in H$$

**命题 4.2**：自伴算子的谱是实的：$\sigma(A) \subseteq \mathbb{R}$。

**证明**：设 $\lambda = \alpha + i\beta \in \sigma(A)$，$\beta \neq 0$。对任意 $x \in H$，
$$\|(A - \lambda I)x\|^2 = \|(A - \alpha I)x\|^2 + \beta^2\|x\|^2 \geq \beta^2\|x\|^2$$

故 $(A - \lambda I)^{-1}$ 存在且有界，矛盾。

### 4.2 紧自伴算子的谱定理

**定理 4.3**（紧自伴算子的谱定理）：设 $A \in \mathcal{K}(H)$ 自伴，则存在 $H$ 的标准正交基 $\{e_n\}$ 和实数列 $\{\lambda_n\}$ 满足：

**(i)** $\lambda_n \to 0$（当 $n \to \infty$）；

**(ii)** $Ae_n = \lambda_n e_n$（特征向量）；

**(iii）** 对任意 $x \in H$：
$$Ax = \sum_{n=1}^\infty \lambda_n \langle x, e_n \rangle e_n$$

**(iv）**（算子范数公式）：
$$\|A\| = \max_{n} |\lambda_n| = \sup_{\|x\|=1} |\langle Ax, x \rangle|$$

**证明概要**：

1. 令 $m = \inf_{\|x\|=1} \langle Ax, x \rangle$，$M = \sup_{\|x\|=1} \langle Ax, x \rangle$。

2. 由紧性，存在单位向量 $e_1$ 使得 $|\langle Ae_1, e_1 \rangle| = \|A\|$。

3. 令 $\lambda_1 = \langle Ae_1, e_1 \rangle$，验证 $Ae_1 = \lambda_1 e_1$。

4. 对 $H_1 = \{e_1\}^\perp$ 重复此过程，由紧性过程终止于可数步。

### 4.3 有界自伴算子的谱定理

对一般有界自伴算子，需要更一般的谱定理。

**定理 4.4**（谱定理——函数演算形式）：
设 $A \in \mathcal{B}(H)$ 自伴，则存在唯一的映射
$$\Phi: C(\sigma(A)) \to \mathcal{B}(H)$$

满足：
- $\Phi$ 是等距*-同态
- $\Phi(1) = I$，$\Phi(\text{id}) = A$
- 对多项式 $p$，$\Phi(p) = p(A)$

**定理 4.5**（谱定理——谱测度形式）：
设 $A \in \mathcal{B}(H)$ 自伴，则存在唯一的**谱测度** $E$（取值于 $H$ 的投影算子），使得
$$A = \int_{\sigma(A)} \lambda\,dE(\lambda)$$

对任意连续函数 $f$，
$$f(A) = \int_{\sigma(A)} f(\lambda)\,dE(\lambda)$$

### 4.4 谱测度的物理意义

在量子力学中，谱测度 $E$ 对应于测量的概率分布：
- 系统处于态 $\psi$（$\|\psi\| = 1$）
- 测量可观测量 $A$ 得到值在集合 $\Delta \subseteq \mathbb{R}$ 中的概率为
$$P(A \in \Delta) = \langle E(\Delta)\psi, \psi \rangle = \|E(\Delta)\psi\|^2$$

## 五、谱理论在量子力学中的应用

### 5.1 量子力学的数学框架

von Neumann于1932年将量子力学建立在严格的数学基础之上：

| 物理概念 | 数学实现 | 谱理论角色 |
|----------|----------|------------|
| 量子态 | $H$ 中的单位向量 | 希尔伯特空间元素 |
| 可观测量 | 自伴算子 $A$ | 谱 $\sigma(A)$ 给出测量可能值 |
| 测量结果 | $\lambda \in \sigma(A)$ | 特征值或连续谱点 |
| 概率分布 | 谱测度 $E$ | $P_\psi(\Delta) = \langle E(\Delta)\psi, \psi \rangle$ |

### 5.2 典型例子

**例 5.1**（谐振子）：
哈密顿量
$$H = \frac{1}{2m}P^2 + \frac{1}{2}m\omega^2 X^2$$

其中 $X$（位置）和 $P = -i\hbar \frac{d}{dx}$（动量）满足 $[X,P] = i\hbar I$。

谱：$\sigma(H) = \{ (n + \frac{1}{2})\hbar\omega : n = 0, 1, 2, \ldots \}$

**例 5.2**（氢原子）：
$$H = -\frac{\hbar^2}{2m}\Delta - \frac{e^2}{r}$$

谱 $\sigma(H) = \{ -\frac{R_H}{n^2} : n = 1, 2, \ldots \} \cup [0, \infty)$
- 离散谱对应束缚态（电子轨道）
- 连续谱对应散射态

### 5.3 与Hilbert学派的联系

von Neumann于1926年来到哥廷根，在Hilbert的指导下工作。他迅速掌握了Hilbert的谱理论，并将其应用于量子力学。这一工作是Hilbert公理化方法在物理学中的最成功应用。

## 六、无界算子简介

### 6.1 必要性

量子力学中的许多重要算子是无界的：
- 位置算子：$(X\psi)(x) = x\psi(x)$
- 动量算子：$(P\psi)(x) = -i\hbar \psi'(x)$
- 哈密顿量：$H = -\frac{\hbar^2}{2m}\Delta + V(x)$

### 6.2 闭算子与自伴性

**定义 6.1**：算子 $A: D(A) \subseteq H \to H$ 称为**闭算子**，若其图像
$$\text{Gr}(A) = \{ (x, Ax) : x \in D(A) \}$$
是 $H \times H$ 的闭子空间。

**定义 6.2**：稠定算子 $A$ 的**伴随** $A^*$ 定义为：
$$D(A^*) = \{ y \in H : \exists z \in H, \forall x \in D(A), \langle Ax, y \rangle = \langle x, z \rangle \}$$

$A^*y = z$。

**定义 6.3**：稠定算子 $A$ 称为**自伴**，若 $A = A^*$（包括定义域相等）。

### 6.3 无界自伴算子的谱定理

**定理 6.4**（无界自伴算子的谱定理）：
设 $A$ 是希尔伯特空间 $H$ 上的自伴算子（可能无界），则存在唯一的谱测度 $E$ 使得
$$A = \int_{\sigma(A)} \lambda\,dE(\lambda)$$

$$D(A) = \left\{ x \in H : \int_{\sigma(A)} \lambda^2\,d\langle E(\lambda)x, x \rangle < \infty \right\}$$

## 七、历史影响与数学家的贡献

### 7.1 Hilbert的原始工作

Hilbert在1906-1910年间建立了紧算子的谱理论。他的方法依赖于：
1. 变分原理（极值问题）
2. 正交展开
3. 无限维类比于二次型理论

### 7.2 后继发展

| 数学家 | 年份 | 贡献 |
|--------|------|------|
| F. Riesz | 1913 | 抽象算子理论，预解式概念 |
| von Neumann | 1929-32 | 无界算子，量子力学应用 |
| Stone | 1932 | 单参数酉群，$e^{itA}$ 的严格理论 |
| Gelfand | 1941 | Banach代数，Gelfand表示 |
| Dunford | 1943 | 谱算子理论 |

### 7.3 与Gödel不完备性定理的关系

Hilbert计划旨在用有穷方法证明数学的一致性。Gödel（1931）证明这不可能。然而，谱理论作为Hilbert计划的"副产品"，其数学价值超越了原始目标。

**von Neumann评论**（1951）：
> "Hilbert关于无限维空间的工作是数学史上最深刻的成就之一。即使他的元数学目标未能完全实现，他创造的工具改变了整个数学的面貌。"

### 7.4 现代应用

- **量子场论**：谱理论是分析场算子的基础
- **随机过程**：Markov过程的生成元是谱理论的应用
- **偏微分方程**：椭圆算子的谱分析
- **数值分析**：谱方法的基础
