## 一、简介

$$e^{i\pi} + 1 = 0$$

这个被理查德·费曼称为"数学中最卓越的公式"、被欧拉本人称为"上帝创造的公式"的恒等式，将数学中五个最重要的常数——$e$（自然对数的底）、$i$（虚数单位）、$\pi$（圆周率）、$1$（乘法单位元）、$0$（加法单位元）——以一种令人震惊的简洁方式联系在一起。它不仅是数学美的巅峰之作，更是复分析（complex analysis）这一宏伟数学分支的基石。

莱昂哈德·欧拉（Leonhard Euler）在1748年出版的《无穷分析引论》（*Introductio in analysin infinitorum*）中首次系统阐述了这一公式。尽管以现代严格性标准衡量，欧拉的推导带有18世纪特有的形式操作色彩，但他的核心洞见——指数函数、三角函数与复数之间存在深刻的内在联系——被后世证明是完全正确的。

本文将从现代复分析的严格观点出发，给出欧拉公式 $e^{iz} = \cos z + i\sin z$ 的多种严格证明，探讨复指数函数的解析性质，并阐述欧拉公式在数学、物理学和工程学中的深远影响。

## 二、欧拉公式的历史发现与数学表述

### 2.1 从指数函数到复数域的延拓

欧拉之前的数学家已经注意到一些令人困惑的公式。1714年，Roger Cotes给出了一个等价的结果：
$$ix = \ln(\cos x + i\sin x)$$

这实际上就是欧拉公式的对数形式。然而，Cotes并没有认识到这一结果的全部意义，也没有将其发展为系统的复函数理论。

欧拉的突破性贡献在于：他不是将 $e^{ix}$ 视为某种奇怪的符号组合，而是将其定义为指数函数在复数域上的**自然延拓**。在欧拉看来，幂级数
$$e^z = 1 + z + \frac{z^2}{2!} + \frac{z^3}{3!} + \cdots$$
不仅对于实数 $z$ 有效，对于复数 $z = x + iy$ 同样有效。将 $z = ix$ 代入，通过形式计算即可得到：
$$e^{ix} = \left(1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots\right) + i\left(x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots\right) = \cos x + i\sin x$$

### 2.2 欧拉公式的严格表述

**定理 2.1（欧拉公式）**：对任意复数 $z \in \mathbb{C}$，有：
$$e^z = e^{x+iy} = e^x(\cos y + i\sin y)$$

特别地，当 $z = iy$（纯虚数）时：
$$e^{iy} = \cos y + i\sin y$$

**定义 2.2（复指数函数）**：复指数函数 $e^z$ 可通过以下等价方式定义：
1. **幂级数定义**：$e^z = \sum_{n=0}^{\infty} \frac{z^n}{n!}$
2. **微分方程定义**：$f(z) = e^z$ 是唯一满足 $f'(z) = f(z)$ 且 $f(0) = 1$ 的整函数
3. **极限定义**：$e^z = \lim_{n \to \infty} \left(1 + \frac{z}{n}\right)^n$

### 2.3 特殊情形与推论

**推论 2.3（欧拉恒等式）**：取 $z = i\pi$，得：
$$e^{i\pi} = \cos\pi + i\sin\pi = -1$$
即：
$$e^{i\pi} + 1 = 0$$

**推论 2.4（三角函数的指数表示）**：
$$\cos z = \frac{e^{iz} + e^{-iz}}{2}, \quad \sin z = \frac{e^{iz} - e^{-iz}}{2i}$$

**推论 2.5（棣莫弗公式）**：对任意整数 $n$ 和实数 $\theta$：
$$(\cos\theta + i\sin\theta)^n = \cos(n\theta) + i\sin(n\theta)$$

这实际上是欧拉公式的直接推论：$(e^{i\theta})^n = e^{in\theta}$。

## 三、欧拉公式的严格证明

### 3.1 方法Ⅰ：幂级数展开法

**定理 3.1**：幂级数 $f(z) = \sum_{n=0}^{\infty} \frac{z^n}{n!}$ 在整个复平面上绝对收敛，且满足 $f(z) = e^x(\cos y + i\sin y)$。

**证明**：

**步骤1：收敛半径**
由比值判别法：
$$\lim_{n \to \infty} \left| \frac{z^{n+1}/(n+1)!}{z^n/n!} \right| = \lim_{n \to \infty} \frac{|z|}{n+1} = 0$$
收敛半径为 $+\infty$，故级数在所有 $z \in \mathbb{C}$ 上收敛，定义了一个**整函数**（entire function）。

**步骤2：代入 $z = iy$**
$$f(iy) = \sum_{n=0}^{\infty} \frac{(iy)^n}{n!} = \sum_{k=0}^{\infty} \frac{(iy)^{2k}}{(2k)!} + \sum_{k=0}^{\infty} \frac{(iy)^{2k+1}}{(2k+1)!}$$

由于 $i^{2k} = (-1)^k$，$i^{2k+1} = i(-1)^k$：
$$f(iy) = \sum_{k=0}^{\infty} \frac{(-1)^k y^{2k}}{(2k)!} + i\sum_{k=0}^{\infty} \frac{(-1)^k y^{2k+1}}{(2k+1)!}$$

**步骤3：识别三角函数的幂级数**
$$\cos y = \sum_{k=0}^{\infty} \frac{(-1)^k y^{2k}}{(2k)!}, \quad \sin y = \sum_{k=0}^{\infty} \frac{(-1)^k y^{2k+1}}{(2k+1)!}$$

因此：
$$f(iy) = \cos y + i\sin y$$

**步骤4：对一般复数 $z = x + iy$**
利用指数律 $f(z_1 + z_2) = f(z_1)f(z_2)$（可由Cauchy乘积严格证明）：
$$f(x+iy) = f(x)f(iy) = e^x(\cos y + i\sin y)$$
$\square$

### 3.2 方法Ⅱ：微分方程法

**定理 3.2**：微分方程 $f'(z) = f(z)$ 在复平面上存在唯一的满足 $f(0) = 1$ 的整函数解，且该解为 $f(z) = e^z$。

**证明**：

**存在性**：幂级数 $f(z) = \sum_{n=0}^{\infty} \frac{z^n}{n!}$ 在 $\mathbb{C}$ 上内闭一致收敛，可逐项求导：
$$f'(z) = \sum_{n=1}^{\infty} \frac{nz^{n-1}}{n!} = \sum_{n=1}^{\infty} \frac{z^{n-1}}{(n-1)!} = \sum_{m=0}^{\infty} \frac{z^m}{m!} = f(z)$$
且 $f(0) = 1$。

**唯一性**：设 $g(z)$ 为另一满足条件的整函数。考虑 $h(z) = f(z)g(-z)$。求导：
$$h'(z) = f'(z)g(-z) - f(z)g'(-z) = f(z)g(-z) - f(z)g(-z) = 0$$
由于 $h(z)$ 在连通区域 $\mathbb{C}$ 上导数恒为零，故 $h(z)$ 为常数。由 $h(0) = f(0)g(0) = 1$，得 $h(z) \equiv 1$，即 $g(-z) = 1/f(z)$。特别地，取 $g = f$，得 $f(z)f(-z) = 1$。

现在令 $g(z) = e^x(\cos y + i\sin y)$（直接验证可知它也满足微分方程）。由唯一性，$f(z) = g(z)$。$\square$

### 3.3 方法Ⅲ：复积分的解析延拓

**定理 3.3**：从实数域上的指数函数出发，通过解析延拓得到的复指数函数满足欧拉公式。

**证明思路**：

**步骤1**：实指数函数 $e^x$ 在 $\mathbb{R}$ 上满足 $f'(x) = f(x)$，$f(0) = 1$。

**步骤2**：寻找一个在整个复平面上解析的函数 $F(z)$，使得 $F(x) = e^x$ 对所有 $x \in \mathbb{R}$ 成立。由解析延拓的唯一性，这样的 $F$ 若存在则唯一。

**步骤3**：验证 $F(z) = e^x(\cos y + i\sin y)$ 满足Cauchy-Riemann方程：
设 $u(x,y) = e^x\cos y$，$v(x,y) = e^x\sin y$，则：
$$\frac{\partial u}{\partial x} = e^x\cos y = \frac{\partial v}{\partial y}, \quad \frac{\partial u}{\partial y} = -e^x\sin y = -\frac{\partial v}{\partial x}$$

C-R方程处处成立，且偏导数连续，故 $F(z)$ 在 $\mathbb{C}$ 上全纯。

**步骤4**：当 $y = 0$ 时，$F(x) = e^x$。由解析延拓的唯一性，$F(z)$ 就是 $e^z$ 的唯一定义。$\square$

## 四、欧拉公式在复分析中的核心地位

### 4.1 复指数函数的周期性与基本域

**定理 4.1**：复指数函数 $e^z$ 是以 $2\pi i$ 为基本周期的周期函数，即：
$$e^{z + 2\pi i} = e^z, \quad \forall z \in \mathbb{C}$$

**证明**：
$$e^{z + 2\pi i} = e^{x + i(y + 2\pi)} = e^x(\cos(y+2\pi) + i\sin(y+2\pi)) = e^x(\cos y + i\sin y) = e^z$$
$\square$

这一周期性揭示了复指数函数与实指数函数的本质区别：实指数函数 $e^x$ 是严格单调的，而复指数函数 $e^z$ 是周期性的。这导致复对数函数必须是**多值函数**。

### 4.2 复对数函数与多值性

**定义 4.2（复对数）**：复对数函数 $w = \log z$ 定义为 $e^w = z$ 的解。设 $z = re^{i\theta}$，则：
$$\log z = \ln r + i(\theta + 2k\pi), \quad k \in \mathbb{Z}$$

**主值分支**：取 $-\pi < \theta \leq \pi$，得到**主对数**（principal logarithm）：
$$\text{Log } z = \ln|z| + i\text{Arg } z$$

复对数的多值性直接源于复指数函数的周期性，而周期性又直接源于欧拉公式。因此，欧拉公式不仅是复指数函数的定义公式，更是整个复对数理论和复幂函数理论的逻辑起点。

### 4.3 三角函数与双曲函数的统一

欧拉公式的一个重要推论是揭示了三角函数与双曲函数之间的深刻联系：

**定理 4.3**：对任意复数 $z$：
$$\cos(iz) = \cosh z, \quad \sin(iz) = i\sinh z$$
$$\cosh(iz) = \cos z, \quad \sinh(iz) = i\sin z$$

**证明**：由欧拉公式：
$$\cos(iz) = \frac{e^{i(iz)} + e^{-i(iz)}}{2} = \frac{e^{-z} + e^{z}}{2} = \cosh z$$
其余三式同理。$\square$

这一结果表明：三角函数与双曲函数不是两种无关的函数族，而是同一复变函数在不同直线（实轴与虚轴）上的限制。这种统一性是复分析统一性力量的典型体现。

## 五、历史背景：18世纪的数学革命

### 5.1 欧拉之前的复数概念

16世纪，意大利数学家卡尔达诺（Cardano）在解三次方程时首次遇到了形如 $\sqrt{-1}$ 的表达式，但他将其视为"无用而精巧"的东西。17世纪，笛卡尔创造了"虚数"（imaginary number）这个略带贬义的术语。

直到18世纪，数学家们才开始认真对待复数。棣莫弗（de Moivre）发现了与欧拉公式密切相关的棣莫弗定理；约翰·伯努利（Johann Bernoulli）在研究积分时系统地使用了复数。但总体而言，复数仍然缺乏坚实的理论基础，主要被视为一种方便的形式计算工具。

### 5.2 《无穷分析引论》的划时代意义

1748年，欧拉的《无穷分析引论》（*Introductio in analysin infinitorum*）出版。这部两卷本的巨著被认为是数学史上最具影响力的教科书之一。在这部著作中，欧拉：

1. **将函数定义为解析表达式**，从而将分析从对几何曲线的依赖中解放出来；
2. **系统研究了无穷级数**，将幂级数作为分析的基本工具；
3. **将指数函数、对数函数和三角函数统一于幂级数框架**，从而自然地将它们的定义域推广到复数；
4. **首次明确表述了欧拉公式**，并推导了大量令人惊叹的恒等式。

### 5.3 从形式运算到严格理论的过渡

以现代标准看，欧拉的推导充满了我们今天会严格质疑的步骤：无穷级数的交换、发散级数的形式处理、对复对数多值性的忽视。然而，欧拉惊人的直觉使他几乎从未在结果上出错。

19世纪，柯西（Cauchy）、黎曼（Riemann）和魏尔斯特拉斯（Weierstrass）将复分析发展为严格的数学理论。柯西引入了复积分和解析函数的概念；黎曼将复函数视为黎曼曲面上的映射；魏尔斯特拉斯则用幂级数和解析延拓奠定了复分析的算术基础。但所有这些工作的核心对象——复指数函数——仍然是欧拉的创造。

## 六、现代意义与应用

### 6.1 信号处理与傅里叶分析

欧拉公式是**傅里叶分析**的数学基础。任何周期信号都可以表示为正弦和余弦函数的叠加：
$$f(t) = \sum_{n=-\infty}^{\infty} c_n e^{int}$$

利用欧拉公式，三角函数的线性组合可以紧凑地写成复指数的线性组合，从而大大简化了分析。在电气工程中，交流电路的分析几乎完全依赖于复数阻抗和欧拉公式；在数字信号处理中，离散傅里叶变换（DFT）和快速傅里叶变换（FFT）是欧拉公式的直接数值实现。

### 6.2 量子力学中的波函数

在量子力学中，粒子的状态由**波函数** $\psi(\mathbf{r}, t)$ 描述，其演化遵循**薛定谔方程**：
$$i\hbar \frac{\partial \psi}{\partial t} = \hat{H}\psi$$

方程中的虚数单位 $i$ 不是数学上的装饰，而是量子力学本质的一部分。自由粒子的平面波解为：
$$\psi(x,t) = Ae^{i(kx - \omega t)}$$

欧拉公式将这一复指数解解释为同时包含正弦和余弦成分的振荡，从而与波粒二象性的物理图像相联系。

### 6.3 拓扑学与欧拉示性数

虽然"欧拉公式"（$e^{i\pi} + 1 = 0$）与"欧拉示性数"（$V - E + F$）在字面上是不同概念，但它们都以欧拉命名，且都在某种意义上体现了数学的统一性。

现代拓扑学中，欧拉示性数可以通过**示性类**和**指标定理**与复分析中的留数理论相联系。Atiyah-Singer指标定理——20世纪数学最伟大的成就之一——将微分算子的解析性质与底流形的拓扑性质统一起来，而复分析（包括欧拉公式所定义的复结构）在其中扮演着核心角色。

**MIT课程引用**：MIT的《Complex Variables with Applications》(18.04) 和《Fourier Analysis》(18.103) 课程将欧拉公式作为整个课程的第一个核心高峰，教授学生"从实分析到复分析的范式转换"。
