## 一、简介

在数学的殿堂中，有些常数如 $\pi$ 和 $e$ 那样家喻户晓，有些则相对默默无闻却又同样神秘。欧拉-马歇罗尼常数（Euler-Mascheroni constant），记作 $\gamma$，正是后者中最具代表性的一个：

$$\gamma = \lim_{n \to \infty} \left( \sum_{k=1}^n \frac{1}{k} - \ln n \right) \approx 0.5772156649\ldots$$

这个常数最早由莱昂哈德·欧拉（Leonhard Euler）在1734年发现，后来由洛伦佐·马歇罗尼（Lorenzo Mascheroni）在1790年进一步研究，因此得名。它度量了调和级数与对数函数之间的渐近差距，是连接离散与连续、算术与分析的关键纽带。

然而，尽管经过了近三个世纪的研究，关于 $\gamma$ 的一个最基本问题至今悬而未决：**$\gamma$ 是有理数还是无理数？** 我们甚至不知道它是否是代数数。这种"无知"本身就说明了 $\gamma$ 的深刻与神秘。

本文将系统阐述欧拉常数的定义、积分与级数表示、它与Gamma函数和Riemann ζ函数的深刻联系、无理性问题的现状，以及它在现代数学和物理学中的广泛应用。

## 二、欧拉常数的定义与基本性质

### 2.1 经典定义：调和级数与自然对数之差

**定义 2.1（欧拉-马歇罗尼常数）**：
$$\gamma = \lim_{n \to \infty} \left( H_n - \ln n \right)$$

其中 $H_n = \sum_{k=1}^n \frac{1}{k}$ 是第 $n$ 个**调和数**（harmonic number）。

**定理 2.2**：上述极限存在且有限。

**证明**：考虑序列 $a_n = H_n - \ln n$。我们有：
$$a_{n+1} - a_n = \frac{1}{n+1} - \ln(n+1) + \ln n = \frac{1}{n+1} - \ln\left(1 + \frac{1}{n}\right)$$

由泰勒展开，对 $x > 0$：
$$\ln(1+x) = x - \frac{x^2}{2} + \frac{x^3}{3} - \cdots$$
取 $x = \frac{1}{n}$：
$$\ln\left(1 + \frac{1}{n}\right) = \frac{1}{n} - \frac{1}{2n^2} + O\left(\frac{1}{n^3}\right)$$
因此：
$$a_{n+1} - a_n = \frac{1}{n+1} - \frac{1}{n} + \frac{1}{2n^2} + O\left(\frac{1}{n^3}\right) = -\frac{1}{n(n+1)} + \frac{1}{2n^2} + O\left(\frac{1}{n^3}\right) = O\left(\frac{1}{n^2}\right)$$

由于 $\sum_{n=1}^{\infty} \frac{1}{n^2}$ 收敛，序列 $a_n$ 收敛。$\square$

**推论 2.3**：调和数有如下渐近展开：
$$H_n = \ln n + \gamma + \frac{1}{2n} - \frac{1}{12n^2} + \frac{1}{120n^4} - \cdots$$

### 2.2 积分表示

**定理 2.4**：欧拉常数有以下等价积分表示：

**(i)** $$\gamma = -\int_0^\infty e^{-x} \ln x \, dx$$

**(ii)** $$\gamma = \int_0^1 \frac{1 - e^{-x} - e^{-1/x}}{x} \, dx$$

**(iii)** $$\gamma = \int_0^1 \left( \frac{1}{\ln x} + \frac{1}{1-x} \right) dx$$

**(iv)** $$\gamma = \int_0^\infty \left( \frac{1}{1-e^{-x}} - \frac{1}{x} \right) e^{-x} \, dx$$

**证明（积分表示i）**：

由Gamma函数的定义：$\Gamma(s) = \int_0^\infty x^{s-1} e^{-x} \, dx$。

对 $s$ 求导：$\Gamma'(s) = \int_0^\infty x^{s-1} e^{-x} \ln x \, dx$。

已知 $\Gamma'(1) = -\gamma$（这由Digamma函数的定义直接得出，见第4.1节），故：
$$\gamma = -\Gamma'(1) = -\int_0^\infty e^{-x} \ln x \, dx$$
$\square$

### 2.3 级数与乘积表示

**定理 2.5（级数表示）**：

**(i) 交错级数**：
$$\gamma = \sum_{k=1}^{\infty} \left( \frac{1}{k} - \ln\frac{k+1}{k} \right)$$

**(ii) 更快的收敛级数（Euler-Maclaurin型）**：
$$\gamma = \sum_{k=1}^{\infty} (-1)^k \frac{\zeta(k)}{k}$$

其中 $\zeta(k)$ 是Riemann ζ函数在正整数点的值。

**(iii) Vacca级数（1910）**：
$$\gamma = \sum_{k=1}^{\infty} (-1)^k \frac{\lfloor \log_2 k \rfloor}{k}$$

**定理 2.6（乘积表示）**：
$$e^\gamma = \prod_{p \text{ prime}} \frac{p}{p-1} \cdot e^{-1/p}$$

这一表示揭示了 $\gamma$ 与素数分布之间的深刻联系。

## 三、γ的无理性与超越性之谜

### 3.1 无理性的困难

$\gamma$ 的无理性问题是数学中最古老、最令人沮丧的未解问题之一。与 $\pi$ 和 $e$ 不同——它们的无理性在18-19世纪就已被证明——$\gamma$ 的无理性至今悬而未决。

问题的困难在于：现有的证明无理性的标准工具（如连分数、Liouville定理、Hermite方法、Nesterenko定理）似乎都"刚好"不足以处理 $\gamma$。

### 3.2 已知结果与猜想

**定理 3.1（Rivoal, 2012）**：在Riemann假设成立的条件下，$\gamma$ 是无理数。

**历史进展**：
- 1978年，Brent计算了 $\gamma$ 的前29,000位小数；
- 2009年，Alexander Yee计算到了29,844,489,545位；
- 连分数展开的前数千项均未显示出周期性（暗示无理）；
- 然而，严格证明仍然缺失。

**猜想 3.2**：$\gamma$ 不仅是无理数，而且是**超越数**。

与这一猜想形成对比的是，我们甚至不能排除 $\gamma = 22/39$（或其他某个简单有理数）的可能性——尽管数值证据强烈反对这一点。

### 3.3 连分数展开与Diophantine逼近

$\gamma$ 的简单连分数展开为：
$$\gamma = [0; 1, 1, 2, 1, 2, 1, 4, 3, 13, 5, 1, 1, 8, 1, 2, 4, 1, 1, 40, \ldots]$$

如果 $\gamma$ 是有理数，其连分数展开终将周期化。目前的数值计算显示没有周期性迹象，但这不能作为证明。

**定理 3.3（Khinchine定理的推论）**：如果 $\gamma$ 是无理数，那么对几乎所有实数，其连分数展开的渐近行为遵循特定的统计规律。然而，$\gamma$ 是否属于这"几乎所有"的集合，同样未知。

## 四、γ与特殊函数的关系

### 4.1 Gamma函数与Digamma函数

**定义 4.1（Digamma函数）**：Digamma函数 $\psi(z)$ 是Gamma函数对数导数：
$$\psi(z) = \frac{d}{dz} \ln \Gamma(z) = \frac{\Gamma'(z)}{\Gamma(z)}$$

**定理 4.2**：
$$\psi(1) = -\gamma$$

**证明**：由Weierstrass的Gamma函数无穷乘积：
$$\frac{1}{\Gamma(z)} = z e^{\gamma z} \prod_{n=1}^{\infty} \left( 1 + \frac{z}{n} \right) e^{-z/n}$$

取对数并求导：
$$-\psi(z) = \frac{1}{z} + \gamma + \sum_{n=1}^{\infty} \left( \frac{1}{n+z} - \frac{1}{n} \right)$$

令 $z = 1$：
$$-\psi(1) = 1 + \gamma + \sum_{n=1}^{\infty} \left( \frac{1}{n+1} - \frac{1}{n} \right) = 1 + \gamma - 1 = \gamma$$

因此 $\psi(1) = -\gamma$。$\square$

**推论 4.3**：对正整数 $n$：
$$\psi(n) = H_{n-1} - \gamma$$

### 4.2 Riemann ζ函数与Stieltjes常数

**定义 4.4（Stieltjes常数）**：Riemann ζ函数在 $s = 1$ 附近的Laurent展开为：
$$\zeta(s) = \frac{1}{s-1} + \sum_{n=0}^{\infty} \frac{(-1)^n}{n!} \gamma_n (s-1)^n$$

其中 $\gamma_0 = \gamma$ 是欧拉-马歇罗尼常数，$\gamma_1, \gamma_2, \ldots$ 称为**广义Stieltjes常数**。

**定理 4.5**：Stieltjes常数可以表示为：
$$\gamma_n = \lim_{m \to \infty} \left[ \sum_{k=1}^m \frac{(\ln k)^n}{k} - \frac{(\ln m)^{n+1}}{n+1} \right]$$

这一定义将欧拉常数的概念自然地推广到了更高阶的"对数加权"情形。

### 4.3 广义欧拉常数

**定义 4.6（广义欧拉常数）**：对正整数 $a$，定义：
$$\gamma(a) = \lim_{n \to \infty} \left( \sum_{k=0}^n \frac{1}{k+a} - \ln n \right)$$

特别地，$\gamma(1) = \gamma$。

**定理 4.7**：
$$\gamma(a) = -\psi(a) = -\frac{\Gamma'(a)}{\Gamma(a)}$$

这些广义常数在特殊函数论和数论中有广泛应用。

## 五、历史背景：欧拉的发现之旅

### 5.1 1734年的突破性论文

1734年，年仅27岁的欧拉向圣彼得堡科学院提交了一篇题为《De progressionibus harmonicis observationes》的论文。在这篇论文中，欧拉不仅给出了调和级数渐近行为的系统研究，还首次计算了今天称为 $\gamma$ 的常数，精确到小数点后6位。

欧拉的推导基于一个天才的观察：虽然调和级数 $H_n$ 本身发散，但它与 $\ln n$ 的差收敛到一个有限常数。欧拉写道：

> "这个常数 $C = 0.577218\ldots$ 非常值得注意，因为它与对数和调和级数都有密切联系。"

### 5.2 欧拉对γ的数值计算

欧拉一生多次计算 $\gamma$ 的值。在他的著作中，我们可以找到以下记录：
- 1734年：$\gamma \approx 0.577218$（6位小数）
- 1736年：$\gamma \approx 0.5772156649015325$（16位小数）
- 晚年：欧拉声称计算到了更多位数，但其中可能包含一些错误

以18世纪的计算工具而言，这些成就令人惊叹。欧拉的计算方法是将调和级数与对数函数的级数展开进行比较，利用加速收敛技巧获得高精度结果。

### 5.3 从Euler到Mascheroni

1790年，意大利数学家洛伦佐·马歇罗尼（Lorenzo Mascheroni）在他的著作《Adnotationes ad calculum integralem Euleri》中进一步研究了 $\gamma$ 的性质。他计算了 $\gamma$ 的32位小数，但其中前19位是正确的。

为了纪念这两位数学家的贡献，这个常数被称为**欧拉-马歇罗尼常数**（Euler-Mascheroni constant）。有趣的是，马歇罗尼曾建议用字母 $A$ 表示这个常数，而今天我们普遍使用 $\gamma$（既可能是因为它与Gamma函数的联系，也可能是对高斯的致敬——高斯在许多场合使用了这个符号）。

## 六、现代意义与应用

### 6.1 数论中的Mertens定理

**定理 6.1（Mertens第一定理，1874）**：
$$\sum_{p \leq x} \frac{\ln p}{p} = \ln x + O(1)$$

**定理 6.2（Mertens第二定理）**：
$$\sum_{p \leq x} \frac{1}{p} = \ln \ln x + M + O\left(\frac{1}{\ln x}\right)$$

其中 $M \approx 0.261497$ 是**Meissel-Mertens常数**，与 $\gamma$ 有密切关系：
$$M = \gamma + \sum_p \left( \ln\left(1 - \frac{1}{p}\right) + \frac{1}{p} \right) = \gamma + \sum_{k=2}^{\infty} \frac{\mu(k) \ln \zeta(k)}{k}$$

Mertens定理是素数定理证明的重要准备，也揭示了 $\gamma$ 在解析数论中的核心地位。

### 6.2 概率论与期望偏差

在概率论中，$\gamma$ 出现在许多与期望和极限相关的问题中：

**记录值问题**：设 $X_1, X_2, \ldots$ 为独立同分布的连续随机变量。定义一个"记录值"为比之前所有值都大的观测。前 $n$ 个观测中记录值的期望个数为 $H_n$。因此，记录值期望个数与 $\ln n$ 的渐近差距正是 $\gamma$。

**优惠券收集问题**：收集全部 $n$ 种不同优惠券所需的期望试验次数为 $nH_n$。其超出于 $n \ln n$ 的部分为 $n\gamma + \frac{1}{2} + o(1)$。

### 6.3 量子场论与重整化

在理论物理学中，$\gamma$ 经常在发散积分的正则化计算中出现。特别是在**维数正规化**（dimensional regularization）中，当空间维度 $d = 4 - \epsilon$ 且 $\epsilon \to 0$ 时，圈图（loop diagrams）的发散部分常常表现为 $1/\epsilon$ 极点乘以一个包含 $\gamma$ 的有限项。

例如，在量子电动力学（QED）中，单圈真空极化的结果包含：
$$\frac{2}{3\epsilon} - \frac{1}{3}\gamma + \ln(4\pi) + \text{(finite terms)}$$

这些计算表明，$\gamma$ 不仅是纯数学的好奇对象，也是描述自然界基本规律的数学语言的一部分。

**Mathlib4引用**：Mathlib4的`Mathlib.Analysis.SpecialFunctions.Gamma.Digamma`模块包含了Digamma函数 $\psi(1) = -\gamma$ 的严格形式化证明，以及Gamma函数Weierstrass乘积展开中与欧拉常数相关的定理。
