# 欧拉-欧拉公式与复分析的严格证明

**msc_primary**: 30A99
**msc_secondary**: 30B10, 01A50, 00A99

---

## 简介

$$e^{i\pi} + 1 = 0$$

这个被理查德·费曼称为"数学中最卓越的公式"的恒等式，将数学中五个最重要的常数——$e$、$i$、$\pi$、$1$、$0$——以一种令人震惊的简洁方式联系在一起。莱昂哈德·欧拉（Leonhard Euler）在1748年出版的《无穷分析引论》（Introductio in analysin infinitorum）中首次系统阐述了这一公式。尽管以现代严格性标准衡量，欧拉的推导带有18世纪特有的形式操作色彩，但他的核心洞见——指数函数、三角函数与复数之间存在深刻的内在联系——被后世证明是完全正确的。

本文将从现代复分析的严格观点出发，给出欧拉公式 $e^{iz} = \cos z + i\sin z$ 的多种严格证明，探讨复指数函数的解析性质，并阐述欧拉公式在数学、物理学和工程学中的深远影响。欧拉公式与[高斯数学理念](../高斯数学理念/)中的复分析应用、[柯西数学理念](../柯西数学理念/)中的解析函数理论形成了复分析发展的核心脉络。

## 一、欧拉公式的历史发现与数学表述

### 1.1 从指数函数到复数域的延拓

欧拉之前的数学家已经注意到一些令人困惑的公式。1714年，Roger Cotes给出了一个等价的结果：
$$ix = \ln(\cos x + i\sin x)$$

这实际上就是欧拉公式的对数形式。然而，Cotes并没有认识到这一结果的全部意义。

欧拉的突破性贡献在于：他不是将 $e^{ix}$ 视为某种奇怪的符号组合，而是将其定义为指数函数在复数域上的**自然延拓**。在欧拉看来，幂级数
$$e^z = 1 + z + \frac{z^2}{2!} + \frac{z^3}{3!} + \cdots$$
不仅对于实数 $z$ 有效，对于复数 $z = x + iy$ 同样有效。

### 1.2 欧拉公式的严格表述

**定理 1.1（欧拉公式）**：对任意复数 $z \in \mathbb{C}$，有：
$$e^z = e^{x+iy} = e^x(\cos y + i\sin y)$$

特别地，当 $z = iy$（纯虚数）时：
$$e^{iy} = \cos y + i\sin y$$

**定义 1.2（复指数函数）**：复指数函数 $e^z$ 可通过以下等价方式定义：
1. **幂级数定义**：$e^z = \sum_{n=0}^{\infty} \frac{z^n}{n!}$
2. **微分方程定义**：$f(z) = e^z$ 是唯一满足 $f'(z) = f(z)$ 且 $f(0) = 1$ 的整函数
3. **极限定义**：$e^z = \lim_{n \to \infty} \left(1 + \frac{z}{n}\right)^n$

### 1.3 特殊情形与推论

**推论 1.3（欧拉恒等式）**：取 $z = i\pi$，得：
$$e^{i\pi} = \cos\pi + i\sin\pi = -1$$
即：$e^{i\pi} + 1 = 0$

**推论 1.4（三角函数的指数表示）**：
$$\cos z = \frac{e^{iz} + e^{-iz}}{2}, \quad \sin z = \frac{e^{iz} - e^{-iz}}{2i}$$

**推论 1.5（棣莫弗公式）**：对任意整数 $n$ 和实数 $\theta$：
$$(\cos\theta + i\sin\theta)^n = \cos(n\theta) + i\sin(n\theta)$$

## 二、欧拉公式的严格证明

### 2.1 方法Ⅰ：幂级数展开法

**定理 2.1**：幂级数 $f(z) = \sum_{n=0}^{\infty} \frac{z^n}{n!}$ 在整个复平面上绝对收敛，且满足 $f(z) = e^x(\cos y + i\sin y)$。

**证明**：

**步骤1：收敛半径**
由比值判别法：
$$\lim_{n \to \infty} \left| \frac{z^{n+1}/(n+1)!}{z^n/n!} \right| = \lim_{n \to \infty} \frac{|z|}{n+1} = 0$$
收敛半径为 $+\infty$，故级数在所有 $z \in \mathbb{C}$ 上收敛，定义了一个**整函数**。

**步骤2：代入 $z = iy$**
$$f(iy) = \sum_{n=0}^{\infty} \frac{(iy)^n}{n!} = \sum_{k=0}^{\infty} \frac{(-1)^k y^{2k}}{(2k)!} + i\sum_{k=0}^{\infty} \frac{(-1)^k y^{2k+1}}{(2k+1)!}$$

**步骤3：识别三角函数的幂级数**
$$f(iy) = \cos y + i\sin y$$

**步骤4：对一般复数 $z = x + iy$**
利用指数律 $f(z_1 + z_2) = f(z_1)f(z_2)$：
$$f(x+iy) = f(x)f(iy) = e^x(\cos y + i\sin y)$$
$\square$

### 2.2 方法Ⅱ：微分方程法

**定理 2.2**：微分方程 $f'(z) = f(z)$ 在复平面上存在唯一的满足 $f(0) = 1$ 的整函数解。

**证明**：

**存在性**：幂级数 $f(z) = \sum_{n=0}^{\infty} \frac{z^n}{n!}$ 可逐项求导，$f'(z) = f(z)$。

**唯一性**：设 $g(z)$ 为另一满足条件的整函数。考虑 $h(z) = f(z)g(-z)$。求导得 $h'(z) = 0$，故 $h(z) \equiv 1$，即 $g(-z) = 1/f(z)$。$\square$

### 2.3 方法Ⅲ：复积分的解析延拓

**定理 2.3**：从实数域上的指数函数出发，通过解析延拓得到的复指数函数满足欧拉公式。

**证明思路**：
1. 实指数函数 $e^x$ 在 $\mathbb{R}$ 上满足 $f'(x) = f(x)$，$f(0) = 1$。
2. 寻找在整个复平面上解析的函数 $F(z)$，使得 $F(x) = e^x$ 对所有 $x \in \mathbb{R}$ 成立。由解析延拓的唯一性，这样的 $F$ 若存在则唯一。
3. 验证 $F(z) = e^x(\cos y + i\sin y)$ 满足Cauchy-Riemann方程，故在 $\mathbb{C}$ 上全纯。
4. 当 $y = 0$ 时，$F(x) = e^x$。由解析延拓的唯一性，$F(z)$ 就是 $e^z$ 的唯一定义。$\square$

## 三、欧拉公式在复分析中的核心地位

### 3.1 复指数函数的周期性与基本域

**定理 3.1**：复指数函数 $e^z$ 是以 $2\pi i$ 为基本周期的周期函数。

**证明**：$e^{z + 2\pi i} = e^{x + i(y + 2\pi)} = e^x(\cos(y+2\pi) + i\sin(y+2\pi)) = e^z$。$\square$

这一周期性揭示了复指数函数与实指数函数的本质区别：实指数函数是严格单调的，而复指数函数是周期性的。这导致复对数函数必须是**多值函数**。

### 3.2 复对数函数与多值性

**定义 3.2（复对数）**：复对数函数 $w = \log z$ 定义为 $e^w = z$ 的解。设 $z = re^{i\theta}$，则：
$$\log z = \ln r + i(\theta + 2k\pi), \quad k \in \mathbb{Z}$$

**主值分支**：取 $-\pi < \theta \leq \pi$，得到**主对数**。

复对数的多值性直接源于复指数函数的周期性，而周期性又直接源于欧拉公式。

### 3.3 三角函数与双曲函数的统一

**定理 3.3**：对任意复数 $z$：
$$\cos(iz) = \cosh z, \quad \sin(iz) = i\sinh z$$
$$\cosh(iz) = \cos z, \quad \sinh(iz) = i\sin z$$

这一结果表明：三角函数与双曲函数不是两种无关的函数族，而是同一复变函数在不同直线上的限制。

## 四、欧拉公式在复分析中的核心地位

### 4.1 复指数函数的周期性与基本域

**定理 4.1**：复指数函数 $e^z$ 是以 $2\pi i$ 为基本周期的周期函数，即：
$$e^{z + 2\pi i} = e^z, \quad \forall z \in \mathbb{C}$$

**证明**：$e^{z + 2\pi i} = e^{x + i(y + 2\pi)} = e^x(\cos(y+2\pi) + i\sin(y+2\pi)) = e^x(\cos y + i\sin y) = e^z$。$\square$

这一周期性揭示了复指数函数与实指数函数的本质区别：实指数函数 $e^x$ 是严格单调的，而复指数函数 $e^z$ 是周期性的。这导致复对数函数必须是**多值函数**。

### 4.2 复对数函数与多值性

**定义 4.2（复对数）**：复对数函数 $w = \log z$ 定义为 $e^w = z$ 的解。设 $z = re^{i\theta}$，则：
$$\log z = \ln r + i(\theta + 2k\pi), \quad k \in \mathbb{Z}$$

**主值分支**：取 $-\pi < \theta \leq \pi$，得到**主对数**（principal logarithm）：
$$\text{Log } z = \ln|z| + i\text{Arg } z$$

复对数的多值性直接源于复指数函数的周期性，而周期性又直接源于欧拉公式。因此，欧拉公式不仅是复指数函数的定义公式，更是整个复对数理论和复幂函数理论的逻辑起点。

### 4.3 三角函数与双曲函数的统一

欧拉公式的一个重要推论是揭示了三角函数与双曲函数之间的深刻联系：

**定理 4.3**：对任意复数 $z$：
$$\cos(iz) = \cosh z, \quad \sin(iz) = i\sinh z$$
$$\cosh(iz) = \cos z, \quad \sinh(iz) = i\sin z$$

**证明**：由欧拉公式：
$$\cos(iz) = \frac{e^{i(iz)} + e^{-i(iz)}}{2} = \frac{e^{-z} + e^{z}}{2} = \cosh z$$
其余三式同理。$\square$

这一结果表明：三角函数与双曲函数不是两种无关的函数族，而是同一复变函数在不同直线（实轴与虚轴）上的限制。这种统一性是复分析统一性力量的典型体现。

### 4.4 复积分与留数定理

欧拉公式为复积分的计算提供了强大的工具。考虑围道积分
$$\oint_C \frac{e^{iz}}{z}\,dz$$
其中 $C$ 为围绕原点的简单闭曲线。由留数定理，该积分等于 $2\pi i \cdot \text{Res}_{z=0}\frac{e^{iz}}{z} = 2\pi i$。

将 $e^{iz} = \cos z + i\sin z$ 代入，并将积分路径取为上半平面的半圆，可以导出重要的Dirichlet积分：
$$\int_0^{\infty} \frac{\sin x}{x}\,dx = \frac{\pi}{2}$$

这一积分在信号处理和傅里叶分析中具有基本重要性。

## 五、留数定理与实积分计算

### 5.1 围道积分的基本定理

**定理 5.1（Cauchy积分定理）**：若 $f$ 在单连通区域 $D$ 上全纯，$\gamma$ 为 $D$ 中的任意可求长闭曲线，则
$$\oint_\gamma f(z)\,dz = 0$$

### 5.2 利用欧拉公式计算实积分

利用留数定理和欧拉公式，可以计算许多重要的实积分。

**例 5.1（Dirichlet积分）**：
$$\int_0^{\infty} \frac{\sin x}{x}\,dx = \frac{\pi}{2}$$

*证明*：考虑围道积分 $\oint_C \frac{e^{iz}}{z}\,dz$，其中 $C$ 为上半平面的半圆形围道。由留数定理，积分等于 $2\pi i$。将 $e^{iz} = \cos z + i\sin z$ 代入，取虚部即得。$\square$

### 5.3 Fresnel积分

**例 5.2（Fresnel积分）**：
$$\int_0^{\infty} \cos(x^2)\,dx = \int_0^{\infty} \sin(x^2)\,dx = \frac{1}{2}\sqrt{\frac{\pi}{2}}$$

这些积分在光学衍射理论中具有基本重要性。

## 六、历史背景

### 4.1 欧拉之前的复数概念

16世纪，卡尔达诺在解三次方程时首次遇到了形如 $\sqrt{-1}$ 的表达式。17世纪，笛卡尔创造了"虚数"这个略带贬义的术语。直到18世纪，数学家们才开始认真对待复数。

### 4.2 《无穷分析引论》的划时代意义

1748年，欧拉的《无穷分析引论》出版。这部巨著被认为是数学史上最具影响力的教科书之一。在这部著作中，欧拉：
1. **将函数定义为解析表达式**；
2. **系统研究了无穷级数**；
3. **将指数函数、对数函数和三角函数统一于幂级数框架**；
4. **首次明确表述了欧拉公式**。

### 4.3 从形式运算到严格理论

以现代标准看，欧拉的推导充满了我们今天会严格质疑的步骤。然而，欧拉惊人的直觉使他几乎从未在结果上出错。19世纪，柯西、黎曼和魏尔斯特拉斯将复分析发展为严格的数学理论。

## 五、现代意义与应用

### 5.1 信号处理与傅里叶分析

欧拉公式是**傅里叶分析**的数学基础。任何周期信号都可以表示为：
$$f(t) = \sum_{n=-\infty}^{\infty} c_n e^{int}$$

利用欧拉公式，三角函数的线性组合可以紧凑地写成复指数的线性组合。

### 5.2 量子力学中的波函数

在量子力学中，自由粒子的平面波解为：
$$\psi(x,t) = Ae^{i(kx - \omega t)}$$

欧拉公式将这一复指数解解释为同时包含正弦和余弦成分的振荡。

### 5.3 拓扑学与欧拉示性数

虽然"欧拉公式"与"欧拉示性数"在字面上是不同概念，但它们都以欧拉命名，且都在某种意义上体现了数学的统一性。

## 七、复分析的现代发展

### 7.1 Riemann曲面

**定义 7.1（Riemann曲面）**：一维复流形 $X$ 称为**Riemann曲面**，如果它带有复结构。

**定理 7.1（Riemann-Roch）**：设 $X$ 为紧Riemann曲面，亏格 $g$，$D$ 为除子。则
$$\ell(D) - \ell(K - D) = \deg(D) + 1 - g$$

### 7.2 模形式

**定义 7.2（模形式）**：上半平面 $\mathbb{H}$ 上的全纯函数 $f$ 称为**权 $k$ 模形式**，如果
$$f\left(\frac{az+b}{cz+d}\right) = (cz+d)^k f(z)$$

### 7.3 复动力系统

**定义 7.3（Julia集）**：对有理函数 $R(z)$，其**Julia集** $J(R)$ 是动力学行为不稳定点的闭包。

### 7.4 共形映射与Riemann映射定理

**定理 7.4（Riemann映射定理）**：设 $U \subset \mathbb{C}$ 为单连通真开子集，$z_0 \in U$。则存在唯一的双全纯映射 $f: U \to \mathbb{D}$（单位圆盘），使得 $f(z_0) = 0$ 且 $f'(z_0) > 0$。

### 7.5 Schwarz引理与度量几何

**定理 7.5（Schwarz-Pick引理）**：设 $f: \mathbb{D} \to \mathbb{D}$ 全纯。则对任意 $z, w \in \mathbb{D}$：
$$\left|\frac{f(z) - f(w)}{1 - \overline{f(w)}f(z)}\right| \leq \left|\frac{z - w}{1 - \overline{w}z}\right|$$
等号成立当且仅当 $f$ 为Möbius变换。

Poincaré度量 $ds = \frac{2|dz|}{1 - |z|^2}$ 在双全纯映射下不变，这是双曲几何的基础。

## 八、整函数与Picard定理

### 8.1 整函数

**定义 8.1（整函数）**：在整个复平面上全纯的函数称为**整函数**。

**定理 8.1（Liouville）**：有界整函数必为常数。

**定理 8.2（Hadamard分解）**：有限阶 $\rho$ 的整函数 $f$ 可以分解为
$$f(z) = z^m e^{P(z)} \prod_{n=1}^\infty E_{p}\left(\frac{z}{a_n}\right)$$
其中 $E_p$ 为初等因子，$P$ 为次数 $\leq \rho$ 的多项式。

### 8.2 Picard定理

**定理 8.3（Picard小定理）**：非常数整函数至多遗漏一个复数值。

**定理 8.4（Picard大定理）**：在本质奇点的任意去心邻域内，解析函数取每个复数值无穷多次，至多一个例外。

## 九：整函数与零点分布

### 9.1 Jensen公式

**定理 9.1（Jensen公式）**：设 $f$ 在 $|z| \leq R$ 上解析，$f(0) \neq 0$，$a_1, \ldots, a_n$ 为 $f$ 在 $|z| < R$ 内的零点（计重数）。则
$$\ln|f(0)| = \sum_{k=1}^n \ln\frac{|a_k|}{R} + \frac{1}{2\pi}\int_0^{2\pi} \ln|f(Re^{i\theta})|d\theta$$

### 9.2 Riemann ξ函数

**定义 9.1（Riemann ξ函数）**：
$$\xi(s) = \frac{1}{2}s(s-1)\pi^{-s/2}\Gamma\left(\frac{s}{2}\right)\zeta(s)$$

**定理 9.2**：$\xi(s)$ 是整函数，满足 $\xi(s) = \xi(1-s)$，且其零点与 $\zeta(s)$ 的非平凡零点重合。

### 9.3 Hadamard分解与Riemann假设

Riemann假设等价于断言：$\xi(s)$ 的所有零点都在 $\text{Re}(s) = 1/2$ 上。
