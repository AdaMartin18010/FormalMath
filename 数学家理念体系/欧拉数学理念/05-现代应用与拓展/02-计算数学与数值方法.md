## 一、欧拉数值方法的开创性贡献

### 1.1 欧拉方法的建立

**欧拉方法（1768）**：

```text
问题：
求解初值问题：
y' = f(t, y)
y(t₀) = y₀

方法：
1. 离散化：tₙ = t₀ + nh
2. 近似：y' ≈ (yₙ₊₁ - yₙ)/h
3. 迭代：yₙ₊₁ = yₙ + h·f(tₙ, yₙ)

其中：
- h：步长
- n：步数
- yₙ：近似值
```

**历史意义**：

```text
意义：
- 第一个系统化的数值方法
- 微分方程数值解的基础
- 影响深远

发展：
- 19世纪：改进方法
- 20世纪：现代理论
- 现代：广泛应用
```

### 7.2 梯度下降

**梯度下降方法**：

```text
方法：
x_{n+1} = x_n - α∇f(x_n)

其中：
- α：学习率
- ∇f：梯度

变种：
1. 批量梯度下降
2. 随机梯度下降
3. 小批量梯度下降
4. 自适应方法

应用：
```
