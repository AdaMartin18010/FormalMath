## 三、具体问题解决案例

### 3.1 Basel问题

**问题**：求 $\sum_{n=1}^{\infty} \frac{1}{n^2}$

**欧拉的解决过程**：

**步骤1：寻找联系**
将问题与已知函数联系。注意到 $\sin x$ 的零点在 $x = n\pi$（$n \in \mathbb{Z}$）。

**步骤2：选择函数**
选择 $\sin x$（零点明确）。

**步骤3：无穷乘积**
$$\sin x = x \prod_{n=1}^{\infty} \left(1 - \frac{x^2}{n^2\pi^2}\right)$$

**步骤4：展开比较**
- $\sin x$ 的Taylor展开：$\sin x = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots$
- 无穷乘积展开：$\sin x = x \left(1 - \frac{x^2}{\pi^2}\sum_{n=1}^{\infty}\frac{1}{n^2} + \cdots\right)$
- 比较 $x^3$ 的系数：$-\frac{1}{6} = -\frac{1}{\pi^2}\sum_{n=1}^{\infty}\frac{1}{n^2}$

**步骤5：得到结果**
$$\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6}$$

**特点**：
- 巧妙联系：将级数问题转化为函数展开问题
- 系统方法：使用无穷乘积和Taylor展开
- 深刻结果：连接数论和分析学

### 3.2 柯尼斯堡七桥问题

**问题**：能否一次走完所有桥？

**欧拉的解决过程**：

**步骤1：抽象化**
将实际问题抽象为数学问题。用图 $G = (V, E)$ 表示：
- 顶点 $V$：表示区域（$A, B, C, D$）
- 边 $E$：表示桥（7条边）

**步骤2：图论模型**
用点表示区域，用边表示桥。

**步骤3：寻找特征**
分析路径的数学特征。对于欧拉路径（经过每条边恰好一次）：
- 起点和终点：度数为奇数
- 中间点：度数为偶数（进入和离开次数相等）

**步骤4：建立判据**
奇度顶点必须 $\leq 2$。

**数学表述**：
图 $G$ 有欧拉路径当且仅当：
- $G$ 连通
- 奇度顶点数 $\leq 2$

**步骤5：应用判断**
七桥问题：4个奇度顶点（$A, B, C, D$ 的度数都是奇数）
→ 不可能

**特点**：
- 抽象思维：将实际问题转化为图论问题
- 开创性：开创图论这一新领域
- 建立新领域：为现代图论奠定基础

### 3.3 最速降线问题

**问题**：什么曲线使下降时间最短？

**欧拉的解决过程**：

**步骤1：参数化**
设曲线为 $y = f(x)$，从 $(0, 0)$ 到 $(a, b)$。

**步骤2：建立泛函**
时间 $T = \int dt = \int \frac{ds}{v}$，其中：
- 弧长元素：$ds = \sqrt{1 + (f'(x))^2} dx$
- 速度：$v = \sqrt{2gy}$（由能量守恒）

所以：
$$T = \int_0^a \frac{\sqrt{1 + (f'(x))^2}}{\sqrt{2gf(x)}} dx$$

**步骤3：变分法**
最小化泛函 $J[f] = \int_0^a L(x, f(x), f'(x)) dx$，其中：
$$L(x, y, y') = \frac{\sqrt{1 + (y')^2}}{\sqrt{2gy}}$$

**步骤4：欧拉-拉格朗日方程**
$$\frac{\partial L}{\partial y} - \frac{d}{dx}\frac{\partial L}{\partial y'} = 0$$

**步骤5：求解**
结果是旋轮线（摆线）：
$$x = R(\theta - \sin\theta), \quad y = R(1 - \cos\theta)$$

**特点**：
- 系统方法：使用变分法系统求解
- 创立变分法：为变分法奠定基础
- 深刻应用：在物理和工程中有广泛应用
