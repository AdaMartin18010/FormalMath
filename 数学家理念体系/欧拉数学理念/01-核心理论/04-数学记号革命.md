## 二、函数记号 f(x)

### 2.1 函数记号的历史

**之前的问题**：

```text
17-18世纪初：
- 无统一函数记号
- 用文字描述："x的函数"
- 表达繁琐
- 难以操作
```

**欧拉的创新**（1734）：

```text
引入记号：
f(x)：x的函数

优势：
- 简洁
- 通用
- 易于复合
- 便于操作

例子：
f(x) = x²
f(x+y) = (x+y)²
f(g(x))：复合函数
```

---

### 2.2 函数记号的影响

**对数学的促进**：

```text
函数概念发展：
- 明确函数对象
- 便于研究函数性质
- 促进函数理论

分析学发展：
- 函数运算方便
- 级数展开清晰
- 微积分应用
```

**现代应用**：

```text
所有数学：
- 函数是基本对象
- f(x)成为标准
- 计算机科学：函数式编程
```

## 三、自然对数的底 e

### 3.1 e的引入

**历史背景**：

```text
问题：
- 对数底的选择
- 自然对数的底未命名
- 表达复杂

定义：
e = lim(n→∞)(1 + 1/n)^n
≈ 2.71828...
```

**欧拉的记号**（1727-1728）：

```text
引入字母 e：
- 可能是"exponential"的首字母
- 简洁美观
- 易于书写

影响：
- 成为标准记号
- 广泛使用
- 连接分析与几何
```

---

### 3.2 e的重要性

**数学中的核心地位**：

```text
指数函数：
e^x：最重要的指数函数

性质：
- d/dx(e^x) = e^x
- 唯一导数等于自身的函数
- 连接微积分

应用：
- 复利计算
- 人口增长
- 放射性衰变
- 欧拉公式
```

**欧拉公式中的角色**：

```text
e^(iπ) + 1 = 0

e连接：
- 分析学（指数）
- 几何学（通过π）
- 代数学（通过i）
```

## 四、虚数单位 i

### 4.1 复数的历史

**虚数的困惑**：

```text
历史：
- Cardano（1545）：发现虚数
- Descartes（1637）：命名"虚数"
- 长期不被接受
- 记号不统一：√(-1), i, j
```

**欧拉的统一**（1777）：

```text
引入记号 i：
i = √(-1)
i² = -1

优势：
- 简洁
- 避免歧义
- 易于运算
- 成为标准
```

---

### 4.2 i的意义

**复数的代数表示**：

```text
标准形式：
z = a + bi

运算：
- 加法：(a+bi) + (c+di) = (a+c) + (b+d)i
- 乘法：(a+bi)(c+di) = (ac-bd) + (ad+bc)i
- 除法：通过共轭

欧拉公式：
e^(ix) = cos x + i sin x
```

**对数学的促进**：

```text
复分析：
- 统一实数与虚数
- 建立复分析
- 解决实数问题

应用：
- 电路分析
- 信号处理
- 量子力学
```

## 六、求和符号 Σ

### 6.1 求和记号的引入

**之前的问题**：

```text
表达繁琐：
"从1到n的所有x_i的和"

例子：
1 + 2 + 3 + ... + n
表达复杂
```

**欧拉的记号**：

```text
引入Σ：
∑(i=1 to n) x_i

优势：
- 简洁
- 清晰
- 易于操作
- 可嵌套
```

---

### 6.2 Σ的应用

**级数表达**：

```text
无穷级数：
∑(n=0 to ∞) aₙxⁿ

双重求和：
∑(i=1 to m) ∑(j=1 to n) a_ij

条件求和：
∑(p prime) 1/p
```

**对数学的促进**：

```text
级数理论：
- 表达清晰
- 操作方便
- 促进发展

现代应用：
- 概率论：期望值
- 统计学：样本和
- 计算机科学：循环
```
