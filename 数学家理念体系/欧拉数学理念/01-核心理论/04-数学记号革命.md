## 一、数学记号的重要性

### 1.1 记号对数学发展的影响

**记号的革命性作用**：

```text
好记号的特征：
- 简洁明了
- 易于操作
- 启发思维
- 促进交流

历史例子：
- 阿拉伯数字（vs罗马数字）
- 代数符号（vs文字描述）
```

**欧拉之前的记号状况**：

```text

```


## 二、函数记号 f(x)

### 2.1 函数记号的历史

**之前的问题**：

```text
17-18世纪初：
- 无统一函数记号
- 用文字描述："x的函数"
- 表达繁琐
- 难以操作
```

**欧拉的创新**（1734）：

```text
引入记号：
f(x)：x的函数

优势：
- 简洁
- 通用
- 易于复合
- 便于操作

例子：
f(x) = x²
f(x+y) = (x+y)²
f(g(x))：复合函数
```

---

### 2.2 函数记号的影响

**对数学的促进**：

```text
函数概念发展：
- 明确函数对象
- 便于研究函数性质
- 促进函数理论

分析学发展：
- 函数运算方便
- 级数展开清晰
- 微积分应用
```

**现代应用**：

```text
所有数学：
- 函数是基本对象
- f(x)成为标准
- 计算机科学：函数式编程
```

## 三、自然对数的底 e

### 3.1 e的引入

**历史背景**：

```text
问题：
- 对数底的选择
- 自然对数的底未命名
- 表达复杂

定义：
$$e = \lim_{n \to \infty} \left(1 + \frac{1}{n}\right)^n \approx 2.71828\ldots$$
```

**欧拉的记号**（1727-1728）：

```text
引入字母 e：
- 可能是"exponential"的首字母
- 简洁美观
- 易于书写

影响：
- 成为标准记号
- 广泛使用
- 连接分析与几何
```

---

### 3.2 e的重要性

**数学中的核心地位**：

```text
指数函数：
e^x：最重要的指数函数

性质：
- $\frac{d}{dx}(e^x) = e^x$
- 唯一导数等于自身的函数
- 连接微积分

应用：
- 复利计算
- 人口增长
- 放射性衰变
- 欧拉公式
```

**欧拉公式中的角色**：

**欧拉公式**：
$$e^{i\pi} + 1 = 0$$

$e$ 连接：
- 分析学（指数）
- 几何学（通过 $\pi$）
- 代数学（通过 $i$）

## 四、虚数单位 i

### 4.1 复数的历史

**虚数的困惑**：

```text
历史：
- Cardano（1545）：发现虚数
- Descartes（1637）：命名"虚数"
- 长期不被接受
- 记号不统一：√(-1), i, j
```

**欧拉的统一**（1777）：

**引入记号** $i$：
$$i = \sqrt{-1}, \quad i^2 = -1$$

**优势**：
- 简洁
- 避免歧义
- 易于运算
- 成为标准

---

### 4.2 i的意义

**复数的代数表示**：

```text
标准形式：
$$z = a + bi$$

运算：
- 加法：$(a+bi) + (c+di) = (a+c) + (b+d)i$
- 乘法：$(a+bi)(c+di) = (ac-bd) + (ad+bc)i$
- 除法：通过共轭

欧拉公式：
$$e^{ix} = \cos x + i \sin x$$
```

**对数学的促进**：

```text
复分析：
- 统一实数与虚数
- 建立复分析
- 解决实数问题

应用：
- 电路分析
- 信号处理
- 量子力学
```

## 五、圆周率 π

### 5.1 π的标准化

**历史**：

```text
古代：
- 各种近似值
- 无统一记号
- 表达复杂

Jones（1706）：
- 首次使用π
- 但未推广
```

**欧拉的推广**（1736）：

```text
广泛使用π：
- 在著作中大量使用
- 确立为标准
- 影响后世

定义：
$$\pi = \frac{\text{圆周}}{\text{直径}} \approx 3.14159\ldots$$
```

---

### 5.2 π在数学中的地位

**无处不在的π**：

```text
几何：
- 圆面积：$\pi r^2$
- 球体积：$\frac{4}{3}\pi r^3$

分析：
- 三角函数周期：$2\pi$
- Basel问题：$\frac{\pi^2}{6}$

数论：
- 素数分布：与 $\pi$ 相关
- 随机数生成

欧拉公式：
$$e^{i\pi} + 1 = 0$$
```

## 六、求和符号 Σ

### 6.1 求和记号的引入

**之前的问题**：

```text
表达繁琐：
"从1到n的所有x_i的和"

例子：
1 + 2 + 3 + ... + n
表达复杂
```

**欧拉的记号**：

```text
引入 $\Sigma$：
$$\sum_{i=1}^n x_i$$

优势：
- 简洁
- 清晰
- 易于操作
- 可嵌套
```

---

### 6.2 Σ的应用

**级数表达**：

```text
无穷级数：
$$\sum_{n=0}^{\infty} a_n x^n$$

双重求和：
$$\sum_{i=1}^m \sum_{j=1}^n a_{ij}$$

条件求和：
$$\sum_{p \text{ prime}} \frac{1}{p}$$
```

**对数学的促进**：

```text
级数理论：
- 表达清晰
- 操作方便
- 促进发展

现代应用：
- 概率论：期望值
- 统计学：样本和
- 计算机科学：循环
```

## 十一、数学公式总结

### 核心公式

1. **自然对数的底**：
   $$e = \lim_{n \to \infty} \left(1 + \frac{1}{n}\right)^n \approx 2.71828\ldots$$

2. **指数函数导数**：
   $$\frac{d}{dx}(e^x) = e^x$$

3. **欧拉公式**：
   $$e^{i\pi} + 1 = 0$$

4. **欧拉公式（一般形式）**：
   $$e^{ix} = \cos x + i \sin x$$

5. **虚数单位**：
   $$i = \sqrt{-1}, \quad i^2 = -1$$

6. **复数标准形式**：
   $$z = a + bi$$

7. **复数加法**：
   $$(a+bi) + (c+di) = (a+c) + (b+d)i$$

8. **复数乘法**：
   $$(a+bi)(c+di) = (ac-bd) + (ad+bc)i$$

9. **圆周率定义**：
   $$\pi = \frac{\text{圆周}}{\text{直径}} \approx 3.14159\ldots$$

10. **圆面积**：
    $$A = \pi r^2$$

11. **球体积**：
    $$V = \frac{4}{3}\pi r^3$$

12. **Basel问题**：
    $$\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6}$$

13. **求和符号**：
    $$\sum_{i=1}^n x_i$$

14. **无穷级数**：
    $$\sum_{n=0}^{\infty} a_n x^n$$

15. **双重求和**：
    $$\sum_{i=1}^m \sum_{j=1}^n a_{ij}$$

16. **三角函数级数展开**：
    $$\sin x = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots$$
    $$\cos x = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots$$

17. **函数记号**：
    $$f(x) = \text{函数值}$$

18. **函数复合**：
    $$f(g(x)) = (f \circ g)(x)$$
