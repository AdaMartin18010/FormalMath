## 二、排列与组合

### 2.1 排列

**排列的定义**：

```text
定义：
从n个不同元素中取r个元素的排列数：
P(n, r) = n!/(n-r)! = n(n-1)...(n-r+1)

特殊：
- 全排列：P(n, n) = n!
- 0! = 1（约定）

例子：
从5个元素中取3个的排列：
P(5, 3) = 5×4×3 = 60
```

**有重复的排列**：

```text
定义：
n个元素中有重复，第i种元素有nᵢ个，
则排列数为：
n!/(n₁! n₂! ... nₖ!)

例子：
"MATHEMATICS"的排列数：
11!/(2! 2! 2! 1! 1! 1! 1! 1!)
```

---

### 2.2 组合

**组合的定义**：

```text
定义：
从n个不同元素中取r个元素的组合数：
C(n, r) = n!/(r!(n-r)!) = (n choose r)

性质：
- C(n, r) = C(n, n-r)
- C(n, 0) = C(n, n) = 1
- C(n, r) = C(n-1, r-1) + C(n-1, r)（Pascal恒等式）

例子：
从5个元素中取3个的组合：
C(5, 3) = 10
```

**二项式定理**：

```text
定理：
(x + y)ⁿ = ∑(k=0 to n) C(n, k) x^k y^(n-k)

系数：
C(n, k)是二项式系数

例子：
(x + y)³ = x³ + 3x²y + 3xy² + y³
系数：1, 3, 3, 1 = C(3,0), C(3,1), C(3,2), C(3,3)
```

## 三、整数分拆

### 3.1 分拆的定义

**整数分拆**：

```text
定义：
将正整数n写成正整数之和的方式

例子：
n = 4的分拆：
4 = 4
4 = 3 + 1
4 = 2 + 2
4 = 2 + 1 + 1
4 = 1 + 1 + 1 + 1

分拆数：p(4) = 5
```

**分拆的表示**：

```text
表示：
分拆可以表示为：
n = a₁ + a₂ + ... + aₖ

其中：
- a₁ ≥ a₂ ≥ ... ≥ aₖ ≥ 1
- k是分拆的长度
```

---

### 3.2 分拆的性质

**基本性质**：

```text
性质1：
p(1) = 1
p(2) = 2
p(3) = 3
p(4) = 5
p(5) = 7
...

增长：
p(n)增长很快，近似：
p(n) ~ (1/(4n√3)) exp(π√(2n/3))
```

**限制条件的分拆**：

```text
问题：
- 分拆成k个部分：p(n, k)
- 分拆成奇数部分：p_odd(n)
- 分拆成不同部分：p_distinct(n)

关系：
p_distinct(n) = p_odd(n)
（Euler的发现）
```

## 四、生成函数方法

### 4.1 生成函数的定义

**普通生成函数**：

```text
定义：
序列{aₙ}的普通生成函数：
A(x) = ∑(n=0 to ∞) aₙ xⁿ

例子：
{1, 1, 1, ...}的生成函数：
1/(1-x) = 1 + x + x² + ...

{C(n,0), C(n,1), ..., C(n,n)}的生成函数：
(1+x)ⁿ = ∑(k=0 to n) C(n,k) x^k
```

**指数生成函数**：

```text
定义：
序列{aₙ}的指数生成函数：
A(x) = ∑(n=0 to ∞) aₙ xⁿ/n!

例子：
{1, 1, 1, ...}的指数生成函数：
e^x = ∑(n=0 to ∞) xⁿ/n!
```

---

### 4.2 分拆的生成函数

**欧拉的发现（1748）**：

```text
定理：
分拆数p(n)的生成函数：
P(x) = ∑(n=0 to ∞) p(n) xⁿ
     = ∏(k=1 to ∞) 1/(1-x^k)

证明思路：
1/(1-x^k) = 1 + x^k + x^(2k) + ...
乘积展开后，xⁿ的系数就是p(n)

意义：
- 连接级数与乘积
- 分析方法
- 计算工具
```

**例子**：

```text
计算p(5)：
P(x) = (1+x+x²+...)(1+x²+x⁴+...)(1+x³+x⁶+...)(1+x⁴+x⁸+...)(1+x⁵+...)

x⁵的系数：
- 5 = 5：1项
- 5 = 4+1：1项
- 5 = 3+2：1项
- 5 = 3+1+1：1项
- 5 = 2+2+1：1项
- 5 = 2+1+1+1：1项
- 5 = 1+1+1+1+1：1项

p(5) = 7 ✓
```

## 五、五边形数定理

### 5.1 五边形数

**五边形数的定义**：

```text
定义：
第n个五边形数：
Pₙ = n(3n-1)/2

序列：
1, 5, 12, 22, 35, 51, ...

几何意义：
可以排列成五边形的点数
```

---

### 5.2 欧拉五边形数定理（1748）

**定理陈述**：

```text
定理：
∏(n=1 to ∞) (1 - xⁿ) = ∑(k=-∞ to ∞) (-1)^k x^(k(3k-1)/2)

展开：
= 1 - x - x² + x⁵ + x⁷ - x¹² - x¹⁵ + ...

系数：
- xⁿ的系数：(-1)^k，如果n = k(3k-1)/2
- 否则：0

意义：
- 深刻的恒等式
- 连接数论与分析
- 分拆理论的基础
```

**证明思路**：

```text
方法：
1. 考虑乘积的展开
2. 使用组合论证
3. 建立递推关系
4. 得到恒等式

现代证明：
- 使用Jacobi三重积恒等式
- 模形式理论
- 组合证明
```

---

### 5.3 分拆数的递推公式

**欧拉递推公式**：

```text
公式：
p(n) = ∑(k≠0) (-1)^(k+1) p(n - k(3k-1)/2)

其中求和对所有k使得n - k(3k-1)/2 ≥ 0

意义：
- 计算分拆数
- 递推关系
- 高效算法

例子：
p(5) = p(4) + p(3) - p(0)
     = 5 + 3 - 1 = 7 ✓
```

