## 二、普通生成函数

### 2.1 定义

**普通生成函数**：

生成函数是组合数学中的重要工具。

**数学表述**：

**定义**：

序列 $\{a_n\}$ 的普通生成函数：
$$A(x) = \sum_{n=0}^{\infty} a_n x^n$$

**数学公式**：

**普通生成函数**：
$$A(x) = \sum_{n=0}^{\infty} a_n x^n$$

**性质**：
- 形式幂级数
- 系数提取：$a_n = [x^n]A(x)$（$x^n$ 的系数）
- 收敛性：在 $|x| < R$ 时收敛，其中 $R$ 是收敛半径

**基本生成函数**：

1. **常数序列**：$\{1, 1, 1, \ldots\}$ 的生成函数：
   $$\frac{1}{1-x} = \sum_{n=0}^{\infty} x^n = 1 + x + x^2 + x^3 + \cdots$$

2. **几何序列**：$\{a, ar, ar^2, \ldots\}$ 的生成函数：
   $$\frac{a}{1-rx} = \sum_{n=0}^{\infty} ar^n x^n$$

3. **二项式系数**：$\{C(n,0), C(n,1), \ldots, C(n,n)\}$ 的生成函数：
   $$(1+x)^n = \sum_{k=0}^{n} C(n,k) x^k$$

4. **斐波那契数列**：$\{F_0, F_1, F_2, \ldots\}$ 的生成函数：
   $$\frac{x}{1-x-x^2} = \sum_{n=0}^{\infty} F_n x^n$$

**例子**：

1. **常数序列**：$\frac{1}{1-x} = 1 + x + x^2 + \cdots$
2. **几何序列**：$\frac{1}{1-2x} = 1 + 2x + 4x^2 + 8x^3 + \cdots$
3. **二项式系数**：$(1+x)^5 = 1 + 5x + 10x^2 + 10x^3 + 5x^4 + x^5$

**基本生成函数**：

```text
常用生成函数：
1. 常数序列{1, 1, 1, ...}：
   1/(1-x) = ∑ xⁿ

2. 几何序列{a, ar, ar², ...}：
   a/(1-rx) = ∑ arⁿ xⁿ

3. 二项式系数{C(n,0), C(n,1), ..., C(n,n)}：
   (1+x)ⁿ = ∑ C(n,k) x^k

4. 斐波那契数列{F₀, F₁, F₂, ...}：
   x/(1-x-x²) = ∑ Fₙ xⁿ
```

---

### 2.2 组合意义

**生成函数的组合解释**：

```text
解释：
生成函数A(x) = ∑ aₙ xⁿ中，
xⁿ的系数aₙ表示"大小为n的对象个数"

例子：
- 硬币问题：xⁿ表示n分钱的方案数
- 分拆问题：xⁿ表示n的分拆数
- 路径问题：xⁿ表示长度为n的路径数
```

## 三、指数生成函数

### 3.1 定义

**指数生成函数**：

```text
定义：
序列{aₙ}的指数生成函数：
A(x) = ∑(n=0 to ∞) aₙ xⁿ/n!

性质：
- 指数级数
- 系数提取：aₙ = n! [xⁿ]A(x)
- 适合排列问题

例子：
{1, 1, 1, ...}的指数生成函数：
e^x = ∑ xⁿ/n!
```

**基本指数生成函数**：

```text
常用指数生成函数：
1. 常数序列{1, 1, 1, ...}：
   e^x = ∑ xⁿ/n!

2. 排列数{n!}：
   1/(1-x) = ∑ n! xⁿ/n!

3. 循环排列{(n-1)!}：
   -ln(1-x) = ∑ (n-1)! xⁿ/n!

4. 子集数{2ⁿ}：
   e^(2x) = ∑ 2ⁿ xⁿ/n!
```

---

### 3.2 应用

**排列问题**：

```text
应用：
指数生成函数适合处理：
- 排列问题
- 有标号对象
- 有序结构

例子：
n个对象的排列数：
n! = n! [xⁿ] (1/(1-x))
```

## 五、应用：组合问题

### 5.1 分拆问题

**整数分拆**：

```text
问题：
计算n的分拆数p(n)

生成函数：
P(x) = ∑ p(n) xⁿ
     = ∏(k=1 to ∞) 1/(1-x^k)

原因：
1/(1-x^k) = 1 + x^k + x^(2k) + ...
表示使用k的任意次
乘积表示所有k的组合

欧拉的发现（1748）：
这是分拆数的生成函数
```

---

### 5.2 递推关系

**斐波那契数列**：

```text
递推关系：
F₀ = 0, F₁ = 1
Fₙ = Fₙ₋₁ + Fₙ₋₂ (n ≥ 2)

生成函数：
F(x) = x/(1-x-x²)

方法：
1. 建立生成函数方程
2. 求解
3. 展开得到通项

通项：
Fₙ = (φⁿ - ψⁿ)/√5
其中φ = (1+√5)/2, ψ = (1-√5)/2
```

---

### 5.3 组合恒等式

**二项式定理**：

```text
恒等式：
(x + y)ⁿ = ∑ C(n, k) x^k y^(n-k)

生成函数证明：
(1 + x)ⁿ = ∑ C(n, k) x^k

应用：
- 组合恒等式
- 概率计算
- 多项式展开
```

## 九、数学公式总结

### 核心公式

1. **普通生成函数的定义**：
   $$A(x) = \sum_{n=0}^{\infty} a_n x^n$$

2. **常数序列的生成函数**：
   $$\frac{1}{1-x} = \sum_{n=0}^{\infty} x^n = 1 + x + x^2 + x^3 + \cdots$$

3. **几何序列的生成函数**：
   $$\frac{a}{1-rx} = \sum_{n=0}^{\infty} ar^n x^n$$

4. **二项式系数的生成函数**：
   $$(1+x)^n = \sum_{k=0}^{n} \binom{n}{k} x^k$$

5. **斐波那契数列的生成函数**：
   $$\frac{x}{1-x-x^2} = \sum_{n=0}^{\infty} F_n x^n$$

6. **指数生成函数的定义**：
   $$A(x) = \sum_{n=0}^{\infty} \frac{a_n}{n!} x^n$$

7. **指数函数的指数生成函数**：
   $$e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!}$$

8. **生成函数的加法**：
   $$(A + B)(x) = A(x) + B(x) = \sum_{n=0}^{\infty} (a_n + b_n) x^n$$

9. **生成函数的乘法**：
   $$(A \cdot B)(x) = A(x) \cdot B(x) = \sum_{n=0}^{\infty} \left(\sum_{i+j=n} a_i b_j\right) x^n$$

10. **生成函数的求导**：
    $$A'(x) = \sum_{n=0}^{\infty} n a_n x^{n-1}$$

11. **生成函数的积分**：
    $$\int A(x) dx = \sum_{n=0}^{\infty} \frac{a_n}{n+1} x^{n+1}$$

12. **分拆数的生成函数**（欧拉）：
    $$P(x) = \sum_{n=0}^{\infty} p(n) x^n = \prod_{k=1}^{\infty} \frac{1}{1-x^k}$$

13. **欧拉五边形数定理**：
    $$\prod_{n=1}^{\infty} (1 - x^n) = \sum_{k=-\infty}^{\infty} (-1)^k x^{k(3k-1)/2}$$

14. **斐波那契数列的通项公式**：
    $$F_n = \frac{\phi^n - \psi^n}{\sqrt{5}}$$
    其中 $\phi = \frac{1+\sqrt{5}}{2}$，$\psi = \frac{1-\sqrt{5}}{2}$。

15. **二项式定理**：
    $$(x + y)^n = \sum_{k=0}^{n} \binom{n}{k} x^k y^{n-k}$$

16. **多元生成函数**：
    $$A(x_1, x_2, \ldots, x_k) = \sum_{n_1, n_2, \ldots, n_k} a_{n_1, n_2, \ldots, n_k} x_1^{n_1} x_2^{n_2} \cdots x_k^{n_k}$$

17. **生成函数的复合**：
    $$(A \circ B)(x) = A(B(x))$$

18. **系数提取记号**：
    $$a_n = [x^n]A(x)$$
