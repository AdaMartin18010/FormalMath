## 一、历史背景：经典问题

### 1.1 柯尼斯堡七桥问题

**历史问题（1736）**：

```text
地点：
柯尼斯堡（Königsberg，现加里宁格勒）

地理：
- 普雷格尔河（Pregel River）
- 两个岛屿
- 七座桥连接

问题：
能否一次走过所有七座桥，每座桥恰好走一次？

图示：
    A
   /|\
  / | \
 B--C--D
  \ | /
   \|/
    E

区域：A, B, C, D（陆地）
岛屿：E（中心岛）
桥：AB, AC, AD, BC, BD, CD, CE（7座）
```

**问题的挑战**：

```text
挑战：
- 看似简单
- 但尝试多次都失败
- 需要理论分析

意义：
- 开创图论
- 抽象化方法
- 影响深远
```

## 二、问题的陈述

### 2.1 精确表述

**问题**：

```text
问题：
给定一个图，是否存在一条路径，
经过每条边恰好一次？

两种形式：
1. 欧拉路径：起点和终点可以不同
2. 欧拉回路：起点和终点相同（闭合路径）

柯尼斯堡问题：
是否存在欧拉路径？
```

---

### 2.2 图的表示

**抽象化**：

欧拉将柯尼斯堡七桥问题抽象为图论问题。

**数学表述**：

1. **图的定义**：
   - 顶点集 $V = \{A, B, C, D, E\}$（区域）
   - 边集 $E = \{AB, AC, AD, BC, BD, CD, CE\}$（桥）
   - 图 $G = (V, E)$

2. **度数分析**：
   - $\deg(A) = 3$
   - $\deg(B) = 3$
   - $\deg(C) = 3$
   - $\deg(D) = 3$
   - $\deg(E) = 3$
   - 所有顶点度数都是奇数

**数学公式**：

**图的定义**：
$$G = (V, E)$$
其中 $V = \{A, B, C, D, E\}$，$E = \{AB, AC, AD, BC, BD, CD, CE\}$

**度数**：
$$\deg(v) = \text{与顶点} v \text{相连的边数}$$

**度数性质**：
$$\sum_{v \in V} \deg(v) = 2|E|$$

**欧拉路径存在的必要条件**：
- 连通图
- 奇度顶点个数为 $0$ 或 $2$

**例子**：

1. **柯尼斯堡图**：所有顶点度数都是 $3$（奇数），有 $5$ 个奇度顶点，因此不存在欧拉路径
2. **简单路径**：如果只有 $2$ 个奇度顶点，则存在欧拉路径
3. **欧拉回路**：如果所有顶点度数都是偶数，则存在欧拉回路

### 3.2 度数的概念

**度数定义**：

度数是图论中的基本概念。

**数学表述**：

1. **度数定义**：
   - 顶点 $v$ 的度数 $\deg(v)$ 是与 $v$ 相连的边的条数
   - 对于无向图，每条边贡献 $2$ 度（两个端点各 $1$）

2. **度数性质**：
   - 所有顶点度数之和等于 $2|E|$
   - 度数为奇数的顶点有偶数个（握手引理）

**数学公式**：

**度数定义**：
$$\deg(v) = |\{e \in E : v \in e\}|$$

**度数性质**：
$$\sum_{v \in V} \deg(v) = 2|E|$$

**握手引理**：
$$\sum_{v \in V} \deg(v) = 2|E| \Rightarrow \text{奇度顶点个数为偶数}$$

**欧拉路径定理**：

连通图 $G$ 存在欧拉路径当且仅当：
- 奇度顶点个数为 $0$ 或 $2$

连通图 $G$ 存在欧拉回路当且仅当：
- 所有顶点度数都是偶数

**例子**：

1. **柯尼斯堡图**：所有顶点度数都是 $3$（奇数），有 $5$ 个奇度顶点，因此不存在欧拉路径
2. **简单图**：如果只有 $2$ 个奇度顶点，则存在欧拉路径（起点和终点是这两个奇度顶点）
3. **欧拉回路**：如果所有顶点度数都是偶数，则存在欧拉回路（可以从任意顶点开始并回到该顶点）

## 四、欧拉路径与欧拉回路

### 4.1 定义

**欧拉路径**：

欧拉路径是图论中的重要概念。

**定义**：

欧拉路径是经过图中每条边恰好一次的路径。

**数学表述**：

欧拉路径：
$$\text{路径 } P \text{ 使得每条边恰好经过一次}$$

**性质**：

- **可以重复经过顶点**：路径可以重复经过顶点
- **但不能重复经过边**：每条边只能经过一次
- **起点和终点可以不同**：起点和终点可以不同

**例子6：欧拉路径的应用**：

欧拉路径在图论中有重要应用。

**欧拉回路**：

欧拉回路是特殊的欧拉路径。

**定义**：

欧拉回路是经过图中每条边恰好一次的回路（闭合路径）。

**数学表述**：

欧拉回路：
$$\text{回路 } C \text{ 使得每条边恰好经过一次}$$

**性质**：

- **起点 = 终点**：起点等于终点
- **经过每条边恰好一次**：每条边恰好经过一次
- **是特殊的欧拉路径**：是特殊的欧拉路径

**例子7：欧拉回路的应用**：

欧拉回路在图论中有重要应用。

---

### 4.2 必要条件

**度数条件**：

欧拉路径和欧拉回路有明确的度数条件。

**观察**：

在欧拉路径中：
- **除了起点和终点**：每个顶点进入和离开次数相等
- **中间顶点的度数必须是偶数**：因此中间顶点的度数必须是偶数

**数学表述**：

度数条件：
- **欧拉路径**：最多2个奇度顶点（起点和终点）
- **欧拉回路**：0个奇度顶点（所有顶点度数都是偶数）

**结论**：

- **欧拉路径**：最多2个奇度顶点（起点和终点）
- **欧拉回路**：0个奇度顶点（所有顶点度数都是偶数）

**例子8：柯尼斯堡问题**：

柯尼斯堡图有5个奇度顶点 $> 2$，因此不存在欧拉路径。

## 五、欧拉定理

### 5.1 定理陈述

**欧拉定理（1736）**：

```text
定理：
连通图存在欧拉路径当且仅当：
- 最多有2个顶点的度数为奇数

连通图存在欧拉回路当且仅当：
- 所有顶点的度数都是偶数

等价：
- 欧拉路径 ⟺ 奇度顶点数 ≤ 2
- 欧拉回路 ⟺ 奇度顶点数 = 0
```

---

### 5.2 证明思路

**必要性**：

```text
证明：
在欧拉路径中：
- 起点：进入次数比离开次数少1
- 终点：离开次数比进入次数少1
- 其他顶点：进入次数 = 离开次数

因此：
- 起点和终点度数必须是奇数
- 其他顶点度数必须是偶数

结论：
最多2个奇度顶点
```

**充分性**：

```text
证明（构造性）：
1. 从奇度顶点（或任意顶点）开始
2. 沿着未访问的边前进
3. 如果无法继续，回到起点
4. 如果还有未访问的边，从已访问的顶点继续
5. 合并路径得到欧拉路径

（Fleury算法或Hierholzer算法）
```

---

### 5.3 柯尼斯堡问题的解答

**应用定理**：

```text
柯尼斯堡图：
- 5个顶点
- 所有顶点度数 = 3（奇数）
- 奇度顶点数 = 5 > 2

结论：
不存在欧拉路径

因此：
柯尼斯堡七桥问题无解
```

## 七、现代发展与应用

### 7.1 算法

**寻找欧拉路径的算法**：

```text
Fleury算法：
1. 从奇度顶点（或任意顶点）开始
2. 选择一条边，删除它
3. 如果删除后图不连通，选择其他边
4. 重复直到所有边都被访问

Hierholzer算法：
1. 从任意顶点开始，找到一条回路
2. 如果还有未访问的边，从已访问的顶点继续
3. 合并回路得到欧拉回路

复杂度：
O(|E|)
```

### 7.3 推广

**有向图**：

```text
有向欧拉路径：
存在当且仅当：
- 最多一个顶点：出度 - 入度 = 1（起点）
- 最多一个顶点：入度 - 出度 = 1（终点）
- 其他顶点：出度 = 入度

有向欧拉回路：
存在当且仅当：
- 所有顶点：出度 = 入度
```

**加权图**：

```text
中国邮递员问题：
在加权图中，寻找最短路径，
经过每条边至少一次

方法：
- 添加重复边
- 转化为欧拉回路
- 最小化总权重
```
