## 一、历史背景：从七桥问题到图论

### 1.1 图论的起源

**欧拉的贡献（1736）**：

```text
起点：
柯尼斯堡七桥问题

方法：
- 抽象化：区域→顶点，桥→边
- 建立图的概念
- 分析图的性质

结果：
- 解决七桥问题
- 开创图论
- 建立基础理论
```

**图论的发展**：

```text
欧拉（1736）：
- 图的概念
- 欧拉路径
- 基础理论

后续发展：
- Cayley（1857）：树的理论
- Kirchhoff（1847）：电路理论
- 现代：算法图论、网络科学
```

---

### 1.2 图论的意义

**为什么重要**：

```text
意义：
1. 统一处理离散结构
2. 抽象化方法
3. 应用广泛
4. 现代数学的基础

应用：
- 计算机科学
- 网络科学
- 运筹学
- 生物信息学
```

## 二、图的基本概念

### 2.1 图的定义

**无向图**：

```text
定义：
无向图G = (V, E)由以下组成：
- V：顶点集（vertices），非空有限集
- E：边集（edges），E ⊆ {{u,v} : u,v ∈ V, u ≠ v}

例子：
V = {1, 2, 3, 4}
E = {{1,2}, {2,3}, {3,4}, {1,4}}
```

**有向图**：

```text
定义：
有向图G = (V, E)由以下组成：
- V：顶点集
- E：边集，E ⊆ V × V（有序对）

例子：
V = {1, 2, 3}
E = {(1,2), (2,3), (3,1)}
```

---

### 2.2 基本术语

**顶点和边**：

```text
术语：
- 顶点（vertex, node）：图的元素
- 边（edge, link）：连接两个顶点
- 相邻（adjacent）：两个顶点之间有边
- 关联（incident）：边与顶点关联

例子：
图G中，顶点1和2相邻（有边连接）
边{1,2}与顶点1和2都关联
```

**度数**：

```text
定义：
顶点v的度数deg(v)是与v关联的边的条数

无向图：
- 每条边贡献2度（两个端点各1）
- ∑_{v∈V} deg(v) = 2|E|

有向图：
- 出度：从v出发的边数
- 入度：进入v的边数
- 出度之和 = 入度之和 = |E|
```

## 三、图的性质

### 3.1 握手引理

**握手引理**：

```text
引理：
对于任意图G = (V, E)：
∑_{v∈V} deg(v) = 2|E|

证明：
每条边连接两个顶点，在度数和中被计算两次

推论：
任意图中，度数为奇数的顶点有偶数个

证明：
所有度数之和为偶数，因此奇数度顶点必须成对出现
```

---

### 3.2 路径与连通性

**路径**：

```text
定义：
路径是顶点序列v₀, v₁, ..., vₖ，使得
对于i = 0, 1, ..., k-1，有{vᵢ, vᵢ₊₁} ∈ E

性质：
- 长度：路径中边的条数
- 简单路径：顶点不重复
- 回路：起点 = 终点（k ≥ 1）
```

**连通性**：

```text
定义：
无向图G是连通的，如果
对于任意两个顶点u, v，存在从u到v的路径

性质：
- 连通图：|E| ≥ |V| - 1
- 树：连通且|E| = |V| - 1
- 森林：无圈图（若干棵树的并）
```

---

### 3.3 欧拉公式（平面图）

**欧拉公式**：

```text
定理：
对于连通平面图G = (V, E, F)：
V - E + F = 2

其中：
- V：顶点数
- E：边数
- F：面数（包括外部面）

例子：
立方体：
V = 8, E = 12, F = 6
8 - 12 + 6 = 2 ✓
```

## 四、特殊类型的图

### 4.1 完全图

**完全图**：

```text
定义：
完全图Kₙ是n个顶点，任意两个顶点之间都有边的图

性质：
- 边数：|E| = n(n-1)/2
- 每个顶点度数：n-1
- 对称性高

例子：
K₃：三角形
K₄：四面体的边
```

---

### 4.2 二分图

**二分图**：

```text
定义：
二分图G = (V₁, V₂, E)的顶点集可以分成
两个独立集V₁和V₂，使得每条边连接
V₁中的一个顶点和V₂中的一个顶点

性质：
- 无奇长度回路
- 可以2-着色
- 应用：匹配问题

例子：
K₃,₃：完全二分图
```

---

### 4.3 树

**树**：

```text
定义：
树是连通无圈图

等价定义：
1. 连通且|E| = |V| - 1
2. 无圈且|E| = |V| - 1
3. 任意两点间存在唯一路径

性质：
- n个顶点的树有n-1条边
- 至少2个叶子（度数为1的顶点）
- 删除任意边后不连通
```

---

### 4.4 平面图

**平面图**：

```text
定义：
平面图是可以画在平面上，边不相交的图

性质：
- 满足欧拉公式：V - E + F = 2
- 边数限制：|E| ≤ 3|V| - 6（|V| ≥ 3）
- Kuratowski定理：不包含K₅或K₃,₃的子式

应用：
- 电路设计
- 地图着色
- 四色定理
```

## 五、图的表示

### 5.1 邻接矩阵

**邻接矩阵**：

```text
定义：
图G的邻接矩阵A是n×n矩阵，其中
A[i][j] = {
  1, 若{i,j} ∈ E（无向图）或(i,j) ∈ E（有向图）
  0, 否则
}

性质：
- 无向图：对称矩阵
- 有向图：一般不对称
- Aᵏ[i][j]：从i到j的长度为k的路径数
```

---

### 5.2 邻接表

**邻接表**：

```text
定义：
邻接表为每个顶点存储其邻接顶点列表

结构：
v₁ → [v₂, v₃, ...]
v₂ → [v₁, v₄, ...]
...

优势：
- 空间效率：O(V + E)
- 适合稀疏图
- 易于遍历邻接顶点
```

## 六、欧拉的贡献

### 6.1 图的概念

**欧拉的创新**：

```text
贡献：
1. 引入图的概念
2. 抽象化方法
3. 建立基础理论

方法：
- 将实际问题抽象为图
- 分析图的性质
- 解决实际问题
```

---

### 6.2 欧拉路径理论

**欧拉定理**：

```text
定理：
连通图存在欧拉路径当且仅当：
- 最多有2个顶点的度数为奇数

连通图存在欧拉回路当且仅当：
- 所有顶点的度数都是偶数

意义：
- 解决路径问题
- 建立图论基础
- 影响深远
```

---

### 6.3 平面图理论

**欧拉公式**：

```text
公式：
对于连通平面图：V - E + F = 2

意义：
- 连接几何与图论
- 拓扑不变量
- 现代图论的基础
```

## 七、现代发展

### 7.1 算法图论

**图算法**：

```text
算法：
1. 最短路径：Dijkstra, Floyd-Warshall
2. 最小生成树：Kruskal, Prim
3. 图的遍历：DFS, BFS
4. 拓扑排序
5. 强连通分量

复杂度：
- 多项式时间算法
- NP完全问题
- 近似算法
```

### 7.2 网络科学

**现代应用**：

```text
应用：
1. 社交网络：Facebook, Twitter
2. 互联网：网页链接
3. 生物网络：蛋白质相互作用
4. 交通网络：道路、航线
5. 电力网络：电网

方法：
```

### 7.3 随机图

**随机图理论**：

```text
Erdős-Rényi模型：
G(n, p)：n个顶点，每条边以概率p独立存在

性质：
- 相变现象
- 连通性阈值
- 小世界现象

应用：
- 网络建模
- 相变理论
- 复杂系统
```
