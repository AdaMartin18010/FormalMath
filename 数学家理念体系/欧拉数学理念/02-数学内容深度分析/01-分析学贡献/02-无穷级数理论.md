# 无穷级数理论：欧拉的分析工具

## 一、级数在欧拉数学中的地位

### 1.1 级数作为分析的基础

**欧拉的信念**（1748, Introductio）：

```text
核心观点：
"所有函数都可以表示为无穷级数"

理论基础：
- 函数可以用幂级数展开
- 级数是分析的通用语言
- 通过级数可以研究函数性质

与Taylor的区别：
- Taylor强调导数的存在性
- Euler更注重实用性和形式操作
```

---

### 1.2 级数的多重作用

**级数的功能**：

```text
1. 函数表示
   f(x) = ∑(n=0 to ∞) aₙxⁿ

2. 数值计算
   通过截断级数近似函数值

3. 函数研究
   从系数推断函数性质

4. 问题求解
   级数系数即问题的解
```

---

## 二、基本级数展开

### 2.1 指数函数

**展开**：

```text
e^x = ∑(n=0 to ∞) xⁿ/n!
= 1 + x + x²/2! + x³/3! + x⁴/4! + ...

收敛性：
- 对所有x ∈ ℝ收敛
- 收敛半径：∞

欧拉的贡献：
- 系统化展开
- 建立恒等式
- 发现e的特殊性质
```

**复指数函数**：

```text
e^z = ∑(n=0 to ∞) zⁿ/n!（z ∈ ℂ）

欧拉公式：
e^(ix) = cos x + i sin x

级数证明：
e^(ix) = ∑(n=0 to ∞) (ix)ⁿ/n!
= ∑(k=0 to ∞) (-1)^k x^(2k)/(2k)! + i∑(k=0 to ∞) (-1)^k x^(2k+1)/(2k+1)!
= cos x + i sin x
```

---

### 2.2 三角函数

**正弦函数**：

```text
sin x = ∑(n=0 to ∞) (-1)^n x^(2n+1)/(2n+1)!
= x - x³/3! + x⁵/5! - x⁷/7! + ...

收敛性：
- 对所有x ∈ ℝ收敛
- 周期函数：sin(x + 2π) = sin x
```

**余弦函数**：

```text
cos x = ∑(n=0 to ∞) (-1)^n x^(2n)/(2n)!
= 1 - x²/2! + x⁴/4! - x⁶/6! + ...

收敛性：
- 对所有x ∈ ℝ收敛
- 周期函数：cos(x + 2π) = cos x
```

**正切函数**：

```text
tan x = x + x³/3 + 2x⁵/15 + 17x⁷/315 + ...

Bernoulli数：
展开系数与Bernoulli数相关
欧拉发展了Bernoulli数理论
```

---

### 2.3 对数函数

**自然对数**：

```text
ln(1 + x) = ∑(n=1 to ∞) (-1)^(n+1) xⁿ/n
= x - x²/2 + x³/3 - x⁴/4 + ...

收敛区间：
- |x| < 1
- x = 1时也收敛（交错级数）

应用：
- 数值计算ln值
- 函数逼近
- 级数技巧
```

**一般对数**：

```text
ln((1+x)/(1-x)) = 2∑(n=0 to ∞) x^(2n+1)/(2n+1)
= 2(x + x³/3 + x⁵/5 + ...)

收敛更快：
用于计算ln值
```

---

### 2.4 二项式级数

**二项式展开**：

```text
(1 + x)^α = ∑(n=0 to ∞) C(α, n) xⁿ

其中：
C(α, n) = α(α-1)...(α-n+1)/n!

特殊情况：
α = -1: 1/(1+x) = 1 - x + x² - x³ + ...
α = 1/2: √(1+x) = 1 + x/2 - x²/8 + ...
α = -1/2: 1/√(1+x) = 1 - x/2 + 3x²/8 - ...

欧拉的贡献：
- 推广到任意实数指数
- 建立收敛理论
- 发现特殊值
```

## 三、特殊级数技巧

### 3.1 无穷乘积展开

**sin x的无穷乘积**：

```text
欧拉的重要发现（1748）：
sin x = x∏(n=1 to ∞)(1 - x²/(n²π²))

证明思路：
1. sin x的零点：x = nπ（n ∈ ℤ\{0}）
2. 类比多项式：P(x) = a∏(x - r_i)
3. sin x = x∏(n=1 to ∞)(1 - x/(nπ))(1 + x/(nπ))
4. 整理得到标准形式

应用：
- 研究三角函数性质
- 解决Basel问题
- 连接离散与连续
```

**其他无穷乘积**：

```text
余弦函数：
cos x = ∏(n=1 to ∞)(1 - 4x²/((2n-1)²π²))

Gamma函数：
1/Γ(z) = ze^(γz)∏(n=1 to ∞)(1 + z/n)e^(-z/n)
```

---

### 3.2 连分式展开

**连分式表示**：

```text
e的连分式：
e = 2 + 1/(1 + 1/(2 + 1/(1 + 1/(1 + 1/(4 + ...)))))

tan x的连分式：
tan x = x/(1 - x²/(3 - x²/(5 - x²/(7 - ...))))

优势：
- 收敛快
- 数值计算
- 函数近似
```

---

### 3.3 级数变换技巧

**级数加速**：

```text
问题：
某些级数收敛慢

方法：
- Abel变换
- Euler变换
- 级数重排

例子：
ln 2 = 1 - 1/2 + 1/3 - 1/4 + ...（收敛慢）

Euler变换：
通过级数变换加速收敛
```

## 四、级数在具体问题中的应用

### 4.1 数值计算

**π的计算**：

```text
方法1：arctan展开
π/4 = arctan 1 = 1 - 1/3 + 1/5 - 1/7 + ...
收敛慢

方法2：Machin公式
π/4 = 4 arctan(1/5) - arctan(1/239)
利用两个级数的组合
收敛更快

欧拉的改进：
发展多种公式
提高计算精度
```

---

### 4.2 函数方程求解

**差分方程**：

```text
问题：斐波那契数列
F₀ = 0, F₁ = 1, Fₙ = F_{n-1} + F_{n-2}

生成函数：
G(x) = ∑(n=0 to ∞) Fₙxⁿ

利用递推关系：
G(x) = x/(1 - x - x²)

级数展开：
- 部分分式分解
- 得到两个几何级数
- 提取系数

通项公式：
Fₙ = (φⁿ - ψⁿ)/√5
其中φ, ψ是黄金比例
```

---

### 4.3 微分方程求解

**幂级数解**：

```text
方程：y'' + xy = 0

假设解：
y = ∑(n=0 to ∞) aₙxⁿ

代入方程：
∑(n=2 to ∞) n(n-1)aₙx^(n-2) + x∑(n=0 to ∞) aₙxⁿ = 0

比较系数：
得到递推关系
确定系数

解：
y = a₀∑(n=0 to ∞) (-1)^n x^(3n)/(3^n n! ∏(k=1 to n)(3k-2)(3k-1))
+ a₁∑(n=0 to ∞) (-1)^n x^(3n+1)/(3^n n! ∏(k=1 to n)(3k-1)(3k))
```

## 五、级数的形式操作

### 5.1 欧拉的形式主义

**形式操作原则**：

```text
欧拉的观点：
"只要级数形式正确，
可以忽略收敛性问题"

例子：
1 + 2 + 4 + 8 + ... = -1

解释：
1/(1-x) = 1 + x + x² + ...
形式地令x=2：1/(1-2) = -1

现代观点：
- 部分可严格化（p-adic分析）
- 启发新理论（解析延拓）
- 需要严格化（Cauchy）
```

---

### 5.2 级数恒等式

**基本恒等式**：

```text
乘法：
(∑aₙxⁿ)(∑bₙxⁿ) = ∑(∑(k=0 to n) aₖb_{n-k})xⁿ

复合：
f(g(x)) = ∑aₙ(∑bₘx^m)ⁿ

求导：
d/dx(∑aₙxⁿ) = ∑naₙx^(n-1)

积分：
∫(∑aₙxⁿ)dx = ∑aₙx^(n+1)/(n+1)
```
