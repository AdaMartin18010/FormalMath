## 二、Gamma函数

### 2.1 阶乘的推广

**问题**：

```text
问题：
n! = 1·2·3·...·n

推广：
- n为非整数？
- 连续推广？

欧拉的创新（1729）：
引入Gamma函数
```

**定义**：

**积分定义**：
$$\Gamma(z) = \int_0^{\infty} t^{z-1} e^{-t} dt$$

**关系**：
$$\Gamma(n+1) = n! \quad \text{（$n$ 为正整数）}$$

**性质**：
- $\Gamma(z+1) = z\Gamma(z)$
- $\Gamma(1) = 1$
- $\Gamma(1/2) = \sqrt{\pi}$

---

### 2.2 Gamma函数的性质

**递推关系**：

**基本性质**：
$$\Gamma(z+1) = z\Gamma(z)$$

**证明**：
$$\Gamma(z+1) = \int_0^{\infty} t^z e^{-t} dt = [-t^z e^{-t}]_0^{\infty} + z\int_0^{\infty} t^{z-1} e^{-t} dt = z\Gamma(z)$$

**应用**：
- 计算非整数阶乘
- 特殊值计算
- 函数方程

**特殊值**：

```text
整数：
Γ(n+1) = n!

半整数：
Γ(1/2) = √π
Γ(3/2) = (1/2)√π
Γ(5/2) = (3/2)(1/2)√π

负值：
通过递推关系定义
```

---

### 2.3 Gamma函数的应用

**概率分布**：

```text
正态分布：
涉及Γ(1/2) = √π

χ²分布：
涉及Gamma函数

应用：
- 统计学
- 概率论
- 随机过程
```

**复分析**：

```text
解析延拓：
Γ(z)可以延拓到整个复平面
（除负整数外）

应用：
- 复变函数理论
- 解析数论
- 特殊函数理论
```

## 三、Beta函数

### 3.1 Beta函数的定义

**定义**：

**积分定义**：
$$B(x, y) = \int_0^1 t^{x-1} (1-t)^{y-1} dt$$

**与Gamma函数关系**：
$$B(x, y) = \frac{\Gamma(x)\Gamma(y)}{\Gamma(x+y)}$$

**对称性**：
$$B(x, y) = B(y, x)$$

---

### 3.2 Beta函数的性质

**基本性质**：

```text
递推关系：
B(x+1, y) = x/(x+y) B(x, y)

对称性：
B(x, y) = B(y, x)

特殊值：
B(1, 1) = 1
B(1/2, 1/2) = π
```

---

### 3.3 Beta函数的应用

**概率论**：

```text
Beta分布：
概率密度函数涉及Beta函数

应用：
- 贝叶斯统计
- 可靠性分析
- 顺序统计量
```

**积分计算**：

```text
积分：
∫₀¹ x^m (1-x)^n dx = B(m+1, n+1)

应用：
- 积分计算
- 特殊积分
- 组合恒等式
```

## 四、Bernoulli数与Euler数

### 4.1 Bernoulli数

**定义**：

**生成函数**：
$$\frac{x}{e^x - 1} = \sum_{n=0}^{\infty} B_n \frac{x^n}{n!}$$

**前几项**：
- $B_0 = 1$
- $B_1 = -\frac{1}{2}$
- $B_2 = \frac{1}{6}$
- $B_3 = 0$
- $B_4 = -\frac{1}{30}$
- $B_5 = 0$
- $B_6 = \frac{1}{42}$
- $\ldots$

**性质**：
- $B_{2n+1} = 0$（$n \geq 1$）
- 有理数
- 与 $\zeta$ 函数相关

---

### 4.2 Bernoulli数的应用

**求和公式**：

```text
Euler-Maclaurin公式：
∑(k=1 to n) f(k) = ∫₁ⁿ f(x)dx + (f(1)+f(n))/2
                  + ∑(k=1 to m) B_{2k}/(2k)! (f^(2k-1)(n) - f^(2k-1)(1))
                  + Rₘ

应用：
- 级数求和
- 数值积分
- 渐近展开
```

**Zeta函数**：

```text
关系：
ζ(2n) = (-1)^(n+1) (2π)^(2n) B_{2n}/(2(2n)!)

例子：
ζ(2) = π²/6 = (2π)² B₂/4 = π²/6
```

---

### 4.3 Euler数

**定义**：

```text
生成函数：
1/cosh x = 2/(e^x + e^(-x)) = ∑(n=0 to ∞) Eₙ xⁿ/n!

前几项：
E₀ = 1
E₁ = 0
E₂ = -1
E₃ = 0
E₄ = 5
E₅ = 0
...

性质：
- E_{2n+1} = 0
- 与三角函数相关
```

## 五、贝塞尔函数

### 5.1 贝塞尔方程

**方程**：

**Bessel方程**：
$$x^2y'' + xy' + (x^2 - n^2)y = 0$$

**解**：
- $J_n(x)$：第一类贝塞尔函数
- $Y_n(x)$：第二类贝塞尔函数

**欧拉的贡献**：
- 早期研究
- 为后人铺路

---

### 5.2 贝塞尔函数的性质

**级数表示**：

```text
第一类：
Jₙ(x) = ∑(m=0 to ∞) (-1)^m (x/2)^(n+2m)/(m!(n+m)!)

性质：
- 正交性
- 递推关系
- 渐近行为
```

---

### 5.3 应用

**物理应用**：

```text
波动方程：
圆柱坐标下的解涉及贝塞尔函数

应用：
- 声学
- 电磁学
- 量子力学
```

## 九、数学公式总结

### 核心公式

1. **Gamma函数定义**：
   $$\Gamma(z) = \int_0^{\infty} t^{z-1} e^{-t} dt$$

2. **Gamma函数与阶乘**：
   $$\Gamma(n+1) = n! \quad \text{（$n$ 为正整数）}$$

3. **Gamma函数递推关系**：
   $$\Gamma(z+1) = z\Gamma(z)$$

4. **Gamma函数特殊值**：
   - $\Gamma(1) = 1$
   - $\Gamma(1/2) = \sqrt{\pi}$
   - $\Gamma(3/2) = \frac{1}{2}\sqrt{\pi}$

5. **Beta函数定义**：
   $$B(x, y) = \int_0^1 t^{x-1} (1-t)^{y-1} dt$$

6. **Beta函数与Gamma函数关系**：
   $$B(x, y) = \frac{\Gamma(x)\Gamma(y)}{\Gamma(x+y)}$$

7. **Beta函数对称性**：
   $$B(x, y) = B(y, x)$$

8. **Bernoulli数生成函数**：
   $$\frac{x}{e^x - 1} = \sum_{n=0}^{\infty} B_n \frac{x^n}{n!}$$

9. **Bernoulli数性质**：
   $$B_{2n+1} = 0 \quad \text{（$n \geq 1$）}$$

10. **Bernoulli数与Zeta函数**：
    $$\zeta(2n) = (-1)^{n+1} \frac{(2\pi)^{2n} B_{2n}}{2(2n)!}$$

11. **Euler数生成函数**：
    $$\frac{1}{\cosh x} = \sum_{n=0}^{\infty} E_n \frac{x^n}{n!}$$

12. **Bessel方程**：
    $$x^2y'' + xy' + (x^2 - n^2)y = 0$$

13. **第一类贝塞尔函数级数**：
    $$J_n(x) = \sum_{m=0}^{\infty} \frac{(-1)^m (x/2)^{n+2m}}{m!(n+m)!}$$

14. **Euler-Maclaurin公式**：
    $$\sum_{k=1}^n f(k) = \int_1^n f(x)dx + \frac{f(1)+f(n)}{2} + \sum_{k=1}^m \frac{B_{2k}}{(2k)!}(f^{(2k-1)}(n) - f^{(2k-1)}(1)) + R_m$$

15. **Beta函数递推关系**：
    $$B(x+1, y) = \frac{x}{x+y} B(x, y)$$
