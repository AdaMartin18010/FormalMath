## 二、Gamma函数

### 2.1 阶乘的推广

**问题**：

```text
问题：
n! = 1·2·3·...·n

推广：
- n为非整数？
- 连续推广？

欧拉的创新（1729）：
引入Gamma函数
```

**定义**：

```text
积分定义：
Γ(z) = ∫₀^∞ t^(z-1) e^(-t) dt

关系：
Γ(n+1) = n!（n为正整数）

性质：
- Γ(z+1) = zΓ(z)
- Γ(1) = 1
- Γ(1/2) = √π
```

---

### 2.2 Gamma函数的性质

**递推关系**：

```text
基本性质：
Γ(z+1) = zΓ(z)

证明：
Γ(z+1) = ∫₀^∞ t^z e^(-t) dt
       = [-t^z e^(-t)]₀^∞ + z∫₀^∞ t^(z-1) e^(-t) dt
       = zΓ(z)

应用：
- 计算非整数阶乘
- 特殊值计算
- 函数方程
```

**特殊值**：

```text
整数：
Γ(n+1) = n!

半整数：
Γ(1/2) = √π
Γ(3/2) = (1/2)√π
Γ(5/2) = (3/2)(1/2)√π

负值：
通过递推关系定义
```

---

### 2.3 Gamma函数的应用

**概率分布**：

```text
正态分布：
涉及Γ(1/2) = √π

χ²分布：
涉及Gamma函数

应用：
- 统计学
- 概率论
- 随机过程
```

**复分析**：

```text
解析延拓：
Γ(z)可以延拓到整个复平面
（除负整数外）

应用：
- 复变函数理论
- 解析数论
- 特殊函数理论
```

## 三、Beta函数

### 3.1 Beta函数的定义

**定义**：

```text
积分定义：
B(x, y) = ∫₀¹ t^(x-1) (1-t)^(y-1) dt

与Gamma函数关系：
B(x, y) = Γ(x)Γ(y)/Γ(x+y)

对称性：
B(x, y) = B(y, x)
```

---

### 3.2 Beta函数的性质

**基本性质**：

```text
递推关系：
B(x+1, y) = x/(x+y) B(x, y)

对称性：
B(x, y) = B(y, x)

特殊值：
B(1, 1) = 1
B(1/2, 1/2) = π
```

---

### 3.3 Beta函数的应用

**概率论**：

```text
Beta分布：
概率密度函数涉及Beta函数

应用：
- 贝叶斯统计
- 可靠性分析
- 顺序统计量
```

**积分计算**：

```text
积分：
∫₀¹ x^m (1-x)^n dx = B(m+1, n+1)

应用：
- 积分计算
- 特殊积分
- 组合恒等式
```

## 四、Bernoulli数与Euler数

### 4.1 Bernoulli数

**定义**：

```text
生成函数：
x/(e^x - 1) = ∑(n=0 to ∞) Bₙ xⁿ/n!

前几项：
B₀ = 1
B₁ = -1/2
B₂ = 1/6
B₃ = 0
B₄ = -1/30
B₅ = 0
B₆ = 1/42
...

性质：
- B_{2n+1} = 0（n ≥ 1）
- 有理数
- 与ζ函数相关
```

---

### 4.2 Bernoulli数的应用

**求和公式**：

```text
Euler-Maclaurin公式：
∑(k=1 to n) f(k) = ∫₁ⁿ f(x)dx + (f(1)+f(n))/2
                  + ∑(k=1 to m) B_{2k}/(2k)! (f^(2k-1)(n) - f^(2k-1)(1))
                  + Rₘ

应用：
- 级数求和
- 数值积分
- 渐近展开
```

**Zeta函数**：

```text
关系：
ζ(2n) = (-1)^(n+1) (2π)^(2n) B_{2n}/(2(2n)!)

例子：
ζ(2) = π²/6 = (2π)² B₂/4 = π²/6
```

---

### 4.3 Euler数

**定义**：

```text
生成函数：
1/cosh x = 2/(e^x + e^(-x)) = ∑(n=0 to ∞) Eₙ xⁿ/n!

前几项：
E₀ = 1
E₁ = 0
E₂ = -1
E₃ = 0
E₄ = 5
E₅ = 0
...

性质：
- E_{2n+1} = 0
- 与三角函数相关
```

## 五、贝塞尔函数

### 5.1 贝塞尔方程

**方程**：

```text
Bessel方程：
x²y'' + xy' + (x² - n²)y = 0

解：
Jₙ(x)：第一类贝塞尔函数
Yₙ(x)：第二类贝塞尔函数

欧拉的贡献：
- 早期研究
- 为后人铺路
```

---

### 5.2 贝塞尔函数的性质

**级数表示**：

```text
第一类：
Jₙ(x) = ∑(m=0 to ∞) (-1)^m (x/2)^(n+2m)/(m!(n+m)!)

性质：
- 正交性
- 递推关系
- 渐近行为
```

---

### 5.3 应用

**物理应用**：

```text
波动方程：
圆柱坐标下的解涉及贝塞尔函数

应用：
- 声学
- 电磁学
- 量子力学
```
