## 二、一阶微分方程

### 2.1 可分离变量方程

**形式**：

```text
方程：
dy/dx = f(x)g(y)

解法：
分离变量：
dy/g(y) = f(x)dx

积分：
∫dy/g(y) = ∫f(x)dx + C

例子：
dy/dx = xy

分离：
dy/y = x dx

积分：
ln|y| = x²/2 + C

解：
y = Ce^(x²/2)
```

---

### 2.2 齐次方程

**形式**：

**方程**：
$$\frac{dy}{dx} = f\left(\frac{y}{x}\right)$$

**方法**：
令$u = y/x$，则$y = ux$

**变换**：
$$\frac{dy}{dx} = u + x \frac{du}{dx} = f(u)$$

**分离**：
$$\frac{du}{f(u) - u} = \frac{dx}{x}$$

积分求解

---

### 2.3 线性方程

**形式**：

**方程**：
$$\frac{dy}{dx} + P(x)y = Q(x)$$

**解法**：积分因子法

**积分因子**：
$$\mu(x) = e^{\int P(x)dx}$$

**乘方程**：
$$\mu(x)\frac{dy}{dx} + \mu(x)P(x)y = \mu(x)Q(x)$$

**左边是$(\mu y)'$**：
$$\frac{d(\mu y)}{dx} = \mu Q$$

**积分**：
$$\mu y = \int \mu Q dx + C$$

**解**：
$$y = \frac{1}{\mu}\left(\int \mu Q dx + C\right)$$

---

### 2.4 伯努利方程

**形式**：

**方程**：
$$\frac{dy}{dx} + P(x)y = Q(x)y^n$$

**方法**：
令$z = y^{1-n}$

变换为线性方程求解

---

## 三、高阶微分方程

### 3.1 线性齐次方程

**形式**：

**方程**：
$$a_n y^{(n)} + a_{n-1} y^{(n-1)} + \cdots + a_1 y' + a_0 y = 0$$

**特征方程**：
$$a_n r^n + a_{n-1} r^{n-1} + \cdots + a_1 r + a_0 = 0$$

**特征根**：
$$r_1, r_2, \ldots, r_n$$

**解**：
- **单根**：$Ce^{rx}$
- **重根**：$(C_1 + C_2 x + \cdots)e^{rx}$
- **复根**：$e^{\alpha x}(C_1\cos \beta x + C_2\sin \beta x)$

---

### 3.2 线性非齐次方程

**形式**：

**方程**：
$$L[y] = f(x)$$

其中$L$是线性微分算子

**解**：
$$y = y_h + y_p$$

其中：
- $y_h$：齐次方程的通解
- $y_p$：非齐次方程的特解

**求特解**：
- 待定系数法
- 变分常数法
- 算子方法

---

## 四、级数解方法

### 4.1 幂级数解

**方法**：

**假设解**：
$$y = \sum_{n=0}^{\infty} a_n x^n$$

**代入方程**：
得到系数递推关系

**确定系数**：
- $a_0, a_1$任意（初始条件）
- 其他由递推关系确定

**例子：Airy方程**
$$y'' - xy = 0$$

**假设**：
$$y = \sum_{n=0}^{\infty} a_n x^n$$

**代入**：
$$\sum_{n=0}^{\infty} n(n-1)a_n x^{n-2} - x\sum_{n=0}^{\infty} a_n x^n = 0$$

**递推**：
$$a_{n+2} = \frac{a_{n-1}}{(n+2)(n+1)}$$

**解**：
$$y = a_0\left[1 + \frac{x^3}{2 \cdot 3} + \frac{x^6}{2 \cdot 3 \cdot 5 \cdot 6} + \cdots\right] + a_1\left[x + \frac{x^4}{3 \cdot 4} + \frac{x^7}{3 \cdot 4 \cdot 6 \cdot 7} + \cdots\right]$$

## 五、偏微分方程

### 5.1 波动方程

**方程**：

**一维波动方程**：
$$\frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^2 u}{\partial x^2}$$

**解法**：分离变量

**假设**：
$$u(x,t) = X(x)T(t)$$

**代入**：
$$X(x)T''(t) = c^2 X''(x)T(t)$$

**分离**：
$$\frac{T''}{c^2 T} = \frac{X''}{X} = -\lambda^2$$

**解**：
$$X(x) = A \cos(\lambda x) + B \sin(\lambda x)$$
$$T(t) = C \cos(c\lambda t) + D \sin(c\lambda t)$$

**解**：
$$u(x,t) = [A \cos(\lambda x) + B \sin(\lambda x)][C \cos(c\lambda t) + D \sin(c\lambda t)]$$

---

### 5.2 热传导方程

**方程**：

**一维热传导**：
$$\frac{\partial u}{\partial t} = \alpha^2 \frac{\partial^2 u}{\partial x^2}$$

**解法**：分离变量

类似方法得到解

---

### 5.3 拉普拉斯方程

**方程**：

**二维拉普拉斯方程**：
$$\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0$$

**极坐标形式**：
$$\frac{\partial^2 u}{\partial r^2} + \frac{1}{r}\frac{\partial u}{\partial r} + \frac{1}{r^2}\frac{\partial^2 u}{\partial \theta^2} = 0$$

**解法**：分离变量

**解**：
$$u(r,\theta) = \sum_{n=0}^{\infty} (A_n r^n + B_n r^{-n})(C_n \cos n\theta + D_n \sin n\theta)$$

## 六、应用

### 6.1 物理应用

**力学**：

**简谐振动方程**：
$$m \frac{d^2 x}{dt^2} + kx = 0$$

**解**：
$$x(t) = A \cos(\omega t + \phi)$$

其中$\omega^2 = k/m$

**应用**：
- 弹簧系统
- 单摆
- 电路
