## 二、一阶微分方程

### 2.1 可分离变量方程

**形式**：

```text
方程：
dy/dx = f(x)g(y)

解法：
分离变量：
dy/g(y) = f(x)dx

积分：
∫dy/g(y) = ∫f(x)dx + C

例子：
dy/dx = xy

分离：
dy/y = x dx

积分：
ln|y| = x²/2 + C

解：
y = Ce^(x²/2)
```

---

### 2.2 齐次方程

**形式**：

```text
方程：
dy/dx = f(y/x)

方法：
令u = y/x，则y = ux

变换：
dy/dx = u + x du/dx = f(u)

分离：
du/(f(u) - u) = dx/x

积分求解
```

---

### 2.3 线性方程

**形式**：

```text
方程：
dy/dx + P(x)y = Q(x)

解法：积分因子法

积分因子：
μ(x) = e^(∫P(x)dx)

乘方程：
μ(x)dy/dx + μ(x)P(x)y = μ(x)Q(x)

左边是(μy)'：
d(μy)/dx = μQ

积分：
μy = ∫μQ dx + C

解：
y = (1/μ)(∫μQ dx + C)
```

---

### 2.4 伯努利方程

**形式**：

```text
方程：
dy/dx + P(x)y = Q(x)yⁿ

方法：
令z = y^(1-n)

变换为线性方程求解
```

## 三、高阶微分方程

### 3.1 线性齐次方程

**形式**：

```text
方程：
aₙy^(n) + a_{n-1}y^(n-1) + ... + a₁y' + a₀y = 0

特征方程：
aₙrⁿ + a_{n-1}r^(n-1) + ... + a₁r + a₀ = 0

特征根：
r₁, r₂, ..., rₙ

解：
- 单根：Ce^(rx)
- 重根：(C₁ + C₂x + ...)e^(rx)
- 复根：e^(αx)(C₁cos βx + C₂sin βx)
```

---

### 3.2 线性非齐次方程

**形式**：

```text
方程：
L[y] = f(x)

其中L是线性微分算子

解：
y = yₕ + yₚ

yₕ：齐次方程的通解
yₚ：非齐次方程的特解

求特解：
- 待定系数法
- 变分常数法
- 算子方法
```

## 四、级数解方法

### 4.1 幂级数解

**方法**：

```text
假设解：
y = ∑(n=0 to ∞) aₙxⁿ

代入方程：
得到系数递推关系

确定系数：
- a₀, a₁任意（初始条件）
- 其他由递推关系确定

例子：Airy方程
y'' - xy = 0

假设：
y = ∑aₙxⁿ

代入：
∑n(n-1)aₙx^(n-2) - x∑aₙxⁿ = 0

递推：
a_{n+2} = a_{n-1}/((n+2)(n+1))

解：
y = a₀[1 + x³/(2·3) + x⁶/(2·3·5·6) + ...]
  + a₁[x + x⁴/(3·4) + x⁷/(3·4·6·7) + ...]
```

---

### 4.2 特殊函数解

**产生特殊函数**：

```text
许多微分方程的解
不能用初等函数表示

产生特殊函数：
- 贝塞尔函数（Bessel方程）
- 勒让德函数（Legendre方程）
- 超几何函数
- 其他

欧拉的贡献：
- 早期研究
- 为后人铺路
```

## 五、偏微分方程

### 5.1 波动方程

**方程**：

```text
一维波动方程：
∂²u/∂t² = c²∂²u/∂x²

解法：分离变量

假设：
u(x,t) = X(x)T(t)

代入：
X(x)T''(t) = c²X''(x)T(t)

分离：
T''/(c²T) = X''/X = -λ²

解：
X(x) = A cos(λx) + B sin(λx)
T(t) = C cos(cλt) + D sin(cλt)

解：
u(x,t) = [A cos(λx) + B sin(λx)][C cos(cλt) + D sin(cλt)]
```

---

### 5.2 热传导方程

**方程**：

```text
一维热传导：
∂u/∂t = α²∂²u/∂x²

解法：分离变量

类似方法得到解
```

---

### 5.3 拉普拉斯方程

**方程**：

```text
二维：
∂²u/∂x² + ∂²u/∂y² = 0

极坐标：
∂²u/∂r² + (1/r)∂u/∂r + (1/r²)∂²u/∂θ² = 0

解法：分离变量

解：
u(r,θ) = ∑(Aₙrⁿ + Bₙr^(-n))(Cₙcos nθ + Dₙsin nθ)
```
