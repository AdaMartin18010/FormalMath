## 一、欧拉公式的发现

### 1.1 历史背景

**18世纪的数学问题**：

```text
核心问题：
- 复数的幂如何定义？
- 三角函数与指数函数的关系？
- 如何统一处理复分析？

已有的发现：
- Cardano（1545）：虚数发现
- De Moivre（1722）：(cos θ + i sin θ)ⁿ = cos(nθ) + i sin(nθ)
- 但仍缺乏统一的理论框架
```

**欧拉的工作**（1740s）：

```text
目标：
建立复分析的统一理论

方法：
- 复指数函数的级数展开
- 发现三角函数与指数的联系
- 建立统一公式
```

---

### 1.2 公式的推导

**欧拉的方法**（1748, Introductio）：

欧拉通过级数展开推导欧拉公式。

**数学表述**：

**步骤1：指数函数的级数展开**：
$$e^z = 1 + z + \frac{z^2}{2!} + \frac{z^3}{3!} + \frac{z^4}{4!} + \cdots = \sum_{n=0}^{\infty} \frac{z^n}{n!}$$

**步骤2：令 $z = ix$**：
$$e^{ix} = 1 + ix + \frac{(ix)^2}{2!} + \frac{(ix)^3}{3!} + \frac{(ix)^4}{4!} + \cdots$$

**步骤3：利用 $i^2 = -1$**：
$$e^{ix} = 1 + ix - \frac{x^2}{2!} - i\frac{x^3}{3!} + \frac{x^4}{4!} + i\frac{x^5}{5!} - \cdots$$

**步骤4：分离实部和虚部**：
$$e^{ix} = \left(1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots\right) + i\left(x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots\right)$$

**步骤5：识别三角函数**：
$$e^{ix} = \cos x + i \sin x$$

**数学公式**：

**级数展开**：
$$e^{ix} = \sum_{n=0}^{\infty} \frac{(ix)^n}{n!} = \sum_{n=0}^{\infty} \frac{i^n x^n}{n!}$$

**分离实虚部**：
$$e^{ix} = \sum_{k=0}^{\infty} \frac{(-1)^k x^{2k}}{(2k)!} + i\sum_{k=0}^{\infty} \frac{(-1)^k x^{2k+1}}{(2k+1)!}$$

**欧拉公式**：
$$e^{ix} = \cos x + i \sin x$$

**特殊情形**：
$$e^{i\pi} + 1 = 0$$

**例子**：

1. **$x = 0$**：$e^0 = \cos 0 + i \sin 0 = 1$
2. **$x = \pi$**：$e^{i\pi} = \cos \pi + i \sin \pi = -1$
3. **$x = \frac{\pi}{2}$**：$e^{i\pi/2} = \cos \frac{\pi}{2} + i \sin \frac{\pi}{2} = i$
4. **$x = 2\pi$**：$e^{2\pi i} = \cos 2\pi + i \sin 2\pi = 1$

**特殊情形**：

```text
令x = π：
e^(iπ) = cos π + i sin π = -1

因此：
e^(iπ) + 1 = 0

"最美丽的公式"：
连接了数学的5个基本常数：
- e：自然对数的底（分析学）
- i：虚数单位（代数）
- π：圆周率（几何）
- 1：乘法单位元（算术）
- 0：加法单位元（算术）
```

## 二、欧拉公式的数学意义

### 2.1 统一性意义

**连接不同数学分支**：

```text
分析学：
- 指数函数：e^x
- 级数展开
- 微积分

几何学：
- 三角函数：cos x, sin x
- 单位圆上的点
- 周期性

代数学：
- 复数：i
- 复平面
- 复数运算
```

**统一表达**：

```text
复数极坐标：
z = r(cos θ + i sin θ) = re^(iθ)

优势：
- 乘法：z₁z₂ = r₁r₂e^(i(θ₁+θ₂))
- 除法：z₁/z₂ = (r₁/r₂)e^(i(θ₁-θ₂))
- 幂运算：zⁿ = rⁿe^(inθ)

几何解释：
- 模长：r
- 幅角：θ
- 旋转：e^(iθ)表示旋转θ角
```

---

### 2.2 三角函数的指数表示

**指数形式的优势**：

```text
三角函数关系：
cos x = (e^(ix) + e^(-ix))/2
sin x = (e^(ix) - e^(-ix))/(2i)

好处：
- 简化恒等式证明
- 统一处理三角函数
- 便于微积分运算

例子：
sin²x + cos²x = 1

证明：
[(e^(ix) - e^(-ix))/(2i)]² + [(e^(ix) + e^(-ix))/2]²
= (e^(2ix) - 2 + e^(-2ix))/(-4) + (e^(2ix) + 2 + e^(-2ix))/4
= 1
```

---

### 2.3 复数的指数形式

**De Moivre定理的推广**：

```text
De Moivre（1722）：
(cos θ + i sin θ)ⁿ = cos(nθ) + i sin(nθ)

欧拉形式：
(e^(iθ))ⁿ = e^(inθ) = cos(nθ) + i sin(nθ)

优势：
- 更简洁
- 易于推广
- 统一理论
```

## 三、复分析的基础

### 3.1 复指数函数

**定义**：

```text
对于复数z = x + iy：
e^z = e^(x+iy) = e^x · e^(iy) = e^x(cos y + i sin y)

性质：
- e^(z₁+z₂) = e^z₁ · e^z₂
- |e^z| = e^x
- arg(e^z) = y
- e^(z+2πi) = e^z（周期性）
```

---

### 3.2 复对数函数

**定义**：

```text
对于z = re^(iθ)（r > 0, -π < θ ≤ π）：
ln z = ln r + iθ

多值性：
- 主值：Ln z = ln r + iArg z
- 一般值：ln z = Ln z + 2πik（k ∈ ℤ）

欧拉公式的应用：
z = e^w ⟺ w = ln z
```

---

### 3.3 复数幂

**定义**：

```text
对于z, w ∈ ℂ：
z^w = e^(w ln z)

例子：
i^i = e^(i ln i) = e^(i(ln 1 + iπ/2)) = e^(-π/2)

实数结果：
i^i ≈ 0.2078...
```

## 四、应用与推广

### 4.1 三角恒等式的简化

**加法公式**：

```text
传统方法：
需要几何构造或复杂计算

欧拉方法：
cos(α + β) + i sin(α + β) = e^(i(α+β))
= e^(iα) · e^(iβ)
= (cos α + i sin α)(cos β + i sin β)
= (cos α cos β - sin α sin β) + i(sin α cos β + cos α sin β)

实部：cos(α + β) = cos α cos β - sin α sin β
虚部：sin(α + β) = sin α cos β + cos α sin β
```

---

### 4.2 傅里叶级数的准备

**欧拉公式与周期函数**：

```text
任意周期函数f(x)可以表示为：
f(x) = ∑(n=-∞ to ∞) cₙe^(inx)

其中：
cₙ = (1/(2π))∫₀^(2π) f(x)e^(-inx) dx

基础：
- e^(inx)形成完备正交系
- 欧拉公式连接实数与复数
- 为傅里叶分析铺路
```

---

### 4.3 微分方程中的应用

**常微分方程**：

```text
方程：y'' + y = 0

特征方程：r² + 1 = 0
根：r = ±i

解：
y = C₁e^(ix) + C₂e^(-ix)
= C₁(cos x + i sin x) + C₂(cos x - i sin x)
= (C₁ + C₂)cos x + i(C₁ - C₂)sin x

实数解：
y = A cos x + B sin x
```

## 六、现代视角

### 6.1 公式的推广

**矩阵指数**：

```text
对于矩阵A：
e^A = I + A + A²/2! + A³/3! + ...

欧拉公式的矩阵版本：
e^(iA) = cos A + i sin A（对于某些矩阵A）

应用：
- 量子力学
- 李群理论
- 数值方法
```

### 6.2 拓扑学视角

**单位圆**：

e^(iθ)（θ ∈ [0, 2π)）：

- 为拓扑学提供例子

应用：

## 七、总结

### 7.1 核心要点

**欧拉公式**：

```text
公式：
e^(ix) = cos x + i sin x
e^(iπ) + 1 = 0

意义：
1. 统一分析、几何、代数
2. 建立复分析基础
3. 简化三角恒等式
4. 连接离散与连续

应用：
- 复分析
- 信号处理
- 量子力学
- 工程应用

影响：
- 18-19世纪复分析发展

```
