## 一、欧拉公式的发现

### 1.1 历史背景

**18世纪的数学问题**：

```text
核心问题：
- 复数的幂如何定义？
- 三角函数与指数函数的关系？
- 如何统一处理复分析？

已有的发现：
- Cardano（1545）：虚数发现
- De Moivre（1722）：(cos θ + i sin θ)ⁿ = cos(nθ) + i sin(nθ)
- 但仍缺乏统一的理论框架
```

**欧拉的工作**（1740s）：

```text
目标：
建立复分析的统一理论

方法：
- 复指数函数的级数展开
- 发现三角函数与指数的联系
- 建立统一公式
```

---

### 1.2 公式的推导

**欧拉的方法**（1748, Introductio）：

欧拉通过级数展开推导欧拉公式。

**数学表述**：

**步骤1：指数函数的级数展开**：
$$e^z = 1 + z + \frac{z^2}{2!} + \frac{z^3}{3!} + \frac{z^4}{4!} + \cdots = \sum_{n=0}^{\infty} \frac{z^n}{n!}$$

**步骤2：令 $z = ix$**：
$$e^{ix} = 1 + ix + \frac{(ix)^2}{2!} + \frac{(ix)^3}{3!} + \frac{(ix)^4}{4!} + \cdots$$

**步骤3：利用 $i^2 = -1$**：
$$e^{ix} = 1 + ix - \frac{x^2}{2!} - i\frac{x^3}{3!} + \frac{x^4}{4!} + i\frac{x^5}{5!} - \cdots$$

**步骤4：分离实部和虚部**：
$$e^{ix} = \left(1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots\right) + i\left(x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots\right)$$

**步骤5：识别三角函数**：
$$e^{ix} = \cos x + i \sin x$$

**数学公式**：

**级数展开**：
$$e^{ix} = \sum_{n=0}^{\infty} \frac{(ix)^n}{n!} = \sum_{n=0}^{\infty} \frac{i^n x^n}{n!}$$

**分离实虚部**：
$$e^{ix} = \sum_{k=0}^{\infty} \frac{(-1)^k x^{2k}}{(2k)!} + i\sum_{k=0}^{\infty} \frac{(-1)^k x^{2k+1}}{(2k+1)!}$$

**欧拉公式**：
$$e^{ix} = \cos x + i \sin x$$

**特殊情形**：
$$e^{i\pi} + 1 = 0$$

**例子**：

1. **$x = 0$**：$e^0 = \cos 0 + i \sin 0 = 1$
2. **$x = \pi$**：$e^{i\pi} = \cos \pi + i \sin \pi = -1$
3. **$x = \frac{\pi}{2}$**：$e^{i\pi/2} = \cos \frac{\pi}{2} + i \sin \frac{\pi}{2} = i$
4. **$x = 2\pi$**：$e^{2\pi i} = \cos 2\pi + i \sin 2\pi = 1$

**特殊情形**：

```text
令x = π：
e^(iπ) = cos π + i sin π = -1

因此：
e^(iπ) + 1 = 0

"最美丽的公式"：
连接了数学的5个基本常数：
- e：自然对数的底（分析学）
- i：虚数单位（代数）
- π：圆周率（几何）
- 1：乘法单位元（算术）
- 0：加法单位元（算术）
```

## 二、欧拉公式的数学意义

### 2.1 统一性意义

**连接不同数学分支**：

```text
分析学：
- 指数函数：e^x
- 级数展开
- 微积分

几何学：
- 三角函数：cos x, sin x
- 单位圆上的点
- 周期性

代数学：
- 复数：i
- 复平面
- 复数运算
```

**统一表达**：

```text
复数极坐标：
z = r(cos θ + i sin θ) = re^(iθ)

优势：
- 乘法：z₁z₂ = r₁r₂e^(i(θ₁+θ₂))
- 除法：z₁/z₂ = (r₁/r₂)e^(i(θ₁-θ₂))
- 幂运算：zⁿ = rⁿe^(inθ)

几何解释：
- 模长：r
- 幅角：θ
- 旋转：e^(iθ)表示旋转θ角
```

---

### 2.2 三角函数的指数表示

**指数形式的优势**：

```text
三角函数关系：
cos x = (e^(ix) + e^(-ix))/2
sin x = (e^(ix) - e^(-ix))/(2i)

好处：
- 简化恒等式证明
- 统一处理三角函数
- 便于微积分运算

例子：
sin²x + cos²x = 1

证明：
[(e^(ix) - e^(-ix))/(2i)]² + [(e^(ix) + e^(-ix))/2]²
= (e^(2ix) - 2 + e^(-2ix))/(-4) + (e^(2ix) + 2 + e^(-2ix))/4
= 1
```

---

### 2.3 复数的指数形式

**De Moivre定理的推广**：

```text
De Moivre（1722）：
(cos θ + i sin θ)ⁿ = cos(nθ) + i sin(nθ)

欧拉形式：
(e^(iθ))ⁿ = e^(inθ) = cos(nθ) + i sin(nθ)

优势：
- 更简洁
- 易于推广
- 统一理论
```

## 三、复分析的基础

### 3.1 复指数函数

**定义**：

```text
对于复数z = x + iy：
e^z = e^(x+iy) = e^x · e^(iy) = e^x(cos y + i sin y)

性质：
- e^(z₁+z₂) = e^z₁ · e^z₂
- |e^z| = e^x
- arg(e^z) = y
- e^(z+2πi) = e^z（周期性）
```

---

### 3.2 复对数函数

**定义**：

```text
对于z = re^(iθ)（r > 0, -π < θ ≤ π）：
ln z = ln r + iθ

多值性：
- 主值：Ln z = ln r + iArg z
- 一般值：ln z = Ln z + 2πik（k ∈ ℤ）

欧拉公式的应用：
z = e^w ⟺ w = ln z
```

---

### 3.3 复数幂

**定义**：

```text
对于z, w ∈ ℂ：
z^w = e^(w ln z)

例子：
i^i = e^(i ln i) = e^(i(ln 1 + iπ/2)) = e^(-π/2)

实数结果：
i^i ≈ 0.2078...
```

## 四、应用与推广

### 4.1 三角恒等式的简化

**加法公式**：

欧拉公式极大地简化了三角恒等式的推导。

**传统方法**：

传统方法需要几何构造或复杂计算。

**欧拉方法**：

使用欧拉公式，三角恒等式变得简单：
$$\cos(\alpha + \beta) + i \sin(\alpha + \beta) = e^{i(\alpha+\beta)} = e^{i\alpha} \cdot e^{i\beta}$$

展开：
$$= (\cos \alpha + i \sin \alpha)(\cos \beta + i \sin \beta)$$
$$= (\cos \alpha \cos \beta - \sin \alpha \sin \beta) + i(\sin \alpha \cos \beta + \cos \alpha \sin \beta)$$

**数学表述**：

实部：$$\cos(\alpha + \beta) = \cos \alpha \cos \beta - \sin \alpha \sin \beta$$

虚部：$$\sin(\alpha + \beta) = \sin \alpha \cos \beta + \cos \alpha \sin \beta$$

**例子6：三角恒等式的应用**：

使用欧拉公式可以轻松推导所有三角恒等式。

---

### 4.2 傅里叶级数的准备

**欧拉公式与周期函数**：

欧拉公式为傅里叶级数提供了基础。

**傅里叶级数**：

任意周期函数 $f(x)$ 可以表示为：
$$f(x) = \sum_{n=-\infty}^{\infty} c_n e^{inx}$$

其中：
$$c_n = \frac{1}{2\pi} \int_0^{2\pi} f(x) e^{-inx} dx$$

**数学表述**：

傅里叶级数：
$$f(x) = \sum_{n=-\infty}^{\infty} c_n e^{inx}$$

Fourier系数：
$$c_n = \frac{1}{2\pi} \int_0^{2\pi} f(x) e^{-inx} dx$$

**例子7：傅里叶级数的应用**：

欧拉公式为傅里叶分析铺平了道路。

**基础**：

- **完备正交系**：$e^{inx}$ 形成完备正交系
- **连接实数与复数**：欧拉公式连接实数与复数
- **为傅里叶分析铺路**：为傅里叶分析提供了基础

### 4.3 微分方程中的应用

**常微分方程**：

欧拉公式在微分方程中有重要应用。

**常微分方程**：

对于方程 $y'' + y = 0$：

特征方程：$$r^2 + 1 = 0$$

根：$$r = \pm i$$

解：
$$y = C_1 e^{ix} + C_2 e^{-ix}$$

使用欧拉公式：
$$= C_1(\cos x + i \sin x) + C_2(\cos x - i \sin x)$$
$$= (C_1 + C_2)\cos x + i(C_1 - C_2)\sin x$$

**数学表述**：

实数解：
$$y = A \cos x + B \sin x$$

**例子8：微分方程的应用**：

欧拉公式在微分方程求解中有重要应用。

## 九、数学公式总结

### 核心公式

1. **指数函数的级数展开**：
   $$e^z = \sum_{n=0}^{\infty} \frac{z^n}{n!} = 1 + z + \frac{z^2}{2!} + \frac{z^3}{3!} + \cdots$$

2. **复指数函数的级数展开**：
   $$e^{ix} = \sum_{n=0}^{\infty} \frac{(ix)^n}{n!} = \sum_{n=0}^{\infty} \frac{i^n x^n}{n!}$$

3. **复指数函数的实虚部分离**：
   $$e^{ix} = \sum_{k=0}^{\infty} \frac{(-1)^k x^{2k}}{(2k)!} + i\sum_{k=0}^{\infty} \frac{(-1)^k x^{2k+1}}{(2k+1)!}$$

4. **欧拉公式**（基本形式）：
   $$e^{ix} = \cos x + i \sin x$$

5. **欧拉恒等式**（最美丽的公式）：
   $$e^{i\pi} + 1 = 0$$

6. **复数的极坐标形式**：
   $$z = r(\cos \theta + i \sin \theta) = re^{i\theta}$$

7. **复数的乘法**：
   $$z_1 z_2 = r_1 r_2 e^{i(\theta_1 + \theta_2)}$$

8. **复数的除法**：
   $$\frac{z_1}{z_2} = \frac{r_1}{r_2} e^{i(\theta_1 - \theta_2)}$$

9. **复数的幂运算**：
   $$z^n = r^n e^{in\theta}$$

10. **三角函数的指数表示**：
    $$\cos x = \frac{e^{ix} + e^{-ix}}{2}$$

11. **正弦函数的指数表示**：
    $$\sin x = \frac{e^{ix} - e^{-ix}}{2i}$$

12. **复指数函数的性质**：
    $$e^{z_1 + z_2} = e^{z_1} \cdot e^{z_2}$$

13. **复指数函数的模**：
    $$|e^z| = e^x$$

14. **复指数函数的周期性**：
    $$e^{z + 2\pi i} = e^z$$

15. **复对数函数**：
    $$\ln z = \ln r + i\theta$$

16. **复数幂的定义**：
    $$z^w = e^{w \ln z}$$

17. **De Moivre定理的欧拉形式**：
    $$(e^{i\theta})^n = e^{in\theta} = \cos(n\theta) + i \sin(n\theta)$$

18. **傅里叶级数的复数形式**：
    $$f(x) = \sum_{n=-\infty}^{\infty} c_n e^{inx}$$
