## 二、球面三角学基础

### 2.1 球面与大圆

**球面**：

```text
定义：
半径为R的球面：
S² = {(x,y,z) : x² + y² + z² = R²}

性质：
- 二维流形
- 常曲率1/R²
- 对称性
```

**大圆**：

```text
定义：
球面上与球心共面的圆

性质：
1. 是球面上两点间的最短路径
2. 是测地线
3. 类似平面上的直线

例子：
- 经线都是大圆
- 赤道是大圆
- 任意通过球心的平面与球面的交线
```

---

### 2.2 球面距离

**定义**：

```text
球面上两点P, Q的距离：
d(P, Q) = R · θ

其中θ是从球心看两点的夹角

公式：
cos θ = (P · Q) / (|P| · |Q|)
     = (P · Q) / R²

（在单位球面上，R = 1）
```

**例子**：

```text
单位球面上：
- 北极点N = (0, 0, 1)
- 赤道上一点P = (1, 0, 0)
- 夹角：θ = π/2
- 距离：d = π/2
```

## 三、球面三角形

### 3.1 定义

**球面三角形**：

```text
定义：
由三条大圆弧围成的区域

顶点：
三个大圆的交点

边：
大圆弧段

角：
两个大圆弧在顶点的夹角
```

**性质**：

```text
性质1：
球面三角形的内角和 > π

性质2：
面积与角度盈余相关

性质3：
边长与角度相关
```

---

### 3.2 球面三角公式

**余弦定理**：

```text
球面余弦定理：
cos c = cos a cos b + sin a sin b cos C

其中：
- a, b, c是边长（弧度）
- C是边a, b之间的角

类比：
平面余弦定理：
c² = a² + b² - 2ab cos C
```

**正弦定理**：

```text
球面正弦定理：
sin A / sin a = sin B / sin b = sin C / sin c

其中：
- A, B, C是角
- a, b, c是对边

类比：
平面正弦定理：
a / sin A = b / sin B = c / sin C
```

**面积公式（Girard定理）**：

```text
球面三角形面积：
Area = R² (A + B + C - π)

其中：
- A, B, C是内角（弧度）
- R是球面半径

意义：
- 面积与角度盈余成正比
- 角度盈余 = A + B + C - π
```

---

### 3.3 例子

**直角球面三角形**：

```text
例子：
球面上三个大圆弧围成直角三角形
- 一个角 = π/2
- 其他两个角待求

利用球面三角公式：
- 余弦定理
- 正弦定理
- 可以计算所有量
```

## 四、球面坐标

### 4.1 球坐标系

**定义**：

```text
球坐标(r, θ, φ)：
- r：到原点的距离
- θ：方位角（0到2π）
- φ：极角（0到π）

转换：
x = r sin φ cos θ
y = r sin φ sin θ
z = r cos φ
```

**欧拉的贡献**：

```text
欧拉系统化球坐标：
- 建立标准记号
- 发展变换公式
- 应用计算
```

---

### 4.2 坐标变换

**与直角坐标的转换**：

```text
球坐标 → 直角坐标：
x = r sin φ cos θ
y = r sin φ sin θ
z = r cos φ

直角坐标 → 球坐标：
r = √(x² + y² + z²)
θ = arctan(y/x)
φ = arccos(z/r)
```

**应用**：

```text
应用：
1. 物理：量子力学、电磁学
2. 天文学：天体坐标
3. 地理学：地球坐标
4. 工程：导航系统
```
