## 一、历史背景：经典几何问题

### 1.1 历史问题

**Dido问题（传说）**：

```text
传说：
迦太基女王Dido用一张牛皮围出最大面积

方法：
- 将牛皮切成细条
- 围成半圆形
- 得到最大面积

问题：
为什么是半圆？如何证明？
```

**经典问题**：

```text
问题：
在所有周长相等的平面图形中，
哪个面积最大？

等价：
在所有面积相等的平面图形中，
哪个周长最小？

答案（直觉）：
圆
```

## 二、等周问题的陈述

### 2.1 问题表述

**等周问题**：

```text
问题：
在所有周长为L的简单闭曲线中，
哪个围成的面积最大？

数学表述：
给定L > 0，求
max Area(γ)
subject to Length(γ) = L

其中γ是简单闭曲线
```

**等周不等式**：

```text
定理（等周不等式）：
对于任意简单闭曲线γ，有
Area(γ) ≤ L²/(4π)

等号成立当且仅当γ是圆

等价：
在所有面积为A的简单闭曲线中，
周长最小的是圆，且
Length(γ) ≥ 2√(πA)
```

---

### 2.2 例子

**圆的情况**：

```text
圆：
- 半径：r
- 周长：L = 2πr
- 面积：A = πr²

关系：
A = πr² = π(L/(2π))² = L²/(4π)

因此：
A = L²/(4π)
满足等号
```

**其他图形**：

```text
正方形（周长L）：
- 边长：L/4
- 面积：A = (L/4)² = L²/16

比较：
L²/16 < L²/(4π) ≈ L²/12.57

因此：
正方形面积 < 圆面积
```

## 三、等周定理

### 3.1 定理陈述

**等周定理**：

```text
定理：
在所有周长为L的简单闭曲线中，
圆围成的面积最大，且最大值为
A_max = L²/(4π)

等价：
在所有面积为A的简单闭曲线中，
圆具有最小周长，且最小值为
L_min = 2√(πA)
```

**等周比**：

```text
定义：
等周比 = 4πA / L²

性质：
- 0 < 等周比 ≤ 1
- 等周比 = 1 ⟺ 曲线是圆
- 等周比衡量"圆度"
```

---

### 3.2 几何意义

**直观理解**：

```text
意义：
- 圆是"最紧凑"的图形
- 给定周长，圆围成最大面积
- 给定面积，圆具有最小周长

应用：
- 自然现象：肥皂泡、水滴
- 工程：最优设计
- 物理：最小能量原理
```

## 四、欧拉的变分法方法

### 4.1 变分法框架

**欧拉的方法（1744）**：

```text
方法：
1. 将曲线参数化
2. 建立面积和长度的积分表达式
3. 使用变分法求极值
4. 得到欧拉-拉格朗日方程
5. 求解得到圆

意义：
- 开创变分法
- 系统化方法
- 影响深远
```

---

### 4.2 变分法推导

**参数化**：

```text
设曲线参数化为：
(x(t), y(t)), t ∈ [0, 2π]

周长：
L = ∫₀²π √(x'² + y'²) dt

面积（Green公式）：
A = (1/2) ∫₀²π (xy' - yx') dt
```

**变分问题**：

```text
问题：
在约束L = 常数下，最大化A

方法：
- 拉格朗日乘数法
- 变分法
- 欧拉-拉格朗日方程
```

**欧拉-拉格朗日方程**：

```text
从变分法得到：
d/dt (∂L/∂x') - ∂L/∂x = 0
d/dt (∂L/∂y') - ∂L/∂y = 0

其中L是拉格朗日函数

解：
得到圆的参数方程
```

## 五、几何证明

### 5.1 Steiner对称化

**Steiner方法（1838）**：

```text
方法：
1. 对任意曲线，进行对称化
2. 保持周长不变
3. 增加面积
4. 重复得到圆

意义：
- 纯几何方法
- 直观
- 严格
```

**对称化过程**：

```text
步骤：
1. 选择一条直线
2. 将曲线关于该直线对称化
3. 得到新曲线，周长不变，面积增加
4. 重复对所有方向

极限：
得到圆
```

---

### 5.2 其他几何方法

**等周不等式证明**：

```text
方法1：Fourier级数
- 将曲线展开为Fourier级数
- 利用Parseval恒等式
- 得到等周不等式

方法2：Brunn-Minkowski不等式
- 凸几何方法
- 现代方法
- 推广到高维
```

## 六、推广与变形

### 6.1 高维推广

**三维等周问题**：

```text
问题：
在所有表面积为S的闭曲面中，
哪个围成的体积最大？

答案：
球

等周不等式：
V ≤ S^(3/2)/(6√π)

等号成立当且仅当是球
```

**n维推广**：

```text
n维球：
- 在所有表面积为S的n维闭曲面中
- 球围成最大体积

等周不等式：
V ≤ C_n S^(n/(n-1))

其中C_n是常数
```

---

### 6.2 其他约束

**固定直径**：

```text
问题：
在所有直径为d的图形中，
哪个面积最大？

答案：
Reuleaux三角形（不是圆）
```

**多边形等周问题**：

```text
问题：
在所有n边形中（固定周长），
哪个面积最大？

答案：
正n边形
```
