## 二、最小作用原理

### 2.1 原理的陈述

**Maupertuis原理（1744）**：

```text
原理：
自然系统沿使"作用量"最小的路径运动

作用量：
S = ∫mv ds = ∫2T dt

其中：
- m：质量
- v：速度
- T：动能

意义：
- 统一原理
- 变分法应用
- 物理基础
```

---

### 2.2 欧拉的贡献

**欧拉的工作（1744）**：

```text
3. 应用于力学
4. 建立理论

方法：

- 力学应用
```

## 三、拉格朗日方程

### 3.1 拉格朗日函数

**定义**：

```text
定义：
拉格朗日函数：
L = T - V

其中：
- T：动能
- V：势能

作用量：
S = ∫L dt

极值条件：
δS = 0
```

---

### 3.2 拉格朗日方程

**方程推导**：

```text
从变分法：
d/dt(∂L/∂q̇ᵢ) - ∂L/∂qᵢ = 0

其中：
- qᵢ：广义坐标
- q̇ᵢ：广义速度

这是拉格朗日方程

意义：
- 统一描述力学系统
- 坐标无关
- 约束自然处理
```

---

### 3.3 与Newton力学的对比

**优势**：

```text
拉格朗日方法：
- 坐标无关
- 约束自然处理
- 对称性明显
- 推广容易

Newton方法：
- 几何直观
- 矢量分析
- 力的概念
- 坐标依赖
```

## 四、刚体动力学

### 4.1 刚体的运动

**欧拉的工作（1765）**：

```text
著作：

(Theoria motus corporum solidorum...)

贡献：
1. 刚体运动方程
2. 角动量理论
3. 旋转动力学
4. 欧拉方程
```

---

### 4.2 欧拉动力学方程

**方程陈述**：

```text
刚体绕固定点的运动：
I₁ω̇₁ - (I₂ - I₃)ω₂ω₃ = M₁
I₂ω̇₂ - (I₃ - I₁)ω₃ω₁ = M₂
I₃ω̇₃ - (I₁ - I₂)ω₁ω₂ = M₃

其中：
- Iᵢ：主转动惯量
- ωᵢ：角速度分量
- Mᵢ：力矩分量

这是欧拉动力学方程
```

---

### 4.3 自由旋转

**无外力情况**：

```text
自由刚体（M = 0）：
I₁ω̇₁ = (I₂ - I₃)ω₂ω₃
I₂ω̇₂ = (I₃ - I₁)ω₃ω₁
I₃ω̇₃ = (I₁ - I₂)ω₁ω₂

性质：
- 角动量守恒
- 能量守恒
- 复杂运动

应用：
- 陀螺运动
- 卫星姿态
- 天体运动
```

## 五、欧拉角与旋转

### 5.1 欧拉角

**定义**：

```text
定义：
欧拉角(φ, θ, ψ)描述刚体在空间中的取向

旋转顺序：
1. 绕z轴旋转φ（进动角）
2. 绕新x轴旋转θ（章动角）
3. 绕新z轴旋转ψ（自转角）

应用：
- 描述刚体取向
- 旋转矩阵
- 运动学
```

---

### 5.2 旋转矩阵

**表示**：

```text
旋转矩阵：
R(φ, θ, ψ) = R_z(ψ) R_x(θ) R_z(φ)

性质：
- 正交矩阵：R^T R = I
- 行列式：det R = 1
- 群结构：SO(3)

应用：
- 坐标变换
- 刚体运动
- 计算机图形学
```

---

### 5.3 角速度

**角速度向量**：

```text
定义：
角速度ω = (ω₁, ω₂, ω₃)

与欧拉角的关系：
ω₁ = φ̇ sin θ sin ψ + θ̇ cos ψ
ω₂ = φ̇ sin θ cos ψ - θ̇ sin ψ
ω₃ = φ̇ cos θ + ψ̇

意义：
- 描述旋转
- 动力学方程
- 应用广泛
```
