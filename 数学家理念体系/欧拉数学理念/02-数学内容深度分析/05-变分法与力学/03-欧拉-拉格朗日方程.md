## 二、方程的基本形式

### 2.1 基本变分问题

**问题陈述**：

变分问题的基本形式。

**数学表述**：

**问题**：

求函数 $y(x)$，使得泛函：
$$J[y] = \int_a^b F(x, y, y') dx$$
达到极值（最大或最小）。

**数学公式**：

**变分问题**：
$$\min_{y} J[y] = \int_a^b F(x, y, y') dx$$
其中：

- $F$ 是给定的函数（拉格朗日函数）
- $y(a) = y_0$，$y(b) = y_1$（边界条件）
- $y' = \frac{dy}{dx}$

**例子**：

1. **最短路径**：$F = \sqrt{1 + (y')^2}$，$J[y] = \int_a^b \sqrt{1 + (y')^2} dx$
2. **最速降线**：$F = \sqrt{\frac{1 + (y')^2}{2gy}}$，$J[y] = \int_a^b \sqrt{\frac{1 + (y')^2}{2gy}} dx$
3. **等周问题**：$F = y$，约束条件 $\int_a^b \sqrt{1 + (y')^2} dx = L$

---

### 2.2 欧拉-拉格朗日方程

**方程陈述**：

欧拉-拉格朗日方程是变分法的核心方程。

**数学表述**：

**定理（欧拉-拉格朗日方程）**：

如果 $y(x)$ 使泛函 $J[y]$ 达到极值，则 $y$ 满足：
$$\frac{\partial F}{\partial y} - \frac{d}{dx}\left(\frac{\partial F}{\partial y'}\right) = 0$$
这是变分问题的必要条件。

**数学公式**：

**欧拉-拉格朗日方程**：
$$\frac{\partial F}{\partial y} - \frac{d}{dx}\left(\frac{\partial F}{\partial y'}\right) = 0$$

**展开形式**：
$$\frac{\partial F}{\partial y} - \frac{\partial^2 F}{\partial x \partial y'} - y'\frac{\partial^2 F}{\partial y \partial y'} - y''\frac{\partial^2 F}{\partial y'^2} = 0$$

**特殊情况**：

如果 $F$ 不显式依赖于 $x$，则：
$$\frac{\partial F}{\partial y'} y' - F = \text{常数}$$

**例子**：

1. **最短路径**：$F = \sqrt{1 + (y')^2}$，欧拉-拉格朗日方程给出 $y'' = 0$，即直线
2. **最速降线**：$F = \sqrt{\frac{1 + (y')^2}{2gy}}$，解是摆线
3. **等周问题**：使用拉格朗日乘数法，解是圆

## 三、推导过程

### 3.1 变分方法

**变分的引入**：

```text
考虑变分：
y_ε(x) = y(x) + εη(x)

其中：
- η(x)是任意函数
- η(a) = η(b) = 0（边界条件）
- ε是小参数

泛函：
J[y_ε] = ∫[a to b] F(x, y_ε, y_ε') dx
```

---

### 3.2 极值条件

**必要条件**：

```text
极值条件：
d/dε|_{ε=0} J[y_ε] = 0

计算：
d/dε J[y_ε] = ∫[a to b] [∂F/∂y · η + ∂F/∂y' · η'] dx

分部积分：
= ∫[a to b] [∂F/∂y - d/dx(∂F/∂y')] η dx
+ [∂F/∂y' · η]|_a^b

由于η(a) = η(b) = 0，边界项为0

因此：
∫[a to b] [∂F/∂y - d/dx(∂F/∂y')] η dx = 0

对任意η，得到：
∂F/∂y - d/dx(∂F/∂y') = 0
```

## 四、特殊情况

### 4.1 F不显式依赖x

**Beltrami恒等式**：

```text
情况：
F = F(y, y')（不依赖x）

结果：
存在首次积分：
F - y'(∂F/∂y') = 常数

证明：
d/dx(F - y'∂F/∂y')
= ∂F/∂y · y' + ∂F/∂y' · y''
  - y''∂F/∂y' - y'd/dx(∂F/∂y')
= y'[∂F/∂y - d/dx(∂F/∂y')]
= 0（由欧拉方程）

因此：
F - y'∂F/∂y' = 常数
```

**应用**：

```text
例子：
最速降线问题：
F = √((1+y'²)/(-y))

首次积分：
√((1+y'²)/(-y)) - y'²/√((1+y'²)(-y)) = 常数

简化：
y(1 + y'²) = 常数
```

---

### 4.2 F不显式依赖y

**简化情况**：

```text
情况：
F = F(x, y')（不依赖y）

结果：
∂F/∂y' = 常数

证明：
欧拉方程：
0 - d/dx(∂F/∂y') = 0

因此：
∂F/∂y' = 常数

例子：
最短路径问题：
F = √(1+y'²)
y'/√(1+y'²) = 常数
→ y' = 常数
→ y = 直线
```

---

### 4.3 F不显式依赖y'

**退化情况**：

```text
情况：
F = F(x, y)（不依赖y'）

结果：
∂F/∂y = 0

证明：
欧拉方程：
∂F/∂y - 0 = 0

因此：
∂F/∂y = 0

这是代数方程，不是微分方程
```

## 五、多变量推广

### 5.1 多函数情况

**多个函数**：

```text
问题：
求函数y₁(x), ..., yₙ(x)，使
J[y₁, ..., yₙ] = ∫ F(x, y₁, ..., yₙ, y₁', ..., yₙ') dx

极值

欧拉-拉格朗日方程：
对每个i：
∂F/∂yᵢ - d/dx(∂F/∂yᵢ') = 0
```

---

### 5.2 高维情况

**多变量函数**：

```text
问题：
求函数u(x₁, ..., xₙ)，使
J[u] = ∫...∫ F(x, u, ∂u/∂x₁, ..., ∂u/∂xₙ) dx₁...dxₙ

极值

欧拉-拉格朗日方程：
∂F/∂u - ∑ᵢ d/dxᵢ(∂F/∂(∂u/∂xᵢ)) = 0
```

---

### 5.3 高阶导数

**高阶导数情况**：

```text
问题：
求函数y(x)，使
J[y] = ∫ F(x, y, y', y'', ..., y⁽ⁿ⁾) dx

极值

欧拉-拉格朗日方程：
∑(k=0 to n) (-1)^k d^k/dx^k (∂F/∂y⁽ᵏ⁾) = 0
```

### 8.1 约束问题

**拉格朗日乘数法**：

```text
问题：
在约束g(x,y) = 0下，求J[y]的极值

方法：
引入拉格朗日乘数λ：
J*[y] = ∫(F + λg) dx

欧拉方程：
∂(F+λg)/∂y - d/dx(∂(F+λg)/∂y') = 0
```

## 十、数学公式总结

### 核心公式

1. **变分问题的泛函**：
   $$J[y] = \int_a^b F(x, y, y') dx$$

2. **变分问题的极值形式**：
   $$\min_{y} J[y] = \int_a^b F(x, y, y') dx$$

3. **欧拉-拉格朗日方程（基本形式）**：
   $$\frac{\partial F}{\partial y} - \frac{d}{dx}\left(\frac{\partial F}{\partial y'}\right) = 0$$

4. **欧拉-拉格朗日方程（展开形式）**：
   $$\frac{\partial F}{\partial y} - \frac{\partial^2 F}{\partial x \partial y'} - y'\frac{\partial^2 F}{\partial y \partial y'} - y''\frac{\partial^2 F}{\partial y'^2} = 0$$

5. **Beltrami恒等式**（$F$不显式依赖$x$）：
   $$\frac{\partial F}{\partial y'} y' - F = \text{常数}$$

6. **最短路径问题的拉格朗日函数**：
   $$F = \sqrt{1 + (y')^2}$$

7. **最短路径问题的泛函**：
   $$L[y] = \int_a^b \sqrt{1 + (y')^2} dx$$

8. **最速降线问题的拉格朗日函数**：
   $$F = \sqrt{\frac{1 + (y')^2}{2gy}}$$

9. **最速降线问题的泛函**：
   $$T[y] = \int_a^b \sqrt{\frac{1 + (y')^2}{2gy}} dx$$

10. **最速降线的首次积分**：
    $$y(1 + y'^2) = \text{常数}$$

11. **多函数情况的欧拉-拉格朗日方程**：
    $$\frac{\partial F}{\partial y_i} - \frac{d}{dx}\left(\frac{\partial F}{\partial y_i'}\right) = 0, \quad i = 1, 2, \ldots, n$$

12. **高维情况的欧拉-拉格朗日方程**：
    $$\frac{\partial F}{\partial u} - \sum_{i=1}^n \frac{d}{dx_i}\left(\frac{\partial F}{\partial (\partial u/\partial x_i)}\right) = 0$$

13. **高阶导数情况的欧拉-拉格朗日方程**：
    $$\sum_{k=0}^n (-1)^k \frac{d^k}{dx^k}\left(\frac{\partial F}{\partial y^{(k)}}\right) = 0$$

14. **约束问题的拉格朗日乘数法**：
    $$J^*[y] = \int_a^b (F + \lambda g) dx$$

15. **约束问题的欧拉-拉格朗日方程**：
    $$\frac{\partial (F+\lambda g)}{\partial y} - \frac{d}{dx}\left(\frac{\partial (F+\lambda g)}{\partial y'}\right) = 0$$

16. **变分方法中的变分函数**：
    $$y_\varepsilon(x) = y(x) + \varepsilon \eta(x)$$

17. **极值条件**：
    $$\left.\frac{d}{d\varepsilon}\right|_{\varepsilon=0} J[y_\varepsilon] = 0$$

18. **等周问题的约束条件**：
    $$\int_a^b \sqrt{1 + (y')^2} dx = L$$
