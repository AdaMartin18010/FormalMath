## 一、历史背景：从最速降线到变分法

### 1.1 最速降线问题

**Johann Bernoulli的问题（1696）**：

```text
问题：
在重力作用下，从点A到点B（A高于B），
什么形状的曲线使质点滑行时间最短？

挑战：
- 不是直线（虽然距离最短）
- 需要"加速"和"最短路径"的平衡
- 需要新的数学方法

答案：
摆线（cycloid）
```

**历史意义**：

```text
意义：
- 开创变分法
- 需要新的数学工具
- 连接几何与物理
- 影响深远
```

## 二、变分问题的陈述

### 2.1 变分问题

**一般变分问题**：

变分法是研究泛函极值的方法。

**数学表述**：

**问题**：

求函数 $y(x)$，使得泛函：
$$J[y] = \int_a^b F(x, y, y') dx$$
达到极值（最大或最小）。

**数学公式**：

**变分问题**：
$$\min_{y} J[y] = \int_a^b F(x, y, y') dx$$
其中：

- $F$ 是给定的函数
- $y(a) = y_0$，$y(b) = y_1$（边界条件）
- $y' = \frac{dy}{dx}$

**例子**：

1. **最速降线问题**：$F = \sqrt{\frac{1 + (y')^2}{2gy}}$，$J[y] = \int_a^b \sqrt{\frac{1 + (y')^2}{2gy}} dx$
2. **最短路径问题**：$F = \sqrt{1 + (y')^2}$，$J[y] = \int_a^b \sqrt{1 + (y')^2} dx$
3. **等周问题**：$F = y$，约束条件 $\int_a^b \sqrt{1 + (y')^2} dx = L$

---

### 2.2 泛函

**泛函的定义**：

```text
定义：
泛函是从函数空间到实数的映射

例子：
1. 长度泛函：
   L[y] = ∫√(1+y'²) dx

2. 面积泛函：
   A[y] = ∫y dx

3. 能量泛函：
   E[y] = ∫(y'² - V(y)) dx
```

## 三、欧拉-拉格朗日方程

### 3.1 欧拉方程

**欧拉方程（1744）**：

```text
定理：
如果y(x)使泛函J[y]达到极值，则y满足：

∂F/∂y - d/dx(∂F/∂y') = 0

这是变分法的基本方程

推导：
1. 考虑变分y(x) + εη(x)
2. 计算J[y + εη]
3. 对ε求导，令ε=0
4. 利用分部积分
5. 得到欧拉方程
```

---

### 3.2 特殊情况

**F不显式依赖x**：

```text
情况：
F = F(y, y')（不依赖x）

结果：
存在首次积分：
F - y'(∂F/∂y') = 常数

意义：
- 简化求解
- 物理意义（能量守恒）
- 应用广泛
```

**F不显式依赖y**：

当 $F$ 不显式依赖 $y$ 时，有特殊结果。

**情况**：

$F = F(x, y')$（不依赖 $y$）

**结果**：

$$\frac{\partial F}{\partial y'} = \text{常数}$$

**数学表述**：

当 $F = F(x, y')$ 时：
$$\frac{\partial F}{\partial y'} = \text{常数}$$

**例子6：最短路径问题**：

最短路径问题：
$$F = \sqrt{1+y'^2}$$

因此：
$$\frac{y'}{\sqrt{1+y'^2}} = \text{常数}$$

这给出：
$$y' = \text{常数} \Rightarrow y = \text{直线}$$

**例子7：F不显式依赖y的应用**：

这种情况在变分法中有重要应用。

## 四、变分法的基本方法

### 4.1 变分

**变分的定义**：

变分是变分法中的基本概念。

**定义**：

函数 $y(x)$ 的变分 $\delta y$ 是满足边界条件的任意函数 $\eta(x)$。

**数学表述**：

变分：
$$\delta y = \eta(x)$$

其中 $\eta(x)$ 满足边界条件。

**性质**：

- **边界条件**：$\delta y(a) = \delta y(b) = 0$
- **导数**：$\delta y' = (\delta y)'$
- **线性性**：$\delta(y_1 + y_2) = \delta y_1 + \delta y_2$

**例子8：变分的应用**：

变分在变分法中有重要应用。

**泛函的变分**：

泛函的变分是变分法中的基本概念。

**定义**：

泛函 $J[y]$ 的变分：
$$\delta J = \left.\frac{d}{d\varepsilon}\right|_{\varepsilon=0} J[y + \varepsilon\delta y]$$

**数学表述**：

泛函的变分：
$$\delta J = \left.\frac{d}{d\varepsilon}\right|_{\varepsilon=0} J[y + \varepsilon\delta y]$$

**极值条件**：

$$\delta J = 0 \quad \text{（对所有变分 $\delta y$）}$$

**例子9：泛函变分的应用**：

泛函的变分在变分法中有重要应用。

---

### 4.2 变分法步骤

**求解步骤**：

变分法求解有标准步骤。

**步骤1：建立泛函**：

$$J[y] = \int F(x, y, y') dx$$

**步骤2：写出欧拉方程**：

$$\frac{\partial F}{\partial y} - \frac{d}{dx}\left(\frac{\partial F}{\partial y'}\right) = 0$$

**步骤3：展开导数**：

$$\frac{\partial F}{\partial y} - \frac{\partial^2 F}{\partial x \partial y'} - y'\frac{\partial^2 F}{\partial y \partial y'} - y''\frac{\partial^2 F}{\partial y'^2} = 0$$

**步骤4：求解微分方程**：

得到 $y(x)$。

**步骤5：利用边界条件**：

确定常数。

**数学表述**：

求解步骤：
1. 建立泛函：$J[y] = \int F(x, y, y') dx$
2. 写出欧拉方程：$\frac{\partial F}{\partial y} - \frac{d}{dx}\left(\frac{\partial F}{\partial y'}\right) = 0$
3. 展开导数：得到二阶微分方程
4. 求解微分方程：得到 $y(x)$
5. 利用边界条件：确定常数

**例子10：求解步骤的应用**：

求解步骤在变分法中有重要应用。

## 五、应用：经典问题

### 5.1 最速降线问题

**问题陈述**：

最速降线问题是变分法的经典问题。

**问题**：

从 $(0,0)$ 到 $(a,b)$，在重力作用下，什么曲线使滑行时间最短？

**数学表述**：

最速降线问题：
$$\min T[y] = \int_0^a \sqrt{\frac{1+y'^2}{2gy}} dx$$

**泛函**：

$$T[y] = \int_0^a \sqrt{\frac{1+y'^2}{2gy}} dx$$

**欧拉方程**：

经过计算得到：
$$y(1+y'^2) = \text{常数}$$

**解**：

摆线参数方程：
$$x = R(\theta - \sin \theta)$$
$$y = R(1 - \cos \theta)$$

**例子11：最速降线问题的应用**：

最速降线问题在变分法中有重要应用。

## 七、现代发展

### 7.1 约束变分问题

**拉格朗日乘数法**：

```text
问题：
在约束g(x,y) = 0下，求J[y]的极值

方法：
引入拉格朗日乘数λ：
J*[y] = ∫(F + λg) dx

欧拉方程：
∂(F+λg)/∂y - d/dx(∂(F+λg)/∂y') = 0
```

---

### 7.2 高维推广

**多变量变分问题**：

```text
问题：
求函数u(x₁, ..., xₙ)，使
J[u] = ∫...∫F(x, u, ∂u/∂x₁, ..., ∂u/∂xₙ) dx₁...dxₙ

极值

欧拉方程：
∂F/∂u - ∑d/dxᵢ(∂F/∂(∂u/∂xᵢ)) = 0
```

## 九、数学公式总结

### 核心公式

1. **变分问题的泛函**：
   $$J[y] = \int_a^b F(x, y, y') dx$$

2. **变分问题的极值形式**：
   $$\min_{y} J[y] = \int_a^b F(x, y, y') dx$$

3. **欧拉方程**（变分法的基本方程）：
   $$\frac{\partial F}{\partial y} - \frac{d}{dx}\left(\frac{\partial F}{\partial y'}\right) = 0$$

4. **欧拉方程的展开形式**：
   $$\frac{\partial F}{\partial y} - \frac{\partial^2 F}{\partial x \partial y'} - y'\frac{\partial^2 F}{\partial y \partial y'} - y''\frac{\partial^2 F}{\partial y'^2} = 0$$

5. **Beltrami恒等式**（$F$不显式依赖$x$）：
   $$F - y'\frac{\partial F}{\partial y'} = \text{常数}$$

6. **最速降线问题的泛函**：
   $$T[y] = \int_a^b \sqrt{\frac{1 + (y')^2}{2gy}} dx$$

7. **最速降线的首次积分**：
   $$y(1 + y'^2) = \text{常数}$$

8. **最短路径问题的泛函**：
   $$L[y] = \int_a^b \sqrt{1 + (y')^2} dx$$

9. **等周问题的约束条件**：
   $$\int_a^b \sqrt{1 + (y')^2} dx = L$$

10. **等周问题的目标泛函**：
    $$A[y] = \int_a^b y dx$$

11. **约束变分问题的拉格朗日乘数法**：
    $$J^*[y] = \int_a^b (F + \lambda g) dx$$

12. **高维变分问题的欧拉方程**：
    $$\frac{\partial F}{\partial u} - \sum_{i=1}^n \frac{d}{dx_i}\left(\frac{\partial F}{\partial (\partial u/\partial x_i)}\right) = 0$$

13. **变分方法中的变分函数**：
    $$y_\varepsilon(x) = y(x) + \varepsilon \eta(x)$$

14. **泛函的变分**：
    $$\delta J = \left.\frac{d}{d\varepsilon}\right|_{\varepsilon=0} J[y + \varepsilon \delta y]$$

15. **极值条件**：
    $$\delta J = 0$$
