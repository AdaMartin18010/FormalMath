## 一、历史背景：从最速降线到变分法

### 1.1 最速降线问题

**Johann Bernoulli的问题（1696）**：

```text
问题：
在重力作用下，从点A到点B（A高于B），
什么形状的曲线使质点滑行时间最短？

挑战：
- 不是直线（虽然距离最短）
- 需要"加速"和"最短路径"的平衡
- 需要新的数学方法

答案：
摆线（cycloid）
```

**历史意义**：

```text
意义：
- 开创变分法
- 需要新的数学工具
- 连接几何与物理
- 影响深远
```

---

```text
著作：

```

## 二、变分问题的陈述

### 2.1 变分问题

**一般变分问题**：

```text
问题：
求函数y(x)，使得泛函
J[y] = ∫[a to b] F(x, y, y') dx

达到极值（最大或最小）

其中：
- F是给定的函数
- y(a) = y₀, y(b) = y₁（边界条件）
- y' = dy/dx

例子：
最速降线问题：
F = √((1+y'²)/(2gy))
```

---

### 2.2 泛函

**泛函的定义**：

```text
定义：
泛函是从函数空间到实数的映射

例子：
1. 长度泛函：
   L[y] = ∫√(1+y'²) dx

2. 面积泛函：
   A[y] = ∫y dx

3. 能量泛函：
   E[y] = ∫(y'² - V(y)) dx
```

## 三、欧拉-拉格朗日方程

### 3.1 欧拉方程

**欧拉方程（1744）**：

```text
定理：
如果y(x)使泛函J[y]达到极值，则y满足：

∂F/∂y - d/dx(∂F/∂y') = 0

这是变分法的基本方程

推导：
1. 考虑变分y(x) + εη(x)
2. 计算J[y + εη]
3. 对ε求导，令ε=0
4. 利用分部积分
5. 得到欧拉方程
```

---

### 3.2 特殊情况

**F不显式依赖x**：

```text
情况：
F = F(y, y')（不依赖x）

结果：
存在首次积分：
F - y'(∂F/∂y') = 常数

意义：
- 简化求解
- 物理意义（能量守恒）
- 应用广泛
```

**F不显式依赖y**：

```text
情况：
F = F(x, y')（不依赖y）

结果：
∂F/∂y' = 常数

例子：
最短路径问题：
F = √(1+y'²)
y'/√(1+y'²) = 常数
→ y' = 常数
→ y = 直线
```

## 四、变分法的基本方法

### 4.1 变分

**变分的定义**：

```text
定义：
函数y(x)的变分δy是满足边界条件的任意函数η(x)

性质：
- δy(a) = δy(b) = 0
- δy' = (δy)'
- δ(y₁ + y₂) = δy₁ + δy₂
```

**泛函的变分**：

```text
定义：
泛函J[y]的变分：
δJ = d/dε|_{ε=0} J[y + εδy]

极值条件：
δJ = 0（对所有变分δy）
```

---

### 4.2 变分法步骤

**求解步骤**：

```text
步骤1：建立泛函
J[y] = ∫F(x, y, y') dx

步骤2：写出欧拉方程
∂F/∂y - d/dx(∂F/∂y') = 0

步骤3：展开导数
∂F/∂y - ∂²F/∂x∂y' - y'∂²F/∂y∂y' - y''∂²F/∂y'² = 0

步骤4：求解微分方程
得到y(x)

步骤5：利用边界条件
确定常数
```

## 五、应用：经典问题

### 5.1 最速降线问题

**问题陈述**：

```text
问题：
从(0,0)到(a,b)，在重力作用下，
什么曲线使滑行时间最短？

泛函：
T[y] = ∫[0 to a] √((1+y'²)/(2gy)) dx

欧拉方程：
经过计算得到：
y(1+y'²) = 常数

解：
摆线参数方程：
x = R(θ - sin θ)
y = R(1 - cos θ)
```

---

### 5.2 等周问题

**问题陈述**：

```text
问题：
在所有周长为L的闭曲线中，
哪个围成的面积最大？

泛函：
在约束∫√(1+y'²) dx = L下，
最大化A = ∫y dx

方法：
拉格朗日乘数法
→ 得到圆
```

---

### 5.3 最短路径问题

**问题陈述**：

```text
问题：
平面上两点间最短路径

泛函：
L[y] = ∫√(1+y'²) dx

欧拉方程：
y' = 常数

解：
直线
```

## 七、现代发展

### 7.1 约束变分问题

**拉格朗日乘数法**：

```text
问题：
在约束g(x,y) = 0下，求J[y]的极值

方法：
引入拉格朗日乘数λ：
J*[y] = ∫(F + λg) dx

欧拉方程：
∂(F+λg)/∂y - d/dx(∂(F+λg)/∂y') = 0
```

---

### 7.2 高维推广

**多变量变分问题**：

```text
问题：
求函数u(x₁, ..., xₙ)，使
J[u] = ∫...∫F(x, u, ∂u/∂x₁, ..., ∂u/∂xₙ) dx₁...dxₙ

极值

欧拉方程：
∂F/∂u - ∑d/dxᵢ(∂F/∂(∂u/∂xᵢ)) = 0
```
