## 二、刚体的基本概念

### 2.1 刚体的定义

**刚体**：

```text
定义：
刚体是形状和大小不变的物体

性质：
- 任意两点间距离不变
- 6个自由度
- 3个平移 + 3个旋转

描述：
- 质心位置：r_cm(t)
- 取向：R(t)（旋转矩阵）
```

---

### 2.2 运动分解

**运动分解**：

```text
Chasles定理：
刚体的任意运动可以分解为：
1. 平移：质心的运动
2. 旋转：绕质心的旋转

数学表述：
r(t) = r_cm(t) + R(t) · r₀

其中：
- r_cm：质心位置
- R：旋转矩阵
- r₀：相对质心的位置
```

## 三、角动量与转动惯量

### 3.1 角动量

**角动量定义**：

```text
定义：
对固定点O，角动量为：
L = ∑ rᵢ × pᵢ

其中：
- rᵢ：位置向量
- pᵢ：动量向量

对刚体：
L = I · ω

其中：
- I：转动惯量张量
- ω：角速度向量
```

---

### 3.2 转动惯量

**转动惯量张量**：

```text
定义：
I = ∑ mᵢ[(rᵢ·rᵢ)E - rᵢ⊗rᵢ]

矩阵形式：
I = [
  Iₓₓ  Iₓᵧ  Iₓᵧ
  Iᵧₓ  Iᵧᵧ  Iᵧᵧ
  Iᵧₓ  Iᵧᵧ  Iᵧᵧ
]

其中：
Iₓₓ = ∑ mᵢ(yᵢ² + zᵢ²)
Iₓᵧ = -∑ mᵢ xᵢ yᵢ
...
```

**主转动惯量**：

```text
定义：
在主轴坐标系中，转动惯量矩阵对角化：
I = diag(I₁, I₂, I₃)

其中I₁, I₂, I₃是主转动惯量

性质：
- 对称矩阵
- 正定
- 坐标依赖
```

## 四、欧拉动力学方程

### 4.1 方程推导

**角动量定理**：

```text
定理：
dL/dt = M

其中：
- L：角动量
- M：外力矩

对刚体：
d(I·ω)/dt = M
```

---

### 4.2 欧拉动力学方程

**方程陈述（1765）**：

```text
在主轴坐标系中：
I₁ω̇₁ - (I₂ - I₃)ω₂ω₃ = M₁
I₂ω̇₂ - (I₃ - I₁)ω₃ω₁ = M₂
I₃ω̇₃ - (I₁ - I₂)ω₁ω₂ = M₃

其中：
- Iᵢ：主转动惯量
- ωᵢ：角速度分量
- Mᵢ：力矩分量

这是欧拉动力学方程
```

**推导思路**：

```text
步骤1：
在旋转坐标系中，角动量变化：
dL/dt = ∂L/∂t + ω × L

步骤2：
利用角动量定理：
dL/dt = M

步骤3：
在主轴坐标系中展开
得到欧拉方程
```

## 五、自由刚体运动

### 5.1 无外力情况

**自由刚体（M = 0）**：

```text
方程：
I₁ω̇₁ = (I₂ - I₃)ω₂ω₃
I₂ω̇₂ = (I₃ - I₁)ω₃ω₁
I₃ω̇₃ = (I₁ - I₂)ω₁ω₂

守恒量：
1. 角动量：L = I·ω（常数）
2. 能量：E = (1/2)ω·I·ω（常数）

性质：
- 复杂运动
- 非线性
- 混沌可能
```

---

### 5.2 对称刚体

**对称情况**：

```text
情况1：I₁ = I₂ ≠ I₃（对称陀螺）
- 绕对称轴旋转稳定
- 进动运动

情况2：I₁ = I₂ = I₃（球对称）
- 任意轴旋转
- 简单运动

情况3：I₁ ≠ I₂ ≠ I₃（非对称）
- 复杂运动
- 可能混沌
```

---

### 5.3 运动分析

**Poinsot构造**：

```text
方法：
1. 能量椭球：ω·I·ω = 2E
2. 角动量平面：L·ω = 常数
3. 交点：运动轨迹

意义：
- 几何描述
- 直观理解
- 分析工具
```

## 六、应用：陀螺运动

### 6.1 陀螺的基本运动

**重力陀螺**：

```text
问题：
重力作用下的对称陀螺

方程：
在重力场中，有重力矩：
M = r_cm × mg

运动：
- 自转：绕对称轴
- 进动：对称轴绕垂直轴
- 章动：对称轴的摆动

应用：
- 导航
- 稳定系统
- 玩具
```

---

### 6.2 进动与章动

**进动**：

```text
定义：
对称轴绕垂直轴的旋转

角速度：
Ω = mgr_cm / (I₃ω₃)

其中：
- m：质量
- g：重力加速度
- r_cm：质心到支点距离
- I₃：绕对称轴转动惯量
- ω₃：自转角速度

性质：
- 与自转角速度成反比
- 快速自转时进动慢
```

**章动**：

```text
定义：
对称轴的摆动

原因：
- 初始条件
- 能量交换
- 非线性效应

性质：
- 小角度近似：简谐振动
- 大角度：复杂运动
```
