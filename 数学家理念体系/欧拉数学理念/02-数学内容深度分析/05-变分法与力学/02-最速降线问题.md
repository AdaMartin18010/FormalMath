## 一、历史背景：Bernoulli的挑战

### 1.1 问题的提出

**Johann Bernoulli的问题（1696）**：

```text
问题：
在重力作用下，从点A到点B（A高于B），
什么形状的曲线使质点滑行时间最短？

挑战：
- 看似简单，实际困难
- 需要新的数学方法
- 吸引当时最优秀的数学家

参与者：
- Johann Bernoulli（提出者）
- Jakob Bernoulli（兄弟）
- Newton（匿名解答）
- Leibniz
- de l'Hôpital
- 欧拉（系统化方法）
```

**历史意义**：

```text
意义：
- 开创变分法
- 连接数学与物理
- 影响深远
- 现代优化理论的起点
```

---

### 1.2 直觉与反直觉

**直觉错误**：

```text
直觉：
最短路径应该是直线（距离最短）

但：
- 直线虽然距离最短
- 但速度增长慢
- 总时间不是最短

正确：
- 需要"加速"和"距离"的平衡
- 曲线更优
- 答案是摆线
```

## 二、问题的陈述

### 2.1 精确表述

**数学问题**：

```text
问题：
在重力场中，从点(0,0)到点(a,b)（b < 0），
求曲线y(x)，使质点沿曲线滑行时间最短

假设：
- 无摩擦
- 重力加速度g
- 初始速度v₀ = 0

目标：
最小化时间T[y] = ∫ dt
```

---

### 2.2 物理约束

**能量守恒**：

```text
在点(x, y)处：
动能：mv²/2
势能：mgy

能量守恒：
mv²/2 + mgy = 0（初始能量）

因此：
v = √(-2gy)

速度只依赖于高度y
```

---

## 三、物理分析

### 3.1 时间计算

**时间元素**：

```text
在曲线元素ds上：
dt = ds/v

其中：
- ds = √(1 + y'²) dx（弧长元素）
- v = √(-2gy)（速度）

因此：
dt = √(1 + y'²)/√(-2gy) dx
```

**总时间**：

```text
总时间：
T[y] = ∫[0 to a] √((1 + y'²)/(-2gy)) dx

简化：
T[y] = (1/√(2g)) ∫[0 to a] √((1 + y'²)/(-y)) dx

这是变分问题
```

---

### 3.2 变分问题

**泛函**：

```text
泛函：
T[y] = ∫[0 to a] F(x, y, y') dx

其中：
F(x, y, y') = √((1 + y'²)/(-y))

边界条件：
y(0) = 0
y(a) = b
```

---

## 四、变分法求解

### 4.1 欧拉方程

**应用欧拉方程**：

```text
欧拉方程：
∂F/∂y - d/dx(∂F/∂y') = 0

计算：
∂F/∂y = (1/2)√((1+y'²)/(-y)) · (1/y²)
       = (1/(2y²))√((1+y'²)/(-y))

∂F/∂y' = (1/2)√((1+y'²)/(-y)) · (2y'/(1+y'²))
        = (y'/√(1+y'²))√(1/(-y))

d/dx(∂F/∂y') = ...
```

---

### 4.2 Beltrami恒等式

**特殊情况**：

```text
由于F不显式依赖x，存在首次积分：

F - y'(∂F/∂y') = 常数

计算：
√((1+y'²)/(-y)) - y' · (y'/√(1+y'²))√(1/(-y))
= √((1+y'²)/(-y)) - (y'²/√(1+y'²))√(1/(-y))
= (1/√(1+y'²))√(1/(-y))
= 常数

简化：
y(1 + y'²) = 常数
```

## 五、摆线解

### 5.1 摆线的定义

**摆线（Cycloid）**：

```text
定义：
摆线是圆在直线上滚动时，圆周上一点的轨迹

参数方程：
x = R(θ - sin θ)
y = R(1 - cos θ)

其中：
- R是圆的半径
- θ是滚动角度

性质：
- 等时性
- 最速降线
- 应用广泛
```

---

### 5.2 验证

**验证摆线是最速降线**：

```text
从参数方程：
dx/dθ = R(1 - cos θ)
dy/dθ = R sin θ

因此：
y' = dy/dx = (dy/dθ)/(dx/dθ)
   = sin θ/(1 - cos θ)
   = cot(θ/2)

计算：
1 + y'² = 1 + cot²(θ/2) = csc²(θ/2)

y(1 + y'²) = R(1 - cos θ) · csc²(θ/2)
           = R(2 sin²(θ/2)) · csc²(θ/2)
           = 2R

常数！✓

因此摆线满足欧拉方程
```

---

### 5.3 边界条件

**确定参数**：

```text
边界条件：
y(0) = 0 → θ = 0
y(a) = b → 需要确定R和θ₁

从：
a = R(θ₁ - sin θ₁)
b = R(1 - cos θ₁)

可以确定R和θ₁

解：
摆线是唯一解
```
