**古代音乐理论**：

```text

- 毕达哥拉斯（公元前6世纪）：音程与数比
- 发现：八度音程 = 频率比 2:1
- 五度音程 = 频率比 3:2
- 四度音程 = 频率比 4:3

问题：
- 如何构造完整的音阶？
- 如何调和不同音程？
- 如何建立调性系统？

挑战：
```

## 二、音阶的数学结构

### 2.1 频率比与音程

**基本音程**：

```text
定义：
音程 = 两个音频率的比

基本音程：
- 八度：2:1
- 五度：3:2
- 四度：4:3
- 大三度：5:4
- 小三度：6:5
- 大二度：9:8
- 小二度：16:15

性质：
- 乘法性：音程叠加 = 频率比相乘
- 对数性：音程相加 = 频率比对数相加
```

**音程的数学表示**：

```text
方法：
- 频率比：f₂/f₁
- 音分数：1200·log₂(f₂/f₁)
- 音程值：log(f₂/f₁)

优势：
- 线性化
- 可加性
- 计算方便
```

---

### 2.2 音阶构造

**自然音阶**：

```text
构造：
从C开始，使用纯五度（3:2）构造：

C → G (3:2)
G → D (3:2) = (3:2)²
D → A (3:2) = (3:2)³
A → E (3:2) = (3:2)⁴
E → B (3:2) = (3:2)⁵

问题：
- 循环不闭合
- 转调困难
- 需要调整
```

**平均律音阶**：

```text
构造：
将八度（2:1）等分为12个半音：
每个半音 = 2^(1/12)

频率：
C: 1
C#: 2^(1/12)
D: 2^(2/12)
...
C': 2^(12/12) = 2

优势：
- 转调方便
- 对称性
- 计算简单

代价：
- 音程不完全纯
- 和声略有偏差
```

---

### 2.3 调式理论

**调式定义**：

```text
定义：
调式 = 音阶中音的选择与排列

主要调式：
- 大调：1, 2, 3, 4, 5, 6, 7
- 小调：1, 2, b3, 4, 5, b6, b7

数学表示：
- 音程序列
- 频率比序列
- 音分数序列

性质：
- 周期性
- 对称性
- 组合结构
```

## 三、和声的数学基础

### 3.1 和弦的频率关系

**三和弦**：

```text
定义：
三个音同时发声

大三和弦：
- 根音：f
- 三音：5f/4（大三度）
- 五音：3f/2（五度）

频率比：1 : 5/4 : 3/2 = 4 : 5 : 6

小三和弦：
- 根音：f
- 三音：6f/5（小三度）
- 五音：3f/2（五度）

频率比：1 : 6/5 : 3/2 = 10 : 12 : 15
```

**和声级数**：

```text
定义：
和声级数 = 频率比为整数比的音

例子：
- 八度：2:1
- 五度：3:2
- 四度：4:3
- 大三度：5:4

性质：
- 简单比 = 协和
- 复杂比 = 不协和
```

---

### 3.2 和声级数

**欧拉的和声级数**：

```text
定义：
和声级数 = 1, 1/2, 1/3, 1/4, ...

频率关系：
- 基音：f
- 第二泛音：2f（八度）
- 第三泛音：3f（十二度）
- 第四泛音：4f（两个八度）

应用：
- 和声分析
- 音色理解
- 乐器设计
```

---

### 3.3 不协和与协和

**协和度**：

```text
定义：
协和度 = 频率比的简单性

欧拉的方法：
- 计算频率比的最简分数
- 分子+分母越小，越协和

例子：
- 八度（2:1）：协和度 = 2+1 = 3（最协和）
- 五度（3:2）：协和度 = 3+2 = 5
- 大三度（5:4）：协和度 = 5+4 = 9
- 小二度（16:15）：协和度 = 16+15 = 31（不协和）

应用：
- 和声分析
- 作曲指导
- 调性理论
```

## 四、欧拉的音阶理论

### 4.1 音阶的数学表示

**音阶的数学模型**：

```text
方法：
- 频率序列：f₀, f₁, f₂, ..., f₁₂
- 音程序列：r₁, r₂, ..., r₁₂
- 关系：fᵢ = f₀ · r₁ · r₂ · ... · rᵢ

约束：
- 八度闭合：f₁₂ = 2f₀
- 音程关系：r₁ · r₂ · ... · r₁₂ = 2

优化：
- 最小化偏差
- 最大化协和度
- 平衡转调
```

---

### 4.2 音阶的优化

**优化问题**：

```text
问题：
在给定约束下，选择最优音阶

目标：
1. 主要音程尽可能纯
2. 转调方便
3. 和声协和

方法：
- 变分法
- 优化理论
- 数值方法

结果：
- 平均律：转调最优
- 纯律：和声最优
- 中庸全音律：折中
```

---

### 4.3 调性系统

**调性系统**：

```text
定义：
调性系统 = 音阶 + 和声规则

要素：
- 主音（tonic）
- 属音（dominant）
- 下属音（subdominant）
- 和声进行

数学表示：
- 音级：0, 1, 2, ..., 11（模12）
- 和声函数：T, D, S
- 进行规则：T → D → T

性质：
- 周期性
- 对称性
- 群结构
```
