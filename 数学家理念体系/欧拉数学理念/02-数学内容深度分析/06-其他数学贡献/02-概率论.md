## 二、组合概率问题

### 2.1 排列与组合

**排列**：

```text
定义：
从n个元素中取r个的排列数：
P(n,r) = n!/(n-r)!

性质：
- 有序
- 不重复
- 计数公式

应用：
- 概率计算
- 组合问题
- 随机抽样
```

**组合**：

```text
定义：
从n个元素中取r个的组合数：
C(n,r) = n!/(r!(n-r)!)

性质：
- 无序
- 不重复
- 对称性：C(n,r) = C(n,n-r)

应用：
- 概率计算
- 组合问题
- 二项分布
```

---

### 2.2 概率计算

**古典概率**：

```text
定义：
P(A) = |A|/|Ω|

其中：
- Ω：样本空间
- A：事件
- |·|：元素个数

条件：
- 等可能性
- 有限样本空间
- 可数事件
```

**概率计算技巧**：

```text
方法：
1. 直接计数
2. 组合方法
3. 生成函数
4. 递推关系

优势：

```

## 三、生成函数方法

### 3.1 生成函数

**普通生成函数**：

```text
定义：
对于序列{aₙ}，生成函数为：
A(x) = ∑ₙ aₙxⁿ

性质：
- 加法：对应项相加
- 乘法：卷积
- 求导：递推关系
- 积分：求和

应用：
- 组合计数
- 概率计算
- 递推关系
```

---

### 3.2 概率生成函数

**概率生成函数**：

```text
定义：
对于离散随机变量X，概率生成函数为：
Gₓ(s) = E[s^X] = ∑ₖ P(X=k)sᵏ

性质：
- Gₓ(1) = 1
- G'ₓ(1) = E[X]
- G''ₓ(1) = E[X(X-1)]

应用：
- 期望值计算
- 方差计算
- 分布性质
```

**独立随机变量**：

```text
定理：
若X, Y独立，则：
Gₓ₊ᵧ(s) = Gₓ(s) · Gᵧ(s)

应用：
- 和分布
- 卷积
- 独立性
```

## 四、期望值与方差

### 4.1 期望值

**期望值定义**：

```text
定义：
对于离散随机变量X：
E[X] = ∑ₖ k·P(X=k)

对于连续随机变量X：
E[X] = ∫ x·f(x)dx

性质：
- 线性性：E[aX+b] = aE[X] + b
- 可加性：E[X+Y] = E[X] + E[Y]（独立）

意义：
- 平均值
- 中心趋势
- 预测值
```

**期望值计算**：

```text
方法：
1. 直接定义
2. 生成函数：G'(1)
3. 条件期望
4. 递推关系

应用：
- 概率计算
- 决策问题
- 优化问题
```

---

### 4.2 方差

**方差定义**：

```text
定义：
Var(X) = E[(X - E[X])²] = E[X²] - (E[X])²

性质：
- Var(aX+b) = a²Var(X)
- Var(X+Y) = Var(X) + Var(Y)（独立）

意义：
- 离散程度
- 风险度量
- 不确定性
```

**方差计算**：

```text
方法：
1. 直接定义
2. 生成函数：G''(1) + G'(1) - (G'(1))²
3. 条件方差
4. 递推关系

应用：
- 风险评估
- 统计分析
- 质量控制
```

## 五、条件概率与独立性

### 5.1 条件概率

**条件概率定义**：

```text
定义：
P(A|B) = P(A∩B)/P(B)

其中P(B) > 0

性质：
- 0 ≤ P(A|B) ≤ 1
- P(Ω|B) = 1
- 可加性：P(A₁∪A₂|B) = P(A₁|B) + P(A₂|B)（互斥）

意义：
- 在B发生条件下A的概率
- 信息更新
- 贝叶斯推理
```

---

### 5.2 独立性

**独立性定义**：

```text
定义：
事件A, B独立 ⟺
P(A∩B) = P(A)·P(B)

等价：
P(A|B) = P(A)

推广：
n个事件独立 ⟺
任意子集的交概率等于概率乘积

意义：
- 无关联
- 信息独立
- 简化计算
```

---

### 5.3 贝叶斯公式

**贝叶斯公式**：

```text
公式：
P(A|B) = P(B|A)·P(A)/P(B)

其中：
- P(A|B)：后验概率
- P(B|A)：似然
- P(A)：先验概率
- P(B)：证据

应用：
- 统计推断
- 机器学习
- 决策理论
```

**全概率公式**：

```text
公式：
P(B) = ∑ᵢ P(B|Aᵢ)·P(Aᵢ)

其中{Aᵢ}是完备事件组

应用：
- 概率分解
- 贝叶斯公式
- 条件概率
```

## 六、随机过程

### 6.1 随机游走

**随机游走**：

```text
定义：
在整数轴上，每步以概率p向右，概率q=1-p向左

状态：
- 位置：Xₙ
- 转移：Xₙ₊₁ = Xₙ + ξₙ
- 其中ξₙ = ±1

性质：
- 马尔可夫性
- 独立增量
- 平稳性

应用：
- 物理：扩散
- 金融：股价
- 生物：随机运动
```

**首达时间**：

```text
问题：
从位置i到达位置j的期望时间

方法：
- 递推关系
- 生成函数
- 分析解

应用：
- 随机过程
- 概率计算
- 优化问题
```

---

### 6.2 马尔可夫链

**马尔可夫链**：

```text
定义：
随机过程{Xₙ}满足：
P(Xₙ₊₁ = j | Xₙ = i, Xₙ₋₁, ..., X₀) = P(Xₙ₊₁ = j | Xₙ = i)

性质：
- 无记忆性
- 转移概率
- 状态空间

应用：
- 随机过程
- 统计模型
- 算法分析
```

**转移矩阵**：

```text
定义：
P = [pᵢⱼ]，其中pᵢⱼ = P(Xₙ₊₁ = j | Xₙ = i)

性质：
- 行和为1
- 非负元素
- 矩阵性质

应用：
- 状态转移
- 稳态分布
- 概率计算
```
