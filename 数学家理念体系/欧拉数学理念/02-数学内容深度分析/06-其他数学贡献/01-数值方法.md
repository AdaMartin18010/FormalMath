## 二、欧拉方法：微分方程数值解

### 2.1 方法陈述

**欧拉方法（1768）**：

```text
问题：
求解初值问题：
y' = f(t, y)
y(t₀) = y₀

方法：
1. 离散化：tₙ = t₀ + nh
2. 近似：y' ≈ (yₙ₊₁ - yₙ)/h
3. 迭代：yₙ₊₁ = yₙ + h·f(tₙ, yₙ)

其中：
- h：步长
- n：步数
- yₙ：近似值
```

**几何解释**：

```text
思想：
- 用切线近似曲线
- 逐步推进
- 局部线性化

图形：
- 斜率场
- 切线方向
- 逐步逼近
```

---

### 2.2 误差分析

**局部截断误差**：

```text
定义：
在单步中的误差：
τₙ = y(tₙ₊₁) - [y(tₙ) + h·f(tₙ, y(tₙ))]

展开：
τₙ = (h²/2)y''(ξₙ) + O(h³)

结论：
局部误差 = O(h²)
```

**全局误差**：

```text
定义：
累积误差：
Eₙ = y(tₙ) - yₙ

分析：
- 每步误差累积
- 全局误差 = O(h)

结论：
欧拉方法是一阶方法
```

## 三、数值积分方法

### 3.1 梯形法则

**梯形法则**：

```text
思想：
用梯形近似曲线下面积

公式：
∫[a,b] f(x)dx ≈ (b-a)/2 · [f(a) + f(b)]

复合公式：
∫[a,b] f(x)dx ≈ h/2 · [f(x₀) + 2∑f(xᵢ) + f(xₙ)]

其中：
- h = (b-a)/n
- xᵢ = a + ih
```

**误差分析**：

```text
误差：
E = -(b-a)³/(12n²) · f''(ξ)

性质：
- 二阶精度
- 误差与步长平方成正比
- 适用于光滑函数
```

---

### 3.2 Simpson法则

**Simpson法则**：

```text
思想：
用抛物线近似曲线

公式：
∫[a,b] f(x)dx ≈ (b-a)/6 · [f(a) + 4f((a+b)/2) + f(b)]

复合公式：
∫[a,b] f(x)dx ≈ h/3 · [f(x₀) + 4∑f(x₂ᵢ₋₁) + 2∑f(x₂ᵢ) + f(xₙ)]

其中：
- n为偶数
- h = (b-a)/n
```

**误差分析**：

```text
误差：
E = -(b-a)⁵/(180n⁴) · f⁽⁴⁾(ξ)

性质：
- 四阶精度
- 高精度
- 适用于光滑函数
```

---

## 四、插值方法

### 4.1 多项式插值

**插值问题**：

```text
问题：
给定n+1个点 (xᵢ, yᵢ)，求多项式P(x)使得：
P(xᵢ) = yᵢ

解：
存在唯一的n次多项式

方法：
- Lagrange插值
- Newton插值
- 差分插值
```

**Lagrange插值**：

```text
公式：
P(x) = ∑ᵢ yᵢ · Lᵢ(x)

其中：
Lᵢ(x) = ∏ⱼ≠ᵢ (x - xⱼ)/(xᵢ - xⱼ)

性质：
- 唯一性
- 显式公式
- 计算复杂度O(n²)
```

---

### 4.2 差分插值

**差分算子**：

```text
定义：
前向差分：
Δf(x) = f(x+h) - f(x)

后向差分：
∇f(x) = f(x) - f(x-h)

中心差分：
δf(x) = f(x+h/2) - f(x-h/2)
```

**Newton前向差分公式**：

```text
公式：
P(x) = f(x₀) + Δf(x₀)·s + Δ²f(x₀)·s(s-1)/2! + ...

其中：
s = (x - x₀)/h

应用：
- 等距节点
- 计算高效
- 递推计算
```

### 4.3 应用

**应用领域**：

```text
应用：
1. 函数近似
2. 数值微分
3. 数值积分
4. 数据拟合

方法：
```

## 五、差分方程

### 5.1 差分算子

**差分算子理论**：

```text
算子：
- Δ：前向差分
- ∇：后向差分
- E：移位算子，Ef(x) = f(x+h)

关系：
Δ = E - I
∇ = I - E⁻¹
E = I + Δ
```

**差分恒等式**：

```text
恒等式：
Δ(fg) = fΔg + (Ef)Δf
Δ(f/g) = [gΔf - fΔg] / [g(Eg)]

应用：
- 差分方程
- 求和公式
- 数值方法
```

---

### 5.2 差分方程求解

**线性差分方程**：

```text
方程：
aₙyₙ₊ₖ + aₙ₋₁yₙ₊ₖ₋₁ + ... + a₀yₙ = fₙ

齐次方程：
特征方程：
aₙλᵏ + aₙ₋₁λᵏ⁻¹ + ... + a₀ = 0

解：
yₙ = ∑ᵢ Cᵢλᵢⁿ

其中λᵢ是特征根
```

**非齐次方程**：

```text
方法：
1. 齐次解
2. 特解
3. 通解 = 齐次解 + 特解

特解方法：
- 待定系数
- 变分常数
- 生成函数
```

## 六、数值级数求和

### 6.1 级数加速

**级数加速方法**：

```text
问题：
级数收敛慢，需要加速

方法：
1. Euler变换
2. Aitken加速
3. Richardson外推
4. 连分式

目标：
- 提高收敛速度
- 减少计算量
- 提高精度
```

**Euler变换**：

```text
方法：
对于交错级数：
∑(-1)ⁿaₙ

变换：
∑(-1)ⁿaₙ = ∑(-1)ⁿΔⁿa₀/2ⁿ⁺¹

其中：
Δⁿa₀是n阶差分

应用：
- 加速收敛
- 数值计算
- 级数求和
```

---

### 6.2 欧拉-麦克劳林公式

**公式陈述**：

```text
公式：
∑ᵢ₌₀ⁿ f(i) = ∫₀ⁿ f(x)dx + [f(0) + f(n)]/2
              + ∑ₖ₌₁ᵐ B₂ₖ/(2k)! [f⁽²ᵏ⁻¹⁾(n) - f⁽²ᵏ⁻¹⁾(0)]
              + Rₘ

其中：
- B₂ₖ：Bernoulli数
- Rₘ：余项

应用：
- 级数求和
- 数值积分
- 渐近展开
```

**应用**：

```text
应用：
1. 级数求和：精确值或近似
2. 数值积分：误差估计
3. 渐近分析：大n行为

优势：
- 高精度
- 系统化
- 理论完善
```
