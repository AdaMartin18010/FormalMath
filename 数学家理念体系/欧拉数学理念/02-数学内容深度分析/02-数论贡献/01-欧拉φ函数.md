## 一、历史背景：从费马到欧拉

### 1.1 费马小定理

**费马的发现（1640）**：

```text
费马小定理：
若p是素数，且gcd(a, p) = 1，则
a^(p-1) ≡ 1 (mod p)

例子：
p = 5, a = 2
2^4 = 16 ≡ 1 (mod 5)

意义：
- 素数的重要性质
- 模运算的基础
- 但费马未给出完整证明
```

**欧拉的证明（1736）**：

```text
欧拉首次给出完整证明：
- 利用同余类的性质
- 证明a, 2a, ..., (p-1)a构成模p的完全剩余系
- 得到a^(p-1) ≡ 1 (mod p)

意义：
- 严格化费马的结果
- 为推广铺路
```

---

### 1.2 欧拉的推广

**问题**：

```text
费马小定理：
- 仅适用于素数p
- 对合数n，a^(n-1)不一定≡1 (mod n)

问题：
能否推广到一般正整数n？
```

**欧拉的洞察（1763）**：

```text
关键观察：
- 不是所有小于n的数都与n互素
- 只有φ(n)个数与n互素
- 这些数形成乘法群

结果：
欧拉定理：若gcd(a, n) = 1，则
a^φ(n) ≡ 1 (mod n)
```

## 二、φ函数的定义与基本性质

### 2.1 定义

**欧拉φ函数的定义**：

```text
定义：
对于正整数n，欧拉φ函数φ(n)定义为：
φ(n) = |{k ∈ ℕ : 1 ≤ k ≤ n, gcd(k, n) = 1}|

即：φ(n)是小于等于n且与n互素的正整数个数

例子：
φ(1) = 1（1与1互素）
φ(2) = 1（只有1与2互素）
φ(3) = 2（1, 2与3互素）
φ(4) = 2（1, 3与4互素）
φ(5) = 4（1, 2, 3, 4与5互素）
φ(6) = 2（1, 5与6互素）
```

**几何解释**：

```text
φ(n)也可以理解为：
在1到n的整数中，与n互素的数的"密度"

例子：
φ(10) = 4
在1, 2, 3, 4, 5, 6, 7, 8, 9, 10中
与10互素的是：1, 3, 7, 9
密度：4/10 = 2/5
```

---

### 2.2 基本性质

**性质1：素数情况**：

```text
定理：
若p是素数，则φ(p) = p - 1

证明：
1, 2, ..., p-1都与p互素
只有p与p不互素（gcd(p, p) = p）

例子：
φ(7) = 6
φ(11) = 10
```

**性质2：素数幂情况**：

```text
定理：
若p是素数，k ≥ 1，则
φ(p^k) = p^k - p^(k-1) = p^(k-1)(p - 1)

证明：
在1到p^k中，与p^k不互素的数是p的倍数：
p, 2p, 3p, ..., p^(k-1)·p = p^k
共p^(k-1)个

因此：
φ(p^k) = p^k - p^(k-1)

例子：
φ(8) = φ(2³) = 2³ - 2² = 8 - 4 = 4
φ(9) = φ(3²) = 3² - 3¹ = 9 - 3 = 6
```

**性质3：乘性函数**：

```text
定理：
若gcd(m, n) = 1，则
φ(mn) = φ(m)φ(n)

证明思路：
利用中国剩余定理
建立双射：
{与mn互素的数} ↔ {与m互素} × {与n互素}

意义：
- φ是乘性函数
- 可以分解计算
- 简化计算
```

## 三、φ函数的计算公式

### 3.1 一般公式

**欧拉的公式**：

```text
定理：
设n = p₁^e₁ · p₂^e₂ · ... · pₖ^eₖ
是n的素因子分解，则

φ(n) = n ∏(i=1 to k) (1 - 1/pᵢ)
     = n (1 - 1/p₁)(1 - 1/p₂)...(1 - 1/pₖ)

证明：
利用乘性：
φ(n) = φ(p₁^e₁) · φ(p₂^e₂) · ... · φ(pₖ^eₖ)
      = p₁^(e₁-1)(p₁-1) · p₂^(e₂-1)(p₂-1) · ... · pₖ^(eₖ-1)(pₖ-1)
      = n ∏(1 - 1/pᵢ)
```

**例子**：

```text
n = 12 = 2² · 3
φ(12) = 12 · (1 - 1/2) · (1 - 1/3)
       = 12 · (1/2) · (2/3)
       = 12 · (1/3)
       = 4

验证：
与12互素的数：1, 5, 7, 11
共4个 ✓

n = 60 = 2² · 3 · 5
φ(60) = 60 · (1 - 1/2) · (1 - 1/3) · (1 - 1/5)
       = 60 · (1/2) · (2/3) · (4/5)
       = 60 · (8/30)
       = 16
```

---

### 3.2 递推关系

**递推公式**：

```text
对于素数p：
φ(p) = p - 1

对于素数幂p^k：
φ(p^k) = p · φ(p^(k-1))

对于互素的m, n：
φ(mn) = φ(m) · φ(n)
```

**计算算法**：

```text
方法1：利用素因子分解
1. 分解n = ∏pᵢ^eᵢ
2. 计算φ(n) = n∏(1 - 1/pᵢ)

方法2：利用递推
1. 若n是素数，φ(n) = n - 1
2. 若n = p^k，φ(n) = p^(k-1)(p - 1)
3. 若n = mn（互素），φ(n) = φ(m)φ(n)
```

## 四、φ函数的应用

### 4.1 欧拉定理

**欧拉定理（1763）**：

```text
定理：
若gcd(a, n) = 1，则
a^φ(n) ≡ 1 (mod n)

证明思路：
1. 设S = {r₁, r₂, ..., r_{φ(n)}}是与n互素的剩余类
2. 证明aS = {ar₁, ar₂, ..., ar_{φ(n)}}也是完全剩余系
3. 因此∏rᵢ ≡ ∏(arᵢ) ≡ a^φ(n)∏rᵢ (mod n)
4. 由于∏rᵢ与n互素，可以约去
5. 得到a^φ(n) ≡ 1 (mod n)
```

**费马小定理的推广**：

```text
费马小定理：
若p是素数，gcd(a, p) = 1，则
a^(p-1) ≡ 1 (mod p)

欧拉定理：
若gcd(a, n) = 1，则
a^φ(n) ≡ 1 (mod n)

关系：
费马小定理是欧拉定理的特殊情况
（当n = p是素数时，φ(p) = p - 1）
```

---

### 4.2 模幂运算

**快速模幂**：

```text
问题：
计算a^b (mod n)，其中b很大

方法：
利用欧拉定理
若gcd(a, n) = 1，则
a^b ≡ a^(b mod φ(n)) (mod n)

例子：
计算2^1000 (mod 7)
φ(7) = 6
1000 = 6·166 + 4
2^1000 ≡ 2^4 = 16 ≡ 2 (mod 7)
```

**应用**：

```text
1. 密码学：RSA加密
2. 数论计算：大数模幂
3. 算法优化：减少计算量
```

---

### 4.3 原根理论

**原根的定义**：

```text
定义：
若gcd(a, n) = 1，且
a的阶（最小正整数k使得a^k ≡ 1 (mod n)）
等于φ(n)，则a是n的原根

即：
ord_n(a) = φ(n)

性质：
- 原根存在 ⟺ n = 2, 4, p^k, 2p^k（p奇素数）
- 若原根存在，则恰有φ(φ(n))个原根
```

**应用**：

```text
1. 离散对数问题
2. 密码学：Diffie-Hellman密钥交换
3. 数论研究
```

## 五、φ函数与群论

### 5.1 乘法群

**模n的乘法群**：

```text
定义：
(ℤ/nℤ)* = {a mod n : gcd(a, n) = 1}

性质：
1. (ℤ/nℤ)*是乘法群
2. |(ℤ/nℤ)*| = φ(n)
3. 群的阶是φ(n)

意义：
- φ(n)是群的阶
- 欧拉定理是Lagrange定理的体现
- 连接数论与群论
```

**Lagrange定理的应用**：

```text
Lagrange定理：
有限群G中，任意元素a的阶整除|G|

应用：
在(ℤ/nℤ)*中：
- |(ℤ/nℤ)*| = φ(n)
- 任意a的阶整除φ(n)
- 因此a^φ(n) = 1

这给出了欧拉定理的群论证明
```

---

### 5.2 循环群

**循环群结构**：

```text
定理：
(ℤ/pℤ)*是循环群（p是奇素数）

即：
存在原根g，使得
(ℤ/pℤ)* = {g, g², ..., g^(p-1)}

意义：
- 所有元素可表示为g的幂
- 简化计算
- 原根理论的基础
```

## 六、现代应用：RSA加密

### 6.1 RSA算法

**密钥生成**：

```text
步骤1：选择两个大素数p, q
步骤2：计算n = pq
步骤3：计算φ(n) = (p-1)(q-1)
步骤4：选择e，使得gcd(e, φ(n)) = 1
步骤5：计算d，使得ed ≡ 1 (mod φ(n))
步骤6：公钥：(n, e)
      私钥：(n, d)
```

**加密与解密**：

```text
加密：
c ≡ m^e (mod n)

解密：
m ≡ c^d (mod n)

正确性：
c^d ≡ (m^e)^d = m^(ed) (mod n)
由于ed ≡ 1 (mod φ(n))，有
ed = 1 + kφ(n)
因此：
m^(ed) = m^(1+kφ(n)) = m · (m^φ(n))^k
       ≡ m · 1^k = m (mod n)
（利用欧拉定理，要求gcd(m, n) = 1）
```

---

### 6.2 安全性

**安全性基础**：

```text
RSA的安全性基于：
1. 大整数因子分解的困难性
2. 计算φ(n)的困难性（已知n的因子分解）

关系：
- 若知道p, q，可计算φ(n) = (p-1)(q-1)
- 若知道φ(n)，可分解n
- 因此计算φ(n)与因子分解等价困难
```

## 七、现代发展与推广

### 7.1 Jordan函数

**推广**：

```text
Jordan函数：
J_k(n) = n^k ∏(1 - 1/p^k)

性质：
- J₁(n) = φ(n)
- 乘性函数
- 数论应用
```

---

### 7.2 Carmichael函数

**定义**：

```text
Carmichael函数λ(n)：
最小的正整数k，使得
对所有gcd(a, n) = 1，有
a^k ≡ 1 (mod n)

性质：
- λ(n) | φ(n)
- 更精确的指数
- 密码学应用
```
