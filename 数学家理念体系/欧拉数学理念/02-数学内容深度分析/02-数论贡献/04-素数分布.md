**经典证明（约公元前300年）**：

```text
定理：
素数有无穷多个

证明（反证法）：
假设素数只有有限个：p₁, p₂, ..., pₙ
考虑数N = p₁p₂...pₙ + 1
N不能被任何pᵢ整除（余数为1）
因此N是素数或包含新的素因子
矛盾！

意义：
- 经典数论结果
- 但未涉及分布
```

**局限**：

问题：

- 分析素数分布

- 建立理论

## 二、欧拉乘积公式

### 2.1 公式的发现

**欧拉乘积公式（1737）**：

**公式**：
$$\zeta(s) = \sum_{n=1}^{\infty} \frac{1}{n^s} = \prod_{p \text{ 素数}} \frac{1}{1 - p^{-s}}$$

其中 $s > 1$（实数）。

**展开**：
$$\prod_{p \text{ 素数}} \frac{1}{1 - p^{-s}} = \prod_{p \text{ 素数}} (1 + p^{-s} + p^{-2s} + \cdots)$$

**原因**：
每个正整数有唯一的素数分解，因此级数展开后，每个 $\frac{1}{n^s}$ 恰好出现一次。

**证明思路**：

**步骤1：几何级数**
对每个素数 $p$：
$$\frac{1}{1 - p^{-s}} = 1 + p^{-s} + p^{-2s} + \cdots$$

**步骤2：乘积展开**
$$\prod_{p \text{ 素数}} \frac{1}{1 - p^{-s}} = \prod_{p \text{ 素数}} (1 + p^{-s} + p^{-2s} + \cdots)$$

**步骤3：唯一分解**
每个正整数 $n = p_1^{e_1} \cdot p_2^{e_2} \cdot \cdots$，在乘积中，$n^{-s}$ 恰好出现一次（来自 $p_1^{e_1}$ 的项、$p_2^{e_2}$ 的项等的乘积）。

**步骤4：得到级数**
$$= \sum_{n=1}^{\infty} \frac{1}{n^s} = \zeta(s)$$

---

### 2.2 公式的意义

**连接级数与乘积**：

```text
意义：
- 级数：∑1/n^s（对所有正整数）
- 乘积：∏1/(1-p^(-s))（对所有素数）

关系：
- 级数的性质 ⟺ 乘积的性质
- 级数发散 ⟺ 乘积发散
- 连接离散与连续
```

**对数形式**：

取对数：
$$\ln \zeta(s) = -\sum_{p \text{ 素数}} \ln(1 - p^{-s}) = \sum_{p \text{ 素数}} \sum_{m=1}^{\infty} \frac{p^{-ms}}{m}$$

**意义**：
- 连接 $\zeta$ 函数与素数
- 为素数定理铺路
- 启发现代数论

## 三、欧拉对素数无穷的分析证明

### 3.1 分析证明

**欧拉的方法（1737）**：

利用乘积公式：
$$\zeta(s) = \prod_{p \text{ 素数}} \frac{1}{1 - p^{-s}}$$

当 $s = 1$ 时：
$$\zeta(1) = \sum_{n=1}^{\infty} \frac{1}{n} = \infty \quad \text{（调和级数发散）}$$

因此：
$$\prod_{p \text{ 素数}} \frac{1}{1 - 1/p} = \infty$$

如果素数只有有限个，则左边是有限乘积，矛盾！

因此素数有无穷多个。

**与欧几里得证明的对比**：

```text
欧几里得方法：
- 初等方法
- 构造性
- 直接

欧拉方法：
- 分析方法
- 利用级数
- 间接但深刻
```

---

### 3.2 素数密度的估计

**欧拉的观察**：

欧拉通过无穷乘积发现了素数分布的重要性质。

**观察**：

$$\prod_{p \text{ prime}} \frac{1}{1 - 1/p} = \infty$$

但：
$$\prod_{p \text{ prime}} \frac{1}{1 - 1/p^2} = \zeta(2) = \frac{\pi^2}{6} \quad \text{（有限）}$$

**数学表述**：

欧拉乘积：
$$\prod_{p \text{ prime}} \frac{1}{1 - 1/p} = \infty$$

平方倒数乘积：
$$\prod_{p \text{ prime}} \frac{1}{1 - 1/p^2} = \zeta(2) = \frac{\pi^2}{6}$$

**意义**：

- **素数"足够多"使乘积发散**：说明素数足够多
- **但"不太多"使某些乘积收敛**：说明素数分布有规律
- **启发密度研究**：为素数密度研究提供了启发

## 四、素数分布的研究

### 4.1 素数计数函数

**定义**：

素数计数函数是研究素数分布的重要工具。

**素数计数函数**：
$$\pi(x) = \#\{p \leq x : p \text{ 是素数}\}$$

即：不超过 $x$ 的素数个数。

**数学表述**：

素数计数函数：
$$\pi(x) = \#\{p \leq x : p \text{ 是素数}\}$$

**例子**：

### 4.2 素数定理的猜想

**Gauss的猜想（1792-1793）**：

Gauss观察到：
$$\pi(x) \sim \frac{x}{\ln x}$$

即：
$$\lim_{x \to \infty} \frac{\pi(x)}{x/\ln x} = 1$$

**意义**：
- 素数密度约为 $\frac{1}{\ln x}$
- 为素数定理铺路

**Legendre的改进（1798）**：

Legendre提出了更精确的近似。

**Legendre公式**：

$$\pi(x) \sim \frac{x}{\ln x - A}$$

其中 $A \approx 1.08366$。

**数学表述**：

Legendre公式：
$$\pi(x) \sim \frac{x}{\ln x - A}$$

其中 $A \approx 1.08366$。

**改进**：

- **更精确的近似**：比Gauss公式更精确
- **但A的选择有问题**：$A$ 的选择有问题

### 5.2 素数定理

**定理陈述（1896）**：

**定理（素数定理）**：
$$\pi(x) \sim \text{Li}(x) = \int_2^x \frac{dt}{\ln t}$$

**等价形式**：
$$\pi(x) \sim \frac{x}{\ln x}$$

即：
$$\lim_{x \to \infty} \frac{\pi(x)}{x/\ln x} = 1$$

**证明方法**：

```text
方法：
1. 利用ζ函数
2. 证明ζ(s)在Re(s) = 1上无零点
3. 利用显式公式
4. 得到渐近估计

关键：
- 无零点在Re(s) = 1
- 这是欧拉工作的延续
```

## 六、现代发展：Riemann假设

### 6.1 Riemann的贡献

**Riemann（1859）**：

```text
论文：
《论小于给定值的素数个数》

贡献：
1. ζ函数解析延拓
2. 函数方程
3. 零点分布
4. Riemann假设
```

**函数方程**：

$$\zeta(s) = 2^s \pi^{s-1} \sin\left(\frac{\pi s}{2}\right) \Gamma(1-s) \zeta(1-s)$$

**意义**：
- 建立 $s$ 与 $1-s$ 的关系
- 解析延拓到全复平面
- 揭示对称性

---

### 6.2 Riemann假设

**假设（RH）**：

**Riemann假设**：
$\zeta(s)$ 的所有非平凡零点满足：
$$\text{Re}(s) = \frac{1}{2}$$

即：所有非平凡零点在临界线上。

**意义**：
- 如果成立，可改进素数分布估计
- 数论中的核心问题
- Clay Millennium问题之一
```

**与素数分布的关系**：

```text
显式公式：
π(x) = Li(x) + 误差项

误差项：
- 与ζ的零点相关
- RH ⟹ 最优误差估计
- O(x^(1/2) log x)
```

## 九、数学公式总结

### 核心公式

1. **欧拉乘积公式**：
   $$\zeta(s) = \sum_{n=1}^{\infty} \frac{1}{n^s} = \prod_{p \text{ 素数}} \frac{1}{1 - p^{-s}} \quad \text{（$s > 1$）}$$

2. **几何级数展开**：
   $$\frac{1}{1 - p^{-s}} = 1 + p^{-s} + p^{-2s} + \cdots$$

3. **对数形式**：
   $$\ln \zeta(s) = \sum_{p \text{ 素数}} \sum_{m=1}^{\infty} \frac{p^{-ms}}{m}$$

4. **调和级数发散**：
   $$\zeta(1) = \sum_{n=1}^{\infty} \frac{1}{n} = \infty$$

5. **素数无穷的分析证明**：
   $$\prod_{p \text{ 素数}} \frac{1}{1 - 1/p} = \infty$$

6. **素数计数函数**：
   $$\pi(x) = \#\{p \leq x : p \text{ 是素数}\}$$

7. **Gauss猜想**：
   $$\pi(x) \sim \frac{x}{\ln x}$$

8. **素数定理**：
   $$\pi(x) \sim \text{Li}(x) = \int_2^x \frac{dt}{\ln t}$$

9. **素数定理等价形式**：
   $$\lim_{x \to \infty} \frac{\pi(x)}{x/\ln x} = 1$$

10. **Riemann ζ函数函数方程**：
    $$\zeta(s) = 2^s \pi^{s-1} \sin\left(\frac{\pi s}{2}\right) \Gamma(1-s) \zeta(1-s)$$

11. **Riemann假设**：
    $$\text{Re}(s) = \frac{1}{2} \quad \text{（$\zeta(s)$ 的所有非平凡零点）}$$

12. **Dirichlet定理**：
    $$\gcd(a, d) = 1 \Rightarrow \text{算术级数 } a, a+d, a+2d, \ldots \text{ 中有无限多个素数}$$

13. **素数密度**：
    $$\text{密度} \approx \frac{1}{\ln x}$$

14. **显式公式**：
    $$\pi(x) = \text{Li}(x) - \sum_{\rho} \text{Li}(x^{\rho}) + \text{误差项}$$

15. **ζ函数的零点**：
    $$\zeta(s) = 0 \Rightarrow s = -2, -4, -6, \ldots \text{（平凡零点）或 } \text{Re}(s) = \frac{1}{2} \text{（假设）}$$
