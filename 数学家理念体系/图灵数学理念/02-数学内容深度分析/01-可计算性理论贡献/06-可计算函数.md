## 一、可计算函数的定义

### 1.1 图灵可计算函数

**定义**：

函数 $f: \mathbb{N}^k \to \mathbb{N}$ 是**图灵可计算的**，如果存在图灵机 $M$，使得对于任意输入 $(n_1, \ldots, n_k) \in \mathbb{N}^k$，$M$ 在有限步内停机并输出 $f(n_1, \ldots, n_k)$。

**形式化表述**：

设图灵机 $M = (Q, \Sigma, \Gamma, \delta, q_0, B, F)$，函数 $f$ 是图灵可计算的，当且仅当：

\[
\forall (n_1, \ldots, n_k) \in \mathbb{N}^k, M(\text{enc}(n_1, \ldots, n_k)) \text{ 停机且输出 } \text{enc}(f(n_1, \ldots, n_k))
\]

其中 $\text{enc}$ 是自然数的编码函数。

### 1.2 部分可计算函数

**定义**：

函数 $f: \mathbb{N}^k \rightharpoonup \mathbb{N}$ 是**部分图灵可计算的**，如果存在图灵机 $M$，使得：

- 如果 $f(n_1, \ldots, n_k)$ 有定义，则 $M$ 在输入 $(n_1, \ldots, n_k)$ 上停机并输出 $f(n_1, \ldots, n_k)$
- 如果 $f(n_1, \ldots, n_k)$ 无定义，则 $M$ 在输入 $(n_1, \ldots, n_k)$ 上不停机

**关系**：

- **可计算函数** ⊆ **部分可计算函数**
- 所有可计算函数都是部分可计算的
- 部分可计算函数包括可计算函数和不可计算函数

## 二、可计算函数的性质

### 2.1 可计算函数的封闭性

**复合**：

如果 $f: \mathbb{N}^k \to \mathbb{N}$ 和 $g_1, \ldots, g_k: \mathbb{N}^m \to \mathbb{N}$ 都是可计算的，则复合函数：

\[
h(x_1, \ldots, x_m) = f(g_1(x_1, \ldots, x_m), \ldots, g_k(x_1, \ldots, x_m))
\]

也是可计算的。

**原始递归**：

如果 $g: \mathbb{N}^k \to \mathbb{N}$ 和 $h: \mathbb{N}^{k+2} \to \mathbb{N}$ 都是可计算的，则原始递归定义的函数：

\[
f(0, x_1, \ldots, x_k) = g(x_1, \ldots, x_k)
\]
\[
f(n+1, x_1, \ldots, x_k) = h(n, f(n, x_1, \ldots, x_k), x_1, \ldots, x_k)
\]

也是可计算的。

**最小化**：

如果 $g: \mathbb{N}^{k+1} \to \mathbb{N}$ 是可计算的，且对于任意 $(x_1, \ldots, x_k)$，存在 $y$ 使得 $g(y, x_1, \ldots, x_k) = 0$，则最小化函数：

\[
f(x_1, \ldots, x_k) = \mu y [g(y, x_1, \ldots, x_k) = 0]
\]

也是可计算的。

### 2.2 可计算函数的等价性

**等价计算模型**：

以下计算模型定义的可计算函数是等价的：

- **图灵可计算函数**：由图灵机计算的函数
- **λ可计算函数**：由λ演算计算的函数
- **递归函数**：由递归函数定义的函数

**丘奇-图灵论题**：

所有"有效可计算"的函数都是图灵可计算的。

## 三、可计算函数的分类

### 3.1 原始递归函数

**定义**：

**原始递归函数**是通过以下方式定义的函数：

1. **基础函数**：
   - 零函数：$Z(n) = 0$
   - 后继函数：$S(n) = n+1$
   - 投影函数：$P_i^k(x_1, \ldots, x_k) = x_i$

2. **复合**：从已有函数通过复合得到新函数

3. **原始递归**：从已有函数通过原始递归得到新函数

**性质**：

- 所有原始递归函数都是可计算的
- 原始递归函数是**全函数**（对所有输入都有定义）
- 存在可计算函数不是原始递归的（如Ackermann函数）

### 3.2 一般递归函数

**定义**：

**一般递归函数**（μ递归函数）是通过以下方式定义的函数：

1. **基础函数**：零函数、后继函数、投影函数

2. **复合**：从已有函数通过复合得到新函数

3. **原始递归**：从已有函数通过原始递归得到新函数

4. **最小化**：从已有函数通过最小化得到新函数

**性质**：

- 所有一般递归函数都是可计算的
- 一般递归函数包括原始递归函数
- 一般递归函数 = 图灵可计算函数（等价性）

## 五、典型例题

### 5.1 例题1：证明加法函数是可计算的

**问题**：

证明加法函数 $f(x, y) = x + y$ 是可计算的。

**解答**：

**构造图灵机**：

设计图灵机 $M$ 计算 $f(x, y) = x + y$：

1. **输入编码**：输入为 $x$ 和 $y$ 的二进制表示（用分隔符分隔）
2. **计算过程**：
   - 从 $x$ 的表示中逐个读取数字
   - 将数字复制到输出带
   - 从 $y$ 的表示中逐个读取数字
   - 将数字追加到输出带
3. **输出**：输出 $x + y$ 的二进制表示

**形式化证明**：

由于图灵机 $M$ 可以在有限步内停机并输出 $x + y$，因此 $f$ 是图灵可计算的。

**复杂度分析**：

- **时间复杂度**：$O(\max(|x|, |y|))$，其中 $|x|$ 是 $x$ 的二进制表示长度
- **空间复杂度**：$O(\max(|x|, |y|))$
- **多项式时间**：时间复杂度是线性的，因此是多项式的

### 5.2 例题2：证明部分可计算函数的存在

**问题**：

证明存在部分可计算函数（在某些输入上未定义）。

**解答**：

**部分可计算函数**：

函数 $f: \mathbb{N} \to \mathbb{N}$ 是**部分可计算的**，如果存在图灵机 $M$，使得：

- 如果 $f(n)$ 有定义，则 $M(n)$ 停机并输出 $f(n)$
- 如果 $f(n)$ 未定义，则 $M(n)$ 不停机

**例子**：

考虑函数 $g: \mathbb{N} \to \mathbb{N}$：

\[
g(n) = \begin{cases}
1 & \text{如果存在 } m \text{ 使得 } m^2 = n \\
\text{未定义} & \text{否则}
\end{cases}
\]

**图灵机实现**：

构造图灵机 $M$ 计算 $g$：

1. 输入：$n$
2. 枚举所有 $m = 0, 1, 2, \ldots$
3. 对每个 $m$，检查 $m^2 = n$
4. 如果找到 $m$ 使得 $m^2 = n$，输出 $1$ 并停机
5. 如果找不到，继续枚举（不停机）

**结论**：

$g$ 是部分可计算的，但不是全函数（在某些输入上未定义）。

### 5.3 例题3：分析可计算函数的闭包性质

**问题**：

分析可计算函数的闭包性质，特别是复合运算的闭包性。

**解答**：

**闭包性质**：

可计算函数在复合运算下封闭：

- **复合运算**：如果 $f$ 和 $g$ 是可计算的，则 $f \circ g$ 也是可计算的
- **递归运算**：如果 $f$ 和 $g$ 是可计算的，则递归定义的函数也是可计算的
- **最小化运算**：如果 $f$ 是可计算的，则最小化定义的函数也是可计算的

**形式化表述**：

设 $f: \mathbb{N} \to \mathbb{N}$ 和 $g: \mathbb{N} \to \mathbb{N}$ 是可计算的，则：

- **复合**：$h = f \circ g$ 是可计算的
- **构造**：给定图灵机 $M_f$ 和 $M_g$，构造图灵机 $M_h$：
  1. 输入：$n$
  2. 使用 $M_g$ 计算 $g(n)$
  3. 使用 $M_f$ 计算 $f(g(n))$
  4. 输出：$f(g(n))$

**证明**：

$M_h$ 的计算过程：

1. $M_h(n)$ 首先运行 $M_g(n)$，得到 $g(n)$
2. 然后运行 $M_f(g(n))$，得到 $f(g(n))$
3. 输出 $f(g(n)) = h(n)$

因此，$h$ 是图灵可计算的。

**意义**：

- **函数构造**：闭包性质允许从简单函数构造复杂函数
- **函数类**：可计算函数形成函数类
- **理论结构**：闭包性质揭示了可计算函数的结构

## 六、跨主题关联小结

### 6.1 可计算函数与图灵机的关联

**核心关联**：

可计算函数由图灵机定义，图灵机是可计算函数的计算模型。

**数学结构分析**：

- **定义关系**：可计算函数由图灵机定义
- **等价性**：图灵可计算函数与其他计算模型等价
- **统一性**：图灵机统一了可计算函数的概念

### 6.2 可计算函数与递归函数的关联

**核心关联**：

可计算函数与递归函数等价，两者从不同角度定义可计算性。

**数学结构分析**：

- **等价性**：图灵可计算函数 = 一般递归函数
- **互补性**：两者从不同角度定义可计算性
- **统一性**：两者统一了可计算性的概念
