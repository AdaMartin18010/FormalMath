## 📚 核心内容方向

### 1. 陈类（Chern Classes）

- **历史背景**: 1940年代发展
- **数学内容**: 向量丛的拓扑不变量
- **应用**: 代数几何、拓扑、物理

### 2. 纤维丛理论

- **主丛与联络**: 规范理论的基础
- **曲率**: 几何不变量
- **应用**: 规范理论、物理

### 3. 整体几何

- **流形上的几何**: 整体性质
- **几何不变量**: 拓扑与几何的联系
- **应用**: 现代几何学
