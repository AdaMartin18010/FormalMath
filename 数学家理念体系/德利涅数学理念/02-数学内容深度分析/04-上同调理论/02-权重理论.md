## 一、权重理论的基础

### 1.1 权重的定义

**权重**：

上同调群H^i_{ét}(X, ℚ_ℓ)的权重定义为Frobenius特征值的绝对值。

**定义**：

设X是有限域F_q上的代数簇，F是Frobenius自同态。对于上同调群H^i_{ét}(X, ℚ_ℓ)的特征值λ，权重定义为：

```
w(λ) = log_q(|λ|)
```

**纯权i**：

如果所有特征值λ满足|λ| = q^{i/2}，则称H^i_{ét}(X, ℚ_ℓ)是纯权i的。

**性质**：

- **权重有界**：对于光滑射影代数簇，|λ| = q^{i/2}（Riemann假设）
- **几何不变量**：权重是几何不变量，不依赖于ℓ的选择
- **应用广泛**：在代数几何、数论中有广泛应用

**意义**：

权重理论描述了上同调群的权重结构，为韦伊猜想的证明提供了基础。

### 1.2 权重滤过

**权重滤过**：

权重滤过W_*是上同调群上的递增滤过，提供了权分解。

**定义**：

对于上同调群H^i_{ét}(X, ℚ_ℓ)，权重滤过W_*定义为：

```
W_0 H^i ⊆ W_1 H^i ⊆ ... ⊆ W_n H^i = H^i
```

其中W_j H^i由权重≤j的所有元素组成。

**权分解**：

权重滤过提供了权分解：

```
H^i = ⊕_j Gr^W_j H^i
```

其中Gr^W_j H^i = W_j H^i / W_{j-1} H^i是纯权j的。

**性质**：

- **递增滤过**：W_j H^i ⊆ W_{j+1} H^i
- **权分解**：H^i = ⊕_j Gr^W_j H^i
- **函子性**：权重滤过具有函子性，可以应用于各种几何对象

**意义**：

权重滤过描述了上同调群的权重结构，为混合Hodge结构提供了基础。
