## 一、经典Hodge理论回顾

### 1.1 Hodge分解

**经典Hodge分解**：

紧Kähler流形X

Hodge分解：

```
H^n(X, C) = ⊕_{p+q=n} H^{p,q}(X)
```

其中：

```
H^{p,q}(X) = H^q(X, Ω^p)
```

**意义**：

- 几何结构
- 上同调结构
- 应用广泛

### 1.2 Hodge结构

**Hodge结构**：

有理向量空间V

Hodge结构：

- 分解：V_C = ⊕_{p+q=n} H^{p,q}
- 性质：H^{p,q} = H^{q,p}（共轭对称）

**意义**：

- 几何结构
- 上同调结构
- 应用广泛

## 二、混合Hodge结构的动机

### 2.1 经典理论的局限性

**局限性**：

- 仅适用于光滑紧Kähler流形
- 不能处理奇异代数簇
- 不能处理非紧流形

**问题**：

如何推广Hodge理论到更一般的情况？

### 2.2 奇异代数簇的需求

**需求**：

- 处理奇异代数簇
- 统一上同调理论
- 应用广泛

**德利涅的突破**：

通过引入权重滤过和Hodge滤过，建立了混合Hodge理论。

## 三、混合Hodge结构的定义

### 3.1 权重滤过

**权重滤过**：

有理向量空间V

权重滤过：

```
W_0 V ⊆ W_1 V ⊆ ... ⊆ W_n V = V
```

**性质**：

- 递增滤过
- 权分解
- 应用广泛

### 3.2 Hodge滤过

**Hodge滤过**：

复向量空间V_C

Hodge滤过：

```
F^0 V_C ⊇ F^1 V_C ⊇ ... ⊇ F^n V_C = 0
```

**性质**：

- 递减滤过
- Hodge分解
- 应用广泛

### 3.3 相容性条件

**相容性条件**：

混合Hodge结构要求权重滤过和Hodge滤过满足相容性条件。

**条件**：

- 权分解与Hodge分解相容
- 函子性
- 应用广泛

## 四、基本性质

### 4.1 权分解

**权分解**：

混合Hodge结构具有权分解：

```
Gr^W_i V = W_i V / W_{i-1} V
```

**性质**：

- 权分解
- Hodge结构
- 应用广泛

### 4.2 Hodge分解

**Hodge分解**：

混合Hodge结构具有Hodge分解：

```
Gr^W_i V_C = ⊕_{p+q=i} H^{p,q}
```

**性质**：

- Hodge分解
- 权分解
- 应用广泛

### 4.3 函子性

**函子性**：

混合Hodge结构具有函子性，可以应用于各种几何对象。

**应用**：

- 代数簇
- 上同调理论
- 应用广泛
