## 一、经典Hodge理论回顾

### 1.1 Hodge分解

**经典Hodge分解**：

紧Kähler流形X

Hodge分解：

```
H^n(X, C) = ⊕_{p+q=n} H^{p,q}(X)
```

其中：

```
H^{p,q}(X) = H^q(X, Ω^p)
```

**意义**：

- 几何结构
- 上同调结构
- 应用广泛

### 1.2 Hodge结构

**Hodge结构**：

有理向量空间V

Hodge结构：

- 分解：V_C = ⊕_{p+q=n} H^{p,q}
- 性质：H^{p,q} = H^{q,p}（共轭对称）

**意义**：

- 几何结构
- 上同调结构
- 应用广泛

## 二、混合Hodge结构的动机

### 2.1 经典理论的局限性

**局限性**：

经典Hodge理论有以下局限性：

- **仅适用于光滑紧Kähler流形**：
  - 需要流形是光滑的
  - 需要流形是紧的
  - 需要流形有Kähler结构

- **不能处理奇异代数簇**：
  - 奇异点破坏了Hodge分解
  - 无法直接应用经典Hodge理论
  - 需要新的理论框架

- **不能处理非紧流形**：
  - 非紧流形的上同调群可能不是有限维的
  - Hodge分解可能不成立
  - 需要推广的理论

**问题**：

如何推广Hodge理论到更一般的情况？这是德利涅面临的核心问题。

**德利涅的解决方案**：

通过引入权重滤过和Hodge滤过，建立混合Hodge结构，推广了经典Hodge理论。

### 2.2 奇异代数簇的需求

**需求**：

- 处理奇异代数簇
- 统一上同调理论
- 应用广泛

**德利涅的突破**：

通过引入权重滤过和Hodge滤过，建立了混合Hodge理论。

## 三、混合Hodge结构的定义

### 3.1 权重滤过

**权重滤过**：

设V是有理向量空间。权重滤过W_*是V上的递增滤过：

```
W_0 V ⊆ W_1 V ⊆ ... ⊆ W_n V = V
```

**权分解**：

权重滤过提供了权分解：

```
Gr^W_i V = W_i V / W_{i-1} V
```

**性质**：

- **递增滤过**：W_i V ⊆ W_{i+1} V
- **权分解**：V = ⊕_i Gr^W_i V
- **函子性**：权重滤过具有函子性，可以应用于各种几何对象

**意义**：

权重滤过描述了上同调群的权重结构，为混合Hodge结构提供了基础。

### 3.2 Hodge滤过

**Hodge滤过**：

设V_C是复向量空间。Hodge滤过F^*是V_C上的递减滤过：

```
F^0 V_C ⊇ F^1 V_C ⊇ ... ⊇ F^n V_C = 0
```

**Hodge分解**：

Hodge滤过提供了Hodge分解：

```
Gr^W_i V_C = ⊕_{p+q=i} H^{p,q}
```

其中H^{p,q}满足H^{p,q} = H^{q,p}（共轭对称）。

**性质**：

- **递减滤过**：F^i V_C ⊇ F^{i+1} V_C
- **Hodge分解**：Gr^W_i V_C = ⊕_{p+q=i} H^{p,q}
- **函子性**：Hodge滤过具有函子性，可以应用于各种几何对象

**意义**：

Hodge滤过描述了上同调群的Hodge结构，为混合Hodge结构提供了基础。

### 3.3 相容性条件

**相容性条件**：

混合Hodge结构要求权重滤过W_*和Hodge滤过F^*满足相容性条件。

**条件**：

对于每个i，Gr^W_i V_C上的Hodge滤过F^*诱导的Hodge分解：

```
Gr^W_i V_C = ⊕_{p+q=i} H^{p,q}
```

满足：

- **共轭对称**：H^{p,q} = H^{q,p}
- **权分解与Hodge分解相容**：权分解和Hodge分解在Gr^W_i V_C上相容
- **函子性**：混合Hodge结构具有函子性

**混合Hodge结构的定义**：

有理向量空间V上的混合Hodge结构由以下数据组成：

1. 权重滤过W_*：递增滤过W_0 V ⊆ W_1 V ⊆ ... ⊆ W_n V = V
2. Hodge滤过F^*：在V_C上的递减滤过F^0 V_C ⊇ F^1 V_C ⊇ ... ⊇ F^n V_C = 0
3. 相容性条件：权重滤过和Hodge滤过满足上述相容性条件

**意义**：

混合Hodge结构统一了权重结构和Hodge结构，为研究奇异代数簇的上同调提供了工具。
