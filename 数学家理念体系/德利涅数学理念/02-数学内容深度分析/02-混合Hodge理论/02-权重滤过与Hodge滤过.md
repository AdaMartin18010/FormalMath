## 一、权重滤过

### 1.1 权重滤过的定义

**权重滤过**：

设V是有理向量空间。权重滤过W_*是V上的递增滤过：

```
W_0 V ⊆ W_1 V ⊆ ... ⊆ W_n V = V
```

**权分解**：

权重滤过提供了权分解：

```
Gr^W_i V = W_i V / W_{i-1} V
```

**性质**：

- **递增滤过**：W_i V ⊆ W_{i+1} V，从W_0 V到W_n V = V
- **权分解**：V = ⊕_i Gr^W_i V，每个Gr^W_i V是纯权i的
- **函子性**：权重滤过具有函子性，可以应用于各种几何对象

**意义**：

权重滤过描述了上同调群的权重结构，为混合Hodge结构提供了基础。在韦伊猜想的证明中，权重滤过起到了关键作用。

### 1.2 权重滤过的性质

**性质**：

- 递增滤过
- 权分解
- 函子性

**递增滤过**：

权重滤过是递增的，从W_0 V到W_n V = V。这意味着：

- W_0 V是最小的权重子空间
- W_n V = V是完整的向量空间
- 中间的W_i V逐步增加权重

**权分解**：

权重滤过提供了权分解：

```
V = ⊕_i Gr^W_i V
```

其中Gr^W_i V = W_i V / W_{i-1} V是纯权i的。这个分解将混合权的向量空间分解为纯权的子空间。

**函子性**：

权重滤过具有函子性，这意味着：

- 对于几何对象之间的映射，权重滤过保持
- 可以应用于各种几何对象（代数簇、上同调群等）
- 为统一上同调理论提供了基础

## 二、Hodge滤过

### 2.1 Hodge滤过的定义

**Hodge滤过**：

设V_C是复向量空间。Hodge滤过F^*是V_C上的递减滤过：

```
F^0 V_C ⊇ F^1 V_C ⊇ ... ⊇ F^n V_C = 0
```

**Hodge分解**：

Hodge滤过提供了Hodge分解：

```
Gr^W_i V_C = ⊕_{p+q=i} H^{p,q}
```

其中H^{p,q}满足H^{p,q} = H^{q,p}（共轭对称）。

**性质**：

- **递减滤过**：F^i V_C ⊇ F^{i+1} V_C，从F^0 V_C = V_C到F^n V_C = 0
- **Hodge分解**：Gr^W_i V_C = ⊕_{p+q=i} H^{p,q}，提供了(p,q)型分解
- **函子性**：Hodge滤过具有函子性，可以应用于各种几何对象

**意义**：

Hodge滤过描述了上同调群的Hodge结构，为混合Hodge结构提供了基础。在混合Hodge理论中，Hodge滤过与权重滤过结合，提供了统一的上同调理论。

### 2.2 Hodge滤过的性质

**性质**：

- 递减滤过
- Hodge分解
- 函子性

**递减滤过**：

Hodge滤过是递减的，从F^0 V_C到F^n V_C。

**Hodge分解**：

Hodge滤过提供了Hodge分解，Gr^W_i V_C = ⊕_{p+q=i} H^{p,q}。

**函子性**：

Hodge滤过具有函子性，可以应用于各种几何对象。

## 三、相容性条件

### 3.1 相容性条件的定义

**相容性条件**：

混合Hodge结构要求权重滤过W_*和Hodge滤过F^*满足相容性条件。

**条件**：

对于每个i，Gr^W_i V_C上的Hodge滤过F^*诱导的Hodge分解：

```
Gr^W_i V_C = ⊕_{p+q=i} H^{p,q}
```

必须满足：

- **共轭对称**：H^{p,q} = H^{q,p}
- **权分解与Hodge分解相容**：权分解和Hodge分解在Gr^W_i V_C上相容
- **函子性**：相容性条件具有函子性

**技术细节**：

相容性条件确保了：

- 权重滤过和Hodge滤过可以同时存在
- 两种滤过在Gr^W_i V_C上产生一致的分解
- 混合Hodge结构可以应用于各种几何对象

**意义**：

相容性条件是混合Hodge结构的核心。它统一了权重结构和Hodge结构，为研究奇异代数簇的上同调提供了工具。

### 3.2 相容性条件的应用

**应用**：

- 混合Hodge结构的构造
- 上同调群的研究
- 几何不变量的计算

**构造**：

相容性条件用于构造混合Hodge结构。

**研究**：

相容性条件用于研究上同调群的结构。

**计算**：

相容性条件用于计算几何不变量。
