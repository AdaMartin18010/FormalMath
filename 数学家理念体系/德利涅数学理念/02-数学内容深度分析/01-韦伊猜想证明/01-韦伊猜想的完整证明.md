## 一、韦伊猜想的陈述

### 1.1 原始猜想

**韦伊猜想**（1949）：

有限域F_q上的光滑射影代数簇X

Zeta函数：

```
ζ_X(s) = exp(∑_{n=1}^∞ N_n T^n/n)
```

其中：

- N_n = |X(F_{q^n})|（F_{q^n}值点个数）

**猜想**：

1. **有理性**：ζ_X(s)是有理函数
2. **函数方程**：满足函数方程
3. **Riemann假设**：零点在Re(s) = 1/2, 3/2, ..., (2d-1)/2上

### 1.2 几何解释

**几何意义**：

Zeta函数：

- 编码点计数
- 几何信息
- 算术性质

Riemann假设：

- 特征值有界
- 几何约束
- 算术应用

## 二、格洛腾迪克的框架

### 2.1 étale上同调框架

**上同调方法**：

格洛腾迪克的想法：

- 用上同调计算Zeta函数
- étale上同调
- Frobenius作用

**方法**：

```
ζ_X(s) = ∏_{i=0}^{2d} P_i(T)^{(-1)^{i+1}}
```

其中：

```
P_i(T) = det(1 - F T | H^i_{ét}(X, ℚ_ℓ))
```

**ℓ进上同调**：

H^i_{ét}(X, ℚ_ℓ)：

- ℚ_ℓ向量空间
- 有限维
- Frobenius F作用

**性质**：

- 特征值有界（|λ| = q^{i/2}）
- 几何不变量

### 2.2 证明策略

**证明策略**：

1. **有理性**：通过上同调群有限维性
2. **函数方程**：通过Poincaré对偶
3. **Riemann假设**：通过权重理论和纯性定理

## 三、德利涅的证明

### 3.1 权重理论

**权重**：

上同调群的权重：

- 纯权i：特征值|λ| = q^{i/2}
- 混合权：不同权重

**纯性**：

- 光滑射影：纯
- 一般：混合

### 3.2 纯性定理

**纯性定理**：

有限域F_q上的光滑射影代数簇X

上同调：
H^i_{ét}(X, ℚ_ℓ)

特征值：
对所有特征值λ：
|λ| = q^{i/2}

## 四、证明的技术细节

### 4.1 有理性

**Zeta函数的有理性**：

通过上同调：

- 上同调群有限维
- Frobenius特征多项式
- 有理性自然

**证明**：

- 使用上同调
- 有限维性
- 特征多项式

### 4.2 函数方程

**函数方程**：

Poincaré对偶：
H^i ≅ H^{2d-i}(d)

函数方程：

```
ζ_X(1/s) = ±q^{dχ/2} s^{-χ} ζ_X(s)
```

其中χ是Euler特征

### 4.3 Riemann假设

**Riemann假设（有限域）**：

Deligne的证明（1974）：

- 使用权重理论
- Hard Lefschetz
- 纯性定理

**结果**：

- 特征值|λ| = q^{i/2}
- Riemann假设成立
