# 高斯-分布与中心极限定理

**msc_primary**: 60F05
**msc_secondary**: 62E20, 60A05, 01A55

---

## 简介

正态分布（Normal distribution），又称高斯分布（Gaussian distribution），是概率论与统计学中最重要、最美丽的概率分布。其密度函数的钟形曲线不仅是自然界和社会现象中无数随机变量的近似模型，更是中心极限定理（Central Limit Theorem, CLT）这一概率论基石定理的极限形式。高斯在1809年研究天体测量误差时系统引入了这一分布，并发展了最小二乘法，从而为现代统计学和误差理论奠定了基础。

本文将严格推导正态分布的数学形式，证明中心极限定理的几种经典版本，阐述最小二乘法的统计理论基础，并探讨高斯分布在现代科学中的广泛应用。高斯的概率论工作与[拉普拉斯数学理念](../拉普拉斯数学理念/)中的误差理论、[帕斯卡数学理念](../帕斯卡数学理念/)中的概率起源形成了概率统计发展的关键链条。

## 第一节：正态分布的推导

### 1.1 误差理论的背景

在天文观测和大地测量中，同一量的多次测量往往得到略有不同的结果。高斯面临的核心问题是：如何从这些带有误差的观测值中估计"真实值"？

高斯的假设是：测量误差 $\epsilon$ 是大量独立微小因素叠加的结果。根据这一直觉，他推导出了误差的概率密度函数应具有的形式。

### 1.2 最大似然推导

**定理 1.1（正态分布的推导）**：设 $\phi(x)$ 为对称的概率密度函数（即 $\phi(-x) = \phi(x)$），满足：
1. 对独立同分布的误差 $\epsilon_1, \ldots, \epsilon_n$，样本均值 $\bar{x} = \frac{1}{n}\sum x_i$ 是真实值 $\mu$ 的最大似然估计；
2. $\phi$ 二次可微。

则 $\phi(x)$ 必为以下形式：
$$\phi(x) = \frac{1}{\sqrt{2\pi}\sigma} \exp\left(-\frac{x^2}{2\sigma^2}\right).$$

*证明思路*：
1. 似然函数为 $L(\mu) = \prod_{i=1}^n \phi(x_i - \mu)$。
2. 最大似然条件要求 $\frac{d\log L}{d\mu}\big|_{\mu=\bar{x}} = 0$，即
   $$\sum_{i=1}^n \frac{\phi'(x_i - \bar{x})}{\phi(x_i - \bar{x})} = 0.$$
3. 令 $\psi(x) = \phi'(x)/\phi(x)$。上式变为 $\sum \psi(x_i - \bar{x}) = 0$。
4. 取 $n=2$，$x_1 = a$，$x_2 = -a$，得 $\psi(a) + \psi(-a) = 0$。由对称性，$\psi$ 为奇函数。
5. 取 $n=3$，利用 $\psi$ 的奇函数性质，可证明 $\psi$ 必须是线性函数：$\psi(x) = cx$。
6. 解微分方程 $\phi'(x)/\phi(x) = cx$ 得 $\phi(x) = Ae^{cx^2/2}$。由归一化条件和 $\phi$ 为密度函数，得 $c < 0$，最终得到正态密度。$\square$

### 1.3 正态分布的基本性质

**定义 1.1（正态分布）**：随机变量 $X$ 服从参数为 $\mu$ 和 $\sigma^2$ 的正态分布，记作 $X \sim N(\mu, \sigma^2)$，如果其概率密度函数为
$$f(x) = \frac{1}{\sqrt{2\pi}\sigma} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right), \quad x \in \mathbb{R}.$$

**定理 1.2（正态分布的性质）**：
1. $E[X] = \mu$，$\text{Var}(X) = \sigma^2$。
2. 若 $X \sim N(\mu, \sigma^2)$，则 $aX + b \sim N(a\mu + b, a^2\sigma^2)$。
3. 若 $X_1 \sim N(\mu_1, \sigma_1^2)$ 和 $X_2 \sim N(\mu_2, \sigma_2^2)$ 独立，则 $X_1 + X_2 \sim N(\mu_1 + \mu_2, \sigma_1^2 + \sigma_2^2)$。

## 第二节：中心极限定理

### 2.1 Lindeberg-Lévy CLT

**定理 2.1（Lindeberg-Lévy 中心极限定理）**：设 $X_1, X_2, \ldots$ 为独立同分布随机变量，$E[X_i] = \mu$，$\text{Var}(X_i) = \sigma^2 < \infty$。令 $S_n = X_1 + \cdots + X_n$。则
$$\frac{S_n - n\mu}{\sigma\sqrt{n}} \xrightarrow{d} N(0, 1) \quad \text{当 } n \to \infty.$$

*证明思路（特征函数法）*：
1. 设 $\varphi(t) = E[e^{itX_1}]$ 为 $X_1$ 的特征函数。
2. 标准化变量的特征函数为
   $$\varphi_n(t) = E\left[\exp\left(it\frac{S_n - n\mu}{\sigma\sqrt{n}}\right)\right] = \left[\varphi\left(\frac{t}{\sigma\sqrt{n}}\right)e^{-it\mu/(\sigma\sqrt{n})}\right]^n.$$
3. 对 $\varphi$ 在0处作Taylor展开：$\varphi(s) = 1 + is\mu - \frac{s^2(\sigma^2+\mu^2)}{2} + o(s^2)$。
4. 代入并取对数：
   $$\log \varphi_n(t) = n\log\left(1 - \frac{t^2}{2n} + o(1/n)\right) \to -\frac{t^2}{2}.$$
5. 因此 $\varphi_n(t) \to e^{-t^2/2}$，即标准正态的特征函数。由Lévy连续性定理，得证。$\square$

### 2.2 Lyapunov CLT

**定理 2.2（Lyapunov 中心极限定理）**：设 $X_1, X_2, \ldots$ 为独立随机变量（不必同分布），$E[X_i] = \mu_i$，$\text{Var}(X_i) = \sigma_i^2$。令 $s_n^2 = \sum_{i=1}^n \sigma_i^2$。若存在 $\delta > 0$ 使得
$$\frac{1}{s_n^{2+\delta}} \sum_{i=1}^n E[|X_i - \mu_i|^{2+\delta}] \to 0 \quad \text{当 } n \to \infty,$$
则
$$\frac{1}{s_n}\sum_{i=1}^n (X_i - \mu_i) \xrightarrow{d} N(0, 1).$$

Lyapunov条件比Lindeberg条件更易验证，是证明非独立同分布情形CLT的常用工具。

### 2.3 多维CLT

**定理 2.3（多维中心极限定理）**：设 $\mathbf{X}_1, \mathbf{X}_2, \ldots$ 为独立同分布的 $d$ 维随机向量，$E[\mathbf{X}_i] = \boldsymbol{\mu}$，$\text{Cov}(\mathbf{X}_i) = \Sigma$。则
$$\frac{1}{\sqrt{n}}\sum_{i=1}^n (\mathbf{X}_i - \boldsymbol{\mu}) \xrightarrow{d} N(\mathbf{0}, \Sigma).$$

## 第三节：最小二乘法

### 3.1 高斯的原始推导

1809年，高斯在《天体运动论》（*Theoria Motus Corporum Coelestium*）中发表了他对最小二乘法的系统处理。高斯证明了：在误差服从正态分布的假设下，最小二乘估计等价于最大似然估计。

**定理 3.1（最小二乘 = 最大似然）**：设线性模型为
$$y_i = \mathbf{x}_i^T\boldsymbol{\beta} + \epsilon_i, \quad i = 1, \ldots, n,$$
其中 $\epsilon_i \overset{iid}{\sim} N(0, \sigma^2)$。则 $\boldsymbol{\beta}$ 的最大似然估计为
$$\hat{\boldsymbol{\beta}} = \arg\min_{\boldsymbol{\beta}} \sum_{i=1}^n (y_i - \mathbf{x}_i^T\boldsymbol{\beta})^2.$$

*证明*：似然函数为
$$L(\boldsymbol{\beta}, \sigma^2) = (2\pi\sigma^2)^{-n/2}\exp\left(-\frac{1}{2\sigma^2}\sum_{i=1}^n (y_i - \mathbf{x}_i^T\boldsymbol{\beta})^2\right).$$
最大化 $L$ 等价于最小化残差平方和。$\square$

### 3.2 Gauss-Markov定理

**定理 3.2（Gauss-Markov定理）**：在线性模型中，若误差满足 $E[\boldsymbol{\epsilon}] = \mathbf{0}$，$\text{Cov}(\boldsymbol{\epsilon}) = \sigma^2 I$（不必正态），则最小二乘估计 $\hat{\boldsymbol{\beta}}$ 是所有线性无偏估计中方差最小的（BLUE, Best Linear Unbiased Estimator）。

*证明思路*：设 $\tilde{\boldsymbol{\beta}} = C\mathbf{y}$ 为任意线性无偏估计。由无偏性，$CX = I$。计算协方差矩阵：
$$\text{Cov}(\tilde{\boldsymbol{\beta}}) = \sigma^2 CC^T = \sigma^2[(C - (X^TX)^{-1}X^T)(C - (X^TX)^{-1}X^T)^T + (X^TX)^{-1}].$$
因第一项半正定，故 $\text{Cov}(\tilde{\boldsymbol{\beta}}) \geq \sigma^2(X^TX)^{-1} = \text{Cov}(\hat{\boldsymbol{\beta}})$。$\square$

## 第四节：特征函数与Lévy连续性定理

### 4.1 特征函数的定义

**定义 4.1（特征函数）**：随机变量 $X$ 的**特征函数**定义为
$$\varphi_X(t) = E[e^{itX}] = \int_{-\infty}^{\infty} e^{itx}\,dF_X(x)$$

**定理 4.1（特征函数的唯一性）**：两个随机变量具有相同的分布，当且仅当它们具有相同的特征函数。

### 4.2 Lévy连续性定理

**定理 4.2（Lévy连续性定理）**：设 $\{X_n\}$ 为随机变量序列，$\varphi_n$ 为其特征函数。若 $\varphi_n(t) \to \varphi(t)$ 对所有 $t$ 成立，且 $\varphi$ 在 $t=0$ 处连续，则 $\varphi$ 是某个随机变量 $X$ 的特征函数，且 $X_n \xrightarrow{d} X$。

*证明思路*：利用Helly选择定理证明分布函数的 tightness，然后证明子序列收敛到以 $\varphi$ 为特征函数的分布。$\square$

### 4.3 Lindeberg条件的证明

利用特征函数可以给出Lindeberg-Lévy CLT的另一种证明：

**定理 4.3（Lindeberg条件）**：设 $\{X_n\}$ 为独立随机变量序列，$E[X_k] = 0$，$\text{Var}(X_k) = \sigma_k^2$。令 $s_n^2 = \sum_{k=1}^n \sigma_k^2$。若Lindeberg条件
$$\frac{1}{s_n^2}\sum_{k=1}^n E[X_k^2 \mathbf{1}_{|X_k| > \epsilon s_n}] \to 0$$
对所有 $\epsilon > 0$ 成立，则
$$\frac{1}{s_n}\sum_{k=1}^n X_k \xrightarrow{d} N(0,1)$$

## 第五节：正态分布在现代科学中的应用

### 4.1 统计推断

正态分布是经典统计推断的基础。假设检验、置信区间、回归分析、方差分析（ANOVA）等统计方法大多建立在正态假设或CLT近似之上。

### 4.2 信号处理与通信

高斯白噪声是通信理论和信号处理中的标准噪声模型。加性高斯白噪声（AWGN）信道是信息论中最基本的研究对象。

### 4.3 量子力学

在量子力学中，谐振子的基态和相干态都具有高斯型波函数。量子场论中的自由场论也大量涉及高斯积分。

### 4.4 金融数学

Black-Scholes期权定价模型假设股票价格服从几何布朗运动，其对数收益服从正态分布。风险价值（VaR）计算通常假设资产收益为正态分布。

## 第五节：现代概率论与统计物理

### 5.1 大偏差理论

**定理 5.1（Cramér大偏差）**：设 $\{X_i\}$ 为i.i.d.随机变量，均值 $\mu$，矩生成函数有限。则对 $a > \mu$：
$$\lim_{n \to \infty} \frac{1}{n} \log P\left(\frac{S_n}{n} > a\right) = -I(a)$$
其中 $I(a) = \sup_\theta (\theta a - \log \mathbb{E}[e^{\theta X}])$ 为速率函数。

### 5.2 高斯自由场

**定义 5.1（高斯自由场）**：设 $D \subset \mathbb{R}^d$ 为有界区域。高斯自由场是在Sobolev空间 $H^{-1}(D)$ 上的高斯测度，其协方差核为Green函数。

高斯自由场在共形场论和随机几何（Liouville量子引力）中发挥核心作用。

### 5.3 随机矩阵理论

**定理 5.2（Wigner半圆律）**：设 $M_n$ 为 $n \times n$ Wigner随机矩阵（上三角元素为i.i.d.，方差 $1/n$）。则其经验谱分布在 $n \to \infty$ 时收敛于半圆分布：
$$\rho(x) = \frac{1}{2\pi}\sqrt{4 - x^2} \cdot \mathbf{1}_{[-2,2]}(x)$$

随机矩阵理论将高斯分布推广到矩阵值随机变量，在量子混沌和深度学习中有重要应用。

## 第六节：随机过程与随机分析

### 6.1 Brown运动与Wiener过程

**定义 6.1（标准Brown运动）**：随机过程 $\{W_t\}_{t \geq 0}$ 称为**标准Brown运动**，如果：
1. $W_0 = 0$；
2. 增量独立；
3. $W_t - W_s \sim N(0, t-s)$（$t > s$）；
4. 样本路径连续。

**定理 6.1（Lévy刻画）**：连续局部鞅 $M$ 是Brown运动当且仅当其二次变差 $\langle M \rangle_t = t$。

### 6.2 Itô积分与随机微分方程

**定义 6.2（Itô积分）**：对适应过程 $H$，Itô积分定义为
$$\int_0^T H_s \, dW_s = \lim_{n \to \infty} \sum_{i=0}^{n-1} H_{t_i}(W_{t_{i+1}} - W_{t_i})$$

**定理 6.2（Itô公式）**：设 $X_t$ 为Itô过程，$f \in C^{2,1}$。则
$$df(t, X_t) = \frac{\partial f}{\partial t}dt + \frac{\partial f}{\partial x}dX_t + \frac{1}{2}\frac{\partial^2 f}{\partial x^2}d\langle X \rangle_t$$

### 6.3 Black-Scholes模型

**定理 6.3（Black-Scholes公式）**：在风险中性测度下，欧式看涨期权价格为
$$C(S, t) = S\Phi(d_1) - Ke^{-r(T-t)}\Phi(d_2)$$
其中
$$d_1 = \frac{\ln(S/K) + (r + \sigma^2/2)(T-t)}{\sigma\sqrt{T-t}}$$
$$d_2 = d_1 - \sigma\sqrt{T-t}$$

## 第七节：统计推断与机器学习

### 7.1 最大似然估计

**定义 7.1（MLE）**：给定样本 $x_1, \ldots, x_n$，**最大似然估计**为
$$\hat{\theta}_{\text{MLE}} = \arg\max_\theta \prod_{i=1}^n f(x_i; \theta)$$

**定理 7.1**：在正则条件下，MLE是渐近正态的：
$$\sqrt{n}(\hat{\theta}_{\text{MLE}} - \theta_0) \xrightarrow{d} N(0, I(\theta_0)^{-1})$$

### 7.2 高斯过程

**定义 7.2（高斯过程）**：随机过程 $\{f(x)\}_{x \in \mathcal{X}}$ 称为**高斯过程**，如果对任意有限点集，$(f(x_1), \ldots, f(x_n))$ 服从多元高斯分布。

高斯过程是贝叶斯优化的核心工具，用于黑盒函数的全局优化。

### 7.3 变分自编码器

在变分自编码器（VAE）中，隐变量 $z$ 的先验通常设为标准高斯分布 $p(z) = N(0, I)$。编码器学习近似后验 $q_\phi(z|x)$，解码器学习似然 $p_\theta(x|z)$。

## 第八节：信息论与高斯信道

### 8.1 高斯信道容量

**定理 8.1（Shannon）**：带限高斯信道的容量为
$$C = W \log_2\left(1 + \frac{P}{N_0 W}\right)$$
其中 $W$ 为带宽，$P$ 为信号功率，$N_0$ 为噪声功率谱密度。

### 8.2 率失真理论

**定理 8.2（Shannon率失真）**：对高斯源 $X \sim N(0, \sigma^2)$，在均方误差失真下的率失真函数为
$$R(D) = \begin{cases} \frac{1}{2}\log_2\frac{\sigma^2}{D} & 0 \leq D \leq \sigma^2 \\ 0 & D > \sigma^2 \end{cases}$$

### 8.3 压缩感知

**定理 8.3（压缩感知）**：若 $s$-稀疏信号 $x \in \mathbb{R}^N$ 通过满足RIP条件的随机测量矩阵 $\Phi \in \mathbb{R}^{M \times N}$（$M \ll N$）获取观测 $y = \Phi x$，则 $x$ 可以通过 $L_1$ 最小化精确恢复。

高斯随机矩阵以高概率满足RIP条件。

## 历史背景

高斯对正态分布和最小二乘法的研究始于1790年代。当时，他参与了对小行星谷神星（Ceres）轨道的计算。意大利天文学家Piazzi在1801年观测到了谷神星，但很快失去了它的踪迹。高斯利用最小二乘法从有限的观测数据中精确计算了谷神星的轨道，使得天文学家能够重新找到这颗小行星。这一成功使高斯声名鹊起，也促使他在1809年系统发表了最小二乘法的理论。

关于正态分布的优先权，历史上存在一些争议。Laplace在1774年也研究过类似的分布，但高斯是第一个将其与最小二乘法联系起来，并给出严格数学推导的人。因此，在英文文献中通常称为"Gaussian distribution"，而在概率统计文献中也常称为"normal distribution"。

中心极限定理的历史同样悠久。De Moivre在1733年证明了二项分布在 $n \to \infty$ 时趋于正态（即De Moivre-Laplace定理）。Laplace在1812年将其推广到了更一般的情形。现代形式的CLT主要由Lindeberg（1922）和Lévy（1935）建立。

## 现代意义

**数据科学**：正态分布在假设检验、贝叶斯推断、高斯过程机器学习等方法中具有核心地位。

**物理学**：从布朗运动到热噪声，从量子谐振子到量子场论的高斯积分，正态分布无处不在。

**工程**：控制理论中的卡尔曼滤波、通信中的调制解调、图像处理中的高斯模糊，都依赖于正态分布。

**经济学与金融**：资产定价模型、风险管理、宏观经济模型中，正态假设或CLT近似是标准工具。

**形式化数学**：Mathlib4中有正态分布和中心极限定理的严格形式化定义与证明，体现了这一理论在现代数学基础中的成熟地位。
