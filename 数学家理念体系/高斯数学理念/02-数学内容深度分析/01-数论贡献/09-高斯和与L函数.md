## 一、高斯和

### 1.1 定义

**二次Gauss和**：

```
对奇素数p和整数n：

G(n, p) = ∑_{k=0}^{p-1} e^{2πink²/p}

其中e^{2πi/p}是p次单位根

特殊情况：
G(0, p) = p
G(1, p) = G(p)（标准Gauss和）
```

**一般Gauss和**：

```
对模m和特征χ：

τ(χ) = ∑_{a=1}^{m-1} χ(a) e^{2πia/m}

这是Dirichlet特征的Gauss和
```

---

### 1.2 基本性质

**绝对值**：

```
对非平凡特征χ：

|τ(χ)|² = {
  m,     若χ是实特征
  m,     若χ是非实特征（一般情况）
}

特别地，对Legendre符号：
|G(p)|² = p
```

**乘法性质**：

```
G(mn, p) = G(m, p) G(n, p) · (m/p) · (n/p)

其中(m/p)是Legendre符号
```

---

### 1.3 计算

**标准Gauss和**：

```
G(p) = {
  √p,     若p ≡ 1 (mod 4)
  i√p,    若p ≡ 3 (mod 4)
}

这给出了Gauss和的精确值
```

**一般情况**：

```
对gcd(n, p) = 1：

G(n, p) = (n/p) · G(p)

其中(n/p)是Legendre符号
```

## 二、在二次互反律中的应用

### 2.1 证明方法

**利用Gauss和**：

```
方法：
1. 将Legendre符号表示为Gauss和
2. 计算Gauss和的乘积
3. 利用特征的性质
4. 得到互反律

这是高斯的第二个证明方法
```

**具体步骤**：

```
1. (p/q) = G(p, q) / G(1, q)
2. 计算G(p, q) · G(q, p)
3. 利用Gauss和的性质
4. 得到(p/q)(q/p) = (-1)^{(p-1)(q-1)/4}
```

## 三、Dirichlet L函数

### 3.1 定义

**Dirichlet L函数**：

```
对Dirichlet特征χ：

L(s, χ) = ∑_{n=1}^∞ χ(n)/n^s

其中Re(s) > 1

解析延拓：
- 延拓到整个复平面
- 若χ非平凡，则L(s, χ)全纯
- 若χ平凡，则L(s, χ) = ζ(s) ∏_{p|m} (1 - p^{-s})
```

**Euler乘积**：

```
L(s, χ) = ∏_p (1 - χ(p)/p^s)^{-1}

其中p遍历素数

这连接了L函数与素数
```

---

### 3.2 函数方程

**函数方程**：

```
对原特征χ mod m：

Λ(s, χ) = (m/π)^{s/2} Γ((s+ε)/2) L(s, χ)

其中ε = (1 - χ(-1))/2

函数方程：
Λ(s, χ) = τ(χ)/(i^ε √m) · Λ(1-s, χ̅)

其中χ̅是χ的复共轭
```

---

### 3.3 特殊值

**L(1, χ)**：

```
对非平凡特征χ：

L(1, χ) = -τ(χ)/m · ∑_{a=1}^{m-1} χ̅(a) ln|1 - e^{2πia/m}|

这给出了L(1, χ)的精确值
```

**在类数公式中**：

```
对二次域Q(√d)：

h(Q(√d)) = (w/2) √|Δ| / L(1, χ_Δ)

其中：
- h是类数
- w是单位数
- Δ是判别式
- χ_Δ是Kronecker符号
```

## 四、在素数分布中的应用

### 4.1 Dirichlet定理

**定理**：

```
对gcd(a, d) = 1：

算术级数a, a+d, a+2d, ...中有无穷多个素数

证明使用L函数：
- 证明L(1, χ) ≠ 0（对非平凡χ）
- 利用L函数的性质
- 得到素数分布
```

---

### 4.2 显式公式

**素数计数**：

```
对算术级数a (mod d)：

∑_{p≤x, p≡a (mod d)} ln(p) =
(1/φ(d)) x + O(x·e^{-c√(ln x)})

其中使用了L函数
```
