## 一、历史背景

### 1.1 发现历程

**欧拉的发现**：

```text
时间：1783年
人物：Leonhard Euler
贡献：通过大量计算发现了二次互反律的模式
局限：没有给出严格证明

欧拉的观察：
- 计算了大量Legendre符号
- 发现了互反的模式
- 但未能证明
```

**勒让德的贡献**：

```text
时间：1785年
人物：Adrien-Marie Legendre
贡献：
- 引入了Legendre符号 (a/p)
- 陈述了二次互反律
- 给出了不完整的证明

Legendre符号：
(a/p) = {
  0,  若 p | a
  1,  若 ∃x: x² ≡ a (mod p)
 -1,  否则
}
```

**高斯的突破**：

```text
时间：1796年（发现），1801年（发表）
人物：Carl Friedrich Gauss
贡献：
- 第一个严格证明
- 在《算术研究》中发表
- 给出了8个不同的证明

历史意义：
- 将猜想变为定理
- 确立了严格证明标准
- 开启了互反律理论
```

### 1.2 定理的表述

**标准表述**：

```text
定理（二次互反律）：
对奇素数p, q：

(p/q)(q/p) = (-1)^{(p-1)(q-1)/4}

其中(p/q)是Legendre符号
```

**等价表述**：

```text
表述1：互反关系
若p ≡ q ≡ 3 (mod 4)，则
  (p/q) = -(q/p)
否则
  (p/q) = (q/p)

表述2：对称性
(p/q) = (q/p) · (-1)^{(p-1)(q-1)/4}

表述3：条件形式
若p ≡ 1 (mod 4) 或 q ≡ 1 (mod 4)，则
  (p/q) = (q/p)
若p ≡ q ≡ 3 (mod 4)，则
  (p/q) = -(q/p)
```

**补充法则**：

```text
第一补充法则：
(-1/p) = (-1)^{(p-1)/2} = {
  1,  若 p ≡ 1 (mod 4)
 -1,  若 p ≡ 3 (mod 4)
}

第二补充法则：
(2/p) = (-1)^{(p²-1)/8} = {
  1,  若 p ≡ ±1 (mod 8)
 -1,  若 p ≡ ±3 (mod 8)
}
```

## 二、高斯的证明

### 2.1 第一个证明（归纳法）

**证明思路**：

```text
方法：对较小的素数归纳
步骤：
1. 验证小素数的情况
2. 建立递推关系
3. 利用辅助引理
4. 完成归纳

特点：
- 完全初等
- 不依赖其他理论
- 但计算复杂
```

**关键引理**：

```text
引理1（Gauss引理）：
设p是奇素数，a与p互素
考虑集合 S = {a, 2a, ..., ((p-1)/2)a}
设其中模p后大于p/2的个数为μ

则：(a/p) = (-1)^μ

意义：
- 给出了Legendre符号的计算方法
- 是证明互反律的关键工具
```

**证明概要**：

```text
步骤1：对p, q应用Gauss引理
- 计算(p/q)和(q/p)
- 得到(-1)的幂次

步骤2：建立联系
- 通过计数论证
- 连接两个符号

步骤3：计算指数
- 计算(p-1)(q-1)/4
- 建立等式

步骤4：完成证明
- 验证所有情况
- 建立互反关系
```

### 2.2 第二个证明（Gauss和）

**Gauss和的定义**：

```text
设p是奇素数，ζ = e^{2πi/p}

Gauss和：
τ(χ) = ∑_{a=1}^{p-1} χ(a) ζ^a

其中χ是模p的Dirichlet特征

性质：
|τ(χ)|² = p（对非平凡特征）
```

**证明思路**：

```text
方法：利用Gauss和的性质

步骤：
1. 将Legendre符号表示为Gauss和
2. 计算Gauss和的乘积
3. 利用特征的性质
4. 得到互反律

优势：
- 更简洁
- 揭示了深层结构
- 可推广到高次
```

## 三、定理的应用

### 3.1 二次剩余的判定

**判定算法**：

```text
问题：判定a是否是模p的二次剩余

方法1：直接计算
- 计算a^{(p-1)/2} (mod p)
- 利用Euler准则

方法2：利用互反律
- 分解p的素因子
- 应用互反律
- 递归计算

优势：
- 避免大指数运算
- 计算效率高
- 适合大素数
```

**计算例子**：

```text
判定：15是否是模47的二次剩余？

步骤：
1. 分解：15 = 3 × 5
2. (15/47) = (3/47)(5/47)
3. 应用互反律：
   (3/47) = (47/3) · (-1)^{(3-1)(47-1)/4}
          = (47/3) · (-1)^{23}
          = (2/3) · (-1)
          = -1 · (-1) = 1

   (5/47) = (47/5) · (-1)^{(5-1)(47-1)/4}
          = (2/5) · (-1)^{23}
          = -1 · (-1) = 1
4. 结果：(15/47) = 1 × 1 = 1
   所以15是模47的二次剩余
```

## 四、推广与发展

### 4.1 高次互反律

**三次互反律**：

```text
时间：1844年
人物：Carl Gustav Jacob Jacobi
基础：Eisenstein整数

定理：
对Eisenstein素数π, ρ：

(π/ρ)₃ · (ρ/π)₃ = ...

其中(·/·)₃是三次剩余符号

意义：
- 二次互反律的推广
- 代数数论的发展
- 类域论的先驱
```

**四次互反律**：

```text
时间：1852年
人物：Gotthold Eisenstein

基础：Gauss整数Z[i]

定理：
对Gauss素数π, ρ：

(π/ρ)₄ · (ρ/π)₄ = ...

其中(·/·)₄是四次剩余符号
```

---

### 4.2 Hilbert互反律

**Hilbert符号**：

```text
定义：对局部域K

Hilbert符号：
(·, ·)_v: K* × K* → {±1}

性质：
- 双线性
- 对称性
- 非退化

Hilbert互反律：
对全局域K，对任意a, b ∈ K*：

∏_v (a, b)_v = 1

其中v遍历K的所有位
```

**意义**：

```text
1. 统一了各种互反律
2. 类域论的基础
3. 现代数论的核心
```

---

### 4.3 Artin互反律

**Artin互反律**：

```text
时间：1927年
人物：Emil Artin

内容：
对Abel扩张L/K：

存在同构：
Gal(L/K) ≅ C_K / N_{L/K}(C_L)

其中：
- C_K是类群
- N_{L/K}是范映射

意义：
- 类域论的核心定理
- 统一互反律
- 现代数论的里程碑
```
