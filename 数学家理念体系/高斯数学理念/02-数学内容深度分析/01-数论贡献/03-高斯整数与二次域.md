## 一、历史背景

### 1.1 高斯整数的发现

**研究背景**：

```text
时间：1825-1832年
问题：哪些整数可以表示为两个平方数之和？

经典问题：
n = x² + y² 何时有整数解？

Fermat的观察：
- 奇素数p = x² + y² ⟺ p ≡ 1 (mod 4)
- 但缺乏严格证明

Euler的工作：
- 给出了部分证明
- 但方法复杂
```

**高斯的突破**：

```text
方法：引入高斯整数Z[i]

思想：
- 将整数问题转化为代数整数问题
- 利用唯一分解性质
- 建立代数数论基础

意义：
- 开创了代数数论
- 为现代数论奠定基础
- 影响至今
```

## 二、高斯整数Z[i]

### 2.1 定义与基本性质

**定义**：

```text
高斯整数：
Z[i] = {a + bi | a, b ∈ Z}

其中i² = -1

例子：
2 + 3i, 5, -i, 1 + i都是高斯整数
```

**环结构**：

```text
运算：
- 加法：(a + bi) + (c + di) = (a+c) + (b+d)i
- 乘法：(a + bi)(c + di) = (ac-bd) + (ad+bc)i

性质：
- Z[i]是交换环
- 单位元：1 = 1 + 0i
- 单位：±1, ±i（共4个）
```

**范数**：

```text
定义：对α = a + bi ∈ Z[i]

N(α) = α·α̅ = (a + bi)(a - bi) = a² + b²

性质：
1. N(α) ≥ 0，且N(α) = 0 ⟺ α = 0
2. N(αβ) = N(α)N(β)
3. N(α) ∈ Z（非负整数）
```

---

### 2.2 整除与素元

**整除关系**：

```text
定义：对α, β ∈ Z[i]
α | β ⟺ 存在γ ∈ Z[i]使得β = αγ

性质：
- 传递性
- 线性组合
- 与范数的关系：α | β ⟹ N(α) | N(β)
```

**素元**：

```text
定义：π ∈ Z[i]是素元，如果：
1. π不是单位
2. 若π = αβ，则α或β是单位

等价表述：
π是素元 ⟺ N(π)是素数或N(π) = p²（其中p ≡ 3 (mod 4)）

分类：
1. 1 + i（范数2）
2. p（其中p ≡ 3 (mod 4)的素数）
3. a + bi（其中a² + b² = p，p ≡ 1 (mod 4)的素数）
```

**唯一分解**：

```text
定理：Z[i]是唯一分解域（UFD）

即：每个非零非单位的高斯整数可以唯一地（在单位意义下）分解为素元的乘积

意义：
- 类似于整数的算术基本定理
- 为代数数论奠定基础
- 证明数论问题的新方法
```

---

### 2.3 欧几里得算法

**欧几里得性质**：

```text
定理：Z[i]是欧几里得域

即：对任意α, β ∈ Z[i]，β ≠ 0
存在γ, ρ ∈ Z[i]使得
α = βγ + ρ，且N(ρ) < N(β)

意义：
- 可以执行欧几里得算法
- 计算最大公因子
- 证明唯一分解
```

**算法**：

```text
输入：α, β ∈ Z[i]，β ≠ 0
输出：gcd(α, β)

步骤：
1. 计算商：γ = ⌊α/β⌋（四舍五入到最近的高斯整数）
2. 计算余数：ρ = α - βγ
3. 若N(ρ) < N(β)，则递归：gcd(β, ρ)
4. 否则，继续

终止：当余数为0时，返回β
```

### 2.4 应用：两平方和问题

**问题**：

```text
哪些正整数n可以表示为两个平方数之和？
n = x² + y²，其中x, y ∈ Z
```

**高斯的解答**：

```text
定理：n = x² + y²有整数解 ⟺
n的每个形如p ≡ 3 (mod 4)的素因子p在n的分解中出现偶数次

证明思路：
1. 在Z[i]中分解n
2. 利用唯一分解
3. 分析素因子
4. 得到表示

例子：
5 = 1² + 2²（因为5 ≡ 1 (mod 4)）
13 = 2² + 3²（因为13 ≡ 1 (mod 4)）
但7不能表示为两平方和（因为7 ≡ 3 (mod 4)）
```

**表示数的公式**：

```text
对n = 2^α ∏p_i^{a_i} ∏q_j^{b_j}

其中p_i ≡ 1 (mod 4)，q_j ≡ 3 (mod 4)

表示数：
r₂(n) = {
  0,              若某个b_j为奇数
  4∏(a_i + 1),    若所有b_j为偶数
}

这给出了表示数的精确公式
```

## 三、二次域Q(√d)

### 3.1 定义与基本性质

**定义**：

```text
二次域：
Q(√d) = {a + b√d | a, b ∈ Q}

其中d是无平方因子整数（d ≠ 0, 1）

分类：
- d > 0：实二次域
- d < 0：虚二次域
```

**整数环**：

```text
二次域的整数环O_K：

若d ≡ 2, 3 (mod 4)：
O_K = Z[√d] = {a + b√d | a, b ∈ Z}

若d ≡ 1 (mod 4)：
O_K = Z[(1+√d)/2] = {a + b(1+√d)/2 | a, b ∈ Z}

例子：
Q(√-1)：O_K = Z[i]（高斯整数）
Q(√-3)：O_K = Z[(1+√-3)/2]（Eisenstein整数）
Q(√2)：O_K = Z[√2]
```

**判别式**：

```text
定义：
Δ = {
  d,        若d ≡ 1 (mod 4)
  4d,       若d ≡ 2, 3 (mod 4)
}

性质：
- Δ是基本不变量
- 决定整数环的结构
- 与二次型判别式一致
```

---

### 3.2 理想与类群

**理想**：

```text
定义：I ⊆ O_K是理想，如果：
1. I是加法子群
2. 对任意α ∈ I, β ∈ O_K，有αβ ∈ I

主理想：
(α) = {αβ | β ∈ O_K}

理想乘法：
IJ = {∑αᵢβᵢ | αᵢ ∈ I, βᵢ ∈ J}
```

**理想类群**：

```text
定义：
- 两个理想I, J等价：I ~ J ⟺ 存在α使得I = αJ
- 理想类群：Cl(K) = {理想类}

性质：
- Cl(K)是有限Abel群
- 阶h(K) = |Cl(K)|是类数
- h(K) = 1 ⟺ O_K是主理想域
```

**类数公式**：

```text
对虚二次域Q(√d)，d < 0：

h(Q(√d)) = (1/π)√|Δ| · L(1, χ_Δ)

其中：
- Δ是判别式
- L(s, χ_Δ)是Dirichlet L函数
- χ_Δ是Kronecker符号

意义：
- 连接代数与解析
- 类域论的基础
- 现代数论的核心
```

---

### 3.3 单位群

**单位**：

```text
定义：ε ∈ O_K是单位 ⟺ 存在η使得εη = 1

等价：N(ε) = ±1

单位群：O_K* = {O_K中的单位}
```

**Dirichlet单位定理（二次域特例）**：

```text
对二次域Q(√d)：

1. 虚二次域（d < 0）：
   - 只有有限个单位
   - 除了Q(√-1), Q(√-3)外，只有±1
   - Q(√-1)* = {±1, ±i}（4个）
   - Q(√-3)* = {±1, ±ω, ±ω²}（6个，ω = e^{2πi/3}）

2. 实二次域（d > 0）：
   - 有无限个单位
   - O_K* = {±εⁿ | n ∈ Z}
   - 其中ε是基本单位（> 1）
```

**基本单位**：

```text
对实二次域Q(√d)：

基本单位ε是满足ε > 1的最小单位

计算：
- 利用连分数
- Pell方程的解
- 算法化方法

例子：
Q(√2)：ε = 1 + √2
Q(√3)：ε = 2 + √3
```
