## 一、原根

### 1.1 定义

**原根**：

```text
对模m，g是模m的原根 ⟺
{g, g², ..., g^φ(m)} 构成模m的简化剩余系

等价条件：
1. g^φ(m) ≡ 1 (mod m)
2. 对任意d < φ(m)，g^d ≢ 1 (mod m)
3. g的阶等于φ(m)
```

**存在性**：

```text
模m有原根 ⟺
m = 2, 4, p^α, 2p^α

其中p是奇素数，α ≥ 1

特殊情况：
- 模素数p总有原根
- 模2和4有原根
- 模2^α (α ≥ 3)无原根
```

---

### 1.2 原根的计算

**小模数**：

```text
直接验证：
- 模3：2是原根
- 模5：2, 3是原根
- 模7：3, 5是原根
- 模11：2, 6, 7, 8是原根
```

**大模数**：

```text
算法：
1. 分解φ(m) = p₁^{α₁} ... pₖ^{αₖ}
2. 对每个pᵢ，验证g^{φ(m)/pᵢ} ≢ 1 (mod m)
3. 若所有验证通过，则g是原根

复杂度：O(φ(m)的因子个数)
```

## 二、指标理论

### 2.1 指标的定义

**指标**：

```text
设g是模m的原根
对任意a满足gcd(a, m) = 1

定义：ind_g(a) = 最小正整数k使得g^k ≡ a (mod m)

性质：
- ind_g(ab) ≡ ind_g(a) + ind_g(b) (mod φ(m))
- ind_g(a^n) ≡ n·ind_g(a) (mod φ(m))
- 将乘法问题转化为加法问题
```

---

### 2.2 指标的性质

**对数性质**：

```text
指标类似于对数：

1. ind_g(ab) = ind_g(a) + ind_g(b) (mod φ(m))
2. ind_g(a^n) = n·ind_g(a) (mod φ(m))
3. ind_g(1) = 0 (mod φ(m))
4. ind_g(g) = 1 (mod φ(m))
```

**换底公式**：

```text
设g, h都是模m的原根：

ind_h(a) = ind_h(g) · ind_g(a) (mod φ(m))

这类似于对数的换底公式
```

## 三、离散对数问题

### 3.1 问题陈述

**离散对数**：

```text
给定：g, h, p（g是模p的原根，h ≡ g^x (mod p)）
求：x = ind_g(h)

这是离散对数问题（DLP）

计算困难性：
- 一般情况：指数时间
- 特殊情况：多项式时间
- 密码学应用的基础
```

---

### 3.2 计算方法

**小步大步算法**：

```text
设m = ⌈√p⌉

步骤：
1. 计算g^j (mod p)，j = 0, 1, ..., m-1（小步）
2. 计算h·(g^{-m})^i (mod p)，i = 0, 1, ..., m-1（大步）
3. 找到匹配：g^j = h·(g^{-m})^i
4. 解：x = j + im

复杂度：O(√p)
```

**指数演算**：

```text
更高效的算法：
- 利用因子基
- 线性代数
- 亚指数时间

应用：密码分析
```
