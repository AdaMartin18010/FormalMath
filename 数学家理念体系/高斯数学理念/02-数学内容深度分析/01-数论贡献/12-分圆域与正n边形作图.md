## 一、正n边形作图问题

### 1.1 经典问题

**尺规作图**：

```text
问题：哪些正n边形可以用尺规作图？

已知：
- 正3, 4, 5, 6边形可作
- 正7边形不可作
- 正17边形可作（Gauss, 1796）

高斯的突破：
- 17岁时发现正17边形可作
- 建立了完整的理论
- 确定了所有可作的正n边形
```

---

### 1.2 高斯的定理

**可作性条件**：

**定理（Gauss-Wantzel）**：
正 $n$ 边形可用尺规作图当且仅当
$$n = 2^\alpha \cdot p_1 \cdot p_2 \cdot \cdots \cdot p_k$$

其中：
- $\alpha \geq 0$
- $p_i$ 是形如 $2^{2^m} + 1$ 的Fermat素数
- $p_i$ 互不相同

**Fermat素数**：
$$F_m = 2^{2^m} + 1$$

已知的Fermat素数：
- $F_0 = 3$
- $F_1 = 5$
- $F_2 = 17$
- $F_3 = 257$
- $F_4 = 65537$

（目前仅知这5个Fermat素数）

## 二、分圆域

### 2.1 定义

**分圆域**：

**$n$ 次分圆域**：
$$\mathbb{Q}(\zeta_n) = \mathbb{Q}(e^{2\pi i/n})$$

其中 $\zeta_n = e^{2\pi i/n}$ 是 $n$ 次本原单位根。

**性质**：
- **次数**：$[\mathbb{Q}(\zeta_n) : \mathbb{Q}] = \phi(n)$（Euler函数）
- **Galois扩张**：$\mathbb{Q}(\zeta_n)/\mathbb{Q}$ 是Galois扩张
- **Abelian扩张**：Galois群是Abelian群

**数学表述**：
$$\text{Gal}(\mathbb{Q}(\zeta_n)/\mathbb{Q}) \cong (\mathbb{Z}/n\mathbb{Z})^*$$

**整数环**：

```text
Q(ζₙ)的整数环：
Z[ζₙ] = {a₀ + a₁ζₙ + ... + a_{n-1}ζₙ^{n-1} | aᵢ ∈ Z}

性质：
- 当n为素数幂时是唯一分解域
- 一般情况更复杂
```

---

### 2.2 分圆多项式

**定义**：

**$n$ 次分圆多项式**：
$$\Phi_n(x) = \prod_{\gcd(k,n)=1} (x - \zeta_n^k)$$

**性质**：
1. **次数**：$\deg \Phi_n = \phi(n)$
2. **首一、整系数、不可约**：$\Phi_n(x) \in \mathbb{Z}[x]$ 且在 $\mathbb{Q}$ 上不可约
3. **分解关系**：$x^n - 1 = \prod_{d|n} \Phi_d(x)$

**例子**：
- $\Phi_3(x) = x^2 + x + 1$
- $\Phi_4(x) = x^2 + 1$
- $\Phi_5(x) = x^4 + x^3 + x^2 + x + 1$
- $\Phi_{17}(x) = x^{16} + x^{15} + \cdots + x + 1$

**例子**：

```text
Φ₃(x) = x² + x + 1
Φ₄(x) = x² + 1
Φ₅(x) = x⁴ + x³ + x² + x + 1
Φ₁₇(x) = x¹⁶ + x¹⁵ + ... + x + 1
```

## 三、可作性与Galois理论

### 3.1 Galois群

**结构**：

$$\text{Gal}(\mathbb{Q}(\zeta_n)/\mathbb{Q}) \cong (\mathbb{Z}/n\mathbb{Z})^*$$

**同构**：
$$\sigma_k: \zeta_n \mapsto \zeta_n^k, \quad \text{其中 } \gcd(k, n) = 1$$

**性质**：
- **Abelian群**：$\sigma_k \circ \sigma_l = \sigma_{kl}$
- **阶为 $\phi(n)$**：$|\text{Gal}(\mathbb{Q}(\zeta_n)/\mathbb{Q})| = \phi(n)$

**数学表述**：
对 $n = 17$，$\text{Gal}(\mathbb{Q}(\zeta_{17})/\mathbb{Q}) \cong (\mathbb{Z}/17\mathbb{Z})^* \cong \mathbb{Z}/16\mathbb{Z}$（循环群）

---

### 3.2 可作性条件

**理论**：

```text
正n边形可作 ⟺
Q(ζₙ)的Galois群是2-群

等价条件：
- φ(n)是2的幂
- n = 2^α · p₁ · ... · pₖ
- 其中pᵢ是Fermat素数
```

## 四、正17边形

### 4.1 高斯的构造

**历史**：

```text
1796年3月30日，高斯（19岁）发现：
正17边形可以用尺规作图

这是他最自豪的发现之一
```

**方法**：

```text
利用：
1. cos(2π/17)的表达式
2. 分圆域的性质
3. 二次方程的求解
4. 尺规作图的步骤
```

---

### 4.2 构造步骤

**理论**：

```text
关键：cos(2π/17)可以用平方根表示

步骤：
1. 利用分圆域的性质
2. 构造中间域
3. 逐步求解二次方程
4. 得到cos(2π/17)
5. 完成作图
```
