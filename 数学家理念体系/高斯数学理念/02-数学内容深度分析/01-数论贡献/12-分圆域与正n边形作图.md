## 一、正n边形作图问题

### 1.1 经典问题

**尺规作图**：

```
问题：哪些正n边形可以用尺规作图？

已知：
- 正3, 4, 5, 6边形可作
- 正7边形不可作
- 正17边形可作（Gauss, 1796）

高斯的突破：
- 17岁时发现正17边形可作
- 建立了完整的理论
- 确定了所有可作的正n边形
```

---

### 1.2 高斯的定理

**可作性条件**：

```
定理（Gauss-Wantzel）：
正n边形可用尺规作图 ⟺
n = 2^α · p₁ · p₂ · ... · pₖ

其中：
- α ≥ 0
- pᵢ是形如2^{2^m} + 1的Fermat素数
- pᵢ互不相同

Fermat素数：
F₀ = 3, F₁ = 5, F₂ = 17, F₃ = 257, F₄ = 65537
（仅知这5个）
```

## 二、分圆域

### 2.1 定义

**分圆域**：

```
n次分圆域：
Q(ζₙ) = Q(e^{2πi/n})

其中ζₙ = e^{2πi/n}是n次本原单位根

性质：
- 次数：φ(n)（Euler函数）
- Galois扩张
- Abelian扩张
```

**整数环**：

```
Q(ζₙ)的整数环：
Z[ζₙ] = {a₀ + a₁ζₙ + ... + a_{n-1}ζₙ^{n-1} | aᵢ ∈ Z}

性质：
- 当n为素数幂时是唯一分解域
- 一般情况更复杂
```

---

### 2.2 分圆多项式

**定义**：

```
n次分圆多项式：
Φₙ(x) = ∏_{gcd(k,n)=1} (x - ζₙ^k)

性质：
1. 次数：φ(n)
2. 首一、整系数、不可约
3. xⁿ - 1 = ∏_{d|n} Φ_d(x)
```

**例子**：

```
Φ₃(x) = x² + x + 1
Φ₄(x) = x² + 1
Φ₅(x) = x⁴ + x³ + x² + x + 1
Φ₁₇(x) = x¹⁶ + x¹⁵ + ... + x + 1
```

## 三、可作性与Galois理论

### 3.1 Galois群

**结构**：

```
Gal(Q(ζₙ)/Q) ≅ (Z/nZ)*

同构：
σₖ: ζₙ ↦ ζₙ^k，其中gcd(k, n) = 1

性质：
- Abelian群
- 阶为φ(n)
```

---

### 3.2 可作性条件

**理论**：

```
正n边形可作 ⟺
Q(ζₙ)的Galois群是2-群

等价条件：
- φ(n)是2的幂
- n = 2^α · p₁ · ... · pₖ
- 其中pᵢ是Fermat素数
```

## 四、正17边形

### 4.1 高斯的构造

**历史**：

```
1796年3月30日，高斯（19岁）发现：
正17边形可以用尺规作图

这是他最自豪的发现之一
```

**方法**：

```
利用：
1. cos(2π/17)的表达式
2. 分圆域的性质
3. 二次方程的求解
4. 尺规作图的步骤
```

---

### 4.2 构造步骤

**理论**：

```
关键：cos(2π/17)可以用平方根表示

步骤：
1. 利用分圆域的性质
2. 构造中间域
3. 逐步求解二次方程
4. 得到cos(2π/17)
5. 完成作图
```
