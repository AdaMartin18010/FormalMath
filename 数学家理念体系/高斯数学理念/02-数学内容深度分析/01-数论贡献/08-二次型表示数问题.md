## 一、问题背景

### 1.1 表示数问题

**核心问题**：

```text
给定：
- 二次型Q(x₁, ..., xₙ)
- 正整数n

问题：方程Q(x₁, ..., xₙ) = n有多少整数解？

记作：r_Q(n) = #{(x₁, ..., xₙ) ∈ Zⁿ | Q(x₁, ..., xₙ) = n}
```

**历史**：

```text
Fermat：哪些数可以表示为两个平方数之和？
Gauss：系统化二次型理论，研究表示数
现代：利用模形式、L函数等工具
```

## 二、二元二次型的表示数

### 2.1 基本理论

**正定型**：

```
对正定型Q(x, y) = ax² + bxy + cy²，Δ < 0

表示数r_Q(n)总是有限的

方法：
1. 利用约化形式
2. 枚举可能的(x, y)
3. 利用类数公式
4. 利用Theta函数
```

**不定型**：

```
对不定型Q(x, y)，Δ > 0

表示数r_Q(n)可能无限

方法：
1. 利用Pell方程
2. 利用基本解
3. 生成所有解
```

---

### 2.2 两平方和问题

**问题**：

```
哪些正整数n可以表示为两个平方数之和？
n = x² + y²
```

**高斯的解答**：

```
定理：n = x² + y²有整数解 ⟺
n的每个形如p ≡ 3 (mod 4)的素因子p在n的分解中出现偶数次

表示数公式：
r₂(n) = 4(d₁(n) - d₃(n))

其中：
- d₁(n)是n的形如4k+1的因子个数
- d₃(n)是n的形如4k+3的因子个数
```

## 三、Theta函数方法

### 3.1 Theta函数

**定义**：

```
对二次型Q：

θ_Q(z) = ∑_{(x₁,...,xₙ)∈Zⁿ} q^{Q(x₁,...,xₙ)}
       = ∑_{n≥0} r_Q(n) q^n

其中q = e^{2πiz}
```

**性质**：

```
1. 模形式：θ_Q(z)是权n/2的模形式
2. 生成函数：r_Q(n)的生成函数
3. 函数方程：与模形式相关
4. 解析性质：全纯函数
```

---

### 3.2 表示数公式

**利用Theta函数**：

```
方法：
1. 计算θ_Q(z)
2. 利用模形式理论
3. 得到Fourier系数
4. 得到表示数公式

例子：两平方和
θ(z)² = (∑_{n∈Z} q^{n²})²
      = ∑_{n≥0} r₂(n) q^n

利用Jacobi恒等式可以得到r₂(n)的公式
```

## 四、类数公式的应用

### 4.1 类数公式

**公式**：

```
对负判别式Δ < 0：

h(Δ) = (1/π)√|Δ| · L(1, χ_Δ)

其中：
- h(Δ)是类数
- L(s, χ_Δ)是Dirichlet L函数
- χ_Δ是Kronecker符号
```

**与表示数的关系**：

```
表示数可以通过类数公式计算

方法：
1. 将表示数问题转化为类数问题
2. 利用类数公式
3. 得到表示数公式
```
