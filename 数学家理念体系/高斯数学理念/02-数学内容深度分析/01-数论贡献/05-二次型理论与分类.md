## 一、历史背景

### 1.1 二次型研究的起源

**早期发展**：

```text
时间：17-18世纪
人物：Fermat, Euler, Lagrange

Fermat的贡献：
- 研究哪些数可以表示为两个平方数之和
- 提出了许多猜想
- 但缺乏系统理论

Euler的工作：
- 研究了二次型x² + ny²
- 发现了某些规律
- 但未建立完整理论

Lagrange的突破：
- 研究了二元二次型的约化
- 引入了等价概念
- 为高斯的工作奠定基础
```

**高斯的系统化**：

```text
时间：1801年
著作：《算术研究》第五章

贡献：
1. 系统化二次型理论
2. 建立分类理论
3. 引入类数概念
4. 建立表示数公式
5. 连接二次型与二次剩余

意义：
- 将零散结果系统化
- 建立代数数论基础
- 影响至今
```

## 二、二元二次型基础

### 2.1 定义与记号

**二元二次型**：

```text
定义：
Q(x, y) = ax² + bxy + cy²

其中a, b, c ∈ Z

矩阵表示：
Q(x, y) = [x y] [a  b/2] [x]
                    [b/2 c] [y]

判别式：
Δ = b² - 4ac

性质：
- Δ ≡ 0 或 1 (mod 4)
- Δ是二次型的本质不变量
```

**约化形式**：

```text
定义：二次型Q是约化的，如果：
1. |b| ≤ a ≤ c
2. 若|b| = a或a = c，则b ≥ 0

约化的意义：
- 每个等价类有唯一的约化形式
- 便于分类和计算
- 算法化处理
```

---

### 2.2 等价关系

**等价定义**：

```text
两个二次型Q₁, Q₂等价（记作Q₁ ~ Q₂）
如果存在线性变换：
[x']   [α β] [x]
[y'] = [γ δ] [y]

其中α, β, γ, δ ∈ Z，且det = αδ - βγ = ±1

使得Q₁(x', y') = Q₂(x, y)

性质：
- 等价是等价关系
- 等价的型有相同判别式
- 等价的型表示相同的数集
```

**等价类的性质**：

```text
1. 每个等价类有唯一的约化形式
2. 等价类由判别式决定
3. 等价类形成群结构（类群）
```

## 三、判别式与分类

### 3.1 判别式的性质

**基本性质**：

```text
判别式：Δ = b² - 4ac

性质：
1. Δ ≡ 0 或 1 (mod 4)
2. 等价的型有相同判别式
3. Δ是基本不变量

分类：
- Δ > 0：不定型（indefinite）
- Δ < 0：定型（definite）
  - Δ < 0, a > 0：正定型
  - Δ < 0, a < 0：负定型
- Δ = 0：退化型
```

**判别式的计算**：

```text
给定二次型Q(x, y) = ax² + bxy + cy²：

Δ = b² - 4ac

例子：
Q(x, y) = x² + xy + y²
Δ = 1² - 4·1·1 = -3

Q(x, y) = x² - 5y²
Δ = 0² - 4·1·(-5) = 20
```

---

### 3.2 定型与不定型

**正定型**：

```text
定义：Δ < 0 且 a > 0

性质：
- Q(x, y) > 0 对所有(x, y) ≠ (0, 0)
- 有界表示数
- 类数有限

例子：
x² + y²（Δ = -4）
x² + xy + y²（Δ = -3）
2x² + 2xy + 3y²（Δ = -20）
```

**不定型**：

```text
定义：Δ > 0

性质：
- 可以取正值和负值
- 表示数可能无限
- 更复杂的结构

例子：
x² - 2y²（Δ = 8）
x² - 5y²（Δ = 20）
```

---

### 3.3 类数

**定义**：

```text
类数h(Δ) = 判别式Δ的二次型等价类的个数

性质：
- h(Δ)总是有限的
- h(Δ) = 1当且仅当所有型等价于主型
- 类数是重要的不变量
```

**类数公式**：

```text
对负判别式Δ < 0：

h(Δ) = (1/π)√|Δ| · L(1, χ_Δ)

其中：
- L(s, χ_Δ)是Dirichlet L函数
- χ_Δ是Kronecker符号

意义：
- 连接二次型与L函数
- 类域论的早期例子
- 解析数论的应用
```

**类数为1的判别式**：

```text
对负判别式，类数为1的判别式：
Δ = -3, -4, -7, -8, -11, -12, -16, -19, -27, -28, -43, -67, -163

共13个（Gauss猜想，后由Heegner证明）

意义：
- 这些判别式有特殊的算术性质
- 与椭圆曲线相关
- 现代数论的重要问题
```

## 四、表示数问题

### 4.1 表示数的定义

**问题陈述**：

```text
给定二次型Q和整数n
求方程Q(x, y) = n的整数解个数

记作：r_Q(n) = #{(x, y) ∈ Z² | Q(x, y) = n}
```

**基本性质**：

```text
1. 等价的型表示相同的数集
2. r_Q(n)可能为0（n不能被表示）
3. r_Q(n)可能为有限或无限
```

---

### 4.2 表示数的计算

**正定型的情况**：

```text
对正定型Q，r_Q(n)总是有限的

方法：
1. 利用约化形式
2. 枚举可能的(x, y)
3. 利用类数公式
4. 利用Theta函数
```

**不定型的情况**：

```text
对不定型，r_Q(n)可能无限

方法：
1. 利用Pell方程
2. 利用基本解
3. 生成所有解
```

**例子**：

```text
Q(x, y) = x² + y²

表示数：
r_Q(n) = 4(d₁(n) - d₃(n))

其中：
- d₁(n)是n的形如4k+1的因子个数
- d₃(n)是n的形如4k+3的因子个数

这给出了哪些数可以表示为两个平方数之和
```

---

### 4.3 表示数与Theta函数

**Theta函数**：

```text
定义：
θ_Q(z) = ∑_{(x,y)∈Z²} q^{Q(x,y)}
       = ∑_{n≥0} r_Q(n) q^n

其中q = e^{2πiz}

性质：
- 模形式
- 连接二次型与分析
- 表示数的生成函数
```

**应用**：

```text
1. 计算表示数
2. 研究表示数的渐近行为
3. 连接数论与分析
4. 模形式理论
```

## 五、类群结构

### 5.1 类群的群结构

**定义**：

```text
判别式Δ的类群：
C(Δ) = {二次型等价类}

运算：合成（composition）

性质：
- 交换群
- 有限群
- 阶为h(Δ)
```

**主型**：

```text
主型：
对Δ < 0：x² - (Δ/4)y²（若Δ ≡ 0 (mod 4)）
          x² + xy + ((1-Δ)/4)y²（若Δ ≡ 1 (mod 4)）

性质：
- 主型是单位元
- 所有型与主型合成得到自身
```

---

### 5.2 类群的计算

**算法**：

```text
1. 列出所有约化形式
2. 计算合成表
3. 确定群结构

例子：Δ = -20

约化形式：
Q₁ = x² + 5y²
Q₂ = 2x² + 2xy + 3y²

类群：C(-20) ≅ Z/2Z（2阶循环群）
```

## 七、现代发展

### 7.1 代数数论视角

**二次域**：

```text
二次域：Q(√d)，其中d是无平方因子整数

类群：
C(Δ) ≅ Cl(Q(√d))

其中Δ是二次域的判别式

意义：
- 二次型类群与理想类群同构
- 连接二次型与代数数论
- 类域论的基础
```

**理想理论**：

```text
二次型 ↔ 理想：

每个二次型对应一个理想类
类群结构相同

这建立了：
- 二次型与代数数论的桥梁
- 统一的观点
- 现代数论的基础
```

---

### 7.2 模形式理论

**Theta级数**：

```text
二次型的Theta函数是模形式：

θ_Q(z) ∈ M_k(Γ₀(N), χ)

其中：
- k是权
- N是级
- χ是特征

意义：
- 连接二次型与模形式
- 表示论的应用
- 现代数论的核心
```
