## 一、类群

### 1.1 定义

**理想类群**：

```text
对数域K，整数环O_K：

分式理想：I = {α/β | α, β ∈ O_K, β ≠ 0}

主理想：(α) = {αβ | β ∈ O_K}

理想类：I ~ J ⟺ 存在α使得I = αJ

类群：Cl(K) = {理想类}

性质：
- Cl(K)是有限Abel群
- 阶h(K) = |Cl(K)|是类数
- h(K) = 1 ⟺ O_K是主理想域（PID）
```

---

### 1.2 二次域的类群

**二次域**：

```text
K = Q(√d)，其中d是无平方因子整数

整数环O_K：
- 若d ≡ 2, 3 (mod 4)：O_K = Z[√d]
- 若d ≡ 1 (mod 4)：O_K = Z[(1+√d)/2]

判别式Δ：
- 若d ≡ 1 (mod 4)：Δ = d
- 若d ≡ 2, 3 (mod 4)：Δ = 4d
```

**与二次型的关系**：

```text
类群Cl(K)与二次型类群C(Δ)同构

这建立了：
- 代数数论与二次型理论的桥梁
- 统一的观点
- 计算方法
```

## 二、类数公式

### 2.1 Dirichlet类数公式

**虚二次域**：

```text
对虚二次域Q(√d)，d < 0：

h(Q(√d)) = (w/2π) √|Δ| · L(1, χ_Δ)

其中：
- h是类数
- w是单位数（通常为2, 4, 6）
- Δ是判别式
- L(s, χ_Δ)是Dirichlet L函数
- χ_Δ是Kronecker符号
```

**实二次域**：

```text
对实二次域Q(√d)，d > 0：

h(Q(√d)) = (1/ln ε) √Δ · L(1, χ_Δ)

其中ε是基本单位（> 1）
```

---

### 2.2 L(1, χ)的计算

**公式**：

```text
对非平凡特征χ mod m：

L(1, χ) = -τ(χ)/m · ∑_{a=1}^{m-1} χ̅(a) ln|1 - e^{2πia/m}|

其中τ(χ)是Gauss和
```

**对二次特征**：

```text
对Kronecker符号χ_Δ：

L(1, χ_Δ)可以用更简单的公式计算

这给出了类数的精确值
```

## 三、类数为1的问题

### 3.1 虚二次域

**类数为1的判别式**：

```text
对虚二次域，类数为1的判别式：
Δ = -3, -4, -7, -8, -11, -12, -16, -19, -27, -28, -43, -67, -163

共13个（Gauss猜想，Heegner-Stark证明）

意义：
- 这些域有特殊的算术性质
- 与椭圆曲线相关
- 现代数论的重要问题
```
