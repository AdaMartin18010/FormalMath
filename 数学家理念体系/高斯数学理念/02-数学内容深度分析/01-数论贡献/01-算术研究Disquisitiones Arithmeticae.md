## 二、核心理论体系

### 2.1 同余理论

**同余的定义**：

```text
定义：对整数a, b, m (m > 0)
a ≡ b (mod m)  ⟺  m | (a - b)

等价表述：
- a和b除以m有相同的余数
- a = b + km，其中k ∈ Z
- a mod m = b mod m
```

**同余的基本性质**：

```text
1. 自反性：a ≡ a (mod m)
2. 对称性：a ≡ b (mod m) ⟹ b ≡ a (mod m)
3. 传递性：a ≡ b, b ≡ c (mod m) ⟹ a ≡ c (mod m)
4. 加法：a ≡ b, c ≡ d (mod m) ⟹ a+c ≡ b+d (mod m)
5. 乘法：a ≡ b, c ≡ d (mod m) ⟹ ac ≡ bd (mod m)
6. 幂：a ≡ b (mod m) ⟹ aⁿ ≡ bⁿ (mod m)
```

**同余类的运算**：

```text
模m的同余类：
[a]_m = {b ∈ Z | b ≡ a (mod m)}

同余类集合：
Z/mZ = {[0]_m, [1]_m, ..., [m-1]_m}

运算：
- [a]_m + [b]_m = [a+b]_m
- [a]_m · [b]_m = [ab]_m

性质：
- Z/mZ构成环
- 当m为素数时，Z/mZ是域
```

---

### 2.2 一次同余方程

**一般形式**：

```text
ax ≡ b (mod m)

解的存在性：
- 有解 ⟺ gcd(a, m) | b
- 解的个数：gcd(a, m)个（模m意义下）
```

**求解方法**：

```text
方法1：扩展欧几里得算法
- 求gcd(a, m) = d
- 若d | b，则解存在
- 求ax + my = d的特解(x₀, y₀)
- 通解：x ≡ (b/d)·x₀ (mod m/d)

方法2：模逆元
- 若gcd(a, m) = 1，则a有模m的逆元a⁻¹
- 解：x ≡ a⁻¹·b (mod m)
```

**中国剩余定理**：

```text
定理：设m₁, m₂, ..., mₖ两两互素
同余方程组：
x ≡ a₁ (mod m₁)
x ≡ a₂ (mod m₂)
...
x ≡ aₖ (mod mₖ)

有唯一解（模M = m₁m₂...mₖ）

构造解：
Mᵢ = M/mᵢ
yᵢ满足：Mᵢyᵢ ≡ 1 (mod mᵢ)
解：x ≡ ∑ᵢ aᵢMᵢyᵢ (mod M)
```

### 2.3 幂剩余与原根

**原根的定义**：

```text
模m的原根：
g是模m的原根 ⟺
{g, g², ..., g^φ(m)} 构成模m的简化剩余系

等价条件：
- g^φ(m) ≡ 1 (mod m)
- 对任意d < φ(m)，g^d ≢ 1 (mod m)
```

**原根的存在性**：

```text
定理：模m有原根 ⟺
m = 2, 4, p^α, 2p^α
其中p是奇素数，α ≥ 1

特殊情况：
- 模素数p总有原根
- 模2和4有原根
- 模2^α (α ≥ 3)无原根
```

**指标理论**：

```text
设g是模m的原根
对任意a满足gcd(a, m) = 1

定义：ind_g(a) = 最小正整数k使得g^k ≡ a (mod m)

性质：
- ind_g(ab) ≡ ind_g(a) + ind_g(b) (mod φ(m))
- ind_g(a^n) ≡ n·ind_g(a) (mod φ(m))
- 将乘法问题转化为加法问题
```

### 2.4 二次剩余

**Legendre符号**：

```text
对奇素数p和整数a，定义：

(a/p) = {
  0,  若 p | a
  1,  若a是模p的二次剩余
 -1,  若a是模p的二次非剩余
}

性质：
- (a/p) ≡ a^{(p-1)/2} (mod p)  (Euler准则)
- (ab/p) = (a/p)(b/p)
- (a²/p) = 1（若p不整除a）
```

**二次互反律**：

```text
定理（高斯，1801）：
对奇素数p, q：

(p/q)(q/p) = (-1)^{(p-1)(q-1)/4}

等价表述：
- 若p ≡ q ≡ 3 (mod 4)，则(p/q) = -(q/p)
- 否则，(p/q) = (q/p)

补充法则：
- (-1/p) = (-1)^{(p-1)/2}
- (2/p) = (-1)^{(p²-1)/8}
```

**二次互反律的意义**：

```text
革命性意义：
1. 建立了二次剩余判定的完整理论
2. 将两个素数的二次剩余性质联系起来
3. 开启了互反律理论的先河

影响：
- 启发高次互反律（Kummer, Hilbert, Artin）
- 类域论的起源
- 现代代数数论的基础
```

### 2.5 二次型理论

**二元二次型**：

```text
形式：f(x, y) = ax² + bxy + cy²

判别式：Δ = b² - 4ac

等价关系：
f ~ g ⟺ 存在线性变换将f变为g

分类问题：
- 给定判别式Δ，有多少不等价的型？
- 给定整数n，哪些型能表示n？
```

**表示数问题**：

```text
问题：给定二次型f和整数n
求方程 f(x, y) = n 的整数解个数

高斯的方法：
- 利用二次型的等价类
- 建立表示数的公式
- 连接二次型与二次剩余
```

**类数公式**：

```text
对负判别式Δ < 0：

h(Δ) = (1/π)√|Δ| · L(1, χ_Δ)

其中：
- h(Δ)是判别式Δ的类数
- χ_Δ是Kronecker符号
- L(s, χ_Δ)是Dirichlet L函数

意义：
- 连接二次型与L函数
- 类域论的早期例子
```

## 三、证明方法与数学风格

### 3.1 严格的证明标准

**高斯的证明哲学**：

```text
原则：
1. 每个定理必须有严格证明
2. 避免依赖几何直观
3. 建立完整的逻辑链条
4. 系统化而非零散

影响：
- 确立了现代数学的证明标准
- 影响了整个19世纪数学发展
- 为数学的严格化奠定基础
```

**证明技巧**：

```text
1. 归纳法：
   - 对模的归纳
   - 对次数的归纳
   - 对参数的归纳

2. 构造性方法：
   - 明确构造解
   - 算法化过程
   - 可计算性

3. 分类讨论：
   - 按模的奇偶性
   - 按剩余类
   - 按等价类
```

---

### 3.2 二次互反律的证明

**高斯的第一个证明**：

```text
方法：归纳法
- 对较小的素数验证
- 建立递推关系
- 利用辅助引理

特点：
- 完全初等
- 不依赖其他高级理论
- 但计算复杂
```

**高斯的其他证明**：

```text
高斯共给出8个不同的证明：
1. 第一个证明（1801）：归纳法
2. 第二个证明：Gauss和
3. 第三个证明：二次型
4. 第四个证明：分圆域
5-8. 其他变体

意义：
- 展示同一问题的多种视角
- 启发后续理论发展
- 体现数学的统一性
```
