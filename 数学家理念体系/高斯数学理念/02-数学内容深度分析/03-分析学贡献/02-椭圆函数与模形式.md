### 1.2 椭圆函数

**定义**：

椭圆函数是在复平面上双周期的亚纯函数。

**数学表述**：
存在 $\omega_1, \omega_2$（线性无关）使得
$$f(z + m\omega_1 + n\omega_2) = f(z)$$

对所有 $m, n \in \mathbb{Z}$。

**基本周期**：
- $\omega_1, \omega_2$ 是基本周期
- 周期平行四边形：$\{z + s\omega_1 + t\omega_2: s, t \in [0, 1)\}$
- 所有周期：$\Lambda = \{m\omega_1 + n\omega_2: m, n \in \mathbb{Z}\}$（周期格）

## 二、Weierstrass椭圆函数

### 2.1 定义

**℘函数**：

**Weierstrass ℘函数**：

$$\wp(z; \omega_1, \omega_2) = \frac{1}{z^2} + \sum' \leqft[\frac{1}{(z - \omega)^2} - \frac{1}{\omega^2}\right]$$

其中 $\sum'$ 表示对所有非零周期 $\omega \in \Lambda \setminus \{0\}$ 求和。

**性质**：
1. **双周期**：$\wp(z + \omega) = \wp(z)$ 对所有 $\omega \in \Lambda$
2. **偶函数**：$\wp(-z) = \wp(z)$
3. **在周期点有2阶极点**：在 $z = 0$ 和所有周期点
4. **满足微分方程**

**微分方程**：

$$\wp'(z)^2 = 4\wp(z)^3 - g_2\wp(z) - g_3$$

其中：
$$g_2 = 60 \sum' \frac{1}{\omega^4}, \quad g_3 = 140 \sum' \frac{1}{\omega^6}$$

这是椭圆曲线方程 $y^2 = 4x^3 - g_2x - g_3$。

---

### 2.2 性质

**基本性质**：

```text
1. 双周期性：
   ℘(z + ω) = ℘(z) 对所有ω ∈ Λ

2. 奇偶性：
   ℘(-z) = ℘(z)（偶函数）
   ℘'(-z) = -℘'(z)（奇函数）

3. 极点：
   在每个周期点有2阶极点

4. 零点：
   在周期平行四边形的中心有零点
```

## 三、Jacobi椭圆函数

### 3.1 定义

**sn, cn, dn函数**：

```text
Jacobi椭圆函数：

sn(u, k) = sin φ
cn(u, k) = cos φ
dn(u, k) = √(1 - k² sin² φ)

其中：
u = ∫₀^φ dθ/√(1 - k² sin² θ)

k是模数（0 < k < 1）
```

**性质**：

```text
1. 双周期：
   sn(u + 4K) = sn(u)
   cn(u + 4K) = cn(u)
   dn(u + 2K) = dn(u)

   其中K是第一类完全椭圆积分

2. 关系：
   sn² + cn² = 1
   dn² + k² sn² = 1
```

## 四、模形式

### 4.1 定义

**模形式**：

**定义**：
在复上半平面 $\mathbb{H} = \{z \in \mathbb{C}: \text{Im } z > 0\}$ 上的全纯函数 $f(z)$ 满足：

**1. 模变换**：
$$f\leqft(\frac{az + b}{cz + d}\right) = (cz + d)^k f(z)$$

对所有 $\begin{pmatrix} a & b \\ c & d \end{pmatrix} \in \text{SL}(2, \mathbb{Z})$。

**2. 增长条件**：
$f$ 在 $i\infty$ 处有Fourier展开：
$$f(z) = \sum_{n=0}^{\infty} a_n e^{2\pi i n z}$$

其中 $k$ 是权（weight）。

**模群**：

```text
SL(2, Z) = {[a b; c d] | ad - bc = 1, a,b,c,d ∈ Z}

作用：
z ↦ (az + b)/(cz + d)

生成元：
T = [1 1; 0 1]（平移）
S = [0 -1; 1 0]（反演）
```

---

### 4.2 例子

**Eisenstein级数**：

$$G_k(z) = \sum' \frac{1}{(mz + n)^k}$$

其中 $\sum'$ 表示对所有 $(m, n) \in \mathbb{Z}^2 \setminus \{(0, 0)\}$ 求和。

**性质**：
- **权 $k$ 的模形式**：满足模变换条件
- **在 $i\infty$ 处有Fourier展开**：$G_k(z) = 2\zeta(k) + \sum_{n=1}^{\infty} \sigma_{k-1}(n) e^{2\pi i n z}$，其中 $\sigma_{k-1}(n) = \sum_{d|n} d^{k-1}$
- **连接数论与分析**：系数与数论函数相关

**Delta函数**：

$$\Delta(z) = (60G_4)^3 - 27(140G_6)^2$$

**性质**：
- **权12的尖点形式**：在 $i\infty$ 处有零点
- **连接椭圆曲线与模形式**：$\Delta(z)$ 的系数与椭圆曲线的 $j$-不变量相关
