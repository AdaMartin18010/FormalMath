## 一、历史背景

### 1.1 椭圆积分

**问题起源**：

```text
17-18世纪：
- 计算椭圆弧长
- 计算单摆周期
- 遇到椭圆积分

椭圆积分：
∫ dx/√(P(x))

其中P(x)是3次或4次多项式

问题：
- 不能用初等函数表示
- 需要新的函数类
```

**高斯的贡献（约1800年）**：

```text
研究：
1. 研究了椭圆积分
2. 发现了双周期性
3. 发展了椭圆函数理论
4. 但未发表（与Abel、Jacobi独立）

意义：
- 椭圆函数理论的起源
- 影响Abel和Jacobi
- 现代数论的基础
```

---

### 1.2 椭圆函数

**定义**：

```text
椭圆函数：
在复平面上双周期的亚纯函数

即：存在ω₁, ω₂（线性无关）使得
f(z + mω₁ + nω₂) = f(z)

对所有m, n ∈ Z

基本周期：
- ω₁, ω₂是基本周期
- 周期平行四边形
- 所有周期：Λ = {mω₁ + nω₂ | m, n ∈ Z}
```

---

## 二、Weierstrass椭圆函数

### 2.1 定义

**℘函数**：

```text
Weierstrass ℘函数：

℘(z; ω₁, ω₂) = 1/z² + ∑' [1/(z - ω)² - 1/ω²]

其中∑'表示对所有非零周期ω求和

性质：
1. 双周期
2. 偶函数
3. 在周期点有2阶极点
4. 满足微分方程
```

**微分方程**：

```text
℘'(z)² = 4℘(z)³ - g₂℘(z) - g₃

其中：
g₂ = 60 ∑' 1/ω⁴
g₃ = 140 ∑' 1/ω⁶

这是椭圆曲线方程
```

---

### 2.2 性质

**基本性质**：

```text
1. 双周期性：
   ℘(z + ω) = ℘(z) 对所有ω ∈ Λ

2. 奇偶性：
   ℘(-z) = ℘(z)（偶函数）
   ℘'(-z) = -℘'(z)（奇函数）

3. 极点：
   在每个周期点有2阶极点

4. 零点：
   在周期平行四边形的中心有零点
```

## 三、Jacobi椭圆函数

### 3.1 定义

**sn, cn, dn函数**：

```text
Jacobi椭圆函数：

sn(u, k) = sin φ
cn(u, k) = cos φ
dn(u, k) = √(1 - k² sin² φ)

其中：
u = ∫₀^φ dθ/√(1 - k² sin² θ)

k是模数（0 < k < 1）
```

**性质**：

```text
1. 双周期：
   sn(u + 4K) = sn(u)
   cn(u + 4K) = cn(u)
   dn(u + 2K) = dn(u)

   其中K是第一类完全椭圆积分

2. 关系：
   sn² + cn² = 1
   dn² + k² sn² = 1
```

## 四、模形式

### 4.1 定义

**模形式**：

```text
定义：
在复上半平面H上的全纯函数f(z)满足：

1. 模变换：
   f((az + b)/(cz + d)) = (cz + d)^k f(z)

   对所有 [a b; c d] ∈ SL(2, Z)

2. 增长条件：
   f在i∞处有Fourier展开

其中k是权（weight）
```

**模群**：

```text
SL(2, Z) = {[a b; c d] | ad - bc = 1, a,b,c,d ∈ Z}

作用：
z ↦ (az + b)/(cz + d)

生成元：
T = [1 1; 0 1]（平移）
S = [0 -1; 1 0]（反演）
```

---

### 4.2 例子

**Eisenstein级数**：

```text
G_k(z) = ∑' 1/(mz + n)^k

其中∑'表示对所有(m, n) ≠ (0, 0)求和

性质：
- 权k的模形式
- 在i∞处有Fourier展开
- 连接数论与分析
```

**Delta函数**：

```text
Δ(z) = (60G₄)³ - 27(140G₆)²

性质：
- 权12的尖点形式
- 在i∞处有零点
- 连接椭圆曲线与模形式
```
