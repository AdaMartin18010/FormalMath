## 一、历史背景

### 1.1 复数的历史

**早期发展**：

```
16-17世纪：
- Cardano：解三次方程时遇到负数开方
- Bombelli：引入虚数单位i
- 但缺乏几何理解

18世纪：
- Euler：发展了复数理论
- 但主要作为计算工具
- 缺乏几何直观
```

**高斯的突破（1799年）**：

```
贡献：
1. 引入复平面（Gauss平面）
2. 将复数几何化
3. 证明代数基本定理
4. 建立复分析基础

意义：
- 复数的几何理解
- 复分析的起点
- 影响整个数学
```

---

### 1.2 Gauss平面的引入

**几何表示**：

```
复数z = x + iy对应平面上的点(x, y)

坐标：
- 横轴：实轴（Re z）
- 纵轴：虚轴（Im z）
- 原点：0

模长：
|z| = √(x² + y²) = 点到原点的距离

幅角：
arg z = arctan(y/x) = 与实轴的夹角
```

**极坐标形式**：

```
z = r e^{iθ} = r(cos θ + i sin θ)

其中：
- r = |z|（模长）
- θ = arg z（幅角）

优势：
- 乘法：模相乘，幅角相加
- 幂：模的幂，幅角的倍数
- 根：均匀分布
```

## 二、复数的运算

### 2.1 基本运算

**加法**：

```
几何意义：向量加法

z₁ + z₂ = (x₁ + x₂) + i(y₁ + y₂)

对应：点(x₁, y₁) + 点(x₂, y₂)
```

**乘法**：

```
代数：
z₁ · z₂ = (x₁x₂ - y₁y₂) + i(x₁y₂ + x₂y₁)

极坐标：
z₁ · z₂ = r₁r₂ e^{i(θ₁ + θ₂)}

几何意义：
- 模相乘：|z₁ · z₂| = |z₁| · |z₂|
- 幅角相加：arg(z₁ · z₂) = arg z₁ + arg z₂
- 旋转和缩放
```

**除法**：

```
z₁ / z₂ = (z₁ · z̄₂) / |z₂|²

极坐标：
z₁ / z₂ = (r₁/r₂) e^{i(θ₁ - θ₂)}

几何意义：
- 模相除
- 幅角相减
```

---

### 2.2 共轭与模

**共轭**：

```
定义：z̄ = x - iy

几何意义：关于实轴的反射

性质：
1. z + z̄ = 2 Re z
2. z - z̄ = 2i Im z
3. z · z̄ = |z|²
4. z̄̄ = z
```

**模的性质**：

```
|z|² = z · z̄ = x² + y²

三角不等式：
|z₁ + z₂| ≤ |z₁| + |z₂|

几何意义：三角形两边之和大于第三边
```

## 三、复函数

### 3.1 基本函数

**多项式**：

```
P(z) = a₀ + a₁z + ... + aₙzⁿ

性质：
- 连续
- 可导
- 解析
```

**指数函数**：

```
e^z = e^x (cos y + i sin y)

性质：
1. e^{z₁ + z₂} = e^{z₁} · e^{z₂}
2. |e^z| = e^x
3. arg(e^z) = y
4. e^{iπ} = -1（Euler公式）
```

**三角函数**：

```
sin z = (e^{iz} - e^{-iz})/(2i)
cos z = (e^{iz} + e^{-iz})/2

性质：
- 周期函数
- 与实函数类似
- 但有复值
```

---

### 3.2 解析函数

**定义**：

```
函数f(z)在点z₀解析，如果：
- 在z₀的邻域内可导
- 导数存在且连续

等价条件（Cauchy-Riemann方程）：
∂u/∂x = ∂v/∂y
∂u/∂y = -∂v/∂x

其中f(z) = u(x, y) + i v(x, y)
```

**性质**：

```
1. 解析函数的和、积、商（分母非零）仍解析
2. 复合函数解析
3. 解析函数无限次可导
4. 有幂级数展开
```

## 四、代数基本定理

### 4.1 定理陈述

**代数基本定理**：

```
定理（Gauss, 1799）：
每个n次复系数多项式
P(z) = a₀ + a₁z + ... + aₙzⁿ  (aₙ ≠ 0)

在复数域内有n个根（计重数）

等价表述：
复数域C是代数闭域
```

**历史意义**：

```
1. 解决了长期悬而未决的问题
2. 证明了复数的必要性
3. 建立了复分析的基础
4. 影响整个数学
```

## 五、复分析的应用

### 5.1 实积分的计算

**留数定理**：

```
应用复分析计算实积分：

∫_{-∞}^∞ f(x) dx = 2πi ∑ Res(f, zₖ)

其中zₖ是f在上半平面的奇点

例子：
∫_{-∞}^∞ dx/(1 + x²) = π
```

---

### 5.2 级数求和

**应用**：

```
利用复分析求级数和：

∑_{n=-∞}^∞ f(n) = -π ∑ Res(f cot πz, zₖ)

其中zₖ是f的奇点

例子：
∑_{n=1}^∞ 1/n² = π²/6
```
