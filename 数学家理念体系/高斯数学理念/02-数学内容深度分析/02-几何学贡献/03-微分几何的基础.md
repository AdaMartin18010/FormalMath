## 一、历史背景

### 1.1 微分几何的起源

**早期发展**：

```
18世纪：
- Euler：研究了曲面的参数表示
- Monge：发展了微分几何
- Lagrange：研究了变分问题

局限：
- 主要研究嵌入在空间中的曲面
- 依赖外部坐标
- 未区分内蕴与外蕴性质
```

**高斯的突破（1827年）**：

```
著作：《关于曲面的一般研究》

贡献：
1. 区分内蕴几何与外蕴几何
2. 建立第一、第二基本形式
3. 证明Theorema Egregium
4. 建立微分几何基础
5. 为黎曼几何奠定基础

意义：
- 现代微分几何的起点
- 几何学的革命
- 影响至今
```

## 二、曲面的参数表示

### 2.1 参数化

**参数表示**：

```
曲面S的参数化：
r(u, v) = (x(u, v), y(u, v), z(u, v))

其中(u, v) ∈ D ⊆ R²

切向量：
r_u = ∂r/∂u = (∂x/∂u, ∂y/∂u, ∂z/∂u)
r_v = ∂r/∂v = (∂x/∂v, ∂y/∂v, ∂z/∂v)

正则性条件：
r_u × r_v ≠ 0（在每点）
```

**例子**：

```
球面：
r(θ, φ) = (R sin θ cos φ, R sin θ sin φ, R cos θ)

柱面：
r(u, v) = (R cos u, R sin u, v)

旋转面：
r(u, v) = (f(v) cos u, f(v) sin u, g(v))
```

---

### 2.2 切空间

**切平面**：

```
在点p = r(u₀, v₀)：

切平面：
T_p S = {p + s r_u + t r_v | s, t ∈ R}

法向量：
n = (r_u × r_v) / |r_u × r_v|

单位法向量：
n̂ = n / |n|
```

## 三、第一基本形式

### 3.1 定义

**第一基本形式**：

```
定义：
I = E du² + 2F du dv + G dv²

其中：
E = r_u · r_u
F = r_u · r_v
G = r_v · r_v

矩阵形式：
[E  F]
[F  G]

行列式：
EG - F² = |r_u × r_v|²
```

**几何意义**：

```
1. 曲线长度：
   ds² = E du² + 2F du dv + G dv²

   曲线长度：
   L = ∫ √(E (du/dt)² + 2F (du/dt)(dv/dt) + G (dv/dt)²) dt

2. 角度：
   两切向量v₁, v₂的夹角：
   cos θ = (v₁ · v₂) / (|v₁| |v₂|)

   用第一基本形式表示

3. 面积：
   dA = √(EG - F²) du dv

   面积：
   A = ∫∫ √(EG - F²) du dv
```

---

### 3.2 内蕴性质

**内蕴几何**：

```
定义：只依赖于第一基本形式的性质

内蕴性质：
1. 曲线长度
2. 角度
3. 面积
4. Gauss曲率（Theorema Egregium）

不依赖：
- 嵌入方式
- 第二基本形式
- 外部坐标
```

## 四、第二基本形式

### 4.1 定义

**第二基本形式**：

```
定义：
II = L du² + 2M du dv + N dv²

其中：
L = r_{uu} · n
M = r_{uv} · n
N = r_{vv} · n

几何意义：
- 描述曲面在空间中的弯曲
- 依赖于嵌入方式
- 不是内蕴的
```

**主曲率**：

```
在点p，主曲率k₁, k₂是二次型II关于I的特征值

即：det(II - k I) = 0

Gauss曲率：
K = k₁ · k₂ = (LN - M²)/(EG - F²)

平均曲率：
H = (k₁ + k₂)/2 = (EN - 2FM + GL)/(2(EG - F²))
```

## 五、Gauss的微分几何方程

### 5.1 Gauss方程

**Gauss方程**：

```
连接第一、第二基本形式：

r_{uu} = Γ¹₁₁ r_u + Γ²₁₁ r_v + L n
r_{uv} = Γ¹₁₂ r_u + Γ²₁₂ r_v + M n
r_{vv} = Γ¹₂₂ r_u + Γ²₂₂ r_v + N n

其中Γᵢⱼᵏ是Christoffel符号
```

**Christoffel符号**：

```
定义：
Γ¹₁₁ = (1/(EG-F²)) [G E_u - F(2E_v - F_u)]/2
Γ²₁₁ = (1/(EG-F²)) [E(2F_u - E_v) - F E_u]/2
...

只依赖于E, F, G及其导数
```

---

### 5.2 Codazzi-Mainardi方程

**相容性条件**：

```
Codazzi-Mainardi方程：
L_v - M_u = L Γ¹₁₂ + M(Γ²₁₂ - Γ¹₁₁) - N Γ²₁₁
M_v - N_u = L Γ¹₂₂ + M(Γ²₂₂ - Γ¹₁₂) - N Γ²₁₂

意义：
- 第一、第二基本形式的相容性
- 曲面的存在性条件
- 微分几何的基本方程
```

## 六、测地线

### 6.1 定义

**测地线**：

```
定义：曲面上局部最短的曲线

等价条件：
1. 测地曲率为0
2. 主法向量平行于曲面法向量
3. 满足测地线方程

测地线方程：
d²u/dt² + Γ¹₁₁ (du/dt)² + 2Γ¹₁₂ (du/dt)(dv/dt) + Γ¹₂₂ (dv/dt)² = 0
d²v/dt² + Γ²₁₁ (du/dt)² + 2Γ²₁₂ (du/dt)(dv/dt) + Γ²₂₂ (dv/dt)² = 0
```

---

### 6.2 性质

**基本性质**：

```
1. 局部最短：
   测地线是连接两点的局部最短路径

2. 唯一性：
   在充分小的邻域内，两点间有唯一测地线

3. 内蕴性：
   测地线只依赖于第一基本形式

4. 变分性质：
   测地线是长度泛函的临界点
```

## 七、Gauss-Bonnet定理

### 7.1 局部形式

**定理**：

```
对曲面S上的区域D，边界∂D由光滑曲线组成：

∫_D K dA + ∫_{∂D} k_g ds + ∑θᵢ = 2π

其中：
- K是Gauss曲率
- k_g是测地曲率
- θᵢ是外角
```

---

### 7.2 整体形式

**紧致曲面**：

```
对紧致定向曲面S：

∫_S K dA = 2πχ(S)

其中χ(S)是Euler示性数

这连接了：
- 局部性质（曲率）
- 整体性质（拓扑）
```
