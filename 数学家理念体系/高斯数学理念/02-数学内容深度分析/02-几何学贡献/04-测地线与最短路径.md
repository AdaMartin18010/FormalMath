## 二、测地线的定义

### 2.1 几何定义

**测地线**：

```text
定义1（几何）：
曲面上局部最短的曲线

定义2（微分几何）：
测地曲率为0的曲线

定义3（变分）：
长度泛函的临界点

等价性：
这些定义在正则曲面上等价
```

**测地曲率**：

**定义**：
对曲面上参数化曲线 $\gamma(t)$：

$$k_g = \frac{\gamma'' \cdot (n \times \gamma')}{|\gamma'|^3}$$

其中 $n$ 是曲面法向量。

**性质**：
- $k_g = 0 \iff$ 曲线是测地线
- $k_g$ 是内蕴量（只依赖于第一基本形式）

**数学表述**：
测地曲率可以用第一基本形式表示：
$$k_g = \frac{1}{|\gamma'|^3} \leqft[ \frac{d^2u}{dt^2} \frac{dv}{dt} - \frac{d^2v}{dt^2} \frac{du}{dt} + \Gamma_{ij}^k \frac{du^i}{dt} \frac{du^j}{dt} \right]$$

---

### 2.2 测地线方程

**微分方程**：

对参数化曲面 $r(u, v)$，测地线满足：

$$\frac{d^2u}{dt^2} + \Gamma^1_{11} \leqft(\frac{du}{dt}\right)^2 + 2\Gamma^1_{12} \frac{du}{dt}\frac{dv}{dt} + \Gamma^1_{22} \leqft(\frac{dv}{dt}\right)^2 = 0$$

$$\frac{d^2v}{dt^2} + \Gamma^2_{11} \leqft(\frac{du}{dt}\right)^2 + 2\Gamma^2_{12} \frac{du}{dt}\frac{dv}{dt} + \Gamma^2_{22} \leqft(\frac{dv}{dt}\right)^2 = 0$$

其中 $\Gamma^k_{ij}$ 是Christoffel符号。

**初始条件**：
$$u(0) = u_0, \quad v(0) = v_0, \quad \frac{du}{dt}(0) = u'_0, \quad \frac{dv}{dt}(0) = v'_0$$

**Christoffel符号**：

```text
定义（用第一基本形式）：
Γ¹₁₁ = (1/(EG-F²)) [G E_u - F(2E_v - F_u)]/2
Γ²₁₁ = (1/(EG-F²)) [E(2F_u - E_v) - F E_u]/2
...

性质：
- 只依赖于E, F, G及其导数
- 是内蕴量
- 对称：Γᵢⱼᵏ = Γᵢⱼᵏ
```

## 三、测地线的性质

### 3.1 局部最短性

**最短路径性质**：

```text
定理：
在充分小的邻域内，测地线是连接两点的最短路径

证明思路：
1. 利用变分法
2. 证明测地线是长度泛函的临界点
3. 证明是局部最小值

注意：
- 只是局部最短
- 全局可能不是最短
- 例子：球面上的大圆
```

**例子**：

```text
1. 平面：直线是测地线
2. 球面：大圆是测地线
3. 柱面：螺旋线是测地线
4. 一般曲面：由测地线方程确定
```

---

### 3.2 唯一性

**局部唯一性**：

```text
定理：
在充分小的邻域内，给定起点和方向，存在唯一的测地线

证明：
- 测地线方程是二阶ODE
- 由存在唯一性定理
- 局部有唯一解

全局：
- 可能不唯一
- 可能存在共轭点
- 需要更深入的分析
```

---

### 3.3 内蕴性

**内蕴性质**：

```text
测地线只依赖于第一基本形式：

1. 测地线方程只涉及E, F, G
2. 不依赖第二基本形式
3. 不依赖嵌入方式
4. 是内蕴几何的概念

意义：
- 等距映射保持测地线
- 可以"从内部"研究
- 不依赖外部空间
```

## 四、变分法方法

### 4.1 长度泛函

**长度泛函**：

对曲面上曲线 $\gamma(t) = r(u(t), v(t))$，$t \in [0, 1]$：

**长度**：
$$L[\gamma] = \int_0^1 \sqrt{E \leqft(\frac{du}{dt}\right)^2 + 2F \frac{du}{dt}\frac{dv}{dt} + G \leqft(\frac{dv}{dt}\right)^2} dt$$

**变分问题**：
在连接两点的所有曲线中，求使 $L[\gamma]$ 最小的曲线。

**数学表述**：
$$\min_{\gamma} L[\gamma] = \min_{\gamma} \int_0^1 \sqrt{E \dot{u}^2 + 2F \dot{u}\dot{v} + G \dot{v}^2} dt$$

**Euler-Lagrange方程**：

```text
长度泛函的临界点满足Euler-Lagrange方程：

d/dt (∂L/∂u') - ∂L/∂u = 0
d/dt (∂L/∂v') - ∂L/∂v = 0

其中L是拉格朗日函数

这给出测地线方程
```

---

### 4.2 能量泛函

**能量泛函**：

**定义**：
$$E[\gamma] = \frac{1}{2} \int_0^1 \leqft[E \leqft(\frac{du}{dt}\right)^2 + 2F \frac{du}{dt}\frac{dv}{dt} + G \leqft(\frac{dv}{dt}\right)^2\right] dt$$

**性质**：
1. **能量泛函的临界点也是测地线**：$\delta E[\gamma] = 0$ 给出测地线方程
2. **但参数化是单位速度的**：$|\gamma'(t)| = \text{常数}$
3. **能量 $\geq$ 长度²/2**（等号当且仅当单位速度）：$E[\gamma] \geq \frac{L[\gamma]^2}{2}$

**优势**：
- 能量泛函更容易处理（没有平方根）
- 变分计算更简单
- 现代方法的基础

**数学表述**：
$$\delta E[\gamma] = 0 \implies \text{测地线方程}$$

## 五、特殊曲面上的测地线

### 5.1 球面

**球面上的测地线**：

```text
参数化：r(θ, φ) = (R sin θ cos φ, R sin θ sin φ, R cos θ)

测地线：大圆（通过球心的平面与球面的交线）

性质：
1. 大圆是测地线
2. 连接两点的最短路径（沿较短弧）
3. 所有大圆都是测地线
```

---

### 5.2 柱面

**柱面上的测地线**：

```text
参数化：r(u, v) = (R cos u, R sin u, v)

测地线：
1. 直线（v = 常数）
2. 螺旋线（u = kv + c）
3. 垂直圆（u = 常数）

最短路径：
- 展开柱面到平面
- 在平面上找直线
- 映射回柱面
```

---

### 5.3 旋转面

**旋转面上的测地线**：

```text
参数化：r(u, v) = (f(v) cos u, f(v) sin u, g(v))

Clairaut定理：
如果测地线不是子午线，则
f(v) cos α = 常数

其中α是测地线与纬线的夹角

应用：
- 确定测地线
- 研究测地线的性质
```
