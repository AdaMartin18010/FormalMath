## 一、历史背景

### 1.1 测地线的概念

**起源**：

```
词源：geodesic（测地线）
- geo = 地球
- desic = 划分

原意：地球表面上的最短路径

数学化：
- 高斯将概念推广到一般曲面
- 建立测地线理论
- 内蕴几何的应用
```

**高斯的贡献（1827年）**：

```
在《关于曲面的一般研究》中：
1. 定义了测地线
2. 建立了测地线方程
3. 研究了测地线的性质
4. 连接了内蕴几何与变分法

意义：
- 内蕴几何的重要应用
- 变分法的几何应用
- 现代微分几何的基础
```

---

### 1.2 最短路径问题

**经典问题**：

```
问题：在曲面上，给定两点，求连接它们的最短路径

历史：
- 古代：地球上的最短路径
- 18世纪：变分法的发展
- 高斯：内蕴几何方法

```

## 二、测地线的定义

### 2.1 几何定义

**测地线**：

```
定义1（几何）：
曲面上局部最短的曲线

定义2（微分几何）：
测地曲率为0的曲线

定义3（变分）：
长度泛函的临界点

等价性：
这些定义在正则曲面上等价
```

**测地曲率**：

```
定义：
对曲面上参数化曲线γ(t)：

测地曲率k_g = (γ'' · (n × γ')) / |γ'|³

其中n是曲面法向量

性质：
- k_g = 0 ⟺ 曲线是测地线
- k_g是内蕴量（只依赖于第一基本形式）
```

---

### 2.2 测地线方程

**微分方程**：

```
对参数化曲面r(u, v)，测地线满足：

d²u/dt² + Γ¹₁₁ (du/dt)² + 2Γ¹₁₂ (du/dt)(dv/dt) + Γ¹₂₂ (dv/dt)² = 0
d²v/dt² + Γ²₁₁ (du/dt)² + 2Γ²₁₂ (du/dt)(dv/dt) + Γ²₂₂ (dv/dt)² = 0

其中Γᵢⱼᵏ是Christoffel符号

初始条件：
u(0) = u₀, v(0) = v₀
du/dt(0) = u'₀, dv/dt(0) = v'₀
```

**Christoffel符号**：

```
定义（用第一基本形式）：
Γ¹₁₁ = (1/(EG-F²)) [G E_u - F(2E_v - F_u)]/2
Γ²₁₁ = (1/(EG-F²)) [E(2F_u - E_v) - F E_u]/2
...

性质：
- 只依赖于E, F, G及其导数
- 是内蕴量
- 对称：Γᵢⱼᵏ = Γᵢⱼᵏ
```

## 三、测地线的性质

### 3.1 局部最短性

**最短路径性质**：

```
定理：
在充分小的邻域内，测地线是连接两点的最短路径

证明思路：
1. 利用变分法
2. 证明测地线是长度泛函的临界点
3. 证明是局部最小值

注意：
- 只是局部最短
- 全局可能不是最短
- 例子：球面上的大圆
```

**例子**：

```
1. 平面：直线是测地线
2. 球面：大圆是测地线
3. 柱面：螺旋线是测地线
4. 一般曲面：由测地线方程确定
```

---

### 3.2 唯一性

**局部唯一性**：

```
定理：
在充分小的邻域内，给定起点和方向，存在唯一的测地线

证明：
- 测地线方程是二阶ODE
- 由存在唯一性定理
- 局部有唯一解

全局：
- 可能不唯一
- 可能存在共轭点
- 需要更深入的分析
```

---

### 3.3 内蕴性

**内蕴性质**：

```
测地线只依赖于第一基本形式：

1. 测地线方程只涉及E, F, G
2. 不依赖第二基本形式
3. 不依赖嵌入方式
4. 是内蕴几何的概念

意义：
- 等距映射保持测地线
- 可以"从内部"研究
- 不依赖外部空间
```

## 四、变分法方法

### 4.1 长度泛函

**长度泛函**：

```
对曲面上曲线γ(t) = r(u(t), v(t))，t ∈ [0, 1]：

长度：
L[γ] = ∫₀¹ √(E (du/dt)² + 2F (du/dt)(dv/dt) + G (dv/dt)²) dt

变分问题：
在连接两点的所有曲线中，求使L[γ]最小的曲线
```

**Euler-Lagrange方程**：

```
长度泛函的临界点满足Euler-Lagrange方程：

d/dt (∂L/∂u') - ∂L/∂u = 0
d/dt (∂L/∂v') - ∂L/∂v = 0

其中L是拉格朗日函数

这给出测地线方程
```

---

### 4.2 能量泛函

**能量泛函**：

```
定义：
E[γ] = (1/2) ∫₀¹ (E (du/dt)² + 2F (du/dt)(dv/dt) + G (dv/dt)²) dt

性质：
1. 能量泛函的临界点也是测地线
2. 但参数化是单位速度的
3. 能量 ≥ 长度²/2（等号当且仅当单位速度）

优势：
- 能量泛函更容易处理
- 变分计算更简单
- 现代方法的基础
```

## 五、特殊曲面上的测地线

### 5.1 球面

**球面上的测地线**：

```
参数化：r(θ, φ) = (R sin θ cos φ, R sin θ sin φ, R cos θ)

测地线：大圆（通过球心的平面与球面的交线）

性质：
1. 大圆是测地线
2. 连接两点的最短路径（沿较短弧）
3. 所有大圆都是测地线
```

---

### 5.2 柱面

**柱面上的测地线**：

```
参数化：r(u, v) = (R cos u, R sin u, v)

测地线：
1. 直线（v = 常数）
2. 螺旋线（u = kv + c）
3. 垂直圆（u = 常数）

最短路径：
- 展开柱面到平面
- 在平面上找直线
- 映射回柱面
```

---

### 5.3 旋转面

**旋转面上的测地线**：

```
参数化：r(u, v) = (f(v) cos u, f(v) sin u, g(v))

Clairaut定理：
如果测地线不是子午线，则
f(v) cos α = 常数

其中α是测地线与纬线的夹角

应用：
- 确定测地线
- 研究测地线的性质
```

```

性质：
- 保持长度

应用：
- 动力系统
- 遍历理论
- 数论（模形式）
```
