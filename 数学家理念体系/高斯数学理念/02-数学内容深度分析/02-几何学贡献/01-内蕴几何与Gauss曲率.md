## 一、历史背景

### 1.1 内蕴几何的起源

**传统几何的局限**：

```text
18世纪的几何学：
- 主要研究嵌入在三维空间中的曲面
- 依赖外部的坐标系统
- 无法区分内蕴性质与外蕴性质

问题：
- 哪些性质是曲面本身固有的？
- 哪些性质依赖于嵌入方式？
- 如何研究曲面的内蕴结构？
```

**高斯的突破（1827年）**：

```text
著作：《关于曲面的一般研究》（Disquisitiones generales circa superficies curvas）

贡献：
1. 区分内蕴几何与外蕴几何
2. 证明Gauss曲率是内蕴不变量
3. 建立内蕴几何理论
4. 为黎曼几何奠定基础

意义：
- 几何学的革命
- 现代微分几何的起点
- 影响至今
```

---

### 1.2 Theorema Egregium

**"绝妙定理"**：

```text
定理（Gauss, 1827）：
Gauss曲率K是曲面的内蕴不变量

即：K只依赖于第一基本形式，不依赖于第二基本形式

意义：
- 曲率是曲面本身的性质
- 不依赖于如何嵌入空间
- 可以"从内部"测量曲率
```

**历史意义**：

```text
1. 打破了传统观念
2. 开启了内蕴几何
3. 为黎曼几何铺路
4. 影响整个数学和物理
```

## 二、曲面的基本形式

### 2.1 第一基本形式

**定义**：

```text
曲面S的参数化：r(u, v)

切向量：
r_u = ∂r/∂u
r_v = ∂r/∂v

第一基本形式：
I = E du² + 2F du dv + G dv²

其中：
E = r_u · r_u
F = r_u · r_v
G = r_v · r_v

几何意义：
- 度量曲面上曲线的长度
- 计算角度
- 计算面积
```

**内蕴性质**：

```text
第一基本形式决定的内蕴性质：
1. 曲线长度
2. 角度
3. 面积
4. Gauss曲率（绝妙定理）

这些性质不依赖于嵌入
```

---

### 2.2 第二基本形式

**定义**：

```text
单位法向量：n = (r_u × r_v) / |r_u × r_v|

第二基本形式：
II = L du² + 2M du dv + N dv²

其中：
L = r_{uu} · n
M = r_{uv} · n
N = r_{vv} · n

几何意义：
- 描述曲面在空间中的弯曲
- 依赖于嵌入方式
- 不是内蕴的
```

**主曲率**：

```text
在点p，主曲率k₁, k₂是二次型II关于I的特征值

Gauss曲率：
K = k₁ · k₂

平均曲率：
H = (k₁ + k₂)/2
```

## 三、Gauss曲率

### 3.1 定义与计算

**定义**：

```text
Gauss曲率：
K = k₁ · k₂ = (LN - M²)/(EG - F²)

其中：
- k₁, k₂是主曲率
- E, F, G是第一基本形式的系数
- L, M, N是第二基本形式的系数

关键：虽然定义涉及第二基本形式，但K只依赖于第一基本形式
```

**Gauss公式**：

```text
K = (1/(EG-F²)²) · {
  |E_{vv}/2 + F_{uv} - G_{uu}/2   F_v - G_u/2|
  |E_u/2                           E          F    |
  |F_u - E_v/2                     F          G    |
  -
  |0                               E_v/2      G_u/2|
  |E_v/2                           E          F    |
  |G_u/2                           F          G    |
}

这证明了K只依赖于E, F, G及其导数
```

---

### 3.2 几何意义

**正曲率**：

```text
K > 0：椭圆点
- 局部像球面
- 两个主曲率同号
- 例子：球面，椭球面
```

**负曲率**：

```text
K < 0：双曲点
- 局部像马鞍面
- 两个主曲率异号
- 例子：双曲抛物面，单叶双曲面
```

**零曲率**：

```text
K = 0：抛物点
- 局部像平面或柱面
- 至少一个主曲率为0
- 例子：平面，柱面，锥面
```

## 四、Theorema Egregium的证明

### 4.1 证明思路

**关键观察**：

```text
虽然K的定义涉及第二基本形式，但：
1. 可以通过Gauss公式用第一基本形式表示
2. 只依赖于E, F, G及其一阶和二阶导数
3. 不依赖于L, M, N

因此K是内蕴的
```

**证明步骤**：

```text
1. 建立Gauss公式
2. 证明K可以用第一基本形式表示
3. 验证等距变换下K不变
4. 得出结论
```

---

### 4.2 等距不变性

**等距映射**：

```text
定义：f: S₁ → S₂是等距映射，如果
- 保持第一基本形式
- 保持曲线长度
- 保持角度

定理：等距映射保持Gauss曲率

即：若f是等距映射，则K₁(p) = K₂(f(p))
```

**应用**：

```text
例子：
- 平面可以等距映射到柱面
- 两者都有K = 0
- 但球面不能等距映射到平面（K不同）
```

## 五、Gauss-Bonnet定理

### 5.1 局部形式

**定理**：

```text
对曲面S上的区域D，边界∂D由光滑曲线组成：

∫_D K dA + ∫_{∂D} k_g ds + ∑θᵢ = 2π

其中：
- K是Gauss曲率
- k_g是测地曲率
- θᵢ是外角
- dA是面积元
- ds是弧长元
```

**特殊情况**：

```text
对测地三角形（三条边都是测地线）：

∫_D K dA + (内角和) = π

这推广了平面几何的三角形内角和定理
```

---

### 5.2 整体形式

**紧致曲面**：

```text
对紧致定向曲面S：

∫_S K dA = 2πχ(S)

其中χ(S)是Euler示性数

这连接了：
- 局部性质（曲率）
- 整体性质（拓扑）
- 深刻的统一
```

**应用**：

```text
1. 计算Euler示性数
2. 研究曲面分类
3. 拓扑与几何的联系
4. 现代几何的核心
```
