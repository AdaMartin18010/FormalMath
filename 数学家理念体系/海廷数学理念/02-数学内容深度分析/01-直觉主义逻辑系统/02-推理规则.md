## 一、自然演绎系统

**自然演绎的特点**：

自然演绎系统使用引入和消去规则，更接近人类的推理方式。

**规则类型**：

- **引入规则**：如何引入连接词或量词
- **消去规则**：如何使用连接词或量词

**证明结构**：

- 使用假设
- 应用推理规则
- 得到结论

## 二、合取规则

**合取引入（∧I）**：

```
  A    B
  ──────
   A ∧ B
```

从A和B可以推出A∧B。

**合取消去（∧E）**：

```
  A ∧ B        A ∧ B
  ────         ────
    A            B
```

从A∧B可以推出A或B。

**构造性意义**：

- 证明A∧B需要同时证明A和B
- 使用A∧B可以分别得到A和B
- 体现了构造性要求

---

## 三、析取规则

**析取引入（∨I）**：

```
    A              B
  ────           ────
  A ∨ B          A ∨ B
```

从A可以推出A∨B，从B也可以推出A∨B。

**析取消去（∨E）**：

```
  A ∨ B    [A]    [B]
           C      C
           ─────────
                C
```

从A∨B、A→C、B→C可以推出C。

**构造性意义**：

- 证明A∨B需要证明A或证明B
- 使用A∨B需要分别处理两种情况
- 体现了构造性要求

---

## 四、蕴含规则

**蕴含引入（→I）**：

```
  [A]
   ...
    B
   ────
   A → B
```

如果从假设A可以推出B，则可以推出A→B。

**蕴含消去（→E，分离规则）**：

```
  A    A → B
  ──────────
       B
```

从A和A→B可以推出B。这是唯一的基本推理规则（在公理系统中）。

**构造性意义**：

- 证明A→B需要提供从A到B的构造方法
- 使用A→B需要提供A的证明
- 体现了函数性

## 五、量词规则

**全称引入（∀I）**：

```
     A(x)
     ────
     ∀x A(x)  （x不在假设中自由出现）
```

从A(x)可以推出∀x A(x)，前提是x不在假设中自由出现。

**全称消去（∀E）**：

```
     ∀x A(x)
     ────────
       A(t)
```

从∀x A(x)可以推出A(t)，其中t是任意项。

**存在引入（∃I）**：

```
       A(t)
       ────
       ∃x A(x)
```

从A(t)可以推出∃x A(x)。

**存在消去（∃E）**：

```
  ∃x A(x)    [A(x)]
               ...
                B
               ────
                 B  （x不在B中自由出现）
```

从∃x A(x)和A(x)→B可以推出B，前提是x不在B中自由出现。

**构造性意义**：

- 证明∀x A(x)需要对所有x证明A(x)
- 证明∃x A(x)需要提供具体的x和A(x)的证明
- 体现了构造性要求

## 六、公理系统

**海廷公理系统**：

除了自然演绎系统，海廷还给出了公理系统：

- **公理**：基本假设
- **推理规则**：分离规则（Modus Ponens）
- **等价性**：公理系统与自然演绎系统等价

**公理示例**：

```
A1: A → (B → A)
A2: (A → (B → C)) → ((A → B) → (A → C))
...
```

---

## 七、推理规则的特点

**构造性**：

- 所有规则都保持构造性
- 证明必须提供构造方法
- 确保可计算性

**可靠性**：

- 规则是可靠的
- 从真前提推出真结论
- 保证推理的正确性

**完备性**：

- 规则是完备的
- 所有有效的推理都可以用规则证明
- 保证推理的完整性
