## 一、命题逻辑的语法

**原子命题**：p, q, r, ...

**连接词**：∧ (合取), ∨ (析取), → (蕴含), ⊥ (矛盾)

**否定**：¬A := A → ⊥

**公式定义**：

```
φ ::= p | ⊥ | (φ ∧ φ) | (φ ∨ φ) | (φ → φ)
```

## 二、一阶逻辑的语法

**项**：

```
t ::= x | f(t₁, ..., tₙ)
```

**原子公式**：P(t₁, ..., tₙ)

**公式**：

```
φ ::= P(t₁, ..., tₙ) | ⊥ | (φ ∧ φ) | (φ ∨ φ) | (φ → φ) | (∀x φ) | (∃x φ)
```
