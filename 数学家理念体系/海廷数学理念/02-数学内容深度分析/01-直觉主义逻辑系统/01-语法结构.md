## 一、命题逻辑的语法

**原子命题**：

- 基本符号：$p, q, r, \ldots$
- 表示基本命题
- 可以是真或假（在经典逻辑中）

**连接词**：

- **∧ (合取)**：$A \land B$ 表示"A且B"
- **∨ (析取)**：$A \lor B$ 表示"A或B"
- **→ (蕴含)**：$A \to B$ 表示"如果A则B"
- **⊥ (矛盾)**：表示矛盾或假

**否定**：

- **¬A**：定义为 $A \to \bot$
- 表示"A蕴含矛盾"

**公式定义**：

直觉主义命题逻辑的公式由以下规则递归定义：

$$\varphi ::= p \mid \bot \mid (\varphi \land \varphi) \mid (\varphi \lor \varphi) \mid (\varphi \to \varphi)$$

其中：

- $p$ 是原子命题
- $\bot$ 是矛盾
- $(\varphi \land \varphi)$ 是合取
- $(\varphi \lor \varphi)$ 是析取
- $(\varphi \to \varphi)$ 是蕴含

**例子**：

1. **原子命题**：$p, q, r$
2. **简单公式**：$p \land q, p \lor q, p \to q$
3. **复合公式**：$(p \land q) \to (r \lor s)$
4. **否定公式**：$\neqg p = p \to \bot$

## 二、一阶逻辑的语法

**项（Terms）**：

项表示对象，由以下规则定义：

$$t ::= x \mid f(t_1, \ldots, t_n)$$

其中：

- $x$ 是变量
- $f$ 是函数符号
- $t_1, \ldots, t_n$ 是项

**原子公式**：

- $P(t_1, \ldots, t_n)$：其中$P$是谓词符号，$t_1, \ldots, t_n$是项
- 表示基本关系

**量词**：

- **∀ (全称量词)**：$\forall x \varphi$ 表示"对所有$x$，$\varphi$成立"
- **∃ (存在量词)**：$\exists x \varphi$ 表示"存在$x$使得$\varphi$成立"

**公式**：

直觉主义一阶逻辑的公式由以下规则递归定义：

$$\varphi ::= P(t_1, \ldots, t_n) \mid \bot \mid (\varphi \land \varphi) \mid (\varphi \lor \varphi) \mid (\varphi \to \varphi) \mid (\forall x \varphi) \mid (\exists x \varphi)$$

**例子**：

1. **项**：$x, f(x), g(x, y), 0, S(0)$（自然数）
2. **原子公式**：$P(x), Q(x, y), x = y$
3. **量词公式**：$\forall x P(x), \exists x (P(x) \land Q(x))$
4. **复合公式**：$\forall x (P(x) \to \exists y Q(x, y))$

## 三、语法规则的特点

**递归定义**：

- 公式通过递归规则定义
- 从简单到复杂
- 确保公式的良构性

**括号约定**：

- 使用括号明确运算顺序
- 通常约定：→ 的优先级低于 ∧ 和 ∨
- 右结合：A → B → C 表示 A → (B → C)

**自由变量与约束变量**：

- 在 ∀x φ 和 ∃x φ 中，x是约束变量
- 不在量词作用范围内的变量是自由变量
- 替换时必须注意变量冲突

## 四、与经典逻辑的语法对比

**语法相同**：

直觉主义逻辑与经典逻辑的语法是相同的：

- 相同的连接词
- 相同的量词
- 相同的公式结构

**差异在语义**：

- 语法结构相同
- 差异在于语义解释
- 差异在于推理规则

**意义**：

- 形式化使得语法可以精确描述
- 语法与语义分离
- 便于比较不同逻辑系统
