## 一、直觉主义算术（HA）

**海廷算术（Heyting Arithmetic, HA）**：

海廷建立了直觉主义算术的形式系统，基于直觉主义逻辑。

**历史背景**：

- 1930年代：海廷完成直觉主义逻辑的形式化
- 随后：建立直觉主义算术系统
- 影响：为构造性数学奠定基础

**基本特征**：

- 基于直觉主义逻辑
- 强调构造性证明
- 拒绝非构造性推理

## 二、公理系统

**皮亚诺公理（直觉主义版本）**：

```
P1: 0 是自然数
P2: 如果 n 是自然数，则 S(n) 是自然数
P3: 0 不是任何数的后继
P4: 如果 S(m) = S(n)，则 m = n
P5: 归纳原理（直觉主义版本）
```

**归纳原理**：

对于任意性质 P：

```
如果 P(0) 成立，且对于所有 n，P(n) → P(S(n)) 成立，
则对于所有 n，P(n) 成立。
```

**构造性要求**：

- 归纳证明必须是构造性的
- 必须提供构造方法
- 确保可计算性

## 三、构造性要求

**存在性证明**：

在直觉主义算术中，证明 ∃x A(x) 必须：

- 提供具体的 x
- 提供 A(x) 的证明
- 证明必须是可计算的

**全称性证明**：

证明 ∀x A(x) 必须：

- 提供对每个 x 证明 A(x) 的方法
- 方法是可计算的
- 构造是明确的

**与经典算术的对比**：

| 特性 | 经典算术 | 直觉主义算术 |
|------|---------|------------|
| 排中律 | 成立 | 不成立 |
| 存在性证明 | 非构造性 | 必须构造性 |
| 可计算性 | 不要求 | 要求 |

## 四、与经典算术的差异

**排中律的拒绝**：

在直觉主义算术中，A ∨ ¬A 不总是成立。

**双重否定**：

¬¬A → A 不成立，但 A → ¬¬A 成立。

**存在性**：

- 经典算术：可以使用非构造性证明
- 直觉主义算术：必须提供构造方法

**实际影响**：

- 某些经典定理不成立
- 需要不同的证明方法
- 强调可计算性

**构造性数学**：

- 为构造性数学提供基础

**类型论**：

- 为类型论提供基础
