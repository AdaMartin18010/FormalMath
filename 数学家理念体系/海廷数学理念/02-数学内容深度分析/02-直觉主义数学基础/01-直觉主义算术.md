## 一、直觉主义算术（HA）

**海廷算术（Heyting Arithmetic, HA）**：

海廷建立了直觉主义算术的形式系统，基于直觉主义逻辑。

**历史背景**：

- 1930年代：海廷完成直觉主义逻辑的形式化
- 随后：建立直觉主义算术系统
- 影响：为构造性数学奠定基础

**基本特征**：

- 基于直觉主义逻辑
- 强调构造性证明
- 拒绝非构造性推理

## 二、公理系统

**皮亚诺公理（直觉主义版本）**：

$$
\begin{align}
\text{P1: } & 0 \in \mathbb{N} \\
\text{P2: } & \forall n \in \mathbb{N}: S(n) \in \mathbb{N} \\
\text{P3: } & \forall n \in \mathbb{N}: S(n) \neq 0 \\
\text{P4: } & \forall m, n \in \mathbb{N}: S(m) = S(n) \to m = n \\
\text{P5: } & \text{归纳原理（直觉主义版本）}
\end{align}
$$

**归纳原理**：

对于任意性质 $P$：

$$\frac{P(0) \quad \forall n (P(n) \to P(S(n)))}{\forall n P(n)}$$

形式化表述：
$$P(0) \land (\forall n (P(n) \to P(S(n)))) \to \forall n P(n)$$

**构造性要求**：

- 归纳证明必须是构造性的
- 必须提供构造方法
- 确保可计算性

**例子**：

证明 $\forall n: n + 0 = n$：
- 基础：$0 + 0 = 0$（定义）
- 归纳：假设 $n + 0 = n$，则 $S(n) + 0 = S(n + 0) = S(n)$
- 构造性：对每个 $n$，可以计算 $n + 0 = n$

## 三、构造性要求

**存在性证明**：

在直觉主义算术中，证明 $\exists x A(x)$ 必须：

- 提供具体的 $x_0$
- 提供 $A(x_0)$ 的证明
- 证明必须是可计算的

**数学表述**：

证明 $\exists x A(x)$ 需要：
$$\exists x_0: A(x_0) \land \text{Proof}(A(x_0))$$

其中 $\text{Proof}(A(x_0))$ 是构造性证明。

**全称性证明**：

证明 $\forall x A(x)$ 必须：

- 提供对每个 $x$ 证明 $A(x)$ 的方法
- 方法是可计算的
- 构造是明确的

**数学表述**：

证明 $\forall x A(x)$ 需要：
$$\exists f: \mathbb{N} \to \text{Proof}, \forall x: f(x) \text{ 是 } A(x) \text{ 的证明}$$

其中 $f$ 是可计算函数。

**与经典算术的对比**：

| 特性 | 经典算术 | 直觉主义算术 |
|------|---------|------------|
| 排中律 | $A \lor \neg A$ 成立 | $A \lor \neg A$ 不总是成立 |
| 存在性证明 | 非构造性：$\neg \neg \exists x A(x) \to \exists x A(x)$ | 必须构造性：$\exists x A(x) \to \exists x_0 A(x_0)$ |
| 可计算性 | 不要求 | 要求所有对象可计算 |

**例子**：

1. **存在性**：证明 $\exists n: n^2 = 4$ 需要提供 $n = 2$ 和 $2^2 = 4$ 的证明
2. **全称性**：证明 $\forall n: n + 0 = n$ 需要对每个 $n$ 提供构造性证明

## 四、与经典算术的差异

**排中律的拒绝**：

在直觉主义算术中，A ∨ ¬A 不总是成立。

**双重否定**：

¬¬A → A 不成立，但 A → ¬¬A 成立。

**存在性**：

- 经典算术：可以使用非构造性证明
- 直觉主义算术：必须提供构造方法

**实际影响**：

- 某些经典定理不成立
- 需要不同的证明方法
- 强调可计算性
