## 一、形式化的基本概念

**形式化的定义**：

形式化是将数学或逻辑概念用精确的符号和规则表达的过程。

**形式化的要素**：

1. **语法**：符号和公式的规则
2. **公理**：基本假设
3. **推理规则**：从前提推出结论的规则
4. **语义**：符号和公式的含义

**形式化的目的**：

- 提供精确的表述
- 使推理可以机械化
- 便于研究和传播

## 二、海廷的形式化方法

**海廷的贡献**：

海廷在1930年完成了直觉主义逻辑的形式化，建立了完整的逻辑系统。

**形式化的特点**：

- **系统性**：完整的语法和推理规则
- **构造性**：保持直觉主义的构造性要求
- **精确性**：使用精确的符号和规则

**形式化的步骤**：

1. 定义语法
2. 建立公理系统
3. 给出推理规则
4. 提供语义解释

## 三、语法定义

**命题逻辑的语法**：

```
原子命题：p, q, r, ...
连接词：∧, ∨, →, ⊥
公式：φ ::= p | ⊥ | (φ ∧ φ) | (φ ∨ φ) | (φ → φ)
```

**一阶逻辑的语法扩展**：

```
项：t ::= x | f(t₁, ..., tₙ)
原子公式：P(t₁, ..., tₙ)
公式：φ ::= P(t₁, ..., tₙ) | ⊥ | (φ ∧ φ) | (φ ∨ φ) | (φ → φ) | (∀x φ) | (∃x φ)
```

**语法规则的重要性**：

- 确保公式的良构性
- 便于形式化处理
- 为推理提供基础

## 四、公理系统

**海廷公理系统**：

海廷给出了直觉主义逻辑的公理系统，包括：

**命题逻辑公理**：

```
H1: A → (B → A)
H2: (A → (B → C)) → ((A → B) → (A → C))
H3: A ∧ B → A
H4: A ∧ B → B
H5: A → (B → A ∧ B)
H6: A → A ∨ B
H7: B → A ∨ B
H8: (A → C) → ((B → C) → (A ∨ B → C))
H9: ⊥ → A
```

**公理系统的性质**：

- **完备性**：所有有效的公式都可证
- **一致性**：不会推出矛盾
- **独立性**：公理之间相互独立

## 五、推理规则

**分离规则（Modus Ponens）**：

直觉主义逻辑的唯一推理规则：

```
从 A 和 A → B 推出 B
```

**自然演绎系统**：

也可以使用自然演绎系统，包括：

- **引入规则**：如何引入连接词
- **消去规则**：如何使用连接词

**推理规则的特点**：

- 保持构造性
- 确保可计算性
- 便于形式化处理

## 六、语义解释

**BHK解释**：

海廷与布劳威尔、柯尔莫哥洛夫共同建立了BHK解释：

- **合取**：A ∧ B 的证明是 A 的证明和 B 的证明
- **析取**：A ∨ B 的证明是 A 的证明或 B 的证明
- **蕴含**：A → B 的证明是将 A 的证明转化为 B 的证明的函数

**海廷代数**：

海廷代数提供代数语义：

- 命题对应代数元素
- 连接词对应代数运算
- 有效性通过代数关系定义

## 七、系统构建原则

**从简单到复杂**：

- 先定义简单的语法
- 逐步扩展到复杂结构
- 保持系统的清晰性

**保持构造性**：

- 所有证明必须是构造性的
- 存在性证明必须提供构造方法
- 确保可计算性

**确保一致性**：

- 系统不能推出矛盾
- 公理之间要协调
- 推理规则要可靠
