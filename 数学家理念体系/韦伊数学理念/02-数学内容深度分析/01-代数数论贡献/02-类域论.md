### 1.1 经典类域论

**Hilbert的类域论（1890s）**：

- **数域的阿贝尔扩张**：Hilbert研究数域的Abel扩张
  - 研究数域 $K$ 的Abel扩张 $L/K$
  - 建立理想类群与Abel扩张的对应关系
- **理想类群的对应**：建立理想类群与Abel扩张的对应
  - 理想类群 $\text{Cl}_K$ 对应Abel扩张
  - 类域的概念：每个理想类对应一个Abel扩张
- **类域的概念**：引入"类域"的概念
  - 类域是数域的Abel扩张
  - 类域论研究类域的存在性和唯一性

**Takagi的类域论（1920）**：

- **一般类域论**：Takagi (1920) 建立一般类域论
  - 推广Hilbert的类域论到一般情况
  - 建立一般类域论的存在性和唯一性定理
- **存在性定理**：证明类域的存在性
  - 对于每个理想类，存在对应的类域
  - 类域的存在性定理是类域论的基础
- **唯一性定理**：证明类域的唯一性
  - 每个理想类对应唯一的类域
  - 类域的唯一性定理是类域论的核心

**Artin互反律（1927）**：

- **建立对应关系**：Artin (1927) 建立互反律
  - 建立理想类群与Galois群的对应关系
  - Artin互反律是类域论的核心定理
- **类域论的核心定理**：Artin互反律是类域论的核心定理
  - 对于数域 $K$ 的Abel扩张 $L/K$，存在同构：
    $$\text{Gal}(L/K) \cong \mathbb{I}_K / K^* N_{L/K}(\mathbb{I}_L)$$
  - 其中 $N_{L/K}$ 是范映射
- **互反律的表述**：建立理想类群与Galois群的对应
  - 理想类群 $\text{Cl}_K$ 对应Galois群 $\text{Gal}(K^{\text{ab}}/K)$
  - 这是类域论的核心对应关系

## 二、韦伊的类域论方法

### 2.1 Adèle环

**定义**：

对于数域 $K$，Adèle环定义为：

$$\mathbb{A}_K = \prod_{v}' K_v = \leqft\{ (x_v) \in \prod_{v} K_v : x_v \in \mathcal{O}_v \text{ 对几乎所有 } v \right\}$$

其中 $\prod'$ 表示**限制直积**（restricted direct product），$\mathcal{O}_v$ 是 $K_v$ 的整数环。

**限制直积的条件**：

- **几乎所有分量在整数环中**：对于几乎所有赋值 $v$，$x_v \in \mathcal{O}_v$
- **只有有限个例外**：只有有限个赋值 $v$，使得 $x_v \notin \mathcal{O}_v$
- **统一构造**：这个构造适用于数域和函数域

**性质**：

- **局部紧拓扑群**：Adèle环是局部紧拓扑群
  - 每个局部域 $K_v$ 是局部紧的
  - 限制直积保持局部紧性
  - Adèle环是局部紧拓扑群
- **在数论中起关键作用**：Adèle环在数论中起关键作用
  - 统一局部与整体的方法
  - 在类域论中的应用
  - 在Langlands纲领中的应用
- **统一局部与整体**：Adèle环统一局部与整体
  - 局部域 $K_v$ 的信息（局部）
  - 数域 $K$ 的信息（整体）
  - 通过Adèle环连接局部与整体

### 2.2 Idèle群

**定义**：

Idèle群是Adèle环的单位群：

$$\mathbb{I}_K = \mathbb{A}_K^* = \leqft\{ (x_v) \in \mathbb{A}_K : x_v \neqq 0 \text{ 对所有 } v, x_v \in \mathcal{O}_v^* \text{ 对几乎所有 } v \right\}$$

**限制直积的条件**：

- **所有分量非零**：对于所有赋值 $v$，$x_v \neqq 0$
- **几乎所有分量在单位群中**：对于几乎所有赋值 $v$，$x_v \in \mathcal{O}_v^*$
- **只有有限个例外**：只有有限个赋值 $v$，使得 $x_v \notin \mathcal{O}_v^*$

**性质**：

- **局部紧拓扑群**：Idèle群是局部紧拓扑群
  - Idèle群是Adèle环的可逆元群
  - 可逆元群保持局部紧性
  - Idèle群是局部紧拓扑群
- **类域论的核心对象**：Idèle群是类域论的核心对象
  - Artin互反律通过Idèle群表述
  - $\mathbb{I}_K / K^* \to \text{Gal}(K^{\text{ab}}/K)$
  - 这是类域论的核心对应关系
- **与理想类群的关系**：$\mathbb{I}_K / K^* \cong \text{Cl}_K$（数域）或 $\text{Pic}_K$（函数域）
  - 数域：$\mathbb{I}_K / K^* \cong \text{Cl}_K$（理想类群）
  - 函数域：$\mathbb{I}_K / K^* \cong \text{Pic}_K$（除子类群）
  - 统一的对应：通过Idèle群统一数域与函数域

## 三、Adèle与Idèle理论

### 3.1 基本理论

**Adèle环的性质**：

- **拓扑结构**：局部紧拓扑群
  - Adèle环具有自然的拓扑结构
  - 每个局部域 $K_v$ 是局部紧的
  - 限制直积保持局部紧性
- **局部紧性**：每个局部域 $K_v$ 是局部紧的
  - 有限素位：$K_v$ 是p进域，局部紧
  - 无限素位：$K_v$ 是 $\mathbb{R}$ 或 $\mathbb{C}$，局部紧
  - Adèle环是局部紧拓扑群
- **在数论中的应用**：统一局部与整体的方法
  - 通过Adèle环连接局部域 $K_v$ 与数域 $K$
  - 在类域论中的应用
  - 在Langlands纲领中的应用

**Idèle群的性质**：

- **类域论的核心**：Artin互反律通过Idèle群表述
  - Idèle群是类域论的核心对象
  - Artin互反律：$\mathbb{I}_K / K^* \to \text{Gal}(K^{\text{ab}}/K)$
  - 这是类域论的核心对应关系
- **与理想类群的关系**：$\mathbb{I}_K / K^* \cong \text{Cl}_K$（数域）或 $\text{Pic}_K$（函数域）
  - 数域：$\mathbb{I}_K / K^* \cong \text{Cl}_K$（理想类群）
  - 函数域：$\mathbb{I}_K / K^* \cong \text{Pic}_K$（除子类群）
  - 统一的对应：通过Idèle群统一数域与函数域
- **在Langlands纲领中的应用**：自守表示的研究
  - 自守表示是 $\text{GL}_n(\mathbb{A}_K)$ 的表示
  - Idèle群在自守表示的研究中起关键作用
  - Langlands对应通过Idèle群实现

### 3.2 Artin互反律的表述

**经典表述**：

对于数域 $K$ 的Abel扩张 $L/K$，存在同构：

$$\text{Gal}(L/K) \cong \mathbb{I}_K / K^* N_{L/K}(\mathbb{I}_L)$$

其中 $N_{L/K}$ 是范映射。

**Idèle表述（韦伊）**：

对于数域或函数域 $K$ 的Abel扩张 $L/K$，存在同态：

$$\mathbb{I}_K / K^* \to \text{Gal}(L/K)$$

这个同态是满射，核是 $K^* N_{L/K}(\mathbb{I}_L)$。

**函数域情况**：

对于函数域，类似地有：

$$\mathbb{I}_K / K^* \to \text{Gal}(K^{\text{ab}}/K)$$

其中 $K^{\text{ab}}$ 是 $K$ 的最大Abel扩张。

**韦伊的统一**：

- **通过Idèle群统一表述**：Artin互反律通过Idèle群统一表述，适用于数域与函数域
- **适用于数域与函数域**：统一的表述适用于数域与函数域
- **为Langlands纲领提供基础**：统一的类域论为Langlands纲领提供基础

**具体优势**：

- **统一框架**：通过Idèle群提供统一的框架
  - 数域与函数域的类域论通过Idèle群统一表述
  - 统一的表述：$\mathbb{I}_K / K^* \to \text{Gal}(K^{\text{ab}}/K)$
  - 为现代数论提供基础
- **局部-整体方法**：通过Adèle/Idèle方法实现局部-整体统一
  - 通过Adèle环连接局部域与数域
  - 通过Idèle群实现局部-整体统一
  - 为现代数论提供统一方法
- **现代发展**：为Langlands纲领提供基础
  - 类域论是Langlands纲领的 $n=1$ 情况
  - 统一的类域论为一般Langlands纲领提供基础
  - Langlands纲领是韦伊统一思想的现代发展

## 四、现代发展

### 4.1 Langlands纲领

**韦伊类域论的推广**：

- **从Abel扩张到一般扩张**：Langlands纲领将类域论推广到非Abel情况
  - 类域论：$n=1$ 的情况，对应Abel扩张
  - Langlands纲领：$n \geqq 2$ 的情况，对应非Abel扩张
  - 这是韦伊统一思想的现代发展
- **Galois表示 ↔ 自守表示**：Langlands对应
  - Galois表示：$\rho: \text{Gal}(\bar{K}/K) \to \text{GL}_n(\mathbb{C})$
  - 自守表示：$\pi$ 是 $\text{GL}_n(\mathbb{A}_K)$ 的自守表示
  - Langlands对应：$\rho \leqftrightarrow \pi$，保持L函数
- **几何Langlands纲领**：几何化的Langlands对应
  - Fargues-Scholze (2021) 实现局部Langlands对应的几何化
  - 几何Langlands纲领连接数论与几何
  - 体现韦伊的统一思想

**Langlands对应**：

对于数域 $K$，存在对应：

$$\text{Gal}(\bar{K}/K) \text{ 的 } n \text{ 维表示} \leqftrightarrow \text{GL}_n(\mathbb{A}_K) \text{ 的自守表示}$$

其中：

- 左边是Galois群的 $n$ 维表示（拓扑群的表示）
- 右边是 $\text{GL}_n(\mathbb{A}_K)$ 的自守表示（拓扑群的表示）
- 对应保持L函数：$L(s, \rho) = L(s, \pi)$

**与类域论的关系**：

- **类域论**：$n=1$ 的情况，对应Abel扩张
  - Artin互反律：$\mathbb{I}_K / K^* \to \text{Gal}(K^{\text{ab}}/K)$
  - 这是Langlands纲领的 $n=1$ 情况
- **一般Langlands纲领**：$n \geqq 2$ 的情况，对应非Abel扩张
  - 这是韦伊类比的现代推广
  - Langlands纲领是韦伊统一思想的现代发展

### 4.2 2024-2025最新进展

**Fargues-Scholze几何化**：

- **局部Langlands对应的几何实现**：Fargues-Scholze (2021) 实现了局部Langlands对应的几何化
  - 局部Langlands对应：局部域的Galois表示与自守表示的对应
  - 几何化：通过几何方法实现局部Langlands对应
  - 完美空间理论的应用：使用完美空间理论
- **完美空间理论的应用**：使用完美空间理论
  - 完美空间是p进几何的新工具
  - Fargues-Fontaine曲线在几何Langlands中的应用
  - 为函数域与数域的类比提供新框架
- **凝聚数学的统一框架**：肖尔策的凝聚数学为统一提供新视角
  - 肖尔策 & Clausen (2020) 建立凝聚数学理论
  - 统一拓扑与代数，为现代数学提供新框架
  - 继承韦伊的统一思想，提供新的统一方法

**最新成果**：

- **几何Langlands纲领的新进展**：
  - Fargues-Scholze的工作继续发展
  - 局部Langlands对应的完整几何实现

  - 局部Langlands对应的几何化
