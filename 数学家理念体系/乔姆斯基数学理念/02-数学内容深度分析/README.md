## 🎯 核心概念

### Chomsky层次

**Chomsky层次**将形式语言按照生成能力分为四个层次：

1. **Type-0**：递归可枚举语言（最强大）
2. **Type-1**：上下文相关语言
3. **Type-2**：上下文无关语言
4. **Type-3**：正则语言（最受限）

### 语言与自动机的对应

- **正则语言** $\leftrightarrow$ **有限自动机**
- **上下文无关语言** $\leftrightarrow$ **下推自动机**
- **上下文相关语言** $\leftrightarrow$ **线性有界自动机**
- **递归可枚举语言** $\leftrightarrow$ **图灵机**

### 泵引理

**泵引理**用于证明语言不属于某个层次：

- **正则语言泵引理**：如果 $L$ 是正则语言，则存在常数 $p$，使得对任意长度 $\geq p$ 的字符串，可以"泵"出无限多个字符串
- **上下文无关语言泵引理**：类似的结论对上下文无关语言成立
