## 二、短语结构语法

### 2.1 基本概念

**短语结构语法（PSG）**：

- 使用产生式规则
- 从起始符号生成句子
- 建立句法结构

**产生式规则**：

- 形如 $A \to \alpha$
- $A$ 是变元（非终结符）
- $\alpha$ 是符号串

---

### 2.2 推导过程

**推导（Derivation）**：

1. 从起始符号 $S$ 开始
2. 应用产生式规则
3. 逐步替换变元
4. 最终得到终结符串

**例子**：

产生式规则：
$$S \to NP\ VP$$
$$NP \to \text{the}\ N$$
$$VP \to V\ NP$$
$$N \to \text{cat} \mid \text{dog}$$
$$V \to \text{chases}$$

推导过程：
$$S \Rightarrow NP\ VP \Rightarrow \text{the}\ N\ VP \Rightarrow \text{the}\ \text{cat}\ VP$$
$$\Rightarrow \text{the}\ \text{cat}\ V\ NP \Rightarrow \text{the}\ \text{cat}\ \text{chases}\ NP$$
$$\Rightarrow \text{the}\ \text{cat}\ \text{chases}\ \text{the}\ N \Rightarrow \text{the}\ \text{cat}\ \text{chases}\ \text{the}\ \text{dog}$$

---

### 2.3 语法树

**语法树（Parse Tree）**：

- 表示推导过程的树形结构
- **节点**：符号（变元或终结符）
- **根节点**：起始符号 $S$
- **叶节点**：终结符（句子中的词）
- **边**：表示产生式规则的应用

**例子**（"the cat chases the dog"）：

```
        S
       / \
      /   \
     NP   VP
    / \   / \
   /   \ /   \
  the  N  V   NP
       |  |   / \
      cat |  /   \
         chases the N
                  |
                 dog
```

**性质**：

- 每个句子对应语法树
- 语法树表示句法结构
- 可能有多个语法树（歧义）

## 三、转换语法

### 3.1 转换规则

**转换语法（Transformational Grammar）**：

- **深层结构（Deep Structure）**：语义表示，基础语法结构
- **表层结构（Surface Structure）**：实际句子
- **转换规则**：连接深层结构与表层结构

**转换规则**：

- 改变句子结构
- 保持语义不变
- 例如：
  - **被动转换**：主动句 $\to$ 被动句
  - **疑问转换**：陈述句 $\to$ 疑问句
  - **否定转换**：肯定句 $\to$ 否定句

**例子**（被动转换）：

- **深层结构**：John loves Mary
- **转换规则**：被动转换
- **表层结构**：Mary is loved by John

---

### 3.2 深层结构与表层结构

**深层结构（Deep Structure）**：

- **语义表示**：句子的语义内容
- **基础语法结构**：通过短语结构语法生成
- **抽象层次**：不一定是实际句子

**性质**：

- 表示句子的语义关系
- 通过短语结构语法生成
- 是转换的起点

**表层结构（Surface Structure）**：

- **实际句子**：人们实际说出或写出的句子
- **转换结果**：通过转换规则从深层结构得到
- **具体形式**：符合语言的表层语法

**关系**：

- 深层结构 $\xrightarrow{\text{转换规则}}$ 表层结构
- 多个表层结构可能对应同一个深层结构
- 一个表层结构可能对应多个深层结构（歧义）

## 九、总结

生成语法理论展示了乔姆斯基的核心贡献，具有深远的历史意义和现代影响：

### 9.1 历史意义

1. **建立生成语法框架**：首次系统地形式化语言生成过程
2. **连接语法与语义**：建立了深层结构与表层结构的理论
3. **影响语言学**：改变了语言学的研究方向

### 9.2 技术贡献

1. **生成机制**：建立了语言生成的数学理论
2. **语法结构**：提供了表示语法结构的方法（语法树）
3. **转换规则**：连接深层结构与表层结构
4. **规则系统**：建立了多层次的规则系统

### 9.3 现代影响

1. **语言学**：为现代语言学提供理论基础
2. **计算语言学**：为自然语言处理提供基础
3. **语言习得**：影响语言习得理论
4. **形式化方法**：推动语言学的形式化

这些成果不仅深化了我们对语言生成的理解，也为现代语言学和计算语言学提供了重要的理论基础，使乔姆斯基成为20世纪最重要的语言学家之一。
