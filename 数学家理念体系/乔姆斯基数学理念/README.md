## 📑 项目概述

本项目旨在系统化分析诺姆·乔姆斯基（Noam Chomsky, 1928-）的数学理念、方法论和具体数学贡献，聚焦其形式语言理论、Chomsky层次、生成语法等核心贡献。

**参照标准**：哥德尔数学理念项目（已完成93%+）

**核心定位**：

- 🔄 **数学思想**：形式语言理论、生成语法、Chomsky层次
- 🔄 **具体数学内容**：形式语法、自动机理论、语言层次、计算复杂性
- 🔄 **历史影响**：对计算理论的影响、与图灵、Post的关系
- 🔄 **现代验证**：在计算机科学、语言学、计算复杂性中的应用

## 🎯 核心主题

### 1. 形式语言理论的创立

- 形式语言的定义
- 形式语法
- 语言层次

### 2. Chomsky层次

- 正则语言（Type-3）
- 上下文无关语言（Type-2）
- 上下文相关语言（Type-1）
- 递归可枚举语言（Type-0）

### 3. 生成语法理论

- 短语结构语法
- 转换语法
- 语言生成

3. **Chomsky, N., & Schützenberger, M. P. (1963).** The algebraic theory of context-free languages. In *Computer Programming and Formal Systems* (pp. 118-161). North-Holland.
