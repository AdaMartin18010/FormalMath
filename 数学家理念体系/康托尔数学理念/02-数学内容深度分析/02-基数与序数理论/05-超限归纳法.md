## 一、历史背景

### 1.1 数学归纳法

**传统归纳法**：

数学归纳法用于证明关于自然数的命题：

- **基础步骤**：证明P(0)
- **归纳步骤**：如果P(n)，则P(n+1)
- **结论**：对所有自然数n，P(n)

**局限性**：

数学归纳法只适用于自然数（有限序数）。

---

### 1.2 康托尔的突破

**超限归纳法**：

康托尔将归纳法推广到所有序数，包括超限序数。

**关键思想**：

- 对0成立
- 如果对所有β < α成立，则对α成立
- 结论：对所有序数成立

**革命性意义**：

这是归纳法从有限到超限的推广。

---

## 二、超限归纳原理

### 2.1 超限归纳的表述

**定理（超限归纳原理）**：

设P(α)是关于序数α的命题，如果：

1. **基础步骤**：P(0)成立
2. **归纳步骤**：对任意序数α，如果对所有β < α，P(β)成立，则P(α)成立

则对所有序数α，P(α)成立。

**证明**：

假设存在序数α使得P(α)不成立，则集合{α : ¬P(α)}非空，有最小元α₀。但根据归纳步骤，P(α₀)应该成立，矛盾。

---

### 2.2 三种情况

**基础情况**：

P(0)成立。

**后继情况**：

如果P(α)成立，则P(α+1)成立。

**极限情况**：

如果α是极限序数，且对所有β < α，P(β)成立，则P(α)成立。

---

### 2.3 与数学归纳法的关系

**数学归纳法**：

只处理基础情况和后继情况。

**超限归纳法**：

还处理极限情况。

**推广**：

超限归纳法是数学归纳法的自然推广。

---

## 三、超限递归

### 3.1 超限递归原理

**定理（超限递归原理）**：

设F是函数，则存在唯一的函数f定义在所有序数上，使得：

```text
f(α) = F(f|_α)
```

其中f|_α是f在{β : β < α}上的限制。

**意义**：

这允许通过"之前的值"定义f(α)。

---

### 3.2 超限递归的例子

**序数加法**：

```text
α + 0 = α
α + (β+1) = (α+β) + 1
α + λ = sup{α+β : β < λ}（λ是极限序数）
```

**序数乘法**：

```text
α · 0 = 0
α · (β+1) = α·β + α
α · λ = sup{α·β : β < λ}（λ是极限序数）
```

**序数幂**：

```text
α^0 = 1
α^{β+1} = α^β · α
α^λ = sup{α^β : β < λ}（λ是极限序数）
```

---

### 3.3 超限递归的唯一性

**唯一性定理**：

满足递归关系的函数是唯一的。

**证明**：

使用超限归纳法。

**意义**：

这保证了递归定义的合理性。

- 超限归纳法更一般

**应用**：
