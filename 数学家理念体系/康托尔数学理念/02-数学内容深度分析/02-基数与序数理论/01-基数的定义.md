## 一、历史背景

### 1.1 康托尔的突破

**1874年**：

康托尔首次系统研究了集合的"大小"，引入了基数的概念。

**关键思想**：

- 集合的"大小"可以用基数来度量
- 存在不同大小的无穷
- 基数可以比较

**革命性意义**：

这是数学史上对无穷概念的根本性突破。

---

### 1.2 从有限到无穷

**有限集**：

有限集的基数就是元素的个数。

**无穷集**：

无穷集的基数需要新的定义。

**康托尔的创新**：

使用"等势"来定义基数。

---

## 二、等势关系

### 2.1 等势的定义

**定义**：

集合A和B等势（equipotent），如果存在双射f: A → B。

**记号**：

|A| = |B| 表示A和B等势。

**例子**：

- 自然数集N和整数集Z等势
- 自然数集N和有理数集Q等势
- 自然数集N和实数集R不等势

---

### 2.2 等势的性质

**自反性**：

|A| = |A|（恒等映射是双射）

**对称性**：

如果|A| = |B|，则|B| = |A|（逆映射是双射）

**传递性**：

如果|A| = |B|且|B| = |C|，则|A| = |C|（复合映射是双射）

**结论**：

等势是等价关系。

---

### 2.3 等势的例子

**自然数与整数**：

```text
f: N → Z
f(1) = 0
f(2n) = n
f(2n+1) = -n
```

这是双射，因此|N| = |Z|。

**自然数与有理数**：

有理数集Q是可数的，因此|N| = |Q|。

**自然数与实数**：

实数集R是不可数的，因此|N| < |R|。

---

## 三、基数的定义

### 3.1 基数的形式定义

**在ZFC中**：

基数可以定义为：

- 有限集的基数：自然数n
- 无穷集的基数：最小的序数α使得|A| = |α|

**等价类定义**：

基数可以定义为等势关系的等价类。

**选择公理**：

在ZFC中，使用选择公理可以证明每个集合都有基数。

---

### 3.2 基数的记号

**有限基数**：

0, 1, 2, 3, ...（自然数）

**可数基数**：

ℵ₀（aleph-zero）：可数无穷的基数

**连续统基数**：

𝔠（continuum）：实数集的基数

**更大的基数**：

ℵ₁, ℵ₂, ℵ₃, ...（假设选择公理）

---

### 3.3 基数的唯一性

**定理**：

在ZFC中，每个集合都有唯一的基数。

**证明**：

使用选择公理和良序定理。

**意义**：

这保证了基数的良好定义。

---

## 四、基数的比较

### 4.1 基数比较的定义

**定义**：

- |A| ≤ |B|：如果存在单射f: A → B
- |A| < |B|：如果|A| ≤ |B|且|A| ≠ |B|

**例子**：

- |N| ≤ |R|
- |N| < |R|

---

### 4.2 康托尔-伯恩斯坦定理

**定理**：

如果|A| ≤ |B|且|B| ≤ |A|，则|A| = |B|。

**证明思路**：

使用Schröder-Bernstein定理构造双射。

**意义**：

这保证了基数比较的合理性。

---

### 4.3 基数的三歧性

**定理（选择公理）**：

对任意集合A和B，以下三者之一成立：

- |A| < |B|
- |A| = |B|
- |A| > |B|

**证明**：

需要选择公理。

**意义**：

这保证了基数的全序性（在AC下）。

---

## 五、有限基数

### 5.1 自然数作为基数

**定义**：

自然数n可以定义为集合{0, 1, 2, ..., n-1}的基数。

**von Neumann构造**：

```text
0 = ∅
1 = {0} = {∅}
2 = {0, 1} = {∅, {∅}}
3 = {0, 1, 2} = {∅, {∅}, {∅, {∅}}}
...
```

**性质**：

这样定义的自然数既是序数也是基数。

---

### 5.2 有限集的基数

**定理**：

有限集A的基数是唯一的自然数n，使得|A| = n。

**证明**：

使用数学归纳法。

**性质**：

- 如果A是有限的，则|A|是自然数
- 如果|A|是自然数，则A是有限的

---

## 六、可数基数

### 6.1 ℵ₀的定义

**定义**：

ℵ₀ = |N|，即可数无穷的基数。

**性质**：

- ℵ₀是最小的无穷基数
- 所有可数集都有基数ℵ₀

---

### 6.2 可数集的例子

**整数集**：

|Z| = ℵ₀

**有理数集**：

|Q| = ℵ₀

**代数数集**：

代数数集是可数的，因此|代数数集| = ℵ₀

---

### 6.3 可数集的性质

**可数集的并**：

有限个可数集的并是可数的。

**可数集的笛卡尔积**：

有限个可数集的笛卡尔积是可数的。

**可数集的子集**：

可数集的子集要么有限，要么可数。

---

## 七、不可数基数

### 7.1 连续统的基数

**定义**：

𝔠 = |R|，即连续统的基数。

**性质**：

- 𝔠 > ℵ₀
- 𝔠 = |P(N)| = 2^ℵ₀

---

### 7.2 不可数集的例子

**实数集**：

|R| = 𝔠

**无理数集**：

无理数集是不可数的，|无理数集| = 𝔠

**超越数集**：

超越数集是不可数的，|超越数集| = 𝔠

---

### 7.3 更大的基数

**幂集的基数**：

根据康托尔定理，|P(A)| > |A|。

**基数层次**：

```text
ℵ₀ < 𝔠 = 2^ℵ₀ < 2^𝔠 < 2^{2^𝔠} < ...
```

**不存在最大基数**：

不存在最大的基数。

## 八、基数的运算

### 8.1 基数加法

**定义**：

如果A ∩ B = ∅，则：

```text
|A| + |B| = |A ∪ B|
```

**一般定义**：

```text
|A| + |B| = |A × {0} ∪ B × {1}|
```

**性质**：

- **交换律**：|A| + |B| = |B| + |A|
- **结合律**：(|A| + |B|) + |C| = |A| + (|B| + |C|)

---

### 8.2 基数乘法

**定义**：

```text
|A| · |B| = |A × B|
```

**性质**：

- **交换律**：|A| · |B| = |B| · |A|
- **结合律**：(|A| · |B|) · |C| = |A| · (|B| · |C|)
- **分配律**：|A| · (|B| + |C|) = |A| · |B| + |A| · |C|

---

### 8.3 基数幂

**定义**：

```text
|B|^{|A|} = |B^A|
```

其中B^A是所有从A到B的函数的集合。

**性质**：

- **幂的幂**：(|B|^{|A|})^{|C|} = |B|^{|A| · |C|}
- **积的幂**：|B|^{|A| + |C|} = |B|^{|A|} · |B|^{|C|}

**例子**：

```text
2^{ℵ₀} = 𝔠
```

## 九、连续统假设

### 9.1 问题的表述

**连续统假设（CH）**：

是否存在基数严格介于ℵ₀和𝔠之间？

等价地：𝔠 = ℵ₁？

**强连续统假设（GCH）**：

对任意序数α，2^ℵ_α = ℵ_{α+1}

---

### 9.2 独立性结果

**哥德尔（1940）**：

CH与ZFC一致。

**科恩（1963）**：

¬CH与ZFC一致。

**结论**：

CH在ZFC中不可判定。

## 十一、总结

### 基数定义的历史地位

**核心贡献**：

基数的定义是集合论的基础：

1. **理论意义**：
   - 提供了度量集合大小的方法
   - 建立了无穷基数的理论
   - 连接了有限与无穷

2. **方法论意义**：
   - **等势**：定义基数的基础
   - **比较**：基数的比较方法
   - **运算**：基数的运算

3. **应用意义**：
   - **数学**：所有数学对象的基础
   - **逻辑学**：模型论
   - **计算机科学**：计算理论

**与数学发展的关系**：

基数的定义是数学史上的重要里程碑：

- **从有限到无穷**：统一的基数概念
- **从直觉到严格**：严格的数学定义
- **从单一到层次**：不同层次的无穷

康托尔的基数定义将数学从有限提升到无穷，从单一提升到层次，为现代数学奠定了基础。从等势到基数，从可数到不可数，从连续统到更大的基数，这一理论展示了数学的深刻统一性和无穷的丰富性。
