# 集合运算：并、交、差、补

## 一、并集运算

### 1.1 两个集合的并集

**定义**：

集合A和B的并集（union）定义为：

```text
A ∪ B = {x : x ∈ A 或 x ∈ B}
```

**例子**：

```text
{1, 2, 3} ∪ {3, 4, 5} = {1, 2, 3, 4, 5}
```

**文氏图**：

并集可以用文氏图表示，是两个集合的"合并"。

---

### 1.2 多个集合的并集

**有限并集**：

```text
A₁ ∪ A₂ ∪ ... ∪ Aₙ = {x : 存在i使得 x ∈ Aᵢ}
```

**无限并集**：

对指标集I和集合族{Aᵢ : i ∈ I}：

```text
∪ᵢ∈ᵢ Aᵢ = {x : 存在i ∈ I使得 x ∈ Aᵢ}
```

**并集公理**：

ZFC的并集公理保证了并集的存在。

---

### 1.3 并集的性质

**交换律**：

```text
A ∪ B = B ∪ A
```

**结合律**：

```text
(A ∪ B) ∪ C = A ∪ (B ∪ C)
```

因此可以写成 A ∪ B ∪ C。

**幂等律**：

```text
A ∪ A = A
```

**单位元**：

```text
A ∪ ∅ = A
```

空集是并集的单位元。

**吸收律**：

```text
A ∪ (A ∩ B) = A
```

---

## 二、交集运算

### 2.1 两个集合的交集

**定义**：

集合A和B的交集（intersection）定义为：

```text
A ∩ B = {x : x ∈ A 且 x ∈ B}
```

**例子**：

```text
{1, 2, 3} ∩ {3, 4, 5} = {3}
```

**文氏图**：

交集是两个集合的"重叠"部分。

---

### 2.2 多个集合的交集

**有限交集**：

```text
A₁ ∩ A₂ ∩ ... ∩ Aₙ = {x : 对所有i都有 x ∈ Aᵢ}
```

**无限交集**：

对指标集I和集合族{Aᵢ : i ∈ I}：

```text
∩ᵢ∈ᵢ Aᵢ = {x : 对所有i ∈ I都有 x ∈ Aᵢ}
```

**注意**：

如果I = ∅，则∩ᵢ∈∅ Aᵢ = 全集（在ZFC中不存在）。

---

### 2.3 交集的性质

**交换律**：

```text
A ∩ B = B ∩ A
```

**结合律**：

```text
(A ∩ B) ∩ C = A ∩ (B ∩ C)
```

因此可以写成 A ∩ B ∩ C。

**幂等律**：

```text
A ∩ A = A
```

**零元**：

```text
A ∩ ∅ = ∅
```

空集是交集的零元。

**吸收律**：

```text
A ∩ (A ∪ B) = A
```

---

## 三、差集运算

### 3.1 差集的定义

**定义**：

集合A和B的差集（difference）定义为：

```text
A \ B = {x : x ∈ A 且 x ∉ B}
```

也记作 A - B。

**例子**：

```text
{1, 2, 3, 4} \ {3, 4, 5} = {1, 2}
```

**文氏图**：

差集是A中"减去"B的部分。

---

### 3.2 差集的性质

**非交换性**：

```text
A \ B ≠ B \ A（一般情况）
```

**与并集的关系**：

```text
A = (A \ B) ∪ (A ∩ B)
```

**与交集的关系**：

```text
A \ B = A \ (A ∩ B)
```

---

## 四、补集运算

### 4.1 相对补集

**定义**：

如果B ⊆ A，则A \ B称为B在A中的补集（complement），记作 B^c 或 A - B。

**例子**：

如果A = {1, 2, 3, 4, 5}，B = {2, 4}，则：

```text
B^c = A \ B = {1, 3, 5}
```

---

### 4.2 绝对补集

**在朴素集合论中**：

如果存在全集U，则集合A的绝对补集为：

```text
A^c = U \ A
```

**在ZFC中**：

不存在全集，因此没有绝对补集的概念。

**相对补集**：

在ZFC中，只能讨论相对补集（在某个集合中的补集）。

---

### 4.3 补集的性质

**双重补集**：

```text
(A^c)^c = A（在相对补集意义下）
```

**德摩根律**：

```text
(A ∪ B)^c = A^c ∩ B^c
(A ∩ B)^c = A^c ∪ B^c
```

**与并集和交集的关系**：

```text
A ∪ A^c = U（全集）
A ∩ A^c = ∅
```

---

## 五、对称差

### 5.1 定义

**对称差（symmetric difference）**：

集合A和B的对称差定义为：

```text
A △ B = (A \ B) ∪ (B \ A)
```

**等价定义**：

```text
A △ B = (A ∪ B) \ (A ∩ B)
```

**例子**：

```text
{1, 2, 3} △ {3, 4, 5} = {1, 2, 4, 5}
```

---

### 5.2 对称差的性质

**交换律**：

```text
A △ B = B △ A
```

**结合律**：

```text
(A △ B) △ C = A △ (B △ C)
```

**单位元**：

```text
A △ ∅ = A
```

**自逆性**：

```text
A △ A = ∅
```

---

## 六、分配律

### 6.1 并集对交集的分配

**分配律**：

```text
A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
```

**证明**：

```text
x ∈ A ∪ (B ∩ C)
⇔ x ∈ A 或 (x ∈ B 且 x ∈ C)
⇔ (x ∈ A 或 x ∈ B) 且 (x ∈ A 或 x ∈ C)
⇔ x ∈ (A ∪ B) ∩ (A ∪ C)
```

**一般形式**：

```text
A ∪ (∩ᵢ Bᵢ) = ∩ᵢ (A ∪ Bᵢ)
```

---

### 6.2 交集对并集的分配

**分配律**：

```text
A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
```

**证明**：

```text
x ∈ A ∩ (B ∪ C)
⇔ x ∈ A 且 (x ∈ B 或 x ∈ C)
⇔ (x ∈ A 且 x ∈ B) 或 (x ∈ A 且 x ∈ C)
⇔ x ∈ (A ∩ B) ∪ (A ∩ C)
```

**一般形式**：

```text
A ∩ (∪ᵢ Bᵢ) = ∪ᵢ (A ∩ Bᵢ)
```

---

## 七、德摩根律

### 7.1 基本形式

**并集的补集**：

```text
(A ∪ B)^c = A^c ∩ B^c
```

**交集的补集**：

```text
(A ∩ B)^c = A^c ∪ B^c
```

**证明**：

```text
x ∈ (A ∪ B)^c
⇔ x ∉ A ∪ B
⇔ x ∉ A 且 x ∉ B
⇔ x ∈ A^c 且 x ∈ B^c
⇔ x ∈ A^c ∩ B^c
```

---

### 7.2 一般形式

**无限并集的补集**：

```text
(∪ᵢ Aᵢ)^c = ∩ᵢ Aᵢ^c
```

**无限交集的补集**：

```text
(∩ᵢ Aᵢ)^c = ∪ᵢ Aᵢ^c
```

---

## 八、集合运算的应用

### 8.1 在逻辑学中的应用

**命题逻辑**：

集合运算对应逻辑运算：

- **并集** ↔ **或**（∨）
- **交集** ↔ **且**（∧）
- **补集** ↔ **非**（¬）

**布尔代数**：

集合的运算形成布尔代数。

---

### 8.2 在概率论中的应用

**事件**：

事件可以表示为集合。

**概率**：

```text
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
```

**独立事件**：

```text
P(A ∩ B) = P(A) · P(B)
```

---

### 8.3 在拓扑学中的应用

**开集**：

拓扑空间中的开集对并集和有限交集封闭。

**闭集**：

闭集对交集和有限并集封闭。

---

## 九、集合运算的基数

### 9.1 并集的基数

**容斥原理**：

```text
|A ∪ B| = |A| + |B| - |A ∩ B|
```

**三个集合**：

```text
|A ∪ B ∪ C| = |A| + |B| + |C|
              - |A ∩ B| - |A ∩ C| - |B ∩ C|
              + |A ∩ B ∩ C|
```

**一般形式**：

容斥原理可以推广到任意有限个集合。

---

### 9.2 交集的基数

**上界**：

```text
|A ∩ B| ≤ min(|A|, |B|)
```

**下界**：

```text
|A ∩ B| ≥ |A| + |B| - |A ∪ B|
```

---

### 9.3 差集的基数

**公式**：

```text
|A \ B| = |A| - |A ∩ B|
```

**如果B ⊆ A**：

```text
|A \ B| = |A| - |B|
```

---

## 十、总结

### 集合运算的历史地位

**核心贡献**：

集合运算是集合论的基础操作：

1. **理论意义**：
   - 提供了集合的基本操作
   - 建立了集合代数
   - 连接了集合论与逻辑学

2. **方法论意义**：
   - **运算律**：交换律、结合律、分配律
   - **德摩根律**：补集与并集、交集的关系
   - **容斥原理**：计算并集的基数

3. **应用意义**：
   - **逻辑学**：对应逻辑运算
   - **概率论**：计算事件的概率
   - **拓扑学**：开集、闭集的性质

**与数学发展的关系**：

集合运算是数学的基础工具：

- **从元素到集合**：集合的运算
- **从具体到抽象**：抽象的代数结构
- **从单一到统一**：统一的数学语言

康托尔的集合运算将集合从静态对象提升到动态操作，为现代数学提供了基础工具。从并集到交集，从差集到补集，从分配律到德摩根律，这一理论展示了数学的深刻统一性和代数结构。
