## 要点列表（初稿，用于分发到子文档）

- 与希尔伯特（形式主义）：
  - 目标差异：完备性/可判定性 vs 不完备性/不可判定性。
  - 技术对照：Hilbert 证明论 vs Gödel 算术化/对角化。
  - 影响：希尔伯特计划受挫；一致性需要更强系统。
- 与康托尔（连续统假设）：
  - 问题承接：CH/GCH 的一致性与独立性；可构造宇宙 L 的角色。
  - 方法差异：集合论基石与阶层分析 vs 自指与算术化。
  - 影响：连续统规模问题的独立性，模型论与内模型理论兴起。
- 与图灵（可计算性与停机）：
  - 对角线范式对照：不可证命题 ↔ 停机反例程序。
  - 不可判定性对应：形式系统不可完备 ↔ 停机不可判定。
  - 影响：递归论/复杂度理论与逻辑基础的互证。
- 与罗素（逻辑主义/类型论）：
  - 目标：逻辑化数学 vs 不完备性对逻辑主义的限制。
  - 技术：类型论消悖与层级化 vs 自指构造的不可避免性。
  - 影响：逻辑主义局限，促生更精细的类型与证明论研究。
