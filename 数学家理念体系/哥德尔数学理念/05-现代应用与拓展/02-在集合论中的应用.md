# 02-在集合论中的应用（骨架）

## 摘要（~250-300字）

哥德尔的可构造宇宙 L 与 CH/GCH 一致性证明奠定了现代集合论的基础：内模型理论、细结构、Core Model 等工具延续了可构造性思想；Forcing 与内模型的互补为独立性证明提供完整框架。本文总结可构造宇宙在现代集合论中的应用：大基数理论、组合原理（平方/钻石）、内模型与 Forcing 的协同，以及 CH/GCH 在现代集合论研究中的持续影响。

## 提纲（6-8 bullet）

- 内模型理论：L、L[U]、Core Model K 的构造与应用。
- 细结构（fine structure）：Jensen 细结构理论、标准参数、可定义性工具。
- 组合原理：平方原则、钻石原则在 V=L 中的证明；与 Forcing 的对照。
- 大基数接口：可构造性与大基数的张力；内模型为大基数提供分层结构。
- Forcing 与内模型互补：独立性上界与一致性下界的双重结构。
- 现代发展：Woodin 基数、强反射、公理选择与 CH 的关系（占位）。

- Jensen, R. Fine Structure 论文。
- Jech, T. *Set Theory*（内模型、Forcing 章节）。
- SEP: "Constructibility"; "Continuum Hypothesis".
