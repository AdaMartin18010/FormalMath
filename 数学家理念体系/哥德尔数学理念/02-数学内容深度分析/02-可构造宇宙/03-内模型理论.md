# 03-内模型理论（骨架）

## 摘要（~300字）

内模型理论研究满足 ZF/ZFC 的“内部”模型，如 L、L[U]、Core Model K，用以在不依赖 Forcing 的情况下分析大基数与组合性质。它为强不可达、可测、Woodin 等基数提供分层的“可构造”近似，并成为相对一致性与细结构（fine structure）研究的主线工具。本节概述内模型的定义、细结构框架、Core Model 的构造思想，以及其在证明“没有大基数时的最大可构造模型”方面的作用，并简述与 Forcing 方法的互补关系。

## 提纲（6-8 bullet）

- 内模型概念：包含所有序数的传递类，满足 ZF/ZFC。
- 细结构（fine structure）：层级划分、标准参数与可定义性工具。
- Core Model K：在无强大基数假设下的“最大”内模型思路。
- L[U] 等示例：可测基数背景下的内模型构造（占位）。
- 应用：相对一致性、反映原理、平方/钻石等组合原理的分析。
- 与 Forcing 的互补：内模型提供下界结构，Forcing 提供扩张与独立性。
