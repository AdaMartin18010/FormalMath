# 01-可构造宇宙的定义（骨架）

## 摘要（~300字）

可构造宇宙 L 由哥德尔定义，通过层级式的可定义元素累积来构建集合论模型。L_0=∅，L_{α+1}=Def(L_α)，极限层 L_λ=⋃_{β<λ}L_β。L 满足 ZF，且在 L 中选择公理与广义连续统成立。此节给出 L 的递归定义、关键引理（反射、最小性）、以及 L 与 V 的关系，指出 L 为“最小”的内模型，便于相对一致性证明。

## 提纲（6-8 bullet）

- 分层定义：L_0=∅；L_{α+1}=Def(L_α)；L_λ=⋃_{β<λ}L_β。
- 可定义性与编码：一阶公式、参数、Def 运算的闭包。
- L 的基本性质：传递性、内模型、包含所有序数。
- L ⊨ ZF 的关键思路：分层证明 Replacement、Power Set 的处理。
- L 与 V 的关系：L⊆V；若 V=L 则 CH/GCH/AC 成立。
- 用途：为 CH/AC 等提供相对一致性平台；与后续 Forcing/内模型理论的对照。
