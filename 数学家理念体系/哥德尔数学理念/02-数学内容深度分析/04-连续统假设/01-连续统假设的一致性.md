# 01-连续统假设的一致性（骨架）

## 摘要（~300字）

连续统假设（CH）断言 2^ℵ0 = ℵ1。哥德尔在可构造宇宙 L 中证明了 CH/GCH 与 ZF 的相对一致性：若 ZF 一致，则 ZF+GCH 一致（下界）；Cohen 通过 Forcing 证明 CH 独立于 ZFC（上界）。本节侧重 Gödel 路线的一致性证明框架：L 的构造、GCH 在 L 中成立的关键步骤、与 Cohen 独立性结果的互补关系，并说明 CH 在集合论发展中的定位（如对基数算术、实数集结构的影响）。

## 提纲（6-8 bullet）

- CH/GCH 陈述与基数背景：2^ℵ0，层级 ℵ1。
- Gödel 路线：构造 L，证明 L ⊨ GCH+AC，给出 Con(ZF) ⇒ Con(ZF+GCH)。
- 关键技术：分层定义、Power Set 控制、可定义性与计数论证。
- 对照 Cohen：Forcing 证明 CH 独立；上界/下界结构。
- 影响：基数算术、实数集结构、组合原理的确定性。
- 后续：内模型与 Forcing 的协同；大型基数背景下的 CH 讨论（占位）。
