# 02-广义连续统假设（骨架）

## 摘要（~300字）

广义连续统假设（GCH）断言对所有无限基数 κ，有 2^κ = κ^+。在可构造宇宙 L 中，GCH 与 AC 成立，从而 ZF 一致 ⇒ ZF+GCH+AC 一致。Cohen Forcing 与后续 Easton 定理展示了在 ZFC 下可灵活控制 2^κ，GCH 也因此被证明可独立。本节总结 GCH 的表述、Gödel 路线的一致性证明框架、与 Easton 结果的对照，并讨论 GCH 对基数算术与结构理论的影响。

## 提纲（6-8 bullet）

- GCH 定义：任意无限 κ，2^κ = κ^+。
- Gödel 路线：在 L 中证明 GCH，给出 Con(ZF) ⇒ Con(ZF+GCH)。
- 关键技术：L 的分层、可定义性与计数论证、Power Set 约束。
- Forcing/Easton：展示 2^κ 可在范围内自由指定（受约束），证明 GCH 独立。
- 影响：基数算术、凝聚与平方/钻石等组合原理的形态。
- 后续：大基数与 GCH 的相容性、内模型与 Forcing 的组合使用（占位）。
