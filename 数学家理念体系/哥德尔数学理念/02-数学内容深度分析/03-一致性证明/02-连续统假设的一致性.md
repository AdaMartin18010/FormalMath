# 02-连续统假设的一致性（骨架）

## 摘要（~300字）

哥德尔利用可构造宇宙 L 证明了 CH/GCH 与 ZF 的相对一致性：若 ZF 一致，则 ZF+GCH 也一致。核心是展示 L 是 ZF 的内模型，且在 L 中成立 GCH 与 AC。随后 Cohen 的 Forcing 证明 CH 可独立于 ZFC，形成“可构造宇宙给出下界、Forcing 给出上界”的经典结构。此节聚焦 Gödel 方向的一致性证明框架、关键引理、以及与 Cohen 独立性结果的并列呈现。

## 提纲（6-8 bullet）

- 目标：Con(ZF) ⇒ Con(ZF+GCH+AC)。
- 方法：构造 L，证明 L ⊨ ZF+GCH+AC。
- 关键性质：分层构造、Power Set 处理、GCH 在 L 中的证明思路。
- 意义：为 CH/GCH 提供相对一致性下界；奠定独立性框架的半壁。
- 对照：Cohen Forcing 证明 CH 独立；上界与下界互补。
- 影响：促进内模型理论与 Forcing 的发展。
