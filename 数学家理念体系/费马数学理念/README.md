## 📚 核心内容方向

### 1. 数论

- **费马小定理**：组合证明、欧拉归纳法证明、群论证明、RSA与素性测试应用
- **费马大定理**：$n=4$ 无限递降法、欧拉 $n=3$、谷山-志村猜想、怀尔斯证明
- **特殊数研究**：费马素数、完全数、亲和数、两平方和定理、四平方和定理

### 2. 解析几何

- **《平面与立体轨迹引论》**：用代数方程描述几何轨迹的系统方法
- **费马 vs 笛卡尔**：独立发现解析几何的历史对比
- **切线与求积**：adequality方法、微分学前夜、积分技巧萌芽

### 3. 概率论

- **点数问题**：费马的组合计数解法、帕斯卡的递推解法
- **数学期望的诞生**：公平分配问题到现代金融数学的演进
- **惠更斯的教科书**：《论赌博中的计算》与概率论的公开化

### 4. 方法论

- **无限递降法**：逻辑结构、在 $n=4$ 中的应用、与数学归纳法的关系、现代高度递降法
- **归纳-猜想-证明**：从具体例子到一般规律的发现模式
- **书信驱动研究**：梅森网络与17世纪学术共同体
