## CH的精确陈述与历史

### 1.1 连续统假设的正式陈述

**定义 1.1（连续统假设，CH）**：

$$2^{\aleph_0} = \aleph_1$$

等价表述：实数集 $\mathbb{R}$ 的基数等于第一个不可数基数。

**广义连续统假设（GCH）**：对所有无穷基数 $\kappa$，$2^{\kappa} = \kappa^+$。

### 1.2 历史脉络

- **1878**：康托尔提出连续统假设
- **1900**：希尔伯特将CH列为第一问题
- **1938**：哥德尔证明CH与ZFC相容
- **1963**：Cohen证明CH独立于ZFC
- **当代**：Woodin计划探索CH的"决定性"真值

> **康托尔的困境**：康托尔终生试图证明CH，但未能成功。他写道："数学的本质在于它的自由"。

## 哥德尔的一致性证明

### 2.1 构造性公理 V = L

**定义 2.1（可定义幂集）**：
$$\text{Def}(A) = \{X \subseteq A : X \text{ 在结构 } (A, \in) \text{ 中可定义}\}$$

**定义 2.2（可构造层级）**：

- $L_0 = \emptyset$
- $L_{\alpha+1} = \text{Def}(L_\alpha)$
- $L_\lambda = \bigcup_{\alpha < \lambda} L_\alpha$（$\lambda$ 极限）

可构造宇宙 $L = \bigcup_{\alpha \in \text{Ord}} L_\alpha$。

### 2.2 哥德尔的关键定理

**定理 2.1（Gödel, 1938）**：
$$L \models \text{ZFC} + \text{V} = \text{L} + \text{GCH}$$

*证明思路*：

1. **$L$ 是ZF的模型**：通过对层级归纳，验证ZF公理在 $L$ 中成立。

2. **$L \models \text{V} = \text{L}$**：由定义，每个 $x \in L$ 都在某 $L_\alpha$ 中，故 $L$ "知道"自己是可构造的。

3. **$L \models \text{GCH}$**（关键步骤）：
   - 对任何无穷基数 $\kappa$，$|L_\kappa| = \kappa$
   - 若 $X \subseteq \kappa$ 在 $L$ 中，则 $X \in L_{\kappa^+}$
   - 因此 $\mathcal{P}(\kappa) \cap L \subseteq L_{\kappa^+}$
   - 所以 $L \models 2^\kappa \leq \kappa^+$

### 2.3 一致性结果的推论

**推论 2.2**：如果ZF是一致的，则ZFC + CH也是一致的。

$$\text{Con}(\text{ZF}) \Rightarrow \text{Con}(\text{ZFC} + \text{CH})$$

*证明*：$L$ 是ZF的内模型，$L \models \text{ZFC} + \text{CH}$。如果ZFC + CH推出矛盾，则ZF也推出矛盾。$\square$

## Cohen的独立性证明

### 3.1 Cohen力迫的构造

**设定**：设 $M \models \text{ZFC} + \text{CH}$ 是可数传递模型。

**定义 3.1（Cohen偏序集）**：
$$\mathbb{C}_{\aleph_2} = \{p : p \text{ 是有限函数}, \, \text{dom}(p) \subseteq \omega_2^M \times \omega, \, \text{ran}(p) \subseteq 2\}$$

序：$p \leq q \Leftrightarrow p \supseteq q$。

### 3.2 构造泛型实数族

**定义 3.2（泛型实数）**：设 $G$ 是 $M$-泛型滤子。对每个 $\alpha < \omega_2^M$，定义：
$$c_\alpha(n) = \begin{cases} 0 & \text{如果 } (\exists p \in G) \, p(\alpha, n) = 0 \\ 1 & \text{如果 } (\exists p \in G) \, p(\alpha, n) = 1 \end{cases}$$

由 $G$ 的泛型性，每个 $c_\alpha : \omega \to 2$ 是良定义的。

### 3.3 关键引理

**引理 3.1**：对 $\alpha \neq \beta$，$c_\alpha \neq c_\beta$。

*证明*：集合 $D_{\alpha,\beta} = \{p : \exists n, \, p(\alpha, n) \neq p(\beta, n)\}$ 是稠密的。$\square$

**引理 3.2**：$\mathbb{C}_{\aleph_2}$ 满足可数链条件（ccc）。

*证明*：由$\Delta$-系统引理，任何不可数函数族都有相容的成员。$\square$

### 3.4 Cohen的独立性定理

**定理 3.3（Cohen, 1963）**：设 $G$ 是 $M$-泛型滤子，则 $M[G] \models \text{ZFC} + \neg\text{CH}$。

*证明*：

1. 由引理3.1，$M[G]$ 中有 $\omega_2^M$ 个不同的实数：$\{c_\alpha : \alpha < \omega_2^M\}$

2. 由引理3.2和ccc的基数保持性质：
   - $\omega_2^M = \omega_2^{M[G]}$
   - $\aleph_1^M = \aleph_1^{M[G]}$

3. 因此：
   $$M[G] \models 2^{\aleph_0} \geq \aleph_2 > \aleph_1$$

即 $M[G] \models \neg\text{CH}$。$\square$

**推论 3.4**：如果ZFC是一致的，则ZFC + ¬CH也是一致的。

## 独立性的技术细节

### 4.1 极小性引理

**引理 4.1**：$M[G]$ 是包含 $M \cup \{G\}$ 的最小ZFC传递模型。

*证明*：任何包含 $M$ 和 $G$ 的传递模型必须包含所有 $\text{val}_G(\tau)$。$\square$

### 4.2 力迫与真值

**定理 4.2**：在Cohen力迫中，$M[G]$ 中关于 $L_\alpha$ 的陈述的真值由 $M$ 中可数信息决定。

这意味着虽然 $M[G]$ 比 $M$ "更大"，但某些性质仍可由 $M$ 的局部信息控制。

### 4.3 打破CH的多种方式

Cohen力迫只是打破CH的一种方式。其他力迫构造包括：

1. **随机力迫（Random forcing）**：添加随机实数
2. **Hechler力迫**：添加支配所有 $M$ 中函数的实数
3. **迭代力迫**：多次扩展以获得更强的基数性质

## 对数学基础的影响

### 5.1 元数学意义

CH的独立性结果彻底改变了数学基础的研究方向：

1. **ZFC的局限性**：存在"自然"的数学命题（CH）独立于ZFC
2. **新公理的探索**：寻找判定CH的新公理系统
3. **多元宇宙观点**：集合论的多元宇宙（multiverse）观点

### 5.2 与希尔伯特第一问题的关系

希尔伯特第一问题询问：
> "康托尔提出的连续统基数问题，即要求证明 $2^{\aleph_0} = \aleph_1$"

Cohen的结果表明：**问题本身有问题**——CH不能在ZFC中被证明。

### 5.3 哲学反思

**形式主义观点**：CH的真值是相对于公理系统的

**柏拉图主义观点**：CH有客观真值，但ZFC不足以发现它

**多元宇宙观点**：存在多个集合论宇宙，在其中CH的真值不同

## 现代集合论视角

### 6.1 Woodin计划

Hugh Woodin提出了一个雄心勃勃的计划：寻找"终极"集合论公理。

**核心思想**：

- 内模型计划：构造类似于 $L$ 但能容纳大基数的内模型
- 如果找到"终极 $L$"，则可以决定CH的真值
- Woodin的猜测：终极 $L$ 将蕴含CH

### 6.2 决定性公理与CH

**定义（决定性公理，AD）**：每个实数子集 $A \subseteq \mathbb{R}$ 都是决定的，即博弈 $G(A)$ 有一方有必胜策略。

**定理**：$\text{ZF} + \text{AD}$ 蕴含所有实数子集都是可测的、具有Baire性质、完美集性质。

**与CH的关系**：AD蕴含 $2^{\aleph_0}$ 是某些大基数，但AD与选择公理矛盾。

### 6.3 当代研究方向

1. **力迫公理**：Martin公理、PFA（Proper Forcing Axiom）
2. **内模型理论**：Steel等人的工作
3. **描述集合论**：与力迫的深刻联系
4. **计算复杂性**：力迫与计算理论的交叉
