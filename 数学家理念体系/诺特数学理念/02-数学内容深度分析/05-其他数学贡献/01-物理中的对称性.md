## 二、诺特定理

### 2.1 定理陈述

**诺特定理（Noether's Theorem）**：

> 对于作用量在连续对称变换下不变的系统，存在相应的守恒量。

**经典表述**：

- **对称性** → **守恒量**
- **平移对称性** → **动量守恒**
- **旋转对称性** → **角动量守恒**
- **时间平移对称性** → **能量守恒**

### 2.2 数学表述

**作用量**：

$$S = \int L(q, \dot{q}, t) \, dt$$

其中 $L$ 是拉格朗日量。

**对称变换**：

$$q(t) \to q'(t') = q(t) + \epsilon \xi(q, t) + O(\epsilon^2)$$

$$t \to t' = t + \epsilon \tau(q, t) + O(\epsilon^2)$$

**不变性条件**：

作用量在对称变换下不变：

$$\delta S = 0$$

**守恒量**：

$$Q = \frac{\partial L}{\partial \dot{q}} \xi - L \tau + \text{边界项}$$

满足：

$$\frac{dQ}{dt} = 0$$

### 2.3 证明思路

**基本思路**：

1. 考虑作用量的变分
2. 使用Euler-Lagrange方程
3. 利用对称性条件
4. 得到守恒量

**关键步骤**：

$$\delta S = \int \left[\frac{\partial L}{\partial q} - \frac{d}{dt}\frac{\partial L}{\partial \dot{q}}\right] \delta q \, dt + \left[\frac{\partial L}{\partial \dot{q}} \delta q\right]_{t_1}^{t_2}$$

利用对称性和Euler-Lagrange方程，得到守恒量。

## 三、经典力学中的应用

### 3.1 平移对称性与动量守恒

**空间平移对称性**：

如果拉格朗日量在空间平移下不变：

$$L(x + a, \dot{x}, t) = L(x, \dot{x}, t)$$

**守恒量**：

$$p = \frac{\partial L}{\partial \dot{x}} = m\dot{x}$$

这是动量守恒。

**例子**：

- **自由粒子**：$L = \frac{1}{2}m\dot{x}^2$，动量守恒
- **其他例子**：其他具有平移对称性的系统

### 3.2 旋转对称性与角动量守恒

**旋转对称性**：

如果拉格朗日量在旋转下不变：

$$L(R(\theta) x, R(\theta) \dot{x}, t) = L(x, \dot{x}, t)$$

**守恒量**：

$$L = r \times p$$

这是角动量守恒。

**例子**：

- **中心力场**：角动量守恒
- **刚体**：角动量守恒
- **其他例子**：其他具有旋转对称性的系统

### 3.3 时间平移对称性与能量守恒

**时间平移对称性**：

如果拉格朗日量不显含时间：

$$L(q, \dot{q})$$

**守恒量**：

$$E = \frac{\partial L}{\partial \dot{q}} \dot{q} - L = T + V$$

这是能量守恒。

**例子**：

- **保守系统**：能量守恒
- **其他例子**：其他具有时间平移对称性的系统

## 四、场论中的应用

### 4.1 规范对称性

**规范对称性**：

场在规范变换下不变：

$$\phi(x) \to e^{i\alpha(x)} \phi(x)$$

**守恒量**：

$$j^\mu = \frac{\partial L}{\partial (\partial_\mu \phi)} \delta \phi$$

满足：

$$\partial_\mu j^\mu = 0$$

这是电荷守恒。

### 4.2 全局对称性

**全局对称性**：

场在全局变换下不变：

$$\phi(x) \to e^{i\alpha} \phi(x)$$

**守恒量**：

相应的守恒流。

### 4.3 局域对称性

**局域对称性**：

场在局域变换下不变：

$$\phi(x) \to e^{i\alpha(x)} \phi(x)$$

**规范场**：

需要引入规范场 $A_\mu$ 来保持局域对称性。

## 五、量子力学中的应用

### 5.1 量子对称性

**量子对称性**：

量子系统在对称变换下不变。

**对称算符**：

$$U^\dagger H U = H$$

其中 $U$ 是幺正算符。

### 5.2 对称性与守恒量

**守恒量**：

如果 $[H, Q] = 0$，则 $Q$ 是守恒量。

**例子**：

- **平移对称性** → **动量算符守恒**
- **旋转对称性** → **角动量算符守恒**
- **时间平移对称性** → **能量守恒**
