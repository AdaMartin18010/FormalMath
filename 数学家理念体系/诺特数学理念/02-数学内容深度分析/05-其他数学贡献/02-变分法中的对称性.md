## 二、变分法基础

### 2.1 作用量原理

**作用量**：

对于拉格朗日函数 $L(q, \dot{q}, t)$，作用量定义为：

$$S[q] = \int_{t_1}^{t_2} L(q(t), \dot{q}(t), t) \, dt$$

**最小作用量原理**：

物理系统的运动路径使作用量取极值：

$$\delta S = 0$$

### 2.2 Euler-Lagrange方程

**Euler-Lagrange方程**：

作用量取极值的必要条件是：

$$\frac{\partial L}{\partial q} - \frac{d}{dt}\frac{\partial L}{\partial \dot{q}} = 0$$

**推导**：

考虑变分 $\delta q$：

$$\delta S = \int_{t_1}^{t_2} \left[\frac{\partial L}{\partial q} - \frac{d}{dt}\frac{\partial L}{\partial \dot{q}}\right] \delta q \, dt + \left[\frac{\partial L}{\partial \dot{q}} \delta q\right]_{t_1}^{t_2}$$

由于 $\delta q(t_1) = \delta q(t_2) = 0$，得到Euler-Lagrange方程。

### 2.3 变分问题

**变分问题**：

求使泛函 $J[q] = \int L(q, \dot{q}, t) \, dt$ 取极值的函数 $q(t)$。

**方法**：

- 使用Euler-Lagrange方程
- 求解微分方程
- 确定边界条件

## 三、对称性与变分

### 3.1 对称变换

**对称变换**：

对于变分问题，对称变换是保持作用量不变的变换。

**一般形式**：

$$t \to t' = t + \epsilon \tau(q, t) + O(\epsilon^2)$$

$$q(t) \to q'(t') = q(t) + \epsilon \xi(q, t) + O(\epsilon^2)$$

**不变性条件**：

$$L(q', \dot{q}', t') \frac{dt'}{dt} = L(q, \dot{q}, t)$$

### 3.2 作用量的不变性

**作用量不变性**：

如果作用量在对称变换下不变：

$$S[q'] = S[q]$$

则称变换是变分对称性。

**条件**：

$$\delta S = 0$$

对任意满足边界条件的变分。

### 3.3 变分对称性

**变分对称性**：

变分对称性是保持作用量不变的变换。

**性质**：

- **群结构**：变分对称性构成群
- **守恒量**：每个对称性对应守恒量

## 四、诺特定理的变分表述

### 4.1 变分对称性定理

**诺特定理（变分表述）**：

如果作用量在单参数对称变换下不变，则存在守恒量。

**数学表述**：

对于对称变换：

$$q(t) \to q'(t') = q(t) + \epsilon \xi(q, t) + O(\epsilon^2)$$

$$t \to t' = t + \epsilon \tau(q, t) + O(\epsilon^2)$$

如果 $\delta S = 0$，则：

$$Q = \frac{\partial L}{\partial \dot{q}} \xi - L \tau + \text{边界项}$$

是守恒量：$\frac{dQ}{dt} = 0$。

### 4.2 守恒量的构造

**守恒量公式**：

$$Q = \sum_i \frac{\partial L}{\partial \dot{q}_i} \xi_i - L \tau + \text{边界项}$$

**推导**：

使用作用量的变分和Euler-Lagrange方程。

### 4.3 证明方法

**证明思路**：

1. 考虑作用量的变分
2. 使用对称性条件
3. 应用Euler-Lagrange方程
4. 得到守恒量

**关键步骤**：

$$\delta S = \int \left[\frac{\partial L}{\partial q} - \frac{d}{dt}\frac{\partial L}{\partial \dot{q}}\right] \delta q \, dt + \left[\frac{\partial L}{\partial \dot{q}} \delta q\right]_{t_1}^{t_2}$$

利用对称性和Euler-Lagrange方程，得到守恒量。

## 六、场论变分中的应用

### 6.1 场的作用量

**场的作用量**：

对于场 $\phi(x)$，作用量定义为：

$$S[\phi] = \int \mathcal{L}(\phi, \partial_\mu \phi, x) \, d^4x$$

其中 $\mathcal{L}$ 是拉格朗日密度。

### 6.2 场的对称性

**场的对称性**：

场在对称变换下不变：

$$\phi(x) \to \phi'(x') = \phi(x) + \epsilon \delta \phi(x) + O(\epsilon^2)$$

**不变性条件**：

$$\delta S = 0$$

### 6.3 守恒流

**守恒流**：

对于场的对称性，存在守恒流：

$$j^\mu = \frac{\partial \mathcal{L}}{\partial (\partial_\mu \phi)} \delta \phi - \mathcal{L} \delta x^\mu$$

满足：

$$\partial_\mu j^\mu = 0$$

**守恒量**：

$$Q = \int j^0 \, d^3x$$

是守恒量。
