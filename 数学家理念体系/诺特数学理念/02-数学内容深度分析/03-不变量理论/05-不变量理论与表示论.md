## 一、表示论与不变量理论的联系

### 1.1 两个理论的共同基础

**共同主题：对称性**：

```text
不变量理论：
- 研究：在群作用下保持不变的多项式
- 方法：代数方法
- 结果：不变量环

表示论：
- 研究：群的线性表示
- 方法：线性代数方法
- 结果：表示空间

联系：
- 都研究对称性
- 都涉及群作用
- 有深刻的对应关系
```

**历史发展**：

```text
19世纪：
- 不变量理论：Cayley, Sylvester, Gordan
- 表示论：Frobenius, Schur

20世纪初：
- 不变量理论：Hilbert, Noether
- 表示论：Weyl, Cartan

现代：
- 不变量理论：GIT, 模空间
- 表示论：几何表示论, 范畴化
```

---

### 1.2 基本对应关系

**对应**：

```text
群表示 ↔ 不变量：
- 群表示：ρ: G → GL(V)
- 不变量：f∈k[V]^G

表示空间 ↔ 不变量环：
- 表示空间：V
- 不变量环：k[V]^G

表示的分类 ↔ 不变量的分类：
- 不可约表示 ↔ 基本不变量
- 表示的分解 ↔ 不变量的生成
```

## 二、群表示与不变量

### 2.1 群表示的定义

**定义**：

```text
定义：
群G的表示是群同态
ρ: G → GL(V)
其中V是向量空间

等价表述：
G作用在V上
G × V → V, (g, v) ↦ ρ(g)v

例子：
- 置换表示：Sₙ → GL(ℝⁿ)
- 线性表示：GL(n) → GL(V)
```

**不变量与表示**：

```text
联系：
给定表示ρ: G → GL(V)
不变量环k[V]^G由所有G-不变多项式组成

即：
k[V]^G = {f∈k[V] | f(ρ(g)v) = f(v)对所有g∈G}
```

---

### 2.2 对称群与对称多项式

**经典例子：对称群Sₙ**：

```text
表示：
Sₙ作用在k[x₁,...,xₙ]上（置换变量）

不变量环：
k[x₁,...,xₙ]^Sₙ = 对称多项式环

生成元：
初等对称多项式
σ₁ = x₁ + ... + xₙ
σ₂ = ∑_{i<j} xᵢxⱼ
...
σₙ = x₁x₂...xₙ

意义：
- 所有对称多项式可表示为σ₁,...,σₙ的多项式
- 这是不变量理论的经典例子
```

**表示论视角**：

```text
Sₙ的表示：
- 标准表示：Sₙ → GL(ℝⁿ)
- 不可约表示：Young图

不变量：
- 对称多项式
- Schur多项式（与Young图对应）
```

---

### 2.3 一般线性群与不变量

**GL(n)的表示**：

```text
表示：
GL(n)作用在k[x₁,...,xₙ]上

不变量：
- 齐次不变量
- 与表示的权重对应

应用：
- 研究GL(n)的表示
- 分类不变量
```

## 三、Schur-Weyl对偶

### 3.1 Schur-Weyl对偶

**对偶关系**：

```text
定理（Schur-Weyl对偶）：
GL(n)和Sₖ在(ℂⁿ)⊗ᵏ上的作用对偶

具体：
1. GL(n)作用在(ℂⁿ)⊗ᵏ上
2. Sₖ也作用在(ℂⁿ)⊗ᵏ上（置换张量因子）
3. 两个作用对易
4. 分解：
   (ℂⁿ)⊗ᵏ = ⨁_λ S_λ(ℂⁿ) ⊗ V_λ
   其中S_λ是GL(n)的不可约表示，V_λ是Sₖ的不可约表示
```

**不变量理论的应用**：

```text
应用：
- 研究不变量环的结构
- 利用表示论工具
- 分类不变量
```

---

### 3.2 Schur多项式

**定义**：

```text
Schur多项式：
与Young图λ对应的对称多项式

性质：
1. 是Sₙ的不变量
2. 与不可约表示对应
3. 形成对称多项式的基

应用：
- 表示论
- 不变量理论
- 组合数学
```

## 四、不变量环的表示论结构

### 4.1 分次结构

**分次不变量环**：

```text
结构：
k[V]^G = ⨁_{d≥0} k[V]^G_d

其中k[V]^G_d是度数d的齐次不变量

表示论意义：
- 每个分次对应表示的权重
- 分次结构反映表示结构
```

**Hilbert级数**：

```text
定义：
H(t) = ∑_{d≥0} dim_k[V]^G_d · tᵈ

意义：
- 编码不变量环的信息
- 与表示论关联
- 计算工具
```

---

### 4.2 Molien公式

**Molien公式**：

```text
公式：
H(t) = (1/|G|)∑_{g∈G} 1/det(1 - t·ρ(g))

意义：
- 用群表示计算Hilbert级数
- 连接不变量与表示
- 计算工具
```

**应用**：

```text
1. 计算不变量环的维数
2. 研究不变量环的结构
3. 分类不变量
```
