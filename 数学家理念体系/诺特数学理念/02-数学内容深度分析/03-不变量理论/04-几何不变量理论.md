## 一、历史背景：从代数到几何

### 1.1 不变量理论的几何化

**问题**：

```text
经典不变量理论：
- 关注：不变量环的代数结构
- 方法：代数方法
- 结果：有限生成性

几何问题：
- 如何构造G-轨道空间的商空间？
- 如何参数化几何对象？
- 如何分类几何对象？
```

**Mumford的突破（1960s）**：

```text
几何不变量理论（GIT）：
- 将不变量理论几何化
- 构造商空间
- 解决分类问题

核心思想：
- 不变量环的谱 = 商空间
- 稳定点理论
- 模空间构造
```

---

### 1.2 从代数到几何的桥梁

**代数 ↔ 几何对应**：

```text
代数方面：
- 不变量环k[V]^G
- 有限生成k-代数
- 代数结构

几何方面：
- Spec(k[V]^G) = V/G
- 商空间
- 几何结构

GIT的作用：
- 连接代数与几何
- 构造几何对象
- 研究几何性质
```

## 二、GIT的基本思想

### 2.1 商空间问题

**问题**：

```text
给定：
- 代数簇X
- 群G作用在X上

问题：
如何构造商空间X/G？

困难：
- 轨道空间可能不是代数簇
- 需要"好的"商空间
```

**GIT的解决**：

```text
方法：
1. 利用不变量环k[X]^G
2. 构造Spec(k[X]^G)
3. 这是"好的"商空间

意义：
- 代数方法构造几何对象
- 保证几何性质
- 解决分类问题
```

---

### 2.2 线性化与射影嵌入

**线性化**：

```text
问题：
如何将G作用线性化？

方法：
1. 选择线性化L（线丛）
2. G作用在L上
3. 诱导作用在H⁰(X,L)上

意义：
- 将几何问题转化为线性问题
- 利用不变量理论
```

**射影嵌入**：

```text
方法：
1. 利用线性化L
2. 构造射影嵌入X ↪ ℙⁿ
3. G作用在ℙⁿ上

意义：
- 将问题转化为射影空间
- 利用射影几何工具
```

## 三、稳定点与半稳定点

### 3.1 稳定点的定义

**定义（Mumford）**：

```text
定义：
点x∈X是稳定的，若：
1. x的轨道G·x是闭的
2. G·x的维数等于G的维数
3. 存在不变量f使得f(x)≠0

记号：
X^s：稳定点集
X^ss：半稳定点集
```

**几何意义**：

```text
稳定点：
- 轨道"好"
- 在商空间中对应"好"的点
- 几何性质良好

半稳定点：
- 轨道可能不闭
- 但仍有"好的"商空间
```

---

### 3.2 半稳定点的定义

**定义**：

```text
定义：
点x∈X是半稳定的，若：
存在不变量f使得f(x)≠0

关系：
X^s ⊆ X^ss ⊆ X

意义：
- 半稳定点更多
- 但商空间可能更复杂
```

**稳定性条件**：

```text
稳定 ⟹ 半稳定

但反之不成立：
- 半稳定点可能不稳定
- 需要额外条件
```

---

### 3.3 稳定性判定

**Hilbert-Mumford准则**：

```text
准则：
点x是稳定的 ⟺
对所有单参数子群λ: 𝔾ₘ → G，
lim_{t→0} λ(t)·x不存在或不在X^ss中

意义：
- 用单参数子群判定稳定性
- 计算工具
- 几何意义
```

**应用**：

```text
1. 判定稳定性
2. 构造稳定点集
3. 研究商空间
```

## 四、商空间的构造

### 4.1 商空间的构造

**构造方法**：

```text
方法1：不变量环方法
X//G = Spec(k[X]^G)
- 代数构造
- 保证有限型
- 但可能不是"好的"商空间

方法2：GIT商
X//G = Proj(⨁_{n≥0} H⁰(X,Lⁿ)^G)
- 几何构造
- 利用线性化
- 更好的几何性质
```

**GIT商的性质**：

```text
性质：
1. X//G是有限型概形
2. 有自然映射X^ss → X//G
3. X^s的像在X//G中开且光滑

意义：
- 构造"好的"商空间
- 几何性质良好
- 可研究几何
```

---

### 4.2 商空间的分类

**分类**：

```text
1. 几何商：
   X/G（轨道空间）
   - 可能不是代数簇
   - 几何性质差

2. 范畴商：
   X//G（GIT商）
   - 是代数簇
   - 几何性质好
   - 但可能"丢失"信息

3. 栈商：
   [X/G]（商栈）
   - 保留所有信息
   - 但更复杂
```

## 五、模空间理论

### 5.1 模空间问题

**问题**：

```text
给定：
- 几何对象类（如曲线、向量丛等）

问题：
如何参数化这些对象？

目标：
构造模空间M
- M的点 ↔ 等价类
- M的几何 ↔ 分类的几何
```

**GIT的解决**：

```text
方法：
1. 构造参数空间X
2. 群G作用在X上（等价关系）
3. 构造GIT商X//G
4. 这是模空间M

例子：
- 曲线的模空间
- 向量丛的模空间
- 其他几何对象
```

---

### 5.2 模空间的例子

**曲线的模空间**：

```text
问题：
参数化亏格g的曲线

方法：
1. 构造Hilbert概形H
2. SL(n)作用在H上
3. 构造GIT商H//SL(n)
4. 这是模空间M_g

性质：
- M_g参数化曲线
- M_g的几何反映曲线的几何
```

**向量丛的模空间**：

```text
问题：
参数化向量丛

方法：
1. 构造Quot概形
2. 群作用
3. GIT商
4. 模空间

应用：
- 研究向量丛
- 分类问题
```
