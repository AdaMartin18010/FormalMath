## 二、有限群的不变量理论

### 2.1 群作用

**群作用的定义**：

```text
定义：
设G是群，V是向量空间
G作用在V上，若存在群同态
ρ: G → GL(V)

等价表述：
存在映射
G × V → V, (g, v) ↦ g·v
满足：
1. e·v = v（e是单位元）
2. (gh)·v = g·(h·v)
3. g·(v+w) = g·v + g·w（线性性）
```

**多项式环上的作用**：

```text
定义：
G作用在多项式环k[V]上：
(g·f)(v) = f(g⁻¹·v)

性质：
1. g·(f+h) = g·f + g·h
2. g·(fh) = (g·f)(g·h)
3. 保持度数
```

---

### 2.2 不变量环

**定义**：

```text
定义：
不变量环k[V]^G = {f∈k[V] | g·f = f对所有g∈G}

性质：
1. k[V]^G是k[V]的子环
2. k[V]^G是k-代数
3. k[V]^G是分次代数
```

**有限生成性**：

```text
定理（Hilbert-Noether）：
若G是有限群，则不变量环k[V]^G是有限生成k-代数

即：
存在f₁,...,fₘ∈k[V]^G使得
k[V]^G = k[f₁,...,fₘ]

意义：
- 解决了Gordan问题
- 建立了有限性理论
```

## 三、群作用与不变量环

### 3.1 Reynolds算子

**定义**：

```text
定义：
Reynolds算子R: k[V] → k[V]^G
R(f) = (1/|G|)∑_{g∈G} g·f

性质：
1. R是k[V]^G-模同态
2. R(f) = f对所有f∈k[V]^G
3. R是投影算子：R² = R
4. R保持度数
```

**应用**：

```text
应用：
1. 构造不变量
2. 证明有限生成性
3. 计算不变量环
```

---

### 3.2 不变量环的结构

**分次结构**：

```text
结构：
k[V]^G = ⨁_{d≥0} k[V]^G_d

其中k[V]^G_d是度数d的齐次不变量

性质：
1. 每个分次是有限维
2. 分次结构反映群的结构
3. 可计算Hilbert级数
```

**Hilbert级数**：

```text
定义：
H(t) = ∑_{d≥0} dim_k[V]^G_d · tᵈ

意义：
- 编码不变量环的信息
- 与群表示关联
- 计算工具
```

## 四、Noether上界定理

### 4.1 定理陈述

**Noether上界定理（1916）**：

```text
定理：
若G是有限群，|G| = n，则
不变量环k[V]^G可由度数≤n的齐次不变量生成

即：
k[V]^G = k[f₁,...,fₘ]
其中每个fᵢ是度数≤n的齐次不变量
```

**意义**：

```text
意义：
1. 给出上界
2. 限制搜索空间
3. 为计算提供基础
```

---

### 4.2 证明思路

**证明策略**：

```text
步骤1：
对任意齐次不变量f，度数d

步骤2：
利用Reynolds算子
R(f) = (1/|G|)∑_{g∈G} g·f

步骤3：
若d≤|G|，则f = R(f)（因为f是不变量）

步骤4：
若d>|G|，则利用Reynolds算子
证明f可表示为度数≤|G|的不变量的多项式

步骤5：
归纳法完成证明
```

**关键技巧**：

```text
1. Reynolds算子的投影性质
2. 度数的控制
3. 归纳法
```

## 五、群表示论的应用

### 5.1 群表示与不变量

**对应关系**：

```text
群表示 ↔ 不变量：
- 群表示：ρ: G → GL(V)
- 不变量：f∈k[V]^G

表示空间 ↔ 不变量环：
- 表示空间：V
- 不变量环：k[V]^G

表示的分类 ↔ 不变量的分类：
- 不可约表示 ↔ 基本不变量
```

---

### 5.2 Molien公式

**Molien公式**：

```text
公式：
H(t) = (1/|G|)∑_{g∈G} 1/det(1 - t·ρ(g))

意义：
- 用群表示计算Hilbert级数
- 连接不变量与表示
- 计算工具
```

**应用**：

```text
应用：
1. 计算不变量环的维数
2. 研究不变量环的结构
3. 分类不变量
```

## 六、在代数几何中的应用

### 6.1 商空间

**商空间的构造**：

```text
问题：
给定群G作用在代数簇X上
如何构造商空间X/G？

方法：
利用不变量环
X//G = Spec(k[X]^G)

性质：
- X//G是有限型概形
- 有自然映射X → X//G
```

---

### 6.2 模空间

**模空间的构造**：

```text
应用：
构造模空间

方法：
1. 构造参数空间X
2. 群G作用在X上
3. 构造GIT商X//G
4. 这是模空间M

例子：
- 曲线的模空间
- 向量丛的模空间
```
