## 一、历史背景：有限性条件的引入

### 1.1 希尔伯特的基底定理

**问题背景（1880s）**：

```text
不变量理论的核心问题：
给定多项式环k[x₁,...,xₙ]和群G，
不变量环k[x₁,...,xₙ]^G是否有有限生成基？

Gordan的结果（1868）：
对二元情况，证明有限基存在（构造性）

未解决问题：
对n≥3，是否仍有有限基？
```

**希尔伯特的突破（1888）**：

```text
基底定理（Hilbert Basis Theorem）：
若R是Noether环，则R[x]也是Noether环。
特别地，k[x₁,...,xₙ]的任意理想有有限生成基。

方法：
非构造性存在性证明

意义：
开创了抽象代数的非构造性方法
```

---

### 1.2 诺特的抽象化

**Noether环概念的引入（1921）**：

```text
诺特的工作：
1. 将Hilbert的条件抽象为"Noether环"
2. 建立Noether环的一般理论
3. 推广到模论

关键论文：
"Idealtheorie in Ringbereichen" (1921)

意义：
从具体条件到抽象公理
建立了有限性条件的统一理论
```

**从环到模**：

```text
诺特的推广（1920s）：
1. Noether环：理想的有限性
2. Noether模：子模的有限性
3. 统一框架：有限性条件

意义：
建立了模论的有限性理论
为同调代数奠定基础
```

## 二、Noether环：定义与等价条件

### 2.1 升链条件（ACC）

**定义**：

```text
定义（Noether, 1921）：
环R是Noether环，若满足升链条件（ACC）：

任意理想升链
I₁ ⊆ I₂ ⊆ I₃ ⊆ ...
必在有限步稳定，即存在N使得Iₙ = Iₙ₊₁对所有n≥N成立

等价表述：
不存在无限严格升链
I₁ ⊊ I₂ ⊊ I₃ ⊊ ...
```

**例子**：

```text
Noether环：
- 域k（只有两个理想）
- 主理想整环ℤ, k[x]
- 有限生成k-代数k[x₁,...,xₙ]/I

非Noether环：
- 无穷变元环k[x₁,x₂,...]
  理想(x₁) ⊊ (x₁,x₂) ⊊ ...（无限升链）
```

---

### 2.2 有限生成性

**等价条件**：

```text
定理：
环R是Noether环 ⟺
R的任意理想I都有有限生成基

即：存在a₁,...,aₘ∈I使得
I = (a₁,...,aₘ) = {r₁a₁ + ... + rₘaₘ | rᵢ∈R}

证明：
(⟹) 利用ACC，构造有限生成基
(⟸) 利用有限生成性，证明ACC
```

**意义**：

```text
有限生成性的重要性：
1. 可计算性：有限生成理想可计算
2. 可表示性：可用有限个元素表示
3. 可控制性：控制无限性

应用：
- 不变量理论：不变量环有限生成
- 代数几何：理想有限生成
- 计算代数：Gröbner基算法
```

---

### 2.3 极大条件

**等价条件**：

```text
定理：
环R是Noether环 ⟺
任意非空理想族有极大元（关于包含关系）

证明：
利用Zorn引理（选择公理）

意义：
提供了另一种刻画Noether性质的方式
```

## 三、Noether模：模论的有限性

### 3.1 模的定义

**模的公理化定义**：

```text
定义：
设R是环，M是Abel群，M是R-模，若存在标量乘法
· : R × M → M
满足：

(1) 分配律：
   r(m+n) = rm + rn
   (r+s)m = rm + sm

(2) 结合律：
   (rs)m = r(sm)

(3) 单位元：
   1·m = m（若R有单位元）

例子：
- 理想I是R-模
- 向量空间是域上的模
- Abel群是ℤ-模
```

---

### 3.2 Noether模

**定义**：

```text
定义：
R-模M是Noether模，若满足下列等价条件：

(1) 升链条件（ACC）：
   任意子模升链
   N₁ ⊆ N₂ ⊆ N₃ ⊆ ...
   必在有限步稳定

(2) 有限生成性：
   M的任意子模都有有限生成基

(3) 极大条件：
   任意非空子模族有极大元
```

**Noether模的例子**：

```text
1. 有限生成自由模：
   Rⁿ = R × ... × R（n次）
   是Noether模（若R是Noether环）

2. 有限生成模：
   若R是Noether环，M是有限生成R-模
   则M是Noether模

3. 理想：
   理想I是R-模
   I是Noether模 ⟺ R是Noether环
```

---

### 3.3 Noether模的性质

**基本性质**：

```text
1. 子模：
   若M是Noether模，N是子模
   则N和M/N都是Noether模

2. 短正合列：
   若0 → M' → M → M'' → 0是正合列
   则M是Noether模 ⟺ M'和M''都是Noether模

3. 有限生成：
   若R是Noether环，M是有限生成R-模
   则M是Noether模
```

## 四、Noether性质的基本定理

### 4.1 Hilbert基底定理

**定理陈述**：

```text
定理（Hilbert, 1888）：
若R是Noether环，则R[x]也是Noether环。

归纳推广：
若R是Noether环，则R[x₁,...,xₙ]也是Noether环。

应用：
k[x₁,...,xₙ]是Noether环（k是域）
```

**证明思路**：

```text
步骤1：反设
假设R[x]中存在理想I无有限生成基

步骤2：构造严格升链
I₀ ⊊ I₁ ⊊ I₂ ⊊ ...（无限长）

步骤3：提取首项系数
对每个Iₖ，选fₖ∈Iₖ\Iₖ₋₁
设首项系数为aₖ

步骤4：应用R的Noether性质
理想J = (a₀,a₁,...) ⊆ R有有限基
J = (a₀,...,aₘ)

步骤5：导出矛盾
利用有限基，构造g∈Iₘ₊₁但g∈Iₘ
矛盾！
```

---

### 4.2 Noether性质的保持

**基本定理**：

```text
1. 商环：
   若R是Noether环，I是理想
   则R/I也是Noether环

2. 局部化：
   若R是Noether环，S是乘闭子集
   则R_S也是Noether环

3. 有限生成代数：
   若R是Noether环，则R[x₁,...,xₙ]也是
   （Hilbert基底定理）

4. 有限生成模：
   若R是Noether环，M是有限生成R-模
   则M是Noether模
```

## 五、Hilbert基底定理的推广

### 5.1 多项式环的推广

**多变量情况**：

```text
定理：
若R是Noether环，则R[x₁,...,xₙ]也是Noether环

证明：
对n归纳，利用Hilbert基底定理

应用：
k[x₁,...,xₙ]是Noether环
（k是域）
```

**形式幂级数环**：

```text
定理：
若R是Noether环，则形式幂级数环R[[x]]也是Noether环

证明：
类似Hilbert基底定理

应用：
局部环的完备化
```

---

### 5.2 模的推广

**有限生成模的Noether性质**：

```text
定理：
若R是Noether环，M是有限生成R-模
则M是Noether模

证明：
1. 有限生成自由模Rⁿ是Noether模
2. M是Rⁿ的商模
3. 商模保持Noether性质

应用：
- 有限生成模的子模有限生成
- 有限生成模的商模有限生成
```

## 六、Noether性质的应用

### 6.1 不变量理论

**Hilbert-Noether定理**：

```text
定理：
若G是可约代数群，则不变量环k[V]^G是有限生成的

证明：
1. k[V]是Noether环（多项式环）
2. k[V]^G是k[V]的子环
3. 利用G的可约性，证明k[V]^G是Noether环
4. 因此k[V]^G有限生成

意义：
解决了Gordan问题
建立了不变量理论的有限性
```

---

### 6.2 代数几何

**理想与簇的对应**：

```text
应用：
1. 代数簇可分解为不可约分支（有限个）
   利用Noether环中理想的准素分解

2. 局部环是Noether环
   研究簇的局部性质

3. 上同调有限维
   利用Noether性质
```

## 七、Artin环：对偶的有限性

### 7.1 Artin环的定义

**降链条件（DCC）**：

```text
定义：
环R是Artin环，若满足降链条件（DCC）：

任意理想降链
I₁ ⊇ I₂ ⊇ I₃ ⊇ ...
必在有限步稳定

等价条件：
任意非空理想族有极小元
```

**Artin环的例子**：

```text
1. 有限环：
   只有有限个理想

2. 有限维k-代数：
   k是域，R是有限维k-向量空间

3. 局部Artin环：
   局部环且Artin环
```

---

### 7.2 Noether vs Artin

**关系**：

```text
定理：
Artin环 ⟹ Noether环
（反之不成立）

例子：
- 有限环既是Artin又是Noether
- ℤ是Noether但不是Artin
  （理想降链(2) ⊇ (4) ⊇ (8) ⊇ ...无限）

特殊情况：
在局部环中，Artin ⟺ Noether（且维数为0）
```

## 八、在代数几何中的应用

### 8.1 Noether概形

**定义**：

```text
定义：
概形X是Noether概形，若：
1. X的拓扑空间是Noether拓扑空间
2. 对每个开集U，O_X(U)是Noether环

性质：
- Noether概形可被有限个仿射概形覆盖
- Noether概形的上同调有限维
```

**应用**：

```text
1. 有限性：
   Noether概形具有良好的有限性

2. 上同调：
   上同调群有限维

3. 分类：
   利用有限性进行分类
```

---

### 8.2 有限型概形

**定义**：

```text
定义：
概形态射f: X → S是有限型的，若：
1. 局部有限型：对每个x∈X，存在仿射邻域
2. 拟紧：f是拟紧的

意义：
有限型概形是"有限生成"的概形
```

**Noether性质**：

```text
定理：
若S是Noether概形，f: X → S是有限型
则X是Noether概形

应用：
- 构造Noether概形
- 研究有限型概形的性质
```
