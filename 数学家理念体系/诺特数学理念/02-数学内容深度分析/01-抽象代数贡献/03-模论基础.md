## 一、历史背景：从向量空间到模

### 1.1 向量空间理论

**向量空间的公理化（1900s）**：

```text
定义：
设k是域，V是k-向量空间，若：
1. (V, +)是Abel群
2. 有标量乘法k × V → V
3. 满足分配律、结合律等

性质：
- 有基（有限维情况）
- 维数唯一
- 结构简单
```

**局限**：

```text
问题：
向量空间要求标量来自域
但许多数学结构需要更一般的"系数"

例子：
- 理想：需要环作为系数
- Abel群：需要ℤ作为系数
- 表示：需要群环作为系数
```

---

### 1.2 诺特的模论

**模论的建立（1920s）**：

```text
诺特的工作：
1. 将向量空间推广到模
2. 建立模的一般理论
3. 引入Noether模概念

关键论文：
"Idealtheorie in Ringbereichen" (1921)
"Abstrakter Aufbau der Idealtheorie" (1926)

意义：
统一了不同领域的代数结构
为同调代数奠定基础
```

**从向量空间到模**：

```text
向量空间（域上） → 模（环上）

推广：
- 标量从域推广到环
- 基的存在性不再保证
- 结构更复杂但更一般
```

## 二、模的公理化定义

### 2.1 左R-模

**定义**：

```text
定义：
设R是环（有单位元），M是Abel群。
M是左R-模，若存在标量乘法
· : R × M → M
满足：

(1) 分配律：
   r·(m + n) = r·m + r·n
   (r + s)·m = r·m + s·m

(2) 结合律：
   (rs)·m = r·(s·m)

(3) 单位元：
   1·m = m

对所有r,s∈R, m,n∈M成立
```

**右R-模**：

```text
定义类似，但标量乘法在右边：
m·r

若R是交换环，左模和右模等价
```

---

### 2.2 模的基本性质

**基本性质**：

```text
1. 0·m = 0（零元作用）
2. r·0 = 0（作用零元）
3. (-r)·m = -(r·m)（负元作用）
4. r·(-m) = -(r·m)（作用负元）

证明：
利用分配律和单位元性质
```

## 三、模的例子与分类

### 3.1 基本例子

**向量空间**：

```text
若R是域k，则R-模就是k-向量空间

例子：
- ℝⁿ是ℝ-向量空间
- ℂⁿ是ℂ-向量空间

性质：
- 有基
- 维数唯一
- 结构简单
```

**Abel群**：

```text
若R = ℤ，则ℤ-模就是Abel群

对应：
- ℤ-模 ⟷ Abel群
- 子模 ⟷ 子群
- 模同态 ⟷ 群同态

意义：
统一了Abel群理论
```

**理想**：

```text
若I是环R的理想，则I是R-模

性质：
- I是R的子模
- 商模R/I是R-模

意义：
理想理论是模论的特例
```

---

### 3.2 特殊模类

**自由模**：

```text
定义：
R-模M是自由的，若存在基B⊆M使得
每个m∈M可唯一表示为B中元素的线性组合

例子：
- Rⁿ = R × ... × R（n次）是自由模
- 基：{(1,0,...,0), (0,1,...,0), ..., (0,...,0,1)}

性质：
- 自由模的基基数唯一（若R是交换环）
- 自由模是投射的
```

**有限生成模**：

```text
定义：
R-模M是有限生成的，若存在m₁,...,mₙ∈M使得
M = Rm₁ + ... + Rmₙ

记号：
M = ⟨m₁,...,mₙ⟩

性质：
- 有限生成模的子模不一定有限生成
- 若R是Noether环，则有限生成模的子模有限生成
```

## 四、模的结构理论

### 4.1 子模与商模

**子模**：

```text
定义：
N ⊆ M是M的子模，若：
1. N是M的子群（加法）
2. 对任意r∈R, n∈N，有r·n∈N

例子：
- {0}是子模（零子模）
- M是子模（平凡子模）
- 理想I是R的子模
```

**商模**：

```text
定义：
设N是M的子模，商模M/N定义为：
- 元素：陪集m+N = {m+n | n∈N}
- 加法：(m+N) + (m'+N) = (m+m') + N
- 标量乘法：r·(m+N) = (r·m) + N

性质：
- M/N是R-模
- 自然投影π: M → M/N是模同态
- ker(π) = N
```

---

### 4.2 直和与直积

**直和**：

```text
定义：
设M₁,...,Mₙ是R-模，直和
M₁ ⊕ ... ⊕ Mₙ = {(m₁,...,mₙ) | mᵢ∈Mᵢ}

运算：
- 加法：分量相加
- 标量乘法：分量相乘

性质：
- 直和是模
- 每个Mᵢ是直和的子模
- 直和是直积的子模（有限情况）
```

**直积**：

```text
定义：
设{Mᵢ}ᵢ∈I是R-模族，直积
∏ᵢ Mᵢ = {(mᵢ)ᵢ∈I | mᵢ∈Mᵢ}

性质：
- 直积是模
- 直和是直积的子模
- 若I有限，则直和=直积
```

## 五、Noether模：有限性条件

### 5.1 Noether模的定义

**定义（Noether, 1920s）**：

```text
定义：
R-模M是Noether模，若满足下列等价条件：

(1) 升链条件（ACC）：
   任意子模升链
   N₁ ⊆ N₂ ⊆ N₃ ⊆ ...
   必在有限步稳定

(2) 有限生成性：
   M的任意子模都有有限生成基

(3) 极大条件：
   任意非空子模族有极大元
```

**等价性证明**：

```text
(1) ⟹ (2)：
反设子模N无有限基
构造严格升链N₀ ⊊ N₁ ⊊ ...（矛盾）

(2) ⟹ (3)：
利用Zorn引理

(3) ⟹ (1)：
直接应用极大条件
```

---

### 5.2 Noether模的性质

**基本性质**：

```text
1. 子模：
   若M是Noether模，N是子模
   则N和M/N都是Noether模

2. 短正合列：
   若0 → M' → M → M'' → 0是正合列
   则M是Noether模 ⟺ M'和M''都是Noether模

3. 有限生成：
   若R是Noether环，M是有限生成R-模
   则M是Noether模
```

**Noether环与Noether模**：

```text
定理：
环R是Noether环 ⟺
R作为R-模是Noether模

证明：
R的理想 ⟷ R的子模
```

## 六、自由模与投射模

### 6.1 自由模

**定义**：

```text
定义：
R-模M是自由的，若存在基B⊆M使得
每个m∈M可唯一表示为
m = ∑_{b∈B} r_b·b（有限和）

性质：
1. 基的基数唯一（若R是交换环）
2. 自由模的秩：rank(M) = |B|
3. 自由模是投射的
```

**自由模的例子**：

```text
1. Rⁿ = R × ... × R（n次）
   基：标准基向量

2. 多项式环R[x]
   基：{1, x, x², ...}（无限基）

3. 若R是域，则自由模就是向量空间
```

---

### 6.2 投射模

**定义**：

```text
定义：
R-模P是投射的，若对任意满同态
f: M → N和同态g: P → N，
存在同态h: P → M使得f ∘ h = g

图示：
    P
    │ g
    ↓
M ──f──→ N (f满)
    ↑
    │ h
    └───
```

**投射模的性质**：

```text
等价条件：
1. P是投射的
2. P是某个自由模的直和项
3. 函子Hom(P, -)是正合的

例子：
- 自由模是投射的
- 主理想整环上的投射模是自由的
- 一般环上投射模不一定自由
```

## 七、模的同态与同构定理

### 7.1 模同态

**定义**：

```text
定义：
设M, N是R-模，映射φ: M → N是模同态，若：
1. φ(m + n) = φ(m) + φ(n)（保持加法）
2. φ(r·m) = r·φ(m)（保持标量乘法）

核：
ker(φ) = {m∈M | φ(m) = 0}
是M的子模

像：
im(φ) = {φ(m) | m∈M}
是N的子模
```

**同态基本定理**：

```text
定理：
设φ: M → N是模同态，则：
M/ker(φ) ≅ im(φ)

证明：
构造同构ψ: M/ker(φ) → im(φ)
ψ(m + ker(φ)) = φ(m)
```

---

### 7.2 同构定理

**第一同构定理**：

```text
定理：
设N是M的子模，H是M的子模且N ⊆ H，则：
(M/N)/(H/N) ≅ M/H

意义：
商模的商模同构于直接商模
```

**第二同构定理**：

```text
定理：
设N, H是M的子模，则：
(N + H)/H ≅ N/(N ∩ H)

意义：
和的商模同构于交的商模
```

## 八、在代数几何中的应用

### 8.1 拟凝聚层

**定义**：

```text
定义：
概形X上的拟凝聚层F是
局部同构于某个模的层

对应：
拟凝聚层 ⟷ 模

意义：
将模论应用到代数几何
```

**凝聚层**：

```text
定义：
拟凝聚层F是凝聚的，若
局部地，F对应有限生成模

性质：
- 凝聚层在Noether概形上良好
- 上同调有限维
```

---

### 8.2 上同调理论

**层上同调**：

```text
定义：
概形X上层的上同调群Hᵢ(X, F)

性质：
- 若X是Noether概形，F是凝聚层
- 则Hᵢ(X, F)是有限维k-向量空间

应用：
- 研究概形的几何性质
- 计算不变量
```

**模论的建立**：
