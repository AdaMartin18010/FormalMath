## 二、域论基础：诺特的公理化方法

### 2.1 域的定义

**公理化定义**：

```text
定义：
域F是带有两种二元运算的集合：
- 加法：+ : F × F → F
- 乘法：· : F × F → F

满足以下公理：

(1) 加法公理：
- (F, +)是Abel群
- 零元：0

(2) 乘法公理：
- (F\{0}, ·)是Abel群
- 单位元：1

(3) 分配律：
- a(b+c) = ab + ac

意义：
- 公理化定义
- 统一不同域
- 抽象框架
```

**例子**：

```text
域的例子：
- ℚ：有理数域
- ℝ：实数域
- ℂ：复数域
- F_p：有限域（p是素数）
- k(x)：有理函数域
```

---

### 2.2 域扩张

**域扩张的定义**：

```text
定义：
设K是域F的扩域（K ⊇ F）
则K/F是域扩张

记号：
[K:F] = dim_F(K)（F-向量空间的维数）

分类：
- 有限扩张：[K:F] < ∞
- 无限扩张：[K:F] = ∞
```

**扩张的性质**：

```text
性质：
1. [K:F] = [K:E][E:F]（塔律）
2. 有限扩张 ⟺ 代数扩张（对有限域）
3. 单扩张：K = F(α)

意义：
- 研究域扩张的结构
- 分类域扩张
```

## 三、Galois理论：代数化的表述

### 3.1 Galois扩张

**定义**：

```text
定义：
域扩张K/F是Galois扩张，若：
1. K/F是有限扩张
2. K/F是正规扩张
3. K/F是可分扩张

等价条件：
|Gal(K/F)| = [K:F]

其中Gal(K/F)是Galois群
```

**Galois群**：

```text
定义：
Gal(K/F) = {σ: K → K | σ是F-自同构}

性质：
1. Gal(K/F)是群（复合运算）
2. |Gal(K/F)| ≤ [K:F]
3. |Gal(K/F)| = [K:F] ⟺ K/F是Galois扩张
```

---

### 3.2 Galois对应

**基本定理**：

```text
定理（Galois对应）：
设K/F是Galois扩张，G = Gal(K/F)

则存在一一对应：
{中间域E | F ⊆ E ⊆ K} ↔ {子群H | H ≤ G}

对应关系：
E ↔ Gal(K/E)
H ↔ Fix(H) = {α∈K | σ(α)=α对所有σ∈H}

性质：
1. E₁ ⊆ E₂ ⟺ Gal(K/E₁) ≥ Gal(K/E₂)
2. H₁ ≤ H₂ ⟺ Fix(H₁) ⊇ Fix(H₂)
3. [E:F] = [G:Gal(K/E)]
```

**意义**：

```text
意义：
1. 域扩张问题 ⟺ 群论问题
2. 中间域 ⟺ 子群
3. 简化问题
```

## 四、诺特正规化定理

### 4.1 定理陈述

**诺特正规化定理（1926）**：

```text
定理：
设k是域，A是有限生成k-代数
则存在代数独立元y₁,...,yₐ使得
A是k[y₁,...,yₐ]的整扩张

即：
存在k-代数同态
k[y₁,...,yₐ] → A
使得A在k[y₁,...,yₐ]上整

几何意义：
存在有限到一的映射
Spec(A) → 𝔸ᵏ
```

**意义**：

```text
意义：
1. 将有限生成k-代数"参数化"
2. 建立与多项式环的联系
3. 为代数几何提供工具
```

## 五、域扩张与Galois群

### 5.1 有限Galois扩张

**性质**：

```text
定理：
设K/F是有限Galois扩张，G = Gal(K/F)

则：
1. K^G = F（固定域是F）
2. |G| = [K:F]
3. K是G的不动点域的Galois闭包

意义：
Galois扩张是"最大"的扩张
```

**例子**：

```text
例子：
ℚ(√2)/ℚ：
- Galois群：{id, σ}，其中σ(√2) = -√2
- |Gal| = 2 = [ℚ(√2):ℚ]
- 是Galois扩张

ℚ(∛2)/ℚ：
- [ℚ(∛2):ℚ] = 3
- 但|Gal| = 1（因为其他根不在ℚ(∛2)中）
- 不是Galois扩张
```

---

### 5.2 可解群与根式可解

**可解群**：

```text
定义：
群G是可解群，若存在子群列
G = G₀ ≥ G₁ ≥ ... ≥ Gₙ = {e}
使得Gᵢ/Gᵢ₊₁是Abel群

等价条件：
G的导群列在有限步终止于{e}
```

**Galois的定理**：

```text
定理（Galois）：
方程可用根式求解 ⟺
Galois群是可解群

意义：
- 连接方程理论与群论
- 解决可解性问题
```

## 六、在代数几何中的应用

### 6.1 函数域

**函数域**：

```text
定义：
设X是代数曲线
函数域k(X)是X的有理函数域

性质：
1. k(X)是k的有限生成扩域
2. 超越度 = 1
3. 对应曲线X

意义：
- 连接代数几何与域论
- 研究曲线的性质
```

---

### 6.2 Galois覆盖

**Galois覆盖**：

```text
定义：
曲线覆盖π: Y → X是Galois覆盖，若
k(Y)/k(X)是Galois扩张

Galois群：
Gal(Y/X) = Gal(k(Y)/k(X))

意义：
- 研究曲线的覆盖
- 分类问题
```

## 七、现代发展：从Noether到现代Galois理论

### 7.1 无限Galois理论

**无限Galois扩张**：

```text
定义：
域扩张K/F是无限Galois扩张，若
K = ∪ᵢ Kᵢ，其中Kᵢ/F是有限Galois扩张

Galois群：
Gal(K/F) = limᵢ Gal(Kᵢ/F)（投射极限）

性质：
- 拓扑群结构
- Krull拓扑
```

---

### 7.2 绝对Galois群

**绝对Galois群**：

```text
定义：
设F是域，Fˢ是F的可分闭包
绝对Galois群：G_F = Gal(Fˢ/F)

性质：
1. G_F是投射有限群
2. 编码F的算术性质
3. 数论中的重要对象
```

**应用**：

```text
应用：
1. 数论
2. 代数几何
3. 表示论
4. Langlands纲领
```
