## 一、历史背景：唯一分解的失效

### 1.1 唯一分解定理

**整数环中的唯一分解**：

```text
定理（算术基本定理）：
在ℤ中，每个大于1的整数可唯一分解为素数的乘积

例子：
12 = 2²·3
分解唯一（不计顺序）

意义：
唯一分解是数论的基础
```

**问题**：

```text
问题：
在代数整数环中，唯一分解是否仍成立？

答案：
不一定！
```

---

### 1.2 唯一分解的失效

**经典例子：ℤ[√-5]**：

```text
定义：
ℤ[√-5] = {a + b√-5 | a, b ∈ ℤ}

问题：
6 = 2·3 = (1+√-5)(1-√-5)

验证：
- 2, 3, 1+√-5, 1-√-5都是"不可约"的
- 但分解不唯一

原因：
缺少"理想数"的概念
```

**其他例子**：

```text
ℤ[√-6]：
10 = 2·5 = (2+√-6)(2-√-6)

ℤ[√-10]：
14 = 2·7 = (2+√-10)(2-√-10)
```

## 二、Dedekind的理想理论

### 2.1 Dedekind的洞察

**问题**：

```text
问题：
如何在代数整数环中恢复唯一分解？

Dedekind的洞察（1870s）：
引入"理想"概念，将数推广为理想
```

**理想的概念**：

```text
定义（Dedekind）：
设K是数域，O_K是K的代数整数环
理想I ⊆ O_K是O_K的子集，满足：
1. I是O_K的子群（加法）
2. 对任意a∈O_K, b∈I，有ab∈I

例子：
主理想：(α) = {αβ | β∈O_K}
```

---

### 2.2 理想的唯一分解

**Dedekind的定理**：

```text
定理（Dedekind, 1871）：
在代数整数环O_K中，每个非零理想可唯一分解为素理想的乘积

即：
I = P₁^e₁ · P₂^e₂ · ... · Pₖ^eₖ
其中Pᵢ是素理想，eᵢ是正整数

唯一性：
分解在理想意义下唯一（不计顺序）
```

**意义**：

```text
1. 恢复了唯一分解
2. 建立了理想理论
3. 为代数数论奠定基础
```

---

### 2.3 素理想

**素理想的定义**：

```text
定义：
理想P是素理想，若：
ab∈P ⟹ a∈P或b∈P

等价条件：
O_K/P是整环

例子：
在ℤ中，素理想对应素数p：(p)
在ℤ[√-5]中，有非主素理想
```

**素理想的例子**：

```text
在ℤ[√-5]中：
理想P = (2, 1+√-5)是素理想
但P不是主理想

意义：
素理想比素数更一般
```

## 三、诺特的抽象化：从具体到抽象

### 3.1 诺特的工作

**诺特的贡献（1920s）**：

```text
诺特的工作（1921-1926）：
1. 将Dedekind的理想理论抽象化
2. 从具体环（代数整数环）到抽象环
3. 建立理想理论的公理化框架

关键论文：
- "Idealtheorie in Ringbereichen" (1921)
- "Abstrakter Aufbau der Idealtheorie" (1926)
```

**抽象化的意义**：

```text
从具体到抽象：
Dedekind：具体环（代数整数环）
诺特：抽象环（满足公理的任意环）

意义：
1. 统一不同环的理想理论
2. 建立一般性理论
3. 为现代代数奠定基础
```

---

### 3.2 理想的公理化定义

**诺特的定义（1921）**：

```text
定义：
设R是环，I ⊆ R是R的理想，若：

(1) I是R的子群（加法）：
   - 0∈I
   - 若a,b∈I，则a+b∈I
   - 若a∈I，则-a∈I

(2) 吸收性：
   - 对任意r∈R, a∈I，有ra∈I
   - 对任意r∈R, a∈I，有ar∈I（若R非交换）

意义：
- 公理化定义
- 适用于任意环
- 统一不同环的理想理论
```

**与Dedekind的关系**：

```text
Dedekind的定义：
- 针对代数整数环
- 具体、计算导向

诺特的定义：
- 针对抽象环
- 公理化、结构导向

关系：
诺特的定义是Dedekind定义的抽象化
```

---

### 3.3 Noether环与有限性

**Noether环的定义**：

```text
定义（Noether, 1921）：
环R是Noether环，若满足下列等价条件之一：

(1) 升链条件（ACC）：
   任意理想升链I₁ ⊆ I₂ ⊆ ...必在有限步稳定

(2) 有限生成性：
   R的任意理想都有有限生成基

(3) 极大条件：
   任意非空理想族有极大元
```

**代数整数环的Noether性质**：

```text
定理：
代数整数环O_K是Noether环

证明思路：
1. O_K是有限生成ℤ-模
2. 利用有限生成性
3. 证明ACC

意义：
代数整数环满足Noether条件
理想理论可应用
```

## 四、理想理论在代数数论中的应用

### 4.1 理想与数

**主理想**：

```text
定义：
理想I是主理想，若存在α∈O_K使得I = (α)

记号：
(α) = {αβ | β∈O_K}

意义：
主理想对应"数"
非主理想对应"理想数"
```

**理想与数的对应**：

```text
对应：
数α ↔ 主理想(α)

性质：
1. (α)(β) = (αβ)
2. (α) + (β) = (α, β)
3. (α) ⊆ (β) ⟺ β|α

意义：
理想运算对应数运算
```

---

### 4.2 理想的范数

**理想的范数**：

```text
定义：
设I是O_K的理想
I的范数N(I) = |O_K/I|

性质：
1. N(IJ) = N(I)N(J)
2. N((α)) = |N_K/ℚ(α)|
3. N(I)是正整数

意义：
- 度量理想的大小
- 连接理想与数
```

**应用**：

```text
应用：
1. 计算理想类群
2. 研究类数
3. 数论问题
```

## 五、Dedekind环：代数整数环的抽象化

### 5.1 Dedekind环的定义

**定义**：

```text
定义：
整环R是Dedekind环，若满足：

(1) Noether性质：
   R是Noether环

(2) 整闭性：
   R在其分式域中整闭

(3) 维数条件：
   R的Krull维数为1

等价条件：
R的每个非零理想可唯一分解为素理想的乘积
```

**例子**：

```text
Dedekind环：
- 代数整数环O_K
- 主理想整环（PID）
- 一维Noether整闭整环

非Dedekind环：
- ℤ[x]（维数2）
- k[x,y]（维数2）
```

---

### 5.2 Dedekind环的性质

**唯一分解性质**：

```text
定理：
在Dedekind环R中，每个非零理想可唯一分解为素理想的乘积

即：
I = P₁^e₁ · P₂^e₂ · ... · Pₖ^eₖ

唯一性：
分解在理想意义下唯一
```

**局部化性质**：

```text
定理：
Dedekind环的局部化是离散赋值环（DVR）

意义：
- 局部结构简单
- 可研究局部性质
```

## 六、理想类群与类数

### 6.1 理想类群

**定义**：

```text
定义：
设K是数域，O_K是K的代数整数环

理想类群：
Cl(K) = 分式理想群 / 主分式理想群

即：
Cl(K) = I(K) / P(K)

其中：
- I(K)：K的分式理想群
- P(K)：K的主分式理想群
```

**性质**：

```text
性质：
1. Cl(K)是有限Abel群
2. Cl(K)的阶是类数h_K
3. Cl(K) = {1} ⟺ O_K是PID

意义：
- 度量O_K与PID的"距离"
- 重要的数论不变量
```

---

### 6.2 类数

**类数的定义**：

```text
定义：
类数h_K = |Cl(K)|

意义：
- h_K = 1 ⟺ O_K是PID
- h_K > 1 ⟺ O_K有非主理想
- h_K度量O_K的"复杂性"
```

**类数的计算**：

```text
计算：
1. 计算理想类群
2. 确定类数

例子：
- ℚ(√-1)：h = 1（PID）
- ℚ(√-5)：h = 2
- ℚ(√-23)：h = 3
```
