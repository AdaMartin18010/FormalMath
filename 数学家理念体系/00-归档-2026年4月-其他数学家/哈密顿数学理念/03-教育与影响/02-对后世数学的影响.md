# 哈密顿 - 对后世数学的影响

## 一、对经典力学的革新

威廉·罗恩·哈密顿（William Rowan Hamilton, 1805–1865）在1834–1835年发表的两篇论文《论动力学的一种普遍方法》将牛顿力学提升到一个全新的数学高度。哈密顿引入了**广义坐标** $q = (q_1, \ldots, q_n)$ 和**广义动量** $p = (p_1, \ldots, p_n)$，将拉格朗日方程转化为对称的**哈密顿正则方程**：
$$\dot{q}_i = \frac{\partial H}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial H}{\partial q_i}$$
其中 $H(q,p,t)$ 为**哈密顿量**，通常对应系统的总能量。这一形式不仅具有数学对称美，更揭示了力学的深层几何结构。

### 相空间与辛几何
哈密顿方程的自然舞台是**相空间**（phase space）$\mathbb{R}^{2n}$（或更一般的辛流形 $(M, \omega)$），装备有**辛形式**：
$$\omega = \sum_{i=1}^n dq_i \wedge dp_i$$
辛形式是一个闭的非退化2-形式（$d\omega = 0$，$\omega^n \neq 0$）。哈密顿方程可写为**辛梯度**形式：
$$\dot{z} = X_H(z), \quad \iota_{X_H} \omega = dH$$
其中 $z = (q,p)$。辛几何由此诞生，并在20世纪后半叶成为数学和物理的核心语言。

### 正则变换与生成函数
哈密顿引入了**正则变换**（canonical transformations）——保持辛形式不变的坐标变换。若变换 $(q,p) \mapsto (Q,P)$ 满足 $dQ \wedge dP = dq \wedge dp$，则新坐标下的方程仍保持哈密顿形式，且新哈密顿量 $K(Q,P,t)$ 可通过**生成函数**与原哈密顿量相联系。生成函数方法在摄动理论、天体力学和量子力学中具有不可替代的作用。

### 作用–角变量与可积系统
对于可积哈密顿系统（存在 $n$ 个独立的守恒量），哈密顿引入了**作用–角变量** $(J_i, \theta_i)$，使得：
$$H = H(J_1, \ldots, J_n), \quad \dot{\theta}_i = \omega_i(J) = \frac{\partial H}{\partial J_i}$$
运动限制在相空间的 $n$ 维环面 $\mathbb{T}^n$ 上。Kolmogorov–Arnold–Moser（KAM）理论证明了：对于轻微不可积扰动，大部分不变环面得以保持。KAM 理论为太阳系的长期稳定性提供了严格的数学基础，是哈密顿力学在20世纪最重要的发展之一。

## 二、对代数的贡献

### 四元数与非交换代数
1843年10月16日，哈密顿在都柏林的布鲁姆桥上刻下了**四元数**的基本公式：
$$i^2 = j^2 = k^2 = ijk = -1$$
四元数构成了实数域上的四维除法代数：
$$\mathbb{H} = \{ a + bi + cj + dk \mid a,b,c,d \in \mathbb{R} \}$$
其中乘法满足非交换律：$ij = k \neq ji = -k$。

四元数的发明打破了乘法交换律的统治地位，开启了**非交换代数**的新纪元。从威廉·金顿·克利福德（William Kingdon Clifford）的**克利福德代数**到 Sophus Lie 的**李代数**，从矩阵代数到量子群，现代代数学在很大程度上建立在哈密顿突破传统观念的基础之上。

四元数在计算机图形学中具有实际应用：三维旋转可通过单位四元数 $q = \cos(\theta/2) + \mathbf{u}\sin(\theta/2)$ 实现，避免了欧拉角的**万向锁**（gimbal lock）问题。对于向量 $\mathbf{v}$，旋转后的向量为 $q\mathbf{v}q^{-1}$。

### 复数与向量分析的萌芽
哈密顿在1833年将复数 $a + bi$ 重新解释为实数对 $(a,b)$，并定义了运算规则。这一观点为后来的**向量分析**奠定了基础。哈密顿的学生彼得·泰特（Peter Tait）和詹姆斯·克拉克·麦克斯韦（James Clerk Maxwell）发展了向量微积分，但其四元数形式最终被吉布斯（Josiah Willard Gibbs）和亥维赛（Oliver Heaviside）的向量分析所取代。

## 三、对物理学的影响

### 量子力学的数学基础
哈密顿力学是量子力学的直接前身。埃尔温·薛定谔在1926年发展波动力学时，将经典哈密顿量 $H(q,p)$ 通过**正则量子化**替换为算子：
$$\hat{H} = H(\hat{q}, \hat{p}), \quad \hat{q}_i = q_i, \quad \hat{p}_i = -i\hbar \frac{\partial}{\partial q_i}$$
从而得到**薛定谔方程**：
$$i\hbar \frac{\partial \psi}{\partial t} = \hat{H} \psi$$

WKB 近似方法通过将波函数设为 $\psi = e^{iS/\hbar}$ 并取 $\hbar \to 0$ 的经典极限，直接从薛定谔方程导出哈密顿–雅可比方程，建立了量子力学与经典力学的严格对应。

### 广义相对论与规范场论
广义相对论的**ADM 形式**（Arnowitt–Deser–Misner, 1962）将爱因斯坦场方程重写为哈密顿形式，其中度规 $g_{ij}$ 及其共轭动量 $\pi^{ij}$ 扮演正则变量的角色。这一形式在数值相对论和量子引力研究中至关重要。

现代规范场论——包括量子电动力学（QED）、量子色动力学（QCD）和电弱理论——均基于**哈密顿作用量原理**：
$$S = \int \left( p_i \dot{q}^i - H \right) dt$$
杨–米尔斯理论的**约束哈密顿形式**直接源自狄拉克（Paul Dirac）对约束系统的广义哈密顿分析。

### 光学与几何光学
哈密顿在1827–1830年代将光学研究数学化，建立了**哈密顿光学**。他证明了费马原理（光沿耗时最短路径传播）可导出程函方程（eikonal equation）：
$$|\nabla S|^2 = n^2$$
其中 $S$ 为程函，$n$ 为折射率。程函方程正是哈密顿–雅可比方程在光学中的类比，展示了哈密顿方法跨越物理领域的普适性。

## 四、对现代数学的深远影响

哈密顿的思想在20世纪催生了多个活跃的数学领域：
- **辛拓扑**（Symplectic Topology）：格罗莫夫（Mikhail Gromov）的伪全纯曲线理论、弗洛尔（Andreas Floer）的同调理论
- **动力系统**：庞加莱截面、混沌理论、遍历理论
- **变分法**：莫尔斯理论（Morse theory）——临界点理论与拓扑的深刻联系
- **非交换几何**：阿兰·孔涅（Alain Connes）将四元数的非交换精神推广到算子代数框架
