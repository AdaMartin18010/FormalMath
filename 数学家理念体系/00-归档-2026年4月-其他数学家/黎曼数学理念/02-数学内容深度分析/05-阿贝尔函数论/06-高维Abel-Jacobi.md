## 一、中间Jacobi簇

### 1.1 定义

**中间Jacobi簇**：

```
代数簇X：

中间Jacobi簇：
J^k(X) = H^{2k-1}(X, C) / H^{2k-1}(X, ℤ)

其中：
- k：中间度
- H：上同调

性质：
- 复环面
- Abel群
- 应用广泛
```

### 1.2 应用

**应用**：

- 几何
- 数论

## 二、Abel映射

### 2.1 推广

**高维Abel映射**：

```
代数簇X：

Abel映射：
A: CH^k(X) → J^k(X)

其中：
- CH^k(X)：Chow群
- J^k(X)：中间Jacobi簇

性质：
- 群同态
- 应用广泛
```

### 2.2 应用

**应用**：

- 除子理论

- 几何应用
