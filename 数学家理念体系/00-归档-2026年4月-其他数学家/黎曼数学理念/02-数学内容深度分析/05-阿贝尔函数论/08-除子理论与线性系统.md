## 一、除子

### 1.1 定义

**除子**：

```
紧致黎曼面R上的除子：

D = ∑_{P∈R} n_P · P

其中：
- n_P ∈ ℤ
- 只有有限个n_P ≠ 0

度：
deg(D) = ∑ n_P
```

---

### 1.2 主除子

**主除子**：

```
全纯函数f：

(f) = ∑_{P} ord_P(f) · P

性质：
- deg((f)) = 0
- 主除子群
```

---

## 二、线性系统

### 2.1 定义

**线性系统**：

```
除子D：

|D| = {E ~ D | E ≥ 0}

其中：
- ~：线性等价
- ≥ 0：有效除子

性质：
- 射影空间
- 应用广泛
```

---

### 2.2 应用

**应用**：

- 分类理论
- 枚举几何
- 几何应用
