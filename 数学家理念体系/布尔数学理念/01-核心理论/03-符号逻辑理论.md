## 一、符号逻辑的建立

### 1.1 符号化的必要性

布尔认识到，要建立精确的逻辑系统，必须使用符号：

1. **消除歧义**：自然语言存在歧义，符号可以精确表达
2. **机械化运算**：符号可以按照规则进行机械化运算
3. **抽象化处理**：符号可以代表任何符合规则的对象

### 1.2 布尔符号系统的特点

布尔的符号系统具有以下特点：

- **简洁性**：使用简单的字母和运算符号
- **直观性**：符号选择尽可能直观
- **系统性**：符号之间有明确的规则关系

## 二、符号系统

### 2.1 基本符号

**类符号**：

- $x, y, z, \ldots$：表示类（classes）或命题

**常量符号**：

- $1$：表示全类（universe）
- $0$：表示空类（nothing）

**运算符号**：

- $+$：表示"或"运算（特殊定义）
- $\times$ 或 $xy$：表示"与"运算
- $-$：表示差运算
- $1-x$：表示补运算（非）

### 2.2 符号的语义

布尔对符号的语义有明确解释：

1. **类符号**：$x$ 表示一个类，可以是任何符合逻辑的集合
2. **运算符号**：
   - $xy$ 表示两个类的交集
   - $x+y$ 表示两个类的并集（排除重叠部分）
   - $1-x$ 表示 $x$ 的补集

## 三、符号运算规则

### 3.1 基本规则

**乘法规则（与运算）**：

$$xy = yx \quad \text{（交换律）}$$
$$x(xy) = xy \quad \text{（结合律）}$$
$$xx = x \quad \text{（幂等律）}$$

**加法规则（或运算）**：

$$x + y = y + x \quad \text{（交换律）}$$
$$x + (y + z) = (x + y) + z \quad \text{（结合律）}$$
$$x + x = x \quad \text{（幂等律，特殊处理）}$$

**补运算规则（非运算）**：

$$1 - (1 - x) = x \quad \text{（双重否定）}$$
$$x + (1 - x) = 1 \quad \text{（互补性）}$$
$$x(1 - x) = 0 \quad \text{（矛盾律）}$$

### 3.2 分配律

$$x(y + z) = xy + xz$$

这是布尔代数中最重要的运算规则之一。

### 3.3 特殊性质

布尔注意到，在逻辑代数中，某些代数性质需要特殊处理：

1. **幂等律**：$xx = x$（不同于普通代数）
2. **加法限制**：$x + y$ 要求 $xy = 0$（排除重叠）
3. **单位元**：$x \cdot 1 = x$，$x + 0 = x$

## 四、符号逻辑的应用

### 4.1 逻辑推理的符号化

**例子**：三段论

前提1：所有 $x$ 是 $y$（符号化：$x(1-y) = 0$）
前提2：所有 $z$ 是 $x$（符号化：$z(1-x) = 0$）
结论：所有 $z$ 是 $y$（符号化：$z(1-y) = 0$）

通过符号运算可以证明结论。

### 4.2 逻辑方程的求解

**例子**：求解逻辑方程

已知：$x = xy$ 且 $x = x(1-y)$

从 $x = xy$ 可得：$x(1-y) = 0$
从 $x = x(1-y)$ 可得：$xy = 0$

因此：$x = 0$

### 4.3 逻辑表达式的化简

使用符号运算规则可以化简复杂的逻辑表达式。

**例子**：化简 $x + xy$

$$x + xy = x(1 + y) = x \cdot 1 = x$$

## 五、对现代逻辑的影响

### 5.1 形式逻辑的基础

布尔的工作为现代形式逻辑奠定了基础：

1. **符号化方法**：启发了后来的形式逻辑系统
2. **公理化方法**：为逻辑系统的公理化提供了范例
3. **机械化方法**：证明了逻辑推理可以机械化

### 5.2 对弗雷格的影响

布尔的工作影响了弗雷格的符号逻辑系统：

- 弗雷格发展了更完善的符号系统
- 建立了谓词逻辑
- 但继承了布尔的符号化思想

### 5.3 对现代逻辑系统的影响

现代逻辑系统（如命题逻辑、谓词逻辑）都继承了布尔的符号化方法：

- 使用符号表示命题
- 使用符号表示逻辑连接词
- 使用形式规则进行推理
