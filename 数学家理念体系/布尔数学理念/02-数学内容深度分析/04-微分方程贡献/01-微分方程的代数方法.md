## 二、微分方程的代数化

### 2.1 微分算子

**微分算子**：

定义微分算子 $D = \frac{d}{dx}$，将微分方程转化为算子方程。

**数学表述**：

微分算子：

$$D = \frac{d}{dx}, \quad D^n = \frac{d^n}{dx^n}$$

**例子1：微分算子**：

微分方程 $y'' + 2y' + y = 0$ 可以表示为：

$$(D^2 + 2D + 1)y = 0$$

### 2.2 算子代数

**算子代数**：

微分算子可以像代数符号一样运算。

**数学表述**：

算子运算：

$$(D + a)(D + b) = D^2 + (a + b)D + ab$$

**例子2：算子分解**：

$$D^2 + 3D + 2 = (D + 1)(D + 2)$$

## 三、求解方法

### 3.1 特征方程法

**特征方程法**：

对于线性常系数微分方程，转化为特征方程求解。

**数学表述**：

特征方程：

$$a_n D^n + a_{n-1} D^{n-1} + \cdots + a_1 D + a_0 = 0$$

对应的特征方程：

$$a_n r^n + a_{n-1} r^{n-1} + \cdots + a_1 r + a_0 = 0$$

**例子3：特征方程法**：

求解 $y'' - 3y' + 2y = 0$：

特征方程：$r^2 - 3r + 2 = 0$，根为 $r = 1, 2$

解：$y = C_1 e^x + C_2 e^{2x}$

### 3.2 算子方法

**算子方法**：

使用算子分解求解微分方程。

**例子4：算子方法**：

求解 $(D^2 + 3D + 2)y = e^x$：

分解：$(D + 1)(D + 2)y = e^x$

先解 $(D + 2)z = e^x$，再解 $(D + 1)y = z$
