## 一、历史背景

**布尔的方法**：

布尔使用算子方法处理微分方程，将微分算子当作代数符号进行运算。

**核心思想**：

算子可以像代数符号一样进行加、减、乘、除等运算。

**数学表述**：

算子代数：

$$\mathcal{D} = \{D, D^2, D^3, \ldots\}$$

其中 $D = \frac{d}{dx}$。

## 二、算子方法

### 2.1 算子的定义

**微分算子**：

定义 $D = \frac{d}{dx}$，则 $D^n = \frac{d^n}{dx^n}$。

**数学表述**：

算子作用：

$$D[f(x)] = \frac{df}{dx}, \quad D^n[f(x)] = \frac{d^n f}{dx^n}$$

**例子1：算子作用**：

$$D[e^{ax}] = a e^{ax}, \quad D^2[e^{ax}] = a^2 e^{ax}$$

### 2.2 算子的运算

**算子的代数运算**：

算子可以像多项式一样运算。

**数学表述**：

算子多项式：

$$P(D) = a_n D^n + a_{n-1} D^{n-1} + \cdots + a_1 D + a_0$$

**例子2：算子运算**：

$$(D + 1)(D + 2) = D^2 + 3D + 2$$

### 2.3 算子的逆

**算子的逆**：

定义算子的逆运算。

**数学表述**：

逆算子：

$$(D - a)^{-1}[f(x)] = e^{ax} \int e^{-ax} f(x) dx$$

**例子3：逆算子**：

$$(D - 1)^{-1}[e^x] = e^x \int e^{-x} \cdot e^x dx = e^x \cdot x$$

## 三、符号计算

### 3.1 符号运算

**符号运算**：

将微分方程转化为符号表达式进行计算。

**数学表述**：

符号计算：

$$\text{微分方程} \to \text{符号表达式} \to \text{代数运算} \to \text{解}$$

**例子4：符号计算**：

求解 $(D^2 + 3D + 2)y = 0$：

因式分解：$(D + 1)(D + 2) = 0$

解：$y = C_1 e^{-x} + C_2 e^{-2x}$

### 3.2 算子分解

**算子分解**：

将复杂算子分解为简单算子的乘积。

**数学表述**：

算子分解：

$$P(D) = (D - r_1)(D - r_2) \cdots (D - r_n)$$

其中 $r_i$ 是特征根。

**例子5：算子分解**：

$$D^2 - 5D + 6 = (D - 2)(D - 3)$$

## 四、应用

### 4.1 微分方程求解

**微分方程求解**：

使用算子方法求解微分方程。

**例子6：非齐次方程**：

求解 $(D^2 + D - 2)y = e^x$：

先解齐次方程：$(D + 2)(D - 1)y = 0$，解为 $y_h = C_1 e^{-2x} + C_2 e^x$

特解：$y_p = \frac{1}{D^2 + D - 2}[e^x] = \frac{1}{0}[e^x] = \frac{x}{2D + 1}[e^x] = \frac{x}{3} e^x$

### 4.2 积分方程

**积分方程**：

算子方法可以应用于积分方程。

## 六、数学公式总结

### 核心公式

1. **微分算子**：
   $$D = \frac{d}{dx}, \quad D^n = \frac{d^n}{dx^n}$$

2. **算子多项式**：
   $$P(D) = a_n D^n + a_{n-1} D^{n-1} + \cdots + a_0$$

3. **算子分解**：
   $$P(D) = (D - r_1)(D - r_2) \cdots (D - r_n)$$

4. **逆算子**：
   $$(D - a)^{-1}[f] = e^{ax} \int e^{-ax} f dx$$

5. **算子作用**：
   $$P(D)[e^{rx}] = P(r) e^{rx}$$
