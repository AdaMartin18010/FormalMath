## 一、历史背景

**布尔的工作**：

布尔建立了线性微分方程的理论，使用代数方法研究线性微分方程。

**核心思想**：

线性微分方程可以转化为代数问题，使用代数方法求解。

**数学表述**：

线性微分方程 $\to$ 算子方程 $\to$ 代数方程

## 二、线性微分方程的定义

### 2.1 线性微分方程

**线性微分方程**：

$n$ 阶线性微分方程的一般形式：

$$a_n(x) y^{(n)} + a_{n-1}(x) y^{(n-1)} + \cdots + a_1(x) y' + a_0(x) y = f(x)$$

**数学表述**：

用算子表示：

$$L[y] = f(x)$$

其中 $L = a_n D^n + a_{n-1} D^{n-1} + \cdots + a_0$ 是线性算子。

**例子1：线性微分方程**：

$$y'' + 2y' + y = e^x$$

是二阶线性非齐次微分方程。

### 2.2 齐次方程

**齐次线性微分方程**：

当 $f(x) = 0$ 时，方程为齐次：

$$L[y] = 0$$

**数学表述**：

齐次方程：

$$a_n y^{(n)} + a_{n-1} y^{(n-1)} + \cdots + a_0 y = 0$$

**例子2：齐次方程**：

$$y'' - 3y' + 2y = 0$$

## 三、求解理论

### 3.1 常系数方程

**常系数线性微分方程**：

当系数 $a_i$ 为常数时，使用特征方程法。

**数学表述**：

特征方程：

$$a_n r^n + a_{n-1} r^{n-1} + \cdots + a_0 = 0$$

**例子3：常系数方程**：

求解 $y'' - 5y' + 6y = 0$：

特征方程：$r^2 - 5r + 6 = 0$，根为 $r = 2, 3$

解：$y = C_1 e^{2x} + C_2 e^{3x}$

### 3.2 变系数方程

**变系数线性微分方程**：

当系数依赖于 $x$ 时，使用其他方法。

**数学表述**：

变系数方程：

$$a_n(x) y^{(n)} + \cdots + a_0(x) y = f(x)$$

**例子4：变系数方程**：

$$x^2 y'' + xy' + y = 0$$

是Euler方程。

### 3.3 非齐次方程

**非齐次线性微分方程**：

使用特解+通解的方法。

**数学表述**：

非齐次方程的解：

$$y = y_h + y_p$$

其中 $y_h$ 是齐次方程的通解，$y_p$ 是非齐次方程的特解。

**例子5：非齐次方程**：

求解 $y'' + y = \sin x$：

齐次解：$y_h = C_1 \cos x + C_2 \sin x$

特解：$y_p = -\frac{x}{2} \cos x$

## 六、数学公式总结

### 核心公式

1. **线性微分方程**：
   $$L[y] = a_n y^{(n)} + \cdots + a_0 y = f(x)$$

2. **特征方程**：
   $$a_n r^n + a_{n-1} r^{n-1} + \cdots + a_0 = 0$$

3. **齐次方程的解**：
   $$y_h = \sum_{i=1}^n C_i e^{r_i x}$$

4. **非齐次方程的解**：
   $$y = y_h + y_p$$

5. **线性算子的性质**：
   $$L[cy_1 + dy_2] = cL[y_1] + dL[y_2]$$
