## 一、布尔代数的定义

### 1.1 集合论定义

一个**布尔代数**是一个六元组 $(B, \land, \lor, \neg, 0, 1)$，其中：

- $B$ 是一个非空集合（称为布尔代数的载体）
- $\land: B \times B \to B$ 是二元运算（称为"与"或"交"）
- $\lor: B \times B \to B$ 是二元运算（称为"或"或"并"）
- $\neg: B \to B$ 是一元运算（称为"非"或"补"）
- $0 \in B$ 是最小元（零元）
- $1 \in B$ 是最大元（单位元）

满足布尔代数的公理。

### 1.2 二元布尔代数

最简单的布尔代数是 **二元布尔代数** $\{0, 1\}$，其运算表如下：

**与运算** $\land$：

| $\land$ | 0 | 1 |
|---------|---|---|
| 0 | 0 | 0 |
| 1 | 0 | 1 |

**或运算** $\lor$：

| $\lor$ | 0 | 1 |
|---------|---|---|
| 0 | 0 | 1 |
| 1 | 1 | 1 |

**非运算** $\neg$：

| $x$ | $\neg x$ |
|-----|----------|
| 0 | 1 |
| 1 | 0 |

## 二、布尔代数的公理系统

### 2.1 标准公理系统

布尔代数满足以下公理：

**公理1：交换律**
$$x \land y = y \land x$$
$$x \lor y = y \lor x$$

**公理2：结合律**
$$(x \land y) \land z = x \land (y \land z)$$
$$(x \lor y) \lor z = x \lor (y \lor z)$$

**公理3：分配律**
$$x \land (y \lor z) = (x \land y) \lor (x \land z)$$
$$x \lor (y \land z) = (x \lor y) \land (x \lor z)$$

**公理4：单位元**
$$x \land 1 = x$$
$$x \lor 0 = x$$

**公理5：互补律**
$$x \land \neg x = 0$$
$$x \lor \neg x = 1$$

### 2.2 亨廷顿公理系统

亨廷顿给出了一个最小公理系统，只包含三个公理：

1. **交换律**：$x \lor y = y \lor x$
2. **分配律**：$x \lor (y \land z) = (x \lor y) \land (x \lor z)$
3. **互补律**：存在 $\neg x$ 使得 $x \lor \neg x = 1$ 且 $x \land \neg x = 0$

从这个最小系统可以推出所有其他性质。

## 三、基本运算的性质

### 3.1 幂等律

$$x \land x = x$$
$$x \lor x = x$$

**证明**：
$$x = x \land 1 = x \land (x \lor \neg x) = (x \land x) \lor (x \land \neg x) = (x \land x) \lor 0 = x \land x$$

### 3.2 吸收律

$$x \lor (x \land y) = x$$
$$x \land (x \lor y) = x$$

**证明**（第一个）：
$$x \lor (x \land y) = (x \land 1) \lor (x \land y) = x \land (1 \lor y) = x \land 1 = x$$

### 3.3 德摩根定律

$$\neg(x \land y) = \neg x \lor \neg y$$
$$\neg(x \lor y) = \neg x \land \neg y$$

## 四、布尔代数的例子

### 4.1 集合代数

给定全集 $U$，$U$ 的幂集 $\mathcal{P}(U)$ 在集合运算下形成布尔代数：

- $B = \mathcal{P}(U)$
- $\land = \cap$（交集）
- $\lor = \cup$（并集）
- $\neg$ = 补集运算
- $0 = \emptyset$（空集）
- $1 = U$（全集）

### 4.2 命题逻辑

命题逻辑的等价类形成布尔代数：

- $B$ = 命题的等价类
- $\land$ = 合取
- $\lor$ = 析取
- $\neg$ = 否定
- $0$ = 矛盾命题
- $1$ = 重言命题

### 4.3 开关电路

开关电路的状态形成布尔代数：

- $B = \{0, 1\}$（开关断开/闭合）
- $\land$ = 串联
- $\lor$ = 并联
- $\neg$ = 反向开关
- $0$ = 断开
- $1$ = 闭合
