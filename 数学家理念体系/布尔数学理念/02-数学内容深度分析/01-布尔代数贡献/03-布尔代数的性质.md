# 布尔代数的性质

## 一、基本性质

### 1.1 幂等律

$$x \land x = x, \quad x \lor x = x$$

### 1.2 吸收律

$$x \lor (x \land y) = x, \quad x \land (x \lor y) = x$$

### 1.3 双重否定律

$$\neg(\neg x) = x$$

---

## 二、德摩根定律

$$\neg(x \land y) = \neg x \lor \neg y$$
$$\neg(x \lor y) = \neg x \land \neg y$$

---

## 三、对偶原理

在布尔代数中，将 $\land$ 和 $\lor$ 互换，$0$ 和 $1$ 互换，得到的表达式仍然有效。
