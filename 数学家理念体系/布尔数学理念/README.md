## 📚 核心内容方向

### 1. 布尔代数（Boolean Algebra）

- **历史背景**: 1854年《思维规律的研究》
- **数学内容**: 逻辑运算的代数系统
- **应用**: 逻辑电路、集合论、逻辑学

### 2. 逻辑代数化

- **核心思想**: 用代数方法处理逻辑推理
- **方法**: 符号逻辑、逻辑方程
- **影响**: 现代逻辑学的基础

### 3. 概率论

- **贡献**: 概率的代数理论
- **方法**: 条件概率的代数表示
- **应用**: 统计推理、决策理论

### 4. 微分方程

- **方法**: 算子方法、符号计算
- **理论**: 线性微分方程
- **应用**: 物理、工程
