## 📖 项目概述

本项目致力于全面梳理和呈现**赫尔曼·外尔**（Hermann Weyl, 1885-1955）的数学理念体系。外尔是20世纪最重要的数学家之一，他在群表示论、规范理论、微分几何等领域做出了重大贡献，对现代数学和物理学产生了深远影响。

---

## 🎯 核心主题

### 1. 群表示论
- 紧李群的表示
- 外尔特征公式
- 表示论的统一

### 2. 规范理论
- 规范不变性
- 规范场论
- 统一场论

### 3. 微分几何
- 外尔张量
- 共形几何
- 几何结构

### 4. 数学哲学
- 数学的统一性
- 数学与物理的关系
- 数学的直觉
