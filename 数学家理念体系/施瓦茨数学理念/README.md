## 📖 项目概述

本项目致力于全面梳理和呈现**赫尔曼·施瓦茨**（Hermann Schwarz, 1843-1921）的数学理念体系。施瓦茨是19世纪重要的数学家，在复分析、几何不等式和变分法方面做出了重要贡献，对现代分析学和几何学产生了深远影响。
