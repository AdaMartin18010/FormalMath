## 一、变分法的定义

### 1.1 变分法的概念

**变分法的定义**：

变分法是研究泛函极值问题的数学分支，它研究函数的函数的极值问题。

**变分法与微积分的区别**：

- **微积分**：研究函数的极值（$f(x)$ 的极值）
- **变分法**：研究泛函的极值（$J[y]$ 的极值，其中 $y$ 是函数）

**变分法的特点**：

- **泛函极值**：不是函数的极值，而是泛函的极值
- **函数空间**：在函数空间中寻找极值
- **约束条件**：通常有边界条件约束

### 1.2 变分问题的形式

**变分问题的定义**：

变分问题是求泛函的极值问题：

$$\min_{y} J[y] = \int_a^b F(x, y(x), y'(x)) \, dx$$

其中：
- $y(x)$ 是待求函数
- $F(x, y, y')$ 是给定的函数
- 边界条件：$y(a) = y_a$，$y(b) = y_b$

**变分问题的例子**：

- **最短路径**：最小化路径长度
- **最速降线**：最小化滑行时间
- **等周问题**：在给定周长下最大化面积

### 1.3 泛函的概念

**泛函的定义**：

泛函是函数的函数，它将函数映射到实数：

$$J: \mathcal{F} \to \mathbb{R}$$

其中 $\mathcal{F}$ 是函数空间。

**泛函的例子**：

**路径长度泛函**：
$$L[y] = \int_a^b \sqrt{1 + (y')^2} \, dx$$

**面积泛函**：
$$A[y] = \int_a^b y(x) \, dx$$

**能量泛函**：
$$E[y] = \int_a^b \left(\frac{1}{2}(y')^2 - V(y)\right) \, dx$$

## 二、变分法的基础概念

### 2.1 变分的定义

**变分的定义**：

函数 $y(x)$ 的变分 $\delta y(x)$ 是满足边界条件的任意函数 $\eta(x)$：

$$\delta y(x) = \eta(x)$$

其中 $\eta(a) = \eta(b) = 0$。

**变分的性质**：

- **边界条件**：$\delta y(a) = \delta y(b) = 0$
- **导数变分**：$\delta(y') = (\delta y)'$
- **线性性**：$\delta(y_1 + y_2) = \delta y_1 + \delta y_2$

**变分的直观理解**：

变分是函数在函数空间中的"微小变化"。

### 2.2 泛函的变分

**泛函的变分**：

泛函 $J[y]$ 的变分定义为：

$$\delta J = \left.\frac{d}{d\varepsilon}\right|_{\varepsilon=0} J[y + \varepsilon \delta y]$$

**变分的计算**：

设 $J[y] = \int_a^b F(x, y, y') \, dx$，则：

$$\delta J = \int_a^b \left[\frac{\partial F}{\partial y} \delta y + \frac{\partial F}{\partial y'} \delta y'\right] \, dx$$

使用分部积分：

$$\delta J = \int_a^b \left[\frac{\partial F}{\partial y} - \frac{d}{dx}\left(\frac{\partial F}{\partial y'}\right)\right] \delta y \, dx$$

### 2.3 极值条件

**极值的必要条件**：

如果 $y(x)$ 使泛函 $J[y]$ 达到极值，则必须满足：

$$\delta J = 0$$

对所有变分 $\delta y$ 成立。

**欧拉-拉格朗日方程**：

由 $\delta J = 0$ 对所有 $\delta y$ 成立，得到：

$$\frac{\partial F}{\partial y} - \frac{d}{dx}\left(\frac{\partial F}{\partial y'}\right) = 0$$

这就是**欧拉-拉格朗日方程**。

## 三、变分法的基本方法

### 3.1 变分方法

**变分方法的基本步骤**：

1. **建立泛函**：将问题转化为泛函极值问题
2. **变分**：对函数进行变分
3. **极值条件**：利用极值条件得到微分方程
4. **求解**：求解微分方程得到极值函数

**变分步骤**：

1. **建立泛函**：
   $$J[y] = \int_a^b F(x, y, y') \, dx$$

2. **变分**：
   $$y(x) \to y(x) + \varepsilon \eta(x)$$

3. **极值条件**：
   $$\left.\frac{d}{d\varepsilon}\right|_{\varepsilon=0} J[y + \varepsilon \eta] = 0$$

4. **得到微分方程**：
   $$\frac{\partial F}{\partial y} - \frac{d}{dx}\left(\frac{\partial F}{\partial y'}\right) = 0$$

### 3.2 欧拉方程

**欧拉-拉格朗日方程**：

$$\frac{\partial F}{\partial y} - \frac{d}{dx}\left(\frac{\partial F}{\partial y'}\right) = 0$$

**展开形式**：

$$\frac{\partial F}{\partial y} - \frac{\partial^2 F}{\partial x \partial y'} - y' \frac{\partial^2 F}{\partial y \partial y'} - y'' \frac{\partial^2 F}{\partial (y')^2} = 0$$

**特殊情况**：

**F不显式依赖x**：

如果 $F = F(y, y')$（不依赖x），则存在首次积分：

$$F - y' \frac{\partial F}{\partial y'} = \text{常数}$$

**F不显式依赖y**：

如果 $F = F(x, y')$（不依赖y），则：

$$\frac{\partial F}{\partial y'} = \text{常数}$$

### 3.3 边界条件

**固定边界条件**：

- **第一类边界条件**：$y(a) = y_a$，$y(b) = y_b$
- **自然边界条件**：$\frac{\partial F}{\partial y'}|_{x=a} = 0$，$\frac{\partial F}{\partial y'}|_{x=b} = 0$

**边界条件的处理**：

- **固定边界**：直接使用边界条件
- **自然边界**：从变分条件得到边界条件

### 4.2 等周问题

**问题陈述**：

在所有周长为L的闭曲线中，哪个围成的面积最大？

**泛函**：

### 4.3 最短路径问题

**问题陈述**：

平面上两点间的最短路径是什么？

**泛函**：

$$L[y] = \int_a^b \sqrt{1 + (y')^2} \, dx$$

**欧拉方程**：

$$\frac{y'}{\sqrt{1 + (y')^2}} = \text{常数}$$

**解**：

## 五、变分法的性质

### 5.1 基本性质

**变分法的基本性质**：

- **极值性**：变分法研究泛函的极值
- **最优性**：变分法寻找最优函数
- **变分性**：变分法使用变分方法

**变分法的特点**：

- **函数空间**：在函数空间中寻找极值
- **约束条件**：通常有边界条件约束
- **微分方程**：极值函数满足微分方程

### 5.2 变分法的结构

**变分法的结构**：

变分法的结构由以下要素决定：

- **函数空间**：待求函数的空间
- **泛函**：函数的函数
- **约束条件**：边界条件等约束

**变分问题的分类**：

- **无约束变分问题**：只有边界条件
- **约束变分问题**：有额外的约束条件
- **等周问题**：有积分约束
