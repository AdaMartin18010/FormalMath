## 一、变分法理论的建立

### 1.1 历史背景

**变分法理论建立的背景**：

17-18世纪，数学家们研究各种优化问题：

- **等周问题**：在所有周长相同的闭曲线中，哪个围成的面积最大？
- **最速降线问题**：在重力作用下，从一点到另一点，什么曲线使滑行时间最短？
- **最短路径问题**：两点间的最短路径是什么？

**传统方法的局限**：

传统的微积分方法只能处理函数的极值问题，无法处理函数的函数的极值问题（泛函的极值问题）。

**变分法的需求**：

需要新的数学方法来处理：

- **泛函的极值**：函数的函数的极值
- **优化问题**：各种优化问题
- **物理问题**：力学中的优化原理

### 1.2 最速降线问题

**Johann Bernoulli的问题（1696）**：

1696年，约翰·伯努利在《Acta Eruditorum》上提出了最速降线问题（Brachistochrone Problem）：

> "在重力作用下，从点A到点B（A高于B），什么形状的曲线使质点滑行时间最短？"

**问题的挑战**：

这个问题看似简单，实际上非常困难：

- **直觉错误**：直觉上认为直线最短，但实际上不是
- **需要新方法**：传统的微积分方法无法解决
- **吸引数学家**：吸引了当时最优秀的数学家

**参与解决的数学家**：

- **Johann Bernoulli**：问题的提出者
- **Jakob Bernoulli**：Johann的兄弟，也解决了问题
- **Newton**：匿名解答，Johann看到解答后说"从爪印认出狮子"
- **Leibniz**：提供了解决方法
- **de l'Hôpital**：也解决了问题
- **欧拉**：后来系统化了变分法

**历史意义**：

最速降线问题的提出和解决标志着：

- **变分法的诞生**：开创了变分法这一新的数学分支
- **数学与物理的连接**：连接了数学优化与物理问题
- **新方法的出现**：为处理泛函极值问题提供了新方法

### 1.3 理论的建立

**伯努利家族的贡献**：

- **Johann Bernoulli（1696）**：提出最速降线问题
- **Jakob Bernoulli**：解决了最速降线问题，发展了变分方法
- **方法创新**：发展了处理变分问题的方法

**变分法的建立**：

虽然欧拉后来系统化了变分法，但伯努利家族：

- **提出问题**：提出了变分法的经典问题
- **解决问题**：解决了最速降线问题
- **方法创新**：发展了变分方法

## 二、变分问题的理论

### 2.1 变分问题的形式

**变分问题的定义**：

变分问题是求泛函的极值问题：

$$\min_{y} J[y] = \int_a^b F(x, y(x), y'(x)) \, dx$$

其中：

- $y(x)$ 是待求函数
- $F(x, y, y')$ 是给定的函数
- 边界条件：$y(a) = y_a$，$y(b) = y_b$

**变分问题的特点**：

- **泛函极值**：不是函数的极值，而是泛函的极值
- **函数空间**：在函数空间中寻找极值
- **约束条件**：通常有边界条件约束

**例子**：

**最短路径问题**：

- 泛函：$L[y] = \int_a^b \sqrt{1 + (y')^2} \, dx$
- 目标：最小化路径长度
- 解：直线

**最速降线问题**：

- 泛函：$T[y] = \int_0^a \sqrt{\frac{1 + (y')^2}{2gy}} \, dx$
- 目标：最小化滑行时间
- 解：摆线

### 2.2 泛函的概念

**泛函的定义**：

泛函是函数的函数，它将函数映射到实数：

$$J: \mathcal{F} \to \mathbb{R}$$

其中 $\mathcal{F}$ 是函数空间。

**例子**：

**路径长度泛函**：
$$L[y] = \int_a^b \sqrt{1 + (y')^2} \, dx$$

**面积泛函**：
$$A[y] = \int_a^b y(x) \, dx$$

**能量泛函**：
$$E[y] = \int_a^b \leqft(\frac{1}{2}(y')^2 - V(y)\right) \, dx$$

### 2.3 变分的概念

**变分的定义**：

函数 $y(x)$ 的变分 $\delta y(x)$ 是满足边界条件的任意函数 $\eta(x)$：

$$\delta y(x) = \eta(x)$$

其中 $\eta(a) = \eta(b) = 0$。

**变分的性质**：

- **边界条件**：$\delta y(a) = \delta y(b) = 0$
- **导数变分**：$\delta(y') = (\delta y)'$
- **线性性**：$\delta(y_1 + y_2) = \delta y_1 + \delta y_2$

**泛函的变分**：

泛函 $J[y]$ 的变分定义为：

$$\delta J = \leqft.\frac{d}{d\varepsilon}\right|_{\varepsilon=0} J[y + \varepsilon \delta y]$$

**极值条件**：

如果 $y(x)$ 使泛函 $J[y]$ 达到极值，则：

$$\delta J = 0$$

对所有变分 $\delta y$ 成立。

## 三、最速降线问题的解决

### 3.1 问题的数学表述

**物理问题**：

在重力场中，从点 $(0, 0)$ 到点 $(a, b)$（$b < 0$），求曲线 $y(x)$，使质点沿曲线滑行时间最短。

**假设条件**：

- 无摩擦
- 重力加速度为 $g$
- 初始速度 $v_0 = 0$

**能量守恒**：

在点 $(x, y)$ 处，由能量守恒：

$$\frac{1}{2}mv^2 + mgy = 0$$

因此速度：

$$v = \sqrt{-2gy}$$

### 3.2 物理分析

**时间元素**：

在曲线元素 $ds$ 上，时间元素为：

$$dt = \frac{ds}{v}$$

其中：

- $ds = \sqrt{1 + (y')^2} \, dx$（弧长元素）
- $v = \sqrt{-2gy}$（速度）

因此：

$$dt = \frac{\sqrt{1 + (y')^2}}{\sqrt{-2gy}} \, dx$$

**总时间**：

总滑行时间为：

$$T[y] = \int_0^a \frac{\sqrt{1 + (y')^2}}{\sqrt{-2gy}} \, dx = \frac{1}{\sqrt{2g}} \int_0^a \sqrt{\frac{1 + (y')^2}{-y}} \, dx$$

**变分问题**：

这是一个变分问题：

$$\min_{y} T[y] = \int_0^a F(x, y, y') \, dx$$

其中：

$$F(x, y, y') = \sqrt{\frac{1 + (y')^2}{-y}}$$

边界条件：$y(0) = 0$，$y(a) = b$。

### 3.3 摆线解

**摆线的定义**：

摆线（Cycloid）是一个圆在直线上滚动时，圆上一点的轨迹。

**摆线的参数方程**：

$$x = R(\theta - \sin \theta)$$
$$y = R(1 - \cos \theta)$$

其中 $R$ 是滚动圆的半径，$\theta$ 是参数。

**摆线的性质**：

- **等时性**：摆线具有等时性，即无论起始位置如何，滑行时间相同
- **最速性**：摆线是最速降线
- **优美性**：摆线是优美的数学曲线

**验证**：

可以验证摆线满足最速降线问题的条件，并且确实使滑行时间最短。

**历史意义**：

摆线解的发现标志着：

- **变分法的成功**：变分法成功解决了优化问题
- **数学与物理的统一**：数学优化与物理问题统一
- **新方法的验证**：验证了变分方法的有效性

## 四、变分法的方法

### 4.1 变分方法

**变分方法的基本思想**：

1. **建立泛函**：将问题转化为泛函极值问题
2. **变分**：对函数进行变分
3. **极值条件**：利用极值条件得到微分方程
4. **求解**：求解微分方程得到极值函数

**变分步骤**：

1. **建立泛函**：
   $$J[y] = \int_a^b F(x, y, y') \, dx$$

2. **变分**：
   $$y(x) \to y(x) + \varepsilon \eta(x)$$

3. **极值条件**：
   $$\leqft.\frac{d}{d\varepsilon}\right|_{\varepsilon=0} J[y + \varepsilon \eta] = 0$$

4. **得到微分方程**：
   $$\frac{\partial F}{\partial y} - \frac{d}{dx}\leqft(\frac{\partial F}{\partial y'}\right) = 0$$

### 4.2 极值条件

**极值的必要条件**：

如果 $y(x)$ 使泛函 $J[y]$ 达到极值，则必须满足：

$$\frac{\partial F}{\partial y} - \frac{d}{dx}\leqft(\frac{\partial F}{\partial y'}\right) = 0$$

这就是**欧拉-拉格朗日方程**（虽然欧拉和拉格朗日后来才系统化）。

**展开形式**：

$$\frac{\partial F}{\partial y} - \frac{\partial^2 F}{\partial x \partial y'} - y' \frac{\partial^2 F}{\partial y \partial y'} - y'' \frac{\partial^2 F}{\partial (y')^2} = 0$$

### 4.3 欧拉方程的前身

**伯努利的方法**：

虽然欧拉后来系统化了欧拉方程，但伯努利家族：

- **使用变分方法**：使用变分方法解决最速降线问题
- **得到微分方程**：通过变分得到微分方程
- **求解问题**：成功求解了最速降线问题

**历史发展**：

- **1696**：Johann Bernoulli提出最速降线问题
- **1697**：多个数学家解决了问题
- **1744**：欧拉系统化变分法，建立欧拉方程
- **1755**：拉格朗日改进方法，建立拉格朗日乘数法
