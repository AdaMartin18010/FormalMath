## 三、Riemann-Roch的推广

### 3.1 经典Riemann-Roch

**黎曼（1857）**：

```
紧致黎曼面R，除子D：

dim L(D) - dim L(K-D) = deg(D) - g + 1
```

---

### 3.2 Hirzebruch-Riemann-Roch

**Hirzebruch（1954）**：

```
光滑射影簇X，向量丛E：

χ(X, E) = ∫_X ch(E) · td(X)
```

**格洛腾迪克的推广**：

```
概形X，向量丛E：

χ(X, E) = ∫_X ch(E) · td(X)

统一框架
```
