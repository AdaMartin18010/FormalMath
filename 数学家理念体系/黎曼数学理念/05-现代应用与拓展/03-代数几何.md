```

```

## 三、上同调

### 3.1 层上同调

**定义**：

```
层F的上同调：
H^i(X, F)

性质：
- 函子性
- 长正合列
- 有限维（紧致）
```

**与黎曼面**：

```
黎曼面：
H^i(R, O) = 上同调

概形：
H^i(X, F) = 层上同调

统一框架
```

---

### 3.2 应用

**Riemann-Roch**：

```
概形X，向量丛E：

χ(X, E) = ∫_X ch(E) · td(X)

其中：
- ch：Chern特征
- td：Todd类
- χ：Euler示性数
```

## 六、黎曼假设的几何化

### 6.1 Weil猜想

**Weil（1949）**：

```
有限域F_q上的代数簇X：

Zeta函数：
Z(X, T) = ∏_{i=0}^{2n} P_i(T)^{(-1)^{i+1}}

Riemann假设：
|α_{ij}| = q^{i/2}
```

**Deligne证明（1974）**：

```
使用：
- Motive理论
- 代数几何
- 层上同调

结果：
- ✅ 证明
- Fields奖
```
