## 二、世界面

### 2.1 定义

**世界面**：

```
弦的时空轨迹：
Σ = {X^μ(σ,τ)}

其中：
- σ：空间坐标（沿弦）
- τ：时间坐标
- X^μ：嵌入到时空

性质：
- 2维曲面
- 黎曼面结构
```

---

### 2.2 度规

**诱导度规**：

```
世界面度规：
h_{αβ} = ∂_α X^μ ∂_β X_ν g_{μν}

其中：
- g_{μν}：时空度规
- α,β = σ,τ

性质：
- 2维Lorentz度规
- 共形类
```

---

## 三、Nambu-Goto作用量

### 3.1 定义

**作用量**：

```
S_{NG} = -T ∫ d²σ √(-det h)

其中：
- T：弦张力
- h_{αβ}：诱导度规
- d²σ = dσ dτ

物理意义：
- 世界面面积
- 最小作用量
```

---

### 3.2 Polyakov形式

**作用量**：

```
S_P = -(T/2) ∫ d²σ √g g^{αβ} ∂_α X^μ ∂_β X_μ

其中：
- g_{αβ}：世界面度规（独立）
- X^μ：嵌入

优势：
- 二次型
- 便于量子化
- 共形不变
```

## 四、共形场论

### 4.1 共形对称性

**共形变换**：

```
世界面坐标：
σ^α → σ'^α(σ)

度规：
g'_{αβ} = e^{φ(σ)} g_{αβ}

性质：
- 保持角度
- 局部缩放
- 对称性
```

---

### 4.2 Virasoro代数

**生成元**：

```
L_n = (1/2) ∑_{m} :α_{n-m}·α_m:

对易关系：
[L_m, L_n] = (m-n) L_{m+n} + (c/12) m(m²-1) δ_{m+n,0}

其中：
- c：中心荷
- 量子修正
```

## 五、弦的量子化

### 5.1 正则量子化

**方法**：

```
1. 作用量S_P
2. 共形规范：g_{αβ} = η_{αβ}
3. 约束：T_{αβ} = 0
4. 量子化：对易关系
```

**结果**：

```
质量谱：
M² = (2/α') (N - a)

其中：
- α'：Regge斜率
- N：激发数
- a：正规化常数
```

---

### 5.2 路径积分

**方法**：

```
配分函数：
Z = ∫ D[g] D[X] e^{-S_P}

模空间积分：
Z = ∫_{M_g} (dg) ∫ D[X] e^{-S_P}

其中：
M_g：亏格g黎曼面的模空间
```

### 6.2 弦振幅

**散射振幅**：

```
n点振幅：
A_n = ∫_{M_{g,n}} (dg) ∫ D[X] e^{-S_P} ∏_{i=1}^n V_i
```
