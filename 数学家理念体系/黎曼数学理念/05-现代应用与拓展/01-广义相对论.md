## 二、时空几何

### 2.1 4维流形

**时空M**：

```
4维Lorentz流形：
- 局部：R⁴
- 度规：Lorentz符号(+, -, -, -)
- 因果结构

坐标：
x^μ = (t, x, y, z)

度规：
ds² = g_{μν} dx^μ dx^ν
```

---

### 2.2 度规

**Lorentz度规**：

```
平坦时空（Minkowski）：
ds² = dt² - dx² - dy² - dz²

弯曲时空：
ds² = g_{μν}(x) dx^μ dx^ν

其中：
g_{μν}：时空度规
- 对称
- 非退化
- 符号(+, -, -, -)
```

---

## 三、Einstein场方程

### 3.1 方程

**Einstein场方程**：

```
G_{μν} = 8πG T_{μν}

其中：
- G_{μν} = R_{μν} - (1/2) R g_{μν}（Einstein张量）
- T_{μν}：能量-动量张量
- G：Newton引力常数

等价：
R_{μν} - (1/2) R g_{μν} = 8πG T_{μν}
```

---

### 3.2 真空方程

**无物质**：

```
T_{μν} = 0

方程：
R_{μν} = 0

等价：
Einstein流形（λ=0）

解：
- Minkowski（平坦）
- Schwarzschild（黑洞）
- Kerr（旋转黑洞）
```

## 四、测地线

### 4.1 粒子轨迹

**自由粒子**：

```
测地线方程：
d²x^μ/dτ² + Γ^μ_{νρ} (dx^ν/dτ)(dx^ρ/dτ) = 0

其中：
- τ：固有时
- Γ^μ_{νρ}：Christoffel符号

物理意义：
- 无外力
- 沿测地线
- 时空弯曲 = "引力"
```

---

### 4.2 光线

**零测地线**：

```
ds² = 0

方程：
d²x^μ/dλ² + Γ^μ_{νρ} (dx^ν/dλ)(dx^ρ/dλ) = 0

其中：
λ：仿射参数

应用：
- 光线偏折
- 引力透镜
- 时间延迟
```

## 七、宇宙学

### 7.1 Friedmann方程

**均匀各向同性**：

```
度规：
ds² = dt² - a(t)² [dr²/(1-kr²) + r²(dθ²+sin²θ dφ²)]

其中：
- a(t)：尺度因子
- k：曲率（+1, 0, -1）

Friedmann方程：
(ȧ/a)² = (8πG/3)ρ - k/a²
ä/a = -(4πG/3)(ρ+3p)
```

---

### 7.2 宇宙演化

**解**：

```
k = 0（平坦）：
- a(t) ∝ t^{2/3}（物质主导）
- a(t) ∝ t^{1/2}（辐射主导）

k = +1（正曲率）：
- 闭合宇宙
- 先膨胀后收缩

k = -1（负曲率）：
- 开放宇宙
- 永远膨胀
```
