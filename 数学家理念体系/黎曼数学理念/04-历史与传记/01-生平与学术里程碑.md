## 一、早期生活

**出生**：

```
1826年9月17日
德国汉诺威的Breselenz
```

**家庭**：

```
父亲：路德派牧师
- 数学兴趣
- 教育影响

母亲：
- 早逝（1842）
- 对黎曼影响
```

**教育**：

```
1840-1846：中学
- 数学天赋
- 准备大学
```

```
学习：
- 神学（父亲期望）

- 转向数学
```

```
- 完成博士
- 高斯指导
- 准备就职
```

## 二、学术里程碑

### 2.1 博士论文（1851）

**题目**：

```
《复变函数论的基础》
(Grundlagen für eine allgemeine Theorie der Functionen einer veränderlichen complexen Grösse)

核心：
- 黎曼面
- 多值函数单值化
- 拓扑与分析统一
```

**意义**：

- 现代复分析基础
- 拓扑学Genesis
- 深远影响

---

### 2.2 就职演讲（1854）

**题目**：

```
《关于几何基础的假设》
(Über die Hypothesen, welche der Geometrie zu Grunde liegen)

核心：
- n维流形
- 度规张量
- 内蕴几何
```

**高斯的评价**：

```
"深刻而原创的思想"

意义：
- 高斯很少赞扬
- 认可突破
- 预示革命
```

---

### 2.3 Riemann-Roch定理（1857）

**工作**：

```
紧致黎曼面上全纯函数的维数

结果：
Riemann-Roch定理

意义：
- 分析-几何-拓扑统一
- 现代代数几何基础
```

---

### 2.4 ζ函数论文（1859）

**题目**：

```
《论小于给定值的素数个数》
(Über die Anzahl der Primzahlen unter einer gegebenen Grösse)

核心：
- ζ函数解析延拓
- 函数方程
- 黎曼假设
```

**意义**：

- 解析数论基础
- 未解决问题
- 现代研究
