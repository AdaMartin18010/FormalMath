# 核心著作解析：黎曼的三大经典

## 一、博士论文（1851）

### 1.1 题目与背景

**题目**：

```
《复变函数论的基础》
(Grundlagen für eine allgemeine Theorie der Functionen einer veränderlichen complexen Grösse)

1851年
哥廷根大学
高斯指导
```

**背景**：

```
问题：
- 多值函数
- 分支点
- 需要新方法

解决：
- 黎曼面
- 拓扑方法
- 统一框架
```

---

### 1.2 核心内容

**黎曼面**：

```
定义：
1维复流形

性质：
- 局部同胚于C
- 全纯结构
- 拓扑结构
```

**多值函数单值化**：

```
问题：
√z, log z

解决：
- 构造覆盖空间
- 多值 → 单值
- 拓扑结构
```

---

### 1.3 影响

**数学**：

- 现代复分析
- 拓扑学Genesis
- 代数几何

**物理**：

- 弦论
- 量子场论

```

- n维空间

```

核心：

## 三、ζ函数论文（1859）

### 3.1 背景

**题目**：

```
《论小于给定值的素数个数》
(Über die Anzahl der Primzahlen unter einer gegebenen Grösse)

1859年
柏林科学院
```

**问题**：

```
素数分布：
- π(x) = 小于x的素数个数
- 渐近行为？
- 误差估计？
```

---

### 3.2 核心内容

**ζ函数**：

```
定义：
ζ(s) = ∑_{n=1}^∞ 1/n^s（Re(s) > 1）

解析延拓：
- 全平面（除s=1）
- 函数方程
- 零点研究
```

**函数方程**：

```
ζ(s) = 2^s π^{s-1} sin(πs/2) Γ(1-s) ζ(1-s)

等价：
ξ(s) = ξ(1-s)
```

**黎曼假设**：

```
所有非平凡零点在Re(s) = 1/2

意义：
- 未解决问题
- 数论核心
- 现代研究
```

---

### 3.3 影响

**数论**：

- 解析数论
- 素数分布
- 现代研究

**应用**：

- 密码学
- 算法
- 物理
