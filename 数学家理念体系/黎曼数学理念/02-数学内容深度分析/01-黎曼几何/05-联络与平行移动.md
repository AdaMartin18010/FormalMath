## 一、联络

### 1.1 定义

**联络（Connection）**：

在流形M上，联络∇是一个算子：

```text
∇: Γ(TM) × Γ(TM) → Γ(TM)
(X, Y) → ∇_X Y

满足：
1. 线性：∇_{fX+gY} Z = f∇_X Z + g∇_Y Z
2. Leibniz法则：∇_X (fY) = X(f)Y + f∇_X Y
   其中f是函数，X, Y是向量场
```

**几何意义**：

联络提供了"在流形上对向量场求导"的方法，这是黎曼几何微分结构的基础。

**历史背景**：

联络的概念由Levi-Civita（1917）引入，用于研究黎曼流形上的平行移动。这是黎曼几何从"局部"到"整体"的关键工具。

---

### 1.2 Christoffel符号

**坐标表示**：

在局部坐标(x¹, ..., xⁿ)下，联络由Christoffel符号Γ^i_{jk}给出：

```text
∇_{∂/∂x^j} (∂/∂x^k) = Γ^i_{jk} (∂/∂x^i)

其中：
- Γ^i_{jk}：Christoffel符号（联络系数）
- 对称性：Γ^i_{jk} = Γ^i_{kj}（如果联络无挠）
```

**Levi-Civita联络的Christoffel符号**：

对于Levi-Civita联络，Christoffel符号由度规决定：

```text
Γ^i_{jk} = (1/2) g^{il} (∂g_{jl}/∂x^k + ∂g_{kl}/∂x^j - ∂g_{jk}/∂x^l)

其中：
- g_{ij}：度规张量
- g^{ij}：度规的逆
```

**坐标变换**：

虽然Christoffel符号不是张量，但在坐标变换下有特定的变换规律。

## 二、平行移动

### 2.1 定义

**平行移动**：

向量场V沿曲线γ(t)平行移动，如果：

```text
∇_{γ'(t)} V(γ(t)) = 0

对所有t成立
```

**等价表述**：

在局部坐标下，平行移动满足：

```text
dV^i/dt + Γ^i_{jk} (dx^j/dt) V^k = 0

其中γ(t) = (x¹(t), ..., xⁿ(t))
```

**几何意义**：

平行移动将向量"沿着曲线移动"，保持某种"平行性"。

---

### 2.2 基本性质

**保持长度和角度**：

对于Levi-Civita联络，平行移动保持：

- **长度**：|V|在平行移动下不变
- **角度**：两个平行移动的向量的夹角不变

**路径依赖性**：

平行移动是路径依赖的：从p到q沿不同路径的平行移动结果不同。这种依赖性由曲率测量。

**与测地线的关系**：

测地线的切向量沿自身平行移动：

```text
如果γ是测地线，则∇_{γ'} γ' = 0
```

这给出了测地线的另一种定义。

---

### 2.3 平行移动的几何意义

**与欧氏空间的类比**：

在欧氏空间中，平行移动是平凡的（向量不变）。在弯曲空间中，平行移动体现了空间的"弯曲"。

**曲率的体现**：

平行移动的路径依赖性体现了曲率：如果空间是平坦的，平行移动与路径无关。

**历史意义**：

Levi-Civita引入平行移动，解决了"如何在弯曲空间中定义平行"的问题。

## 三、协变导数

### 3.1 定义

**协变导数**：

```text
向量场V：

∇_X V = X^j (∂V^i/∂x^j + Γ^i_{jk} V^k) (∂/∂x^i)

性质：
- 张量
- 坐标无关
- 应用广泛
```

---

### 3.2 推广到张量场

**张量场的协变导数**：

联络可以推广到任意张量场。对(k,l)型张量T：

```text
∇_X T = 协变导数

性质：
- 张量（坐标无关）
- 满足Leibniz法则
- 与缩并交换
```

**外形式**：

对k形式ω，外协变导数定义为：

```text
d_∇ ω = 协变外微分

这是外微分的推广
```

**应用**：

张量场的协变导数用于：

- 研究张量场的几何性质
- 构造几何不变量
- 研究流形的几何

## 四、曲率

### 4.1 曲率张量

**定义**：

```text
曲率张量：

R(X,Y)Z = ∇_X ∇_Y Z - ∇_Y ∇_X Z - ∇_{[X,Y]} Z

坐标：
R^i_{jkl} = ∂Γ^i_{jl}/∂x^k - ∂Γ^i_{jk}/∂x^l
          + Γ^i_{mk} Γ^m_{jl} - Γ^i_{ml} Γ^m_{jk}
```

---

### 4.2 基本性质

**张量性质**：

曲率张量是(1,3)型张量，坐标无关。

**对称性**：

曲率张量满足：

```text
R(X,Y)Z = -R(Y,X)Z（反对称）
R(X,Y)Z + R(Y,Z)X + R(Z,X)Y = 0（第一Bianchi恒等式）
```

**Bianchi恒等式**：

```text
∇_X R(Y,Z)W + ∇_Y R(Z,X)W + ∇_Z R(X,Y)W = 0

（第二Bianchi恒等式）
```

**几何意义**：

曲率张量测量了平行移动的路径依赖性，是流形"弯曲"程度的度量。

## 五、Levi-Civita联络

### 5.1 定义

**Levi-Civita联络**：

在黎曼流形(M, g)上，存在唯一的联络∇满足：

1. **无挠（Torsion-free）**：

   ```text
   ∇_X Y - ∇_Y X = [X,Y]
   ```

   其中[X,Y]是Lie括号

2. **度规相容（Metric compatibility）**：

   ```text
   ∇_X g = 0
   或等价地：
   X(g(Y,Z)) = g(∇_X Y, Z) + g(Y, ∇_X Z)
   ```

**唯一性定理**：

Levi-Civita联络是唯一满足上述两个条件的联络。

**历史背景**：

Levi-Civita（1917）证明了这一定理，这是黎曼几何的基础定理。

---

### 5.2 唯一性的证明

**证明思路**：

通过Koszul公式可以唯一确定Levi-Civita联络：

```text
2g(∇_X Y, Z) = X(g(Y,Z)) + Y(g(Z,X)) - Z(g(X,Y))
             + g([X,Y], Z) - g([Y,Z], X) + g([Z,X], Y)
```

这个公式完全由度规和Lie括号决定，因此联络是唯一的。

**几何意义**：

Levi-Civita联络是"最自然"的联络，它：

- 与度规相容（保持长度和角度）
- 无挠（与Lie括号一致）
- 由度规唯一决定

---

### 5.2 应用

**应用**：

- 测地线
- 曲率
- 几何分析

## 六、与测地线和曲率的关系

### 6.1 测地线

**测地线方程**：

使用联络，测地线方程可以写成：

```text
∇_{γ'} γ' = 0

这是测地线的联络定义
```

**与平行移动的关系**：

测地线的切向量沿自身平行移动，这给出了测地线的几何意义。

---

### 6.2 曲率

**曲率与平行移动**：

曲率张量测量了平行移动的路径依赖性：

```text
如果沿闭曲线平行移动，向量会旋转，旋转角度由曲率决定
```

**平坦性**：

流形是平坦的（曲率为零）当且仅当平行移动与路径无关。

---

### 6.3 和乐群

**定义**：

在点p，和乐群Hol_p是沿闭曲线平行移动的变换群。

**性质**：

- 和乐群是Lie群
- 与曲率相关
- 用于研究流形的几何

   - **几何**：测地线、曲率、几何分析

## 九、数学公式总结

### 核心公式

1. **联络定义**：
   $$\nabla: \Gamma(TM) \times \Gamma(TM) \to \Gamma(TM), \quad \nabla_X Y \text{ 满足 Leibniz 法则}$$

2. **Christoffel符号**：
   $$\nabla_{\partial_j} \partial_k = \Gamma^i_{jk} \partial_i$$

3. **Levi-Civita联络**：
   $$\Gamma^i_{jk} = \frac{1}{2} g^{il} \left(\frac{\partial g_{jl}}{\partial x^k} + \frac{\partial g_{kl}}{\partial x^j} - \frac{\partial g_{jk}}{\partial x^l}\right)$$

4. **协变导数**：
   $$\nabla_X Y = X^i \left(\frac{\partial Y^j}{\partial x^i} + \Gamma^j_{ik} Y^k\right) \partial_j$$

5. **平行移动方程**：
   $$\frac{dV^i}{dt} + \Gamma^i_{jk} \frac{dx^j}{dt} V^k = 0$$

6. **曲率张量**：
   $$R^i_{jkl} = \frac{\partial \Gamma^i_{jl}}{\partial x^k} - \frac{\partial \Gamma^i_{jk}}{\partial x^l} + \Gamma^i_{mk} \Gamma^m_{jl} - \Gamma^i_{ml} \Gamma^m_{jk}$$

7. **测地线方程**：
   $$\frac{d^2 x^i}{dt^2} + \Gamma^i_{jk} \frac{dx^j}{dt} \frac{dx^k}{dt} = 0$$

8. **和乐群**：
   $$\text{Hol}_p(M) = \{\text{平行移动沿闭曲线}\} \subset \text{GL}(T_p M)$$

9. **联络与曲率关系**：
   $$[\nabla_X, \nabla_Y] Z - \nabla_{[X,Y]} Z = R(X, Y) Z$$

10. **联络与度规相容**：
    $$\nabla g = 0 \iff \text{Levi-Civita 联络}$$
