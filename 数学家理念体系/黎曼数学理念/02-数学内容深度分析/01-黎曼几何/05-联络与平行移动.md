# 联络与平行移动：黎曼几何的微分结构

## 一、联络

### 1.1 定义

**联络**：

```
流形M上的联络∇：

∇: Γ(TM) × Γ(TM) → Γ(TM)

性质：
- 线性
- Leibniz法则
- 坐标表示：Christoffel符号
```

---

### 1.2 Christoffel符号

**坐标表示**：

```
∇_{∂/∂x^j} (∂/∂x^k) = Γ^i_{jk} (∂/∂x^i)

其中：
Γ^i_{jk}：Christoffel符号

公式：
Γ^i_{jk} = (1/2) g^{il} (∂g_{jl}/∂x^k + ∂g_{kl}/∂x^j - ∂g_{jk}/∂x^l)
```

---

## 二、平行移动

### 2.1 定义

**平行移动**：

```
向量场V沿曲线γ：

∇_{γ'} V = 0

性质：
- 保持长度
- 保持角度
- 路径依赖
```

---

### 2.2 性质

**性质**：

- 路径依赖
- 曲率相关
- 几何意义

---

## 三、协变导数

### 3.1 定义

**协变导数**：

```
向量场V：

∇_X V = X^j (∂V^i/∂x^j + Γ^i_{jk} V^k) (∂/∂x^i)

性质：
- 张量
- 坐标无关
- 应用广泛
```

---

### 3.2 推广

**张量场**：

```
(k,l)型张量T：

∇_X T = 协变导数

性质：
- 张量
- 坐标无关
```

---

## 四、曲率

### 4.1 曲率张量

**定义**：

```
曲率张量：

R(X,Y)Z = ∇_X ∇_Y Z - ∇_Y ∇_X Z - ∇_{[X,Y]} Z

坐标：
R^i_{jkl} = ∂Γ^i_{jl}/∂x^k - ∂Γ^i_{jk}/∂x^l
          + Γ^i_{mk} Γ^m_{jl} - Γ^i_{ml} Γ^m_{jk}
```

---

### 4.2 性质

**性质**：

- 张量
- 对称性
- Bianchi恒等式

---

## 五、Levi-Civita联络

### 5.1 定义

**Levi-Civita联络**：

```
黎曼流形(M, g)：

唯一联络∇满足：
1. 无挠：∇_X Y - ∇_Y X = [X,Y]
2. 度规相容：∇g = 0

性质：
- 唯一性
- 自然
- 应用广泛
```

---

### 5.2 应用

**应用**：

- 测地线
- 曲率
- 几何分析
