# 子流形理论：嵌入与浸入

## 一、嵌入与浸入

### 1.1 定义

**嵌入**：

```
映射f: M → N：

嵌入条件：
- 单射
- 浸入
- 同胚到像

性质：
- 子流形
- 应用广泛
```

---

### 1.2 浸入

**浸入**：

```
映射f: M → N：

浸入条件：
- 切映射单射
- 局部嵌入

性质：
- 自交可能
- 应用广泛
```

## 二、Nash嵌入定理

### 2.1 定理

**Nash嵌入**：

```
n维黎曼流形M：

存在等距嵌入：
M → R^N（N充分大）

性质：
- 等距
- 光滑
- 应用
```
