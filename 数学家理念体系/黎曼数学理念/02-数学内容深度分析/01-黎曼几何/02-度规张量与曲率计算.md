# 度规张量与曲率计算：黎曼几何的技术细节

## 一、度规张量

### 1.1 局部表示

**坐标表示**：

```
n维流形M，局部坐标(x¹, ..., xⁿ)

度规：
ds² = ∑_{i,j=1}^n g_{ij}(x) dx^i dx^j

其中：
g_{ij}(x) = g(∂/∂x^i, ∂/∂x^j)

性质：
- 对称：g_{ij} = g_{ji}
- 正定：对任意v≠0，∑g_{ij}v^iv^j > 0
- 光滑
```

**矩阵形式**：

```text
g = (g_{ij})_{n×n}

行列式：
g = det(g_{ij})

逆矩阵：
g^{-1} = (g^{ij})
```

---

### 1.2 坐标变换

**变换规则**：

```text
坐标变换：x → x'

g'_{kl} = (∂x^i/∂x'^k)(∂x^j/∂x'^l) g_{ij}

张量性质：
- 协变2阶张量
- 坐标无关
```

---

## 二、Christoffel符号

### 2.1 定义

**公式**：

```text
Γ^k_{ij} = (1/2) g^{kl} (∂g_{il}/∂x^j + ∂g_{jl}/∂x^i - ∂g_{ij}/∂x^l)

性质：
- 对称：Γ^k_{ij} = Γ^k_{ji}
- 坐标依赖
- 不是张量
```

**物理意义**：

```text
Γ = 联络系数

作用：
- 定义平行移动
- 定义协变导数
- 定义曲率
```

---

### 2.2 计算例子

**球面坐标**：

```text
2维球面S²：
ds² = dθ² + sin²θ dφ²

度规：
g_{θθ} = 1
g_{φφ} = sin²θ
g_{θφ} = 0

Christoffel符号：
Γ^θ_{φφ} = -sinθ cosθ
Γ^φ_{θφ} = cotθ
其他为0
```

---

## 三、协变导数

### 3.1 向量场

**定义**：

```text
向量场V = V^i ∂/∂x^i

协变导数：
∇_j V^i = ∂V^i/∂x^j + Γ^i_{jk} V^k

性质：
- 张量
- 满足Leibniz法则
```

---

### 3.2 张量场

**一般张量**：

```text
(k,l)型张量T：

∇_m T^{i₁...iₖ}_{j₁...jₗ} =
∂T^{i₁...iₖ}_{j₁...jₗ}/∂x^m
+ ∑_{r=1}^k Γ^{i_r}_{mn} T^{...n...}
- ∑_{s=1}^l Γ^n_{mj_s} T^{...n...}
```

---

## 四、曲率张量

### 4.1 黎曼曲率张量

**定义**：

```text
R^i_{jkl} = ∂Γ^i_{jl}/∂x^k - ∂Γ^i_{jk}/∂x^l
          + Γ^i_{mk} Γ^m_{jl} - Γ^i_{ml} Γ^m_{jk}

协变形式：
R_{ijkl} = g_{im} R^m_{jkl}
```

**对称性**：

```text
- R_{ijkl} = -R_{jikl}
- R_{ijkl} = -R_{ijlk}
- R_{ijkl} = R_{klij}
- Bianchi恒等式：
  R_{ijkl;m} + R_{ijlm;k} + R_{ijmk;l} = 0
```

---

### 4.2 计算例子

**2维曲面**：

```text
度规：
ds² = E du² + 2F du dv + G dv²

Gauss曲率：
K = (1/(EG-F²)) [∂/∂v(Γ¹₁₂/E) - ∂/∂u(Γ¹₂₂/E) + ...]

简化：
K = (1/(EG-F²)) [∂²F/∂u∂v - (1/2)∂²E/∂v² - (1/2)∂²G/∂u² + ...]
```

---

## 五、Ricci张量与标量曲率

### 5.1 Ricci张量

**定义**：

```text
R_{ij} = R^k_{ikj} = g^{kl} R_{kilj}

性质：
- 对称：R_{ij} = R_{ji}
- 2阶协变张量
- 迹：R = g^{ij} R_{ij}（标量曲率）
```

**计算**：

```text
R_{ij} = ∂Γ^k_{ik}/∂x^j - ∂Γ^k_{ij}/∂x^k
       + Γ^k_{ik} Γ^l_{lj} - Γ^k_{ij} Γ^l_{lk}
```

---

### 5.2 Einstein张量

**定义**：

```text
G_{ij} = R_{ij} - (1/2) R g_{ij}

性质：
- 对称
- 散度为零：G^{ij}_{;j} = 0
- 物理意义：引力场
```

---

## 六、测地线方程

### 6.1 变分原理

**作用量**：

```text
S = ∫ √(g_{ij} dx^i dx^j)

Euler-Lagrange：
d/dt(∂L/∂ẋ^i) - ∂L/∂x^i = 0

其中：
L = √(g_{ij} ẋ^i ẋ^j)
```

---

### 6.2 测地线方程

**标准形式**：

```text
d²x^i/dt² + Γ^i_{jk} (dx^j/dt)(dx^k/dt) = 0

性质：
- 二阶ODE
- 局部存在唯一
- 最短路径
```

**例子：球面**：

```text
球面测地线：
- 大圆
- 沿经线或纬线
- 一般：球面上的"直线"
```

---

## 七、特殊度规

### 7.1 常曲率空间

**定义**：

```text
常曲率K：
R_{ijkl} = K (g_{ik} g_{jl} - g_{il} g_{jk})

Ricci：
R_{ij} = (n-1)K g_{ij}

标量：
R = n(n-1)K
```

**例子**：

```text
- K > 0：球面Sⁿ
- K = 0：欧氏空间Rⁿ
- K < 0：双曲空间Hⁿ
```

---

### 7.2 Einstein流形

**定义**：

```
R_{ij} = λ g_{ij}

其中：
λ：常数（Einstein常数）

例子：
- 球面
- 双曲空间
- Calabi-Yau流形
```

## 八、应用

### 8.1 广义相对论

**Einstein场方程**：

```
G_{ij} = 8πG T_{ij}

其中：
- G_{ij}：Einstein张量
- T_{ij}：能量-动量张量
- G：引力常数

解：
- Schwarzschild（黑洞）
- Friedmann（宇宙学）
- Kerr（旋转黑洞）
```
