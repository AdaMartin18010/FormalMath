# 测地线与变分原理：黎曼几何的运动学

## 一、测地线的定义

### 1.1 最短路径

**定义**：

```
测地线 = 流形上的"直线"

性质：
- 局部最短路径
- 曲率为零
- 自平行
```

**物理意义**：

```
- 自由粒子轨迹
- 无外力运动
- 时空中的"直线"
```

---

### 1.2 变分原理

**作用量**：

```
S = ∫ √(g_{ij} dx^i dx^j)

其中：
- g_{ij}：度规张量
- dx^i：坐标微分

变分：
δS = 0
```

---

## 二、测地线方程

### 2.1 Euler-Lagrange方程

**Lagrangian**：

```
L = √(g_{ij} ẋ^i ẋ^j)

其中：
ẋ^i = dx^i/dt
```

**Euler-Lagrange**：

```
d/dt(∂L/∂ẋ^i) - ∂L/∂x^i = 0
```

---

### 2.2 标准形式

**测地线方程**：

```
d²x^i/dt² + Γ^i_{jk} (dx^j/dt)(dx^k/dt) = 0

其中：
- Γ^i_{jk}：Christoffel符号
- t：仿射参数
```

**性质**：

- 二阶ODE
- 局部存在唯一
- 坐标无关

---

## 三、仿射参数

### 3.1 定义

**仿射参数**：

```
参数t使得：
d²x^i/dt² + Γ^i_{jk} (dx^j/dt)(dx^k/dt) = 0

性质：
- 线性变换不变
- 自然参数
```

---

### 3.2 弧长参数

**弧长**：

```
s = ∫ √(g_{ij} dx^i dx^j)

性质：
- 自然参数
- ds² = g_{ij} dx^i dx^j
```

---

## 四、例子

### 4.1 球面

**2维球面S²**：

```
度规：
ds² = dθ² + sin²θ dφ²

测地线：
- 大圆
- 沿经线或纬线
- 一般：球面上的"直线"
```

**方程**：

```
d²θ/dt² - sinθ cosθ (dφ/dt)² = 0
d²φ/dt² + 2cotθ (dθ/dt)(dφ/dt) = 0
```

---

### 4.2 双曲平面

**Poincaré模型**：

```
度规：
ds² = 4(dx²+dy²)/(1-x²-y²)²

测地线：
- 垂直于边界的圆弧
- 通过中心的直线
```

---

## 五、变分方法

### 5.1 能量泛函

**能量**：

```
E = (1/2) ∫ g_{ij} ẋ^i ẋ^j dt

变分：
δE = 0
```

**优势**：

- 二次型
- 便于计算
- 等价于长度

---

### 5.2 最小作用量

**原理**：

```
作用量：
S = ∫ L dt

最小化：
δS = 0

结果：
测地线方程
```

## 六、应用

### 6.1 广义相对论

**自由粒子**：

```
测地线 = 自由粒子轨迹

Einstein：
- 引力 = 时空曲率
- 粒子 = 沿测地线
- 无外力
```

---

### 6.2 光线

**零测地线**：

```
ds² = 0

方程：
d²x^μ/dλ² + Γ^μ_{νρ} (dx^ν/dλ)(dx^ρ/dλ) = 0

应用：
- 光线偏折
- 引力透镜
- 时间延迟
```
