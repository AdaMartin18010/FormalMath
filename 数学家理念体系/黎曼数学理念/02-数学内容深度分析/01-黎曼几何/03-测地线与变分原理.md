## 一、测地线的定义

### 1.1 最短路径

**定义**：

```text
测地线 = 流形上的"直线"

性质：
- 局部最短路径
- 曲率为零
- 自平行
```

**数学表述**：

在黎曼流形(M, g)上，测地线γ: I → M是满足以下等价条件之一的曲线：

1. **变分定义**：对任意固定端点的变分，长度泛函L(γ)的临界点
2. **微分方程定义**：满足测地线方程
3. **几何定义**：切向量场沿曲线平行移动

**历史背景**：

黎曼在1854年的就职演讲中，虽然没有明确使用"测地线"这一术语，但已经隐含了测地线的概念。他认识到，在一般流形上，最短路径的概念需要由度规张量来定义，而不是预先给定的欧氏几何。

**物理意义**：

```text
- 自由粒子轨迹
- 无外力运动
- 时空中的"直线"
```

在广义相对论中，Einstein（1915）将测地线提升为基本物理原理：自由粒子（包括光子）在时空中沿测地线运动，这体现了"引力即几何"的核心思想。

---

### 1.2 变分原理

**长度泛函**：

给定曲线γ: [a,b] → M，其长度为：

```text
L(γ) = ∫_a^b √(g_{ij}(γ(t)) ẋ^i(t) ẋ^j(t)) dt

其中：
- g_{ij}：度规张量
- ẋ^i = dx^i/dt：切向量分量
```

**变分问题**：

固定端点p = γ(a)和q = γ(b)，考虑变分γ_ε(t) = γ(t) + εη(t)，其中η(a) = η(b) = 0。

测地线是使长度泛函L(γ)达到临界值的曲线，即：

```text
δL = d/dε|_{ε=0} L(γ_ε) = 0
```

**历史意义**：

变分原理将几何问题转化为分析问题，这是黎曼几何与经典变分法结合的典型例子。这一方法后来在物理中得到了广泛应用，特别是在Lagrangian力学和广义相对论中。

## 二、测地线方程

### 2.1 Euler-Lagrange方程

**Lagrangian**：

长度泛函的Lagrangian为：

```text
L(x, ẋ) = √(g_{ij}(x) ẋ^i ẋ^j)

其中：
- x = (x^1, ..., x^n)：坐标
- ẋ^i = dx^i/dt：速度分量
```

**变分计算**：

对长度泛函L(γ) = ∫ L(x, ẋ) dt进行变分：

```text
δL = ∫ [∂L/∂x^i δx^i + ∂L/∂ẋ^i δẋ^i] dt
   = ∫ [∂L/∂x^i - d/dt(∂L/∂ẋ^i)] δx^i dt
```

**Euler-Lagrange方程**：

由于δx^i在端点为零，临界条件给出：

```text
d/dt(∂L/∂ẋ^i) - ∂L/∂x^i = 0
```

**计算细节**：

对于L = √(g_{jk} ẋ^j ẋ^k)，有：

```text
∂L/∂ẋ^i = (1/L) g_{ij} ẋ^j
∂L/∂x^i = (1/2L) (∂g_{jk}/∂x^i) ẋ^j ẋ^k
```

直接计算较为复杂，因此通常使用能量泛函方法。

---

### 2.2 标准形式

**测地线方程**：

通过Euler-Lagrange方程或直接变分，得到测地线方程的标准形式：

```text
d²x^i/dt² + Γ^i_{jk} (dx^j/dt)(dx^k/dt) = 0

其中：
- Γ^i_{jk}：Christoffel符号（Levi-Civita联络系数）
- t：仿射参数
```

**Christoffel符号**：

```text
Γ^i_{jk} = (1/2) g^{il} (∂g_{jl}/∂x^k + ∂g_{kl}/∂x^j - ∂g_{jk}/∂x^l)
```

这是Levi-Civita联络在坐标基下的表示，完全由度规张量决定。

**推导要点**：

从能量泛函E = (1/2)∫ g_{ij} ẋ^i ẋ^j dt的变分，得到：

```text
δE = ∫ [g_{ij} ẍ^j + (∂g_{ij}/∂x^k) ẋ^k ẋ^j - (1/2)(∂g_{jk}/∂x^i) ẋ^j ẋ^k] δx^i dt
```

重新整理并利用度规的对称性，得到标准形式。

**性质**：

- **二阶ODE系统**：n个二阶常微分方程
- **局部存在唯一性**：给定初始位置x^i(0)和初始速度ẋ^i(0)，存在唯一的局部解
- **坐标无关性**：虽然方程形式依赖坐标，但测地线的几何性质是内蕴的
- **仿射参数化**：方程在仿射参数变换t → at + b下不变

## 三、仿射参数

### 3.1 定义

**仿射参数**：

参数t称为仿射参数，如果测地线方程在该参数下具有标准形式：

```text
d²x^i/dt² + Γ^i_{jk} (dx^j/dt)(dx^k/dt) = 0
```

**关键性质**：

1. **线性变换不变性**：如果t是仿射参数，则t' = at + b（a ≠ 0）也是仿射参数
2. **速度模长**：对于类时或类空测地线，|ẋ|² = g_{ij} ẋ^i ẋ^j在仿射参数下为常数
3. **自然性**：仿射参数是"最自然"的参数化，使得方程形式最简单

**物理意义**：

在广义相对论中：

- **类时测地线**（ds² > 0）：质点的世界线，仿射参数正比于固有时
- **类空测地线**（ds² < 0）：空间曲线
- **零测地线**（ds² = 0）：光线的世界线，仿射参数是任意参数

---

### 3.2 弧长参数

**弧长**：

```text
s = ∫ √(g_{ij} dx^i dx^j)

性质：
- 自然参数
- ds² = g_{ij} dx^i dx^j
```

## 四、例子

### 4.1 球面

**2维球面S²**：

在标准球坐标(θ, φ)下，度规为：

```text
ds² = dθ² + sin²θ dφ²

其中：
- θ ∈ [0, π]：极角（从北极）
- φ ∈ [0, 2π)：方位角
```

**Christoffel符号**：

```text
Γ^θ_{φφ} = -sinθ cosθ
Γ^φ_{θφ} = Γ^φ_{φθ} = cotθ
其他分量为零
```

**测地线方程**：

```text
d²θ/dt² - sinθ cosθ (dφ/dt)² = 0
d²φ/dt² + 2cotθ (dθ/dt)(dφ/dt) = 0
```

**解的几何意义**：

测地线是**大圆**（通过球心的平面与球面的交线）：

- **经线**（φ = 常数）：从北极到南极的大圆
- **赤道**（θ = π/2）：特殊的大圆
- **一般大圆**：可以通过坐标旋转得到

**验证**：

对于经线（φ = φ₀常数），方程简化为d²θ/dt² = 0，解为θ = at + b，确实是测地线。

对于赤道（θ = π/2），方程给出d²φ/dt² = 0，也是测地线。

---

### 4.2 双曲平面

**Poincaré圆盘模型**：

在单位圆盘D = {(x,y) : x² + y² < 1}上，双曲度规为：

```text
ds² = 4(dx²+dy²)/(1-x²-y²)²

其中：
- (x,y)：欧氏坐标
- 边界|x²+y²| = 1：无穷远点
```

**测地线的几何特征**：

在Poincaré模型中，测地线是：

1. **垂直于边界的圆弧**：圆心在单位圆周上
2. **通过原点的直线**：特殊的测地线（半径）

**计算示例**：

对于通过原点的直线（如x轴），设y = 0，度规简化为：

```text
ds² = 4dx²/(1-x²)²
```

测地线方程给出d²x/dt² = 0（在适当的参数化下），验证了直线是测地线。

**与欧氏几何的对比**：

- **欧氏平面**：测地线是直线
- **双曲平面**：测地线是"弯曲的"，但在双曲几何意义下是"直线"
- 这体现了黎曼几何的核心思想：几何由度规决定，而非预先给定

## 五、变分方法

### 5.1 能量泛函

**能量泛函**：

与长度泛函L(γ)不同，能量泛函定义为：

```text
E(γ) = (1/2) ∫_a^b g_{ij}(γ(t)) ẋ^i(t) ẋ^j(t) dt
```

**变分原理**：

测地线也是能量泛函的临界点：

```text
δE = 0
```

**为什么使用能量泛函？**

1. **数学优势**：
   - 能量泛函是二次型，变分计算更简单
   - 避免了长度泛函中平方根带来的技术困难
   - Euler-Lagrange方程形式更简洁

2. **等价性**：
   - 对于固定端点的变分，能量和长度的临界点相同
   - 但能量泛函的临界曲线自动以常数速度参数化

3. **物理意义**：
   - 在力学中，能量对应动能
   - 在广义相对论中，能量泛函有明确的物理解释

**变分计算**：

```text
δE = (1/2) ∫ [∂g_{ij}/∂x^k ẋ^i ẋ^j δx^k + 2g_{ij} ẋ^i δẋ^j] dt
   = ∫ [g_{ij} ẍ^j + (∂g_{ij}/∂x^k - (1/2)∂g_{jk}/∂x^i) ẋ^i ẋ^k] δx^j dt
```

利用度规的对称性和Christoffel符号的定义，得到测地线方程。

---

### 5.2 最小作用量

**原理**：

```text
作用量：
S = ∫ L dt

最小化：
δS = 0

结果：
测地线方程
```

## 七、存在性与唯一性

### 7.1 局部存在

**定理**：

```text
给定初始条件：
- x^i(0) = x₀^i
- ẋ^i(0) = v₀^i

存在唯一测地线：
- 局部存在
- 唯一解
```

---

### 7.2 全局性质

**共轭点**：

点p, q ∈ M称为沿测地线γ共轭，如果存在沿γ的Jacobi场（测地线变分场）在p和q处为零。

**几何意义**：

- 共轭点标志着测地线不再是局部最短路径
- 在共轭点之后，存在"更短"的路径（虽然可能不光滑）

**割迹（Cut Locus）**：

从点p出发，测地线γ(t)的割点是使得：

- γ在[0,t]上是最短路径
- 但γ在[0,t+ε]上不是最短路径（对任意ε > 0）

所有割点的集合称为p的割迹。

**例子**：

- **球面**：对极点是共轭点，也是割点
- **双曲平面**：无共轭点（负曲率）
- **欧氏空间**：无共轭点，无割点

**应用**：

- **比较定理**：利用共轭点出现的时间比较不同曲率流形的几何
- **体积估计**：割迹的几何性质用于估计流形的体积
- **几何分析**：共轭点和割迹是理解流形全局几何的关键工具

## 八、测地流

### 8.1 定义

**测地流**：

在切丛TM上，测地流φ_t: TM → TM定义为：

```text
φ_t(x, v) = (γ(t), γ'(t))

其中：
- (x, v) ∈ TM：初始位置和速度
- γ：从x出发、初始速度为v的测地线
```

**几何意义**：

测地流是切丛上的动力学系统，描述了"测地线如何演化"。

**关键性质**：

1. **保持能量**：E(φ_t(x,v)) = (1/2)|v|² = 常数
2. **Hamiltonian结构**：测地流是切丛上自然Hamiltonian系统的流
3. **测地线对应**：每条测地线对应测地流的一条轨道

**Liouville定理**：

测地流保持切丛上的自然体积形式（Liouville测度），这是辛几何的结果。

---

### 8.2 应用

**几何**：

- 流形分类
- 曲率估计
- 几何分析

**物理**：

- 动力学
- 统计力学

## 九、与黎曼几何其他部分的联系

### 9.1 与曲率的关系

**Jacobi方程**：

测地线变分满足Jacobi方程，该方程直接涉及曲率张量：

```text
∇²_γ' J + R(γ', J)γ' = 0

其中：
- J：Jacobi场（测地线变分场）
- R：曲率张量
```

这揭示了测地线的聚焦性质与曲率的直接关系。

### 9.2 与联络的关系

**平行移动**：

测地线的切向量沿自身平行移动：

```text
∇_{γ'} γ' = 0
```

这给出了测地线的另一种等价定义，体现了联络在黎曼几何中的核心地位。

### 9.3 与比较几何的关系

**Rauch比较定理**：

通过比较不同曲率流形上的测地线行为，可以得到流形的几何性质。这是比较几何学的核心方法。

## 十、总结

### 测地线的历史地位

**发展历程**：

- **1854年**：黎曼在就职演讲中隐含了测地线的概念，虽然未明确使用这一术语
- **1917年**：Levi-Civita引入平行移动，给出了测地线的现代定义
- **1915年**：Einstein将测地线应用于广义相对论，建立了"引力即几何"的物理框架
- **现代**：测地线成为几何分析、动力系统和数学物理的核心工具

**核心贡献**：

测地线是**黎曼几何的运动学基础**，它：

1. 连接了变分法与微分几何
2. 提供了几何与物理的桥梁
3. 是现代几何分析的基本研究对象
4. 体现了黎曼"内蕴几何"的核心思想

**现代意义**：

测地线理论不仅解决了"流形上的直线是什么"这一基本问题，更开启了现代几何分析的大门。从Perelman的Ricci流到现代动力系统理论，测地线都扮演着关键角色。

## 十一、数学公式总结

### 核心公式

1. **长度泛函**：
   $$L(\gamma) = \int_a^b \sqrt{g_{ij}(\gamma(t)) \dot{\gamma}^i(t) \dot{\gamma}^j(t)} dt$$

2. **能量泛函**：
   $$E(\gamma) = \frac{1}{2}\int_a^b g_{ij}(\gamma(t)) \dot{\gamma}^i(t) \dot{\gamma}^j(t) dt$$

3. **测地线方程**：
   $$\frac{d^2 x^i}{dt^2} + \sum_{j,k} \Gamma_{jk}^i \frac{dx^j}{dt} \frac{dx^k}{dt} = 0$$

4. **Euler-Lagrange方程**：
   $$\frac{\partial L}{\partial x^i} - \frac{d}{dt}\\left(\frac{\partial L}{\partial \dot{x}^i}\right) = 0$$

5. **弧长参数**：
   $$s(t) = \int_0^t \sqrt{g_{ij}(\gamma(\tau)) \dot{\gamma}^i(\tau) \dot{\gamma}^j(\tau)} d\tau$$

6. **仿射参数**：
   $$\frac{d^2 x^i}{d\lambda^2} + \Gamma_{jk}^i \frac{dx^j}{d\lambda} \frac{dx^k}{d\lambda} = 0$$

7. **测地线存在性**：
   $$\text{对任意 } p \in M, v \in T_pM, \text{ 存在唯一测地线 } \gamma \text{ 满足 } \gamma(0) = p, \dot{\gamma}(0) = v$$

8. **指数映射**：
   $$\exp_p(v) = \gamma(1), \quad \text{其中 } \gamma \text{ 是满足 } \gamma(0) = p, \dot{\gamma}(0) = v \text{ 的测地线}$$

9. **Jacobi方程**：
   $$\frac{D^2 J}{dt^2} + R(\dot{\gamma}, J)\dot{\gamma} = 0$$

10. **比较定理**：
    $$\text{若 } K \leq 0, \text{ 则测地线不聚焦}$$
