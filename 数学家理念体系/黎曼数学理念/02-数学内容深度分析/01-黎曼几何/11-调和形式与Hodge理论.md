# 调和形式与Hodge理论：黎曼流形的分析工具

## 一、调和形式

### 1.1 定义

**调和形式**：

```
黎曼流形M上的k形式ω：

调和条件：
Δω = 0

其中：
Δ = dd* + d*d（Hodge-Laplace算子）

性质：
- 闭：dω = 0
- 上闭：d*ω = 0
- 应用广泛
```

---

### 1.2 性质

**性质**：

- 能量最小
- 唯一性
- 应用广泛

---

## 二、Hodge分解

### 2.1 定理

**Hodge分解定理**：

```
Ω^k(M) = H^k(M) ⊕ dΩ^{k-1}(M) ⊕ d*Ω^{k+1}(M)

其中：
- H^k(M)：调和k形式
- dΩ^{k-1}：恰当形式
- d*Ω^{k+1}：余恰当形式

性质：
- 正交分解
- 唯一性
- 应用广泛
```

---

### 2.2 应用

**应用**：

- 上同调计算
- 几何分析
- 现代发展

---

## 三、Hodge理论

### 3.1 基本定理

**Hodge定理**：

```
紧致黎曼流形M：

H^k(M, ℝ) ≅ H^k(M)

其中：
- H^k(M, ℝ)：de Rham上同调
- H^k(M)：调和k形式空间

性质：
- 有限维
- 应用广泛
```

---

### 3.2 应用

**应用**：

- 上同调计算
- 几何分析
- 现代发展

---

## 四、Kähler流形

### 4.1 Hodge分解

**Kähler流形**：

```
Kähler流形M：

Hodge分解：
H^k(M, ℂ) = ⊕_{p+q=k} H^{p,q}(M)

其中：
- H^{p,q}：调和(p,q)形式
- 应用广泛
```

---

### 4.2 应用

**应用**：

- 代数几何
- 分类理论
