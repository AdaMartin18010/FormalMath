# 黎曼几何基础理论：从度量到曲率

## 概述

黎曼几何是19世纪数学最深刻的发明之一，由Bernhard Riemann在1854年就职演讲《论几何学基础中的假设》中首次系统阐述。这一理论彻底改变了人们对空间本质的理解，从欧几里得空间的刚性框架中解放出来，建立了内蕴几何的普遍理论。黎曼几何不仅是现代微分几何的核心，更是广义相对论的数学基础。

**MSC分类**: 53B20 (Local Riemannian geometry), 53C20 (Global Riemannian geometry)  
**关键词**: 黎曼度量, Levi-Civita联络, 曲率张量, 测地线, 截面曲率  
**参考文献**: do Carmo "Riemannian Geometry", Petersen "Riemannian Geometry"

## 一、历史背景与思想起源

### 1.1 高斯的启发

1827年，Carl Friedrich Gauss发表《关于曲面的一般研究》，证明了著名的**绝妙定理**（Theorema Egregium）：曲面的Gauss曲率是内蕴的，不依赖于曲面在三维欧氏空间中的嵌入方式。这一发现深刻影响了年轻的黎曼。

Gauss的工作表明，曲面的几何性质可以分为两类：
- **外蕴性质**：依赖于嵌入空间（如第二基本形式）
- **内蕴性质**：仅由曲面本身决定（如Gauss曲率、测地线）

黎曼将这一思想推广到任意维度，建立了**n维流形的内蕴几何理论**。

### 1.2 黎曼的革命性论文

1854年6月10日，黎曼在哥廷根大学发表就职演讲《论几何学基础中的假设》(Über die Hypothesen, welche der Geometrie zu Grunde liegen)。在这篇演讲中，黎曼提出了 radically new ideas：

1. **空间是n维流形**：突破了3维空间的限制
2. **度量决定几何**：通过度量张量定义距离和角度
3. **曲率测量偏离欧氏性的程度**：建立了曲率的一般理论
4. **空间可能是弯曲的**：为后来Einstein的广义相对论奠定数学基础

黎曼的演讲仅有约30页，但其影响深远，被誉为现代微分几何的奠基之作。

## 二、黎曼度量的严格定义

### 2.1 光滑流形上的度量

**定义 2.1（黎曼度量）**：设 $M$ 是一个 $n$ 维光滑流形。$M$ 上的**黎曼度量** $g$ 是一个 $(0,2)$ 型光滑张量场，满足：

1. **对称性**：$g(X, Y) = g(Y, X)$，对所有 $X, Y \in \mathfrak{X}(M)$
2. **正定性**：$g(X, X) \geq 0$，且 $g(X, X) = 0 \iff X = 0$
3. **光滑性**：在局部坐标下，分量 $g_{ij}$ 是光滑函数

装备了黎曼度量的流形 $(M, g)$ 称为**黎曼流形**。

### 2.2 局部坐标表示

在局部坐标 $(U, (x^1, \ldots, x^n))$ 下，黎曼度量可表示为：

$$g = \sum_{i,j=1}^n g_{ij}(x) \, dx^i \otimes dx^j$$

其中 $g_{ij}(x) = g\left(\frac{\partial}{\partial x^i}, \frac{\partial}{\partial x^j}\right)$ 构成对称正定矩阵。

**线元形式**：
$$ds^2 = \sum_{i,j=1}^n g_{ij}(x) \, dx^i dx^j$$

**逆度量**：记 $(g^{ij})$ 为 $(g_{ij})$ 的逆矩阵，则 $g^{ij}g_{jk} = \delta^i_k$。

### 2.3 度量的存在性

**定理 2.1（黎曼度量的存在性）**：任何光滑流形 $M$ 上都可以定义黎曼度量。

**证明思路**：
1. 利用单位分解将局部度量拼合为全局度量
2. 由于正定性是凸条件，凸组合保持正定性
3. 因此存在性由流形的仿紧性保证

**注**：该定理表明黎曼几何的适用范围极广，任何光滑流形都可以作为黎曼流形研究。

## 三、Levi-Civita联络的存在唯一性

### 3.1 仿射联络的基本概念

**定义 3.1（仿射联络）**：流形 $M$ 上的**仿射联络**是一个映射 $\nabla: \mathfrak{X}(M) \times \mathfrak{X}(M) \to \mathfrak{X}(M)$，记作 $\nabla_X Y$，满足：

1. **$C^\infty(M)$-线性于 $X$**：$\nabla_{fX+hZ}Y = f\nabla_X Y + h\nabla_Z Y$
2. **线性于 $Y$**：$\nabla_X(aY + bZ) = a\nabla_X Y + b\nabla_X Z$
3. **Leibniz法则**：$\nabla_X(fY) = X(f)Y + f\nabla_X Y$

### 3.2 Levi-Civita联络的刻画

**定理 3.1（Levi-Civita，1917）**：设 $(M, g)$ 是黎曼流形，则存在唯一的仿射联络 $\nabla$，满足：

1. **无挠性**（Torsion-free）：$\nabla_X Y - \nabla_Y X = [X, Y]$
2. **度量相容性**（Metric compatible）：$X(g(Y,Z)) = g(\nabla_X Y, Z) + g(Y, \nabla_X Z)$

该联络称为**Levi-Civita联络**或**黎曼联络**。

### 3.3 Christoffel符号的显式公式

在局部坐标下，Levi-Civita联络由**Christoffel符号** $\Gamma^k_{ij}$ 表示：

$$\nabla_{\partial_i} \partial_j = \sum_{k=1}^n \Gamma^k_{ij} \frac{\partial}{\partial x^k}$$

**定理 3.2（Christoffel符号公式）**：
$$\Gamma^k_{ij} = \frac{1}{2} \sum_{l=1}^n g^{kl} \left(\frac{\partial g_{il}}{\partial x^j} + \frac{\partial g_{jl}}{\partial x^i} - \frac{\partial g_{ij}}{\partial x^l}\right)$$

**证明**：由度量相容性和无挠性，通过直接计算可得。具体地：
1. 由无挠性：$\Gamma^k_{ij} = \Gamma^k_{ji}$
2. 由度量相容性导出三个方程
3. 联立求解即得上述公式

### 3.4 协变导数与平行移动

**协变导数**：对向量场 $V = \sum V^i \partial_i$，其协变导数为：
$$\nabla_j V^i = \frac{\partial V^i}{\partial x^j} + \sum_{k=1}^n \Gamma^i_{jk} V^k$$

**平行移动**：沿曲线 $\gamma(t)$ 的向量场 $V(t)$ 称为**平行的**，如果 $\nabla_{\dot{\gamma}} V = 0$。这给出了一种"比较"不同点切向量的方法。

## 四、黎曼曲率张量的定义与性质

### 4.1 曲率张量的定义

曲率度量了空间偏离平直的程度，定义为协变导数的不可交换性。

**定义 4.1（黎曼曲率张量）**：黎曼曲率张量 $R$ 是 $(1,3)$ 型张量，定义为：
$$R(X, Y)Z = \nabla_X \nabla_Y Z - \nabla_Y \nabla_X Z - \nabla_{[X,Y]} Z$$

在局部坐标下，分量为：
$$R^i_{jkl} = \frac{\partial \Gamma^i_{jl}}{\partial x^k} - \frac{\partial \Gamma^i_{jk}}{\partial x^l} + \sum_{m=1}^n \left(\Gamma^i_{mk}\Gamma^m_{jl} - \Gamma^i_{ml}\Gamma^m_{jk}\right)$$

**协变形式**：$R_{ijkl} = \sum_{m=1}^n g_{im} R^m_{jkl}$

### 4.2 曲率张量的代数性质

**定理 4.1（曲率张量的对称性）**：黎曼曲率张量满足：

1. **反对称性（前两个指标）**：$R_{ijkl} = -R_{jikl}$
2. **反对称性（后两个指标）**：$R_{ijkl} = -R_{ijlk}$
3. **对称性（交换前后）**：$R_{ijkl} = R_{klij}$
4. **第一Bianchi恒等式**：$R_{ijkl} + R_{iklj} + R_{iljk} = 0$

**第二Bianchi恒等式**：
$$\nabla_m R_{ijkl} + \nabla_k R_{ijlm} + \nabla_l R_{ijmk} = 0$$

其中 $\nabla$ 表示协变导数。

### 4.3 独立分量数目

**定理 4.2**：在 $n$ 维流形上，黎曼曲率张量的独立分量数目为：
$$\frac{n^2(n^2-1)}{12}$$

**特例**：
- $n=2$：1个独立分量（Gauss曲率）
- $n=3$：6个独立分量
- $n=4$：20个独立分量

## 五、截面曲率、Ricci曲率与数量曲率

### 5.1 截面曲率

**定义 5.1（截面曲率）**：设 $\sigma \subset T_p M$ 是由两个线性无关向量 $X, Y$ 张成的2维平面，**截面曲率**定义为：
$$K(\sigma) = \frac{R(X, Y, X, Y)}{|X|^2|Y|^2 - \langle X, Y \rangle^2} = \frac{R_{ijkl}X^iY^jX^kY^l}{(g_{ik}g_{jl} - g_{il}g_{jk})X^iY^jX^kY^l}$$

**几何意义**：截面曲率是2维子流形的Gauss曲率，完全决定了黎曼曲率张量。

**定理 5.1**：如果知道所有2维平面的截面曲率，则可以重构完整的黎曼曲率张量。

### 5.2 Ricci曲率

**定义 5.2（Ricci张量）**：Ricci张量是黎曼曲率张量的缩并：
$$R_{ij} = R^k_{ikj} = \sum_{k=1}^n g^{kl} R_{kilj}$$

**对称性**：$R_{ij} = R_{ji}$

**几何意义**：Ricci曲率度量了测地球体积与欧氏测地球体积的偏差。

**定义 5.3（Ricci曲率方向）**：对单位向量 $X$，$\text{Ric}(X) = R_{ij}X^iX^j$ 表示该方向的Ricci曲率。

### 5.3 数量曲率

**定义 5.4（数量曲率）**：数量曲率是Ricci张量的迹：
$$R = g^{ij}R_{ij} = \sum_{i,j=1}^n g^{ij}R_{ij}$$

**Einstein张量**：在广义相对论中起核心作用：
$$G_{ij} = R_{ij} - \frac{1}{2}Rg_{ij}$$

满足 $G^{ij}_{\ ;j} = 0$（由Bianchi恒等式导出）。

### 5.4 常曲率空间

**定义 5.5（常曲率空间）**：如果所有截面曲率都等于同一常数 $K$，则称 $(M, g)$ 为**常曲率空间**。

**定理 5.2**：常曲率空间的曲率张量具有形式：
$$R_{ijkl} = K(g_{ik}g_{jl} - g_{il}g_{jk})$$

对应的Ricci张量和数量曲率为：
- $R_{ij} = (n-1)Kg_{ij}$
- $R = n(n-1)K$

**分类定理**：单连通常曲率空间只有三种：
1. $K > 0$：球面 $S^n$
2. $K = 0$：欧氏空间 $\mathbb{R}^n$
3. $K < 0$：双曲空间 $\mathbb{H}^n$

## 六、测地线与指数映射

### 6.1 测地线方程

**定义 6.1（测地线）**：测地线是局部最短的曲线，满足微分方程：
$$\frac{d^2x^i}{dt^2} + \sum_{j,k=1}^n \Gamma^i_{jk} \frac{dx^j}{dt} \frac{dx^k}{dt} = 0$$

**变分原理**：测地线是能量泛函的临界点：
$$E[\gamma] = \frac{1}{2} \int_a^b g_{ij}(\gamma(t)) \dot{\gamma}^i(t) \dot{\gamma}^j(t) \, dt$$

### 6.2 指数映射与法坐标

**定义 6.2（指数映射）**：对 $p \in M$ 和 $v \in T_p M$，定义指数映射：
$$\exp_p(v) = \gamma_v(1)$$
其中 $\gamma_v$ 是以 $p$ 为起点、$v$ 为初始速度的测地线。

**定理 6.1（Gauss引理）**：在法坐标系下，径向测地线正交于测地球面。

## 七、与现代数学物理的联系

### 7.1 广义相对论

1915年，Albert Einstein建立广义相对论，其核心是**Einstein场方程**：
$$G_{ij} = \frac{8\pi G}{c^4} T_{ij}$$

其中：
- $G_{ij} = R_{ij} - \frac{1}{2}Rg_{ij}$ 是Einstein张量
- $T_{ij}$ 是能量-动量张量
- $G$ 是引力常数

Einstein方程将时空的几何与物质分布联系起来，是黎曼几何在物理学中最辉煌的应用。

### 7.2 与Weierstrass、Dedekind的关系

黎曼的工作对其同时代人产生了深远影响：
- **Weierstrass**：虽然批评黎曼的直观方法，但其严格分析学派补充了黎曼理论的严格性
- **Dedekind**：与黎曼在哥廷根共事，将黎曼的函数论思想发展为Dedekind割和代数数论
- **Christoffel、Ricci、Levi-Civita**：发展了张量分析，为Einstein提供了数学工具

## 八、总结与评价

### 8.1 历史地位

黎曼几何是19世纪最具革命性的数学成就之一：

1. **几何观念的革命**：从欧氏空间的刚性框架中解放，建立了任意弯曲空间的几何学
2. **内蕴方法的胜利**：证明了空间的几何性质可以内蕴地定义，不依赖嵌入
3. **跨学科的桥梁**：连接了纯数学与理论物理

### 8.2 现代发展

黎曼几何在20世纪和21世纪继续蓬勃发展：
- **比较几何学**（Rauch、Toponogov）
- **几何分析**（丘成桐、Hamilton、Perelman）
- **Ricci流与Poincaré猜想**
- **Calabi-Yau流形与弦论**
