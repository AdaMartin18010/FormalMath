# Einstein流形与特殊几何：黎曼几何的分类

## 一、Einstein流形

### 1.1 定义

**Einstein流形**：

```
Ricci张量满足：
R_{ij} = λ g_{ij}

其中：
- λ：Einstein常数
- g_{ij}：度规

性质：
- 常曲率（标量）
- 特殊几何
- 应用广泛
```

---

### 1.2 例子

**常曲率空间**：

```
- 球面Sⁿ：λ > 0
- 欧氏空间Rⁿ：λ = 0
- 双曲空间Hⁿ：λ < 0
```

**其他例子**：

- Calabi-Yau流形
- Kähler-Einstein流形
- 特殊几何

---

## 二、常曲率空间

### 2.1 定义

**常曲率K**：

```
曲率张量：
R_{ijkl} = K (g_{ik} g_{jl} - g_{il} g_{jk})

Ricci：
R_{ij} = (n-1)K g_{ij}

标量：
R = n(n-1)K
```

---

### 2.2 分类

**单连通情况**：

```
- K > 0：球面Sⁿ
- K = 0：欧氏空间Rⁿ
- K < 0：双曲空间Hⁿ
```

**一般情况**：

- 商空间
- 覆盖空间
- 分类理论

---

## 三、Kähler流形

### 3.1 定义

**Kähler流形**：

```
复流形M：

Kähler条件：
- 复结构J
- 度规g
- Kähler形式ω = g(J·, ·)
- dω = 0

性质：
- 特殊几何
- 应用广泛
```

---

### 3.2 Kähler-Einstein

**定义**：

```
Kähler流形 + Einstein条件：

R_{ij} = λ g_{ij}

性质：
- 特殊几何
- 应用广泛
- 现代研究
```

---

## 四、Calabi-Yau流形

### 4.1 定义

**Calabi-Yau流形**：

```
Kähler流形M：

条件：
- 第一Chern类：c₁(M) = 0
- 或：Ricci平坦

性质：
- 特殊几何
- 弦论应用
- 现代研究
```

---

### 4.2 应用
