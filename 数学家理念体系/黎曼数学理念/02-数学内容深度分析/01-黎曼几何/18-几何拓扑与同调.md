# 几何拓扑与同调：流形的拓扑不变量

## 一、同调群

### 1.1 定义

**奇异同调群**：

在拓扑空间X上，奇异同调群定义为：

```text
H_k(X, ℤ) = Z_k(X) / B_k(X)

其中：
- Z_k(X) = {k维闭链} = ker(∂_k)
- B_k(X) = {k维边界链} = im(∂_{k+1})
- ∂_k: C_k(X) → C_{k-1}(X)是边界算子
```

**链复形**：

```text
... → C_{k+1} →^{∂_{k+1}} C_k →^{∂_k} C_{k-1} → ...

性质：
- ∂_k ∘ ∂_{k+1} = 0（边界无边界）
```

**历史背景**：

同调理论由Poincaré（1895）开创，是代数拓扑的基础。它将拓扑问题转化为代数问题。

---

### 1.2 基本性质

**拓扑不变量**：

同调群是拓扑不变量：

- **同伦不变量**：同伦等价的空间有同构的同调群
- **同胚不变量**：同胚的空间有同构的同调群

**Betti数**：

```text
b_k(M) = rank H_k(M, ℤ)

这是流形的重要拓扑不变量
```

**Euler示性数**：

```text
χ(M) = Σ_{k=0}^n (-1)^k b_k(M)

这是同调群的组合不变量
```

---

### 1.3 计算同调

**Mayer-Vietoris序列**：

如果M = U ∪ V，则有长正合序列：

```text
... → H_k(U ∩ V) → H_k(U) ⊕ H_k(V) → H_k(M)
     → H_{k-1}(U ∩ V) → ...
```

**应用**：

Mayer-Vietoris序列是计算同调的重要工具。

**例子**：

- **球面Sⁿ**：H_0(Sⁿ) = ℤ, H_n(Sⁿ) = ℤ, 其他为0
- **环面T²**：H_0 = ℤ, H_1 = ℤ², H_2 = ℤ
- **高亏格曲面**：H_1 = ℤ^{2g}

## 二、上同调

### 2.1 de Rham上同调

**定义**：

在光滑流形M上，de Rham上同调定义为：

```text
H^k_{dR}(M, ℝ) = {闭k形式} / {恰当k形式}
              = ker(d: Ω^k → Ω^{k+1}) / im(d: Ω^{k-1} → Ω^k)

其中：
- 闭形式：dω = 0
- 恰当形式：ω = dη
```

**de Rham定理**：

de Rham上同调与奇异上同调同构：

```text
H^k_{dR}(M, ℝ) ≅ H^k(M, ℝ)

其中H^k(M, ℝ)是奇异上同调
```

**几何意义**：

de Rham上同调将分析（微分形式）与拓扑（上同调）联系起来。

---

### 2.2 Hodge理论

**Hodge定理**：

在紧致黎曼流形上，每个de Rham上同调类有唯一的调和代表元：

```text
H^k_{dR}(M) ≅ H^k(M) = {调和k形式}

其中H^k(M)是调和k形式空间
```

**应用**：

Hodge理论提供了计算上同调的具体方法（见Hodge理论文档）。

---

### 2.3 上同调环

**杯积**：

上同调有自然的乘法结构（杯积）：

```text
∪: H^k(M) × H^l(M) → H^{k+l}(M)

性质：
- 结合律
- 交换律（在系数域上）
- 单位元：1 ∈ H^0(M)
```

**上同调环**：

```text
H*(M) = ⊕_{k=0}^n H^k(M)

这是分次交换环
```

**应用**：

上同调环是流形分类的重要不变量。

## 三、Poincaré对偶

### 3.1 定理

**Poincaré对偶定理**：

在n维紧致可定向流形M上：

```text
H^k(M, ℤ) ≅ H_{n-k}(M, ℤ)

对偶映射：
D: H^k(M) → H_{n-k}(M)

通过交积实现
```

**历史背景**：

Poincaré对偶由Poincaré（1895）提出，是代数拓扑的基本定理。

---

### 3.2 交积

**定义**：

在紧致可定向流形M上，有交积：

```text
∩: H_k(M) × H_l(M) → H_{k+l-n}(M)

几何意义：
两个子流形的交
```

**与杯积的关系**：

在可定向流形上，交积与杯积对偶：

```text
⟨α ∪ β, γ⟩ = ⟨α, β ∩ γ⟩

其中⟨·, ·⟩是配对
```

---

### 3.3 应用

**流形分类**：

Poincaré对偶提供了流形分类的工具：

- **Betti数对称性**：b_k = b_{n-k}
- **Euler示性数**：在奇数维为0（可定向）
- **分类工具**：通过上同调分类流形

**几何分析**：

Poincaré对偶在几何分析中有应用，特别是在指标定理中。

---

## 四、示性类

### 4.1 Chern类

**定义**：

复向量丛E的Chern类c_k(E) ∈ H^{2k}(M, ℤ)可以通过曲率形式计算（Chern-Weil理论）。

**性质**：

- **拓扑不变量**：Chern类是拓扑不变量
- **乘法性**：c(E ⊕ F) = c(E) · c(F)
- **应用**：向量丛的分类

---

### 4.2 Pontryagin类

**定义**：

实向量丛E的Pontryagin类p_k(E) ∈ H^{4k}(M, ℤ)是Chern类的实形式。

**应用**：

Pontryagin类用于实向量丛的分类。

---

### 4.3 Euler类

**定义**：

定向向量丛E的Euler类e(E) ∈ H^n(M, ℤ)是最高Chern类。

**Gauss-Bonnet定理**：

在2维流形上，Euler类与Euler示性数相关：

```text
χ(M) = ∫_M e(TM)
```

### 5.1 流形分类

---

### 5.2 指标定理

**Atiyah-Singer指标定理**：

指标定理将拓扑不变量（指标）与分析不变量（通过示性类）联系起来：

```text
ind(D) = ∫_M ch(E) · td(TM)

其中：
- D：椭圆算子
- ch(E)：Chern特征
- td(TM)：Todd类
```

**应用**：

指标定理是几何分析的重要工具。

---

### 5.3 几何分析

**Hodge理论**：

Hodge理论将上同调与调和形式联系起来，提供了计算上同调的方法。

**几何不变量**：

同调群是几何不变量，用于研究流形的几何。
