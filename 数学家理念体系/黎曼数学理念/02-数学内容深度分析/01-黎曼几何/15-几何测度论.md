# 几何测度论：流形的测度与积分

## 一、Hausdorff测度

### 1.1 定义

**Hausdorff测度**：

```
n维Hausdorff测度：

H^n(E) = lim_{δ→0} inf{∑(diam U_i)^n}

其中：
- E：集合
- U_i：覆盖
- diam：直径

性质：
- 测度
- 应用广泛
```

---

### 1.2 应用

**应用**：

- 流形测度
- 几何分析
- 现代发展

---

## 二、面积公式

### 2.1 定理

**面积公式**：

```
映射f: M → N：

∫_N #(f^{-1}(y)) dH^n(y) = ∫_M |det Df| dH^n(x)

其中：
- #：计数
- Df：微分
- 应用广泛
```

---

### 2.2 应用

**应用**：

- 测度计算
- 几何分析
- 现代发展

## 三、等周不等式

### 3.1 定理

**等周不等式**：

```
n维流形M：

Vol(M) ≤ C (Area(∂M))^{n/(n-1)}

其中：
- C：常数
- 应用广泛
```
