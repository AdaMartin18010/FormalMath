# 谱几何与特征值：流形的"声音"

## 一、Laplace算子

### 1.1 定义

**Laplace算子**：

```
黎曼流形(M, g)上的Laplace算子：

Δ = -div grad = -g^{ij} ∇_i ∇_j

性质：
- 自伴
- 非负
- 应用广泛
```

---

### 1.2 特征值

**特征值问题**：

```
Δφ = λφ

其中：
- λ ≥ 0：特征值
- φ：特征函数

性质：
- 离散谱（紧致流形）
- 应用广泛
```

---

## 二、Weyl渐近

### 2.1 定理

**Weyl渐近公式**：

```
特征值计数：

N(λ) = #{λ_k ≤ λ} ~ (ω_n/(2π)^n) Vol(M) λ^{n/2}

其中：
- ω_n：单位球体积
- n：流形维数

性质：
- 渐近公式
- 应用广泛
```

---

### 2.2 应用

**应用**：

- 几何不变量
- 流形分类
- 现代发展
