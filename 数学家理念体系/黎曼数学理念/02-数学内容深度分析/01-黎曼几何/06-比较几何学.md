# 比较几何学：曲率与几何性质

## 一、比较定理

### 1.1 Rauch比较定理

**定理**：

```
流形M₁, M₂：

曲率条件：
K_M₁ ≥ K_M₂

测地线：
- 长度比较
- 角度比较
- 应用
```

---

### 1.2 应用

**应用**：

- 体积估计
- 直径估计
- 几何分析

---

## 二、体积比较

### 2.1 Bishop-Gromov

**定理**：

```
流形M：

曲率条件：
Ric ≥ (n-1)K

体积：
Vol(B_r) ≤ Vol_K(B_r)

其中：
Vol_K：常曲率K空间的体积
```

---

### 2.2 应用

**应用**：

- 体积估计
- 几何分析
- 流形分类

---

## 三、直径估计

### 3.1 Myers定理

**定理**：

```
流形M：

曲率条件：
Ric ≥ (n-1)K > 0

直径：
diam(M) ≤ π/√K

性质：
- 紧致
- 有限基本群
```

---

### 3.2 应用

**应用**：

- 流形分类
- 几何分析
- 现代发展
