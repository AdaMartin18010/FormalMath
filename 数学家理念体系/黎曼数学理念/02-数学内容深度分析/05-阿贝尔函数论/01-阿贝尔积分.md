# 阿贝尔积分：黎曼面的积分理论

## 一、阿贝尔积分

### 1.1 定义

**阿贝尔积分**：

```
紧致黎曼面R上的积分：

∫_γ ω

其中：
- γ：路径
- ω：全纯1形式

性质：
- 路径依赖
- 同调类
- 周期
```

---

### 1.2 分类

**第一类**：

```
全纯1形式的积分：
∫_γ ω（ω全纯）

性质：
- 有限
- 周期
- 应用广泛
```

**第二类**：

```
有单极点的积分：
∫_γ ω（ω有单极点）

性质：
- 对数项
- 应用
```

**第三类**：

```
有双极点的积分：
∫_γ ω（ω有双极点）

性质：
- 复杂
- 应用
```

---

## 二、周期

### 2.1 周期矩阵

**定义**：

```
紧致黎曼面R（亏格g）：

周期矩阵：
Π = (∫_{a_j} ω_i)

其中：
- {a_j}：同调基
- {ω_i}：全纯1形式基

性质：
- g×g复矩阵
- 对称性
- 正定性
```

---

### 2.2 Riemann双线性关系

**关系**：

```
∫_R ω_i ∧ ω̄_j = 0

其中：
- ω_i：全纯1形式
- ω̄_j：反全纯1形式

意义：
- 周期关系
- 几何应用
```

---

## 三、Abel-Jacobi映射

### 3.1 Jacobi簇

**定义**：

```
紧致黎曼面R（亏格g）：

Jacobi簇：
J(R) = H⁰(R, Ω¹)* / H₁(R, ℤ)

性质：
- g维复环面
- Abel群
- 全纯结构
```

---

### 3.2 Abel映射

**定义**：

```
A: R → J(R)

A(P) = (∫_{P₀}^P ω₁, ..., ∫_{P₀}^P ω_g)

其中：
- P₀：基点
- {ω₁, ..., ω_g}：全纯1形式基

性质：
- 全纯映射
- 单射（一般）
- 应用广泛
```

---

## 四、Abel定理

### 4.1 定理

**Abel定理**：

```
除子D = ∑n_i P_i：

D是主除子 ⟺
∑n_i A(P_i) = 0（在J(R)中）

意义：
- 线性等价
- 除子类群
- 几何应用
```

---

### 4.2 应用

**几何**：

- 除子理论
- 线性系统
- 分类理论
