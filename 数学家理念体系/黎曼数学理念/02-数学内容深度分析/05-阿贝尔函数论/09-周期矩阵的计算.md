## 一、周期矩阵的定义

### 1.1 阿贝尔簇的周期矩阵

**定义**：

```
给定g维阿贝尔簇A：
- 全纯1形式：ω₁,...,ω_g
- 同调基：γ₁,...,γ_{2g}

周期矩阵：
Π = (∫_{γ_j} ωᵢ)_{i,j}
```

**性质**：

- 2g×g复矩阵
- 定义Hodge结构
- 模空间坐标

### 1.2 黎曼关系

**黎曼双线性关系**：

```
周期矩阵Π满足：
- Π^T J Π = 0
- i Π^T J Π̄ > 0

其中：
J = (0  I)
    (-I 0)
```

**意义**：

- 极化条件
- 模空间条件
- 几何意义
