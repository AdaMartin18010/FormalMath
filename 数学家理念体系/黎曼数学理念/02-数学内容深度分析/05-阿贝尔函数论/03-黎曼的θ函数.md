## 一、θ函数

### 1.1 定义

**θ函数**：

```
θ(z, τ) = ∑_{n∈ℤ} exp(πi n²τ + 2πi n z)

其中：
- z ∈ C
- τ ∈ H（上半平面）

性质：
- 全纯
- 拟周期
- 应用广泛
```

---

### 1.2 性质

**拟周期性**：

```
θ(z+1, τ) = θ(z, τ)
θ(z+τ, τ) = exp(-πiτ - 2πiz) θ(z, τ)
```

**零点**：

- 简单零点
- 周期性
- 应用

## 二、Jacobi簇

### 2.1 对应

**对应**：

```
紧致黎曼面R（亏格g）：

Jacobi簇：
J(R) = C^g / (Π, I)

θ函数：
- 在J(R)上定义
- 显式描述
- 应用广泛
```

---

### 2.2 周期矩阵

**周期矩阵**：

```
Π = (∫_{a_j} ω_i)

θ函数：
θ(z, Π) = ∑_{n∈ℤ^g} exp(πi n^T Π n + 2πi n^T z)
```

## 三、Riemann θ函数

### 3.1 定义

**Riemann θ函数**：

```
紧致黎曼面R（亏格g）：

θ(z) = ∑_{n∈ℤ^g} exp(πi n^T Π n + 2πi n^T z)

其中：
- z ∈ C^g
- Π：周期矩阵
```
