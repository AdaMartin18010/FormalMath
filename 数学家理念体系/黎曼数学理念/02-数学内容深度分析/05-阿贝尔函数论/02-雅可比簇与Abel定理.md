## 一、雅可比簇

### 1.1 定义

**Jacobi簇**：

```
紧致黎曼面R（亏格g）：

J(R) = H⁰(R, Ω¹)* / H₁(R, ℤ)

其中：
- H⁰(R, Ω¹)*：全纯1形式的对偶
- H₁(R, ℤ)：同调群

性质：
- g维复环面
- Abel群
- 全纯结构
```

---

### 1.2 构造

**方法**：

```
1. 全纯1形式基：{ω₁, ..., ω_g}
2. 同调基：{a₁, b₁, ..., a_g, b_g}
3. 周期矩阵：Π = (∫_{a_j} ω_i)
4. Jacobi簇：C^g / (Π, I)
```

---

## 二、Abel映射

### 2.1 定义

**Abel映射**：

```
A: R → J(R)

A(P) = (∫_{P₀}^P ω₁, ..., ∫_{P₀}^P ω_g)

其中：
- P₀：基点
- {ω₁, ..., ω_g}：全纯1形式基

性质：
- 全纯映射
- 单射（一般）
- 应用广泛
```

---

### 2.2 推广

**除子映射**：

```
A: Div(R) → J(R)

A(∑n_i P_i) = ∑n_i A(P_i)

性质：
- 群同态
- 主除子 → 0
- 应用
```

---

## 三、Abel定理

### 3.1 定理

**Abel定理**：

```
除子D = ∑n_i P_i：

D是主除子 ⟺
∑n_i A(P_i) = 0（在J(R)中）

等价：
D ~ 0 ⟺ A(D) = 0
```

---

### 3.2 证明思路

**方法**：

```
1. 主除子 → 0：直接计算
2. 0 → 主除子：构造全纯函数
3. 使用Riemann-Roch
```

---

## 四、应用

### 4.1 除子类群

**Picard群**：

```
Pic(R) = Div(R) / 主除子

Abel映射：
A: Pic⁰(R) → J(R)

其中：
Pic⁰(R) = {度数为0的除子类}

同构：
Pic⁰(R) ≅ J(R)
```

---

### 4.2 线性系统

**应用**：

```
线性系统：
|D| = {E ~ D | E ≥ 0}

Abel定理：
- 判断线性等价
- 计算维数
- 分类理论
```

## 五、特殊情况

### 5.1 g = 1（椭圆曲线）

```
Abel映射：

性质：
- 同构
- 群结构
- 应用
```
