## 一、Shimura簇的定义与构造

### 1.1 Shimura簇的起源

**Shimura的贡献（1960s）**：

Shimura在1960年代引入了Shimura簇的概念，这是算术几何中的一个重要构造。

**基本思想**：

```
Shimura簇 = 模空间
    ↓
参数化：
- 带额外结构的代数簇
- 算术性质
- 应用广泛
```

**历史背景**：

- 椭圆曲线的模空间（经典）
- 高维推广（Shimura）
- 算术几何基础

### 1.2 定义

**Shimura簇**：

```
给定：
- 约化群G
- 共轭类h: S → G_R
- 紧开子群K ⊆ G(A_f)

Shimura簇：
Sh_K(G, h) = G(Q)\G(A_f)/K × h
```

**关键性质**：

- 代数簇结构
- 算术模型
- 应用广泛
