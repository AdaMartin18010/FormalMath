# 模空间的微分几何：度规与曲率

## 一、Weil-Petersson度量

### 1.1 定义

**Weil-Petersson度量**：

```
模空间M_g上的Weil-Petersson度量：

⟨μ, ν⟩_{WP} = ∫_Σ (μ · ν̄) / (ρ²) dA

其中：
- μ, ν：Beltrami微分
- ρ：双曲度规

性质：
- Kähler度量
- 应用广泛
```
