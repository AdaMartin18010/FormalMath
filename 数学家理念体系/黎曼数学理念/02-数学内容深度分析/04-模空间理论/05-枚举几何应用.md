## 二、Gromov-Witten不变量

### 2.1 定义

**Gromov-Witten不变量**：

设 $X$ 是光滑射影代数簇，$\beta \in H_2(X, \mathbb{Z})$ 是曲线类。

**稳定映射的模空间**：
$$\overline{M}_{g,n}(X, \beta) = \{\text{稳定映射 } f: C \to X \text{ 满足 } f_*[C] = \beta\}$$

**Gromov-Witten不变量**：
$$\langle \gamma_1, \ldots, \gamma_n \rangle_{g,\beta} = \int_{[\overline{M}_{g,n}(X, \beta)]^{vir}} \prod_{i=1}^n \text{ev}_i^*(\gamma_i)$$

其中：
- $\gamma_i \in H^*(X)$：上同调类
- $\text{ev}_i$：评估映射
- $[\overline{M}_{g,n}(X, \beta)]^{vir}$：虚拟基本类

---

### 2.2 性质

**Gromov-Witten不变量的性质**：

1. **对称性**：
   $$\langle \gamma_1, \ldots, \gamma_n \rangle_{g,\beta} = \langle \gamma_{\sigma(1)}, \ldots, \gamma_{\sigma(n)} \rangle_{g,\beta}$$

2. **度条件**：
   $$\sum_{i=1}^n \deg(\gamma_i) = 2(\dim X - 3)(1-g) + 2\int_\beta c_1(X)$$

3. **分裂公式**：连接不同亏格的GW不变量

## 三、应用实例

### 3.1 曲线计数

**问题**：计算通过给定点的曲线数

**例子**：$\mathbb{P}^2$ 中的有理曲线

```
问题：
计算通过5个一般点的3次有理曲线数

方法：
- 模空间：M̄_{0,5}(P², 3H)
- GW不变量：⟨pt, pt, pt, pt, pt⟩_{0,3H}
- 结果：12
```

**数学公式**：
$$\langle \text{pt}, \ldots, \text{pt} \rangle_{0,dH} = N_d$$

其中 $N_d$ 是通过 $3d-1$ 个一般点的 $d$ 次有理曲线数。

## 六、数学公式总结

### 核心公式

1. **Gromov-Witten不变量**：
   $$\langle \gamma_1, \ldots, \gamma_n \rangle_{g,\beta} = \int_{[\overline{M}_{g,n}(X, \beta)]^{vir}} \prod_{i=1}^n \text{ev}_i^*(\gamma_i)$$

2. **度条件**：
   $$\sum_{i=1}^n \deg(\gamma_i) = 2(\dim X - 3)(1-g) + 2\int_\beta c_1(X)$$

3. **虚拟基本类**：
   $$[\overline{M}_{g,n}(X, \beta)]^{vir} \in H_*(\overline{M}_{g,n}(X, \beta))$$

4. **评估映射**：
   $$\text{ev}_i: \overline{M}_{g,n}(X, \beta) \to X, \quad (f: C \to X) \mapsto f(p_i)$$

5. **稳定映射条件**：
   $$\text{Aut}(f) < \infty \iff \text{稳定}$$

6. **模空间的维数**：
   $$\dim \overline{M}_{g,n}(X, \beta) = (3 - \dim X)(g-1) + n + \int_\beta c_1(X)$$

7. **GW不变量的对称性**：
   $$\langle \gamma_1, \ldots, \gamma_n \rangle_{g,\beta} = \langle \gamma_{\sigma(1)}, \ldots, \gamma_{\sigma(n)} \rangle_{g,\beta}$$

8. **分裂公式**：
   $$\langle \gamma_1, \ldots, \gamma_n \rangle_{g,\beta} = \sum \langle \gamma_I \rangle_{g_1,\beta_1} \cdot \langle \gamma_J \rangle_{g_2,\beta_2}$$

9. **曲线计数**：
   $$N_d = \langle \text{pt}, \ldots, \text{pt} \rangle_{0,dH}$$

10. **量子乘积**：
    $$\gamma_1 \star \gamma_2 = \sum_{\beta} q^\beta \langle \gamma_1, \gamma_2, \cdot \rangle_{0,\beta}$$
