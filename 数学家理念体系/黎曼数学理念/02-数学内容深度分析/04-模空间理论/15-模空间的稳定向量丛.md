# 模空间的稳定向量丛：几何与物理

## 一、稳定向量丛

### 1.1 定义

**稳定向量丛**：

```
黎曼面R上的稳定向量丛E：

μ(E) < μ(F)（对所有子丛F）

其中：
μ(E) = deg(E) / rank(E)

性质：
- 稳定性
- 应用广泛
```

---

### 1.2 应用

**应用**：

- 模空间
- 分类理论
- 几何应用
