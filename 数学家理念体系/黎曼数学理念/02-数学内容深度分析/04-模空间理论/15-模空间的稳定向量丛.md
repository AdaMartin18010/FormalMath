## 一、稳定向量丛

### 1.1 定义

**稳定向量丛**：

```
黎曼面R上的稳定向量丛E：

μ(E) < μ(F)（对所有子丛F）

其中：
μ(E) = deg(E) / rank(E)（斜率）

性质：
- 稳定性
- 有限自同构群
- 应用广泛
```

**历史**：

- Mumford（1960s）
- 稳定化方法
- 现代发展

### 1.2 稳定条件

**稳定条件**：

```
稳定条件：
- 斜率条件
- 有限自同构群
- 模空间性质
```

**应用**：

- 模空间构造

- 现代研究

## 二、模空间

### 2.1 定义

**模空间M(r, d)**：

```
稳定向量丛的模空间：
M(r, d) = {稳定向量丛} / 等价

其中：
- r：秩
- d：度
- 等价：同构
```

**性质**：

- 复流形
- 有限维
- 应用广泛

### 2.2 性质

**几何性质**：

- 光滑
- 紧化
- 应用广泛

**应用**：

- 分类理论
- 枚举几何
- 现代研究
