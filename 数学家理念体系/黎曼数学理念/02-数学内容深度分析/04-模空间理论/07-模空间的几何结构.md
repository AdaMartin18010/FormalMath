## 一、模空间的几何结构

### 1.1 Teichmüller度量

**Teichmüller度量**（1940s）：

```
Teichmüller空间T_g上的度量：

d_T(R₁, R₂) = (1/2) log K(f)

其中：
- f：R₁到R₂的极值拟共形映射
- K(f)：最大伸缩比
```

**数学公式**：

$$d_T(R_1, R_2) = \frac{1}{2} \log K(f)$$

其中$K(f)$是极值拟共形映射$f: R_1 \to R_2$的最大伸缩比。

**性质**：

- 完备度量
- 非正曲率
- 应用广泛

---

### 1.2 Weil-Petersson度量

**Weil-Petersson度量**（1960s）：

```
Teichmüller空间T_g上的Kähler度量：

g_{WP} = ∫_R (μ, ν̄) dA

其中：
- μ, ν：Beltrami微分
- dA：面积元
```

**数学公式**：

$$g_{WP}(\mu, \nu) = \int_R \mu \bar{\nu} \, dA$$

其中$\mu, \nu$是Beltrami微分。

**性质**：

- Kähler度量
- 负曲率
- 应用广泛

## 七、具体例子

### 7.1 Teichmüller度量的例子

**例子1：亏格1的情况**：

对于亏格1（椭圆曲线），Teichmüller空间：
$$T_1 = \mathbb{H} / \text{SL}(2, \mathbb{Z})$$

其中$\mathbb{H}$是上半平面。

**Teichmüller距离**：
$$d_T(\tau_1, \tau_2) = \frac{1}{2} \log K(f)$$

其中$f$是$\tau_1$到$\tau_2$的极值拟共形映射。

**例子2：亏格2的情况**：

对于亏格2，Teichmüller空间：
$$T_2 \cong \mathbb{H}^3$$

是3维复流形。

### 7.2 Weil-Petersson度量的例子

**例子1：Weil-Petersson内积**：

在$T_g$的切空间上，Weil-Petersson内积：
$$g_{WP}(\mu, \nu) = \int_X \mu \bar{\nu} \, dA$$

其中$\mu, \nu$是Beltrami微分。

**例子2：曲率计算**：

Weil-Petersson度量的Ricci曲率：
$$\text{Ric}_{WP} = -\frac{1}{2\pi(2g-2+n)} g_{WP}$$

其中$n$是标记点数。

## 八、数学公式总结

### 核心公式

1. **Teichmüller距离**：
   $$d_T(R_1, R_2) = \frac{1}{2} \log K(f)$$

2. **Weil-Petersson度量**：
   $$g_{WP}(\mu, \nu) = \int_R \mu \bar{\nu} \, dA$$

3. **切空间**：
   $$T_X T_g \cong H^1(X, T_X)$$

4. **Ricci曲率**：
   $$\text{Ric}(g_{WP}) = -\frac{1}{2\pi(2g-2+n)} g_{WP}$$

5. **模空间**：
   $$M_g = T_g / \text{Mod}_g$$

6. **维数**：
   $$\dim T_g = 3g - 3 \quad (g \geqqq 2)$$
