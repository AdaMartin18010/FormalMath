# 模形式与模曲线：模空间的函数理论

## 一、模形式

### 1.1 定义

**模形式**：

```
函数f: H → C（H：上半平面）：

模形式条件：
1. 全纯
2. f(γz) = (cz+d)^k f(z)（γ ∈ SL(2,ℤ)）
3. 增长条件

其中：
k：权
```

---

### 1.2 性质

**性质**：

- 全纯
- 模不变
- 应用广泛

---

## 二、模曲线

### 2.1 定义

**模曲线**：

```
模曲线 = H/Γ

其中：
Γ：Fuchsian群

例子：
- X(1) = H/SL(2,ℤ)
- j-不变量
```

---

### 2.2 性质

**性质**：

- 紧致化
- 代数曲线
- 应用
