## 一、模空间的定义

### 1.1 基本概念

**模空间**：

```
M_g = {亏格g紧致黎曼面的模空间}

等价关系：
R₁ ~ R₂ ⟺ R₁ ≅ R₂（同构）

性质：
- 参数化所有黎曼面
- 分类工具
- 几何结构
```

**维数**：

```
dim M_g = 3g - 3（g ≥ 2）

例子：
- g = 0：M₀ = {点}（唯一P¹）
- g = 1：M₁ = H/SL(2,ℤ)（1维）
- g ≥ 2：M_g（3g-3维）
```

## 三、模空间的构造

### 3.1 Teichmüller空间

**定义**：

```
T_g = {标记的亏格g黎曼面}

性质：
- 复流形
- 3g-3维
- 万有覆盖
```

**模空间**：

```
M_g = T_g / Mod_g

其中：
Mod_g：映射类群
```

---

### 3.2 映射类群

**定义**：

```
Mod_g = Diff⁺(R) / Diff₀(R)

其中：
- Diff⁺(R)：保向微分同胚
- Diff₀(R)：同伦于恒等

性质：
- 离散群
- 作用在T_g上
```
