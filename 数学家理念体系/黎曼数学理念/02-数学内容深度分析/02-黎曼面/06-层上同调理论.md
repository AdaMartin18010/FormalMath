## 三、Riemann-Roch

### 3.1 层形式

**Riemann-Roch**：

```
紧致黎曼面R，线丛L：

χ(R, L) = dim H⁰(R, L) - dim H¹(R, L)
        = deg(L) - g + 1

其中：
- χ：Euler示性数
- g：亏格
```

---

### 3.2 应用

**应用**：

- 除子理论
- 分类理论
- 几何应用

## 四、Serre对偶

### 4.1 对偶定理

**Serre对偶**：

```
H⁰(R, L) ≅ H¹(R, K ⊗ L⁻¹)*

其中：
- K：典范线丛
- *：对偶

应用：
- Riemann-Roch证明
- 上同调计算
```

### 4.2 应用

**应用**：
