## 三、Riemann-Roch

### 3.1 层形式

**Riemann-Roch**：

```
紧致黎曼面R，线丛L：

χ(R, L) = dim H⁰(R, L) - dim H¹(R, L)
        = deg(L) - g + 1

其中：
- χ：Euler示性数
- g：亏格
```

---

### 3.2 应用

**应用**：

- 除子理论

## 四、Serre对偶

### 4.1 对偶定理

```
H⁰(R, L) ≅ H¹(R, K ⊗ L⁻¹)*

其中：
- K：典范线丛
- *：对偶

应用：
- Riemann-Roch证明
- 上同调计算
```

## 七、数学公式总结

### 核心公式

1. **层定义**：
   $$\mathcal{F}: \text{Open}(R)^{\text{op}} \to \text{Ab}, \quad \text{满足层公理}$$

2. **上同调群**：
   $$H^i(R, \mathcal{F}) = \check{H}^i(R, \mathcal{F}) = \text{Ext}^i(O_R, \mathcal{F})$$

3. **Riemann-Roch（层形式）**：
   $$\chi(R, \mathcal{F}) = h^0(R, \mathcal{F}) - h^1(R, \mathcal{F}) = \deg(\mathcal{F}) - g + 1$$

4. **Serre对偶**：
   $$H^i(R, \mathcal{F}) \cong H^{1-i}(R, \mathcal{F}^\vee \otimes \omega_R)^*, \quad \omega_R = \Omega_R^1$$

5. **上同调与全纯1形式**：
   $$H^1(R, O_R) \cong H^0(R, \Omega_R^1)^* = \mathbb{C}^g$$

6. **上同调与线丛**：
   $$H^i(R, L) = 0 \text{（$i > 1$，$R$ 紧致）}$$

7. **上同调长正合列**：
   $$0 \to H^0(R, \mathcal{F}) \to H^0(R, \mathcal{G}) \to H^0(R, \mathcal{H}) \to H^1(R, \mathcal{F}) \to \cdots$$

8. **上同调与拉回**：
   $$H^i(R, f^*\mathcal{F}) \cong H^i(S, \mathcal{F}) \text{（$f: R \to S$ 有限覆盖）}$$

9. **上同调与推前**：
   $$H^i(S, f_*\mathcal{F}) \cong H^i(R, \mathcal{F}) \text{（$f: R \to S$ 有限覆盖）}$$

10. **上同调维数**：
    $$\text{cd}(R) = 1, \quad H^i(R, \mathcal{F}) = 0 \text{（$i > 1$）}$$
