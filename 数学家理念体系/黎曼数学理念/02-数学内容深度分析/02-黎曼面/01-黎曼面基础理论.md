# 黎曼面基础理论：复分析的拓扑化

## 一、基本定义

### 1.1 黎曼面

**定义**：

```
1维复流形R：

1. 拓扑空间（Hausdorff，第二可数）
2. 复坐标图册
3. 转移函数全纯

等价：
- 2维实流形
- 复结构
- 可定向
```

**例子**：

```
- 复平面C
- 黎曼球面P¹ = C ∪ {∞}
- 环面（椭圆曲线）
- 高亏格曲面
```

---

### 1.2 全纯函数

**定义**：

```
f: R → C全纯 ⟺
局部表示为全纯函数

等价：
- Cauchy-Riemann方程
- 复可微
- 解析
```

**性质**：

- 开映射
- 最大值原理
- 零点孤立

---

## 二、多值函数的单值化

### 2.1 问题

**多值函数**：

```
√z, log z, arcsin z

问题：
- 分支点
- 多值性
- 需要新方法
```

**黎曼的解决**：

```
构造覆盖空间：
- 多值 → 单值
- 分支点 → 分支覆盖
- 拓扑结构
```

---

### 2.2 构造

**例子：√z**：

```
z = w²

黎曼面：
- 两个分支
- 在0处连接
- 单值化√z
```

**一般情况**：

```
代数函数：
F(z,w) = 0

黎曼面：
{(z,w) | F(z,w) = 0}
    ↓
紧致黎曼面
```

---

## 三、单值化定理

### 3.1 基本单值化

**定理**（Poincaré-Koebe, 1907）：

```
单连通黎曼面R：

R ≅
- C（复平面）
- P¹（黎曼球面）
- D（单位圆盘）

分类：
- 椭圆型：P¹
- 抛物型：C
- 双曲型：D
```

---

### 3.2 一般单值化

**定理**：

```
任意黎曼面R：

R ≅ D/Γ

其中：
- D：万有覆盖（C, P¹, 或D）
- Γ：Fuchsian群（离散自同构群）

分类：
- 亏格0：P¹
- 亏格1：C/Λ（环面）
- 亏格≥2：D/Γ（双曲）
```

---

## 四、亏格与分类

### 4.1 亏格

**定义**：

```
紧致黎曼面R的亏格g：

g = (1/2) dim H¹(R, C)

几何意义：
- 洞的个数
- 拓扑不变量
- 分类工具
```

**例子**：

```
- g = 0：P¹（球面）
- g = 1：环面
- g ≥ 2：高亏格曲面
```

---

### 4.2 分类

**亏格分类**：

```
g = 0：
- 唯一：P¹
- 自同构群：PGL(2,C)

g = 1：
- 椭圆曲线
- 模空间：H/SL(2,ℤ)

g ≥ 2：
- 一般型
- 模空间：M_g（3g-3维）
```

---

## 五、全纯1形式

### 5.1 定义

**全纯1形式**：

```
ω = f(z)dz

其中：
- f：全纯函数
- dz：全纯微分

性质：
- 局部表示
- 全纯
- 闭（dω = 0）
```

---

### 5.2 上同调

**Hodge理论**：

```
H¹(R, C) = H⁰(R, Ω¹) ⊕ H⁰(R, Ω¹)⁻

其中：
- H⁰(R, Ω¹)：全纯1形式
- 维数：g（亏格）
```

**应用**：

- Abel-Jacobi映射
- 周期矩阵
- 模空间

---

## 六、代数函数

### 6.1 定义

**代数函数**：

```
F(z,w) = 0

其中：
F：多项式

例子：
w² = z³ + az + b（椭圆曲线）
wⁿ = P(z)（超椭圆曲线）
```

**黎曼面**：

```
代数函数 → 黎曼面

F(z,w) = 0
    ↓
{(z,w) | F(z,w) = 0}
    ↓
紧致黎曼面
```

---

### 6.2 亏格公式

**Riemann-Hurwitz**：

```
f: R → S（分支覆盖）

2g(R) - 2 = d(2g(S) - 2) + ∑(e_p - 1)

其中：
- d：度数
- e_p：分支指数
```

---

## 七、现代发展

### 7.1 代数几何

**概形**：

```
黎曼面（1维复流形）
    ↓
代数曲线（1维概形）

统一：
- 全纯函数 → 正则函数
- 上同调 → 层上同调
```

---

### 7.2 模空间

**定义**：

```
M_g = {亏格g紧致黎曼面的模空间}

性质：
- 3g-3维（g≥2）
- 复流形
- 紧化：M̄_g
```
