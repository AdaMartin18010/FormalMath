# 复结构与形变理论：黎曼面的现代方法

## 一、复结构

### 1.1 定义

**复结构**：

```
流形M上的复结构J：

J: TM → TM

性质：
- J² = -id
- 可积性
- 应用广泛
```

---

### 1.2 黎曼面

**黎曼面**：

```
1维复流形 = 2维实流形 + 复结构

性质：
- 可定向
- 复结构
- 应用广泛
```

---

## 二、形变理论

### 2.1 形变

**形变**：

```
复结构的形变：

J(t) = J₀ + t·δJ + ...

其中：
- J₀：初始复结构
- δJ：形变
- t：参数

性质：
- 局部理论
- 应用广泛
```

---

### 2.2 应用

**应用**：

- 模空间
- 分类理论
- 几何应用

---

## 三、Kuranishi理论

### 3.1 形变空间

**Kuranishi空间**：

```
复结构J的形变空间：

Def(J) = H¹(M, T_M)

其中：
- T_M：切丛
- H¹：上同调

性质：
- 局部模空间
- 应用广泛
```

---

### 3.2 应用

**应用**：

- 模空间
- 分类理论
- 几何应用
