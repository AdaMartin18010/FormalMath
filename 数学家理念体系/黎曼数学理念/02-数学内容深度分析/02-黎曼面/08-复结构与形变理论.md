```
流形M上的复结构J：

J: TM → TM

性质：
- J² = -id
- 可积性
- 应用广泛
```

## 七、复结构的数学理论

### 7.1 复结构的严格定义

**定义**：

- 流形$M$上的复结构$J$是切丛$TM$的自同态
- 满足$J^2 = -\text{id}$
- 可积性条件：Nijenhuis张量为零

**数学公式**：

- 复结构：$$J: TM \to TM, \quad J^2 = -\text{id}$$
- Nijenhuis张量：$$N_J(X, Y) = [JX, JY] - J[JX, Y] - J[X, JY] + [X, Y]$$
- 可积性：$$N_J = 0 \iff \text{复结构可积}$$

---

### 7.2 黎曼面的复结构

**2维情况**：

- 1维复流形 = 2维实流形 + 复结构
- 所有2维可定向流形都有复结构
- 复结构唯一（模共形等价）

**数学公式**：

- 复坐标：$$z = x + iy, \quad \bar{z} = x - iy$$
- 复结构：$$J\leqqqqqft(\frac{\partial}{\partial x}\right) = \frac{\partial}{\partial y}, \quad J\leqqqqqft(\frac{\partial}{\partial y}\right) = -\frac{\partial}{\partial x}$$

## 八、形变理论的数学基础

### 8.1 形变空间的严格定义

**Kuranishi空间**：

- 复结构$J$的形变空间：$\text{Def}(J) = H^1(M, T_M)$
- 其中$T_M$是切丛，$H^1$是第一上同调群
- 形变空间是局部模空间

**数学公式**：

- 形变空间：$$\text{Def}(J) = H^1(M, T_M)$$
- 切空间：$$T_{[J]}\text{Def}(J) = H^1(M, T_M)$$
- 障碍空间：$$\text{Ob}(J) = H^2(M, T_M)$$

---

### 8.2 形变方程

**形变方程**：

- 复结构的形变满足Maurer-Cartan方程
- 障碍类在$H^2(M, T_M)$中
- 如果障碍为零，则形变空间光滑

**数学公式**：

- Maurer-Cartan方程：$$\bar{\partial}\eta + \frac{1}{2}[\eta, \eta] = 0$$
- 其中$\eta \in \Omega^{0,1}(M, T_M)$是形变参数

## 九、Kuranishi理论的详细内容

### 9.1 Kuranishi定理

**定理**（Kuranishi, 1962）：

- 紧致复流形$M$的复结构形变空间局部由Kuranishi族参数化
- 形变空间是$H^1(M, T_M)$中的解析子集
- 如果$H^2(M, T_M) = 0$，则形变空间光滑

**数学公式**：

- Kuranishi映射：$$\kappa: H^1(M, T_M) \to H^2(M, T_M)$$
- 形变空间：$$\text{Def}(M) = \kappa^{-1}(0)$$

---

### 9.2 应用

**应用**：

- 模空间的构造
- 分类理论
- 几何应用

**数学公式**：

- 模空间：$$\mathcal{M}_g = \{\text{亏格}g\text{的黎曼面}\}/\text{同构}$$
- 维数：$$\dim \mathcal{M}_g = 3g - 3 \quad (g \geqqqqqq 2)$$
