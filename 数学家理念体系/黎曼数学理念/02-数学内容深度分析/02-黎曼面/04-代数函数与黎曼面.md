## 一、代数函数

### 1.1 定义

**代数函数**：

```
F(z, w) = 0

其中：
F：多项式

例子：
- w² = z³ + az + b（椭圆曲线）
- wⁿ = P(z)（超椭圆曲线）
```

---

### 1.2 多值性

**问题**：

```
代数函数：
- 多值
- 分支点
- 需要新方法
```

**解决**：

- 黎曼面
- 单值化
- 拓扑方法

---

## 二、黎曼面的构造

### 2.1 从代数函数

**构造**：

```
代数函数F(z, w) = 0
    ↓
黎曼面：
{(z, w) | F(z, w) = 0}
    ↓
紧致化
    ↓
紧致黎曼面
```

---

### 2.2 例子

**椭圆曲线**：

```
w² = z³ + az + b

黎曼面：
- 亏格1
- 环面
- 椭圆曲线
```

**超椭圆曲线**：

```
w² = P(z)（P：多项式）

黎曼面：
- 亏格g
- 超椭圆曲线
```

## 三、亏格公式

### 3.1 Riemann-Hurwitz

**公式**：

```
f: R → S（分支覆盖）

2g(R) - 2 = d(2g(S) - 2) + ∑(e_p - 1)

其中：
- d：度数
- e_p：分支指数
```

---

### 3.2 应用

**计算亏格**：

- 代数曲线
- 分支覆盖
- 分类理论

## 四、代数曲线

- 上同调 ↔ 层上同调
