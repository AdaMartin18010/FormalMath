## 一、历史背景

### 1.1 问题的提出

**Poincaré（1882）**：

```
问题：
单连通黎曼面的分类？

猜想：
只有三种类型
```

**Koebe（1907）**：

```
证明：
单值化定理

结果：
完全分类
```

---

### 1.2 定理的陈述

**单值化定理**：

```
单连通黎曼面R：

R ≅
- P¹（黎曼球面）
- C（复平面）
- D（单位圆盘）

分类：
- 椭圆型：P¹
- 抛物型：C
- 双曲型：D
```

---

## 二、椭圆型：P¹

### 2.1 黎曼球面

**定义**：

```
P¹ = C ∪ {∞}

坐标：
- z ∈ C
- w = 1/z（在∞附近）

性质：
- 紧致
- 单连通
- 亏格0
```

**自同构群**：

```
Aut(P¹) = PGL(2,C)

作用：
z → (az+b)/(cz+d)

性质：
- 3维复Lie群
- 传递作用
```

---

### 2.2 分类

**唯一性**：

```
单连通紧致黎曼面：
- 唯一：P¹
- 亏格0
- 椭圆型
```

---

## 三、抛物型：C

### 3.1 复平面

**定义**：

```
C = 复平面

性质：
- 非紧致
- 单连通
- 全纯自同构：z → az+b
```

**分类**：

```
单连通非紧致黎曼面：

R ≅ C ⟺
- 非双曲
- 非椭圆
- 抛物型
```

---

### 3.2 例子

**典型例子**：

```
- C本身
- C* = C\{0}（非单连通）
- 环面（非单连通）
```

---

## 四、双曲型：D

### 4.1 单位圆盘

**定义**：

```
D = {z ∈ C | |z| < 1}

Poincaré度规：
ds² = 4|dz|²/(1-|z|²)²

性质：
- 常负曲率-1
- 双曲几何
- 自同构：Möbius变换
```

**自同构群**：

```
Aut(D) = PSU(1,1)

作用：
z → (az+b)/(b̄z+ā̄)

其中：
|a|² - |b|² = 1
```

---

### 4.2 分类

**双曲型**：

```
单连通黎曼面R：

R ≅ D ⟺
- 非椭圆
- 非抛物
- 双曲型

等价：
- 亏格≥2（紧致）
- 或非紧致且非C
```

---

## 五、一般单值化

### 5.1 万有覆盖

**定义**：

```
任意黎曼面R：

万有覆盖R̃：
- 单连通
- 覆盖R
- 万有性质

分类：
R̃ ≅ P¹, C, 或D
```

---

### 5.2 覆盖群

**定义**：

```
覆盖映射：
π: R̃ → R

覆盖群：
Γ = Aut(R̃/R)（离散）

结果：
R ≅ R̃/Γ
```

**分类**：

```
- R̃ = P¹：Γ = {e}（平凡）
- R̃ = C：Γ = ℤ或ℤ²（平移）
- R̃ = D：Γ = Fuchsian群
```

---

## 六、Fuchsian群

### 6.1 定义

**Fuchsian群**：

```
Γ ⊆ Aut(D)（离散子群）

性质：
- 离散
- 不连续作用
- 商空间：D/Γ（黎曼面）
```

**例子**：

```
模群：
SL(2,ℤ)作用在H（上半平面）

商：
H/SL(2,ℤ) = 模曲线

亏格：
g = 0（j-不变量）
```

---

### 6.2 分类应用

**紧致黎曼面**：

```
亏格g：

- g = 0：P¹（椭圆）
- g = 1：C/Λ（抛物，Λ：格）
- g ≥ 2：D/Γ（双曲，Γ：Fuchsian）
```

## 七、证明思路

### 7.1 基本步骤

**方法**：

```
1. 构造万有覆盖R̃
2. 证明R̃单连通
3. 分类R̃（P¹, C, 或D）
4. 确定覆盖群Γ
5. 得到R = R̃/Γ
```

## 十一、数学公式总结

### 核心公式

1. **单值化定理**：
   $$\text{单连通黎曼面 } R \cong \begin{cases} \hat{\mathbb{C}} & g = 0 \\ \mathbb{C} & g = 1 \\ \mathbb{D} & g \geq 2 \end{cases}$$

2. **黎曼球面**：
   $$\hat{\mathbb{C}} = \mathbb{C} \cup \{\infty\} \cong S^2$$

3. **单位圆盘**：
   $$\mathbb{D} = \{z \in \mathbb{C} : |z| < 1\}$$

4. **Poincaré度规**：
   $$ds^2 = \frac{4|dz|^2}{(1-|z|^2)^2}$$

5. **Fuchs群作用**：
   $$\Gamma \times \mathbb{D} \to \mathbb{D}, \quad \gamma \cdot z = \frac{az + b}{cz + d}$$

6. **自守函数**：
   $$f(\gamma z) = f(z) \text{ 对所有 } \gamma \in \Gamma$$

7. **万有覆盖**：
   $$p: \tilde{R} \to R, \quad \tilde{R} \cong \mathbb{D}$$

8. **覆盖群**：
   $$\pi_1(R) \cong \Gamma \subset \text{PSL}(2, \mathbb{R})$$

9. **单值化映射**：
   $$R \cong \mathbb{D} / \Gamma$$

10. **Riemann-Hurwitz公式**：
    $$2g - 2 = n(2g' - 2) + \sum (e_p - 1)$$
