# 黎曼面的微分几何：度规与曲率

## 一、度规

### 1.1 定义

**度规**：

```
黎曼面R上的度规：

ds² = g(z)|dz|²

其中：
- g(z)：正函数
- dz：全纯微分

性质：
- 共形
- 应用广泛
```

---

### 1.2 例子

**例子**：

- 球面：常曲率
- 环面：平坦
- 双曲平面：负曲率

## 二、曲率

### 2.1 Gauss曲率

**Gauss曲率**：

```
2维黎曼面R：

Gauss曲率：
K = -Δ log g / (2g)

其中：
- g：度规
- Δ：Laplace算子

性质：
- 内蕴
- 应用广泛
```

### 2.2 应用

## 三、Gauss-Bonnet

### 3.1 定理

**Gauss-Bonnet定理**：

```
紧致黎曼面R：

∫_R K dA = 2π(2-2g)

其中：
- K：Gauss曲率
- g：亏格

意义：
- 曲率与拓扑
- 应用广泛
```
