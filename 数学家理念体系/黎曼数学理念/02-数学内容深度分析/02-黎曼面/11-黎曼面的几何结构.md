## 一、复结构

### 1.1 复结构的定义

**复结构J**：

```
黎曼面R上的复结构：
- J: TR → TR（自同构）
- J² = -id
- 可积性条件
```

**性质**：

- 可定向
- 复维数1
- 应用广泛

### 1.2 复坐标

**局部复坐标**：

```
局部复坐标z：
- 全纯坐标
- 复结构
- 应用广泛
```

**应用**：

- 全纯函数
- 复分析
- 几何应用

## 二、度规结构

### 2.1 度规定义

**度规g**：

```
黎曼度规g：
- 正定对称双线性形式
- 局部：ds² = g(z)|dz|²
- 应用广泛
```

**性质**：

- 正定性
- 对称性
- 应用广泛

### 2.2 曲率

**Gauss曲率K**：

```
Gauss曲率：
- K = -Δ log g
- 几何不变量
- 应用广泛
```

**应用**：

- 几何分类
- 几何分析
- 现代研究
