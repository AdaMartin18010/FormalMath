## 一、全纯1形式

### 1.1 定义

**全纯1形式**：

```
ω = f(z)dz

其中：
- f：全纯函数
- dz：全纯微分

性质：
- 局部表示
- 全纯
- 闭（dω = 0）
```

**坐标变换**：

```
坐标变换z → w：

ω = f(z)dz = f(z(w)) (dz/dw) dw = g(w)dw

其中：
g(w) = f(z(w)) (dz/dw)
```

---

### 1.2 例子

**复平面C**：

```
全纯1形式：
ω = dz

性质：
- 全局定义
- 无零点
```

**环面**：

```
全纯1形式：
ω = dz（在覆盖空间C上）

性质：
- 周期
- 模空间
```

## 二、上同调

### 2.1 定义

**全纯1形式空间**：

```
H⁰(R, Ω¹) = {全纯1形式}

维数：
dim H⁰(R, Ω¹) = g（亏格）

基：
{ω₁, ..., ω_g}
```

---

### 2.2 Hodge理论

**分解**：

```
H¹(R, C) = H⁰(R, Ω¹) ⊕ H⁰(R, Ω¹)⁻

其中：
- H⁰(R, Ω¹)：全纯1形式
- H⁰(R, Ω¹)⁻：反全纯1形式

维数：
dim H¹(R, C) = 2g
```

## 三、周期

### 3.1 周期矩阵

**定义**：

```
紧致黎曼面R（亏格g）：

周期矩阵：
Π = (∫_{a_j} ω_i)_{i,j=1}^g

其中：
- {a_j}：同调基
- {ω_i}：全纯1形式基
```

**性质**：

- g×g复矩阵
- 对称性
- 正定性

---

### 3.2 Riemann双线性关系

**关系**：

```
∫_R ω_i ∧ ω̄_j = 0

其中：
- ω_i：全纯1形式
- ω̄_j：反全纯1形式
```

## 四、Abel-Jacobi映射

### 4.1 Jacobi簇

**定义**：

```
紧致黎曼面R（亏格g）：

Jacobi簇：
J(R) = H⁰(R, Ω¹)* / H₁(R, ℤ)

性质：
- g维复环面
- Abel群
- 全纯结构
```

---

### 4.2 Abel映射

**定义**：

```
A: R → J(R)

A(P) = (∫_{P₀}^P ω₁, ..., ∫_{P₀}^P ω_g)

其中：
- P₀：基点
- {ω₁, ..., ω_g}：全纯1形式基
```

**性质**：

- 全纯映射
- 单射（一般）
- 应用广泛

## 五、应用

### 5.1 Abel定理

**定理**：

```
除子D = ∑n_i P_i：

D是主除子 ⟺
∑n_i A(P_i) = 0（在J(R)中）

意义：
- 线性等价
- 除子类群
- 几何应用
```

## 九、数学公式总结

### 核心公式

1. **全纯1形式定义**：
   $$\omega = f(z)dz, \quad f \text{ 全纯函数}$$

2. **坐标变换**：
   $$\omega = f(z)dz = f(z(w)) \frac{dz}{dw} dw = g(w)dw$$

3. **全纯1形式空间**：
   $$H^0(R, \Omega^1) = \{\text{全纯1形式}\}, \quad \dim H^0(R, \Omega^1) = g$$

4. **上同调群**：
   $$H^1(R, \mathbb{C}) = H^{1,0}(R) \oplus H^{0,1}(R), \quad H^{1,0} = H^0(R, \Omega^1)$$

5. **周期矩阵**：
   $$\Pi = \begin{pmatrix} A & B \end{pmatrix}, \quad A, B \in M_{g \times g}(\mathbb{C})$$

6. **Riemann双线性关系**：
   $$A^T B = B^T A, \quad \text{Im}(A^{-1}B) > 0$$

7. **Abel-Jacobi映射**：
   $$\Phi: R \to J(R) = \mathbb{C}^g / \Lambda, \quad \Phi(p) = \leqqqqqqft(\int_{p_0}^p \omega_1, \ldots, \int_{p_0}^p \omega_g\right)$$

8. **Jacobi簇**：
   $$J(R) = \mathbb{C}^g / \Lambda, \quad \Lambda = \mathbb{Z}^g + \Pi \mathbb{Z}^g$$

9. **Riemann-Roch（1形式）**：
   $$h^0(R, \Omega^1) = g, \quad h^1(R, \Omega^1) = 1$$

10. **周期积分**：
    $$\int_{\gamma} \omega = \sum_{i=1}^g \leqqqqqqft(a_i \int_{\alpha_i} \omega + b_i \int_{\beta_i} \omega\right), \quad \gamma = \sum a_i \alpha_i + b_i \beta_i$$
