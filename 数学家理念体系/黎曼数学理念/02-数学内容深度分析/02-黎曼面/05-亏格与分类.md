# 亏格与分类：黎曼面的拓扑不变量

## 一、亏格

### 1.1 定义

**亏格**：

```
紧致黎曼面R的亏格g：

g = (1/2) dim H¹(R, C)

几何意义：
- 洞的个数
- 拓扑不变量
- 分类工具
```

---

### 1.2 例子

**例子**：

```
- g = 0：P¹（球面）
- g = 1：环面
- g ≥ 2：高亏格曲面
```

**分类**：

## 三、Riemann-Hurwitz公式

### 3.1 公式

**公式**：

```
f: R → S（分支覆盖）

2g(R) - 2 = d(2g(S) - 2) + ∑(e_p - 1)

其中：
- d：度数
- e_p：分支指数
```

### 3.2 应用
