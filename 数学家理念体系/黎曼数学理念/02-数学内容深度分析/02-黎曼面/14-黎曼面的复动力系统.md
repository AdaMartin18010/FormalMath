
## 二、Julia集

### 2.1 定义

**Julia集**：

```
映射f的Julia集：

J(f) = {z | 轨道不稳定}

性质：
- 闭集
- 完全不变
- 应用广泛
```

---

### 2.2 性质

## 三、Mandelbrot集

### 3.1 定义

**Mandelbrot集**：

```
二次映射族：

M = {c ∈ ℂ | J(f_c)连通}

其中：
f_c(z) = z² + c

性质：
- 分形
- 应用广泛
```
