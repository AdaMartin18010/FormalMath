## 一、复动力系统

### 1.1 定义

**复动力系统**：

```
映射f: ℂ̂ → ℂ̂：

迭代：
z_{n+1} = f(z_n)

其中：
- ℂ̂：黎曼球面
- 应用广泛
```

点分类：

## 二、Julia集

### 2.1 定义

**Julia集**：

```
映射f的Julia集：

J(f) = {z | 轨道不稳定}

性质：
- 闭集
- 完全不变
- 应用广泛
```

---

### 2.2 性质

**性质**：

- 分形结构
- 自相似
- 应用广泛

---

## 三、Mandelbrot集

### 3.1 定义

**Mandelbrot集**：

```
二次映射族：

M = {c ∈ ℂ | J(f_c)连通}

其中：
f_c(z) = z² + c

性质：
- 分形
- 应用广泛
```

### 3.2 应用

**应用**：

- 复动力系统
- 分形几何
