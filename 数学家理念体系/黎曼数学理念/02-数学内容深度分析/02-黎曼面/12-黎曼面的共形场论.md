## 一、共形对称性

### 1.1 定义

**共形变换**：

```
黎曼面R上的共形变换：

度规变换：
g' = e^{φ} g

其中：
- φ：共形因子
- 保持角度
- 应用广泛
```

---

### 1.2 共形场

**共形场**：

```
共形场φ(z, z̄)：

变换：
φ'(w, w̄) = (∂w/∂z)^{-h} (∂w̄/∂z̄)^{-h̄} φ(z, z̄)

其中：
- (h, h̄)：共形权重
- 应用广泛
```

## 二、Virasoro代数

### 2.1 定义

**Virasoro代数**：

```
生成元L_n：

[L_m, L_n] = (m-n) L_{m+n} + (c/12) m(m²-1) δ_{m+n,0}

其中：
- c：中心荷
- 量子修正
- 应用广泛
```

## 三、算子乘积展开

### 3.1 OPE

**OPE**：

```
算子乘积：

A(z) B(w) = ∑_k C_{AB}^k (z-w)^{h_k - h_A - h_B} O_k(w)

其中：
- C_{AB}^k：结构常数
- 应用广泛
```
