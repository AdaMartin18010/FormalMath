## 一、解析延拓

### 1.1 原始定义

**Dirichlet级数**：

```text
ζ(s) = ∑_{n=1}^∞ 1/n^s（Re(s) > 1）

收敛性：
- Re(s) > 1：绝对收敛
- Re(s) = 1：发散
- Re(s) < 1：未定义
```

**问题**：

```text
需要：
- 延拓到全平面
- 研究零点
- 函数方程
```

---

### 1.2 积分表示

**Mellin变换**：

```text
ζ(s) = (1/Γ(s)) ∫₀^∞ x^{s-1}/(e^x - 1) dx

其中：
- Re(s) > 1
- Γ(s)：Gamma函数

优势：
- 积分表示
- 便于延拓
```

---

### 1.3 延拓方法

**Hankel围道**：

```text
围道C：
- 沿正实轴
- 绕过原点
- 返回

积分：
ζ(s) = (1/2πi) ∫_C (-z)^{s-1}/(e^z - 1) dz

结果：
- 全平面定义
- 除s=1外全纯
```

## 二、函数方程

### 2.1 Riemann函数方程

**定理**：

```text
ζ(s) = 2^s π^{s-1} sin(πs/2) Γ(1-s) ζ(1-s)

等价形式：
ξ(s) = ξ(1-s)

其中：
ξ(s) = π^{-s/2} Γ(s/2) ζ(s)
```

**性质**：

- 对称性
- 函数性质
- 零点分布

---

### 2.2 证明思路

**方法1：积分表示**：

```text
使用Mellin变换
    ↓
围道积分
    ↓
函数方程
```

**方法2：Theta函数**：

```text
Theta函数：
θ(t) = ∑_{n=-∞}^∞ e^{-πn²t}

关系：
ξ(s) = ∫₀^∞ t^{s/2-1} (θ(t)-1) dt

对称性：
θ(1/t) = √t θ(t)
    ↓
函数方程
```

## 三、特殊值

### 3.1 正整数

**Euler结果**：

```text
ζ(2) = π²/6
ζ(4) = π⁴/90
ζ(6) = π⁶/945

一般：
ζ(2n) = (-1)^{n+1} B_{2n} (2π)^{2n} / (2(2n)!)

其中：
B_{2n}：Bernoulli数
```

---

### 3.2 负整数

**函数方程**：

```text
ζ(-n) = (-1)^n B_{n+1}/(n+1)

例子：
ζ(0) = -1/2
ζ(-1) = -1/12
ζ(-2) = 0
ζ(-3) = 1/120
```

**物理意义**：

```text
ζ(-1) = -1/12：
- 弦论中的正规化
- 1+2+3+... = -1/12（形式）
```

---

### 3.3 临界值

**s = 1/2**：

```text
ζ(1/2) ≈ -1.4603545...

意义：
- 临界线
- RH相关
```

## 四、零点

### 4.1 平凡零点

**定义**：

```text
ζ(-2n) = 0（n = 1, 2, 3, ...）

原因：
sin(πs/2) = 0

性质：
- 显式
- 已知
- "平凡"
```

---

### 4.2 非平凡零点

**定义**：

```text
非平凡零点：
0 < Re(s) < 1（临界带）

性质：
- 无限多个（Hardy, 1914）
- 对称：s ↔ 1-s̄
- 在临界线上？(RH)
```

**已知结果**：

```text
临界线上：
- 无限多个（Hardy, 1914）
- 至少40%（Conrey, 1989）

数值：
- 前10¹³个零点
- 全部在临界线上
```

## 六、推广

### 6.1 Dirichlet L函数

**定义**：

```text
L(s, χ) = ∑_{n=1}^∞ χ(n)/n^s

函数方程：
L(s, χ) = ε(χ) L(1-s, χ̄)

其中：
ε(χ)：Gauss和
```

---

### 6.2 Dedekind ζ函数

**定义**：

```text
数域K：
ζ_K(s) = ∑_I N(I)^{-s}

函数方程：
ξ_K(s) = ξ_K(1-s)

其中：
ξ_K(s) = |Δ_K|^{s/2} (函数)
```

## 十、数学公式总结

### 核心公式

1. **ζ函数定义**：
   $$\zeta(s) = \sum_{n=1}^\infty \frac{1}{n^s} \quad (\text{Re}(s) > 1)$$

2. **Mellin变换**：
   $$\zeta(s) = \frac{1}{\Gamma(s)} \int_0^\infty \frac{x^{s-1}}{e^x - 1} dx$$

3. **Riemann函数方程**：
   $$\zeta(s) = 2^s \pi^{s-1} \sin\\\left(\frac{\pi s}{2}\right) \Gamma(1-s) \zeta(1-s)$$

4. **对称形式（ξ函数）**：
   $$\xi(s) = \xi(1-s), \quad \xi(s) = \frac{1}{2} s(s-1) \pi^{-s/2} \Gamma(s/2) \zeta(s)$$

5. **函数方程等价形式**：
   $$\pi^{-s/2} \Gamma(s/2) \zeta(s) = \pi^{-(1-s)/2} \Gamma((1-s)/2) \zeta(1-s)$$

6. **解析延拓**：
   $$\zeta(s) = \frac{1}{2\pi i} \int_C \frac{(-z)^{s-1}}{e^z - 1} dz \text{（Hankel围道）}$$

7. **特殊值（正整数）**：
   $$\zeta(2n) = \frac{(-1)^{n+1} B_{2n} (2\pi)^{2n}}{2(2n)!}$$

8. **特殊值（负整数）**：
   $$\zeta(-2n) = 0, \quad \zeta(1-2n) = \frac{B_{2n}}{2n}$$

9. **函数方程应用**：
   $$\zeta(1-s) = 2(2\pi)^{-s} \cos\\\left(\frac{\pi s}{2}\right) \Gamma(s) \zeta(s)$$

10. **零点对称性**：
    $$\zeta(s) = 0 \iff \zeta(1-s) = 0 \text{（函数方程）}$$
