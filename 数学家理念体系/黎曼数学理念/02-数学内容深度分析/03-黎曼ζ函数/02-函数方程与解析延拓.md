## 一、解析延拓

### 1.1 原始定义

**Dirichlet级数**：

```text
ζ(s) = ∑_{n=1}^∞ 1/n^s（Re(s) > 1）

收敛性：
- Re(s) > 1：绝对收敛
- Re(s) = 1：发散
- Re(s) < 1：未定义
```

**问题**：

```text
需要：
- 延拓到全平面
- 研究零点
- 函数方程
```

---

### 1.2 积分表示

**Mellin变换**：

```text
ζ(s) = (1/Γ(s)) ∫₀^∞ x^{s-1}/(e^x - 1) dx

其中：
- Re(s) > 1
- Γ(s)：Gamma函数

优势：
- 积分表示
- 便于延拓
```

---

### 1.3 延拓方法

**Hankel围道**：

```text
围道C：
- 沿正实轴
- 绕过原点
- 返回

积分：
ζ(s) = (1/2πi) ∫_C (-z)^{s-1}/(e^z - 1) dz

结果：
- 全平面定义
- 除s=1外全纯
```

## 二、函数方程

### 2.1 Riemann函数方程

**定理**：

```text
ζ(s) = 2^s π^{s-1} sin(πs/2) Γ(1-s) ζ(1-s)

等价形式：
ξ(s) = ξ(1-s)

其中：
ξ(s) = π^{-s/2} Γ(s/2) ζ(s)
```

**性质**：

- 对称性
- 函数性质
- 零点分布

---

### 2.2 证明思路

**方法1：积分表示**：

```text
使用Mellin变换
    ↓
围道积分
    ↓
函数方程
```

**方法2：Theta函数**：

```text
Theta函数：
θ(t) = ∑_{n=-∞}^∞ e^{-πn²t}

关系：
ξ(s) = ∫₀^∞ t^{s/2-1} (θ(t)-1) dt

对称性：
θ(1/t) = √t θ(t)
    ↓
函数方程
```

## 十、数学公式总结

### 核心公式

1. **ζ函数定义**：
   $$\zeta(s) = \sum_{n=1}^\infty \frac{1}{n^s} \quad (\text{Re}(s) > 1)$$

2. **Mellin变换**：
   $$\zeta(s) = \frac{1}{\Gamma(s)} \int_0^\infty \frac{x^{s-1}}{e^x - 1} dx$$

3. **Riemann函数方程**：
   $$\zeta(s) = 2^s \pi^{s-1} \sin\left(\frac{\pi s}{2}\right) \Gamma(1-s) \zeta(1-s)$$

4. **对称形式（ξ函数）**：
   $$\xi(s) = \xi(1-s), \quad \xi(s) = \frac{1}{2} s(s-1) \pi^{-s/2} \Gamma(s/2) \zeta(s)$$

5. **函数方程等价形式**：
   $$\pi^{-s/2} \Gamma(s/2) \zeta(s) = \pi^{-(1-s)/2} \Gamma((1-s)/2) \zeta(1-s)$$

6. **解析延拓**：
   $$\zeta(s) = \frac{1}{2\pi i} \int_C \frac{(-z)^{s-1}}{e^z - 1} dz \text{（Hankel围道）}$$

7. **特殊值（正整数）**：
   $$\zeta(2n) = \frac{(-1)^{n+1} B_{2n} (2\pi)^{2n}}{2(2n)!}$$

8. **特殊值（负整数）**：
   $$\zeta(-2n) = 0, \quad \zeta(1-2n) = \frac{B_{2n}}{2n}$$

9. **函数方程应用**：
   $$\zeta(1-s) = 2(2\pi)^{-s} \cos\left(\frac{\pi s}{2}\right) \Gamma(s) \zeta(s)$$

10. **零点对称性**：
    $$\zeta(s) = 0 \iff \zeta(1-s) = 0 \text{（函数方程）}$$

11. **Hankel围道积分**：
    使用Hankel围道进行解析延拓：
    $$\zeta(s) = \frac{\Gamma(1-s)}{2\pi i} \int_C \frac{(-z)^{s-1}}{e^z - 1} dz$$

12. **Theta函数的函数方程**：
    Theta函数的对称性：
    $$\theta(1/t) = \sqrt{t} \theta(t), \quad \theta(t) = \sum_{n=-\infty}^\infty e^{-\pi n^2 t}$$

13. **ξ函数的积分表示**：
    通过Theta函数表示ξ函数：
    $$\xi(s) = \int_0^\infty t^{s/2-1} (\theta(t) - 1) dt$$

14. **函数方程的证明（Theta方法）**：
    使用Theta函数的对称性证明函数方程：
    $$\xi(s) = \xi(1-s) \text{（通过Theta函数对称性）}$$

15. **解析延拓的留数**：
    在$s=1$处的留数：
    $$\text{Res}_{s=1} \zeta(s) = 1$$

## 十一、解析延拓与函数方程的详细数学表述

### 11.1 Hankel围道方法的严格构造

**Hankel围道**：

使用**Hankel围道**$C$进行解析延拓：
$$\zeta(s) = \frac{\Gamma(1-s)}{2\pi i} \int_C \frac{(-z)^{s-1}}{e^z - 1} dz$$

其中围道$C$从$+\infty$沿正实轴到$\delta$，沿小圆绕过原点，再沿正实轴返回$+\infty$。

**解析延拓**：

这个积分定义了对所有$s \neq 1$的$\zeta(s)$，并且是全纯的。

**数学公式**：
- Hankel围道积分：$$\zeta(s) = \frac{\Gamma(1-s)}{2\pi i} \int_C \frac{(-z)^{s-1}}{e^z - 1} dz$$
- 解析延拓：$$\zeta(s) \text{ 全纯于 } \mathbb{C} \setminus \{1\}$$
- 留数：$$\text{Res}_{s=1} \zeta(s) = 1$$

---

### 11.2 Theta函数方法的严格证明

**Theta函数**：

定义**Theta函数**：
$$\theta(t) = \sum_{n=-\infty}^\infty e^{-\pi n^2 t}, \quad t > 0$$

**对称性**：

Theta函数满足对称性：
$$\theta(1/t) = \sqrt{t} \theta(t)$$

**ξ函数的积分表示**：

通过Theta函数，ξ函数可以表示为：
$$\xi(s) = \int_0^\infty t^{s/2-1} (\theta(t) - 1) dt$$

**函数方程的证明**：

使用Theta函数的对称性，可以证明：
$$\xi(s) = \xi(1-s)$$

**数学公式**：
- Theta函数：$$\theta(t) = \sum_{n=-\infty}^\infty e^{-\pi n^2 t}$$
- 对称性：$$\theta(1/t) = \sqrt{t} \theta(t)$$
- ξ函数积分：$$\xi(s) = \int_0^\infty t^{s/2-1} (\theta(t) - 1) dt$$

### 11.3 函数方程的应用

**零点对称性**：

函数方程$\xi(s) = \xi(1-s)$意味着：
$$\zeta(s) = 0 \iff \zeta(1-s) = 0$$

**临界线对称**：

如果$\rho$是零点，则$\bar{\rho}$和$1-\bar{\rho}$也是零点。

**研究策略**：

由于函数方程，只需研究$\text{Re}(s) \geq 1/2$的零点即可。

**数学公式**：
- 零点对称：$$\zeta(s) = 0 \iff \zeta(1-s) = 0$$
- 临界线对称：$$\text{Re}(\rho) = \frac{1}{2} \iff \text{Re}(1-\bar{\rho}) = \frac{1}{2}$$
- 研究范围：$$\text{只需研究 } \text{Re}(s) \geq \frac{1}{2} \text{ 的零点}$$
