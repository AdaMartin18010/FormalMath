## 一、Dirichlet级数

### 1.1 定义

**Dirichlet级数**：

```
级数：
D(s) = ∑_{n=1}^∞ a_n n^{-s}

其中：
- a_n：系数
- s：复变量
- 收敛域：Re(s) > σ_a
```

**性质**：

- 解析性质
- 收敛性
- 应用广泛

### 1.2 收敛性

**收敛域**：

```
绝对收敛域：
Re(s) > σ_a

其中：
σ_a = inf{σ | ∑|a_n|n^{-σ} < ∞}
```

**应用**：

- 解析性质
- 数论应用
- 现代研究

## 二、ζ函数

### 2.1 定义

**Riemann ζ函数**：

```
ζ(s) = ∑_{n=1}^∞ n^{-s}

收敛域：
Re(s) > 1

性质：
- Dirichlet级数
- 解析性质
- 应用广泛
```

**历史**：

- Euler（1737）
- Riemann（1859）
- 现代发展

### 2.2 解析延拓

**解析延拓**：

```
ζ(s)可延拓到整个复平面（除s=1）

函数方程：
ζ(s) = 2^s π^{s-1} sin(πs/2) Γ(1-s) ζ(1-s)
```

**应用**：

- 解析性质
- 数论应用
- 现代研究
