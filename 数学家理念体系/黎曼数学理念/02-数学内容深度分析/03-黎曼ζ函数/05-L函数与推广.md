# L函数与推广：ζ函数的现代发展

## 一、Dirichlet L函数

### 1.1 定义

**Dirichlet L函数**：

```
L(s, χ) = ∑_{n=1}^∞ χ(n)/n^s

其中：
- χ：Dirichlet特征
- s > 1

性质：
- 类似ζ函数
- 解析延拓
- 函数方程
```

---

### 1.2 函数方程

**函数方程**：

```
L(s, χ) = ε(χ) L(1-s, χ̄)

其中：
ε(χ)：Gauss和

性质：
- 对称性
- 零点分布
```

---

## 二、广义黎曼假设

### 2.1 GRH

**广义RH**：

```
L(s, χ)的所有非平凡零点满足：
Re(s) = 1/2

应用：
- 素数分布
- 算法（素性检验）
```

---

### 2.2 应用

**应用**：

- 数论
- 算法
- 密码学

---

## 三、Dedekind ζ函数

### 3.1 定义

**Dedekind ζ函数**：

```
数域K：
ζ_K(s) = ∑_I N(I)^{-s}

其中：
- I：理想
- N(I)：范数

性质：
- 类似ζ函数
- 解析延拓
- 函数方程
```

---

### 3.2 扩展RH

**扩展RH**：

```
所有非平凡零点在Re(s) = 1/2

应用：
- 代数数论
- 类域论
```

---

## 四、Artin L函数

### 4.1 定义

**Artin L函数**：

```
数域K的Galois表示ρ：

L(s, ρ) = ∏_p det(1 - ρ(Frob_p) p^{-s})^{-1}

其中：
- Frob_p：Frobenius
- ρ：表示

性质：
- 解析延拓
- 函数方程
```

---

### 4.2 应用

**应用**：

- Langlands纲领
- 数论
- 现代研究

---

## 五、自守L函数

### 5.1 定义

**自守L函数**：

```
GL(n)的自守表示π：

L(s, π) = ∏_p L_p(s, π)

性质：
- 解析延拓
- 函数方程
- 应用广泛
```

---

### 5.2 Langlands纲领

**对应**：

```
Galois表示
    ⟺
自守表示

对应：
- L函数
- 零点
- 统一框架
```
