## 一、Dirichlet L函数

### 1.1 定义

**Dirichlet L函数**：

```text
L(s, χ) = ∑_{n=1}^∞ χ(n)/n^s

其中：
- χ：Dirichlet特征
- s > 1

性质：
- 类似ζ函数
- 解析延拓
- 函数方程
```

---

### 1.2 函数方程

**函数方程**：

```text
L(s, χ) = ε(χ) L(1-s, χ̄)

其中：
ε(χ)：Gauss和

性质：
- 对称性
- 零点分布
```

---

应用：

## 三、Dedekind ζ函数

### 3.1 定义

**Dedekind ζ函数**：

```text
数域K：
ζ_K(s) = ∑_I N(I)^{-s}

其中：
- I：理想
- N(I)：范数

性质：
- 类似ζ函数
- 解析延拓
- 函数方程
```

---

### 3.2 扩展RH

**扩展RH**：

```text
所有非平凡零点在Re(s) = 1/2

应用：
- 代数数论
- 类域论
```

## 四、Artin L函数

### 4.1 定义

**Artin L函数**：

```text
数域K的Galois表示ρ：

L(s, ρ) = ∏_p det(1 - ρ(Frob_p) p^{-s})^{-1}

其中：
- Frob_p：Frobenius
- ρ：表示

性质：
- 解析延拓
- 函数方程
```

---

### 4.2 应用

**应用**：

- Langlands纲领
- 数论
- 现代研究

## 八、数学公式总结

### 核心公式

1. **Dirichlet L函数**：
   $$L(s, \chi) = \sum_{n=1}^\infty \frac{\chi(n)}{n^s} = \prod_p \left(1 - \frac{\chi(p)}{p^s}\right)^{-1}$$

2. **函数方程（Dirichlet）**：
   $$L(s, \chi) = \varepsilon(\chi) L(1-s, \bar{\chi}), \quad \varepsilon(\chi) = \frac{\tau(\chi)}{i^a \sqrt{q}}$$

3. **Dedekind ζ函数**：
   $$\zeta_K(s) = \sum_{\mathfrak{a}} \frac{1}{N(\mathfrak{a})^s} = \prod_{\mathfrak{p}} \left(1 - \frac{1}{N(\mathfrak{p})^s}\right)^{-1}$$

4. **Artin L函数**：
   $$L(s, \rho) = \prod_{\mathfrak{p}} \det\left(1 - \rho(\text{Frob}_\mathfrak{p}) N(\mathfrak{p})^{-s}\right)^{-1}$$

5. **自守L函数**：
   $$L(s, \pi) = \prod_p L_p(s, \pi_p), \quad \pi \text{ 自守表示}$$

6. **Langlands对应**：
   $$\text{Galois表示} \leqftrightarrow \text{自守表示}$$

7. **广义黎曼假设（GRH）**：
   $$L(s, \chi) \text{ 的所有非平凡零点满足 } \text{Re}(s) = \frac{1}{2}$$

8. **函数方程（一般）**：
   $$L(s) = \varepsilon(s) L^*(1-s), \quad L^*(s) = L(\bar{s})$$

9. **特殊值**：
   $$L(1, \chi) = \frac{\pi}{\sqrt{q}} \sum_{a=1}^{q-1} \chi(a) \cot\left(\frac{\pi a}{q}\right)$$

10. **零点分布**：
    $$N_L(T) = \frac{d}{2\pi} T \log T + O(T), \quad d = \text{函数阶}$$

11. **Dirichlet特征的Gauss和**：
    Dirichlet特征的Gauss和：
    $$\tau(\chi) = \sum_{a=1}^{q-1} \chi(a) e^{2\pi i a / q}$$

12. **函数方程的根因子**：
    Dirichlet L函数的函数方程：
    $$L(s, \chi) = \varepsilon(\chi) L(1-s, \bar{\chi}), \quad \varepsilon(\chi) = \frac{\tau(\chi)}{i^a \sqrt{q}}$$

13. **Dedekind ζ函数的函数方程**：
    Dedekind ζ函数的完整函数方程：
    $$\xi_K(s) = |\Delta_K|^{s/2} \Gamma_{\mathbb{R}}(s)^{r_1} \Gamma_{\mathbb{C}}(s)^{r_2} \zeta_K(s)$$

14. **Artin L函数的局部因子**：
    Artin L函数的局部因子：
    $$L_p(s, \rho) = \det(1 - \rho(\text{Frob}_p) p^{-s})^{-1}$$

15. **Langlands对应的L函数**：
    Langlands对应下的L函数相等：
    $$L(s, \rho) = L(s, \pi) \text{（如果对应成立）}$$

## 九、L函数的详细数学表述

### 9.1 Dirichlet L函数的严格构造

**Dirichlet特征**：

**Dirichlet特征**$\chi: \mathbb{Z} \to \mathbb{C}$是模$q$的乘法特征：
$$\chi(n+q) = \chi(n), \quad \chi(mn) = \chi(m)\chi(n)$$

**Dirichlet L函数**：

$$L(s, \chi) = \sum_{n=1}^\infty \frac{\chi(n)}{n^s} = \prod_p \left(1 - \frac{\chi(p)}{p^s}\right)^{-1}$$

**Gauss和**：

**Gauss和**定义为：
$$\tau(\chi) = \sum_{a=1}^{q-1} \chi(a) e^{2\pi i a / q}$$

**函数方程**：

$$L(s, \chi) = \varepsilon(\chi) L(1-s, \bar{\chi})$$

其中：
$$\varepsilon(\chi) = \frac{\tau(\chi)}{i^a \sqrt{q}}$$

**数学公式**：

- Dirichlet L函数：$$L(s, \chi) = \sum_{n=1}^\infty \frac{\chi(n)}{n^s}$$
- Gauss和：$$\tau(\chi) = \sum_{a=1}^{q-1} \chi(a) e^{2\pi i a / q}$$
- 函数方程：$$L(s, \chi) = \varepsilon(\chi) L(1-s, \bar{\chi})$$

---

### 9.2 Dedekind ζ函数的严格构造

**数域的ζ函数**：

对于数域$K$，**Dedekind ζ函数**定义为：
$$\zeta_K(s) = \sum_{\mathfrak{a}} \frac{1}{N(\mathfrak{a})^s} = \prod_{\mathfrak{p}} \left(1 - \frac{1}{N(\mathfrak{p})^s}\right)^{-1}$$

其中$\mathfrak{a}$遍历$K$的非零理想，$\mathfrak{p}$遍历素理想。

**函数方程**：

$$\xi_K(s) = |\Delta_K|^{s/2} \Gamma_{\mathbb{R}}(s)^{r_1} \Gamma_{\mathbb{C}}(s)^{r_2} \zeta_K(s)$$

满足：
$$\xi_K(s) = \xi_K(1-s)$$

其中$r_1$和$r_2$分别是实嵌入和复嵌入的个数。

**数学公式**：

- Dedekind ζ函数：$$\zeta_K(s) = \sum_{\mathfrak{a}} \frac{1}{N(\mathfrak{a})^s}$$
- 函数方程：$$\xi_K(s) = |\Delta_K|^{s/2} \Gamma_{\mathbb{R}}(s)^{r_1} \Gamma_{\mathbb{C}}(s)^{r_2} \zeta_K(s)$$
- 对称性：$$\xi_K(s) = \xi_K(1-s)$$

---

### 9.3 Artin L函数与Langlands纲领

**Artin L函数**：

对于数域$K$的Galois表示$\rho: \text{Gal}(\bar{K}/K) \to \text{GL}_n(\mathbb{C})$，**Artin L函数**定义为：
$$L(s, \rho) = \prod_{\mathfrak{p}} \det\left(1 - \rho(\text{Frob}_\mathfrak{p}) N(\mathfrak{p})^{-s}\right)^{-1}$$

**Langlands对应**：

**Langlands纲领**预测Galois表示与自守表示之间的对应：
$$\text{Galois表示} \rho \longleftrightarrow \text{自守表示} \pi$$

**L函数相等**：

如果对应成立，则：
$$L(s, \rho) = L(s, \pi)$$

**数学公式**：

- Artin L函数：$$L(s, \rho) = \prod_{\mathfrak{p}} \det\left(1 - \rho(\text{Frob}_\mathfrak{p}) N(\mathfrak{p})^{-s}\right)^{-1}$$
- Langlands对应：$$\rho \longleftrightarrow \pi$$
- L函数相等：$$L(s, \rho) = L(s, \pi)$$

## 十、Langlands纲领的详细发展

### 10.1 Langlands对应的严格表述

**Galois表示**：

对于数域$K$，**Galois表示**是连续同态：
$$\rho: \text{Gal}(\bar{K}/K) \to \text{GL}_n(\mathbb{C})$$

**自守表示**：

**自守表示**$\pi$是$\text{GL}_n(\mathbb{A}_K)$的不可约表示。

**Langlands对应**：

**Langlands纲领**预测：
$$\text{Galois表示} \rho \longleftrightarrow \text{自守表示} \pi$$

**L函数相等**：

如果对应成立，则：
$$L(s, \rho) = L(s, \pi)$$

**数学公式**：

- Galois表示：$$\rho: \text{Gal}(\bar{K}/K) \to \text{GL}_n(\mathbb{C})$$
- Langlands对应：$$\rho \longleftrightarrow \pi$$
- L函数相等：$$L(s, \rho) = L(s, \pi)$$

### 10.2 Motive理论与L函数

**Motive**：

**Motive**$M$是代数几何中的统一对象，对应一个L函数$L(M, s)$。

**L函数**：

$$L(M, s) = \prod_p \frac{1}{P_p(M, p^{-s})}$$

其中$P_p(M, T)$是Motive的局部多项式。

**Riemann假设**：

**Motive的Riemann假设**：$L(M, s)$的所有非平凡零点满足$\text{Re}(s) = \frac{1}{2}$。

**数学公式**：

- Motive：$$M \text{ 是Motive}$$
- L函数：$$L(M, s) = \prod_p \frac{1}{P_p(M, p^{-s})}$$
- Riemann假设：$$\text{Re}(s) = \frac{1}{2}$$

其中$P_p(M, T)$是Motive的局部多项式。
