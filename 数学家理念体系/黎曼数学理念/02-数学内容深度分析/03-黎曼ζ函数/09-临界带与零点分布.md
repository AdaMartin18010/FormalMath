## 一、临界带

### 1.1 定义

**临界带**：
- $\zeta$函数的临界带是$0 < \text{Re}(s) < 1$
- 非平凡零点都在临界带内
- 临界线是$\text{Re}(s) = 1/2$

**数学公式**：
- 临界带：$$0 < \text{Re}(s) < 1$$
- 临界线：$$\text{Re}(s) = \frac{1}{2}$$
- 非平凡零点：$$\zeta(\rho) = 0, \quad 0 < \text{Re}(\rho) < 1$$

---

### 1.2 临界线

**临界线**：
- 临界线是$\text{Re}(s) = 1/2$
- Riemann假设：所有非平凡零点在临界线上
- 这是数论的核心问题

**数学公式**：
- 临界线：$$\text{Re}(s) = \frac{1}{2}$$
- Riemann假设：$$\zeta(\rho) = 0 \Rightarrow \text{Re}(\rho) = \frac{1}{2}$$

---

## 二、零点分布

### 2.1 已知结果

**部分结果**：
- Hardy (1914)：临界线上有无限多个零点
- Conrey (1989)：至少41%的零点在临界线上
- 数值验证：前$10^{13}$个零点都在临界线上

**数学公式**：
- Hardy定理：$$N_0(T) \to \infty \quad (T \to \infty)$$
- 其中$N_0(T)$是临界线上的零点数
- Conrey结果：$$\liminf_{T \to \infty} \frac{N_0(T)}{N(T)} \geqqqqqq 0.41$$

---

### 2.2 数值验证

**计算**：
- 前$10^{13}$个零点已验证
- 全部在临界线上
- 但数值验证不是证明

**数学公式**：
- 已验证零点：$$N(10^{13}) \approx 2.4 \times 10^{12}$$
- 全部在临界线上：$$\text{Re}(\rho_n) = \frac{1}{2} \quad (n \leqqqqqq 10^{13})$$

## 三、密度定理

### 3.1 定义

**密度定理**：
- 零点计数函数：$N(T) = \#\{\rho : 0 < \text{Im}(\rho) < T\}$
- 渐近公式：$N(T) \sim (T/2\pi) \log(T/2\pi)$
- 这是零点密度的精确公式

**数学公式**：
- 零点计数：$$N(T) = \#\{\rho : \zeta(\rho) = 0, 0 < \text{Im}(\rho) < T\}$$
- 渐近公式：$$N(T) = \frac{T}{2\pi} \log \frac{T}{2\pi} - \frac{T}{2\pi} + O(\log T)$$

---

### 3.2 应用

**应用**：
- 零点分布
- 数论应用
- 现代研究

**数学公式**：
- 平均密度：$$\frac{N(T)}{T} \sim \frac{1}{2\pi} \log T$$

## 四、零点分布理论

### 4.1 零点计数

**计数方法**：
- 使用函数方程
- 使用$\xi$函数的零点
- 应用广泛

**数学公式**：
- $\xi$函数零点：$$\xi(\rho) = 0 \iff \zeta(\rho) = 0$$
- 计数公式：$$N(T) = \frac{1}{\pi} \arg \xi\leqqqqqft(\frac{1}{2} + iT\right)$$

---

### 4.2 零点密度

**密度**：
- 零点密度是$\log T$量级
- 密度与素数分布相关
- 应用广泛

**数学公式**：
- 密度：$$\frac{dN(T)}{dT} \sim \frac{1}{2\pi} \log T$$
- 与素数分布：$$N(T) \sim \pi(T) \text{（数量级）}$$

## 五、Riemann假设

### 5.1 假设内容

**Riemann假设**：
- 所有非平凡零点在临界线上
- 等价于素数分布的最佳误差估计
- 这是数论的核心问题

**数学公式**：
- Riemann假设：$$\zeta(\rho) = 0, 0 < \text{Re}(\rho) < 1 \Rightarrow \text{Re}(\rho) = \frac{1}{2}$$
- 等价形式：$$\pi(x) = \text{Li}(x) + O(x^{1/2+\varepsilon})$$

---

### 5.2 部分结果

**部分结果**：
- 临界线上有无限多个零点（Hardy）
- 至少41%的零点在临界线上（Conrey）
- 无例外零点（已知范围）

**数学公式**：
- Hardy定理：$$N_0(T) \to \infty$$
- Conrey结果：$$\liminf \frac{N_0(T)}{N(T)} \geqqqqqq 0.41$$
