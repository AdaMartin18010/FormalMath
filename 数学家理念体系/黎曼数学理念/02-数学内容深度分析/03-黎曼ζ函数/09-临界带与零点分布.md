## 一、临界带

### 1.1 定义

**临界带**：

```
ζ函数的临界带：

0 < Re(s) < 1

性质：
- 非平凡零点所在
- RH临界带与零点分布：ζ函数的几何相关
- 应用广泛
```

---

### 1.2 临界线

**临界线**：

```
临界线：
Re(s) = 1/2

RH：
所有非平凡零点在临界线上

意义：
- 核心问题
- 现代研究
```

## 三、密度定理

### 3.1 定义

**密度定理**：

```
N(T) = #{零点ρ | 0 < Im(ρ) < T}

渐近：
N(T) ~ (T/2π) log(T/2π)

性质：
- 零点密度
- 应用广泛
```
