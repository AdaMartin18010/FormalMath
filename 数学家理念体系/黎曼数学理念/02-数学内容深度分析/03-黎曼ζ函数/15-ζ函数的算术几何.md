## 二、ζ函数的几何解释

### 2.1 Hasse-Weil ζ函数

**定义**：

```
代数簇X的Hasse-Weil ζ函数：
ζ_X(s) = ∏_{p} (1 - N_p p^{-s})^{-1}

其中：
- N_p = |X(F_p)|
- 算术性质
```

**性质**：

- 解析延拓
- 函数方程
- 应用广泛

### 2.2 Weil猜想

```
Weil猜想：
- 函数方程
- Riemann假设（有限域版本）
- 应用广泛
```

**证明**：
