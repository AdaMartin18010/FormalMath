# ζ函数的解析性质：全纯性与奇点

## 一、全纯性

### 1.1 定义域

**定义域**：

```
ζ(s)的定义：

Re(s) > 1：
ζ(s) = ∑_{n=1}^∞ n^{-s}

解析延拓：
全平面（除s=1）

性质：
- 全纯
- 应用广泛
```

---

### 1.2 奇点

**奇点**：

```
s = 1：
- 单极点
- 留数 = 1
- 应用广泛
```

## 二、函数方程

### 2.1 对称性

**函数方程**：

```
ξ(s) = ξ(1-s)

ξ(s) = π^{-s/2} Γ(s/2) ζ(s)

性质：
- 对称性
- 零点分布
- 应用广泛
```
