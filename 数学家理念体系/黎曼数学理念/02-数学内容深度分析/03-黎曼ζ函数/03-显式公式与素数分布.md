## 一、显式公式

### 1.1 Riemann-von Mangoldt公式

**公式**：

```text
ψ(x) = x - ∑_ρ x^ρ/ρ - log(2π) - (1/2)log(1-x^{-2})

其中：
- ψ(x)：Chebyshev函数
- ρ：ζ的零点
- 求和：按|Im(ρ)|排序
```

**意义**：

- 连接零点与素数
- 显式关系
- 误差估计

---

### 1.2 Chebyshev函数

**定义**：

```text
ψ(x) = ∑_{n≤x} Λ(n)

其中：
Λ(n) = log p（n = p^k）
     = 0（其他）

性质：
- 与素数显式公式与素数分布：ζ函数的数论应用相关
- 便于分析
```

## 三、素数定理

### 3.1 定理

**素数定理**：

```text
π(x) ~ Li(x) = ∫_2^x dt/log(t)

等价：
π(x) ~ x/log x
```

**证明**：

- Hadamard, de la Vallée-Poussin（1896）
- 使用ζ函数
- 无零点在Re(s) = 1

---

### 3.2 误差项

**误差**：

```text
π(x) - Li(x) = O(x^{1/2+ε})

RH ⟺ 误差最优：
π(x) - Li(x) = O(x^{1/2} log x)
```

## 十、数学公式总结

### 核心公式

1. **Riemann-von Mangoldt显式公式**：
   $$\psi(x) = x - \sum_{\rho} \frac{x^\rho}{\rho} - \log(2\pi) - \frac{1}{2}\log(1-x^{-2})$$

2. **Chebyshev函数**：
   $$\psi(x) = \sum_{n \leq x} \Lambda(n), \quad \Lambda(n) = \begin{cases} \log p & n = p^k \\ 0 & \text{其他} \end{cases}$$

3. **素数计数函数**：
   $$\pi(x) = \#\{p \leq x : p \text{ 是素数}\}$$

4. **素数定理**：
   $$\pi(x) \sim \frac{x}{\log x} \text{ 或等价地 } \psi(x) \sim x$$

5. **误差项**：
   $$\pi(x) = \text{Li}(x) + O(x^{1/2+\varepsilon}) \text{（若RH成立）}$$

6. **Li函数**：
   $$\text{Li}(x) = \int_2^x \frac{dt}{\log t} = \frac{x}{\log x} + \frac{x}{(\log x)^2} + \cdots$$

7. **零点求和**：
   $$\sum_{\rho} \frac{x^\rho}{\rho} = \sum_{\text{Im}(\rho) > 0} \left(\frac{x^\rho}{\rho} + \frac{x^{\bar{\rho}}}{\bar{\rho}}\right)$$

8. **Mertens函数**：
   $$M(x) = \sum_{n \leq x} \mu(n), \quad M(x) = O(x^{1/2+\varepsilon}) \iff \text{RH}$$

9. **von Mangoldt函数关系**：
   $$\psi(x) = \sum_{p^k \leq x} \log p = \sum_{n \leq x} \Lambda(n)$$

10. **显式公式的零点项**：
    $$\text{误差项主要由零点项 } \sum_{\rho} x^\rho/\rho \text{ 控制}$$

11. **Perron公式**：
    使用Perron公式推导显式公式：
    $$\psi(x) = \frac{1}{2\pi i} \int_{c-i\infty}^{c+i\infty} -\frac{\zeta'(s)}{\zeta(s)} \frac{x^s}{s} ds$$

12. **留数计算**：
    显式公式通过计算留数得到：
    $$\text{Res}_{s=1} + \sum_\rho \text{Res}_{s=\rho} + \text{其他项}$$

13. **von Mangoldt函数的生成函数**：
    $$-\frac{\zeta'(s)}{\zeta(s)} = \sum_{n=1}^\infty \frac{\Lambda(n)}{n^s}, \quad \text{Re}(s) > 1$$

14. **零点项的收敛性**：
    零点项的条件收敛：
    $$\sum_\rho \frac{x^\rho}{\rho} = \lim_{T \to \infty} \sum_{|\text{Im}(\rho)| \leq T} \frac{x^\rho}{\rho}$$

15. **误差项的精确估计**：
    如果RH成立，则：
    $$\pi(x) = \text{Li}(x) + O(\sqrt{x} \log x)$$

## 十、显式公式的详细数学表述

### 10.1 Riemann-von Mangoldt公式的严格推导

**Perron公式**：

使用**Perron公式**可以推导显式公式：
$$\psi(x) = \frac{1}{2\pi i} \int_{c-i\infty}^{c+i\infty} -\frac{\zeta'(s)}{\zeta(s)} \frac{x^s}{s} ds$$

其中$c > 1$。

**留数计算**：

通过计算围道积分的留数：
1. **$s = 1$处的留数**：$\text{Res}_{s=1} = x$
2. **零点处的留数**：$\text{Res}_{s=\rho} = -\frac{x^\rho}{\rho}$
3. **其他项**：$\log(2\pi) + \frac{1}{2}\log(1-x^{-2})$

**数学公式**：
- Perron公式：$$\psi(x) = \frac{1}{2\pi i} \int_{c-i\infty}^{c+i\infty} -\frac{\zeta'(s)}{\zeta(s)} \frac{x^s}{s} ds$$
- 留数计算：$$\psi(x) = \sum_{\text{Res}} = x - \sum_\rho \frac{x^\rho}{\rho} - \log(2\pi) - \frac{1}{2}\log(1-x^{-2})$$
- 生成函数：$$-\frac{\zeta'(s)}{\zeta(s)} = \sum_{n=1}^\infty \frac{\Lambda(n)}{n^s}$$

---

### 10.2 零点项的收敛性

**条件收敛**：

零点项$\sum_\rho \frac{x^\rho}{\rho}$是**条件收敛**的，需要按虚部大小排序求和。

**收敛性证明**：

使用**Abel求和**或**部分求和**方法：
$$\sum_{|\text{Im}(\rho)| \leq T} \frac{x^\rho}{\rho} = \sum_{|\text{Im}(\rho)| \leq T} \frac{x^{\beta + i\gamma}}{\beta + i\gamma}$$

**估计**：

对于$\text{Re}(\rho) = \beta$，有：
$$\left|\frac{x^\rho}{\rho}\right| = \frac{x^\beta}{|\rho|} \leq \frac{x^\beta}{|\gamma|}$$

**数学公式**：
- 条件收敛：$$\sum_\rho \frac{x^\rho}{\rho} = \lim_{T \to \infty} \sum_{|\text{Im}(\rho)| \leq T} \frac{x^\rho}{\rho}$$
- 估计：$$\left|\frac{x^\rho}{\rho}\right| \leq \frac{x^\beta}{|\gamma|}$$
- 收敛性：$$\sum_{|\gamma| \leq T} \frac{1}{|\gamma|} \sim \log T$$

---

### 10.3 误差项的精确估计

**RH与误差估计**：

如果**黎曼假设**成立，即所有非平凡零点满足$\text{Re}(\rho) = \frac{1}{2}$，则：
$$\pi(x) = \text{Li}(x) + O(\sqrt{x} \log x)$$

**证明思路**：

1. 显式公式：$\psi(x) = x - \sum_\rho \frac{x^\rho}{\rho} + \text{小项}$
2. 如果$\text{Re}(\rho) = \frac{1}{2}$，则$x^\rho = x^{1/2} \cdot x^{i\gamma}$
3. 误差项：$O(x^{1/2} \log x)$

**等价性**：

实际上可以证明：
$$\text{RH} \iff \pi(x) = \text{Li}(x) + O(\sqrt{x} \log x)$$

**数学公式**：
- RH成立：$$\text{Re}(\rho) = \frac{1}{2} \Rightarrow x^\rho = x^{1/2} \cdot x^{i\gamma}$$
- 误差估计：$$\pi(x) = \text{Li}(x) + O(\sqrt{x} \log x)$$
- 等价性：$$\text{RH} \iff \pi(x) = \text{Li}(x) + O(\sqrt{x} \log x)$$

### 11.1 素数计数的改进

**显式公式的应用**：

使用显式公式可以改进素数计数：
$$\pi(x) = \text{Li}(x) - \sum_{\rho} \text{Li}(x^\rho) + \text{小项}$$

**数值计算**：

通过计算前$N$个零点，可以得到更精确的估计：
$$\pi(x) \approx \text{Li}(x) - \sum_{|\text{Im}(\rho)| \leq T} \text{Li}(x^\rho)$$

**误差估计**：

误差项为：
$$R(x, T) = O\left(\frac{x \log^2(xT)}{T}\right)$$

**数学公式**：

- 改进公式：$$\pi(x) = \text{Li}(x) - \sum_{\rho} \text{Li}(x^\rho) + R(x, T)$$
- 数值计算：$$\pi(x) \approx \text{Li}(x) - \sum_{|\text{Im}(\rho)| \leq T} \text{Li}(x^\rho)$$
- 误差估计：$$R(x, T) = O\left(\frac{x \log^2(xT)}{T}\right)$$
