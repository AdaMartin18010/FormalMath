# 黎曼几何核心：度规、曲率与测地线

## 一、度规张量

### 1.1 定义

**n维流形上的度规**：

```text
ds² = ∑_{i,j=1}^n g_{ij}(x) dx^i dx^j

其中：
- g_{ij}：度规张量分量
- 对称：g_{ij} = g_{ji}
- 正定：对任意非零向量v，∑g_{ij}v^iv^j > 0

坐标变换：
g'_{kl} = (∂x^i/∂x'^k)(∂x^j/∂x'^l) g_{ij}
```

**矩阵形式**：

```text
g = (g_{ij})_{n×n}

性质：
- 对称矩阵
- 正定矩阵
- 可逆：g^{-1} = (g^{ij})
```

---

### 1.2 距离与角度

**距离**：

```text
两点间距离：
d(P,Q) = ∫_γ √(g_{ij} dx^i dx^j)

其中：
γ：连接P和Q的路径

最短路径：
测地线
```

**角度**：

```text
两向量u, v的夹角：
cos θ = (g_{ij} u^i v^j) / (||u|| ||v||)

其中：
||u|| = √(g_{ij} u^i u^j)
```

---

## 二、联络与协变导数

### 2.1 Christoffel符号

**定义**：

```text
Γ^k_{ij} = (1/2) g^{kl} (∂g_{il}/∂x^j + ∂g_{jl}/∂x^i - ∂g_{ij}/∂x^l)

性质：
- 对称：Γ^k_{ij} = Γ^k_{ji}
- 坐标依赖
- 不是张量
```

**物理意义**：

```text
Γ = 联络系数
- 定义平行移动
- 定义协变导数
- 定义曲率
```

---

### 2.2 协变导数

**定义**：

```text
向量场V的协变导数：
∇_j V^i = ∂V^i/∂x^j + Γ^i_{jk} V^k

张量场T的协变导数：
∇_k T^{ij} = ∂T^{ij}/∂x^k + Γ^i_{kl} T^{lj} + Γ^j_{kl} T^{il}
```

**性质**：

- 张量性
- 满足Leibniz法则
- 与度规相容

---

## 三、曲率张量

### 3.1 黎曼曲率张量

**定义**：

```text
R^i_{jkl} = ∂Γ^i_{jl}/∂x^k - ∂Γ^i_{jk}/∂x^l
          + Γ^i_{mk} Γ^m_{jl} - Γ^i_{ml} Γ^m_{jk}

对称性：
- R_{ijkl} = -R_{jikl}
- R_{ijkl} = -R_{ijlk}
- R_{ijkl} = R_{klij}
- Bianchi恒等式
```

**几何意义**：

```text
曲率 = 度规的非平坦性

R = 0 ⟺ 平坦空间（欧氏）
R ≠ 0 ⟺ 弯曲空间（非欧）
```

---

### 3.2 Ricci曲率与标量曲率

**Ricci张量**：

```text
R_{ij} = R^k_{ikj} = g^{kl} R_{kilj}

性质：
- 对称：R_{ij} = R_{ji}
- 迹：R = g^{ij} R_{ij}（标量曲率）
```

**物理意义**：

```text
爱因斯坦场方程：
R_{ij} - (1/2) R g_{ij} = 8πG T_{ij}

其中：
- R_{ij}：Ricci曲率
- T_{ij}：能量-动量张量
- G：引力常数
```

---

## 四、测地线

### 4.1 定义

**测地线方程**：

```text
d²x^i/dt² + Γ^i_{jk} (dx^j/dt)(dx^k/dt) = 0

性质：
- 局部最短路径
- 自平行（协变导数=0）
- 只依赖度规
```

**变分原理**：

```text
测地线 = 作用量S的极值

S = ∫ √(g_{ij} dx^i dx^j)

Euler-Lagrange方程 → 测地线方程
```

---

### 4.2 应用

**广义相对论**：

```text
自由粒子轨迹 = 测地线

- 无外力时
- 沿测地线运动
- 时空弯曲导致"引力"
```

**几何**：

```text
测地线 = 流形上的"直线"

- 最短路径
- 平行移动
- 几何性质
```

## 五、特殊流形

### 5.1 常曲率空间

**定义**：

```text
常曲率K：
R_{ijkl} = K (g_{ik} g_{jl} - g_{il} g_{jk})

例子：
- K > 0：球面
- K = 0：欧氏空间
- K < 0：双曲空间
```

---

### 5.2 爱因斯坦流形

**定义**：

```text
R_{ij} = λ g_{ij}

其中：
λ：常数

例子：
- 球面
- 双曲空间
- Calabi-Yau流形
```
