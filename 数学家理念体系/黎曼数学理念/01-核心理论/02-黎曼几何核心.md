## 一、度规张量

### 1.1 定义

**n维流形上的度规**：

在n维光滑流形M上，度规是一个2阶协变张量场，在局部坐标 $(x^1, x^2, \ldots, x^n)$ 下表示为：

$$ds^2 = \sum_{i,j=1}^n g_{ij}(x) dx^i dx^j$$

其中：

- $g_{ij}(x)$：度规张量的分量，是流形上点的光滑函数
- $dx^i$：坐标的微分
- $ds^2$：无穷小距离的平方

**度规的基本性质**：

1. **对称性**：$g_{ij} = g_{ji}$（度规是对称张量）
2. **正定性**：对任意非零切向量 $v = (v^1, \ldots, v^n)$，有：
   $$\sum_{i,j=1}^n g_{ij} v^i v^j > 0$$
3. **非退化性**：$\det(g_{ij}) \neqq 0$

**坐标变换下的变换规律**：

在坐标变换 $x^i \to x'^k$ 下，度规分量按协变张量的规律变换：

$$g'_{kl} = \sum_{i,j=1}^n \frac{\partial x^i}{\partial x'^k} \frac{\partial x^j}{\partial x'^l} g_{ij}$$

这保证了度规的几何不变性：虽然分量依赖于坐标选择，但度规本身是流形上的几何对象。

**矩阵形式**：

度规可以表示为 $n \times n$ 对称正定矩阵：

$$g = (g_{ij})_{n \times n}$$

其逆矩阵记为：

$$g^{-1} = (g^{ij})_{n \times n}$$

满足：

$$\sum_{k=1}^n g^{ik} g_{kj} = \delta^i_j$$

其中 $\delta^i_j$ 是Kronecker符号。

**例子：欧氏空间**：

在n维欧氏空间 $\mathbb{R}^n$ 中，标准度规为：

$$ds^2 = (dx^1)^2 + (dx^2)^2 + \cdots + (dx^n)^2$$

即 $g_{ij} = \delta_{ij}$（单位矩阵）。

**例子：球面度规**：

在2维球面 $S^2$ 上，使用球坐标 $(\theta, \phi)$，度规为：

$$ds^2 = R^2 d\theta^2 + R^2 \sin^2\theta d\phi^2$$

其中 $R$ 是球面半径。度规矩阵为：

$$g = \begin{pmatrix} R^2 & 0 \\ 0 & R^2 \sin^2\theta \end{pmatrix}$$

---

### 1.2 距离与角度

**距离的定义**：

度规定义了流形上两点之间的距离。对于流形上的两点 $P$ 和 $Q$，它们之间的距离定义为连接这两点的所有曲线中长度的下确界：

$$d(P, Q) = \inf_{\gamma} \int_\gamma \sqrt{\sum_{i,j=1}^n g_{ij} \frac{dx^i}{dt} \frac{dx^j}{dt}} dt$$

其中 $\gamma: [a,b] \to M$ 是连接 $P$ 和 $Q$ 的曲线，参数化为 $x^i(t)$。

**最短路径：测地线**：

使距离取到最小值的曲线称为**测地线**。测地线是欧氏空间中"直线"概念的推广，满足测地线方程（见第四节）。

**例子：球面上的距离**：

在半径为 $R$ 的球面上，两点之间的最短距离沿着大圆弧（测地线）。如果两点在球面上的球坐标分别为 $(\theta_1, \phi_1)$ 和 $(\theta_2, \phi_2)$，则它们之间的球面距离为：

$$d = R \arccos(\cos\theta_1 \cos\theta_2 + \sin\theta_1 \sin\theta_2 \cos(\phi_1 - \phi_2))$$

**向量的长度**：

在切空间 $T_P M$ 中，向量 $v = (v^1, \ldots, v^n)$ 的长度（范数）定义为：

$$\|v\| = \sqrt{\sum_{i,j=1}^n g_{ij}(P) v^i v^j}$$

**角度**：

两个非零切向量 $u = (u^1, \ldots, u^n)$ 和 $v = (v^1, \ldots, v^n)$ 之间的夹角 $\theta$ 由下式定义：

$$\cos\theta = \frac{\sum_{i,j=1}^n g_{ij} u^i v^j}{\|u\| \|v\|}$$

其中：

- $\|u\| = \sqrt{\sum_{i,j} g_{ij} u^i u^j}$ 是向量 $u$ 的长度
- $\|v\| = \sqrt{\sum_{i,j} g_{ij} v^i v^j}$ 是向量 $v$ 的长度

**例子：欧氏空间中的角度**：

在欧氏空间中，$g_{ij} = \delta_{ij}$，因此角度公式简化为：

$$\cos\theta = \frac{\sum_{i=1}^n u^i v^i}{\sqrt{\sum_{i=1}^n (u^i)^2} \sqrt{\sum_{i=1}^n (v^i)^2}}$$

这就是标准的点积公式。

**体积元**：

度规还定义了流形上的体积元。n维体积元为：

$$dV = \sqrt{\det(g_{ij})} dx^1 dx^2 \cdots dx^n$$

这在积分计算中非常重要。

## 二、联络与协变导数

### 2.1 Christoffel符号

**定义**：

Christoffel符号（也称为Christoffel联络系数）是黎曼几何中的基本量，完全由度规决定：

$$\Gamma^k_{ij} = \frac{1}{2} \sum_{l=1}^n g^{kl} \leqft( \frac{\partial g_{il}}{\partial x^j} + \frac{\partial g_{jl}}{\partial x^i} - \frac{\partial g_{ij}}{\partial x^l} \right)$$

其中 $g^{kl}$ 是度规的逆矩阵元素。

**重要性质**：

1. **对称性**：$\Gamma^k_{ij} = \Gamma^k_{ji}$（对下指标对称）
2. **坐标依赖性**：Christoffel符号不是张量，它们依赖于坐标选择
3. **度规相容性**：由度规唯一确定，满足：
   $$\nabla_k g_{ij} = \frac{\partial g_{ij}}{\partial x^k} - \Gamma^l_{ik} g_{lj} - \Gamma^l_{jk} g_{il} = 0$$

**几何意义**：

Christoffel符号定义了流形上的**联络**（connection），它告诉我们如何将切空间中的向量从一个点"平行移动"到另一个点。这是定义协变导数和曲率的基础。

**例子：欧氏空间**：

在欧氏空间中，$g_{ij} = \delta_{ij}$ 是常数，因此所有Christoffel符号为零：

$$\Gamma^k_{ij} = 0$$

这反映了欧氏空间的"平坦性"。

**例子：球面上的Christoffel符号**：

在2维球面 $S^2$ 上，使用球坐标 $(\theta, \phi)$，度规为 $ds^2 = R^2 d\theta^2 + R^2 \sin^2\theta d\phi^2$，非零的Christoffel符号为：

- $\Gamma^\phi_{\theta\phi} = \Gamma^\phi_{\phi\theta} = \cot\theta$
- $\Gamma^\theta_{\phi\phi} = -\sin\theta \cos\theta$

其他分量均为零。

**物理意义**：

在广义相对论中，Christoffel符号对应于**引力势的梯度**，描述了时空的弯曲程度。它们出现在测地线方程中，决定了自由粒子的运动轨迹。

---

### 2.2 协变导数

**定义**：

协变导数是普通偏导数的推广，它考虑了流形的弯曲性质。对于向量场 $V = (V^1, \ldots, V^n)$，其协变导数为：

$$\nabla_j V^i = \frac{\partial V^i}{\partial x^j} + \sum_{k=1}^n \Gamma^i_{jk} V^k$$

**关键特性**：

1. **张量性**：虽然 $\frac{\partial V^i}{\partial x^j}$ 不是张量，但协变导数 $\nabla_j V^i$ 是张量
2. **坐标无关性**：协变导数的结果不依赖于坐标选择

**张量场的协变导数**：

对于一般的张量场，协变导数遵循以下规则：

- **(1,1)型张量** $T^i_j$：
  $$\nabla_k T^i_j = \frac{\partial T^i_j}{\partial x^k} + \Gamma^i_{kl} T^l_j - \Gamma^l_{kj} T^i_l$$

- **(2,0)型张量** $T^{ij}$：
  $$\nabla_k T^{ij} = \frac{\partial T^{ij}}{\partial x^k} + \Gamma^i_{kl} T^{lj} + \Gamma^j_{kl} T^{il}$$

- **(0,2)型张量** $T_{ij}$：
  $$\nabla_k T_{ij} = \frac{\partial T_{ij}}{\partial x^k} - \Gamma^l_{ki} T_{lj} - \Gamma^l_{kj} T_{il}$$

**Leibniz法则**：

协变导数满足乘积法则。对于两个张量场 $A$ 和 $B$：

$$\nabla_k (A \otimes B) = (\nabla_k A) \otimes B + A \otimes (\nabla_k B)$$

**与度规的相容性**：

度规的协变导数为零：

$$\nabla_k g_{ij} = 0$$

这保证了度规在平行移动下保持不变，是黎曼几何的基本要求。

**几何意义**：

协变导数告诉我们如何沿着曲线"平行移动"向量。如果向量场 $V$ 沿曲线 $\gamma(t)$ 满足：

$$\frac{DV^i}{dt} = \frac{dV^i}{dt} + \Gamma^i_{jk} V^j \frac{dx^k}{dt} = 0$$

则称 $V$ 沿 $\gamma$ 平行移动。测地线就是切向量沿自身平行移动的曲线。

## 三、曲率张量

### 3.1 黎曼曲率张量

**定义**：

黎曼曲率张量是描述流形弯曲程度的核心量。它的定义为：

$$R^i_{jkl} = \frac{\partial \Gamma^i_{jl}}{\partial x^k} - \frac{\partial \Gamma^i_{jk}}{\partial x^l} + \sum_{m=1}^n \leqft( \Gamma^i_{mk} \Gamma^m_{jl} - \Gamma^i_{ml} \Gamma^m_{jk} \right)$$

**协变形式**：

将上指标降下来，得到协变曲率张量：

$$R_{ijkl} = \sum_{m=1}^n g_{im} R^m_{jkl}$$

**对称性**：

黎曼曲率张量具有以下对称性质：

1. **反对称性**：$R_{ijkl} = -R_{jikl}$（对前两个指标反对称）
2. **反对称性**：$R_{ijkl} = -R_{ijlk}$（对后两个指标反对称）
3. **对称性**：$R_{ijkl} = R_{klij}$（交换前后两对指标）
4. **Bianchi恒等式**（第一）：
   $$R_{ijkl} + R_{iklj} + R_{iljk} = 0$$
5. **Bianchi恒等式**（第二）：
   $$\nabla_m R_{ijkl} + \nabla_k R_{ijlm} + \nabla_l R_{ijmk} = 0$$

**几何意义**：

曲率张量衡量了流形的**非平坦性**：

- **$R_{ijkl} = 0$** ⟺ 流形是**平坦的**（局部等距于欧氏空间）
- **$R_{ijkl} \neqq 0$** ⟺ 流形是**弯曲的**（非欧几何）

**曲率的直观理解**：

考虑在流形上沿一个小的闭合回路平行移动一个向量。如果流形是平坦的，向量回到起点时方向不变；如果流形是弯曲的，向量会旋转，旋转的角度由曲率张量决定。

**例子：2维曲面的高斯曲率**：

对于2维曲面，黎曼曲率张量只有一个独立分量，与高斯曲率 $K$ 的关系为：

$$R_{1212} = K \det(g_{ij}) = K (g_{11} g_{22} - g_{12}^2)$$

对于球面（半径 $R$），$K = 1/R^2$；对于平面，$K = 0$。

---

### 3.2 Ricci曲率与标量曲率

**Ricci张量**：

Ricci曲率张量是黎曼曲率张量的缩并，定义为：

$$R_{ij} = \sum_{k=1}^n R^k_{ikj} = \sum_{k,l=1}^n g^{kl} R_{kilj}$$

**重要性质**：

1. **对称性**：$R_{ij} = R_{ji}$（Ricci张量是对称的）
2. **迹**：标量曲率定义为：
   $$R = \sum_{i,j=1}^n g^{ij} R_{ij} = \sum_{i=1}^n R^i_i$$

**几何意义**：

Ricci曲率描述了流形在各个方向上的平均弯曲程度。具体来说：

- 如果 $R_{ij} > 0$（正定），流形在该方向上是"凸的"
- 如果 $R_{ij} < 0$（负定），流形在该方向上是"凹的"
- 如果 $R_{ij} = 0$，流形在该方向上是"平坦的"

**标量曲率的几何意义**：

标量曲率 $R$ 是流形上每一点的**总弯曲度**的标量度量。它是Ricci曲率的迹，给出了流形弯曲的总体特征。

**例子：常曲率空间**：

在常曲率 $K$ 的n维流形上：

- Ricci曲率：$R_{ij} = (n-1)K g_{ij}$
- 标量曲率：$R = n(n-1)K$

对于球面 $S^n$（半径 $R$），$K = 1/R^2$，因此 $R = n(n-1)/R^2$。

**物理意义：爱因斯坦场方程**：

在广义相对论中，Ricci曲率和标量曲率出现在爱因斯坦场方程中：

$$R_{ij} - \frac{1}{2} R g_{ij} = \frac{8\pi G}{c^4} T_{ij}$$

其中：

- $R_{ij}$：Ricci曲率张量（描述时空弯曲）
- $R$：标量曲率
- $g_{ij}$：度规张量（描述时空结构）
- $T_{ij}$：能量-动量张量（描述物质分布）
- $G$：引力常数
- $c$：光速

这个方程表明：**时空的弯曲（几何）由物质的分布（物理）决定**，实现了几何与物理的深刻统一。

**爱因斯坦张量**：

方程左边的量 $G_{ij} = R_{ij} - \frac{1}{2} R g_{ij}$ 称为**爱因斯坦张量**，它具有重要的性质：

$$\sum_{j=1}^n \nabla_j G^{ij} = 0$$

这保证了能量-动量守恒。

## 四、测地线

### 4.1 定义

**测地线方程**：

测地线是流形上的"直线"，定义为满足测地线方程的曲线 $\gamma(t) = (x^1(t), \ldots, x^n(t))$：

$$\frac{d^2 x^i}{dt^2} + \sum_{j,k=1}^n \Gamma^i_{jk} \frac{dx^j}{dt} \frac{dx^k}{dt} = 0$$

**关键性质**：

1. **局部最短路径**：在足够小的邻域内，测地线是连接两点的最短路径
2. **自平行性**：测地线的切向量沿自身平行移动，即：
   $$\frac{D}{dt} \leqft( \frac{dx^i}{dt} \right) = 0$$
   其中 $\frac{D}{dt}$ 是沿曲线的协变导数
3. **内蕴性**：测地线只依赖于度规，不依赖于嵌入方式

**变分原理**：

测地线可以通过变分原理得到。定义作用量：

$$S[\gamma] = \int_a^b \sqrt{\sum_{i,j=1}^n g_{ij} \frac{dx^i}{dt} \frac{dx^j}{dt}} dt = \int_a^b L dt$$

其中 $L = \sqrt{g_{ij} \dot{x}^i \dot{x}^j}$ 是拉格朗日量。测地线使作用量 $S$ 取极值。

**Euler-Lagrange方程**：

应用变分法，得到Euler-Lagrange方程：

$$\frac{d}{dt} \frac{\partial L}{\partial \dot{x}^i} - \frac{\partial L}{\partial x^i} = 0$$

这等价于测地线方程。

**例子：欧氏空间中的测地线**：

在欧氏空间中，$\Gamma^i_{jk} = 0$，因此测地线方程简化为：

$$\frac{d^2 x^i}{dt^2} = 0$$

解为 $x^i(t) = x^i_0 + v^i t$，即**直线**。

**例子：球面上的测地线**：

在2维球面上，测地线是**大圆**（通过球心的平面与球面的交线）。例如，赤道 $\phi = \text{常数}$ 是一条测地线。

---

### 4.2 应用

**广义相对论中的应用**：

在广义相对论中，测地线具有根本重要性：

1. **自由粒子的运动**：
   - 在无外力作用下，自由粒子沿着时空的测地线运动
   - 这实现了"引力是几何"的思想：引力不是力，而是时空弯曲的效应

2. **光线的传播**：
   - 光线沿着**零测地线**（$ds^2 = 0$）传播
   - 这解释了引力透镜效应：光线在引力场中弯曲

3. **测地偏差方程**：
   两条相邻测地线的分离由测地偏差方程描述：
   $$\frac{D^2 \xi^i}{dt^2} + R^i_{jkl} \frac{dx^j}{dt} \xi^k \frac{dx^l}{dt} = 0$$
   其中 $\xi^i$ 是两条测地线的分离向量。这描述了潮汐力。

**几何中的应用**：

1. **最短路径问题**：
   测地线提供了流形上两点之间的最短路径（在局部意义下）

2. **指数映射**：
   从点 $p$ 出发，沿所有方向的测地线定义了**指数映射**：
   $$\exp_p: T_p M \to M$$
   这是流形局部几何的重要工具

3. **测地坐标系**：
   使用测地线可以构造特殊的坐标系（Riemann法坐标），简化计算

**计算几何中的应用**：

在计算机图形学和计算几何中，测地线用于：

- 曲面上的路径规划
- 纹理映射
- 形状分析

**例子：地球表面的最短路径**：

在地球表面（近似为球面），两点之间的最短路径是沿着**大圆弧**。这就是为什么飞机航线看起来是弯曲的原因。

```

```

## 九、数学公式总结

### 核心公式

1. **度规张量**：
   $$ds^2 = \sum_{i,j} g_{ij} dx^i dx^j$$

2. **协变导数**：
   $$\nabla_j V^i = \frac{\partial V^i}{\partial x^j} + \sum_k \Gamma^i_{jk} V^k$$

3. **黎曼曲率张量**：
   $$R^i_{jkl} = \frac{\partial \Gamma^i_{jl}}{\partial x^k} - \frac{\partial \Gamma^i_{jk}}{\partial x^l} + \sum_m \leqft( \Gamma^i_{mk} \Gamma^m_{jl} - \Gamma^i_{ml} \Gamma^m_{jk} \right)$$

4. **Ricci张量**：
   $$R_{ij} = \sum_k R^k_{ikj}$$

5. **标量曲率**：
   $$R = \sum_{i,j} g^{ij} R_{ij}$$

6. **测地线方程**：
   $$\frac{d^2 x^i}{dt^2} + \sum_{j,k} \Gamma^i_{jk} \frac{dx^j}{dt} \frac{dx^k}{dt} = 0$$

7. **Einstein场方程**：
   $$R_{ij} - \frac{1}{2} R g_{ij} = \frac{8\pi G}{c^4} T_{ij}$$

8. **Bianchi恒等式**：
   $$R_{ijkl} + R_{iklj} + R_{iljk} = 0, \quad \nabla_m R_{ijkl} + \nabla_k R_{ijlm} + \nabla_l R_{ijmk} = 0$$

9. **常曲率空间**：
   $$R_{ijkl} = K (g_{ik} g_{jl} - g_{il} g_{jk})$$

10. **Einstein流形**：
    $$R_{ij} = \lambda g_{ij}, \quad \lambda \text{ 常数}$$
