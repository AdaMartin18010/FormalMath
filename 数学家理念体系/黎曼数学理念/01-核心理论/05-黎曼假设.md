# 黎曼假设：数论的核心猜想

## 一、历史背景

### 1.1 素数分布问题

**Euler（1737）**：

```
ζ(s) = ∑_{n=1}^∞ 1/n^s = ∏_p (1 - p^{-s})^{-1}

其中：
- s > 1（实数）
- p：素数

意义：
- 连接级数与乘积
- 素数分布
```

**Dirichlet（1837）**：

```
L函数：
L(s, χ) = ∑_{n=1}^∞ χ(n)/n^s

应用：
- 算术级数中素数
- 解析数论
```

---

### 1.2 黎曼的突破（1859）

**论文**：

```
《论小于给定值的素数个数》

核心：
- ζ函数的解析延拓
- 函数方程
- 零点分布
- 黎曼假设
```

## 二、ζ函数

### 2.1 定义

**Riemann ζ函数**：

```
ζ(s) = ∑_{n=1}^∞ 1/n^s（Re(s) > 1）

解析延拓：
- 全平面（除s=1）
- 单极点：s=1（留数1）

函数方程：
ζ(s) = 2^s π^{s-1} sin(πs/2) Γ(1-s) ζ(1-s)
```

---

### 2.2 性质

**特殊值**：

```
ζ(2) = π²/6
ζ(4) = π⁴/90
ζ(-1) = -1/12
ζ(0) = -1/2

负偶数：
ζ(-2n) = 0（n≥1）
```

**零点**：

```
平凡零点：
s = -2, -4, -6, ...

非平凡零点：
0 < Re(s) < 1（临界带）
```

---

## 三、黎曼假设

### 3.1 陈述

**假设**（RH）：

```
ζ(s)的所有非平凡零点满足：
Re(s) = 1/2

等价：
所有非平凡零点在临界线上
```

**等价形式**：

```
1. 素数计数函数：
π(x) = Li(x) + O(x^{1/2+ε})

2. Mertens函数：
M(x) = O(x^{1/2+ε})

3. 其他数论函数
```

---

### 3.2 已知结果

**部分结果**：

```
1. 临界线上有无限多个零点（Hardy, 1914）
2. 至少1/3零点在临界线上（Selberg, 1942）
3. 至少40%零点在临界线上（Levinson, 1974）
4. 至少41%零点在临界线上（Conrey, 1989）
```

**数值验证**：

```
前10¹³个零点：
- 全部在临界线上
- 计算验证
- 但非证明
```

## 四、与素数分布

### 4.1 显式公式

**Riemann-von Mangoldt公式**：

```
ψ(x) = x - ∑_ρ x^ρ/ρ - log(2π) - (1/2)log(1-x^{-2})

其中：
- ψ(x)：Chebyshev函数
- ρ：ζ的零点
- 求和：按|Im(ρ)|排序
```

**意义**：

- 连接零点与素数
- 显式关系
- 误差估计

---

### 4.2 误差项

**素数定理**：

```
π(x) ~ Li(x) = ∫_2^x dt/log(t)

误差：
π(x) - Li(x) = O(x^{1/2+ε})

RH ⟺ 误差最优
```

## 五、推广

### 5.1 广义黎曼假设

```
应用：

```

---

### 5.2 扩展ζ函数

**Dedekind ζ函数**：

```
数域K：
ζ_K(s) = ∑_I N(I)^{-s}

其中：
- I：理想
- N(I)：范数

扩展RH：
所有非平凡零点在Re(s) = 1/2
```

## 七、Motive理论

### 7.1 Weil猜想

**Weil（1949）**：

```
有限域F_q上的代数簇X：

Zeta函数：
Z(X, T) = ∏_{i=0}^{2n} P_i(T)^{(-1)^{i+1}}

其中：
P_i(T) = det(1 - Frob·T | H^i)

Riemann假设（Weil）：
|α_{ij}| = q^{i/2}
```

**Deligne证明（1974）**：

- 使用Motive理论
- 代数几何方法
- Fields奖
