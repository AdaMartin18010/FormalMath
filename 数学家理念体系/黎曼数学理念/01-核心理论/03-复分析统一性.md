## 一、历史背景

### 1.1 复分析的早期发展

**Cauchy的工作（1820s）**：

```text
复函数理论：
- 复可微性
- Cauchy积分定理
- 留数理论

但：
- 多值函数问题
- 分支点处理
- 需要新方法
```

**黎曼的突破（1851）**：

```text
博士论文：
《复变函数论的基础》

核心思想：
- 黎曼面
- 多值函数的单值化
- 拓扑与分析的统一
```

---

### 1.2 黎曼面的概念

**基本思想**：

```text
多值函数问题：
√z, log z, arcsin z

解决：
- 构造"覆盖空间"
- 多值 → 单值
- 拓扑结构
```

---

## 二、黎曼面的定义

### 2.1 基本定义

**黎曼面**：

```text
1维复流形R：

1. 局部同胚于C
2. 复结构（全纯坐标）
3. 连通（通常）

性质：
- 2维实流形
- 可定向
- 紧致或非紧致
```

**例子**：

```text
- 复平面C
- 黎曼球面P¹ = C ∪ {∞}
- 环面（椭圆曲线）
- 高亏格曲面
```

---

### 2.2 全纯函数

**定义**：

```text
f: R → C全纯 ⟺
局部表示为全纯函数

等价：
- Cauchy-Riemann方程
- 复可微
- 解析
```

**性质**：

- 开映射
- 最大值原理
- 零点孤立

---

## 三、单值化定理

### 3.1 基本单值化

**定理**（Poincaré-Koebe, 1907）：

```text
单连通黎曼面R：

R ≅
- C（复平面）
- P¹（黎曼球面）
- D（单位圆盘）

分类：
- 椭圆型：P¹
- 抛物型：C
- 双曲型：D
```

**意义**：

- 单连通完全分类
- 万有覆盖
- 几何结构

---

### 3.2 一般单值化

**定理**：

```text
任意黎曼面R：

R ≅ D/Γ

其中：
- D：万有覆盖（C, P¹, 或D）
- Γ：Fuchsian群（离散自同构群）

分类：
- 亏格0：P¹
- 亏格1：C/Λ（环面）
- 亏格≥2：D/Γ（双曲）
```

## 四、代数函数

### 4.1 定义

**代数函数**：

```text
F(z,w) = 0

其中：
F：多项式

例子：
w² = z³ + az + b（椭圆曲线）
wⁿ = P(z)（超椭圆曲线）
```

**黎曼面**：

```text
代数函数 → 黎曼面

F(z,w) = 0
    ↓
{(z,w) | F(z,w) = 0}
    ↓
紧致黎曼面
```

---

### 4.2 亏格

**定义**：

```text
紧致黎曼面R的亏格g：

g = (1/2) dim H¹(R, C)

几何意义：
- 洞的个数
- 拓扑不变量
- 分类工具
```

**Riemann-Hurwitz公式**：

```text
f: R → S（分支覆盖）

2g(R) - 2 = d(2g(S) - 2) + ∑(e_p - 1)

其中：
- d：度数
- e_p：分支指数
```

---

## 五、Abel-Jacobi映射

### 5.1 定义

**Jacobi簇**：

```text
紧致黎曼面R（亏格g）：

J(R) = H⁰(R, Ω¹)* / H₁(R, ℤ)

性质：
- g维复环面
- Abel群
- 全纯结构
```

**Abel映射**：

```text
A: R → J(R)

A(P) = (∫_{P₀}^P ω₁, ..., ∫_{P₀}^P ω_g)

其中：
{ω₁, ..., ω_g}：全纯1形式基
```

---

### 5.2 Abel定理

**定理**：

```text
除子D = ∑n_i P_i：

D是主除子 ⟺
∑n_i A(P_i) = 0（在J(R)中）

意义：
- 线性等价
- 除子类群
- 几何应用
```

---

## 六、与拓扑的统一

### 6.1 基本群

**对应**：

```text
黎曼面R ↔ 拓扑曲面

基本群：
π₁(R) = 自由群（2g生成元）

关系：
- 亏格 = 拓扑不变量
- 同调群 = 分析工具
```

---

### 6.2 上同调

**全纯上同调**：

```text
H⁰(R, Ω¹) = 全纯1形式

维数：
dim H⁰(R, Ω¹) = g（亏格）

Hodge理论：
H¹(R, C) = H⁰(R, Ω¹) ⊕ H⁰(R, Ω¹)⁻
```

## 七、现代发展

### 7.1 代数几何

**概形**：

```text
黎曼面（1维复流形）
    ↓
代数曲线（1维概形）

统一：
- 全纯函数 → 正则函数
- 上同调 → 层上同调
- 亏格 → 几何亏格
```

---

### 7.2 模空间

**定义**：

```text
M_g = {亏格g紧致黎曼面的模空间}

性质：
- 3g-3维（g≥2）
- 复流形
- 紧化：M̄_g
```

**应用**：

- 分类理论
- 枚举几何
- 物理（弦论）
