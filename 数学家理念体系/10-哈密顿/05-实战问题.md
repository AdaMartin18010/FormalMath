## 第一部分：四元数应用 (Q1-Q5)

### Q1: 四元数基础运算

**问题**：给定四元数 $p = 2 + i + 3j - k$ 和 $q = 1 - 2i + j + 2k$。

(a) 计算 $p + q$, $pq$, 和 $qp$

(b) 计算共轭 $p^*$, $q^*$ 和模 $|p|$, $|q|$

(c) 验证 $(pq)^* = q^* p^*$

**(d) 挑战**：求 $p^{-1}$ 并验证 $pp^{-1} = 1$

<details>
<summary>提示</summary>
使用哈密顿乘积公式。注意 $pq \neq qp$。
</details>

---

### Q2: 三维旋转

**问题**：将向量 $\mathbf{v} = (1, 0, 0)$ 绕轴 $\mathbf{u} = (0, 0, 1)$ 旋转 $90°$。

(a) 构造对应的单位四元数 $q$

(b) 用四元数旋转公式计算 $\mathbf{v}'$

(c) 验证结果与旋转矩阵方法一致

**(d) 挑战**：若连续旋转 $\mathbf{v}$ 先绕 $z$ 轴转 $90°$，再绕 $x$ 轴转 $90°$，求最终位置。

<details>
<summary>提示</summary>
$q = \cos(\theta/2) + \mathbf{u}\sin(\theta/2)$。注意旋转顺序。
</details>

### Q3: SLERP插值

**问题**：给定两个单位四元数表示的旋转：
- $q_0 = 1$ (单位旋转)
- $q_1 = \frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2}k$ (绕 $z$ 轴转 $90°$)

(a) 计算 $\theta$ 使得 $\cos\theta = q_0 \cdot q_1$

(b) 写出 SLERP 公式 $\text{slerp}(q_0, q_1; t)$

(c) 计算 $t = 0.5$ 时的插值结果

**(d) 挑战**：证明 SLERP 保持单位长度，即 $|\text{slerp}(q_0, q_1; t)| = 1$。

### Q5: 纯虚四元数与叉积

**问题**：设 $u = 2i - j + k$，$v = i + 3j - 2k$ 为纯虚四元数。

(a) 计算 $uv$ 并用点积和叉积分解

(b) 验证：$u \cdot v = -\frac{1}{2}(uv + vu)$

(c) 验证：$u \times v = \frac{1}{2}(uv - vu)$

**(d) 挑战**：证明对任意纯虚四元数 $u, v, w$，有雅可比恒等式：$u \times (v \times w) + v \times (w \times u) + w \times (u \times v) = 0$。

## 参考答案与详解

### Q1 解答

**(a)** 基础运算：

$$p + q = 3 - i + 4j + k$$

$$pq = (2)(1) - (1)(-2) - (3)(1) - (-1)(2) = 2 + 2 - 3 + 2 = 3$$

$$+ [2(-2) + 1(1) + 3(2) - (-1)(1)]i = [-4 + 1 + 6 + 1]i = 4i$$

$$+ [2(1) - 1(2) + 3(1) + (-1)(-2)]j = [2 - 2 + 3 + 2]j = 5j$$

$$+ [2(2) + 1(1) - 3(-2) + (-1)(1)]k = [4 + 1 + 6 - 1]k = 10k$$

$$pq = 3 + 4i + 5j + 10k$$

类似计算 $qp = 3 + 0i + 5j + 6k$，验证 $pq \neq qp$。

**(b)** 共轭与模：

$$p^* = 2 - i - 3j + k, \quad q^* = 1 + 2i - j - 2k$$

$$|p| = \sqrt{4 + 1 + 9 + 1} = \sqrt{15}$$

$$|q| = \sqrt{1 + 4 + 1 + 4} = \sqrt{10}$$

**(c)** 验证：

$$(pq)^* = 3 - 4i - 5j - 10k$$

$$q^*p^* = (1 + 2i - j - 2k)(2 - i - 3j + k)$$

展开验证相等。

**(d)** 逆元：

$$p^{-1} = \frac{p^*}{|p|^2} = \frac{2 - i - 3j + k}{15}$$

---

### H1 解答

**(a)** 拉格朗日量与哈密顿量：

$$L = \frac{1}{2}m\dot{x}^2 - \frac{1}{2}kx^2$$

$$p = \frac{\partial L}{\partial \dot{x}} = m\dot{x}$$

$$H = p\dot{x} - L = \frac{p^2}{2m} + \frac{1}{2}kx^2$$

**(b)** 正则方程：

$$\dot{x} = \frac{\partial H}{\partial p} = \frac{p}{m}$$

$$\dot{p} = -\frac{\partial H}{\partial x} = -kx$$

**(c)** 求解：

$$\ddot{x} + \frac{k}{m}x = 0$$

$$x(t) = A\cos(\omega t + \phi), \quad \omega = \sqrt{\frac{k}{m}}$$

**(d)** 相空间：

相轨迹是椭圆：$\frac{p^2}{2mE} + \frac{x^2}{2E/k} = 1$

面积 = $\pi \cdot \sqrt{2mE} \cdot \sqrt{2E/k} = 2\pi E\sqrt{m/k} = \frac{E}{\nu}$（其中 $\nu = \omega/2\pi$）

这正比于能量，与量子化条件 $E = (n + \frac{1}{2})\hbar\omega$ 相关。
