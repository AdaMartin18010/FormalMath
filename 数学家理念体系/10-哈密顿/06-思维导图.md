## 9. 核心公式速查表

### 四元数
```
i² = j² = k² = ijk = -1
ij = k,  jk = i,  ki = j
q = q₀ + q₁i + q₂j + q₃k
q⁻¹ = q*/|q|²
旋转: v' = qvq*
SLERP: slerp(q₀,q₁;t) = sin((1-t)θ)/sin(θ)·q₀ + sin(tθ)/sin(θ)·q₁
```

### 哈密顿力学
```
正则方程: q̇ = ∂H/∂p,  ṗ = -∂H/∂q
泊松括号: {f,g} = Σ(∂f/∂qᵢ·∂g/∂pᵢ - ∂f/∂pᵢ·∂g/∂qᵢ)
运动方程: df/dt = {f,H} + ∂f/∂t
辛形式: ω = Σ dpᵢ ∧ dqᵢ
HJ方程: ∂S/∂t + H(q,∂S/∂q,t) = 0
```

### 光学
```
费马原理: δ∫n ds = 0
程函方程: |∇V|² = n²
光线方程: d/ds(n dr/ds) = ∇n
成像条件: V(P,P') = 常数
正弦条件: n sinθ = n' sinθ'
```

### 图论
```
狄拉克定理: δ(G) ≥ n/2 ⇒ 哈密顿
奥尔定理: σ₂(G) ≥ n ⇒ 哈密顿
```
