## 核心数学贡献

### 1. 四元数 (Quaternions, 1843)

1843年10月16日，哈密顿在布鲁姆桥上刻下了四元数的基本公式：

$$i^2 = j^2 = k^2 = ijk = -1$$

**突破性意义**：
- 首次构建了非交换代数系统
- 为三维空间旋转提供了优雅表示
- 奠定了现代向量分析的基础
- 在计算机图形学、航天工程中广泛应用

### 2. 哈密顿力学 (1834-1835)

哈密顿将拉格朗日力学重新表述为更对称的形式：

$$\dot{q}_i = \frac{\partial H}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial H}{\partial q_i}$$

**理论价值**：
- 揭示了经典力学的几何结构（辛几何）
- 成为统计力学和量子力学的出发点
- 在控制论和优化理论中有重要应用

### 3. 特征函数方法 (1828-1837)

将变分原理引入光学，建立了现代几何光学的数学基础。

### 4. 哈密顿路径与图论

虽然哈密顿本人主要研究伊科西亚游戏（Icosian Calculus），但由此引出的哈密顿路径问题成为图论的核心问题之一。
