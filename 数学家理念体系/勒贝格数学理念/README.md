## 📚 核心内容方向

### 1. 测度论

- **Lebesgue外测度**：基于开区间覆盖的下确界定义
- **Carathéodory可测性条件**：可测集族的$\sigma$-代数结构
- **Lebesgue测度性质**：可数可加性、平移不变性、正则性、完备性
- **Vitali不可测集**：选择公理与测度论边界的经典案例

### 2. 勒贝格积分

- **三层构造**：简单函数积分 → 非负可测函数积分 → 一般可积函数
- **三大收敛定理**：单调收敛定理（MCT）、Fatou引理、控制收敛定理（DCT）
- **与黎曼积分的对比**：可积函数类、分割方法、极限交换条件、狄利克雷函数
- **微积分基本定理**：Lebesgue框架下的绝对连续性与原函数

### 3. 函数空间理论

- **$L^p$空间定义**：$p$次幂可积函数的等价类、范数结构
- **Hölder与Minkowski不等式**：$L^p$空间赋范结构的基础
- **Riesz-Fischer定理（1907）**：$L^p$空间的完备性、泛函分析的诞生
- **$L^2$ Hilbert空间**：内积结构、正交基、Plancherel定理、量子力学应用

### 4. 可测函数

- **可测性定义**：Borel集原像、水平集可测性
- **运算封闭性**：和、积、极限、上确界、下确界的可测性
- **简单函数逼近定理**：可测函数作为简单函数序列的极限
- **Lusin定理**：可测函数的"几乎连续性"
- **Egorov定理**：几乎处处收敛与一致收敛的联系
