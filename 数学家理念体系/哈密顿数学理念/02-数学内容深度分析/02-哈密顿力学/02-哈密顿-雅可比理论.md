# 哈密顿-雅可比理论：哈密顿力学的深化

## 一、哈密顿-雅可比方程的历史起源

哈密顿-雅可比方程是经典力学的三大表述形式之一（另两种为牛顿形式和拉格朗日形式），它将力学的基本问题转化为一个一阶非线性偏微分方程的求解问题。威廉·罗恩·哈密顿（William Rowan Hamilton, 1805-1865）在1834-1835年间的两篇奠基性论文"On a General Method in Dynamics"中首次引入了这一理论框架。卡尔·古斯塔夫·雅各布·雅可比（C. G. J. Jacobi）在19世纪40年代对其进行了系统发展，特别是证明了完整积分与运动常数之间的联系，因此这一理论被后人称为**哈密顿-雅可比理论**。

哈密顿最初的目标是为光学和力学建立一个统一的数学框架。他观察到：光线在介质中的传播与质点在势场中的运动具有深刻的数学相似性——两者都可以通过"最小作用量原理"和"特征函数"来描述。这一洞见最终发展为哈密顿-雅可比方程，并成为连接经典力学与量子力学的重要桥梁。

## 二、哈密顿-雅可比方程的推导与形式

### 2.1 从作用量到H-J方程

考虑 $n$ 自由度力学系统，拉格朗日函数为 $L(q, \dot{q}, t)$。作用量泛函定义为
$$S[q] = \int_{t_0}^{t_1} L(q(t), \dot{q}(t), t)\, dt.$$

固定起点 $(q_0, t_0)$，让终点 $(q, t)$ 变化。定义**Hamilton主函数**
$$S(q, t) = \int_{t_0}^{t} L(q_{\text{cl}}(\tau), \dot{q}_{\text{cl}}(\tau), \tau)\, d\tau,$$
其中 $q_{\text{cl}}$ 为连接 $(q_0, t_0)$ 和 $(q, t)$ 的真实经典轨道。

根据作用量变分原理，Hamilton主函数满足
$$\frac{\partial S}{\partial t} = -H, \quad \frac{\partial S}{\partial q_i} = p_i,$$
其中 $H(q, p, t)$ 为哈密顿函数。将 $p_i = \partial S/\partial q_i$ 代入哈密顿函数，得到

**哈密顿-雅可比方程**：
$$\boxed{\frac{\partial S}{\partial t} + H\left(q, \frac{\partial S}{\partial q}, t\right) = 0}$$

这是一个关于 $S(q, t)$ 的一阶非线性偏微分方程。若哈密顿函数不显含时间，$H = H(q, p)$，则可进行变量分离 $S(q, t) = W(q) - Et$，其中 $E$ 为能量常数。代入H-J方程得**定态哈密顿-雅可比方程**：
$$H\left(q, \frac{\partial W}{\partial q}\right) = E.$$
函数 $W(q)$ 称为**Hamilton特征函数**。

### 2.2 完整积分与Jacobi定理

**定义**：含 $n$ 个独立常数 $\alpha_1, \dots, \alpha_n$ 的解 $S(q, t; \alpha)$ 称为哈密顿-雅可比方程的**完整积分（complete integral）**。

**Jacobi定理**：若已知完整积分 $S(q, t; \alpha)$，且矩阵 $(\partial^2 S / \partial q_i \partial \alpha_j)$ 非奇异，则原哈密顿方程的解可以由以下隐式方程给出：
$$\beta_i = \frac{\partial S}{\partial \alpha_i}, \quad p_i = \frac{\partial S}{\partial q_i},$$
其中 $\beta_i$ 为另一组 $n$ 个常数。

Jacobi定理的深刻之处在于：它将求解 $2n$ 个一阶常微分方程（哈密顿方程）的问题，转化为求解一个一阶偏微分方程的问题。在某些具有足够对称性的系统中（如中心力场、可积系统），变量分离法使得H-J方程可精确求解。

## 三、典型例子：中心力场与开普勒问题

### 3.1 球坐标下的变量分离

考虑质量为 $m$ 的粒子在中心势 $V(r)$ 中运动。在球坐标 $(r, \theta, \phi)$ 下，哈密顿函数为
$$H = \frac{1}{2m}\left(p_r^2 + \frac{p_\theta^2}{r^2} + \frac{p_\phi^2}{r^2 \sin^2\theta}\right) + V(r).$$

设Hamilton特征函数 $W(r, \theta, \phi) = W_r(r) + W_\theta(\theta) + W_\phi(\phi)$。由 $p_\phi = \partial W/\partial \phi = \text{const} = L_z$，得
$$W_\phi(\phi) = L_z \phi.$$

对 $\theta$ 方程，设 $p_\theta^2 + p_\phi^2/\sin^2\theta = L^2$（总角动量平方），则
$$\left(\frac{dW_\theta}{d\theta}\right)^2 + \frac{L_z^2}{\sin^2\theta} = L^2.$$
积分得
$$W_\theta(\theta) = \int \sqrt{L^2 - \frac{L_z^2}{\sin^2\theta}}\, d\theta.$$

最后，径向方程为
$$\frac{1}{2m}\left(\frac{dW_r}{dr}\right)^2 + \frac{L^2}{2mr^2} + V(r) = E,$$
即
$$W_r(r) = \int \sqrt{2m(E - V(r)) - \frac{L^2}{r^2}}\, dr.$$

对于开普勒问题 $V(r) = -k/r$，上述积分可以显式求出（涉及椭圆积分）。轨道方程 $r(\theta)$ 可以通过 $\partial S/\partial L = \text{const}$ 导出，最终给出圆锥曲线轨道。

### 3.2 作用-角变量

对于周期系统（或多周期可积系统），可以引入**作用-角变量** $(J_i, \theta_i)$。作用变量定义为
$$J_i = \frac{1}{2\pi} \oint p_i\, dq_i,$$
积分沿第 $i$ 个自由度的闭合轨道进行。在新变量下，哈密顿函数仅依赖于作用变量 $H = H(J)$，运动方程简化为
$$\dot{J}_i = 0, \quad \dot{\theta}_i = \frac{\partial H}{\partial J_i} = \omega_i(J).$$

角变量以恒定频率 $\omega_i$ 旋转：$\theta_i(t) = \omega_i t + \theta_i(0)$。这一形式在微扰理论（KAM理论）和旧量子论（Bohr-Sommerfeld量子化条件 $J_i = n_i \hbar$）中具有核心地位。

## 四、哈密顿-雅可比理论与量子力学

### 4.1 从经典到量子的过渡

哈密顿-雅可比理论为量子力学提供了最深刻的几何和解析桥梁。1926年，薛定谔在构造他的波动力学时，正是从定态H-J方程 $H(q, \partial W/\partial q) = E$ 出发，通过形式替换 $p_i = \partial W/\partial q_i \to \hat{p}_i = -i\hbar \partial/\partial q_i$，得到了定态薛定谔方程
$$\hat{H}\psi = E\psi.$$

更精确地，设波函数具有形式
$$\psi(q, t) = A(q, t) \exp\left(\frac{i}{\hbar} S(q, t)\right),$$
其中 $A$ 为振幅，$S$ 为相位。将其代入含时薛定谔方程
$$i\hbar \frac{\partial \psi}{\partial t} = \left(-\frac{\hbar^2}{2m}\nabla^2 + V\right)\psi,$$
分离实部和虚部，得到两个方程：
1. **相位方程**（$\hbar \to 0$ 极限）：$\partial S/\partial t + \frac{1}{2m}(\nabla S)^2 + V = 0$，即哈密顿-雅可比方程；
2. **连续性方程**：$\partial A^2/\partial t + \nabla \cdot (A^2 \nabla S/m) = 0$。

这表明：在经典极限 $\hbar \to 0$ 下，量子力学的相位演化精确退化为经典力学的H-J方程。哈密顿主函数 $S$ 因此可以被理解为量子波函数的"经典相位"。

### 4.2 WKB近似

WKB（Wentzel-Kramers-Brillouin）近似是求解一维薛定谔方程的重要半经典方法。其基本假设为波函数具有上述指数形式，且振幅 $A$ 缓变。在势垒区域（$E > V$），近似解为
$$\psi(x) \approx \frac{C}{\sqrt{p(x)}} \exp\left(\pm \frac{i}{\hbar} \int^x p(x')\, dx'\right),$$
其中 $p(x) = \sqrt{2m(E - V(x))}$ 为经典动量。相位中的积分正是哈密顿特征函数 $W(x)$。

WKB近似的适用条件要求势能在一个de Broglie波长内变化很小。在转折点（$E = V$）附近，近似失效，需要通过Airy函数进行连接。WKB量子化条件
$$\oint p\, dq = \left(n + \frac{1}{2}\right) 2\pi \hbar$$
是Bohr-Sommerfeld条件的半经典修正，对于可积系统可以精确给出量子能级。

### 4.3 路径积分表述

费曼在1948年发展的路径积分理论，可以视为对哈密顿-雅可比作用量原理的量子推广。传播子可以写成所有路径的泛函积分：
$$K(q_f, t_f; q_i, t_i) = \int_{q(t_i)=q_i}^{q(t_f)=q_f} e^{iS[q]/\hbar} \mathcal{D}q(t).$$
在经典极限下，$\hbar \to 0$，快速振荡的相位使得只有使作用量取极值的经典路径才有显著贡献——这正是最小作用量原理的量子版本。

## 五、哈密顿-雅可比方程的几何意义

### 5.1 生成函数与辛几何

从现代辛几何观点看，哈密顿-雅可比方程描述的是Lagrange子流形的生成函数。设 $(M, \omega)$ 为辛流形，$T^*Q$ 为构型空间 $Q$ 的余切丛，配备标准辛形式 $\omega = \sum_i dq_i \wedge dp_i$。

一个函数 $S: Q \to \mathbb{R}$ 的图
$$L_S = \left\{\left(q, \frac{\partial S}{\partial q}\right) : q \in Q\right\} \subset T^*Q$$
是一个Lagrange子流形（即 $\omega|_{L_S} = 0$）。哈密顿-雅可比方程
$$\frac{\partial S}{\partial t} + H\left(q, \frac{\partial S}{\partial q}\right) = 0$$
表明 $L_S$ 在哈密顿流 $X_H$ 下的演化保持Lagrange性质。

### 5.2 特征线法

哈密顿-雅可比方程是一个一阶非线性偏微分方程，可以用**特征线法**求解。其特征方程正是哈密顿正则方程：
$$\dot{q}_i = \frac{\partial H}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial H}{\partial q_i}.$$
这意味着：H-J方程的解可以通过经典轨道的"扫掠"来构造。给定初始超曲面 $S(q, 0) = S_0(q)$，从每一点发出的经典轨道族确定了 $S(q, t)$ 在不发生焦散（caustic）的区域内的值。

---

## 六、总结

哈密顿-雅可比方程是经典力学的巅峰成就之一。它将 $2n$ 个一阶常微分方程的求解问题转化为一个偏微分方程的积分问题，在具有足够对称性的系统中可以通过变量分离精确求解。更为重要的是，H-J理论架起了经典力学与量子力学之间的桥梁：薛定谔方程在 $\hbar \to 0$ 极限下退化为H-J方程，WKB近似的核心正是哈密顿特征函数的量子化，而费曼路径积分则是作用量原理的量子推广。从开普勒问题到量子隧穿，从辛几何到偏微分方程，哈密顿-雅可比理论的影响无处不在。
