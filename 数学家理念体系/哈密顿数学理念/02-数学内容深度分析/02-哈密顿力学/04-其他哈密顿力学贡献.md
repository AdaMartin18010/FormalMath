## 一、正则变换与生成函数

哈密顿不仅是哈密顿方程的创立者，也是**正则变换（canonical transformation）**理论的奠基人。正则变换是保持哈密顿方程形式不变的相空间坐标变换，它是经典力学中最强大的分析工具之一。

### 1.1 生成函数方法

哈密顿证明了，任何正则变换都可以从一个**生成函数**导出。常见的生成函数类型包括：
- **$F_1(q, Q, t)$**：旧坐标和新坐标为自变量；
- **$F_2(q, P, t)$**：旧坐标和新动量为自变量；
- **$F_3(p, Q, t)$**：旧动量和新坐标为自变量；
- **$F_4(p, P, t)$**：旧动量和新动量为自变量。

由生成函数，新旧变量之间的关系为：
$$p_i = \frac{\partial F}{\partial q_i}, \quad P_i = -\frac{\partial F}{\partial Q_i}, \quad K = H + \frac{\partial F}{\partial t}$$
（以 $F_1$ 为例）。

### 1.2 恒等变换与点变换

- **恒等变换**由生成函数 $F_2 = \sum_i q_i P_i$ 产生；
- **点变换**（只改变坐标，不改变动量的变换）也是正则变换的特例。

正则变换的重要性在于：它允许我们选择最方便的坐标系来求解力学问题。例如，通过适当的正则变换，可以将可积系统的哈密顿函数化为仅依赖于作用变量的形式。

## 二、泊松括号

### 2.1 定义与性质

哈密顿力学中，**泊松括号（Poisson bracket）** 定义为
$$\{f, g\} = \sum_i \left(\frac{\partial f}{\partial q_i} \frac{\partial g}{\partial p_i} - \frac{\partial f}{\partial p_i} \frac{\partial g}{\partial q_i}\right).$$

对于任意力学量 $A(q, p, t)$，其时间演化由
$$\frac{dA}{dt} = \{A, H\} + \frac{\partial A}{\partial t}$$
给出。特别地，若 $\{A, H\} = 0$ 且 $A$ 不显含时间，则 $A$ 是守恒量。

### 2.2 基本关系

坐标和动量满足基本泊松括号关系：
$$\{q_i, q_j\} = 0, \quad \{p_i, p_j\} = 0, \quad \{q_i, p_j\} = \delta_{ij}.$$

这些关系在量子力学中被直接推广为正则对易关系：
$$[\hat{q}_i, \hat{p}_j] = i\hbar \delta_{ij}.$$

Dirac 在1925年发现了这一对应关系，称之为"量子 Poisson 括号"。这成为正则量子化的核心原则。

## 三、可积系统与刘维尔定理

### 3.1 刘维尔可积性

一个 $n$ 自由度的哈密顿系统称为**刘维尔可积**的，如果存在 $n$ 个相互对合（即两两泊松括号 vanishing）的独立守恒量 $F_1 = H, F_2, \dots, F_n$：
$$\{F_i, F_j\} = 0 \quad \text{对所有 } i, j.$$

**刘维尔定理**断言：对于刘维尔可积系统，运动方程可以通过求积（quadrature）求解。具体地，相空间可以被 action-angle 变量 $(J_i, \theta_i)$ 参数化，其中 $J_i$ 为常数，$\theta_i$ 以恒定频率旋转。

### 3.2 作用-角变量

作用变量定义为
$$J_i = \frac{1}{2\pi} \oint_{\gamma_i} p \, dq,$$
其中 $\gamma_i$ 为 $n$ 维环面上的基本闭合回路。在新变量下，哈密顿函数仅依赖于作用变量 $H = H(J)$，运动方程简化为
$$\dot{J}_i = 0, \quad \dot{\theta}_i = \frac{\partial H}{\partial J_i} = \omega_i(J).$$

这一形式在天体力学、分子动力学和旧量子论中具有核心地位。Bohr-Sommerfeld 量子化条件 $J_i = n_i \hbar$ 正是基于作用-角变量的量子化规则。

## 四、哈密顿力学与辛几何

### 4.1 辛流形的结构

哈密顿力学的现代数学形式是**辛几何**。相空间 $T^*Q$ 具有自然的辛结构
$$\omega = \sum_i dq_i \wedge dp_i.$$

哈密顿向量场 $X_H$ 由
$$\iota_{X_H} \omega = dH$$
定义，其积分曲线就是哈密顿方程的解。

### 4.2 正则变换的辛刻画

正则变换等价于保持辛形式的微分同胚：
$$\phi^* \omega = \omega.$$

这一刻画将哈密顿力学的坐标变换提升为几何对象，使得我们可以用拓扑和几何工具来研究力学系统。

---

## 五、总结

哈密顿对力学的贡献远不止哈密顿方程本身。正则变换理论为坐标选择提供了系统方法；泊松括号成为经典力学与量子力学之间的桥梁；作用-角变量理论为可积系统和半经典量子化提供了核心工具；而辛几何则将哈密顿力学提升为现代数学的一个独立分支。从经典力学到量子场论，从天体力学到最优控制，哈密顿的数学遗产无处不在，持续为人类理解自然界的动力学行为提供最深刻的力量。
