## 一、哈密顿方程的定义

### 1.1 哈密顿函数

**哈密顿函数的定义**：

对于拉格朗日函数 $L(q, \dot{q}, t)$，哈密顿函数定义为：

$$H(q, p, t) = \sum_i p_i \dot{q}_i - L(q, \dot{q}, t)$$

其中 $p_i = \frac{\partial L}{\partial \dot{q}_i}$ 是广义动量。

**Legendre变换**：

哈密顿函数是通过Legendre变换从拉格朗日函数得到的。Legendre变换将变量从 $(q, \dot{q})$ 变换到 $(q, p)$。

**数学表述**：

Legendre变换：
$$H(q, p, t) = \sum_i p_i \dot{q}_i - L(q, \dot{q}, t)$$

其中 $\dot{q}_i$ 通过关系 $p_i = \frac{\partial L}{\partial \dot{q}_i}$ 用 $(q, p)$ 表示。

**例子1：一维谐振子**：

拉格朗日函数：
$$L = \frac{1}{2}m\dot{x}^2 - \frac{1}{2}kx^2$$

广义动量：
$$p = \frac{\partial L}{\partial \dot{x}} = m\dot{x}$$

因此 $\dot{x} = \frac{p}{m}$。

哈密顿函数：
$$H = p\dot{x} - L = p \cdot \frac{p}{m} - \left(\frac{1}{2}m\left(\frac{p}{m}\right)^2 - \frac{1}{2}kx^2\right)$$

$$= \frac{p^2}{m} - \frac{p^2}{2m} + \frac{1}{2}kx^2 = \frac{p^2}{2m} + \frac{1}{2}kx^2$$

这是动能加势能的形式。

### 1.2 哈密顿方程

**哈密顿方程的定义**：

$$\dot{q}_i = \frac{\partial H}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial H}{\partial q_i}$$

**推导**：

从哈密顿函数的全微分：
$$dH = \sum_i \left(\frac{\partial H}{\partial q_i} dq_i + \frac{\partial H}{\partial p_i} dp_i\right) + \frac{\partial H}{\partial t} dt$$

另一方面，从定义 $H = \sum_i p_i \dot{q}_i - L$：
$$dH = \sum_i (p_i d\dot{q}_i + \dot{q}_i dp_i) - dL$$

使用拉格朗日方程 $\frac{d}{dt}\frac{\partial L}{\partial \dot{q}_i} = \frac{\partial L}{\partial q_i}$ 和 $p_i = \frac{\partial L}{\partial \dot{q}_i}$：
$$dL = \sum_i \left(\frac{\partial L}{\partial q_i} dq_i + \frac{\partial L}{\partial \dot{q}_i} d\dot{q}_i\right) = \sum_i (\dot{p}_i dq_i + p_i d\dot{q}_i)$$

因此：
$$dH = \sum_i (\dot{q}_i dp_i - \dot{p}_i dq_i)$$

比较系数得到哈密顿方程。

**数学表述**：

哈密顿方程（标准形式）：
$$\dot{q}_i = \frac{\partial H}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial H}{\partial q_i}$$

哈密顿方程（向量形式）：
$$\begin{pmatrix} \dot{q} \\ \dot{p} \end{pmatrix} = \begin{pmatrix} 0 & I \\ -I & 0 \end{pmatrix} \begin{pmatrix} \frac{\partial H}{\partial q} \\ \frac{\partial H}{\partial p} \end{pmatrix}$$

其中 $I$ 是单位矩阵。

**例子2：一维谐振子的哈密顿方程**：

哈密顿函数：
$$H = \frac{p^2}{2m} + \frac{1}{2}kx^2$$

哈密顿方程：
$$\dot{x} = \frac{\partial H}{\partial p} = \frac{p}{m}$$

$$\dot{p} = -\frac{\partial H}{\partial x} = -kx$$

第二个方程给出 $m\ddot{x} = -kx$，这是谐振子方程。

## 二、哈密顿方程的性质

### 2.1 对称性

**哈密顿方程的对称性**：

哈密顿方程在坐标和动量之间具有对称性。这种对称性反映了相空间的结构。

**对称性表述**：

哈密顿方程：
$$\dot{q}_i = \frac{\partial H}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial H}{\partial q_i}$$

如果将 $(q, p)$ 替换为 $(p, -q)$，方程形式保持不变（除了符号）。

**辛结构**：

相空间具有辛结构，由辛形式：
$$\omega = \sum_i dq_i \wedge dp_i$$

定义。哈密顿方程可以写成：
$$i_{X_H} \omega = dH$$

其中 $X_H$ 是哈密顿向量场。

**数学表述**：

辛形式：
$$\omega = \sum_i dq_i \wedge dp_i$$

哈密顿向量场：
$$X_H = \sum_i \left(\frac{\partial H}{\partial p_i} \frac{\partial}{\partial q_i} - \frac{\partial H}{\partial q_i} \frac{\partial}{\partial p_i}\right)$$

**例子3：相空间的对称性**：

对于一维系统，相空间是 $(q, p)$ 平面。哈密顿方程：
$$\dot{q} = \frac{\partial H}{\partial p}, \quad \dot{p} = -\frac{\partial H}{\partial q}$$

在变换 $(q, p) \mapsto (p, -q)$ 下，方程变为：
$$\dot{p} = \frac{\partial H}{\partial (-q)}, \quad \dot{(-q)} = -\frac{\partial H}{\partial p}$$

这保持了方程的结构。

### 2.2 守恒量

**哈密顿方程与守恒量**：

如果哈密顿函数不显含时间，则能量守恒。

**能量守恒**：

如果 $\frac{\partial H}{\partial t} = 0$，则：
$$\frac{dH}{dt} = \sum_i \left(\frac{\partial H}{\partial q_i} \dot{q}_i + \frac{\partial H}{\partial p_i} \dot{p}_i\right) + \frac{\partial H}{\partial t}$$

使用哈密顿方程：
$$= \sum_i \left(\frac{\partial H}{\partial q_i} \frac{\partial H}{\partial p_i} - \frac{\partial H}{\partial p_i} \frac{\partial H}{\partial q_i}\right) = 0$$

因此 $H$ 是守恒量。

**Noether定理**：

如果哈密顿函数在某个变换下不变，则存在相应的守恒量。

**数学表述**：

能量守恒：
$$\frac{dH}{dt} = 0 \quad \text{（如果 } \frac{\partial H}{\partial t} = 0\text{）}$$

动量守恒：
如果 $H$ 不依赖于 $q_i$，则 $\frac{\partial H}{\partial q_i} = 0$，因此 $\dot{p}_i = 0$，$p_i$ 是守恒量。

**例子4：守恒量**：

对于一维谐振子：
$$H = \frac{p^2}{2m} + \frac{1}{2}kx^2$$

由于 $\frac{\partial H}{\partial t} = 0$，能量 $H$ 守恒。

对于自由粒子：
$$H = \frac{p^2}{2m}$$

由于 $H$ 不依赖于 $x$，动量 $p$ 守恒。
