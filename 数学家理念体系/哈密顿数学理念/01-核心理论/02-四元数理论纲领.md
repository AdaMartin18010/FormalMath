## 一、四元数的发明

### 1.1 历史背景

**四元数发明的背景**：

哈密顿在寻找三维空间中旋转的代数表示时，发现了四元数。在此之前，复数已经成功表示二维旋转，但三维旋转的代数表示一直是一个难题。

**历史问题**：

- **复数与二维旋转**：复数 $z = a + bi$ 可以表示二维旋转，其中 $i^2 = -1$
- **三维旋转的困难**：寻找类似复数的结构来表示三维旋转
- **代数表示的需求**：需要一种代数结构，能够像复数一样自然地表示旋转

**哈密顿的探索**：

哈密顿花费了十多年时间寻找三维旋转的代数表示。他最初尝试三元数（triplets），但发现无法满足所有期望的性质。

**数学表述**：

设 $q = a + bi + cj$ 是三元数，其中 $a, b, c \in \mathbb{R}$，$i, j$ 是单位。问题在于如何定义乘法使得：
- 模长保持：$|q_1 q_2| = |q_1| |q_2|$
- 除法存在：对任意非零 $q$，存在 $q^{-1}$ 使得 $q q^{-1} = 1$

### 1.2 发明过程

**四元数的发明**：

1843年10月16日，哈密顿在都柏林的布鲁姆桥（Broom Bridge）上发现了四元数的乘法规则。这一发现是数学史上的重要时刻。

**发现过程**：

哈密顿意识到，要表示三维旋转，需要四个分量而不是三个。他发现了四元数的基本乘法规则：

$$i^2 = j^2 = k^2 = ijk = -1$$

**关键洞察**：

- **非交换性**：$ij = k$，但 $ji = -k$，因此 $ij \neq ji$
- **循环性**：$ij = k$，$jk = i$，$ki = j$
- **反循环性**：$ji = -k$，$kj = -i$，$ik = -j$

**数学表述**：

四元数的乘法表：

| × | 1 | i | j | k |
|---|---|---|---|---|
| 1 | 1 | i | j | k |
| i | i | -1 | k | -j |
| j | j | -k | -1 | i |
| k | k | j | -i | -1 |

**例子1：四元数乘法**：

设 $q_1 = 1 + 2i + 3j$，$q_2 = 2 - i + j + k$，则：

$$q_1 q_2 = (1 + 2i + 3j)(2 - i + j + k)$$

展开计算：

$$= 2 - i + j + k + 4i - 2i^2 + 2ij + 2ik + 6j - 3ji + 3j^2 + 3jk$$

使用乘法规则 $i^2 = j^2 = -1$，$ij = k$，$ik = -j$，$ji = -k$，$jk = i$：

$$= 2 - i + j + k + 4i + 2 + 2k - 2j + 6j + 3k - 3 - 3i$$

$$= (2 + 2 - 3) + (-1 + 4 - 3)i + (1 - 2 + 6)j + (1 + 2 + 3)k$$

$$= 1 + 0i + 5j + 6k = 1 + 5j + 6k$$

## 二、四元数的性质

### 2.1 四元数的定义

**四元数的定义**：

四元数是形如 $q = a + bi + cj + dk$ 的数，其中 $a, b, c, d$ 是实数，$i, j, k$ 满足：

$$i^2 = j^2 = k^2 = ijk = -1$$

**四元数的结构**：

- **实部**：$a$ 是实部
- **虚部**：$bi + cj + dk$ 是虚部
- **纯四元数**：如果 $a = 0$，则 $q$ 是纯四元数

**数学表述**：

四元数集合：
$$\mathbb{H} = \{q = a + bi + cj + dk \mid a, b, c, d \in \mathbb{R}\}$$

四元数的加法：
$$(a_1 + b_1 i + c_1 j + d_1 k) + (a_2 + b_2 i + c_2 j + d_2 k) = (a_1 + a_2) + (b_1 + b_2)i + (c_1 + c_2)j + (d_1 + d_2)k$$

四元数的乘法（使用分配律和乘法规则）：
$$q_1 q_2 = (a_1 a_2 - b_1 b_2 - c_1 c_2 - d_1 d_2) + (a_1 b_2 + b_1 a_2 + c_1 d_2 - d_1 c_2)i + (a_1 c_2 - b_1 d_2 + c_1 a_2 + d_1 b_2)j + (a_1 d_2 + b_1 c_2 - c_1 b_2 + d_1 a_2)k$$

**例子2：四元数的表示**：

四元数 $q = 1 + 2i + 3j + 4k$ 可以表示为：
- 实部：$1$
- 虚部：$2i + 3j + 4k$
- 向量表示：$(1, 2, 3, 4)$

### 2.2 非交换性

**四元数的非交换性**：

四元数乘法不满足交换律，这是四元数的重要特征。

**非交换性的例子**：

$$ij = k, \quad ji = -k$$

因此 $ij \neq ji$。

**数学表述**：

非交换性：
$$q_1 q_2 \neq q_2 q_1 \quad \text{（一般情况）}$$

交换子：
$$[q_1, q_2] = q_1 q_2 - q_2 q_1$$

对于纯四元数 $u = bi + cj + dk$ 和 $v = b'i + c'j + d'k$：
$$uv - vu = 2(bc' - cb')k + 2(cd' - dc')i + 2(db' - bd')j$$

**例子3：非交换性的计算**：

设 $q_1 = i$，$q_2 = j$，则：
- $q_1 q_2 = ij = k$
- $q_2 q_1 = ji = -k$

因此 $q_1 q_2 \neq q_2 q_1$。

**四元数的其他性质**：

1. **结合性**：$(q_1 q_2) q_3 = q_1 (q_2 q_3)$
2. **分配性**：$q_1(q_2 + q_3) = q_1 q_2 + q_1 q_3$
3. **共轭**：$\bar{q} = a - bi - cj - dk$
4. **模长**：$|q| = \sqrt{a^2 + b^2 + c^2 + d^2}$
5. **逆元**：$q^{-1} = \frac{\bar{q}}{|q|^2}$（如果 $q \neq 0$）

**数学表述**：

共轭：
$$\bar{q} = a - bi - cj - dk$$

模长：
$$|q|^2 = q \bar{q} = a^2 + b^2 + c^2 + d^2$$

逆元：
$$q^{-1} = \frac{\bar{q}}{|q|^2}$$

**例子4：共轭和模长**：

设 $q = 1 + 2i + 3j + 4k$，则：
- 共轭：$\bar{q} = 1 - 2i - 3j - 4k$
- 模长：$|q| = \sqrt{1^2 + 2^2 + 3^2 + 4^2} = \sqrt{30}$
- 逆元：$q^{-1} = \frac{1 - 2i - 3j - 4k}{30}$
