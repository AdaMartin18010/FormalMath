## 一、哈密顿力学在统计力学中的应用

### 1.1 相空间与刘维尔定理

统计力学研究由大量粒子组成的宏观系统的热力学性质。哈密顿力学为统计力学提供了天然的语言：**相空间**。对于 $N$ 粒子系统，相空间的维度为 $6N$（$3N$ 个位置坐标和 $3N$ 个动量坐标）。系统的微观态对应于相空间中的一个点，而宏观态对应于相空间中的概率分布 $\rho(q, p, t)$。

**刘维尔定理**断言：在哈密顿演化下，相空间中的体积元保持不变。用数学语言表述，概率密度沿着哈密顿流满足
$$\frac{d\rho}{dt} = \frac{\partial\rho}{\partial t} + \{\rho, H\} = 0,$$
其中 $\{\cdot, \cdot\}$ 为Poisson括号。这意味着相空间流是不可压缩的，宏观可逆性在微观层次上得到保证。

### 1.2 系综理论与配分函数

Gibbs的系综理论基于相空间上的概率分布：
- **微正则系综**：孤立系统，能量固定，$\rho \propto \delta(E - H)$；
- **正则系综**：与热库接触，温度固定，$\rho \propto e^{-\beta H}$；
- **巨正则系综**：粒子数可变，$\rho \propto e^{-\beta(H - \mu N)}$。

配分函数
$$Z = \int e^{-\beta H(q,p)} \frac{dq\,dp}{N! h^{3N}}$$
是统计力学的核心量，所有热力学量（内能、熵、自由能）都可以通过对 $Z$ 求导得到。这一相空间积分直接依赖于哈密顿函数 $H$ 的形式。

### 1.3 遍历假设与混沌

Boltzmann的**遍历假设**认为：在长时极限下，孤立系统的轨迹会遍历能量面上的所有点，从而时间平均等于系综平均。虽然严格遍历性在一般系统中不成立（Fermi-Pasta-Ulam悖论展示了非遍历行为），但现代统计力学通过**混合性（mixing）**和**混沌**等更弱的概念来理解热化过程。

KAM理论揭示了可积系统中存在大量规则运动（不变环面），而混沌区域则表现为Arnold扩散。对于大量自由度的系统（如 $N \sim 10^{23}$ 的气体），KAM环面的体积趋于零，系统的绝大部分相空间被混沌轨道填充——这为统计力学的有效性提供了某种辩护。

## 二、哈密顿力学在量子力学中的应用

### 2.1 正则量子化

量子力学的标准构建方法——**正则量子化**——直接源于哈密顿力学。其规则为：
- 将经典广义坐标 $q_i$ 和动量 $p_i$ 提升为算符 $\hat{q}_i$ 和 $\hat{p}_i$；
- 要求它们满足正则对易关系：$[\hat{q}_i, \hat{p}_j] = i\hbar \delta_{ij}$；
- 将经典哈密顿函数 $H(q, p)$ 替换为量子哈密顿算符 $\hat{H} = H(\hat{q}, \hat{p})$（注意排序问题）。

含时薛定谔方程
$$i\hbar \frac{\partial \psi}{\partial t} = \hat{H} \psi$$
可以看作是经典哈密顿方程的量子对应。在Heisenberg绘景中，算符随时间的演化
$$\frac{d\hat{A}}{dt} = \frac{i}{\hbar}[\hat{H}, \hat{A}]$$
与经典Poisson括号方程 $\dot{A} = \{H, A\}$ 形成直接类比（Dirac的对应原理）。

### 2.2 路径积分表述

费曼（1948）的**路径积分**量子力学是另一种深刻的量子化方案。传播子（probability amplitude）可以写成所有经典路径的叠加：
$$K(q_f, t_f; q_i, t_i) = \int_{q(t_i)=q_i}^{q(t_f)=q_f} e^{iS[q]/\hbar} \mathcal{D}q(t),$$
其中 $S[q] = \int L(q, \dot{q}) dt$ 为经典作用量。这一表述将量子力学奠基在经典力学的最小作用量原理之上。

在 $\hbar \to 0$ 的经典极限下，快速振荡的相位使得只有使 $S$ 取极值的经典路径才有显著贡献——这正是**驻相近似**和哈密顿-雅可比方程的物理来源。

### 2.3 WKB近似与半经典物理

WKB近似是求解一维薛定谔方程的半经典方法。设波函数为
$$\psi(x) = A(x) e^{iW(x)/\hbar},$$
其中 $W(x)$ 为Hamilton特征函数。代入薛定谔方程后，主导阶给出
$$\frac{1}{2m}\left(\frac{dW}{dx}\right)^2 + V(x) = E,$$
这正是**定态哈密顿-雅可比方程**。WKB量子化条件
$$\oint p\, dx = \left(n + \frac{1}{2}\right) 2\pi \hbar$$
是旧量子论Bohr-Sommerfeld条件的半经典修正，对于可积系统可以给出精确的量子能级。

## 三、相对论性场论与引力

### 3.1 广义相对论中的哈密顿形式

虽然爱因斯坦的广义相对论本质上是一个几何理论，但它也可以写成哈密顿形式——即 **ADM形式**（Arnowitt-Deser-Misner, 1962）。在 $(3+1)$ 分解下，时空度量被分解为
$$ds^2 = -N^2 dt^2 + g_{ij}(dx^i + N^i dt)(dx^j + N^j dt),$$
其中 $N$ 为时移函数（lapse），$N^i$ 为位移函数（shift），$g_{ij}$ 为空间切片上的诱导度量。

ADM哈密顿量为
$$H_{ADM} = \int d^3x \left(N \mathcal{H} + N^i \mathcal{H}_i\right),$$
其中 $\mathcal{H}$ 和 $\mathcal{H}_i$ 为哈密顿约束和动量约束。这些约束是广义相对论的核心结构，反映了理论的规范不变性（微分同胚不变性）。

哈密顿形式在数值相对论（双黑洞合并的模拟）和圈量子引力（LQG）中具有核心地位。在LQG中，Ashtekar变量将引力场的相空间重新参数化，使得约束的量子化变得更加可行。

### 3.2 相对论性粒子的哈密顿形式

相对论性粒子的运动也可以写成哈密顿形式。对于自由粒子，世界线上的作用量为
$$S = -m \int d\tau = -m \int \sqrt{-g_{\mu\nu} \dot{x}^\mu \dot{x}^\nu}\, d\lambda.$$
这一作用量对参数 $\lambda$ 是重参数化不变的，因此存在初级约束（mass-shell约束）：
$$p_\mu p^\mu + m^2 = 0.$$
弦理论中弦的世界面作用量（Nambu-Goto作用量或Polyakov作用量）也是这一思想的直接推广。

## 四、总结

哈密顿力学是现代物理学的通用语言之一。在统计力学中，刘维尔定理和相空间积分奠定了Gibbs系综理论的基础；在量子力学中，正则量子化和路径积分将经典力学提升为量子理论；在广义相对论中，ADM哈密顿形式揭示了时空动力学的约束结构和规范不变性。从原子到星系，从超弦到黑洞，哈密顿力学的结构无处不在。哈密顿在1834年写下的那些方程，经过近两世纪的演化，仍然是物理学家理解自然界最深刻工具之一。
