## 📖 项目概述

本项目致力于全面梳理和呈现**威廉·罗恩·哈密顿**（William Rowan Hamilton, 1805-1865）的数学理念体系。哈密顿是19世纪最重要的数学家和物理学家之一，他发明了四元数，发展了哈密顿力学，对现代数学和物理产生了深远影响。

---

## 🎯 核心主题

### 1. 四元数理论
- 四元数的发明与性质
- 四元数代数
- 四元数在几何中的应用

### 2. 哈密顿力学
- 哈密顿方程
- 哈密顿-雅可比理论
- 经典力学的哈密顿形式

### 3. 光学理论
- 特征函数方法
- 光学系统的数学描述

### 4. 数学哲学
- 数学发现的直觉
- 数学与物理的统一
