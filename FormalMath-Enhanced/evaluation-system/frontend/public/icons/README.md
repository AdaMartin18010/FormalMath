# 图标目录

请将以下尺寸的PNG图标放置在此目录：

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

同时需要在根目录放置：
- favicon.ico (多尺寸: 16x16, 32x32, 64x64)
- logo192.png
- logo512.png
- apple-touch-icon.png

图标建议使用项目品牌色 #1a365d 作为背景色。
