# 同伦类型论 (Homotopy Type Theory)

> **创始人**: Vladimir Voevodsky (2006-2014)
> **核心概念**: 单值基础 (Univalent Foundations) · 高阶归纳类型 (HITs)
> **标志性著作**: "Homotopy Type Theory: Univalent Foundations of Mathematics" (HoTT Book, 2013)
> **与Lean4**: 深刻联系，共享数学基础的新范式

---

## 背景与动机

### 数学基础的危机与重生

**集合论的局限**:

- ZFC集合论作为数学基础已有百年历史
- 但它在处理结构化数学时显得笨拙
- 范畴论提供了更好的抽象，但缺乏"基础"地位

**计算机形式化的挑战**:

- 数学证明的计算机验证日益重要
- 传统基础在证明助手中的表达不够自然
- 需要与数学实践更契合的基础

### Voevodsky的洞察

**Vladimir Voevodsky** (2006年菲尔兹奖得主， motivic同伦论的创立者) 发现：

> Martin-Löf类型论与无穷群胚(∞-groupoids)之间存在深刻联系。

**核心观察**:

- 类型 $A$ 可以看作空间
- 项 $a : A$ 是点
- 恒等类型 $a =_A b$ 是路径空间
- 恒等的恒等是路径的同伦
- ...无限继续

**单值公理** (Univalence Axiom):

> 等价即相等。

形式化地，对于类型 $A, B$:

$$(A =_\mathcal{U} B) \simeq (A \simeq B)$$

这意味着：

- 同构的结构可以被识别
- 不变性自动成立
- 数学实践中"自然"的识别被形式化

### 为什么这很重要？

1. **数学基础的新范式**: 提供与数学直觉更契合的基础
2. **计算机形式化的友好**: 证明助手中的自然表达
3. **同伦论的计算化**: 高维结构的计算处理
4. **数学结构的正确性**: 构造性数学的实现

## 核心概念

### 1. 类型即空间 (Types as Spaces)

**核心类比**:

```
类型论              同伦论
─────────────────────────────────
类型 A              空间 X
项 a : A            点 x ∈ X
等式 a = b          路径 p : x → y
等式的证明          路径的同伦
...                 ...
```

**Brouwer-Heyting-Kolmogorov (BHK) 解释的同伦版本**:

- $A \times B$: 乘积空间
- $A + B$: 不交并
- $A \to B$: 映射空间
- $\prod_{x:A} B(x)$: 截面空间 (dependent product)
- $\sum_{x:A} B(x)$: 全空间 (dependent sum)

### 2. 恒等类型 (Identity Types)

**定义**: 给定类型 $A$ 和 $a, b : A$，恒等类型 $\text{Id}_A(a, b)$（或 $a =_A b$）是...类型的类型？

**归纳原理** (路径归纳):

要证明 $C(x, y, p)$ 对所有 $x, y : A$ 和 $p : x =_A y$ 成立，只需证明 $C(x, x, \text{refl}_x)$ 对所有 $x$ 成立。

**高阶结构**:

- $\text{refl}_a : a = a$ (自反性)
- $p^{-1} : b = a$ (对称性)
- $p \cdot q : a = c$ (传递性)
- 这些运算满足群胚律（直到更高阶同伦）

**无限塔**:

$$A \to (a =_A b) \to (p =_{a=b} q) \to \ldots$$

这形成了一个**弱∞-群胚**的结构。

### 3. 单值公理 (Univalence Axiom)

**陈述**:

对于宇宙 $\mathcal{U}$ 中的类型 $A, B$，有：

$$(A =_\mathcal{U} B) \simeq (A \simeq B)$$

即：类型的相等等价于它们之间的等价。

**意义**:

1. **同构传递**: 同构的结构可以互换使用
2. **结构不变性**: 数学性质在同构下自动保持
3. **商构造**: 可以定义有效的商类型

**示例**:

- 自然数的两种定义（Peano vs 二进制）是同构的，因此在单值基础下是"相等"的
- 有限维向量空间由维数分类

### 4. 高阶归纳类型 (Higher Inductive Types, HITs)

**动机**: 传统归纳类型只能生成自由代数结构，但我们需要：

- 商类型（如 $\mathbb{Z} = \mathbb{N} \times \mathbb{N} / \sim$）
- 拓扑空间（如圆 $S^1$）
- 同伦论中的复杂构造

**定义**: HIT通过同时引入：

- 点构造子（生成元素）
- 路径构造子（生成等式/关系）
- 高阶路径构造子（生成高阶关系）

**示例：圆 $S^1$**:

```
HIT S¹ :=
| base : S¹
| loop : base = base
```

- `base` 是基点
- `loop` 是从base到base的非平凡路径

**性质**: $\Omega(S^1, \text{base}) \simeq \mathbb{Z}$（基本群是整数）

**示例：区间 $I$**:

```
HIT I :=
| 0_I : I
| 1_I : I
| seg : 0_I = 1_I
```

用于定义同伦和函数扩展性。

**示例：商类型**:

```
HIT A/R :=
| [·] : A → A/R
| eq : ∏_{x,y:A} R(x,y) → [x] = [y]
| set : isSet(A/R)
```

### 5. 构造性与计算性

**构造性数学**:

- 存在性证明必须提供构造
- 排中律不假设
- 选择公理有计算内容

**HoTT的计算解释**:

- 类型论有自然的计算语义
- 证明归约为计算
- 与函数式编程的联系

**同伦水平 (h-levels)**:

- **h-level 0**: 可缩类型（contractible）
- **h-level 1**: 命题（至多一个元素）
- **h-level 2**: 集合（离散，无高阶结构）
- **h-level 3**: 群胚
- ...

这允许在需要时恢复经典数学（集合层次），同时保留高阶结构。

## 关键结果

### 定理1: 弱∞-群胚模型

**定理** (Kapulkin-Lumsdaine, 2012):

Martin-Löf类型论有弱∞-群胚的模型。

这意味着类型论确实有同伦论的解释。

### 定理2: 单值公理的一致性

**定理** (Voevodsky, 2012):

单值公理与Martin-Löf类型论一致（相对于单纯集合模型）。

### 定理3: 圆周的基本群计算

**定理**: $\pi_1(S^1) \simeq \mathbb{Z}$

这是HoTT中的第一个非平凡同伦计算。

### 定理4: Blakers-Massey定理

**定理** (Favonia, Harper, 2016):

在HoTT中证明了经典的Blakers-Massey定理，展示了HoTT处理复杂同伦论的能力。

### 定理5: 同伦群论的进展

**定理**: 许多经典同伦论结果已在HoTT中重新证明，包括：

- Freudenthal悬挂定理
- van Kampen定理
- Hurewicz定理

## 现代发展 (2020-2025)

### 2020: Cubical Type Theory的成熟

**问题**: 传统HoTT中，单值公理破坏了计算性。

**解决方案**: **Cubical Type Theory** (Coquand, Huber, Mörtberg等)

- 使用立方体（而非单形）作为高维结构的基本形状
- 单值公理有计算内容
- 实现了**计算单值基础**

**实现**:

- **Cubical Agda**: Agda的立方体扩展
- **cubicaltt**: 实验性实现
- **redtt**: 新的证明助手

### 2021: 综合同伦论的扩展

**综合微分几何**与HoTT的结合：

- 综合数学的新领域
- 综合代数几何
- 综合拓扑学

### 2022: ∞-范畴语义

**关系**: HoTT与∞-范畴论的精确联系

**工作**:

- Kapulkin, Lumsdaine: 拟范畴模型
- Riehl, Shulman: ∞-范畴的语义
- 类型论作为∞-范畴的内在语言

### 2023: 形式化项目的进展

**UniMath** (Voevodsky遗产):

- 大量的数学形式化
- 包括代数、拓扑、范畴论

**Lean4中的HoTT**:

- 虽然Lean主要使用经典基础
- 但HoTT的思想影响了Lean的设计
- 可以形式化部分HoTT构造

### 2024: 实用证明助手

**Arend**:

- 基于HoTT的证明助手
- 强调可用性

**新型语言设计**:

- 结合HoTT与依赖类型
- 更友好的用户界面

### 2025: 新兴方向

1. **模态HoTT**: 引入模态算子（如形状模态）
2. **有向类型论**: 处理有向同伦（如高维范畴）
3. **线性HoTT**: 结合线性逻辑
4. **量子HoTT**: 量子计算的类型论基础

## 开放问题

### 问题1: 计算单值基础

**问题**: 如何在保持计算性的同时实现完整的单值基础？

**现状**: 立方体类型论提供了部分答案，但仍有局限。

### 问题2: 高阶归纳类型的规范形式

**问题**: 如何系统地理解高阶归纳类型的规范形式？

**挑战**: HITs的消去规则复杂。

### 问题3: 无穷范畴的类型论

**问题**: 如何扩展HoTT以直接处理∞-范畴（而不仅仅是∞-群胚）？

**进展**: Riehl-Shulman的定向类型论。

### 问题4: 经典数学的形式化

**问题**: 如何在HoTT中高效地形式化经典数学（分析、代数几何等）？

**策略**: 使用适当的h-level（通常是集合层）。

### 问题5: 性能优化

**问题**: 如何提高HoTT证明助手的性能？

**挑战**: 高维结构的计算开销大。

### 问题6: 教育推广

**问题**: 如何将HoTT纳入数学和计算机科学教育？

**现状**: 已有一些课程，但需要更多教材和课程。

## 总结

同伦类型论是数学基础领域最激动人心的发展之一。Voevodsky的单值基础不仅提供了一个与数学实践更契合的形式系统，更打开了连接类型论、同伦论和计算机科学的全新通道。

从HoTT Book的出版到立方体类型论的成熟，从综合同伦论的发展到∞-范畴语义的澄清，这一领域正在迅速发展。

**核心贡献**:

1. ✅ 新的数学基础范式
2. ✅ 计算同伦论的框架
3. ✅ 计算机形式化的友好基础
4. ✅ 连接了类型论与同伦论

**未来展望**:

- 完全可计算的单值基础
