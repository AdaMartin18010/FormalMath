# ∞-范畴论 (∞-Category Theory)

> **核心人物**: Jacob Lurie  
> **标志性著作**: "Higher Topos Theory" (2009), "Higher Algebra" (2017)  
> **应用领域**: 代数拓扑、代数几何、表示论、数学物理  
> **Mathlib4状态**: 部分形式化，活跃开发中

---

## 背景与动机

### 从范畴论到高维范畴论

**经典范畴论的核心洞察**:
> 数学结构的研究应该包含结构之间的态射。

**自然的问题**:
> 为什么不继续考虑"态射之间的态射"？

这就是高维范畴论的起点。

### 同伦论的视角

在代数拓扑中，同伦等价是比同胚更自然的等价概念：

```
同胚: X ≅ Y (严格的拓扑等价)
同伦等价: X ≃ Y (保持代数结构的等价)
```

**关键观察**: 同伦等价的传递性是"直到同伦"的，而非严格的。

这意味着：如果我们想要一个尊重同伦论的范畴框架，我们需要允许"态射的相等"也是"直到同伦"的。

### 导出代数的需要

现代代数几何和代数拓扑中广泛使用的导出构造(derived constructions)：

- 导出范畴 (Derived Categories)
- 谱 (Spectra)
- 代数K-理论
- 拓扑Hochschild同调

这些构造在经典范畴论中难以严格处理，因为：
1. 需要处理"直到同伦"的交换性
2. 高阶一致性条件复杂
3. 缺乏统一的理论框架

**∞-范畴论提供了一个解决方案**。

## 核心概念

### 1. ∞-范畴的定义

**直观定义**: 一个∞-范畴 C 包含：
- **对象**: X, Y, Z, ...
- **1-态射**: f: X → Y (普通态射)
- **2-态射**: α: f ⇒ g (1-态射之间的同伦)
- **3-态射**: β: α ⇒ α' (2-态射之间的同伦)
- **...**: 无限继续

**形式化定义** ( quasicategory / 拟范畴 ):

一个**拟范畴**是一个单纯集合 X: Δ^op → Set，满足**内Kan条件**:

对于所有 0 < i < n，任何内角 (inner horn) Λ^n_i → X 都可以填充为 n-单形 Δ^n → X。

**为什么这个定义有效**:

- n-单形 Δ^n 对应 n 个可复合的1-态射
- 内Kan条件保证了：可复合的态射确实有复合
- 填充的非唯一性对应"同伦唯一性"

### 2. ∞-范畴 vs 普通范畴

| 特性 | 普通范畴 | ∞-范畴 |
|------|---------|--------|
| 态射相等 | 严格相等 | 直到同伦等价 |
| 结合律 | 严格成立 | 直到同伦成立 |
| 复合 | 唯一确定 | 同伦唯一 |
| 图表交换 | 严格交换 | 直到同伦交换 |

### 3. ∞-群胚 (∞-Groupoids)

**定义**: 所有1-态射都是可逆的∞-范畴。

**关键定理** (Grothendieck假设):

∞-群胚的范畴等价于拓扑空间的同伦范畴（模去弱等价）。

```
∞-Groupoids ≃ Spaces
```

**意义**: 
- ∞-群胚是"空间的代数模型"
- 同伦群 π_n 对应 n-态射的"自同态"

### 4. 稳定∞-范畴 (Stable ∞-Categories)

**动机**: 代数拓扑中的谱(spectra)有极好的性质：
- 有正合三角形 (而非短正合序列)
- 有无穷环路/悬挂
- 同调论可以取值于谱

**定义**: 一个∞-范畴 C 是**稳定的**，如果：
1. 有零对象
2. 每个态射有纤维和余纤维
3. 三角形既是纤维序列又是余纤维序列

**核心例子**:
- 谱的∞-范畴 Sp
- 链复形的导出∞-范畴 D(R)
- 凝聚层/拟凝聚层的导出范畴

**定理**: 稳定∞-范畴等价于三角范畴加上某种增强。

### 5. ∞-Topos理论

**高阶Topos的定义**: 

一个**∞-topos**是形如 Fun(C^op, Spaces) 的∞-范畴的左正合局部化，其中 C 是小∞-范畴。

**与普通Topos的关系**:

普通(Grothendieck)topos是∞-topos的离散部分(0-截断)。

**应用**:
- 同伦层理论
- 导出代数几何
- 高阶堆栈(stacks)

## 关键结果

### 定理1: 等价性判据

**定理** (Joyal): 拟范畴之间的映射 f: X → Y 是范畴等价当且仅当：
1. 在0-截断层面诱导等价
2. 诱导完全的映射空间等价

这为验证∞-函子的等价性提供了实用标准。

### 定理2: 伴随函子定理

**定理** (Lurie): ∞-范畴版本的可表性和伴随函子定理。

特别地，对于导出范畴，这给出了导出Hom和导出张量积的存在性条件。

### 定理3: 可表性定理

**定理**: 一个∞-范畴是∞-topos当且仅当它满足Giraud公理的∞-版本。

这是topos理论的核心结果的完美推广。

### 定理4: 稳定Deligne猜想

**定理**: E_n-代数的 Hochschild 上链复形具有 E_{n+1}-代数结构。

这一经典猜想的解决展示了∞-范畴方法的威力。

### 定理5: 导出代数几何的基础

**定理**: 使用谱代数几何可以建立严格的导出代数几何基础。

Lurie的《Spectral Algebraic Geometry》完成了这一宏伟计划。

## 现代发展 (2020-2025)

### 2020: ∞-范畴的等价模型

多种∞-范畴的模型被证明等价：

- 拟范畴 (Quasicategories)
- 单纯范畴 (Simplicial Categories)  
- 完备Segal空间 (Complete Segal Spaces)
- 相对范畴 (Relative Categories)

这种等价性保证了理论的内禀性。

### 2021: ∞-范畴的初等理论

Riehl和Verity的工作：
- 发展了∞-范畴的"初等"理论
- 不依赖于复杂的单纯技术
- 更易于数学家使用

**代表性著作**: "Elements of ∞-Category Theory" (2022)

### 2022: 导出代数几何的突破

- **完美oid空间**: 使用∞-范畴方法简化理论
- **凝聚数学**: 与∞-topos理论的结合
- **p-进Hodge理论**: 导出方法的系统性应用

### 2023: 拓扑数据科学的∞-范畴方法

将∞-范畴应用于：
- 持续同调 (Persistent Homology)
- 层方法的数据分析
- 拓扑机器学习

### 2024: 形式化的进展

- **Mathlib4**: ∞-范畴的基础定义和引理
- **Univalent Foundations**: HoTT与∞-范畴的联系
- **计算机验证**: 关键定理的机械证明

### 2025: 新兴方向

1. **Dirac几何的∞-版本**: 高阶结构在几何物理中的应用
2. **量子场论的∞-范畴表述**: 因子化代数的进一步发展
3. **同伦类型论与∞-范畴**: 两种高阶数学基础的融合

## 与其他领域的联系

### 与代数拓扑的联系

```
经典代数拓扑          ∞-范畴论视角
─────────────────────────────────────
同伦群 π_n(X)    →   自同态(n-态射)
同伦类 [X,Y]      →   映射空间 Map(X,Y)
纤维序列         →   纤维∞-范畴
谱 (Spectra)     →   稳定∞-范畴
广义上同调论      →   可表函子
```

### 与代数几何的联系

**导出代数几何**: 
- 将概形(schemes)替换为谱概形(spectral schemes)
- 结构层取值于E_∞-环谱
- 相交理论的自然导出化

**应用**:
- 虚拟基本类
- Gromov-Witten理论的严格基础
- Donaldson-Thomas理论

### 与表示论的联系

**导出表示论**:
- 代数群的导出范畴
- 几何表示论的自然框架
- Soergel双模范畴的∞-版本

### 与数学物理的联系

**拓扑场论**:
- 扩展拓扑场论的∞-范畴表述
- Cobordism假设的证明
- 因子化代数

**弦理论**:
- 导出范畴在弦紧化中的应用
- D-膜范畴的严格定义

## 开放问题

### 问题1: ∞-范畴的初等公理化

**问题**: 能否为∞-范畴找到一个简洁的初等公理系统？

**现状**: 
- Riehl-Verity的工作提供了部分答案
- 但仍依赖于复杂的模型范畴技术

**意义**: 使∞-范畴更容易被广大数学家接受和使用。

### 问题2: 无穷维导出范畴

**问题**: 如何定义和处理无穷维代数结构的导出∞-范畴？

**挑战**: 
- 通常的导出范畴构造假设有限性条件
- 泛函分析中的导出方法尚不完善

### 问题3: ∞-范畴的K-理论

**问题**: 如何定义一般∞-范畴的代数K-理论？

**进展**: 
- Blumberg-Gepner-Tabuada在2013年有重要工作
- 但与经典K-理论的完全联系仍待澄清

### 问题4: 拓扑循环同调的几何解释

**问题**: 拓扑循环同调(TC)的几何意义是什么？

**背景**: TC通过∞-范畴方法定义，但缺乏经典同调论那样的直观解释。

### 问题5: 非可换几何的∞-版本

**问题**: 如何将Connes的非可换几何纳入∞-范畴框架？

**可能方向**: 
- 谱三元组的导出版本
- KK-理论的∞-范畴表述

### 问题6: 同伦类型论与∞-范畴的完全对应

**问题**: 同伦类型论(HoTT)中的概念与∞-范畴论中的概念能否建立完全对应？

**现状**: 
- 部分对应已建立
- 完整的语义对应仍是一个活跃研究领域

## 总结

∞-范畴论是21世纪数学最重要的发展之一。它不仅为代数拓扑中长期的同伦相干性问题提供了优雅的解决方案，更成为了现代代数几何、表示论和数学物理的基础语言。

**核心贡献**:
1. ✅ 统一的高阶结构理论
2. ✅ 导出代数的严格基础
3. ✅ 同伦论与代数的深度融合
4. ✅ 数学物理的新工具

**未来展望**:
- 更友好的初等理论
- 更广泛的应用领域
- 与形式化证明的深度融合
