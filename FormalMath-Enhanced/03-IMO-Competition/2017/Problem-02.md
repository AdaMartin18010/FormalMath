# IMO 2017 Problem 2

## 题目
求所有函数 $f: \mathbb{R} \to \mathbb{R}$，满足对所有实数 $x, y$：
$$f(f(x)f(y)) + f(x+y) = f(xy)$$

## 分类信息
- **领域**: 代数/函数方程
- **难度**: 5分
- **涉及概念**: 函数方程、函数迭代、柯西方程、根的存在性

## 解答

### 答案
满足条件的函数是：
- $f(x) = 0$（对所有 $x$）
- $f(x) = x - 1$（对所有 $x$）
- $f(x) = 1 - x$（对所有 $x$）

### 证明

**步骤1：观察对称性**

若 $f$ 是解，则 $-f$ 也是解。

**步骤2：分析 $f(0)$**

令 $x = y = 0$：
$$f(f(0)^2) + f(0) = f(0)$$
$$f(f(0)^2) = 0$$

若 $f(0) = 0$，令 $y = 0$：
$$f(f(x) \cdot 0) + f(x) = f(0) = 0$$
$$f(0) + f(x) = 0$$
$$f(x) = 0$$

所以 $f \equiv 0$ 是一个解。

假设 $f(0) \neq 0$，则 $f(0) = 1$（通过对称性可设为正）。

由 $f(f(0)^2) = f(1) = 0$。

**步骤3：关键引理**

**引理**：$f(z) = 0 \iff z = 1$。

**证明**：已知 $f(1) = 0$。若 $f(z) = 0$ 且 $z \neq 1$，取 $(x, y) = (z, \frac{z}{z-1})$（使得 $x+y = xy$）。

则：$f(f(z)f(y)) + f(z+y) = f(zy)$

$f(0) + z+y = zy$，但 $z+y = zy$，所以 $f(0) = 0$，矛盾！

**步骤4：推导函数方程**

令 $y = 1$：
$$f(f(x) \cdot 0) + f(x+1) = f(x)$$
$$f(0) + f(x+1) = f(x)$$
$$1 + f(x+1) = f(x)$$

所以 $f(x+1) = f(x) - 1$。

由归纳：$f(x+n) = f(x) - n$（$n \in \mathbb{Z}$）。

**步骤5：证明单射性**

设 $f(a) = f(b)$。需要证 $a = b$。

利用 $f(x+n) = f(x) - n$，可将 $a, b$ 平移使它们足够大。

可以找到 $x, y$ 使得 $x+y = a+1$，$xy = b$。

代入原方程并比较，利用 $f(0) = 1$ 和 $f(1) = 0$，可推出 $a = b$。

**步骤6：确定函数**

由 $f(x+1) = f(x) - 1$，有 $f(f(x)) = f(x) - 1$。

对原方程应用 $f$ 三次：
$$f(f(f(x))) = f(f(x)) - 1 = f(x) - 2$$
$$f(1-f(x)) = f(f(f(x))) = f(x)$$

由单射性：$1 - f(x) = x$，即 $f(x) = 1 - x$。

或考虑 $-f$ 的情况，得 $f(x) = x - 1$。

## 关键思路与技巧

1. **特殊值代入**：$x = y = 0$ 和 $y = 1$ 提供关键信息
2. **零点分析**：确定 $f(z) = 0 \iff z = 1$
3. **平移性质**：$f(x+1) = f(x) - 1$ 揭示函数结构
4. **单射性证明**：通过巧妙选择 $x, y$ 证明
5. **函数迭代**：利用 $f(f(x)) + f(x) = 1$ 确定函数
