# IMO 2017 Problem 3

## 题目
一个**不可约格点**是指满足 $\gcd(x, y) = 1$ 的有序整数对 $(x, y)$。

证明：如果 $S$ 是不可约格点的有限集，则存在非常数的整系数齐次多项式 $f(x, y)$，使得对所有 $(x, y) \in S$，都有 $f(x, y) = 1$。

## 分类信息
- **领域**: 代数/数论
- **难度**: 7分
- **涉及概念**: 齐次多项式、贝祖定理、格点、中国剩余定理、插值

## 解答

### 解答1（归纳法 + 线性变换）

对 $|S|$ 进行归纳。

**基础情形 $|S| = 1$**：设 $S = \{(a, b)\}$，$\gcd(a, b) = 1$。

由**贝祖定理**，存在整数 $u, v$ 使得 $ua + vb = 1$。

取 $f(x, y) = ux + vy$，这是次数为1的齐次多项式。

---

**归纳步骤**：假设结论对 $|S| = k-1$ 成立，证 $|S| = k$ 时成立。

设 $S = \{(a_1, b_1), \ldots, (a_{k-1}, b_{k-1}), (a_k, b_k)\}$。

由归纳假设，存在次数为 $n$ 的齐次多项式 $P(x, y)$，使得 $P(a_i, b_i) = 1$ 对 $i = 1, \ldots, k-1$ 成立。

**关键引理**：存在 $2 \times 2$ 整数矩阵 $T = \begin{pmatrix} u & v \\ s & t \end{pmatrix}$，$\det T = 1$，使得：
$$T \cdot \begin{pmatrix} a_k \\ b_k \end{pmatrix} = \begin{pmatrix} 1 \\ 0 \end{pmatrix}$$

**证明**：由 $\gcd(a_k, b_k) = 1$，存在 $u, v$ 使得 $ua_k + vb_k = 1$。取 $s = -b_k$，$t = a_k$，则 $ut - vs = ua_k + vb_k = 1$。✓

---

应用变换 $T$ 到所有点（保持不可约性，因为 $\det T = 1$ 保证 $T^{-1}$ 也是整数矩阵）。

若 $g$ 对新点集有效，则 $f(x, y) = g(ux + vy, sx + ty)$ 对原集合有效。

因此不妨设 $(a_k, b_k) = (1, 0)$。

我们需要构造 $Q$ 使得：
- $Q(a_i, b_i) = 1$，$i = 1, \ldots, k-1$
- $Q(1, 0) = 1$

取 $Q(x, y) = P(x, y)^M - C \cdot y \cdot \prod_{i=1}^{k-1}(b_i x - a_i y) \cdot R(x, y)$

其中 $M$ 足够大，$C$ 适当选择，$R$ 使等式成立。

具体地，设 $L_i(x, y) = b_i x - a_i y$（在 $(a_i, b_i)$ 处为0）。

取 $Q(x, y) = P(x, y)^{k-1} - \sum_{j=1}^{k-1} c_j L_j(x, y) \cdot \prod_{i \neq j} L_i(x, y)$

选择 $c_j$ 使 $Q(a_j, b_j) = 1$ 且 $Q(1, 0) = 1$。

由**中国剩余定理**思想，这是可行的。

## 关键思路与技巧

1. **贝祖定理**：基础情形的核心工具
2. **线性变换**：通过 $SL_2(\mathbb{Z})$ 变换简化问题
3. **归纳构造**：逐点添加并调整多项式
4. **拉格朗日插值**：齐次版本的插值技术
5. **中国剩余定理**：在不同点处控制多项式值

## 现代联系

本题与**代数几何**中的**希尔伯特第17问题**（正定有理函数表示为有理函数平方和）有关。在**计算代数**中，这类问题涉及**多项式插值**和**理想论**。在**密码学**中，格上的困难问题（如SVP、CVP）是后量子密码的基础。
