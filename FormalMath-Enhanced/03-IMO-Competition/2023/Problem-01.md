# IMO 2023 Problem 1

## 题目
求所有满足以下性质的合数 $n > 1$：若 $d_1 < d_2 < \cdots < d_k$ 是 $n$ 的所有正因数，则对每个 $1 \leq i \leq k-2$，$d_i$ 整除 $d_{i+1} + d_{i+2}$。

## 分类信息
- **领域**: 数论
- **难度**: 4分
- **涉及概念**: 因数、合数、整除性、素数幂

## 解答

### 答案
满足条件的 $n$ 是所有素数幂 $p^e$（$p$ 为素数，$e \geq 2$）。

### 证明

**步骤1：验证素数幂满足条件**

设 $n = p^e$，则因数为 $d_i = p^{i-1}$（$i = 1, 2, \ldots, e+1$）。

需要验证：$p^{i-1} \mid p^i + p^{i+1}$。

由于 $p^i + p^{i+1} = p^i(1 + p)$，显然被 $p^{i-1}$ 整除。

---

**步骤2：证明没有其他合数满足条件**

设 $n$ 至少有两个不同的素因子，设 $p < q$ 是最小的两个素因子。

设 $p^e$ 是整除 $n$ 的最大的 $p$ 的幂，且 $p^e < q$。

则 $n$ 的最小因数为：$1, p, p^2, \ldots, p^e, q, \ldots$

考虑条件 $d_i \mid d_{i+1} + d_{i+2}$ 对 $d_i = \frac{n}{q}$（即 $d_i$ 是 $n$ 除以 $q$）：

需要 $\frac{n}{q} \mid \frac{n}{p^e} + \frac{n}{p^{e-1}} = \frac{(p+1)n}{p^e}$。

这意味着 $\frac{q(p+1)}{p^e}$ 必须是整数。

但 $p^e \geq p$ 且 $q > p^e \geq p$，所以 $p^e$ 与 $q$ 互素。

需要 $p^e \mid p+1$，但 $p^e > p+1$（当 $e \geq 2$ 或 $p \geq 3$），矛盾。

详细分析表明，如果 $n$ 有多个素因子，条件无法满足。

## 关键思路与技巧

1. **素数幂验证**：直接计算验证充分性
2. **反证法**：假设存在多素因子导致矛盾
3. **极端因数分析**：考虑最大/最小的特殊因数
4. **估值论证**：利用素数幂的整除性质
