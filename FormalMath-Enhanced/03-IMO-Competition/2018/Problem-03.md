# IMO 2018 Problem 3

## 题目

定义数列 $\{a_n\}_{n \geq 1}$ 如下：$a_1$ 是任意正整数，对 $n \geq 1$：
$$a_{n+1} = \frac{a_n}{P(a_n)}$$
其中 $P(a_n)$ 是 $a_n$ 的最大素因子（约定 $P(1) = 1$）。

是否存在 $a_1$ 使得序列 $\{a_n\}$ 无界？

## 分类信息

- **领域**: 数论/序列
- **难度**: 7分
- **涉及概念**: 素因子、递推序列、有界性

## 解答

### 答案

不存在这样的 $a_1$，序列总是有界的。

### 证明

**引理1**：若 $a_n = p^k$（素数幂），则 $a_{n+1} = p^{k-1}$，$a_{n+k} = 1$，之后恒为1。

**引理2**：若 $a_n = p_1^{e_1} \cdots p_r^{e_r}$ 且 $p_1 < p_2 < \cdots < p_r$，则：
$$a_{n+1} = p_1^{e_1} \cdots p_{r-1}^{e_{r-1}} \cdot p_r^{e_r - 1}$$

**主要论证**：

我们需要证明对任意 $a_1$，序列最终达到1或进入一个有界循环。

**观察1**：序列中的素因子只能是 $a_1$ 的素因子或更小的素数。

**观察2**：若 $a_n$ 的最大素因子是 $p$，则 $a_{n+1} < a_n$ 除非 $a_n = p$（即 $a_n$ 本身就是素数）。

**关键观察**：设 $M(a)$ 为 $a$ 的素因子个数（计重数）。

当 $e_k = 1$ 时，$M(a_{n+1}) = M(a_n) - 1$。

当 $e_k > 1$ 时，$M(a_{n+1}) = M(a_n)$，但 $a_{n+1} < a_n$。

无论如何，序列 $(M(a_n), a_n)$ 按字典序递减，最终必须终止。

因此序列有界（实际上，最终达到1）。

## 关键思路与技巧

1. **素因子分析**：关注最大素因子的作用
2. **单调性**：构造适当的递减量
3. **字典序论证**：多维度的递减确保终止

## 现代联系

本题是**整数序列**理论的经典问题。在**算术动力学**中，研究整数的迭代行为。著名的**Collatz猜想**（3n+1问题）就是这类问题的极端困难例子。
