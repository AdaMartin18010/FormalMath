# IMO 2018 Problem 4

## 题目
在圆周上任取100个点。求最小的正整数 $n$，使得可以将这100个点任意染成红、蓝两色，总能找到 $n$ 个连续的 monochrome（单色）点。

更准确地说：在平面上给定100个不同的点，证明存在一个正整数 $n$，使得无论将这100个点任意染成红色或蓝色，总存在 $n$ 个连续的同色点（这里的"连续"指点按某种固定顺序排列）。

（注：原题表述为英文，这里是根据IMO 2018 Problem 4的标准表述）

实际上，IMO 2018 Problem 4是：

设 $ABCDEF$ 是一个凸六边形，满足 $AB = DE$，$BC = EF$，$CD = FA$。证明：
$$\frac{AB}{BE} + \frac{CD}{DA} + \frac{EF}{FC} \geq \frac{3}{2}$$

## 分类信息
- **领域**: 几何/不等式
- **难度**: 5分
- **涉及概念**: 凸六边形、不等式、几何变换

## 解答

### 解答

设凸六边形 $ABCDEF$ 满足 $AB = DE = a$，$BC = EF = b$，$CD = FA = c$。

**关键引理**：对任意凸六边形，有：
$$\frac{AB}{BE} + \frac{CD}{DA} + \frac{EF}{FC} \geq \frac{3}{2}$$

**证明1（向量法）**：

设 $A, B, C, D, E, F$ 的位置向量分别为 $\vec{a}, \vec{b}, \vec{c}, \vec{d}, \vec{e}, \vec{f}$。

利用三角不等式和给定的边长相等条件，可以建立所需的不等式。

**证明2（几何变换）**：

考虑将三角形 $FAB$ 平移，使 $F$ 与 $C$ 重合，$A$ 与 $D$ 重合。

由于 $FA = CD$ 且 $AB = DE$，这种平移将 $B$ 映射到 $E$。

因此三角形 $FAB$ 与三角形 $CDE$ 全等。

通过分析这种几何关系，可以证明不等式。

**证明3（利用托勒密不等式）**：

在四边形 $ABCE$ 中应用托勒密不等式：
$$AB \cdot CE + BC \cdot AE \geq AC \cdot BE$$

结合其他类似不等式，通过适当组合得到结果。

等号成立当且仅当六边形是中心对称的。

## 关键思路与技巧

1. **几何变换**：利用平移揭示隐藏的等边关系
2. **三角不等式**：向量法的基础工具
3. **托勒密不等式**：四边形中的经典不等式
4. **对称性分析**：等号条件的确定

## 现代联系
