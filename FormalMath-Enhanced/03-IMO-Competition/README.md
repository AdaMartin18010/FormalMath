# IMO 竞赛数学模块

## 模块概述

本模块是FormalMath项目的重要组成部分，专注于整理和分析**国际数学奥林匹克（IMO）**2006-2025年的竞赛题目与解答。

## 模块目标

- 📚 **题目收集**：整理2006-2025年共20年的IMO题目（目标120题）
- 🎯 **深度分析**：每道题目提供完整的分类、解答和数学概念解析
- 🔗 **知识链接**：与FormalMath主项目的数学概念体系建立连接
- 📖 **备赛指南**：提供系统性的IMO备赛策略和资源

## 模块结构

```

├── README.md                    # 本文件
├── IMO备赛指南.md               # 系统性备赛指南

├── 2006/                        # 2006年IMO题目

```
