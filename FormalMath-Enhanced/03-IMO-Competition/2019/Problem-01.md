# IMO 2019 Problem 1

## 题目
求所有函数 $f: \mathbb{Z} \to \mathbb{Z}$，满足对所有整数 $a, b$：
$$f(2a) + 2f(b) = f(f(a+b))$$

## 分类信息
- **领域**: 代数/函数方程
- **难度**: 4分
- **涉及概念**: 函数方程、整数映射、线性函数

## 解答

### 答案
满足条件的函数是 $f(x) = 2x + c$（$c$ 为任意整数常数）和 $f(x) = 0$。

### 证明

**步骤1：探索线性解**

设 $f(x) = kx + c$，代入：
$$k(2a) + c + 2(kb + c) = k(k(a+b) + c) + c$$
$$2ka + c + 2kb + 2c = k^2(a+b) + kc + c$$
$$2k(a+b) + 3c = k^2(a+b) + kc + c$$

对所有 $a, b$ 成立，需要：
- $2k = k^2$，即 $k = 0$ 或 $k = 2$
- $3c = kc + c$，即 $3c = kc + c$，所以 $2c = kc$

若 $k = 0$：$2c = 0$，所以 $c = 0$。$f(x) = 0$ 验证成立。

若 $k = 2$：$2c = 2c$，恒成立。$f(x) = 2x + c$ 验证成立。

**步骤2：证明没有其他解**

令 $a = 0$：$f(0) + 2f(b) = f(f(b))$，即 $f(f(b)) = 2f(b) + f(0)$。

令 $b = 0$：$f(2a) + 2f(0) = f(f(a)) = 2f(a) + f(0)$。

所以 $f(2a) = 2f(a) - f(0)$。

令 $c = f(0)$，则 $f(2a) = 2f(a) - c$。

原方程变为：
$$2f(a) - c + 2f(b) = 2f(a+b) + c$$
$$2f(a) + 2f(b) - 2c = 2f(a+b)$$
$$f(a) + f(b) - c = f(a+b)$$

令 $g(x) = f(x) - c$，则：
$$g(a) + c + g(b) + c - c = g(a+b) + c$$
$$g(a) + g(b) = g(a+b)$$

这是**柯西方程**！由于 $f: \mathbb{Z} \to \mathbb{Z}$，$g$ 也是整数到整数的映射。

柯西方程的解为 $g(x) = kx$（$k$ 为整数）。

因此 $f(x) = kx + c$。

代回原方程验证，只有 $k = 0$（且 $c = 0$）或 $k = 2$（任意 $c$）满足。

## 关键思路与技巧

1. **试探线性解**：猜测并验证多项式形式
2. **特殊值代入**：$a = 0$ 和 $b = 0$ 提供关键信息
3. **转化柯西方程**：通过变量替换揭示函数结构
4. **整数域分析**：利用整数到整数的限制确定解
