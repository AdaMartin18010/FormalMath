# IMO 2015 Problem 06

## 题目

整数序列 $a_1, a_2, \ldots$ 满足条件：(i) $1 \leq a_j \leq 2015$ 对所有 $j \geq 1$；(ii) $k + a_k \neq \ell + a_\ell$ 对所有 $1 \leq k < \ell$。证明：存在两个正整数 $b$ 和 $N$，使得对所有满足 $n > m \geq N$ 的整数 $m, n$：
$$\left|\sum_{j=m+1}^{n}(a_j - b)\right| \leq 1007^2$$
