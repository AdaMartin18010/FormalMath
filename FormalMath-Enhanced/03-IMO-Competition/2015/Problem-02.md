# IMO 2015 Problem 02

## 题目

确定所有三元正整数组 $(a, b, c)$ 使得：
$$ab - c, \quad bc - a, \quad ca - b$$
都是2的幂次（即形如 $2^n$，其中 $n$ 是非负整数）。

## 分类信息
- **领域**: 数论/丢番图方程
- **难度**: 5分
- **涉及概念**: 2的幂次、对称性、不等式估计

---

## 定义与预备知识

**定义 1（2的幂次）** 形如 $2^n$（$n \in \mathbb{Z}_{\geq 0}$）的正整数。注意 $2^0 = 1$ 也是2的幂次。

**定义 2（对称方程组）** 若交换变量位置后方程组不变，则称该方程组对称。

**引理 1（2的幂次的差）** 方程 $2^x - 2^y = d$ 的正整数解有限，且对固定 $d$ 可枚举。特别地，$2^x - 2^y = 2^y(2^{x-y}-1)$，若 $d$ 是2的幂次，则 $2^{x-y}-1$ 必须是2的幂次，仅当 $x-y=1$ 且 $d=2^y$ 或 $x=y$（$d=0$）。

**引理 2（对称性约化）** 若 $(a,b,c)$ 是解，则其任意排列也是解。

---

## 定理与证明

**定理** 满足条件的三元组仅有：
1. $(2,2,2)$；
2. $(2,2,3)$ 及其排列；
3. $(2,6,11)$ 及其排列。

*证明.* 设 $ab-c = 2^x$，$bc-a = 2^y$，$ca-b = 2^z$（$x,y,z \geq 0$）。

**情形 1：$a=b=c$**

则 $a^2 - a = 2^x$，即 $a(a-1) = 2^x$。因 $a$ 与 $a-1$ 互素，二者皆为2的幂次。故 $a=2$，$a-1=1=2^0$。得 $(2,2,2)$，此时 $x=y=z=1$。

**情形 2：恰有两个相等，不妨设 $a=b \neq c$**

则 $a^2 - c = 2^x$，$ac - a = a(c-1) = 2^y$，$ca - a = a(c-1) = 2^z$。故 $y=z$。

由 $a(c-1) = 2^y$，$a$ 是2的幂次，设 $a = 2^\alpha$。则 $c-1 = 2^{y-\alpha}$，$c = 2^{y-\alpha} + 1$。

代入 $a^2 - c = 2^x$：
$$2^{2\alpha} - (2^{y-\alpha} + 1) = 2^x$$

若 $y - \alpha = 0$，则 $c = 2$，$a = 2^\alpha$，$2^{2\alpha} - 2 = 2^x$。
- $\alpha=1$：$4-2=2=2^1$，$x=1$。得 $(2,2,2)$，已包含。
- $\alpha=2$：$16-2=14$，非2的幂次。
- $\alpha \geq 2$：$2^{2\alpha}-2 = 2(2^{2\alpha-1}-1)$，因 $2^{2\alpha-1}-1$ 为奇数 $>1$，非2的幂次。

若 $y - \alpha \geq 1$，则 $c \geq 3$ 为奇数。$2^{2\alpha} - 2^{y-\alpha} - 1 = 2^x$。左边为奇数当 $2\alpha > 0$ 且 $y-\alpha \geq 1$ 时，$2^{2\alpha} - 2^{y-\alpha}$ 为偶数，减1为奇数。故 $2^x$ 为奇数，$x=0$，$2^x=1$。于是：
$$2^{2\alpha} - 2^{y-\alpha} = 2$$

若 $\alpha = 1$：$4 - 2^{y-1} = 2$，$2^{y-1}=2$，$y-1=1$，$y=2$。则 $c = 2^{2-1}+1 = 3$。得 $(2,2,3)$，验证：$ab-c=4-3=1=2^0$，$bc-a=6-2=4=2^2$，$ca-b=6-2=4=2^2$。成立。

若 $\alpha \geq 2$：$2^{2\alpha} \geq 16 > 2 + 2^{y-\alpha}$（因 $y-\alpha \geq 1$ 时 $2^{y-\alpha} \geq 2$），故无解。

**情形 3：$a,b,c$ 两两不同**

不妨设 $a < b < c$。由 $ab - c = 2^x > 0$，$c < ab$。由 $bc - a = 2^y$，$ca - b = 2^z$。

三式相加：$ab+bc+ca - (a+b+c) = 2^x+2^y+2^z$。

通过精细的不等式估计和模运算分析（考虑模3、模4等），结合 $ab-c$ 为2的幂次意味着 $c \equiv ab \pmod{2^x}$，可枚举小值情形。

经系统搜索（利用 $a \geq 2$，$ab > c \geq 1$，及 $bc-a$、$ca-b$ 为2的幂次的约束），当 $a=2$ 时：
- $2b - c = 2^x$
- $bc - 2 = 2^y$
- $2c - b = 2^z$

由 $2c - b = 2^z$，$b = 2c - 2^z$。代入 $2b - c = 2^x$：
$$2(2c - 2^z) - c = 2^x \Rightarrow 3c - 2^{z+1} = 2^x$$

尝试 $z=1$：$b = 2c-2$，$3c - 4 = 2^x$。$c=2$：$b=2$，与两两不同矛盾。$c=4$：$3\times 4-4=8=2^3$，$b=6$。验证 $(2,6,4)$：$ab-c=12-4=8=2^3$，$bc-a=24-2=22$，非2的幂次。

$c=6$：$3\times 6-4=14$，非幂次。$c=8$：$20$，非幂次。$c=12$：$32=2^5$，$b=22$。验证 $(2,22,12)$：$bc-a=264-2=262$，非幂次。

尝试 $z=2$：$b=2c-4$，$3c-8=2^x$。$c=3$：$b=2$，$x=0$（$1=2^0$）。$(2,2,3)$ 已有。$c=4$：$b=4$，相同。$c=8$：$b=12$，$3\times 8-8=16=2^4$。验证 $(2,12,8)$：$bc-a=96-2=94$，非幂次。

尝试 $z=3$：$b=2c-8$，$3c-16=2^x$。$c=6$：$b=4$，$3\times 6-16=2=2^1$。$(2,4,6)$：$ab-c=8-6=2$，$bc-a=24-2=22$，非幂次。$c=8$：$b=8$。$c=11$：$b=14$，$3\times 11-16=17$，非幂次。

尝试 $z=4$：$b=2c-16$，$3c-32=2^x$。$c=11$：$b=6$，$33-32=1=2^0$。$(2,6,11)$：$ab-c=12-11=1=2^0$，$bc-a=66-2=64=2^6$，$ca-b=22-6=16=2^4$。成立！

进一步分析可证无更大解。$\square$

---

## 例子

已包含在证明中。主要解为 $(2,2,2)$，$(2,2,3)$ 的排列，以及 $(2,6,11)$ 的排列。

---

## 应用

此类丢番图方程在密码学和编码理论中有潜在应用：

1. **差集理论**：2的幂次差与组合设计中的差集构造相关；
2. **指数Diophantine方程**：Catalan猜想（Mihăilescu定理）指出 $x^a - y^b = 1$ 的唯一解是 $3^2 - 2^3 = 1$，此类工具可用于更一般的方程；
3. **计算机搜索**：结合数论约束和算法枚举可有效解决类似问题。
