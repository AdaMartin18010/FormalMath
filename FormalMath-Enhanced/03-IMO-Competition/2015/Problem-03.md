# IMO 2015 Problem 03

## 题目

设锐角三角形 $ABC$ 有外心 $O$，$H$ 是垂心。设 $F$ 是从 $A$ 到 $BC$ 的垂足。点 $M$ 是 $BC$ 的中点。点 $Q$ 在 $\triangle ABC$ 的外接圆上满足 $\angle HQA = 90°$，点 $K$ 在外接圆上满足 $\angle HKQ = 90°$。假设点 $A, B, C, K, Q$ 互不相同且按此顺序在外接圆上。证明：三角形 $KQH$ 和 $FKM$ 的外接圆相切。
