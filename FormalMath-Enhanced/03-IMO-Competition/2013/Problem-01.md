# IMO 2013 Problem 1

## 题目

证明对于任意一对正整数 $k$ 和 $n$，存在 $k$ 个（不必不同的）正整数 $m_1, m_2, \ldots, m_k$ 使得：
$$1 + \frac{2^k - 1}{n} = \left(1 + \frac{1}{m_1}\right)\left(1 + \frac{1}{m_2}\right)\cdots\left(1 + \frac{1}{m_k}\right)$$

## 分类信息
- **领域**: 数论/代数
- **难度**: 4分
- **涉及概念**: 连分数展开、归纳构造、埃及分数

## 解答

### 方法一：对 $k$ 归纳

**步骤1：基础情况 $k = 1$**

需要 $1 + \frac{2 - 1}{n} = 1 + \frac{1}{n} = 1 + \frac{1}{m_1}$。

取 $m_1 = n$。成立。

**步骤2：归纳假设**

假设对 $k$ 和任意 $n$，存在 $m_1, \ldots, m_k$ 使得：
$$1 + \frac{2^k - 1}{n} = \prod_{i=1}^k \left(1 + \frac{1}{m_i}\right)$$

**步骤3：对 $k+1$ 的构造**

我们需要：
$$1 + \frac{2^{k+1} - 1}{n} = \prod_{i=1}^{k+1} \left(1 + \frac{1}{m_i}\right)$$

关键观察：
$$1 + \frac{2^{k+1} - 1}{n} = \frac{n + 2^{k+1} - 1}{n}$$

尝试写成 $\left(1 + \frac{1}{m}\right)\left(1 + \frac{2^k - 1}{n'}\right)$ 的形式。

设：
$$\left(1 + \frac{1}{m}\right)\left(1 + \frac{2^k - 1}{n'}\right) = \frac{m+1}{m} \cdot \frac{n' + 2^k - 1}{n'} = \frac{n + 2^{k+1} - 1}{n}$$

选择 $n' = n$：
$$\frac{m+1}{m} \cdot \frac{n + 2^k - 1}{n} = \frac{n + 2^{k+1} - 1}{n}$$

$$\frac{(m+1)(n + 2^k - 1)}{m} = n + 2^{k+1} - 1$$

$$(m+1)(n + 2^k - 1) = m(n + 2^{k+1} - 1)$$

$$mn + m(2^k - 1) + n + 2^k - 1 = mn + m(2^{k+1} - 1)$$

$$m(2^k - 1) + n + 2^k - 1 = m(2^{k+1} - 1)$$

$$n + 2^k - 1 = m(2^{k+1} - 1 - 2^k + 1) = m \cdot 2^k$$

$$m = \frac{n + 2^k - 1}{2^k}$$

这不一定是整数。例如 $n = 1, k = 2$：$m = \frac{1+4-1}{4} = 1$。整数！

$n = 2, k = 2$：$m = \frac{2+4-1}{4} = \frac{5}{4}$。不是整数。

所以这种直接构造不总是有效。

**步骤4：分奇偶构造**

**情况A：$n$ 是偶数**

设 $n = 2t$。取 $m_1 = n = 2t$。则：
$$1 + \frac{1}{m_1} = 1 + \frac{1}{2t} = \frac{2t+1}{2t}$$

剩余需要：
$$\frac{1 + \frac{2^{k+1}-1}{2t}}{\frac{2t+1}{2t}} = \frac{2t + 2^{k+1} - 1}{2t+1}$$

$$= 1 + \frac{2^{k+1} - 2}{2t+1} = 1 + \frac{2(2^k - 1)}{2t+1}$$

这不是 $1 + \frac{2^k - 1}{n'}$ 的形式。

**步骤5：正确的归纳构造**

对 $n$ 分两种情况：

**情况1：$n$ 是奇数**

取 $m_1 = n$。则：
$$\left(1 + \frac{1}{n}\right)^{-1}\left(1 + \frac{2^{k+1}-1}{n}\right) = \frac{n}{n+1} \cdot \frac{n + 2^{k+1} - 1}{n} = \frac{n + 2^{k+1} - 1}{n+1}$$

$$= 1 + \frac{2^{k+1} - 2}{n+1} = 1 + \frac{2(2^k - 1)}{n+1}$$

由于 $n$ 是奇数，$n+1$ 是偶数。设 $n+1 = 2t$。

$$= 1 + \frac{2^k - 1}{t} = 1 + \frac{2^k - 1}{(n+1)/2}$$

由归纳假设，存在 $m_2, \ldots, m_{k+1}$ 使得：
$$1 + \frac{2^k - 1}{(n+1)/2} = \prod_{i=2}^{k+1}\left(1 + \frac{1}{m_i}\right)$$

所以：
$$1 + \frac{2^{k+1}-1}{n} = \left(1 + \frac{1}{n}\right)\prod_{i=2}^{k+1}\left(1 + \frac{1}{m_i}\right)$$

令 $m_1 = n$。完成！

**情况2：$n$ 是偶数**

取 $m_1 = 2n$。则：
$$\left(1 + \frac{1}{2n}\right)^{-1}\left(1 + \frac{2^{k+1}-1}{n}\right) = \frac{2n}{2n+1} \cdot \frac{n + 2^{k+1} - 1}{n} = \frac{2(n + 2^{k+1} - 1)}{2n+1}$$

$$= \frac{2n + 2^{k+2} - 2}{2n+1} = 1 + \frac{2^{k+2} - 3}{2n+1}$$

这不是 $1 + \frac{2^k - 1}{n'}$ 的形式。这个构造不对。

尝试另一种方法。取 $m_1 = n + 2^{k+1} - 2$？太随意了。

**正确的偶数情况构造**：

设 $n = 2^s \cdot t$（$t$ 奇数，$s \geq 1$）。

取 $m_1 = m_2 = \cdots = m_s = 2n - 1$？不。

实际上，标准解法如下：

对 $n$ 偶数，取 $m_1 = n + 2^{k+1} - 2$？验证：

更好的方法：利用恒等式
$$1 + \frac{2^{k+1}-1}{n} = \left(1 + \frac{1}{n + 2^{k+1} - 2}\right)\left(1 + \frac{2^k - 1}{n/2}\right)$$

验证右边：
$$\frac{n + 2^{k+1} - 1}{n + 2^{k+1} - 2} \cdot \frac{n/2 + 2^k - 1}{n/2}$$

$$= \frac{n + 2^{k+1} - 1}{n + 2^{k+1} - 2} \cdot \frac{n + 2^{k+1} - 2}{n}$$

$$= \frac{n + 2^{k+1} - 1}{n} = 1 + \frac{2^{k+1}-1}{n}$$

✓ 完美！

所以：
- 若 $n$ 是偶数，取 $m_1 = n + 2^{k+1} - 2$，然后对 $n' = n/2$ 和 $k$ 用归纳假设。
- 若 $n$ 是奇数，取 $m_1 = n$，然后对 $n' = (n+1)/2$ 和 $k$ 用归纳假设。

**归纳有效性**：

- 奇数情况：$n' = (n+1)/2 \leq n$（对 $n \geq 1$）。
- 偶数情况：$n' = n/2 < n$。

两种情况都使 $n$ 严格减小或保持有界，归纳有效。

**基础情况验证**：

$k = 1$：$1 + \frac{1}{n} = 1 + \frac{1}{m_1}$，$m_1 = n$。

对任意 $n$，$k = 1$ 成立。

**归纳步骤总结**：

假设对 $k$ 和任意 $n$ 成立。对 $k+1$ 和 $n$：

- 若 $n$ 奇数：$m_1 = n$，剩余 $k$ 个因子处理 $n' = (n+1)/2$。
- 若 $n$ 偶数：$m_1 = n + 2^{k+1} - 2$，剩余 $k$ 个因子处理 $n' = n/2$。

由归纳假设，剩余部分可构造。$\square$

## 关键思路

1. **归纳法**：对 $k$ 归纳，将 $k+1$ 个因子的构造归结为 $k$ 个因子的构造
2. **分奇偶**：$n$ 的奇偶性决定第一步的拆分方式
3. **恒等式设计**：关键恒等式 $\frac{n + 2^{k+1} - 1}{n} = \frac{n + 2^{k+1} - 1}{n + 2^{k+1} - 2} \cdot \frac{n + 2^{k+1} - 2}{n}$ 的巧妙构造
4. **降阶策略**：奇数时 $(n+1)/2 < n$，偶数时 $n/2 < n$，确保归纳终止

## 相关定理与概念
- **数学归纳法**：证明对所有正整数成立的命题
- **埃及分数**：将分数表示为单位分数之和
- **连分数展开**：数的递归表示方法
