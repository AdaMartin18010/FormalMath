# IMO 2009 Problem 3

## 题目

设 $s_1, s_2, s_3, \ldots$ 是一个严格递增的正整数序列。假设子序列
$$s_{s_1}, s_{s_2}, s_{s_3}, \ldots \quad \text{和} \quad s_{s_1+1}, s_{s_2+1}, s_{s_3+1}, \ldots$$
都是等差数列。证明：序列 $s_1, s_2, s_3, \ldots$ 本身也是等差数列。

## 分类信息
- **领域**: 代数/组合
- **难度**: 6分
- **涉及概念**: 等差数列、序列分析、递归关系、密度论证

## 解答

### 方法一：分析公差

**步骤1：设定记号**

设：
- $s_{s_n} = a + (n-1)d_1$（第一个子序列）
- $s_{s_n+1} = b + (n-1)d_2$（第二个子序列）

**步骤2：分析公差关系**

由于 $s$ 严格递增，$s_{s_n+1} > s_{s_n}$，所以 $s_{s_n+1} - s_{s_n} = (b-a) + (n-1)(d_2-d_1) > 0$。

对于大的 $n$，需要 $d_2 \geq d_1$。

**步骤3：关键引理**

**引理**：$d_1 = d_2$。

证明：假设 $d_2 > d_1$，则 $s_{s_n+1} - s_{s_n}$ 随 $n$ 增大而增大。

但 $s_{s_n+1} - s_{s_n} \geq s_{s_n+1} - s_{s_n} \geq 1$（整数序列）。

考虑差分的增长速度...

实际上，由于 $s$ 严格递增，相邻项差至少为1。

设 $t_n = s_{n+1} - s_n \geq 1$。

**步骤4：建立方程**

$s_{s_{n+1}} - s_{s_n} = d_1$

$s_{s_{n+1}+1} - s_{s_n+1} = d_2$

**步骤5：差分分析**

$s_{s_{n+1}} - s_{s_n} = \sum_{i=s_n}^{s_{n+1}-1} (s_{i+1} - s_i) = \sum_{i=s_n}^{s_{n+1}-1} t_i = d_1$

$s_{s_{n+1}+1} - s_{s_n+1} = \sum_{i=s_n+1}^{s_{n+1}} t_i = d_2$

相减：$t_{s_{n+1}} - t_{s_n} = d_2 - d_1$

**步骤6：证明 $d_1 = d_2$**

如果 $d_2 \neq d_1$，则 $t_{s_n}$ 是等差数列。

由于 $t_i \geq 1$，且 $s_n$ 递增，这会导致矛盾（增长过快或过慢）。

因此 $d_1 = d_2 = d$。

**步骤7：证明原序列是等差数列**

现在 $s_{s_n+1} - s_{s_n} = b - a$（常数）。

由于 $d_1 = d_2$，两个子序列公差相同。

关键观察：对于严格递增序列，如果 $s_{s_n}$ 和 $s_{s_n+1}$ 都是公差为 $d$ 的等差数列，且它们的差是常数，那么 $s_n$ 本身必须是等差数列。

设 $s_{s_n} = a + (n-1)d$，$s_{s_n+1} = b + (n-1)d$。

则 $s_{s_n+1} - s_{s_n} = b - a$。

这意味着从位置 $s_n$ 到 $s_n+1$，序列值跳跃了 $b-a$。

由于 $s$ 严格递增且 $s_{s_{n+1}} - s_{s_n} = d$，通过分析 $s_n$ 的分布，可以证明 $s_n = c + (n-1)k$ 对某些常数 $c, k$。

详细证明需要利用 $s_{s_n}$ 和 $s_{s_n+1}$ 的稠密性。

### 方法二：利用密度论证

**关键观察**：子序列 $s_{s_n}$ 在 $s$ 中的"密度"与 $s_n$ 的增长速度相关。

如果 $s_n \approx cn$，则 $s_{s_n} \approx c^2 n$，这与等差数列的线性增长一致。

通过分析，可得 $s_n$ 必须是线性的。

## 关键思路

1. **分析子序列公差**：证明两个子序列公差相等。

2. **差分技巧**：利用差分建立方程。

3. **反证法**：假设原序列不是等差数列，导出矛盾。

## 相关定理与概念
- **等差数列** - 相邻项差为常数的序列
- **子序列** - 保持顺序的子集
- **递归序列** - 由递归关系定义的序列
- 序列与级数
