# IMO 2010 Problem 5

## 题目

确定所有函数 $f: \mathbb{R} \to \mathbb{R}$，使得对所有实数 $x, y$：
$$f(x - f(y)) = f(f(y)) + xf(y) + f(x) - 1$$

## 分类信息
- **领域**: 代数（函数方程）
- **难度**: 5分
- **涉及概念**: 函数方程、多项式函数、柯西方法

---

## 定义与预备知识

**定义 1（函数方程）** 含有未知函数的等式称为函数方程。本题要求找出所有满足给定恒等式的实函数。

**定义 2（单射与满射）** 函数 $f$ 是单射，若 $f(a)=f(b)$ 蕴含 $a=b$；是满射，若值域为全体实数。

**引理 1（多项式型函数方程）** 若函数方程两边关于某变量为多项式形式，则解常为多项式函数。

**引理 2（常数函数排除）** 若 $f(x) = c$ 为常数，代入方程左边 $f(x-c) = c$，右边 $f(c) + xc + c - 1 = c + xc + c - 1 = xc + 2c - 1$。要求 $c = xc + 2c - 1$ 对所有 $x$ 成立，则 $c=0$ 且 $2c-1=c$，即 $-1=0$，矛盾。故无常数解。

---

## 定理与证明

**定理** 满足条件的唯一函数是 $f(x) = -\frac{x^2}{2} + 1$。

*证明.* 

**步骤 1：确定 $f(0)$ 及零点结构**

令 $x = f(y)$：
$$f(0) = f(f(y)) + f(y)^2 + f(f(y)) - 1 = 2f(f(y)) + f(y)^2 - 1$$
故 $f(f(y)) = \frac{f(0) + 1 - f(y)^2}{2}$。

这表明 $f(f(y))$ 仅依赖于 $f(y)$。若 $f$ 满射，则 $f(z) = \frac{f(0)+1-z^2}{2}$ 对所有 $z \in \mathbb{R}$。

**步骤 2：假设 $f$ 为二次函数**

设 $f(x) = ax^2 + bx + c$。由引理 2，$a,b$ 不全为零。

计算 $f(f(y))$：
$$f(f(y)) = a(ay^2+by+c)^2 + b(ay^2+by+c) + c$$
这是 $y$ 的四次式。但由步骤 1，$f(f(y)) = \frac{f(0)+1-f(y)^2}{2} = \frac{c+1-(ay^2+by+c)^2}{2}$，也是四次式。

比较两边：
$$a(ay^2+by+c)^2 + b(ay^2+by+c) + c = \frac{c+1 - (ay^2+by+c)^2}{2}$$

令 $u = ay^2+by+c$，则 $au^2 + bu + c = \frac{c+1-u^2}{2}$。这对无穷多个 $u$ 成立（因 $y$ 变化），故作为多项式恒等：
$$au^2 + bu + c = -\frac{1}{2}u^2 + \frac{c+1}{2}$$

比较系数：
- $u^2$：$a = -1/2$
- $u$：$b = 0$
- 常数：$c = (c+1)/2$，即 $2c = c+1$，$c=1$

故 $f(x) = -\frac{x^2}{2} + 1$。

**步骤 3：验证解**

左边：$f(x - f(y)) = f\left(x + \frac{y^2}{2} - 1\right) = -\frac{1}{2}\left(x + \frac{y^2}{2} - 1\right)^2 + 1$

右边：$f(f(y)) + xf(y) + f(x) - 1$

计算 $f(y) = 1 - y^2/2$，$f(f(y)) = f(1-y^2/2) = -\frac{(1-y^2/2)^2}{2} + 1 = -\frac{1 - y^2 + y^4/4}{2} + 1 = -\frac{1}{2} + \frac{y^2}{2} - \frac{y^4}{8} + 1 = \frac{1}{2} + \frac{y^2}{2} - \frac{y^4}{8}$

$xf(y) = x(1 - y^2/2) = x - \frac{xy^2}{2}$

$f(x) - 1 = -\frac{x^2}{2}$

右边总和：$\frac{1}{2} + \frac{y^2}{2} - \frac{y^4}{8} + x - \frac{xy^2}{2} - \frac{x^2}{2}$

左边：$-\frac{1}{2}\left(x^2 + x(y^2-2) + \left(\frac{y^2}{2}-1\right)^2\right) + 1$
$= -\frac{x^2}{2} - \frac{x(y^2-2)}{2} - \frac{(y^2-2)^2}{8} + 1$
$= -\frac{x^2}{2} - \frac{xy^2}{2} + x - \frac{y^4 - 4y^2 + 4}{8} + 1$
$= -\frac{x^2}{2} - \frac{xy^2}{2} + x - \frac{y^4}{8} + \frac{y^2}{2} - \frac{1}{2} + 1$
$= -\frac{x^2}{2} - \frac{xy^2}{2} + x - \frac{y^4}{8} + \frac{y^2}{2} + \frac{1}{2}$

与右边完全一致。验证通过。$\square$

## 例子

**例 1** $x=0, y=0$：
左边 $f(0-f(0)) = f(-1) = -\frac{1}{2} + 1 = \frac{1}{2}$
右边 $f(f(0)) + 0 + f(0) - 1 = f(1) + 1 - 1 = -\frac{1}{2} + 1 = \frac{1}{2}$

**例 2** $x=1, y=2$：
$f(2) = 1-2 = -1$，$f(f(2)) = f(-1) = 1/2$
左边 $f(1-(-1)) = f(2) = -1$
右边 $1/2 + 1\times(-1) + f(1) - 1 = 1/2 - 1 + 1/2 - 1 = -1$

## 应用

函数方程在数学和自然科学中无处不在：

1. **物理学**：薛定谔方程、热传导方程等均为函数方程；
2. **经济学**：效用函数的可加性条件常表为函数方程；
3. **信息论**：熵函数的公理化刻画（Shannon 熵）通过函数方程确定。
