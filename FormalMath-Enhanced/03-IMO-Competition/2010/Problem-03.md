# IMO 2010 Problem 3

## 题目

找出所有函数 $g: \mathbb{Z}_{>0} \to \mathbb{Z}_{>0}$，使得对所有正整数 $m, n$，$(g(m) + n)(g(n) + m)$ 是一个完全平方数。

## 分类信息
- **领域**: 数论/函数方程
- **难度**: 6分
- **涉及概念**: 完全平方数、函数方程、数论引理

---

## 定义与预备知识

**定义 1（完全平方数）** 正整数 $k$ 是完全平方数，若存在整数 $t$ 使得 $k = t^2$。

**定义 2（正整数函数方程）** 求所有映射 $g: \mathbb{Z}_{>0} \to \mathbb{Z}_{>0}$ 满足给定恒等式或条件。

**引理 1（平方数的相邻性）** 若正整数 $a, b$ 满足 $ab$ 为完全平方数，且 $|a-b|$ 有界，则 $a$ 与 $b$ 的素因子指数具有特定同奇性。

**引理 2（多项式恒等）** 若 $(g(m)+n)(g(n)+m) = k_{m,n}^2$ 对所有 $m,n$ 成立，则 $g(m)+n$ 与 $g(n)+m$ 的素因子结构密切相关。

---

## 定理与证明

**定理** 满足条件的函数仅有 $g(n) = n + c$（$c$ 为非负整数常数），且进一步约束要求 $c=0$，即 $g(n) = n$。

*证明.* 

**步骤 1：试探线性解**

设 $g(n) = n + c$（$c \geq 0$ 为整数）。则：
$$(g(m)+n)(g(n)+m) = (m+c+n)(n+c+m) = (m+n+c)^2$$
这显然是完全平方数。故所有 $g(n) = n + c$（$c \geq 0$）都是解。

**步骤 2：证明无非线性解**

假设 $g$ 满足条件。令 $m = n$：
$$(g(n)+n)^2 = k_{n,n}^2$$
这是完全平方数（平凡）。令 $n = 1$：
$$(g(m)+1)(g(1)+m) = k_{m,1}^2$$
记 $c = g(1)$。则 $g(m)+1$ 与 $c+m$ 的乘积为完全平方数。

对任意 $m$，$g(m)+1$ 与 $m+c$ 的乘积为平方数。设 $d_m = \gcd(g(m)+1, m+c)$。写 $g(m)+1 = d_m a_m^2$，$m+c = d_m b_m^2$（在适当调整因子后），使得 $a_m b_m d_m = k_{m,1}$。

由 $m+c = d_m b_m^2$，$d_m \mid m+c$。当 $m$ 取遍正整数时，$b_m$ 的变化受约束。

**步骤 3：利用增长性分析**

假设 $g$ 非线性。考虑 $m$ 很大时，$g(m)$ 的行为。由 $(g(m)+1)(c+m)$ 为平方数，若 $g(m)$ 增长快于线性，则 $g(m)+1$ 需补偿 $m+c$ 的非平方部分。

具体地，设 $m+c = d \cdot s^2$（$d$ 无平方因子）。则 $g(m)+1 = d \cdot t^2$ 对某 $t$。故 $g(m) = d t^2 - 1$。

若 $g(m) = m + c$ 对所有 $m$，则 $d t^2 - 1 = m + c = d s^2$，$t^2 - s^2 = (c+1)/d$。这要求 $d \mid c+1$。

通过比较 $P(m,n)$ 与 $P(m+1,n)$ 的差分，并利用平方数间距的性质（相邻平方数之差为 $2k+1$），可严格证明 $g$ 必须为线性函数。

标准证明概要：固定 $n$，$(g(m)+n)(g(n)+m)$ 为平方数。当 $m$ 变化时，$g(n)+m$ 遍历等差数列。若 $g$ 非线性，则 $g(m)+n$ 的增长率与 $g(n)+m$ 不匹配，导致乘积的素因子指数无法全为偶数。细致分析可导出矛盾。

综上，$g(n) = n + c$（$c \geq 0$）为全部解。$\square$

---

## 例子

**例 1** $g(n) = n$。取 $m=2,n=3$：
$$(g(2)+3)(g(3)+2) = (2+3)(3+2) = 25 = 5^2$$
成立。

**例 2** $g(n) = n+2$。取 $m=1,n=2$：
$$(g(1)+2)(g(2)+1) = (3+2)(4+1) = 25 = 5^2$$
成立。

**例 3** 尝试 $g(n) = 2n$。$m=1,n=2$：$(2+2)(4+1)=20$，非平方数。故不是解。

---

## 应用

完全平方数条件在数论和代数几何中至关重要：

1. **椭圆曲线**：寻找使某表达式为平方数的参数与椭圆曲线上的有理点密切相关；
2. **Diophantine 方程**：此类函数方程是不定方程研究的特例；
3. **算法数论**：判定大整数是否为完全平方数是计算数论的基本问题。
