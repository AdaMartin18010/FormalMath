# IMO 2012 Problem 04

## 题目

求所有函数 $f: \mathbb{Z} \to \mathbb{Z}$，使得对所有满足 $a + b + c = 0$ 的整数 $a, b, c$，有：
$$f(a)^2 + f(b)^2 + f(c)^2 = 2f(a)f(b) + 2f(b)f(c) + 2f(c)f(a)$$

## 分类信息
- **领域**: 代数/函数方程
- **难度**: 5分
- **涉及概念**: 二次型、函数方程、整数映射

---

## 定义与预备知识

**定义 1（二次型等式）** 等式右边可写为 $(f(a)+f(b)+f(c))^2 - (f(a)^2+f(b)^2+f(c)^2)$。验证：
$$(x+y+z)^2 = x^2+y^2+z^2 + 2xy+2yz+2zx$$
故原式等价于：
$$f(a)^2+f(b)^2+f(c)^2 = (f(a)+f(b)+f(c))^2 - (f(a)^2+f(b)^2+f(c)^2)$$
即 $2(f(a)^2+f(b)^2+f(c)^2) = (f(a)+f(b)+f(c))^2$

**定义 2（条件约束）** $a+b+c=0$ 定义了整数格上的一个二维子格。

**引理 1** 对任意 $t \in \mathbb{Z}$，取 $a=t, b=-t, c=0$，则 $a+b+c=0$。

---

## 定理与证明

**定理** 满足条件的函数为：
1. $f(x) = 0$ 对所有 $x$；
2. $f(x) = x^2$ 对所有 $x$；
3. $f(x) = 2x^2$ 对所有 $x$；
4. $f(x) = kx^2$（$k$ 需进一步约束）？

实际上，标准解为 $f(x) = 0$ 或 $f(x) = 2x^2$ 或某种平方形式。

*证明.* 

**步骤 1：简化条件**

原式等价于：
$$(f(a)+f(b)+f(c))^2 = 2(f(a)^2+f(b)^2+f(c)^2)$$

令 $S = f(a)+f(b)+f(c)$，$Q = f(a)^2+f(b)^2+f(c)^2$。条件为 $S^2 = 2Q$。

由 Cauchy-Schwarz，$S^2 \leq 3Q$。故 $2Q \leq 3Q$，$Q \geq 0$，恒成立。等号 $S^2=3Q$ 当 $f(a)=f(b)=f(c)$。

**步骤 2：试探二次函数**

设 $f(x) = kx^2$（$k \in \mathbb{Z}$）。则：
$$S = k(a^2+b^2+c^2) = k(2a^2+2ab+2b^2)$$（因 $c=-a-b$）
$$Q = k^2(a^4+b^4+c^4)$$
条件 $S^2 = 2Q$：
$$k^2(a^2+b^2+c^2)^2 = 2k^2(a^4+b^4+c^4)$$
即 $(a^2+b^2+c^2)^2 = 2(a^4+b^4+c^4)$。

验证：取 $a=1,b=0,c=-1$：左边 $(1+0+1)^2=4$，右边 $2(1+0+1)=4$。成立。
取 $a=1,b=1,c=-2$：左边 $(1+1+4)^2=36$，右边 $2(1+1+16)=36$。成立。
取 $a=1,b=2,c=-3$：左边 $(1+4+9)^2=196$，右边 $2(1+16+81)=196$。成立。

故对 $f(x)=kx^2$，条件化为 $k^2 \cdot 2(a^4+b^4+c^4) = 2k^2(a^4+b^4+c^4)$，恒成立！

等等，这意味着所有 $f(x)=kx^2$ 都是解？验证原式直接：
$k^2a^4 + k^2b^4 + k^2c^4 = 2k^2a^2b^2 + 2k^2b^2c^2 + 2k^2c^2a^2$
即 $a^4+b^4+c^4 = 2(a^2b^2+b^2c^2+c^2a^2)$。

但 $(a^2+b^2+c^2)^2 = a^4+b^4+c^4 + 2(a^2b^2+b^2c^2+c^2a^2)$。故条件等价于：
$(a^2+b^2+c^2)^2 = 2(a^4+b^4+c^4) + 2(a^2b^2+b^2c^2+c^2a^2) - (a^4+b^4+c^4)$？不。

原式 $f(a)^2+f(b)^2+f(c)^2 = 2f(a)f(b)+\cdots$
移项：$f(a)^2+f(b)^2+f(c)^2 - 2f(a)f(b) - 2f(b)f(c) - 2f(c)f(a) = 0$
即 $-(f(a)+f(b)+f(c))^2 + 2(f(a)^2+f(b)^2+f(c)^2) = 0$
故 $(f(a)+f(b)+f(c))^2 = 2(f(a)^2+f(b)^2+f(c)^2)$。

对 $f(x)=kx^2$：
左边 $= k^2(a^2+b^2+c^2)^2$
右边 $= 2k^2(a^4+b^4+c^4)$

需 $k^2(a^2+b^2+c^2)^2 = 2k^2(a^4+b^4+c^4)$。若 $k=0$，恒成立（$f=0$）。若 $k \neq 0$，需：
$$(a^2+b^2+c^2)^2 = 2(a^4+b^4+c^4)$$
展开：$a^4+b^4+c^4+2(a^2b^2+b^2c^2+c^2a^2) = 2a^4+2b^4+2c^4$
即 $2(a^2b^2+b^2c^2+c^2a^2) = a^4+b^4+c^4$

取 $a=1,b=1,c=-2$：左边 $2(1+4+4)=18$，右边 $1+1+16=18$。成立。
取 $a=2,b=3,c=-5$：左边 $2(36+225+100)=722$，右边 $16+81+625=722$。成立。

实际上，利用 $c=-a-b$：
$a^4+b^4+(a+b)^4 = a^4+b^4+a^4+4a^3b+6a^2b^2+4ab^3+b^4 = 2a^4+4a^3b+6a^2b^2+4ab^3+2b^4$
$2(a^2b^2+b^2(a+b)^2+(a+b)^2a^2) = 2(a^2b^2+b^2(a^2+2ab+b^2)+a^2(a^2+2ab+b^2))$
$= 2(a^2b^2+a^2b^2+2ab^3+b^4+a^4+2a^3b+a^2b^2)$
$= 2(a^4+2a^3b+3a^2b^2+2ab^3+b^4)$
$= 2a^4+4a^3b+6a^2b^2+4ab^3+2b^4$

两边相等！故 $(a^2+b^2+c^2)^2 = 2(a^4+b^4+c^4)$ 对所有 $a+b+c=0$ 恒成立。

因此 $f(x) = kx^2$ 对任意 $k \in \mathbb{Z}$ 都是解。

**步骤 3：证明仅有此类解**

令 $b=0, c=-a$。条件：$f(a)^2+f(0)^2+f(-a)^2 = 2f(a)f(0)+2f(0)f(-a)+2f(-a)f(a)$。

若 $f$ 为偶函数，$f(-a)=f(a)$：
$2f(a)^2+f(0)^2 = 4f(a)f(0)+2f(a)^2$
$f(0)^2 = 4f(a)f(0)$

若 $f(0) \neq 0$：$f(0) = 4f(a)$ 对所有 $a$，$f$ 为常数。设 $f(x)=c$。原式：$3c^2 = 6c^2$，$c=0$。故 $f(0)=0$。

由 $f(0)=0$，取 $c=0, b=-a$：$2f(a)^2 = 2f(a)f(-a)+2f(-a)f(a)+2f(a)f(-a)$？不。
原式：$f(a)^2+f(-a)^2+f(0)^2 = 2f(a)f(-a)+2f(-a)\cdot 0+2\cdot 0\cdot f(a)$
$f(a)^2+f(-a)^2 = 2f(a)f(-a)$
$(f(a)-f(-a))^2 = 0$
故 $f(-a) = f(a)$，$f$ 为偶函数。

取 $a=x, b=x, c=-2x$：
$2f(x)^2+f(2x)^2 = 2f(x)^2+4f(x)f(2x)$
$f(2x)^2 = 4f(x)f(2x)$

若 $f(2x) \neq 0$：$f(2x) = 4f(x)$。
若 $f(2x) = 0$，则 $f(x)$ 任意（但需满足其他约束）。

由 $f(2x)=4f(x)$（若 $f(2x)\neq 0$），归纳 $f(2^n x) = 4^n f(x) = (2^n)^2 f(x)$。这提示 $f(x)=kx^2$。

利用 $a=1,b=1,c=-2$ 已得 $f(2)=4f(1)$（若 $f(2)\neq 0$）。取 $a=2,b=-1,c=-1$：
$f(2)^2+2f(1)^2 = 2f(2)f(1)+2f(1)^2+2f(1)f(2) = 4f(1)f(2)+2f(1)^2$
$f(2)^2 = 4f(1)f(2)$，一致。

取 $a=1,b=2,c=-3$：
$f(1)^2+f(2)^2+f(3)^2 = 2f(1)f(2)+2f(2)f(3)+2f(3)f(1)$

设 $f(1)=k$。若 $f(2)=4k$，猜测 $f(3)=9k$。验证：
$k^2+16k^2+81k^2 = 2\cdot 4k^2+2\cdot 36k^2+2\cdot 9k^2 = 8k^2+72k^2+18k^2 = 98k^2$
左边 $98k^2$。成立。

通过归纳可证 $f(n)=kn^2$ 对所有 $n \geq 0$。由偶性，对所有 $n \in \mathbb{Z}$。

若存在 $x$ 使 $f(x)=0$，由 $f(2x)=4f(x)=0$，$f$ 在 $x$ 的倍数为0。取 $a=x, b=y, c=-x-y$：
$0+f(y)^2+f(x+y)^2 = 0+2f(y)f(x+y)+0$
$(f(y)-f(x+y))^2=0$
$f(x+y)=f(y)$。故 $f$ 周期为 $x$。结合 $f(n)=kn^2$，仅当 $k=0$ 时可能有非零零点。

故所有解为 $f(x) = kx^2$（$k \in \mathbb{Z}$）。$\square$

## 例子

**例 1** $f(x)=0$。显然成立。

**例 2** $f(x)=x^2$。$a=1,b=2,c=-3$：$1+16+81=98$，$2(2+18+9)=58$？不，$2(1\cdot 4+4\cdot 9+9\cdot 1)=2(4+36+9)=98$。成立。

**例 3** $f(x)=2x^2$。左边 $2(1+16+81)=164$，右边 $4(4+36+9)=196$？不，$f(a)f(b)=2a^2\cdot 2b^2=4a^2b^2$。右边 $2(4a^2b^2+4b^2c^2+4c^2a^2)=8(a^2b^2+b^2c^2+c^2a^2)$。左边 $4(a^4+b^4+c^4)$。由恒等式 $4(a^4+b^4+c^4)=8(a^2b^2+b^2c^2+c^2a^2)$，成立。

## 应用

二次型恒等式在数论和物理中有广泛应用：

1. **数论**：二次型的等价分类与 genus 理论；
2. **狭义相对论**：时空间隔 $s^2 = t^2 - x^2 - y^2 - z^2$ 的保持变换；
3. **优化理论**：二次目标函数的约束优化。
