# MSC2020 数学主题分类编码标准说明

## 1. MSC2020 简介

MSC2020 (Mathematics Subject Classification 2020) 是由美国数学学会(AMS)和德国数学文摘(Zentralblatt MATH)联合维护的数学文献分类体系。

### 主要特点

- **层级结构**：三级编码体系 (XX-XX-XX)
- **覆盖范围**：涵盖纯数学、应用数学及交叉领域
- **使用范围**：Math Reviews、Zentralblatt MATH、arXiv等
- **更新周期**：约10年更新一次

### 统计数据

- 63个顶级二位码分类
- 529个三级分类码
- 6,022个五级分类码

## 2. 编码结构

### 2.1 编码格式

MSC2020采用层级编码结构：

```
[顶级二位码]-[二级字母码]-[三级二位码]
   │              │            │
   │              │            └── 具体子主题 (01-99)
   │              └── 二级分类字母 (A-Z, -)
   └── 顶级数学领域 (00-97)
```

### 2.2 编码示例

| 编码 | 含义 |
|------|------|
| `11` | 数论 (顶级) |
| `11F` | 模形式与自守形式 (二级) |
| `11F27` | Theta级数；Weil表示 (三级) |

### 2.3 特殊编码

| 编码模式 | 含义 |
|----------|------|
| `XX-XX` | 表示该级别下的所有主题 |
| `XX-xx` | 与XX-XX等效，用于统一格式 |
| `99` | 表示"本节中未定义的其他主题" |
| `-` | 特殊材料类型（综述、软件、会议录等） |

## 4. 标注规范

### 4.1 YAML 格式

每个文档的MSC标注采用YAML frontmatter格式：

```yaml
---
msc_primary: "XX-XX-XX"      # 主要分类（1个）
msc_secondary:               # 次要分类（0-5个）
  - "XX-XX-XX"
  - "XX-XX-XX"
keywords:                    # 关键词（3-8个）
  - keyword1
  - keyword2
  - keyword3
---
```

### 4.2 标注原则

1. **主要分类**：文档最核心的数学领域（唯一）
2. **次要分类**：相关的辅助领域（最多5个）
3. **精确度**：优先使用五级码，必要时可用三级码
4. **时效性**：使用MSC2020版本，不使用旧版编码

### 4.3 编码选择指南

| 文档特征 | 推荐编码级别 |
|----------|--------------|
| 综述性文档 | 二级或三级码 |
| 研究论文 | 五级码 |
| 跨领域研究 | 多个五级码 |
| 教学材料 | 三级码 |
