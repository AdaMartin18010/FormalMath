## 标注格式

每个文档使用YAML frontmatter格式进行标注：

```yaml
---
msc_primary: "XX-XX-XX"      # 主要分类（1个）
msc_secondary:               # 次要分类（0-5个）
  - "XX-XX-XX"
  - "XX-XX-XX"
keywords:                    # 关键词（3-8个）
  - keyword1
  - keyword2
  - keyword3
---
```

## 使用指南

### 1. 查找编码

1. 根据文档主题选择对应的分支索引文件
2. 在索引中查找最相关的五级编码（格式：XX-XX-XX）
3. 确认编码在MSC2020官方列表中存在

### 2. 应用标注

1. 在文档开头添加YAML frontmatter
2. 填写`msc_primary`为主要分类（1个）
3. 填写`msc_secondary`为次要分类（最多5个）
4. 添加3-8个相关关键词

### 3. 质量检查

- [ ] 编码存在于MSC2020官方列表
- [ ] 主要分类唯一且准确
- [ ] 次要分类不超过5个
- [ ] 关键词与内容一致
- [ ] YAML格式正确
