# FormalMath 备份配置指南

## 概述

本文档描述如何配置 FormalMath 系统的自动备份任务，包括备份计划、存储管理和监控设置。

## 目录结构

```
backup/
├── scripts/          # 备份脚本
│   ├── backup.ps1           # 主备份脚本
│   ├── restore.ps1          # 恢复脚本
│   └── verify_backup.ps1    # 备份验证脚本
├── docs/             # 文档
│   ├── 恢复流程文档.md
│   ├── 灾难恢复计划.md
│   └── 备份配置指南.md
├── archive/          # 备份存储
│   ├── backup_full_*.zip
│   ├── backup_db_*.zip
│   └── ...
└── logs/             # 日志文件
    ├── backup_*.log
    ├── restore_*.log
    └── verify_*.log
```

## 备份类型

### 1. 完整备份 (Full)

备份所有数据：数据库、配置文件、数据目录、知识图谱、Lean4代码。

```powershell
.\scripts\backup.ps1 -BackupType full
```

### 2. 数据库备份 (DB)

仅备份数据库文件。

```powershell
.\scripts\backup.ps1 -BackupType db
```

### 3. 配置备份 (Config)

仅备份配置文件。

```powershell
.\scripts\backup.ps1 -BackupType config
```

### 4. 增量备份 (Incremental)

仅备份最近24小时内修改的文件。

```powershell
.\scripts\backup.ps1 -BackupType incremental
```

## Windows 任务计划程序配置

### 创建每日完整备份任务

1. **打开任务计划程序**

   ```
   开始菜单 → Windows管理工具 → 任务计划程序
   ```

2. **创建基本任务**
   - 名称: `FormalMath Full Backup`
   - 描述: `FormalMath 系统每日完整备份`

3. **设置触发器**
   - 频率: 每天
   - 开始时间: 02:00:00
   - 每: 1 天

4. **设置操作**
   - 操作: 启动程序
   - 程序/脚本: `powershell.exe`
   - 参数:

     ```
     -ExecutionPolicy Bypass -File "G:\_src\FormalMath\FormalMath-Enhanced\backup\scripts\backup.ps1" -BackupType full -RetentionDays 30
     ```

   - 起始于: `G:\_src\FormalMath\FormalMath-Enhanced\backup`

5. **设置条件**
   - 仅在以下网络连接可用时启动: 任何连接
   - 如果计算机使用电池电源，则停止: 否
   - 唤醒计算机运行此任务: 是

6. **设置高级选项**
   - 允许按需运行任务: 是
   - 如果任务失败，按以下频率重新启动: 每 15 分钟，尝试 3 次
   - 如果运行时间超过: 2 小时，则停止任务
   - 如果请求后任务还在运行，强制将其停止: 是

### 创建每小时数据库备份任务

1. **创建任务**
   - 名称: `FormalMath DB Backup`
   - 描述: `FormalMath 数据库每小时备份`

2. **设置触发器**
   - 开始任务: 按预定计划
   - 设置: 每天
   - 开始时间: 00:00:00
   - 重复任务间隔: 1 小时
   - 持续时间: 无限期

3. **设置操作**

   ```
   -ExecutionPolicy Bypass -File "G:\_src\FormalMath\FormalMath-Enhanced\backup\scripts\backup.ps1" -BackupType db -RetentionDays 7
   ```

### 创建每日验证任务

1. **创建任务**
   - 名称: `FormalMath Backup Verify`
   - 描述: `验证最近备份的完整性`

2. **设置触发器**
   - 每天 06:00:00

3. **设置操作**

   ```
   -ExecutionPolicy Bypass -File "G:\_src\FormalMath\FormalMath-Enhanced\backup\scripts\verify_backup.ps1" -Detailed
   ```

## 使用 PowerShell 配置计划任务

### 完整备份任务

```powershell
$action = New-ScheduledTaskAction `
    -Execute "powershell.exe" `
    -Argument '-ExecutionPolicy Bypass -File "G:\_src\FormalMath\FormalMath-Enhanced\backup\scripts\backup.ps1" -BackupType full -RetentionDays 30' `
    -WorkingDirectory "G:\_src\FormalMath\FormalMath-Enhanced\backup"

$trigger = New-ScheduledTaskTrigger `
    -Daily `
    -At "02:00"

$settings = New-ScheduledTaskSettingsSet `
    -AllowStartIfOnBatteries `
    -DontStopIfGoingOnBatteries `
    -WakeToRun `
    -StartWhenAvailable

$principal = New-ScheduledTaskPrincipal `
    -UserId "$env:USERDOMAIN\$env:USERNAME" `
    -LogonType Interactive

Register-ScheduledTask `
    -TaskName "FormalMath Full Backup" `
    -Action $action `
    -Trigger $trigger `
    -Settings $settings `
    -Principal $principal `
    -Description "FormalMath 系统每日完整备份"
```

### 数据库备份任务

```powershell
$action = New-ScheduledTaskAction `
    -Execute "powershell.exe" `
    -Argument '-ExecutionPolicy Bypass -File "G:\_src\FormalMath\FormalMath-Enhanced\backup\scripts\backup.ps1" -BackupType db -RetentionDays 7'

$trigger = New-ScheduledTaskTrigger `
    -Once `
    -At (Get-Date) `
    -RepetitionInterval (New-TimeSpan -Hours 1) `
    -RepetitionDuration (New-TimeSpan -Days 365)

$settings = New-ScheduledTaskSettingsSet `
    -AllowStartIfOnBatteries `
    -DontStopIfGoingOnBatteries

Register-ScheduledTask `
    -TaskName "FormalMath DB Backup" `
    -Action $action `
    -Trigger $trigger `
    -Settings $settings `
    -Description "FormalMath 数据库每小时备份"
```

### 验证任务

```powershell
$action = New-ScheduledTaskAction `
    -Execute "powershell.exe" `
    -Argument '-ExecutionPolicy Bypass -File "G:\_src\FormalMath\FormalMath-Enhanced\backup\scripts\verify_backup.ps1" -Detailed'

$trigger = New-ScheduledTaskTrigger -Daily -At "06:00"

Register-ScheduledTask `
    -TaskName "FormalMath Backup Verify" `
    -Action $action `
    -Trigger $trigger `
    -Description "验证最近备份的完整性"
```

## 存储管理

### 本地存储

```powershell
# 设置备份路径
$BackupPath = "G:\_src\FormalMath\FormalMath-Enhanced\backup\archive"

# 检查磁盘空间
$drive = Get-PSDrive G
$freeSpaceGB = [math]::Round($drive.Free / 1GB, 2)
$usedSpacePercent = [math]::Round(($drive.Used / ($drive.Used + $drive.Free)) * 100, 2)

Write-Host "可用空间: $freeSpaceGB GB"
Write-Host "已使用: $usedSpacePercent%"

# 自动清理旧备份
.\scripts\backup.ps1 -BackupType full -RetentionDays 30
```

### 异地备份同步

```powershell
# 使用 Robocopy 同步到网络位置
$source = "G:\_src\FormalMath\FormalMath-Enhanced\backup\archive"
$destination = "\\backup-server\FormalMath\backups"

robocopy $source $destination /MIR /R:3 /W:10 /LOG:"G:\_src\FormalMath\FormalMath-Enhanced\backup\logs\sync.log"

# 或使用 PowerShell
Copy-Item -Path "$source\*" -Destination $destination -Recurse -Force
```

### 云存储备份（可选）

```powershell
# Azure Blob Storage 示例
# 需要安装 Az 模块: Install-Module Az

$storageAccount = "yourstorageaccount"
$container = "formalmath-backups"
$context = New-AzStorageContext -StorageAccountName $storageAccount -UseConnectedAccount

Get-ChildItem -Path "G:\_src\FormalMath\FormalMath-Enhanced\backup\archive" -Filter "backup_*.zip" |
    ForEach-Object {
        Set-AzStorageBlobContent `
            -File $_.FullName `
            -Container $container `
            -Blob $_.Name `
            -Context $context `
            -Force
    }
```

## 监控与告警

### 备份状态监控

```powershell
# 检查最近备份
$latestBackup = Get-ChildItem -Path "G:\_src\FormalMath\FormalMath-Enhanced\backup\archive" -Filter "backup_*.zip" |
    Sort-Object LastWriteTime -Descending |
    Select-Object -First 1

if ($latestBackup) {
    $hoursSinceBackup = ((Get-Date) - $latestBackup.LastWriteTime).TotalHours
    if ($hoursSinceBackup -gt 26) {
        Write-Warning "警告: 超过26小时没有成功备份!"
        # 发送告警邮件或通知
    }
}
```

### 磁盘空间监控

```powershell
$drive = Get-PSDrive G
$freePercent = ($drive.Free / ($drive.Used + $drive.Free)) * 100

if ($freePercent -lt 20) {
    Write-Warning "警告: 磁盘空间不足 ($([math]::Round($freePercent, 2))% 剩余)!"
    # 清理旧备份
    .\scripts\backup.ps1 -BackupType full -RetentionDays 14
}
```

### 备份失败告警

```powershell
# 检查最近备份日志
$latestLog = Get-ChildItem -Path "G:\_src\FormalMath\FormalMath-Enhanced\backup\logs" -Filter "backup_*.log" |
    Sort-Object LastWriteTime -Descending |
    Select-Object -First 1

if ($latestLog) {
    $logContent = Get-Content $latestLog.FullName -Raw
    if ($logContent -match "ERROR") {
        Write-Warning "警告: 最近备份包含错误!"
        # 发送告警
    }
}
```

## 故障排除

### 备份任务未运行

1. 检查任务计划程序历史记录
2. 确认 PowerShell 执行策略
3. 检查日志文件

```powershell
# 检查执行策略
Get-ExecutionPolicy

# 临时设置执行策略
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process
```

### 磁盘空间不足

```powershell
# 查看备份目录大小
Get-ChildItem -Path "G:\_src\FormalMath\FormalMath-Enhanced\backup\archive" |
    Measure-Object -Property Length -Sum

# 手动清理旧备份
Get-ChildItem -Path "G:\_src\FormalMath\FormalMath-Enhanced\backup\archive" -Filter "backup_*.zip" |
    Where-Object { $_.LastWriteTime -lt (Get-Date).AddDays(-30) } |
    Remove-Item -Force
```

### 备份文件损坏

```powershell
# 验证所有备份
.\scripts\verify_backup.ps1 -CheckAll -Detailed
```

## 最佳实践

1. **定期测试恢复**: 每季度执行一次完整恢复测试
2. **多重备份**: 本地 + 异地 + 云端
3. **监控告警**: 设置备份失败和磁盘空间告警
4. **文档更新**: 保持文档和脚本同步更新
5. **权限管理**: 限制备份目录访问权限

## 附录

### 备份文件命名规范

```
backup_<type>_<timestamp>.zip

类型:
- full: 完整备份
- db: 数据库备份
- config: 配置备份
- incremental: 增量备份

时间戳格式: yyyyMMdd_HHmmss

示例:
- backup_full_20250401_020000.zip
- backup_db_20250401_120000.zip
```

# 可选: 设置环境变量
[Environment]::SetEnvironmentVariable("FORMALMATH_BACKUP_PATH", "G:\Backups", "Machine")
