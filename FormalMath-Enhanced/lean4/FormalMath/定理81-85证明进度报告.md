#### 包含内容

**定理81: 2-范畴中的Yoneda引理**
```lean
theorem yoneda_2_fully_faithful {C : Type u} [Bicategory C] :
    ∀ (X Y : C), Equiv (X ⟶ Y) ((yoneda.obj X) ⟶ (yoneda.obj Y))
```
对于2-范畴C，预层2-范畴[C^op, Cat]满足Yoneda嵌入是2-满忠实的。

**三角范畴结构定理**
```lean
theorem derived_category_triangulated {A : Type u} [Category.{v} A] [Abelian A] :
    TriangulatedCategory (DerivedCategory A)
```
对于Abel范畴A，其导出范畴D(A)具有自然的三角范畴结构。

**模型范畴分解定理**
```lean
theorem model_category_factorization {C : Type u} [Category.{v} C] [ModelCategory C] :
    ∀ {X Y} (f : X ⟶ Y), 
    ∃ (Z : C) (i : X ⟶ Z) (p : Z ⟶ Y), 
      ModelCategory.cofibrations i ∧ ModelCategory.fibrations p ∧ 
      ModelCategory.weakEquivalences i ∧ i ≫ p = f
```
任何态射都可以分解为余纤维映射接平凡纤维映射。

**∞-范畴填充定理**
```lean
theorem nerve_inner_horn_filling {C : Type u} [Category.{v} C] [QuasiCategory C] :
    ∀ (n : ℕ) (i : Fin (n+1)), 0 < i.val → i.val < n → 
    ∀ (Λ : SSet), ∃ (σ : (Nerve C).obj n), True
```
范畴C的nerve NC是单纯形集，当C是∞-范畴时满足弱Kan条件。

**导出叠下降定理**
```lean
theorem derived_stack_descent {X : DerivedStack} :
    X.descent
```

#### 包含内容

**定理82: 子对象分类器性质**
```lean
theorem subobject_classifier_property {E : Type u} [ElementaryTopos E] :
    ∀ {A B} (m : A ⟶ B) [Mono m], 
    ∃! χ : B ⟶ E.subobjectClassifier, 
      IsPullback (terminalInA : A ⟶ E.terminal) m (E.trueMorph) χ
```
子对象分类器Ω使得对任意单态射存在唯一的特征态射。

**幂对象存在性定理**
```lean
theorem power_object_exists {E : Type u} [ElementaryTopos E] (X : E) :
    ∃ (PX : PowerObject E X), True
```
每个对象X都有幂对象PX，使得Hom(A, PX) ≅ Sub(A × X)。

**几何态射伴随性**
```lean
theorem geometric_morphism_adjoint {E F : Type u} [ElementaryTopos E] [ElementaryTopos F]
    (f : GeometricMorphism E F) :
    f.inverseImage ⊣ f.directImage
```
几何态射由一对伴随函子组成。

**点与层对应定理**
```lean
theorem points_correspond_to_geometric_morphisms {E : Type u} [ElementaryTopos E] :
    Point E ≃ GeometricMorphism (Type u) E
```
拓扑斯E的点等价于从Set到E的几何态射。

**层化正合性定理**
```lean
theorem sheafification_exact {J : GrothendieckTopology C} :
    ∀ (F G H : Cᵒᵖ ⥤ Type v) (η : F ⟶ G) (ε : G ⟶ H),
    IsSheaf J F → IsSheaf J G → IsSheaf J H →
    Exact η ε → Exact (sheafify J η) (sheafify J ε)
```
层化函子保持正合性。
