# 路径评估指标体系

## 概述

本文档定义了用于评估学习路径质量的完整指标体系。所有指标均在0-1范围内归一化，便于综合评估。

## 指标体系架构

```
路径质量评分
├── 效率指标 (40%)
│   ├── 完成率 (25%)
│   ├── 时间效率 (10%)
│   └── 学习速度 (5%)
├── 效果指标 (40%)
│   ├── 掌握度提升 (20%)
│   ├── 知识保持率 (12%)
│   └── 迁移能力 (8%)
└── 体验指标 (20%)
    ├── 参与度 (10%)
    ├── 满意度 (6%)
    └── 辍学风险 (4%)
```

## 详细指标定义

### 一、效率指标

#### 1. 完成率 (Completion Rate)
**权重**: 25%

**定义**: 成功完成路径节点的比例

**计算公式**:
```
完成率 = 已完成节点数 / 总节点数
```

**评分标准**:
- 0.9-1.0: 优秀
- 0.7-0.9: 良好
- 0.5-0.7: 一般
- <0.5: 需改进

**改进建议**:
- 完成率低 → 降低路径难度、增加支持材料

---

#### 2. 时间效率 (Time Efficiency)
**权重**: 10%

**定义**: 实际用时与估计用时的比值

**计算公式**:
```
时间效率 = min(估计时间 / 实际时间, 1.5) / 1.5
```

**评分标准**:
- >0.85: 高效
- 0.7-0.85: 正常
- <0.7: 效率偏低

**影响因素**:
- 难度估计准确性
- 用户能力匹配度
- 干扰因素

---

#### 3. 学习速度 (Learning Velocity)
**权重**: 5%

**定义**: 单位时间内的掌握度提升

**计算公式**:
```
学习速度 = 总掌握度提升 / 总学习时间(小时)
```

**参考值**:
- >0.8: 快速学习
- 0.4-0.8: 正常速度
- <0.4: 需要支持

---

### 二、效果指标

#### 4. 掌握度提升 (Mastery Improvement)
**权重**: 20%

**定义**: 学习前后掌握度的平均提升

**计算公式**:
```
掌握度提升 = mean(最终掌握度 - 初始掌握度)
```

**数据来源**:
- 前测/后测成绩
- 知识追踪模型估计
- 练习正确率

**评分标准**:
- >0.6: 显著提升
- 0.3-0.6: 正常提升
- <0.3: 效果不佳

---

#### 5. 知识保持率 (Retention Rate)
**权重**: 12%

**定义**: 延迟测试表现与即时测试表现的比值

**计算公式**:
```
保持率 = 延迟测试分数 / 即时测试分数
```

**测试间隔**:
- 推荐: 7天后
- 最低: 3天后

**参考值**:
- >0.85: 优秀保持
- 0.7-0.85: 正常
- <0.7: 需要复习

---

#### 6. 知识迁移能力 (Transfer Score)
**权重**: 8%

**定义**: 解决新类型问题的能力

**测量方法**:
- 变式题目测试
- 跨情境应用题
- 综合问题解决

**计算公式**:
```
迁移分数 = 迁移题目正确率
```

**评分标准**:
- >0.75: 强迁移能力
- 0.5-0.75: 一般
- <0.5: 需加强

---

### 三、体验指标

#### 7. 参与度 (Engagement Score)
**权重**: 10%

**定义**: 用户在学习过程中的投入程度

**计算维度**:
- 会话时长 (30%)
- 交互次数 (30%)
- 重试率 (20%)
- 主动探索 (20%)

**计算公式**:
```
参与度 = 0.3*会话归一化 + 0.3*交互归一化 + 
         0.2*重试率 + 0.2*探索度
```

**参考值**:
- >0.8: 高度参与
- 0.5-0.8: 正常参与
- <0.5: 参与度低

---

#### 8. 用户满意度 (Satisfaction Score)
**权重**: 6%

**定义**: 用户对学习体验的满意程度

**测量方式**:
- 5分制评分
- NPS净推荐值
- 开放式反馈

**计算公式**:
```
满意度 = 用户评分 / 5
```

**评分标准**:
- >0.85: 非常满意
- 0.6-0.85: 满意
- <0.6: 需改进

---

#### 9. 辍学风险 (Dropout Risk)
**权重**: 4%

**定义**: 用户放弃学习的可能性

**风险因素**:
- 完成率低 (<30%)
- 参与度低 (<30%)
- 进展缓慢 (<0.1掌握度/小时)

**计算公式**:
```
辍学风险 = min(风险因素之和, 1.0)
```

**风险等级**:
- 0-0.3: 低风险
- 0.3-0.6: 中风险
- >0.6: 高风险

**干预措施**:
- 高风险 → 人工干预、简化路径
- 中风险 → 增加提醒、提供支持

---

## 综合评分

### 总体公式

```
综合评分 = 
    0.25 * 完成率 +
    0.10 * 时间效率 +
    0.05 * 学习速度 +
    0.20 * 掌握度提升 +
    0.12 * 知识保持率 +
    0.08 * 迁移能力 +
    0.10 * 参与度 +
    0.06 * 满意度 +
    0.04 * (1 - 辍学风险)
```

### 评分等级

| 综合评分 | 等级 | 建议 |
|---------|------|------|
| 0.90-1.00 | A+ (卓越) | 推广为最佳实践 |
| 0.80-0.90 | A (优秀) | 可作为标准路径 |
| 0.70-0.80 | B (良好) | 小幅优化即可 |
| 0.60-0.70 | C (及格) | 需要优化改进 |
| 0.50-0.60 | D (较差) | 大幅修改或重做 |
| <0.50 | F (失败) | 废弃或完全重做 |

## 数据收集要求

### 必需数据
- 用户ID
- 路径ID
- 开始/结束时间
- 完成节点记录
- 掌握度评估
- 时间花费

### 可选数据
- 交互日志
- 满意度评分
- 延迟测试成绩
- 迁移测试成绩
- 学习行为数据

### 隐私保护
- 所有数据匿名化处理
- 仅用于系统改进
- 遵守数据保护法规
