## 🚀 使用指南

### 1. 快速开始

```bash
cd FormalMath-Enhanced

# 查看项目总览
cat 00-Project-Status/00-PROJECT-OVERVIEW.md

# 使用AI工具
cd 05-AI-Formalization
pip install -r requirements.txt
python tools/auto_formalize.py

# 或使用Docker
docker build -t formalmath-ai .
docker run -it formalmath-ai
```

### 2. 查找MSC编码

cat 01-MSC-Coding/02-代数结构-MSC索引.md

### 3. 查看定理注释

cat 02-Mathlib4-Annotations/Algebra/lagrange-theorem.md

### 4. 查看IMO题目

cat 03-IMO-Competition/2024/Problem-06.md

### 5. 运行AI形式化

```python
from tools.auto_formalize import quick_formalize
result = quick_formalize("证明拉格朗日定理...")
```
