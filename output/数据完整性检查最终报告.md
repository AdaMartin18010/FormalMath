### 方案C: 批量修复链接

对于链接错误，可以使用批量替换：

```powershell
# 示例：批量修复常见路径变更
Get-ChildItem -Recurse -Filter "*.md" | ForEach-Object {
    $content = Get-Content $_.FullName -Raw
    $content = $content -replace "../view/资料库/", "../00-归档/03-view文件夹/view/资料库/"
    Set-Content $_.FullName $content -NoNewline
}
```
