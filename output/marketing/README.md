## 🎯 使用指南

### 场景一：项目发布会

**推荐材料**：
1. 使用 [01-项目介绍文案.md] 准备演讲稿
2. 使用 [05-新闻稿模板.md] 的"项目发布通稿"向媒体发布
3. 使用 [06-演示视频脚本.md] 制作发布会演示视频

### 场景二：社交媒体推广

**推荐材料**：
1. 使用 [04-社交媒体文案.md] 直接发布或改编
2. 针对不同平台选择对应文案（Twitter/微博/LinkedIn/知乎等）

**推荐材料**：
1. 使用 [01-项目介绍文案.md] 准备核心话术

3. 使用 [03-竞品对比表格.md] 说明差异化优势

**推荐材料**：
1. 使用 [01-项目介绍文案.md] 的"完整版本"

## 📝 关键话术

### 一句话定位

> FormalMath —— 全球首个集数学家理念体系、形式化证明、智能学习系统、国际权威对齐于一体的综合性数学知识平台

### 三大核心卖点

1. **独特的数学家理念体系** —— 62位数学家深度分析，从思想史角度理解数学
2. **国际权威课程对齐** —— 与MIT、Harvard等九校121+门课程详细对应
3. **智能学习生态系统** —— 诊断、评估、自适应路径三位一体的个性化学习

### 差异化优势

- 比维基百科更系统、更有深度
- 比nLab更易入门、更教育友好
- 比Mathlib4更注重教育解释
- 比传统教材更便捷、更新更快
