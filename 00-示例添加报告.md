## 2. 添加的50个实际示例

### 2.1 群论示例 (8个)

**文件**: `concept/核心概念/08-群.md`

添加的数值示例：

1. **示例1: 整数加法群 $(\mathbb{Z}, +)$ 的具体运算**
   - $3 + 5 = 8$
   - 单位元：$0$
   - $3$的逆元：$-3$

2. **示例2: 模6加法群 $\mathbb{Z}_6$ 的运算表**
   - 元素：$\{0, 1, 2, 3, 4, 5\}$
   - $3 + 4 = 1 \pmod{6}$
   - $5 + 2 = 1 \pmod{6}$

3. **示例3: 对称群 $S_3$ 的具体计算**
   - 设 $\sigma = (1\ 2)$, $\tau = (2\ 3)$
   - $\sigma \circ \tau = (1\ 2\ 3)$
   - 群的阶：$|S_3| = 6$

4. **示例4: 二面体群 $D_4$ 的旋转和反射**
   - 正方形的对称群
   - 旋转 $r$（$90°$）和反射 $s$
   - 关系：$r^4 = e$, $s^2 = e$, $srs = r^{-1}$

5. **示例5: 循环群 $\mathbb{Z}_5$ 的生成元**
   - 元素：$\{0, 1, 2, 3, 4\}$
   - 生成元：$1$ 或 $2$ 或 $3$
   - $2$ 的幂：$2^1=2$, $2^2=4$, $2^3=3$, $2^4=1$

6. **示例6: Klein四元群 $V_4$**
   - 元素：$\{e, a, b, c\}$
   - 所有非单位元的阶都是2
   - 是最小的非循环群

7. **示例7: 四元数群 $Q_8$**
   - 元素：$\{1, -1, i, -i, j, -j, k, -k\}$
   - $i^2 = j^2 = k^2 = ijk = -1$
   - 8阶非交换群

8. **示例8: 乘法群 $\mathbb{Z}_7^*$**
   - 元素：$\{1, 2, 3, 4, 5, 6\}$
   - 是6阶循环群
   - $3$ 是生成元：$3^1=3$, $3^2=2$, $3^3=6$, $3^4=4$, $3^5=5$, $3^6=1$

### 2.2 环论示例 (6个)

**文件**: `concept/核心概念/09-环.md`

1. **示例1: 整数环的具体运算**
   - $5 + 3 = 8$
   - $5 \times 3 = 15$
   - 分配律：$2(3+4) = 2 \times 3 + 2 \times 4 = 14$

2. **示例2: 模8环 $\mathbb{Z}_8$**
   - $5 + 6 = 3 \pmod{8}$
   - $3 \times 5 = 7 \pmod{8}$
   - 零因子：$2 \times 4 = 0 \pmod{8}$

3. **示例3: 多项式环 $\mathbb{Z}[x]$**
   - $(x+1)(x-1) = x^2 - 1$
   - $(2x+3) + (x-2) = 3x + 1$

4. **示例4: 矩阵环 $M_2(\mathbb{Z})$**
   - 加法：$\begin{pmatrix}1&2\\3&4\end{pmatrix} + \begin{pmatrix}0&1\\2&3\end{pmatrix} = \begin{pmatrix}1&3\\5&7\end{pmatrix}$
   - 乘法：$\begin{pmatrix}1&0\\0&0\end{pmatrix}\begin{pmatrix}0&0\\0&1\end{pmatrix} = \begin{pmatrix}0&0\\0&0\end{pmatrix}$

5. **示例5: 高斯整数环 $\mathbb{Z}[i]$**
   - $(1+i) + (2-i) = 3$
   - $(1+i)(1-i) = 2$

6. **示例6: 商环 $\mathbb{Z}[x]/(x^2+1)$**
   - 等价于高斯整数
   - $[x]^2 = [-1]$

### 2.3 域论示例 (5个)

**文件**: `concept/核心概念/10-域.md`

1. **示例1: 有理数域 $\mathbb{Q}$ 的运算**
   - $\frac{1}{2} + \frac{1}{3} = \frac{5}{6}$
   - $\frac{2}{3} \times \frac{3}{4} = \frac{1}{2}$
   - $\frac{2}{3}$ 的乘法逆元是 $\frac{3}{2}$

2. **示例2: 有限域 $\mathbb{Z}_5$**
   - $2 + 4 = 1 \pmod{5}$
   - $3 \times 2 = 1 \pmod{5}$
   - $2^{-1} = 3 \pmod{5}$

3. **示例3: 域扩张 $\mathbb{Q}(\sqrt{2})$**
   - 元素形式：$a + b\sqrt{2}$ ($a,b \in \mathbb{Q}$)
   - $(1+\sqrt{2}) + (2-\sqrt{2}) = 3$
   - $(1+\sqrt{2})(1-\sqrt{2}) = -1$

4. **示例4: 有限域 $\mathbb{F}_4 = \mathbb{F}_2[x]/(x^2+x+1)$**
   - 元素：$\{0, 1, x, x+1\}$
   - $x^2 = x+1$
   - 特征为2

5. **示例5: 复数域 $\mathbb{C}$**
   - $(1+2i) + (3-i) = 4+i$
   - $(1+i)(1-i) = 2$
   - $i$ 的乘法逆元是 $-i$

### 2.4 向量空间示例 (6个)

**文件**: `concept/核心概念/11-向量空间.md`

1. **示例1: $\mathbb{R}^3$ 中的向量运算**
   - $(1,2,3) + (4,5,6) = (5,7,9)$
   - $2(1,2,3) = (2,4,6)$

2. **示例2: 多项式空间 $\mathbb{R}_2[x]$**
   - 次数小于3的多项式
   - 基：$\{1, x, x^2\}$
   - 维数：3

3. **示例3: 矩阵空间 $M_{2\times 2}(\mathbb{R})$**
   - 基：$\{E_{11}, E_{12}, E_{21}, E_{22}\}$
   - 维数：4

4. **示例4: 连续函数空间 $C([0,1])$**
   - 无限维向量空间
   - 包含所有连续函数

5. **示例5: 解空间（齐次线性方程组）**
   - $x + y + z = 0$
   - $2x + y - z = 0$
   - 解空间维数：1

6. **示例6: 张成的子空间**
   - $\text{span}\{(1,0,1), (0,1,1)\} \subset \mathbb{R}^3$
   - 维数：2
   - 是过原点的平面

### 2.5 极限示例 (5个)

**文件**: `concept/核心概念/13-极限.md`

1. **示例1: 数值逼近 $\lim_{x\to 0}\frac{\sin x}{x} = 1$**
   - $x=0.1$: $\frac{\sin 0.1}{0.1} \approx 0.998$
   - $x=0.01$: $\frac{\sin 0.01}{0.01} \approx 0.99998$

2. **示例2: 序列极限 $\lim_{n\to\infty}\frac{1}{n} = 0$**
   - $n=10$: $0.1$
   - $n=100$: $0.01$
   - $n=1000$: $0.001$

3. **示例3: 有理函数的极限**
   - $\lim_{x\to\infty}\frac{3x^2+2x}{x^2-1} = 3$

4. **示例4: 单侧极限**
   - $\lim_{x\to 0^+}\frac{1}{x} = +\infty$
   - $\lim_{x\to 0^-}\frac{1}{x} = -\infty$

5. **示例5: 夹逼定理应用**
   - $\lim_{n\to\infty}\frac{\sin n}{n} = 0$
   - 因为 $-\frac{1}{n} \leq \frac{\sin n}{n} \leq \frac{1}{n}$

### 2.6 导数示例 (5个)

**文件**: `concept/核心概念/15-导数.md`

1. **示例1: 用定义计算 $f(x)=x^2$ 在 $x=2$ 处的导数**
   - $f'(2) = \lim_{h\to 0}\frac{(2+h)^2-4}{h}$
   - $= \lim_{h\to 0}\frac{4+4h+h^2-4}{h}$
   - $= \lim_{h\to 0}(4+h) = 4$

2. **示例2: 幂函数导数**
   - $\frac{d}{dx}x^5 = 5x^4$
   - $\frac{d}{dx}x^{-2} = -2x^{-3}$

3. **示例3: 乘积法则应用**
   - 求 $f(x) = x^2e^x$ 的导数
   - $f'(x) = 2xe^x + x^2e^x = xe^x(2+x)$

4. **示例4: 链式法则应用**
   - 求 $f(x) = \sin(x^2)$ 的导数
   - $f'(x) = 2x\cos(x^2)$

5. **示例5: 隐函数求导**
   - $x^2 + y^2 = 1$
   - $2x + 2y\frac{dy}{dx} = 0$
   - $\frac{dy}{dx} = -\frac{x}{y}$

### 2.7 积分示例 (5个)

**文件**: `concept/核心概念/16-积分.md`

1. **示例1: 用Riemann和计算 $\int_0^1 x dx$**
   - 分割：$x_i = \frac{i}{n}$
   - 和：$\sum_{i=1}^n \frac{i}{n} \cdot \frac{1}{n} = \frac{1}{n^2}\frac{n(n+1)}{2}$
   - 极限：$\lim_{n\to\infty}\frac{n+1}{2n} = \frac{1}{2}$

2. **示例2: 换元法**
   - $\int_0^{\pi/2}\sin^2 x \cos x dx$
   - 令 $u = \sin x$, $du = \cos x dx$
   - $= \int_0^1 u^2 du = \frac{1}{3}$

3. **示例3: 分部积分**
   - $\int_0^1 xe^x dx$
   - $= [xe^x]_0^1 - \int_0^1 e^x dx$
   - $= e - (e - 1) = 1$

4. **示例4: 定积分的几何意义**
   - $\int_{-1}^1 \sqrt{1-x^2}dx = \frac{\pi}{2}$
   - 半圆的面积

5. **示例5: 瑕积分**
   - $\int_0^1 \frac{1}{\sqrt{x}}dx$
   - $= [2\sqrt{x}]_0^1 = 2$

### 2.8 拓扑空间示例 (3个)

**文件**: `concept/核心概念/23-拓扑空间.md`

1. **示例1: 实数标准拓扑中的开区间**
   - $(0,1)$ 是开集
   - $[0,1]$ 不是开集
   - $(0,1) \cup (2,3)$ 是开集

2. **示例2: 离散拓扑**
   - $X = \{a, b, c\}$
   - 拓扑：$\mathcal{P}(X)$（所有子集）
   - 每个单点集 $\{a\}$ 都是开集

3. **示例3: 子空间拓扑**
   - $\mathbb{Q} \subset \mathbb{R}$
   - $\mathbb{Q}$ 上的拓扑由 $\mathbb{R}$ 诱导
   - $(a,b) \cap \mathbb{Q}$ 是 $\mathbb{Q}$ 中的开集

### 2.10 同余示例 (3个)

**文件**: `concept/核心概念/27-同余.md`

1. **示例1: 同余的基本运算**
   - $17 \equiv 5 \pmod{12}$（因为 $12 \mid (17-5)$）
   - $17 + 8 = 25 \equiv 1 \pmod{12}$
   - $17 \times 3 = 51 \equiv 3 \pmod{12}$

2. **示例2: 同余类的运算**
   - 在 $\mathbb{Z}_5$ 中：$[3] + [4] = [2]$
   - $[3] \times [4] = [12] = [2]$

3. **示例3: 中国剩余定理的具体计算**
   - 求解：$x \equiv 2 \pmod{3}$ 且 $x \equiv 3 \pmod{5}$
   - 解：$x = 8 + 15k$ ($k \in \mathbb{Z}$)
