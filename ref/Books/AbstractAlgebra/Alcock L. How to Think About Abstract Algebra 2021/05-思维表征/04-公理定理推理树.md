## 一、群公理 → 基本定理链

```text
【群公理】(G, *) 为群
├── 封闭性：∀a,b∈G, ab∈G
├── 结合律：∀a,b,c∈G, (ab)c = a(bc)
├── 单位元：∃e∈G, ∀a∈G, ea = ae = a
└── 逆元：∀a∈G, ∃a⁻¹∈G, aa⁻¹ = a⁻¹a = e

    ↓ 由公理直接推出

【基本定理】
├── 单位元唯一性：若 e,e' 均为单位元，则 e = e'
├── 逆元唯一性：每个元的逆元唯一
├── 消去律：若 ab = ac 则 b = c；若 ba = ca 则 b = c
└── (a⁻¹)⁻¹ = a，(ab)⁻¹ = b⁻¹a⁻¹
```

**对应文档**: [01-概念关联网络](../04-知识关联/01-概念关联网络.md) 群论部分；与 MIT 18.701 / AMS 群定义一致。

## 二、子群与正规子群推理链

```text
【子群定义】H ⊆ G，且 (H, *) 为群

    ↓ 等价刻画

【子群判定】
├── 定理：H ≤ G ⇔ H≠∅ 且 ∀a,b∈H 有 ab⁻¹∈H
└── 推论：⟨S⟩ = ⋂{H ≤ G : S ⊆ H} 为子群

【正规子群】N ◁ G：∀g∈G, gN = Ng

    ↓ 构造商结构

【商群】
├── G/N = {gN : g∈G}，运算 (aN)(bN) = (ab)N
├── |G/N| = [G:N] = |G|/|N|（有限时）
└── 自然同态 π: G → G/N, π(g) = gN 为满同态且 ker π = N
```

**对应文档**: [02-理论关联图谱](../04-知识关联/02-理论关联图谱.md)。

## 三、环公理 → 基本定理链

```text
【环公理】(R, +, ·) 为环
├── (R, +) 为阿贝尔群（加法单位元 0，逆元 -a）
├── 乘法结合律：(ab)c = a(bc)
├── 分配律：a(b+c) = ab+ac，(a+b)c = ac+bc
└── 可选：乘法单位元 1（有单位元环）

    ↓ 由公理推出

【基本定理】
├── a·0 = 0·a = 0
├── (-a)b = a(-b) = -(ab)，(-a)(-b) = ab
├── 子环判定：S ⊆ R 为子环 ⇔ S 对加减法封闭、含 0、对乘法封闭
└── 理想 I：I 为加法子群且 ∀r∈R, rI ⊆ I, Ir ⊆ I ⇒ 商环 R/I 良定义
```

**对应文档**: [01-概念关联网络](../04-知识关联/01-概念关联网络.md) 环论部分；与 MIT 18.702 / Cambridge 环论一致。

## 四、同态与同构定理链

```text
【群同态】φ: G → H 满足 φ(ab) = φ(a)φ(b)

    ↓ 基本性质

├── φ(e_G) = e_H，φ(a⁻¹) = φ(a)⁻¹
├── ker φ = φ⁻¹(e_H) 为 G 的正规子群
├── im φ = φ(G) 为 H 的子群
└── 第一同构定理：G/ker φ ≅ im φ（自然同构 φ̃: g·ker φ ↦ φ(g)）

【环同态】φ: R → S 满足 φ(a+b)=φ(a)+φ(b)，φ(ab)=φ(a)φ(b)

    ↓ 平行结论

├── ker φ 为 R 的理想
├── im φ 为 S 的子环
└── 第一同构定理：R/ker φ ≅ im φ
```

**对应文档**: [03-跨主题关联](../04-知识关联/03-跨主题关联.md)；与 Harvard Math 122 / Cambridge 同态定理对应。
