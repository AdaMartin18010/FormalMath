## 一、群公理 → 基本定理链

```text
【群公理】(G, *) 为群
├── 封闭性：∀a,b∈G, ab∈G
├── 结合律：∀a,b,c∈G, (ab)c = a(bc)
├── 单位元：∃e∈G, ∀a∈G, ea = ae = a
└── 逆元：∀a∈G, ∃a⁻¹∈G, aa⁻¹ = a⁻¹a = e

    ↓ 由公理直接推出

【基本定理】
├── 单位元唯一性；逆元唯一性
├── 消去律：ab = ac ⇒ b = c（左消去）；同理右消去
├── (ab)⁻¹ = b⁻¹a⁻¹；|a| 有限时 |a⁻¹| = |a|
└── 方程 ax = b 与 ya = b 在 G 中唯一可解
```

**对应文档**: [01-概念关联网络](../04-知识关联/01-概念关联网络.md)；与 MIT 18.701 / AMS 群定义一致。

## 二、子群与 Lagrange 定理链

```text
【子群】H ≤ G：H ⊆ G 且 (H, *) 为群

    ↓ 等价判定与指数

【子群判定】H ≤ G ⇔ H≠∅ 且 ∀a,b∈H 有 ab⁻¹∈H
【指数】[G:H] = 左陪集个数 = |G|/|H|（当 G 有限）

    ↓ Lagrange 定理

【Lagrange】若 G 有限、H ≤ G，则 |G| = [G:H]·|H|
├── 推论：|a| 整除 |G|（⟨a⟩ ≤ G）
├── 推论：|G| 为素数 ⇒ G 为循环群
└── 正规子群 N ◁ G ⇒ 商群 G/N，|G/N| = [G:N]

【商群与同态】
├── 自然同态 π: G → G/N 满足 ker π = N
└── 第一同构定理：G/ker φ ≅ im φ（φ 为群同态）
```

**对应文档**: [02-理论关联图谱](../04-知识关联/02-理论关联图谱.md)；与 MIT 18.701 / Cambridge Part IA 对应。

## 三、环公理 → 理想与商环链

```text
【环公理】(R, +, ·)
├── (R, +) 阿贝尔群；乘法结合；分配律
└── 可选：乘法单位元 1

    ↓ 理想与商环

【理想】I ⊆ R：I 为加法子群且 rI ⊆ I, Ir ⊆ I（∀r∈R）
├── 主理想：(a) = Ra = {ra : r∈R}（有单位元时）
├── 商环 R/I：陪集 a+I，运算 (a+I)(b+I) = ab+I
└── 自然同态 π: R → R/I 满足 ker π = I

【特殊环】
├── 整环：无零因子的交换环（含 1）
├── 域：非零元均可逆的整环
└── 定理：R 为域 ⇔ R 仅有平凡理想 {0} 与 R
```

**对应文档**: [01-概念关联网络](../04-知识关联/01-概念关联网络.md) 环论部分；与 MIT 18.702 / Cambridge 环论一致。

## 四、同态与同构定理链

```text
【群同态】φ: G → H，φ(ab) = φ(a)φ(b)

├── ker φ ◁ G；im φ ≤ H
├── 第一同构定理：G/ker φ ≅ im φ
├── 第二同构定理（H≤G, N◁G）：HN/N ≅ H/(H∩N)
└── 第三同构定理（N⊆K◁G）：(G/N)/(K/N) ≅ G/K

【环同态】φ: R → S，保加法和乘法

├── ker φ 为理想；im φ 为子环
└── 第一同构定理：R/ker φ ≅ im φ
```

**对应文档**: [03-跨主题关联](../04-知识关联/03-跨主题关联.md)；与 Harvard Math 122 / Cambridge 同态定理对应。
