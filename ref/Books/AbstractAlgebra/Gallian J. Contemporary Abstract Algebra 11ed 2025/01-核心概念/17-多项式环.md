## 一、定义与记号

### 1.1 多项式环的定义

**定义**: 设 $R$ 是交换环。形式符号集合
$$
R[x] = \{a_n x^n + a_{n-1} x^{n-1} + \dots + a_1 x + a_0 \mid a_i \in R, n \text{ 是非负整数}\}
$$
称为 $R$ 上在未定元 $x$ 中的**多项式环 (Ring of Polynomials)**。

- *Gallian原文*: "Definition Ring of Polynomials over $R$" (Chapter 16)

两个元素
$$
a_n x^n + a_{n-1} x^{n-1} + \dots + a_1 x + a_0
$$
和
$$
b_m x^m + b_{m-1} x^{m-1} + \dots + b_1 x + b_0
$$
被认为是相等的，当且仅当 $a_i = b_i$ 对于所有非负整数 $i$。（定义 $a_i = 0$ 当 $i > n$ 且 $b_i = 0$ 当 $i > m$。）

### 1.2 加法和乘法的定义

**定义**: 设 $R$ 是交换环，且设
$$
f(x) = a_n x^n + a_{n-1} x^{n-1} + \dots + a_1 x + a_0
$$
和
$$
g(x) = b_m x^m + b_{m-1} x^{m-1} + \dots + b_1 x + b_0
$$
属于 $R[x]$。则
$$
f(x) + g(x) = (a_s + b_s)x^s + (a_{s-1} + b_{s-1})x^{s-1} + \dots + (a_1 + b_1)x + a_0 + b_0,
$$
其中 $s$ 是 $m$ 和 $n$ 的最大值，$a_i = 0$ 对于 $i > n$，且 $b_i = 0$ 对于 $i > m$。此外，
$$
f(x)g(x) = c_{m+n}x^{m+n} + c_{m+n-1}x^{m+n-1} + \dots + c_1 x + c_0,
$$
其中
$$
c_k = a_k b_0 + a_{k-1} b_1 + \dots + a_1 b_{k-1} + a_0 b_k
$$
对于 $k = 0, \ldots, m+n$。

- *Gallian原文*: "Definition Addition and Multiplication in $R[x]$" (Chapter 16)

### 1.3 记号与术语

在这个定义中，符号 $x, x^2, \ldots, x^n$ 不代表来自环 $R$ 的"未知"元素或变量。相反，它们的目的是作为方便的占位符，将环元素 $a_n, a_{n-1}, \ldots, a_0$ 分开。我们可以通过将多项式定义为无限序列 $a_0, a_1, a_2, \ldots, a_n, 0, 0, 0, \ldots$ 来避免 $x$，但我们的方法利用了学生在操作多项式方面的经验，其中 $x$ 确实代表变量。我们方法的缺点是必须注意不要将多项式与由多项式确定的函数混淆。

- *Gallian原文*: "In this definition, the symbols $x, x^2, \ldots, x^n$ do not represent 'unknown' elements or variables from the ring $R$." (Chapter 16)

**例子**: 在 $Z_3[x]$ 中，多项式 $f(x) = x$ 和 $g(x) = x^3$ 确定从 $Z_3$ 到 $Z_3$ 的相同函数，因为 $f(a) = g(a)$ 对于 $Z_3$ 中的所有 $a$。但 $f(x)$ 和 $g(x)$ 是 $Z_3[x]$ 的不同元素。

**术语**:

- 如果 $f(x) = a_n x^n + a_{n-1} x^{n-1} + \dots + a_1 x + a_0$，其中 $a_n \neq 0$，我们说 $f(x)$ 的**次数 (Degree)** 为 $n$。
- 项 $a_n$ 称为 $f(x)$ 的**首项系数 (Leading Coefficient)**。
- 如果首项系数是 $R$ 的乘法单位元，我们说 $f(x)$ 是**首一多项式 (Monic Polynomial)**。
- 多项式 $f(x) = 0$ 没有次数。
- 形式为 $f(x) = a_0$ 的多项式称为**常数 (Constant)**。
- 我们经常写 $\deg f(x) = n$ 来表示 $f(x)$ 的次数为 $n$。

## 二、多项式环的性质

### 2.1 整环性质 (定理 16.1)

**定理 16.1**: 如果 $D$ 是整环，则 $D[x]$ 是整环。

- *Gallian原文*: "Theorem 16.1 $D$ an Integral Domain Implies $D[x]$ an Integral Domain" (Chapter 16)

**证明**: 由于我们已经知道 $D[x]$ 是环，我们只需要证明 $D[x]$ 是有单位元的交换环且没有零因子。显然，$D[x]$ 是交换的，只要 $D$ 是。如果 1 是 $D$ 的单位元，显然 $f(x) = 1$ 是 $D[x]$ 的单位元。最后，假设
$$
f(x) = a_n x^n + a_{n-1} x^{n-1} + \dots + a_0
$$
和
$$
g(x) = b_m x^m + b_{m-1} x^{m-1} + \dots + b_0,
$$
其中 $a_n \neq 0$ 且 $b_m \neq 0$。则，根据定义，$f(x)g(x)$ 的首项系数为 $a_n b_m$，且由于 $D$ 是整环，$a_n b_m \neq 0$。

### 2.2 次数规则

**次数规则**: 设 $D$ 是整环，$f(x), g(x) \in D[x]$。则 $\deg(f(x) \cdot g(x)) = \deg f(x) + \deg g(x)$。

- *Gallian原文*: "(Degree Rule) Let $D$ be an integral domain and $f(x), g(x) \in D[x]$." (Chapter 16, Exercise 19)

对于交换环 $R$，可能 $\deg f(x)g(x) < \deg f(x) + \deg g(x)$，其中 $f(x)$ 和 $g(x)$ 是 $R[x]$ 中的非零元素。

## 三、除法算法 (定理 16.2)

### 3.1 定理内容

**定理 16.2**: 设 $F$ 是域，$f(x), g(x) \in F[x]$ 且 $g(x) \neq 0$。则存在唯一的多项式 $q(x)$ 和 $r(x)$ 在 $F[x]$ 中，使得 $f(x) = g(x)q(x) + r(x)$ 且 $r(x) = 0$ 或 $\deg r(x) < \deg g(x)$。

- *Gallian原文*: "Theorem 16.2 Division Algorithm for $F[x]$" (Chapter 16)

### 3.2 证明思路

我们首先证明 $q(x)$ 和 $r(x)$ 的存在性。如果 $f(x) = 0$ 或 $\deg f(x) < \deg g(x)$，我们简单地设 $q(x) = 0$ 且 $r(x) = f(x)$。所以，我们可以假设 $n = \deg f(x) \geq \deg g(x) = m$ 且设 $f(x) = a_n x^n + \dots + a_0$ 和 $g(x) = b_m x^m + \dots + b_0$。这个证明背后的想法是开始就像你要"长除" $g(x)$ 到 $f(x)$ 中，然后使用关于 $\deg f(x)$ 的第二数学归纳原理来完成。因此，求助于长除法，我们设 $f_1(x) = f(x) - a_n b_m^{-1}x^{n-m}g(x)$。则，$f_1(x) = 0$ 或 $\deg f_1(x) < \deg f(x)$；所以，由我们的归纳假设，存在 $q_1(x)$ 和 $r_1(x)$ 在 $F[x]$ 中，使得 $f_1(x) = g(x)q_1(x) + r_1(x)$，其中 $r_1(x) = 0$ 或 $\deg r_1(x) < \deg g(x)$。

因此，
$$
f(x) = a_n b_m^{-1} x^{n-m} g(x) + f_1(x) = a_n b_m^{-1} x^{n-m} g(x) + q_1(x) g(x) + r_1(x) = [a_n b_m^{-1} x^{n-m} + q_1(x)] g(x) + r_1(x).
$$
所以，多项式 $q(x) = a_n b_m^{-1} x^{n-m} + q_1(x)$ 和 $r(x) = r_1(x)$ 具有所需性质。

要证明唯一性，假设 $f(x) = g(x)q(x) + r(x)$ 且 $f(x) = g(x)\bar{q}(x) + \bar{r}(x)$，其中 $r(x) = 0$ 或 $\deg r(x) < \deg g(x)$ 且 $\bar{r}(x) = 0$ 或 $\deg \bar{r}(x) < \deg g(x)$。则，减去这两个方程，我们得到
$$
0 = g(x)[q(x) - \bar{q}(x)] + [r(x) - \bar{r}(x)]
$$
或
$$
\bar{r}(x) - r(x) = g(x)(q(x) - \bar{q}(x)).
$$
因此，$\bar{r}(x) - r(x)$ 是 0，或 $\bar{r}(x) - r(x)$ 的次数至少是 $g(x)$ 的次数。由于后者显然不可能，我们有 $\bar{r}(x) = r(x)$ 且 $q(x) = \bar{q}(x)$。

### 3.3 应用：余数定理和因子定理

**推论 1 余数定理**: 设 $F$ 是域，$a \in F$，且 $f(x) \in F[x]$。则 $f(a)$ 是 $f(x)$ 除以 $x - a$ 的余数。

- *Gallian原文*: "Corollary 1 Remainder Theorem" (Chapter 16)

**推论 2 因子定理**: 设 $F$ 是域，$a \in F$，且 $f(x) \in F[x]$。则 $a$ 是 $f(x)$ 的零点当且仅当 $x - a$ 是 $f(x)$ 的因子。

- *Gallian原文*: "Corollary 2 Factor Theorem" (Chapter 16)

## 四、多项式的零点

### 4.1 零点的定义

**定义**: 元素 $a$ 是多项式 $f(x)$ 的**零点 (Zero)**（或**根 (Root)**），如果 $f(a) = 0$。[回忆 $f(a)$ 意味着在 $f(x)$ 的表达式中用 $a$ 替换 $x$。]

当 $F$ 是域，$a \in F$，且 $f(x) \in F[x]$ 时，我们说 $a$ 是**重数 (Multiplicity)** $k$（$k \geq 1$）的零点，如果 $(x - a)^k$ 是 $f(x)$ 的因子但 $(x - a)^{k+1}$ 不是 $f(x)$ 的因子。

### 4.2 零点定理 (定理 16.3)

**定理 16.3**: 域上的 $n$ 次多项式最多有 $n$ 个零点，计算重数。

- *Gallian原文*: "Theorem 16.3 Polynomials of Degree $n$ Have at Most $n$ Zeros" (Chapter 16)

**证明**: 我们对 $n$ 进行归纳。显然，域上的 0 次多项式没有零点。现在假设 $f(x)$ 是域上的 $n$ 次多项式，且 $a$ 是 $f(x)$ 的重数为 $k$ 的零点。则，$f(x) = (x - a)^k q(x)$ 且 $q(a) \neq 0$；且，由于 $n = \deg f(x) = \deg (x - a)^k q(x) = k + \deg q(x)$，我们有 $k \leq n$（见练习 19）。如果 $f(x)$ 没有除 $a$ 之外的零点，我们完成。另一方面，如果 $b \neq a$ 且 $b$ 是 $f(x)$ 的零点，则 $0 = f(b) = (b - a)^k q(b)$，所以 $b$ 也是 $q(x)$ 的零点，具有与 $f(x)$ 相同的重数（见练习 21）。由第二数学归纳原理，我们知道 $q(x)$ 最多有 $\deg q(x) = n - k$ 个零点，计算重数。因此，$f(x)$ 最多有 $k + n - k = n$ 个零点，计算重数。

### 4.3 重数

**例子 2**: 我们找到 $x^n - 1$ 的所有复零点。设 $\omega = \cos(360^\circ / n) + i\sin(360^\circ / n)$。由 DeMoivre 定理（见第 0 章的例子 11），$\omega^n = 1$ 且 $\omega^k \neq 1$ 对于 $1 \leq k < n$。因此，$1, \omega, \omega^2, \ldots, \omega^{n-1}$ 中的每一个都是 $x^n - 1$ 的零点，且由定理 16.4，没有其他的。

- *Gallian原文*: "EXAMPLE 2 The Complex Zeros of $x^n - 1$" (Chapter 16)

例子 2 中的复数 $\omega$ 称为**本原 $n$ 次单位根 (Primitive $n$th Root of Unity)**。

## 五、主理想整环 (PID)

### 5.1 PID 的定义

**定义**: **主理想整环 (Principal Ideal Domain, PID)** 是整环 $R$，其中每个理想都有形式 $\langle a \rangle = \{ra \mid r \in R\}$ 对于 $R$ 中的某个 $a$。

- *Gallian原文*: "Definition Principal Ideal Domain (PID)" (Chapter 16)

### 5.2 $F[x]$ 是 PID (定理 16.4)

**定理 16.4**: 设 $F$ 是域。则 $F[x]$ 是主理想整环。

- *Gallian原文*: "Theorem 16.4 $F[x]$ Is a PID" (Chapter 16)

**证明**: 由定理 16.1，我们知道 $F[x]$ 是整环。现在，设 $I$ 是 $F[x]$ 中的理想。如果 $I = \{0\}$，则 $I = \langle 0 \rangle$。如果 $I \neq \{0\}$，则在 $I$ 的所有元素中，设 $g(x)$ 是次数最小的一个。我们将证明 $I = \langle g(x) \rangle$。由于 $g(x) \in I$，我们有 $\langle g(x) \rangle \subseteq I$。现在设 $f(x) \in I$。则，由除法算法，我们可以写 $f(x) = g(x)q(x) + r(x)$，其中 $r(x) = 0$ 或 $\deg r(x) < \deg g(x)$。由于 $r(x) = f(x) - g(x)q(x) \in I$，$\deg g(x)$ 的最小性意味着后一个条件不能成立。所以，$r(x) = 0$，因此，$f(x) \in \langle g(x) \rangle$。这证明 $I \subseteq \langle g(x) \rangle$。

**定理 16.5**: 设 $F$ 是域，$I$ 是 $F[x]$ 中的非零理想，且 $g(x)$ 是 $F[x]$ 的元素。则，$I = \langle g(x) \rangle$ 当且仅当 $g(x)$ 是 $I$ 中次数最小的多项式。

- *Gallian原文*: "Theorem 16.5 Criterion for $I = \langle g(x)\rangle$" (Chapter 16)

## 六、数学意义与应用

### 6.1 代数的基础

多项式环是代数的基础。许多代数问题可以表述为多项式环中的问题。多项式环的性质（特别是除法算法和 PID 性质）使得我们可以研究这些问题的结构。

### 6.2 数论中的应用

多项式环在数论中有许多应用。例如，我们可以使用多项式来研究整数的性质，或者研究模 $p$ 的整数环。

### 6.3 代数几何

多项式环是代数几何的基础。代数几何研究由多项式方程定义的几何对象。多项式环的理想对应于这些几何对象。

## 七、学习要点与难点

### 7.1 理解形式符号

在多项式环的定义中，$x, x^2, \ldots, x^n$ 不代表来自环 $R$ 的"未知"元素或变量。相反，它们的目的是作为方便的占位符。我们必须注意不要将多项式与由多项式确定的函数混淆。

### 7.2 区分多项式与函数

在 $Z_3[x]$ 中，多项式 $f(x) = x$ 和 $g(x) = x^3$ 确定从 $Z_3$ 到 $Z_3$ 的相同函数，因为 $f(a) = g(a)$ 对于 $Z_3$ 中的所有 $a$。但 $f(x)$ 和 $g(x)$ 是 $Z_3[x]$ 的不同元素。理解这个区别很重要。

此外，在环 $Z_n[x]$ 中，注意只约化系数而不约化指数模 $n$。例如，在 $Z_3[x]$ 中，$5x = 2x$，但 $x^5 \neq x^2$。
