## 一、群的验证

### 例题1：验证 $(\mathbb{Z}, +)$ 是群

**形式证明**（参考Wikipedia和MIT 18.701课程标准）：

**目标**：证明 $(\mathbb{Z}, +)$ 满足群的四个公理。

**证明**：

**步骤1：封闭性（Closure）**

**陈述**：对任意 $a, b \in \mathbb{Z}$，有 $a + b \in \mathbb{Z}$。

**证明**：

根据整数的定义，$\mathbb{Z} = \{\ldots, -2, -1, 0, 1, 2, \ldots\}$。

对任意 $a, b \in \mathbb{Z}$，整数加法 $a + b$ 的结果仍然是整数（这是整数加法的基本性质）。

因此封闭性成立。✓

**步骤2：结合律（Associativity）**

**陈述**：对任意 $a, b, c \in \mathbb{Z}$，有 $(a + b) + c = a + (b + c)$。

**证明**：

这是整数加法的基本性质，可以从整数的构造（Peano公理）或整数的定义直接得出。

因此结合律成立。✓

**步骤3：单位元（Identity）**

**陈述**：存在 $e \in \mathbb{Z}$，使得对任意 $a \in \mathbb{Z}$，有 $e + a = a + e = a$。

**证明**：

设 $e = 0$。

对任意 $a \in \mathbb{Z}$：

- $0 + a = a$（整数加法的性质）
- $a + 0 = a$（整数加法的性质）

因此 $0$ 是单位元。✓

**步骤4：逆元（Inverse）**

**陈述**：对任意 $a \in \mathbb{Z}$，存在 $a^{-1} \in \mathbb{Z}$，使得 $a + a^{-1} = a^{-1} + a = 0$。

**证明**：

对任意 $a \in \mathbb{Z}$，设 $a^{-1} = -a$（$a$ 的相反数）。

验证：

- $a + (-a) = 0$（整数加法的性质）
- $(-a) + a = 0$（整数加法的性质）

且 $-a \in \mathbb{Z}$（因为 $\mathbb{Z}$ 对取相反数封闭）。

因此 $a^{-1} = -a$ 是 $a$ 的逆元。✓

**结论**：

由于 $(\mathbb{Z}, +)$ 满足所有四个群公理，所以它是群。

**交换性**：

对任意 $a, b \in \mathbb{Z}$，有 $a + b = b + a$（整数加法交换律）。

因此 $(\mathbb{Z}, +)$ 是交换群（Abel群）。✓

**参考**：此证明遵循标准的形式证明结构，参考了Wikipedia的"群（数学）"条目和MIT 18.701课程的标准证明格式。

### 例题2：验证 $U(n)$ 在模 $n$ 乘法下是群

**解答**：

- $U(n) = \{k \in \{1, 2, \ldots, n-1\} \mid \gcd(k, n) = 1\}$
- **封闭性**：如果 $\gcd(a, n) = 1$ 且 $\gcd(b, n) = 1$，则 $\gcd(ab, n) = 1$ ✅
- **结合性**：继承自整数乘法 ✅
- **单位元**：$1$ 是单位元 ✅
- **逆元**：由欧拉定理，如果 $\gcd(a, n) = 1$，则存在 $b$ 使得 $ab \equiv 1 \pmod{n}$ ✅

因此 $U(n)$ 是群。

## 二、子群的识别

### 例题3：找出 $\mathbb{Z}_{12}$ 的所有子群

**解答**：
$\mathbb{Z}_{12}$ 是循环群，其子群都是循环群。

- $\langle 0 \rangle = \{0\}$（1阶）
- $\langle 1 \rangle = \mathbb{Z}_{12}$（12阶）
- $\langle 2 \rangle = \{0, 2, 4, 6, 8, 10\}$（6阶）
- $\langle 3 \rangle = \{0, 3, 6, 9\}$（4阶）
- $\langle 4 \rangle = \{0, 4, 8\}$（3阶）
- $\langle 6 \rangle = \{0, 6\}$（2阶）

### 例题4：证明 $Z(G)$ 是 $G$ 的子群

**解答**：

- $Z(G) = \{a \in G \mid ax = xa \text{ 对所有 } x \in G\}$
- $Z(G)$ 非空（包含 $e$）
- 对任意 $a, b \in Z(G)$ 和 $x \in G$：
  $$(ab)x = a(bx) = a(xb) = (ax)b = x(ab)$$
  所以 $ab \in Z(G)$
- 对任意 $a \in Z(G)$ 和 $x \in G$：
  $$a^{-1}x = a^{-1}(xa)a^{-1} = a^{-1}(ax)a^{-1} = xa^{-1}$$
  所以 $a^{-1} \in Z(G)$

因此 $Z(G) \leq G$。

## 三、循环群问题

### 例题5：找出 $\mathbb{Z}_{12}$ 的所有生成元

**解答**：
$\mathbb{Z}_{12} = \langle 1 \rangle$，生成元是那些与12互素的元素。

与12互素的元素：$1, 5, 7, 11$

因此生成元是：$1, 5, 7, 11$。

### 例题6：证明素数阶群是循环群

**形式证明**（参考Wikipedia和Harvard Math 122课程标准）：

**定理**：设 $G$ 是有限群，$|G| = p$，其中 $p$ 是素数。则 $G$ 是循环群。

**证明方法**：直接证明 + Lagrange定理

**证明**：

**步骤1：选择非单位元**

设 $a \in G$，$a \neq e$。

**存在性证明**：因为 $|G| = p \geq 2$（素数至少为2），所以 $G$ 中至少有两个元素：$e$ 和某个 $a \neq e$。

因此存在非单位元 $a$。

**步骤2：应用Lagrange定理**

**Lagrange定理**（参考Wikipedia）：

> 如果 $H$ 是有限群 $G$ 的子群，则 $|H|$ 整除 $|G|$。

考虑循环子群 $\langle a \rangle = \{a^n : n \in \mathbb{Z}\} \leq G$。

根据Lagrange定理：$|\langle a \rangle|$ 整除 $|G| = p$。

**步骤3：确定 $|\langle a \rangle|$**

因为 $p$ 是素数，$p$ 的正因子只有 $1$ 和 $p$。

因此 $|\langle a \rangle| \in \{1, p\}$。

**步骤4：排除 $|\langle a \rangle| = 1$**

如果 $|\langle a \rangle| = 1$，则 $\langle a \rangle = \{e\}$。

这意味着 $a = e$，与 $a \neq e$ 矛盾。

因此 $|\langle a \rangle| \neq 1$。

**步骤5：得出结论**

因此 $|\langle a \rangle| = p$。

因为 $|\langle a \rangle| = |G| = p$ 且 $\langle a \rangle \subseteq G$，所以 $\langle a \rangle = G$。

因此 $G = \langle a \rangle$ 是循环群，$a$ 是生成元。✓

**步骤6：唯一性（补充）**

**定理**：素数阶群的所有非单位元都是生成元。

**证明**：

设 $G$ 是 $p$ 阶群，$a, b \in G$，$a, b \neq e$。

根据上述证明，$|\langle a \rangle| = |\langle b \rangle| = p$。

因此 $\langle a \rangle = G = \langle b \rangle$。

所以 $a$ 和 $b$ 都是 $G$ 的生成元。✓

**参考**：此证明遵循标准的形式证明结构，参考了Wikipedia的"循环群"条目和Harvard Math 122课程的标准证明格式。

**应用**：

- 所有素数阶群都同构于 $\mathbb{Z}_p$
- 素数阶群都是Abel群（因为循环群是Abel群）

## 四、同构判断

### 例题7：证明 $S_2 \cong \mathbb{Z}_2$

**解答**：

- $S_2 = \{e, (12)\}$，有2个元素
- $\mathbb{Z}_2 = \{0, 1\}$，有2个元素
- 定义映射 $\phi: S_2 \to \mathbb{Z}_2$，$\phi(e) = 0$，$\phi((12)) = 1$
- 验证同构：
  - $\phi$ 是双射 ✅
  - $\phi((12) \circ (12)) = \phi(e) = 0 = 1 + 1 = \phi((12)) + \phi((12))$ ✅

因此 $S_2 \cong \mathbb{Z}_2$。

### 例题8：证明 $D_3 \cong S_3$

**解答**：

- $D_3$ 有6个元素（3个旋转 + 3个反射）
- $S_3$ 有 $3! = 6$ 个元素
- 可以建立同构映射：将 $D_3$ 的对称变换映射到 $S_3$ 的对应置换

因此 $D_3 \cong S_3$。

## 五、拉格朗日定理应用

### 例题9：证明6阶群必有3阶子群

**解答**：
设 $|G| = 6$，$a \in G$，$a \neq e$。

由拉格朗日定理，$|a|$ 整除6，所以 $|a| \in \{1, 2, 3, 6\}$。

- 如果 $|a| = 3$，则 $\langle a \rangle$ 是3阶子群 ✅
- 如果 $|a| = 6$，则 $\langle a^2 \rangle$ 是3阶子群 ✅
- 如果 $|a| = 2$，考虑另一个元素 $b \notin \langle a \rangle$，可以证明存在3阶元素

因此6阶群必有3阶子群。

### 例题10：证明群中元素的阶整除群的阶

**形式证明**（参考Wikipedia和MIT 18.701课程标准）：

**定理**：设 $G$ 是有限群，$|G| = n$，$a \in G$，$|a| = m$。则 $m$ 整除 $n$。

**证明方法**：直接应用Lagrange定理

**证明**：

**步骤1：元素的阶的定义**

**定义**：元素 $a \in G$ 的阶 $|a|$ 是使得 $a^m = e$ 的最小正整数 $m$。

如果不存在这样的 $m$，则 $|a| = \infty$。

**步骤2：循环子群**

考虑由 $a$ 生成的循环子群：

$$\langle a \rangle = \{e, a, a^2, \ldots, a^{m-1}\}$$

其中 $m = |a|$ 是 $a$ 的阶。

**步骤3：应用Lagrange定理**

**Lagrange定理**（参考Wikipedia）：

> 如果 $H$ 是有限群 $G$ 的子群，则 $|H|$ 整除 $|G|$。

因为 $\langle a \rangle \leq G$，根据Lagrange定理：

$$|\langle a \rangle| \mid |G|$$

**步骤4：确定 $|\langle a \rangle|$**

因为 $\langle a \rangle = \{e, a, a^2, \ldots, a^{m-1}\}$ 且 $a^m = e$，所以：

$$|\langle a \rangle| = m = |a|$$

**步骤5：得出结论**

因此：

$$m = |a| = |\langle a \rangle| \mid |G| = n$$

所以 $m$ 整除 $n$。✓

**步骤6：应用**

**推论**：在 $n$ 阶群中，每个元素的阶都是 $n$ 的因子。

**例子**：

- 在 $12$ 阶群中，元素的阶可能是：$1, 2, 3, 4, 6, 12$
- 在 $p$ 阶群中（$p$ 是素数），非单位元的阶都是 $p$

**参考**：此证明遵循标准的形式证明结构，参考了Wikipedia的"拉格朗日定理（群论）"条目和MIT 18.701课程的标准证明格式。

## 六、形式证明方法总结

### 6.1 证明方法分类

**直接证明**：从已知条件直接推导出结论

- 例子：例题1（验证群）、例题10（元素阶整除群阶）

**反证法**：假设结论不成立，推导出矛盾

- 例子：例题9的部分证明

**构造性证明**：通过构造具体例子来证明存在性

- 例子：例题3（找出所有子群）

**应用定理**：直接应用已知定理

- 例子：例题6（Lagrange定理）、例题10（Lagrange定理）

### 6.2 证明写作规范

**标准结构**：

1. 陈述定理
2. 选择证明方法
3. 逐步推导
4. 得出结论
5. 验证结果
