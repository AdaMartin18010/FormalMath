## 一、群论理论体系

### 1.1 群的定义与性质

**群的定义**：

群 $(G, *)$ 是集合 $G$ 配上一个二元运算 $*$，满足以下四个公理：

1. **封闭性**：$\forall a, b \in G: a * b \in G$
2. **结合律**：$\forall a, b, c \in G: (a * b) * c = a * (b * c)$
3. **单位元**：$\exists e \in G: \forall a \in G: e * a = a * e = a$
4. **逆元**：$\forall a \in G: \exists a^{-1} \in G: a * a^{-1} = a^{-1} * a = e$

**群的基本性质**：

1. **单位元唯一性**：
   - **定理**：群的单位元是唯一的
   - **证明**：设 $e$ 和 $e'$ 都是单位元，则 $e = e * e' = e'$

2. **逆元唯一性**：
   - **定理**：群中每个元素的逆元是唯一的
   - **证明**：设 $b$ 和 $c$ 都是 $a$ 的逆元，则 $b = b * e = b * (a * c) = (b * a) * c = e * c = c$

3. **消去律**：
   - **左消去律**：如果 $a * b = a * c$，则 $b = c$
   - **右消去律**：如果 $b * a = c * a$，则 $b = c$
   - **证明**：$b = e * b = (a^{-1} * a) * b = a^{-1} * (a * b) = a^{-1} * (a * c) = (a^{-1} * a) * c = e * c = c$

### 1.2 子群理论

**子群定义**：

子群 $H \subseteq G$ 是群 $G$ 的子集，且 $(H, *)$ 本身是群。

**子群判定定理**：

**定理**：$H \subseteq G$ 是子群当且仅当：

- $H \neq \emptyset$
- $\forall a, b \in H: ab^{-1} \in H$

**等价条件**：

- $H \neq \emptyset$
- $\forall a, b \in H: ab \in H$（封闭性）
- $\forall a \in H: a^{-1} \in H$（逆元存在）

**生成子群**：

设 $S \subseteq G$，由 $S$ 生成的子群：
$$\langle S \rangle = \bigcap \{H \leq G : S \subseteq H\}$$

如果 $S = \{a\}$，则 $\langle a \rangle = \{a^n : n \in \mathbb{Z}\}$ 是循环子群。

**例子**：

- $\mathbb{Z}_6$ 的子群：$\{0\}$，$\{0, 3\}$，$\{0, 2, 4\}$，$\mathbb{Z}_6$
- $\langle 2 \rangle = \{0, 2, 4\}$（在 $\mathbb{Z}_6$ 中）

### 1.3 正规子群与商群

**正规子群定义**：

子群 $N \leq G$ 是正规子群（记作 $N \triangleleft G$），如果：
$$\forall g \in G: gN = Ng$$

等价条件：

- $\forall g \in G: gNg^{-1} = N$
- $\forall g \in G, \forall n \in N: gng^{-1} \in N$

**商群构造**：

如果 $N \triangleleft G$，则商群：
$$G/N = \{gN : g \in G\}$$

运算：

- $(aN)(bN) = abN$
- 单位元：$N = eN$
- 逆元：$(aN)^{-1} = a^{-1}N$

**Lagrange定理**：

**定理**：如果 $H \leq G$（$G$ 有限），则：
$$|G| = |H| \cdot |G:H|$$

其中 $|G:H|$ 是 $H$ 在 $G$ 中的指数（陪集数量）。

**推论**：

- 有限群的子群的阶整除群的阶
- 如果 $N \triangleleft G$，则 $|G/N| = |G|/|N|$

**例子**：

- $G = \mathbb{Z}_6$，$H = \{0, 3\}$，则 $|G| = 6$，$|H| = 2$，$|G:H| = 3$，$6 = 2 \times 3$ ✓

### 1.4 同态与同构

**同态定义**：

映射 $\phi: G \to H$ 是群同态，如果：
$$\forall a, b \in G: \phi(ab) = \phi(a)\phi(b)$$

**核与像**：

- **核**：$\ker \phi = \{g \in G : \phi(g) = e_H\}$
- **像**：$\text{im } \phi = \{\phi(g) : g \in G\}$

**性质**：

- $\ker \phi \triangleleft G$
- $\text{im } \phi \leq H$

**同构定义**：

同态 $\phi: G \to H$ 是同构，如果它是双射。记作 $G \cong H$。

**第一同构定理**：

**定理**：如果 $\phi: G \to H$ 是群同态，则：
$$G/\ker \phi \cong \text{im } \phi$$

**证明思路**：

- 定义映射 $\bar{\phi}: G/\ker \phi \to \text{im } \phi$，$\bar{\phi}(g\ker \phi) = \phi(g)$
- 证明 $\bar{\phi}$ 是良定义的、单射、满射、同态

**第二同构定理**：

**定理**：如果 $H \leq G$ 且 $N \triangleleft G$，则：
$$H/(H \cap N) \cong HN/N$$

**第三同构定理**：

**定理**：如果 $N \triangleleft G$ 且 $K \triangleleft G$ 且 $N \subseteq K$，则：
$$(G/N)/(K/N) \cong G/K$$

## 二、环论理论体系

### 2.1 环的定义与性质

**环的定义**：

环 $(R, +, \cdot)$ 是集合 $R$ 配两个二元运算（加法和乘法），满足：

1. **$(R, +)$ 是Abel群**：
   - 封闭性、结合律、单位元 $0$、逆元 $-a$、交换律

2. **$(R, \cdot)$ 是半群**：
   - 封闭性、结合律

3. **分配律**：
   - 左分配律：$a(b + c) = ab + ac$
   - 右分配律：$(a + b)c = ac + bc$

**环的基本性质**：

1. **零元性质**：
   - $0 \cdot a = a \cdot 0 = 0$（对所有 $a \in R$）

2. **负元性质**：
   - $(-a)b = -(ab) = a(-b)$
   - $(-a)(-b) = ab$

3. **分配律的推广**：
   - $a(b_1 + \cdots + b_n) = ab_1 + \cdots + ab_n$
   - $(a_1 + \cdots + a_m)b = a_1b + \cdots + a_mb$

**特殊类型的环**：

- **交换环**：$\forall a, b \in R: ab = ba$
- **有单位元环**：$\exists 1 \in R: \forall a \in R: 1a = a1 = a$
- **整环**：交换环，有单位元，无零因子
- **域**：交换除环（每个非零元都有乘法逆元）

### 2.2 子环与理想

**子环定义**：

子环 $S \subseteq R$ 是 $R$ 的子集，且 $(S, +, \cdot)$ 本身是环。

**子环判定**：

$S \subseteq R$ 是子环当且仅当：

- $S \neq \emptyset$
- $\forall a, b \in S: a - b \in S$ 且 $ab \in S$

**理想定义**：

理想 $I \subseteq R$ 是子环，且满足吸收性质：

- $\forall r \in R, \forall i \in I: ri \in I$ 且 $ir \in I$

**主理想**：

如果 $R$ 是交换环，由 $a \in R$ 生成的主理想：
$$\langle a \rangle = \{ra : r \in R\}$$

**商环构造**：

如果 $I \triangleleft R$，则商环：
$$R/I = \{r + I : r \in R\}$$

运算：

- $(a + I) + (b + I) = (a + b) + I$
- $(a + I)(b + I) = ab + I$
- 单位元：$1 + I$（如果 $R$ 有单位元）

### 2.3 环同态

**环同态定义**：

映射 $\phi: R \to S$ 是环同态，如果：

- $\phi(a + b) = \phi(a) + \phi(b)$
- $\phi(ab) = \phi(a)\phi(b)$
- $\phi(1_R) = 1_S$（如果 $R$ 和 $S$ 有单位元）

**核与像**：

- **核**：$\ker \phi = \{r \in R : \phi(r) = 0_S\}$
- **像**：$\text{im } \phi = \{\phi(r) : r \in R\}$

**性质**：

- $\ker \phi \triangleleft R$
- $\text{im } \phi \leq S$（作为子环）

**第一同构定理（环）**：

**定理**：如果 $\phi: R \to S$ 是环同态，则：
$$R/\ker \phi \cong \text{im } \phi$$

**证明思路**：类似群的第一同构定理，定义 $\bar{\phi}: R/\ker \phi \to \text{im } \phi$。

## 三、域论理论体系

### 3.1 域的定义

**域的定义**：

域 $(F, +, \cdot)$ 是交换除环，即：

- $(F, +, \cdot)$ 是环
- $(F, +)$ 是Abel群
- $(F \setminus \{0\}, \cdot)$ 是Abel群（每个非零元都有乘法逆元）

**等价定义**：

域是满足以下条件的环：

- 交换环
- 有单位元 $1 \neq 0$
- 每个非零元都有乘法逆元

**整环定义**：

整环是满足以下条件的交换环：

- 有单位元 $1 \neq 0$
- 无零因子（如果 $ab = 0$，则 $a = 0$ 或 $b = 0$）

**关系**：

$$\text{域} \subset \text{整环} \subset \text{交换环}$$

### 3.2 域的性质

**域的基本性质**：

1. **每个非零元都有逆元**：
   - $\forall a \in F, a \neq 0: \exists a^{-1} \in F: aa^{-1} = 1$

2. **无零因子**：
   - 如果 $ab = 0$，则 $a = 0$ 或 $b = 0$
   - **证明**：如果 $a \neq 0$，则 $b = a^{-1}(ab) = a^{-1} \cdot 0 = 0$

3. **消去律**：
   - 如果 $a \neq 0$ 且 $ab = ac$，则 $b = c$
   - **证明**：$b = a^{-1}(ab) = a^{-1}(ac) = c$

**域的例子**：

- **有限域**：$\mathbb{Z}_p$（$p$ 是质数）
- **有理数域**：$\mathbb{Q}$
- **实数域**：$\mathbb{R}$
- **复数域**：$\mathbb{C}$

**域扩张**：

如果 $F \subseteq E$ 且 $E$ 是域，则 $E$ 是 $F$ 的域扩张，记作 $E/F$。

**例子**：

- $\mathbb{R}/\mathbb{Q}$：实数域是有理数域的扩张
- $\mathbb{C}/\mathbb{R}$：复数域是实数域的扩张

## 四、抽象代数的历史发展（参考Wikipedia）

### 4.1 群论历史发展

**时间线**：

```
19世纪早期（1830s）
├── Galois（1832）：引入群的概念
│   └── 研究多项式方程的根式可解性
├── Abel（1824）：证明五次方程无根式解
└── Ruffini（1799）：五次方程无根式解的早期尝试

19世纪中期（1850s-1870s）
├── Cayley（1854）：抽象群的定义
│   └── 群的公理化定义
├── Jordan（1870）：置换群理论
├── Sylow（1872）：Sylow定理
└── Burnside（1897）：有限群理论

20世纪早期（1900s-1930s）
├── Noether（1921）：抽象代数方法
├── Artin（1927）：Galois理论
└── Schreier（1927）：群扩张理论

20世纪中期（1940s-1960s）
├── Brauer（1940s）：模表示论
├── Feit-Thompson（1963）：奇数阶群可解
└── Thompson（1960s）：有限单群分类

20世纪后期（1970s-2000s）
└── 有限单群分类（1980s完成）
```

### 4.2 环论历史发展

**时间线**：

```
19世纪早期（1830s-1850s）
├── Gauss（1801）：数论中的模运算
└── Kummer（1847）：理想数

19世纪晚期（1870s-1890s）
├── Dedekind（1871）：理想的概念
├── Hilbert（1890）：Hilbert基定理
└── Kronecker（1882）：代数数论

20世纪早期（1900s-1930s）
├── Noether（1921）：环的理想理论
├── Artin（1927）：Artin环
└── Jacobson（1945）：Jacobson根

20世纪中期（1940s-1960s）
├── Zariski（1940s）：代数几何
└── Grothendieck（1960s）：概形理论

20世纪后期（1970s-现在）
└── 交换代数和非交换代数的发展
```

### 4.3 域论历史发展

**时间线**：

```
19世纪早期（1830s）

19世纪晚期（1880s-1890s）

20世纪早期（1900s-1930s）
```
