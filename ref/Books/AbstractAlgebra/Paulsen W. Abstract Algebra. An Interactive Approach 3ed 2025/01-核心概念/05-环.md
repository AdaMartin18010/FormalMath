## 一、环的定义

### 1.1 基本定义

**定义 9.1**：环 $(R, +, \cdot)$ 是一个集合 $R$ 配两个二元运算 $+$（加法）和 $\cdot$（乘法），满足：

1. **$(R, +)$ 是Abel群**
2. **乘法结合律**：$(ab)c = a(bc)$
3. **分配律**：$a(b + c) = ab + ac$ 和 $(a + b)c = ac + bc$

**数学表述**：

$$\text{环 } (R, +, \cdot) \text{ 满足：}$$
$$(R, +) \text{ 是Abel群}$$
$$\forall a, b, c \in R: (ab)c = a(bc)$$
$$\forall a, b, c \in R: a(b + c) = ab + ac, (a + b)c = ac + bc$$

### 1.2 交换环

**交换环 (Commutative Ring)**：如果环 $R$ 还满足乘法交换律，即对任意 $a, b \in R$，有 $ab = ba$，则称 $R$ 为交换环。

**数学表述**：

$$\forall a, b \in R: ab = ba$$

### 1.3 有单位元的环

**有单位元的环 (Ring with Unity)**：如果环 $R$ 存在乘法单位元 $1$，使得对任意 $a \in R$，有 $1 \cdot a = a \cdot 1 = a$，则称 $R$ 为有单位元的环。

## 二、环的性质

### 2.1 基本性质

**性质**：

- $0 \cdot a = a \cdot 0 = 0$（零元性质）
- $(-a)b = a(-b) = -(ab)$（负元性质）
- $(-a)(-b) = ab$

### 2.2 零因子

**零因子 (Zero Divisor)**：设 $R$ 是环，$a \in R$，$a \neq 0$，如果存在 $b \in R$，$b \neq 0$，使得 $ab = 0$ 或 $ba = 0$，则称 $a$ 是零因子。

**整环 (Integral Domain)**：没有非零零因子的交换环。

**性质**：

- 整环中，如果 $ab = 0$，则 $a = 0$ 或 $b = 0$（消去律）
- 如果 $R$ 是整环，$a \neq 0$，$ab = ac$，则 $b = c$

**例子**：

- $\mathbb{Z}$ 是整环（无零因子）
- $\mathbb{Z}_6$ 不是整环（$2 \cdot 3 = 0$，但 $2, 3 \neq 0$）
- $\mathbb{Z}_p$（$p$ 是质数）是整环（实际上是域）

## 三、环的例子

### 3.1 整数环

**例子**：$(\mathbb{Z}, +, \cdot)$ 是环。

- **加法**：整数加法，$(R, +)$ 是Abel群
- **乘法**：整数乘法，满足结合律
- **分配律**：$a(b + c) = ab + ac$，$(a + b)c = ac + bc$
- **性质**：交换环，有单位元 $1$，整环（无零因子）

**验证**：

- $(\mathbb{Z}, +)$ 是Abel群：封闭性、结合律、单位元 $0$、逆元 $-a$、交换律 ✓
- 乘法结合律：$(ab)c = a(bc)$ ✓
- 分配律：$a(b + c) = ab + ac$ ✓

### 3.2 模 $n$ 环

**例子**：$(\mathbb{Z}_n, +, \cdot)$ 是环，其中 $\mathbb{Z}_n = \{0, 1, 2, \ldots, n-1\}$，运算为模 $n$ 加法和乘法。

- **加法**：模 $n$ 加法，$(R, +)$ 是Abel群
- **乘法**：模 $n$ 乘法，满足结合律
- **分配律**：模 $n$ 运算满足分配律
- **性质**：交换环，有单位元 $1$

**验证**：

- $(\mathbb{Z}_n, +)$ 是Abel群（已在群论中证明）✓
- 乘法结合律：$(a \cdot b) \cdot c \equiv a \cdot (b \cdot c) \pmod{n}$ ✓
- 分配律：$a \cdot (b + c) \equiv ab + ac \pmod{n}$ ✓

**零因子**：

- 在 $\mathbb{Z}_n$ 中，$a$ 是零因子当且仅当 $\gcd(a, n) > 1$
- **例子**：在 $\mathbb{Z}_6$ 中，$2 \cdot 3 = 6 \equiv 0 \pmod{6}$，所以 $2$ 和 $3$ 是零因子

### 3.3 矩阵环

**例子**：$n \times n$ 矩阵环 $M_n(R)$，其中 $R$ 是环。

- **加法**：矩阵加法 $(A + B)_{ij} = A_{ij} + B_{ij}$
- **乘法**：矩阵乘法 $(AB)_{ij} = \sum_{k=1}^n A_{ik}B_{kj}$
- **性质**：有单位元 $I_n$（单位矩阵），非交换（当 $n > 1$）

**验证**：

- $(M_n(R), +)$ 是Abel群 ✓
- 矩阵乘法满足结合律 ✓
- 分配律：$A(B + C) = AB + AC$，$(A + B)C = AC + BC$ ✓

**例子**：$M_2(\mathbb{R})$（$2 \times 2$ 实矩阵环）

- 单位元：$I_2 = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}$
- 非交换：$\begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}\begin{pmatrix} 0 & 0 \\ 1 & 0 \end{pmatrix} \neq \begin{pmatrix} 0 & 0 \\ 1 & 0 \end{pmatrix}\begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}$

### 3.4 多项式环

**例子**：$R[x]$（系数在环 $R$ 中的多项式环）。

- **加法**：多项式加法（按系数相加）
- **乘法**：多项式乘法（卷积）
- **性质**：如果 $R$ 是交换环，则 $R[x]$ 是交换环；如果 $R$ 有单位元，则 $R[x]$ 有单位元

**例子**：$\mathbb{Z}[x]$（整数系数多项式环）

- $f(x) = 2x^2 + 3x + 1$，$g(x) = x - 1$
- $f(x) + g(x) = 2x^2 + 4x$
- $f(x) \cdot g(x) = 2x^3 + x^2 - 2x - 1$

## 四、子环与理想

### 4.1 子环

**定义**：设 $R$ 是环，$S$ 是 $R$ 的子集，如果 $S$ 在 $R$ 的运算下也构成环，则称 $S$ 是 $R$ 的子环。

**判定**：$S$ 是子环当且仅当：

- $S$ 非空
- 对任意 $a, b \in S$，有 $a - b \in S$
- 对任意 $a, b \in S$，有 $ab \in S$

### 4.2 理想

**定义**：设 $R$ 是环，$I$ 是 $R$ 的子集，如果：

- $I$ 是 $(R, +)$ 的子群
- 对任意 $a \in I$ 和 $r \in R$，有 $ar \in I$ 和 $ra \in I$（吸收性质）

则称 $I$ 是 $R$ 的理想，记为 $I \trianglelefteq R$。

**数学表述**：

$$I \trianglelefteq R \Leftrightarrow \begin{cases}
I \text{ 是 } (R, +) \text{ 的子群} \\
\forall a \in I, r \in R: ar \in I, ra \in I
\end{cases}$$

**主理想**：

如果 $R$ 是交换环，由 $a \in R$ 生成的主理想：

$$\langle a \rangle = \{ra : r \in R\}$$

**例子**：

1. **$\mathbb{Z}$ 中的理想**：
   - $\mathbb{Z}$ 的所有理想都是主理想：$\langle n \rangle = n\mathbb{Z} = \{nk : k \in \mathbb{Z}\}$
   - **证明**：设 $I \trianglelefteq \mathbb{Z}$，如果 $I = \{0\}$，则 $I = \langle 0 \rangle$
   - 如果 $I \neq \{0\}$，设 $n$ 是 $I$ 中最小正整数，则 $\langle n \rangle \subseteq I$
   - 对任意 $a \in I$，用除法算法：$a = qn + r$，其中 $0 \leq r < n$
   - 由于 $a, qn \in I$，有 $r = a - qn \in I$，由 $n$ 的最小性，$r = 0$，因此 $a = qn \in \langle n \rangle$
   - 所以 $I = \langle n \rangle$

2. **$\mathbb{Z}_6$ 中的理想**：
   - $\langle 0 \rangle = \{0\}$
   - $\langle 1 \rangle = \mathbb{Z}_6$
   - $\langle 2 \rangle = \{0, 2, 4\}$
   - $\langle 3 \rangle = \{0, 3\}$
   - $\langle 4 \rangle = \{0, 2, 4\} = \langle 2 \rangle$
   - $\langle 5 \rangle = \mathbb{Z}_6 = \langle 1 \rangle$

**商环**：

如果 $I \trianglelefteq R$，则商环：

$$R/I = \{r + I : r \in R\}$$

运算：
- $(a + I) + (b + I) = (a + b) + I$
- $(a + I)(b + I) = ab + I$

**例子**：$\mathbb{Z}_6 / \langle 3 \rangle$：

- $\langle 3 \rangle = \{0, 3\}$
- $R/I = \{\{0, 3\}, \{1, 4\}, \{2, 5\}\}$
- 可以验证 $\mathbb{Z}_6 / \langle 3 \rangle \cong \mathbb{Z}_3$
