## 一、群论例题

### 例题1：验证群

**问题**：验证 $(\mathbb{Z}_n, +)$ 是群，其中 $\mathbb{Z}_n = \{0, 1, 2, \ldots, n-1\}$，运算为模 $n$ 加法。

**形式证明**：

**目标**：证明 $(\mathbb{Z}_n, +)$ 满足群的四个公理。

**证明**：

**步骤1：封闭性（Closure）**

**陈述**：对任意 $a, b \in \mathbb{Z}_n$，有 $a + b \in \mathbb{Z}_n$。

**证明**：

设 $a, b \in \mathbb{Z}_n$，则 $0 \leq a, b < n$。

根据模运算的定义，$a + b \bmod n$ 是 $a + b$ 除以 $n$ 的余数。

设 $a + b = qn + r$，其中 $0 \leq r < n$（带余除法）。

则 $a + b \equiv r \pmod{n}$，且 $r \in \{0, 1, 2, \ldots, n-1\} = \mathbb{Z}_n$。

因此 $a + b \in \mathbb{Z}_n$。✓

**步骤2：结合律（Associativity）**

**陈述**：对任意 $a, b, c \in \mathbb{Z}_n$，有 $(a + b) + c = a + (b + c)$。

**证明**：

设 $a, b, c \in \mathbb{Z}_n$。

根据模运算的性质，$(a + b) + c \equiv a + b + c \pmod{n}$。

类似地，$a + (b + c) \equiv a + b + c \pmod{n}$。

因为整数加法满足结合律：$(a + b) + c = a + (b + c)$。

模 $n$ 运算保持这个性质，所以：

$$(a + b) + c \equiv a + b + c \equiv a + (b + c) \pmod{n}$$

由于两边都在 $\mathbb{Z}_n$ 中，且模 $n$ 同余，所以：

$$(a + b) + c = a + (b + c)$$

因此结合律成立。✓

**步骤3：单位元（Identity）**

**陈述**：存在 $e \in \mathbb{Z}_n$，使得对任意 $a \in \mathbb{Z}_n$，有 $e + a = a + e = a$。

**证明**：

设 $e = 0 \in \mathbb{Z}_n$。

对任意 $a \in \mathbb{Z}_n$：

- $0 + a \equiv a \pmod{n}$，且 $0 + a \in \mathbb{Z}_n$，所以 $0 + a = a$。
- $a + 0 \equiv a \pmod{n}$，且 $a + 0 \in \mathbb{Z}_n$，所以 $a + 0 = a$。

因此 $0$ 是单位元。✓

**步骤4：逆元（Inverse）**

**陈述**：对任意 $a \in \mathbb{Z}_n$，存在 $a^{-1} \in \mathbb{Z}_n$，使得 $a + a^{-1} = a^{-1} + a = 0$。

**证明**：

设 $a \in \mathbb{Z}_n$。

**情况1**：如果 $a = 0$，则 $0 + 0 = 0$，所以 $0^{-1} = 0$。

**情况2**：如果 $a \neq 0$，设 $a^{-1} = n - a$。

验证：

- $a + (n - a) = n \equiv 0 \pmod{n}$，且 $a + (n - a) \in \mathbb{Z}_n$，所以 $a + (n - a) = 0$。
- $(n - a) + a = n \equiv 0 \pmod{n}$，且 $(n - a) + a \in \mathbb{Z}_n$，所以 $(n - a) + a = 0$。

因此 $a^{-1} = n - a$ 是 $a$ 的逆元。✓

**结论**：

由于 $(\mathbb{Z}_n, +)$ 满足所有四个群公理，所以它是群。

**交换性**：

对任意 $a, b \in \mathbb{Z}_n$，有 $a + b = b + a$（因为整数加法交换）。

因此 $(\mathbb{Z}_n, +)$ 是交换群（Abel群）。✓

**参考**：此证明遵循标准的形式证明结构，参考了Wikipedia的"群（数学）"条目和MIT 18.701课程的标准证明格式。

### 例题2：子群判定

**问题**：判定 $H = \{0, 2, 4\} \subseteq \mathbb{Z}_6$ 是否是子群。

**解答**：

使用子群判定定理：$H$ 是子群当且仅当对任意 $a, b \in H$，有 $a - b \in H$（或等价地，$a + b^{-1} \in H$）。

验证封闭性和逆元：

1. **封闭性**：
   - $0 + 0 = 0 \in H$ ✓
   - $0 + 2 = 2 \in H$ ✓
   - $0 + 4 = 4 \in H$ ✓
   - $2 + 2 = 4 \in H$ ✓
   - $2 + 4 = 6 \equiv 0 \pmod{6} \in H$ ✓
   - $4 + 4 = 8 \equiv 2 \pmod{6} \in H$ ✓

2. **逆元**：
   - $0^{-1} = 0 \in H$ ✓
   - $2^{-1} = 6 - 2 = 4 \in H$ ✓
   - $4^{-1} = 6 - 4 = 2 \in H$ ✓

因此 $H = \{0, 2, 4\}$ 是 $\mathbb{Z}_6$ 的子群。

**注**：$H$ 实际上是 $\mathbb{Z}_6$ 的循环子群，由元素 $2$（或 $4$）生成：$\langle 2 \rangle = \{0, 2, 4\}$。

### 例题3：循环群的性质

**问题**：设 $G = \langle a \rangle$ 是 $n$ 阶循环群，证明对任意 $k \in \mathbb{Z}$，有 $a^k = a^{k \bmod n}$。

**解答**：

设 $G = \{e, a, a^2, \ldots, a^{n-1}\}$，其中 $a^n = e$。

对任意 $k \in \mathbb{Z}$，使用带余除法：$k = qn + r$，其中 $0 \leq r < n$。

则：

$$a^k = a^{qn + r} = (a^n)^q \cdot a^r = e^q \cdot a^r = a^r$$

其中 $r = k \bmod n$。

因此 $a^k = a^{k \bmod n}$。

**应用**：在循环群中，指数可以模 $n$ 简化，这简化了计算。

### 例题4：Lagrange定理的应用

**问题**：证明 $6$ 阶群不可能有 $4$ 阶子群。

**形式证明（反证法）**：

**目标**：证明不存在 $6$ 阶群具有 $4$ 阶子群。

**证明方法**：反证法（Proof by Contradiction）

**证明**：

**步骤1：假设**

假设存在 $6$ 阶群 $G$ 具有 $4$ 阶子群 $H$。

即：$|G| = 6$，$H \leq G$，且 $|H| = 4$。

**步骤2：应用Lagrange定理**

**Lagrange定理**（参考Wikipedia和MIT 18.701课程）：

> 如果 $H$ 是有限群 $G$ 的子群，则 $|H|$ 整除 $|G|$。
>
> 即：$|H| \mid |G|$。

**证明思路**（Lagrange定理的证明）：

设 $G$ 是有限群，$H \leq G$。

对任意 $a \in G$，定义左陪集：$aH = \{ah : h \in H\}$。

**引理1**：任意两个左陪集要么相等，要么不相交。

**引理2**：每个左陪集的大小等于 $|H|$。

**引理3**：$G$ 可以表示为不相交左陪集的并：$G = \bigcup_{i=1}^k a_i H$。

因此：$|G| = k \cdot |H|$，所以 $|H| \mid |G|$。✓

**步骤3：应用定理**

根据Lagrange定理，如果 $H \leq G$，则 $|H| \mid |G|$。

在我们的情况下：

- $|H| = 4$
- $|G| = 6$

因此必须有：$4 \mid 6$。

**步骤4：检查整除性**

检查 $4$ 是否整除 $6$：

$6 = 4 \cdot 1 + 2$，余数为 $2 \neq 0$。

因此 $4 \nmid 6$。

**步骤5：得出矛盾**

我们得到：

- 根据Lagrange定理：$4 \mid 6$（如果 $H \leq G$）
- 根据整除性检查：$4 \nmid 6$

这是矛盾！

**步骤6：结论**

矛盾表明我们的假设是错误的。

因此：不存在 $6$ 阶群具有 $4$ 阶子群。✓

**一般结论**：

**定理**：$n$ 阶群的子群的阶必须是 $n$ 的因子。

**证明**：直接应用Lagrange定理。

**应用**：

- $6$ 阶群的子群可能的阶：$1, 2, 3, 6$（$6$ 的因子）
- $6$ 阶群不可能有 $4$ 阶或 $5$ 阶子群

**参考**：此证明遵循标准的形式证明结构，参考了Wikipedia的"拉格朗日定理（群论）"条目和Harvard Math 122课程的标准证明格式。

## 二、环论例题

### 例题5：验证环

**问题**：验证 $(\mathbb{Z}_n, +, \cdot)$ 是环，其中 $\mathbb{Z}_n = \{0, 1, 2, \ldots, n-1\}$，运算为模 $n$ 加法和乘法。

**解答**：

需要验证环的公理：

1. **$(R, +)$ 是Abel群**：已在例题1中证明。

2. **乘法结合律**：对任意 $a, b, c \in \mathbb{Z}_n$，有 $(a \cdot b) \cdot c = a \cdot (b \cdot c)$。
   - 因为整数乘法满足结合律，模 $n$ 运算保持结合律，所以 $(a \cdot b) \cdot c \equiv a \cdot (b \cdot c) \pmod{n}$。

3. **分配律**：
   - **左分配律**：$a \cdot (b + c) = a \cdot b + a \cdot c$
   - **右分配律**：$(a + b) \cdot c = a \cdot c + b \cdot c$
   - 因为整数运算满足分配律，模 $n$ 运算保持分配律。

因此 $(\mathbb{Z}_n, +, \cdot)$ 是环。

**注**：$\mathbb{Z}_n$ 是交换环（因为整数乘法交换），且有单位元 $1$。

---

### 例题6：零因子

**问题**：在环 $\mathbb{Z}_6$ 中，找出所有零因子。

**解答**：

**零因子定义**：非零元素 $a \in R$ 是零因子，如果存在非零元素 $b \in R$ 使得 $ab = 0$。

检查 $\mathbb{Z}_6 = \{0, 1, 2, 3, 4, 5\}$ 中的每个非零元素：

- $1$：$1 \cdot a = a \neq 0$（当 $a \neq 0$），所以 $1$ 不是零因子。
- $2$：$2 \cdot 3 = 6 \equiv 0 \pmod{6}$，所以 $2$ 是零因子。
- $3$：$3 \cdot 2 = 6 \equiv 0 \pmod{6}$，所以 $3$ 是零因子。
- $4$：$4 \cdot 3 = 12 \equiv 0 \pmod{6}$，所以 $4$ 是零因子。
- $5$：$5 \cdot a$ 在 $\mathbb{Z}_6$ 中，需要检查：
  - $5 \cdot 1 = 5 \neq 0$
  - $5 \cdot 2 = 10 \equiv 4 \pmod{6} \neq 0$
  - $5 \cdot 3 = 15 \equiv 3 \pmod{6} \neq 0$
  - $5 \cdot 4 = 20 \equiv 2 \pmod{6} \neq 0$
  - $5 \cdot 5 = 25 \equiv 1 \pmod{6} \neq 0$
  - 所以 $5$ 不是零因子（实际上 $5^{-1} = 5$，因为 $5 \cdot 5 = 25 \equiv 1 \pmod{6}$）。

**答案**：$\mathbb{Z}_6$ 中的零因子是 $2, 3, 4$。

**一般结论**：在 $\mathbb{Z}_n$ 中，$a$ 是零因子当且仅当 $\gcd(a, n) > 1$。

### 例题7：理想的性质

**问题**：设 $I = \{0, 3\} \subseteq \mathbb{Z}_6$，验证 $I$ 是理想。

**解答**：

**理想定义**：$I$ 是环 $R$ 的理想，如果：

1. $(I, +)$ 是 $(R, +)$ 的子群
2. 对任意 $r \in R$ 和 $i \in I$，有 $ri \in I$ 和 $ir \in I$

验证：

1. **$(I, +)$ 是子群**：
   - $0 + 0 = 0 \in I$ ✓
   - $0 + 3 = 3 \in I$ ✓
   - $3 + 3 = 6 \equiv 0 \pmod{6} \in I$ ✓
   - $3^{-1} = 3 \in I$ ✓

2. **吸收性质**：对任意 $r \in \mathbb{Z}_6$ 和 $i \in I$：
   - $r \cdot 0 = 0 \in I$ ✓
   - $r \cdot 3$：需要检查所有 $r \in \mathbb{Z}_6$
     - $0 \cdot 3 = 0 \in I$ ✓
     - $1 \cdot 3 = 3 \in I$ ✓
     - $2 \cdot 3 = 6 \equiv 0 \in I$ ✓
     - $3 \cdot 3 = 9 \equiv 3 \in I$ ✓
     - $4 \cdot 3 = 12 \equiv 0 \in I$ ✓
     - $5 \cdot 3 = 15 \equiv 3 \in I$ ✓
   - 类似地，$3 \cdot r \in I$（因为 $\mathbb{Z}_6$ 是交换环）

因此 $I = \{0, 3\}$ 是 $\mathbb{Z}_6$ 的理想。

**注**：$I = \langle 3 \rangle$ 是由 $3$ 生成的主理想。

## 三、形式证明方法总结

### 3.1 直接证明（Direct Proof）

**方法**：从已知条件直接推导出结论。

**例子**：例题1（验证群）使用直接证明方法。

**结构**：

1. 陈述已知条件
2. 应用定义和定理
3. 逐步推导
4. 得出结论

### 3.2 反证法（Proof by Contradiction）

**方法**：假设结论不成立，推导出矛盾。

**例子**：例题4（Lagrange定理应用）使用反证法。

**结构**：

1. 假设结论不成立
2. 应用已知定理
3. 推导出矛盾
4. 得出结论

### 3.3 数学归纳法（Mathematical Induction）

**方法**：证明基础情况，然后证明归纳步骤。

**结构**：

1. **基础步骤**：证明 $P(1)$ 成立
2. **归纳步骤**：假设 $P(k)$ 成立，证明 $P(k+1)$ 成立
3. **结论**：对所有 $n$，$P(n)$ 成立

### 3.4 构造性证明（Constructive Proof）

**方法**：通过构造具体例子来证明存在性。

**例子**：例题2（子群判定）通过构造验证来证明。

### 3.5 分类讨论（Proof by Cases）

**方法**：将所有可能情况分类，分别证明。

**例子**：例题6（零因子）通过检查每个元素来证明。
