## 一、群公理 → 基本定理链

```text
【群公理】(G, *) 为群
├── 封闭性：∀a,b∈G, ab∈G
├── 结合律：∀a,b,c∈G, (ab)c = a(bc)
├── 单位元：∃e∈G, ∀a∈G, ea = ae = a
└── 逆元：∀a∈G, ∃a⁻¹∈G, aa⁻¹ = a⁻¹a = e

    ↓ 由公理推出

【基本定理】
├── 单位元唯一性；逆元唯一性
├── 消去律：ab = ac ⇒ b = c（及右消去）
├── (a⁻¹)⁻¹ = a；(ab)⁻¹ = b⁻¹a⁻¹
└── 方程 ax = b 在 G 中唯一可解
```

**对应文档**: [04-知识关联/01-概念关联网络](../04-知识关联/01-概念关联网络.md)；与 MIT 18.701 / AMS 群定义一致。

## 二、子群与商群推理链

```text
【子群】H ≤ G：H ⊆ G 且 (H, *) 为群

    ↓ 判定定理

【子群判定】H ≤ G ⇔ H≠∅ 且 ∀a,b∈H 有 ab⁻¹∈H

【正规子群】N ◁ G：∀g∈G, gN = Ng

    ↓ 商群构造

【商群】G/N = {gN : g∈G}，运算 (aN)(bN) = (ab)N
├── 自然同态 π: G → G/N 满足 ker π = N
└── |G/N| = [G:N]（有限时 |G|/|N|）
```

**对应文档**: [04-知识关联/01-概念关联网络](../04-知识关联/01-概念关联网络.md)；与 MIT 18.701 / Cambridge 对应。

## 三、环公理 → 理想与商环链

```text
【环公理】(R, +, ·)
├── (R, +) 阿贝尔群；乘法结合；分配律
└── 可选：乘法单位元 1

    ↓ 理想与商环

【理想】I ⊆ R：加法子群且 rI ⊆ I, Ir ⊆ I
├── 商环 R/I：运算 (a+I)(b+I) = ab+I
├── 自然同态 π: R → R/I 满足 ker π = I
└── 第一同构定理：R/ker φ ≅ im φ（φ 为环同态）
```

**对应文档**: [04-知识关联/01-概念关联网络](../04-知识关联/01-概念关联网络.md) 环论部分；与 MIT 18.702 / Cambridge 一致。

## 四、同态与同构定理链

```text
【群同态】φ: G → H，φ(ab) = φ(a)φ(b)

├── ker φ ◁ G；im φ ≤ H
├── φ(e_G) = e_H；φ(a⁻¹) = φ(a)⁻¹
└── 第一同构定理：G/ker φ ≅ im φ（φ̃: g·ker φ ↦ φ(g)）

【环同态】φ: R → S，保加法和乘法

├── ker φ 为理想；im φ 为子环
└── 第一同构定理：R/ker φ ≅ im φ
```

**对应文档**: [04-知识关联](../04-知识关联/01-概念关联网络.md)；与 Harvard Math 122 / Cambridge 同态定理对应。
