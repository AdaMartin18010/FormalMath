## 一、映射计数问题

### 例题1：从有限集合到有限集合的映射数量

**问题**：设 $A = \{John, Mary, Sam\}$，$B = \{eggs, coffee\}$。有多少个不同的映射 $f: A \to B$？

**解答**：

映射 $f$ 需要为 $A$ 中的每个元素指定 $B$ 中的一个元素。

- $f(John)$ 可以是 $eggs$ 或 $coffee$：2种选择
- $f(Mary)$ 可以是 $eggs$ 或 $coffee$：2种选择
- $f(Sam)$ 可以是 $eggs$ 或 $coffee$：2种选择

因此，总共有 $2 \times 2 \times 2 = 2^3 = 8$ 个不同的映射。

**一般结论**：从 $n$ 元素集合到 $m$ 元素集合的映射数量是 $m^n$。

**验证**：列出所有8个映射：

1. $f_1(John) = eggs$，$f_1(Mary) = eggs$，$f_1(Sam) = eggs$
2. $f_2(John) = eggs$，$f_2(Mary) = eggs$，$f_2(Sam) = coffee$
3. $f_3(John) = eggs$，$f_3(Mary) = coffee$，$f_3(Sam) = eggs$
4. $f_4(John) = eggs$，$f_4(Mary) = coffee$，$f_4(Sam) = coffee$
5. $f_5(John) = coffee$，$f_5(Mary) = eggs$，$f_5(Sam) = eggs$
6. $f_6(John) = coffee$，$f_6(Mary) = eggs$，$f_6(Sam) = coffee$
7. $f_7(John) = coffee$，$f_7(Mary) = coffee$，$f_7(Sam) = eggs$
8. $f_8(John) = coffee$，$f_8(Mary) = coffee$，$f_8(Sam) = coffee$

### 例题2：自映射的数量

**问题**：设 $A = \{John, Mary, Sam\}$。有多少个不同的自映射 $f: A \to A$？

**解答**：

自映射 $f$ 需要为 $A$ 中的每个元素指定 $A$ 中的一个元素。

- $f(John)$ 可以是 $John$、$Mary$ 或 $Sam$：3种选择
- $f(Mary)$ 可以是 $John$、$Mary$ 或 $Sam$：3种选择
- $f(Sam)$ 可以是 $John$、$Mary$ 或 $Sam$：3种选择

因此，总共有 $3 \times 3 \times 3 = 3^3 = 27$ 个不同的自映射。

**一般结论**：$n$ 元素集合的自映射数量是 $n^n$。

### 例题3：从较小集合到较大集合的映射数量

**问题**：设 $A = \{eggs, coffee\}$，$B = \{John, Mary, Sam\}$。有多少个不同的映射 $f: A \to B$？

**解答**：

映射 $f$ 需要为 $A$ 中的每个元素指定 $B$ 中的一个元素。

- $f(eggs)$ 可以是 $John$、$Mary$ 或 $Sam$：3种选择
- $f(coffee)$ 可以是 $John$、$Mary$ 或 $Sam$：3种选择

因此，总共有 $3 \times 3 = 3^2 = 9$ 个不同的映射。

**一般结论**：从 $m$ 元素集合到 $n$ 元素集合的映射数量是 $n^m$。

## 二、复合映射问题

### 例题4：复合映射的计算

**问题**：设 $A = \{1, 2, 3\}$，$B = \{a, b\}$，$C = \{x, y, z\}$。

定义映射：

- $g: A \to B$，$g(1) = a$，$g(2) = b$，$g(3) = a$
- $f: B \to C$，$f(a) = x$，$f(b) = y$

计算复合映射 $f \circ g: A \to C$。

**解答**：

对每个 $a \in A$，计算 $(f \circ g)(a) = f(g(a))$：

- $(f \circ g)(1) = f(g(1)) = f(a) = x$
- $(f \circ g)(2) = f(g(2)) = f(b) = y$
- $(f \circ g)(3) = f(g(3)) = f(a) = x$

因此，$f \circ g: A \to C$ 定义为：

- $(f \circ g)(1) = x$
- $(f \circ g)(2) = y$
- $(f \circ g)(3) = x$

### 例题5：复合映射的结合律验证

**问题**：设 $A = \{1, 2\}$，$B = \{a, b\}$，$C = \{x, y\}$，$D = \{p, q\}$。

定义映射：

- $f: A \to B$，$f(1) = a$，$f(2) = b$
- $g: B \to C$，$g(a) = x$，$g(b) = y$
- $h: C \to D$，$h(x) = p$，$h(y) = q$

验证 $(h \circ g) \circ f = h \circ (g \circ f)$。

**解答**：

**左边**：$(h \circ g) \circ f$

首先计算 $h \circ g: B \to D$：

- $(h \circ g)(a) = h(g(a)) = h(x) = p$
- $(h \circ g)(b) = h(g(b)) = h(y) = q$

然后计算 $(h \circ g) \circ f: A \to D$：

- $((h \circ g) \circ f)(1) = (h \circ g)(f(1)) = (h \circ g)(a) = p$
- $((h \circ g) \circ f)(2) = (h \circ g)(f(2)) = (h \circ g)(b) = q$

**右边**：$h \circ (g \circ f)$

首先计算 $g \circ f: A \to C$：

- $(g \circ f)(1) = g(f(1)) = g(a) = x$
- $(g \circ f)(2) = g(f(2)) = g(b) = y$

然后计算 $h \circ (g \circ f): A \to D$：

- $(h \circ (g \circ f))(1) = h((g \circ f)(1)) = h(x) = p$
- $(h \circ (g \circ f))(2) = h((g \circ f)(2)) = h(y) = q$

**结论**：左边和右边相等，验证了结合律。

## 三、恒等映射问题

### 例题6：恒等映射的复合

**问题**：设 $A = \{1, 2, 3\}$，$B = \{a, b\}$，映射 $f: A \to B$ 定义为 $f(1) = a$，$f(2) = b$，$f(3) = a$。

验证：

1. $f \circ 1_A = f$
2. $1_B \circ f = f$

**解答**：

**1. 验证 $f \circ 1_A = f$**：

恒等映射 $1_A: A \to A$ 定义为 $1_A(1) = 1$，$1_A(2) = 2$，$1_A(3) = 3$。

计算 $f \circ 1_A: A \to B$：

- $(f \circ 1_A)(1) = f(1_A(1)) = f(1) = a$
- $(f \circ 1_A)(2) = f(1_A(2)) = f(2) = b$
- $(f \circ 1_A)(3) = f(1_A(3)) = f(3) = a$

因此，$f \circ 1_A = f$。

**2. 验证 $1_B \circ f = f$**：

恒等映射 $1_B: B \to B$ 定义为 $1_B(a) = a$，$1_B(b) = b$。

计算 $1_B \circ f: A \to B$：

- $(1_B \circ f)(1) = 1_B(f(1)) = 1_B(a) = a$
- $(1_B \circ f)(2) = 1_B(f(2)) = 1_B(b) = b$
- $(1_B \circ f)(3) = 1_B(f(3)) = 1_B(a) = a$

因此，$1_B \circ f = f$。

## 四、映射性质判定问题

### 例题7：单射、满射、双射的判定

**问题**：判定以下映射的性质（单射、满射、双射）：

1. $f: \{1, 2, 3\} \to \{a, b, c\}$，$f(1) = a$，$f(2) = b$，$f(3) = c$
2. $g: \{1, 2, 3\} \to \{a, b\}$，$g(1) = a$，$g(2) = b$，$g(3) = b$
3. $h: \{1, 2\} \to \{a, b, c\}$，$h(1) = a$，$h(2) = b$

**解答**：

**1. 映射 $f$**：

- **单射性**：$f(1) = a$，$f(2) = b$，$f(3) = c$ 都不同，因此 $f$ 是单射。
- **满射性**：$f$ 的值域是 $\{a, b, c\}$，等于值域，因此 $f$ 是满射。
- **双射性**：由于 $f$ 既是单射又是满射，因此 $f$ 是双射。

**2. 映射 $g$**：

- **单射性**：$g(2) = b$ 且 $g(3) = b$，因此 $g$ 不是单射。
- **满射性**：$g$ 的值域是 $\{a, b\}$，等于值域，因此 $g$ 是满射。
- **双射性**：由于 $g$ 不是单射，因此 $g$ 不是双射。

**3. 映射 $h$**：

- **单射性**：$h(1) = a$，$h(2) = b$ 不同，因此 $h$ 是单射。
- **满射性**：$h$ 的值域是 $\{a, b\}$，但值域是 $\{a, b, c\}$，因此 $h$ 不是满射。
- **双射性**：由于 $h$ 不是满射，因此 $h$ 不是双射。

## 五、同构问题

### 例题8：构造同构映射

**问题**：证明集合 $\{1, 2, 3\}$ 和 $\{a, b, c\}$ 是同构的，并构造一个同构映射。

**解答**：

**步骤1**：定义映射 $f: \{1, 2, 3\} \to \{a, b, c\}$：

- $f(1) = a$
- $f(2) = b$
- $f(3) = c$

**步骤2**：验证 $f$ 是双射：

- **单射性**：$f(1) = a$，$f(2) = b$，$f(3) = c$ 都不同，因此 $f$ 是单射。
- **满射性**：$f$ 的值域是 $\{a, b, c\}$，等于值域，因此 $f$ 是满射。

因此，$f$ 是双射，从而是同构。

**步骤3**：构造逆映射 $f^{-1}: \{a, b, c\} \to \{1, 2, 3\}$：

- $f^{-1}(a) = 1$
- $f^{-1}(b) = 2$
- $f^{-1}(c) = 3$

**步骤4**：验证 $f^{-1}$ 是 $f$ 的逆：

- $f^{-1} \circ f = 1_{\{1,2,3\}}$：
  - $(f^{-1} \circ f)(1) = f^{-1}(f(1)) = f^{-1}(a) = 1$
  - $(f^{-1} \circ f)(2) = f^{-1}(f(2)) = f^{-1}(b) = 2$
  - $(f^{-1} \circ f)(3) = f^{-1}(f(3)) = f^{-1}(c) = 3$

- $f \circ f^{-1} = 1_{\{a,b,c\}}$：
  - $(f \circ f^{-1})(a) = f(f^{-1}(a)) = f(1) = a$
  - $(f \circ f^{-1})(b) = f(f^{-1}(b)) = f(2) = b$
  - $(f \circ f^{-1})(c) = f(f^{-1}(c)) = f(3) = c$

因此，$f$ 是同构映射。

### 例题9：自同构群

**问题**：求集合 $\{1, 2\}$ 的所有自同构，并验证它们构成群。

**解答**：

**步骤1**：列出所有自映射：

集合 $\{1, 2\}$ 有 $2^2 = 4$ 个自映射：

1. $f_1(1) = 1$，$f_1(2) = 1$（不是双射）
2. $f_2(1) = 1$，$f_2(2) = 2$（恒等映射，是双射）
3. $f_3(1) = 2$，$f_3(2) = 1$（对换，是双射）
4. $f_4(1) = 2$，$f_4(2) = 2$（不是双射）

**步骤2**：识别自同构：

自同构是双射的自映射，因此有：

- $1_{\{1,2\}} = f_2$：恒等映射
- $\sigma = f_3$：对换映射，$\sigma(1) = 2$，$\sigma(2) = 1$

**步骤3**：验证群结构：

- **单位元**：$1_{\{1,2\}}$ 是单位元
- **逆元**：
  - $1_{\{1,2\}}^{-1} = 1_{\{1,2\}}$
  - $\sigma^{-1} = \sigma$（因为 $\sigma \circ \sigma = 1_{\{1,2\}}$）
- **封闭性**：
  - $1_{\{1,2\}} \circ 1_{\{1,2\}} = 1_{\{1,2\}}$
  - $1_{\{1,2\}} \circ \sigma = \sigma$
  - $\sigma \circ 1_{\{1,2\}} = \sigma$
  - $\sigma \circ \sigma = 1_{\{1,2\}}$
- **结合律**：映射的复合满足结合律

因此，$\text{Aut}(\{1, 2\}) = \{1_{\{1,2\}}, \sigma\}$ 是一个2阶群，同构于 $\mathbb{Z}_2$。

## 六、形式证明方法总结

### 6.1 证明方法分类

**直接证明**：从已知条件直接推导出结论

- 例子：例题1-3（映射计数）、例题4-5（复合映射）

**构造性证明**：通过构造具体例子来证明存在性

- 例子：例题8（构造同构映射）

**验证性证明**：通过验证性质来证明

- 例子：例题7（映射性质判定）、例题9（自同构群）

### 6.2 证明写作规范

**标准结构**：

1. 陈述问题
2. 选择证明方法
3. 逐步推导
4. 得出结论
5. 验证结果
