## 二、概念定义权威对照表（至少 5 项）

| 概念 | 本书定义摘要 | 权威对应（nLab/课程） | 备注 |
|------|--------------|------------------------|------|
| **范畴** | 对象、映射、恒等、复合（结合律与单位律） | nLab: Category | 一致 |
| **同构** | 可逆映射，逆复合为恒等 | nLab: Isomorphism | 一致 |
| **终对象** | 对任意对象存在唯一态射 | nLab: Terminal object | 一致 |
| **积** | 对象与投影的泛性质 | nLab: Product | 一致 |
| **函子** | 保持恒等与复合的映射 | nLab: Functor | 一致 |
| **自然变换** | 态射族满足自然性方块 | nLab: Natural transformation | 一致 |
