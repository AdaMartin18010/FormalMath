## 一、范畴公理 → 函子定律 → 自然性

```text
【范畴公理】
├── 恒等律：f ∘ 1_A = f，1_B ∘ f = f
├── 结合律：(h ∘ g) ∘ f = h ∘ (g ∘ f)
└── 推论：恒等唯一；复合唯一到定义域/陪域

    ↓ 函子 F: 𝒞 → 𝒟 保持结构

【函子定律】
├── F(1_A) = 1_{F(A)}
├── F(g ∘ f) = F(g) ∘ F(f)
└── 推论：函子保持同构

    ↓ 自然变换 η: F ⇒ G

【自然性】
├── 对任意 f: A → B，η_B ∘ F(f) = G(f) ∘ η_A
└── 推论：自然同构 ⇒ 函子等价（在适当意义下）
```

**对应文档**: [01-范畴](../01-核心概念/01-范畴.md)、[04-知识关联](../04-知识关联/01-概念关联网络.md)；与 nLab Category / Functor / Natural transformation 一致。

## 二、泛性质推理链（终对象、积）

```text
【终对象】T：对任意对象 A，存在唯一态射 A → T
├── 唯一性由泛性质保证
└── 例：Set 中单元素集为终对象

【积】A×B：对象带投影 π_A: A×B → A，π_B: A×B → B
├── 泛性质：对任意 (f: C→A, g: C→B)，存在唯一 u: C → A×B 使 π_A∘u=f，π_B∘u=g
└── 推论：积在同构意义下唯一

【余积】A+B：对偶地由入射与泛性质定义
└── 例：Set 中不交并
```

**对应文档**: [04-知识关联](../04-知识关联/01-概念关联网络.md) 泛性质部分；与 nLab Terminal object / Product 一致。

## 三、同构与等价

```text
【同构】f: A → B 为同构 ⇔ 存在 g: B → A 使 g∘f=1_A，f∘g=1_B
├── 推论：恒等为同构；同构的复合为同构；同构的逆为同构
└── 本书强调：同构即「可逆的映射」

【范畴等价】函子 F: 𝒞 → 𝒟，G: 𝒟 → 𝒞
├── 若 GF ≅ Id_𝒞 且 FG ≅ Id_𝒟（自然同构），则 𝒞 与 𝒟 等价
└── 比「同构」弱，但保持许多范畴论性质
```

**对应文档**: [04-同构](../01-核心概念/04-同构.md)；与 nLab Isomorphism / Equivalence of categories 一致。
