## 一、学习路径决策树

```text
开始学习范畴论（本书 Conceptual Mathematics）
  ├─ 是否理解集合与函数？
  │   ├─ 否 → 先复习集合论与函数
  │   └─ 是 → 继续
  ├─ 是否掌握范畴的定义（对象、映射、恒等、复合）？
  │   ├─ 否 → 学习 01-范畴、02-映射、03-复合
  │   └─ 是 → 继续
  ├─ 是否理解同构与泛性质（终对象、积）？
  │   ├─ 否 → 学习 04-同构 与 04-知识关联 泛性质部分
  │   └─ 是 → 继续
  └─ 是否接触函子与自然变换？
      ├─ 否 → 学习 04-知识关联 函子与自然变换
      └─ 是 → 可拓展伴随、极限（nLab/MIT）
```

## 二、概念选择决策树

```text
需要理解「结构」
  ├─ 仅看对象与映射？ → 范畴、态射、恒等、复合
  ├─ 看「等价」？ → 同构（可逆态射）
  ├─ 看「唯一性」？ → 终对象、始对象、泛性质
  ├─ 看「积/和」？ → 积（泛性质）、余积
  └─ 看「范畴间关系」？ → 函子、自然变换、伴随
```
