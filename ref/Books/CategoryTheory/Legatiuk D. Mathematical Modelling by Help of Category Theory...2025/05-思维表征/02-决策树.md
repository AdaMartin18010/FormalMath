## 一、学习路径决策树

```text
开始学习范畴论建模（本书 Legatiuk）
  ├─ 是否理解数学模型 (S, F, C)？
  │   ├─ 否 → 学习 01-数学模型、02-建模错误
  │   └─ 是 → 继续
  ├─ 是否掌握范畴与函子？
  │   ├─ 否 → 学习 03-数学模型范畴、04-知识关联 范畴与函子
  │   └─ 是 → 继续
  ├─ 是否理解耦合与复合系统？
  │   ├─ 否 → 学习 04-耦合数学模型
  │   └─ 是 → 继续
  └─ 是否接触 Olog/本体？
      ├─ 否 → 学习 05-范畴本体日志
      └─ 是 → 可拓展应用案例、nLab/MIT 应用
```

## 二、建模选择决策树

```text
需要建立或分析模型
  ├─ 单模型、单层次？ → 数学模型 (S,F,C)、建模错误分类
  ├─ 多模型、需比较？ → 模型范畴、函子
  ├─ 多系统、需耦合？ → 耦合数学模型、自然变换
  └─ 知识/本体表示？ → Olog、可交换图
```
