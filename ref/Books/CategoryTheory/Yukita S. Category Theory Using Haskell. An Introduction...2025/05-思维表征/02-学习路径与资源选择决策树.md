## 一、学习路径选择决策树

```text
你的背景与目标？
├── 先打好数学基础再写代码
│   └── 路径：范畴 → 函子 → 自然变换 → 极限/余极限 → 伴随 → 单子 → Yoneda
│       资源：Yukita 数学部分 + Cambridge CAT 或 nLab 补充
├── 先会写 Haskell 再补范畴论
│   └── 路径：Hask 与类型 → Functor → 多态与自然变换 → Monad → 伴随与 List → Yoneda 应用
│       资源：Yukita Haskell 部分 + Bartosz 视频
└── 数学与代码并进（推荐）
    └── 路径：每章先读定义与定律，再做对应 Haskell 例题
        资源：Yukita 全书 + 03-典型例题 + 可选 MIT problem sets
```

## 三、概念困难时的查阅与练习决策树

```text
某个概念（如伴随、Yoneda）看不懂？
├── 定义不清楚
│   └── 查 nLab 对应词条；对照 01-核心概念 中该概念的「数学定义」小节
├── 不知道在 Haskell 里长什么样
│   └── 查本书 01-核心概念 中该概念的「Haskell 对应」；运行 03-典型例题 中相关代码
├── 自然性/定律不会证
│   └── 查 05-思维表征/03-公理定理推理树；对照 nLab 或 Cambridge 证明
└── 不知道有什么用
    └── 查 06-应用拓展；Bartosz 或 Seven Sketches 的应用章节
```
