## 一、范畴公理 → 函子定律 → 自然性

```text
【范畴公理】
├── 恒等律：f ∘ 1_A = f，1_B ∘ f = f
├── 结合律：(h ∘ g) ∘ f = h ∘ (g ∘ f)
└── 推论：恒等唯一；复合唯一到定义域/陪域

    ↓ 函子 F: 𝒞 → 𝒟 保持结构

【函子定律】
├── F(1_A) = 1_{F(A)}
├── F(g ∘ f) = F(g) ∘ F(f)
└── 推论：fmap id = id；fmap (g . f) = fmap g . fmap f（Haskell）

    ↓ 自然变换 η: F ⇒ G 为函子之间的映射

【自然性】
├── 对任意 f: A → B，η_B ∘ F(f) = G(f) ∘ η_A
└── 推论：Haskell 中 forall a. F a -> G a 自动满足自然性
```

## 二、伴随 ⇔ 单位–余单位 ⇔ 单子

```text
【伴随 F ⊣ G】
├── 定义：Hom_𝒟(F(A), B) ≅ Hom_𝒞(A, G(B)) 自然于 A, B
├── 等价：单位 η: Id → GF，余单位 ε: FG → Id 满足三角恒等式
│   ├── (εF)∘(Fη) = id_F
│   └── (Gε)∘(ηG) = id_G
└── 推论：单子 T = GF，η 为单位，μ = G(εF) 为乘法

【从伴随得到单子】
├── T = G∘F，η_A 为单位，μ_A = G(ε_{F(A)})
├── 单子定律由三角恒等式推出
└── 例：Free ⊣ U（集合→幺半群）→ List 单子

【从单子得到伴随（Kleisli）】
├── Kleisli 范畴 𝒞_T：对象同 𝒞，Hom_𝒞_T(A,B) = Hom_𝒞(A, T(B))
├── F_T: A ↦ A, f ↦ η∘f；G_T: A ↦ T(A), f ↦ μ∘T(f)
└── F_T ⊣ G_T，且 G_T∘F_T = T
```

**对应文档**: [04-单子](../01-核心概念/04-单子.md)、[06-伴随](../01-核心概念/06-伴随.md)。

## 三、Yoneda 引理 → 表示与唯一性

```text
【协变 Yoneda】
├── 设 F: 𝒞^op → Set（或 𝒞 → Set），A ∈ 𝒞
├── Hom(Hom_𝒞(-, A), F) ≅ F(A) 自然于 A 与 F
└── 推论：函子 F 由其在 A 处的“值”与 Hom(-,A) 的映射唯一决定（到自然同构）

【反变 Yoneda】
├── Hom(Hom_𝒞(A, -), F) ≅ F(A)
└── 同上，用于反变函子

【Haskell / 编程】
├── Yoneda 类型封装 f a 为 (forall x. (a -> x) -> f x)
├── 与 Coyoneda 等用于优化与 API 设计（Yoneda 机器）
└── 见 05-Yoneda引理、06-应用拓展
```

**对应文档**: [05-Yoneda引理](../01-核心概念/05-Yoneda引理.md)。

## 四、极限/余极限与泛性质

```text
【极限】
├── 图 F: J → 𝒞；锥 (L, π_j: L → F(j)) 满足 F(α)∘π_j = π_k
├── 极限：泛锥——对任意锥 (C, p_j)，存在唯一 u: C → L 使 p_j = π_j∘u
└── 特例：终对象、积、等化子、拉回

【余极限】
├── 余锥 (L, ι_j: F(j) → L)；余极限：泛余锥
└── 特例：始对象、余积、余等化子、推出

【Haskell】
├── 积 ↔ (a, b)；余积 ↔ Either a b；终对象 ↔ ()；始对象 ↔ Void
└── 见 07-极限与余极限、03-典型例题
```

**对应文档**: [07-极限与余极限](../01-核心概念/07-极限与余极限.md)。
