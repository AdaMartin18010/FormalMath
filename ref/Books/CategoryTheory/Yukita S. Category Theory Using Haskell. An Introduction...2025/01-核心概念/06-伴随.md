## 一、伴随的定义

### 1.1 伴随的数学定义

**伴随 (Adjoint)** 函子对 $F \dashv G: \mathcal{C} \to \mathcal{D}$ 满足：

对任意对象 $A \in \mathcal{C}$ 和 $B \in \mathcal{D}$，存在自然同构：

$$\text{Hom}_{\mathcal{D}}(F(A), B) \cong \text{Hom}_{\mathcal{C}}(A, G(B))$$

**记号**：$F \dashv G$ 表示 $F$ 左伴随于 $G$，$G$ 右伴随于 $F$。

### 1.2 伴随的等价定义

**等价定义1**：存在自然变换：

- **单位 (Unit)**: $\eta: 1_{\mathcal{C}} \Rightarrow G \circ F$
- **余单位 (Counit)**: $\epsilon: F \circ G \Rightarrow 1_{\mathcal{D}}$

满足三角恒等式。

**等价定义2**：对每个对象 $A \in \mathcal{C}$，存在泛箭头：

- 泛箭头：$\eta_A: A \to G(F(A))$
- 泛性质：对任意 $f: A \to G(B)$，存在唯一的 $g: F(A) \to B$ 使得 $f = G(g) \circ \eta_A$

## 二、伴随的单位与余单位

### 2.1 单位自然变换

**单位 (Unit)** $\eta: 1_{\mathcal{C}} \Rightarrow G \circ F$：

- 对每个对象 $A \in \mathcal{C}$，$\eta_A: A \to G(F(A))$
- 自然性：对 $f: A \to B$，$G(F(f)) \circ \eta_A = \eta_B \circ f$

**Haskell对应**：

- 单位对应 `return :: a -> m a`（在单子中）
- 或 `pure :: a -> f a`（在Applicative中）

### 2.2 余单位自然变换

**余单位 (Counit)** $\epsilon: F \circ G \Rightarrow 1_{\mathcal{D}}$：

- 对每个对象 $B \in \mathcal{D}$，$\epsilon_B: F(G(B)) \to B$
- 自然性：对 $g: B \to B'$，$g \circ \epsilon_B = \epsilon_{B'} \circ F(G(g))$

**Haskell对应**：

- 余单位对应某些"提取"操作
- 在单子中，对应 `join` 或 `(>>=)` 的某种形式

### 2.3 三角恒等式

**三角恒等式**：

1. **左三角恒等式**：$(\epsilon F) \circ (F \eta) = 1_F$
   - 对对象 $A$：$\epsilon_{F(A)} \circ F(\eta_A) = 1_{F(A)}$

2. **右三角恒等式**：$(G \epsilon) \circ (\eta G) = 1_G$
   - 对对象 $B$：$G(\epsilon_B) \circ \eta_{G(B)} = 1_{G(B)}$

**Haskell对应**：

- 左三角恒等式对应单子的左单位律
- 右三角恒等式对应单子的右单位律

## 三、伴随与单子

### 3.1 从伴随到单子

**定理**：伴随 $F \dashv G: \mathcal{C} \to \mathcal{D}$ 产生单子 $(T, \eta, \mu)$，其中：

- **函子**: $T = G \circ F$
- **单位**: $\eta: 1_{\mathcal{C}} \Rightarrow T$（伴随的单位）
- **乘法**: $\mu = G(\epsilon_F): T \circ T \Rightarrow T$（伴随的余单位）

**证明思路**：

- 单子定律由伴随的三角恒等式保证
- 左单位律：$\mu_A \circ T(\eta_A) = G(\epsilon_{F(A)}) \circ G(F(\eta_A)) = G(\epsilon_{F(A)} \circ F(\eta_A)) = G(1_{F(A)}) = 1_{G(F(A))}$
- 右单位律和结合律类似

### 3.2 从单子到伴随

**定理**：单子 $(T, \eta, \mu)$ 产生Kleisli伴随 $F_T \dashv G_T$。

**构造**：

- **Kleisli范畴** $\mathcal{C}_T$：
  - 对象：与 $\mathcal{C}$ 相同
  - 映射：$A \to B$ 对应 $\mathcal{C}$ 中的映射 $A \to T(B)$
- **函子**：
  - $F_T: \mathcal{C} \to \mathcal{C}_T$，$F_T(A) = A$，$F_T(f) = \eta_B \circ f$
  - $G_T: \mathcal{C}_T \to \mathcal{C}$，$G_T(A) = T(A)$，$G_T(f) = \mu_B \circ T(f)$

**Haskell对应**：

- Kleisli范畴对应Haskell中的单子计算
- `(>>=)` 对应Kleisli范畴中的复合

## 四、Haskell中的伴随

### 4.1 自由函子与遗忘函子

**自由函子与遗忘函子构成伴随**：

```haskell
-- 自由单子函子
data Free f a = Pure a | Free (f (Free f a))

-- 遗忘函子（到Hask）
forget :: Free f a -> a
forget (Pure a) = a
forget (Free _) = error "Cannot forget structure"
```

**数学对应**：

- `Free f` 对应自由函子 $F: \mathbf{Hask} \to \mathbf{Hask}_f$
- 遗忘函子对应 $G: \mathbf{Hask}_f \to \mathbf{Hask}$
- $F \dashv G$ 构成伴随

### 4.2 List单子作为伴随

**List单子来自伴随**：

```haskell
-- List单子
instance Monad [] where
    return x = [x]
    xs >>= f = concat (fmap f xs)

-- List单子来自自由单子与遗忘函子的伴随
-- Free Monoid ⊣ Forgetful Functor
```

**数学原理**：

- List单子对应自由单子与遗忘函子的伴随
- `return` 对应伴随的单位
- `(>>=)` 对应伴随的余单位

### 4.3 伴随的Haskell表示

**伴随的Haskell类型**：

```haskell
-- 伴随的类型
class (Functor f, Functor g) => Adjunction f g where
    leftAdjoint :: (f a -> b) -> (a -> g b)
    rightAdjoint :: (a -> g b) -> (f a -> b)
```

**数学对应**：

- `leftAdjoint` 对应 $\text{Hom}(F(A), B) \to \text{Hom}(A, G(B))$
- `rightAdjoint` 对应 $\text{Hom}(A, G(B)) \to \text{Hom}(F(A), B)$
- 它们应该是互逆的
