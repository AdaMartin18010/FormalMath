## 一、函子的定义

### 1.1 函子的数学定义

**函子 (Functor)** $F: \mathcal{C} \to \mathcal{D}$ 由以下数据组成：

1. **对象映射**: 对每个对象 $A \in \mathcal{C}$，对象 $F(A) \in \mathcal{D}$
2. **映射映射**: 对每个映射 $f: A \to B$，映射 $F(f): F(A) \to F(B)$

**满足的条件**：

- **恒等保持**: $F(1_A) = 1_{F(A)}$
- **复合保持**: $F(g \circ f) = F(g) \circ F(f)$

### 1.2 函子的记号

**数学记号**：

- $F: \mathcal{C} \to \mathcal{D}$ 表示函子从范畴 $\mathcal{C}$ 到范畴 $\mathcal{D}$
- $F(A)$ 表示对象 $A$ 的像
- $F(f)$ 表示映射 $f$ 的像

**交换图**：
$$
\begin{CD}
A @>{f}>> B \\
@V{F}VV @VV{F}V \\
F(A) @>{F(f)}>> F(B)
\end{CD}
$$

## 二、Haskell中的函子

### 2.1 Functor类型类

**Functor类型类**定义：

```haskell
class Functor f where
    fmap :: (a -> b) -> f a -> f b
```

**数学对应**：

- `Functor f` 对应函子 $F: \mathbf{Hask} \to \mathbf{Hask}$
- `fmap` 对应映射映射 $F(f): F(A) \to F(B)$
- `f a` 对应对象 $F(A)$

### 2.2 函子定律

**Functor必须满足的定律**：

1. **恒等律**: `fmap id = id`
   - 数学表示：$F(1_A) = 1_{F(A)}$

2. **复合律**: `fmap (g . f) = fmap g . fmap f`
   - 数学表示：$F(g \circ f) = F(g) \circ F(f)$

**例子**：

```haskell
-- Maybe是函子
instance Functor Maybe where
    fmap f Nothing = Nothing
    fmap f (Just x) = Just (f x)

-- 验证恒等律
fmap id Nothing = Nothing = id Nothing
fmap id (Just x) = Just (id x) = Just x = id (Just x)

-- 验证复合律
fmap (g . f) (Just x) = Just ((g . f) x) = Just (g (f x))
(fmap g . fmap f) (Just x) = fmap g (Just (f x)) = Just (g (f x))
```

## 三、协变函子与反变函子

### 3.1 协变函子

**协变函子 (Covariant Functor)** 保持映射方向：

- 映射 $f: A \to B$ 映射到 $F(f): F(A) \to F(B)$
- 方向：$A \to B$ 保持为 $F(A) \to F(B)$

**Haskell中的协变函子**：

- `Maybe`, `[]`, `Either a` 都是协变函子
- `Functor` 类型类定义协变函子

### 3.2 反变函子

**反变函子 (Contravariant Functor)** 反转映射方向：

- 映射 $f: A \to B$ 映射到 $F(f): F(B) \to F(A)$
- 方向：$A \to B$ 反转为 $F(B) \to F(A)$

**Haskell中的反变函子**：

```haskell
class Contravariant f where
    contramap :: (b -> a) -> f a -> f b
```

**数学定义**：

- 反变函子 $F: \mathcal{C}^{\text{op}} \to \mathcal{D}$ 对应协变函子 $F: \mathcal{C} \to \mathcal{D}^{\text{op}}$
- 其中 $\mathcal{C}^{\text{op}}$ 是 $\mathcal{C}$ 的对偶范畴

**例子**：

```haskell
-- 函数类型 (-> r) 是反变函子
instance Contravariant ((->) r) where
    contramap f g = g . f
    -- contramap :: (b -> a) -> (a -> r) -> (b -> r)
```

## 四、函子的性质

### 4.1 忠实函子与满函子

**忠实函子 (Faithful Functor)**：

- 定义：对任意对象 $A, B$，映射 $\text{Hom}(A, B) \to \text{Hom}(F(A), F(B))$ 是单射
- 意义：不同的映射映射到不同的映射

**满函子 (Full Functor)**：

- 定义：对任意对象 $A, B$，映射 $\text{Hom}(A, B) \to \text{Hom}(F(A), F(B))$ 是满射
- 意义：目标范畴中的每个映射都来自源范畴

**完全忠实函子 (Fully Faithful Functor)**：

- 定义：既是忠实函子又是满函子
- 意义：$\text{Hom}(A, B) \cong \text{Hom}(F(A), F(B))$

### 4.2 函子的复合

**函子的复合**：

- 给定函子 $F: \mathcal{C} \to \mathcal{D}$ 和 $G: \mathcal{D} \to \mathcal{E}$，复合 $G \circ F: \mathcal{C} \to \mathcal{E}$ 也是函子
- $(G \circ F)(A) = G(F(A))$
- $(G \circ F)(f) = G(F(f))$

**Haskell中的函子复合**：

```haskell
-- 函子复合
newtype Compose f g a = Compose { getCompose :: f (g a) }

instance (Functor f, Functor g) => Functor (Compose f g) where
    fmap f (Compose x) = Compose (fmap (fmap f) x)
```
