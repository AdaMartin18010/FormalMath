## 概述

### 对齐目标

本文档旨在建立FormalMath项目与澳大利亚国立大学(Australian National University, ANU)数学系课程体系的全面对齐关系，识别内容覆盖情况，发现差距，并提出改进建议。

### ANU数学系简介

ANU数学系是澳大利亚顶尖数学研究机构之一，其课程设计具有以下特点：
- **严格的数学基础**：强调公理化方法和严格证明
- **荣誉路径课程(Honours Pathway Courses, HPC)**：为优秀学生提供更深层次的数学训练
- **理论与实践结合**：注重数学在现代科学和工程中的应用
- **渐进式难度**：从基础到高级的清晰学习路径

### 对齐方法

1. **课程大纲分析**：研究ANU官方课程描述和学习成果
2. **概念映射**：将ANU课程内容映射到FormalMath知识体系
3. **差距分析**：识别FormalMath未覆盖或覆盖不足的内容
4. **建议制定**：提出内容补充和改进的具体建议

## ANU数学课程体系

### 本科基础课程

#### MATH1115 - 高等数学与应用1 (Advanced Mathematics and Applications 1)

| 属性 | 内容 |
|------|------|
| **学分** | 6 units |
| **先修要求** | 无（但要求较强的数学基础） |
| **并行课程** | MATH1116（第二学期） |
| **核心内容** | 线性代数、微积分基础、数学证明入门 |

**详细主题列表**：
1. **线性代数**
   - 向量空间与线性变换
   - 矩阵理论与行列式
   - 特征值与特征向量
   - 内积空间与正交性

2. **单变量微积分**
   - 实数系统与完备性
   - 极限与连续性（ε-δ定义）
   - 微分学：导数定义与性质
   - 积分学：Riemann积分理论

3. **数学证明基础**
   - 逻辑推理与证明技巧
   - 数学归纳法
   - 反证法与构造性证明

**学习成果**：
- 解释实分析和线性代数的基本概念
- 准确高效地使用分析技术
- 通过分析、证明和解释展示数学推理能力
- 将问题解决技巧应用于物理学、工程学等情境

#### MATH1116 - 高等数学与应用2 (Advanced Mathematics and Applications 2)

| 属性 | 内容 |
|------|------|
| **学分** | 6 units |
| **先修要求** | MATH1115（60分以上） |
| **核心内容** | 多变量微积分、级数、常微分方程 |

**详细主题列表**：
1. **多变量微积分**
   - Rⁿ空间中的拓扑基础
   - 多元函数的极限与连续性
   - 偏导数与方向导数
   - 梯度、散度、旋度
   - 多重积分与变量替换

2. **级数理论**
   - 数项级数收敛判别法
   - 幂级数与Taylor展开
   - 一致收敛与函数项级数
   - Fourier级数基础

3. **常微分方程**
   - 一阶ODE解法
   - 高阶线性ODE
   - 级数解法
   - 存在唯一性定理

## 澳洲数学教育特点分析

### 1. 学习成果导向 (Learning Outcomes Focus)

澳洲高等教育强调**阈值学习成果(Threshold Learning Outcomes, TLO)**，数学学科的学习成果包括：

| 维度 | 描述 |
|------|------|
| **理解** | 展示对数学科学的连贯理解 |
| **知识** | 展示数学科学知识的深度和广度 |
| **探究与问题解决** | 使用数学和统计方法研究和解决问题 |
| **沟通** | 使用多种方式沟通数学和统计信息、论证或结果 |
| **责任** | 展示个人、专业和社会责任 |

### 2. 评估结构特点

| 课程级别 | 典型评估方式 | 比例 |
|----------|-------------|------|
| 基础课程 | 作业(Assignment) + 期末考试 | 30% + 70% |
| 高级课程 | 作业 + 期中考试 + 期末考试 | 30% + 30% + 40% |
| 荣誉课程 | 作业 + 论文 + 回家考试 | 30% + 20% + 50% |

### 3. Honours Pathway Courses (HPC) 特点

- **更严格的数学训练**：强调严格证明和抽象思维
- **更深的内容覆盖**：包含标准课程不覆盖的高级主题
- **替代评估方式**：包含论文、演讲等形式的评估
- **理论导向**：从抽象视角发展现代数学

### 4. 先修制度严格

- ANU数学课程有严格的先修要求和成绩门槛（通常要求先修课程达到60分以上）
- 确保学生具备足够的数学基础才能进入高级课程

### 5. 国际视野

- 课程内容与国际顶尖大学保持一致
- 参考国际权威教材（如Munkres、Rudin、Hungerford等）
- 强调数学的全球通用性
