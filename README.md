- 链接规范：
  - 同目录链接使用 `./文件名.md`
  - 上级目录使用 `../`，避免冗余前缀（例如在同目录内不要使用 `../当前目录名/文件`）
- 向量符号：文档统一使用 `\mathbf` 粗体向量表示。
- 术语与符号统一：对标国际标准，差异由抽检脚本标注并周度复核。

## 贡献指南（Documentation）

提交文档时请遵循：

- 链接规范：优先使用相对路径；同目录 `./`，上级 `../`；与 `docs/项目索引.md` 保持一致。
- 数学符号：向量统一使用 `\mathbf`；符号表见规范文档。
- 参考文献：优先补充 DOI/ISBN；高质量权威来源优先。
- 变更说明：在相关看板/周报中同步关键进展与指标变动。

如遇到链接问题或死链，请在 PR 描述附上“修复前/后”对照，便于自动化校验与回归。
