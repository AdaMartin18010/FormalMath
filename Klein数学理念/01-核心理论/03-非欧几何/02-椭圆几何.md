## 🔷 二、椭圆几何的基本概念

### 2.1 椭圆几何的定义

**定义**：

椭圆几何是满足椭圆平行公设的几何。

**椭圆平行公设**：

通过直线外一点，没有直线与给定直线平行。

### 2.2 椭圆平行公设

**公设**：

在椭圆几何中，任意两条直线都相交。

### 2.3 椭圆几何的性质

**性质1：无平行线**

在椭圆几何中，任意两条直线都相交，不存在平行线。

**性质2：三角形内角和>180°**

椭圆三角形的内角和：
$$\alpha + \beta + \gamma > 180° = \pi$$

**角度盈余**：
$$\epsilon = (\alpha + \beta + \gamma) - \pi > 0$$

**性质3：面积公式**

椭圆三角形面积与角度相关：
$$\text{Area} = R^2[(\alpha + \beta + \gamma) - \pi]$$

其中$R$是球面半径。

**性质4：直线长度有限**

椭圆几何中，直线（大圆）是闭合的：

- 周长：$2\pi R$
- 有限且无边界

**性质5：两点间有多条"直线"**

球面上对径点之间有无穷多条大圆（经线）。

**三种几何对比**：

| 性质 | 欧氏几何 | 双曲几何 | 椭圆几何 |
|------|---------|---------|---------|
| 平行线 | 唯一 | 无穷多 | 不存在 |
| 三角形内角和 | =180° | <180° | >180° |
| 直线 | 无限长 | 无限长 | 闭合有限 |
| 相似三角形 | 存在 | 不存在 | 不存在 |
| 曲率 | 0 | 负 | 正 |

## 📐 三、球面几何模型

### 3.1 球面模型

**球面模型定义**：

半径为$R$的球面$S^2 = \{(x,y,z) : x^2 + y^2 + z^2 = R^2\}$

**点**：球面上的点

**直线（测地线）**：大圆

- 与球心共面的圆
- 是球面上两点间的最短路径
- 例如：经线都是大圆

**距离**：
沿大圆的弧长：
$$d(P, Q) = R \cdot \theta$$

其中$\theta$是从球心看两点的夹角。

**注意事项**：

球面几何实际上是"双椭圆几何"：

- 对径点需要识别为同一点
- 真正的椭圆几何是$\mathbb{RP}^2$（实射影平面）

### 3.2 球面三角形

**球面三角形**：

由三条大圆弧围成的区域。

**球面三角公式**：

**余弦定理**：
$$\cos c = \cos a \cos b + \sin a \sin b \cos C$$

**正弦定理**：
$$\frac{\sin A}{\sin a} = \frac{\sin B}{\sin b} = \frac{\sin C}{\sin c}$$

**面积公式**（Girard定理）：
$$\text{Area} = R^2(\alpha + \beta + \gamma - \pi)$$

### 3.3 应用

**地理学**：

- 地球表面的几何
- 大圆航线（最短路径）
- 地图投影

**天文学**：

- 天球坐标
- 星座位置
- 天体运动
- **距离**：球面距离

### 3.2 模型的几何性质

**性质**：

- 保持角度
- 保持某些几何关系
- 几何直观

### 3.3 模型的应用

**应用**：

- 地球几何
- 球面三角学
- 现代应用

### 4.1 变换群视角

**椭圆几何的等距群**：

椭圆几何对应旋转群$SO(3)$（或射影版本$PSO(3)$）。

**群结构**：

- 三维李群
- 紧群
- 双覆盖：$SU(2) \to SO(3)$

### 4.2 三种几何的统一

**Gauss曲率统一**：

| 几何 | 曲率 | 变换群 |
|------|------|-------|
| 欧氏 | $\kappa = 0$ | $E(n)$ |
| 双曲 | $\kappa < 0$ | $PSL(2,\mathbb{R})$ |
| 椭圆 | $\kappa > 0$ | $SO(3)$ |

**Klein的统一视角**：

所有几何都是射影几何的子几何，通过保持特定二次曲线来区分。

## 🌟 扩展：椭圆几何的深入理解

### 为什么球面几何是椭圆几何？

**对踵点的等同**：

在真正的椭圆几何中，球面上的对踵点（antipodal points）被等同：
- 北极点 = 南极点
- $(x,y,z) = (-x,-y,-z)$

**结果**：
- 椭圆几何 = $S^2/\{\pm 1\}$ = $\mathbb{RP}^2$（实射影平面）
- 没有对踵点，真正的"点"
- 每两条线恰好交于一点

### 椭圆几何的模型对比

**球面模型**（易可视化）：
- 直观
- 但有对踵点的冗余
- 两条"直线"交于两点

**实射影平面**（真正的椭圆几何）：
- 抽象
- 每两条线恰好交一点
- 无边界、紧致、不可定向

**半球模型**：
- 取球面的上半球
- 赤道对径点等同
- 可视化椭圆几何

### 椭圆几何的奇异性质

**三角形内角和 > 180°**：

**公式**：
$$\alpha + \beta + \gamma = \pi + \frac{A}{R^2}$$

其中$A$是三角形面积，$R$是球面半径。

**例子**：
- 球面上的"等边三角形"（三个大圆弧）
- 每个角90°
- 内角和 = 270° > 180°

**不存在相似形**：

在椭圆几何中：
- **不存在不全等的相似三角形**
- 形状决定大小
- 与欧氏几何根本不同

**原因**：曲率为常数，没有尺度不变性。

### 椭圆几何的对称群

**SO(3)详解**：

椭圆几何的对称群是$SO(3)$：
$$SO(3) = \{R \in M_{3\times3} : R^TR = I, \det R = 1\}$$

**维数**：3维Lie群

**生成元**：绕三个坐标轴的旋转
- $R_x(\theta)$：绕$x$轴
- $R_y(\theta)$：绕$y$轴
- $R_z(\theta)$：绕$z$轴

**结构**：
- 紧致群
- 非交换（$R_xR_y \neq R_yR_x$）
- 每个元素对应球面上的旋转

**与射影群的关系**：
$$PSO(3) = SO(3)/\{\pm I\}$$

是$\mathbb{RP}^2$的对称群。

### 椭圆几何的应用

**地球测量学**（Geodesy）：

**大地线问题**：
- 地球表面两点间最短路径
- 沿大圆弧（椭圆几何的"直线"）
- 航空、航海路线规划

**例子**：北京→纽约
- 看似向北飞（接近北极）
- 实际是大圆航线
- 比"直线"（恒定纬度）更短

**天文学**：

**天球**：
- 恒星在天球（球面）上的位置
- 椭圆几何描述天球几何
- 球面三角学计算

**宇宙学**：
- 正曲率宇宙模型
- 封闭宇宙（有限但无边界）
- Einstein-de Sitter模型的一种

**计算机图形学**：

**球面插值**：
- 旋转插值（SLERP: Spherical Linear Interpolation）
- 计算机动画
- 机器人运动规划

### 三种几何的统一视角（Klein）

**Cayley-Klein度量**：

所有三种几何统一于射影几何：

| 几何 | 基本二次曲线 | 度量 |
|------|------------|------|
| 椭圆 | 虚二次曲线 | $\kappa > 0$ |
| 欧氏 | 无穷远直线 | $\kappa = 0$ |
| 双曲 | 实二次曲线 | $\kappa < 0$ |

**Klein的贡献**：
- 用射影几何统一三种几何
- 用二次曲线的类型区分
- 埃尔兰根纲领的实现
