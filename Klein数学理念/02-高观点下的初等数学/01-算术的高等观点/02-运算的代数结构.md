## 🔷 二、基本运算的代数结构

### 2.1 加法的群结构

**加法群（Additive Group）** / **Additive Gruppe**：

整数集合在加法运算下构成群。

**群的性质**：

- **封闭性**：两个整数相加仍为整数
- **结合律**：$(a+b)+c = a+(b+c)$
- **单位元**：$0$是加法单位元
- **逆元**：每个整数都有加法逆元

### 2.2 乘法的群结构

**乘法群（Multiplicative Group）** / **Multiplikative Gruppe**：

非零有理数在乘法运算下构成群。

**群的性质**：

- **封闭性**：两个非零有理数相乘仍为非零有理数
- **结合律**：$(ab)c = a(bc)$
- **单位元**：$1$是乘法单位元
- **逆元**：每个非零有理数都有乘法逆元

### 2.3 运算的结合性

**结合性（Associativity）** / **Assoziativität**：

运算满足结合律。

**意义**：

- 允许改变运算顺序
- 简化运算过程
- 是群结构的基础

## 📐 三、数系的代数结构

### 3.1 自然数的半群结构

**半群（Semigroup）** / **Halbgruppe**：

自然数在加法下构成半群。

**结构**：

- 有结合律
- 有单位元$0$
- 但没有逆元（不是群）

### 3.2 整数的群结构

**群（Group）** / **Gruppe**：

整数在加法下构成群。

**结构**：

- 有结合律
- 有单位元$0$
- 每个元素都有逆元

### 3.3 有理数的域结构

**域（Field）** / **Körper**：

有理数在加法和乘法下构成域。

**域的性质**：

- 加法构成交换群
- 乘法（除零外）构成交换群
- 分配律成立

## 🔗 四、运算律的深层意义

### 4.1 交换律的意义

**交换律（Commutativity）** / **Kommutativität**：

$a + b = b + a$，$ab = ba$（在相应数系中）。

**意义**：

- 允许改变运算顺序
- 简化运算
- 是交换群的基础

### 4.2 结合律的意义

**结合律（Associativity）** / **Assoziativität**：

$(a + b) + c = a + (b + c)$。

**意义**：

- 允许改变运算顺序
- 是群结构的基础
- 简化运算过程

### 4.3 分配律的意义

**分配律（Distributivity）** / **Distributivität**：

$a(b + c) = ab + ac$。

**意义**：

- 连接加法和乘法
- 是环和域的基础
- 简化运算过程
