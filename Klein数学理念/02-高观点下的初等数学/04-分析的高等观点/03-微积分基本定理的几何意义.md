## 🔷 二、微积分基本定理的表述

### 2.1 第一基本定理

**第一基本定理（First Fundamental Theorem）**：

如果$f$在$[a,b]$上连续，则函数
$$F(x) = \int_a^x f(t) dt$$
在$[a,b]$上可导，且$F'(x) = f(x)$。

**几何意义**：

- 面积函数的导数是原函数
- 积分和导数的关系

### 2.2 第二基本定理

**第二基本定理（Second Fundamental Theorem）**：

如果$F$是$f$的原函数，则
$$\int_a^b f(x) dx = F(b) - F(a)$$

**几何意义**：

- 定积分等于原函数在端点的差
- 面积的计算方法

### 2.3 定理的统一表述

**统一表述**：

微积分基本定理建立了微分和积分的互逆关系。

**关系**：

- 积分是微分的逆运算
- 微分是积分的逆运算
- 两者互为逆运算

## 🔬 扩展内容

### 几何直观的深化

**第一基本定理的几何意义**：

想象$F(x) = \int_a^x f(t) dt$是从$a$到$x$的"累积面积"：

- $F(x)$：曲线下从$a$到$x$的面积
- $F(x + h) - F(x)$：从$x$到$x+h$的窄条面积
- $\frac{F(x+h) - F(x)}{h} \approx f(x)$：窄条面积/宽度 ≈ 高度
- $F'(x) = f(x)$：面积增长率就是高度！

**第二基本定理的几何意义**：

$$\int_a^b f(x) dx = F(b) - F(a)$$

**直观理解**：

- 左边：曲线下总面积（累加无穷多小矩形）
- 右边：原函数的净变化
- **深刻洞察**：总面积=端点值之差！

### Stokes定理：终极推广

**微积分基本定理是1维Stokes定理**：

$$\int_a^b f'(x) dx = f(b) - f(a)$$

**推广**：

**2维（Green定理）**：
$$\iint_D \leqft(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) dA = \oint_{\partial D} (P dx + Q dy)$$

**3维（Gauss定理）**：
$$\iiint_V (\nabla \cdot \mathbf{F}) dV = \iint_{\partial V} \mathbf{F} \cdot d\mathbf{S}$$

**n维（Stokes定理）**：
$$\int_M d\omega = \int_{\partial M} \omega$$

**统一主题**："区域上的积分=边界上的积分"

### 教学案例

**案例**：计算$\int_0^{\pi} \sin x dx$

**传统方法**：

- 找原函数：$F(x) = -\cos x$
- 代入：$F(\pi) - F(0) = -\cos\pi - (-\cos 0) = 1 + 1 = 2$

**几何理解**：

- 正弦曲线下从0到π的面积是2
- 原函数是"累积面积函数"
- 端点值之差给出总面积

## 🌟 扩展：微积分基本定理的深层意义

### 微分形式的视角

**1-形式**：$\omega = f(x) dx$

**外微分**：$d\omega = \frac{\partial f}{\partial x} dx \wedge dx = 0$（1维）

**Stokes定理**（1维情况）：
$$\int_\gamma d\omega = \int_{\partial \gamma} \omega$$

**具体化**：

- 设$\omega = F'(x) dx$
- $d\omega = F''(x) dx \wedge dx = 0$
- $\int_a^b F'(x) dx = F(b) - F(a) = \int_{\partial [a,b]} F(x)$

**推广到高维**：

**2-形式**：$\omega = P dx + Q dy$
$$d\omega = \leqft(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) dx \wedge dy$$

**Green定理**：
$$\iint_D d\omega = \oint_{\partial D} \omega$$

### 向量场的视角

**梯度场**（保守场）：$\mathbf{F} = \nabla f$

**定理**：梯度场的路径积分只依赖端点

**数学表达**：
$$\int_C \nabla f \cdot d\mathbf{r} = f(B) - f(A)$$

**物理意义**：

- $f$是势能
- $\mathbf{F} = -\nabla f$是力
- 路径积分=功=势能变化

**例子**：重力场

- 势能：$U(h) = mgh$
- 力：$\mathbf{F} = -\nabla U = -mg\mathbf{k}$
- 功：$W = U(h_1) - U(h_2)$（只依赖高度差）

### 测度论视角

**Lebesgue积分**：

**定理**（Lebesgue形式的微积分基本定理）：

如果$F$绝对连续，则
$$F(b) - F(a) = \int_a^b F'(x) dx$$

其中$F'$几乎处处存在（Lebesgue意义）。

**推广**：

- 不要求$F$处处可微
- 允许可数个不可微点
- 只需"平均意义"下可微

### 信息论视角

**Shannon熵**：
$$H(X) = -\sum p_i \log p_i$$

**微分熵**（连续情况）：
$$h(X) = -\int f(x) \log f(x) dx$$

**关键**：积分（全局）←→密度函数（局部）

### 概率论视角

**累积分布函数**$F(x) = P(X \leq x)$
**概率密度函数**$f(x) = F'(x)$

**微积分基本定理**：
$$P(a < X \leq b) = \int_a^b f(x) dx = F(b) - F(a)$$

**意义**：概率（全局）可以从密度（局部）积分得到

### 教学中的高级案例

**案例1：为什么导数的原函数相差常数？**

**问题**：$F'(x) = G'(x)$ ⟹ $F(x) = G(x) + C$？

**拓扑证明**：

- 定义$H(x) = F(x) - G(x)$
- $H'(x) = 0$
- 连通区间上$H'(x) = 0$ ⟹ $H(x)$为常数
- 因此$F(x) = G(x) + C$

**案例2：变上限积分的导数**

**定理**：
$$\frac{d}{dx} \int_a^x f(t) dt = f(x)$$

**几何意义**：

- 左边：累积面积函数的变化率
- 右边：当前"高度"
- **直观**：面积的瞬时变化率=当前高度

**案例3：分部积分的几何意义**

**公式**：
$$\int_a^b u dv = [uv]_a^b - \int_a^b v du$$

**几何**：

- 考虑矩形$uv$
- 总面积变化=$u$变化×$v$ + $u$×$v$变化
- 积分 ⟹ 分部积分公式
