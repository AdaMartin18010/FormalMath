## 🔷 二、微积分基本定理的表述

### 2.1 第一基本定理

**第一基本定理（First Fundamental Theorem）**：

如果$f$在$[a,b]$上连续，则函数
$$F(x) = \int_a^x f(t) dt$$
在$[a,b]$上可导，且$F'(x) = f(x)$。

**几何意义**：

- 面积函数的导数是原函数
- 积分和导数的关系

### 2.2 第二基本定理

**第二基本定理（Second Fundamental Theorem）**：

如果$F$是$f$的原函数，则
$$\int_a^b f(x) dx = F(b) - F(a)$$

**几何意义**：

- 定积分等于原函数在端点的差
- 面积的计算方法

### 2.3 定理的统一表述

**统一表述**：

微积分基本定理建立了微分和积分的互逆关系。

**关系**：

- 积分是微分的逆运算
- 微分是积分的逆运算
- 两者互为逆运算

## 🔬 扩展内容

### 几何直观的深化

**第一基本定理的几何意义**：

想象$F(x) = \int_a^x f(t) dt$是从$a$到$x$的"累积面积"：

- $F(x)$：曲线下从$a$到$x$的面积
- $F(x + h) - F(x)$：从$x$到$x+h$的窄条面积
- $\frac{F(x+h) - F(x)}{h} \approx f(x)$：窄条面积/宽度 ≈ 高度
- $F'(x) = f(x)$：面积增长率就是高度！

**第二基本定理的几何意义**：

$$\int_a^b f(x) dx = F(b) - F(a)$$

**直观理解**：

- 左边：曲线下总面积（累加无穷多小矩形）
- 右边：原函数的净变化
- **深刻洞察**：总面积=端点值之差！

### Stokes定理：终极推广

**微积分基本定理是1维Stokes定理**：

$$\int_a^b f'(x) dx = f(b) - f(a)$$

**推广**：

**2维（Green定理）**：
$$\iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) dA = \oint_{\partial D} (P dx + Q dy)$$

**3维（Gauss定理）**：
$$\iiint_V (\nabla \cdot \mathbf{F}) dV = \iint_{\partial V} \mathbf{F} \cdot d\mathbf{S}$$

**n维（Stokes定理）**：
$$\int_M d\omega = \int_{\partial M} \omega$$

**统一主题**："区域上的积分=边界上的积分"

### 教学案例

**案例**：计算$\int_0^{\pi} \sin x dx$

**传统方法**：

- 找原函数：$F(x) = -\cos x$
- 代入：$F(\pi) - F(0) = -\cos\pi - (-\cos 0) = 1 + 1 = 2$

**几何理解**：

- 正弦曲线下从0到π的面积是2
- 原函数是"累积面积函数"
- 端点值之差给出总面积
