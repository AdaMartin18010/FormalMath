### 2.2 对齐方法

#### 2.2.1 内容对齐

**步骤1：识别关键概念**

- 提取文档中的核心概念、定理、方法
- 建立概念清单

**步骤2：权威检索**

- 在Wikipedia、nLab等权威来源检索对应条目
- 记录权威定义、表述、引用

**步骤3：对比分析**

- 对比项目内容与权威来源的差异
- 识别缺失、错误、过时内容

**步骤4：补充完善**

- 补充缺失内容
- 修正错误内容
- 更新过时内容
- 添加权威引用

#### 2.2.2 表述对齐

**标准**：

- 使用权威来源的标准术语
- 遵循权威来源的表述方式
- 引用权威来源的原始表述

**示例**：

- ❌ 错误："变换群是几何的本质"
- ✅ 正确："几何学可以定义为研究在给定变换群下保持不变性质的学科"（Klein, 1872）

#### 2.2.3 引用对齐

**引用格式**：

- 数学理论：作者(年份). 标题. 期刊/出版社.
- 网络资源：来源名称. 条目名称. URL. 访问日期.

**引用要求**：

- 每个核心概念至少1个权威引用
- 每个重要定理至少2个权威引用
- 每个现代应用至少1个最新研究引用（2020-2026）

### 2.3 对齐检查清单

**每个文档需检查**：

- [ ] **概念定义对齐**：核心概念是否与Wikipedia/nLab对齐？
- [ ] **术语使用对齐**：术语是否使用标准表述？
- [ ] **定理表述对齐**：定理是否与权威来源一致？
- [ ] **历史事实对齐**：历史事实是否与权威来源一致？

## 三、多维思维表征体系

### 3.1 思维表征类型

#### 3.1.1 多维主题相关概念子概念矩阵对比

**定义**：以矩阵形式展示主题、相关概念、子概念之间的关系和对比。

**格式**：

| 主题 | 相关概念1 | 相关概念2 | 相关概念3 | 子概念1.1 | 子概念1.2 | 子概念2.1 | ... |
|------|----------|----------|----------|----------|----------|----------|-----|
| **定义** | ... | ... | ... | ... | ... | ... | ... |
| **属性** | ... | ... | ... | ... | ... | ... | ... |
| **关系** | ... | ... | ... | ... | ... | ... | ... |
| **应用** | ... | ... | ... | ... | ... | ... | ... |

**示例**：几何变换矩阵（见`00-KLEIN数学理念全面梳理与多维表征系统.md` 3.2节）

**应用范围**：

- 所有核心概念（变换群、不变量、对称性等）
- 所有数系扩张（ℕ→ℤ→ℚ→ℝ→ℂ）
- 所有几何类型（欧氏、仿射、射影等）
- 所有教育理论（Piaget、Bruner、Vygotsky等）

#### 3.1.2 概念属性定义关系解释论证形式证明等树图

**定义**：以树状结构展示概念的属性、定义、关系、解释、论证、形式证明等。

**格式**：

```text
概念名称
├─ 定义
│  ├─ 形式定义
│  ├─ 直观定义
│  └─ 历史定义
├─ 属性
│  ├─ 基本性质
│  ├─ 运算性质
│  └─ 结构性质
├─ 关系
│  ├─ 与其他概念的关系
│  ├─ 包含关系
│  └─ 等价关系
├─ 解释
│  ├─ 几何解释
│  ├─ 代数解释
│  └─ 物理解释
├─ 论证
│  ├─ 存在性论证
│  ├─ 唯一性论证
│  └─ 重要性论证
└─ 形式证明
   ├─ 严格证明
   ├─ 构造性证明
   └─ 反证法证明
```

**应用范围**：

- 核心数学概念（群、变换、不变量等）
- 教育概念（高观点、螺旋式课程等）
- 认知概念（操作思维、最近发展区等）

#### 3.1.3 决策分析论证树图

**定义**：以树状结构展示决策过程、分析步骤、论证链条。

**格式**：

```text
决策问题
├─ 条件1？
│  ├─ 是 → 决策A
│  │     ├─ 理由1
│  │     ├─ 理由2
│  │     └─ 证据支持
│  └─ 否 → 继续判断
│        └─ 条件2？
│           ├─ 是 → 决策B
│           └─ 否 → 决策C
```

**应用范围**：

- 教学决策（如何讲授某个概念？）
- 问题求解（如何解决某个问题？）
- 方法选择（选择何种方法？）

**示例**：见`00-KLEIN数学理念全面梳理与多维表征系统.md` 6.1-6.2节

#### 3.1.4 公理定理推理树图

**定义**：以树状结构展示从公理到定理的推理过程。

**格式**：

```text
目标定理
├─ 前提1（公理/引理）
│  └─ 推理步骤1
├─ 前提2（公理/引理）
│  └─ 推理步骤2
├─ 前提3（已证定理）
│  └─ 推理步骤3
└─ 综合推理
   └─ 结论（目标定理）
```

**应用范围**：

- Klein纲领的证明
- 代数基本定理的证明
- Noether定理的证明
- 其他重要定理的证明

**示例**：见`00-KLEIN数学理念全面梳理与多维表征系统.md` 3.4节

#### 3.1.5 模型层次映射函数等树图

**定义**：以树状结构展示模型的层次结构、映射关系、函数关系等。

**格式**：

```text
顶层模型
├─ 层次1：抽象层
│  ├─ 概念1
│  │  └─ 映射到 → 层次2概念1.1
│  ├─ 概念2
│  │  └─ 映射到 → 层次2概念2.1
│  └─ 函数关系：f: 概念1 → 概念2
├─ 层次2：具体层
│  ├─ 概念1.1
│  │  └─ 映射到 → 层次3实例1.1.1
│  └─ 概念2.1
│     └─ 映射到 → 层次3实例2.1.1
└─ 层次3：实例层
   ├─ 实例1.1.1
   └─ 实例2.1.1
```

**应用范围**：

- 知识层次结构（抽象→具体）
- 教育层次结构（理论→实践）
- 数学结构层次（公理→定理→应用）

## 七、质量评估标准

### 7.1 评估维度

#### 7.1.1 权威对齐度（30%）

**评分标准**：

- **90-100分**：全面对齐，引用充分，术语标准

- **70-79分**：部分对齐，引用不足，术语部分标准
- **<70分**：对齐不足，引用缺乏，术语不标准

**检查项**：

- [ ] 核心概念是否与Wikipedia/nLab对齐？
- [ ] 术语是否使用标准表述？
- [ ] 是否包含足够的权威引用？
- [ ] 现代发展是否包含最新研究？

#### 7.1.2 思维表征度（25%）

**评分标准**：

- **90-100分**：多种表征，质量高，覆盖率≥90%
- **80-89分**：部分表征，质量一般，覆盖率≥70%
- **70-79分**：少量表征，质量一般，覆盖率≥50%
- **<70分**：缺少表征，质量低，覆盖率<50%

**检查项**：

- [ ] 是否包含多维矩阵？
- [ ] 是否包含概念树图？
- [ ] 是否包含决策树图？
- [ ] 是否包含推理树图？
- [ ] 是否包含层次树图？

#### 7.1.3 内容完整度（25%）

**评分标准**：

- **90-100分**：全面完整，深度足够，论证充分

- **70-79分**：部分完整，深度不足，论证部分充分
- **<70分**：不完整，深度不足，论证不足

**检查项**：

- [ ] 概述是否完整？
- [ ] 定义是否清晰？
- [ ] 性质是否全面？
- [ ] 应用是否丰富？
- [ ] 论证是否充分？

#### 7.1.4 现代性（20%）

**评分标准**：

- **90-100分**：最新研究（2020-2026），前沿应用，教育研究最新

- **<70分**：缺少新研究，应用不足，教育研究不足

**检查项**：

- [ ] 是否包含2020-2026最新研究？
