# 神经网络数学基础

## 前馈网络

### 感知机
$$y = \sigma(w^T x + b)$$

激活函数：
- Sigmoid：$\sigma(z) = \frac{1}{1+e^{-z}}$
- ReLU：$\sigma(z) = \max(0, z)$

### 反向传播
链式法则计算梯度：
$$\frac{\partial L}{\partial w} = \frac{\partial L}{\partial y} \frac{\partial y}{\partial z} \frac{\partial z}{\partial w}$$
