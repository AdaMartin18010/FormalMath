# 抽屉原理

## 描述

n+1个物体放入n个抽屉，至少有一个抽屉有≥2个物体。

## 可视化

```mermaid
graph TD
    A[n+1个物体] --> B[n个抽屉]
    B --> C[存在抽屉有≥2个]

    D[应用] --> E[生日问题]
    D --> F[拉姆齐理论]
end

```
