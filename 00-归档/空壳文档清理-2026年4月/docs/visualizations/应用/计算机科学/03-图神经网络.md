# 图神经网络

## 描述

GNN将神经网络推广到图结构数据，利用谱图理论进行消息传递。

## 可视化

```mermaid
graph TD
    A[图 G = (V,E)] --> B[邻接矩阵 A]
    B --> C[拉普拉斯 L = D - A]
    C --> D[图卷积]
    D --> E[消息传递]
end

```

## 参考

Kipf, T. N., & Welling, M. (2017). Semi-Supervised Classification with Graph Convolutional Networks.
