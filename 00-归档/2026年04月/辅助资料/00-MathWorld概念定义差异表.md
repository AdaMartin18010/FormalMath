## 1. 代数结构差异表

### 1.1 基础代数概念

| 概念 | MathWorld定义要点 | FormalMath定义要点 | 差异类型 | 建议措施 |
|-----|------------------|-------------------|---------|---------|
| **群 (Group)** | 四个性质：封闭性、结合律、单位元、逆元 | 相同四性质，强调代数结构 | 表述方式 | 增加Wolfram代码 |
| **环 (Ring)** | 加法阿贝尔群+乘法半群+分配律 | 相同，强调理想理论 | 侧重点 | 保持一致 |
| **域 (Field)** | 非零元有乘法逆元的交换环 | 相同，强调域扩张 | 侧重点 | 保持一致 |
| **理想 (Ideal)** | 子环，对环乘法封闭 | 相同，强调主理想整环 | 一致 | 无 |
| **模 (Module)** | 环上的向量空间推广 | 相同，强调模同态 | 一致 | 无 |

### 1.2 线性代数

| 概念 | MathWorld表示 | FormalMath表示 | 差异 | 优先级 |
|-----|--------------|---------------|------|-------|
| **矩阵** | 线性变换表示 | 代数结构 | 侧重点不同 | 低 |
| **行列式** | det(A) 或 \|A\| | det(A) 或 \|A\| | 一致 | 无 |
| **特征值** | λ满足Av=λv | 相同 | 一致 | 无 |
| **特征向量** | 非零解v | 非零解v | 一致 | 无 |
| **秩** | rank(A) | rank(A) | 一致 | 无 |

### 1.3 抽象代数进阶

| 概念 | MathWorld特点 | FormalMath特点 | 差距分析 |
|-----|--------------|---------------|---------|
| **群同态** | 有Wolfram实现 | 有Lean 4实现 | 互补 |
| **Sylow定理** | 标准表述 | 标准+形式化证明 | FormalMath更优 |
| **伽罗瓦理论** | 标准定义 | 历史+哲学分析 | FormalMath更详 |
