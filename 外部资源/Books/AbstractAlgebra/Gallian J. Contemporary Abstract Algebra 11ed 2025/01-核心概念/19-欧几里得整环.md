## 一、定义与例子

### 1.1 欧几里得整环的定义

**定义**: 整环 $D$ 称为**欧几里得整环 (Euclidean Domain, ED)**，如果存在从 $D$ 的非零元素到非负整数的函数 $d$（称为**测度 (Measure)**），使得：

1. 对于 $D$ 中的所有非零 $a, b$，有 $d(a) \leq d(ab)$；且
2. 如果 $a, b \in D, b \neq 0$，则存在 $D$ 中的元素 $q$ 和 $r$，使得 $a = bq + r$，其中 $r = 0$ 或 $d(r) < d(b)$。

- *Gallian原文*: "Definition Euclidean Domain (ED)" (Chapter 18)

### 1.2 欧几里得函数

函数 $d$ 称为**欧几里得函数 (Euclidean Function)** 或**测度函数 (Measure Function)**。它满足两个条件：

- **单调性**: $d(a) \leq d(ab)$ 对于所有非零 $a, b$
- **除法算法**: 对于任何 $a, b$（$b \neq 0$），存在 $q, r$ 使得 $a = bq + r$，其中 $r = 0$ 或 $d(r) < d(b)$

### 1.3 例子：整数环 $Z$

**例子 5**: 环 $Z$ 是欧几里得整环，其中 $d(a) = |a|$（$a$ 的绝对值）。

- *Gallian原文*: "EXAMPLE 5" (Chapter 18)

这里，除法算法就是整数的带余除法：对于任何整数 $a$ 和正整数 $b$，存在唯一的整数 $q$ 和 $r$，使得 $a = bq + r$，其中 $0 \leq r < b$。

### 1.4 例子：域上的多项式环 $F[x]$

**例子 6**: 设 $F$ 是域。则 $F[x]$ 是欧几里得整环，其中 $d(f(x)) = \deg f(x)$（见定理 16.2）。

- *Gallian原文*: "EXAMPLE 6" (Chapter 18)

这里，除法算法就是多项式的带余除法：对于任何多项式 $f(x)$ 和非零多项式 $g(x)$，存在唯一的多项式 $q(x)$ 和 $r(x)$，使得 $f(x) = g(x)q(x) + r(x)$，其中 $r(x) = 0$ 或 $\deg r(x) < \deg g(x)$。

### 1.5 例子：高斯整数环 $Z[i]$

**例子 7**: 高斯整数环 $Z[i] = \{a + bi \mid a, b \in Z\}$ 是欧几里得整环，其中 $d(a + bi) = a^2 + b^2$。与前两个例子不同，在这个例子中，函数 $d$ 并不明显满足必要条件。$d(x) \leq d(xy)$ 对于 $x, y \in Z[i]$ 直接从 $d(xy) = d(x)d(y)$ 这一事实得出（练习 7）。要验证条件 2 成立，观察到如果 $x, y \in Z[i]$ 且 $y \neq 0$，则 $xy^{-1} \in Q[i]$，$Z[i]$ 的商域（第 15 章练习 65）。设 $xy^{-1} = s + ti$，其中 $s, t \in Q$。现在设 $m$ 是最接近 $s$ 的整数，设 $n$ 是最接近 $t$ 的整数。（这些整数可能不是唯一确定的，但这无关紧要。）因此，$|m - s| \leq 1/2$ 且 $|n - t|\leq 1 / 2$。则
$$
x y ^ {- 1} = s + t i = (m - m + s) + (n - n + t) i = (m + n i) + [ (s - m) + (t - n) i ].
$$
所以，
$$
x = (m + n i) y + [ (s - m) + (t - n) i ] y.
$$
我们声称欧几里得整环定义的除法条件满足，其中 $q = m + ni$ 且
$$
r = [ (s - m) + (t - n) i ] y.
$$
显然，$q$ 属于 $Z[i]$，且由于 $r = x - qy$，$r$ 也属于 $Z[i]$。最后，
$$
\begin{array}{l} d (r) = d ([ (s - m) + (t - n) i ]) d (y) \\ = [ (s - m) ^ {2} + (t - n) ^ {2} ] d (y) \\ \leq \left(\frac {1}{4} + \frac {1}{4}\right) d (y) <   d (y). \\ \end{array}
$$

- *Gallian原文*: "EXAMPLE 7" (Chapter 18)

## 二、ED 蕴含 PID (定理 18.4)

### 2.1 定理内容

**定理 18.4**: 每个欧几里得整环都是主理想整环。

- *Gallian原文*: "Theorem 18.4 ED Implies PID" (Chapter 18)

### 2.2 证明思路

**证明**: 设 $D$ 是欧几里得整环，$I$ 是 $D$ 的非零理想。在所有 $I$ 的非零元素中，设 $a$ 使得 $d(a)$ 是最小的。则 $I = \langle a \rangle$。因为，如果 $b \in I$，存在元素 $q$ 和 $r$，使得 $b = aq + r$，其中 $r = 0$ 或 $d(r) < d(a)$。但 $r = b - aq \in I$，所以 $d(r)$ 不能小于 $d(a)$。因此，$r = 0$ 且 $b \in \langle a \rangle$。最后，零理想是 $\langle 0 \rangle$。

## 三、ED 蕴含 UFD (推论)

### 3.1 推论内容

**推论**: 每个欧几里得整环都是唯一分解整环。

- *Gallian原文*: "Corollary ED Implies UFD" (Chapter 18)

**证明**: 这直接来自定理 18.3 和 18.4。

### 3.2 关系总结

我们可以总结我们的定理和评论如下：
$$
\begin{array}{l} \mathrm {E D} \Rightarrow \mathrm {P I D} \Rightarrow \mathrm {U F D}; \\ \mathrm {U F D} \not \Rightarrow \mathrm {P I D} \not \Rightarrow \mathrm {E D}. \\ \end{array}
$$
（你可以通过按字母顺序列出类型来记住这些蕴含关系。）

换句话说：

- **ED $\Rightarrow$ PID**: 每个欧几里得整环都是主理想整环
- **PID $\Rightarrow$ UFD**: 每个主理想整环都是唯一分解整环
- **UFD $\not\Rightarrow$ PID**: 存在不是主理想整环的唯一分解整环（例如，$Z[x]$）
- **PID $\not\Rightarrow$ ED**: 存在不是欧几里得整环的主理想整环（Motzkin 的例子）

## 四、$Z$ 和 $F[x]$ 的相似性

### 4.1 相似性表格

例子 5 和 6 说明了环 $Z$ 和 $F[x]$ 之间的许多相似性之一。其他相似性总结在表 18.1 中。

| $Z$ | ↔ | $F[x]$ |
|-----|---|--------|
| 欧几里得整环: | | 欧几里得整环: |
| $d(a) = \|a\|$ | | $d(f(x)) = \deg f(x)$ |
| 单位: | | 单位: |
| $a$ 是单位当且仅当 $\|a\| = 1$ | | $f(x)$ 是单位当且仅当 $\deg f(x) = 0$ |
| 除法算法: | | 除法算法: |
| 对于 $a,b \in Z, b \neq 0$，存在 $q,r \in Z$ 使得 $a = bq + r, 0 \leq r < \|b\|$ | | 对于 $f(x), g(x) \in F[x], g(x) \neq 0$，存在 $q(x), r(x) \in F[x]$ 使得 $f(x) = g(x)q(x) + r(x), 0 \leq \deg r(x) < \deg g(x)$ 或 $r(x) = 0$ |
| PID: | | PID: |
| 每个非零理想 $I = \langle a \rangle$，其中 $a \neq 0$ 且 $\|a\|$ 最小 | | 每个非零理想 $I = \langle f(x) \rangle$，其中 $\deg f(x)$ 最小 |
| 素元: | | 不可约元: |
| 没有非平凡因子 | | 没有非平凡因子 |
| UFD: | | UFD: |
| 每个元素是素元的"唯一"乘积 | | 每个元素是不可约元的"唯一"乘积 |

- *Gallian原文*: "Table 18.1 Similarities Between $Z$ and $F[x]$" (Chapter 18)

### 4.2 类比的意义

这个类比表明，整数和域上的多项式在许多方面是相似的。这种相似性使得我们可以将整数的许多性质推广到多项式。

## 五、非 ED 的 PID

### 5.1 Motzkin 的例子

虽然不容易验证，但我们注意到存在不是欧几里得整环的主理想整环。第一个这样的例子是由 T. Motzkin 在 1949 年给出的。

- *Gallian原文*: "Although it is not easy to verify, we remark that there are principal ideal domains that are not Euclidean domains. The first such example was given by T. Motzkin in 1949." (Chapter 18)

### 5.2 重要性

这个例子表明，虽然每个欧几里得整环都是主理想整环，但逆命题并不成立。这强调了欧几里得整环是一个比主理想整环更强的条件。

## 六、数学意义与应用

### 6.1 计算的基础

欧几里得整环是计算的基础。欧几里得算法（用于计算最大公因子）可以在任何欧几里得整环中使用。

### 6.2 算法理论

欧几里得整环的性质使得我们可以开发高效的算法。这些算法在计算机科学和计算数论中很重要。

### 6.3 数论应用

欧几里得整环在数论中有许多应用。例如，高斯整数环 $Z[i]$ 用于研究平方和问题。

## 七、学习要点与难点

### 7.1 理解欧几里得函数

欧几里得函数 $d$ 必须满足两个条件：

1. **单调性**: $d(a) \leq d(ab)$ 对于所有非零 $a, b$
2. **除法算法**: 对于任何 $a, b$（$b \neq 0$），存在 $q, r$ 使得 $a = bq + r$，其中 $r = 0$ 或 $d(r) < d(b)$

理解这两个条件都很重要。第一个条件确保测度函数是"合理的"，第二个条件确保我们可以执行除法算法。

### 7.2 掌握除法算法

除法算法是欧几里得整环的核心。它允许我们计算最大公因子，这对于许多应用都很重要。理解如何在高斯整数环等更复杂的例子中执行除法算法很重要。
