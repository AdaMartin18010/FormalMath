## 一、群论概念关联

### 1.1 基础概念关联

```
群 (Group)
├── 子群 (Subgroup)
│   ├── 循环群 (Cyclic Group)
│   ├── 正规子群 (Normal Subgroup)
│   └── Sylow子群 (Sylow Subgroup)
├── 同构 (Isomorphism)
├── 同态 (Homomorphism)
│   └── 第一同构定理 (First Isomorphism Theorem)
└── 商群 (Factor Group)
    └── 陪集 (Coset)
```

### 1.2 群的结构关联

- **群** → **子群** → **循环群** → **生成元**
- **群** → **正规子群** → **商群**
- **群** → **同态** → **核** → **像**
- **群** → **同构** → **分类**

### 1.3 特殊群关联

- **循环群** ↔ **整数模n群** ($\mathbb{Z}_n$)
- **置换群** ($S_n$) ↔ **对称群**
- **二面体群** ($D_n$) ↔ **对称群**
- **有限单群** → **分类定理**

## 二、环论概念关联

### 2.1 基础概念关联

```
环 (Ring)
├── 整环 (Integral Domain)
│   └── 域 (Field)
├── 子环 (Subring)
├── 理想 (Ideal)
│   ├── 素理想 (Prime Ideal)
│   └── 极大理想 (Maximal Ideal)
├── 商环 (Factor Ring)
└── 环同态 (Ring Homomorphism)
```

### 2.2 环的结构关联

- **环** → **整环** → **域**
- **环** → **理想** → **商环**
- **环** → **多项式环** → **因式分解**
- **整环** → **唯一分解整环** → **欧几里得整环**

---

## 三、域论概念关联

### 3.1 基础概念关联

```
域 (Field)
├── 域扩张 (Field Extension)
│   ├── 代数扩张 (Algebraic Extension)
│   ├── 有限扩张 (Finite Extension)
│   └── 分裂域 (Splitting Field)
├── 有限域 (Finite Field)
└── 伽罗瓦理论 (Galois Theory)
    └── 分圆扩张 (Cyclotomic Extension)
```

### 3.2 域的结构关联

- **域** → **域扩张** → **代数扩张** → **有限扩张**
- **域** → **有限域** → **分类**
- **域扩张** → **伽罗瓦群** → **伽罗瓦理论**

## 四、跨主题关联

### 4.1 群与环的关联

- **群** ↔ **环的加法群**
- **子群** ↔ **理想**（在环中）
- **商群** ↔ **商环**
- **群同态** ↔ **环同态**

### 4.2 环与域的关联

- **环** → **整环** → **域**
- **理想** → **商环** → **域**（当理想是极大理想时）
- **多项式环** → **域扩张**

### 4.3 应用关联

- **群论** → **编码理论** → **线性码**
- **群论** → **密码学** → **RSA、椭圆曲线**
- **域论** → **几何构造** → **可作图性**
- **群论** → **对称性** → **晶体结构**

## 五、概念依赖关系

### 5.1 学习路径

**基础路径**：

1. 群 → 子群 → 循环群
2. 群 → 同构 → 同态 → 商群
3. 环 → 整环 → 域
4. 域 → 域扩张 → 伽罗瓦理论

**进阶路径**：

1. Sylow定理 → 有限单群分类
2. 唯一分解整环 → 欧几里得整环
3. 有限域 → 编码理论
4. 伽罗瓦理论 → 几何构造
