## 一、域扩张的构造

### 例题1：构造 $Q(\sqrt{2})$

**题目**: 构造域扩张 $Q(\sqrt{2})$ 并描述其元素。

**解答**：

**定义**: $Q(\sqrt{2})$ 是包含 $Q$ 和 $\sqrt{2}$ 的最小域。

**步骤1**: 由于 $Q(\sqrt{2})$ 是域，它必须包含所有形式为 $a + b\sqrt{2}$ 的元素，其中 $a, b \in Q$。

**步骤2**: 验证 $Q(\sqrt{2}) = \{a + b\sqrt{2} \mid a, b \in Q\}$ 是域：

- **加法**: $(a + b\sqrt{2}) + (c + d\sqrt{2}) = (a + c) + (b + d)\sqrt{2}$ ✅
- **乘法**: $(a + b\sqrt{2})(c + d\sqrt{2}) = (ac + 2bd) + (ad + bc)\sqrt{2}$ ✅
- **逆元**: 对于 $a + b\sqrt{2} \neq 0$，
  $$\frac{1}{a + b\sqrt{2}} = \frac{a - b\sqrt{2}}{a^2 - 2b^2} = \frac{a}{a^2 - 2b^2} - \frac{b}{a^2 - 2b^2}\sqrt{2}$$
  注意 $a^2 - 2b^2 \neq 0$（因为如果 $a^2 = 2b^2$，则 $\sqrt{2} = a/b \in Q$，矛盾）✅

**结论**: $Q(\sqrt{2}) = \{a + b\sqrt{2} \mid a, b \in Q\}$。

**扩张次数**: $[Q(\sqrt{2}):Q] = 2$，基为 $\{1, \sqrt{2}\}$。

### 例题2：构造 $Q(\sqrt[3]{2})$

**题目**: 构造域扩张 $Q(\sqrt[3]{2})$ 并描述其元素。

**解答**：

**步骤1**: $Q(\sqrt[3]{2})$ 是包含 $Q$ 和 $\sqrt[3]{2}$ 的最小域。

**步骤2**: 由于 $\sqrt[3]{2}$ 满足 $x^3 - 2 = 0$，$Q(\sqrt[3]{2})$ 的元素可以表示为：
$$Q(\sqrt[3]{2}) = \{a + b\sqrt[3]{2} + c(\sqrt[3]{2})^2 \mid a, b, c \in Q\}$$

**步骤3**: 验证这是域：

- **加法**: 显然封闭 ✅
- **乘法**: 使用关系 $(\sqrt[3]{2})^3 = 2$ 来简化乘积 ✅
- **逆元**: 对于 $a + b\sqrt[3]{2} + c(\sqrt[3]{2})^2 \neq 0$，可以使用线性代数方法找到逆元 ✅

**结论**: $Q(\sqrt[3]{2}) = \{a + b\sqrt[3]{2} + c(\sqrt[3]{2})^2 \mid a, b, c \in Q\}$。

**扩张次数**: $[Q(\sqrt[3]{2}):Q] = 3$，基为 $\{1, \sqrt[3]{2}, (\sqrt[3]{2})^2\}$。

### 例题3：构造 $Q(i)$

**题目**: 构造域扩张 $Q(i)$ 并描述其元素。

**解答**：

**步骤1**: $Q(i)$ 是包含 $Q$ 和 $i = \sqrt{-1}$ 的最小域。

**步骤2**: 由于 $i^2 = -1$，$Q(i)$ 的元素可以表示为：
$$Q(i) = \{a + bi \mid a, b \in Q\}$$

**步骤3**: 验证这是域：

- **加法**: $(a + bi) + (c + di) = (a + c) + (b + d)i$ ✅
- **乘法**: $(a + bi)(c + di) = (ac - bd) + (ad + bc)i$ ✅
- **逆元**: 对于 $a + bi \neq 0$，
  $$\frac{1}{a + bi} = \frac{a - bi}{a^2 + b^2} = \frac{a}{a^2 + b^2} - \frac{b}{a^2 + b^2}i$$
  注意 $a^2 + b^2 \neq 0$（因为 $a + bi \neq 0$）✅

**结论**: $Q(i) = \{a + bi \mid a, b \in Q\}$。

**扩张次数**: $[Q(i):Q] = 2$，基为 $\{1, i\}$。

**注意**: 这类似于例题1，但使用 $i$ 而不是 $\sqrt{2}$。

### 例题4：应用Kronecker定理构造域扩张

**题目**: 使用Kronecker定理构造 $Q$ 的扩张，使得 $x^2 + x + 1$ 在其中有一个零点。

**解答**：

**Kronecker定理**: 设 $F$ 是域，$f(x)$ 是 $F[x]$ 中的非常数多项式。则存在 $F$ 的扩张域 $E$，使得 $f(x)$ 在 $E$ 中有零点。

**步骤1**: 检查 $x^2 + x + 1$ 在 $Q$ 上是否不可约。

- 判别式：$\Delta = 1 - 4 = -3 < 0$
- 因此 $x^2 + x + 1$ 在 $Q$ 上不可约

**步骤2**: 构造 $E = Q[x] / \langle x^2 + x + 1 \rangle$。

**步骤3**: 根据定理19.3，$E \cong Q(a)$，其中 $a$ 是 $x^2 + x + 1$ 的零点。

**步骤4**: $E$ 的元素可以表示为 $c + da$，其中 $c, d \in Q$，且 $a^2 + a + 1 = 0$，即 $a^2 = -a - 1$。

**步骤5**: 乘法规则：
$$(c + da)(e + fa) = ce + (cf + de)a + df a^2 = ce + (cf + de)a + df(-a - 1) = (ce - df) + (cf + de - df)a$$

**结论**: $E = Q[x] / \langle x^2 + x + 1 \rangle \cong Q(a)$，其中 $a$ 满足 $a^2 + a + 1 = 0$。

**扩张次数**: $[E:Q] = 2$。

## 二、扩张次数

### 例题5：计算 $[Q(\sqrt{2}):Q]$

**题目**: 计算域扩张 $Q(\sqrt{2})$ 在 $Q$ 上的次数。

**解答**：

**方法1**: 使用基。

**步骤1**: $Q(\sqrt{2}) = \{a + b\sqrt{2} \mid a, b \in Q\}$。

**步骤2**: $\{1, \sqrt{2}\}$ 是 $Q(\sqrt{2})$ 在 $Q$ 上的基。

**步骤3**: 因此 $[Q(\sqrt{2}):Q] = 2$。

**方法2**: 使用最小多项式。

**步骤1**: $\sqrt{2}$ 在 $Q$ 上的最小多项式是 $x^2 - 2$。

**步骤2**: 根据定理19.3，$[Q(\sqrt{2}):Q] = \deg(x^2 - 2) = 2$。

**结论**: $[Q(\sqrt{2}):Q] = 2$。

---

### 例题6：计算 $[Q(\sqrt{2}, \sqrt{3}):Q]$

**题目**: 计算域扩张 $Q(\sqrt{2}, \sqrt{3})$ 在 $Q$ 上的次数。

**解答**：

**方法1**: 使用塔规则。

**步骤1**: 考虑塔 $Q \subseteq Q(\sqrt{2}) \subseteq Q(\sqrt{2}, \sqrt{3})$。

**步骤2**: $[Q(\sqrt{2}):Q] = 2$（从例题5）。

**步骤3**: 计算 $[Q(\sqrt{2}, \sqrt{3}):Q(\sqrt{2})]$。

- $\sqrt{3}$ 在 $Q(\sqrt{2})$ 上的最小多项式是 $x^2 - 3$（因为 $\sqrt{3} \notin Q(\sqrt{2})$）
- 因此 $[Q(\sqrt{2}, \sqrt{3}):Q(\sqrt{2})] = 2$

**步骤4**: 应用塔规则：
$$[Q(\sqrt{2}, \sqrt{3}):Q] = [Q(\sqrt{2}, \sqrt{3}):Q(\sqrt{2})][Q(\sqrt{2}):Q] = 2 \cdot 2 = 4$$

**方法2**: 直接找基。

**步骤1**: $Q(\sqrt{2}, \sqrt{3})$ 的元素可以表示为：
$$a + b\sqrt{2} + c\sqrt{3} + d\sqrt{6}$$
其中 $a, b, c, d \in Q$。

**步骤2**: 基为 $\{1, \sqrt{2}, \sqrt{3}, \sqrt{6}\}$。

**步骤3**: 因此 $[Q(\sqrt{2}, \sqrt{3}):Q] = 4$。

**结论**: $[Q(\sqrt{2}, \sqrt{3}):Q] = 4$。

### 例题7：应用塔规则

**题目**: 设 $F \subseteq E \subseteq K$ 是域扩张的塔。如果 $[K:E] = 3$ 且 $[E:F] = 4$，计算 $[K:F]$。

**解答**：

**塔规则**: 如果 $F \subseteq E \subseteq K$ 是域扩张的塔，则
$$[K:F] = [K:E][E:F]$$

**应用**:
$$[K:F] = [K:E][E:F] = 3 \cdot 4 = 12$$

**结论**: $[K:F] = 12$。

**一般形式**: 对于塔 $F_1 \subseteq F_2 \subseteq \cdots \subseteq F_n$，
$$[F_n:F_1] = [F_n:F_{n-1}][F_{n-1}:F_{n-2}] \cdots [F_2:F_1]$$

## 三、代数元与超越元

### 例题8：证明 $\sqrt{2}$ 在 $Q$ 上是代数的

**题目**: 证明 $\sqrt{2}$ 在 $Q$ 上是代数的。

**解答**：

**定义**: 元素 $a$ 在域 $F$ 上是代数的，如果存在 $F[x]$ 中的非零多项式 $f(x)$，使得 $f(a) = 0$。

**证明**:

**步骤1**: 考虑多项式 $f(x) = x^2 - 2 \in Q[x]$。

**步骤2**: $f(\sqrt{2}) = (\sqrt{2})^2 - 2 = 2 - 2 = 0$。

**步骤3**: 因此 $\sqrt{2}$ 是 $f(x)$ 的零点。

**结论**: $\sqrt{2}$ 在 $Q$ 上是代数的。

**最小多项式**: $\sqrt{2}$ 在 $Q$ 上的最小多项式是 $x^2 - 2$（因为它不可约且首一）。

---

### 例题9：找出 $\sqrt{2}$ 在 $Q$ 上的最小多项式

**题目**: 找出 $\sqrt{2}$ 在 $Q$ 上的最小多项式。

**解答**：

**步骤1**: 我们知道 $x^2 - 2$ 是 $\sqrt{2}$ 的零化多项式。

**步骤2**: 检查 $x^2 - 2$ 是否不可约。

- 如果 $x^2 - 2$ 可约，它可以分解为 $(x - a)(x - b) = x^2 - (a + b)x + ab$，其中 $a, b \in Q$
- 比较系数：$a + b = 0$ 且 $ab = -2$
- 因此 $a^2 = 2$，即 $a = \pm \sqrt{2} \notin Q$
- 矛盾！因此 $x^2 - 2$ 在 $Q$ 上不可约

**步骤3**: $x^2 - 2$ 是首一多项式。

**结论**: $x^2 - 2$ 是 $\sqrt{2}$ 在 $Q$ 上的最小多项式。

**一般方法**: 要找到 $a$ 在 $F$ 上的最小多项式，找到 $F[x]$ 中使得 $f(a) = 0$ 的最低次首一不可约多项式。

---

### 例题10：证明 $\pi$ 在 $Q$ 上是超越的

**题目**: 证明 $\pi$ 在 $Q$ 上是超越的（即不是代数的）。

**解答**：

**注意**: 这是一个非常困难的问题！$\pi$ 的超越性由 Lindemann 在 1882 年证明。

**证明思路**（简化版）:

**Lindemann-Weierstrass定理**: 如果 $\alpha_1, \ldots, \alpha_n$ 是代数数且线性无关于 $Q$，则 $e^{\alpha_1}, \ldots, e^{\alpha_n}$ 在 $Q$ 上代数无关。

**应用**:

- 如果 $\pi$ 是代数的，则 $i\pi$ 也是代数的
- 根据 Euler 公式，$e^{i\pi} = -1$ 是代数的
- 但根据 Lindemann-Weierstrass 定理，如果 $i\pi$ 是非零代数数，则 $e^{i\pi}$ 是超越的
- 矛盾！因此 $\pi$ 是超越的

**结论**: $\pi$ 在 $Q$ 上是超越的。

**历史**: 这是数学史上的一个重要结果，解决了"化圆为方"问题（使用尺规作图不可能）。

## 四、分裂域

### 例题11：找出 $x^2 - 2$ 在 $Q$ 上的分裂域

**题目**: 找出 $x^2 - 2$ 在 $Q$ 上的分裂域。

**解答**：

**步骤1**: $x^2 - 2$ 的零点是 $\sqrt{2}$ 和 $-\sqrt{2}$。

**步骤2**: 分裂域必须包含 $Q$ 和所有零点，即 $Q(\sqrt{2}, -\sqrt{2}) = Q(\sqrt{2})$。

**步骤3**: 验证 $x^2 - 2$ 在 $Q(\sqrt{2})$ 中分裂：
$$x^2 - 2 = (x - \sqrt{2})(x + \sqrt{2})$$

**结论**: $x^2 - 2$ 在 $Q$ 上的分裂域是 $Q(\sqrt{2})$。

**扩张次数**: $[Q(\sqrt{2}):Q] = 2$。

---

### 例题12：找出 $x^3 - 2$ 在 $Q$ 上的分裂域

**题目**: 找出 $x^3 - 2$ 在 $Q$ 上的分裂域。

**解答**：

**步骤1**: $x^3 - 2$ 的零点是 $\sqrt[3]{2}$，$\omega\sqrt[3]{2}$，$\omega^2\sqrt[3]{2}$，其中 $\omega = e^{2\pi i/3} = (-1 + i\sqrt{3})/2$ 是单位立方根。

**步骤2**: 分裂域必须包含 $Q$ 和所有零点，即 $Q(\sqrt[3]{2}, \omega)$。

**步骤3**: 计算扩张次数：

- $[Q(\sqrt[3]{2}):Q] = 3$（因为 $x^3 - 2$ 在 $Q$ 上不可约）
- $\omega$ 满足 $x^2 + x + 1 = 0$，因此 $[Q(\omega):Q] = 2$
- 由于 $\omega \notin Q(\sqrt[3]{2})$（因为 $Q(\sqrt[3]{2}) \subseteq \mathbb{R}$），我们有 $[Q(\sqrt[3]{2}, \omega):Q(\sqrt[3]{2})] = 2$
- 应用塔规则：$[Q(\sqrt[3]{2}, \omega):Q] = 3 \cdot 2 = 6$

**步骤4**: 验证 $x^3 - 2$ 在 $Q(\sqrt[3]{2}, \omega)$ 中分裂：
$$x^3 - 2 = (x - \sqrt[3]{2})(x - \omega\sqrt[3]{2})(x - \omega^2\sqrt[3]{2})$$

**结论**: $x^3 - 2$ 在 $Q$ 上的分裂域是 $Q(\sqrt[3]{2}, \omega)$，扩张次数为 $6$。

---

### 例题13：找出 $x^4 + 1$ 在 $Q$ 上的分裂域

**题目**: 找出 $x^4 + 1$ 在 $Q$ 上的分裂域。

**解答**：

**步骤1**: $x^4 + 1$ 的零点是 $e^{i\pi/4}$，$e^{3i\pi/4}$，$e^{5i\pi/4}$，$e^{7i\pi/4}$。

**步骤2**: 这些可以写成 $\pm(1 \pm i)/\sqrt{2}$。

**步骤3**: 分裂域是 $Q(i, \sqrt{2})$（因为需要 $i$ 和 $\sqrt{2}$）。

**步骤4**: 计算扩张次数：

- $[Q(\sqrt{2}):Q] = 2$
- $[Q(\sqrt{2}, i):Q(\sqrt{2})] = 2$（因为 $i \notin Q(\sqrt{2})$）
- 因此 $[Q(\sqrt{2}, i):Q] = 2 \cdot 2 = 4$

**步骤5**: 验证 $x^4 + 1$ 在 $Q(i, \sqrt{2})$ 中分裂。

**结论**: $x^4 + 1$ 在 $Q$ 上的分裂域是 $Q(i, \sqrt{2})$，扩张次数为 $4$。

## 五、有限域

### 例题14：构造 $\mathrm{GF}(4)$

**题目**: 构造4元素的有限域 $\mathrm{GF}(4)$。

**解答**：

**定理**: 每个有限域都有素幂阶，且在同构意义下唯一。

**步骤1**: 由于 $4 = 2^2$，$\mathrm{GF}(4)$ 是 $Z_2$ 的扩张。

**步骤2**: 找到 $Z_2[x]$ 中的2次不可约多项式。

- $x^2$ - 可约（$x \cdot x$）
- $x^2 + 1 = (x + 1)^2$ - 可约
- $x^2 + x = x(x + 1)$ - 可约
- $x^2 + x + 1$ - 不可约（在 $Z_2$ 中，$0^2 + 0 + 1 = 1 \neq 0$，$1^2 + 1 + 1 = 1 \neq 0$）

**步骤3**: 构造 $\mathrm{GF}(4) = Z_2[x] / \langle x^2 + x + 1 \rangle$。

**步骤4**: $\mathrm{GF}(4)$ 的元素：

- $0 = 0 + \langle x^2 + x + 1 \rangle$
- $1 = 1 + \langle x^2 + x + 1 \rangle$
- $a = x + \langle x^2 + x + 1 \rangle$
- $a + 1 = x + 1 + \langle x^2 + x + 1 \rangle$

**步骤5**: 乘法表（使用 $a^2 = a + 1$）：

- $a^2 = a + 1$
- $a^3 = a(a^2) = a(a + 1) = a^2 + a = (a + 1) + a = 1$

**结论**: $\mathrm{GF}(4) = \{0, 1, a, a + 1\}$，其中 $a^2 + a + 1 = 0$。

### 例题15：找出 $\mathrm{GF}(16)$ 的所有子域

**题目**: 找出16元素有限域 $\mathrm{GF}(16)$ 的所有子域。

**解答**：

**定理**: 对于 $n$ 的每个因子 $d$，$\mathrm{GF}(p^n)$ 有一个唯一的 $p^d$ 阶子域。

**步骤1**: $16 = 2^4$，因此 $p = 2$，$n = 4$。

**步骤2**: $n = 4$ 的因子是 $1, 2, 4$。

**步骤3**: 子域：

- $d = 1$: $\mathrm{GF}(2^1) = Z_2$（2元素）
- $d = 2$: $\mathrm{GF}(2^2) = \mathrm{GF}(4)$（4元素）
- $d = 4$: $\mathrm{GF}(2^4) = \mathrm{GF}(16)$（16元素，即自身）

**步骤4**: 验证包含关系：

- $Z_2 \subseteq \mathrm{GF}(4) \subseteq \mathrm{GF}(16)$

**结论**: $\mathrm{GF}(16)$ 的子域是 $Z_2$，$\mathrm{GF}(4)$，和 $\mathrm{GF}(16)$（自身）。

**一般结论**: $\mathrm{GF}(p^n)$ 的子域是 $\mathrm{GF}(p^d)$，其中 $d$ 整除 $n$。

---

### 例题16：找出 $\mathrm{GF}(9)$ 的乘法生成元

**题目**: 找出9元素有限域 $\mathrm{GF}(9)$ 的乘法生成元。

**解答**：

**定理**: 有限域的非零元素在乘法下形成循环群。

**步骤1**: $9 = 3^2$，因此 $\mathrm{GF}(9)$ 是 $Z_3$ 的扩张。

**步骤2**: 找到 $Z_3[x]$ 中的2次不可约多项式。

- $x^2 + 1$ - 检查：$0^2 + 1 = 1 \neq 0$，$1^2 + 1 = 2 \neq 0$，$2^2 + 1 = 5 \equiv 2 \pmod{3} \neq 0$
- 因此 $x^2 + 1$ 在 $Z_3$ 上不可约

**步骤3**: 构造 $\mathrm{GF}(9) = Z_3[x] / \langle x^2 + 1 \rangle$。

**步骤4**: $\mathrm{GF}(9)$ 的非零元素是 $\{1, a, a^2, a^3, a^4, a^5, a^6, a^7\}$，其中 $a = x + \langle x^2 + 1 \rangle$，$a^2 = -1 = 2$。

**步骤5**: 计算 $a$ 的幂：

- $a^1 = a$
- $a^2 = 2$
- $a^3 = 2a$
- $a^4 = 2a^2 = 2 \cdot 2 = 4 \equiv 1 \pmod{3}$

**步骤6**: 由于 $a^4 = 1$，$a$ 的阶是 $4$，不是生成元。

**步骤7**: 尝试 $a + 1$：

- $(a + 1)^2 = a^2 + 2a + 1 = 2 + 2a + 1 = 2a$
- $(a + 1)^4 = (2a)^2 = 4a^2 = 4 \cdot 2 = 8 \equiv 2 \pmod{3}$
- $(a + 1)^8 = 2^2 = 4 \equiv 1 \pmod{3}$

**步骤8**: 因此 $a + 1$ 的阶是 $8$，是 $\mathrm{GF}(9)^*$ 的生成元。

**结论**: $a + 1$ 是 $\mathrm{GF}(9)$ 的乘法生成元。

**一般方法**: 对于 $\mathrm{GF}(p^n)$，非零元素形成 $p^n - 1$ 阶循环群。生成元是阶为 $p^n - 1$ 的元素。

## 六、代数扩张的性质

### 例题17：证明有限扩张是代数扩张

**题目**: 证明如果 $E$ 是 $F$ 的有限扩张，则 $E$ 是 $F$ 的代数扩张。

**解答**：

**定理 20.4**: 如果 $E$ 是 $F$ 的有限扩张，则 $E$ 是 $F$ 的代数扩张。

**证明**:

**步骤1**: 设 $[E:F] = n$，$a \in E$。

**步骤2**: 考虑集合 $\{1, a, a^2, \ldots, a^n\}$。这个集合有 $n + 1$ 个元素。

**步骤3**: 由于 $E$ 在 $F$ 上的维数是 $n$，这 $n + 1$ 个元素在 $F$ 上线性相关。

**步骤4**: 因此存在不全为零的 $c_0, c_1, \ldots, c_n \in F$，使得
$$c_0 + c_1 a + c_2 a^2 + \cdots + c_n a^n = 0$$

**步骤5**: 设 $f(x) = c_0 + c_1 x + c_2 x^2 + \cdots + c_n x^n \in F[x]$。则 $f(a) = 0$。

**步骤6**: 因此 $a$ 在 $F$ 上是代数的。

**结论**: $E$ 的每个元素在 $F$ 上都是代数的，因此 $E$ 是 $F$ 的代数扩张。

**注意**: 逆命题不成立。存在代数扩张不是有限扩张。

### 例题18：证明代数扩张的传递性

**题目**: 证明如果 $K$ 是 $E$ 的代数扩张且 $E$ 是 $F$ 的代数扩张，则 $K$ 是 $F$ 的代数扩张。

**解答**：

**定理 20.7**: 如果 $K$ 是 $E$ 的代数扩张且 $E$ 是 $F$ 的代数扩张，则 $K$ 是 $F$ 的代数扩张。

**证明**:

**步骤1**: 设 $a \in K$。我们需要证明 $a$ 在 $F$ 上是代数的。

**步骤2**: 由于 $K$ 是 $E$ 的代数扩张，$a$ 在 $E$ 上是代数的。设 $a$ 在 $E$ 上的最小多项式为
$$p(x) = x^n + e_{n-1} x^{n-1} + \cdots + e_1 x + e_0$$
其中 $e_i \in E$。

**步骤3**: 考虑域 $F(e_0, e_1, \ldots, e_{n-1})$。由于 $E$ 是 $F$ 的代数扩张，每个 $e_i$ 在 $F$ 上都是代数的。

**步骤4**: 因此 $F(e_0, e_1, \ldots, e_{n-1})$ 是 $F$ 的有限扩张（因为它是有限个代数元的扩张）。

**步骤5**: $a$ 在 $F(e_0, e_1, \ldots, e_{n-1})$ 上是代数的（因为 $p(x) \in F(e_0, e_1, \ldots, e_{n-1})[x]$）。

**步骤6**: 因此 $F(e_0, e_1, \ldots, e_{n-1}, a)$ 是 $F(e_0, e_1, \ldots, e_{n-1})$ 的有限扩张。

**步骤7**: 应用塔规则，$F(e_0, e_1, \ldots, e_{n-1}, a)$ 是 $F$ 的有限扩张。

**步骤8**: 根据定理20.4，有限扩张是代数扩张，因此 $a$ 在 $F$ 上是代数的。

**结论**: $K$ 的每个元素在 $F$ 上都是代数的，因此 $K$ 是 $F$ 的代数扩张。

### 例题19：构造 $Q$ 的代数闭包

**题目**: 描述 $Q$ 的代数闭包的结构。

**解答**：

**定义**: 域 $F$ 的**代数闭包**是 $F$ 的代数闭的代数扩张。

**Steinitz定理**: 每个域都有一个唯一的（在同构意义下）代数闭包。

**步骤1**: $Q$ 的代数闭包记作 $\overline{Q}$。

**步骤2**: $\overline{Q}$ 由所有在 $Q$ 上是代数的复数组成。

**步骤3**: $\overline{Q}$ 的性质：

- $\overline{Q}$ 是 $Q$ 的代数扩张
- $\overline{Q}$ 是代数闭的（即 $\overline{Q}[x]$ 中的每个非常数多项式在 $\overline{Q}$ 中都有零点）
- $\overline{Q}$ 是 $C$ 的子域

**步骤4**: 例子：

- $\sqrt{2} \in \overline{Q}$（因为 $x^2 - 2$ 的零点）
- $\sqrt[3]{5} \in \overline{Q}$（因为 $x^3 - 5$ 的零点）
- $\pi \notin \overline{Q}$（因为 $\pi$ 是超越的）
- $e \notin \overline{Q}$（因为 $e$ 是超越的）

**步骤5**: $\overline{Q}$ 是 $C$ 的真子域（因为 $C$ 包含超越数如 $\pi$ 和 $e$）。

**结论**: $\overline{Q}$ 是所有代数数的集合，是 $Q$ 的代数闭包。

**注意**: $\overline{Q}$ 不是有限扩张，但它是代数扩张。
