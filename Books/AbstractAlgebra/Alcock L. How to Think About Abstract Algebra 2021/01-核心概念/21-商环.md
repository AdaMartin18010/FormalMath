## 一、定义

### 1.1 商环的定义

设 $R$ 是环，$I$ 是 $R$ 的理想。则 $R/I$ 是**商环 (Quotient Ring)**，其元素是陪集 $r + I$。

### 1.2 商环的运算

$$(r_1 + I) + (r_2 + I) = (r_1 + r_2) + I$$
$$(r_1 + I)(r_2 + I) = (r_1 r_2) + I$$

### 2.1 构造步骤

2. 计算所有陪集 $r + I$
3. 定义陪集之间的运算

## 三、典型例子

### 3.1 $\mathbb{Z}/n\mathbb{Z} \cong \mathbb{Z}_n$

整数环关于理想 $n\mathbb{Z}$ 的商环同构于模 $n$ 剩余类环。

### 3.2 多项式环的商环

$\mathbb{Z}[x]/(x)$ 同构于 $\mathbb{Z}$。

## 四、与商群的关系

商环的构造方法与商群类似，都通过正规子群/理想构造。
