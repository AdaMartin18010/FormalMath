## 一、梁理论例题

### 例题1：梁理论的范畴结构

**问题**：描述Bernoulli-Euler、Rayleigh和Timoshenko梁理论的范畴结构。

**解答**：

**假设集合**：

- **Bernoulli-Euler**：$\mathbf{Set}_{\text{B-E}} = \{\text{忽略转动惯量}, \text{忽略剪切变形}\}$
- **Rayleigh**：$\mathbf{Set}_{\text{R}} = \{\text{考虑转动惯量}, \text{忽略剪切变形}\}$
- **Timoshenko**：$\mathbf{Set}_{\text{T}} = \{\text{考虑转动惯量}, \text{考虑剪切变形}\}$

**复杂性关系**：

$$\mathbf{Set}_{\text{B-E}} \subset \mathbf{Set}_{\text{R}} \subset \mathbf{Set}_{\text{T}}$$

**范畴结构**：全序范畴 $\mathbf{Beam}$。

**数学表述**：

设 $\mathbf{Beam}$ 是梁理论范畴，对象是假设集合，态射是包含关系：

- 对象：$\mathbf{Set}_{\text{B-E}}, \mathbf{Set}_{\text{R}}, \mathbf{Set}_{\text{T}}$
- 态射：$\mathbf{Set}_{\text{B-E}} \hookrightarrow \mathbf{Set}_{\text{R}} \hookrightarrow \mathbf{Set}_{\text{T}}$

**控制方程**：

1. **Bernoulli-Euler梁**：
   $$EI \frac{\partial^4 w}{\partial x^4} = q(x)$$
   其中 $w$ 是横向位移，$EI$ 是弯曲刚度，$q$ 是分布载荷。

2. **Rayleigh梁**（考虑转动惯量）：
   $$EI \frac{\partial^4 w}{\partial x^4} + \rho I \frac{\partial^2 \theta}{\partial t^2} = q(x)$$
   其中 $\theta$ 是转角，$\rho I$ 是转动惯量。

3. **Timoshenko梁**（考虑转动惯量和剪切变形）：
   $$\begin{cases}
   EI \frac{\partial^2 \theta}{\partial x^2} + kGA\leqft(\frac{\partial w}{\partial x} - \theta\right) = \rho I \frac{\partial^2 \theta}{\partial t^2} \\
   kGA\leqft(\frac{\partial^2 w}{\partial x^2} - \frac{\partial \theta}{\partial x}\right) = \rho A \frac{\partial^2 w}{\partial t^2} + q(x)
   \end{cases}$$
   其中 $kGA$ 是剪切刚度，$A$ 是横截面积。

**范畴解释**：更复杂的模型包含更多假设，形成全序结构。

## 二、弹性理论例题

### 例题2：弹性模型的可转换性

**问题**：证明经典张量形式、Lamé方程和四元数形式的弹性模型是可转换的。

**解答**：

**假设集合**：所有三种形式都对应相同的假设集合：

$$\mathbf{Set}_{\text{Linear Elasticity}} = \{\text{线性应力-应变关系}, \text{小变形}, \text{均匀材料}, \text{各向同性}\}$$

**形式化映射**：

- $F: \mathbf{Set}_{\text{Linear Elasticity}} \to \text{Tensor Form}$
- $G: \mathbf{Set}_{\text{Linear Elasticity}} \to \text{Lamé Equations}$
- $H: \mathbf{Set}_{\text{Linear Elasticity}} \to \text{Quaternion Form}$

**1. 张量形式**：

$$\sigma_{ij} = \lambda \delta_{ij} \varepsilon_{kk} + 2\mu \varepsilon_{ij}$$

其中 $\lambda, \mu$ 是Lamé常数，$\sigma_{ij}$ 是应力张量，$\varepsilon_{ij}$ 是应变张量。

**2. Lamé方程形式**：

$$\mu \nabla^2 \mathbf{u} + (\lambda + \mu) \nabla(\nabla \cdot \mathbf{u}) + \mathbf{f} = 0$$

其中 $\mathbf{u}$ 是位移向量，$\mathbf{f}$ 是体积力。

**3. 四元数形式**：

使用四元数表示旋转和变形：

$$\mathbf{q} = q_0 + q_1 \mathbf{i} + q_2 \mathbf{j} + q_3 \mathbf{k}$$

弹性关系用四元数运算表示。

**自然变换**：存在自然变换 $\alpha: F \Rightarrow G$ 和 $\beta: G \Rightarrow H$，使得：

- $\alpha$ 将张量形式转换为Lamé方程形式
- $\beta$ 将Lamé方程形式转换为四元数形式

**转换关系**：

- **张量 → Lamé**：通过平衡方程 $\sigma_{ij,j} + f_i = 0$ 和几何关系 $\varepsilon_{ij} = \frac{1}{2}(u_{i,j} + u_{j,i})$ 转换
- **Lamé → 四元数**：通过四元数表示旋转和变形转换

**结论**：这三种模型表述是可转换的，具有相同的复杂性（相同的假设集合），只是数学表示形式不同。

## 三、耦合模型例题

### 例题3：结构-流体耦合

**问题**：描述结构模型和流体模型的耦合，并给出范畴结构。

**解答**：

**结构模型**：来自 $\mathbf{Structure}$ 范畴

**平衡方程**：
$$\sigma_{ij,j} + f_i = 0$$

**本构关系**（线性弹性）：
$$\sigma_{ij} = C_{ijkl} \varepsilon_{kl}$$

**几何关系**：
$$\varepsilon_{ij} = \frac{1}{2}(u_{i,j} + u_{j,i})$$

其中 $u_i$ 是位移，$\sigma_{ij}$ 是应力，$\varepsilon_{ij}$ 是应变，$C_{ijkl}$ 是弹性张量。

**流体模型**：来自 $\mathbf{Fluid}$ 范畴

**Navier-Stokes方程**：
$$\rho \frac{\partial v_i}{\partial t} + \rho v_j v_{i,j} = -p_{,i} + \mu v_{i,jj} + f_i$$

**连续性方程**：
$$\frac{\partial \rho}{\partial t} + (\rho v_i)_{,i} = 0$$

其中 $v_i$ 是速度，$p$ 是压力，$\mu$ 是动力粘度，$\rho$ 是密度。

**耦合条件**（在界面 $\Gamma$ 上）：

1. **运动学条件**（无滑移）：
   $$v_i = \frac{\partial u_i}{\partial t} \quad \text{在 } \Gamma \text{ 上}$$

2. **动力学条件**（力平衡）：
   $$\sigma_{ij} n_j = -p n_i + \tau_{ij} n_j \quad \text{在 } \Gamma \text{ 上}$$
   其中 $n_j$ 是界面法向量，$\tau_{ij}$ 是粘性应力。

**范畴结构**：

使用函子范畴 $\text{Funct}(\mathbf{Structure}, \mathbf{Fluid})$ 描述耦合：

- **对象**：函子 $F: \mathbf{Structure} \to \mathbf{Fluid}$，将结构模型映射到流体模型
- **态射**：自然变换，表示不同的耦合方式

**耦合函子**：

$$F: \mathbf{Structure} \to \mathbf{Fluid}$$

- $F(\text{结构域}) = \text{流体域}$
- $F(\text{结构方程}) = \text{耦合方程}$
- $F(\text{结构边界条件}) = \text{耦合边界条件}$

**自然变换**：$\alpha: F \Rightarrow G$ 表示不同的耦合策略（强耦合、弱耦合、分区耦合等）。

**数学表述**：

耦合系统可以表示为：

$$\begin{cases}
\text{结构方程} & \text{在 } \Omega_s \text{ 上} \\
\text{流体方程} & \text{在 } \Omega_f \text{ 上} \\
\text{耦合条件} & \text{在 } \Gamma = \partial \Omega_s \cap \partial \Omega_f \text{ 上}
\end{cases}$$

其中 $\Omega_s$ 是结构域，$\Omega_f$ 是流体域，$\Gamma$ 是耦合界面。

**范畴解释**：耦合模型通过函子将两个不同范畴的模型连接起来，自然变换表示不同的耦合方法。
