## 一、数制系统

### 1.1 十进制

**十进制 (Base-10)**：使用10个符号（$0, 1, 2, \ldots, 9$）表示数字。

**位置系统**：

- **个位**：$10^0 = 1$
- **十位**：$10^1 = 10$
- **百位**：$10^2 = 100$
- **千位**：$10^3 = 1,000$
- $\ldots$

**例子**：$1,011$（十进制）

$$1,011 = 1 \times 10^3 + 0 \times 10^2 + 1 \times 10^1 + 1 \times 10^0$$

$$= 1,000 + 0 + 10 + 1 = 1,011$$

### 1.2 其他进制

**二进制 (Base-2)**：使用2个符号（$0, 1$）表示数字。

**八进制 (Base-8)**：使用8个符号（$0, 1, 2, \ldots, 7$）表示数字。

**十六进制 (Base-16)**：使用16个符号（$0, 1, 2, \ldots, 9, A, B, C, D, E, F$）表示数字。

**一般进制**：Base-$b$ 使用 $b$ 个符号。

## 二、二进制的定义

### 2.1 二进制表示

**二进制 (Binary / Base-2)**：使用2个符号（$0$ 和 $1$）表示数字。

**位置系统**：

- **第0位**：$2^0 = 1$
- **第1位**：$2^1 = 2$
- **第2位**：$2^2 = 4$
- **第3位**：$2^3 = 8$
- **第4位**：$2^4 = 16$
- $\ldots$

**例子**：$1011$（二进制）

$$1011_2 = 1 \times 2^3 + 0 \times 2^2 + 1 \times 2^1 + 1 \times 2^0$$

$$= 8 + 0 + 2 + 1 = 11_{10}$$

### 2.2 二进制数字

**二进制数字 (Binary Digits / Bits)**：

- $0$：表示"关"或"否"
- $1$：表示"开"或"是"

**例子**：

- $10_2$（二进制）= $2_{10}$（十进制）
- 因此"10只猫"（二进制）= "2只猫"（十进制）
- 这是唯一一次说"10只猫不如2只猫"的情况！

### 2.3 二进制计数

**前几个二进制数**：

| 十进制 | 二进制 |

| 1      | 1      |

## 三、进制转换

### 3.1 二进制转十进制

**方法**：按位展开并求和。

**例子**：$1011_2$ 转十进制

$$1011_2 = 1 \times 2^3 + 0 \times 2^2 + 1 \times 2^1 + 1 \times 2^0$$

$$= 8 + 0 + 2 + 1 = 11_{10}$$

**一般公式**：

$$d_n d_{n-1} \ldots d_1 d_0_2 = \sum_{i=0}^n d_i \times 2^i$$

### 3.2 十进制转二进制

**方法1：除2取余法**

1. 将十进制数除以2，记录余数
2. 将商继续除以2，记录余数
3. 重复直到商为0
4. 将余数从下往上排列

**例子**：$11_{10}$ 转二进制

- $11 \div 2 = 5$ 余 $1$
- $5 \div 2 = 2$ 余 $1$
- $2 \div 2 = 1$ 余 $0$
- $1 \div 2 = 0$ 余 $1$

因此 $11_{10} = 1011_2$。

**方法2：减幂法**

1. 找到小于等于该数的最大2的幂
2. 减去该幂，标记对应位为1
3. 重复直到数为0

**例子**：$11_{10}$ 转二进制

- $8 = 2^3 \leq 11$，$11 - 8 = 3$，第3位为1
- $2 = 2^1 \leq 3$，$3 - 2 = 1$，第1位为1
- $1 = 2^0 \leq 1$，$1 - 1 = 0$，第0位为1
- 其他位为0

因此 $11_{10} = 1011_2$。

### 3.3 一般进制转换

**Base-$b$ 转十进制**：

$$d_n d_{n-1} \ldots d_1 d_0_b = \sum_{i=0}^n d_i \times b^i$$

**十进制转 Base-$b$**：

使用除 $b$ 取余法或减幂法。

## 四、应用

### 4.1 计算机科学

**数字电路**：

- **开关**：$0$ = 关，$1$ = 开
- **逻辑门**：AND、OR、NOT等
- **处理器**：所有计算都基于二进制

**数据存储**：

- **位 (Bit)**：单个二进制数字
- **字节 (Byte)**：8位 = 1字节
- **文件**：所有数字文件都以二进制形式存储

**国际电工委员会符号**：

- **开**：符号像 $1$
- **关**：符号像 $0$
- **电源按钮**：这两个符号的组合

### 4.2 通信

**数字通信**：

- **编码**：将信息转换为二进制
- **传输**：通过二进制信号传输
- **解码**：将二进制转换回信息

**错误检测与纠正**：

- **奇偶校验**：检测单个位错误
- **校验和**：检测多个位错误
- **纠错码**：纠正错误

### 4.3 数学应用

**组合数学**：

- **子集**：可以用二进制表示
- **幂集**：$n$ 个元素的集合有 $2^n$ 个子集
- **布尔代数**：基于二进制逻辑

**算法**：

- **位运算**：高效的数值运算
- **哈希函数**：使用二进制操作
- **加密**：基于二进制运算

### 4.4 日常应用

**时间**：

- **二进制时钟**：用二进制显示时间
- **倒计时**：使用二进制表示

**游戏**：

- **猜数字游戏**：使用二进制搜索
- **益智游戏**：涉及二进制逻辑

**教育**：

- **数学教学**：理解数制系统
- **计算机科学入门**：理解计算机基础
