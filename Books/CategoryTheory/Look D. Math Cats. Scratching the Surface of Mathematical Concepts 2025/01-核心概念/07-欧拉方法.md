## 一、微分方程背景

### 1.1 微分方程

**微分方程 (Differential Equation)**：包含未知函数及其导数的方程。

**一阶常微分方程**：

$$\frac{dy}{dx} = f(x, y)$$

其中 $f(x, y)$ 是已知函数。

**初值问题 (Initial Value Problem)**：

$$\frac{dy}{dx} = f(x, y), \quad y(x_0) = y_0$$

**例子**：

- **指数增长**：$\frac{dy}{dx} = ky$，$y(0) = y_0$
- **逻辑增长**：$\frac{dy}{dx} = ky(1 - \frac{y}{M})$，$y(0) = y_0$
- **抛体运动**：$\frac{d^2y}{dt^2} = -g$（二阶方程）

### 1.2 解析解与数值解

**解析解 (Analytical Solution)**：用初等函数表示的解。

**例子**：

- $\frac{dy}{dx} = ky$ 的解：$y(x) = y_0 e^{kx}$

**数值解 (Numerical Solution)**：通过数值方法得到的近似解。

**问题**：大多数微分方程没有解析解，需要使用数值方法。

### 1.3 数值方法的重要性

**应用场景**：

- **物理建模**：预测物体运动轨迹
- **工程计算**：模拟系统行为
- **生物建模**：预测种群增长
- **经济建模**：预测市场变化

**例子**：猫的跳跃轨迹

- 已知初始速度和角度
- 需要预测着陆位置
- 使用数值方法近似轨迹

## 二、欧拉方法的算法

### 2.1 基本思想

**欧拉方法 (Euler's Method)**：用直线段近似曲线，逐步计算函数值。

**几何直观**：

- 在点 $(x_0, y_0)$ 处，切线斜率为 $f(x_0, y_0)$
- 沿切线方向前进一小步 $h$（步长）
- 得到下一个近似点 $(x_1, y_1)$
- 重复此过程

### 2.2 算法步骤

**算法描述**：

1. **输入**：$f(x, y)$，初值 $(x_0, y_0)$，步长 $h$，终点 $x_n$
2. **初始化**：$x = x_0$，$y = y_0$
3. **迭代**：对 $i = 0, 1, 2, \ldots, n-1$：
   - $y_{i+1} = y_i + h \cdot f(x_i, y_i)$
   - $x_{i+1} = x_i + h$
4. **输出**：近似解 $(x_i, y_i)$，$i = 0, 1, \ldots, n$

**数学表述**：

$$y_{i+1} = y_i + h \cdot f(x_i, y_i)$$

$$x_{i+1} = x_i + h$$

### 2.3 例子

**例子**：$\frac{dy}{dx} = y$，$y(0) = 1$，$h = 0.1$

**解析解**：$y(x) = e^x$

**数值解**：

- $x_0 = 0$，$y_0 = 1$
- $x_1 = 0.1$，$y_1 = 1 + 0.1 \times 1 = 1.1$
- $x_2 = 0.2$，$y_2 = 1.1 + 0.1 \times 1.1 = 1.21$
- $x_3 = 0.3$，$y_3 = 1.21 + 0.1 \times 1.21 = 1.331$
- $\ldots$

**比较**：

- $y(0.3) = e^{0.3} \approx 1.3499$
- 欧拉方法：$y_3 = 1.331$
- 误差：约 $0.0189$

## 三、误差分析

### 3.1 局部截断误差

**局部截断误差 (Local Truncation Error)**：单步的误差。

**泰勒展开**：

$$y(x_{i+1}) = y(x_i) + h \cdot y'(x_i) + \frac{h^2}{2} y''(\xi_i)$$

其中 $\xi_i \in (x_i, x_{i+1})$。

**欧拉方法**：

$$y_{i+1} = y_i + h \cdot f(x_i, y_i) = y_i + h \cdot y'(x_i)$$

**局部截断误差**：

$$e_i = y(x_{i+1}) - y_{i+1} = \frac{h^2}{2} y''(\xi_i) = O(h^2)$$

### 3.2 全局误差

**全局误差 (Global Error)**：累积误差。

**估计**：

如果 $|y''(x)| \leq M$ 在区间上，则全局误差：

$$E_n \leq \frac{Mh}{2} (x_n - x_0) = O(h)$$

**结论**：欧拉方法是一阶方法（全局误差与 $h$ 成正比）。

### 3.3 步长的影响

**步长越小，误差越小**：

- $h = 0.1$：误差较大
- $h = 0.01$：误差较小
- $h = 0.001$：误差更小

**权衡**：

- 步长小：计算量大，但精度高
- 步长大：计算量小，但精度低

**例子**：猫的跳跃

- 步长太大：预测不准确，可能错过目标
- 步长太小：计算时间长，但预测准确

## 四、应用与改进

### 4.1 实际应用

**物理建模**：

- **抛体运动**：预测物体轨迹
- **弹簧振动**：模拟振动系统
- **电路分析**：分析电路行为

**生物建模**：

- **种群增长**：预测种群数量
- **疾病传播**：模拟疫情发展
- **化学反应**：模拟反应过程

**工程应用**：

- **结构分析**：分析结构响应
- **流体力学**：模拟流体流动
- **控制系统**：设计控制系统

### 4.2 改进方法

**改进的欧拉方法 (Improved Euler's Method)**：

$$y_{i+1} = y_i + \frac{h}{2}[f(x_i, y_i) + f(x_{i+1}, y_i + h \cdot f(x_i, y_i))]$$

**龙格-库塔方法 (Runge-Kutta Method)**：

四阶龙格-库塔方法（RK4）：

$$k_1 = h \cdot f(x_i, y_i)$$

$$k_2 = h \cdot f(x_i + \frac{h}{2}, y_i + \frac{k_1}{2})$$

$$k_3 = h \cdot f(x_i + \frac{h}{2}, y_i + \frac{k_2}{2})$$

$$k_4 = h \cdot f(x_i + h, y_i + k_3)$$

$$y_{i+1} = y_i + \frac{1}{6}(k_1 + 2k_2 + 2k_3 + k_4)$$

**优势**：RK4是四阶方法，精度更高。

### 4.3 历史背景

**欧拉 (Leonhard Euler, 1707-1783)**：

- 瑞士数学家
- 在《积分学原理》（Institutionum calculi integralis，1768-1770）中提出欧拉方法
- 对数学的贡献：决定使用 $\pi$ 表示圆周率

**轶事**：

- 1738年失去右眼视力后说："现在我会少一些分心。"
- 展现了数学家的专注和奉献精神

### 4.4 现代发展

**自适应方法**：

- 根据误差自动调整步长
- 在误差大的地方使用小步长
- 在误差小的地方使用大步长

**并行计算**：

- 利用并行计算加速数值求解
- 适用于大规模问题

**软件工具**：

- MATLAB、Python（SciPy）、Mathematica等
- 提供高效的数值求解器
