## 一、埃及分数的定义

### 1.1 基本定义

**埃及分数 (Egyptian Fraction)**：分子为1的单位分数的和。

**数学表述**：

$$\frac{a}{b} = \frac{1}{n_1} + \frac{1}{n_2} + \cdots + \frac{1}{n_k}$$

其中 $n_1, n_2, \ldots, n_k$ 是互不相同的正整数。

**例子**：

- $\frac{1}{2}$ 是埃及分数
- $\frac{1}{50}$ 是埃及分数
- $\frac{1}{13} + \frac{1}{22} + \frac{1}{213}$ 是埃及分数
- $\frac{2}{3}$ 不是埃及分数，但可以表示为 $\frac{1}{2} + \frac{1}{6}$

### 1.2 与普通分数的区别

**普通分数 (Vulgar Fraction)**：形如 $\frac{a}{b}$ 的分数，其中 $a$ 和 $b$ 是整数，$b \neq 0$。

**埃及分数表示**：将普通分数表示为单位分数的和。

**例子**：

- $\frac{2}{3} = \frac{1}{2} + \frac{1}{6}$
- $\frac{5}{22} = \frac{1}{5} + \frac{1}{37} + \frac{1}{4070}$
- $\frac{5}{6} = \frac{1}{2} + \frac{1}{3}$

### 1.3 存在性定理

**定理**：任何正有理数都可以表示为有限个互不相同的单位分数的和。

**证明思路**：

使用贪心算法（见下文）可以构造出表示。

## 二、埃及分数的表示

### 2.1 表示方法

**方法一：贪心算法**

对分数 $\frac{a}{b}$（$0 < a < b$），找到最小的 $n$ 使得 $\frac{1}{n} \leq \frac{a}{b} < \frac{1}{n-1}$，然后递归处理 $\frac{a}{b} - \frac{1}{n}$。

**例子**：$\frac{5}{6}$

- $\frac{1}{2} \leq \frac{5}{6}$，取 $\frac{1}{2}$
- $\frac{5}{6} - \frac{1}{2} = \frac{5-3}{6} = \frac{2}{6} = \frac{1}{3}$
- 因此 $\frac{5}{6} = \frac{1}{2} + \frac{1}{3}$

**方法二：分数分解**

利用恒等式：

$$\frac{1}{n} = \frac{1}{n+1} + \frac{1}{n(n+1)}$$

**例子**：

$$\frac{1}{2} = \frac{1}{3} + \frac{1}{6}$$

### 2.2 唯一性

**问题**：埃及分数表示是否唯一？

**答案**：不唯一。同一个分数可能有多种埃及分数表示。

**例子**：

$$\frac{2}{3} = \frac{1}{2} + \frac{1}{6} = \frac{1}{3} + \frac{1}{3}$$

（但第二种表示有重复项，通常要求互不相同）

### 2.3 最短表示

**问题**：给定分数，求最短的埃及分数表示（项数最少）。

**例子**：

- $\frac{5}{6} = \frac{1}{2} + \frac{1}{3}$（2项）
- 这是最短表示吗？是的。

## 三、贪心算法

### 3.1 算法描述

**贪心算法 (Greedy Algorithm)**：

1. 输入：分数 $\frac{a}{b}$（$0 < a < b$）
2. 如果 $a = 1$，返回 $\frac{1}{b}$
3. 否则，找到最小的 $n$ 使得 $\frac{1}{n} \leq \frac{a}{b}$
4. 输出 $\frac{1}{n}$，递归处理 $\frac{a}{b} - \frac{1}{n}$

**数学表述**：

设 $n = \lceil \frac{b}{a} \rceil$（向上取整），则：

$$\frac{a}{b} = \frac{1}{n} + \frac{an - b}{bn}$$

### 3.2 算法正确性

**定理**：贪心算法总是终止，并给出埃及分数表示。

**证明思路**：

1. 每次选择的 $n$ 满足 $\frac{1}{n} \leq \frac{a}{b}$
2. 剩余部分 $\frac{an - b}{bn}$ 的分子 $an - b < a$（因为 $n < \frac{b}{a} + 1$）
3. 因此分子严格递减，算法必然终止

### 3.3 算法效率

**项数上界**：

- 最坏情况：$O(\log b)$ 项
- 实际中通常更少

**例子**：

- $\frac{5}{6} = \frac{1}{2} + \frac{1}{3}$（2项）
- $\frac{5}{22} = \frac{1}{5} + \frac{1}{37} + \frac{1}{4070}$（3项）

## 四、应用与历史

### 4.1 实际应用

**分配问题**：

**例子**：6只猫，5罐猫粮

- 普通分数：每只猫 $\frac{5}{6}$ 罐（难以分配）
- 埃及分数：$\frac{5}{6} = \frac{1}{2} + \frac{1}{3}$
- 分配方法：
  - 3罐分成两半，得到6份 $\frac{1}{2}$ 罐
  - 2罐分成三等份，得到6份 $\frac{1}{3}$ 罐
  - 每只猫得到 $\frac{1}{2} + \frac{1}{3} = \frac{5}{6}$ 罐

**优势**：使用埃及分数表示，分配更容易操作。

### 4.2 历史背景

**古埃及**：

- 古埃及人使用埃及分数进行数学计算
- 记录在纸草书上（如Rhind纸草书）
- 可能与荷鲁斯之眼的神话有关

**荷鲁斯之眼**：

- 传说荷鲁斯的眼睛被分成6部分
- 各部分对应分数：$\frac{1}{2}, \frac{1}{4}, \frac{1}{8}, \frac{1}{16}, \frac{1}{32}, \frac{1}{64}$
- 总和：$\frac{1}{2} + \frac{1}{4} + \frac{1}{8} + \frac{1}{16} + \frac{1}{32} + \frac{1}{64} = \frac{63}{64}$
- 缺少 $\frac{1}{64}$（可能代表魔法或修复）

**Heqat分数**：

- Heqat是古埃及的小麦计量单位
- Heqat分数是通过连续对半分得到的分数
- 形式：$\frac{1}{2}, \frac{1}{4}, \frac{1}{8}, \frac{1}{16}, \ldots$

### 4.3 数学发展

**斐波那契**：

- 1202年《算盘书》（Liber Abaci）中有关于埃及分数的章节
- 介绍了贪心算法

**现代研究**：

- **Erdős-Straus猜想**：对任意 $n \geq 2$，$\frac{4}{n}$ 可以表示为3个单位分数的和
- **Sylvester序列**：用于构造某些埃及分数表示
- **计算复杂性**：寻找最短表示是NP难问题

### 4.4 现代应用

**数论**：

- 研究单位分数的性质
- 与连分数相关

**算法**：

- 贪心算法的经典例子
- 启发式搜索算法

**教育**：

- 分数教学的辅助工具
- 展示数学的多样性和历史
