## 一、定理的数学表述

### 1.1 基本表述

**毕达哥拉斯定理 (Pythagorean Theorem)**：在直角三角形中，两条直角边的平方和等于斜边的平方。

**数学表述**：

设直角三角形 $ABC$，其中 $\angle C = 90°$，$a$ 和 $b$ 分别为两条直角边的长度，$c$ 为斜边的长度，则：

$$a^2 + b^2 = c^2$$

### 1.2 几何表述

**几何表述**：在直角三角形中，以两条直角边为边长的两个正方形的面积之和，等于以斜边为边长的正方形的面积。

**数学表示**：

- 以 $a$ 为边长的正方形面积：$a^2$
- 以 $b$ 为边长的正方形面积：$b^2$
- 以 $c$ 为边长的正方形面积：$c^2$
- 关系：$a^2 + b^2 = c^2$

### 1.3 特殊情况

**等腰直角三角形**：

如果 $a = b$，则：

$$a^2 + a^2 = c^2$$

$$2a^2 = c^2$$

$$c = a\sqrt{2}$$

**例子**：如果 $a = b = 8$，则：

$$c^2 = 8^2 + 8^2 = 64 + 64 = 128$$

$$c = \sqrt{128} = 8\sqrt{2} \approx 11.31$$

## 二、定理的证明

### 2.1 几何证明

**方法一：面积法**

构造四个全等的直角三角形，排列成一个大正方形：

- 大正方形边长：$a + b$
- 大正方形面积：$(a + b)^2 = a^2 + 2ab + b^2$
- 内部小正方形边长：$c$
- 内部小正方形面积：$c^2$
- 四个三角形总面积：$4 \times \frac{1}{2}ab = 2ab$

因此：

$$(a + b)^2 = c^2 + 2ab$$

$$a^2 + 2ab + b^2 = c^2 + 2ab$$

$$a^2 + b^2 = c^2$$

### 2.2 代数证明

**方法二：相似三角形法**

在直角三角形 $ABC$ 中，从直角顶点 $C$ 向斜边 $AB$ 作高 $CD$。

根据相似三角形：

- $\triangle ABC \sim \triangle ACD$：$\frac{a}{c} = \frac{AD}{a}$，即 $a^2 = c \cdot AD$
- $\triangle ABC \sim \triangle BCD$：$\frac{b}{c} = \frac{BD}{b}$，即 $b^2 = c \cdot BD$

因此：

$$a^2 + b^2 = c \cdot AD + c \cdot BD = c(AD + BD) = c \cdot c = c^2$$

## 三、定理的应用

### 3.1 计算斜边长度

**已知两条直角边，求斜边**：

$$c = \sqrt{a^2 + b^2}$$

**例子**：$a = 3$，$b = 4$，则：

$$c = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

### 3.2 计算直角边长度

**已知斜边和一条直角边，求另一条直角边**：

$$a = \sqrt{c^2 - b^2}$$

**例子**：$c = 5$，$b = 4$，则：

$$a = \sqrt{5^2 - 4^2} = \sqrt{25 - 16} = \sqrt{9} = 3$$

### 3.3 判断三角形类型

**判断是否为直角三角形**：

如果 $a^2 + b^2 = c^2$，则三角形是直角三角形。

**判断三角形类型**：

- 如果 $a^2 + b^2 > c^2$，则三角形是锐角三角形
- 如果 $a^2 + b^2 < c^2$，则三角形是钝角三角形

## 四、历史背景

### 4.1 历史发展

**古代发现**：

- **埃及**：埃及人在公元前2000年左右就已经知道这个关系
- **巴比伦**：巴比伦人在公元前1800年左右也有相关记录
- **中国**：《周髀算经》中记载了"勾股定理"

**命名**：

- 定理以古希腊数学家毕达哥拉斯（Pythagoras，约570-495 BCE）命名
- 但毕达哥拉斯可能不是第一个发现这个定理的人
- 毕达哥拉斯学派可能集体研究并推广了这个定理

### 4.2 毕达哥拉斯学派

**学派特点**：

- 强调数学和几何的重要性
- 认为数是万物的本质
- 有严格的入会仪式（据说需要5年）
- 禁止吃肉和豆类

**数学贡献**：

- 毕达哥拉斯定理
- 无理数的发现（$\sqrt{2}$）
- 音乐和数学的关系
- 数的分类（完全数、友好数等）
