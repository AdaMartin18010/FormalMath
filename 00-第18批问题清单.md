## 🔴 高优先级问题 (发布前必须修复)

### P1: K8s Secret使用占位符值

- **位置**: `k8s/03-secret.yaml`
- **问题描述**: 所有Secret值使用占位符，如 `ChangeMeInProduction123!`
- **安全风险**: 生产环境部署后可能导致安全漏洞
- **修复措施**:

  ```bash
  # 生成强密码
  openssl rand -base64 32

  # 或使用kubectl创建Secret
  kubectl create secret generic formalmath-secrets \
    --from-literal=DB_PASSWORD='<强密码>' \
    --from-literal=JWT_SECRET='<随机密钥>' \
    --from-literal=JWT_REFRESH_SECRET='<随机密钥>' \
    --from-literal=SESSION_SECRET='<随机密钥>' \
    --from-literal=API_KEY='<随机密钥>' \
    -n formalmath-prod
  ```

- **预计工时**: 0.5小时
- **责任人**: DevOps团队
- **截止日期**: 2026-04-05

## 🟢 低优先级问题 (1个月内修复)

### L1: 缺少K8s HPA配置

- **位置**: `k8s/`
- **问题描述**: 缺少HorizontalPodAutoscaler资源配置
- **影响**: 无法根据负载自动伸缩
- **修复措施**:

  ```yaml
  # 创建 k8s/07-hpa.yaml
  apiVersion: autoscaling/v2
  kind: HorizontalPodAutoscaler
  metadata:
    name: backend-hpa
    namespace: formalmath-prod
  spec:
    scaleTargetRef:
      apiVersion: apps/v1
      kind: Deployment
      name: backend-deployment
    minReplicas: 3
    maxReplicas: 10
    metrics:
    - type: Resource
      resource:
        name: cpu
        target:
          type: Utilization
          averageUtilization: 70
    - type: Resource
      resource:
        name: memory
        target:
          type: Utilization
          averageUtilization: 80
  ```

- **预计工时**: 2小时
- **责任人**: DevOps团队
- **截止日期**: 2026-05-04

### L2: 缺少K8s Ingress配置

- **位置**: `k8s/`
- **问题描述**: 缺少Ingress资源配置，无法处理外部流量
- **影响**: 需要直接暴露Service或使用LoadBalancer
- **修复措施**:

  ```yaml
  # 创建 k8s/08-ingress.yaml
  apiVersion: networking.k8s.io/v1
  kind: Ingress
  metadata:
    name: formalmath-ingress
    namespace: formalmath-prod
    annotations:
      nginx.ingress.kubernetes.io/rewrite-target: /
      cert-manager.io/cluster-issuer: "letsencrypt-prod"
  spec:
    tls:
    - hosts:
      - formalmath.example.com
      secretName: formalmath-tls
    rules:
    - host: formalmath.example.com
      http:
        paths:
        - path: /
          pathType: Prefix
          backend:
            service:
              name: frontend-service
              port:
                number: 80
        - path: /api
          pathType: Prefix
          backend:
            service:
              name: backend-service
              port:
                number: 3000
  ```

- **预计工时**: 2小时
- **责任人**: DevOps团队
- **截止日期**: 2026-05-04
