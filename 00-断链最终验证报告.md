## 断链原因分析

### 1. 索引文件链接失效 (主要问题)

**影响范围:** 约12,000+ 断链

**原因:**
- `docs/00-概念索引-2025年12月.md` 和 `数学家理念体系/00-数学家索引-2025年12月.md` 是大型索引文件
- 引用了大量不存在或已移动的目标文件
- 目录结构变更后，索引未同步更新

**典型断链模式:**
```markdown
[群论](docs/02-代数结构/02-核心理论/群论/01-群论.md)  → 文件不存在
[代数几何](docs/13-代数几何/01-代数几何基础.md)  → 文件不存在
```

### 2. 锚点格式不匹配 (次要问题)

**影响范围:** 约18,000+ 断链

**原因:**
- 中文锚点ID生成规则不一致
- 特殊字符（如数学符号、括号）处理差异
- 标题修改后锚点未更新

**典型问题锚点:**
```markdown
[链接](#1-代数几何-mathamathag)  → 实际锚点为 #1-代数几何-mathag
[链接](#二math-55深度分析)  → 实际锚点为 #二math-55深度分析-最难本科数学
```

### 3. 术语词典分散文件 (中等问题)

**影响范围:** 约1,000+ 断链

**原因:**
- 多个 `FormalMath术语词典-*.md` 文件分散在不同目录
- 部分术语词典已被整合到 `00-标准术语表-8语言.md`
- 旧链接未更新到新位置

### 4. 目录结构变更遗留 (中等问题)

**影响范围:** 约3,000+ 断链

**原因:**
- 项目 restructuring 后旧路径失效
- 如 `docs/07-逻辑学/` → `docs/33-数理逻辑/`
- `docs/09-形式化证明/` 目录内容调整
