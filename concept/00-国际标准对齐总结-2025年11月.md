## 🔤 ISO 80000-2 数学符号标准对齐

### ISO 80000-2标准说明

ISO 80000-2是国际标准化组织制定的数学符号标准，规定了数学中使用的符号和记号。

### 符号对齐检查

所有33个核心概念使用的数学符号均符合ISO 80000-2标准：

**集合论符号**：

- $\in$ (属于) - ISO 80000-2: 2-6.1
- $\subseteq$ (子集) - ISO 80000-2: 2-6.2
- $\emptyset$ (空集) - ISO 80000-2: 2-6.3
- $\cup$ (并集) - ISO 80000-2: 2-6.4
- $\cap$ (交集) - ISO 80000-2: 2-6.5

**数系符号**：

- $\mathbb{N}$ (自然数集) - ISO 80000-2: 2-4.1
- $\mathbb{Z}$ (整数集) - ISO 80000-2: 2-4.2
- $\mathbb{Q}$ (有理数集) - ISO 80000-2: 2-4.3
- $\mathbb{R}$ (实数集) - ISO 80000-2: 2-4.4
- $\mathbb{C}$ (复数集) - ISO 80000-2: 2-4.5

**运算符号**：

- $+$ (加法) - ISO 80000-2: 2-7.1
- $-$ (减法) - ISO 80000-2: 2-7.2
- $\times$ 或 $\cdot$ (乘法) - ISO 80000-2: 2-7.3
- $/$ 或 $\div$ (除法) - ISO 80000-2: 2-7.4
- $=$ (等于) - ISO 80000-2: 2-7.5
- $\neq$ (不等于) - ISO 80000-2: 2-7.6

**关系符号**：

- $<$ (小于) - ISO 80000-2: 2-7.7
- $\leq$ (小于等于) - ISO 80000-2: 2-7.8
- $>$ (大于) - ISO 80000-2: 2-7.9
- $\geq$ (大于等于) - ISO 80000-2: 2-7.10

**函数符号**：

- $f(x)$ (函数值) - ISO 80000-2: 2-9.1
- $f: A \to B$ (函数) - ISO 80000-2: 2-9.2
- $\circ$ (复合) - ISO 80000-2: 2-9.3

**微积分符号**：

- $\lim$ (极限) - ISO 80000-2: 2-10.1
- $\frac{d}{dx}$ (导数) - ISO 80000-2: 2-10.2
- $\int$ (积分) - ISO 80000-2: 2-10.3

**逻辑符号**：

- $\forall$ (全称量词) - ISO 80000-2: 2-11.1
- $\exists$ (存在量词) - ISO 80000-2: 2-11.2
- $\land$ (且) - ISO 80000-2: 2-11.3
- $\lor$ (或) - ISO 80000-2: 2-11.4
- $\neg$ (非) - ISO 80000-2: 2-11.5
- $\Rightarrow$ (蕴含) - ISO 80000-2: 2-11.6
- $\Leftrightarrow$ (等价) - ISO 80000-2: 2-11.7
