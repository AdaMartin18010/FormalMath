## 2. 主题编号示例

### 2.1 主题编号格式

**格式**: `X.YY.ZZ`

**示例**:

```markdown
## 1. 群论基础 (编号: B.02.01)

### 1.1 群的定义 (编号: B.02.01.01)

### 1.2 群的性质 (编号: B.02.01.02)

## 2. 子群理论 (编号: B.02.02)

### 2.1 子群的定义 (编号: B.02.02.01)

### 2.2 子群的性质 (编号: B.02.02.02)
```

### 2.2 编号使用说明

1. **一级编号 (X)**: 文档类型
   - A: 术语词典
   - B: 学科理论
   - C: 系统文档
   - D: 学习资源
   - E: 标准规范
   - F: 项目文档

2. **二级编号 (YY)**: 数学分支或文档类型
   - 01-16: 数学分支编号
   - 01-99: 文档类型编号

3. **三级编号 (ZZ)**: 具体主题
   - 01-99: 主题编号

## 3. 关联关系示例

### 3.1 概念关联示例

```markdown
## 相关概念

### 强关联概念

- **群** → **环**: 环是群的推广
- **拓扑空间** → **度量空间**: 度量空间是拓扑空间的特殊化
- **连续函数** → **可微函数**: 可微函数是连续函数的特殊化

### 中等关联概念

- **群** ↔ **域**: 域包含群结构
- **拓扑** ↔ **几何**: 拓扑几何

### 弱关联概念

- **群** ↔ **数论**: 群在数论中的应用
```

## 📝 使用说明

### 如何使用模板

1. **选择模板**: 根据文档类型选择合适模板
2. **填写内容**: 按照模板结构填写内容
3. **添加编号**: 使用主题编号体系添加编号
4. **建立关联**: 添加相关文档和概念关联
5. **完善目录**: 确保目录完整
