## **标准说明**

本文档规定资料库中所有资料的标注标准，确保资料的一致性和可检索性。

**适用范围**：

- 所有文献资料
- 所有案例资料
- 所有行业资料

---

## **一、文件命名标准**

### **命名格式**

**基本格式**：`[类型]-[主题]-[作者/来源]-[日期].md`

**格式说明**：

- **类型**：文献/案例/行业资料
- **主题**：资料主题（简短）
- **作者/来源**：作者姓名或来源机构
- **日期**：收集或发布日期（YYYY或YYYY-MM）

### **命名示例**

- `文献-数学认知-喻平-2004.md`
- `案例-概念教学-张老师-2024.md`
- `行业资料-教育报告-教育部-2023.md`

---

## **二、文件内容标准**

### **必需字段**

1. **标题**：资料标题
2. **作者/来源**：作者或来源
3. **日期**：收集或发布日期
4. **类型**：文献/案例/行业资料
5. **主题**：资料主题分类
6. **摘要**：资料摘要（可选但推荐）
7. **关键词**：关键词列表
8. **内容**：资料内容
9. **关联**：与其他资料的关联

### **可选字段**

- **备注**：备注信息
- **标签**：标签列表
- **分类**：详细分类
- **状态**：资料状态（已收集/已分析/已整合）

---

## **三、标注字段说明**

### **1. 标题**

**格式**：`# [资料标题]`

**要求**：

- 清晰明确
- 反映资料核心内容
- 不超过50字

### **2. 作者/来源**

**格式**：`**作者**：[作者姓名]` 或 `**来源**：[来源信息]`

**要求**：

- 文献资料：使用"作者"
- 案例资料：使用"来源"
- 行业资料：使用"来源"

### **3. 日期**

**格式**：`**日期**：[YYYY]` 或 `**日期**：[YYYY-MM]`

**要求**：

- 文献资料：使用发布日期
- 案例资料：使用收集日期
- 行业资料：使用发布日期

### **4. 类型**

**格式**：`**类型**：文献/案例/行业资料`

**要求**：

- 必须选择一种类型
- 保持一致

### **5. 主题**

**格式**：`**主题**：[主题分类]`

**要求**：

- 使用统一的主题分类
- 可以多个主题，用逗号分隔

**主题分类**：

- 神经机制
- 认知模型
- 心理学视角
- 语言学视角
- 发展与教育
- 计算模型
- 国际数学教育
- 数学哲学
- 三层结构理论

### **6. 摘要**

**格式**：`**摘要**：[资料摘要]`

**要求**：

- 简洁明了
- 不超过200字
- 反映资料核心内容

### **7. 关键词**

**格式**：`**关键词**：[关键词1, 关键词2, 关键词3]`

**要求**：

- 3-10个关键词
- 用逗号分隔
- 反映资料核心概念

### **8. 内容**

**格式**：`**内容**：[资料内容]`

**要求**：

- 完整准确
- 格式规范
- 结构清晰

### **9. 关联**

**格式**：

```markdown
**关联**：
- 相关文献：[文献列表]
- 相关案例：[案例列表]
- 相关理论：[理论列表]
```

**要求**：

- 列出相关资料的链接
- 使用相对路径
- 保持更新

## **四、标注示例**

### **示例1：文献资料**

```markdown
# 数学学习心理的CPFS结构理论

**作者**：喻平

**日期**：2004

**类型**：文献

**主题**：心理学视角, CPFS结构理论

**来源**：《数学教育学报》

**摘要**：本文提出数学学习心理的CPFS结构理论，包括概念域、概念系、命题域、命题系。该理论为数学学习提供了新的认知视角。

**关键词**：CPFS结构、概念域、概念系、命题域、命题系、数学学习

**内容**：
[文献内容]

**关联**：
- 相关文献：[三层结构理论整合.md](../三层结构理论整合.md)
- 相关案例：[CPFS结构教学案例](./04-数学教育实践/教学案例/)
- 相关理论：三层结构理论、CPFS结构理论

**备注**：这是CPFS结构理论的重要文献，为数学认知研究提供了理论基础。
```

### **示例2：案例资料**

```markdown
# 数学概念教学案例：函数概念的形成

**来源**：数学教学实践

**日期**：2024

**类型**：案例

**主题**：发展与教育, 概念形成

**背景**：高中数学函数概念教学

**内容**：
[案例内容]

**分析**：
[案例分析]

**关联**：
- 相关文献：[CPFS结构理论](./03-数学认知理论/)
- 相关案例：[其他概念教学案例](./04-数学教育实践/教学案例/)
- 相关理论：三层结构理论、CPFS结构理论

**备注**：这是一个典型的数学概念形成案例，展示了从具体到抽象的过程。
```
