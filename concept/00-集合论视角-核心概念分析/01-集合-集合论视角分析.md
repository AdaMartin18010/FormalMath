## 🎯 一、集合的基础概念组合 (编号: C.00.36.01.01)

### 1.1 集合的原子性

**定义**：集合是最基础的原子概念，无法进一步分解。

**集合论结构**：

$$\text{Set} = \emptyset$$

即集合本身是原子概念，不依赖于其他概念。

**层次定位**：

- **层次**：L0（基础层）
- **层次深度**：$d(\text{Set}) = 0$
- **基础概念依赖**：无

### 1.2 集合的公理化定义

**ZFC公理体系**：

集合通过ZFC公理体系定义，包括：

1. **外延公理**：两个集合相等当且仅当它们有相同的元素
2. **空集公理**：存在空集
3. **配对公理**：对任意集合 $a, b$，存在集合 $\{a, b\}$
4. **并集公理**：对任意集合 $A$，存在集合 $\bigcup A$
5. **幂集公理**：对任意集合 $A$，存在集合 $\mathcal{P}(A)$
6. **无穷公理**：存在无穷集合
7. **替换公理**：对任意函数和集合，像也是集合
8. **正则公理**：每个非空集合都有 $\in$-极小元
9. **选择公理**：每个集合族都有选择函数

**集合的构造性**：

虽然集合本身是原子概念，但通过ZFC公理可以构造各种集合：

- **空集**：$\emptyset$
- **单元素集**：$\{a\}$
- **对集**：$\{a, b\}$
- **并集**：$A \cup B$
- **交集**：$A \cap B$
- **幂集**：$\mathcal{P}(A)$
- **笛卡尔积**：$A \times B$

### 1.3 集合与其他基础概念的关系

**集合 → 元素**：

$$\text{Element} \subseteq \text{Set}$$

元素是集合的组成部分，但元素本身也是集合（在ZFC中）。

**集合 → 关系**：

$$\text{Relation} = \text{Set} \times \text{Set}$$

关系是集合的笛卡尔积的子集。

**集合 → 函数**：

$$\text{Function} \subseteq \text{Relation}$$

函数是满足单值性的关系，因此是集合的特殊子集。

## 🔗 二、集合的层次定位 (编号: C.00.36.01.02)

### 2.1 集合在概念体系中的位置

**层次结构**：

```text
L0层（基础层）
└── 集合 (Set) - 层次深度 0
    ├── 元素 (Element) - 层次深度 0-1
    ├── 关系 (Relation) - 层次深度 1
    └── 函数 (Function) - 层次深度 1
```

**基础地位**：

- 集合是所有数学概念的基础
- 所有其他概念都可以用集合构造
- 集合是概念体系的"原子"

### 2.2 集合的认知结构映射

**认知结构映射**：

$$\phi(\text{Set}) = \{\text{分类}, \text{归属}, \text{包含}\}$$

- **分类**：集合对应认知中的"类别"概念
- **归属**：元素属于集合对应认知中的"属于"关系
- **包含**：集合的包含关系对应认知中的"层次"关系

**映射函数**：

$$\phi: \text{Set} \to \{\text{分类}, \text{归属}, \text{包含}\}$$

### 2.3 集合的静态层次结构

**基础概念依赖表**：

| 基础概念 | 在集合中的作用 | 依赖关系 |
|---------|-------------|---------|
| 无 | 集合是原子概念 | - |

**层次深度**：$d(\text{Set}) = 0$

**组合关系**：

集合不依赖于其他概念，但其他概念都依赖于集合：

- $\text{Relation} = f(\text{Set})$
- $\text{Function} = f(\text{Set}, \text{Relation})$
- $\text{Group} = f(\text{Set}, \text{Function})$

## 📊 三、集合的组合构造能力 (编号: C.00.36.01.03)

### 3.1 集合运算

**基本运算**：

1. **并集**：$A \cup B = \{x : x \in A \text{ 或 } x \in B\}$
2. **交集**：$A \cap B = \{x : x \in A \text{ 且 } x \in B\}$
3. **差集**：$A \setminus B = \{x : x \in A \text{ 且 } x \notin B\}$
4. **补集**：$A^c = U \setminus A$（$U$ 是全集）
5. **笛卡尔积**：$A \times B = \{(a, b) : a \in A, b \in B\}$
6. **幂集**：$\mathcal{P}(A) = \{B : B \subseteq A\}$

### 3.2 集合构造其他概念的能力

**关系构造**：

$$\text{Relation} = \mathcal{P}(A \times B)$$

关系是笛卡尔积的幂集的子集。

**函数构造**：

$$\text{Function} = \{R \in \mathcal{P}(A \times B) : R \text{ 满足单值性}\}$$

函数是满足特定条件的关系。

**群构造**：

$$\text{Group} = (G, \cdot, e, ^{-1})$$

其中：

- $G$ 是集合
- $\cdot: G \times G \to G$ 是函数
- $e \in G$ 是元素
- $^{-1}: G \to G$ 是函数
