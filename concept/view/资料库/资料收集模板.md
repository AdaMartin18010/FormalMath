# **资料收集模板**

---

## **模板说明**

本模板用于统一资料收集的格式和标准，确保资料的质量和一致性。

---

## **模板格式**

### **文献资料模板**

```markdown
# [文献标题]

**作者**：[作者姓名]

**日期**：[发布日期]

**类型**：文献

**主题**：[主题分类]

**来源**：[来源信息]

**摘要**：[文献摘要]

**关键词**：[关键词1, 关键词2, 关键词3]

**内容**：
[文献内容]

**关联**：
- 相关文献：[相关文献列表]
- 相关案例：[相关案例列表]
- 相关理论：[相关理论列表]

**备注**：[备注信息]
```

## **使用说明**

1. **选择模板**：根据资料类型选择相应的模板
2. **填写信息**：按照模板格式填写资料信息
3. **保存文件**：按照文件命名标准保存文件
4. **分类存储**：将文件存储到相应的分类文件夹

## **示例**

### **文献资料示例**

```markdown
# 数学学习心理的CPFS结构理论

**作者**：喻平

**日期**：2004

**类型**：文献

**主题**：CPFS结构理论

**来源**：《数学教育学报》

**摘要**：本文提出数学学习心理的CPFS结构理论，包括概念域、概念系、命题域、命题系。该理论为数学学习提供了新的认知视角。

**关键词**：CPFS结构、概念域、概念系、命题域、命题系、数学学习

**内容**：
[文献内容]

**关联**：
- 相关文献：数学认知结构研究、数学概念形成研究
- 相关案例：数学概念教学案例
- 相关理论：三层结构理论

**备注**：这是CPFS结构理论的重要文献，为数学认知研究提供了理论基础。
```
