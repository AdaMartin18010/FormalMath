## **目录格式标准**

### **格式要求**

```markdown
## **目录**

- [一级标题](#锚点)
  - [二级标题](#锚点)
    - [三级标题](#锚点)
```

### **锚点生成规则**

1. 转换为小写
2. 移除Markdown格式标记（**、*等）
3. 空格替换为连字符（-）
4. 中文标点保持不变
5. 特殊字符移除

**示例**：

- `## **研究目标**` → `#研究目标`
- `### **核心思想**` → `#核心思想`
- `## **LIDA认知架构概述**` → `#lida认知架构概述`

---

## **批量处理建议**

由于文件较多，建议使用以下方法：

1. **使用grep提取标题**：

   ```bash
   grep "^## " filename.md
   grep "^### " filename.md
   ```

2. **手动生成目录**：根据提取的标题生成目录

3. **插入目录**：在文件开头（标题后）插入目录
