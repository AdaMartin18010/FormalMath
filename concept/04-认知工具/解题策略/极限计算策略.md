# 解题策略：极限计算策略

## 严格定义

极限计算是数学分析的核心技能，涉及序列极限、函数极限以及它们在各类不定式中的应用。形式化地，设 $(X, d)$ 为度量空间（通常为 $\mathbb{R}$ 或 $\mathbb{C}$），$a$ 为 $X$ 的极限点。

**序列极限**：$\lim_{n \to \infty} a_n = L$ 定义为
$$\forall \varepsilon > 0, \exists N \in \mathbb{N}, \forall n \geq N: |a_n - L| < \varepsilon。$$

**函数极限**：$\lim_{x \to a} f(x) = L$ 定义为
$$\forall \varepsilon > 0, \exists \delta > 0, \forall x \in X: 0 < d(x, a) < \delta \Rightarrow |f(x) - L| < \varepsilon。$$

极限计算策略的核心是**识别不定式类型**并选择最有效的展开或变换方法。

## 历史背景

极限概念的历史可以追溯到古希腊的穷竭法（Eudoxus, Archimedes），但严格的极限定义直到 19 世纪才形成。

**古典时期（公元前–1600）**：Zeno 悖论揭示了无限过程的直觉困难；Archimedes 用穷竭法计算面积和体积，但未使用极限语言。

**微积分的诞生（1600–1800）**：Newton 和 Leibniz 发展了流数法和微分法，但他们的极限概念基于"无穷小"的直觉，缺乏严格性。Euler 大胆地操纵发散级数，获得了许多正确结果，但也引发了"第二次数学危机"。

**严格化时期（1800–1900）**：Bolzano（1817）和 Cauchy（1821）给出了极限的严格 $ \varepsilon$-$\delta$ 定义；Weierstrass 将分析完全算术化，消除了无穷小；Dedekind 和 Cantor 建立了实数的严格构造。

**现代发展（1900–至今）**：非标准分析（Robinson, 1961）重新赋予无穷小严格意义；渐近分析研究函数在奇异点附近的展开；计算机代数系统实现了符号极限计算。

## 核心策略体系

### Taylor 展开

**原理**：若 $f$ 在 $a$ 处 $n$ 阶可微，则
$$f(x) = f(a) + f'(a)(x-a) + \frac{f''(a)}{2!}(x-a)^2 + \cdots + \frac{f^{(n)}(a)}{n!}(x-a)^n + R_n(x)，$$
其中 $R_n(x) = o((x-a)^n)$（Peano 余项）或 $R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!}(x-a)^{n+1}$（Lagrange 余项）。

**适用场景**：$0/0$ 或 $\infty/\infty$ 型不定式，其中分子分母都可局部展开。

**典型例子**：求 $\lim_{x \to 0} \frac{e^x - 1 - x}{x^2}$。

$e^x = 1 + x + \frac{x^2}{2} + O(x^3)$，故
$$\frac{e^x - 1 - x}{x^2} = \frac{\frac{x^2}{2} + O(x^3)}{x^2} = \frac{1}{2} + O(x) \to \frac{1}{2}。$$

### L'Hôpital 法则

**原理**：设 $f, g$ 在 $a$ 附近可微，$g'(x) \neq 0$。若

- $\lim_{x \to a} f(x) = \lim_{x \to a} g(x) = 0$（或 $\pm\infty$），且
- $\lim_{x \to a} \frac{f'(x)}{g'(x)}$ 存在（或为 $\pm\infty$），

则
$$\lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)}。$$

**适用场景**：$0/0$ 或 $\infty/\infty$ 型，且导数之比更易计算。

**典型例子**：求 $\lim_{x \to 0^+} x \ln x$。

改写为 $\frac{\ln x}{1/x}$（$\infty/\infty$ 型）。由 L'Hôpital：
$$\lim_{x \to 0^+} \frac{\ln x}{1/x} = \lim_{x \to 0^+} \frac{1/x}{-1/x^2} = \lim_{x \to 0^+} (-x) = 0。$$

### 夹逼定理

**原理**：若 $a_n \leq b_n \leq c_n$ 且 $\lim a_n = \lim c_n = L$，则 $\lim b_n = L$。

**适用场景**：表达式复杂但可被上下界夹逼，如含三角函数、取整函数或组合数的极限。

**典型例子**：证明 $\lim_{n \to \infty} \frac{n!}{n^n} = 0$。

$$0 \leq \frac{n!}{n^n} = \frac{1 \cdot 2 \cdots n}{n \cdot n \cdots n} \leq \frac{1 \cdot n \cdots n}{n^n} = \frac{1}{n} \to 0。$$

### 积分判别与 Riemann 和

**原理**：将序列极限转化为定积分：
$$\lim_{n \to \infty} \frac{1}{n} \sum_{k=1}^n f\left(\frac{k}{n}\right) = \int_0^1 f(x) \, \mathrm{d}x。$$

**适用场景**：含求和式的极限，尤其是 $\frac{1}{n}\sum f(k/n)$ 形式的 Riemann 和。

**典型例子**：求 $\lim_{n \to \infty} \sum_{k=1}^n \frac{1}{n + k}$。

$$\sum_{k=1}^n \frac{1}{n + k} = \frac{1}{n} \sum_{k=1}^n \frac{1}{1 + k/n} \to \int_0^1 \frac{1}{1 + x} \, \mathrm{d}x = \ln 2。$$

### Stolz-Cesàro 定理

**原理**：设 $\{b_n\}$ 严格单调趋于 $+\infty$。若
$$\lim_{n \to \infty} \frac{a_{n+1} - a_n}{b_{n+1} - b_n} = L，$$
则 $\lim_{n \to \infty} a_n / b_n = L$。

这是离散版本的 L'Hôpital 法则。

**典型例子**：求 $\lim_{n \to \infty} \frac{1 + 2 + \cdots + n}{n^2}$。

由 Stolz-Cesàro：
$$\lim \frac{a_n}{b_n} = \lim \frac{a_{n+1} - a_n}{b_{n+1} - b_n} = \lim \frac{n+1}{(n+1)^2 - n^2} = \lim \frac{n+1}{2n+1} = \frac{1}{2}。$$

## 策略选择指南

极限计算的关键是识别不定式类型：

1. **$0/0$ 型**：Taylor 展开、L'Hôpital 法则、因式分解、有理化；
2. **$\infty/\infty$ 型**：L'Hôpital、比较最高次项、Stolz-Cesàro；
3. **$0 \cdot \infty$ 型**：改写为 $0/0$ 或 $\infty/\infty$；
4. **$\infty - \infty$ 型**：通分、有理化、提取公因子；
5. **$1^\infty, 0^0, \infty^0$ 型**：取对数化为 $0 \cdot \infty$。

## 常见陷阱

- **L'Hôpital 条件未验证**：必须验证是 $0/0$ 或 $\infty/\infty$ 型，且导数之比极限存在；
- **循环使用 L'Hôpital**：$\lim_{x \to \infty} \frac{x}{\sqrt{x^2+1}}$ 用 L'Hôpital 会循环，应直接分子分母除以 $x$；
- **忽略单侧极限**：$\lim_{x \to 0} \frac{1}{x}$ 不存在（左右极限不同）；
- **Stolz-Cesàro 的单调性条件**：$\{b_n\}$ 必须严格单调。

## 相关定理

- **Bolzano-Weierstrass 定理**：有界序列必有收敛子列；
- **单调收敛定理**：单调有界序列必收敛；
- **Cauchy 收敛准则**：序列收敛当且仅当为 Cauchy 列；
- **Heine 定理**：函数极限与序列极限的等价性；
- **控制收敛定理**：积分与极限交换的条件。
