# 解题策略：不等式证明策略

## 严格定义

不等式证明是分析学的核心技能，涉及比较两个数学表达式的大小关系。形式化地，不等式是形如 $A \leq B$（或 $<, \geq, >$）的命题，其中 $A, B$ 为涉及变量、函数或序列的表达式。

不等式证明策略可按其依赖的核心数学工具分类：

- **代数不等式**：利用代数恒等变形、因式分解、配方；
- **分析不等式**：利用单调性、凸性、积分比较；
- **泛函不等式**：利用范数、内积、算子理论；
- **概率不等式**：利用期望、方差、集中度。

## 历史背景

不等式研究贯穿整个数学史。

**古典时期**：古希腊数学家研究了几何不等式（如三角形两边之和大于第三边）；AM-GM 不等式的二元形式 $(a+b)/2 \geq \sqrt{ab}$ 可追溯到 Euclid。

**近代发展**：Cauchy（1821）系统研究了均值不等式和 Cauchy-Schwarz 不等式；Chebyshev、Hölder、Minkowski 在 19 世纪末发展了 $L^p$ 空间的基本不等式；Jensen（1906）将凸函数不等式系统化；Hardy、Littlewood 和 Pólya 的《不等式》（1934）是该领域的经典。

**现代发展**：Sobolev 不等式、Poincaré 不等式、等周不等式成为偏微分方程和几何分析的核心工具；熵不等式、对数 Sobolev 不等式在概率论和信息论中至关重要；计算机辅助证明（如验证 BMI 界和最优常数）扩展了不等式研究的边界。

## 核心策略体系

### Cauchy-Schwarz 不等式

**原理**：对实或复内积空间中的向量 $x, y$，
$$|\langle x, y \rangle| \leq \|x\| \cdot \|y\|，$$
等号成立当且仅当 $x$ 与 $y$ 线性相关。

**离散形式**：
$$\left(\sum_{i=1}^n a_i b_i\right)^2 \leq \left(\sum_{i=1}^n a_i^2\right)\left(\sum_{i=1}^n b_i^2\right)。$$

**积分形式**：
$$\left(\int f g\right)^2 \leq \left(\int f^2\right)\left(\int g^2\right)。$$

**典型例子**：证明 $\frac{a^2}{x} + \frac{b^2}{y} \geq \frac{(a+b)^2}{x+y}$（$x, y > 0$）。

由 Cauchy-Schwarz：
$$(a + b)^2 = \left(\frac{a}{\sqrt{x}} \cdot \sqrt{x} + \frac{b}{\sqrt{y}} \cdot \sqrt{y}\right)^2 \leq \left(\frac{a^2}{x} + \frac{b^2}{y}\right)(x + y)。$$

### Jensen 不等式

**原理**：设 $f$ 为区间 $I$ 上的凸函数。则对任意 $x_1, \ldots, x_n \in I$ 和权重 $\lambda_i \geq 0$（$\sum \lambda_i = 1$），
$$f\left(\sum_{i=1}^n \lambda_i x_i\right) \leq \sum_{i=1}^n \lambda_i f(x_i)。$$

对凹函数，不等号反向。

**典型例子**：证明 AM-GM 不等式。

$\ln x$ 是凹函数。由 Jensen：
$$\ln\left(\frac{x_1 + \cdots + x_n}{n}\right) \geq \frac{\ln x_1 + \cdots + \ln x_n}{n} = \ln\left((x_1 \cdots x_n)^{1/n}\right)。$$
取指数即得 $(x_1 + \cdots + x_n)/n \geq (x_1 \cdots x_n)^{1/n}$。

### Hölder 不等式

**原理**：设 $p, q > 1$ 且 $1/p + 1/q = 1$（共轭指数）。则
$$\sum_{i=1}^n |a_i b_i| \leq \left(\sum_{i=1}^n |a_i|^p\right)^{1/p} \left(\sum_{i=1}^n |b_i|^q\right)^{1/q}。$$

积分形式类似。$p = q = 2$ 时退化为 Cauchy-Schwarz。

**典型例子**：证明 Minkowski 不等式（三角不等式在 $L^p$ 中）。

$$\|f + g\|_p^p = \int |f + g|^p \leq \int |f + g|^{p-1}|f| + \int |f + g|^{p-1}|g|。$$
对右边两项分别用 Hölder（共轭指数为 $p$ 和 $q = p/(p-1)$）：
$$\int |f + g|^{p-1}|f| \leq \left(\int |f + g|^p\right)^{(p-1)/p} \|f\|_p。$$
整理即得 $\|f + g\|_p \leq \|f\|_p + \|g\|_p$。

### 排序不等式

**原理**：设 $a_1 \leq a_2 \leq \cdots \leq a_n$，$b_1 \leq b_2 \leq \cdots \leq b_n$。对任意排列 $\sigma$，
$$\sum a_i b_{n+1-i} \leq \sum a_i b_{\sigma(i)} \leq \sum a_i b_i。$$
即**同序和最大，逆序和最小**。

**典型例子**：证明 Chebyshev 和不等式：
$$\frac{1}{n}\sum a_i b_i \geq \left(\frac{1}{n}\sum a_i\right)\left(\frac{1}{n}\sum b_i\right)$$
（$a_i, b_i$ 同序）。

由排序不等式，$n$ 个同序和 $\sum a_i b_i$ 不小于任意排列和，特别地：
$$n \sum a_i b_i \geq \sum_{k=0}^{n-1} \sum_i a_i b_{i+k} = \left(\sum a_i\right)\left(\sum b_i\right)。$$

### 调整法与对称化

**原理**：若不等式在变量对称时取等，可通过逐步调整变量向对称位置移动，证明目标函数单调变化。

**典型例子**：证明在周长固定的三角形中，等边三角形面积最大。

设三角形边长为 $a, b, c$，半周长 $s = (a+b+c)/2$ 固定。面积 $S = \sqrt{s(s-a)(s-b)(s-c)}$。固定 $a+b$，调整使 $a = b$（向对称移动），则 $(s-a)(s-b)$ 在 $a = b$ 时取最大（由 AM-GM）。重复此过程直到 $a = b = c$。

## 策略选择指南

不等式证明工具的选择取决于：

1. **变量个数**：单变量优先考虑微积分（单调性、凸性）；多变量考虑对称化、齐次化；
2. **齐次性**：齐次不等式可设归一化条件（如 $a + b + c = 1$）；
3. **对称性**：对称不等式常在等号条件处取极值；
4. **等号条件**：等号成立情形暗示应使用的工具（等号在 $a = b$ 时暗示 Jensen 或对称化）。

## 常见陷阱

- **等号条件遗漏**：证明后未验证等号何时成立；
- **凸凹性混淆**：Jensen 不等式的方向取决于凸凹性；
- **非负性假设误用**：AM-GM 要求非负；Cauchy-Schwarz 对实数成立，复数需取模；
- **积分不等式的可积性**：应用积分形式的不等式前需验证函数可积。

## 相关定理

- **AM-GM 不等式**：$\frac{a_1 + \cdots + a_n}{n} \geq \sqrt[n]{a_1 \cdots a_n}$；
- **Cauchy-Schwarz 不等式**：内积空间的基本不等式；
- **Hölder 不等式**：$L^p$ 空间的对偶性基础；
- **Minkowski 不等式**：$L^p$ 范数的三角不等式；
- **Wirtinger 不等式**：$\int_0^{2\pi} f^2 \leq \int_0^{2\pi} (f')^2$（$f$ 零均值周期函数）；
- **Sobolev 不等式**：$\|f\|_{L^{p^*}} \leq C \|\nabla f\|_{L^p}$。
