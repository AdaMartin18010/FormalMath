## 🎯 一、理论框架总结 (编号: C.04.17.01)

### 1.1 认知维度框架

**核心模型**: 直观-知性-理性三维认知模型

```mermaid
graph TD
    A[认知维度] --> B[直观维度]
    A --> C[知性维度]
    A --> D[理性维度]

    B --> B1[感性认识]
    B --> B2[形象思维]
    B --> B3[具体实例]

    C --> C1[概念理解]
    C --> C2[分类归纳]
    C --> C3[抽象思维]

    D --> D1[逻辑推理]
    D --> D2[形式化证明]
    D --> D3[系统建构]

    style A fill:#e1f5ff
    style B fill:#fff4e1
    style C fill:#e8f5e9
    style D fill:#f3e5f5
```

**特征矩阵**:

| 认知维度 | 特征 | 工具支持 | 学习阶段 |
|---------|------|---------|---------|
| **直观维度** | 感性认识、形象思维、具体实例 | 思维导图、可视化图表 | 入门阶段 |
| **知性维度** | 概念理解、分类归纳、抽象思维 | 概念地图、知识矩阵 | 中级阶段 |
| **理性维度** | 逻辑推理、形式化证明、系统建构 | 证明系统、形式化工具 | 高级阶段 |
