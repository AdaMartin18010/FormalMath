## 📋 概述 (编号: C.04.06.01)

本文档提供FormalMath项目的问题解决工具，帮助学习者系统化地解决数学问题。

**目标**: 建立系统化的问题解决工具体系

---

## 🔍 一、问题分析工具 (编号: C.04.06.02)

### 1.1 问题理解框架

**步骤**:

1. **理解问题陈述**
   - 明确已知条件
   - 明确求解目标
   - 明确约束条件

2. **识别问题类型**
   - 证明题
   - 计算题
   - 构造题
   - 存在性问题

3. **分析问题结构**
   - 识别关键概念
   - 识别关键关系
   - 识别隐含条件

### 1.2 问题分解工具

**分解方法**:

- **按步骤分解**: 将问题分解为多个步骤
- **按部分分解**: 将问题分解为多个部分
- **按条件分解**: 按不同条件分类讨论

**示例**: 证明群G的阶为素数p时，G是循环群

**分解**:

1. 步骤1: 证明G有非平凡子群
2. 步骤2: 证明非平凡子群就是G本身
3. 步骤3: 证明G是循环群

### 1.3 问题转化工具

**转化方法**:

- **等价转化**: 转化为等价问题
- **特殊化**: 先解决特殊情况
- **一般化**: 从特殊情况推广
- **类比转化**: 类比相似问题

## 🛠️ 三、方法应用工具 (编号: C.04.06.04)

### 3.1 代数方法

**常用方法**:

- **因式分解**: 分解多项式
- **配方法**: 配方求解
- **换元法**: 变量替换
- **消元法**: 消去变量

**示例**: 求解方程 x² + 2x - 3 = 0

- 方法: 因式分解
- 步骤: (x+3)(x-1) = 0
- 解: x = -3 或 x = 1

### 3.2 几何方法

- **坐标法**: 建立坐标系
- **向量法**: 使用向量
- **变换法**: 几何变换
- **面积法**: 利用面积

### 3.3 分析方法

- **导数法**: 求导数
- **积分法**: 求积分
- **级数法**: 使用级数

### 3.4 组合方法

**常用方法**:

- **计数法**: 计数原理
- **递推法**: 递推关系
- **生成函数**: 生成函数
- **容斥原理**: 容斥原理

## ✅ 四、验证检查工具 (编号: C.04.06.05)

### 4.1 逻辑检查

**检查项**:

- ✅ 推理是否严密
- ✅ 条件是否充分
- ✅ 结论是否合理
- ✅ 是否有遗漏情况

### 4.2 计算检查

**检查项**:

- ✅ 计算是否正确
- ✅ 单位是否一致
- ✅ 精度是否足够
- ✅ 结果是否合理

### 4.3 特例检查

**检查方法**:

- 用简单特例验证
- 用边界情况验证
- 用已知结果验证

**示例**: 验证公式 (a+b)² = a² + 2ab + b²

- 特例1: a=1, b=1 → (1+1)² = 4 = 1+2+1 ✓
- 特例2: a=2, b=3 → (2+3)² = 25 = 4+12+9 ✓

### 4.4 反例检查

**检查方法**:

- 寻找反例
- 检查反例是否成立
- 如果反例成立，则结论错误
