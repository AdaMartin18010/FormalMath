# 概念卡片：素数定理

## 严格定义

设 $\pi(x)$ 为不超过 $x$ 的素数个数，即
$$\pi(x) = \sum_{p \leq x} 1，$$
其中求和遍历所有素数 $p \leq x$。

**素数定理（Prime Number Theorem, PNT）**：
$$\pi(x) \sim \frac{x}{\ln x} \quad (x \to \infty)，$$
即
$$\lim_{x \to \infty} \frac{\pi(x)}{x / \ln x} = 1。$$

**等价形式**：

- 第 $n$ 个素数 $p_n \sim n \ln n$；
- $\psi(x) = \sum_{p^k \leq x} \ln p \sim x$（Chebyshev 函数）；
- $\vartheta(x) = \sum_{p \leq x} \ln p \sim x$；
- $\sum_{p \leq x} \frac{\ln p}{p} \sim \ln x$。

**更精细的渐近**：由 Riemann 假设（RH），
$$\pi(x) = \mathrm{Li}(x) + O(\sqrt{x} \ln x)，$$
其中 $\mathrm{Li}(x) = \int_2^x \frac{\mathrm{d}t}{\ln t}$ 为对数积分。 unconditionally，已知
$$\pi(x) = \mathrm{Li}(x) + O\left(x \exp\left(-c \frac{(\ln x)^{3/5}}{(\ln \ln x)^{1/5}}\right)\right)。$$

## 历史背景

素数定理的历史是解析数论发展的主线。

**经验观察（1790–1800）**：Legendre（1798）和 Gauss（1792–1793，未发表）独立地通过数值计算猜测了素数分布的规律。Legendre 提出 $\pi(x) \approx x / (\ln x - A)$（$A \approx 1.08366$）；Gauss 猜测 $\pi(x) \approx \mathrm{Li}(x)$。Gauss 的猜测更为精确：$\mathrm{Li}(x) - \pi(x)$ 始终为正（对 $x \geq 2$），尽管 Littlewood 后来证明了它无穷多次变号。

**Chebyshev 的突破（1850）**：Pafnuty Chebyshev 证明了存在常数 $c_1, c_2 > 0$ 使得
$$c_1 \frac{x}{\ln x} \leq \pi(x) \leq c_2 \frac{x}{\ln x}$$
对足够大的 $x$ 成立。他引入了 $\psi(x)$ 和 $\vartheta(x)$ 等辅助函数，为后续证明奠定了基础。

**Riemann 的革命（1859）**：Bernhard Riemann 的论文《论小于给定值的素数个数》开创了用复分析研究素数分布的方法。他引入了 Riemann $\zeta$ 函数
$$\zeta(s) = \sum_{n=1}^\infty \frac{1}{n^s} = \prod_p \frac{1}{1 - p^{-s}} \quad (\mathrm{Re}(s) > 1)，$$
并证明了 $\zeta(s)$ 可解析延拓到 $\mathbb{C} \setminus \{1\}$，在 $s = 1$ 处有单极点。他提出了著名的**Riemann 假设**：$\zeta(s)$ 的所有非平凡零点都位于直线 $\mathrm{Re}(s) = 1/2$ 上。

**PNT 的证明（1896）**：Jacques Hadamard 和 Charles de la Vallée Poussin 独立地利用 $\zeta(1 + it) \neq 0$（$t \in \mathbb{R}$）证明了素数定理。这是复分析方法在数论中的首次重大胜利。

**初等证明（1949）**：Paul Erdős 和 Atle Selberg 独立地发现了不依赖复分析的初等证明。这一突破震惊了数学界——此前人们普遍认为复分析是证明 PNT 的必需工具。

## 直观理解

素数定理的直觉可以从概率角度来理解。

**素数的"密度"**：在 $x$ 附近，整数为素数的"概率"约为 $1/\ln x$。这是因为一个随机整数 $n$ 不被任何小素数 $p$ 整除的概率约为 $1 - 1/p$，由独立性启发（虽不严格成立），
$$P(n \text{ 为素数}) \approx \prod_{p \leq \sqrt{n}} \left(1 - \frac{1}{p}\right)。$$
由 Mertens 定理，此乘积渐近于 $e^{-\gamma} / \ln \sqrt{n} = 2e^{-\gamma} / \ln n \approx 1.12 / \ln n$，与 $1/\ln n$ 同阶。

**素数间隙**：素数定理暗示平均素数间隙约为 $\ln p$。但实际间隙可能远大于或小于此平均值。已知存在任意长的素数间隙（$n! + 2, n! + 3, \ldots, n! + n$ 都是合数），也存在任意小的相对间隙（孪生素数猜想说有无穷多对 $p, p+2$）。

**随机矩阵与素数**：Montgomery-Odlyzko 定律说 $\zeta$ 函数零点的间隔分布与随机 Hermite 矩阵（GUE）的特征值间隔分布一致。这一惊人的联系（通过 Hilbert-Pólya 猜想）将素数分布与量子力学谱理论联系起来。

## 数学例子

### 解析证明的核心：$\zeta(1 + it) \neq 0$

**引理**：$\zeta(1 + it) \neq 0$ 对所有 $t \in \mathbb{R} \setminus \{0\}$。

**证明概要**：对 $\sigma > 1$，由 Euler 乘积取对数：
$$\ln \zeta(s) = \sum_p \sum_{k=1}^\infty \frac{1}{k p^{ks}} = \sum_p \frac{1}{p^s} + O(1)。$$

利用 $3 + 4\cos\theta + \cos 2\theta = 2(1 + \cos\theta)^2 \geq 0$，对 $s = \sigma + it$ 有
$$\ln |\zeta(\sigma)^3 \zeta(\sigma + it)^4 \zeta(\sigma + 2it)| = \sum_p \sum_k \frac{1}{k p^{k\sigma}} (3 + 4\cos(kt\ln p) + \cos(2kt\ln p)) \geq 0。$$

故 $|\zeta(\sigma)^3 \zeta(\sigma + it)^4 \zeta(\sigma + 2it)| \geq 1$。

若 $\zeta(1 + it) = 0$（$t \neq 0$），则当 $\sigma \to 1^+$ 时，$\zeta(\sigma) \sim 1/(\sigma - 1)$，$\zeta(\sigma + it) = O(\sigma - 1)$，$\zeta(\sigma + 2it) = O(1)$。左边 $= O((\sigma - 1)^{-3} \cdot (\sigma - 1)^4) = O(\sigma - 1) \to 0$，矛盾。

### 显式公式

**von Mangoldt 显式公式**：设 $\psi(x) = \sum_{n \leq x} \Lambda(n)$，其中 $\Lambda(n) = \ln p$（若 $n = p^k$）否则 $0$。则
$$\psi(x) = x - \sum_\rho \frac{x^\rho}{\rho} - \frac{\zeta'(0)}{\zeta(0)} - \frac{1}{2} \ln(1 - x^{-2})，$$
其中 $\rho$ 遍历 $\zeta(s)$ 的非平凡零点。

此公式将素数分布与 $\zeta$ 函数的零点直接联系起来：若所有零点满足 $\mathrm{Re}(\rho) = 1/2$（Riemann 假设），则 $\sum x^\rho / \rho = O(\sqrt{x} \ln^2 x)$，给出 $\psi(x) = x + O(\sqrt{x} \ln^2 x)$。

### 算术级数中的素数

**Dirichlet 定理（1837）**：若 $\gcd(a, m) = 1$，则算术级数 $a, a+m, a+2m, \ldots$ 中有无穷多个素数。

**Dirichlet 定理的定量版**：
$$\pi(x; m, a) = \sum_{\substack{p \leq x \\ p \equiv a \pmod m}} 1 \sim \frac{1}{\varphi(m)} \frac{x}{\ln x}。$$

即素数在 $\varphi(m)$ 个允许的剩余类中渐近均匀分布。这要求证明 Dirichlet $L$-函数 $L(s, \chi)$ 在 $s = 1$ 处非零（对所有非主特征 $\chi$）。

## 与其他概念的联系

### 与 Riemann 假设的联系

素数定理的误差项与 $\zeta$ 函数零点的实部直接相关。设 $\Theta = \sup\{\mathrm{Re}(\rho) : \zeta(\rho) = 0\}$。则
$$\pi(x) = \mathrm{Li}(x) + O(x^\Theta \ln x)。$$

已知 $1/2 \leq \Theta \leq 1$。Riemann 假设等价于 $\Theta = 1/2$。 unconditionally，Hadamard-de la Vallée Poussin 证明了 $\Theta < 1$，这已足以推出 PNT。

### 与信息论的联系

素数定理在信息论中有深刻应用。设 $p_n$ 为第 $n$ 个素数。整数的**素因子编码**将 $n = p_1^{a_1} \cdots p_k^{a_k}$ 编码为 $(a_1, a_2, \ldots)$。由 PNT，$p_n \sim n \ln n$，故编码 $n$ 所需的素数指标约为 $\ln n / \ln \ln n$。

### 与计算复杂性

素数定理是素数判定算法和素数生成算法的理论基础。素数定理说 $n$ 附近素数的密度约为 $1/\ln n$，因此随机检验 $O(\ln n)$ 个 $n$ 附近的整数即可高概率找到素数。这是 RSA 密钥生成和随机化素数测试（Miller-Rabin）的效率保证。
