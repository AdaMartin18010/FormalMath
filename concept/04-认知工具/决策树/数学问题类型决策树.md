# 数学问题类型决策树

## 根节点：你面对的是什么类型的问题？

### 分支1：证明题

#### 子分支1.1：证明"存在性"

- 构造法 → 直接给出具体例子
- 反证法 → 假设不存在导出矛盾
- 鸽巢原理 → 适用于离散组合
- Zorn引理 → 适用于无限偏序集

#### 子分支1.2：证明"唯一性"

- 反证法 → 假设两个不同解
- 直接比较 → 证明任意两个解相等

#### 子分支1.3：证明"对所有成立"

- 数学归纳法 → ℕ上的命题
- 直接证明 → 从已知条件推导
- 反证法 → 假设存在反例
- 分类讨论 → 穷尽所有情况

### 分支2：计算题

#### 子分支2.1：极限计算

- 直接代入 → 连续函数
- ε-δ定义 → 严格证明
- L'Hôpital法则 → 0/0或∞/∞
- Taylor展开 → 复杂函数

#### 子分支2.2：积分计算

- 换元法 → 复合函数
- 分部积分 → 乘积函数
- 留数定理 → 复围道积分
- 数值积分 → 无解析解

### 分支3：分类题

#### 子分支3.1：方程分类

- 线性/非线性
- 常微分/偏微分
- 齐次/非齐次
- 可解群分析

#### 子分支3.2：空间分类

- 拓扑空间类型
- 流形维数
- 度量性质
