## 1. 📋 概述

积分是微积分的核心概念，是导数的逆运算。
积分理论在数学、物理学、工程学等领域有广泛应用，是计算面积、体积、质量等量的基础工具。

## 2. 🎯 严格定义

### 2.1 基础定义 (L0)

**直观理解**: 积分是计算曲线下面积、体积等量的工具，是"累加"的连续版本。

**基本定义**: 函数 $f$ 在区间 $[a,b]$ 上的定积分 $\int_a^b f(x)dx$ 表示曲线 $y = f(x)$ 下从 $a$ 到 $b$ 的面积。

**简单例子**:

- $\int_0^1 x dx = \frac{1}{2}$
- $\int_0^{\pi} \sin x dx = 2$
- $\int_1^e \frac{1}{x} dx = 1$

### 2.2 形式化定义 (L1)

**Riemann积分**: 函数 $f$ 在 $[a,b]$ 上的Riemann积分定义为：

$$\int_a^b f(x)dx = \lim_{||P|| \to 0} \sum_{i=1}^n f(\xi_i)(x_i - x_{i-1})$$

其中 $P$ 是 $[a,b]$ 的分割，$||P||$ 是分割的细度。

**Lebesgue积分**: 更一般的积分理论，基于测度论。

**记号**:

- $\int_a^b f(x)dx$: 定积分
- $\int f(x)dx$: 不定积分
- $\int f$: 简记

## 4. 🔍 性质与定理

### 4.1 基本性质 (L1)

**性质1: 积分的线性性**:

- **和**: $\int (f+g) = \int f + \int g$
- **数乘**: $\int (cf) = c\int f$

**性质2: 积分的区间可加性**:

- **陈述**: $\int_a^b f + \int_b^c f = \int_a^c f$

**性质3: 积分的单调性**:

- **陈述**: 若 $f \leqq g$，则 $\int f \leqq \int g$

### 4.2 重要定理 (L2)

**定理1: 微积分基本定理**:

- **第一部分**: 若 $F' = f$，则 $\int_a^b f = F(b) - F(a)$
- **第二部分**: $\frac{d}{dx}\int_a^x f(t)dt = f(x)$

**定理2: 换元积分法**:

- **陈述**: $\int f(g(x))g'(x)dx = \int f(u)du$，其中 $u = g(x)$

**定理3: 分部积分法**:

- **陈述**: $\int fg' = fg - \int f'g$

## 5. 🔬 形式化证明

### 定理1: 微积分基本定理的形式化证明

**定理陈述（第一部分）**:
$$\forall f \forall a [\text{Cont}(f) \to \forall x (F'(x) = f(x))]$$
其中 $F(x) = \int_a^x f(t) \, dt$

**前提**:

- 连续函数的定义
- 导数的定义
- 积分的定义
- 积分中值定理

**形式化证明**:

```text
步骤1: 假设条件
  设: Cont(f) and F(x) = int_a^x f(t) dt

步骤2: 计算F的导数
  F'(x) = lim_{h->0} (F(x+h) - F(x))/h
        = lim_{h->0} (1/h) int_x^{x+h} f(t) dt

步骤3: 应用积分中值定理
  对充分小的h: exists c in [x, x+h], int_x^{x+h} f(t) dt = f(c)h
  因此: (1/h) int_x^{x+h} f(t) dt = f(c)

步骤4: 取极限
  lim_{h->0} f(c) = f(x)  (连续性: c -> x as h -> 0)
  因此: F'(x) = f(x)

步骤5: 结论
  因此: Cont(f) -> forall x, F'(x) = f(x)
```

**定理陈述（第二部分）**:
$$\forall f \forall F [F' = f \to \int_a^b f(x) \, dx = F(b) - F(a)]$$

**形式化证明**:

```text
步骤1: 假设条件
  设: F' = f

步骤2: 定义辅助函数
  定义: G(x) = int_a^x f(t) dt
  由第一部分: G' = f

步骤3: 比较F和G
  (G - F)' = G' - F' = f - f = 0
  因此: G - F是常数

步骤4: 计算常数
  (G - F)(a) = G(a) - F(a) = 0 - F(a) = -F(a)
  因此: G(x) - F(x) = -F(a)
  即: G(x) = F(x) - F(a)

步骤5: 代入b
  G(b) = F(b) - F(a)
  即: int_a^b f(x) dx = F(b) - F(a)

步骤6: 结论
  因此: F' = f -> int_a^b f(x) dx = F(b) - F(a)
```

## 6. 💡 应用实例

### 6.1 理论应用

- 微积分（积分的计算和应用）
- 实分析（Lebesgue积分）
- 函数分析（$L^p$ 空间）

### 6.2 实际应用

#### 应用1: 物理学 - 功的计算

**问题描述**:
物体在力 $F(x) = 2x + 1$（牛顿）作用下从 $x = 0$ 移动到 $x = 5$（米），求所做的功。

**数学建模**:
功是力对位移的积分：$W = \int_0^5 F(x)dx$

**计算过程**:

- $W = \int_0^5 (2x + 1)dx$
- $W = [x^2 + x]_0^5 = (25 + 5) - (0 + 0) = 30$ 焦耳

**结果解释**:
物体在力作用下移动5米所做的功是30焦耳。

**数据**:

- 位移: 0到5米
- 力函数: $F(x) = 2x + 1$
- 功: 30焦耳

#### 应用2: 工程学 - 面积计算

**问题描述**:
计算曲线 $y = x^2$ 与 $x$ 轴在区间 $[0, 2]$ 围成的面积。

**数学建模**:
面积是函数在区间上的定积分：$A = \int_0^2 x^2 dx$

**计算过程**:

- $A = \int_0^2 x^2 dx = \leqft[\frac{x^3}{3}\right]_0^2 = \frac{8}{3} - 0 = \frac{8}{3}$ 平方单位

**结果解释**:
曲线 $y = x^2$ 与 $x$ 轴在 $[0, 2]$ 围成的面积是 $\frac{8}{3}$ 平方单位。

**数据**:

- 区间: $[0, 2]$
- 函数: $y = x^2$
- 面积: $\frac{8}{3}$ 平方单位

#### 应用3: 经济学 - 累积收益计算

**问题描述**:
某公司的收益率为 $r(t) = 1000 + 50t$（元/年），求前10年的累积收益。

**数学建模**:
累积收益是收益率对时间的积分：$R = \int_0^{10} r(t)dt$

**计算过程**:

- $R = \int_0^{10} (1000 + 50t)dt$
- $R = [1000t + 25t^2]_0^{10} = 10000 + 2500 = 12500$ 元

**结果解释**:
前10年的累积收益是12500元。

**数据**:

- 时间区间: 0到10年
- 收益率函数: $r(t) = 1000 + 50t$
- 累积收益: 12500元

- **类比**: 积分就像"计算曲线下面积"或"累加的连续版本"
- **例子**:
  - 曲线下面积：$\int_a^b f(x)dx$表示曲线$y=f(x)$下从$a$到$b$的面积
  - 体积：$\int_a^b A(x)dx$表示横截面积为$A(x)$的立体体积
  - 累积量：$\int_0^t v(t)dt$表示速度$v(t)$的累积位移

**2. Riemann积分的直观理解**:

- **类比**: Riemann积分就像"用矩形逼近面积"
- **解释**:
  - 将区间分成小段
  - 每段用矩形近似
  - 当分割越来越细时，矩形面积和趋于真实面积

**3. 微积分基本定理的直观理解**:

- **类比**: 微积分基本定理就像"积分和导数是互逆运算"
- **解释**:
  - 第一部分：如果知道导数，可以通过积分计算原函数
  - 第二部分：如果知道积分，可以通过求导得到被积函数

### 认知科学视角

**1. 数学教育家Dienes的观点**:

- **多表征原则**: 通过几何（面积）、物理（累积）、代数（公式）等多种方式理解积分
- **变化性原则**: 通过不同的积分例子理解积分的本质
- **教学启示**: 使用面积可视化、物理类比、符号计算等多种方法

**2. 数学认知学家Tall的观点**:

- **过程-对象对偶**: 理解"积分过程"（如何计算）和"积分"（对象）
- **认知层次**: 从直观理解（"面积"）到形式化理解（Riemann积分定义）

## 9.5 📚 习题库

### L0基础题（5道）

**EX.CORE.016.01** (L0, 计算)

- **题目**: 计算：$\int_0^1 x \, dx$。
- **答案**: $\int_0^1 x \, dx = \leqft[\frac{x^2}{2}\right]_0^1 = \frac{1}{2}$。

**EX.CORE.016.02** (L0, 计算)

- **题目**: 计算：$\int_0^{\pi} \sin x \, dx$。
- **答案**: $\int_0^{\pi} \sin x \, dx = [-\cos x]_0^{\pi} = -(-1) - (-1) = 2$。

**EX.CORE.016.03** (L0, 概念理解)

- **题目**: 用Riemann和计算 $\int_0^1 x^2 \, dx$（使用 $n$ 等分）。
- **答案**: $\int_0^1 x^2 \, dx = \lim_{n \to \infty} \sum_{i=1}^n \leqft(\frac{i}{n}\right)^2 \cdot \frac{1}{n} = \lim_{n \to \infty} \frac{1}{n^3} \sum_{i=1}^n i^2 = \lim_{n \to \infty} \frac{n(n+1)(2n+1)}{6n^3} = \frac{1}{3}$。

**EX.CORE.016.04** (L0, 计算)

- **题目**: 使用换元法计算：$\int_0^1 x(1 + x^2)^2 \, dx$。
- **答案**: 设 $u = 1 + x^2$，则 $du = 2x \, dx$，$\int_0^1 x(1 + x^2)^2 \, dx = \frac{1}{2} \int_1^2 u^2 \, du = \frac{1}{2} \leqft[\frac{u^3}{3}\right]_1^2 = \frac{7}{6}$。

**EX.CORE.016.05** (L0, 应用)

- **题目**: 计算曲线 $y = x^2$ 在 $[0, 1]$ 上的面积。
- **答案**: $\int_0^1 x^2 \, dx = \frac{1}{3}$。

### L1中级题（6道）

**EX.CORE.016.06** (L1, 证明)

- **题目**: 证明：若 $f$ 在 $[a, b]$ 上Riemann可积，则 $f$ 在 $[a, b]$ 上有界。
- **提示**: 使用Riemann可积的定义。
- **答案**: 假设 $f$ 无界，则对任意分割，存在子区间上 $f$ 无界，因此Riemann和可以任意大，矛盾。

**EX.CORE.016.07** (L1, 证明)

- **题目**: 证明微积分基本定理第一部分：若 $F(x) = \int_a^x f(t) \, dt$，则 $F'(x) = f(x)$（$f$ 连续）。
- **提示**: 使用导数的定义。
- **答案**: $F'(x) = \lim_{h \to 0} \frac{F(x+h) - F(x)}{h} = \lim_{h \to 0} \frac{1}{h} \int_x^{x+h} f(t) \, dt = f(x)$（由积分中值定理）。

**EX.CORE.016.08** (L1, 证明)

- **题目**: 证明微积分基本定理第二部分：若 $F' = f$，则 $\int_a^b f(x) \, dx = F(b) - F(a)$。
- **提示**: 使用第一部分。
- **答案**: 设 $G(x) = \int_a^x f(t) \, dt$，则 $G' = f = F'$，因此 $G - F$ 是常数，所以 $G(b) - G(a) = F(b) - F(a)$，即 $\int_a^b f(x) \, dx = F(b) - F(a)$。

**EX.CORE.016.09** (L1, 计算)

- **题目**: 使用分部积分计算：$\int_0^{\pi} x \sin x \, dx$。
- **答案**: $\int_0^{\pi} x \sin x \, dx = [-x \cos x]_0^{\pi} + \int_0^{\pi} \cos x \, dx = \pi + [\sin x]_0^{\pi} = \pi$。

**EX.CORE.016.10** (L1, 证明)

- **题目**: 证明：若 $f$ 在 $[a, b]$ 上连续，则存在 $c \in [a, b]$ 使得 $\int_a^b f(x) \, dx = f(c)(b - a)$（积分中值定理）。
- **提示**: 使用连续函数的最值定理。
- **答案**: 设 $m = \min f$，$M = \max f$，则 $m(b - a) \leqq \int_a^b f(x) \, dx \leqq M(b - a)$，由中间值定理存在 $c \in [a, b]$ 使得 $f(c) = \frac{1}{b-a} \int_a^b f(x) \, dx$。

**EX.CORE.016.11** (L1, 应用)

- **题目**: 计算：$\int_0^1 \frac{1}{\sqrt{1 - x^2}} \, dx$。
- **答案**: 设 $x = \sin t$，则 $\int_0^1 \frac{1}{\sqrt{1 - x^2}} \, dx = \int_0^{\pi/2} \frac{\cos t}{\cos t} \, dt = \frac{\pi}{2}$。

### L2高级题（4道）

**EX.CORE.016.12** (L2, 证明)

- **题目**: 证明：连续函数在 $[a, b]$ 上Riemann可积。
- **提示**: 使用一致连续性。
- **答案**: $f$ 在 $[a, b]$ 上一致连续，因此对任意 $\varepsilon > 0$，存在 $\delta > 0$ 使得当 $|x - y| < \delta$ 时 $|f(x) - f(y)| < \varepsilon$。选择分割使得最大子区间长度 $< \delta$，则上Riemann和与下Riemann和的差 $< \varepsilon(b - a)$，因此 $f$ 可积。

**EX.CORE.016.13** (L2, 证明)

- **题目**: 证明：若 $f$ 在 $[a, b]$ 上单调，则 $f$ 在 $[a, b]$ 上Riemann可积。
- **提示**: 使用单调函数的性质。
- **答案**: 设 $f$ 单调递增，对分割 $P$，上Riemann和与下Riemann和的差 $= \sum (M_i - m_i)\Delta x_i \leqq \max(M_i - m_i) \sum \Delta x_i = (f(b) - f(a))\|P\|$，当 $\|P\| \to 0$ 时趋于 $0$。

**EX.CORE.016.14** (L2, 综合)

- **题目**: 证明：若 $f$ 在 $[a, b]$ 上Riemann可积，$g$ 连续，则 $g \circ f$ 不一定Riemann可积。
- **提示**: 构造反例。
- **答案**: 设 $f$ 是Riemann函数（在有理点取 $1/q$，在无理点取 $0$），$g(x) = x$，则 $f$ 可积但 $g \circ f = f$ 可积。但若 $g$ 不连续，可以构造反例。

**EX.CORE.016.15** (L2, 证明)

- **题目**: 证明：Lebesgue可积函数是Riemann可积当且仅当它的不连续点集是零测集。
- **提示**: 使用Lebesgue判别法。
- **答案**: 这是Lebesgue判别法：函数Riemann可积当且仅当它的不连续点集是Lebesgue零测集。
