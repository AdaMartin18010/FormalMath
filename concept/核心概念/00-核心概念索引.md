## 🔗 三、按知识关联索引 (编号: C.CORE.00.03)

### R1: 依赖关系

**基础数学 → 代数**:

- 集合 → 群、环、域、向量空间
- 函数 → 线性映射
- 数系 → 域（有理数域、实数域、复数域）

**代数内部**:

- 群 → 环（环的加法群）
- 环 → 域（域是特殊的环）
- 域 → 向量空间（向量空间是域上的）
- 向量空间 → 线性映射（线性映射的定义域和值域）

### R2: 等价关系

- Dedekind分割 ↔ Cauchy序列（实数构造）
- 公理化定义 ↔ 集合论构造（自然数、整数、有理数、实数）
- 群定义 ↔ 范畴定义（群、环、域）

### R3: 推广关系

**数系推广**:

- 自然数 → 整数 → 有理数 → 实数 → 复数

**代数结构推广**:

- 群 → 环 → 域
- 向量空间 → 模（一般化到环上）

### R4: 应用关系

**理论应用**:

- 集合 → 所有数学分支的基础
- 函数 → 数学分析、代数、拓扑
- 数系 → 数学分析、数论
- 群 → 代数、几何、拓扑、数论
- 环 → 代数数论、代数几何
- 域 → 代数数论、代数几何、Galois理论
- 向量空间 → 线性代数、函数分析
- 线性映射 → 线性代数、算子理论

**实际应用**:

- 集合 → 数据库理论、计算机科学
- 函数 → 物理学、计算机科学
- 数系 → 测量、金融、工程
- 群 → 物理学、化学、密码学
- 环 → 编码理论、密码学
- 域 → 编码理论、密码学
- 向量空间 → 物理学、工程学、机器学习
- 线性映射 → 物理学、工程学、图像处理
