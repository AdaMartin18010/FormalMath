## 📋 概述

本文档制定FormalMath项目应用实例补充的标准模板，确保每个核心概念都有3-5个高质量的应用实例。

**目标**:
- 为每个核心概念添加3-5个应用实例
- 包含实际数据和计算过程
- 覆盖多个应用领域（物理、工程、计算机等）
- 提供实际问题的解决方案

**标准**:
- 每个实例包含：问题描述、数学建模、计算过程、结果解释
- 实例应具有实际意义和教育价值
- 实例应覆盖不同难度层次

## 📖 示例

### 示例1: 集合的应用实例

```markdown
### 应用实例1: 数据库系统 - 关系数据库查询

**问题描述**:
在关系数据库中，需要查询同时选修"数学"和"物理"课程的学生。

**数学建模**:
设 $A$ 为选修"数学"课程的学生集合，$B$ 为选修"物理"课程的学生集合。
查询结果为 $A \cap B$（两个集合的交集）。

**计算过程**:
- $A = \{Alice, Bob, Charlie, David\}$
- $B = \{Bob, Charlie, Eve, Frank\}$
- $A \cap B = \{Bob, Charlie\}$

**结果解释**:
Bob和Charlie同时选修了"数学"和"物理"课程。

**数据**:
- 学生总数: 100
- 选修数学: 40
- 选修物理: 35
- 同时选修: 15
```

### 示例2: 导数的应用实例

```markdown
### 应用实例1: 物理学 - 速度计算

**问题描述**:
物体沿直线运动，位置函数为 $s(t) = t^2 + 2t$（米），求 $t = 3$ 秒时的速度。

**数学建模**:
速度是位置对时间的导数：$v(t) = s'(t)$

**计算过程**:
- $s(t) = t^2 + 2t$
- $s'(t) = 2t + 2$
- $v(3) = s'(3) = 2(3) + 2 = 8$ 米/秒

**结果解释**:
在 $t = 3$ 秒时，物体的瞬时速度为 8 米/秒。

**数据**:
- 时间: 3秒
- 位置: $s(3) = 15$ 米
- 速度: 8 米/秒
```
