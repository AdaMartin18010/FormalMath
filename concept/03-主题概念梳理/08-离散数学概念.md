## 📚 一、图论核心概念 (编号: C.03.08.02)

### 1.1 图的基本概念

| 概念 | 定义 | 重要性 | 关联概念 |
|-----|------|--------|---------|
| **图** | 由顶点集和边集组成的数学结构 | ⭐⭐⭐⭐⭐ | 顶点、边 |
| | **严格定义**: 设 $V$ 是非空有限集合，$E \subseteq V \times V$，则 $G=(V,E)$ 称为图 | | |
| | **性质**: $n$ 个顶点的简单无向图最多有 $\binom{n}{2}$ 条边 | | |
| **顶点** | 图中的节点 | ⭐⭐⭐⭐⭐ | 度、邻接 |
| | **定义**: 顶点集 $V$ 中的元素，表示图中的基本单元 | | |
| **边** | 连接两个顶点的线段 | ⭐⭐⭐⭐⭐ | 有向边、无向边 |
| | **定义**: 边集 $E$ 中的元素，表示顶点间的连接关系 | | |
| | **无向边**: $(u,v) \in E \Leftrightarrow (v,u) \in E$ | | |
| | **有向边**: $(u,v) \in E$ 表示从 $u$ 指向 $v$ | | |
| **有向图** | 边有方向的图 | ⭐⭐⭐⭐ | 有向边、路径 |
| **无向图** | 边无方向的图 | ⭐⭐⭐⭐⭐ | 边、路径 |
| **简单图** | 无自环、无重边的图 | ⭐⭐⭐⭐ | 图、多重图 |
| **多重图** | 允许重边的图 | ⭐⭐⭐ | 图、简单图 |

### 1.2 图的度与路径

| 概念 | 定义 | 重要性 | 关联概念 |
|-----|------|--------|---------|
| **度** | 顶点连接的边数 | ⭐⭐⭐⭐⭐ | 顶点、边 |
| | **定义**: 在无向图中，$d(v) = |\{u : (u,v) \in E\}|$ | | |
| | **握手引理**: $\sum_{v \in V} d(v) = 2|E|$（每条边被计算两次） | | |
| | **推论**: 度数为奇数的顶点个数为偶数 | | |
| **入度** | 有向图中指向顶点的边数 | ⭐⭐⭐⭐ | 有向图、出度 |
| | **定义**: $d^-(v) = |\{u : (u,v) \in E\}|$ | | |
| **出度** | 有向图中从顶点出发的边数 | ⭐⭐⭐⭐ | 有向图、入度 |
| | **定义**: $d^+(v) = |\{u : (v,u) \in E\}|$ | | |
| | **性质**: $\sum_{v \in V} d^-(v) = \sum_{v \in V} d^+(v) = |E|$ | | |
| **路径** | 顶点序列，相邻顶点有边连接 | ⭐⭐⭐⭐⭐ | 顶点、边、回路 |
| **回路** | 起点和终点相同的路径 | ⭐⭐⭐⭐ | 路径、圈 |
| **圈** | 长度至少为3的简单回路 | ⭐⭐⭐⭐ | 回路、路径 |
| **连通性** | 图中任意两点间存在路径 | ⭐⭐⭐⭐⭐ | 路径、连通图 |

### 1.3 特殊图

| 概念 | 定义 | 重要性 | 关联概念 |
|-----|------|--------|---------|
| **树** | 连通无圈图 | ⭐⭐⭐⭐⭐ | 图、连通性 |
| | **等价定义**: (1) 连通且 $|E| = |V| - 1$; (2) 无圈且 $|E| = |V| - 1$; (3) 任意两点间存在唯一路径 | | |
| | **性质**: $n$ 个顶点的树有 $n-1$ 条边 | | |
| | **证明思路**: 使用数学归纳法，删除一条边后分成两个连通分量，每个都是树 | | |
| **森林** | 无圈图的集合 | ⭐⭐⭐ | 树、图 |
| | **定义**: 若干棵树的并集，$n$ 个顶点 $k$ 个连通分量的森林有 $n-k$ 条边 | | |
| **完全图** | 任意两点间都有边的图 | ⭐⭐⭐⭐ | 图、边 |
| **二分图** | 顶点可分成两个独立集的图 | ⭐⭐⭐⭐ | 图、匹配 |
| **平面图** | 可在平面上无交叉绘制的图 | ⭐⭐⭐⭐ | 图、欧拉公式 |
| **正则图** | 所有顶点度数相同的图 | ⭐⭐⭐ | 图、度 |

### 1.4 图的性质

| 概念 | 定义 | 重要性 | 关联概念 |
|-----|------|--------|---------|
| **欧拉路径** | 经过每条边恰好一次的路径 | ⭐⭐⭐⭐ | 路径、欧拉回路 |
| | **存在性定理**: 连通无向图存在欧拉路径当且仅当恰好有 0 个或 2 个度数为奇数的顶点 | | |
| | **证明思路**: 必要性：除起点和终点外，每个顶点进入和离开次数相等；充分性：使用贪心构造 | | |
| **欧拉回路** | 起点和终点相同的欧拉路径 | ⭐⭐⭐⭐ | 欧拉路径、图 |
| | **存在性定理**: 连通无向图存在欧拉回路当且仅当所有顶点度数为偶数 | | |
| | **应用**: 七桥问题、邮递员问题 | | |
| **哈密顿路径** | 经过每个顶点恰好一次的路径 | ⭐⭐⭐⭐ | 路径、哈密顿回路 |
| **哈密顿回路** | 起点和终点相同的哈密顿路径 | ⭐⭐⭐⭐ | 哈密顿路径、图 |
| **匹配** | 边集，任意两条边不相邻 | ⭐⭐⭐⭐ | 边、二分图 |
| **完美匹配** | 覆盖所有顶点的匹配 | ⭐⭐⭐⭐ | 匹配、顶点 |
| **着色** | 给顶点或边分配颜色 | ⭐⭐⭐⭐ | 顶点、边、色数 |

## 🔢 二、组合数学核心概念 (编号: C.03.08.03)

### 2.1 计数原理

| 概念 | 定义 | 重要性 | 关联概念 |
|-----|------|--------|---------|
| **加法原理** | 互斥事件的总数等于各事件数之和 | ⭐⭐⭐⭐⭐ | 计数、事件 |
| | **严格表述**: 若 $A_i \cap A_j = \emptyset$ ($i \neq j$)，则 $|A_1 \cup \cdots \cup A_k| = \sum |A_i|$ | | |
| | **证明**: 互斥保证每个元素只属于一个集合，因此并集大小等于各集合大小之和 | | |
| **乘法原理** | 独立事件的总数等于各事件数之积 | ⭐⭐⭐⭐⭐ | 计数、事件 |
| | **严格表述**: $k$ 步独立选择，第 $i$ 步有 $n_i$ 种选择，则总数为 $\prod n_i$ | | |
| | **证明思路**: 数学归纳法，每步选择对应前一步的所有可能组合 | | |
| **容斥原理** | 计算并集大小的公式 | ⭐⭐⭐⭐⭐ | 集合、计数 |
| | **两个集合**: $|A \cup B| = |A| + |B| - |A \cap B|$ | | |
| | **一般形式**: $|\bigcup A_i| = \sum |A_i| - \sum |A_i \cap A_j| + \sum |A_i \cap A_j \cap A_k| - \cdots$ | | |
| | **证明思路**: 考虑并集中每个元素被计算的次数，使用二项式定理验证 | | |
| **鸽笼原理** | n+1个物体放入n个容器，必有容器含多个物体 | ⭐⭐⭐⭐⭐ | 计数、存在性 |

### 2.2 排列与组合

| 概念 | 定义 | 重要性 | 关联概念 |
|-----|------|--------|---------|
| **排列** | 从n个元素中取r个的有序排列 | ⭐⭐⭐⭐⭐ | 计数、顺序 |
| | **公式**: $P(n,r) = \frac{n!}{(n-r)!} = n(n-1)\cdots(n-r+1)$ | | |
| | **证明思路**: 第1个位置$n$种选择，第2个位置$n-1$种，...，由乘法原理 | | |
| | **性质**: $P(n,n) = n!$, $P(n,0) = 1$, $P(n,1) = n$ | | |
| **组合** | 从n个元素中取r个的无序选择 | ⭐⭐⭐⭐⭐ | 计数、选择 |
| | **公式**: $C(n,r) = \binom{n}{r} = \frac{n!}{r!(n-r)!} = \frac{P(n,r)}{r!}$ | | |
| | **证明思路**: 每个$r$元素组合对应$r!$个排列，因此组合数=排列数/$r!$ | | |
| | **性质**: 对称性 $\binom{n}{r} = \binom{n}{n-r}$; 递推 $\binom{n}{r} = \binom{n-1}{r-1} + \binom{n-1}{r}$ | | |
| **全排列** | n个元素的所有排列 | ⭐⭐⭐⭐ | 排列、阶乘 |
| **重复排列** | 允许重复的排列 | ⭐⭐⭐ | 排列、计数 |
| **重复组合** | 允许重复的组合 | ⭐⭐⭐ | 组合、计数 |
| **二项式系数** | C(n,k) = n!/(k!(n-k)!) | ⭐⭐⭐⭐⭐ | 组合、二项式定理 |
| | **二项式定理**: $(x+y)^n = \sum_{k=0}^n \binom{n}{k} x^{n-k} y^k$ | | |
| | **证明思路**: 展开$(x+y)^n$，项$x^{n-k}y^k$的系数是从$n$个因子中选$k$个取$y$的组合数 | | |
| | **应用**: $(1+1)^n = 2^n = \sum \binom{n}{k}$; $(1-1)^n = 0 = \sum (-1)^k \binom{n}{k}$ | | |

### 2.3 生成函数

| 概念 | 定义 | 重要性 | 关联概念 |
|-----|------|--------|---------|
| **生成函数** | 将序列编码为形式幂级数 | ⭐⭐⭐⭐ | 序列、级数 |
| **普通生成函数** | 用于组合问题的生成函数 | ⭐⭐⭐⭐ | 生成函数、组合 |
| **指数生成函数** | 用于排列问题的生成函数 | ⭐⭐⭐⭐ | 生成函数、排列 |
| **递推关系** | 序列项之间的关系式 | ⭐⭐⭐⭐⭐ | 序列、递归 |

| 概念 | 定义 | 重要性 | 关联概念 |
|-----|------|--------|---------|
| **斯特林数** | 第一类和第二类斯特林数 | ⭐⭐⭐ | 计数、排列 |
