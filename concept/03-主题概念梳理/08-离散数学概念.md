**哲学背景**: 离散数学的发展体现了逻辑主义、形式主义和建构主义的影响。布尔代数（Boole）体现了逻辑主义的思想，将逻辑代数化。图灵机（Turing）体现了形式主义的思想，用形式系统定义计算。算法理论和计算理论的发展体现了建构主义的要求，强调可计算性和可构造性。详见[数学哲学维度矩阵](../02-知识矩阵/09-数学哲学维度矩阵.md)。

**权威资源对齐**: 本文档的概念定义和结构参考了Wikipedia离散数学条目、MIT 6.042离散数学课程、Stanford的图论和组合数学课程，以及Metamath的逻辑和集合论定理。详见[权威资源对齐说明](../00-权威资源对齐说明.md)和[权威资源对标改进计划](../00-权威资源对标改进计划.md)。

## 📚 一、图论核心概念 (编号: C.03.08.02)

### 1.1 图的基本概念

| 概念 | 定义 | 重要性 | 关联概念 |
|-----|------|--------|---------|
| **图** | 由顶点集和边集组成的数学结构 | ⭐⭐⭐⭐⭐ | 顶点、边 |
| | **严格定义**: 设 $V$ 是非空有限集合，$E \subseteq V \times V$，则 $G=(V,E)$ 称为图 | | |
| | **历史背景**: 图论起源于Euler（1736）的七桥问题，König（1936）的《有限图与无限图理论》建立了现代图论基础 | | |
| | **性质**: $n$ 个顶点的简单无向图最多有 $\binom{n}{2}$ 条边 | | |
| | **应用**: 网络分析、社交网络、算法设计、运筹学 | | |
| | **参考文献**: Wikipedia: [Graph Theory](https://en.wikipedia.org/wiki/Graph_theory); Diestel, R. (2017). *Graph Theory*. Springer. | | |
| **顶点** | 图中的节点 | ⭐⭐⭐⭐⭐ | 度、邻接 |
| | **定义**: 顶点集 $V$ 中的元素，表示图中的基本单元 | | |
| **边** | 连接两个顶点的线段 | ⭐⭐⭐⭐⭐ | 有向边、无向边 |
| | **定义**: 边集 $E$ 中的元素，表示顶点间的连接关系 | | |
| | **无向边**: $(u,v) \in E \Leftrightarrow (v,u) \in E$ | | |
| | **有向边**: $(u,v) \in E$ 表示从 $u$ 指向 $v$ | | |
| **有向图** | 边有方向的图 | ⭐⭐⭐⭐ | 有向边、路径 |
| **无向图** | 边无方向的图 | ⭐⭐⭐⭐⭐ | 边、路径 |
| **简单图** | 无自环、无重边的图 | ⭐⭐⭐⭐ | 图、多重图 |
| **多重图** | 允许重边的图 | ⭐⭐⭐ | 图、简单图 |

### 1.2 图的度与路径

| 概念 | 定义 | 重要性 | 关联概念 |
|-----|------|--------|---------|
| **度** | 顶点连接的边数 | ⭐⭐⭐⭐⭐ | 顶点、边 |
| | **定义**: 在无向图中，$d(v) = \|\{u : (u,v) \in E\}\|$ | | |
| | **握手引理**: $\sum_{v \in V} d(v) = 2\|E\|$（每条边被计算两次） | | |
| | **历史背景**: 握手引理由Euler（1736）在解决七桥问题时使用 | | |
| | **推论**: 度数为奇数的顶点个数为偶数 | | |
| | **应用**: 图的性质分析、存在性证明 | | |
| **入度** | 有向图中指向顶点的边数 | ⭐⭐⭐⭐ | 有向图、出度 |
| | **定义**: $d^-(v) = \|\{u : (u,v) \in E\}\|$ | | |
| **出度** | 有向图中从顶点出发的边数 | ⭐⭐⭐⭐ | 有向图、入度 |
| | **定义**: $d^+(v) = \|\{u : (v,u) \in E\}\|$ | | |
| | **性质**: $\sum_{v \in V} d^-(v) = \sum_{v \in V} d^+(v) = \|E\|$ | | |
| **路径** | 顶点序列，相邻顶点有边连接 | ⭐⭐⭐⭐⭐ | 顶点、边、回路 |
| **回路** | 起点和终点相同的路径 | ⭐⭐⭐⭐ | 路径、圈 |
| **圈** | 长度至少为3的简单回路 | ⭐⭐⭐⭐ | 回路、路径 |
| **连通性** | 图中任意两点间存在路径 | ⭐⭐⭐⭐⭐ | 路径、连通图 |

### 1.3 特殊图

| 概念 | 定义 | 重要性 | 关联概念 |
|-----|------|--------|---------|
| **树** | 连通无圈图 | ⭐⭐⭐⭐⭐ | 图、连通性 |
| | **等价定义**: (1) 连通且 $\|E\| = \|V\| - 1$; (2) 无圈且 $\|E\| = \|V\| - 1$; (3) 任意两点间存在唯一路径 | | |
| | **历史背景**: 树的概念由Cayley（1857）在研究化学分子结构时引入，Kirchhoff（1847）在研究电路时也独立发现 | | |
| | **性质**: $n$ 个顶点的树有 $n-1$ 条边 | | |
| | **证明思路**: 使用数学归纳法，删除一条边后分成两个连通分量，每个都是树 | | |
| | **应用**: 最小生成树、数据结构、算法设计 | | |
| | **参考文献**: Wikipedia: [Tree (Graph Theory)](https://en.wikipedia.org/wiki/Tree_(graph_theory)); Cayley, A. (1857). "On the theory of the analytical forms called trees". | | |
| **森林** | 无圈图的集合 | ⭐⭐⭐ | 树、图 |
| | **定义**: 若干棵树的并集，$n$ 个顶点 $k$ 个连通分量的森林有 $n-k$ 条边 | | |
| **完全图** | 任意两点间都有边的图 | ⭐⭐⭐⭐ | 图、边 |
| **二分图** | 顶点可分成两个独立集的图 | ⭐⭐⭐⭐ | 图、匹配 |
| **平面图** | 可在平面上无交叉绘制的图 | ⭐⭐⭐⭐ | 图、欧拉公式 |
| **正则图** | 所有顶点度数相同的图 | ⭐⭐⭐ | 图、度 |

### 1.4 图的性质

| 概念 | 定义 | 重要性 | 关联概念 |
|-----|------|--------|---------|
| **欧拉路径** | 经过每条边恰好一次的路径 | ⭐⭐⭐⭐ | 路径、欧拉回路 |
| | **存在性定理**: 连通无向图存在欧拉路径当且仅当恰好有 0 个或 2 个度数为奇数的顶点 | | |
| | **历史背景**: 由Euler（1736）在解决Königsberg七桥问题时提出，开创了图论 | | |
| | **证明思路**: 必要性：除起点和终点外，每个顶点进入和离开次数相等；充分性：使用贪心构造 | | |
| | **应用**: 路径规划、网络遍历 | | |
| | **参考文献**: Wikipedia: [Eulerian Path](https://en.wikipedia.org/wiki/Eulerian_path); Euler, L. (1736). "Solutio problematis ad geometriam situs pertinentis". | | |
| **欧拉回路** | 起点和终点相同的欧拉路径 | ⭐⭐⭐⭐ | 欧拉路径、图 |
| | **存在性定理**: 连通无向图存在欧拉回路当且仅当所有顶点度数为偶数 | | |
| | **历史背景**: 由Euler（1736）证明，是图论的第一个重要定理 | | |
| | **应用**: 七桥问题、邮递员问题、电路设计 | | |
| | **参考文献**: Wikipedia: [Eulerian Path](https://en.wikipedia.org/wiki/Eulerian_path) | | |
| **哈密顿路径** | 经过每个顶点恰好一次的路径 | ⭐⭐⭐⭐ | 路径、哈密顿回路 |
| **哈密顿回路** | 起点和终点相同的哈密顿路径 | ⭐⭐⭐⭐ | 哈密顿路径、图 |
| **匹配** | 边集，任意两条边不相邻 | ⭐⭐⭐⭐ | 边、二分图 |
| **完美匹配** | 覆盖所有顶点的匹配 | ⭐⭐⭐⭐ | 匹配、顶点 |
| **着色** | 给顶点或边分配颜色 | ⭐⭐⭐⭐ | 顶点、边、色数 |

## 🔢 二、组合数学核心概念 (编号: C.03.08.03)

### 2.1 计数原理

| 概念 | 定义 | 重要性 | 关联概念 |
|-----|------|--------|---------|
| **加法原理** | 互斥事件的总数等于各事件数之和 | ⭐⭐⭐⭐⭐ | 计数、事件 |
| | **严格表述**: 若 $A_i \cap A_j = \emptyset$ ($i \neq j$)，则 $\|A_1 \cup \cdots \cup A_k\| = \sum \|A_i\|$ | | |
| | **证明**: 互斥保证每个元素只属于一个集合，因此并集大小等于各集合大小之和 | | |
| **乘法原理** | 独立事件的总数等于各事件数之积 | ⭐⭐⭐⭐⭐ | 计数、事件 |
| | **严格表述**: $k$ 步独立选择，第 $i$ 步有 $n_i$ 种选择，则总数为 $\prod n_i$ | | |
| | **证明思路**: 数学归纳法，每步选择对应前一步的所有可能组合 | | |
| **容斥原理** | 计算并集大小的公式 | ⭐⭐⭐⭐⭐ | 集合、计数 |
| | **两个集合**: $\|A \cup B\| = \|A\| + \|B\| - \|A \cap B\|$ | | |
| | **一般形式**: $\|\bigcup A_i\| = \sum \|A_i\| - \sum \|A_i \cap A_j\| + \sum \|A_i \cap A_j \cap A_k\| - \cdots$ | | |
| | **证明思路**: 考虑并集中每个元素被计算的次数，使用二项式定理验证 | | |
| **鸽笼原理** | n+1个物体放入n个容器，必有容器含多个物体 | ⭐⭐⭐⭐⭐ | 计数、存在性 |

### 2.2 排列与组合

| 概念 | 定义 | 重要性 | 关联概念 |
|-----|------|--------|---------|
| **排列** | 从n个元素中取r个的有序排列 | ⭐⭐⭐⭐⭐ | 计数、顺序 |
| | **公式**: $P(n,r) = \frac{n!}{(n-r)!} = n(n-1)\cdots(n-r+1)$ | | |
| | **历史背景**: 排列的研究可以追溯到古代，Pascal（1654）在《算术三角形》中系统研究 | | |
| | **证明思路**: 第1个位置$n$种选择，第2个位置$n-1$种，...，由乘法原理 | | |
| | **性质**: $P(n,n) = n!$, $P(n,0) = 1$, $P(n,1) = n$ | | |
| | **应用**: 密码学、概率论、算法设计 | | |
| | **参考文献**: Wikipedia: [Permutation](https://en.wikipedia.org/wiki/Permutation); Stanley, R. P. (2011). *Enumerative Combinatorics*. Cambridge University Press. | | |
| **组合** | 从n个元素中取r个的无序选择 | ⭐⭐⭐⭐⭐ | 计数、选择 |
| | **公式**: $C(n,r) = \binom{n}{r} = \frac{n!}{r!(n-r)!} = \frac{P(n,r)}{r!}$ | | |
| | **历史背景**: 组合数学起源于Pascal（1654）的算术三角形，Euler（18世纪）进一步发展 | | |
| | **证明思路**: 每个$r$元素组合对应$r!$个排列，因此组合数=排列数/$r!$ | | |
| | **性质**: 对称性 $\binom{n}{r} = \binom{n}{n-r}$; 递推 $\binom{n}{r} = \binom{n-1}{r-1} + \binom{n-1}{r}$（Pascal恒等式） | | |
| | **应用**: 概率论、统计学、算法分析、密码学 | | |
| | **参考文献**: Wikipedia: [Combination](https://en.wikipedia.org/wiki/Combination); Stanley, R. P. (2011). *Enumerative Combinatorics*. Cambridge University Press. | | |
| **全排列** | n个元素的所有排列 | ⭐⭐⭐⭐ | 排列、阶乘 |
| **重复排列** | 允许重复的排列 | ⭐⭐⭐ | 排列、计数 |
| **重复组合** | 允许重复的组合 | ⭐⭐⭐ | 组合、计数 |
| **二项式系数** | C(n,k) = n!/(k!(n-k)!) | ⭐⭐⭐⭐⭐ | 组合、二项式定理 |
| | **二项式定理**: $(x+y)^n = \sum_{k=0}^n \binom{n}{k} x^{n-k} y^k$ | | |
| | **历史背景**: 二项式定理由Newton（1665）发现，是组合数学的基础定理 | | |
| | **证明思路**: 展开$(x+y)^n$，项$x^{n-k}y^k$的系数是从$n$个因子中选$k$个取$y$的组合数 | | |
| | **应用**: $(1+1)^n = 2^n = \sum \binom{n}{k}$; $(1-1)^n = 0 = \sum (-1)^k \binom{n}{k}$ | | |
| | **参考文献**: Wikipedia: [Binomial Theorem](https://en.wikipedia.org/wiki/Binomial_theorem); Newton, I. (1665). "De analysi per aequationes numero terminorum infinitas". | | |

## 🔗 三、离散结构核心概念 (编号: C.03.08.04)

### 3.1 关系与函数

| 概念 | 定义 | 重要性 | 关联概念 |
|-----|------|--------|---------|
| **二元关系** | 两个集合的笛卡尔积的子集 | ⭐⭐⭐⭐⭐ | 集合、关系 |
| **自反关系** | 每个元素与自身相关 | ⭐⭐⭐⭐ | 关系、性质 |
| **对称关系** | 若aRb则bRa | ⭐⭐⭐⭐ | 关系、性质 |
| **传递关系** | 若aRb且bRc则aRc | ⭐⭐⭐⭐ | 关系、性质 |
| **等价关系** | 自反、对称、传递的关系 | ⭐⭐⭐⭐⭐ | 关系、等价类 |
| **偏序关系** | 自反、反对称、传递的关系 | ⭐⭐⭐⭐⭐ | 关系、偏序集 |
| **函数** | 特殊的二元关系 | ⭐⭐⭐⭐⭐ | 关系、映射 |

### 3.2 偏序集与格

#### 偏序集

- **偏序集 (Poset)**: 带有偏序关系的集合
  - **严格定义**: 有序对 $(P, \leq)$，其中 $P$ 是集合，$\leq$ 是 $P$ 上的偏序关系（自反、反对称、传递）
  - **性质**: 偏序集是序理论的基本对象，允许不可比较的元素
  - **应用**: 集合论、代数、拓扑、计算机科学
  - **例子**: $(\mathcal{P}(A), \subseteq)$（幂集的包含关系）、$(\mathbb{N}, \mid)$（自然数的整除关系）

- **全序集 (Totally Ordered Set)**: 任意两元素可比较的偏序集
  - **严格定义**: 偏序集 $(T, \leq)$ 满足 $\forall a,b \in T, a \leq b \lor b \leq a$
  - **性质**: 全序集是偏序集的特殊情况，所有元素都可比较
  - **应用**: 排序算法、有序结构、实数理论
  - **例子**: $(\mathbb{R}, \leq)$（实数的大小关系）

- **良序集 (Well-Ordered Set)**: 每个非空子集有最小元的全序集
  - **严格定义**: 全序集 $(W, \leq)$ 满足 $\forall S \subseteq W (S \neq \emptyset \Rightarrow \exists m \in S, \forall s \in S, m \leq s)$
  - **性质**: 良序集允许超限归纳法
  - **应用**: 序数理论、超限归纳、选择公理
  - **例子**: $(\mathbb{N}, \leq)$（自然数的标准序）

#### 格

- **格 (Lattice)**: 任意两元素有上确界和下确界的偏序集
  - **严格定义**: 偏序集 $(L, \leq)$ 满足 $\forall a,b \in L, \exists \sup\{a,b\}, \exists \inf\{a,b\}$
  - **等价定义**: 格是配备两个二元运算（并 $\vee$ 和交 $\wedge$）的代数结构，满足交换律、结合律、吸收律
  - **性质**: 格是序结构和代数结构的结合
  - **应用**: 布尔代数、逻辑、拓扑、计算机科学
  - **例子**: $(\mathcal{P}(A), \subseteq)$（幂集的包含关系构成格）

- **分配格 (Distributive Lattice)**: 满足分配律的格
  - **严格定义**: 格 $(L, \vee, \wedge)$ 满足 $a \vee (b \wedge c) = (a \vee b) \wedge (a \vee c)$ 和 $a \wedge (b \vee c) = (a \wedge b) \vee (a \wedge c)$
  - **性质**: 分配格是格的子类，具有更好的结构性质
  - **应用**: 布尔代数、逻辑、集合论
  - **例子**: 布尔格、集合的幂集格

- **布尔格 (Boolean Lattice)**: 有补分配格
  - **严格定义**: 分配格 $(B, \vee, \wedge, \neg, 0, 1)$ 满足：
    - 有最大元 $1$ 和最小元 $0$
    - 每个元素 $a$ 有补元 $\neg a$，使得 $a \vee \neg a = 1$ 且 $a \wedge \neg a = 0$
  - **性质**: 布尔格等价于布尔代数
  - **应用**: 逻辑、集合论、计算机科学、电路设计
  - **例子**: $(\mathcal{P}(A), \cup, \cap, ^c, \emptyset, A)$（幂集的布尔格）

- **完备格 (Complete Lattice)**: 任意子集都有上确界和下确界的格
  - **严格定义**: 格 $(L, \leq)$ 满足 $\forall S \subseteq L, \exists \sup S, \exists \inf S$
  - **性质**: 完备格是格的推广，允许无限运算
  - **应用**: 拓扑、域理论、程序语义
  - **例子**: 闭区间的集合、拓扑空间的开集族

## 🎓 学习路径建议 (编号: C.03.08.09)

### 基础路径

1. **离散结构基础** (4-6周)
   - 集合论基础
   - 关系与函数
   - 逻辑基础

2. **图论入门** (6-8周)
   - 图的基本概念
   - 路径与连通性
   - 树与特殊图

3. **组合数学基础** (6-8周)
   - 计数原理
   - 排列与组合
   - 生成函数

### 进阶路径

1. **布尔代数与逻辑电路** (4-6周)
   - 布尔运算
   - 布尔函数
   - 逻辑电路

2. **离散概率** (4-6周)
   - 基本概率
   - 离散随机变量
   - 常见分布

3. **算法理论** (8-10周)
   - 算法基础
   - 算法设计技术
   - 数据结构

### 高级路径

1. **计算理论** (8-10周)
   - 形式语言
   - 自动机
   - 可计算性与复杂性
