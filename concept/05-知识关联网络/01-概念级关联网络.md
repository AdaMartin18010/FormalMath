## 📋 概述 (编号: C.05.01.01)

**核心概念三视角版本**: 本文档中涉及的所有33个核心概念已完成三视角转换，每个概念都包含认知学、教育学、数学家三个视角的深度分析。详见：[核心概念索引](../核心概念/00-核心概念索引.md) ⭐ - 所有33个核心概念的三视角版本

**集合论与范畴论视角**: 本文档中的概念关联网络可以从集合论和范畴论两个视角理解：

- **集合论视角**: 概念之间的关联关系可以用集合关系（包含、交集、并集等）描述，概念层次结构反映了基础概念的组合方式。概念网络中的节点对应概念集合，边对应概念之间的关系集合。详见：[集合论视角的概念层次结构分析](../00-集合论视角的概念层次结构分析.md) ⭐ - 集合论视角的详细分析

- **范畴论视角**: 从范畴论视角来看，每个概念之间的联系都可以被范畴论的概念把握。概念之间的关系（包含关系、推广关系、应用关系、等价关系）可以用范畴论的态射描述，概念之间的变换可以用函子描述，概念关系的动态变化可以用自然变换、极限、伴随等工具描述。概念网络中的节点对应范畴的对象，边对应范畴的态射。详见：[范畴论视角的概念关系动态表征](../00-范畴论视角的概念关系动态表征.md) ⭐ - 范畴论视角的详细分析

- **双视角整合**: 集合论视角描述概念的静态层次结构，范畴论视角描述概念的动态关系变换。两个视角互补，共同构成完整的概念关联网络理解。详见：[集合论与范畴论视角整合分析](../00-集合论与范畴论视角整合分析.md) ⭐ 新增

## 🔗 一、基础数学概念网络 (编号: C.05.01.02)

### 1.1 集合论概念网络

```mermaid
graph TD
    A[集合 C.CORE.001] --> B[子集]
    A --> C[并集]
    A --> D[交集]
    A --> E[补集]
    B --> F[真子集]
    C --> G[幂集]
    A --> H[笛卡尔积]
    H --> I[关系]
    I --> J[函数 C.CORE.002]
    I --> K[等价关系]
    I --> N[序关系]
    K --> L[等价类]
    L --> M[商集]
    N --> O[偏序关系]
    O --> P[偏序集]
    O --> Q[全序关系]
    Q --> R[全序集]
    Q --> S[良序关系]
    S --> T[良序集]
    P --> U[格]
    U --> V[分配格]
    V --> W[布尔代数]

    style A fill:#e1f5ff
    style J fill:#e1f5ff
```

#### 关系说明

- **集合 → 子集**: 子集是集合的特殊情况，$A \subseteq B$ 当且仅当 $\forall x (x \in A \Rightarrow x \in B)$
- **集合 → 并集/交集/补集**: 集合的基本运算，满足交换律、结合律、分配律、De Morgan律
- **集合 → 幂集**: 幂集 $\mathcal{P}(A)$ 是 $A$ 的所有子集构成的集合，$|\mathcal{P}(A)| = 2^{|A|}$
- **笛卡尔积 → 关系**: 关系 $R \subseteq A \times B$ 是笛卡尔积的子集
- **关系 → 函数**: 函数是满足单值性的关系：$\forall a \in A, \exists! b \in B, (a,b) \in f$
- **关系 → 等价关系**: 等价关系满足自反、对称、传递性
- **关系 → 序关系**: 序关系是满足自反、反对称、传递性的关系
- **等价关系 → 等价类**: 等价类 $[a] = \{x : (a,x) \in R\}$ 是等价关系下的元素分类
- **等价类 → 商集**: 商集 $A/R = \{[a] : a \in A\}$ 是等价类的集合，构成 $A$ 的划分
- **序关系 → 偏序关系**: 偏序关系满足自反性、反对称性、传递性；允许不可比较的元素
- **偏序关系 → 偏序集**: 偏序集 $(P, \\leq)$ 是配备偏序关系的集合；序理论的基本对象
- **偏序关系 → 全序关系**: 全序关系是任意两元素都可比较的偏序关系
- **全序关系 → 全序集**: 全序集 $(T, \\leq)$ 是配备全序关系的集合；所有元素都可比较
- **全序关系 → 良序关系**: 良序关系是每个非空子集都有最小元的全序关系
- **良序关系 → 良序集**: 良序集 $(W, \\leq)$ 是配备良序关系的集合；允许超限归纳法
- **偏序集 → 格**: 格是任意两元素都有上确界和下确界的偏序集；序结构和代数结构的结合
- **格 → 分配格**: 分配格是满足分配律的格；具有更好的结构性质
- **分配格 → 布尔代数**: 布尔代数是有补分配格；逻辑和集合论的代数化

## 🔗 二、代数结构概念网络 (编号: C.05.01.03)

### 2.1 群论概念网络

```mermaid
graph TD
    A[群 C.CORE.008] --> B[子群]
    A --> C[群同态]
    B --> D[正规子群]
    D --> E[商群]
    C --> F[群同构]
    A --> G[群作用]
    G --> H[轨道]
    G --> I[稳定子]
    A --> J[循环群]
    A --> K[对称群]
    A --> L[置换群]
    A --> M[表示 C.CORE.032]

    style A fill:#e1f5ff
    style M fill:#e1f5ff
```

#### 关系说明

- **群 → 子群**: 子群 $H \subseteq G$ 是群，满足封闭性和逆元封闭性；判定：$\forall a,b \in H, ab^{-1} \in H$
- **群 → 群同态**: 群同态 $\phi: G \to H$ 保持群运算：$\phi(ab) = \phi(a)\phi(b)$
- **子群 → 正规子群**: 正规子群 $H \trianglelefteq G$ 满足 $\forall g \in G, gHg^{-1} = H$，是构造商群的条件
- **正规子群 → 商群**: 商群 $G/H$ 的运算 $(g_1H)(g_2H) = (g_1g_2)H$ 需要正规性保证良定义
- **群同态 → 群同构**: 群同构是双射的群同态，表示两个群在代数上完全相同
- **群 → 群作用**: 群作用 $G \times X \to X$ 将群元素映射为集合上的变换
- **群作用 → 轨道**: 轨道 $G \cdot x = \{g \cdot x : g \in G\}$ 是 $x$ 在群作用下的像
- **群作用 → 稳定子**: 稳定子 $G_x = \{g \in G : g \cdot x = x\}$ 是保持 $x$ 不变的群元素
- **群 → 循环群**: 循环群由单个元素生成，$G = \langle a \rangle = \{a^n : n \in \mathbb{Z}\}$
- **群 → 对称群**: 对称群 $S_n$ 是 $n$ 个元素的集合上所有置换构成的群

#### 关系说明

- **环 → 子环**: 子环 $S \subseteq R$ 是环，满足封闭性和逆元封闭性；判定：$\forall a,b \in S, a-b, ab \in S$
- **环 → 环同态**: 环同态 $\phi: R \to S$ 保持环运算：$\phi(a+b) = \phi(a) + \phi(b)$，$\phi(ab) = \phi(a)\phi(b)$
- **环 → 理想**: 理想 $I \trianglelefteq R$ 是子环，满足 $\forall r \in R, a \in I, ra, ar \in I$；是构造商环的条件
- **理想 → 主理想**: 主理想 $(a) = \{ra : r \in R\}$ 由单个元素生成
- **理想 → 素理想**: 素理想 $P$ 满足：若 $ab \in P$，则 $a \in P$ 或 $b \in P$；对应整环
- **理想 → 极大理想**: 极大理想 $M$ 是极大真理想；$R/M$ 是域
- **理想 → 商环**: 商环 $R/I$ 的运算 $(a+I)(b+I) = ab+I$ 需要理想保证良定义
- **环 → 整环**: 整环是无零因子的交换环；$ab = 0 \Rightarrow a = 0$ 或 $b = 0$
- **整环 → 域**: 域是每个非零元素都有乘法逆元的整环；$R$ 是域当且仅当 $(0)$ 和 $R$ 是仅有的理想
- **环 → 交换环**: 交换环满足 $ab = ba$；大多数环论研究交换环

#### 关系说明

- **域 → 子域**: 子域 $F \subseteq K$ 是域；$K$ 是 $F$ 的扩域
- **域 → 域扩张**: 域扩张 $K/F$ 是包含关系 $F \subseteq K$；$K$ 是 $F$ 上的向量空间
- **域扩张 → 代数扩张**: 代数扩张中每个元素都是代数元（是某个多项式的根）
- **域扩张 → 超越扩张**: 超越扩张中存在超越元（不是任何多项式的根）
- **代数扩张 → 有限扩张**: 有限扩张 $[K:F] < \infty$；有限扩张是代数扩张
- **代数扩张 → Galois扩张**: Galois扩张是正规且可分的有限扩张
- **Galois扩张 → Galois群**: Galois群 $\text{Gal}(K/F) = \{\sigma \in \text{Aut}(K) : \sigma|_F = \text{id}\}$ 是自同构群

### 2.4 线性代数概念网络 ⭐ 新增

```mermaid
graph TD
    A[向量空间 C.CORE.011] --> B[子空间]
    A --> C[基]
    A --> D[维数]
    A --> E[线性映射 C.CORE.012]
    E --> F[核]
    E --> G[像]
    E --> H[秩]
    E --> I[矩阵表示]
    A --> J[线性组合]
    A --> K[线性无关]
    C --> L[标准基]
    C --> M[正交基]

    style A fill:#e1f5ff
    style E fill:#e1f5ff
```

#### 关系说明

- **向量空间 → 子空间**: 子空间 $W \subseteq V$ 是向量空间，满足封闭性；判定：$\forall u,v \in W, a \in F, u+v, au \in W$
- **向量空间 → 基**: 基是线性无关的生成集；所有基有相同的元素个数（维数）
- **向量空间 → 维数**: 维数 $\dim V$ 是基的元素个数；有限维向量空间的分类
- **向量空间 → 线性映射**: 线性映射 $T: V \to W$ 保持线性组合：$T(au+bv) = aT(u) + bT(v)$
- **线性映射 → 核**: 核 $\ker(T) = \{v \in V : T(v) = 0\}$ 是 $V$ 的子空间
- **线性映射 → 像**: 像 $\text{im}(T) = \{T(v) : v \in V\}$ 是 $W$ 的子空间
- **线性映射 → 秩**: 秩 $\text{rank}(T) = \dim \text{im}(T)$；秩-零度定理：$\dim V = \text{rank}(T) + \dim \ker(T)$

#### 关系说明（代数-几何关联）

- **环 → 概形**: 概形 $\text{Spec}(R)$ 是环 $R$ 的素理想集合，配备Zariski拓扑和结构层；代数几何的基础
- **概形 → 代数簇**: 代数簇是概形的特殊情况；由多项式方程定义的几何对象
- **环 → 理想**: 理想对应概形的闭子集；$V(I) = \{p \in \text{Spec}(R) : I \subseteq p\}$
- **理想 → 代数集**: 代数集 $V(I)$ 是理想 $I$ 的零点集；Hilbert零点定理
- **代数集 → 代数簇**: 代数簇是不可约的代数集；代数几何的基本对象
- **群 → 群概形**: 群概形是概形上的群结构；群作用、群表示
- **域 → 函数域**: 函数域是代数簇上的有理函数域；代数几何的重要工具
- **函数域 → 代数簇**: 函数域描述代数簇的几何性质；维数、奇点

- **函数 → 流形上的函数**: 流形上的函数 $f: M \to \mathbb{R}$ 是光滑函数；定义在流形上
