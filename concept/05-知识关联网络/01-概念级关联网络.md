## 🔗 一、基础数学概念网络 (编号: C.05.01.02)

### 1.1 集合论概念网络

```mermaid
graph TD
    A[集合] --> B[子集]
    A --> C[并集]
    A --> D[交集]
    A --> E[补集]
    B --> F[真子集]
    C --> G[幂集]
    A --> H[笛卡尔积]
    H --> I[关系]
    I --> J[函数]
    I --> K[等价关系]
    K --> L[等价类]
    L --> M[商集]
```

#### 关系说明

- **集合 → 子集**: 子集是集合的特殊情况，$A \subseteq B$ 当且仅当 $\forall x (x \in A \Rightarrow x \in B)$
- **集合 → 并集/交集/补集**: 集合的基本运算，满足交换律、结合律、分配律、De Morgan律
- **集合 → 幂集**: 幂集 $\mathcal{P}(A)$ 是 $A$ 的所有子集构成的集合，$|\mathcal{P}(A)| = 2^{|A|}$
- **笛卡尔积 → 关系**: 关系 $R \subseteq A \times B$ 是笛卡尔积的子集
- **关系 → 函数**: 函数是满足单值性的关系：$\forall a \in A, \exists! b \in B, (a,b) \in f$
- **关系 → 等价关系**: 等价关系满足自反、对称、传递性
- **等价关系 → 等价类**: 等价类 $[a] = \{x : (a,x) \in R\}$ 是等价关系下的元素分类
- **等价类 → 商集**: 商集 $A/R = \{[a] : a \in A\}$ 是等价类的集合，构成 $A$ 的划分

## 🔗 二、代数结构概念网络 (编号: C.05.01.03)

### 2.1 群论概念网络

```mermaid
graph TD
    A[群] --> B[子群]
    A --> C[群同态]
    B --> D[正规子群]
    D --> E[商群]
    C --> F[群同构]
    A --> G[群作用]
    G --> H[轨道]
    G --> I[稳定子]
    A --> J[循环群]
    A --> K[对称群]
    A --> L[置换群]
```

#### 关系说明

- **群 → 子群**: 子群 $H \subseteq G$ 是群，满足封闭性和逆元封闭性；判定：$\forall a,b \in H, ab^{-1} \in H$
- **群 → 群同态**: 群同态 $\phi: G \to H$ 保持群运算：$\phi(ab) = \phi(a)\phi(b)$
- **子群 → 正规子群**: 正规子群 $H \trianglelefteq G$ 满足 $\forall g \in G, gHg^{-1} = H$，是构造商群的条件
- **正规子群 → 商群**: 商群 $G/H$ 的运算 $(g_1H)(g_2H) = (g_1g_2)H$ 需要正规性保证良定义
- **群同态 → 群同构**: 群同构是双射的群同态，表示两个群在代数上完全相同
- **群 → 群作用**: 群作用 $G \times X \to X$ 将群元素映射为集合上的变换
- **群作用 → 轨道**: 轨道 $G \cdot x = \{g \cdot x : g \in G\}$ 是 $x$ 在群作用下的像
- **群作用 → 稳定子**: 稳定子 $G_x = \{g \in G : g \cdot x = x\}$ 是保持 $x$ 不变的群元素
- **群 → 循环群**: 循环群由单个元素生成，$G = \langle a \rangle = \{a^n : n \in \mathbb{Z}\}$
- **群 → 对称群**: 对称群 $S_n$ 是 $n$ 个元素的集合上所有置换构成的群
