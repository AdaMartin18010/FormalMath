## 📋 概述

本目录包含FormalMath项目的全面概念梳理体系，通过思维导图、知识矩阵、主题概念梳理和认知工具，系统化地组织和呈现数学知识结构。

**创建日期**: 2025年11月21日
**目标**: 建立完整的知识认知体系，帮助理解数学知识的整体结构和关联关系

**权威资源对齐**: 本概念体系与当前网络上的权威资源对齐，包括Wikipedia、国际著名大学（斯坦福、普林斯顿、MIT、哈佛等）的数学课程、Metamath形式化数学项目等。详见[权威资源对齐说明](./00-权威资源对齐说明.md)。

## 🎯 使用指南

### 1. 总体思维导图

- **用途**: 快速了解数学知识体系的整体结构
- **适用场景**: 初学者入门、知识体系概览、学习路径规划

### 2. 知识矩阵

- **用途**: 多维度分析数学概念之间的关系
- **适用场景**: 深入研究、关联分析、知识发现
- **扩展内容**:
  - 历史发展矩阵：了解数学分支的历史演进
  - 研究热点矩阵：掌握当前研究动态和前沿问题
  - 时间维度矩阵：规划学习时间和理解理论成熟度

### 3. 主题概念梳理

- **用途**: 系统梳理各数学分支的核心概念
- **适用场景**: 专题学习、概念理解、知识巩固

### 4. 认知工具

- **用途**: 提供思维模型和学习方法
- **适用场景**: 学习方法优化、知识组织、记忆增强
- **扩展内容**:
  - 学习策略工具详细版：5种系统化学习策略
  - 记忆与理解工具详细版：5种记忆与理解方法
  - 问题解决工具：系统化的问题解决框架

### 5. 知识关联网络

- **用途**: 展示知识之间的复杂关联关系
- **适用场景**: 知识发现、路径规划、交叉学习
- **扩展内容**:
  - 概念级关联网络：细粒度的概念关联分析
  - 定理级关联网络：定理之间的依赖关系
  - 方法级关联网络：方法之间的关联关系

### 6. 文档主题分析

- **用途**: 全面分析docs文件夹中所有文档的主题结构
- **适用场景**: 文档导航、主题查找、知识梳理
