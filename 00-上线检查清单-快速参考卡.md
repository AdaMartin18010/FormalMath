## 🔍 快速诊断

### 网站无法访问
```bash
nslookup formalmath.org
curl -I https://formalmath.org
kubectl get pods -n formalmath
```

### 页面加载慢
```bash
curl -s -w "Total: %{time_total}s\n" https://formalmath.org
kubectl top pods -n formalmath
```

### API错误
```bash
kubectl logs -f deployment/api -n formalmath | grep ERROR
```

### 数据库问题
```sql
-- 检查连接数
SELECT count(*) FROM pg_stat_activity;

-- 检查慢查询
SELECT query, mean_time FROM pg_stat_statements ORDER BY mean_time DESC LIMIT 5;
```
