## 5. 双语 Lean4 代码使用方案

### 5.1 使用原则

- **Python 优先应用，Lean4 优先严谨**：计算密集型任务（SVD 数值计算、图像压缩）用 Python；定理声明与证明结构理解用 Lean4。
- **课上**：投影展示 Lean4 的**类型签名和注释**，重点解释数学假设如何被编码为类型约束。
- **课后**：学生在本地 Lean4 环境中打开项目文件，运行 `lake build`，完成预设的 `sorry` 填空。

### 5.2 课上投影片段示例

**片段 1：Eckart-Young-Mirsky 定理的形式化声明**

```lean
/-- Eckart-Young-Mirsky 定理（谱范数形式）：
    截断到前 k 个奇异值的矩阵 A_k 是最佳秩 k 逼近 -/
theorem eckart_young_mirsky_spectral
    (A : Matrix (Fin m) (Fin n) ℝ) (k : ℕ) (hk : k > 0) :
    ∃ (A_k : Matrix (Fin m) (Fin n) ℝ),
      Matrix.rank A_k ≤ k ∧
      (∀ B : Matrix (Fin m) (Fin n) ℝ, Matrix.rank B ≤ k →
        ‖A - A_k‖ ≤ ‖A - B‖) := by
  sorry  -- 需要依赖 SVD 的具体构造和算子范数的性质
```

**讲解要点**：
- `Matrix (Fin m) (Fin n) ℝ` 表示 m×n 实矩阵；
- `Matrix.rank A_k ≤ k` 是"秩不超过 k"的形式化表达；
- `‖A - A_k‖ ≤ ‖A - B‖` 对应数学中的最佳逼近；
- `sorry` 代表 Mathlib4 中尚未有直接可用的简单接口，是当代形式化数学的前沿问题之一。

**片段 2：PCA 主成分的形式化声明**

```lean
/-- PCA 主成分方向对应数据矩阵 X 的前 k 个左奇异向量 -/
theorem pca_principal_components
    (X : Matrix (Fin d) (Fin n) ℝ) (k : ℕ) :
    ∃ (U : Matrix (Fin d) (Fin d) ℝ) (Σ : Matrix (Fin d) (Fin n) ℝ)
      (V : Matrix (Fin n) (Fin n) ℝ),
      X = U * Σ * V.transpose ∧
      (U.transpose * U = 1) ∧
      (V.transpose * V = 1) ∧
      IsDiagonal Σ ∧
      PrincipalComponents X (U.submatrix (·) (Finset.range k)) := by
  sorry
```

### 5.3 课后实验任务单

**Python 实验 1：SVD 几何可视化（60 min）**

1. 用 NumPy 生成矩阵 A = [[3, 0], [0, 1]]，计算其 SVD：U, Σ, Vᵀ。
2. 绘制单位圆及其在 A 作用下的像（椭圆），并标出长轴和短轴方向。
3. 验证：椭圆的长半轴长度等于最大奇异值 σ₁，短半轴长度等于 σ₂。
4. 对矩阵 A = [[2, 1], [1, 2]] 重复上述步骤。

**Python 实验 2：图像压缩与降噪（60 min）**

1. 从项目提供的 `data/lena_gray.png` 读取 512×512 灰度图像。
2. 对图像矩阵 A 做 SVD：A = UΣVᵀ。
3. 分别取 k = 10, 50, 100 构造低秩逼近 Aₖ，显示重构图像。
4. 计算并记录：
   - 压缩比 = k(m + n + 1) / (mn)
   - 相对 Frobenius 误差 = ‖A − Aₖ‖_F / ‖A‖_F
   - 实际值与理论值 √(Σᵢ₌ₖ₊₁ʳ σᵢ²) / √(Σᵢ₌₁ʳ σᵢ²) 对比
5. （选做）向图像添加高斯噪声，观察截断 SVD 的降噪效果。

**Lean4 实验：阅读与填空（60 min）**

1. 打开 `docs/02-代数结构/线性代数与矩阵理论/MIT-18.06/12-奇异值分解的应用.md`，找到文末的 Lean4 代码块。
2. 将代码复制到本地 Lean4 项目的 `SVDApplication.lean` 文件中。
3. 运行 `lake build`，确认编译通过（`sorry` 不会导致编译错误）。
4. 在 `eckart_young_mirsky_spectral` 前添加注释，用中文说明该定理的数学含义。
5. （选做）尝试用 Mathlib4 中的 `Matrix.rank` 和范数相关引理，将 `sorry` 替换为更具体的证明骨架（不要求完全证明）。
