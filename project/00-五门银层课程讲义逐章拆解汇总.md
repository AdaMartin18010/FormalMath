## 3. P1-T3 严重缺失区汇总（银层攻坚优先级）

以下 4 项为 P1-T3 内容审计中发现的**致命短板**，必须在银层文档生产启动前优先修复：

### 3.1 MIT 18.06 L4–L6 完全缺失 🔴 P0

- **证据**：`project/` 目录下不存在 `MIT-18.06-L4-定理对齐表.md`、`MIT-18.06-L5-习题对齐表.md`、`MIT-18.06-L6-思想方法提炼.md`。
- **影响**：18.06 是本科线性代数标杆课程，L4–L6 缺失导致无法进入定理证明、习题解答与教学方法文档的生产。
- **修复动作**：
  1. 生成 `MIT-18.06-L3-定义对齐表` 并补全关联矩阵、傅里叶矩阵、稳定/不稳定子空间等缺失定义；
  2. 依次生成 L4、L5、L6 对齐表；
  3. 优先覆盖 Ch 3–Ch 6（向量空间、正交性、行列式、特征值）核心定理。

### 3.2 MIT 18.100A L4 定理完整率仅 16.7% 🔴 P0

- **证据**：`project/MIT-18.100A-L4-定理对齐表.md` 统计：12 道核心定理中仅 2 道（中间值定理、中值定理）为 ✅，其余 10 道为 ⏳。
- **影响**：实分析银层文档缺少收敛性、紧致性、一致收敛等核心证明，学生无法完成自学。
- **修复动作**：
  1. 补充单调收敛定理、Bolzano-Weierstrass 定理、Cauchy 准则、Heine-Borel 定理、一致收敛保持连续性/积分等证明文档；
  2. 同步修复 L3 定义缺失（可数集、limsup/liminf、一致连续等）。

### 3.3 Harvard 232br L5 习题覆盖率仅 ~4.5% 🔴 P0

- **证据**：`project/Harvard-232br-L5-习题对齐表.md` 显示，课程以 Hartshorne Ch II–V 习题为核心（约 80+ 道精选习题），但项目文档中仅有极少数（如 AG-EX-001、AG-EX-002）有完整对应解答。
- **影响**：研究生代数几何课程缺少可动手练习的解答资源，无法支撑资格考试复习。
- **修复动作**：
  1. 按 Hartshorne 章节批量建立习题解答文档（优先 Ch II Schemes、Ch III Cohomology）；
  2. 为 Riemann-Roch、Serre 对偶、Hurwitz 定理等设计配套计算题。

### 3.4 Stanford FOAG Part VI (Ch 23–29) 证明极度稀缺 🟡 P1

- **证据**：`project/Stanford-FOAG-L6-思想方法提炼.md` 对导出函子、平坦性、深度、Cohen-Macaulayness、形式函数定理、Serre 对偶等仅有哲学/历史叙述，缺少手写数学证明。
- **影响**：FOAG 后半部分（研究生第二、三学期内容）在项目中的数学密度极低。
- **修复动作**：
  1. 为 Part VI 各章建立独立的定理证明文档；
  2. 优先补充“形式函数定理”与“Serre 对偶”的完整证明。

## 4. 跨课程共性缺口

| 缺口类型 | 影响课程 | 说明 |
|----------|----------|------|
| **定义层缺失** | 18.100A、18.701、18.06、FOAG | 大量基础或进阶定义（如一致连续、双线性型、关联矩阵、stalk）在项目文档中完全缺失。 |
| **证明完整率低** | 全部 5 门 | 除少数定理（如 18.100A 的 IVT/MVT）外，大多数核心定理处于“仅有陈述”或“证明骨架”状态。 |
| **习题解答稀缺** | 18.06、232br、FOAG | 18.06 与 232br 的习题对齐表覆盖率最低；18.701 与 FOAG 次之。 |
| **思想方法提炼不足** | 18.100A、FOAG | L6 文档多停留在历史/哲学叙述，缺少“技巧卡片”与“常见错误警示”。 |

## 5. 后续银层生产建议

### 5.1 生产顺序建议

基于缺失严重度与课程基础重要性，建议按以下优先级启动银层文档生产：

1. **MIT 18.06**（最高）：必须先补齐 L4–L6，因为它是本科工程/数学的通用基础，且目前为**零对齐**状态。
2. **MIT 18.100A**（次高）：L4 定理完整率最低（16.7%），但 syllabus 与 L3 已有较好基础，适合快速补证。
3. **Harvard 232br**（高）：L5 习题覆盖率仅 4.5%，需大量习题生产，但 L3–L4 已有一定骨架。
4. **MIT 18.701**（中）：Ch 8–9 定义与证明有缺口，但群论与线性代数部分相对完整。
5. **Stanford FOAG**（持续）：Part I–V 可逐步推进，Part VI 作为研究生高阶内容可稍延后。

### 5.2 资源调配建议

- **定理证明文档**：优先由有分析/代数几何背景的子代理负责，严格依据公开教材（Lebl、Artin、Strang、Hartshorne、Vakil）撰写。
- **习题解答文档**：可采用“教材习题 + 课程 Problem Set”双源对齐策略，确保解答与课程要求一致。
- **Lean4 形式化**：对于 18.06 与 18.701，可并行推进 Mathlib4 路径验证；对于 232br 与 FOAG，形式化优先级可适当降低，先保证手写数学内容的完整性。
