# 学生知情同意书

## 研究目的

本研究旨在评估 FormalMath 银层核心课程（MIT 18.06/18.100A/18.701/Harvard 232br/Stanford FOAG）在真实课堂环境中的教学效果，特别是 Lean4 双语对照学习模式对学生数学理解深度的影响。

## 研究流程

1. **前测**（30分钟）：线性代数/实分析/抽象代数基础概念测试
2. **课程学习**（8周）：使用 FormalMath 银层材料进行学习
3. **后测**（30分钟）：与前测平行的测试题
4. **访谈**（可选，20分钟）：学习体验与反馈

## 数据使用

- 所有数据仅用于学术研究，匿名化处理后发表
- 学生可随时退出，不影响课程成绩
- 原始数据保存期限为研究结束后5年
