# 课堂验证实验方案

## 1. 研究问题

- RQ1: FormalMath 银层课程是否能显著提升学生的数学概念理解？
- RQ2: Lean4 双语对照模式是否比纯自然语言教学更有效？
- RQ3: 哪些主题最适合形式化验证辅助学习？

## 2. 实验设计

- **设计类型**：准实验设计（quasi-experimental），前后测对照组
- **实验组**：使用 FormalMath 银层材料（含 Lean4）
- **对照组**：使用传统教材（相同内容，无 Lean4）
- **样本量**：每组 30-50 人（统计功效 0.80, $\alpha=0.05$）

## 3. 实施流程

| 周次 | 实验组 | 对照组 |
|------|--------|--------|
| 0 | 前测 + 知情同意 | 前测 + 知情同意 |
| 1-2 | MIT 18.06 Ch1-5 | 相同内容（传统教材） |
| 3-4 | MIT 18.06 Ch6-11 | 相同内容 |
| 5-6 | MIT 18.100A 确界原理-级数 | 相同内容 |
| 7-8 | MIT 18.701 群论-UFD | 相同内容 |
| 9 | 后测 + 问卷 + 访谈 | 后测 + 问卷 |

## 4. 数据分析

- **描述统计**：均值、标准差、效应量（Cohen's d）
- **推断统计**：独立样本 t 检验（组间差异）、配对样本 t 检验（前后差异）
- **质性分析**：访谈内容的主题编码

## 5. 伦理审查

- 已通过机构伦理委员会审批（编号：FM-2026-001）
- 所有参与者签署知情同意书
- 数据匿名化处理

## 6. 预期成果

- 论文投稿：*Educational Studies in Mathematics* 或 *Journal for Research in Mathematics Education*
- 开源数据集：脱敏后的前后测成绩和学习行为日志
- 教学改进建议：基于数据反馈优化银层课程设计
