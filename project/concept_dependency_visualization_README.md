# FormalMath 概念依赖网络可视化

本目录包含 FormalMath 项目概念依赖关系的可视化图表和分析报告。

## 📊 生成文件概览

### 1. 层次结构图
**文件**: `concept_hierarchy.mmd`

展示所有382个概念按学习层级（L0-L13）的组织结构。层级基于概念的依赖关系计算，入度为0的概念位于第0层，后续概念根据前置概念的最大层级加1确定。

**使用方式**: 在支持Mermaid的编辑器（如GitHub、VS Code、Typora）中打开即可渲染。

### 2. 关键路径图
**文件**: `concept_critical_path.mmd`

展示四大数学分支的核心概念依赖链：
- **分析学核心**: 极限 → 连续性 → 导数 → 积分 → 级数 → 测度论 → 泛函分析
- **代数学核心**: 群 → 环 → 域 → 向量空间 → 线性代数 → 模论 → 同调代数
- **几何拓扑核心**: 拓扑空间 → 流形 → 黎曼度量 → 曲率
- **数论高级核心**: 集合论 → 同余 → 椭圆曲线 → 模形式 → Langlands纲领

### 3. 分支依赖图
**文件**: `concept_branch_dependency.mmd`

展示21个数学分支之间的依赖关系：
- **核心基础分支**: 分析学、代数学、几何拓扑、数论逻辑
- **应用分支**: 概率统计、最优化、控制论、信息论、计算数学
- **交叉分支**: 范畴论、同调代数、代数几何、表示论、数论高级、动力系统、数理逻辑
- **专业应用**: 密码学、金融数学、生物数学、物理数学、数据科学

依赖强度分为三级：强依赖(>=6)、中等依赖(3-5)、弱依赖(<=2)。

### 4. 学习路径推荐图
**文件**: `concept_learning_path.mmd`

提供四条推荐的学习路线：
- **数学分析之路**: 从极限到泛函分析的完整分析学路径
- **代数结构之路**: 从集合论到代数几何的代数学路径
- **几何拓扑之路**: 从拓扑空间到辛几何的几何学路径
- **数论探索之路**: 从整除到Langlands纲领的数论路径

每条路径标注了概念难度等级(D1-D5)。

### 5. 分析报告
**文件**: `concept_dependency_analysis_report.md`

包含以下分析内容：
- 总体统计数据
- 层级分布分析
- 各分支统计（概念数、平均难度、学习时长）
- 枢纽概念识别
- 基础概念和高级概念识别
- 分支间依赖关系分析
- 学习路径建议

## 🔧 使用方法

### 查看Mermaid图表

1. **GitHub**: 直接在浏览器中打开.mmd文件
2. **VS Code**: 安装Mermaid插件后预览
3. **在线编辑器**: 复制内容到 [Mermaid Live Editor](https://mermaid.live/)[需更新]
4. **Typora/Obsidian**: 直接渲染显示

### 使用JSON数据

```python
import json

with open('concept_dependency_network_summary.json', 'r', encoding='utf-8') as f:
    data = json.load(f)

# 获取统计数据
print(data['statistics'])

# 获取L0层概念
print(data['concepts_by_level']['L0'])

# 获取枢纽概念
print(data['hub_concepts'])
```
