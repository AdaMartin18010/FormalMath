## 课程概览

### 课程描述

本课程是Harvard代数几何研究生课程序列的第二部分（232ar为第一部分），聚焦凝聚层上同调、Grothendieck对偶、曲线与曲面的几何。课程采用概形语言的现代观点，强调上同调技术的系统应用。

### 先修要求

- Harvard Math 232ar（或同等代数几何入门）
- 交换代数基础（Atiyah-Macdonald水平）
- 同调代数基础（导出范畴、谱序列）

**证明树示例**:
```
Grothendieck谱序列
│
├─ 输入: 左正合函子 F: A→B, G: B→C
│
├─ 关键: 导出函子 RF, RG, R(G∘F)
│
├─ 条件: F映内射对象为G-非循环对象
│
└─ 结论: E₂^{p,q} = R^pG(R^qF(X)) ⇒ R^{p+q}(G∘F)(X)
```

**重要定理**:

**Serre消失定理**:
> 设X为射影概形，O_X(1)为极丰富线丛，F为凝聚层。则对足够大的n，有：
> $$H^i(X, F(n)) = 0, \quad \forall i > 0$$

**应用**: 
- 射影概形的高阶上同调控制
- Riemann-Roch定理的证明

- `docs/13-代数几何/03-上同调理论/06-Riemann-Roch定理-深度版.md`

**Serre对偶（曲线情形）**:
> 设X为光滑射影曲线，K_X为典则除子。则对任意除子D：
> $$H^0(X, O_X(D)) \cong H^1(X, O_X(K_X-D))^*$$

**与Riemann-Roch的关系**:
$$\chi(O_X(D)) = \deg(D) + 1 - g$$
