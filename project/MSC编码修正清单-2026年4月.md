## 一、格式修正（全局替换）

### 1.1 小写"xx" → 大写"XX"

**影响范围**: 约975个编码，76种不同编码

**修正方法**:

```powershell
# PowerShell命令
Get-ChildItem -Path "e:\_src\FormalMath" -Recurse -Filter "*.md" | ForEach-Object {
    (Get-Content $_.FullName -Raw) -replace 'xx', 'XX' | Set-Content $_.FullName -NoNewline
}
```

**注意**: 执行前请备份，此命令会替换所有"xx"，需要确认不会影响其他内容。

**建议安全方法**:

```python
import os
import re

for root, dirs, files in os.walk('e:/_src/FormalMath'):
    for f in files:
        if f.endswith('.md'):
            fp = os.path.join(root, f)
            with open(fp, 'r', encoding='utf-8') as file:
                content = file.read()

            # 仅替换MSC编码中的xx
            new_content = re.sub(r'^(msc_primary|msc_secondary):\s*\[?["\']?\d{2}[A-Z]([a-z]{2})["\']?\]?',
                                lambda m: m.group(0).replace('xx', 'XX').replace('x', 'X'),
                                content, flags=re.MULTILINE)

            if new_content != content:
                with open(fp, 'w', encoding='utf-8') as file:
                    file.write(new_content)
```

## 四、MSC编码规范建议

### 4.1 编码选择原则

1. **主要编码 (msc_primary)**
   - 选择最能代表文档核心主题的3位分类
   - 如文档涉及多个主题，选择最核心的一个
   - 格式: `XX-XX` 或 `XXXxx`

2. **次要编码 (msc_secondary)**
   - 列出与文档相关的其他主题
   - 最多3-5个
   - 按相关性排序

3. **特殊情况**
   - 综述/概述文档: 使用 00A05
   - 历史/传记文档: 使用 01-XX
   - 教育/教学文档: 使用 97-XX
