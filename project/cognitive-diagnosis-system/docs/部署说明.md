# FormalMath认知诊断系统 - 部署说明

## 系统要求

### 硬件要求
- CPU: 2核及以上
- 内存: 4GB及以上
- 存储: 20GB可用空间

### 软件要求
- Python 3.10+
- Node.js 18+
- PostgreSQL 14+
- Redis 7+

## 部署步骤

### 1. 后端部署

#### 1.1 安装依赖
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

#### 1.2 配置环境变量
创建 `.env` 文件：
```env
DATABASE_URL=postgresql://postgres:password@localhost:5432/formalmath_cds
REDIS_URL=redis://localhost:6379/0
JWT_SECRET_KEY=your-secret-key
DEBUG=False
```

#### 1.3 初始化数据库
```bash
# 创建数据库
createdb formalmath_cds

# 运行迁移
alembic upgrade head
```

#### 1.4 启动服务
```bash
# 开发模式
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 生产模式
uvicorn app.main:app --host 0.0.0.0 --port 8000 --workers 4
```

### 2. 前端部署

#### 2.1 安装依赖
```bash
cd frontend
npm install
```

#### 2.2 配置API地址
编辑 `.env` 文件：
```env
VITE_API_URL=http://localhost:8000/api/v1
```

#### 2.3 构建
```bash
npm run build
```

#### 2.4 部署
将 `dist` 目录部署到Web服务器（Nginx/Apache）。

### 3. Docker部署

#### 3.1 构建镜像
```bash
docker-compose build
```

#### 3.2 启动服务
```bash
docker-compose up -d
```

## 配置说明

### 后端配置 (config.py)

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| DATABASE_URL | 数据库连接URL | postgresql://... |
| REDIS_URL | Redis连接URL | redis://... |
| JWT_SECRET_KEY | JWT密钥 | - |
| HCD_MAX_ITER | HCD算法最大迭代次数 | 100 |
| DEFAULT_QUESTION_COUNT | 默认题目数量 | 30 |

### 前端配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| API_BASE_URL | API基础地址 | /api/v1 |

## 监控与日志

### 日志配置
- 后端日志: `logs/app.log`
- 访问日志: `logs/access.log`
- 错误日志: `logs/error.log`

### 健康检查
```bash
curl http://localhost:8000/health
```

## 故障排除

### 数据库连接失败
1. 检查PostgreSQL服务是否运行
2. 验证数据库连接URL
3. 检查防火墙设置

### Redis连接失败
1. 检查Redis服务是否运行
2. 验证Redis连接URL

### 前端API请求失败
1. 检查后端服务是否运行
2. 验证API地址配置
3. 检查CORS配置
