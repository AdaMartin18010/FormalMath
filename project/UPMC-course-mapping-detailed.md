# FormalMath与巴黎第六大学(UPMC)数学课程深度对齐报告

> **版本**: 2026年4月
> **适用范围**: UPMC (Université Pierre et Marie Curie) - 现索邦大学理学院
> **目标**: 与法国Bourbaki学派传统对齐，建立FormalMath与法国精英数学教育体系的映射

---

## 📋 概览

### 巴黎第六大学简介

```
历史背景：
- 成立于1971年，继承巴黎大学科学学院
- 2018年与巴黎四大合并为索邦大学
- 法国最大的科学和医学大学
- 位于巴黎拉丁区，Jussieu校区

数学传统：
- Bourbaki学派的发源地和核心
- 严格的公理化方法传统
- 结构主义数学观
- 纯数学与应用数学并重

著名数学家：
- Henri Cartan (Bourbaki创始人之一)
- André Weil (Bourbaki创始人之一)
- Jean-Pierre Serre (Fields奖1954)
- Alain Connes (Fields奖1982)
- Pierre-Louis Lions (Fields奖1994)
- Wendelin Werner (Fields奖2006)
- Cédric Villani (Fields奖2010)
- Hugo Duminil-Copin (Fields奖2022)
```

### Bourbaki学派风格分析

| 特征 | 描述 | FormalMath对应 |
|------|------|----------------|
| **严格公理化** | 从公理出发，逻辑推导 | `docs/01-基础数学/ZFC公理体系/` |
| **结构主义** | 强调代数/序/拓扑三大母结构 | `docs/02-代数结构/`、`docs/05-拓扑学/` |
| **高度抽象** | 在一般框架下处理问题 | `数学家理念体系/格洛腾迪克数学理念/` |
| **符号精确** | 精确的数学语言，避免含糊直观 | `docs/00-术语标准化词典-国际标准版.md` |
| **统一处理** | 相同结构统一处理 | `docs/00-概念关联图谱/` |
