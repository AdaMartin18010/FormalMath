## 课程概览

### 课程描述

本课程涵盖实分析基础：序列和级数的收敛性、连续性、可微性、Riemann积分、函数序列和级数、一致收敛性、极限运算的交换。

### 先修要求

- 单变量微积分（18.01）
- 基本证明写作经验

- 免费下载: https://www.jirka.org/ra/

- Bartle & Sherbert "Introduction to Real Analysis"

#### 定义对齐细节

**定义1: 序列收敛**

MIT 18.100A表述:
> A sequence {xₙ} converges to L if for every ε > 0, there exists N ∈ ℕ such that for all n ≥ N, |xₙ - L| < ε.

FormalMath对应:

```markdown
定义（序列收敛）：设{xₙ}为数列，L∈ℝ。若对任意ε>0，存在N∈ℕ，
使得当n≥N时，有|xₙ - L|<ε，则称{xₙ}收敛于L，记作 lim_{n→∞} xₙ = L。
```

等价性: ✅ 严格等价
差异: 无
