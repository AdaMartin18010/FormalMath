# 课堂验证伦理审查与数据管理协议

## 1. 研究性质声明

本课堂验证属于**教育干预研究**（educational intervention study），目的是评估 FormalMath 项目银层课程文档在真实教学环境中的学习效果。研究不涉及医学、心理学敏感内容，不收集个人隐私信息。

## 2. 学生知情同意书模板

---

**FormalMath 课堂验证知情同意书**

亲爱的同学：

欢迎你参加 FormalMath 课堂验证项目。在接下来的 2-3 周课程中，我们将使用一套新的数学教学文档（含完整证明和形式化代码对照）替代部分传统教材内容。

**你的权利**：

- 参与完全自愿，你可以随时退出，不影响课程成绩
- 所有测试数据和问卷反馈将**匿名化处理**
- 你的个人信息（姓名、学号）不会出现在任何研究报告中

**数据用途**：

- 仅用于改进教学文档质量和撰写学术研究论文
- 研究成果可能在教育技术会议或期刊上发表

## 3. 数据匿名化处理方案

- **去标识化**：在数据录入时立即用随机编号替代学生姓名和学号
- **敏感信息过滤**：问卷中的开放题回答将人工审核，去除可能泄露身份的具体描述
- **数据存储**：匿名化数据存储在项目内部 Git 仓库的加密分支中
- **访问控制**：只有核心研究团队可以访问原始数据

## 4. 学习分析数据收集范围

| 数据类型 | 收集方式 | 匿名化 | 用途 |
|----------|---------|--------|------|
| 前测/后测分数 | 纸质/在线测试 | 是 | 评估学习效果 |
| 习题完成率 | 作业提交系统 | 是 | 评估文档可用性 |
| Lean4 编译尝试次数 | 在线 Lean 环境日志 | 是 | 评估形式化辅助效果 |
| 文档阅读时长 | 学习平台日志（如有） | 是 | 评估内容吸引力 |
| 学生问卷 | 纸质/在线问卷 | 是 | 收集质性反馈 |
| 焦点小组访谈录音 | 录音+转录 | 是 | 深入理解学习体验 |

## 5. 退出机制

学生可在任何时候口头或书面通知教师退出验证，退出后：

- 其后续数据不再被收集
- 已收集的匿名化数据可选择性删除
- 不影响课程参与和成绩评定
