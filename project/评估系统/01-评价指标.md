## 1. 指标体系概述

### 1.1 设计原则

FormalMath评估系统的指标体系遵循以下原则：

| 原则 | 描述 |
|------|------|
| **科学性** | 基于数学教育理论和认知科学 |
| **全面性** | 覆盖知识、技能、能力、思维多个维度 |
| **可操作性** | 每个指标都有明确的测量方法 |
| **发展性** | 关注成长轨迹，而非仅看绝对水平 |

### 1.2 指标体系架构

```
┌─────────────────────────────────────────────────────────────────┐
│                    综合评价指数 (CEI)                           │
```

## 2. 知识掌握度 (30%)

### 2.1 指标定义

知识掌握度评估学生对数学知识的理解、记忆和应用能力。

### 2.2 子指标体系

```yaml
知识掌握度:
  权重: 0.30
  
  概念理解度 (40%):
    定义: 对数学概念的准确理解和解释能力
    
    测量维度:
      - 概念识别:
          描述: 识别概念实例的能力
          测量方法: 概念判断题正确率
          权重: 0.25
          
      - 概念解释:
          描述: 用自己的语言解释概念
          测量方法: 概念解释题评分
          评分标准:
            - 优秀(0.9-1.0): 解释准确、完整、有深度
            - 良好(0.7-0.89): 解释准确、基本完整
            - 合格(0.5-0.69): 解释基本正确但不完整
            - 不合格(<0.5): 解释错误或不理解
          权重: 0.35
          
      - 概念联系:
          描述: 建立概念间联系的能力
          测量方法: 概念关联题得分
          权重: 0.25
          
      - 概念应用:
          描述: 在新情境中应用概念
          测量方法: 概念应用题正确率
          权重: 0.15
    
    计算公式: |
      概念理解度 = 0.25×概念识别 + 0.35×概念解释 + 
                  0.25×概念联系 + 0.15×概念应用
  
  定理掌握度 (35%):
    定义: 对数学定理的条件、结论和证明的掌握程度
    
    测量维度:
      - 定理陈述:
          描述: 准确陈述定理
          测量方法: 定理填空/陈述题
          权重: 0.30
          
      - 条件理解:
          描述: 理解定理成立的条件
          测量方法: 条件判断题
          权重: 0.25
          
      - 定理证明:
          描述: 理解或构造定理证明
          测量方法: 证明题得分
          权重: 0.25
          
      - 定理应用:
          描述: 正确应用定理解决问题
          测量方法: 应用题正确率
          权重: 0.20
    
    计算公式: |
      定理掌握度 = 0.30×定理陈述 + 0.25×条件理解 +
                  0.25×定理证明 + 0.20×定理应用
  
  方法熟练度 (25%):
    定义: 对数学方法和算法的熟练掌握
    
    测量维度:
      - 方法识别:
          描述: 识别适用方法的能力
          测量方法: 方法选择题
          权重: 0.20
          
      - 方法执行:
          描述: 正确执行方法步骤
          测量方法: 步骤分得分
          权重: 0.35
          
      - 方法选择:
          描述: 选择最优方法的能力
          测量方法: 开放题方法适当性评分
          权重: 0.25
          
      - 方法迁移:
          描述: 将方法应用于新情境
          测量方法: 变式题正确率
          权重: 0.20
    
    计算公式: |
      方法熟练度 = 0.20×方法识别 + 0.35×方法执行 +
                  0.25×方法选择 + 0.20×方法迁移
```

### 2.3 计算公式

```
知识掌握度 = 0.40 × 概念理解度 + 0.35 × 定理掌握度 + 0.25 × 方法熟练度

等级划分:
- 优秀: [0.90, 1.00]
- 良好: [0.75, 0.89)
- 中等: [0.60, 0.74)
- 及格: [0.50, 0.59)
- 不及格: [0.00, 0.50)
```

## 3. 技能熟练度 (25%)

### 3.1 指标定义

技能熟练度评估学生运用数学技能解决问题的熟练程度。

### 3.2 子指标体系

```yaml
技能熟练度:
  权重: 0.25
  
  计算能力 (35%):
    定义: 准确、快速进行数学运算的能力
    
    测量维度:
      - 计算准确性:
          描述: 计算结果的正确率
          测量方法: 计算题正确率
          权重: 0.45
          
      - 计算速度:
          描述: 完成计算的效率
          测量方法: 单位时间完成题数
          标准: 基础题30秒/题，中等题90秒/题
          权重: 0.30
          
      - 计算策略:
          描述: 选择简便方法的能力
          测量方法: 最优策略使用率
          权重: 0.25
    
    计算公式: |
      计算能力 = 0.45×准确性 + 0.30×速度 + 0.25×策略
  
  证明能力 (40%):
    定义: 构造数学证明的能力
    
    测量维度:
      - 证明完整性:
          描述: 证明步骤的完整性
          测量方法: 证明题步骤分
          权重: 0.25
          
      - 证明严谨性:
          描述: 逻辑推理的严密性
          测量方法: 逻辑错误扣分
          权重: 0.30
          
      - 证明方法多样性:
          描述: 掌握多种证明方法
          测量方法: 证明方法使用种类
          权重: 0.20
          
      - 证明简洁性:
          描述: 证明的简洁优美程度
          测量方法: 与标准证明对比
          权重: 0.15
          
      - 反例构造:
          描述: 构造反例的能力
          测量方法: 反例题得分
          权重: 0.10
    
    计算公式: |
      证明能力 = 0.25×完整性 + 0.30×严谨性 + 0.20×多样性 +
                0.15×简洁性 + 0.10×反例构造
  
  建模能力 (25%):
    定义: 将实际问题转化为数学问题的能力
    
    测量维度:
      - 问题理解:
          描述: 理解实际问题的能力
          测量方法: 问题重述准确性
          权重: 0.25
          
      - 模型建立:
          描述: 建立数学模型的能力
          测量方法: 模型适当性评分
          权重: 0.35
          
      - 模型求解:
          描述: 求解数学模型的能力
          测量方法: 求解过程分
          权重: 0.25
          
      - 结果解释:
          描述: 将结果解释为实际问题答案
          测量方法: 答案解释评分
          权重: 0.15
    
    计算公式: |
      建模能力 = 0.25×问题理解 + 0.35×模型建立 +
                0.25×模型求解 + 0.15×结果解释
```

### 3.3 计算公式

```
技能熟练度 = 0.35 × 计算能力 + 0.40 × 证明能力 + 0.25 × 建模能力
```

## 4. 问题解决能力 (25%)

### 4.1 指标定义

问题解决能力评估学生面对数学问题时分析问题、选择策略、执行解决的能力。

### 4.2 子指标体系

```yaml
问题解决能力:
  权重: 0.25
  
  问题分析 (30%):
    定义: 理解问题、分解问题的能力
    
    测量维度:
      - 问题理解:
          描述: 准确理解问题要求
          测量方法: 问题复述准确性
          权重: 0.35
          
      - 信息提取:
          描述: 提取关键信息的能力
          测量方法: 关键信息识别率
          权重: 0.30
          
      - 问题分解:
          描述: 将复杂问题分解为子问题
          测量方法: 分解策略适当性
          权重: 0.20
          
      - 目标明确:
          描述: 明确问题的目标
          测量方法: 目标陈述准确性
          权重: 0.15
    
    计算公式: |
      问题分析 = 0.35×问题理解 + 0.30×信息提取 +
                0.20×问题分解 + 0.15×目标明确
  
  策略选择 (35%):
    定义: 选择适当解题策略的能力
    
    测量维度:
      - 策略识别:
          描述: 识别可用策略的能力
          测量方法: 策略列举完整性
          权重: 0.25
          
      - 策略评估:
          描述: 评估策略优劣的能力
          测量方法: 策略比较合理性
          权重: 0.30
          
      - 策略选择:
          描述: 选择最优策略的能力
          测量方法: 所选策略最优性
          权重: 0.30
          
      - 备选策略:
          描述: 准备备选方案的意识
          测量方法: 多策略准备情况
          权重: 0.15
    
    计算公式: |
      策略选择 = 0.25×策略识别 + 0.30×策略评估 +
                0.30×策略选择 + 0.15×备选策略
  
  执行与验证 (35%):
    定义: 准确执行并验证结果的能力
    
    测量维度:
      - 执行准确性:
          描述: 准确执行解题步骤
          测量方法: 过程分得分
          权重: 0.35
          
      - 执行效率:
          描述: 高效完成解题
          测量方法: 解题时间
          权重: 0.20
          
      - 结果验证:
          描述: 验证结果正确性
          测量方法: 验算完整性
          权重: 0.25
          
      - 错误修正:
          描述: 发现并修正错误
          测量方法: 自纠能力
          权重: 0.20
    
    计算公式: |
      执行与验证 = 0.35×执行准确性 + 0.20×执行效率 +
                  0.25×结果验证 + 0.20×错误修正
```

### 4.3 计算公式

```
问题解决能力 = 0.30 × 问题分析 + 0.35 × 策略选择 + 0.35 × 执行与验证
```

## 5. 创新思维能力 (20%)

### 5.1 指标定义

创新思维能力评估学生的数学创新思维，包括发散性、创造性和批判性思维。

### 5.2 子指标体系

```yaml
创新思维能力:
  权重: 0.20
  
  发散性 (35%):
    定义: 多角度、多方向思考问题的能力
    
    测量维度:
      - 思维流畅性:
          描述: 产生大量想法的能力
          测量方法: 一题多解数量
          标准: 基础解法+2种以上为优
          权重: 0.30
          
      - 思维灵活性:
          描述: 思维方向转换的能力
          测量方法: 不同思路类型数
          权重: 0.25
          
      - 思维广度:
          描述: 涉及知识领域的广度
          测量方法: 跨分支方法使用
          权重: 0.25
          
      - 视角多样性:
          描述: 从不同角度看待问题
          测量方法: 视角转换题得分
          权重: 0.20
    
    计算公式: |
      发散性 = 0.30×流畅性 + 0.25×灵活性 + 0.25×广度 + 0.20×多样性
  
  创造性 (40%):
    定义: 产生新颖、独特想法的能力
    
    测量维度:
      - 解法新颖性:
          描述: 解法的创新程度
          测量方法: 与标准解法差异度
          权重: 0.30
          
      - 方法独创性:
          描述: 独创方法的能力
          测量方法: 原创方法评分
          权重: 0.25
          
      - 问题重构:
          描述: 重构问题的能力
          测量方法: 问题变式质量
          权重: 0.20
          
      - 拓展推广:
          描述: 结果推广的能力
          测量方法: 推广深度
          权重: 0.15
          
      - 猜想提出:
          描述: 提出数学猜想
          测量方法: 猜想质量评分
          权重: 0.10
    
    计算公式: |
      创造性 = 0.30×新颖性 + 0.25×独创性 + 0.20×问题重构 +
              0.15×拓展推广 + 0.10×猜想提出
  
  批判性 (25%):
    定义: 批判性分析和评价的能力
    
    测量维度:
      - 质疑能力:
          描述: 提出质疑的能力
          测量方法: 质疑题得分
          权重: 0.25
          
      - 评价能力:
          描述: 评价解法优劣
          测量方法: 评价题评分
          权重: 0.30
          
      - 反思能力:
          描述: 反思解题过程
          测量方法: 反思日志质量
          权重: 0.25
          
      - 改进能力:
          描述: 提出改进方案
          测量方法: 改进建议质量
          权重: 0.20
    
    计算公式: |
      批判性 = 0.25×质疑能力 + 0.30×评价能力 +
              0.25×反思能力 + 0.20×改进能力
```

### 5.3 计算公式

```
创新思维能力 = 0.35 × 发散性 + 0.40 × 创造性 + 0.25 × 批判性
```

## 6. 综合评价指数 (CEI)

### 6.1 计算模型

```
综合评价指数 (CEI) = 
    0.30 × 知识掌握度 +
    0.25 × 技能熟练度 +
    0.25 × 问题解决能力 +
    0.20 × 创新思维能力
```

### 6.2 等级标准

| 等级 | CEI范围 | 描述 | 建议 |
|------|---------|------|------|
| S (卓越) | [0.95, 1.00] | 全面优秀，表现突出 | 挑战更高层次内容 |
| A (优秀) | [0.85, 0.95) | 整体优秀，个别方面可提升 | 针对性强化 |
| B (良好) | [0.70, 0.85) | 整体良好，有明显优势 | 保持优势，补齐短板 |
| C (中等) | [0.55, 0.70) | 中等水平，有提升空间 | 系统学习 |
| D (待提高) | [0.40, 0.55) | 需要努力，基础待加强 | 夯实基础 |
| F (需帮助) | [0.00, 0.40) | 需要额外支持和指导 | 一对一辅导 |

## 8. 指标应用

### 8.1 个人层面

- **学习诊断**: 识别个人优势和薄弱环节
- **目标设定**: 基于现状设定合理目标
- **路径规划**: 根据指标缺口规划学习

### 8.2 教学层面

- **学情分析**: 了解班级整体情况
- **教学调整**: 针对薄弱环节调整教学
- **效果评估**: 评估教学改进效果

### 8.3 系统层面

- **内容推荐**: 基于指标推荐学习内容
- **难度调节**: 根据能力调整内容难度
- **进度控制**: 监控学习进度是否合理
