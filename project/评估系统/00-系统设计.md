## 1. 系统概述

### 1.1 设计目标

FormalMath评估系统是一个**多元化、过程化、发展性**的学习评估平台，旨在：

- **全面评估**: 多维度评估学习效果，不仅关注结果更关注过程
- **促进发展**: 通过增值评价，关注学生的进步和成长
- **即时反馈**: 提供及时、具体、可操作的学习反馈
- **支持决策**: 为教学决策和学习规划提供数据支持

### 1.2 评估理念

基于《深化新时代教育评价改革总体方案》的指导，本系统采用：

| 评估类型 | 关注点 | 实施方式 |
|----------|--------|----------|
| **过程性评价** | 学习过程、参与程度 | 实时跟踪学习行为 |
| **增值评价** | 能力增长、进步幅度 | 对比前后测数据 |
| **表现性评价** | 实际问题解决能力 | 情境任务评估 |
| **发散思维评价** | 创新思维、独特见解 | 开放性任务评估 |

```yaml
评价维度:

    权重: 0.30

      - 方法熟练度:
          描述: 解题方法的熟练掌握
          测量: 方法选择正确率 + 解题速度
  
  技能熟练度:
    权重: 0.25
    子指标:
      - 计算能力:
          描述: 准确快速完成计算
          测量: 计算题正确率 + 平均用时
      - 证明能力:
          描述: 构造数学证明的能力
          测量: 证明题完成率 + 证明严谨性评分
      - 建模能力:
          描述: 将实际问题转化为数学问题
          测量: 应用题得分 + 建模步骤完整性
  
  问题解决能力:
    权重: 0.25
    子指标:
      - 问题分析:
          描述: 理解问题、分解问题的能力
          测量: 问题分解准确性 + 关键信息提取
      - 策略选择:
          描述: 选择合适解题策略的能力
          测量: 策略选择正确率
      - 执行与验证:
          描述: 准确执行并验证结果
          测量: 过程分 + 结果正确率
  
  创新思维能力:
    权重: 0.20
    子指标:
      - 发散性:
          描述: 多角度思考问题的能力
          测量: 一题多解数量 + 解法独特性
      - 创造性:
          描述: 提出新颖解法的能力
          测量: 创新性解法评分
      - 批判性:
          描述: 批判性分析问题和解法
          测量: 问题质疑深度 + 改进建议质量
```

### 2.3 过程性评价模块

#### 评价模型

```python
class ProcessEvaluationModel:
    """过程性评价模型"""
    
    def evaluate(self, learner_id: str, period: str) -> ProcessEvaluationResult:
        """
        执行过程性评价
        
        Args:
            learner_id: 学习者ID
            period: 评价周期 (week, month, quarter)
        """
        # 1. 获取学习行为数据
        behavior_data = self.fetch_behavior_data(learner_id, period)
        
        # 2. 计算各项指标
        participation = self.evaluate_participation(behavior_data)
        initiative = self.evaluate_initiative(behavior_data)
        consistency = self.evaluate_consistency(behavior_data)
        engagement = self.evaluate_engagement(behavior_data)
        
        # 3. 生成评价结果
        return ProcessEvaluationResult(
            participation=participation,
            initiative=initiative,
            consistency=consistency,
            engagement=engagement,
            overall_score=self._calculate_overall(
                participation, initiative, consistency, engagement
            )
        )
    
    def evaluate_participation(self, data: LearningBehaviorData) -> ParticipationScore:
        """评估学习参与度"""
        return ParticipationScore(
            time_score=min(data.session_duration / 3600, 1.0),  # 最高1小时满分
            frequency_score=min(data.weekly_frequency / 5, 1.0),  # 每周5次满分
            content_breadth=min(data.content_pages_viewed / 20, 1.0),
            depth_score=data.videos_completion_rate
        )
    
    def evaluate_initiative(self, data: LearningBehaviorData) -> InitiativeScore:
        """评估学习主动性"""
        # 主动学习行为占比
        total_actions = len(data.interactions)
        active_actions = sum(1 for i in data.interactions 
                           if i.action_type in ['note_create', 'search', 'question_ask'])
        
        return InitiativeScore(
            exploration_score=active_actions / max(total_actions, 1),
            questioning_score=min(data.questions_asked / 3, 1.0),
            note_taking_score=min(data.notes_taken / 5, 1.0)
        )
```

### 2.4 增值评价模块

#### 增值计算模型

```
增值评价核心公式:

1. 绝对增值:
   ΔA = A_post - A_pre
   
   其中:
   - A_pre: 前测能力水平
   - A_post: 后测能力水平

2. 相对增值:
   ΔA_rel = (A_post - A_pre) / A_pre × 100%

3. 标准化增值分 (SGP - Student Growth Percentile):
   SGP = percentile(ΔA_student, ΔA_peers)
   
   学生的增值在同侪中的百分位排名

4. 增值效应量 (Cohen's d):
   d = (M_post - M_pre) / SD_pooled
```

#### 增值追踪

```python
class ValueAddedTracker:
    """增值追踪器"""
    
    def track_growth(self, learner_id: str, start_date: date, end_date: date) -> GrowthReport:
        """追踪学习者在指定期间的成长"""
        
        # 获取期初诊断结果
        baseline = self.get_diagnosis_result(learner_id, start_date)
        
        # 获取期末诊断结果
        current = self.get_diagnosis_result(learner_id, end_date)
        
        # 计算各领域增值
        growth = {}
        for dimension in ['knowledge', 'skill', 'problem_solving', 'creativity']:
            growth[dimension] = {
                'baseline': baseline[dimension],
                'current': current[dimension],
                'absolute_gain': current[dimension] - baseline[dimension],
                'relative_gain': (current[dimension] - baseline[dimension]) / baseline[dimension],
                'growth_level': self._classify_growth(
                    current[dimension] - baseline[dimension]
                )
            }
        
        # 生成成长报告
        return GrowthReport(
            learner_id=learner_id,
            period=(start_date, end_date),
            growth_data=growth,
            trajectory=self._analyze_trajectory(learner_id, start_date, end_date),
            projection=self._project_future(growth)
        )
    
    def _classify_growth(self, gain: float) -> str:
        """对增值进行分类"""
        if gain < -0.1:
            return "退步"
        elif gain < 0.05:
            return "停滞"
        elif gain < 0.15:
            return "缓慢增长"
        elif gain < 0.3:
            return "稳健增长"
        else:
            return "快速增长"
```

### 3.2 反馈生成算法

```python
class FeedbackGenerator:
    """智能反馈生成器"""
    
    def generate_feedback(self, evaluation_result: EvaluationResult) -> Feedback:
        """基于评价结果生成反馈"""
        
        # 1. 识别关键发现
        findings = self._identify_findings(evaluation_result)
        
        # 2. 生成反馈内容
        feedback_parts = []
        
        # 强项肯定
        strengths = findings['strengths']
        if strengths:
            feedback_parts.append(self._generate_strength_feedback(strengths))
        
        # 弱项提醒与建议
        weaknesses = findings['weaknesses']
        if weaknesses:
            feedback_parts.append(self._generate_improvement_feedback(weaknesses))
        
        # 进步认可
        growth = findings['growth']
        if growth and growth > 0:
            feedback_parts.append(self._generate_growth_feedback(growth))
        
        # 下一步行动建议
        next_actions = self._generate_next_actions(evaluation_result)
        
        return Feedback(
            summary=". ".join(feedback_parts),
            strengths=strengths,
            weaknesses=weaknesses,
            next_actions=next_actions,
            encouragement=self._generate_encouragement(evaluation_result),
            resources=self._recommend_resources(weaknesses)
        )
    
    def _generate_strength_feedback(self, strengths: List[str]) -> str:
        """生成强项反馈"""
        templates = [
            "您在{strength}方面表现出色，继续保持！",
            "值得称赞的是您的{strength}，这是您的优势。",
            "{strength}是您的一大亮点，可以在此基础上进一步发展。"
        ]
        import random
        return random.choice(templates).format(strength="、".join(strengths))
    
    def _generate_improvement_feedback(self, weaknesses: List[Dict]) -> str:
        """生成改进反馈"""
        priority_weakness = weaknesses[0]
        return (
            f"建议重点关注{priority_weakness['dimension']}，"
            f"特别是{priority_weakness['aspect']}方面。"
            f"可以通过{priority_weakness['suggestion']}来提升。"
        )
    
    def _generate_encouragement(self, result: EvaluationResult) -> str:
        """生成鼓励话语"""
        overall = result.overall_score
        
        if overall > 0.8:
            return "您的学习状态非常好，相信您能取得更大的成就！"
        elif overall > 0.6:
            return "您的进步有目共睹，继续努力一定会有更大突破！"
        elif result.growth > 0:
            return "虽然还有提升空间，但您的进步让我们看到了希望，继续加油！"
        else:
            return "学习是一个过程，不要灰心，我们一起找到适合您的方法！"
```

## 6. 数据模型

### 6.1 核心实体

```python
# 评价记录
class EvaluationRecord:
    id: str
    learner_id: str
    evaluation_type: str      # process|growth|performance|creative
    period_start: datetime
    period_end: datetime
    scores: Dict[str, float]  # 各项得分
    feedback: Feedback
    created_at: datetime

# 学习行为记录
class BehaviorRecord:
    id: str
    learner_id: str
    session_id: str
    behavior_type: str
    timestamp: datetime
    metadata: Dict           # 行为详情
    
# 成长轨迹
class GrowthTrajectory:
    learner_id: str
    dimension: str           # 评价维度
    data_points: List[{
        "date": datetime,
        "score": float,
        "context": str
    }]
    trend_line: List[float]  # 趋势线
    projection: List[float]  # 预测

# 报告
class Report:
    id: str
    type: str
    target_id: str           # 学生ID或班级ID
    period: str
    content: Dict            # 报告内容
    file_urls: Dict          # 各格式文件URL
    generated_at: datetime
```
