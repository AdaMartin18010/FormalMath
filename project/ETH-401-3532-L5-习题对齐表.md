## 1. ETH 401-3532习题体系概述

### 1.1 ETH Zurich习题结构

ETH 401-3532采用"Strenger Stil"（严格风格）教学传统，习题强调**严格证明**与**技术计算**的深度融合：

| 类型 | 德语标记 | 说明 | 占比 | ETH特色 |
|------|---------|------|------|---------|
| **严格证明型** | *Strenge Beweis* | 完整形式化证明 | 40% | 德国数学传统 |
| **技术计算型** | *Technische Berechnung* | 显式计算与验证 | 30% | 苏黎世计算传统 |
| **综合分析型** | *Synthese* | 多定理综合应用 | 20% | 系统方法论 |
| **研究探索型** | *Forschung* | 前沿问题初步 | 10% | 研究导向 |

### 1.2 "Strenger Stil"（严格风格）特色

ETH Zurich代数几何习题的核心理念：

```
Strenger Stil（严格风格）三原则
│
├─ 1. 定义的严格性 (Strenge Definitionalität)
│   └─ 每个概念必须有明确的形式化定义
│   └─ 例: "平坦性"必须明确是模的平坦还是态射的平坦
│
├─ 2. 证明的完整性 (Beweisvollständigkeit)
│   └─ 不省略任何非平凡的论证步骤
│   └─ 例: 自然性的验证必须显式给出
│
└─ 3. 技术的精确性 (Technische Präzision)
    └─ 计算必须显式展示所有中间步骤
    └─ 例: 谱序列的微分计算必须完整呈现
```

## 6. 严格形式化风格习题

### 6.1 "Strenger Stil"习题设计原则

ETH 401-3532的习题遵循以下设计原则：

```
严格形式化风格 (Strenger Stil Formalisierung)
│
├─ 1. 前提条件明确化 (Prämissenexplizierung)
│   └─ 所有假设必须在问题陈述中明确列出
│   └─ 例: "设 k 为代数闭域，X 为 k 上的光滑射影曲线"
│
├─ 2. 结论精确化 (Konklusionspräzisierung)
│   └─ 要证明的结论必须用形式化语言精确陈述
│   └─ 例: "证明存在典范同构 H^i(X, F) ≅ Ext^{n-i}(F, ω_X)^∨"
│
├─ 3. 证明步骤系统化 (Beweissystematisierung)
│   └─ 证明必须按逻辑步骤组织
│   └─ 每个关键步骤必须引用相关定理或定义
│
└─ 4. 验证完备化 (Verifikationsvollständigkeit)
    └─ 自然性、函子性必须显式验证
    └─ 例: 证明映射的自然性，即证明图表交换
```

### 6.2 严格风格习题示例

**示例**: AG-ETH-001 概形的局部性质

```
【ETH严格风格陈述】
```
