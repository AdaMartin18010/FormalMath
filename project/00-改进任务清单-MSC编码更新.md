## 六、质量保证

### 6.1 编码规范

```yaml
# MSC编码Front Matter标准格式
---
msc_primary: "XX-XX"        # 主编码，单个字符串
msc_secondary:              # 次编码，数组格式
  - 'XX-XX'
  - 'XX-XX'
---
```

### 6.2 验证脚本

```powershell
# 每周执行的验证脚本
$files = Get-ChildItem -Path e:\_src\FormalMath -Recurse -Filter "*.md"
$stats = @{}
foreach($file in $files) {
    $dir = $file.Directory.Name
    if(-not $stats[$dir]) { $stats[$dir] = @{Total=0; MSC=0} }
    $stats[$dir].Total++
    if(Select-String -Path $file.FullName -Pattern "^msc_primary:" -Quiet) {
        $stats[$dir].MSC++
    }
}
# 生成周报
```

### 6.3 错误防范

- **编码验证**: 所有编码必须在MSC2020标准中存在
- **格式检查**: 定期扫描小写xx等格式问题
- **重复检查**: 防止同一文档重复标注
