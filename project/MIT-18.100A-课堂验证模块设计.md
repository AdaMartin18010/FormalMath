# MIT 18.100A 实分析课堂验证模块设计

> **文档编号**：P4-CLASSROOM-18.100A-001  
> **课程代码**：MIT 18.100A  
> **模块主题**：确界原理 + 介值定理 + 中值定理（Completeness → Continuity → Differentiability 三连模块）  
> **适用阶段**：本科二年级（已修单变量微积分、具备初步证明写作能力）  
> **验证周期**：2 周（4 学时课堂 + 2 学时课后 Lean4 实验）

---

## 1. 模块定位与选取理由

### 1.1 为何选取此模块

| 维度 | 说明 |
|------|------|
| **理论连贯性** | 确界原理（实数完备性）→ 极值定理/介值定理（连续性核心性质）→ 中值定理（微分学基石），构成实分析从"存在性"到"应用性"的完整逻辑链 |
| **文档成熟度** | 项目现有文档中，中间值定理与中值定理的 Lean4 形式化已完成（`IntermediateValueTheorem.lean`、`MeanValueTheorem.lean`），单调收敛定理、Bolzano-Weierstrass 定理的自然语言证明已补全 |
| **课堂可操作性** | 4 学时刚好覆盖三连模块的核心证明思路，2 学时 Lean4 实验可让学生亲手验证定理条件与结论 |
| **评估友好性** | 前后测可设计从概念理解（L1）到定理应用（L2）再到简短证明（L3）的梯度题目，效应量易于量化 |

### 1.2 与 MIT 18.100A 原课程的映射

- **原课程 Lecture 2–6**：实数系统、确界原理、序列收敛
- **原课程 Lecture 13–16**：连续性、介值定理、极值定理
- **原课程 Lecture 17–20**：导数、中值定理

本模块将上述内容压缩为一条"最小可行证明链"，重点突出**定理之间的逻辑依赖**与**形式化验证的互补作用**。

## 5. 双语 Lean4 代码使用方案

### 5.1 使用原则

- **课上**：投影展示 Lean4 代码的**关键片段**（每段不超过 15 行），用中文注释解释 tactic 的作用。
- **课后**：学生在自己的 Lean4 环境（VS Code + Lean4 插件）中打开项目文件，完成实验任务单。
- **语言策略**：Lean4 代码保留英文关键字（`theorem`、`have`、`apply`、`exact`），但文档和注释使用中文，降低非英语母语学生的认知负荷。

### 5.2 课上投影片段示例

**片段 1：介值定理（连通性证明的核心步骤）**

```lean
-- 设 f 在 [a, b] 上连续，y 在 f(a) 与 f(b) 之间
theorem intermediate_value {f : ℝ → ℝ} {a b : ℝ} (hab : a ≤ b)
    (hf : ContinuousOn f (Icc a b)) {y : ℝ} 
    (hy : f a ≤ y ∧ y ≤ f b ∨ f b ≤ y ∧ y ≤ f a) :
    ∃ (c : ℝ), c ∈ Icc a b ∧ f c = y := by
  /- 步骤1: [a, b] 是连通的 -/
  have h_connected : IsPreconnected (Icc a b) := connected_Icc hab
  /- 步骤2: 连续函数保持连通性，所以 f([a,b]) 也是连通的 -/
  have h_image_connected : IsPreconnected (f '' Icc a b) := by
    apply IsPreconnected.image h_connected hf
  /- 步骤3: ℝ 中连通集是区间，必包含 f(a) 和 f(b) 之间的所有值 -/
  -- ...（后续步骤略）
```

**片段 2：中值定理（辅助函数构造）**

```lean
/- 构造辅助函数 g(x) = f(x) - [(f(b)-f(a))/(b-a)](x-a) - f(a) -/
let g : ℝ → ℝ := fun x => f x - ((f b - f a) / (b - a)) * (x - a) - f a

/- 验证 g(a) = 0 且 g(b) = 0，从而可应用 Rolle 定理 -/
have h_ga : g a = 0 := by simp [g]
have h_gb : g b = 0 := by simp [g]; field_simp; ring
```

### 5.3 课后实验任务单

**实验 1：确界与单调收敛（60 min）**

## 9. 质量控制与风险预案

### 9.1 教师培训要点（课前 1 周）

- 试讲 30 min，重点练习从自然语言证明过渡到 Lean4 代码的对照讲解。

- 分发教师用"常见学生问答速查表"（基于 `project/MIT-18.100A-L6-思想方法提炼.md`）。

### 9.2 技术风险与应对

| 风险 | 应对措施 |
|------|----------|
| 学生电脑无法配置 Lean4 环境 | 提前准备基于 Docker 的一键启动脚本；或提供远程 JupyterLab 环境 |
| Lean4 编译报错挫伤学生积极性 | 实验任务单提供"完全可编译的 starter 代码"，学生只需填空 |
| 文档中的数学公式显示异常 | 统一使用 VS Code 的 Markdown+Math 插件预览；课前测试 PDF 导出 |

- 前后测试卷版本分为 A/B 卷，题目顺序和数字参数不同，防止抄袭。
- Lean4 实验要求学生提交 git commit 历史，以区分独立完成与复制粘贴。
