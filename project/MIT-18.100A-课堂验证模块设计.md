# MIT 18.100A 课堂验证模块设计

## 1. 模块概述

**课程来源**：MIT 18.100A Real Analysis (Spring 2010)
**模块主题**：实数完备性、连续性与中值定理
**建议课时**：2 周，共 4 学时（每学时 45 分钟）
**适用对象**：大学本科二年级数学/理工科学生

## 2. 学习目标

| 维度 | 目标 |
|------|------|
| **知识 (Knowledge)** | 掌握确界原理、介值定理、中值定理的严格数学陈述 |
| **技能 (Skill)** | 能够写出完整的 $\varepsilon$-$\delta$ 证明；能够阅读并验证 Lean4 形式化代码 |
| **态度 (Attitude)** | 认同严格证明在数学学习中的核心地位；愿意尝试机器辅助验证 |

## 4. 双语 Lean4 代码使用方案

- **课前**：学生预习对应的双语对照文档 `docs/09-形式化证明/双语对照/介值定理-自然语言与Lean4对照.md`
- **课中**：教师投影展示 Lean4 代码，学生逐行对照自然语言证明
- **课后**：学生在本地或在线 Lean4 环境中尝试修改参数并重新编译

## 5. 诊断性测试（前测）

**题目 1**（概念辨析）：陈述实数完备性的三种等价形式（确界原理、单调收敛定理、Cauchy 完备性），并简述它们之间的逻辑依赖关系。

**题目 2**（证明写作）：用 $\varepsilon$-$\delta$ 语言证明 $f(x) = x^2$ 在 $[0, 1]$ 上一致连续。

**题目 3**（形式化理解）：阅读以下 Lean4 代码片段，解释 `intermediate_value_Icc` 的前提条件分别对应介值定理的哪些假设。

## 6. 总结性测试（后测）

**题目 1**（综合证明）：设 $f: [a, b] \to \mathbb{R}$ 连续，$f(a) < 0 < f(b)$。不使用任何"显然"，给出介值定理的完整证明。

**题目 2**（错误诊断）：下面是一个学生写的中值定理证明，请找出至少 3 个逻辑漏洞或表述不严谨之处。

**题目 3**（形式化实践）：在 Lean4 中补全以下 `sorry`：证明若 $f$ 在 $[0,1]$ 上可导且 $f(0)=f(1)$，则存在 $c \in (0,1)$ 使得 $f'(c)=0$（Rolle 定理）。
