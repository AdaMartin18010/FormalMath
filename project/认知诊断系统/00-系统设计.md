## 1. 系统概述

### 1.1 设计目标

FormalMath认知诊断系统基于**层次约束感知认知诊断框架（HCD）**，旨在：

- 准确评估学生在L0-L3四个知识层次的能力表现
- 识别学习优势与薄弱环节
- 生成个性化学习路径建议
- 为自适应学习系统提供数据支持

### 1.2 核心特性

| 特性 | 描述 |
|------|------|
| 层次化诊断 | 基于L0-L3知识层次进行分层诊断 |
| 知识图谱关联 | 关联知识图谱进行关系分析 |
| 个性化报告 | 生成个性化诊断报告与学习建议 |
| 实时反馈 | 支持即时诊断与反馈 |

### 2.3 诊断算法模块

#### HCD算法流程

```
┌─────────────────────────────────────────────────────────────┐
│                      HCD诊断流程                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  输入                                                        │
│  ├─ 学生答题响应 R = {r₁, r₂, ..., rₙ}                     │
│  ├─ 测试题信息 Q = {q₁, q₂, ..., qₙ}                       │
│  └─ 层次约束 C = {c₁, c₂, ..., cₘ}                         │
│                                                             │
│  步骤1: 初始化                                               │
│  ├─ 知识状态 α₀ = 0.5 (均匀初始化)                         │
│  └─ 能力水平 θ₀ = 0.0                                       │
│                                                             │
│  步骤2: 迭代优化 (EM算法)                                    │
│  ├─ E步: 估计知识状态 P(α|R,Q,θ)                           │
│  ├─ M步: 更新能力参数 θ = argmax P(R|Q,θ,α)               │
│  ├─ 应用层次约束 α' = ApplyConstraints(α, C)              │
│  └─ 检查收敛 ‖αₜ - αₜ₋₁‖ < ε                              │
│                                                             │
│  步骤3: 输出                                                 │
│  ├─ 知识状态 α = {α₁, α₂, ..., αₖ}                        │
│  ├─ 能力水平 θ = {θ_L0, θ_L1, θ_L2, θ_L3}                 │
│  └─ 学习建议 S = GenerateSuggestions(α, θ)                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 核心算法伪代码

```python
class HCDAlgorithm:
    """层次约束感知认知诊断算法"""
    
    def __init__(self, knowledge_graph, constraints):
        self.kg = knowledge_graph
        self.constraints = constraints
        self.max_iter = 100
        self.epsilon = 1e-6
    
    def diagnose(self, responses, questions, student_id=None):
        """
        执行认知诊断
        
        Args:
            responses: 答题响应列表 [{question_id, answer, correct, time_spent}]
            questions: 题目信息列表 [Question]
            student_id: 学生ID(可选)
        
        Returns:
            DiagnosisResult: 诊断结果
        """
        # 1. 初始化知识状态
        knowledge_state = self._init_knowledge_state()
        
        # 2. EM迭代优化
        for iteration in range(self.max_iter):
            # E步: 估计知识状态
            knowledge_state = self._e_step(
                responses, questions, knowledge_state
            )
            
            # 应用层次约束
            knowledge_state = self._apply_constraints(knowledge_state)
            
            # 检查收敛
            if self._check_convergence(knowledge_state):
                break
        
        # 3. 评估能力水平
        ability_level = self._assess_ability(knowledge_state)
        
        # 4. 生成学习建议
        suggestions = self._generate_suggestions(
            knowledge_state, ability_level
        )
        
        return DiagnosisResult(
            student_id=student_id,
            knowledge_state=knowledge_state,
            ability_level=ability_level,
            suggestions=suggestions,
            confidence=self._calculate_confidence(responses)
        )
    
    def _apply_constraints(self, knowledge_state):
        """应用层次约束"""
        constrained_state = knowledge_state.copy()
        
        for constraint in self.constraints:
            lower_level = constraint['from']
            upper_level = constraint['to']
            strength = constraint['strength']
            
            # 获取对应层次的知识点
            lower_concepts = self.kg.get_concepts_by_level(lower_level)
            upper_concepts = self.kg.get_concepts_by_level(upper_level)
            
            # 计算下层平均掌握度
            lower_mastery = np.mean([
                knowledge_state.get(c, 0) for c in lower_concepts
            ])
            
            # 约束上层掌握度不能超过下层太多
            for concept in upper_concepts:
                current = constrained_state.get(concept, 0)
                max_allowed = min(1.0, lower_mastery + (1 - strength) * 0.3)
                constrained_state[concept] = min(current, max_allowed)
        
        return constrained_state
```

### 3.2 数据表结构

#### 学生表 (students)

```sql
CREATE TABLE students (
    id VARCHAR(36) PRIMARY KEY,
    username VARCHAR(50) NOT NULL,
    email VARCHAR(100) UNIQUE,
    current_level INTEGER DEFAULT 0,  -- 当前学习层次 L0-L3
    learning_style VARCHAR(20),       -- 学习风格
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

#### 题目表 (questions)

```sql
CREATE TABLE questions (
    id VARCHAR(36) PRIMARY KEY,
    content TEXT NOT NULL,            -- 题目内容
    type VARCHAR(10) NOT NULL,        -- SC/MC/FB/SA/PR
    level INTEGER NOT NULL,           -- 0-3 对应L0-L3
    difficulty INTEGER NOT NULL,      -- 1-5
    branch VARCHAR(50),               -- 数学分支
    concepts JSON,                    -- 关联知识点 ["concept1", "concept2"]
    skills JSON,                      -- 考察技能
    options JSON,                     -- 选项(选择题)
    answer TEXT,                      -- 正确答案
    explanation TEXT,                 -- 解析
    time_estimate INTEGER,            -- 预估用时(秒)
    score DECIMAL(5,2),               -- 分值
    prerequisite_concepts JSON,       -- 先修知识点
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

#### 测试记录表 (test_records)

```sql
CREATE TABLE test_records (
    id VARCHAR(36) PRIMARY KEY,
    student_id VARCHAR(36) NOT NULL,
    test_type VARCHAR(20),            -- diagnostic/practice/exam
    start_time TIMESTAMP,
    end_time TIMESTAMP,
    status VARCHAR(20),               -- pending/completed/aborted
    total_score DECIMAL(5,2),
    FOREIGN KEY (student_id) REFERENCES students(id)
);
```

#### 答题记录表 (responses)

```sql
CREATE TABLE responses (
    id VARCHAR(36) PRIMARY KEY,
    test_id VARCHAR(36) NOT NULL,
    question_id VARCHAR(36) NOT NULL,
    answer TEXT,                      -- 学生答案
    is_correct BOOLEAN,
    score DECIMAL(5,2),
    time_spent INTEGER,               -- 用时(秒)
    FOREIGN KEY (test_id) REFERENCES test_records(id),
    FOREIGN KEY (question_id) REFERENCES questions(id)
);
```

#### 诊断结果表 (diagnosis_results)

```sql
CREATE TABLE diagnosis_results (
    id VARCHAR(36) PRIMARY KEY,
    student_id VARCHAR(36) NOT NULL,
    test_id VARCHAR(36),
    knowledge_state JSON,             -- 知识状态 {"concept": mastery_level}
    ability_level JSON,               -- 能力水平 {"L0": score, "L1": score...}
    weak_areas JSON,                  -- 薄弱环节
    strong_areas JSON,                -- 优势领域
    confidence DECIMAL(3,2),          -- 诊断置信度
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (student_id) REFERENCES students(id)
);
```
