# UI原型设计

**文档编号**: CDS-UI-001  
**版本**: v1.0  
**创建日期**: 2026年4月2日

---

## 1. 界面架构

### 1.1 系统结构

```
FormalMath认知诊断系统
├── 学生端
│   ├── 登录/注册页面
│   ├── 仪表盘 (Dashboard)
│   ├── 诊断测试页面
│   ├── 诊断报告页面
│   ├── 学习路径页面
│   └── 个人中心
│
└── 教师/管理端
    ├── 仪表盘
    ├── 题库管理
    ├── 学生管理
    ├── 诊断分析
    └── 系统设置
```

### 1.2 设计原则

| 原则 | 描述 |
|------|------|
| 清晰性 | 数学公式清晰展示，使用LaTeX渲染 |
| 渐进式 | 诊断流程循序渐进，避免认知过载 |
| 反馈性 | 及时反馈，可视化展示诊断结果 |
| 个性化 | 根据诊断结果个性化推荐 |

### 5.3 核心组件示例

```tsx
// FormulaRender.tsx - 数学公式渲染组件
import React from 'react';
import katex from 'katex';
import 'katex/dist/katex.min.css';

interface FormulaRenderProps {
  formula: string;
  display?: boolean;
}

export const FormulaRender: React.FC<FormulaRenderProps> = ({
  formula,
  display = false
}) => {
  const html = katex.renderToString(formula, {
    throwOnError: false,
    displayMode: display
  });

  return (
    <span 
      dangerouslySetInnerHTML={{ __html: html }}
      className={display ? 'formula-display' : 'formula-inline'}
    />
  );
};
```

```tsx
// AbilityChart.tsx - 能力水平雷达图
import React from 'react';

interface AbilityChartProps {

    L0: number;
    L1: number;
    L2: number;
    L3: number;

        { name: 'L2\n定理应用', max: 100 },
        { name: 'L3\n综合证明', max: 100 }
      ]

      type: 'radar',
      data: [{
        value: [data.L0 * 100, data.L1 * 100, data.L2 * 100, data.L3 * 100],
        name: '能力水平',
        areaStyle: { color: 'rgba(0, 128, 255, 0.3)' }
      }]
    }]
  };

  return <ReactECharts option={option} style={{ height: 300 }} />;
};
```

## 6. 交互设计

### 6.1 诊断流程交互

```
用户点击[开始诊断]
    ↓
系统显示准备页面（说明诊断流程）
    ↓
用户点击[开始测试]
    ↓
系统加载题目（显示loading）
    ↓
逐题作答（支持跳题、标记）
    ↓
用户点击[提交]
    ↓
系统显示处理中（进度条动画）
    ↓
跳转至报告页面
    ↓
展示诊断结果 + 学习建议
```

### 6.2 反馈设计

| 场景 | 反馈方式 |
|------|----------|
| 答案提交 | Toast提示，显示正误 |
| 诊断完成 | 庆祝动画 + 报告展示 |
| 能力提升 | 徽章系统 + 进度提示 |
| 错误输入 | 即时表单验证 |
