## 🎯 如何使用这些学习计划

### 1. 确定您的起点

根据您的数学背景选择合适级别的课程开始：
- **初学者**: 从18.090和18.700开始
- **有基础者**: 可直接从Stage 2开始
- **准备研究生学习**: 重点参考Stage 3课程

### 2. 检查先修要求

每门课程的学习计划都包含详细的先修要求检查清单。在开始之前，请确保您已掌握：
- 列出的前置知识
- 建议的FormalMath文档

### 3. 按周学习计划进行

每门课程都分为14周的学习计划，包含：
- **主题**: 本周学习重点
- **FormalMath文档**: 具体对应文档及阅读时间
- **MIT教材**: 参考章节
- **自测检查点**: 验证理解程度

### 4. 定期自测

利用学习计划中的自测检查点，定期验证您的理解：
- 完成每周的练习题
- 尝试证明课程中的定理
- 解决应用问题

## 📊 学习计划结构

每份学习计划包含以下部分：

```

   - 课程信息
   - 学习目标

   - 必备知识检查表
   - 推荐复习文档

📅 周学习计划 (14周)
   - Week 1-2: 主题
     * MIT教材章节
     * FormalMath对应文档（含时间）
     * 核心定理
     * 练习题
     * 自测检查点
   - ...

⚠️ FormalMath覆盖缺口
   - 需补充的内容

📚 补充阅读材料
   - 必读经典
   - 进阶参考

📝 评估与自测
   - 期中/期末测试
```
