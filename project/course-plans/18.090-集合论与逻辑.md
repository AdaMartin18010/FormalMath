# MIT 18.090 集合论与逻辑 自学路径

> **课程**: 18.090 Set Theory and Logic  
> **学分**: 12  
> **先修要求**: 无  
> **MIT教材**: Enderton, *Elements of Set Theory*; Hamilton, *Logic for Mathematicians*

---

## 📋 课程概述

18.090是MIT数学系的入门课程，建立严格的集合论基础和数理逻辑基础。这门课是所有后续数学课程的基石。
