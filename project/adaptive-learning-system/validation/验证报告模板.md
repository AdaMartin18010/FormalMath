## 1. 引言 / Introduction

### 1.1 研究背景

[在此插入研究背景：传统数学教材在证明教学中的不足、形式化方法在教育中的兴起、FormalMath 项目的愿景等。]

### 1.2 研究问题

本研究试图回答以下核心问题：

1. **RQ1**：使用 FormalMath 银层文档的学生是否在期末综合测试中表现显著优于使用传统教材的学生？
2. **RQ2**：实验组在证明题模块上的优势是否大于计算题模块？
3. **RQ3**：实验组学生对文档的满意度是否显著高于对照组学生对教材的满意度？
4. **RQ4**：双语对照（自然语言 + Lean4）对学生的理解是促进作用还是认知负担？

### 1.3 研究假设

- **H1**：实验组期末综合测试成绩显著高于对照组（Cohen's d ≥ 0.5）。
- **H2**：实验组证明题得分率显著高于对照组（差距 ≥ 15%）。
- **H3**：实验组学生满意度 ≥ 4.0/5.0，且显著高于对照组。
- **H4**：至少 60% 的实验组学生认为 Lean4 双语对照有助于理解。
