# FormalMath 银层文档质量评估标准

## 评分维度 (满分100)

### 1. 数学准确性 (30分)
- **定义**: 严格、无歧义、符号规范
- **定理**: 陈述完整、条件明确
- **证明**: 逻辑严密、无跳步
- **评分**: 无错误30分，每处错误扣5分

### 2. 教学有效性 (25分)
- **直观解释**: 有清晰的几何/物理/计算直观
- **例子**: 有典型例子和反例
- **难度梯度**: 适合目标读者
- **评分**: 优秀25分，良好20分，一般15分

### 3. Lean4形式化 (20分)
- **import**: 使用正确的mathlib模块
- **定义**: 与数学定义一致
- **定理**: 可编译通过
- **评分**: 完整20分，部分10分，缺失0分

### 4. 习题设计 (15分)
- **数量**: 每文档≥3题
- **难度**: 有梯度（易/中/难）
- **解答**: 有提示或完整解答
- **评分**: 完整15分，部分10分

### 5. 交叉引用 (10分)
- **前置**: 明确列出前置知识
- **后继**: 链接到相关高级主题
- **内部**: 链接到项目内相关文档
- **评分**: 完整10分，部分5分

## 质量等级

| 总分 | 等级 | 说明 |
|------|------|------|
| 90-100 | 金层候选 | 可升级为金层 |
| 75-89 | 银层优秀 | 维持银层 |
| 60-74 | 银层合格 | 需要改进 |
| <60 | 铜层 | 需要大幅修改 |
