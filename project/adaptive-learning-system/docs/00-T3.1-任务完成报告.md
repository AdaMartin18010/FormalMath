## 🎯 核心功能演示

### 1. 学习者特征分析

```python
# 创建学习者
learner = LearnerProfile(name="张三", email="zhangsan@example.com")

# 评估学习风格
learner.learning_style = LearningStyleProfile(
    visual_score=0.7,    # 视觉型
    auditory_score=0.1,  # 听觉型
    kinesthetic_score=0.1,  # 动手型
    reading_score=0.1    # 阅读型
)

# 设置目标和时间
goal = GoalSetting(
    target_concepts=["群", "环", "域"],
    target_level=AbilityLevel.L2
)
time = TimeAvailability(
    daily_available_hours=2,
    weekly_available_days=5
)
```

### 2. 学习路径生成

```python
# 创建规划器
planner = AStarPathPlanner(knowledge_graph, learner)

# 生成路径
constraints = PathConstraints(
    max_total_time_hours=100,
    diversity_weight=0.2
)
path = planner.find_optimal_path(
    start_concepts=[],
    goal_concepts=["群", "环", "域"],
    constraints=constraints
)
```

### 3. 实时调整

```python
# 检测困难
detection = monitor.detect_difficulty(
    node_id="node-1",
    current_metrics={
        "time_spent": 120,  # 分钟
        "error_count": 3,
        "help_requests": 2
    }
)

# 如果太难，自动调整
if detection.signal == DifficultySignal.TOO_HARD:
    adjustment = engine.adjust_based_on_feedback(progress, detection)
```

### 4. 间隔重复

```python
# 计算复习间隔
scheduler = SpacedRepetitionScheduler(path, learner)
intervals = scheduler.calculate_review_intervals(
    concept_id="集合",
    mastery_level=0.85,
    review_count=0
)
# 结果: [3, 7, 14, 30] 天
```
